> **編**: α編

## 計算式

- `F′ = μ₅P`
- `F′ = μ₆P`
- `μ² = kP/d₁`
- `トラクション(traction)`
- `P′ = Wcos α, F′ = Wsinα = P′tanα`
- `F = Wsinα + μ₆P′ = W(sinα + μ₆cos α)`
- `F = W(μ₆cos α - sinα)`
- `ΔF′ = ΔT = μ₆TΔϕ`
- `T₂ = T₁e⁻`
- `F = 2(Psinα + F′cos α) = 2P(sinα + μ₆cos α)`
- `= 2P(sinα + tanρ·cos α)`
- `F = 2P(μ₆cos α - sinα) = 2P(tanρ·cos α - sinα)`
- `Fcos α - Wsinα = μ₆P′`
- `Fsinα + Wcos α = P′`
- `F = W(sinα + μ₆cos α)/(cos α - μ₆sinα)`
- `= W(tanα + tanρ)/(1 - tanρ·tanα)`
- `= Wtan(α + ρ)`
- `F = W(dp + πdμ₆)/(πd - pμ₆)`
- `F = Wtan(ρ - α)`
