# 1・1・1 機械(machine)とは

> **章**: 第1章 機械工学概説
> **節**: 1・1 総説

機械とは何か.これを簡潔かつ適切に表現することは難しい.機械の種類は極めて多く,千差万別だからである.日本語の機械という言葉は紀元前3~4世紀ころ,中国の古典,『荘子』ならびに『韓非子』という書物にさかのぼる.機とは仕掛けのこと,械は刑罰の道具だから,機械とは仕掛けをもってむりやり何かをさせるもの,ということになる.英語のマシン(machine)はギリシャ語のメカネー,ラテン語のマキナに由来し,これも機構や巧みな仕掛けを意味する言葉である.古今東西,さまざまな人が与えた機械の定義はその時代を反映している.特に有名なものを以下に掲げる.・荘周(c.365~c.290BC,古代中国,戦国時代の思想家)ここに械あり,力を用うること甚だ寡(すくな)くして,而も功を見(あら)わすこと多し.『荘子(外編・天地12)』より・韓非(c.280~c.233BC,古代中国,戦国時代の思想家)舟車機械の利は力の少なきを用いて功大を致す.すなわち多くの利入る.『韓非子(巻15・雑:第37)』より・ヴィトルヴィウス(Vitruvius Polio,M.,紀元前1世紀ころ,古代ローマの建築家)機械(マキナ)とは木材を組合わせたもので,おもに重いものを動かすのに用いる.『建築術』より・ロイポルト(1674~1727,ドイツの機械学者)機械とは人工的製作物で,その助けをかりて運動を起こすことができ,時間または力が節約される.『機械の劇場』(1724年)より・ルーロー(Reuleaux,F.,1829~1905,ドイツの機械工学者)機械とは,力に対して抵抗を示す物体が組合わせられて定まった相対運動を行い,外部からエネルギーを受取って有用な仕事をするものをいう.『理論運動学』(1875年)よりルーローの定義は現代の辞書・事典類にそのまま引用されることが多いが,これにも19世紀後半という時代的な制約がある.この定義は動力機械と作業機械が機械の主流だった当時の状況を反映しており,現代の観点からはいくつかの問題がある.一つは物理量や機械量の指示・記録を目的とする(すなわち,機械的な仕事の遂行が目的ではない)計測機器がこの定義から外れること,もう一つはこの定義ではエレクトロニクスが含まれないことである.計測機器の発展と普及は20世紀前半から,エレクトロニクスの発展と機械への応用は20世紀後半のことである.機械は時代とともに発達し変化していくから,その定義もまた変わっていくのは当然である.これらを考慮して現代における機械を改めて定義すれば,機械とは次の三つの要件を満たすもの,ということができる.(1) 機械は人工物であり,それぞれ特定の役割を担う物体(機械要素,流体要素,電気・電子要素など)の集合体である.(2) これらの物体は意味あるように組合わされ,全体として目的にかなった機能を現出する.(3) 目的とする機能の現出に当たっては,機械的な力と運動の両方あるいはいずれか一方が重要な役割を果たす.ところで,はさみやノギスなどは上記三つの要件をすべて満たしてはいるが,慣習上これらは工具あるいは道具といって機械とはいわない.また,機械工学が扱う範囲のものであっても,構造部分が運動を伴わない溶鉱炉や圧力容器(タンク)などは装置といって,機械とは区別する.また,計測器のように小型で,かつ機械的な仕事の遂行を目的としないものを器具ということがある.機械と器1―1具,さらに装置をも含めて機器という.現代ではコンピュータを俗にマシンということがあるが,これは機械とは考えないのがふつうである.コンピュータが目的とする機能(論理演算,情報処理など)はもっぱら電子装置によって達成され,機械的な力または運動によるものではないからである.逆に情報関連機器であっても,プリンタやビデオ機器,自動改札機などは機械構造とその運動が重要な役割を担うので,りっぱな機械である.
