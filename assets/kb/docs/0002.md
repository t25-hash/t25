# 1・1・2 機械の種類と分類

> **章**: 第1章 機械工学概説
> **節**: 1・1 総説

機械の種類を区分するには機能別、用途別など、視点によるさまざまな分類法がある。下記は最も一般的なものである。a. 動力機械(prime mover)動力(力と運動速度の相乗積)の発生を目的とする機械で、原動機ともいう。動力機械は力を発生するだけではなく、運動の発生を伴う[例:蒸気タービン、内燃機関、動力水車(電気モータを含めることもある)].b. 作業機械(working machine)外部から動力の供給を受けて所要の作業を行う機械で、一般に作業対象物の変形あるいは移動・運動を伴う[例:工作機械、産業機械(紡績機械、ブルドーザなど)、交通・輸送機械、流体機械、空調機、ミシン、洗濯機].c. 計測機械(measuring machine or instrument)物理量・機械量の測定(指示、記録、判定など)を目的とする機械(例:測長機、材料試験機、振動計、流量計、時計)。d. 情報・知能機械(information or intelligent machine)情報を扱い、その記憶・演算・判断等の機能を備えた機械(例:コンピュータの周辺機器、ビデオ機器、自動改札機、知能ロボット)。e. その他医療・福祉用機器、娯楽・遊戯機械、タイプライタ、カメラなど。ただしこれらは広義の作業機械(カメラは計測機械でもある)と見ることもできる。機械の分類には上記のほか、産業・用途別分類(農業機械、土木機械、鉱山機械、繊維機械、事務機械、家庭用機械など)、動作原理による分類(水力機械、電気機械、化学機械など)がある。機械が発現する運動の形態による分類(往復機械、回転機械)もある。
