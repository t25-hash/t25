# 1・1・3 機械工学(mechanical engineering)とは

> **章**: 第1章 機械工学概説
> **節**: 1・1 総説

機械(単体ならびにシステム)の開発、設計、製作、運転、保全に関わる技術が機械技術であり、機械技術と機械製品に関する学問と知識の体系が機械工学である。技術と工学の関係については項を改め、1・3節で詳しく述べる。機械技術と機械工学では製品の機能・性能はもちろんのこと、経済性と安全性が十分に確保されなければならない。その他、諸法規への適合、人間性(生理、心理、倫理道徳など)との衝突を避けることなど、人文・社会事象への配慮も重要なことがらである。現代の機械工学はおおよそ次のような内容をもつ。a. 基礎工学分野機械的エネルギーあるいは機械情報のふるまいと操作を扱う工学分野である。工業力学、機械力学、材料力学、流体工学、熱工学を中心として、計算工学、情報関連工学を含む。工業数学をここに含めることもある。b. 生産工学分野機械単体ならびに機械システムの設計と生産に関係する工学分野である。これには材料学、材料工学、加工学と加工機器、機構学、機械要素・トライボロジー、計測・制御工学、生産システム工学、生体工学などがある。機械設計法と機械製図法もここに含まれる。c. 応用工学分野特定の機械あるいは機械システムの種類ごとに固有な技術を扱う工学分野である。対象となる機械の種類には、産業機械・装置、流体機械、熱原動機、熱機器、交通機械、ロボット、情報機器、医療・福祉機器、資源・環境システム、宇宙機器などがある(例:自動車工学、ロボット工学)。d. 関連工学分野それ自身は機械工学には属さないが、機械の設計・製作・取扱い等に密接に関係する他の工学分野である。電1―2気・電子工学、管理工学、化学工学などがこれに当たる。生物工学を含めることもある。e. 社会関連分野技術者が社会の中で専門職(profession-al)として職業活動を行うとき、その地位にふさわしい識見と行動規範を与える学問・知識である。技術史・工学史、技術論(技術哲学、技術社会学)、技術倫理、法工学(関連諸法規、知的財産権)などがある。この機械工学便覧においては、上記の5分野を再編成して、次の3編から成る三部構成としている。α. 基礎編:上記のa項、ならびにd項とe項の一部β. デザイン編:上記のb項、ならびにd項とe項の一部γ. 応用システム編:上記のc項論
