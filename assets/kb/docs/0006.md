# 1・2・3 近代ヨーロッパにおける動力学の誕生と発展

> **章**: 第1章 機械工学概説
> **節**: 1・2 ).しかしこの知識も学問化されることはなかった.西洋では中

a. 社会の思想的背景現代機械工学の基礎学問となる力学,特に動力学(どうりきがく,dynamics)の誕生と発展は17~18世紀の西ヨーロッパの産物である.これには当時の社会における思想的な背景があった.17世紀のヨーロッパは国民を国家統合の要とする近代国民国家の形成期に当たる.経済の発展で社会は活気に満ち,学者の間にも新しい動きが見られた.16世紀の宗教改革を通して精神の高揚を見た学者たちの間には,キリスト教の信仰に基礎をおいた,燃えるような自然探究心が起こった.目標は自然の中に隠された神の創造に関わる秩序の発見である.神が定めたこの秩序に「自然法則(lex naturae)」という名を与えたのは中世13世紀の神学者でスコラ哲学者のトマス・アクィナスであるが,ガリレイからニュートンに至る学者たちの研究の動機はまさに,自然法則の発見にあったのである.1―4機械工学総論図1・4ガリレイによる木材片持ばりの強度試験,『新科学対話』(1638年)よりb. ガリレイ,機械学と動力学の芽生え機械とその作用を真の意味で初めて学問的に扱ったのはガリレオ・ガリレイ(GalileoGalilei)である.彼はヴェネチアの造船工場に頻繁に出入りして技術者の仕事ぶりを観察した.実験をたいせつにする彼の態度はこれに由来している.彼の初期の論文に『レ・メカニケー(機械の学問)』(1593~1599年ころ)がある.ここで彼は初めて力のモーメントの概念を案出して,「機械は得をしないこと」を明らかにした.てこでの力の利得はてこの動きの増大で相殺されるのである.ガリレイの晩年の著作『新科学対話』(1638年)は動力学に関する彼の研究の総決算である.落体の運動はここで扱われている.この書物で彼ははりや棒の強度を扱い,近代的な材料力学の先駆となった(図1・4).彼はまた幾何学的に相似な2物体の強度を論じて,「巨人の骨は並外れて太いはずだ」といっている.力学的相似則の萌芽である.c. ホイヘンス,動力学(dynamics)への貢献ガリレイの動力学をさらに発展させたのはオランダのホイヘンス(Huygens,C.)である.彼が最も力を注いだのは航海用クロノメータの開発であった.著書『振子時計』(1673年)は彼の研究の結晶であるが,この書物は機械書というよりは力学書であって,サイクロイド曲線の性質,慣性モーメント,遠心力などが扱われており,回転運動に関する基礎概念の導出とその解析への端緒はこの書物によって与えられた.d. ニュートン,動力学の完成ガリレイに始まった動力学の理論的研究はホイヘンスを経て,ニュートン(Newton,I.)によって集大成された.その成果は著書『自然哲学の数学的原理(略称:プリンキピア)』(1687年)にまとめられている.この書物の核心は重力(万有引力)の導入である.ニュートンが力の概念を導入したことで従来の運動学は力学となり,力学の基本原理として運動の3法則が提示された.彼はこの書物の中で流体力学,振動,波動をも扱っている.ただしラテン語で書かれたこの書物の文章表現は極めて難解で,物体の運動に関する記述はすべて幾何学(図形)で表現され,代数式は一つもない.これでは本書の内容を技術に応用することは不可能である.本書はもともと学者のために書かれた哲学論考であって,ニュートンの時代には科学(当時の言葉では自然哲学)を技術に応用するという考えは全くなかったのである.e. 解析力学(analytical mechanics)への道難解なニュートンの力学を,解析力学という誰にでも理解できる理論体系にまとめあげたのはベルヌーイ一族[特にヨハン・ベルヌーイ一世(Ber-noulli, JohannI)とその息子ダニエル],オイラー,ダランベール,ラグランジュら,18世紀ヨーロッパ大陸の数学者たちである.彼らのほとんどは宮廷のお抱え数学者であって,当時生まれたばかりの新数学(微積分学と三角関数)を駆使してニュートンが示した力学の原理と体系を代数的表現に書き直し,質点の運動だけでなく剛体から多自由度系,さらに連続体までを含む理論体系を作り上げた.解析力学である.彼らはまた棒・はり・板などの変形や振動を図1・5 オイラーと『力学;解析による運動の科学』(1736年)解析し,流体の運動を解析して現代の材料力学と流体力学の基礎を築いた.現代の機械工学にとって特に重要な業績としては,ダニエル・ベルヌーイ(Bernoulli, D.)では流れに関するベルヌーイの定理のほか,多自由度系と連続系の振動モードがある.オイラー(Euler, L.)には長柱の座屈,歯車の歯形曲線,流体運動の基礎方程式があり,反動水車の研究ではキャビテーションにも言及する.ダランベール(D’Alembert, J. le Rond)には弦の振動(波動方程式)と流体抵抗,ラグランジュ(Lagrange,J.L.)には流体運動の解析がある.力学原理の整理と代数式による定式化,それに力学の体系化に関して彼らにはそれぞれ重要な著作がある.中でもオイラーの『力学;解析による運動の科学』(1736年,図1・5),ダランベールの『動力学概論』(1743年),そしてラグランジュの『解析力学』(1788年)が特に重要である.理論的な流体力学ではダニエル・ベルヌーイの『流体力学』(1738年)のほか,オイラーとダランベールにもそれぞれ重要な論文がある.ただし,彼らが作った解析力学や流体力学の体系は理論的すぎて,経験を重んじる当時の機械技術者の関心を呼ぶことはなかった.彼ら18世紀の数学者たちが力学の問題を扱ったのは,もっぱら応用数学の題材としての学問的関心によるものであって,実際の技術問題への適用を考えたのではなかった.解析力学を完成させたラグランジュは「力学は経験科学ではなく,応用数学にすぎない」とまでいっている.彼らの作った学問が実際の技術に用いられるようになるのは,産業革命が成熟した19世紀になって各国に高級工科学校が設立されて工学の研究と教育が進み,学校出の技術者が産業化社会で活躍するようになってからのことである.
