# 1・2・4 エネルギー・環境関連法規

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・2 エネルギーシステム

環境エネルギー関係の法律は，環境六法 によくまとめられて
いる．熱工学の関係者に必要な関連法規は以下のとおりである．
a.環境関係 環境問題（environmental problem）に対する
国の姿勢は「環境基本法」に示され，そこに，国は「環境基本計
画」を策定し発表することが定められている．これに従って策定さ
れた環境基本計画には，国の環境保全に関する基本方針が示されて
いる．その全体は膨大な内容であるが，国が対策を実施すべき環境
問題の分野は，①地球規模の大気環境の保全：地球温暖化対策，オ
ゾン層保護対策，②そのほかの大気環境保全：酸性雨，光化学オキ
シダントなど，③水環境，土壌環境，地盤環境，④廃棄物・リサイ
クル対策等の物質循環，⑤化学物質対策，⑥自然環境の保全と自然
とのふれあい，の6項目であるとしている．
この基本計画に沿って，個々の環境問題に対処するようそれぞれ
の法律が定められており，また，今後整備されていく．現在の法律
として注目すべきものは，
全般：環境基本法，環境基本計画，環境影響評価法
地球温暖化防止およびオゾン層保護関係：地球温暖化対策の推進に
関する法律，特定物質の規制等によるオゾン層の保護に関する
法律
大気汚染防止関係：大気汚染防止法
水質汚濁防止関係：水質汚濁防止法
廃棄物および資源関係：廃棄物の処理及び清掃に関する法律，資源
の有効な利用の促進に関する法律，ダイオキシン類対策特別措
置法，循環型社会形成推進基本法
を挙げることができる．
b.エネルギー関係 エネルギーを扱う事業のための法律とし
て，電気事業法，ガス事業法，石油事業法，熱供給事業法がある．
それぞれその事業を行うときは事業の計画を提出し，経済産業大臣
の許可を受けることなどが定められている．
c.省エネルギーの法律 昨今のエネルギー供給の状況により，
新しいエネルギー源の導入が必要であるとの見地から，新エネルギ
ー利用等の促進に関する特別措置法，石油代替エネルギーの開発及
び導入の促進に関する特別措置法が定められ，政府がその分野の事
業を推進すべきであるとされている．
また，わが国はエネルギーの合理的使用すなわち省エネルギー
を，産業，民政，運輸の各分野において，産業政策および地球環境
保全の両面から強力に推進する必要があるとの判断から，エネルギ
ー使用の合理化に関する法律（1979年6月22日制定，最終2006
年4月1日一部改訂）が制定され，国民の省エネルギーへの努力が
要請されている．この法律は第二次オイルショックの年に制定さ
れ，細かい改訂を加えて現在に至っているもので，その体系を図
1･14 に図示する．
本法律は燃料資源の有効な利用の確保に資するため，工場，建築
物および機械器具についてエネルギーの使用の合理化に関する所要
の措置を講じて，国民経済の健全な発展に寄与することを目的とし
ている．
