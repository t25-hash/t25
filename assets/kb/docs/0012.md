# 1・2・8 流体工学(fluids engineering)の誕生と発展

> **章**: 第1章 機械工学概説
> **節**: 1・2 ).しかしこの知識も学問化されることはなかった.西洋では中

流体工学とは,流れに関する学問と,管路・噴流・翼列など実際の流体機器に関わる技術的・工学的知識の体系をいう.歴史的には水路・管路・水力機械に現れる現象を扱う経験的色彩の濃い水力学と,流体の運動を数理物理学的に解析する理論的色彩の濃い流体力学,およびその応用に大別される.a. 水力学(すいりきがく,hydraulics)流体の知識と学問は古代にまでさかのぼる.静止流体については古代のアルキメデス[Archimedes(of Syracuse)]からルネサンス期のステヴィン(Stevin,S.)を経て17世紀のパスカル(Pascal,B.)でほぼ完成する.17~18世紀のヨーロッパは大土木時代となって,ポンプと動力水車が先端機械として重用された.技術者の間では流れる水の性質と挙動に関心が高まり,18世紀の中ごろまでに水力学の基礎ができあがった(1・2・4・c項参照).19世紀に入ると新型の水力機械や水力タービンが続々と現れ,管路や水力機械に関してイギリスのスミートン(Smeaton,J.),フランスのマリオット(Mariotte,E.)やパラン(Parent, A.)らは実用的な研究をさらに進めた.マリオットには『水・流体の運動理論』(1686年)という有名な著書がある.b. 流体力学(fluid dynamics)・応用流体力学流れの解析的扱いはニュートン(Newton, I.)が『プリンキピア』(1687年)第2編に示した粘性法則に始まった.18世紀の数学者ダニエル・ベルヌーイ(Bernoulli, D.),オイラー(Euler, L.),ダランベール(D’Alembert, J. le Rond),ラグランジュ(Lagrange, J.L.)らは当時最新の微積分学を駆使して理論的な流体力学の基礎を築いた.1―9ベルヌーイの定理(1738年),オイラーの基礎方程式(1755年)は特に有名である.ただし,ここで扱われたのは粘性も圧縮性もなく渦もない,完全流体であった.19世紀に入ると水力機械はいっそう発展し,大型蒸気船も出現したことで流れの力学は新しい段階を迎えた.流れを全体として水力学的にとらえるのではなく,乱流とか渦とかの現象ごとに分けて解析的に扱う応用流体力学の出現である.実在流体が研究の対象となり,19世紀の半ば,イギリスの数学者ストークス(Stokes,G.G.)は粘性流体の運動を研究し,フランスの土木技師ナビエ(Navier, C.L.M.H.)が先に行った研究を補完して1845年,ベルヌーイの式に粘性項を加えたナビエ・ストークスの方程式(N-S 方程式)を明らかにした.これは実在流体の基礎方程式として名高い.イギリスのレイノルズ(Reynolds, O.)も粘性流の研究を行い,1883年には流れの力学的相似性を与え,かつ層流と乱流を区別する指標としてレイノルズ数を示した(図る(1・2・5・b項参照).これはトライボロジー(表面・摩擦工学)という学問の始まりともいえる出来事である.c. 高速気体力学(high-speed gas dynamics)1903年,ライト兄弟(Wright Brothers)が発明した飛行機は第一次世界大戦(1914~1918年)によって構造も性能も格段に進歩し,現代にまで及ぶ航空機時代を生み出した.航空機の性能をさらに向上させるためには実験と経験だけではもはや不可能で,高速気体力学を含む流体工学の発展が不可欠となった.翼の揚力理論の発見と展開は1902~1908年ごろのことで,クッタ,ジュウコフスキー,ランチェスターの3人の功績である.プラントル(Prandtl, L.)は境界層の発見(1904年)や風洞実験法の確立などの功績で近代流体力学の父といわれる.『航空流体力学』(1929年)はプラントルの講義を弟子のティーチェンスが書物にしたもので,名著の誉れが高い(図1・13).図1・12 レイノルズと流れの状態に関するレイノルズの実験図1・13 プラントルと『航空流体力学』第2巻(1931年),「境界層」の部分1―10機械工学総論第二次世界大戦後,航空機の速度は音速の壁を超えた.この時代の高速気体力学をリードしたのはプラントルの弟子カルマン(Ka썝rma썝n,T.von)である.彼はカルマン渦の発見で名高く(1・2・7・e項参照),のちにアメリカ航空宇宙局(NASA)の初代所長として超音速機の研究に当たった.ロケットの研究ではオーストリアのゼンガーの著書『ロケット航空工学』(1933年)が先駆的なものである.d. 計算流体力学(computational fluid dynamics),その他最近のコンピュータの進歩は流体力学に大きな影響を与えた.これは1970年代以降のことであって,与えられた境界条件のもとでN-S 方程式の解を数値的に求めることが可能となり,これにより流体の学術と技術は大きく発展した.この分野は計算流体力学(数値流体力学)といわれる.実験流体力学の新しい展開でも特筆すべきものがある.流れの可視化をはじめとする最近の流体計測の発達は電子機器の発達に負うところが大きい.コンピュータによる流れの数値シミュレーションや,コンピュータ支援による実験技術の進歩もまた著しい.その他,流体力学における最近の話題では,希薄気体力学,非ニュートン流体を扱うレオロジー(流動学),電磁流体力学,混相流を扱う力学,機能流体(磁性流体や電気粘性流体など,流れが制御できる流体),生体内の流体(血管,呼吸器系など)等がある.
