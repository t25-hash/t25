# 1・2・12 第二次世界大戦後の機械工学の新しい展開

> **章**: 第1章 機械工学概説
> **節**: 1・12 )。滑り軸受の潤滑理論を打立てたことも彼の大きな功績であ

a. 戦後の技術革新(technology innovation)と機械工学の発展第二次世界大戦(1939~1945年)以後の機械技術と機械工学の発1―13達は戦時中の技術、中でも精密計測・制御機器、航空機、電子機器の発達に負うところが大きい。戦争末期にはロケットやレーダ、コンピュータも現れた。これらの機器の発達と並行する形で、戦後間もないころ、計測・制御工学がまず学問として体系化された。第二次世界大戦中のOR(operations research、作戦研究)に出発し、戦後の1950年代に発展した技術分野にシステム工学、インダストリアルエンジニアリングがある。これはシステム全体の設計と特性を調べ、信頼性・安全性を研究するもので、「技術を扱う技術」を研究する学問といえる。ここで得られた知見はいよいよ大規模化する機械システムの発展にも大きな役割を果たすこととなった。戦後間もない1948年にはトランジスタが発明され、演算増幅器を含む電子機器の小型化と高性能化が進んだ。この時期にはエレクトロニクス(電子工学技術)が大きく発展して、機械においてもその動作の支援と制御に応用され、メカトロニクス(mecha-tronics)という新分野が生まれた。メカトロニクスという言葉は、もとは安川電機が1972年に商標として作った和製英語である。のちに機械・電子融合技術が一つの工学技術分野に発展したとき、同社は商標権を放棄してこの語を一般に開放した。いまでは、メカトロニクスは世界中で通用する言葉となっている。ロボット工学はメカトロニクスを基礎におく工学技術であって、その応用面は産業ロボットに始まり、医療・福祉・生活支援のほか、娯楽の分野にまで及ぶ。材料力学の関係では、戦時型船舶の脆性破壊に端を発して材料の破壊に関する研究がアメリカのアーウィン(Irwin, G.R.)らによって進められ、戦後になって破壊力学という学問が生まれた。これはさらに物性物理学と結びついて、材料強度学という学問へと展開した。b. 現在の先端機械工学(advanced mechanical engineering)現代の機械工学の最先端は時代の様相を強く反映している。第一はコンピュータ応用、第二は資源・エネルギー・環境対応、第三は健康・医療・福祉分野への展開である。ここではコンピュータ応用と在来工学分野の新しい展開を扱い、第二と第三はc項で述べる。コンピュータ利用はさまざまな現象、特に非線形現象の数値解析とシミュレーションによる模擬実験を可能としたほか、膨大なデータを扱う実験技術にも革命的な変化をもたらした。1956年ごろアメリカの航空機産業でFEM(finite element method、有限要素法)が開発され、これはただちに機械の構造・運動解析や熱・流体現象の解析に用いられるようになった。1965年のFFT(高速フーリエ変換)アルゴリズムの出現は機械振動など動的現象の解析には不可欠のもので、機械力学および計測・制御工学とその関連技術に大きな発展をもたらした。従来の機械工学では、研究を進めるための手段として理論と実験の二つの方法があった。コンピュータが発達したことで、いまでは数値計算に頼る「モデル化とシミュレーションによる模擬実験」という第三の方法が加わった。機械技術へのコンピュータ応用はさらに広範に及んでいる。機械要素の能動化と機械の知能化、機械の設計と製作段階におけるCAD/CAM(コンピュータ支援設計・加工)という手法をはじめ、FMS(flexible manufacturing system、多種少量生産システム)、FA(factory automation、工場の自動化)はコンピュータなしには成り立たない。機械技術の可能性を広げるコンピュータ利用と支援技術に関して、計算力学、コンピュータ機械工学などという新しい工学分野が生まれている。1980年ごろからはマイクロ加工の研究が進み、関連技術が開発されてマイクロマシンの製作が現実味を帯びてきた(図1・20)。1980年代中ごろの新素材開発は材料科学や材料工学の発展と表裏の関係にある。鉄道工学や自動車工学のような個別機種を扱う古くからの工学分野でも、新素材の出現や計測・制御技術の発展などで新しい内容が加えられた。新機種の開発で新分野も誕生した。宇宙工学はその一例で、航空工学から発展して分離・独立したものであ1―14機械工学総論る。これらと並んで、他の在来分野においても、技術のブレークスルーを目指して、超高速、超高圧、超高温、高放射能、あるいは極微、極地、深海など、これまでにない極限環境や極限状況に挑戦する新技術とこれを生み出す新しい工学がいま形を成しつつある。c. 技術を取巻く社会環境(social environment)の変化と機械工学1960年代の日本は経済優先、高度成長の時代だった。しかし高度成長に関わる技術の爆発的進歩の一方で公害が各地に発生した。1970年代に入ると技術先進国では大量生産・大量消費による資源・エネルギー・環境問題が顕在化して、野放図な技術発展に対する深刻な反省の声が世界中に高まった。このような社会情勢に呼応して、機械の分野では省資源と省エネルギー、それに環境問題の解決を目指す動きが活発になった。例えばエネルギーの高効率利用を図る研究が行われ、熱工学の内容はいっそう深まった。エネルギー変換工学、環境工学などはこの延長上に生まれた新しい工学分野である。1970年代は人間への関心がとみに高まった時代でもある。一方では遺伝子DNA の発見(1953年)と遺伝子組換え技術の開発(1973年)で生命科学と生命技術の発展があり、他方には先進国での少子・高齢社会の到来と高度技術社会における人々の精神的負担(ストレス)の増大があって、人間の心身の健康への関心がこれまでになく高まった。生体・医用工学(bioengineering,medicalengineering)や福祉工学(welfare engineering)などはこのような時代の要請に応える分野である。これらの学問が対象とする現場では、工学技術は人間個人の心の問題(好き嫌い、安心感、価値観など)や社会の仕組みとあり方(政治、経済、法律など)が不可避なり大づかみなものではあるが、まとめて表1・2に掲げておく。図1・20
