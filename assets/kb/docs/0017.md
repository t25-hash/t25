# 1・2・13 まとめ、歴史に見る機械と機械工学の変遷(表1・1)

> **章**: 第1章 機械工学概説
> **節**: 1・12 )。滑り軸受の潤滑理論を打立てたことも彼の大きな功績であ

機械工学は機械を設計し、作り、動かすことに関わる学問と知識の体系である。機械と機械工学が歴史の中で時代の状況と要請のもとでどのように生まれ、時代の移り変わりとともにいかなる特徴をもって変遷してきたかを以下に要約する。(1) 機械とは、昔は何よりも「重い物を動かすもの」のことだった。材料と構造の強さが技術者最大の関心事であり、ここで材料と静力学の知識が蓄えられた。(2) 近代になると機械には大きな動力(力と速度の相乗積)が求められるようになり、速度が重要な役割を果たすものとなった。特に鉄道と近代造船は学問としての機械工学の形成と発展を促した。動力学と機械力学(機構学を含む)、材料力学、それにエネルギーを扱う流体工学と熱工学がここで生まれた。(3) 20世紀に入ると機械の種類は増え、機械は巨大かつ複雑なものとなった。このような機械の機能や性能を安定で確実なものとするために、20世紀の中ごろにはメカトロニクスの支援のもとに計測・制御工学が生まれた。最近では機械の設計、製作、運転等のあらゆる面でコンピュータの利用がふつうになり、コンピュータ機械工学が編まれ、計算力学も体系化された。(4) 20世紀の末期には人間の健康・福祉と地球環境の保全に人々の関心が集まり、生体・福祉工学のほか安全工学や環境工学も現れた。このような人文・社会現象に関わる技術を扱う工学(文理融合工学)がいま生まれつつある。(5) 機械の材料と工作に関する知識と学問は、どの時代も変わることなく、つねに発展して機械工学に新風を送っている。機械技術と機械工学の誕生と発展の姿を見るには、歴史上の各時代に何が技術を推進する駆動源となったか(技術は主として何に奉仕したか)、時代の先端技術領域は何であったか(これは時代が求めたものでもある)をあわせ眺めることがたいせつである。こうすることで過去だけでなく現代という時代の特徴をつかむことができ、機械技術と機械工学が今後進んでいく方向が展望できるようになるだろう。最後に、各時代における技術の駆動力、先端技術領域と代表的な機械、このときに生まれ発展した機械工学の分野を、か表1・1機械技術史・機械工学史年表(その1)西暦機械技術・機械工学関連社会・一般歴史1637
