# 1・3・1 項では,新聞やテレビでは類義語として扱われる,多くの

> **章**: 第1章 機械工学概説
> **節**: 1・3 技術と工学

似たような言葉の意味,すなわち技術(technology),工学(engi-neering),設計(design),創造(creation),ものづくり(manu-facturing)などの言葉の意味を考えることを通して,本書で解説する機械工学はどのような学問か,どのように学べばよいか,を考えてみる.内容は多分に主観的であるが,結果として特に読者に伝えたいメッセージは簡単で,工学を学んで創造しよう,ということに尽きる.しかし,1・3・2項で詳説するが,現在の日本では,創造こそが21世紀の成熟経済における二段ロケット的な再加速に必要不可欠である,と国民に理解されている.それにもかかわらず,工学を学んでも創造にまで達しないエンジニアが実に多い.これは,これまでの日本の工業の発展過程や大学の教育方法が影響しているためかもしれない.日本では,欧米に比べても真理をそれなりに発見して特許を得ているのだが,それを組合わせて社会の要求に応えるものを試作し,自分が起業して新産業を立上げる,という実績が欧米に比較して少ない.これは,どうやって創造するのかという方法論の知識が足りないためであろうか.それとも創造を目指す気概や,創造を評価する駆動力が足りないためであろうか.今後の機械工学は,従来の力学の知識と他の分野の知識とを融合する方向に,期待される知識分野が広がるであろう.そして,新たなものを創造するプロデューサ向けの学問としても発展し,その結果,上述の創造を支援する方法論や気概,駆動力のような,人材開発・組織経営・社会構築の人文科学的な分野に進展していくであろう.
