# 1・3・3 これからの機械工学が進む方向

> **章**: 第1章 機械工学概説
> **節**: 1・3 技術と工学

日本では1965年ごろまでは、大学の組織と産業の分野とが完全に1対1に対応していた。つまり大学の組織図を見れば、産業の縮図が読み取れたのである。しかし、上述したように、民間では、機械工業に電気や情報の産業が融合して新たに強い組織が生まれてきたのに、大学や学会では、機械工学と電気工学、情報工学、生命工学、医療福祉工学などとが互いにマージすることはなかった。例えば、マイクロマシンはマシンが付くから機械学会だろうと思うと大間違いで、これは電気学会に論文を提出しなければならない。ますます、機械工学の柱である力学の知識体系だけでモデル化できる新製品が減り、その分、力学と何か他の知識とを組合わせて創出するビジネスが増えてくるだろう。例えば、力学を再生医学で作った細胞の運動の記述に使い、障害者の機能再生に役立てることが挙げられる。しかし、その要求に対して、自分は高校で生物学を勉強していないと言ったり、生体物は本質的に金属とは異なるまたはバイオの研究者とはそもそも文化が違うと言って機械工学者が逃げていては、その社会的使命を終えたと思われてもしかたがない。幸いにして、若い学生やエンジニアは、戦後50年間の古い世代が背負ってきた会社人間思考から脱皮し、自分の生き方を世間の評価を気にせずに打立てられるようになってきた。もちろん父親の会社人間世代がリストラで早期退職していく時代背景を考えると、まずは「自分流」で生きたいと願うことも理解できる。しかし、新産業創出に挑戦する気概はこれまでの人材にない性格である。後は、それに投資するという民間の金持ちのエクイティマインドだけが必要であるが、それに至るには自分の意志というよりも時代の勢いで投機に失敗した、というバブル期の痛手がいえるまで時間がかかるだろう。当分は、引続き、国が援助していく必要があるかもしれない。このように、機械工学のまわりには融合化を期待するビジネスが生まれ、機械工学は他の工学文化との融合だけでなく、医学や理学、生物学、経済学などとの融合が必要となるだろう。また、融合だけでなく、人間の思考にまで新たに手を伸ばして、設計の方法論や知識の活用論にも力を入れ、多くのプロデューサを育てる必要があるだろう。このような方向に向かってエンジニアが気概をもって挑戦すれば、日本の製造文化がひと皮むけて活性化し、その結果、高コストで過密なうえに原料、食料をもたない日本でも、世界と共生しながら繁栄を続けることができると考える。文献(1) 三輪修三、機械工学史、(2000)、丸善。(2) カーヴィル、J.(三輪修三訳)、工学を創った天才たち、(1986)、工業調査会。(3) 伊東俊太郎・ほか3名編、科学史技術史事典、(1983)、弘文堂。(4) 陸敬厳・華覚明(主編)、中国科学技術史機械巻、(2000)、305、北京・科学出版社。(6) Eckholdt, C., Kraftmaschinen I, (1983)、 34, DeutschesMuseum.(7) ティモシェンコ、S.P.(川口晶宏訳)、材料力学史、(1974)、鹿島出版会。1―22機械工学総論(8) 菊川真・北川浩、材料力学の発展、技術のこころ(一)、(1984)、173、丸善。(11) 三輪修三、日本舶用機関学会誌、25-9(1990)、567.(12) ラウス、H.・インス、S.(高橋裕・鈴木高明訳)、水理学史、(1980)、鹿島出版会。(13) 木村雄吉、流体力学の発展、技術のこころ(一)、(1984)、200、丸善。(14) レイノルズ、 T.S.
