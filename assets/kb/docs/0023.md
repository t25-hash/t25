# 2・1・2 道具の使用と火の利用

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

私たち人類の特徴は，(１)霊長類ヒト科の動物で，(２)自立姿勢
により二足歩行し，(３)文化をもつ，ということである．ここでい
う文化とは，社会活動を通じて学習し伝えられる行動や生活様式の
ことであり，道具を使い作ること，火を利用すること，狩りや農耕
作業を行うこと，言語をもちそれを使うことなどの痕跡や現状はす
べて文化に含まれる．
地球上に人類の祖先である猿人（アウストラロピテクス）が誕生
したのはいまから約200万年前といわれるが，1994年にエチオピ
アのアラミス地域で約440万年前の化石（ラミダス猿人）が発見さ
れその説が更新された．彼らはそれまでの樹上生活から気候など
環境の変化により地上生活に移行したともいわれ，前足と後ろ足を
巧みに使って移動し食物を採る生活から，後ろ足だけで地上を直立
歩行して生きなければならない生活へと変わった．そして体の骨格
や他の部位にも，直立二足歩行に適するような対応が見られるよう
になる．前足は食物を採り，ものをつかむ機能に特化し，後ろ足に
比べてはるかに高い能力をもつ手へと発達した．直立姿勢になった
ことで目の位置が上がり，そこから得られた情報（生きていくため
に必要なさまざまな情報）と手足を巧みに使う行動との組合せが，
彼らに続く後の人類に与えた影響ははかりしれない．目でものを
見，手（指と手のひら）でものをつかむ行動と感覚は別な情報とな
って脳を刺激し，もののもつ属性として記憶される．こうしたたび
重なる刺激が脳の容積増大を促したと考えられる．
ところで地上は人類の祖先だけが住んでいたのではなく，先住の
動物が生きる場でもあった．それゆえ人類の祖先は後から割り込ん
だ形となり，先住の動物と干渉し相互の生存を争うことになる．多
くの犠牲を払いながらそれが人類の祖先の教訓として，さらに経験
として蓄積され，敵から身を守る術をおのずと学びこれを生命の維
持に結び付けたのであろう．地面に落ちていた身近な木の枝を手に
持ち，向かってくる動物に一撃を加え倒すという簡単な防御法か
ら，木の枝が武器になることを知ったのだろう．この木の枝こそ，
初期段階としての道具の役割を十分に担ったといえよう．こうした
さまざまな経験を積み重ねて，外敵から身を守ったり食料を獲得す
るために，自らの生存に不可欠な道具を所有するという段階に進
む．同じような使い方をしても折れやすい枝と折れにくい枝があ
り，使いやすい枝と使いにくい枝があるなど，枝がもつ情報や機能
を識別し目的に合わせて選別する能力が養成されていったと考えら
れる．
枝ばかりでなく，地面に転がっている石も重要な道具であった．
石が何かにぶつかれば割れて破壊面に鋭い端面が生じるさまを知

がったことは，旧石器文化圏の形成としてよく知られている．中国
