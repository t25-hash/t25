# 2・1・2 道具の使用と火の利用

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

私たち人類の特徴は、(1)霊長類ヒト科の動物で、(2)自立姿勢により二足歩行し、(3)文化をもつ、ということである。ここでいう文化とは、社会活動を通じて学習し伝えられる行動や生活様式のことであり、道具を使い作ること、火を利用すること、狩りや農耕作業を行うこと、言語をもちそれを使うことなどの痕跡や現状はすべて文化に含まれる。地球上に人類の祖先である猿人(アウストラロピテクス)が誕生したのはいまから約200万年前といわれるが、1994年にエチオピアのアラミス地域で約440万年前の化石(ラミダス猿人)が発見されその説が更新された。彼らはそれまでの樹上生活から気候など環境の変化により地上生活に移行したともいわれ、前足と後ろ足を巧みに使って移動し食物を採る生活から、後ろ足だけで地上を直立歩行して生きなければならない生活へと変わった。そして体の骨格や他の部位にも、直立二足歩行に適するような対応が見られるようになる。前足は食物を採り、ものをつかむ機能に特化し、後ろ足に比べてはるかに高い能力をもつ手へと発達した。直立姿勢になったことで目の位置が上がり、そこから得られた情報(生きていくために必要なさまざまな情報)と手足を巧みに使う行動との組合せが、1―23彼らに続く後の人類に与えた影響ははかりしれない。目でものを見、手(指と手のひら)でものをつかむ行動と感覚は別な情報となって脳を刺激し、もののもつ属性として記憶される。こうしたたび重なる刺激が脳の容積増大を促したと考えられる。ところで地上は人類の祖先だけが住んでいたのではなく、先住の動物が生きる場でもあった。それゆえ人類の祖先は後から割り込んだ形となり、先住の動物と干渉し相互の生存を争うことになる。多くの犠牲を払いながらそれが人類の祖先の教訓として、さらに経験として蓄積され、敵から身を守る術をおのずと学びこれを生命の維持に結び付けたのであろう。地面に落ちていた身近な木の枝を手に持ち、向かってくる動物に一撃を加え倒すという簡単な防御法から、木の枝が武器になることを知ったのだろう。この木の枝こそ、初期段階としての道具の役割を十分に担ったといえよう。こうしたさまざまな経験を積み重ねて、外敵から身を守ったり食料を獲得するために、自らの生存に不可欠な道具を所有するという段階に進む。同じような使い方をしても折れやすい枝と折れにくい枝があり、使いやすい枝と使いにくい枝があるなど、枝がもつ情報や機能を識別し目的に合わせて選別する能力が養成されていったと考えられる。枝ばかりでなく、地面に転がっている石も重要な道具であった。石が何かにぶつかれば割れて破壊面に鋭い端面が生じるさまを知り、これが切れ刃として使えることから打製石器としての用途が広がったことは、旧石器文化圏の形成としてよく知られている。中国北京市の南西部にある周口店の洞窟で発見された北京原人(Pek-ing man:ホモ-エレクトゥス-ペキネンシスまたはシナントロプス-ペキネンシス、Homo-erectus pekinensis,Sinanthropus pe-kinensis)は打製石器を使っていたが、その石器は彼らが生活する場所ではなくほかから持ち込んだものであり、石の材質を選別する能力があったことがわかる。また日本では、長野県の美ヶ原に近い和田峠付近が、へき開により鋭い端面をもち刃物として使える黒曜石の産出地として知られているが、この黒曜石が峠を越えて遠く関東地方にまで運ばれ使われていたことはあまり知られていない。これは黒曜石の物流ルートが古代から形成され、同時に道が開かれていたことを裏付けている。一方、繰返しの使用により打製石器は表面が摩耗するとともに滑らかになり、結果として端面が長もちすることも経験的にわかってきた。この現象を逆に応用し、石の端面を磨くという工夫がなされたのではなかろうか。エジプトでピラミッドを建造する際に、ナイル川の水面を基準にして石材の表面を磨いたことも知られている。現在、各地の博物館や資料館には旧石器、新石器文化を象徴する打製、磨製石器がたくさん展示されているが、それらの大きさに注目すると、人が手で握ったときちょうどよい大きさと厚さになっていることに気づく。これは驚くべきことである。当時の人々は道具として使う石器の大きさと重さに最適化の考え方をすでに適用していたのである。石を打ち割り鋭い端面を得るという手法は、現代でも身近なところで使われている。私たちはカッタの刃が切れなくなるとそれを折って新しい刃を作り出すが、これなどは打製石器の製作法と原理的になんら変わるところはない。初めて火を使った人類は北京原人で、いまから約50万年前のことであり、彼らにはことばを話す能力もあったといわれる。先に述べた周口店の遺跡からは、火を使った痕跡が見つかっている。火を使うメリットは、(1)寒い時期でも暖が取れ、照明にもなる、(2)外敵から身を守ることができる、(3)調理に使えば食物中の細菌を殺し、味もよくなる、などであろうか。これらは人類の生存への可能性につながる重要なことがらである。落雷や山火事などによる自然火の採取から、ものとものとを叩いたり擦ったりしながら、摩擦により自らが火を得るというすばらしい知恵ももっていた。火そ1―24れ自体は現在においても原理的になんら変わるところはないが、先の用途に加えて多方面にわたる応用がなされ、私たちの生活にとって重要で不可欠な要素となっている。しかし、北京原人が火を使い始めたときから、人的な大気汚染が始まったことも忘れてはならない。文化の進展と自然環境保全は互いに背反するのである。人類の祖先たちによる道具の使用と火の利用はともに、現代に生きる私たちにとって最も重要で身近なできごとといえよう。
