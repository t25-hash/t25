# 2・1・3 五つの単一機械

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

機械はたくさんの機能をもつ要素の集合体であり、その要素は単純な機能をもつ、五つの単一機械に集約される。五つの単一機械とは、(1)てこ(lever)、(2)滑車(block,pulley block,tackle)、(3)輪軸(wheel set)、(4)斜面(くさび、slope,in-clined plane,wedge)、(5)ねじ(screw)、のことである。1世紀ごろエジプトの都市アレクサンドリアで活躍したヘロン(Heron)による力学の論文には、これらの単一機械を使うことにより力の大きさや向きを変えられることが記されている。エジプトに見られるような大型石造物の建造では、そりに載せられた巨石を運ぶため、数多くの奴隷が使われた。このとき、木の枝(丸太)などをてことし、それを地面に置いてころとして使う様子も壁画には描かれている。オリーブ油などが、そりところの間の潤滑剤として使われた痕跡もあるといわれる。こうした重量物の運搬法はかつて日本の山林でも、伐採した原木をふもとまで運び下ろすために盛んに使われた。敷き詰められた丸太で構成される木道で直接原木を下ろしたり(図2・1)、案内軌条に従って木馬と呼ばれるそりに載せられた太い原木の上に乗った作業者が、ブレーキを巧みにかけながらふもとの貯木場まで運び下ろしていた(図2・2)。また、急な斜面を使ったインクライン式木馬道も設置され(図2・3)、原木を搭載した木馬がロープと滑車により重力方式で上り下りしていた。現在の重力式ケーブルカーと原理は全く同じである。中世ドイツの炭鉱では、石炭を運ぶ箱車の先に案内溝とはまりあう棒が取付けられ、この棒と木板の上を転がる車輪とが組合わされていた。また木製レールとフランジ付き車輪による箱車も使われるなど(図2・4)、今日のレールを使った鉄道の原点に相当する輸送システムがすでに確立していた。ドイツ、ミュンヘンのドイツ博物館にこの展示品がある。地面に置いたころをそりなどの運搬具に直接取付ける工夫もなされ、これが軸と車輪の組合せへと発展した。この場合重要なことは、軸をどのように運搬具に固定するかということである。軸をじかに運搬具に固定すれば、軸径よりも大きな穴をもつ車輪を軸の周りに自由に回転させ、しかも抜け落ちないような工夫(輪止め)が必要になる。一方、車輪と軸を一体としてはめ込み固定すれば、軸を支える部品が必要になる。これが軸受の考案につながった。初期の軸受は滑り軸受であり、接触面での摩擦を減少させるため潤滑剤が使われるようにもなった。もっとも巨石運搬に使われたころ自体が回転するのであれば、転動体を使った軸受の登場も早い時期からといわねばならない。車輪の構造も、丸太を切断したような一体型から、板材を互いに組合わせた組立型へと移行し、さらにスポークの考案により軽量で強い構造の組立型車輪が誕生した。車輪をもつ運搬具も、当初の用途は死体を運ぶためだったといわれている。車輪の付いた運搬具を人が押したり引いたりする場合はかじ取りを自分で直接行えるが、家畜にけん引させる場合は人が間接的なかじ取りをしなければならない。運搬具を安定させ、積載量を大きくするという二つの目的から軸が二つになり、構造上の必要性から前軸にかじ取り機能をもつ装置が考案された。現代の自動車のかじ取り装置のスタートはこういったところに見いだせるのである。滑車はてこの応用であり、回転運動を利用してものを高い所に持ち上げるために考案されたといえる。井戸水を地上までくみ上げる図2・1 pump)、らせんポンプ]である。筒の中にらせん状の斜面があり、この筒を回転させると斜面に沿って下から上に水がくみ上げられるというもので、日本でも富山県を中心に東日本一帯に普及し、かんがい用や排水用さらに脱穀機の動力として使われた。現在でも非常用に設置されるらせん階段やプラスチックの射出成形機のスクリュー押出機はいずれもねじの応用であり、木工作業に使われる木ねじはねじとくさびを組合わせた巧みな構造になっていることに注目したい。このほか、かんがい用としてリンク機構(関節)を組合わせた龍骨車(ryukotsu-sha)もあり、ねじポンプ同様古くから利用されていた(図2・6)。1―25図2・7
