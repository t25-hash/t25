# 2・1・4 中国の三大発明

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

古代中国は、現在私たちが恩恵を浴している技術の発祥地であった。その典型的な事例は、a.火薬、b.磁石、c.紙、の三つの発明である。いずれも重要な事項で一般にはヨーロッパのルネサンス期三大発明といわれるが、これらのルーツはすべて中国である。a. 火薬火薬は黒色火薬のことで、木炭の粉末(炭素)と硝石を混合して作られる。中国では850年ごろ(唐代中期)に火薬を使った簡単な火矢が考案され、950年ごろ(五代後期から北宋初期)には本格的に実用化された。火矢とは現在のロケットの原点に相当し、底を粘土で封じて火薬を詰めた紙筒を矢先付近に取付け、導火線に着火し弓で敵に放つ火器のことである(図2・7)。日本人と火矢との初対面は元寇のときとされるが、日本へは直接中国から伝わったようで、以後文化面での花火として発達した。火薬はイスラム圏を経てヨーロッパに到達し、14世紀のドイツでは筒の中に込めた弾丸を火薬の爆発力で発射する火砲として発達した。これが、それまで戦闘の中心的存在だった騎士階級の没落を早めたといわれる。日本へは1543年、種子島に流れ着いたポルトガル人が鉄砲としてもたらしたことはよく知られている。このとき、銃尾にあったねじ部は日本人が鉄製ねじと出会った最初ともいわれ、おねじとめねじの作り方には諸説があり非常に興味深い。1―26機械工学総論b. 磁石磁石は古代から中国で指南車や指南魚、磁気さじなど、方位を知るための計器として実用化されていた。また指南車(shinan-sha,south-pointing carriage)には今日の差動歯車機構が用いられており、古代中国の技術レベルの高さをうかがい知ることができる(図2・8)。宋代には早くも航海用羅針盤として実用化された。火薬と同様、これもイスラム圏を経てヨーロッパへと伝わり、14世紀にイタリアで改良された。天文学と地理学の発達と合わせて、北ヨーロッパのバイキング船と地中海船の長所を組合わせた木造帆船に、改良された羅針盤を積み、地中海商人たちは遠く外洋航海に乗り出した。羅針盤と時間を知るための航海用時計(マリンクロノメータ)は、新大陸発見になくてはならない重要な計器であった。c. 紙文字は言語情報を後世に保存し、遠距離にも伝達できる有効な媒体である。しかし文字を記録する適当な手段がなかったため大衆には普及せず、僧侶や役人などもっぱら知的な仕事に携わる人々の間で使われた。中国の漢字の原点は紀元前1000年ごろの殷で使われた甲骨文字で、紀元ごろ現在の漢字になったようである。当時は木簡(木の板)や竹簡(竹の板)に漆で記されていたが、後に筆と墨(すすをにかわで固めたもの)と紙という好都合な記録システムが登場してから急激に発達した。紙は長らく、紀元105年に蔡倫により発明された(『後漢書』の記述による)とされていたが、紀元1~2世紀ごろの長安郊外の遺跡から麻を使った紙が出土したことから、蔡倫以前にも紙の実用化があったことが明らかになった。彼の発明では、木皮、麻くず、ぼろ布、漁網をアルカリ(木灰や石灰)で煮て繊維以外の不要部分を溶かし去り、繊維素だけで紙を作る方法を考案したことが記されているのであり、むしろ技術面での改良といえる。紙もイスラム圏を経てヨーロッパへと伝わったが、ブドウ搾り機を応用したグーテンベルク(Gutenberg)の活版印刷術と組合わされて宗教改革をなし得たことはよく知られている。こうした意味において、文化面、社会面で技術の果たした役割は実に大きいといわねばならない。
