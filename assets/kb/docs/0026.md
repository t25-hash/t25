# 2・1・5 自然力の利用

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

古くから人は人力や畜力のほか、風力や水力のような自然力を利用してきた。自然力を利用する場合には地域性や地理的条件がかなり左右するにもかかわらず、人はそうした自然の恵みである風や水の流れから巧みに力を取出す装置を考案し生活に応用してきたのである。これが風車(wind mill)と水車(waterwheel,water tur-bine)であり、回転型の石うす(stone mill)と組合わせて穀物(小麦)を粉砕する用途のほか、かんがい用、揚水用、加工作業用など実に多くの用途が見いだせる。風車の起源は明らかではないが、紀元前1700年にハムラビ王がメソポタミア高原のかんがい用に風車を使ったといわれる。これは、紀元前5000年ごろに小麦の栽培がこの地方で始まったことと無縁ではない。古代の風車は縦軸型風車であり、当初は主軸の真上または真下に石うすを置き、これを直接回転させる方式であったが、のちには歯車を利用した間接的な方式へと発展した(図2・9)。ヨーロッパでは12世紀になり風車が導入され始めるが、北ヨーロッパへは13世紀に、そして14世紀には広域的に普及した。ヨーロッパの風車は主軸が横軸で、風車をもつ大きな建物全体が風の吹く向きに回転できることが特徴で、これが技術面での大きな進歩といえる。これらは箱型風車(図2・10)や塔型風車(図2・11)と呼ばれているが、こうした大型の風車では、(1)風車が建物内の石うすとは垂直関係にあり、石うすを回転させるため軸を垂直に変換させる必要性から、歯車が用いられる、(2)この歯車は丸太の切断面に垂直に、かつ円周上等間隔に軸を植え込んだ簡単な構造のもの(ピン歯車)を互いに直角に組合わせて実用化され、今日のかさ歯車のルーツになった、(3)風車の羽根には一般には帆布が張られたが、薄い板を一定間隔で並べて窓のブラインドのように可変とし、風力の強さに応じて調節できる可変ピッチ構造のものもあった、(4)石うすの回転数が高くなりすぎないように調速機(gov-ernor)が取付けられていた、(5)風力を石うすの回転に使うだけでなく風車塔内にも伝達し、石うすに穀物を供給する容器を振動させる、製粉した穀物の袋を持ち上げたり、外から供給される穀物袋を塔の上まで持ち上げるリフトの駆動など、実に集約された工場機能をもっていた。ミル(石うす)ということばが工場の意味に使われるのは、こうした事例があるからである。このほか、南ヨーロッパの地中海地方にも風車が見られるが、建物が石造りでそれ自体は回転しない。これは海風の方向がつねに一定で、年間を通じて得られるからであろう。風車の最大の欠点は、(1)風が吹かなければ役に立たないこと、(2)風車で得られた動力は遠方に運ぶのが困難なこと、(3)材料や構造面で風車の出力に限界があること、などである。現在各地に建設されている大型風車は、立地条件の選定、機械エネルギーの電気エネルギー変換、複合材料をはじめ新材料の採用と構造面での改善などによりこうした問題点を克服しながら、地域的エネルギーの新しい担い手として期待されている。一方、水車は北ヨーロッパで使われた縦軸のノルウェー型水車(図2・12)が古く、これは家畜1頭程度の力しか得られなかった。おもに山間地の急流に設置され、タービン型水車のルーツといえる。こうした縦軸水車は現在でも中国西南部の雲南省とその周辺で、小水力利用のためにしばしば使われているが、水を受ける羽根は木材を削ってさじ状にし、水車主軸に穴をあけ斜めに差し込んだだけの簡易工法である(図2・13)。紀元前1世紀ごろ、ローマでは図2・9縦軸型風車(1600年ごろ)図2・10箱型風車(1630年ごろ)図2・11
