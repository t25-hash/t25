# 2・1・6 動力革命と産業革命

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

18世紀中ごろからイギリスで始まった産業革命には,(1)家内制手工業から工場制機械工業への転換,(2)水車動力から新しい蒸気機関への転換と併用,(3)木炭から石炭へのエネルギー革命と炭鉱の開発,(4)鉄鋼材料の大量生産方式確立,(5)工作機械の機能,構造改善と精度の向上,(6)河川・運河による内陸水上交通から小型高圧蒸気機関を搭載した蒸気機関車による鉄道への移行,(7)港湾整備と帆船,機帆船による海外航路確立,などといった特徴が見られる.繊維工業を手始めに機械工業で開花した産業革命はまた,社会制度や生産様式の変革,経済規模の変化なども引き起こしていった.近代化と呼ばれる一連のできごとは,この産業革命を経て実現されたともいえるのである.こうした中でも,蒸気機関(steam engine,beam engine)による動力革命は産業革命の中心的なことがらであった.1765年の春,グラスゴー大学構内で教材修理を担当していた機械職人ジェームズ・ワット(Watt, J.)は,トーマス・ニューコメン(Newcomen, T.)が発明した大気圧蒸気機関(atmosphericengine)の改良に成功し,産業革命の中心となる高効率蒸気機関への道を開いた.彼の改良は,シリンダ内の熱い水蒸気を冷えた真空の凝水器(コンデンサ)に送り込むことであり,実際にこの方式による蒸気機関が作られたのは1769年になってからである.その機関のシリンダ直径は18インチ(457mm),行程は5フィート(1524mm)であった.この蒸気機関は量産され,炭鉱での地下水くみ上げや工場用動力源として多用された(図2・18).しかしその大きさは巨大で,いったん設置すると移設は容易ではなかった.ところで,ワットの蒸気機関には新しい考案が盛り込まれてい図2・18 ワットの蒸気機関(1788年製)た.それは,揺動するビームとシリンダとの間にリンクによる擬似平行運動装置が組込まれていることである.これはすでにクランクを使った特許が取られているのを避けるための,彼の工夫であった.さらに遊星歯車装置も取付け,ピストン1往復に対し駆動軸は2回転するという効率の良いものであった.また調速機の改良にも取組み,遠心力を使って蒸気弁装置の開閉を自動的に行えるようにした.これは自動制御面での功績としてしばしば取上げられる話題だが,彼は風車に使われ実績のあった遠心式調速機を応用したのであり,発明者ではなかったことを知ってほしい.産業革命を推進したもう一つの立役者は,鉄鋼材料の大量生産であった.コート(Cort, H.)の発明による木炭を使うパドル法(かくはん法)に比べて,1855年ベッセマー(Bessemer, H.)が特許を取った転炉(converter)法は鉄鋼材料の大量生産を可能にした(図2・19).続いて1865年マルタン(Martine, P.E.)が発明した平炉(open-hearth furnace)法は,転炉法よりも製品が優れていた.こうして鉄鋼材料が工業化を推進し,それまで木製だった機械もだんだんと鉄鋼製に変わり,大馬力の蒸気機関で駆動され生産がいちだんと進んだ.一般に産業革命の担い手は石炭と鉄である図2・19転炉法による製鋼(1860年ごろ)図2・20 トレヴィシックの高圧型蒸気機関と蒸気機関車図2・21 スチーブンソンのロケット号蒸気機関車といわれるが,その集約された姿を蒸気機関に見ることができる.これを車台上に搭載するために小型化し,出力を向上させたものが蒸気車と蒸気機関車である.1804年,イギリスのコーンウォール出身のトレヴィシック(Trevithick, R.)は,高圧で軽量の蒸気機関の試作に成功した(図2・20).この成果を使い,南ウェールズの鉱山から運河までの線路上に初めて蒸気機関車(steam locomotive)を走らせた.その後1825年には,スチーブンソン(Stephenson, G.)がロコモーション号と名付けた蒸気機関車を使い,ストックトンアンドダーリントン鉄道で運転を始めた.しかしこれらは一般人が利用できる公共用鉄道ではなく,石炭運搬の産業用鉄道であった.1830年になり,スチーブンソンのロケット号(図2・21)がリバプールアンドマンチェスター鉄道で走り始めた.これが旅客と貨物を運ぶ世界初の公共用鉄道であった.この成功は蒸気機関車による鉄道が輸送革命をなした記念すべきことがらで,同年フランスで,1835年ドイツで,1837年オランダで,そして日本では1872年に公共用鉄道が開通し,人と物の輸送に大いに貢献した.鉄道の開通は国家の経済活性化もさることながら,有事の際の兵員と武器・弾薬,物資輸送にも重要な存在であったことはいうまでもない.これは日本でも同様で,民間資本で建設された幹線的大手私鉄を鉄道国有法により1906年買収した.このとき買収された私鉄が,東北本線など現在の幹線的鉄道の前身であり,線路の幅(ゲージ)はすべて1067mmに統一されていた.産業革命の進展は,国土の荒廃と環境悪化を引き起こしていた.イギリスでは木炭を作るため森林が激減し,煙突から排出されるスモッグで大都市はいつも暗く曇っていた.石炭の生産量増大は婦人や年少者までも労働に駆り立て,資本家の搾取により労働環境はいっそう悪化し大きな社会問題となった.この時期に活躍した人が,ロバート・オーウェン(Owen, R.)であった.自らが工場主であった彼は,労働者の待遇改善,工場法制定,労働組合結成などの社会改革運動の創始者としてよく知られている.彼の努力は1833年,工場法制定として結実した.このように産業革命は光ばかりではなく影の部分もあり,技術者の創意工夫と多くの現場労働者の犠牲のうえに近代化が達成されていったのである.
