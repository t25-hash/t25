# 2・1・7 加工法と大量生産

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

人類の祖先が石を叩いて打製石器を得たように、素材の一部を破壊・除去して必要なものを作り出す除去加工は人類と同じくらい長い歴史をもっている。さらに、あらかじめ用意された型の中に溶かした金属を流し込み、冷却後取出して型の複製品を作る鋳造(casting)法も、除去加工同様に歴史が古い。紀元前2000年ごろ、中国の殷では、すでに貨幣が鋳造法により作られていたことが知られている(図2・22)。高貴な人々が愛用したエジプトやメソポタミア出土の装飾品(図2・23)にもかなり精巧な細工が施されており、古代人の精密鋳造技術レベルの高さを知ることができる。貴重なものや祭祀用には玉や金属が用いられている反面、庶民が使うものは石や木や土で作られているという、階層による材料の使い分け1―29図2・22 古代中国で使われた貨幣鋳造用鋳型(中国、上海博物館展示)図2・23 メソポタミアの装飾品(紀元前1200年ごろ)/平山郁夫シルクロード美術館蔵図2・24 古代エジプトの壁画に残る穴あけ作業(紀元前2500年)がなされていた様子がうかがえる。これらの見事な品々には、美を意識したすばらしい形状と細工、模様が施されている。紀元前2500年ごろに描かれたエジプトの壁画には、穴あけ(drilling)作業の場面が残されている(図2・24)。この壁画には注目すべき箇所がある。それは、(1)職人が座って加工作業をしている、(2)職人は弓を前後に動かし、巻付けられた弦で軸に回転運動を与えている、(3)職人は軸受を介して軸を支えながら加工具を工作物に押付けている、という点である。加工具先端形状は不明りょうだが、軸が交互に回転することから、刃先が軸先端の両側に取付けられていた可能性もある。工具材料は石材なのか金属なのかが非常に興味深い。この弓式きりを横に寝かせて軸の両端を軸受で支えれば、横軸ろくろ(rokuro,lathe)となる。軸の一端に工作物を取付ける工夫をし、弓式きりの代わりに先手と呼ばれる人が回転運動を与えながら加工が行われた。日本でも、こけしを作る木地師がこうした横軸ろくろを使っていた(図2・25)。日本に横軸ろくろが入ってきたのは、仏教といっしょであろう。それは、朝鮮半島西側の百済にいた仏師たちが、仏具をひくろくろ師(木地師)を伴い新しい技術文化をもたらしたときであった。帰化人となった彼らは琵琶湖周辺に居住し、半島から伝来の技術を日本人に伝えた。中国西南部やチベット地域では、町の市場で家具職人がいまでもこの横軸ろくろを弓で駆動させながら部品加工を行っている。また古代には図2・26に示すような穴あけ機も使われ、これは現在のボール盤の原型といえる。1―30機械工学総論軸が縦方向の縦軸ろくろは、陶工用として縄文末期から弥生期にかけて米作技術といっしょにもたらされたといえる。陶土をろくろ成形して作ったかめや容器は、収穫された穀物の保存用として、また水の貯蔵など日常生活用品として必要だったからである。石材などの表面を平らに仕上げる場合、ある程度まで荒く平らに仕上げた後に硬い砂などを研磨剤として使い、長い時間をかけながら丹念に仕上げたことだろう。エジプトのピラミッドを構成する石は直方体状であるが、この石材表面はかなり平らに磨かれている。これはナイル川の川面を模範として仕上げたといわれている。古墳から発掘されるまが玉などもある程度までの大まかな形を作った後、仕上げは研磨(polishing)法で行われていた。特に玉に穴をあける場合は少しずつたたいて穴を広げるか、玉に穴をあけたい箇所に砂粒をまき、そこに鳥の硬骨を上から当てて回転させながら穴あけをし、最後の仕上げは砂粒を含んだ水中で麻縄などの植物繊維を往復させて穴を研磨したのだろう。現在のラッピングに相当する加工法である。古代の加工では、穴をあけること、平面を作り出すこと、丸いものを削り出すこと、の三つが基本で、これは今日の加工にもそのまま当てはまる基礎的なことがらである。同じものをたくさん生産するためには、それなりの創意工夫が必要である。日本で初めて大量生産されたものに百万塔(図2・27)図2・25 二人びき横軸ろくろとかんなと呼ばれる工具(宮城蔵王こけし館展示)がある。これは孝謙天皇(718~770年)が恵美押勝の乱を平定した後、罪障消滅を目的として『陀羅尼経』を入れる小さな塔を百万基作らせ、東大寺をはじめ十大寺に奉納した。ヒノキ、サクラを主材に相輪はモクセイやカツラを使い、横軸ろくろでひいて6年がかりで作られたが、塔の中は中空でここに縦6cm、横17~50cmの経典が巻いて納められている。この経典は大量印刷しなければならず、丸い軸木の周囲に経典の文字を彫り込んで、これを紙の上で転がして印刷した(今日の凸版印刷法)可能性もある。長い経典はこうした軸木を何本も用意して、次々に転がして印刷すればできあがる。現在の輪転機はこの方法をそのまま応用しただけであり、先人の技術文化のすばらしさに感動するばかりである。このほか、貨幣の鋳造でも大量生産が必要であった。このため、貨幣の形と模様を刻んだ高温に耐える鋳型に溶けた金属を流し込んで作られたが、あらかじめこうしたパターンをもつ原型を用意し、これを粘土に押付けて型を複製してから生産したのかもしれない。できあがった製品の観察もさることながら、製品にならなかった不良品の観察からは多くの有益な技術情報が得られるものである。日本で近年見つかった富本銭では、鋳造法で使用する鋳型と銭の不良品が出土し、これは古代生産技術の姿をいまに伝える重要な証となっている。イギリスでは、産業革命初期の加工現場で木製工作機械(おもに旋盤)が多く使われ、加工を担う工具に長い柄を付け、これを作業者が体で支えて削るという方法で作業が行われていた(図2・28)。形状・寸法精度を要求される工作物を加工するにはこうした構造の機械では無理で、どうしても鉄鋼材料を使った剛性の高い工作機械が必要であった。すでに述べたベッセマーやマルチンらによる鉄鋼材料の大量生産がこれを大幅に可能にし、工作機械の高剛性化・大型化が図られていった。さらに画期的なことは、加工時に工具を人が体で支えるのではなく、工作機械の送り装置と工具固定台(スライドレスト)とを連動させ、一定の速さで移動させることであった。これが自動送り装置付き刃物台を備えた工作機械(旋盤)で、1797年にヘンリー・モーズレイ(Maudslay, H.)が考案し、現在もロンドンの科学博物館に保存・展示されている。現在の工作機械ではしごくあたりまえの装置となっているが、当時としてはすばらしい発明であった。また彼の工場で働いていたジョゼフ・ウィットワース(Whitworth, J.)は1825年、定盤(surfaceplate)の平面を仕上げる画期的な方法を考案した。それは同時に三つの平面板を作り、これらの間に光明丹を塗って互いにすり合わせ、あたりをきさげで削り取ってすべての面を平面に仕上げるという方法であった。定盤は加工作業に不可欠な平面の基準であるから、彼のこの方法は現在でも三面すり合せ(fitting)法として加工現場で使われている。工作機械が鉄鋼材料で作られるようになっても、それらの多くは水車から得られた回転動力をベルトとプーリによる動力伝達装置から得ていた。水車が蒸気機関に代わってもこの動力伝達装置は多くの生産工場でそのまま残り、紡績工場も含めて20世紀まで継続的図2・26 古代の穴あけ機図2・27 百万塔図2・28 手持ち式刃物台と自動送り装置付き刃物台に使われていた。こうした動力伝達装置は産業革命を担った脇役であるが、日本では近年急激に数が減少し、博物館と地方鉄道の鉄道工場でわずかに残るだけである。工作物の大量生産になくてはならないものに測定機器がある。1765年、ワットは機械製造に使うため、マイクロメータを考案した。このマイクロメータをさらに改良したのが、ウィットワースであった。精密測定ができるようになることは、機械部品の互換性を前提とする生産が可能になることを意味している。現在のようなマイクロメータが実用化されたのは1868年のことで、ブラウン・シャープ社が発売したものである。材料の一部を除去する切削加工とは異なり、材料に外力を加えて変形させて加工する塑性加工も産業革命期には大いに発達した。なかでもジェームズ・ナスミス(Nasmyth, J.)が1839年に作ったスチームハンマ(steam hammer、図2・29)はその代表的な機械で、大量生産にも貢献した。1842年には特許を取得、1851年にロンドンで開催された第1回万国博覧会にも出品し好評を得ている。大量生産は新大陸アメリカではどうしても不可欠な方式であった。その理由は1789年のフランス革命後、ナポレオン軍がヨーロッパ各地に攻め入り戦争が始まったからである。その影響でアメリカでは産業や生活に必要な機械や物資が輸入されず、自国内で大量生産せざるを得なくなった。それと同時に、機械が壊れたときにはどこでも誰でもが部品を取り替えて修理し、再び使えることができるように互換性をもたせることが必要であった。これは特に軍用品には重要なことで、銃では絶対に不可欠な方式であった。大量生産方式の開拓者の一人に、エリー・ホイットニー(Whitney, E.)がいる。彼は綿繰機に大量生産方式を適用し、その成果を銃の大量生産に応用した。ここでたいせつなことは、生産にジグ(jig)を使ったことである。有名なサミュエル・コルト(Colt, S.)が発明した拳銃も、またシンガー(Singer, I.M.)のミシンも、そしてヘンリー・フォード(Ford,H.)の自動車も、ともに大量生産に互換性方式を兼ね備えていた。フォードはコンベヤシステムを使いT型フォードの大量生産を行ったことでよく知られているが、これは彼が町の肉屋で大きな肉の塊を数人が順に作業分担しながら解体する様子を見て、その作業を逆に行えば自動車の組立作業ができることに気がついたといわれている。問題解決策は案外身近にあるという一つの事例である。大量生産でもう一つたいせつなことは、作業能率である。アメリカの技術者フレデリック・ウィンスロー・テイラー(Taylor,F.W.)は作業者の作業時間を実測し、生産能率向上への研究(作図2・29ナスミスが作ったスチームハンマ1―31業研究、work study)を行い、科学的管理法を提唱した人として著名である。加工作業では作業者の行動観察もさることながら、加工を担う工具の形状と材質がその寿命を大きく左右することに気づいた。後にテイラー・ホワイト鋼と呼ばれる著名な高速度鋼(high-speed steel)は彼の研究成果として発見され、1900年にパリで開かれた万国博覧会にも出品されて切削速度をそれまでの125ft/min(38.1m/min)から145ft/min(44.2m/min)に向上させ軟鋼を切削してみせた。高速度鋼の実用化は高速回転型工作機械の設計を促し、機械性能の向上にも大きく貢献した。
