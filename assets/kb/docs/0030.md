# 2・1・9 終わりに

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

これまで,機械技術史上重要と思われる話題をいくつか取上げ,機械技術はどのような過程を経て今日に至ったかという課題を中心に,人と技術と社会の関わりや生活と文化の質的変容などにも触れながら概説してきた.従来の技術展開は,19世紀の経済学にそのよりどころを求め,資源とエネルギーは無限,技術に限界はなく,大量生産・大量消費は良い,などといった価値観に支えられている.地球資源の有限性や環境問題など,21世紀の機械工学が直面する課題はさらに複雑かつ困難が予想される.機械技術史はこうした課題に直面したとき,その解決策を示してくれる唯一の重要なデータベースであるといえよう.機械技術史についての各論は,2・2節から詳細に記されている.
