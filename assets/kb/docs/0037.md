# 3・1・1 機械工学(mechanical engineering)の位置付け

> **章**: 第3章 現代の機械工学の構成
> **節**: 3・1 機械工学の縦糸構成

人類は石器時代よりものづくりを始め、長い歴史の中で実にさまざまな道具を作ってきた。そして、機械文明ということばが示すように、人類は機械を作ることによって文明を発展させてきた。また、産業革命を契機としてエネルギーに関する科学技術を進展させてきた。さらに、第二の産業革命と呼ばれる情報化によって機械分野のみならず、社会全体の構造を変革させてきた。これらを文明の移り変わりとしてとらえると、狩猟採集文明から始まり、農耕牧畜文明へ、そして、化石燃料文明から情報創造文明へと変わってきたといってよいであろう。一方、学問体系は、法学、医学、工学、文学、理学、農学などに次第に分化し、この中で工学は、これまではものを中心とした価値を創造することに貢献してきた。今後は、地球資源の回復、多様性のある文明の構築などへの貢献が期待される。さらにこれまで実現してきた真理の探究と未踏分野開拓への挑戦、技術革新による社会や人の活動への支援、産業と文明の発展への貢献をよりいっそう進めていかねばならない。しかし、それだけではなく、われわれ機械技術者は科学技術が人類の環境と生存とに重要な影響を与えることを認識して、人類の安全、健康、福祉の向上・増進と環境の保全のために尽くす必要がある。工学は、おおまかには、土木・建築工学、機械工学、電気工学、化学工学、材料工学などの分野に分けられる。なかでも機械工学は、個別技術が重要であることはいうまでもないが、さらに、細分化された工学やそれを超えた全体を統括する役目も果たしている。この意味で機械工学は工学の中心に位置するといっても決して過言ではない。
