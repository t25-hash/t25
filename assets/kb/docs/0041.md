# 1・2・3 活性劣化（deactivation)

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

反応中に触媒活性点が変質することにより，活性が劣化すること
は触媒反応装置にとって最も悩ましい問題である．担体や助触媒の
使用，移動層・流動層などの反応器，モノリス型触媒（mono-
lithic catalyst ），ガードリアクタの利用など触媒反応プロセス工学
の工夫は触媒活性劣化との戦いの中から生まれてきた．その原因を
表1･2に示す．この対処法は触媒それ自身の改良，触媒の再生を組
込む反応操作法の工夫，劣化をもたらす物質を極小化するようなプ
ロセスシステムとしての工夫，など原因に応じた多種多様な方法が
ある．詳しくは成書 を参考にしてほしい．
