# 1・2・5 表面積，細孔構造（pore structure)

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

触媒作用は表面現象である．
したがって第一義的に活性は表面積で，その大きさは構成微粒子の径に強く依存する．
ちなみに通常の酸化物ではρ＝3×10gm程度であるnm（100オングストローム）程度ということになる．
このようなの一次微粒子間には空げきが存在する．
それが細孔といわれるものとその可燃性物質が除去され，細孔として残る．
このような手法は1･4(ｂ)(ｃ)］．
したがってゼオライトの結晶によりその細孔構造は度であり，触媒機能に大きな影響を与える．
例えばトルエンのメチを用いると，分子径の大きなオルトキシレンおよびメタキシレンは形状選択性という．
ゼオライトの細孔は特殊な結晶構造に固有のものである［図に比例する．
［ただし， ：一次粒子径(m)，ρ：密度( g･m)］ので，200m ･ gの表面積をもつ酸化物触媒の一次微粒子径は10一次微粒子が集合してわれわれの見ている粒子を形成しており，そである．
触媒調製時に可燃性物質を加えて成形し，その後焼成する細孔径制御法として広く使われるようになっている．
一意的に決定されている．
その細孔の大きさは分子の大きさと同程ル化によるキシレン合成のとき，適切な細孔径をもつZSM-5触媒生成せず，パラキシレンのみが選択的に生成する．
これを（分子）り，物質は拡散によって移動する．
