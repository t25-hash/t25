# 1・2・6 細孔構造と活性・選択性

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

細孔の中の流体は反応器内の流動状態とは無関係に静止してお
うに影響するかは，触媒反応工学の重要な研究課題である．
触媒粒子内で反応物は消費され，外部から拡散により補給され
る．反応速度が大きいとき，物質移動速度が遅いとき，あるいは粒
子が大きいとき，触媒粒子内の原料物質濃度は低くなり見掛けの反
応速度も低下する．すなわち触媒の有効度［触媒有効係数
（eﬀectiveness factor ）といい，ηで表す］は低下することとなる．
ηは式(1･33)で定義されるチーレ数（Thiele number ） の関数と
して，式(1･34)のように表される．

触媒有効係数：η＝11tanh3－13 (1･34)

は有効拡散係数（m ･ s ）である．またチーレ数の中に現れる3は
いろいろな形状の触媒粒子との比較を容易にするために導入された
球に対する形状係数である．ηと との関係を図1･5に示す．反
応を早くするためには，粒子径を小さくする，有効拡散係数を増す
ために細孔径を大きくする，触媒有効成分を担体の外表層に担持す
るなどの対策がとられるが，これらはすべて触媒有効係数を大きく
しようとの試みである．
触媒反応の選択性とはいくつかの反応の相対的な反応速度によっ
て決まる．したがって，選択性もまたチーレ数によって大きく影響
される．詳しくは成書 を参照してほしい．
