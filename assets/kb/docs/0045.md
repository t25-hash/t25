# 1・2・7 触媒反応装置（catalytic reactor ）

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

運常反応は表面で起こるため，その反応の起こる場での着目物質
の濃度や温度は流体本体のそれらとは異なる．したがって触媒反応
装置の設計に当たっては熱・物質移動の効果を正当に評価しなけれ
ばならない．反応装置内の触媒の形状，保持の仕方，熱の除去手
法，反応物や生成物の相によっていろいろな触媒反応装置が開発さ
れている．
多種多様な触媒反応装置は結局のところ触媒の機能をいかに生か
すか，化学的には解決がつかなかった触媒の欠点をいかにプロセス
の工夫で補うか，という努力の結果である．反応装置の形状，操作
法に関係する因子を挙げておく．
(１) 生産量：少量生産か，大量生産か．
(２) 反応熱：発熱反応か吸熱反応か，反応熱は大きいか小さい
か．
(３) 平衡論的制約：高温が有利か，低温が有利か，高圧が有利
か，低圧が有利か．
(４) 触媒反応速度論：反応速度，反応速度式（特に濃度依存性)．
(ｂ) 多段断熱式固定層
(ｃ) 自己熱交換式固定層
(ｄ) 外部熱交換
式固定層
(ｅ)流動層
(ｆ) 多段流動層
(ｇ)移動層
図1･6種々の触媒反応装置
(５)相：不均一系か均一系か，反応物は均相か多相か．
(６)触媒寿命：その時定数，再生の可能性．
(７)触媒への毒物質：どのような不純物が存在するか．
これらの要因を考慮して触媒反応装置の形状が決められる．触媒
反応装置の一例を図1･6に示す ．反応の特性，それに有効な触媒
の特性は個々のプロセスで異なっている．極言すれば指定された一
つの反応に一つのプロセスが必要であり，これが触媒反応設計の難
しさであり，また楽しさでもある．
