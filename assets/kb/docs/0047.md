# 4・1・1 社会と技術と文明の歴史

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・1 社会技術・安全技術

東アフリカで猿人アウストラロピテクスが二足歩行を始めたのは、いまからおよそ350万年前とされる。それによって、猿人たちは両手が自由に使えるとともに、重くなった頭脳を支えられるようになった。次いで人工の道具を作り出し、自然物を素材として、いろいろな人工物(artifacts)を創造し始めた。手と頭を使うことで生み出されたものづくりの技術が、ほかの動物とは異なる動物としての人間を誕生させたのである。世界の各地に発達した文明の歴史を研究する比較文明論では、人類革命-都市革命-精神革命-産業革命-科学革命-情報革命の順に、パラダイム(時代に共通な思考の枠組)が変化して社会の進歩が起こったとされている。そして現在は情報革命の時代であり、また生命革命の夜明けでもある。人工の道具の創造と活用によって、猿人から人類が誕生し人類革命期が始まる。それからは、日々の食料を確保するために、狩猟と採集の技術を工夫しながらの移動生活が何世代も続いた。そのうちに、家畜の飼育と植物の栽培の技術を獲得して定住生活ができるようになり、紀元前7000年ころからの農業革命期になる。牧畜と農耕の社会における食料生産の増加で、生活に余裕が生まれて余剰人口を養えるようになると、いろいろな人工物を作る技術をもつ専門職人が集まって生活する集落や都市が形成された。そして紀元前3500年ころから、チグリス川とユーフラテス川流域のメソポタミア、ナイル川流域のエジプト、インダス川流域のインド、黄河流域の中国で、河川の流れを制御する技術によるかんがい農業を基礎に、都市文明が開花し都市革命期を迎えた。次いで紀元前7~4世紀ころからの、ギリシャの思想と自然哲学、インドの哲学と仏教、イスラエルのユダヤ教とキリスト教、中国の儒教と諸子百家、そして紀元7世紀にはアラビアのイスラム教などと、思想と宗教の普及が始まる精神革命期になる。中世を経て13~15世紀には、イタリアを中心にルネサンスが芸術と思想の革新運動として起こり、全ヨーロッパに広がった。17世紀、ガリレイやニュートンらによる神の摂理の証明を目指した近代科学が芽生える。18世紀後半からのイギリスでの紡糸・紡績と動力の機械化に始まる産業革命期には、人工物の生産は、それまでの家族による家内制手工業から労働者による工場制機械工業に代わり、人々の生活が都市に集中するようになる。また、1789年に始まったフランス革命を契機に、人々の自由・平等・博愛の実現とともに、欲望の解放を目指す近代化の思想が広がり、20世紀の消費を美徳とみなす社会思想の背景が生まれた。産業革命ごろから世界人口の急増が始まり、それに応じて工場生産される人工物の種類と量の急速な増加が起こった。19世紀になると、自然の構造の解明を目的とした近代科学によって、本格的な科学革命期になる。同時に産業の面では機械、電気、材料、化学などについての経験的な技術を基に、ワット、エジソン、ベル、マルコーニ、ダイムラー、フォードなどの起業家による鉄鋼、鉄道、蒸気船、発電、照明、電話、無線通信、自動車など、人工物の発明と工場生産が始まる。20世紀には、力学、電磁気学、化学、生物学、量子力学などの近代科学を基盤にした技術によって、材料、薬品、工作機械、船舶、自動車、航空機、通信機、映像機器、家庭電器、原子力、半導体、人工衛星、情報機器、ロボットなど、新しい機能をもった人工物が次々に発明されて産業になり、社会に普及した。その結果、機械文明と呼ばれる社会が誕生し、全世界の人々に快適な未来生活を約束するかのように見えた。ところが20世紀後半のコンピュータと衛星通信を使う情報技術による情報革命期になると、60億人を超える世界人口の増加と大1―9 7量生産された人工物のはんらんから地球環境の汚染が問題化した。そして自然の生態系の崩壊が現実の問題になると同時に、地球上の鉱物資源と化石資源の枯渇が予測された。こうして有限な地球社会というコンセプト(概念)が生まれ、それまで機械文明の社会基盤を支えてきた人工物が、逆に人々の生活と安全をおびやかす危険な存在に変わったのである。そして21世紀の生命革命期を迎え、科学による生命現象の解明が始まった。
