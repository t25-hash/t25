# 4・1・2 技術と工学の社会的機能

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・1 社会技術・安全技術

現代の科学と技術と工学との間の関係と,それらが現代社会の中で果たしている役割と機能について述べる.人間の知的活動の中で最も基本的なものは,遺伝子として組込まれている先天的な生きるための知としての生活知である.この生活知を基盤にして,生活をさらに上手に遂行するスキル(技能)としての知である技術(technology)が生まれた.技術知には,ある特定の個人にだけ可能な,特殊で局所的な知的活動も含まれている.この生活知と技術知の本質は,人と社会に役立つというなんらかの目的の実現を遂行するための遂行知の体系なのである.それに対して,物理学,化学,生物学,地学など自然についての科学(science)は,宇宙に存在する物質とエネルギーと情報との間にある相互関係を,数学のような論理形式を使って一般・普遍的に説明するための知的活動である.したがって科学は,何かの実現を遂行するという目的は必要としないのである.そして,万物が宇宙に存在する理由までを説明するのが,哲学の一分野である形而上学である.つまり,科学と哲学の本質は宇宙に存在する自然物の間にある関係の説明知の体系なのである.機械,航空,宇宙,電気,電子,材料,化学などの領域別の工学(engineering)は,遂行知である技術と説明知である科学との間を結ぶ知的活動の体系である.すなわち一般に工学は,人と社会に役立つという目的を遂行するための特殊・局所的な技術知を,科学によって一般・普遍的になるように解析し,それらの間にある相互関係を説明するための知の体系であった.19世紀になって工業社会が形成され,人間の誕生以来蓄積されてきた多くのいろいろな技術を,社会的に制度化された大学において,多くの学生に伝承するという必要が生まれた.これらの技術を近代科学によって説明する知の体系化の試みが工学の起源であったのである.そして,世界で初めて総合大学に工学部が創設されたのは,日本の帝国大学工科大学で1886年のことである.それ以後,欧米でも総合大学に工学部が設置されるようになる.現代の工学は,科学から導かれる予測を基にして未知の人工物を創造するため,遂行知としての機能も果たすように変化しつつある.すなわち,技術と工学の社会的機能は人と社会に役立つという目的の実現を遂行するための総合(シンセシス)による知的活動である.これに対して,科学の社会的機能は,実現すべき目的は必要とせず,宇宙に起こる現象の間に存在する相互関係を説明するための解析(アナリシス)による知的活動なのである.そして説明知である科学は,専門家の集団である学会によっておもに新規性の観点から評価される.遂行知である技術は,それを使う人(ユーザ)である一般の人々によっておもに有用性の観点から評価され,社会的に選択されて淘汰される.遂行知の説明知である工学は学会およびユーザによって新規性および有用性の両面から評価されているのである(図4・1).人間の知的活動遂行知生活知技術工学説明知科学哲学図4・1 人間の知的活動の社会的機能1―9 8図4・2人工文物の蓄積と自然界の循環
