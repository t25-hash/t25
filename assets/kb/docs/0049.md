# 4・1・3 文化と文明と技術の関係

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・1 社会技術・安全技術

まず文明と文化の区別を明らかにしておく.文明(civiliza-tion)とは,人間を取り囲むいろいろな機器や構造物である装置群から構成される人工物に加えて,社会の制度や組織までを含めた人工文とで構成されるシステムとされている.人工物のシステムはおもに物質とエネルギーにより,人工文のシステムは情報によって,ともに遂行知としての技術を使って構築される.それに対して文化(culture)は,ある人間集団の成員によって後天的に学習・共有され,世代を通して継承される人間の行動様式と価値観,とされている.そのため,人工物と人工文すなわち人工文物の実現を目的とする技術は,形として見える文明の変革には深く関係するが,民族の習慣を中心に形成される形の見えない文化との関わりは少ないとされている.現代の文明社会は,技術によって実現された人工文物システムから,快適で便利という恩恵と同時に,緩やかに進行する地球環境の破壊という恐怖と危険がもたらされている.そこでいま,人間を含めたすべての生物の未来における生存権を認めたうえで,地球環境を持続可能にするために,できる限り自然の循環系に近づける人工文物システムを創成する技術が求められているのである(図4・2).
