# 1・4・1 合成法の分類

> **章**: 第1章 化学反応の基礎
> **節**: 1・4 薄膜・微粒子生成反応

無機系薄膜や微粒子の合成法は，表1･3に示した気相，液相，固
相の3種類に分類される．気相からの薄膜や微粒子など固体の合成
は，大きく分けると蒸発・凝縮を基礎とした物理的気相析出
（PVD:physical vapor deposition）法と化合物気体の反応を用い
た核発生・成長による化学的気相析出（CVD:chemical vapor
deposition）法に分けられる．液相での粒子や薄膜の合成法は，水
溶液法，水熱法，金属アルコキシドを用いたゾルゲル法などが代表
的な手法である．これらの手法は，高純度の製品が得られ，分子，
イオンレベルから核生成・成長させてナノレベルの大きさや厚さの
微粒子や薄膜を合成するのに適しているが，一般に大きなサイズの
ものを作るにはコストが高い．固相法は，熱分解や還元反応など高
温下での反応や焼結現象を利用する方法で，一般にバルク体を作る
のに適しているが，微粒子化するには粉砕等の操作が必要で，純度
が犠牲になるが，コスト的には安く製造が可能である．
