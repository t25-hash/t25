# 4・1・5 現代社会における安全性の問題

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・1 社会技術・安全技術

現代社会の中で稼働する機械システムが増えてくると、その運用の信頼性と安全性が社会問題になってくる。信頼性とは機械システムなど人工物の故障による機能停止に関わる概念(コンセプト)であり、安全性とは個人の生命と財産を危険から解放するコンセプトである。人工物システムが故障して機能が停止し、さらに事故にまで拡大すると、人々の生命や財産に関わる危険が発生する。このとき、システムの信頼性と人々の安全性とは同じ事象になるのである。そこで文明社会における安全技術の問題を生物系、物質系、文明系に分けて述べる。a. 生物系地球環境の安全問題人類革命期の生物系環境では、人々は野獣や病原菌という他の生物からの攻撃の危険にさらされてきた。ただし人間以外の生物どうしは、自然の食物連鎖の必要量以上には、殺し合いをしないのが自然界の摂理である。しかし人間どうしは、部族、民族、国家の間で戦争という無制限の殺りくを、人間だけが創造した技術で作った武器を使って今日まで続けている。1914年に始まる第一次世界大戦以後の兵器には、それまでの刀、弓、鉄砲、大砲、戦艦に加えて、強力な破壊力をもつ機関銃、戦車、航空機、航空母艦、潜水艦、原子爆弾、ロケット、ミサイルなどの機械化兵器が開発された。その究極は核分裂を使う原子爆弾による人類最終戦争の危険である。1―9 戦後の日本における家庭電化製品の普及率[資料:消費動向調査(経済企画庁)]ところで日本は、1543年ヨーロッパから種子島に火縄銃が持ち込まれてから32年後、織田信長が3000丁の銃を使った長篠の戦いのように、世界最大の鉄砲の生産と使用国になりながら、江戸時代初期から幕末に至る260年間の鎖国で、その使用を禁止した世界で唯一の特異な文化をもつ国であった。19世紀の本格的な科学革命期以後、500万種を超える自然および人工的に合成された物質が知られ、そのうち約6万5000種をいま産業界で使っている。その中には有毒、発がん、内分泌かく乱(環境ホルモン)を起こす物質が含まれているので、これらが大量に自然環境に拡散すると、大気と水質の汚染として社会を危険にさらすことになる。1953年ワトソンとクリックによって生物の遺伝子DNA の分子構造が明らかにされ、生命現象のメカニズム解明が進んできた。この遺伝子操作技術から生まれた遺伝子組換え植物やクローン動物などの人工生物は、その安全性が完全には確認されないまま、いま人々の食物になっている。その害はただちに発現するのではなく、長い潜伏期間の後に子孫に影響が現れる可能性をもっている。生物系地球環境の循環プロセスは、これまで主体である生物と自然環境との間の相互関係という生態系の原則に従うとされてきた。すなわち、環境との間に働く自然の相互作用力によって生物は遷移(succession)して変化し、その自己発展運動の結果、最終的には極相(climax)に達して繁栄するという自然の動的変化のプロセスであった。しかし最近は、生物と自然環境の間に大量の人工物と人工生物が介在する複雑な生物系地球環境に変わってきた。そのために新しい安全問題が発生しているのである。b. 物質系地球環境の安全問題人類革命期の物質系環境では、火山の噴火、地震、落雷、暴風雨、洪水など、自然災害だけが人々の安全をおびやかした。産業革命以後では、人工物の大量生産の結果として、鉱物と化石資源の採掘、工場用地、港湾、道路、ダム、大都市化などのために自然環境の破壊が進行した。また、化石資源の枯渇の予測、産業と都市と家庭から発生する廃棄物の処理法、廃棄物の焼却で発生する有毒ガスによる環境汚染、化石資源の燃焼で発生する炭酸ガスによる地球温暖化、工業用洗浄剤と冷凍機媒体のフルオロカーボン(いわゆるフロン)ガスによるオゾン層破壊などが社会の安全問題になっている。これらは、自然界に存在する物質群の中に大量の人工物質が介在し始め、複雑な物質系環境の新たな安全問題をもたらしているのである。それは、人間の欲望解放をもって近代化とする思想のもとで、全世界の人々が豊かな消費社会を目指していることの結果でもある。そのため、先進国の生活水準の引下げまで視野に入れる必要も出ている。しかし、火山の噴火、地震、台風、豪雨、豪雪など、大規模な自然現象の発生を制御する技術はまだ開発されていない。そのため高1―100機械工学総論度に集積化された現代社会を守るには、自然災害が発生した後の被害を最小にするための安全技術が求められる。c. 文明系地球環境の安全問題最近は人工物および社会の制度と組織まで含めた文明系環境における災害が多発するようになった。これまで病気の治療と予防のために、多くの医薬品や医療機器が開発され、人々の長寿化に役立てられてきた。しかし医療の分野では、薬物の副作用とともに、各種の診断と治療や手術のための医療機械、計測機器、人工臓器、臓器移植などの技術の安全性が社会問題になっている。さらに、少子高齢社会の家庭で、老人介護の技術が安全問題になっている。産業社会の現場では、労働災害の防止が古くから運動として実施され成果を上げてきた。最近の生産は、大規模機械システムとして生産の自動化が進み、作業者の安全は確保しやすい環境になっている。しかし頭脳労働者の過労死など、多忙と疎外による精神医療の問題が新たな安全問題として浮上してきた。公共社会における最大の安全問題は、交通事故と排ガス公害である。保有台数7000万台の日本では毎年90万件の自動車事故があり、そのうち24時間以内の死者はおよそ8000人、負傷者はおよそ120万人である。自動車は世界でいま約7億台が使われ、世界の自動車事故が減少する兆しはない。また、自動車の排気ガスは人々に直接の害を与えるとともに、酸性雨や地球温暖化を促す間接的な被害の原因にもなっている。さらに航空機、鉄道、原子力発電所など、大規模機械システムの事故も後を絶たない。これらの社会システムでは、従来からある機械の生産、稼働、廃棄における安全問題に加えて、機械システムの操作者と制御用コンピュータとの間のインタフェース、またコンピュータソフトのバグ、コンピュータウイルスによる誤作動、ハッカーによる意図的な妨害など、新たな安全問題が発生している。さらに公害規制、廃棄物処理、医薬品認可など、社会制度に関係する技術では、制度化の遅れや不備、また制度運用者の情報操作のような技術者の倫理に反する安全問題まで起こっている。
