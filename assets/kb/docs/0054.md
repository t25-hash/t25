# 4・2・1 はじめに

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・2 地球環境技術

ひとくちに地球環境問題(global environmental issues)といっても、その内容は地球温暖化、オゾン層破壊、酸性雨、熱帯林減少、砂漠化、海洋汚染、有害廃棄物の越境移動、野生生物種の減少、途上国の公害など多岐にわたっているが、最も注目を集めているのは地球温暖化問題である。この地球温暖化[より一般的には気候変動(climate change)]については不確定要素を含み、異論のあることも事実であるが、これに先立って指摘されたフロンガスによるオゾン層破壊の可能性への対応の遅れに対する反省もあり、科学的知見に対する完全な合意以前に対策の必要性が広く認識され、予防原則(precautionary principle)に基づいて行動が開始されるという状況を招いているのがこの問題の特徴である。また、大気中のCO(二酸化炭素、carbon dioxide)濃度が550ppm(産業革命前の濃度の2倍)という一つの安定化目標を達成したとしても、そのCO濃度そのものが海洋表層中の生態系に影響をもたらす懸念も指摘されるようになってきている。地球温暖化への寄与率が約60%にすぎないCOが特に重要視されるのは、大気中濃度の増加の主原因が、現代文明を支えている化石燃料消費の加速的増大にあり、その排出抑制には極めて大きな困難が予測されるためである。図4・5は、2001年に発行された気候変動に関する政府間パネル(IPCC: Intergovernmental Panel onClimate Change)の第三次評価報告書に示されている、過去1000年にわたる大気中CO濃度の変化と今後100年間の予測値である。図中に示されているように、予測値はシナリオ(世界の人口や経済発展、エネルギー需給構造、技術進歩などに依存)によって異なるが、いずれにおいても、2100年までの間には大気中CO濃度が産業革命以前の倍以上に増大することは避けられないとみなされている。このIPCCの第三次評価報告書第一作業部会報告「科学的根拠」の、「政策決定者向けの要約(SPM : Summary for Policyma-kers)」では、世界的な規模での温暖化および気候システムの変化についての全体像が明らかになってきており、最近50年間に観測された温暖化のほとんどは人間活動によるものであるという強力な証拠が得られており、また、21世紀を通して人間活動が大気組成を変化させ続けると見込まれるとの指摘がなされている。また、第三作業部会報告「緩和策」のSPM では、気候変動によってもたらされる問題がわれわれにとって無視し得なくなってしまう10年あるいは20年後ではなく、ただちに対策が必要であると述べている。この理由の一つは、図4・6に概念的に示されているように、たとえ今後100年程度の期間をかけて温室効果ガスの大幅な排出削減がなされたとしても、地球の気候システム等の時間遅れにより、大気中濃度の急激な増大による影響が数百年規模で続くためである。すなわち、この温室効果ガス(greenhouse effect gas)の大気中濃度の増加による気候変動問題は、その性質を考えても、短期的・中期的対策のみで解決できるものでなく、今後100年、あるいは200年を見通した長期的観点のもとに、しかも緊急に世界規模で取組む必要のある課題であることは明らかである。しかし、IPCCでも議論されているように、長期的に大気中のCO濃度を550ppmに安定化させようとした場合、2050年ごろには現在の全世界の排図4・5 過去および将来の大気中CO濃度図4・6 大気中CO濃度上昇による影響の時間遅れと持続性出量を1990年レベルの半分以下にする必要があることを考えても対策の困難さは明白である。ここでは、特に1997年にわが国で開催された気候変動に関する国際連合枠組条約(気候変動枠組条約、UNFCCC: UnitedNations Framework Convention on Climate Change)第3回締約国会議、いわゆるCOP3における京都議定書の成立に至る背景とその後の展開について概観するとともに、持続可能な将来へ向けた技術動向について解説する。
