# 4・2・2 京都議定書(Kyoto protocol)への道

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・2 地球環境技術

1988年は,気候変動問題にとって記憶に残る年であったといえよう.アメリカの穀倉地帯は異常気象による猛暑と干ばつに襲われていた.これとタイミングを合わせるかのように,1988年6月,アメリカ上院公聴会においてアメリカ航空宇宙局(NASA)のハンセン博士が,「現在,温室効果のために気候に変化が起き始めていることは99%確実である」との証言を行い,アメリカ社会にセンセーションを巻き起こし,地球温暖化という用語がマスコミにも取上げられ,世界的にも大きな注目を集めるようになった.同じく6月にトロントで開催された大気変動に関する国際会議(World Conference on the Changing Atmosphere: Implicationsfor Global Security)には,46箇国と国際機関から政治家,科学者等が集まり,温暖化が国際的に重要な政策課題として初めて議論された.この会議では,当面の目標としてCO排出量を2005年に1988年レベルの20%削減することが提案された(20%削減宣言).このように,気候変動問題が深刻な問題として認識され始め,11月には,気候変動問題に関する初めての国際的な検討の場として,世界気象機関(WMO)と国連環境計画(UNEP)が共同で,気候変動に関する最新の自然科学的および社会科学的知見をとりまとめ,気候変動防止政策に科学的な基礎を与えることを目的とした,気候変動に関する政府間パネル(IPCC)を設立した.このようななか,地球温暖化対策に関わる国際条約締結の必要性が高まり,1989年5月に開催されたUNEP管理理事会では,IPCCの報告が発行され次第,気候変動枠組条約の条約交渉を開始することが決定された.1990年8月,IPCCの第一次評価報告書がとりまとめられ,10~11月に第2回世界気候会議が137箇国の参加を得て開催され,立場の違いを越え協力して地球温暖化防止に取組んでいくべきことが合意された.これを出発点として,1991年2月から条約作りに向けての外交交渉(第1回条約交渉会議)が開始された.その後,計5回の条約交渉が行われ,1992年5月9日,温室効果ガスの大気中濃度の安定化の達成を究極の目的とし,地球温暖化を防止するための取組みの枠組を確立するための「気候変動に関する国際連合枠組条約(気候変動枠組条約):UNFCCC」が採択された.1992年6月に180箇国の代表を集めて開催された国連環境開発会議いわゆる地球サミットの場において,UNFCCCは計154箇国により署名が行われた.このUNFCCCは,拘束力のある温室効果ガス(GHG)の排出削減目標を定めるものではないが,気候系に危険な人為的影響を与えることとならない水準において,大気中の温室効果ガスの濃度を安定化させることを究極の目標とした.そして,全締約国共通の約束として,(1) 温室効果ガスの排出量・吸収量の目録(インベントリー)を作ること(2) 温暖化対策の国別計画を策定し実施することなどを定めた.さらに,先進国(附属書I締約国:Annex-I coun-tries)に対しては,途上国(non-Annex-I countries)が上の排出インベントリーを作成するための財政的・技術的支援の提供とともに,(1) CO等の温室効果ガスの人為的な排出量を1990年代の終わりまでに従前の水準に戻すことの有用性を認識して政策・措置を講じることおよび1―103(2) CO等の温室効果ガスの人為的な排出量を1990年の水準に戻すという目的をもって採用した政策・措置の詳細および将来の排出量・吸収量の予測等について締約国会議に通報することなどの追加的約束事項が定められた.UNFCCCは1994年3月21日に発効し,わが国を含む先進締約国は条約に従い,1994年9月までに,(1) 地球温暖化に関連する一般的状況(2) 温室効果ガスの排出および吸収の目録(基準年である1990年時点の値)(3) 地球温暖化対策に関わる政策および措置,およびその効果の予測などを内容とする国別報告書を提出した.しかし,先進締約国のコミットメントについては,当初より,(1) 温室効果ガス排出に関する目標が示唆的な努力目標でしかない(2) 取るべき政策・措置について具体的な言及がない(3) 国際約束の履行の適否の審査は形式的(条約の約束の履行を担保する措置が弱い)などの問題が指摘されていた.このような指摘を踏まえて1995年3~4月にベルリンで開催された気候変動枠組条約第1回締約国会合(COP1)では,2000年までにGHG 排出量を1990年レベルまで低減させるといった現行の約束の内容が不十分であるとの点で合意し,(1) 政策および措置の詳細を定めること(2) 先進国のコミットメントを強化し,例えば2005年,2010年,2020年といった特定の時間的な枠組における数量目的について国際的検討を行うことを決め,1997年のCOP3における新たな国際的約束のとりまとめに向けた検討を行うことが決定された.これがいわゆるベルリンマンデートと呼ばれるものである.また,発展途上国は,この新しいコミットメントから除外することも合意された.一方,1997年12月にはIPCCの第二次評価報告書が採択され,新たな施策の検討に向けての科学的基礎が提供された.報告書においては,大気中の温室効果ガス濃度は上昇を続けており,過去1世紀の間に気候は変化し,この傾向は将来も継続すると見込まれるとしているものの,依然として不確実性が残っていることも併記している.1996年7月にジュネーブで開催されたCOP2では,このIPCCの第二次評価報告書の内容を高く評価するとともに,現在では意外な印象を受けると思われるが,アメリカの強い主張により,ベルリンマンデートを達成するため温室効果ガスの全地球規模での削減に向けて,法的拘束力のある数量的な排出抑制・削減目標を含む議定書またはその他の法的文書を第3回締約国会議で採択すべきであるとされた.上記のような背景のもと,法的拘束力のある数量的な排出抑制・削減目標に関するさまざまな思惑が入り乱れた事前交渉を経て1997年12月に開催されたCOP3京都会議では,予定を延期して行われた議論の結果,先進国における温室効果ガスの排出削減目標などを定めた「京都議定書」が採択された.表4・1に京都議定書の主要点を示す.また,各国の数量目標値を表4・2に示す.採択された京都議定書では,先進各国の数値目標が決定されるとともに,その目標を経済効率良く達成するための手段として,排出権取引,先進国間での排出削減プロジェクトである共同実施(AIJ),先進国と途上国間での排出削減共同プロジェクトを推奨するクリーンディベロップメントメカニズム(CDM)など,いわゆるメカニズムの基本的考え方が示されている.しかし,これらメカニズムに関わる運用規則や排出削減の評価法や検証法といった詳細についての議論は,これ以降の締約国会議に持ち越された.しかし,京都議定書は,先進各国の数値目標を規定するとともに,途上1―104機械工学総論表4・1
