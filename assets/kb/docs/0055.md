# 1・4・5 固相合成反応

> **章**: 第1章 化学反応の基礎
> **節**: 1・4 薄膜・微粒子生成反応

固相合成法は，固体の熱分解，固相-気相反応，固相-固相反応な
どに分類される．比較的安価に粒子の合成が可能であるが，固相反
応では粗い粒子やバルク固体が得られるので，合成後，一般に粉砕
操作により微粒子を得る．近年，微粉砕技術の進歩は著しく，湿式
粉砕を用いればサブミクロン領域から数十nmの微粒子を比較的短
時間の粉砕で得ることが可能である．しかし，粉砕媒体や壁面から
の磨耗によるコンタミネーションが避けられないため，粒子と同じ
材質の粉砕媒体や容器を用いることが多い．

化学の基礎 第1章 化学反応の基礎
熱分解反応を利用した代表的な粒子製造法は，炭酸カルシウムの
熱分解によるCaOの生成，酸化鉄の還元反応などがある．固気反
応としては，金属の酸化，ハロゲン化，ポリマの気相合成など多岐
にわたり，固気接触効率を高めるため，流動層などが反応装置とし
て利用される．固相反応もさまざまな化合物に用いられているが，
一般に反応効率を高めるため，原料は微粉化して比表面積を上げ，
固相拡散等による物質移動を高め反応速度の向上をはかる．
