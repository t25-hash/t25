# 1・4・5 固相合成反応

> **章**: 第1章 化学反応の基礎
> **節**: 1・4 薄膜・微粒子生成反応

化学の基礎 第1章 化学反応の基礎熱分解によるCaOの生成，酸化鉄の還元反応などがある．
固気反にわたり，固気接触効率を高めるため，流動層などが反応装置とし一般に反応効率を高めるため，原料は微粉化して比表面積を上げ，どに分類される．
比較的安価に粒子の合成が可能であるが，固相反操作により微粒子を得る．
近年，微粉砕技術の進歩は著しく，湿式時間の粉砕で得ることが可能である．
しかし，粉砕媒体や壁面から材質の粉砕媒体や容器を用いることが多い．
熱分解反応を利用した代表的な粒子製造法は，炭酸カルシウムの応としては，金属の酸化，ハロゲン化，ポリマの気相合成など多岐て利用される．
固相反応もさまざまな化合物に用いられているが，固相拡散等による物質移動を高め反応速度の向上をはかる．
