# 4・2・3 京都議定書発効へ向けた交渉

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・2 地球環境技術

1998年11月にブエノスアイレスで開催されたCOP4では、今後のタイムフレームを伴う目標およびそのための具体的取組を規定する行動計画[いわゆるブエノスアイレス行動計画(BuenosAires Plan of Action)]が採択された。この中では、最大の交渉の焦点であったメカニズムについて、その原則、手続きなどにつき、COP6に最終決定を行うことを目的とした作業計画・日程が決定されたほか、資金メカニズム、技術移転などについても具体的吸収源の取扱い限定的な活動(1990年以降の新規の植林、再植林および森林減少)を対象とした温室効果ガス吸収量を加味。ただし、COP3で限定的に認められた他の吸収源の扱いについては議定書の第1回締約会議(COP 1)以降検討、決定することとする。附属書I締約国全体の対象ガスの人為的な総排出量を、目標期間中に基準年に比べ全体で少なくとも5%削減する。各附属書I締約国は、目標期間中の対象ガスの人為的な排出量が、個別の割当量を超過しないことを確保する。例えば、日本の割当量:基準年の94%(6%削減)アメリカの割当量:基準年の93%(7%削減)EU の割当量:基準年の92%(8%削減)バンキング目標期間中の割当量に比べて排出量が下回る場合には、その差は、次期以降の目標期間中の割当量に加えることができる。数量目標の達成の約束を共同で果たすことに合意した附属書I締約国は、これら諸国の総排出量が各締約国の割当量の合計量を上回らない場合には、その約束を果たしたとみなされる。すなわち、事前から問題視されていたEU バブルについては、削減目標について深掘りを求めるとともに、議定書上、責任関係の明確化、バブルの拡大時の扱い(EU が拡大した場合も拡大前の加盟国の目標に変更をもたらさないこと)について規定を設けることを条件にこれを認めた。附属書I締約国は、議定書の約束を達成するために、排出権取引に参加できる。条約の締約国会議は、排出権取引に関連する原則やルール、ガイドライン等を決定する。数量目標の達成を果たすため、すべての附属書I締約国は、他の附属書I締約国から、割当量を移転または獲得することができる。地球温暖化防止京都会議においては、排出権取引に関し、COP4において関連規則などの作成を行うことなどを決めた。数量目標を達成するため、附属書I締約国は、発生源による人為的排出を削減することあるいは吸収源による人為的除去を増進することを目的としたプロジェクトによる排出削減ユニットを他の附属書I締約国に移転し、または他の附属書I締約国から獲得することができる。附属書I締約国と非附属書I締約国との共同実施は、クリーン開発メカニズムのもとで行うことができる。クリーン開発メカニズムは、非附属書I締約国の持続可能な開発と気候変動枠組条約の目的達成を支援し、かつ附属書I締約国の数量目標の達成を支援するもの。本メカニズムにより、非附属書I締約国は排出削減につながるプロジェクト実施による利益が得られ、附属書I締約国はこうしたプロジェクトによって生じる承認された削減量を自国の数量目標の達成のために使用できる。作業計画が決定された。また、遵守問題については、COP6における決定を目的として検討作業を進めることが決定された。COP5は、1999年10~11月にかけてボンで開催され、京都議定書に関わる技術的問題に関して進展が見られ、議定書の遵守問題や、メカニズムの利用方法に関して政治的判断を下すに必要な材料が出そろったとみなされている。2000年11月にはハーグでCOP6が開催され、京都議定書に関わる懸案事項について決着が図られるものと期待されていたが、吸収源(Kyoto Forestとも称されるシンクの扱い)、途上国問題、京都メカニズム(Kyoto Mechanism)の上限値の設定、議定書の不遵守に対する法的拘束力の是非などについて、夜を徹しての交渉にもかかわらず、EU とアンプレラグループ(非EU の先進国による交渉グループ。アメリカ、日本、カナダ、オーストラリア、ニュージーランド、ロシアなど)の間で合意が得られず、COP6をいったん中断し、時を改めて議論を再開することとなった。このような状況下、2001年3月13日、アメリカのブッシュ大統領は、京都議定書を批准しないという選挙時の主張を再確認し、発展途上国が削減義務を負っていないことは不公平であり、またアメリカ経済への悪影響に対する懸念を理由として京都議定書交渉からの離脱を宣言した。アメリカ抜きでの京都議定書の発効の可能性を危惧する声が高まるなか、COP6再開会合は2001年7月にボンで開催され、京都議定書のいわゆる中核的要素に関する基本的合意(ボン合意)が得られ、京都議定書の2002年発効に向けて勢いが高まった。しかし、ボン合意の細則作りの協議においては、途上国問題についての合意が得られたものの、LULUCF(land use, land-use change andforestry)すなわち吸収源の扱い、遵守問題、京都メカニズム等に関しては継続協議となり、COP7での採択を目指すこととなった。COP7は2001年10~11月にモロッコのマラケシュで開催され、COP3から4年を経て、ようやくボン合意の内容に沿って、表4・3に示すように京都メカニズム、吸収源(LULUCF)、遵守制度、途表4・3 COP7における京都議定書の運用ルールに関する最終合意案の概要途上国問題・新たに三つの基金、技術移転のための専門家グループを設置することなどが決められた。京都メカニズム(Kyoto Mechanism)・削減目標を達成するために、各国内で実施する対策に対して補完的にのみ利用できる(定量的な制限は設けられていない)。森林吸収源(LULUCF)遵守制度(不遵守の場合の対応)上国問題を実施する仕組みや詳細な運用ルールの最終案が確定した(マラケシュ合意)。この合意の成立により京都議定書はアメリカ抜きの状況ではあるものの発効に向けて大きく前進した。その後、ロシアが当初の予想を覆して議定書批准に消極的な姿勢に転じたことによって再度発効が危ぶまれていたものの、2004年9月になってロシア議会による批准決定がなされ、議定書は2005年2月に発効する運びとなった。
