# 4・2・5 終わりに

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・2 地球環境技術

以上、地球環境問題とそれに関わる国際的経緯、および最近になって注目が高まっているCOの回収隔離技術について述べてきた。繰返しになるが、CO問題とエネルギー問題の共通性を考えると、再生可能エネルギー技術や省エネルギー技術など、従来から研究開発や導入が活発に行われてきたエネルギー技術は、その多くがCO対策としても有効であるものの、発展途上国を中心とする化石燃料の急速な消費増大や、地球温暖化が進行するといわれている数十年という期間内でのこれら新技術の世界的規模での導入にも限界があることなどを考慮すると、その間にも化石燃料から放出され続けるCOを大気から隔離するため、安定な形で長期的バッファに貯蔵する技術を選択肢として確立しておくことも緊急の課題である。しかし、問題の規模、すなわち対応すべきCOの量の膨大さを考えると、たとえ複数の対策技術を採ったとしても、それぞれの導入規模も大きくならざるを得ず、必然的に社会との摩擦を引き起こす可能性がある。例えば、再生可能エネルギーの中でも現状ではコスト面で最も期待されている風力発電については、すでに世界各地で地元住民やNGOなどとの間で論争を呼ぶようになってきている。また、規模の面では有力視されているCOの回収隔離については、さらに大きな論争を呼ぶことはほぼ確実である。地球環境問題そのものが必然的に先進国と発展途上国の間の不公平に関する問題を包含しているという性格があることと併せて、こういった技術の社会的側面をつねに念頭におくことが必須であるのが、この問題に対応する技術開発の大きな特徴であるといえる。
