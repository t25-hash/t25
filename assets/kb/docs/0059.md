# 4・3・1 はじめに

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

国内で進められている規制緩和を含めたグローバル化は,米ソ冷戦終結を機に急速に進んでいる.冷戦中は,資本主義圏と共産主義圏の二つのグループの戦いであり,商業分野でも各グループ内で相互に協力して政策立案,貿易を行っていた.冷戦終結後,環境は大きく変化したが,その変化の最も大きなものの一つにボーダレスエコノミーに変わったことが挙げられる.これはすべての国が貿易の競争相手国になったことを意味している.ボーダレスエコノミーの中で自由競争するためには,世界共通のルールを確立しておく必要がある.それらのルールが国際標準であり,その役割が非常に大きくなっている.ここでは,国際標準とは何か,その役割およびそれらに対する欧米の戦略を中心に述べ,その後に関係する技術倫理,製造物責任,知的所有権,最後に日本はどう対応すべきかを述べる.1―107
