# 1・2・4 物理量の数値の換算

> **章**: 第1章 単位系と次元
> **節**: 1・3 国際単位系

関はもとより，日本工業規格 においても，SI専用化の傾向が顕
すべての物理量を国際単位系のみで表現する限り単位の換算
著である．一方，わが国の計量法の体系についても1992年の改正
により，全面的なSI化が図られた．それは原則として，法定計量
単位は，SIおよびSIと併用される単位とし，特定の分野に限り用
途を限定して（あるものについては時限的に）従来単位を認めたも
のであり，これにより，工学単位系，CGS単位系などの単位のほ
とんどすべてが排除された．
