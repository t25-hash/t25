# 4・3・2 国際標準とは

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

標準(標準は規格ともいわれるが,ここでは標準として用語を統一する)の重要性は,身の回りを見ればよくわかる.ねじ,ビデオテープ,フロッピーディスクなど数え上げたら切りがないほどの製品は標準に則って製作されている.これらが標準化されていなかった場合,生活が不便となる.さらに安全にも影響があり,人命に直接影響する場合もある.国際標準を制定している組織も数多くあり,一般によく知られている組織としては,機械関係ではISO(International Organiza-tion for Standardization,国際標準化機構),電気関係ではIEC(International Engineering Consortium,国際電気標準会議)などがある.国際標準の例としては身近な,非常にインパクトのある品質保証標準ISO 9000がよく知られている.最近ISO標準が重要になった理由は,上に述べた米ソ冷戦終結に関係している.国際標準化機構(ISO)は非営利団体組織として1828年に発足し,冷戦前から多くの国際標準を発行してきている.米ソ冷戦中は,資本主義圏と共産主義圏が対立していたため,国際標準はあったもののその利用は非常に限られた範囲であった.冷戦終結後,両者の対立はなくなり,グローバル化が進む中で国際標準の重要性が増したといえる.さらに,ヨーロッパをはじめ購入条件にISO 9000の取得を条件付けるに至って,ビジネススタンダードとなりISO 9000の重要性は一気に高まった.その結果,ISO 9000認定を受けていなければ輸出ができないといわれるまでになり,日本の企業でISO9000を取得している企業は3万5000社を超えるに至っている.
