# 4・3・3 国際標準に対する各国の認識

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

国際標準で重要なことは,国際標準の内容をどうするかである.自由貿易のため各国の規制を撤廃し,国際標準に従っていればどこへでも輸出できるようにしようとすることがねらいであるが,万一,自国に不利な標準となった場合,結局は国益を損なう結果となる.このため欧米では,自国にメリットがある国際標準とするため官民を挙げて自国の産業の優位性を確立すべく産業政策を実施している.これだけ企業活動がグローバル化しているときに,自国がどうのという議論でもないという考えもある.しかし,諸外国の企業の行動,ヨーロッパ委員会やアメリカ政府の打出していることを考えれば,国家の利益をかけてし烈な争いに入っているといえる.国際標準化機構などで作られる国際標準は,世界で共通に用いられるインフラストラクチャである.日本は世界第2位の経済大国であるが,国際標準を作成し,決定する場面において,日本の果たしている役割と貢献は非常に小さい.その結果,日本は各方面で非常に損をしている.ISO標準などの公的な標準,あるいは市場で形成されるデファクト標準も含め,国際標準とは共有のインフラストラクチャであると同時に,各種活動を制約する一種のルールである.少しでも自国に合ったルールを作ろうと競合しているときに,そのルール作りに積極的に参画し,日本の意見を国際標準に反映する努力をしていないのは,日本の権利を放棄していることに等しい.
