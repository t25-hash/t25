# 4・3・4 国際標準のシステム

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

国際標準は技術標準(technical standards)、品質保証標準(quality assurance standards)と認証・認定標準(conformityassessment standards)で構成されることを理解する必要がある。それらは通常図4・9のように相互補完してその機能を発揮する。こ図4・9国際標準の体系1―108機械工学総論の考えは、規制基準、民間標準においても同様である。技術標準は、最新の技術に基づき認定製品としての技術的細目(例えば許容値、許容公差等)をまとめたものである。技術標準は技術の集約とはいえ、多くの技術判断(エンジニアリングジャッジメント)が含められており、完全ではない。このため、技術標準の欠点を補うために品質保証標準が発行される。認証・認定標準は、技術標準に従って設備、機器が製作されているか、また品質保証標準どおりに管理が行われているかを確認する方法を述べたもので、これらがすべてそろって初めて標準が有効となる。ISOでは、各国が責任をもって認証・認定を行えばよいと考えているが、ヨーロッパでは、ヨーロッパ内の認証・認定作業はヨーロッパ内の組織が行えばよいと考えている。グローバライゼーションの考え方は良いが、その実施となると各国の利害関係が現れてくる。
