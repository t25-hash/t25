# 4・3・6 欧米の国際標準への対応

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

国際標準の重要性を述べたが，以下に国際標準に対する各国の戦
略を述べる．
a.ヨーロッパにおける国際標準戦略 ヨーロッパには，国ご
とに日本のJISと同じような工業標準がある．例えば，イギリス
ではBS（British Standards，英国標準），イタリアではUNI
（Nazionale Italiano di Unificazione，イタリア標準）等である．
それぞれに長い歴史のある標準であるが，市場統合の進むヨーロッ
パでは，自由な物流を実現するためにこれら各国の標準の統合を進
めている．その主体となっているのが，CEN（European Commit-
tee for Standardization，欧州標準化委員会）とCENELEC（Eur-
opean Committee for Electrotechnical Standardization，欧州電
気標準化委員会）である．いずれも1965年に創設され，加盟国は
2003年末の時点でEUメンバ15箇国に，スイス，ノルウェー，ア
イスランド，チェコ等の13箇国を加えた28箇国である．
標準の作り方は，ISO（国際標準化機構）やIEC（国際電気標準
会議）と基本的に同じであり，関係者が委員会に集まって原案を作
り，最終的には加盟各国の投票で承認する．すでにCENおよび
CENELECは約2000のヨーロッパ標準を作成しており，加盟各国
はこのヨーロッパ標準を国家標準として導入し，矛盾する国家標準
については撤廃・変更する義務を負っている．
ヨーロッパ域内で各国ばらばらの標準に代えて統一標準の作成に
取組むのは，ヨーロッパ内の自由貿易のためである．ヨーロッパは
ヨーロッパ標準をそのまま国際標準にすることを明確にして戦略化
し，意図的に進めている．標準作成の技術委員会の開催場所に制限
はないが，多くの場合，ヨーロッパ内で開かれる．主要なものだけ
でISOには約800，IECには約200の標準作りの委員会があるが，
これら委員会のすべてにそれぞれ幹事が決められている（図4･
10)．幹事は国際標準の作成のために，主導的な役割を果たし，標
準への影響力が大きいため，幹事引受け数を見れば，その国の国際
標準への影響力がわかる．引受け数を国別に見ると，ISOとIEC
の合計で，上位からドイツ，アメリカ，イギリス，フランス，スウ
ェーデン，日本の順となっている．アメリカ以外ではヨーロッパの
国々が圧倒的に多く，欧州連合（CENメンバ国）を合計してみる
と，ISOの全幹事数の61%，同じくIECの66 %をヨーロッパのど
こかの国が引受けていることになり，ISOとIECをヨーロッパが
主導していることを示している．
ISO，IEC標準の決定で非常に重要なものに幹事国の数と並んで
票数がある．国際標準は1国1票の投票で決められるためだが，こ
の点でもヨーロッパは優位に立っている．とはいってもISOや
IECの規則では，投票総数の4分の1以上が反対すると国際標準は
成立しない．ヨーロッパは国が多いからといって，ヨーロッパ票だ
けでは国際標準は成立しない．このため，ヨーロッパでは，反対し
そうな国に事前にコンタクトし，調整を行う．
b.欧州連合の協定 ヨーロッパにはウィーン協定（Vienna
Agreement）とドレスデン協定（Dresden Agreement）がある．
ウィーン協定はISOとCENの間，ドレスデン協定はIECと
CENELECの間で結ばれており，両協定とも1998年に発効してい
る．内容はほぼ同じなのでウィーン協定を例にとると，ISOと
CENが一定の条件のもとで合意すれば，CENの委員会で作った標
準案をそのままISOの標準原案として扱い，ISOにおける委員会
審議を経ることなく，いきなり国際標準にするための全体投票
（CENとの並行投票）にかけることができる．
すなわち，ヨーロッパ内で作った標準案に特権的取扱いを認め，
そのまま国際標準にするバイパスが作られている．国際標準作りの
過程で重要なのは，委員会審議の段階で各国の利害関係者が技術的
な意見を述べ，それを国際標準原案に反映させるプロセスを経るこ
とにあるが，ウィーン協定を使われると，ヨーロッパ以外のISO
メンバは，最後に投票するだけでISO標準の原案審議に実質的に
関与できない．
これらの協定に，日本などヨーロッパ以外の先進国は反対してい
るが，協定廃止論はいまだISO，IECのすう勢とはならず，これ
らの協定は維持されたまま今日に至っている．
