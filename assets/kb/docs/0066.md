# 4・3・8 関連事項

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

次に,国際標準の話をする際に避けて通れない製造者責任,知的所有権,技術者倫理について説明する.これらも国際化の中で重要性が増してきたといえる.a. 製造者責任(producer responsibility)WTO/TBT 協定により,国際規格を満足した設備,機器は国境を越えて自由貿易が可能になった.こうなると,これまでのように各国の政府がオーナに対して規制を課して安全を確保するのではなく,製造者自身が標準を満足していることを保証する必要性が生じた.このため,各国では製造者責任ルールを明確化した.日本では1994年に購入者を保護するために,製品の欠陥によって生命,身体または財産に損害を被ったことを証明した場合に,被害者は製造会社などに対して損害賠償を求めることができる法律を制定した.具体的には,製造業者等が自ら製造,加工,輸入または一定の表示をして引渡した製造物の欠陥により,他人の生命,身体または財産を侵害したときは,過失の有無にかかわらず,これによって生じた損害を賠償する責任があることを定めている.また製造業者等の免責事由や期間の制限についても定め,製造業者,消費者がお互い自己責任も踏まえながら,製品の安全確保に向けていっそうの努力を払い,安全で安心できる消費生活を実現するために制定された法律である.ただし,製造物をその製造業者等が引渡したときにおける科学または技術に関する知見によっては,当該製造物にその欠陥があることを認識することができなかった場合の責任には免責事項がある.また,製品が部品または原材料を使用して製造され,その欠陥に製造者の過失がない場合,製造者の責任は免責される.法律的には製造者の責任は明確になっているが,その内容が明確になるものではなく,各種条件によって異なる.引渡したときにおける科学または技術に関する知見によっては,製品に欠陥があることを認識することができなかったことが,裁判において非常に大きな問題となる.国内の訴訟の場合はまだ簡単であるが,海外との取引きで問題が生じたときはより複雑になる.その背景には,日本のルールと海外のルールに差があることが挙げられる.あらかじめその差を理解し対応しておくことで,結果として責任の程度を軽減できる.b. 技術者倫理(ethics of engineers)技術者倫理を簡単に述べれば,技術が社会および自然に及ぼす影響・効果に関する理解力や責任など,技術者として社会に対する責任を自覚する能力となる.グローバル化した社会では,各国共通の技術者としての倫理をもつ必要がある.国内と国外でその内容を比較した場合,大きな差がある.国内の場合,一般にその内容は常識的かつ一般的な内容で読み手によっては翻訳が変わるものであった.価値観が同一の場合はこのようなものでも良かったが,グローバル化した社会では,明確に記載するのが普通である.特に国際標準のように世界で使われる場合,できるだけ解釈によって意味が異ならないように書く必要がある.社会は分業で成り立っている.現代社会では分業化が非常に進み,個人個人の生活に必要なさまざまな手段の多くを専門家にゆだねて暮らしている.公衆は日々の生活を専門家にゆだねているのであるから,専門家を信頼しなければならない立場にある.例えば人となりを知らないで,専門家が設計したものの安全性を信じるにはどうしたらよいか.国家試験などの資格制度,国の検査制度,事故を起こした場合への厳しい処罰制度などはもちろん有効である.しかしそれだけでは十分ではない.最後は人と人との信頼関係が必要である.専門家がきちんとしたモラルを有していることが見えれば,われわれは専門家に安心を託せる.このためには技術者に倫理が必要とされる.倫理規定には,上に述べた意味以外に,組織の存続に関係した役割がある.製造者責任にも関係するものであるが,学会から発行する標準,企業が製造する製品はいかに完璧を図って作成,製造しても完全ではない.このため,標準に従って作成した製品でも事故・故障が発生し損害が発生し,損害賠償責任が生じる.その場合は,標準策定組織,製造企業そして従業員個人の責任が追及されることになる.この場合,倫理規定が非常に重要な役割をもつ.グローバルスタンダードでは,一生懸命やりましたとの精神論だけで責任から逃れられることはない.この場合,組織,企業には明確な倫理規定があり,その内容を従業員に理解させるために適切な教育をタイムリーに行い,かつ企業と従業員は倫理規定に従って仕事を行うことを約束し,その取り決めを従業員が守って業務を行っていることを証明することが重要で,それにより,組織,企業および個人の責任の程度は大きく変わることになる.このため,海外の企業では多くの場合,倫理規定は非常に詳細なものになっており,従業員とはその倫理規定に従って業務を行うと契約することが一般的である.例えば,アメリカ機械学会の規格委員会の委員になる場合,倫理規定,独占禁止法遵守要領,民法遵守要領ほかがメンバ就任時にパンフレットとして送られてきて,それを遵守する誓約書を学会に送ることが要求される.これは学会を含めた組織の保護のために非常に重要であることを認識しなければならない.グローバル化した社会では,先に述べたように価値観の異なる人々と協力して業務を行わなければならなくなる.このことを前提に,読み手によって解釈が異ならない詳細な技術者倫理が必要となる.c. 知的所有権(intellectual property)知的所有権(著作権を含む)は,最初に考えた人・組織の権利を定め,文化的な所産の公平な利用に留意しつつ,権者の保護を図り,もって文化の発展に寄与することを目的としている.このための法律も知的所有権,著作権法として制定されており,近年産業活動において非常に重要な役割を果たすようになってきている.知的所有権の法律上の定義は「人間の幅広い知的創造の成果について,その製作者に一定期間の権利保護を与える」となっている.知的財産には,特許権,実用新案権,意匠権,著作権,回路配置権,植物品種権,企業秘密権があり,営業標識について商標権,商号権,著作票標・原産地表等(不当競争防止法)がある.知的所有権,著作権には国内の法律以外に国際条約があり,その一つが万国著作権条約パリ条約と世界知的所有権機関条約(ベルヌ条約)である.パリ条約は,著作権者の権利を示し,利用者が守らなければならないことを示したもので,ベルヌ条約は,有権者が守らなければならないことを示している.有権者が守らなければならないこととは,権利が無断で使用され,それによって事故・トラブルが発生し,損害が生じないように有権者の管理責任を問うもので,有権者の管理義務を負わせたものになっている.有権者の義務についても述べているとはいえ,基本的にこれらの法律・条約は所有権者を非常に広い範囲で保護するものとなっている.著作権でいえば,著者が登録する必要はなく,書かれたものであればすべて著作権があると理解しなければならない.対象のものが著作権に抵触するかの基準は,有権者が決めることになっている.さらに,著作権対象物の利用における費用については,有権者の判断にゆだねられている.このことを十分理解をせず,勝手な判断で使用すると,突如として訴状が舞い込み,そのために莫大な損害賠償請求をされ,企業の存続が危ぶまれるということになる.すでに,何億円,何十億円の損害賠償がされたということがニュースになっている.これらの有権者保護はあまりに厳しく,公共の福祉に反すると思われるかもしれないが,権者を保護することが結果として公共の利益をもたらすものと考えるのが国際標準の考えであり,この考えを理解し,対応しなければならない.
