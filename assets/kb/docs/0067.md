# 4・3・9 21世紀に向けた体制作り

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

国内では,同様な標準を各省庁もしくは監督している協会で制定または策定している.例えば,危険物タンクの標準は内容物および移動方法の差により,同様な標準を異なる組織で策定している.このため,その内容を審議する委員会も別々となり,関係者は同じような審議のため,異なる委員会に参加することになる.また,各委員会は縦割り行政のため,相互交流がほとんどなく,人的資源の有効活用の面から全く効率的でなかった.研究についても各組織が個別に計画を立てて研究を行ってきたため,同種の研究を重複して行うなど,資源の有効利用の面からもむだが多かった.米ソ冷戦中は市場も制限され,国際標準の影響もさほど大きくなく,国内のこのような非効率さも国益にさほど影響を与えることはなかったが,グローバル化した社会では,日本が不利にならないようにするためにはどうするかを考える必要がある.国際標準の重要性に,最初に気がついたのがヨーロッパであり,それに続いてアメリカがその対応を開始した.日本はこの面で,欧米に相当遅れを取っている.この現状を打開するにはまず,関係者が標準の重要性を認識し,世界の動きに対して危機認識をもち,協力することが条件となる.このためには,官・民・学が有機的に結合し,各人の能力を最大限に発揮する環境を作ることがその第一歩となる.すでに,欧米では数年前1―111図4・11 官・民・学一体の標準策定体制から官民が一体となって21世紀に向け貿易戦略を進めてきており,日本も猶予なく体制を固め,行動を開始しなければならない.
