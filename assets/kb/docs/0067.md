# 4・3・9 21世紀に向けた体制作り

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

国内では，同様な標準を各省庁もしくは監督している協会で制定
または策定している．例えば，危険物タンクの標準は内容物および
移動方法の差により，同様な標準を異なる組織で策定している．こ
のため，その内容を審議する委員会も別々となり，関係者は同じよ
うな審議のため，異なる委員会に参加することになる．また，各委
員会は縦割り行政のため，相互交流がほとんどなく，人的資源の有
効活用の面から全く効率的でなかった．研究についても各組織が個
別に計画を立てて研究を行ってきたため，同種の研究を重複して行
うなど，資源の有効利用の面からもむだが多かった．米ソ冷戦中は
市場も制限され，国際標準の影響もさほど大きくなく，国内のこの
ような非効率さも国益にさほど影響を与えることはなかったが，グ
ローバル化した社会では，日本が不利にならないようにするために
はどうするかを考える必要がある．国際標準の重要性に，最初に気
がついたのがヨーロッパであり，それに続いてアメリカがその対応
