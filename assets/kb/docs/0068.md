# 4・3・10 終わりに

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

が、国益のため非常に重要であることを述べた。このためにはまず国内の体制の充実が必要であり、特に人的、金銭的資源を有効に機能させるため、個人個人の経験・能力が標準作成に効率良く反映できる体制作りから始めなければならない。標準の策定には中立、公正、透明性が要求されることから、学会を中心とした体制とすることが必要であり、関係者がこの流れをサポートしていくことが必要である。その結果、学会に知識が集中し、学会が力をもてば国際標準への影響力も増し、日本のさらなる発展につながると考える。その体制を図4・11に示す。この基本は、各種研究機関が行った研究成果を、国および産業から独立しボランティアで運営されている専門家集団である学会・協会に持ち寄り、公正、中立、公開の場で議論しコンセンサスプロセスを経て、公衆審査を受けた後で発行する。国をはじめ企業はその目的に合致する標準(規格)を使い(国の場合は規制に呼び込み、企業の場合には企業マニュアルに呼び込んで)利用する。これにより学会・協会にリソースが集中し、結果として、国際標準への影響力が強化され、これが国民の安全と福祉の向上につながり、日本の国益に損なうことなくグローバリゼーションが図られることになる。以上、国際化における標準の役割およびそれに関係する項目について述べた。ここに述べられた内容は、標準化に対する各国(特に欧米)の思惑を中心に述べており、一般に公表された内容ではない。ISO,IEC,JIS,CEN,CENELEC等の詳細な情報が必要な場合は下記のウェブサイトから入手可能である。ISO http://www.iso.ch/iso/en/ISOOnline.openerpageIEC http://www.iso.ch/iso/en/ISOOnline.frontpageCEN http://www.cenorm.be/CENELEC http://www.cenelec.org/
