# 4・4・1 はじめに

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・4 少子高齢化

現在先進国を中心に急速な高齢社会の進展に対応するために、高齢者の生活を支援する技術の確立が重要性を増してきている。特に、高齢者の増加は介助者人口の相対的な減少も意味しており、高齢者や障害者の自立生活を支援するための機器開発が急務となっている。このため、寝たきりの人や車いす使用者はむろんのこと、その可能性のある人も含めて、できるだけ他人に頼らず自立した生きがいのある人生が送れること、その結果として21世紀の社会が活力あるものになることが期待されており、高齢者や障害者の自立や社会参加を促進する福祉機器を開発することが重要になる。
