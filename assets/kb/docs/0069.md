# 1・4・3 フート・ポンド系（foot -pound system)

> **章**: 第1章 単位系と次元
> **節**: 1・4 その他の単位系

基本単位として長さのフート（ft ），質量（または重量）のポン
ド（lb），時間の秒（s ）に基づいた単位系で，欧米特にイギリス連
邦諸国とアメリカで広く用いられてきたが，SIへの移行に伴って
廃止の方向にある．基本単位の一つに質量を用いる絶対単位系と力
（重量）を用いる重力単位系があることは1･4･2項の場合と同様で
あり ，同一のポンドという名称を用いるため，重力単位系では重
量ポンド（lbf ，lbw）などのように表し区別する．長さ，質量，時
間以外に温度の単位として華氏（ファーレンハイト）度（℉）［温
度差を表す単位はランキン度（° R)］，熱量の単位として英熱量単位
［British Thermal Unit （BTU，Btu）］，工率の単位として英馬力
（hp，HP，㏋）(1hp＝550ft ･ lbf /s ）などである．長さ，質量な
どの基準がSIとは異なるため，SIと共通する単位はほとんどな
い．同単位系内の換算関係は十進法的でないため複雑であるが，お
もなものは次のとおりである．
長さ：1フート（複数はフィート)＝(1/3）ヤード＝12イン
チ＝(1/5280）マイル＝0.3048m
質量：1ポンド＝16オンス＝(1/2240）英トン＝(1/2000）米
トン＝0.45359237kg
