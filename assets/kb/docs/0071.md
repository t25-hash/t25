# 4・4・3 感染症から生活習慣病へ

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・4 少子高齢化

図4・15には加齢に伴う死因の変化を示す。生活習慣病が死因の多くを占めることがよくわかる。20世紀の末にはエイズ・西ナイル熱・サーズ(SARS)等従来の抗生物質では完治できない感染病が世界全土でまん延してきており、今後の医療の課題になりつつある。図4・16に示すように20世紀の日本における病気は、前半は肺結核をはじめとした感染症が圧倒的に多かったのに対して、後半はがん・心臓病・脳卒中を中心とした生活習慣病が急激に増加した。医療の進歩に伴い、細菌やウイルスによる感染を抑える抗生物質に図4・12 総人口に占める高齢者比率図4・13 21世紀の社会図4・14 加齢に伴う身体能力の変化より感染症が飛躍的に減少した。一方で第二次世界大戦からの復興後は食生活の向上を中心とした生活環境の変化により、大幅な寿命の延びとそれに連動した形で生活習慣病が急激に増加した。
