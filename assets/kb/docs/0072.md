# 4・4・4 医療費の暴騰とその解決法

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・4 少子高齢化

上述の背景から,図4・17に示すように国民医療費が20世紀末には30兆円を突破した.これは医療全体の市場規模といえるが,このうちで機器が占める市場は2兆円である.従来は感染症を対象とし,創薬を商品としていた時代の流れを継続した状態にある現状が読み取れる.具体的にどのような分野が成長しているかを示したのが図4・18である.創薬・材料や診断用機器から治療機器へ移っていることが読み取れる.少子高齢社会では,高齢者は寝たきりではなく生活の質を確保できる支援を得る必要がある.このような視点から,高齢者の生活習慣病に対する低侵襲診断・治療の実現や,細胞分裂中の胎児の低侵襲診断・治療の実現で障害のない新生児の誕生も求められている.また,身体能力の低下した高齢者や障害をもった若年者のリハビリや日常生活を支援する知的機械システムも需要が高まり,先進国を中心にさまざまな試みの成果として登場してきている.図4・15 加齢に伴う死因の変化(1996年度厚生省統計)図4・16 日本国民の病気の変化図4・17 国民医療費と医療市場の推移IT の進展は,経済産業構造を改革するとともに,国民生活に多様なライフスタイルを実現するものとして期待されている.しかしながら,2000年問題は幸い大事には至らなかったものの,さまざまな危険性もはらんでいる.例えば,IT 社会[IT (informationtechnology) society]の進展に伴って,情報生産量が加速度的に増加することが予測されている.この情報の大洪水にさいなまれながらも,多様化する情報通信サービスを実現するために,さらには,次世代に残すべき情報(人類の文化)を着実に記録・蓄積するために,超高密度な記憶装置の実現が待たれている.また,IT 社会の進展に伴って,情報通信サービスに必要とされるエネルギー消費量も大幅に増加する傾向にあり,地球環境保護のうえからも大きな問題となっている.このようなIT 社会の進展とともに発生する課題の解決に,機械工学・機械技術はどのような役割を果たすことができるだろうか.以下では,IT 社会の進展を俯瞰すると同時に,IT の進展とともに増大する情報生産量やエネルギー消費量について考察する.さらに,機械工学・機械技術への期待を込めて,高密度記憶技術とクリーンエネルギー技術の現状をまとめ,その将来を展望する.
