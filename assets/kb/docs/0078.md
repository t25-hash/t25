# 4・6・3 ナノテクノロジーの研究開発の現状と特徴

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・27 に示すが,以下の三つのキーワードが挙げられる.大きな特

人間環境調和型産業技術を作り上げるための共通基盤技術としてのナノマニュファクチャリング技術に関して,ナノテクノロジー全体における位置付けは図4・28に示したとおりであり,情報通信分野での技術開発から,バイオ,環境分野でのナノテクノロジーの発展を生み出すためのキーテクノロジーとなっている.現在のナノテクノロジーは,以下のような特徴をもつと考えられる.(1) ナノテクノロジー全体のロードマップを作成することが重要であるとともに,明確な目標をもって,ナノのサイエンスに本腰を入れて取組むことがたいせつである.これは,現在,多くの関心を集めているナノテクノロジーであるが,5年から10年で出口が見え成果が出てこないと,経済情勢が厳しいなか,企業の多くで本格的な取組みはできないと予測される.ナノレベルのエレクトロニクス技術を活用するバイオ,医療,環境技術という展開で,英知を結集して出口を考案するとともに,成果に至るまでの加工・計測技術などの基盤技術の確立も,ロードマップに位置付ける必要がある.連続体力学から量子力学的効果が顕著になりだすメゾスコピック領域にかけての学問体系は,まだまだ確立しておらず,技術としての体系化はさらに時間を要すると推測される.しかしながら,現在は新しい機能を見出して,技術として活用する方法を考案し,基本的な特許をどんどん取っていく段階であり,その競争が行われている段階である.例えば,生物のメカニズムを模倣する技術でも,生物のメカニズムを正確に解明することは重要であるが,応用する視点で,エンジニアリングとしてのバイオミメティックな応用技術を考案することがよりいっそう求められているといえよう.(2) ナノ加工装置,ナノ計測装置は,設備等が大がかりになるので,多くの企業や多くの研究者が協力して使用することが重要になる.マイクロマシン技術やMEMS 技術では,クリーンルームやリソグラフィー装置が必須で,かつ,種々のエッチング装置などをそろえることを考えると,場所的にも,資金的にも,運営面でも,大きな規模にならざるを得ない状況であった.これから本格化するナノ加工装置,ナノ計測装置の分野では,さらに大きな規模が必要になるので,多くの研究者の協力のもとに実施することが極めて有効である.さらに,ナノテクノロジーの場合,従来の工学でイメージされているクリーンという概念が通用しないことが多い.固体表面への吸着分子の性質で,表面の加工特性等は大きく変化し,また,汚れが多い中では,ナノバブル(nano-bubble,直径数百nm以下の気泡)等は計測できない場合が多く,マイクロ・ナノ機能を議論するには,再現性のある清浄な環境を実現することが前提となる.
