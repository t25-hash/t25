# 4・6・4 ナノマニュファクチャリングのロードマップ

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・27 に示すが,以下の三つのキーワードが挙げられる.大きな特

ナノマニュファクチャリング,マイクロマニュファクチャリングについてのロードマップの試案を図4・29に示す.このロードマップは,製造技術を,設計,加工,組立,計測の四つの技術に分けて検討したもので,さらにアウトプットとして,製品として予測される素子部品,機器装置を記述した.各技術分野ごとに発展が予測され,出口としての機器装置でもナノ加工のマザーマシンが10年以内に実用化されるのではないかと期待される.一つの特徴として,ナノメートルの領域に入ってくると,表面張力,分子間力,また,静電気力などの従来の部品では無視できた力が相対的に大きくなるため,部品を加工した後に組立を実施することは困難になるので,加工しながら組立を同時に行うという,加工と機能を発揮する構造体化を同時並行的に実施する必要性が生じてくる.
