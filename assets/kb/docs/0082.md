# 1・10・2 仮想仕事の原理

> **章**: 第1章 基礎
> **節**: 1・10 固体力学における変分原理

以上の場の方程式をすべて満足する解を以下では真の解という．
物体内部で連続な1階の導関数を有し，かつ 上で幾何学的境界
条件(1･73)を満足する変位場を運動学的可容変位場（kinematical-
ly admissible displacement distribution）という．また物体内部
で平衡方程式(1･70)を満足し，かつ 上で力学的境界条件(1･74)
を満足する応力場を静力学的可容応力場（statically admissible
stress distribution）という．いま物体が平衡状態にあり，変位
，応力σは真の解であるとする．物体の各点において連続な1
階の導関数を有する仮想的な無限小変位δを与える．ただし，
仮想変位（virtual displacement ）δは 上で零とする．このと
き仮想変位場＋δは運動学的可容変位場となる．この変位場
は，必ずしも物理的に存在するものではないという意味で仮想的で
ある．応力場σ ，表面力 ，物体力 の値は変化しないものと
考え，外力が仮想変位δに関してなす外部仮想仕事δ
δ＝ δd＋ δd (1･75)
を考える． 上で δが零であり 上の面積分を 全体に広げ
られることに留意してガウスの発散定理を適用し，応力場σが平
衡方程式(1･70)を満足することに留意し，δを式(1･71)に代入し
て得られる仮想ひずみ場δεを導入すれば，式(1･75)は次のよう
に書ける．
δd＋ δd＝ σδεd (1･76)
上式は，もし応力場が静力学的に可容であるならば，外力が任意の
δに関してなす仮想仕事が，応力場が仮想ひずみ場に関してなす
内部仮想仕事に等しいことを示している．逆に，式(1･76)から出発
して逆の順序で変形すれば，
－ σ＋δd＋ －δd＝0 (1･77)
を導くことができる．δは連続な1階の導関数を有すること以外
は 内と 上で任意であるから，式(1･70)および(1･74)が成立
し，その応力場は静力学的に可容である．この関係を，変形する物
体に対する仮想仕事の原理（principle of virtual work，仮想変位
の原理ともいう）という．上記の誘導過程では構成式(1･72)を使用
しておらず，したがって仮想仕事の原理は任意の力学特性を有する
物体について成立する．
