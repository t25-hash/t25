# 4・6・8 まとめ

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・27 に示すが、以下の三つのキーワードが挙げられる。大きな特

術全体の中におけるナノテクノロジーの位置付け、製造技術やその研究動向との関係、また、ナノテクノロジーの研究開発の現状と特徴を説明した。次に、ナノテクノロジーを体系的に整理する試みとして、基礎現象、製造技術、出口を含む応用分野の視点から整理し、動的プロセスである輸送現象との関わりについて考察した。さらに、マイクロ・ナノ現象をいかにして製品の大きさという広域に発現させ、実用に役立てるかという観点から、ミクロなスケールゆえに必要になる制御された自己組織化現象の活用による広域発現化の重要性と、熱・運動量・物質輸送現象がキーテクノロジーになりうることを説明した。さらに、機能発現は局所のナノ構造に基づき、実用的な効果はメートルオーダの製品全体に有効という、ナノ機能の広域発現技術を実現するのは、ものを作り上げる技術に永年取組んできた機械工学にポテンシャルがあることを説明した。機械工学を専門にする研究者が、一人でも多くナノテクノロジーの分野で産業技術の高度化を目指して活躍されることを期待したい。物理学、化学、生物学などの基礎科学の分野は、19世紀後半にエネルギー保存の法則、電磁場の基礎方程式、元素の性質と周期律、進化論およびX 線の発見など数々の成果によってそれまでの自然科学の集大成として進展し、現代の科学・技術の基礎を築き上げた。このような科学の進展は、実験・計測技術と情報技術によるデータ処理技術の発展に支えられてきており、科学と技術は真に車の両輪的関係で発展している。ここでは、宇宙の誕生とその成り立ちの神秘を探る宇宙科学と、生命の神秘とその仕組みの謎に迫り、医療・医薬、農林水産、食料、環境などその応用分野への寄与が期待されている生命科学(ライフサイエンス)について関連機械技術の役割について述べる。
