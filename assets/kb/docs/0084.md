# 4・7・1 宇宙科学

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・7 科学と技術

a. 天文学の変遷宇宙科学の始まりは、近代観測天文学の父ガリレオ・ガリレイ(Galileo Galilei)が自作の望遠鏡を空に向けた1609年にさかのぼる。ガリレイの使用した望遠鏡の倍率は約30倍といわれており、新世代の望遠鏡が肉眼で見える天体の10億倍以上暗い天体を見ることができるようになったことを考えれば、技術の進歩は著しいものである。次世代の宇宙科学における発見や新理論を支えていくのは、このたゆまぬ技術進歩でもある。表光学望遠鏡を用いた観測天文学が始まって以来約400年たち、観測対象は、可視光線から赤外線、電波、X 線、ガンマ線と広がり、赤外線天文学、X 線天文学、電波天文学などを生んだ。その過程で、天文学の発展を支えてきた技術は、観測感度と精度を高めるための光学技術に始まり、各種電磁波の計測技術、宇宙衛星技術、通信・データ処理技術等幅広い分野に及んでいる。最近のトピックスは、1990年にNASA によって打上げられたハッブル宇宙望遠鏡(Hubble space telescope)である。大気の揺らぎなど地上の大気にじゃまされずに観測できる高度約600kmの宇宙空間に打上げられた宇宙望遠鏡によって、これまで地上では観測できなかった100億光年以上の遠方の観測が可能になる。宇宙空間から見た星雲や星の姿や色彩は、これまで見たことのなかった鮮明な映像であり、まさに宇宙映像新時代の始まりといえる。この映像1―124機械工学総論年代씗宇宙科学分野での出来事>씗宇宙科学関連技術分野での成果>・色収差を除く凹レンズを使用した反射望遠鏡の発明・反射望遠鏡で天王星を発見(ハーシェル)・スペクトル分析法確立(ブンゼン・キルヒホッフ)・X telescope“Subaru”)は、これまでの望遠鏡の概念を変えたアクティブ制御技術を本格的に採用した。これは、鏡面の変形や大気の揺らぎ等を刻々検知し、機械制御技術、情報処理技術および光学技術で補正するアクティブ制御により高精度を実現した超精密マシンとしての大型光学赤外線望遠鏡である。この日本のすばるとハッブル宇宙望遠鏡によって、また新たな宇宙科学の成果が期待される。b. 国立天文台大型光学赤外線望遠鏡すばる日本で初の世界最大級の望遠鏡として、ハワイ島のマウナケア山頂(標高4200m)に建設されたすばるは、従来の望遠鏡の概念を変えたアクティブ制御方式などの最先端技術が採用され、一枚鏡としての有効口径8.2m、解像度0.2秒角という世界最高の性能を実現できた(図4・34)。そのキーテクノロジーとして高性能実現を支えたのは機械制御技術であった。すばるのドームは従来の球形ドームでなく、円筒形である。標高4200mの空気は、低温で乾燥した安定な条件を備えているが、さらに空気の揺らぎを低減するために、熱流体力学、構造力学、材料力学等の解析結果に基づいて設置場所やドーム形状が決定された。光学望遠鏡の進歩は主鏡口径の大型化に代表されるが、例えばヘール望遠鏡では、口径5m、厚さ0.9mの主鏡重量は20tであり、自重変形によるゆがみをパッシブに補正するための支持機構は、大型かつ複雑となり限界にきていた。すばるは口径8m、主鏡厚さは0.2mと極端に薄くし、自重変形を抑えた。鏡の剛性が減って変形しやすくなった分は支持点数を増やし、変形が生じても支持力を制御してアクティブにゆがみを補正するという従来の概念を覆した方表4・9宇宙科学および宇宙科学関連技術の変遷・凹凸レンズを組合わせた望遠鏡を発明(リッペルスハイ)この情報を基に倍率30倍の望遠鏡を作製(ガリレオ・ガリレイ)・月のクレータ、木星の衛星4個を発見(ガリレオ・ガリレイ)近代観測天文学の始まり宇宙映像新時代の到来・NASA、ハッブル宇宙望遠鏡(口径:2.4m)打上げ(1993:スペースシャトルで修理完了)・ハッブル望遠鏡でクエーサ発見、ブラックホールとの関連究明の手がかりとなる・(日本)電波望遠鏡「はるか」打上げ、搭載した望遠鏡と地上電波望遠鏡とが連携して高感度観測を実現(VSOP:VLBI m、表面粗さは0.01 m以下に加工される。主鏡形状制御系は、図4・35に示すように、高度角によって変化する鏡体の自重分の支持力を決める機能、鏡面形状誤差を補正する機能および各支点(261点)ごとに支持力を追従させる機能を有している。力アクチュエータはボールスクリューにモータでばねを圧縮させて、その力を力センサで検出しフィードバックする方式を採用した。鏡面測定装置には、主鏡からの光を平行にし、マイクロレンズを通して多数の点に集光するシャックル・ハルトマン方式が採用された。このように力アクチュエータおよび主鏡面測定装置には、それぞれ高精度化技術が採用され、直径8.2mの鏡の形状を50nmの精度以上で保持している。また、望遠鏡駆動方式には、ダイレクトドライブ方式や高剛性化サーボ機構を開発し、0.07秒角の追尾精度が得られた。光学系でも空気揺らぎ除去のための補償光学系が開発され、世界最高レベルの精度を実現できた。すばるは、機械技術、電子技術、光学技術および情報技術の融合した成果といえる。c.
