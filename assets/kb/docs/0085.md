# 4・7・2 生命科学(life science)

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・9 に宇宙科学分野での科学と技術の変遷をまとめた。

21世紀は、生命の神秘をさぐる生命科学の時代といわれており、国内外ともに産・官・学で精力的に研究開発が進められている。表の変遷をまとめて示した。1865年メンデルによって遺伝子という概念が導入された法則が発見されて以来、生命科学分野の発展は、1953年キャヴェンディッシュ研究所の生物学者ワトソンと物理学者クリックの専門分野の異なる2人の共同作業による世紀の大発見、DNA の二重らせん構造につながった。この発見の背景には、X 線の発見(1895年)、X線管の発明(1913年)およびX 線解析技術の進歩がタイミングよく合致していたことを忘れてはならない。二重らせん構造のヒントは、キングズカレッジの研究者が撮ったX 線回折写真の中にあった。このように、技術のシナジー効果が科学の進歩に密接に関係している。2000年6月28日、人間の全ゲノム情報(約30億塩基対)のドラフトシーケンス(概要版)の解読データが発表された。これは、国際ヒトゲノム計画(Human Genome Project)とアメリカベンチャー企業(セレラ・ジェノミクス社)の共同発表であったが、高速遺伝子解析装置(DNA シーケンサ)を300台設置し、膨大な実験データを高性能コンピュータシステムで処理した物量作戦の強さを示す出来事であった。このように、ライフサイエンス産業分野は、サイエンスとビジネスが同時進行するというこれまでに見られない知識集約産業構造をもつ。生命活動の根源とされる遺伝子・タンパク質の解明をはじめとする生命科学の発展は、それを支える分析・計測技術や情報技術など関連技術の高度化と産業化なしには実現されない。その成果表4・10 細胞すべての生物は細胞から構成されており、細胞の分裂で増殖し、種の保存がなされる。ヒトの体は約60兆個の細胞から構成されており、細胞の大きさは10~30 mである。(2) 染色体細胞の中で遺伝子DNA がタンパク質に巻付いた構造の物質で、遺伝子の保存庫。ヒトは父母からそれぞれ受け継いだ23対(46本)の染色体をもつ。(3) DNAデオキシリボ核酸(deoxyribo nucleic acidの略)。染色体を構成する物質でA,C,G,T の4種類の塩基が対をなして結合した二重らせん構造をもつ。この塩基配列に生物の遺伝情報が書き込まれており、分裂によって細胞から細胞へ情報が伝えられ、親から子へDNA を介して遺伝情報が伝えられる。DNAの直径:2.37nm、らせんのピッチ:3.4nm(10塩基ピッチ)、また、ヒトの一つの細胞中にあるDNA の全長は2mにも達する。(4) 遺伝子DNA 塩基配列の中でタンパク質の設計図になっている部分。ヒト遺伝子は約3万箇所。DNA(30億塩基対)全体の1.5%.(5) ゲノムある生物のもつすべての遺伝子情報の総称。ヒトゲノム、イネゲノム、マウスゲノムなど。(6) mRNAmessenger-RNA の略。DNA に書き込まれた遺伝子情報は、いったん二重鎖が解けてDNA の必要な部分がコピー(転写)されたmRNA が作られ、その情報に基づきアミノ酸配列に翻訳されてタンパク質が作られる。(7) アミノ酸タンパク質を構成する基本分子で、アミノ基とカルボキシル基をもつ化合物。DNA 塩基の三つの配列ごとにアミノ酸が決まり、タンパク質は数十のアミノ酸配列より構成される。(8) SNP(single nucleotide polymorphism、一塩基多型)遺伝子の中で約1000塩基に1箇所、塩基が個人により異なるところがあり、体質や疾病、薬剤への感受性や副作用などの個人差の要因になっているところ。(9) プロテオームある生物のもつすべてのタンパク質およびその情報。(10) 制限酵素DNA の特定の塩基配列を認識し、その部分を切断する酵素。遺伝子組換え等に使われる。(11) 遺伝子発現DNA に書かれた暗号が必要な部分だけコピーされたmRNA という物質がタンパク質を作る過程。(12) DNA チップガラス等の基板上に張り付けた多種類のDNA と蛍光色素によって標識したDNA の二重鎖結合作用を使ったDNA 検出方法。SNP解析や発現解析において一度に多量の遺伝子を解析できる特徴を有する。今後、DNA チップやマイクロアレイ技術は、DNA 解析の高速化、微量化へ進み、細胞からのDNA 抽出、PCR 反応、電気泳動など基本プロセスを統合した高集積化チップあるいはナノテクノロジー活用へと進歩する。(13) タンパク質構造解析タンパク質の三次元構造解析。X線結晶構造解析法とNMR 法が二大手法。強力で均一なX 線源としてシンクロトロン放射光(Spring 8)が使われる。(14) X 線結晶構造解析結晶にX 線を照射して試料中の原子によって散乱された回折パターンより結晶構造を推測する。1970年代よりタンパク質の三次元構造解析に広く使われている。(15) NMR(核磁気共鳴)nuclear magnetic resonanceの略。強力な磁場の中で整列している試料の原子核にラジオ波(RF)パルスを照射すると原子核によるエネルギー吸収が起こり(核磁気共鳴)、これをスペクトルとして測定し、アミノ酸からの水素原子核の状態を解析する。(16) バイオインフォマティクス情報科学と生命科学の融合領域。c. DNA・タンパク質解析技術DNA 塩基配列解析やタンパク質相互作用解析関連の計測・解析装置に活用されている機械技術を紹介する。. DNA シーケンサ(DNA sequencer)DNA 塩基配列(A,C,G,T の並び)を解読する機械をシーケンサ(sequen-cer)と呼ぶ。電気泳動法は、DNA やタンパク質などコロイド溶液に電圧をかけて電荷された分子の大きさや形によって移動速度が異なるのを検出する方法である。従来の平板型電気泳動を用いたシーケンサでは、ジュール発熱に起因する泳動電圧の限界があり、高速化は困難であった。この問題を解決するため、神原らは、平板の代わりに放熱効果の高い毛細管(キャピラリー)を複数本同時計測できるマルチキャピラリー方式シーケンサを開発した。図4・36にマルチキャピラリー方式シーケンサの構成を示す。本装置はDNA 断片を高感度に一括蛍光検出できるシースフロー技術と、各塩基に対応した蛍光を同時計測できる4色同時分離検出法を特徴としている。DNA シーケンス試料は、4種(A,C,G,T)の塩基に対応する蛍光色素で標識され、ゲルを充てんされたキャピラリーで分離されて電気泳動処理される。各キャピラリー端からシースフローセル内に泳動流出するDNA 断片は、シース液槽から供給される緩衝液の幕で安定な流れが生じ、DNA 断片どうしが混じり合うことなくシースフローとなる。このシースフロー端部にレーザ照射を行い、得られた蛍光像をプリズムで四つの像に分割してCCD カメラで検出し、画像データ処理装置を介してDNA 塩基配列結果を得る。シースフローの採用で散乱光がない高感度な一括蛍光検出が可能となった。キャピラリーの採用により、従来の平板電気泳動に比べて1けた高い泳動電圧(100~300V/cm)を印加でき、高速分析が可能になった。この結果、従来の20~50倍のハイスループットDNA 塩基配列解析(400~1000kb/d)が可能になり、ゲノム配列解析に多大な貢献をしている。. その他関連技術DNA 配列解析のみならず、SNP(一塩基多型)決定のための大規模解析や膨大な量のタンパク質解析などが世界的に進む中で、それを支える関連技術の高度化が緊急の課題である。例えば、藤田らは、電気泳動中のゲル温度を熱伝達モデルにより高精度に予測するモデル式を決定し、高感度・高スループットSSCP装置(一塩基多型解析)を開発した。電気泳動解析装置に本格的に熱力学的な解析がなされ、高機能化が実現できた図4・36
