# 1・1・1 力の釣合い(equilibrium)

> **章**: 第1章 静力学
> **節**: 1・1 力

物体が多くの力の作用を受けてその位置を変えずに静止の状態を保っている場合,物体に働くそれらの力は釣合っているといい,物体は平衡の状態にあるという.このことは,物体の部分についてもいえることで,その場合,物体の断面に働く内力はその断面の片方のみを考えるとき,これを外力とみなしてよい.剛体では,その質点間の距離が外力によって変わらないので,大きさ相等しく,方向相反する任意の2力を任意の直線上に付け加えても,あるいは取り去っても剛体に働く力の作用はこれによる影響を受けない.物体がその平衡の位置から少しの変位を受けたとき,その物体に働く力により始めの平衡の位置に戻される場合を安定な平衡といい,変位を受けると漸次始めの位置から遠ざかろうとする場合を不安定な平衡という.また変位を受けても新しい位置で平衡の状態を保つ場合を中立の平衡という.
