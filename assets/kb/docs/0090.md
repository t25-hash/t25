# 1・2・3 偶力とそのモーメント

> **章**: 第1章 静力学
> **節**: 1・2 力のモーメント

a. 偶力図1・14に示すように互いに平行で、大きさが等しく、向きが反対で、作用線の一致しない2力を偶力(couple)という。図1・12 合力のモーメントの図式解法(着力点の異なる多くの力の場合)図1・13 合力のモーメントの図式解法(平行力の場合)(b)(a)図1・14 偶力と偶力の腕2―3偶力は物体を特定の方向へ動かそうとする作用をせず、回転作用だけをする。その作用の程度を偶力のモーメントで表す。したがって、偶力のモーメントは、偶力によって生じる回転モーメントの大きさに等しい。図1・15で垂直軸につけたハンドルaになる偶力を働かせることと、ハンドルbに同一の大きさの偶力を働かせることは回転モーメントの点では同一である。すなわち、偶力は平行移動が行える。また、図1・16に示すようにその平面内で偶力を(a)から(b)のように任意に移動しても、あるいは偶力の働く回転軸をA からB のように平行移動してもその回転モーメントはであり、回転モーメントの点では同一となる。b. 偶力の表し方偶力とは大きさが等しく方向が相反する2力をいう。偶力の大きさは偶力のモーメントの大きさに等しい。偶力のモーメントは大きさと方向を有するからベクトル量である。図示し、そのベクトルの長さ=はベクトルの大きさ、すなわち偶力の大きさになる。ベクトルの符号は、モーメントの作用が反時計回りに働くときは正、その反対のときは負とする。c. 偶力の釣合い図1・17で各モーメントの代数和が零となる場合には、釣合いの状態にある。d. 偶力の置換図1・19で一つの力が力の作用線外の点Aに及ぼす作用を知るため、点A に大きさがに等しく方向の相反する力′,-′を働かせる。′,-′はそれ自身で釣合っているから剛体にはなんらの作用を及ぼさない。すなわち、一平面内のいくつかの力は、物体全体の釣合いに関する限り、その平面内の任意に定めた一点に作用する一つの力と一つの偶力とに置換えられる。ただし、偶力とその偶力の面に垂直な力とを合成することはできない。
