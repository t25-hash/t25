# 1・3・2 支点での反力

> **章**: 第1章 静力学
> **節**: 1・3 力の平衡と支点に働く力

a. 支点での反力図1・21に示すように支点で支えられた物体がなる外力を受けて平衡を保っているとする。この場合、外力と大きさの和が等しく方向が反対の力,が支点に働く。この力を支点での反力(reaction force)という。平衡の条件から支点での反力を求めうる場合を静力学的に決めうる、すなわち静定(statically determinate)といい、そうでない場合を静力学的に決め得ない、すなわち不静定(statically in-determinate)という。後者では、物体の弾性変形を考慮することにより反力を求めることができる。支点には、図1・22(a)(b)に示すような固定支点と自由支点の2種類がある。固定支点(fixed support)とは反力が支点を通る場合で、反力を水平、垂直の2分力,に分けることができる。自由支点(free support)では単に垂直方向の力のみを考えることができ、水平方向に移動可能である。図1・23は2支点からなる代表的なはりの例で、固定と自由の支点がある。反力を求めるには、,,を求める必要があり静力学的に決めうる場合である。両支点が固定の場合は、四つの力を求めなければならないが、平面の平衡条件は三つしかないので四つのうちの一つは静力学的には決めることができない。また、自由(a)(b)図1・21 支点での反力図1・22 固定支点と自由支点図1・24 静力学に反力を決めうる支点図1・23 はりの代表的な支点な3支点からなるはりでは3支点が水平にならんでいる場合、∑=0の条件が欠けているため静力学的に決めることができない。しかし、3支点が、図1・24のように同一直線上にないときは静力学的に決めうる場合となる。b. 平衡条件の応用図1・25(a)に示すはりに外力が働く場合の反力を平衡の条件から求める。点B での反力は水平方向のみに働き、点A での反力は任意の方向に働きうるとすれば、平衡の状態になるためには、3力,,の作用線は一点Cで交わらなければならない。したがって、点A での反力はACの方向で、図1・25(b)に示す力の三角形から,の大きさを求めることができる。なお、は水平、垂直の2分力,に分けうる。
