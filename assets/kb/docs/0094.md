# 1・4・2 トラス

> **章**: 第1章 静力学
> **節**: 1・4 機械・構造物の静力学

鉄塔、クレーン、橋りょうなど多くの棒状部材を用いて構成される構造物を骨組構造(frame work、あるいはframed structure)という。ピン結合は回転自由なので、ピンの中心のまわりにモーメントが発生せず、また部材が節点に及ぼす力も、部材が節点から受ける反力もともにピンの中心を通る。部材は両端でピン結合されているから、部材に直接外力が作用しなければ両端の節点から受ける力は釣合わなければならない。すなわち、両端に作用する二つの力はピンの中心と中心とを結ぶ作用線をもち、互いに大きさが等しく、方向が反対でなければならない。力としては、引張り、圧縮のいずれかの方向の力が働き、引張力を受ける部材を引張材(tension member)、圧縮を受ける部材を圧縮材(compres-sion member)という。一般に自重は、部材間に働く力に比べて小さいので省略することが多い。図1・26は代表的なトラスを示す。節点がピンでなく角度の変化に抵抗するように接合された構造をラーメン(rahmen,rigid frame)という。
