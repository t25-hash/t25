# 1・5・1 変形とひずみ

> **章**: 第1章 基礎
> **節**: 1・5 ひずみの定義

外力の作用などによって物体が変形（deformation）するとき，
物体の代表寸法に比べて変形が十分小さければ微小変形，そうでな
い場合は有限変形という．力やモーメントの釣合いは，厳密には変
形後の物体形状について記述すべきものであるが，微小変形の場合
は変形前の物体形状について考えてよい．
ひずみ（strain）は物体の変形の程度を表す量であり，単位寸法
当たりの変形量，あるいは角度の変化として表される．ひずみに
は，以下の各項のようなさまざまな種類がある．
