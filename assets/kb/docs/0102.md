# 2・2・1 質点系と重心

> **章**: 第2章 質点系の力学
> **節**: 2・2 質点系の運動

a. 質点系質点が多数集まってそれらが相互に関連付けら2―9れ、一つのシステムとみなされるとき、これを質点系と呼ぶ。質点は質点間に作用する内力(internal force)によって結びつけられ、また系の外から作用する外力(external force)を受ける。質点間の運動を拘束するように力が作用するとき、これを拘束力(forceof constraint)と呼ぶ場合がある。質点が無数に集まり、質点間の距離が固定されたものが剛体である。重力場においては重心を支えれば系は平衡を保つ。また、剛体の重心を実験的に求めるには、剛体の1点をもって糸でつるしたとき糸の方向は重心を通るから、剛体の2点についてそれぞれ糸でつるしたときの糸の方向の交点を求めればよい。
