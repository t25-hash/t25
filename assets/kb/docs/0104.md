# 2・3・1 運動量・角運動量と運動方程式

> **章**: 第2章 質点系の力学
> **節**: 2・3 運動量と角運動量

a. 運動量、力積質点の運動方程式(2・21)は以下のようにも書くことができる。よって、ニュートンの運動の第二法則は、運動量の時間変化割合は作用する力に等しいということもできる。式(2・35)は、力積は運動量の変化に等しいことを表している。式(2・36)はある固定点まわりの角運動量の時間的変化割合はその点まわりのモーメントに等しいことを表しており、これを角運動方程式という。式(2・37)は、角力積は角運動量の変化に等しいことを示している。これから、質点系に働く外力は質点系の運動量(または重心の運動量)の時間的変化割合に等しいことがいえる。式(2・36)を系全体について加算する。これから、ある固定点まわりの質点系に働くモーメントはその点まわりの系全体の角運動量の時間的変化割合に等しいことがいえる。式(2・43)より、原点まわりのモーメントおよび角運動量は、重心に全外力および全質量が集中したとしたときのそれぞれの値に、重心まわりのそれぞれの値を加えたもので表されることがわかる。質点系の運動を重心の並進運動と重心まわりの回転運動としてまとめると、並進運動に対しては外力および質量が重心に集中していると考えたように運動を行い、回転運動に対しては重心まわりの外力のモーメントが重心に関する角運動量の時間的変化割合に等しいように運動を行うといえる。これらの関係は剛体の運動の基礎を与えるものである。
