# 2・5・1 拘束条件での運動(motion in constraint condition)

> **章**: 第2章 質点系の力学
> **節**: 2・5 ラグランジュの運動方程式

空間内での質点は3自由度の運動が可能であり,O-座標系を設定すれば,座標軸から質点位置が定まる.運動を-平面内に制限すれば=0の条件を加えればよい.運動を制限する条件は拘束条件(constraint condition)と呼ばれる.拘束条件の数だけ,運動の自由度は減少する.拘束条件が,質点の位置関係として,次式で示されたものとする.,=0(2・62)2―12機械力学上式の等式を満たす場合には,質点の位置に一定の幾何学的関係をもちホロノミックな拘束(holonomic constraint)と呼ばれる.一方,等式が満たされない拘束を非ホロノミックな拘束(non-holonomic constraint)という.例えば,長さの弦でつり下げられた質点が重力の下で振動する場合を考える.直角座標系で,弦が直線で一定長さを保つならば,++-=0となり,質点は球面上を運動しホロノミックな拘束の状態となる.質点が振れ上がった際に,弦がたわめば,++-<0となり,質点は球面内部にも至り,非ホロノミックな拘束となる.なお,式(2・62)の拘束が時間を陽に含む場合,レオノーマス(rheonomous)と呼ばれ,弦長を時間変動させた場合に対応する.なお,時間依存のない拘束はスクレオノーマス(scleronomous)といわれる.
