# 2・5・5 ハミルトンの原理(Hamilton’s principle)

> **章**: 第2章 質点系の力学
> **節**: 2・5 ラグランジュの運動方程式

ダランベールの原理を導入した運動方程式(2・84)と仮想変位δとの内積をとると、次式を得る。時間からまでの時間領域での仮想仕事の総和として次式に示す時間積分を考える。第2項に、変分の交換法則と部分積分を適用すると次式となる。運動エネルギーと仕事の変分が、時間領域で停留値をとることに対応する。これを上式に代入して次式を得る。ハミルトンの原理を用いて、ラグランジュ関数の停留値を求めると、運動方程式を解析的に導くことができる。個の質点系におけるハミルトンの原理も同様に定まる。ラグランジュ関数はスカラ量であるため要素数全体について総和をとると、系全体のラグランジュ関数が次式のように求まる。
