# 3・1・1 剛体の運動学

> **章**: 第3章 剛体の力学
> **節**: 3・1 剛体の運動

運動の原因となる力をもち出さないで、剛体の運動を記述する方法を論じる運動学(kinematics)の立場から、剛体の運動を検討する。a. 平面運動剛体の各点が一つの平面に平行に運動するとき、この運動を平面運動(plane motion)という。平面運動する剛体を取上げる。この剛体をB と名付ける。運動の平面に平行に、剛体B 上に直線PQを固定する。直線PQによって剛体の運動を代表させることができる。図3・1に示すように、直線PQが、時刻のときPQの位置にあり、短い時間が経過したときP′Q′の位置にきたとする。このときの運動を、ある点のまわりの回転運動として表すことがつねに可能である。これを示すため、線分PP′,QQ′の垂直二等分線を引き、その交点をCとする。△CPQ≡△CP′Q′となることが容易に示され、したがって∠PCP′=∠QCQ′が成り立つ。このようにして時間の間における直線PQの運動は、点Cを中心とし、∠PCP′=∠QCQ′の角度の回転運動であると考えることができる。0とした極限におけるCの位置を瞬間中心(instantaneouscenter)という。時刻における剛体B の平面運動は、瞬間中心Cを中心とする回転運動であるということができる。b. 固定点まわりの空間運動次に固定点まわりを自由に回転する剛体B の運動を取上げる。いま剛体B が固定点Oのまわりに回転するとする。平面運動を論じたとき、剛体の運動を代表するものとして、剛体上に線分を定めた。固定点Oのまわりに回転する剛体でこの線分に相当するものとして、固定点Oを中心とし適当な半径の球面上にとった一つの大円の弧PQを剛体上に定める。図3・2に示すように、弧PQが、時刻においてPQの位置にあり、短い時間が経過したときP′Q′の位置にきたとする。PQが球面運動してP′Q′の位置にきたのであるから、P′Q′も一つの大円上にある。PQからP′Q′への運動をある軸まわりの回転運動として表すことがつねに可能である。これを示すため、2点P,P′を通る大円、2点Q,Q′を通る大円を定め、それぞれの大円上の弧PP′,QQ′を垂直に二等分する大円の交点Cを求める。a項の場合と同様に、球面上で△CPQ≡△CP′Q′を証明することができる。したがってPQからP′Q′への運動が、原点Oと点Cを結んだ軸OCまわりの回転運動と考えることができる。平面機構の場合と同様に、0とした極限における軸OCを瞬間回転軸(instantaneousrotational axis)という。時刻における剛体B の空間運動は、この軸まわりの回転運動と考えることができる。2―15c. 一般の空間運動剛体B が最も一般的な運動をする場合を考える。この問題では、剛体B 上に、1点Oと、Oを中心とする任意の半径の球面上に一つの大円の弧PQを固定し、これらの運動を調べればよい。剛体B を代表する点Oと弧PQが時刻においてOとPQの位置にあり、短い時間後にOはO′の位置、PQはP′Q′の位置に移動したとする。この間の運動を二つの段階に分けて考えることができる。第一段階は点Oが点O′に一致するまでの剛体B の平行移動である。この結果PQはPQの位置にくる。第二段階はPQがP′Q′に一致するまでの移動である。第二段階の移動では点O′は移動しないので、この移動をb項で示したように、O′を通るある一つの軸O′C まわりの回転運動と表すことがつねに可能である。0とした極限を考えれば、時刻における剛体B の変位を、平行運動とある軸まわりの回転運動の合成として表すことができる。剛体B 上の固定点Oを上では任意に定めた。この点を特定の位置にとれば、平行移動の方向と、回転運動の軸方向を一致させることができることを次に示そう。このため図3・3に示すように、O′を通り直線O′C に垂直な平面Π と、Oを通り直線O′C に平行な直線を考え、両者の交点をH とする。OからO′への平行移動はOからH,H からO′への移動の二段階に分けられる。前者の移動は回転の軸に平行である。後者の移動は平面Π 内の運動である。この平面運動と軸O′C まわりの回転運動は両方を合わせても平面Π内の運動であり、この平面運動を一つの軸まわりの回転運動で置換えることができるはずである。この回転運動の軸を求め、この上に初めの点Oをとれば、時間内の全変位は、新しく定めた点Oを通るある直線に平行な移動とこの直線まわりの回転運動の合成で与えられる。平行移動と平行移動の方向に平行な直線まわりの回転との合成をらせん変位(screw displacement)という。平行移動の距離の回転角に対する比をピッチ(pitch)という。らせん変位においては、平行移動と回転運動を行う順序を変えても全体の運動に変わりはない。
