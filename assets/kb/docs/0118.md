# 3・1・3 剛体運動の解析的表示

> **章**: 第3章 剛体の力学
> **節**: 3・1 剛体の運動

剛体の運動を解析的に表示する方法を,いくつかの場合について述べる.a. 固定軸まわりの回転運動剛体が固定軸まわりに回転するように拘束されているとする.空間に固定座標系O-を定め,固定軸を軸と一致させる.この場合の剛体の運動は基準の位置からの剛体の回転角φで定められる.角速度ベクトルは軸の方向にあり,その成分は軸方向にω=φで,他の軸方向に零である.b. 平面運動の場合平面運動する剛体を考える.平面が運動の生じている平面に平行になるように,固定座標系O-を定める.またξη平面が同じ平面内にあるように,剛体に固定した座標系O′-ξηζを定める.このとき剛体の運動は,O′の,座標,とξ軸が軸となす角φで定められる.角速度ベクトルは軸の方向にあり,その成分は,軸方向にω=φ,他の軸方向に零である.c. 固定点まわりの運動剛体が固定点のまわりに自由に回転するものとする.図3・5に示すように,固定点Oを原点とし空間に固定された座標系O-と,同じ固定点Oを原点とし剛体に固定された座標系O-ξηζを定める.剛体の運動を表すのに,ξ,η,ζ軸の位置を示せばよい.ξ,η,ζ軸の位置を示す量を考えよう.座標系O-をまず軸まわりに図3・5に示す角φだけ回転すると,軸はON の方向に,軸はOK の方向に移動する.次にOK 軸まわりに図3・5に示す角θだけ回転すると,軸はζ軸に,線分ON は線分OM の方向に移動する.最後にζ軸まわりに図3・5に示す角ψだけ回転すると,線分OM はξ軸の方向に,線分OK はηの方向に移動する.この順序で座標系O-を回転すると座標系O-ξηζが得られるので,ξ,η,ζ軸の位置を示す量としてθ,φ,ψを用いることができる.これをオイラー角(Euler angles)という.角速度ベクトルの,,方向の成分ω,ω,ω욨は図3・5 オイラー角表3・1 方向余弦ξcosθcosφcosψ-sinφsinψcosθsinφcosψ+cosφsinψ-sinθcosψη-cosθcosφsinψ-sinφcosψ-cosθsinφsinψ+cosφcosψsinθsinψζsinθcosφsinθsinφcosθω=-θsinφ+ψsinθcos φω=θcos φ+ψsinθsinφω욨=φ+ψcosθ(3・13)である.また角速度ベクトルのξ,η,ζ軸方向の成分成分ω,ω,ωは,ω=θsinψ-φsinθcos ψω=θcos ψ+φsinθsinψω=φcos θ+ψ(3・14)である.ξ,η,ζ軸方向と,,軸方向の間の方向余弦がしばしば必要になるので,ここでそれを示しておく.例えばξ軸方向の方向余弦を求めるため,ξ軸方向の単位ベクトルを考える.まずこの単位ベクトルのOM,OK 方向の成分を求めるとcos ψ,sinψとなる.前者の軸方向の成分はcos θcos φcos ψであり,後者の軸方向の成分は-sinφsinψである.これらを加え合わせるとξ軸方向と軸方向の間の方向余弦が得られる.同じように考えて表3・1の結果を得る.剛体の力学的特性を表す量として,重心と慣性モーメントを考える.剛体の運動を論じるとき,これらの量が必要になる.剛体内の点Oを原点として,座標系O-を定める.
