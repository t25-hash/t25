# 3・3・1 運動量

> **章**: 第3章 剛体の力学
> **節**: 3・3 剛体の運動量、角運動量、運動エネルギー

剛体が質点の集まりであるとし、質点の質量を욡、位置ベクトルを욡、速度ベクトルを욡とする。このとき、욡=욡욡=욡욡(3・47)で与えられる量を質点の運動量(momentum)という。剛体を構成する各質点の運動量を加え合わせた=∑욡=∑욡욡=∑욡욡(3・48)を、剛体の運動量という。式(3・16)を時間で微分し、式(3・48)と比較すると、=(3・49)を得る。は重心の速度を表すので、式(3・49)から、剛体の運動量は、全質量が重心に集まったと考えたときの重心の運動による運動量に等しいということができる。
