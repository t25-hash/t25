# 3・4・1 固定軸のある剛体の運動

> **章**: 第3章 剛体の力学
> **節**: 3・4 剛体の運動方程式

a. 運動方程式図3・8に示すように、剛体が固定軸まわりに回転するように拘束されている場合を考える。固定軸を軸に一致させる。この場合の剛体は基準の位置からの回転角φで指定することができる。運動を定める方程式は1個あればよい。このための方程式として、式(3・78)の軸方向の成分の式を用いることができる。この式は式(3・80)に示されるように、욨=욨(3・94)である。ここで、욨=∑욡욡욡-욡욡(3・95)である。固定軸のある剛体の運動を求めるとき、上式を別の形に書き直した式が便利である。これを求めるため、各質点の座標욡,욡を、極座標욡,θ욡を用いて、욡=욡cos θ욡,욡=욡sinθ욡(3・96)と表す。これを時間で微分して욡,욡を求め、剛体ではどの質点においてもθ욡が同一であることに注意し、これをωとおくと、욡=-욡ωsinθ욡,욡=욡ωcos θ욡(3・97)を得る。これを用いると式(3・95)から욨=∑욡욡ω(3・98)となる。この式のωの係数욨=∑욡욡(3・99)は、剛体の軸まわりの慣性モーメントである。式(3・98)(3・99)を式(3・94)に代入すると、욨ω=욨(3・100)となる。剛体の基準位置からの回転角φを用いれば、この式は、욨φ=욨(3・101)と書くこともできる。式(3・100)あるいは式(3・101)が求める方程式である。b. 例:物理振り子図3・9に示すように、重力の作用を受けて水平な固定軸まわりに回転する構造の剛体を物理振り子(physi-cal pendulum)という。固定軸を軸とし、重心G を通って運動の平面と平行な平面と固定軸の交点を原点Oとする。点Oと重心G の間の距離をOG=とおけば、式(3・101)によって運動方程式は、φ=-sinφ(3・102)となる。この式を、長さの、質量の単振り子の式φ=-sinφ(3・103)と比較すると、物理振り子の方程式と、長さを=(3・104)とした場合の単振り子の式は同じになる。このを相当振り子の長さという。式(3・102)に戻って平衡点まわりの小さい運動を考える。φが小さいとして、式(3・102)においてsinφ≒φとおくと、線形方程式を得る。この式から、物理振り子は、周期=2π(3・105)の調和運動をすることがわかる。
