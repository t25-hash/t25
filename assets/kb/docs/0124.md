# 1・1・4 入出力装置と入出力インタフェース

> **章**: 第1章 情報システム
> **節**: 1・1 計算機ハードウェア

ハードディスクドライブ（HDD:hard disk drive）は，二次記
憶装置として用いられる外部記憶装置であり，大容量で高速なアク
セスが可能である．磁性体を塗った高速の円盤にデータを読み書き
するもので，ランダムにアクセスできる．1990年代初頭で数十メ
ガバイトであった容量は現在，数十ギガバイトとなり，年々大容量
化している．データを保存する装置としては，従来，フロッピーデ
ィスクや磁気テープドライブなどが使われていたが，最近ではCD
（compact disk，compact disc）やフラッシュメモリ，DVD（dig-
ital versatile disk）などが使われるようになってきた．
人間とのインタフェースとなる外部入出力装置としては，キーボ
ードやマウス，ディスプレイがあるが，スキャナやタッチパッド，
ジョイスティックなどを含めて多様化している．
インタネットの発達に伴い，ネットワーク装置は最も重要な入出
力装置の一つになっている．従来はさまざまなネットワークがあっ
たが，現在ではイーサネット（Ethernet ）が標準的に使われるよ
うになっている．1990年代に10Mbps （bit per second，1秒間で
転送するビット数）だった伝送速度は100Mbpsと高速化し，1
Gbpsの転送速度をもつギガビットイーサネットが普及しつつあ
る．
CPUと外部の入出力装置でデータのやり取りを行う部分が入出
力インタフェースである．CPU自体は外部とのデータのやり取り
にバスを通してデータをやり取りするが，入出力インタフェースは
外部装置に合わせてバスの信号を変換する．
CPUが入出力インタフェースとの間でデータをやり取りする場
合には入出力命令を用いる．入出力装置からメモリにデータを読込
む場合，入出力命令を用いてデータをCPUに読込み，これをメモ
リに書き込むという動作を繰返す．しかし，この場合，入出力のた
めにCPUが介在するために，データ量が多くなるとCPUが占有
されてしまうという問題点がある．そのため，大量のデータを入出
力装置とメモリの間でやり取りする場合にはDMA（direct mem-
ory access ）という方法を用いる．DMAはCPUを介さずに入出
力インタフェースとメモリ間でデータ転送を行うもので，これを制
御する回路をDMAコントローラという．CPUは入出力命令で
DMAコントローラに外部インタフェースと転送先のメモリのアド
レス，データ量などをセットすることにより，DMAコントローラ
に自動的に転送を行わせることができる．
外部入出力装置のインタフェースにはさまざまなものがあるが，
大きく分けて，シリアルインタフェースとパラレルインタフェース
の二つがある．
シリアルインタフェースとは，1本の信号線で1ビットずつデー
タを伝送するものである．パーソナルコンピュータでモデムなどの
接続に使われるRS- 232Cは最も普及しているシリアルインタフェ
ースで，最大のデータ転送速度は115.2kbpsである．最近では，
さらに高速なUSB（universal serial bus ）やIEEE1394（Fire
Wire ）が使われるようになってきた．当初，キーボードやマウス
などの比較的低速の機器を接続することを想定してUSB1.1（最
大データ転送速度12Mbps ）が策定された．USB1.1が広く普及
したため，CDやHDDも接続されることが多くなり，高速データ
転送の需要が高まってUSB2.0（最高データ転送速度480Mbps ）
が策定された．これにより，二次記憶装置を接続しても十分な性能
を発揮できるようになっている．なお，USB2.0はUSB1.1の上
位互換が確保されており，USB1.1の機器もそのまま接続できる．
IEEE1394は高速なシリアル転送を実現する規格で，最大400
Mbpsまでの規格が決められている．機器を数珠つなぎにする，い
わゆるディジチェーン接続することが可能で，複数の機器を同時に
接続することができる．この規格はディジタルビデオの外部接続に
採用されている．
このほかに，シリアルインタフェースとしては，赤外線で外部機
器と接続するIrDA（Infrared Data Association）がある．
パラレルインタフェースは，複数のデータ線で情報を一度に転送
する方式である．二次記憶装置として使われるディスク装置など大
量のデータの転送に使われる．PCなどで内蔵のディスクドライブ
の接続に使われているのがIDE（integrated drive electronics ）イ
ンタフェースである．製造コストが安く，高速であるために広く普
及している．IDEを正式に標準化したものがATA規格である．
接続するためのパラレルインタフェースの一つである．当初の
SCSIは，8本の信号線で8ビットデータ転送を行い最大転送速度
は5MB（メガバイト)/ sであったが，その後，転送速度を向上さ
せてfast SCSIでは10MB/s ，さらにultra SCSIでは20MB/sの
転送速度が可能になった．同時に，8ビットのデータ幅を16ビッ
図っている．ultra wide SCSIでは，40MB/ sの転送速度が可能で
ある．
中心とした団体で策定されたインタフェースの規格で，多くのPC
で使われている．データ転送幅は32ビットで，転送周波数は33
MHzのため，データ転送速度は133MB/sである．複数の外部
HDDがある場合などはデータ転送速度が足りなくなりつつある．
そのため，さらにサーバなどに用途に高速化した64ビット，周波
数が66MHzのPCI規格では，最大533MB/ sの転送速度となる．
