# 3・4・1 固定軸のある剛体の運動

> **章**: 第3章 剛体の力学
> **節**: 3・4 剛体の運動方程式

a. 運動方程式図3・8に示すように、剛体が固定軸まわりに回転するように拘束されている場合を考える。固定軸を軸に一致させる。この場合の剛体は基準の位置からの回転角φで指定することができる。運動を定める方程式は1個あればよい。このための方程式として、式(3・78)の軸方向の成分の式を用いることができる。固定軸のある剛体の運動を求めるとき、上式を別の形に書き直した式が便利である。式(3・100)あるいは式(3・101)が求める方程式である。b. 例:物理振り子図3・9に示すように、重力の作用を受けて水平な固定軸まわりに回転する構造の剛体を物理振り子(physi-cal pendulum)という。固定軸を軸とし、重心G を通って運動の平面と平行な平面と固定軸の交点を原点Oとする。このを相当振り子の長さという。式(3・102)に戻って平衡点まわりの小さい運動を考える。φが小さいとして、式(3・102)においてsinφ≒φとおくと、線形方程式を得る。
