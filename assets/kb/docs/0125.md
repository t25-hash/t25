# 3・4・2 剛体の平面運動

> **章**: 第3章 剛体の力学
> **節**: 3・4 剛体の運動方程式

a. 運動方程式平面運動する剛体の自由度は3である。この変数として、図3・10に示すように、重心G の位置と重心を通る直線の傾きを用いることができる。b. 例:粗い斜面の上を滑らずに転がる円板の運動図3・11に示すように、半径の円板が水平面と角αをなす斜面を滑らずに転がるとする。円板は一様で、重心G は円板の中心にあるとする。図3・11に示すように、斜面に沿って下方に軸、それに垂直に軸をとる。この式から重心G は一定の加速度で運動することがわかる。したがって、静止摩擦係数μは、図3・11 斜面を転がる円板図3・12 固定点のある剛体2―25μ≧13 tanα(3・118)でなければならない。
