# 4・1・1 摩擦

> **章**: 第4章 摩擦
> **節**: 4・1 滑り摩擦(sliding friction)

接触する二つの面が相対運動しようとするとき，あるいは運動し
ているとき，接触していることに起因して相対運動に抵抗する力を
生じる現象が摩擦（friction）であり，その力を摩擦力（friction
forceまたはfriction）と呼ぶ．
ここでは，二つの面は固体面であるとし，流体の摩擦については
二つの固体間の潤滑剤としての役割のみを紹介するにとどめ，それ
以外の部分についてはα4編の流体工学を参照していただきたい．
二つの固体間の摩擦については相対運動の形態により滑り摩擦
（sliding friction）と転がり摩擦（rolling friction）に分類される
が，通常，単に摩擦といえば，滑り摩擦を指す場合が多い．4･1節
では，滑り摩擦について，その基本的な性質を示す．なお，潤滑
（lubrication）や摩耗（wear）などを含めた摩擦に関連する現象を
広くとらえた分野はトライボロジー（tribology）と呼ばれ，その
現象についてβ4編に示されている．ここでは，おもに摩擦として
機械力学に関連する部分を示す．
