# 4・1・4 摩擦と振動

> **章**: 第4章 摩擦
> **節**: 4・1 滑り摩擦(sliding friction)

摩擦はエネルギーを消散することによって振動を抑制する場合や、逆にいろいろな種類の自励振動の発生原因となる場合があり、摩擦は振動に非常に大きい影響を与えるものの一つであると考えられる。以下に、流体関連振動(第13章熱・流体と構造系の連成振動、α4編の流体工学参照)は除いて、摩擦と振動の関係のおもなものを示す。a. 摩擦による減衰力図4・3で速度こう配が正である流体潤滑の範囲内の振動に対しては、摩擦力は摩擦方向の振動に対しては粘性減衰的な特性を示す。また、接触面が滑っていない状態を中心とした振動のように、1サイクル中滑り面の相対速度の符号が変化する場合には、通常、摩擦力は滑り方向についての非線形減衰力として働く。その効果を応用したものに摩擦ダンパ(20・2節参照)などがある。b. 速度-摩擦力特性に起因する自励振動図4・3の混合潤滑のように負の速度こう配を示す領域や、静摩擦係数が動摩擦係数より大きい場合、すなわち、摩擦係数が速度とともに低下する場合には、振動系の中に摩擦による負の減衰が存在することになる。負の減衰項が存在すれば、エネルギーが系に流入することになり、摩擦が生じる方向の自励振動を励起する場合がある。一般的に負のこう配が存在するのは、図4・3のようにある速度範囲だけであるので、振動の振幅がある程度大きくなれば、静止と滑りを繰返すスティックスリップ(stick-slip)という形の振動になる場合が多い。c. 2方向の連成による自励振動摩擦力が垂直抗力にほぼ比例するという摩擦の法則が成り立つ場合において、仮に垂直方法の振動により垂直抗力が変動すれば、滑り方向の力である摩擦力もそれに比例して変動する。一方、滑り方向に振動しても摩擦を介しては垂直方向に力を及ぼさないので、垂直方向と滑り方向との間で摩擦力による非対称の連成力が発生し、2方向の連成による自励振動を励起させる場合がある。発生メカニズムの詳細については、d. 回転体の内部摩擦による自励振動a項で示した滑っていない状態を中心とした振動は、静止座標系であれば摩擦が振動抑制に働く。一方、回転体における回転座標系での摩擦の場合には、回転速度が危険速度を越えると摩擦は振動抑制効果から不安定化力に変化し、内部減衰による自励振動を発生させる場合がある。
