# 4・2・1 転がり摩擦の法則

> **章**: 第4章 摩擦
> **節**: 4・2 転がり摩擦

転がり摩擦(rolling friction)とは、ある物体が接触している物体の上を転がるときに起こる抵抗であり、乾燥摩擦などと比べれば、その大きさは非常に小さい。例として、図4・4のように直径の円筒の中心に平面と平行な力を加えた場合を考える。円筒が平面と平行に滑る場合の滑り摩擦力よりも小さい力で、接触部を中心とした回転運動を生じれば、円筒は平面上を転がることになる。このときに円筒が接触部から転がり方向と反対方向に受ける抵抗力′を転がり摩擦と呼ぶ。転がり摩擦係数(coefficient of rolling friction)μ욥については、いくつかの定義があるが、滑り摩擦の場合と同様、転がり摩擦力′と垂直抗力との比で表現される場合が多い。転がり摩擦は、おもに弾性ヒステリシス損失や微小滑りなどにより生じるといわれているが、その値が小さいため、いろいろな因子の影響を受けやす表4・3 鋼球の各種平面上のころがり摩擦係数硬鋼0.00002軟鋼0.00004~0.0001黄銅0.000045銅0.00012アルミニウム0.001スズ0.0012鉛0.0014ガラス0.000014鋼球直径1.59mm、荷重約1.5N図4・5 トラクションの例図4・7 滑車とベルトの摩擦図4・4転がり摩擦い。したがって、普遍性のある法則を得ることは難しいが、一般的に、を定数として、μ욥=욖욗(4・3)という形で表現できる。指数,については、いろいろな説があるが、については軟質金属で1に近い値を取る場合を除き零と考えてよいとされており、については、0.4~1.7程度の数値が実験的に報告されている。表4・3に、乾燥摩擦に対応する状態での転がり摩擦係数の測定例を示すが、表より非常に小さい値であることが確認できる。
