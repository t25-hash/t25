# 1・4・3 無線通信技術

初期の無線通信は，広範囲にわたるサービスエリアを少数の基地
局で対応するシステムがおもであった．これに対して，多数の基地
局を用意し，サービスエリアを複数のセル（cell ）と呼ばれるエリ
アに分割するセルラシステム（cellular system）が登場した．この
システムでは，出力信号のレベルが低く済むため，通信端末が使用
する電力は小さい．このため，大型のバッテリが不要となり，通信
端末自体の小型化・軽量化が可能となる．本項では，おもにセルラ
システムによる無線通信方式について概説する．
ところで，無線通信の方式として人工衛星を用いる方式もあるの
で簡単に触れておく．人工衛星は，地上約36000kmの地点で地
球の自転速度と同じ速度で周回する静止衛星と，これよりも低い軌
道上で地球の自転速度よりも遅い速度で周回する低軌道衛星に分類
される．いずれの場合も，地上の通信端末は信号を人工衛星に送
る．人工衛星はこれを増幅した後，地上にいる別の通信端末に転送
する．ここで，後者の衛星を用いた通信では，前者のそれと比べ
て，衛星との通信の信号レベルを低くすることができ，また，信号
の伝搬遅延時間が小さくて済む．その反面，一つの衛星がカバーす
る通信範囲が静止衛星と比べて狭く，さらに，速度が遅いために衛
星が地球の陰に隠れることがある．このため，多数の衛星が必要と
なるほか，通信端末は衛星がどこにいるかをつねに追跡する必要が
ある．
a.セルラシステム セルラシステムによる無線通信方式は，
1970年代後半から1990年代前半にかけてアナログ方式がおもに用
いられた．これは第一世代と呼ばれ，おもに音声中心の通信を目的
に用いられた．1990年代前半以降にディジタル方式が登場し，音
声のほか，64kbps以下の低速なデータ通信が行われるようになっ
た．これは第二世代と呼ばれる．また，より高速なデータ通信を行
うため，第三世代の通信システムであるIMT-2000（interna-
tional mobile telecommunications2000）が1997年に標準化され
ている．
セルラシステムにおけるセルとは，基地局と通信端末が通信可能
な範囲を示す．その形状は基地局を中心とした円となることが多
く，図1･27に示すようにセルを配置することで，サービスエリア
全体をカバーする．各基地局は交換局と有線で接続されており，こ
の交換局を経由して，異なるセルにいる通信端末や，他のネットワ
ーク網との接続が行われる．
以下では，セルラシステムにおける通信技術についていくつか述
べる．
.チャネル利用方式 無線通信方式では，限られた周波数
帯域を用いて通信が行われる．このため，この周波数帯域を効率的
に使用し，収容可能な通信端末数を高めることが求められる．その
方法の一つに多重化方式があり，利用可能な周波数帯域をさらに細
かい周波数帯域で分割するFDMA（frequency division multiple
access ）方式と，周波数帯域を短い時間（スロット）単位で区切る
時分割多重TDMA（time division multiple access ）方式がある．
前者は1人のユーザが分割された周波数帯域を占有するのに対し，
後者は各ユーザは定期的に利用可能なスロットが割当てられる．ま
た，FDMAで分割された周波数帯域をTDMAでスロットに分割
することも可能である．
これに対して，すべてのユーザからの通信を広範囲な周波数帯域

セルで送信に用いる周波数を再利用することが可能である．例え

また，セルの大きさはサービスエリア全体で一定である必要はな
い．ユーザが集中する地点のセルサイズを小さくして収容人数を増
加するとともに，ユーザが少ない地点のセルサイズを大きくするこ

末は自分が通信している地点のセルの大きさに応じて，出力信号の
レベルを変える必要がある．

user agent ）とは区別される．

.ローミングとハンドオフ 通信中に通信端末が，現在い
るセルから別のセルに移動したとする．この動作をローミング
（roaming）と呼び，このとき，接続する基地局を切換えたり，出
力信号のレベルを変更するなどの処理が必要となる．この処理をハ
ンドオフ（hand oﬀ）と呼ぶ．ハンドオフの処理時間が長いと通信
が一時的に切断されるため，切換え操作の高速化が要求される．
