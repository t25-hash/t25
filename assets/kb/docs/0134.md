# 4・2・2 トラクション(traction)

> **章**: 第4章 摩擦
> **節**: 4・2 転がり摩擦

転がり接触と滑りが共存した状態で、接触する2面に微小な速度差がある場合には、それに起因して摩擦力が生じる。その摩擦力をトラクションと呼び、通常、純粋な転がり摩擦よりも大きい値を示す。その例としては、トラクションドライブによる動力の伝達、鉄道における駆動輪とレールの間の相互作用などがある。図4・5のような二つの円筒が、接触面に垂直抗力を受け、わずかに滑りながら回転している場合を考える。摩擦係数の場合と同様に、μ욧=をトラクション係数(coefficient of traction)と呼ぶ。二つの円筒の周速の平均値を、速度差をΔとすれば、μ욧は滑り率Δの関数となる。乾燥摩擦の状態でのトラクション係数μ욧は、Δの増加とともに増加し、滑り摩擦における摩擦係数μ욅に漸近する。また、接触面において転がり摩擦と滑り摩擦が共存している状態では、Δが増加すれば、滑りの割合が増加していき、トラクション係数がμ욅に一致したときには、接触面全体が滑り出す。一方、潤滑状態でのトラクションでは、μ욧はΔの増加に伴いあるところまでは増加するが、それ以降は温度上昇による粘性低下のために、徐々に減少していく。
