# 4・3・1 斜面の摩擦

> **章**: 第4章 摩擦
> **節**: 4・3 機械要素の摩擦

図4・6のように斜面に静止している自重の物体に関する力の釣合いを考える.傾斜の角度をαとすると,は垂直抗力′と静摩擦力′の合力と釣合っているので,′=cos α, ′=sinα=′tanα(4・4)となる.静摩擦係数をμ욦とすれば,′が最大静摩擦力=μ욦′以下であれば,すなわちtanα≦μ욦であれば滑り出さない.ここで,等号が成り立つときの角度ρを静摩擦角(static frictionangle)と呼ぶ.斜面に沿って,物体を静かに引き上げるには,摩2―29図4・6 傾斜の摩擦擦力は,運動に抵抗する方向,すなわち図4・6の′と逆方向に働くので,動き始めるための力(図4・6に破線で示している)として次式の大きさが必要である.=sinα+μ욦′=sinα+μ욦cos α(4・5)また,自重では滑らない状態で,斜面に沿って下方に静かに動かすには,下向きの力として,=μ욦cos α-sinα(4・6)が必要である.
