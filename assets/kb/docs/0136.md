# 1・1・1 機械（machine）とは

> **章**: 第1章 機械工学概説
> **節**: 1・1 総説

機械とは何か．これを簡潔かつ適切に表現することは難しい．機
械の種類は極めて多く，千差万別だからである．日本語の機械とい
う言葉は紀元前3～4世紀ころ，中国の古典，『荘子』ならびに『韓
非子』という書物にさかのぼる．機とは仕掛けのこと，械は刑罰の
道具だから，機械とは仕掛けをもってむりやり何かをさせるもの，
ということになる．英語のマシン（machine）はギリシャ語のメカ
ネー，ラテン語のマキナに由来し，これも機構や巧みな仕掛けを意
味する言葉である．古今東西，さまざまな人が与えた機械の定義は
その時代を反映している．特に有名なものを以下に掲げる．
･荘周（c. 365～c. 290BC，古代中国，戦国時代の思想家）
ここに械あり，力を用うること甚だ寡（すくな）くして，而も功
を見（あら）わすこと多し．
『荘子（外編・天地12）』より
･韓非（c. 280～c. 233BC，古代中国，戦国時代の思想家）
舟車機械の利は力の少なきを用いて功大を致す．すなわち多くの
利入る．
『韓非子（巻15・雑：第37）』より
･ヴィトルヴィウス（Vitruvius Polio,M. ，紀元前1世紀ころ，古
代ローマの建築家）
機械（マキナ）とは木材を組合わせたもので，おもに重いものを
動かすのに用いる．
『建築術』より
･ロイポルト（1674～1727，ドイツの機械学者）
機械とは人工的製作物で，その助けをかりて運動を起こすことが
でき，時間または力が節約される．
『機械の劇場』（1724年）より
･ルーロー（Reuleaux,F. ，1829～1905，ドイツの機械工学者）
機械とは，力に対して抵抗を示す物体が組合わせられて定まった
相対運動を行い，外部からエネルギーを受取って有用な仕事をす
るものをいう．
『理論運動学』（1875年）より
ルーローの定義は現代の辞書・事典類にそのまま引用されること
が多いが，これにも19世紀後半という時代的な制約がある．この
定義は動力機械と作業機械が機械の主流だった当時の状況を反映し
ており，現代の観点からはいくつかの問題がある．一つは物理量や
機械量の指示・記録を目的とする（すなわち，機械的な仕事の遂行
が目的ではない）計測機器がこの定義から外れること，もう一つは
この定義ではエレクトロニクスが含まれないことである．計測機器
の発展と普及は20世紀前半から，エレクトロニクスの発展と機械
への応用は20世紀後半のことである．機械は時代とともに発達し
変化していくから，その定義もまた変わっていくのは当然である．
これらを考慮して現代における機械を改めて定義すれば，機械と
は次の三つの要件を満たすもの，ということができる．
(１) 機械は人工物であり，それぞれ特定の役割を担う物体（機
械要素，流体要素，電気・電子要素など）の集合体である．
(２) これらの物体は意味あるように組合わされ，全体として目
的にかなった機能を現出する．
(３) 目的とする機能の現出に当たっては，機械的な力と運動の
両方あるいはいずれか一方が重要な役割を果たす．
ところで，はさみやノギスなどは上記三つの要件をすべて満たし
てはいるが，慣習上これらは工具あるいは道具といって機械とはい
わない．また，機械工学が扱う範囲のものであっても，構造部分が
運動を伴わない溶鉱炉や圧力容器（タンク）などは装置といって，
機械とは区別する．また，計測器のように小型で，かつ機械的な仕
事の遂行を目的としないものを器具ということがある．機械と器
具，さらに装置をも含めて機器という．
現代ではコンピュータを俗にマシンということがあるが，これは
機械とは考えないのがふつうである．コンピュータが目的とする機
能（論理演算，情報処理など）はもっぱら電子装置によって達成さ
れ，機械的な力または運動によるものではないからである．逆に情
報関連機器であっても，プリンタやビデオ機器，自動改札機などは
機械構造とその運動が重要な役割を担うので，りっぱな機械であ
る．
