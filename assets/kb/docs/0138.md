# 1・1・3 機械工学（mechanical engineering）とは

> **章**: 第1章 機械工学概説
> **節**: 1・1 総説

機械（単体ならびにシステム）の開発，設計，製作，運転，保全
に関わる技術が機械技術であり，機械技術と機械製品に関する学問
と知識の体系が機械工学である．技術と工学の関係については項を
も重要なことがらである．
現代の機械工学はおおよそ次のような内容をもつ．
a.基礎工学分野 機械的エネルギーあるいは機械情報のふる
まいと操作を扱う工学分野である．工業力学，機械力学，材料力
む．工業数学をここに含めることもある．
b.生産工学分野 機械単体ならびに機械システムの設計と生
と加工機器，機構学，機械要素・トライボロジー，計測・制御工
c .応用工学分野 特定の機械あるいは機械システムの種類ご
とに固有な技術を扱う工学分野である．対象となる機械の種類に
は，産業機械・装置，流体機械，熱原動機，熱機器，交通機械，ロ
ボット，情報機器，医療・福祉機器，資源・環境システム，宇宙機
d.関連工学分野 それ自身は機械工学には属さないが，機械
気・電子工学，管理工学，化学工学などがこれに当たる．生物工学
を含めることもある．

規範を与える学問・知識である．技術史・工学史，技術論（技術哲
学，技術社会学），技術倫理，法工学（関連諸法規，知的財産権）
