# 6・2・1 振動系を構成する要素

> **章**: 第6章 線形系の振動
> **節**: 6・2 1自由度系の振動

1自由度の振動系には、図6・1に示すように、質量とばねおよび減衰で構成される直線振動系のほか、ねじり振動系および抵抗、コイル、コンデンサによって構成される電気振動系などがある。図6・1(a)において、物体が平衡位置を中心として鉛直方向に振動するものとする。実際には理想的な保存系は存在せず、すべての振動系にはなんらかのエネルギーの損失がある。物体の運動の速度に比例する減衰力が作用する場合を粘性減衰(viscous damping)という。ねじり振動系や電気振動系も同様の運動方程式に支配される。それらの振動系の変数の対応を表6・1に示す。
