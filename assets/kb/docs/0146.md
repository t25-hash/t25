# 6・2・3 強制振動

> **章**: 第6章 線形系の振動
> **節**: 6・2 1自由度系の振動

ここでは加振力の振幅、ωは角振動数である。その一般解は右辺を零とした自由振動の解と、強制力が作用する場合の特解の和で与えられるが、時間が経過するにつれて自由振動はやがて減衰し、加振力による定常的な強制振動(forced vibration)だけが残る。は強制振動の振幅、φは位相角で、この角度だけ応答変位が加振力の位相より遅れていることになる。図6・3からわかるように、加振力の振動数が系の固有振動数に比べて小さいとき振幅倍率は1に近く、振動の振幅はほぼ静たわみに図6・2 減衰振動図6・3 正弦加振力による粘性減衰系の振幅倍率曲線と位相曲線等しい。加振力の振動数が固有振動数に近づくと、系の減衰が小さいとき振幅倍率は非常に大きくなる。この現象を共振(reso-nance)と呼び、振幅倍率がピーク値となる振動数を共振点と呼ぶ。特に不減衰系ではω=ωが共振点で、そのときの振幅倍率は無限大となる。減衰系に正弦力を加えて振動させ、共振点における振幅を測定することができればζの値が求まり、これより系の減衰係数の値を知ることができる。加振振動数が共振点を超えると、振動数の増加につれて振幅は小さくなってくる。位相角に関しては、振動数比が小さいときはφ<90°で、変位は加振力と同位相であるのに対し、振動数比が大きくなって1を超えるとφ>90°で、互いに逆位相となる。ωω=1では減衰の大きさに関係なく位相角は90°で、減衰の小さい振動系ほどωω=1の近傍で位相角が急激に変化する。
