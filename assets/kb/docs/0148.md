# 6・2・5 等価粘性減衰

> **章**: 第6章 線形系の振動
> **節**: 6・2 1自由度系の振動

機械や構造物の減衰は一般に複雑な現象である。多くの場合、互いに接触する固体面の摩擦、流体の抵抗、不完全弾性体内の応力によるもの、電磁力によるものなどがおもな原因となっている。一般にエネルギーの消費のある系では必ず減衰力が作用する。減衰力の大きさは系の運動状態によって変わるほか、流体の粘性が温度変化に敏感であるなど環境条件による影響が大きい。粘性減衰系に正弦加振力cos ωが働く場合、定常状態の変位は=cos ω-φで、1サイクルの間に加振力によってなされる仕事は、=cos ω=cos ω=-ωcos ωsinω-φ=πsinφ(6・31)である。力と変位の間の位相角が0°か180°のときはこの仕事は零で、系にエネルギーの流入はないが、位相角が90°のとき、加振力による仕事は最大となる。一方、減衰力による1サイクルの間の消費エネルギーは、===ωsinω-φ=πω(6・32)のように振幅の二乗に比例する。この二つのエネルギーを等しいとおき、sinφ=2ζω욈ωω욈-ω+2ζω욈ωであることを考慮すれば、振幅が求められる。すなわち、=sinφω=욦욧1-ωω욈+2ζωω욈(6・33)となり、この結果は式(6・24)と一致する。他の形の減衰力が働くときはこのような簡単な式は成立せず、その運動は必ずしも正弦的とはならない。しかし減衰の形に関係なく定常な単振動を仮定し、系の減衰力により1サイクルの間になされた仕事を求め、これを式(6・32)と等しくおくことによって、等価な粘性減衰係数욄욤におきかえて近似することができる。固体摩擦(クーロン減衰力(Coulomb damping))では運動の方向と反対に、変位や速度に無関係な一定の力が働く。この場合の運動方程式は、+±=cos ω(6・34)となり、これは非線形系の強制振動である。この方程式の厳密な解はフーリエ級数を用いて得られているが、それはかなり複雑である。ここでは等価粘性減衰(equivalent viscous damping)による近似解を示す。定常振動における1サイクルの運動の変位は4であるから、消費エネルギー4と式(6・32)を等しくおくことによって、等価粘性減衰係数욄욤=4πωが得られる。ただし욄욤は一定値ではなく、振動数と振幅に逆比例する。この場合の定常振動の振幅は式(6・24)でζ=욄욤욂とおくことによって、욦욧=11-ωω욈+4πと求まる。これを解くと、=욦욧1-4π1-ωω욈(6・35)が得られる。式(6・35)は摩擦力が正弦的な加振力の大きさに対して>π4のときにのみ実数値をとり、このとき定常振動となる。一方、<π4のとき1サイクルのエネルギー消費量4に比べて流入量πのほうが大きくなり、振幅は無限大となる。厳密解による振幅倍率と振動数比の関係は図6・5のようになる。図中の破線より下の条件では、質量が最大振幅点で一定時間停止するスティック・スリップ運動となるが、破線の上では質量は滑らかに振動する。2―38機械力学
