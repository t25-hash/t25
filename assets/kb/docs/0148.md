# 6・2・5 等価粘性減衰

> **章**: 第6章 線形系の振動
> **節**: 6・2 1自由度系の振動

機械や構造物の減衰は一般に複雑な現象である。多くの場合、互いに接触する固体面の摩擦、流体の抵抗、不完全弾性体内の応力によるもの、電磁力によるものなどがおもな原因となっている。一般にエネルギーの消費のある系では必ず減衰力が作用する。減衰力の大きさは系の運動状態によって変わるほか、流体の粘性が温度変化に敏感であるなど環境条件による影響が大きい。力と変位の間の位相角が0°か180°のときはこの仕事は零で、系にエネルギーの流入はないが、位相角が90°のとき、加振力による仕事は最大となる。他の形の減衰力が働くときはこのような簡単な式は成立せず、その運動は必ずしも正弦的とはならない。しかし減衰の形に関係なく定常な単振動を仮定し、系の減衰力により1サイクルの間になされた仕事を求め、これを式(6・32)と等しくおくことによって、等価な粘性減衰係数におきかえて近似することができる。固体摩擦(クーロン減衰力(Coulomb damping))では運動の方向と反対に、変位や速度に無関係な一定の力が働く。この方程式の厳密な解はフーリエ級数を用いて得られているが、それはかなり複雑である。ここでは等価粘性減衰(equivalent viscous damping)による近似解を示す。式(6・35)は摩擦力が正弦的な加振力の大きさに対して>π4のときにのみ実数値をとり、このとき定常振動となる。厳密解による振幅倍率と振動数比の関係は図6・5のようになる。図中の破線より下の条件では、質量が最大振幅点で一定時間停止するスティック・スリップ運動となるが、破線の上では質量は滑らかに振動する。
