# 1・2・8 流体工学（fluids engineering）の誕生と発展

> **章**: 第1章 機械工学概説
> **節**: 1・2 機械工学史（history of mechanical engineering）通

流体工学とは，流れに関する学問と，管路・噴流・翼列など実際
の流体機器に関わる技術的・工学的知識の体系をいう．歴史的には
水路・管路・水力機械に現れる現象を扱う経験的色彩の濃い水力学
と，流体の運動を数理物理学的に解析する理論的色彩の濃い流体力
学，およびその応用に大別される．
a.水力学（すいりきがく，hydraulics ) 流体の知識と学問は
古代にまでさかのぼる．静止流体については古代のアルキメデス
［Archimedes(of Syracuse)］からルネサンス期のステヴィン
（Stevin,S. ）を経て17世紀のパスカル（Pascal,B. ）でほぼ完成す
る．17～18世紀のヨーロッパは大土木時代となって，ポンプと動
力水車が先端機械として重用された．技術者の間では流れる水の性
質と挙動に関心が高まり，18世紀の中ごろまでに水力学の基礎が
できあがった（1･2･4･ c項参照）．19世紀に入ると新型の水力機械
や水力タービンが続々と現れ，管路や水力機械に関してイギリスの
スミートン（Smeaton,J. ），フランスのマリオット（Mariotte,E. ）
やパラン（Parent,A. ）らは実用的な研究をさらに進めた．マリオ
ットには『水・流体の運動理論』（1686年）という有名な著書があ
る．
b.流体力学（fluid dynamics ）・応用流体力学 流れの解析的
扱いはニュートン（Newton,I. ）が『プリンキピア』（1687年）第
2編に示した粘性法則に始まった．18世紀の数学者ダニエル・ベル
ヌーイ（Bernoulli,D. ），オイラー（Euler,L. ），ダランベール
（D’Alembert,J.le Rond），ラグランジュ（Lagrange,J.L. ）らは
当時最新の微積分学を駆使して理論的な流体力学の基礎を築いた．
ベルヌーイの定理（1738年），オイラーの基礎方程式（1755年）は
特に有名である．ただし，ここで扱われたのは粘性も圧縮性もなく
渦もない，完全流体であった．
19世紀に入ると水力機械はいっそう発展し，大型蒸気船も出現
したことで流れの力学は新しい段階を迎えた．流れを全体として水
力学的にとらえるのではなく，乱流とか渦とかの現象ごとに分けて
解析的に扱う応用流体力学の出現である．
実在流体が研究の対象となり，19世紀の半ば，イギリスの数学
者ストークス（Stokes,G.G. ）は粘性流体の運動を研究し，フラン
スの土木技師ナビエ（Navier,C.L.M.H. ）が先に行った研究を補
完して1845年，ベルヌーイの式に粘性項を加えたナビエ・ストー
クスの方程式（N- S方程式）を明らかにした．これは実在流体の
基礎方程式として名高い．イギリスのレイノルズ（Reynolds,O. ）
も粘性流の研究を行い，1883年には流れの力学的相似性を与え，
かつ層流と乱流を区別する指標としてレイノルズ数を示した（図
1･12)．滑り軸受の潤滑理論を打立てたことも彼の大きな功績であ
る（1･2･5･ b項参照）．これはトライボロジー（表面・摩擦工学）
という学問の始まりともいえる出来事である．
