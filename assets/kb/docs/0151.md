# 1・2・9 熱工学（thermal engineering）の誕生と発展

> **章**: 第1章 機械工学概説
> **節**: 1・2 機械工学史（history of mechanical engineering）通

熱工学とは，熱に関する学問と熱関連機器に関わる技術的・工学
的知識の体系をいう．熱工学で扱われる分野は熱力学を中心とし
て，熱物性・伝熱・燃焼・エネルギー変換などを含む．
a.熱力学（thermodynamics ) 熱力学は材料力学や流体力学
に比べると新しい学問である．熱の本性は古くから論じられてきた
が，熱力学という学問は19世紀初め，2人のフランス人の研究に
始まる（1･2･4･ d項参照）．1人は軍人技術者フーリエ（Fourier,J.
B.J. ）である．彼は大砲の安全性を求めて金属内の熱伝導を解析
し，研究成果を1822年に発表した．フーリエ級数はこの研究の中
から生まれた．もう1人はエコル・ポリテクニク出身の若手軍人サ
ディ・カルノー（Carnot,N.L.S. ）である．彼は蒸気機関が発生
しうる最大動力は何によって決まるかの理由を求めて研究を行い，
熱落差と熱サイクルの概念を創出して1824年に熱機関の動作原理
を明らかにした．熱それ自身の性質についてはこのあと研究が進め
られ，マイアー（Mayer,J.R. ）によるエネルギー保存則の確立
（1842年），ジュール（Joule,J.P. ）による熱の仕事当量の決定
（1843年），ケルヴィン（Kelvin,Lord）の絶対温度概念（1848
年），ランキン（Rankine,W.J.M. ）による蒸気の性質の研究
（1849年）と続く．熱の流れの非可逆性（つねに高温から低温へと
向かう）についてはクラウジウス（Clausius,R.J.E. ）が熱の状態
量としてエントロピー概念を提唱してエントロピー増大の法則を明
らかにし（1865年），ボルツマン（Boltzmann,L. ）がエントロピ
ー増大の法則に解析的証明を与えて（1872年）この問題に決着を
与えた．
b.熱原動機（heat engine）と熱工学の並行的発展 19世紀
末近くには蒸気機関に代わる新しい熱機関が現れた．蒸気タービン
（1884年），4サイクルガス機関（1887年），ディーゼル機関（1897
年）などである．リンデ（Linde,K. ）の冷凍機は1875年に現れ
た．これらの熱機関と熱機器の発明は熱工学の研究を生み出す刺激
となったし，工学研究の成果でもあった．技術の発展と工学研究は
表裏をなして並行的に展開したのである．ランキンの著書『蒸気機
関ならびに他の原動機』（1859年）は熱力学という学問が蒸気機関
にいかなる意味をもつかを明らかにした名著であって，ランキンサ
イクルの考えはこの書物の中に示されている（図1･14)．
熱関係の工学技術が発展していく中で異種エネルギー間の変換方
法とその効果について研究が行われ，いまではエネルギー変換とい
う，熱工学の重要な一分野が形づくられるに至っている．
c .熱物性（thermophysical property) 熱機関の作動流体の
熱物性データは機関を設計するうえで基礎的な情報である．ランキ
ンは1849年に蒸気の性質を研究した．19世紀末に蒸気タービンが
現れると水蒸気の熱的性質を知る必要はますます高まった．リンデ
の弟子モリエ（Mollier,R. ）は熱力学を機械に応用しようとして，
1892年にいくつかの蒸気線図を考案した．有名なモリエ線図はこ
の中にある．1920年ごろになると先進工業国ではエネルギー需要
が急増し，蒸気動力プラント（火力発電所）ではより高温・高圧の
蒸気が用いられるようになった．先進各国では関係学会が信頼でき
る蒸気データ集（蒸気表）を作成してこれに対処した．高温水蒸気
の熱物性の本格的研究は日本では1960年ごろから始められ，今で
は日本が世界をリードするようになっている．
d.伝熱工学（heat transfer engineering) 伝熱工学は熱と物
質の移動を扱う技術の学問である．熱伝導ではフーリエの古典的研
究（1822年）があるが，20世紀になって熱動力プラントの高性能
化が本格的な伝熱研究を促進した．1910年代から1920年代にかけ
て熱伝達の研究が進み，ドイツのヌセルト（Nusselt,E.K.W. ）は
流体中の熱流を研究して熱伝達の基礎法則を示し，1930年には熱
伝達と質量伝達の相似性を明らかにした．最近では原子力プラント
の出現と宇宙工学の進展で伝熱工学の重要性がいっそう増してい
る．沸騰熱伝達では東北大学の抜山四郎（ぬきやま・しろう，
Nukiyama,S. ）の研究（1934年）が先駆的なものであるが，最近
では原子炉の熱除去に関して必要性が生じ，1950年代から研究が
活発になった．ボイラの水循環に関しては気液二相流（混相流）の
研究があり，1970年ごろからは超臨界状態の伝熱現象も新しい研
究課題となった．
伝熱工学の分野でも現在ではコンピュータによる数値解析が広く
行われ，現象の研究や機器の設計に利用されている．非線形や非定
常の複雑な伝熱現象も，いまでは計算力学の発達によって扱うこと
ができるようになった．
e .燃焼工学（combustion engineering) ものが燃えるとい
うことの科学的研究は，古くはルシャトリエの火炎伝ぱの理論
（1883年）があるが，本格的な研究はやはり熱動力プラントの普及
と関係があり，20世紀になってからである．燃焼に関する技術的
な関心の高まりの中で1928年，初の工学的な国際燃焼シンポジウ
ムが開かれた．燃焼の研究が大きく発展するのは第二次世界大戦後
である．ジェットエンジンやロケットの発明によって，燃料の技術
開発とこれを支える基礎研究が求められたからである．燃焼を反応
ガス力学として広義に扱う近代的な燃焼学はここで始まった．1970
年代以降は燃焼排ガスによる大気汚染への対処が切実になり，燃焼
の研究はいっそう促進された．
最近ではレーザ応用計測の進歩によって燃焼現象の測定が可能と
なり，コンピュータ利用の数値解析も容易になって，燃焼の研究は
新しい段階を迎えている．
