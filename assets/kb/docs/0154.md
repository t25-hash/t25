# 1・3・3 これからの機械工学が進む方向

> **章**: 第1章 機械工学概説
> **節**: 1・3 技術と工学

日本では1965年ごろまでは，大学の組織と産業の分野とが完全
に1対1に対応していた．つまり大学の組織図を見れば，産業の縮
図が読み取れたのである．しかし，上述したように，民間では，機
械工業に電気や情報の産業が融合して新たに強い組織が生まれてき
たのに，大学や学会では，機械工学と電気工学，情報工学，生命工
学，医療福祉工学などとが互いにマージすることはなかった．例え
ば，マイクロマシンはマシンが付くから機械学会だろうと思うと大
間違いで，これは電気学会に論文を提出しなければならない．
ますます，機械工学の柱である力学の知識体系だけでモデル化で
きる新製品が減り，その分，力学と何か他の知識とを組合わせて創
出するビジネスが増えてくるだろう．例えば，力学を再生医学で作
った細胞の運動の記述に使い，障害者の機能再生に役立てることが
挙げられる．しかし，その要求に対して，自分は高校で生物学を勉
強していないと言ったり，生体物は本質的に金属とは異なるまたは
バイオの研究者とはそもそも文化が違うと言って機械工学者が逃げ
ていては，その社会的使命を終えたと思われてもしかたがない．
幸いにして，若い学生やエンジニアは，戦後50年間の古い世代
が背負ってきた会社人間思考から脱皮し，自分の生き方を世間の評
価を気にせずに打立てられるようになってきた．もちろん父親の会
社人間世代がリストラで早期退職していく時代背景を考えると，ま
ずは「自分流」で生きたいと願うことも理解できる．しかし，新産
業創出に挑戦する気概はこれまでの人材にない性格である．後は，
それに投資するという民間の金持ちのエクイティマインドだけが必
要であるが，それに至るには自分の意志というよりも時代の勢いで
投機に失敗した，というバブル期の痛手がいえるまで時間がかかる
だろう．当分は，引続き，国が援助していく必要があるかもしれな
い．
このように，機械工学のまわりには融合化を期待するビジネスが
生まれ，機械工学は他の工学文化との融合だけでなく，医学や理
学，生物学，経済学などとの融合が必要となるだろう．また，融合
だけでなく，人間の思考にまで新たに手を伸ばして，設計の方法論
や知識の活用論にも力を入れ，多くのプロデューサを育てる必要が
あるだろう．このような方向に向かってエンジニアが気概をもって
挑戦すれば，日本の製造文化がひと皮むけて活性化し，その結果，
高コストで過密なうえに原料，食料をもたない日本でも，世界と共
生しながら繁栄を続けることができると考える．
