# 6・3・4 モード解析(modal analysis)

> **章**: 第6章 線形系の振動
> **節**: 6・3 多自由度系の振動

モード解析は、振動系の質量、剛性、減衰などがわかっているときに、外力に対する応答を固有モード(固有関数)展開によって求める解析手法である。一方で、外力に対する振動系の応答が実験的にわかっているときに、振動系の固有振動数、固有振動モード、減衰比などを求めるための解析手法を実験モード解析という。実験モード解析は、モード解析の理論に基づくモードパラメータの実験的同定であり、有限要素モデルの改良や構造変更シミュレーションなどに有用である。はじめにモード解析について述べる。次にモード行列を用いて一般座標からモード座標に変数変換し、モード座標による、連成項のない、自由度の数だけの1自由度振動系の運動方程式を得る。これらの運動方程式を解き、その結果を一般座標に戻すと、6・3・2項の最後に示したような外力と応答の関係が得られる。一般座標とモード座標の間の変換の関係式からわかるように、モード座標ξの値は、一般座標における応答のうちの次モードの大きさに反映されるから、モード座標に変換して解く方法は固有振動モードごとの応答を求めていることになる。そこで6・3・2項に述べたような応答解析方法をモード解析と呼んでいる。このモード解析を行うためには、モード行列を求めることが必要であるが、実際にはすべての固有振動モードを考える必要性は少なく、一般には低い次数の少数の固有振動モードを考えれば十分なことが多い。次に実験モード解析について述べる。構造物は本来は連続体であるが、例えば構造物上に3個の点を取り、各点で直交三方向の変位を考えることにすれば、連続体を自由度振動系で置換えたことになる。伝達関数は角振動数ωの関数としてボード線図やナイキスト線図の形で表すことができる。これは、理論モデルを実験的に得られた線図にあてはめることであるから、曲線適合(curve fit)と呼ばれる。実際上は、伝達関数の実験値が得られている角振動数の範囲に、自由度振動系のすべての固有振動が含まれているわけではない。以上の関係を用いると、伝達関数は次のように表せる。固有振動ごとのパラメータの決定にはいくつかの方法があるが、一般には最小二乗法と繰返し収束計算が用いられる。例えば、まず実験結果のボード線図などを見て、ω~ωの範囲に存在する共振の個数から固有振動の個数を決める。また、減衰が小さいとき共振振動数は固有角振動数にほぼ等しく、周波数応答曲線は共振点付近で共振振動数に関してほぼ対称となる。振動系に副振動系を取付けると、主振動系の振動が緩和されることがある。この目的で付加される副振動系を動吸振器(dynamicvibration absorber)という。動吸振器の特性は、制振の目的と使用条件に応じて設定される。以下では動吸振器の例と調整方法について説明する。
