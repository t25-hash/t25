# 1・2・3 流体の状態を表す量

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

a.密度，圧力，温度の定義 流体の巨視的な状態を表す物理
量（分子の集合体としての統計的性質）は状態量と呼ばれ，密度，
圧力，温度などがある．
密度（density）は，その流体が占める空間の単位体積当たりの
質量（kg/m ）で定義され，流体を構成する分子の重さと単位体積
中に含まれる分子の数によって決まる量である．
圧力（pressure）は，流体内の任意の面に垂直に作用する単位面
積当たりの力（N/ m ，Pa）である．分子間隔が小さい液体では，
圧力は主として分子間力を介して作用する．しかし，分子間隔が大
きい気体の場合，圧力は単位時間に単位面積を通過する（または単
位面積の固体表面に衝突する）分子の運動量に等しく，分子の重
さ，個数，速度に関係する．このことは，運動量の時間的変化が力
と等価であるというニュートンの運動の法則を考えると理解でき
る．分子が存在しない完全真空の状態を絶対圧力が零と定義すれ
ば，内部エネルギーが存在する限り分子は運動するので，負の絶対
圧力はあり得ない．
温度（temperature）は，上述の内部エネルギーに関係する量で
ある．同じ物質が長時間接していると，分子の振動的運動または不
規則運動を介して両者の単位質量当たりの内部エネルギーは等しく
なり熱的に平衡状態に達するが，異なる物質が接しているときの平
衡状態では二つの物質の内部エネルギーは等しくない．物質が異な
っても熱的平衡状態にあれば等しくなる量として温度が定義され，
温度差は平衡状態からのずれを表す尺度である．すなわち，互いに
接する物質の内部エネルギーは温度が異なると物質間を移動し，等
しいときは移動しない．温度は，標準大気圧（1atm＝0.1013
MPa）における水の融点と沸点の間を100等分した尺度で計測さ
れ，水の融点を基準にした温度を摂氏温度（0℃），同じ尺度で内部
エネルギーが零の状態を基準にした温度を絶対温度（K）という．
温度差1℃（1K）に対応する内部エネルギーの差は物質によって
異なる（1･3･5項参照）．
状態量には圧力，温度，密度のほかに，単位質量当たりの流体の
もつ内部エネルギー（J/ kg，m / s ），エンタルピー（J / kg，m /
s ），エントロピー［J/( kgK)］などの熱力学的状態量がある（α5
編「熱工学」参照）．
b.相変化と状態曲面 1種類の分子からなる（1成分系の）
気体から圧力一定の条件で熱を除去すると温度が下がり，ある温度
に到達すると局所的に液化が始まるが，気相と液相が共存する領域
では除去熱は潜熱に取られて温度は変化しない．この温度を飽和温
度（saturation temperature）といい，このときの圧力をその温度
に対する飽和圧力（saturation pressure）という．この状態からさ
らに熱を除去すると液相のみの状態となり，液相と固相が共存する
領域を経て固体になる．気相と液相が共存しうる圧力と温度の領域
は限られ，その最大値を臨界圧力（critical pressure）および臨界
温度（critical temperature）という．臨界圧力より高い状態で熱
を除去すると密度が滑らかに変化して液体になる．

相と固相が同時に共存するときがある．この圧力と温度を三重点と
