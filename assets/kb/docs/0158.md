# 6・4・2 粘性減衰型動吸振器

> **章**: 第6章 線形系の振動
> **節**: 6・13 )。そこで、外力の角振動数ωが一定している場合は、動吸振

a. 定点による調整動吸振器の減衰比だけを変えるとき、共振曲線は二つの定点を通るため、これをP,Qとする(図6・15)。定点の高さは最大振幅倍率の下限であるが、P,Qを同時に下げることはできないから、高さが一致する条件から固有振動数比を決める。この調整方法によると共振曲線はP,2―46機械力学Qの付近でほぼ極大となり、図6・16の実線のように最大振幅倍率が近似的に最小化されることが知られている。上記の調整方法は広く用いられている。安定度が最大のとき、特性方程式は重根をもつ。主振動系が非減衰系とするとき、同調比は定点による調整と等しい。定点による調整と異なり共振点は1個であり、最大振幅倍率は増加してしまう(図6・16の破線)。
