# 1・2・5 気液界面における蒸発・凝縮と溶解

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

液体と気体の界面では絶えず液体の分子は気体中に飛び出し，気
体分子は液体中に飛び込んでいる．液体と同じ分子の気体（蒸気）
が界面を介して往来する現象において，気体中に飛び出す分子のほ
うが多いときが蒸発（evaporation），少ないときが凝縮（conden-
sation）である．飛び出す分子の数と飛び込む分子の数が等しくな
ると平衡状態に達する．蒸発か凝縮かは気体中に含まれる蒸気の分
子数の割合による．気体中における蒸気の分子数の割合は，同じ体
積を蒸気分子が単独に占めるときの分圧（partial pressure）に比
例する．蒸気の分圧を蒸気圧（vapor pressure）といい，平衡状態
に達したときの蒸気圧を飽和蒸気圧（saturated vapor pressure）
という．すなわち，蒸発現象は飽和蒸気圧より蒸気圧が低いときに
起こる．飽和蒸気圧は温度の関数であり，液体温度の上昇とともに
高くなる．
