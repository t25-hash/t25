# 6・4・4 固体摩擦式ダンパ

> **章**: 第6章 線形系の振動

図6・17に示す固体摩擦式ダンパ(ランチェスタダンパ)は非線型振動系であり,周期的外力の角振動数ωの値によって,主振動系とダンパとの間に相対的滑りのある場合と,主振動体とダンパが一体になって動く場合がある.このような系の共振特性を求めることは困難であるが,主振動体とダンパとの間につねに相対的な滑りがあり,両者の間に摩擦力が作用するときの運動方程式はに対して+±=cos ω-α, 욃=となる.摩擦力をフーリエ級数に展開してこの場合の強制振動解を求め,固体摩擦ダンパと粘性減衰型動吸振器を比較すると,固体摩擦ダンパでは,ダンパがありながら振幅が非常に大きくなることがある.最良の摩擦力は外力の大きさに関係し,最良の調整を行ったとしても最大振幅は粘性減衰型動吸振器より大きい.すなわち,性能的に粘性減衰型に劣る.文献(1) Den Hartog,J.P.,Trans.
