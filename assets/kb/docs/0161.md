# 1・3・1 密度

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・3 流体の物性

液体に含まれる気体の溶解度が低いと，飛び込む気体分子の数が外
部に飛び出る分子の数より多く溶解が進行する．液体中の気体分子
が増えるにつれて界面で気体中に戻る分子が増加し，飛び出す分子
の数と飛び込む分子の数が等しくなると平衡状態に達する．平衡状
の水の密度に対する密度の比で定義される．
態になる溶解度を飽和溶解度（saturated solubility）という．飽
和溶解度は圧力と温度の関数であり，一般に気体の圧力にほぼ比例
して増加し，また液体の温度が低いほど大きい．
