# 7・2・1 項で述べた単位インパルス応答関数が得られると、

> **章**: 第7章 過渡応答・衝撃
> **節**: 7・2 1自由度系の衝撃応答

任意形状の外力を受ける振動系の過渡応答を求めることができる。図7・4に示すような外力を微小な時間幅Δτに分割して、各時間のインパルスによって発生する応答を重ね合わせることで線形系の応答を求める。=τにおける1個のインパルスを取上げると、その力積はτΔτであるので、その後の時間に及ぼす寄与は経過時間-τに依存し、-ττΔτで表すことができる。これらを重ね合わせて任意の外力に対する過渡応答は、=-τττ(7・9)なる積分で表現することができる。この積分を畳み込み積分(con-volution integral)という。
