# 7・2・4 理想衝撃パルスによる過渡応答

> **章**: 第7章 過渡応答・衝撃
> **節**: 7・2 1自由度系の衝撃応答

理想衝撃パルスに関してはISO 2041に定められており、そのおもなものを表7・1に示す。これらの理想衝撃パルスの場合、パルス形状が簡単な関数で与えられるので、その応答も比較的容易に求めることができる。いずれの場合にも、(I)衝撃パルスが作用している時間区間と、(II)その後の自由振動を行う時間区間とに分けて考える必要がある。したがって、=-なる変数変換を行うと、図7・6 pulse)直線的に最大値まで増加し、その後瞬時に零となる三角波形の時刻歴をもつ理想衝撃パルス。逆のこぎり波衝撃パルス(initialpeak sawtooth shock pulse)瞬時に最大値に達し、その後直線的に減衰して零となる時刻歴をもつ理想衝撃パルス。三角波衝撃パルス(symmetricaltriangular shock pulse)時間履歴曲線が二等辺三角形の時刻歴をもつ理想衝撃パルス。バーサイン衝撃パルス(versineshock pulse, haversine shockpulse)零から始まるバーサイン曲線(サイン二乗曲線)の1サイクルの時刻歴をもつ理想衝撃パルス。方形波衝撃パルス(rectangularshock pulse)時刻歴が瞬時にある値に達し、パルスが持続している間一定の値をとり、瞬時に零となる理想衝撃パルス。台形波衝撃パルス(trapezoidalshock pulse)時刻歴が直線的にある値まで増加し、その後ある時間内は一定の値を取り、その後直線的に減少して零となる理想衝撃パルス。変位の最大値を、インパルス力の最大値が静的に作用しているときの変位で割った値を動的増幅係数(dynamic amplification factor)と呼ぶ。まず、ω<ωにおける最大応答値について考える。ω≠ωの場合には、式(7・13)を時間に関して微分して零とおき、最大値をとる時刻を求める。一方、ω>ωのときは最大応答は自由振動領域で生じる。このとき、初期変位と初期速度はω=πを式(7・13)に代入して得られる。b. 方形波衝撃パルスの場合図7・8に示すような長方形形状のパルス入力に対する応答を調べる。前述のように二つの時間区間に分けて考える。図7・9は正弦半波衝撃パルス、方形波衝撃パルスを入力したときの動的増幅係数を求め、パルス長さ比に対して描いたものである。このグラフは7・4節で述べる衝撃スペクトルの表現に対応するものである。図7・10は粘性減衰を有する振動系のステップ応答の典型的な例である。この場合、減衰比ζと減衰固有円振動数ωは次式により推定できる。このとき、周期やの値として、単一波形からとりうる種々の組合せの平均値を用いれば、推定値の精度は向上する。このほかに動特性の推定方法として、周波数応答関数の曲線適合による方法などが知られている。衝撃の問題においては、フーリエスペクトルの概念のほかに衝撃スペクトル(shock spectrum)が用いられることがある。これは一連の1自由度系に一定の衝撃が作用するときの応答の最大値を、系の固有振動数または固有周期に対する関係として表した曲線である。一般に振動系は不減衰系として扱われることが多いが、粘性減衰をもつ系についても減衰比をパラメータにして表すこともできる。衝撃スペクトルは変位、速度、加速度それぞれに対して定義することができる。系の減衰が小さいときはそれぞれ次の近似式で与えられる。したがって、この近似式を用いれば角振動数ωが与えられたときに、縦軸にlog ωをとり、横軸には式(7・21)の条件を満足するように対数で座標軸をとれば、図7・12に示すような関係が得られる。このようなグラフを三重衝撃スペクトル(tripartiteshock spectrum)と呼ぶ。地震工学の分野では応答スペクトルに関して同様な取扱いを行い、三重応答スペクトルと呼ばれている。衝撃スペクトルには使用目的に応じて次のようなものがある。(1) 初期衝撃スペクトル:衝撃パルスの作用時間中に発生する変位、速度、加速度の最大応答を連ねたスペクトルであり、最も一般的に用いられる。(2) 残留衝撃スペクトル:衝撃作用が終了した後の変位、速度、加速度の最大値に対応するスペクトルで、衝撃測定用振動計などのように励振後に静止復帰が要求される場合に利用される。(3) 最大ひずみ衝撃スペクトル:最大変位に対応した衝撃スペクトルで、最大ひずみは最大応力に対応するため設計上重要である。図7・13は代表的な衝撃パルスの初期衝撃スペクトルと残留衝撃スペクトルを求めた例である。
