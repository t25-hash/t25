# 8・1・1 線形系と非線形系

> **章**: 第8章 非線形振動
> **節**: 8・1 非線形振動の概説

機械や構造物の振動を理論的に解析するのに、対象物の運動方程式を用いる。この運動方程式に基づいて、対象物を線形系と非線形系に分けることができる。簡単な例で線形系、非線形系の意味を考えよう。例として1自由度ばね質量系を取上げる。得られた式は線形方程式である。この例のように、線形方程式で支配される系を線形系(linear system)という。得られた式は非線形方程式である。この例のように、非線形方程式で支配される系を非線形系(nonlinear system)という。対象物を非線形系でモデル化する必要があるかどうかを決定するのに、非線形振動の特徴を知っておくことが重要である。
