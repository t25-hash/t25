# 8・1・1 線形系と非線形系

> **章**: 第8章 非線形振動
> **節**: 8・1 非線形振動の概説

機械や構造物の振動を理論的に解析するのに、対象物の運動方程式を用いる。この運動方程式に基づいて、対象物を線形系と非線形系に分けることができる。簡単な例で線形系、非線形系の意味を考えよう。例として1自由度ばね質量系を取上げる。質量を、質点の変位をとおく。対象とする系の復元力、減衰力がそれぞれ変位、速度に比例すると考えることができるとき、,を定数として、運動方程式は、++=0(8・1)となる。得られた式は線形方程式である。この例のように、線形方程式で支配される系を線形系(linear system)という。実在の対象物では、復元力や減衰力は厳密には変位や速度に比例せず、,の高次の式となる。復元力と減衰力を合わせて,とおくと、運動方程式は、+,=0(8・2)となる。得られた式は非線形方程式である。この例のように、非線形方程式で支配される系を非線形系(nonlinear system)という。対象物を非線形系でモデル化する必要があるかどうかを決定するのに、非線形振動の特徴を知っておくことが重要である。
