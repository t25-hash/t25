# 8・1・2 非線形要素

> **章**: 第8章 非線形振動
> **節**: 8・1 非線形振動の概説

機械や構造物を非線形系にする主要な要素をここで列挙する.a. 材料非線形性対象とする機械・機器・構造物が変形させられたとき,その復元力は変位に比例すると仮定される場合が多い.多くの材料,特に金属材料について,変位が小さい場合はこの仮定は精度よく成り立つ.しかしゴムやコンクリートのような材料では,復元力と変位は比例しない.これらの材料から作られる対象物をモデル化すると,復元力の特性に起因して対象物は非線形系となる.この非線形性を材料非線形性という.b. 幾何学的非線形性例として両端を軸方向に拘束されたはりを考える.このはりは,小さい変位に対しては,曲げに起因する復元力だけを考慮すればよく,材料特性が線形であれば,線形系としてモデル化できる.しかし変位が大きくなると,小さい変位に対して無視できた軸方向の張力が大きくなり,材料特性が線形であっても,はり全体としては非線形系となる.このように,大きい変位に起因して引き起こされる非線形性を幾何学的非線形性という.c. 非線形減衰力運動している対象物が受ける減衰力は種々の特性をもつ.例えば粘性の低い流体に起因する減衰力では,減衰力は速度にほぼ比例すると考えることができる.この減衰力は線形要素である.しかし粘性の高い液体に起因する減衰力は非線形要素となる.クーロン摩擦も非線形減衰力の例である.また変位が小さい間は減衰力と逆に運動を促進する作用をし,変位が大きくなると通常の減衰力に転じる,ファン・デア・ポール系に見られる減衰力も非線形減衰力の例である.d. がた・あそび・リミッタ機械にはがたやあそびが存在し,必要に応じてリミッタが設けられる.これらの要素が対象物に存在するとき,復元力が不連続に変化する.これらは非線形要素である.e. 流体関連力対象物が流体内に置かれたとき,対象物は流体力を受ける.流体力は対象物の変位と相互作用して一般に複雑な特性を示すので,流体力は非線形要素となることが多い.f. 非線形境界条件機械や構造物の境界部は,実際の問題では,固定支持や単純支持のような簡単なものではない.境界部は復元力と減衰力を生じ,これらが非線形性をもつことが多い.この場合,対象物は全体として非線形系となる.2―53g. 電磁力対象物が電場,磁場に置かれたとき,それらは電磁力を受ける.この電磁力は,流体関連力と同じように非線形性を示し,電磁力が関連する系は,多くの場合,非線形系である.
