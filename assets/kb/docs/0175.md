# 1・2・2 磁性体（magnetics ）

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 物質と電磁界

物質のもつ磁気的性質や磁気現象から見るとき，その物質を磁性
体と呼び，磁界の影響で磁気双極子モーメントをもつ現象を物質の
磁化（magnetization）という．外部から磁界をかけると磁性体の
内部の微視的磁気モーメントを強める方向になるものを常磁性体
（paramagnetic substance）といい，外部磁界を打消すものを反磁
性体（diamagnetic substance）という．常磁性体には白金，アル
ミニウムなどがあり，反磁性体には銅，銀，水などがある．これに
対して，鉄，コバルト，ニッケルなどは大きな磁気モーメントを有
しており，強磁性体（ferromagnetic substance）と呼ばれる．こ
れは，磁区と呼ばれる領域内で磁気モーメントのもとになる電子ス
ピンが一方向に整列するためといわれている．外部の磁界がなくて
も磁化がもとのままで保持され，その方向に強い磁界を発生させる
ものが，磁石である．
磁性体は実用的に，変圧器，モータなどの磁心材料や磁石材料，
あるいは磁気記録やセンサなどに応用される特殊磁気応用材料とし
て重要である．

必要があることなどである．
