# 8・3・1 厳密解

> **章**: 第8章 非線形振動
> **節**: 8・3 解析手法と非線形振動特性

非線形系においては、ごくわずかな例外を除いて厳密解(exactsolution)を求めることはできない。以下ではその例外の中からいくつかの例を紹介する。a. 非線形保存系の自由振動次のような1自由度の非線形保存系(nonlinear conservative system)を考える。式(8・14)の両辺にを乗じてエネルギー積分を行うと、次式を得る。また、その周期および角振動数ωは、式(8・17)から次式で与えられる。以下にその結果のみを示す。表8・1に各ばね特性に対する厳密解とその周期および母数の決定方程式を示す。漸硬ばねおよび漸軟ばねの場合の自由振動解は、それぞれ8・3・5項の図8・16および図8・17中の背骨曲線に相当する。両図からわかるように、その周期は最大振幅が大きくなるにつれて漸硬ばねの場合には短くなり、漸軟ばねの場合には長くなる。この系では、各領域ごとに解析解が求められるので、それらを接合することによって厳密解が求められる。これを接合法(inosculating method)という。ここでは、いくつかの代表的な断片線形ばね特性に対して、1自由度ばね-質点系の不減衰自由振動の周期を示す。厳密解はそれらを接合すればよい。式(8・30)のように、クーロン摩擦が作用する系では最大振幅が直線的に(半周期ごとに2ずつ)減少するのが特徴である。. 漸硬ばね系支配方程式は次式で与えられる。
