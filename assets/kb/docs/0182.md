# 8・3・2 項で示した手法によれば、周期解に関してはいずれも安定

> **章**: 第8章 非線形振動
> **節**: 8・3 解析手法と非線形振動特性

解(stable solution)・不安定解(unstable solution)ともに区別なく求められる。両者ともに数学的には存在しうるが、安定解は物理的にも発生可能な解であるのに対し、不安定解はその近傍の微小な擾乱が時間とともに大きく成長するので、現実には発生し得ない解である。したがって、非線形系に対しては解を求めるだけでは不十分で、得られた解の安定性(stability)を判別しなければならない。8・3・3項では、得られた周期解に対する代表的な安定判別法(stability criterion)について説明する。a. 振幅・位相に関する微分方程式を利用する方法多重尺度法、平均法および漸近法では、振幅と位相θに関する自律系の微分方程式が求められる。それらをまとめて次式のように表す。得られた解を,θ、その微小な変分(variation)をξ,ηで表す。=+ξ,θ=θ+ηとして式(8・154)に代入すると、ξ,ηに関する一次近似は次式となる。したがって、その基本解(fundamental solution)は次式のように表される。すなわち、2個の特性根の実部がいずれも負であればξ,ηは∞で零に漸近するので周期解は漸近安定(asymptotically sta-ble)、一つでも正のものがあれば不安定(unstable)である。実部が零のときはなんらかの形で解の特性が急激に変化する分岐点(bifurcation point)に対応し、その安定性の判別にはより高度な解析が必要となる。ただし、分岐点上での解の挙動を問題とするのでない限り、現実の系ではこのことをそれほど深刻に考える必要はない。確かにこの判別法は比較的手軽で直感にも訴えやすい。しかしながら、1自由度系に対する1項近似解の場合には、この判別法で判別できる安定・不安定境界は、周波数応答曲線の垂直接線部だけであることが多い(1自由度系でも減衰項に非線形性を含む場合や高次近似解を用いる場合、あるいは多自由度系の場合にはこの限りではない)。垂直接線部は一般にサドルノード分岐点(saddle-nodebifurcation point)に一致する。ところが、非線形系には種々のタイプの分岐点が存在し、それが系の安定性を支配することが知られている。垂直接線部がサドルノード分岐に基づく安定・不安定境界であることは安定判別を行うまでもなくほとんど自明のことであるから、それ以外のタイプの分岐点を求めることが不得意なこの種の安定判別法は極めて不十分なものであるといえる。結局のところ、平均法、漸近法、多重尺度法などの解法でも、種々のタイプの分岐点を判別して周期解の安定性の全容を把握するためには、次に述べる変分方程式を利用する方法に頼らざるを得ないことが多い。b. 変分方程式を利用する方法周期解に関する安定判別法として最も一般的で信頼性の高いのが、変分方程式(variationalequation)を利用して無限小安定性(infinitesimal stability)を判別する方法である。ここでは、次のような式(8・36)を一般化した非自律系の元連立非線形常微分方程式を対象として、その周期解の安定判別法について説明する。さらに、はおよびに関して、式(8・158)に一意な解が存在する程度の滑らかさを有するものと仮定する。すなわち、式(8・159)は周期係数型の線形常微分方程式である。これを零解と呼ぶ。このことは、が上記で仮定した条件を満たす場合には、数学的に正しいことが証明できる。つまり、式(8・159)で省略したの二次以上の項の影響を無視できるというわけである。すなわち、周期解の安定性を変分方程式の零解の安定性に置換えて議論できることを意味している。このような安定性を無限小安定性と呼ぶ。周期解の安定性とは、無限小安定性を指すことが多い。周期係数型の線形常微分方程式(8・159)の零解の安定性を解析するに当たっては、フローケの定理(Floquet’s exponent)を求めることによって判別することができる。ただし、これらを解析的に求めることは極めて困難であり、通常は数値解法にたよらざるを得ない。その代表的な数値解法としては次の二つがある。状態遷移行列とは、単位行列を初期値とする基本解行列(fundamental matrix of
