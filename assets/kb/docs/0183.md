# 8・3・4 多自由度強非線形系に対する周期解の高精度計算法

> **章**: 第8章 非線形振動
> **節**: 2・8 項参照)が重要かつ有効である。

非線形振動系におけるカオスの発見以来、学問的興味の対象を定常周期振動に限定するわけにはいかなくなったが、振動工学的な観点からすれば、定常周期振動(周期解)の占める位置はいまだ重要であり、第一義的な意味をもっている。一方、機械システムは一般に大きな自由度を有しており、しかもガタや摩擦力の作用する系では非線形性がかなり強くなる。したがって、大規模な強非線形系に対して任意の精度の近似解を能率よく求めることのできる解析法が必要になる。8・3・2項で示した種々の解法も、このような要請に応えられる形式を整えてはいるが、数式処理等を含む計算機の利用を前提とした場合でも、実際上は無力であることが多い。そこで、8・3・4項では、多自由度強非線形系における周期解の有力な計算法であるシューティング法および調和バランス法による計算方法の概要を紹介する。これら二つの手法によれば、計算機を利用することによって、多自由度強非線形系における周期解を任意の精度で計算することができる。a.
