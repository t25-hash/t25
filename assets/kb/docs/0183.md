# 8・3・4 多自由度強非線形系に対する周期解の高精度計算法

> **章**: 第8章 非線形振動
> **節**: 2・8 項参照)が重要かつ有効である。

非線形振動系におけるカオスの発見以来、学問的興味の対象を定常周期振動に限定するわけにはいかなくなったが、振動工学的な観点からすれば、定常周期振動(周期解)の占める位置はいまだ重要であり、第一義的な意味をもっている。一方、機械システムは一般に大きな自由度を有しており、しかもガタや摩擦力の作用する系では非線形性がかなり強くなる。したがって、大規模な強非線形系に対して任意の精度の近似解を能率よく求めることのできる解析法が必要になる。8・3・2項で示した種々の解法も、このような要請に応えられる形式を整えてはいるが、数式処理等を含む計算機の利用を前提とした場合でも、実際上は無力であることが多い。そこで、8・3・4項では、多自由度強非線形系における周期解の有力な計算法であるシューティング法および調和バランス法による計算方法の概要を紹介する。これら二つの手法によれば、計算機を利用することによって、多自由度強非線形系における周期解を任意の精度で計算することができる。a. シューティング法(shooting method)シューティング法は、周期解の初期値を求める問題を二点境界値問題としてとらえて、直接数値積分法とニュートン法とを併用して周期解の初期値を効率的に求める手法である。次のような自律系の元連立非線形常微分方程式を取扱う。そこで、自律系の場合には解の位相に任意性がある(時間の原点を自由に選べる)ことに着目して、この位相を適当に拘束するものとし、次のような初期値に対する拘束条件式を1本付加する。=0(8・165)その一例としては、自励振動が発生している(恒等的に零ではない)成分のうち、いずれか一つの初期変位または初速度を零に拘束することなどが考えられる。式(8・164)および(8・165)を満足する初期値および未知周期をニュートン法により逐次近似計算することを考える。ただし、その点を除けば計算手続きは自律系の場合とほぼ同様であり、初期値の逐次近似式は次のようになる。判定基準は自律系の場合と同様である。b. 調和バランス法(多項近似)調和バランス法は、かなり広範な系に対して近似解近傍における厳密解の存在条件や厳密解への収束性が確立されており、数学的基礎付けの明確な手法であるといえる。また、アルゴリズムの規則性が高く、計算機の利用に適しており、多自由度系に対しても任意の精度で周期解を求めることが可能である。以下では、非自律周期系の式(8・158)を対象として解法の概要を説明する。ただし、式表示を簡潔にするため、一般性を損なうことなく基礎式の周期は2πであるものとする。なお、自律系に対しても調和バランス法を適用することは可能であるが、ここでは省略する。そこで、その(正整数)次までの有限級数で近似解を仮定し、次のように表す。なお、比較的広範なクラスの非線形振動系において、奇数次のみのフーリエ級数に展開されるような周期解が存在する。このような周期解を調和バランス法で求めるには奇数次成分のみを計算すればよいので、仮定する近似解の最高次数(ここでは正の奇数)が同一ならば、近似解計算並びに安定判別ともに、取扱うべき行列とベクトルの次元を、上記の約半分にすることができる。ただし、安定判別に用いるべき個の主特性指数が存在するμの領域は、次のように変更する必要がある。
