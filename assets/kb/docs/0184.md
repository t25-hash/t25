# 8・3・5 代表的な非線形強制振動の特性

> **章**: 第8章 非線形振動

漸硬ばね系ダフィング方程式の周波数応答図8・16に、式(8・38)の漸硬ばね系ダフィング方程式において、ε=1.0およびδ=1.0のときの周波数応答を示す。ただし、ノルムは、次式によって定義している。図中の実線は安定解、破線は不安定解であり、ζ=0.005のときの結果を太線で、ζ=0.15のときの結果を細線で示した。一点鎖線はζ=δ=0のときの自由振動解、いわゆる背骨曲線(skeletoncurve)である。また、○印および□印は、それぞれサドルノード分岐点およびピッチフォーク分岐点を表す。なお、8・3・5項では、奇数次のみのフーリエ級数で表される解を奇数次解、偶数次成分をも含む解を非奇数次解と呼ぶ。このように共振時の周波数応答曲線が右傾するのが、漸硬ばね系ダフィング方程式の定常周期解の特徴である。主共振領域では、減衰比ζが微小なときには安定解の分枝と不安定解の分枝(両分枝ともに奇数次解である)が背骨曲線を挟むようにΩω>1の広い範囲に存在しており、両者が結合してサドルノード分岐を形成する。サドルノード分岐点は周波数応答曲線の垂直接線部に対応しており、外力の角振動数Ωを徐々に変動させたとき、この分岐点で跳躍現象(jump phenomena)が発生する。ζが大きくなるにつれて振幅の大きな側のサドルノード分岐点がΩωの小さい側に移動し、ζがある値を超えると不安定分枝は消滅する。Ωωの大きな側から小さい側にかけて低次から高次の高調波共振の発生領域が局所的に存在し、図示の系パラメータのときには、二次と三次の高調波共振が顕著に現れている。偶数次の高調波共振と奇数次の高調波共振とは互いに性質がやや異なっており、前者は主共振の周波数応答曲線の左側すそ部に存在するピッチフォーク分岐点から発生する非奇数次解であるのに対し、後者はすそ部の分枝から連続的に発生する奇数次解である。両者ともに系パラメータ次第では広い範囲で存在し、サドルノード分岐点では跳躍現象も見られることもあるが、高次のものほど減衰比ζの影響が大きく、わずかなζで消滅する。図示の範囲内では1/2次と1/3次の分数調波共振が得られているが、両者ともにζ>0のときには安定解と不安定解とで孤立した島状の分枝を形成し、その結合部はサドルノード分岐点となっている。偶数次の分数調波共振は非奇数次解、奇数次の分数調波共振は奇数次解である。減衰比ζの影響は、奇数次よりも偶数次の分数調波共振に対するほうが大きいが、高調波共振に対する影響と比較すると分数調波共振に対するほうがかなり小さい。なお、式(8・38)のダフィング方程式においては、分数調波共振は主共振の右側すそ部の解とは全く独立に共存する解であり、対応した初期条件を与えない限り自然に発生することはないが、いったん発生すると存在領域がかなり広いので注意が必要である。以上のように、主共振だけではなく高調波共振や分数調波共振が発生すること、周波数応答曲線が傾斜することに起因して跳躍現象や同一のΩに対する解の多値性が現れること、および共振の発生に関して種々の分岐現象が関与していることなどが非線形系の強制振動の大きな特徴である。なお、8・3・2・h項で述べたように、弱非線形系に対する解析的手法では主共振、三次の高調波共振および1/3次分数調波共振の存在しか示すことができないが、図8・16の結果のように、系パラメータ次第ではそれ以外の次数の共振が多数発生する可能性があることに注意を要する。表示方法は図8・16とほぼ同様であり、図中の線幅の太いものから順にζ=0.15,0.175および0.2のときの結果である。また、△印は周期倍分岐点を表す。漸軟ばね系の周波数応答の特徴は、図示のように応答曲線が左傾することである。図中のすそ部をもつ分枝は奇数次解であり、ζが小さいときにはΩωが小さい領域まで存在するが、ζが大きくなるにつれて、すそ部をもつ分枝と図の左上から垂れ下がる半島状の分枝とに分離し、存在範囲がともに後退する。安定領域をもつ奇数次解の分枝は、Ωωが小さくかつ振幅が大きくなるとピッチフォーク分岐により不安定化し、分岐点から非奇数次解が発生する。そ図8・16 その他の特徴的な現象上記は、非線形系に特有の現象の中で最も基本的かつ一般的なものであるが、これら以外にも系の非線形特性、自由度、励振形態に対応して、それぞれに固有の多種多様な現象が発生する。その全体像を示すことは事実上不可能なので、以下では、8・3節でこれまでに触れることのできなかった興味深い現象の中からごく一部を選んで補足説明を行う。まず、1自由度系においても発生する現象として、複数の振動数成分からなる強制外力が作用したときの結合共振(combinationresonance for multifrequency excitation)がある。これを結合共振と呼ぶ。さらに、例えばω3ΩΩ3であるような場合には、角振動数Ωの加振力に対する三次高調波共振と角振動数Ωの加振力に対する1/3次分数調波共振が同時に発生して、複雑な特性を呈することもある。ただし、このような系で応答が周期的になりうるのはΩとΩが有理数比である場合だけであり、そうでない場合には概周期振動(quasi peri-odic vibration)となる。したがって、このような系の解析には、発生しうる主要な振動数成分の調波のみを仮定したうえで形式的に調和バランスの原理を適用する方法、平均法、多重尺度法等がよく利用される。次に、多自由度非線形系に特有の現象について述べる。周知のように、一般的な線形の保存系の場合には、各モードは力学的に独立(非連成)しており、モード間でエネルギーの授受は起こらない。これを内部共振(internal resonance)と呼ぶ。一例として、ばね振り子(spring pendulum)を考える。いま、一端が支持されたばね定数のばねの他端に質量の質点を取付け、鉛直面内で自由に振動させるとする。ただし、減衰は微小なので無視する。すなわち、一方のモードのみが発生するような初期条件を与えて自由振動させたとき、他方のモードにエネルギーが流れ、結果として両モードともにうなり状の振動(ただし、一方のモードの振幅が大きいときは他方の振幅は小さい)が発生する。これが内部共振の典型例である。一方、多自由度系に角振動数Ωの周期外力が作用し、Ωと固有角振動数ωが次のような関係を満たすとする。これをモード間の結合共振(combinationresonance among some modes)と呼ぶ。結合共振の角振動数ωとΩとの間には次の関係が成立する。以上の例からもわかるように、非線形系の振動現象は線形系に比べて非常に複雑で多様性に富んでおり、その性質に関して一般的な議論を行うことが極めて困難である。また、厳密解が得られないことが多いので、解析上の難点も多い。したがって、非線形系に対しては、解明したい現象の個別的性質と要求精度に応じた近似解法や数値解法を適切に選択して解析を行うことが肝要である。文献(1)
