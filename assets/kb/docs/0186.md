# 9・2・2 1自由度系の不安定振動系

> **章**: 第9章 自励振動
> **節**: 9・2 線形系の不安定振動

まず、運動方程式(9・12)の係数がすべて正、すなわち、>0,>0,>0の場合には、解(9・13)は減衰振動となり、安定解であるので大きな振動となることはない。その結果、<0のとき、式(9・13)から変位振幅は振動しながら時間とともに指数関数的に増大する-ζω>0.このように、減衰係数が負またはそれと等価な作用を生じさせる要因を負性抵抗と呼ぶ。したがって、<0のときが負性抵抗による不安定に相当する。次に、運動方程式(9・12)のばね定数が負となる不安定系の例を見てみよう。ばね定数、減衰係数、自然長のばねを滑らかな管の中に入れ、その一端に質量の質点を取付け、他端を回転中心Oに固定する。管が点Oを中心に水平面内を一定角速度ωで回転するときの遠心力場における系の安定性を検討しよう。右辺=0とおいた同次方程式にフルビッツの安定判別法を適用すると、漸近安定の条件は、>0,>0および-ω>0となる。
