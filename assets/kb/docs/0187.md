# 9・2・3 負性抵抗に基づく不安定振動

> **章**: 第9章 自励振動
> **節**: 9・2 線形系の不安定振動

線形系の運動方程式において減衰係数が負となって発生する不安定振動の例を調べる.a. 乾性摩擦による自励振動機械系には摩擦に起因する自励振動,すなわち,摩擦振動(frictional vibration)が発生し,大きなトラブルとなることがある.摩擦は原則的には振動を抑制する働きがあるが,その摩擦が振動を助長して自励振動を生じさせるので,注意が必要である.接触部に相対的な滑りを生じるときの摩擦特性には以下の種類がある.(1) 垂直抗力に比例し,接触面積に依存しない,運動の向きとは常に逆向きに作用するクーロン摩擦(Coulomb’s fric-tion)(2) 相対速度に関して負のこう配を有する乾性摩擦(dry fric-tion)(3) わずかな相対滑りを伴って転がり接触している部分に作用するクリープ力(creep force)(4) 相対速度に比例した粘性減衰(viscous damping)図9・2の横軸に相対滑り速度または滑り率,縦軸に摩擦力またはクリープ力をとって,これらの摩擦特性を示す.さらに,図9・2の結合型として,滑りが大きくなるにつれてクリープ力からクーロン摩擦へ,あるいは,クリープ力から乾性摩擦へと変化するような摩擦特性もある.図9・3のように,一定速度で移動するベルト上に置かれた質量の物体がばね定数のばねで支持されている.物体のベルト移動方向の変位をとし,=0でばねは自然長であるとする.はじめに,ベルトの面は滑らかで,かつ油がひかれており,物体とベルトとの相対速度に比例した抵抗力(比例定数>0)が作用しているとすれば,物体の運動方程式は次式となる.+-+=0(9・14)ここに,,,>0.-の項はベルトの定速移動によるエネルギーが振動系に与えられることによって生じた項である.変数変換ξ=-を施すと,式(9・14)は次式に変換される.ξ+ξ+ξ=0(9・15)したがって,ξは減衰振動となる.結局,振動系に流入したエネルギーはほどの静的変位を生じさせるが,動的には振動を抑制する作用をもっている.よって,上記(4)の粘性減衰は自励振動を発生させる原因とはなり得ない.一方,ベルトと物体の間に上記(2)の乾性摩擦が作用する場合には,乾性摩擦力を相対速度=-の関数として-で表すと,運動方程式は次式となる.図9・2
