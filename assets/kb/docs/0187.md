# 9・2・3 負性抵抗に基づく不安定振動

> **章**: 第9章 自励振動
> **節**: 9・2 線形系の不安定振動

線形系の運動方程式において減衰係数が負となって発生する不安定振動の例を調べる。a. 乾性摩擦による自励振動機械系には摩擦に起因する自励振動、すなわち、摩擦振動(frictional vibration)が発生し、大きなトラブルとなることがある。摩擦は原則的には振動を抑制する働きがあるが、その摩擦が振動を助長して自励振動を生じさせるので、注意が必要である。接触部に相対的な滑りを生じるときの摩擦特性には以下の種類がある。(1) 垂直抗力に比例し、接触面積に依存しない、運動の向きとは常に逆向きに作用するクーロン摩擦(Coulomb’s fric-tion)(2) 相対速度に関して負のこう配を有する乾性摩擦(dry fric-tion)(3) わずかな相対滑りを伴って転がり接触している部分に作用するクリープ力(creep force)(4) 相対速度に比例した粘性減衰(viscous damping)図9・2の横軸に相対滑り速度または滑り率、縦軸に摩擦力またはクリープ力をとって、これらの摩擦特性を示す。さらに、図9・2の結合型として、滑りが大きくなるにつれてクリープ力からクーロン摩擦へ、あるいは、クリープ力から乾性摩擦へと変化するような摩擦特性もある。図9・3のように、一定速度で移動するベルト上に置かれた質量の物体がばね定数のばねで支持されている。物体のベルト移動方向の変位をとし、=0でばねは自然長であるとする。はじめに、ベルトの面は滑らかで、かつ油がひかれており、物体とベルトとの相対速度に比例した抵抗力(比例定数>0)が作用しているとすれば、物体の運動方程式は次式となる。結局、振動系に流入したエネルギーはほどの静的変位を生じさせるが、動的には振動を抑制する作用をもっている。よって、上記(4)の粘性減衰は自励振動を発生させる原因とはなり得ない。また、ベルトの運動方向を逆にしても同じように不安定振動は生じる。いま、物体がばねとダッシュポットで支持されているとする。すなわち、系に存在する減衰よりも負のこう配が大きいと負性抵抗による不安定振動が生じる。乾性摩擦に起因する自励振動の特徴を以下にまとめる。(1) 滑りの向きを逆にしても自励振動は生じる。一般に、方向性をもたない。(2) 1自由度系でも多自由度系でも生じうる。(3) 振動の主成分は摩擦力の作用する方向である。(4) 摩擦特性の負のこう配が振動的不安定を生じさせる原因である。摩擦力の大きさは不安定振動の発生に影響しない。(5) 多自由度系では、減衰行列の対角項が負となる負性抵抗による。(6) 相対滑り速度に対する負のこう配に起因する負性抵抗と構造物に内在する正の減衰との大小で自励振動の発生が決定される。乾性摩擦による自励振動の事例としては、ワイングラスの鳴き、バイオリンの弦の振動などがある。b. ポンプのサージング図9・4(a)のように大きな貯水池Aから中間の管路にあるポンプで他のサージタンクB へ揚水する。タンクからの流出量をとする。c. ギャロッピング送電線に氷結が生じ、その断面形状が円形から変化すると、一様な風の中で送電線が大きく振動することがある。このギャロッピング(galloping)と呼ばれる不安定振動を考察する。図9・5に示すような流速の一様流れの中に置かれた質量の物体がばねとダッシュポットで支持され、流れと垂直方向に振動する1自由度系を考える。無次元流速が大きいときに発生する流体力の特性は物体が流れに向かう角度(迎え角-α)に依存し、次のような揚力と抗力で表現される。よって、流速が次式を満たすとき、ギャロッピングが生じることになる。
