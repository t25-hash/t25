# 9・3・1 概説

> **章**: 第9章 自励振動
> **節**: 9・3 非線形自励振動の定常応答

線形系で不安定振動が発生すると、その振幅は指数関数的に増大する。しかし、一般の機械系では振幅が大きくなると構造的な非線形性の影響が現れ、この非線形性によって振幅の増加が抑制されて、振幅の有界な振動に移行することが多い。特に、1自由度非線形自励系の場合には、ポアンカレ・ベンディクソンの定理(Poin-care-Bendixson’s theorem)により、かならず周期振動が発生することが証明されている。上記の例を含めて、1自由度非線形自励系の運動方程式は一般的に次のように表すことができる。これがポアンカレ・ベンディクソンの定理の内容である。→-∞のときにも同様のことがいえ、→∞のとき安定な平衡点または周期解、→-∞のとき不安定な平衡点または周期解となる。この定理から、次のようにして1自由度非線形自励系に周期解が存在するか否かを判定することができる。平面内に任意の閉曲線CおよびCを取り、CはCの内側に含まれるとする。この二つの閉曲線で囲まれる領域を考え、その境界CおよびC上で定義される式(9・53)のベクトル場が常にの内部を向く(境界上を初期値とする解がすべての内部に向かう)とする。このとき、の内部に式(9・53)の平衡点が存在しなければ、ポアンカレ・ベンディクソンの定理により周期閉軌道が少なくとも一つは存在するので、安定な周期振動が発生すると結論できる。このように、1自由度非線形自励系の場合には、静的平衡状態、周期振動あるいは発散振動のいずれかに分類できる。一方、多自由度非線形自励系の場合にはポアンカレ・ベンディクソンの定理に相当するような定理は知られておらず、有界な振動は周期振動や概周期振動であったりカオスになることもある。ただし、周期振動が発生する場合には、その振動数と振動モードは線形化された系のある一つの次数の固有振動数と固有モードにほぼ一致すると考えてよい。
