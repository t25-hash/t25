# 9・3・2 リエナール型の方程式

> **章**: 第9章 自励振動
> **節**: 9・3 非線形自励振動の定常応答

1自由度非線形自励系の典型例として、次のようなリエナールの方程式(Lie썝nard’s equation)が古くから詳しく研究されている。++=0(9・54)いま、,が次の四つの条件を満足すると仮定する。(i) ,はすべてのについて連続で、はリプシッツ条件(-≦-,は任意の正数)を満足する。(ii) は偶関数で0<0であり、は奇関数で≠0のとき>0である。(iii) =ξξとおけば、→±∞のとき→±∞.(iv) =0は>0でただ一つの根=をもち、≧0では単調増加である。この四つの条件のもとで、式(9・54)は安定な唯一の周期解をもつ。これをリエナールの定理(Lie썝nard’s theorem)という。リエナールの定理の成立する系の代表が、次のファン・デル・ポールの式(van der Pol’s equation)である。-ε1-+=0(9・55)また、関連するものにレイリーの式(Rayleigh’s equation):-ε1-3+=0(9・56)がある。レイリーの式をについて微分し、をに置換えるとファン・デル・ポールの式となる。ε>0のとき、式(9・55)は明らかに条件(i)~(iv)を満足するので、安定な周期解がただ一つ存在する。この周期解の位相面上での解曲線(解軌道)は閉曲線になる。これを極限閉軌道またはリミットサイクル(limit cycle)と呼ぶ。一方、ε<0のときには原点の近傍で正の減衰となるので、=0,=0が安定な平衡点となり、周期振動は発生しない。このようにε=0が安定な平衡点と安定な周期振動の分岐点となっている。このような分岐をホップ分岐(Hopf bifurcation)と呼ぶ。位相面上に描いたレイリーの式の解軌道および波形を図9・18に示す。図9・18(a)はε=1のときの解軌道であり、時間が経過するにつれて、一つの周期閉軌道(リミットサイクル)にその外部および内部の任意の点を初期値とする解軌道が漸近する。図9・18(b)はε=1から10まで1刻みで求めたリミットサイクル、図9・18(c)はε=1および10のときの波形である。の波形はファン・デル・ポールの式の解に相当する。系パラメータε>0が小さいときには閉軌道は円に近く、およびの波形はいずれも正弦波に近い。一方、εが大きくなるにつれて、はのこぎり波に、は方形波に近づく。εが大きいときののように、物理量が非常に緩やかな変化と急激な変化とを繰返す自励振動を弛緩振動(relaxation oscillation)という。日本庭園にある鹿追がその典型例である。ε>0が微小であるときのファン・デル・ポールの式の周期解を平均法により求めよう。,=-1-,τ=+θとおき、解を=cos τ,=-sinτとすれば、8・3・2・e項の手順により次のような平均化方程式が求められる。=ε2πcos τ,-sinτsinττ=ε21-θ=ε2πcos τ,-sinτcos ττ=0(9・57)初期条件=0で=として上式を解くと、=expε21+expε-14(9・58)となり、→∞で一定振幅=2のリミットサイクルに漸近する。また、その周期は2πとなる。一方、ε>0が大きなときをも含めて、シューティング法(8・(a)(b)(c)図9・18 レイリー方程式のリミットサイクルと波形表9・1ファン・デル・ポール方程式の周期解の周期ε/2π0.016.283224576991.000006249980.056.284166990921.000156239820.106.287111272291.000624836750.506.380675801771.015516094121.006.663286859321.060495040905.0011.61223066771.8481439110910.019.07836956693.03641682271方程式の周期解の周期を表9・1に示す。この結果から、εが大きくなるにつれて2πよりも大きくなっており、平均法の結果である2πからのずれが拡大することがわかる。
