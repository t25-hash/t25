# 9・3・2 リエナール型の方程式

> **章**: 第9章 自励振動
> **節**: 9・3 非線形自励振動の定常応答

1自由度非線形自励系の典型例として、次のようなリエナールの方程式(Lienard’s equation)が古くから詳しく研究されている。この四つの条件のもとで、式(9・54)は安定な唯一の周期解をもつ。これをリエナールの定理(Lienard’s theorem)という。リエナールの定理の成立する系の代表が、次のファン・デル・ポールの式(van der Pol’s equation)である。ε>0のとき、式(9・55)は明らかに条件(i)~(iv)を満足するので、安定な周期解がただ一つ存在する。この周期解の位相面上での解曲線(解軌道)は閉曲線になる。これを極限閉軌道またはリミットサイクル(limit cycle)と呼ぶ。このようにε=0が安定な平衡点と安定な周期振動の分岐点となっている。このような分岐をホップ分岐(Hopf bifurcation)と呼ぶ。位相面上に描いたレイリーの式の解軌道および波形を図9・18に示す。図9・18(a)はε=1のときの解軌道であり、時間が経過するにつれて、一つの周期閉軌道(リミットサイクル)にその外部および内部の任意の点を初期値とする解軌道が漸近する。図9・18(b)はε=1から10まで1刻みで求めたリミットサイクル、図9・18(c)はε=1および10のときの波形である。の波形はファン・デル・ポールの式の解に相当する。εが大きいときののように、物理量が非常に緩やかな変化と急激な変化とを繰返す自励振動を弛緩振動(relaxation oscillation)という。日本庭園にある鹿追がその典型例である。ε>0が微小であるときのファン・デル・ポールの式の周期解を平均法により求めよう。また、その周期は2πとなる。一方、ε>0が大きなときをも含めて、シューティング法(8・(a)(b)(c)図9・18 レイリー方程式のリミットサイクルと波形表9・1ファン・デル・ポール方程式の周期解の周期ε/2π0.016.283224576991.000006249980.056.284166990921.000156239820.106.287111272291.000624836750.506.380675801771.015516094121.006.663286859321.060495040905.0011.61223066771.8481439110910.019.07836956693.03641682271方程式の周期解の周期を表9・1に示す。この結果から、εが大きくなるにつれて2πよりも大きくなっており、平均法の結果である2πからのずれが拡大することがわかる。
