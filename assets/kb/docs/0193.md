# 9・3・3 非線形係数励振系

> **章**: 第9章 自励振動
> **節**: 3・4 ・a項参照)により非常に高精度で求めたファン・デル・ポール

系を線形とみなしたときの不安定振動の発生原因が係数励振の場合であっても、非線形性の影響で定常振動が発生することがある。次式のような、マシュー型方程式に対して粘性減衰と三次の非線形復元力が付加された系について考えよう。9・2・5・c項で述べたように、ζ=0,β=0とおいた線形不減衰マシュー方程式では、Ωωの主共振領域で不安定振動が発生する。そこで、こ2―79図9・19非線形減衰マシュー方程式の周波数応答の領域における式(9・59)の周期解を次式のように仮定して、調和バランス法(8・3・2・g 項参照)により解析する。このことから、ζが微小で線形減衰マシュー方程式に不安定領域が存在するとき、非線形減衰マシュー方程式の周期解は、線形減衰マシュー方程式の安定・不安定境界からホップ分岐して発生することがわかる。そのような境界ν,νν<νは次式で与えられる。図中の実線は安定解、破線は不安定解である。β>0のときは振幅の応答曲線は右傾しており、ν=νから分岐する解が安定で、ζ>0のとき解の存在領域に関してνの上限値が存在する。一方、β<0のときは振幅の応答曲線は左傾し、ν=νから分岐する解が安定である。ただし、図示の範囲よりもさらにνが小さくなると不安定化する。β<0はぶらんこの事例で振幅が比較的大きな場合に相当し、図9・19の結果は、ある程度振幅が大きくなった状態で安定な周期振動が実現しうることを示している。
