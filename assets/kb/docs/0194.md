# 9・4・1 概説

> **章**: 第9章 自励振動
> **節**: 9・4 自己同期

同期化現象は自励振動と関連して非線形現象に分類され,次の二つに大別される.(1) ある非線形自励振動系に周期的な外力が作用するときに生じる同期化現象で,強制引込みともいう.自励系の固有振動数ω욈と外力の振動数Ωが接近し,振動数差ω욈-Ωがある範囲以内になると自励振動の振動数が周期外力の振動数に引込まれて同期す2―80機械力学る.引込まれた振動の振動数は外力の振動数Ωである.また,外力の振幅が大きいほど振動数差が大きくても同期が可能となる.一方,両方の振動数が遠く離れている場合や強制外力が弱い場合には,うなりが発生する.(2) 自励系1(固有振動数ω)と自励系2(固有振動数ω)の二つの自励振動系を結合したときにも,双方の固有振動数が近いωωと一つの自励系となって同期する.これを相互引込みという.ω<ωと仮定すると,普通は,ω<ω<ωを満たす二つの自励系の特性によって決まる振動数ωで同期する.しかし,同期振動数は常に等しいωとは限らず,二つの自励系の振動数がある整数比で同期するときもある.このように振動体が同期して振動する現象を自己同期(self-syn-chronization)という.
