# 9・4・2 非線形自励系振動の強制引込み

> **章**: 第9章 自励振動
> **節**: 9・4 自己同期

自励系であるファン・デル・ポールの式に強制外力が作用するときの運動方程式は次式で表せる。-ε1-+=cos ω(9・63)強制力の角振動数と系の線形固有角振動数との差ω-1を離調度(detuning)と呼ぶ。この差が大きいときは自励振動と強制振動の振動成分が合わさったうなりを生じるが、ωを1に近づけると、うなりは消滅し外力の振動数に一致した振動が出現する。この現象を自励振動における振動数の引込み(entrainment)という。引込み振動を、=sinω+cos ωと仮定して式(9・57)に代入すると、次式となる。σ+1-=0, 1--σ=εωここに、=,=,=+,=2,σ=1-ωεω.は外力がないときの自励振動の振幅、σは離調度を表すパラメータである。したがって、振動振幅と外力の振幅との関係は、1-+σ=εω(9・64)となる。引込み振動解の安定性は省略し、調和振動の引込み特性の結果のみを図9・20に示す。実太線が安定解、すなわち、引込みが生じる領域、波線が不安定解、実細線が安定/不安定境界の曲線を表す。外力の振幅が小さいと引込み領域が狭く、離調度が大きくなると解は不安定化する。
