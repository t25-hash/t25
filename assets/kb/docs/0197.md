# 9・4・4 自励回転

> **章**: 第9章 自励振動
> **節**: 9・4 自己同期

リングや輪ゴムに内側から指をかけて動かすと、これらは指に調子を合わせて回転する。すなわち、指の振動数とリングや輪ゴムの回転数は同期する。同様な現象はフラフープの運動にも見られる。図9・20調和外力を受けるファン・デル・ポール方程式の引込み振動これらの運動と等価な系として、図9・21に示すような自由に回転する長さ、質量の振り子の支点が振幅、角振動数ωの余弦的な運動をするときの振り子の回転運動を考える。減衰係数をとすると、振り子の回転角θに関する運動方程式は、次式で表せる。定常回転を考えて、φ=φ=0とし、定常回転時の位相φをφとおくと、次式を得る。すなわち、振り子の定常回転は振り子の支点の振動よりφだけ遅れている。また、位相面解析から初期角速度θが支点の角振動数ωより大きいときにこの定常回転が実現しやすいことがわかる。
