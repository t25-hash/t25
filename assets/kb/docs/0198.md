# 9・4・5 自動平衡装置

> **章**: 第9章 自励振動
> **節**: 9・4 自己同期

図9・22に示すように、回転軸A に偏心距離がOG=のロータB の中に自由に動きうる質量の2個の鋼球Cを入れて軸を危険速度以上で回転させると、2個の鋼球はロータの中で自動的にロータの偏心を修正する位置に配置されて、回転軸の振動および軸受の動荷重を減少させる。この装置を自動平衡装置(automatic balan-cer)と呼ぶ。この現象を図9・23に示す系で考察する。いま、ロータの回転速度が危険速度以上の高速回転運転を考える。そのとき、上式の第3,4式の左辺の減衰項および復元項は慣性項に比べて無視できる。この応用として、鋼球の代わりに塩水を用いた流体バランサが日本の洗濯機の脱水時の制振に利用されている。
