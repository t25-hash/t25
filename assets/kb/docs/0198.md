# 2・1・3 制御装置

> **章**: 第2章 メカトロニクス
> **節**: 2・1 ハードウェア

メカトロニクスシステムのための制御装置の役割は，時間やセン
サからの情報に応じたアクチュエータの動作を管理することであ
る．
a.専用コントローラとはん用コントローラ システムとして
制御する対象が特定されている場合には，そのシステム専用のコン
トローラを用意することがコスト面等で有利な場合が多い．特に単
純なシステムであれば，専用コントローラを開発するコストもそれ
ほどかからないためその傾向は強い．
しかし，仕様の変更等に応じてコントローラを変更する場合に開
発をやり直す必要がでてくる．またIC，LSIの集積能力の向上に
より冗長なはん用コントローラを製作してもあまりコストがかから
ず，むしろ大量に安価に作れることから，近年ははん用的なコント
ローラを用いることが多い．
b.オープンアーキテクチャ 近年，はん用性をさらに進め，
共通の仕様を定めようという試みが，大規模メカトロニクスシステ
ムであるロボットを中心として行われ始めている．センサとアクチ
ュエータ間のやり取りだけでなく，情報のやり取りを含めて規格化
し，開発効率の向上，ロボットモジュールの開発・商品化を目指し
ている．
c .さまざまなコントローラ 単純なものから複雑なものまで
さまざまなシステムを対象としたコントローラがある．その中のい
くつかを紹介する．
(１)シーケンサ（sequencer ）
ON/OFFだけで動作するスイッチやモータなどを，リレー等を
介して接続し，望んだ手順で動作するように構成したもの．専用コ
ントローラとして構成される．
(２)プログラマブルコントローラ
シーケンサをはん用化したもの．多数の入力端子と出力端子をも
ち，動作対応をプログラムとして登録することにより動作させる．
さまざまな種類のものが市販されている．
(３)PIC・ワンチップマイコン
(４)シングルボードコンピュータ
くは信頼性を高めたコントローラとして使用される．
(５)はん用PC
(ａ)ウォームギヤ
タルI / OやアナログI られてきている．
