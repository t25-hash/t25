# 10・1・1 離散力学系

> **章**: 第10章 分岐現象とカオス
> **節**: 10・1 概説

生物の個体数の変化や人口問題を表す成長の方程式はロジスティック写像(logistic map)と呼ばれている次の差分方程式で表される。このとき、1周期不動点は零および1-1/である。=3.3[図10・1(c)]ではは完全な2周期になり、さらに=4.0[図10・1(d)]になると、非周期的となり、カオスが発生する。図10・2はを2.8から4.0まで0.002きざみで、初期値を0.5として、500回の過渡状態を除いた後の200回写像した解を示す。さらに、が大きくなると順次周期倍化を繰返した後、カオスになる。パラメータを変化させたとき、急激に系の特性が変化する現象を分岐現象(10・2・1項参照)、そのときのパラメータの値を分岐点という。このように周期が倍化する分岐を周期倍化分岐(period doubling bifurcation)(あるい(a) =0.7(b) =1.5(c) =3.3(d) =4.0図10・1 ロジスティック写像はフリップ分岐)という。図10・3は図10・2の周期倍化分岐からカオスに至る分岐図を模式的に示す。式(10・2)(10・3)は定義域内で極値を一つもち、シュワルツの導関数が負となるような写像について成立する。δやαをファイゲンバウム定数(Feigen-baum number)という。周期倍化を無限回繰返した点であるの集積点=3.569 945...では、無限個の周期点が存在する。図10・2の分岐図には、カオス領域の間に窓のように開いた周期軌道(周期解)が見られる。特に、=3.83付近では、3周期解が生じている。この付近を拡大した分岐図を図10・4に示す。このようにカオス領域で安定な周期解が発生するとき、周期の窓(window)という。周期の窓はサドルノード分岐(saddle-nodebifurcation)(10・2・2項参照)から生じる。でサドルノード分岐から生じた安定な3周期解が周期倍化して、三つの狭い領域に閉図10・2 ロジスティック写像の分岐図図10・3 ロジスティック写像の分岐図の模式図図10・4 ロジスティック写像の分岐図の拡大図じ込められたカオス(アトラクタが3個の閉区間に分割される3バンドアトラクタ)になる。アトラクタ(attractor)とは近傍のトラジェクトリを引きつける性質がある軌道をいう。窓中の一つのバンドアトラクタには矢印で示すように3周期解に対応する窓が存在し、図10・2の分岐図を縮小したものになっている。このように、カオスは自己相似性(self-similarity)、つまりフラクタル構造をもっている。さらに、が大きくなると、3バンドアトラクタがで一つのアトラクタになる。このように急激にストレンジアトラクタ(10・3・1項参照)が大きくなることをカオスの危機(crisis ofchaos)(あるいは爆発)という。この場合のカオスの危機はサドルノード分岐から生じる不安定な3周期解が3バンドアトラクタと接触するとき発生する。二次元2パラメータ写像として、3階のローレンツ微分方程式を簡単にモデル化したエノン写像(Henon map)がある。なお、分岐の過程で、どのような不動点がどのような順序で現れるかなど系の挙動を記号列を用いて表す記号力学がある。また、周期軌道が現れる順序や共存に関して、連続な一次元写像の場合について一般性のあるシャルコフスキーの定理(Sarkovskii theo-rem)がある。また、この定理はリー・ヨークの定理『周期3の軌道があればカオスになる』を一般化したものになっている。しかし、このシャルコフスキーの定理は周期軌道の最初の出現の順序を示すのみで、すべての周期軌道の位置を記述するものではない。
