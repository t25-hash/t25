# 10・1・2 連続力学系

> **章**: 第10章 分岐現象とカオス
> **節**: 10・1 概説

非自律系の連続力学系の例として,2個の磁石と片持はりからなる系に正弦波の強制力が作用したときに得られるダフィング方程式を取上げる.+-+=γcos ω(10・4)ここで,は減衰係数,γは強制力の振幅を表す.γ=0.175,=0.1について,初期値を,=-1,0としてωを1.05から1.20まで増大させたときのの分岐図および数値計算による1周期解をそれぞれ図10・5(a)(b)に示す.図10・5(b)において,実線は安定解,破線は不安定解を示す.分岐図には二つの1周期解のうち初期値に近い安定な1周期解が現れる.この1周期解がサドルノード分岐した後,安定解が存在しないため,カオスが生じる.カオス領域には5周期の窓が見られる.さらに,ωを増大させると周期倍化分岐の逆の過程を経て,1周期解になる.図の周期に対する変位波形,変位のパワースペクトル密度およびポアンカレ写像(Poincare썝map, 10・3・1項参照)を示す.この図(a)(b)図10・5ダフィング系の分岐図および1周期解2―83数ωおよび5周期に相当するω/5にピークをもつ以外ほとんどピークのないスペクトル,(3)ストレンジアトラクタを示していることがわかる.なお,非自律系の連続力学系はポアンカレ写像によって離散力学系と考えることもできる.
