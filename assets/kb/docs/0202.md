# 10・1・3 カオスへのルートおよび判定法

> **章**: 第10章 分岐現象とカオス
> **節**: 10・6 からカオスの特徴である(1)非周期性な波形、(2)強制振動

カオスへのルートとしては、おもに、(1)周期倍化分岐からのカオス、(2)周期的な振動状態から間欠的に乱れた振動が現れる間欠性カオス、(3)準周期(有理比の関係にない周期をもつ周期解の和からなる解)からのカオス、(4)パラメータを変化させたとき、不動点や周期状態が不安定化し、突然発生するカオスなどがある。また、カオスの判定法として、(1)振動波形の非周期性、(2)ポアンカレ写像によるストレンジアトラクタの存在、(3)パワースペクトル密度の幅広い周波数分布、(4)リアプノフ指数(10・3・1項参照)、(5)フラクタル次元、(6)不変多様体におけるホモクリニック点の存在などが挙げられる。力学系(dynamical system)の特性がそれに含まれるパラメータの変化によって定性的に変わることを分岐という。物質の相転移、対流、座屈、また振動問題においては、周波数応答に見られるジャンプ現象、自励振動、係数励振など質的な変化を伴う現象の多くは、数学的に解の分岐としてとらえられる。まず分岐現象の最も簡単な例として、図10・7に示された圧縮荷重を受ける2リンク系に生じる座屈現象を考える。すなわちパラメータが点を超えて変化するとき、この力学系の挙動は定性的に変化し分岐が発生している。
