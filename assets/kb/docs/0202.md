# 10・1・3 カオスへのルートおよび判定法

> **章**: 第10章 分岐現象とカオス
> **節**: 10・6 からカオスの特徴である(1)非周期性な波形、(2)強制振動

カオスへのルートとしては、おもに、(1)周期倍化分岐からのカオス、(2)周期的な振動状態から間欠的に乱れた振動が現れる間欠性カオス、(3)準周期(有理比の関係にない周期をもつ周期解の和からなる解)からのカオス、(4)パラメータを変化させたとき、不動点や周期状態が不安定化し、突然発生するカオスなどがある。また、カオスの判定法として、(1)振動波形の非周期性、(2)ポアンカレ写像によるストレンジアトラクタの存在、(3)パワースペクトル密度の幅広い周波数分布、(4)リアプノフ指数(10・3・1項参照)、(5)フラクタル次元、(6)不変多様体におけるホモクリニック点の存在などが挙げられる。力学系(dynamical system)の特性がそれに含まれるパラメータの変化によって定性的に変わることを分岐という。物質の相転移、対流、座屈、また振動問題においては、周波数応答に見られるジャンプ現象、自励振動、係数励振など質的な変化を伴う現象の多くは、数学的に解の分岐としてとらえられる。まず分岐現象の最も簡単な例として、図10・7に示された圧縮荷重を受ける2リンク系に生じる座屈現象を考える。無次元状態方程式は状態量=,=を用いて、+=/2-α(10・5)-1-1のように表せる(粘性減衰効果-を考慮し、・は無次元時間に関する微分を意味する)。無次元圧縮荷重とリンクの平衡状態にお図10・6 変位波形、パワースペクトル密度およびポアンカレ写像図10・7 圧縮力を受ける2リンクモデル2―84ける無次元横変位との関係はばね復元力の三次の非線形性αがα>1/2の場合、=1の近傍で図10・8(a)のように表せるμ=-1.ここで実線は安定な平衡状態(equilibrium state)、破線は不安定な平衡状態を表す。図10・8は自明な平衡状態=0がパラメータ値=1≡욂욥μ=0を境に安定から不安定に変わる(座屈現象)点および平衡状態が一つから三つに変わる点で特徴的である。すなわちパラメータが点욂욥を超えて変化するとき、この力学系の挙動は定性的に変化し分岐が発生している。このとき욂욥は分岐点と呼ばれる。
