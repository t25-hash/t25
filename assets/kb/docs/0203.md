# 10・2・1 不動点とその安定性(stability)

> **章**: 第10章 分岐現象とカオス
> **節**: 10・2 分岐

以下のような微分方程式に支配される力学系を考える。=; (10・6)ここで、は次元ベクトル、はパラメータを表す次元ベクトル、; はベクトルとの次元ベクトル値関数である。不動点(平衡点)は/=0すなわち; =0(10・7)を満たす≡である。ヤコビアン=の固有値より点の局所的安定性が求まる。固有値の実部がすべて負の場合は安定、それ以外の場合は不安定と判別される。実部零の固有値が存在しない場合、不動点(fixed point)は双曲型不動点と呼ばれる(図10・7のシステムにおいて不動点=0は≠1のとき双曲型不動点)。一方、存在する場合不動点を非双曲型不動点と呼ぶ。はパラメータの関数であり、パラメータの値によってその固有値は変化するが双曲型不動点は、微小なパラメータの変化に対してその定性的性質を変えないことが知られている。したがって平衡点が分岐するためには、少なくともそれが非双曲型でなければならない。図10・7のシステムにおいて不動点=0は=1のとき非双曲型不動点であり、図10・8(a)からわかるようにこの点で分岐が生じている。
