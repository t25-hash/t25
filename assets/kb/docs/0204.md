# 10・2・2 分岐の分類

> **章**: 第10章 分岐現象とカオス
> **節**: 10・2 分岐

ここでは一つのパラメータが変化して生じる分岐現象の例を示す。a. サドルノード分岐(saddle-node bifurcation)二次の非線形項を含んだ以下の力学系の分岐図は図10・8(b)のようになる。これはサドルノード分岐と呼ばれる。b. トランスクリティカル分岐(transcritical bifurcation)二次の非線形性に起因して生じる分岐にはもう一つトランスクリティカル分岐がある[図10・8(c)参照].この力学系は以下の式で支配される。c. ピッチフォーク分岐(pitchfork bifurcation)三次の非線形項を含んだ以下の力学系の分岐は図10・8(a)のようになる。このような分岐はピッチフォーク分岐と呼ばれる。d. ホップ分岐(Hopf bifurcation)以上三つの分岐は不動点でのヤコビアンが固有値零をもつような非双曲型不動点で発生したものであるが、以下の力学系ω≠0では平衡点==0が純虚数の固有値をもつ非双曲型不動点μ=0で分岐が発生する。このような分岐はホップ分岐と呼ばれる。
