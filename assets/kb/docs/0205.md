# 10・2・3 中心多様体(center manifold)

> **章**: 第10章 分岐現象とカオス
> **節**: 10・2 分岐

a. 平衡点と不変部分空間不動点の安定性は10・2・1項で述べたようにヤコビ行列の固有値によって決まるが、各固有値が張る固有ベクトルはさらに以下のように分類される。負の実部をもつ固有値に対応する固有ベクトルが張る空間は安定部分空間욦、正の実部をもつ固有値に対応する固有ベクトルが張る空間は不安定部分空間욊、実部零の固有値に対応する固有ベクトルが張る空間は中心部分空間욂と呼ばれる。욦上での解は時間とともに平衡点に近づき、욊上では時間とともに平衡点から遠ざかる。これに対して욂上での解は初期状態を保ったまま変化しない。中心部分空間をもてばその平衡点は非双曲型であり、さらに不安定部分空間をもたない場合は、その平衡点はヤコビ行列の固有値からは安定判別不可能である。次に述べる中心多様体理論はこのような場合の安定判別を可能にする。b. 中心多様体を用いた非双曲型不動点の安定判別と力学系の低次元化式(10・6)のが級の滑らかさをもつ場合非線形性の影響により、不変部分空間は平衡点で接した不変多様体に変形されることが知られている(図10・9)。例えば式(10・5)のヤコビアンは不動点==0に関して=1のとき一つの零固有値と負の実固有値をもつ。したがって、図10・9のように安定部分空間욦と中心部分空間욂が存在し、さらに非線形性の効果によりこれらは不変多様体욦,욂のように変形される。ここで욦,욂は特に安定多様体、中心多様体と呼ばれ区別される。このとき解は図沿って変化する。中心多様体は以下の方法によりべき級数表示可能である。=1における式(10・5)の非双曲型不動点について中心多様体を求めてみる。まず線形変換,=+,-を施す(a) 中心多様体(b) 解挙動図10・9不変多様体と、式(10・5)は=1/2-α+=,(10・13)=--1/2-α+=,(10・14)となる。このときの中心部分空間욂と安定部分空間욦は図10・9のように表せる。さて中心多様体は自明な平衡点==0で中心部分空間に接するのであるから、==++...(ここで,,...は定数)のようにおける。このとき=/=/・,であり、一方=,であるからこれらを等置することによって,,...が得られ、中心多様体=が求まる。これを式(10・13)に代入した=,は中心多様体に沿った解挙動を表しこの式より非双曲型不動点の安定性が調べられる。上記の例では、==-1/2-α(10・15)=1/2-α(10・16)のように求まり、自明平衡点はα>1/2のとき安定、α<1/2のとき不安定であることがわかる。このように中心多様体を利用した解析は非双曲型不動点(分岐点)の安定判別および力学系の低次元化に有効である(この例では二次元の力学系[式(10・13)(10・14)]が一次元の力学系[式(10・16)]に低次元化された)。c. 懸垂(suspension)による分岐点近傍での力学系の低次元化懸垂の技法を用いると分岐点近傍での力学系の低次元化が実現できる。パラメータμ≡1-を状態変数と同様に従属変数とみなし力学系に自明な式μ=0(10・17)を式(10・13)(10・14)に加えた方程式系を新たな力学系を考える。
