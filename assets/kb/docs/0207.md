# 10・2・4 標準形(normal form)

> **章**: 第10章 分岐現象とカオス
> **節**: 10・9 のように中心多様体上に急速に落ち込みその後中心多様体に

適当な非線形座標変換によって非線形力学系を簡単にする(非線形項の数を減らす)ことを考える。以下のような線形部分をもつ非線形力学系が与えられたとする[はに関する非線形項].=+(10・19)上式を非線形変換=+(10・20)を用いて[はの成分から構成される二次以上のべき級数]以下のような方程式に変換することを考える。=+(10・21)ここではベクトルの成分から構成された二次以上のべき級数を表す。目的は、できるだけ非線形項を構成する項が少なくなるような、座標変換を導くことである。まず式(10・20)を時間で1回微分すると以下の式を得る。=+・=+++(10・22)一方は式(10・19)に式(10・20)を代入すると以下のようにも記述できる。=+++(10・23)ここで式(10・22)と式(10・23)を等置しの各べきの係数を適切に設定してできるだけのべき級数の係数を零にする。その結果得られた力学系は可能な限り非線形項が取除かれた系となり標準形と呼ばれる。なお取除けずに残った項は共振現象を発生する項(共鳴項と呼ばれる)であり、このような項はシステムに含まれる本質的な非線形特性を表現している項である。
