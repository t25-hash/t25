# 10・2・5 非線形ノーマルモード(nonlinear normal mode)

> **章**: 第10章 分岐現象とカオス
> **節**: 10・9 のように中心多様体上に急速に落ち込みその後中心多様体に

非線形ノーマルモードの概念は、従来の線形系に対するノーマルモードの概念を完全に包含するより一般的な理論であり、これによ2―85図10・10 対称な非線形ばね質量系図10・11 非線形ノーマルモードの分岐り非線形多自由度系に発生するモード局在化現象やモード分岐現象などの解析が可能になる。本稿では非減衰系のノーマルモードの定義[減衰系の非線形ノーマルモードについては文献(12)参照]を示した後、2自由度非減衰非線形系のノーマルモードを導出し、モード分岐とモード局在化に関して簡単に述べる。非減衰系に対するノーマルモードの定義は以下のとおりである。1. すべての質点は同じ周期で周期運動する。2. すべての質点は同時に静的平衡位置を通る。3. すべての質点の位置は任意の時刻において任意に選んだ一つの質点の位置によって決定される。図10・10に示す2自由度非線形系について考える。運動方程式は以下のようになる。=----(10・24)=----(10・25)定義の条件3より=(モード座標に相当する)を上式に代入すると、モード座標上の運動における質点の振幅比と非線形ばね定数との関係が以下のように求まる。-1+1-2-1+=0(10・26)したがって、>1/4のときは二つのモード(モード座標は=,=-)が存在する。<1/4のときは2自由度系であるにもかかわらず>1/4で存在したモードに、二つのモード(モード座標は=2-1+1-4/2および、=2-1-1-42を加えた合計四つのモードが存在することになる。したがって非線形連成の効果と各モードにおける質点の振幅比との関係は図10・11のようになる。図中の実線は安定なモード、破線は不安定なモードである。なお=1/4でのモードの増減をモードの分岐という。また<1/4で発生した二つの非線形システム特有のモードは、→0の極限において、またはのみが運動することを意味する。このような現象をモードの局在化(localization)という。
