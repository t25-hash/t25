# 10・3・2 カオスのメカニズム

> **章**: 第10章 分岐現象とカオス
> **節**: 10・3 カオス

点は横断的なホモクリニック点(homoclinic point)と呼ばれ、点を通過する軌道を横断的なホモクリニック軌道(homoclinic orbit)という。さらに、写像の連続性により、安定多様体あるいは不安定多様体の近くにある点はこれらの点の近くに移されていく。したがって、適当な回数、例えば、回だけ写像を作用させると、領域Δ上の点は、図10・14に示された逆U 字型あるいは馬蹄型の領域に移されることになる。これは、完全に確率的な現象と考えられるコイン投げの試行と同じである(コインを投げて表が出ると1を、裏が出ると2を割当て、過去と現在にわたって無限回試行すると、このような数列が得られる)。
