# 11・2・1 基礎的な統計量

> **章**: 第11章 不規則振動
> **節**: 11・2 確率変数

統計量としてよく用いられるものに、平均値(mean value)ま図11・1 不規則励振、系、応答の関係図11・2 ヒストグラムたは期待値(expected value,expectation)がある。確率量で表されるような変数を確率変数という。平均値または期待値は次式で与えられる。これを表すために、次式で表される二乗平均値(mean square value)もよく用いられる。分散の平方根をとったものは標準偏差(standard deviation)と呼ばれる。例えば、平均値まわりの三乗の平均値は次式で表されるひずみ度(skew-ness)の計算に用いられ、平均値からの偏り具合を評価するために使われる。一方、人間の身長のように連続した値をとる場合には連続的(continuous)であるという。以下ではは連続的である場合を扱う。
