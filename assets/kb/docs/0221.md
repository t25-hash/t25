# 11・4・3 不規則過程の種類

> **章**: 第11章 不規則振動
> **節**: 11・4 自己相関関数とパワースペクトル密度

a. 白色雑音またはホワイトノイズ(white noise)ω=である不規則波のことで、すべての周波数成分を均等に含むため非常に不規則性が強い。b. 狭帯域不規則過程(narrow band random process)ωの広がりが比較的狭い不規則波で、特定の周波数成分を多く含む。c. 広帯域不規則過程(wide band random process)ωの広がりが比較的広い不規則波で、多くの周波数成分を含む。図11・5にそれぞれの波形、自己相関関数、パワースペクトル密度関数の例を示す。
