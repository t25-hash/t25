# 2・3・1 ガス吸収の基礎

> **章**: 第2章 分離操作
> **節**: 2・3 ガス吸収

液体中へのガスの溶解度は吸収液の選択，吸収速度の計算などに
必要な基本的物性で，気相中の吸収される溶質ガスの分圧により液
中の溶質濃度すなわち溶解度が決まる．一般に溶解度は気相中の溶
質ガス分圧が高いほど大きく，温度が高いほど小さい．溶解度を分
圧の関数として表した曲線を溶解度曲線，または平衡曲線と呼ぶ．
一定温度で溶質の溶解度 が小さい範囲では溶質ガス分圧 は
に比例するが，これをヘンリーの法則（Henry’s law）といい，気
相，液相中の溶質の濃度表示により，式(2･47)～(2･49)のように書
かれ，ヘンリー定数（Henry’s law constant ）が異なる．ここで，
気相と液相は平衡状態にある．
①＝ (2･47)
：平衡にある気相中の溶質ガスの分圧 Pa
：平衡にある溶液中の溶質の濃度 kmol / m
：ヘンリー定数 mPa/kmol

：平衡にある溶液中の溶質のモル分率

されたい．
