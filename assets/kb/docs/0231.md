# 11・7・2 項

> **章**: 第11章 不規則振動
> **節**: 11・7 節では、定常雑音が式(11・115)を満たす場合につい

と同様に、次式によって表される1自由度振動系を考える。これらの値は、外力2―9 6機械力学がデルタ相関を有するホワイトノイズの場合とは異なり、応答の全履歴に依存する。さらに、応答の確率密度を仮定することにより、非ガウス応答解析への拡張が行われている。成形フィルタを導入することによって拡大システムは多自由度系となるので、次に述べる多自由度非線形系の解析法を適用して解析を行う。また、非定常応答解析を行う場合には、成形フィルタの過渡応答の影響を除去するため、まず成形フィルタのみで解析を行って、成形フィルタが定常応答に達してから、元の系を付加して解析を行うといった工夫が必要である。
