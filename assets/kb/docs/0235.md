# 11・8・2 パラメトリック励振を受ける多自由度系

> **章**: 第11章 不規則振動
> **節**: 11・7 節と同様に式(11・173)を状態変数表示すると、次式が得ら

入力が非白色雑音の場合のパラメトリック励振を受ける自由度系は一般に次式で表される。式(11・181)を式(11・154)と同様に状態変数表示すると、次式が得られる。振動入力を受ける系の破壊の様式として、破壊レベルを最初に通過した瞬間に破壊が起こる初通過破壊と、繰返し力を受けて破壊する累積疲労破壊がある。不規則振動入力では、破壊も確率論的に起こることになるので、信頼性を評価する場合に破壊確率を求める必要がある。特に初通過破壊確率は不規則振動入力を受ける系の信頼性を評価する重要な指標の一つである。
