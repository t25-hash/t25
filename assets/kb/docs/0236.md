# 2・4・5 金属イオンの抽出プロセス

> **章**: 第2章 分離操作
> **節**: 2・4 抽出

より－＝－となり，隣り合った段と段の間での二つの液相量の差は等しく一定である．
つまり，全物質の収支および溶質成分の収支より次式が得られる．
これより，一般にR点とE点を結んだ直線はすべてD点を通る．
共通点であるD点は操作点といわれ，F点とE点およびC点とR点とを結んだ二つの直線の交点として与えられる．
分離に必要な理論段数はE点から出発し，まずEと溶液平衡にあるR，すなわちタイラインERを求める．
次にD点とR点を直線で結び溶解度曲線上に点Eを与える．
この操作を繰返し，最終抽残液Rを超えるまで行う．
求める理論段数はタイラインの数で与えられる．
抽出操作は，すでに多くの金属イオンの分離操作として実用化されている．
周知のとおり，水溶液中に存在している金属イオンは，水和して電荷を有しているため，有機相中には単独では全く溶解しない．
したがって，これら金属イオンを有機相中に抽出させるためには，金属イオンの電荷を中和し，かつ配位している水分子を親油性の大きい分子で置換える必要がある．
抽出剤は，有機相に抽出される化学種の型により以下の三つの抽出系に大別される．
a.キレート抽出（chelating extraction）系 オキシム誘導体などキレート系抽出剤による金属イオンの抽出がこの系に属する．
