# 2・5・1 晶析の特徴

> **章**: 第2章 分離操作
> **節**: 2・5 晶析

晶析（crystallization）の目的は，まず第一に物質の分離・精製
である．晶析を用いると，固容体（後述）を形成する場合を除け
ば，原理的に一段の操作で純度100 %の製品が得られる．何段もの
操作を繰返して純度を上げる蒸留などとは，この点が大きく異な
る．また，操作温度が比較的低いことも，晶析の特徴である．この
ため，熱に弱い物質の分離・精製にも適している．このような優れ
た特徴をもつにもかかわらず，晶析は蒸留ほど広く使われていな
い．それは，晶析操作および晶析装置設計が非常に難しいためであ
る．その難しさは，通常晶析が懸濁系で行われることに起因する．
すなわち，結晶個数の複雑な変化をも考慮しなくてはならないから
である．しかし，晶析現象に対する理解も進み，最近では高度分離
技術として注目されてきている．
晶析のもう一つの目的は，結晶性粒子の製造である．この場合，
得られる結晶そのものが製品である．粉体粒子製造法としては粉砕
法があるが，その場合は粉砕の対象となる固体素材の存在が前提と
なるのに対して，晶析法はそのような前提は不要で，いわば，分子
原子レベルから固体粒子を製造する方法である．化学製品から医薬
品まで広い範囲で，晶析により粉体粒子が製造されている．製造法
を工夫することにより，製品にさまざまな特性を付与することも可
能である．これも粉砕にはない特徴である．
その目的が分離・精製あるいは粉体粒子の製造のいずれであれ，
晶析においてまず第一に問題となるのは，結晶純度，粒度分布，粒
子形状である．したがって，これらの制御が晶析の重要な課題であ
る．その他，医薬品などにおいては，結晶内の分子の並び方に起因
する結晶多形（2･5･2･c項）の問題，結晶格子の乱れに関係する結
晶性の制御も重要になる．これらは，医薬品の効き方に見られるよ
うな機能性に関係するからである．
ここでは，晶析に関する基本的事項を解説する．ただし，エレク
トロニクス材料としてのシリコンや光学結晶のような大型単結晶の
製造についてはふれない．分離・精製操作としての晶析および粉体
材料製造のための晶析について述べる．それも溶液における晶析を
対象とする．
