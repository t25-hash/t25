# 11・10・2 項の手順によって生成した疑似不規則入力のそれぞれに

> **章**: 第11章 不規則振動
> **節**: 11・7 節に示した不規則入力モデルのよ

対して応答の解析を行って、それらの応答の時刻歴の統計処理を行うことにより、不規則応答のシミュレーション結果を得ることができる。このそれぞれの応答の数値解析では、生成された疑似不規則入力を確定励振として扱うことになるので、通常の確定励振での応答解析法である、ルンゲ・クッタ法、ルンゲ・クッタ・ギル法、予測子修正子法などがよく用いられる。近年では、数値計算によってシミュレーションを行うことがほとんどである。文献(1) Risken, H., The Fokker-Planck Equation, 2rd ed., (1989)、Springer-Verlag,Berlin.(2) Lin, Y.K.
