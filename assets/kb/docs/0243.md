# 12・1・1 連続体とは

> **章**: 第12章 連続体の振動
> **節**: 12・1 連続体の振動概説

振動系の自由度が複数になると,各質点の位置,,...,욈を基点として運動中の位置(変位)を表す変数,,...,욈を必要とする.この系を自由度の系(多自由度系)と呼び,系は個の固有振動数と固有振動モード形をもつ.その概念を∞と拡張すると,質量は連続的に分布し,各位置の変位も連続的な関数により表現される.この場合,慣性を表す質量と剛性を表すばねを空間的に分離できず,すべての位置に両方の性質をもつ振動系となる.この系を連続体(continuoussystem)と呼ぶ.連続体のモデルには,復元力が張力で与えられる弦,膜と,復元力が曲げ剛性などの弾性力で与えられるはり,板,殻などの構造(要素)がある.これらの振動現象は,時間と空間の変数に関する微分方程式により記述される.連続体を三次元空間の弾性体とすると,その振動を解析するため,六つの応力,三つの変位,六つのひずみの力学量を導入して,これらに対する3本の平衡方程式,6本の変位-ひずみ関係式,6本の構成方程式を必要とする.そのうえで方程式と境界条件(拘束条件)を満足する解を定める.しかし三次元弾性論に基づいて,さまざまな境界条件を満足する数学的に厳密な解を得ることは一般に困難である.有限要素法など数値解法により近似解を得るときも,構造全体をつねに三次元問題として解くことは実用的ではない.そこで他の寸法に比べて肉厚が小さい,形が特定の曲率をもつなどの特徴から,低い振動数に卓越して現れる変形を考慮する.その変位の仮定に基づいて,はり,平板,殻など一次元,二次元の力学モデルへ還元する.
