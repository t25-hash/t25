# 2・5・6 晶析装置（crystallizer )

> **章**: 第2章 分離操作
> **節**: 2・5 晶析

ここでは，晶析装置に関する詳しい解説は行わず，典型的な装置
を紹介するにとどめる．晶析装置に関する詳細は，文献 を参照
されたい．
晶析装置は，大きく分けて連続装置とバッチ晶析装置に分けられ
る．バッチ晶析装置は，基本的にはかくはん機を備えたタンク式
で，例えば図2･48のような形 をしている．かくはん機にはい
ろいろなタイプがある．冷却によって過飽和状態を作る方式が多
い．
一方，連続式は，バッチ晶析装置とは違い，いくつかの種類
がある．図2･49に示したのは，DTB（draft - tube-baﬄed）晶析
装置である．名前のとおり内部にドラフトチューブとバッフルを備
え，ドラフトチューブの下部にかくはん機を備えて，結晶懸濁液を
循環させている．溶液は装置下部から供給され，結晶は供給液の上
昇流により分級されある程度大きくなったもののみが製品として取
出される．DTB晶析装置のドラフトチューブとかくはん羽根を改
良して，結晶の破損を少なくしたダブルプロペラ型装置 もある．
そのほか，外部に備えた循環ポンプで懸濁液を循環させ，装置内の
結晶を流動層状に保って成長させるクリスタルオスロ型の装置もあ
るが，ここでは装置図は省略する．連続装置の場合は，冷却よりも
蒸発濃縮によって過飽和状態を作り出す方式が多い．
