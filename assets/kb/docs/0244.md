# 12・1・2 連続体に現れる振動現象

> **章**: 第12章 連続体の振動
> **節**: 12・1 連続体の振動概説

連続体の振動は、離散系と同じく外力との関連から、自由振動、強制振動、その他の振動(自励振動等)の三つに分類される。自由振動(free vibration)は、運動中に外部の力が作用せず運動を続けるためのエネルギー補給がない。平衡(釣合い)位置からずらして物体を離すか、運動中に外力の作用を停止したときに生じる。連続体の自由振動解析では、離散系と同様に、系の特徴を表す固有振動特性(固有振動数、固有振動モード)に着目する。強制振動(forced vibration)は、強制外力や強制変位を与えられたときに発生して、初めは自由振動と混在するが減衰作用により自由振動は消滅して強制振動だけが残る。定常的な外力や強制変位に対して示す定常強制振動と、地震波や波浪波などの非定常的なランダム外力を受ける系の非定常強制振動がある。その他の振動には、バイオリン弦の摩擦や流体力による航空機翼の自励振動がある。振動の方向に関して、真直な棒には縦方向の(棒の)伸縮振動、横方向の(はりの)曲げ振動、(軸の)ねじり振動があり、単純な断面の棒の微小(線形)振動では互いの変形は非連成である。棒が平面内に曲率をもつと面内、面外、ねじり振動があり、コイルばねのように空間に三次元の曲率をもつ棒にはあらゆる変位に関して連成がある。平板では、厚さ方向に材料の対称性があると微小振動の範囲内で、面内振動と面外振動は非連成化される。厚さ方向に非対称な材料構成をもつ非対称板は面内と面外振動が連成する。形状に曲率をもつ殻では、材料構成にかかわらず各方向の変位が幾何的に連成する。
