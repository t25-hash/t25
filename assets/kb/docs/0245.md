# 12・1・3 連続体の振動解析

> **章**: 第12章 連続体の振動
> **節**: 12・1 連続体の振動概説

連続体の振動解析では,力学モデルの振動を支配する微分方程式を得た後,その数学的な解を導く.ここでは自由振動を中心として2―101非減衰系の固有振動数や振動モードを示す.工業的に問題となるのはおもに強制振動であるが,その応答も自由振動の特性に依存するため,自由振動解析は連続体の振動解析の主要な役割を果たす.一様断面をもつ真直棒(はり)では,縦振動,横振動,ねじり振動において,代表的な境界条件の組合せに対して厳密解が得られる.変断面や不均質材料など,それ以上の複雑さを含む問題では近似解法を利用する.等方性材料や直交異方性の長方形板では対向する二辺が単純支持の場合は,レビ解(Levy solution)と呼ばれる厳密解法が適用可能である.同様に等方性の円板や円環板では,ベッセル関数を用いた厳密解が得られる.殻では両端が単純支持の円筒殻や,長方形状の偏平殻では厳密解がある.このように連続体の振動解析では,特別な境界条件のとき厳密解が得られる.しかしさまざまな境界条件や一様でない肉厚など,問題が少し複雑化すると厳密解を得ることは不可能になる.この場合に適用可能な近似解法については12・5節において説明する.
