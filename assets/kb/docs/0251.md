# 12・2・4 項で考察したはり理論は、変形前に中立軸に垂直な平面

> **章**: 第12章 連続体の振動
> **節**: 12・2 に一致する。断面の図心とせん断中心が一致する限り、断面

は、変形後も中立軸に垂直な平面を保つというベルヌーイ・オイラー(Bernoulli-Euler)の仮定に基づく。このはりはオイラーはり(Euler beam)と呼ばれる。この仮定から、はりの厚さ方向のせん断変形は無視されるが、これは短く太いはりに対しては成立しない。そこで変形前の垂直な平面は、変形後に中立軸に新たな回転角度をもつ平面となると仮定を緩めて、平面の回転角度を新たな自由度とする。これに回転慣性の効果を含めると、チモシェンコはり(Timoshenko beam)の方程式옿옿옿옿-Ψ-ρ옿옿=0옿옿-Ψ +옿옿옿Ψ옿-ρ옿Ψ옿=0を得る。ここでΨ:面外せん断変形による角度変位、:断面積、:縦弾性係数、:断面二次モーメント、:断面形状により決まる係数で、クーパーによれば、長方形:=101+ν12+11ν、円:=61+ν7+6ν ,薄肉円:=21+ν4+νとが一定の一様はりについて方程式は、옿옿+ρ옿옿-ρ1+옿옿옿+ρ옿옿=0となる。両端支持の境界条件には厳密解があり、それを用いてチモシェンコはりとオイラーはりの固有振動数の比が計算できる。固有振動数はオイラーはりに比較して、新たな自由度の導入により低下して、せん断変形が卓越した振動モードも現れる。
