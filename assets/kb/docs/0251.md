# 12・2・4 項で考察したはり理論は、変形前に中立軸に垂直な平面

> **章**: 第12章 連続体の振動
> **節**: 12・2 に一致する。断面の図心とせん断中心が一致する限り、断面

は、変形後も中立軸に垂直な平面を保つというベルヌーイ・オイラー(Bernoulli-Euler)の仮定に基づく。このはりはオイラーはり(Euler beam)と呼ばれる。この仮定から、はりの厚さ方向のせん断変形は無視されるが、これは短く太いはりに対しては成立しない。そこで変形前の垂直な平面は、変形後に中立軸に新たな回転角度をもつ平面となると仮定を緩めて、平面の回転角度を新たな自由度とする。両端支持の境界条件には厳密解があり、それを用いてチモシェンコはりとオイラーはりの固有振動数の比が計算できる。固有振動数はオイラーはりに比較して、新たな自由度の導入により低下して、せん断変形が卓越した振動モードも現れる。
