# 2・7・10 浸透気化（pervaporation）による膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離（membrane separation）

浸透気化膜としては高分子膜とゼオライト膜が実用化されてい
る．図2･82には，実用化されている高分子膜（PVA膜）とゼオ
ライト膜（A型）を用いた場合の，エタノール/水系における分離
性能を示す．気液平衡と比べて，非常に高い水選択性を示し，アル
コール水溶液の共沸点付近での脱水プロセスで実用化されている．
一方，気液平衡以上のエタノール選択性を示す膜としては，シリコ
ンゴム膜やゼオライト膜（シリカライト）が報告されているが，実
用化はあまり進んでいない．輸送方程式としては， 成分の透過率
mol / m･ s ，成分 の供給液におけるモル分率 ，活量係数
γ ，透過側モル分率 ，および透過側全圧 ， 成分の飽和蒸気圧
を用い，溶解拡散モデルに基づいた次式
＝γ－ (2･216)
が有効である．
