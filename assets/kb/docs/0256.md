# 12・3・5 積層板の横振動

> **章**: 第12章 連続体の振動
> **節**: 12・3 膜・板の振動

複合材料(composite material)のうち、高分子材料を母材として連続繊維で補強する長繊維型のFRP(fiber reinforced plastics)は力学的異方性を有し、その層材を重ねて積層構造材料として利用する。積層板(laminated plate)では、繊維配向角度(材料主軸)が層内で軸となす角度をαとすると、層どうしの主軸を直交させるクロスプライ積層と、ある角度αと-αを交互に重ねるアングルプライ積層がよく利用される。積層する角度が中央面に関して非対称になると、板の面内変位と面外変位が連成するが、対称積層板では非連成化されて面外の横振動は独立に解くことができる。変位とモーメント、せん断力、等価なせん断力は等方性材料より複雑な式となる。表12・3は、周辺単純支持と固定の長方形単層板αの基本次数について、繊維配向角度αの変化に対する振動数を与える。
