# 12・3・6 ミンドリン板(Mindlin plate)の横振動

> **章**: 第12章 連続体の振動
> **節**: 12・3 膜・板の振動

オイラーはりに面外せん断変形を考慮してチモシェンコはり理論が導かれたように、薄板に有効な古典板理論を改良して面外のせん断と回転慣性を考慮したライスナー・ミンドリン板理論(Reis-sner-Mindlin plate theory)がある。横変位に加えて、軸、軸回りの角度変位Ψ,Ψを導入して横方向の3本の平衡方程式をまとめると、∇-ρ옿옿∇-ρ옿옿+ρ옿옿=0を得る。ここではせん断修正係数で、等方性板について=23,56,π12などが提案されているが、=56が多く用いられる。この方程式から、を含む項とρ12の項を省略すると、せん断変形と回転慣性の効果が省略され、古典板理論の方程式に一致す2―105る。
