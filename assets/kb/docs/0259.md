# 2・7・6 精密ろ過（microfiltration）の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離（membrane separation）

液相系ろ過でも精密ろ過では，分離対象とするのはおもに懸濁粒異なる．
膜（構造や細孔径など）および懸濁粒子（サイズ，濃度など）によるが，粒子は膜表面に堆積層（ケーク層）を形成，あるいりのろ過方式が用いられる．
液を膜透過させるものであり，図2･72の非透過に相当する流れがるために，処理量の増大とともに，ケーク層のろ過抵抗が増大し，ける濃縮工程に用いられている．
