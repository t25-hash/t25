# 2・7・8 ガス分離（gas separation）の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離（membrane separation）

ケーク層形成のために透過流束は徐々に低下するものの，クロスフ
ローでは膜面方向の流れ（非透過流）が存在するためにケーク層を
はぎ取ることが可能となり，やがて定常透過流束に達する．
(３) 逆洗を伴うろ過：クロスフローろ過において，間欠的に透
mを用いて，
過側から供給側に透過液あるいは加圧ガスを透過させることで，ケ
ーク層などのファウリング物質をはく離しながらろ過を行うもので
あり，高い透過流束を維持できる．
