# 12・4・4 回転殻の定常振動

> **章**: 第12章 連続体の振動
> **節**: 12・4 複雑な形状物体の振動

回転する殻(rotating shell)は、高速遠心分離機などに応用されている。回転殻の定常振動の解析は、一定の角速度により回転する殻の平衡状態から、始めに殻内部に生じる初期応力を求める。次に回転中の平衡状態を中心として微小振動する殻の運動方程式を、初期応力を考慮して導く。回転殻の振動問題が、他の静止した殻の振動問題と異なる点は、回転中の振動から生じるコリオリ力の効果により、回転方向に進む進行波(forward wave)と後退波(backward wave)の二つの移動波(traveling wave)が発生することである。合応力等の式と境界条件は12・4・3項と同じ式である。数値例は文献(21)に詳しい。
