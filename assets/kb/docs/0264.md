# 12・4・5 積層殻

> **章**: 第12章 連続体の振動
> **節**: 12・4 複雑な形状物体の振動

この場合の力学量、境界条件は12・4・2項の等方性偏平殻に、異方性による特性が含まれた式となる。全周に沿って単純支持されたクロスプライ偏平殻には12・4・2項に与えた厳密解が成り立つ。各要素の下線部分を省略するとドンネル理論と一致する。両端で単純支持されたクロスプライ円筒殻には12・4・3項に与えた厳密解が成り立つ。その一方向強化円筒殻(繊維配向角度がすべて軸方向)とクロスプライ円筒殻の数値例を、それぞれ表12・5,12・6に与えた。ここの材料定数は、=138GPa,=8.96GPa,=7.1GPa,ν=0.3である。等方性円筒殻と同様に、は周方向波数、(ドンネル)に示すドンネル理論の振動数は少し高い振動数を与える。
