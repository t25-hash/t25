# 12・4・6 直方体の振動

> **章**: 第12章 連続体の振動
> **節**: 12・4 複雑な形状物体の振動

その厳密な解は特別な境界条件に限られ、他の実用的な境界条件に対する解には級数解やエネルギー法が使われ、自由や単純支持、片持ちの境界条件をもつ弾性直方体の振動解析が行われた。固有振動数を求める自由振動解析では、時間的変化はsinωcos ωと仮定して、運動方程式から簡単に分離されるため、その後は空間座標に関する方程式だけを解く。このとき12・1~12・4節で述べた場合以外に、厳密な解は得られないことが多く、近似解法を用いる。連続体の近似解法は、無限自由度をもつ問題をその特性をなるべく失わないように離散化して、有限自由度の問題に還元することである。はりの自由振動は、一つの空間変数に関する常微分方程式により表されるため二点境界値問題の解法が適用できる。例えばべき級数解法の一つであるフロベニウス法(Frobenius method)、はりを区間に分割して区分多項式を使うスプライン解法(spline method)がある。また連立1階微分方程式に書き直した後に、数値積分により精度よい解が得られる伝達マトリックス法(transfer matrixmethod)、その条件により数値不安定が生じる欠点を改良した影響係数法がある。また残差に重みを乗じて積分を評価する重み付残差法の一つであるガラーキン法(Galerkin method)や、微分方程式を差分方程式化する差分法も使える。長方形板ではレビ型の条件を適用すると、残った対向辺の方向に変厚、異方性などの複雑さがあっても、二点境界値問題に還元されるので、はりの解法に適した方法が使える。またレビ型の解をうまく組合わせると解の重ね合わせ法(method of superposition)を適用して精度よく解を得られる。リッツ法(Ritz process)においては、領域全体を対象として全体の振動モード形を(幾何学的な)境界条件を満たす多数の関数で表し、その重ね合わせの係数を求める。これに対して、有限要素法(finite element method)は、解析対象の領域を多数の要素(有限要素)に分割して物理的近似を行い、要素内の力学的関係を定式化する。この後、領域全体について力学的関係を集積して全体系の方程式を数学的に厳密に解く解析方法であり、形状に対する適用性に優れている。これに対して数値解法のもう一つの有力な方法である差分法(finite difference method)は、現象を表す微分方程式はそのまま用いるが、微分を差分に置換えて扱う数学的な近似解法である。多自由度系の振動において、固有振動の直交性という基本性質がある。すなわちベクトルが直交する。無限自由度をもつ連続体では、ベクトルの直交性を一般化して、二つの固有関数を掛け合わせて、関数が定義された領域で積分したときに積分値が零となる。はりや板の連続体の強制振動では、変位を固有関数で展開して、右辺の外力項もその形に固有関数を乗じたものを積分する。すると固有関数の直交性がうまく使えて、応答を表す関数を求めるのが基本的な解析方法である。モード解析は、固有関数の直交性を積極的に利用する方法である。モード解析は、実験による周波数応答関数の測定技術の進歩とともに大きく発展した。文献(1) Timoshenko,S.P.、ほか2名、Vibration Problems in Engineer-ing,4ed.、(1974)、405,John Wiley& Sons.(2) 文献(1)の432-435ページ。(3) Cowper,G.R.,Trans.
