# 13・1・1 流体関連振動(flow-induced vibration)騒音の発生

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・1 熱・流体と構造系の連成振動の特徴

機構各種プラントでは,流体と構造の連成した振動だけでなく,流体そのものの振動や流れによって発生する騒音も観察される.これらは,流体構造連成および流体音響連成と呼ばれ,そのメカニズムは以下のような線図で表される(図13・1).非定常流,自由せん断流,後流,噴流など各種の流れが,流れの振動,圧力変動,速度変動を引き起こし,それらが弾性支持剛体や柔軟構造物,音響共鳴器等と連成して自励振動を引き起こす.また,連成作用が弱い場合は乱れとして構造系や流体系に強制振動を発生させる.
