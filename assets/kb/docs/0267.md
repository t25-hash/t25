# 13・1・2 励振メカニズムによる分類

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・1 熱・流体と構造系の連成振動の特徴

流れに注目して励振メカニズムを分類すると、以下のようになる。a. 外部励振流れの圧力変動が強制的外乱として振動体を励振する。ただし振動体の振動が流れに与える影響は無視できる。例として乱流による構造物の振動、往復動ポンプによる圧力脈動が挙げられる。b. 不安定励振流れの不安定性から誘発される交番的流体力が振動体を励振する。振動体の振動が流れに影響を与えない場合、すなわち流体力フィードバックがない場合と、流れに影響を与える場合、すなわち流体力フィードバックがある場合とがある。前者の例として、流体交番力による構造物や管内流体の強制振動、後者の例として、渦の放出による流体交番力による構造物の振動(同期共振)がある。c. 運動励起励振振動体の振動がフィードバック交番流体力を誘発し、さらに振動体を励振する。この場合、静的流体力特性に起因して交番流体力が誘起する場合と、非定常交番流体力が誘起する場合の2種類がある。前者の例として、ギャロッピング振動やフラッタがあり、後者の例として、ストールフラッタがある。これら励振メカニズムをまとめると、表13・1のようになる。図13・1 流体関連振動騒音発生機構表13・1 励振メカニズムの分類流れの不安定性による励振力運動励起流体力外乱渦放出渦と流体共振渦と振動体共振乱れた流れにおける圧力変動によって発生する流体力固定物体まわりの流れのはく離によって生じる渦の放出による流体力渦の放出による圧力変動と流体波動との共振から生成される流体力渦の放出と物体の振動との共振から生成される流体力流れの中での物体の運動によって励起される流体力2―109
