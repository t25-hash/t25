# 13・2・1 流体力

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・2 流体力および付加質量

流れの中に置かれた物体に作用する流体力(fluid force)は,一般に物体表面に作用する圧力と粘性応力の面積分=욡+σ욡욅′욅(13・1)で記述される.ただし,σ욡욅′は物体表面での粘性応力テンソル,욡=,,は,面の単位法線ベクトルの方向余弦である.,σ욡욅′は,物体表面の圧力と流速の速度こう配を用いて計算する.13章で取扱うような流体と構造系の連成振動(coupled vibra-tion)の場合,原理的には流体の質量と運動量の保存則を記述する方程式と,物体の運動を記述する方程式を連成させて解くことになる.その際,物体に作用する流体力は式(13・1)の形で物体の運動方程式に考慮され,流体の運動を記述する方程式には境界条件を介して物体の運動の影響が考慮される.さらに熱が加わる場合には,エネルギーの保存則を記述する方程式を連成させて解くことになる.しかし,このような方法で解析可能な場合は,流体の対流的加速度項が無視できるほどレイノルズ数が小さい場合の解析的な取扱い,あるいは対流的加速度項は無視できず,物体のまわりからの渦はく離はあるが,レイノルズ数はそれほど大きくなく全体の流れは層流である場合の数値解析に,目下のところ限られているようである.そこで現実の問題では,式(13・1)の基になる運動量の定理を直接用いて物体に作用する流体力を評価したり,実験的に物体の受ける流体力を求める.このようにして求まる流体力が,物体の振動方程式の加速度項,速度項,強制外力項さらに複雑な励振力項として作用する代表的な例を以下に示す.a. 強制外力として物体に作用する例(1) 容積型あるいはターボ型流体機械で発生する,回転速度や羽根数などに比例する振動数の励振力,流体の乱れによる不規則圧力脈動による励振力などで,近似的な取扱いが行われている.(2) 流出渦列など流体運動の不安定性に起因した圧力変動に伴う励振力で,カルマン渦による煙突の振動,エッジトーンなどがある.この場合,流体と物体の運動の両者を考慮に入れた数値計算も一部可能で,本来,流体運動も物体の運動の影響を受ける.しかし通常は,流体運動の変化は無視できる場合が多い.c. 係数励振力として物体に作用する例流体管路の場合,管軸が曲率によって生じる遠心力,管内流の脈動成分による遠心力の変動などが,管の横振動を励起させることがある.また管の横振動と管軸方向の流れの相互作用により生じるコリオリ力も管の横振動の安定性に影響を与える.この場合,流体運動は,一次元流れとして取扱われ,係数励振領域を求める場合などは,管路の運動に伴う流体運動の変化は無視してよい.d. 熱および二相流に係わる複雑な励振力として物体に作用する場合熱源と気流による管内気柱振動,液面振動の気泡による励振などがあり,実験的な解明が行われている.以上に挙げた例の中で,b,c,d,e項の一部は,本来,物体の運動と流体運動との連成現象が不安定なもので,その固有値の実数部分が負つまり線形の不安定現象としてとらえるべき問題もある.さらに振動発生後の状況あるいは外乱の大きさの影響を調べるには,非線形安定問題として取扱うべきである.しかし,目下のところ解決が困難なため,近似的な取扱いを行っている.
