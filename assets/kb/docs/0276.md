# 2・7・3 キー溝（key way）を有する軸のねじり

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・6 コイルばね

キー溝をもつ丸棒がトルクを伝達するときは純ねじりによるキー
溝底のせん断応力の集中とキーの伝達力による溝側面加圧による垂
直応力の集中が重ね合わさったものである．図2･13にねじりモー
メント のみによる溝底の集中せん断応力のαを示す．
最大せん断応力はキー溝底とすみの丸みの接点から丸みの側に約
15°離れたA点に生じる．基準応力を，キー溝がないときの丸棒の
最大せん断応力τ＝16πとすれば，α＝ττは図2･13のよう
になる．キー溝の幅 が大きいほど，深さ が小さくなるほど，
すみの丸みρが大きくなるほど，αは小さくなる．
