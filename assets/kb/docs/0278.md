# 13・4・5 平行流(parallel flow)

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・10 に示すようなとの関係が求められる。

直管、管群および弾性平板に流れが沿って流れる場合、および狭いすきまを介して二つの構造物が相対している場合に、このすきまを高速で流れるすきま流れによる振動を総称して平行流による振動と呼ぶ。直管、管群で流力振動が問題となる場合の多くは流れの乱れによるランダム振動であり、かなり流速が速い場合には、フラッタ(flutter)・ダイバージェンス(divergence)の不安定現象も発生する場合がある。一方、構造物の剛性が比較的低い弾性平板および構造物の振動に対する流体フィードバック力が大きくなるすきま流れにさらされる構造物では、比較的低い流速でもフラッタ・ダイバージェンスが発生する場合がある。a. 直管、管群のランダム振動(random vibration)外部平行流によるランダム振動の励振源には、対象物近傍の境界層で発生する乱れ(near-field flow noise)によるものと対象物の上流遠方で発生する乱れ(far-field flow noise)によるものとがある。near-field flow noiseによる励振源は圧力のゆらぎであり、壁面近傍の乱流境界層中の乱れが主要因である。一方、far-field flownoiseは流体脈動・流体中の構造物からの渦放出・曲り管などで発生する乱流・キャビテーションなどである。対象物近傍の境界層で発生する乱れは、その統計的性質を設計相関式として整理可能であるが、far-field flow noiseによるものはその励振源に依存して変化し、現状では簡便な設計相関式は見当たらない。near-field flow noiseにより励振されるランダム振動の評価法は、チェンとワムスガンスらによって提案されており、励振力である流体力には空間的な相関、すなわち軸方向および周方向の相関関係を考慮している。流体力の相互相関関数には、クロコスの壁面圧力場の現象論的モデル式を利用している。管壁面圧力相互相関スペクトル密度を定めた後でこれを用いてランダム振動理論に従って管の振動振幅rms値や振幅パワースペクトル密度を求めることができる。なお、手続きの詳細は参考文献(23)、(28)を参照。ほぼ比例することから、円柱の振動振幅rms値は流速の1.5乗から3.0乗に比例することが明らかになっている。また、管群においては、パイドウシスらによって当該の管とその周辺管群との連成の影響が理論計算と実験結果との比較により評価されている。b. 外部平行流による管および弾性平板の安定性(stability)外部平行流の場合の安定性については、ホーソンが初めてダイバージェンス型の不安定の発生を示し、その後パイドウシスが基礎式を導出し、解析的にフラッタおよびダイバージェンスの発生条件を調べるとともに、実験により検証を行っている。モデルをさらに詳細化したり、不安定現象の発生メカニズムを調べる研究等、1980年代まで多くの研究が行われた。基本的には外部平行流中の管の応答は、流体輸送管と、振動に対する流体力の応答や系全体の動特性が極めて類似している。図13・16のように周辺を支持され片面を高速流れにさらされる平板に生じる自励振動はパネルフラッタ(panel flutter)と呼ばれる。第二次世界大戦中、ドイツV-2ロケットが実験中にパネルフラッタのため多数事故を起こしたのは有名である。超音速パネルフラッタに関する理論研究は1950年代初頭より見られる。特に高速では空力加熱のためパネルに圧縮応力が生じてパネルフラッタが起きやすいことが知られており、この種の問題で平衡状態での面内方向応力が安定性に影響することが伺われる。また、事故例は特に報告されていないものの、空気に比べ密度がはるかに高い水の流れに片面をさらされたパネルの振動に関する研究も実施されている。解析方法は、平衡点周りの微小な調和振動を仮定し、外部流れをポテンシャル流れとして扱い、流体力をパネルの非減衰固有モードで展開したときの一般化力の形で表現して、複素固有値解析により安定判別を行う。図13・16の場合の安定性計算例を図13・17に示す。質量比が一定のもとで流速を上げていくとまずダイバージェンスが発生し、さらに流速を上げていくとフラッタが発生することがわかる。相対的に流体の質量が大きくなると臨界流速は低下する。c.
