# 13・4・6 環状流(annular flow)

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・10 に示すようなとの関係が求められる。

環状流に起因する流体構造連成問題は大きく分けて環状流路内を流れる高速流体によって発生するフラッタと、環状すきま流路内で発生するスロッシングと越流せき(overflow weir)との連成による自励振動に分けられる。a. 環状すきま流によるフラッタ狭いすきまを有する環状流路を挟んで対面する構造物が並進または回転の1自由度振動する場合、軸方向のすきま形状を自励振動が発生しにくい先細とすることが、自励振動を止めるのに有効である。これは一次元テーパすきま流路の場合と同様である。より正確な自励振動発生限界流速の評価を目指した研究例としては、李らの差分法による数値計算などがある。内筒がシェルモード振動する場合については、ジェットエンジンアフタバーナ熱遮へいシェル(図13・20)の自励振動があり、ジアダらはわずかに先細な片持コニカル殻を用いて再現実験を行い、環状すきま部の流速がある限界を超えると自励振動が発生し、内筒内部の流速は逆に系を安定化することを明らかにした。また理論解析研究は、同心二重円筒殻の流動励起振動に関してパイドウシスらが精力的に行っているが、メカニズムの検討はほとんど行われていない。b. 環状プレナム内流体のスロッシングと越流せきとの連成によるシェル振動(shell vibration)フランスの高速増殖炉スーパフェニックスの炉壁冷却系では、越流せきの自励振動が発生した。図13・21に示すように、これは、下端を支持された円筒せきの外面に沿って上昇する流体が上端部で越流し、円筒せき内面側のプレナムに流入する系において、プレナム内流体スロッシングと円筒せきオーバル振動が連成したものである。系の安定性は内側プレナム液面揺動と内側プレナムへの流入量変動の位相差に依存し、位相差を支配するのは越流量変動のせき振動に対する一次遅れと越流流体がプレナムへ落下するまでに要するむだ時間である。(a) FEM,BEM など(b) CFD(差分法、FEM など)図13・22 構造系と流体系のモデル化(modeling)
