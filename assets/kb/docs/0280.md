# 13・4・7 CFD の利用

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・10 に示すようなとの関係が求められる。

流体構造連成問題を数値計算コードを用いて解析する場合には，
流れ場と構造物の時間的変化を同時に扱えるプログラムの利用が便
利に思えるが，一般に，流れのみを取扱うプログラムのほうが流れ
場の解析には適していることが多く，構造振動を取扱うプログラム
でも同様のことがいえ，構造系と流体系とは別々にして図13･22の
ようにモデル化することが多い．
流れ解析プログラムでの解析は，時系列解析が中心で，各種パラ
メータの影響を検討し，一般的な知見を得るには時間を要する場合
が多く，見通しをもった評価をするのであれば，現状では，流れ解
析は定常流解析結果のみを利用し，構造系の振動に関しては，流体
関連振動分野の知見から評価するほうがよい．
流体構造連成問題は，はん用の解析プログラムが多数市販されて
いるので，これらを利用するのが便利である．ただし，すべての問
題に対応できるわけではなく，それぞれに得意分野があるので注意
が必要である．また，はん用プログラムは精力的にアップデートさ
れている場合が多く，次々に改良版が発表されることが多いので，
最新情報を集めて分析し利用方針を立てることも重要である．な
お，はん用解析プログラムの内容に関しては別途数値流体力学や有
限要素法の教科書を参照して，内容をある程度理解したうえで使用
する必要がある．数値流体力学で使われる代表的手法には，定常流
解析用の - ε法，非定常流解析用のLES（large eddy simulation）
法，DNS（direct numerical simulation）法などがあるが，計算
時間や必要とされる記憶容量に違いがあり，構造物との連成解析も
適合格子を使えるプログラムやそうでないために複雑な形状に対応
しにくいものなど，さまざまであるので注意が必要である．
13・4・8学会基準（JSME standard）
日本機械学会規格は研究開発推進センター標準事業部の監督のも
とに策定されている．そこでは2種類の委員会が活動している．一
つは常置委員会である発電用設備規格委員会である．もう1種類は
会員から規格制定の提案があったとき設置される委員会である．
発電用設備の構造設計に関する規格は，その守備範囲が広く，つ
ねに最新の知見を反映させようとするために常置委員会が不可欠で
ある．一方，流体関連振動に関する指針などは常時見直しが必要と
いうこともない．このため上記のような形態となっている．歴史的
には後者の形態の規格作りが先行している．
流体関連振動の関係では，これまでに二つの指針が制定されてい
る．一つはもんじゅの温度計破損事故を踏まえた「配管内円柱状構
造物の流力振動評価指針」 であり，もう一つは美浜2号機蒸気発
生器伝熱管破断事故とその後のR&Dを踏まえた「蒸気発生器伝
熱管U字管流力弾性振動防止指針」 である．これらの指針に従
い定められた手順どおりに評価を実施していけば，必ずしも振動の
専門家でなくとも振動発生を回避できるように作られている．
