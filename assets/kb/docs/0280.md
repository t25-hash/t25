# 13・4・7 CFD の利用

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・10 に示すようなとの関係が求められる。

流体構造連成問題を数値計算コードを用いて解析する場合には、流れ場と構造物の時間的変化を同時に扱えるプログラムの利用が便利に思えるが、一般に、流れのみを取扱うプログラムのほうが流れ場の解析には適していることが多く、構造振動を取扱うプログラムでも同様のことがいえ、構造系と流体系とは別々にして図13・22のようにモデル化することが多い。流れ解析プログラムでの解析は、時系列解析が中心で、各種パラメータの影響を検討し、一般的な知見を得るには時間を要する場合が多く、見通しをもった評価をするのであれば、現状では、流れ解析は定常流解析結果のみを利用し、構造系の振動に関しては、流体関連振動分野の知見から評価するほうがよい。流体構造連成問題は、はん用の解析プログラムが多数市販されているので、これらを利用するのが便利である。ただし、すべての問題に対応できるわけではなく、それぞれに得意分野があるので注意が必要である。また、はん用プログラムは精力的にアップデートされている場合が多く、次々に改良版が発表されることが多いので、最新情報を集めて分析し利用方針を立てることも重要である。なお、はん用解析プログラムの内容に関しては別途数値流体力学や有限要素法の教科書を参照して、内容をある程度理解したうえで使用する必要がある。数値流体力学で使われる代表的手法には、定常流解析用の-ε法、非定常流解析用のLES(large eddy simulation)法、DNS(direct numerical simulation)法などがあるが、計算時間や必要とされる記憶容量に違いがあり、構造物との連成解析も適合格子を使えるプログラムやそうでないために複雑な形状に対応しにくいものなど、さまざまであるので注意が必要である。
