# 13・5・1 流れの中の非円形断面柱自励振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・5 翼と翼列の自励振動

流れと直交する構造物の断面が非円形のときは、抗力のほかに揚力も作用するため、円柱では生じない不安定現象を生じる。これは(1)迎え角増加とともに揚力が減少する領域で生じる失速フラッタ(stall flutter)、(2)構造物の曲げ振動とねじり振動が連成するため生じる曲げねじりフラッタ(classical flutter)、(3)翼群において翼間の流れの変化による非定常力が発生させる翼列フラッタ(cascade flutter)、に分けられる。(1)は単独の1自由度の構造物でも生じ、(2)は2自由度以上の構造物、(3)は翼列にのみ生じる。実際にはこれらの複合現象も生じる。翼列では、失速部分が回転方向に伝ぱしていく旋回失速(rotating stall)という現象もある。なお、翼と翼列の振動についてはα4編流体工学やγ2編流体機械も参照されたい。
