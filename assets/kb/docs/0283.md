# 13・5・2 失速フラッタ

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・5 翼と翼列の自励振動

1自由度系の振動である。振動モードが曲げ振動(翼断面の並進運動)の場合とねじり振動(翼断面の回転運動)の場合があるが、以下では曲げ振動を例にとって説明する。なお、曲げ振動の一例である着雪電線の振動などでは本現象をギャロッピング(gallop-ing)と呼ぶ。図13・23のように翼の一断面を切り出して考える。振動方向には復元力-と揚力が作用する。揚力は迎え角αの関数となるが、流速の流れと直角方向に運動している翼の実際の迎え角とその際の揚力はそれぞれα-(α:静止時の迎え角)、α-옿옿αとなるので、運動方程式は、+옿옿α+=α(13・45)と書ける。静止時の迎え角がαの増加とともにが減少する条件(失速状態)にあるとき、左辺第2項は負性抵抗として作用し、自励振動が発生する。
