# 13・5・4 翼列フラッタ

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・24 の翼の曲げ運動,ねじり運動は次式で記述できる.

翼列が振動するとき,翼列間隔の変化により流況が変化する.これが翼に働く流体力を変化させ,条件によっては自励振動を発生させる.一般には,各翼は同一振幅,隣接翼間で一定の位相差δをもった振動をすると仮定できる.図13・25のように着目する翼以外の翼をその位置の渦で置換える.ポテンシャル流れとして,当該翼位置の流れ(循環)の変化は上下の翼との距離の変化と上下の翼まわりの循環の変化の影響の和で表せる.これから当該翼の揚力の変動を翼長と翼間距離の比,翼列食違い角θ,位相差δの関数として計算できる.これが翼の運動方向に働くとき,翼列は不安定となる.このモデルは翼が失速状態に入る場合には適用しにくい.失速状態にも拡張できるモデルも提案されている.
