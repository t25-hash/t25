# 13・5・5 旋回失速

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・24 の翼の曲げ運動、ねじり運動は次式で記述できる。

なんらかの原因により翼列の一部分だけで失速が起きたとする。失速した部分では流れが翼からはく離するため抵抗が大きくなり、流れは翼列の他の部分に回り込む。回り込む流れは図13・26に示すように隣接する翼の一方では迎え角を大きくし、反対側では小さくする。迎え角が大きくなった翼は新たに失速し、反対側では失速を回復させる。このため失速部分は一方向に伝ぱしていく。これを旋回失速という。大型液体貯糟、タンカー、ロケットの燃料タンク等に代表されるように、内部に液体を有する構造物は多数存在する。容器内液体の挙動を把握することは、構造物の安全性を向上させるために重要な課題となっており、多数の研究がなされている。図13・25 翼列フラッタの解析モデル図13・26 旋回失速の発生機構2―120機械力学
