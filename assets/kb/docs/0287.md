# 13・6・1 容器内の流体の固有振動数

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・6 容器内の流体の振動

a. 剛体容器内の流体の弾性振動剛体容器内に満たされた流体の弾性振動による固有振動数に着目する。微小振動を仮定すると速度ポテンシャルφ、圧力、は波動方程式を満たす。固定境界では境界に直角な速度成分が零、自由境界では圧力の変動分が零になる。代表的な容器形状内での流体の弾性振動の固有振動数を表13・6に示す。図13・27に示すように、容積が断面積の小さい開管の一端に取付けられているとき、容積内の流体は管内の流体柱が剛体ピストンとして運動する場合の固有振動数をもつ。これはヘルムホルツ共鳴器(Helmholtz resonator)と呼ばれる。b. 容器内液体の液面振動容器内の液体が自由表面をもつとき、平衡位置から液面が移動すると重力による復元力が作用する。この復元力と液体の慣性で振動系を構成する。着目している液面が、水平を保ったまま、振動するとき、液体の振動を1自由度系とみなせる場合がある。表13・7にこの1自由度系の例と固有振動数を示す。自由液面が広がりをもつ場合には、液面の振動を考慮する必要がある。この液面振動はスロッシング(sloshing)と呼ばれ、顕著な非線形性を示すことが知られている(自由液面の境界条件に非線形性が存在する)。微小振動を考えたときには、ラプラスの方程式を満たす速度ポテンシャルを解くことにより、さまざまな容器内液面振動の固有振動数を求めることができる。図13・29に代表的な液面振動モードを示す。円周方向一次以上のモードは、同じ固有振動数を有する直交する二つのモード(cosとsinの関係)が存在する。代表的な液面振動のモード形状を図13・30に示す。また、低重力下での液面振動の固有振動数も調べられている。
