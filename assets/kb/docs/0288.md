# 13・6・2 スロッシング特性

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・6 容器内の流体の振動

a. 水平方向の加振を受ける場合内部に液体を有する容器を水平方向に加振する場合には、すべての液面振動モードが励振されるわけではなく、励振力の直接作用しないモードが多数存在するという特徴がある。励振されるモードは直方体容器では奇数次のモード[例えば、図13・29(a)(c)等]、円筒容器では円周方向一次(節直径が1本)モードのみである。励振されるモードは微小振動の範囲では、内部液体は上述の固有振動数で共振する。応答振幅が増加すると非線形性が顕著に現れる。直方体容器内液面の応答例を図13・32に示す。円筒容器や球形の軸対称容器の場合には共振領域で液面に回転揺動が生じる場合がある。円筒容器内の液面振動では、直交する二つの円周方向一次モード間には、見掛け上の互いに係数励振を引図13・31 球形容器内液体の固有振動数(:振幅、ω:線形解共振振動数)図13・32 直方体容器液体の非線形応答2―122機械力学(c) 1,1モードの回転揺動図13・33 円筒容器内の回転揺動き起こすような非線形連成がある。この連成により、直接励振される一方のモードが直接励振されない他方のモードを励起し、両方のモードが同時に成長すると、節直径が回転する回転揺動を引き起こす場合があることが指摘されている。この概略を図13・33に示す。図13・33(a)(b)は、同じ固有振動数を有し、空間的に直交する振動モードである。円筒容器を方向に励振した場合、直接励振力が作用するのは図13・33(a)のモード(図中の太線は節直径)で、(b)のモードは直接励振されない。上述の非線形連成により(b)のモードが成長する場合があるため、図13・33(c)のように節直径が回転する(節直径が回転するためには、二つのモードの振動に位相差が必要)。液面振動の非線形性により、直接励振されないモードに対して、直接励振されるモードが、振幅の二乗の形で連成し、見掛け上の励振力として作用して、内部共振(internal resonance)が生じ、直接励振されないモードが成長する場合がある。この内部共振の一例として、円筒容器の場合には、容器半径で無次元化した液深0.3付近で、円周方向一次・半径方向一次モード固有振動数近傍で加振すると、円周方向零次・半径方向一次モード[図13・30(a)]が内部共振により成長する。これは、円周方向零次・半径方向一次モードの固有振動数が円周方向一次・半径方向一次モードの固有振動数の2倍近傍になるためである。さらに、直方体容器や円筒容器内液面の振動以外にも、二重円筒容器や多液面を有する容器内の液面振動も調べられている。b. 鉛直方向の加振を受ける場合鉛直方向の加振を受ける場合には、液面振動の復元力である重力が変動したことに対応するので、係数励振系の応答となる。水平加振時とは異なり、すべてのモードが係数励振を受ける。固有振動数の2倍の振動数で加振された場合に液面振動が生じる。応答の例を図13・35に示す。共振領域から離れると液面振動は生じないが、加振加速度をさらに増加すると、気泡の形成や液滴の飛び出しが生じる。ただし、回転方向の励振を受ける場合には、励振軸と静止液面との距離等により、共振点近傍の励振力が作用しても、液面振動が生じない場合があることが指摘されている。
