# 13・6・2 スロッシング特性

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・6 容器内の流体の振動

a. 水平方向の加振を受ける場合内部に液体を有する容器を水平方向に加振する場合には,すべての液面振動モードが励振されるわけではなく,励振力の直接作用しないモードが多数存在するという特徴がある.励振されるモードは直方体容器では奇数次のモード[例えば,図13・29(a)(c)等],円筒容器では円周方向一次(節直径が1本)モードのみである.励振されるモードは微小振動の範囲では,内部液体は上述の固有振動数で共振する.応答振幅が増加すると非線形性が顕著に現れる.液深が小さい場合には漸硬ばね,大きい場合には漸軟ばね特性を示し,その境界は液深をとすると,直方体容器では加振方向の容器の長さをとすると0.34,円筒容器の場合には半径をとすると0.5である.直方体容器内液面の応答例を図13・32に示す.円筒容器や球形の軸対称容器の場合には共振領域で液面に回転揺動が生じる場合がある.円筒容器内の液面振動では,直交する二つの円周方向一次モード間には,見掛け上の互いに係数励振を引図13・31 円筒容器内の回転揺動き起こすような非線形連成がある.この連成により,直接励振される一方のモードが直接励振されない他方のモードを励起し,両方のモードが同時に成長すると,節直径が回転する回転揺動を引き起こす場合があることが指摘されている.この概略を図13・33に示す.図13・33(a)(b)は,同じ固有振動数を有し,空間的に直交する振動モードである.円筒容器を方向に励振した場合,直接励振力が作用するのは図13・33(a)のモード(図中の太線は節直径)で,(b)のモードは直接励振されない.上述の非線形連成により(b)のモードが成長する場合があるため,図13・33(c)のように節直径が回転する(節直径が回転するためには,二つのモードの振動に位相差が必要).直方体容器でも,容器底面の縦横比が整数になる場合には,同様のメカニズムが指摘されており,例えば図13・34に示すように,容器底面の縦横比が2の容器が軸方向に励振された場合,図13・34(a)に示す方向一次・方向零次モードが直接励振されるが,非線形連成により,直接励振力の作用しない図13・34(b)に示す方向零次・方向二次モード(このモードは方向一次・方向零次モードと同じ固有振動数である)が成長し,結果的に液面振動は図13・34(c)のような形状になり,二次元的な振動が三次元的な振動に遷移することが指摘されている.液面振動の非線形性により,直接励振されないモードに対して,直接励振されるモードが,振幅の二乗の形で連成し,見掛け上の励振力として作用して,内部共振(internal resonance)が生じ,直接励振されないモードが成長する場合がある.この内部共振の一例として,円筒容器の場合には,容器半径で無次元化した液深0.3付近で,円周方向一次・半径方向一次モード固有振動数近傍で加振すると,円周方向零次・半径方向一次モード[図13・30(a)]が内部共振により成長する.これは,円周方向零次・半径方向一次モードの固有振動数が円周方向一次・半径方向一次モードの固有振動数の2倍近傍になるためである.さらに,直方体容器や円筒容器内液面の振動以外にも,二重円筒容器や多液面を有する容器内の液面振動も調べられている.b.
