# 13・7・2 浮体に作用する流体力

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・7 浮体の運動

浮体に対して、規則的に働く流体力として、静水圧に基づく静的復元力、浮体が運動することによって生じるラディエイション流体力(radiation force)、波による強制力、および、粘性流体力等が作用する。また、浮体に前進速度がある場合には、前進速度による流体力、浮遊構造物の場合には、係留による力等も考慮する必要がある。以下で代表的な流体力について概説する。傾斜角φが小さいときには、傾く前の浮心B を通る浮力線とB′を通る浮力線は一定な点で交わる。この点をメタセンタ(metacenter)と呼ばれる。ここで、ρは流体の密度、は重力加速度、は直立時の水線面積(浮体の静水面による断面の面積)である。静的復元力は、上下揺れ、横揺れおよび縦揺れに対して、復元力または復元モーメントとして作用する。b. ラディエイション流体力(付加質量力と造波減衰力)浮体が運動するとき、浮体に付随して浮体とともに振動する波と浮体から離れて遠方に拡散していく2種類の波が発生する。これらの波から受ける流体力をラディエイション流体力と呼び、浮体の加速度に比例する成分(付加質量力)と速度に比例する成分(造波抵抗力)に分けて表す。c. 波強制力波強制力はフルード・クリロフ力(Froude-Krylov force)とディフラクション力(diffraction force)に分けられる。前者は入射波が浮体の存在によって乱されないと仮定して、入射による圧力を仮想的な浮体表面で積分して求める流体力、後者は流体が浮体表面で妨げられて発生する散乱波による流体力である。d. 粘性流体力船舶のような浮体では、造波抵抗力に比べて粘性減衰力は小さいので、横揺れ運動以外は無視でき、横揺れ運動に対して非線形の減衰力として作用する。e. 流体力の計算法船体のような浮体の場合、浮体を構成する部材を二次元の細長体と見て、各断面をストリップ状に輪切りにして各断面に作用する流体力を求め、これを積分して三次元浮体に働く流体力とする方法を、ストリップ法(strip method)と呼び、OSM(ordinary strip method)、NSM (new stripmethod)等が開発されている。また、境界要素法等の手法を用いた流体力の計算法も開発されている。
