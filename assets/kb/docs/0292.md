# 13・7・3 浮体の運動方程式

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・7 浮体の運動

浮体の慣性力と、考慮する流体力を等しいとして、一般に以下のように表される。욡욢+욡욢욢+욡욢+욡욢욢+욡욢욢∑=욡+욡=1,2,...,6(13・58)ここで、욡욢:構造物の質量または慣性モーメント、욡욢:付加質量または付加慣性モーメント、욡욢:造波抵抗係数、욡욢:粘性による減衰係数、욡욢:係留力を含む復元力、または復元モーメント、욡:粘性によらない波強制力、またはモーメント、욡:粘性による波強制力、またはモーメント。なお、욡욢および욡に非線形性が含まれている。図13・38 浮体の傾斜と復元力2―124機械力学
