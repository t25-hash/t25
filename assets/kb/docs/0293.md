# 2・4・1 直接法（direct method）

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

直接法の基礎はガウスの消去法である．対称行列，非対称行列を
問わず100元以下の行列であれば，ガウスの消去法は強力で高速な
解法である．しかし，1000元以上の巨大行列に対しては，より有
効な解法が考えられている．
対称行列の解法としてはコレスキー法または改良コレスキー法が
有効であり，非対称行列の解法としてはクロー法がよい．コレスキ
ー法は行列の対称性を考慮してLU分解する方法で，ガウスの消去
法よりも速い．改良コレスキー法は対称行列をLDU分解するもの
で，コレスキー法の改良版になっている．クロー法は非対称行列を
LDU分解して解く方法で，対称行列の改良コレスキー法に対応す
る．
