# 13・8・2 熱応力(thermal stress)による構造物の振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・8 熱・気泡・相変化に起因する振動

はりや弦に熱応力により振動が発生する条件は,振動の運動の方向と熱応力の方向が一致することである.伝熱には位相遅れがあるので,準定常状態を仮定して得られる方向とは逆方向に熱応力が働くことが多いので注意が必要である.図13・39,13・40に自己発熱のあるはりや弦の自励振動の例を示す.図13・41のように外部熱源に近接して置かれたはりや弦,太陽からの受熱量がたわみ角により変化する人工衛星アンテナも振動する.これらでははりや弦自身の固有振動数での振動が励起されるが,伝熱状態の変化を引き起こす機構によっては系の熱容量で決まる長周期の振動を生じることもある.必ずしも熱応力による振動ではないが,弁のリーク量変化と弁体の熱膨張が連成した振動,熱交換器胴の変形と内部流体の温度分布変化が連成した振動なども報告されている.
