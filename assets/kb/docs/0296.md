# 13・8・3 燃焼振動(pulsating combustion)

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・8 熱・気泡・相変化に起因する振動

燃焼室あるいは燃焼室と燃料や空気の供給流路などからなる系において,その系の固有周波数の音が発生することがある.この燃焼振動は,工業炉,ボイラ,ガスタービン燃焼器,ジェットエンジンのアフタバーナ等でしばしば観測され,騒音問題だけでなく構造物の振動そして機器の損傷を引き起こす場合がある.燃焼振動の発生には,エネルギーを供給する燃焼発熱反応と振動系の存在,そしてそれらのフィードバック結合が必要である.発熱率の変動による発熱量変動が系の振動を助長するように作用し,それが振動系の減衰を上回ると不安定となり,燃焼振動が発生する.燃焼振動は振動モードに着目すると次の二つに分類できる.a. ヘルムホルツ型燃焼振動燃焼室とそれ以外の系が連成して起こる連成振動モードで,周波数は比較的低く,低周波燃焼振動(a) 片持ばり:運動前面収縮(b) 両端固定ばり:運動前面膨張図13・39発熱のあるはりの熱応力による振動図13・40 発熱のある弦の熱応力による上下振動(b) 外部熱源による弦の振動(a) 外部熱源によるはりの振動図13・41 外部熱源で生じる熱応力による振動とも呼ばれる.燃焼室と燃料や空気の供給流路などからなる系全体の振動で,燃焼室内の圧力を集中定数で近似できるヘルムホルツ共鳴器(Helmholtz resonator)のようなモードの振動となる.図は燃料供給量により変化するフィードバック系となっている.燃料が燃焼室内に噴射された後,熱を発生するまでには有限の時間が必要で,これを燃焼の時間遅れと呼ぶ.燃焼の位相遅れのため,燃焼室の圧力変動による発熱量変動が圧力変動を増幅する位相となると,発振に至る.燃焼の時間遅れには燃料と空気の混合に要する時間なども含まれる.b. 音響型燃焼振動振動モードは燃焼室内の音響定在波で,燃焼室の外部との連成は小さい.周波数は比較的高く,高周波燃焼振動とも呼ばれる.また熱気柱共鳴とも呼ばれるが,気柱は共鳴管の働きだけをするわけではない.音響振動の発熱量変動へのフィードバックが重要な役割を果たしている.図13・43(a)のように単純管に火炎を挿入したとき生じるシンギングフレーム(singingflame)は古くから知られる現象である.基礎研究ではこのような軸方向低次の音響モードが扱われることが多い.一方,ジェットエンジン等では燃焼装置断面内のモードの非常に高周波の燃焼振動が発生する場合がある.発振条件を考える際には燃焼室内の圧力変動と発熱量の変動の空間分布の考慮が必要である.次式のレイリーの条件を満たすとき機械的エネルギーが生成され,これが減衰による散逸量を上回れば発振する.>0(13・59)シンギングフレームの火炎を図13・43(b)のように熱した金網等に置換えても発振する.気柱型燃焼振動の多くは燃焼反応より熱源の存在が主要な役割を果たすもので,13・8・4項の熱駆動気柱振動の一種である.
