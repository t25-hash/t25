# 13・8・4 熱駆動気柱振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・42 において燃料供給量は燃焼室圧力により変化し,燃焼室圧力

図13・43(a)および(b)では上昇気流が存在するので管の上下の区別がある.振動を生じるのは図13・44の色塗り位置に熱源が存在するときである.普通は発生しうる最も低い周波数の振動を生じる.図13・43(c)の場合に存在するのは下降気流なので,気流方図13・42 ロケットエンジン概念図(b)(c)(d)(e)(a)図13・43 各種の熱駆動気柱振動図13・44 熱駆動気柱振動が発生する高温熱源位置向は逆になるが,やはり図13・44の色塗り位置で発振する.図13・43(d)のようにヒータとクーラが接近してある系では,気流がなくとも発振する.なお,(d)でヒータとクーラの位置が逆のときや,(a)~(c)で熱源位置が色塗り位置以外にあるときは,その振動モードを強く制振する.気柱振動では圧力変動と速度変動の位相が90°ずれていて,圧力変動の節から腹の間で腹側への速度大のとき圧力は上昇中である.この瞬間に温度低・密度大になるなら,機械的エネルギーが生成され発振する.図13・43(e)のように極低温気体で両端の絶対温度の比が大きいときは,壁からの入熱だけで発振を生じる.熱駆動気柱振動は熱音響工学として研究され,図13・43(d)の体系に外部から音波を加えてクーラ位置に冷熱を作り出す冷凍機への利用も試みられている.
