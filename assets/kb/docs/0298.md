# 13・8・5 気泡による液面振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・42 において燃料供給量は燃焼室圧力により変化し,燃焼室圧力

容器内の液中に気泡を吹込むとき,液面が振動することがある.図13・45(a)のように吹込み口が液面振動の節の位置にあるとき,液の振動によって気泡が流される側に,より大きな浮力が作用する.これが液の振動を助長するように作用するなら,振動が自励的に発生する.二つのタンクを下部で連結するV 字管に気泡を吹込むときも同様の原理の振動を生じる.一方,図13・45(b)のような腹の位置からの吹込みによる振動は,気泡吹込み量の変化が生じさせる浮力の時間変化が原因である.
