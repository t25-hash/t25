# 2・5・2 予測子・修正子法（predictor - corrector method）

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

気象学の分野ではこれを松野法と呼んでいる．
くなり，より大規模な問題への適用が可能となる．
ゴリズムは図2･18に示すとおりである．
計算の反復部分として，内側ループ（inner loop，計算の繰返し部分として，適当な反復回近似解を，初期近似解として再スタートさせる部分）がある．
マトリックスフリー法では，反復法内部における要素ごとの行列とベクトルの積の重ね合わせを行う．
そして，このループ内で要素ごとの行列を作成する．
