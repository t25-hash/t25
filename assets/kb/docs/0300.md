# 13・8・7 蒸気の凝縮による振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・53 に示すような並列管においては全流量は一定でも管と管で流

a. 静止液中への蒸気吹込みに伴う振動容器内の液中に蒸気を直接吹込み、凝縮させる。この技術は原子炉の圧力抑制型格納容器や簡易凝縮器に利用されている。このとき激しい圧力変動が発生することがある。この様子を図13・54に示す。蒸気流量が非常に小さければ、蒸気は管内で凝縮を完了し、管内の液面は上下動するものの大きな圧力変動は生じない。流量がある程度大きくなると、管出口の蒸気泡生成、その急激な凝縮と管内の負圧発生、液の管内流入と圧力回復を間欠的に繰返す。これをチャギング(chugging)と呼ぶ。蒸気流量が非常に大きくなると、周期的圧力変動が観測される。これを凝縮振動と呼ぶ。このとき管外の気液界面は激しく振動している。この液体プラグ位置が振動し圧力も変動する。この現象もチャギング(chugging)と呼ばれる。蒸気流量が小さいときは液体プラグ上流側界面位置は注入点の上流と下流の間を往復し、周期は比較的長く、圧力振幅も大きい。蒸気流量の増加とともに界面位置は下流に移動し、注入点の下流だけで振動するようになる。周期は短く振幅は小さくなり、ある蒸気流量以上では界面は振動しなくなる。c. サブクール液中にトラップされた蒸気凝縮による水撃管内を流れるサブクール液中に蒸気がトラップされると、ボイド消滅による水撃現象を発生する。図13・55 蒸気流中へのサブクール液注入時の液体プラグの振動図13・54容器水中での蒸気凝縮に伴う振動d. 管内凝縮二相流の振動管内凝縮二相流においても凝縮液膜の界面波の成長、凝縮量の変化が引き金となる上記b項のような振動が発生することがある。
