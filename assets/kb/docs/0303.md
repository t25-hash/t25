# 13・9・2 パッシブ制御(passive control)

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・9 流体を含む系の制振(flow-induced vibration con

流体そのものが振動するシステムを対象とする場合の例として、容器内で発生するスロッシングを防振するために中央部に隔壁を挿入して制振する方法や逆U 字管を利用したダイナミックダンパを構成して制振する研究が行われている。流体振動現象を利用した構造物用のパッシブ制振装置の代表例はスロッシングダンパである。長周期構造物用のダイナミックダンパとして、容器内の液体の同調運動を利用した同調液柱管ダンパ(TLCD)や同調液体ダンパ(TLD)がある。TLCD は任意形状の液柱管内の液体の往復運動と液柱管内部に設けたオリフィスの減衰により制振効果を発生させている。一方、TLD は容器内のスロッシングを利用したダンパであるが、非線形分布定数系であるため取扱いが面倒である。このダンパについても、最適な減衰を得るためには、TLCD のオリフィスに相当するエネルギー散逸要素が必要となる。最適減衰を得るための方策は、容器底面の表面粗さを増す、作動液体の粘度を増す、容器の上面に蓋あるいは反射板を付けて砕波を起こす、容器の中に抵抗となるような十字の断面をもつ棒やネットを挿入するというものである。ネットを要素として挿入するタイプのダンパは、浅水波型スロッシングダンパおよび深水波型スロッシングダンパの二つのタイプがあるが、両方について減衰性能を理論的に予測することが可能である。図13・56には、深水波型スロッシングダンパを斜張橋頂部に設置した場合の設置図を示す。磁石を利用したパッシブダンパとしては、図13・57に示す斜張橋ケーブル制振用ダンパが開発されている。レインバイブレーション、ウェークギャロッピングの抑制を目的として、磁石と金属板との衝突を利用することにより低次モードで発生するケーブル振動を安定な高次モードへと変換し、発生する空力減衰を負から正に変えることで風による自励振動の抑制が可能である。
