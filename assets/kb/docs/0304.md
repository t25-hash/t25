# 13・9・3 アクティブ制御(active control)

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・9 流体を含む系の制振(flow-induced vibration con

アクティブ制振の研究分野の起源は,1970年代にNASA で行われた弾性翼のアクティブフラッタサプレッションに遡ることができる.このころ,原子力をはじめとする分野で,各種の流体関連振動の発生機構が構造物の振動によって発生する流体からのフィードバック力によっているという事実が認識され始めた.それ以降,振動解析から得られた知見に基づいて,さまざまな系を対象に自励振動のアクティブ制振に関する研究が続けられてきた.研究対象は,スロッシング,ダクト内共鳴音,サージング等のように,流体振動子が励振されるものと,ブラフボディの渦励振,ギャロッピング,翼や送水管のフラッタのように構造系と流体系の相互作用で発生する振動とに大別される.制御力としては,圧電素子や電磁石によって直接構造系に力を加えるものとアクチュエータによって付加流体力を作り出し系を安定化させる場合とに分けられる.a.
