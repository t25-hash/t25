# 14・1・1 構造物中の電磁力(electromagnetic force)

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・1 電磁力による振動の概要

磁場中における構造物の挙動を知るためには構造物に生じる電磁力を評価することが必要である.対象となる電磁力は,電流に働くローレンツ力と磁性体に働く磁気力である.磁場中に置かれた導電性の構造物には,時間的に変化する磁場が作用することにより渦電流が生じる.その渦電流と外部磁場との相互作用によりローレンツ力(Lorentz force)が生じる.また,静磁場中においても磁場と電流との相互作用によりローレンツ力が生じ構造物を変形させる.単位体積当たりのローレンツ力욢N/mは,욢=×(14・1)で表される.ここでは電流密度(A/m),は磁束密度(T)である.強磁性体の構造物には,磁場の非一様分布から生じる単位体積当たりの磁気力욇N/m욇=μ욉・∇(14・2)が生じる.ここで,μ욉は真空の透磁率(H/m),は磁化(A/m),は磁場(A/m)である.電磁力を利用している発電機やモータなどの回転機や,磁気浮上,電磁力が作用している磁場中で使われる超伝導マグネットなどでは,ローレンツ力および磁気力による変形や振動を考慮して構造物の挙動を評価する必要がある.
