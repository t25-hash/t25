# 14・1・2 電磁力による構造物の振動

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・1 電磁力による振動の概要

構造物の振動を1自由度系の運動方程式により概念的に表現する。このような場合に変形や振動を評価する際にはやを評価する必要はなく、電磁力による効果は右辺のみに現れる。一般に磁気減衰は構造物中でのジュール損失に起因することから正となる。また、磁気剛性は磁気エネルギーが一般化変位の二乗に比例することから生じる。磁気剛性は正、負の値をとりうる。超伝導マグネットで磁場あるいは電流が大きくなることで超伝導マグネットに電磁弾性座屈が生じることでわかるように磁気剛性は負となり全体としての剛性を減じるように作用する。電磁場と構造物の連成効果の取扱方法は、式(14・3)を用いると以下のように分類できる。ただし、ローレンツ力、磁気力は通常体積力となることに注意すべきである。(b) 磁気剛性効果が存在する場合超伝導マグネットのように巨大な電磁力が発生する場合や、強磁性導板が磁化による電磁力によって変形する場合には、磁気剛性を考慮する必要がある。(c) 磁気減衰効果が存在する場合強磁場中での金属薄板が振動する場合には≠0となる。構造物中でのジュール損により電磁的エネルギーが消費されるため、>0となる。上記(a)については、回転機での取扱いを14・1・3項に示す。また、(b)(c)については、14・2節にその理論的取扱いを、11・4節に解析的な取扱い方法を示す。
