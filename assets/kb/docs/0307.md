# 14・1・3 連成効果が小さい場合の電磁力と振動

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・1 電磁力による振動の概要

磁場中に置かれた構造物や機器の振動現象のうち、変形と磁場との連成、特に磁気剛性、磁気減衰を考慮する必要性のある場合については14・2節で、また、さらに高温超伝導浮上系の場合のように、非線形性が強い場合については14・3節で記述する。発電機やモータなどの回転機では、磁場中において電流が流れるが、磁気剛性効果や磁気減衰効果を考慮する必要性は少ない。電磁力解析により得られた電磁力を振動外力として応答解析している例が多い。回転機は小型化、軽量化の要求のため、構造全体の剛性が低下する傾向にある。さらにインバータ運転では、電源に高調波成分が多く、周波数が変化することから構造系の固有振動数との共振の可能性がある。ステータ(固定子)は近似的に円環(リング)の一種と考えられる。おもに半径方向に作用する電磁力が構造系の円環の固有振動を励起するため、電磁力に対する共振の回避が設計の課題である点は、発電機、モータについて共通である。モータについては、その使用環境から特に騒音低減が要求されるので、電磁力、振動、騒音まで解析した例も報告されている。発電機の場合、振動による強度低下や騒音が問題となり、大型で重量もあるので、円環一次の固有値が電源周波数の2倍、いわゆる2の変動周波数に近くなる可能性がある。このため電磁加振力の基本波に対する共振の回避が課題となる。モータは発電機と比較して小型・軽量のものが多いので、一般には円環一次の固有振動数は2よりも十分高くなる。このため電磁加振力の基本波よりも、電磁力の高調波成分に対する高次モードの振動の共振回避が課題となる。高調波に対する振動解析と、さらに偏心(ステータ軸とロータ軸中心間のずれ)がある場合の振動解析がなされている。このような連成効果が小さい場合において、回転機の解析技術についてはおおむね整備されており、実際の設計への適用が進んでいる。
