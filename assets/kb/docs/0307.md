# 2・1・4 中国の三大発明

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

古代中国は，現在私たちが恩恵を浴している技術の発祥地であっ
た．その典型的な事例は，a.火薬，b.磁石，c.紙，の三つの発明
である．いずれも重要な事項で一般にはヨーロッパのルネサンス期
三大発明といわれるが，これらのルーツはすべて中国である．
a.火薬 火薬は黒色火薬のことで，木炭の粉末（炭素）と硝
石を混合して作られる．中国では850年ごろ（唐代中期）に火薬を
使った簡単な火矢が考案され，950年ごろ（五代後期から北宋初
期）には本格的に実用化された ．火矢とは現在のロケットの原点
に相当し，底を粘土で封じて火薬を詰めた紙筒を矢先付近に取付
け，導火線に着火し弓で敵に放つ火器のことである（図2･7)．
日本人と火矢との初対面は元寇のときとされるが，日本へは直接中
国から伝わったようで，以後文化面での花火として発達した．火薬
はイスラム圏を経てヨーロッパに到達し，14世紀のドイツでは筒
の中に込めた弾丸を火薬の爆発力で発射する火砲として発達した．
これが，それまで戦闘の中心的存在だった騎士階級の没落を早めた
といわれる．日本へは1543年，種子島に流れ着いたポルトガル人
が鉄砲としてもたらしたことはよく知られている．このとき，銃尾
にあったねじ部は日本人が鉄製ねじと出会った最初ともいわれ，お
ねじとめねじの作り方には諸説があり非常に興味深い．
b.磁石 磁石は古代から中国で指南車や指南魚，磁気さじな
ど，方位を知るための計器として実用化されていた．また指南車
（shinan- sha，south-pointing carriage）には今日の差動歯車機構
が用いられており，古代中国の技術レベルの高さをうかがい知るこ
とができる（図2･8)．宋代には早くも航海用羅針盤として実用
化された．火薬と同様，これもイスラム圏を経てヨーロッパへと伝
わり，14世紀にイタリアで改良された．天文学と地理学の発達と
合わせて，北ヨーロッパのバイキング船と地中海船の長所を組合わ
せた木造帆船に，改良された羅針盤を積み，地中海商人たちは遠く
外洋航海に乗り出した ．羅針盤と時間を知るための航海用時計
（マリンクロノメータ）は，新大陸発見になくてはならない重要な
計器であった．
c .紙 文字は言語情報を後世に保存し，遠距離にも伝達でき
る有効な媒体である．しかし文字を記録する適当な手段がなかった
ため大衆には普及せず，僧侶や役人などもっぱら知的な仕事に携わ
る人々の間で使われた．中国の漢字の原点は紀元前1000年ごろの
殷で使われた甲骨文字で，紀元ごろ現在の漢字になったようであ
る．当時は木簡（木の板）や竹簡（竹の板）に漆で記されていた
が，後に筆と墨（すすをにかわで固めたもの）と紙という好都合な
記録システムが登場してから急激に発達した．
紙は長らく，紀元105年に蔡倫により発明された（『後漢書』の
記述による）とされていたが，紀元1～2世紀ごろの長安郊外の遺
跡から麻を使った紙が出土したことから，蔡倫以前にも紙の実用化
があったことが明らかになった．彼の発明では，木皮，麻くず，ぼ
ろ布，漁網をアルカリ（木灰や石灰）で煮て繊維以外の不要部分を
溶かし去り，繊維素だけで紙を作る方法を考案したことが記されて
いるのであり ，むしろ技術面での改良といえる．紙もイスラム
圏を経てヨーロッパへと伝わったが，ブドウ搾り機を応用したグー
テンベルク（Gutenberg）の活版印刷術と組合わされて宗教改革を
れるからであろう．
なし得たことはよく知られている．こうした意味において，文化
面，社会面で技術の果たした役割は実に大きいといわねばならな
い．
用してきた．自然力を利用する場合には地域性や地理的条件がかな

など実に多くの用途が見いだせる．

風車の起源は明らかではないが，紀元前1700年にハムラビ王が
メソポタミア高原のかんがい用に風車を使ったといわれる．これ
は，紀元前5000年ごろに小麦の栽培がこの地方で始まったことと
無縁ではない．古代の風車は縦軸型風車であり，当初は主軸の真上
または真下に石うすを置き，これを直接回転させる方式であった
が，のちには歯車を利用した間接的な方式へと発展した（図2･
9) ．ヨーロッパでは12世紀になり風車が導入され始めるが，北
ヨーロッパへは13世紀に，そして14世紀には広域的に普及した．
ヨーロッパの風車は主軸が横軸で，風車をもつ大きな建物全体が風
の吹く向きに回転できることが特徴で，これが技術面での大きな進
歩といえる．これらは箱型風車（図2･10) や塔型風車（図2･
11) と呼ばれているが，こうした大型の風車では，(１)風車が建
物内の石うすとは垂直関係にあり，石うすを回転させるため軸を垂
直に変換させる必要性から，歯車が用いられる，(２)この歯車は丸
太の切断面に垂直に，かつ円周上等間隔に軸を植え込んだ簡単な構
造のもの（ピン歯車）を互いに直角に組合わせて実用化され，今日
のかさ歯車のルーツになった，(３)風車の羽根には一般には帆布が
張られたが，薄い板を一定間隔で並べて窓のブラインドのように可
変とし，風力の強さに応じて調節できる可変ピッチ構造のものもあ
った，(４)石うすの回転数が高くなりすぎないように調速機（gov-
ernor ）が取付けられていた，(５)風力を石うすの回転に使うだけ
でなく風車塔内にも伝達し，石うすに穀物を供給する容器を振動さ
せる，製粉した穀物の袋を持ち上げたり，外から供給される穀物袋
を塔の上まで持ち上げるリフトの駆動など，実に集約された工場機
能をもっていた．ミル（石うす）ということばが工場の意味に使わ
れるのは，こうした事例があるからである．このほか，南ヨーロッ
パの地中海地方にも風車が見られるが，建物が石造りでそれ自体は

(２)風車で得られた動力は遠方に運ぶのが困難なこと，(３)材料や
構造面で風車の出力に限界があること，などである．現在各地に建
