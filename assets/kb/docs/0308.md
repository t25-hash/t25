# 14・2・1 電磁構造連成

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・2 磁気剛性と磁気減衰

磁場中において金属の構造物が変形あるいは弾性振動することに伴って,金属構造物中には誘導電流である渦電流が生じ,磁性体では渦電流に加えて,磁性体に生じる磁化が変化する.また,これにより外部磁場を変化させる.通常は構造物の変形は小さいためこのような連成効果は無視してよい場合が多い.しかし,超伝導マグネットのように巨大な電磁力が発生する場合や,強磁性体の薄板が磁気力によって変形する場合などには,電磁場と構造変形との連成現象として定量的に評価する必要がある.連成現象として磁気剛性効果と磁気減衰効果がある.それぞれ,運動方程式における剛性項と粘性減衰項に影響を与える.磁気剛性の影響が大きくなると静的には座屈が生じ,動的には固有振動数の低下が生じる.磁気減衰は変形に比例して導体中に起電力が生じることによって現れ,振動振幅の減衰,固有振動数の低下などの現象を引き起こす.
