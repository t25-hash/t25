# 14・2・2 磁気剛性

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・2 磁気剛性と磁気減衰

磁気剛性の理論的取扱いには、磁気エネルギー(magneticenergy)を考慮したラグランジュの方程式(Lagrange’s equa-tion)から出発するのが最も簡便である。ラグランジアン(La-grangian)はこの場合次式で与えられる。ここで、強磁場中に置かれている平板を取上げる。これらは、平板に対してはそれぞれ次式のように表される。機械系、電気系を支配するラグランジュ方程式はそれぞれ次式で与えられる。式(14・5)~(14・7)を用い、式(14・4)を機械系のラグランジュ方程式(14・8)に代入すると、次の運動方程式が得られる。変形が小さい場合にはおよびは変位の関数として表すことができる。具体的な計算例については11・4・2項に示されている。また、式(14・4)を電気系のラグランジュ方程式(14・9)に代入すると回路方程式に相当する式が得られる。この回路方程式と、式(14・10)を連成して解くのが最も一般的であるが、電圧一定か電流一定の条件のどちらかが満たされているとすると、式(14・10)だけについて検討することで十分である。式(14・10)中の連成項は磁気剛性の概念と密接な関係がある。式(14・10)の第1項が加速度項であるのに対して、第2項、第3項は、たわみに関する項が導出される。このような電磁弾性座屈現象は構造物が対称性を有する場合に生じ、代表的な例がトカマク型核融合炉のトロイダル超伝導マグネット(superconductingmagnet)である。超伝導マグネットの電磁弾性座屈、電磁弾性振動の具体的な数値解法については文献(16)(17)が参考になる。強磁性体のはりは強磁場中で座屈を生じることが知られている。非磁性導体および磁性体によって作製された片持ばりの定常磁場中での振動に関する実験とその数値解析が、発表されている。また、磁気剛性と幾何学的非線形性を考慮した座屈解析が報告されている。
