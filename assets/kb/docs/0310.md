# 14・2・3 磁気減衰

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・2 磁気剛性と磁気減衰

磁気剛性が電磁的運動エネルギーから導出されるのに対して、磁気減衰は物質中で生じる電磁的エネルギーの散逸過程によって生じる動的連成効果である。ここでσは導電率である。この右辺第2項が磁気減衰効果の原因となる。この電流と磁場により発生する電磁力であるローレンツ力は、主として運動を妨げる方向に働く。式(14・12)の右辺第2項から、磁気減衰力が速度に比例した粘性減衰力になっていることがわかる。この場合にはジュール損失(Joule loss)がエネルギー散逸の原因である。導体の運動エネルギーはジュール発熱として失われることになる。磁気減衰効果がある場合の振動の解析方法と解析例については11・4・3項において示されている。
