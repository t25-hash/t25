# 14・3・1 常電導浮上における非線形性

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・3 電磁力が関連する非線形振動

a. 吸引磁気浮上電磁石の吸引力で物体を浮上する方式は磁気軸受,磁気浮上搬送車など,多くの分野で用いられる一般的な磁気浮上法である.この浮上系では,電流と空げき(物体の変位により変動)に対する非線形性を有する.しかし,この系は吸引方向に不安定で,一定電流で浮上させることは困難であるので,一般には物体の変位を計測し,それをフィードバックして電流制御を行って安定な浮上を得る.このようなフィードバック制御を行うと,振動変位をかなり小さく抑えることができるので,非線形の影響は小さくなり,このような系では非線形の影響を無視してよい場合が多い.しかし,振幅が大きい場合は,吸引力を式(14・13)の右辺第2項のように近似する.ただし,符号は+となる.なお,吸引磁気浮上では,吸引と垂直な方向は安定となる.b. 反発磁気浮上二つの磁石の同極を対向させ,軸方向に近づけていくと,その反発力は距離の二乗にほぼ反比例して大きくなり,安定した復元力が得られる.しかし,この場合は軸直角方向にずれる力(不安定力)が発生するので,軸直角方向に浮上体を支持する必要がある.いま,復元力特性を有する系の浮上力と空げきの関係を表す曲線上の点A で物体が釣合いを保ち浮上しているとすると,このときの運動方程式は近似的に次式で与えられる.+=욃-++(14・13)ここに,は物体の変位,욃は加振力,は空気抵抗などに起因する減衰係数,~は点A を原点として復元力を最小二乗近以したときの定数である.式(14・13)は典型的な振幅依存性を有する非線形振動の方程式であり,その解法については,8章を参照されたい.
