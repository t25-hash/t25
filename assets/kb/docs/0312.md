# 14・3・2 高温超電導浮上系の非線形振動

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・3 電磁力が関連する非線形振動

高温超電導浮上系の場合は、上述の常電導浮上系とはかなり異なる非線形特性を有するので、以下にその特徴を述べる。a. ピン止め力超電導体には第I種と第II種があり、第II種の超電導体を液体窒素で冷却し(77K)、その上に磁石を置くと軸方向および軸直角方向の2方向に浮上力が発生する。冷却温度が第I種の液体ヘリウム温度に比べて高いので、第II種は高温超電導体と呼ばれている。この浮上力はピン止め力と呼ばれ、第II種の超電導体にその上部、下部臨界磁界の中間の磁界を印加したときに得られる浮上力である。現在第II種の超電導体として、YBCuOのかなり大きなバルク材の製造が可能となった。高温超電導体のピン止めの原理は、超電導体を臨界温度以下に冷却すると、超電導体の部分は超電導状態に相転移する。しかし、超電導体に格子欠陥などの非超電導部(ピン止め点)があると、その部分では、磁束が貫通する。磁束をピン止め点から外すためにはピン止め点周囲の超電導状態を壊さなければならないので、力が必要2―132となる。この力がピン止め力である。高温超電導体を磁界中で冷却すると、超電導体の欠陥部分に磁束が保持(トラップ)される。次いで、外部磁界を変化させた場合を考えると、高温超電導体に超電導遮へい電流が流れる。いま簡単のため、とが直交する場合を考えると、超電導遮へい電流密度は臨界電流密度に等しく、次式で与えられる。超電導体内の磁束密度は磁場の加え方によって大きく異なる。いま磁場のない状態で超電導体を冷却(零磁場冷却)してから、永久磁石を浮かせ、負荷をかけて磁石を超電導体に近づけていくと、磁場は超電導体の外周部から内部に侵入する。このとき、超電導体内の磁場は外周部で大きく中心部で小さくなる。またこのとき、侵入した磁場は超電導体内にトラップされる。したがって、今度は負荷を取り除き、磁石を離していくと、超電導体の外周部の磁場は小さくなるが、内部にトラップされた磁場が残っているため、磁石を近づけたときと同じ磁束分布とはならず、外部磁場を完全に取り去っても超電導体にはトラップされた磁場が残る。そのため、高温超電導体では、磁石を近づけたときと離したときの浮上力が異なる。ある磁石と超電導体でこのことを実験により示したものが図14・1である。図14・1から高温超電導体の場合は大きな履歴特性があることがわかる。しかし、この履歴特性は磁石を動かす距離によって異なり、変位が小さい場合は、履歴特性も小さくなる。一方磁場をかけた状態で冷却(磁場中冷却)する場合は、超電導体の位置における永久磁石の磁束分布がほぼそのまま超電導体にトラップされる。その状態から負荷を加えると反発力で、逆方向に負荷を加えると、吸引力で支持する。どちらも負荷の方向と逆向きの力が得られるので復元力となる。b. フラックスフローとフラックスクリープ前述のピン止め力より大きな力が浮上体に作用すると、磁束はピン止め点から外れる。この現象をフラックスフロー(磁束フロー)という。一方、超電導状態では電気抵抗は零であるから、高温超電導体は原理的には抵抗零で大電流を流すことができる。しかし、ピン止め点に捕捉された磁束には、つねにローレンツ力が働いており、熱活性でピン止め点から磁束が外れる現象がある。このため超電導体の図14・1ピン止め浮上力特性図14・2 非線形振動のモデリング前述のように超電導浮上系でも振動振幅が小さい場合は、復元力を線形としてモデリングして差支えない。振幅が大きい場合は非線形性が無視できない。またその要因は前述のようにさまざまである。まず、復元力が時間とともに減少するが、これはある時間で一定となるので、定常強制振動では無視して差支えない。さらに履歴特性による減衰特性もあるが、これについては、履歴エネルギー(履歴部分の面積)と減衰エネルギーを等値して等価減衰係数を求める方法を用いるか、あるいは実験的に求める。一方、復元力は文献(30)により、図14・5のような結果となることから、理論的あるいは実験的に浮上釣合位置からの復元力と振幅の関係を、復元力が飽和する領域までいくつかの振動数について求める。それらを図14・5のように表し、変位を原点として最小二乗法で復元力と振幅の関係を求める。実際の計算では、数値計算の過程で振動数に応じて復元力の式を入替える。具体例として、高温超電導体の上に浮上する磁石に起振力=sinωを作用させたときの浮上体の運動方程式は近似的に次式で与えられる。上式は典型的な非線形方程式であるので、ルンゲ・クッタ・ギル法などを用いることにより、数値的に解を求めることができる。なお空げきに対2―133して変位が小さく、厳密性を要求しないときは、振動数依存性を無視しても差支えない。文献ロナ社。(2)
