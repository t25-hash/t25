# 2・1・9 終わりに

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・1 機械技術史通論

これまで，機械技術史上重要と思われる話題をいくつか取上げ，
機械技術はどのような過程を経て今日に至ったかという課題を中心
に，人と技術と社会の関わりや生活と文化の質的変容などにも触れ
ながら概説してきた．従来の技術展開は，19世紀の経済学にその
よりどころを求め，資源とエネルギーは無限，技術に限界はなく，
大量生産・大量消費は良い，などといった価値観に支えられてい
る．地球資源の有限性や環境問題など，21世紀の機械工学が直面
する課題はさらに複雑かつ困難が予想される．機械技術史はこうし
た課題に直面したとき，その解決策を示してくれる唯一の重要なデ
ータベースであるといえよう．機械技術史についての各論は，2･2
節から詳細に記されている．
