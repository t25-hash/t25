# 2・2・1 機械要素の歴史

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・2 機械技術史各論

a.単一機械 五つの単一機械［てこ，滑車，輪軸（軸受），
くさび，ねじ］や歯車の歴史は極めて古く，石器時代にまでさかの
ぼるものがある．例えば太古の穴あけや火起こし道具である舞錐
（まいぎり）に滑り軸受の萌芽を認めることができる．紀元前3000
年紀の中国龍山文化やインダス文明の彩陶はろくろで作られ，滑り
軸受が明らかに用いられている．
編物や織物の起源は石器時代に始まるが，機械の元祖である織機
は新石器時代の初めごろに現れた ．漢字は象形文字に始まるか
ら，機は織機の構造を示し ，現代に至るまでその基本構造は変
わっていない．おもなものの歴史を見よう．
多種類の機械要素は永い歴史を経て，西洋ではギリシャ・ローマ
時代からヘレニズム時代に，中国を中心とする東アジアでは春秋・
戦国時代から秦・漢時代にすべて出そろった．ただ，中国では紀元
前にはねじは現れなかった．
.てこ（lever ) 旧石器時代に始まる非常に古い機械要素
であるが，紀元前6世紀のギリシャのつぼ絵にビームプレスが描か
れている．てこの応用である．オリーブ油を搾るビームプレスは，
カトー（Cato，紀元前234～149年）のビームウィンチプレス，ヘ
ロンのビームプーリプレス，プリニウス（Plinius ，紀元前23/
24～後79年）のレバーねじプレスへと発展した ．今日でも北ア
フリカのチュニジアの山中ではビームプーリプレスでオリーブ油を
搾っている．
.ねじ（screw) ねじの起源は古代地中海世界である．
ねじをギリシャのアルキュタス（Archytas ，紀元前430～365年）
の発明とする伝承がある．彼は紀元前4世紀前半のターレントゥム
（現イタリア南部の港湾都市）生まれの政治家，数学・天文学者で
ある．空間曲線の解析に優れた見解を示し，力学的現象に初めて数
学的解説を試みたため，ねじ・複滑車などを発明したとされる．
ねじと密接な関連にある旋盤原理の起源は非常に古く，紀元前
2000年ごろ，地中海マルタ島出土の石碑にその原型が見られ
る ．エフェソスの神殿の円柱や装飾，細工には見事なねじらせ
んが認められる．紀元79年，イタリアのベスビオ火山の大噴火で
埋没したポンペイの壁画 に明快なねじプレスがあり，復原展示
されている（図2･31）．
地中海シチリア島のシュラクサ出身のアルキメデス（Ar-
chimedes ，紀元前287～212年）はエジプト北端の港町アレクサン
ドリアで活躍し，かんがい用揚水機としてねじらせん面を応用した
コクリアス（kokhlias ，渦巻殻のカタツムリねじ），すなわちねじ
ポンプ を発明したといわれる（図2･32）．彼はプラトン的理論
数学を実際問題に応用し，理論と実際の一体化を図り，ペリ・ヘリ
コン『らせんについて（Peri helicon）』を著した．ねじポンプは
17世紀初めに中国，1630年ごろ日本に伝わり，佐渡金山で長年使
用された．ナイル川流域の一部ではいまでも使われている．
アレクサンドリアを中心に開花したヘレニズム文化は，天文学・
地理学の進展が著しく，精密測定器の誕生を促した．ヘロンは天文
観測用経緯儀ディオプトラ （dioptre）を考案した．角度の微調
節に精密ウォーム歯車2組が使用されている．広い意味でのねじの
応用である．

ころ，乾燥地の農耕に必要な水をくみ上げるため，サーキア
（saqiya）が考案された．牛やラクダに引き回されるアームによ

ホイールを回す．ホイールに固定された多数の水容器により，低水
位の水がくみ上げられる（図2･35)．ナイル河畔やシナイ半島の砂
漠で今日も稼働し，ナイルデルタでは鋼製の角ピン歯車をもつ鋼製
サーキアが動いている．インドのデカン高原では極めて深い地下水
をくみ上げるため，サーキアのメインホイールからバケットチェー
ンを長く垂らしてくみ上げている ．

b.巨石運搬
.石舞台古墳 奈良県明日香村飛鳥にある石舞台古墳は，
イギリスのストーンヘンジと並ぶ巨石文明の顕著な遺構である．蘇
我馬子（626年没）の墓といわれる飛鳥の石舞台の天井第1石は
638kN，第2石は770kN，ストーンヘンジのヒールストーンは
350kNである．
1932年，京都大学の浜田耕作・末永雅雄らによって初めて本格
的な調査 が行われ，高橋逸夫による人力・綱・ろくろ・滑車・
てこ・ころ・修羅などを用いた詳しい力学的論証がある．
巨石を載せるそりを修羅（しゅら，wood sleigh）と呼ぶ．修羅
は阿修羅の略で，古代インドの梵（ぼん）語（サンスクリット）古

重量物を動かす修羅には二つの意味がある．一つは丸太をレール
状または枕木状に並べ，その上を大石や大木を滑らせる滑動方式で
あり，もう一つは大型木製そりに重量物を積載して運搬する方式で
ある．ここでは大型そりを指す．
1984年2月，石舞台古墳（ancient tomb of Ishi -butai ）発掘50
周年記念事業の一環として，橿原考古学研究所と関西大学考古学研
究室により，石舞台古墳再構築実験 が行われた．修羅の製作は
1978年4月，大阪府藤井寺市の仲津媛陵の陪塚とされる三ツ塚よ
り出土した修羅［Y字形，赤樫（あかがし）製，全長8.8m，近つ
飛鳥博物館蔵］を参考として，左野勝司氏が設計製作した (図
2･37）．
強固な運搬路の上に道板を水平に2列に敷き，ころを並べ，その
上に300kNの巨石を載せた修羅を置いた．強靱な麻綱を縦軸ろく
ろにつなぎ，4人で押し回した．修羅と巨石は4人の力で軽々と動
いた ．
.古代エジプト 紀元前2000年紀の古墳壁画 には巨石
運搬技術が明確に記録されている（図2･38）．そりに載った巨大石
像は大人数によって綱で引かれている．ころは見えない．そりの先
端に乗る人が，潤滑剤を道路とそりの間に注いでいる．このときの
摩擦係数は約0.2とされる．ダウソン は人力・人数・巨石重量
は合理的と論考している．

c .チャリオット（chariot ) 交通機械とりわけ自動車の元
祖は，紀元前3000年紀の終わりごろ現れたチャリオット，すなわ
ち2頭の馬が引く一軸二輪の軽快な馬車である．
.オリエント
(１)ヒッタイト（Hittite) 紀元前2000年紀，北方草原か
ら遊牧民のヒッタイトが南下し，小アジア（現在のトルコ）のアナ
トリア高原に定着し，王国を建国した．
ヒッタイトは世界で初めて鉄を作った民族として技術史上にその
名を残している．また，古代オリエントで活躍した諸民族のうち，
唯一の印欧語族として，西欧人の関心を呼んでいる．ヒッタイトは
南方メソポタミアの二軸四輪車（a･ⅳ項）の技術と，北方遊牧民
族の馬の制御技術とを鋭利な鉄工具で結合させ，馬がけん引する一
軸二輪のチャリオットを作り上げた．方向転換が難しい二軸四輪車
に比べ，チャリオットは極めて容易に方向転換でき，軽快な乗り物
として重用された．アンカラの国立考古博物館には，オルトスタッ
ト（orthostat ，彫刻のある飾り腰石板）に彫刻されたチャリオッ
ト が多数展示されている（図2･39）．

.中国の筒車 中国黄河の上流，甘粛省蘭州付近では最近
まで多数の中国式揚水水車，筒車（つつしゃ）が稼働していた．明
代の1556年，曽任が雲南省より伝えた といわれる．明末の
1637年に宋応星（1587年～没年不詳）が伝統技術を集大成した
『天工開物』 に見られる筒車は，蘭州市の水車園に2基が復元さ
れ，実働している（図2･43）．

.日本の朝倉水車群 九州の筑後川の中流，朝倉には

車である．水車匠（たくみ）妹川矩雄氏が永年尽力されてきた．九

.水車の起源（origin of waterwheel ) 古代インドの古
典の製粉水車讃美の詩より，水車の起源をインドとする説 があ
る．世界各地の水車の実地調査より，西アジア（トルコやメソポタ
ミア），ヒマラヤ南縁（インド・ネパール），東アジア（華南山地）
等を起源とする多元説が有力である．
