# 2・2・4 情報通信機器の歴史

> **章**: 第2章 歴史から見た機械技術
> **節**: 2・2 機械技術史各論

a.はじめに：情報通信機器について 情報通信機器というの
は，これを字義どおりに解釈すれば，情報通信に関する機器のこと
である．情報処理技術/通信処理技術あるいはこれをあわせた情報
通信技術は，わが国をはじめとする先進諸国においては，社会にあ
まねく普及するに至っている．これらの技術を総称してIT［infor-
mation technologyあるいはICT(information and communica-
tions technology)］と呼ぶことにすれば，ITなしに機械設計や機
械技術に関わる開発や研究を語ることはできなくなっている．ここ
で，ITつまり情報通信技術（information and communications
technology）について若干触れておこう．情報処理とはそもそも
なんらかの記号の集合であるデータを処理して，意味のある記号の
列つまり情報に変換する技術である．一方，通信処理とは送信側が
送ろうとする信号をそのまま受信側に伝達する技術であり，途中の
ノイズをいかに消去して，送信側の信号を忠実に受信側に伝達でき
るかが重要なこととなる．ところが，コンピュータ技術の発展とと
もに，情報処理技術と通信処理技術が結合して情報通信技術となる
に及び，これら個別の処理技術は事実上その意義を失い，これらを
あわせた情報通信技術つまりITの中に両技術が包含されるに至っ
ている．
ITがハイテクの一翼を担うものとして人々から特段の注目を集
めているのは言うまでもない．こうした中で，機械技術者あるいは
一般に機械技術を学ぼうとする人々にとって，ITつまり情報通信
機器やその技術についての知識をもつことが必須になっている．そ
れに加えて，ITに関する歴史を知ることもそれと同じくらい重要
である．というのは，第一に，機械技術のこれからの発展を洞察す
るには，ITならびにその歴史を知ることが不可欠であるからであ
る．それに，ITの歴史の中には，機械技術者，あるいは広く工学
者/技術者にとって参考になる歴史的な経験が多く含まれているか
らである．
2･2･4項ではこうしたITつまり情報通信機器やその技術に関す
る歴史について考察する．ITの歴史からさまざまなことを学び取
って，技術開発や技術研究の方向を探るために，人類の始源から話
を始めることとしたい．歴史において，起こったその事実は一つか
もしれないが，その解釈は時とともに変わる．これはITの歴史に
おいても同様である．そこで，21世紀という時代，つまりITの進
歩によりマルチメディアとインタネットの利用があまねく普及し
て，コンピュータが社会に遍在（ユビキタス）しつつある時代とい
う視点から，情報通信の歴史を省察していくことにしよう．
すでに述べたように，情報通信ということばは情報と通信に分け
ることができる．情報ということばは明治時代に造られたことばだ
が，現在のように情報処理あるいは情報技術などという意味で使わ
れるようになったのは，1945年に計数型自動電子計算機（コンピ
ュータ）が誕生して以来のことである．そこで，以下においてはコ
ンピュータが登場する以前の情報通信機器の歴史と，それ以後に区
分して話を進めることにする．
情報通信機器を人が手段として利用するもともとの過程とはどの
ようなものであろうか．それは，端的には人と人とが意思や情報を
伝達する過程であり，その際に情報を加工することがある．これを
コミュニケーションと呼ぶことにすれば，このコミュニケーション
に関わる過程を機器その他の技術手段を用いて支援するのが，情報
通信機器である．なお，コミュニケーションは通常，通信と翻訳さ
れるが，適宜コミュニケーションということにする．
人と人とが行うコミュニケーション過程について，これを工学的
には時間，空間，エネルギーという視点からとらえるのがわかりや
すい．太古の時代には，会話ならぬ叫び声や身振り手振りなど五感
を利用して，一定の狭い空間でコミュニケーション活動を図ってい
たにちがいない．現在でも学校の授業などは教室という一定の空間
で営まれるし，歌舞伎や演劇，音楽などはホールや劇場で上演され
たり演奏される．一方，人と人とのコミュニケーションは，絵画や
文字あるいは録音，録画などの技術手段により，時間を超えて伝達
することができる．これがマクロな視点でのコミュニケーションに
おける時間軸の意味である．
空間を超え，あるいは時間を超えることにより，人と人とのコミ
ュニケーションは対人的で特定の他者との間のコミュニケーション
過程から，不特定で多数の人との間のコミュニケーション過程へと
広がる．そこで，前者を対人コミュニケーションと呼び，後者をマ
スコミュニケーション（mass communication）と呼ぶことにす
る．
情報とはそもそも不確実性が存在するときに，物質的ないしエネ
ルギー的に展開されたパターンのことであり，エネルギーがなけれ
ば存在しない．通信路（チャネル）上を流れる情報つまり信号には
アナログ型とディジタル型がある．アナログ型とは工学的には連続
的に変化する物理量を表すが，元来は類似しているないし相似であ
るという意味であり，アナログ型ないし相似型と呼ぶことにする．
ディジタル型とは，数字その他の有限個の離散符号で表現したもの
である．ところで，エネルギーというのはあまり一般的でないの
で，以下においては情報あるいは信号ということにする．
そうすると，情報通信機器に関わる新技術あるいはイノベーショ
ンとは，時間や空間あるいは情報をなんらかの方法で改善したり，
技術的な突破を図るものということになる．そこで，このような視
点から情報通信機器の歴史について以下に考えてみる．
