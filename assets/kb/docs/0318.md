# 15・1・4 レイリー波(Rayleigh wave)

> **章**: 第15章 波動・音響
> **節**: 15・1 波動(wave motion)

半無限弾性体においては、主として表面付近を伝ぱする波動が存在し、表面波(surface wave)と呼ばれている。その一つがレイリー波であり、はじめレイリー(Rayleigh)が理論的に存在を示し、後に存在が確認されている。いま均質な半無限弾性体において、表面が平面であるとし、直交座標を,軸が表面上にあり、軸の負方向が弾性体内部に向かうようにとるものとする。角振動数ωの調和振動の場合、変位,は次のように与えられる。変位は深さの増加に伴って急速に減少している。方向の変位は表面と深部では位相が反転しており、その境界はν=0.25のとき=-0.1951λである。図15・2においてレイリー波が紙面に平行に左から右に伝ぱするとすれば、この境界より浅部では弾性体の粒子運動は反時計回りの縦長のだ円運動、境界では方向の往復運動、深部では時計回りの縦長のだ円運動である。機械基礎の振動などによって生じる地盤振動は、縦波、横波、レイリー波のいずれをも含む。円形基礎の上下方向振動によって地盤に与えられる振動エネルギーは、67%がレイリー波となり、26%が横波となり、7%が縦波となる。このようにレイリー波はもともとのエネルギーの割合が大きく、しかも減衰が小さいので、地盤振動の主要部分はレイリー波であることがわかる。レイリー波は弾性体が均質でなく、層状構造を有する場合にも存在する。この場合には波動は分散性を有する。
