# 15・2・1 音波

> **章**: 第15章 波動・音響
> **節**: 15・2 音響

通常、音(sound、音響)とは人間の五感の一つである聴覚によって感知される感覚のことであり、この感覚は音波(sound wave)によって引き起こされる。音波とは、流体・固体中の弾性振動および弾性振動の伝搬現象のことである。ここでは、物理現象としての音波の基本的事項を中心に解説する。なお、望ましくない音として定義される騒音(noise)の問題では生理的影響や心理的影響を考慮することが重要であるが、音の生理的側面、心理的側面についてはここでは述べない。波を伝える物質を媒質(medium)といい、音波の場合は弾性体である。空気などの流体は体積変位に対してのみ弾性を示し、固体のような変形に対する弾性は示さないから、流体中の音波は圧力振動が伝わる粗密波であり、媒質の粒子(particle)の振動の方向に2―135波が伝わる縦波である。なお、媒質粒子とは、媒質の微小部分のことであり、媒質の分子を意味するものではない。音波によって生じる媒質内圧力の静圧からの変化分を音圧(sound pressure)といい、音波の存在する空間を音場(sound field)という。媒質の中で物体が振動したり、穴から媒質が出入りしたり、気流を途中で周期的に妨げたりすると音波が発生する。音波を発生する源を音源(sound source)、音源から音波が媒質中に出て行くことを音の放射(radiation)という。音の伝搬は、媒質粒子の振動状態が伝わっていく現象であるから、振動のエネルギーが伝搬していくことになる。そこで、媒質中のある部分の全エネルギーのうち音によって生じた力学的エネルギーを音響エネルギー(sound energy)と呼ぶ。また、媒質中のある点の音の強さ(sound intensity)を、その点において音波の進行方向に垂直な単位面積を単位時間に通過する音響エネルギーで表す。音圧の実効値(二乗の時間平均値の平方根)に対して、その基準値に対する比の常用対数の20倍の量が音圧レベル(sound pres-sure level)として定義されている。また、音の強さに対しても、その基準値に対する比の常用対数の10倍の量が音の強さのレベル(sound intensity level)として定義されている。いずれも単位はデシベル(dB)である。それぞれの基準値は、空気中の音の場合、音圧について20 Pa、音の強さについて1pW/mと定められている。これらの基準値は、1kHz の音の人の耳が聴きうる最小値にほぼ相当する。媒質中を伝わる音波の様子は、音波の波動方程式を解くことにより知ることができる。問題の空間内で媒質の生成や消滅がないことを示す連続の条件、媒質の運動方程式、および媒質の弾性的性質から、次の音波の波動方程式が求められる。ただし、式(15・1)の導出では線形近似などが行われており、強力な超音波の音場や、爆発の近傍では成立しない。また、気体の場合では断熱変化が仮定されており、極めて低い周波数でも成立しない。しかし、通常の音の場合には、式(15・1)で十分である。これを用いると、式(15・1)は次のように書換えることができる。空間での音圧の変化の位相の等しい点を連ねた連続面を波面(surface wavefront)という。波面が平面である平面波(planewave)や球面である球面波(spherical wave)は理論的に簡単で扱いやすい。完全な平面波や球面波は実際には存在しないが、音源から遠い場所での音波は平面波に近く、波長に比べて十分小さい寸法の音源の近くでの音波は球面波に近いことから、それぞれ近似的2―136に平面波、球面波として扱うことができる。
