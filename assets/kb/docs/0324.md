# 15・2・5 管内の音波

> **章**: 第15章 波動・音響
> **節**: 15・2 音響

フルートやパイプオルガンのような楽器は一種の音響管であり、その長さを変えて音の高低を調節する。管内には定在波(standingwave)が存在し管の長さや管端の開閉条件によって音波の分布や共鳴周波数(resonance frequency)が決まる。管内の定在波は式(15・33)と式(15・34)の和より以下の式となる。以下の境界条件を与えると管内の音圧分布、粒子速度分布および共鳴周波数が求められる。b. 閉管(closed tube)閉管は管の一方が閉じ、一方が開放された管である。いま、音の波長が管の断面寸法に比べて十分大きいと仮定し、図15・7に示すような長さの閉管を考える。管は剛体により作られ、管の内部は滑らかで一様とする。このρを空気の固有音響抵抗と呼ぶ。閉管の共鳴の波長λと周波数は音速をとして以下となる。開管では、一般に=では音圧振幅は零、粒子速度振幅は最大となる。音圧分布と粒子速度分布はそれぞれ以下の式である。このΔを開口端補正という。Δは管の断面形状や管端の状態で異なり、管が半径の円形断面をもつ場合には、Δ≒0.6(15・48)である。また、管端に無限大バッフルが付くような場合には、Δ≒0.85(15・49)となる。開口端補正は開口部の流体がピストン運動をし、音波を放射する際の放射インピーダンスと関係している。f. ヘルムホルツ共鳴器(Helmholtz resonator)大きさの異なる二つの開管と閉管が結合されるとヘルムホルツ共鳴器が構成される。ビンのような容器がその例であり、口のところを吹くと固有周波数で共鳴が発生する。図15・9のような共鳴器(resonator)を考え、音波の波長が管の寸法に比べて十分に大きいとする。太い管2の内部では閉管の場合の式が使え、細い管1の内部には定在波の式が成り立つ。管の結合部においては音圧が連続であり、気体の流出量と流入量は等しい。ただしは太い管の容積である。についても開口端補正が必要である。ヘルムホルツ共鳴器の1の部分は首または咽、2の部分は胴体と呼ばれることがある。共鳴時には共鳴器中の音響エネルギーは非常に大きなものとなり、外部の音場に対しては音の吸収作用がある。エネルギーの吸収は首部の流体振動に対する抵抗が作用し、熱エネルギーとして失われる。また、ヘルムホルツ共鳴器には共鳴によって音の放射作用もある。
