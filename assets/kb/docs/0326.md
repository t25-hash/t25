# 15・2・7 遮音

> **章**: 第15章 波動・音響
> **節**: 15・2 音響

建物の外壁、室間の壁、窓面などは音波が外部から侵入するのを防ぐ役割をもつ。壁などによって音を遮断することを遮音(soundinsulation)といい、その性能を透過損失と呼ぶ量で表現する。また遮音に用いる材料を遮音材という。遮音材の種類は多く、コンクリート板、鉄板、ガラス板、スレート板などあらゆる建材が遮音性能をもっている。a. 透過損失図15・13に示すように面密度(kg/m)をもつ無限大の単板壁を考える。音波は壁の法線方向に入射するとし、壁は音波から力を受けて曲げを伴わない往復振動をし、またその振動によって反対側には透過波が発生すると仮定する。このとき、壁に働く単位面積当たりの力と壁の振動速度の間には次の力学的運動の式が成り立つ。これを質量則(mass law)という。ランダム入射と垂直入射の関係はおよそ次式となる。このとき透過損失が低下し問題となる。いま、音波が壁に角度θで入射するとき、壁面上へ投影した音の波長λ/sinθと、壁の曲げ波の波長λが一致する際には壁は共振を起こし、音波は背面から効率よく放射される。このことをコインシデンス効果と呼ぶ。この効果には板の剛性がかかわっている。壁面に対する法線方向を軸にとり、それと直角方向の一つを軸にとると壁に関する運動の式は次のように表現される。図15・13 音波が単板壁の法線方向に入射する条件図15・14 単板の透過損失の模式図図15・15 音源-受音点配置c. 遮音塀(noise barrier)高速道路や新幹線鉄道には遮音塀が建てられており、騒音の防止に利用されている。遮音塀は遮音壁とも呼ばれるが、これまで述べた遮音材とは異なる方法で音の制御をする。すなわち音源の傍に塀を設け、その先端部から回り込む音の量を塀の高さにより調整する方法である。音の回り込みを回折(diffraction)といい、回折による音の減少を回折効果と呼ぶ。回折効果にはフレネル・キルヒホッフの回折理論をはじめさまざまな研究があるが、実務的には厚みのない塀に対する実験から作成された前川チャートが広く使われている。図15・15に示すように音源をS、障害物の先端をO、受音点をPとすれば、前川による回折効果(減音量)(dB)は行路差δ(m)と音の波長λで決まるフレネル数で表現される。すなわち、行路差δと、フレネル数はそれぞれ以下の式である。図15・16に前川のチャートの近似式直線、フレネル・キルヒホッフの理論曲線、クルツェの理論曲線を示す(図中の●印は前川チャートオリジナル版からの読取り値)。文献(1) 国立天文台、理科年表、(1995)、493、丸善。(2) 小橋豊、音と音波、(1971)、117、裳華房。(3) JIS A 1405、(1998)。(4) JIS A 1409、(1998)。(5) Kimura,K.and Yamamoto,K.,Appl.
