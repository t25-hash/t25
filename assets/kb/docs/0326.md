# 15・2・7 遮音

> **章**: 第15章 波動・音響
> **節**: 15・2 音響

建物の外壁,室間の壁,窓面などは音波が外部から侵入するのを防ぐ役割をもつ.壁などによって音を遮断することを遮音(soundinsulation)といい,その性能を透過損失と呼ぶ量で表現する.また遮音に用いる材料を遮音材という.遮音材の種類は多く,コンクリート板,鉄板,ガラス板,スレート板などあらゆる建材が遮音性能をもっている.a.
