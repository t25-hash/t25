# 16・1・3 ピストン・クランク機構のバランシング

> **章**: 第16章 往復機械の力学
> **節**: 16・1 ピストン・クランク機構の運動と力学

往復部質量による慣性力は、単シリンダ(single cylinder)のみでは釣合わせることはできない。これを往復慣性力の一次成分の部分的釣合せと呼ぶ。しかし方向の慣性力は増加する。これらが小さいほど周辺への振動伝達が小さい。前提としてシリンダ列・クランク室を剛体と仮想している。図16・5は4シリンダ機関に作用している回転慣性力と往復慣性力を示す。この機関釣合い論では軸まわりのシリンダ列・クランク室トルクを考慮しないので、これによるシリンダ・クランク室の振動もある。なお、現実のシリンダ列・クランク室は弾性体であるから、合力が零であっても、力成分と振動数が整えば高次モードの共振を引き起こすおそれがある。特に船用大型機関は剛体に遠く、船体も変形するから、各シリンダの励振荷重が各個に機関・船体結合構造物の各所に加わるとすべきであり、これらによる振動を低減するには、シリンダごとの釣合せが望ましい。. 往復質量のバランシング(balancing of reciprocatingmass)個の等価質量による慣性力は慣性力の二次成分までを考慮して以下のようになる。次に往復慣性トルクについては、往復慣性力と同様にして、以下のバランシング条件を求めることができる。白丸印は慣性力ベクトルの起点と終点であり、太い線は不釣合いベクトル、数字はそのクランクのベクトル先端部を示す。
