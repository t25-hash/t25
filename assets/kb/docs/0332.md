# 2・1・3 温度

> **章**: 第2章 熱力学
> **節**: 2・1 熱平衡と温度・熱エネルギー

化の過程で物質に加えられる熱エネルギーは，物質の相変化のエネルギーとして費やされ，物質の温度・圧力は一定に保たれる．
単位質量の物質の相変化に際して，物質を出入りする熱エネルギーを相変化の潜熱（latent heat，J その系は熱平衡の状態にあり，等しくなければ，高温の部分から低tion）などという．
温度（temperature）は，一つの物質の系において，あるいは複数の系の間で熱平衡が実現しているか否かの判定の指標になる．

一つの系と他の系の間の関係についても同様である．
