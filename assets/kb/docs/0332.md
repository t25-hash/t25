# 16・3・1 クランク軸系の振動モード

> **章**: 第16章 往復機械の力学
> **節**: 16・3 往復機械の自由振動

一般に,往復機械(reciprocating engine)は往復動部と回転部を結合するクランク機構により,力や動力を受け渡している.ここでは複雑な構造をもつクランク軸系(crankshaft system)を中心にその動的挙動を対象とする.図16・7に,おもなクランク軸系の振動モード(vibration modes)を示す.クランク軸系の軽量化,鋳造化などによる軸の三次元的剛性の低下や内燃機関での高出力化,燃焼方式の変更などによる加振力増大により,稼働条件下での共振問題(resonant problem)が発生する場合がある.さらに,図中の主たる振動モードだけでなく,いくつかの振動モードの連成(coupled vibration)を考慮する場合も生じる.
