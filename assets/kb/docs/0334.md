# 16・3・3 クランク軸系の曲げ,縦振動モード

> **章**: 第16章 往復機械の力学
> **節**: 16・3 往復機械の自由振動

クランク軸系の曲げモード(bending mode)が主体の場合は,主軸受部(main bearing cap)への軸系の動的な加振力(軸受荷重)が問題となり,図16・9,16・10に示すような回転座標系(軸系部)を固定座標系(エンジン本体構造)で弾性支持するモデル化が必要となる.クランク軸系は回転軸を含む面の方向により曲げ剛性が異なるため,また曲げや縦振動モードとの連成を表現するために,三次元的な質量ばね系の表現(図16・9など)が必要となる.クランク形状に沿って複数個の三次元的な質量ばね系に分割した伝達マトリックス法(前述)や動剛性マトリックス法(dynamicstiffness matrix approach)は多少式の扱いが異なるが同様な手法である.計算機の処理能力向上により,構造解析ソフトを用いて,三次元形状データから直接モデルのパラメータを求める手法が主流になりつつある.1クランクスローの有限要素法モデルを縮退させる方法,効率良くその結果を利用する動的サブストラクチャ法(dynamic analysis of substructure method,モード合成法など),またはクランク軸系全体の有限要素法(finite element method)モデルから縮退して自由度を落としたモデル(reduced model)を用いる手法も用いられている.特に,クランク軸に発生するひずみや応力分布の解析を行う場合は,図16・10,16・11に示すような有限要素法フルモデルが用いられる事例が出てきた.主軸受部の弾性支持は,油膜,主軸受部およびエンジン本体構造の粘性・弾性特性を考慮する必要がある.軸受部油膜の動的特性は,軸受部の平均荷重からゾンマーフェルト数と油膜特性から弾性・粘性要素を設定する事例,クランク軸回転数による変動分を考慮した油膜形成のシミュレーションから弾性・粘性要素を計算する事例などがある.クランク軸系の縦振動[図16・7(b)]は,ロングストローク化などでクランク形状の軸方向口開き剛性が低下し,固有振動数(natural frequency)が低下すると共振問題を起こす場合がある.この口開きモードは軸系の曲げ・ねじりモードと連成する場合が多く,主軸受部への加振,先端部プーリの発音などが発生する.図16・9弾性支持連続はりモデル
