# 16・4・1 加振力

> **章**: 第16章 往復機械の力学
> **節**: 16・4 往復機械の強制振動と振動低減

ガス圧によりクランク軸に加わる加振力すなわち起振トルクの調和成分項はトルクハーモニックス(torque harmonics)と総称され,その高調波成分は機関形式や燃焼方式により変化する.特に,最近の高出力化,低燃費化,排出ガスクリーン化などの要求により,燃焼システム(予燃焼室方式,直接噴射方式など),燃料噴射システム,過給器有無や燃焼制御が選択され,爆発筒内平均有効圧力(mean effective pressure)や熱発生パターンが変化している.これら爆発筒内圧の燃焼サイクルパターンの違いによりトルクハーモニックスが決まる.対象の往復動内燃機関の指圧線図実測値を用いるのが原則であるが,予測手法への適用のため,実用的なトルクハーモニックスのデータベースやシミュレーション結果を用いる場合も多い.図16・12に,クランク軸系に加わる動的加振力による固定座標系上の主軸受部への加振力の概念図を示す.両者間では,お互いの座標変換が必要となる.クランクピンには,回転質量による遠心力とクランク軸ねじり振動による円周方向の慣性力が加わる.ねじりと曲げの連成がある場合は,さらに軸系の曲げ平面方向への軸受支持反力が加算される.これら回転している加振力の固定座標軸,への投影結果を求めると,軸系の回転次加振力により誘発される共振による軸系⇒主軸受部への加振周波数は,θ=±1ωのように,座標軸の回転数分だけ上下する.図16・11 クランク軸系FEM モデル図16・12 主軸受反力図16・10 クランク軸系の有限要素法モデルと弾性支持
