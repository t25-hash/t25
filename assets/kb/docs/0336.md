# 16・4・2 クランク軸系の共振問題

> **章**: 第16章 往復機械の力学
> **節**: 16・4 往復機械の強制振動と振動低減

a. 共振モードについてクランク軸系の各種振動モードにおいて、図16・7(a)(d)は、クランク軸後端フライホイール、先端プーリの首振り振動を伴う局部的曲げモードである。前者は後方の主軸受部への加振が振動騒音問題となり、また後者はNo.1気筒の爆発加振によるプーリの発音がアイドル騒音時に問題となる。図16・7(c)では、ねじり振動モードが主体的な場合はねじり共振からねじり応力による疲労破壊問題が生じる。特に、信頼性を要求される舶用往復機関では重要視されてきた。しかし最近では、ねじりと曲げの連成モードの共振時に主軸受部が加振され、エンジン本体構造の振動騒音問題となる場合が多く、疲労問題よりいっそう厳しいねじり振動低減を要求される。図16・7(b)は、前述のように、曲げ・ねじりモードにより誘発される縦振動モードである。いずれの場合も、共振振幅が大きければ疲労破壊の危険があるが、それよりも小さい振幅で、パワープラントの振動騒音問題が発生する。そのため最近では、より低いレベルの振動低減手法が要求される。図16・13には、クランク軸のねじり曲げ振動がエンジン騒音レベルに及ぼす影響について示す。図中には、クランク軸ねじり振動ダンパ有無によるエンジン騒音の周波数分析結果の比較を示す。これによると、クランク軸の共振モードにより誘発される加振周波数帯と軸受部のがた系で生じる衝撃力によって発生するより高周波領域の騒音レベルも同時に有意差が見られる。したがってクランク軸系のねじり振動低減により、騒音寄与の大きい(500Hz 以上)領域での騒音低減にも効果があることがわかる。クランク軸系による主軸受部での加振周波数とエンジン本体構造の曲げモードとの共振問題が生じる場合もある。b. クランク軸系の振動低減手法について定格回転での稼働であればその共振周波数の移動、稼働回転域が広い船舶や車両であれば稼働時の加振周波数領域での最大振幅の低減が検討される。図より、クランク軸後端部とフライホイールの首振り曲げ振動モードの固有振動数を低い方向に移動させ、騒音が問題となる周波数領域図16・13 クランク軸系ねじり振動とエンジン騒音2―147より下方に移動させる手法を用いている。クランク軸系自身の基本設計の変更は、大幅な設計変更を伴うため、一般にはクランク軸系に振動特性を合わせたチューンドねじり振動ダンパ(tuned torsional damper)の装着が検討される。図仕事量負荷の少ないラバーダンパ(rubber damper)、シリコンオイル(silicone oil)を用いたビスカスダンパ、その折衷型のビスカスラバーダンパ(viscous rubber damper)がある。これらのダンパは、クランク軸系の先端部に装着され、図16・8のクランク軸系のねじり振動モデルに付加されて振動特性を適合させるチューニングが検討される。ラバー部は、ねじりに関する粘弾性体として複素ばね定数(complex stiffness ratio)=1+ηが用いられる。ここで、はばね定数、ηは損失係数(loss factor)である。ビスカスダンパのシリコンオイル部は、粘度(viscosity)の低い場合に粘性ダンパとして扱われる場合が多いが、高粘度では弾性と粘性の特性をもっており、複素ダンピングとして表現される場合もある。粘性に弾性効果が加わりダンパとしての制振効果は向上する。ビスカスラバーダンパは、ビスカスダンパにラバーの弾性を組合せチューニングの範囲を拡大しようとするものである。図16・16に示すように、クランク軸系とダンパの2自由度の振動系モデルにおけるダンパチューニングの最適化の指標が各種検討されている。c.
