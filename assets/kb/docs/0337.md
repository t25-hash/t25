# 17・2・1 原動機と負荷の特性

> **章**: 第17章 回転機械の力学
> **節**: 17・2 駆動トルクと負荷トルク

回転機械において原動機(prime mover)の出力軸に現れるトルク욃[負荷側から見れば駆動トルク(driving torque)],ならびに負荷が所定の運動と仕事を行うに必要なトルク욥[原動機側から見れば負荷(load)トルク]は一般に軸の回転角速度ωによって異なり,その変化の様子をトルクの回転速度特性(torque-speed characteristics),あるいは単にトルク特性という.トルク特性は機械の種類ごとに特有の形をもっており,その代表例を図(1) 原動機の特性曲線は原点を通らず,速度零のときにもトルク,すなわちストールトルク(stall torque)の発生がある.また,曲線には必ず右下がりの部分があって,この部分で使用するのが機械の正常な用い方である.(2) 負荷の特性曲線は摩擦性のものを除き,すべて原点を通り,一般に右上がりの曲線である.回転機械の発生動力または所要動力はトルクと回転角速度ωの積=ωである.したがって図17・1のトルク特性に対応した動力特性曲線が描かれ,これより機械が最大動力を発生または消費する回転速度がわかる.原動機と負荷とでこの速度が異なるときは,中間に変速機を用いることが多い.
