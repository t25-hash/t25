# 17・3・1 剛性ロータ(rigid rotor)

> **章**: 第17章 回転機械の力学
> **節**: 17・3 不釣合い力とロータの挙動

図17・5に示すように、通常、回転体の軸受中心線上に重心は一致しておらず、このため回転させるとこの偏心εによって遠心力が生じる。この遠心力を不釣合い(unbalance)力と呼び、その大きさは(偏心)×(回転角速度)に比例している。回転体が比較的低速で回転している場合、回転体の弾性変形(曲がり)は小さく無視でき、このような回転体を剛性ロータと呼ぶ。回転体の変形が無視できるとき、不釣合い力は偏心(eccentricity)に比例し、回転体の質量と偏心の積を不釣合いと呼ぶ。不釣合い力は軸受を介して軸受を支える構造物に作用し、構造物の固有振動数が回転速度域に図17・2 簡単な回転軸系図17・3 平衡速度とその変動図17・4 平衡速度の安定性(点A:安定、点B:不安定)2―150あると、固有振動数付近の回転速度では構造物は回転速度と同じ振動数で加振されるので共振する。
