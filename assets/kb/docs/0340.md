# 17・3・2 弾性ロータ(flexible rotor)

> **章**: 第17章 回転機械の力学
> **節**: 17・3 不釣合い力とロータの挙動

回転速度が増加すると、不釣合い力(遠心力)による回転体の弾性曲げ変形が無視できなくなる。このような回転体のことを弾性ロータと呼ぶ。一般にはロータの曲げ危険速度付近以上の回転速度で運転されるロータを指す。弾性ロータの力学的特性を述べるには個々の回転体について論じる必要がある。しかし、図17・6に示すような質量のない均一な弾性軸の中央に1個の剛体円板が取付けられたロータの特性を調べることによって、実用上重要な力学特性を議論することができる。このロータを基本ロータ(あるいは、Jef-cott rotor, Laval rotor)と呼ぶ。以下では特に断らない限り基本ロータを考察の対象とする。
