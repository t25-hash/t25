# 17・4・1 危険速度(critical speed)

> **章**: 第17章 回転機械の力学
> **節**: 17・4 不釣合い応答

両端が単純支持され、一定角速度ωで回転している基本ロータの円板の中心(軸心)S に関する運動方程式は次のように表される(図17・7)。たわみの振幅は初め回転速度に伴って増加する。このとき、重心G はOSの延長線上に位置し、軸は弓なりに曲がったまま回転速度と同じ速さで同じ向きに振れまわる。回転角速度がロータの曲げの固有角振図17・5剛性ロータに作用する不釣合い力(G:重心、:円板の質量、ε:偏心、ω:回転角速度)図17・6 基本ロータ図17・7 座標系動数に一致するとは発散する。この回転速度を特に危険速度と呼ぶ。振幅は時間とともに指数関数的に増大し、たわみはらせんを描いて増大する。危険速度以上に回転速度をさらに上げると、振幅は次第に減少する。このとき重心G は点Oと点S の中間に位置する。非常に高い回転速度では軸は偏心だけ曲がって、重心G は点Oに一致する。このような現象をロータの自動調心作用と呼ぶ。
