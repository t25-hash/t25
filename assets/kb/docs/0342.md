# 17・4・2 支持剛性の影響

> **章**: 第17章 回転機械の力学
> **節**: 17・4 不釣合い応答

実際の回転機械ではロータを支持する軸受などの剛性を無視できないことが多い。さらに、その支持の剛性は垂直方向と水平方向とでは異なっているのが普通である。図17・9のように水平方向、垂直方向の支持剛性をそれぞれ,とする系の運動方程式は次式で表される。不釣合い応答はこれらの式の特解として次のように求められる。危険角速度は二つ存在し、水平方向はω、垂直方向はωとなる。二つの危険速度の間の回転速度域では後ろ向き、すなわち、ロータの回転と逆向きに、それ以外の回転速度域では前向き、すなわち、ロータの回転と同じ向きに振れまわる。
