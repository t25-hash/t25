# 17・4・3 減衰の影響

> **章**: 第17章 回転機械の力学
> **節**: 17・4 不釣合い応答

系に減衰が作用するときの応答を考える。減衰としては円板周囲の流体の抵抗や滑り軸受内の軸に働く油膜の粘性力が考えられる(滑り軸受の油膜力については17・7・2項参照)。これらはエネルギーを消費して系の運動を消滅させようとするので、近似的に粘性減衰とみなすことができる。簡単のため円板の速度に比例した減衰力が速度と逆向きに作用すると考えると、運動方程式は次のように表すことができる。式(17・12)は角速度ωで前向きと後ろ向きに旋回する成分からなっている。は時間に伴い指数関数的に減少するので、式(17・12)で与えられる自由振動は時間が経過すると次第に消滅する。すなわち、このような減衰力の作用するロータの運動はつねに安定である。次に、不釣合い力による運動はロータと同じ向きに同じ角速度で振れまわるとすると次のように求まる。減衰力が作用すると固有角振動数ω付近においても振幅は発散しない。また、図17・11(b)に示すように円板の変位は、0~180°の範囲で遅れる。減衰比が大きくなると最大振幅は減少し、振幅が最大となる回転角速度はωより高くなる。
