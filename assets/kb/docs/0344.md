# 17・4・4 ジャイロモーメント(gyroscopic moment)

> **章**: 第17章 回転機械の力学
> **節**: 17・4 不釣合い応答

a. 運動方程式基本ロータでは軸がたわんでも円板は傾かな図17・10非等方弾性支持された基本ロータの不釣合い応答2―151い。しかし、図17・12に示すようにロータが片持ち支持されたり、円板が中央にない場合には、軸がたわんだときに円板は傾き、首振り運動をしながら振れまわる。回転している円板が傾くとき、傾こうとする向きと直角な向きにモーメントが作用する。このモーメントをジャイロモーメントと呼ぶ。特に、回転速度が高いとき、あるいは、円板が大きい場合にはその影響は無視できない。図17・13に示す質量のない軸に取付けられた円板の運動方程式は次式で表される。軸のまわりに円板が回転するときにも同様である。これらのモーメントはジャイロモーメントと呼ばれ、円板が回転しているときにのみ発生する。左辺第3項は軸から円板に作用するモーメントおよび力である。一般に相反定理が成り立つので行列(剛性行列)は対称となる。例えば、一様な断面の片持ちの軸では、軸の長さを、縦弾性係数を、断面二次モーメントをとすると次のようになる。固有角振動数ωは回転角速度ωに依存しており、一般にその解を解析的に求めることは困難である。定性的な関係を図17・14に示す。円板は並進運動と傾き運動を行うので、固有振動数は回転角速度に対して四つ存在している。図は原点に関して点対称であり、ω<0は回転の向きが逆になっただけで基本的な特性は同じなので、ω>0について考察する。回転角速度ω=0では、固有振動数は四つあるが、そのうちの二つずつは絶対値が同じである。しかし、ロータが回転すると正の固有角振動数と負の固有角振動数は絶対値が異なり、回転角速度ωが増加する(ω>0)と正の固有振動数は増加し、負の固有角振動数は絶対値が減少する。正の固有角振動数は前向き振れまわり(forward whirling)、すなわち、ロータの回転角速度ωと同じ向きに振れまわる固有角振動数を表す。一方、負の固有角振動数は後ろ向き振れまわり(backward whirling)、すなわち、回転角速度と逆向きに振れまわる固有角振動数を表す。c. 危険速度(critical speed)危険速度は回転速度と系の固有振動数が一致したときに、円板の偏心による不釣合い力によって振れまわり振動が大きくなる現象である。この不釣合い力はロータとともに回転しているので、不釣合い力によって励振されるのは不釣合いと同じ向きに振れまわる固有振動数成分だけである。図17・14に示す固有角振動数と直線(固有角振動数ω=回転角速度ω)との交点が危険速度となる。
