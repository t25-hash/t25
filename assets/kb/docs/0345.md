# 2・13・2 カルノーサイクル，スターリングサイクル，エリクソ

> **章**: 第2章 熱力学
> **節**: 2・13 各種サイクル

ε：圧縮比（＝）である．
100%再生することができれば，熱効率はカルノーサイクルと同じとなる．
実在流体の場合は， -線図上における二つの等積線てもカルノーサイクルの効率より低くなる．
等積変化から成る．
熱機関の場合は，a→b：等積加熱，b→c：の過程を経て，サイクルを完了する．
作動流体を理想気体とするである．
ここで， ：比熱比（＝），τ：温度比（＝），理想気体の場合は＝であるから，これを熱交換器でab，dcは傾きが異なり，＞となるので，再生を行ったとしc.エリクソンサイクル（Ericsson cycle) エリクソンサイク変化で構成されている．
このサイクルの場合もスターリングサイク体の場合カルノーサイクルと同じ熱効率になる．
