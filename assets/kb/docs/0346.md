# 17・5・1 危険速度通過(passing through critical speed)問題

> **章**: 第17章 回転機械の力学
> **節**: 17・5 危険速度通過

と基礎方程式運転速度範囲内に曲げ振動やねじり振動の危険速度(共振が起きる回転速度)が存在する回転機械では、起動停止時にロータはある速さで危険速度を通過する。危険速度を通過すると、最大振幅のあとにうなりが生じること、通過速度が速いほど最大振幅は小さくなり、そのときの回転速度は増速時には高速側へ(減速時には低速側へ)ずれることが知られている。また、駆動トルクが小さくロータの偏重心が大きい場合には、振動が大きくなるとロータの増速割合が低下して、時には危険速度を通過できなくなることもある。こうした非定常振動特性は一般に複雑で、多くの場合、ロータの運動方程式を直接数値積分して求めるが、場合によっては理論的な解析も可能である。
