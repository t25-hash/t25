# 17・5・4 駆動トルク特性および振動による負荷トルクの影響

> **章**: 第17章 回転機械の力学
> **節**: 17・5 危険速度通過

図17・18は,モータの駆動トルク特性およびロータの振動による負荷トルクを考慮して危険速度通過時の振動特性を計算した最初の結果である.駆動トルク特性(-曲線:図17・18の最上段の式)は右下がりで,ロータの偏重心はε=0.05cmと大きな値となっている.図中,横軸は加速開始からの時間,縦軸は振動振幅,θは変位と偏重心との位相角,は回転角速度,φは振動の角振動数である.図17・18より,回転速度はほぼ指数関数図17・18 トルク特性と振動負荷トルクを考慮した非定常振動的に増加するが,振幅の大きい範囲で駆動エネルギーが振動の負荷トルクに消費され,回転速度の増加率が鈍化していることがわかる.このときの最大振幅は定常振動時のそれより10~20%程度小さい.駆動トルクがさらに小さくなったり,ロータの偏重心が大きくなったりすると,回転速度の増加率の鈍化が著しくなり,ロータは危険速度通過できなくなる.ロータが危険速度を通過するために必要な最小駆動トルクは,式(17・19)の第3式を極座標表示(=cos φ,=sinφ)して,≧0とおき,振動による負荷トルクの最大値を考慮すると次のように書ける.≧ε윯율융=ερ윯율융(17・33)ここで,は駆動トルクの危険速度付近の平均トルク,ρ윯율융=윯율융εは一定角加速度α=で危険速度を通過した場合の無次元最大振幅[式(17・31)(17・32)で推定する]である.また,危険速度における駆動トルクの大きさが同じ場合には,危険速度における-曲線がより右上がりのほうが,危険速度を通過しやすい.
