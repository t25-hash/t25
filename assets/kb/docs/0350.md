# 17・5・5 危険速度通過問題の拡張

> **章**: 第17章 回転機械の力学
> **節**: 17・5 危険速度通過

危険速度通過問題は、線形の多自由度系(曲げ振動、ねじり振動)、ジャイロ系、非対称回転軸系、がた系、非線形ばね系へと拡張される一方、モータのより複雑な駆動トルク特性や三相モータの電流や発生トルクに関する運動方程式を考慮するなどの拡張が行われている。また、最近では、危険速度通過時にロータが振れ止めやケーシングに接触し、接触時の摩擦力により後向き振れまわりを含むさまざまな振動様式が発生することが報告されている。回転機械の部品で軸を有する回転部分をロータ(rotor)という。加工、組立、熱変形などにより、ロータの重心や慣性主軸は軸受中心線からそれぞれ微小距離ε(偏重心)、微小角度τ(偏角)ずれる。ロータの質量を、ロータの極慣性モーメントおよび直径まわりの慣性モーメントをそれぞれ욣,욃とすると、욦=εを不釣合い(静不釣合い、static unbalance)、욂=욃-욣τを偶不釣合い(couple unbalance)、両者が同時に存在する状態を動不釣合い(dynamic unbalance)という。こうした不釣合いをもつロータを一定の角速度ωで回転させると遠心力εωやモーメント욃-욣τωが生じ、これらがロータをたわませ、軸受を加振し、基礎に伝達される。運転時に不釣合いによるロータのたわみが問題にならない場合(運転速度がロータの曲げ振動の一次危険速度の1/2以下)を剛性ロータ、たわみが問題になる場合を弾性ロータとして区別する。ロータのたわみや軸受の振動を低減するためには、加振力である不釣合いを小さくすることが重要である。このためロータを釣合い試験機(balancing machine)上や実際の組立状態で運転し、ロータ上の指定した位置(修正面、balancing plane, correc-tion plane)における不釣合いの大きさおよび角位置を検出する。その後、修正面で所要の質量(修正質量=検出した不釣合い量/修正半径)をグラインダやボール盤でロータから除去するか、あるいは、新たな質量を180°ずれた角位置に取付けて、残留不釣合いを許容値以下にする。この一連の作業を釣合せ(balancing)という。不釣合いを完全に零にすれば不釣合いによる振動は生じないが、釣合せに多くの時間がかかり経済的でない。このためロータ形状、支持条件、使用回転速度が異なる剛性ロータや弾性ロータをどの程度まで釣合わせれば振動の少ない実機運転が期待できるかを示したのが釣合せ規格である。釣合せの方法は剛性ロータと弾性ロータで異なり、以下にこれらを説明する。
