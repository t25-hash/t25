# 17・6・1 剛性ロータの釣合せ

> **章**: 第17章 回転機械の力学
> **節**: 17・6 釣合せ

図17・19に示すように、剛体とみなせるロータが軸受中心線(軸)に垂直な微小厚さの円板個からなっていると仮定する。ロータに作用する不釣合い遠心力が力とモーメントの釣合いを満足すれば、軸受A,B にはロータの自重以外の力は作用しない。これをロータの完全釣合い状態といい、次式が成立する。通常のロータでは式(17・35)(17・36)が満足されていないので、ロータの重心から任意の距離,にある2面(修正面、例えばロータの左右端面)を選んで、そこに新たな不釣合い,を付加して式(17・37)が満足されるようにする。実際の釣合せでは、釣合い試験機上でロータを回転させて二つの軸受の振動または動荷重を測定し、指定した2個の修正面位置での不釣合いの大きさと角位置を検出する。
