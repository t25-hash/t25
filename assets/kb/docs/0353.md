# 17・6・3 釣合せ規格

> **章**: 第17章 回転機械の力学
> **節**: 17・6 釣合せ

剛性ロータについては,ISO 1940-1: 1986やJIS B 0905:1992(剛性ロータの釣合いよさ,balance quality)がある.これらの規格では,残留不釣合いをロータ質量で除した比不釣合い=とロータの使用最高速度ω(rad/s)の積ω(mm/s)を釣合いよさと定義し,11に区分した各剛性ロータ群についてωの許容値(=釣合いよさの等級:G 0.4-G 4000)を定めている(図17・21).ロータの種類,使用最高速度ωおよび質量を指定すると,釣合いよさの等級から許容比不釣合いおよび許容残留不釣合い(permissible residual unbalance)읋윬읍=욄(g・mm)が得られる.二面釣合せでは,ロータの形状,寸法,重心位置および修正面位置に応じて읋윬읍を2修正面へ振り分ける.詳細は規格を参照願いたい.弾性ロータについては,ISO 11342: 1998がある.この規格では,(i)弾性ロータを軸,円板,剛円筒,弾性円筒の要素の組合せと考えて6組に区分し,ロータの特徴を考慮して推奨する釣合せ法を示している.(ii)一般に,弾性ロータは高速釣合せ(モード釣合せ)を行うが,ロータがスパン中央部に1円板や1個の剛円筒をもつ場合などでは,低速釣合せ(一面または二面釣合せなど)で十分なことがある.(iii)釣合い試験設備上で高速釣合せを行う際,釣合せ基準を,軸または軸受の振動の許容値で示す場合とロータの許容残留不釣合いで示す場合とがある.前者では,ISO 7919やISO図17・21 剛性ロータの釣合いよさ図17・22 振動感度10816に示される実機運転での振動許容値(オーバオール値)をもとに,振動数成分,ロータ支持条件,振動測定位置の影響を考慮した釣合い試験設備上での振動許容値を算出して用い,後者では,ISO 1940-1に示される剛性ロータの許容残留不釣合い읋윬읍をもとに,ロータの一次,二次の等価モード不釣合いおよび低速釣合せ時または定格速度での等価残留不釣合いをそれぞれ읋윬읍の60%,100%以内としている.なお,ISO 10814: 1996では,実機ロータ系の振動応答倍率욈が大きいと,不釣合いに対するロータの振動感度が高くなるとして,3タイプに分けた弾性ロータ群に対して5段階(A~E)の振動感度領域を設定し,욈-η욈平面上にそれらの領域を示している(図17・22).η욈=運転速度次の曲げ危険速度である.領域D,Eは振動感度が高く,D(7.5<욈<10,タイプII)では,フィールドバランスが必要になり,Eの領域の設計は避けるように指示されている.
