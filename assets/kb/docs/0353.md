# 17・6・3 釣合せ規格

> **章**: 第17章 回転機械の力学
> **節**: 17・6 釣合せ

剛性ロータについては、ISO 1940-1: 1986やJIS B 0905:1992(剛性ロータの釣合いよさ、balance quality)がある。二面釣合せでは、ロータの形状、寸法、重心位置および修正面位置に応じてを2修正面へ振り分ける。詳細は規格を参照願いたい。弾性ロータについては、ISO 11342: 1998がある。この規格では、(i)弾性ロータを軸、円板、剛円筒、弾性円筒の要素の組合せと考えて6組に区分し、ロータの特徴を考慮して推奨する釣合せ法を示している。(ii)一般に、弾性ロータは高速釣合せ(モード釣合せ)を行うが、ロータがスパン中央部に1円板や1個の剛円筒をもつ場合などでは、低速釣合せ(一面または二面釣合せなど)で十分なことがある。(iii)釣合い試験設備上で高速釣合せを行う際、釣合せ基準を、軸または軸受の振動の許容値で示す場合とロータの許容残留不釣合いで示す場合とがある。領域D,Eは振動感度が高く、D(7.5<<10、タイプII)では、フィールドバランスが必要になり、Eの領域の設計は避けるように指示されている。
