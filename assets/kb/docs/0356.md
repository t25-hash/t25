# 17・7・3 気体軸受(gas bearing)

気体軸受は油により潤滑される滑り軸受に比して摩擦トルクが著しく小さく,清浄で使用可能温度範囲が広く,また,転がり軸受に比べて励振力発生要因をもたないので,小型・高速回転ロータ用軸受および精密回転ロータ用軸受として用いられる.高速回転用には主として動圧軸受が,また,精密回転用としては主として静圧軸受が使用される.図17・31におもな動圧気体ジャーナル軸受の構造を示す.気体軸受で支えるロータはほとんど剛性ロータであり,その不安定振動は回転角速度ωの1の角振動数をもつ.真円軸受または溝付軸受などの場合,2となるが,後者の場合2とは異なる振動モードが不安定になる場合がある.溝付軸受では軸心付近にジャーナルがあっても溝によって気体がかき込まれるために圧力が発生し,真円軸受よりも安定性はよい.静圧軸受の場合,軸受すきまが大きくなるに従って,は2よりも大きくなる.ティルティングパッド軸受やフォイル軸受は安定性に優れた軸受である.気体軸受は負荷容量が小さく,また,減衰力も小さいので軸系の安定性ばかりでなく不釣合い力による振れまわりの大きさにも注意が必要である.軸系の安定限界,不釣合い力によるロータの振れまわりは,圧縮2―162機械力学(a) 真円軸受(c) ティルティングパッド軸受(d) フォイル軸受(b) ヘリングボーン溝付軸受図17・31 動圧気体ジャーナル軸受の構造例性流体に対するレイノルズ方程式(17・66)を解いて得られる弾性および減衰係数を用いてかなり正確に予測できる.a. 弾性および減衰係数の計算軸受気体膜の作用および弾性および減衰係数の定義に関しては,油により潤滑された滑り軸受の場合と同様である.ただし,ジャーナルと軸受とのすきまに生じる圧力は,気体の圧縮性を考慮した,次のレイノルズ方程式(Reynolds equation)に支配される.옿옿θ 옿옿θ +옿옿옿옿=6μ 옿옿θ+2옿옿(17・66)ここで,θおよびは軸受滑り面に沿って設けた円周方向および軸方向の座標,は膜厚,は周方向(θ方向)の滑り速度,μは気体の粘度,は軸受半径,は平均軸受すきまを表す.式(17・66)は圧力に関して非線形なので数値的に解くのが一般的である.弾性・減衰係数を求めるには圧力を時間に依存しない静的な成分と変動圧力成分の和と考える摂動法による.静圧軸受では軸受すきま内に加圧された気体が流入するので,式(17・66)を解く際にはこの流入流量も含めて質量保存則が成り立つようにする必要がある.弾性・減衰係数は,通常,くさび膜効果の影響を表すベアリング数Λ(bearing number)に対して表す.Λ=6μω욀(17・67)ただし,욀は周囲圧力である.b. 安定性(stability)図17・32は真円軸受で支持された水平ロータの安定判別図である.無次元安定限界回転速度をベアリング数Λに対し示してある.偏心率εが大きいほど安定限界回転速度は高くなる.図17・33に真円軸受とへリングボーン溝付軸受の負荷容量および剛性욥をベアリング数Λに対して示す.Λが約10以上の領域においては,溝のポンピング効果のためへリングボーン溝付軸受のほうが負荷容量,剛性ともに高くなる.図17・34にへリングボーン溝付軸受で支持された水平ロータについて,ベアリング数Λと安定限界質量で表した安定判別図を示す.Λが約22以下の領域ではΛが大きくなるほどワール不安定(振れまわり不安定)が発生する安定限界質量は小さくなる.さらにΛが30以上の領域ではΛが大きくなるほどは急激に小さくなる.またΛが22以上では気体膜の圧縮性に起因するニューマチックハンマのため,小さいにおいても不安定になる.なお,真円軸受で支持された水平ロータは図に示す領域ではすべて不安定となる.
