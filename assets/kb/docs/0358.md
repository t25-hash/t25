# 17・8・1 二次的危険速度

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

不釣合いによる回転軸の振動は回転角速度ωが系の固有振動数ωに一致したときに大きな振動が起こりこれを危険速度と呼んだ。しかし、場合によっては回転数と一致しない振動数で回転体が大きな振れまわり振動をすることがある。これを二次的危険速度(sec-ondary critical speed)という。a. 回転速度が変動する場合駆動側に往復動機関や電動機、従動側に往復動コンプレッサやポンプ、また、継手として自在継手などが使われていると、これらのトルク変動成分は一般に平均トルクの0.5~5%で小さいが、振動の発生の原因となる。ここでは回転角、ωは回転角速度の平均値である。一般に機械のトルク変動または角速度変動は、回転速度の整数倍か、分数倍であることが多いので、=12,1,2のとき式(17・83)から、共振を起こす回転速度比ωωを求めると表17・3に示すようになる。この危険速度は速度の変動によって生じる二次的な成分であることから、二次的危険速度と呼ばれる。この危険速度による励振力は主危険速度の励振力に比べて非常に小さな力であるが、主危険速度よりもかなり低い回転速度で大きな振動を起こすことがあり、二次的危険速度を避けるように設計することが望ましい。b. 重力の作用を受ける非等方軸剛性(nonaxisymmetric shaftstiffness)発電機やキー溝のある水平に置かれた回転軸では、直交する二方向で曲げ剛性が異なるために、つねに鉛直下方に働いている重力によって重心が加振されることになる。基本ロータの直角方向の剛性が異なる系の運動方程式は軸と同じ回転角速度ωで回転する回転座標系O-ξηで表すと、運動方程式は次のように表され、剛性が時間的に変化しなくなる。軸剛性の非等方性のためξ軸、η軸方向で曲げの固有振動数が異なり、それぞれ1-μω,1+μωである。式(17・84)の重力による振動解は次のようになる。軸の非等方性μが小さいとき、二次的危険速度は主危険速度ωの約1/2となる。
