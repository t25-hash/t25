# 2・2・1 熱エネルギーと力学的なエネルギー

> **章**: 第2章 熱力学
> **節**: 2・2 熱力学の状態量

熱エネルギーと力学的なエネルギーである仕事（work）は，と
もにエネルギーであり，その単位はJ （＝N･ m）である．熱エネ
ルギーの単位には，1atmのもとで重量1kgfの水の温度 を，
14.5℃から15.5℃まで1Kだけ増加させるのに要する熱エネルギ
ーを1kcal （＝4185.5J ）とするものを用いることがある．動力
（power ）は単位時間当たりになされる仕事（仕事率）であり，そ
の単位はW（＝J / s ）である．
