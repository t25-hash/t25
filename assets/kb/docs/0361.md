# 17・8・3 内部減衰(internal damping)

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

水平に置かれたロータの軸が重力の影響でたわんで回転しているとき、軸が回転するたびに伸び縮みが繰返され、軸の各部分は相対的に伸びの量が異なるために摩擦が起こる。このような摩擦に起因した減衰を内部減衰と呼ぶ。内部減衰力を速度比例型減衰として比例係数を욡とすると、基本ロータの運動方程式は角速度ωで回転する回転座標系O-ξηにより次のように表すことができる。ξξη=0η+ζ욡ω욈-2ωη+ω욈-ωξ2ωζ욡ω욈ω욈-ω(17・90)ここで、ζ욡=욡2ω욈(17・91)である。この解をξ=,η=とおいて固有値を求めると、=-ω+ζ욡ω욈± ω+ζ욡ω욈-ω욈-ω(17・92)となる。通常ζ욡≪1であることを考慮すると、≒-ω욈±ωζ욡±(複号同順)(17・93)と近似できる。これから固有値の実部はω<ω욈では負であり、振動は時間とともに消滅する。一方、ω>ω욈では固有値の一つの実部が正なので、振動は時間とともに発散する。内部減衰のあるロータを危険速度以上の高速で運転すると、外部減衰が十分でないときには不安定になる。軸に円板などが圧入されているとき軸が変形すると、軸との間で摩擦が生じる。このような構造減衰を正しく表現することは困難だが、内部減衰同様ω>ω욈の回転速度域で不安定化作用をもつ。
