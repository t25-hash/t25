# 17・8・4 シール

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

ポンプ、遠心圧縮機や蒸気タービンの流体回転機械では、段落ごとやケーシング外への流体漏れを抑えるためにさまざまなシールが用いられている。これらの流体回転機械では負荷を上げていく過程で異常振動を経験している。その振動発生はシールで漏れている流体が関与していることから、ポンプの場合にはハイドロホワール、遠心圧縮機の場合はガスホワール、そして蒸気タービンはスチームホワールと呼ばれている。これらの振動は自励振動であり、流体力を線形化したとき得られる弾性係数の連成項による。観察された振動を以下に述べる。a. ガスホワール化学プラントの合成ガス等の遠心圧縮機では、負荷を上げていくと突然異常振動が発生し、これ以上増産できなくなるという経験が多く報告されている。この振動数成分は回転速度より低い周波数からサブシンクロナスホワールとも呼ばれる。例として、アンモニア合成塔に高圧の合成ガスを送る遠心圧縮機トレンのガスホワール振動を図17・44に示す。負荷95%である回転速度10750rpmで、図17・44下部に示すような発散的な振動が低圧(LP)圧縮機を中心に発生した。この振動周波数は80Hz で前向きに振れまわっており、回転速度、すなわち負荷を下げると振動は消滅した。回転体の安定性を分析した結果、LP圧縮機のみが不安定であり、安定な圧縮機と交換することで解決した。ガスホワールの原因はおもにラビリンスシールであり、ラビリンスシールの動特性解析についてはコスチュークがモデル化し、岩壺らが解法を与えている。ラビリンスシールの不安定効果である弾性係数の連成項を大きくする因子は、入口旋回流である。すでに円周方向に流速をもつガスがシールに入ってくることが想定される。ラビリンスシールの膨張室でのガスの円周方向速度は、軸の周速のほぼ半分である。この膨張室でのガスの周速より速い、または遅い旋回速度をもつガスがシールに入ったときの速度分布は図17・45のように変化する。速いガスが入れば膨張室の円周速度も速くなり、弾性係数の連成項は正となり、軸は前向き振れまわりが励起される。一方、遅ければ膨張室の円周速度も遅くなり、連成項は負になり、前向き振れまわりは励起されない。軸の周速の半分より大きい入口旋回速度となるラビリンスシールではガスの流入口に旋回速度を低減するスワールキャンセラ(整流板等)を入れるなどの注意が必要である。c.
