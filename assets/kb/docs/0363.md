# 17・8・5 液体を部分的に封入した回転軸系

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

回転機械の中には、回転軸部分を中空にして作動流体または冷却水を流したり、あるいは中空部分にドレンや潤滑油がたまってしまったまま回転させるということが多々見られる。このような状態での運転は、不安定振動が発生することが知られている。以下では、回転速度は十分高く、液体の挙動は自由表面を有する遠心場が支配的とする。(1) 液体が自由表面をもつ状態での危険速度は次式のように封入液体量に依存せず、軸心から容器の内径までの大きさで決まる。ω=ω+ρπ(17・94)ここで、添字,はそれぞれ封入時、無封入時の危険速度を、は回転体のみの質量、ρ,,は液体の密度、封入容器の内半径、長さを表す。(2) 封入液体は剛体運動を行うので減衰は零と考える。液体は危険速度以下では容器の偏心に伴う不釣合いを出すが、危険速度以上では自動バランサとして働く。(3) 危険速度を通過したあと、約2倍の速度までの間不安定振図17・47液体を部分的に封入した回転軸の振動図17・48 歯車軸系動を発生する。この不安定振動は回転方向とは逆に伝ぱする一つの直径節をもつ後進波動モードと回転軸系の固有値との連成による自励振動である。液体を非粘性流体として、ウォルフはωが複素数になると不安定であるという以下の安定判別式を導いていた。-ω=ρπω-4ωΩ+2Ωγω-2γ+1ωΩ+γ+1Ω(17・95)γ=+1-1(17・96)ここで、は回転体の支持弾性係数、Ωは回転角速度、は自由表面内半径である。図17・47に示す横置きモデルロータでの実験結果でも上記ウォルフの安定判別結果と比較的よい一致を示す。
