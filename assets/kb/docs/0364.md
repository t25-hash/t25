# 17・8・6 曲げ・ねじり連成振動(coupled vibration)

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

歯車を含む動力伝達系において,歯車を支える軸や軸受の剛性が高くない場合は,負荷運転時に歯車は回転するだけでなく並進運動する.このとき,歯車のかみあい線方向の接触条件から,歯車の回転角と並進変位との間に拘束が生じるため,軸系のねじり振動と曲げ振動(横振動)が関連し合う,いわゆる連成が生じる.連成系の固有振動数や危険速度は連成がない場合の値と異なるので,設計時など注意が必要である.以下に最も単純な例を示す.剛性軸(軸1)と弾性軸(軸2,曲げのばね定数)をもつ歯車軸系(図17・48)のねじりおよび曲げ振動の運動方程式は次のように書ける.θ=-θ=(17・97)=-+=-(17・98)歯車1,2の歯面の接触条件より,θ=θ+(17・99)ここで,θ,θは歯車1,2の回転角,,は歯車2のかみあい線,滑り線方向の並進変位,はかみあい歯面力,,は歯車1,2の基礎円半径である.上式より,θ,θ,が連成し,が独立であることがわかる.式(17・97)~(17・99)からとθを消去して,θ,に関する運動方程式を導き,連成系の固有角振動数を求めると次式が得ら2―167図17・49後向き振れまわりの発生過程れる.=+α,α=+(17・100)一方,連成がないの曲げ振動の固有角振動数ω욈は式(17・98)より,ω욈=(17・101)式(17・100)(17・101)より,連成の結果,軸2の方向の曲げ振動の固有角振動数は低下し,かつ,が大きいほどが低くなることがわかる.この歯車系を負荷運転し,軸2の曲げ振動を計測すると,一般にはおよびω욈で二つの共振のピークが観察される.この例では,ねじり振動の固有角振動数は存在しない.なお,式(17・99)の接触条件は,=θ-θ-(17・102)のように,かみあい歯面力(:かみあいのばね定数)の式を用いる場合もある.
