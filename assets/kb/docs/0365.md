# 17・8・7 ロータの接触振動

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

角速度ωで回転しているロータは、通常、不釣合い遠心力によってたわみを生じ、回転方向と同じ方向に角速度ωで円またはだ円軌跡を描いて旋回(前向き振れまわり)する。しかし、危険速度通過時や事故などでロータの振れまわりが大きくなりケーシングなどに接触すると、摩擦力の作用によってロータが回転方向と逆方向に大きく振れまわる、いわゆる後ろ向き振れまわり(backwardwhirling)が発生する場合がある。このとき、振動、騒音、接触力がともに大きくなり、回転機械にとって極めて危険である。後向き振れまわりの発生過程は、シミュレーション結果(図17・49)から次のように説明できる。反射角が負になるとロータの振れまわりが後向きになる。反射角が90°を超えるとロータはケーシングに食込むようになり、接触力や摩擦力が増加し回転速度が低下する。ケーシングを弾性支持した場合は、その振動速度を含めてロータ・ケーシング間の相対速度が零になるようにωが決まる。後向き振れまわりは、摩擦係数、ケーシングの質量および支持ばね定数が大きいほど、ロータ・ケーシング間の接触剛性が小さいほど発生しやすい。文献(1) 三輪修三・坂田勝、機械力学、(1984)、98、コロナ社。(2) 文献(1)の121ページ。(3) ガッシュ・ピュッツナー(三輪修三訳)、回転体の力学、(1978)、9、森北出版。(4) Dunkerley,S.,Phil.
