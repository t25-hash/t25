# 18・2・1 運動学解析問題

> **章**: 第18章 ロボットアームの力学
> **節**: 18・2 ロボットアームの運動学

シリアル機構の剛体アームは、図18・2に示したように、リンク2―170が直列につながり、終端リンクがどこにも閉じていない、いわゆる開ループ構造となっている。各ジョイントは回転またはスライドの1自由度ジョイントとする。各リンクは剛体とみなされるので、アームの運動は、各リンクをつなぐジョイントの回転またはスライド変位により決定される。したがって、ロボットアームの運動学解析問題は、ジョイントの変位、速度、加速度と各リンクの位置、姿勢、速度、加速度との間の関係を解析する問題となる。ロボットアームの目的は、ジョイントの回転角やスライド量を変化させることにより、アーム先端のエンドエフェクタの位置、姿勢、速度、加速度を作業空間内で決定し、対象物のマニピュレーションを行うことである。は作業ベクトル(task vector)と呼ばれる。一般には=であるが、>の場合も研究されており、このようなアームは冗長自由度アーム(redundant arm)と呼ばれる。前者は順運動学(forward kinematics)、後者は逆運動学(inverse kinematics)と呼ばれる。
