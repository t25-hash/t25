# 18・2・1 運動学解析問題

> **章**: 第18章 ロボットアームの力学
> **節**: 18・2 ロボットアームの運動学

シリアル機構の剛体アームは,図18・2に示したように,リンク2―170が直列につながり,終端リンクがどこにも閉じていない,いわゆる開ループ構造となっている.各ジョイントは回転またはスライドの1自由度ジョイントとする.各リンクは剛体とみなされるので,アームの運動は,各リンクをつなぐジョイントの回転またはスライド変位により決定される.したがって,ロボットアームの運動学解析問題は,ジョイントの変位,速度,加速度と各リンクの位置,姿勢,速度,加速度との間の関係を解析する問題となる.ロボットアームの目的は,ジョイントの回転角やスライド量を変化させることにより,アーム先端のエンドエフェクタの位置,姿勢,速度,加速度を作業空間内で決定し,対象物のマニピュレーションを行うことである.ジョイントの回転角やスライド量よりなるベクトルをジョイント変数ベクトル(joint variable vector)と呼び,=,,...,욈(18・1)により表す.また,作業空間内でのエンドエフェクタの位置,姿勢など,作業座標(task coordinates)を表すベクトルを,=,,...,욇(18・2)と書くことにしよう.は作業ベクトル(task vector)と呼ばれる.はアームの自由度数,は作業座標の次元である.一般には=であるが,>の場合も研究されており,このようなアームは冗長自由度アーム(redundant arm)と呼ばれる.通常は,=6である.さて,このように記号を定義すると,アームの運動学解析で制御上重要な問題は,,,より,,を求める問題と,逆に,,より,,を求める問題である.前者は順運動学(forward kinematics),後者は逆運動学(inverse kinematics)と呼ばれる.このような機構解析は,せんじつめれば,をに変換する座標変換(coordinate transformation)の関数=(18・3)の構造を,リンク機構の構造と対応づけて解析することにほかならない.
