# 18・2・2 デナビット・ハーテンバーグの記法

> **章**: 第18章 ロボットアームの力学
> **節**: 18・2 ロボットアームの運動学

ロボットアームの運動は、機構を構成する各リンクの空間運動により決定される。したがって、その運動を解析するためには、各リンクの空間運動を解析すればよい。各リンクを剛体と仮定すれば、その運動は、各リンクに固定した座標系の運動により表すことができる。図18・4のリンク連鎖を考え、各リンクに図のような直交座標系O욡-욡욡욡を固定したとしよう。この座標系をリンク座標系(linkcoordinate system)と呼び、Σ욡で表す。Σ욡は、原点O욡の位置と各座標軸の単位方向ベクトル욡,욡,욡により表すことができる。リンク座標系の選び方は、種々考えられるが、一般に用いられる記法は、デナビット・ハーテンバーグの記法(Denavit-Hartenbergnotation)である。これを図18・5に示す。図18・5は、回転ジョイントに対するリンク座標系の設定を示す。直動ジョイントに対するリンク座標系の設定については、文献(9)を参照されたい。リンクには、台座側から、台座に固定されたリンクを0として、先端リンクへ向って、1,2, 3, ...と番号を付ける。ジョイントには、やはり台座に近い側から先端へ向けて、1, 2, 3, ...と番号を付ける。番めのリンクに固定されるリンク座標系Σ욡(O욡-욡욡욡)は次のように定義される。(1) +1番めのジョイントの回転または直動軸を욡軸とする。욡軸の方向は回転または直動の正方向とする。(2) 욡욪軸と욡軸の共通法線と욡軸の交点を原点O욡とする。욡욪軸と욡軸が交わるときは、その交点を原点O욡とする。OはOに一致させる。(3) 욡욪軸と욡軸の共通法線を욡욪軸から욡軸方向に延長した軸を욡軸とする。욡욪軸と욡軸が交わるときは욡욪×욡に平行に욡軸をとる。向きはどちらでもよい。図18・4 リンク座標系による機構の表現図18・5 デナビット・ハーテンバーグの記法(4) 욡軸と욡軸に対して、右手系となるように욡軸を定める。リンク座標系Σ욡のΣ욡욪に対する位置と姿勢は、図18・5に示す四つのパラメータ욡,α욡,욡,θ욡により表される。욡,α욡,욡,θ욡は次のように定義される。(1) 욡욪軸と욡軸の共通法線の長さを욡とする。욡はリンクの長さと呼ばれる。(2) 욡욪軸に対して욡軸のなす角度をα욡とする。α욡はリンクのねじれと呼ばれる。α욡の正方向は욡軸の方向とする。(3) O욡욪から욡욪軸方向に沿って測った욡욪軸と욡軸の共通法線の足の位置を욡とする。(4) 욡욪軸に対して욡軸のなす角度をθ욡とする。θ욡の正方向は욡욪軸の方向とする。以上定義した四つのパラメータの中で、回転ジョイントではθ욡が、直動ジョイントでは욡がジョイントの動きを表すジョイント変数(joint variable)となる。他の三つのパラメータはリンクの形状を表すリンクパラメータ(link parameters)である。デナビット・ハーテンバーグの記法では、リンク座標系は必ずしも一意に定まらない。例えば、スライドジョイントでは、욡軸の方向は決まっても、その位置は一意に定まらない。デナビット・ハーテンバーグの記法では、このようなときは、解析者が適宜決定するものとしている。また、以上述べたリンク座標系の定義からは、先端リンクすなわちエンドエフェクタに固定する座標系は決定されない。先端リンクの座標系の原点は、通常、その一つ手前のリンク座標系の原点に一致させる。さて、いずれにせよ、リンク座標系が定義されれば、各座標系間の相対位置、姿勢関係を解析することにより、ロボットの運動を解析することができる。Σ욡とΣ욡욪の位置、姿勢関係は、Σ욡のΣ욡욪に対する姿勢の変換行列욡と、O욡욪からO욡へ至る位置ベクトル욡により表すことができる。ここで、욡の左肩字-1は、このベクトルがΣ욡욪で成分表示されていることを示す。욡,욡は、욡,α욡,욡,θ욡を用いて、sinα욡sinθ욡-cos α욡sinθ욡cos θ욡(18・4)-sinα욡cos θ욡cos α욡cos θ욡욡=sinθ욡cos α욡sinα욡욡cos θ욡욡=욡sinθ욡(18・5)욡と表される。욡,욡は、ジョイント変数の関数となっている。욡,욡を用いると、リンク機構を組織的に解析することが可能となる。
