# 18・2・2 デナビット・ハーテンバーグの記法

> **章**: 第18章 ロボットアームの力学
> **節**: 18・2 ロボットアームの運動学

ロボットアームの運動は、機構を構成する各リンクの空間運動により決定される。したがって、その運動を解析するためには、各リンクの空間運動を解析すればよい。各リンクを剛体と仮定すれば、その運動は、各リンクに固定した座標系の運動により表すことができる。この座標系をリンク座標系(linkcoordinate system)と呼び、Σで表す。リンク座標系の選び方は、種々考えられるが、一般に用いられる記法は、デナビット・ハーテンバーグの記法(Denavit-Hartenbergnotation)である。これを図18・5に示す。図18・5は、回転ジョイントに対するリンク座標系の設定を示す。直動ジョイントに対するリンク座標系の設定については、文献(9)を参照されたい。リンクには、台座側から、台座に固定されたリンクを0として、先端リンクへ向って、1,2, 3, ...と番号を付ける。ジョイントには、やはり台座に近い側から先端へ向けて、1, 2, 3, ...と番号を付ける。OはOに一致させる。向きはどちらでもよい。αはリンクのねじれと呼ばれる。以上定義した四つのパラメータの中で、回転ジョイントではθが、直動ジョイントではがジョイントの動きを表すジョイント変数(joint variable)となる。他の三つのパラメータはリンクの形状を表すリンクパラメータ(link parameters)である。デナビット・ハーテンバーグの記法では、リンク座標系は必ずしも一意に定まらない。例えば、スライドジョイントでは、軸の方向は決まっても、その位置は一意に定まらない。デナビット・ハーテンバーグの記法では、このようなときは、解析者が適宜決定するものとしている。また、以上述べたリンク座標系の定義からは、先端リンクすなわちエンドエフェクタに固定する座標系は決定されない。先端リンクの座標系の原点は、通常、その一つ手前のリンク座標系の原点に一致させる。さて、いずれにせよ、リンク座標系が定義されれば、各座標系間の相対位置、姿勢関係を解析することにより、ロボットの運動を解析することができる。
