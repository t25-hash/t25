# 18・2・4 ヤコビ行列の計算

> **章**: 第18章 ロボットアームの力学
> **節**: 18・2 ロボットアームの運動学

a. ヤコビ行列の定義ヤコビ行列(Jacobian matrix)は、式(18・3)で表される座標変換の関数を局所線形化した変換行列として定義される。はエンドエフェクタの位置を表すベクトル、は姿勢を表すベクトルである。なお、はオイラー角(Eulerangles)あるいはロール、ピッチ、ヨー角(roll, pitch, yawangles)を成分とし、ベクトル形式で表記されるが、物理量としては、ベクトルの性質を有しない。式(18・24)によるヤコビ行列の定義は直接的であるが、が式(18・26)で与えられるとき、その姿勢角成分の微分が何を意味するか、直感的にわかりにくい。これに対して、ヤコビ行列のもう一2―172機械力学つの定義法として、エンドエフェクタの速度と角速度(または微小変位と微小回転)に注目する方法がある。作業ベクトルが式(18・26)により定義されているとき、式(18・28)のと式(18・24)のの間には次の関係が成り立つ。の定義として、オイラー角またはロール、ピッチ、ヨー角を考えると、図18・7の(a)(b)に示すように、の微分は、を図の斜交座標系で成分表示したものと考えることができる。上記の説明では、速度、角速度を表記する座標系については触れなかったが、実際の計算ではこれを考慮する必要がある。詳細は文献(13)(14)を参照されたい。
