# 18・2・5 逆運動学

> **章**: 第18章 ロボットアームの力学
> **節**: 18・2 ロボットアームの運動学

逆運動学とは、18・2・1項で述べたように、リンクの位置、姿勢、速度、角速度、加速度、角加速度が与えられたとき、それを実現するジョイント変数の位置、速度、加速度を求める問題である。ロボットアームの制御においては、先端リンクすなわちエンドエフェクタの位置、姿勢、速度、角速度、加速度、角加速度の制御が問題となることがほとんどで、したがって、逆運動学問題は、エンドエフェクタの位置、姿勢、速度、角速度、加速度、角加速度よりジョイント変数の位置、速度、加速度を求める問題がおもな問題となる。以上の三つの関係式に対して、それぞれ、以下の三つの逆運動学問題が定義される。が与えられたときを求める問題は、分解速度制御(resolved motion rate con-trol)の問題である。こうして計算されたを式(18・39)に代入し、右辺に含まれる未知ベクトルについて解けば、逆座標変換の解が得られる。しかし、容易にわかるように、この逆座標変換問題は、複雑な非線形方程式を解く問題となり、一般的な解析解は存在しない。一般解は数値解法による。ただし、産業用に使用されているほとんどのロボットアームに対しては、個別的な解析解が存在する。種々のアームに対して種々の解法が提案されている。解析解は、通常複数個存在する。が正方でない冗長自由度アームの場合の解法については、文献(7)を参照されたい。が正則でない場合の解法については、文献(18)を参照されたい。なお、6自由度アームに対しては、式(18・68)の解を解析的に求める手法も提案されている。前者の問題は分解速度制御の問題と等価である。静力学解析では、ロボットアームが静止していると仮定して、ロボットアームに加えられる外力や外モーメントの、各リンク、各ジョイントへの影響を解析する。対象物の重量を支えたり、対象物を押付けたりするときの、各アクチュエータの出力トルクの計算や、各部材の強度計算などに、静力学解析は必要となる。
