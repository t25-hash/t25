# 18・4・1 順動力学と逆動力学

> **章**: 第18章 ロボットアームの力学
> **節**: 18・4 ロボットアームの動力学

a. ロボットアームの運動方程式ロボットアームは複数のリンクが連結された空間多リンク機構であり、その動特性は非線形でかつ各関節相互に力学的干渉が存在する複雑なものである。18・3節まではおもにロボットアームの運動学と静力学について述べられてきたが、18・4節ではロボットアームの動力学について述べる。対象とするロボットアームは前節までと同様に、シリアル機構の剛体アームを前提とする。まず図18・9に示すような自由度のロボットアームを考える。図18・9に示すように、台座に固定されたリンクをリンク0とし、順次手先に向かってリンク1、リンク2,...とし、手先のリンク(エンドエフェクタが取付けられたリンク)をリンクとする。このようなロボットアームの各リンクの運動エネルギーと重力によるポテンシャルエネルギーからロボットアーム全体のラグランジュ関数を求めることにより、最終的に次式のようなロボットアームの運動方程式を得る。の第要素τは、ジョイントが回転ジョイントのときはジョイント軸回りの回転トルクを、直動ジョイントのときにはジョイント軸方向の並進力を表す。b. ロボットアームの順動力学問題式(18・79)は、ある関節駆動力を与えたときにロボットアームがどのような運動をするかを規定するものであり、通常運動方程式を解くとはこのように駆動力の結果生じる運動を求めることを意味する。具体的には式(18・79)を次式のように変形する。計算機シミュレーションなどでは、式(18・80)を時間軸に沿って繰返し解くことになる。式(18・80)の計算と式(18・81)(18・82)等の積分計算とをあわせて順動力学問題と呼ぶこともある。c. ロボットアームの逆動力学式(18・79)を形式的に次式で表す。順動力学問題の目的は計算機シミュレーションであったのに対し、逆動力学問題はロボットアームの軌道制御(trajectory control)に深く関係する。このように式(18・84)などで得られたある目標加速度を実現する関節駆動力を式(18・85)により計算で求めて制御に用いる方法を計算トルク法(computed torquemethod)という。計算トルク法の簡便なものとして、式(18・85)において修正加速度の代わりに単に目標加速度を代入して得られた関節駆動力をフィードフォワード項として加える方法もある。ロボットアームの軌道制御に計算トルク法を用いるとアームの動特性が補償されるので、高加減速を含む目標軌道でも精度よく追従させることができる。ただし、ロボットアームのリンクの質量や重心位置、慣性テンソルなどアームの動特性を表すパラメータがある程度正確にわかっていることが前提となる。この動特性パラメータ図18・9自由度ロボットアームの同定法については例えば文献(20)を参照されたい。次に目標軌道がアームのエンドエフェクタの位置で与えられる場合についても考えてみよう。まず式(18・3)を1階、2階微分して次式を得る。式(18・90)(18・91)と式(18・85)による制御法も一種の計算トルク法であるが、ルウらによって示されたこの方法を特に分解加速度制御という。分解加速度制御は、関節変数を目標軌道として指定する先の方法と比較すると、エンドエフェクタの位置を目標軌道として直接的に与えることができ、またその応答特性もゲイン行列,によって直接的に指定できる点が特徴である。
