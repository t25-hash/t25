# 18・4・1 項では、ロボットアームの動力学として順動力学問題と

> **章**: 第18章 ロボットアームの力学
> **節**: 2・1 項で定義したとおりである。∈

逆動力学問題について概略を説明した。18・4・2項では、ロボットアームの動力学の具体的な計算方法としてまずラグランジュ法に基づく定式化を行う。運動エネルギーは、式(18・16)で求められた4×4の座標変換行列(同次変換行列とも呼ばれる)を用いて次式で与えられる。または3×3単位行列である。また方向のみが意味をもつでは、4番めの要素には零を付加する。このとき式(18・98)左辺の第1項が慣性項、第2項が遠心力・コリオリ力の項、第3項が重力項に対応する。
