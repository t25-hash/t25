# 2・7・1 純粋物質の相と 関係

> **章**: 第2章 熱力学
> **節**: 2・7 気体・液体・固体・超臨界流体

1種類の分子から構成され一定の化学組成をもつ物質を純粋物質
（pure substance）と呼び，二つ以上の純粋物質から成る混合物
（mixture）と区別する．このような単一の組成をもつすべての純
粋物質は，液体，気体または固体のどの状態においても存在しうる
し，固体はいろいろな同素変態のもとで存在しうる．物質の三つの
状態は一般に相（phase ）と呼ばれ，単一の相から成る物質の状態
は均質であると考えられるのに対して，二つ以上の相すなわち共存
相から成る系の状態は不均質であるという．純粋物質で認められる
共存相の数は最大3相までであり，4相またはそれ以上の共存相が
存在する不均質な系は，混合物の場合にのみ認められる．共存しう
