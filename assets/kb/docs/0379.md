# 18・4・2 項で紹介し

> **章**: 第18章 ロボットアームの力学
> **節**: 2・1 項で定義したとおりである.∈

たラグランジュ法に基づく動力学の定式化は,最初にウィッカー,カーンによってなされた.表18・1に示すように,ウィッカー,カーンの方法では6自由度のロボットアームの場合で乗算が66271回,加算が51548回必要である.これを50kflops程度の1980年代前半当時の標準的な16bitのマイクロプロセッサで計算すると約2.3sかかることになる.これは,制御入力の更新周期は少なくともロボットアームの機械的な共振周波数(10Hz程度)の5倍強の60Hz 以上(すなわち1回の計算は16ms以下)とすべきといわれている要求値からはほど遠く,当時の計算機の性能では計算トルク法を実際に適用するのは事実上不可能であった.
