# 19・1・1 自動車に作用する力

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・1 自動車の振動

自動車はエンジンでタイヤを駆動して加速し,ブレーキで制動して減速し,ステアリングを操作して旋回することができる.そのとき,自動車にはタイヤの接地面で駆動力と制動力が作用するのに加えて車体重心に慣性力が作用する.また,路面の凹凸によってタイヤに上下方向の力,さらには坂路,横風などの環境条件によっても車体に外力が作用する.その他,自動車における振動発生源からも力が作用する.エンジンのトルク変動,回転部アンバランス,歯車のかみあいなどである.一般に自動車の運動は,車体重心を原点Oとして前後方向を,左右方向を,上下方向をとして,座標軸回りの回転を加えた合計6自由度で取扱われることが多い.
