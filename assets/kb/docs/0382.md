# 19・1・2 操縦性・安定性

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・1 自動車の振動

自動車の横方向の運動に関する性能を操縦性・安定性と呼ぶ。具体的には、図19・1に示す軸回りのヨーイング、軸方向の横滑り、軸回りのローリングがおもな運動である。a. タイヤに作用する力自動車が運動を行う場合、おもな力はタイヤに依存することになる。このタイヤを力学的に取扱う座標系を図19・2に示す。タイヤには鉛直荷重、駆動力、制動力、求心力が作用する。b. コーナリングフォースとコーナリングパワータイヤに発生する力とその名称を図19・3に示す。転動するタイヤの回転面と進行方向が異なる場合、それらのなす角を横滑り角と呼ぶ。横滑りするタイヤに発生する摩擦力の進行方向に対し直角成分をコーナリングフォース(cornering force)と呼ぶ。通常の走行状態で横滑り角は5~6度以下であり、横滑り角(slip angle)とコーナリングフォースはほぼ直線関係にある。そして、この傾きを特性値として用いてコーナリングパワーと呼んでいる。c. 自動車の運動方程式自動車の基本的な運動解析は図19・4に示されるように、質量の自動車が一定速度で走行中に重心点回りにθ回転して横滑り角βで旋回する状態が取扱われる。したがって、車両の横方向の運動は次式で表せる。まず1番めは、自動車の定常円旋回時のステア特性について述べる。極低速時の旋回で図19・1 自動車の座標系図19・3 横滑りをもつタイヤに発生する力図19・4 自動車に作用する力は遠心力が零であるが、一般には遠心力が発生するので極低速時の旋回半径とは異なってくる。このように速度を上げて旋回した場合、極低速の旋回半径より大きくなる特性をアンダステア(US)という。逆に、<0の自動車は車速を上げるにつれて極低速の円よりも内側に入るようになる。この特性をオーバステア(OS)という。そして=0の自動車は、車速を上げても旋回半径が変わらない。この特性をニュートラルステア(NS)という。通常、市販されている自動車は弱アンダステアの特性を有する。そしてさらに車速を上げて>になると旋回半径は物理的には意味のない負の値となり静的不安定状態となる。この安定限界速度は式(19・12)の左辺を零とおくことによって次式から導かれる。
