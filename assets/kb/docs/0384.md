# 19・1・4 衝突安全性

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・1 自動車の振動

a. 自動車の安全性能自動車の安全性能は(1)予防安全、2―179図19・5 理想制動力配分線図の例(2)衝突安全、(3)事故後安全の三つに分けられる。予防安全は事故を未然に防ぐための性能でアクティブセーフティと呼ばれており、ブレーキ性能、操縦性・安定性などの性能に依存する。衝突安全は衝突時の乗員および歩行者などの被害を軽減するための性能であり、おもに車体構造、乗員保護装置、操舵ステアリングなどの衝突エネルギー吸収性能などに依存する。事故後安全は衝突後の乗員救出のために容易に開きうるドアシステム、容易にはずせるシートベルトをはじめ、火災などの二次災害防止のための燃料タンク構造や燃料タンク注入口、配管などの配置などが重要である。b. 自動車の衝突自動車が障害物に衝突すると、車体は数十~数百ms(ミリ秒)といった短時間で停止するため車体には大きな減速度が発生する。そのときに、自動車の運動エネルギーを車体のクラッシュで吸収して、乗員の受ける減速度を低減させて被害を軽減することは、衝突安全性能において重要な技術課題である。いま、固定壁への衝突を考える。衝突前の自動車のもつ運動エネルギーの大部分が衝突時の車体変形によって吸収される。したがって、自動車の質量を、速度を、変形量をδとし、衝突荷重は変形量δとともに直線的に増加すると仮定=δすると、次の式が成り立つ。12=δ=δδ=12δ(19・14)したがって、変形量をδ、車体剛性は次式で求められる。δ=(19・15)=δ(19・16)車体のクラッシュ特性δが求まると、次の式により車体の変形によるエネルギー吸収特性δを求めることができる。δ=δδ(19・17)車体のクラッシュ特性およびエネルギー吸収特性は、乗員保護装置とともに衝突時の安全性能に直接結びつく重要な特性である。最近では、実車衝突試験とかなりの精度で合致する衝突シミュレーション技術が開発され、衝突特性の評価に用いられている。これらの手法を大別すると、(1)非線形ばね・質量モデル解析、(2)有限要素モデル解析、およびこれらを組合わせた(3)ハイブリッドモデル解析の三つの手法に分類できる。
