# 19・2・2 蛇行動特性

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・17 によりの関数として与えられる。クリープ力とクリープ

a. 幾何学的蛇行動軸箱支持ばねがなく輪軸単体がレール上をゆっくり転送する場合を考える。輪軸の運動運動方程式(19・27)にてばね力の項、慣性力の項を零とおくと、運動方程式の第1式より次式のようにヨーイングと左右速度の関係が得られる。この方程式は減衰のない単振動の解をもつ。輪軸は正弦波を描いて運動しその波長は輪軸の幾何学的蛇行動波長と呼ばれ次式で与えられる。固有値の実数部から安定性が、虚数部から蛇行動の振動数がわかる。式(19・27)でばねがない輪軸単体の場合には不安定な系である。安定化するために支持剛性が必要である。前後、左右のクリープ係数を等しいと近似してとする。λ,λは安定な実根であり、蛇行動には関係しない。共役複素根が蛇行動特性を表し、実部が負であれば蛇行動は安定、正であると不安定となる。この近似解によれば、蛇行動の限界速度を向上させるためには、幾何学的蛇行動波長を増加させることが一つの方策となるので、等価踏面こう配を小さくすること、車輪半径を大きくすること、軌間を広げることが望ましい。また、支持剛性を硬くする、あるいは輪軸の質量を軽量化して固有振動数を大きくすることが、限界速度向上につながることになる。より一般的に台車、車両の蛇行動特性を検討する場合も基本的な取組みは同じで、連立微分方程式の固有値解析を行えばよい。
