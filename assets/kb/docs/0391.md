# 19・2・4 通り変位のモデリング

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・17 によりの関数として与えられる。クリープ力とクリープ

車両の乗り心地を阻害する外乱として代表的なものはレールの変位である。高低(上下)変位、通り(左右)変位、水準(傾斜)変位、軌間(間隔)変位の各方向の変位が存在するが、輪軸の左右系運動方程式に関連する通り変位のモデリングについて述べる。a. 通り変位考慮法高低変位に対しては車輪とレールは離れることがないので、変位量だけ車輪に上下に強制的な変位が与えら2―185れる系として扱える。したがって一般の機械構造物の定式化と全く同じである。それに対して、通り変位による車両への外乱は、車輪・レール間に働くクリープ力を通して与えられる点が高低変位の場合との大きな違いである。最も基本となる式(19・27)の一輪軸の運動方程式での通り変位の考慮法を以下に述べる。ヨーイング運動方程式のの項は左右車輪の半径差により生じる前後クリープ力による旋回モーメントの項である。この左右車輪の半径差は、輪軸のレールに対する相対左右変位により決まるのでを-で置換えなければならない。注意しなければいけないのは、左右運動方程式のの項である。通り変位はレールの左右位置がずれている状態で、レール自身は振動していないので、-とするのは間違いである。通り変位のない場合と同じ式でよい。b. 絶対軌道変位検測車で検測される通り変位、高低変位は10m弦正矢法によるデータである。実際の軌道変位に対する車両の応答を検討する場合、絶対的な変位に復元された波形を用いなければならない。曲線やこう配走行のときは、曲線線形、こう配線形からの絶対変位である。文献(1) 自動車技術会編、 自動車技術ハンドブック基礎理論編、 (1990)、177、自動車技術会。(2) 自動車技術会編、日本の自動車規格JASO Z 208、(1978)、33、自動車技術会。(3) 景山克三・景山一郎、自動車力学、(1991)、52、理工図書。(4) 安部正人、自動車の運動と制御、(1994)、53、山海堂。(5) 文献(4)の69ページ。(6) 文献(4)の71ページ。(7) 自動車技術会編、新編自動車工学便覧、第2編、(1982)、2-34、自動車技術会。(8) 文献(1)の343ページ。(9) 藤岡健彦、自動車技術、54-10(2000)、104.(10) 谷口修、振動工学ハンドブック、(1987)、1133、養賢堂。(11) 文献(7)の2-74ページ。(12) Janeway,R.N.,SAE J.,56-8(1948)、48.(13) 文献(10)の1136ページ。(14) 文献(10)の1138ページ。クス、(1994)、1-33、電気車研究会。(16) 宮本昌幸、 ダイナミクスハンドブック、 21章ビークルダイナミクス・3節鉄道、(1993)、972-992、朝倉書店。(17) 宮本昌幸・須田義大、 車両の運動力学入門(第1回~24回)、 鉄道車両と技術、1-1~3-7(1995~1997)。(18) 宮本昌幸・須田義大、 車両システムのダイナミクスと制御・4章鉄道車両の運動と制御、(1999)、106-155、養賢堂。(19) 宮本昌幸、 車輪/レールの接触と鉄道車両のダイナミクス、 機械学(20) 文献(15)の付録II.1686.(22) 例えば、Timoshenko,S.and Goodier,J.N,Theory of Elasicity、(1951)、362-382,McGraw-Hill.(23) Kalker,J.J.,Vehicle System Dynamics,8-4(1979)、317.→ Next Chapter
