# 20・1・1 ばねとダンパによる制振と絶縁

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・1 受動的制振および絶縁の理論

機械や構造物に減衰を付加したり、動吸振器で機械の振動を抑えることを制振という。機械で発生した振動が基礎に伝わるのを減らすこと、および基礎の振動が機械に伝わらないようにすることを振動絶縁(vibration isolation)という。なお、建築や土木の分野では地震による揺れを抑えることを免震という。ダンパは速度比例の減衰力を生じるダッシュポットのような減衰要素を意味する場合と、アクティブマスダンパのように減衰装置全体を指す場合がある。ダンパは振動のエネルギーを熱に変えて消散させ、振動を減衰させる。ばねはエネルギーの消散はないが、ばね係数を変えることによって、固有振動数を変化させて、振動や力の伝達を小さくすることができる。a. 外力による振動図20・1に示す質量、ばね、ダンパからなる1自由度振動系を考える。まず、基礎が動かない=0とし、質量に振幅、角振動数ωの調和外力=sinωが働く場合の運動方程式は、++=sinω(20・1)となる。定常応答の変位振幅は無次元化した次の式で与えられる。욦욧=1-+4ζ(20・2)ここで、=ω/ω욈(振動数比)、욦욧=/(静たわみ)、ω욈=/(固有角振動数)、ζ=/2 (減衰比)である。このとき、ばねとダンパを介して床へ伝達する力は、=+(20・3)で与えられる。伝達力の振幅と入力の振幅の比、すなわち力の伝達率(transmissibility)は、욞==1+4ζ1-+4ζ(20・4)となる。b. 基礎変位による振動次に、基礎の変位=sinωによる振動を考える。運動方程式は、+-+-=0(20・5)となる。定常応答の変位振幅と基礎の変位振幅の比、すなわち変位の伝達率は、욃==1+4ζ1-+4ζ(20・6)となり、これは力の伝達率[式(20・4)]と同じ式である。c. ばねとダンパの選択式(20・2)の振幅および式(20・4)(20・6)の伝達率を、それぞれ図20・2および図20・3に示す。一般的に、質量を変えることは現実的でなく、ばね定数と減衰係数を変えることによって、振動や伝達率を減少させる。まず、図20・2の無次元振動振幅/욦욧であるが、≪1、すなわち加振振動数が固有振動数より低い領域では1で、1で共振のピークをもち、≫1ではに比例して減少する。共振ピークは減衰比を大きくすることによって下げることができる。振幅について考えると、≪1では1/に比例し、1では1/に比例し、≫1では1/ωに比例する。よって、振幅を小図20・1 1自由度振動系図20・4 2自由度振動系さくするには、ばね定数および減衰係数は大きいほうがよい。なお、これは自明のことであり、基礎に剛結合すればは零になる。図20・3の伝達率であるが、≪1では、1であり、1でピーク値をとり、=2 で1になり、が≫2 では伝達率はに反比例して減少する。よって、運転領域でのを2 よりも大きくすることによって、伝達率を下げることができる。すなわち、は1/ に比例するので、柔らかいばねを使えばは大きな値となり、伝達率は低下し、振動絶縁がはかられる。ただし、そのときは外力による振動は大きくなる。また、伝達率のピーク値はζに反比例し、2 では伝達率はζに比例する。したがって、減衰係数を大きくすると共振ピークを低下させられるが、>2 では逆に伝導率は増加し、振動絶縁の低下につながる。伝達率を小さくするためには、機械の定常運転の振動数が固有振動数より十分高くなるように柔らかいばねで支持し、ダンパの減衰係数も小さくする。しかし、回転数を上げていくときや下げていくときに共振点を通過し、そこで大きな振動が生じる。これは、例えば洗濯器の脱水において回転数の上昇過程でがたがたと揺れ、定常回転数になると静かになる現象である。よって、ばねおよびダンパの選択においては、制振および絶縁の両者を総合的に考える必要がある。
