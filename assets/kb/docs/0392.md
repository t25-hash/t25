# 20・1・1 ばねとダンパによる制振と絶縁

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・1 受動的制振および絶縁の理論

機械や構造物に減衰を付加したり、動吸振器で機械の振動を抑えることを制振という。機械で発生した振動が基礎に伝わるのを減らすこと、および基礎の振動が機械に伝わらないようにすることを振動絶縁(vibration isolation)という。なお、建築や土木の分野では地震による揺れを抑えることを免震という。ダンパは速度比例の減衰力を生じるダッシュポットのような減衰要素を意味する場合と、アクティブマスダンパのように減衰装置全体を指す場合がある。ダンパは振動のエネルギーを熱に変えて消散させ、振動を減衰させる。ばねはエネルギーの消散はないが、ばね係数を変えることによって、固有振動数を変化させて、振動や力の伝達を小さくすることができる。a. 外力による振動図20・1に示す質量、ばね、ダンパからなる1自由度振動系を考える。定常応答の変位振幅は無次元化した次の式で与えられる。b. 基礎変位による振動次に、基礎の変位=sinωによる振動を考える。c. ばねとダンパの選択式(20・2)の振幅および式(20・4)(20・6)の伝達率を、それぞれ図20・2および図20・3に示す。一般的に、質量を変えることは現実的でなく、ばね定数と減衰係数を変えることによって、振動や伝達率を減少させる。共振ピークは減衰比を大きくすることによって下げることができる。よって、振幅を小図20・1 1自由度振動系図20・4 2自由度振動系さくするには、ばね定数および減衰係数は大きいほうがよい。なお、これは自明のことであり、基礎に剛結合すればは零になる。よって、運転領域でのを2 よりも大きくすることによって、伝達率を下げることができる。ただし、そのときは外力による振動は大きくなる。また、伝達率のピーク値はζに反比例し、2 では伝達率はζに比例する。伝達率を小さくするためには、機械の定常運転の振動数が固有振動数より十分高くなるように柔らかいばねで支持し、ダンパの減衰係数も小さくする。しかし、回転数を上げていくときや下げていくときに共振点を通過し、そこで大きな振動が生じる。これは、例えば洗濯器の脱水において回転数の上昇過程でがたがたと揺れ、定常回転数になると静かになる現象である。よって、ばねおよびダンパの選択においては、制振および絶縁の両者を総合的に考える必要がある。
