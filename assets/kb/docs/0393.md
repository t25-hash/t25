# 20・1・2 動吸振器による制振と絶縁

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・1 受動的制振および絶縁の理論

図20・1の振動系を制振するには減衰を付加すればよいが、現実の機械や建物ではそれが難しい場合が多い。ここでは図20・4に示すように、,からなる副振動系を付加し、2自由度系にし、反共振を利用して制振する。動吸振器(dynamic damper)の理論に関しては6章にて詳しく論じられているので、ここでは、動吸振器による制振および絶縁について簡単に説明する。これは反共振を利用した制振である。外力の周期が一定でない場合は、減衰を小さくすると、二つの共振点で大きなピークをもつので、ピークの最大値を最小にするという規範で,を近似的に求める定点理論(fixed points theory)が一般に用いられている。この周波数応答曲線を図20・5の実線で示す。よって、動吸振器は絶縁にも制振と同様に有効である。鉄道の枕木の下の砕石、自動車のタイヤ、電気製品のゴム製の足など振動や衝撃が付随するものには古くから振動絶縁の工夫がなされている。振動絶縁を防振ともいい、それに使用するばね、ダンパを防振材料、それらを組合わせた支持機構を防振機器ということが多い。また、制振材料というとサンドイッチ鋼板など主振動系自体の振動を抑える高減衰材料を指す場合が多い。
