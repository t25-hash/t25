# 20・2・1 絶縁用ばね

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・2 振動絶縁用ばね・ダンパ

絶縁用ばねを素材で分類すると,金属ばね,空気ばね,ゴムばねになる.弾性特性としては,荷重とたわみが比例する線形ばね,たわみが大きくなると硬くなる漸硬ばね(hardening spring),逆に柔らかくなる漸軟ばね(softening spring)がある.また,荷重を加えていくときと減らしていくときで荷重-たわみ曲線でループを描く,すなわちヒステリシスをもつばねもある.摩擦をもつばねがこれにあたる.実際のばねの選定に当たっては,防振で要求される特性,すなわち,ばね定数,減衰特性,ストローク,動作方向,温2―187(a)(b)(c)図20・7 皿ばね図20・6 重ね板ばね図20・9ダイヤフラム型空気ばね図20・8 ベローズ型空気ばね度などの使用環境によって最適なものが選ばれる.a. 金属ばね. コイルばねコイルばね(helical spring)は最も多く使用されているばねであり,圧縮型と引張型がある.一般的に,コイルばねで支持した振動系の固有振動数は数Hz 以上となり,それより高振動数域での絶縁が可能となる.また,減衰は小さいので,ダンパを並列使用する場合が多い.他の特性としては,線形性が良い,安価で構造が簡単,常用温度範囲が広い,耐油性・耐老化性が良いなどが挙げられる.また,ばね自体が質量をもつので,分布振動系を形成し,高振動数領域に複数個の固有振動数をもつ.よって,そこで共振が起こる可能性があるので注意を要する.また,横方向の変位に関しては柔らかくなるので,ガイドなどが必要な場合もある.. 重ね板ばね重ね板ばね(lamellar spring)は図20・6に示すように,長さの異なる板ばねを重ねたもので,古くから鉄道車両やトラックの懸架ばねとして使われてきた.この特徴としては,板の間で摩擦が生じ,摩擦減衰特性をもつことにある.また構造上,主負荷方向以外の変位に対しては剛であることがコイルばねとの違いである.. 皿ばね皿ばね(coned disc spring)は図20・7に示すように,茶たくのような形状のコンパクトな板ばねである.特徴としては,形状によるが,ある変位域で負のばね定数をもつ場合がありうることである.図20・7(b)に示すように,複数の皿ばねを同じ向きに重ねるとばねが剛になり,同図20・7(c)のように逆向き互い違いに重ねるとばねが柔らかくなりストロークが長くなる.b. 空気ばね空気ばね(air suspension)は空気の圧縮弾性を利用したもので,自動車,鉄道車両,大型プレス機械,半導体生産設備の微振動対策など幅広く利用されている.構造としては,図ベローズ型は胴部をゴム膜のベローズで形成し,それを上面と下面の金属板に接合したものである.ダイヤフラム型は,ゴムのダイヤフラムを金属製の内筒と外筒に接合したものである.空気ばねで支持した振動系の固有振動数は1~5Hz とすることができ金属ばねより低い固有振動数が実現できる.また,空気の流路に絞りを設けることにより,減衰をもたすことができる.圧縮空気を補給するシステムをもっている空気ばねでは静的な高さ調整ができる.常用温度範囲,耐油性・耐老化性などはゴムを用いているので,金属ばねに比べると劣るが,適切に使えば長期間の使用に耐える.c. ゴムばね一般に防振ゴム(rubber isolator)と称されて,多く使用されている.金属の上面板と下面板の間にゴムが挟まれて2―188いるが,ゴムをせん断変形させるもの,圧縮変形させるもの,その両者を用いるものがある.また,変形方向は水平3方向,軸方向,それに回転方向があり,それらによって形状は異なるが,いずれも直方体,円柱,M 型など簡単な構造である.また,内部減衰をもつので,弾性と減衰を併せもつ多方向弾性支持が実現できる.また,液体を封入しオリフィスによって減衰を付加したゴムマウントが自動車のエンジン支持などに使用されている.d. 積層ゴム地震対策としてのビルの免震には,鉛直方向の自重に耐えかつ水平方向には柔らかいばねが必要である.それを満足するのが,軸方向の荷重には硬く,せん断方向には柔らかい積層ゴム(laminated rubber)である.これは,図20・10に示すように,薄い金属板とゴムを積層したものである.ゴムに圧縮荷重がかかると図20・10(a)のように側面にゴムがはみ出る.しかし,図20・10(c)のように薄く積層されると,側面へのはみ出しが拘束されて,非常に高い剛性となる.すなわち,図20・10(a)は通常の縦弾性係数,(c)は体積弾性係数に支配されるので,積層することによって100倍から1000倍も硬くなる.せん断方向に関しては両者とも同じせん断弾性係数に支配される.ゴムのせん断弾性係数は縦弾性係数の約1/3である.よって,鉛直方向には十分硬く,水平方向には柔らかいばねが実現できる.積層ゴムを使った免震建物は,1985年から2001年までに,日本で約800棟建設された.e. その他のばね以上,典型的なばねについて述べてきたが,そのほかに,木,コルク,皮,フェルト,じゅうたんなどもばねとダンパの機能をもっている.また,複数の素材を組合わせたものが多く使用されている.例えば,ゴムパッドの中にコイルばねを埋込んだスプリングパッドもある.スチールワイヤは摩擦減衰と弾性をもった金属ばねとして使用されている.
