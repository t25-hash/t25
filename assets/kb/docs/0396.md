# 20・3・1 振動制御の目的と手法

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・3 能動的振動制御

振動制御の目的は、制御対象を平衡点近傍に保持する、あるいは出力をより小さな値にすることであり、制御の分野においてはレギュレータ問題(regulator problem)と呼ばれる領域に含まれる。入力された外乱によって生じた平衡点からの制御対象のずれを制御アクチュエータの発生する操作力によって抑制する。このときの制御対象の平衡点を与える座標系に、慣性(絶対)座標系と運動(相対)座標系の問題がある。これはとりもなおさず絶対振動と相対振動の問題と考えられる。振動問題の多くは人の振動の知覚に関係する絶対振動の問題であるが、機械の精度に関係しているのは相対振動の問題である。相対振動の問題は、位置・姿勢の制御問題とも関係して、機械・構造物のサーボ問題(追従制御問題)と呼ばれ、レギュレータ問題(振動制御問題)とともに重要な工学的関係を有する。振動制御の問題をレギュレータ問題としてとらえるときに、外乱の入力のありようから大きく二つに分けることができる。一つは振動の伝達に関する絶縁問題であり、もう一つは力入力に対する応答に関する制振問題である。絶縁問題は系の支点が揺らいだときに慣性系にいかに力を伝えないかという問題であり、制振問題は慣性系に加わった力に対していかに揺れないようにするかという問題である。受動要素によって構成された範囲において、絶縁は支持剛性が柔らかいほうが有利であり、制振は堅いほうが有利であるということはよく知られている。受動要素においては絶縁と制振を同時に実現する条件は質量を大きくする以外にはない。しかし制御技術を用いることによって双方の要請を満たすより自由な構成が可能となる。振動制御対象としての機械・構造物をモデル化すると、質量要素、減衰要素、ばね要素とし、その質点の変位をとすることによって図20・15(a)のように表現することができる。質量に対して、ばね・減衰は質量要素を支点に拘束する役割を果たしており、一種のフィードバック要素として機能している。言い換えるといずれの変位においても作用力を発生する。このような慣性座標系から支持したダンパはスカイフックダンパ(a) MCK モデル(b) スカイフックMCK モデル図20・15 振動要素モデルと座標系図20・16 振動要素のブロック表現図20・17 振動伝達特性とスカイフック効果(skyhook damper)と呼ばれ、理想の振動制御要素として概念化されている。ただこの関係はダンパだけではなく質量要素やばね要素においても同様な関係が成立することから、ここではそれらを含めてスカイフック要素と呼ぶ。この関係をブロック図で表したものが図20・16である。支点の運動と無関係な減衰要素は、機械・構造系にとって多くの利点をもたらし、能動的振動制御(active vibration control)の導入の契機にもなっている。その最も大きな利点は、スカイフックダンパが絶縁と制振のトレードオフを解消できることである。通常の減衰要素の場合には減衰作用を大きくすると絶縁能力が低下することは避けられないが、支点の運動と関係しないスカイフックダンパは絶縁能力と無関係に減衰を大きくできる。慣性系からの支持は具体的に絶対速度に比例するダンパと考えられるから、絶対速度が計測可能であれば、能動振動制御において制御力をスカイフックダンパとして作用させることが可能となる。同じように絶対変位に比例するスプリング要素や、絶対加速度に比例する質量要素のように仮想物理要素を制御対象に付加して、系の物理パラメータの性質を変更することができる。図20・17に支点の振動から質点の振動までの振動伝達特性の変化をスカイフック要素別に示した。左からばね要素、減衰要素、質量要素を順に大きくしている(ばね、質量を単独に大きくすると減衰比が下がるために付加的に減衰も大きくしている)。これらの効果は受動的な物理要素を変更しても得られない。基本的なフィードバック制御について語ったが、目的の実現のためにはフィードフォワード制御を併用することが高い効果を生む。例えば絶縁問題であれば、支点の振動を計測してその出力を零にするようなフィードフォワード制御を形成することによって、フィードバック制御では実現できない目的を達成することも可能となる。また制振問題においても、精密機器のステージ移動に伴う反力消去に同技術が適用されている。二つの制御方式を併用することによる実用的なメリットは大きい。
