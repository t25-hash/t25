# 20・3・2 除振装置の振動制御

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・3 能動的振動制御

a. 除振装置の制御とその目的精密機器の支持機構である除振装置(vibration isolation system)を例にとって具体的な振動制御について述べる。除振装置の制御目的には、(1)振動の絶縁、(2)直接外乱の制振、(3)位置・姿勢の保全、を挙げることができ2―19 1図20・18 除振装置の剛体系モデル化る。従来、精密機器の除振は空気ばねのような剛性の低い除振材料を用いて、床からの振動の伝達を極力小さくするということが行われてきた。しかし、最近の精密機器は機器の内部になんらかの振動源を含み、さらには機器の生産性の要請に伴って大きな加工対象を早く移動、あるいは加工することが求められている。この要請に応えようとすると、柔らかい支持機構では精密機器に揺れが発生して、生産性に支障がでてしまうのである。受動的な除振技術をもってすると、機械自身を実用性がないほど重くする以外に方策がないことから、能動的振動制御が用いられるようになってきているのである。さらに、最近の機器は単独で存在することはまれで、周辺のシステムと結合してその機能を果たしている。例えば半導体の製造ラインではシリコンウェハがいろいろな装置の間を行き来し、露光装置は外部光源とアライメントが必要であるなど、装置の位置・姿勢の制御も重要な要素となっている。位置・姿勢の制御は、その目的からして振動絶縁とトレードオフの関係にあり、結局除振装置の制御は先に挙げた三つのそれぞれに相反する目的を同時に実現しようというものである。b. 除振装置の構成とモデル化除振装置は質量要素を剛体としてモデル化される。精密機器は一般に剛性は高く設計されており、除振性能を検討する領域においては剛体モデルだけで十分な場合が多い。図20・18に示す6自由度の剛体モデル(rigid model)の生成について述べる。このとき個別座標の質量要素とばね要素は位置ベクトルで与えられる座標変換マトリックスを用いると、原点の直交座標系に変換することができ、式(20・12)のような6自由度表現が与えられる。また、ここには記さないが減衰マトリックスも同様に与えることができる。c. 制御系設計と特性剛体モデルを可観測(observable)、可制御(controllable)に制御するためには、独立した6点以上の信号と6点以上のアクチュエータが必要とされる。このとき用いるセンサ、アクチュエータは制御座標(直交座標やモード座標)の信号や力をそのまま出力できるわけではなく、個別座標から制御座標に変換することが必要である。もちろんセンサ、アクチュエータをできる限り同じ位置、同じ向きに配置して直接フィードバック制御を行うという手法(direct velocity feedback)も実用的であるが、ここではより一般的な手法として座標変換を用いる。センサの場合と同様に、アクチュエータの向きと位置座標を使って変換マトリックスが定義できる。図20・19にモード座標系で記述した除振装置の制御ブロック図を示すが、このようにモデル化された制御対象に対するコントローラは制御理論を用いて求めることができる。制御理論の中でも最も代表的なものは最適レギュレータ(optimal regulator)理論である。この理論は応答と制御力の二次形式(linear quadratic)の和を制御の評価関数として用い、これを最小とする制御を実現するものであり、LQ制御とも呼ばれている。
