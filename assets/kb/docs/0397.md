# 20・3・2 除振装置の振動制御

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・3 能動的振動制御

a. 除振装置の制御とその目的精密機器の支持機構である除振装置(vibration isolation system)を例にとって具体的な振動制御について述べる.除振装置の制御目的には,(1)振動の絶縁,(2)直接外乱の制振,(3)位置・姿勢の保全,を挙げることができ2―19 1図20・18 除振装置の剛体系モデル化る.従来,精密機器の除振は空気ばねのような剛性の低い除振材料を用いて,床からの振動の伝達を極力小さくするということが行われてきた.しかし,最近の精密機器は機器の内部になんらかの振動源を含み,さらには機器の生産性の要請に伴って大きな加工対象を早く移動,あるいは加工することが求められている.この要請に応えようとすると,柔らかい支持機構では精密機器に揺れが発生して,生産性に支障がでてしまうのである.受動的な除振技術をもってすると,機械自身を実用性がないほど重くする以外に方策がないことから,能動的振動制御が用いられるようになってきているのである.さらに,最近の機器は単独で存在することはまれで,周辺のシステムと結合してその機能を果たしている.例えば半導体の製造ラインではシリコンウェハがいろいろな装置の間を行き来し,露光装置は外部光源とアライメントが必要であるなど,装置の位置・姿勢の制御も重要な要素となっている.位置・姿勢の制御は,その目的からして振動絶縁とトレードオフの関係にあり,結局除振装置の制御は先に挙げた三つのそれぞれに相反する目的を同時に実現しようというものである.b.
