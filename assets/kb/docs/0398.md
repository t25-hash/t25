# 20・3・3 構造物の振動制御

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・3 能動的振動制御

a. 構造物の制御とその目的構造物、特に住宅や高層ビルに焦点を当てるなら、振動制御の目的としては居住環境の快適性、安全性への要請がおもなものである。居住環境への外乱としては、支点外乱としての入力と、直接に構造物に力が作用する直接振動としての入力の二つである。支点外乱の要素としては交通振動、地震が代表的である。直接振動の要因としては風、内部利用者、設備機器などがある。これら外乱に対して絶縁問題として考える場合と制振問題として考える場合があるが、ここでは制振問題としての振動制御に焦点を当て、さらに構造物全体が振動するような低周波の水平振動モデルに限定するものとする。能動的制振技術の構造物への代表的な適用方法は、付加的な質量をアクチュエータで駆動してその反力を制御対象構造物に操作力として作用させる方式である。この方式による振動制御装置はアクティブ制振装置(active damper)あるいはアクティブマスダンパと呼ばれている。高層ビルなどの構造物に対して最も有効に制御力を作用させることができるのは屋上などの高所であり、反力構造を使って直接に操作力を構造物に作用させることを期待することはできないため、このような付加質量の慣性力が用いられている。この手法は効果的に直接振動を抑制することができるが、大入力の地震動に対しては付加質量のストロークの制約もあり、十分な制振力を与えることが実用上困難であることが示され、大入力時には制御を打切り、受動的な制振機構で対応するなどの工夫もなされている。しかし、このような限定はありつつも微小入力から大入力まで広い範囲に効果を有する構造物振動制御の代表的手法として用いられている。は制振装置の設置位置(階層)を示すベクトルで、0... 1は設置位置が最上階であることを示す。このとき操作力は最上階のみに作用している。制振装置にばね・減衰要素を含ませると、制振装置はその固有振動数で振動を増幅する特性を示す。その動特性を利用してアクチュエータの動力(エネルギー消費)を小さくするハイブリッド型の制御方式[図20・20(b)]が提案されて用いられている。この場合の制御モデルは制御対象に制振装置のダイナミクスを含むように拡張したモデルを作成して、そのダイナミクスを生かす制御解を求めるという手法が取られる。状態量ベクトルは簡単のために同じ記号を用いている。このようなモデルに対して、制御器を与えるアルゴリズムに多くの手法が提案されているのは除振装置の場合と同様である。ハイブリッドの手法で動力の低減効果が得られるのは、装置の固有振動数付近に限定され、多質点モデルの構造物の多くの固有振動数を制御対象とするときには利用できない。また、モード座標に変換することで、除振装置の場合と同じように独立な制御器を構成することができる。
