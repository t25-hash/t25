# 20・3・3 構造物の振動制御

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・3 能動的振動制御

a. 構造物の制御とその目的構造物、特に住宅や高層ビルに焦点を当てるなら、振動制御の目的としては居住環境の快適性、安全性への要請がおもなものである。居住環境への外乱としては、支点外乱としての入力と、直接に構造物に力が作用する直接振動としての入力の二つである。支点外乱の要素としては交通振動、地震が代表的である。直接振動の要因としては風、内部利用者、設備機器などがある。これら外乱に対して絶縁問題として考える場合と制振問題として考える場合があるが、ここでは制振問題としての振動制御に焦点を当て、さらに構造物全体が振動するような低周波の水平振動モデルに限定するものとする。能動的制振技術の構造物への代表的な適用方法は、付加的な質量をアクチュエータで駆動してその反力を制御対象構造物に操作力として作用させる方式である。この方式による振動制御装置はアクティブ制振装置(active damper)あるいはアクティブマスダンパと呼ばれている。高層ビルなどの構造物に対して最も有効に制御力を作用させることができるのは屋上などの高所であり、反力構造を使って直接に操作力を構造物に作用させることを期待することはできないため、このような付加質量の慣性力が用いられている。この手法は効果的に直接振動を抑制することができるが、大入力の地震動に対しては付加質量のストロークの制約もあり、十分な制振力を与えることが実用上困難であることが示され、大入力時には制御を打切り、受動的な制振機構で対応するなどの工夫もなされている。しかし、このような限定はありつつも微小入力から大入力まで広い範囲に効果を有する構造物振動制御の代表的手法として用いられている。b.
