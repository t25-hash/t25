# 20・4・1 振動絶縁(vibration isolation)

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・4 振動・衝撃絶縁

ポンプ、送風機、コンプレッサなどの振動が外部に伝わらないように、あるいは建物内外からの振動が精密機械や測定装置に伝わらないように、振動絶縁が行われる(後者の場合は除振とも呼ばれる)。振動絶縁装置(vibration isolator)(または防振装置または除振装置)の代表的なものとして、金属ばね・オイルダンパ、防振ゴム、空気ばねなどが挙げられる。a. 金属ばね・オイルダンパ(metal spring-oil damper unit)図20・21(a)(b)に示すように、質量の機械をばね定数の金属ばねと粘性減衰係数の線形特性をもつオイルダンパで支持した場合を考える。図20・22からわかるように、オイルダンパは共振を抑える効果は大きいが、高振動数領域での振動絶縁には有害である。この高振動数領域におけるオイルダンパの欠点は、ダンパに直列にばねを図20・21 振動絶縁系モデル2―19 4機械力学図20・22 振動伝達率(ばね・オイルダンパ)図20・23 振動伝達率(防振ゴム)入れる(すなわちダンパを弾性支持する)ことによって改善される。b. 防振ゴム(rubber isolator)防振ゴムは比較的簡単な構造をもち、取付けも容易であるので、各種機械・機器の振動絶縁用として広く用いられている。防振ゴムはヒステリシス減衰をもっているので、その動特性を1+γのように複素表示するのが合理的である。ここで、γは損失係数と呼ばれる。式(20・32)を図示すると図20・23のようになる。図用はほとんどない反面、共振を抑える効果はあまり大きくないことがわかる。c. 空気ばね(air spring) 空気ばねはベローズ型、ダイヤフラム型およびそれらの複合型に分類され、ばね特性の異なるさまざまな形のものが使用されている。また、空気ばねと補助タンクとの間の空気通路に絞りを入れることによって減衰効果を与えることもできる。質量の機械を空気ばねで支持したときの等価振動モデルは図20・24(a)のように表され、その振動伝達率は次式のように表される。実際に使用されている各種金属ばね、空気ばね、防振ゴムおよび(a) 等価モデル(b)振動伝達率図20・24 空気ばね系の等価モデルと振動伝達率(b) 伝達力波形(a) 衝撃絶縁モデル図20・25 衝撃絶縁モデルと床への伝達力波形オイルダンパの形状および特性が文献(16)に示されている。
