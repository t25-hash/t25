# 21・1・1 耐震技術の変遷

> **章**: 第21章 耐震設計
> **節**: 21・1 耐震工学の基礎と地震

1923年の関東大震災の経験から、1922年に内藤多仲により提案された静的震度法による耐震設計法が1924年に市街地建築物法に取り入れられた形で一般に普及した。これがその後、建築基準法(construction standard act)(1950年制定)となり、1980年に同施行令の改訂が行われた。機械構造物を含む産業施設の耐震設計(aseismic design)についての研究は、土木や建築構造物に対するものと異なり、その歴史は浅く、1960年代以降に発展を見た。1964年の新潟地震および1968年の十勝沖地震による産業施設の被害体験が、その後の産業施設の耐震設計技術の発展の大きな契機となった。特に、大型石油タンクなどの貯槽類や配管系の耐震研究は、精力的に産官学の連携の基で進められた。1971年のサンフェルナンド地震では産業施設やライフライン系の施設に大きな被害を出し、その教訓から産業業種別に、耐震設計や地震防災に関する技術的規制、基準、指針が制定されていった。また、原子力発電施設については、1970年の本格稼働、施設開業以降、重要度分類という新しい設計思想を軸に、国家的な規模で耐震設計化技術が進められ、巨大地震の経験がない状況のもとで、例えば1982年四国の多度津に設立された積載最大重量1000tの大型振動台によって重要な施設の耐震性を実証している。一方、1981年に建築物の耐震設計に対して弾塑性変形時に伴う保有耐力の概念を導入した新耐震設計法が施行されたこと、さらにアメリカにおいて原子力発電プラントの配管サポートなどにエネルギー吸収効果を期待する新しいダンパが開発された影響もあり、1980年以降の機械構造物の耐震技術にも大きな変化が現れてきた。免震床によるコンピュータ機器などの支持技術、塔状構造物や高層建築物への能動・準能動制振法の適用、粘弾性材、制振材などを導入した新型ダンパの開発などが精力的になされてきた。さらに、1995年の兵庫県南部地震(阪神・淡路大震災)における多くの生産施設の被害経験から、今後の耐震設計が経済性を考慮した生産活動の軸の中でとらえられる必要が増してきている。
