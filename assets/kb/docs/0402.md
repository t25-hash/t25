# 21・1・2 震害

> **章**: 第21章 耐震設計
> **節**: 21・1 耐震工学の基礎と地震

阪神・淡路大震災では多くの生産施設に被害を与えた。この地震による代表的な設備・機械の損傷形態、損傷モード(mode of fail-ure)を述べる。損傷モードは、例えば、以下のようにいくつかのグループに分類される。(1) 重量構造物、立型構造物の転倒・落下(図21・1)(2) アンカーボルトの破断などによる機械類の移動、転倒(3) 工場建屋などの支持構造系の倒壊や、変形に連動した損傷・落壊(4) 支持構造物系との共振による振動破壊(5) 地盤・支持基礎の液状化損傷に伴う機械類の沈下・傾斜・水没図21・1 横中ぐり盤の転倒2―19 7(6) 内容物のスロッシング(sloshing)による格納容器の座屈および内容物の流出このほか、主として電気系統や制御機器系の途絶や電力、ガスを含むエネルギー供給系にかかわる損傷・被害も多数あった。(1)は明らかに予想を上回る水平加速度の作用により生じたもので、実際、長軸が東西方向に取付けてあった多くの暖房用ボイラや冷凍機が滑動破損していた。(2)は、(1)のほか、上下動の影響によるものも多く見られ、床面に固定した約10cmのアンカーボルトを飛び越えて移動した工作機械もあった。次に(3)、(4)の被害パターンは、主として建屋や支持構造物との連成効果によるものであるが、次の2点に特徴が見られた。(a) 主構造物が変形または圧壊したために、機械設備などの支持機能を失った(天井クレーン、エレベータなど)(b) 建屋や下部構造物系の振動により、それと共振して引きちぎれたり、落壊した(ジブクレーン、屋上型空気清浄機など)これらとは別に、支持系と機械類とを結合する装置にも両者間の相対的変形により数多くの損傷が生じた。配管つり下げ用ハンガ、発電用大型ボイラの振止め装置などがその例である(図21・2)。(5)の液状化による被害は、大型構造物としての貯槽類やコンテナクレーンでは支持基礎の沈下による傾斜や座屈として現れた(図いは水冠、水没の被害を受けたのも、液状化がおもな原因であった。この地震では、(6)のスロッシングによる被害はそれほど顕著ではなかった。震源が近かったこと、主要動の継続時間が短かったことなどがその理由であるが、一方では、生産工程の中で、高温のチョコレート溶液や製靴工場のアルミニウム溶液の飛散という現象も見られた(図21・4)。
