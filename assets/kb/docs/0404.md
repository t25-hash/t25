# 21・1・4 応答スペクトル

> **章**: 第21章 耐震設計
> **節**: 21・3 )。また、多くの工場内の機械類が、傾斜による芯変位、ある

耐震設計の考え方が他の振動工学と異なる点は応答スペクトル(response spectrum)の概念であるが、その詳細は21・3・3項へ譲り、ここでは地震波形の振動数特性を表す一つの方法としての応答スペクトルの考え方を述べる。これは固有振動数(固有周期)と臨界減衰比で1質点減衰ばね系を代表させ、その系の地震波入力に対する応答の最大値(最大変位応答、最大速度応答、最大絶対加速度応答)をとる。この最大値と系の固有振動数(固有周期)を臨界減衰比をパラメータとして図示したグラフが応答スペクトルであり、各々変位応答スペクトル、速度応答スペクトル、加速度応答スペクトルと呼ぶ。図21・6に1940年のインペリアルバレー(ImperialValley)地震のエルセントロ(El Centro)波の代表的な応答スペクトルを示す。一般に1~5%の臨界減衰比に対するものが多く作成され、この範囲で凹凸の激しい折線グラフを呈する。実用的には平均、標準偏差分の上積、包絡などの方法で平滑されたもの、さらに数段の折線で近似されたものなどが設計用として使われる。代表的な応答スペクトルとしては、ハウスナーの平均応答スペクトル、土木研究所による平均応答スペクトル、コンビナート保安・防災技術指針によるスペクトル、大崎スペクトルなどが提案されている。また、地震動の経時特性のうち最大応答の発生に関するものは応答スペクトルに含まれているが、継続時間および応答の時間経過に関するものは含まれていないので、別途規定することが多くの場合必要である。
