# 21・2・1 耐震設計(aseismic design)の概要

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

世界有数の地震国であるわが国においては、機械構造物は地震に対して安全でなければならないことが基本である。この場合、一律に大規模地震に耐えうるように設計することが望ましいと考えられがちではあるが、地震の強さとその発生確率および設備が損傷を受けたと仮定したときの影響の度合いをあわせて考えて設計することが合理的である。また、地震災害が構造的なものに対してのみならず、機能的な面への影響の重要性が指摘されるとともに、根本的に当該構造物への地震入力自体を低減させる免震構造や、当該構造物の減衰性能を向上させる意味での制振構造が着目され、広い意味での新たな耐震技術として注目されている。わが国の耐震設計技術は、(特に建築/土木分野において)地震学、地震工学の発展とともに成熟していったが、その歴史は、1880年の横浜地震(M 5.5~6)にさかのぼる。同年には日本地震学会が、1886年には造家学会(現日本建築学会)がそれぞれ設立された。1891年に発生した濃尾地震は7000人以上の命を奪うとともに、建築物をはじめとする構造物に甚大な被害をもたらした。濃尾地震の翌年1892年には震災予防協会が設立、これが耐震構造研究の幕開けとなった。1913年には、日本建築学会が指導して東京市建築条例案が公表され、ここで木造、石造、れんが造、鉄骨造、鉄筋コンクリート造に対する構造規定が初めて定められた。1919年には市街地建築物法および施行令が制定され、現在の長期許容応力度に相当する許容応力度が規定されるとともに、100尺(約30.3m)という高さ制限が設けられた。1923年の関東大震災を受けて1924年に改定された市街地建築物法では、耐震規定が追加され、0.1の水平震度が導入された。また同年、日本建築学会は構造強度計算規準を発行し、水平力に対する応力計算法が提案された。これらが耐震設計基規準の始まりとなる。関東大震災時、東京下町の地震動は水平震度に換算して0.3程度であったこと、また当時の基準による材料許容応力度は材料強度の1/3程度であったことから、水平震度を0.1に設定すれば、材料強度を超える応力は作用せず、したがって建築物は崩壊することはないという議論が当時水平震度を0.1に設定した背景にあるといわれている。その後、死者3700人あまりを出した1948年福井地震を経て、1950年には建築基準法が制定され、それまでの震度0.1が0.2へと2倍に引上げられた。しかし、震度0.1の時代においては、地震力に対して長期許容応力度が参照されていたものを、震度0.2としてからは、長期許容応力度の約2倍の値をもつ短期許容応力度が参照されるようになったので、実設計においてそれほど大きな変化はなかったといえる。さらに、戦前の100尺制限を踏襲し、31mの高さ制限が設けられた。その後、1964年新潟地震、1968年十勝沖地震、1978年宮城県沖地震など建築物に大きな被害をもたらした地震があったものの、1981年の建築基準法大改定に至るまで、震度0.2が耐震設計における地震力の基本として位置付けられていた。高度経済成長期に入った日本では、高層建築物への期待が高まり、また耐震工学研究・技術の進展ともあいまって、1964年に日本建築学会は高層建築物技術指針を刊行、翌1965年には31mの高さ制限が撤廃され、高層建築物建設への道がひらかれた。これら高層建築物の設計に対しては、建築基準法諸規定を必ずしも満足する必要はなく、高度な解析や最新技術を適用して安全な高層建築物を設計することとし、またそれを構造評定という識者らによる審査を経ることによって確認する手順をとることになった。1981年の建築基準法改定に伴って、設計地震力は大幅に見直され、そこでは、中小地震(レベル1地震)と大地震(レベル2地震)という二つのレベルが設定された。中小地震に対しては、ベ一スシア係数の基準値として0.2、また大地震に対しては、べ一スシア係数の基準値として1.0が設定された。ベ一スシア係数0.2は震度0.2にほぼ対応するもので、この係数に対して短期許容応力度が参照されていることからも、中小地震に対する設計地震力は震度0.2時代とそれほど変わるものではない。大地震に対しては1.0という大きな係数が採用されたが、そこでは塑性変形とそれによるエネルギー消費が勘案され、具体的には係数(震動減衰性、各層の性に関係する係数)を用いて構造物が保有すべき耐力が低減される。係数は構造物がもつ塑性変形能力に応じて0.25~0.5の2―19 9値を取るので、大地震に対するべ一スシア係数1.0にこの値を乗じることから、構造物はベ一スシア係数換算で0.25~0.5程度の耐力を保有すればよいことになる。中小地震に対する設計から派生する余力や別途チェックされる変形の制限などによって、べ一スシア係数0.2に対して短期許容応力度設計された構造物の多くは(特に塑性変形能力に優れ小さな係数でよいとされる純ラーメン構造物は)、べ一スシア係数換算で0.25~0.3程度の耐力を自動的に保有している。つまり、べ一スシア係数0.2に対する短期許容応力度設計が耐震設計を支配している。このように、震度0.1の時代と1981年改定後を見ると、見掛けの設計地震力ははるかに増加しているものの、その実態はそれほど変わっていないという側面もある。さらに、1999年の建築基準法の改定に伴い、性能設計の概念が取入れられることとなった。このような二段階レベルの地震動に対して構造物を設計する考え方は、兵庫県南部地震以降、さまざまな分野に適用され基準化されていった。各基準における考え方に多少の差はあるものの、おおむね、レベル1地震動は、構造物の供用期間中に当該地点において発生する確率の高い地震動と定義され、およそ50~100年の再現周期のものであり、基本的に構造物は弾性設計範囲内の応答に収まることが求められる。一方、レベル2地震動は、構造物の供用期間中に当該地点において発生する確率は低いが大きな強度をもつ地震動と定義され、現在から将来にわたり当該地域で考えられる最大級の地震動とされる。一方、原子力発電所の耐震設計に採用された重要度分類の考え方は、わが国では石油コンビナートなどの二次災害の危険性の高い重要な産業施設の耐震設計問題に適用されていった。21・2節冒頭にも述べたように、発生確率の小さな大地震に対する耐震設計手法としては、この重要度分類の考え方を取込むほうが合理的であるとの考えから、広く産業施設の耐震設計に用いられている。ここで、一般的な耐震設計の流れを図21・7に示し、各要件について
