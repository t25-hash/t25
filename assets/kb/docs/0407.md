# 21・2・2 重要度分類

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

重要度分類は原子力耐震設計で始まった概念である。もし、ある施設において設計対象に耐震性がなく地震により損傷を生じたと仮定し、それにより(i)周辺公衆の安全、(ii)システム全体の機能の維持にどの程度の影響を与えるかを評価し、その想定被害に応じてその耐震設計の程度(重要度)を定めるというのが、重要度分類とそれに応じた耐震設計といえる。(i)については、その設備が耐震設計されていないと仮定し、地震で全損状態になったときの周辺の被害を評価して定める。評価方式としては、(a)具体的に被害を算定する方式、(b)被害を与えるであろう因子群から求めた総合的な指数を算定する方式、(c)設備に内蔵される被害を与える可能性のある物質の量と、周辺の人口密度分布とを主体とする表(表21・1に一例を示す)による方式、(d)設備の安全に関する機能分析に基づく表による方式が現在主として使われている。原子力発電所は(d)の方式によっている。これらについて、同じ地点にある施設が同一の地震によって破損する程度が異なるのはおかしいという意見もある。(ii)については、いわゆるライフラインなど、その機能維持が社会の機能維持に多大な影響を与える施設が第一に挙げられる。一般の工場などの生産施設については、工場からの災害の拡大を防ぐための設備の機能維持と、生産設備のそれがある。前者については(i)に準じて考えることが必要であり、後者はいわゆる法規制による安全設計などとしての耐震設計の対象とはならないが、施設所有者としては損傷による諸損失と設備の耐震化に要する経費とを均衡をとって考えるべきものである。
