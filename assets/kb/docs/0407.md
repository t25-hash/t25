# 21・2・2 重要度分類

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

重要度分類は原子力耐震設計で始まった概念である.もし,ある施設において設計対象に耐震性がなく地震により損傷を生じたと仮定し,それにより(i)周辺公衆の安全,(ii)システム全体の機能の維持にどの程度の影響を与えるかを評価し,その想定被害に応じてその耐震設計の程度(重要度)を定めるというのが,重要度分類とそれに応じた耐震設計といえる.(i)については,その設備が耐震設計されていないと仮定し,地震で全損状態になったときの周辺の被害を評価して定める.評価方式としては,(a)具体的に被害を算定する方式,(b)被害を与えるであろう因子群から求めた総合的な指数を算定する方式,(c)設備に内蔵される被害を与える可能性のある物質の量と,周辺の人口密度分布とを主体とする表(表21・1に一例を示す)による方式,(d)設備の安全に関する機能分析に基づく表による方式が現在主として使われている.原子力発電所は(d)の方式によっている.これらについて,同じ地点にある施設が同一の地震によって破損する程度が異なるのはおかしいという意見もある.(ii)については,いわゆるライフラインなど,その機能維持が社会の機能維持に多大な影響を与える施設が第一に挙げられる.一般の工場などの生産施設については,工場からの災害の拡大を防ぐための設備の機能維持と,生産設備のそれがある.前者については(i)に準じて考えることが必要であり,後者はいわゆる法規制による安全設計などとしての耐震設計の対象とはならないが,施設所有者としては損傷による諸損失と設備の耐震化に要する経費とを均衡をとって考えるべきものである.半導体工場のように生産密度が高く,また,1999年に発生した台湾921地震で見られたように,その損害が世界的な経済への影響を生じうる施設に対しては,先に述べた免震構造等の地震入力自体を低減するような新しい耐震表21・1重要度分類の例(第3種毒性ガスおよび可燃性ガスにかかわる耐震設計構造物の重要度分類)10未満10以上100未満100以上1000未満1000以上10000未満10000以上20未満III율I율I율20以上40未満IIII율I율I율40以上90未満IIIII율I율I율90以上200未満IIIIIII율I율200以上400未満IIIIIIIII율400以上900未満IIIIIIIIII900以上2000未満IIIIIIIIIII2000以上IIIIIIIIIIII備考::貯蔵能力(tonf)塔槽類の外面から当該耐震設計構造物が設置される事業所の境界線(当該境界線に連接する海,河川,湖沼またはこれらと同等の効用を有する施設もしくは土地がある場合は,その外縁)までの距離のうち最短のもの(単位m)重要度I율設備:その損傷もしくは機能喪失が,事業所外の公衆,公共財産,環境に壊滅的損害を与えるおそれのあるもの重要度I設備:その損傷もしくは機能喪失が,事業所外の公衆,公共財産,環境に多少の損害を与えるおそれのあるもの重要度II設備:その損傷もしくは機能喪失が,事業所外の人命を損なうおそれのないもの重要度III設備:通常の耐震性を有するもの技術を積極的に導入することも考えられる.
