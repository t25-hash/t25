# 21・2・3 設計地震動(design basis earthquake)

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

軽水炉耐震設計の基本方針は原子力安全委員会の発電用原子炉施設に関する耐震設計審査指針に示されているように,建設地点ごとに想定されるいかなる地震力に対してもそれが大きな事故の誘因とならぬよう設計することであり,現実には起こりえないような限界的な地震が発生しても原子炉は冷却材の喪失事故を生じずに安全に停止・維持されるようになっている.上記の地震力を与える地震としては,発電所建設地点を中心とした広い範囲の地域にわたる地震学的調査結果をもとに,以下に示す2種の地震,すなわち設計用最強地震Sと設計用限界地震Sが想定されている.設計用最強地震S:歴史的資料から過去において,敷地またはその近傍に影響を与えたと考えられる地震が再び起こり,敷地およびその周辺に同様の影響を与えるおそれのある活動度の高い活断層による地震のうちの最も影響の大きいもの.設計用限界地震S:地震学的見地に立脚し設計用最強地震を上回る地震について,過去の地震の発生状況,敷地周辺の活断層の性質および地震地体構造に基づき工学的見地からの検討を加え,最も影響の大きいもの.一方,一般の機械構造物などではその耐震設計において建築分野で用いられる地震波を用いることが多い.例えば発電用大型ボイラなどは日本建築センター高層建築物高層評定委員会においてその設計に関しての審査が行われる場合が多い.そこで,以下にここで用いられることの多い設計用地震波について紹介する.(財)日本建築センターの高層建築物構造評定委員会が作成した構造建築物(高さ60m以上)の設計ガイドラインである高層建築物の動的解析用地震波についてでは,設計用地震動波形として,少なくとも以下に示す3種類以上の既往の観測波形を用いることを原則とすると述べている.(1) 標準的な地震動波形:例えば,エルセントロNS,タフトEW 等(2) 地域特性を表すような地震動波形:例えば,東京-101 NS,大阪-205EW,仙台TH 0-1FL EW 等(3) 長周期成分等を含む地震動波形:例えば,八戸NS,EW 等地震動の強さは地表面位置での最大速度で定義し,地震活動度が低い地域以外では,レベル1で25kine(cm/s)以上,レベル2で50kine(cm/s)以上を要求している.したがって,建築構造物の動的解析に用いる最終的な地震動の時刻歴波形は,観測地震動の最大速度をレベル1とレベル2に応じてそれぞれ規準化したものとなり,実記録の最大化速度値とは異なったものとなる.しかしながら,そのスペクトル特性は当然のことながら観測地震動のそれと同一となる.このような考え方は,強震動は元来不規則で予測できないものであるという前提に立っており,地震動特性が異なる複数の既往の観測波を用いることにより,強震動の不規則性や予測の困難さの問題を克服しようとしたものである.ここで,(1)から(3)に挙げた観測地震動の中から,特によく用いられてきたいくつかの地震波について簡単に紹介しておく.エルセントロNS 波:1940年5月18日にアメリカ,カリフォルニア州南部で発生したインペリアルバレー地震(M 7.1)の際に,断層から約10km離れたエルセントロという場所の建物地下室に設置された強震計により記録された.地盤は,よく締まった洪積地盤であり,地表から20mはP波速度360m/sの粘土層,それ以深ではP波速度1800m/sのローム層,厚いれき,砂,粘土層が続いている.なお,この地震では構造物に対する著しい被害は生じていない.この記録は,1940年当時では世界で数少ない強震記録で,その後標準的な設計用地震動波形として非常によく使用されてきた.最大加速度340gal(cm/s)強で,最大速度40kine(cm/s)弱と大きく,スペクトル特性が比較的なだらかな特徴がある.タフトEW 波:1952年7月21日にアメリカ,カリフォルニア州中部で起こったカーン・カウンティ地震(M 7.8)の際に,震源距離約40kmの位置にあるタフト・リンカーン・スクールに設置された強震計で記録された.地盤はよく締まった洪積地盤で,最大加速度は176gal(cm/s)弱,最大速度20kine(cm/s)弱であり,El Centro NS 波とともによく用いられ,わが国においても発電用ボイラ耐震設計の標準的な設計用地震動波として知られている.八戸NS 波:1968年5月16日に発生した十勝沖地震(M 7.9)の際に,震央から約180km離れた八戸港の地表に設置された強震計で記録された.地盤は表層10mは値が10から20程度の軟弱な沖積層である.八戸では,鉄筋コンクリート造の建物の被害が著しく,最大加速度は200gal(cm/s)強,最大速度40kine(cm/s)弱であり,継続時間が長く,やや長周期の波形成分を多く含むという特徴があり,(3)の長周期成分を含む設計用地震動波形としてよく用いられてきた.東京101 NS 波:1956年2月14日に発生した埼玉地震の際に東京大学地震研究所の地下1階で記録されたもので,最大加速度は100gal(cm/s)以下であるが,東京の地域特性を表す地震動波形として用いられてきた.その他,高層ビルや免震構造物の設計用地震入力波として,近年よく用いられているものに通称臨海波と建築センター波がある.先に述べたように既往の観測波をその最大速度を25kine,50kineに引伸ばし,慣用的に耐震設計に用いてきた経緯がある.堅固な工学的基盤上に基礎を直接設置できる場合には,大きな問題はないとされてきていたが,堅固な地盤が地中深くにあるような軟弱地盤では,地表で地震動を規定すると基礎の検討に不都合が生じる.そこで,震源断層を考慮した入力地震動評価により得られる工学的基盤上での地震動を設計で採用するためにこうした地震波が開発された.臨海波は東京の臨海部の工学的基盤である東京れき層や江戸川層を対象に,東京周辺で起こった過去の地震を想定して,断層モデルを含め,さまざまな方法で地震動を評価し,応答スペクトルのレベルを定めたものである.想定した地震は,関東地震,安政江戸地震および東海地震であるが,関東地震が他の二つの地震の評価結果をほぼ満足していることから,関東地震に対して評価されたレベルをレベル2地震動として定めている.建築センター波の元となるスペクトルは,臨海波と同じく,東京における工学的基盤を念頭に,安政江戸地震,関東地震,東海地震の影響を種々の方法で検討し,それらをほぼ包絡する形で求めたものである.臨海波との大きな違いは,おもに東海地震による地震動が大きく評価されたため,スペクトルの長周期側が周期10秒まで100kine(cm/s)フラットになっている点である.
