# 21・2・4 損傷様式(mode of failure)

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

産業施設内の機械構造物は建築物、橋りょうなどに比べ、一般的にその力学的形状は複雑である。したがって、地震時にどの部位において、どのように、そしてどの程度損傷あるいは破壊(損傷とは誤動作も含め機能を失うこと、および破壊することを含む。破壊とは設備の全体または一部が過度の変形もしくは破断に至ることをいう)を生じるかを、設計時に正確に把握することが重要となる。そのためには理論的考察もさることながら、実際の地震での損傷例をできるだけ数多く集めて、分類・分析することが重要である。表表21・2 地震動による損傷機構の例設計変数損傷機構衝撃破壊剛体応答加速度動的応答共振応答力基準変位応答強制変位・変形地盤・支持物変状断層変位変位基準2―201してはさらにどの部位が、どのような損傷機構(瞬時の破断、切損、圧壊、座屈など、また累積性、進行性損傷、例えば低サイクル疲労、ラチェッティング、き裂進展、延性変形進行など)で、全体系として損傷に至るかを検討する必要がある。このような観点から、実地震の被害調査は極めて重要なものとして位置づけられ、1995年に生じた兵庫県南部地震の地震被害調査においては地震工学にかかわる各学会において速報版的な報告書が発行されるとともに、地盤工学会、土木学会、日本建築学会、日本地震学会、日本機械学会の5学会からの代表委員らにより阪神・淡路大震災調査報告編集委員会が設置され、全26巻からなる公式な地震被害調査報告書が刊行された。その中で、地震被害のデータベース化の試みも行われた。さらに、近年になり、1999年のトルココジャエリ地震、台湾921地震、2001年のインド西部地震と世界的に見ても、巨大災害地震が頻発していることから、地震被害調査活動を効率的にかつ正確に実施するための地震被害調査マニュアル作成も進められている。表21・2の内容は大別して、(a)振動的被害、(b)強制変形被害、(c)断層被害、(d)地盤液状化被害になるが、いわゆる耐震設計解析は(a)が主体であり、(b)がそれについで解析的である。(c)は、わが国では地表に破壊力のある断層が出現することは、むしろまれであるとされていて、アメリカにおけるパイプラインについてのような設計解析はほとんど実施されていない。(d)になると発生予測の解析は行われても、全体としては地盤改良等による対策が中心となる。地震被害調査においては、損傷の直接の誘因が何であるかを判断し、またその損傷過程を現地にて判断する必要に迫られることから、高度な専門的知識と判断力が要求されるため、その体制作りと、若い研究者/技術者のこの種の調査への積極的な参加が求められる。
