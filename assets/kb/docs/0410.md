# 21・2・5 振動特性解析・応答解析

> **章**: 第21章 耐震設計
> **節**: 21・2 は力学的な損傷の分類の例であるが,前述のように設計に際

損傷の様式が定まると,それに基づき力学モデルを構成し,その振動特性解析を行う.一般に振動特性は固有振動数,振動モード,減衰特性から成り立っている.このうち前二者は対象構造物が決定すれば,それを適当にモデル化することにより計算で精度よく(少なくとも工学的には)求めることができる.しかしながら,減衰特性に関しては,例えば制振構造の場合のように特に大きな減衰要素を人為的に付加した場合を除いては,計算で求めることは一般的に困難である.したがって,これは通常,振動実験などにより実験的に求められる.構造物において,減衰は構造部材の結合部や構造物内可動物に生じる摩擦,がた,衝撃によって,運動エネルギーが消散するものと,構造物からこれを支持する基礎・地盤や他の結合された構造物へ逸散していくものがある.地震時における構造部材の降伏など材料そのものによるエネルギー吸収は,一般的な機械構造物における通常の設計限度である弾性ないしは弱い弾塑性領域では大きく寄与しない程度であり,問題ないとされている.また,大型な構造物で地震時においても弾性変形が支配的となる原子炉建屋などでは,地盤に逸散するものが主たるものとなり,計算で比較的精度よくこの量を推定できる.構造物の地震応答解析については21・3節において詳細に述べられるので,ここでは結果の評価に対する留意点について記載する.すなわち,より精度の高い解析手法を用いたほうが,設計の信頼性が向上するものと一般に信じられているが,地震を含む設計条件に対しては精度は向上するが,自然現象である地震そのものには多くの不確定な因子が含まれており,実際の地震時における設備の挙動の推定精度が向上するわけでないことにまず留意する必要があるということである.このような統計的な性質は入力である地震波自体および出力である応答結果ともに存在する.柴田らによれば,応答倍率の揺らぎは意外に大きく,特定地点のデータをまとめて解析した結果は,変動係数(標準偏差/平均値でと呼ばれる)で30%程度となり,さらにその地点に固有の地震については,3σを超えた応答倍率を示すこともあることが指摘されている.地震動特性を表す応答スペクトル曲線は,一般的にいくつかの地2―202機械力学震応答スペクトルを統計的に処理して求められる.歴史的には,ハウスナーが提案した平均値をさらに平滑化したものに始まり,その後,平均値に標準偏差にある倍率を乗じたものを上乗せしたものや,包絡したものなどが提案されている.形状的にも滑らかなもの,折れ線で示されるもの,短周期側で値を減じているもの,最大値をそのまま長周期側に延長しているもの(例えば免震構造用設計スペクトルなど)など種々のものが,対象構造物ごとにその特性に合わせて個別の議論がなされ実務的に用いられている.
