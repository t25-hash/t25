# 21・2・6 許容応力

> **章**: 第21章 耐震設計
> **節**: 21・2 は力学的な損傷の分類の例であるが,前述のように設計に際

鋼構造の許容応力の体系は最大主応力説に基づくものと,最大せん断応力説に基づくものとがあり,その基本的な考え方は,建築・土木のものと機械系のものと分けて考えることもできる.機械設備を支持する構造物,鉄筋コンクリート,基礎などについては建築系の許容応力度体系の影響を強く受けている.建築・土木系では,常時荷重に対して長期許容応力度,変動荷重に対しては短期許容応力度とし,後者を50~100%大きくとっている.一方,圧力容器などを中心とした機械系の許容応力は,当初は主応力説により,弾性範囲内の設計であったが,原子力施設のためのASMEのBoiler andPressure Vessel CodeのSection IIIが整備されてからせん断応力説に基づく考え方が普遍的になってきた.応力種別,運転状態,シェイクダウン,材料疲労などの考え方を取入れて,かなり合理的になっており,原子力発電施設以外の分野へも適用可能である.運転状態を正常,異常,緊急,損傷と分け,前2状態は平常ありうることとして,基準の応力値を用い,緊急状態は寿命中に特定回数しか起きないこととして,平常時の120~150%増しの応力値を用い,若干の弾塑性状態を許す.また,損傷状態はあってはならない状態であるから,万一そのような状態になった状態でも,周辺の安全は確保できることを目標に塑性状態の評価を行って,安全機能維持を確認する.この考え方は,一般の危険性をはらむ施設について,適宜,簡略化して用いることができ有用である.なお,許容応力以外の許容限界はその対象ごとに,実験その他の方法により検討して決めることが多く,例えば,高温材料ではクリープとの関連でまだ定まった手法が確立していない.また,一方,低温における材料脆性によるものも含め,脆性材料の破壊限度の設定法も明確でなく,脆性材料を避けるか,欠陥の検出を厳重にするかのいずれかによっている.
