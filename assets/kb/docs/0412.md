# 21・2・7 設計細目(general design criteria)

> **章**: 第21章 耐震設計
> **節**: 21・2 は力学的な損傷の分類の例であるが、前述のように設計に際

設計上の細部に関する制限事項で、計算や解析によらず、耐震性を増加することである。前出の脆性材料の使用制限、配管支持部材の形状の標準化、アンカーボルトの工法の制限、防振架台の振幅制限装置の使用、緊急遮断弁の装着方法の指定などがその例であるが、建築・土木分野に比べて遅れているとの意見もある。一般に設計解析にのらない点で耐震性を改善すること、もしくは構造を簡略化して設計解析を実施しやすくすること、システム工学的観点から全体系としての安全を増加するようなことが対象となることが多い。建築の分野では構造細目と呼ばれる。
