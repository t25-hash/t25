# 21・3・1 振動モデル

> **章**: 第21章 耐震設計
> **節**: 21・3 機械構造物の地震応答解析

狭義の耐震設計は地震動によって発生する慣性力に対する構造強度設計であり,地震応答解析の目的は,この慣性力によって構造物に作用するせん断力や曲げモーメントなど(特にその最大値)を求めることである.対象構造物の振動モデルは応答解析,さらには耐震設計の良否を左右する重要なものであるが,経験によるところが大きい.一般に,地震応答解析のための振動モデルの作成には次の点に留意する必要がある.(1) 振動特性(固有円振動数や固有モード等)を模擬できること.(2) 想定する種々の損傷モードに対して,クリティカルな箇所の応答が算出できること.(3) 安全側の評価結果を得ること.振動モデルの一例として,BWR の原子力発電所・機器設備の耐震設計に用いられる振動モデルを示す.BWR の原子炉格納容器には,MARK-I型とMARK-II型がある.最近では,保守点検作業性の向上等を図ったそれぞれの改良型が一般的になっている.図図21・9にそのMARK-II改良型の地震応答解析モデルを示す.建屋と地盤の連成を考慮している.地震動は建屋の振動特性によって増幅されて,機械設備の据付位置から機械設備に入力される.原子炉建屋と圧力容器支持構造物は連成運動を行う系とし,各部分の剛性は曲げとせん断を考慮して求め,質量を集中質量とした多質点系モデルとしている.モデルの質点位置は,配管ノズルの取付位置,構造体の板厚変化点,主要応力評価点などが考慮されて定められている.一般の機器は建屋とは非連成でモデル化される.(a) MARK-I改良型原子炉格納容器(b) MARK-II改良型原子炉格納容器図21・8 BWR 原子炉格納容器図21・9原子炉炉内構造地震応答解析モデル図21・10原子炉再循環配管系のモデル化の例図21・10はBWR 型原子炉再循環配管系の振動モデルである.モデルの質点は,配管支持装置取付け点,ポンプ,バルブなどの設置点,配管口径変化点,ノズル位置などを基準として,応力評価上重要な位置に定められている.また,配管系の曲り部については剛性の変化が考慮され,バルブ,ポンプなどについては等価な剛性と質量が考慮されている.
