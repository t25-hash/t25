# 21・3・3 地震応答スペクトルを用いたモード解析法

> **章**: 第21章 耐震設計
> **節**: 21・8 にMARK-I,MARK-II改良型を示している。また、

入力地震動の時刻歴波形が与えられると、21・3・2項の式(21・14)から数値積分によって1自由度系の時刻歴応答が求められ、さらに式(21・15)によって多自由度系の時刻歴応答が求められる。しかし、耐震設計では、応答の最大値だけを必要とする場合がほとんどである。この最大値を簡便に求める方法として、地震応答スペクトルを用いたモード解析法があり、耐震設計での標準的な応答解析法になっている。例として、エルセントロNS 波(1940年アメリカのインペリアルバレー地震、最大加速度342cm/s)の応答スペクトルを図21・11に示す。このように、設計に用いる地震波の応答スペクトルを作成しておけば、1自由度系でモデル化される機器の最大応答量はただちに読み取ることができる。また、原子力施設の耐震設計用標準地震動の設計スペクトルは、設計用最強地震に対応する基準地震動と設計用限界地震に対応する基準地震動のそれぞれに応じて選ばれた地震規模と震央距離との組合せをもとに作られている。図21・12に低地震帯用および高地震帯用の標準用水平地震動(臨界減衰比0.05)のスペクトルを示す。しかも、質量比の関係から建物と非連成で取扱うことのできる場合が多い。そのために、建物の床応答加速度を入力地震動とする応答スペクトル(通常は加速度応答スペクトル)が原子力発電所・機械設備の耐震設計などでよく用いられている。これを床応答スペクトル(floor response spectrum)と呼ぶ。図21・12にその一例を示すが、建物の固有周期と一致した周期でピークをもつような応答スペクトルとなる。機械設備の耐震設計では、人工地震波を建屋への入力として計算した設置点の応答波形をもとにして作成した床応答スペクトルにその変動幅を考慮して固有周期領域について(一般に10%)拡幅したものを一般に用いる。図21・13はBWR 原子炉建屋の標高18mの床面における拡幅した設計用床応答スペクトルの例である。b. モード応答の合成法
