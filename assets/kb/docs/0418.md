# 21・3・4 代表的な機械構造物の地震応答解析

> **章**: 第21章 耐震設計
> **節**: 1・2 ・d項参照)。

原子炉炉心はどのような炉型の場合にも複雑な振動特性を有しており、BWR 炉心では燃料集合体群と流体間の連成を考慮した解析、PWR 炉心では燃料集合体どうしの衝突を考慮した解析がなされている。また、高温ガス炉炉心、CADU 炉炉心についての解析も研究されている。配管系の地震応答解析は、特に原子力施設において重要であり、支持点から異なった入力地震動を受ける場合の多入力応答解析、減衰比評価などが研究されている。さらに、配管系と支持装置との間のがたや摩擦を考慮した研究も数多く行われている。
