# 21・4・1 基礎式

> **章**: 第21章 耐震設計
> **節**: 21・4 液体貯槽の地震応答解析

液体貯槽(liquid storage tank)の地震応答には、(1)やや長周期地震動(数秒から十数秒の成分)に対して、自由液面が大きく動揺するスロッシング(sloshing)現象を呈すること、(2)短周期地震動(数秒以下の成分)に対して、液体と貯槽間の連成振動を呈する特徴をもっている。ここでは、貯槽に関してこれらの振動を表す支配方程式について触れる。図21・14に示す軸対称の剛体貯槽内の全流体系のラグランジアンは自由表面の波高ηを一般化座標、速度ポテンシャル(velocitypotential)φをそれに共役な一般化運動量とみなすと質点系の力学とのアナロジーにより次のような形が期待できる。このときハミルトンの原理は次のようになる。こうして変分式(21・25)(21・26)は液体の運動方程式と等価である。
