# 21・4・1 基礎式

> **章**: 第21章 耐震設計
> **節**: 21・4 液体貯槽の地震応答解析

液体貯槽(liquid storage tank)の地震応答には、(1)やや長周期地震動(数秒から十数秒の成分)に対して、自由液面が大きく動揺するスロッシング(sloshing)現象を呈すること、(2)短周期地震動(数秒以下の成分)に対して、液体と貯槽間の連成振動を呈する特徴をもっている。ここでは、貯槽に関してこれらの振動を表す支配方程式について触れる。図21・14に示す軸対称の剛体貯槽内の全流体系のラグランジアンは自由表面の波高ηを一般化座標、速度ポテンシャル(velocitypotential)φをそれに共役な一般化運動量とみなすと質点系の力学とのアナロジーにより次のような形が期待できる。=φ옿η옿-2Δφ+(21・25)ここでは自由表面を表し、は液体の占める領域を表し、液体の密度を1としている。は重力加速度である。このときハミルトンの原理は次のようになる。η,φ=0(21・26)δ욧ここで、自由表面波高ηとφを独立な変数として、容器壁面では옿옿φ=0を考慮しつつ変分を実行すると次の2式を得ることができる。図21・15二次元長方形貯槽内液のスロッシング振動解析の座標系図21・16 液体貯槽の解析モデル化2―205옿φ옿+12∇φ+η=0(21・27)옿η옿-옿φ옿+옿φ옿옿η옿+옿φ옿옿η옿=0(21・28)これらの式のうち式(21・27)はベルヌーイの積分を自由表面に適用したもので、圧力=0としたものに等しく、式(21・28)は自由表面での連続の式に等しい。こうして変分式(21・25)(21・26)は液体の運動方程式と等価である。
