メモリLSIの基本的な動作状態は，大別すると読出しと書込み
とチップ非選択の三つである．読出しと書込みとは，チップ選択の
状態でもある．状態の種類を真理値表に示す．この三つの状態を区
別するためにチップ選択信号CS（chip select ）と書込み制御信号
WEが使われる．説明図では，チップ選択をCS＝1，チップ非選
択では，CS＝0としている．読出し状態では，CS＝1かつWE＝
0，書込み状態では，CS＝1かつWE＝1になるものとしている．
この逆に，読出し状態でWE＝1，書込み状態でWE＝0とするこ
ともできる．
チップ選択信号CSは複数個のメモリLSIを接続して，大きな
容量のメモリとして使う場合に便利な端子である．チップ非選択の
ときには，データ出力はハイインピーダンス状態になる．こうすれ
ば，複数チップの出力を数珠つなぎにしたとき，その線には，チッ
プ選択されたチップのデータ出力が現れる．もちろん，2個以上の
チップを選択してはならない．
読出し状態では，アドレスの指定する場所に（あるメモリセル
に）記憶されていたデータが出力端子DO（dataout ）に読み出さ
れる．書込み状態では，アドレスの指定する場所に（あるメモリセ
ルに）書込みデータDI （datain）端子の信号を書き込んで記憶さ
せる．
入力端子としては，アドレスと書込みデータと書込み制御の3種
類がある．また，出力端子は，読出しデータの1種類である．この
例は，最も基本的なメモリLSIの例を示していて，これ以外の入
出力端子をもつものもある．
