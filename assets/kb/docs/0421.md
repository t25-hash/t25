g.乗算器（multiplier ）と除算器（divider ) 2個の二進数を
乗算する回路を乗算回路という．加算器1個を用いて，加算を繰返
すことにより，乗算を実行できる．最初に，積を保存するレジスタ
に0を記憶する．乗数の最下位ビットが1であれば，0に被乗数を
加算し，積とする．0であれば，加算を行わない．その後，被乗数
を左に1ビットシフトする．これは，被乗数を2倍したことに相当
する．次に，乗数を右へ1ビット右へシフトする．これは，乗数を
1/2にすることに相当する．この後，乗数の最下位ビットが1であ
れば，0に被乗数を加算し，積とする．0であれば，加算を行わな
い．その後，被乗数を左に1ビットシフトする．次に，乗数を1ビ
ット右へシフトする，といった動作を繰返す．このようにシフトと
加算を繰返すことにより積が求まる．これを逐次乗算法という．
高速化の手法としては加算の回数を減らすブースの方法などがあ
る．また，複数個の加算器を配列状に並べて，加算をハードウェア
で行う高速化の方法としては，部分積を木状に加算するウォレスの
木（Wallace tree）を用いた乗算法，冗長二進加算を木状に実行す
る方法，およびLUT（look-up table）を用いる方法などが知られ
ている．これらは，単位回路が規則正しく，繰返し配列されるの
で，LSI化に適している．
