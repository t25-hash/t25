# 21・5・1 耐震診断と耐震補強

> **章**: 第21章 耐震設計
> **節**: 21・5 耐震性向上技術

建築物に対しては、耐震診断、耐震補強という工法は確立している。既存の建築物には老朽化しているものが多いだけでなく、旧規定としての建築基準法をもとに設計、建設されたものも多数残存している。したがって、新しい基準に定められた設計用地震動に対応する地震動を受けたとき、その安全性が保障できない可能性がある。既存の建築物が現行基準が要求している耐震強度を有しているか否かを判定することを耐震診断という。耐震診断の結果、耐震強度が不足していると認定されたものには必要な耐震補強を施すことになる。建築物に対しては、例えば「改訂版既存鉄筋コンクリート造建物の耐震診断基準・耐震改修設計指針・同解説」(日本建築図21・17 耐震診断と耐震補強のフロー図21・18 免震技術と制振技術の構成防災協会)など、耐震診断や補強の工法についての基準等が施行され、実施例も多い。図21・17には、建築物に対する耐震診断と耐震補強のフローを示している。
