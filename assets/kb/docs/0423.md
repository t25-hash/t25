# 21・5・2 耐震性向上法

> **章**: 第21章 耐震設計
> **節**: 21・5 耐震性向上技術

建築物に限らず一般に構造物の耐震性能を向上させるには次の4種の方法が考えられる。(1) 耐震構造強度(剛性)向上法(2) 耐震変形能向上法(3) 地震応答低減法(4) その他の方法(上記3方法の併用を含む)(1)は旧来から多く採用されている方法であり、建築物を例にとれば壁や柱などの増強、ブレースの増設などが相当し、剛性の増加により、固有振動数は高域側にシフトする。(2)の方法は、構造要素に弾塑性域で大きな変形能をもたせ、その性を向上させようとするものである。建築物でいえば、柱の性を増やすために金網や帯板、炭素繊維を巻く工法などがある。これらに対して、(3)の方法は対象構造物の特性自体を変更せずに、免震技術、制振技術を適用して、免震装置、制振装置等を付与することにより、地震動による構造物の応答を低減することにより耐震性の向上を図ろうとするものである。建築物と異なり機械系の構造物、設備、配管などは、それらに要請されるさまざまな機能により構造の形態、設置条件などが定められていることが多く、耐震性向上のために新設および既存の機械自体の構造要素の変更(例えば、配管の直径や肉厚を増加すること)は不可能なことが多い。したがって、以下では(3)の方法を主体にまとめることにする。なお、(4)のその他の方法としては、(1)(2)(3)の方法の併用によるものがあるが、建築物を例にとれば、重要性の低い設備や耐力を担わない壁などを撤去して重量を軽減する方法や構造体の脆弱部やき裂発生部に樹脂などを注入して経年寿命を向上させる方法などがある。
