# 21・5・4 産業機械システムとしての耐震性向上技術

> **章**: 第21章 耐震設計
> **節**: 21・5 耐震性向上技術

特定の構造物や設備に対する耐震性の向上対策については,すでに21・5・3項までに記した方法や技術が定められたり,推奨されている.しかし,機械工業,機械工学に密接に関連する産業機械システムの耐震性向上を考えると,各種の機器,設備,ネットワークシステムなどの結合システムとしての対策を講じる必要がある.1999年度から2年間,新エネルギー・産業技術総合開発機構(NEDO)は,(財)エンジニアリング振興協会に委託して産業機械システムの耐震防災技術に関する調査研究を実施した.ここでは,情報技術(IT)を活用したリアルタイム対震防災技術総合システム構想が提案されている.具体的には,耐震防災技術を(1)事前対策防災システム,(2)リアルタイム地震防災システム,(3)早期復旧システムの三つの基幹システムに分類し,そのそれぞれを次のような技術の総合として構築することを提案している.(1) 事前対策防災システム耐震構造化技術として総合化する.内容的には,(a) 大変位対応,即時起動型免震システム(b) 三次元免震および能動制振システム(c) 損傷破壊評価技術システム等の構築によって生産機能の維持を企てる.(2) リアルタイム地震防災システムこのシステムは,化学プラント,焼却炉,加熱炉,半導体製造など各種重要産業施設を対象とする即時安全起動技術(安全停止/起動システム,震度予測システム,警報システムなど)と産業集中地域,重要産業地域を対象とする早期被害センター技術(被災度診断システム,被災情報収集システム,避難誘導総合システムなど)から成り,主として人的被害の軽減と二次被害の軽減および応急対応技術の構築を企てる.(3) 早期復旧システムこのシステムも既存の重要産業施設や鉄道などのライフライン施設等に対する迅速な復旧計画やそのための技術指導を実施するために構築されるシステムで,モジュール組立復旧機械,鉄道等の迅速復旧用機械,および遠隔操作復旧用機械などの諸システムから構成される.近い将来,これらの地震防災技術が,内閣府,国土交通省,産業経済省,文部科学省など国家機関を中核とする大型プロジェクトとして開発されることが期待されている.文献(1)
