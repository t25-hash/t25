# 22・1・1 レイリー法(Rayleigh method)

> **章**: 第22章 振動・音響の計算法
> **節**: 22・1 近似解法

減衰や摩擦などエネルギー散逸を無視し、エネルギー保存則を用いて線形系の基本振動数を求める代表的な方法で、単にエネルギー法とも呼ばれている。最も簡単な例として、1自由度のばね質点系を考える。一方、ポテンシャルエネルギーは、ばねに蓄えられるひずみエネルギーであり次式で与えられる。/をレイリー商(Rayleigh quotient)という。一般の複雑な系に本法を適用するには、まず、境界条件を満足する変形モードを仮定することが必要である。仮定するモードとしては種々考えられるが、レイリーは任意の仮定した変形モードから求めた固有振動数は、真の値よりもつねに高くなることを示した。これは、真の固有モードに対して、仮定された変形モードはなんらかの外部拘束を与えたことになっているため、見掛け上系の剛性が高くなることから、固有振動数も高くなるという物理的解釈が成り立つ。以上のことから、本法で近似的に求めた固有振動数の計算精度は、仮定する変形モードが真の値にどれだけ近いかによって決まる。しかし、一般には幾何学的境界条件を満足さえしていれば、かなり粗い近似の変形モードを用いても、実用上十分な精度で固有振動数を計算できる点に特徴がある。はりの曲げ振動レイリー法を一般の弾性体に適用する場合、仮定する変形モードとして静荷重によるたわみ曲線が用いられることが多い。単純な形状については、材料力学で用いられている公式などを利用すればよいが、複雑なものについてはこれらを組合わせるなどの工夫が必要である。いずれの場合でも、幾何学的境界条件を満足するように選定することが重要である。いま、仮定するはりの変形モードをとし、円振動数ωで単振2―209動しているとすると、ポテンシャルエネルギーと運動エネルギーの最大値はそれぞれ次のようになる。一様なはりについては、式(22・1)から固有振動数は次式で計算できる。
