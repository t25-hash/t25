# 22・1・2 リッツ法(Ritz process)

> **章**: 第22章 振動・音響の計算法
> **節**: 22・1 近似解法

レイリー法では、固有振動数の計算値は真の値よりつねに大きな値となることから、仮定した変形モードにあるパラメータを導入し、固有振動数が最小となるようにこのパラメータを決めることから計算精度を向上させることができる。リッツ法では、この原理により、変形モードをいくつかの独立な関数の線形結合で表し、基本振動数の精度を向上させるとともに、高次の固有振動を系統的に求めることができる。ここでは、変断面はりの曲げ振動を例として取上げる。この係数行列式を零とおくことにより、ωに関する次方程式が得られ、これを解くことにより個の固有振動数が求まる。この方法は、変形モードの項数を増すことにより、精度をさらに高めることができる。なお、仮定する個々の関数(基底関数という)としては、変位および傾斜角の境界条件を満たす三角関数や多項式などが用いられる。
