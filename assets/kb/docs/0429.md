# 22・2・1 伝達マトリックス法の概要

> **章**: 第22章 振動・音響の計算法
> **節**: 22・2 伝達マトリックス法

伝達マトリックス法(transfer matrix method)は複雑な振動系を要素の自由度とほぼ同一の自由度で解析できる手法の一つで、系の固有振動数や固有モードの計算および定常応答の計算に用いられ2―210る。この手法を拡張して時刻歴応答や非線形系の解析に適用できる手法として、いくつかの手法がある。その一例として、増分伝達マトリックス法が提案されている(22・2・6項参照)。全体系をいくつかの基本要素に分割し、各構成要素の力学的特性を伝達マトリックスの形でマトリックス表示し、一定の手順を逐次実行して全体特性を求める手法で、電子計算機で数値計算を行うのに適している。一般には、系の種類に応じて大型電子計算機用のはん用プログラムの形で実用に供されており、特に、複雑な回転軸系や配管系の振動解析に多く用いられている。本手法の基本概念は、多自由度系の数値計算法として従来から知られているホルツァー法(Holzer’s method)やマイクルスタッド法(Myklestad’s method)と同一のものであるが、これをマトリックス表示により系統的かつはん用的に展開させたところに本手法の特徴がある。
