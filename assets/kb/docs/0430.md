# 22・2・2 状態量ベクトルと伝達マトリックス

> **章**: 第22章 振動・音響の計算法
> **節**: 22・2 伝達マトリックス法

伝達マトリックス法の基本は、まず系をいくつかの基本要素に分割することである。この基本要素としては、一般には要素の力学的特性がすでに解析的に明確になっているものを用いる。例えば、集中質量やばね、ダンパなどの集中付加要素および一様なはりや平板などである。いま、各要素の接合部、いい換えれば、全体系での分割点を便宜上節点と名付ける。各節点では、この点の物理的状態を表すいくつかの状態量が規定できるが、これらをすべて含めてベクトル表示したものを状態量ベクトルという。このような正方マトリックスを伝達マトリックスという。図22・1 連続系の節点と状態量ベクトル
