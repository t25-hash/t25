# 22・2・3 伝達マトリックス法による振動解析

> **章**: 第22章 振動・音響の計算法
> **節**: 22・2 伝達マトリックス法

a. 自由振動解析図22・1の系で、両端の状態量ベクトルが全体系の伝達マトリックスを介して式(22・8)で関係付けられることがわかったが、さらに式(22・8)を変位と力の関係に分解して考える。ここで、両端の境界条件に注目する。には系の慣性要素や減衰要素に対応して仮定した固有角振動数ωが未知の形で入っているので、上の方程式を解くことにより、これを得る。しかし、一般には自体の各要素が複雑な形になっており、式(22・10)を解析的に解くことは困難である。b. 強制振動解析任意の節点に振動数が一定の正弦強制変位あるいは正弦強制加振力が外部から作用する場合、各要素の力学的特性には影響がなく、単に状態量が変化したものとしてとらえることができる。つまり、強制振動問題を解く場合には、状態量ベクトルとして新たに′を式(22・11)のように定義し、各要素の伝達マトリックスを外部加振の振幅を含めて式(22・11)の′のような形に拡張することにより、先の自由振動の解析と全く同様に取扱えることがわかる。次に両端の境界条件から右端の未知の状態量を求める。
