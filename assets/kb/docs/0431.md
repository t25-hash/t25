# 3・1・1 外力と支持反力

> **章**: 第3章 はりの曲げ
> **節**: 3・1 はりのせん断力と曲げモーメント

自由物体の 方向の平衡条件より，完全に固定している（並進および回転運動がない）場合を考える．
せん断力および曲げモーメントが，はりの長さ方向に変化する状態いるので，壁からの力 や偶力 を支持反力（広義）と呼ぶ．
こa.単純支持（simple support) 支持点で自由物体の並進運動伝達するが，偶力 は伝達されない（例えば図3･2）．
び回転運動の両者を拘束する．
つまり支持点で および の両者例えば図3･1(ａ)のように，外力 が作用しているはりを壁に壁からはりを仮想的に切断し，自由物体図を描くと，図3･1(ｂ)のようになる．
このとき，もし壁からの支持がなければ，自由物体は加速度をもって時計まわりに回転しながら下に落下する．
壁はこのような運動を妨げ，自由物体を静止のまま（平衡状態）に支持してこで，支持の仕方として，次の二とおりがある．
のみを拘束し，回転運動は拘束しない．
つまり，支持点で力 はb.固定支持（fixed support) 支持点で自由物体の並進およが伝達される（例えば図3･3）．
