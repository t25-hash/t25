# 22・3・1 歴史

> **章**: 第22章 振動・音響の計算法
> **節**: 22・3 有限要素法

有限要素法(finite element method)は,1950年代初期に欧米航空宇宙工学の分野における構造力学への応用に成功して誕生した.特に1956年のターナー,クロー,マーチン,トップの共著論文が有名であり,この年を有限要素法誕生の年として解説する本も多い.構造力学での成功後,有限要素法は動力学などの応用力学への適用研究とともに,応用数学者も巻込んで理論的研究が大いに展開された.すなわち,有限要素法はレイリー・リッツ法の応用手法であることを明らかにした1963年のメロシュの理論的研究から,重み付き残差法に基づく定式化へ展開されていった.そして,アイソパラメトリック要素など適用性の高い要素の開発もなされた.その結果,有限要素法は,熱伝導,流体力学,電磁気学など物理学および工学のさまざまな問題の解析に利用される解析法に発展した.今日では有限要素法は対象を限定せずに広く微分方程式を数値近似解析するための手法と位置付けられる.以下の解説からもわかるとおり,理論が形式化されており,まさにコンピュータによる解析手法として適している.それゆえ欧米で開発されたいくつかのはん用プログラムは世界的に市販され広く利用されている.上記の有限要素法では,解析精度を上げるためには有限要素分割規模を大規模にする.換言すれば,より小さな有限要素を多量に組上げて解析対象物をモデル化する.そのために,モデルデータを作成するための負担と解析自由度の莫大化が実用の問題点である.前者の解決のために,いろいろと便利な機能の前処理プログラムの開発研究がなされた.例えば,有限要素モデルをグラフィック表示してモデル形状の妥当性を検討しやすくするものや自動的に多くの有限要素で分割して入力データのファイルを作成してくれるものである.後者の問題については,有限要素法のアルゴリズム自体を改良する研究もなされた.その一つがp法有限要素法である.これに対応して従来の方法はh法有限要素法と呼ばれて識別されるようになった.p法有限要素法の開発の歴史を概観すると次のようである.ザボとリーの論文“Stiffness matrix for plates by Galerkin’s Method”が1969年に発表されており,その中で有限要素の数を増やすか各要素自由度を増やすことで解析精度の向上を図ることができると述べられている.このことなどからp法は1960年代後期には始まっていたと考えられる.1975年にはペアノがp法に関する研究の博士論文を発表し,イタリアに戻って1978年にp法有限要素法プログラムFIESTA を開発し,ダムの解析に適用している.1984年には彼が学位を取得した大学のあるセントルイスにノエティックテクノロジー社が設立され,PROBEと名付けられたp法有限要素プログラムが開発された.それは後にMSC社のMSC/PROBEとなり,やがてMSC/NASTRAN に組込まれた.1980年代にはp法に関する多くの論文が発表され,1990年代には欧米製の市販ソフトも数多く出回り,日本にも広く知れわたって実用的に利用されるようになっている.
