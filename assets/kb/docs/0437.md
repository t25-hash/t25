# 22・3・2 エネルギー法による定式化

> **章**: 第22章 振動・音響の計算法
> **節**: 22・3 有限要素法

有限要素法ではあらかじめ利用できる何種類かの有限要素(ブロック)を多数適当に結合して、解析対象の構造物にできるだけ似た形状を構築する。さらに各有限要素について実際の構造材料定数と材料力学の基礎理論に基づいた有限要素法理論を定式化して、それらを対象構造物モデル全体にわたる有限要素間の結合状態に対応させて合成することにより、構造物全体の運動方程式を構成する。一般的に有限要素法では多くの有限要素を用いれば用いるほど解析精度は向上する。さて、一つの有限要素を取上げて、その特性表現の定式化をする。この有限要素はいくつかの節点から構成されているので、その要素内部任意の位置の変位をある適当に設定した関数(形状関数と呼ぶ)により構成節点の変位で表現できるものとする。式(22・24)の表現は一般的に近似である。さて、剛性行列を導出する。ここで、はひずみ-応力関係を表現するための弾性行列である。ここで、は要素領域を示す。さらに、式(22・28)の関係は任意の節点変位について成立するはずなので、式(22・29)が導かれる。ここで、ρは密度である。式(22・31)と式(22・35)により個々の有限要素の剛性行列と質量行列が作成できるので、構造物を構成するすべての有限要素についてそれらを求めて、節点の結合状態に整合させて構造物全体に関する剛性行列と質量行列を組上げればよい。なお、一般的に式(22・31)と式(22・35)の積分計算はガウス積分による数値積分演算で行われる。以上解説した理論は元来の有限要素法であり、現在ではh法有限要素法と呼ばれる。h法で解析精度を向上させるためには解析対象構造物をより小さな有限要素で構成する(要素分割規模を増大)のに対して、アダプティブp法有限要素法では、要素分割規模は変えずに要素内部の変位を表現する関数の次数を節点自由度で表現できる以上に増加させて定式化するものである。したがって、運動方程式の自由度として節点自由度以外に一般自由度と呼ばれる各有限要素ごとに独立の自由度が追加される。p法はh法に比べてそれほど大規模な要素分割をせずに解析できるので、入力データ作成の手間と時間の節減が図られるメリットを有している。一方で適応的に関数の次数を上げて計算を繰返して解の収束判定から結果を得るのは、かなり大規模な実用解析においては計算時間が多くかかることに留意する必要がある。しやすい一般定式化の方法であろう。しかし、その他の定式化の方法の一つとして、重み付き残差法の一種であるガラーキン法による2―214機械力学定式化もよく知られていることをここに記しておく。
