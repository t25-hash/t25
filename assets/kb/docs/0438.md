# 22・3・3 固有値解析(eigenvalue analysis)

> **章**: 第22章 振動・音響の計算法
> **節**: 22・3 節のエネルギー法による定式化は、有限要素法の最も理解

有限要素法により解析対象構造物を多くの有限要素でモデル化し、各有限要素に関して導出された要素剛性行列と要素質量行列を要素節点の結合に従って重ね合わせると、解析対象物に関する多自由度の剛性行列と質量行列を得ることができる。振動解析の基本は、対象物の動特性、すなわち固有振動数と固有モードを求めることである。これらは、式(22・37)の固有値問題から求めることができる。なお、その自由度をとする。この式は数学的には一般的固有値問題として取扱われる。固有値λとそれに対応する固有ベクトルは組得られる。実際の計算は、行列の自由度が一般的に大きいので、大自由度行列に適する固有値解析法である逆反復法やランチョス法またはサブスペース法などの数値計算手法を利用する。これにより実用的に必要な低次のいくつかの固有値および固有ベクトルのみを効率よく計算することができる。
