# 22・3・4 自由振動解析

> **章**: 第22章 振動・音響の計算法
> **節**: 22・3 節のエネルギー法による定式化は、有限要素法の最も理解

有限要素法では、対象構造物の質量行列と剛性行列の導出はできるが、減衰行列については合理的な減衰モデルの理論が確立されていない。そこで、便宜的な減衰行列作成手法が用いられる。その一つが剛性行列と質量行列から式(22・39)に示すように比例減衰(proportional viscous damping)行列を構成する方法である。=α+β(22・39)ここで、αとβはスカラ実係数でなんらかの便宜的根拠に基づいて適切に設定する。式(22・40)を式(22・41)のように変形する。なお、式(22・41)で行列とは一度計算しさえすればよい。また、実際には有限要素法で作成される特性行列(質量行列、剛性行列、減衰行列の総称)の自由度は非常に大きくなる場合がある。その場合には計算効率を上げるために22・4節のモード解析を利用して実用的に有効な低次のいくつかの固有モードのみを考慮した近似計算を行う。
