# 22・3・5 強制振動

> **章**: 第22章 振動・音響の計算法
> **節**: 22・3 節のエネルギー法による定式化は、有限要素法の最も理解

強制振動の運動方程式は、++=(22・42)である。この式を=--+=++(22・43)の形式にして、自由振動解析と同様に初期値と時々刻々の外力を与えることで数値積分を行う。周波数応答の計算は、基本的には式(22・42)を-ω+ω+ω=ω(22・44)と周波数領域に変換して、ω=-ω+ω+ω(22・45)で解けばよい。ただし、22・3・4項と同様の理由から、実用的には理論モード解析を利用して解くのが一般的である。
