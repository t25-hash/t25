# 22・4・1 理論モード解析

> **章**: 第22章 振動・音響の計算法
> **節**: 22・4 モード解析

a. 不減衰系不減衰系の固有値解析(22・3・3項)で得られる組の固有モードは互いに一次独立な関係にあるから、いかなる強制振動応答でもそれらの一次結合で表現できる。それらは物理空間座標上での応答変位を固有モードを一般化座標とした場合のモード座標(modal coordinate)上で表現するものであり、固有モード行列は物理空間座標とモード座標間の座標変換行列とみなすことができる。ここでは以下に周波数領域のみの運動方程式について取扱おう(時間領域についても全く同様に展開できる)。ここで、対角成分とはそれぞれモード剛性(modalstiffness)、モード質量(modal mass)と呼ばれる。そこで、式(22・48)を構成している個の方程式は、互いに非連成であるので逆行列計算(この計算は時間がかかる)が不要で、容易に解くことができる。結局、振動応答は式(22・50)のように計算できる。以上の式展開から理解できるように、理論モード解析では固有値解析を行う必要はあるものの、一般的解析条件(多くの周波数点や時間きざみ数の振動解析)においては、原始的に式(22・44)の連立方程式を繰返し解く場合より大幅に計算効率がよい。そのおもな理由は二点ある。第一は逆行列計算の繰返しが不必要である点、第二は解析周波数帯域からかなり外れた固有振動数に対する固有モードは省略しても近似計算としては実用的な精度が得られる点である。したがって、例えば有限要素法で作成された大自由度の剛性行列と質量行列による解析では、解析対象周波数帯域に対して若干の上下限周波数に余裕をもった帯域に存在する固有値と固有ベクトルのみを選択的に算出できる固有値解析手法を利用してモード解析(modal analysis)を行えばよい。加振点自身の周波数応答関数(自己応答)は、式(22・51)の右辺各項の分子が必ず非負値となるので、図22・4のボード線図に例示するように共振峰と反共振溝が交互に存在する。共振峰の周波数は図22・4 自己応答の例(FEM 計算)固有振動数にほかならないから、式(22・38)で求められる値であり、式(22・51)の右辺各項の分母が零となる角振動数ωに対応する。一方、反共振溝の周波数は伝達関数の振幅が零となる周波数であるから、式(22・51)の右辺を通分したときの分子が零となる周波数である。物理的解釈は加振自由度を拘束した場合の固有振動数である。ここで、分子は分母の行列の行列成分に対する余因子行列の行列式である。反共振溝の周波数はこの余因子行列式が零となる周波数であるから、まさに対象構造物の加振自由度を拘束した場合の固有振動数を意味している。そして、反共振溝付近の伝達関数の振幅レベルは低いので、式(22・51)からわかるように、たとえ解析対象周波数帯域よりかなり高い固有振動数の固有モードでも、それらを省略したことによる影響はかなりはっきり現れる。b. 減衰系。しかし、現実的にはλが含まれる式(22・54)では固有値解析が難しいので、次のような固有値問題の形式に変えて解析する方法がとられる。そして、式(22・57)からわかるように固有ベクトルの上半分の部分ベクトルが変位固有モードを、下半分はその時間微分であるから速度固有モードを表す。実数部は安定な減衰系の場合に値は負となり、σが減衰の大きさを表す。系が周辺自由境界条件下など剛体的に運動できる場合には剛体運動モード分の数の固有値は零として得られる。ここで、便宜的に右辺各項の分子は実部と虚部に分離して複素数で表現しておく。式(22・65)によれば、実数で与えられる減衰、共振角振動数、モード留数(分子のこと)の実部と虚部のモーダルパラメータと振動角周波数ωで伝達関数を表現できたことになり、22・4・2項で示す実験モード解析のカーブフィット(curve fit)に都合がよい。
