# 22・4・2 実験モード解析(experimental modal analysis)

> **章**: 第22章 振動・音響の計算法
> **節**: 22・4 モード解析

理論モード解析で不減衰系の周波数応答変位関数は式(22・51)、一般粘性減衰系の周波数応答変位関数は式(22・62)に示されている。決定されたモード特性は、固有角振動数や固有モードを物理的に表現するものである。ここで、式(22・68)と式(22・69)の右辺第1項は対象周波数帯域より低い周波数成分の影響を考慮した項(慣性項)、第3項は高い周波数帯域の成分の影響を近似考慮した項(剰余項)である。また第2項のは対象周波数帯域内の固有振動数の数を表す。曲線適合法として多くの手法が提案されている。それらは時間領域法、周波数領域法およびそれらのハイブリッド法に大別される。時間領域法は、周波数応答関数を逆FFT して得られる単位衝撃応答データを利用して時間領域の理論式に基づいてモード特性を同定する手法である。周波数領域法は周波数応答関数データを直接フィットさせるように周波数領域で同定解析する手法である。
