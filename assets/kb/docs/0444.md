# 22・4・4 部分構造合成法

> **章**: 第22章 振動・音響の計算法
> **節**: 22・4 モード解析

一般に機械構造物/システムは複数の部品の結合で構成されている。そこで、機械の動特性は、それらを構成するそれぞれの部品の動特性および部品間の結合特性によって作り上げられているはずである。したがって、各部品ごとの動特性および結合部の結合特性をなんらかの便利な形式でモデル化表現してデータベース化しておけば、それらを適宜引用することによって設計段階において機械の動特性予測ができるとともに、問題視される動特性に対する各部品の寄与度合を把握して有効な改善対策方針をすみやかに得られることが期待できる。この概念に基づく解析技術は、コンピュータを利用した現代および将来の設計工学の一つの重要な体系的技術である。このように解析対象物の解析を、実体的構成に基づいて、または解析上仮想的に、いくつかの部分構造に分解して、まずそれぞれの動特性を得て、それからの動特性の合成を行うことにより効率的に解析する手法を部分構造合成法(substructure synthesismethod)と呼ぶ。部分構造合成法を歴史的に眺めると、誕生当時に利用可能なコンピュータの能力に比して大規模すぎる機械構造物の解析を実現するために、解析の都合に合わせて仮想的に対象物をいくつかの部分構造に分解して解析しようとした試みがきっかけとなっている。静解析に関する部分構造合成法は先進的にかなり古くから開発されているが、動解析に関しては1950年代のヒュームやマクニールの提案論文から始まり、1960年代のハーティやクレイグとバンプトンの提案が初期の成果であろうと思われる。そして、現在までにその派生的な手法が提案されてきている。a. 分類現在までに部分構造合成法として提案された手法はかなり多くあるが、おもだったものを大別すると図22・5のように整理できるであろう。すなわち、動解析に関しては、各部分構造の伝達関数を陽な形で合成していく伝達関数合成法(transfer func-tion synthesis method)、各部分構造単体に関する固有モードを一般化座標として用いて合成していくモード合成法(mode synthesismethod)、各部分構造単体に関する自由度の小さな特性行列(質量行列、減衰行列および剛性行列の総称)を合成していく特性行列合成法(spatial matrix synthesis method)、そして最後にそれらの混合型である。なお、有限要素法に基づくある構造解析システムプログラムにお図22・5 部分構造合成法の分類いては、モード型と特性行列型の部分構造合成法を概念に分類せずに統一的にスーパーエレメント法と名付けた手法で取扱えるようにしている。すなわち、モード座標上の一般化自由度と物理座標上の自由度を自由に選択でき、それにより作成した各部分構造の動特性はスーパーエレメント(super-element)と名付けた特性行列の形式に縮小化され、それを組上げることにより全体系の運動方程式を構築して解析する。伝達関数合成法は、基本的には外力の作用する自由度と応答評価のための自由度と部分構造の結合自由度のみの伝達関数を陽な形式で合成することによって、結合点に生じる内力を消去して応答を解析する手法であり、まさに原理的な手法である。その際に通常、振動実験では伝達関数が計測されるので、実験ベースとして便利な部分構造合成法として提案された。そのため、結合部分などの非線形性を組込むことも容易にできる長所を有している。他方、直接固有振動数を出力できない点や、ある程度の広い周波数帯域の応答では他の部分構造合成法より計算時間がかかってしまうなどの欠点を有している。モード合成法は、各部分構造単体について、隣接する部分構造との結合自由度を自由端もしくは拘束自由度とした境界条件で得られる低次側からのいくつかの固有モード(採用モードと呼ぶ)を一般化座標として、その部分構造の変形(物理座標上での変位ベクトル)を表現するように定式化して解析する手法である。そのために、解析対象全系に関する運動方程式の自由度は各部分構造の採用モード数の合計となり、まともに物理座標上で一体解析としての運動方程式を構成するより大幅に縮小でき、かつ、質量行列、減衰行列、剛性行列がモード座標上での特性行列で表現されるので、固有値解析により直接共振周波数が得られる。周波数応答解析はモード解析により効率よく実行できる長所を有する。他方、採用モード数の不適切な設定によっては剰余成分の影響の補償が不十分となり解析精度が大幅に悪化してしまう点や、各部分構造についてそれが有する結合自由度より少ない採用モード数の設定(実験ベースの場合に多々発生する事態であると思われる)であると全体系に関して合成される特性行列が特異となってしまい計算不可能な事態を生じかねないので解析上慎重な判断を要するという点などの短所がある。特性行列合成法は、各部分構造単体について、まず隣接する部分2―217構造間の結合自由度を自由端とする境界条件のもとで、なんらかの手段によって結合自由度と外力作用自由度および応答評価点自由度のみに関する自由度の比較的小さな特性行列を準備し、次にそれらを結合自由度に関して重ね合わせることによって、結合部で発生している内力(作用・反作用力)を方程式から消去して全体系の運動方程式を構成して解析する手法である。各部分構造の特性行列を準備するための手法に関して、理論的アプローチとしてまずグヤンの静縮小法、動的縮小法、さらにモード解析の基礎理論を用いて正方行列になるように自由度調整した固有モード行列と固有振動数から特性行列を算出する手法などが挙げられる。実験的なアプローチとしては実験的特性行列同定法による方法が挙げられる。特性行列合成法は、概念的には最もすっきりとした手法であり、理論的アプローチのデータと実験的なアプローチによるデータをまったく区別なく扱うことができ、CAEとしての解析手法として適していると思われる。上記3種類以外には、二者を混合利用した手法や、羽根車など周期的対称構造物の解析を単位周期分の部分構造についての解析データを周期対称条件で合成して解析する周期対称法(cyclic symme-try method)、部分構造間の結合自由度を1自由度ずつ次々に結合していく状態の解析を進めることで、最終的に全結合点を結合して対象物全体の解析を行うダイヤコップスティックス法などがある。なお、伝達関数型をはじめ、どの種類の部分構造合成法でも合成の多重化が可能である。すなわち、例えば二重化の合成を行う場合には、解析対象物をいくつかの部分構造(ここでは第一次分系と呼ぶことにする)にまず分割するが、もしある第一次分系がまだ解析には大規模すぎると判断される場合には、それをさらに小さないくつかの部分構造(ここでは第二次分系と呼ぶことにする)に分割して、第二次分系の合成計算によりその第一次分系の特性を求め、それをほかの第一次分系と合成することにより対象物全系の解析を行う。b.
