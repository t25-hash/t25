# 22・5・1 ・a項の問題を、重み付き残

> **章**: 第22章 振動・音響の計算法
> **節**: 22・5 境界要素法

差法(weighted residual method)で解くことを考える。ここでは重み関数である。これがいまの問題の重み付き残差の条件式である。に適当な関数を与えたとき、上式を満たすΦが、求める近似解である。ここでこの式は、境界条件として与えられている値については、その値を代入したものを意味する。ここで例として取上げた以外の別の境界条件の問題でも、代入する値が異なるだけで、重み付き残差の条件式は式(22・88)と同じになる。以下式(22・88)を問題の重み付き残差の条件式とする。式(22・86)のような問題において、左辺にデルタ関数δ,ξを加えた方程式の解を基本解(fundamental solu-tion)という。式(22・90)は、領域の内部の点=ξにおけるΦの値が、境界におけるΦ,の値を用いて得られることを示している。未知関数Φを求めるため、式(22・90)においてξ→0およびξ→とすると、境界におけるΦ,の間の二つの関係式を得る。この式に、境界条件として与えられているΦまたはの値を代入すると、境界条件で与えられていないΦまたはの境界上の値が求められる。一般に22・5節のような問題の場合、境界上でΦまたはのどちらかが与えられるので、以上の扱いによって、与えられた問題の境界上のΦ,がすべて求められる。これを式(22・90)に代入すれば、点=ξにおけるΦの値を求めることができる。ξは任意にとりうるので、このようにして任意の点=ξにおけるΦの値を知ることができ、未知関数Φが定められたことになる。上述のように、もとの問題を、境界における関数値の関係に直して扱うという考え方が境界要素法の基本である。ただ一次元の問題では、境界は両端の2点となり、境界を要素に分割するという作業を伴わないので、この解法を境界法(boundary method)という。
