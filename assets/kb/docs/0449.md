# 22・5・3 三次元の音響問題の境界解法

> **章**: 第22章 振動・音響の計算法
> **節**: 22・5 境界要素法

時間に関してのように変化する音圧あるいは粒子速度が境界上で与えられる三次元空間の音響問題を考える。この問題の重み付き残差の条件式は、式(22・94)で左辺の積分を領域V の積分、右辺の積分を領域S の積分に置換えたものとなる。これ以降の取扱いは二次元の音響問題と同じである。これが成り立つとき、無限遠方の境界に関する積分が零となり、要素分割を音源の境界だけについて行えばよいことが導かれる。ここでは振動・音響の解析によく用いられる固有値解析法と直接数値積分法を述べる。
