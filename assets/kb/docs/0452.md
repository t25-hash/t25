# 23・1・1 振動と騒音の計測の目的と用途

> **章**: 第23章 計測と信号処理
> **節**: 23・1 振動と騒音の計測

振動や騒音を計測する目的をまとめたのが図23・1である。その一つは製品の品質を評価することであり、ルールに基づいた計測器の仕様や計測方法が必要である。もう一つは振動の現象や原因を究明するための調査である。また最近では両者の中間的な位置付けとして、振動や騒音の監視や異常診断が発達してきた。これらの調査や監視ではルールとは関係なしに可能な限り詳細な計測を実施する必要があり、適切なセンサの選択などが不可欠である。いずれにせよ振動や騒音の計測方法や評価方法についてはISO(Interna-tional Standard Organization)やJIS(Japan Industrial Stan-dard)等で多くの規格が整備されているのでこれを参照する必要がある。
