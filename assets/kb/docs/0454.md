# 23・1・3 計測の具体例

> **章**: 第23章 計測と信号処理
> **節**: 23・1 振動と騒音の計測

振動,騒音の計測の具体例として,回転機械の例を図23・10に示す.この例のように,回転機械の計測では回転の基準を示す回転パルス(キーフェザーシグナル)が重要であり,データ処理に大いに役立つ.また実際の機械の計測では,振動,騒音だけでなく流体の圧力脈動や各部の温度など多くのデータの同時計測が必要であるこ図23・9エレクトレットマイクロホンの構造図23・10 回転機械振動計測例2―226機械力学とが多い.この種の計測には時間的に連続な記録が必要であり,表23・2に示す記録方法が用いられる.以前多く用いられていたアナログテープレコーダからディジタルテープレコーダへと変化し,さらに最近では大容量ディスクの発展により,A-D 変換器とパーソナルコンピュータを用いる方法へと変化しつつあり,各種システムやソフトが市販されている.
