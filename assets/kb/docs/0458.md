# 23・2・1 連続および離散フーリエ変換

> **章**: 第23章 計測と信号処理
> **節**: 23・2 データ処理

連続フーリエ変換(continuous Fourier transform)は,無限大の時間軸の関数を無限大の周波数軸の関数に,またはその逆に変換する方法である.現実の世界の信号を対象とする場合は,積分は数値的(離散的)に行われ,時間軸および周波数軸も有限でなければならない.連続フーリエ変換に対応する離散的な変換は離散フーリエ変換(DFT : discrete Fourier transform)と呼ばれる.離散フーリエ変換をより少ない演算量で実行するアルゴリズムが1965年にクーリーとチューキーにより高速フーリエ変換(FFT : fastFourier transform)として考案された.ここでは,FFT のアルゴリズムの詳細の説明は省略し,それを正しく有効に利用するために知っておくべき,離散フーリエ変換の基本的な性質について述べる.a. 折返しひずみ(エリアジング)サンプリングする際に考慮すべき重要な定理がシャノンの標本化(サンプリング)定理(sampling theorem)である.サンプリングにより信号を正しく(ゆがませずに)取込むには,連続信号に含まれている最大周波数の2倍以上のサンプリング周波数を用いることが必要である.この条件を満たさない場合には,サンプリングの半分の周波数を超える周波数成分がそれ以下の帯域に折返すように含まれるエリアジング(aliasing)と呼ばれる現象が発生し,正しい分析が行えなくなる.エリアジングによって発生する余分な成分は折返しひずみと呼ばれる.そのため,アナログ信号をサンプリングする際には必ず標本化定理を満たすようにすることが必要である.図23・18は多くの高調波を含む矩形波をサンプリングした例である.エリアジングを防止する低域通過フィルタを用いることによって,矩形波の正しい周波図23・16
