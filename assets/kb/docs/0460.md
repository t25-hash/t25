# 23・2・1 項では時間軸信号を周波数軸の成分(スペクトル)に変

> **章**: 第23章 計測と信号処理
> **節**: 23・2 データ処理

図23・18 エリアジングフィルタをON(下)およびOFF(上)としたときの矩形波のパワースペクトル図23・17ハニング窓をかけたときの図23・16に対応する結果2―230機械力学換する方法について述べた。これらは基本の物理量(温度、変位、電圧など)で表される現象を、時間軸の信号として表すか周波数スペクトルとして表すかの違いである。さらに、二次の物理量として、相関(correlation)やパワースペクトル(power spectrum)がある。ここではそれらについて述べることとする。二つの信号およびの相互相関関数(cross correlationfunction)Φτは一般的に次式で与えられる。Φτ=+τ(23・18)ここで、は期待値を表し、τは時間遅れである。もし、およびが定常ランダムプロセスであれば、相互相関関数はに依存せず、期待値は時間平均で置換えられる。욪+τ(23・19)Φτ=lim次に、クロスパワースペクトル(cross power spectrum)を次のように定義する。=lim(23・20)このとき、Φτ=욪욭(23・21)=욪Φτ욭τ(23・22)の関係がある。すなわち、相互相関関数とクロススペクトルは互いにフーリエ変換の関係にある。とが等しい場合は自己相関関数(autocorrelation function)とオートパワースペクトル(auto power spectrum)について同じ関係が成り立つ。式(23・20)で与えられるクロスパワースペクトルを実際に計算するためには連続信号を離散化する必要があり、下式によって求める。욇욇(23・23)=1∑욖は平均回数である。相関関数は次式で与えられる。욪욗요(23・24)Φ=1∑요変数として速度と力、電圧と電流などを用いてクロスパワースペクトルを求めると、周波数ごとに複素パワー(有効パワーはその実数部)が求められる。速度の代わりに加速度または変位を用いて積分または微分することによりパワーを求めることも可能である。離散フーリエ解析により、有効および無効パワーの周波数成分を詳細に調べることができるので、機器の診断等に極めて有効である。
