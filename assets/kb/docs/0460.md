# 23・2・1 項では時間軸信号を周波数軸の成分(スペクトル)に変

> **章**: 第23章 計測と信号処理
> **節**: 23・2 データ処理

図23・18 エリアジングフィルタをON(下)およびOFF(上)としたときの矩形波のパワースペクトル図23・17ハニング窓をかけたときの図23・16に対応する結果2―230機械力学換する方法について述べた。これらは基本の物理量(温度、変位、電圧など)で表される現象を、時間軸の信号として表すか周波数スペクトルとして表すかの違いである。さらに、二次の物理量として、相関(correlation)やパワースペクトル(power spectrum)がある。ここではそれらについて述べることとする。すなわち、相互相関関数とクロススペクトルは互いにフーリエ変換の関係にある。式(23・20)で与えられるクロスパワースペクトルを実際に計算するためには連続信号を離散化する必要があり、下式によって求める。相関関数は次式で与えられる。速度の代わりに加速度または変位を用いて積分または微分することによりパワーを求めることも可能である。離散フーリエ解析により、有効および無効パワーの周波数成分を詳細に調べることができるので、機器の診断等に極めて有効である。
