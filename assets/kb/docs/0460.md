g.定性推論 定性推論（qualitative reasoning）とは，
物理系などの本来連続的である動的システムの挙動を，運動方程式
などの定量的・数値的なモデルを用いず，より抽象的・定性的な局
面に基づいて記述されたモデルを用いて推論しようとする方法論で
ある．
つまり定性推論は，ある対象物理システムに関する問題に対して
詳細かつ完全な事前知識が与えられず数値的に厳密な解が求められ
なくても，不完全で粗い知識から定性的には正しいような近似解を
得ることを目的とする．
具体的な定性推論技術の方法論は，おもに対象世界に対する定性
的な表現をいかにして得るかという問題と，それによって得られた
定性的表現を用いていかに推論を行うかという二つの問題に分けら
れる．
前者に関しては，量，時間，空間，関係など本来は連続的な実体
をいかに量子化して表現するかということが問題になる．例えば，
本来は実数値である温度や質量などの物理量を特定の境界標
（landmark）を境にして有限個の区間に分割したり，本来数学的
な微分方程式によって表現されるような物理量間の関係を，一方が
増加すると他方は減少するというような定性的な比例関係に抽象化
して表現したりする．
一方，後者の定性的表現を用いた推論に関しては，システムに発
生したある事象を他の事象によって説明付けようとする因果解析
（causal reasoning）や，システムの可能なふるまいを求める挙動
推定（envisioning）とそれを用いた定性的シミュレーション
（qualitative simulation），系の一部のパラメータ値を変えた場合
にどのような変化が発生するかを推定する比較解析（comparative
analysis ）などの技術が開発されている．
定性推論の技術は，電子回路から宇宙機システムまで，さまざま
な工学分野において故障診断，製品設計，挙動シミュレーションな
どの用途に応用されている．
