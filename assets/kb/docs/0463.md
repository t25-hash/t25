# 1・1・2 分布様式による分類

> **章**: 第1章 基礎
> **節**: 1・2 のように分類される.繰返し荷重(repeated load)のうち,

荷重の分布様式に応じて,図1・3のように分類される.固体の外部表面全体に一様な圧力が作用する場合にいう静水圧(hydros-tatic pressure)も一様分布荷重(uniformly distributed load)の例である.荷重の分布の領域が表面の広がりに比べて局所的であれば,その分布荷重は近似的に1点に作用するとみなし,集中荷重(concentrated load)と考えることができる.分布荷重がその点に及ぼす効果を合力と合モーメントで置換えて力学的考察をすることがよく行われる.この背景をなすものに,次の原理がある.すなわち,荷重の分布状態に差があっても,合力および合モーメントが等しい場合には荷重領域から十分に離れた断面に生じる変位および応力は等しく,これをサンブナンの原理(Saint Venant’s principle)という.
