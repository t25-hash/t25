# 1・1・3 作用による分類

> **章**: 第1章 基礎
> **節**: 1・2 のように分類される。繰返し荷重(repeated load)のうち、

荷重が固体表面に作用する際、その作用方向によって垂直荷重(normal load)とせん断荷重(tangential load)に分類される。その定義を表1・3に示す。流体力の固体に及ぼす影響を考えるとき、非粘性流体の場合には垂直荷重のみを考えればよいが、粘性流体の場合にはせん断荷重も考慮しなければならない。また、固体に作用する荷重の種類に応じて図1・3のように分類される。作用する荷重が分布荷重の場合でも、これを合力や合モーメントに置換えたうえで、十分に離れた位置での変形や応力の評価を行うことができる(サンブナンの原理)。軸荷重(axial load)、横荷重(lateralload)、曲げモーメント(bending moment)、ねじりモーメント(torsional moment)などが複合して作用する場合にはこれらを組合せ荷重(combined load)という。一般に、固体の表面は、荷重が与えられた表面と変位が拘束された表面からなると考えることができる。変位を拘束する条件を支持条件という。例えば、はりに対する支持条件の例を通常用いられる(a) 繰返し荷重(b) ランダム荷重(c) 衝撃荷重図1・2 動荷重の分類(a) 一様分布荷重(b) 不等分布荷重(c) 集中荷重(d) 集中モーメント図1・3 分布荷重の分類表1・3 作用方向による分類荷重分類定義垂直荷重固体の表面に垂直に作用する荷重せん断荷重固体の表面に沿って作用する荷重3―2記号とともに表1・4に示す。荷重が作用していない表面は、荷重が零として与えられるとみなし、このような表面を自由表面(freesurface)という。自由表面も支持条件の一種とみなし、自由支持ということがある。構造物が支持物体から受ける反力を支持反力という。固体と固体が互いに押付けられて接触している場合には、接触面には垂直荷重とすべりによるせん断荷重が分布して作用し、これらを支持反力の一種とみなすこともできる(7・3節参照)。
