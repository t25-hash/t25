# 1・3・1 内力と応力

> **章**: 第1章 基礎
> **節**: 1・3 応力の定義

図1・4のように物体の表面AB に表面集中力が作用して釣合い状態にあるとき、物体内の仮想断面Cを考える。物体AB を仮想断面Cで切断すると、物体CB はやはり釣合い状態にあり、仮想断面Cには外力に釣合う力′が作用し、これを内力という。このとき、仮想断面における単位面積当たりの内力を応力(stress)という。上記のように、応力は外力が作用する結果として生じるが、複数の部材を組立てる際に、寸法が適正でない場合や、部材の一部が塑性変形した場合には、外力が作用しなくても応力が存在し、これを残留応力(residual stress)という。また温度変化によって生じる変位が拘束される場合には熱応力(thermal stress)という。
