# 3・4・3 動画像処理

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・4 視覚情報処理

3･4･2項で述べた形状認識では基本的に1枚の入力画像ごとに処
理が行われる．これに対して，対象物体が動いている場合にその動
きのパターンを解析したり，静止している環境内を移動する視覚セ
ンサで撮影した際の外界の見え方の変化からセンサ自体の動きを解
析したりするには，時間的に連続して撮影された画像列を取扱う必
要がある．このような画像処理は動画像処理と呼ばれ，これまでに
さまざまな処理手法が開発されている．
1枚の静止画像を処理する手法と比べて，動画像処理には固有の
難しさが存在する．例えば，通常のビデオ映像は毎秒30フレーム
の画像で構成されているため，数分間の映像でも静止画と比較して
膨大なデータ量となることが容易に理解できる．画像処理システム
では実時間処理が要求されることが多いが，このように膨大なデー
タ量となる動画像を実時間で処理するためには，できる限り効率の
良い処理アルゴリズムを開発することが重要となる．また，動画像
処理では，対象物体の移動や視覚センサ自体の運動により同一の物
体やシーンであってもその見え方が変化してしまう．そのため，見
掛けの形状が異なっていても，同一の物体と認識し，また時間的に
連続する画像間で各部の対応関係を決めるためのアルゴリズムが必
要となる．
画像処理システムの目的に応じて，さまざまな動画像処理が用い
られるが，ここでは基本的な処理にかぎり，移動領域の検出，オプ
ティカルフロー（optical flow）の計算，オプティカルフローに基
づくセンサ運動の推定，のそれぞれについて述べることにする．
a.移動領域の検出 シーン内を移動する物体を撮影した映像
に対する動画像処理では，画像中で観察される移動領域とその他の
領域とを区別することが必要となる．このための処理として画像間
の差分に基づく手法がよく用いられる．ここで，移動物体に対応す
る領域とその他の背景領域を考えた場合，もし背景領域の見え方に
変化が少ないと仮定できる場合には，あらかじめ物体が存在しない
状態でシーンの画像（背景画像）を撮影しておき，その背景画像と
新たな入力画像との差分を考えることにより，移動する物体に対応
する領域を特定することが可能となる．このような考え方に基づく
手法は背景差分（background subtraction）と呼ばれ，画像処理
システムで頻繁に用いられる基礎的な処理である．しかしながら，
屋外などのように照明条件が常に変化してしまう状況や，風になび
く木など小さな動きのある物体がシーンに存在する状況では，単純
に背景画像と入力画像との差分を求めるだけでは移動物体領域を安
定に抽出することができない．そのため，背景画像が一定であると
して取扱うのではなく，その明るさの時間的な変化の度合いなどを
考慮した背景モデルへと拡張するさまざまな工夫が提案されてい
る．
背景差分以外にも，差分による移動領域の検出として，時間的に
連続する2枚の画像間における差分を考慮した手法が用いられる．
このような処理は一般に時間差分と呼ばれ，背景差分の場合と異な
り，ほぼ同時刻の画像を2枚用いるために背景における照明変動な
どの影響を受けにくいという特徴がある．一方で，移動領域の色や
明るさが一様な場合などは，時間差分で検出される領域は実際の物
体の一部のみとなる．そのため，物体の全体を抽出するためには，
時間差分で検出された変化領域を手がかりに周辺の領域をさらに解
析しなければならない．
