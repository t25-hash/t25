# 1・3・2 垂直応力とせん断応力

> **章**: 第1章 基礎
> **節**: 1・3 応力の定義

仮想断面内の微小断面積Δに作用する内力の法線成分がΔであるとき、次式で定義される応力を垂直応力(normal stress)という。また、内力の接線成分がΔであるとき、次式で定義される応力をせん断応力(shear stress)という。垂直応力はσ,σのように表し、添字は垂直応力の作用方向を意味する。これをせん断応力の共役性(conjugation)または対称性(symmetricity)という。表1・4 はりに対する支持条件支持条件記号特徴単純支持垂直方向変位拘束。水平方向の移動は自由。回転は自由。ピン支持垂直、水平方向変位拘束。回転は自由。固定支持垂直、水平方向変位拘束。回転も拘束。図1・4 外力と内力図1・5 平面応力
