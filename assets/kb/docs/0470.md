# 1・3・6 座標変換と主応力

> **章**: 第1章 基礎
> **節**: 1・3 応力の定義

二つの主軸は直交する。一方、せん断応力は次式を満足する角度で極値をとる。このときのせん断応力を主せん断応力(principal shearstress)といい、次式で与えられる。このときの垂直応力を主応力といい、三軸応力では三つの主応力σ>σ>σがある。主応力の作用面が主応力面、主応力の方向が主応力軸であり、主応力軸は直交する。三つの主応力は次式を満足する根となる。σ-σ+σ+σσ+σσ+σσ+σσσ-σσσ=0 (1・15)上二式の係数は等しく、次式が成立する。また、ある面で合せん断応力は次のような極値をとり、これを主せん断応力という。
