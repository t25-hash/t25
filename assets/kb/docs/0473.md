# 1・3・6 項で述べた応力の座標変換および

> **章**: 第1章 基礎
> **節**: 1・3 応力の定義

主応力は、図式的に求めることができる。これをモールの応力円(Mohr’s stress circle)という。この円とσ軸との交点をA,B とすると、σの最大値および最小値はOA およびOB となり、これらはそれぞれ主応力σ,σに対応する。次いで、CA から反時計方向に2θの角度をなし、円の中心点Cを通る線CD を引くと、点D の座標は、σ方向から反時計方向にθだけ傾いた方向の垂直応力σと、その作用面に生じるせん断応力τを表す。また、このとき点D′の座標は、軸方向の垂直応力σと、その作用面に生じるせん断応力τを表し、その角度はCA に対して2θ+πとなる。また、σ,σおよびτが与えられた場合の主応力とその作用方向を求める場合には次のように考える。まず、σ=OE,σ=OE′となるようにσ軸上に点Eおよび点E′をとり、その中点を点Cとする。点Cを中心として、半径CD の円を描くとモールの応力円になる。σ軸と円の交点A およびB は主応力を表し、OA=σ,OB=σとなる。このとき、∠DCA を2θとすると、σは軸からθの角度をなす方向に作用することとなる。b. 三軸応力の場合応力分布が三次元的であるときのモールの応力円は次のようになる。三次元における任意の面の法線と、,軸からの角度がそれぞれα,β,γである面に作用する垂直応力σと合せん断応力τを考える。いま主応力σ,σ,σが与えられ、その作用方向が,軸に等しいとき、σとτは主応力を用いて次式のように関係する。いまσ>σ>σとして、縦軸にτ、横軸にσをとり、図1・8のように主応力をOA=σ,OB=σ,OC=σとする。上式の第1式は、点Oを中心とする破線の円となる。ただし、図1・8では下側半分は省略した。この円は、αを一定としてβおよびγを変化させたときのσとτの関係を表している。同様に第2式はOを中心とする破線の円となり、βを一定としたときのσとτを与える。また第3式は、Oを中心とする破線の円で、γ=90のとき最小半径σ-σ2のOを中心とする実線の円IIIとなる。円I,II,IIIはそれぞれσ,σ,σを含む面のσとτの関係を表し、任意の面の応力は、円IとIIIの外側で、かつ円IIの内側の領域に存在する。なお、三つの破線の円は一点Pで交わり、この点の座標が上三式の連立方程式の解を与え、この面に作用する垂直応力σと合せん断応力τを与える。次に、作用面の法線方向を表す角度α,β,γとモールの応力円の関係について考える。次いで、点A,B,Cを通り、τ軸に平行な線を引く。点A では、τ軸と平行な線から反時計方向にαの角度をなし、点Aを通る直線を引き、円IIとの交点をaとする。次いで、点aを通り点Oを中心とする斜線の円を描く。また、同様に点B,Cにおいても、τ軸に平行な線から時計方向にβ,γの角度をなす線を引き、円IIIおよびIとの交点をそれぞれb,cとする。点b,cを通り、中心が点O,Oの斜線の円を描くと、三つの斜線の円は一点Pで交わり、その交点が作用面の法線方向に平行な垂直応力σと作用面における合せん断応力τを表す。一般に物体内での応力は場所によって変化する。任意の点における応力は、その点を含む微小要素の周辺に生じている応力成分によって代表することができる。微小要素周辺に作用する応力成分と要素内に作用する物体力は釣合わなければならない。同様に三次元での釣合い方程式は、6個の応力成分と3個の物体力成分を用いて次式になる。
