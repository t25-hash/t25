# 1・5・2 垂直ひずみとせん断ひずみ

> **章**: 第1章 基礎
> **節**: 1・5 ひずみの定義

図1・10(a)のように、棒が引張りの軸力を受けて長さがから+λに変化するとき、棒の単位長さ当たりの伸びε=λを垂直ひずみ(normal strain)という。図1・10(a)のような引張りの場合、伸びλとひずみεはともに正であるが、圧縮を受けた場合は負となる。垂直ひずみが正ならば引張ひずみ(tensile strain)、負ならば圧縮ひずみ(compressive strain)という。また、軸力に対して垂直方向の寸法がから+Δに変化するとき[図1・10(a)において伸びΔは負]、この方向の単位長さ当たりの伸びε′=Δも垂直ひずみである。特に単軸引張りの場合、εを縦ひずみ(longitudinal strain)、ε′を横ひずみ(lateral strain)という。図1・10(b)のように、下辺を固定した長方形板にせん断を与えると、上辺が相対的にλだけ移動する。このとき生じる角度変化γをせん断ひずみ(shear strain)という。微小変形の場合はγ=tanγ=λとなる。
