# 1・5・2 垂直ひずみとせん断ひずみ

> **章**: 第1章 基礎
> **節**: 1・5 ひずみの定義

図1・10(a)のような引張りの場合、伸びλとひずみεはともに正であるが、圧縮を受けた場合は負となる。垂直ひずみが正ならば引張ひずみ(tensile strain)、負ならば圧縮ひずみ(compressive strain)という。特に単軸引張りの場合、εを縦ひずみ(longitudinal strain)、ε′を横ひずみ(lateral strain)という。図1・10(b)のように、下辺を固定した長方形板にせん断を与えると、上辺が相対的にλだけ移動する。このとき生じる角度変化γをせん断ひずみ(shear strain)という。
