# 1・5・4 平面ひずみ

> **章**: 第1章 基礎
> **節**: 1・5 ひずみの定義

図1・11のように、物体中の-平面上にある微小な長方形ABCD が、二次元変形によってA′B′C′D′となった場合を考える。AA′のような、変形前の点から変形後の点までのベクトルを変位(displacement)といい、その方向成分を,方向成分をとする。変形が微小な場合、点A の方向の垂直ひずみεと方向の垂直ひずみεはそれぞれ次のように表される。式(1・25)と式(1・26)をまとめて、二次元変形におけるひずみ-変位関係式(strain-displacement relations)という。このように、ある方向のひずみがすべて零となるとき、物体が平面ひずみ(planestrain)の状態にあるという。
