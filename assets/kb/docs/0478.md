# 1・5・5 三軸ひずみ

> **章**: 第1章 基礎
> **節**: 1・5 ひずみの定義

三次元の変形の場合も二次元変形(1・5・4項参照)と同様に考え,ひずみ-変位関係式を得ることができる.変形が微小な場合,,,方向の変位をそれぞれ,,とすれば,ひずみはε=옿옿ε=옿옿ε욨=옿옿γ=옿옿+옿옿(1・27)γ욨=옿옿+옿옿γ욨=옿옿+옿옿となる.ここでε,ε,ε욨はそれぞれ,,軸方向の垂直ひずみ,γ,γ욨,γ욨はそれぞれ-面,-面,-面のせん断ひずみであり,これらをまとめてひずみ成分(strain component)という.また,,,軸まわりの変形を伴わない回転の角度はω=1옿옿-옿옿ω=1옿옿-옿옿ω욨=1옿옿-옿옿(1・28)で表される.これらを回転(rotation)という.物体中に,,,軸方向の変形前の辺長がそれぞれd,d,dの微小直方体要素を考えたとき,微小変形後の各方向の辺長はそれぞれ1+εd,1+εd,1+ε욨dとなる.1・5・3項で述べた体積ひずみε욋はε욋=1+ε1+ε1+ε욨ddd-ddddddε+ε+ε욨(1・29)となる.微小変形の場合,せん断ひずみは体積変化を生じない.なお,1・5・2項から1・5・5項の式(1・27)までで述べたひずみは工学ひずみ(engineering strain)という.これに対してε=εε=εε욨욨=ε욨(1・30)ε=ε=γ2ε욨=ε욨=γ욨2ε욨=ε욨=γ욨2で定義されるε욡욢をひずみテンソル(strain tensor)という.
