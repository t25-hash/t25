# 1・9・2 二次元弾性問題

> **章**: 第1章 基礎
> **節**: 1・9 弾性問題

弾性体内の応力がつねに平面に平行(考えている平面に垂直な方向の応力成分は零)であるような状態(例:面内荷重を受ける薄い平板)を平面応力問題(plane stress problem)、剛体変位を除く変位ベクトルがつねに平面に平行(平面に垂直方向のひずみ成分は零)であるような状態(例:長手方向に変形を拘束された一様断面の柱体で、長手方向に一様分布の横方向荷重を受ける柱体断面)を平面ひずみ問題(plane strain problem)として扱う。このほか、有限厚さの平板が中央面に関して対称な面内荷重を受ける場合、中央面に垂直方向の応力成分を零とみなし、かつ面内応力については厚さ方向の平均を考えて平面応力状態として扱える。これを一般化平面応力(generalized plane stress)問題という。平面内で-座標系を取り、平面応力問題と平面ひずみ問題の方程式系を表1・6に示す。A 欄(釣合い式)、B 欄(ひずみ-変位関係式)は両者に共通である。それぞれの問題における仮定(C欄)を式(1・51)の応力-ひずみ関係式に代入すればD 欄の関係式が得られる。釣合い式は、エアリーの応力関数(Airy’s stress func-tion),を導入して応力をE欄のようにおくことにより恒等的に満足される。物体力がないとき、式(1・40)のひずみの適合条件により,は二次元重調和方程式옿옿+2 옿옿옿+옿옿=0(1・66)を満たす重調和関数(biharmonic function)となる。境界条件が応力のみに関して与えられる単連結領域の平面応力問題および平面ひずみ問題では、σ욨を除いて弾性体内の応力分布はE欄の応力関数を解くことにより、弾性係数に無関係に定まる。多連結領域の問題では、変位の多価性を取り除く必要があるため、応力関数だけでは応力分布を決定できず、B 欄、D 欄の方程式を合わせて解析する。ただし、多連結問題でも、穴の境界に働く外力が個々の穴で釣合っている場合は、σ욨を除く応力分布は両問題に共通で、弾性係数に関係しないことが知られている。ある平面応力問題に対する解が求められている場合、この解を弾性係数とνで表し、νν1-νと置換えると、対応する平面ひずみ問題の解が得られる。同様に平面ひずみ問題の解は、νν1+νの置換えによって平面応力問題の解となる。
