# 1・10・2 仮想仕事の原理

> **章**: 第1章 基礎
> **節**: 1・10 固体力学における変分原理

以上の場の方程式をすべて満足する解を以下では真の解という.物体内部で連続な1階の導関数を有し,かつ욊上で幾何学的境界条件(1・73)を満足する変位場を運動学的可容変位場(kinematical-ly admissible displacement distribution)という.また物体内部で平衡方程式(1・70)を満足し,かつ욧上で力学的境界条件(1・74)を満足する応力場を静力学的可容応力場(statically admissiblestress distribution)という.いま物体が平衡状態にあり,変位욡,応力σ욡욢は真の解であるとする.物体の各点において連続な1階の導関数を有する仮想的な無限小変位δ욡を与える.ただし,仮想変位(virtual displacement)δ욡は욊上で零とする.このとき仮想変位場욡+δ욡は運動学的可容変位場となる.この変位場は,必ずしも物理的に存在するものではないという意味で仮想的である.応力場σ욡욢,表面力욡,物体力욡の値は変化しないものと考え,外力が仮想変位δ욡に関してなす外部仮想仕事δ욄욧δ욄욧=욡δ욡d+욡δ욡d(1・75)を考える.욊上でδ욡が零であり욧上の面積分を全体に広げられることに留意してガウスの発散定理を適用し,応力場σ욡욢が平衡方程式(1・70)を満足することに留意し,δ욡を式(1・71)に代入して得られる仮想ひずみ場δε욡욢を導入すれば,式(1・75)は次のように書ける.욡δ욡d+욡δ욡d=σ욡욢δε욡욢d(1・76)上式は,もし応力場が静力学的に可容であるならば,外力が任意のδ욡に関してなす仮想仕事が,応力場が仮想ひずみ場に関してなす内部仮想仕事に等しいことを示している.逆に,式(1・76)から出発して逆の順序で変形すれば,-σ욢욡욢+욡δ욡d+욡-욡δ욡d=0(1・77)を導くことができる.δ욡は連続な1階の導関数を有すること以外は内と욧上で任意であるから,式(1・70)および(1・74)が成立し,その応力場は静力学的に可容である.この関係を,変形する物体に対する仮想仕事の原理(principle of virtual work,仮想変位の原理ともいう)という.上記の誘導過程では構成式(1・72)を使用しておらず,したがって仮想仕事の原理は任意の力学特性を有する物体について成立する.
