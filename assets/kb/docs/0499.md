# 1・10・2 仮想仕事の原理

> **章**: 第1章 基礎
> **節**: 1・10 固体力学における変分原理

以上の場の方程式をすべて満足する解を以下では真の解という。物体内部で連続な1階の導関数を有し、かつ上で幾何学的境界条件(1・73)を満足する変位場を運動学的可容変位場(kinematical-ly admissible displacement distribution)という。また物体内部で平衡方程式(1・70)を満足し、かつ上で力学的境界条件(1・74)を満足する応力場を静力学的可容応力場(statically admissiblestress distribution)という。物体の各点において連続な1階の導関数を有する仮想的な無限小変位δを与える。この変位場は、必ずしも物理的に存在するものではないという意味で仮想的である。この関係を、変形する物体に対する仮想仕事の原理(principle of virtual work、仮想変位の原理ともいう)という。上記の誘導過程では構成式(1・72)を使用しておらず、したがって仮想仕事の原理は任意の力学特性を有する物体について成立する。
