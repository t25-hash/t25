# 1・10・5 ヘリンガー・ライスナーの原理

> **章**: 第1章 基礎
> **節**: 1・10 固体力学における変分原理

フー・鷲津の原理において、運動学的関係式(1・71)は成立しているものと考え、ε욡욢を消去する。そのため、式(1・72)を用いて式(1・80)のを욂で置換えると、次の汎関数を得る。=-욂σ욡욢+12σ욡욢욡욢+욢욡-욡욡d-욡욡d-욡욡-욡d(1・82)変分を受ける変数は욡,σ욡욢,욡であって付帯条件はない。変分をとり停留条件を求めると次式を得る。δ=2욡욢+욢욡-옿욂옿σ욡욢δσ욡욢-σ욢욡욢+욡δ욡d-욡-욡δ욡+욡-σ욢욡욢δ욡d-욡-σ욢욡욢δ욡d=0(1・83)任意の変分に対して式(1・83)が成立する条件から、式(1・70)および(1・72)~(1・74)を得る。これをヘリンガー・ライスナーの原理(Hellinger-Reissner’s principle)という。
