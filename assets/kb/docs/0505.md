# 1・12・1 変形とひずみ

> **章**: 第1章 基礎
> **節**: 1・12 有限変形理論

時刻=0のときを基準にして、物質の運動を計測する。このことは、次のように表すことができる。変位は粒子の移動量であり、次のように定義できる。この線素が時刻のとき、線素pqに変化し、十分に小さい線素なら直線で近似でき、その位置ベクトルはdになったと仮定できる。線素pqは最初の線素PQに比べ、その大きさも向きも変化する。変形こう配(deformation gradient)テンソルとは、線素ベクトルdの線素ベクトルdへの写像として次のように定義できる。または、デカルト座標を用いると、直接的に、線素ベクトル間の微分として次のように表すことができる。変形は隣接する粒子で変化し、速度ベクトルdほど変化する。速度ベクトルは、現在の時刻と座標で表すことができる。また、変形後のベクトルdを基準にしたアルマンシのひずみ(Almansi’sstrain)は次のように定義できる。このため、ひずみ速度は、速度こう配テンソルの対称成分として次のように求めることができる。
