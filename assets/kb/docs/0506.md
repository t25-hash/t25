# 1・12・2 応力

> **章**: 第1章 基礎
> **節**: 1・12 有限変形理論

変形後の状態について、単位面積当たりの内力として、定義する面と方向に基づく応力が定義できる。この量を真応力またはコーシー応力(Cauchy’s stress)という。変形後の微小面素dに働く表面力ベクトルdは、コーシーの公式から次のように求めることができる。変形後の微小面積dと単位法線ベクトルは、変形前の形状では同一粒子からなる面素はdであり、単位法線ベクトルをとする。変形前の形状に対して、第一種ピオラ・キルヒホッフ応力(first Piola-Kirchhoffstress)をとすると、同一の表面力ベクトルdを与えるものとして定義する。これから、第一および第二種ピオラ・キルヒホッフ応力が次のように計算できる。
