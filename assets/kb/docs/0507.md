# 3・3・5 プラガーの線形移動硬化則

> **章**: 第3章 応力解析の基礎
> **節**: 3・4 幾何学的非線形性

位関係式の非線形性に起因して発生する非線形問題を，幾何学的非
降伏曲面はその大きさを変えず，図3･4(ｂ)に示すように，剛体
線形問題（geometrically nonlinear problem）と呼んでいる．さ
らに，接触問題（contact problem）などのように，境界条件が変
形に依存して変化することに起因する非線形問題もある．
本節では，幾何学的非線形問題の基礎理論（3･4･2項）と変分原
理（3･4･3項），幾何学的非線形問題の分類（3･4･4，3･4･5項）お
よび具体例（3･4･6項）について述べ，最後に接触問題（3･4･7項）
について簡単に説明する．
