# 3・4・4 有限変形問題（finite deformation problem）

> **章**: 第3章 応力解析の基礎
> **節**: 3・4 幾何学的非線形性

ひずみ-変位関係の非線形性に起因して，荷重-変位関係に図3･5
に示すような非線形性を伴う問題を，有限変形問題と呼んでい
る ．図3･5(ａ)は荷重漸増型の有限変形問題と呼ばれ，周辺が固
定された平板が横荷重を受けたときの荷重-たわみ関係などが典型
例である．また，図3･5(ｂ)は荷重低減型の有限変形問題と呼ば
れ，わずかにたわんだ単純支持柱が圧縮荷重を受けた場合の初期の
荷重-たわみ関係などに見られる．図中の点線は線形の荷重-変位関
係である．
