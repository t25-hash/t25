# 2・3・1 弾性変形

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・3 一様断面真直軸のねじり

a. ねじり応力関数任意の断面形をもつ無限に長い軸を軸のまわりにねじるとき、軸に直交する断面(面)には、せん断応力τ욨,τ욨が生じる(図2・4)。このねじりによるせん断応力の性質を知るために応力関数(stress function)ψを用いると便利である。このψは断面内のすべての点で옿ψ옿+옿ψ옿=-2θ(2・14)を満足すると同時に境界線上でψ=定数=を満足すればよい。中空断面に対しては、外周辺上と内周辺上とでの値は異なる。b. せん断応力、ねじりモーメントおよびねじれ角の関係応力関数ψを用いると、せん断応力はτ욨=-옿ψ옿,τ욨=옿ψ옿で与えられる。せん断応力の合応力の作用方向を結んだ曲線をせん断応力線といい、これはψ=(定数)で与えられる。境界周辺も一つのせん断応力線であり、に種々の値を入れることによりせん断応力線群を描くことができる(図2・5)。また、ねじりモーメントは外周辺でψ==0、内周辺でψ=、穴の面積をとす図2・4 ねじりの応力図2・5 山型材のねじりせん断応力線れば、=τ욨-τ욨dd=2ψdd+2(2・15)種々な断面についての応力関数が求められている。c. ねじりと薄膜のふくらみとの相似複雑な形状の断面に対して応力関数ψを求めることは容易でないから、薄膜理論(mem-brane theory)との相似を利用して実験的にねじりによる応力を推論すると便利である。すなわち、断面と相似の周辺を有する薄膜(せっけん膜または薄いゴム膜)を張り、一方から圧力を加えると薄膜はふくらんで曲面を形づくる。この曲面とねじりによるせん断応力とは曲面のふくらみが小さい場合には、次の関係がある。(1)薄膜曲面の一点における傾斜はせん断応力の大きさに比例し、(2)曲面の等高線はせん断応力線を表す。(3)ふくらみの全容積はねじりモーメントに相当する。なお中空軸の場合には内周辺を外周辺より適当に高くしておき、内周辺と外周辺の間に薄膜を張り圧力をかける。d. せん断応力の分布せん断応力はc項のように薄膜との相似を利用して求めることができる。一般に(1)中実軸ではねじりによる最大せん断応力は必ず外周辺上に生じる。そうして断面の重心から最も近い辺上に起こることが多い。(2)外周辺および内周辺のせん断応力は周辺の接線方向に作用する。(3)突出角の頂点(図う角のすみ部(図2・5のD 点)では応力集中が起こる(2・5・2項参照)。また、応力集中は次の場合にも生じる。軸にあけられた穴、軸のキー溝、断面の変化する棒など(2・7節参照)。
