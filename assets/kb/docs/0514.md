# 2・3・1 弾性変形

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・3 一様断面真直軸のねじり

このねじりによるせん断応力の性質を知るために応力関数(stress function)ψを用いると便利である。中空断面に対しては、外周辺上と内周辺上とでの値は異なる。せん断応力の合応力の作用方向を結んだ曲線をせん断応力線といい、これはψ=(定数)で与えられる。境界周辺も一つのせん断応力線であり、に種々の値を入れることによりせん断応力線群を描くことができる(図2・5)。c. ねじりと薄膜のふくらみとの相似複雑な形状の断面に対して応力関数ψを求めることは容易でないから、薄膜理論(mem-brane theory)との相似を利用して実験的にねじりによる応力を推論すると便利である。すなわち、断面と相似の周辺を有する薄膜(せっけん膜または薄いゴム膜)を張り、一方から圧力を加えると薄膜はふくらんで曲面を形づくる。この曲面とねじりによるせん断応力とは曲面のふくらみが小さい場合には、次の関係がある。(1)薄膜曲面の一点における傾斜はせん断応力の大きさに比例し、(2)曲面の等高線はせん断応力線を表す。(3)ふくらみの全容積はねじりモーメントに相当する。なお中空軸の場合には内周辺を外周辺より適当に高くしておき、内周辺と外周辺の間に薄膜を張り圧力をかける。d. せん断応力の分布せん断応力はc項のように薄膜との相似を利用して求めることができる。一般に(1)中実軸ではねじりによる最大せん断応力は必ず外周辺上に生じる。そうして断面の重心から最も近い辺上に起こることが多い。(2)外周辺および内周辺のせん断応力は周辺の接線方向に作用する。(3)突出角の頂点(図う角のすみ部(図2・5のD 点)では応力集中が起こる(2・5・2項参照)。また、応力集中は次の場合にも生じる。軸にあけられた穴、軸のキー溝、断面の変化する棒など(2・7節参照)。
