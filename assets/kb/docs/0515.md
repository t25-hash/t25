# 2・3・2 塑性変形

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・5 のA,B,C,E,F点)では、せん断応力は零である。(4)お

2・3・1・c項の薄膜との相似では、塑性域での曲面の傾斜は一定で、弾塑性境界でψは連続である。b. 全面塑性全断面が降伏した極限の状態では、合せん断応力はいたるところでτとなり、周辺から内側に向かってτの傾斜の屋根を作れば、この屋根と底面とで囲む体積は、ねじりモーメントの半分である。表2・1に各種の断面形をもつ軸にねじりモーメントが作用したときのせん断応力、単位長さ当たりのねじれ角、単位体積に対する弾性ひずみエネルギー(elastic strain energy)およびねじり応力関数ψを示す。だ円形断面のゆがみの等高線は図2・6のような双曲線となる。
