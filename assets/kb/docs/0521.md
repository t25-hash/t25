# 2・7・3 キー溝(key way)を有する軸のねじり

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・7 軸の応力集中

キー溝をもつ丸棒がトルクを伝達するときは純ねじりによるキー溝底のせん断応力の集中とキーの伝達力による溝側面加圧による垂直応力の集中が重ね合わさったものである。図2・13にねじりモーメントのみによる溝底の集中せん断応力のαを示す。最大せん断応力はキー溝底とすみの丸みの接点から丸みの側に約15°離れたA 点に生じる。基準応力を、キー溝がないときの丸棒の最大せん断応力τ=16πとすれば、α=ττは図2・13のようになる。キー溝の幅が大きいほど、深さが小さくなるほど、すみの丸みρが大きくなるほど、αは小さくなる。
