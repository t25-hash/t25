# 2・8・2 塑性変形

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・8 組合せ応力を受ける軸

a. ねじりと曲げを受ける軸軸の断面全体が塑性状態になるときの,ねじりモーメント,曲げモーメントの下界の値(安全側の近似値),は次式の関係がある.+=1(2・31)ここで,,はそれぞれ単独に作用した全領域塑性の荷重である.b. ねじりと軸力を受ける軸このとき軸力の下界の値ととの関係は+=1(2・32)であり,は軸力だけ作用したときの全塑性荷重である.c. ねじりと曲げと軸力を受ける軸このとき下界の値の関係として,++1+=1(2・33)が成立する.
