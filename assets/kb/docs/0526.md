# 2・10・1 弾性波(elastic wave)

> **章**: 第2章 引張り・圧縮とねじり
> **節**: 2・10 応力波の伝ぱ

a. 棒を伝ぱする縦波図2・15に示すように、一様断面を有する弾性棒の横断面に、軸方向の垂直応力が一様に分布すると仮定する。したがって、式(2・37)で表される波動は、変位の方向と波動の進行方向が一致する縦波である。以上の理論は、断面寸法に比べて波長が十分大きい場合に成立する。そうでない場合には、棒の軸に垂直な方向の変形により生じる慣性力の影響が無視できなくなる。このとき波数は角周波数ωの関数となり、したがって伝ぱ速度も角周波数の関数となるので、波動の伝ぱとともに波形が変わる。群速度は波動のエネルギーの伝ぱ速度に等しい。棒を伝ぱする縦波に関するより厳密な理論は、例えば文献(14)(15)にまとめられている。b. 不連続境界における縦波の反射と透過棒の断面積や材質が不連続な境界に縦波が入射すると、一部は反射され、一部は透過3―20材料力学図2・15 弾性棒の微小要素に作用する力される。これらを考慮すれば、境界における力の釣合いσ=σと粒子速度の連続性=の二つの条件から以下の関係式を得る。d. 丸棒を伝ぱする横波一様断面を有する弾性丸棒にねじりモーメントが作用する場合を考える。は三次元弾性体中を伝ぱするせん断波の速度に等しい。この理論による結果は、三次元動弾性理論による厳密解の一次モードと一致する。
