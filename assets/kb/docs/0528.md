# 3・1・1 外力と支持反力

> **章**: 第3章 はりの曲げ
> **節**: 3・1 はりのせん断力と曲げモーメント

例えば図3・1(a)のように、外力が作用しているはりを壁に完全に固定している(並進および回転運動がない)場合を考える。壁からはりを仮想的に切断し、自由物体図を描くと、図3・1(b)のようになる。このとき、もし壁からの支持がなければ、自由物体は加速度をもって時計まわりに回転しながら下に落下する。壁はこのような運動を妨げ、自由物体を静止のまま(平衡状態)に支持しているので、壁からの力や偶力を支持反力(広義)と呼ぶ。ここで、支持の仕方として、次の二とおりがある。a. 単純支持(simple support)支持点で自由物体の並進運動のみを拘束し、回転運動は拘束しない。つまり、支持点で力は伝達するが、偶力は伝達されない(例えば図3・2)。b. 固定支持(fixed support)支持点で自由物体の並進および回転運動の両者を拘束する。つまり支持点でおよびの両者が伝達される(例えば図3・3)。
