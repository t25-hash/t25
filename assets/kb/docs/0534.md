# 3・7・5 ストローハル数

> **章**: 第3章 流体力学の基礎式
> **節**: 3・7 流れの相似則と無次元パラメータ

開水路の模型実験などでは，フルード数を合わせるとレイノルズ
無次元パラメータ＝は非定常流においてのみ問題と
数は合わなくなることが起こる．しかし，レイノルズ数の相違に基
づく流れの様子の変化は連続的であまり著しくないのが通例である
速度 の流れの中に置かれた直径 の円柱の後流中にカルマン
ので，レイノルズ数が実物のそれと異なっていても，実物の場合を
ほぼ推定できる．ただし，模型実験と実物のレイノルズ数との間に
するとその周期が代表時間＝1となり，また代表長さ＝
