# 3・10・3 エネルギー法(energy method)

> **章**: 第3章 はりの曲げ
> **節**: 3・10 不静定ばり

はりの長さに沿って曲げ応力σ=が一様なはりを平等強さのはり(beam of uniform strength)という。このはりを得るためには、はりの各断面の断面係数を曲げモーメントに比例して変化させればよい。曲げモーメント零の断面にもせん断力を支える断面積が必要であるから、実際に用いうる平等強さのはりは上の定義とは少し違った近似的な形状になる。表3・7に長方形断面の平等強さのはりの形状とたわみの公式の例を示す。図中破線で示した形は近似形であり、記号については、:集中荷重、σ:一様な曲げ応力である。車両によく用いられる重ね板ばね(lamellar spring)は平等強さのはりの原理を利用した適例である。軽量設計(light weight design)という立場では、単に断面の高さや幅を変えて平等強さとするだけでなく、断面の形を工夫することも必要である。丸棒よりは中空円管を、長方形断面よりはI型断面を、というように、はりの材料がなるべく中立軸から離れた位置に配置された形を選ぶことによって、一様断面でも大幅の軽量化が図られる。肉を薄くするには座屈などによる制限がある点に注意を要する。強さならびに弾性係数の異なる2種以上の縦部材をせん断力に対し、十分な結合強度で組合わせたはりを組合せはり(compositebeam)という。
