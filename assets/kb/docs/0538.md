# 3・13・2 極限曲げモーメント

> **章**: 第3章 はりの曲げ
> **節**: 3・13 はりの塑性曲げ

長方形断面の弾完全塑性体では、式(3・68)より明らかなように、3―36材料力学塑性曲げの進行(ρの減少)につれては増加し、ρ→0の極限において曲げモーメントの値は욣=1.5욦=σ욦4に収束する。この極限値욣を極限曲げモーメント(ultimate bendingmoment)あるいは全塑性曲げモーメント(totally plastic bend-ing moment)と呼ぶ。長方形以外の断面に対する極限曲げモーメントは욣=σ욦욣で計算できる。ただし、욣は塑性断面係数(3・2節参照)である。極限曲げモーメント욣と降伏開始曲げモーメント욦の比は形状係数=욣(3・2節、表3・1参照)に等しい。
