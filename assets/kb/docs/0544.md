# 3・16・1 はりの横座屈

> **章**: 第3章 はりの曲げ
> **節**: 3・16 はりの座屈

図3・30 二軸対称断面はりの横座屈荷重異なる場合に、曲げ剛性最大の方向にはりが曲げられると、負荷がある値ではりが横に倒れ、ねじれて座屈する。これをはりの横座屈あるいは横倒れ座屈(lateral buckling)という。二軸対称断面をもつはりに限れば、無次元量は図3・30の値をとる。またとΓについては次を参考のこと。なお、この断面変形(distortion)は、断面面外の変形(そり、warping)とは別の量であることに注意を要する。一般に負荷点がはりの中立面よりも上にあれば座屈荷重が小さくなり、下にあれば大きくなる。
