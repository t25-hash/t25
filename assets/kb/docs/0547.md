# 3・3・1 各種燃料の燃焼

> **章**: 第3章 燃焼
> **節**: 3・3 燃焼装置内の燃焼

気体燃料の燃焼形態は，予混合燃焼（premixed combustion）は，燃料と空気（酸化剤）をあらかじめ混合させた後に燃焼室内で一定の関係が成立している必要がある．
これに対して拡散燃焼では，燃焼装置内に燃料と空気を別々に供給焼負荷率は予混合燃焼より小さくなるが，燃料と空気を別々に燃焼る．
これらの理由から，特に大型の燃焼装置で拡散燃焼方式を採用燃焼）させることによって，窒素酸化物（NO）を比較的容易に蒸発過程と反応過程が密接に結びついている．
燃焼形態としては，ことにより微粒化し，空気との混合をよくして燃焼させる噴霧燃焼分が気化して空気（酸化剤）と反応する形態と，固体のまま表面で混合特性が，燃焼状態を支配する．
また，燃料と空気の流動は火炎炎形状の関連を把握しておくことが重要である．
熱速度の変動が流速や圧力の変動を駆動し，誘起された音響的固有フィードバックされる必要がある．
