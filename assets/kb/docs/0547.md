# 3・3・1 各種燃料の燃焼

> **章**: 第3章 燃焼
> **節**: 3・3 燃焼装置内の燃焼

気体燃料の燃焼形態は，予混合燃焼（premixed combustion）
と拡散燃焼（diﬀusion combustion）に大別される．予混合燃焼
は，燃料と空気（酸化剤）をあらかじめ混合させた後に燃焼室内で
燃焼させる方式であり，単位体積当たりの発熱率（燃焼負荷率）を
一定の関係が成立している必要がある．
高くとれるが，可燃性混合気を形成するため逆火の危険性がある．
これに対して拡散燃焼では，燃焼装置内に燃料と空気を別々に供給
し，燃焼室内で混合と燃焼を並行して生じさせる．拡散燃焼では燃
焼負荷率は予混合燃焼より小さくなるが，燃料と空気を別々に燃焼
室に供給するため逆火の危険性はなく，かつ操作範囲が広くとれ
る．これらの理由から，特に大型の燃焼装置で拡散燃焼方式を採用
する場合が多い．一方，予混合燃焼では空気過剰燃焼（予混合希薄
燃焼）させることによって，窒素酸化物（NO ）を比較的容易に
抑制することが可能である．
蒸発過程と反応過程が密接に結びついている．燃焼形態としては，
噴霧燃焼，液面燃焼等がある．工業的には，液体燃料を噴霧させる
ことにより微粒化し，空気との混合をよくして燃焼させる噴霧燃焼
の形態を採用する場合が多い（3･2･6項参照）．
分が気化して空気（酸化剤）と反応する形態と，固体のまま表面で
空気と反応する形態がある（3･2･7項参照）．
混合特性が，燃焼状態を支配する．また，燃料と空気の流動は火炎
安定に重要な役割を果たす．このため，燃焼装置内の流動様式と火
炎形状の関連を把握しておくことが重要である．
