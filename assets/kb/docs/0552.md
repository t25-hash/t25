# 4・4・1 初期たわみのある長柱

> **章**: 第4章 柱の圧縮・座屈
> **節**: 4・4 初期たわみおよび偏心荷重を受ける長柱

一般に微小な初期たわみ(initial deflection)を有する長柱に軸圧縮荷重が作用する場合、弾性域・微小たわみの範囲での問題は、平衡方程式が++=0となるので、分布荷重=-が作用する同一端末条件のまっすぐな柱の問題に帰着される。したがってその解法は、4・3節のそれと同様であり、生じるたわみ、曲げモーメントなどの性質も同様であって、がオイラー座屈荷重욅に近づくとそれらは急速に増大する。代表例として、両端回転端の長柱(長さ:、一端を原点とする長手座標:)において、初期たわみ:=∑욖욇sinπが存在する場合の解析結果を示せば、オイラー座屈荷重を욅=πとして、たわみ:=∑욖욇욅-1sinπ(4・10)(初期たわみからの)曲げモーメント:=∑욖욇욅욅-1sinπ(4・11)となる。この種の柱の強度については、4・5節参照。
