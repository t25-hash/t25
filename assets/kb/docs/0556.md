# 3・4・7 汚染物質の拡散

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

大気汚染のヒト，生物などへの影響を知り，適切な対策を立てる
ためには，排出量と環境濃度の定量的な関係を把握することが重要
である．大気中での汚染物質の拡散は，風向，風速，風の乱れ，気
温の鉛直変化などの気象条件と地形条件により大きく変化する．こ
のような複雑な条件を組込んで拡散結果を予測することは，通常は
容易ではない．
大気の流れが単純で，汚染物質の大気中での変化がなければ，簡
単な拡散方程式を適用して汚染物質濃度を推定することが可能であ
る．例えば，平坦な場所で大気状態が安定しており，汚染物質も安
定で変化しにくいときは，煙突から排出された汚染物質は，まず煙
突から出た煙の運動量と煙と大気の温度差による浮力で大気中を上
昇し，同時に風に流されつつ拡散する．煙はいったん上昇した後，
拡散により煙突からの距離に応じて地表での濃度（着地濃度）を増
していく．排出された汚染物質の質量と拡散条件が同じであれば，
高い煙突から排出された汚染物質の着地濃度は，低い煙突に比べて
必ず低くなり，最大着地濃度の位置は遠くなる．
しかし，同じような汚染物質の拡散を定量的に考える場合でも，
対象となる空間スケールは数mから数千km，時間スケールは数
分から数年と幅広いため，拡散過程を統一的に扱うことは通常は困
難である．例えば，複雑な地形や構造物の周辺での拡散などは，風
洞を用いた実験や，現地でのトレーサ実験，あるいは拡散方程式を
数値的に解く方法によらなければ定量的な答えは出てこない．ま
た，地球規模の拡散であれば，成層圏まで含めた大気循環や，森
林，海洋などの吸収，放出を考慮しなければならない．
