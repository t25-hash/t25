h.リニアモータ（linear motor ) 交流機の電機子を円筒状か
ら切り開いた状態の直線状とした場合，直線的に移動する移動磁界
が得られる．その移動磁界に相対する部分に界磁極あるいははしご
導体（かご導体を切り開いたもの）を置けば，直線方向に動くリニ
ア同期機あるいはリニア誘導機が構成される．
リニア誘導機は構造が簡単なために，リニア地下鉄，物流システ
ム，磁気浮上鉄道などに応用されている．一方，リニア同期機は構
造簡単な永久磁石タイプが実用化され，例えば，工作機械あるいは
精密位置決め装置などに応用されている．
