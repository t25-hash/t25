# 3・2・1 回転機（electrical rotating machine）の種類

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・2 電動機の種類と特徴

a.直流機（direct current machine) 機械動力を受けて直流
電力を発生する機械（直流発電機），または直流電力を受けて機械
動力を発生する機械（直流電動機）をいう．主として電動機として
用いられている．直流機の国内標準規格はJEC-2120「直流機」で
ある．
b.誘導機（induction machine) 固定子および回転子にそれ
ぞれ独立した巻線を有し，電磁誘導作用によって両巻線間にエネル
ギーを伝えて動作する非同期機（同期速度以外の速度で運転する交
流機）である．誘導機は主として電動機として使用される．誘導発
電機は出力が比較的小さく（7～10MW以下），回転速度が大きい
（300～400min 以上）範囲の小容量水力発電所や，風力発電用に
多く用いられている．誘導機の国内標準規格はJEC-2137「誘導
機」である．
c.同期機（synchronous machine) 定常運転状態において，
機械の極数と周波数によって一義的に決まる同期速度で運転する機
械である．通常，固定子に電機子巻線を，回転子に界磁巻線を有
し，主として電力用の発電機として用いられている．磁極を界磁巻
線の代わりに磁石にしたものを磁石同期機といい，電動機または発
電機が製作されている．一方，界磁に界磁巻線または永久磁石をも
たない突極構造の小容量の同期電動機をリラクタンスモータとい
う．同期機の国内標準規格はJEC-2130「同期機」である．なお，
誘導機と同期機は交流電力で駆動，または交流電力を発生するので
交流機ともいう．
