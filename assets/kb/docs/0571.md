# 3・2・3 直流機（direct current machine)

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・2 電動機の種類と特徴

a.直流電動機の構造 図3･26は直流電動機の構造と原理を
示す図である．コイル（電機子巻線）の両端は整流子S ，Sに接
続され，ブラシB ，Bを介して直流電圧 が与えられる．コイ
ルは磁界中を運動するので誘導起電力 を発生する．コイルには
両者の差電圧－に比例する電流 が流れ，磁束と電流の間
に働く電磁力によってトルクを発生し回転する．図3･26において，
コイルが180 °回転すると整流子に接触するブラシが機械的に切換
わるので，磁界とコイル電流の関係は同一となり，つねに同一方向
のトルクを発生する．この整流子の作用を整流という．図3･26は
1コイルのみを示しているが，実際の直流機は多くのコイルで構成
され，各コイルを直列に接続してブラシから取出している．直流機
の主要構成部品は，固定子では主磁極，補極およびブラシ保持器で
あり，回転子では電機子と整流子である．
.固定子 直流機の固定子継鉄は，鋳鉄または鋼板で作ら
れ，磁路を形成すると同時に，機械的構造部分として用いられる．
急速な制御性能が要求される直流機では，界磁電流変化に対する磁
束変化の追従を良くするため，継鉄部を積層構造にすることがあ
る．主磁極は，1～3.2mm厚の積層鉄心からなる界磁鉄心と界磁
巻線を一体に組立てた後，継鉄に取付けられる．小型機では主磁極
を永久磁石で構成することが多い．補極は，主磁極の中間に設けら
れる幅の小さい磁極で，補極巻線には電機子電流を流すことによっ
て整流作用の改善を行っている．また，中型機以上では主磁極の磁
極片部に多くのスロットを設け，その中に納めた巻線に電機子電流
を流す場合がある．この巻線を補償巻線といい，電機子反作用（電
機子電流によりギャップ磁束が変化する作用）を打消してギャップ
磁束分布の改善を行う．
.回転子 電機子鉄心は界磁とともに磁路を形成するが，
界磁鉄心と異なり交流磁束が流れる．したがって，交流機固定子鉄
心と同様に，電機子鉄心は0.5mm厚のケイ素鋼板を所要の形に
打抜いたものを積層し，これを回転子軸に焼ばめて回転子を作る．
電機子鉄心は，積厚40～60mmごとに約10mm幅の冷却ダクト
を設けている．現在製作されている産業用の直流機の電機子巻線
は，ほとんど耐熱クラスF（耐熱温度155℃）で電機子鉄心に巻線
を納めた後に，鉄心と巻線をエポキシ樹脂で一体含浸する全含浸絶
縁方式が広く採用されている．整流子は，硬銅材料の整流子片をマ
イカ板で相互に絶縁し，これを円筒状に組立てる．整流子片と電機
子コイルとの接続は，中型機以上ではライザを用い，小型機では直
接ろう付けされる．
b.直流電動機の等価回路と電動機基本式 直流機の励磁方式
は，他励式，分巻式，直巻式および複巻式がある．サイリスタ，ト
ランジスタなどの半導体電力変換装置によって，励磁電源が容易に
作れるようになったので，他励式が鉄鋼用駆動機，工作機，製紙機
などすべての用途に広く用いられた．図3･27に直流他励電動機の
結線と等価回路を示す．誘導起電力 は可変電圧で，界磁電流と
回転速度の関数である．鉄損，機械損およびブラシ電圧降下を無視
すると，直流電動機の基本式は次式で与えられる．
