# 6・1・8 円板の変形挙動の数値解法(numerical method)

> **章**: 第6章 圧力・遠心力などを受ける円板、円筒および球
> **節**: 6・4 )は、どのようなに対しても、適用することができる。

厚さの変化する円板(disk of varying thickness)を適当に分割し、その各々を一様厚さの円板とみなし、これらを接続して円板全体の応力を求めるドナートの方法(Donath’s method)がよく用いられてきたが、より一般的な加工硬化特性を有する有孔円板の変形解析には有限要素法が最も有力である。円板の厚さの変化が大きくない場合ならびに変形によって円板の板厚に大きな変化が生じない場合、軸対称平面応力状態が成立するとして、半径方向のみに各種物理量が変化するとする、一次元解析図6・7 円板表面の熱衝撃応力図6・8 熱衝撃最大応力σ윯율융とビオの係数βの関係が可能となる。板厚が一様でない場合については、軸対称体とみなすことによって、変形解析をすることができる。弾塑性変形、クリープ変形については、対応する構成式を軸対称体に対する有限要素法に導入することにより、熱応力については、軸対称熱伝導有限要素法、ラプラス変換によって、衝撃変形については、動的問題に対する有限要素法、ラプラス変換によって解析が可能である。また、円周方向に分布する力、変形ならびに温度が作用した場合の円板の非軸対称変形を、フーリエ級数と有限要素法を用いた解析法も提案され、モードごとの軸対称解析を行うことにより、非軸対称変形の解析を可能とする。しかしながら、計算速度ならびに記憶容量の向上によって、一般三次元体として解析を行う傾向にある。円板の内周に一様な半径方向変位を加えた状態は、円板の深絞り加工中のフランジ部の変形を近似しており、円周方向に面外の変形、しわの発生を評価することによって、加工性を知ることができる。関連した研究として、軸対称平面応力状態にある円板が曲げ変形を伴った非軸対称変形への分岐(bifurcation)ととらえ、ヒルの変形の分岐定理と有限要素法を用いた解析法が提案されている。円板外周に半径方向一様引張変形を加えた場合、円板内周にくびれ(necking)が発生する。この場合、軸対称平面応力変形から非軸対称平面応力変形への分岐として扱うことができる。このような分岐挙動の解析は、円筒の軸対称変形から非軸対称変形への分岐解析と同様に行うことができる。回転円板については、遠心力を物体力として扱うことによって、容易に解析可能である。この場合も、回転速度が限界速度を超えると、非軸対称変形が生じる。延性材料の場合の限界速度を、軸対称変形から非軸対称変形への分岐として解析した研究などがある。
