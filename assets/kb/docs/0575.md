# 6・2・1 内外圧(internal and external pressure)を受ける円

> **章**: 第6章 圧力・遠心力などを受ける円板、円筒および球
> **節**: 6・2 円筒(cylinder)

筒の弾性変形(elastic deformation of cylinder)内外圧を受ける円筒の支配方程式は、平衡方程式と適合条件式については円板の式と同じであるが、構成式のみ次式のように異なる。このとき、解の形は6・1・1項と同形になり、σを除く面内応力成分σ,σは内外径が同じ円板と同じ値になる。円筒の端末条件としては、次の3種類の境界条件を取上げている。両端がプラグなどで閉じられている場合(両端閉じ)、軸方向の変形のみが拘束されている場合(平面ひずみ)、両端が自由になっている場合(両端開放)についてである。外図6・9内外圧を受ける円筒3―63圧が作用した場合の応力にはチルダ(~)を付けて示すと、内圧が作用した場合の応力との間に下記の関係が成立する。
