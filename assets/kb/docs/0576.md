ソレノイドの基本的な構造は，図3･49に示すように電磁石のコ
イルを励磁して，可動鉄心であるプランジャを吸引動作するもの

持型ソレノイドなどもある．
c .ムービングコイルアクチュエータ（moving coil actuator )
ムービングコイルアクチュエータの代表的なものはVCM（voice
coil motor ，ボイスコイルモータ）であり，ダイナミックスピーカ
のボイスコイル部と同じ原理によって作動するので，ボイスコイル
モータと呼ばれる．図3･50に示すように永久磁石によって一様な
磁界を円筒状の空げき部に作り，その中に円筒状の可動コイルが挿
入されている．コイルに電流を流すと，フレミングの左手の法則に
よりコイルに推力が発生し，この力は電流に比例する．可動コイル
は鉄心がないので可動部の慣性が小さく，電気的時定数も小さいの
が特徴である．
図3･51にムービングコイルアクチュエータの別の構造例を示す．
VCMに比べ構造が簡単なのではん用性がある方式である．
ムービングコイルアクチュエータは，ソレノイドと比べ線形性と
応答速度に優れており，磁気ディスク装置の磁気ヘッドなどの位置
決めが代表的な用途である．
d.リニアDCモータ（LDM:linear DC motor ) 直流のリニ
アモータ全般が，当初はリニア直流モータと呼ばれていた．例えば
c項のムービングコイルアクチュエータや，e項のブラシレスDC
モータのリニア型もLDMとする分類方法もあるが，分類の細分化
が進んでブラシレス型はLSM（リニア同期モータ）に分類される
ようになったので，LDM自体は回転型のDCモータをリニアに展
開したものに限るほうが，わかりやすい分類法と考えられる．
これまでにブラシ付のDCモータを，リニアモータ化した例は存
在するが，実用的ではないので，現実にはブラシレス化が進み，e
項のLSMに置換わってきている．
e.LSM（linear synchronous motor ，リニア同期モータ)
回転型のブラシレスDCモータの電磁的構造を，直線上に展開して
直線運動するモータである．永久磁石界磁と電機子コイルの組合せ
電気・電子の基礎 第3章 電気機器とパワーエレクトロニクス
が最も標準的な構造で，空芯コイルを用いたコアレスと電機子鉄心
を用いたコア付に分けられる．
リニアモータは回転機と異なり，ストロークの全範囲にわたって
推力を発生するギャップが対向しなければならない．すなわち，ブ
ラシレスDCモータの固定子の電機子巻線側か，回転子の永久磁石
側のいずれかをストローク全長に配置する必要がある．一般に電機
子を可動側にして，永久磁石を固定子全長に配置することが多い
が，その場合可動側の電機子に給電する必要がある．可動部への給
電をなくすため，永久磁石を可動側とすることもある．
電機子鉄心と永久磁石の間には，強力な磁気吸引力（magnetic
attraction）が発生する．回転型のモータは軸中心の構造から，ギ
ャップ部で吸引力がバランスし外部に出ることはないが，リニアモ
ータはこの吸引力に対応した構造を採る必要がある．
.コアレス 永久磁石とヨーク間，あるいは永久磁石を両
側に配置した磁界中に，渦巻状に巻かれた空芯（コアレス）のコイ
ルを進行方向に並べて配置したもので，可動子となるコイル部は樹
脂モールドしたものが一般的である．コア付に比べ，空げき長が長
くなるので推力は小さい．磁気吸引力は固定子内で処理すればよ
く，コアレス構造でコギング力の発生もないので，推力リップルを
いかに低減するかということと，推力と直角方向に発生するサイド
フォースを小さく抑えることが，この方式のポイントとなる．図
3･52に断面構造を示す．

.HD-LSM（high-density-type linear synchronous
motor ，高力密度型リニア同期モータ) 図3･55に示すHD
（high density）型と呼ばれるLSMは，電機子内部に永久磁石が組

f.LIM（linear induction motor ，リニア誘導モータ) 回転
型の誘導型ACモータの電磁的構造を，直線上に展開して直線運動
をするモータである．ACサーボモータの場合と同様に，大型のモ
ータになるとマグネットの体積が増え高価になる．そこで大型大出
力の搬送用などの用途では，誘導型が主流になる．
g.LPM（linear pulse motor ，リニアパルスモータ) 回
転型のステッピングモータに対応させると，直線運動型の呼び名は
リニアステッピングモータとなるが，略称がLSMとなりリニア同
期モータと同じになるので，ここではリニアパルスモータという呼
び方を使用する．
回転型のステッピングモータと同様に，LPMというモータ構造
が特別にあるわけではなく，モータ構造はLSMと基本的に同じで
あるが，送りピッチを小さくするため小歯（誘導子）を設けた構造
が多い．
代表的な方式は，図3･56に示すPM型（permanent magnet
type：永久磁石型）とVR型（variable reluctance type）である．
