# 6・2・2 内外圧を受ける薄肉円筒

> **章**: 第6章 圧力・遠心力などを受ける円板、円筒および球
> **節**: 6・2 円筒(cylinder)

厚さ方向の周方向応力の変化が無視できるほど十分に肉厚が小さい場合には、薄肉円筒(thin-walled cylinder)として次式のような近似式が用いられる。σ=욇욀(内圧の場合)(6・44)σ=-욇욁(外圧の場合)(6・45)ここで、욇は平均半径、は肉厚を示す。表6・1の厚肉円筒の結果を用いて、内圧あるいは外圧を受けるときに内外縁で生じる周方向応力の比を求めると、=1.1(肉厚が内径の1/10)のときには、σ욥욀がσ욥욁より約10%大きいだけである。この程度ならば、近似的に応力が一様に分布しているとする薄肉円筒の式を用いても差支えない。また、外圧が作用する薄肉円筒では、座屈を考慮しなければならないことに注意が必要である(8・5・3項参照)。種々の端末条件における薄肉円筒に対して、内半径の増加δ、軸方向ひずみε욨、および内容積の増加δを示したのが表6・2である。
