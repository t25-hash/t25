b.各種パワー半導体デバイス
.サイリスタ（thyristor ) サイリスタは三つ以上のpn
接合を有し，オンとオフの二つの安定状態がある半導体デバイスの
総称であるが，一般的には，順電圧（カソードに対しアノードの電
圧が高電圧）のみならず，逆電圧（カソードに対しアノードの電圧
が低電圧）を印加した場合も電流を阻止する逆阻止三端子サイリス
タのことを指す．サイリスタを導通状態にする（ターンオンと呼
ぶ）ためには，ゲートからカソードにパルス状のゲートトリガ電流
を流す．サイリスタには，高電圧の領域でも損失が小さいという利
点があるが，導通状態の電流を遮断する（ターンオフと呼ぶ）ため
には，アノード・カソード間に逆電圧を印加することが必要で，使
い方によっては，転流回路と呼ばれる専用の外付回路が必要であ
る．おもな適用先は直流送電をはじめとする電力分野である．
.GTO（gate turn-oﬀthyristor ) GTO（ゲートターン
オフサイリスタ）はサイリスタの一種であるが，ゲートに負の電流
を流すことによりターンオフする機能を有することが特徴である．
1960年代に発表されたが，1980年代になって大容量化が進み，電
力，圧延機，鉄道車両等の分野で実用化された．ターンオフさせる
ためには，瞬時ではあるが，アノード電流の1/3～1/5程度の大電
流をゲート電流として流す必要がある．また，多くのユニットを並
列接続して大電流化を図るため，デバイス上の電流の広がり時間に
より，過渡的な電流集中やターンオフ動作の不具合が起こる可能性
がある．これを防ぐために，外部にスナバ回路と呼ばれる付加回路
やアノードリアクトルが必要で，装置が複雑で大型になり，損失も
大きくなる．この問題を解決するために，ゲートのインダクタンス
を低減し，電流集中を防ぐデバイス構造も提案されている．
.バイポーラパワートランジスタ（bipolar power transis-
tor ) バイポーラパワートランジスタは，ベース電流によって高
速のスイッチングができ，増幅率も高いため，オーディオ，自動車
電装品，小型電源等に使用されていた．1980年ごろから，並列接
続のモジュールによって大容量化が進み，産業用機器分野でも使用
されるようになった．しかし，次第に，次に述べるパワーMOS-
FETやIGBTに置換えられた．
.パワーMOSFET（power MOSFET) パワーMOS-
FETはLSIに使用されるMOSFETをパワー分野向けに改良した
デバイスで，1970年代に提案された．高周波化に適している横型
パワーMOSFETは移動体通信用の高周波パワーアンプなどに，
高耐圧化・低オン抵抗化に適している縦型パワーMOSFETはス
イッチング電源，自動車電装品，OA機器のモータ駆動などで商用
化されている．
パワーMOSFETは，電圧駆動のため，駆動回路が簡単で駆動
電力も小さい．また，電流の担体として電子と正孔が利用されるバ
イポーラデバイスと異なり，電子または正孔だけが使われるユニポ
ーラデバイスであるため，キャリヤの蓄積による遅延がなく，高速
スイッチングが可能である．
低電圧の領域では，微細加工技術の導入により，低オン抵抗化が
進められた．また，ゲートが半導体表面上に形成されるプレーナ構
造からゲートをトレンチ溝の中に形成するトレンチゲート構造を用
いることにより，さらに低オン抵抗化が図られている．
パワーMOSFETのようなユニポーラデバイスの場合，高電圧
の領域では，ドレイン耐圧を確保するために，半導体材料の物性に
起因するオン抵抗の理論限界が存在する．この点が，キャリヤの蓄
積に基づく伝導度変調効果を使い，低オン抵抗化が期待できるバイ
ポーラデバイスと異なる．ただし，各種スイッチング電源，照明用
インバータ等，高周波動作を必要とする分野ではMOSFETの要
求が強く，スーパージャンクション型 という構造で，高耐圧で
オン抵抗が小さいパワーMOSFETが実現されている．オン抵抗
と耐圧が最適条件になるリサーフ層を，p領域とn領域の繰返しパ
ターンにして形成するものであるが，プロセス工程は複雑になる．
.IGBT（insulated gate bipolar transistor ) 縁ゲートバイポーラトランジスタ）はパワーMOSFETのn型ド
レイン領域にp型領域を追加し，バイポーラトランジスタを複合
化したものである．パワーMOSFETと同様に，駆動回路が簡単
で，駆動電力が小さく，スイッチング速度が速い．一方で，バイポ
ーラパワートランジスタと同様に大電力化が可能である．高耐圧
化・低損失化技術の進歩により，250～6500Vクラスのデバイス
が開発されており，パワーエレクトロニクスの広範囲な分野で使用
されている．
IGBTの低損失化はパワーMOSFETと同様に微細加工技術や
トレンチ構造化によって達成された．ドリフト領域でのキャリヤ蓄
積効果を促進させることにより，低損失化を図った改良型の開発も
進んでいる．さらに，p型コレクタ拡散層やn型ドリフト領域の不
純物濃度や寸法に関しても最適化の検討がされており，高耐圧化，
低損失化，高速化に向けた改良が進められている．
