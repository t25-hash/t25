# 3・4・4 直流変換装置

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・4 半導体電力変換装置

交流電圧の大きさを変えるためには変圧器が用いられるが，直流
電圧を変圧器で直接変えることはできない．しかし，各種直流電源
や直流電動機駆動のために可変の直流電圧を必要とする用途は多
い．そのような場合に用いられるのが直流変換装置（DC con-
verter ）である．いわば直流の変圧器に相当するものである．直流
から他の直流への変換を直流変換と呼ぶ．
直流変換の代表例は直流チョッパで，単にチョッパ（chopper ）
と呼ばれることが多い．バルブデバイスによって高頻度で直流のオ
ンオフを繰返し，交流を介することなく直接，変換を行う．チョッ
パの基本的な構成としては，降圧チョッパ，昇圧チョッパ，昇降圧
チョッパの3種類がある．降圧チョッパ回路の構成と動作波形を図
3･63に，昇圧チョッパの回路の構成を図3･64に示す．
