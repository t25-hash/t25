# 6・2・7 内圧を受ける円筒の破壊

> **章**: 第6章 圧力・遠心力などを受ける円板、円筒および球
> **節**: 6・2 円筒(cylinder)

a. 延性破壊(ductile fracture)円筒の内圧破壊強度推定によく用いられる評価式を表6・4に示す。しかしながら、≧1.2では誤差が次第に大きくなり、≧1.8では平均径公式がほぼ妥当な推定値を与える。円筒が延性材料でできている場合、内圧が増加するとまず内面が降伏する。次に、外面も降伏する全塑性状態を経て、ついには不安定変形を生じて破壊する。一般のひずみ硬化材では、材料の応力-ひずみ曲線を用いて弾塑性解析を行わなければならない。トレスカ材を用いた場合の内圧と内外周せん断ひずみは、次式によって与えられる。通常、εは円筒の破壊内圧に大きな影響を及ぼさないので、ε=0として求めたγの式[マニングの式(Manning’s formula)]を用いる。これらの式で計算された内圧の最大値が破壊内圧を与えるが、その結果は一般的に次の形で書ける。応力腐食、低サイクル疲労は、しばしばき裂発生の要因となることにも注意が必要である。
