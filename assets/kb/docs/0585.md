.電動機の運転象限（operation quadrant ) 電動機の運
転モードは，回転方向とトルク方向の組合せにより，図3･77に示
す四つの象限で表すことができる．上下は回転方向を示し，左右は
トルク方向を示す．第Ⅰ象限と第Ⅲ象限は通常の電動機運転の領域
であるが，第Ⅱ象限と第Ⅳ象限ではトルク方向が回転方向と逆にな
るため，駆動対象からトルクを受ける発電機運転の領域になる．発
電機運転では，駆動対象が発生するエネルギーを，電力変換器を介
して電源に戻すことができる．これを回生動作と呼ぶ．
ドライブシステムの構成や制御方式を決めるに際しては，あらゆ
る運転パターンを想定して，どの運転象限を使うかを把握すること
が重要である．使う運転象限によって電力変換器の回路構成や制御
アルゴリズムの処理内容が決まる．

の速度を基底速度と呼ぶ．この状態から，電機子電圧を一定に保っ
て，界磁電流を減少させる弱め界磁の制御を行うと，速度はさらに
上昇する．界磁を弱めながら到達できる上限速度を最高速度と呼
ぶ．
