# 3・5・3 ドライブシステムの変遷

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・5 各種ドライブシステム

ドライブシステムの開発が本格化したのは，最初のパワー半導体
デバイスであるサイリスタが実用化された1960年ごろからである．
最初は圧延機や鉄道車両の駆動，工作機械やロボットのサーボがけ
ん引し，パワー半導体デバイスとマイコンの進歩によって幅広い分
野に普及してきた．その変遷の経緯をけん引役になった半導体技術
の進歩，および応用側からの要求と対比して示したのが図3･79で
ある．
初期のドライブシステムには直流電動機が使われた．駆動電源の
構成が簡単であるうえに，電動機の制御パラメータと駆動特性の対
応が明確であるため，比較的簡単な制御で高い性能が得られたため
である．制御の面では，それぞれの応用に対して満足すべき性能が
得られ，各種の装置やシステムの性能向上に貢献した．しかし，直
流電動機の本質的な問題として，機械的な接触部である整流子とブ
ラシの保守の問題が残った．ドライブシステムの発展は，整流子や
ブラシのない交流電動機を使い，直流電動機と同等以上の性能を出
すことに向かって進んできたといってよい．この中で，マイコンと
パワー半導体デバイスの進歩が重要な役割を果たした．交流電動機
を使ったドライブシステムは二つの方向に向かって進んだ．
一つは，直流電動機のブラシと整流子の機能をパワー半導体デバ
イスで置換えようとする方向である．現実には，直流電動機の整流
子の数ほどパワー半導体デバイスを使うことはできないため，三相
の同期電動機を使い，磁極位置のセンサによって，各相に電流を供
給するタイミングを決める方式が一般的だった．当初はパワー半導
体デバイスとしてサイリスタが使われたため，サイリスタモータと
呼ばれた．ここでは，電動機の速度を制御するために，別の電源で
電圧を制御することが必要であった．しかし，IGBT（insulated
gate bipolar transistor ）とMOSFET（metal - oxide semiconduc-
tor field eﬀect transistor ）の登場でPWM（pulse-width modula-
tion，パルス幅変調）制御が可能となり，電圧制御が容易になっ
て，応用が小型電動機まで広がった．電動機のほうも，磁石材料の
性能向上もあって，永久磁石型同期電動機が普及した．この方式
は，直流電動機と同様に制御は行いやすい反面，トルク脈動が大き
いことが欠点である．
もう一つは，可変周波数インバータで誘導電動機を駆動する方向
である．ゲート信号で電流を遮断できるGTO（gate turn-oﬀ
thyristor ）の登場でPWM制御が容易になり，周波数と電圧を容
易に制御できるようになって普及が進んだ．PWM制御では，電圧
を制御できるだけでなく，電動機電流を正弦波に近づけることもで
きる．また，IGBT，MOSFETによって小型化が可能になり，応
用が小型電動機にまで広がった．この方式の欠点は高応答のトルク
制御ができないことである．また，無効電流による損失も発生す
る．
上で述べた交流電動機駆動の欠点を解消するのがベクトル制御で
ある．この狙いは，正弦波駆動のインバータでトルクの脈動をなく
したうえで，トルク電流を励磁電流から分離して制御することで，
高応答のトルク制御と効率を直流電動機と同等にすることにある．
これ以降の部分では，次の順序でドライブシステムの具体的な内
容を説明する．まず，直流電動機によって，ドライブシステムの基
本的な動作を述べる．次に，上に述べた2種の交流電動機のドライ
ブについて述べ，この基本的な動作とともに，ベクトル制御の狙い
を明らかにする．この後，ベクトル制御の原理を述べ，最後に代表
的な応用事例を示す．
