# 7・1・1 定義と概説

> **章**: 第7章 応力集中、接触応力および接合応力
> **節**: 7・1 応力集中

棒やはりなどの構造部材に引張りや曲げなどの荷重が作用するとき、部材の断面が一様ならば、内部に生じる応力は一様あるいは一次的に分布する。しかし、実際の構造部材には孔(あな、hole)、切欠(notch)、空か(cavity)やピット(pit)、フィレット(fillet)、溝(groove)、き裂(crack)などがあり、これらの近傍では応力は一様に分布せず、局所的な乱れが生じて応力が大きく変化する。例えば、部材に切欠がある場合、切欠から十分離れたところでは応力は一様に分布するが、切欠に近づくにつれて徐々に大きくなり、さらに近づくと応力は急激に増大して切欠底で最大になる。このように切欠の近傍で見られる応力が急激に大きく変化する現象を応力集中(stress concentration)という。応力集中は、切欠などによって部材の幾何学的形状の一様性が失われるために生じる現象であるが、それ以外にも、介在物(inclusion)などの弾性的性質が異なる部分、集中荷重や不連続的に分布する荷重などの荷重の不連続部、さらには温度分布の不連続部などにも生じる。このような応力集中部では、一般に、大きな応力や応力分布のこう配によって微小な割れやボイドが生じてき裂の発生につながる場合が多い。例えば、疲労破壊は、繰返し荷重によって応力集中部に生じたき裂が破壊の起点となって生じることはよく知られている。したがって、疲労設計においては応力集中に十分注意することが必要である。さて、応力集中の度合いを定量的に表すために応力集中係数(stress concentration factor)が用いられる。ここで、基準応力は公称応力(nominal stress)と呼ばれることもあり、応力集中が生じないとして最小断面などに基づいて計算される応力や応力集中部から十分離れた一様応力状態での応力などが用いられることが多いが、そのほかにもいくつかの基準応力の取り方があるので注意が必要である。なお、応力集中係数の値は、二次元均質等方性弾性体の場合、弾性体の形状と荷重様式が与えられると、弾性係数や荷重の大きさにかかわらず一定になるため、αを形状係数(shape factor)と呼ぶこともある。
