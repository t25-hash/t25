# 7・1・2 応力集中の例

> **章**: 第7章 応力集中、接触応力および接合応力
> **節**: 7・1 応力集中

孔や切欠および空かをもつ弾性体の応力集中問題は最も基本的で重要である。ここでは、主として孔や切欠をもつ二次元弾性平板、および空かやピット・円周切欠をもつ三次元弾性体の応力集中について、代表的な例を紹介する。a. 円孔やだ円孔をもつ平板の面内変形(in-plane deformation)。円孔をもつ無限平板の引張り図7・1のように、無限に広い平板(infinite plate)の中に直径2の円孔(circular hole)がある。この平板を軸方向に単位面積当たりσで引っ張る場合、軸上の応力σは次式で与えられる。このような応力集中が生じる理由は、円孔がない場合にAA 断面が均等に負担していた力が、円孔がある場合には、A 点近傍で集中的に負担されるからである。一方、円孔から遠ざかるにつれて応力σの大きさは急速に減少し、十分離れたところでは、σ=σとなって円孔がない場合の一様な応力状態になる。. 円孔をもつ帯板の曲げ同様の帯板に、図7・3(a)のように、大きさの曲げモーメントをその面内で作用させることを面内曲げ(in-plane bending)という。この場合、円孔縁のA 点の応力σは円孔がない場合よりも大きくなるが、必ずしも最大応力にはならない。. 円孔をもつ半無限板の引張り図7・4のように、直線縁から距離の位置に直径2の円孔がある半無限板(semi-infiniteplate)が引張荷重を受けて、遠方で一軸引張応力σの一様な応力状態にあるとする。このとき、軸方向応力に注目すると、円孔縁のA 点の応力σは直線縁の影響を受けて著しく大きくなるが、A′点の応力σの増加はそれほどでもない。また、直線縁のB 点の応(b)(a)(a)(b)図7・3 円孔をもつ帯板の面内曲げ図7・4 円孔をもつ半無限板の引張り力σは円孔が直線縁に近づくにつれて減少する。一方、C点は直線縁で最大応力が生じる点であり、円孔が直線縁に近づくにつれて、C点はB U 型円周切欠をもつ丸棒の引張り・曲げ・ねじり切欠底の曲率半径がρであるU 型円周切欠(U-shaped circumfer-ential notch)をもつ丸棒に引張荷重が作用すると、最大引張応力は切欠底のA 点に生じる。図7・23より、曲率半径が小さいほど、また、切欠が深いほどαの値は大きくなる。一方、同じ丸棒に曲げモーメントやねじりモーメントが作用する場合も、A 点で最大引張応力や最大せん断応力が生じる。f. その他ここでは、最も基本的な弾性体の応力集中問題に限定して、応力集中係数の具体的な例を紹介した。しかし、弾性体の応力集中問題は、円孔やだ円孔以外に正四角形孔・長方形孔・正八角形孔の場合、双曲線形切欠やV 形切欠の場合、球か以外にだ円体状空かや偏長・扁平だ円体状空かの場合、弾性体内部に種々の介在物がある場合、また、孔(あな)や空かが弾性体の中心から偏心して存在する場合、さらにこれらの孔・切欠・空か・介在物などが二つ以上存在する場合など、多岐にわたる。これらの応力集中係数については、例えばノイバーの著書には弾性理論による具体的な解析手法と解析結果が述べられており、また、西田による応力集中ハンドブックには、孔・切欠・空か等の各種の応力集中源の形状と荷重様式に対して、応力集中係数の値が多くの図表にまとめられている。
