# 7・2・1 接触応力概説

> **章**: 第7章 応力集中,接触応力および接合応力
> **節**: 7・2 接触応力

車輪とレール,圧延機のロールと圧延材,歯車の歯面と歯面,転がり軸受の転動体と内外輪などの接触のように,二つの物体を互いに押付けると局部的に変形して接触するようになり,この面に生じる圧縮応力を接触応力(contact stress)という.接触面は一般に小さいため,合力が小さくとも接触応力は局所的に極めて大きくなることがある.また,せん断応力は接触面直下の物体内部で最大になり,この部分で破壊して表面層がはく離することもある.物体の材料定数が著しく異なる場合には一方の物体を剛体とみなすことができ,種々の剛体パンチと弾性体表面との接触応力が求められている.弾性体どうしの接触応力を厳密な数理解析で求めることは困難なことが多く,ヘルツの公式や数値解析が用いられる.
