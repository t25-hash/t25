# 7・2・2 項ではいずれの場合にも:縦弾性係数、ν:ポアソン比

> **章**: 第7章 応力集中、接触応力および接合応力
> **節**: 7・2 接触応力

とし、接触面には摩擦が作用しないものとする。図7・28 球面どうしまたは円柱面どうしの間の接触図7・27 回転放物状剛体パンチによる接触応力3―77となる。なお、摩擦を考慮した場合の剛体パンチを半無限弾性体に押付けた解析例は、文献(7)に紹介されている。b. ヘルツの公式ヘルツの公式(Hertz’s formula)は任意の曲面と曲面の弾性体の接触に適用できるが、ここでは、図7・28に示すような球面と球面、円柱面と円柱面の場合のみを示す。いずれの場合にも曲率半径は、凸面のとき正、凹面のとき負、平面のとき無限大とする。応力特異性(stress singularity)の中で最もよく知られているのは、き裂の場合における1の特異性である(はき裂先端からの距離)。線形破壊力学は、このような応力特異性に着目し、き裂の特異応力場に基づいて強度評価を行うものである。き裂の問題と同様に接合角部(jointed corner)の強度解析においても、線形弾性論による特異応力場の解析結果は有益な情報となる。接合モデルとして最もよく用いられるのは、二つのくさびによる接合角部の問題である。この問題はさらに、くさびの一つの縁のみが接合されるモデル(全平面を形成しない場合)とくさびの二つの縁がともに接合されるモデル(全平面を形成する場合)との二とおりに分けられる。以下では、これらの場合について、応力特異性に関して示す。なお、他の場合についてはハンドブックを参照。
