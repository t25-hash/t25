# 7・3・1 全平面を形成しない二つのくさびによる接合モデル

> **章**: 第7章 応力集中、接触応力および接合応力
> **節**: 7・3 接合応力

図7・29に示すように、二つのくさび(それぞれの横弾性定数、ポアソン比を,νおよび,νとする。また、くさび1の角点の角度をθとして、くさび2の角点の角度をθ-θとする)が一つの縁に沿って接合された角部の問題を考える。a. 面内変形面内変形(plane deformation)に対して特性方程式は、次式で与えられている。また、αとβは、材料1,2の弾性定数から決まる量で、ダンダースの複合パラメータと呼ばれている。特性方程式(7・11)において材料の弾性定数に関係するパラメータがαとβだけであることに注意する必要がある。これは、応力特異性に対する二つの材料の計4個の弾性定数,ν,とνの影響は、二つのパラメータα,βだけで表されることを意味している。このモデルに関して、特にθ=180の場合(すなわち二つのくさびが接合されて半無限板となる場合)およびθ=180,θ=360の場合(すなわち接合界面に沿ってき裂が生じる場合)は応用範囲が広く、多くの解析が行われている。θ=180の場合の応力特異性について、次のような特徴が挙げられる。すなわち、この領域内のα,βに対して、応力特異性が生じない。したがって、二つの90のくさびによる接合半無限板については、その接合角部に応力特異性が生じるかどうかを次のように判断することができる。
