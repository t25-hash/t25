# 7・3・2 全平面を形成する場合

> **章**: 第7章 応力集中、接触応力および接合応力
> **節**: 7・3 接合応力

図7・30に示すような、二つのくさび(くさび1の角点の角度をγとし、くさび2の角点の角度を2π-γとする)が接合して全平面を形成する場合の接合角部、すなわちコーナを有する介在物の角部について考える。a. 面内変形この場合、面内変形の特性方程式は次式で与えられる。逆対称形の変形による応力特異性の指数λは、α,βの全範囲内でその数がせいぜい1個となり、またそれがつねに実数であるが、対称形の変形による応力特異性の指数λについては、それがα,βの値によって実数または複素数となり、多数の根が生じる場合もあ図7・292個のくさびの接合モデル図7・30 2個のくさびの接合モデル(介在物の角部)るなど、逆対称形の場合より複雑である。したがって、面内変形と面外せん断変形による介在物角部の特異応力場については、それぞれを対称形の変形様式と逆対称形の変形様式とに分けることができるが、面外せん断変形による特異応力場では、対称形と逆対称形の特異応力場は同時に生じることができない。材料2の横弾性定数が材料1のそれより小さいとき逆対称形の特異応力場のみが生じる。これに対して、材料2の横弾性定数が材料1のそれより大きいときには対称形の特異応力場のみが生じる。
