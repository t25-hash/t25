# 4・2・3 マルチメディア/マルチモーダルインタフェース

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・2 マルチメディア

a.マルチメディアインタフェース（multimedia interface)
場の提示や，視野の広い三次元視覚ディスプレイ，力覚や触覚，嗅
覚のディスプレイを組み合わせることにより，仮想世界を体験した
り，実世界と仮想世界を重ねたりできるマルチメディアインタフェ
ースが試作されている．バーチャルリアリティ（virtual real-
ity)，オーグメンティッドリアリティ（augmented reality)，
インタラクティブアート（interactive art ）はこのようなインタフ
ェースの究極の応用例といえる．
b.マルチモーダルインタフェース（multimodal interface)
コミュニケーションにおける人間の情報表現は運動器官を通してな
され，音声，身振り，表情，視線などとして表出される．人間どう
しの対面コミュニケーションでは髪型や容ぼう，衣服などもメッセ
ージ伝達に用いられる．音声を除く身体による情報表現媒体は身体
メディア，ノンバーバルメディア（nonverbal media）などと呼ば
れ，キーボード，マウス，スライダなどの接触型インタフェースと
異なって，人間にとって自然であり，直感的に扱うことができ
る ．この場合の情報は身体メディアごとに異なるモード
［mode，またはモダリティ（modality)，表現様式］をもち，人間
は同時に多数のモードを用いて情報を表出できる特性をもってい
る．この特性を人間とシステムの対話に適用して，多様なモードを
通して人間がシステムに情報や操作を伝えるためのインタフェース
をマルチモーダルインタフェースと呼ぶ ．マルチメディアインタ
フェースが主としてシステムから人間への情報提示に用いられるの
に対して，マルチモーダルインタフェースは通常人間からシステム
への情報伝達に使われる．ノンバーバルメディアが人間どうしのコ
ミュニケーションで果たす役割は広範囲に及んでおり ，コンピュ
ータをはじめとする機械と人間のコミュニケーションにおいてもマ
ルチモーダルインタフェースを通じてそれらの機能が活用できると
期待されている ．
世界で最初に実現されたマルチモーダルインタフェースは，1970
年代末にMITメディアラボで作られた「それをあそこに（Put
That There）」（図4･2）で，音声と指差しを用いてディスプレイ
上の図形を操作するシステムであった ．現在は，音声，身振り，
姿勢，視線，うなずきの組合せがよく使われ ，これにキーボ
ードやマウスが追加されることもある．
ヒューマンインタフェース（human interface）の一種に擬人化
エージェント（personified agent ）がある ．これはユーザまたは
システムの代理として機能するソフトウェアを人物に似せて作り，
画面上に表示したもので，人間とシステムの対話を容易にする目的
をもっている．また物理的，心理的に遠隔の相手との対話を自然に
するために，相手の化身であるアバタ（avatar ）を画面上に提示
する仮想コミュニケーション環境も構築されている ．この場合，
目前のアバタとの対話はそのまま遠隔の相手とのコミュニケーショ
ンとなるわけである．これらエージェントやアバタは人間に似た容
姿をもち，またマルチモーダルなふるまいをすることが望ましい．
この場合にはシステムから人間への情報提示にもマルチモーダルイ
ンタフェースが活用できる．図4･3は手指動作に加えて，表情，ま
ばたき，うなずきを交えた手話表現をするアバタの例で，容ぼう，
衣服などへの配慮も要求される．
