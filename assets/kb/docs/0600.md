# 8・1・2 トラスの解析

> **章**: 第8章 構造力学
> **節**: 8・1 骨組構造

a. 応力法図8・1に示すトラスを考えよう。解析の手順は(1) 不静定次数の決定。(2) 不静定次数に相当する数の適当な部材を選び、その部材を切断する。この操作を静定化という。(4) 静定化した際に切断した部材に実際に作用する不静定力を図8・1 不静定平面トラス,...,とする。(7) に関する元連立一次方程式(8・5)を解くと不静定力が定まるので、これを式(8・2)に代入してすべての軸力が計算できる。図8・1のトラスの場合、不静定次数=1である。節点2において部材5を切断し、この不静定力をとする。b. 変位法各部材が一様断面をもつ平面トラスを考え、荷重は節点のみに作用するものとする。解析の手順は以下のとおりである。(1) 全部材に部材番号1,2,...,◯を付す。ここで、節点変位の自由度の総数を、拘束される変位成分の総数をとする。このとき、部材レベルの自由度番号1,2,3,4と全体レベルの自由度番号1,2,1,2との対応は部材の構造全体への結合(connectivity)を表す重要な対応関係である。これを解けば未知変位成分,...,が定まり、骨組の全体変形が確定する。さらに、得られた変位を式(8・16)に代入すると部材の軸力が定まる。上述の(1)~(8)の手続きにより記述される変位法による骨組解析は、コンピュータによるプログラム化が容易であり有限要素法による構造解析の最も簡単な例とみなすことができる。
