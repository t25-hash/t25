# 8・1・3 ラーメンの解析

> **章**: 第8章 構造力学
> **節**: 8・1 骨組構造

a. 応力法図8・3に示すラーメンを考えよう。図8・3のように各部材の長手方向に局部座標系をとる。解析の手順はトラスの場合と同様であるが、ラーメンの場合の内力成分はせん断力と曲げモーメントになる。解析手順は以下のとおりである。(1) 不静定次数の決定。(2) 不静定次数に相当する数の余剰な拘束を解放するような切断をして、骨組を静定化する。(4) 静定化した際に切断した部材に実際に作用する不静定量を,...,とする。不静定量,...,が定まれば、これを式(8・24)に代入して各部材の曲げモーメントが定まる。図8・4(a)の例題の場合、不静定次数は1であるから、節点4の上下方向変位の拘束を解放し、その不静定力をとする。節点の自由度は変位2成分と回転角の計3成分であり、対応する外力成分も力2成分とモーメン(a)(b)ト1成分の計3成分である。解析の手順は以下のとおりである。(1) 全部材に部材番号1,2,...,◯を付す。ここで、節点変位の自由度の総数を、拘束される変位成分の総数をとする。さらに、得られた変位を式(8・34)により局部座標系の成分に変換し、さらに式(8・37)に代入すると部材の内力が定まる。c. 直接部材に作用する荷重ここまでは力やモーメントなどの外荷重が節点のみに作用するとして問題を扱ってきたが、実際の構造では分布荷重など部材に直接外力が作用することも多い。ラーメン部材に直接分布荷重や集中荷重が作用する場合には、まず当該部材の両端を仮に固着し、両端固着はりとしての部材力および固着端反力と反モーメントを計算する。いま、これを解aとしよう。次に、この反力および反モーメントと大きさが等しく反対向きの力およびモーメントを当該節点に作用させた場合の骨組構造の解を解3―83bとする。解aと解bの和を計算すると、直接部材に荷重が作用し、かつ固着端反力も打消され、解くべき問題の解が得られる。トラスの場合には、前述の解aを求めるに際して両端単純支持とすればよい。
