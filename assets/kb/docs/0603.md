# 8・2・1 せん断場理論

> **章**: 第8章 構造力学
> **節**: 8・2 薄板構造

薄板を使った構造では集中荷重の構造全体への伝達など薄板のせん断剛性が無限大とは考えられないことによる問題が生じる。この近似的解明にはせん断場理論が使われる。この理論は補強材(フランジ等)は軸力のみを、薄板(ウェブ)はせん断力のみを負担するという考えに基づいている。以下ではせん断場理論の説明を行う。a. 静定補強板の例1図8・12に示す補強板にせん断場理論を適用する。フランジは軸力のみ、薄板のウェブ(板厚)はせん断力のみを負担すると考える。ウェブは純粋せん断場となり、一様せん断応力をτ、板厚をとして≡τなるせん断流(shear flow)を定義する。このときの補強材の軸力分布を図8・13に示す。計算結果を図8・14(b)(c)に示す。この図は上下部材の端末でかけた軸力が中央の部材に伝わるまでにはある距離が必要なことを示している。この現象をせん断遅れ(shear lag)と呼ぶ。d. 連続外板補強板図8・15のように端末を除いて横材がない場合は、薄板全面にわたりせん断流一定と考えることはできない。図8・14(a)の問題で横材を取り去った場合についての計算結果を図8・14(b)(c)に一点鎖線で示した。
