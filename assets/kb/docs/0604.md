# 8・2・2 張力場理論

> **章**: 第8章 構造力学
> **節**: 8・2 薄板構造

薄板ウェブ、フランジ、支柱からなる図8・16のようなプレートガーダにせん断荷重が作用する場合を考える。薄板が座屈するまでは、薄板に働く一様なせん断応力(純粋せん断場)はフランジと45°および135°の方向に働く引張応力と圧縮応力とに分解することができる。せん断荷重が増大し、薄板がせん断座屈(shearbuckling)すると斜に座屈波(しわ)が発生する。しかし、この構造は座屈後も依然耐荷能力がある。この耐荷能力は主としてしわの方向の剛性によるものである。せん断荷重が座屈荷重より大幅に大きいときは薄板の圧縮方向(しわと直角方向)の剛性=0と近似的に考えることができる。圧縮方向の剛性を零としない理論を不完全張力場の理論といい、せん断座屈直後の解析に有効である。a. 完全張力場図8・16は座屈波がαの方向に発生していることを示している。せん断力は全部薄板のしわが取ると考えると、しわの方向の引張応力σは次式で与えられる。フランジおよび支柱の軸力,は次式で与えられる。フランジにはσによる分布荷重がかかり、フランジの曲げモーメントは支柱位置で最大となり、次式で与えられる。
