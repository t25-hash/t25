# 8・3・4 開断面薄肉はりの曲げねじり

> **章**: 第8章 構造力学
> **節**: 8・17 に示すようなはりの両端のみにねじりモーメント

せん断中心点S まわりにはりの各断面が回転すると仮定する。ねじりモーメントが軸方向に分布し、あるいは端部で断面の反りが拘束されていて、θ′が各断面で変化するとき、サンブナンのねじりによるせん断応力以外に、付加的なせん断応力が生じる。このために付加的なねじりモーメントが生じるが、これを曲げねじりモーメント(warping torque)と呼ぶ。薄肉はりの場合、曲げねじりモーメントは次のように求められる。閉断面ではねじり剛性が相対的に曲げ剛性より大きく、このとき、曲げ剛性はあまり重要ではない。次式によってバイモーメント(bimoment)を定義する。
