# 8・4・1 殻理論の基礎と分類

> **章**: 第8章 構造力学
> **節**: 8・4 殻構造

中央面が曲面で厚さが曲率半径および面内寸法[正確には応力、ひずみの中央面上の波長(wave length)]に比べて十分に小さい固体を殻(shell)という。殻を三次元理論で扱うと一般に解析が難しくなる。そこで簡易化して中央面の変位を変数とする二次元理論が用いられる。簡易の程度に応じて種々の殻理論がある。以下フックの法則に従う弾性殻理論について述べる。a. キルヒホッフ・ラブ(K-L)の仮定K-Lの仮定によれば、1変位-ひずみ関係の誘導において、変形前の中央面に垂直な線要素は変形後も垂直で長さが不変である。2構成方程式の誘導において、殻厚方向の垂直応力成分を無視する。ラブの一次近似理論(first approximation theory for shells)によれば、ひずみエネルギーが、中央面のひずみによるエネルギーと曲げひずみによる、エネルギーの和で与えられる。ラブの一次近似理論は、三次元弾性体の変形の近似理論として合理的(consis-tent)であり、K-Lの仮定(または等価な仮定)から導くことができる。二次近似理論(second approximation theory for shells)はさらに高次の項を留保しており、ひずみエネルギーの表示に中央面のひずみ成分と曲げひずみ成分の積の項が現れる。二次近似理論にはフリュゲ・ルレ・ビルン(Flugge-Lure-Byrne)の式など多くの式が提案されている。ただし二次近似理論におけるK-Lの仮定の妥当性についても問題があり、合理性は十分に明らかにされたとはいえない。ほかに、殻厚方向のせん断変形を考慮したK-Lの仮定に拘束されない理論もあるが、ここでは述べない。b. たわみの大小および応力の様相による分類中央面のたわみが殻厚に比べて小さいとし、変位-ひずみ関係で一次線形項のみを考慮した(i)線形理論と、中央面のたわみが殻厚と同程度の変形まで有効であるようにたわみの二次の項まで考慮した(ii)非線形理論がある。殻の面内応力(membrane stress)[膜応力(membranestress)]と曲げ応力(bending stress)の大小に応じて殻理論は、膜理論(曲げ応力が無視できるように負荷、支持され、面内応力のみによる変形理論)と曲げ理論(面内応力とともに曲げ応力が大きい場合の変形理論)に分類される。このほかに、曲げ応力に比べ面3―89内応力が無視できる変形を対象とした伸びなし変形理論がある。線形理論(linear theory for shells)による殻の解析を行う場合、図8・23に示すように膜理論による解を求めておき、曲げ理論による補正を行うことによって境界条件を満たす方法がしばしば用いられる。
