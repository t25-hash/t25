# 8・4・4 殻の非線形理論

> **章**: 第8章 構造力学
> **節**: 8・4 殻構造

線形殻理論においてたわみが殻厚に比べて小さいという仮説を除外すると、変位-ひずみ関係は非線形となり、また平衡方程式に曲率変化による面内力の分力が現れ、幾何学的非線形問題となる。弾性殻の一次近似非線形理論(non-linear the-ory for shells)の式を以下に示す。式(8・66)の面内の平衡方程式を満足するように次の応力関数を導入する。式(8・68)(8・69)の非線形連立偏微分方程式を解析的に解くことは困難であり、通常近似解法が用いられる。例えば境界条件を満足するようにたわみを仮定し、を求めた後にガレルキン(Galerkin)法を適用するか、エネルギー停留原理で解いて非線形平衡曲線を求める解法、級数展開による解法などがある。また、有限要素法による数値解法も有用である。殻の非線形二次元理論の合理性の評価を三次元固体の変形に基づいて初めて系統的に論じたのがF. ジョン(John)であり、一次近似(最低次)の方程式を誘導している。それらは式(8・67)~(8・69)に等価である。式(8・67)~(8・69)の合理性が示されたわけであるが、あくまでも近似理論としてであり、この近似の程度を越える形状、荷重条件ないし変形量などに対して妥当であるということはできない。
