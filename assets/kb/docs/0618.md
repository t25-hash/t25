# 9・2・1 応力とひずみ

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

材料試験では慣例として次式で定義される公称応力σ[nominal (or conventional)stress]と公称ひずみε[nominal (or conventional) strain]を用いて応力-ひずみ曲線を描くことが多い。この長さがとなった棒を引張り、さらにdだけ伸ばしたとする。このときのひずみ増分はdε=dとなるので、これを積分して次のようなひずみεが得られる。材料の塑性的性質を正確に表すためには(公称応力-公称ひずみ曲線ではなく)真応力-真ひずみ曲線によらねばならない。
