# 4・10・8 電磁流体（magnetohydrodynamic flow）の対流熱伝

> **章**: 第4章 伝熱（物質移動を含む）
> **節**: 4・10 その他の重要な対流熱伝達

a.特徴 電磁流体には，水銀や液体ナトリウムなどの液体金
(ｂ)熱伝導流束
図4･51管内反応乱流におけるヌセルト数とレイノルズ数の関係
ここでは主として連続流体として扱えるプラズマを対象とする．
プラズマ中には電子やイオンが存在しているので，プラズマ中に
おける伝熱現象は通常の伝熱現象とは異なった以下のような特徴を
有する．
(１)熱プラズマは高温であるので，伝熱量が非常に大きく，さ
らに放射による伝熱量が無視できなくなる．
(２)プラズマ中に存在する電子やイオンが大きく関与している
ので，化学反応を伴う反応性流体と同様に，解離エネルギーの輸送
を考慮する必要がある．高温領域で電離して生成した電子とイオン
が拡散によって低温領域へ移動し再結合すると，高温領域で得た電
離エネルギーを低温領域で放出することになる．つまり，電子とイ
オンのペアによって電離エネルギーが輸送される．プラズマ伝熱で
は電離エネルギーによる機構が大きな割合を占める．
(３)電子やイオンは電磁場の影響を受けるので，外部からの電
磁場によって伝熱量をプロセスに応じて制御することができる．電
磁場がある場合の伝熱現象は，プラズマの流れの場が大きく影響を
受けることから複雑になる．
(４)荷電粒子はプラズマの物性値に対して大きな影響を与え
る．荷電粒子間の衝突であるクーロン衝突は遠い距離でも起こるの
で，中性粒子間の衝突とは大きく異なる．クーロン衝突は極めて起
こりやすいので，粘性係数等の輸送係数は小さくなる．アルゴンの
粘性係数は温度に対して上昇するが，電離が起こる温度から減少す
るのはこのためである．これとは逆に，電子の熱運動速度が大きい
ために，電子が関与する熱伝導率等の輸送係数は大きくなる．ただ
し，粘性係数に対しては電子の運動量が小さいのでほとんど寄与し
ない．
b.電磁場のない場合 プラズマにおいてはイオンと電子の電
離・再結合反応があることが特徴で，化学反応を伴う熱伝達と類似
に扱うことができる．高温領域で電離して生成したイオンと電子は
低温領域で再結合し，電離エネルギーを放出するので，通常の流体
の伝熱機構に加えて，イオンと電子のペアの両極性拡散によるエネ
ルギー移動がある．
電磁場のない場合の伝熱量は，一般の流体と同じようにヌセルト
数を用いて推算することができる．しかし，プラズマ中の球の境界
層内では物性値が大きく変化するので，一般の流体における熱伝達
率（伝熱係数）の推算式をそのまま用いることはできない．さまざ
まな熱伝達率の推算式が提案されているが，式(4･392)がよく用い

d.電磁場のある場合 電場に加えて磁場が加わると伝熱現象
はさらに複雑になるが，電磁場が存在するプラズマの挙動は電磁流
体力学の分野で詳細な研究が行われている．磁場があるとプラズマ
の物性値が磁場に垂直方向と平行方向とで異なり，非等方性にな
る．磁場に垂直方向の輸送係数は平行方向に比べて減少する．一般
に磁場によって荷電粒子の運動が抑制されるので，伝熱量は減少す
る．
