# 9・2・3 結晶の塑性変形と転位

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

a. 理想的すべり強度と転位すべり変形は結晶学的に定まった特定の結晶面(すべり面,slip plane)上で特定の結晶方向(すべり方向,slip direction)にのみ起こる.なお,すべり面とすべり方向を合わせた組をすべり系(slip system)と呼ぶ.完全結晶にせん断応力が作用して,すべり面を挟んだ上下の結晶が相対的に同時にずれると仮定する.このときに必要な最大せん断応力の値(理想的すべり強度)を計算してみると,これは実際の材料の降伏応力の10~10倍にもなってしまう.このような理想的すべり強度と実際の降伏応力の大きな差は,結晶中に存在する原子配列の乱れである転位(dislocation)の存在とその運動により説明される.図9・4(a)~(d)は,刃状転位(edge dislocation)がせん断応力τの作用によって移動し,最終的にすべり面の上下で1原子間隔分のずれを生じる様子を模式的に描いてある.すべり面の上半分に存在する余分な原子面は図9・4(a)で最初は位置4にあるが,転位の周辺の格子はひずんでいる(内部応力が存在する)ので,せん断応力作用により容易に隣の位置5に移ることができる[図9・4(b)].こうした過程が連続することによって塑性すべりが実現す(b)図9・3 種々の金属材料の応力-ひずみ曲線図9・5 単結晶体の引張りにおけるすべり面とすべり方向る[図9・4(c)(d)].このときのせん断応力の値は,転位近傍の原子の局所的なわずかな移動に必要なものであるから,理想的すべり強度よりけた違いに小さくなり,実験で観察されるものにほぼ等しくなる.b. 塑性変形の非圧縮性塑性変形のおもな機構はa項で述べたようなすべり変形であるので,塑性変形による体積変化はない.塑性変形の非圧縮性は次式で表される.ε+ε+ε욨욨=0(9・8)c. ひずみ硬化と転位実際の材料中の転位の運動に対してはさまざまな障害物(例えば,他の転位との弾性相互作用,結晶粒界,析出物,林立転位の切合い)が変形抵抗として作用する.ひずみ硬化は塑性変形による転位の増殖(代表的な増殖機構は,ピン止めされた転位の引張出しによる転位ループの形成とその増殖によるフランク・リード機構)と転位どうしの相互作用によって説明される.すなわち,変形とともに転位密度が増大すると転位どうしが互いの運動を妨害し合うようになり,それがひずみ硬化の原因となる.一般に,せん断変形抵抗τと転位密度ρの関係は次式で与えられる.τ=τ욉+αρ(9・9)ここでτ욉,α=0.3~0.6,=せん断弾性係数)は材料定数,はバーガースベクトルの大きさである.
