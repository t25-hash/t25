# 9・2・5 集合組織と異方性

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

多結晶体から種々の方向に試験片を切出して引張試験を行うとしよう.もしこの材料中の個々の結晶粒の方位が全くランダムであるならば,応力-ひずみ曲線は試験片の切出し方向によらず同一となるはずであり,このような材料は等方性材料といわれる.一方,結晶方位の分布が特定の方向に偏っている場合に,これを集合組織(texture)が形成されているという.これはシュミット因子の分布の偏りを意味し,降伏強さ(変形抵抗)に方向性(異方性,anisotropy)が生じる.板材の圧延再結晶集合組織はその代表的なものである.こうした異方性材料の板の単軸引張りでは,引張軸に直交する方向のひずみ(板幅ひずみε욌)と板厚方向のひずみ(板厚ひずみε욧)は等しくならない.次式で定義される値(またはランクフォード値,Lankford value)は板材の異方性を表す重要なパラメータである(等方性材料では=1).=ε욌ε욧(9・12)一般には値の高い板材ほど深絞り成形性が良いので,これは成形性の良否の尺度としても用いられる.ところで,集合組織による異方性とは別に,塑性変形によって誘起される異方性があり,その代表的なものはバウシンガー効果である.多結晶体のバウシンガー効果のおもな原因は,ある方向の応力作用によって個々の結晶粒に生じた内部応力の存在がその後の応力反転における巨視的な降伏応力を低くすることであるといえる.また,転位の運動に対して方向性をもつ障害物の存在によってこの現象を説明することもできる.すなわち,ある方向の応力作用によって粒界に堆積(pile-up)したり,微細析出物等によってピン止めされた転位は応力反転作用により,容易に可動性を回復するので反転側での降伏応力が低くなる.
