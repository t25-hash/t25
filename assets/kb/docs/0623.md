# 9・2・6 繰返し硬化・軟化特性とラチェット変形

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

繰返し塑性変形(cyclic plasticity)において応力振幅が繰返し数とともに大きくなる現象を繰返し硬化(cyclic hardening),徐々に小さくなる場合を繰返し軟化(cyclic softening)という.繰返し硬化・軟化特性は材料ごとに大きく異なるとともに,繰返し変形条件(繰返しひずみ幅,平均ひずみなど)にも強く依存する.繰返し応力作用により繰返し数とともにひずみが一方向に累積する現象はラチェット変形(ratchetting)と呼ばれる.これには,単軸応力の繰返しにおいて平均応力方向にひずみが累積する単軸ラチェット(図9・6),応力の多軸性に起因する多軸ラチェットがある.単軸ラチェットは主として正・負応力方向でひずみ硬化(バ3―9 9図9・6 ステンレス鋼(SUS 304)の単軸ラチェット変形挙動図9・7 炭素鋼(0.1%C-0.2%Si-1.1%Mn)の応力-ひずみ曲線に及ぼす温度の影響ウシンガー効果)の挙動がわずかに異なることに起因し,材料や温度条件によっては粘塑性変形(クリープ変形)も加わって生じる.熱応力の繰返しによって起こる場合をサーマルラチェット(ther-mal ratchetting)と呼ぶ.構造物に繰返し荷重が作用するときに生じるラチェット変形は,繰返し(粘)塑性特性に起因するものとあわせて,繰返しごとに構造物内の応力分布が変化することによっても生じる.なお,ある繰返し数でラチェット変形が止まる現象はシェイクダウン(shakedown)と呼ばれる.
