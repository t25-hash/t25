# 9・2・7 変形抵抗の温度およびひずみ速度依存性

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

塑性変形は一般には速度依存性が小さいとされるが,転位の運動速度は作用する応力に依存するため,室温でもある程度の速度依存性が見られるのが普通である.転位の運動は本質的に熱活性化過程(応力作用とともに原子の熱振動エネルギーの助けを借りて,転位が障害物を乗り越えて進む過程)に支配されるため,変形抵抗は一般に温度およびひずみ速度に依存する.高温においては粒界すべり(粘性的な変形挙動を示す)が活発になる場合も多く,ある温度以上では変形中の回復や再結晶(これらは動的復旧過程といわれる)も影響するため温度・速度依存性がさらに顕著となる.図9・7には変形抵抗の温度依存性を,図9・8にはひずみ速度依存性を示す軟鋼の応力-ひずみ曲線の例を描いてある.変形抵抗の温度・ひずみ速度依存性は,塑性ひずみ速度(あるいは非弾性ひずみ速度)εが応力σと温度に依存する現象ということもでき,一般的には次のような形で表現することができる.ε=ε욉σ,ξexp-(9・13)ε욉σ,ξは応力σと変形履歴に依存する内部状態変数ξ(一般には複数個)の関数である.はガス定数であり,は活性化エネルギーと呼ばれる(変形機構に支配される)材料定数である.ここで,≡εexpで定義されるゼナー・ホロモンパラメータ(Zener-Hollomon parameter)を用いると,式(9・13)より変形抵抗σはパラメータと内部状態変数ξの関数としてσ=σ,ξ(9・14)3―100材料力学図9・8炭素鋼(0.35%C-2.9%Si-0.25%Mn)の600°Cにおける応力-ひずみ曲線に及ぼすひずみ速度の影響のように表せる.
