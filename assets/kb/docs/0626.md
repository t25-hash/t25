# 9・3・2 完全塑性体

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・3 塑性構成式(plastic constitutive equation)

初期降伏点を超えて負荷すると、一般にひずみ硬化(strain har-dening)により応力は増加する。しかし、ひずみ硬化が無視できる程度であれば、応力一定で塑性変形が進展するような材料モデルを考えることができる(図9・11)。このような材料モデルは完全塑性体(perfectly plastic material)と呼ばれ、弾性ひずみの有無によって弾完全塑性体(elastic-perfectly plastic material)と剛完全塑性体(rigid-perfectly plastic material)に分類される。完全塑性体では、降伏後も応力状態は初期降伏条件を満足し続ける。(a) 弾完全塑性体(b) 剛完全塑性体図9・11完全塑性体の単軸引張応力-ひずみ曲線図9・12 再負荷と逆負荷
