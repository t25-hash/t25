# 9・3・3 後続降伏曲面

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・3 塑性構成式(plastic constitutive equation)

ひずみ硬化する場合には、塑性変形の進行とともに応力は増加する。このように負荷した後、除荷し、続けて再負荷する場合(図合は、除荷開始点の近くで再降伏するので、再降伏点の応力σ′は初期降伏応力σより大きくなる。一方、逆負荷のもとでの再降伏応力σ′′の大きさは、初期降伏応力σより小さくなる傾向にある。これは、9・2・2項で述べたようにバウシンガー効果として知られる現象であり、降伏曲面が塑性変形の進行とともに変化することを意味する(図9・9)。塑性変形が生じた後の降伏曲面は、後続降伏曲面(subsequent yieldsurface)と呼ばれ、一般に塑性変形の履歴に依存する。変数ξ,ξ,...,ξは、負荷履歴パラメータ(load-ing history parameter)、内部状態変数(internal state variable)あるいは内部変数(internal variable)と呼ばれ、スカラあるいはテンソル量である。
