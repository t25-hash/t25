# 9・3・4 最大塑性仕事の原理と流れ則

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 のO′A)と逆負荷する場合(O′B)を比較する。再負荷の場

安定な材料において応力を任意の状態から変化させ、塑性変形を起こさせた後、応力を元の状態に戻すサイクルを考える。このとき、そのような変化を引き起こす原因となった外力のなす仕事は非負となるはずである。これは、ドラッカーの仮説(Drucker’shypothesis)と呼ばれる。このとき正味仕事は次のようになる。なお、結晶塑性理論でシュミット則(Schmid rule)が成立する場合には最大塑性仕事の原理が成り立つことが示される。このことから、降伏曲面は凸であること、また降伏曲面が滑らかであれば、塑性ひずみ速度は次式のように降伏曲面の法線方向に生じることが示される(図9・13)。この流れ則(flow rule)は連合流れ則(associatedflow rule)とも呼ばれる。また、塑性ひずみ速度が式(9・28)の形に書かれる塑性構成式は流れ理論(flow theory)といい、その降伏関数は塑性ポテンシャル(plastic potential)と呼ばれる。式(9・28)のλは、次のように求められる。塑性変形の進行に伴って内部状態変数ξは変化するから、ξの時間変化率ξはεの一次の同次関数となる。この式は、プラガー(Prager)の適応条件(consistency condition)と呼ばれる。
