# 9・3・7 変形理論(deformation theory)

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 の破線のように予測され,バウシンガー効果が表されない.

式(9・28)の形の塑性構成式は,すでに述べたように流れ理論(flow theory)と呼ばれ,徐荷や中立負荷が生じる場合にも有効で(a)(b)図9・16 ガーソンの降伏関数による降伏曲面図9・15 移動硬化理論による単軸引張応力-ひずみ曲線(徐荷,逆負荷,再負荷を含む)3―103ある.負荷が比例的あるいはほぼ比例的でかつ単調に強くなる場合には,徐荷や中立負荷は生じない.このような場合の塑性変形に対しては,塑性ひずみを応力の非線形関数によって表す変形理論(deformation theory)がしばしば用いられる.相当応力と相当塑性ひずみをそれぞれσ=3/2욡욢욡욢,ε=2/3ε욡욢ε욡욢と定義し,単軸引張負荷における応力と塑性ひずみの関係がε=σと表されるとすると,変形理論による塑性構成式はε욡욢=32σ욡욢σ(9・52)と書ける.変形理論は全ひずみ理論(total strain theory)とも呼ばれる.
