# 9・3・8 静水圧依存の構成式

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 の破線のように予測され、バウシンガー効果が表されない。

金属材料の塑性変形は、通常、非圧縮であるため応力の静水圧成分σ/3に依存しないと仮定される。しかし、金属材料が塑性変形を受け、内部に微小なボイドが発生すると、その巨視的塑性変形は、非圧縮でなくなり静水圧成分σ3に依存するようになる。ガーソン(Gurson)は、このような場合の巨視的塑性構成式を定式化するため、まずボイド1個を含む単位セルの剛塑性極限解析を行うことによって、ボイドを含む連続体としての降伏関数を次式のように表した。上式の第2項が静水圧依存性を表す項であり、これによって降伏条件は静水圧とともに図9・16に示すように変化する。土壌や岩石などのように間げきやき裂を含む材料では、内部摩擦や静水圧が塑性変形に大きく影響する。ルドゥニッキ・ライス(Rudnicki-Rice)は、このような場合の塑性構成式を検討し、次式を導いた。上式では、塑性ひずみ速度の方向と降伏曲面の法線方向は一般に一致しない。このような流れ則は非連合流れ則(non-associated flowrule)と呼ばれる。3―104材料力学図9・17 降伏曲面の尖り点の発達
