# 9・3・10 粘塑性構成式(viscoplastic constitutive equation)

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 の破線のように予測され,バウシンガー効果が表されない.

高速変形では,通常,ひずみ速度依存性が著しいため,9・3・1~9・3・9項で述べた速度非依存の塑性構成式が適さないことが多い.高速変形でのひずみ速度依存性を表現できる粘塑性構成式として,マルバーン(Malvern)やペジュナ(Perzyna)が提案したモデルがある.このモデルでは,準静的変形での応力-ひずみ曲線はひずみ速度非依存の塑性構成式により表されると考え,応力状態が準静的状態を超えると,粘塑性ひずみ速度が生じると仮定される.ひずみ速度非依存の塑性構成式として,まず等方硬化理論[式(9・38)]を考えると,準静的変形ではσ-ε=0(9・56)が成り立つ.したがって,等方硬化理論に基づく粘塑性構成式はε욡욢=32σ-ε욡욢σ(9・57)と書ける.ここでは=,≧0<0(9・58)0,と定義される.次に,複合硬化理論[式(9・50)]を考えると,準静的変形ではσ윬읆읆-ε=0(9・59)が成り立つ.ここでσ윬읆읆は,有効応力を表し,σ윬읆읆=32욡욢-욡욢욡욢-욡욢(9・60)と定義される.したがって,複合硬化理論に基づく粘塑性構成式はε욡욢=32σ윬읆읆-ε욡욢-욡욢σ윬읆읆(9・61)と書ける.
