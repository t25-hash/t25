# 9・4・2 極限解析の上下界定理

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・4 塑性の一般的諸定理

一般に非硬化弾塑性体では,降伏応力が一定であるから,物体が支えうる荷重には限界があり(極限荷重),その荷重のもとでは変形が進行していく(塑性崩壊)状態となる.塑性崩壊状態,極限荷重を求める解析を極限解析(limit analysis)という.極限解析は,弾塑性解析に比べて簡単に最終状態の評価を行うことができ,塑性加工の解析などに用いられる.極限解析は上下界定理を用いて行われる.物体の表面上で表面力が,表面욊上で速度がそれぞれ境界条件として与えられているとする.力学的境界条件と物体内で応力の平衡方程式を満たし,かつ降伏条件を破らない応力場σ(可容応力場)を考える.可容応力場に対して得られる速度境界上の表面力をとする.また,速度境界条件を満たし,物体内で体積一定の条件を満足する速度場を욘(可容速度場)とし,これより得られるひずみ速度場をε욘,構成関係によってε욘より求められる応力をσ욘とそれぞれ記する.仮想仕事の原理と最大塑性仕事の原理を用いると,正解の表面力,速度と,これらの量の間には次の不等式が成り立つことを示すことができる.d≦d≦σ욘ε욘d-욘d上式を剛塑性体の上下界定理という.物体中に速度の不連続面Γが存在する場合には,不連続面での接線方向不連続速度をΔとすれば上式は次の形となる.d≦d≦σ욘ε욘d+욘ΔdΓ-욘d物体中に種々の速度不連続面Γを含む可容速度場욘を仮定することで,上式の右側不等式から上界荷重を求めることができ,適当な可容応力場を仮定することにより左側不等式から下界荷重を求めることができる.可容速度場から求められるひずみ速度,応力場が,可容応力場から求められるものと互いに一致するとき,正解が得られ,下界荷重,上界荷重は正解荷重と一致する.
