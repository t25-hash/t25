# 9・5・1 定義

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・5 すべり線場の理論

塑性変形状態にある物体の各点において,その接線が最大せん断応力方向と一致する曲線をすべり線(slip line)と呼ぶ.最大せん断応力線,せん断線(shear line)とも呼ばれる.代数的に最大の主応力の方向から時計方向に45°の位置の最大せん断応力の方向を第一せん断方向(α方向),反時計方向に45°の位置の最大せん断応力の方向を第二せん断方向(β方向)と呼び,対応するすべり線をそれぞれ第一せん断線(αすべり線,α線),第二せん断線(βすべり線,β線)といい,お互いに直行する曲線群をなす(図9・18).剛完全等方塑性体の平面ひずみ問題は双曲型の偏微分方程式に支配され,応力場と速度場の特性曲線が一致する.この特性曲線がすべり線である.
