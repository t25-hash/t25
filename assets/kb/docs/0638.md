# 9・5・2 平面問題のヘンキーの式

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・5 すべり線場の理論

剛完全等方塑性体の平面ひずみ問題において、応力場の支配方程式(応力の釣合い式と降伏条件)の特性曲線であるすべり線に沿って、平均圧縮応力と降伏応力に関して成り立つ関係式をヘンキー(Henkey)の式という。平均圧縮応力を=-σ+σ/2、降伏応力を(一定値)、αすべり線と軸のなす角をφとするとき、すべり線に沿ってαすべり線上+2φ=一定βすべり線上-2φ=一定として表される。これより、境界上で,φの値が既知であれば、境界を通るすべり線上での,φを求めることができる。αβ座標系での応力成分はσ=σ=-,τ=であるので、座標変換によって座標系の応力成分はすべり線上でαすべり線上σ+sin2φ-2φ=一定、τ=cos 2φβすべり線上σ+sin2φ+2φ=一定、τ=cos 2φとなる。
