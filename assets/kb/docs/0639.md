# 9・5・3 ガイリンガーの式とホドグラフ

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・5 すべり線場の理論

剛完全等方塑性体の平面ひずみ問題では、速度場の支配方程式(ひずみ速度-変位速度関係と構成式)の特性曲線は、応力場の特性曲線(すべり線)と一致する。すべり線に沿う速度成分の間に成り立つ関係式をガイリンガー(Geiringer)の式という。αすべり線、βすべり線方向の速度成分をそれぞれ,とし、αすべり線と軸のなす角をφとするとき、すべり線に沿う微分関係αすべり線上d-dφ=0βすべり線上d+dφ=0として表される。ある点の速度,を、,を座標軸とする座標面(ホドグラフ面)に束縛ベクトルとして表示して得られる線図をホドグラフという(図9・19)。ガイリンガーの式により、αおよびβ線上の速度を写像して得られるα′およびβ′線はそれぞれαおよびβに直交する。ホドグラフは、塑性状態における各点の運動を、一見して知るうえで便利である。
