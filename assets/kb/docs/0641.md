# 9・6・1 残留応力の分類

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・6 残留応力

残留応力(residual stress)とは、塑性変形、鋳造、熱処理、溶接などにより材料内部に生じ残留する応力のことである。不均一な機械的あるいは熱的負荷が原因となり発生する残留応力を巨視的残留応力と呼ぶ。一方、材料の組織の不均一が原因で発生する残留応力を微視的残留応力と呼ぶ。a. 巨視的残留応力(macroscopic residual stress)巨視的残留応力が発生する例として、はりの曲げを取上げる。図9・20(a)に示す応力-ひずみ関係をもつ弾完全塑性体のはりに、図9・20(b)に示すような曲げモーメントによる曲げを与え、中立軸-′から距離±より表面側が降伏したとすると、図9・20(b)の斜線部で示す応力分布が生じる。その後、図9・20(c)に示すように、曲げモーメントを取り除く。つまり、-の曲げモーメントで曲げ戻すと、このときは塑性変形が生じないので、図9・20(c)の網掛部で示す応力分布が生じる。すなわち、図9・20(b)の応力分布と図9・20(c)の応力分布を重ね合わせることにより、図9・20(d)のような応力がはり内部に残留することになる。これは、巨視的残留応力のよく知られた例である。巨視的残留応力は、溶接により材料内部に温度こう配が生じるような、不均一な加熱・冷却による不均等な熱膨張によっても生じる。熱処理などによる変態に伴い体積変化が材料内部で不均一に起きる場合や、さらに、熱負荷により塑性変形が材料内部に不均一に生じる場合にも巨視的残留応力が発生する。b. 微視的残留応力(microscopic residual stress)初期等方性を保っている多結晶金属材料に塑性変形が生じる負荷を与えると、ランダム配向していた結晶粒が負荷により規定されるある特定な方向に回転し、材料は異方性をもつようになる。負荷により異方性をもった材料の結晶粒の方向は、負荷を取り除いても初期の状態には戻らず、各結晶粒内部に大きさの異なる残留応力が生じる。バウシンガー効果の一つの原因として、このような微視的残留応力が考えられる。
