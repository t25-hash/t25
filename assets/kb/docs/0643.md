# 9・6・3 残留応力の利用

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・6 残留応力

残留応力を利用して材料特性を向上することが可能である。材料特性向上の代表的な例として、圧縮残留応力を利用した疲労強度の向上がある。疲労強度は、材料に引張りの残留応力があると低下し、圧縮の残留応力があると向上する。この特性を利用したのが、(a)(b)(c)(d)図9・19速度場とホドグラフ図9・20 はりの曲げによる残留応力3―106材料力学表面圧延やショットピーニング加工を用いた表面改質による疲労強度の向上である。すなわち、材料に表面圧延やショットピーニング加工を施すことにより、材料表層面に圧縮残留応力を発生させ、これにより疲労強度を向上できる。
