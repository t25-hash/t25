# 9・6・4 残留応力の除去

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・6 残留応力

残留応力は時効割れや応力腐食割れを起こす原因となる。このため、残留応力を取り除く作業が必要となる場合がある。熱処理により残留応力を除去する場合は、普通、焼なましが用いられる。焼なまし条件は材料により異なるが、例えばオーステナイト系ステンレス鋼の場合には、850~900°Cで2~4時間保持する方法や、340~430°Cで30分~3時間保持する低温応力除去熱処理がある。また、軽度の引張りやロール成形などの機械的負荷により残留応力を軽減する方法もある。これは、圧延などの塑性加工を経た単純な形状をもつ部材に対しておもに行われる。引張力が作用して塑性化した弾塑性材料は、ひずみの増加とともにその剛性(塑性係数)が減少し続ける。やがて荷重極大点が発生した直後に断面の一部のみが急激な減少を示すいわゆるくびれが始まり、その後、材料の極めて狭い領域にひずみが集中することによって破断に至る。このような一連の挙動は一般的に塑性不安定(plastic instability)現象として知られている。引張力状態における塑性不安定条件はスイフト(Swift)により初めて与えられた。スイフトの塑性不安定条件は、釣合い状態にある材料に微小な増分変形が生じた際に、それによる加工硬化量δσと新しい応力状態になる結果生じる相当応力増分σの間の関係として以下のように表される。δσ≦σ(9・62)このスイフトの塑性不安定条件は、応力-ひずみ関係が硬化型軟化型によらず、引張力が作用する材料の変形が均一な状態において、荷重極大点が発生しうることを示している。単軸引張を受ける棒に対しては、真応力をσ、対数ひずみをε、単軸下でのσ-ε曲線の接線係数を욧とおくとき、욧≦σ(9・63)となり、安定の限界点は最高荷重点(公称応力最大点)である。スイフトの塑性不安定条件のように、材料レベルでの塑性不安定問題の議論は、古典的理論として広く知られている。材料不安定の古典的理論では、応力速度テンソルとひずみ速度テンソルの内積の符号によりその議論がなされ、内積が正である場合は安定となる。>0(9・64)ここで、応力速度テンソルとひずみ速度テンソルを関連付ける4階の構成方程式テンソルを用いた場合の構成関係=(9・65)を不等式(9・64)に代入すると、安定条件は以下のようになる。>0(9・66)以上で述べた材料不安定に関する古典的理論は、微小変形状態を仮定したものである。塑性不安定状態のような有限変形問題に古典的理論を適用する際には、上記の応力速度とひずみ速度による議論を、客観応力速度と速度こう配による議論へと変換する必要があり、その場合には、構成方程式テンソルはそれに整合したものへと変換される。例えば、更新ラグランジュ形式を用いるものとして、公称応力速度をπ욡욢、トルスデル応力速度を욡욢、コーシー応力をσ욡욢、変位をとすれば、π욢욡=욢욡+σ욢욅욡욅(9・67)となる。構成則として例えば욡욢=욡욢욅욆ε욅욆(9・68)ε욅욆=욅욆+욆욅2(9・69)を導入すれば、式(9・65)はテンソルの対称性を利用して、π욢욡=욡욢욅욆욅욆(9・70)욡욢욅욆=욡욢욅욆+δ욡욅σ욢욆(9・71)となる。ただしδ욡욅はクロネッカのデルタである。したがって、式(9・65)と式(9・70)を比較すれば、有限変形問題においては、욡욢욅욆がだ円性を消失したときにその材料が不安定状態になると考えることができる。さらに、塑性不安定状態において材料内での不連続面が発生する場合がある。この不連続面の法線ベクトルをとし、以下のようなアコースティックテンソルを定義する。욡욅=욡욢욅욆욢욆(9・72)そしてこのアコースティックテンソルに対して、不連続面が発生する場合の不安定条件は以下のように与えられる。det=0(9・73)式(9・72)からも明らかなように、アコースティックテンソルは、不連続面の法線方向ベクトルの関数であり二次元問題においては=cos neck)が発生している。平面応力状態の場合では分岐によって除荷が鋼材の端部から始まり、その後に中央部から再び除荷が始まる。そして除荷域に挟まれた×状の細い塑性域が残されるとともに、その×状の部分に変形が集中することによって局部くびれを形成する。よって平面応力状態に関しては、この局部くびれが破断を意味するものと考えられる。平面応力状態においては、図9・22のように×状の局部くびれが発生しているにもかかわらず、分岐モードは拡散くびれ(diffused neck)モードである。よって分岐直後は拡散くびれが発生して、その後に次の段階の局部くびれへと変形が進行したことがわかる。分岐経路上に平衡点が得られると、ただ単にその分岐経路上の平衡点を追跡するだけで拡散くびれから局部くびれへと変形が移行する。このとき分岐経路上では、式(9・78)の塑性不安定条件を満たす平衡点は存在しない。よって均一変形から拡散くびれ、そして局部くびれへと移行する塑性不安定現象は2回の分岐が発生しているのではない。薄板の塑性不安定解析では、分岐は1回のみであり、この分岐挙動として拡散くびれが発生する。局部くびれは1回めの分岐によって引き起こされた除荷域に挟まれる塑性部分に、変形が局所化しているにすぎない。引張力下での弾塑性材料における一連の塑性不安定現象において、荷重極大点は材料的な不安定点であり、くびれの発生は大局的(構造的)不安定としての分岐点ということになる。現実の構造物においては、幾何学的および材料的な非線形性や不安定性が連成するので、有限要素法などの非線形解析により、材料不安定および構造不安定さらには破断に至る際の挙動などの場合分けと、その解釈図9・21
