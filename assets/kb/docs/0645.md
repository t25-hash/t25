# 4・16・1 熱交換器流路の基本形式

> **章**: 第4章 伝熱（物質移動を含む）
> **節**: 4・16 熱交換器の基礎

高温流体から低温流体へ熱エネルギーを移動させるための機器
を，一般に熱交換器（heat exchanger ）という．熱交換器の基本
的な流路形式は，高温側および低温側流体の相対的な流れ方向によ
り，並流（parallel flow），向流（counter flow），直交流（cross
flow）などに分類される（図4･70）．これら比較的簡単な流路形式
については，いくつかの仮定のもとに設計計算手法が確立されてい
る．直交流において，一方の流体が，他方の流体の流れ方向に混合
しない場合を非混合という．例えば，図4･70(ｄ)において，管に
直角方向に流れる流体は混合するが，管内を流れる流体は混合しな
い．
となる．この場合の熱交換器伝熱性能を向上させるためには，熱伝
達率 および伝熱面積 のいずれか（あるいは双方）の向上を図
る必要があることがわかる．しかし，高温側・低温側いずれかの抵
抗が他方に比べて非常に大きい場合には，まずは抵抗の大きい側の
や の増大を目指すべきであることはいうまでもない．
