# 9・8・1 定義

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・8 粘弾性変形

高分子材料、コンクリートなどは弾性的性質とともに時間に依存する粘性的性質を示す。このような材料を一般に粘弾性体(vis-coelastic material)と呼び、その応力とひずみの状態は過去の負荷履歴に依存する。粘弾性体で作られた試験片に一定の引張応力σを図9・23(a)のように時間の間加えたとすると、ひずみは図9・23(b)のように時間とともに変化する。ここで、OA は瞬間弾性ひずみ、AB では遅延弾性ひずみと粘性流動が共存する。時間で応力を除いた場合のBCは瞬間弾性回復ひずみ、CD は遅延弾性回復ひずみであり、粘性流動した部分は最後までひずみは回復しない。粘弾性体の挙動は弾性ばねと粘性ダッシュポットからなるモデルにより定性的に説明できる。図9・24はばねとダッシュポットを直列に結合したマクスウェルモデル(Maxwell model)と呼ばれるモデルである。このモデルから応力σとひずみεは次のように関連付けることができる。式(9・80)(9・81)からεとεを消去すると、応力とひずみの挙動は次のように定式化できる。その応答は、時間とともに応力が低下する応力緩和挙動を示す。図9・25はばねとダッシュポットを並列に結合したフォークトモデル(Voigt model)あるいはケルビンモデル(Kelvin model)と呼ばれるものである。このモデルの応力とひずみの関係は、次のようになる。式(9・85)は時間とともにひずみが増加するクリープ挙動を表し、式(9・86)は応力除去後に時間とともにひずみが減少するひずみ回復を表す。これらのモデルは粘弾性体の力学現象を定性的に表すことができるが、実際の粘弾性体の挙動を正確に表すには、さらに多くのばねやダッシュポットの要素を含むモデルが必要になる。このようなモデルとして図9・26(a)の一般化マクスウェルモデル(generalizedMaxwell model)と図9・26(b)の一般化フォークトモデル(generalized Voigt model)が用いられる。
