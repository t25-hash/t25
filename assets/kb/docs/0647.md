# 9・8・3 非線形粘弾性理論

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・8 粘弾性変形

応力とひずみ、あるいはひずみ速度との間に線形関係が成り立たない材料を非線形粘弾性体(nonlinear viscoelastic material)と呼ぶ。このような材料に対しては、重ね合わせの原理が成立しない。線形粘弾性体の履歴積分に対し、応力あるいは応力速度の高次の項を考慮して、非線形粘弾性体に対する履歴積分は次式で与えられる。ε=σ0+-τdστdτdτ+,σ0+-τ,-τdστdτdσττdτdτ+,,σ0+-τ,-τ,-τ×dστdτdστdτdστdτdτdτdτ+...(9・98)上式の,,,...を実験により決めることは一般に困難であるので、次のような近似式も用いられる。ε=σ,σ0+σ,-τdστdτ dτ(9・99)ここで、σ,は応力レベルσによって変化するクリープコンプライアンスである。
