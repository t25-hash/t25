# 9・9・1 定義

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・9 クリープ変形

一定温度のもとで一定の大きさの応力が材料に作用するとき,時間とともにひずみが生じる現象をクリープ(creep)という.高温下で金属材料に一定応力σを与えたときに観察される(全)ひずみεと時間の関係(クリープ曲線:creep curve)を図9・27に模式的に示す.応力σを与えると瞬間ひずみ(instantaneousstrain)εが生じ,これに続く一定応力下で時間とともにクリープひずみε욑が生じる.クリープ曲線の接線の傾きを表すクリープ速度(creep rate)ε욑=dε욑dは,時間とともに低下し,その後ほぼ一定となり,最後に増加へと転じる.これらの三つの段階(stage)は,それぞれ第1期クリープ(primary creep),第2期クリープ(secondary creep),および第3期クリープ(tertiarycreep)と呼ばれる.これらのクリープ段階は,それぞれ遷移クリープ(transient creep),定常クリープ(steady-state creep),および加速クリープ(accelerating creep)ともいわれる.クリープ曲線の形状は材料,応力および温度によって大きく異なり,これら三つのクリープ段階のいずれかがほとんど現れないこともある.一定保持する応力の大きさが途中で変化すると,この応力変動直後に複雑なクリープ応答が現れる.例えば,引張応力σ>0をσ>0からσ>0へ増加させた場合,応力増加直後においてクリープ速度は一時的に増加し,その後時間の経過とともに後続の応力をはじめから一定負荷した場合の定常クリープ速度に漸近する.応力を低下させた直後のクリープ応答はもう少し複雑になる.すなわち,応力の低下が小さいとき,応力低下によるクリープ速度の過渡的な低下はあまり大きくない.しかし,応力を完全に取り除いた場合σ=0,除荷に伴う瞬間ひずみの分だけひずみが小さくなった後,ひずみが時間とともにさらに小さくなる現象が観察される.一定応力による予クリープ(prior creep)を受けた材料がその後の無応力下で負のひずみ速度を示す現象はクリープ回復(creeprecovery)と呼ばれる.クリープ回復は,部分除荷後にも過渡的に現れることがある.応力変動が繰返し作用する場合,クリープひ図9・27 クリープ曲線3―109ずみの複雑な漸増現象が起きる.このような現象はクリープラチェッティング(creep ratchetting)と呼ばれており,クリープひずみの漸増速度は繰返し応力変動条件に大きく依存する.クリープ変形の物理機構(physical mechanism)は材料の種類によって大きく異なる.多結晶金属材料の場合,結晶粒が微細なときあるいは応力が非常に低く温度が非常に高いときは,原子空孔の拡散(diffusion)によってクリープ変形が生じる.しかし,それ以外の実用上重要な荷重条件下におけるクリープは転位(disloca-tion)の運動によって生じる.転位は,原子の熱振動によって時間的に一定な応力下においても漸進的に運動を継続することができ,高温になるほど低応力で運動を継続することができる.転位運動は,転位の増殖(転位密度の増加)を促し,転位の障害物への堆積や転位間の反応による不動化を引き起こす.この結果,一定応力条件下の転位の可動性は徐々に低下する.このような転位機構は,巨視的にはクリープ速度の低下,すなわちクリープ硬化(creephardening)をもたらす.一方,原子空孔の吸収や放出による転位の上昇(下降),転位反応による転位の消滅(転位密度の低下)あるいは転位運動に対する障害物の拡散は,一定応力条件下の転位の可動性を回復させる.このような回復機構の作用はクリープ軟化(creep softening)をもたらし,その程度は高温になるほど著しくなる.転位運動に付随するこのような硬化機構と回復機構は同時に作用している.したがって,転位運動に律速されるクリープの場合,遷移クリープ域では硬化機構が回復機構よりも強く作用し,定常クリープ域では両機構が釣合っていると解釈することができる.応力変動によって複雑なクリープ現象が現れるのは,クリープ硬化とクリープ軟化が現在の応力条件だけでなくクリープ履歴(creephistory)にも強く依存するためである.金属材料の変形を支配する物理機構の応力依存性および温度依存性は,弾性係数で無次元化した応力と融点温度で無次元化した温度(相対温度)を用いて体系的に整理されている変形機構領域図(deformation mechanismmap)に見ることができる.加速クリープは,ボイド(void)や微視き裂(microcrack)がおもに結晶粒界で急激に増加することによって生じるもので,遷移クリープや定常クリープとは異なる物理機構に支配されている.加速クリープを支配する材料内部の劣化はクリープ損傷(creep damage)と呼ばれている.
