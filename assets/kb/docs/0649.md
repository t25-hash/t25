# 9・9・2 定常クリープ

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・9 クリープ変形

まず、一定応力下のクリープ曲線が定常クリープを仮定することで近似的に表現できる場合を考える。単軸定常クリープ則に含まれる応力σを相当応力σに置換えることによって多軸応力下の定常クリープ速度(定常クリープ速度テンソルの強さ)の応力依存性を記述することができる。式(9・101)のノルトン則は、取扱いが容易なべき関数型を有することから、ミーゼス型の相当応力を用いて多軸形式に拡張したうえで、構造要素の定常クリープ解析に広く用いられている。なお、多軸定常クリープ則を用いて求めた構造要素の応力場は、類似の形式の非線形弾性構成式を用いて求めたそれと一致する。このことは、ホフのアナロジーと呼ばれている。
