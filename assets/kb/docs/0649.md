# 9・9・2 定常クリープ

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・9 クリープ変形

まず,一定応力下のクリープ曲線が定常クリープを仮定することで近似的に表現できる場合を考える.定常クリープ速度(steady-state creep rate)あるいは最小クリープ速度(minimum creeprate)ε욦욑は,温度と応力σの関数として,ε욦욑=Ασ(9・100)のように表現されることが多い.定常クリープ速度の応力依存性は,低応力に対して応力のべき関数σ=σ욗(ノルトン)(9・101)また高応力に対しては応力の指数関数σ=expβσ(ルードウィク)(9・102)を用いて表現できる.低応力から高応力までのすべての応力範囲については,双曲線関教を用いた定常クリープ速度の表示σ=sinhασ욗(ガロファロ)(9・103)が有効である.単軸定常クリープ則に含まれる応力σを相当応力σ욄욤に置換えることによって多軸応力下の定常クリープ速度(定常クリープ速度テンソルの強さ)の応力依存性を記述することができる.式(9・101)のノルトン則は,取扱いが容易なべき関数型を有することから,ミーゼス型の相当応力を用いて多軸形式に拡張したうえで,構造要素の定常クリープ解析に広く用いられている.なお,多軸定常クリープ則を用いて求めた構造要素の応力場は,類似の形式の非線形弾性構成式を用いて求めたそれと一致する.このことは,ホフのアナロジーと呼ばれている.3―110材料力学
