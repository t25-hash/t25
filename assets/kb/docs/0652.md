# 9・10・2 内部状態変数

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・10 統一型非弾性構成式

材料の粘塑性挙動は、変形の履歴に依存するため、通常の観察可能な状態変数(例えば応力や温度)だけで変形挙動を表現することはできない。このために、テンソルあるいはスカラ量の内部状態変数(または内部変数、internal state variable)を導入する。内部変数は内部構造を表現するものであって、金属材料の非弾性変形に限れば、転位や欠陥などの微視的な構造変化が巨視的な応力-ひずみ関係に及ぼす効果を現象論的に表したものと考えられる。式(9・113)に記したような背応力の場合、バウシンガー効果を観測することによって背応力を同定することは可能であるが、変形中に制御することはできない。一般に内部変数は、外部から直接観測可能な変数とその時間変化率および内部変数それ自身によって、発展式(evolution equation)が規定されなければならない。塑性構成式で用いられる移動硬化のための背応力や等方硬化のための降伏応力についても、基本的には上式の枠組に従って発展式が定義されている。発展式中に表れる具体的な材料定数の決定方法としては、9・10・1項に述べた単軸での応力-ひずみ曲線やクリープ曲線の特徴から定めることになるが、材料定数を唯一定めることが困難な場合は試行錯誤して求めることが多い。さらにいくつかの統一型非弾性構成式では、現象論的な意味での転位密度の変化を非弾性変形に関連付けることもあり、関数の特徴や応力-ひずみ曲線への影響を勘案して同定する。
