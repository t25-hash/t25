# 9・10・3 具体的な構成式と発展式

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・10 統一型非弾性構成式

全ひずみは弾性ひずみと非弾性ひずみに分けて、弾性ひずみ速度についてはフックの法則で表される。温度は一定であるとしておく。a. シャボッシュモデル非弾性ひずみ速度がべき乗則に従うとして、静的な降伏応力+を超えた超過応力が非弾性変形に寄与すると仮定する。とはそれぞれ背応力と降伏応力であり、変形や時間とともに変化する。関数sinh・は変数・について鋭い立上りを示すので、そのべき乗を取ることによって疑似的に降伏挙動を記述できる。これを平衡応力-ひずみ曲線という。粘塑性変形に実質的に寄与するσ-εを過応力(overstress)という。このモデルは単軸応力状態で定義されたが、多軸応力状態へ拡張されている。このモデルは、エストリンらによって金属学的な考察を加えられて、さらに発展したモデルとなっている。e. エンドクロニックモデルバラニスは変形に対する材料固有の時間測度(intrinsic time measure)ζをζ=αξ+β(9・126)として定義した。ξはひずみに関連するパラメータであり、は実時間である。結晶形態を考慮し、定義したすべり系に生じるせん断変形と分解せん断応力の関係を基にして構成式を求める。有限変形を想定し速度形の定式化を示す。結晶形態の因子としては、(1)結晶格子構造(FCC,BCC,HPCなど)、(2)結晶方位(方位分布、集合組織)、(3)結晶粒(粒形状、粒界)、(4)析出物(金属間化合物)、があり、硬化・軟化発展の因子としては、(1)転位密度(可動転位、不可動転位、林転位)、(2)粒界効果(有効転位運動行程、サブグレイン、セル構造、粒界拡散)、(3)析出物効果(金属間化合物、界面への転位の堆積)、(4)相効果(塑性変形誘起相変態、多相結晶の複合効果)が考えられる。
