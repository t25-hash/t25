# 9・12・1 定義

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 損傷変数

材料、あるいは負荷条件により材料内部に生じる分布空げきの形状、分布、力学的効果は異なるが、このような損傷状態を表現するためにスカラ、ベクトル、各階のテンソルなどが損傷変数として用いられる。材料の損傷が球形の空洞のように形状に方向性がなく、その分布も等方である場合、あるいは、分布空げきの個々の形状、局所的な分布には方向性があっても大域的特性が等方とみなせる場合など、等方損傷(isotropic damage)に対して損傷変数はスカラで定義される。ここで、スカラで定義される損傷変数をとするとき、その値は次のように定義される。この損傷変数は、損傷に伴う分布空げきの存在による有効面積の減少率と解釈することができる。図9・32のように丸棒の引張りを考えるとき、実質的に荷重を負担する丸棒の断面積は、分布空げきの存在により、丸棒の実際の断面積よりも小さな有効面積となる。ただし、有効面積には、断面に存在する分布空げきの総断面積分の面積減少だけではなく、分布空げきの存在による応力集中の効果も含まれる。このとき損傷変数は、次式で定義される。このほか、損傷変数は微視的き裂密度、微小空洞の体積率と直接関係付けて定義されたり、あるいは、弾性係数などの材料の力学的特性の変化率として定量化されることもある。また、材料内部に生じる分布空げきの形状や分布に方向性があったり、その後の損傷の発達に異方性があり、材料の力学的特性や変形挙動に及ぼす効果に異方性が見られる異方損傷(anisotropicdamage)に対しては、2階や4階のテンソル、あるいはベクトルが用いられる。
