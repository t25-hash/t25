# 9・12・2 損傷発展式

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 損傷変数

連続体損傷力学における理論は,カチャノフによって初めて提案され,ラバトノフによって拡張されたクリープ損傷理論(creepdamage theory)が,その後の多くの損傷理論の原型となっている.ここで,単軸応力状態のクリープひずみをε욑とおくとき,カチャノフ・ラバトノフのクリープ損傷理論として次式がよく用いられる.3―114材料力学ε욑=σ욗1-욖=σ1-(9・137)ここで,,,,,μ,νは材料定数である.その後,クリープ損傷以外にも種々の負荷条件下で生じる損傷に対して損傷発展式は定式化されている.例えば,時間非依存な塑性損傷は,応力σがある限界値σに達して初めて発生すると考え,次式が提案されている.dε=σ-α1-,,dλ, =ε(9・138)d=σ-σ1-dσ(9・139)ここで,εは塑性ひずみ,αは背応力,は加工硬化パラメータ,は温度を表し,,は材料定数である.また,疲労損傷に対しては,繰返しの負荷条件に応じて式(9・139)を積分すれば計算可能であるが,次式のような繰返し数に対する損傷発展式が提案されている.=1-1-σ윯율융-σ욇1-(9・140)ここで,σ윯율융は最大応力,σ욇は平均応力,αはσ윯율융とσ욇の材料関数,はσ욇の材料関数,βは材料定数である.また,クリープ-疲労損傷に対しては,式(9・137)と式(9・140)を組合わせた発展式も提案されている.なお,ここでは単軸応力状態での損傷発展式,ならびに構成式を示したが,多軸応力状態への拡張もなされている.
