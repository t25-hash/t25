# 9・12・2 損傷発展式

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 損傷変数

連続体損傷力学における理論は、カチャノフによって初めて提案され、ラバトノフによって拡張されたクリープ損傷理論(creepdamage theory)が、その後の多くの損傷理論の原型となっている。ここで、単軸応力状態のクリープひずみをεとおくとき、カチャノフ・ラバトノフのクリープ損傷理論として次式がよく用いられる。その後、クリープ損傷以外にも種々の負荷条件下で生じる損傷に対して損傷発展式は定式化されている。例えば、時間非依存な塑性損傷は、応力σがある限界値σに達して初めて発生すると考え、次式が提案されている。また、疲労損傷に対しては、繰返しの負荷条件に応じて式(9・139)を積分すれば計算可能であるが、次式のような繰返し数に対する損傷発展式が提案されている。また、クリープ-疲労損傷に対しては、式(9・137)と式(9・140)を組合わせた発展式も提案されている。なお、ここでは単軸応力状態での損傷発展式、ならびに構成式を示したが、多軸応力状態への拡張もなされている。
