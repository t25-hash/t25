# 10・1・3 多軸応力下の静的強度

> **章**: 第10章 材料の強度
> **節**: 10・1 に示した引張破壊、カップアンドコーン破壊、せん断破壊、

等三軸圧縮応力(静水圧)下では、材料に塑性変形も破壊も生じない。しかし、等三軸引張応力下では、塑性変形することなく、脆性破壊を生じる。等三軸圧縮応力に加えて、一軸方向にのみ引張応力を負荷した場合、材料は延性を示し、破断ひずみは著しく増大する。図10・3に一例を示す。逆に、等三軸引張応力に加えて、一軸方向のみに引張応力を負荷した場合、材料は脆性を示し、破断ひずみは著しく減少する。多軸応力下の破壊条件(fracture criterion in multiaxial stateof stress)と適用の対象を以下に示す。ただし、σ,σ,σは主応力σ>σ>σ,σは単軸応力下の静的強度である。3―117図10・2 公称応力-公称ひずみ関係図10・3 破断ひずみと等三軸圧縮応力の関係a. 最大主応力説σ=σ、脆性破壊。b. 最大せん断応力説σ-σ=σ、延性破壊、トレスカの降伏条件に同じ(9・3・1項参照)。ただし、μはすべり面の摩擦係数、τは純せん断のすべり強度である。また、種々の場合について、多軸応力下の静的強度を実験し、結果をモールの応力円として描けば、それらの包絡線が降伏または破壊条件を与える。両側あるいは環状切欠試験片に単軸応力を負荷した場合、切欠底断面で塑性変形の結果として生じる断面収縮は、上下の塑性変形していない断面の拘束により妨げられる。したがって、切欠底断面では多軸応力状態となる。これを塑性拘束(plastic constraint)という。多軸応力下の降伏条件からわかるように、全断面降伏した場合の軸方向引張応力σは、平滑試験片の単軸応力下の降伏応力σよりもはるかに大きくなる。両者の比=σ/σが塑性拘束係数(plastic constraint factor)である。その結果、静的強度もほぼ倍に増大するが、一方、脆性破壊を生じやすくなる。ただし、1≦≦3である。
