# 10・2・1 疲労破壊

> **章**: 第10章 材料の強度
> **節**: 10・2 疲労強度

疲労破壊(fatigue fracture)は,静的破壊(static fracture)より低い負荷応力の繰返しによって起こる現象であり,き裂の発生,進展およびリガメント部の静的最終破断という経過をたどる.典型的な疲労破壊過程のモデルは最初にフォーサイスにより示されたが,図10・7はその模式図である.一般にき裂発生過程(crackinitiation process)を含めてすべり帯に沿う初期進展過程を第I段階(first stage of fatigue crack growth),それに引続く引張応力に依存した進展過程を第II段階(second stage of fatigue crackgrowth)のき裂進展と称している.一般の構造物の耐疲労設計においては,疲労き裂発生とき裂進展に分けて考える必要がある.すなわち,比較的小さい要素では疲労き裂進展寿命(fatigue crackgrowth life)は短く,き裂発生が即座に要素の破壊を導くため,疲労き裂発生寿命(fatigue crack initiation life)により設計を行う.一方大型構造物や航空機等の冗長度(redundancy)の高い構造物では疲労き裂が発生してもただちに破壊に至ることはないので,後述するように疲労き裂進展過程を理解したうえでき裂進展寿命を考慮した設計を行う.
