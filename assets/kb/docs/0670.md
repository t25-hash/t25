# 10・2・4 疲労強度のばらつき

> **章**: 第10章 材料の強度
> **節**: 10・26 中の実線はハダッドらによる提案で、実際2の長さのき

疲労破壊は極めて局所的損傷により生じるものであるため、材料の不均質性や種々の欠陥等の微視組織に敏感である。耐疲労信頼性設計を行うためには、それらの分布特性を把握しておく必要がある。図10・27は材料の確率疲労特性の代表例であり、同一応力振幅レベルにおいても破断寿命には大きなばらつきが現れている。疲労破壊のばらつき特性を表すものとして図10・27に示すように疲労寿命分布と時間強度分布がある。前者は同一応力振幅に対する寿命の分布特性を表したものであり、後者は同一の破断寿命に対する応力振幅のばらつき特性を評価したものである。図10・28は鉄鋼材料の統計的疲労試験結果の例であるが、図中には疲労寿命分布から求められる同一破壊確率(fracture probability)に対する-曲線群が描かれている。寿命分布は有限寿命域では対数正規分布もしくは2母数ワイブル分布にほぼ従う。一方、時間強度分布は正規分布として扱うことが多い。疲労寿命と同様に疲労限度も大きなばらつきを示す。疲労限度の分布形は正規分布として扱ってよい場合が多く、その分布を求める手法としてはプロビット法、ステアケース法が用いられる。手法の詳細は文献(37)を参考願いたい。
