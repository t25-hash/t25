# 10・2・6 低サイクル疲労

> **章**: 第10章 材料の強度
> **節**: 10・34 は回転曲げと引張圧縮試験での疲労限度に及ぼす直径の影響

破断繰返し数が10サイクル程度以下である領域を低サイクル疲労(low cycle fatigue)と呼ぶ。低サイクル疲労損傷は、前述した固執すべり帯のような局所すべりに関連するものではなく、大変形の繰返しによって結晶粒内で多重すべりが生じることによって起こるものである。金属材料の低サイクル疲労特性は、荷重制御もしくは変位制御の2種類の試験で一般に得られているが、実機の部材が受ける荷重・変位履歴は変位制御状態が多いようである。図10・48は低サイクル領域における応力-ひずみヒステリシスループ(stress-strain hyteresis loop)であり、以下の諸量は図10・48に示されるように定義される。繰返し応力下の応力-ひずみ応答(stress-strain response)は、通常の静的引張試験で得られる応答とは異なる。図10・49は全ひずみ制御下での疲労試験における繰返しに伴う応力振幅の変化である。焼なまし材ではひずみの繰返しとともに徐々に応力振幅が増大していき、繰返し硬化(cyclic hardening)挙動を示す。一図10・49一定ひずみ制御下の繰返し変形挙動(a) 銅焼なまし材(b) S 35C焼ならし材(c) SCM 435材方、冷間引抜き材では、逆に応力振幅が減少していく繰返し軟化挙動(cyclic softening)を示している。材料の疲労変形抵抗の尺度としては繰返し軟化もしくは繰返し硬化が定常状態に達した時点での応力-ひずみ関係を用いる。これを繰返し応力-ひずみ曲線(cyclic stress-strain curve)と呼ぶ。図10・50に引張試験における応力-ひずみ関係と繰返し応力-ひずみ関係を示すが、繰返し硬化する材料の繰返し応力-ひずみ曲線は静的応力-ひずみ曲線より高応力側に位置し、繰返し軟化する材料では逆になる。定常状態における繰返し応力-ひずみ曲線は、一般に次式で表される。′は材質の影響をあまり受けず、多くの材料において0.10~0.20程度、あるいは0.15に近い値となることが知られている。低サイクル疲労寿命を支配する力学量はおもに塑性ひずみ幅(plastic strain range)である。図10・51は破断寿命と塑性ひずみ幅の関係を両対数紙上に示したものである。両者の関係は全寿命領域で1本の直線で表現でき、いわゆる次式で表されるマンソン・コフィン則(Manson-Coffin law)が成立する。Δε・=(10・23)マンソン・コフィン則に含まれる定数αは材料によらず0.5~0.6程度の値をとる。一方、材料定数が材料の静的破断延性εに密接に関係していることは広く知られている。実機において力学量として塑性ひずみ幅が得られることはまれで、力学情報としては一般にはひずみゲージなどから測定される全ひずみ幅Δεであり、Δεに基づく寿命評価が重要となる。図10・52は共通こう配法による寿命推定法の説明図であり、全ひずみ幅に対する疲労寿命曲線は次式で与えられる。
