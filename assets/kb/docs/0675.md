# 10・3・1 クリープ変形

> **章**: 第10章 材料の強度
> **節**: 10・3 高温強度

材料に一定温度のもとで一定の大きさの応力が作用するとき、ひずみが時間とともに増加する現象をクリープ(creep)という。図10・65は、純ニッケルの変形応力と温度の関係を示す。ただし、縦軸のせん断応力τは横弾性係数で、横軸の絶対温度は融点욇でそれぞれ除して、正規化して表されている。この図度と応力に依存するためτ--γ関係(γ:せん断クリープ速度)が異なることを説明している。この図10・65を変形機構領域図(deformation mechanism map)という。クリープ変形の機構は、高応力下の転位の交差すべりや上昇運動によるものと、低応力下の原子空孔の拡散によるものに大別される。前者の転位クリープでは、定常クリープ速度ε욦は応力のべき乗則で表され、温度依存性を含めてε욦∝σ욗exp -(10・26)となる。はクリープ指数、はガス定数、はクリープの活性化エネルギーである。後者は、低温側で認められる粒界拡散クリープと高温側で認められる格子拡散クリープに分けられる。いずれも、原子の拡散による移動に起因する変形であるが、それが粒界に沿って輸送される場合を粒界拡散クリープといい、粒内を輸送される場合を格子拡散クリープ(または、体拡散クリープ)という。両者ともクリープ速度は応力に比例する(=1)が、拡散定数等が異なるので両者のクリープ速度は相違する。なお、図10・65の変形機構領域は、結晶構造、結晶粒径、積層欠陥エネルギー等に依存する。実用耐熱合金では、溶質原子の添加による固溶強化、炭化物や金属間化合物による析出強化、高融点金属の添加による拡散抑制強化などが施され、耐クリープ性を向上させている。このような強化機図10・64 溶接品質の等級と許容応力図10・65 純ニッケルの変形領域図3―132材料力学構は、負荷中、長時間にわたって安定に持続させることが望ましい。しかし、多くの耐熱合金においてはこれらの金属中の組織に経時変化(経年劣化)が生じる。このために、長時間クリープほど低ひずみレベルから加速(第3期)クリープが出現したりすることがある。応力や温度が変動する場合に生じる時間依存性変形も広義のクリープである。この場合も変形機構は定応力、定温度の場合のそれと基本的に同じであるが、負荷あるいは温度履歴の影響が現れるところに特徴がある。例えば、応力をσからσに増加させると、σ一定の場合のクリープ速度より大きなクリープ速度を生じ、時間が経過すると次第に減速して定常化する。応力をσからσに減少させても、一時的にクリープ速度が減速する傾向がある。このような遷移クリープに類似の過渡的非定常クリープ変形が生じるのは、一定応力下で加工硬化と回復が釣合って、微視構造が定常状態にあったところを、応力変動によって微視構造が変化し、内部応力場が変動して釣合いが乱されたためと考えられている。
