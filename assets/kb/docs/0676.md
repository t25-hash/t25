# 10・3・2 クリープ破断

> **章**: 第10章 材料の強度
> **節**: 10・65 は、クリープ変形にはいくつかの微視機構があり、それが温

クリープ変形によって材料が破断する現象を、クリープ破断(creep rupture)という。通常は、その時間強度を得るために、種々の温度レベルでのクリープ破断曲線(creep rupture curve、応力と破断時間の関係)を求める。図10・66は、SUS 304(ボイラ用鋼管)に関する温度650°Cのクリープ破断試験の結果である。ボイラや蒸気タービンなど、長時間にわたって高温高圧下で使用される構造物においては設計の際の許容応力(allowable stress)はクリープ破断強度によって規定される。例えば、10時間クリープ破断強度の平均値の67%、あるいは10時間クリープ破断強度の最小値の80%などである。ここで、10時間(11.4年)という長期間破断試験の結果が得られていない場合には、それより短時間のデータに基づいて外挿しなければならない。この外挿法(extrapolation method)として、温度補償時間パラメータを用いる方法がある。図10・66に示した温度によって異なるσ-関係を、を調整することによって温度によらない1本のσ-関係(主クリープ破断曲線)にまとめることができる。この方法において問題となるのは、広い温度範囲にわたるとσ-曲線が1本で定まらなくなることである。また、10倍以上の長時間側への外挿は一般性に乏しいとされている。
