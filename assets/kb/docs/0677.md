# 10・3・3 クリープき裂の発生と進展

> **章**: 第10章 材料の強度
> **節**: 10・65 は、クリープ変形にはいくつかの微視機構があり、それが温

図10・67は、応力と温度による破壊機構の相異を整理した破壊機構領域図(fracture mechanism map)である。クリープ破壊は、低温・高応力下の粒内破壊と高温・低応力下の粒界破壊に大別図10・66 SUS 304の650°Cにおけるクリープ破断曲線できる。後者の粒界破壊(intergranular fracture)がクリープ破壊の代表的形態としてよく知られており、比較的低温・高応力下ではくさび型、高温・低応力下では球形(キャビティ)の粒界き裂となる。くさび型き裂の発生・成長は粒内のクリープ変形およびそれに基づく粒界すべりの寄与が大きいが、キャビティについては拡散が関与し、高温・低応力になるほど純拡散支配下の発生・成長が起こりうる。また、長時間クリープでは粒界炭化物がσ相に変わり、それを核としたキャビティの発生・成長がある。比較的大きな欠陥や鋭い切欠などが存在する材料や構造部材においては、それから単一のクリープき裂が進展する可能性がある。クリープ積分はき裂先端近傍のクリープ速度場を代表する破壊力学パラメータであり、き裂進展がき裂先端近傍のクリープ速度によって支配されているため、上式が成立する。なお、応力負荷直後ではクリープひずみが弾性ひずみよりもかなり小さいことがあり、この場合には、き裂先端から離れた部分は弾性場が支配し、先端近傍はクリープ場が支配する。これを小規模クリープ(small scale creep)状態という。時間が経過すると、クリープ支配領域が拡大して部材全体を覆うようになる。これを大規模クリープ(large scale creep)状態という。小規模クリープでは大規模クリープよりき裂先端近傍の応力場・ひずみ速度場の強さは大きくなっており、応力・温度・き裂長さが同一でもき裂は図10・67 Nimonic 80A の破壊機構領域図図10・68 クリープき裂進展速度とクリープ積分の関係より速く伝ぱすることに注意を要する。
