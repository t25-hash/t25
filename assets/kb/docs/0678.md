# 10・3・4 高温疲労寿命

> **章**: 第10章 材料の強度
> **節**: 10・65 は、クリープ変形にはいくつかの微視機構があり、それが温

高温で繰返し負荷を受けると、疲労損傷(fatigue damage、繰返し数依存性損傷)だけではなく、時間に依存したクリープ損傷(creep damage)が発生する。実際高温疲労においては、クリープと疲労によって破壊機構が複雑になっており、これをクリープ・疲労相互作用(creep-fatigue interaction)という。図10・69(a)は、SUS 304の高温疲労における破壊形態の相異を示したものである。この破壊形態の相異に対応して、図10・69(b)に示すように高温疲労寿命も大きく異なっている。クリープ・疲労相互作用を受ける材料の寿命を推定する最も簡単な方法は、クリープ損傷φと疲労損傷φの線形和が1に達したときに破損するとする線形累積損傷則(linear cumulative dam-age rule)に基づく方法である。一方、疲労損傷φはクリープ変形が含まれない一定ひずみ振幅下の疲労寿命に対する実際の繰返し数の比の総和である。この評価方法は簡便であるため高温構造の設計によく用いられるが、材料・温度・ひずみ波形によって損傷の線形和が1にならない場合も多い。現実的には適切な安全係数を設定することによって線形損傷則を拡張適用して用いられている。
