# 4・8・6 外部流の強制対流乱流熱伝達

> **章**: 第4章 伝熱（物質移動を含む）
> **節**: 4・8 強制対流乱流熱伝達

流れの中に置かれた円柱のような物体の背後では，レイノルズ数
が大きいと流れがはく離し，乱れが生成されるとともに，その影響
ははく離点の上流にも現れる．また，大規模渦塊が形成，そして放
出されるために，物体まわりの流れは極めて複雑になる．このよう
な複雑な流れ状態の熱伝達率を理論的に求めることは困難であり，
主として実験的に求められている．

れた円柱まわりの流れは，レイノルズ数の増加とともに，前面と後
面が対称な流れから後面に双子渦を伴う流れ，周期的な渦放出を伴
う流れへと大きく変化する ので，熱伝達率もこれに伴って変化
する．図4･35にレイノルズ数＝νが比較的大きな場合の

いわれている．レイノルズ数が4.26×10では後面における熱伝達
