# 10・3・5 高温疲労き裂の発生と進展

> **章**: 第10章 材料の強度
> **節**: 10・65 は、クリープ変形にはいくつかの微視機構があり、それが温

低サイクルクリープ疲労においては、寿命の初期段階からき裂が発生し始め、き裂の数はクリープ疲労繰返し数とともに増加する。発生した個々のき裂は、微視組織の影響を受けて発生直後から停留と進展を繰返す。また、き裂の発生部および進展経路はクリープ疲労条件に強く依存している[図10・69(a)].このき裂発生と初期進展に及ぼす微視組織の影響は、定量的・力学的に明らかになっていない。発生寿命やき裂進展速度に関する不規則な挙動に対して、モンテカルロ法等による確率論的な取扱いが検討されている。高温疲労における巨視き裂の進展は、クリープ変形に起因する時間依存性のものと繰返し数依存性(室温疲労におけるき裂進展と同機構)のものに分けられる。前者は、高温および1サイクルの周期が長い疲労において現れ、後者は比較的低温短周期の疲労において現れる。時間依存性き裂進展はクリープによるものであるので、その進展速度(単位時間当たりの伝ぱ速度)ddに対しては、式(10・28)の破壊力学法則が成立する。3―134材料力学また、小規模クリープも毎サイクル現れる可能性がある。一方、繰返し数依存性き裂進展は、疲労によるものである。
