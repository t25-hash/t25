# 10・4・1 応力腐食割れ

> **章**: 第10章 材料の強度
> **節**: 10・4 環境強度

応力腐食割れ(SCC: stress corrosion cracking)とは,材料因子,環境因子に加え,応力(残留応力を含む)負荷の組合せによって,環境の影響を受けない条件下の静的強度よりも小さな荷重負荷条件下において,き裂が発生・進展し破壊に至る現象をいう.SCCの機構には,き裂先端が溶解して進む活性路溶解型SCC(activepath corrosion type SCC)と材料中に水素が供給されて脆化し,破壊する水素脆化割れ(10・4・2項参照)の二つに大きく分けられるが,例えばAl合金のように両者の機構の区別があまり明確ではない場合もある.SCCで重要な点は,先に述べた3因子が同時に作用したときに生じることであり,一つでも欠けるとSCCは生じない.材料因子では,化学成分,引張強度等の機械的性質,偏析やステンレス鋼などの鋭敏化の原因となるCr欠乏層などに代表される析出などの材料の不均一性が重要である.環境因子としては,材料特有の有害アニオン(例えばCl,OHなど)や,電位や溶存酸素,pH などが影響を与える.また,すきまなどがあると沖合溶液に比べてpH の低下やアニオンの局部的濃縮が生じてSCCが生じる環境条件になる場合がある.このようなSCCの特徴をまとめると,(1)合金で生じ,純金属では生じない,(2)腐食がほとんど認められない環境でもSCCを生じ,全面腐食が生じる環境ではSCCは生じない,(3)応力成分としては引張応力が重要である,(4)破面は粒界割れと粒内割れがあり,材料と環境に依存する,などが挙げられる.表10・3に金属材料がSCCを生じる環境をまとめて示しておく.また,SCC条件下において,低繰返し速度の大変動応力(応力比0)や高平均応力下に高繰返し速度の振動応力が重畳するとき(応力比1),一定持続荷重あるいは一定変位条件下に比べてSCC強度が低下したり,SCCき裂進展速度が上昇することがある.このとき重要な点は,SCCの下限界値よりも小さな最大応力下においてもSCCの影響が現れ,下限界値が低下する場合があることであり,これらを繰返し応力腐食割れ(cyclic stress corrosion cracking,cyclic SCC),動応力腐食割れ(dynamic stress corrosion crack-ing,dynamic SCC)と呼んでいる.SCC機構には,種々の機構が提案されている.有害イオン吸着説では,吸着した有害イオンにより表面エネルギーが低下し,き裂が進展するとするものである.また,二段階説は,変色皮表10・3 の制御,あるいはインヒビター添加などによる環境の制御,(4)ライニングなどのコーティングやSCCに優れる材料とのクラッド材を使用するなどの環境との遮断,(5)起動・停止時の温度-溶存酸素濃度の制御などのプラントの運転管理,(6)構造設計の見直しなどが用いられている.
