# 10・4・2 水素脆化割れ

> **章**: 第10章 材料の強度
> **節**: 10・4 環境強度

水素脆化割れ(hydrogen embrittlement)は,材料中に水素が吸蔵されることにより,劣化を生じて割れに至る現象であり,めっきや酸洗い,カソード防食下,あるいは腐食により水素が発生する条件下で問題となる.水素脆化としては,湿潤硫化水素環境中のラインパイプなどで問題となる水素誘起割れ(硫化物応力腐食割れ)や高強度ボルトで生じるいわゆる遅れ破壊が代表的な事例である3―135図10・73 引張強度と材料中の水素溶解量で整理した水素脆化感受性が,チタン,ジルコニウム,あるいは金属間化合物など多くの材料で環境脆化の要因として問題となっている.HEの機構としては,材料中の空げきに水素が集まり,空げきで水素分子となり,その高い圧力により割れが生じるとする水素ガス圧説[ザッフェ(Zapffe)の説],トロイアノ(Troiano)やオリアニ(Oriani)らが提唱した材料内部のき裂などの欠陥に外部応力負荷により生じる三軸応力場に応力誘起拡散により水素が集中し,その結果格子間結合力を低下させて破壊に至るとする格子脆化説,さらに水素が変形中に生じる原子空孔を安定化させ,その密度が上昇し,それらが連結,あるいは格子を不安定化させて破壊に至るとする,原子空孔説などがある.このうち,水素ガス圧説は湿潤HS 環境の硫化物応力腐食割れやブリスターの要因となっている.また,高強度鋼などでは,格子脆化説や原子空孔説などで説明されるが,現時点では明確な結論は得られていない.図10・73に水素による鋼の損傷事例を引張強度と材料中の水素溶解量(hydrogen content)の関係で整理した結果を示す.図このことは,低強度材料では硫化水素を含んだような過酷な環境がHEに対して必要であるのに対して,強度が高い場合は雨水などのマイルドな環境下の腐食により発生する微量水素の吸蔵によりHEを生じることを意味している.また,一般に水素の溶解量が小さく,また格子ひずみが大きくなる体心立方格子のほうが面心立方格子よりもHEが生じやすい.なお,水素脆化割れ特性の改善には,粒界に偏析するP,S を減少させる,高温での焼戻しが可能なようにV などの特殊炭化物生成元素を添加する,旧オーステナイト粒界にフィルム状の炭化物の析出を抑えるなどが用いられている.
