# 10・4・2 水素脆化割れ

> **章**: 第10章 材料の強度
> **節**: 10・4 環境強度

水素脆化割れ（hydrogen embrittlement）は，材料中に水素が
吸蔵されることにより，劣化を生じて割れに至る現象であり，めっ
きや酸洗い，カソード防食下，あるいは腐食により水素が発生する
条件下で問題となる．水素脆化としては，湿潤硫化水素環境中のラ
インパイプなどで問題となる水素誘起割れ（硫化物応力腐食割れ）
や高強度ボルトで生じるいわゆる遅れ破壊が代表的な事例である
が，チタン，ジルコニウム，あるいは金属間化合物など多くの材料
で環境脆化の要因として問題となっている．HEの機構としては，
材料中の空げきに水素が集まり，空げきで水素分子となり，その高
い圧力により割れが生じるとする水素ガス圧説［ザッフェ
（Zapﬀe）の説 ］，トロイアノ（Troiano）やオリアニ（Oriani）
らが提唱した材料内部のき裂などの欠陥に外部応力負荷により生じ
る三軸応力場に応力誘起拡散により水素が集中し，その結果格子間
結合力を低下させて破壊に至るとする格子脆化説，さらに水
素が変形中に生じる原子空孔を安定化させ，その密度が上昇し，そ
れらが連結，あるいは格子を不安定化させて破壊に至るとする，原
子空孔説 などがある．このうち，水素ガス圧説は湿潤HS環
境の硫化物応力腐食割れやブリスターの要因となっている．また，
高強度鋼などでは，格子脆化説や原子空孔説などで説明されるが，
現時点では明確な結論は得られていない．
図10･73 に水素による鋼の損傷事例を引張強度と材料中の水
素溶解量（hydrogen content）の関係で整理した結果を示す．図
10･73より強度が高いほどHEの限界水素溶解量が低下している．
このことは，低強度材料では硫化水素を含んだような過酷な環境が
HEに対して必要であるのに対して，強度が高い場合は雨水などの
マイルドな環境下の腐食により発生する微量水素の吸蔵によりHE
を生じることを意味している．また，一般に水素の溶解量が小さ
く，また格子ひずみが大きくなる体心立方格子のほうが面心立方格
子よりもHEが生じやすい．なお，水素脆化割れ特性の改善には，
粒界に偏析するP，Sを減少させる，高温での焼戻しが可能なよう
にVなどの特殊炭化物生成元素を添加する，旧オーステナイト粒
界にフィルム状の炭化物の析出を抑えるなどが用いられている．
10・4・3環境疲労
水溶液中あるいは種々のガス環境下において，乾燥空気あるいは
Arのような不活性ガス環境中と異なる疲労挙動特性を示す場合を
総称して環境疲労（腐食疲労）と呼んでいる．特に海水，湿潤ガス
などの腐食性環境中においては，参照環境中に比べて疲労寿命が短
くなり，またき裂進展速度が上昇することがある．これを環境疲労
の狭義の意味で，腐食疲労（corrosion fatigue）と呼ぶことがあ
る．SCCの場合は，材料と環境がある組合せのときに生じるが，
腐食疲労の場合は必ずしも明確な材料と環境の組合せは存在せず，
どのような材料においても生じることに注意を要する．参照環境と
して大気環境を選ぶこともあるが，大気中の湿気はき裂やすきまに
おいて毛管凝縮を生じるので，高強度鋼や高強度Al合金において
は大気も腐食性環境として無視できない．また，腐食疲労において
は，環境と変動応力が同時に作用するときに害作用が大きく現れ，
前腐食による表面の凹凸に起因する疲労強度の低下よりも大きくな
る．
