# 10・4・3 環境疲労

> **章**: 第10章 材料の強度
> **節**: 10・73 より強度が高いほどHEの限界水素溶解量が低下している.

水溶液中あるいは種々のガス環境下において,乾燥空気あるいはArのような不活性ガス環境中と異なる疲労挙動特性を示す場合を総称して環境疲労(腐食疲労)と呼んでいる.特に海水,湿潤ガスなどの腐食性環境中においては,参照環境中に比べて疲労寿命が短くなり,またき裂進展速度が上昇することがある.これを環境疲労の狭義の意味で,腐食疲労(corrosion fatigue)と呼ぶことがある.SCCの場合は,材料と環境がある組合せのときに生じるが,腐食疲労の場合は必ずしも明確な材料と環境の組合せは存在せず,どのような材料においても生じることに注意を要する.参照環境として大気環境を選ぶこともあるが,大気中の湿気はき裂やすきまにおいて毛管凝縮を生じるので,高強度鋼や高強度Al合金においては大気も腐食性環境として無視できない.また,腐食疲労においては,環境と変動応力が同時に作用するときに害作用が大きく現れ,前腐食による表面の凹凸に起因する疲労強度の低下よりも大きくなる.腐食疲労に及ぼす影響因子としては,力学因子,材料因子,およ3―136材料力学図10・75種々の鋼の引張強度と人工海水中の腐食疲労限の関係び環境因子があるが,このうち力学因子としては,応力繰返し速度,応力波形,応力比あるいは最大応力などが挙げられる.参照環境中においては,応力繰返し速度の影響はほとんど見られないが,腐食環境中では応力繰返し速度が小さいほど腐食疲労寿命が低下したり,疲労き裂進展速度が上昇する.また,参照環境中においては,応力波形効果はほとんど見られないが,腐食環境中では応力波形の影響が現れ,応力上昇時の割合が大きいほど腐食の害作用が大きくなる.平均応力の効果としては,応力振幅との組合せによっては,保護皮膜を破壊したり,あるいはSCCを誘発して,繰返し応力腐食割れ,動応力腐食割れを生じて,大きく腐食疲労強度が低下することがある.材料因子としては,材料と環境との反応性,すなわち耐食性が第一義的に重要であるが,破面にほとんど腐食が見られない耐食材料においてもき裂進展速度が上昇することがある.図10・74にHT 780鋼の人工海水中の-曲線を示す.図労強度にはほとんど差異は見られないが,低応力域では腐食疲労寿命の低下が顕著であり,疲労限は著しく小さくなる.また応力繰返し速度が小さいほど腐食疲労強度が低下する.引張強度と人工海水中疲労限の関係を図10・75に示すように,炭素鋼,低合金鋼では引張強度による腐食疲労強度の上昇はほとんど期待できない.耐腐食疲労対策としては,種々の防食法の適用は一般的には効果的であるが,特に高強度材ではカソード防食は水素脆化の影響が生じることがあるので,注意が必要である.また,SCCの場合と同様にマイルドな腐食環境中においても,局部的濃縮が生じて強度低下が生じることも極めて多いので,構造変更や環境のコントロールも重要である.
