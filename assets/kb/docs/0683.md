# 10・4・4 腐食疲労の発生と進展

> **章**: 第10章 材料の強度
> **節**: 10・74 より明らかなように高応力域では参照環境と人工海水中の疲

腐食疲労の特徴は、疲労寿命の初期にき裂が発生し、徐々に進展していくことにある。疲労荷重を受けると、金属表面にはすべりが生じて新生面が現れるが、腐食環境中においては、この新生面の部分がアノードに、またすべりを生じていない部分がカソードとなり、局部電池が形成され、この結果すべり帯において優先的な溶解図10・74 HT 780鋼の人工海水中の腐食疲労強度特性図10・76 種々の高張力鋼の人工海水中腐食疲労試験(=0.1,=0.17Hz)におけるピット深さと負荷応力σの関係(中実印:き裂発生、中空印:き裂なし)が生じ、き裂が発生したり、条件によっては腐食ピット(孔食)が形成される。腐食ピット(corrosion pit)中では、自己被毒作用により溶液が酸性化して溶解がさらに促進され、ある限界の大きさを超えるとピット底からき裂が発生し、伝ぱする。腐食ピットから腐食疲労き裂が発生する条件としては、(1)腐食ピットがある限界の大きさを超えたときに腐食疲労き裂が発生する、あるいは(2)腐食ピットの成長速度を腐食疲労き裂進展速度が上回ったときにき裂が発生する、とする説がある。いずれにおいても、き裂発生の力学条件として腐食ピットを鋭いき裂とみなして求めた応力拡大係数幅が有用である。図10・76に、腐食疲労試験において、腐食ピット底にき裂が発生しているものを中実印で、またき裂が発生していないものを中空印で示しているが、これよりき裂は腐食ピット深さがある限界値を超えたときに発生していること、負荷応力が小さいほど腐食疲労き裂が発生する限界深さは大きくなる。き裂が発生している腐食ピットを鋭いき裂とみなして、腐食ピット底の応力拡大係数幅(Δ)を求め、これらの平均値を基に個々の腐食ピットのアスペクト比の平均値に対して、応力振幅と腐食ピット深さから腐食疲労き裂が発生する限界線を引いているが、これより腐食ピットを鋭いき裂としてみなして求めた応力拡大係数幅により腐食ピットからのき裂発生をほぼ推定できることがわかる。さらに、き裂が進展すると参照環境中の疲労き裂進展速度に比較して、き裂進展速度が加速したり、あるいは減速し、その挙動は複雑である。減速因子としては、き裂の分岐やき裂内部に腐食生成物や人工海水中のカソード防食条件下で生じる白色生成物などによるき裂閉口[腐食生成物のくさび効果(wedge effect、腐食生成物誘起き裂閉口)、白色生成物誘起き裂閉口]が、また加速因子としては腐食溶解や水素脆化割れが重畳することによる。
