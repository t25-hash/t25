# 10・4・4 腐食疲労の発生と進展

> **章**: 第10章 材料の強度
> **節**: 10・74 より明らかなように高応力域では参照環境と人工海水中の疲

腐食疲労の特徴は，疲労寿命の初期にき裂が発生し，徐々に進展
していくことにある．疲労荷重を受けると，金属表面にはすべりが
生じて新生面が現れるが，腐食環境中においては，この新生面の部
分がアノードに，またすべりを生じていない部分がカソードとな
り，局部電池が形成され，この結果すべり帯において優先的な溶解
が生じ，き裂が発生したり，条件によっては腐食ピット（孔食）が
形成される．腐食ピット（corrosion pit）中では，自己被毒作用に
より溶液が酸性化して溶解がさらに促進され，ある限界の大きさを
超えるとピット底からき裂が発生し，伝ぱする．腐食ピットから腐
食疲労き裂が発生する条件としては，(１)腐食ピットがある限界の
大きさを超えたときに腐食疲労き裂が発生する，あるいは(２)腐食
ピットの成長速度を腐食疲労き裂進展速度が上回ったときにき裂が
発生する，とする説がある．いずれにおいても，き裂発生の力学条
件として腐食ピットを鋭いき裂とみなして求めた応力拡大係数幅が
有用である．図10･76 に，腐食疲労試験において，腐食ピット
底にき裂が発生しているものを中実印で，またき裂が発生していな
いものを中空印で示しているが，これよりき裂は腐食ピット深さが
ある限界値を超えたときに発生していること，負荷応力が小さいほ
ど腐食疲労き裂が発生する限界深さは大きくなる．き裂が発生して
いる腐食ピットを鋭いき裂とみなして，腐食ピット底の応力拡大係
数幅（Δ）を求め，これらの平均値を基に個々の腐食ピットの
アスペクト比の平均値に対して，応力振幅と腐食ピット深さから腐
食疲労き裂が発生する限界線を引いているが，これより腐食ピット
を鋭いき裂としてみなして求めた応力拡大係数幅により腐食ピット
からのき裂発生をほぼ推定できることがわかる．
さらに，き裂が進展すると参照環境中の疲労き裂進展速度に比較
して，き裂進展速度が加速したり，あるいは減速し，その挙動は複
雑である．減速因子としては，き裂の分岐やき裂内部に腐食生成物
や人工海水中のカソード防食条件下で生じる白色生成物などによる
き裂閉口［腐食生成物のくさび効果（wedge eﬀect，腐食生成物誘
起き裂閉口)，白色生成物誘起き裂閉口］が，また加速因子として
は腐食溶解や水素脆化割れが重畳することによる．

10・5・1衝撃強度
物体が衝突する場合のように，急激かつ短時間に作用する荷重を
静荷重と区別して衝撃荷重（impact load）という．衝撃荷重を受
ける物体に生じる衝撃応力は波動として伝ぱし，反射，屈折および
干渉を繰返すため応力場は時間的に変動する．衝撃荷重に対する材
料の強度を衝撃強度（impact strength）といい，多くの材料では
負荷速度の増大，すなわちひずみ速度が増大すると降伏強度や引張
強度も増大する．安全設計の基礎資料となる衝撃応力-ひずみ関係
を精密に決定するための標準衝撃試験法はいまだ規格化されていな

計測が可能であり，衝撃圧縮試験のほか，引張り，ねじり，せん断
試験などにも拡張使用されている．
