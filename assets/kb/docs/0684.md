# 10・5・1 衝撃強度

> **章**: 第10章 材料の強度
> **節**: 10・5 衝撃強度

物体が衝突する場合のように、急激かつ短時間に作用する荷重を静荷重と区別して衝撃荷重(impact load)という。衝撃荷重を受ける物体に生じる衝撃応力は波動として伝ぱし、反射、屈折および干渉を繰返すため応力場は時間的に変動する。衝撃荷重に対する材料の強度を衝撃強度(impact strength)といい、多くの材料では負荷速度の増大、すなわちひずみ速度が増大すると降伏強度や引張強度も増大する。安全設計の基礎資料となる衝撃応力-ひずみ関係を精密に決定するための標準衝撃試験法はいまだ規格化されていないが、ホプキンソン棒法(13・5・1項参照)によって比較的正確に計測が可能であり、衝撃圧縮試験のほか、引張り、ねじり、せん断試験などにも拡張使用されている。ひずみ速度が10s程度以下では、アルミニウムや銅などのfcc金属および亜鉛などにおいては、ひずみ速度が大きいほど、また温度が高いほど、降伏応力および衝撃強度のひずみ速度依存性が図10・77 アルミニウムの各ひずみ速度εにおける応力-ひずみ曲線図10・78純鉄の各ひずみ速度εにおける応力-ひずみ曲線図10・79モルタルの衝撃引張強度と静的強度の比較高くなる。図10・77にホプキンソン棒法による衝撃圧縮試験より得られたアルミニウムの室温における各ひずみ速度に対する応力-ひずみ曲線を示す。鉄やリチウムなどbcc金属では、降伏応力のひずみ速度依存性はfcc金属よりも顕著であり、また降伏現象が現れるのが特徴である。図10・78に室温における純鉄の衝撃引張試験結果の例を示す。他の材料、例えばセラミックスでは、ひずみ速度依存性が小さいものが多いが、コンクリートや岩石はひずみ速度に比較的敏感である。モルタルの強度に関してワイブル確率紙にプロットした例を図10・79に示す。ひずみ速度がおよそ10sの場合であるが、○印で示される衝撃引張強度は平均値で5.5MPaになっており、静的引張(割裂)強度3.6MPaの1.5倍強になっている。衝撃強度の評価方法として、応力のほか破断に要した衝撃エネルギーや動的破壊靱性も用いられている。試験片の破壊に費やされた衝撃エネルギーを求めるシャルピー試験は簡便な比較試験として工業的に広く用いられている。しかし、強度設計資料としては、エネルギー表示の結果がそのまま用いられることは少なく、応力表示および動的破壊靱性評価が望まれている。
