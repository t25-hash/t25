# 10・5・2 シャルピー衝撃値と遷移温度

> **章**: 第10章 材料の強度
> **節**: 10・5 衝撃強度

衝撃曲げ試験の一種のシャルピー試験(13・5・2項参照)において、試験片の破壊に費やされた衝撃エネルギーを試験片断面で除したものをシャルピー衝撃値(Charpy impact value)といい、これ3―137図10・80 各種炭素鋼のシャルピー衝撃エネルギー図10・81 シャルピー遷移曲線に及ぼす諸因子の影響を単に衝撃値と略称することもある。衝撃値が小さい試験片の破面は塑性変形が小さいために脆性破面となり、衝撃値が大きい場合は塑性変形が大きく延性破面が観察される。それゆえ衝撃値の大小は材料の脆性破壊の程度を知るための目安になると考えられ、耐衝撃性のほか、焼入れや焼戻し脆性などの熱処理の評価にも衝撃値が用いられる。図10・80は炭素量の異なる鋼の衝撃値と試験温度の関係を示したもので、衝撃値は温度の影響を受け、特に常温付近で急激に低下する傾向がある。このようにエネルギーが低下急変する温度を遷移温度(transition temperature)という。遷移温度が低いことは脆性破壊温度が低いことを意味する。このような鋼の遷移温度は材質や寸法などの影響を受ける。図10・81はシャルピー衝撃試験時の鋼の遷移現象が負荷速度、試験片寸法、切欠の鋭さなどで変化する状況を説明したもので、試験条件で靱性の評価が大きく異なるため、靱性の絶対的評価が難しい点が理解される。図10・81に見られるように、負荷速度の上昇により遷移温度が上昇し、また上部棚での吸収エネルギーは流動応力の上昇によって向上するのが一般的である。シャルピー衝撃値や遷移温度はあくまでも定性的比較値であり、破壊力学に立脚した定量的評価が行えない欠点がある。このため予き裂付きシャルピー衝撃片を用いた計装化シャルピー衝撃試験(13・5・3項参照)より動的破壊靱性を求める研究が進められている。
