# 10・5・3 熱衝撃

> **章**: 第10章 材料の強度
> **節**: 10・5 衝撃強度

物体の表面が急激に加熱または冷却されると、物体内部と表面の温度差によって、急激な熱ひずみおよび熱応力が発生する。この応力は一定時間で最大値に達し、これが材料によっては限界値を超えて破壊することがある。このように急激な非定常の熱応力が作用する現象を熱衝撃(thermal shock)という。発生する熱応力の大き、分布およびその時間変化は、その物体の物理的性質、形状寸法および加熱・冷却条件によって異なる。熱応力に対する材料の強度は破壊応力によって決まるので、熱衝撃はセラミックスのような破壊引張応力の小さい脆性材料で問題となる場合が多い。材料の熱衝撃強度を一義的に表すのは難しいが、以下のような各種の耐熱衝撃破壊パラメータで評価されている。熱衝撃係数(coefficient of thermal shock)は、板の熱衝撃3―138材料力学表10・4おもな破損形態と関連する力学量、材料特性ラチェット変形繰返し負荷塑性ひずみ破断延性疲労破壊繰返し負荷応力範囲、ひずみ範囲疲労強度クリープ破壊高温定常負荷応力、クリープひずみクリープ破断強度破壊に対する抵抗度を示す係数で、熱衝撃の程度を示すビオー(Biot)数β=γλが小さいとき=λσαで、またβが極めて大きいときは′=σαで表される。ここで、:板厚、γ:板表面の熱伝達率、λ:材料の熱伝導率であり、またσ:引張強さ、:縦弾性係数、α:線膨張係数である。熱衝撃に耐えうる最大温度変化は熱衝撃係数と1-νの積に比例する。ここで、νはポアソン比である。したがって、耐熱衝撃用材料としては、βが小のとき1-νが大きいもの、βが大のとき1-ν′が大きいものを選定すればよい。なお、上記は無限平板の場合であるが、無限円柱についても同様である。=1-ν′は熱衝撃破壊抵抗係数ともいわれ、セラミックスなど、材料中の微小き裂が熱衝撃によって成長し、それによって強度低下が問題となるときの熱衝撃破壊抵抗の評価に用いられている。以上の係数は材料の破壊に至るまでの耐熱衝撃性を表すが、熱衝撃損傷抵抗係数は破壊に至ってからの損傷の度合いを表し、クラックによる材料の崩壊やはく落などの損傷が重要となる場合に用いられている。また、熱応力が材料の破壊強度以下でも、環境によってはき裂が緩やかに成長するため、繰返し熱衝撃を与えると徐々にき裂が成長し、最後には熱疲労破壊が生じる。このときの抵抗性は熱疲労破壊抵抗という。
