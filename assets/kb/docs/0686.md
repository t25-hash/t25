# 10・5・3 熱衝撃

> **章**: 第10章 材料の強度
> **節**: 10・5 衝撃強度

物体の表面が急激に加熱または冷却されると、物体内部と表面の温度差によって、急激な熱ひずみおよび熱応力が発生する。この応力は一定時間で最大値に達し、これが材料によっては限界値を超えて破壊することがある。このように急激な非定常の熱応力が作用する現象を熱衝撃(thermal shock)という。発生する熱応力の大き、分布およびその時間変化は、その物体の物理的性質、形状寸法および加熱・冷却条件によって異なる。熱応力に対する材料の強度は破壊応力によって決まるので、熱衝撃はセラミックスのような破壊引張応力の小さい脆性材料で問題となる場合が多い。材料の熱衝撃強度を一義的に表すのは難しいが、以下のような各種の耐熱衝撃破壊パラメータで評価されている。ここで、:板厚、γ:板表面の熱伝達率、λ:材料の熱伝導率であり、またσ:引張強さ、:縦弾性係数、α:線膨張係数である。熱衝撃に耐えうる最大温度変化は熱衝撃係数と1-νの積に比例する。ここで、νはポアソン比である。なお、上記は無限平板の場合であるが、無限円柱についても同様である。=1-ν′は熱衝撃破壊抵抗係数ともいわれ、セラミックスなど、材料中の微小き裂が熱衝撃によって成長し、それによって強度低下が問題となるときの熱衝撃破壊抵抗の評価に用いられている。以上の係数は材料の破壊に至るまでの耐熱衝撃性を表すが、熱衝撃損傷抵抗係数は破壊に至ってからの損傷の度合いを表し、クラックによる材料の崩壊やはく落などの損傷が重要となる場合に用いられている。また、熱応力が材料の破壊強度以下でも、環境によってはき裂が緩やかに成長するため、繰返し熱衝撃を与えると徐々にき裂が成長し、最後には熱疲労破壊が生じる。このときの抵抗性は熱疲労破壊抵抗という。
