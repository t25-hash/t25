# 10・6・1 強度設計の目的と考慮するべき破損形態

> **章**: 第10章 材料の強度
> **節**: 10・6 強度設計

強度設計の目的は,所望の機能を発揮するように設計された機械やプラントが,想定された寿命の間,破壊することなく機能を維持することができるかどうかを強度面から評価し,必要に応じて設計変更を要求することである.機械の部品やプラント機器は各種の荷重を受け,さまざまな形態で破損する可能性があることが知られているため,これらの設計に当たっては使用中に予想されるすべての荷重下で,いずれの形態の破損も生じないようにすることが必要である.対象によって異なるが,以下のような荷重が考慮されることが多い.・内圧,外圧・遠心力(回転体の場合)・他機器から伝達される力(重力,振動等を含む)・熱応力・流体力・地震一方,破損の形態としては,以下が想定されることが多い.・過大負荷による塑性不安定破壊(延性破壊)・繰返し負荷による過大変形(ラチェット変形)・繰返し負荷による疲労破壊・高温状態での定常負荷によるクリープ破壊・クリープ疲労破壊このうち,前三者は,一般にどのような温度でも起こりうるが,後二者は,クリープ変形が生じうる高温でのみ問題となる.その他,使用環境によっては,腐食によるき裂の発生や進展の加速のような環境効果を考慮することが必要となる場合もある.機器の強度設計では,それぞれの破損が起きる条件を明らかにしたうえで,使用中に想定される負荷と使用する材料がもつ強度との比較によって,設計の妥当性が評価される.破損が生じる力学的条破損形態原因となる負荷関連する力学量おもに関連する材料特性塑性不安定破壊一方向負荷応力,塑性ひずみ降伏応力,引張強さ,破断延性クリープ疲労破壊繰返し+高温定常負荷ひずみ範囲,クリープひずみ疲労強度,クリープ破断延性件は,通常,応力やひずみを用いて表現されるため,使用中にこれらがどのような変化をするか予測するとともに,対応する材料強度特性を把握しておくことが必要である(表10・4).
