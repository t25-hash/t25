# 10・6・2 応力の分類とそれぞれに対する評価

> **章**: 第10章 材料の強度
> **節**: 10・6 強度設計

機器にはさまざまな要因で応力が発生する。一般的な応力源としては、外・内圧や遠心力、重力などの機械的荷重と、温度分布に伴って生じる熱応力を考慮する必要のあることが多い。また、多くの機器では地震荷重や振動荷重等、動的な荷重も考慮することが必要である。これらは、破損への寄与という観点から異なった特性を示すため、別々に扱う方が合理的である。また、応力やひずみの最大値のみならず、これらの分布も破損モードに影響を与える。このような見地から、圧力容器などの設計基準では、応力を機械的荷重による一次応力、熱応力に代表される二次応力および、応力集中分に対応するピーク応力の3種に分類し、それぞれに対する評価法が詳細に定められている。以下、それぞれの応力とこれらに対する制限の概要を示す。なお、多軸状態の等価応力としては、最大せん断エネルギー降伏説に基づく応力強さ(最大主応力と最大主応力の差)が用いられることが多い。a. 一次応力(primary stress)一次応力は、圧力や重力によって生じる応力で、構造物の変形によってほとんど変化しないので、荷重制御型応力とも呼ばれる。なお、機械荷重によるものでも、応力集中などが存在する場合には、これを除外した平均的な膜または曲げ応力が用いられる。その意味で、一次応力は個々の点での応力というよりは、軸力を断面積で除したり、曲げモーメントを断面二次モーメントで除すことによって求められる平均的な応力であるとみなすことができる。原子力機器に対する設計基準などでは、一次応力は、さらに一般膜応力、局部膜応力および曲げ応力に分類され、それぞれ独自またはその組合せに対して異なった制限値が与えられている。最も単純な一次応力は、内圧による一般膜応力であり、基準によって異なるが、通常運転状態では、降伏応力の2/3~9/10または引張強さの1/4~1/3よりも小さくすることが要求されている。さらに、局部膜応力+曲げ応力は、曲げ荷重下での崩壊荷重を考慮して、その1.5倍まで許容されることが多い。なお、起きる確率が低い事象に対しては、より高い許容応力が用いられる。また、クリープ変形が無視できない温度で長時間使用される機器では、これらの短時間破断強度のほか、使用温度や使用時間に対応した材料のクリープ破断強度に関わる応力制限をも満足しなければならない。b. 二次応力(secondary stress)機器に発生する応力としては、上記のような機械的荷重のほか、熱応力のように材料が変形することによって緩和する特性を示すものも多い。これは、機械的荷重に置換えて考えると変位制御型の負荷に近く、一次応力とは区別されて、二次応力として評価されることが多い。ただし、一般には二次応力として評価される熱応力でも、配管系の熱膨張反力のように一部分に対しては荷重制御的に働く場合があり(弾性追従と呼ばれる)、このような場合には保守的に一次応力として扱うか、変形の再分布を考慮した評価を行う必要がある。一次応力が低い値に制限されていても、一次応力と二次応力の和が材料の降伏応力を超えた場合、非可逆的な塑性変形が生じるが、特にこれらの応力が繰返された場合、塑性変形が一方向に蓄積していき、初期の形状から離れていく可能性がある。特に高い熱応力が繰返し作用する場合にこのような現象が生じることが多く、熱ラチェット現象と呼ばれている。熱ラチェットの生起は、一次応力と二次応力の組合せで決まり、図10・82に代表される評価線図も作成されている。しかし、熱ラチェット現象は、構造形状や応力の物質のほか、材料の変形特性によっても大きな影響を受け、精度良く評価するのが難しい現象の一つであるため、簡易評価の前提と異なる条件への適用に当たっては十分な注意が必要である。変形の制限はおもに機能的な観点から決定されるが、累積ひずみが疲労強度を低下させる場合もあり、高温原子力プラント設計基準では、断面内平均値、等価線形近似値、局所値に対して、それぞれ1%,2%,5%の制限が与えられている。c. ピーク応力(peak stress)一次応力と二次応力は、断面内の平均的応力(膜応力)および直線的に変化する成分(曲げ応力)に分離されるが、形状的な応力集中部や非線形な温度分布によって生じる応力の非線形成分については、別途ピーク応力として評価される。疲労による微視き裂の発生は局所的な現象であるため、一次応力、二次応力およびピーク応力のすべてを考慮することが必要となる。材料の塑性変形が無視できるような低応力での高サイクル疲労領域では、応力振幅をパラメータとした評価が可能であるが、高応力では、ひずみ範囲をパラメータとした評価が必要となる。特に後者の場合には、塑性変形によってひずみ範囲が拡大するため、この効果を適切に評価することが重要である。原子力プラトン用設計基準では、一次応力+二次応力強さの範囲が降伏応力の2倍を超えた場合に、ひずみ範囲の増加を考慮する方法が規定されている(図10・83)が、高温プラントに対しては熱応力を対象により柔軟性をもった評価法も開発されている。なお、疲労破損の場合には、発生するひずみ範囲に加えてその繰返し数が影響を及ぼし、疲図10・82 熱ラチェット評価線図の例(一定軸力と板厚方向線形温度こう配を受ける円筒に対する解、Bree線図)3―139労損傷度は、実際に想定する繰返し数をひずみ範囲に対応した許容繰返し数で割ることによって求められる。許容繰返し数は、繰返し数やひずみ範囲に対して安全係数を考慮して定められる。高い熱応力と高温保持が繰返されるような高温機器に対しては、定常運転中に働く応力によって生じるクリープ損傷を考慮したクリープ疲労評価を行う必要がある。クリープ損傷は負荷応力に対するクリープ破断時間の分数和として求めるのが最も簡単であるが、クリープ変形や損傷が応力変動によって加速されることがあり、このような場合には、応力ベースの評価法は非保守的な傾向を示すため、クリープひずみの精度良い推定とこれに基づく損傷評価法などを用いることによって、この効果を考慮することが必要である。なお、疲労やクリープ疲労によるき裂発生に対しては、環境、温度、負荷速度、応力多軸性、負荷履歴、寸法など多くの要素が影響を与えるため、これらに関する実機と材料試験の相違については十分な注意が必要である。また、長寿命の機器では、材料データの取得や評価法の検討が実機より短時間で行われることが多く、データや評価法の多挿性についても留意が必要である。さらに、実プラントでは、内包する流体の非定常な流れに起因する温度ゆらぎや、動的機器の運転によって生じる機械的振動などがき裂の発生・進展を引き起こす原因となることが少なくない。これらの現象は設計時点で精確に予測することが困難であり、過去の事故例などによってこれらの可能性が懸念される場合には、確認試験や検査を充実させることが重要と考えられる。また、製造過程で溶接が用いられる機器では、材料特性の違いや残留応力の存在などによって、母材よりも寿命が短くなる場合が少なくない。このようなことが予想される場合には、強度低減係数などによって母材に対する評価を補正することが必要である。
