# 10・6・3 安全確保と信頼性評価

> **章**: 第10章 材料の強度
> **節**: 10・6 強度設計

強度設計においては、各種の不確定性を考慮することが不可欠である。なかでも材料特性のばらつきはすべての機器の強度設計において無視できない問題である。また、地震荷重はその典型であるが、機器が受ける荷重の大きさについても不確定性がある場合が多い。さらに、実際の機器の使用経験から、強度に影響を及ぼすが評価上十分な精度で考慮できない因子が存在することが推察される場合がある。このような不確定性または不確実性に対して安全側に対応するため、安全係数(または安全率、safety factor)が用いられる。安全係数を用いた設計では、以下の関係を満足することが要求される。材料強度>(安全保数)×(負荷強度)この場合の材料強度および負荷の大きさとしては、平均的な値が用いられる場合と、強度の下限や荷重の上限が用いられる場合がある。安全係数や強度、荷重の上下限値は工学的判断で設定されるが、その妥当性を客観的に示すことが困難であることから、より合理的な方法として、破損確率評価に基づく強度評価法が開発されてきており、構造信頼性工学(structural reliability engineering)と呼ばれている。この場合、破損確率は、材料強度と負荷の変図10・84 強度、負荷の分布と破損確率の関係図10・83 ひずみ範囲増倍係数評価線図の例3―140材料力学動から図10・84のようにして評価される。両者の最確値の比である安全係数を大きく設定するほど、破損確率を減少させることができる。このような手法を適用するためには、強度と荷重のばらつきの程度を数式で表す必要がある。各種材料特性のばらつきを表す確率分布関数としては、正規分布、対数正規分布がよく用いられるが、脆性的な材料に対しては最弱リンク説に基づくワイブル分布が用いられることも多い。なおこの方法を適用するためには、精確な破損条件と材料強度、荷重の統計的性質に関する十分な知見を有することが不可欠である。また、許容される破損確率については、安全性への影響、経済性への影響などを考慮して総合的に決定する必要がある。
