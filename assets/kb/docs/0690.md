# 4・2・2 トラクション（traction)

> **章**: 第4章 摩擦
> **節**: 4・2 転がり摩擦

転がり接触と滑りが共存した状態で，接触する2面に微小な速度
差がある場合には，それに起因して摩擦力が生じる．その摩擦力
をトラクションと呼び，通常，純粋な転がり摩擦よりも大きい
値を示す．その例としては，トラクションドライブによる動力の伝
達，鉄道における駆動輪とレールの間の相互作用などがある．
図4･5のような二つの円筒が，接触面に垂直抗力 を受け，わ
ずかに滑りながら回転している場合を考える．摩擦係数の場合と同
様に，μ＝をトラクション係数（coeﬃcient of traction）と
呼ぶ．二つの円筒の周速の平均値を ，速度差をΔとすれば，
μは滑り率Δの関数となる．
乾燥摩擦の状態でのトラクション係数μは，Δの増加とと
もに増加し，滑り摩擦における摩擦係数μに漸近する．また，接
触面において転がり摩擦と滑り摩擦が共存している状態では，
Δが増加すれば，滑りの割合が増加していき，トラクション
係数がμに一致したときには，接触面全体が滑り出す．
一方，潤滑状態でのトラクションでは，μはΔの増加に伴
いあるところまでは増加するが，それ以降は温度上昇による粘性低
下のために，徐々に減少していく．
