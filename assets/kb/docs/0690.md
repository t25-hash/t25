# 10・6・4 詳細解析の適用

> **章**: 第10章 材料の強度
> **節**: 10・6 強度設計

計算機性能の飛躍的な向上によって、現在、有限要素法を用いた構造解析プログラムによって大規模な三次元解析を実施することはさほど困難ではなくなってきている。一方、上述した応力分類に基づく強度評価法はベースとして単純な軸対称構造を想定しており、評価断面が自明でないなどの理由により、複雑な形状をもった機器に対して得られた解析結果に適用するのが容易でないことも多い。このため、近年、応力分類を行わずに、三次元解析の結果から直接的に強度評価を行う手法も検討されている。特に材料の硬化を無視した弾完全塑性体を仮定した応力解析を行えば、塑性不安定破壊に対する裕度を保守的に評価することが可能である。さらには、材料の塑性、クリープなどの非弾性変形挙動を精度よく表現する構成モデルを用いた非弾性解析によって、直接、変形挙動や応力やひずみ、損傷の分布を求めることも可能である。これによって、任意の機器に対してより信頼性の高い評価を与えられることから、構成モデルの高精度化や計算機性能の向上に伴って、今後さまざまな分野での適用が進められることが予想される。
