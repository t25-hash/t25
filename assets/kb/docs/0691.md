# 10・6・5 破壊力学の適用

> **章**: 第10章 材料の強度
> **節**: 10・6 強度設計

近年、き裂状の欠陥が存在したときの強度を評価する手法として、破壊力学(fracture mechanics)が発達してきた。通常の強度設計では、初期欠陥の存在を仮定せず、き裂発生防止を主眼とした評価が行われるが、溶接部などでは製造時に欠陥が生じる可能性が無視できない場合があり、安全上重要性の高い機器では、バックアップとして破壊力学を用いた強度評価が行われることもある。また、供用中に検査などで欠陥が発見された場合にも、機器取替の必要性を検討するために破壊力学が用いられる。破壊力学は、脆性破壊の評価から始まったが、非線形破壊力学の進歩によって、現在では、疲労き裂進展、クリープき裂進展、延性破壊などさまざまな形態のき裂進展の予測に適用できるようになっている。破壊力学を用いた強度評価では、初期欠陥を仮定し、疲労、クリープなどの時間依存性き裂進展を予測しながら、各時点で脆性的または延性的な破壊が生じるか否かを評価する。これらの評価に当たっては、応力拡大係数、積分などの破壊力学パラメータの評価が不可欠となる。き裂をモデル化した機器に対する詳細な数値解析を用いてこれらを評価する手法が確立されているほか、ハンドブックを用いた評価など、簡易評価法の整備も進められている。前述の構造信頼性工学と同様、各種の不確定性を考慮する確率論的破壊力学も開発されている。破壊力学の対象は主として巨視的なき裂であるが、通常は発見されないような1mm以下の微小なき裂に対しても、破壊力学が適用できることが確認されており、微小き裂発生寿命の評価に対しても破壊力学を活用できることが明らかとなっている。これによって、通常は半経験的にしか評価できない寿命に対する各種効果が、き裂進展速度に対する効果としてより明確にとらえられ、強度評価の信頼性向上に資することが期待できる。文献(1) 中沢一・小林英男、固体の強度、(1976)、146、共立出版。(2) Kelly,A.、ほか2名、Phil.
