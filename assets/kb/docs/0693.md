# 11・1・2 エネルギー解放率

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・1 線形弾性破壊力学

エネルギー解放率(energy release rate)の概念は、き裂が進展するためには、き裂進展に伴って解放されるエネルギー(考えている力学系より失われる有効なエネルギー)が新破面形成のために必要とされるエネルギー(表面エネルギー、surface energy)より大きいことが必要であるとするグリフィス(Griffith)の脆性破壊の理論にはじまり、破壊の物理的背景を与える重要なパラメータとして応力拡大係数とともに破壊力学における基本的な役割を果たす概念とされる。エネルギー解放率はき裂進展に際して対象とするき裂材-荷重系より失われる力学的に有効なエネルギーを、き裂の単位面積当たりの進展に対して表したものと定義される。静的なき裂の進展を考えており、dだけき裂面積が増大する間に、dは外力のする仕事、dは弾性ひずみエネルギーの増大分である。a. エネルギー解放率と応力拡大係数き裂が進展するとき、き裂進展前に図11・1のθ=0の面に作用していた応力は新たなき裂表面で零となり、その結果として新たなき裂表面の上下には相対的な開口変位が生じてエネルギーを放出する。したがって、き裂が進展するときの弾性ひずみエネルギーの増大分dはき裂が進展する間に外力がする仕事dからこのき裂先端でのエネルギーの放出分を差引いたものとなり、式(11・16)を参照するとこのエネルギー放出分がき裂進展に伴い解放されるエネルギーdであることがわかる。き裂進展前にθ=0の面に作用していた応力は式(11・3)(11・5)(11・7)においてθ=0とすることにより、またき裂進展後の開口変位は式(11・4)(11・6)(11・8)においてθ=±πとすることにより求まる上下面の変位の差として与えられるので、これらを適用することにより応力拡大係数とエネルギー解放率の間に成り立つ次の関係が得られる。b. エネルギー解放率とコンプライアンス図11・22に示すような、き裂材(線形弾性体)が荷重を受け、荷重点変位が生じている状態からのき裂進展を考える。荷重と荷重点変位の間には図11・22 負荷を受けるき裂材図11・21弾性体中におけるき裂の進展図11・23 き裂端近傍の塑性域λをコンプライアンス(compliance)として=λ(11・18)が成り立つ。特に単一変形モードの場合、式(11・17)と併せ考えることによりコンプライアンスより応力拡大係数が、逆に応力拡大係数よりき裂材のコンプライアンスが評価できる。
