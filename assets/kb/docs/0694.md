# 11・1・3 小規模降伏

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・1 線形弾性破壊力学

応力拡大係数は,線形弾性体における,図11・2の各変形様式に対して現れるき裂端近傍の特異応力場の強さを一義的に代表するパラメータとして意味をもつ.この線形弾性特異応力場は応力・ひずみが比較的小さい間に成り立つフックの法則が,応力・ひずみがいくら大きくなっても成り立つとして得られるものであり,実際の材料においては応力・ひずみの増加とともに降伏現象に代表される非弾性挙動が現れるので,その状況は線形弾性特異応力場とは異なってくる.図11・23はモードIの変形様式のもとで現れる塑性変形領域を模式的に描いたものである.Ω욣は塑性変形領域,Ω욄は線形弾性体を想定して,応力場が式(11・1)の第1項[いまの場合,具体的には式(11・3)の特異応力場]のみでほぼ表しうる,すなわち応力拡大係数で場を代表できると考えられる領域である.図11・23において,塑性域Ω욣は負荷の増大ともに広がっていき,それとともにΩ욄の境界あたりでの応力状態は線形弾性体としての特異応力場から外れていくが,Ω욣の領域がΩ욄に比べて十分小さいときにはΩ욣の存在はΩ욄の境界あたりでの状況にはほとんど影響を及ぼさない.したがってこのような状況においては,応力拡大係数はΩ욄内の弾塑性変形状態を代表するパラメータになっていると考えられる.すなわち同じ材料からできた部材に存在するき裂であるならば,き裂長さとき裂材の形状,負荷条件によらず,それらを線形弾性体とみなして評価される応力拡大係数が同じ値をとるとき,いずれのき裂先端近傍においてもほぼ同一の弾塑性変形状態が生じていると考えられることになる.上のような条件が満たされている状態を小規模降伏(smallscale yielding)という.通常,小規模降伏の条件はθ=0方向の塑性域寸法욣のき裂長さまたは部材寸法に対する割合で表され,相対的にそれが十分小さいとき小規模降伏は実現される.この十分小さいときという表現はあいまいであるが,塑性域が存在しても部材の全体的挙動が弾性的である場合,具体的には例えば荷重-荷重点変位の関係がほぼ線形とみなしうる範囲では,多くの場合,応力拡大係数は有効であり,線形弾性破壊力学ではこのような条件が満たされる範囲でのき裂の挙動を対象とする.11・3・1項で示すように,破壊靱性の評価においては小規模降伏を満たすための具体的な条件が与えられる.
