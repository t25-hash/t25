# 11・1・4 ダグデールモデル

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・1 線形弾性破壊力学

き裂先端近傍に塑性域が生じた状態を線形弾性解の重ね合わせで近似するものとして提案されたモデルである。長さ2のき裂を有する無限板が遠方で一様引張応力σと図11・24においてで表現される部分のき裂内面に降伏応力σに等しい応力がき裂を閉じるように作用する場合を例に考える。このとき、の幅を調節することにより、き裂先端B での特異性が消えるように、すなわち、二つの荷重による応力拡大係数が相殺されるようにすることができ、その状態では、き裂を含む面上において図(Dugdale model)は、で表現される黒塗り部分を長さ2のき裂が存在する場合の塑性域とみなすモデルであり、平面応力状態の場合に当てはまる。ダグデールモデルの解析では2が実際のき裂長さであり、応力σに対応して仮想き裂先端B での応力拡大係数が零となるように塑性域寸法(仮想き裂長さ2)を求めることになる。ここで考えた無限板に1個のき裂が存在する場合の諸量の関係は次式で与えられる。塑性域の大きさが線形弾性特異応力場による応力拡大係数の支配域に比べ十分大きくなり、いわゆる小規模降伏条件を逸脱した大規模降伏条件下では、応力拡大係数を用いてき裂先端近傍の応力・ひずみ場を表すことはできない。このため、弾塑性破壊力学が用いられる。
