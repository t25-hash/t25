# 11・2・1 弾塑性特異応力場と積分

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・2 弾塑性破壊力学

a. エネルギー解放率としての積分ポテンシャルエネルギーをΠ で表したとき、線形弾性体に対するエネルギー解放率を、応力とひずみの間に非線形関係がある場合に拡張し、き裂の単位面積成長当たりのΠ の解放率を表すものが、積分(-inte-gral)である。この式を式(11・21)に代入すると、き裂先端近傍の塑性の特異応力・ひずみ場は、次のようににより表現できる。b. 積分の経路積分表示式(11・22)のエネルギー的関係より、物体力、慣性力と初期ひずみがないときの積分は、図11・25のようにき裂先端を囲む経路Γに沿って評価した次式の線積分で表される。は積分経路Γのとり方に依存しないという、経路独立性(path-independence)を有している。このため、破壊進行域の大きさが弾塑性特異場の支配領域より十分小さい必要がある。もう一つの問題は、非線形弾性体に対して定義された積分が、通常の弾塑性体の挙動を表す増分塑性に対しては、ひずみエネルギーの消散のためエネルギー解放率としての意味を失うことである。しかし、除荷がない間は増分塑性体と非線形弾性体の応答にほとんど差はなく、したがって積分はき裂先端の特異応力・ひずみ場の強さを表している。基本的にき裂進展があり、除荷の影響が大きい場合には、積分を適用することができない。しかし、き裂進展に伴い抵抗が急速に増大する場合には、除荷域の影響が相対的に小さくなり、積分を近似的に適用することができる。これを支配き裂成長(-controlled crack growth)という。疲労によりき裂が進展する場合にも、き裂の後方に除荷域ができる。小規模降伏のもとでは、き裂先端近傍の応力とひずみの変動範囲は、応力拡大係数範囲(stress intensity factor range)と対応付けられる。大規模降伏条件下では、繰返し応力-ひずみ関係が安定している場合には、き裂開口点を基準にすることにより、き裂先端近傍の応力とひずみの変動範囲を積分範囲(-integralrange)、と対応付けることもできる。d. 積分の評価法積分の評価法としては、解析的評価法、簡便評価法および数値的評価法がある。積分を評価するための解析的な式はあまり報告されていない。ここに、εはσに対応するひずみである。このようには代表応力、代表ひずみと代表寸法の積の形に表される。の近似式としては次の二つがある。変位制御のもとでは、ポテンシャルエネルギーΠは、図から求められる。このことより次の関係式が得られる。したがって、曲げを受ける深い片側き裂平板では、η=2(11・38)である。コンパクト試験片(CT 試験片)では引張りの効果を考慮した次式を用いてηを評価することができる。形状と負荷条件が複雑な場合には、有限要素法などによる応力・ひずみの数値解析結果があれば、き裂先端より離れた経路に対して式(11・27)の経路積分を実行することにより、が評価できる。式(11・22)に基礎をおく仮想き裂進展法もよく用いられている。e. 弾塑性破壊力学の役割弾塑性破壊力学により、材料が線形挙動を示さない場合、あるいは塑性域の大きさがき裂長さと同程度以上となるような大規模降伏および全面降伏条件に対して、き裂の挙動を評価することが可能となった。弾塑性破壊力学の別の大きな役割は、大きな試験片でないと求められない小規模降伏条件下の破壊特性が、小さな試験片で求められることである。この考え方は、平面ひずみ破壊靱性の評価に用いられている。図11・29のように、小規模降伏条件下および大規模降伏条件下にある試験片を考え、双方で破壊が開始しようとしているものとする。小規模降伏条件下では、弾性特異場に着目すればが適用でき、またその中の弾塑性特異場に着目すればが適用できる。これらのパラメータで表現した破壊靱性,の間には、例えばモードIのもとでは式(11・23)より次式が成立する。したがって、大規模降伏条件のもとで求めたを式(11・41)に代入することにより、小規模降伏条件下でないと求められないが求められる。
