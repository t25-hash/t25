# 11・2・2 積分とき裂先端開口変位CTOD との関連

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・26 のように外力のした仕事に等しく、-線図の下の面積

き裂先端開口変位CTOD,δ욧も、き裂先端近傍の力学状態の代表値であり、弾塑性破壊力学パラメータとして用いられる。δ욧と積分の関係は、σを降伏応力として、=σδ욧(11・42)で表される。ここに、は定数であり、ダグデールモデルでは=1、すべり線場理論を用いたき裂の鈍化解析では=1.5となる。安定進展するき裂に対しては、図11・30に示すき裂先端開口角、3―149図11・28 深いき裂を有する曲げ試験片、CT 試験片、中央き裂試験片、両側き裂試験片に対する積分およびの簡便評価法CTOA(crack tip opening angle)が用いられることがある。き裂成長条件としてCTOA 一定の条件が用いられている。
