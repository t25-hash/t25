# 4・4・1 初期たわみのある長柱

> **章**: 第4章 柱の圧縮・座屈
> **節**: 4・4 初期たわみおよび偏心荷重を受ける長柱

一般に微小な初期たわみ（initial deflection） を有する長の問題は，平衡方程式が＋＋＝0となるのすぐな柱の問題に帰着される．
したがってその解法は，4･3節のそも同様であって， がオイラー座屈荷重 に近づくとそれらは急する．
この種の柱の強度については，4･5節参照．
柱に軸圧縮荷重 が作用する場合，弾性域・微小たわみの範囲でで，分布荷重＝－が作用する同一端末条件のまっれと同様であり，生じるたわみ，曲げモーメント などの性質速に増大する．
