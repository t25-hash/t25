# 11・3・1 種々の材料の破壊靱性

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・3 破壊靱性

破壊靱性とは材料の破壊に対する抵抗値の総称である。理論的には弾性状態でき裂先端応力場は無限大になり、また小規模降伏状態、大規模降伏状態でもき裂先端には応力特異性が存在し、き裂先端での応力場は極めて大きくなる。それでも材料がすべてただちに破壊に至らないのは、材料固有の破壊に対する抵抗力が存在するからである。それらを破壊靱性といい、これは材料固有の値という意味で材料定数である。3種類の破壊モードに対応してそれぞれの破壊靱性が定義されるが、以下では実用上最も重要なモードIについてのみ説明する。a. 完全脆性材料の破壊靱性グリフィスは、図11・31に示すように、上下両端を固定された、長さ2のき裂がある板が引張りを受ける場合を考えた。この状態からき裂進展により新しいき裂表面が形成されるとき、物体中のポテンシャルエネルギーの変化量が新しい表面を形成するのに必要な表面エネルギーより大きくなるときにき裂が進展するものとした。この関係は次式で表現できる。ここで、2γとなっているのはき裂の上下面にき裂面が生じるためである。これより破壊条件として次式を得る。しかし、この理論は必ずしも表面エネルギーのみを破壊に必要なエネルギーとする必要はない。例えば、へき開破壊の際のへき開段の形成や、き裂先端でのわずかな塑性変形に要するエネルギーを加えて、γの代わりに有効表面エネルギーγを用いることでそうした問題へのこの理論の拡張が試みられてきた。それらを一般化すると次式の破壊条件式が得られる。これがグリフィスの理論に端を発するエネルギー論に基づく破壊条件式である。b. 小規模降伏状態の破壊靱性小規模降伏状態でのき裂先端近傍での変位、ひずみ、応力等の力学量は、応力拡大係数により一義的に記述できる。したがって破壊を力学的に論じるなら、破壊条件はこの唯一の支配量を用いて次式のように記述されることとなる。がと等しくなったとき生じる破壊は完全脆性材料のときと同様、不安定破壊である。エネルギー解法率と応力拡大係数との間には式(11・23)の関係があることが示されている。このように、式(11・45)と式(11・46)とは等価であり、と、またととは1対1の対応関係にある。図11・32に示すように、見掛け上の破壊靱性はき裂長さ、板厚などの試験片寸法に依存し、特に板厚の増加とともに低下し、ある板厚以上で一定値を示す。これは試験片の厚さ方向に沿って、表面層を除いてほとんどの領域で平面ひずみ条件が満足されたときに対応している。これは材料が示す最小の破壊抵抗と考えられ、寸法に無関係な材料定数である。c. 大規模降伏状態の破壊靱性き裂先端に大きな塑性域が生じ、小規模降伏条件が成立しない状態を大規模降伏という。このような大規模降伏後の破壊では、不安定破壊に先立ってゆっくり破壊が進行する安定破壊段階が存在する。この場合の破壊パラメータとしては積分が用いられる(積分の定義については11・2・1項参照)。積分が破壊パラメータとなるためには、大規模降伏状態でのき裂先端の変位、ひずみ、応力が積分のみをパラメータとして記述できることが必要条件である。その条件を満たす領域をHRR 場という。図11・33に模式的に示すように、この場合、き裂先端のごく近傍には破壊が実際に進行しているプロセスゾーン(強変形域、破壊進行域)、その外側にHRR 支配の場、さらにその外側に応力拡大係数支配の場が存在している。プロセスゾーン内は連続体力学の適用できない領域であり、き裂先端開口変位(CTOD)をδとして、き裂先端から約2δの領域であるとされている。HRR 場がこの外側に十分の大きさで存在していれば、このときの安定破壊の開始条件は次式で表される。小規模降伏の場合と同様、弾塑性破壊靱性も試験片寸法、負荷形式に大きく依存する。平面ひずみ条件が満足されるときの値が平面ひずみ弾塑性破壊靱性となり、これが材料定数とみなされる。,それぞれの平面ひずみ破壊靱性の寸法規定と測定法については同様の弾塑性破壊パラメータとして、き裂先端開口変位がある。これは図11・33に示すように定義され、積分と式(11・42)を拡張した以下の関係にあることが知られている。δはき裂先端の量であるから測定は簡単ではないが、き裂材の表面でのき裂開口変位(COD)とCTOD の間には良い対応関係があるから、実用的にはCOD を破壊基準として使用することが可能である。同一材料の小型試験片を用いて大規模降伏状態で得られたと、大型試験片を用いて小規模降伏状態で得られたとの間には延性引裂き破壊のみを対象として式(11・41)の関係が成立する。すなわち、小型の試験片で大型部材のが評価できる。
