# 11・3・3 き裂先端の拘束効果

> **章**: 第11章 破壊力学と界面力学
> **節**: 11・3 破壊靱性

윪,윪などの破壊靱性は,き裂先端での力学的環境がやなどで一義的に記述できるいわゆる支配場,支配場が存在するときに有効な破壊条件を与える.これは平面ひずみ状態で満足されることが多い.しかし,近年より複雑でかつ小型の機器に破壊力学が適用されるようになると,き裂先端の応力状態は平面応力,または三次元応力状態となり,平面ひずみ破壊靱性値より見掛け上大きな破壊靱性を示すことになる.これらはき裂の拘束効果といわれる.実用的にこれらの環境で妥当な破壊靱性を評価,予測しようとする研究が行われている.その一つとして,最弱リンク説に基づき,き裂先端領域でのワイブル応力分布を考慮したローカルアプローチが,実用的な予測法として有効に利用されている.
