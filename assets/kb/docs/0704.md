# 12・1・1 複合材料の特質

> **章**: 第12章 複合材料の力学
> **節**: 12・1 複合材料の特質

複合材料は2種類以上の基材を組合わせて元の基材になかった特性を生み出した人工の材料で,しかも出来上がった後,元の基材のアイデンティティが残っているものをいう.力学的に見たときの複合材料の特質は,(1)軽くて強い,(2)設計できる材料,の2点で,ほかに(3)大型の製品が一体成形できる,(4)耐食性があり,保守が少なくて済む,(5)基材の組合せによって各種機能的特性を付与できる,などの特質がある.(1)を表す指標として比強度(=材料の強さ/単位体積当たりの重さ,specific strength)が定義されているが,図12・1にそれを比弾性率(=ヤング率/単位体積当たりの重さ,specific modulus)とともに示す.金属では4の壁といわれ,比強度は4×10m,比弾性率は4×10mをなかなか超えられないが,複合材料では容易に突破している.
