ある．炭化ケイ素，アルミナ繊維なども特殊用途に使用される．
12・1・3母材の種類
強化される側（母材）は樹脂，金属，セラミックスなどで，でき
あがった複合材料はそれぞれPMC（polymer matrix composites，
高分子基複合材料），MMC（metal matrix composites，金属基複
合材料），CMC（ceramic matrix composites，セラミック基複合
材料）などと呼ばれる．繊維で強化したものはFRP（fiber rein-
forced plastics，繊維強化プラスチック），FRM（fiber reinforced
metal，繊維強化金属），FRC（fiber reinforced ceramics，繊維強
化セラミック）である．ほかにコンクリートにスチール繊維を混入
したものや，タイヤのようにゴムにコードを入れたものも複合材料
