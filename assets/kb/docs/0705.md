# 12・1・2 複合材料の強化形態

> **章**: 第12章 複合材料の力学
> **節**: 12・1 複合材料の特質

複合材料はかなり広い概念であり、種類も豊富である。まず図composite materials(先進複合材料)の略号である。図12・1 複合材料の比強度、比弾性率図12・2 複合材料ラインアップ力学的には強化材が大きな役割を果たす。この強化材の形態には、(1)粒子(例:SiC粒子)、(2)フレーク(例:マイカフレーク)、(3)繊維(例:ガラス繊維)などがある。構造材料としての複合材料の強化材には繊維を用いることが多い。繊維はさらに(3-1)不連続繊維、(3-2)連続繊維に分けられ、不連続繊維にはガラスマットのような長さが25~50mmレベルのものと、射出成形材のように1mm以下に粉砕されたものがある。SMC(sheet molding compound)、BMC(Bulk molding com-pound)は半硬化樹脂に短く切った繊維を練りこんだ中間基材である。また図12・2中のHLU はハンドレイアップ、FW はフィラメントワインディングで、ともに成形法の名称である。連続繊維の場合には、どの方向に繊維を配置するか、繊維がまっすぐか織られたり編まれたりしているかによって、一方向連続繊維、織物、多軸織物、ニットファブリック、編物(ニッティング)、組み物、三次元織物などに細分化される(図12・3)。織物にも多くの種類があり、ここでは代表的な平織りクロスとしゅす織りクロスを示している。織物は繊維がうねっているのに対して、ニットファブリックは繊維の束をポリエステル糸等でかがったもので、繊維束がまっすぐなため性能がよく、今後の発展が期待される。また、薄いシートを何枚も重ねた(積層)ような場合、板厚方向に強化繊維がなくはがれやすいので、あらかじめミシンで縫いとり(ステッチング)をし、はく離を防ぐ技術も開発されている(図12・4)。複合材料の強化繊維には、ガラス繊維(glass fiber)が最も多く使われ、さらに高性能が要求される航空・宇宙分野やスポーツ用品では炭素繊維(carbon fiber)、アラミド繊維(有機繊維)なども使われる。炭素繊維には出発原料によりPAN 系、ピッチ系などがある。炭化ケイ素、アルミナ繊維なども特殊用途に使用される。
