# 12・1・3 母材の種類

> **章**: 第12章 複合材料の力学
> **節**: 12・2 にその概要を若干の成形法とともに示す.ACM はadvanced

強化される側(母材)は樹脂,金属,セラミックスなどで,できあがった複合材料はそれぞれPMC(polymer matrix composites,高分子基複合材料),MMC(metal matrix composites,金属基複合材料),CMC(ceramic matrix composites,セラミック基複合材料)などと呼ばれる.繊維で強化したものはFRP(fiber rein-forced plastics,繊維強化プラスチック),FRM(fiber reinforcedmetal,繊維強化金属),FRC(fiber reinforced ceramics,繊維強化セラミック)である.ほかにコンクリートにスチール繊維を混入したものや,タイヤのようにゴムにコードを入れたものも複合材料とみなされる.ただし,加硫ゴムや鉄に炭素を入れた鋼(合金)な図12・3 強化材の形態図12・4 ステッチングの例どは,元の原料の特性が残っていないので,複合材料とは呼ばない.プラスチックは大別して熱硬化性樹脂(TS : thermosetting)と熱可塑性樹脂(TP: thermoplastic)に分けられる.熱可塑性樹脂を用いた場合にはFRTPと呼ぶ.複合材料に用いられる熱硬化性樹脂には不飽和ポリエステル(UP),ビニルエステル(VE),ビスマレイミド(BMI),エポキシ(EP)などがあり,はん用のものにはUP,高性能品にはEPが多用される.フェノール樹脂は難燃性があり,建材や車両用に適用が検討されている.熱可塑性樹脂もナイロン等,複合材料に用いられる.ポリイミド(PI)は耐熱樹脂で,短時間なら450°Cに耐えられ,PEEK(ポリエーテルエーテルケトン,耐熱TP)とともに宇宙・航空分野での適用検討がなされている.金属母材では,アルミニウムの検討が最も多く,強化繊維としては炭素繊維,炭化ケイ素繊維などがある.チタンをボロン繊維等で強化する試みもなされている.一方向凝固共晶合金というのがあり,ある成分が繊維状になり,別の成分がそれを取り巻くという形態をとり,複合材料の仲間に入れられるが,母材という概念はあまりない.これはin-situ コンポジットとも呼ばれる.図12・2中の金属間化合物を複合材料と呼ぶかどうかは微妙である.セラミック母材では,アルミナ(AlO)や炭化ケイ素(SiC)がある.ガラス母材もこのカテゴリーに入れることが多い.C/C複合材料はCFRPを蒸し焼きにして,母材のプラスチックも炭素化したもので,酸素のない雰囲気では究極の耐熱材料で,宇宙往還機やエンジンの分野で期待が寄せられており,レーシングカーのディスクブレーキなどにも使われている.同じくSiC/SiC複合材料も耐熱用途である.なお,A/B と表記した場合,A が強化材(繊維),B が母材である.
