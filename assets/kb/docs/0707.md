# 12・1・4 界面(interface)問題

> **章**: 第12章 複合材料の力学
> **節**: 12・2 にその概要を若干の成形法とともに示す.ACM はadvanced

強化材,母材に加えて,両者の界面が複合材料の特性を支配する.一般には界面の結合が強いほうがよい.そのため繊維に表面処3―155図12・5 シランカップリング剤の結合モデル図12・6 炭素繊維/樹脂界面理を施す.ガラス繊維ではカップリング剤(coupling agent)というものを用いることが多い.図12・5はガラス繊維と樹脂のシランカップリング剤による結合モデルで,両者は化学結合している.炭素繊維の表面には官能基がほとんどなく,樹脂と化学結合しない.そこで表面に電解酸化,オゾンやプラズマによる処理を施し,樹脂との接着性を上げ,サイジングを施す.図12・6は炭素繊維/樹脂界面付近のモデルで,図12・7は酸素プラズマ処理によるCFRP一方向強化材の横方向強さの向上例である.酸化処理により繊維の強さは若干低下するが,横方向の強さの向上は著しい.ただし,界面の結合を強くしすぎると複合材料としての引張りや曲げ強さがかえって低下するという報告もあり,どこかに最適な条件がある.3―156図12・7 表面処理とCFRPの強さ以上はPMCの場合であるが,CMCでは母材のセラミックスの破断伸びは一般に小さく,繊維より先に母材にき裂が入る.その際,界面が強いと,き裂はそのまま繊維を横切ることになり,強さあるいは破壊靱性の向上につながらない.むしろ界面が弱いほうが,繊維が引抜けるため,破壊に余分のエネルギーが必要となり,特にCMCでは界面をいかに弱くするかが技術的課題となる.CMCあるいはMMCでは高温で成形することが多く,繊維と母材の化学反応が起こる.この反応生成物は往々にしてもろく,有害な反応生成物を抑えることも界面に関係した課題である.
