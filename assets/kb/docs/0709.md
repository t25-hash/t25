# 12・2・2 繊維直角方向弾性率とせん断弾性率

> **章**: 第12章 複合材料の力学
> **節**: 12・2 節では、長繊維強化材料の最も単純な構成である、一方向

次に、繊維垂直方向の引張モデルとして図12・10を考える。基本的な仮定として、繊維とマトリックスの応力σは共通であり、かつ、どちらも等方性弾性体とする。ただし、ひずみは繊維とマトリックスとでは異なる。なかでも、半経験式であるHaipin-Tsaiの式がよく使われている。ξは未知量であり、数値計算結果と一致するように決められる。また、以上からも明らかなように、繊維体積含有率は、複合材料の力学特性を決定する重要なパラメータである。一方向連続繊維強化材について、断面内に詰められる繊維の量には上限が存在する。図12・12に示すように、繊維断面を円形と仮定すると、正方配列では0.785、六方配列では0.905が上限である。しかし実用材料では、一方向繊維強化材については=0.5~0.6程度が通常である。
