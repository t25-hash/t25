# 12・2・3 織物材等の力学特性

> **章**: 第12章 複合材料の力学
> **節**: 12・2 節では、長繊維強化材料の最も単純な構成である、一方向

ここでは、長繊維強化複合材料の一種である織布複合材料(woven composite)の弾性定数を求めるための理論について、概説する。二次元織布は、図12・13のように経糸(warp)と緯糸(fill)によって構成され、交差点(織り目)の繰返し数によって分類される。点線で囲まれた部分は各パターンの最小繰返し領域を表す。このような二次元織布の弾性定数を求めるための、いちばん簡単なモデルはモザイクモデルである。これは実際の織物の糸のうねりおよび連続性を無視して、二層の直交積層板の集合体とみなすものであり、例えば=8の場合には、図12・14のようにモデル化される。このようなモデルに対し、12・5・1項で説明される古典積層理論を適用し、一様ひずみまたは一様応力の仮定を導入することで、弾性係数の上下界が求められる。この方法は、織布弾性係数の粗い推算に便利である。次に、クリンプモデルは糸のうねりおよび連続性を考慮したモデルである。板厚と経緯糸の糸厚の和とが異なる場合は、純マトリックス層が表面にでるように仮定する。このモデルの解析においては、古典積層理論が、dの無限小長さをもつ板の一部分にも適用できるという仮定を設ける。そして緯糸のうねり部分のoff-axis異方性を考慮し、このモデルに巨視的面内膜応力を作用させることで、全体の弾性係数が求められる。これを克服するため、ブリッジングモデルが提案された。図12・16(a)は=8の繰返しパターンの最小形状であり、(b)はその形状を等積的に簡略化した正方形である。その正方形は(c)のように分割され、領域Cだけに糸のうねりを考慮し、領域B,C,D は同一の平均基準面ひずみおよび曲率をもつと仮定する。うねりをもつ領域Cより剛性の高い領域B,D は、より多くの荷重を分担する架橋の役割を果たす。この三領域の平均剛性を求め、さらに領域A,Eの剛性と平均すれば、全体の弾性係数が求められる。以上のモデル化において、ブリッジングモデルについてはしゅす織り材の弾性係数の記述に極めて有効であり、クリンプモデルは層数によって変動する平織り材の弾性係数の上下界の推定に有効である。
