# 12・3・2 ・b項のエシェルビーの等価介在物法において、マトリック

> **章**: 第12章 複合材料の力学
> **節**: 12・22 に示すように、複合材料の弾性係数をもつマトリックス中に

スの弾性係数を複合材料の弾性係数に置換えることにより、以下のように得られる。以上、エシェルビーの等価介在物法の応用例として、森・田中の平均場理論に基づく場合とセルフコンシステント理論に基づく場合について、複合材料の構成式を定式化した。これらの理論は、だ円体介在物の形状を変えることにより、粒子、短繊維そして長繊維で強化された複合材料に応用できることから、有力な方法として広く使われている。また、マトリックスの弾塑性変形を考慮した場合、強化材-マトリックス界面のはく離損傷あるいは強化材の割れ損傷を考慮した場合、強化材が集合したクラスタを考慮した場合に対して、これらの理論の拡張が行われている。
