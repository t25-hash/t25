# 12・4・1 異方性弾性の構成式

> **章**: 第12章 複合材料の力学
> **節**: 12・4 異方性

a. 異方性材料一般的な異方性材料(anisotropic mate-rial)の線形の応力-ひずみ関係は36個の独立な弾性定数を有する。つまり、次式が成立して、独立な弾性定数は21個になる。例えば、一軸、二軸のどちらにも180回転しても特性が同じ場合にはその材料は単斜晶系(monoclinic)と呼ばれ、以下の13個の独立な弾性定数を有する。この場合、独立な弾性定数は9個である。これを横等方性(transversely isotropic)と呼ぶ。例えば、三軸まわりの特性が等方性であれば、下記になる。直交異方性の薄板の場合、次式で表される。一軸方位の引張試験、二軸方位の引張試験、せん断試験で得られる工学的弾性定数縦弾性係数、横弾性係数、ポアソン比,νを用いると、式(12・59)の面内弾性定数は次式で表される。例として、炭素繊維(T 300)、エポキシ樹脂(5208)の一方向強化複合材料の場合、繊維方位=181GPa、繊維直角方位=10.3GPa、せん断=7.17GPa、主ポアソン比ν=0.28がよく用いられる。ガラス繊維(E-glass)エポキシ樹脂複合材料の場合、繊維方位=38.6GPa、繊維直角方位=8.27GPa、せん断=4.14GPa、主ポアソン比ν=0.26がよく用いられる。
