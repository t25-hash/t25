# 12・4・3 温度・湿度の影響

> **章**: 第12章 複合材料の力学
> **節**: 12・4 異方性

直交異方性薄板材料の材料主軸、1-2軸と構造軸が同一な薄板を考える。樹脂系複合材料においては、温度だけでなく、複合材料の吸湿によっても固有ひずみが生じる。繊維方位を一軸、繊維直角方位を二軸とした場合の熱・吸湿ひずみの合計は次式で表される。εαβε=α+β(12・69)γここで、αは熱膨張係数、βは吸湿膨張係数、は温度変化、は吸湿度変化である。炭素繊維強化エポキシ複合材料では、α=0.02×10(1/°C)、α=22.5×10(1/°C)の程度であり、ガラス繊維強化エポキシではα=8.6×10(1/°C)、α=22.1×10(1/°C)の程度である。吸湿度は乾燥時重量と吸湿時の重量の差を最大吸湿時重量と乾燥時重量の差で除した値であり、無次元数である。材料主軸と構造軸の異なる斜交軸平板(off-axis plate)の場合、式(12・69)にひずみの座標変換[式(12・63)]を行えばよい。ひずみ変換行列を掛けると、温度変化、吸湿度変化によって斜交軸平板にせん断ひずみが発生する場合があることがわかる。
