# 12・4・4 異方性材料の強度則

> **章**: 第12章 複合材料の力学
> **節**: 12・4 異方性

薄板を積層して作成する積層複合材料では、各層の破壊を二次元平面状態で判定する。以下に薄板の破壊判定則を述べる。a. 最大応力則薄板の繊維方向、繊維直角方向、せん断方向の応力が材料固有の極限値に達したときに破壊すると判定する。引張強度と圧縮強度が異なる場合には、応力の正負でそれぞれ引張強度と圧縮強度を入れ替えて判定する。三次元への拡張も容易である。b. 最大ひずみ則薄板の繊維方向、繊維直角方向、せん断方向のひずみが材料固有の極限値に達したときに破壊すると判定する。引張破壊ひずみと圧縮破壊ひずみが異なる場合には、ひずみの正負でそれぞれ引張強度と圧縮強度を入れ替えて判定する。材料のポアソン比が零ではないので、最大応力則と最大ひずみ則は同じではない。多軸負荷時の相関がないため、正確な破壊判定ができないという欠点があるが、計算が簡単であること、必要な実験が少ないため低コストであるという利点を有している。三次元への拡張も容易である。c. ヒルの破壊則ヒルの破壊則は、異方性塑性材料の降伏条件判定式をそのまま用いるものである。このため、実用的には応力の正負で破壊強度を引張りと圧縮を入れ替えて区別する必要がある。e. ホフマンの破壊則ホフマンの破壊則では、破壊則に一次項を導入して引張りと圧縮の区別を導入した。f. サイ・ウの破壊則サイ・ウの破壊則は応力の二次形式の破壊則のすべての項を考慮し、せん断の正負などの不要項を削除したものである。等方性材料の主応力場において、式(12・75)がミーゼスの降伏条件に等しくなるためにはが次式を満足する必要がある。g. ハシンの破壊則ハシンの破壊則では、破壊のメカニズムを考慮して破壊判定式が分類されている。h. ひずみエネルギー密度基準破壊則この破壊則は、a~g項の応力基準と異なり、非線形の応力-ひずみ関係を考慮し、蓄積されたひずみエネルギーが極限値に達すると破壊するという破壊判定基準である。
