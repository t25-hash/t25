# 12・5・1 古典積層理論

> **章**: 第12章 複合材料の力学
> **節**: 12・5 積層板理論

繊維強化複合材料の設計では、目標とする剛性・強度特性を得るために、図12・24に示すように、一方向繊維強化複合材料からなる積層材の繊維配向角とプライ数を決定することが重要なステップとなる。この積層材の弾性特性は古典積層理論(classical lamina-tion theory)と呼ばれる単純な板理論を適用することにより求まる。この番め層の応力-ひずみ関係は12・4節の式(12・64)で表される。積層板では、図12・26に示すように、(,)点のひずみを板厚方向の座標の一次関数で近似する。また、図12・25に示すように、通常=0を幾何学的な中央面に取ることが多い。積層板の繊維角が層ごとに異なるとき、図12・26からわかるように、ひずみはの線形関数であるが弾性係数が層ごとに異なるため、応力は各層内では線形分布、層と層の境界で不連続的に変化することになる。式(12・82)のひずみのに関する線形近似に対応して、応力については次の合力と合モーメントを考える。これら合力および合モーメントは単位長さ当たりの力とモーメントを示す。式(12・82)を応力-ひずみ関係式(12・64)に代入して、番め層の応力を中央面ひずみおよび曲率で表した後、式(12・83)の右辺の積分を行うと次式が得られる。積層板では、通常、熱残留変形を防ぐために中央面に対称に積層した対称積層材が用いられる。一方、積層パラメータは材料には関係なく、繊維角分布θにより決まる幾何学的パラメータである。積層パラメータは互いに独立ではなく、例えば、4個の面内積層パラメータの関係は次式で与えられる。2ξ-1≦ξ≦1(12・91)
