# 12・5・2 積層構成と弾性特性

> **章**: 第12章 複合材料の力学
> **節**: 12・5 積層板理論

積層材の剛性特性は個々の繊維角や層厚の複雑な関数となっているが、積層パラメータ座標上に表示すれば、弾性特性と積層構成の関係が明確となる。図12・27,12・28に炭素繊維/エポキシ積層材(一方向材の特性:繊維方向縦弾性係数=142、繊維と直角方向縦弾性係数=10.8、横弾性係数=5.49 GPa、ポアソン比ν=0.3)につい(a) 縦弾性係数,(b) 横弾性係数、ポアソン比ν図12・27 ξ-ξ座標における面内弾性特性3―163て、面内積層パラメータ座標上に弾性特性を示す。ここで、面内弾性特性,,,νは、それぞれ、,方向の縦弾性係数、横弾性係数、方向ポアソン比である。カップリングがないときξ,ξ=0,0、図12・27に示されるように、弾性特性は(ξ,ξ)の関数で表される。一方、カップリングのあるときには、弾性特性は4個の積層パラメータの関数となる。例えば、ξ,ξ=0,0のときのξ-ξ座標上での弾性特性が図12・28に示されている。このとき、式(12・90)より(ξ,ξ)の許容領域は2ξ+ξ≦1で与えられる。図12・28に示すように、縦弾性係数、横弾性係数はカップリング(ξ,ξ)が(0,0)のときに最大となる。上記の特性は面内の弾性特性に関するものであるが、面外の弾性特性についても同様である。面外積層パラメータの許容領域は式(12・90)(12・91)においてξ-ξをξ-ξに置換することにより与えられる。また、図12・27,12・28の座標をそれぞれξ-ξ,ξ-ξとおくと面外の弾性特性表示となる。さらに、線膨張係数も同様に積層パラメータ座標上に表示することにより積層構成との関係が明確となり、例えば、炭素繊維/エポキシ積層材について方向の線膨張係数が零となる積層構成が求められている。次に、積層材の代表的な積層構成と面内弾性特性の関係を示す。ここでは簡単に、ξ,ξ=0,0であるようなカップリング項を生じない積層構成を対象とする。なお、ξ,ξ=0,0となる積層構成として、例えば、+θ層と-θ層の厚さの等しいバランスト積層がある。a. クロスプライ積層0層と90層からなる積層構成であり、層厚比をおよび(ただし、+=1,≧0,≧0)とすると、ξ=-,ξ=1より、図12・29の積層パラメータ座標上ではAC(A:0一方向材、C:90一方向材に対応)上の積層構成を示す。b. アングルプライ積層厚さの等しい+θ層(θ=一定)と-θ層からなる積層構成では、ξ=cos 2θ,ξ=cos 4θよりξ=2ξ-1-1≦ξ≦1の関係が成り立つことから、図12・29の放物線ABC上の積層構成に対応する。例えば、ξ,ξ=0,-1は±45積層に対応する。c. 0±4590積層航空機構造等でよく用いられる積層構成(b) 横弾性係数(GPa)(a) 縦弾性係数(GPa)図12・28 ξ-ξ座標における面内弾性特性図12・29積層パラメータξ-ξ座標上の積層構成3―164材料力学で、0,±45,90層の割合に応じて、図12・29の三角形ABCの斜線内の一点を示す。すなわち、点A(0層)、点B(±45層)、点C(90層)の層厚比を重みとする内分点に対応する。d. 擬似等方性積層面内剛性が等方性を示す積層構成であり、ξ,ξ=0,0に対応する。例えば、0,±45,90層を1:2:1の比に積層した0±4590읎積層材(sは対称の意味)は擬似等方性を示す。あるいは、±22.5層と±67.5層を1:1の比で積層した±22.5±67.5읎積層材も擬似等方性材である。
