# 12・5・2 積層構成と弾性特性

> **章**: 第12章 複合材料の力学
> **節**: 12・5 積層板理論

積層材の剛性特性は個々の繊維角や層厚の複雑な関数となっているが、積層パラメータ座標上に表示すれば、弾性特性と積層構成の関係が明確となる。一方、カップリングのあるときには、弾性特性は4個の積層パラメータの関数となる。このとき、式(12・90)より(ξ,ξ)の許容領域は2ξ+ξ≦1で与えられる。図12・28に示すように、縦弾性係数、横弾性係数はカップリング(ξ,ξ)が(0,0)のときに最大となる。上記の特性は面内の弾性特性に関するものであるが、面外の弾性特性についても同様である。面外積層パラメータの許容領域は式(12・90)(12・91)においてξ-ξをξ-ξに置換することにより与えられる。また、図12・27,12・28の座標をそれぞれξ-ξ,ξ-ξとおくと面外の弾性特性表示となる。さらに、線膨張係数も同様に積層パラメータ座標上に表示することにより積層構成との関係が明確となり、例えば、炭素繊維/エポキシ積層材について方向の線膨張係数が零となる積層構成が求められている。次に、積層材の代表的な積層構成と面内弾性特性の関係を示す。なお、ξ,ξ=0,0となる積層構成として、例えば、+θ層と-θ層の厚さの等しいバランスト積層がある。b. アングルプライ積層厚さの等しい+θ層(θ=一定)と-θ層からなる積層構成では、ξ=cos 2θ,ξ=cos 4θよりξ=2ξ-1-1≦ξ≦1の関係が成り立つことから、図12・29の放物線ABC上の積層構成に対応する。すなわち、点A(0層)、点B(±45層)、点C(90層)の層厚比を重みとする内分点に対応する。
