# 4・3・3 高温構造解析

> **章**: 第4章 構造解析のための有限要素法
> **節**: 4・3 各種問題への応用

a.はじめに エネルギー機器など，定常運転時に高温で使用
される機器では，起動・停止などの過渡運転時に材料がさらされる
雰囲気温度が時間的に変化するが，熱伝導の遅れによって温度の空
間的な分布が生じることが多い．これに伴う熱膨張ひずみの非均一
分布によって熱応力が発生し，構造健全性の確保のためには，その
影響の的確な評価が不可欠である．さらに，運転温度が材料のクリ
ープ変形域に入った場合には，クリープ変形やこれに伴う損傷を考
慮して設計や寿命評価を行う必要がある．これらのためには，構造
物中に発生する応力やひずみの分布を，精度良く推定することが必
要である．
これまでの構造設計では，線形弾性体を仮定した応力解析の結果
を基に，非弾性変形の影響を簡易的に考慮して応力やひずみの評価
を行う，簡易非弾性解析的な手法がおもに用いられてきた．しか
し，これらは特定の材料特性や構造形状の仮定のうえに導かれたも
のが多く，個々の材料や構造に対する適用性が不明な場合が少なく
ない．一方，信頼できる数値解析法を用いて，構成材料の非弾性変
形特性をそのまま考慮した非弾性解析を行えば，人間の判断に依存
することなく，自動的に構造の非弾性応答が評価できるため，裕度
のばらつきがより小さく，高い信頼性をもった設計が実現できると
予想される．このようなことから，材料非線形性を考慮した有限要
素解析が適用されることが多くなってきている．本項では，有限要
素法を用いて，これらの評価を実施する手法の概要とその適用例を
示す．
b.非弾性構成式（inelastic constitutive equation) 応力と
非弾性ひずみの関係を与える非弾性構成式は，非弾性変形が生じる
のを無視できない高温機器に対する非弾性解析を行ううえで不可欠
であるが，応力，ひずみなど健全性を評価するための量に対して大
きな影響をもつため，使用される材料の挙動をできるだけ忠実に表
現できるものを用いることが必要である．非弾性ひずみは便宜的
に，時間依存性を考慮しない塑性ひずみと，時間とともに変化する
クリープひずみの2種に分けてモデル化されることが多いが，非弾
性変形は基本的に時間に依存するため，これらを統一的に表現する
統一型粘塑性構成式の開発も進められている．以下，これらの開発
現状について概説する．

.クリープ構成式（creep constitutive equation) クリ
ープ変形は，高温で境界条件や温度分布など外的条件が変化しない
状態でも漸進的に進行する，時間依存性変形である．一定応力下で
のクリープ変形挙動は，多くの場合三段階に分けられる．比較的低
温では，初期にクリープ変形速度が低減する効果が見られ，続いて
ほぼ一定の速度で変形が進行する定常クリープ状態を経た後，再
度，変形速度が加速する第三期クリープ状態に至る．ただし，明確
な定常クリープ状態が存在するような材料からなる高温構造に対す
る評価では，第三期クリープ状態まで考慮されることはまれであ
る．クリープ変形の評価に当たっては，一定応力下でのクリープひ
ずみ式を導いたのち，クリープひずみ速度が応力と累積クリープひ
ずみの関数で与えられるものとして，評価されることが多い（ひず
み硬化則，strain hardening rule）．ただし，この方法は硬化を累
積クリープひずみのみの関数として，等方硬化的に取扱うことを意
味しており，負荷応力が反転した場合などでは，クリープ変形でも
移動硬化的な挙動が観察されるため，良好な予測を与えない．これ
に対して，修正ひずみ硬化理論や回復可能な硬化領域モデルなどの
解決策が考案されている．さらに，塑性変形によって，クリープ変
形特性が変化したり，クリープ変形によって，塑性変形特性が変化
することもよく知られており，それぞれの構成式で硬化を表す変数
に対して，これらを考慮するための補正項を加えることが重要とな
る場合も少なくない．逆方向の塑性変形によって，クリープ変形に
対する硬化が回復することの考慮は，クリープ疲労状態でのクリー
プ損傷評価において大きな影響をもつため，特に重要である．
.統一型構成式（unified constitutive equation) 統一型
構成式は，非弾性ひずみを塑性とクリープに分離せず，統一的に取
扱う方法であり，本来的にはその整備が望まれる．これに関しても
これまで多くの研究が行われ，さまざまなモデルが提案されている
が，なかでもシャボッシュ（Chaboche）モデル は高速炉などの
構造解析に適用されている．なお，弾塑性構成式やクリープ構成式
に比較して，定数の決定が難しいのが難点である．
また，非弾性変形が小さい条件で片側だけの保持を受けた場合，
ヒステリシスループが保持を与えたのと逆方向に移動する現象が，
多くの材料で観察される．これによって生じる平均応力はき裂進展
速度に影響を及ぼすため，その考慮は重要であるが，通常の粘塑性
モデルでは，この現象を的確に表現することが困難であるため，移
動硬化則の修正も提案されている ．修正移動硬化則のガスター
ビン用超合金に対する適用例を図4･20に示す．

c .非弾性解析アルゴリズム
.弾塑性解析 応力とひずみが1対1の関係をもつ線形弾
性体と異なり，非弾性変形は負荷履歴によって異なり，構成式も微
分形で表現されるのがふつうである．このため，評価する必要があ
るのが最終的な状態だけであっても，それに至る過程を増分的に解
く必要がある．増分は計算精度の点からは細かいほうが望ましい
が，計算時間の点からはある程度大きくとるのが望ましく，両者の
バランスをとる必要がある．
増分形解析では，各点での応力増分とひずみ増分の関係を満足さ
せながら，各点での適合条件式（変位-ひずみ関係）と平衡方程式
（応力の釣合い条件）を満足させる解を求めることが必要である．
線形弾性ひずみと塑性ひずみの両者を考慮する弾塑性解析では，時
間は陽には含まれてこないので，増分は実時間とは無関係に，温度
など負荷の変化の程度に応じて決定することができる．
変位型有限要素法では，各積分点でひずみ増分から応力増分を求
める必要がある．このときに，材料固有の構成式が使用されるが，
非弾性構成式は基本的に微分形で表されているため，有限なひずみ
増分に対してこれをそのまま前進差分化して用いると，降伏条件か
らの逸脱のような不具合が生じる．これを避けるため，さまざまな
方法が考案されているが，なかでも応力を降伏曲面の中心に向けて
縮退させる法線引戻し法（radial return method，ラジアルリター
ン法）は，簡単でありながら有効性の高いアルゴリズムである．法
線引戻し法は，最初は弾完全塑性体に対して提案されたが ，近
年，非線形な移動硬化則を含む構成式に対しても適用法が開発さ
れ ，その有効性が確認されている．
次に応力増分を用いて更新された応力を，要素内で積分すること
によって得られる各節点での節点力の釣合いによって，収束性が判
定される．このときの不釣合い量（残差節点力）が十分小さければ
収束とみなされるが，そうでない場合にはこれを小さくするため
に，残差節点力を荷重項とした変位増分評価が必要となる．このた
めには，剛性方程式を再度解く必要があるが，剛性行列作成時に用
いる材料の剛性によって，収束性が大きく変化する．弾性関係を用
いると，多数の繰返しが必要となるのが一般的であり，弾塑性関係
式を用いることが有効である．単純な微分関係を用いる方法は接線
剛性法と呼ばれ，古くから用いられているが，単純な微分関係を用
いるよりも，応力積分法と整合した関係を用いるほうが収束性が向
上する．特に法線引戻し法に対応したコンシスタント接線剛性行列
（consistent tangent stiﬀness matrix）は，有効性を発揮する ．
