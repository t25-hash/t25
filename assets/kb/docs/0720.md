# 12・5・3 最適設計

> **章**: 第12章 複合材料の力学
> **節**: 12・5 積層板理論

積層板では繊維配向角やプライ数を変化させることにより、さまざまな剛性・強度特性を示す。そこで、目標とする剛性・強度特性を得るために、積層構成を最適化することが重要となる。非線形計画法や遺伝的アルゴリズム等の最適化手法を用いて、積層板の座屈・振動・強度・変形等に関する多くの最適化研究が行われており、さらにより複雑な構造である航空機複合材構造についての最適設計の研究も活発に行われている。等方性材に比べ、繊維角・層厚等の設計自由度の非常に大きな複合材構造の設計においては、最適設計の効果もまた大きくなる。複合材料の場合には、材料の異方性や異種材料を組合わせたことによる非均質性などの影響を考慮する必要があるだけでなく、材料設計を行うことができる(必要がある)のでさらに進んだ形での座屈や振動の問題の理解が要求される。複合材料の場合も等方性材料と同様、座屈振動特性は構造全体の変形に応じて生じる復元力によって支配されるので、大きさ・形状・境界条件などに依存するが、さらに異方性・非均質性の影響を受けるので、無限といっていいほどの影響パラメータが存在し、その特性を少数の表やグラフで表すことはほとんど不可能で、個々の問題に関して解を求めることを要求される場合が多い。しかし、異方性や非均質性の程度を適切に考えることにより、等方性材料の結果から座屈・振動特性をある程度推定することは不可能ではない。
