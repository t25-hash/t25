# 12・6・1 座屈の基礎式と解析例

> **章**: 第12章 複合材料の力学
> **節**: 12・6 座屈と振動

複合材料積層板の場合、積層理論の式(12・5節参照)にあるよ(a) 等方性材料板の釣合い曲線(b) 一般の積層板の釣合い曲線。荷重の初期から荷重の増加に従い変位が増加する。明りょうな座屈点はなくなり、初期不整のある等方性板の挙動に類似する図12・30等方性材料板および一般の複合材料積層板の荷重と中央点のたわみの関係うに、板厚方向の不均質性のために任意に積層をすると面内の問題と面外の問題が連成する。この場合非線形項まで考慮したときには図12・30(a)に太線で示した等方性板の圧縮時の平衡曲線と異なり、図12・30(b)のようにつねに安定な釣合い状態が連続的に存在し、厳密には座屈現象は存在しない。実際問題としては、初期不整のある等方性板[図12・30(a)細線]と同様、たわみが急に増加し始める荷重が存在し、この状態を工学的には座屈と考えてよいであろう。このような値を評価することは設計上重要である。座屈前の変形が小さい場合には座屈前のたわみを無視して座屈荷重を求めると上記の曲げ変形が急増する点の近似解となる。各係数に関しては12・5節を参照のこと。中立面に対して対称に積層すると連成項が零になり、一般の異方性板の問題として扱える。複合材料積層板は、面内剛性に比べてせん断剛性が小さいので、せん断変形の寄与が等方性板に比べて大きく、古典積層理論(キルヒホッフの仮説)に従って求めた座屈荷重は、実際の座屈荷重よりかなり大きくなる傾向がある。a. 直交異方性板の座屈図12・31のような長方形直交異方性板の座屈を考える。もちろん、一般の境界条件の場合には等方性板と同様、陽な解を導くことは困難で数値解析手法を用いる必要がある。長方形板の縦横比に関してプロットすると図12・32のようになる。曲げ剛性が剛な方向の実寸法が小さくなったと考えることにより、等方性板の結果から直交異方性板の結果を類推することができる。例えば繊維方向が軸と0°,45°,90°,-45°をなす層を重ね合わせると、面内の弾性率が方向に寄らず一定になり擬似等方性板と呼ばれるが、曲げに関しては等方性とはならない。このときおよびの成分が零とならないので、理論解は得られない。同じ方向成分をもつ板であっても積層順序の違いにより座屈荷重が大きく異なり、板厚方向に材料特性が一様でないことの影響は小さくないことがわかる。c. 層間はく離と座屈荷重低下複合材料積層板は面内の剛性に比べて層間靱性が小さいために、層間はく離が発生しやすい。さらに層間はく離が生じた部分では重ねばりの問題から類推できるように局部的に曲げ剛性の低下が著しく、局部座屈からの損傷の拡大により圧縮特性に極端な劣化を引き起こす。したがって、設計者はこのことをつねに念頭において複合材料構造を設計することが要求される。
