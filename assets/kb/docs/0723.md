# 12・7・1 応力集中

> **章**: 第12章 複合材料の力学
> **節**: 12・7 強度と破壊

異方性材や積層材における応力集中について、強化繊維の直径よりも大寸法の円孔が存在する巨視的な応力集中により生じる強度の低下は、設計上重要な問題となる。特に、円孔を有する複合材料の引張りでは、破壊時における公称応力は円孔径の増加とともに減少する円孔寸法効果(hole size effect)が現れるが、円孔径がある限界値以下では、円孔縁は破壊の起点にはならないとされている。異方性体の場合、等方性材と異なり、応力集中係数は径のみならず弾性係数にも依存する。a. 応力集中係数図12・36(a)のだ円孔を有する直交異方性無限平板に一様応力σが作用した場合、孔縁,における応力集中係数は次式で示される。b. 点応力説(point stress criterion)による強度予測図12・36(b)に示す円孔を有する直交異方性材の無限平板に一様応力σが作用した際の軸方向応力は次式で示される。点応力説は、ホイットニーらが提案した複合材の有孔部の強度に関する理論である。次式に示すように、円孔縁からある一定距離だけ離れた点の応力が材料強度(無孔材強度)σに達した場合に破壊が生じると考える。ホイットニーらは、典型的なガラス/エポキシ系複合材料について=1.02mmを採用している。したがって、点応力説では有孔材の破壊強度と円孔径の関係をσとの二つのパラメータを用いて説明することができる。c. 平均応力説(average stress criterion)による強度予測点応力説と同じくホイットニーらは、円孔縁から一定距離間の平均応力がその材料の引張強度に達したとき破壊が生じると考え、次式の平均応力説を提案している。ホイットニーらは、ガラス/エポキシ系複合(a) だ円孔(c) 切欠(b) 円孔図12・36 孔および切欠を有する平板図12・37 CFRP 6層積層材の強度に及ぼす孔径の影響材料について=3.81mmを採用している。d. 切欠材の強度予測図12・36(c)に示す長さ2cの切欠を有する直交異方性材の無限平板に一様応力sが作用した際の軸方向応力は次式で示される。e. マーとリンによる強度予測(Mar-Lin model)マーとリンは、切欠を有する複合材料に対して、次式の破壊応力の予測式を提案している。は繊維/母材界面に起因する不連続部先端での応力特異性を示すパラメータである。母材および繊維のポアソン比、せん断弾性係数に依存し、ガラス/エポキシ系複合材料で0.28と報告されている。は複合材料の破壊靱性を示すパラメータであり、マーらは各孔径での実験結果より得た破壊応力から種々のを求め、平均値を採用している。図12・37に示すように、孔径ならびに積層順序の破壊強度に及ぼす影響がラガースらにより調査されている。その結果、マー・リンの手法が点応力説に比べて良い相関を示すという報告がなされている。
