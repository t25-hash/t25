# 12・7・2 時間依存特性

> **章**: 第12章 複合材料の力学
> **節**: 12・7 強度と破壊

樹脂の力学挙動には程度の差はあれ時間依存性があり、いわゆる粘弾性挙動を示す。樹脂をマトリックスとする複合材料もまた樹脂に起因して粘弾性挙動を示す。粘弾性挙動は、具体的には一定ひずみを加えたときに生じる応力が時間の経過とともに減少する応力緩和現象や、一定の応力を加えたときに生じるひずみが時間の経過とともに増加するクリープ現象として観察される。FRPのマトリックスとして一般的に用いられるエポキシ樹脂のクリープコンプライアンスを種々の温度で測定した例を図12・38に示す。この樹脂の耐熱性の目安であるガラス転移温度は142°Cであるが、この温度以上はもちろんのこと以下においても、は時間の経過とともに増加してクリープ現象を示し、またこの現象は温度の上昇によっても大きく増加する。樹脂単体と同様に、CFRPのも時間の経過とともに増加してクリープ現象を示し、またこの現象は温度の上昇によっても大きく増加する。FRPのマトリックスとして用いられる熱硬化性樹脂は熱レオロジー的に単純である。すなわち、力学挙動には高温で短時間で観察される現象と同じ現象が低温では長時間において観察されるとする時間-温度換算則が成立する。これを式で表現すると次のようになる。これらのマスタ曲線を利用すれば、極めて短時間から長時間の広い時間範囲におけるエポキシ樹脂およびCFRPのクリープ現象を予測することが可能となる。
