b.複合材料板への横衝撃問題の概要 この問題については，
入門的講座 があり，本書もそれによるところが多い．まず最初
に実施されたのは，複合材料積層板を積層による非均質性が無視で
きる均質直交異方性板とみなした解析 であった．これは図12･
43(ａ)のように直交異方性板に横衝撃が作用したときの波動伝ぱの
近似解を求めたもので，図12･43(ｂ)にいわゆるアングルプライ積
層板の交角φを変化させたときの，純縦波（位相速度大）と純横
波（位相速度小）の方向依存性を示している．ただし，実際に複合
材積層板に衝撃を与えたときの挙動は，このような面内波だけでは
決定されず，面外曲げ波の応答，最も実用的な決定要因としての板
厚方向の応力波の影響などが混在した極めて複雑な現象となり，実
際には積層による非均質性が無視できないこともあって，完全に解
くことは，現在の進んだ数値計算技術をもってしても非常に困難な
いし非効率と考えられる．それよりも，以下に述べるような実用的
に重要な低速衝撃の応答は，むしろ静的な点荷重による曲げの問題
との類似性が想定され，このようなアプローチがいくつか研究され
ている．
c .複合材料板の衝撃後残留圧縮強度（CAI）特性の概要 こ
の問題の理解は大きく二つの局面に分けられる．すなわち，①複合
材積層板に横衝撃を与えたときに，どのようなサイズ・様式の損傷
を生じるかの決定，②ある様式・サイズの損傷を生じた積層板が圧
縮されたときに，どのように破損するかを解明して，強度低下を減
らす要因を特定することである．①は上記b項にごく簡単に述べ
たように，厳密な予測は非常に困難ないし非効率なので，実態に即
した上手な近似が必要である．c項では，おもに②の局面に関する
最近の理解について述べる．
この特性の概観を把握するため，いわゆる第一世代の低靱性炭素
繊維強化樹脂複合材料［CFRP（素材名KA410）］について実施し
たCAI試験結果のうち，まず，衝撃エネルギーと層間はく離投影
面積の関係に関する試験結果 を図12･44に示す．図中
SACMA，NASAとあるのはCAI試験法を示しており，前者が実
質標準となっているSACMA法，後者がもう少し航空機実機に近
いNASA法を示す．STD，PFとあるのは，衝撃付与時の複合材
支持条件の標準型と額縁型の略であるが，詳細は省略する．ある一
定以上の衝撃エネルギーがないと層間はく離が生じないしきい値が
あること，エネルギーを増加すると層間はく離面積が飽和していく
ことがわかる．後者は衝撃試験時の支持枠にはく離境界が近づき，
それ以上の進展が不可能という単純な理由による．このようなはく
離が生じると圧縮強度（CAI）が低下する．この強度低下の状
況 を示したのが図12･45である．これは複合材の航空機主構造，
特に翼の設計に関しては大問題であって，この問題を放置したま
ま，主翼のような大型構造にCFRPを適用すると，少しも軽量化
されない，という難点が生じる．
