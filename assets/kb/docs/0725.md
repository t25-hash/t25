# 12・7・3 衝突特性

> **章**: 第12章 複合材料の力学

a. 複合材の衝突問題への工学的概観複合材の構造物への応用、特に航空宇宙構造への応用が進み始めたとき、この技術的流れの阻害要因として認識されたのが、複合材構造に、板面に直角の衝突荷重(横衝撃)によって層間はく離が生じた後に、面内の圧縮負荷が作用すると、圧縮強度が激減する性質、いわゆるCAI(com-pression after impact、衝撃後圧縮)特性であった。この問題の重要性の認識を契機として、複合材構造への低速横衝撃の問題、衝撃を受けた後の圧縮強度低下解明/はく離板の非線形座屈解析の研究が進展した。また、航空エンジンに複合材製ファンブレードを適用するに際して、異物衝突の問題があり、もう少し早い速度域の横衝撃問題が研究された。複合材の強化繊維面内の衝撃荷重問題は、構成式の検討などの観点で純学理的にいくつか研究されているが、工学的には横衝撃(lateral impact)のほうがはるかに重要であるといってよい。そのため、12・7・3項では横衝撃問題の概要と、CAI現象について解説する。b. 複合材料板への横衝撃問題の概要この問題については、入門的講座があり、本書もそれによるところが多い。まず最初に実施されたのは、複合材料積層板を積層による非均質性が無視できる均質直交異方性板とみなした解析であった。これは図12・43(a)のように直交異方性板に横衝撃が作用したときの波動伝ぱの近似解を求めたもので、図12・43(b)にいわゆるアングルプライ積層板の交角φを変化させたときの、純縦波(位相速度大)と純横波(位相速度小)の方向依存性を示している。ただし、実際に複合材積層板に衝撃を与えたときの挙動は、このような面内波だけでは決定されず、面外曲げ波の応答、最も実用的な決定要因としての板厚方向の応力波の影響などが混在した極めて複雑な現象となり、実際には積層による非均質性が無視できないこともあって、完全に解くことは、現在の進んだ数値計算技術をもってしても非常に困難ないし非効率と考えられる。それよりも、以下に述べるような実用的に重要な低速衝撃の応答は、むしろ静的な点荷重による曲げの問題との類似性が想定され、このようなアプローチがいくつか研究されている。図12・44 複合材料板の衝撃後残留圧縮強度(CAI)特性の概要この問題の理解は大きく二つの局面に分けられる。すなわち、1複合材積層板に横衝撃を与えたときに、どのようなサイズ・様式の損傷を生じるかの決定、2ある様式・サイズの損傷を生じた積層板が圧縮されたときに、どのように破損するかを解明して、強度低下を減らす要因を特定することである。1は上記b項にごく簡単に述べたように、厳密な予測は非常に困難ないし非効率なので、実態に即した上手な近似が必要である。c項では、おもに2の局面に関する最近の理解について述べる。この特性の概観を把握するため、いわゆる第一世代の低靱性炭素繊維強化樹脂複合材料[CFRP(素材名KA 410)]について実施したCAI試験結果のうち、まず、衝撃エネルギーと層間はく離投影面積の関係に関する試験結果を図12・44に示す。図中SACMA,NASA とあるのはCAI試験法を示しており、前者が実質標準となっているSACMA 法、後者がもう少し航空機実機に近いNASA 法を示す。STD,PFとあるのは、衝撃付与時の複合材支持条件の標準型と額縁型の略であるが、詳細は省略する。ある一定以上の衝撃エネルギーがないと層間はく離が生じないしきい値があること、エネルギーを増加すると層間はく離面積が飽和していくことがわかる。後者は衝撃試験時の支持枠にはく離境界が近づき、それ以上の進展が不可能という単純な理由による。このようなはく離が生じると圧縮強度(CAI)が低下する。この強度低下の状況を示したのが図12・45である。これは複合材の航空機主構造、特に翼の設計に関しては大問題であって、この問題を放置したまま、主翼のような大型構造にCFRPを適用すると、少しも軽量化されない、という難点が生じる。図12・45 複合材板の横衝撃、衝撃後残留圧縮強度(CAI)特性まとめ積層複合材料の横衝撃問題は、大変複雑であるので、あまりに正直にこれを数値解析するのでなく、実験と理論の突合せを行いながら、上手な近似法を取り入れつつ、合理的な説明を行うという姿勢が必要である。実用的にこの特性を向上させるには、強化樹脂の高靱化を行うか、板厚方向に繊維を入れて強化するかの二つの方向が考えられ、複合材構造の低コスト化とも関連して、技術の潮流が変化しつつある状態である。
