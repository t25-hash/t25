# 4・1・2 技術と工学の社会的機能

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・1 社会技術・安全技術

そして自然の生態系の崩壊が現実の問題になると同時に，地球上のというコンセプト（概念）が生まれ，それまで機械文明の社会基盤存在に変わったのである．
そして21世紀の生命革命期を迎え，科で果たしている役割と機能について述べる．
人間の知的活動の中るための知としての生活知である．
この生活知を基盤にして，生活（technology）が生まれた．
技術知には，ある特定の個人にだけ可知の本質は，人と社会に役立つというなんらかの目的の実現を遂行科学（science）は，宇宙に存在する物質とエネルギーと情報とのに説明するための知的活動である．
したがって科学は，何かの実現宙に存在する理由までを説明するのが，哲学の一分野である形而上にある関係の説明知の体系なのである．
（engineering）は，遂行知である技術と説明知である科学との間を立つという目的を遂行するための特殊・局所的な技術知を，科学に係を説明するための知の体系であった．
19世紀になって工業社会を，社会的に制度化された大学において，多くの学生に伝承するとの体系化の試みが工学の起源であったのである．
そして，世界で初で1886年のことである．
それ以後，欧米でも総合大学に工学部が現代の工学は，科学から導かれる予測を基にして未知の人工物をる．
すなわち，技術と工学の社会的機能は人と社会に役立つというある．
これに対して，科学の社会的機能は，実現すべき目的は必要の解析（アナリシス）による知的活動なのである．
もに新規性の観点から評価される．
評価され，社会的に選択されて淘汰される．
遂行知の説明知である量生産された人工物のはんらんから地球環境の汚染が問題化した．
鉱物資源と化石資源の枯渇が予測された．
こうして有限な地球社会を支えてきた人工物が，逆に人々の生活と安全をおびやかす危険な現代の科学と技術と工学との間の関係と，それらが現代社会の中で最も基本的なものは，遺伝子として組込まれている先天的な生きをさらに上手に遂行するスキル（技能）としての知である技術能な，特殊で局所的な知的活動も含まれている．
この生活知と技術それに対して，物理学，化学，生物学，地学など自然についての間にある相互関係を，数学のような論理形式を使って一般・普遍的を遂行するという目的は必要としないのである．
そして，万物が宇学である．
つまり，科学と哲学の本質は宇宙に存在する自然物の間機械，航空，宇宙，電気，電子，材料，化学などの領域別の工学結ぶ知的活動の体系である．
すなわち一般に工学は，人と社会に役よって一般・普遍的になるように解析し，それらの間にある相互関が形成され，人間の誕生以来蓄積されてきた多くのいろいろな技術いう必要が生まれた．
これらの技術を近代科学によって説明する知めて総合大学に工学部が創設されたのは，日本の帝国大学工科大学設置されるようになる．
創造するため，遂行知としての機能も果たすように変化しつつあ目的の実現を遂行するための総合（シンセシス）による知的活動でとせず，宇宙に起こる現象の間に存在する相互関係を説明するためそして説明知である科学は，専門家の集団である学会によっておう人（ユーザ）である一般の人々によっておもに有用性の観点から舶，自動車，航空機，通信機，映像機器，家庭電器，原子力，半導物が次々に発明されて産業になり，社会に普及した．
その結果，機約束するかのように見えた．
