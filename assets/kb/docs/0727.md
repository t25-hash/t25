# 12・7・5 層間破壊

> **章**: 第12章 複合材料の力学
> **節**: 12・50 )。基礎-関係として周波数10Hz の一定振幅疲労試験結

複合材料ラミナ(lamina、単層板)は強度と弾性率に著しい異方性(anisotropy)を有する場合が多く、設計上必要とする強度や弾性特性の方向性をもたせるため(例えば擬似等方性など)、複合材料は複数のラミナを重ね合わせた積層材料として用いられることが多い。積層板(laminated plate)では板厚方向が繊維で強化されていないため、層間方向の強度や靱性は面内と比較して著しく低く、層間はく離(delamination)などの損傷が発生しやすい。複合材料構造の破壊は層間はく離を起点としている場合が少なくない。層間はく離は、厚肉殻、殻の曲面部、板厚変化部分、面外力を受ける部材、自由端や円孔縁などで発生しやすい。層間応力の分布は板厚方向の非均質性を考慮した三次元的応力解析を行わなければ3―172材料力学図12・55曲げモーメントとせん断力を受ける層間はく離求められないので、弾性学的には複雑な問題に属する。しかし、層間はく離の発生・進展はエネルギー解放率を用い、破壊力学的手法によって、はりや殻の理論を利用して比較的簡単に予測することができる。図12・56に模式的に示すように、はく離を有する分割はりに3とおりの荷重が作用する場合のエネルギー解放率を例示する。図12・56(a)はDCB 試験片(DCB specimen)と呼ばれ、純枠なモードIとなっていることがわかる。オブライエンは、一軸引張りを受ける積層板のエネルギー解放率を、熱残留応力を無視して次式で近似した。ラミナどうしが互いに拘束してが高くなっている積層構成ほど自由端での層間はく離を起こしやすく、また同じ積層枚数ではラミナの厚い試験片ほど層間はく離を起こしやすいことがわかる。図12・57に示すような二次元き裂を考える。き裂面に含まれる主法線を軸、き裂面に垂直な陪法線を軸、き裂前縁方向を軸に取る。直交異方性材料で、その弾性主軸がこの--座標系と一致する場合、各モードの応力拡大係数とエネルギー解放率は1対1に対応し、平面応力を仮定すると、両者の間には次式の関係が成立する。
