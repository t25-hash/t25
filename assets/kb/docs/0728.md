# 4・1・4 現代社会における機械システムの役割

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・1 社会技術・安全技術

機械文明と呼ばれる現代社会を築き上げた技術の歴史を振り返資源を動力機械で大量生産する産業から始まった．
19世紀には公社会基盤（インフラストラクチャ）のシステムが発達した．
生活が便利で快適になるとともに，先進諸国では女性の社会進出と半からは，生活の便利さや快適さと引換えに，大気や水質の汚染，ルギーの枯渇などが社会問題となって現れ，有限な地球環境を持続に，高齢社会でのいろいろな課題の解決が求められている．
これらいるのである．
景に，世界の技術が軍需用に集中した中で，民生用に特化した日本てみる．
採掘で，農業，林業，牧畜業，水産業，鉱業などが含まれる．
日本いように機械化されている．
例えば稲作では，耕地，施肥，播種，作業のほとんどが機械化されている．
農業では生産物の位置が固定化には多くの技術開発が必要であった．
こうした機械化は，すでににもかかわらず，食糧問題の解決に役立てられてきた．
最近では，る生物工場まで稼働している．
家庭内で家族によって少規模生産されてきた生活用のいろいろな人産と動力の機械を使い，大工場で大量生産されるようになった．
そて今日では，衣・食・住から公共設備に関わるほとんどすべての人れる大規模工場システムによって自動生産されている．
機械を使う中核になっている．
共社会を支えているシステムは，現代社会を形成する基盤のインフの交通と運輸，郵便，通信などの公共設備には，それぞれ固有の機る．
また，教育，医療，警察，消防，福祉，行政のような公共サーさらに，公共設備の建設には，建築物，構造物，設備，河川，道械，その素材である鉄鋼，非鉄金属，セメント，セラミックス，化共と生活を支えるすべての事務作業には事務機器が必要である．
大ーソナルコンピュータ，印刷機，複写機，通信装置，ファクシミリまた家庭や個人の生活を支える機械群が増えている．
情報（電卓，VTR，CD，パーソナルコンピュータ，DVD，ディ日本の家庭で，さらに高い水準で達成されてしまった．
これら家庭などの商品群は，第二次世界大戦後の日本が特化した民生産業の技間，欧米では東西の冷戦を背景にした軍需産業に，多くの技術者が第三次産業（tertiary industries）は人工物を使うサービス業で交通・運輸，通信・放送，商業，金融・保険，娯楽などの産業でなどの機械システムの支援なしには事業が成立しなくなっている．
センサとコンピュータを機能的にも，また物理的にも機械と一体と機械のメカトロニクスのシステムに変身している．
例えば，いま1タ，モータが搭載され，その究極は人間型二足歩行ロボットである．
18世紀後半からの産業革命は，まず衣服などの基本的な生活共，家庭，個人の生活を便利で快適にするために，動力機械を使う20世紀になって大量生産・大量消費の社会が形成され，人々の少子化が進んで社会構造が変わる兆しが現れた．
しかし20世紀後自動車による公害，産業と家庭から出る廃棄物の処理，資源とエネさせる技術課題が生まれた．
され，作業機械自身が移動する．
また対象も多種多様なので，機械世界の大規模農業でも実用化されて生産性を上げ，世界人口の急増多くの食用の植物，家畜，魚貝類を制御された環境で栽培・飼育す第二次産業（secondary industries）は人工物の製造業である．
工物は，18世紀後半からの産業革命によって新たに創造された生の社会的背景には，世界人口の増加による大量消費があった．
そし工物が，固有の生産機械を多数組合わせ，動力機械によって駆動さ自動生産ものづくりシステムは，現代社会の基盤を形成する技術のいま，人工物は用途によって多くの種類が作られている．
まず公ラストラクチャである．
電気，ガス，上下水道，陸上・海上・空中能をもつ多数の機械を組合わせた大規模機械システムが使われていビスの事業にも機械システムによる業務の遂行が欠かせない．
路，港湾，橋りょう，トンネルなどの工事を施工する土木・建設機成品などの素材を生産する産業機械が必要である．
また，生産と公量の事務を迅速で正確に処理しているのは，大型コンピュータ，パなど，事務と情報と通信のための機械システムである．
20世紀初頭からアメリカで普及し始め，現代文明の象徴として世にする生活空間における機械群である．
すなわち，年間を通して空皿洗い機，風呂，水洗トイレ，ラジオ，テレビジョン，レコードプメラ，そして自動車のある家庭生活である．
神器（洗濯機，冷蔵庫，白黒テレビ）時代，1965年ごろからの3C（カラーテレビ，クーラ，乗用車）時代，1975年ごろからの映像・ジタルカメラなど）の時代に始まり，20世紀後半にはほとんどの生活用機械である家庭電器，聴視覚機器，情報・通信機器，自動車術によって築き上げられ，世界中の家庭に供給したのである．
この従事していた．
ここでも機械システムの果たす役割が増えている．
例えば，も，コンピュータに支えられた輸送，情報，通信，映像，ロボットそして最近のほとんどの機械は，小型で軽量で低コストになったして組込み，人の機能に限りなく近い働きをする知能を備えた賢い台の乗用車にはそれぞれ50個ほどのセンサ，マイクロコンピューる．
