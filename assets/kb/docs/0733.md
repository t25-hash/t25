# 13・1・4 ねじり試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

ねじり試験(torsion test)は引張試験法ほどは広く普及していないが特色のある試験法である。ねじり試験は二つの目的で行われている。一つは回転軸やドリルのようにねじり応力(せん断応力)が作用する状態で使われる機械部品の強度を調べるためであり、もう一つは鍛造材のように強加工して使う材料の機械的性質を調べるためである。また、金属材料の機械的性質は引張試験によって通常求められるが、ひずみ量が50%を超えるような大変形を問題にするような場合には、引張試験ではくびれが生じるため、正確な応力-ひずみ関係を求めることはできない。しかし、ねじり試験を行えばくびれは生じず、容易に大変形を扱うことが可能になるため、金属材料の加工性に関する検討や、塑性変形に関する理論的な研究の分野ではねじり試験が広く行われている。ねじり試験機は駆動部とねじりモーメント(トルク)検出部によりなっており、試験片は両者の間に取付けられる。ねじり試験の結果はねじりモーメント-ねじれ角曲線として描かれ、これが引張試験における荷重-伸び曲線に相当する。ねじり試験の場合、試験片に生じる応力の算定を容易にするため、円形の断面を有する試験片が広く用いられている。中実の試験片をねじり試験で用いると、試験片の半径に依存した応力、ひずみの分布を生じる。例えば、弾性変形領域における応力は中心部において零で、表面において最大となり、その間では半径に比例して直線的に増加するという分布となる。ねじり試験で求めたせん断応力-ひずみ曲線と一軸引張りで求めた応力-ひずみ曲線とを比較する場合には、ミーゼス型やトレスカ型の相当応力や相当ひずみを用いれば、両者の間にはよい一致が見られる。ねじり試験の例として、極軟鋼を広範囲の温度およびひずみ速度にわたってねじり変形させたときのせん断応力-せん断ひずみ曲線を図13・7に示す。
