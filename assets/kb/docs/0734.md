# 13・1・5 その他の試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

薄板の深絞り性を判定するために行われる試験法の一つとして,エリクセン試験(Erichsen test)がある.これはエリクセン試験機を用い,直径または幅90mmの円形,正方形または帯状試験片を図13・8のしわ押さえとダイスの間に置き,ポンチを押込む.試験片の表裏面に貫通したき裂が生じるまでにポンチの進入した深さ(mm)の数値をもってエリクセン値を表す.試験中のしわ押さえとダイスの間隔には試料の厚さのほかに0.05mmのすきまを設けて行うのがエリクセン試験A 方法であり,別にすきまを設けずに9800N の荷重を加え,押さえつけた状態で行うのがエリクセン試験B 方法である.セラミックス等の脆性材料は引張強度試験を行うと試験片つかみ部での応力集中や,荷重線の鉛直性からのわずかなずれにより試験片が破壊し,正確な強度評価が困難となる.このような場合,三点曲げ試験(three point bending test)あるいは四点曲げ試験(fourpoint bending test)が用いられている.特に四点曲げ試験は支点間で最大応力が一定に保たれているので,推奨されている.これを図13・9に示す.試験片は,3×4×35mmの角柱試験片が一般的である.
