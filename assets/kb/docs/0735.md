# 13・1・6 材料試験機

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

a. 万能試験機引張試験のほかに圧縮,曲げなどの試験にも利用できるものを万能試験機という.一般に引張試験は万能試験機を用いて行われる.目盛ざおと送りおもりとをもって荷重を測定する試験機は概して精度が良好である.単てこ型のものは特に精度が良いため荷重検定用標準試験機として用いられる.しかし,この種のものは取扱いが不便であることから多用する向きには適さない.油圧駆動式の万能試験機では,荷重測定機構として,アムスラー式振り子力計,すなわち振り子の角変位によって荷重を測る装置をもつものと,レバーにより縮小された荷重を電気的にとらえる装置を備えたもの(おもりを移動させゼロ点法で荷重を測る)とが主流である.測定精度は後者が優れている.これらは荷重-伸び線図を書かせるにも,降伏点や引張荷重の測定にも好都合であり,万能試験機として多用されている.ほかに,直流サーボモータを用いて駆動する試験機(通称インストロン型という)も普及している.この試験機は,荷重の変動を忠実に記録できること,一定速度で変形を与えることが可能であること,などの長所をもっている.荷重-変位曲線が正確に記録できるので,上降伏点とかセレーション変形による荷重の急激な変動も明りょうに求められる.b. 圧縮試験機セメント,コンクリート,鋼球,タイヤなどのようにもっぱら圧縮試験のみ行われる材料に対しては,専用の圧縮試験機が用いられることが多い.専用の試験機は荷重用のシリンダが床上に配置された構造であるため,試験の実施が容易でかつ安価である.c. ねじり試験機ねじり試験には専用の試験機が用いられる.試験片の一端に歯車装置などを用いてねじりモーメントを与え,他端に連結した振り子の変位角あるいは抵抗線ひずみ計を用いたロードセル,荷重を電気的にとらえる装置などによってトルクの測定を行う.高速ねじり変形用に考案されたねじり試験機もある.d. 精度検査引張試験機や万能試験機ではひょう量とその5分の1に相当する荷重間を使用範囲とするので精度検査もこの間で行われる.ひょう量を変更しうるものについてはひょう量ごとに行う.使用範囲内における荷重の許容誤差は±1.0%以内でなければならない.これらの精度検査には材料試験機用荷重検定器を用いる.一般に,小荷重の検定にはたわみ型検定器(ループ)を,大荷重の検定には容積検定器(スタンダータイジングボックス)を用いる.e. ナノインデンテーション試験電子材料の分野における極薄膜(<100nm)デバイスの硬度,弾性率測定に用いられる試験法である.硬度試験時の最大押込み深さは,膜厚の10分の1に制限され,数十nmの圧痕を観察・解析する.具体的には,走査型プローブ顕微鏡(SPM : scanning probe microscope)と組合わせて圧痕の形状解析を行い,さらに,押込みにより得られる荷重-変位曲線から硬度と弾性率を算出する.疲労試験(fatigue test)には,試験片による標準試験と実機あるいは模型による実物試験とがある.標準試験は定まった形状・寸法の試験片を用い,繰返し応力またはひずみに対する破断(破損)繰返し数を求める.標準的には,一定の応力あるいはひずみ振幅を負荷する試験が行われるが,実際の構造物に加わる応力を模擬した実働応力や変動応力などによる疲労試験も行われている.また,欠陥材やき裂材の強度を破壊力学的に求めるため,疲労き裂進展試験も多く行われている.
