# 13・2・1 疲労試験機

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・2 疲労試験

疲労試験機(fatigue testing machine)は負荷荷重形式、荷重発生機構の違いにより多種多様の形式のものがある。荷重形式により大別すると、(i)引張圧縮疲労試験機、(ii)回転曲げ疲労試験機、(iii)平面曲げ疲労試験機、(iv)ねじり疲労試験機、(v)組合せ応力疲労試験機などとなる。試験片に繰返し荷重を与える駆動方式によって分類すると、(i)クランク機構式または偏心機構式、(ii)試験片または荷重の回転式、(iii)遠心方式、(iv)電磁式、(v)油圧式、(vi)圧縮空気式などがある。また、試験片を含む試験機の振動系の共振点との関係から分類すると、(i)非共振型、(ii)準共振型、(iii)共振型に分類できる。さらに、負荷荷重を任意のプログラムに従って変動させるプログラム疲労試験機、あるいは荷重を衝撃的に与える衝撃疲労試験機などがある。引張圧縮疲労試験は、応力が試験片断面にわたり一様であるので、基本的な疲労強度の特性を調べることができるものとして、多くの種類の試験機が考案され使用されている。しかしながら、試験片の試験機への取付けにおいて軸合せが悪いと曲げ応力がかかるおそれがあるので、細心の注意が必要である。荷重負荷様式としては電動機駆動によるクランク式、電磁式および油圧式などが代表的なものとして広く用いられているが、最近は適用分野が広く、高荷重の負荷ができ、かつ精度の高い制御が可能なことより、閉ループ式の電気油圧サーボ疲労試験機(electro-hydraulic servo fatiguetesting machine)が多く使われている。これは図13・10に示すように、例えばロードセルにより検出された荷重信号をコントロールユニットにフィードバックして、信号発生器からの基準信号とフィードバック信号の差が零になるように制御する閉ループ(closed-loop)を構成する。この制御信号を用いてサーボバルブによりアクチュエータに送られる圧力油の量を調整して、所定の荷重を負荷するものである。荷重以外にもフィードバック信号の切換えにより、ひずみまたは変位信号を制御した試験が可能である。この方式により負荷容量10kN 以下から10MN 程度、繰返し速度0~100Hz 程度の広範囲の試験機が製作されている。また、省エネルギーの立場から油圧共振型の試験機が用いられ、小さな駆動力で大型部材の試験ができ、しかも繰返し速度を高くすることができる。最近、10回を超える長寿命域の疲労試験の必要性から、1000Hz 程度の高周波で疲労試験の行える試験機が開発されている。最近の閉ループ式の電気油圧サーボ疲労試験機は、マイクロコンピュータやパーソナルコンピュータを用いて負荷波形信号の発生からデータの計測、処理まで自動化されており、精度の高いコンピュータ制御試験(computer controlled test)が行えるようになっている。この図13・10 電気油圧サーボ疲労試験機3―181試験機を用いることによって、例えば疲労き裂進展試験における応力拡大係数制御試験のように、計測値に基づきコンピュータで演算処理した所定量を制御する疲労試験、実機で計測された負荷波形をそのまま模擬したプログラム変動疲労試験などを容易に行うことができる。また、高精度な制御が行える特徴を生かして、電子デバイス用材料やマイクロマシン用材料の疲労試験を行うために、特に微小な荷重制御を容易にしたものや、走査型電子顕微鏡との組合せによって疲労過程中の挙動を連続的に観察できる装置が開発されている。回転曲げ疲労試験機(rotary bending fatigue testing machine)は、丸棒に静的な曲げ応力を負荷しながら試験片を電動機で回転させ、繰返し応力を加えるものである。曲げモーメントの加え方としては片持曲げと四点曲げで均一な曲げモーメントを加える方式(図썥hler)によって最初に用いられたのでベーラー型ともいわれ、構造が簡単であるので現在も広く用いられている。後者の代表例としてわが国では小野式回転曲げ疲労試験機が広く用いられている。回転曲げ疲労試験機は構造上から両振り応力で正弦波の波形しか負荷できないが、おもりで負荷するので精度や長期安定性に優れている。標準の試験片寸法は直径6~12mm程度のものが多く用いられているが、直径250mm程度の大型試験片による試験も行われている。平面曲げ疲労試験機(plane bending fatigue testing machine)は同じ曲げ負荷であるが、曲げモーメントの軸が試験片に対して一方向に限られており、回転しない場合であって、回転曲げの場合と異なり平均応力を与えて種々の応力比の試験が実施可能である。通常は平板状試験片の面外曲げ試験が行われる。試験機駆動方式としてはクランク式、電磁式あるいは電気式など各種のものが使われている。図13・12はクランク式(西原式板曲げ疲労試験機)であり、曲げモーメントの大きさははずみ車の慣性力により求めるが、この代わりにねじり応力棒によりモーメントを測定するようにしたシェンク式曲げ(ねじり)疲労試験機がある。一般的にいって荷重精度はあまりよくないので、動荷重の検定は定常的に行う必要がある。ねじり疲労試験機(torsional fatigue testing machine)は平面曲げ疲労試験機と同じ機構のもので、試験片の軸をモーメントの軸図13・11 回転曲げ疲労試験機(四点曲げ)図13・12 平面曲げ疲労試験機3―182材料力学と一致させ、試験片にねじり応力が加わるようにしたものが多い。また、油圧式のロータリアクチュエータを使った、コンピュータ制御による閉ループ式電気油圧サーボ疲労試験機も使用されている。ねじり疲労試験には中空または中実の円形断面の試験片が通常用いられる。組合せ応力疲労試験機は引張圧縮、曲げ、ねじりの変動する2種類の応力を同時に試験片に加えられるようにしたものである。これには組合わされた両応力の位相が一致する場合と位相差を自由に与えうる場合とがある。従来から使われている組合せ応力疲労試験機には、前述の平面曲げまたはねじり疲労試験機に試験片軸をモーメント軸と傾けて取付け、傾け角の大きさにより曲げとねじり応力の比率を変えた組合せ応力が負荷できるようにしたものがある。また、複数の油圧アクチュエータを組合わせた二軸引張圧縮疲労試験機や、引張圧縮とねじりの組合せ疲労試験機などが使われており、コンピュータ制御試験により位相差を変えた実験も容易に行えるようになっている。
