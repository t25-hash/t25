# 4・2・5 終わりに

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・2 地球環境技術

入規模も大きくならざるを得ず，必然的に社会との摩擦を引き起こスト面で最も期待されている風力発電については，すでに世界各地る．
また，規模の面では有力視されているCOの回収隔離につい地球環境問題そのものが必然的に先進国と発展途上国の間の不公ういった技術の社会的側面をつねに念頭におくことが必須であるのって注目が高まっているCOの回収隔離技術について述べてきた．
再生可能エネルギー技術や省エネルギー技術など，従来から研究開CO対策としても有効であるものの，発展途上国を中心とする化数十年という期間内でのこれら新技術の世界的規模での導入にも限続けるCOを大気から隔離するため，安定な形で長期的バッファる．
考えると，たとえ複数の対策技術を採ったとしても，それぞれの導す可能性がある．
例えば，再生可能エネルギーの中でも現状ではコで地元住民やNGOなどとの間で論争を呼ぶようになってきていては，さらに大きな論争を呼ぶことはほぼ確実である．
平に関する問題を包含しているという性格があることと併せて，こが，この問題に対応する技術開発の大きな特徴であるといえる．
