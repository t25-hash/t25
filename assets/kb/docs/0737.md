# 13・2・2 疲労強度試験と疲労限度の決定法

疲労強度(fatigue strength)は通常10本程度の試験片を用い、それぞれに異なった大きさの繰返し応力を与え、応力振幅と破断繰返し数の関係から-曲線(10・2・2項参照)を描いて求める。繰返し数10回以上の高サイクル疲労寿命の標準的な試験に関してはJIS 規格があり、用語、試験片、試験方法、疲労強度の決定方法などが定められている。また、ISOやASTMにも同様の規格が設けられている。疲労強度は静的強度と異なり、試験片の形状寸法、表面性状などによって大きな影響を受けるので、試験片の作製には十分な注意が必要である。回転曲げおよび平面曲げの小型の標準試験片については上記JIS 規格によって形状寸法が定められている。JIS による疲労強度(時間強度)の決定法は、-曲線を試験結果を表す各点のほぼ中央に引き、指定された繰返し数に対応する応力振幅を-曲線上から求め、時間強度と定める(ただし、-曲線の外挿は許されない)。疲労限度(fatigue limit,endur-ance limit)の決定法は、(1)その応力段階で未破断の試験片が半数以上となる最大応力、あるいは(2)上記(1)で求められた応力段階での試験片がすべて未破断の場合は、その応力段階と一つ上の応力段階の平均の応力とすると規定している。疲労強度は一般にばらつきを示すものであり、上記JIS の規定は疲労強度をばらつきの中央値(破壊確率の50%)として定めるものであるが、疲労強度を統計的にとらえ、破壊確率をパラメータとした--曲線を描くことも行われる。統計的な性質を考え方法がある。時間強度や疲労限度についての平均値や標準偏差を統計的に求めるには、通常、プロビット法(probit method)やステアケース法(staircase method,up and down method)が用いられるが、前者は多数の試験片本数を必要とするので、疲労限度の平均値や標準偏差を求めるには後者が用いられることが多い。プロビット法は疲労限度の平均値の予想値욌を中心として一定階差のいくつかの試験応力レベルを設定する。各試験応力で複数個の試験片を疲労試験し、所定の打切り繰返し数までに破断する確率を求める。次に、疲労限度が正規分布に従うという前提で、各試験応力での破壊確率を正規確率紙にプロットして直線を当てはめる。このとき、=0.5に対応する応力が疲労限度の平均値욌を与える。また、=0.841に対応する応力と욌との差が疲労限度の標準偏差σを与える。ステアケース法の試験方法は、予想される疲労限度욌の上下に一定の応力階差をもつ数段の試験応力レベルを設定し、疲労限度と予想される応力で試験を実施する。これが所定の繰返し数(通常10回)で破断しないときは応力レベルを一段上げ、あるいは所定の繰返し数までに破断したときは応力レベルを一段下げて試験を行う。以下同じ操作で試験を続ける(図13・13)。試験結果の解析図13・13 ステアケース法による実験手法は、破断した試験片数と破断しなかった試験片数を数え、和の少ないほうをとってそれをとし、この個の試験片について行う。応力レベルを,,...,욡,...,욅とし、各応力レベルにおけるデータ数を,,...,욡,...,욅とすると、疲労限度の平均値욌と標準偏差σの近似値は、욌=+±0.5⎫⎬⎭(13・5)σ=1.620-+0.029ただし、욅욡=∑욅욡=∑욅욡=∑で与えられる。式(13・5)における複号は非破壊事象について計算するときは正、破壊事象について行うときは負とする。破断繰返し数が10回以下のいわゆる低サイクル疲労試験において、通常、延性材料では応力が降伏点以上となるので、大きな塑性変形を生じる。したがって、応力振幅を一定にして試験を行う場合(定応力試験)と、全ひずみ振幅または塑性ひずみ振幅を一定にして試験を行う場合(定ひずみ試験)とでは、応力-ひずみ挙動が大きく異なる。定応力試験では、応力のわずかな変動で塑性ひずみが大きく変動するので試験が不安定になりやすく、また一方向への大きな変形(ラチェット変形)を生じる場合もあるので、一般にはひずみ制御試験が行われる。試験結果は全ひずみ振幅または塑性ひずみ振幅と破損繰返し数の関係で表される(10・2・6項参照)。
