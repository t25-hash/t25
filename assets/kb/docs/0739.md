# 4・3・2 国際標準とは

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

標準（標準は規格ともいわれるが，ここでは標準として用語を統
一する）の重要性は，身の回りを見ればよくわかる．ねじ，ビデオ
テープ，フロッピーディスクなど数え上げたら切りがないほどの製
品は標準に則って製作されている．これらが標準化されていなかっ
た場合，生活が不便となる．さらに安全にも影響があり，人命に直
接影響する場合もある．
国際標準を制定している組織も数多くあり，一般によく知られて
いる組織としては，機械関係ではISO（International Organiza-
tion for Standardization，国際標準化機構），電気関係ではIEC
（International Engineering Consortium，国際電気標準会議）な
どがある．国際標準の例としては身近な，非常にインパクトのある
品質保証標準ISO9000がよく知られている．最近ISO標準が重
要になった理由は，上に述べた米ソ冷戦終結に関係している．国際
標準化機構（ISO）は非営利団体組織として1828年に発足し，冷
戦前から多くの国際標準を発行してきている．米ソ冷戦中は，資本
主義圏と共産主義圏が対立していたため，国際標準はあったものの
その利用は非常に限られた範囲であった．冷戦終結後，両者の対立
はなくなり，グローバル化が進む中で国際標準の重要性が増したと
いえる．さらに，ヨーロッパをはじめ購入条件にISO9000の取得
を条件付けるに至って，ビジネススタンダードとなりISO9000の
重要性は一気に高まった．その結果，ISO9000認定を受けていな
ければ輸出ができないといわれるまでになり，日本の企業でISO
9000を取得している企業は3万5000社を超えるに至っている．
