# 13・3・2 応力リラクゼーション試験

> **章**: 第13章 析
> **節**: 13・3 クリープ試験

応力リラクゼーション(stress relaxation、応力緩和)とは、初期の全ひずみを一定に保ったとき、時間とともに応力が減少する現象をいう。高温下で用いられるボルトの締付力、焼ばめ力、ばね力、溶接部などの残留応力除去焼なまし、運転開始時に発生する熱応力の運転中の低下、PCコンクリートにおけるPC鋼材の室温付近での緊張力の低下などがその代表例である。したがって、リラクゼーション試験は時々刻々の応力低下を測定するのが目的である。最も一般的な試験は引張リラクゼーション試験(tensile relaxa-tion test)であり、その試験法はJIS に規格化されている。クリープ試験方法と異なるおもな点は、試験片の標点距離が長いこと、制御するための伸び測定精度が高いこと、全ひずみ保持の許容範囲は初期値の±1.5%以内であること、などである。全ひずみ一定に保持するために自動制御型の試験機が用いられるが、通常のクリープ試験機を用いて応力を段階的に減少させるステップダウン試験と呼ばれる試験法もある。⬅分割入っ注段表13・1 引張クリープ試験方法の規格(JIS Z 2271:1999)ています伸び測定装置の精度荷重精度試験温度の許容範囲0.01mmの変化が読み取れるもの±0.5%以内300<≦600 °C:±3°C600<≦800 °C:±4°C800<≦1000°C:±6°C3―184材料力学
