# 13・3・3 クリープ支配下の高温低サイクル疲労試験

> **章**: 第13章 析
> **節**: 13・3 クリープ試験

クリープ支配下の高温低サイクル疲労試験(low cycle fatiguetest at elevated temperature)における破損繰返し数は、材料の種類、温度、応力あるいはひずみ波形、ひずみ速度あるいは繰返し速度によって異なり、それは材料内の局所におけるクリープひずみの蓄積および破壊の機構と密接に関係している。同時に、それらに対して雰囲気が大きな影響を及ぼす場合のあることに関心が寄せられており、真空中の高温低サイクル疲労試験なども実施されている。試験法としては、日本材料学会報告やJIS 規格がある。電気炉あるいは高周波誘導加熱装置を備えた電気油圧サーボ疲労試験機が多く用いられている。このほかに、通常のクリープ試験機を用いておもりを上下させて片振りあるいは部分片振り応力下の疲労試験を行ったり、偏心レバーによって繰返し変位を与える機械式疲労試験機を用いたりする場合もある。環境強度試験には大きく分けて、水素脆化も含む応力腐食割れと腐食疲労に関する試験法が存在するが、ここでは腐食疲労試験法(corrosion fatigue test method)について記載する。腐食疲労試験では、想定される使用条件で腐食に影響を及ぼす因子を的確に把握して、試験条件に反映することが重要である。環境因子では、試験液中のpH や塩素イオン濃度などの主要な因子のみならず、硫酸イオンなど腐食に影響するわずかな不純物や溶液中の溶存酸素濃度を模擬することが重要であるケースがある。これらの環境を試験中に維持するためには、試験環境を保持する試験容器と管理した環境水を蓄え、供給するタンクを有した循環装置を使用することが望ましい。循環装置のタンクの水質はオンラインまたはオフラインで定期的にモニタし、適切な水質をつねに保持することが必要である。また、試験液の制御のみだけではなく、流速などの流動条件、異種金属を試験ジグなどに使用することで生じる電気化学的電位の変化などにも留意する必要があり、試験中に電気化学的電位を計測しておくことが望ましい。腐食疲労の力学因子として、繰返し速度(周波数)、応力波形(台形波、鋸波など)、平均応力が腐食疲労特性に大きく影響する場合が多く、想定する機器の変動応力条件を反映した試験条件の選定が必要である。例えば、回転機器の微小振動が対象となる場合には、遠心応力を模擬した平均応力下で高周波数・低応力振幅の試験が適切であり、機器の起動停止で生じる大きく緩やかな応力変動が対象となる場合には、低周波数・高応力振幅(必要に応じて高ひずみ振幅)の試験が適切となる。腐食疲労試験も、腐食疲労き裂発生の疲労寿命、疲労強度を求めるき裂発生試験、き裂進展速度を求めるき裂進展試験に大きく分かれる。腐食疲労き裂発生試験は、通常の疲労試験と同様に、回転曲げ疲労試験機や油圧サーボ式疲労試験機などに試験環境を保持する試験容器を付設して行われる場合が多い。高圧水素ガスや高温水中など特殊な環境で試験を行う場合には、専用の圧力容器と疲労試験機を組合わせて試験がなされており、高温高圧水中で適用できる軸ひずみ計など特殊な計測装置が使用されている。腐食疲労き裂発生試験で留意すべき点は、腐食疲労では時間依存の腐食現象が関連することから、疲労寿命を示す繰返し数のみでなく試験時間が重要な因子となることである。炭素鋼など耐食性の劣る材料を塩素イオン環境下において腐食疲労試験に供した場合、大気中では繰返し数10~10回で工学的に認められる疲労限は出現せず、疲労強度が低下する現象が認められる。このケースは時間の経過とともに、試験片表面に腐食ピットが発生・成長することで腐食疲労強度が低下したものであり、腐食ピットのような疲労強度低下要因の有無を見極めるために必要である十分に長い時間の試験を行うことが重要である。腐食疲労き裂進展試験では、破壊力学的な試験片(多くはCT 試験片)を用いることが多い。腐食疲労試験は試験環境を保持する試験容器中で行われるため、き裂進展速度を求めるために必要であるき裂長さ測定にはオンライン計測が可能な方法が望ましく、試験片の開口変位や背面ひずみを用いたコンプライアンス法、交流または直流の電気ポテンシャル法が適用される場合が多い。直流の電気ポテンシャル法では、配線の接点等で生じる起電力の経時変化の低減と試験片に直流電流を通電することで生じうる電気化学的な影響を低減する目的から、通電方向を変化させる交番式電気ポテンシャル法が適用される場合が多い。低応力拡大係数幅条件などでき裂進展速度が遅い場合には、き裂内の腐食生成物によるくさび効果のためにき裂閉口が生じて、徐々にき裂進展速度が低下する現象が認められる場合がある。このような腐食生成物によるき裂閉口は条件が変われば挙動が変化することが予想され、試験結果をそのまま実機の評価に適用すると問題が生じる可能性がある。この対策として、き裂閉口分を除いた有効応力拡大係数幅を用いて評価する必要があると提案されており、き裂進展試験中に除荷弾性コンプライアンス法等により、き裂開閉口を計測することが望ましい。
