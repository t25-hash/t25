# 13・5・1 衝撃引張り、圧縮試験

> **章**: 第13章 析
> **節**: 13・5 衝撃試験

試験片を高速で直接的に引っ張る方法はいくつか考えられ、回転慣性が十分大きな回転円板を利用するもの、落下おもりをゴムバンドや圧縮ガスで加速するもの、火薬の爆圧を利用するもの、後述するホプキンソン棒法を引張型に改良したものなどがあり、各種の衝撃引張試験(impact tensile test)が行われている。図13・15は回転円板を利用した試験装置の概略を示したものである。光電管を利用して、所定の速度のとき試験片フランジ部にこれを引っ掛けるためのヨークが飛び出すように別にトリガ回路を設けておき、つめがヨークに当たって試験片を定速で引っ張るようにしてある。試験片に作用する荷重はロードセルのひずみゲージとシンクロスコープまたは各種ディジタルメモリなどの波形測定装置によって測定され、荷重-時間関係が求められる。定速引張りであるから、荷重-時間関係はそのままほぼ応力-ひずみ関係と考えられ、:速度、:試験片の平行部長さ、とすれば、ひずみ速度はで与えられる。この種の測定では、高速になると主としてロードセルの固有振動周期に関係した振動波形が測定波形に重畳し、測定荷重に誤差が含まれるおそれがある。圧縮試験法には、カムの高速回転を利用して試験片の一端に圧縮変位を与えるカムプラストメータ、落下おもりで試験片を衝撃圧縮するもの、棒の縦衝撃を利用したホプキンソン棒法、大変形塑性ひずみ域までの測定が可能なテーラー試験などがある。図13・16は分割型ホプキンソン棒法(split-Hopkinson pressurebar testing)の概略を示したものである。入力棒と出力棒の間にこれらの棒より低強度の短い試験片を挿入してある。打撃棒が入力棒を打撃すれば、打撃面に衝撃圧縮応力が発生し、これが弾性応力波として一定速度で入力棒中を伝ぱし、試験片の一端に到達する。ここで応力波の一部は入力棒へと反射され、残りが試験片を圧縮し、図13・15 回転円板による衝撃引張試験装置の概略図13・16分割型ホプキンソン棒法による衝撃圧縮試験装置の概略図13・17 分割型ホプキンソン棒法圧縮試験により測定されたひずみ波形の例さらにその一部が試験片の他端から出力棒へと透過することになる。このように応力波の伝ぱによって試験片は衝撃的に圧縮されることになるが、応力-ひずみ関係はひずみ速度とともに入・出力棒に張り付けられた二組のひずみゲージによる測定波形(図13・17)から一次元弾性波伝ぱ理論に基づいて以下のように求められる。dεd=ε-ε-ε(13・6)ε=ε-ε-εd(13・7)σ=2ε+ε+ε(13・8)ここで、,,は入・出力棒の弾性波伝ぱ速度、縦弾性係数、横断面積、,は試験片の長さおよび横断面積である。もし、試験片の変形が一様でかつ試験片両端面の応力が等しいとみなせることができれば、すなわちε+ε=ε(13・9)が成立すれば、式(13・6)と式(13・8)は次のようになる。dεd=-2ε(13・10)σ=ε(13・11)なお、ホプキンソン棒法は圧縮のみでなく、引張りやねじり試験にも応用されている。
