# 13・5・2 シャルピー試験

> **章**: 第13章 析
> **節**: 13・5 衝撃試験

両端を支持されたはり状試験片の中央部を,一定のシャルピーハンマで打撃して破断に要したエネルギーを求める衝撃曲げ試験をシャルピー試験(Charpy impact test)といい,試験機,試験片,試験法ともにJIS をはじめ,国ごとあるいは国際規格(ISO)に規定されている.ASTM 規格のハンマ先端の形状は他の規格と異なっている.類似のものに試験片を片持形式としたアイゾット試験(Izod impact test)がある.図13・18はシャルピー試験法の概略を示したもので,ハンマが所定の位置まで持ち上げられた後,振りおろされて試験片が曲げ破断される.破断に要したエネルギーは=cos β-cos α(13・12)で求められ,ここで,:ハンマ重量,:ハンマの回転軸中心から重心までの距離,β:ハンマ振り上がり角度,α:持ち上げ角3―185図13・18 シャルピー衝撃試験法の概略図13・19シャルピー衝撃試験片(V 型切欠)度である.,,αは試験機に固有な値であるから,目盛板上の指針によるβの読みからがただちに求まるようになっている.にはハンマ軸の摩擦や空気抵抗などによる損失が含まれているが,これらは通常極めて小さい.なお試験機の容量は1-cosαで表され,金属材料用として294J のものが多く用いられている.図13・19はV 型切欠試験片を示したもので,ほかにU 型切欠やキーホールを付したものがある.このような試験片の切欠底断面積で上述のエネルギーを除したものがシャルピー衝撃値である.
