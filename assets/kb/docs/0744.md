# 4・3・7 日本の国際標準への対応

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・3 国際化と標準化

a.国際標準への日本の貢献 日本の場合，産業技術についカの標準を原案にしながら，日本なりのアレンジを加えるという手物の欠陥により，他人の生命，身体または財産を侵害したときは，過失の有無にかかわらず，これによって生じた損害を賠償する責任があることを定めている．
また製造業者等の免責事由や期間の制限についても定め，製造業者，消費者がお互い自己責任も踏まえながら，製品の安全確保に向けていっそうの努力を払い，安全で安心できる消費生活を実現するために制定された法律である．
ただし，製造物をその製造業者等が引渡したときにおける科学または技術に関する知見によっては，当該製造物にその欠陥があることを認識することができなかった場合の責任には免責事項がある．
また，製品が部品または原材料を使用して製造され，その欠陥に製造者の過失がない場合，製造者の責任は免責される．
法律的には製造者の責任は明確になっているが，その内容が明確になるものではなく，各種条件によって異なる．
引渡したときにおける科学または技術に関する知見によっては，製品に欠陥があることを認識することができなかったことが，裁判において非常に大きな問題となる．
国内の訴訟の場合はまだ簡単であるが，海外との取引きで問題が生じたときはより複雑になる．
その背景には，日本のルールと海外のルールに差があることが挙げられる．
あらかじめその差を理解し対応しておくことで，結果として責任の程度を軽減できる．
b.技術者倫理（ethics of engineers) 技術者倫理を簡単に述べれば，技術が社会および自然に及ぼす影響・効果に関する理解力や責任など，技術者として社会に対する責任を自覚する能力となる．
グローバル化した社会では，各国共通の技術者としての倫理をもつ必要がある．
国内と国外でその内容を比較した場合，大きな差がある．
国内の場合，一般にその内容は常識的かつ一般的な内容で読み手によっては翻訳が変わるものであった．
価値観が同一の場合はこのようなものでも良かったが，グローバル化した社会では，明確に記載するのが普通である．
特に国際標準のように世界で使われる場合，できるだけ解釈によって意味が異ならないように書く必要いが，日本の多くの企業は標準化活動に参加することを考えたことるという行為は自分の属している企業にとって重要であり，また，は，社会的な貢献でもある．
法として，まず日本の技術を国際標準に反映し，日本の存在を明確欧米ではすでに国際化戦略を練り，そのための準備を進めている．

その変化の最大のものは，ボーダレスエコノミーであり，それによって国際標準の重要性が増したということができる．

ヨーロッパもアメリカもこうした変化に対応して，あるいは変化を自ら主導して，それぞれに目標を明確に示したうえで戦略を体系化し，着実にそれを実践しつつある．
こうした状況の中，日本も戦略的な対応を迫られている．
b.国際連携の必要性 アジア諸国は日本と同じ標準を使っていることが多い．
これはアジア地域が市場としても生産拠点としても，これまでヨーロッパよりもはるかに日本やアメリカとの関係が深かったからである．
例えば，韓国の工業標準には多くのJIS標準が採用されているし，東南アジア諸国でもJISは広く利用されている．
中国でもJISが使われている例がある．
これは，日本企業の現地法人がJIS標準を使いながら生産する，また日本から輸出する製品の仕様をJIS標準に準じたものとすることなどによって，いわばデファクト（実績）標準として普及したものである．
同様に，アジア諸国ではアメリカ標準（ASME：アメリカ機械学会標準，ASTM：アメリカ材料試験協会標準等）も広く利用されている．
一方，アジア諸国の多くはISOやIECのメンバである．
また，大多数はWTOにも加盟しているし，中国のように最近加盟した国もある．
WTOに加盟すればTBT協定も守らなければならず，ISOやIECの標準を自国標準として採用する義務を負う．
自分たちに使いにくい国際標準を作られては困る事情は日本と同じである．
このような状況において，自国の不利益を招くことのないよう国際連携が重要となる．
この国際連携のためにも標準化活動への積極的な参加および貢献度の向上が不可欠となる．
能力の底上げという観点からも，日本企業の果たすべき役割は大きもないというところがその実態である．
企業活動の場を提供してくれている標準化の仕事に協力することれからは，知識（ソフトウェア）の輸出がたいせつである．
その方にし，かつ，アジア諸国の信頼を得ることから始める必要がある．
国際標準に対しアジアの影響力を増すため，日本が中心となりアジダとなり，またアメリカ，ヨーロッパから対等な立場で見られるこ所有権，技術者倫理について説明する．
これらも国際化の中で重要により，国際規格を満足した設備，機器は国境を越えて自由貿易がに対して規制を課して安全を確保するのではなく，製造者自身が標では製造者責任ルールを明確化した．
