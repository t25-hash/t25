c .知的所有権（intellectual property) 知的所有権（著作権
を含む）は，最初に考えた人・組織の権利を定め，文化的な所産の
公平な利用に留意しつつ，権者の保護を図り，もって文化の発展に
寄与することを目的としている．このための法律も知的所有権，著
作権法として制定されており，近年産業活動において非常に重要な
役割を果たすようになってきている．知的所有権の法律上の定義は
「人間の幅広い知的創造の成果について，その製作者に一定期間の
権利保護を与える」となっている．知的財産には，特許権，実用新
案権，意匠権，著作権，回路配置権，植物品種権，企業秘密権があ
り，営業標識について商標権，商号権，著作票標・原産地表等（不
当競争防止法）がある．知的所有権，著作権には国内の法律以外に
国際条約があり，その一つが万国著作権条約パリ条約と世界知的所
有権機関条約（ベルヌ条約）である．パリ条約は，著作権者の権利
を示し，利用者が守らなければならないことを示したもので，ベル
ヌ条約は，有権者が守らなければならないことを示している．有権
者が守らなければならないこととは，権利が無断で使用され，それ
によって事故・トラブルが発生し，損害が生じないように有権者の
管理責任を問うもので，有権者の管理義務を負わせたものになって
いる．
有権者の義務についても述べているとはいえ，基本的にこれらの
法律・条約は所有権者を非常に広い範囲で保護するものとなってい
る．著作権でいえば，著者が登録する必要はなく，書かれたもので
あればすべて著作権があると理解しなければならない．対象のもの
が著作権に抵触するかの基準は，有権者が決めることになってい
る．さらに，著作権対象物の利用における費用については，有権者
の判断にゆだねられている．このことを十分理解をせず，勝手な判
断で使用すると，突如として訴状が舞い込み，そのために莫大な損
害賠償請求をされ，企業の存続が危ぶまれるということになる．す
でに，何億円，何十億円の損害賠償がされたということがニュース
になっている．
これらの有権者保護はあまりに厳しく，公共の福祉に反すると思
われるかもしれないが，権者を保護することが結果として公共の利
益をもたらすものと考えるのが国際標準の考えであり，この考えを
理解し，対応しなければならない．
