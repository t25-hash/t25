# 13・5・4 熱衝撃試験(thermal shock test)

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・5 衝撃試験

急激な温度変化を試験片表面に与える方法はいくつかある.例えば試験片を炉で加熱し,所定の温度になった後,水(冷却液)面または水中に試験片を投入するものがある.そのほかに,熱疲労試験に用いられるものとして,試験片に通電して直接発熱させ,これに加圧空気と冷却水をノズルで吹付けて急冷するもの,イメージ炉を用いるものなどがある.硬さ(hardness)は硬い物体を押付けたり,落下させたりしたときに現れる材料の変形抵抗を,永久変形量や跳ね上がり高さの大小で表した比較値である.材料の硬さと他の機械的性質との関係は複雑で,不明のところが多いが,硬さ試験は簡便さのため,実用的に広く用いられているものである.硬さ試験には多くの方法があるが,ブリネル,ビッカースおよびロックウェル硬さ試験は最も広く用いられている方法であり,ISOの国際規格が作成されている.これを基に,技術内容を変更することなく日本工業規格(JIS)が作成されている.ショア硬さ試験に対するISO規格はないが,国内では広く用いられている試験であり,JIS が作成されている.3―186材料力学
