# 13・6・1 ブリネル硬さ(Brinell hardness)

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・6 硬さ試験

a. 原理と定義超硬合金球の圧子を、試料の表面に押込み、その試験力を解除した後、表面に残ったくぼみの直径を測定する。ブリネル硬さは、試験力を圧子と同じ球形の一部と仮定したときのくぼみの表面積で除した値に比例し、HBW という硬さ記号で表す。定義式は次のとおりである。HBW=0.1022π--(13・13)ここで、は球圧子の直径(mm)、は試験力(N)、はくぼみの平均直径(mm)である。記号HBW の前に硬さ値を表示し、その後に試験条件のおよびの数字を順次追記する。例えば、ブリネル硬さが350,=10mm,=29.42kN の場合には、350HBW 10/3000と表記する。b. 試料試料面は、平面で滑らかで凹凸がなく、酸化スケール、異物など特に潤滑剤の付着していない表面とする。前処理を行う際は、加熱、冷間加工などによる表面硬さの変化を最小限に抑える。試料の厚さは、少なくともくぼみ深さの8倍とし、試験後に試料の裏面に見える変形がないようにする。c. 試験試験の周囲温度は、一般に10~35°Cとし、温度管理条件下では23±5°Cとする。=10,5,2.5または1mmが選択できるが、試料の平均硬さを測るために、できるだけの大きな圧子を用いる。(N)は、0.102=30,15,10,5,2.5または1となる値から、=0.24~0.6となるように選択する。試料は頑丈な支持装置の上に置き、試験中に動かないようにする。圧子を試験面に接触させ、その面と垂直方向に衝撃や振動のないように試験力をかける。試験力をかけ始めてから規定のに達するまでの時間は2~8秒とし、を10~15秒保持する。これより長い保持時間を採用する場合、許容誤差は±2秒とする。隣接するくぼみの中心間距離は3以上、くぼみの中心から試料の縁までの距離は2.5以上とする。
