# 13・6・2 ビッカース硬さ(Vickers hardness)

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・6 硬さ試験

a. 原理と定義ビッカース硬さは、対面角が136°のダイヤモンド四角すい圧子を用い、試験面にくぼみを付けたときの試験力(N)とくぼみの表面積から求めた硬さであり、HV という硬さ記号で表す。定義式は次のとおりである。記号HV の前に硬さ値を表示し、その後に試験条件のの数字、試験力の保持時間を順次追記する。例えば、ビッカース硬さが640,=294.2N、試験力の保持時間が20秒の場合には、640HV30/20と表記する。試験力の保持時間が10~15秒の場合には、表記を省略する。b. 試料試料面には、油や酸化物などの異物があってはならず、の0.5%または0.2 mまでの測定が行えるように仕上げを施す。硬さ値に仕上げが影響しないように注意する。試料の厚さはの1.5倍以上とし、試験後に試料の裏面に見える変形がないようにする。c. 試験試験の周囲温度は、一般に10~35°Cとし、温度管理条件下では23±5°Cとする。一般的なは0.09807~980.7N であるが、他のを用いてもよい。試料面は圧子軸に垂直に置き、試験力は衝撃・振動を伴うことなく、しかも負荷機構の慣性による誤差が無視できる程度の速度によって規定のまで増加させる。の保持時間は10~15秒とする。鋼・銅および銅合金の場合、隣接するくぼみの中心間距離は3以上、くぼみの中心から試料の縁までの距離は2.5以上とする。軽金属・鉛・スズおよびそれらの合金の場合、それぞれ6以上および3以上とする。
