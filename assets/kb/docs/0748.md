# 4・4・2 日本の少子高齢社会

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・4 少子高齢化

図4･12に示したように日本は高齢者比率が先進各国の中でも高
く，まもなく25%（4人に1人）を超えようとしており，さらには
3人に1人までになろうとしている ．このままの状況では図4･
13中左に示したように子供を含む3組の若年者が1組の高齢者を
支えながら社会の活力を維持していかなければならない社会が訪れ
てしまうと懸念されている．そこではRT（robot technology）の
支援を受けて，子供も高齢者も皆で図4･13中右に示すような活力
ある社会を実現していく必要がある．このような支援にはRT単
独でなくBT（bio- technology）やIT（information technol-
ogy）も融合することが有効であると考えられている．人間は図4･
14に示すように誰でも加齢に伴い肉体的能力の低下が避けられな
いが ，このような能力の変化は自然なことであり，医療はこの
変化を上手に予防・支援して健康を実現することが望ましい．先進
各国はいずれも超高齢社会を迎えようとしているが，このような社
会では人々が健康を確保するための産業は有望と考えられており，
そのためのさまざまな施策が採られている．
