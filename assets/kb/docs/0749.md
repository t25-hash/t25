# 13・6・4 ショア硬さ(Shore hardness)

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・6 硬さ試験

a. 原理と定義ショア硬さは,ダイヤモンドハンマを一定の高さから落下させ,その跳ね上がり高さに比例する値であり,HS という硬さ記号で表す.定義式は次のとおりである.HS=(13・16)ここで,はハンマの落下高さ,は跳ね上がり高さ比をショア硬さに交換する係数であり,試験機の計測筒の形式によって異なる.反発高さをダイヤルゲージで読み取るD 型試験機が多く用いられている.この場合,=1000065で,試験形式の表示が必要であればHSD の硬さ記号を用いる.その他,目測によるC型試験機を用いたHSCおよびビッカース硬さからの換算ショア硬さVHS がある.硬さの表示は,硬さ値,硬さ記号の順に行う.b. 試料試料は,試料面を平面とし,質量が0.1kg 以上でなるべく大きくする.試料の厚さは硬さの測定に試料受台の硬さが影響しない厚さとする.c. 試験試験の周囲温度は,一般に10~35°Cとし,温度管理条件下では23±5°Cとする.試料を試料受台に置き,約200N の力で押付ける.D 型試験機の場合,操作開始からハンマの落下までの時間を約1秒とし,その戻し操作は緩やかに行う.硬さを測定する位置は試料の縁から約4mm以上,くぼみ相互の中心間距離は1mm以上とする.ショア硬さの測定では,0.5HS まで読み取り,連続して測定した5点の平均値を硬さ値とする.
