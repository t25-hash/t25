# 13・7・1 試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・7 破壊靱性試験

小規模降伏条件(応力拡大係数の有効性を規定)と平面ひずみ条件(き裂先端における塑性拘束が高いことを規定)が満足されるときの開口モード(モードI)のき裂進展に対する破壊抵抗は、試験片形状や負荷方式に依存しない材料定数であり、平面ひずみ破壊靱性(plane strain fracture toughness)と呼ばれる。を評価するための規格としては、歴史的にASTM 規格E399が有名であり、その他、イギリス規格とISO規格(およびこのISO規格を翻訳したJIS G 0564:1999)があるが、内容に本質的な差はない。使用する試験片は図13・20のような三点曲げ試験片とコンパクト(CT)試験片であるが、そのほかに円弧型(C型)試験片と円盤型(ディスク型)試験片も用意されている。試験片に導入する疲労予き裂長さは1.3mm以上で、かつ(切欠を含む)予き裂長さの5%以上である必要がある。得られた-曲線はタイプI,II,IIIのいずれかに属する。図中の直線OA は-曲線の初期こう配を延長したものである。次に、OA のこう配より5%少ないこう配の直線(5%オフセット線)OBを引き、OB と-曲線との交点をPとする。直線OA とOB の範囲内で得られる最大荷重がである。したがって、タイプIの場合には=Pであるが、ポップインや不安定破壊が発生したタイプIIやIIIの場合には≧Pとなる。弾性計算によれば、5%オフセット線はき裂長さが約2%増加した場合のこう配に相当し、実際にはき裂進展のほかにき裂先端での塑性変形(の過小評価)の効果も加わる。さらに、安定なき裂進展が生じれば破壊抵抗の上昇(の過大評価)が生じる可能性がある。これらの効果を抑制するため、ASTM において規格の簡便性と正確性との両立という観点から実験が重ねられ、最終的に上記の3条件が課せられている。特に、式(13・17・b)は小規模降伏条件を、式(13・17・c)は平面ひずみ条件をそれぞれ規定している。
