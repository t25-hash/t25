# 13・7・1 윪試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・7 破壊靱性試験

小規模降伏条件(応力拡大係数の有効性を規定)と平面ひずみ条件(き裂先端における塑性拘束が高いことを規定)が満足されるときの開口モード(モードI)のき裂進展に対する破壊抵抗は、試験片形状や負荷方式に依存しない材料定数であり、平面ひずみ破壊靱性(plane strain fracture toughness)윪と呼ばれる。윪を評価するための規格としては、歴史的にASTM 規格E399が有名であり、その他、イギリス規格とISO規格(およびこのISO規格を翻訳したJIS G 0564:1999)があるが、内容に本質的な差はない。この規格には、試験装置、試験片、の計算方法、疲(a) 曲げ試験片(b) コンパクト試験片(ピン径0.25)図13・20 윪試験片図13・21 윪試験におけるの決定方法労き裂の導入条件および有効性の判定条件などに関する規定がある。使用する試験片は図13・20のような三点曲げ試験片とコンパクト(CT)試験片であるが、そのほかに円弧型(C型)試験片と円盤型(ディスク型)試験片も用意されている。試験片に導入する疲労予き裂長さは1.3mm以上で、かつ(切欠を含む)予き裂長さの5%以上である必要がある。標準試験片の、試験片幅と試験片厚さは、≒,≒2と規定されているが、1≦≦4(三点曲げ)、2≦≦4(CT)までの範囲は許容される。試験片に疲労予き裂を導入する際、予き裂長さの2.5%に相当する最終部分については、最大応力拡大係数욞윯율융は次式の条件を満足しなければならない。욞윯율융≦0.00032m욞윯율융≦0.6σσ-1≦욞윯읉윰욞윯율융≦0.1ここで、は縦弾性係数、σは疲労予き裂導入温度での降伏応力、σは試験温度での降伏応力、は本項の後で述べる方法によって決定される仮の破壊靱性、욞윯읉윰は疲労予き裂導入時の最小応力拡大係数である。욞윯율융が上記以上になると、予荷重効果によりが増大する場合があるので注意が必要である。試験時には、応力拡大係数の増加速度=ddが(0.55~2.75)MPa・m/sの範囲内に納まるように、荷重速度を制御する[ISO規格では(0.5~3)MPa・m/s].また、切欠端部にはクリップゲージを取付け、荷重とクリップゲージ変位との関係を-記録計に記録する。図13・21にの決定方法を示す。得られた-曲線はタイプI,II,IIIのいずれかに属する。図中の直線OA は-曲線の初期こう配を延長したものである。次に、OA のこう配より5%少ないこう配の直線(5%オフセット線)OBを引き、OB と-曲線との交点をPとする。直線OA とOB の範囲内で得られる最大荷重がである。したがって、タイプIの場合には=Pであるが、ポップインや不安定破壊が発生したタイプIIやIIIの場合には≧Pとなる。に対応する応力拡大係数を計算し、とが次式を満足する場合には、このを平面ひずみ破壊靱性윪と認める。윯율융≦1.1(13・17・a),-≧2.5σ(13・17・b)≧2.5σ(13・17・c)ここで、윯율융は試験片の最大荷重である。弾性計算によれば、5%オフセット線はき裂長さが約2%増加した場合のこう配に相当し、実際にはき裂進展のほかにき裂先端での塑性変形(の過小評価)の効果も加わる。さらに、安定なき裂進展が生じれば破壊抵抗の上昇(の過大評価)が生じる可能性がある。これらの効果を抑制するため、ASTM において規格の簡便性と正確性との両立という観点から実験が重ねられ、最終的に上記の3条件が課せられている。特に、式(13・17・b)は小規模降伏条件を、式(13・17・c)は平面ひずみ条件をそれぞれ規定している。式(13・17・b)と式(13・17・c)から明らかなように、試験片寸法の妥当性は試験後に初めて判明するので、規格には有効な윪が得られる板厚とき裂長さの推奨最低値がσの大きさ別に与えられている。
