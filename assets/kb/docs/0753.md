# 13・7・2 試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・7 破壊靱性試験

弾塑性破壊靱性（elastic-plastic fracture toughness） とは予
き裂からモードIの微小空洞の合体による延性引裂き破壊が開始す
るときの限界 積分であり，試験片形状や負荷方式に依存しない
材料定数である． が板厚全体にわたる平面ひずみ条件を要求
し，延性引裂き破壊に加えて粒界破壊や低・中強度鋼などのへき開
破壊なども対象としているのに対して， 試験では延性引裂き破
壊のみを対象として，板厚の中央部においてのみ平面ひずみ条件を
要求しているので，注意が必要である．なお，低・中強度鋼のよう
に，遷移温度領域でへき開破壊が関与する場合にはこの規格は適用
できない．遷移温度領域で小型試験片を用いて延性引裂き破壊の開
始点として を決定しても，有効な の板厚条件を超えてなお
板厚依存性が認められるので，大型試験片ではそれよりも小さな靱
性で直接にへき開破壊する場合があることがその理由である．
試験方法はアメリカを中心として発展してきた．規格として
は，ASTMにおいてE813 が1981年に制定され，日本でも日
本機械学会基準としてJSME S001 が1981年に制定された．
E813はその後E1152「 曲線（破壊抵抗曲線）の評価方法」と統
合されてE1737が制定され，さらにき裂先端開口変位CTOD試験
方法とも統合され，E1820 として現在に至っている．このよう
な統合は，イギリス規格 やISO規格 でも同様に行われてき
ている．ただし，後述するようにイギリス規格 やISO規格
では という表記は用いられていない．
