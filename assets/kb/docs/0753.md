# 13・7・2 試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・7 破壊靱性試験

弾塑性破壊靱性(elastic-plastic fracture toughness)とは予き裂からモードIの微小空洞の合体による延性引裂き破壊が開始するときの限界積分であり、試験片形状や負荷方式に依存しない材料定数である。が板厚全体にわたる平面ひずみ条件を要求し、延性引裂き破壊に加えて粒界破壊や低・中強度鋼などのへき開破壊なども対象としているのに対して、試験では延性引裂き破壊のみを対象として、板厚の中央部においてのみ平面ひずみ条件を要求しているので、注意が必要である。なお、低・中強度鋼のように、遷移温度領域でへき開破壊が関与する場合にはこの規格は適用できない。遷移温度領域で小型試験片を用いて延性引裂き破壊の開始点としてを決定しても、有効なの板厚条件を超えてなお板厚依存性が認められるので、大型試験片ではそれよりも小さな靱性で直接にへき開破壊する場合があることがその理由である。規格としては、ASTM においてE813が1981年に制定され、日本でも日本機械学会基準としてJSME S 001が1981年に制定された。E813はその後E1152「曲線(破壊抵抗曲線)の評価方法」と統合されてE1737が制定され、さらにき裂先端開口変位CTOD 試験方法とも統合され、E1820として現在に至っている。このような統合は、イギリス規格やISO規格でも同様に行われてきている。ただし、後述するようにイギリス規格やISO規格ではという表記は用いられていない。以下では、まず複数試験片による破壊抵抗曲線法(曲線法)についてASTM 規格に基づき説明する。使用する試験片は図CT 試験片に関しては、荷重線変位(両ピン穴の中心点を結ぶ線上のき裂開口変位)を測定する必要から、ピン穴間距離、ピン穴径、切欠形状に違いがある。負荷は変位制御で行う。試験には、複数試験法と単一試験片法(除荷弾性コンプライアンスや電位差法によりき裂進展量を評価)がある。以下に複数試験法の手順の概略を示す(図13・22)。(2) その際、荷重-荷重線変位を記録し、規格で指定された式により積分を計算する。(3) 試験片厚さを八等分し、き裂前縁に沿う9点平均(両表面の平均は1点とみなす)として鈍化量(ストレッチゾーン幅)を含むき裂進展量Δを測定する。(4) 鈍化直線として次式あるいは実験式を用いる。(6) 鈍化直線に平行な0.2mmオフセット線と曲線との交点を仮の破壊靱性とする。なお、に関する規定は板厚中心部が平面ひずみ状態を満足するための規定であり、またに関する規定はパラメータが有効であるための降伏規模に関する規定である。は上記(7)の条件を満足する大規模降伏状態においても評価可能であるので、原理的には小型試験片により高靱性材料のを容易に評価できることになる。ただし、試験方法(5%オフセット線法)と試験方法(0.2mmオフセット線法)とでは定義が異なることもあり、イギリス規格とISO規格では小型試験片からの換算という記述はない。例えば、ISO規格では0.2mmオフセット線との交点を安定き裂発生点とみなす(engineering estimatesとする)考えは同様であるが、ではなくと表記している。一方、ISO規格には破壊靱性(fracture toughness)に加えて、実際の破壊開始靱性(actual initiation toughness)の評価方法が規定されている。この方法では、複数試験片を用いることは図13・22と同様であるが、Δの測定と並行して、走査型電子顕微鏡(SEM)により破壊開始後の試験片破面から限界ストレッチゾーン幅(限界SZW)を(フラクトグラフィーの専門家が)測3―19 0材料力学定する。SEM による単一観察画像を予き裂前縁に沿って九等分して、その平均値として限界SZW を求める。その概要を図13・23に示す。図13・23に示すように、原点からを示す交点に直線を引き、この線と0.2mmオフセット線との間に少なくとも1点の曲線データが必要である。点をおいているという点に特徴がある。そのため、曲線法とともに限界SZW を用いる方法が1981年の初版においてすでに取入れられており、この意味において図13・23の手法に先鞭をつけたと考えられる。
