# 13・7・2 윪試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・7 破壊靱性試験

弾塑性破壊靱性(elastic-plastic fracture toughness)윪とは予き裂からモードIの微小空洞の合体による延性引裂き破壊が開始するときの限界積分であり,試験片形状や負荷方式に依存しない材料定数である.윪が板厚全体にわたる平面ひずみ条件を要求し,延性引裂き破壊に加えて粒界破壊や低・中強度鋼などのへき開破壊なども対象としているのに対して,윪試験では延性引裂き破壊のみを対象として,板厚の中央部においてのみ平面ひずみ条件を要求しているので,注意が必要である.なお,低・中強度鋼のように,遷移温度領域でへき開破壊が関与する場合にはこの規格は適用できない.遷移温度領域で小型試験片を用いて延性引裂き破壊の開始点として윪を決定しても,有効な윪の板厚条件を超えてなお板厚依存性が認められるので,大型試験片ではそれよりも小さな靱性で直接にへき開破壊する場合があることがその理由である.윪試験方法はアメリカを中心として発展してきた.規格としては,ASTM においてE813が1981年に制定され,日本でも日本機械学会基準としてJSME S 001が1981年に制定された.E813はその後E1152「曲線(破壊抵抗曲線)の評価方法」と統合されてE1737が制定され,さらにき裂先端開口変位CTOD 試験方法とも統合され,E1820として現在に至っている.このような統合は,イギリス規格やISO規格でも同様に行われてきている.ただし,後述するようにイギリス規格やISO規格では윪という表記は用いられていない.以下では,まず複数試験片による破壊抵抗曲線法(曲線法)についてASTM 規格に基づき説明する.使用する試験片は図CT 試験片に関しては,荷重線変位(両ピン穴の中心点を結ぶ線上のき裂開口変位)を測定する必要から,ピン穴間距離,ピン穴径,切欠形状に違いがある.また,予き裂長さの範囲は0.45≦≦0.70である.疲労予き裂導入時においては욞윯율융が0.0002m以下か,または予き裂導入荷重の最大値が試験時の到達最大荷重の70%以下である必要がある.負荷は変位制御で行う.試験には,複数試験法と単一試験片法(除荷弾性コンプライアンスや電位差法によりき裂進展量を評価)がある.以下に複数試験法の手順の概略を示す(図13・22).3―189図13・22
