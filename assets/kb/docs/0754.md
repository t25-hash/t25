# 13・7・3 CTOD 試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・20 に類似した三点曲げ試験片とCT 試験片である.ただし,

き裂先端開口変位(CTOD : crack tip opening displacement)は弾塑性破壊力学パラメータの一つであり,破壊時におけるこの値を評価するための試験がCTOD 試験である.CTOD 試験に関する規格はイギリス規格BS 5762としてまず規格化され,前述のように윪試験および限界積分試験方法などと統合され,現在に至っている.以下ではその概要を説明する.試験片としては図13・20に類似した三点曲げ試験片のみが推奨されていて,手順は윪試験方法と類似している.ただし,歴史的に溶接構造用鋼のへき開破壊を対象として発展してきた経緯があり,へき開破壊の板厚効果が明確ではないとの観点から,試験片厚さは実際に使用する厚さ(原厚)に一致させる必要がある.負荷時に,切欠端においてクリップゲージにより変位を測定する.規格から荷重-変位曲線のうちで,CTOD 試験に関する部分を抜粋して図13・24に示す.破壊様式は,(a)予き裂から直接へき開により不安定破壊が生じるかあるいはポップイン挙動が生じる[図13・24中(1)~(3)],(b)予き裂から安定な延性引裂き破壊が発生・進展し,その後にへき開により不安定破壊が生じる[図13・24中(4)(5)],(c)最大荷重近傍で全断面降伏によるプラトーを示した後,荷重低下を示す[図13・24中(6)],の三つに分けられる.なお,ポップインの判定に関しては別に規定があるので注意が必要である.それら3種類に対応するクリップゲージ変位,욂,욊,욇から,次式によりそれに対応するCTOD =δ,δ욂,δ욊,δ욇を求める.δ=1-ν2σ욦+0.4-0.4+0.6+(13・21)ここで,は荷重욂,욊,욇に対応する応力拡大係数,σ욦は試験温度での降伏応力,=-욄はクリップゲージ変位の塑性成分,は切欠端からクリップゲージ変位の計測位置までの距離である.なお,規格では,上式で得られるδ욂とδ욊の区別は,SZW を含めてΔが0.2mm以下の場合には前者,それ以上の場合には後図13・23 破壊開始靱性욡の評価方法図13・24 CTOD 試験における荷重-変位曲線の例(CTOD 試験関連のみの抜粋)図13・25 욃のddおよび試験温度依存性者と便宜上定めている.なお,これらのδはあくまでもその板厚における値であり,δ욊についてはさらに平面寸法にも依存する場合がある.なお,δ욇は塑性不安定に対する値であり,材料選択の際の参考データにすぎない.ISO規格では,図13・23で示したような評価方法により実際の破壊開始時のCTOD,δ욡を評価する方法がそれらとは別に規定されている.ASTM 規格では,CT 試験片の使用も認めている.また,SZW を除いた量をΔと定義している.そして,次式を満足する場合には,平面寸法に依存しない材料定数値δ욂となると規定している., >300δ욂(13・22)ただし,この条件についてはまだ検討が続けられている.国内では,日本溶接協会規格のCTOD 試験方法(WES 1108:1995)がある.
