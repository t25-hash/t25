# 13・7・4 動的破壊靱性試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・20 に類似した三点曲げ試験片とCT 試験片である。ただし、

動的破壊靱性(dynamic fracture toughness)としては、(a)静止き裂に動的負荷が加えられた際の進展開始靱性(衝撃破壊靱性、impact fracture toughness)、(b)高速で進展しているき裂の進展抵抗(動的破壊抵抗)があり、これらは、試験片寸法、試験温度(特に鋼)、負荷速度[上記(a)の場合]、き裂進展速度[上記(b)の場合]などに依存する。試験に際しては、厳密には動的応力拡大係数の評価が必要である。ただし、静的な解析で近似する場合も多い。なお、(a)に関しては、これまでにおよびCTOD による実験的な評価もなされている。また、ASTM 規格のE1221には短いき裂の進展停止を準静的過程の近似のもとにで解析する規定がある。ただし、衝撃荷重下での規格はまだない。イギリス規格では、BS 7448のPart IIIにおいて動的破壊靱性試験方法の検討が進められている。
