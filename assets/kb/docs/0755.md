# 13・7・4 動的破壊靱性試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・20 に類似した三点曲げ試験片とCT 試験片である.ただし,

動的破壊靱性(dynamic fracture toughness)としては,(a)静止き裂に動的負荷が加えられた際の進展開始靱性(衝撃破壊靱性,impact fracture toughness)욃,(b)高速で進展しているき裂の進展抵抗(動的破壊抵抗)があり,これらは,試験片寸法,試験温度(特に鋼),負荷速度[上記(a)の場合],き裂進展速度[上記(b)の場合]などに依存する.試験に際しては,厳密には動的応力拡大係数の評価が必要である.ただし,静的な解析で近似する場合も多い.A 533B-1鋼について,욃のddおよび試験温度依存性の一例を図13・25に示す.なお,(a)に関しては,これまでにおよびCTOD による実験的な評価もなされている.ASTM 規格のE1820 A 13には윪試験の負荷速度の上限dd=2.75MPamsを超える場合の急速(rapid loading)試験[ただし,落重や落槌などによる衝撃(impact)試験や準衝撃(quasi-impact)試験とは異なる]の規定がある.また,ASTM 規格のE1221には短いき裂の進展停止を準静的過程の近似のもとにで解析する規定がある.ただし,衝撃荷重下での規格はまだない.イギリス規格では,BS 7448のPart IIIにおいて動的破壊靱性試験方法の検討が進められている.
