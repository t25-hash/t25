# 13・7・5 その他の破壊靱性

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・20 に類似した三点曲げ試験片とCT 試験片である。ただし、

以下では、その他の主要な試験方法について規格あるいは実施要領の名称を中心として紹介する。ASTM 関連では、平面ひずみき裂停止破壊靱性試験方法(ASTM E1221-96:2002)がある。この試験は、フェライト鋼の高速進展き裂が進展停止(アレスト)する際の破壊靱性評価を目的としている。また、フェライト鋼の遷移温度領域におけるへき開破壊開始時の破壊靱性を特徴付ける参照温度の決定を目的とした、参照温度決定の試験方法(ASTM E1921-05:2005)がある。この規格には、3母数ワイブル分布を仮定した統計的な評価手法が取入れられている。そのほかに、シェブロン切欠試験片による金属材料の平面ひずみ破壊靱性評価(ASTM E1304-97:2002)、アルミニウム合金に関して、(a)平面ひずみ破壊靱性評価(Standard Prac-tice, ASTM B 645-02:2002)、(b)破壊靱性評価(StandardPractice, ASTM B 646-04:2004)、(c)応力除去処理を施していないアルミニウム合金の平面ひずみ破壊靱性評価(StandardGuide,ASTM B 909-00:2000)がある。イギリス規格関連では、BS 7448Part II:1997には溶接部における,CTOD および試験方法が、またBS 7448Part IV:1997には破壊靱性延性引裂き抵抗曲線(曲線)の決定方法がある。ISO関連では、室温におけるファインセラミックスの破壊靱性評価法[ISO 15732:2003(SEPB 法:single edge precrackedbeam法)、ISO 18756:2003(surface crack in flexure法)]、プラスチックスを対象とした線形弾性破壊力学に基づく破壊靱性評価方法[ISO 13586:2000,ISO 17281:2002(at moderately highloading rates)]がある。JIS 関連では、ファインセラミックスの破壊靱性試験方法(JISR 1607:1995)、金属材料の液体ヘリウム中での弾塑性破壊靱性試験方法(JIS Z 2284:1998)がある。日本溶接協会規格には溶接熱影響部CTOD 試験に関する指針(WES 1109:1995)がある。一方、ASTM 規格E647-00において、その標準的な試験法が述べられている。その中で、試験片としてコンパクト引張型[C(T)、従来CT]と中央引張型[M(T)、従来CCT]に、新たに偏心荷重片側端き裂引張型[ESE(T)]を加え、いずれかの使用を推奨している。線形弾性論に基づくによる整理が有効であるためには、試験片寸法は次の条件が必要とされている。疲労き裂進展試験における急激な荷重変動、特に荷重減少は、ddに大きな影響を与える。き裂長さの測定は、通常数十倍の倍率をもつ移動読取り顕微鏡で行われる。しかしながら、疲労き裂進展挙動は平面応力状態の試験片表面と平面ひずみ状態に近い試験片内部で異なり、一般には試験片内部の挙動がき裂進展を支配することから試験片表面のみでの測定では不十分な場合がある。試験片内部をも含むき裂長さの測定には、コンプライアンス法、電位差法、超音波法などが用いられる。特に、試験片表面の測定では、き裂進展方向の投影長で試験片表裏の平均値をき裂長さとするが、き裂の屈曲および表裏の差が大きい場合の限界値や板厚方向き裂前縁のわん曲に対する補正について、ASTM 規格で示されている。なお、電位差法、超音波法においても荷重と検出量のヒステリシスを描かせることによりを計測することができる。しかしながら、通常の材料、試験片寸法に対して上記のおよびの条件は、試験時間が極めて長くなり容易ではない。また微小疲労き裂(small fatiguecracks)進展試験のガイドラインや標準試験法として注意すべき他の事項も、ASTM 規格で示されている。応力腐食割れ(SCC: stress corrosion cracking)試験は、使用環境あるいはその加速環境において金属材料に荷重を負荷して、割れの発生・進展の有無あるいはそれらの速度を測定する試験であり、応力あるいはひずみの負荷方式により、定ひずみ試験(con-stant strain test)、定荷重試験(constant load test)、低ひずみ速度試験(SSRT : slow strain rate test)に大別される。定ひずみ試験法は、試験片に一定の変形を与えたまま固定して試験片自身の弾性ひずみにより応力を負荷する方法で、U ベンド型あるいはCリング型の試験片が用いられることが多い。定ひずみ試験では、き裂の発生・進展に伴って応力が解放されるため、破断時間ではなく、き裂発生の有無あるいはき裂密度やき裂深さが割れ感受性の指標として用いられる。定荷重試験法では、一定断面積の平行部を有する平板型あるいは丸棒型の試験片に一定の単軸引張荷重を加える3―19 2材料力学のが一般的である。破断までに要する時間が評価指標となるが、応力腐食割れの核ができて安定成長が始まるまでの時間は試験条件が同一であっても大きくばらつくため、確率・統計論的なデータの取扱いが必要となる。低ひずみ速度試験法は、ごく低速の動的ひずみを与えることによって迅速に割れ感受性を評価する試験法であり、一般には一定断面積の平行部を有する単軸引張型試験片が用いられる。破断面に占める応力腐食割れ破面の割合、最大応力、破断ひずみ等が評価指標となる。見掛けの割れ感受性はひずみ速度に依存し、水素割れ機構の場合はひずみ速度が小さいほど割れ感受性が増加し、活性経路割れ機構の場合はある中間のひずみ速度領域で割れ感受性が極大となることが知られている。一般には10~10/sのひずみ速度が用いられることが多い。巨視的き裂の進展過程に対象を絞り、き裂進展速度を定量的に評価する目的には、予き裂を導入したWOL型などの破壊力学試験片が用いられる。値とともにき裂進展速度が上昇する第I領域、進展速度がに依存せず一定となる第II領域、そして再びに依存し、急速破壊へと遷移する第III領域が現れるのが一般的である。応力腐食割れ試験法の規格としては、ISOに定められたものが現時点で最も体系付けられている。破面には破壊過程や荷重の大きさに応じた特徴的な幾何学的模様や形状が残されている。これらを目視あるいは顕微鏡を用いて観察を行い、破壊機構や破壊原因および負荷応力の大きさを究明する方法をフラクトグラフィー(fractography)という。また、破面写真をフラクトグラフ(fractograph)という。
