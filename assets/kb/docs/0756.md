# 4・6・3 ナノテクノロジーの研究開発の現状と特徴

> **章**: 第4章 現代の機械工学・機械技術と社会の関係
> **節**: 4・6 ナノテクノロジーと機械工学

人間環境調和型産業技術を作り上げるための共通基盤技術として
のナノマニュファクチャリング技術に関して，ナノテクノロジー全
体における位置付けは図4･28に示したとおりであり，情報通信分
野での技術開発から，バイオ，環境分野でのナノテクノロジーの発
展を生み出すためのキーテクノロジーとなっている ．現在の
ナノテクノロジーは，以下のような特徴をもつと考えられる．
(１)ナノテクノロジー全体のロードマップを作成することが重
要であるとともに，明確な目標をもって，ナノのサイエンスに本腰
を入れて取組むことがたいせつである．
これは，現在，多くの関心を集めているナノテクノロジーである
が，5年から10年で出口が見え成果が出てこないと，経済情勢が
厳しいなか，企業の多くで本格的な取組みはできないと予測され
る．ナノレベルのエレクトロニクス技術を活用するバイオ，医療，
環境技術という展開で，英知を結集して出口を考案するとともに，
成果に至るまでの加工・計測技術などの基盤技術の確立も，ロード
マップに位置付ける必要がある．連続体力学から量子力学的効果が
顕著になりだすメゾスコピック領域にかけての学問体系は，まだま
だ確立しておらず，技術としての体系化はさらに時間を要すると推
測される．しかしながら，現在は新しい機能を見出して，技術とし
て活用する方法を考案し，基本的な特許をどんどん取っていく段階
であり，その競争が行われている段階である．例えば，生物のメカ
ニズムを模倣する技術でも，生物のメカニズムを正確に解明するこ
とは重要であるが，応用する視点で，エンジニアリングとしてのバ
イオミメティックな応用技術を考案することがよりいっそう求めら
れているといえよう．
(２)ナノ加工装置，ナノ計測装置は，設備等が大がかりになる
ので，多くの企業や多くの研究者が協力して使用することが重要に
なる．
マイクロマシン技術やMEMS技術では，クリーンルームやリソ
グラフィー装置が必須で，かつ，種々のエッチング装置などをそろ
えることを考えると，場所的にも，資金的にも，運営面でも，大き
な規模にならざるを得ない状況であった．これから本格化するナノ
加工装置，ナノ計測装置の分野では，さらに大きな規模が必要にな
るので，多くの研究者の協力のもとに実施することが極めて有効で
ある．さらに，ナノテクノロジーの場合，従来の工学でイメージさ
れているクリーンという概念が通用しないことが多い．固体表面へ
の吸着分子の性質で，表面の加工特性等は大きく変化し，また，汚
れが多い中では，ナノバブル（nano- bubble，直径数百nm以下の
気泡 ）等は計測できない場合が多く，マイクロ・ナノ機能を議
論するには，再現性のある清浄な環境を実現することが前提とな
る．
