# 13・10・1 破面観察、計測技術

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・10 フラクトグラフィー(破面解析)

フラクトグラフィーは、破面観察の倍率に応じて3種類に分類できる。すなわち、実体からせいぜい数十倍程度までの低倍率でミリメートルレベルの観察を行うマクロフラクトグラフィー(macro-fractography)、一般的に数千倍程度までの高倍率でミクロンメートルレベルの観察を行うミクロフラクトグラフィー(microfracto-graphy)、および非常に高倍率でナノメートルレベルの観察を行うナノフラクトグラフィー(nanofractography)である。これらはそれぞれ得られる情報が異なり、破壊原因究明には通常複数の方法が用いられる。a. マクロフラクトグラフィーマクロ破面の観察は破面解析において通常最初に、肉眼、実体顕微鏡あるいは走査型電子顕微鏡(SEM : scanning electron microscope)による低倍率観察で行う。マクロ破面観察では破面の特徴的パターンにより破壊機構が推定できる(後出表13・4)とともに、破断部と構造物全体との関連も把握できるので非常に重要である。b. ミクロフラクトグラフィーミクロ破面の観察は一般的にSEM や共焦点型レーザ走査顕微鏡(LSM : con-focal laser scan-ning microscope)を用いて行う。SEM は倍率可変域が大きく、マクロ破面とミクロ破面の関係付けが容易であり、また焦点深度が深いため凹凸の大きな破面の観察に適している。最近では分解能の向上等、改良が加えられた電界放出型SEM(FE-SEM : fieldemission SEM)や三次元の形状計測が可能な三次元SEM(TOPO-SEM)が開発され、破面解析に適用されている。LSM(分解能;0.01 m)はSEM(分解能;数nm)に比べると解像度は劣るが、(1)大気中での高倍率観察が可能、(2)非電導性材料もそのまま観察が可能、(3)焦点深度は無限大、(4)三次元の表面形状を非接触で計測可能、などの特徴がある。c. ナノフラクトグラフィーナノメートルレベルの観察は走査トンネル顕微鏡(STM : scanning tunneling microscope)や原子間力顕微鏡(AFM : atomic force microscope)等の走査プローブ顕微鏡(SPM : scanning probe microscope)を用いて行う。STM やAFM は分解能が格段に高く(0.01nm)極微小領域の高精度の三次元形状計測が可能である。d. X 線フラクトグラフィー破面近傍の塑性変形量や残留応力を、破面にX 線を照射してX 線回折により求める方法である。計測にはX 線回折測定装置が用いられ、計測および解析の結果は上記3種類のフラクトグラフィーを補完するものとして利用される。
