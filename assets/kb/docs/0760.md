# 13・10・4 数値的破面解析技術

> **章**: 第13章 材料試験および実験応力解析

破面の特徴的パターンに基づく破面解析は多くの場合かなりの熟練を要し、ストライエーションが認められる場合を除いて定量的評価はほとんどの場合困難である。これに対し、三次元SEM,LSM,SPM 等では破面の数値的形状データが得られ、これを解析することにより定量的評価が可能である。a. 破面粗さ法破面粗さを計測して粗さパラメータから作用応力を推定するものである。粗さパラメータを用いた評価例を図13・31に示す。なお、は次式で表される。=∑[(粗さレンジペア)×(頻度)]/(計測距離)b. FRASTA 法FRASTA(fracture surface topographyanalysis)法は、一対の破面の計測結果を用いて破面形状をコンピュータ内で再構築し、破壊過程を再現する手法である。図13・32に模式図を示すが、破面を重ね合わせたあと徐々に引き離すことに3―19 4材料力学よって各段階での横断面図(XSP: cross-sectional plot)および透視図(FAPP: fractured-area-projection plot)が得られる。また、破面間距離と破壊面積率の関係が求められる。これらの結果から、き裂の起点、進展形態、き裂進展速度等の評価が可能である。素材や製品を破壊せずに調べることを非破壊試験(nondes-tructive testing)、この結果を基に規格などの基準に従って合否を判定する方法を非破壊検査(nondestructive inspection)という。非破壊試験には種々の物理現象を用いた方法が考案されており、それぞれの適用範囲や特徴を把握して適切な方法を選択することが必要である。
