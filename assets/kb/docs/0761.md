# 13・11・1 目視試験、浸透探傷試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

目視試験(visual test)は被検査材の表面に開口するき裂や欠陥を、肉眼や拡大鏡を用いた観察により検出する方法である。特別な機器を必要とせずに欠陥を最も簡単に検出できる反面、検出感度は検査員の熟練度等に左右されやすい。浸透探傷試験(penetrant testing)は目視試験と同様、表面に開口したき裂の検出に用いられる。液体が狭いすきまに浸入するという毛管現象を利用した方法である。き裂中に浸入させた浸透液を表面に染み出させ、現像膜中に指示模様を形成させる。得られた指示模様を肉眼等によって観察し、き裂の位置や表面上の形状を知る。浸透液として赤色塗料を用いて自然光のもとで観察する方法と、蛍光物質を用いて紫外線のもとで観察する方法がある。目視試験および浸透探傷試験は、き裂の深さ方向の寸法に関する情報が得られないものの、特別な装置が不要であり、金属・非金属を問わず幅広い材料の表面上のき裂や欠陥に適用可能である。
