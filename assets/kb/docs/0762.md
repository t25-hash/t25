# 13・11・2 超音波探傷試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

超音波探傷試験(ultrasonic testing)は材料内の不連続部で超音波が反射される性質を利用して欠陥を検査する方法である.通常,周波数帯域1~10MHz 程度の超音波を探触子(probe)により被検査材内に入射し,底面や欠陥による反射波を探触子で受信する.なお探触子と被検査材の間には水や油などの接触媒質を介し,超音波が透過しやすくする必要がある.得られた探傷信号より欠陥の位置や大きさを知る.代表的なき裂評価法として表面波法(sur-face wave technique),端部エコー法(tip-diffraction tech-nique)などがある.また二つの縦波斜角探触子を用い,試料表面を直進するラテラル波とき裂端部からの散乱波の到達時間差よりき裂の深さや高さを評価するTOFD(time of flight diffraction)法も開発されている.閉じたき裂に対して超音波探傷試験を適用する場合,音波が閉口き裂面の接触突起を介して透過し,反射波の強度が低下して検出が困難になる.これに対し,静的荷重や振動荷重を負荷してき裂を開かせて探傷する方法や,無負荷状態でき裂寸法とき裂の閉じ具合を未知量とした逆問題解析により評価する方法が開発されている.
