# 13・11・3 アコースティックエミッション

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

アコースティックエミッション(AE: acoustic emission)は、材料内における塑性変形やき裂の発生・進展に伴って放出される弾性波であり、これを計測することにより破壊の進行を評価する方法をAE法という。AEの周波数帯域は可聴域から数MHz であり、その計測には圧電素子などのAE変換子(AE transducer)が用いられる。地震の震源を知るのと同様に、複数の場所で計測したAE信号を解析することにより、破壊の発生位置や規模などを評価することができる。応力履歴のある材料では、過去に受けた最大応力以下ではAEは計測されない。この現象はカイザー効果と呼ばれ、材料の応力履歴を知る指標になる。AE法の特徴は、現在進行中の破壊をリアルタイムにモニタリングできることであり、稼動中の監視に適している。圧力容器の保証・保守検査や発電タービン軸受の摩耗状態の監視などに利用されているほか、地熱・石油井における水圧破砕のモニタリングや人工き裂の評価などにも用いられている。
