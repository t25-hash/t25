# 13・11・4 放射線透過試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

放射線透過試験(radiography)は、X 線やγ線などの放射線を被検査材に照射し、物質の種類や厚さによる放射線の吸収・透過の差異に基づく透視画像を得て内部の欠陥を検査する方法である。直径の異なる針金や孔を並べた構造の透過度計、厚さの異なる板を並べた階調計を被検査材と同時に撮影し、微小な欠陥の識別度や濃度の較正に利用する。放射線透過試験の特徴は、材料内部に埋没した欠陥の検出が可能なことであり、複数の方向から透視画像を得れば欠陥の形状や位置を正確に知ることができる。このため溶接施工後のブローホールの検査などに広く用いられている。放射線透過試験は放射線の透過媒体による減衰率の差を利用しているため、体積を有する欠陥の検出には有効であるが、き裂など体積の極めて小さな欠陥の検出は困難である。
