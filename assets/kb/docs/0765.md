# 13・11・5 赤外線による方法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

材料内にき裂やはく離などの欠陥が存在すると欠陥部分で熱伝導性に変化が生じる.この材料をなんらかの方法で加熱すれば,得られる温度場には欠陥による乱れが生じる.物体の表面からはその温度に応じた赤外線が放射されており,これを赤外線サーモグラフィー(infrared thermography)で計測して表面温度分布画像を得れば,同画像を基に欠陥の検査を行うことができる.この方法は非接触での検査が可能である.欠陥検出に必要な温度場の与え方には,キセノンフラッシュランプなどによるパルス加熱やプロジェクタを用いたランプ加熱のような外部からの加熱のほか,熱弾性効果を利用した内部発熱による方法がある.近年,赤外線サーモグラフィーの高性能化や小型・軽量化が進んでおり,同装置を用いた欠陥計測の高精度化や定量化に向けた研究が進められている.
