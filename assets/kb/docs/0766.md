# 13・11・6 マイクロ波,ミリ波による方法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

マイクロ波(microwave)は周波数300MHz~300GHz(波長1m~1mm)の帯域の電磁波である.同帯域の電磁波のうち,周波数30~300GHz のものはミリ波(millimeter wave)とも呼ばれる.マイクロ波を材料に照射したときの反射,透過,減衰などの性質は,材料の電磁気的特性やその不連続によって異なる.この性質を欠陥などの検査に用いることができる.マイクロ波は導電材料を透過できないので,金属材料については表面欠陥のみの検出が可能である.一方,マイクロ波は誘電材料を容易に透過するので,セラミックス,ポリマ,複合材料等を対象として,はく離の検出など内部の検査が可能である.また測定結果が材料の誘電特性に関係するので,誘電率の計測や吸水率の測定など欠陥以外の評価への応用も行われている.マイクロ波による非破壊試験では,非接触測定が可能で接触媒質が不要である.また振幅以外に位相や周波数等に関する測定も容易であり,得られる情報量が多い.
