# 13・11・7 磁気探傷試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

磁気探傷試験(magnetic testing)は強磁性材を対象とした探傷法であり,被検査材を磁化したとき表面付近の欠陥によって磁束の一部が空気中に漏えいする現象を利用して欠陥を検出する.漏えい磁束の検出方法により磁粉探傷試験(magnetic particle testing)と漏えい磁束探傷試験(flux leakage testing)に分類される.被検査材の磁化方法には電磁石を用いる極間法,コイルにより非接触で磁化するコイル法,被検査材に直接電流を流す通電法などがある.磁粉探傷試験は探傷面に強磁性粉末を散布し,漏えい磁束に吸着させて磁紛模様を観察する方法である.乾燥した磁紛を用いる乾式法と液体に懸濁させた検査液を用いる湿式法がある.磁化方向と平行なき裂に対しては感度が極端に低くなるので,き裂の方向が未知の場合には磁化方向を90°変えて探傷を行う必要がある.磁粉探傷は強磁性材の表面欠陥や溶接部の欠陥検出に広く用いられており,適用法を適切に選択すれば微小な欠陥の検出も可能である.漏えい磁束探傷試験はコイルやホール素子などの磁気センサを用い,欠陥による漏えい磁束の強さを電気信号に変換して検出する方法である.非接触で高速・自動化した探傷が可能であり,漏えい磁束の強さからき裂の深さに関する情報も得られる.
