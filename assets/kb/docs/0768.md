# 13・11・8 渦流探傷試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

渦流探傷試験(eddy current testing)は、電磁誘導現象を利用して導電性材料内の欠陥を検出する方法である。交流を流したコイルを導体に接近させると導体内には渦電流が誘導される。このときコイルには渦電流より生じる二次磁界によって逆起電力が発生する。導体表面にき裂などの欠陥が存在すると渦電流の流れが妨げられ、コイルに生じる逆起電力が変化する。同変化をコイルインピーダンスの変化として検出することにより、欠陥の検査を行うことができる。渦流探傷試験は非接触での探傷が可能であり、欠陥信号が電気信号で得られるので高速・自動化した探傷が可能である。一方、渦電流は表皮効果により材料表面に集中するため、表面の微小な欠陥を高感度に検出できる反面、薄肉部材の場合を除き、内部や裏面にある欠陥の検出は困難である。また欠陥以外にリフトオフや被検査材の導電率・透磁率などの影響によって信号が変化する。したがって多様な用途に応用できるが、欠陥のみを検出する場合にはそれ以外の要素の変動に対する対処が必要である。特に加工等により透磁率が不均一になりやすい鉄鋼材料などの強磁性体では/比が低下する。これに対し、鋼管を対象に数十Hz 程度の励磁周波数を用い、励磁コイルから管径の2~3倍以上離れた位置に検出コイルを配置することにより、透磁率の不均一から生じる雑音を受けずに管内外面の欠陥を検出できるリモートフィールド(remotefield)渦流探傷試験法が開発されている。また、き裂信号をもとに逆問題解析を行い、き裂の断面形状を評価する手法の開発も進められている。
