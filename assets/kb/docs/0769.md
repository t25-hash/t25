# 13・11・9 電位差法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

電位差法（potential drop technique）は導電性材料に電流を流
し，材料表面における電位差を計測することによる非破壊検査法で
ある．材料内にき裂が存在する場合には電流場が乱され，計測され
る電位差に変化が生じる．電流場の乱れはき裂の寸法や形状によっ
て異なるので，電位差法はき裂の定量的な評価に適してい
る．
電位差法には，用いる電流によって交流電位差法と直流電位差法
がある．周波数の高い交流電位差法では表皮効果によって電流が材
料表面付近に集中するので，表面に開口した微小なき裂に対して高
感度である反面，内部に埋没したき裂や裏面にあるき裂の検出は困
難である．また電流を供給する配線や材料表面に流れる電流により
空気中に時間変動する磁場が誘導され，計測回路に誘導起電力が発
生するので対処が必要である．一方，直流電位差法は電流が材料内
部にも流れるため，材料表面上のき裂はもちろん埋没したき裂によ
っても電流場が乱されやすく，これらの評価への適用も可能であ
る．ただし比較的大きな電流を必要とする場合が多い．磁場は時間
変動しないので誘導起電力は発生しない．電流の入出力と電位差の
計測端子をともにき裂に近接して配置することにより，より小さな
電流で微小な表面き裂を高感度に評価できる方法も開発されてい
る．
13・12材質劣化の検査法
多くの機器・構造物では，使用中に機能，強度などに変化が生じ
ることがある．これらの変化の多くは劣化（degradation）であ
り，いわゆる経年劣化（ageing）である．経年劣化現象は，大別
すれば，材質型劣化，き裂型劣化そして表面損傷型劣化に分けられ
る．材質型劣化は，基本的には材料の組織変化によって生じるもの
である．き裂型劣化は，現象としてき裂の発生，進展による劣化現
象であり，これもまたき裂の発生・進展の要因が多種多様である．
表面損傷型劣化は，劣化現象が材料の表面で生じるもので，その原
因も多様なものが考えられる．以下に特に材質劣化の計測法につい
