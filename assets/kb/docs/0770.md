# 13・12・1 焼戻し脆化

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・12 材質劣化の検査法

特定の元素を含む材料(例えば、Cr-Mo鋼やCr-Mo-V 鋼)において、温度450°C程度に長時間さらされることにより、粒界にリン、スズ、ヒ素、アンチモン等の不純物が偏析し、材料が脆化する現象を焼戻し脆化という。脆化としては、延性-脆性遷移温度の上昇として顕在化する。結合力が低下した粒界に沿ってき裂は進行し、粒界破壊が生じる。破壊的機械試験としては、例えばシャルピー衝撃試験により容易に計測することができるが、非破壊的には電気化学的な手法により微量の粒界偏析を検出することができることが報告されている。
