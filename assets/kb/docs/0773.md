# 13・12・4 熱時効脆化(スピノーダル分解による脆化)

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・12 材質劣化の検査法

高Cr合金はシグマ相析出より低い温度領域においても長時間の時効によりナノスケールの相分離を生じ,脆化する.この現象は熱時効脆化(スピノーダル分解による脆化)と呼ばれる.原子力発電プラントにおけるステンレス鋳鋼のフェライトが長時間の運転により硬くなるのはこの現象による.溶接金属のデルタフェライトもこの機構で硬化・脆化する.明らかな析出はない.この検査法については,アトムプローブ顕微鏡による相分離元素濃度の直接計測あるいはナノスケールの濃度の揺らぎを直接画像処理化して検査できる.また電気化学的な手法も提案されている.
