# 13・12・9 軟化

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・12 材質劣化の検査法

材料の軟化をもたらす原因はいくつか考えられる．例えば，加熱
軟化，クリープ軟化そして繰返しひずみ軟化等が例として挙げられ
よう．軟化の検査については，硬さ試験により直接的に計測するこ
とができる．
このほかに，炭化物粗大化，水素吸収軟化，脱炭あるいは力学負
荷による損傷として累積疲労損傷，クリープ損傷等が挙げられる
が，累積疲労損傷に関しては，陽電子消滅法，X線回折法，さら
には磁気的手法（MAE法，BHN法および磁化曲線法）の有用
性が報告されている．クリープ損傷や，疲労損傷で微小き裂の発生
を伴う累積損傷の評価には，レプリカ法の有用性が報告されてい
る．一方，硬さの変化を伴うものは，硬さ計測に加えて超音波の音
速変化あるいは減衰率等を用いて検査することが可能である．また
クリープ損傷については電気抵抗法や密度法による評価も可能であ
る．より詳細は文献(109)を参照していただきたい．
