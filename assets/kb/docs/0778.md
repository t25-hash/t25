# 13・12・9 軟化

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・12 材質劣化の検査法

材料の軟化をもたらす原因はいくつか考えられる.例えば,加熱軟化,クリープ軟化そして繰返しひずみ軟化等が例として挙げられよう.軟化の検査については,硬さ試験により直接的に計測することができる.このほかに,炭化物粗大化,水素吸収軟化,脱炭あるいは力学負荷による損傷として累積疲労損傷,クリープ損傷等が挙げられるが,累積疲労損傷に関しては,陽電子消滅法,X 線回折法,さらには磁気的手法(MAE法,BHN 法および磁化曲線法)の有用性が報告されている.クリープ損傷や,疲労損傷で微小き裂の発生を伴う累積損傷の評価には,レプリカ法の有用性が報告されている.一方,硬さの変化を伴うものは,硬さ計測に加えて超音波の音速変化あるいは減衰率等を用いて検査することが可能である.またクリープ損傷については電気抵抗法や密度法による評価も可能である.より詳細は文献(109)を参照していただきたい.応力を直接測定できるものは少なく,ひずみを測定して,応力とひずみの関係式を用いて換算している.測定には各点測定法と全視野法がある.各点測定法は一般には精度は高いが,高ひずみ点を見逃すおそれがあり,全視野法は全面にわたって連続的に測定できるが,一般に精度の低いものが多い.また測定のゲージ長は適用対象物体により選ばれるが,精度と相反する性質があるので,最適のひずみ測定法を選択する必要がある.表13・5に各測定法の長所・短所,得られる情報,適用面を示した.
