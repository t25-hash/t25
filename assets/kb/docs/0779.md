# 13・13・1 変位計

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

引張試験など被測定物形状や負荷形式が比較的簡単な場合には,伸びなどの変形に伴う変位を計測することによって,ひずみが評価できる.ここでは,各点測定用の変位計に注目する.変位計には大きく分けて,接触式と非接触式がある.接触式変位計には,測定子(触針)の機械的変位を,差動変圧器,静電容量変化,ひずみゲージ(13・13・2項)の抵抗変化などを利用して電気量に変換するもの(電気マイクロメータ)や機械的に指針の角度変位量に変換するもの(ダイヤルゲージ)がある.また,測定子変位を光学式スケールや磁気スケールまたは干渉測長光学系を用いて計数方式で計測するもの(ディジタルマイクロメータ)がある.一方,非接触式変位計には,圧縮空気流体素子を利用した空気マイクロメータ,測定プローブ用電極と被測定物の間の電気容量変化を検出する静電容量型変位計,測定プローブの一次コイルによって被測定物の導電体に渦電流を発生させ,これにより誘起されるプローブ内二次コイルの電圧を検出する渦電流型変位計がある.また,投光用・受光用の光ファイバを束ねた測定プローブを用い,ファイバ端と反射面(被測定面)の距離によって変化する受光量を計測する光ファイバ変位計,被測定面にレーザビームを入射した際の乱反射光あるいは正反射光を結像レンズで受け,その結像スポット位置を検出する光三角法レーザ変位計や斜入射型光学式変位計,対物レンズの焦点誤差を検出する光触針式変位計,マイケルソン干渉計などの光波干渉法による変位計がある.
