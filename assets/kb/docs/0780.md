# 13・13・2 電気抵抗ひずみゲージ、光ファイバひずみセン

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

サ電気抵抗ひずみゲージ(electric resistance strain gage)は、基本的には図13・33(a)に示すように抵抗体をフォトエッチングなどでグリッド状に加工して、電気的絶縁体であるゲージベースの上に設置した構造となっている。物体の表面にひずみゲージを接着し、物体が変形したとき、その変形に追随して変化するひずみゲージの電気抵抗変化を、ホイートストンブリッジを組込んだ測定器により増幅して読み取り、ひずみを測定する。抵抗体の種類から分類すれば、温度による抵抗の変化が小さい銅・ニッケル系合金(アドバンスなど)や、ニッケル・クロム系合金(カルマなど)のはくゲージや線ゲージ、シリコンなどの半導体ゲージに大別できる。ゲージベースには主としてエポキシ樹脂などが用いられているが、特に高温使用を目的としたひずみゲージの中には溶接形という金属ベースのもの、また、あらかじめ被測定物の表面にAlOなどを高温で吹付けて絶縁層を作り、この上に抵抗素子を埋め込むもの(溶射型)もあり、最高使用温度は500°C以上である。ゲージ長は通常0.2~10mmである。半導体ゲージでは、半導体のピエゾ抵抗効果によって、ゲージ率が100以上にもなる。ゲージ率が大きいために小型のゲージが利用できるが、温度変動が大きい場合やひずみが大きい場合には適さない。被測定物がLSI等の半導体チップの場合には、その表面に半導体ひずみゲージを直接作り込むこともできる。図13・33(a)に示したような表面接着型のほかに埋込み用もある。また、図13・33(b)に示すようなロゼットゲージと呼ばれる多軸用もある。これを用いて表面内3方向のひずみε,ε,εを計測すれば面内の二つの主ひずみとその方向が評価できる。ひずみは無次元量であり単位をもたないが、一般に非常に小さい量であるため、習慣として1×10のひずみを1(1マイクロストレインとε読む)として表すことが多い。また百分率(%)で表すこともある。ゲージが損傷することなく動作するひずみの限界は、一般用ゲージを室温で使用した場合には、20000~50000程度である。εなお、プラスチックなどの測定を目的とした大ひずみ用ゲージではひずみ限界が200000以上のものもある。水中での使用、湿気εの多いところや長時間の使用にはワックスなどのコーティング剤をゲージ表面に塗布すればよい。光ファイバひずみセンサ(fiber optic strain sensor)では光ファイバの各種光学的特性を利用する。光ファイバの一部(ゲージ部)を被測定物に貼り付けるか埋め込んだ場合、被測定物の変形に追随してゲージ部の長さや屈折率が変化し、伝送光の位相がシフトする。この位相シフトを光波干渉法によって検出すれば、ゲージ部のひずみが計測できる。あらかじめ屈曲させたゲージ部の曲率変化に伴う光伝送損失の変化を検出すれば、ゲージ部のひずみ変化がわかる(光ファイバ式ひずみゲージ)。さらに、ゲージ部コア内に長さ方向の周期的な屈折率変化構造(ファイバグレーティング)を有するひずみセンサでは、ゲージ部でブラッグ(Bragg)の条件を満たす波長(グレーティング周期に比例する波長)をもった光のみが選択的に反射され、変形によってグレーティング周期が変化すると反射波の波長も変化する。この波長変化を光スペクトルアナライザによって検出すれば、ゲージ部のひずみがわかる(ファイバグレーティングひずみセンサ)。このほか、光ファイバに生じるブリルアン(Brillouin)後方散乱光のひずみによるスペクトルシフトに着目した方法もある。
