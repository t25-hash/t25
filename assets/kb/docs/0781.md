# 13・13・3 X 線による応力測定法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

試料表面に照射した入射X 線が、試料表面で反射して出てくるX 線の散乱状態を解析することで、非破壊・非接触的に試料表面の応力状態を測定することができる(X-ray method)。すなわち、試料表面に対して適当な角度ψでX 線を照射したときの反射X線を調べると、その強度がある角度ψ+2ηで極値をとる(図子面(回折面)でX 線が回折するためである。ψ+η方向に反射るX 線を回折X 線と呼び、入射と回折X 線のなす角2θを回折す角と呼ぶ。さらに、測定対象の試料が、多くの金属がうであるように方位を異にする無数の結晶の集そ合体であれば、任意のψ方向のひずみを同方向の回折角から評価でき応。これの原理から、面終的に次式によって試料表力の最るらめれるが求ら。なお、X 線入射角度ψを変化させ、数点のψについて2θを測定し、sinψに対してプロットすると、2θとsinψの関係は直線になる。
