# 13・13・5 赤外線による応力測定

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 )。これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

物体の温度に依存して放射される赤外線を測定して、物体の温度分布を知り、これにより、種々の試験を行う方法をサーモグラフィー(thermography method)という。このような温度の変化を知ることにより、応力測定を行うことができる。固体に引張り・圧縮の力が加わると、体積変化が断熱的に行われ、吸熱・発熱による温度変化が生じる。これは熱弾性(ther-moelasticity)効果として知られる現象であり、弾性限界内では単位要素に作用する力の量(主応力の和の変化量)と発熱量が比例関係にある。したがって、繰返し負荷を受ける物体の温度変化量の計測により、主応力和の変化量を求めることができる。負荷による温度変化はわずかなので、繰返し負荷を作用させて変化を積算して計測する工夫が必要であり、静的な計測には向かない。赤外線応力測定法では、全視野的な応力測定が可能なので、き裂端やはく離端などの応力集中箇所の見逃しがなく、非接触での測定のため微小部分から大構造物まで、形状が複雑なものでも測定できるという特徴がある。
