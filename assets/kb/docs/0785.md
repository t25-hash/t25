# 13・13・7 レーザラマンによる応力測定法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 ).これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

試料にレーザ光を照射したときの反射光として得られるラマン散乱光を測定し,そのスペクトルを解析することで,X 線法と同様に非破壊・非接触的に試料表面の応力状態を測定することができる(laser Raman spectroscopy).すなわち,レーザ光のフォトンと結晶格子フォノン間のエネルギー変換(非弾性散乱)により,反射光には材料固有のスペクトルをもったラマン散乱光が含まれる.結晶格子にひずみが発生すると,フォノンのエネルギー状態がひずみエネルギーに対応して変化するため,ラマン散乱光のスペクトルのピーク位置が変化する.また,このピーク位置の変化量と結晶ひずみ(応力)の間には比例関係があることから,この変化量を測定することで応力の測定が可能となる.直径1 m程度と極めて微小な領域の測定が可能であることから,半導体チップ内部における微細部分の応力評価への適用が試みられている.
