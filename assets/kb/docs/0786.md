# 13・13・8 光弾性による応力測定法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 )。これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

エポキシ樹脂、ガラス、セルロイドのような透明の均質等方体に荷重を加えると、一時的に異方性を示し、光を入射させると複屈折現象を呈するようになる。これを光弾性効果といい、この現象を利用して物体に生じる応力を求めるのが光弾性応力測定法(photoelastic stress measurement)である。おもな手法に、二次元モデルの応力を求める透過光弾性法、三次元モデルの応力を求める凍結スライス法や光散乱光弾性法、構造物の表面に光弾性被膜を接着して応力を測定する光弾性被膜法がある。二次元モデルの応力を求めるときには、まず単色光を偏光子、1/4波長板の順に通して偏光とし、この光を試料板表面より入射させる。光は主応力方向に振動する二つの光に分かれ、主応力面に沿って進行する。二つに分かれた光は屈折率の異なった面を進むため速度が異なり、位相差を生じる。この位相差は主応力差に比例する。位相の異なる二つの光を1/4波長板、偏光子の順に通して合成すれば主応力差の等高線を表す等色線が得られる。一方、主応力の方向を二つの1/4波長板を取り除いたときに等色線とともに現れる等傾線によって求める。
