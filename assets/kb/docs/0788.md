# 13・13・10 格子法,モアレ法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 ).これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

格子法(grating method)は試料の表面に格子模様など目印となるマークを描き,変形前後のマークの位置変化を測定して,ひずみおよび応力を算出するものである.押出し加工時の塑性変形の測定や,ゲージ長の短いことが要求されるき裂先端付近の塑性変形の測定などに用いられる.モアレじまを利用した形状やひずみの測定法をモアレ法(moire썝method)という.等間隔の平行線からなる試料格子を試料に描いておく.これに間隔,方向ともに等しい基準格子を重ねる.試料が格子線に垂直な方向に伸びると,それまで見えなかったモアレじまが観察できる.このときモアレじまは定変位線となり,しまの間隔は伸びひずみに反比例する.モアレ法により試料の変形を拡大して観察することができる.試料格子の間隔を光の回折が起こる程度に細くしておき,2方向からレーザ光を照射し,変形前後でそれぞれのレーザ光の回折角度が変化する.これを利用して試料の変形量を測定するモアレ干渉法もある.モアレ法は金属材料のみならず電子デバイス,複合材料および高分子材料の変形計測に用いられる.
