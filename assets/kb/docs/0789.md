# 13・13・11 ホログラフィー干渉法,スペックル法,コー

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 ).これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

スティックス法ホログラフィー干渉法(holography interferometry)では,試料にレーザ光を照射して得られる物体光と参照光を重ねてできる干渉模様を感光材料に記録したホログラムを利用する.おもな手法に,二重露光法,実時間法,時間平均法がある.二重露光法は変形前後の干渉模様を同一の感光材料に記録し,変位に相当する干渉じまを得る方法である.実時間法は変形前に作成したホログラムからの再生光と変形中の試料からの反射光を干渉させ,変位に相当する干渉じまを観察する方法である.時間平均法は試料が変形中のすべての時間露光させる方法である.ホログラフィー干渉法は観察方向の変位は測定しやすく,実機の振動実験に適している.スペックル法(speckle method)は,レーザ光を粗面に照射すると,粗面の各点で散乱した光が不規則な位相関係で干渉し合い生じるスペックルパターンを利用する方法である.スペックル法には,スペックル写真法,スペックル相関法およびスペックル干渉法がある.スペックル写真法は,変形前後のスペックルパターンを二重露光して得られたネガにレーザ光を照射し,生じるヤングじまの方向と間隔からスペックル移動の方向と大きさを求める方法である.スペックル相関法は変形前後のスペックル移動量を直接測定する方法である.スペックル干渉法は変形前後のスペックル像の明るさの差を求めて変位成分の等高線を得る方法である.スペックル法はホログラフィー干渉法に比べると光学系の配置が単純であり,テレビカメラで直接像をとらえることができるので電子化が容易である.き裂を有する透明な試料に負荷を加えると,き裂先端付近で板厚や屈折率が変化する.平行光を照射すると,き裂先端付近で光路程が変化して光の方向が曲げられる.このため,スクリーン上に光のこない部分ができ,その境界は光が集まって明るい部分となりコースティック像が生じる.この像の形と大きさから応力拡大係数の値,,を直接求める方法をコースティックス法(causticsmethod)という.
