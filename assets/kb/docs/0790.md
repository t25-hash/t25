# 13・13・12 切抜き、切込みおよび穴あけによる残留応力測定

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 )。これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

法残留応力が生じている試料に切込みを入れる、あるいは、その一部を切抜くと、部分的に残留応力が解放されて試料が変形する。図13・35(a)は、切込みによる変形量を直接測定する場合の例であり、図中の式を使って円管壁面の周方向残留応力が求められる。一方、図13・35(b)は、切抜き(穴あけ)前に接着したひずみゲージによって切抜きに伴う板の変形をひずみ量として測定する場合であり、ひずみ量から図中の式を使って残留応力の主値が求められる。
