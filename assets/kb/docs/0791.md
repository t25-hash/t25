# 13・13・13 画像処理法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・34 )。これは角度ψ(=ψ+η)の向きを法線方向とする結晶格

高速化、高精度化を図るために、光弾性応力測定法、モアレ法、光学的干渉法などのひずみや応力の測定法では、測定対象領域全体の画像をディジタル化してコンピュータに取込み画像処理(imageprocessing)し、ひずみ分布や応力分布を求めることが一般的になってきた。ディジタル化された画像の画質は、空間フィルタや周波数帯域フィルタを用いて高周波成分を有する雑音を除去し平滑化される。平滑化され濃淡のある画像を白と黒に2値化し、得られたしま模様を細線化してひずみや応力を求めるためのデータを得る。ホログラフィー干渉法やスペックル干渉法などでは変形前後の画像を干渉させ、しま模様を得るために画像処理法が用いられる。変形格子をコンピュータ上で平行移動して重ね合わせ等ひずみ線を表す干渉じまを得るモアレのシフト法やコンピュータ上で基準格子を作る走査モアレ法も行われている。モアレ法、光学的干渉法などしまの間隔がひずみに関係しているときには、しま模様を波紋ととらえて、周波数解析や位相解析を用いてひずみを求める方法がある。周波数解析や位相解析にはフーリエ変換、ガボール変換、ウェーブレット変換およびアダマール変換等が用いられる。各場所での変形前後の周波数を求め、周波数の差を変形後の周波数で割ってひずみを求める。また、変位を表す位相差を微分することによりひずみ分布を求めることができる。変形格子のスペクトルと位相解析のみで変形を解析するフーリエ変換モアレ法も行われている。近年の技術革新に伴い、エレクトロニクスデバイスやマイクロマシンといった従来とは異なる形態の構造物が広く使用されている。これらを構成する材料は、薄膜に代表されるように一般に寸法が小さく、また複数の材料が接合されて機能する場合が多いため、機械工学が従来から取扱ってきた材料とは異なる評価手法が要求される。ここではこれらの微小材料と総称される材料の評価手法について概説する。
