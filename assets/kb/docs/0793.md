# 13・14・2 内部応力評価試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・14 微小材料の試験

a. 基板の反り測定基板上に薄膜を形成すると,薄膜の内部応力により基板に曲げ変形が生じる.このときの基板の曲率を計測することにより,薄膜材料の内部応力を求める.片持はり上に薄膜を成膜し,これによる組合せはりの反り変形から評価することも行われている.b. ラマン分光法(Raman spectroscopy)応力作用下での結晶格子振動の周波数変化を利用した応力測定法である.レーザ照射によるラマン散乱光の周波数ピークの移動を計測するが,顕微光学系を用いることによりレーザスポット径をmオーダまで絞ることができるため,微小領域の内部応力の評価にしばしば用いられる.高空間分解能,非接触評価という特徴に加えてin-situ 評価が容易に可能であるが,一方,測定対象が結晶性の材料に限られ,また金属材料についての計測ができない.c. X 線回折法(X-ray diffractometry)応力作用下の結晶格子面間隔の変化に起因したX 線回折角の変化を利用した応力計測法である.多結晶薄膜材料に関しては通常の材料と同様に試料表面に対して傾斜角ψを有する格子面での回折角θを計測し,2θとsinψの線形関係を得て,この傾きから応力を求める(sinψ法)が,非常に薄い膜を対象とする場合はX 線の入射を低角度とし,回折角を直接評価する方法も用いられる.非接触での応力評価が可能である一方,測定対象が結晶材料に限られ,またX 線照射径を絞ることができないためラマン分光法に比し,空間分解能が劣る.d. 基板分離後の薄膜変形計測法薄膜と基板を分離し,内部応力の解放後の変形を利用して内部応力を評価するものである.測定する変形量が小さいため一般に精度は悪い.次の2種類の変形を計測する方法がある.. 伸縮計測片持はり,T 型やH 型に形成された試験片の基板分離後の長さ変化を計測することにより内部応力を評価する.. 座屈計測犠牲層エッチングにより種々の長さの両端固定はりを同一の薄膜状材料に形成する.圧縮の内部応力が存在する場合は特定の長さ以上のはりで座屈が発生するので,この臨界長さを求め,座屈発生の臨界条件から内部応力を評価する.
