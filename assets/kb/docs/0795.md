# 1・2・1 分子の状態から見た流体

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

すべての物質は分子の集合体で,固体(solid),液体(liquid)または気体(gas)のいずれかの相(phase)の形態で存在するが,物質がどの相で存在するかは,分子のポテンシャルエネルギーと分子の不規則な自由運動のエネルギーとの大小関係による.ポテンシャルエネルギーは,図1・1に示すように,分子どうしに作用する力(分子間力)に関係し,分子の中心からある半径以上離れると距離の二乗に逆比例する引力として作用し,近すぎると斥力として作用する.固体では,分子間力が強いので分子どうしの引力と斥力が打消し合う相対的平均位置に分子が固定され,平均位置を中心として振動的運動を行う.液体では,個々の分子が分子間力の作用を強く受ける距離を保ちながら自由に移動できる状態にある.分子どうしが近づきすぎると斥力が働き,離れすぎると凝集力が作用するので,一定量の液体が占める体積は限定されており,その体積より小さい容器に入れることはできないが,大きな容器に入れると気体を含む空間が生じて液体との間に自由表面が形成される.気体は,自由運動のエネルギーが大きいので,分子どうしの間隔が大きく分子は無秩序に激しく運動して衝突を繰返しており,分子間力は衝突の際にわずかに作用するだけで,気体の体積は無限に広がろうとする性質をもっている.このように物質の相を決定する分子相互間のポテンシャルエネルギーと分子の不規則運動による運動エネルギーは,内部エネルギー(internal energy)あるいは熱エネルギー(thermal energy)と呼ばれ,分子を構成する原子の結合エネルギー(化学反応に伴って放出または吸収される化学的エネルギー)や原子核を構成する素粒子の結合エネルギー(核融合や核分裂に伴って放出されるエネルギー)などと異なった形態で物質内に保有され,物質の組成を変えることなく外に取出したり,外から与えることができる.これらの分子の微視的な運動の統計的性質によって物質の巨視的な物理的性質は決まる.固体が力を加えると変形するが取り除くと元に戻る性質,いわゆる弾性(elasticity)を有するのは分子間力が強いためであり,これを上回る力を加えると破壊する.これに対して液体と気体は,分子どうしの相対的位置が容易に変化するので,体積変化を伴わなければ極めて小さい力(せん断力)が加わるだけで容易に変形して流動する.また,力を取り除いても元の形には戻らない.このように液体と気体は力学的に共通の性質を有しているので,総称して流体(fluid)と呼ばれる.
