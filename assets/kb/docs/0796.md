# 1・2・2 連続体としての流体

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

流体の巨視的な流動現象を支配するのは流体を構成する個々の分子の運動の結果である。しかし、ある流動現象を考える際にその代表長さが分子相互の干渉距離に比べて十分大きく代表時間が分子相互の干渉時間に比べて十分長ければ、個々の分子の運動に関わりなく分子の集合体としての統計的性質が流動現象を支配する。したがって、統計的性質が空間的・時間的に連続である限り、流体を連続図1・1 分子間力ポテンシャル体(continuum)として取扱うことができる。例えば、われわれがつねに恩恵を被っている水や空気について、一般の工学分野では点とみなせる1辺1 mの微小立方体に含まれる分子の数は、常温の水で約3.4×10個、大気圧の空気で約2.7×10個であり、取扱う物体の寸法に比して分子間の平均距離は無視できるほど小さい。したがって、流体の状態を表す物理量は流体内で連続的に変化すると考える連続体の概念が適用できる。連続体の概念の導入により数学的に微分および積分の操作が可能になる。連続体として流体運動を扱う場合、しばしば流体粒子(fluidparticle)という言葉が用いられる。これは流体の分子のことではなく流体運動の最小単位と考えられる微小部分であり、その内部では流体の状態を表すすべての物理量が等しいと近似できる。したがって、流体粒子の大きさは取扱う問題によって異なる。
