# 1・2・3 流体の状態を表す量

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

a. 密度、圧力、温度の定義流体の巨視的な状態を表す物理量(分子の集合体としての統計的性質)は状態量と呼ばれ、密度、圧力、温度などがある。密度(density)は、その流体が占める空間の単位体積当たりの質量(kg/m)で定義され、流体を構成する分子の重さと単位体積中に含まれる分子の数によって決まる量である。圧力(pressure)は、流体内の任意の面に垂直に作用する単位面積当たりの力(N/m,Pa)である。分子間隔が小さい液体では、圧力は主として分子間力を介して作用する。しかし、分子間隔が大きい気体の場合、圧力は単位時間に単位面積を通過する(または単位面積の固体表面に衝突する)分子の運動量に等しく、分子の重さ、個数、速度に関係する。このことは、運動量の時間的変化が力と等価であるというニュートンの運動の法則を考えると理解できる。分子が存在しない完全真空の状態を絶対圧力が零と定義すれば、内部エネルギーが存在する限り分子は運動するので、負の絶対圧力はあり得ない。温度(temperature)は、上述の内部エネルギーに関係する量である。同じ物質が長時間接していると、分子の振動的運動または不規則運動を介して両者の単位質量当たりの内部エネルギーは等しくなり熱的に平衡状態に達するが、異なる物質が接しているときの平衡状態では二つの物質の内部エネルギーは等しくない。物質が異なっても熱的平衡状態にあれば等しくなる量として温度が定義され、温度差は平衡状態からのずれを表す尺度である。すなわち、互いに接する物質の内部エネルギーは温度が異なると物質間を移動し、等しいときは移動しない。温度は、標準大気圧(1atm=0.1013MPa)における水の融点と沸点の間を100等分した尺度で計測され、水の融点を基準にした温度を摂氏温度(0°C)、同じ尺度で内部エネルギーが零の状態を基準にした温度を絶対温度(K)という。温度差1°C(1K)に対応する内部エネルギーの差は物質によって異なる(1・3・5項参照)。状態量には圧力、温度、密度のほかに、単位質量当たりの流体のもつ内部エネルギー(J/kg,m/s)、エンタルピー(J/kg,m/s)、エントロピー[J/(kgK)]などの熱力学的状態量がある(α5編「熱工学」参照)。b. 相変化と状態曲面1種類の分子からなる(1成分系の)気体から圧力一定の条件で熱を除去すると温度が下がり、ある温度に到達すると局所的に液化が始まるが、気相と液相が共存する領域では除去熱は潜熱に取られて温度は変化しない。この温度を飽和温度(saturation temperature)といい、このときの圧力をその温度に対する飽和圧力(saturation pressure)という。この状態からさらに熱を除去すると液相のみの状態となり、液相と固相が共存する領域を経て固体になる。気相と液相が共存しうる圧力と温度の領域は限られ、その最大値を臨界圧力(critical pressure)および臨界温度(critical temperature)という。臨界圧力より高い状態で熱を除去すると密度が滑らかに変化して液体になる。一方、圧力が十分低い状態で熱を除去するとある温度で気相と液4―3図1・2 水の立体状態図相と固相が同時に共存するときがある。この圧力と温度を三重点という。三重点より低い圧力で熱を除去すると局所的に固化が始まり気相と固相の共存する温度に到達する。このような相の変化を圧力、温度、密度を直交3軸に取った空間で表すと一般に図1・2のようになり、この面を状態曲面、曲面を表す式を状態方程式という。二相状態から気相、液相、固相に変わる曲線をそれぞれ飽和蒸気線、飽和液線、飽和固体線という。水の場合には三重点から臨界点近くの圧力の領域で固相の密度が、固液二相状態の密度よりも小さく、この領域で状態曲面はかなり複雑になる。
