# 1・2・5 気液界面における蒸発・凝縮と溶解

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

液体と気体の界面では絶えず液体の分子は気体中に飛び出し、気体分子は液体中に飛び込んでいる。液体と同じ分子の気体(蒸気)が界面を介して往来する現象において、気体中に飛び出す分子のほうが多いときが蒸発(evaporation)、少ないときが凝縮(conden-sation)である。飛び出す分子の数と飛び込む分子の数が等しくなると平衡状態に達する。蒸発か凝縮かは気体中に含まれる蒸気の分子数の割合による。気体中における蒸気の分子数の割合は、同じ体積を蒸気分子が単独に占めるときの分圧(partial pressure)に比例する。蒸気の分圧を蒸気圧(vapor pressure)といい、平衡状態に達したときの蒸気圧を飽和蒸気圧(saturated vapor pressure)4―4流体工学という。すなわち、蒸発現象は飽和蒸気圧より蒸気圧が低いときに起こる。飽和蒸気圧は温度の関数であり、液体温度の上昇とともに高くなる。液体の圧力が飽和蒸気圧より急激に下がると沸騰して液体は蒸気になる。したがって、常温の液体でも流れの中に局所的にその液体温度における飽和蒸気圧よりも圧力の低い場所があると、そこに蒸気泡ができる。このような現象をキャビテーション(cavitation)と呼ぶ。キャビテーションは液体に溶解している気体を核として成長する。溶解(dissolution)は、液体に異種の気体、液体または固体物質が混合して均質な状態になる現象である。液体に含まれる異種物質の質量の割合を溶解度(solubility)という。気液界面において液体に含まれる気体の溶解度が低いと、飛び込む気体分子の数が外部に飛び出る分子の数より多く溶解が進行する。液体中の気体分子が増えるにつれて界面で気体中に戻る分子が増加し、飛び出す分子の数と飛び込む分子の数が等しくなると平衡状態に達する。平衡状態になる溶解度を飽和溶解度(saturated solubility)という。飽和溶解度は圧力と温度の関数であり、一般に気体の圧力にほぼ比例して増加し、また液体の温度が低いほど大きい。
