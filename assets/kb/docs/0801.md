# 1・2・6 液体界面の力学的性質

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・2 流体の諸性質

a. 表面張力分子のポテンシャルエネルギーにより液体分子には互いに凝集力が働く。液体内部では凝集力は等方的に釣合う状態にあるが、液体と気体との界面すなわち自由表面ではこれが非等方的になり、液体の表面が収縮しようとする張力作用を生じる。液面上の単位長さ当たりの張力を表面張力(surface tension)という。表面張力の値は接触している液体と気体の組合せにより異なる。また、温度が高いほど小さくなる。b. 毛細管現象液体の分子間に凝集力が働くのと同様に、液体と固体の間にも分子間力が働く。これを付着力(adhesion)という。固体壁面に液面が接触するところでは、固体と液体の付着力と液体の凝集力との大小関係に応じて、壁面と液面の接触角が変化し、液面の形状は壁面で局所的に凸状または凹状になる。付着力のほうが大きいときは液面が壁面をぬらすように凹状になり、付着力が小さいと壁面をぬらさないように凸状になる。そのため、細い円管を液面に垂直に挿入すると、円管の壁面をぬらす場合には管内の圧力MPa12.81812.49411.63810.89510.2429.66499.15008.68796.94075.78164.33643.470265.19563.35158.58154.54451.07048.04245.37643.00534.20228.46021.36117.119131.36127.27116.88108.29101.0394.78589.34484.55067.01655.74741.91033.657480.48469.81441.02415.45392.69372.37354.15337.75275.57234.01181.07148.28646.21637.15611.94588.48566.61546.19527.10509.24435.31380.74306.34257.55圧力MPa1023.91023.01017.91010.11000.4988.84975.78961.34869.29734.71218.58123.531045.51044.01037.31028.91019.11008.0995.71982.26899.44790.80482.82266.05液面は凹状になって上昇し、ぬらさない場合は凸状となり下降する。この現象は毛細管現象(capillarity)としてよく知られている。連続体としての流体の流動現象は流体の物理的性質(physicalproperties)の影響を受ける。作動流体が異なると流れ特性や圧力損失などは変わることがあるが、それらは流体の物理的性質の違いによって生じる。流体の流動に影響する物理的性質としては、密度、粘度、表面張力、比熱などがあり、それらは一般に温度・圧力に依存する。物理的性質は略して物性、その値は物性値と呼ばれる。
