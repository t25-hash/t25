# 1・3・2 圧縮率・体積弾性係数・音速

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・3 流体の物性

圧力が増すと密度が増大し体積は減少する。この性質を圧縮性という(1・2・4項参照)。いま、圧力がdに増加することによって密度ρがdρ変化し、また比体積がd変化したとすると、圧縮率(coefficient of compressibility)βは圧力変化に対する密度変化の割合として式(1・3)で定義される。=(1・6)圧力変化が高速で微小な場合には圧縮・膨張が断熱的に行われるので、式(1・6)が成り立つ。標準大気圧・常温の空気の体積弾性係数は=0.14MPaである。b. 液体の体積弾性係数表1・6に水の体積弾性係数を、表1・7に各種流体の体積弾性係数を示す。液体は体積変化を起こしにくいので、気体に比べて大きな値をとる。なお、液体の場合、体積弾性係数の圧力依存性は極めて小さい。c. 音速表1・8に空気の音速(acoustic velocity)を示す。気体の音速は絶対温度の平方根に比例するが、圧力に対する依存性は小さい。完全気体とみなされる場合(1・3・6項参照)、音速は式(1・7)で与えられる。液体の音速の圧力・温度についての依存性は極めて小さいことがわかる。表1・10に種々の流体中の音速を示す。
