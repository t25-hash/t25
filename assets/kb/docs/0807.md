# 4・2・6 磁気を利用した機械量センサ

> **章**: 第4章 電気計測
> **節**: 4・2 トランスデューサ

a.渦電流式変位センサ（eddy current type displacement sen-
sor ) 図4･25に渦電流式変位センサの原理を示す．すなわち，1
MHz程度の高周波電流を流したコイルに導体を近づけると，導体
中に磁束の変化を妨げるように渦電流が流れる．この電磁誘導作用
によって，コイルのインピーダンスが変化する．コイルと測定対象
となる金属の間の距離，すなわち，変位が小さいと渦電流が多く流
れ，コイルインピーダンスの変化も大きくなる．
コイルインピーダンス変化をブリッジ回路で取出して，電圧信号
とすることによって，変位にほぼ比例した信号を取出すことができ
るが，その直線性を改善するためリニアライズ回路が使用されてい
る．一般に，このリニアライズ回路は，測定対象に対応して調整・
作成されるので，センサの製作時と異なる金属を測定対象とすると
きには注意が必要である．
渦電流式変位センサには，以下の特徴がある．油，水，ほこりな
どの影響は受けにくい．比較的小型のセンサヘッドで高速応答
（DC～20kHz)，高分解能（0. 01m）である．
b.応力測定用磁気異方性センサ 強磁性体に応力が加わると
磁わい効果等により，磁気特性が変化するため，磁気特性変化から
応力を非接触・非破壊で計測する試みがいろいろ行われている．磁
気特性変化を利用した応力測定法は，実構造物の応力測定に適用で
き，突合せ溶接した溶接部近傍の残留応力分布測定，鋼板の張力測
定，レール軸応力測定，橋りょうの応力測定，ボルトの軸応力測
定，高周波焼入れ層の判定などへの応用例が報告されている．
磁気異方性センサ（magnetic anisotropy sensor ）は，測定対象
の磁気異方性を測定するプローブと，励振回路や同期検波回路から
なる信号処理部から構成され，測定対象内の応力を被接触で測定す
ることができる．
図4･26は，磁気異方性センサの原理を示している．正の磁わい
定数をもつ測定対象に，応力を印加すると，引張方向の透磁率が大
きくなり，直交方向の透磁率が小さくなる．すなわち方向別の透磁
率分布の異方性が生じる．磁気異方性センサを測定対象に近づけ励
振コアを励振し，測定対象上でセンサヘッドを回転させると，検出
コイルに電圧が誘起される．誘起された電圧を同期検波し直流電圧
にし，これを検出電圧とすると以下のように表される．
＝σ－σcos2θ V (4･24)
ここで，σ ，σは測定対象に加わっている応力であり，θはプロー
ブの方向と主応力σ方向の間の角である．
検出電圧は，センサヘッドの回転角πに対して，一周期の信号
になる．この信号の最大値と最小値の差をセンサの出力電圧とす
る．この信号出力から測定対象の磁気異方性の大きさと方向がわか
る．これをもとにして，主応力（principal stress ）差σ－σと
その主応力の方向がわかる．
図4･27は，厚さ1mmの低炭素鋼板（SPCC）に曲げ応力を加
えたときの磁気異方性センサの出力特性の実験値と計算値を示して
いる．磁束の浸透深さを考慮した計算値は，実験値によく一致して
いる．
磁気異方性センサによって，主応力方向θと主応力差σ－σ
を測定できる．ここで，せん断応力差積分法を適用すると，任意方
向の平面応力を測定することができる．この方法は，比較的単純な
応力状態の場合，ひずみゲージで応力開放により測定した値との差
は約±15MPa，応力こう配がある場合でも約±20MPaの差で測
定できることが報告されている．
磁気ひずみによる応力測定法は，応力の絶対値測定が可能であ
り，塗装などを除去するなどの必要がなく，皮膜上から非接触で応
力を測定できるという利点を有している．
トルクによって，シャフト表面に応力が生じることを利用して，
磁気ひずみ方式のトルクセンサが開発研究されている．
