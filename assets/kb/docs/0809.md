# 1・3・7 蒸気

> **章**: 第1章 流体工学の概要と流体の諸性質
> **節**: 1・3 流体の物性

気体の温度と圧力を下げていくとやがては液体になる。液相に近い温度・圧力状態の気体を蒸気(vapor)という。蒸気と液体が共存する平衡状態を飽和状態といい(1・2・5項参照)、その温度・圧力を飽和温度(saturation temperature)・飽和圧力(saturationpressure)という。これら蒸気の特性は完全ガスで近似することはできず、物性値は蒸気表、蒸気線図による必要がある。表1・19に水の飽和状態における液と蒸気の物性値を示す。水の蒸気の特性は機械学会蒸気線図に詳しい。
