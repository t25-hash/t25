# 4・4・1 PC応用計測システムの構築

> **章**: 第4章 電気計測
> **節**: 4・4 計測システム・測定器

PC（パーソナルコンピュータ）のカードススロットに取付け可
能なA- D変換ボードが種々市販されている．これらの入力電圧は
通常0～5V，0～±5V，0～12Vなど，サンプリング周波数10
MHz程度，分解能12bit程度，多チャンネル入力にも対応するも
のもある．また，PCへのデータ転送速度のボトルネックを解決し
て高速サンプリングに対応するために搭載したメモリにいったん変
換結果を蓄えるものもある．またPCから回り込む雑音やグランド
ループに起因する雑音を避けるために，入力回路部をPC側とは絶
縁した構造のものが多い．
入力電圧範囲についていえば，適当なプリアンプ，プログラマブ
ルゲインアンプ，アッテネータなどを付加すれば，さらに入力範囲
を拡大することも可能である．
PCとは古くから用いられているGPIB（general purpose inter-
face bus )，VMEバス，最近のものではPCI （peripheral compo-
nent interconnect )，ISA（industry standard architecture)，
USB（universal serial bus )，PCMCIA（personal computer
memory card international association）バスなどを介して接続さ
れる．このようにさまざまなバス形式のものがあるので採用に当た
っては使用するPCとA-D変換ボードのバス形式を確認すること
が必要である．
一方PCのソフトウェアについても，入力データを表形式に整理
することをはじめ，グラフ化して表示するもの，オシロスコープの
ように表示するもの，入力データに統計処理，高速フーリエ変換を
施すものなどが種々用意されている．このように考えると，入力信
号の性質を考慮すれば，これらの信号を観測したりさまざまな処理
を施したりすることも可能であり，新たな処理を試みたり開発する
に当たっては非常にフレキシブルである．一方，一般の測定器は，
このようなベースの機能を特化・高度化してユーザインタフェース
を充実して専用化したものともいえよう．事実，最近では画面によ
るユーザインタフェースを特徴とするPC用のオペレーティングシ
ステムが動作する計測器が登場している．
信号処理のためにプログラムをすることも可能であるし，ほかの
ソフトウェアを利用する，あるいは，種々の信号処理を試したり，
新たな処理を検討したりするときにはこのようなシステムが有効な
場合もある．特に同様な計測を繰返し実施する場合にはPC応用計
測システムを構築することを検討してみる意義がある．
