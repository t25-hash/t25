# 2・1・3 圧力の単位

> **章**: 第2章 流体静力学
> **節**: 2・1 圧力

圧力の単位は国際単位系(SI単位)ではパスカルPa(N/m,kg・m・s)であるが、相当する0°Cの水銀柱の高さ(mmHg)、相当する4°Cの水柱の高さ(mAq)、標準大気圧(2・3・2項参照)を基準にした気圧(atm)など、多くの単位が使用される。特に真空工学では、絶対圧力をmmHg で記述したTorr(トル、1Torr=1mmHg)がよく使われる。工業界では古くは1kg の質量の物体が重力により受ける力1kgf(キログラム重)を力の単位としてkgf/mを圧力の単位とする工業単位系、これをポンド・インチ系で表したlbf/in(psi)、1kgf/cmを1気圧と考える工業気圧(at)が使用され、世界各国で単位系をSI単位に統一することになった現在でも、慣用的に使用されることがある。また、気象学では大気圧に近い10Paを単位とするバールbar(またはb)が使用されていたが、SI単位系で表示するために1mb(ミリバール)を1hPa(ヘクトパスカル)として表示するようになった。これらの単位をPaに換算すると以下のようになる。1atm=101325Pa1mmHg=1Torr=133.322Pa1mAq=9806.65Pa1bar=1000mb=1000hPa=100000Pa1kgf/cm=10000kgf/m=1at=98066.5Pa1psi=6894.76Pa真空状態を基準に測った圧力を絶対圧力(absolute pressure)、大気の圧力すなわち大気圧を基準に測った圧力をゲージ圧力またはゲージ圧(gage pressure)という。すなわち、(絶対圧力)=(ゲージ圧力)+(大気圧)である。大気圧より低い圧力をゲージ圧力で呼ぶとき、特に真空圧(vacuum pressure)ということがある。絶対圧力とゲージ圧力を特に明確に区別する必要があるときは、それぞれの単位のあとに、absおよびgage(またはg)を付ける。静止流体中の任意の点における力の釣合い条件は式(2・2)で与えられる。ρ옿옿=ρ옿옿=(2・2)ρ옿옿=ここに、,,:直交座標、,,:流体の単位質量当たりに作用する外力の,,方向成分、ρ:流体の密度、:圧力である。式(2・2)は流体静力学(fluid statics)の基礎式で、流体に作用する外力(体積力)は流体中の圧力こう配(pressure gradient)と釣合うことを示している。偏微分に関する公式よりd=옿옿d+옿옿d+옿옿dが成立するので上式を式(2・2)に代入すれば、圧力の変化dはd=ρd+d+d(2・3)圧力が等しくなる面を等圧面といい、一つの面上で圧力の等しい点を結んだ線を等圧線という。静止流体中の等圧面は、式(2・3)においてd=0、すなわち式(2・4)で与えられる。d+d+d=0(2・4)
