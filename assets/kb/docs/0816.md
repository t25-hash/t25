# 2・3・2 大気中の圧力

> **章**: 第2章 流体静力学
> **節**: 2・3 重力場における圧力

図2・1のように、80cmより長い片方を閉じたガラス管に水銀を満たし、水銀を入れた容器の中に立てると、ガラス管内の水銀面の高さは容器の水銀面から約76cmになる。常温における水銀の飽和蒸気圧(=2.73×10Pa)は極めて低いので、その上部は真空と考えてよい。これはトリチェリ(Torricelli)の実験としてよく知られている。大気圧において、温度が0°Cで水銀柱の高さが760mmになるときを標準大気圧(normal atmospheric pressure)という。標準大気圧は1.01325×10Pa(1013.25hPa)である。
