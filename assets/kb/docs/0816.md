# 2・3・2 大気中の圧力

> **章**: 第2章 流体静力学
> **節**: 2・3 重力場における圧力

図2・1のように、80cmより長い片方を閉じたガラス管に水銀を満たし、水銀を入れた容器の中に立てると、ガラス管内の水銀面の高さは容器の水銀面から約76cmになる。常温における水銀の飽和蒸気圧(욦=2.73×10Pa)は極めて低いので、その上部は真空と考えてよい。これはトリチェリ(Torricelli)の実験としてよく知られている。式(2・6)によれば、大気の圧力は真空(絶対圧力=0)より水銀の密度ρ=13.595×10kgm×重力加速度=9.80665ms×水銀柱の高さ=0.76mだけ高いことを示している。大気圧において、温度が0°Cで水銀柱の高さが760mmになるときを標準大気圧(normal atmospheric pressure)という。標準大気圧は1.01325×10Pa(1013.25hPa)である。大気圧は高度(altitude、海抜)によって変化し、海抜욠=0m、温度288.2K(15°C)、圧力101.3kPaの状態を基準とした場合、욠(m)における温度は=288.2-6.5×10욠K(2・7)となる。空気は完全気体の状態方程式=ρ[=287.0JkgK、空気のガス定数]に従うものと近似して、これらの関係式を式(2・5)に代入し、=9.807msとして積分すると、海抜욠(m)における大気の圧力と密度は式(2・8)(2・9)で与えられる。=101.3×1-2.26×10욠kPa(2・8)ρ=1.226×1-2.26×10욠kgm(2・9)図2・1 トリチェリの実験
