# 2・3・3 マノメータ

> **章**: 第2章 流体静力学
> **節**: 2・3 重力場における圧力

図2・2のように,圧力を測定したい流体(密度ρ′)と混ざらない密度ρの液体を鉛直に立てたU 字管に入れ,測定点A とU 字管をつないで液面の高さの差を読むことにより,点A の圧力がわかる.このように,密度が既知の液体の液面高さの差によって圧力を計測する計器をマノメータ(manometer,液柱計)という.いま,点A の圧力を,点B の圧力をとすれば,式(2・6)を利用して下方の液面の圧力の釣合いより,+ρ′=+ρ∴=+ρ1-ρ′ρ(2・10)測定流体が気体の場合,ρ′はρに対して無視できるので,=+ρ(2・11)2点間の圧力差を計測する場合は,両端をそれぞれの測定点につなげばよい.圧力差を計測するマノメータを差圧マノメータという.
