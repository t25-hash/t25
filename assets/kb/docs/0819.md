# 2・3・5 液体中の曲面壁に働く力

> **章**: 第2章 流体静力学
> **節**: 2・3 重力場における圧力

a. 水平分力考える方向に垂直な鉛直面上へ曲面を投影すれば、曲面に作用する力の大きさと作用線は、投影された平面上に働く力と等しくなる。図2・4において図2・2 マノメータ図2・3 平面壁に作用する全圧力および圧力の中心4―11方向の分力:=ρd욨(2・15)作用線の座標:η=d욨d욨(2・16)作用線の座標:ζ=d욨d욨(2・17)方向の分力:=ρd욨(2・18)作用線の座標:ζ=d욨d욨(2・19)作用線の座標:ζ=d욨d욨(2・20)ここで、d욨,d욨は曲面上の微小面積dを,面に投影した面積である。また、各分力作用線の座標ξ,η,ζの添字は分力の方向を示す。b. 鉛直分力曲面の鉛直分力は、曲面を底面としその上方に液面まで立つように考えられた液柱の重量に等しく、その作用線は液中の質量中心(重心)を通る鉛直線になる。鉛直分力:욨=ρd=ρ(2・21)作用線の座標:ξ욨=d(2・22)作用線の座標:η욨=d(2・23)ここに、d:dを面に投影した面積、:曲面上に液面まで伸びた柱体の体積である。
