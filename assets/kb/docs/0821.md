# 2・3・7 浮体の安定

> **章**: 第2章 流体静力学
> **節**: 2・3 重力場における圧力

図2・5のように、浮力を受けて浮かぶ物体を浮体(floatingbody)という。浮体が静止しているとき、重力と浮力の作用線は一致し、これを浮揚軸という。また、液面が物体を切断する面を浮揚面、その図心を浮面心という。図2・4 曲面壁に作用する圧力図2・5 浮体(静止)4―12図2・6のように、浮体を釣合い状態から微小角だけ傾けたとき、新しい浮力の作用線と元の浮揚軸との交点M をメタセンタ(metacenter)という。メタセンタの高さをとすると、浮体の安定性はの正負で判断される。
