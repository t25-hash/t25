# 2・4・1 相対的静止

> **章**: 第2章 流体静力学
> **節**: 2・4 相対的圧力平衡

流体が運動していても，すべての流体粒子の相対的な位置関係を
変えない場合には，静止状態にある流体としての取扱いが可能で，
運動する座標系に対して流体静力学の圧力平衡方程式(2･2)が適用
できる．これを相対的静止（relative rest）の状態という．この場
合には流体に作用する外力として，重力以外にダランベールの原理
に基づく加速度による慣性力を考慮する．
