# 5・1・2 シミュレーション

> **章**: 第5章 技術情報システム
> **節**: 5・1 情報解析

a.数値シミュレーション シミュレーション（simulation）
とは，もともと模擬実験を指す．ある問題に対して，対象を模擬し
た装置による実験を行い，問題の分析や予測を行う手法である．コ
ンピュータシミュレーションは，この実験装置をコンピュータ上に
生成する方法であるが，特に数値モデルに従った計算を行う場合を
数値シミュレーション（numerical simulation）という．シミュレ
ーションは，実システムで試すには時間や費用がかかりすぎる，あ
るいは危険が伴うなどの場合に有効な方法であり，自然科学の問題
から社会経済の問題まで幅広く利用することができる．
物理現象等の自然科学の問題を例にすると，数値シミュレーショ
ンの処理は一般的に次のような手順で行われる．
.定式化 まず対象とする現象に対して，保存則等を基に
（偏）微分方程式等の形式による数学モデルとして定式化を行う．
この方程式と問題に対して与えられる初期条件，境界条件を合わせ
て支配方程式という．
.離散化 数学モデルとして与えられた支配方程式から，
時空間での離散化近似（差分近似，有限要素近似，境界要素近似
等）を行い，代数方程式で表される数値モデルを構築する．この
際，解の安定性，計算精度，演算の高速性等を備えた数値モデルの
構築が必要である．
.計算の実行 数値モデルに従ったプログラムを計算機上
に作成し，プログラムの実行により数値解を求める．数値計算用の
プログラム言語としては，FORTRAN，C言語，あるいは最近で
はJava等が用いられる．
.理解，分析 得られた数値解に対して可視化等の処理を
行い，計算結果の解釈を行う．この計算結果を解析対象の現象と照
らし合わせ，解の精度や計算に用いたモデルの妥当性に関する検
討，また計算結果を基に現象の理解，分析等を行う．
工学的な諸問題の中には偏微分方程式を用いて定式化される問題
が多い．例えば，次式の
＝ ＋ ＝0
で表される偏微分方程式は一般にラプラスの方程式（Laplace’s
equation）と呼ばれるが，式中の関数 の物理的意味を変えるこ
とによって，上式は非圧縮性流体の速度ポテンシャル，静電場のポ
テンシャル，熱伝導現象における定常温度分布等，種々の問題を表
現する数学モデルとして扱うことができる．
偏微分方程式の初期値境界値問題の代表的な数値解法として，差
分法，有限要素法，境界要素法が知られている ．これらの数
値解法の詳細については他編に譲るとして，ここでは各手法の特徴
について概略を述べる．
b.差分法 差分法（FDM:finite diﬀerence method）は，解
析領域を規則正しい格子によって分解し，偏微分方程式に現れる導
関数を差分で置換えることで近似方程式を作成する方法である．対
象とする関数は連続的な値をもつが，格子点上の有限個の値のみを
扱うことで離散化することができる．最終的には連立一次方程式で
問題が表現され，これを解くことで解を得ることができる．
差分法は，近似式を作成することは比較的容易であるが，境界と
格子点が一致しないため境界条件を正確に指定することが難しいな
ど，計算精度上の問題は指摘されている．精度を上げるためには全
体の格子のサイズを小さくする必要があり，特定部分だけ，より詳
細に解析を行いたいなどの要求に対応することは困難である．
c .有限要素法 有限要素法（FEM:finite element method）
は，1950年代に航空構造問題に対する解法として誕生した計算手
法であるが，1970年代には流体力学，伝熱工学，電磁気学，量子
力学等，偏微分方程式でモデル化が可能なあらゆる分野で使用され
るようになってきた．
有限要素法は，解析領域を三角形や四角形等の有限要素（finite
element ）と呼ばれる単純な形状に分割し，その挙動を変分法等に
基づいて求める計算方法である．差分法のように直接偏微分方程式
を近似するのではなく，関連する積分表示式を近似することが特徴
となる．最終的に得られる近似方程式は，差分法と同様に格子点
（有限要素法では節点と呼ぶ）の関数値に関する連立一次方程式の
形で表される．
有限要素法は，要素分割において節点を自由に配置することがで
きるため，境界形状を比較的精度よく近似することができる．ま
た，特定部分の要素を小さくし，節点の配置を密にすることで，部
分的に計算精度を向上することができる．しかし，解析領域を三角
形や四角形で要素分割を行う処理に手間がかかり，特に三次元領域
の要素分割を人手で行うことは困難である．そのため，自動要素分
割等に関する研究も盛んである．
d.境界要素法 境界要素法（BEM:boundary element
method）は，偏微分方程式を解きたい領域の境界上の積分方程式
に帰着させ，これを離散化することによって数値的に解く方法であ
る．解析領域の全体にわたって要素分割を行う有限要素法に対し
て，境界要素法では解析領域の次元が一次元小さくなるため，要素
分割に要する手間が大幅に軽減され，計算量も少なくなるという利
点がある．また，境界が無限遠にあるような場合の解析領域を容易
に扱うことができるため，流体力学や音場，電磁場等の問題にも有
効に利用することができる．
一方，境界要素法は不均質領域や形状の複雑な問題，非線形性を
有する問題にはあまり適していない．また，計算によって直接求め
られる解が境界での値であるため，内部の変位や応力等の値を求め
るにはそれぞれの点で計算し直さなければならないなどの問題もあ
る．
e .種々のシミュレーション 数値シミュレーションは，数学
モデルによって定式化が可能な問題に対して有効な方法であるが，
工学分野で用いられるシミュレーションには，それ以外にも種々の
手法がある．例えば，景観シミュレーションは，コンピュータグラ
フィックス（CG）を利用したシミュレーション手法である．これ
は，建築設計，土木設計，都市開発等において，対象となる建造物
が周辺の景観にどのような影響を与えるかを事前に検討するため，
CGを用いて可視化を行う手法である．三次元CGを用いること
で，建造物をいろいろな方向から見て，色彩や質感，配置，日照等
の問題について検討を行うことができる．また実写とCGの合成技
術により，現在の町の景観がどのように変わるかなどの確認を行う
ことも可能である．
また，近年新しいシミュレーション技術として注目されてきたも
のにバーチャルリアリティが挙げられる．これはインタラクティブ
な三次元環境の中で臨場感の高い映像を用いることで，仮想世界を
体験する技術である．例えば，建築物の設計を行う場合，部屋の大
きさや天井の高さなどの人間の感覚に係わる問題については，仮想
体験を通して検討することが可能になる．従来のシミュレーション
は計算機の中だけで行われていたのに対し，これらの手法では人間
の行動や判断がシミュレーションの枠組みの中に含まれるところが
特徴である．
