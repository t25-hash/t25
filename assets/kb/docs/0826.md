# 3・1・2 ラグランジュの方法とオイラーの方法

> **章**: 第3章 流体力学の基礎式
> **節**: 3・1 流体運動の記述

流体運動を記述する方法として、ラグランジュの方法(Lagran-gian method)とオイラーの方法(Eulerian method)がある。ラグランジュの方法では、個々の流体粒子に着目して任意の時刻における流体粒子の位置ベクトルの3成分とその位置の物理量を調べる。これを対流加速度(convectionalacceleration)という。拡大管や縮小管の流れは定常状態でもこの項により減速流れや増速流れになる。これに対して式(3・4)の右辺第1項は局所加速度(local acceleration)と呼ばれ、両者の和が4―13真の加速度(substantial acceleration)である。加速度の導入から明らかなように、オイラーの方法では流れている流体粒子の物理量の時間的な変化は式(3・5)で与えられる。ラグランジュの方法は考え方が質点系の力学と類似しており、一見わかりやすそうであるが数学的取扱いは面倒であり、一般にはオイラーの方法が用いられる。
