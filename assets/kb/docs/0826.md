# 3・1・2 ラグランジュの方法とオイラーの方法

> **章**: 第3章 流体力学の基礎式
> **節**: 3・1 流体運動の記述

流体運動を記述する方法として、ラグランジュの方法(Lagran-gian method)とオイラーの方法(Eulerian method)がある。ラグランジュの方法では、個々の流体粒子に着目して任意の時刻における流体粒子の位置ベクトルの3成分とその位置の物理量を調べる。すなわち、流体粒子を識別するために初期時刻=0における流れ場の位置욣=욣,욣,욣と経過時間を独立変数に選ぶ。座標욣,욣,욣を物質座標(material coordinate)という。空間座標,,は流体粒子の時刻における位置=,を表す従属変数であり、=욣,(3・1)流れの速度ベクトル=욣,および加速度ベクトル=욣,は式(3・2)(3・3)で表示できる。δ욣,δ=옿옿욣(3・2)=limδδ=옿옿욣=옿옿욣(3・3)=lim一方、オイラーの方法では流れ場の任意の位置に着目し、流体の物理量を位置(空間座標)と時間の関数として調べる。時刻において位置を占める流体粒子욣は一つしかなく、逆に流体粒子욣は時刻において一つの位置しか占められないので、と욣は1対1の対応関係があり、独立変数を욣,욣,욣,から,,,に座標変換できる。式(3・2)の関係を用いて式(3・3)を座標変換すれば、オイラーの方法における加速度は=옿옿옿옿욣+옿옿옿옿욣+옿옿옿옿욣+옿옿옿옿욣=옿옿+옿옿+옿옿+옿옿(3・4)となる。式(3・3)と式(3・4)からわかるように、ラグランジュの方法では加速度は単に時間の微分で記述されるが、オイラーの方法では加速度は空間座標の偏微分にも関係する、これは、流体の速度が空間で連続的に変化するためであり、速度が空間で一様でない場合は時間的に変化しない流れ(定常流:옿옿=0)でも式(3・4)の右辺第2項以下の加速度が存在する。これを対流加速度(convectionalacceleration)という。拡大管や縮小管の流れは定常状態でもこの項により減速流れや増速流れになる。これに対して式(3・4)の右辺第1項は局所加速度(local acceleration)と呼ばれ、両者の和が4―13真の加速度(substantial acceleration)である。加速度の導入から明らかなように、オイラーの方法では流れている流体粒子の物理量の時間的な変化は式(3・5)で与えられる。옿옿욣=DD(3・5)ここに、DD≡옿옿+옿옿+옿옿+옿옿(3・6)DDは実質微分(substantial derivative)または物質微分(material derivative)と呼ばれ、オイラーの方法では、内部エネルギー、エンタルピー、エントロピーなどの状態量の変化はDD,DD,DDで記述される。ラグランジュの方法は考え方が質点系の力学と類似しており、一見わかりやすそうであるが数学的取扱いは面倒であり、一般にはオイラーの方法が用いられる。
