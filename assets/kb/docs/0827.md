# 3・2・1 層流と乱流

> **章**: 第3章 流体力学の基礎式
> **節**: 3・2 流体運動に関する用語

流れの中に細い注射針を挿入して静かに染料を注入すると,流速が小さいと染料は1本の糸を引いて下流に流れるが,流速が大きくなると染料は乱れて拡散し下流では流路いっぱいに広がる.前者の場合,流体粒子は層状をなして滑らかに流れており,このような流れを層流(laminar flow)という.後者は流体粒子が時間的および空間的に不規則に変動して互いに混合しながら流れており,乱流(turbulent flow)と呼ばれる.中間の速度では上流は層流状態であるが途中で乱流に変わる.層流から乱流へ変わることを遷移(transition)といい,遷移する位置を遷移点(transition point)という.遷移点はある領域で絶えず上下流に変動しており,この領域を遷移領域(transition region)という.すなわち遷移領域では層流状態と乱流状態が間欠的に現れる.
