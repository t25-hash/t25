# 1・3・3 項で述べたように流れと直角方向に速度こう配ddが

> **章**: 第3章 流体力学の基礎式
> **節**: 3・2 流体運動に関する用語

あるとき、粘性によりせん断応力が作用するが、層流の場合せん断応力は流体分子どうしの衝突による運動量交換によって生じ、ニュートン流体では式(1・7)に従う。一方、乱流の場合、微視的には流体粒子どうしのせん断応力は同じメカニズムで発生するが、巨視的に平均的な流れを取扱うときには、早い流れと遅い流れの間には流体粒子の混合(乱流混合、turbulent mixing)による運動量交換があるので、平均的な(見掛け上の)せん断応力は式(1・7)よりはるかに大きな値になる。乱流混合は渦運動を伴うので乱流の見掛け上の粘性を渦粘性(eddy viscosity)という。これに対して層流の粘性を分子粘性(molecular viscosity)という。このような巨視的なせん断応力発生のメカニズムの違いにより、管内を流れる流体の速度分布は層流と乱流では異なる。
