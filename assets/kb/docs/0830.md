# 3・2・3 流線、流跡、流脈

> **章**: 第3章 流体力学の基礎式
> **節**: 3・2 流体運動に関する用語

ある時刻において、流体中の一つの線上のすべての点における接線がその点の流れの方向を与えるような線を流線(streamline)という。流線どうしが交わることはない。もし交われば、交点で流体粒子は同時に二つの速度ベクトルをもつことになり矛盾するからである。流線の集合により作られる管を流管(streamtube)という。特定の流体粒子が流れ場に描く軌跡を流跡(path)といい、それを連ねた線を流跡線(pathline)という。すなわち、流跡はラグランジュの方法で見たときの流れの様子を表す。例えば、比重が1の微小蛍光粒子を水に流し、写真のシャッターを全開にして撮った写真は流跡線を示す。流れ場の特定の一点を通過するすべての流体粒子の並びを流脈(streak)といい、ある時刻においてそれらを連ねた線を流脈線(streakline)という。流れの中の一点から煙や染料を流して瞬間的に撮った写真は流脈を示している。定常流では流線と流跡線と流脈線は一致するが、非定常流では異なる。流れを可視化する際に、塗料、微小粒子、煙などを流して写真を撮ることがあるが、非定常流の場合には、その写真が流線、流跡線、流脈線のいずれを表しているかを吟味することが重要である。図3・1は、平面上で一辺がδの矩形の流体粒子について、それぞれの変形と運動の様子を説明している。ある時刻において、流体中の一つの線上のすべての点における接線がその点の渦度ベクトルの方向を与えるような線を渦線(vortex line)という。また、式(3・9)の中辺および右辺の第2項に現れる3行3列の行列はテンソル量で、それぞれ速度こう配テンソル(velocitygradient tensor)、変形速度テンソル(rate-of-deformation ten-sor)と呼ばれる。また、右辺第3項をスピンテンソルと呼ぶことがある。(a) 並進運動(b) 方向伸縮変形(c) 方向伸縮変形(e) 回転運動図3・1 流体の運動と変形
