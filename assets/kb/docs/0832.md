# 3・4・2 内部応力

> **章**: 第3章 流体力学の基礎式
> **節**: 3・4 流体粒子に作用する力

流体が流動して慣性力が作用したり上述のような外力が働くとき、流体粒子どうしが接する面および流体と固体壁の間には、それに抗する表面力(surface force)が生じる。単位面積当たりの表面力を内部応力または単に応力(stress)という。ここで、σの添字は、流体粒子の面の外向き法線ベクトルの方向を示し、外向き法線ベクトルの方向が負の場合にはσに-記号を付けて表す。また、τについては、前の添字が作用する面の法線方向、後の添字が作用する方向を示し、外向き法線ベクトルの方向が負の場合には-記号を付けて表す。ところで、静止流体においては垂直応力は面に内向きに作用する圧力であるが、流動している場合には、一般にσ≠σ≠σであり、式(3・13)で表される。式(3・14)の左辺を応力テンソル(stress tensor)、右辺第2項をせん断応力テンソル(shear stress tensor)という。すなわち、流動している流体粒子には、外力のほかに圧力こう配による力と粘性力が作用する。
