# 3・4・3 せん断変形速度とせん断応力の関係

> **章**: 第3章 流体力学の基礎式
> **節**: 3・4 流体粒子に作用する力

固体(弾性体)にせん断力を加えると,せん断変形(shearingdeformation)に比例してせん断応力(shear stress)が生じ,力を取り去ると元の形に戻るが,流体では,せん断応力はせん断変形の量ではなく,せん断変形の速度(rate of shearing deformation)に依存し,このせん断変形速度に抗する性質が粘性である.水や空気などのニュートン流体では,せん断応力に対して式(3・18)と式(3・19)が成立する.τ=2μ옿옿+λ-23μ divτ=2μ옿옿+λ-23μ div(3・18)τ욨욨=2μ옿옿+λ-23μ divここに,λ:体積粘性係数,div=옿옿+옿옿+옿옿.τ욨=τ욨=μγ=μ 옿옿+옿옿τ욨=τ욨=μγ=μ 옿옿+옿옿(3・19)τ=τ=μγ욨=μ 옿옿+옿옿式(3・18)と式(3・19)はニュートン流体の運動方程式を解く際に必要な構成方程式(constitutive equation)である.非ニュートン流体の構成方程式については15章を参照されたい.
