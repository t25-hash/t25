# 3・5・3 エネルギー方程式

> **章**: 第3章 流体力学の基礎式
> **節**: 3・5 保存則

式(3・28)の左辺は単位質量の流体のもつ内部エネルギーと運動エネルギーの変化を表し、右辺の第1~4項は、それぞれ、外力、圧力こう配、体積変化、粘性力に対してなす仕事であり、第5項はエントロピー変化の項である。一方、運動方程式(3・23)は単位質量の流体の力の釣合いを示しており、速度ベクトルとの内積をとれば、次の力学的なエネルギー方程式(energy equation)が得られる。式(3・28)から式(3・29)を差引くと、次の熱力学的エネルギーの式(熱力学第一法則)が得られる。このことは、エネルギー方程式として式(3・28)の代わりに熱力学第一法則[式(3・30)または式(3・31)]を使用すればよいことを示している。
