# 3・6・3 角運動量の保存と物体の受ける力

> **章**: 第3章 流体力学の基礎式
> **節**: 3・3 )。一次元流れを仮定しているから、すべての流線上の速度、圧

例1:オイラー(Euler)のタービン羽根車理論(turbine equa-tion、図3・8)半径流タービンの羽根車に流入する流体の絶対速度を、その方向が羽根車の回転方向となす角をαとし、その点の半径をとする。また、羽根車内を単位時間に通過する流体の質量をとする。流体は回転する羽根車内を、相対的にほぼ羽根表面の方向に流図3・6平面に衝突する噴流図3・7 局壁面に沿う噴流図3・8 半径流タービン羽根車れる。半径の出口における相対速度を羽根車の周速度と合成すれば、出口における絶対速度(角度αの方向)が得られる。したがって、流体がタービンに及ぼすモーメント(あるいはトルク)は式(3・51)より次式で与えられる。
