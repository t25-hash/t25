# 3・6・4 エネルギー積分とベルヌーイの式

> **章**: 第3章 流体力学の基礎式

b. ベルヌーイの式。渦なし運動の場においては、はすべての流線に対して同じ値をもつ。圧力はPa、速度はm/s、基準水平面からの高さはmで表され、重力の加速度は9.80665m/sである。. 実用式実用上、管路内の流れを取扱う場合には、は管路断面の平均流速、およびはその断面の中心における値をとる。流れが発達し、速度分布が飽和状態になっている断面においては、乱流の場合にはα=1.03~1.10、層流の場合にはα=2である。普通の管路では乱流状態の流れにあるから、α≒1としてよい。管路に沿って圧力指示用ガラス管を立てると、その管内を流体がそのにρを加えた高さに相当する点c,d,...まで上昇することが認められる。これらの上昇点をつないでできた線acd(図3・10)を水力こう配線という。水力こう配線の傾斜角をθとするとき=tanθを相当こう配、または水力こう配と呼ぶ。もし、水力こう配線が管路の中心線から下記のの値より低い点があれば、その点で流れがとぎれる。c. 動圧、静圧、全圧水平な定常流の中に物体があるとき、物体の正面中央の一点で流れは完全にせき止められる。これに対してを静圧(static pressure)、を全圧(total pressure)という。d. 回転系における流れ(pressure and velocity in rotating sys-tem)ターボ機械の羽根車のように、一定速度で回転している系内にある流路においては、次の関係が成り立つ。これは式(3・59)に対応するエネルギー式であるが、回転系の遠心力に相当する値2だけ異なる。羽根車においては流体は羽根車より強制力を受けるので、入口と出口で全比エネルギーに次の差ができる。
