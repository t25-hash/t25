# 3・7・1 相似の条件

> **章**: 第3章 流体力学の基礎式
> **節**: 3・7 流れの相似則と無次元パラメータ

流体は,運動方程式からわかるように外力,圧力こう配による力,粘性力などと慣性力とが釣合うように流れるから,境界が幾何学的に相似な二つの流れに対してこれらの力の比が同一であれば,力学的相似性が成り立つことを意味している.このことを具体的に調べるには運動方程式を無次元化して考えると便利である.簡単のために二次元流れを取上げ,物体の大きさを表す代表長さ,流れの速さを表す代表速度,代表時間,代表圧力を用いて,=′,=′,=′,=′,=′,=′とおき,軸方向の運動方程式を無次元化すれば,옿′옿′+′옿′옿′+′옿′옿′=-ρ′옿′옿′+ν옿′옿′+옿′옿′+1옿옿′옿옿′+옿′옿′(3・73)となる.式(3・73)から,無次元パラメータ,,ρ,νの値が二つの流れに対して同じであるときに方程式は同一となり,対応する時刻における対応する位置での′,′および′がそれぞれ等しくなり,二つの流れに対する力学的相似性が成り立つことがわかる.
