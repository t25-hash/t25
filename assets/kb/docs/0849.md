# 4・2・2 渦なし流れ

> **章**: 第4章 理想流体の流れ
> **節**: 4・2 ベルヌーイの式と渦なし流れ

流れ場のすべての領域において3・3節で定義した渦度が零である流れを渦なし流れ(または非回転流れ、irrotational flow)という。すなわち、渦なし流れでは流れ場のすべての領域で左辺( )内の値は一定である。すなわち、ベルヌーイの式では1本1本の流線に対して式(4・6)が成立するのに対し、渦なし流れ(ポテンシャル流れ)の場合には、流れ場のすべての領域で流体力学的なエネルギーが一定で式(4・6)が成立する。式(4・6)は一般化したベルヌーイの式と呼ばれる。例えば、大気中で運動する流線形物体のまわりの相対流れは、物体のごく近傍を除いてエネルギーが一定とみなせるので、渦なし流れの理論が適用できる。渦なし流れでは、4・3節で述べるように、ポアソン方程式∇φ=0を解いて式(4・5)により速度場を求めた後、式(4・6)を用いて圧力場を求める。したがって、式(4・6)は圧力方程式とも呼ばれる。
