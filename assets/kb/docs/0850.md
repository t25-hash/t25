# 4・3・1 流れ関数(stream function)と速度ポテンシャル

> **章**: 第4章 理想流体の流れ
> **節**: 4・3 二次元渦なし流れ

流体粒子の運動が一平面に限られ、その平面と平行なあらゆる平面で同一になる流れを二次元流れという。図4・1でψ=const.の流線上の点1とψ=const.の流線上の点2を結ぶ任意の曲線を横切って、その左側のA の領域から右側のB の領域へ通過する流量を求める。一方、二次元流れの場合について、式(4・5)を式(4・1)に代入すれば、式(4・12)が成立し、φも調和関数でなければならない。また式(4・5)と式(4・7)との比較から、次のφとψに関するコーシー・リーマンの関係式が成り立つ。
