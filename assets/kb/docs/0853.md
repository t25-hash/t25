# 4・4・2 任意形状物体に働く力

> **章**: 第4章 理想流体の流れ
> **節**: 4・4 物体に作用する力

通常、物体近傍に原点をとるので、∞では右辺第2項以下は近似的に零となる。このことは、式(4・31)は遠方において速度が一様にであることを満足している。これをクッタ・ジューコフスキーの定理(Kutta-Joukowski’s theorem)という。一般には物体からのわき出しはないので=0となり、物体には全く抗力が作用しないという現実の現象には合わない結果を得る。これが理想流体の最も大きい欠点である。これをダランベールのパラドックスという。また、物体に循環がある場合、流れに直角方向の力、揚力を受ける。回転するボールが曲がるのはその一例である。図4・13の場合には循環は時計回りであるからΓ<0であり、>0の向きにρΓの揚力を受ける。
