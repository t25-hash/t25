# 4・5・1 等角写像(conformal transformation)

> **章**: 第4章 理想流体の流れ
> **節**: 4・5 等角写像とその応用

二つの複素変数=+とζ=ξ+ηを一つの解析関数で関係付ける.=ζ(4・34)式(4・34)によって図4・15に示すようにζ平面上の一点は平面上の一点に対応付けられるから,ζ平面上の曲線は平面上の曲線に写される.これを写像といい,ζを写像関数という.式(4・34)をζで微分すれば,d=′ζdζ(4・35)ここで,d,dσ,′ζをそれぞれ,以下のように絶対値と偏差で表す.d=ddζ=dσ′ζ=(4・36)これらを式(4・35)に代入して両辺を比較すれば,d=dσ,θ=φ+αが成り立つ.つまり面上で点ζの近傍ではdはdσの倍になり,角度はαだけ回転することになる.このことは,ζ近傍の微小な形状は面上に相似に写像されることを意味する.
