# 5・3・6 逆問題

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

12･7節で示すように，逆問題（inverse problem）とは結果から
原因を推定するもの，あるいは別の表現で，出力をもとに入力を求
めるものである．逆問題解析においても計算の過程において有限要
素法（FEM:finite element method），境界要素法（BEM:
boundary element method），差分法（FDM:finite diﬀerence
method）などの数値解析法が必要となる．数値解析法は取扱う逆
問題によって異なるものの，入力を仮定して数値解析によって得ら
れる解（出力）が，実験や計測などによって得られる測定データか
らの一次情報にできるだけ近づくように入力を繰返し修正したり，
一次情報を用いて直接未知入力を計算する方法として利用される．
ところで，計測などに基づく一次情報は境界値として得られる場
合が多い．また，同定すべき対象が隠れて見えない部分や内部の欠
陥のような境界形状であったり，測定できない部分の境界値であっ
たりする場合もある．そこで，逆問題解析で用いる数値解析手法と
して境界要素法 がよく適用される．12･7節でもいくつかの例
が示されるが，ここでは境界要素法を逆問題に利用した例を以下に
示す．
a.境界値逆問題 金属に外部から電流を加え，防食す
るカソード防食（cathodic protection）での最適な電極配置と，
電流の大きさを決定する問題への境界要素法の応用例 があ
る．この問題では，ポテンシャル問題の境界要素法を利用できる．
アノードとカソードの部分のポテンシャルと流束がともに未知（不
完全規定境界），他の部分は通常の境界条件が与えられている境界，
ほかに内点数点のポテンシャル値を利用して，境界積分方程式と内
点のポテンシャル値を計算するグリーン（Green）の公式を，それ
ぞれ離散化したものを連立し，不完全境界の未知量を直接求めてい
る．安定な解を得るために，特異値分解（singular value decom-
position) を利用して，連立方程式の係数行列の階数低下を行っ
ている．
音響問題では，音波の干渉を利用して，騒音源から放射される音
を消すために必要な付加音源の振動状態を同定する問題への応用
例 がある．定常音響問題として定式化し，ヘルムホルツ（Hel-
mholtz）方程式に対する境界要素法を利用している．付加音源の
振動状態の仮定値を用い，境界要素法で計算された測定点の音圧の
平方和が最小になるような振動状態のパラメータを，最適化手法を
用いた反復計算により同定している．
静弾性問題での例 では，領域内部で得られる主応力和から境
界の表面力分布を推定する問題に境界要素法が応用されている．主
応力和の計算を境界要素法で定式化し，特異値分解による階数低下
法で適切化を図っている．
b.境界形状同定問題 高炉の寿命を延ばすために必要
な，炉床部の耐火物の侵食状況を同定するために，境界要素法を応
用した例 がある．耐火物外表面の温度の実測値と，境界要素
法を用いた定常熱伝導解析による計算値との残差平方和を最小化す
る形状を，侵食ラインとして同定している．
また，構造物中のき裂を同定する問題へ，ポテンシャル問題の境
界要素法を適用した例 がある．これは電気ポテンシャルCT
（computed tomography）法（electric potential CT method）と
呼ばれる方法であり，ポテンシャル問題における境界積分方程式を
未知数に対して逆に解いてき裂面での境界値を求め，得られた境界
値の中で，流束が零となる部分をき裂の存在位置とする方法（未知
境界同定法) と，マルチ電流負荷方式により得られた境界上のポ
テンシャル分布の計算値と測定値の残差を，最小化させる方法（境
界残差最小化法) とがある．この問題で，き裂を一意に同定する
効であるとしている．
題とし，ヘルムホルツ方程式に関する境界要素法を用いて，観測さ
ために必要な電流負荷条件も示されている ．

定常動弾性問題に対する適用例 として，動弾性問題のひず
み応答を利用して，構造物に内在する欠陥の位置と大きさを同定す

要素法が用いられている．境界上のいくつかの点で測定したひずみ
と，境界要素法の順解析で得られるひずみの残差平方和を最小にす
るように，欠陥の位置と大きさのパラメータを共役こう配法 に
より計算している．複数の箇所を加振した場合の応答を観測値に利
用することや，応答の感度が大きい箇所の加振点としての選定が有
同様の例として，静弾性問題において，弾性体内部の欠陥形状と
位置を弾性体境界に作用する集中力による変位応答から同定する問
題への適用例 がある．解析には境界要素法を用い，欠陥を表
現するパラメータ値の探索には拡張カルマンフィルタ（extended
Kalman filter ) が用いられている．
c .負荷逆問題 拡散方程式の時間領域境界要素法を用
いた応用例 として，時間に依存して変化するソース項同定問題
（source identification problem）への適用例がある．ソース項は既
知関数の線形結合で近似し，観測値と計算値の残差平方和を最小化
することにより，その係数を求めている．
また，スカラ波動方程式の時間領域解法に関して，集中負荷の位
置と強さおよび作用時間を同定した例 がある．
音響問題における負荷逆問題の例としては，騒音の低減に関連し
て，騒音源の位置と大きさを探査する問題がある ．定常音響問
れた音響インテンシティと計算値の残差平方和を最小化するように
同定している．
静弾性問題の負荷逆問題としては，トンネル掘削時の測定変位か
ら地下の初期応力を同定するための適用例 がある．内部の初期
応力はソース項とし，地上の変形と関係付ける式に境界要素法を利
用している．測定値が求めるべき初期応力の成分数より多いとき
は，最小二乗基準で初期応力を決定している．
また，残留応力（residual stress ）の同定問題に境界要素法を適
