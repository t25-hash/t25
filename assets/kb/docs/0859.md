# 4・6・3 物体を含む流れの例

> **章**: 第4章 理想流体の流れ
> **節**: 4・6 特異点法とその応用

これよりわき出し分布は翼厚分布からただちに求まるが、渦分布は反りからクッタの条件を満足する積分方程式(4・59)の解として求められる。したがって、薄翼に働く圧力および揚力はそれぞれ式(4・60)から決定される。この解法としては積分領域を微小区間に分割し、各区間上であらかじめ渦分布を一定、一次変化、および三次変化などに仮定することにより、連立一次代数方程式に帰着させる。物体形状が複雑な場合には、パネル法(panel method,4・9・2項参照)により、物体表面を微小区間に分割し、この上に渦とわき出しを分布させ、区間中点で境界条件が満足されるようにそれらの強さを決定すれば、さらに物体表面圧力分布や流体力を求めることができる。最も基本的なものは一つの直線的な区間上では渦とわき出しの強さを一定とする。
