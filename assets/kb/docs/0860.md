# 4・7・1 翼の場合

> **章**: 第4章 理想流体の流れ
> **節**: 4・7 不連続流れ(discontinuous flow)

速度の中に平板AB が置かれており、図4・23に示すようにその前縁と後縁の両方から流れがはく離し、自由流線ACとBD とを境界として後流が形成されているとする。このような流れは4・ンシャル流れで、後流内部では流体は静止し(この状態を死水という)、圧力は一定であるとする。この場合、自由流線ACとBDに沿って圧力は一定であり、ベルヌーイの定理から流速も一定である。平板には主流に垂直に下記の力が働く。これをキルヒホッフの死水理論という。式(4・61)は後流が空洞である場合には比較的実際の値に近いが、後流が同じ流体の場合には実際より小さい値を与える。
