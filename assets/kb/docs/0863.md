# 4・8・1 速度ポテンシャルとベクトル速度ポテンシャル

> **章**: 第4章 理想流体の流れ
> **節**: 4・8 三次元流れ

外力が保存力で=∇Ωで写えられるとき、オイラーの運動方程式(4・2)は式(4・67)で与えられる。ただし、φは式(4・5)を満たす速度ポテンシャルである。したがって、ポテンシャル流れにおける流れ解析では、式(4・67)において、まず右辺の∇×=0から速度場を求め、その後、左辺を流線に沿って積分した、次の圧力方程式から圧力場を求める手法が用いられる。そこで、三次元流れ場においては、渦度ωによって誘起される速度場を想定して、式(4・71)で定義されるベクトル速度ポテンシャル(vector velocity potential)ψが用いられる。∇ψ=-ω(4・74)これと、オイラー方程式から圧力項を消去して導かれる渦度輸送方程式[式(4・75)]とを組合わせた解析から流れ場が求まる。
