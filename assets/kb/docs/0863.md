# 5・5・3 ナノカロリメトリー（nanocalorimetry）

> **章**: 第5章 熱測定法
> **節**: 5・5 ミクロ物性計測法

シリコンテクノロジーを用いて微小温度センサやヒータを有する
MEMSを作成することにより，従来よりもはるかに微量な試量を
用いて熱分析を行うナノカロリメトリー法が開発された．これによ
り，極微量試料の熱容量や相転移熱が計測できるだけでなく，白金
触媒表面における水素と酸素の反応における発熱を，リアルタイム
に検出することも可能である．また，最近では細胞活動に伴う温度
変化などの検出も試みられている．
