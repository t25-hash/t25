# 4・9・1 渦の性質

> **章**: 第4章 理想流体の流れ
> **節**: 4・9 渦運動

渦は以下の性質をもつ.(1) 非粘性の流体が運動する場合,渦は発生することもなく,消滅することもない.これをラグランジュの渦定理(Lagrange’stheorem on vorticity)という.(2) 側面が渦線(3・3節参照)で作られる管を渦管(vortextube)という.一つの渦管の周りの循環Γは断面によらず一定である.渦管の循環Γを一定に保って断面を無限小にした仮想の線を渦糸(vortex filament)という.渦管(または渦糸)は渦管(または渦糸)として行動し,その強さは一定不変である.これをヘルムホルツの渦定理(Helmholtz’s theorem on vortex)という.(3) 渦線は流体の内部で切れることはなく,流れの境界から境界まで連続しているか,自分自身で閉じて渦輪(vortex ring)を作るかのいずれかである.
