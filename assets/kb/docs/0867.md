# 4・9・3 渦の運動

> **章**: 第4章 理想流体の流れ
> **節**: 4・9 渦運動

無限流体中に存在する一つの直線渦糸は,流体の流れが一様であれば,その形を保ったまま流れに乗って移動し,静止流体の場合には渦も静止したままである.しかし,渦の近くに例えば平面の壁がある場合には,図4・8に示した鏡像を利用した流れ場に相当し,渦A は鏡像渦B が誘起する速度,Γ4π[式(4・90)]で壁に平行に下方へ移動する.同時に渦B は渦A が誘起する速度に乗って,同様に壁に平行して下方へ移動する.このように強さが等しく循環の符号が逆の二つの渦を渦対(twin vortex)といい,これらは同じ距離を保ちつつ互いに平行に直線状に移動する.多数の渦が存在する場合,その中の一つの渦の運動は,その渦以外の渦が誘起する速度ベクトルの和に従う.
