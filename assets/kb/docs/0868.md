# 4・9・4 渦層(vortex sheet)

> **章**: 第4章 理想流体の流れ
> **節**: 4・9 渦運動

図4・30に示すように、同じ密度の流体が速度とで隣り合って流れると、その不連続面では速度こう配が大きい自由せん断層(渦層)が形成される。その不連続面には単位長さ当たり-の強さの循環の渦が並んでいるとみなせる。≠0のときはωの虚数部が正となるものが必ず存在し、すべてのに対して擾乱は時間の経過とともにその指数乗で増大する。すなわち微小擾乱は不安定で、この渦層は図4・31のように巻上がっていく。これを渦層のヘルムホルツ不安定(Helmholtzinstability)という。
