# 4・9・4 渦層(vortex sheet)

> **章**: 第4章 理想流体の流れ
> **節**: 4・9 渦運動

図4・30に示すように、同じ密度の流体が速度とで隣り合って流れると、その不連続面では速度こう配が大きい自由せん断層(渦層)が形成される。その不連続面には単位長さ当たり-の強さの循環の渦が並んでいるとみなせる。この不連続面に次式、η,=εReexp-ω(4・91)で表される波数の微小擾乱ηを与えると、複素数ωを決定する条件として、ω--2ω-+1=0(4・92)が得られ、これを解けばωは次式で与えられる。すなわち、ω-=1±2(4・93)を得る。≠0のときはωの虚数部が正となるものが必ず存在し、すべてのに対して擾乱は時間の経過とともにその指数乗で増大する。すなわち微小擾乱は不安定で、この渦層は図4・31のように巻上がっていく。これを渦層のヘルムホルツ不安定(Helmholtzinstability)という。
