# 5・2・4 境界層制御（boundary layer control )

> **章**: 第5章 粘性流体の流れ
> **節**: 5・2 境界層

ここでは，はく離に関するものを除いた乱流境界層の制御につい
て紹介する．境界層制御の目的の代表としては，流体抵抗の低減や
騒音の抑制および伝熱の促進などが考えられる．抵抗低減は流体工
学の課題として重要であり，壁面せん断応力が乱流境界層における
種々のスケールを与えることを考慮すると，壁面による流体抵抗の
発生機構は重要な事柄である．乱流における摩擦抵抗の低減と遷移
の遅延が実用上注目される技術となる．乱流遷移への遅れや乱流摩
擦抵抗の低減は直接運転コストの節約をもたらす．流れを制御する
方法としては，境界条件や物性変化を通じて操作する受動的手法
f .秩序構造（coherent structure) エネルギー散逸の源
（passive method）とエネルギーを直接に注入する能動的手法
（active method）とがある．経済性を考慮した応用を検討すると，
前者が圧倒的に有利であり，後者はシミュレーションのみの研究や
乱流の動的構造の調査に利用されている．能動的手法は制御のため
に導入した動力を考慮した実質的有効性を検討する必要がある．ま
た，制御の対象としては，流れ場全体を包括的（global ），局所的
領域や構造のみの選択的（selective）とに分けられる．近年
MEMS技術の発達と整構造に関する研究成果とニューラルネット
ワークなどの制御法の発展により，特定の現象のみをねらった能動
的手法が可能となっている．

調査されており，それぞれについて簡単に紹介する．
