# 5・1・2 非平行流

> **章**: 第5章 粘性流体の流れ
> **節**: 5・1 節では非圧縮流れのみを扱う。すなわち、非圧縮流れの連続

平行流でなくとも、次に示す円形流は運動方程式が自動的に線形方程式になるために、容易に厳密解を求めることができる。平行流でも円形流でもない場合には、一般に非線形方程式を解かねばならない。a. 減衰渦(decaying vortex)流線がすべて同心円であるような軸対称な二次元流は円形流と呼ばれる。流線が円なので流速ベクトルは周方向成分しかもたない。連続の式は自動的に満たされる。初期時刻に渦度が対称軸上に集中し、そのほかのすべての場所で静止一様状態であったとして、その後の流れ場の時間発展を求めよう。ただし、Γは初期時刻の軸まわりの循環を表す。これを減衰渦という。図5・4の実線は各時刻の流速分布を示している。b. 軸対称噴流(axisymmetric jet)無限に広がる流体中の一点から噴出する定常な軸対称噴流(ジェット)を表す厳密解も得られている。ジェットの噴出点を原点とし、原点からの距離を、ジェットの噴出方向から測った角度θを導入し、軸対称流れを仮定する。図5・5にジェットの流線を示す。定数A が1に近いほど噴出点前方の流速が速くなる。c. その他の非平行流同軸円筒間のクエット流れ、よどみ点流れ、くさび型領域内の流れなどが厳密に解かれている。
