# 5・1・1 項の平行流や5・1・2・a項の円形流で厳密解を求めることが

> **章**: 第5章 粘性流体の流れ
> **節**: 5・1 節では非圧縮流れのみを扱う。すなわち、非圧縮流れの連続

可能であったのは、解くべき方程式が自動的に線形方程式となったためであった。しかし、一般の場合には、非線形方程式を取扱わなければならず、それゆえ厳密解を得ることは極めて困難である。以下では、遅い流れに有効な近似解法について述べる。a. ストークス近似(Stokes’approximation)流体の速度が十分に小さければ、速度の積からなるナビエ・ストークス方程式(5・7)の非線形項を、粘性項に比べて小さいとみなして無視することは自然な近似であろう。ただし、簡単のために外力は働かないとした。ストークス近似は、遅い(レイノルズ数が1に比べて十分に小さい)粘性流の解析に役立つ。特に有界領域の中の遅い流れの記述に適している。無限遠方で速度をもつ一様流の中の半径の球を過ぎる定常流をストークス方程式から求める。球の中心を原点にとり、原点からの距離を、原点を通り一様流と平行な直線から測った角度をθとする。軸対称流れを仮定し、式(5・33)を満足するストークスの流れ関数を導入すると、式(5・34)から次式が導かれる。b. オゼーン近似(Oseen’s approximation)ストークス近似には、円柱を過ぎる一様流の解がない(ストークスのパラドックス、Stokes’paradox)ことや、球を過ぎる一様流において無限遠の境界条件を満たす第二近似解がない(ホワイトヘッドのパラドックス、Whitehead’s paradox)という欠点がある。ここで、は無限遠の一様流の流速である。これをオゼーン近似といい、式(5・7)に式(5・41)の近似を当てはめて得られる式をオゼーン方程式という。オゼーン近似もストークス近似同様にナビエ・ストークス方程式を線形化する近似であるが、その解の構造は、ストークス近似の場合より複雑である。また、ストークス近似が(注意さえすれば)遅い粘性流に広く適用可能であるのに対して、オゼーン近似は遅い一様流の中に置かれた物体の周りの流れを対象とする。c. 球を過ぎる流れの近似解
