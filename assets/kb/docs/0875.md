# 5・2・1 境界層の概念と性質(図5・6)

> **章**: 第5章 粘性流体の流れ
> **節**: 5・2 境界層

レイノルズ数の極めて高い流れにおいても,航空機翼表面等の固体壁近傍には粘性の無視できない薄い層が存在する.層の厚さが物体の代表寸法に比べて十分に薄い場合,この層を境界層(bound-ary layer)と呼ぶ.境界層の概念はプラントル(Prandtl)が1904年に発表した境界層理論(boundary layer theory)によって明確な定義がなされ,実用的高レイノルズ数流れの解析に対する突破口を開いた.境界層外側は渦なし非粘性流体として取扱われ,境界層内側は渦あり粘性流体として取扱われる.境界層の内と外は,境界層厚さ(boundary layer thickness,便宜上,主流速度の99.5%または99%の速度をとる壁からの距離として定義される)により分割される.境界層の厚さのより明確な定義には式(5・53)(5・54)の積分厚さが使用される.排除厚さ:δ=1-/욄d(5・53)運動量厚さ:θ=/욄1-/욄d(5・54)レイノルズ数をパラメータとして,連続した流れ場を異なる領域が接続したものとして取扱う手法は,数学的には漸近接合展開(matched asymptotic expansion)あるいは特異摂動問題(singu-(momentum integral equation)が得られる.図5・6 平板境界層の概念と主要記号4―33lar perturbation problem)と呼ばれる.一方で,境界層の概念に基づいて得られる運動方程式の変形は,薄せん断層近似(thinshear layer approximation)という概念を提供し,物体表面上の薄い流れに限らず噴流や後流等の流れ解析に対しても有用である.また,この種の解析手法は平板後縁の流れなどに見られるような三層理論(triple deck theory)などのアイデアの基礎をなすものである.境界層はレイノルズ数が大きくなると層流から乱流へと遷移する.乱流境界層においては,さまざまなスケールをもつ渦塊からなる乱流運動の構造が運動量,熱および物質の輸送に対して支配的となる.だ円型であるナビエ・ストークス方程式は,境界層近似により放物型の偏微分方程式となり,その解法に対しては流れ場の相似性が極めて重要な意味を成す.層流境界層については相似変換(simi-lar transformation)が,乱流境界層については自己保存(self-preserving)が考察の基礎となり,運動方程式の常微分方程式への変換を可能とする.乱流境界層は多層構造をもち,それぞれの層における適切な尺度法則(scaling law)とレイノルズ数相似(Reynolds number similarity)の確立が理想となる.層全体において厳密な相似性が成立する流れ場は特定の境界条件を必要とするが,相似解は流れの本質を表現し,標準的な解として種々の付加的影響下の流れ場においてもその表現と展開に有用である.主流速度が流れ方向に変化する場合,境界層は慣性力と粘性力との釣合いに対して圧力こう配が加わった流れ場となる.流れ方向に圧力が上昇する場合を逆圧力こう配(adverse pressure gradient),下降する場合を順圧力こう配(favorable pressure gradient)と呼ぶ.順圧力こう配下においては,圧力こう配が流れ方向速度を増加させる働きを成すため,流れ場は安定する.一方,逆圧力こう配下においては,粘性力と圧力こう配とがともに慣性力に抗して作用するため流速の減少を招き,はく離(separation)に至ることがある.流れのはく離は翼面上やディフューザにおいて見られ,流体機器の性能を著しく低下させるため,はく離の機構解明と制御は流体工学の最も重要な課題の一つである.乱流境界層は層流境界層に比べて運動量の輸送が活発で速度分布形状がより凸であるため,はく離しにくい性質を有する.境界層の理解は流体機器の設計や制御に対して定量的指針を与えることのほかに,乱流では伝熱および化学物質の拡散といったスカラ量の輸送係数のモデリングにも利用される.したがって,ここで述べる流体の運動に関する半経験式や解は運動量の輸送に限らず,他のスカラ量に対する拡散係数の見積りに対しても実質的な基準となるものである.
