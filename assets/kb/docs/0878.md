# 5・2・4 境界層制御(boundary layer control)

> **章**: 第5章 粘性流体の流れ
> **節**: 5・2 境界層

ここでは,はく離に関するものを除いた乱流境界層の制御について紹介する.境界層制御の目的の代表としては,流体抵抗の低減や騒音の抑制および伝熱の促進などが考えられる.抵抗低減は流体工学の課題として重要であり,壁面せん断応力が乱流境界層における種々のスケールを与えることを考慮すると,壁面による流体抵抗の発生機構は重要な事柄である.乱流における摩擦抵抗の低減と遷移の遅延が実用上注目される技術となる.乱流遷移への遅れや乱流摩擦抵抗の低減は直接運転コストの節約をもたらす.流れを制御する方法としては,境界条件や物性変化を通じて操作する受動的手法(passive method)とエネルギーを直接に注入する能動的手法(active method)とがある.経済性を考慮した応用を検討すると,前者が圧倒的に有利であり,後者はシミュレーションのみの研究や乱流の動的構造の調査に利用されている.能動的手法は制御のために導入した動力を考慮した実質的有効性を検討する必要がある.また,制御の対象としては,流れ場全体を包括的(global),局所的領域や構造のみの選択的(selective)とに分けられる.近年MEMS 技術の発達と整構造に関する研究成果とニューラルネットワークなどの制御法の発展により,特定の現象のみをねらった能動的手法が可能となっている.壁面摩擦抵抗の低減技術としては以下の方法の有効性が実験的に調査されており,それぞれについて簡単に紹介する.(1) 流体中への高分子溶液の注入高分子溶液(トムズ効果)や界面活性剤により乱流運動を抑制す(a) リブレット(b) 外層デバイス図5・12 抵抗低減パッシブデバイスる方法で,水など液体の流れに限られる.低減率は80%近くに及ぶ場合もあり,熱交換器の循環系に採用されている.(2) 柔軟壁および波状壁イルカなど柔軟壁をもつ水中生物の遊泳速度が動力に比べて極めて速いことに由来しており,グレイ(Gray)のパラドックスとして知られている.柔軟壁材料の開発が鍵となる.(3) 撥水性塗料等のコーティング撥水性塗料を塗布することにより摩擦抵抗低減を図る.層流に対しては有効であることが確認されている.(4) マイクロバブル多孔質性の壁面から数mの微小気泡を放出し,摩擦抵抗の低減を図る方法である.微小気泡の放出に必要な動力を見積もり,実質的な有効性を考慮する必要がある.(5) リブレット[riblets,図5・12(a)]流れ方向の微小溝からなり,最適寸法は溝深さ=5~20,溝間隔=3~25程度であり,抵抗の最大低減率は6~8%である.溝の流れ方向に対する角度の有効な範囲は±15°である.逆圧力こう配に対しても有効であることが確認されている.三次元リブレット形状も考案されている.(6) 外層デバイス[LEBU’s: large eddy break-up devices,図5・12(b)]乱流境界層の外層における大規模渦構造を破壊し,それにより運動エネルギーの輸送と消費を抑制する方法である.できるだけ自身の抗力を低く抑えた薄い翼を,境界層の外層/δ=0.75~0.8に設置する条件が効果的である.流れ方向間隔を7.7δにとり,タンデムに配置するとより大きな効果が得られる.(7) 壁面の横方向振動周囲から切り離した平板を左右に適当な周波数で振動させると流れ方向の壁面摩擦抵抗が低減する方法である.振動に要する動力を実質的有効性の評価に考慮する必要がある.速度のより小さい周囲の流体中に噴出する流れを噴流(jet)と呼ぶ.噴流は,工業分野の機器内で多方面にわたって現れ,また利用されており,流体工学における基本流れの一つである.
