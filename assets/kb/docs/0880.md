# 5・3・2 壁面噴流と衝突噴流

> **章**: 第5章 粘性流体の流れ
> **節**: 5・1 のものとは異なる。自由噴流では、このほかに半径方向に噴出

固体壁に沿って流出する噴流を壁面噴流(wall jet)と呼び、壁面が平面のときの二次元および放射状噴流が代表的なものである。壁面噴流の概略を図5・18に示す。その他の壁面噴流として、噴出口形状が長方形、円形の場合、壁面が曲面の場合などがある。図5・19(a)に示すように、円柱面上の二次元壁面噴流はかなりの範囲にわたって壁面に沿って流れる。これをコアンダ効果(Coanda effect)と呼ぶ。このような円柱壁面噴流の流動特性(速度分布、圧力分布、噴流の広がり、はく離位置など)はノズル出口速度分布形状および噴出角に影響される。図5・19(b)のように、二次元自由噴流または放射状噴流が壁面近くにあるときは、噴流軸が壁面方向に曲がり、ついには壁面に付着する。これもコアンダ効果の一形態である。壁面噴流は、固体表面の膜冷却、加熱、熱的遮断などに広く利用されている。ガスタービン翼、ディスクブレーキの冷却などもその例である。壁面噴流が、曲面固体壁に沿って流れる性質(コアンダ効果)は、航空機翼の循環制御、VTOL航空機のジェットの偏向、ガラス工業における鋳型の冷却などに用いられる。円柱を冷却するには、一様流よりもコアンダ効果を伴う噴流のほうが有利である。側面付着噴流は、純流体素子におけるオンオフ信号の発生、パワーフルディスクにおける流れの切換えに利用される。自由噴流が直角に近い角度で固体表面に衝突するとき、これを衝突噴流(impinging jet)と呼ぶ(図5・20)。衝突噴流は、図5・21 水噴流図5・22 同軸噴流(i) 自由噴流領域(ii) 衝突噴流領域(iii) 壁面噴流領域の3領域があり、(i)は自由噴流、(iii)は壁面噴流と同一の特性をもつ。(ii)は(i)と(iii)の移行領域で、流れの方向が急激に変化するので、噴流軸および壁面に沿って大きな圧力こう配が生じる。衝突噴流中には顕著な渦構造が発達する。衝突噴流は、ガラス、製紙、繊維、製鉄工業などにおける加熱、冷却、乾燥などに応用される。その利点は、動力源の容量が比較的少なくて済むこと、流量制御による熱および物質伝達率の調節が得られることなどである。
