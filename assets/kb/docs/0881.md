# 5・3・3 液体噴流

> **章**: 第5章 粘性流体の流れ
> **節**: 5・1 のものとは異なる。自由噴流では、このほかに半径方向に噴出

液体噴流(liquid jet)の代表例は、空気中に噴出する水噴流(water jet)である。水噴流は、空気との間に自由表面が形成されるため、表面張力の作用を含む複雑な気液混相流となる。水噴流の概略を図5・21に示す。ノズルから噴出した水噴流の界面に発生した表面波が増幅され渦が発生し、噴流は周囲の空気と混合し、やがて大きい液塊や小さい液滴に分裂し、さらに霧状になって拡散する。水液噴流は、初期には水力採炭(hydraulic excavation)に応用された。3~10MPaの高圧水を切羽まで導き、直径15~30mmのノズルから、1.5~6m離れた炭壁に水噴流を当て採炭を行った。その後の技術革新により、水噴流を用いて金属材料の切削を行う方法も開発された。液体を0.2~1GPaに加圧して、直径0.1mm~数mmのノズルから、数百m/s~千数百m/sの速度で噴出させて加工物に当て、そのときの衝撃圧力の作用によって材料除去を行う加工法で、これを噴流切削(jet cutting)と呼ぶ。なお、噴流を用いて金属表面のさびやミル汚れを除去することも行われている。水中における水噴流では、ノズル圧力がある大きさになると、キャビテーションが発生しキャビテーションジェットとなる。キャビテーション気泡は、噴流境界の速度こう配の大きい領域で生じる強い渦運動の中心部の低圧領域に発生し、流れとともに下流に移動してやがて消滅する。このような水噴流を固体表面に衝突させると大きな衝突力が発生するので、洗浄や各種加工に利用されている。
