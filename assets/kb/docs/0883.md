# 5・3・5 横風噴流(cross-flow jet)と対向噴流(opposed jet)

> **章**: 第5章 粘性流体の流れ
> **節**: 5・23 )。ノズル出口近傍には、環状に噴出した噴流によって囲まれ

速度の主流中に、ある角度θをもって噴出した噴流では、主流の影響により中心軸がわん曲する。円形ノズルの場合、噴流断面はしだいに図5・24のような腎臓形に変形していく。このような噴流の特性は、主として角度θ、速度比/によって規定される。主流と逆方向に噴出した噴流(対向噴流)は、ある距離욣だけ4―40流体工学周囲流体中に貫通したあと、流れの向きを逆転する。平均流れ場は図5・25のようになる。욣はノズル出口から噴流軸上のよどみ点までの長さとして定義され、軸対称噴流のときは、욣/=2.7/である。対向噴流では軸方向に圧力がかなり変化する。対向噴流の流れ場は時間的に変動し、拡散・混合特性は噴流外縁の混合層の渦挙動に関連している。主流中に放出される噴流の例は、ガスタービン翼の膜冷却用噴出流体、燃焼器の中へ噴出する二次空気流、気流中へ噴出する噴霧燃料流、煙突から流出する煙の流れ、海や川に放出される温排水の流れなどに見られる。設計上、噴流の軌跡、主流中への浸透距離および主流との混合を知ることが必要である。対向噴流は、ジェットエンジンのアフタバーナ用燃料の供給に用いられることがある。
