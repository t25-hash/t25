# 5・3・7 噴流の制御

> **章**: 第5章 粘性流体の流れ
> **節**: 5・23 ).ノズル出口近傍には,環状に噴出した噴流によって囲まれ

噴流の諸特性(混合・拡散,騒音など)は,噴流中の渦の挙動に支配されるので,渦の操作により噴流を制御することが有効である.音波,振動板,アクチュエータなどにより,噴出口から流出する混合層にかく乱を与え,流れの不安定性を増幅して噴流中の渦の発達を促進し渦の巻込み効果を強めて,噴流の混合・拡散を増大することができる.混合層の自然周波数が욈のときに욈/(は整数)の周波数で励起すると,N 個の渦の合体が促進され大規模渦が生成される.複数の周波数の合成波による励起も大規模渦の生成に有効である.また,욈と욈/2の周波数をもつ二つの信号の合成波により励起すると,渦の合体は信号間の位相差に依存し,ある特定の位相差のときに合体が抑制され,混合層の拡散が著しく減少する.円形噴流の混合作用を増大するためには,噴流を=0.85で励(a) 渦構造(b) 噴流断面積(c) 連行量図5・26 長方形噴流の制御起し,大規模な渦合体を生じさせることが有効である.噴流中の渦挙動を利用した噴流制御技術として非円形噴流がある.これは非円形渦の変形と干渉を利用して混合・拡散を促進するものである.長方形噴流の例を図5・26に示す.音波により励起された噴流中の長方形渦輪列が対になって干渉し,三次元的な渦構造の誘起速度の効果により,円形噴流と比較して,長方形噴流の断面積と連行量が著しく増大している.図中の욄は噴出口の等価直径,は噴出口の断面積,は速度が噴出速度の10%以上である領域の噴流断面積,は噴出口での流量,はにおける流量である.また,噴出口を波状や花弁状にすることにより縦渦を生成して,混合を促進したり騒音を低減する手法が広く利用されている.噴出口の周方向に位相差をもったかく乱によりせん断層を励起して,噴流中にらせん状の渦を発生させて混合を促進することもできる.この手法については,MEMS(micro-electro-mechanicalsystems)を利用した実験も行われ,その有効性が示されている.噴出口にタブを取付けると混合促進および騒音低減に効果がある.この手法は特に超音速噴流のスクリーチ音の低減に有効である.噴流中の渦の生成を弱める手法として,噴出口内面に柔毛を取付けることも有効である.旋回を伴う軸対称噴流は,せん断流不安定と遠心力不安定に起因する渦の干渉により,大きな拡散性を有するので,燃焼などの工学分野で広く利用されている.
