# 5・4・1 後流の基礎

> **章**: 第5章 粘性流体の流れ
> **節**: 5・4 後流

物体の下流に生じる流れを後流(wake)、あるいは伴流という。流れの中に置かれた物体や、静止流体中を移動する物体の壁面には、境界層が形成される。この境界層は物体の後縁、あるいはそれよりも上流ではく離し、はく離点からはく離せん断層(separationshear layer)が放出される。後流は、このはく離せん断層のさまざまな干渉や変形過程を経て、種々の特性を呈しながら下流方向に発達していく。固体壁の影響を受けないこのようなせん断流れを、自由せん断流(free shear flow)という。図5・27に、後流の一般的なフローパターンの概略を示す。非常に低いレイノルズ数の場合を除き、鈍い物体(bluffbody)と呼ばれる円柱や球の後流では、物体背後の下流部に逆流を伴うはく離領域が形成される。このはく離領域の存在は、物体に働く大きな圧力抵抗の原因となっている。はく離せん断層の厚さは、物体のスケールに比べて十分薄いので、はく離せん断層は渦層(vortex sheet)によって近似できる。渦層は擾乱に対して不安定であるため、はく離せん断層は物体背後の低い背圧とケルビン・ヘルムホルツの不安定性、およびはく離せん断層自身の循環による誘起速度によって巻(a) 円柱後流(b) 水平平板後流図5・27 後流の概略上がり、物体幅よりも数倍程度大きな渦に成長し、大規模な渦となって下流に放出される。この非定常な渦放出現象に伴って、物体まわりの循環や圧力分布は大きく変動するため、物体には変動流体力(揚力・抗力)が生じる。したがって、渦放出に周期性があると、物体には周期的な変動流体力が働き、これが騒音や振動の原因となることがある。二次元物体からの渦放出に周期性がある場合は、回転方向の異なる渦が互い違いに規則的に並んだ渦列が形成される。カルマン(Ka썝rma썝n,1911年)は理想流体中で無限に続く渦列(渦糸列)の安定性を調べた。その結果、渦列の一周期に相当する渦間隔を、これに垂直な方向の渦列の間隔をとしたとき、渦配列が/=0.2806の千鳥状にある場合のみ渦列は安定に存在できることを示した。このような千鳥配列で周期性のある二次元渦列を、一般にカルマン渦列(Ka썝rma썝n vortex street)と呼ぶ。なお実際の円柱後流の実験では、/<0.2806となる報告が多い。カルマン渦列のように渦が規則的に並んだ状態の後流を渦列後流(vortex street wake)という。渦列後流は下流でさまざまな三次元変形、分裂、合体等を繰返し、十分遠方では周期性のない発達した乱流後流(turbulent wake)となる。薄い水平平板のような流線形物体の後流では、はく離が物体下流端の一点で起こるため、はく離領域は形成されず、はく離直前の境界層速度分布が、そのまま後流の初期速度分布となる。したがって、後流の初期状態は、互いに異なる符号の渦度をもつ薄い渦層が平行に接した状態のせん断層となる。レイノルズ数が低いとき、このせん断層は粘性によって渦度を弱めながらほぼ一直線状に流下する。しかし、レイノルズ数が臨界値以上になると、せん断層はケルビン・ヘルムホルツの不安定性によって波状に振動し、やがて下流で巻上がる。この臨界レイノルズ数は、平板長さを代表長さとすると、およそ700といわれている。この渦形成に周期性のある場合はカルマン渦列が形成されるが、その周波数や周期性の強さ、および渦形成距離等は、レイノルズ数や主流乱れの状態など、さまざまな要因に対して敏感に影響を受けるため、不確定性が強い。後流領域を物体からの距離と構造の特徴によって分類すると、物体から近い順に近傍後流(near wake)、中間後流(intermediatewake)、遠方後流(far wake)に分けることができる。近傍後流は、境界層がはく離し、はく離せん断層が巻上がり、渦が形成され、その渦が放出されるまでの渦形成領域(vortex formationregion)をいう。近傍後流の特性は、物体形状によって異なる。中間後流は、周期的な渦放出によって形成された渦列が明りょうに存在する渦列後流領域をいう。遠方後流は、複雑に乱れた乱流後流領域をいう。乱流後流では、後流内部の乱流域と後流外部(主流)の非乱流域との境界面は大規模にわん曲しており、したがって後流外縁部では乱流と非乱流が間欠的(intermittent)に現れる。これは乱流中の組織的構造(coherent structure)によるものと考えられている。遠方後流の特性は物体形状に依存せず、相似則が成り立つ。なお、後流を遠方後流と近傍後流の2領域に分ける場合には、近傍後流に中間後流を含める。図5・28に後流の主要パラメータを、図5・29に遠方後流速度の時間平均特性の一例を示す。ここで主流方向を軸とし、これと直角に,軸をとり、瞬時の速度ベクトル,,を、その時間平均成分,,と変動成分′,′,′の和として表す。また流体の密度をρとする。後流内部の時間平均速度は、主流速度に比べて小さく、その差욃=-を欠損速度(defect
