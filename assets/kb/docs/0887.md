# 5・4・2 二次元物体の後流

> **章**: 第5章 粘性流体の流れ
> **節**: 5・4 後流

a. 円柱・垂直平板二次元円柱の近傍~中間後流のフローパターンは、レイノルズ数=/νによって大きく変化する。ここに:円柱直径、ν:動粘度である。が約1以下のとき、流体は円柱に沿って上下・左右対称に流れ、層流状態を維持する。が約1を超えると、円柱表面の境界層ははく離し、円柱背後に安定した双子渦が形成される。双子渦はの増加とともに大きくなり、が約40を超えると不安定な振動を始め、渦が交互に放出され、後流にはカルマン渦列が形成される。が約200以下のカルマン渦列は、はく離せん断層が層流状態を保ったまま巻上がるため、渦列内部も層流状態となり、後流内部の速度変動波形は、渦配置に基づく正弦波状の周期変動成分のみとなる。このような渦列を層流渦列(laminar vortex street)という。しかしが約4004―42流体工学を超えると、はく離せん断層は巻上がり時に乱流に遷移し、渦列内部にさまざまな不規則変動成分が発生する。このような状態の渦列を乱流渦列(turbulent vortex street)という。をさらに増加させると、渦列内の不規則変動成分は増加するが、10の高いにおいても、後流内に渦列の存在することが確認されている。図5・30に、二次元乱流渦列後流内の速度変動波形とそのスペクトルの一例を示す。速度変動波形は、渦列の配置に基づく振幅の大きな周期的変動に、さまざまな周波数成分の不規則変動が重畳していることがわかる。この波形のスペクトルには、渦放出周波数で、卓越したスペクトルが見られ、強い周期性をもつ変動であることがわかる。また前後の比較的低い周波数帯にも大きなエネルギーをもち、周波数が高くなるに従ってエネルギーは減少する。スペクトルの低周波数成分は、はく離流脈線の三次元性、カルマン渦列の渦転位現象(vortex dislocation)、縦渦の存在など、三次元構造に起因するものと考えられているが、その詳細は物体形状によっても異なり、不明な点が多い。一方、スペクトルの高周波成分は、定性的には物体形状によらず、ほぼ周波数の-5/3乗に比例する。これは等方性乱流におけるコルモゴロフ(Kolmogorov)の-5/3乗則と一致する。したがって、渦列に重畳された小スケールの渦は等方性乱流的な性質をもち、渦列を構成する大規模な渦からエネルギーを受取り、順次小規模な渦に分裂していく。これをエネルギーのカスケードダウンという。ここでは物体の代表長さ(投影長さ)であり、円柱の場合は直径、垂直平板の場合は高さを用いる。ストローハル数は一般にレイノルズ数の関数となる。図5・31に円柱と垂直平板のストローハル数のレイノルズ数に対する変化を示す。しかし、乱流渦列が形成される300≦≦2×10の範囲では、=0.19~0.21のほぼ一定値となる。一様せん断流中に置かれた円柱からの渦放出は、レイノルズ数だけでなく、せん断パラメータによって変化する。図5・32は、との組合せに対して、周期的な渦放出の有無の境界を示す臨界曲線を表している。図5・32から一様流中では渦放出の起きない低レイノルズ数の場合でも、わずかなせん断があると渦放出が起きること、および一様流中では明りょうな渦放出が起きる場合でも、臨界せん断パラメータを超えると渦放出が消滅することがわかる。b. 角柱図5・33に、迎え角のある正方形断面柱のストローハル数を示す。の増加傾向は、迎え角α=13°で大きく変化している。これはこの迎え角で角aからはく離したせん断層が物体壁面abに再付着するためである。正方形断面柱の後流には、迎え角によらずカルマン渦列が形成される。ただし0°<α<13°では、互いに逆回転する渦の循環の大きさはどちらも等しく、その形状や最大渦度の大きさはそれぞれ大きく異なる非対称な渦列が形成される。図5・34に、非対称性が最も強くなるα=3°の渦度分布を示す。
