# 5・4・3 三次元物体の後流

> **章**: 第5章 粘性流体の流れ
> **節**: 5・4 後流

球後流のフローパターンは、円柱と同様にレイノルズ数によって大きく変化する。図5・36に、一様流中に置かれた球の後流フローパターンの概略との関係を示す。およそ≦130では、球背後に定常的に固定された渦輪が形成されるが、が約130を超えると渦輪の後端部が揺動し始め、渦輪から波状の流脈が放出される。が約300を超えるとヘアピン状の渦環(vortex loop)が周期的に放出される。この渦環は、が約420を超えると放出方向に不規則性が現れ、が約480を超えると後流中心軸まわりにゆっくりと回転しながら流下する。が約800を超えると球表面からはく離した筒状の渦層に周期性のある脈動が生じ、この筒状渦層が渦形成領域を包むように成長する。また球後部から放出される渦環は乱流化した状態で放出され、筒状渦層から分離したいくつかの小さな渦管と複雑に絡み合った状態で流下し、大きく揺動した後流を形成する。したがって、このときのストローハル数は、筒状渦層の脈動に基づく高周波数と、後流の揺動に基づく低周波数の二つが観測される。球が一様せん断流中にある場合は、渦の放出方向は高速側に固定され、その周波数はせん断パラメータの増加とともに大きくなる。またループ状渦の放出開始レイノルズ数は、せん断パラメータの増加とともに小さくなる。円板後流のフローパターンは、基本的には球と同様であるが、そのレイノルズ数依存性は比較的小さく、ストローハル数は10≦≦10程度の範囲で0.13~0.14であることが知られている。だ円板の後流は、短・長軸の存在により、球や円板等の軸対称物体のそれと異なった特徴を示す。例えば物体の短・長軸と後流幅の短・長軸が逆転する現象(軸転換)などが報告されている。
