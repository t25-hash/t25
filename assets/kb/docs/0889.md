# 5・4・4 境界層中に埋没する物体の後流

> **章**: 第5章 粘性流体の流れ
> **節**: 5・4 後流

乱流境界層中に埋没する二次元円柱の場合、フローパターンは境界層厚さδにあまり影響を受けず、おもに/の値によって分類される。ここには円柱と平面壁とのすきま間隔、は円柱の直径である。/<0.3のとき、渦放出は抑制され、円柱背面には停滞渦が形成される。また円柱上面ではく離したせん断層は、平面4―43図5・38 乱流境界層に埋没する二次元円柱のストローハル数図5・39境界層中に直立する有限長円柱からの渦放出壁に再付着する。/≧0.3では、周期的な渦放出が起こり、下流にはカルマン渦列が形成されるが、壁面の影響が完全に消滅し、一様流中の円柱と同様の流れとなるのは、/≧3.0の場合である。ここには物体の平面壁からの高さ、は物体の幅(円柱の場合直径)である。フローパターンは/に対して臨界値が存在し、これより大きいとカルマン渦が、小さいとアーチ形渦が形成される。図5・39にこれらの渦の概略を示す。乱流境界層に直立する物体の/の臨界値は、角柱、円柱、平板のそれぞれに対して、2.0,2.5,1.25である。
