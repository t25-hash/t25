# 5・4・5 後流の干渉

> **章**: 第5章 粘性流体の流れ
> **節**: 5・4 後流

二つ以上の物体が近接して設置されている場合や、物体が別の物体の後流中に設置されている場合は、複雑に干渉を受けた後流が形成される。図5・40は、中心間隔で主流に対して垂直に配置された二円柱(並列二円柱)からの渦放出ストローハル数である。円柱直径を代表長さとしたレイノルズ数が、10≦≦10の範囲にあれば、/>2.0のとき単独円柱とほぼ同じストローハル数でカルマン渦列が形成される。二円柱がこれより近接すると円柱間の流れは噴流状態となり、噴流はコアンダ効果により片側円柱に偏向し、抗力、後流幅、ストローハル数などの特性が各円柱で異なる非対称流れとなる。このとき噴流の偏向した側の円柱後流は狭くなり、は増加するが、もう一方の円柱後流は、その逆となる。二円柱がほとんど密着すると後流は合体し、直径が2倍の円柱と同様な後流特性をもつ。並列多円柱の場合は、並列二円柱と同様に、円柱間隔が小さくなると円柱間噴流は偏向するが、その方向は隣接円柱で交互に変化する。二円柱を主流方向に中心間隔で配置する(直列二円柱)と、上流側円柱の後流と下流側円柱が複雑に干渉し合い、渦放出特性が大きく変化する。が十分大きい場合は、両円柱とも同一の周波数で渦放出を行うが、が小さい場合は、上流側円柱のはく離せん断層は下流側円柱に再付着したり、あるいは下流側円柱に再付着することなく流下し、完全はく離型の細長物体と同様の後流特性に近付く。この再付着現象は/およびに大きく影響を受けるため、ストローハル数は急変する。図5・41に、直列二円柱のスト4―44流体工学ローハル数を示す。およそ/=3.8以下で再付着現象の起こることがわかる。直列多円柱の後流は、が小さい場合、基本的には直列二円柱の上流側円柱と同様になる。すなわち最下流端の円柱を除き、上流側円柱の渦形成領域内には下流側円柱があるため、周期的な渦放出は最下流端円柱以外では生じない。一方、がある程度大きくなると、直列二円柱の渦放出特性に近付く。が10~10程度の範囲における直列多円柱では、/≦1.5で周期的渦放出は消滅し、/≧3.0で直列二円柱のストローハル数と同程度になる。
