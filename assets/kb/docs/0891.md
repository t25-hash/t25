# 5・4・6 後流の制御

> **章**: 第5章 粘性流体の流れ
> **節**: 5・4 後流

渦放出の周期性を制御する方法の一つとして、スプリッタプレート(splitter plate)が知られている。これは物体下流部の渦形成領域に設置された主流に対して平行な平板であり、この平板の長さや設置位置によって、渦放出の周期性の有無やその周波数を変化させることができる。図5・42に、円柱および垂直平板の背面に設置されたスプリッタプレートの長さとストローハル数の関係を示す。が増加すると、上下二つのはく離せん断層の相互干渉が抑制され、はく図5・40 並列二円柱のストローハル数図5・41 直列二円柱のストローハル数図5・42 スプリッタプレート付き物体のストローハル数図5・43 円柱とスプリッタプレートのすきまとストローハル数離せん断層の巻上がりはスプリッタプレート下流端以降で起こるため、渦形成領域が下流側に拡大する。ところが/が円柱の場合は4、垂直平板の場合は2.5を超えると渦放出の周期性は減少し、さらにが増大すると、はく離せん断層はスプリッタプレート上に再付着して周期的渦放出は消滅する。一方、スプリッタプレートと物体背面間にすきまがある場合は、このすきま間隔との組合せにより、ストローハル数は大きく変化する。図5・43に、これらの関係を示す。円柱を半円状に二分割し、分割面にすきまを設けた円柱(スリットをもつ円柱)の後流では、スリット幅が増加するとストローハル数は減少する。また、円柱後縁からの吹出しあるいは吸込みによって後流特性を制御する例もある。吹出し流量を増加させるとストローハル数は緩やかに増加したのち、急激に増加し、その後周期性は消滅する。一方、吸込み流量を増加させると、は緩やかにかつ単調に減少し、速度変動の強さも減少する。
