# 5・5・3 はく離せん断層のかく乱に対する応答特性

> **章**: 第5章 粘性流体の流れ
> **節**: 5・47 に示されるように壁面上にはく離点は存在せず、次の条件を

a. 自由せん断層この流れのレイノルズ数は、薄板後端のはく離点における運動量厚さθを用いて=θ/νと表される。≒1000を境に、それより小さければ層流、それ以上であれば乱流となる。自由せん断層の速度分布には変曲点が存在するため、いかなる周波数のかく乱も増幅される。b. 壁面近くのはく離せん断層:はく離泡図5・44(b)に示すように、厚板の先端角からはく離したせん断層は、先の位置で再付着する。はく離点から再付着点までに形成される循環領域がはく離泡である。ここに、θははく離点における運動量厚さである。なお、は位置におけるはく離せん断層で観測される特徴的な速度変動周波数である。この周波数で再付着点から渦が放出される。このような渦放出周波数の決定機構には流れのフィードバック機構が作用していると考えられている。鈍頭円柱の場合、軸対称のはく離泡が形成される。かく乱周波数が高くなると=20において再付着長さは極大値をとる。c. 近接する二つのはく離せん断層:円柱、噴流流速の一様流中に置かれた直径の円柱において、かく乱を与えない場合の後流における渦の通過に伴う速度変動の周波数をとする。主流速度に周期的速度変動を重畳させる場合には、主流に平行な4―48流体工学方向の速度変動がはく離点およびはく離せん断層全体に与えられる。これに対し、円柱を流れと平行に微小振幅で振動させる場合には、流れ方向の速度変動がはく離点近傍にだけ与えられる。これら両者を流れ方向加振(in-line forcing)という。この状態をロックオン(lock-on)またはロックイン(lock-in)という。与えるかく乱の強さに依存し、それが強いほどは大きい。このように2付近のかく乱周波数を与えると後流の渦の通過周波数はほぼであり、二つのせん断層における渦が互い違いに配列する。なお、3および4付近のかく乱周波数を与えてもロックオンするが、二つのせん断層における渦配列は対称となる。上述の場合と同様、与えるかく乱の強さに依存し、それが強いほどは大きい。噴流のせん断層が層流である場合、噴出口付近においてK-H 不安定により周波数で小さな渦が巻上がり、下流においてそれらが合体することにより大きな渦構造が形成される。
