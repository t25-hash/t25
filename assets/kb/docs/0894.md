# 5・5・3 はく離せん断層のかく乱に対する応答特性

> **章**: 第5章 粘性流体の流れ
> **節**: 5・47 に示されるように壁面上にはく離点は存在せず、次の条件を

a. 自由せん断層この流れのレイノルズ数は、薄板後端のはく離点における運動量厚さθを用いて=θ/νと表される。≒1000を境に、それより小さければ層流、それ以上であれば乱流となる。自由せん断層の速度分布には変曲点が存在するため、いかなる周波数のかく乱も増幅される。その中で最も増幅率の高いかく乱の周波数욈は、層流では、욈θ/=0.032、乱流では、욈θ/=0.044~0.048である。はく離点近傍に設けたスリットから周期的変動を伴う吹出し、およびスピーカによる音響加振などによっての0.01~0.1%程度の振幅をもつ周波数욞のかく乱を与えると、渦の合体に起因する욥の周波数をもつ周期的速度変動が観測される。b. 壁面近くのはく離せん断層:はく離泡図5・44(b)に示すように、厚板の先端角からはく離したせん断層は、先の位置で再付着する。はく離点から再付着点までに形成される循環領域がはく離泡である。レイノルズ数=/νが約300以上になると、はく離せん断層にはケルビン・ヘルムホルツ(K-H)不安定(Kelvin-Helmholtz instability)により最初に巻上がる渦が観測され、その周波数はθ/=0.024で与えられる。ここに、θははく離点における運動量厚さである。また、これをはく離点から測った距離を用いて表すと、욈/=0.50となる。なお、욈は位置におけるはく離せん断層で観測される特徴的な速度変動周波数である。したがって、再付着点において観測される速度変動周波数욋は、욋=0.50/で与えられる。この周波数で再付着点から渦が放出される。このような渦放出周波数の決定機構には流れのフィードバック機構が作用していると考えられている。鈍頭円柱の場合、軸対称のはく離泡が形成される。はく離点近傍に周波数욞の周期的かく乱を与えた場合、かく乱強さが主流速度の10%以下であれば、無次元かく乱周波数=욞/,:円柱の直径)が1.6~2.1のとき、はく離泡が縮退し、再付着長さが最小となる。かく乱強さが主流速度の14~20%の場合、2~3のときが最小となる。このとき、かく乱周波数と再付着点からの渦放出周波数との間に욞=5욋の関係がある。かく乱周波数が高くなると=20において再付着長さは極大値をとる。この周波数は/2に相当する。c. 近接する二つのはく離せん断層:円柱、噴流流速の一様流中に置かれた直径の円柱において、かく乱を与えない場合の後流における渦の通過に伴う速度変動の周波数を욋とする。この無次元周波数=욋/は1.0×10≦=/ν≦2.0×10において=0.2である。主流速度に周期的速度変動を重畳させる場合には、主流に平行な4―48流体工学方向の速度変動がはく離点およびはく離せん断層全体に与えられる。これに対し、円柱を流れと平行に微小振幅で振動させる場合には、流れ方向の速度変動がはく離点近傍にだけ与えられる。これら両者を流れ方向加振(in-line forcing)という。与えるかく乱周波数욞を욞=2욋±に選ぶと、後流における渦の通過に伴う速度変動の周波数욋は욋=1/2욞となる。この状態をロックオン(lock-on)またはロックイン(lock-in)という。なお、の範囲は<1.2욋程度である。与えるかく乱の強さに依存し、それが強いほどは大きい。このように2욋付近のかく乱周波数を与えると後流の渦の通過周波数はほぼ욋であり、二つのせん断層における渦が互い違いに配列する。なお、3욋および4욋付近のかく乱周波数を与えてもロックオンするが、二つのせん断層における渦配列は対称となる。円柱を流れと垂直に微小振幅で振動させる場合(transverse for-cing)、욞=욋±でロックオンが起こる。なお、の範囲は<0.6욋程度のものである。上述の場合と同様、与えるかく乱の強さに依存し、それが強いほどは大きい。また、円柱の回転振動および音波によって与えるかく乱においても욞=욋でロックオンが起こる。噴流のせん断層が層流である場合、噴出口付近においてK-H 不安定により周波数で小さな渦が巻上がり、下流においてそれらが合体することにより大きな渦構造が形成される。直径の円形オリフィスから流速で噴出する噴流において、大規模構造渦の発生周波数に起因する速度変動周波数욋は욋/=0.63~0.80で表される。また、前述と同じ直径の円形ノズルによる噴流の場合では、욋/=0.3~0.5である。直径の円管内において発達した管内乱流が噴出する場合、明確な速度変動のスペクトルピークを示さないが、なだらかなピークの周波数は4.6×10<=/ν<2.1×10において욋/=0.3~0.5となる。種々の断面形状のオリフィスから噴出する噴流を욞=/4または/8の周波数で励起すると、いずれの場合も渦の合体が促進され、連行(entrainment)が増加し噴流の広がりが大きくなる。
