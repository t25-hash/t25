# 5・5・4 はく離の制御

> **章**: 第5章 粘性流体の流れ
> **節**: 5・47 に示されるように壁面上にはく離点は存在せず、次の条件を

a. 受動制御(passive control)制御のために付加的な動力を必要としない方法を受動制御と呼ぶ。翼の負圧面上に取付けたボルテックスジェネレータまたはラフネス、翼前縁に取付けたセレーション(serration)などによって誘起される縦渦を利用するものがある。縦渦の連行作用により、主流から運動量の大きな流れを壁面近傍に取込み、速度分布においてつねに/<0となるようにする。翼においては、スラット、フラップのように、翼の正圧面側と負圧面側の圧力差を利用してすきまから流れを吹出させ、壁面近くの速度を増速することではく離を防止する。管路内においては、直角コーナ部の角部およびディフューザにおけるはく離を抑えるためにガイドベーンを取付ける。ディフューザにおいては流れに直角に挿入した金網がはく離防止に効果がある。この制御方法では、はく離していないときにも制御装置そのものが流れにさらされており、有害抵抗となること、予想外の状況で失速が起こったときには迅速な対処ができないことなどが問題である。b. 能動制御(active control)制御のために付加的エネルギーを必要とする制御方法で、あらかじめ決めた方法で制御するもの(predetermined,open-loop control)、ある計測量を基に制御量を決定するもの(reactive,feedforward,open-loop control)、制御量を入力に再び戻し、制御量を決定するもの(reactive,feed-back,closed-loop control)がある。はく離点近傍のスリットから吹出す一定速度の壁面噴流または周期的に強さを変化させる壁面噴流、ボルテックスジェネレータジェット(VGJ)、ケーシングから間欠噴流(渦輪)を打込むもの、壁面にあけた小さな孔からの吸込み、回転円柱・ムービングウォールなどがある。はく離流れを生じる流れと制御方法の組合せは多い。また、ステップ後方のはく離領域の縮退には、はく離点における運動量厚さθを用いた無次元周波数θ/0.01のかく乱が最も効果的である。運動エネルギーを流れに注入するために、制御装置分の重量が増加すること、取付けのためのスペースが必要となることが問題となる。これを解決できる可能性を秘めているものに、MEMS の技術があるが、1ユニットのMEMS のパワーが小さいので、はく離制御に対して用いるのにまだ課題が残されている。c. 適応・最適制御積極的に失速を予知し、抑制し、さらにそれが適切かどうか診断し、もう一度抑制動作に還元するような制御が、流れの変化にきめ細かく対応するものとして重要である。また、これは制御エネルギーの節約にもつながる。はく離が生じる前に制御を開始し、制御エネルギーを最小にできる適切な制御方法、および最小消費エネルギーで済む制御開始時刻が求められている。また、制御を開始するためにははく離の開始を予測しなければならない。そのため、層流境界層においてはく離が起こる直前の壁面圧力変動波形および速度変動波形に、境界層の変曲点不安定に起因する変動が出現することが見いだされており、それが層流はく離の前兆信号の一つと考えられている。文献(1) Temam,R.,Navier-Stokes Equations、(1985)、North-Holland.(2) 巽友正・後藤金英、流れの安定性理論、(1976)、産業図書。(3) Drazin,P.G.and Reid,W.H.,Hydrodynamic Stability、(1981)、Cambridge University Press.(4) Squire,H.B.,Q.
