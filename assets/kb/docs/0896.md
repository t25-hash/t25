# 6・1・1 線形安定論

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・1 乱流の発生、遷移

安定論(stability theory)では、層流に加えられたかく乱(dis-turbance)の増幅、減衰を扱うが、かく乱が微少でかく乱どうしや主流との干渉が無視できるとした線形安定論は、オア・ゾンマーフェルト方程式(Orr-Sommerfeld equation)に基づいて展開される。=0の解を中立曲線といい、中立曲線上でが最小となる点が遷移の始まる臨界レイノルズ数(critical Reynolds num-ber)である。この場合、流れ方向に進行する(二次元)かく乱の臨界レイノルズ数は、流れに傾いて進行する(三次元)かく乱よりつねに小さいことが証明されている。トルミーン(Tollmien)とシュリヒティング(Schlicht-ing)は境界層に平行流近似を適用してかく乱(T-S 波)の安定限界と増幅率を求め、=∞(非粘性流)では中立曲線が閉じて安定となることを示した(図6・1)。この結果は速度分布に変曲点をもつ噴流や混合層、後流などの自由せん断層が、非粘性の極限において不安定解をもつ(変曲点不安定、inflexional instability)こととは基本的に異なる。境界層などの速度分布に変曲点をもたない流れが=∞で安定であることは、レイリーの第一定理として知られているが、このような流れが粘性の寄与する有限のレイノルズ数範囲で不安定になるのは、分子運動によって平均流のエネルギーがかく乱に供給されるためである。回転二重円筒に現れるテイラー渦(Taylor vortex)は、流れの曲率による遠心力不安定(内筒の回転速度が外筒のそれよりも大きい場合に限る)によるものである。なお、遠心力による不安定は、凹面に沿う境界層にも現れ、流れ方向に並んだ渦対列(ゲルトラー渦)を形成する。また、加熱した容器の流れでは浮力による不安定によって細胞渦(ベナール渦)が形成される。
