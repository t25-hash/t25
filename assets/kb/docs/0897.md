# 6・1・2 非線形安定論

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・1 乱流の発生,遷移

かく乱が大きくなると,相互の干渉や平均流の変形が起こって,エネルギーの流れが大きく変わることが実験的に知られているが,これについての理論的解析はほとんどなされていない.ただ,かく乱の振幅があまり大きくない場合には,dd=2α+αとおいて,線形解(右辺第1項)からのずれとして追跡することができる.これによれば,初段階でのかく乱の増幅,減衰がさらに促進される場合,増幅が抑制されて一定値-ααに漸近する図6・1平板境界層の線形安定特性[:主流速度,δ:排除厚さ,文献(8)では욂=520]場合[(α>0,α<0),超臨界平衡,supercritical equilibrium],およびいったん減衰したのち再び増幅に転じる場合α<0,α>0などが生じる.最後の形の不安定(亜臨界不安定,subcriticalinstability)は非線形域でかく乱の振幅が-ααを超えたときに起こる.
