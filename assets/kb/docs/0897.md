# 5・2・1 電力ネットワークの発展

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・2 電力ネットワーク

電源の立地が遠隔地化，大容量化することに対しては，送電線を
高電圧化して大容量化することで対応してきた．電力ネットワーク
の発展は図5･12に示すとおりで，発電機容量が伸びるとともに，
変電所の容量，送電線や配電線の亘長が増加してきた．また，送電
容量の増加に対して，ネットワークの構成機器の高電圧，大容量
化，ならびに高機能化が進められてきた．
a.架空送電線 架空送電線は大電力を長距離に送電するため
の基本となる設備で，電気事業が始まって以来，着実に発展を続け
ている．電力再編以来，戦後復興期における遠距離の水力電源から
の送電，経済成長期には需要家近傍の火力電源の連系，さらにオイ
ルショック以降は遠距離立地のLNG，原子力電源からの送電とし
て発展を続けてきた．また，電力の経済的利用，緊急時応援などの
ための系統間連系にも活用され，現在は沖縄を除き，一部に直流連
系，周波数連系を介して全国連系が完成している．架空送電線の大
容量化の様相を図5･13に示す．
154kV送電線は，戦後しばらくの間は，水力電源からの送電線
として基幹系統に用いられてきたが，275kV送電線の普及，500
kV送電線の開発に伴い，二次系統として扱われるようになった．
この154kV高圧送電線と275kVを含む超高圧送電線において，
絶縁設計の大幅な違いがある．前者では中性線の抵抗接地など有効
接地方式を採用し，落雷等による送電線事故において，健全相に過
電圧が生じるのを許容して地絡電流を抑制しているのに対し，後者
では直接接地方式を採用し，地絡電流が増えることを許容して絶縁
設計の合理化を実現している（図5･14）．
b.地中送電線 地中送電は，都市部の電力供給方式として，
としている．
図5･15のような形で発展してきた．高度成長期に154kV，275
kVの送電線に採用されてきたが，500kVケーブルや地下変電所
が開発されるに伴い，外輪系統から都市中心部への電力供給系統が
構築されるに至った．地中送電線は油で絶縁，冷却を行うOFケー
ブルから実用化が始まったが，油の供給等の設備を必要としない固
体絶縁のCVケーブルの実用化が進んでいる．500kV級CVケー
ブル（長距離）の普及には，接続部の絶縁厚の低減が重要な課題と
(ｂ)直接接地方式（超高圧送電線) (ａ)有効接地方式（高圧送電線)
供給する系統と，産業などに高品質な電力を供給する系統に分ける
ことができ，前者は樹枝状構成で，後者はネットワーク構成となっ
ている（図5･18）．樹枝状系統では電力が順次分配されるが，幹の
系統での電流が大きく，端に行くほど小さくなるため，線路損失に
よる電圧低下に注意する必要がある．また，幹の部分事故が末端の
需要家に影響する．このため，配電系統の電圧を一定の範囲内に維
持できるように，配電用変圧器の出口電圧の調整に留意する必要が
あり，事故の影響を軽減するため別ルートからの緊急時供給を可能
一方，ネットワーク系統は電圧維持が容易で，系統事故に対して
も，供給支障の発生を抑えることが可能である．しかしながら，系
統事故時の事故遮断が複雑であるため，負荷の運転状態を十分に考
慮した保護方式の開発が重要である．
