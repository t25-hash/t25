# 6・1・3 かく乱の乱雑化とカオス理論

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・1 乱流の発生、遷移

安定論では線形、非線形いずれの場合も、選択的に増幅する特性周波数のかく乱を対象としており、これが乱流に移行していく過程を解明するには、さらに波数や位相の変化を織り込んだ乱雑化の理論が必要であるが、これについての定式化は全くなされていない。乱流を予想不能な事象(カオス、chaos)として追求し、遷移を分岐論により説明しようとする試みがなされているが、現在のところ有限次元のモデル方程式による類推だけで、流体運動の方程式を取扱ったものはない。乱流は不規則運動であるという点を除いて層流と区別する根拠はなく、層流を記述するナビエ・ストークス方程式、連続の式および境界条件としての粘着条件がそのまま適用される。しかし、初期条件は一義的ではなく、これに応じて無数個の解が存在することになるから、速度分布や流れの挙動を知るという実用的な目的に対しては統計的方法が有効である。式(6・2)はレイノルズ方程式(Reynolds equation)と呼ばれ、変動による見掛け応力(レイノルズ応力、Reynoldsstress)-ρを含む点でナビエ・ストークス方程式と異なる。式(6・4)の右辺第1項はレイノルズ応力によるエネルギーの生産項(production term)で、その大きさは平均流のエネルギーの損失に等しい。第2項は粘性による散逸項(dissipation term)、第3項は拡散項(diffusion term)である。レイノルズ方程式(6・2)は、平均値のほかに変動の二重積を未知量にもつためにこのままでは解けないので、応力輸送方程式(6・3)を援用するが、これにはさらに高次の乗積が含まれていて、系を閉じることができない(完結問題、closure problem)。ナビエ・ストークス方程式から出発した方程式(6・2)および方程式(6・3)がこのような不完全形となったのは、平均値という概念を導入したためである。系を閉じるには変動量の乗積平均についての考察が必要であり、統計的取扱いはその一つである。
