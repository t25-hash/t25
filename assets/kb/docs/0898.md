# 6・1・3 かく乱の乱雑化とカオス理論

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・1 乱流の発生,遷移

安定論では線形,非線形いずれの場合も,選択的に増幅する特性周波数のかく乱を対象としており,これが乱流に移行していく過程を解明するには,さらに波数や位相の変化を織り込んだ乱雑化の理論が必要であるが,これについての定式化は全くなされていない.乱流を予想不能な事象(カオス,chaos)として追求し,遷移を分岐論により説明しようとする試みがなされているが,現在のところ有限次元のモデル方程式による類推だけで,流体運動の方程式を取扱ったものはない.乱流は不規則運動であるという点を除いて層流と区別する根拠はなく,層流を記述するナビエ・ストークス方程式,連続の式および境界条件としての粘着条件がそのまま適用される.しかし,初期条件は一義的ではなく,これに応じて無数個の解が存在することになるから,速度分布や流れの挙動を知るという実用的な目的に対しては統計的方法が有効である.そのため,流れの速度などの物理量を時間平均=lim1욧욪d(平均は変動の時間尺度に比べて十分長い時間間隔についてとるが,平均値の緩やかな変化には及ばないものとする)と変動値=-とに分け,平均量に対する方程式を作ると,速度を,圧力をとして,非圧縮性流体の連続の式と運動方程式が옿욡옿욡=0(6・1)옿욡옿+욅옿욡옿욅=-1ρ옿옿욡+1ρ옿옿욅μ옿욡옿욅-ρ욡욅(6・2)となる.式(6・2)はレイノルズ方程式(Reynolds equation)と呼ばれ,変動による見掛け応力(レイノルズ応力,Reynoldsstress)-ρ욡욅を含む点でナビエ・ストークス方程式と異なる.レイノルズ応力の輸送方程式は옿옿욡욢+욅옿옿욅욡욢=-욡욅옿욢옿욅+욢욅옿욡옿욅+ρ옿욡옿욢+옿욢옿욡-2ν옿욡옿욆옿욢옿욆+옿옿욆욡욢욆+ρ욡δ욢욡+욢δ욡욆-ν옿욡욢옿욆(6・3)で与えられ,=とおくと変動のエネルギー方程式옿옿+욆옿옿욆=-욡욆옿욡옿욆-ε-옿옿욆ρ+욡욡욆-옿옿욆-ν옿욡욆옿욆(6・4)が得られる.ただし,=욡욡2は単位質量当たりの乱れの運動エネルギー,ε=ν옿욢옿욡옿욢옿욡+옿욡옿욢はエネルギー散逸率(dissipation rate)である.式(6・4)の右辺第1項はレイノルズ応力によるエネルギーの生産項(production term)で,その大きさは平均流のエネルギーの損失に等しい.第2項は粘性による散逸項(dissipation term),第3項は拡散項(diffusion term)である.レイノルズ方程式(6・2)は,平均値のほかに変動の二重積を未知量にもつためにこのままでは解けないので,応力輸送方程式(6・3)を援用するが,これにはさらに高次の乗積が含まれていて,系を閉じることができない(完結問題,closure problem).ナビエ・ストークス方程式から出発した方程式(6・2)および方程式(6・3)がこのような不完全形となったのは,平均値という概念を導入したためである.系を閉じるには変動量の乗積平均についての考察が必要であり,統計的取扱いはその一つである.
