# 6・3・1 確率分布とモーメント

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・3 乱流の統計理論

乱流の統計理論では座標、時刻における変動速度を時空間の点,における偶然関数とみなす。変動速度,の統計的状態は、時空間における任意の点,,,,...,욈,욈における速度,,...,욈の結合確率密度(joint probability den-sity),,...,욈によって記述されるが、これを決定するには変動量のすべての相乗積モーメントを知る必要がある。流体運動の方程式に含まれるモーメントを一般的に表すことはもちろん不可能であるが、特殊な乱流構造を仮定すれば、その具体的な記述がある程度可能になる。
