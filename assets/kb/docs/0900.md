# 5・3・2 ビルの電源システム

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・3 需要家システム

ビルの電気設備は，従来，照明，空調，エレベータ等の負荷が主
体であったが，最近では，インテリジェント化された24時間稼働
ビルや，IT化のインフラとなるデータセンタ等が普及し，負荷の
容量が大幅に増えてきている．また，電源供給の信頼度に関しても
さらなる向上を求められ，負荷の重要性に応じて，UPS（uninter-
ruptible power supply，無停電電源装置）などの停電対策機器が
設けられる．図5･33は大型テナントビルの66kV受変電設備の単
線接続図の一例である．
工場設備の場合，開放型や全天候型空気絶縁キュービクルが採用
される場合もあるが，ビルの場合，66kVまたは77kV系統にお
いては，設置スペースを縮小するため，ほとんどのケースでSF6
ガスで絶縁されたキュービクル型ガス絶縁開閉装置が採用されてい
る．ただし，22kVまたは33kV受電の場合には，サイズの縮小
という面で，66kVまたは77kVほどガス絶縁による効果が大き
くないため，空気絶縁型のキュービクルが採用されることも多く，
最近では，空気絶縁で主回路部分が密封され，キュービクル型ガス
絶縁開閉装置と同等の機能のものも登場している．
VCTと並列に設置されている断路器は，15年ごとに実施される
電力計量用VCT（MOF）の取替えの際，停電が許されない設備の
ために設けられたバイパス用断路器である．電力会社によってはこ
のバイパス方式を認めていない場合もあり，その場合はバイパス断
路器の代わりに，もう1台電力計量用VCT（MOF）を設け2
VCT（MOF）方式とする．
