c .マイクロガスタービン マイクロガスタービンは，従来の
ガスタービン発電より1けた以上低い出力で電力が供給できること
から，電力自由化の流れに乗って，小工場やスーパーマーケット，
集合住宅などの需要家に注目されている．背景には，1995年の電
気事業法の改正により10kW未満の発電機は一般工作物となり，
導入に際し電気主任技術者が不要であることがある．
数十kWのMGTは，古くは潜水艦の電源や自動車のターボチ
ャージャとして製品化されているが，小工場やスーパーマーケッ
ト，集合住宅等で用いるためには，メンテナンスフリー化のほか
に，騒音やNOなどの環境対策が重要となる．また，熱の有効利
用を考えたコージェネレーションシステムとの併用も，普及のため
に欠かせない．海外ではMGT本体の開発が進んでいるが，わが
国では評価試験やシステム化技術により，導入の方策を検討してい
る状況である．
