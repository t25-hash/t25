# 6・3・5 非等方性乱流

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・3 乱流の統計理論

実際の流れに見られる乱流は非等方で,しかも自由乱流境界の間欠的構造で代表される大規模運動(large scale motion)や壁面近くのバースト(burst)現象のような秩序運動(coherentmotion)が重畳していて,これを純粋な統計理論の対象とすることが困難なため,その記述には半経験的乱流モデルが用いられている.ただ,速度こう配が一定で乱れの統計的性質が座標軸の平衡移動に対して普遍とみなされる発達した自由せん断乱流の部分に対しては,一様乱流を仮定した解析を適用してエネルギーの生成や配分についての検討が行われている.乱流理論の近年の理論については文献(23)に詳しい.
