# 6・2・2 項で導出されたレイノルズ方程式と同様に、乱流場において

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・4 乱流拡散

流体の単位質量に含まれる任意のスカラ量Φの輸送は以下の式で表される。右辺第3項は、平均操作の際に対流項の非線形性によって現れる乱流拡散を表す。この項に含まれる変動速度と変動濃度の積はベクトル量であり、乱流流束(turbulent flux)と呼ばれる。すなわち、乱流拡散は乱流中の不規則な速度変動による対流輸送に起因することを表している。とτは流れ場の性質によって局所的に決定されるべき量である。式(6・17)は、乱流の作用が拡散係数を増加させることを表し、その際、実効的な拡散係数はΓとなる。乱流拡散係数Γと分子拡散係数Γの比は乱流ペクレ数(turbulent Peclet number)であり、発達した乱流場では100~1000のオーダである。ただし、その値は流れ場の内部でも大きく変化する。固体壁面近傍で粘性により速度が零に漸近するような場では乱流ペクレ数は1かそれ以下のオーダになる。このように、乱流拡散を拡散係数の増大として表す古典的な方法は直感的ではあるが、乱流現象の一断面を表しているにすぎない。例えば、式(6・15)は変動速度ベクトルと拡散物質Φの空間こう配ベクトルが平行であることを表すが、その物理的な根拠はなく、実験・シミュレーションともにむしろその反証となる結果を示している。
