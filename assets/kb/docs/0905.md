# 6・2・2 項で導出されたレイノルズ方程式と同様に,乱流場において

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・4 乱流拡散

流体の単位質量に含まれる任意のスカラ量Φの輸送は以下の式で表される.DΦD=옿옿욢Γ옿Φ옿욢+-옿옿욢욢′φ′(6・14)式(6・14)中,ΓはΦの分子拡散係数であり,は化学反応などによるΦの単位質量当たりの生成・消散を表す.右辺第3項は,平均操作の際に対流項の非線形性によって現れる乱流拡散を表す.この項に含まれる変動速度と変動濃度の積はベクトル量であり,乱流流束(turbulent flux)と呼ばれる.すなわち,乱流拡散は乱流中の不規則な速度変動による対流輸送に起因することを表している.乱流流束をこう配拡散の形で近似して욢′φ′=-Γ욧옿Φ옿욢(6・15)と表すとき,拡散係数Γ욧を乱流拡散係数[turbulent diffusivity,または渦拡散係数(eddy diffusivity)]と呼ぶ.Γ욧は場の関数で,その次元はなので乱流の長さスケール욧と時間スケールτによってΓ욧=욧τ(6・16)で表すことができる.욧とτは流れ場の性質によって局所的に決定されるべき量である.式(6・15)を式(6・14)に代入すると,DΦD=옿옿욢Γ+Γ욧옿Φ옿욢+=옿옿욢Γ윬읆읆옿Φ옿욢+(6・17)となる.式(6・17)は,乱流の作用が拡散係数を増加させることを表し,その際,実効的な拡散係数はΓ윬읆읆となる.乱流拡散係数Γ욧と分子拡散係数Γの比は乱流ペクレ数(turbulent Pe썝clet number)であり,発達した乱流場では100~1000のオーダである.ただし,その値は流れ場の内部でも大きく変化する.固体壁面近傍で粘性により速度が零に漸近するような場では乱流ペクレ数は1かそれ以下のオーダになる.このように,乱流拡散を拡散係数の増大として表す古典的な方法は直感的ではあるが,乱流現象の一断面を表しているにすぎない.例えば,式(6・15)は変動速度ベクトルと拡散物質Φの空間こう配ベクトルが平行であることを表すが,その物理的な根拠はなく,実験・シミュレーションともにむしろその反証となる結果を示している.
