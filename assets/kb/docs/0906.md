# 6・4・2 運動量の輸送

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・4 乱流拡散

運動量の乱流輸送は、式(6・14)の被輸送量φを速度ベクトルの成分욡に置換えることで表現される。このとき、6・2・2項で示されたとおり、乱流流束に相当する物理量がレイノルズ応力욡욢である。ここで、式(6・15)と同様の考え方でレイノルズ応力を表現すれば욡욢=-ν욧옿욡옿욢+옿욢옿욡+23δ욡욢(6・18)となる。式の右辺のν욧は渦動粘性係数(kinematic eddy viscos-ity)で、乱流拡散係数と同じ次元をもつ。右辺2項めのは=12 욡(6・19)で定義される乱流エネルギー(turbulent kinetic energy)であり、δ욡욢はクロネッカのデルタで=のとき1、それ以外では零となる。右辺第1項は平均速度場による変形速度テンソルで対角和が零の対称行列となる。第2項は両辺の対角和が等しくなるように形式的に導入された。渦動粘性係数ν욧は便宜的に乱流の速度スケール욧と長さスケール욧の積としてν욧=욧욧(6・20)で表すことができ、ν욧と動粘性係数νの比ν욧ν=욧욧ν ≡욧(6・21)は乱流レイノルズ数(turbulent Reynolds number)である。式(6・18)を運動量の式に代入すれば、乱流場での実効的な粘性係数ν윬읆읆はν윬읆읆=ν+ν욧=1+욧νであり、乱流レイノルズ数が十分大きければ分子粘性の作用は無視される。ただし、本来のレイノルズ応力の空間分布は、6・2・3項の応力輸送方程式によって表されるとおり、種々のプロセスの釣合いから決定されるべきものであり、式(6・18)の成立根拠は式(6・15)の場合と同様にそれほど高くない。
