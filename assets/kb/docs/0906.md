# 6・4・2 運動量の輸送

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・4 乱流拡散

運動量の乱流輸送は、式(6・14)の被輸送量φを速度ベクトルの成分に置換えることで表現される。式の右辺のνは渦動粘性係数(kinematic eddy viscos-ity)で、乱流拡散係数と同じ次元をもつ。右辺第1項は平均速度場による変形速度テンソルで対角和が零の対称行列となる。第2項は両辺の対角和が等しくなるように形式的に導入された。ただし、本来のレイノルズ応力の空間分布は、6・2・3項の応力輸送方程式によって表されるとおり、種々のプロセスの釣合いから決定されるべきものであり、式(6・18)の成立根拠は式(6・15)の場合と同様にそれほど高くない。
