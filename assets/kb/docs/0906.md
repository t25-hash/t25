# 6・1・1 線形安定論

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・1 乱流の発生，遷移

安定論では線形，非線形いずれの場合も，選択的に増幅する特性周波数のかく乱を対象としており，これが乱流に移行していく過程を解明するには，さらに波数や位相の変化を織り込んだ乱雑化の理論が必要であるが，これについての定式化は全くなされていない．
乱流を予想不能な事象（カオス，chaos）として追求し，遷移を分岐論により説明しようとする試みがなされているが，現在のところ有限次元のモデル方程式による類推だけで，流体運動の方程式を取扱ったものはない．
安定論（stability theory）では，層流に加えられたかく乱（dis-turbance）の増幅，減衰を扱うが，かく乱が微少でかく乱どうしや主流との干渉が無視できるとした線形安定論は，オア・ゾンマーフェルト方程式（Orr -Sommerfeld equation) に基づいて展開される．
この方程式は4階の常微分方程式で，固有値＝＋＝α,によりかく乱の増幅特性を与える．
はかく乱の進行速度， は対数増幅率，αは波数， はレイノルズ数で，＝ －1である．
