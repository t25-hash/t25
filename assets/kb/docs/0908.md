# 6・4・4 物質の拡散

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・4 乱流拡散

物質の拡散の場合も同様に式(6・15)の被輸送量の変動φ′を変動濃度′に,乱流拡散係数を乱流物質拡散係数욧にそれぞれ置換え,욢′′=-욧옿옿욢(6・24)と表される.乱流シュミット数(turbulent Schmidt number)욧を定義して욧=ν욧욧(6・25)とすれば욧を渦動粘性係数から求める場合がある.乱流プラントル数욧と同様に욧も1に等しいとおかれる場合が多いが,これは実験的な情報の欠如に起因するところが大きく,不確実な要素が多い.いずれにしても乱流拡散係数の導入により乱流拡散をこう配拡散として近似すること自体,乱流現象の大胆な近似にほかならず,それに立脚する係数を厳密に決定することにはそれほど物理的な意義はない.
