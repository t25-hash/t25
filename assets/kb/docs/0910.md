# 6・4・6 圧縮性流体の乱流

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・4 乱流拡散

流体の圧縮性を考える際,レイノルズ平均法の立場をとるならば,流体密度の変化を考慮する必要がある.密度の時間変化を,平均密度と変動密度の和でρ=ρ+ρ′と表すとすると,アンサンブル平均を施した質量保存の式は옿ρ옿+옿옿욢ρ욢+옿옿욢ρ′욢′=0となり,変動密度と変動速度の相関項が新たに考慮すべき未知数となる.ファーブル平均法(Favre averaging)はこれを避けるために密度で重みを付けたオペレータを導入する.任意の確率密度関数P,すなわち욪Pd=1を用いて,物理量φの密度重み付き確率密度をPφ=1ρρPρ,φdρと定義することで,φの平均値および二次モーメントを以下のように定義する.φ=ρφρ=φPφdφφ′′=ρφ′′ρ=φ-PPφdφここで,通常のレイノルズ平均と異なる点は,瞬時の物理量をφ=φ+φ′′と表現することにあるが,φ=ρφρ,φ′′≠0となることから,この関係を用いれば質量保存の式は옿ρ옿+옿옿욢ρ욢=0となり,密度変動と速度変動の相関項を考える必要がない.しかし,相関項を消去することはできても,圧縮性乱流の速度場を決定するためにはエネルギー式(エンタルピー輸送方程式)を考えなければならず,非圧縮の場合に比較すると事情が複雑になることに変わりはない.
