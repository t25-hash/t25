# 6・5・1 レイノルズ平均(RANS)モデル

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・5 乱流モデル

RANS モデルは、レイノルズ応力を直接未知量とした輸送方程式を導入するタイプ(応力方程式モデル)と、渦粘性概念を利用するタイプ(渦粘性モデル)に分類できる。6・5節ではこれらの代表的なもののみを概説する。詳細については文献(26)などを参照されたい。まず、渦粘性概念に基づく乱流モデルについて述べる。乱流スペクトルの普遍則(コルモゴロフ則)に従えば、乱れエネルギー(あるいは、速度スケール)と長さスケールの二つによって乱流効果を代表するのは自然な考え方といえ、これらは2方程式モデル(twoequation model)として分類される。これらモデルは原理的には相互に変換可能であり、個々の事例で予測性能が異なるのはモデル定数や付加項の相違によるものと考えられる。また、境界層など限定された問題ではスケールの一つ(もしくは二つとも)を代数式で与える1方程式モデル(0方程式モデル)がある。工学的に最も応用例の多い-εモデルについて一般的な表式は以下のように表される。以下に、-εモデルのいくつかのバリエーションを分類して示す。. ロンダー・スポルディング(Launder-Spalding)モデル(標準モデル)-εモデルの原型といえるもので、乱れが十分発達し、渦粘性が分子粘性に卓越する高レイノルズ数状態を対象としており、壁面の修正付加項やダンピング効果は考慮していない。モデル定数は、一様等方乱流(格子乱流)、平板境界層、二次元噴流などの実験データと試行計算により次のように与えられている。第1格子点が<300ならば比較的良好に適用できる。計算法としては微分方程式の解からを与えて他の境界条件を算出することが多い。. レイノルズ平均モデルの計算法応用計算に際しては、多変数が非線形に連成する乱流モデルにはしばしば計算の不安定性が問題となる。乱流モデルの計算に際しての重要なポイントをいくつか挙げておく。. 計算格子速度場に比べて乱流量,εはより急峻な分布をもつことが多いので、格子配分に注意が必要である。特に、低レイノルズ数型モデルを用いる場合の壁近傍の解像度は重要で、例えば、-εでは第1格子点が<2に、-ωでは2<<6にあれば実用上十分との報告がある。応力方程式モデルの条件はやや厳しく、<5に9点以上の格子が望ましいとされる。また、格子均一性が悪いところでは、数値誤差により乱流量に非物理的な解を与えて発散を生じるおそれがある。. 解の健全性(realizability)RANS モデルが物理的に妥当性を保つには、そこに含まれる乱流の時間スケール(例えば、-εモデルでは=ε)が適切な値でなければいけない。また、RANS モデルは乱流エネルギーの生成と散逸がほぼ等しい平衡状態を基準として構築されているので、両者の比が著しく1から離れると計算上の不安定となりやすい。このような問題は、物理的な条件よりもむしろ、数値誤差や、不適切な初期条件や流入条件によっても生じることが多い。
