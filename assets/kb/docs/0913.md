# 6・6・1 壁乱流の制御

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・6 乱流の制御

固体壁面に沿う乱流の壁面近傍には縦渦とそれに誘起される低速ストリーク構造が存在し、その不安定化がバースト、レイノルズ応力生成や高波数へのカスケードへと発展する。この縦渦を中心とした乱流構造(準秩序構造)は有限な寿命をもち、やがて減衰するが、別の新たなストリークが発生し、その繰返しによって全体として一つの平衡状態が保たれている。その過程で引き起こされる壁面に向かっての運動量輸送が乱流摩擦を生み出している。この一連の過程に介入することによって別の平衡状態を実現し、乱流摩擦を低減することが壁乱流制御のおもなねらいである。従来の提案の大部分は直感と経験から出発したものであったが、その後の実験や数値解析技術の進歩のおかげで、効果があると認められたものについては、効果の機構が乱流構造との関連においてほぼ説明できるようになっている。a. ストリーク発生の予防境界層中に迎え角をもった翼型などを設置することで乱流摩擦が低下する場合がある。LEBU(large eddy break-up device)と呼ばれるこの手法は、LEBU 板の後縁から離脱する孤立渦を利用して外層領域を流下するスパン方向大規模渦を弱め、縦渦・ストリークへの進化を妨げているものと解釈されている。速度変動が最も抑制される無次元周波数範囲があり、これが乱流境界層中の大規模渦構造の特性周波数範囲に入るように寸法を決めるのが効果的とされている。ただし、期待どおり供試面上の摩擦抵抗は減少させることができたとしても、同時にLEBU 板自身の表面での摩擦が加わるため、総合的に見て効果が得られるとは限らない。b. 発生したストリークの消去いったん発生したストリークを弱め、あるいは消去することでバースト発生率を低下させる方法にアクティブフィードバック制御がある。チョイ(Choi)らは、=10での速度情報を検知し、縦渦の回転に対抗する方向に壁面から吹出し・吸込み流を与えることで、15%程度の抵抗低減が可能としている。制御入力としてより実用性の高い壁面せん断応力や壁面静圧を用いる方法も提案されているが、いずれもDNS による可能性の調査の段階であり、実現のためには準秩序構造と実際に干渉できる程度の時・空間スケールと十分な変位量やエネルギーを有するアクチュエータおよび制御アルゴリズムの開発が課題であり、十分高いレイノルズ数への適用が可能かどうかは予断をゆるさない。c. ストリークの安定化ストリークの平均間隔より狭い間隔で、主流方向に縦溝を多数、平行に配置したリブレット面が乱流摩擦低減に有効であることが知られている。ストリークのスパン方向への遥動を阻止して安定化させることによって、バースト発生や小スケールへのエネルギーカスケードが抑制されたものと理解されている。d. ストリークのランダム化ストリークとそれに重畳される斜行波の干渉がバースト発生に深く関係しているとの見通しから、V 型の突起により縦渦と斜行波の位相関係をランダム化することでバースト発生率を低下させる試みが、チャネル流について試みられ、12%の抵抗低減が可能であることがDNS により示されている。e. その他壁面のスパン方向への加振によって摩擦抵抗が大幅に減少することがDNSおよび実験によって示されている。加振により生じる渦糸の転向により生まれたスパン方向渦度成分が、壁面近傍の速度を減速させることによると説明されているが、準秩序構造との関連は明らかでない。微量の鎖状高分子溶液の添加が摩擦抵抗低減をもたらすことが知られていて、壁面垂直方向の速度変動の減少、ストリーク間隔の増大、バースト発生率の低下が報告されている。また界面活性剤やマイクロバブルの添加も、遷移層におけるレイノルズ応力の生成運動を抑制する効果が期待されている。これらは流体自身のレオロジー特性に起因すると考えられているが、乱流構造との因果関係を含めて、物理的機構については十分明らかとはいえない。
