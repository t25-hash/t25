# 6・6・2 自由乱流の制御

> **章**: 第6章 乱流理論と乱流拡散
> **節**: 6・6 乱流の制御

壁乱流制御のおもな目的が、乱流の活動を弱めて摩擦抵抗を減らすことにあるのに対し、噴流など自由乱流の制御では混合の促進・抑制およびその結果として得られる拡大率の制御がおもな目的である。自由乱流の特性を主として支配するのは、せん断面に直角な軸をもつ大規模渦(ローラ渦)で、その形成、相互干渉、合体、三次元化と縦渦形成、崩壊などの過程が制御のターゲットとなる。変極点不安定、渦の自己誘導・伸張・転向つなぎ換えなどの、主として非粘性・非線形過程が一連の現象を支配する点で、粘性支配の領域が重要な役割を果たす壁乱流とは本質的に異なり、また壁乱流の制御では壁面全域が対象となるのに対し、自由乱流では形成初期段階がおもな制御対象となる点でも壁乱流の制御とは異なっている。円形噴流の出口端近傍の流れを支配する渦輪の放出を、軸方向の平面音波かく乱の重畳によりロックすれば、その後の渦輪の合体の促進、混合や噴流拡大率の向上が得られる。一方、乱流混合にとって欠かせない下流での三次元化と縦渦の形成は、初期段階において幾何学条件あるいは流入条件に含まれていた微小な三次元性に特に敏感であることから、ノズルの断面形状の非軸対称化や上流における三次元かく乱の付与によって噴流の成長を制御することが期待される。なかでも非円形断面ノズルの使用は、非軸対称噴流特有の周方向不安定による渦輪の変形により、混合と連行作用の活発化をもたらす。この効果は圧縮性噴流において特に顕著である。噴流出口(jet lip)での局所的吸込みによる大振幅変動の励起と混合の促進、多重周期変動の付加による渦輪の偏心、渦輪軸の偏向、進行方向の異なる多数の渦輪への分裂と、噴流の拡大率の見掛け上の大幅増加が報告されている。文献(1) Reynolds,O.,Phil.
