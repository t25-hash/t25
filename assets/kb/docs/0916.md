# 7・1・2 音速とマッハ数

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・1 圧縮性流れの基礎

振幅が十分小さい微小な圧力の擾乱、すなわち音波が伝ぱする速度は音速(acoustic velocity,sonic velocity)と呼ばれる。音波の通過による気体の状態変化はエントロピーが一定に保たれる可逆断熱変化(等エントロピー変化)とみなすことができ、気体の比熱比をγとすれば、完全気体の音速は式(7・2)で表される。温度は場所によって異なるから、音速も場所によって変化する。密度変化の割合dρρとマッハ数との間には次式が成り立つ。すなわち、流体の運動に伴う密度変化(圧縮性)の程度を表す量は速度そのものではなくマッハ数である。
