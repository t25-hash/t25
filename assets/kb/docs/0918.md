# 7・1・4 圧縮性流れの分類

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・1 圧縮性流れの基礎

一般に圧縮性流れはマッハ数によって次のように分類される。4―59a. 非圧縮性流れ(incompressible flow)厳密には密度が一定で変化せず、したがって音速が無限大で、=0の流れをいうが、通常工学上の問題では、dρρ<0.05の範囲を近似的に非圧縮性流れとみなして差支えない。これは<0.3の流れに相当する。b. 亜音速流れ(subsonic flow)<1の流れをいうが、一般には流体の圧縮性の効果が無視できないマッハ数0.3程度以上で、流路内や流れの中に置かれた物体まわりに衝撃波が発生する状態までの範囲をいう。上限のマッハ数は流路形状や物体形状により異なるが、0.8程度である。管路における定常な亜音速流れでは、断面積の小さい所で速度が大きく圧力が低いなどの定性的性質は非圧縮性流れと同じである。c. 遷音速流れ(transonic flow)流れ場に亜音速流れの領域と超音速流れの領域が共存する流れをいう。このような流れは一般に主流のマッハ数が約0.8より大きく、約1.2より小さい場合で、衝撃波を伴う。翼のまわりの遷音速流れでは、マッハ数のわずかな変化で揚力と抗力が急激に変化する。d. 超音速流れ(supersonic flow)>1の流れをいうが、一般には遷音速流れと極超音速流れの間の約1.2<<約5の流れをいう。>1の超音速流れの重要な特徴は、流速が音速より大きいため、流れ場の中の一つの微小な点擾乱の影響は、擾乱源を頂点とし半頂角αがsinα=1で表される円すいの内部に限られることである。角αをマッハ角(Mach angle)、この円すいをマッハ円すい(Mach cone)という。また、超音速流れの中の任意の点において、流線に対してマッハ角αだけ傾いた線を、マッハ線(Mach line)という。マッハ線は流れの中に生じた微小な擾乱が伝わる軌跡を表しており、マッハ波(Mach wave)とも呼ばれる。超音速流れが亜音速に減速される際には、ほとんどの場合、衝撃波が発生する。e. 極超音速流れ(hypersonic flow)超音速流れにおいて、マッハ数が約5以上の流れを極超音速流れといい、それ以下のマッハ数の超音速流れと区別する。極超音速流れの中の物体の先端には極めて強い衝撃波が生じ、物体表面では気体の粘性による空力加熱が大きくなる。このため一般に極超音速流れでは、気体を完全気体として取扱うことはできない。
