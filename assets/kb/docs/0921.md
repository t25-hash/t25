# 7・2・3 垂直衝撃波

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・2 一次元定常流

音波はその通過によって生じる気体の圧力変化が十分小さい微小振幅波である.圧力変化が大きい有限振幅波は,その通過によって気体の圧力,温度,および密度が増加する圧縮波(compressionwave)と,減少する膨張波(expansion wave)に分類される.流れの中に生じる圧縮波は次第に集中して,その幅が小さくなる傾向があり,ついには圧力,温度,および密度が不連続的に増加する波面を形成して安定する.このような波を衝撃波(shock wave)という.衝撃波は流れに相対的に音速より速い速度で伝ぱする.したがって,流れの中に静止した衝撃波がある場合,衝撃波の上流は超音速で,流れは衝撃波を通過すると亜音速に減速される.衝撃波による気体の状態変化は断熱不可逆変化で,流れのエントロピーは衝撃波を通過すると増加する.上流の流れ方向に対し,波面が垂直の衝撃波を垂直衝撃波(nor-mal shock wave),斜めの衝撃波を斜め衝撃波(oblique shockwave)という.図7・3に示すように,定常な垂直衝撃波を囲む薄い検査体積を考え,衝撃波の上流に添字1,下流に添字2を付けて表すと,連続の式,運動方程式,およびエネルギー式はρ=ρ+ρ=+ργγ-1ρ+12=γγ-1ρ+12(7・18)となる.これらの基礎式より,衝撃波前後の状態量に関する次式が得られ,これらはランキン・ユゴニオの式(Rankine-Hugoniotequation)と呼ばれる.ρρ=γ+1γ-1+1+γ+1γ-1(7・19)図7・3垂直衝撃波の検査体積図7・4 垂直衝撃波前後の関係(γ=1.4)(a) 分枝衝撃波(約1.3<<約1.5)(b) ショックトレーン(>約1.5)図7・5管内における垂直衝撃波と境界層の干渉=+γ+1γ-1γ+1γ-1+1(7・20)また,垂直衝撃波による諸量の変化を衝撃波上流のマッハ数の関数として表すと=2γ-γ-1γ+1(7・21)=2γ-γ-1γ-1+2γ+1(7・22)ρρ=γ+1γ-1+2(7・23)=γ+1γ-1+2γ+12γ-γ-1(7・24)=γ-1+22γ-γ-1(7・25)となる.これらの関係をγ=1.4の場合について図7・4に示す.
