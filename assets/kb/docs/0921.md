# 7・2・3 垂直衝撃波

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・2 一次元定常流

音波はその通過によって生じる気体の圧力変化が十分小さい微小振幅波である。圧力変化が大きい有限振幅波は、その通過によって気体の圧力、温度、および密度が増加する圧縮波(compressionwave)と、減少する膨張波(expansion wave)に分類される。流れの中に生じる圧縮波は次第に集中して、その幅が小さくなる傾向があり、ついには圧力、温度、および密度が不連続的に増加する波面を形成して安定する。このような波を衝撃波(shock wave)という。衝撃波は流れに相対的に音速より速い速度で伝ぱする。したがって、流れの中に静止した衝撃波がある場合、衝撃波の上流は超音速で、流れは衝撃波を通過すると亜音速に減速される。衝撃波による気体の状態変化は断熱不可逆変化で、流れのエントロピーは衝撃波を通過すると増加する。上流の流れ方向に対し、波面が垂直の衝撃波を垂直衝撃波(nor-mal shock wave)、斜めの衝撃波を斜め衝撃波(oblique shockwave)という。これらの基礎式より、衝撃波前後の状態量に関する次式が得られ、これらはランキン・ユゴニオの式(Rankine-Hugoniotequation)と呼ばれる。これらの関係をγ=1.4の場合について図7・4に示す。
