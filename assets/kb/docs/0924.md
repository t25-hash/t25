# 7・2・6 ファノー流れ

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・2 一次元定常流

断面積が一定の管路における圧縮性流体の流れは、管路の長さが比較的短い場合には断熱流れと考えてよい場合が多い。しかし、実在気体では粘性による摩擦があるから、等エントロピー流れではない。このような摩擦を伴う断面積一定の断熱流れをファノー流れ(Fanno flow)という。ファノー流れのある断面における流れの状態と管路の摩擦係数が既知であるとすると、流れの連続の式、運動方程式、エネルギー方程式、および状態方程式を用い、別の断面における流れの状態を知ることができる。図7・8に示すように、エントロピーとエンタルピーをそれぞれ横軸と縦軸にとり、ファノー流れにおける流れの状態変化の軌跡を描いた曲線をファノー線(Fanno line)という。ファノー線上のエントロピーが最大の点aは、マッハ数が1の臨界状態を示し、点aより上の部分は亜音速状態、下の部分は超音速状態に対応する。ファノー流れではエントロピーはつねに流れ方向に増大するから、亜音速流れでは流れは加速されてマッハ数は1に近づく。一方、超音速流れでは減速されて、マッハ数はやはり1に近づく。したがって、管路がある程度長い場合には、流れのチョークが起こる。
