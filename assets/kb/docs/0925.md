# 7・2・7 レイリー流れ

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・2 一次元定常流

断面積が一定の管路において、外部との熱の授受はあるが、流体摩擦を無視できる流れをレイリー流れ(Rayleigh flow)という。このような流れにおいても、ある断面における流れの状態と外部との間で授受される熱量が既知であるとすると、流れの連続の式、運動方程式、エネルギー方程式、および状態方程式を用い、別の断面における流れの状態を知ることができる。ある流れに対して授受される熱量を変化させたときに得られる状態を-線図上に描いた曲線のことを、レイリー線(Rayleigh line)という。図7・8に示図7・8 ファノー線とレイリー線したレイリー線上のエントロピーが最大の点bは、マッハ数が1の臨界状態を示し、点bより上半分は亜音速状態、下半分は超音速状態に対応する。レイリー流れでは摩擦がないから、外部からの加熱はエントロピーの増加、冷却はエントロピーの減少をもたらす。したがって、亜音速流れでは流れは加熱によって加速されてマッハ数は1に近づき、冷却によって減速される。一方、超音速流れでは加熱により減速されて、マッハ数はやはり1に近づく。したがって、レイリー流れにおいても、加熱量によっては流れのチョークが起こる。また、垂直衝撃波の前後では質量流量、よどみ点エンタルピー、および衝撃関数が一定に保たれる。したがって、図7・8に示すように、-平面上に垂直衝撃波上流の点1を与えると、下流の点2は、点1を通るファノー線とレイリー線の交点2で与えられる。エントロピーは衝撃波によって必ず増加するから、点2は点1より右側にある。
