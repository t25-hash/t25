# 6・1・3 材料プロセスシミュレーションへの展開

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・1 個別要素法

一例として，図6･5に示すような粘度試験（capillary viscom-
eter ）の計算機シミュレーションを行った．非ニュートン粘性体で
も，チャネル内で発達し，一定のせん断ひずみ速度γに対する面
応力（せん断応力）τを測定すれば，見掛け粘度（apparent visco-
sity）νをν＝τγで求められるというのが，この粘度測定法の原
理である．そこで，実際の試験法と同様に，一定のストローク速度
で粒状体モデルを圧下し，試験装置モデルのチャネル壁での反力・
すべり速度を計算し，反力の単位面積平均をせん断応力として，見
掛け粘性を求めた．実験結果と粒状体モデルによる解析値とを比較
すると，図6･6において両者は良好に一致し，図6･4に示すモデル
が実際の粉末コンパウンドの粘性挙動を，定量的に再現できること
がわかった．興味深い点は，図6･4のモデルによって，容器壁での
すべり速度の定量的な予測に加えて，流動速度の増加に伴うチャネ
ル内の速度分布を精度良く記述できることであり，粉体・粒状体の
レオロジー（rheology）現象を解明するうえでも，粉末の離散挙
動を記述できる粒状体モデルが極めて有効であることがわかる．実
際，種々の粉末射出成形プロセス（powder injection molding）・
コロイドプロセスなどにおいて，その有用性が検証されている．
