# 7・3・2 微小振幅波と波動方程式

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・3 一次元非定常流と波動

断面積一定の管内の静止した気体中を波が伝わる問題を考える。すなわち微小振幅波の伝ぱを考える。この場合、一次元非定常等エントロピー流れに対する連続の式と運動方程式は線形方程式となり、これらの式と等エントロピーの式を連立することにより、式(7・28)が得られる。式(7・28)の第2式の一般解は式(7・29)で表される。同様に、式(7・29)の右辺第2項は、密度の変動ρ′がの負方向に速度で伝ぱすることを示し、波の形状は変化しない。の正方向に伝ぱする波を、前進波(forward-running wave)、の負方向に伝ぱする波を後進波(backward-running wave)という。したがって、式(7・29)は音速で伝ぱする前進波と後進波の二つの波の重ね合わせを示しており、式(7・28)の波動方程式が示す波の伝ぱ速度は音速である。
