# 7・3・2 微小振幅波と波動方程式

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・3 一次元非定常流と波動

断面積一定の管内の静止した気体中を波が伝わる問題を考える.乱されていない静止した気体の圧力を,密度をρとし,波の通過による圧力,密度,速度の変動量をそれぞれ′,ρ′,′とすれば,通過後の圧力は=+′,密度はρ=ρ+ρ′,速度は=′である.いま,波の通過による気体の状態変化は等エントロピー変化で,かつその変化は微小(′≪1,ρ′ρ≪1,′≪1,ただしは乱されていない静止気体の音速)であると仮定する.すなわち微小振幅波の伝ぱを考える.この場合,一次元非定常等エントロピー流れに対する連続の式と運動方程式は線形方程式となり,これらの式と等エントロピーの式を連立することにより,式(7・28)が得られる.옿′옿-옿′옿=0옿ρ′옿-옿ρ′옿=0(7・28)式(7・28)の形の方程式は一次元の波動方程式(wave equation)と呼ばれ,微小振幅波がある定まった速度で伝ぱする現象を記述するのに特有の方程式である.式(7・28)の第2式の一般解は式(7・29)で表される.ρ′,=-++(7・29)式(7・29)の右辺第1項は,密度の変動ρ′がの正方向に速度で伝ぱするが,式の形から明らかなように単なる平行移動であるから,波の形状は変化しないことを示している.同様に,式(7・29)の右辺第2項は,密度の変動ρ′がの負方向に速度で伝ぱすることを示し,波の形状は変化しない.の正方向に伝ぱする波を,前進波(forward-running wave),の負方向に伝ぱする波を後進波(backward-running wave)という.したがって,式(7・29)は音速で伝ぱする前進波と後進波の二つの波の重ね合わせを示しており,式(7・28)の波動方程式が示す波の伝ぱ速度は音速である.
