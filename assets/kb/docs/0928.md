# 7・3・3 有限振幅等エントロピー波

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・3 一次元非定常流と波動

振幅は有限の大きさをもつが、波によって生じる流れの速度や温度のこう配は小さく、波の通過による気体の状態変化は等エントロピー変化とみなせる波、すなわち有限振幅等エントロピー波を考える。これは、波の通過によって気体の圧力や密度が増加する圧縮波と、減少する膨張波に分類される。断面積一定の管内を有限振幅等エントロピー波が伝ぱする場合、前進波の前後で式(7・30)のが、また後進波の前後で式(7・30)のが一定に保たれる。有限振幅等エントロピー波の具体例として、図7・9に示すように、断面積が一定の長い管内に薄いピストンを置き、これを静止状態からの正方向に連続的に加速してある一定速度にする問題を考える。図7・9のOA で示す加速過程で、ピストンの右側には圧縮波、左側には膨張波が発生する。この圧縮波はの正方向に伝ぱする前図7・9ピストンの運動により形成される圧縮波と膨張波図7・10有限振幅波の伝ぱに伴う波形の変形4―63進波、膨張波は負方向に伝ぱする後進波である。また、後進膨張波に対してもリーマン不変量が一定に保たれることを用いると、膨張波通過後の状態を求めることができる。有限振幅等エントロピー波の伝ぱ速度は波形の各部分で異なり、圧力(あるいは密度)が高い部分ほど速い。例えば図7・10に示すように、時刻=0における圧力分布が正弦関数で表される有限振幅等エントロピー前進波を考える。この波はの正方向に伝ぱするから、図7・10のbcdの部分は圧縮波領域で、伝ぱ速度は点dで最も速く、点bで最も遅い。また、abとdOの部分は膨張波領域である。したがって、波形は時間とともに変形し、圧縮波領域は時間とともに急峻になるが、膨張波領域は次第に緩やかになる。圧縮波領域における速度や温度のこう配が非常に大きくなると、気体の粘性や熱伝導性が無視できなくなり、衝撃波が形成される。
