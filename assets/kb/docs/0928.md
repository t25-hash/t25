# 7・3・3 有限振幅等エントロピー波

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・3 一次元非定常流と波動

振幅は有限の大きさをもつが、波によって生じる流れの速度や温度のこう配は小さく、波の通過による気体の状態変化は等エントロピー変化とみなせる波、すなわち有限振幅等エントロピー波を考える。これは、波の通過によって気体の圧力や密度が増加する圧縮波と、減少する膨張波に分類される。断面積一定の管内を有限振幅等エントロピー波が伝ぱする場合、前進波の前後で式(7・30)のが、また後進波の前後で式(7・30)のが一定に保たれる。=+2γ-1=-2γ-1(7・30)ただし、式(7・30)のとはそれぞれ流れの速度と音速で、とは、リーマン不変量(Riemann invariant)と呼ばれる。有限振幅等エントロピー波の具体例として、図7・9に示すように、断面積が一定の長い管内に薄いピストンを置き、これを静止状態からの正方向に連続的に加速してある一定速度にする問題を考える。図7・9のOA で示す加速過程で、ピストンの右側には圧縮波、左側には膨張波が発生する。この圧縮波はの正方向に伝ぱする前図7・9ピストンの運動により形成される圧縮波と膨張波図7・10有限振幅波の伝ぱに伴う波形の変形4―63進波、膨張波は負方向に伝ぱする後進波である。前進圧縮波の前方1と後方2のリーマン不変量は等しいから、圧縮波によって加速された流れのマッハ数=を用いて、次式が得られる。==1-γ-1(7・31)静止気体の音速とピストン速度が既知の場合、式(7・31)の第2式の未知数はのみであるから、したがってが求められ、等エントロピーの関係を用いると、領域2の圧力や密度が得られる。また、後進膨張波に対してもリーマン不変量が一定に保たれることを用いると、膨張波通過後の状態を求めることができる。有限振幅等エントロピー波の伝ぱ速度は波形の各部分で異なり、圧力(あるいは密度)が高い部分ほど速い。例えば図7・10に示すように、時刻=0における圧力分布が正弦関数で表される有限振幅等エントロピー前進波を考える。この波はの正方向に伝ぱするから、図7・10のbcdの部分は圧縮波領域で、伝ぱ速度は点dで最も速く、点bで最も遅い。また、abとdOの部分は膨張波領域である。したがって、波形は時間とともに変形し、圧縮波領域は時間とともに急峻になるが、膨張波領域は次第に緩やかになる。圧縮波領域における速度や温度のこう配が非常に大きくなると、気体の粘性や熱伝導性が無視できなくなり、衝撃波が形成される。
