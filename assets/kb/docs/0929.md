# 7・3・4 ショックチューブ

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・3 一次元非定常流と波動

ショックチューブ(shock tube,衝撃波管)は,図7・11(f)に示すように,断面積が一定の長い管を隔膜で高圧室と低圧室の二つに仕切り,高圧室には高圧気体を,低圧室には低圧気体を充てんした極めて簡単な装置である.隔膜を前後の圧力差あるいは撃針で破ると,高圧気体は急激に膨張して低圧気体を圧縮する.その結果,低圧室内に衝撃波が形成され,右方向に伝ぱし,その背後に衝撃波によって誘起された高速気流が続く.ショックチューブは衝撃波の反射や回折など,衝撃波の特性を研究するには最適の装置である.また,衝撃波の後方に続く高速気流を利用して高速風洞としても使用されるが,ショックチューブによって得られる流れの持続時間は一般に数ms程度で,極めて短い.図7・11(a)において,最初低圧室にあった領域1の静止気体(a)(b)(c)(d)(e)(f)図7・11 ショックチューブにおける流れ4―64流体工学(比熱比γ,音速)は衝撃波の通過により加熱加圧され,領域2の気体となる.この気体を熱気体(hot gas)という.また,最初高圧室にあった領域4の静止気体(比熱比γ,音速)は膨張波により冷却膨張し,領域3の気体となる.この気体を冷気体(coldgas)という.図7・11(a)に一点鎖線で示す熱気体と冷気体の境界は接触面(contact surface)と呼ばれる.領域1と領域2を垂直衝撃波の関係で結び,また領域3と領域4を後進膨張波の関係で結び,接触面の両側の圧力と流速は等しいという条件を用いると,隔膜前後の初期圧力比と形成される衝撃波のマッハ数욦の関係として,式(7・32)が得られる.=2γ욦-γ-1γ-11-γ-1γ-1욦-1욦(7・32)式(7・32)より,が大きいほど욦は大きくなる.を無限大としたときの極限値は욦윯율융=γ+1γ-1(7・33)となる.式(7・33)より욦윯율융は高圧室気体の初期状態における音速に比例する.このため,強い衝撃波を得るには高圧気体としてヘリウムなどの軽い気体が用いられる.
