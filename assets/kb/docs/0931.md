# 7・4・2 ポテンシャル方程式の線形化

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・4 二次元定常流

式(7・39)は非線形で、その厳密解を求めるのは極めて困難である。しかし、流れの中に生じる擾乱が一様流れの値に比べて微小な場合、式(7・39)は線形化できる。すなわち、方向の速度(マッハ数)をもつ一様流れの中に、流れと平行に置かれた薄翼などの薄い二次元物体を考える。物体によって流れの中に生じる圧力や速度の擾乱は一様流れの値に比べて微小であるから、擾乱ポテンシャルをφとし、Φ=+φを式(7・39)に代入すると、式(7・40)が得られる。また、この線は上流から下流に向かって立つ観察者から見て左側に伝ぱするから、左向きマッハ線(left-running Mach line)と呼ばれる。同様に、右辺第2項は右側に伝ぱする右向きマッハ線(right-running Mach line)を表し、式(7・41)の解は両者の重ね合わせである。
