# 6・2・5 補足事項

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・2 離散転位動力学法

離散転位動力学は，メゾスコピック領域を扱う手法として発展途
上にあり，現時点で力学理論としてほぼ確立している事項につい
て，ここでは，等方線形弾性体の微小変形論の枠組内での，古典論
的取扱いについて概説した．異方性体の場合は，解析解の表現が難
しくなる点を除いて原理的に同じであり，また，特異性を示さない
解や非局所理論による解も，原理的にはここで述べたシミュレーシ
ョンに適用可能と考えられる．有限変形については現在のところ簡
便な方法は示されていない．また，転位の非古典的運動，例えばト
ンネル効果による転位運動や相対論的効果については，いくつかの
議論があり，特別なケースについて検討された例があるが，いまの
ところ一般的な方法論は確立されていない．PU（partition of
unity）有限要素法の離散転位動力学法への応用などの新しい手法
も提案されており，今後の発展が待たれる．
