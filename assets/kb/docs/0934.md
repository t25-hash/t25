# 7・4・5 斜め衝撃波

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・4 二次元定常流

波面が流線に対して傾いた衝撃波を、斜め衝撃波(obliqueshock wave)という。図7・14はその説明図で、波面が上流の流線となす角βを衝撃波角(shock angle)、流線が斜め衝撃波を通過することにより曲げられる角θを、転向角(turning angle)あるいは偏角(deviation angle)という。垂直衝撃波は斜め衝撃波においてβ=π2の特別の場合である。斜め衝撃波の前後の諸量の関係は、波面に垂直方向速度成分との間に垂直衝撃波前後の関係が成立すること、および波面に平行方向速度成分が波面の前後で等しいことから導かれる。前者は垂直衝撃波、後者はマッハ波に対応する。θ<θの場合の二つの解のうち、大きいほうのβに対応する衝撃波を、強い斜め衝撃波、小さいβに対応する衝撃波を、弱い斜め衝撃波という。図7・16はγ=1.4の場合の衝撃波極線図(shock polardiagram)で、斜め衝撃波を含む流れの解析に用いられる。
