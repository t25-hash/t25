# 7・5・1 層流圧縮性境界層(compressible boundary layer)

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・5 圧縮性境界層

流体の圧縮性が無視できない高速流では、粘性摩擦による発熱量が大きいため必然的に温度境界層が生じ、温度変化に伴う密度ρ、粘性係数μ、熱伝導率の変化を無視することができない。このため境界層を支配する運動方程式とエネルギー方程式は局所的に連成している。二次元圧縮性層流境界層方程式においては、ρ,μおよびの温度依存性、および流れ方向主流圧力こう配が特別な条件を満たすとき、適当な座標変換により自己相似解を見いだすことができる。
