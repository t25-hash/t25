# 7・5・5 境界層と斜め衝撃波の干渉

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・5 圧縮性境界層

境界層と垂直衝撃波との干渉に関しては,7・2・4項に述べられて図7・17 層流境界層内の温度と速度の関係図7・18 斜め衝撃波と層流境界層の干渉(b)(a)図7・19斜め衝撃波と乱流境界層の干渉いる.7・5・5項では,斜め衝撃波が境界層と干渉する流れについて,簡単に説明する.この干渉はタービン翼列内の流れでよく観察される.斜め衝撃波が境界層と干渉する場合,衝撃波による擾乱が境界層内の亜音速部を通って上流におよび,それが境界層自身の状態(厚さ,はく離の有無,遷移点)を変えるのみならず,境界層外の流れおよび衝撃波の状態を変える.この干渉状態を支配する主要パラメータは,レイノルズ数,マッハ数および衝撃波の強さ(圧力比または圧力係数)である.平板上の層流境界層が,斜め衝撃波との干渉によりはく離し,その下流で再付着するような場合の流れの模式図を図7・18に示す.衝撃波の入射によって,境界層はかなり上流より持ち上げられ,境界層内の圧力上昇は境界層厚さの数十倍程度上流まで及ぶ.境界層の外縁が凹面状となるため,圧縮波が発生する.入射衝撃波は,境界層の外縁において,自由境界面での反射のように反射されるため膨張波が発生する.衝撃波の入射の下流では,境界層の外縁が再び凸面状となるため,圧縮波が発生する.再付着後,境界層は乱流であることが多い.平板上の乱流境界層と斜め衝撃波の干渉の模式図を図7・19に示す.図7・19(a)は,入射衝撃波が弱く,はく離が起こらない場合で,入射衝撃波は境界層がない固体表面からの反射の場合とほぼ同じように反射される.入射衝撃波が強く,境界層がはく離する場合には,図7・19(b)に示すように,さまざまな反射波が発生するが,平板の表面から十分離れたところで,合体して一つの反射衝撃波となる.乱流境界層の場合,境界層厚さの変化は衝撃波の入射点のごく近傍において生じるので,層流境界層に比べて境界層内の圧力上昇の起こる範囲も小さい.の数値解法値計算法を述べる.この方法は散逸効果が若干大きい欠点はあるが,コードの生成が容易,安定性に優れるなどの利点がある.以下に理論的根拠は省略し,一般座標系による計算方法を述べる.以下の式において,はスカラ量,は4項列ベクトルで,=で,욕(=1~4)はベクトルの成分を示す.また,7・6節ではこの列ベクトルを=と表記する.はすべての成分が0である4項列ベクトルを示す.は4×4の行列である.下添字,は物理空間の座標点욡욢,욡욢における諸量で,욡욢,욡욢は物体の形状,予想される流線などを考慮して適当に定める.욡욢,욡욢に対応する一般座標の点をξ욡,η욢とする.,はの,に関する,,はξ,ηに関する偏微分係数である.一般座標に対するオイラー方程式は,옿옿+옿옿ξ+옿옿η+=(7・54)ここで,=1ρ,ρ,ρ,=1ρ,ρ+ξ,ρ+ξ,+=1ρ,ρ+η,ρ+η,+は変換ヤコビアンで,=1-.これを用いると,ξ=, ξ=-, η=-, η=は単位体積当たりの内部エネルギーで,比熱比をγとすると,=γ-1+12ρ+である.,は=ξ+ξ=η+ηは二次元流れでは,軸対称流れでは,=1ρ,ρ,ρ,+なお,=γρを音速として,次式でを定義する.≡γ-1+12+
