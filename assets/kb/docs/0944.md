# 6・5・4 終わりに

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・5 第一原理解析

以上，現在の固体材料の第一原理計算の標準的な手法について概
説した．VASP，CASTEPといったはん用プログラムの中身は基
本的に上記の内容である．一方，最近，実空間メッシュ法
が開発されてきている．密度汎関数理論に基づく第一原理擬ポテン
シャル法と効率的な固有値解法を用いる点は共通だが，波動関数を
平面波基底で展開するのでなく，固有関数の 部分を実空間
メッシュ点データとして収束計算を行う．コーン・シャム方程式の
ラプラシアンに関する演算は，差分法 や有限要素法 で数値的
に扱われる．波動関数を実空間の領域ごとに分けて扱えば，高効率
の並列計算が可能となり，大規模系への適用が期待されている．
