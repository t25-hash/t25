# 8・1・5 案内および整流装置

> **章**: 第8章 管路内の流れおよび流体中の物体に働く力
> **節**: 8・5 に示すようになる。表中では弁座口の直径、′は弁の上

a. 案内羽根入りベンドベンドに案内羽根を挿入して二次流れやはく離を抑制すれば、ベンドによる損失を著しく減少し、下流の速度分布を改善することができる。薄板を中心角90°の円弧に曲げた案内をベンドの中の同心に挿入する場合には、図8・29のように、案内羽根により分割される部分流路が同じ半径比になる位置ρに取付けると、損失が最小になる。ρ=1では1枚と2枚のいずれも80%程度にとどまり、ρがそれ以上になると損失係数の減少はなく、むしろ案内羽根の挿入により損失が増大する。長方形断面の90°ベンドに対する案内羽根挿入の効果についても、円形断面と同様のことがいえる。断面形状の縦横比の影響はあまりない。b. 翼列案内羽根入りエルボ風洞のコーナのように比較的寸法の大きいダクトでは、案内羽根として図8・30(a)のように翼列を用いると、損失はいっそう少なく、下流の速度分布に偏りのない一様に近い流れを得ることができる。案内羽根は、厚みのある翼型よりはむしろ、薄板を曲げて制作した翼型を多数配列することが推奨されている。図8・30(b)に風洞用翼列案内羽根の一例を示す。この薄翼は、コーナの対角線に沿って等間隔に配置され、前縁は流入方向に対し迎え角αが約4°~5°、後縁の流出角は0°になるように取付けられている。翼の間隔は0.28程度、アスペクト比は3以上、枚数は15枚ないし20枚以上とする。c. 金網均一な抵抗係数をもつ金網(wire gauze,screen)を流路の中に置くと、その下流の速度分布は一様になり、乱れが減少するので、風洞など気流を用いる実験装置の整流部に使用される。添字は、上流を1、下流を2とする。ただし、は式(8・64)による。円形断面の針金で正方形網目の織金網に対して、次の実験式がある。整流用の金網としては、(1)さびない材質で正方形網目の織金網(a) 管状式(b) 多孔板式図8・32 整流装置の例であること、(2)開口面積比βは下流に不安定な乱れの増加が起こらないようにβ>0.57であって、制作精度や目づまりを考慮した余裕をとること、(3)大きいの金網1枚より=1~2の金網を複数枚用いること、(4)金網の間隔は500以上にすること、(5)低乱流を得るにはダクト内で金網の継目がないこと、などの注意が必要である。各種の金網を組合わせた実験例がある。金網付きの広角度ディフューザ(wide-angle diffuser)では、金網は、はく離を抑え速度分布の偏りを改善する役割をもつ。d. 多孔板同じ直径の孔を一様に開けた多孔板(perforatedplate)は、金網に比べ、堅牢で使用中に目づまりが起こりにくいため、水流などの整流に使用される。多孔板による速度分布不均一の減衰効果は、多孔板の抵抗係数に依存する。多孔板は、単独ないしコークス層と組合わせて、せきによる流量測定のための整流装置としても使用される。e. 整流格子整流格子(honeycomb)は、薄板を格子状に組合わせて流路を多数のセル状に分割する、奥行の長い整流装置で、管路やダクトに挿入することによって、主流に垂直方向の速度成分を減少させ、上流側の旋回流やスケールの大きい乱れを除くために用いられる。風洞用整流格子としては、セルの寸法が流路幅に対し50分の1であること、およびセルの長さは流入角が10°より小さいとき、6~8ないしそれ以上とされている。セルの形状は整流効果にあまり影響しない。f. その他の整流装置図8・32の二例は、絞り機構による流量測定に対しASMEが推奨している整流装置である。図8・32(b)の多孔板式は、小さい孔を多数あけた多孔板を1の間隔に3枚組合わせたもので、角多孔板の開口面積比は少なくとも0.5とする。同様の構造がJIS にも述べられている。
