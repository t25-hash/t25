# 8・1・6 開きょの流れ

> **章**: 第8章 管路内の流れおよび流体中の物体に働く力

a. 等流における平均流速図8・33に示す断面形状およびこう配が一定の開きょ(open channel)内を水が定常状態で一様に流れる流れを等流(uniform flow)という。等流における平均流速(m/s)の代表的な公式には次のものがある。の値はガンギエ・クッタ(Ganguillet-Kutta)によれば粗度係数を用いて次の経験式で示される。の値はシェジーの式にも共通に使用され、表8・11にその範囲を示す。ただし、開きょ表面の仕上げの程度、水路の曲り状態、自然河川の状況などに応じて適正なの値を採用しなければならない。b. 流量が最大となる開きょの断面断面積が一定な開きょにおいて流量を最大にするには、平均流速の式からも明らかなようにを最大(ぬれ縁の長さを最小)にすればよい。c. 流速分布図8・35に開きょ断面における等速曲線の一例を示す。開きょ断面内の流速分布はかなり複雑であるが、横断面が左右対称でまっすぐな開きょでは、一般に開きょ中央の表面より少し下に最大流速の部分があり、壁面に近づくに従い流速が小さくな図8・33 等流表8・11マニングまたはガンギエ・クッタ公式の係数水路の種類の値閉管路黄銅管0.009~0.013鋳鉄管0.011~0.015純セメント平滑面0.010~0.013コンクリート管0.012~0.016人工水路滑らかな木材0.010~0.014コンクリート巻0.012~0.018粗石空積0.025~0.035土の開さく水路、直線状で等断面0.017~0.025自然河川線形、断面とも規則正しく、水深が大0.025~0.033同上で河床がれき(礫)、草岸のもの0.030~0.040蛇行していて、水深が小さいもの0.040~0.055図8・37 開きょに現れる水面形状4―79る。また、底面の境界層内では圧力が隅部で高く、中央部で低く、隅から中央に向かう二次流れが発生する。d. 比エネルギーと限界水深単位重量当たりの運動エネルギーと開きょ底を基準とした位置エネルギーの和を比エネルギー(全ヘッド)という。e. 背水曲線開きょのこう配が変化したり、せきや水門などがあると、水面のこう配が変化し、流れ方向に流れの状態が変わる(不等流)。流れの水面の形状を示す曲線を背水曲線(backwatercurves)と呼ぶ。開きょ幅が一定の長方形断面で、ゆるやかなこう配の場合(漸変不等流)、シェジーの式とベルヌーイの式から水深に関して次の関係式が得られる。背水曲線の計算には長方形断面の場合ではブレッス(Bresse)の公式、放物形断面ではトルクミット(Tolkmitt)の公式などに基づいたに対する背水関数表が便利であるが、最近では、任意断面形状に対して電子計算機を用いて差分法で厳密な曲線を求めることが多い。f. 跳水図8・37に示すように開きょの流れが射流から常流に遷移すると、水面付近に渦を伴った跳水(hydraulic jump)が発生し、自由表面は急激に変化する。水平水路において上流が=1~1.7の射流では跳水の下流の水面は渦の形成がない波状を呈するが、の増加とともに表面は不安定となる。上流が=4.5~9で表面渦とともに大きな水面上昇を伴う安定した跳水となり、下流側は比較的静穏な常流となる。上流が>9になると内部の激しい渦運動によって下流側は不安定に波立つ強い跳水となる。
