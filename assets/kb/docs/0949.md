# 8・2・1 揚力、抗力およびモーメント

> **章**: 第8章 管路内の流れおよび流体中の物体に働く力
> **節**: 8・2 流体中の物体に働く力

流体中の三次元物体には一般に揚力(lift)、抗力(drag)、横力(side force)、縦揺れモーメント(pitching moment)、偏揺れモーメント(yawing moment)および横揺れモーメント(rollingmoment)が作用する。それぞれの力、モーメントの方向は図が多いが、物体軸を中心に定義されることもある。それぞれの力、モーメントの係数は次式で定義される。ここでは物体の代表長さ、νは流体の動粘性係数である。が全抗力に占める割合は物体の形状、姿勢、表面粗さ(surface roughness)、レイノルズ数によって異なる。図8・39は球まわりの流れで、球前面にトリッピングワイヤ(tripping wire)を付けると、比較的低いレイノルズ数で球表面の境界層が層流から乱流に遷移(transition)し、乱流はく離(turbulent separation)に変わり、後流幅(wake width)が大きく縮小する。球表面が滑らかな場合には=3~4×10で乱流遷移(turbulent transition)が起こり、球まわりの圧力分布(pressure distribution, sphere)は図8・40のように変化する。円柱まわりの圧力分布(pressure distribution,circular cylinder)も図8・41のように変化する。このの値はトリッピングワイヤの有無や物体表面粗さ、主流の乱れ度(turbulent intensity)などにより異なる。
