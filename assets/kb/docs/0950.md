# 8・2・2 球に働く力

> **章**: 第8章 管路内の流れおよび流体中の物体に働く力
> **節**: 8・38 のように主流の方向を一主軸とする座標系で定義されること

球の抗力係数(drag coefficient, sphere)はレイノルズ数4―81図8・45 球の抗力係数の,による変化図8・46 球の抗力係数図8・47 円柱の抗力係数と円柱のストローハル数により図8・46のように変化する.低いレイノルズ数では解析的に求めたストークスの式(Stokes’formula),=24:(<1)やオゼーンの式(Oseen’s formula),=241+316:(<2)は実測値とよく一致している.中間領域の=30~300ではアレン(Allen)の実験式,=10で近似できる.亜臨界域の<4×10では=0.28+6+21の近似式,<10では=0.40の一定値,<3×10では=241+0.15+0.421+4.25×10の近似式がある.レイノルズ数욂욥=2.5×10~3.5×10で臨界レイノルズ数に達すると,球面上の境界層は層流から乱流に遷移し,その下流で境界層は乱流はく離して,は0.1以下に減少する.超臨界領域から極超臨界領域のに対しては次の近似式がある.=3.3~4×10では=29.78-5.3,=4×10~10では=0.1-0.49,<10では=0.19-8×10.ここで,=logである.
