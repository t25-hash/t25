# 6・1・6 回転円板の破壊（fracture of rotating disk)

> **章**: 第6章 圧力・遠心力などを受ける円板，円筒および球
> **節**: 6・1 円板

a.遠心破壊 普通の遠心破壊は，高い応力集中部をもたない
円板の延性破壊と，材料欠陥・き裂・鋭い切欠などをもつ円板の脆
性破壊に分けて考えることができる．
高い応力集中部をもたない円板の遠心破壊の強度 を簡単に推
定するにはσ＝σと考えればよい（平均応力説，average
stress theory）．ここで，σは円周方向弾性応力の最弱直径断
面上の平均値，σは円板材料の引張強さである．材料の延性が増
すと実際の破壊強度は平均応力説による計算値よりも低くなるが，
これはσの代わりに，内・外周自由な回転円輪の変形が不安定に
なる応力σ′を採用することによって修正される（修正平均応力
説）．公称応力-ひずみ曲線にひずみ軸上の－1の点から接線を引い
たとき，応力軸を切る点の応力値がσ′を与える．なお，材料の延
性が大きい場合は，塑性変形の計算法を応用して変形安定限界強度
を求めることによっても破壊強度を推定できる．
繊維強化複合材料の円板についてはミーゼス則，ホフマン則など
が破壊をよく予測するという研究がある ．
材料欠陥・き裂・鋭い切欠などをもつ低靱性材で製作された円板
は脆性破壊しやすく，平均応力説による推定値よりも低い回転数で
破壊する ．回転体の脆性破壊強度の推定には線形破壊力学を
応用して，＝の破壊条件を用いる．ここで， および

数（stress intensity factor ) および運転温度における円板材
料の破壊靱性（fracture toughness ）を表す．

き裂などの欠陥が存在しない円板の低サイクル疲労寿命は，回転
円板内の危険箇所（応力が最大になる位置）の発停に伴って繰返さ
れる応力変化，平均応力，塑性ひずみ幅および平均ひずみなどを計
算し，これらの値を使って円板材料の寿命特性曲線から推定す
る ．その際，負荷の変動波形や高温での保持時間の影響も考慮
する ．
c .クリープ破壊 回転円板のクリープ破壊（creep rupture of
rotating disk）寿命は，高応力のときはクリープ変形に基づく真
応力の増大あるいは幾何学的不安定の発生の面から ，低応力の
場合は微視的き裂の発生などに基づく材料組織の劣化の面から
推定が企てられている．高温において，クリープと疲労が重畳する
場合の回転体の寿命を推定する試みも見られる ．
