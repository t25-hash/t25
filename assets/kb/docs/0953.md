# 8・2・5 壁面の影響

> **章**: 第8章 管路内の流れおよび流体中の物体に働く力
> **節**: 8・57 に示す。

a. 閉塞比の影響物体まわりの流れは流路壁の影響、閉塞比(blockage ratio)、流れと垂直な断面への物体の投影面積と流路面積の比の影響を受ける。辺長比0.5~5.0の二次元長方形柱に対する補正式がある。図8・61(a)(b)は、抗力係数と背圧係数が一定の関係にあることを正三角柱と円柱に対して示している。二次元物体、円柱、三角柱、直交平板への修正近寄り速度を導入して、閉塞比のある抗力係数を精度良く見積もることができる。b. 平面壁の影響平面壁近くの物体に働く力は壁からの距離によって変化する。図8・62は壁近くにおかれた円柱まわりの流れである。図8・62(a)はすきまがない場合である。平面上で流れははく離し、円柱からのはく離せん断層は平板に再付着する。円柱の前面と背面に停留渦が形成される。図8・62(b)はすきまから噴流が吹出す場合でやはり不規則な後流を形成する。図8・62(c)は円柱の両側からはく離せん断層が形成され周期的な渦放出が行われる。さらに臨界すきま以下で揚力は急増する。この傾向は境界層厚さδが小さいほど顕著である。
