# 9・1・1 一次元解析

> **章**: 第9章 流体機械の流れ
> **節**: 9・1 ターボ機械の流動解析法

ターボ機械は回転する羽根車の作用により流体との間でエネルギーの授受を行う流体機械であり、羽根車には翼(動翼)が周方向に等間隔で配置される。その羽根車の子午面形状により、ターボ機械は軸流型、斜流型および遠心型に分類される(図9・1)。いずれの形式のターボ機械でもその内部流れは極めて複雑であり、高性能なターボ機械の設計には三次元流れ解析が不可欠であるが、まず流れ場の諸量が流れ方向のみに変化する一次元流れを仮定して、ターボ機械の作動原理について述べる。図9・2のような任意の羽根車の上流および下流に軸対称な環状の横断面を設定し、その断面を通過する一次元流れを絶対(静止)座標系から考える。図9・2に示すとおり、断面に垂直な速度成分を、断面内の速度成分のうち子午面に垂直な成分(周方向速度成分)をとする。一次元流れを仮定していることから、断面内の速度成分のうち子午面内成分を無視する。なお、羽根車の上流および下流の断面における値にそれぞれ添字1および2を付けて表す。この速度三角形の幾何学的な関係から式(9・3)が得られる。ここで、:羽根車を通過する流体の質量流量、ρ:流体の密度である。次に、上述の検査体積に対して、羽根車の回転軸まわりの角運動量保存則を適用すると式(9・5)を得る。被動機(ポンプや圧縮機)では>0、原動機(水車やタービン)では<0となる。式(9・6)の関係は角運動量保存則より導かれたものであるから、流れの圧縮性や損失にかかわりなく成立する。被動機では、羽根車にだけの動力を外部から加えることによって、羽根車が流体に旋回を与え、その結果として流体のもつエネルギーが増加する。一方原動機では、流体が羽根車を通過する際に流体のもつ旋回が減少することによって、だけの動力が羽根車から取出され、その結果として流体のもつエネルギーが減少する。は単位時間に羽根車になされる単位質量流量当たりの仕事であり、流れに損失がない場合には、この仕事のすべてが単位質量当たりの流体における単位時間当たりの全エネルギー変化量となる。はオイラーヘッド(Euler’s head)とも呼ばれ図9・2 羽根車を通過する一次元流れ図9・3 速度三角形る。速度三角形から得られた式(9・3)の関係を式(9・8)に適用すると、式(9・10)が導かれる。理論ヘッドから運動エネルギーの変化によるヘッド変化量を差引いた量を理論静ヘッドという。理論静ヘッドが理論ヘッドに占める割合は反動度(degree of reaction)Λと呼ばれ、式(9・11)で表される。したがって、羽根車の流体効率ηは式(9・14)(9・15)で定義される。しかしながら、実際には流れの諸量が流路断面にわたって一様でないことから、各種保存則の適用において厳密には断面内の流れ分布を積分して評価する必要がある。したがって、各断面での代表値で羽根車の流れを取扱う上述の一次元解析は、以下に述べる経験的な修正係数の導入によって初めて実用に供される。角運動量保存則に基づいて導出された式(9・8)は式(9・16)のとおり作動係数(ワークダンファクタ、work-done factor)を導入して修正される。
