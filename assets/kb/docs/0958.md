# 9・1・2 二次元解析

> **章**: 第9章 流体機械の流れ
> **節**: 9・1 ターボ機械の流動解析法

ターボ機械内部の三次元流れを2種類の二次元問題に帰着させてそれらを組合わせることにより、近似的に流れ場を解析する準三次元理論(quasi-three-dimensional theory)がある。このような取扱いは、三次元解析と比べて計算量を低減できることから、設計問題に適している。準三次元理論では、図9・4に示すとおり、ハブとケーシング間に複数の流面(S面)、翼間にも複数の流面(S面)をとって、それぞれの流面上で二次元流れ問題を解き、これらを組合わせることにより三次元流れが近似的に求められる。S面およびS面上の流れをそれぞれ、翼間流れ(blade-to-bladeflow)および通り抜け流れ(through flow)と呼ぶ。一般には、通り抜け流れは一つの代表S面(図9・4中のS욇)上でのみ解析され、図9・5に示されるとおり、その代表S面上の流れは機械軸まわりに回転し子午面(面)上に投影して一意に表現できることから、代表S面上の通り抜け流れは子午面流れ(meridional4―9 5図9・4 準三次元理論図9・5 子午面流れflow)とも呼ばれる。S面上の流れを子午面に投影して得られる流線(図9・5中の線)を子午面流線(meridional streamline)と呼ぶ。a. 子午面流れ解析子午面流れの定式化の方法は次の三つに大別される。(i) 代表S面上の流れを取扱う方法(ii) 軸対称流れを仮定する方法(iii) 基礎方程式を周方向に厳密に平均化する方法方法(i)では、翼間に代表S面を設定し、その面上に通り抜け流れを規定する。すなわち、S面の形状が与えられると、周方向座標は従属変数となり、S面上の流れは二次元問題に帰着される。二次元問題に帰着される過程で、S面に垂直に働く見掛けの体積力に相当する項が運動方程式に表れる。この項は、あたかも流れをS面上に強制的に規定するために必要となる体積力とみなせ、翼力(blade force)と呼ばれる。流体力学的には、この翼力は翼両面の圧力差に基づく羽根作用に相当する。代表S面として、翼の反り線、S面上の質量流量を二等分する流線、S面上の流れを質量流量平均して得られる流線などから構成される面が採用される。方法(ii)においては、軸対称流れを仮定して子午面上で二次元流れ問題を解析するが、以下のような考察のもとに羽根作用をモデル化する。羽根車と流体との間でエネルギーの授受がある流体機械では、流体が羽根車を通過する間に翼両面の圧力差によってその角運動量が変化する[式(9・7)].翼枚数が無限となり軸対称流れとなると、1枚の翼から受ける力が零に近づくが、同時に翼間の流量も零に近づき、両者の比は一定値に漸近して羽根作用は分布体積力で置換えられる。さらに非粘性流れを仮定すると、翼面に平行な力の成分は存在しないことから、この分布体積力は翼面に垂直な方向に働く。このような考察から、この方法では翼枚数が無限で流れ面が翼面に一致する非粘性軸対称流れを仮定して、基礎式中に含まれる周方向の偏微分項をすべて無視するとともに、羽根作用を表すために翼面に垂直に働く体積力が基礎式に付加される。方法(iii)では、流れの基礎方程式を翼間1ピッチにわたって周方向に厳密に平均化することによって、通り抜け流れを二次元問題に帰着させる。この方法では、S面の設定や翼面が代表流面であると仮定する必要がなく、方法(i)および(ii)よりも合理的に羽根作4―9 6図9・6 翼間流れ用を考慮できるとともに、翼両面の圧力差に基づく翼力だけでなく、翼枚数が有限であることに起因する流れの非軸対称性も考慮することができる。周方向の平均操作として単純平均および質量平均などが用いられるが、翼のブロッケージ効果の取扱いという観点からは質量平均が合理的である。b. 翼間流れ解析図9・4に示したとおり、ハブとケーシング間に位置する複数のS面上で翼間流れ(blade-to-blade flow)が解析される。S面は流れの三次元性からねじれているが、図9・6に示すとおり、一般に回転面と仮定される。すなわち、子午面流れ解析から求まった子午面流線(図9・5中の線)を機械軸まわりに回転して得られる回転面(回転流面)をS面としてハブからケーシングまで複数設定し、各回転流面上で翼間流れを解析する。S面とは異なって、S面は複数設定されることから、その設定に関するモデル化に任意性は少ない。翼間流れ解析において非粘性流れを仮定する場合、絶対流れが渦なしである条件から基礎方程式が定式化される(9・3・1項参照)。代表S面上の通り抜け流れとは異なり、翼間流れを粘性流れとして取扱って各S面上でナビエ・ストークス方程式を解くことも比較的容易である。
