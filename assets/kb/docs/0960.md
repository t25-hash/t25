# 9・2・1 直線翼列(linear cascade)

> **章**: 第9章 流体機械の流れ
> **節**: 9・2 軸流機械

図9・1に示したように軸流機械では羽根車の入口と出口の半径差が小さいことから、機械軸と同心の円筒面に沿って流体が流れると仮定する。この円筒面上の流れを平面に展開すると、図9・8に示すとおり、無数の翼が直線に並んだ直線翼列まわりの流れが得られる。一つの円筒面上に現れる翼断面は翼素と呼ばれ、この翼素を積重ねることで三次元の翼が形成される。以上のパラメータを図9・8に示す。式(9・24)(9・25)から、軸流機械では理論静ヘッドが相対速度エネルギーの変化のみによって決まることがわかる。前者はポンプや圧縮機などのように流体の圧力を上昇させるターボ機械に、後者は水車やタービンなどのように圧力を降下させるターボ機械に用いられる。増速翼列では、増速流れとなることから、壁面境界層のはく離が抑制され、減速翼列よりも大きな転向角(翼負荷)をとることができる。羽根車内で流体が円筒面に沿って流れると仮定すると、9・1・3項で述べた相対座標系での運動方程式(9・19)において、左辺第4項のコリオリ力および左辺第5項の遠心力はいずれも円筒面内成分をもたない。このことは、同形状の翼素をもつ、回転する直線翼列と静止した直線翼列を考えた場合、前者まわりの相対流れと後者まわりの絶対流れとが全く同じ基礎方程式で支配されていることを意味する。すなわち、直線翼列では回転の効果が流れ場に現れず、静止した直線翼列の絶対流れと回転する直線翼列の相対流れは同一となる。以上のことに基づいて、風洞試験等で静止直線翼列の流体力学4―9 7的性能が既知であれば、それを用いて軸流機械の羽根車の翼素を設計することができる。直線減速翼列を例にとって、翼列の上流・下流での流れと翼列の性能との関係について以下に述べる。直線翼列まわりの非圧縮性の相対流れを考えると、連続の式から翼列前後で軸流速度成分は等しく、翼列前後での速度三角形は図9・9のようになる。この損失が発生しない場合、翼素に作用する抗力が零となり、このとき翼素に作用する揚力は式(9・28)で定義されるベクトル平均流れ角方向と垂直に働く。なお、式(9・11)で定義される反動度Λは直線翼列に対して以下のように表せる。
