# 9・2・1 項で述べた直線翼列に基づいて羽根車流れを取扱うこと

> **章**: 第9章 流体機械の流れ
> **節**: 9・2 軸流機械

は、9・1・2項の準三次元理論において、円筒流面に沿った翼間流れを仮定することに相当し、ハブ面とケーシング面との間に複数の円筒流面を設定することによって羽根車の全体性能が見積もられる。この各円筒流面上の翼間流れは独立に成立することはなく、子午面図9・8 直線翼列図9・9直線翼列前後の非圧縮性流れの速度三角形4―9 8流体工学流れ(通り抜け流れ)における力学的な平衡条件を介してお互いに関係し合っている。このことは、ある円筒流面上における翼列前後の速度三角形(図9・9)が、近接した別の円筒流面上の流れも考慮して決定されなければならないことを意味する。この円筒流面間の相互関係(平衡条件)は9・1・2項で述べた子午面流れ解析から得られる。以下では、最も簡略化された平衡条件を示すとともに、その条件を満足するフローパターン、すなわち旋回速度成分の翼スパン方向分布(渦形式)および軸方向速度分布について述べる。子午面流れを支配する基礎方程式を最も簡略化して得られる平衡条件は、単純半径平衡条件と呼ばれる。羽根車前後の翼が存在しないダクト領域の流れのみに着目して、非圧縮性・非粘性流体が軸対称でかつ円筒面に沿って流れると仮定する。このとき、半径方向の運動方程式から式(9・32)が得られる。式(9・33)において、は積分定数であり、次の連続の条件から決められる。式(9・33)からわかるとおり、羽根車前後での軸方向速度分布は、絶対流れの旋回速度成分の半径方向分布に依存している。このことは、ターボ機械の設計において、羽根車前後における各速度成分の半径方向分布は任意に与えることができず、半径平衡条件を考慮して与えなければならないことを意味する。旋回速度成分の半径方向分布は渦形式と呼ばれ、式(9・24)および(9・31)からわかるとおり、この渦形式は羽根車の理論ヘッドや反動度Λの半径方向分布を規定する重要な設計パラメータである。以上の観点から、次におもな渦形式を示す。羽根車の入口および出口での渦形式を式(9・35)で与える。この渦形式を自由渦形式という。さらに、羽根車の入口で全圧分布が一様であり、羽根車内での全圧損失の半径方向こう配が無視できる場合、羽根車の出口での全圧分布も一様になる。このとき、式(9・33)から羽根車前後での軸方向速度分布も式(9・38)のとおり一様となる。したがって、複数の羽根車が軸方向に配置された多段軸流機械の場合、半径方向の全圧こう配が下流で極めて大きくなり、混合損失が増えることから、強制渦形式は多段型であまり採用されない。≠-1の場合、この渦形式は半強制渦形式と呼ばれ、多段軸流機械に採用されることが多い。定反動度形式では、理論ヘッド、反動度、軸方向速度分布は次のように表せる。
