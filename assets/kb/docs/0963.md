# 9・2・3 二次流れ(secondary flow)

> **章**: 第9章 流体機械の流れ
> **節**: 9・2 軸流機械

ターボ機械における代表的構成要素である羽根車内部には、複雑な三次元流れが形成される。図9・10は軸流圧縮機の羽根車(動翼列)内の三次元流れ構造を示す。このように複雑な流れ場の場合、三次元流れを主流(基本流れ)と主流に垂直な断面内の流れ(二次流れ)とに分離して取扱うと、流動現象の把握が比較的容易となる。図9・10を例にとって、二次流れを引き起こす代表的な流れ挙動を以下に示す。翼列の上流において、ハブ面およびケーシング面上には境界層が発達しており、それらが翼列に流入する際に翼前縁で巻上がって馬てい形渦が形成され、翼列内を転向する際に流路渦へと成長する。翼先端がシュラウドで覆われていない翼列の場合には、ケーシング側よりもハブ側で馬てい形渦と流路渦による二次流れが顕在化する。また、この二次流れは減速翼列よりも転向角が大きな増速翼列で顕著に現れる。流路渦から引き起こされる二次流れは、曲り流路における二次流れ(8・1・8項)と同じ発生メカニズムをもち、図9・10中に示すとおり、ハブ面境界層では翼背面(翼負圧面)に向かう二次流れが起きる。翼面上に発達する境界層内でも二次流れが発生するが、動翼列と静翼列によって二次流れの向きが逆転する。まず動翼列では、図速度よりも小さく、速度三角形の関係から、境界層内の絶対速度の旋回成分′が主流よりも大きくなる。したがって、動翼の境界層内では、遠心力ρ′が半径方向圧力こう配よりも大きくなっており、その結果、図9・10に示すとおり境界層内では半径外向きの二次流れが発生する。次に静翼列の翼面境界層内においては、図9・11に示すように、絶対速度自身′が主流の絶対速度よりも小さいことから、旋回に伴う遠心力が主流よりも小さくなっている。その結果、静翼の境界層内では半径内向きの二次流れが起きる。以上からわかるとおり、翼面境界層内の低エネルギー流体は、動翼列で翼先端側に、静翼列で翼根元側に集積する。動翼列の場合、ハブ面と翼背面とのコーナ部で、前述した流路渦によるハブ面上の二次流れと遠心力による翼背面上の半径外向きの二次流れとの相互作用によって、局所的はく離(コーナはく離)が現れることがある。また、シュラウドのない翼列では、図9・10中に示すとおり、翼先端とケーシング面とのすきま(翼先端すきま)から翼両面の圧力差により漏れ流れが生じる。この翼先端漏れ流れは翼間内で巻上がって、漏れ渦が形成される。減速翼列では、増速翼列と比べて転向角を大きくとれない結果、前述した流路渦による二次流れが弱く、漏れ渦による二次流れが最も支配的である。さらに、シュラウドのない翼列では、動翼とケーシング面との間に相対運動がある結果、動翼がケーシング面境界層をかき上げて、翼先端とケーシング面とのコーナ部にかき上げ渦が形成される。
