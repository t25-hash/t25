# 9・3・1 円形翼列、円形翼列への写像

> **章**: 第9章 流体機械の流れ
> **節**: 9・3 遠心機械・斜流機械

羽根車内に現れる相対流線は圧力こう配との釣合いにより翼先端側や根元側に偏るなど、個々に異なる。実際の、このようなゆがんだ流れを取扱うには計算が面倒となり、羽根車を通過する流れの本質を知る方法としては適切ではない。そこで、羽根車内の三次元流れを子午面流れと翼間流れに分けて考えることが実用的である。これを準三次元流れ(quasi-three-dimensional flow)という(9・1節参照、図9・12)。この流面に対して翼間流れを考察する。この回転流面上で直接、流れを考えるには計算が煩雑となるため、一般に、円形翼列に写像して解析されることが多い。なお、羽根車の回転角速度ωは両面で同じである(図9・13)。式(9・44)は、ρ=一定の場合、ポアソン方程式となる。さらに、回転軸に平行な流れsinφ=0の場合、右辺は零となり、ラプラス方程式となる。軸に平行な流れでは、静止翼列における絶対流れにおいても同じラプラス方程式(式の誘導は省略)が得られる。このことは、羽根車(直線翼列)が回転している場合の非粘性相対流れは、静止翼列の場合の絶対流れと等価であり、回転翼列の翼素選定に風洞試験資料が利用できることを示している。一方、角速度ωで回転する円形翼列φ=90°内の-Θ面の相対流れ場には回転方向に-2ωの渦が分布しており、ω=0の静止円形翼列内の絶対流れとは異なる。したがってこの場合、静止翼列の風洞試験資料を用いることはできず、その翼素選定資料は、回転翼列を用いた実際の実験か流れ計算に頼らざるを得ない。すなわち図9・14において、静止円形翼列では(a)通り抜け流れと(b)循環流れを重ね合わせた流れとなるのに対し、回転円形翼列では(a)と(b)に加えて、4―100流体工学-2ωの渦分布が作り出す(c)押しのけ流れを合成した流れとなる。回転円形翼列の非粘性流れ解析には、差分法のほか境界要素法が多用されている。流れ解析から相対速度場(速度)が得られると、相対流線に対して、次に示す回転系のベルヌーイの式から圧力場(圧力)が算出される。
