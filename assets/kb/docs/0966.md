# 9・3・3 インデューサ、エクスデューサ

> **章**: 第9章 流体機械の流れ
> **節**: 9・3 遠心機械・斜流機械

遠心型のポンプや圧縮機の入口では軸方向から半径方向に急に流れの向きを変えるので、シュラウド側の急激な増減速域において流れのはく離・逆流を引き起こし、それに伴う損失をもたらす。図化(横軸φは流量を、縦軸ψは揚程を無次元化して表示)を示す。図9・16から、羽根車入口を軸流流入部まで延長することにより、低流量域での揚程改善が認められ、入口での滑らかな流入には羽根の上流側への延長が有効なことが知られる。遠心圧縮機では羽根の上流側への延長が積極的に採用され、この延長部分をインデューサ(inducer)と呼んでいる。液体を取扱うポンプや水車における最近の高速小型化に伴い、ポンプでは羽根車入口圧、水車では羽根車出口圧の低下が著しく、羽根車入口(ポンプ)や出口(水車)部に生じるキャビテーション図9・15 遠心羽根車出口での速度三角形と滑り速度(cavitation、局部的に圧力がその液体の飽和蒸気圧以下となり、液体から気体へと相変化を起こす現象。12章参照)が問題となっている。キャビテーションの発生は、流体機械の性能を低下させるだけではなく、キャビテーション消滅時に生じる強い衝撃圧が羽根車材料の疲労破壊・損傷を引き起こしたり、キャビテーションの不安定性が振動や騒音を招くなど、さまざまなトラブルの原因となる。円形翼列(遠心羽根車)のポンプ入口部・水車出口部はその半径の減少とともに流路面積が狭くなるため、キャビテーションによる流路閉塞が軸流型よりも著しい。そこで最近では、遠心機械の機能として不可欠な主羽根車(遠心羽根車)においてキャビテーションの発生を回避し、ポンプではその上流にインデューサと呼ばれる、水車ではその下流にエクスデューサ(exducer)と呼ばれる軸流型または斜流型の羽根車を付設して、キャビテーションの発生を軸流型羽根車内にとどめる方策が採用されている(図9・17)。
