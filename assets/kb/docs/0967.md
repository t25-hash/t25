# 9・3・4 二次流れ

> **章**: 第9章 流体機械の流れ
> **節**: 9・16 は、羽根車入口形状を変えたときの渦巻ポンプの特性曲線変

遠心羽根車には、軸流型や斜流型と同様に、翼端とケーシング静止壁との間にすきまをもつ開放型[これを半開放型(semi-open)といい、翼根元にも一部、すきまをつける場合を開放型(open)ということもある]と流路全域が回転壁に囲まれた密閉型(closed)とがある。その回転流路内部の相対流れは、静止流路と同様に、軸(半径)方向から半径(軸)方向へと流れの向きを変える板面の曲りや、羽根のわん曲の影響を受けるとともに、羽根車の翼作用に加えて、遠心力やコリオリ力の影響[式(9・46)参照]を受けるため、さまざまな方向に非一様な圧力分布が生じる。この圧力分布は、壁面上に成長する境界層の外部にある主流域の相対流れ(非粘性流体の流れとみなしうる)が作り出す。このとき、主流方向とは垂直に流速分布が存在すると、その垂直方向には流速分布に応じた圧力こう配が生じる。主流方向に垂直な流れを二次流れ(secondaryflow)と呼ぶ。流れが軸方向から流入し半径方向に流出するポンプ作用をもつ遠心羽根車流路について考えてみる(図9・18)。流れが軸方向を向いている羽根車入口部では、軸流羽根車と同様に、ハブ面およびケーシング面に、羽根間で高圧となる羽根圧力面から低圧となる負圧面への二次流れが生じるとともに、半径方向には、おもに次式(9・58・a)に従って圧力こう配が存在するため、相対速度の周方向成分が<となる羽根面上の境界層内の流体には、半径方向外向きの二次流れが生じる。一般には、羽根が回転方向に凸状の後向き羽根をもつ羽根車においても、式(9・58・c)において右辺第1項(コリオリ力の項)が第2項(流線曲率の項)より強く現れるため、板面境界層の流体は回転の方向、すなわち羽根圧力面から負圧面側へと流される[図9・18(d)および(e)].その結果、羽根車出口では羽根負圧面側に低エネルギー流体が集積し、出口下流で後流域が形成される[図9・18(e)].図9・19には遠心羽根車の板面上に現れた限界流線(板面上せん断応力の方向)の可視化写真を示す。図9・19から、羽根圧力面側の板面境界層内の流体が遠心力とコリオリ力の影響を受けて負圧面側へと流され、負圧面上出口近傍ではく離域に大きな低エネルギー流体が蓄積されている様子がわかる。
