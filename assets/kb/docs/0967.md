# 9・3・4 二次流れ

> **章**: 第9章 流体機械の流れ
> **節**: 9・16 は,羽根車入口形状を変えたときの渦巻ポンプの特性曲線変

遠心羽根車には,軸流型や斜流型と同様に,翼端とケーシング静止壁との間にすきまをもつ開放型[これを半開放型(semi-open)といい,翼根元にも一部,すきまをつける場合を開放型(open)ということもある]と流路全域が回転壁に囲まれた密閉型(closed)とがある.その回転流路内部の相対流れは,静止流路と同様に,軸(半径)方向から半径(軸)方向へと流れの向きを変える板面の曲りや,羽根のわん曲の影響を受けるとともに,羽根車の翼作用に加えて,遠心力やコリオリ力の影響[式(9・46)参照]を受けるため,さまざまな方向に非一様な圧力分布が生じる.この圧力分布は,壁面上に成長する境界層の外部にある主流域の相対流れ(非粘性流体の流れとみなしうる)が作り出す.このとき,主流方向とは垂直に流速分布が存在すると,その垂直方向には流速分布に応じた圧力こう配が生じる.圧力こう配は壁面境界層内の流体にもそのまま作用するが,境界層内流体の速度は主流速度より遅い<ため,流体粒子は圧力こう配には抗しきれずに主流に垂直な方向の流れを誘発してねじれ境界層(screwed boundarylayer)となるとともに,流路断面の一部に低エネルギー流体の蓄積をもたらす.主流方向に垂直な流れを二次流れ(secondaryflow)と呼ぶ.流れが軸方向から流入し半径方向に流出するポンプ作用をもつ遠心羽根車流路について考えてみる(図9・18).流れが軸方向を向いている羽根車入口部では,軸流羽根車と同様に,ハブ面およびケーシング面に,羽根間で高圧となる羽根圧力面から低圧となる負圧面への二次流れが生じるとともに,半径方向には,おもに次式(9・58・a)に従って圧力こう配が存在するため,相対速度の周方向成分が<となる羽根面上の境界層内の流体には,半径方向外向きの二次流れが生じる.ddρ+ω(9・58・a)また羽根先端すきまがある場合には,羽根圧力面から負圧面へのすきま流れが流路内に流入する[図9・18(b)].次に流れが軸方向図9・17 主羽根車とインデューサ,エクスデューサとの組合せ(c) 軸方向から半径方向への曲り(断面II)(b) 軸方向流入部(断面I)(a) 遠心羽根車子午断面4―102流体工学図9・19遠心ポンプ羽根車板面上限界流線の可視化写真から半径方向に向きを変えるところでは,ハブおよびケーシング面での二次流れと羽根先端漏れ流れに加えて,子午面流線が曲率半径욇で半径方向に曲がることにより,流線に垂直な욇方向の圧力こう配は子午面流速욇に対して式(9・58・b)が支配的となり,욇<욇となる羽根面上の境界層内の流体には羽根先端側への二次流れが生じる[図9・18(c)].dd욇-ρ욇욇(9・58・b)流れが半径方向を向くと,羽根間に現れる流線に垂直な方向욦には,流速に対して2ωで表されるコリオリ力と流線の曲り(曲率半径욦)による圧力こう配が現れる[式(9・58・c)].dd-2ρω+ρ(9・58・c)板面上境界層内の流体の流速は<であるため,圧力こう配の方向に二次流れが生じる.一般には,羽根が回転方向に凸状の後向き羽根をもつ羽根車においても,式(9・58・c)において右辺第1項(コリオリ力の項)が第2項(流線曲率の項)より強く現れるため,板面境界層の流体は回転の方向,すなわち羽根圧力面から負圧面側へと流される[図9・18(d)および(e)].その結果,羽根車出口では羽根負圧面側に低エネルギー流体が集積し,出口下流で後流域が形成される[図9・18(e)].図9・19には遠心羽根車の板面上に現れた限界流線(板面上せん断応力の方向)の可視化写真を示す.図9・19から,羽根圧力面側の板面境界層内の流体が遠心力とコリオリ力の影響を受けて負圧面側へと流され,負圧面上出口近傍ではく離域に大きな低エネルギー流体が蓄積されている様子がわかる.
