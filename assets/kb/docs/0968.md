# 9・4・1 円すいディフューザ、二次元ディフューザ

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ、ノズルなどの内部流れ

遠心圧縮機のような被動機において、流体に伝えられる力学的エネルギーの一部は運動エネルギーの増加で、高流速のままでは損失が増えるため、流れをディフューザ(diffuser)に導いて減速させ圧力回復を図る方式がとられる。その通路形状は多種多様であるから、その設計に際しては、管路要素としても基本的な形状である円すいディフューザ(conical diffuser)や直線壁二次元ディフューザ(two dimensional diffuser)で得られた知見が利用される。いずれにおいても、圧力回復量にもっぱら関心が寄せられるので、ディフューザ(出入口面積比)の性能は式(9・59)(9・60)の圧力回復係数(pressure recovery coefficient)욣や有効度(effective-ness)η욃で評価される。出入口間の圧力上昇量を、入口流速をで表すと욣=ρ2(9・59)η욃=욣욣욡(9・60)ここで、욣욡:理想的圧力回復係数[=1-1].上述のディフューザでは幾何形状を表す広がり角2φ、長さ比または(:ディフューザ長さ、:入口半径、:入口幅)、面積比のうちの二つが独立変数となるから、それらを使って性能試験結果が多数の線図として整理されている。その利用に当たり、有効な減速量はディフューザ内の乱流境界層の成長により定められるので、圧力回復量が幾何形状だけでなく入口境界層(δ:排除厚さ)に支配されることを考慮する必要がある。図9・20 point)の形状係数욦に対して式(9・61)の判別基準が提案されている。욦=1.8+7.5δ(9・61)ここで、δ:排除厚さ、:その位置の通路幅(円すいの場合は管半径)で、両者の比は干渉効果を表す安定性パラメータに相当する。ターボ機械で使われる長方形断面の二次元ディフューザは、アスペクト比が通常小さいので流れの三次元性が強い。その影響に留意して、従来の研究結果を参考にする必要がある。ところで、流れの向きを変える必要があるときは管軸を曲げることになる。その措置で壁に沿う主流方向圧力分布に負のこう配を経て正のこう配に転じる領域が現れるため、過大な圧力上昇になって、はく離が発生しやすくなる。また、流線が曲率をもつと境界層厚さ方向への圧力変化も無視できなくなり、乱流状態に違いが現れる。凸壁面では乱流の混合作用が抑制されるので、圧力上昇に弱い性質が見られる。形状選定の際の指針としては、可能な限り曲率半径を大きくとることになる。円すいディフューザは出口損失を減らすためにターボ機械の出口部に設けられることも多い。設計点では無旋回としても作動点が変わると旋回を有する流れになるが、その場合も上述の好適形状は良好なものとみなしうる。ただし、流れに旋回がわずかに付いている場合には圧力回復性能は改善されるが、旋回が強い場合は中心軸付近に逆流が現れて性能低下を生む。旋回流(swirl flow)を表す条件として、その強さが主パラメータとみなされており、次の運動量パラメータが実験結果の整理に使用されている。=욊욀d욀d(9・62)ここで、욊:周方向分速度、욀:軸方向分速度、:半径位置、:管半径である。ランキン渦的な速度分布の場合、の値は壁面位置における流れ角の正接に対応する。例えば=1は流れ角45°程度になる。旋回速度成分を有効に減らして圧力回収するためには流路中に翼を入れるか、出口半径を大きくするしかないので、強い旋回を扱うのは性能の面で厄介である。
