# 6・5・4 燃焼生成物

> **章**: 第6章 熱物性値
> **節**: 6・5 燃料・燃焼生成物の熱物性値

燃焼生成物（combustion product ）はその組成が，燃料の種類，
温度，圧力や空気過剰率など諸条件によって大幅に異なるので，厳
密には個々の場合について計算する必要がある．物性値表 や
計算のための資料集 ，代表的な燃料についての計算例やコンピ
ュータプログラムなどの文献調査報告がある ので，参照され
たい．
燃焼ガスの線図としては，温度-エンタルピー線図 ，温度-
エントロピー線図 ，エンタルピー-エントロピー線図 な
どがある．
