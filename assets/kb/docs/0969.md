# 9・4・2 ベーンレスディフューザ、ベーンディフューザ

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ、ノズルなどの内部流れ

diffuser)の基本形状は平行壁の半径方向軸対称流路である。これに旋回流が外向きに入る場合、壁摩擦が無視されるならば、速度の半径方向成分욥および周方向成分욊は連続の式と角運動量収支式に従い半径に反比例していずれも減速するので、流れは流れ角α=tan욥욊=一定の対数らせんを描く。そのような理想流れの場合の圧力回復係数욣は式(9・63)で表される。出入口半径比をとすれば욣=1-1(9・63)このようにベーンレスディフューザは、限度があるものの、出口半径を大きくするだけで旋回速度を有効に減速して動圧を静圧に回収できる流路であるために、反動度が小さい遠心機械では圧力回復要素として羽根車の外側に設置される。現実の羽根車を出る相対流は二次流れや羽根の後流によって周方向にひずんでいるために、その絶対流速は大きさ、方向ともに変動するので、大きな損失がディフューザ入口領域で生じる。なお、後流は羽根車外半径の約1.05~1.25倍下流のディフューザ内位置に至ると羽根車回転の影響によりほとんど消滅してしまい、流れはほぼ軸対称となる。一方、幅方向の速度分布のひずみは成長するために、その影響でディフューザに入る流れの周方向からの角度が小さい場合には内向き流れが壁近傍にやがて発生して、条件によっては旋回失速(rotating stall)と呼ばれる不安定な流れを誘発する。不安定限界の流れ角α욥と通路幅との関係を図9・21に示す。安定性能を高めて運転範囲を広げるために、流れ角が小さくならないようにディフューザハブ側壁面の一部を粗くすることや、浅い溝を壁に設けることで失速を抑制する方法が提案されている。ディフューザの平行壁を傾斜壁に変えることも有効である。流量が低下すると、羽根車からディフューザに入る絶対流れは角度を減らし、一定の半径位置に至るまでに長い距離を要するので、より大きな壁面摩擦の影響を受ける。したがって、ベーンレスディフューザの圧力-流量曲線は右上がり特性を有するといえるため、羽根車も同様な特性をもつ場合にはサージング(surging)に陥りやすい。流出角が小さいと予測される場合には羽根(翼とも呼ばれる)をディフューザ部に設置して適切に減速しながら必要半径位置に流れを導く、ベーンディフューザ(vaned diffuser)が広く使用される。積極的に流れを導くには、羽根前縁部を羽根車に近づけることが望まれるが、羽根車からの非一様な流れに基づく入射角変動の問題や、羽根車とディフューザ羽根との干渉の問題が現れるので、前図9・21 ベーンレスディフューザ内流れの失速限界4―103縁位置は羽根車出口流れの性状を考慮して決めることが望まれる。図9・22は薄翼からなる翼列ディフューザ(cascade diffuser)である。ベーンレスディフューザ部を経てこれを通り抜ける流れは、次の二つに分けて取扱うことができる。(1)羽根前縁から羽根間スロートまでの半開放域、および(2)スロートから羽根出口までの通路ディフューザ域である。流れが定格点において羽根に沿って流入して適度にスロート断面まで減速できるように形状は定められる。通路ディフューザ部には二次元ディフューザの好適形状が選ばれる。運転点を変化させた場合のディフューザ内流速変化と流量との定性的な関係が図9・23に示される。羽根車を出る流れの速度は流量が変化してもあまり大きさを変えないが、羽根間通路の断面平均流速は流量に比例するため、低流量の場合には半開放部で大きな減速が要求される。羽根の迎え角が増えて減速割合が過大になれば流れは壁面からはがれ、ディフューザの圧力回復も減少する。一方、高流量域ではスロートにおける平均速度が増加するので、ディフューザ入口速度よりも大きくなれば摩擦損失が増えて、ディフューザ性能は低下する。図9・23はこれらで定まる流量限界を示す。したがって、ベーンディフューザは定格点近傍ではベーンレスディフューザより圧力回復係数は高くなるものの、ディフューザが有効に作動する流量範囲は狭い。薄翼のディフューザでは広がり率が過大にならないように、通常羽根の圧力面を凸面に、負圧面を凹面にする。その場合、流線曲率の影響が強いと圧力は凹面から凸面に向かって低下するため、側壁境界層内の低エネルギー流体には圧力面に集積する傾向が現れる。そのため圧力面境界層は厚くなり、圧力損失の増大や流れのはく離の原因となる場合がある。このようなディフューザ通路部での二次流れを抑制するために、その区間を直線壁の広がり通路にすることがある。その内部ではく離が生じないように10°程度以下の広がり角2φとするために、図なす角は360°を羽根数で割った値となるから、くさび角はβ=360-2φで表される。この角度は羽根枚数が少ないほど大きいので、流路の中心線が円周方向となす角や広がり角が一定の場合、羽根先端部の負圧面は羽根数が少なくなるにつれて周方向に傾く。そのためにディフューザ失速につながる流入角は通常小さくなるか図9・22
