# 9・4・3 スクロール

> **章**: 第9章 流体機械の流れ

外向き旋回流を集めて吐出し管に導くスクロール(scroll)は、図9・26のような形状をとるので渦巻室(volute)とも呼ばれる。巻き始め位置の基準半径における円周上で一様流と仮定して、スクロール内で断面平均流速が周方向に一定とする条件をおき、連続の関係と角運動量保存を基に巻き始めの位置から断面積を線形的に増加するように、通常、渦巻形状が定められる。式(9・64)にその関係を示す。=2πθ(9・64)ここで、:角度θにおけるスクロール断面積、:巻き終わり部の断面積である。スクロールの断面形状として円形、長方形、台形などが使われる。送風機で使われることの多い平行壁の場合には、スクロール外壁は巻き角一定の対数らせんとなる。流路の設計では一次元流れの仮定が使用されるものの、半径方向への圧力こう配によって壁面に沿う内向きの二次流れが生じ、現実の流れは複雑である。吐出し量が減少すると羽根車出口流れの周方向速度成分は増えるが、スクロール各断面における平均流速は流量図9・25 小弦節比翼列ディフューザ図9・26 スクロールにほぼ比例して低下する。そのため、スクロールに入る流れはその内部で円周方向の運動量を減らし、それに見合った圧力上昇が出口に向かって現れる。一方、吐出し量が増える場合には逆の効果で圧力降下する。このように設計点からはずれた運転点では、周方向に圧力が変化して羽根車出口流れも円周方向に一様ではなくなり、結果として半径方向推力が羽根車に作用する。スクロールに羽根車からの流れが直接入る場合には、巻き始めを形成する舌部の半径位置や形状が流力性能に大きな影響を及ぼすので注意を払う必要がある。
