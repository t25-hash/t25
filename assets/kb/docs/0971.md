# 9・4・4 吸出し管

水車ランナ下流に置かれる固定の広がり流路を吸出し管(drafttube)という。円すいディフューザも使用されるが、構造上の都合から曲り吸出し管が広く使われている。低落差の場合は水車の水力性能が吸出し管の圧力回復性能に支配されるので、有効度が高くなるような形状が選ばれる。フランシス水車のような高落差の場合には、次に述べる安定性能のほうが重視される。フランシス水車ランナ出口の流れは定格運転点ではほとんど旋回をもたないが、負荷に応じて流量を減らしていくとランナ回転方向の旋回流になる。旋回が強まると流れは外周に片寄る傾向を有するので、入口円すい部の管軸に死水域が生じる。それを取巻くように流れる旋回流との間のせん断層は渦面としての性質から巻上がり、らせん状渦心(vortex core)となって死水域の周りを回転する。死水域の挙動によって渦心は種々の形態をとるが、強い旋回で死水域がほぼ吸出し管入口まで拡大すると、らせん状渦心のピッチは伸び、その規則的な振れまわりが管内で観測される。図9・27は渦心(渦ロープとも呼ばれる)の模式図である。非軸対称となる圧力場も渦心とともに回転するために、圧力は周方向に脈動する。通常、らせん状渦心内部の圧力は低いのでキャビテーションが生じている。この空洞部の体積が増えると吸出し管内流れの固有振動数は低下するので、渦心の振れまわり周波数と一致して共振状態に陥り、フット部内水流の管軸方向の大きな動揺を伴う、ドラフトサージと呼ばれる激しい水圧脈動が発生することがある。水車流量が増えた過負荷状態では、ランナ回転方向と逆の旋回流になるが、軸方向速度は管軸付近が大きい分布となるために部分負荷とは性質の異なる脈動現象が観測される。なお、図9・27と同種の渦心の振れまわりは、順流と逆流が見られる管内旋回流でしばしば観測される。
