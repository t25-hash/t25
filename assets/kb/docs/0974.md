# 9・4・7 中間案内通路

> **章**: 第9章 流体機械の流れ
> **節**: 9・24 に示すようなくさび形状の羽根が使われる。隣接する羽根の

多段遠心圧縮機や多段遠心ポンプで使用される中間案内通路は、ディフューザからの旋回流を外向きから内向きに変えるU 字形通路と、内向き旋回流を軸方向に向け所定の流れにして次段羽根車に導く戻り通路(return channel)から構成される。U 字形通路は子午面内流路の曲率と流れの旋回により外壁面で圧力が高く内壁面で低くなるために、二次流れが生じて低エネルギー流体が内壁側に集積する性質をもつ。それが成長するとはく離が生じて損失増につながるので、内壁面の曲率半径はできるだけ大きくとりたい。U 字形通路に入る流れの旋回が小さくなるようなディフューザ形状にしておくことも望まれる。内向き旋回流は半径位置が小さくなるほど周方向速度成分が大きくなる性質を有するから、U 字形通路からの流れの旋回をとるために戻り通路内には戻り羽根が置かれる。その設計では、羽根前半部は緩やかな曲面として羽根に作用する負荷を軽減し、後半部を大きく曲げて半径方向に流れを導くような形状が採用される。図9・29はその一例である。流路断面積は内側に向かって狭まるので設計しやすいといわれるが、二次流れの発達が著しくて所定の流れが得がたい場合もある。流れの案内が主であるものの、上流に置かれたディフューザで十分に減速されていない場合にはこの通路で圧力回復を図る必要が出てくる。高性能化のために、ディフューザから戻り流路までを一体に見て三次元羽根を使って流れを積極的に導く設計も利用されることがある。内部の流れは三次元性が強く、低エネルギー流体が二次流れで集積し巻上がって通路から流出する場合があるから、次段羽根車への影響に注意を払う必要がある。
