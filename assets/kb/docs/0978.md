# 9・6・3 ラビリンスシールの設計

> **章**: 第9章 流体機械の流れ
> **節**: 9・34 に示す。絞り片を軸心に垂直に設置する軸流型が一般的であ

ラビリンスシールの基本設計は、作動流体が圧縮性の場合には理想ラビリンス[複合直通型、図9・33(c)]に対する理論解析に基づいて、非圧縮性の場合には連続する流体抵抗を有する流路として行われる。いずれも実際の形状に対応させるために、修正係数を導入して補正を加える半経験的な設計方法となっている。図9・33に示したような基本的な形態に関しては豊富な実験データが整備されており、実用上十分な精度で設計を行うことができる。ただし、回転による弾性変形(遠心力効果)および温度変化に伴う熱膨張によってすきまが変化するため、これらの効果による変形量を別途求めておき、稼働時のすきまを用いて設計を行うことが必要である。最近のラビリンスシールでは、漏れ性能を向上するために絞り片や段差の形状を複雑化・最適化することが行われている。また、ガスタービンのような場合には、内部構造の冷却に漏れ流れが利用されており、漏れを低減しつつ冷却に必要な流量と温度を同時に満足4―107することが求められている。このため、コンピュータシミュレーション(数値流体力学)によって多数の形状に対する性能を比較検討して、その中から最適なものを選択し、最終的に実験を行って性能を確認するという設計方法がとられるようになりつつある。例えば、ロード(Rhode)らは、ステップラビリンスシールを対象として標準型-ε乱流モデルを組込んだプログラムを開発し、絞り片高さ、ステップ幅、すきま幅などをパラメータとしたコンピュータシミュレーションを実施し、各パラメータの最適化を図っている。この研究の結果、彼らは、検証実験により流れ場に対する計算誤差がおおむね7%以内であることを確認し、初期形状に対して約60%の漏れ流量の低減を実現している。また、ジマーマン(Zim-merman)らは、摩耗による絞り片先端形状の変化が直通型ラビリンスの性能に及ぼす影響を評価している。文献(1) 大橋秀雄・黒川淳一編、流体機械ハンドブック、(1997)、16、朝倉書店。(2) Wu,C.H.,NACA TN ,2604(1952)。(3) Katsanis,T.,Trans.
