# 10・1・1 管内の波動伝ぱ

> **章**: 第10章 非定常流れ
> **節**: 10・1 流体過渡現象(fluid transients)、管内流れの非定

管路内の非定常流れにおいて、非定常現象の原因となる流体機器の動作時間や系の固有周期など、流れを支配する時間スケールが管路内を伝ぱする圧力波(pressure wave)の往復時間に比べて同程度の大きさであると、圧力波の伝ぱ速度(propagation velocity)が重要な意味をもってくる。流体中を伝ぱする圧力波は、その流速変化が流体の音速(acous-tic velocity)に比べて十分小さい場合、音波(acoustic wave)とみなすことができる。単相流の場合、音波は可逆断熱変化とみなされ、音速の値は基準の圧力と温度に依存する物性値として扱われる(1章参照)。特に水の場合、圧力と密度の関係は広い圧力範囲において(10MPa以下)、次のテイト(Tait)型状態方程式で近似できることが知られている。式(10・1)(10・3)より水の音速の圧力依存性は次のようになる。実際に観測される液体中の圧力波伝ぱ速度は上述の値よりも低くなることが多い。その一つの理由として、液体中の非溶解気体の存在があり、わずかな量でもその容量効果によって音速は著しく低下する。気泡が微小かつ一様に分布し、液相と同一速度で動くとき、すなわち滑りなし均質流とみなせるとき、絶対圧力における気液二相流体の密度ρは次のように表される。一般に液体の熱容量は気体に比べて非常に大きいので気体は等温変化とみなせる。このことから密度と圧力の間にバロトロピックな関係が成立し、式(10・1)より気液二相流の音速は次のようになる。大気圧下の空気-水系の気泡流に関する式(10・6)の成立範囲は図10・1のようになることが知られている。図10・1は気泡流の音速が水中ばかりでなく、空気中の音速に比べても非常に小さくなること、空気はほぼ等温変化とみなせること、式(10・5)は滑りなし均質流の仮定にもかかわらず50%を超えるボイド率まで有効なことを示している。ここで、大気圧における気体のボイド率をα、液体の密度をρとし、液体の密度変化は無視できるものとすれば、圧力のときのボイド率α4―109は次のようになる。ボイド率αが1%を超えるような気泡流では、気体の圧縮性に比べて液体の圧縮性はほとんど影響しなくなるので、式(10・6)は次式で近似できる。管路が薄肉の弾性円筒とすれば、式(10・6)はさらに次のように拡張される。:管路の支持方法と管材料のポアソン比μによって決まる係数で次式によって与えられる。音速は管路内流れの過渡現象解析にとって非常に重要なパラメータであるにもかかわらず、実際の管路系で気体のボイド率や管路の影響因子を正確に把握することは難しい。
