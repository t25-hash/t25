# 10・1・2 管内非定常流れの基礎理論

> **章**: 第10章 非定常流れ
> **節**: 10・1 流体過渡現象(fluid transients)、管内流れの非定

a. 基礎式流管の軸方向に沿う断面積変化が十分ゆるやかで軸の曲率が十分小さいとき、流管内の流れは軸方向のみ重要で半径方向の流れが無視できる。このような流れを準一次元流れ(quasi-one-dimensional flow)と呼ぶ。ここではこのような準一次元の流管内の非定常流れの基礎式を示す(3章参照)。図10・3に示す流管の断面は単相または均質な混相流体で満たされ、流れは連続な変化をするものとすれば、流管の各断面、各時刻で次の質量および運動量の保存則が成立する。熱力学の第一法則に基づくエネルギー保存則は、流体の密度と圧力の間にバロトロピックな関係が存在すれば、流れ場を定めてから温度分布を求める式となる。上述の保存則は流管の一部に自由表面があっても成立するが、以下ではおもに管路内の液体流れを対象に式(10・9)(10・10)を具体化する。このため、さらに次の仮定を追加する。・流体は液体または均質な気液二相流体とし、流れはバロトロピックな変化に従う。・管断面上で静圧力は一定とする。・管路の断面積については、軸方向のゆるい変化と各位置での圧力に対する弾性変形を考慮する。密度ρと音速は上述の仮定から式(10・5)(10・8)で与えられる。一方、運動量保存則の式(10・10)で、単位長さの流体要素に作用する合力は圧力と管内面の壁面せん断応力および重力の項からなり、その軸方向成分は次のように表される。ただし、, は転置ベクトルを表す(以下同様)。実用上最も重要な円管路で、管摩擦が定常値で近似できる場合を考える。これを音響近似(acoustic approximation)といい、式(10・13)で対流項を省略することにほかならない。また、このような均質流体の一次元流れでも、液体中に気泡が存在すると波動の伝ぱ速度が圧力によって大きく異なり(図10・2参照)、方程式の非線形性が顕著となって数値解の精度を維持することが難しくなる。これらの分布は通常、系の定常流れを表すものであるが、必ずしも簡単な関数表示が可能とは限らない。そのような場合には式(10・13)に仮の初期値を与えて数値解析を行い、定常流れに収束したところで初期分布とすることが行われる。系の境界条件を与えるためには、管路系の末端あるいは途中に設置された種々の流体機器あるいは管路要素の力学特性をモデル化する必要がある。過渡現象解析においては、このモデル化は管内定常流に比べてかなり単純化されたものを用いる。その例を以下に示す。特に管路閉端では=0である。特に、式(10・19)で=(大気圧一定)とすると、大気圧開放のサージタンクのモデルとなる。以上の例からわかるように、境界条件となる各要素の動作特性は、機器の代表長さが管路内を往復する波の波長に比べて十分短いものとし、準静的に変化するものとみなしてモデル化される。
