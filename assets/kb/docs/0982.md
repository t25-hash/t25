# 10・1・4 脈動流

> **章**: 第10章 非定常流れ
> **節**: 10・11 から高粘度流体の過渡現象では、定常管摩擦よりも非定常管

往復動式のポンプや圧縮機ばかりでなく遠心式ポンプや圧縮機でも、羽根車と舌部あるいは案内羽根との干渉により回転数に起因する周期的な圧力変動が発生する。この圧力変動は流体機械に接続する管路内に伝ぱし、管路途中あるいは管路端の要素で反射して平均流に周期的な変動をもたらす。これを脈動流(pulsating flow)という。脈動流は配管系の騒音や振動の原因となることがあり、その正確な把握は配管系の設計や運転管理に必要となる。脈動流を数値的に解析する場合は、10・1・3項の特性曲線法や差分解法の適用が可能である。しかし、音響近似と微小擾乱の仮定が成り立つときは、流れの基本変数を定常成分と脈動成分に分離し、基礎式と境界条件の特性を線形化すると解の重ね合わせの原理が使えるようになる。この原理に基づく代表的な解法が伝達マトリックス法(transfer matrix method)である。伝達マトリックス法では、対象とする配管系を有限個の要素に分割し。各要素の特性を分布定数系か集中定数系でモデル化する。図10・4に示した配管中の機器要素の線形特性は、集中定数系でモデル化される。
