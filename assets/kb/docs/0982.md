# 10・1・4 脈動流

> **章**: 第10章 非定常流れ
> **節**: 10・11 から高粘度流体の過渡現象では、定常管摩擦よりも非定常管

flow)という。脈動流は配管系の騒音や振動の原因となることがあり、その正確な把握は配管系の設計や運転管理に必要となる。脈動流を数値的に解析する場合は、10・1・3項の特性曲線法や差分解法の適用が可能である。しかし、音響近似と微小擾乱の仮定が成り立つときは、流れの基本変数を定常成分と脈動成分に分離し、基礎式と境界条件の特性を線形化すると解の重ね合わせの原理が使えるようになる。この原理に基づく代表的な解法が伝達マトリックス法(transfer
