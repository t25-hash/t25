# 10・1・5 サージング

> **章**: 第10章 非定常流れ
> **節**: 10・11 から高粘度流体の過渡現象では、定常管摩擦よりも非定常管

流体にエネルギーを与える速度型流体機械、すなわちターボ型のポンプ、送風機、圧縮機などを低流量域で運転すると、管路内の流れが周期的な変動を起こし、安定な運転が維持できなくなることがある。このような現象をサージング(surging)という。サージングが発生すると、管路系の振動や騒音を引き起こし、場合によっては流体機械や管路要素の破損につながるおそれがある。このため、低流量域の運転が避けられない場合、サージングの発生機構を明らかにしておくことが重要である。サージングはターボ型流体機械の流量-圧力曲線に右上がり特性を含む場合に発生するが、それ以外に管路要素の特性も関与するため、旋回失速など羽根車内部の局所的な不安定流動とは異なるものである。ここではこのようなサージング現象の基本的な性質を明らかにするため、図10・12に示すような最も単純化されたポンプ、管路、水槽、弁からなる送水管路系のサージングを取上げる。通常、サージングによる流れの変動周期は圧力波の往復時間よりも長い。そこで図10・12において、水の圧縮性と管摩擦は無視できるものとし、ポンプと弁の圧力-流量の関係は定常特性が成り立つものとする。このとき管路の非定常流れの基礎方程式は、水柱の運動方程式と水槽における連続の式から以下のようになる。問題はこのような平衡点の安定性(stability)である。平衡点が安定ならばサージングは発生せず、平衡点が不安定な場合にサージングが発生する。一般に平衡点の安定性は中立安定を除けば、平衡点まわりの線形近似系の原点の安定性と一致する。このことから、元の系の平衡点が不安定になるのは、次の2ケースである。以上のように平衡点の安定性判別には線形近似系が有効であるが、平衡点が不安定な場合の解析は元の非線形系に戻る必要がある。平衡点が不安定の場合、式(10・66)(10・67)の解,はリミットサイクル(limit cycle)に発達し、サージングとなる。したがって、サージングが発生するためには、式(10・71)(10・72)より少なくともポンプの揚程曲線に右上がりの傾向>0が必要であることがわかる(十分条件ではない)。ここで、ターボ型流体機械の性能に関する相似則を考慮し、元の方程式系(10・66)(10・67)を無次元表示することにより、上述の結果を一般化する。ポンプが単段の遠心式の場合、羽根車周速度をとして、流量と圧力を次のように無次元化する。平衡点が不安定な場合、式(10・66)(10・67)から時間を消去して、無次元化すると、次の流量-圧力平面の相軌道(trajectory)を表す方程式が得られる。値が大きくなるとともに、リミットサイクルはヘルムホルツ型の自由振動から弛緩振動に移行し、圧力振幅は小さくなる反面、周期は長くなる。一方、静的不安定の場合、平衡点は一般に複数となり、非常に複雑でサージングが発生しやすくなる。以上、液体を輸送するポンプ系でサージングが発生するのは、ポンプの揚程曲線に右上がり特性が存在することと、ポンプ下流側に容量要素を有することが必要条件であることを述べた。しかし、明確な容量要素がなくても、管路系の一部に空気が停留したりポンプ内部にキャビテーションが発生すると、その容量効果によってサージングが発生することがある。特にロケットポンプなどで使用されるインデューサ付き遠心羽根車に伴うキャビテーションサージは、定常なポンプ揚程曲線の右上がり特性も必要とせず、設計点流量でもサージングが発生する可能性があるので、十分な検討と対策が必要である。他方、送風機、圧縮機では、管路途中に容量要素がなくても、気体の圧縮性や羽根車段数の影響でポンプよりもサージングが発生しやすい。そこで、送風機、圧縮機のサージングの発生限界(surge line)を、安全のため流量-圧力曲線の最高点を結ぶラインとし、それより低流量域では定常な運転を避けるようにしている(γ2編「流体機械」第I部4章参照)。
