# 10・1・5 サージング

> **章**: 第10章 非定常流れ
> **節**: 10・11 から高粘度流体の過渡現象では,定常管摩擦よりも非定常管

流体にエネルギーを与える速度型流体機械,すなわちターボ型のポンプ,送風機,圧縮機などを低流量域で運転すると,管路内の流れが周期的な変動を起こし,安定な運転が維持できなくなることがある.このような現象をサージング(surging)という.サージングが発生すると,管路系の振動や騒音を引き起こし,場合によっては流体機械や管路要素の破損につながるおそれがある.このため,低流量域の運転が避けられない場合,サージングの発生機構を明らかにしておくことが重要である.サージングはターボ型流体機械の流量-圧力曲線に右上がり特性を含む場合に発生するが,それ以外に管路要素の特性も関与するため,旋回失速など羽根車内部の局所的な不安定流動とは異なるものである.ここではこのようなサージング現象の基本的な性質を明らかにするため,図10・12に示すような最も単純化されたポンプ,管路,水槽,弁からなる送水管路系のサージングを取上げる.通常,サージングによる流れの変動周期は圧力波の往復時間よりも長い.そこで図10・12において,水の圧縮性と管摩擦は無視できるものとし,ポンプと弁の圧力-流量の関係は定常特性が成り立つものとする.このとき管路の非定常流れの基礎方程式は,水柱の運動方程式と水槽における連続の式から以下のようになる.dd=-(10・66)dd=1-욋(10・67)ここに,:時間,:ポンプ流量,:ポンプ全揚程[=],:タンク水頭,욋:タンク吐出し弁流量[=욋],:管路断面積,:管路長,:水槽断面積,:重力加速度.4―115この系の状態を決定する変数はポンプ吐出し量=と水槽内の水頭=で,ポンプ揚程曲線=と吐出し弁の抵抗曲線욋=욋が与えられれば,初期条件を指定することにより,系の状態の時間変化が求められる.ポンプ-送水管路系の定常運転点は式(10・66)(10・67)においてdd=0,dd=0,すなわち流量-圧力平面(状態平面)においてポンプ揚程曲線と弁の抵抗曲線の交点=,=として求められ,系の平衡点(equilibrium
