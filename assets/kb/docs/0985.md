# 10・2・2 静止流体中で振動物体に作用する非定常流体力と付加

> **章**: 第10章 非定常流れ
> **節**: 10・2 各種物体に作用する非定常流体力

この場合の非定常流体力は流体の付加質量の値と等しく、表10・1に種々な形状をもつ物体の付加質量の例を挙げる。表10・1の係数はポテンシャル流れを仮定して算出されたものであるが、流れや振動条件のかなり広い範囲にわたり10%以内で実験値によく合う。物体に接近して固定壁が存在するときは、壁面に直角な方向の運動に対する付加質量は増大する。また、流体の自由表面がある場合には反対に減少し、表面波の効果を考えなければならない場合もある。
