# 10・2・3 一様流中で振動物体に作用する非定常流体力

> **章**: 第10章 非定常流れ
> **節**: 10・2 各種物体に作用する非定常流体力

a. 振動円柱の非定常流体力特性。直角方向振動の場合一様流中で円柱を強制的に振動さ4―118流体工学せたときの非定常揚力特性の代表的な例として、円柱に作用する揚力振幅および位相φを図10・15に示す。このような変化は渦励振の応答特性に現れる強い非線形性と対応している。. 流れ方向振動の場合流れ方向振動円柱の抗力における変動分は、流れに直角方向の揚力変動に比較して振幅は極端に小さい。b. 振動角柱の非定常流体力特性。一方、フラッタ発生を判別する際に重要な、揚力の位相φの変化を図10・16(b)に示す。角柱の自由振動結果の=5付近における強い振幅依存性をもつ低流速励振の挙動と一致する。c. 軸方向流れによる流体力弾性管の内部および外部に軸方向流れが存在する場合には、弾性管の横変位に伴って遠心力およびコリオリ力の形で流体力が作用する。管表面の流体摩擦力の横変位に与える効果は、上記の力に比して小さい。遠心力およびコリオリ力は、弾性管の静的不安定の原因になる。さらに境界条件によって、管の端面の横変位が零でない場合には、管に単位時間に流出入する流体の運動量すなわち慣性力の仕事としてエネルギーの出入が生じ、動的不安定の原因にもなる。容器内で自由表面を有する液体(液面)が振動する現象を、スロッシング(sloshing)という。
