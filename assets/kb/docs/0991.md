# 10・4・1 自励振動と強制振動

> **章**: 第10章 非定常流れ
> **節**: 10・4 一様流中における物体の流力振動

流れによって物体に誘起される空力弾性的不安定振動は,基本的には物体が自励的に振動する場合と流れの変動によって強制的に振動する場合に分けられる.自励振動としてはフラッタ(flutter)が挙げられるが,これは図10・22の概念図に示すように,物体の運動に伴って流体振動系を構成する後流渦が変化する.そして励振力として後流渦により誘起される付加流体力が発生して物体振動系へフィードバックし,物体の振動が持続される.この現象をフラッタと呼ぶ.フラッタには,振動モードや振動挙動により,曲げ1自由度のフラッタである角柱などで生じるギャロッピング(galloping)振動,タコマ橋(アメリカ)の風による崩壊の主因であるねじりフラッタ(torsional flutter),そして航空機の翼などで生じる2自由度の曲げねじり連成フラッタなどがある.さらに,自励振動的要素に強制振動の要素が加味している渦励振(vortex inducedexcitation),また強制振動の中にはバフェティング振動(buffet-ing oscillation)などが挙げられる.その他,特殊な条件下で生じる自励振動として,円形物体に生じるレインバイブレーション(rain vibration)や後流中の物体に生じるウェーク(後流)ギャロッピング(wake galloping)振動などがあり,おもな流れにより誘起される流力振動(flow-induced vibration)現象は表10・2のように分類される.このような種々の流れによる振動現象の発生は,物体の断面形状に強く依存し,大まかにいえば,断面形状が扁平,すなわち流線形(streamlined)形状の物体になるほど,曲げ図10・22 物体の流力振動発生機構の概念図表10・2
