# 10・4・3 流れ方向振動

> **章**: 第10章 非定常流れ
> **節**: 10・4 一様流中における物体の流力振動

a. 円柱の場合静止円柱後流に交互に渦が放出されている場合、流れ方向に定常抗力のほかに振動数2で変動する変動抗力が作用する。流れ方向振動(in-line oscillation)は、水などの液体流中で質量比が小さく、しかも構造減衰が極めて小さくてが極端に小さい場合には、욥욂욥2=12付近で流れ方向振動が卓越する振動が発生する。図10・27に円柱の流れ方向の振動応答特性(response characteristics)を示す。が直角方向振動に比べて極端に小さい=0.81~1.41の場合、ある程度大きい振動が発生し、욥욂욥2前後の욥=1.2~2.5と2.6~3.6の2箇所に励振域があり、それぞれの励振機構が異なる。このときの振動円4―122流体工学柱後流の流れパターンは、욥욂욥2以下の第一励振域では上下対称の渦配置パターンが、욥욂욥2以上の第二励振域では上下非対称の交互渦パターンが対応する。第二励振域では、後流渦の卓越ストローハル数は욌=욂2にロックインするが、第一励振域では、後流は非ロックイン状態で、自然発生渦の욈の成分が卓越する。また、第一励振域では流れ方向振動が支配的であるが、第二励振域ではさらに直角方向振動も加わって8の字のリサジューを描く。b. 角柱の場合図10・28には、角柱の流れ方向の応答振動挙動を示す。욥욂욥2前後で円柱の場合に類似した二つの励振域が見られるが、この場合욥<욥욂욥2の領域では、かなり小さいでも円柱に比較して励振振幅は小さく、욥>욥욂욥2の励振は顕著になる。
