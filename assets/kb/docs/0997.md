# 11・1・1 基礎方程式

> **章**: 第11章 空力音響および水中音響
> **節**: 11・1 空力音響の基礎式

または式(11・6)で与えられ、これをライトヒルの応力テンソル(Lighthill’s stress tensor)と呼んでいる。式(11・5)の右辺が零であれば、静止媒体中の音の伝ぱを支配するいわゆる波動方程式である。式(11・5)は一般的には非線形の方程式で、乱れなどと発生音の関係を表す。つまり式(11・5)の右辺の乱れなどによる音源によって発生した音波は、乱流の場を通過しつつ逆に乱れなどに影響を与え、それがまた発生音波を変えるというフィードバック関係にある。すなわち式(11・5)の右辺は単なる音源であると解釈することができる。
