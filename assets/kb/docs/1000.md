# 11・2・2 遠距離場の条件

> **章**: 第11章 空力音響および水中音響
> **節**: 11・2 空力音響の基礎式の解

式(11・16)および(11・17)の右辺において、音源から観測点までの距離が大きくなるほど第2項や第3項は第1項に比較して小さくなる。第1項が第2項より大きい条件はそれぞれ、옿옿≧(11・19)옿옿≧3옿옿(11・20)である。式(11・19)に演算子옿옿=ω=2π,=λを代入すれば、≧λ2π、同様に式(11・20)から≧3λ2πが得られる。これらの結果を考慮して遠距離場の条件は、対象としている音の半波長より大なる距離とされている。
