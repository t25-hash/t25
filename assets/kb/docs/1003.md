# 11・2・5 空力騒音の低減策

> **章**: 第11章 空力音響および水中音響
> **節**: 11・2 空力音響の基礎式の解

式(11・24)および(11・25)は、空力騒音の低減化には、(a)主流速度を小さくすること、(b)代表寸法のスケールを小さくすること、とりわけ前者が効果的であることを示す。パウエル(Powell)はライトヒルのモデルにおける音源であるレイノルズ応力の物理的意味を考察して、渦運動が音源であるという、物理的に理解しやすい理論を提案した。それによれば、渦度と速度のベクトル積が重要な音源である。このことを考慮すれば、式(11・24)および(11・25)のは乱れの面積的スケール(相関面積、11・2・3項参照)を意味しており、騒音低減のためには大規模構造の渦(乱れ)の生成を抑えなければならない。また、渦のごく近傍に固体物体が存在する場合には、式(11・23)からわかるように渦の影響が物体の表面の圧力に反映されて、大規模の圧力の相関面積(correlation area)が生じ、大きな音の発生原因となる。
