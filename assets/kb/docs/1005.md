# 11・3・2 プロペラ騒音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・3 回転音源による騒音

翼のように固体物体のまわりに循環があると、その周辺には非一様な圧力分布が形成される。例えばプロペラを考えると、この圧力分布はプロペラとともに回転する座標軸から観察すると時間的に定常であり、音としては感知されない。しかし、これを絶対座標系から観察すると、その非一様な圧力場が回転することによって圧力は時間的に変動し、音として観察される。翼の場合、個々の翼の周辺の圧力分布はほぼ同じであると考えられるので、この圧力場によって誘起される音は翼の枚数と軸の回転数との積を基本周波数とする周期変動であり、いわゆる離散周波数騒音(discretefrequency noise)となる。観測される音のスペクトルでは周波数にピークが形成される。を翼通過周波数(blade passingfrequency)と呼ぶ。回転する翼に一様な流れが流入する場合の音(プロペラ音あるいはグーティン音)のレベルの解析は、最初にグーティン(Gutin)によって成されている。また式(11・27)の導入に当たっては、翼にかかる揚力が半径0.35に集中していると仮定している。
