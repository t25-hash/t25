# 11・3・2 プロペラ騒音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・3 回転音源による騒音

翼のように固体物体のまわりに循環があると、その周辺には非一様な圧力分布が形成される。例えばプロペラを考えると、この圧力分布はプロペラとともに回転する座標軸から観察すると時間的に定常であり、音としては感知されない。しかし、これを絶対座標系から観察すると、その非一様な圧力場が回転することによって圧力は時間的に変動し、音として観察される。翼の場合、個々の翼の周辺の圧力分布はほぼ同じであると考えられるので、この圧力場によって誘起される音は翼の枚数と軸の回転数との積を基本周波数とする周期変動であり、いわゆる離散周波数騒音(discretefrequency noise)となる。観測される音のスペクトルでは周波数にピークが形成される。を翼通過周波数(blade passingfrequency)と呼ぶ。回転する翼に一様な流れが流入する場合の音(プロペラ音あるいはグーティン音)のレベルの解析は、最初にグーティン(Gutin)によって成されている。図11・3に示すように、回転数で直径の円内で回転している枚数のプロペラが、速度욀で前進する場合、点X の観測点での次数の音圧は、式(11・27)で表される。욇,욧=-π1+πcos β+2 Ω0.35×욇Ω0.35sinβ図11・3 プロペラ音×exp Ω+θ-π2 -Ω(11・27)ここで、Ω=2π,=욀,=,=ρ+욀2は音源の速度に基づく動圧であり、:動翼先端の回転速度≡Ωである。また、とを翼にかかるトルクとスラストとすると、=Ωρ,=ρであり、それぞれトルク係数とスラスト係数である。また式(11・27)の導入に当たっては、翼にかかる揚力が半径0.35に集中していると仮定している。
