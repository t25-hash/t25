# 11・3・4 ターボ機械の広帯域騒音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・3 回転音源による騒音

騒音のスペクトル分布において、そのエネルギーが広い周波数帯域になだらかに分布するものを広帯域騒音(broadband noise)という。翼などの流線形状物体からの広帯域騒音の発生要因は、次のように流れの乱れの成分によっているので、広帯域騒音は乱流騒音(turbulence noise)と呼ばれる。(1) 翼面上の境界層内の乱れ(渦)(2) 翼後縁近傍での流れのはく離によって生じる放出渦(3) 主流中の乱れ(渦)などによる主流速度の時間的変化これらの要因のうち、(1)の場合、境界層内の乱れによる壁面上の圧力変動が直接音として放射される成分は、一般に無視できるとされているが、境界層内の組織的な大規模構造の渦が、翼後縁近傍4―128流体工学で翼と干渉して放射される成分は大きい。これは後縁騒音(trai-ling edge noise)と呼ばれている。(2)の場合については、設計流量より過大な流量における場合のように、主流が翼の凸面側に流入する条件下においては、凸面側の翼面境界層が層流となり、これが翼後縁近傍の逆圧力こう配の領域ではく離し、凹面側の翼後縁から出発した自由せん断層と対を成してカルマン渦列を形成し、離散周波数音(いわゆるエオルス音、aeolian tone)を発生する。しかし、翼面に流入する主流速度がハブから翼先端に向けて増加するため、その周波数はハブから先端へと増加する。したがって、測定される騒音のスペクトルはこれらの積分値であるため、各ピークの包絡線であるなだらかないわゆる広帯域分布を示す。このカルマン渦による騒音発生が過大流量域での騒音の増加の原因となる。このような場合の発生騒音のスペクトル分布の半理論的予測もされている。また(3)に関しては、翼に流入する主流の乱れ度が高い場合と、主流の迎え角が時間的に大きく変化する場合の二つが考えられるが、騒音発生の物理的機構は同じである。すなわち、いずれも翼に対する迎え角が変化するために、物体まわりの循環が時間的に変化することによって騒音が発生する。したがって、前者は乱流騒音となり、後者は時間的変化が周期的である場合は、離散周波数騒音となる。11・3・3・b項で述べたように、流入する流れが偏流していたり、動翼の上流の物体の後流が流入して発生した離散周波数騒音がその例である。これらのいずれの場合も流れと翼との干渉によって音が生じるので、双極子型の音源である。
