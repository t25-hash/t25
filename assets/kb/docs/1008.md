# 11・3・5 騒音低減対策

> **章**: 第11章 空力音響および水中音響
> **節**: 11・3 回転音源による騒音

a. 音源ファンの場合、翼に流入する主流速度(翼に対する相対速度)は最も速い翼の先端においても音速より十分低い。したがって、動翼から発生する騒音は双極子型であり、発生騒音の音圧レベルは基本的にはほぼ回転数の6乗則に合う。ジェットエンジンの圧縮機のように翼先端速度が音速に近い場合は、動翼単独の回転および動・静翼の干渉のいずれの場合にも、高いレベルの離散周波数騒音が発生するうえに、がに近い値となりダクト内での減衰は期待できない。また、>の場合には全く減衰しないので、この低減策としてはダクト壁の吸音処置に頼らざるを得ない。乱流騒音の低減策は、主流速度(動翼回転数)を下げる、および翼から大規模な渦が放出されないように工夫する、という二つにつきる。ファン騒音の低減化に当たっては、最も支配的な騒音をまず低減しなければならない。このことは、回転体から発生する騒音の場合、その原因の究明に当たって翼先端近傍に着目しなければならないということを意味している。b. 管路の抵抗特性とのマッチングターボ機械の作動点は、ターボ機械の圧力特性曲線と管路の抵抗特性曲線との交点で定まる。したがって、ターボ機械は設計どおりに作られていても、管路抵抗の予測を誤ると設計点では作動しない。その場合、一般に乱流騒音の音圧レベルが著しく増加する。図11・5はその例を模式的に示したものである。図11・5には軸流ファンの圧力特性曲線と抵抗特性曲線を示しているが、流量はこの2曲線の交点で決まる。抵抗特性曲線が適正なA の場合には設計流量φで作動し、騒音の音圧レベルは低い。しかし、管路抵抗が大きくなって抵抗曲線がB のようになると、流量はφと減少し、騒音は高くなる。この例からわかるように、騒音低減化に当たっては、どのような作動点で運転されているかを把握する必要がある。
