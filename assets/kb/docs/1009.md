# 11・4・1 ジェット騒音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・4 乱流騒音

乱流騒音(turbulence noise)の音源項に関してはすでに11・1・1項で述べた.これらの音源は,乱流の種類によって支配的な音源になる非定常現象が異なるが,レイノルズ応力の変動が主音源になることが一般的である.a. 乱れによる音レイノルズ応力に変動を与える乱れの主成分は渦度の時間的空間的変化である.渦輪一つが受音点の近くを通過すると1周期の音波を観測でき,連続した二つの渦輪が引込み現象を伴って通過すると,変調された複数の周波数の音波を観測できる.よって,複数の渦が周期的な間隔を保って移動すると離散周波数音(discrete frequency tone)が発生し,複数の渦が互いに干渉しあって移流すると広帯域騒音(broad band noise)に近い音が発生する.ジェット騒音(jet noise)は後者の典型的な現象である.b. ジェット騒音の低減亜音速噴流についての上述の全音響出力に関する式からわかるように,ジェット騒音を低減するためには,噴流速度を弱めるか,乱れの空間的相関スケールを小さくすることが必要になる.ジェットエンジンでは,二重噴流にすることで推進力を保ったまま噴流速度を低下させて騒音低減を実現している.噴流出口に口径の15%程度の大きさの微小縦渦発生器(vor-tex generator)を取付けると空間的相関スケールが小さくなり,低周波数成分が約5dB 低減している.また,二重噴流の速度比を逆転すると騒音を抑制したり促進したりできるので,噴流外周のせん断流の制御がジェット騒音の大小に敏感に影響することが推察できる.噴流出口でせん断流にストローハル数0.018の圧力変動を与えると,乱れの時間的相関スケールが大きくなり,発生騒音が約5dB 低減する.超音速ジェット騒音にも二重噴流の有効性が確認されており,욢=1.5の不足膨張主噴流の外周に=0.84の副噴流を加えると,広帯域騒音が約15dB 低減している.
