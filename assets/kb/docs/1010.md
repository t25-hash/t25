# 11・4・2 固体物体まわり流れからの騒音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・6 内の表に示すように、混合領域でほぼ一定で、乱流領域で

ライトヒルの応力テンソルの変動を作るような乱れの塊(渦の集合体)が四重極子とみなされており、この四重極子が固体物体近傍を通過するとその鏡像四重極子音源が固体表面に誘起される。この四重極子音源の波長より固体物体のスケールが十分に小さい(例えば10%以下)場合には、遠距離場に対して実音源と鏡像音源(mirror sound source)が合成されて双極子音源(dipole source)が固体表面に発生したとみなすことができ、そこからの音波の放射4―130流体工学効率が単独の四重極子以上に大きくなる。この実現象に基づいてライトヒルのアナロジーにおける見掛け上の音源第2項(体積力の変動に起因する音源)を、より実務的に取扱うことが可能になる。また全音響出力は代表流速の6乗に比例する。一般的な流体機械や地上の交通機械では、周囲流体との相対速度が0.03<<0.4の範囲であり、これらから発生する空力騒音のほとんどの支配的騒音源が固体表面の圧力変動に起因するものである。壁面せん断応力τは乱流境界層厚さや表面粗さおよび主流速度と比例関係があるが、詳細は乱流境界層の性状に左右される。c. はく離流れ騒音(separated flow noise)流体中の固体表面から流れがはく離し、表面に再付着しないで後流域を形成するような完全はく離流れでは、後流中の乱流応力による四重極子音源と固体表面はく離域内の圧力変動による双極子音源の発生を推察できるが、後者が支配的であることが実験的に確認されている。よって、騒音強さの速度依存性や指向性は式(11・37)に関する説明と同様であり、表面の圧力変動の特性が解明されると式(11・37)に基づいて発生騒音のスペクトルなどを知ることができる。円柱を横切る流れがカルマン渦放出条件よりも大きなレイノルズ数(10以上)の場合、円柱後流は完全はく離流れであり、流れに直角な方向で観測する騒音のスペクトルは図11・9に示すようになる。横切る流れに含まれている初期乱れが少ないとストローハル数(Strouhalnumber)0.5付近で最大値となるが、初期乱れが大きくなると最大値を示すストローハル数が0.1程度にまで変化する。大きな迎え角をもつ平板まわりからはく離流れが発生すると、発生騒音のスペクトルはストローハル数が0.3前後に最大値を示し、その両側の周波数域で周波数の0.5~1.0乗に比例して増加または減少する周波数分布になっている。また、主流の周期的変動に応じてはく離流れの発生消滅やはく離流れ域の増加減少があると、はく離流れ騒音のスペクトルに主流変動周波数とその高調波の離散周波数成分が重なり合って生じる。はく離流れが固体表面で再付着する場合は、そのはく離泡の変動周波数が騒音の支配的成分になったり、自励音が発生することもある。また、流体中の固体物体の後流渦もはく離流れ現象の一種であるが、周期的な渦放出に伴う体積力の変化が発生騒音の主要音源になり、渦放出音(vortex shedding tone)として数多くの報告がある。なお、はく離流れ領域境界に相当する強いせん断流域が固体境界面から大きく離れているような流れでは式(11・37)とは異なって、せん断流内の渦度変化に基づく空間的双極子音源が支配的になって、高周波数域まで強く広がる広帯域騒音を発生する。d. 固体物体騒音の低減式(11・37)における表面圧力変動の表面二重積分は、圧力変動の自己相関とその空間的相関スケール(相関面積)の積で表すことができ、時間微分した表面圧力変動強さか、相関スケール(correlation scale)を小さくすることで、固体表面からの発生騒音を低減できる。表面の圧力変動を小さくするためには、固体境界周囲の速度変動をより弱くするように境界層流れやはく離流れを制御することである。固体表面に吸音材を張り付けることで表面圧力変動を低減することも一手段である。相関スケールの縮小化のためには、表面のスパン方向に連続な渦度分布の分断や、固体境界面のスパン方向への形状の一様性や連続性を崩すことが必要である。流体力学的な不安定現象が存在する流れ場において、特定形状の物体と流体とが干渉して、その流れ場に特定周波数の流力振動を励起する。気体の場合それに伴って発生する離散周波数音を自励音(self excited flow tone)と呼ぶ。流体に接する構造体の振動が流力振動を励起する現象も存在するが、ここでは構造振動を媒介しない流体の自励振動現象に関してだけを自励音の対象とする。乱れ(または渦)の移流による音波(または圧力場の変動)の発生と音波のフィードバックによる乱れの発生という閉ループ系において、入力エネルギーが系の散逸項と釣合うリミットサイクルが形成されると自励音が発生する。
