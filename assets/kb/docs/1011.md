# 11・5・1 カルマン渦励起音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・5 自励音

流体中の二次元物体の後流にカルマン渦列が形成され、このカルマン渦列の放出に伴って離散周波数音が励起される。カルマン渦列の発生自体が流体の自励振動現象である。物体から流れ出した渦がある距離流れたときに物体周辺に誘起する圧力場は次の渦の流れ出しに影響を与えているので、周期的に渦が放出されると推論できる。渦を周期的に放出することで物体表面に周期的な圧力変動が誘起されて、物体表面が双極子音源になって音が放射される。なお、カルマン渦列の発生は渦と物体の干渉によって発生する自励振動現象である。a. エオルス音(aeolian tone,aiolos tone)円柱を横切る気流に伴うカルマン渦励起音がエオルス音と称する現象である。エオルス音のスペクトルの一例を図11・10に示す。b. 後流渦放出音角柱や平板などの二次元物体からもカルマン渦励起音が発生し、これもエオルス音の例である。小さな迎え角の平板の後縁近傍にはく離域が発生すると周期的渦放出が励起されて、離散周波数音が発生する。平板後縁での境界層排除厚を代表寸法にするとストローハル数が約0.2になる。対象翼を一様気流図11・9乱流中円柱まわりはく離流れからの発生騒音スペクトル図11・10単独円柱および交差円柱を横切る流れからの空力音響図11・11 自励音状態の平板列からの渦放出の位相関係図11・12物体列近傍のモードと物体弦長との相関関係中に設置すると、微小な迎え角において離散周波数音が発生する。これは翼後縁からのカルマン渦列放出に伴う空力自励音である。翼表面の境界層内に発生する不安定波を介した自励音が発生する場合もある。c. カルマン渦励起の自励音音波によるフィードバックがカルマン渦励起音に顕著に影響すると強い自励音が発生する。例えば、図11・11に示すような列状物体を気流ダクト内に設置すると、気流に直角方向のダクト内の音響定在波(acoustic standingwave)の影響でスパン方向に位相がそろった周期的渦放出と音波の発生が列状物体に強く励起される。このような構成の実例は、長方形断面ダクト内の円柱列や平板列あるいは熱交換器管群、コーナベーンなどであり、流体機械内の環状翼列も該当する。環状翼列の場合は定在波モードだけではなく、回転モードの音響波も、それらのモード周波数の翼列放出渦によって自励音を誘起する。これらの自励音の周波数は、流速変化に対して列状物体の代表厚さに基づくストローハル数一定の比例関係に沿っているが、周波数が階段状に変化していく。渦放出の位相を支配する定在波の波数を、ダクト長さを、発生音波数を、物体弦長をとすると、二次元モードに対して図11・12に示すような関係がある。弦長が長い物体列ほどダクト遮断されやすいモードになる。なお、列状物体を流4―131図11・13 せん断流自励音の原理的発生機構れ方向に間隔を空けて設置すると、その間隔から定まるダクト軸方向の音響定在波の影響を受ける場合もある。d. カルマン渦励起音の制御音響定在波に強く影響を受けている場合には、定在波の周波数と渦放出の周波数が大きく異なるようにダクトの寸法を変更したり、干渉仕切板(baffle plate)をダクト内に設けると自励音が弱くなる。通常のカルマン渦励起音でも、カルマン渦の柱状体軸方向の相関長さ(correlation length)を小さくすると放射音が低減する。図11・10に示したように1本の円柱の直下流に追加の円柱を交差設置して相関長さを短くをすると、エオルス音が低減する。円柱よりも角柱のほうが渦放出音の卓越性が強い現象は、角柱の相関長さのほうがより長いことによる。柱状体表面にトリッピングワイヤを巻付けたり、微小突起を取付けると相関長さが短くなる。表面を一様に粗面にしても相関長さはあまり変化せず、見掛け状の代表径が太くなることで周波数が低くなる。一方、柱状体を流れ方向に傾斜させると相関長さが長くなるが、横切る流速が低下するので発生音周波数が低くなる。円柱を下流方向へ凸状となるように曲げると、相関長さが極端に短くなってエオルス音が消え、はく離騒音だけになる。渦放出自体を制御するためには能動的なアクチュエータを用いることが必要であるが、相関長さを制御するためには多様な手法が考案可能である。
