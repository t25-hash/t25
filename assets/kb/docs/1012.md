# 11・5・2 エッジトーン

> **章**: 第11章 空力音響および水中音響
> **節**: 11・5 自励音

膜状の噴流が同軸上にこれと平行に設置されているくさびに衝突する状態で自励音が発生し、エッジトーン(edge tone)と称する。しかし、噴流がある程度厚くなって、くさび先端が噴流のポテンシャルコアに包み込まれるようになると自励音は発生しなくなる。噴流のせん断流がくさびと干渉することが自励音の発生条件であり、せん断流の自励振動現象を理解することが重要である。a. せん断流の自励振動ある固体境界端から流出するせん断流(shear flow)が下流にある別の固体境界近傍を通過すると、流体に自励振動を生じる干渉現象が起こる。これはエッジトーンに関連する各種自励音に共通する基本機構であり、図11・13に示すように説明できる。上流側固体境界端から流出する主流境界部のせん断層が秩序だった不安定運動(例えば渦列)を保有していて、不安定せん断層が発達しながら移流していく。不安定運動が下流側の固体境界縁のごく近傍を通過するときに第二の擾乱(音波)を誘起する。この擾乱が上流側にフィードバックして、上流側の固体境界端に至るときにせん断層の不安定運動の発生に刺激を与えることで、閉鎖フィードバックループが形成される。不安定運動と擾乱の位相が一致してループが成長すると強い音波が周囲へ放射されるようになる。したがって、自励音の周波数は次のようにして定まる。発生する自励音の強さは主流速度の4~6乗に比例するが、不安定運動から音波の誘起と、遡上音波が不安定運動を刺激する各過程でのエネルギー変換効率が100%になることはないので、不安定せん断層が移流する過程で不安定運動が成長することが不可欠である。また、せん断層が移流する空間に音響的共鳴機構が存在すると、フィードバックする擾乱も移流する不安定運動も強調され、自励周波数も共鳴周波数に支配された極めて強い自励音に発達する。b. エッジトーン周波数の変化特性板や噴流のスパン方向に平行な円柱であっても自励音が発生する。噴流-円柱系の自励音においては、円柱表面の噴流せん断層が衝突する付近でのはく離渦が交互に発生することが擾乱の誘起に重要な役割を果たしている。c. ホールトーン(hole tone)円形断面の噴流が噴出口の下流にあるオリフィス板に衝突する構成で発生する自励音である[図11・14(b)].オリフィス孔の周縁がエッジトーンにおけるくさびの役割を演じている。円形噴流が1枚のオリフィス板に衝突しなくても、2枚のオリフィス板に一方向から気流が流れると、1枚めのオリフィス孔を通過した気流が2枚めのオリフィス板に衝突して同様な自励現象を生じる。また、円形噴流が同軸上にあるリングに衝突しても自励音が発生し[図11・14(c)]、これをリングトーン(ring tone)と呼んでいる。一方、軸対象噴流が金網に衝突する場合にも同様な自励音が発生する。このことは金網の整流効果によってせん断流の自励振動を抑制することが困難であることを示唆している。なお、せん断流の影響がない円形自由噴流において離散周波数音が生じることがある。図11・14(d)に示すように、噴出口での渦輪放出の不安定性が噴出口上流側の音響共鳴の影響を受けて、周期的な渦輪が放出されて音波が発生する現象であり、バードトーン(bird tones)と称する。口笛もこの現象の一つである。d. キャビティトーン(cavity tone)流路壁面にくぼみ(cavity)や溝あるいは箱状の孔が開いていると、壁面に沿うせん断流によって自励音が発生する。これがキャビティトーンである。流体機械の内部や管路系およびベローズ管などで発生する離散周波数騒音の原因の一つであり、航空機の脚格納部からも発生することが多い。図11・14(e)に示すように、くぼみの上流側縁部から境界層がはく離してせん断流が流出し、せん断流中の渦(不安定運動)が移流してくぼみ下流側の縁に接近して通過するときに音波が誘起されて、フィードバックする擾乱になる。さまざまな寸法の矩形くぼみにおいて、下流側の縁近傍の表面圧力変動が極めて強くなることが実験的に確認されている。発生音の周波数は上述(a項)の式のようになるが、渦の移流速度と主流速度との関係がくぼみ上流側縁部での境界層厚さに支配されるので、実験的にはさまざまな係数βが報告されていて確定的ではない。また乱流境界層であったり、スパン方向にコヒーレントではない境界層であったりするとキャビティトーンが発生しない場合もある。さらに発生する自励音強さを定量的に予測できるまでには解明が進んでいない。また、幾何学的なくぼみ形状ではなくても、流路の壁面のさまざまな形状がキャビティトーンを発生しやすい形状とみなされることが経験的にわかっている。管路系において、オリフィスが複数枚並んでいたり、断面の拡大縮小部[図11・14(f)]があったり、ベローズ部があると、環状のキャビティとなってキャビティトーンを発生する。この場合、管路の軸方向の音響共鳴モードの影響を受けて、極めて強い自励音が生じることもあり、自励音が成長しない場合もある。また、管路壁に環状の空げきがある場合にも同様な自励音が発生しやすいが、空げき部を移流する環状渦の周方向位相分布が励起に強く影響する。e. 後流干渉の自励音一様流中に2本の円柱が流れ方向に並んでいる場合[図11・14(g)]、上流側の円柱の放出渦が下流側の円柱の前面に衝突して圧力波を誘起する。この圧力波が擾乱となってフィードバックして上流側円柱の渦放出に刺激を与えるので、閉ループが成長して自励音が生じる。閉ループの周波数と円柱本来の渦放出の周波数が一致する場合に、最も顕著な自励音になる。この現象は平板(角柱)が2本、流れ方向に並んでいる場合にも発生し、上流側物体からの後流のせん断層が不安定運動を保有しているために、下流側物体との干渉によって自励音を誘起するものである。流体機械内部の支持板と静翼列との関係や配管系での整流板と支持板(仕切板)との関係などにおいて、この自励音は異音問題の原因の一つである。
