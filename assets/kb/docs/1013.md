# 11・5・3 超音速流の自励音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・5 自励音

超音速噴流は擾乱を発生させるような固体物体に衝突しなくても、図11・14(h)に示すように衝撃波セルの振動によって自励音が発生する。これをスクリーチトーン(screech tone)と呼び、自励ループは次のような機構である。第三衝撃波セル内を移動する斜め衝撃波がセル下流の垂直衝撃波にぶつかると音波が発生し、音波は噴流の外側を音速で伝わって噴出口のせん断流における渦放出に刺激を与える。噴出口から放出された渦はセル内を移流する斜め衝撃波を誘起することで閉ループが構成される。スクリーチトーンの周波数は次のようになる。욣=욂2π1+욂ここで、욂:移流速度、:音速、:波数である。円形超音速噴流の場合、衝撃波セル内を移流する斜め衝撃波はらせん状に回転して音波を発生する。噴流のせん断流が物体に衝突して自励音を生じる現象は超音速噴流でも存在し、ハルトマン笛だけではなく、ホールトーンや平板衝突噴流音[図11・14(i)]も確認されている。また噴流ではなくて物体まわりの膜状超音速流において衝撃波セルが構成されるとスクリーチトーンが自励する。一方、超音速噴流の自励音を低減するためには、噴出口に微小突起(tab)を装着すると効果的である。水中において観測される音には、(1)空気中にある音源からの音が水面から水中に透過してきた場合のもの、(2)水中にある音源から発せられた場合のものがある。空気中の音源からの音が水中に透過する際、音響(固有)インピーダンス(acoustic impedance)が異なる水面における透過率は式(11・38)のように与えられる。α=욧욡=4욌욀욌+욀(11・38)ここに、욡は入射音波の、욧は透過音波の音の強さを、욀および욌はそれぞれ空気および水の音響インピーダンスを表す。伝わる音波が平面波であれば、욀=ρ욀욀、および욌=ρ욌욌と表される。ここに、ρ욀,ρ욌:それぞれ空気および水の密度、욀,욌:それぞれ空気および水中における音速を表す。なお、反射率は=1-αと定義される。20°C,1気圧における空気の音響インピーダンス욀=415(Nsm)および水のそれである욌=1.5×10(Nsm)を式(11・38)に代入すると、α=0.001を得る。これは、空気中から入射する音のエネルギーの1/1000程度が透過し、ほとんどは反射されることを示す。その逆も同様であり、水中における音のほとんどは水面において反射される。水中の音圧レベルの基準値として、욌=1×10Paを用いる。空気中で計測した音圧レベルが욀dBであるとすると、水中での音圧レベル욌は式(11・39)で表される。욌=욀+20log욌+10log 욌욀α(11・39)ここに、は空気中における基準となる圧力で、1kHz の正弦波の音の最小可聴音圧であり、=2.5×10Paである。式(11・39)に、これとインピーダンスおよび透過率などを代入すると、20°C,1気圧のとき次の関係式を得る。욌=욀+31.6(11・40)これは、例えば空港周辺領域において、航空機の騒音が水中環境に及ぼす影響を調べる際に便利である。なお、水中の音速は式(11・41)で与えられる。욌=욌ρ욌(11・41)ここに、욌は水の体積弾性率で、20°Cにおける値0.22×104―133Pa、および、同温度の水の密度998kg/mを式(11・41)に代入すると、욌=1484m/sとなる。前述したように水中において観測される音の発生源として、水中にある固体の振動のほかに、以下に述べる気泡の振動がある。これらの音の伝ぱや固体振動による音源については空気中における取扱いを水中に適用すればよい。水中音響が空気中と異なるのは、気泡が音源となる場合である。以下に気泡音(bubble tone)について述べる。
