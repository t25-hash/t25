# 11・5・3 超音速流の自励音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・5 自励音

超音速噴流は擾乱を発生させるような固体物体に衝突しなくても、図11・14(h)に示すように衝撃波セルの振動によって自励音が発生する。これをスクリーチトーン(screech tone)と呼び、自励ループは次のような機構である。第三衝撃波セル内を移動する斜め衝撃波がセル下流の垂直衝撃波にぶつかると音波が発生し、音波は噴流の外側を音速で伝わって噴出口のせん断流における渦放出に刺激を与える。噴出口から放出された渦はセル内を移流する斜め衝撃波を誘起することで閉ループが構成される。スクリーチトーンの周波数は次のようになる。円形超音速噴流の場合、衝撃波セル内を移流する斜め衝撃波はらせん状に回転して音波を発生する。噴流のせん断流が物体に衝突して自励音を生じる現象は超音速噴流でも存在し、ハルトマン笛だけではなく、ホールトーンや平板衝突噴流音[図11・14(i)]も確認されている。また噴流ではなくて物体まわりの膜状超音速流において衝撃波セルが構成されるとスクリーチトーンが自励する。一方、超音速噴流の自励音を低減するためには、噴出口に微小突起(tab)を装着すると効果的である。水中において観測される音には、(1)空気中にある音源からの音が水面から水中に透過してきた場合のもの、(2)水中にある音源から発せられた場合のものがある。空気中の音源からの音が水中に透過する際、音響(固有)インピーダンス(acoustic impedance)が異なる水面における透過率は式(11・38)のように与えられる。これは、空気中から入射する音のエネルギーの1/1000程度が透過し、ほとんどは反射されることを示す。その逆も同様であり、水中における音のほとんどは水面において反射される。式(11・39)に、これとインピーダンスおよび透過率などを代入すると、20°C,1気圧のとき次の関係式を得る。なお、水中の音速は式(11・41)で与えられる。前述したように水中において観測される音の発生源として、水中にある固体の振動のほかに、以下に述べる気泡の振動がある。これらの音の伝ぱや固体振動による音源については空気中における取扱いを水中に適用すればよい。水中音響が空気中と異なるのは、気泡が音源となる場合である。以下に気泡音(bubble tone)について述べる。
