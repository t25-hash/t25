# 7・3・2 高速フーリエ変換（FFT:fast Fourier transform）

> **章**: 第7章 フーリエ変換
> **節**: 7・2 フーリエ級数

離散フーリエ変換式(Ⅱ-7･11)は 回の複素数の積であるが，
eの周期性を利用して，この計算回数をlogのオーダ
まで減らすことができる．このアルゴリズムを高速フーリエ変換と
いう．大きい のときの計算効率が顕著によくなり，離散フーリ
エ変換が実用化された．例えば，＝1024のとき，＝1048576
に対してlog＝10240で，計算回数は約1/100になる．FFT
は離散フーリエ変換式(Ⅱ-7･11)の代名詞として使われることもあ
る．
