# 11・6・2 気泡音の実際例

> **章**: 第11章 空力音響および水中音響
> **節**: 11・6 水中音響

水中に気泡が存在する原因には、(1)キャビテーションおよび沸騰による気泡生成、(2)水面からの気泡取込みがある。どのような原因にせよ、いったん気泡が水中に発生するかもしくは取込まれれば、式(11・57)および(11・59)からわかるように、気泡の大きさに依存する周波数の音が発生する。サブクール沸騰による気泡の最大径は=1.3mm程度であり、式(11・57)より、2.5kHz 程度と計算される。雨粒の水面への衝突によって取込まれる気泡の代表的半径は0.25mmであり、13kHzの音が卓越する。実際には、多数の気泡が集まって存在する。このような場合、気泡の存在によって音速욁は変化し、式(11・61)で与えられる。욁=3욦ρ욌욁4π3=욦ρ욌(11・61)ここに、욁:単位体積当たりの気泡の数m,:ボイド率(void fraction)を表す。いま、半径の気泡が代表寸法욂の気泡クラウド(bubblecloud)を形成しているとする。ここに、ボイド率は=욁욂と表される。욁は気泡クラウド中の気泡の総数である。このとき、式(11・57)で与えられる単独の半径の気泡が発生する音の基本周波数ωに対して、気泡クラウドから発生する音は式(11・62)で与えられる周波数にシフトする。ω욈=ω욁(11・62)ここに、は整数であり、気泡クラウドは円柱調和関数の振動モードであり、=1が最低次のモードとなる。例えば、=0.001m,욂=0.1m,욁=1000の場合、ω욈=ω3となる。したがって、この気泡が単独で存在する場合3.3kHz の音が発生するが、上記の条件の気泡クラウドになると、1.1kHz の音が卓越することになる。
