# 11・6・3 水中超音波

> **章**: 第11章 空力音響および水中音響
> **節**: 11・6 水中音響

音圧は周波数に比例するので、周波数の高い超音波では、音圧が大きい。このため水中ではキャビテーションが容易に起こり、発生した気泡はつぶれるとき、衝撃波を伴った大きな音を発生する。これが壁面近くで起こると壁面を腐食させる。また、発熱や発光現象である音響ルミネッセンスが生じる。音波の減衰の大きさは周波数の二乗に比例し、空気中における減衰は水中のそれに比べて1000倍程度大きい。このため、超音波の空気中での使用は少なく、液体または固体中での応用がほとんどである。なお、水中に気泡などが存在すると超音波の減衰はさらに大きくなる。超音波は波長が短いため、鋭い指向性をもつので幾何光学的扱いができることなどの特徴をもつ。このような性質を利用し、音速の異なる材質で作った音響レンズを用いて超音波を屈折させて集音し、焦点に強力な音場を作ることができる。このことを超音波加工や胆石の破壊などに用いている。また、反射を利用してソナーとして用い、魚群探知機や測深機に応用している。超音波を水中から水面に向けて発射すると、水面にさざ波状の表面波を励起する。波の加速度が大きいと、この表面波の山が砕け、小さな水滴として飛び散る。水滴の大きさは周波数が高いほど細かく霧状になる。これを応用したのが、超音波加湿器や吸入器である。文献(1) Lighthill,M.J.,Proc.
