# 12・2・1 発生条件

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

キャビテーションは液体の速度が増加し、流れ場の局所圧力がある臨界圧より低下すると、液相が気相に相変化してキャビテーション気泡が発生する現象である。この点では液体の温度がその圧力の蒸発温度より高くなったときに発生する沸騰現象とは相異がある。キャビテーションが発生すると、流体機器の性能を劣化させたり、材料表面に壊食(erosion)を引き起こしたりするので、機器の設計や信頼性の観点から考慮すべき重要な課題となる。一般に、キャビテーションは液中に存在する微小なキャビテーション核(cavitation nuclei)を引き金として初生(inception)し、その最も活性なものとして直径10~100 m程度の空気泡からなる気泡核の存在が知られている。したがって、主要な発生条件には次のものがある。(1) 液中に活性な気泡核が存在すること(2) 局所圧力が蒸気圧より低いこと(3) 気泡核が目に見えるほどの大きさに十分成長できる滞在時間がとれることいま、半径の球状気泡の準静的な安定性を考えると、気泡内外の圧力が釣合っているとき、気泡内の蒸気および非凝縮ガスの分圧と表面張力を考慮すると式(12・1)が成り立つ。通常の気泡核は曲線B 上にあり、周囲圧が低下してに達すると急激に膨張し、初生すると考えられる。通常、キャビテーションの発生は主として流れ場の特性、特に圧力場に依存するので、式(12・3)で定義するキャビテーション数(cavitation number)σがその発達の程度を表す指標として用いられる。系の圧力と主流速度を変化させ、キャビテーションが発生し始める状態のσを初生キャビテーション数(incipient cavitation num-ber)σ、一度発生したキャビテーションが消える状態のσを消滅キャビテーション数(desinent cavitation number)σと呼ぶ。一般にσ<σとなり、発生条件の一つである気泡核の役割の程度が異なることに起因する。この現象はキャビテーションヒステリシス(cavitation hysteresis)と呼ばれる。また、液中の気泡核は時空間に分布するため、σの値にばらつきが大きい。このため、再現性が良く安全側の指標であるσが用いられることも多い。無次元量であるσの相似性を調査するため、世界各国のキャビテーションタンネルで同一軸対称物体を用いて、国際試験水槽会議(ITTC)による比較実験が行われた。その結果を図12・2に示すが、σの値は各実験ごとに大きく異なっており、12・4節で述べるように、その発生形態にも異なる種々のものがあることが確認された。このことは、同一試験物体を用いても、タンネル内の速度分布図12・1 気泡核の安定性図12・2 ITTC によるσの比較実験や乱れ、物体表面上の境界層遷移やはく離、試験液中の気泡核分布やガス含有度などの多くの要因によって、キャビテーションの発生に相異があることを示している。
