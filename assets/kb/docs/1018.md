# 12・2・2 キャビテーション核

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

気泡核の大きさと数について種々の計測が試みられた。おもな計測法としては、(1)空間的情報を有するホログラフの再生像から計測するホログラフィ法、(2)気泡核によるレーザ光の散乱強度から計測する光散乱法、(3)口径0.1mm程度のオリフィスを気泡核が通過する際に生じる電極間の電気抵抗変化から計測するコールタカウンタ法などがある。(1)は気泡核の画像から計測する直接法であるが、(2)(3)は電気信号から計測する間接法で検定が必要となる。気泡核分布の実測例を図12・3に示す。しかし、計測法にもよるが、前処理や真水と海水の違いなどにより核分布のの値には大きな差異が見られる。初生キャビテーションの相違の一つの原因は、これらの核分布から求まる臨界半径以上の核個数に差があるためで、核分布などの試験液の性状をあらかじめ特定しておくことが重要となる。
