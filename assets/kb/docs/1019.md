# 12・2・3 初生則と寸法効果

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

気泡核が物体近くの低圧場に流入し、成長してキャビテーションが発生するとき、式(12・1)では>0の条件下にある。式(12・4)の初生則は最も基本となるものであるが、実際では12・4節で分類されているように、種々の発生形態に対応した半経験的な初生則が試みられている。図12・3 気泡核分布4―137a. バブルキャビテーション(bubble cavitation)気泡核の個数が少なく、その半径が小さくなると、初生には気泡核の周囲圧が表面張力に打勝つよう負の引張圧力として作用することが必要となる。c. 付着キャビテーション(attached cavitation)気泡核分布にはほとんど影響せず、境界層の特性に依存する。このようにキャビテーションの発生は主流中の気泡核分布はもとより、発生領域の境界層の特性などの種々の要因に影響され、初生の正確な予測は難しい課題である。一方、同一形状の物体であっても、模型と実物のように、その寸法が異なることによってσの値が相違する。これは、液中の核分布が同一であっても、大きな物体ほど多くの気泡核が発生領域に流入するため、初生の確率が高まるからである。一般に、寸法の違いによる影響も含めて、式(12・4)の初生則からのずれは寸法効果(scale effect)と呼ばれる。したがって、多くの初生実験データから寸法効果を考慮し、より一般性の高い初生則を見出す試みも続けられている。ここで、:経験定数、添字0は参照値を表す。
