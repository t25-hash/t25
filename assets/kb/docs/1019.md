# 12・2・3 初生則と寸法効果

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

気泡核が物体近くの低圧場に流入し,成長してキャビテーションが発生するとき,式(12・1)では>0の条件下にある.したがって,初生時には気泡核が目に見えるほどの気泡径に成長するので,욋->0,すなわちσ<-욣の状態になっていると考えられる.ここで,욣=-12ρは圧力係数である.初生が圧力最低点の욣윯읉윰で生じるとすれば,σ욡=-욣윯읉윰(12・4)さらに,液中に溶存しているガス濃度が高い場合には,σ욡=-욣윯읉윰+αβ12ρ(12・5)ただし,α:液中のガス濃度,β:ヘンリー定数,=0.4-1.0は修正係数である.式(12・4)の初生則は最も基本となるものであるが,実際では12・4節で分類されているように,種々の発生形態に対応した半経験的な初生則が試みられている.図12・3 気泡核分布4―137a. バブルキャビテーション(bubble cavitation)気泡核の個数が少なく,その半径が小さくなると,初生には気泡核の周囲圧が表面張力に打勝つよう負の引張圧力として作用することが必要となる.σ욡=-욣윯읉윰-2σ욦12ρb. 渦キャビテーション(vortex cavitation)流れ場によって種々の成因の渦があるので一般化は難しいが,一例として流れに垂直な円板後流中のはく離せん断層に生じる渦キャビテーションに対しては,σ욡=-욣윯읉윰+(=0.0036,:レイノルズ数)となる.c. 付着キャビテーション(attached cavitation)気泡核分布にはほとんど影響せず,境界層の特性に依存する.σ욡=-욣욦(層流はく離型:욣욦ははく離点の圧力係数)σ욡=-욣욧욥(乱流遷移型:욣욧욥は遷移点の圧力係数)その他,乱れによる気泡核分布の変化や圧力変動などの動的な影響も重要とされる.このようにキャビテーションの発生は主流中の気泡核分布はもとより,発生領域の境界層の特性などの種々の要因に影響され,初生の正確な予測は難しい課題である.一方,同一形状の物体であっても,模型と実物のように,その寸法が異なることによってσ욡の値が相違する.これは,液中の核分布が同一であっても,大きな物体ほど多くの気泡核が発生領域に流入するため,初生の確率が高まるからである.一般に,寸法の違いによる影響も含めて,式(12・4)の初生則からのずれは寸法効果(scale effect)と呼ばれる.したがって,多くの初生実験データから寸法効果を考慮し,より一般性の高い初生則を見出す試みも続けられている.一例として,物体の代表寸法,液体の動粘性係数ν,主流速度,主流の乱れ度を考慮したσ욡の経験式は,σ욡=νν1+1+(12・6)となる.ここで,:経験定数,添字0は参照値を表す.
