# 12・2・4 熱力学的効果

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

一般に、ある液体が高温状態にあるほど熱力学的効果(ther-modynamic effect)が大きい。いま、気泡界面を通して周囲の液体から潜熱の供給を受け蒸発により気泡が成長するとき、界面温度は遠方の温度より低くなる。したがって、式(12・1)の욋は욋=욋=욋-ρΘ(12・7)となる。ここで、蒸気圧の差を表すρΘはクラジウス・クラペイロン(Clausius-Clapeyron)の式と、熱拡散および界面での熱収支を考慮して、近似的に式(12・8)で与えられる。Θ=∑dd(12・8)ただし、∑=ρ욋ρ욣αは熱力学的パラメータで、ρ욋,ρ:蒸気および液体の密度、:潜熱、욣,α:液体の定圧比熱および熱拡散率である。式(12・7)を用いれば、式(12・4)の初生則はσ욡=-욣윯읉윰-Θ12(12・9)となる。通常∑の値は高温になるほど急速に大きくなるため、熱力学的効果によりσ욡は低下し、初生およびキャビテーションの発達は大幅に抑制される。
