# 12・2・4 熱力学的効果

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

一般に、ある液体が高温状態にあるほど熱力学的効果(ther-modynamic effect)が大きい。いま、気泡界面を通して周囲の液体から潜熱の供給を受け蒸発により気泡が成長するとき、界面温度は遠方の温度より低くなる。ここで、蒸気圧の差を表すρΘはクラジウス・クラペイロン(Clausius-Clapeyron)の式と、熱拡散および界面での熱収支を考慮して、近似的に式(12・8)で与えられる。通常∑の値は高温になるほど急速に大きくなるため、熱力学的効果によりσは低下し、初生およびキャビテーションの発達は大幅に抑制される。
