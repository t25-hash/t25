# 12・3・1 単一気泡の力学

> **章**: 第12章 キャビテーション
> **節**: 12・3 気泡の力学

無限に広がる非圧縮性液体中の球状気泡(spherical bubble)の非線形運動はレイリー・プレセットの式(Rayleigh-Plesset equa-tion)で表される。+32=1ρ-(12・10)ここに、:気泡半径、=dd,=dd,:時間、ρ:液体密度、:気泡から十分離れた位置での液体圧力、また4―138:気泡表面での液体圧力で、半径方向の力の釣合い条件から=욡-2σ욦-4μ(12・11)となる。ここに、욡:気泡内部圧力(=욋and/or )、욋:蒸気圧、:ガス分圧、σ욦とμは液体の表面張力係数と粘度である。式(12・10)から初期に最大半径윯율융の空洞[=0]が完全に消滅するレイリーの崩壊時間(Rayleigh’s collapse time)はτ=0.91468윯율융ρ(12・12)となる。윯율융=10m,=0.1MPaでは気泡は約10sで崩壊する。式(12・10)のの項に時間依存の数値が代入され、流れ場、音場中などの気泡の運動が調べられる。
