# 12・3・2 気泡運動に及ぼす圧縮性、気体拡散、熱拡散の影響

> **章**: 第12章 キャビテーション
> **節**: 12・3 気泡の力学

a. 圧縮性の影響気泡表面速度が音速に近づくと圧縮性の影響が重要となり、気泡エネルギーの一部は流体の圧縮仕事や衝撃波(shock wave)のエネルギーに変換される。このとき式(12・13)は±4%以内の誤差で実験値と一致する。また、同様なエンタルピー表現のギルモア(Gilmore)の式は部分的に気泡表面マッハ数の二次の項を含みマッハ数5まで数値解とよく一致するが、マッハ線からのゆがみが顕著となる衝撃波形成過程では解の精度は保証されない。図12・4に特性曲線法によるガス気泡リバウンド(rebound)直後の液中の圧力分布を示した。ここに、:半径方向距離、また気泡内ガスの初期圧力は=10MPa、比熱比=1.4.気泡最小時に最大2.5GPaの衝撃圧が発生し、気泡の運動が外向きに転じると液中に衝撃波が形成される。水中衝撃波は固体表面に繰返し作用して材料の疲労破壊の原因となる。ここに、:気体の拡散係数、ρ:気泡内のガス密度。不飽和度が大きくなると溶解時間は減少するが、キャビテーション気泡の運動周期に比べれば非常に遅い。c. 熱拡散の影響崩壊末期の気泡内部、高温液体中の気泡挙動など熱的影響が著しい場合には、気泡の運動方程式とエネルギーの式を解く必要がある。しかし、液中の温度境界層が薄い条件では、次に示す球対称非定常熱伝導式のゼロ次解が利用できる。式(12・17)を用いて気液界面での蒸発・凝縮を考慮した圧力波動が検討され、また気液両相の質量、運動量およびエネルギーの各保存式を直接数値的に解いて気泡運動に及ぼす内部現象の影響が明らかにされている。
