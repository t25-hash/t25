# 7・4・2 マイクロマシン

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・4 マイクロ熱工学

マイクロマシン（MEMS:micro- electro-mechanical systems ）
の応用分野として，光通信機器，工業・医療用センサ，マイクロ生
化学分析デバイス，電子デバイスの冷却システム，伝熱・流体制
御，マイクロ無線機器などが考えられているが，その中で熱工学と
マイクロマシンの接点としては，以下の三つが挙げられる．
(１)マイクロマシン周りの伝熱，流れ現象の解析を通じて，熱工
学の知識をマイクロデバイスの設計に役立てる．
(２)マイクロセンサ，アクチュエータを，既存の伝熱・熱流体機
器の診断，熱流動の制御に応用する．
(３)両者の融合により，新たな機械システムを創造する．
スケールが小さくなるに従って粘性力，界面張力などの表面力の
影響が増大するだけでなく，希薄気体効果，ブラウン運動，電気二
重層，温度こう配や電位差による表面張力駆動の対流などが顕在化
するため，通常スケールとは質的にも異なったデバイスも提案され
つつある．以下では，伝熱関連のマイクロデバイスについての例を
示す．

ロセンサは，流速，流量，壁面せん断応力の測定を目的として多く
試作されている ．ヒータ部を微小にすることによって熱容量が
減少するため，高い周波数応答が実現できる．ヒータの材質として
は，多結晶シリコン，白金などが用いられる．図7･9に，マイクロ

真空キャビティになっており，基板への熱損失を防ぎ，消費電力の

b.熱式アクチュエータ 通常スケールでは伝熱現象の時間応
答は遅いが，スケールが小さくなることによって熱容量が極めて小
さくなるため，マイクロアクチュエータの動作原理として伝熱を用
いたものも多く提案されている．図7･10に，熱変形を用いたフラ
ップ型アクチュエータの例を示す ．ヒンジ部のV溝にポリイミ
ドを埋め込み，ポリイミドの大きな熱収縮率と熱膨張率を利用する
ことによって，三次元構造の組立，動的な大変形が可能であり，マ
イクロスケールでの搬送やマイクロロボットの歩行などへの応用が
検討されている．図7･11に，表面張力の温度変化を利用した気泡
駆動の光スイッチの例を示す ．光導波路に基板と屈折率の等し
い液滴を極微量封入した微細流路を形成し，マイクロヒータで温度
こう配を変化させることにより液滴を移動させ，入射光を透過・反
射させる．時間応答は100msであり，全反射を利用するため光の
損失が少ない．そのほかに，表面張力の電場依存性を利用したアク
チュエータ，液体から気体，あるいは固体から液体への相変化を用
いたアクチュエータなどが提案されている．
