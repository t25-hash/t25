# 12・4・1 発生形態と発達過程

> **章**: 第12章 キャビテーション
> **節**: 12・4 キャビテーション流れの諸特性

a. 発生形態キャビテーションの発生形態に応じて種々の分類が可能である.発生形態は発生機構や気泡形態そして気泡崩壊時の衝撃強さとも関連し重要である.キャビテーション気泡は,静的あるいは動的な方法で液体の圧力を低下することにより,液体中に存在する微視的スケールのキャビテーション核から成長・発生する.温度上昇による気泡生成が主である沸騰とは熱的作用の有無の点から区別されるが,(高温液体など)その境界があいまいな場合も多い.圧力低下による気泡発生現象は,また,1蒸気的キャビテーション(vaporous cavitation)と2気体的キャビテーション(gaseouscavitation)とに分けられる.前者は,おもに液体そのものが気泡核中へ蒸発することにより発生し,後者は,おもに液中に溶解している気体が気泡核中へ拡散することにより発生する.前者を単にキャビテーションと呼ぶこともあるが,一般に気泡内部は気体と蒸気との混合気体から成り複雑である.キャビテーションはさらに次の視点から分類される.(1) diffusion)の役割が指摘される.このうち,流れ中で生じる流力的キャビテーションは,流体工学上特に重要であり,表12・1に示すような流れの構造と関連する低圧場において発生する.キャビテーション気泡はそれぞれの圧力場に応じた発生形態をとり,以下のように分類される.代表的な様相を図12・7に示す.(1) バブルキャビテーション(bubble cavitation)代表例は移動型バブルキャビテーション(traveling bubblecavitation)で,物体境界面に沿う低圧領域において,流れとともに流入した気泡核から成長し,ほぼ球形あるいは半球形の気泡を形成しながら下流へ移動・消滅する.(2) 渦キャビテーション(vortex cavitation)流れ場中に形成された渦核が低圧になることにより発生する.翼端渦キャビテーション(tip cavitation)のように安定なものからせん断層キャビテーション(shear cavitation)のように過渡的(transient)なものまである.主流方向に対して横渦型(trans-verse-type)および縦渦型(longitudinal-type)のような分類もできる.後者は比較的安定である.これらの代表例は円柱後流に生じるカルマン渦キャビテーションなどに認められる.(3) 付着キャビテーション(attached cavitation)見掛け上,物体面に付着あるいは一定位置に発生しているような形態を指す.境界層状態と密接に関連がある.(i) 層流はく離泡型境界層のはく離・再付着に伴い形成されるはく離泡内において界面を通しての拡散あるいは蒸発作用によりキャビテーションが発生・成長する.フィンガ状の形状をとる場合があり気体的側面が強い.(ii) cavitation)多数の微小なキャビテーション気泡が集団(キャビテーションクラウド)としてトランジエントな挙動を示すもの(大規模な渦構造を核にする場合もある).付着キャビテーションから周期的に放出されるものなど,高衝撃性の観点から特に注目されている.b. 発達過程キャビテーション数σの低下とともにキャビテーションは発達していく.まず,キャビテーション気泡の発生に伴いキャビテーションノイズあるいは衝撃加速度などの値が急に立上がる(初生の判断はストロボ光などを利用した肉眼による場合もある).この付近が1初生キャビテーション(incipient cavita-tion)段階であり,キャビテーション事象は当初間欠的に現れるが,キャビテーション数の低下とともに連続的に計測されるようになり,2サブキャビテーション(subcavitation)段階となる.この段階では流れの局所構造はキャビテーションの影響を受けるが,流れの大局的構造は質的には大きく変化しない.次いで,3遷移(transition)段階そして4スーパーキャビテーション(super-caviation)段階と進む.34における流れ場の特性は,キャビテーションの存在によってキャビテーションなし状態に比して大きく変化し,キャビテーション特有の流れ状態となる.以上のキャビテーション発達過程とそれに伴うキャビテーション特性変化を模式的に図12・8に示す.なお,初生キャビテーション,部分キャビテーション,スーパーキャビテーションと分ける場合もある.
