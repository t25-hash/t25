# 12・4・2 翼型のキャビテーション特性

> **章**: 第12章 キャビテーション
> **節**: 12・4 キャビテーション流れの諸特性

a. サブ(および遷移)キャビテーション翼型の形状,迎え角そして流体力学的諸条件に依存する.特に図12・9に示すように境界層特性(層流はく離,乱流はく離,再付着の有無,乱流遷移など)や気泡核分布は他の流体機器の場合と同様にキャビテーション特性や発生形態と密接な関係をもつ.各種翼型のキャビテーション特性については揚・抗力係数をはじめ系統的な研究がなされている.図12・8に模式的に示したように揚力係数が初生後σの低下とともにさらに増加する場合もあるが,これは翼背面でのキャビテーション域の広がりにつれて負圧領域が広がるためである.σのさらなる低下とともにキャビティはく離状態となり,通常の翼型性能は大幅に低下する.振動,騒音,壊食の問題も重要であり,特にシートキャビテーションから放出されるキャビテーションクラウドとの関連が指摘される.b. スーパーキャビテーション通常,翼において強いキャビテーションが避けられない場合,比較的定常性の高いスーパーキャビテーション状態での使用が望ましい場合がある.不安定なキャビテーション状態である遷移段階でも,外部から空気を強制的にキャビテーション発生域に通気(ventilation)することにより類似の流れ状態に移行することができる.翼型背面全体を積極的に空洞で覆い,かつ腹面でのキャビテーション発生を抑えることにより,腹面正圧・背面蒸気圧の状態で超高速(低キャビテーション数)域での作動を可能としたものをスーパーキャビテーション翼と呼ぶ.この翼型のにはその作動原理上,≦1+σの上限がある.好適な翼型形状について種々の提案がなされているが,基本的には後縁側に腹面圧のピークを近づけ腹面形状をS 字形にすることが肝要であり,腹面正圧,背面空洞厚み正の必要条件とともに翼前縁強度を考慮しなければならない.c. 翼特性の数値シミュレーション部分キャビテーションあるいはスーパーキャビテーション状態における翼まわりの流れ特性は,ポテンシャル理論に基づき古くから自由流線理論や特異点法を用いて解析されてきた.この際,空洞,特に空洞後端のモデル化および空洞先端位置の確定が必要であり,空洞内圧一定の仮定のもとに種々の非線形・線形モデルが提案され翼特性が解析されている.一方,近年の数値流体力学の進展により,なるべく実際のキャビテーション流れに近い状態で数値的に解こうとする試みが盛んになりつつある.キャビテーション流れは液体と気体が混ざり合った高速の気液二相流であり,相変化を伴う流体のモデル化(擬似単相媒体モデル,気泡流モデルなど)が必要である.
