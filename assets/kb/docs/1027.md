# 7・4・4 マイクロ熱交換器（micro-heat exchanger ）

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・4 マイクロ熱工学

MEMS， -TASなどマイクロ機器の開発が進展する一方で，そ
の周辺機器である電源，駆動制御系，熱交換器等は依然として大き
いままであることが多く，これら周辺機器の小型化の要求は大き
い．熱交換器やヒートシンクの場合，小型化により熱伝達率が大き
くなること，また熱伝導距離を小さくすることができる可能性があ
ることから，高性能な熱交換器が期待できる．また，マイクロ熱交
換器は冷媒等の使用量も少なくて済むことから，万が一，リーク事
故が発生したとしても危険性は少ないなどのメリットが考えられ
る．CPUなどの電子デバイスの冷却における除熱の厳しさを評価
するパラメータとして，デバイスと周囲環境との最大温度差
と，発熱密度 で定義される熱抵抗（thermal resistance）＝
が用いられる．小さい温度差でできるだけ多くの熱交換を行
うという点を考えると，この熱抵抗というパラメータはマイクロ熱
交換器の性能の指標としても有効である．
マイクロ熱交換器としては，マイクロチャネル型や噴流型がこれ
までに製作されており，ここではマイクロチャネル熱交換器につい
て述べる．
a.マイクロチャネル熱交換器 マイクロチャネル熱交換器の
熱抵抗は，熱伝達によって生じる伝達熱抵抗 と隔壁の伝導熱
抵抗 ，作動流体の温度上昇により実効的な温度差が減少する
ことで生じる容量熱抵抗 の和で表すことができる．マイクロ
チャネル熱交換器の隔壁の厚さは数百 m程度であることが多く，
一般的には伝導熱抵抗は伝達熱抵抗よりもかなり小さい．容量熱抵
抗は作動流体の流量とともに減少するが，大きさとしては伝達熱抵
抗と同じ程度である．伝達熱抵抗を小さくするためには，流路直径
を小さくすること，熱伝導率の良い流体を用いることが考えられ
る．容量熱抵抗を減少させるためには，流量を増やすことが挙げら
れる．以上がマイクロチャネル熱交換器で小さい熱抵抗を達成する
ための指針であるが，流路直径を小さくすると必要となるポンプ仕
事および圧力が大きくなってしまうので，与えられた条件や制約に
よって，どの程度まで流路直径を小さくするかが決まることにな
る．例えば，水を作動媒体とした場合には伝達熱抵抗 を0.1
K･cm / W程度まで小さくすることが可能である．図7･17は，水
冷マイクロチャネルと他の代表的な冷却方法の適用領域を比較した
ものであり，小さい熱抵抗が要求される場合にはマイクロチャネル
冷却が最も優れていることがわかる ．

ない場合のマイクロチャネルの伝熱特性としては，以下の実験整理
式が提案されている（図7･18) ．

ここで， ：流路水力直径， ：溝深さ， ：溝幅， ：溝の間
隔である．一方，流動特性としては，流量が小さい層流領域におい
てはマクロな配管やベント部における圧力損失の整理式を用いて予
測可能であり，流入部の圧力損失は極めて小さいが，流量が多くな
ると入口部において流れのはく離が発生することにより，予測値よ
り圧力損失が増大することがある．ヘッダ部の最適形状に関して
図7･17各冷却手法の適用範囲
は，圧力損失だけでなく流量配分の均一化の観点からも今後の検討
が必要である．
c .沸騰相変化を伴う流れの伝熱・流動特性 熱流束が増加し
た場合にはチャネル内で沸騰が生じ，表面張力の効果が顕著に現れ
て，スラグ流と呼ばれる流動様式が多くなる．この場合の伝熱およ
び流動特性は単相の場合とは大きく異なり，沸騰により圧力損失の
増大が顕著となる．この圧力損失のおもな原因は，沸騰により気体
が急激に膨張することによって，液相が急加速されるときの慣性力
である．図7･19は沸騰相変化を伴うマイクロチャネル流れの実験
から得られた流量，圧力損失および熱流束の関係の一例であ
る ．沸騰によって圧力損失は急激に増大するが，伝熱特性はそ
れほど良くはならないこと，流路途中でドライアウトが生じた場合
には，下流における熱交換性能は大きく低下することを考慮する
と，マイクロチャネル熱交換器は液単相の条件で用いたほうが有利
である．
