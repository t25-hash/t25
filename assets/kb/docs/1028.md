# 12・5・1 壊食(erosion)の機構

> **章**: 第12章 キャビテーション
> **節**: 12・5 キャビテーションによる壊食と騒音

キャビテーション(cavitation)気泡は、気泡崩壊時にマイクロジェットや衝撃波に起因する衝撃力を生じる。図12・10にその概要を示す。一般にキャビテーションは、水中の微細な気泡などのキャビテーション核が高速・低圧領域において、キャビテーション気泡として発達する。速度が低下して圧力が回復すると、キャビテーション気泡が収縮してやがて崩壊する。キャビテーション気泡が固体壁近傍で崩壊する場合には、気泡の固体壁の反対側が固体壁側よりも変形しやすいためにへこみ、このへこみ部が1500m/s以上の流速をもつマイクロジェットとして発達して固体壁をたたく。またキャビテーション気泡が、キャビテーション核となる微細な気泡から発達しているときには、気泡内部の圧力が低いために水中から気体を取り込むため、気泡崩壊時には気泡内にある気体のために気泡が収縮した後に崩壊せずに再膨張する。この気泡の収縮・再膨張がごく短時間に行われるので、再膨張により衝撃波を生じる。この衝撃波が固体壁に衝撃力を与える。これらのマイクロジェットや衝撃波による衝撃力が、流体機械の材料表面に作用して壊食を生じる。キャビテーションは、そのタイプにより壊食率(erosion rate、図12・10 キャビテーション気泡の発達と衝撃力発生の概要4―142流体工学図12・12 壊食率の経時変化質量損失の時間変化率)が大きく異なる。図12・11には、遠心ポンプに壊食が生じる場合のキャビテーションの様相と壊食面の様相を示す。流体機械に致命的損傷を生じるキャビテーションは、単一の気泡ではなく、図12・11に示すように、微細な気泡が集まったキャビテーションである。微細な気泡の集積には局所的な渦流れが関わっており、高いキャビテーション衝撃圧を生じるU 字形渦や縦渦が観察されており、渦キャビテーション(vortexcavitation)が高壊食性キャビテーションである。このような高壊食性渦キャビテーションは、キャビテーションが発生しても機器や翼の性能に影響を与えないサブキャビテーション領域と、キャビテーションが機器の性能に直結するスーパーキャビテーション領域の間の遷移領域において顕著に発生する。なお、単一のキャビテーション気泡が崩壊した場合の衝撃圧に比べて、同体積の微細な気泡から成るキャビテーションが崩壊した場合のほうが衝撃圧がはるかに大となる。上記の衝撃力による材料の壊食においては、単発の衝撃力により材料が質量損失する壊食を生じる場合と、衝撃力が繰返し負荷されて疲労破壊的に壊食する場合があり、主として疲労により壊食する場合が多い。図12・12に、疲労破壊的に壊食が発達する場合の質量損失の経時変化を示す。壊食初期は、衝撃力により塑性変形が繰返されるだけで質量損失は生じないので、潜伏期と呼ばれる。壊食が発達すると、質量損失が生じて時間とともに壊食率が加速的に増大する加速期、壊食が定常的に進行する定常期を経て、壊食率が減少する減速期に至る。したがって、壊食の過程により壊食率が大幅に異なるので、材料の耐壊食性を推定する場合には壊食期を把握する必要がある。壊食率は、計量ごとに求める瞬間壊食率と、全損失量から求める累積壊食率の二つを定義できるが、ASTM G 134:1995では累積壊食率を使用して、累積壊食率が極大を示す最大累積壊食率を材料の壊食性を示す壊食率とすることを推奨している。以上のごとく、キャビテーション気泡の崩壊に伴う衝撃力は壊食を生じるが、この衝撃力をショットピーニングのごとく機械材料の高強度化に用いる試みも行われている。
