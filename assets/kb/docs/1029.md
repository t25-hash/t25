# 12・5・2 壊食に及ぼす諸因子とその予測

> **章**: 第12章 キャビテーション
> **節**: 12・5 キャビテーションによる壊食と騒音

壊食には、おもに以下のような流体力学的因子と材料学的因子が関わっている。a. 流体力学的因子流体力学的因子としては、キャビテーション流れの支配パラメータであるキャビテーション数、主流の流速、流体機器の寸法、キャビテーション気泡が崩壊する領域の静圧などがある。キャビテーション数が小さくなるほど、キャビテーションが発達、すなわち高壊食性渦キャビテーションが発達するために、衝撃力が大きくなり壊食が増加する。ポンプなどの流体機械においてはNPSH がこれに相当する。また12・5・1項で述べたように、サブキャビテーション領域からスーパーキャビテーション領域の遷移領域において壊食が極端に激しくなることがあるので注意を要する。同一のキャビテーション数ならば、キャビテーション長さなどは相似となるが、主流の流速や機器の寸法が大きくなるほど渦キャビテーションが発達して、壊食率は大きくなる。流速や機器の寸法については、以下のようなべき乗則が提案されている。キャビテーション気泡の崩壊領域の静圧は、大きくなるほど衝撃力が大きくなるので、静圧が高い領域においてキャビテーションが発生する場合には、壊食の危険性が大となる。以上のような流体力学的因子を個々に把握してキャビテーションの壊食率を予測する方法とは別に、キャビテーションの強さを直接計測して、壊食率を定量的に評価する方法がある。キャビテーションの強さの計測には、圧電材料を用いた衝撃力計測センサによる方法、材料表面に生じた塑性変形ピットの深さや大きさによる方法などがある。また圧力に応じて発色する感圧紙や、ペイントにより壊食が発生する領域を推定する方法もある。b. 材料学的因子材料学的因子としては、引張強さ、硬さ、ひずみエネルギーが考えられる。引張強さσが大きくなるほど、また硬さが大きくなるほど耐壊食性(例えば、壊食率の逆数など)が大きい傾向がある。また、ひずみエネルギー(1/2×引張強さ×伸び)が大きくなるほど、耐壊食性が比較的良い相関を示すという報告もある。材料の耐壊食性を定量的に予測するためには、キャビテーション噴流式壊食試験(erosion test)を用いて、各材料について壊食の加速試験を行う必要がある。本法は、従来、振動式試験装置を用いてキャビテーションを発生させていた方法に比べて、流体機械に壊食を生じるキャビテーションと同タイプの渦キャビテーションにより壊食を行える、強力なキャビテーションを容易に発生できるなどの利点がある。
