# 12・5・3 騒音(noise)の発生と特徴

> **章**: 第12章 キャビテーション
> **節**: 12・5 キャビテーションによる壊食と騒音

キャビテーションが発生すると、キャビテーション気泡の崩壊に伴う騒音が生じる。キャビテーションによる騒音は、個々のキャビテーション気泡の崩壊に伴う高周波ノイズと、12・5・1項で述べた遷移領域に起因する低周波ノイズがある。したがって、大気圧下で=1mmのキャビテーション気泡が崩壊するときはτは約0.1msとなり、10kHz 程度の騒音を生じることになる。実際に流体機械に生じるキャビテーションは、さらに気泡径が小さい場合が多いので、キャビテーションによる騒音は高周波であることが特徴である。逆に、高周波ノイズからキャビテーションの発生を検知することも可能である。騒音計により大気中で計測したり、水中マイクロホンにより水中を伝ぱするノイズを計測して、キャビテーションの発生を検知できる。なお、キャビテーション気泡は、騒音源となるばかりでなく、気液界面で騒音を減少させる効果も併せ持つので、キャビテーションの強さを計測する場合には注意が必要である。また、高周波振動加速度計やアコースティックエミッション(AE)センサを用いてキャビテーションの発生の程度を検出することも可能である。キャビテーションにより数十Hz の低周波ノイズを生じることもある。これは遷移領域において、サブキャビテーション状態とスーパーキャビテーション状態が、数十Hz の周期で交番的に生じることに起因する。この遷移領域では、騒音ばかりでなく振動も顕著になる特徴がある。
