# 7・1・3 圧縮性流れ（compressible flow）の特徴

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・1 圧縮性流れの基礎

流れの中に生じた微小な圧力変動などの擾乱は流れに相対的に流
れの上下流方向に音速で伝わる．したがって，7･2･5項の先細ノズ
ルの出口断面やラバルノズルのスロート断面のように，流れのある
断面で流速が音速に等しい，すなわちマッハ数が1であれば，その
断面より下流から上流に向かって伝わってくる微小擾乱はその断面
で上流への伝ぱ速度が零となり，それより上流へはさかのぼれな
い．このため，その断面より下流の圧力が変動してもその断面を通
る質量流量は一定に保たれる．このように，ある断面で流れのマッ
ハ数＝1の状態（臨界状態と呼ばれる）になることをチョーク
（choking，閉塞）という．チョークは等エントロピー流れにおい
て断面積が減少するとき，壁面摩擦によって全圧が減少するとき，
加熱によって全温度が増加するときなどに起こる．チョーク現象は
非圧縮性流れに見られない圧縮性流れの重要な特徴の一つである．
流れがある断面でチョークし，その下流で超音速に加速されると，
その流れが亜音速に減速される際には衝撃波が形成される．衝撃波
の発生も非圧縮性流れに見られない圧縮性流れの重要な特徴であ
る．
