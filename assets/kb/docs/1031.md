# 12・5・4 壊食および騒音の低減法

> **章**: 第12章 キャビテーション
> **節**: 12・5 キャビテーションによる壊食と騒音

壊食および騒音の低減法としては,壊食・騒音源であるキャビテーションの強さを弱める方法がまず考えられる.そのためには,キャビテーションの発達領域を小さくし,キャビテーション気泡や高壊食性渦キャビテーションの寸法を小さくして衝撃力を弱めることが有効である.具体的にキャビテーションの発達領域を小さくするためには,キャビテーション数やNPSH を大きく,すなわちキャビテーションが発生する領域の静圧を大きくする.特に,遷移領域で激しい壊食が生じている場合には,サブキャビテーション領域,もしくはスーパーキャビテーション領域に移行させて壊食を低減させることも有効である.上流側で発達・崩壊したキャビテーションが,下流のキャビテーションに豊富なキャビテーション核(cavitation nuclei)を供給してキャビテーションを発達させ,激しい壊食を生じる場合がある.したがって,壊食や騒音が予測される領域の上流でのキャビテーションの発生は,発生領域が小さくても極力避ける必要がある.キャビテーションが発達する領域に空気を吹込む通気(ventila-tion)により,気泡崩壊時の気泡内のガス圧を高めて気泡の崩壊衝撃力を弱め,壊食を回避・低減する方法もある.壊食に対しては,耐壊食性が強い材料を使用する方法も有効である.材料の耐壊食性は,前述した噴流式壊食試験装置により実験的に求める方法が最も信頼性が高い.素材を強くするばかりでなく,耐壊食性のコーティング材なども市販されている.また一般的には,引張強さが大,あるいは硬い材料ほど耐壊食性がある.騒音については騒音の伝ぱを減衰させる方法がある.通気はキャビテーション気泡の崩壊衝撃力を弱めるばかりでなく,気泡により騒音の伝ぱを低減させる効果もある.文献(1) Young,F.R.,Cavitation,(1999),Imperial College Press.(2) Barnaby,S.W.and Parsons,C.A.,Trans.Inst.Nav.Arch.,38(1897),232.(3) Rayleigh,L.,Phil.
