# 13・1・1 フルード数(Froude number)

> **章**: 第13章 自由表面のある流れ
> **節**: 13・1 開水路の流れ

開水路流れは、管路流や境界層流と同じ壁面乱流に分類され類似した挙動を示すが、自由水面をもつため流れの特性によって水深が時空間的に変化し、複雑である。流れの支配パラメータは、レイノルズ数以外にフルード数が重要である。は重力ρに対する慣性力ρの比の平方根であって、式(13・1)で定義される。ここで、は水深、はバルクの断面平均流速である。≡=, ≡(13・1)は水面の微小かく乱の伝ぱ速度であり、フルード数はマッハ数と類似している。
