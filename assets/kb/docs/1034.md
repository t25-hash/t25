# 13・1・3 比エネルギーと比力

> **章**: 第13章 自由表面のある流れ
> **節**: 13・1 開水路の流れ

河床から測ったエネルギーを比エネルギー,運動量に圧力を加えて拡張された運動量を比力といい,開水路水理学の基本量である.≡α2+cos θ=α2+cos θ(13・2)≡β+12cos θ =β+12cos θ (13・3)ここで,:流量,:流水断面積(河川工学では河積という)で,≡である.長方形断面では=である(は川幅).θは河床こう配で,通常の河川ではcos θ1である.αはエネルギー補正係数あるいはコリオリ係数,βは運動量補正係数あるいはブシネスク係数と呼び,N-S 方程式にガウスの発散定理を適用して運動方程式の積分型を誘導する際に導入される.なお,乱流では,α1.1,β1.03程度であり,複雑な実務に直結しない基礎学習上ではαβ1と近似してよい.
