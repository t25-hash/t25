# 7・2・4 ショックトレーンと擬似衝撃波

> **章**: 第7章 圧縮性流体の流れ
> **節**: 7・2 一次元定常流

管路や通路に垂直衝撃波が形成されると，壁面の境界層との干渉
が起こる．この干渉は衝撃波上流のマッハ数 が大きいほど強
く， が約1.3以上の場合，この干渉により境界層のはく離が起
こる．図7･5はこのような場合の干渉の様子を模式的に示したもの
である．まず，図7･5(ａ)の約1.3＜ ＜約1.5では，はく離によ
って凹面状になった境界層外縁の部分から圧縮波が発生する．この
圧縮波は衝撃波を形成し，管中心部の衝撃波と交わるため，衝撃波
は境界層の近くで枝分かれした状態となる．これを衝撃波の分枝
（bifurcation）という．分枝の程度は衝撃波が強いほど激しく，
が約1.5以上になると，図7･5(ｂ)に示すように，分枝した衝撃波
の下流に新たに一つあるいはそれ以上の衝撃波が形成される．この
ような一連の衝撃波はショックトレーン（shock train）と呼ばれ
る．
一般にショックトレーンの下流端断面では超音速流れと亜音速流
れが混在しており，管路が長い場合にはこの両者が混合するため，
図7･6に示すように，ショックトレーンの下流では静圧がある距離
にわたり増加し続ける．したがって，このような流れでは静圧の回
復はショックトレーン領域とそれに続く混合領域を通じて行われ
る．ショックトレーンの先端から混合領域の後端までの干渉領域全
図7･6 断面積一定のダクト内の垂直衝撃波と境界層の干渉領域における
壁面および中心線に沿う静圧分布の概要
体は，流れを超音速から亜音速に減速するという意味で1個の垂直
衝撃波と同じ役割を果たしており，擬似衝撃波（pseudo- shock
wave）と呼ばれる．超音速ノズルや超音速ディフューザ，超音速
エジェクタ，あるいは超音速圧縮機の翼間流路など，超音速の内部
流れが亜音速に減速される際には衝撃波が発生するが，このような
流れの現象は発生する衝撃波を垂直衝撃波ではなく，擬似衝撃波と
考えることによって詳しく説明される．
