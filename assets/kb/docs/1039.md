# 13・1・8 開水路の乱流構造

> **章**: 第13章 自由表面のある流れ
> **節**: 13・1 開水路の流れ

上述までは開水路水理学のマクロな扱いであるが、その乱流構造も境界層乱流と同程度に解明されている。開水路の内層特性は境界層や管路の内層とほぼ同じであり、外層で自由水面の影響が現れる。式(13・13)は滑面乱流の式で、粘性底層およびバッファ層(≦30)の分布も境界層のものと同じである。開水路のカルマン定数と積分定数は、禰津・ロディ(Nezu-Rodi)によるレーザ流速計から=0.412,=5.29 と得られ、境界層・管路とほぼ同じで、普遍特性がある。粗面乱流ではニクラゼの対数則に置換えればよい。後流強度係数Π はレイノルズ数の関数で、低レイノルズ数ではΠ=0であり、対数則が水面まで成立する。高レイノルズ数ではΠ=0.2となり、対数則からずれる。Π 値は、圧力こう配や二次流によっても変化する。このため、乱れの非等方性すなわち渦度発生項′-′が境界層や管路流に比べて大きくなり、縦渦(二次流セル)が発生しやすい点である。縦渦が発生すると、最大流速点が降下したり、河床が移動床のとき縦筋が発生し、流れが三次元性となる。このような三次元乱流構造に関しては文献(5)を参照していただきたい。開水路流れの組織乱流構造に関しても、ある程度解明されている。内層にはバースティング現象が存在し、乱れエネルギーの発生機構になっている。外層には河床波などの影響で大規模なコルク渦が発生し、これが水面まで発達し、ボイル渦になる。浮遊土砂はボイル渦によって水面まで間欠的に浮上する。河川のさまざまな渦構造はこのような組織乱流によって構成されている。液膜流れ(liquid film flow)とは液体が固体壁に沿って薄い膜状に流れる状態をいい、境界層近似が成り立つ流れである。また、液膜の内部に乱れが存在すれば、その情報は液面の凹凸として現れる程度に薄い流れともいえる。
