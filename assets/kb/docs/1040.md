# 13・2・1 落下液膜流れ

> **章**: 第13章 自由表面のある流れ
> **節**: 13・2 液膜流れ

図13・1に示すような傾斜平板上の液膜流れ、あるいは垂直円管の壁に沿って落下する液膜流れでは、壁面と気液界面におけるせん断力の和と重力とが釣合ったときに流れは平衡状態に達する。ここに、Γ:単位幅当たりの流量、θ:平板の傾斜角、ν:動粘性係数を表す。したがって、十分小さなレイノルズ数においても液面には二次元的な波動が生じ、これが下流に向かって増幅する。垂直円管に沿って落下する液膜の、十分下流(>1.5m)における流れを例にとると流動状態はおおむね以下のように区分できる。また、乱流域の時間平均速度分布は乱流境界層におけるカルマンの一般速度分布にほぼ一致する。
