# 13・2・2 水平平板上の液膜流れ

> **章**: 第13章 自由表面のある流れ
> **節**: 13・2 液膜流れ

図13・2のように、円形の液体噴流が水平に置かれた平板に落下すると、液体は放射状に流れて環状の跳水(hydraulic jump)が生じる。跳水はフルード数が2~6となる半径位置で生じる。跳水に至るまでの射流域の液膜が層流であると仮定すると、(1)よどみ点流れの領域、(2)ブラジウス(Blasius)型境界層の発達する領域、(3)速度分布がブラジウス型から相似形へ変化しながら発達して液面に達するまでの領域、および(4)液膜全体が相似形速度分布をもって流れる領域に区分できる。ここに、は母数sin75°のヤコビ(Jacobi)のだ円関数、は1.402である。>0.3155における放射状液膜流れでは半径の増加とともに局所のレイノルズ数は減少する。しかし、>80000のときには下流へ流れる途中で、液膜内部のかく乱の増幅により平板境界層と同様な層流から乱流への遷移が生じる。乱流遷移(laminar-turbulenttransition)が生じても境界層と同様な乱流が維持されるのは遷移点下流のごく狭い領域であり、下流に向かって再層流化が進行し、乱れは緩慢になる。
