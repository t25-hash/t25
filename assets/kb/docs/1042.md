# 13・2・3 回転円板(rotating disk)上の液膜流れ

> **章**: 第13章 自由表面のある流れ
> **節**: 13・2 液膜流れ

液噴流が回転している円盤の中心に落下すると,放射状に流れる液膜が形成される.このような流れではよどみ点近傍からねじれ型境界層が発達し,境界層が液面に達したあとは液膜全体が一種のねじれ型境界層として流れる.層流の場合にはねじれ型境界層の速度分布を多項式近似し,運動量積分方程式を用いれば容易に計算することができる.これについては文献(12)を参照されたい.半径≫4πνωでは液膜は十分に薄くなり流れは準平行流とみなせる.ここに,ωは円盤の角速度を表す.このとき,液膜は遠心力のみに支配され,液膜厚さδ,半径方向速度分布,および接線方向速度はδ=2πνω(13・29)=ωδνδ-1δ(13・30)=ω(13・31)で与えられる.
