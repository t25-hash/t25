# 13・3・1 表面波の基礎式

> **章**: 第13章 自由表面のある流れ
> **節**: 13・3 表面波

水深に比べて波の振幅がはるかに小さく、波動が水面付近のある深さにしか及ばないような波を表面波(surface wave)と呼ぶ。この表面波に対しては、粘性の影響を無視した非圧縮性完全流体としての取扱いがなされる。また、自由表面(=ζ)では、圧力が大気圧に等しく、表面張力も無視できるとすると、一般化されたベルヌーイの式[4・2節の式(4・6)参照]から式(13・34)が得られる。式(13・33)(13・36)の水底および自由表面での境界条件のもとで、式(13・32)を解けば、Φから流速が、式(13・34)より自由表面形状が決定される。しかし、式(13・36)は非線形方程式であるから解析解が求まらない。なお、波長が数cm以下、周期が約0.1s以下のさざ波の場合のように表面張力の効果が重要となる場合は、式(13・36)の左辺に表面張力に伴う圧力の実質微分項が付加される。
