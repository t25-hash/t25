# 13・3・4 流速と運動軌跡

> **章**: 第13章 自由表面のある流れ
> **節**: 13・3 表面波

特定の流体要素(水粒子)をラグランジュ的に追跡する場合,水粒子が第一近似として-平面上での初期位置,からオイラー速度[式(13・39)より求められる]で動いたとすると,微小振幅波に対する水粒子の軌跡は+-sinh+sinh-cosh+sinh=1(13・45)で示すだ円形で与えられる.また,波の進行方向が図13・3に示す方向の場合には,水粒子は図13・3(b)に示すように時計方向に回る.このように,有限水深をもつ波の場合は,水粒子はだ4―148流体工学円軌道を描くが,深水波の場合は∞であるから式(13・45)より半径요の円軌道を描く.特定粒子の移動速度として位置の変化による速度変化をも考慮した第二次近似を用いると,流体粒子の時間平均速度の方向成分は周期関数となるため零となるが,方向成分は욧=σcosh2+2sinh(13・46)であり零とはならない.これは,有限振幅波の場合に,水粒子の軌道は式(13・45)に示すだ円軌道として閉じたものになるのではなく,욧に乗って方向に漂流することを意味している(ストークスドリフトと呼ばれている).したがって,水粒子の軌道は図13・3(c)に示すように閉曲線ではない栗の実形の軌道を描く.この욧のことを質量輸送速度と呼ぶ.
