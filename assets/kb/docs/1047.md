# 13・3・5 波のエネルギーとエネルギー輸送

> **章**: 第13章 自由表面のある流れ
> **節**: 13・3 表面波

1波長分の微小振幅波がもつポテンシャルエネルギーは式(13・38)を用いて=12ρζd=14ρλ(13・47)となる。また、1波長分の波がもつ運動エネルギーは=d욪욠ρ2+d=14ρλ(13・48)となりと同じ値をもつ。よって、単位水平面積当たりの全エネルギー密度は=+=12ρ(13・49)となる。なお、表面張力波の場合は、=12(13・50)で与えられる。波のエネルギーの進行方向(軸)に垂直で方向に単位幅をもつ断面を通して単位時間に輸送されるエネルギー流束は=2 1+2sinh2=×(13・51)となる。ここで、は波長と周期がわずかに異なる二つの波を合成した場合に得られる波の位相速度(phase velocity)で、群速度(group velocity of wave)と呼ばれる。群速度は=dσd=+dd(13・52)で定義され、微小振幅の深水波では2、表面張力波では32となる。有限振幅波の場合のは、有限振幅波に対するζ,,を、式(13・47)(13・48)に代入することにより得られるが、この場合、質量輸送速度式(13・46)が存在するため、はよりも大きくなる。流れの中に存在する波や、他の波と共存する波の場合には、波と流れの間および波と波の間に非線形相互作用があり、波のモード間のエネルギー移動を考慮しなければならない。
