# 13・4・2 風波の特徴

> **章**: 第13章 自由表面のある流れ
> **節**: 13・4 風波

気液界面上の風速が低い場合には,表面張力波に近いさざ波が形成され,風速の増加とともに典型的な風波が形成される.さらに風速が増加する場合には,風波は崩壊し,砕波となる.このような風波は,成分波の重ね合わせで記述することができる線形性の強いうねりとは異なり,成分波間の相互作用,気流の乱流構造と気液界面に与えるせん断力,液側の乱流構造,砕波機構などが複雑に絡み合った非線形現象を呈する.この風波は気液接触型の工業装置の中の流れにも見られるが,一般的には,大気・海洋間の海水面に見られるもので,地球の気候変化予測につながる大気・海洋間での熱や物質交換量の正確な評価の問題にも関連して重要である.
