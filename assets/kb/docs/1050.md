# 13・4・3 風波の相似則

> **章**: 第13章 自由表面のある流れ
> **節**: 13・4 風波

一定時間に現れる波高の時系列において、波高の大きいほうから全体の波の数の3分の1を選んで、それらの波の波高を平均したものを有義波高()、それらの波の周期を平均したものを有義波周期()と呼ぶ。ここで、は水面上10mの位置での風速である。吹走距離が十分大きくなると波は発達した飽和状態に漸近する。さらに、風と風波とが局所平衡にあることを示す統計的法則として、鳥羽の3/2乗則がある。この3/2乗則はうねりのエネルギーの小さな純風波に対して成立するものであり、うねりが大きい場合には適用できなくなる。
