# 13・4・3 風波の相似則

> **章**: 第13章 自由表面のある流れ
> **節**: 13・4 風波

一定時間に現れる波高の時系列において、波高の大きいほうから全体の波の数の3分の1を選んで、それらの波の波高を平均したものを有義波高()、それらの波の周期を平均したものを有義波周期()と呼ぶ。=0の位置から風が存在するとした場合の吹走距離の関数として、とはウィルソン(Wilson)の公式により=0.31-1+0.004(13・53)2π=1.371-1+0.008(13・54)で与えられる。ここで、は水面上10mの位置での風速である。<10の比較的短い吹走距離の場合は、=0.0024(13・55)2π=0.0548(13・56)となる。吹走距離が十分大きくなると波は発達した飽和状態に漸近する。さらに、風と風波とが局所平衡にあることを示す統計的法則として、鳥羽の3/2乗則がある。=(13・57)ここで、は気側の摩擦速度であり、経験定数は=0.062であるが、風の変動に対応して±20%程度変動する。この3/2乗則はうねりのエネルギーの小さな純風波に対して成立するものであり、うねりが大きい場合には適用できなくなる。
