# 14・1・2 各種変数およびパラメータの定義

> **章**: 第14章 混相流
> **節**: 14・1 基礎的概念

混相流を整理したり解析する場合に最も重要なパラメータは、各相の体積率(volume fraction)である。体積率は一般に式(14・1)(14・2)で定義される。気相の体積率は、ボイド率(void fraction)またはガスホールドアップ(gas hold-up)と呼ばれることも多い。分散相が固体粒子の場合、連続相の体積率は空げき率(porosity)と呼ばれる。物質密度(material density)または真密度(true density)と呼ばれる物性値に対し、混相流では分散相の分布状態に応じて変化する種々の密度が以下のように定義される。そのほか、乾き度(quality,dryness)も気液二相流ではよく使われるパラメータである。気相の質量流量と全質量流量の比は、質量流量乾き度(mass flow quality)、気相の質量と全質量の比は質量乾き度(mass quality)と呼ばれる。粒子運動の観点から重要となる因子は粒子の流体運動に対する追従性である。このは移動度(mobility)と呼ばれる。τは運動応答時間(momentum response time)または緩和時間(relaxation time)と呼ばれ、粒子の流体運動に対する追従性を表すパラメータとなる。すなわち、τが小さい場合は追従しやすく、大きい場合、追従しにくい。これより=τのとき、粒子の速度は最終値の約63%となることがわかる。はストークス数(Stokes number)または慣性パラメータ(inertia parameter)と呼ばれる。
