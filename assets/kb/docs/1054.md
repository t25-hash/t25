# 14・1・3 相間相互作用

> **章**: 第14章 混相流
> **節**: 14・1 基礎的概念

単相流にはない混相流の物理現象としての特徴は,混相流を構成する各相が相互に作用し合うことにある.すなわち相間で運動量,エネルギー,物質が移動する.粒子(液滴,気泡も含む)に働く流体抵抗を通じて分散相の運動と連続相の運動が相互に干渉し合うが,これは運動量の移動を意味する.内部エネルギーの移動は例えば分散相と連続相の間に温度差がある場合に生じる.物質の移動は蒸発,凝縮,燃焼,化学反応などによって相間で質量の交換が生じる現象として現れる.一般に分散相の濃度が低い場合,分散相から連続相への作用が無視できる.例えば粒子は流体抵抗の影響を受けるが,流体は粒子の影響を受けない.分散相の濃度が高くなると,両相が相互に干渉し合う.前者の場合の相互作用はワンウェイ(one way coupling),後者の場合はツーウェイ(two way coupling)と呼ばれる.
