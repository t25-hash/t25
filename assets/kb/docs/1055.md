# 14・1・4 流体中の粒子(固体、液滴、気泡)の運動

> **章**: 第14章 混相流
> **節**: 14・1 基礎的概念

一般に粒子は種々の外力のもとで運動するが、流体中にある場合に最も重要な力は流体抵抗である。また粒子が回転する場合には横向きの力も働く。粒子に働く慣性力が粘性力に比べて十分小さい低レイノルズ数流れでは、ストークスの抵抗[式(14・6)の右辺]に代表される理論解が古くから知られている。レイノルズ数が大きい場合には経験則であるニュートン抵抗がよく知られている。これらについては8・2節に詳しい説明がある。解析的な表現で明示される抵抗式は理論展開を進めるうえで都合がよく、混相流の解析に果たした役割は大きい。しかし、工業装置で扱われる混相流においては、粒子形状は球形のように単純とは限らないし、流体が乱れている場合も多い。また粒子濃度が高くなると、粒子どうしの相互干渉も考慮しなければならない。さらに粒子自身が作り出す変動流れ場の影響を受けて自由に運動する場合に粒子に働く流体力は、流れ場に固定された場合とは異なる。混相流中の粒子に働く流体力に関して既存の理論解や経験則を適用することは第一近似的には許されるが、さらに詳細な解析を行う場合には用いる式の成立条件や限界に注意する必要がある。近年、流体中で運動する多数の粒子や気泡のまわりの瞬時の流れをナビエ・ストークス方程式の直接数値解析によって求め、法線応力、接線応力の粒子表面にわたる積分から粒子に働く流体力を求める方法が広がりつつある。しかし、実装置内にある粒子数は膨大な数値になるので、この方法が適用できる場合は限られている。低レイノルズ数(粒径と相対速度で定義)の単一球の運動方程式は式(14・16)で与えられる。ただし、応力こう配は一定と仮定する。第3項は仮想質量項、第4項はバセット項、第5項は重力による力である。粒子が回転する場合には揚力(横向きの力)が働く。粒子の真密度が流体の真密度に比べて十分大きな値をもつ固気二相流では、仮想質量項やバセット項などの非定常項は無視できるが、固液二相流や気液二相流では無視できない。レイノルズ数が高くなると、抵抗、揚力、非定常項について経験式が適用される。粒子の体積率が低い場合には、単一粒子の運動方程式と同様の式が流れ場の中に多数の粒子に対しても適用される。体積率が高くなると、個々の粒子のおかれる条件は充てん層内にある場合に類似し、したがって流体抵抗についても、充てん層内の圧力損失から個々の粒子に働く流体抵抗が推算される。
