# 14・1・5 混相流の数理モデル

> **章**: 第14章 混相流
> **節**: 14・1 基礎的概念

混相流の数理モデルは、大きく分類すると次のように連続体モデルと離散モデルに分かれる。一流体モデルは、連続相と分散4―151相の区別をせず、混相流体の一つの新しい流体とみなすものである。粘度や密度などの性質は粒子の影響を受ける。二流体モデルでは連続相と分散相のそれぞれに流体型方程式が適用される。ただし、それらの式においては、連続相と分散相の相互作用が考慮される。離散モデルでは、分散相を構成する個々の粒子、液滴、気泡に対して14・1・4項で説明した運動方程式を適用し、その運動が計算される。多数の粒子に対して得られた結果を平均することによって分散相の特性が求められる。離散モデルであっても、連続相の扱いは連続体モデルの場合の連続相の扱いと本質的には同じであるが、その扱い方によってワンウェイ法とツーウェイ法に分類される。連続相が分散相の影響を受けないと仮定するワンウェイ法は、粒子濃度が極めて低い場合に用いられる。連続相が満たすべき方程式は質量保存式(連続の式)、運動量保存式(運動方程式)、エネルギー保存式である。これらの式は基本的には同じ考えに基づいて誘導することができる。すなわち、検査体積(control volume)内で増加する質量、運動量、エネルギーなどの量は、検査体積に流入する量と検査体積から流出する量の差、検査体積内で発生する量の和に等しいとおくことによって得られる。二流体モデルにおいては、分散相を連続体とみなすので、連続相と同じ考えで分散相の質量保存式(連続の式)、運動量保存式(運動方程式)、エネルギー保存式が誘導される。以上のように誘導に関する基本的な考え方は単純であるが、具体的な表現は多様である。特に分散相と連続相の乱れが相互に影響し合うことを考慮しなければならない場合の数理モデルは、いまなお発展途上の状態にある。
