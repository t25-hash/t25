# 14・2・1 流動様式(flow pattern)の定義

> **章**: 第14章 混相流
> **節**: 14・2 気液二相流

気液二相流(gas-liquid two-phase flow)では気相と液相が混在して流れ、いずれもいかようにも変形できるので、その流動様相は気液それぞれの流量によって複雑に変化する。また圧力損失、ボイド率(void fraction)あるいは熱伝達などが流動様式に強く依存しており、問題にしている流れ場がどの様式であるかをできるだけ精度良く予測することが必要である。以下では主として従来多くの研究で対象とされた内径20mm程度の管についての基本的な様相について述べる。a. 鉛直管内上昇流(図14・1)気泡流(bubble flow):連続した液相中に管径より十分小さい気泡が分散した流れ。スラグ流(slug flow):管径より長い砲弾状の大気泡と気泡流状の液体スラグが交互に流れる脈動性の強い流れ。フロス流(froth flow):チャーン流とも呼ばれている。液相と気相の流量がともに大であると、液体スラグ部は多くの小気泡が含まれて白濁する。その白濁した液体が大気泡のまわりに厚い液膜となって流入するため、全体としても泡立った様相をして高速で流れる。脈動性が最も強い流れである。環状噴霧流(annular dispersed flow、環状流ともいう):液相流図14・1 流動様式(鉛直管内上昇流)4―152量が比較的少なく気相流量が大であると、液相は管の壁面上を液膜となって流動し、気相は管断面の中心部に気柱を形成して流動する。気相中には大規模の波(擾乱波、disturbance wave)から発生した液滴が多数含まれている場合が多い。このような場合には環状噴霧流とも呼ばれる。なお、液体流量が多い場合には気相中に白濁した液滴の大きな塊が存在した流れになり、これをウィスピー環状流という。b. 鉛直管内下降流基本的には上昇流と変わりがない。しかし、例えば管壁を液膜が下降する液膜流の場合、上昇流のように気相流量が大きくなくても流動様相は環状流に分離される。c. 水平管内流(図14・2)気液二相流では流動系の圧力が臨界圧力に近い、あるいは微小重力の場のような場合を除けば、気液それぞれの密度の差は極めて重要な意味をもっている。この差によって、水平管では液相は管の低部に、気相は管頂部に偏って流動し、そのため鉛直管内の流動様式とは異なる。研究者によって異なる呼称を用いているが、以下では重力場で見られる一般的なものに従って説明する。気泡流(bubble flow):連続した液相中に管径より十分小さい気泡がいくぶん管頂部に偏って流れる流れ。プラグ流(plug flow):管軸方向に長い大気泡が管頂部に遍在する流れ。層状流(laminate flow):液相は管底部を、気相は管頂部を層状に分離して流れる流れ。気液界面は鏡面状に平滑である。図14・2 流動様式(水平流)図14・3水平流の流動様式に与える重力の影響波状流(wavy flow):層状流と同様に分離した流れであるが、気相の流速が高くなって気液界面が不安定となり、波立っている流れ。スラグ流(slug flow):波状流より液流量が多い場合で、液相の一部が管頂部に達して管路断面を閉塞した液体スラグとして流れる流れ。液体スラグ部には多数の小気泡が含まれていることがある。環状流(annular flow、環状噴霧流ともいう):重力に逆らって液膜が管頂部まで形成され、鉛直管の場合と同様の様相を呈する。ただし、液膜は管低部ほど厚い。通常気体コア部には擾乱波から発生した液滴が含まれている。d. 細管内流管径が3mm程度より小さくなると表面張力の効果が重力の効果に勝って、管断面内の気相の偏りが少なくなり、流動様式は鉛直管内のそれと類似してくる。すなわち水平流でも層状流や波状流などの分離流がなくなり、鉛直上昇流におけるスラグ流と同様になる。ただし、速度が遅いと大気泡の後端も先端と同様に丸みをもつ。また、管径が1mm程度より細い管では、大気泡のまわりの液膜厚さも管低部と管頂部とで差がほとんどなくなる。e. 微小重力下の管内流微小重力下では気液両相の密度差による相の偏りがほとんどなくなり、管径が20mm程度の水平流の場合でも鉛直管内流とほとんど同じ流動様式を呈する。図14・3は気液の流量が同じ場合で、重力の加速度が地球の場合、月の場合、および微小重力の場合の流動様相をそれぞれ比較したものである。
