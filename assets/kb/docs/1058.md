# 14・2・2 流動様式線図(flow regime map)

> **章**: 第14章 混相流
> **節**: 14・2 気液二相流

いろいろな条件下の気液二相流の流動様式を精度良く予測できる流動様式線図はまだない。ここではよく使われるものを例として挙げる。a. 鉛直管内上昇流の場合図14・4はヒューイット・ロバーツ(Hewitt-Roberts)の線図である。とはそれぞれ気相と液相の体積流束(見掛けの速度)で、それぞれに管断面積を乗じれば体積流量に等しい。その例を図14・5に示す。図14・6は流体の物性値を含む量を軸として整理されているため、水以外の流体の実験結果の整理によく使われているが、実際には物性値の異なる流体を用いて得られた線図ではないので、使用には注意が必要である。水平管に対してもとを用いた線図がマンデェーン(Mandhane)により提案されている(図14・7)。c. 細管の場合図14・8は鉛直上昇流と水平流の場合の区分線図を比較したものである。鉛直上昇流と水平流との差はあまり大きくはない。図中には大きな管径のマンデェーンの線図も比較のために記している。
