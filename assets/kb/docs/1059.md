# 14・2・3 各流動様式の特徴

> **章**: 第14章 混相流
> **節**: 14・2 気液二相流

a. 気泡流。. 気泡の上昇速度静止水中における単一気泡の上昇速度は、気泡の径と形状に強く依存するが、周囲液体中の不純物の影響も大きい。不純物は気液界面に吸着し、気泡下部へ蓄積するため濃度によるマランゴニ対流(Marangoni convection)が誘起される。その流れが気泡上昇中の気泡表面の上部から下部への液体の流下の妨げとなる結果、気泡表面があたかも固体表面のように振舞い、上昇速度を決定する抗力係数が高純度の液体の場合の1.5倍にも達する。. 気泡挙動の解析鉛直管内気泡運動方程式の一例は、式(14・18)で与えられる。これらは次第に明確になりつつある。. 気泡流からスラグ流への遷移気泡密度が増加すると液相の乱れに応じて小気泡の部分的な集合、気泡クラスタ(bubblecluster)が生じ、上昇速度が大気泡と同じになり、あたかも大気泡のような挙動をする。気泡クラスタは気泡の集積度が上がるとまもなく小気泡の全体が瞬間的に合体し、大気泡へ変化する。気泡クラスタはスラグ流への遷移の重要な過程である。b. 鉛直管内スラグおよびフロス流。単一大気泡の上昇速度(静止水中)内径の管内の静止水中を上昇する大気泡速度は通常式(14・19)で与えられる。. スラグ流中の大気泡上昇速度式(14・19)との和として式(14・21)の形式で表す場合が多い。式(14・21)はニクリン(Nick-lin)の式としてよく用いられている。個々の大気泡の速度のばらつきは実際には非常に大きいが、その平均値は水-空気系の場合、式(14・21)とよく合う。 大気泡周辺の液膜流のモデル化スラグ流やフロス流は、絶対座標系から観察すると、ボイド率が時間的に大幅に変動する非定常性の強い流れであるが、大気泡に固定した座標系から観察すれば非加熱の十分発達した流れでは、気泡周辺の液体の流れは定常であるとみなすことができ、液体は気泡の上部から近寄り、気泡のまわりを2~3m/s(管径25mm、大気泡終端近傍のホールドアップを0.1と仮定)に近い速度で流下する。この流れの概要は、層流、一次元流れ、大気泡内静圧分布は一様と仮定すれば、壁面におけるせん断力を考慮した基礎式を数値的に解くことによって、液膜厚さ分布(気泡形状)、平均速度分布などが容易に求められる。. 大気泡周辺液膜の逆流大気泡周辺の液膜は、気液の合計速度+がiii項で述べた気泡終端での速度より小さい場合には絶対座標から観察しても逆流する。この逆流液体は後続する上昇液体スラグへ高速で突入し、その液膜の運動エネルギーは液体スラグ内で渦となって消滅する。このためスラグおよびフロス流領域では単相流に比較してエネルギー損失が格段に大きい。この根本の原因は、気液の密度差によって生じる大気泡と液体スラグ中の液体との速度の差にある。. 液体塊速度スラグ流や典型的なフロス流では液体塊は液体スラグであり、これは管の断面全体を満たしているので、その上昇速度はニクリンの式(14・21)によく一致する。一方、環状流への遷移領域でのフロス流では、一部の液体スラグが気流に貫通されて速度が低下し、液体塊の集合平均は式(14・21)より小さくなる。c. 水平管内スラグ流。 静止液体中の先端速度水平な管路に液体を充満し、その一端を解放すれば、液体は解放端から流出するので、その分だけ気体が大気泡を形成しつつ管内に流入する。すなわち大気泡に挟まれた液体スラグは、その前後の大気泡内の圧力差による力によって移動する。したがって、液体スラグ周辺の液体の流れは図14・10に示すように、液体スラグの先端では遅い速度の液膜部流体がその液体スラグへ流入し、後端部では同量の液体が後方の液膜部へ流出する。. スラグ速度ヒューマーク(Hughmark)は水平管の場合には重力の影響がないとして、式(14・25)を提案した。. 鉛直上昇環状流液体は管壁面上を液膜となって流動し、高速の気流は管のコア部を貫通した気柱を成して流動する。薄い液膜の表面上にはリップルと呼ばれる微細な波が生じており、通常この上を管周を取巻くリング状の大規模な構造を有する擾乱波が、40~50cm間隔で高速で通過する。擾乱波の波頭からは気流のせん断力によって微小液滴(エントレインメント、entrainment)が発生し、気流に乗って流動する。液膜と液滴の速度は大幅に異なっている。したがって、この流れではこれらに加えて気流が存在するので、気流、液膜および液滴からなる三流体モデルで解析する。. 水平環状流この流れでは重力の影響は流れ方向にはないが、管軸に直角な方向に現れ、液体は管の低部に遍在しがちであるため、管周方向の液膜厚さ分布は一様ではない。しかし、開きょの流れのように液面が水平にはならない。このことは重力に逆らって液膜を管頂部へ輸送し、環状の液膜を形成する機構が存在することを意味している。この機構として、気流の二次流れ、エントレインメント、波の広がりおよび擾乱波のポンプ作用の四つの機構が提案されている。内径が20~30mm程度の管の場合、擾乱波のポンプ作用が支配的である流動モデルが提案され、そのことが数値解析から明らかにされている。
