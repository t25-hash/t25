# 14・2・4 基礎式

> **章**: 第14章 混相流
> **節**: 14・11 を与えている。

a. 基礎式最もわかりやすい分離層状流モデル(図14・12)の場合は単相流に対する式が応用され、以下のとおりである。ただし、:重力加速度、:エンタルピー、:運動量、ρ:圧力、:管周方向もしくは気液界面の長さ、:時間、:流れ方向の座標、θ:管軸の水平方向からの傾き角度、添字:とは気相と液相を、は気液界面、もしくは気体から液体への輸送量、はその逆を表す。ηは1-αに等しく、ホールドアップ(hold-up)と呼ばれる。b. 気相と液相の質量保存の式옿옿ρα+옿옿ρα=(14・26)옿옿ρη+옿옿ρη=(14・27)図14・12 分離層状流モデルとはそれぞれの相に物質移動により流入する単位長さ当たりの質量であり、=が成り立つ。したがって、二相流全体の質量保存の式は、옿옿ρα+ρη+옿옿ρα+ρη=0(14・28)となる。c. 気相と液相の運動量保存の式옿옿ρα+옿옿ρα+α옿ρ옿+ραsinθ+욌τ욌+τ+=0(14・29)옿옿ρη+옿옿ρη+η옿ρ옿+ρηsinθ+욌τ욌+τ+=0(14・30)ここで욌と욌は気相と液相が壁面と接している管周の長さ、τ욌とτ욌はその部分のせん断応力、,τおよびτは気液界面のそれらの値を表し、τ=τである。また、とはそれぞれ気相から液相へ、およびその逆の運動量輸送量で、=-である。二相流全体に対する運動量の保存式は、式(14・31)と式(14・32)の辺々を加えて、次のようになる。옿옿ρα+ρη+옿옿ρα+ρη+옿옿+ρα+ρηsinθ+욌τ욌+욌τ욌=0(14・31)式(14・31)の物理的意味は、順に非定常項、対流項、静圧項、静水圧項、摩擦項である。d. 気相および液相のエネルギー保存の式をエネルギーの移動量、あるいは各相中で発生もしくは供給されたエネルギーであるとすれば、それぞれ、次のようになる。옿옿ρα-ρ++옿옿ρα++ραsinθ+=α(14・32)옿옿ρη-ρ++옿옿ρη++ρηsinθ+=η(14・33)二相流全体のエネルギー式は、옿옿ρα-ρ++ρη-ρ++옿옿ρα++ρη++ρα+ρηsinθ=α+η(14・34)となる。
