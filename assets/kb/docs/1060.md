# 7・1・5 応力集中の簡便解析

> **章**: 第7章 応力集中，接触応力および接合応力
> **節**: 7・1 応力集中

強度設計において応力集中係数の値を必要とする場合は，ここでらその値を評価すればよい．
一方，これらの図表が適用できない場よる理論解析や有限要素法・境界要素法などのコンピュータによる前者の理論解析が適用できるのは，部材および応力集中源の形状や合でも，応力集中部の近傍では応力の変化が大きいことから，要素が必要である．
一方，解析結果の精度が工学的に許容できる範囲内もある．
例えば，帯板の両側に深い双曲線状切欠，あるいは浅い半理論による解析結果が知られている．
このとき，任意の深さの切欠の関係式から，次式で評価することができる．
また，有限要素解析においても，比較的粗い要素分割と基本的U型切欠をもつ帯板が引張荷重を受けるときの応力集中係数αのさを曲率半径の30分の1として計算した結果を，実験結果や7･1･簡便解析でも工学的には十分な結果が得られることがわかる．
紹介した結果や応力集中ハンドブック等にまとめられている図表か合や応力集中係数の値をより正確に評価したい場合には，弾性論に数値解析によって応力集中係数を評価することができる．
