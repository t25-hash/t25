# 7・1・5 応力集中の簡便解析

> **章**: 第7章 応力集中，接触応力および接合応力
> **節**: 7・1 応力集中

強度設計において応力集中係数の値を必要とする場合は，ここで
紹介した結果や応力集中ハンドブック等にまとめられている図表か
らその値を評価すればよい．一方，これらの図表が適用できない場
合や応力集中係数の値をより正確に評価したい場合には，弾性論に
よる理論解析や有限要素法・境界要素法などのコンピュータによる
数値解析によって応力集中係数を評価することができる．しかし，
前者の理論解析が適用できるのは，部材および応力集中源の形状や
荷重様式が比較的単純な場合に限られ，また，後者の数値解析の場
合でも，応力集中部の近傍では応力の変化が大きいことから，要素
分割を細かくしたり，特殊な要素を用いるなど，解析に特別の工夫
が必要である．一方，解析結果の精度が工学的に許容できる範囲内
であれば，比較的簡便な方法で応力集中係数の値を評価できる場合
もある．例えば，帯板の両側に深い双曲線状切欠，あるいは浅い半
だ円状切欠がある場合の応力集中係数αとαの値については弾性
理論による解析結果が知られている．このとき，任意の深さの切欠
に対する応力集中係数αは，ノイバーの応力集中係数の三角則
（Neuber’s triangle rule of stress concentration），すなわち
1α－1＝1α－1＋1α－1 (7･5)
の関係式から，次式で評価することができる．
α＝1＋ α－1α－1
－1＋α－1 (7･6)
この応力集中係数の三角則には理論的根拠はないが，便利な式であ
る．また，有限要素解析においても，比較的粗い要素分割と基本的
な要素を用いて応力集中係数を評価することができる．表7･1は
U型切欠をもつ帯板が引張荷重を受けるときの応力集中係数αの
値について，基本的な定ひずみ要素を用いて，切欠底の要素の大き
さを曲率半径の30分の1として計算した結果を，実験結果や7･1･
2項の図7･10から評価した結果と比べたものであり，この程度の
簡便解析でも工学的には十分な結果が得られることがわかる．
