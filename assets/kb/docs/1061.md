# 7・1・6 応力集中と塑性変形

> **章**: 第7章 応力集中，接触応力および接合応力
> **節**: 7・1 応力集中

セラミックスのように非常にもろい材料が静的荷重を受けると，
応力集中部にき裂が発生してただちに破壊に至ることから，脆性材
料を用いる場合の強度設計では，応力集中係数は重要である．一
方，鋼のように比較的延性のある材料の場合は，最大応力が生じる
点で材料は降伏して塑性流動を起こすため，応力集中の度合いは相
対的に低下して応力はほぼ均一に分布するようになる．したがっ
て，延性材料を用いる場合には，応力集中を無視して強度設計を行
うこともできる．しかし，疲労破壊のように，微視的レベルで見る
と，繰返し塑性変形による損傷の累積結果とみなせる破壊に対して
は，弾塑性変形状態での応力集中を考えることが必要である．この
場合，応力とひずみは比例しないことから，応力集中係数αとひ
