# 14・3・1 スラリーの物性

> **章**: 第14章 混相流
> **節**: 14・3 固液二相流

厳密には微粒子と液体との混合物(泥しょう)をスラリー(slur-ry)と呼ぶが,固体粒子と液体との混合体を広義にスラリーと呼ぶ.スラリーと固液二相流(solid-liquid two-phase flow,liquid-solid two-phase flow)とはほぼ同義に扱われるが,固体粒子の水力輸送(hydraulic transport,slurry transport)に重点をおいた場合に,スラリーと呼ぶことが多い.スラリーにおける固体粒子と液体との密度比は実用的には10以下であり,液体運動に対する固体粒子の追従性は一般的に高いと考えてよい.水力輸送においては,固体粒子の沈降性を重要な指標と考え,スラリーを非沈降性(安定)スラリー(fine particle slurry)と沈降性スラリー(coarseparticle slurry,settling mixture)とに大別する.前者は微粒子から成り,均質かつ輸送過程における粒子の沈降性を無視しうる.後者は粗大固体粒子から成り,全体として沈降しやすい.このため,沈降性スラリーでは,粒子の沈降・堆積が生じて,流れが不安定となり,一義的に物性値を求めることが困難となる.したがっ図14・13 混合体の物性沈降性スラリーは,流速,粒子物性(粒径,粒度分布,密度),粒子濃度,搬送流体の物性,抗力や揚力等の粒子・搬送流体間相互作用の組合せにより種々の流動状態を呈する.このため,一義的にその物性を決定することは困難であり,一般的に,流動様式,遷移速度,粒子濃度分布,圧力損失という視点から沈降性スラリーの流動を把握する手法がとられる.
