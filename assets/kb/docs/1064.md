# 14・3・2 沈降速度と抗力

> **章**: 第14章 混相流
> **節**: 14・3 固液二相流

a. 単一粒子の沈降速度固液二相流の流動特性を決定する基礎量の一つである。沈降速度(settling velocity)と終速度(ter-minal velocity、終端速度)とは同義であり、無限広さの静止流体中を単一粒子が重力を受けて定常状態で沈降する際の速度と定義される。単一粒子の球の場合には、ストークスおよびニュートンの各領域における終速度욦は、욦=욦18ν욆ρ욦ρ욆-1、ストークス領域(14・57)욦=4욦3ρ욦ρ욆-1、ニュートン領域(14・58)で与えられる。ここで、:重力加速度、욦:粒径、:抗力係数、ν욆:搬送流体の動粘度、ρ욆およびρ욦:搬送流体および粒子の密度である。実用上、スラリー粒子の多くは不規則形状粒子であるから、욦には球等価径が使用される。ただし、この場合、粒子形状を表す係数等による補正が必要となる。さらに、スラリー粒子は粒径分布をもつので、中位径に相当する球等価径で構成粒子の径を代表させて、単一粒子の沈降速度を定義する。b. 抗力粒子レイノルズ数욣は욣=욦욦ν(14・59)と定義される。球形粒子が受ける抗力욥および抗力係数(dragcoefficient)は、概略次の各近似式により表される。욥=3πμ욦욦=24<1(14・60)=1030<<500(14・61)욥=1.25 μρ욆욦욦욥=0.055πρ욆욦욦=0.44500<<10(14・62)c. 壁の影響(管壁の干渉)単一粒子が液体で満たされた管路内を沈降あるいは上昇するとき、粒子と管壁との間に形成される流体速度分布により、粒子運動は無限容器内とは異なったものとなる。これを管壁の干渉という。粒子が壁の干渉を受けて沈降する場合、干渉沈降速度(hindered settling velocity)という。ニュートン抵抗則で近似される粗大粒子の場合<500にこの干渉を考慮することが多い。干渉沈降速度욦욠は粒径と管径との比욦욣により終端速度を補正する実験式により表すことができる。욦욠=욦1-욦욣욖(14・63)には1.5~2.0が使用される。なお、スラリー輸送の実用管径範囲においては、<500の微小粒子の場合、욦욣が無視小となり、管壁の干渉を考慮する必要はない。d. 粒子群の抗力係数粗大粒子のスラリーにおいては、上記の管壁の干渉に加えて粒子間の干渉を考慮する必要がある。粒子群の抗力係数は、平均粒径욦の粒子の抗力係数をとし、管径욣および粒子の管内体積濃度(special concentration)욦を考慮して、次の実験式により表すことができる。=1-욦욣exp -4욦1+욦(14・64)なお、本実験式の推奨使用範囲は2mm<욦<50mm,25mm<욣<250mm,욦<0.24である。
