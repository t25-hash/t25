# 14・3・2 沈降速度と抗力

> **章**: 第14章 混相流
> **節**: 14・3 固液二相流

a. 単一粒子の沈降速度固液二相流の流動特性を決定する基礎量の一つである。沈降速度(settling velocity)と終速度(ter-minal velocity、終端速度)とは同義であり、無限広さの静止流体中を単一粒子が重力を受けて定常状態で沈降する際の速度と定義される。実用上、スラリー粒子の多くは不規則形状粒子であるから、には球等価径が使用される。ただし、この場合、粒子形状を表す係数等による補正が必要となる。さらに、スラリー粒子は粒径分布をもつので、中位径に相当する球等価径で構成粒子の径を代表させて、単一粒子の沈降速度を定義する。球形粒子が受ける抗力および抗力係数(dragcoefficient)は、概略次の各近似式により表される。これを管壁の干渉という。粒子が壁の干渉を受けて沈降する場合、干渉沈降速度(hindered settling velocity)という。d. 粒子群の抗力係数粗大粒子のスラリーにおいては、上記の管壁の干渉に加えて粒子間の干渉を考慮する必要がある。
