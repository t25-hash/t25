# 14・3・3 遷移速度(堆積速度、浮遊速度)

> **章**: 第14章 混相流
> **節**: 14・3 固液二相流

遷移速度は、粒子群の運動状態から定義される速度である。水平および傾斜管路による固体粒子の輸送という視点からは、粒子の流動状態あるいはスラリーの圧力損失を見積もるうえで重要な指標となる。ただし、遷移速度は固液二相流の目視観察を基に決定された図14・14 沈降性固液二相流の流動様式表14・1 粒子物性値平均粒径mm平均密度kg/m平均終端速度m/s粒子レイノルズ数(-)平均抗力係数(-)4.970.4632.3×100.4710.30.6997.2×100.44ものであり、同じ遷移速度であっても研究者によりその値には幅があることに注意を払わなければならない。a. 堆積速度、堆積限界速度、臨界速度沈降性スラリーにおいて、流速を徐々に低下させると管底部の粒子が一時的に停止するようになる。このときの固液混合体の平均速度を堆積速度(deposit velocity)という。さらに、流速を低下させると管底部に明りょうな粒子静止層が形成される。このときの速度を堆積限界速度(limit deposit velocity)という。粒子吐出し濃度を一定とした場合の圧力損失と固液混合体の平均流速との関係において、圧力損失が極小となるときの固液混合体の平均流速を臨界流速(crit-ical velocity)という。b. 浮遊限界速度固液混合体の平均流速が堆積速度より大きい速度領域においては、管底の静止粒子層は消失し、粒子層は管底をすって移動するしゅう動層(sliding bed)となる。さらに流速を増大させるとこのしゅう動層も消失し、管底より管頂部に向かって濃度分布をもつ非均質流れ(heterogeneous flow)となる。このときの流速を浮遊限界速度(suspending velocity)という。c. 遷移速度に関する実験式管径:1.26~69.9cm、粒子密度:1.16~11.3g/cm、平均粒径:0.0297~38mm、粒子の吐出し体積濃度:0.006~42%、混合体平均流速:0.009~6.7m/sの流速条件における2848データを基に、堆積速度、浮遊限界速度および非均質流れから均質流れへの遷移速度を推定する下記の実験式が提案されている。=31.93욋욌욣ρ욦ρ욆-1(14・65)=2.411욋욌욣ρ욦ρ욆-1(14・66)=0.2859욋욌욣ρ욦ρ욆-1(14・67)ただし、욋:粒子の吐出し体積濃度、:粒子の抗力係数、욣:管径、욌:スラリーの平均流速と等しい液相流速における管摩擦係数、:重力加速度、ρ욆およびρ욦:液相密度および粒子密度である。
