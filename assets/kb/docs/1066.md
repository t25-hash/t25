# 7・3・3 対数型の応力特異性

> **章**: 第7章 応力集中，接触応力および接合応力
> **節**: 7・3 接合応力

自由縁に零ではない表面力を受ける二つのくさびの接合角部の先
端において対数型の応力特異性（logarithmic stress singularity）
が生じる可能性がある ．例えば，二つのくさびの角度の和が
180となる場合においては，次の二つの条件式：
α＝βcos2θ－1cos2θ (7･17)
または
α＝π2tanθ＋π－2θ (7･18)
のいずれかが満足されるときに，対数型の応力特異性が生じる．
