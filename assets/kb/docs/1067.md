# 14・3・5 圧力損失

> **章**: 第14章 混相流
> **節**: 14・3 固液二相流

粒子輸送に必要なポンプ動力を見積もるうえで、圧力損失の算定は重要となる。単一流体とみなしうる場合と二相流とみなす場合とで、圧力損失の算定方法は異なる。a. 単一流体として扱える場合非沈降性(安定)スラリーならびに沈降性スラリーの均質流れ、すなわちスラリーを単一流体として扱うことができる場合には、単一流体が管路を流れる際の管摩擦係数および固液混合体密度を用いてファニング(Fanning)の式により、スラリーの圧力損失を評価することができる。b. 二相流として扱うことが必要な場合。 水平管内図14・14に示したように圧力損失と流動様式とは密接に関係しており、粒子輸送の観点からは、圧力損失は流動様式と関係付けて議論されることが多い。固液二相流の代表密度を液相密度ρとして、単位長さ当たりの水頭損失(hydraulic gradient、水力こう配)で表すことが多い。スラリーの圧力損失は、式(14・72)で定義される圧力損失係数φと修正フルード数の関数として、式(14・74)のように整理される。式(14・74)中の係数および指数には粒子の種類、粒子濃度、流動様式ごとに古くから種々の値が提案されている(表14・2)。なお、実験範囲の参考値は、管径1.26~69.9cm、粒子密度:1.16~11.3g/cm、平均粒径:0.0297~38mm、粒子の吐出し体積濃度:0.006~42%、混合体平均流速:0.009~6.7m/sである。粒子の流動状態をより詳細に反映させて圧力損失を見積もるべきとの考えから、粒度分位と粒子の濃度分布とを考慮して、沈降性スラリーの圧力損失を算定する方法が提案されている。. 鉛直管内粒子の吐出し体積濃度と粒子群の干渉沈降速度とを考慮することにより、次の実験式により算定できる。固気二相流(solid-gas two-phase flow,gas-solid two-phaseflow)には内部流として管による粉粒体の空気輸送、乾燥や反応等の操作を行う固定層や流動層内の流れがある。外部流として物体まわりの含じん気流、自由空間での流れとして含じん噴流がある。固気系では固体粒子と気体の密度比が10程度であるので、重力下では流れの方向によって流動様式は変化する。
