# 7・2・1 分子動力学法（molecular dynamics method）

> **章**: 第7章 熱流体方程式の選択と離散化手法
> **節**: 7・2 分子流れの解法

分子動力学法は，計算対象を原子や分子を表すモデル粒子の集合
体としてモデル化し，そのモデルにおける各粒子の運動や空間配置
の特徴を解析する手法であり，これを用いて構造的物性，熱力学的
物性，動力学的物性，輸送現象的物性などが解析される．分子レベ
ルの挙動を可視化して観察することもでき，ミクロな現象の理解に
役立つ．
分子動力学法は，分子が古典的な物理法則に従って運動するとし
て，個々の分子の時々刻々の動きを計算する手法である．量子力学
的な効果はすべてポテンシャルの中に含まれているとし，各分子の
運動量と位置は同時に決定できるものと仮定している．分子運動の
支配方程式はニュートン方程式である．ポテンシャルがわかってい
れば，分子に働く力がわかり，時々刻々の位置と速度が計算され
る．分子の運動方程式を解くに当たって，初期条件，境界条件を与
える必要がある．初期条件としては分子の位置，速度ベクトルを与
えることになる．計算を効率良く行おうとすれば，気相，液相，固
相，気液共存相，固気共存相など計算対象に合わせて，適当な初期
条件を与える必要がある．空間的な境界条件については，ある大き
さの計算空間（基本セル）を考え，その計算空間に周期境界条件を
課して，単位セルを周期的に並べ，無限大の大きさの計算領域を考
えることが多い．この場合，基本セルから分子が出ていくと，反対
側の境界の相対する位置から分子が流入することになる．基本セル
の計算空間よりも大きな規模をもつ構造に関する情報は，計算でき
ないことに注意が必要である．

計算できる．連続体としての密度は，空間内の分子の数の時間平均
の平均速度から求められ，温度は分子の速度の平均からの分散とし

その値から計算する方法（アインシュタインの方法）などがある．
