# 14・4・2 浮遊速度

> **章**: 第14章 混相流
> **節**: 14・4 固気二相流

粒子の流体力学的特性として最終沈降速度がある。最終自由落下速度、終末沈降速度(terminal settling velocity)、終速度等とも呼ばれる。粒子が気中を重力のもとで気流抵抗、浮力と釣合って等速で落下するときの速度である。空気輸送では重力下で粒子を気流によって浮遊させる速度に等しいので浮遊速度(floatingvelocity)と呼ばれる。粒子の形状、密度、大きさなどに関係する。さらに、が減少するとブラウン運動の影響が出てくる。一般に、このような10 m以下の粒子は浮遊粒子状物質(SPM)といわれる。c. 粒子群の沈降粒子群が有限の容器内を均一に沈降する場合、他の粒子や容器壁との干渉、さらに置換気流の影響を受けるため、沈降速度は単一粒子と比べて減少する。しかしながら、粒子が高濃度のクラスタ状に落下する場合、クラスタ内の気体は群に取込まれ、等価的に大きな粒子となって落下し、沈降速度は単一粒子のものより大きくなる。粒子が無限気体中に孔から連続的に排出される場合、周囲気体が落下粉体群に引込まれ、その中に下降気流が形成され、粒子はこの気流に相対的に沈降するため、この場合も沈降速度は単一粒子のものより大きくなり、粒子の質量流量の増加とともに増大する。図14・24 浮遊速度図14・25 最小流動化速度4―162流体工学
