# 14・4・3 流動化速度

> **章**: 第14章 混相流
> **節**: 14・4 固気二相流

最小流動化速度욇욞は均質流動層の最小の流動化速度であり、固定層と流動層の遷移点での空塔速度である。粒子径、形状、密度、粒度分布によって変化する。実験的には図14・20のように、最初、固定層からを上げていくと点A で流動化が開始し、その後を上げても圧力損失はほぼ一定である。この流動化状態からを下げていくと点B から固定層の状態になる。욇욞はB での速度とされる。この差は固定層の空げき率の違いによるもので、一度気流で固定層がほぐされることによって、再現性のある욇욞が求められる。욇욞は計算では固定層の圧力損失と重さが釣合うとして求められる。図14・25はウエン(Wen)とユー(Yu)の方法で20°Cの空気での욇욞をρ욦をパラメータとして示す。
