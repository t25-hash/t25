# 14・4・3 流動化速度

> **章**: 第14章 混相流
> **節**: 14・4 固気二相流

最小流動化速度は均質流動層の最小の流動化速度であり、固定層と流動層の遷移点での空塔速度である。粒子径、形状、密度、粒度分布によって変化する。実験的には図14・20のように、最初、固定層からを上げていくと点A で流動化が開始し、その後を上げても圧力損失はほぼ一定である。この流動化状態からを下げていくと点B から固定層の状態になる。この差は固定層の空げき率の違いによるもので、一度気流で固定層がほぐされることによって、再現性のあるが求められる。図14・25はウエン(Wen)とユー(Yu)の方法で20°Cの空気でのをρをパラメータとして示す。
