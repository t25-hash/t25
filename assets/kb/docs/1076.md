# 7・5・2 連続体の衝撃応答

> **章**: 第7章 過渡応答・衝撃
> **節**: 7・5 多自由度系と連続体の衝撃応答

7･5･1項では2自由度系を例にとって多自由度系の衝撃応答について解説したが，自由度が増した場合も同様な取扱いが可能である．
さらに，無限自由度を有する連続体の衝撃応答については，有限要素法などにより有限自由度に落として数値計算を用いることにより応答を求めるのも有効な手段である．
一方，モード座標系を導入すれば連続体を有限自由度系として扱うことが可能であり，しかる扱いをそのまま利用することができる．
絶縁系の最大変位δと速度ステップ との関係として，絶縁系の最大伝達力 および最大応答加速度 は，このような，ばねのみによる絶縁系の衝撃エネルギー吸収効率ηは，図7･18を参照して，もモードの独立性を背景として，7･2節で示した1自由度系に対すで与えられる．
