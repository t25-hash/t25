# 15・3・1 単純せん断流れ

> **章**: 第15章 非ニュートン流体
> **節**: 15・3 基本的な流れ

図15・1に示すように、非圧縮性の非ニュートン流体が2枚の平行平板間を十分発達して流れている場合を考える。下側の板は静止しており、上側の板が速度で下側の板に平行に動いているとする。このような流れを単純せん断流れ(simple shear flow)と呼ぶ。非圧縮性流体では圧力を決めることはできないので、せん断応力τと二つの法線応力差τ-τとτ-τのみに意味がある。ニュートン流体ではせん断速度γが変化しても粘性率は一定であるが、非ニュートン流体ではγの関数である。多くの高分子流体はγが大きくなるとη4―165が小さくなる。この現象をシアシニング(shear-thinning)あるいはずり流動化と呼び、非ニュートン流体の第一の特徴である。非ニュートン流体ではシアシニングのため、例えば円管内の流れでニュートン流体と異なった速度分布になるなど、多くの違った流動状態を示す。次に、第二の特徴として法線応力差(normal stressdifference)がある。ニュートン流体の単純せん断流れではτ,τ,τはすべて零であるが非ニュートン流体では零とはならず、このためニュートン流体に比べ、非ニュートン流体の流れは複雑となる。第二法線応力差は第一法線応力差に比べ、その値は小さく、負の値となることが知られている。単純せん断流れでは、η,Ψ,Ψの三つの関数を粘度計関数(viscometric function)と呼び、この関数によって応力状態が決まる。したがって、この関数形を決定することが構成方程式との関連のもとで重要である。
