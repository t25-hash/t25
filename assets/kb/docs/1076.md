# 15・3・1 単純せん断流れ

> **章**: 第15章 非ニュートン流体
> **節**: 15・3 基本的な流れ

図15・1に示すように、非圧縮性の非ニュートン流体が2枚の平行平板間を十分発達して流れている場合を考える。流れの方向に軸、板に垂直な方向に軸をとる。下側の板は静止しており、上側の板が速度で下側の板に平行に動いているとする。このときの速度分布は、2枚の板の間隔をとすると==0=0である。このような流れを単純せん断流れ(simple shear flow)と呼ぶ。さらに、液晶性の流体以外は等方性(isotropic)であるから、単純せん断流れにおける全応力はτ-+τ-+τ=-+=τ-+τとなる。非圧縮性流体では圧力を決めることはできないので、せん断応力τと二つの法線応力差τ-τとτ-τのみに意味がある。ニュートン流体ではせん断速度(shear rate)をγ=、粘性率をμとおくと、τ=μγである。非ニュートン流体でμに相当するものを非ニュートン粘度(non-Newtonian viscosity,shear-rate dependent viscosity)と呼び、ηγと書くと、τ=ηγγとなる。ここで、γ:γの大きさである。ニュートン流体ではせん断速度γが変化しても粘性率は一定であるが、非ニュートン流体ではγの関数である。多くの高分子流体はγが大きくなるとη4―165が小さくなる。この現象をシアシニング(shear-thinning)あるいはずり流動化と呼び、非ニュートン流体の第一の特徴である。非ニュートン流体ではシアシニングのため、例えば円管内の流れでニュートン流体と異なった速度分布になるなど、多くの違った流動状態を示す。次に、第二の特徴として法線応力差(normal stressdifference)がある。これを=σ-σ=Ψγγ=σ-σ=Ψγγと書き、,を第一、第二法線応力差、Ψ,Ψを第一、第二法線応力差係数と呼ぶ。ニュートン流体の単純せん断流れではτ,τ,τはすべて零であるが非ニュートン流体では零とはならず、このためニュートン流体に比べ、非ニュートン流体の流れは複雑となる。第二法線応力差は第一法線応力差に比べ、その値は小さく、負の値となることが知られている。単純せん断流れでは、η,Ψ,Ψの三つの関数を粘度計関数(viscometric function)と呼び、この関数によって応力状態が決まる。したがって、この関数形を決定することが構成方程式との関連のもとで重要である。
