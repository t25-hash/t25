# 15・3・2 伸長流れ

> **章**: 第15章 非ニュートン流体
> **節**: 15・3 基本的な流れ

粘度というときは,ニュートン流体からの連想でせん断粘度を指すことが多い.しかし,非ニュートン流体の流れでは伸長粘度(elongational viscosity)が重要な役割を担うことがしばしばある.伸長流れとは,例えば小さなノズルから溶融高分子を糸状に押出し,それを軸方向に引っ張ったときの流れで,このときせん断応力は作用していない.引張方向(軸方向)の伸長ひずみの速度をεとすると=ε=-12ε=-12εのような速度場である.この流れを一軸伸長流れという.一般に伸長流れは=ε=-12ε1+=-12ε1-なる速度場で表され,ε:伸長速度(elongational rate),パラメータは0≦≦1である.そして,パラメータによって,次のように分類される(図15・2).1=0かつε>0のとき,一軸伸長流れ(elongational flow)2=0かつε<0のとき,二軸伸長流れ(biaxial stretchingflow)3=1のとき,平面伸長流れ(planer elongational flow)である.定常伸長流れでは,流体粒子は互いに指数関数的な相対運動をし図15・1 単純せん断流れ図15・2 3種類の伸長流れ4―166ている.例えば,初めの距離にあった軸上の二つの流体粒子は時間の後には=expεとなる.これに対し,定常せん断流れでは,軸上の二つの流体粒子は時間の後にはγの距離となる.ただし,は十分に大きいとする.伸長流れにおける応力は-+τ=-+=-+τ-+τである.一軸伸長流れや二軸伸長流れでは軸と軸を区別できないから,τ-τ=0となる.伸長のみの流れでは粘度関数はεと流れの型を表すパラメータの関数となる.定常伸長流れではτ-τ=ηε,ετ-τ=ηε,εで定義される粘度関数ηとηがある.=0,すなわち一軸および二軸伸長流れではηε=ηε,0ηε,0=0となり,伸長粘度(elongational viscosity)ηのみを考えればよい.また,伸長粘度はトルートン粘度(Trouton viscosity)とも呼ばれる.
