# 15・3・2 伸長流れ

> **章**: 第15章 非ニュートン流体
> **節**: 15・3 基本的な流れ

粘度というときは、ニュートン流体からの連想でせん断粘度を指すことが多い。しかし、非ニュートン流体の流れでは伸長粘度(elongational viscosity)が重要な役割を担うことがしばしばある。伸長流れとは、例えば小さなノズルから溶融高分子を糸状に押出し、それを軸方向に引っ張ったときの流れで、このときせん断応力は作用していない。この流れを一軸伸長流れという。そして、パラメータによって、次のように分類される(図15・2)。定常伸長流れでは、流体粒子は互いに指数関数的な相対運動をし図15・1 単純せん断流れ図15・2 3種類の伸長流れ4―166ている。ただし、は十分に大きいとする。一軸伸長流れや二軸伸長流れでは軸と軸を区別できないから、τ-τ=0となる。また、伸長粘度はトルートン粘度(Trouton viscosity)とも呼ばれる。
