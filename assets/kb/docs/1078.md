# 7・6・2 加速度パルス入力に対する衝撃絶縁

> **章**: 第7章 過渡応答・衝撃
> **節**: 7・6 衝撃絶縁

衝撃入力を速度ステップにより近似をする場合，加速度パルス入
力に対する応答に誤差を含むことになる．線形1自由度系を例にと
ると図7･21に示すように応答加速度の最大値が過大評価を受ける．
このような誤差が問題となる場合は入力を加速度パルスで扱う必要
が生じる．

図7･22には，代表的理想衝撃パルス入力に対する加速度衝撃ス
ペクトルを示している．絶縁系に減衰がない場合を考える．まず，
与えられた加速度パルスの最大値 ，その積分値 ，およびパ
ルス作用時間 からそのパルスを理想衝撃パルスに置換える．図
7･22で示される衝撃スペクトル曲線から，最大応答値 がその
許容限界値 以下となるωの範囲を読取る．これにより固有
円振動数ωの上限値が決定される．特に線形ばねの場合には，変
位の許容限界値δおよびωの下限値も決定することができる．
一般に固有振動数の下限値は，相対変位衝撃スペクトルを用いて最
大変位量がその許容限界値以下となるωの範囲から求める必要
がある．このように固有円振動数の範囲が決定されれば，それに伴
ってばね特性の範囲も決まることになる．
絶縁系に減衰を含む場合には，減衰比ζをパラメータとして描
かれた衝撃スペクトルを用いるか，またはある減衰比の標準値をも
とにした応答量から他の減衰比に対する応答を換算できる式や表を
用いて絶縁系の特性を決定することが可能である．図7･23には一
例として正弦半波パルス入力による減衰比をパラメータとした衝撃
スペクトルを示している．このような図を用いると，まず減衰比の
値を仮定して，所要のωに関する範囲を読取り固有円振動数の
上限値を決める．固有円振動数が決まれば，ばね定数も決まり，さ
らに減衰比から減衰係数を求めることができる．固有円振動数の下
限値を決めるためには，種種の減衰比の値に対する相対変位の衝撃
スペクトルが必要である．
