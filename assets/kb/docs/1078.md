# 15・4・1 流動曲線による分類

> **章**: 第15章 非ニュートン流体
> **節**: 15・4 純粘性流体

いろいろな高分子流体の粘度を測定すると、粘度はせん断速度が増すにつれ減少すること(シアシニング)が多い。このような性質をもつ流体を擬塑性流体(pseudo-plastic fluid)と呼ぶ。また、せん断速度が増加すると粘度が大きくなる流体もある。これをシアシックニング(shear-thickening)あるいはずり粘ちょう化といい、その流体をダイラタント流体(dilatant fluid)という。このほかに、スラリーや練り歯磨きなどではある応力以上でないと流動しない、いわゆる塑性流体もある。これら三つの流体におけるせん断応力とせん断速度γの関係を図15・3に示す。高分子流体は一般にシアシニング粘性を示す。図15・4に示すよ図15・3 流動曲線図15・4 高分子流体の代表的粘度曲線うに、せん断速度が小さくなると粘度は一定となり、またせん断速度が大きくなっても粘度は一定となる。すなわち、せん断速度が小さい領域と大きい領域でともにニュートン粘性を示し、前者を第一ニュートン領域(first Newtonian region)、後者を第二ニュートン領域(second Newtonian region)と呼んでいる。
