# 15・4・3 非ニュートン粘度の測定

> **章**: 第15章 非ニュートン流体
> **節**: 15・4 純粘性流体

非ニュートン流体の粘度はせん断速度の関数となるので、せん断速度を変えて粘度を測定しなければならない。比較的低せん断速度の領域には、回転型の粘度計が、高せん断速度の領域には細管式粘度計が用いられる。a. 円すい平板粘度計図15・5に示す円すい平板粘度計は、半径の円すいと平板の角度δのすきまに試料流体を入れ、円すいあるいは平板のどちらかを一定回転角速度Ωで回転させる。このとき、流体のすべての部分において近似的に一定せん断速度γ=Ωδの単純せん断流れが実現され、そのときに生じる回転トルクを測定することにより、非ニュートン粘度ηγを次式により決定することができる。図15・5 円すい平板型粘度計ηγ=3δ2πΩb. 細管粘度計円管内定常流の圧力こう配と流量を測定することにより、せん断速度γとせん断応力τの関係を求めることができる。これより非ニュートン粘度が求まる。管壁=におけるせん断速度をγ욌とすると、次の関係が成立する。γ욌=π3+ddこの式をラビノヴィッチ(Rabinowitsch)の式という。ここで、ニュートン流体の壁面せん断速度をΓ=4πとおくと、τ욌=2とΓの間に次の関係が成立する。γ욌=34+1dlnΓdlnτ욌Γ(15・8)ここで、カッコ内はラビノヴィッチの補正項である。非ニュートン粘度はη=τ욌γ욌であるから、とを測定し、Γとτ욌の線図を作れば式(15・8)から粘度ηが求まる。
