# 15・6・1 構成方程式

> **章**: 第15章 非ニュートン流体
> **節**: 15・6 粘弾性流体

一般にひも状のフレキシブルな高分子の流体は顕著な粘弾性を示すが、そのような高分子流体だけでなく多くの流体が粘弾性的な特性を示すことが知られている。したがって、すべての粘弾性流体(viscoelastic fluid)を一つの統一的な構成方程式(constitutiveequation)で表すことは不可能である。これまでにさまざまな構成方程式が提案されてきたが、形式的には微分型と積分型がある。以下に、粘弾性流体の代表的な構成方程式を示す。ただし、これらの構成方程式を用いて、高分子成形加工の数値シミュレーションを行い定量的な検討をする場合に、多重緩和モードのモデルを用いることがある。a. 上対流微分マクスウェルモデル(upper-convected deriva-tive Maxwell model)最も簡単な線形粘弾性一次元モデルであるマクスウェルモデルを、三次元に拡張したモデルである。+λ∇=2ηここで、λ:緩和時間(relaxation time)、η:粘度である。ここで、λ:緩和時間、λ:遅延時間(retardation time)である。c. ホワイト・メッツナーモデル(White-Metzner model)上対流微分マクスウェルモデルの粘度と緩和時間を変形速度の第二不変量の関数とすると、実在の粘弾性流体に近いレオロジー物性を与えることのできるホワイト・メッツナーモデルが得られる。e. ギーゼクスモデル(Giesekus model)弾性ダンベルの溶液におけるビーズの抵抗係数に非等方性を導入することによって、高分子融液や濃厚溶液の構成方程式として、次式が提案された。f. ファンチェン・タナーモデル(Phan Thien-Tannermodel)ネットワークモデルにより導出された構成方程式で、次のように表される。g. ラルソンモデル(Larson model)管模型理論を拡張した微分型の構成方程式である。h. ロッジモデル(Lodge model)ゴム弾性の理論を粘弾性流体に拡張したもので、積分型構成方程式で与えられる。しかし、この構成方程式は閉じた形になっていないので、クロージャー近似を導入した種々の構成方程式が提案されている。その一つに、FENE-Pモデルがある。また、:ダンベルの両ビーズを結ぶ直線の単位方向ベクトルである。
