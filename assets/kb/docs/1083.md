# 15・6・2 レオロジー関数

> **章**: 第15章 非ニュートン流体
> **節**: 15・6 粘弾性流体

粘弾性流体の場合には、粘度や法線応力差(normal stressdifference)あるいは伸長粘度(elongational viscosity)などは一般にせん断速度や伸長速度の関数となる。種々の構成方程式を実在の粘弾性流体に適用するには、構成方程式のレオロジー関数を測定データと比較することにより、構成方程式の適否や構成方程式に含まれる定数を決定することが必要となる。その際に、定常せん断流れや一軸伸長流れにおけるレオロジー関数が利用されることが多い。したがって、オールドロイド-B モデルは、非ニュートン粘性を記述できず、第二法線応力差も零となることがわかる。b. ギーゼクスモデルレオロジー関数は、次のようになる。また、伸長粘度は次式となる。このモデルでは、せん断粘度のシアシニング性と伸長粘度のストレッチシックニング性を表すことができる。また、第二法線応力差も実験結果と同様に負の値となる。図15・8 ギーゼクスモデルのせん断粘度曲線図15・9ギーゼクスモデルの一軸伸長粘度曲線図15・10 ラルソンモデルの一軸伸長粘度曲線4―169c. ラルソンモデルこのモデルでは、図15・10に示すように伸長粘度がピークを示したのちに減少し、実際の高分子融液の伸長粘度の挙動とよく一致する。
