# 15・6・2 レオロジー関数

> **章**: 第15章 非ニュートン流体
> **節**: 15・6 粘弾性流体

粘弾性流体の場合には,粘度や法線応力差(normal stressdifference)あるいは伸長粘度(elongational viscosity)などは一般にせん断速度や伸長速度の関数となる.種々の構成方程式を実在の粘弾性流体に適用するには,構成方程式のレオロジー関数を測定データと比較することにより,構成方程式の適否や構成方程式に含まれる定数を決定することが必要となる.その際に,定常せん断流れや一軸伸長流れにおけるレオロジー関数が利用されることが多い.a. オールドロイド-B モデル定常せん断流れにおけるせん断応力と第一および第二法線応力差は,τ=ηγτ-τ=2ηλ-λγτ-τ=0となり,粘度η=η,第一法線応力差係数Ψ=2ηλ-λ,第二法線応力差係数Ψ=0を得る.したがって,オールドロイド-B モデルは,非ニュートン粘性を記述できず,第二法線応力差も零となることがわかる.一方,定常伸長粘度ηは,η3η=λλ+1-λλ1+λε1-2λεとなり,ストレッチシックニング(stretch-thickening)性を示す.b. ギーゼクスモデルレオロジー関数は,次のようになる.ηη=1-1+1-2αΨ2ηλ=1-αλγα1-Ψηλ=-λγここで,=1-ζ1+1-2αζζ=1+16α1-αλγ-18α1-αλγである.また,伸長粘度は次式となる.η3η=16α 3+1λε1-41-2αλε+4λε-1+21-2αλε+λεこれらのうち,せん断粘度と伸長粘度を図15・8と図15・9に示す.このモデルでは,せん断粘度のシアシニング性と伸長粘度のストレッチシックニング性を表すことができる.また,第二法線応力差も実験結果と同様に負の値となる.図15・8 ギーゼクスモデルのせん断粘度曲線図15・9ギーゼクスモデルの一軸伸長粘度曲線図15・10 ラルソンモデルの一軸伸長粘度曲線4―169c. ラルソンモデルこのモデルでは,図15・10に示すように伸長粘度がピークを示したのちに減少し,実際の高分子融液の伸長粘度の挙動とよく一致する.
