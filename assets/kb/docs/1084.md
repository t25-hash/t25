# 15・6・3 粘弾性流体の特異現象

> **章**: 第15章 非ニュートン流体
> **節**: 15・6 粘弾性流体

粘弾性流体(viscoelastic fluid)は単純せん断流れにおいて法線応力差を生じ、また、顕著な伸長粘度(elongational viscosity)のストレッチシックニング性を示すことがある。それらの特性により、ニュートン流体には見られない特異な現象を生じる。a. ロッドクライミング現象(rod-climbing phenomena)高分子溶液に突っ込んだ棒を回転させるとき、その棒に巻付いてはい登る現象(図15・11)をロッドクライミング現象あるいはワイセンベルグ(Weissenberg)効果という。このような流れでは、閉じた流線に沿って法線応力が生じ、それが回転中心に向かって流体を締付ける力となり、高分子溶液が棒をはい登る結果を引き起こす。b. ダイスウェル(die swell)図15・12のように、粘弾性流体が毛細管から空気中にジェットとして流出するときに、ジェットの直径が毛細管の直径の2~4倍にもなる。この現象をダイスウェルあるいはバラス(Barus)効果と呼んでおり、高分子押出し成形などにおいて重要となる。毛細管中の流体要素は、流れ方向に引伸ばされているが、毛細管から流出するとそれが緩和し、その結果としてジェットの径が膨れることになる。また、長さの短い毛細管を使用した場合には、毛細管入口での縮流の効果によってスウェル比が大きくなる。c. 測圧孔効果(hole pressure effect)固体壁面において粘弾性流体の圧力を測定するときに、ニュートン流体に対して行われるように測圧孔を設けて壁面での圧力を測定すると、法線応力効果により測定誤差が生じる。これに関しては、測圧孔を二次元スロットでモデル化した場合に次のような関係が提案されている。=τ-τ욌4ここで、τ-τ욌は壁面での第一法線応力差である。d. チューブレスサイフォン容器中の粘弾性流体に管を突っ込んで吸い上げると流体は管を流れるが、このときに管の入口を空気中に持ち上げても、流体は途切れることなく容器から吸い上がってくる(図15・13)。この現象を、チューブレスサイフォンと呼んでいる。チューブレスサイフォンにおける液柱内の流れは明らかに伸長流れである。そのために伸長粘度の顕著なストレッチシックニング性を示す粘弾性流体では大きな伸長応力が発生し、液柱を破断図15・11 ロッドクライミング現象図15・12 ダイスウェル図15・13 チューブレスサイフォン図15・14 急縮小流路における二次渦4―170流体工学図15・15正方形断面を有する直管における二次流れ図15・16 配向ベクトルさせることなく吸い上げることができると理解されている。e. 急縮小流路における二次渦直径の大きな容器から小さな管に流れるような急縮小流れにおいて、粘弾性流体は急縮小部において、図15・14に示すような大きな二次渦を生じる。この二次渦の発生も管入口部における伸長流れが原因であると考えられており、多くの研究がなされている。この流れは、粘弾性流体の構成方程式や数値計算方法のチェックのためのモデル流れとしても多く用いられてきた。さらに、流速が増すと不安定流動に移行する。f. 非円形断面直管における二次流れ一様な断面を有する直管内をニュートン流体が層流で流れるときには、流れは管軸に平行な一方向流れとなる。しかし、粘弾性流体の場合には、円管や環状断面以外の断面(例えば、矩形やだ円など)を有する直管に対して、一般には一方向流れとはならずに、管軸と直角な速度成分を生じる。図15・15に正方形断面に対する二次流れの模式図を示す。この二次流れの発生は、粘弾性流体の第二法線応力差が零でないことに起因する。g. 乱流抵抗減少乱流において、溶媒にわずかの量(数ppm)の高分子を添加することによって、著しい抵抗減少が生じることが古くから知られており、トムズ効果(Toms effect)とも呼ばれている。この現象のメカニズムについては、いまだに十分には解明されていない。また、高分子だけでなく界面活性剤においても同様の効果が見出されており、この場合にはせん断誘起構造(shear-induced structure)が関連していると考えられている。
