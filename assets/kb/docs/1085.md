# 15・6・4 液晶性流体

> **章**: 第15章 非ニュートン流体
> **節**: 15・6 粘弾性流体

分子形状が剛直な棒状をしているときには、その流体は液晶(liquid crystal)を示すことがある。ここで液晶とは、結晶的な固体と非晶性の液体との中間的な状態で、自発的に分子の配向に秩序を有する相のことである。また液晶は、ネマチック(nematic)液晶、コレステリック(cholesteric)液晶、スメクチック(smectic)液晶に分類される。そのうちでもネマチック液晶は基本的な液晶で、分子の重心位置はランダムで、配向が平均的にある方向を向いている。平衡状態におけるネマチック液晶は、その平均的な配向方向を示す単位ベクトル(配向ベクトル、direc-tor)(図15・16)とその配向の程度を意味する配向秩序度で与えられる。液晶ディスプレイの材料として用いられている低分子液晶の多くは、ネマチック液晶である。低分子のネマチック液晶に対する連続体理論としては、レスリー・エリクセン理論(Leslie-Ericksentheory)がある。この理論では配向秩序度は外場によって変化しないことが仮定されており、そのために配向ベクトルのみが未知変数として導入される。偏差応力テンソルは、次のように与えられる。さらに、は配向のひずみに対して発生するフランクの自由エネルギーで、次式で与えられる。は分子場(molecularfield)と呼ばれ、磁場の影響も考慮して次のように与えられる。これら一連の式を連続の方程式、コーシーの運動方程式とともに解けば、低分子のネマチック液晶の流れを計算できる。なお種々の配向処理によって、低分子液晶に対して、固体表面上で平行配向や垂直配向などの配向を与えることができる。一方、合成ポリペプチド、DNA、タバコモザイクウイルス以外にも、全芳香族ポリアミド溶液や全芳香族ポリエステル融液などの剛直な高分子の流体も液晶性を示す。これらの高分子液晶は繊維材料や射出成形用材料として用いられ、その製品は高強度・高剛性を有する。高分子液晶の構成方程式としては、棒状高分子の濃厚溶液に対するものとして、土井モデル(Doi model)がある。このモデルでは、配向秩序テンソルの発展方程式が次のように与えられている。また、偏差応力テンソルとして、配向による弾性の寄与分は次のようになる。しかし、この近似は高分子液晶の特徴である単純せん断流れにおけるタンブリングやワギングといった複雑な配向挙動を記述できず、そのために実在の高分子液晶が示す負の第一法線応力差の存在を予測できない。そこで、種々のクロージャー近似の有用性が検討されている。また、長距離秩序弾性の効果を取り入れる試みもなされている。文献(1) 中村喜代次、非ニュートン流体力学、(1997)、48、コロナ社。(2) Christiansen,E.B.and Leppard,W.R.,Trans.Soc.Rheol.,18-1(1974)、65.(3) Bird,R.B.、ほか2名、Dynamics of Polymeric Liquids,2nd ed.,1 (1987)、181,John Wiley& Sons.(4) Papanastasiou,T.C.,J.
