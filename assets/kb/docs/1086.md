# 16・1・1 連続流(continuous flow)

> **章**: 第16章 希薄気体力学
> **節**: 16・1 希薄気体流の分類

分子の運動を特徴付ける統計量として、ある空間に占める分子の質量(密度)、分子の集団としての移動速度(速度)、分子がある仮想面に及ぼす力(圧力、せん断応力)などが定義される。流れの特性時間、特性長さが流体を構成する分子運動の平均自由時間、平均自由行程に比較して十分大きければ、密度、速度、圧力などマクロな性質は、考えている流体の体積、位置、時間に関して連続的に変化すると考えられ、流れは連続体としての取扱いが可能となる。体積内に存在する流体分子の質量をとすると、密度ρ=で定義されるが、体積を小さくしていけば、分子の数の揺らぎにより密度は大きく変動する。連続体では、→0でρに収れんする仮想的な物質を考え、物理量を定義することとする。おおよそ<0.01の状態で、流体は連続体と考えられる。そのような流れを連続流という。
