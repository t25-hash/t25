# 16・1・2 滑り流(slip flow)

> **章**: 第16章 希薄気体力学
> **節**: 16・1 希薄気体流の分類

希薄化の度合いが弱く,流体の内部では連続体として扱えるが,固体面上では有意に流れの速度は零ではなく,滑り速度をもつ状態の流れを滑り流という.おおよそ0.01<<0.1の状態で流れは滑り流となる.気体の平均自由行程をλとすると,固体面よりλ上方の面から流入する気体分子の壁接線方向速度は읉윰=욌+λddと考えてよい.ここで,욌は壁面上の接線方向速度である.一方,壁から脱離して気体中に戻る分子の平均的な接線方向速度윱윲읏=0とすると,壁面上の接線方向速度욌はそれらの平均,욌=읉윰+윱윲읏2=욌+λdd+02となる.すなわち,これから滑り速度욌=λddとなる.
