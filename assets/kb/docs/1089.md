# 16・1・4 自由分子流(free molecular flow)

> **章**: 第16章 希薄気体力学
> **節**: 16・1 希薄気体流の分類

が大きい場合で、分子衝突頻度は、気体分子・固体表面間衝突頻度に比べて無視できるほど高度に希薄化された場合である。おおよそ>10の状態は自由分子流と考えてよい。気体分子はほかの気体分子に衝突することなく、壁面から壁面へと衝突を繰返し、このような気体分子の軌跡を追跡していけば流れを解析することができる。ボルツマン方程式(Boltzmann equation)は気体の流れに対する支配方程式であり、自由分子流から連続流までのすべての流れに適用できる。しかしクヌッセン数が0.01以下の流れではナビエ・ストークス方程式が成立するので、ボルツマン方程式はが0.01以上の流れに対して適用すればよい。が求まると、流れ場の諸量はのモーメントとして与えられる。
