# 8・2・8 信号圧縮（signal compression）

> **章**: 第8章 ウェーブレット変換
> **節**: 8・2 離散ウェーブレット変換

トータルで信号の長さもエネルギーも変えないが，離散ウェーブ
レット変換によって信号のエネルギーが ，＝－1,－2,…に
再配分される．その結果，少数個の係数 に信号エネルギーが
集中する．したがって，絶対値の小さい を零と置換えても信
号のエネルギーはそれほど失われない．こうして，エネルギーを概
ね保持して実質的に信号の長さを減らすこと，すなわち信号圧縮が
できる．その割合は信号の特性と使うフィルタの種類によって異な
るが，典型的には，98 %のエネルギーを保持するのに10%ほどの
個数の を残せばよい．どのような基準で重要な を選ぶか
は，フーリエ変換を利用する信号圧縮と同様で，フーリエ係数の代
わりにウェーブレット係数を対象とする．残された係数からは，逆
ウェーブレット変換によってもとの信号の近似信号が再構成され
る．
