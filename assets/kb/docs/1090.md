# 16・2・1 衝突断面積

> **章**: 第16章 希薄気体力学
> **節**: 16・2 ボルツマン方程式

分子間に働くポテンシャル(:分子間距離)を与えると、散乱角χが相対速度と衝突径数の関数として決まる。散乱角χ,より微分断面積(differential cross-section)が求まりσ,χ=sinχ옿χ옿あるいはが増加するとχは減少する。直径の剛体球分子では=cosχ2となるのでσ=4となる。逆べき型ポテンシャル=は実際の分子間力をかなり正確に表している。ここで、>0,α≧4である。このに対してはχはβ=2αのみの関数となる。このときσ,χ=係数は、剛体球分子α→∞の場合を除きχに依存する。この依存性を無視し、を定数で近似した分子模型を擬似剛体球分子(VHS モデル、variable hard sphere model)と呼ぶ。α=∞では=4とする。一般のαに対するの値は、VHS モデルから求めた粘性係数が実験値と一致するように決める。一般のσ,χに対し全衝突断面積σ윔はσ윔=2πσsinχdχ運動量移行断面積はσ=2πσ1-cos χsinχdχ粘性断面積はσ=2πσsinχdχで定義される。VHS モデルに対してはσ윔=4πまたσ=σ윔,σ=23σ윔となる。剛体球分子ではσ윔=πである。
