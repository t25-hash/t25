# 8・2・9 ノンパラメトリック推定（non-parametric esti

> **章**: 第8章 ウェーブレット変換
> **節**: 8・2 離散ウェーブレット変換

-
ダムな要素が含まれている信号のノンパラメトリック推定に有効な
手段を与える．ノイズを除去したり特徴を抽出する処理がこれに当
たる．例えば簡単なホワイトノイズは，高い解像度のいくつかのレ
ベルにエネルギーが均一に分散する．さまざまなノイズはそれぞれ
の統計的性質によって区別されるから，これを利用してノイズを弁
別し，適切なしきい値を選択する．絶対値がしきい値を超えるウェ
ーブレット係数を処理するには，値をそのまま残すハードしきい値
（hard threshold）と，しきい値の分を差引いて値を縮めて残すソ
フトしきい値（soft threshold）の二とおりがある．これらをうま
く組合わせて所望のノイズ処理を行い，あるいはデータの特徴抽出
を行う方法をウェーブレット縮小法（wavelet contraction）とい
う．ウェーブレットの直交性は，ノンパラメトリック推定に極めて
有利である．
