# 16・2・3 チャップマン・エンスコグ解

> **章**: 第16章 希薄気体力学
> **節**: 16・2 ボルツマン方程式

速度分布関数のマクスウェル分布からのずれが小さい場合には、ボルツマン方程式の一般解(チャップマン・エンスコグ解)が求められている。はオイラー方程式で表される流れ場中の分子の速度分布を与えるが、チャップマン・エンスコグ解はナビエ・ストークス方程式(NS 方程式)で記述される流れ場中の分子の速度分布を表している。チャップマン・エンスコグ解からNS 方程式が求まるばかりでなく、輸送係数,μも決定される。逆に、μの実験値からを決めることができる。上式のαは、μの温度依存性がμ∝となるので、μの実験値を用いて決める。このように気体が指定されるとVHS モデルの係数が定まる。混合気体では異種分子間の衝突を考慮する。通常、標準状態(0°C,1atm)でのの値を用いてを決める。HeとArの混合気体では=6.14×10m/sである。
