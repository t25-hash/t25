c .直接部材に作用する荷重 ここまでは力やモーメントなど
の外荷重が節点のみに作用するとして問題を扱ってきたが，実際の
構造では分布荷重など部材に直接外力が作用することも多い．ラー
メン部材に直接分布荷重や集中荷重が作用する場合には，まず当該
部材の両端を仮に固着し，両端固着はりとしての部材力および固着
端反力と反モーメントを計算する．いま，これを解aとしよう．
次に，この反力および反モーメントと大きさが等しく反対向きの力
およびモーメントを当該節点に作用させた場合の骨組構造の解を解
bとする．解aと解bの和を計算すると，直接部材に荷重が作用
し，かつ固着端反力も打消され，解くべき問題の解が得られる．
トラスの場合には，前述の解aを求めるに際して両端単純支持
とすればよい．
