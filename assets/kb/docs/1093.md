# 16・2・4 ボルツマン方程式の確率解法

> **章**: 第16章 希薄気体力学
> **節**: 16・2 ボルツマン方程式

ボルツマン方程式は直接シミュレーションモンテカルロ法(DSMC法、direct simulation Monte Carlo method)という確率的方法によって数値的に解くことができる。分子間衝突の取扱いをボルツマン方程式から導出した方法もあるが、カッツ(Kac)方程式に基づいた方法が広く用いられている。DSMC法は、原理的にどのようなクヌッセン数の流れにも適用できるが、計算負荷の点から中間流や滑り流を対象にするのがよい。DSMC法では、計算領域にある分子の集団(母集団)から無作為に分子を抽出し、これらの分子から成るはるかに小さな分子集団(標本集団)の挙動を追跡する。標本集団の一つの分子(サンプル分子)は、母集団の個の分子を代表している。サンプル分子の運動の法則はボルツマン方程式から誘導される。サンプル分子の運動の法則を簡単にするために、二つの基本的な近似が導入される。一つは分離の原理であり、もう一つは流れ場のセル分割である。a. 分離の原理標本集団の時間発展を計算するための時間ステップを平均自由時間より十分小にとると、時間における分子の移動とこの間の分子の衝突を分離して考えてよい。これを分離の原理と呼ぶ。分子は自由運動中に固体壁に入射し反射されたり、計算領域から出入りする。これらはすべて自由運動の計算で考慮する。b. セル分割流れ場を平均自由行程程度の大きさのセルに分割すると、分子間の衝突は各セルで独立に考慮してよい。すなわちあるセルの中にある分子は、同じセルに属する分子とのみ衝突することになる。さらに分子間の衝突は互いの空間的な遠近によらないと考えてよい。セルの大きさを平均自由行程程度に選ぶことから、各セルの中では数密度、流速、温度は空間的に一様とみなせる。c. 分子間衝突個のサンプル分子を含むあるセルに着目し、このセル内で時間に起こる分子間衝突について考える。衝突したものは、自由運動後の速度を衝突後の速度で置換える。d. 最大衝突数法の計算時間はに比例し、能率が悪い。最大衝突数法(maximum collision number method)はこれを解消する方法である。衝突計算の手順は次のようになる。衝突するときは(2)へ進む。剛体球分子やVHS モデルでは等方散乱となり、cos χ=1-2,sinχ=1-cosχである。方位角はε=2πである。異なる乱数を同じで表している。例題により流れ図の各部を説明する。以下、異なる乱数をすべて同じで表す。(3) 分子の自由運動すべての分子を時間だけ走らせる。(4) 分子間衝突分子間衝突はセルごとに独立に計算する。衝突はすべて時刻+で瞬時に起こるとし、衝突計算を自由運動の計算から切離してよい。ただし、この分離の原理が成立するためには、を平均自由時間より小に選ばなければならない。すべての分子の自由運動を計算したのち、各セルに入っている分子の数を求める。いま、あるセルに着目し、ここに個の分子が入っていたとする。ただし、は選ばれた分子対の相対速度である。和はセル内の分子についてとる。がさほど大きくないため、これらの量は定常になったあとも時間的に変動する。変動を小さくするには、定常後も計算を続行し、得られたデータを時間平均にすればよい。データの採取間隔は平均自由時間程度にとる。以上が定常流の計算法である。DSMC法では固体面に入射する数流束を直接求めることができるので、半導体製造プロセスで重要な表面反応(ウェーハのエッチング)を伴う流れを容易に計算できる。
