# 8・1・4 ラーメンの塑性崩壊解析

> **章**: 第8章 構造力学
> **節**: 8・1 骨組構造

a.塑性崩壊 前述の骨組に関する議論は部材の応力が弾性範
囲内にあり，かつ座屈が生じない限りは適用できる．骨組構造が不
静定であり，また材料として延性に富んだものを用いる場合には，
荷重の増加とともに安定な塑性変形を生じることがある．このよう
な場合，構造は塑性変形開始後ただちに崩壊するわけではない．構
造物が実際に塑性崩壊する荷重を最終強度（ultimate strength）
と呼ぶ．この最終強度を規準とした構造設計法を極限設計（limit
design）と呼び，そのための崩壊荷重の解析を極限解析（limit
analysis ）という．
