# 16・3・2 超音速自由噴流(supersonic free jet)

> **章**: 第16章 希薄気体力学
> **節**: 16・3 希薄気体の流れ

オリフィスやノズルから真空中に噴出するジェットでは、出口近くでは連続流であるが下流にいくに従って希薄化が進み、順次中間流、自由分子流となる。連続流の範囲ではジェット中心線上のマッハ数は=η-1+1-1η+η>1で与えられる。ここで、η=-,:比熱比、:オリフィスまたはノズルの出口からジェット中心線上に沿って測った距離、:オリフィスまたはノズルの出口口径である。,,は気体によって異なり、2原子分子気体=1.40では=3.65,=0.20,=0.40、単原子気体=1.67では=3.26,=0.31,=0.075である。ジェット温度は等エントロピー関係から=1+-1ここで、:貯気槽温度である。連続流の領域では下流に入っていくに従ってが増大し、が減少する。しかし、中間流の領域に入るとの減少は等エントロピー式で与えられるものよりゆるやかになり、ついには自由分子流の領域で温度は一定値となる。流速も一定値になるから結局マッハ数も一定値をとる。この終端マッハ数は=で与えられ、単原子気体では=1.17である。クヌッセン数は貯気槽状態での平均自由行程とを用いて定義されている。
