# 16・4・6 種々の断面形状配管の自由分子流コンダクタンス

> **章**: 第16章 希薄気体力学
> **節**: 16・4 自由分子流コンダクタンス

真空装置では製作が容易で、外圧に耐える構造がとりやすいという理由から円形配管が用いられてきたが、最近先端科学技術においては装置の複雑化に伴い、円形以外の配管が用いられる場合が増えてきた。いくつかの配管形状の自由分子流コンダクタンスの計算式を、20°Cの空気の場合について表16・3に示す。表16・3の式はスリットを除いて、いずれも配管が長い場合であるから、短い管の場合には式(16・12)と同様な管端の補正が必要である。希薄気体流では、気体分子の固体表面との衝突回数が気体分子どうしのそれに比べて無視できず、気体分子の固体表面における挙動が極めて重要となる。壁面における滑り速度(slip velocity)・温度飛躍(temperature jump)、半導体製膜過程における気体分子の付着確率などでは特に重要である。
