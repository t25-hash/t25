# 16・5・2 適応係数

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

気体分子が固体原子と干渉すると、気体分子と固体原子の間でエネルギーと運動量が交換される。エネルギー交換と運動量交換の素過程の表現として適応係数(accommodation coefficient)が定義されている。例えば、エネルギー適応係数は式(16・20)で与えられる。α=욡-욥욡-욦(16・20)ここで、욡:入射分子の平均並進運動エネルギー、욥:反射分子の平均並進運動エネルギー、욦:反射分子速度分布が固体表面の温度に相当するマクスウェル分布であるときの平均並進運動エネルギーである。運動量適応係数も、表面に対する接線方向および法線方向の運動量について同様に定義される。拡散反射の場合にはα=1となる。固体表面温度より低い温度の平衡状態で散乱する場合には、反射分布束はcos θに比例するがα≠1となり、これを余弦散乱と呼ぶ。
