# 16・5・3 速度滑りと温度飛躍

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

希薄化の度合いが大きくなると,固体表面に接する流体の速度は固体表面の速度と異なり,流体は固体表面で滑る.これを速度滑り(velocity slip)と呼ぶ.同様に,固体表面に接する気体温度も固体の表面温度と異なり,これを温度飛躍と呼ぶ.図16・7に,希薄気体流の固体表面(静止)近傍での代表的な接線方向速度分布を示す.固体表面上の気体の接線方向速度,すなわち滑り速度욦は,滑り距離をとすると,욦=dd욨(16・21)と表される.욡:固体表面に入射する分子群の平均接線速度,욥:固体表面から反射する分子群の平均接線速度とすれば,욦は式(16・22)となる.욦=욡+욥2(16・22)また,μ:粘性係数,:分子の質量とすれば,μdd=욡욡-욥(16・23)であり,入射分子束は욡=14욡で与えられる.なお,は入射分子群の数密度である.固体表面が静止していることを考慮して,接線方向運動量適応係数α욧を式(16・24)で定義する.α욧=욡-욥욡(16・24)これらの式を用いて滑り距離を求めると,=2-α욧α욧2μρ욡(16・25)となる.ここでρ:入射分子群の密度である.粘性係数μと平均自由行程λ욡の間には,μ=0.499ρλ욡욡の関係があるから,α욧≒1の場合には≒λ욡となる.温度飛躍距離についても同じように解くことができ,気体の比熱比がγ=53でエネルギー適応係数がα≒1の場合には≒2λ욡となる.
