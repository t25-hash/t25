# 16・5・3 速度滑りと温度飛躍

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

希薄化の度合いが大きくなると、固体表面に接する流体の速度は固体表面の速度と異なり、流体は固体表面で滑る。これを速度滑り(velocity slip)と呼ぶ。同様に、固体表面に接する気体温度も固体の表面温度と異なり、これを温度飛躍と呼ぶ。なお、は入射分子群の数密度である。固体表面が静止していることを考慮して、接線方向運動量適応係数αを式(16・24)で定義する。ここでρ:入射分子群の密度である。温度飛躍距離についても同じように解くことができ、気体の比熱比がγ=53でエネルギー適応係数がα≒1の場合には≒2λとなる。
