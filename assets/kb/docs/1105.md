# 16・5・4 気体分子と固体表面との干渉に関する実験

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

気体分子と固体表面の相互作用は、固体表面に入射する気体分子のエネルギー、質量、入射角、固体原子の質量や熱運動、固体の結晶構造、および気体分子と固体原子のポテンシャルなどの多くのパラメータに依存している。このようなパラメータを考慮に入れた気体分子と固体表面との干渉に関する実験には、分子線を用いた方法が利用されている。a. 分子線実験装置と飛行時間法による速度分布の計測ノズルビームを利用した分子線実験装置の概要を図16・8に示す。貯気室[1.3kPa~0.1MPa(10Torrから数百Torr)]からノズルを通して膨張室に噴出した気体は超音速自由噴流となる。この噴流にスキマ(入口内径0.5mm以下)を挿入し、膨張の結果高いマッハ数となった流れの中心軸近傍から分子線を取出す。そのビームを照準孔を経て標的室に入れ、固体表面に衝突させる。散乱分子はある反射角方向に設置した真空計や質量分析器などの検出器によって計測される。図16・9に分子線を用いた入射および反射分子の速度分布の測定原理を示した。分子線はチョッパによってある周期で断続された分子群になり、これはチョッパから検出器(質量分析器)までの距離図16・8 分子線実験装置図16・9飛行時間法図16・10 TOFスペクトルへの理論スペクトルのフィッティング4―179を飛行する。分子速度によって検出器への到着時間が異なるため、この到着信号の解析によって分子速度分布関数を求めることができる。これをTOF(time of flight、飛行時間)法と呼ぶ。チョッパスリットの幅が考慮された到着信号の理論式を、実験で得られたスペクトルにベストフィットさせることで、入射分子や固体表面から散乱された分子の速度分布関数を決定する。図16・10は入射分子線のTOF信号に理論スペクトルをフィッティングした例である。b. 流束強度分布非常に粗い表面や汚染された表面では、流束強度分布が余弦法則に従う散乱、すなわち拡散反射の特性を示す。しかし、図16・11に示すように、汚染された多結晶面(Pt,Niなど)からの散乱においては、表面温度の上昇とともに表面が清浄となり、流束強度分布が円形の拡散反射から葉状の分布へと変化する。多くの散乱実験では拡散反射と葉状散乱が組合わさった分布となる。c. 散乱分子の速度分布散乱分子の速度分布はエネルギーや温度の定量的な情報を含み、固体表面との干渉特性を評価する場合には非常に有用である。粗い表面や汚染された表面では吸着・脱離過程が優勢で、散乱分子の速度分布は巨視的流速がなく、表面温度とほぼ同じ温度をもつマクスウェル分布でフィッティングすることができる。しかし、比較的表面温度の低い清浄な表面の場合には、図16・12に示すように散乱分子の速度分布は巨視的流速がなく、表面温度とほぼ同じ温度をもつマクスウェル分布に加えて、巨視的流速をもつマクスウェル分布もTOFスペクトルに含まれる結果となる。後者の成分は固体原子と少ない衝突回数で散乱された気体分子、すなわちほとんど吸着しない分子の速度分布に対応していると考えられる。
