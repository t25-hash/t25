# 16・5・5 気体分子・固体表面干渉モデル

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

気体分子の固体表面との干渉に関するモデルには、DSMC法(直接シミュレーションモンテカルロ法)などの境界条件を与えるための数理モデルと分子線実験における気体分子の固体表面からの散乱現象を解析するための物理モデルがある。a. 数理モデル(鏡面反射モデル、拡散反射モデル、マクスウェルモデル)分子散乱の最も単純なモデルは、気体分子が固体表面で光のように反射すると考える鏡面反射モデルである。このモデ図16・11 汚染表面から散乱された気体分子の流束強度分布図16・12 二つのTOF スペクトルによるフィッティング4―180流体工学ルでは、気体分子の固体表面の接線方向成分は保存され、法線方向成分のみが反転する。拡散反射モデルでは、気体分子はいったん固体表面に吸着し、気体分子のエネルギーが固体のそれに十分平衡した後に、入射方向とは無関係に等方的に反射される。このモデルにより散乱される分子の数密度分布は反射角θ(θは固体表面の法線から計測)の余弦に比例する(余弦法則)。マクスウェルモデルは拡散反射と鏡面反射が混在して起きると考え、拡散反射が起きる確率をα、鏡面反射が起きる確率を1-αで与える。b. 物理モデル分子線による気体分子の散乱現象を解析するには、気体分子のエネルギー、固体表面の原子スケールでの粗さ、固体表面原子の熱振動、気体分子と固体表面原子とのポテンシャルなどを考慮する必要がある。グッドマン(Goodman)らは無次元干渉パラメータを導入して干渉の分類を行い、気体分子の固体表面への入射エネルギーが低く、固体表面の粗さや固体表面原子の熱振動を考慮する必要のない熱散乱領域と、入射エネルギーが高く固体表面の構造を考慮しなければならない構造散乱領域に大別している。高温の気体分子を利用しない一般の分子線実験では気体分子の入射エネルギーが低く、熱散乱領域と考えられる。. ハードキューブモデル、ソフトキューブモデルハードキューブモデルは、熱散乱領域の気体分子の固体表面との干渉を説明するために、固体表面が図16・13に示すような立方体からなるとしたモデルである。このモデルでは、立方体(hard-cube)は表面温度に相当する熱運動をもって垂直方向に運動、気体分子と立方体は弾性衝突、気体分子の接線方向の速度成分は変化しないなどが仮定される。ハードキューブモデルは散乱分子の流束強度分布などはよく説明するものの、適応係数などの微視的な現象を正確には再現できないことが示されている。ソフトキューブモデルは立方体に気体分子との間のポテンシャルを付け加え、さらに立方体を平面にばねで連結したモデルである。ポテンシャルにより固体表面の吸着エネルギーを、バネにより固体原子の熱振動を再現した。このモデルに三次元的な散乱の効果を取込むため、バネの下側に分子レベルの波打ちを考慮したウォッシュボードモデルも提案されている。. 格子モデル格子モデルは、固体を三次元的なばね-質量系(spring-mass system)で近似したモデルである。固体原子の一つに上から垂直に衝突した気体分子の挙動が解析され、高温ではエネルギー適応係数が次式の値に漸近することが示された。α=4μ1+μここで、μ:気体分子と固体原子の質量比である。この結果は希ガスをタングステン表面に衝突させた場合の実験結果とよく一致する。. チェルチニャーニ・ランピス・ロード(Cercignani-Lampis-Lord)モデル入射速度욡で固体表面に衝突した気体分子が反射速度욥で固体表面から離脱する確率욡→욥を考え、接線方向と法線方向の二つの運動量適応係数をパラメータとして含む、熱平衡状態での詳細釣合いを満たす関数により、流束強度分布の再現を可能にしたモデルである。それぞれの方向の適応係数を正確に与えれば、実験結果とよく一致する結果が得られる。. マルチステージ(multi stage)モデル気体分子の固体表面との衝突を三つのステージに分けたモデルである。ステージ1では入射条件から衝突後の並進および回転エネルギーを決定し、ステージ2では等ポテンシャル面で模擬した固体表面の凹凸により気体分子の散乱挙動を決定し、ステージ3では気体分子の離脱または再入射などの衝突後の挙動を決定する。このモデルにより実験結果とよく一致する流束強度分布やTOFスペクトルが再現されている。
