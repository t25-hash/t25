# 17・1・1 基礎方程式

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・1 電磁流体力学

電磁流体力学(magnetohydrodynamics)は,流体の流れ場と温度場と電磁場(electromagnetic field)の相互作用を研究する学問である.温度は流体に浮力として作用し,粘性係数は温度に大きく依存する.流体中を電流が流れると,抵抗によりジュール(Joule)発熱が生じ,温度は上昇する.また,磁場が印加されているとその電流にはローレンツ力が作用する.流体が導電性の場合,磁場中を流体が運動することにより,誘起電圧が発生する.このように,流れ場と温度場と電磁場は相互に干渉し合っている.したがって,流れ場・温度場・電磁場を支配する基礎方程式が必要になる.流れ場と温度場は質量・運動量・角運動量・エネルギーの保存または釣合い方程式により記述される.一方,電磁場はマクスウェルの方程式により記述される.プラズマ・磁性流体・電気粘性流体は多粒子系よりなる混相流である.そして,粒子の回転運動は内部偶力を発生し,応力テンソルは非対称となる.したがって,粒子レベルのミクロな立場から研究する場合と平均量のマクロな立場から研究する場合でその取扱い方が異なる.ここではまず連続体としてのマクロな立場から出発し,その基礎方程式について調べる.すべての流体は粘性があり,圧縮性がある.理想流体や非圧縮流体はその特殊な場合である.同様に,すべての流体は電場により電気分極し,磁場により磁気分極(磁化)する.一般にプラズマや電磁流体図17・1 電磁流体力学の分類では電気分極や磁気分極が無視できるが,それは理想化された特殊な場合と考える.さらに微粒子による内部角運動量の存在を認める.すなわち,連続体ではあるが,マクロからメゾの中へ一歩踏み込む.このような立場で電磁熱流体の理論を構築するのがマイクロポーラ(micropolar)電磁熱流体である.プラズマ・MHD・磁性流体・電気粘性流体の基礎方程式は,マイクロポーラ電磁熱流体の基礎方程式の特殊な場合として導ける.a.
