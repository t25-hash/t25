重が著しく低下する．一方，均一外圧やねじりを受ける長い円筒殻
は初期不整敏感度が低い．一般的な傾向として，座屈変形が不伸長
変形に近い場合ほど，初期不整敏感度が低い．
分岐座屈に関しては，コイター（Koiter ）理論によって分岐点
近傍の初期不整敏感時曲線を表し，既知の初期不整変位に対して座
屈荷重の低下量を推定することができる．飛び移り座屈に関して
は，コイター理論が適用できないので，非線形解析により初期不整
敏感度を求めなければならない．しかし，一般に初期不整は不規則
な未知量であるから，特定の初期不整形状に基づいて得られたこれ
らの解析結果を，そのままはん用的な設計公式とするのは危険であ
る．定量的な設計値は，製作精度，使用環境など初期不整の介在に
関与する因子を考慮して設定しなければならない．
