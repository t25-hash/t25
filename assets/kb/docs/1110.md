# 17・1・3 種々の流れ

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・1 電磁流体力学

a. ポアズイユ・ハルトマン流れ(Poiseuille-Haltmann flow)液体金属の平行平板間の定常流れに、磁場を流れに垂直に印加するとポアズイユ・ハルトマン流れが得られる。電磁流体の厳密解として最も基本的な解である。その速度分布は욈=1-coshηcosh1-tanhで与えられる。ここで욈:断面の平均速度、2:平行平板間の距離、η=,:ハルトマン数[=σμ]である。印加磁場を大きくしていくと速度分布は平担になり、壁面付近にハルトマン境界層が発達する。境界層の厚さはおよそδ~で与えられる(図17・4)。=0は平面ポアズイユ流れとなる。b.
