# 17・1・3 種々の流れ

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・1 電磁流体力学

a. ポアズイユ・ハルトマン流れ(Poiseuille-Haltmann flow)液体金属の平行平板間の定常流れに、磁場を流れに垂直に印加するとポアズイユ・ハルトマン流れが得られる。電磁流体の厳密解として最も基本的な解である。印加磁場を大きくしていくと速度分布は平担になり、壁面付近にハルトマン境界層が発達する。b. MHD クエット流れ(MHD Couette flow)流体金属の平行平板間の単純せん断流れに、磁場を垂直に印加するとMHDクエット流れとなる。図17・5にそれぞれの速度分布が示されている。絶縁壁の場合は流体中を電流が逆流し、閉回路を構成する。完全導体壁の場合はが大きくなると速度分布は平らになり、零に近づく。そして、移動壁のごく近くに薄いハルトマン境界層が形成される。c. アルベン波(Alfven wave)電磁気的な力と流体力学的な力の干渉によって生じる波を、電磁流体波またはアルベン波と呼ぶ。アルベン波は横波の電磁波である。流体の電気伝導度σは無限大で磁気拡散はないものとする。また、流体は非圧縮でその粘性は無視できるものとする。すなわち、ローレンツ力を受ける完全流体のオイラー方程式と完全導体の誘導方程式の連成によってアルベン波は生じる。アルベン波は乱れの方向に垂直で磁場の方向に伝ぱする平面波である。d. MHD レイリー問題(MHD Rayleigh problem)ここでは流体の粘性と磁気粘性の干渉を取扱う。平板を軸方向に一定速度で急に運動すると流体粘性のために平板上に非定常境界層が発達する(図17・6)。ここでνは動粘性係数、λ=1σμは磁気拡散係数である。これがMHD レイリー問題の基礎方程式である。これを静止初期条件のもとに解く。
