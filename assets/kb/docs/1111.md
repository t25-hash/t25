# 17・2・1 液体金属

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・2 機能性流体

金属は常温で一般に固体である。しかし、温度を上げて融点以上に保持すると金属は溶けて液体となる。このような液状の金属を液体金属と呼ぶ。液体金属は一般に高温である。しかし、常温の液体金属もある。そのよい例が水銀や液体ナトリウムである。水銀は毒性が強く、液体ナトリウムは水と反応し危険であるため、よく実験に用いられる低融点の液体金属としてはガリウムとウッドメタルがある。ガリウムは高価であるがウッドメタルは比較的安い。産業界においては溶けた鉄、アルミニウム、チタンなどが液体金属である。これらの金属は一度高温のるつぼで溶かされ、その後精練される。一方、半導体産業においては高純度の単結晶が要求される。この場合も溶けた多結晶は液体金属である。液体金属の特徴は多くの場合、高温で電気伝導度が大きく、かつ非圧縮である。低プラントル数流体で溶けた状態では粘度も低くニュートン流体として取扱われる。最近、地球環境科学が盛んになっている。地球内部の液体核も高温の溶けた鉄が主成分で液体金属である。ここでは地球磁場やダイナモ作用の研究が盛んである。次に、機械工学として重要なトピックスについて記述する。液体金属は電気伝導度が大きいのでよく電流が流れる。磁場の中を電流4―185が流れると、電流にローレンツ力が作用する。このローレンツ力によって液体金属の流れが制御できる。液体金属の電磁場制御の目的はいろいろあるが、流れ場の制御によって目的とする品質の良い金属や単結晶を作ることができる。液体金属には電磁力としてローレンツ力×が作用する。また、液体金属の表面には接線応力:σ욧=1μ욈욧法線応力:σ욈=12μ욈-욧が作用する。a. コールドクルーシブル(cold crucible)溶融金属は高温でるつぼに接触している。この接触はるつぼをエロージョン(ero-sion、壊食)し、その寿命を短くすると同時に不純物が液体金属の内部に混入する。るつぼのまわりに高周波数の交流磁場を印加すると、ファラデーの電磁誘導の法則により溶融金属内部に渦電流が流れる。この渦電流は溶融金属をジュール加熱する。これを誘導加熱と呼ぶ。一方、この電流に作用するローレンツ力によって、溶融金属は表面よりピンチ力を受ける。このピンチ力が大きい場合、るつぼと溶融金属の間に狭いすきまが生じ、溶融金属は浮遊した状態になる。この場合、るつぼは水冷することができるので、コールドクルーシブルと呼ばれる。すきまはるつぼと液体金属を非接触にするため、るつぼの寿命が非常に長くなり、経済効果も大きくなる。今後、このようなコールドクルーシブルの研究発展がプロセス産業において期待されている。b. 電磁せき(magnetic weir)流れる液体はせきによって止めることができる。いま二つの回転ロールの間を溶けた鉄が流れる場合を考える。問題になるのはロールの両端である。両端から流出する溶けた鉄を止めるには、せきが必要である。せきはいろいろな材料によって作られるが、回転ロールと接触するためにすぐに摩耗する。すなわち、部品の交換が必要になり、交換するごとにプロセスを停止する必要がある。溶けた鉄の流出を防ぐために電磁力を使用するのが電磁せきの原理である。c. 単結晶育成(single crystal growth)コンピュータ産業の発展に伴って多量のウェーハが必要になっている。ウェーハの素材は単結晶のインゴットである。この単結晶を育成するのにいろいろな方法がある。電磁場を利用する代表的な方法はCZ 法とFZ 法である。CZ 法に磁場を印加したものがMCZ 法であり、さらに電極に電流を流したものがEMCZ 法である。CZ 法の場合、るつぼの中に自然対流とマランゴニ対流と回転による強制対流が混在する。これらの対流を電磁場により抑制し、流れ場を均一化し、より品質の高い単結晶を作る。印加磁場の方法には水平磁場、垂直磁場、カスプ磁場などがある。誘導電流の分布は磁場分布と流速分布に垂直であり、液体に作用するローレンツ力は流れる電流と磁場に垂直である。その結果、印加磁場に水平な速度成分は制御されず、垂直な成分のみが抑制される。一様磁場の場合、そのメカニズムは容易であるが、カスプ印加磁場の場合は複雑である。重要なことは、結晶の成長界面の流動をうまく制御することである。CZ 法では直流磁場が利用されるのに対して、FZ 法には交流磁場が利用される。これは単に、誘導加熱を利用するからである。この場合、液柱表面に作用するマランゴニ対流の抑制が大きな課題である。d. 電磁ポンプ(magnetic pump)高速増殖炉プラントの冷却材ナトリウムを駆動するのに、電磁ポンプが補助として用いられる。電磁ポンプは可動部がないこと、および電磁力により完全密封型でナトリウムを駆動できることから、安全性と信頼性に優れている。電磁ポンプは大きく伝導型と誘導型に分類される。伝導型の一つに直流式のフローカプラポンプがある。フローカプラポンプは外部から通電するための電極が不要で、ダクトの構造が単純である。ダクトの配列構造や流体の流れる方向により、向流型、並流型、環状型に分類される。4―186流体工学e. 超伝導船(Techno Super Liner)海水は液体金属ではないが、電気が流れる流体である。よってその電気的性質は液体金属と同様である。すなわち、海水に磁場を印加し大電流を流すと、ローレンツ力が作用し、その反力として船は進む。これが超伝導船の原理である。
