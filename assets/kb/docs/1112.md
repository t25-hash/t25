# 17・2・2 磁性流体、MR 流体

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・2 機能性流体

a. いろいろな磁性流体磁性流体(magnetic fluid)は溶媒(母液)と分散粒子(磁性粒子)の組合せにより、いろいろな磁性流体が用途別に開発されている。a項ではそれらを概観することにする。. 水ベース磁性流体マグネタイトなどの強磁性体微粒子の表面を親水基と疎水基をもつ界面活性剤(surfactant)で被覆して、溶媒として水を用いて粒子を分散させたものが水ベース磁性流体である。粒子表面に吸着している界面活性剤は粒子どうしの合体や凝集(agglomeration)を防ぎ、溶媒との親和性により粒子は安定に分散する。微粒子のブラウン運動により、粒子が安定に分散するためには粒子の平均直径が10nm程度までに微細化する必要がある。. ケロシンベース磁性流体界面活性剤にオレイン酸を、溶媒に水の代わりにケロシンを用いたものがケロシンベース磁性流体である。一般に溶媒と界面活性剤と強磁性体微粒子の種類によって磁性流体の特性は変化する。ケロシンベースの磁性流体は、磁場の印加による有効粘度の増加率が一様分散を仮定した理論により、ほぼ記述できることが知られている。. 金属性磁性流体普通の磁性流体は飽和磁束密度が低いためその用途が限られている。しかし現在、飽和磁束密度の高い磁性流体として、金属性磁性流体を開発中である。これには、強磁性体微粒子としてコバルトやニッケルなどの強磁性の金属あるいは合金を用いる。普通の磁性流体では飽和磁束密度が低く0.04T 程度であるが、金属性磁性流体の飽和磁束密度は高く約0.23T である。. 感温性磁性流体磁化の温度依存性が大きい磁性流体を感温性磁性流体と呼ぶ。磁化の温度依存性が大きいと、わずかな温度変化に対して高い駆動力が得られ、磁気対流を制御しやすい。このことは対流熱伝達を促進し、小型高性能の熱交換器および磁性流体ヒートパイプを開発するのに重要である。. 導電性磁性流体油や水を母液とする磁性流体は電気伝導度が低いため、一般に非導電性流体と呼ばれる。これに対して、母液として水銀やナトリウムなどの電気伝導度の高い液体金属を用いた磁性流体を、導電性磁性流体と呼ぶ。導電性磁性流体には体積力としての磁気力(magnetic force)と電流に働くローレンツ力が作用するため、高効率のMHD 発電が可能である。そのためには液体金属の中に、安定な状態で強磁性体微粒子を浮遊させる技術が必要である。. イオン性磁性流体イオン性磁性流体とは、界面活性剤を用いることなく、強磁性体微粒子を安定に分散させた新しいタイプの磁性流体である。これはフェライトの電位決定イオンであるHとOHの吸着を利用し、水溶液中で粒子の表面電気二重層の相互作用の結果生じる反発力によって、粒子を安定に分散させたものである。現在、電気二重層の相互作用による反発力を大きくする技術を開発中である。. 磁性気体普通の磁性流体は母液がすべて液体である。しかし固気混相流の固体粒子として磁性粉を用いると、その挙動は磁性気体のように振舞う。現在この分野で伝熱促進の研究が進められているが、気体中に強磁性流体を浮遊させる技術が開発されると、いままでとは異なった応用製品ができそうである。さらに酸素ガスや窒素ガスも極低温において磁性気体として振舞うことが知られている。b. MR 流体。 生成法および組成MR 流体(magneto-rheologicalfluid)は、シリコン油、ケロシンの液相中に表面処理した1~10mサイズの強磁性体微粒子(カルボニル鉄、ソフト磁性粒子)を体積濃度で10~30%、あるいは、粗密充てんに近い高濃度で、懸濁および沈殿させた黒灰色の流体である。. 機能性とレオロジー特性MR 流体は図17・7の流動曲線に示すように、磁場下で降伏応力(yield stress)が存在するビンガム流体(Bingham fluid)である。磁性流体と比較すると、大きな磁化、磁場印加により100kPa程度に変化する降伏応力、10~10倍の見掛け粘度変化、広範囲な温度作動域、数msオーダの応答時間、強磁性体微粒子の沈殿現象、磁力線方向の強磁性体微粒子のクラスタ(cluster)形成と磁場下での弱い流動性や固化現象が特徴である。. 構成方程式(constitutive equation)図17・7に示すように、ビンガム塑性モデル(Bingham plastic model)の基礎式は、以下のようになる。γ=0, τ≦ττ=τ+η욣γ, τ>ττ=욖=1~2(17・1)ここで、γ:ずり速度、τ:せん断応力、τ:降伏応力、η욣:塑性粘度、:磁場の強さである。また、シアシニング(shear-thinning)粘性を記述するために、高次のモデルとしてハーシェル・バルクレイモデル(Herschel-Bulkley model)は、γ=0, τ≦ττ=τ+γ욗γ, τ>τ(17・2)ここで、塑性粘度は、以下のようになる。η욣γ=γ욗. 種々の流動様式MR 流体をダンパに適用するとき、せん断流れモード(shear flow mode)、バルブ流動機構を有する圧力流れモード(pressure flow mode)、および大きな減衰力を生じる表17・7 定数および物性値ボルツマン定数=1.38×10J/K真空中の透磁率μ=4π×10=1.257×10H/m絶対温度(常温15°C)=288 K粒子の半径=5×10m粒子の流体力学的半径2粒子の数密度10個/m粒子1個の体積욀=4/3π4.1×10m粒子の体積分率φ=욀욈0.041磁気異方性定数(Fe)=0.47×10J/m母液の粘度(水)η=0.1 Pa・sコロイド粒子の飽和磁化욦=0.040 T粒子1個の磁気モーメント=욦욀粒子の密度ρ욦=1.45×10kg/m慣性モーメント=8π/15ρ욦10kg/m渦粘性係数4η=/τ욦=6ηφ=2.4×10Pa・s拡散係数γ1/2τ, γ=η/ρ4―187スクイーズモード(squeeze mode)がある。. 計測法MR 流体の磁場下での流動曲線を求めるために、平行平板粘度計、円すい・平板型回転粘度計および同心二重円筒型回転粘度計がある。そのほか、表面張力計や磁化特性には、磁気天びん計を用いる。. 工学的応用ii項で記述した機能性を利用して、磁性流体に比べ、耐圧性の高いMR 流体シール、mサイズの強磁性体微粒子を利用した研磨、制御しやすく数百kN の大きな力を生じるピストン型MR 流体ダンパや磁場印加による固化を利用したMR 流体クラッチ、バルブ、ブレーキ、また、福祉機器や患部へのガン治療にも試されている。
