# 17・2・3 ER 流体,液晶

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・7 に定数および計算に用いられる物性値のオーダを示しておく.

外部電場の作用のもとで流体の内部構造が変化しそのレオロジー的性質が大きく変化する現象をER 効果(electro-rheologicaleffect)といい,そのような機能を有する流体をER 流体(elec-tro-rheological fluid)と呼ぶ.ER 流体は,特にその粘性を数msのオーダで電気的に可逆的かつ連続的に変化させられるため,電気信号によって直接力の信号に変換し制御できる特筆すべき機能を有している.a. ER 流体の分類これまでさまざまな種類のER 流体が発見され,また新たに開発されてきている.これらのER 流体は,その組成とER 特性の違いから大きく分けて,粒子を分散した粒子分散系ER 流体と粒子を含まない均一系ER 流体に分類できる.粒子分散系ER 流体は,シリコンオイル等の絶縁性の分散媒中に電場下で分極しやすい物質からなる微粒子(一般に,直径1~10m程度)を分散した懸濁液であり,ER 効果発現に水分が関与する粒子を有する含水系ER 流体と,水分を全く含まない粒子を有する非含水系ER 流体に分類できる.含水系ER 流体の粒子としては,粒子表面に付着水として水分をもつ粒子,粒子内部に構造水として水分を含む粒子,およびイオン解離基を有する高分子電解質粒子などが挙げられる.また,非含水系ER 流体粒子としては,電子の移動がER 効果に関与する半導体粒子や導電体粒子,バルクとして誘電率の大きな強誘電体粒子などがある.均一系ER 流体の代表として,低分子液晶や高分子液晶があるが,ほかに,極性低分子液体(ニトロベンゼン,ニトロメタン),非極性低分子液体(ベンゼン),極性高分子溶液(ポリウレタン系溶液)なども挙げられる.b. 粒子分散系ER 流体. 機能発現のメカニズム粒子分散系ER 流体の機能発現は,基本的には電場下で電気分極(electric polarization)した多数の粒子によるクラスタ(架橋構造)の形成に起因する.すなわち,図17・8(a)に示すように,電界の方向に電気的に分極した一つの粒子に着目すると,電界方向に対して±55°以内に位置する他の粒子とはお互い静電引力で引付け合うが,その範囲外にある他の粒子とは斥力が生じて反発する.そのため,無電場下で分散媒中に不規則に散在していた微粒子は,電場を印加すると電界方向に粒子どうしが鎖状につながり多数のクラスタ構造,さらには,複数のクラスタが凝集した集合体(カラム)構造を形成して電極間を物理的に結合する.流動方向に対して直角に形成されるこれらのクラスタが流動抵抗となり,見掛け上流体の粘性が増加したように振舞うの(a) 分極した粒子間の相互作用(b) 電場下でのクラスタ形成図17・7 流動曲線図17・8 粒子分散系ER 流体の電場下での粒子クラスタ構造の形成4―188流体工学である.電極間に形成された微粒子からなるクラスタが,電極が相対運動をする場合にはせん断流れに対する抵抗となり,移動電極には抵抗力が働く.また,電極が固定されて電極流路間の圧力差による流れがある場合には,クラスタがその流動抵抗となり,電極間への印加電圧によって電極流路間の圧力差や流量を変化させることができる.すなわち,ER 流体の圧力や流量を制御するバルブとしての機能をもたせることができる.分散粒子の電気分極のメカニズムは,基本的にはイオン性解離基含有型,電子分極型および強誘電体型の3種に分類される.イオン性解離基型粒子では,そのまわりにイオン解離基がほぼ均等に配列された電気二重層が形成されるが,電場印加のもとでは解離イオンが移動してイオン分布にひずみが生じ,電気双極子を構成することによって粒子間に静電引力が発生する.半導体粒子のような電子分極型粒子では,電場印加によって粒子内の電子が電場の方向に移動し,粒子全体が電気的に分極する.. ER 効果と流動特性粒子分散系ER 流体は,基本的には分散粒子の分極によるクラスタ構造の形成に起因してそのレオロジー的性質が変化することから,電場印加によって形成された粒子クラスタ構造を崩壊する力,すなわち降伏せん断応力τが現れ,その応力以下の応力では流動せず,応力が降伏せん断応力を超えると流動するビンガム流体(Bingham fluid)に類似した挙動を呈する.そのため,そのER 特性は降伏前と降伏後との特性に大きく分けられる.図17・9には,二つの平行平板電極が相対滑り運動する,せん断流れにおける粒子分散系ER 流体のせん断応力τの,降伏前と降伏後の一般的なER 特性を示す.図17・9(a)に示すように,降伏ひずみγ以下のせん断ひずみでは,せん断応力がせん断ひずみに比例して増加する弾性体としての特性が主となり,電場強度(=印加電圧/電極間げき)が高くなるにつれて降伏せん断応力が大きくなり,弾性的特性がより顕著になる.降伏ひずみ以上のせん断ひずみを与えると,降伏現象と塑性流動が起こり粘性流体としての挙動を呈し,図17・9(b)に示すように,せん断応力はせん断速度に比例して大きくなる.結局,粒子分散系ER 流体の誘起せん断応力τは式(17・3)のように表せる.τ=τ+ηγ(17・3)ここで,η:流体の無電場時の基底粘度,γ:せん断速度である.また,i項で記したように,粒子間相互作用は粒子の電気分極によって発生するものであり,粒子の界面効果が無視できるものと考えると,降伏せん断応力τは式(17・4)で表される.τφεε욂ε욣-ε욂ε욣-2ε욂(17・4)ここで,φ:粒子の体積分率,ε:真空の誘電率,ε욂およびε욣:それぞれ分散媒および粒子の比誘電率である.このように,降伏せん断応力τは粒子濃度に比例し,外部電場強度の二乗に比例して増加する特性を示す.以上のように,粒子分散系ER 効果が現れる.一般に,液晶の誘起せん断応力は粒子分散系ER 流体とは異なり,有意な降伏せん断応力は現れず,電場下でもせん断速度にほぼ比例して増加し,ニュートン流体に類似した特性を示す.その増加割合が電場強度によって変化するER 特性を示す.液晶のER 効果発現のメカニズムは,基本的には電場下における液晶の棒状分子の配向挙動に起因する.すなわち,せん断流動下における液晶の棒状分子の平均的な配向方向(ダイレクタ)と流動方向との相対的な方位関係によって,図17・10に示すような異方的なミエゾヴィク粘度(Mies-owicz viscosity)η욀,η욁,η욂η욂>η욁>η욀が定義できる.定常的なせん断流動下においては,液晶の分子は流動方向に配向して粘度はη욁に漸近するものと考えられる.この状態でせん断方向に垂直に電場が印加されると,誘電率の異方性によって液晶の棒状分子(あるいは液晶ドメイン)が電場方向に配向して,理想的には異方的な粘度がη욁からη욂に上昇すると予想される.η욂がη욁に比してはるかに大きな値を示す場合には,極めて大きなER 効果が発揮される.η욂は液晶棒状分子(あるいは液晶ドメイン)の高分子量化によって増大すると予想される.実際に,電場下で電界方向に棒状分子が配向する正の誘電異方性をもつp型のネマチックの低分子液晶では,η욂が小さいと考えられ数倍程度のER 効果しか認められていないが,液晶ドメイン間に相互作用をもたせるために,一つのフレキシブルな分子鎖に複数個の液晶性基を結合させた液晶性高分子溶液では,10~30倍程度のER 効果が確認されている.d. 工学的応用ER 流体は,産業応用という面からも夢の膨らむ機能性流体であり,流体を媒体として動力を伝達するクラッチ,流量や圧力を制御するバルブ,そして液体粘性を利用したダンパなど,流体が介在する機械要素への応用と同時に,振動制御,ロボット制御,アクチュエータ制御,マイクロマシン制御などの各種制御システムへの応用も期待され,種々の産業応用が提案され試みられ,活発な研究展開がなされてきている.1990年代当初,ER
