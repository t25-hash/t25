# 17・2・4 プラズマ流体(plasma flow)

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・7 に定数および計算に用いられる物性値のオーダを示しておく。

a. 生成法と分類プラズマは、気体にエネルギーを与えることにより気体の一部が電離した状態であり、イオン、電子などの荷電粒子と原子、分子の多成分粒子から構成される。全体としては電気的中性で、プラズマ構成粒子は、個別的挙動およびデバイの長さ(debye length)λが関係する集団的挙動を示す。プラズマの性質は、作動圧力が重要因子で、プラズマ密度とプラズマ温度により大きく変化する。生成法としては、コロナ放電、グロー放電、アーク放電などの直流(DC)放電と、高周波(RF)放電、マイクロ波放電やパルス放電がある。これにより、大気圧程度で用いられる熱プラズマ(thermal plasma)と、0.1~10Paの低圧下で電子温度が重粒子温度に比べて高い低温プラズマ(cold plasma)がある。b. 物理化学的性質と素過程プラズマ流体の機能性発現には、以下の性質が関与する。解離(dissociation)、励起(ex-citation)、電離(ionization)、再結合(recombination)などの素過程による荷電粒子の生成と消滅、粒子間の弾性または非弾性衝突および電磁場下での荷電粒子の加減速運動やラーモア半径のサイクロトロン運動、シース(sheath)に代表されるプラズマと壁面との干渉、温度と圧力の関数であるプラズマ物性、高温下で大きな変物性を有する乱流構造、アーク柱の変動や電離不安定性に起因するプラズマ流の不安定現象、異なる反応速度や低圧下で起こる温度こう配や濃度こう配の大きな領域での熱非平衡状態(thermalnonequilibrium condition)、高エネルギー電子や活性ラジカル自身、または窒素、水素、アルカリ金属などの反応性気体との混合による高機能化、放電形態、流動形式や電磁場による制御などである。c. 基礎方程式粒子間衝突が頻繁な高圧下での熱プラズマでは、局所平衡(local thermodynamic equilibrium)を仮定し、連続流体として流体モデル(fluid model)を用いる解法がある。また、低圧下の低密度プラズマでは、個々の粒子軌道を追跡する粒子モデル(particle model)による解法がある。熱平衡プラズマでは、流体モデルに機能性を考慮して、前述のMHD 方程式(17・1・1項参照)が適用できる。また、電離平衡下での荷電粒子密度は、サハの式(Saha equation)から求められる。욄욡욈=2욡욈2π욄요(17・6)ここで、욄,욡,욈:電子、イオン、中性粒子の数密度、욡,욈:イオンおよび中性粒子の分配関数、ε욡:電離エネルギー、:プランク定数、:ボルツマン定数である。熱非平衡プラズマでは、二温度モデルを用いる。この場合、電子温度욄のエネルギー方程式は、以下になる。DD욄32욄+ε욡4―189=-욄52욄+ε욡∇-∇욄-ε욡욄2욄욠ν욄욠욄32욄--(17・7)+욄+×-∑욯ここで、:磁束密度、:電場の強さ、욄:電子電流密度、욄:電子熱流束、:放射損失、ν욄욠:電子・重粒子間平均運動量交換衝突周波数である。さらには、電子、イオン、中性粒子からなる弱電離プラズマに適用する三流体モデルがある。一方、一様な直流電磁場下の電荷の単一荷電粒子の運動に粒子モデルを適用すると、dd=dd=+×(17・8)ここで、,⊥は電磁場に平行または垂直な成分である。また、両者の中間領域では、流体を粒子間の衝突を考慮して運動論的(kinetic)に取扱うことが必要であり、前述のボルツマン方程式(16・2・3項参照)に基づく解法が必要である。d. 種々の流れ流動場の状態は、高密度プラズマでは連続流体、低密度プラズマでは希薄気体の取扱いとなる。また、層流および乱流、熱あるいは化学的に平衡または非平衡な状態もある。噴流としてはプラズマジェット、外部流では、平板上、円柱まわり、また内部流としては、円管内、ダクト内流れ、そのほかに、相変化や化学反応を伴う粒子や液滴を含んだ複雑プラズマ流、電極間のアーク流もある。e. 診断法電子温度や電子密度のプラズマパラメータや熱流体力学特性、そして分光特性は、パターン認識、可視化などを統合した計測診断から明らかにされる。受動的測定として、放電電流・電圧計測、ラングミュア探針(Langmuir probe)測定、プラズマから放射される電磁波を分光法(spectroscopy)によるスペクトルの絶対強度法(absolute spectral intensity method)や相対強度法(relative spectral intensity method)を用いて測定する方法などがある。また、レーザによる能動的測定として、プラズマに電磁波を入射し、その反射、干渉、透過、散乱、屈折を利用する方法もある。f. 工学的応用エネルギー変換機器として、プラズマ燃焼、MHD 発電、プラズマ推進機、制御熱核融合、プラズマプロセシングとして、プラズマ溶射、微粒子創製、ナノ構造体、CVD、スパッタリング、イオンプレーティング、エッチング、表面改質、さらには、アーク溶接・切断やプラズマ製鉄などがある。また、環境浄化プロセスとして窒素酸化物、硫黄酸化物、フロン、ダイオキシンの分解や焼却灰の溶融固化処理、難溶性ごみの減容化、滅菌などがある。さらには、光源やプラズマディスプレイパネルなどの電磁波への応用もある。生体内においてはリンパ系、泌尿器系、消化器系などさまざまな流れがあるが、17・3節では代表例として血液の輸送を担う循環器系と、外気とのガス交換を行う呼吸器系について述べる。
