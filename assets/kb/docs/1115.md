# 17・3・1 血液のレオロジー

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・3 生体内の流れ

血液は液体成分である血しょう(漿)に血球などの有形成分が分散したものである。血しょうは水(90%)、タンパク質(7%)などからなる。有形成分のほとんどは赤血球(99%)であり、ほかに微量の白血球、血小板などが含まれている。血液中の赤血球の体積割合をヘマトクリット(hematocrit)と呼び、ヒトの場合、通常0.4~0.45の範囲にある。血液のレオロジー特性は非ニュートン性を示すことが知られている。血液のレオロジー特性を記述する最も簡便な方法は、ニュートン流体の定常層流を仮定して、円管内の圧力こう配から求めた見掛けの粘度を用いる方法である。実験によれば、せん断速度が30s4―19 0流体工学表17・9ヒト血液の物性値比重粘度Pa・s全血1.064.0×10血しょう1.031.3×10赤血球1.09―*:37°C、ヘマトクリット0.45、男子表17・10 ヒトの血管における血流の条件内径cm平均速度cm/sレイノルズ数γsμ욂γPa大動脈0.5毛細血管0.00050.050.0012.6γ:壁面での平均せん断ひずみ速度、μ욂:高せん断速度での血液粘度表17・11 ヒトの気道の諸元程度以上で、管直径が1mm程度以上であれば、見掛けの粘度はほぼ一定値となることが知られている。このようにニュートン流体として扱える場合の血液の物性値を、血しょうおよび赤血球の値とともに表17・9に示す。これに対して、血流の速度が小さい場合や管直径が小さい場合には血液の非ニュートン性が顕著となってくる。前者については赤血球が低速度領域で凝集体を形成する性質が、後者については赤血球が外力により容易に変形する性質がおもに関係している。赤血球は中央がくぼんだ円板形(直径8 m、厚み2 m程度)である。静止状態の血液においては、赤血球間の結合力によりそれらは凝集体を形成する。血液の流動に伴うせん断速度の増加とともに、結合が破壊し見掛けの粘度は減少する。この血液のレオロジー特性は一般の分散媒質で成立するカッソンの式(Casson’s equa-tion)で記述できる。τ=τ+μ욂γτ=0≦0.1>0.10.625-0.1μ욂=μ욉1-≦0.56(17・9)ここで、τ:せん断応力、τ:降伏応力、μ욂:高せん断速度における血液粘度の漸近値、μ욉:血しょうの粘度、はヘマトクリットである。ヒトの血管内の血流の条件を表17・10に示す。式(17・9)でヘマトクリットを0.4とすると降伏応力はτ=0.006Paとなり、表血管内の平均的な血流では式(17・9)の第1式右辺第1項を省略し、血液をニュートン流体で近似できる。ただし、心臓の拍動に伴って血流速度が低下した状態や、血管分岐部において血流が停滞するような領域では降伏応力の寄与が無視できない。血液のレオロジー特性を決定する第二の要因である赤血球の変形能の影響は、以下に述べる血管径が小さい微小循環において顕著に現れる。
