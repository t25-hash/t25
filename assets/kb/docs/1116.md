# 17・3・2 血液と微小循環

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・10 の最右列の応力値に比べて十分小さい.したがって,実際の

心臓から出た動脈は分岐して細動脈となり,さらに分岐して毛細血管(最小直径は5 m程度)となったのち,合流して細静脈となる.これら細動脈から細静脈までの直径が数十m以下の微小血管における血液循環を微小循環と呼ぶ.微小循環における血流は赤血球の変形能の影響を強く受ける.柔軟な粒子が分散した媒質が管内を流れる際に,粒子には管軸方向に力が作用することが知られている.血管内の血流においても,血管壁面近傍には赤血球の存在しない厚さ数m程度の血しょう層が存在する.血液が細管内を通過する際に,ヘマトクリットが初期の値より低下する現象が見られる.これはファーレウス効果(Fa°hraeuseffect)と呼ばれており,図17・11に示すように,管の直径が1図17・11 毛細管内の血液のレオロジー特性の模式図内径cm長さcm平均流速cm/sレイノルズ数気管1.820世代気管支0.050.080.30.1mm程度以下に減少するにつれてヘマトクリットが減少し,15~20 mで最小値をとる.赤血球が管軸方向に移動するため,赤血球の平均速度が血液全体の平均速度より大きくなり,その結果ヘマトクリットが低下するものである.ヘマトクリットの減少に伴い,見掛けの粘度も減少するが,これはファーレウス・リンドクビイスト効果(Fahraeus-Lindqvist effect)と呼ばれる.管の直径がさらに小さくなると,赤血球が流れにくくなってヘマトクリットは増加するようになる(逆ファーレウス効果).また見掛けの粘度もそれに伴い増加する(逆ファーレウス・リンドクビイスト効果).上で述べた赤血球の変形能に起因する諸現象は,血管でも,ガラス管でも共通に見られる.生体に特徴的な現象としては,血しょうタンパクなどが複雑に絡み合った0.5 m程度の内皮表面層と呼ばれる層が血管内表面上に存在し,血液の流動特性に影響を与えることが知られている.
