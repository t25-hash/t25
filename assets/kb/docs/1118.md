# 17・4・1 超流動(superfluidity)ヘリウムの性質

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・4 超流動

ヘリウムは,絶対温度4.2K で気体から液体に変わり,絶対零度まで冷却しても固体にならない.この液体ヘリウムは,さらにラムダ温度2.172K 以下で,粘性が消失した超流動という劇的な状態に転じる(ラムダ温度より高温側をヘリウムI,低温側をヘリウムIIという).この現象の本質は,本来ミクロな世界の支配則である量子力学的性質が,極低温に冷却することによりマクロなスケールで現れることにある.液体ヘリウムは量子効果が顕著に現れる液体という意味で,量子液体または量子流体(quantum fluid)と呼ばれる.極低温で量子効果が顕著になる理由は次のとおりである.量子力学の本質は粒子と波動の二重性にある.粒子と考えられていた原子や電子が実は波の性質も有し(この波を物質波という),波と考えられていた光が粒子の性質をも有すというものである.もっとも十分温度が高ければ,原子の物質波の波長は平均原子間距離に比べて十分短く,波動性は事実上現れない.しかし,ヘリウムのような軽い原子を極低温に冷却すると,その物質波の波長は増大して平均原子間距離と同等またはそれ以上になり,波動性すなわち量子効果が現れるようになる.超流動は,量子統計力学により初めて理解できる現象である.一般に,十分高温ではすべての粒子はマクスウェル・ボルツマン統計に従うが,低温で量子効果が現れると,その波動関数の対称性に依存して,ボース・アインシュタイン統計かフェルミ・ディラック統計かのどちらかに従うようになる.自然界には2種類の安定なヘリウム原子の同位体HeとHeが存在し,Heはボース統計に従うボース粒子,Heはフェルミ統計に従うフェルミ粒子である.両者の差は,原子核の中の中性子1個の多寡だが,それが両者の従う統計性を分かち,ひいてはその液体のマクロな性質をも決定的に変えてしまう.17・4節は液体Heの超流動現象を述べる.これはボース粒子Heからなる多体系がボース・アインシュタイン凝縮[以下,ボース凝縮(Bose condensation)という]を起こすことにより生じる.フェルミ粒子Heからなる液体Heは,約1mK以下の超低温で,やはり超流動になることを付記しておく.
