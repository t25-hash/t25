# 17・4・2 量子効果と特異な現象

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・4 超流動

ヘリウムIIは,通常流体には見られない特異な現象を示す.まず,ヘリウムIIは粘性がないため,普通の粘性流体が通り抜けられないような直径ミクロン程度の毛細管でも通り抜ける(スーパーリーク).この場合,単位時間当たりの流量は管両端の圧力差に依存しない.次に,フィルムフローと呼ばれる現象がある.ヘリウムIIの入ったビーカを,より大きな容器に入ったヘリウムIIに漬けたとしよう.ビーカ内の液面が外部の液面より低ければ,外部のヘリウムIIがビーカをはい登って侵入し,内外の液面を同じくする.逆にビーカ内の液面が高ければ,内部のヘリウムIIがはい出し,やはり内外の液面が等しくなる.このとき,厚さ200~300Åの薄膜がビーカの壁面に沿って流れていることがわかっている.次に,流体中で温度差と圧力差が直接結び付くことにより,熱機械効果および機械熱効果と呼ばれる現象が起こる.ヘリウムIがヘリウムIIに転移すると,熱伝導が非常に良くなることも知られている(超熱伝導).超流動の本質は,ボース粒子Heの多体系が引き起こすボース凝縮である.凝縮という言葉で普通連想されるのは,例えば,気体の温度を下げたときに起こる液滴の凝縮であろう.これは,空間中に一様に分散していた気体分子が特定の場所に凝縮したものだが,これが運動量空間で起こったものがボース凝縮である.液体ヘリウムをある臨界温度以下に冷却すると,系の全原子数に匹敵する大量のヘリウム原子が運動量最低状態にたまるようになる.通常,流体の粘性は分子間の相対速度のために生じるが,このように大量のヘリウム原子が同一の運動量最低状態を占めると,粘性も効きようがない.これが超流動の本質である.このボース凝縮の考え方を現象論的に発展させたのが二流体モデル(two-fluid model)である.二流体モデルでは,ヘリウムIIが独立な二成分,超流体と常流体から構成されるとする.常流体はエントロピーをもつ粘性流体だが,超流体はエントロピーを担わない非粘性流体(理想流体)である.ラムダ温度以上のヘリウムIではすべて常流体だが,ラムダ温度以下でヘリウムIIを冷却していくと常流体が減少して超流体が増加し,絶対零度ではすべて超流体となる.この二流体モデルにより,先に述べたヘリウムIIの特異な熱流動特性はよく説明できる.二流体の混合比の温度変化も観測されている.二流体モデルは単なる現象論にとどまらず,ボース凝縮系の本質に関わっており,量子統計力学により理解することができる.
