# 17・4・3 量子渦(quantized vortex)の力学

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・4 超流動

粘性がなく熱伝導が良いヘリウムIIは優れた冷媒として使えそうだが、管内を流すと比較的低い流速で乱流状態に遷移し超流動は壊れる。この超流動乱流では、ボース凝縮に起因する量子渦と4―19 1いう特殊な渦が重要な役割を担う。量子渦は超流体流れが作る渦であり、その循環はを単位として量子化される(:プランク定数、:ヘリウム原子の質量)。通常流体の渦の循環が任意の連続的な値をとりうることとは異なる。また量子渦の芯半径は原子サイズと、非常に細いこともわかっている。ヘリウムIIをある臨界流速以上で流すと、量子渦が高密度で絡まり合った渦タングルが形成され(超流動乱流)、それと常流体流れとの間に生じる摩擦により、ヘリウムIIの非粘性は破壊される。これはヘリウムII管内流の実験と、量子渦の運動の数値シミュレーションにより明らかにされている。
