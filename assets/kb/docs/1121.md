# 17・5・1 化学反応を伴う流れの基礎式

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・5 反応流れ

大きな密度変化や発熱を伴う化学反応系は燃焼に準じるので、ここでは、反応による密度変化や発熱が無視できる均一化学反応について述べる。νを量論係数(stoichiometric coefficient)として化学反応には、一つの化学式で与えられる単一反応[式(17・11)]と式(17・12)のように反応がいくつかの反応を経て進行する逐次反応、並発反応などの複合反応がある。式(17・11)(17・12)の量論係数νが1でない場合、一つの反応式中の一成分に対する反応速度(reaction rate)が決まれば、各成分の反応速度を各量論係数νで除した量論式に対する反応速度が等しくなる関係を用いて、各成分に対するを求めることができる。このほかにも律速段階近似法がある。なお、反応速度定数はアレニウス(Arrhenius)の式で与えられる。
