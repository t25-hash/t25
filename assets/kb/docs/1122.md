# 8・2・8 物体に働く力の軽減

> **章**: 第8章 管路内の流れおよび流体中の物体に働く力
> **節**: 8・2 流体中の物体に働く力

線上，またはやや内側の最適位置に設置する場合 ，コアンダ効
物体に働く抗力軽減（drag reduction）は重要な問題である．球
において，臨界レイノルズ数以下でも人為的に境界層を乱流遷移さ
果によるせん断層の再付着，あるいはせん断層の伸長が起こり，
は30%軽減され＝1.0の揚力が発生する．予測された大幅
せると，抗力係数が激減することは図8･39のトリッピングワイヤ
な変動流体力の抑制が実証されている ．他の一つは物体の上流
の実験でよく知られている．乱流遷移の典型的な例がゴルフボール
側にロッドを設置する方法である．図8･82に示すように，正方形
のディンプルに見られる．粗さによる球の乱流遷移 は図8･42
断面柱の抗力軽減（drag reduction,square prism) は著しい．
のようにレイノルズ数 ～10で実現する．図8･80の円柱 の
また，同図の上段に示すように流れ場は一変する．ロッドからの渦
抗力係数（drag coeﬃcient,circular cylinder ）も40 %以下に減少
放出のあるパターンAと，渦放出が停止し再循環領域が形成され
する．また，円柱の乱流遷移の条件 が与えられている．ここで
るパターンBに大別される．ロッドの付加抗力を含んだ総抗力係
は，外部動力を必要としない簡便な方法，小さな制御用ロッド
数はパターンAで50 %，パターンB（両物体の軸間距離＝
（control rod）を用いた二次元物体の抗力軽減法を述べる．一つは
2.5，ロッド径＝0.3）では70%も減少する．ここで， は正
図8･81に示す小さなロッドを円柱からはく離したせん断層の分割
方形柱の辺長である．この場合40%の伝熱促進（heat transfer en-
hancement ) が達成される．また，円柱 の抗力係数は図8･
83に示すように，パターンBの円柱直径 に対しロッド径＝
0.15～0.25，ロッド位置＝1.75～2.0のとき最も軽減され，レ

63%に減少する．この場合の伝熱促進 は30～40%である．ロッ
ドの代わりに小平板をおいても正方形柱に働く変動流体力は大幅に

mal flat plate ) は最大30 %である．
なお，図8･84のように円柱の前面をθ＝53°でカットすると，
端部からはく離したせん断層が物体面と一致し，乱流境界層を形成
し乱流はく離する．このため抗力は円柱の抗力の50%に減少す
る ．球の場合も同様で，球の抗力の25%に減少する ．
