# 17・6・1 MEMS における流れ

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・6 マイクロ流れ

微小電気機械システム(MEMS : micro-electro-mechanicalsystems)において,超微小液滴の連続生成技術が注目されている.超高解像度プリンタ技術,それを利用したDNA チップ,および,μ-TAS(μ-total analysis system)への応用が期待されている.数pLの体積をもつ液滴の生成は実用段階にあり,さらなる微粒化が模索されている.粒径の予測・制御に最も重要なことは,液滴の生成過程,すなわち,液体界面の変動やその破断現象を的確にモデル化することである.このような微小スケ―ルの界面流動現象では,通常の流体力のみでなく,分子間引力が支配的になってくる.工業的にはレナード・ジョーンズポテンシャル(Lennard-Jones potential)φ욡욢=4εσ욡욢-σ욡욢(17・19)に基づく分子動力学シミュレーション(molecular dynamic simu-lation)が有用である.욡욢は分子間距離を表す.εおよびσは,それぞれ,エネルギーおよび距離の次元をもつパラメータであり,種々の物質に対するデータテーブルが用意されている.レナード・ジョーンズポテンシャルを用いた超微小液滴生成のシミュレーションと線形安定性理論による結果との比較検討がなされている.図17・12は静止した超微細円筒液体が崩壊し,液滴が生成される様子をシミュレーションしたものであり,無次元波数は線形安定性理論による解析結果:=2π욧λ=0.697とほぼ同じ値になることが明らかになっている.ただし,욧:円筒液体半径,λ:界面変動の波長である.これは,生成される液滴の半径が約1.89욧になることに相当する.円筒液体が噴流状の場合,さらに小さな液滴が生成される.
