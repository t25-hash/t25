# 17・7・2 微小重力実験

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・7 無重力流体

a. 微小重力下の界面不安定現象重力の影響を除くと、おもに表面張力が支配的な界面不安定現象の観察に有利である。特に、液滴や気泡は完全な球形にすることができるので、理論との比較が容易にできる。液柱の振動では、大直径の液柱の実験でレイリー不安定の観察が行われ、理論どおりの不安定波が生じることが確かめられている。液滴の振動では、三次元大振幅の液滴振動が実現でき、理論との比較が行われている。気泡や液滴の界面不安定は気泡流や液滴流のような二相流にとってもその流動様式を決定する要因となり、重要である。容器内の液面の挙動や不安定性は液体のぬれ性や表面張力の影響が卓越する。磁性流体の界面不安定も微小重力場と重力場とでは異なる挙動をとることが知られている。b. 微小重力下の熱流動熱流動現象として重要な現象としては、沸騰とマランゴニ対流(Marangoni convection)が挙げられる。沸騰現象では、加熱面で発生した蒸気気泡が伝熱面から離脱しないか、あるいは伝熱面近傍に停滞するので、伝熱面への液体の供給を妨げることになる。その結果、伝熱面の温度が上昇しバーンアウトする。マランゴニ対流は表面張力が温度依存性をもつために生じる流れである。自由界面での表面張力は一般に低温側で大きく、高温側で小さい。したがって、両端の表面張力差が粘性力を上まわれば、流体表面に流れが生じることになる。これが、マランゴニ対流と呼ばれる流れである。表面張力と粘性との関係は、二つの無次元数を用いて表現され、マランゴニ数(,)と呼ばれている。は表面張力と動粘性係数に熱拡散係数を乗じたものの比であり、は表面張力と粘性力の比を表す。マランゴニ対流は≧(臨界マランゴニ数)の条件が満たされたときに開始する。c. 毛細管現象を利用した液体移動液体のぬれ性を利用し、液体の移送、位置制御を行うことと関連して、細管内に流入する気泡の挙動が調べられている。流量が増大し、気泡界面が不安定となると細管内に気泡が流入するようになる。特に、ロケットなどの液体燃料に気泡が混入してはならないので、細管内に気泡が流入する条件を求めることは重要である。
