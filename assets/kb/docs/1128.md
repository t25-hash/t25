# 18・3・1 風洞実験

> **章**: 第18章 流体実験および計測
> **節**: 18・3 流体実験設備

風洞は流速が低く,流れの特性が圧縮性の影響を受けない領域で使用される低速風洞(low speed wind tunnel)と,マッハ数が0.7以上の領域で使用される高速風洞(high speed wind tunnel)とに大別される.a. 低速風洞低速風洞を形式により分類すると図18・1に示すように回流式と開放式とになる.さらに,開放式は送風機が測定部の下流にある吸込み式と,送風機が測定部の上流にある吹出し式とに分けられる.回流式のものは流路が密閉されているために外界の影響を受けることが少なく,かつ流れが循環しているためにエネルギー損失も少ないという特徴がある.このため,回流式風洞は大型の風洞として用いられることが多い.流路の曲り部では遠心力のために内側の壁に沿う流れにはく離が発生する.これを防ぐために円弧状の薄板や翼からなる変流板を挿入し,流れの向きを滑らかに変える.変流板は,その長さと間隔の比を0.3~0.5程度に配置するとよい.開放式のものは構造が簡単で設置面積も少なくて済むが,エネルギー損失が大きい.また,開放式のものでは吹出し口や吸込み口における外気の影響が,測定部に及ばないように工夫をする必要がある.さらに,吸込み式のものでは測定部が大気圧以下になるので,測定部を密閉式にしなければならない.開放式のものは小型の風洞に用いられることが多い.低速風洞として最も多く使用されているのは吹出し式の風洞であり,基本的には送風機,拡散胴,整流胴,縮流ノズル,および測定(a) 回流式(b) 開放式(吸込み式)(c) 開放式(吹出し式)部より構成される.低速風洞で使用される送風機は通常単段の軸流送風機が用いられる.ねじれや脈動の少ない流れを得るためには,動翼の上流あるいは下流に静翼を設ける.送風機やモータの発生する振動や騒音を低減するためには,送風機やモータを風洞本体と別の土台に設置したり,下流の胴部とはゴムや布などで接続して振動の伝ぱを遮断することが有効である.送風機を出た流れは拡散胴によって減速されるが,流れのはく離を防止するために広がり角としては10°程度が推奨される.しかし,設置面積の制限から広がり角を大きくする場合には,途中に網を3~5枚挿入し,流れのはく離を防止する.整流胴では流れの不均一を取り除き,乱れを減衰させるために大きな空間が必要である.整流胴内には20メッシュ程度(1インチ当たりの目の数が20)の整流網が複数枚挿入される.流れにねじれが残っているときにはハニカムも併用される.しかし,ハニカムの厚さや網の線の直径が大きいと,その後流が流れを乱すことになるので,そのレイノルズ数が十分小さくなるように寸法を選ぶ必要がある.整流胴を出た流れは縮流ノズルにて測定部の断面積まで絞られる.測定部における速度分布を一様にし,乱れを小さくするためには絞り比(整流胴と測定部との面積比)とノズルの形状が重要となる.絞り比はできるだけ大きくする(5~20程度)ことが望ましい.縮流ノズルの形状を決定するために多くの方法が提案されているが,単純なものとしては図18・2に示す余弦曲線があり,比較的良い結果が得られる.b. 高速風洞高速風洞はマッハ数により次の3種類に区別される.(1) 遷音速風洞(マッハ数0.7~1.5)(2) 超音速風洞(マッハ数1.5~5.0)(3) 極超音速風洞(マッハ数5.0以上)遷音速風洞では測定部での気流の閉塞現象を避けるために多孔壁または多溝壁を使用する必要があり,強制的に抽気を行うこともある.極超音速風洞では大きな圧力比が要求されるとともに,空気の凝結を防ぐための加熱装置が必要となる.超音速風洞では壁からの抽気や風洞気流の加熱は必要としない.また,高速風洞は,運転方式により間欠運転式と連続運転式の二つに大別される.間欠運転式の構成例を図18・3に示す.圧縮機によって圧縮された空気を貯気槽にため,これを整流胴,ノズル,測定部,拡散胴を通して大気に放出する.大きなマッハ数が比較的容易に得られるが,貯気槽の容量制限のため,運転時間は10~100s程度に制限される.連続運転式の構成例を図18・4に示す.連続運転式は回流式の風洞となり,大型の装置となる.圧縮機としては多段軸流あるいはターボ圧縮機が用いられる.圧縮機から出た空気は冷却器で所定の温度まで冷却される.冷却器を出ると整流胴に入り,ここで気流の脈動などが除かれる.その後,ノズルで超音速まで加速され,測定部,拡散胴を経て,再び圧縮機に導かれる.連続運転式は長時間の連続運転が可能なため,測定に長い時間が必要となる実験に適し図18・2
