# 18・3・2 水槽実験

> **章**: 第18章 流体実験および計測
> **節**: 18・3 流体実験設備

風洞が気流を用いる実験装置であるのに対し、水流を用いる装置を水槽実験装置という。水槽には流体は静止させて模型を移動させるものと、風洞と同様に水の流れを作るものとがある。前者の方式のものはえい行水槽(towing tank)あるいは静止水槽と呼ばれ、後者は流水槽(running water channel)と呼ばれる。a. えい行水槽えい行水槽の構成例を図18・5に示す。模型はレールの上を走る台車に取付けられ、静水中を移動する。流れの速さは台車の駆動装置によって調整される。このえい行水槽では、流れの中の乱れを零に近い状態にすることができるため、遷移の実験や低レイノルズ数の実験、流れの可視化実験などに適する。しかし、測定時間が短くかつ次の実験を行うまで流れが完全に静止するのを待たねばならない。水槽の長さは模型の速度、測定時間、始動・停止に要する距離によって決まるが、実験室用としては3~4m程度で使用されることが多い。幅と深さの決定には二次元性の確保、壁の干渉などを考慮する必要がある。台車の移動速度は表面張力波の発生限界を考慮して20cm/s以下に抑えるか、台車に波抑え用の板を取付けることが必要である。b. 流水槽流水槽には測定部が密閉され水が充満して流れる閉水路型と、測定部の上面が開かれ自由表面をもつ開水路型とがある。閉水路型のものはおもにキャビテーションの研究や伝熱の研究などに多く用いられる。また、流水槽にも風洞の場合と同様に回流式と放流式とがある。回流式のものはポンプで水を駆動し、水流が連続的に循環する形式で、放流式は貯水槽に蓄えられた水を水路に放水する形式である。図18・6は開水路型回流水槽の構成例を示している。貯水槽はポンプから送られてきた水を受ける部分であり、貯水槽の下流には水流の乱れの均一化を図るために有孔板が設置される。整流部は風洞と同様な形式をとり、内部にハニカムや整流網が設けられる。整流部を出た流れは、測定部に入る前に縮流ノズルによって加速される。絞り比は風洞の場合ほど大きくとる必要はな図18・4 高速風洞(連続運転式)図18・5 えい行水槽図18・6 開水路型回流水槽4―19 7く、2から3程度が多いが、ノズルを省略して直接測定部に導くこともある。流れを可視化するために測定部の側壁や底面の一部を透明壁にすることが多く、ガラスやアクリル板が用いられる。前者は板厚を十分にとり破損に注意することが必要で、後者は平面の不均一性や汚れの付着による透明度の低下に注意する必要がある。水位の調整は、水路の下流端に取付けられた高さの調整が可能なせき、ならびに弁の開度とによって行う。水槽実験を行うに際しては、気泡の除去、水質管理、温度管理、水圧に対する強度対策、腐食対策、漏水対策、ポンプなどの振動対策など注意しなければならない点が多く存在するので、専門書を参考にすることが望ましい。
