# 18・5・1 ピトー管

> **章**: 第18章 流体実験および計測
> **節**: 18・5 速度計測

流れの中に置かれた先端が球形な物体の前面において、流れが静止するよどみ点における圧力を計測する。エネルギー損失が無視できれば、ピトー管によって全圧(総圧)を計測することができる。静圧を求めるものを静圧管(static pressure tube)という。ここで、はピトー管係数で1に近いが、静圧が必ずしも真の値にならないので、一般に1.0にならない。使用する流速範囲に対してあらかじめ校正を行ってピトー管係数を求めておく。図18・18に示すJIS 型標準ピトー管は、広い流速範囲に対して、ピトー管係数が1.0かそれに極めて近い。静圧孔は頭部から6、支柱から8のところにあり(:管の外径)、静圧の計測に及ぼす頭部と支柱の影響が互いに打消し合う位置である。その他、プラントル型ピトー管などもある。ピトー管が流れに対して傾いているとき、測定圧力は全圧よりも低い値を示すが、実際には、圧力計の読みが最大になるように傾きを調節して流れの方向に一致させることができる。境界層や後流などのせん断流中においては速度こう配の影響、また固体壁面までおよそ2以下の近いところでは壁面の影響を受け、ピトー管は速度こう配の方向にずれた点(速度の増加する方向)での全圧を読み取ることになる。速度の大きさと方向が局所的に変化している場合は図18・19に示す3管型ピトー管などのヨーメータ(yawmeter)を用いる。これは、ピトー管よどみ点圧は流れに対して傾いていると総圧より低い値を示すことを利用したもので、あらかじめ傾き角とよどみ点圧の関係を校正することによって、3管型や5管型、さらに5孔球形ピトー管によって、方向と全圧を計測する。静圧は別に流れの方向に一致するようにした静圧管で計測する。流れが高速の場合には圧縮性の影響が現れる。マッハ数0.7以下の亜音速流れであれば図18・18 JIS 型ピトー管4―200流体工学図18・193管型ピトー管非圧縮性流れ用のピトー静圧管を用いてもよい。遷音速流れで局所的に音速を超える流れが形成され、等エントロピーの仮定が成り立たなくなるときは、鋭く尖った円すい状の管に圧力孔を設け静圧を計測する。超音速では、低速の静圧管の頭部を円すい状にしたものを用いる。全圧は先端をほぼ直角に切り落としたピトー管で測り、静圧と別々の計器を用いることが望ましい。よどみ点温度は熱電対を組込んだプローブ[ウィンクラー(Winkler)プローブなど]によって計測する。
