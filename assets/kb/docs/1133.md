# 18・5・2 熱線流速計

> **章**: 第18章 流体実験および計測
> **節**: 18・5 速度計測

金属細線に電流を流し、加熱細線からの強制対流熱伝達と気流速度との関係式を用いて流速を測ろうとする流速計を熱線流速計(hot-wire velocimeter)という。理想化された無限の長さをもつ細線においては、熱伝導損失が無視され、電流による発熱量と流れによる放熱量が釣合う。ここで、,は流体に対して実験によって与えられる校正定数である。空気、水、油を使った実験により、広範囲なレイノルズ数とプラントル数について以下の結果が与えられている。定温度型熱線流速計は図18・20(a)のように速度変動による熱線の温度変化非平衡電圧-を補償増幅器を通して負帰還し、熱線抵抗を一定に保つ。また、図18・20(b)に示すような定電流方式もあるが、流速を求めるには定温度方式と比べて周波数特性が劣る。しかし、低い加熱比を用いることによって温度測定として用いることができる。熱線プローブの形状の例を図18・21に示すが、速度の一成分、二成分、さらに三成分を計測する場合に応じてプローブを選択する(例えば、二成分計測のときはX 型を選択する)。乱流の平均特性を取扱う場合は、傾斜一次元プローブを回転させることによって用いることもできる。本項後述の特別な熱膜プローブを用いること以外には一般に逆流の識別が難しい。熱線流速計は計測前後に校正を行うようにする。気流温度のゆっくりした変化を補償する熱線流速計も多い。高速流体における熱線プローブは、プロング(prong)によって発生する衝撃波の影響をできるだけ小さくするようにしている。一般に振動対策としてプロングは短く、プロング先端からの衝撃波の影響を受けにくいようにプロング近くはコーティングされた熱線を使用することもたいせつである。熱膜プローブはくさび状や円すい状の石英ガラス(quartzglass)の表面にニッケルや白金を蒸着させたものを用い、水や油などの液体の流れ、さらに高速気流の計測に用いられる。MEMS 技術の発達によって壁面せん断応力を計測するための小型熱線プローブが開発されている。渦度を計測するプローブや、同時に多点計測を行う熊手型などの微小熱線(rake)が開発されている。一成分熱線プローブではピッチ角βの変化に対して、プロング幅が小さいほど影響が大きいが、直径5 m、有効長さ1mmのタングステン熱線で最大15%程度である。直線化回路を用いれば流速に対応する出力電圧を直線的にすることができる。
