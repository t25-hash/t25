# 18・5・4 PTV 法(粒子追跡法)

> **章**: 第18章 流体実験および計測
> **節**: 18・5 速度計測

Particle-tracking velocimetryの頭文字をとってPTV と呼ぶ。4―201この方法は粒子の軌跡をマニュアルで追跡したのが始まりで、粒子の移動や粒子配列パターンから流速を求める方法がある。流速計システムは粒子群の可視化を行う可視化光学系と可視化画像を解析する画像解析システムから構成される。トレーサ粒子の分布パターンの類似性から流速を計測する粒子画像流速計測法(PIV : particle image velocimetry)における粒子像の密度が十分小さい場合に対応する。画像解析システムにとって良好な可視化画像を得ることが重要であり、また、図18・23に示すように、高出ダブルパルスYAG レーザによりストロボ的にレーザ光を照明に使うなど、レーザパルス周波数と高解像度CCD カメラのフレームレートによって時間分解能を向上させる方法がある。粒子の散乱光強度が観察方向に対して十分なものが望まれる。粒子径が20m程度の微小粒子を用いる。粒子画像の取得後は、(1)粒子像の認識、(2)粒子の番号付け、(3)追跡、(4)速度ベクトルの算出を行う。短い時間間隔で取得された可視化画像において粒子像の移動を検出する。四時刻追跡法や粒子分布相関法などによって粒子像の移動先をさがす。四時刻追跡法とは、4枚の画像を用いて、粒子の位置が最も直線に近い組合せが正しい移動経路である確率が高いという考え方に基づいている。一方、流れの中で粒子群は隣り合う粒子は類似な動きをするので、粒子群のパターンは大きく変わらないことを利用した方法が図18・24に示す分布相関法(二値化相関法)や粒子群の相対的位置変化を考慮したばねモデル法と呼ばれる方法がある。また、輝度差累積法に代表されるような可視化画像の濃淡パターンの相互相関係数から画像の移動先を求め、流速ベクトルを決める相関法と呼ばれる方法がある。解析対象が可視化画像の輝度分布であればなんでもよく、煙や染料を流した可視化画像でも解析が可能である。種々の解析手法に対して、標準的な評価基準である評価法が提案されている。その他、画像ノイズ除去後にフィルタリング処理など、画像改善法が提案されている。一方、三次元流れ場の計測は、ステレオ写真法を応用した3D-PTVがある。粒子密度が低いために、一般に速度ベクトルは少なく空間解像度が低くならざるを得ない。さらに、ホログラムを使って三次元粒子像を取得して三次元速度場計測に応用するホログラフィックPIV法も開発されている。微細な粒子をホログラムに高密度で記録する。PIV とレーザ誘起蛍光法を用いることによって、流速と温度や密度の同時計測を行うことができる。流体中の蛍光物質の濃度を一様に保てば、蛍光強度は温度の関数になり、流体の温度を一図18・23 PTV図18・22 レーザ流速計図18・24 粒子分布相関法の計測原理4―202流体工学定に保てば蛍光強度は濃度の関数になる。速度計測のために粒子と蛍光物質を流れ場に混入し、これらを光学フィルタにより分別して2種類の画像として取得する。流量計測は、産業計測の基幹技術の一つで、気体や液体などの測定対象、航空機エンジンの特性試験、プラント、環境計測、医療などの応用分野、圧力や流量範囲などの測定条件など、これらに適した多様の種類・原理の流量計が開発され使用されている。表18・1、分野をまとめてある。ここで記述されていない流量計については文献(44)を参照されたい。流量は、複合量であり直接それ自体を測定することはできない。言い換えれば、流量計とは、流れの情報を圧力測定などの間接的な手段で検出し、流量に換算する計測器である。このことは、流量計の測定精度が間接的な測定手段の精度によって決まることを意味する。ゆえに、必ず上位の精度を有する装置で測定された流量と計測器の読みとの関係を決定する校正という作業が必要であり、この校正の精度を保証するためには、計測器が実際に使用されるのと同じ条件で行われることが重要である。
