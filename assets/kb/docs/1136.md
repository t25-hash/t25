# 18・6・1 差圧式流量計(differential pressure flow meter)

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

差圧式流量計は、大まかに絞り流量計と層流流量計に分類される。絞り流量計は、オリフィス、ベンチュリ、ノズルなどに分類され、ベルヌーイの式を基本にしていて、管路途中の絞り部の上流・下流の圧力差から流量を算出する流量計である。絞り流量計を使用する場合、その形状とレイノルズ数に応じた補正係数(流出係数)が必要である。オリフィスは絞り流量計の中で最も多く使用されていて、その流出係数はJIS で規定された設置条件や形状を満足していれば、JIS で与えられた値を用いて高い精度の計測が可能である。オリフィスは、プラントなどにおける大流量計測において主要な流量計であり、さまざまな設置条件や流れ状態などがもたら流量範囲空気流量使用流量計利用分野2MPa以上10000m/h以上UV O,SV航空機エンジン等の性能評価UV,T天然ガスパイプライン天然ガス発電所大気圧~1MPa10000m/h以上ピトー管式、O,SV圧縮送風機航空機空力評価UV,T,SV天然ガスパイプライン地域冷暖房大気圧~0.5MPa1000m/h以上渦式、O,SV蒸気プロセス計測制御UV,T,SV3MW 以上の燃料電池大工場への天然ガス大気圧~0.3MPa0.002~5m/h流量範囲石油・化学製品の流量使用流量計利用分野1000m/h以上UV,T石油輸入・備蓄基地オープンチャネル河川、用水路の流量10m/h以上容積式、コリオリ式燃焼バーナ電磁式、T病院・ホテルへの給水0.1m/h以上熱式半導体製造潤滑油供給熱式、P薬品工業0.001m/h以上マイクロセンサマイクロTASマイクロセンサ医療機器UV:超音波流量計、T:タービンメータ、P:ピストンプルーバす誤差要因、その補正方法などが研究されている。ハーゲン・ポアズイユの式を利用して細管の上流・下流の圧力差から流量を算出する層流流量計は、小流量用に広く使用されている。また、細管でなくて環状のすきまを使用し、二つの高精度絶対圧力計と温度計を本体に内蔵した質量流量を計測できる層流流量計が開発された。この流量計は、外部の温度変動に鈍感なように熱容量の大きい本体をもち、このタイプの流量計の弱点を逃れている。精度の高い校正をすれば、0.1%程度の精度と同程度の再現性をもつ。
