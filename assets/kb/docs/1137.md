# 18・6・2 熱式流量計(thermal type flow meter)

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

さまざまな方式が提案されているが、基本的にはセンサが流れに直接接しているタイプと管路の外部にコイルを巻いて加熱するタイプとがある。内部にセンサが置かれたタイプのものは、応答性は速いが測定対象流体を選ぶ。また、汚れなどによる感度劣化が問題になる。半導体産業などで使用されている熱式質量流量計は、センサ管路の外側にコイルを巻いたタイプである。この流量計は、センサ管路とバイパス部とから構成されている。センサ管路には内径0.2~0.3mmの細管のまわりに加熱と温度計測を兼ねた一対のコイルが巻かれており、流体が流れることによって生じる温度分布の対称性の崩れからセンサ部を流れる流量を求める。バイパス部にはセンサ部の流量に比例する形で毛細管等の抵抗体が挿入され、全体として必要な流量が得られるようになっている。流量욇は、センサ間の温度差から次の式を用いて算出される。욇=욣ここで、はセンサ部分とバイパス部分との分流比率など流量計に依存した補正係数で、校正によって決定される。気体の定圧比熱욣を含むために、計測される質量流量욇は気体によって異なる。そのため、窒素の計測結果を基に任意の気体の流量を推定する変換係数(CF)が提案されているが、測定精度を必要とする場合は、測定対象ガスによる直接校正が不可欠である。測定精度は、校正方法にもよるが2%程度、再現性は±0.5%である。表18・1 測定対象および流量と使用される流量計の種類およびその利用分野(気体)天然ガス・その他のガス流量使用流量計利用分野大気圧~0.5MPa5m/h以上面積式、SVプロセス計測制御自動車空燃費制御容積式、GM,SV10kW 以上の燃料電池病院、ホテルへの都市ガス熱式、GM,SV大気捕集、化学薬品熱式、GM,SVセル単位の燃料電池家庭への都市ガス大気圧~0.3MPa0.0001~0.002m/hマイクロノズルリーク計測マイクロノズル環境分析、半導体製造技術SV:音速ノズル、O:オリフィス、T:タービンメータ、GM:膜式ガスメータ、P:ピストンプルーバ、UV:超音波流量計表18・2 測定対象および流量と使用される流量計の種類およびその利用分野(液体)水・排水の流量使用流量計利用分野100m/h以上容積式、T石油化学コンビナート電磁式、差圧式上水・下水、揚水ポンプ1m/h以上膜式、P自動車燃費、家庭用燃料油翼車式、面積式家庭用水道水、流体機械
