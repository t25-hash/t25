# 18・6・3 渦式流量計(Ka

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

썝rma썝n-vortex type flow meter)渦式流量計は、管路内に置かれた渦発生体から生じるカルマン渦のストローハル数を利用するため、レイノルズ数が同じならば流体の種類によらないという利点をもち、一般的にはストローハル数がほぼ一定になる=2×10以上の範囲で使用される。ストローハル数の定義式から得られた平均流速に管路の断面積を掛けることによって、そのときの体積流量が計算できる。広い流量範囲でストローハル数を一定にするため、外部から強制的に擾乱を与えて渦の安定を図るなど、多くの研究がある。渦発生体である柱状物体と管路壁の結合部分のすきまが、柱状物体と管壁近傍に発生する馬てい形渦の挙動に影響し、カルマン渦の安定性に大きく影響する。この実験結果を基に規格化された標準渦発生体をもつ標準渦式流量計がJIS で規定されている。この標準渦式流量計の校正係数の不確かさは、レイノルズ数が10~2.5×10の範囲で±0.2%である。標準渦式流量計の校正係数は管路直径が決まれば計算から求めることができ、実流校正が不可能な大流量の流れや水素などの危険な気体の計測などへの適用が期待される。
