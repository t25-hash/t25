# 18・6・4 超音波流量計(ultrasonic flow meter)

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

流れによる超音波伝搬の変化量の計測から平均流速を求めるというのが超音波流量計の基本的な原理である。その変化量の検出方法として時間差方式、位相差方式、シングアラウンド法(singaround method)、ドップラー法などさまざまな方式が提案されている。一般的な方法では、二つの超音波センサを管路を挟んで斜めに配置し、超音波パルスを同時に打ち、到達時間差から超音波のパス上の平均流速を求め流量を算出する。この方法は、最終計算に気体の音速を必要としないという利点をもつ。この流量計は、非接触、長期安定性、広い測定レンジ(1:1000)、高精度(±1%程度)などの特性から、環境計測用の微小流用流量計、天然ガス計測用の大流量用流量計、混相流用の流量計などその応用範囲は広い。そのため、微小な時間差(ナノ秒のオーダ)計測、ノイズの改善、また、上流側流速分布に影響されないなどの研究が進められている。流速分布対策として開発されたマルチパス方式の超音波流量計でも、上流側擾乱から10離すと二重曲りでは、0.5%、旋回流では2.5%不確かさが増大する。
