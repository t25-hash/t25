# 18・6・5 音速ノズル(critical nozzle)

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

気体の流れが管路に設けられた絞り部で臨界になると、この管を流れる流量は下流側の流れ場に影響されずに一定になり、そのときの流量は上流側条件(圧力、温度)にのみ依存する。この原理を利用したのが音速ノズル流量計である。ISOではトロイダルスロート(toroidal throat)とシリンドリカルスロート(cylindricalthroat)の2種類のノズル形状が規定されている。ここで、:スロート断面積、,:それぞれ上流側よどみ点圧力と温度である。は臨界係数と呼ばれ、理想気体の場合は比熱比だけの関数である。境界層が層流で実在気体効果が大きくない場合には、理論的に取扱われるノズル形状と正確に同じ形状をもつ超精密加工音速ノズルを使用することによって、理論的に流出係数を±0.05%以下で決定できる。レイノルズ数が大きくなり、境界層が遷移状態や乱流になる場合の理論的取扱いはできていない。気体温度の流出係数への影響、ガス種による流出係数の変化などの研究も行われている。また、0.1mm以下のスロート直径をもつ微小音速ノズルに4―203よって0.01g/min以下の気体の流量計測も可能になっている。
