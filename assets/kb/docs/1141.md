# 18・6・6 コリオリ式流量計(Coriolis flow meter)

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

コリオリ式流量計はコリオリ加速度を利用する流量計である。2本のU 字形のチューブをその軸に垂直方向に相反する方向に振動させる。このU 字形チューブに流体を流すと、コリオリ力のために2本のチューブの振動に位相差が生じる。この位相差からそのときの質量流量を求める。位相差の代わりに2本のチューブの基準点を通過するときの時間差を,U 字管チューブの幅を、チューブに加えられる強制振動数をω、流体の流れによって生じるチューブのねじれ振動数をω욦、チューブのばね定数を욦とすると質量流量욇は、욇=욦1-ωω욦2で与えられる。は校正によって決定される補正係数である。この式は、流体の熱物性量を含まないので測定対象流体によらないが、温度によって変化するばね定数を含んでいるので温度補正は必要となる。コリオリ式流量計は、外乱振動がなく、急激な圧力や温度変化などがない条件下では0.1%程度の測定精度ももつ。文献(61)には、コリオリ式流量計のさまざまな応用分野の研究が、文献(62)は最新のコリオリ式流量計についてそれぞれまとめてある。この流量計で気体のように密度が小さい流体を測定する場合、検出すべき位相差が小さくなり、液体の場合と比べて測定精度は5倍以上悪くなる。密度を大きくするために気体を10MPa以上に加圧するとチューブが変形する、変形の起こらないようにチューブの剛性を高くすると感度が落ちるなどの理由で、気体の場合の高精度計測は難しい。
