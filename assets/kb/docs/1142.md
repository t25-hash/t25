# 18・6・7 その他の流量計

> **章**: 第18章 流体実験および計測
> **節**: 18・2 に測定対象と使用される流量計のおもな種類およびその利用

タービンメータ、歯車式、ロータリ式、膜式ガスメータ、湿式ガスメータなどの測定原理は、いずれも内部の回転子が回転、もしくは往復運動をして気体を吐出し、その回転数や振動数からそのときの流量を求めるものである。この種の流量計は、測定対象流体の性質に強く影響される。また、回転子と内部壁のすきまからの漏れが高精度測定においては問題となる。さらに、旋回流など上流側の流れの状態に強く影響される。水など電導性の良い液体の計測には電磁流量計が使用される。しかし、上流側の流速分布の影響を強く受けるために、複数の電極対を配置しそれに対応するための研究がされている。配管の外部に電極を配置して流量を計測できる非接触型の流量計の特性を生かして血管内部の血流量計測などにも使用されている。
