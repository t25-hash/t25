# 18・7・1 音圧、音の強さ、音響パワー

> **章**: 第18章 流体実験および計測
> **節**: 18・7 騒音計測

音圧は媒質(空気中であれば空気)における微小圧力変動の実効値によって表される。この大きさ(強さ)を表すのに、以下に示すようないくつかの表現がある。対数表示された相対的な音の大きさを表す単位にはdB を用い、これをデシベルと呼ぶ。音圧レベル(sound pressure level)は式(18・21)のように定義される。=10log=20logdB(18・21)ここに、は1kHz の周波数の正弦波振動する音の音圧2×10Paを表し、一般の人がこの周波数の音を知覚できる最小音圧の平均として定められたものである。物理的な音の強さを表現するために、平面音波における関係を基に音の強さ(sound intensity)を式(18・22)のように定義する。=1d=ρWm(18・22)これを用いた音の強さのレベル(sound intensity level)は=10logdB(18・23)である。ここに、:媒質の流体粒子速度、ρ:密度、:音速を4―204流体工学表し、:標準状態における空気に対応する音の強さである。しかし、式(18・22)からわかるように、空気の状態によって密度および音速が異なるため、それに近い値である10W/mを基準値として用いる。したがって、この若干の差を無視すれば、音の強さのレベルと音圧レベルとは一致する。音源の強さを表すための音源パワーレベル(sound power level)は次のように定義される。=10logdB(18・24)ここで、=10Wである。いま、音源から毎秒Wのエネルギーがあらゆる方向に放射されていると仮定すれば、音源からm離れた球面上において、単位面積、単位時間当たりに音源から遠ざかる方向へ流れるエネルギーW/mは、=4πW/m(18・25)である。音の強さのレベルは==-20log-11(18・26)と表される。また、地上にある無指向性の音源から半球状に放射するものでは、==-20log-8(18・27)と表される。音圧レベルがとである二つの音が同時に存在したときのレベルは、=10log10+10(18・28)と求められる。この関係を用いて、測定対象音に対する暗騒音の影響を補正できる。
