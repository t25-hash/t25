# 18・7・1 音圧、音の強さ、音響パワー

> **章**: 第18章 流体実験および計測
> **節**: 18・7 騒音計測

音圧は媒質(空気中であれば空気)における微小圧力変動の実効値によって表される。この大きさ(強さ)を表すのに、以下に示すようないくつかの表現がある。対数表示された相対的な音の大きさを表す単位にはdB を用い、これをデシベルと呼ぶ。音圧レベル(sound pressure level)は式(18・21)のように定義される。物理的な音の強さを表現するために、平面音波における関係を基に音の強さ(sound intensity)を式(18・22)のように定義する。ここに、:媒質の流体粒子速度、ρ:密度、:音速を4―204流体工学表し、:標準状態における空気に対応する音の強さである。しかし、式(18・22)からわかるように、空気の状態によって密度および音速が異なるため、それに近い値である10W/mを基準値として用いる。したがって、この若干の差を無視すれば、音の強さのレベルと音圧レベルとは一致する。音源の強さを表すための音源パワーレベル(sound power level)は次のように定義される。この関係を用いて、測定対象音に対する暗騒音の影響を補正できる。
