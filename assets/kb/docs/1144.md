# 18・7・2 音響インテンシティ(acoustic intensity)

> **章**: 第18章 流体実験および計測
> **節**: 18・7 騒音計測

ここに、は時間平均を表す。原理的には近接して設置された2個のマイクロホンを用いて音圧を計測して音響インテンシティを得る。計測システムを図18・25に示す。2チャネル入力をもつFFT アナライザを使い相互相関を求め、その虚数部が音響インテンシティに相当することを利用する。普通、処理時間短縮のために、オクターブ、1/3オクターブ、1/12オクターブのディジタルフィルタを用いた音響インテンシティ計測システムとなっている。計測誤差に影響するのはマイクロホン間の距離である。小さな誤差の範囲内で計測可能な音の最大周波数はが小さいほど高くなる。
