# 18・7・3 マイクロホン,騒音計

> **章**: 第18章 流体実験および計測
> **節**: 18・7 騒音計測

a. マイクロホンコンデンサマイクロホン(condensermicrophone)では,振動板と固定電極(背板)との間に蓄えられる静電容量変化を電圧変化として出力するものである.一般的に1インチ,1/2インチ,1/4インチの直径のものが用いられる.周波数特性が平坦になるよう設計されたものを音圧型マイクロホンという.このタイプは,周波数帯で20~5000Hz を,ダイナミックレンジで35~140dB の範囲をカバーできる.これに対して,マイクロホンの設置により反射,回折のために音場を乱す影響をあらかじ図18・25 音響インテンシティ計測のためのブロック図め考慮した音場型マイクロホンがある.計測できる高周波数帯を100000Hz 程度まで広げることができ,使用環境温度範囲を-50~150°Cの範囲で選ぶことができる.エレクトレットコンデンサマイクロホン(electret condensermicrophone)は,電荷を蓄えた高分子フィルムを固定電極に張り付けたエレクトレットを用いたものである.直流高圧電源が不要である.エレクトレットフィルムには負の電荷が蓄えられており,負の直流電圧を印可したコンデンサマイクロホンと等価と考えることができる.したがって,音圧信号と電気信号は同位相となる.このマイクロホンは,構造が簡単であること,丈夫であること,湿度による影響を受けにくい,などの特徴をもっている.しかし,使用温度は50°Cぐらいまでである.ピエゾマイクロホン(piezoelectric microphone)は人間の耳と類似した簡単な構造をもっている.振動板の変位がそれに接触しているバイモルフ(bimorph)によって電気信号に変換される.バイモルフにロッセル塩(Rochelle salt)のような結晶が用いられる.安価で周波数応答の良いものができる反面,使用温度が45°C以下と低く,また湿度の影響を受けやすい欠点がある.変換素子にジルコン酸鉛などのセラミックスを用いた場合,ロッセル塩のものより感度は低いが,湿度の影響を受けにくいものができる.また,使用温度も100°Cまでのものがある.ムービングコイルマイクロホン(moving coil microphone)はスピーカと逆の原理を用いている.したがって,原理的に機械振動であるので,音波面に対して直角方向の受感特性が最も良い.他のマイクロホンに比べ,温度・湿度の影響を受けにくく,出力インピーダンスが小さいために長いケーブルを用いることができる.野外の風に対してはナイロン布やポリウレタンでできたウインドスクリーン(wind screen)をかぶせ,風による雑音を下げる.また,ダクト内の気流に対しては,側面の一部がメッシュ状になった金属キャップであるノーズコーン(nose cone)を受感部に装着し,気流方向とマイクロホンの軸方向を一致させて用いる.マイクロホンを校正するには,無響室内において標準マイクロホンと校正したいものとを10~20cm離して並べて置く.スピーカからマイクロホンまでの距離は1m程度にする.スピーカから種々の周波数の音を発生させ,周波数ごとに二つのマイクロホンの出力を測定し,両者を比較することで被検査マイクロホンの特性を知る.b. 騒音計騒音計(sound level meter)には,精密騒音計,普通騒音計および簡易騒音計がある.基本的には,これらはマイクロホンから受取った信号の前置増幅器,周波数補正回路,増幅器,出力指示器および外部出力部からなる.周波数補正回路の特性にはA 特性,C特性および平坦(flat)特性がある.後述の図18・26に示す聴覚曲線(equal loudness contour)を上下逆さにしたような特性曲線をもつ補正回路の特性がA 特性である.この場合,低周波数の音ほど大きな減音を施して出力させる.C特性にはより大きいラウドネス曲線を反転させたようなほぼ平坦な特性を用いている.したがって,A 特性ほど減音量は大きくない.平坦特性はすべての周波数に対して減音を施さないで出力させるものである.なお,出力値の単位としてA 特性を用いた場合にはdB(A),C特性図18・26 聴覚曲線(等聴感曲線)の場合にはdB(C)および平坦特性の場合には単にdB と表示する.
