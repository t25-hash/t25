# 18・7・4 周波数分析

> **章**: 第18章 流体実験および計測
> **節**: 18・7 騒音計測

騒音計を用いた音圧レベルの計測はすべての周波数帯における音圧レベルを評価するものである。これに対し、ある周波数ごとに区分した音圧レベルを評価しなければ騒音の特徴を表現できない場合が多い。人間の感覚では、周波数に対してもほぼ対数的なとらえ方をするため、オクターブ間隔に周端数を区切って音圧レベルを表示する。このときの音圧レベルをオクターブレベルという。周波数ととの間に=2の関係があるとき、からの間をオクターブバンド(octave band)といい、とをカットオフ周波数(cut-offfrequency)という。また、式(18・30)のように定義される周波数をオクターブバンドの中心周波数(center frequency)と呼ぶ。욂=(18・30)1オクターブバンドでは、バンド幅(1octave bandwidth)は次のように表される。=-=2욗-2욗욂(18・31)オクターブ幅ごとにオクターブレベルを示したものを、オクターブバンドスペクトルという。
