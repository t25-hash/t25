# 18・7・5 感覚特性

> **章**: 第18章 流体実験および計測
> **節**: 18・7 騒音計測

音の評価は人間の感性に依存するので,一概に物理的な音の大きさだけではその音がうるさいか否かは決められない.同じ強さ(エネルギー)の音でも,周波数が異なると強さが違うものと感じる.それをグラフに表したのが図18・26の聴覚曲線(等聴感曲線)である.単位phonで表された音の大きさは対数尺度なので,例えば80phonは感覚上40phonの音の2倍の大きさにはならない.中心周波数1kHz のオクターブバンドレベル40dB の音を基準として,感覚的に2倍になる音圧レベルはオクターブバンドレベルが10dB増加するときである.逆に,感覚的に半分の強さとなったと感じるのは,オクターブバンドレベルが10dB 減少したときである.この関係をわかりやすくするために,中心周波数1kHz のオクターブバンドレベル40dB の音を1soneと定義する.したがって,50dB の音の大きさは2倍になるので,2soneであり,30dB の音は半分なので,0.5soneである.これを式で表すと,次のようになる.=2(18・32)ここに,はうるささ(loudness)を表し,単位はsoneである.また,はうるささのレベル(loudness level)であり,単位はphonである.4―205目に見えない流れを可視化(flow visualization)することで,流体の運動を定性的もしくは定量的にとらえることができるようになる.近年は,画像解析を応用した定量可視化(quantitativevisualization)が計測方法としても定着している.
