# 18・8・2 気流における可視化

> **章**: 第18章 流体実験および計測
> **節**: 18・8 流れの可視化

a. タフト法水流の場合と同様に,流れの方向を知ることができる.気体の密度が小さいため,タフト材質もなるべく軽いものを用いるなど,重力の影響を考慮する必要がある.b. トレーサ法. 注入トレーサ法気流の場合,一般にトレーサと気体との密度差が大きくなる傾向にある.流体への追従性を考慮するため,微小でかつ光の散乱が強い粒子を用いることが多い.また,実験装置も大きくなることが多いので,トレーサによる装置の汚染や人体への影響も考慮する必要がある.例えば,シリカ粒子(SiO)やオイルミスト,水蒸気,煙などが用いられる.後述するPIV などでは,直径1 m程度のオリーブ油滴やDEHS(di-2-ethyhexyl-sebacat)滴がよく用いられている.. 化学反応トレーサ法水流の場合と同様である.薬品としては,四塩化チタンと湿り空気などが用いられるが,人体などへの影響も十分考慮する必要がある.. 電気制御トレーサ法細線に油を塗布し,通電することで白煙を生じさせる(スモークワイヤ法).水素気泡法と同様に電気制御が可能であるとともに装置が簡便である.火花追跡法は,気中に置かれた電極間に高周波でパルス状高電圧を掛けると,放電により電極間にイオン化された経路が形成され1msほど保持される.この経路は流れとともに移動するが,次のパルスによって経路に電流が流れることで発光し,その火花列がタイムラインとして可視化される.c. 壁面トレース法水流と同様である.表面張力などを考慮し,比較的低粘度の油を用いる.d. 光学的可視化法. シャドーグラフ法(shadowgraph method)流体に密度や温度分布があると屈折率が変わり,空間に屈折率分布[,,]が生じる.この光の屈折により,光路が曲がることで,スクリーン上には光強度の分布ができる(図18・27).この光強度分布は,流体の屈折率こう配の変化(密度や温度の2階微分値)を表している.よって,強度変化から密度変動をとらえることが可能となる.なお,水流にも用いることが可能である.. シュリーレン法(Schlieren method)シャドーグラフと同様に光の屈折を利用する.図18・28に示すように,レンズとナイフエッジにより屈折した光を止めることで,流体の密度こう配図18・27 シャドーグラフの原理(密度の1階微分値)を光強度変化に変換する.この強度変化から密度変動をとらえる.密度こう配の大きな衝撃波などの可視化に用いられることが多い.. マッハツェンダ干渉計(Mach-Zehnder interferometer)光が波であることを利用し干渉じまを用いて可視化を行う.図18・29にあるように,変調を受けていない参照光と対象によって波面が変調を受けた物体光を干渉させることで干渉じまが生じる.この干渉じまが,対象中の屈折率分布の積分値を表すことから,内部の密度変動をとらえる.参照光/物体光の配置を変えたマイケルソン干渉計なども用いられる.. ホログラフィック干渉計(horographic interferometer,図18・30)マッハツェンダ干渉計と同様に干渉じまを用いる.対象がない状態で,変調を受けていない光の情報をあらかじめホログラムに記録する.このホログラムにより再生される変調を受けない光と,対象により変調される光の干渉を記録する.物体光の光路上に波面を変調させる窓などがあるような場合であっても,その窓などの変調をキャンセルアウトできるので,容易に干渉計が構成できる.ホログラム記録媒体として,PPH(光導電プラスチックホログラム)などのホログラムがその場で形成できるものを用いると,過渡変動も容易に可視化することが可能となる.. スペックル法(speckle method,図18・31)シャドーグラフと同様に,光の屈折による光路の微小ずれを用いる.この微小な光路のずれを,光強度ではなく,レーザスペックルパターンの移動としてとらえる.パターンの移動量が屈折率分布こう配の積分値に対応する.旧来は写真を用いることが多かったが,ディジタル記録することで,c項に示すPIV と同様の画像解析手法を用いた高精度解析が可能である.. ドップラーグローバル流速計(DGV : Doppler globalvelocimetry)レーザドップラー流速計(18・5・3項参照)は流れによる光振動数のドップラーシフトを干渉によって取得する.ドップラーグローバル流速計は,このドップラーシフトを直接計測図18・28 シュリーレンの原理図18・29マッハツェンダ干渉計の原理図18・30 ホログラフィック干渉計の原理図18・31 スペックル法の原理する.可視光領域の光は数100THz の高周波であるのに対して,ドップラーシフトはMHz オーダであり,7けた以上小さい.そこでアイオダイン(iodine)の共鳴吸収の線幅がMHz オーダであることを利用し,周波数のわずかな変化を吸収量の変化としてとらえ,結果として透過光強度の変化としてとらえる.数百m/s程度の高速気流に対して適用され,速度分布を二次元的にとらえることに成功している.. サーモグラフ法物体表面からのふく射による赤外線をとらえる.ふく射量が表面温度の関数となることを利用して表面の温度を算出する手法.. 感圧塗料(PSP: pressure sensitive paint)法ある種のリン光物質は,そのクエンチ(quench)速度が酸素濃度や温度に強く依存する.紫外線を照射した場合の発光強度が酸素濃度に依存することを利用し,物体表面の酸素分圧を求め,これから圧力を算出する手法.PSPと呼ばれる.
