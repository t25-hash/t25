# 19・1・1 差分法と有限体積法

> **章**: 第19章 流れの数値解析
> **節**: 19・1 離散化手法

a. 差分法(finite difference method)差分法とは微分方程式に現れる微分商を差分商で近似して解く方法である.差分法では,解くべき領域を差分格子と呼ばれる格子状に分割して,格子点における解の近似値を求める.考え方が非常に単純であるため,どのような微分方程式にも適用できるという長所がある.実用上よく現れる線形偏微分方程式の境界値問題に適用した場合には,最終的に連立一次方程式を解くことになる.このとき,差分法では構造格子を用いることが多いため,連立一次方程式を行列の形に書いたとき,係数行列は規則的な疎行列になる.複雑な幾何形状をもった領域においても,不等間隔の格子を用いたり曲線格子を用いることによって近似解を求めることができる.b. 差分近似式微分を差分近似する公式を導く方法はいろいろあるが,最も簡単な方法はテイラー展開を使えばよい.例えば,の点=욡における2階微分の近似を,3点욡-,욡,욡+を用いて行うとする.このとき,dd~욡-+욡+욡+(19・1)とおいて,右辺を点욡のまわりにテイラー展開して同類項をまとめ,0階と1階微分の係数は0,2階微分の係数は1になるように,,を決めればよい.その結果dd~욡욪-2욡+욡용+(19・2)となる.ここで욡などは욡などである.この場合,3階微分以上の高次の項を無視したことになり,それが差分近似式の誤差になる.上の近似では3階微分項はたまたま消えて誤差の主要項はに比例する.このとき,近似式は二次精度であるという.この方法は不等間隔の格子についても使える.なお,偏微分に対しても変数が増えるだけで一つの変数に注目すれば常微分と同じ公式が使える.表19・1に代表的な差分近似式をまとめておく.その他,陰的な差分法としてコンパクト差分と呼ばれる一連の差分法がある.これは連立方程式を解いて近似値を求める.例えば1階微分の一つの近似として,6욡욪′+4욡′+욡용′=욡용-욡욪2(19・3)がある.上式(19・3)が各格子点で成り立つため,욡が数値で求まる.c.
