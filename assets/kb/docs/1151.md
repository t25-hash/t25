# 19・1・1 差分法と有限体積法

> **章**: 第19章 流れの数値解析
> **節**: 19・1 離散化手法

a. 差分法(finite difference method)差分法とは微分方程式に現れる微分商を差分商で近似して解く方法である。差分法では、解くべき領域を差分格子と呼ばれる格子状に分割して、格子点における解の近似値を求める。考え方が非常に単純であるため、どのような微分方程式にも適用できるという長所がある。実用上よく現れる線形偏微分方程式の境界値問題に適用した場合には、最終的に連立一次方程式を解くことになる。このとき、差分法では構造格子を用いることが多いため、連立一次方程式を行列の形に書いたとき、係数行列は規則的な疎行列になる。複雑な幾何形状をもった領域においても、不等間隔の格子を用いたり曲線格子を用いることによって近似解を求めることができる。b.
