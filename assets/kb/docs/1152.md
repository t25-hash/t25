# 19・1・2 有限要素法

> **章**: 第19章 流れの数値解析
> **節**: 19・1 離散化手法

有限差分法が基礎式の微分をテイラー展開に基づき置換える直接近似法であるのに対して、有限要素法(finite element method)は基礎式を変分法あるいは重み付き残差式に変換したのちに近似を行う間接近似法である。このようなことから格子分割について有限差分法は構造格子を、有限要素法は図19・2に示されているように、二次元では三角形あるいは四角形、三次元では四面体、プリズム要素あるいは六面体などの非構造格子を用いている。したがって、有限差分法と比較して有限要素法は複雑な形状への適合性がよく、複雑な形状を有する工学的な流体解析の応用問題に対して用いられることが多い。a. meth-od)により基礎方程式の弱形式化を行う。構造解析で見られるように、一般に有限要素法では変分原理により汎関数を導き出すことが多い。しかし、流体解析では基礎方程式の一つであるNS(ナビエ・ストークス)運動方程式が非線形方程式であるため、変分原理による汎関数が存在せず、重み付き残差法を用いて弱形式化する。重み付き残差法は、基礎方程式を近似した際に生じる残差に対して重み関数を乗じて全体積分し、このようにして得られた積分方程式が零となるように近似解を求める手法である。ここでは、具体的に非圧縮性の流体の基礎方程式を例に取上げ、有限要素法による離散化を試みる。非圧縮性の流体の基礎方程式は、以下の連続の式とNS 有限要素法による空間離散化有限要素法では図19・2に示すようなさまざまな形状の要素、または異なる形状の要素の組合せを用いて領域分割をする。このとき、各有限要素において節点値を補間する関数を用いて近似解を求める。一般に補間関数について、変数と同じ補間関数を重み関数(weight function)に用いるガラーキン法(Galerkin method)を用いるのが多い。流体解析では速度と圧力の補間関数の組合せが不適当であると、圧力に振動が発生する。したがって、速度と圧力の補間関数を選択する際には、圧力の補間次数に比べて流速の補間次数が十分である条件を満たすようにする必要がある。ここで具体的な例として、三次元の六面要素を用いる場合を取上げる。任意形状の六面体は図座標変換される。ここで速度は一次内挿関数、圧力は要素一定の場合を例に取上げると、各要素補間関数は式(19・29)(19・30)で表される。次にこれらの要素補間関数を弱形式化された式(19・24)と式(19・25)に代入することにより、ガラーキン法による有限要素行列式が導かれる。これらの有限要素行列をマトリックス表示すると以下のようになる。このようにして得られた要素ごとの有限要素方程式を解析領域全体に重ね合わせて全体化を図る。全体化した方程式は大規模な連立一次方程式となり、この方程式を解いて近似解を求める。導出された大規模連立一次方程式は有限差分法が帯行列になるのに対して、有限要素法は粗密行列となる、このため、有限差分法と比較して有限要素法は計算容量および計算時間の負荷が大きい。b. 連続の式とのカップリングと時間積分法有限要素法で離散化された連続の式とNS 方程式(19・31)と式(19・32)を解く手法にはさまざまなものが提案されている。おもな手法としては、二つの基礎方程式を代数的な式操作によってカップリングする手法と、二つの式を直接カップリングさせずに連立させて解く手法に大別される。前者は有限差分法でも用いられているMAC法をはじめとしてSMAC法、フラクショナルステップ法などが提案されている。後者は有限要素法の特徴である重み付き残差法に特有の解法である。基礎方程式のカップリング手法は、場合によっては時間進行法や計算の安定化手法とも密接に関連している。b項ではカップリング手法の代表として、有限要素法によるMAC法の定式化について説明し、そのあとに二つの基礎方程式を直接連立させる解法について説明する。. MAC法による定式化MAC法(marker and cellmethod)は、1965年にウェルチ(Welch, J.E.)らによって提案された自由表面を有する非圧縮性粘性流れの数値解法である。もともとは自由表面を有する流れの解析手法であったが、今日では自由表面の有無にかかわらず、その中で用いられているNS 方程式と連続の式とのカップリング手法を指してMAC法と呼んでいる。. 重み付き残差法による直接解法直接解法では連続式とNS 方程式の両方を含めた以下の弱形式化した式を用いる。式(19・36)に対して、19・1・2・a項で説明したようにガラーキン法を用いて空間離散化を行う。直接法を用いて解く際にはNS方程式に含まれる非線形の対流項を線形化するため、式(19・36)の変数をニュートン・ラプソン法に従って増分分解して速度と圧力を求める。ここで、移流項が卓越した場合には、有限差分法同様に風上化を用いて数値安定性を得る必要がある。有限差分法では差分のステンシルの選択により風上化を導入するが、有限要素法には重み関数を変化させるペトロフ・ガラーキン法を用いることが多い。そこで、式(19・26)のNS 方程式に人工粘性(artificial viscosity)を付加して弱形式化すると次のようになる。しかし、式(19・39)はそのままでは単に人工粘性を付加しただけであり、もともとの方程式が表現している物理現象と同じものを表しているとはいえない。このことからこの安定化法はSUPG(streamline-upwind/Petrov-Galerkin)法と呼ばれている。通常、六面体要素を用いた場合には速度に線形の一次補間関数、圧力には要素一定を用いる。しかし、問題によっては速度と圧力に同じ補間関数を用いる手法、すなわち有限差分法でのコロケーショングリッドに相当する手法を用いたほうが有利な場合がある。速度と圧力の補間関数の組合せは、前述したように非圧縮性の条件に起因する数値不安定性をいわゆる下限上限条件(inf-sup condition)、またはバブスカ・ブレッツィの条件と呼ばれる条件を満たすように選択をする必要がある。このため、速度・圧力ともに一次の補間関数を用いた場合には、数値不安定性が生じ、なんらかの形で圧力を安定化させる必要がある。そこで、以下に示す項を付加するPSPG(pressure-stabilized/Petrov-Galerkin)法と呼ばれる手法を用いることにより、数値不安定性を回避する。その中でも特に有限要素法の特徴を生かしたものとして時間空間有限要素法(space-time finite element method)が提案されている。
