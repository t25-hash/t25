# 19・1・2 有限要素法

> **章**: 第19章 流れの数値解析
> **節**: 19・1 離散化手法

有限差分法が基礎式の微分をテイラー展開に基づき置換える直接近似法であるのに対して,有限要素法(finite element method)は基礎式を変分法あるいは重み付き残差式に変換したのちに近似を行う間接近似法である.このようなことから格子分割について有限差分法は構造格子を,有限要素法は図19・2に示されているように,二次元では三角形あるいは四角形,三次元では四面体,プリズム要素あるいは六面体などの非構造格子を用いている.したがって,有限差分法と比較して有限要素法は複雑な形状への適合性がよく,複雑な形状を有する工学的な流体解析の応用問題に対して用いられることが多い.a.
