# 19・2・2 SIMPLE 法(SIMPLE method)による定常解法

> **章**: 第19章 流れの数値解析

定常解を求めたり大きな時間刻みを用いる場合には陰解法(im-plicit solution method)が適している。そのために、パタンカーらはSIMPLE(semi-implicit method for pressure linked equa-tion)法を提案した。以下にその手法を概説する。運動方程式の時間発展を陰解法で近似すると、一般に全格子点の未知数,に対する線形連立方程式となり、욣욗+∑욗=-1ρδ욗δ+욣욗+∑욗=-1ρδ욗δ+(19・66)4―218流体工学と表される。ここで、,は、速度成分についての係数行列の対角とそれ以外の成分(:定義点、:離散化式に現れる近傍点を示す)、は既知時刻で表されるすべての量を含む定数項である。左辺圧力項にはMAC法と同様にスタッガード格子またはコロケーション格子に基づく離散化がされるものとする。非圧縮性流れでは、これと同時に連続の式を満たすように、もう一つの未知変数と連立した解を得なければならない。SIM PLE 法では、以下の反復法(iterativemethod)を用いる。適当な仮定値から式(19・66)の係数行列,と圧力項を与えて、速度の新しい近似値を求める(も同様)。+∑=-1ρ옿δδ+(19・67)この解は一般に連続の式を満たさないため、修正量′=욗-,′=욗-を求めることが必要となる。そこで、式(19・66)と式(19・67)を差引いて′+∑′=-1ρδ′δ(19・68)を得たのち、簡略化のため右辺第2項を省略して、′=-1ρδ′δ, ′=-1ρδ′δ(19・69)と近似するのがSIMPLE法の要点である。求める解욗,욗が連続の式を満たす条件から、修正圧力′についてのポアソン方程式に類似の式が導かれる。ρδδδ′δ+δδδ′δ=δδ+δδ(19・70)SIMPLE法の反復計算アルゴリズムはMAC法(あるいはフラクショナル法)の時間進行法と類似しており、以下のようにまとめられる。ステップ0:反復=0の初期仮定値を与える。ステップ1:前回-1の反復値から係数行列,を算出する。ステップ2:式(19・66)を用いて=요に対する仮定値,を求める。ステップ3:式(19・70)を用いて修正圧力場′を解く。ステップ4:回めの近似値を요=+′,요=+′を得る。ステップ5:ステップ1~4を繰返し収束解を得る。定常流れ計算ではこの反復計算の収束値が求める解となるが、時間発展を計算する場合には各時間刻みごとにこの反復計算を実行することになる。SIMPLE法において式(19・69)の簡略化は、収束解が得られたときの結果には関係しないが、反復計算の収束性には大きな影響を与える。そのため、乱流モデルなどを用いた高レイノルズ数流れの定常解析では、ステップ2およびステップ4において次のような不足緩和を用いる必要がある。ステップ2:α욈+∑=-1ρδδ++1-α욊α욊요(19・71・a)ステップ4:요=+α′(19・71・b)緩和係数(relaxation factor)α=0.7,α욊=0.5程度の値が用いられる。式(19・69)の簡略化については改良法(PISO法、SIM-PLEC法など)があり、詳細は文献(17)を参照されたい。
