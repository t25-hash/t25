# 19・2・2 SIMPLE 法(SIMPLE method)による定常解法

> **章**: 第19章 流れの数値解析
> **節**: 19・2 節では、非圧縮性流れ(incompressible flow)に対しての

定常解を求めたり大きな時間刻みを用いる場合には陰解法(im-plicit solution method)が適している。そのために、パタンカーらはSIMPLE(semi-implicit method for pressure linked equa-tion)法を提案した。以下にその手法を概説する。左辺圧力項にはMAC法と同様にスタッガード格子またはコロケーション格子に基づく離散化がされるものとする。非圧縮性流れでは、これと同時に連続の式を満たすように、もう一つの未知変数と連立した解を得なければならない。SIM PLE 法では、以下の反復法(iterativemethod)を用いる。ステップ0:反復=0の初期仮定値を与える。ステップ3:式(19・70)を用いて修正圧力場′を解く。ステップ5:ステップ1~4を繰返し収束解を得る。定常流れ計算ではこの反復計算の収束値が求める解となるが、時間発展を計算する場合には各時間刻みごとにこの反復計算を実行することになる。SIMPLE法において式(19・69)の簡略化は、収束解が得られたときの結果には関係しないが、反復計算の収束性には大きな影響を与える。そのため、乱流モデルなどを用いた高レイノルズ数流れの定常解析では、ステップ2およびステップ4において次のような不足緩和を用いる必要がある。式(19・69)の簡略化については改良法(PISO法、SIM-PLEC法など)があり、詳細は文献(17)を参照されたい。
