# 19・2・3 密度変化のある流れの解法

> **章**: 第19章 流れの数値解析
> **節**: 19・2 節では、非圧縮性流れ(incompressible flow)に対しての

上述のMAC法およびSIMPLE法は流体の密度が一定な非圧縮性流れを前提としているが、熱生成や相変化を伴う反応流れ・混相流などの大きな密度変化があってもマッハ数(Mach number)が十分小さい流れにおいては拡張された解法が有効である。マッハ数が小さいことを仮定して速度、圧力、温度をマッハ数の関数として展開し、これを、圧縮性流れの基礎方程式に代入してオーダ評価をすると、基礎式は以下のように近似される。そこで、とを別のステップに分けて漸近的に解くとするならば、非圧縮性流れの解法と同様の解法が適用できる。低マッハ数の仮定が一般に有効である流れの例としては大きな温度分布のある浮力流れなどが挙げられ、このとき、熱力学的圧力が既知で密度変化が主として温度分布によって与えられるとすると計算手順は次のようになる。0) 第0近似の圧力を境界条件などから定める。1) エネルギー方程式を解いてを得る。2) 状態方程式より密度ρを得る。第0近似レベル。..............3) ρ既知として運動力学的圧力を解く。4) を用いて運動方程式より解いて速度場を得る。...............第1近似レベルここでは、手順3)、4)にMAC法を適用したが、上に述べたほかの非圧縮性流れ解法も同じように適用できる。ここでは圧縮性流れの解析に用いられる基礎方程式とその解法について概説する。個々の手法の詳細は19・1節や文献(27)~(29)に譲る。
