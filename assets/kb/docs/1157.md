# 19・3・1 圧縮性NS 方程式

> **章**: 第19章 流れの数値解析
> **節**: 19・3 圧縮性流れ

圧縮性流れの数値解析は、圧縮性NS 方程式(もしくは圧縮性オイラー方程式)を基礎方程式として、差分法や有限要素法などの離散化手法で解くのが主流である。a. 対流項流体の圧縮性が重要となる流れは、高速で高レイノルズ数流れである場合が多く、粘性項よりも対流項の影響が支配的となり、数値計算の精度や安定性は対流項の取扱いによって決まってくる。特に圧縮性流れでは流れ場の中に衝撃波やせん断層などの不連続が存在するので、これらを捕らえつつ、それ以外の領域でも離散化の近似精度を落とさない方法が要求される。このような手法を衝撃波捕獲法(shock capturing method)という。一方、衝撃波が離脱衝撃波のみであれば、そこを計算領域の境界として衝撃波の下流のみを計算する、衝撃波適合法(shock fitting method)という手法もある。しかし、衝撃波捕獲法も衝撃波のない滑らかな領域で十分な精度を有しているので、現在では衝撃波適合法であっても、衝撃波捕獲法の計算スキームをそのまま用いることが多い。. 風上差分に基づく方法現在最もよく用いられているのは、風上差分によって安定性を確保し、衝撃波などの不連続は、その付近で近似精度を最大で一次精度にまで落として格子点2,3点の幅で近似的にとらえる方法である。具体的には、一次元で隣合う2点間の物理量の差の絶対値の総和(total variation)が時間とともに増加しないというTVD(total variation diminishing)条件を満たすTVD スキームや、類似の考え方で差分をとる点の範囲を動的に変更するENO(essentially non-oscillatory)スキームなどがよく用いられる。また、これらを多次元に拡張して、多次元で局所的に最大値・最小値が成長しないようにするLED(localextremum diminishing)スキームや、正値条件(差分の係数がすべて正になるようにして解の安定性を保証する条件)に基づくスキームなども提案されている。これらのスキームの多くは、19・1節で述べたスカラ輸送方程式に対する風上法をNS(オイラー)方程式に拡張している。Λは対角行列なので、式(19・77)は未知数の数だけのスカラの輸送方程式になる。これにスカラの風上法を適用し、左からを掛けて戻す、というのが一般的な手順になる。Λの要素は流速および流速±音速となる。混相流、化学反応流などより複雑な方程式系でも同じ考え方が利用できる。. 人工粘性(artificial viscosity)に基づく方法風上差分が主流となる以前によく用いられていた方法で、中心差分に加える人工粘性の大きさを衝撃波などの流れ場の状態によって調節するものである。i項とは別の方法と思われがちであるが、風上差分も中心差分+人工粘性の形に書き換えることができるので、うまく設計された人工粘性であれば十分に実用的である。歴史的にはジェームソンのものがよく知られている。b. 粘性項粘性項は特別に精度を要求するのでなければ二次精度中心差分(またはそれと同等の離散化)とする。c. 時間積分境界層を解像するなどの理由で局所的に格子や要素が細かい場合は、時間刻みの制約から陰解法を用いることが多い。陰解法では近似因子化法(AF法: approximate factoriza-tion method)が最もよく用いられている。定常問題に限れば緩和法やこう配法なども有効であり、マルチグリッドなどの収束加速法ともしばしば組合わされる。また、時間精度が要求される非定常問題では内部反復を用いることがある。この場合は各時間ステップで収束計算が行われるので、AF法に限らず定常問題に使われるさまざまな解法が適用される。粘性項を省略したオイラー方程式を解くときなど格子が比較的大きい場合や、ほかの理由からいずれにしても大きい時間刻みがとれない場合は、陽解法も用いられる。陽解法の場合は、安定性や時間精度からルンゲ・クッタ法のような多段階積分を用いることが多い。d. 初期条件と境界条件。 初期条件初期条件は問題依存であるが、物理的に適当な条件がない場合は、流れ場全体に一様流(もしくは静止流)を与えることが多い。. 遠方境界非圧縮性流体に比べると、圧縮性流体は遠方境界条件の制約が少ない。それは、境界での多少の数値的な誤差や擾乱は圧縮性に吸収され、物体に大きな影響を与えないからである。したがって、遠方境界は一様流で固定するなど比較的簡単に条件を与えることができる。境界が物体に近い場合、非一様な流れが4―219流出する場合などは別途、条件を与えなければならない。. 流入・流出境界外部境界は極力計算対象から離れて影響の少ないところに設定するのが第一であるが、流れが非一様に流入出する境界では、そこに計算上の境界を設定した影響が内部に及ばないようにすることが必要である。一般には、境界面に垂直方向に各特性量ごとに伝ぱ速度を調べて、流出であれば内部から外挿し、流入であれば固定(外部から与える)する方法がある。より簡便に、特性量を計算せず、密度・速度は流速で外挿するか固定するかを判断し、圧力は(流速-音速)の符号でいずれをとるか判断するという方法もある。. 物体境界物体境界の条件を厳密に与えるのは簡単でない。明らかなのは粘性計算(NS 方程式を解く)の場合に物体上で流速が零であるということだけである。それ以外については近似的・離散的に与えざるを得ない。まず、非粘性計算(オイラー方程式)の流速は、物体に沿う方向に速度があるので、流れ場から速度を外挿し、物体に垂直な方向の成分のみ零とする。圧力は物体に垂直な方向のこう配を零として離散的に求めたり、NS 方程式の、物体に垂直方向の成分を(離散的に)解いて求めたりする。密度はこれも物体に垂直な方向のこう配を零とすることが多い。これらはいずれも近似であり、問題によっては変更が必要となる。また、有限体積法や有限要素法において物体表面で物理量を与えないようになっている場合は、物体表面で速度が零になるなどの条件を満足するように境界条件を工夫する。e. 乱流モデル圧縮性の乱流モデルは開発途上であるが、乱流モデルなしでは境界層が不自然にはく離するなどの問題が出やすいので、現在は非圧縮性流体の乱流モデルを利用することが多い。-εモデルのほか、代数型のボールドウィン・ロマックスモデルもよく用いられる。圧縮性流れは概してレイノルズ数が高いので、現在の計算機能力ではLES やDNS は困難である。
