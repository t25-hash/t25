# 19・3・2 その他の方程式

> **章**: 第19章 流れの数値解析
> **節**: 19・3 圧縮性流れ

a. ポテンシャル方程式非粘性で衝撃波が弱いときは,等エントロピー,渦なしを仮定することができ,支配方程式としてポテンシャル方程式を用いることができる.この場合も弱い衝撃波をとらえるための工夫が必要になるが,演算量がオイラー方程式に比べて少ないので,設計などにはしばしば用いられている.b. 低マッハ数近似流れの速さでなく,温度変化のために圧縮性を考慮しなければならない場合,19・3・1項の方法は必ずしも適当でない.それは,上記の方法が暗黙のうちに流速と音速のオーダが近いということを期待しているからである.音速に比べて流速が著しく(例えば2けた以上)小さいときは,19・3・1項の方法は誤差のために安定した解が得られないか,得られたとしても精度がよくないことが多い.そこで,圧縮性を考慮する低マッハ数流れでは別の方法が用意されている.それは低マッハ数近似と呼ばれる近似によって元の圧縮性NS 方程式を書き換え,これを解くものである.得られる方程式は非圧縮性流れの式に近いので,解法もこれにならったものになる.また,19・3・1項の方法でも,前処理によって特性速度を調節し,誤差を少なくして安定に解く方法も提案されている.流れの数値計算では,実験で流れ場を決めてから装置を設計し測定機器の選択をするのと同様に,目的を明確にしてから計算する領域と境界条件を設定し,次にモデルや数値計算法を適切に選択する必要がある.ここではその順序で乱流の数値計算について述べる.乱流解析におけるモデルと数値計算法についての解説として文献(30)~(34)を挙げておく.
