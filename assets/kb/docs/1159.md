# 19・4・1 解析可能なスケール幅

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

乱流は、流れの支配方程式の非線形性に起因し、レイノルズ数(Reynolds number)が高いときに発生する現象である。したがって、流れの数値解析法をそのまま適用すれば、原理的には乱流現象4―220流体工学を計算することができる。しかし、乱流は大小さまざまなスケールの渦によって構成されており、そのスケール幅はレイノルズ数とともに拡大する。例えば、流路や物体に関連して発生する渦の最大スケールを、エネルギーが散逸されるスケールをηとすると、両者の比はレイノルズ数に対してη∝と見積もられている。乱流変動は本質的に三次元現象であるから、すべてのスケールにわたって計算するには程度の格子分割数が必要であることになる。このことは、格子数1000の大規模計算であっても低レイノルズ数乱流しか扱えないことを意味する。そこで、乱流の数値計算においては、使用できる計算機の容量に応じて、実際の乱流場の中から解析するスケール幅の切出しを行う必要がある。計算領域よりも大きなスケールと格子幅よりも小さなスケールにはなんらかの仮定やモデルが必要となる。
