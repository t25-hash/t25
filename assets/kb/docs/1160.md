# 19・4・2 乱流計算のための境界条件

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

解析スケールの切出しと関連して重要な点は境界条件(bound-ary condition)である。速度変動まで計算する場合には、計算領域から流入・流出する時々刻々の乱れも与えなければならない。流入する乱れは一般には未知であるが、平均流れに適切な近似変動を与え、十分な助走区間を設ける必要がある。流出条件としては、一般に無反射境界条件が用いられる。特に圧縮性乱流においては、境界からの非物理的な反射が解析する乱れに影響を及ぼさないようにすることが不可欠である。物体遠方では、物体まわりの流れの影響を無視できない場合には一様流やポテンシャル流とするのは適切ではなく、トラクションフリー条件が用いられる。流入する乱流の統計量は一般には未知であるから、実験データを援用するか、(例えば主流方向に十分発達した仮定など)低次元の流れ場を別途計算した結果を用いるなどして流入境界条件とする。遠方や流出側の境界では非物理的な反射のない条件とする。固体壁や気液境界などにおいて速度のノンスリップ条件を用いる場合、速度変動強度などの乱流統計量の関数形も漸近挙動を合理的に満足するものでなければならない。界面を通して熱や物質の移動を伴う問題においては、境界近傍の乱れの影響が極めて大きいため、壁近傍でのふるまいには十分に注意する必要がある。
