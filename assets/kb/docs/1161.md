# 19・4・3 直接数値シミュレーション

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

直接数値シミュレーション(DNS : direct numerical simula-tion)は、乱流モデルを用いずに流れの支配方程式を計算するものであるから、最も小規模な散逸スケールの渦まで格子で解像することを前提としている。DNS では、計算領域を大きく確保することができないため、大スケール側の設定の影響が大きくなる。高レイノルズ数流れや複雑な形状の流れ場では大小の渦のスケール比が大きくなるので、DNS の適用範囲は単純な形状の流れ場における低レイノルズ数乱流に限られる。また、十分な助走区間を確保する余裕がないため、周期性を仮定できる流れ場を扱うことが多い。代表例として、一様等方乱流(三方向に周期的)、十分に発達した平行平板間乱流(二方向)やダクト内乱流(一方向)がある。これらの例から明らかなように、DNS は工業的な実用目的よりも乱流現象の基礎研究に用いられる。すなわち、DNS は、乱流理論やモデルの検証、外乱や境界からの操作に対する応答など、狭いスケール領域での数値実験手段として適している。DNS における空間微係数の近似としてはスペクトル法(spec-tral method)と高次精度中心差分法がよく用いられる。前述のような周期性を仮定できる方向があれば、精度の高いフーリエスペクトル法を適用しやすい。差分法の場合には、テイラー展開に基づく差分とパデ展開に基づくいわゆるコンパクト差分が用いられる。精度次数を高めるほど高波数(細かい変動)まで解像度が高くなるが、計算の負担も増大する。二次精度では格子を細かくしてカットオフ波数(計算できる最高の波数であり、等間隔格子では格子幅δに対して=πδ)を高くする必要がある。一方、六次以上では高次化の効果は次第に小さくなる。差分法では、プログラミングの手間も考慮すれば四次精度程度が妥当であろう。時間発展もこれに対応して二次以上の精度であることが望ましい。精度次数よりも重要な性質は離散化における整合性である。さらに、乱流理論やモデルの検証においては各種の統計量を残差なく算出する条件でもある。乱流変動の直接計算には、境界条件と同様に初期条件にも乱れの情報が必要である。非圧縮流れでは連続の式=0を満たす変動速度場を作成することは困難である。また、同様の方法でランダム外力を与えれば、統計的に定常な乱流場を生成することができる。しかし、一般に発達した乱流場を得るには、(連続の式を必ずしも満足しないが時間進行計算が破綻しない程度の)弱い乱数を初期値としたり、不安定モードを加振する方法があるが、それらの影響がなくなるまで十分に時間発展させる必要がある。
