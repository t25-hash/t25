# 19・4・4 数値粘性を用いた計算

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

物理モデルを使用しない乱流計算としては,上流化(風上化)を施した数値計算も行われている.具体的には,三次精度ないし五次精度の風上差分法やQUICK 法がよく使われる.前述のスペクトル法でもエイリアス誤差の除去操作が必要であり,中心差分法でも数値誤差は不可避であるが,上流化は拡散的な誤差[数値粘性(numerical viscosity)または数値拡散(numericaldiffusivity)]を加えるという意味でこれらと異なる.数値粘性は,本来はカットオフ波数付近で発生する数値的不安定を取り除くことを目的とするが,乱流に適用する場合には散逸的な最小渦に対する解像度不足を補完する手段として用いられる.以上のことから,上流化による乱流の計算はDNS とも19・4・5項のLES(large eddysimulation)とも区別されている.しかし,実用目的の乱流解析においては数値粘性の有用性は軽視できない.格子以下のスケールの渦に対しては,本来ならば物理的なSGS(subgrid scale)モデルを用いるべきであるが,複雑な形状の流れ場や複雑な組成の流体に対して普遍的に確立されていない.また,一部のSGS モデルは散逸性をもたないために数値的に不安定である.そこで,数値粘性が単独あるいはSGS モデルに加えて用いられることがある.その場合に注意すべきことは,高階微係数の数値粘性は細かい変動に作用することから,計算結果の中で比較的大規模な変動成分しか信頼できないということである.また,格子幅あるいは精度次数(数値粘性の微分階数)への依存性を調べ,解析したい大スケール変動には影響のないことを確認することが必要である.
