# 19・4・5 LES の数値計算

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

LES は、計算格子でとらえられない小スケールの変動にはSGSモデルを与えるので、DNS に比較して解析スケール幅を大スケール側にシフトさせることができる。換言すれば、高レイノルズ数乱流への適用範囲が著しく広がる。一方、19・4・6項のレイノルズ平均解析と比較すると、平均的流れを得るための計算量は多くなる。しかし、多くの実用計算に現れる複雑な流路の乱流では、非定常な大規模変動が支配的になることが多い。これを計算の対象とするLES の用途は、今後の計算機性能の向上を考慮すれば大幅に拡張されると思われる。LES ではDNS と同様に流れの基礎方程式に対する高精度計算を行う。スマゴリンスキーモデル(Smagorinsky model)やこれを基礎とするダイナミックスマゴリンスキーモデルなどでは、渦粘性型SGS モデルはフィルタ幅Δ(格子幅δのおおむね1~2倍)に対してΔのオーダである。モデルの不確かさを考慮すれば、LES の空間精度も二次程度で十分と思われがちである。また、三次以上の精度であれば、上流差分の数値粘性はSGS モデルよりも高次項となる。そのため、LES ではDNS に比べて数値精度にはあまり関心が払われていない。しかし、DNS のカットオフ波数は散逸スケールにあるが、LES のそれはエネルギー保有スケールに近いことが多いので、LES では高波数端までの解像度と精度の要請がむしろDNS よりも高いはずである。また、スケール相似則モデルやダイナミックモデルでは、変数にフィルタをかける操作やモデルそのものの性質を計算結果に的確に反映させるためには少なくとも四次精度が必要であろう。
