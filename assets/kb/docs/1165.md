# 19・5・1 格子形成手順

> **章**: 第19章 流れの数値解析
> **節**: 19・5 格子形成法

流体運動を記述する偏微分方程式を差分法なり有限要素法で数値的に解く場合の手順を図19・5に示す.まず,計算対象の形状データは三次元CAD の出力として与えられることが多い.三次元4―221図19・5 流れの計算手順CAD では,サーフェスは一般的にはNURBS(non-uniformrational B-spline)曲面で表現され,さらに有効領域の情報をもったものをトリム面と呼ぶ.NURBS からの出力データ形式にはIGES,STEPなどがあり,また小さな三角形(facet)の集まりとして三次元表面を表現するSTL(stereo lithography)データ形式も使われる.CAD のトリム面に対して表面格子(surface mesh)の形成を行う.次に,流入や流出境界,遠方境界などの計算領域境界に対しても表面格子を張る.そして表面格子によって完全に囲まれた空間に対し空間格子(volume mesh)を形成する.表面格子から空間格子までの格子形成(mesh generation)を計算の前処理(preprocessing)とも呼ぶ.一方,流れ場の計算結果から必要な情報を取出したり,等圧線図や流跡線などの画像処理をすることを計算の後処理(post-processing)と呼ぶ.
