# 19・5・2 格子と計算精度

> **章**: 第19章 流れの数値解析
> **節**: 19・5 格子形成法

格子の細かさ(格子密度)は,流れの特性,計算機環境やどの程度の精度を要求されるかにより決まる.例えば,航空機の翼のRANS 計算(6・5・1項参照)では壁面摩擦係数までを正確に予測するだけの精度が要求されるため,壁に鉛直方向の最小格子幅は境界層の粘性底層の厚さ以下にする必要がある.LES やDNS での格子幅については厳密な基準はないが,DNSでは,乱流変動の最小スケールと考えられるコルモゴロフスケールを基準に,その数倍程度に格子を切ることが行われている.また,壁近くの流れでは粘性底層内に数点の格子が必要であり,同程度の細かさは主流方向,スパン方向にも必要とされている.LES では,乱流の渦スケールをとらえることができる細かさの格子が要求されるが,普遍的な基準はない.
