# 9・2・5 粘性項に対する空間離散化スキーム

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・2 構造格子による解析

粘性効果が等方的な拡散であり，物理量分布の凸凹を平滑化する
作用であることを考えると，中心差分を用いて粘性項を離散化する
のが最も妥当であることがわかる．実際，ほとんどの計算において
二次精度もしくは四次精度中心差分が用いられている．
これまで見てきたように，圧縮性流体の数値計算におけるスキー
ムは，非粘性の特性理論に基づき，時間項と非粘性項の離散化に主
眼をおいて構築されてきている．このため，粘性項を陽的に取扱
い，基本的なスキーム（例えば，TVDスキーム）の中に粘性効果
を取り込むことはあまり行われていない．慣性効果と粘性効果の時
間スケールを考えると，圧倒的に慣性効果が速い現象であるため，
このような取扱いでも数値計算上問題の生じることはほとんどない
ものと考えられる．ただし，境界層の底部のような慣性効果と粘性
効果が競合するような領域では，慣性効果に基づいて構築されたス
キームが破綻することは十分予想でき，時間ステップ幅のとり方等
に注意を要する．
