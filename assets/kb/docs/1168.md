構造格子による複雑形状への対応複雑な形状まわりの流れ場では単一の境界適合構造格子では困難である。そのため、マルチブロック法や重なり格子法などが用いられる。. マルチブロック法リー(Lee)らによって提案されたマルチブロック法(multi-block method)は、三次元物体を直方体ブロックの集まりに置換え、そのまわりに格子のブロックを4―223積み上げて流れ場全体を覆う方法である。個々のブロックは直方体に近い形であり、ブロック内にH 型格子を形成することは容易である。隣接するブロックの境界面では格子点を共有している。流れ場の計算時には、ブロックごとに計算を行って逐次ブロック境界面で情報交換をする方法と、直方体で置換えた物体の内部にも格子を張って全空間を単一のH 型格子のようにして取扱う方法とがある。後者の場合、物体内外を区別するフラッグが必要であるものの、全場を一度に解くことによりベクトル演算機をもつ計算機には計算効率の点で有利となる。沢田らはこの方法を航空機全機まわりに適用し、その有効性を示した。. 重なり格子法ステーガーらにより提案された重なり格子法(overset grid method,chimera grid method)では、例えば飛行機全機まわりの計算の場合、翼や胴体といった部分ごとに格子を形成して複雑な形状まわり全体の場を表現する方法である。格子どうしは重なり、その重なった領域で格子間の情報のやりとりを行いながら計算を進める。マルチブロック法に対して重なり格子法を用いる有利な点は、格子どうしが時間的に移動することもできることであり、スペースシャトルのブースタが分離する過程のシミュレーション、ヘリコプタ全機まわりの数値計算や、トンネル内に突入する高速列車まわりの非定常流れのシミュレーションも行われている。複数の格子を用いて移動物体を扱うというアプローチは、非構造格子でも用いられる。
