# 19・5・6 直交格子

> **章**: 第19章 流れの数値解析
> **節**: 19・5 格子形成法

直交格子は,空間を単に,,座標方向に切る手法であり,その格子形成は,物体の表面を格子線が横切る点,および各格子点が物体の中か外かを判断することが基本である.そのため,複雑な三次元形状でも完全自動メッシュが可能なアプローチである.4―226流体工学ただし,全領域を等間隔の格子で切ると,飛行機まわりのように,物体からかなり遠方に境界をとるような外部流問題では膨大な格子点数を必要とする.このような場合は,場所に応じて細分度合いを変える方法がとられる.さらには,物理量変化の激しいところを細かくする解適合格子細分化(AMR : adaptive meshrefinement)も直交格子法の大きな利点として多用されている.直交格子では,図19・15(a)に示すような曲がった物体境界の取扱いに注意が必要である.現在用いられているおもな手法を図19・15(b)~(d)に示す.最も単純な方法は,図19・15(b)のように曲がった物体面を階段状の境界として表現するものである.二次元問題では,密な格子を使うことによってかなり正確な物体形状再現ができるが,三次元問題では格子点数が限られるため物体境界の精度の良い表現が難しくなる.図19・15(c)は,物体境界によって切り取られるセルを多角形セルとして非構造格子的に扱う方法である.物体境界再現は図19・15(b)よりも正確になるが,多角形セルに対するソルバの対応が必要となり,また非常に小さなセルが生じることへの対策も必要となる.これらの複雑さは,容易な格子形成という直交格子の利点を損ないかねない.図19・15(d)は,物体境界を含むセルにさらに細かなセルをはめ込む方法であり,図19・15(b)の単純さと壁境界再現の正確さをねらったものである.この他に埋設境界法(immersed boundary method)も注目されている.高レイノルズ数流れのNS 計算では,非構造格子と同じく,境界層領域に物体適合格子なりプリズム格子を用いるハイブリッド的なアプローチがしばしば用いられる.直交格子の境界での精度問題は,高レイノルズ数流れでかつ航空機のように壁面摩擦係数までの精度を要求する場合には確かに厳しいが,そこまでの精度を要求しない問題,例えばクリーンルーム内の流れや自動車エンジンルーム内,建築物内外の流れなどでは強力なアプローチである.
