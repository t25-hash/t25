# 19・5・7 解適合格子法

> **章**: 第19章 流れの数値解析
> **節**: 19・5 格子形成法

解適合格子（solution-adaptive grid）は，計算結果を基に格子
構造格子では格子点数および格子トポロジーを変更することはで
非構造格子法では，r法のほかに格子点を追加・削除するh法
分化を規則なしに繰返すとセル形状が急激に悪くなり，計算精度が
むしろ悪化する場合もある．そのため，ルールを決めて細分化する
等方的分割アルゴリズム，および二分割アルゴリズム がよく
用いられる．
直交格子は特に細分化が容易でありAMR と呼ばれている．
解適合格子法で，格子点を集めるための流れ場の解のモニタ量が
重要である．理想的には，離散化による打切り誤差を計算場全体で
最小になるように格子点を配置することが望ましい．しかしながら
打切り誤差の評価は複雑になり，またノイズも多いため，解適合格
子のためのモニタ量として必ずしも適切とはいえない．一般的には
なんらかの物理量をモニタすることが多い．例えば，衝撃波が支配
的な超音速流れでは，圧力や密度の1階微分，あるいは2階微分の
絶対値を用いることが多い．非圧縮性流れでは渦度も重要なモニタ
量となりうる．また，衝撃波や渦中心などの流れの特徴線を抽出
し，その近傍を細分化することも有効である．
19・6可視化手法
数値解析結果を可視化することは結果の理解に欠かせないもので
ある．可視化に用いるソフトウェアは，現在，有償/無償を問わず
多くのものが利用可能であるので，ここではそれらを用いることを
前提として，どのような可視化が可能であるかについて述べる．
19・6・1スカラ量の可視化
可視化するのは，格子点・節点ごと，もしくは要素ごとの物理量
である．物理量は大きくスカラ量とベクトル量に分かれる．ここで
は，スカラ量の可視化について述べる．それぞれの手法の詳細は文
献(81)(82)を参照のこと．
a.スカラ関数の種類 スカラ関数は，基本変数である密度，
圧力（静圧）のほか，マッハ数，総圧，動圧，温度，渦度などがし
ばしば可視化される．これらの空間微分値も用いることができる．
特に密度の空間微分はシュリーレン写真などとの対応から利用され
ることがある．また，数値的な不具合を調べる目的では，エントロ
ピーの分布を見て不自然なエントロピー生成がないか確認すること
もある．
b.メッシュ図 メッシュ図は，計算格子や要素の形を示すも
ので，それ自体はスカラ量の可視化にならないが，基本の一つとし
て取上げた．メッシュ図は図19･16のようなもので，格子の形状や
分布を示すほか，物体の形状などを表すのにも用いられる．また，
辺や要素の面にスカラ量に応じた色を付けることで簡易的にスカラ
量の分布を示すこともできる．
