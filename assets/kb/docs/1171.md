# 19・5・7 解適合格子法

> **章**: 第19章 流れの数値解析
> **節**: 19・5 格子形成法

解適合格子(solution-adaptive grid)は,計算結果を基に格子点を移動なり追加,削除を行い,計算精度を改善する方法である.解適合格子法には,流れの計算結果を基に格子点を移動させる方法(r法),新たに格子点を追加あるいは削除する方法(h法)とがある.また,有限要素法では要素の高次化により局所的な計算精度を上げる方法(p法)も用いられる.構造格子では格子点数および格子トポロジーを変更することはできないため,r法による解適合格子法が用いられる.これらの包括的な解説は文献(78)に詳しい.非構造格子法では,r法のほかに格子点を追加・削除するh法(格子細分化,mesh refinement)もよく用いられる.ただし,細(b) 階段状表現(c) カットセル表現(a) 直交格子原図(d) 細分化セル挿入分化を規則なしに繰返すとセル形状が急激に悪くなり,計算精度がむしろ悪化する場合もある.そのため,ルールを決めて細分化する等方的分割アルゴリズム,および二分割アルゴリズムがよく用いられる.直交格子は特に細分化が容易でありAMRと呼ばれている.解適合格子法で,格子点を集めるための流れ場の解のモニタ量が重要である.理想的には,離散化による打切り誤差を計算場全体で最小になるように格子点を配置することが望ましい.しかしながら打切り誤差の評価は複雑になり,またノイズも多いため,解適合格子のためのモニタ量として必ずしも適切とはいえない.一般的にはなんらかの物理量をモニタすることが多い.例えば,衝撃波が支配的な超音速流れでは,圧力や密度の1階微分,あるいは2階微分の絶対値を用いることが多い.非圧縮性流れでは渦度も重要なモニタ量となりうる.また,衝撃波や渦中心などの流れの特徴線を抽出し,その近傍を細分化することも有効である.数値解析結果を可視化することは結果の理解に欠かせないものである.可視化に用いるソフトウェアは,現在,有償/無償を問わず多くのものが利用可能であるので,ここではそれらを用いることを前提として,どのような可視化が可能であるかについて述べる.
