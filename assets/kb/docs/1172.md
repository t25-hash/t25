# 19・6・1 スカラ量の可視化

> **章**: 第19章 流れの数値解析
> **節**: 19・6 可視化手法

可視化するのは、格子点・節点ごと、もしくは要素ごとの物理量である。物理量は大きくスカラ量とベクトル量に分かれる。ここでは、スカラ量の可視化について述べる。それぞれの手法の詳細は文献(81)(82)を参照のこと。a. スカラ関数の種類スカラ関数は、基本変数である密度、圧力(静圧)のほか、マッハ数、総圧、動圧、温度、渦度などがしばしば可視化される。これらの空間微分値も用いることができる。特に密度の空間微分はシュリーレン写真などとの対応から利用されることがある。また、数値的な不具合を調べる目的では、エントロピーの分布を見て不自然なエントロピー生成がないか確認することもある。b. メッシュ図メッシュ図は、計算格子や要素の形を示すもので、それ自体はスカラ量の可視化にならないが、基本の一つとして取上げた。メッシュ図は図19・16のようなもので、格子の形状や分布を示すほか、物体の形状などを表すのにも用いられる。また、辺や要素の面にスカラ量に応じた色を付けることで簡易的にスカラ量の分布を示すこともできる。c. 等高線(contour lines)等高線は、ある面上でスカラ量が一定の線を描くものである(図19・17)。面は、構造格子であれば適当な格子面を選ぶことができるが、非構造格子の場合は、滑らかな面がないので、要素とは関係なく、適当に断面を設定して描くことになる。一般的には、断面と要素の面との交線でできる多角形を基に等高線を描く。もちろん、構造格子であっても断面をとることができる。面が決まると、各格子、要素内でスカラ量の分布を考え図19・16 三角形翼まわりの計算格子のメッシュ図図19・15 直交格子における物体境界の取扱いて等高線を引く。等高線ごとのスカラ量の値は、色付けによって区別する方法と、等高線上に数値を記入する方法がある。等高線はスカラ量の分布を基に描く方法であるが、等高線の疎密がスカラ量のこう配を表すことから、スカラ量の変化率を示すためによりよく用いられる。衝撃波を表すために密度の等高線を描くなどがその例である。d. カラー面塗りカラー面塗りは等高線と同じく面を選び、その面内のスカラ量の分布を色の変化で表すものである(図19・18)。格子点や節点など多角形の頂点でスカラ量を与え、スカラ量の値に対応する色を決めると、多角形の内部は頂点の色を基に適当に内挿して塗られる。等高線とよく似ているが、カラー面塗りはスカラ量の分布がそのまま見てとれるので、圧力や温度といったスカラ関数そのものの分布を示すのに適した方法である。e. 等値面(iso-value surfaces)これまでは二次元面内のスカラ量の分布を示す方法について述べたが、三次元空間内のスカラ量分布を表す方法として等値面がある。等値面は、スカラ量が一定の面を数値的に取出して表示する。手法としては、格子や要素の頂点でスカラ量を与え、内部は線形補間などをすると仮定すると、スカラ量が一定となる面を計算することができるので、これを要素ごとに求めて表示すると結果としてつながった面に見えるというものである。等高線の三次元版とも考えられるが、等高線と違い重なった面の向こう側が見えないので、描くことのできる面の数に限りがあり、どの値の面を表示するかが重要となる。端的な例としては、渦度の大きさが一定の面を描いて渦の強い領域を示す場合(図19・19)などが挙げられる。図19・17 二次元翼型まわりの流れの圧力の等高線図19・18二次元翼型まわりの流れの圧力のカラー面塗り表示図19・19チャネル内流れの乱流渦の等値面表示図19・22 円柱まわりの定常流れの流線4―227f. ボリュームレンダリング(volume rendering)等値面に対し、三次元空間内全域でのスカラ量の分布を表す手法としてボリュームレンダリングがある。ボリュームレンダリングはもともと医療分野の可視化から始まっているが、流体に利用する場合は空間をサイコロ状の小空間(ボクセル)に分け、それぞれのボクセルに物理量から色や透明度を与える。これを外から眺めると半透明なものを通してずっと先までを見通すことになり、煙や雲のようなイメージができあがる(図19・20)。もともと空間に解が分布している流体の計算結果はボリュームレンダリングの考え方に合っており、三次元的な分布を見るのに適した方法である。一方、得られる画像が積分の結果なので大ざっぱな様子はわかるが、各点の詳細な情報は得られない、ほかの可視化手法に比べ演算量が多く、表示に時間がかかる、必ずしもすべての可視化ソフトウェアに用意されていない、などの問題点もある。
