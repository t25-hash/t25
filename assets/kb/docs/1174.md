# 19・6・3 画像ファイルと動画

> **章**: 第19章 流れの数値解析
> **節**: 19・6 可視化手法

a. 画像ファイル可視化した結果は画像ファイルとして保存したり、あるいは文書などに張り付けたりする。画像ファイルの形式としては、EPS,TIFF,BMP,PICT,JPEG,PNG,GIFなどさまざまなものがあるが、多くものはそれぞれに変換が可能であるので問題はない。ただし、これらのいくつかは画質を落としたり色数を減らしたりしてファイルサイズを小さくするので、希望する画質によって使い分けることが必要である。b. 動画動画(アニメーション)を作る方法はおもに二とおりある。一つは、可視化しているPCやWS(ワークステーション)に、ディスプレイに表示されている映像の一部をTV 信号として取出す機能がある場合、可視化ソフトウェアによって動いている画像をそのままビデオに録画するものである。これはハードウェア/ソフトウェアの条件がそろっていること、ディスプレイ上で動いて見える程度に少ないデータ量の可視化であることが必要となる。もう一つは、静止画の画像ファイルを複数枚用意し、これを並べて動画にする方法である。この場合、1枚の画像を作るのに要する時間には制約がないので、どんな複雑な画像でも動画にできる。動画はディジタルビデオの形式となるが、周辺機器やソフトウェアを整えれば、TV 信号として出力し、ビデオに記録することもまた可能である。文献(1) 日本数値流体力学会有限要素法研究委員会編、 有限要素法による流れのシミュレーション、(1998)、シュプリンガー。(2) 亀本喬司、数値流体力学、2-1(1993)、20.(3) 亀本喬司、数値流体力学、2-2(1994)、28.(4) Zhu,B.、ほか2名、Comput.
