# 1・1・1 概説

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・1 エネルギー

わが国のエネルギー源の大部分は、輸入を主体とした石油・天然ガス等の化石燃料、そして原子力に依存している。石油に関しては、1999年の1年間でその原油価格が中東産油国の減産政策から約3倍もの上昇となり、わが国の石油依存体質の脱却に向けて、新たな展開の必要に迫られている。さらに、東海村のウラン加工施設において発生した臨界事故は、将来原子力エネルギーへの依存度を増大しようとするわが国のエネルギー政策に大きな影を落としている。これら二つの象徴的な出来事や、1997年の地球温暖化防止京都会議(COP3)における、炭酸ガスなどの地球温暖化ガス削減量の国際的取り決め等で、新たなエネルギー源の探索や効果的な環境負荷低減法の展開に向けての、革新的な技術開発やその方法論の構築が急務となっている。このようなエネルギーや環境対策に向けて、さまざまな規制緩和策や責任義務の強化策および市民意識の向上などから、閉塞感のあったエネルギー関連分野の技術開発やサービスなどで、ダイナミックな動きが見られ始めた。例えば、コンピュータシステム技術の飛躍的進歩によるコンピュータのダウンサイジング化や、有線型固定式電話からディジタル型の携帯電話への移行など、需用側がその利用に対して大きな自由度をもてる、高性能小型化かつ移動型へ多くのものが変革しつつある。エネルギー分野では、石油、ガスや電力の規制緩和策から、外国供給者の参入など自由化競争時代に入り、それらの価格・料金の値下げと需用者側のエネルギーに対する選択枝の拡大が可能となりそうである。環境対策としての電気自動車への取組みが、固体高分子形燃料電池の出現により一気に加速し、車メーカはその開発にしのぎを削っている現状にある。当初ガス供給会社が主体となり、数百kW 程度のリン酸形燃料電池の普及促進に力を入れてきたが、その導入普及は停滞ぎみである。ここにきてアメリカなどから、小型分散電源として数十kW から数百kW のマイクロガスタービンコージェネレーション(熱電併給発電)の、業務用などへ向けての売り込みが活発化し、国内メーカもこの分野へ参入せざるを得ない状況になっている。さらに、固体高分子形燃料電池を用いた数kW の一般家庭向けコージェネレーションシステム開発に向けての取組みも、家電メーカやガス会社により行われ始めている。特に、従来の電力供給やガス供給という縦割りのエネルギー供給に対して、ガス供給産業が電力を販売するといった総合エネルギー供給産業の出現が、外資系資本の参入などでわが国でも起こりつつある。このようなエネルギー関連産業の動向から、電力供給会社が分散型コージェネレーション分野へ進出する動きもあり、わが国の電力を中心としたエネルギー供給体制や形態が大幅に変化する時代へと移りつつある。また、21世紀におけるエネルギービジョンに関する調査によると、現在の電力供給を含む大規模集中型から、総合効率の高い分散型エネルギーシステムへの変革を推進すべきであるとの回答が、90%以上に達するとの報告もある。さらに、2000年からの改正省エネルギー法によるエネルギー管理指定工場の拡大や、そのエネルギー使用合理化の強化、長期エネルギー需給見通しにおける各種の分野での省エネルギーや新エネルギーの活用など、政策的にもエネルギー資源の多様化やその利用の効率化が叫ばれている状況にある。すなわち21世紀は、エネルギー、環境問題と経済の持続的発展(3E、トリレンマ)、さらに環境・エネルギー教育(4E)を含めた大きな四つの課題の克服に向けて、大きく動き出す時期にある。一方、物質の大量生産・消費・廃棄の時代から、4R(reuse,repair, remanufacture (reduce)、 recycle)の時代へと、環境を全面に出した資源保全型の社会システムの構築に関するさまざまな試みが起こっている。例えば、一部のメジャー石油会社は、大量の地5―1球温暖化ガス排出の元凶となる石油製品販売が将来の訴訟対象となることを見越して、太陽や風力エネルギー等の再生可能エネルギー供給への転換を計画している。さらに、自動車メーカでは、エネルギー利用効率の悪い自動車の製造から、電気自動車などの脱化石燃料を用いた交通システムへの転換を模索し、燃料電池自動車の開発技術の家庭向け分散型コージェネレーションシステムへの利用を考えている。環境ISOといわれるISO14000シリーズ取得数は、わが国において2641件(1999年10月まで)となり、その数は世界第1位である。さらに、割高でも環境配慮を優先した商品の購入を希望するグリーンコンシューマ化が進んでいる。環境関連の社会的動きとしては、政策として環境税や税のグリーン化等の検討が進められており、環境負荷の少ない製品やサービスを優先するグリーン購入、環境関連を志向する会社の株を売るエコファンド、再生可能なエネルギーを利用したグリーン電力購入、企業における環境コスト会計の公表などが企業を中心に進行している。エネルギー供給の大部分を海外に依存するわが国は、エネルギー資源の有限性および産地の偏在や政治的要因により、絶えずその供給の不安定な状態におかれ、エネルギー購入価格も他の先進国に比べて割高な構造となっている。さらに、わが国の全般的なエネルギー多消費構造がエネルギー需要の増加を導き、結果的にはさまざまな環境問題やエネルギー需給の不均衡を引き起こしている現状にある。この需給の不均衡を解消するには、単純に考えても、新エネルギーなどの供給の増大や安定化、そしてエネルギーの有効利用と使用量の絶対的縮小による需要の抑制を図ることである。21世紀後半のエネルギーシステムは、太陽等の自然エネルギーと原子力エネルギーを基幹エネルギー源に配置し、小規模化石燃料などを補助エネルギー源とし、中間媒体には電力や燃料電池によるコージェネレーションに必要な水素などの気体を利用するシステムとなることが予想される。21世紀前半のエネルギーシステムとしては、環境保全とエネルギーの安定供給を考慮したエネルギー源の多様化(ベストミックス)、およびエネルギーの有効利用や省エネルギーなどを中心としたものとならざるを得ない。すなわち、エネルギー供給源の多様化の観点からは、天然ガス(メタンハイドレートを含む)、再生可能な自然エネルギー、リサイクル型エネルギー、排熱を含む未利用エネルギーなど、いわゆる新エネルギー依存率を大幅に増大する必要がある。しかしながら、現在のエネルギーシステムは、文明の発達とともに自然発生的に出来上がったもので、最適性を満足するように構築されたものではない。すなわち、このシステムの要素はほかとは独立に設計されており、エクセルギー効率の観点からは極めてむだの多いシステム構造となっている。鉄鋼業などある特殊な産業内では、その利用に対するシステム化が行われているが、産業間や産業民生間のエネルギー連携はほとんど進んでいない。各プロセスで熱の温度は広い範囲にわたっていることを考えると、これらの熱利用を連携する熱のカスケード利用(heat cascading)の拡大が今後の大きな課題である。例えば、現在のエネルギーシステムを理想的に改編し、あるプロセスの排熱を他プロセスへの入熱とするように効率的に組合わせた場合、現行のヒートポンプ等の技術を利用すれば、わが国でもエネルギー需要がほぼ3分の1に低減できるとの試算例もある。このように排熱を動かすにはその間に熱を売買する産業が必要で、分散型地域エネルギーセンターやネットワークの構築が必要である。すなわち、積算エネルギー消費の増加傾向の抑制を目指して、エネルギーの総合利用効率の向上を図るには、地域冷暖房、工業団地規模のエネルギー連携、産業間のエネルギー連携、すなわち分散型地域エネルギーネットワークの段階的普及を着実に推し進める必要がある。今後は、エネルギー源の多様化すなわち複合エネルギー時代になるものと思われ、エネルギーの発生から最終利用に至るエネルギーフロー(エネルギーの流れ)の構築のため、エネルギー変換、輸5―2送、貯蔵、そして利用に関する先駆的な要素技術や革新的システム化技術を組込むことにより、新たなエネルギーシステムの提案が可能となる。魅力的なエネルギーシステムの獲得は世界的な趨勢(すうせい)にある。
