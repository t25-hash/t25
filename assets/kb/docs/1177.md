# 1・1・3 エネルギー変換

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・1 エネルギー

一つのエネルギー形態からほかのものへのエネルギー変換(energy conversion)はさまざまなものがあり、工業的に重要なものを図1・1に示した。化石燃料を燃やして(または原子力を使って)得られた熱エネルギーを、熱機関を用いて仕事(力学エネルギー)に変え、さらに発電機を用いて電気を得る、または電気を作らずに熱機関によって輸送に使う、あるいは熱エネルギーのまま使う、というのが一般的な方法である。量としてまだ多くはないが、化石燃料を直接電気に変換する燃料電池や太陽光から直接発電する図1・1 エネルギーの相互変換図1・2 わが国のエネルギーの流れ(2000年)5―3太陽電池があり、それらは今後増えると考えられる。この各種エネルギー変換においては必ずエネルギー損失を伴うが、その損失は熱エネルギーとなり、変換された各種エネルギーも最終的には熱エネルギーになる。具体的にエネルギー供給から消費に至るエネルギーの流れを図に示したものをエネルギーフロー(energy flow)といい、わが国のエネルギーフローは図1・2のようになっている。一次エネルギー総供給は原子力、石油、石炭、天然ガス、水力・地熱・新エネルギーであり、二次エネルギーは電力、石油製品、都市ガス、石炭製品等である。それが消費される状態のエネルギーとなり、最終的に熱の形に変わって散逸する。また、エネルギー変換のどの過程においてもエネルギーの総和は等しいという、エネルギー保存則が存在する。エネルギー変換には、一次エネルギーから二次エネルギーへの変換、二次エネルギーの輸送・貯蔵のためのエネルギー変換、エネルギーを利用するためのエネルギー変換などがある。現在、エネルギーフローにおける損失をなるべく少なくする変換の改良、従来捨てられていたエネルギーを有効に利用する方法の開発、効率のよい新しい変換方法の開発などが進められている。エネルギー変換の性能の評価には、熱力学の第一法則に基づく変換効率(conversion efficiency)と、第二法則に基づく有効率(effectiveness)がある。前者には、一般的に出力エネルギーを入力エネルギーで割ったものが用いられる。有効率は、出力側の有効エネルギーの増加量を入力側の有効エネルギーの減少量で割ったもので、エクセルギー効率(second law efficiency)ともいわれている。エネルギー変換はその内容により、要素型、サイクル型、システム型に分類できる。要素型のものは、単一の変換機器だけで二つの異なったエネルギー間の変換を行うものである。要素型の簡単なものの一つは、蒸気タービンやガスタービンにおける気体の膨張である。その評価は実際の出力と等エントロピー変化をするときの出力욦の比で表され、断熱効率(adiabatic efficiency)η욦とも呼ばれる。η욦=욦これは、理想的には1の値をもつ。化石燃料の化学エネルギーを熱エネルギーに変える燃焼器も、要素型変換を行う。変換効率は理想的には1であるが有効率は0.7程5―4熱工学度で、燃料を燃焼させると有効エネルギーが下がる。化学エネルギーと電気エネルギーの変換を行うものに燃料電池があり、反応物と生成物のギブスの自由エネルギーの差を、エンタルピーの差をとすれば、変換効率ηは次式となる。η=損失は変換過程で起こるエントロピー生成に起因する。現在燃料電池は、人工衛星、車両、家電、分散型エネルギーシステム等に使われつつある。電磁流体発電(magnetohydrodynamic electric power,MHDelectric power)は力学的エネルギーを電気エネルギーに、熱電発電(thermo-electric electricity generation)はp-n半導体を用いて熱エネルギーを電気エネルギーに、また熱電子発電は熱電子放射現象を用いて熱エネルギーを電気エネルギーに変換する。太陽電池(solar cell)は、p-n半導体などを用いて太陽光を電気エネルギーに変換する。燃料電池、太陽電池、電磁流体発電、熱電発電、熱電子発電などは、単一の機器で化石燃料、太陽光、熱を直接電気エネルギーに変換するので直接発電(direct electricity generation)ともいう。サイクル型変換は、一つの装置でいくつかの熱力学的過程を組合わせて行うもの、または、要素型変換を行ういくつかの機器の組合せである。前者には内燃機関のサイクル、スターリングサイクル(Stirling cycle)が、後者にはランキンサイクル(Rankinecycle)、ブレイトンサイクル(Brayton cycle)がおもなものとして挙げられる。これらのサイクル型変換で高熱源より得る熱エネルギーを、変換され出力として取出すエネルギーをとすると、サイクル変換効率には熱エネルギーを使うので、熱効率(thermalefficiency)ηと呼ばれる次式が用いられる。η=システム型変換には、いくつかのエネルギー変換のサイクルを組合わせたもの、または熱機関サイクルとその排ガスの熱エネルギーを有効利用するように組合わせたものなどがある。例えば、高温側のサイクル変換と低温側のサイクル変換を組合わせたシステム型変換が複合サイクルである。有効率によるエネルギー変換の評価では、タービンのように熱エネルギーを力学エネルギーに変換するようなときは、有効率と変換効率とは大差がないが、出力側が熱エネルギーであるときは、有効率は著しく低くなる。例えば、電気加熱器、電気湯沸器などは変換効率が100%に近いが、電気エネルギーをジュール熱で熱エネルギー変換しているため大きなエントロピー生成をきたし、有効率は20~30%と低い。一般にエネルギー機器またはプラントにおいては、変換効率のみでなく有効率について検討する必要がある。
