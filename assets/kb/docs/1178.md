# 1・1・4 エネルギーの輸送(輸送形態やその特徴)

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・1 エネルギー

エネルギーは唯一の普遍的な通貨として変換されて何かをなしており(スミル、Smil)、あらゆる現象に関与している。人類の生活・経済活動を支えているエネルギーは、さまざまな形態で輸送され、消費される。エネルギー源から輸送あるいは消費するために変換され、形態が変えられ、元の形態に戻されることもある。生活・経済活動を支えるエネルギーが輸送される形態・種類としては、一次エネルギーとしての化石燃料、その加工燃料、原子燃料、廃棄物やバイオマス、ならびに、これらの一次エネルギーや自然エネルギー(水力・地熱・太陽・風力など)が変換された二次エネルギーとしての電力や、加工燃料(メタノールや水素の輸送も提案されている)、あるいは熱エネルギー(流体)などがある。輸送されたエネルギーは消費システムにおいて再び変換・利用され、便益として消費される。このエネルギー源から消費されるシステムまでのエネルギーの生産・輸送・変換・消費の鎖を、エネルギーチェーンあるいはエネルギーフローともいう。多様なエネルギー資源を有効に利用し、地域環境、地球規模の環境に与える影響を評価するためには、エネルギー変換・輸送のプロセスに係わる選択肢に含まれる要素・システムの技術革新を考慮し、消費側で用いられるエネルギー変換システムを含めて、エネルギー利用の便益から見たエネルギーチェーン全体でとらえることが重要である。代表的なエネルギーの輸送形態としては、1880年にエジソンが設立した世界最初の電気事業の会社以来、自然独占的に構築されてきた電力系統(electric power system、電力システムあるいは電力ネットワークともいう)、都市のガスネットワークや、温熱・冷熱を輸送する地域熱供給システム、運輸システムによる化石燃料や加工燃料の輸送ネットワーク、工場内システムなどがある。また上記の燃料輸送ネットワークを利用し、小型のエネルギー変換機器を用いて消費エネルギー種別を消費側で総合的に発生・消費するオンサイトエネルギーシステムなどがある。さらに、新しい二次燃料や水素などの加工・人工燃料を利用した輸送形態・システム、そのための新しい要素技術が提案されている。電力系統は現代に不可欠なものとして、社会のエネルギー基盤となっている。日本の電力需要は21世紀初頭も増加の傾向にあり、電力化率(一次エネルギーに占める電力用エネルギーの比)も40%を超えている。日本の人口当たりの消費電力量を考慮すれば、人口は21世紀の前半でピークを迎えると予測されているので、発電・電力輸送の規模も21世紀半ば当たりで飽和するであろうとの展望もある。ある電力系統の負荷(kW)は、不等時性(負荷が最大となる時間帯の差異)をもった需要家や地域の負荷が合成されたものである。異なる電力系統を日本全体で合成したものの年間最大値(最大電力)や負荷のプロフィルは直接的な意味はないが、日本の発電・電力輸送・消費の規模や平均的な負荷構造を知るうえで参考になる。すなわち、1999年に電源にかかった最大電力(負荷/発電端)は電力9社の合計が167.43GWである。この値はその年の気候に大きく左右され、猛暑(2001年)により180GW をも超えることもあるので、電源容量(能力)は予備力が必要であり、電力系統も冗長な設備にならない程度に十分な輸送能力が必要となる。また、電力系統においては、年負荷率(年間の最大電力に対する発電電力量積分平均負荷の比率)が、電源を含む電力システムの指標として電気事業の創始以来重要視されている。日本の電力系統はほぼ地域全体の電力需要に応える形で構成されているので、電力系統の負荷はその地域の生活・経済活動、気候などを反映したものになっている。相互に連携されている日本の電力9社を合わせた年負荷率は、56%(1996年)程度である。日間の電力負荷については、1990年代で見ると、日間最大電力に対する深夜の最低負荷の比率すなわち深夜率は季節で大きく変化し、夏季の45%程度から春季など中間期の60%、ならびに年間特殊日(正月、盆)の平坦な負荷となっている。低い年負荷率や深夜率に対する対策としては、電気料金を時間帯や季節で差をつける季時別料金制度の徹底のほかに、揚水発電に代わる安価で貯蔵効率の高い電力貯蔵技術の開発が望まれているが、候補技術それぞれはコスト・性能面で課題を抱えているといわれる。日本の電力系統は地域(電力9社+沖縄)で大きく分かれており、大部分は連系されている。電力系統の構成はおおむね送電系統と配電系統から成っている。送電電圧は、長距離送電の安定性確保や損失抑制のために最高1000kV(建設済み)、都市を囲むような環状幹線や大都市中心部への供給ラインの500kV あるいは275kV が用いられ、需要地に向けて下位の系統に順次電圧を下げている。配電電圧は、22kV,6.6kV といった電圧が用いられ、住宅需要家ではさらに降圧されて100V(200V)となっている。住宅用には世界的に200V 以上が多く用いられているが、日本でもヒートポンプエアコンディショナなど電力消費の伸びとともに、大型家電機器に備えて200V 化が進みつつある。発電端から需要家に至るまでの電力の損失は、発電所における所内動力、送配電系統における変圧器損失、送配電線路損失であり、これらを合わせた総合損失率は約9%程度(1998年)である。需要端における発電効率[HHV(higher heating value)基準]は、改正省エネルギー法(1998年)によって評価のために規定されているが、LNG(liquefied natural gas)火力発電のコンバインドサイクル化、石炭火力の蒸気条件向上、既設火力発電プラントのリプレースなどで、次第に高効率化が進みつつある。電力系統は生産(発電)と消費が同時である。電力系統の周波数と電圧の変動はほとんど許されず、また輸送能力も送配電線路に対して最大値が決まっており、品質の異なる電力が途中で随意に連系されるようになるには、電力品質制御や運用に関する技術的、経済的ブレークスルーが必要といわれている。また、社会のエネルギー基盤としては、都市ガスネットワークがある。一般の需要に応じるために導管によりガスを供給する一般ガス事業(導管利用事業のすべてではない)がこれに相当する。都市ガスの原料は液化石油ガス(LPG : liquefied petroleumgas)、液化天然ガス(LNG)、天然ガスがおもなもので、ほかに石炭系購入ガス、石油系購入ガス、灯油揮発油などがある。LNG は気化の後LPG を用いて熱量調整後導管へ送る。LPG、ナフサなどは水蒸気を、石炭はコークス炉を用いて改質し、熱量調整用ガスを混合して水素、メタンなどを主成分とする発熱量15.07~100.46MJ/mN(0°C)のガスとして供給される。全国の私営・公営のガス事業者が販売するガス消費量(発電用を除く)は9.3×10MJ で、家庭用需要、商業用、工業用はそれぞれ約40%、約16%、約36%である(1997年)。また導管の総延長は、全国で約21万km(1997年末)である。導管のネットワークは、例えば大規模な東京ガスの場合、高圧ガスプラントから高圧導管(圧力1.08MPa以上)で長距離輸送(途中供給を含む)し、中圧ガスホルダを有する中圧導管(圧力0.395~1.08MPa未満、0.160~0.395MPa未満)ネットワークが構成され、さらに圧力を下げて低圧導管(圧力:100~250mmHO=245~981Pa)によるガスネットワークが構成されている。また圧力の異なる輸送本管はガバナで接続されて制御される。ガスネットワークはループを構成して、万が一の事故時も供給が可能となるよう工夫されている。天然ガスの利用は、以上のように発電用、家庭用、商業用、工業用など多岐にわたっている。そのための輸送手段は、世界的には需要の多い地域に天然ガスパイプラインネットワークが構築されているが、日本の場合は大規模ガス田から遠隔であるため、液化してLNG タンカーで輸送され、日本各地のLNG 基地(17箇所、1999年)において気化し、発電用や都市ガス用などに供給している(日本全体で約5000万t,1997年)。一方、国産天然ガスは、小規模ながら用途を越えた国内パイプラインが用いられている。国産の天然ガス田から需要地に輸送するもの(例:新潟から東京地区までの330km、北海道、秋田、千葉などの地区)、LNG 受け入れ基地から多目的用途の大量消費地へ輸送するもの(例:新潟から仙台市地区発電所までの250kmほか)である。ネットワークを構成して輸送されている電力、都市ガスのほかに、石油類、石炭、液化ガス等、原子燃料の運輸システムによる輸送がある。21世紀の前半は、世界的に天然ガス資源が豊富といわれており、具体的にはサハリンのガス田から新潟あるいは関東に至る1200~1600kmのパイプラインの可能性が議論されている(2001年)。このほかに将来の燃料として、各種エンジン、ガスタービン、燃料電池などエネルギー消費設備の技術革新や、燃料のクリーン化、輸送・ハンドリング性の追求を背景に、硫黄分を含まず窒素酸化物の発生も少ない図1・3のような二次燃料の候補がある。現在、運輸部門のエネルギー需要や分散型エネルギーシステムの需要に向けて、製造技術の改良、これらの輸送・供給インフラストラクチャ整備の検討がなされている。5―5
