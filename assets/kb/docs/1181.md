# 1・2・1 エネルギーシステムとエネルギー要素技術

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・2 エネルギーシステム

a. 地球温暖化抑止を目指し水素の時代へ大気中の二酸化炭素濃度は増加しつつある.このことが地球温暖化(global warm-ing)をもたらすか否かについては,実験的な確証が得られたわけではないが,実証されるまで待っていれば手遅れになるかもしれない.したがって,二酸化炭素の排出を可能な限り抑制する技術を開発することが,エネルギーに関わる機械技術者の使命であろう.化石燃料をエネルギー源として燃焼させる限り二酸化炭素の排出は避けられない.19世紀までの燃料は薪,石炭などの固体燃料であった.20世紀は液体燃料,つまり石油の世紀であり,内燃機関が発明され,自動車が走り飛行機が飛んだ.固体,液体と進化してきたから,21世紀は気体燃料の時代であろう.事実,石油のシェアは1970年代の後半に44%を超えたあたりでピークをとり,天然ガスに主役の座を奪われつつある.天然ガスはメタンが主成分であり,もともと硫黄分を含まないので燃焼させてもSOは排出しない.気体燃料であるから大量の空気と混合して希薄燃焼すればNOの排出も低く抑えられる.二酸化炭素の発生量も同じ発熱量に対して石炭の60%,石油の75%程度であるので,化石燃料の中では最もクリーンな燃料である.一方,21世紀の後半には燃料電池が広く普及し,気体燃料の中でも水素が燃料の主役となる時代と予測されてきたが,d項に述べるように,燃料電池技術の進歩は極めて早く,自動車用,あるいは小型コージェネレーション用として家庭用に燃料電池が用いられる時代が2010年ごろには始まると予測されるので,水素の時代もはるかに早まり,そのためのインフラストラクチャの構築などに着手しておく必要がある.b. エネルギーの有効利用これまでの日本のエネルギー供給構造は,供給義務を負った電気事業者が独占的に発電し,送電・配電を行って電気を消費者に販売してきた.一方,ガスおよび石油事業者は,工場のプロセス用の蒸気や家庭用・業務用の暖房・給湯などのために,都市ガスや石油を販売してきた.この二つのエネルギー供給の流れは並列であり,相互に入り交じることはなかった.電気事業者は電気の安定供給が使命であるから,発電だけを目的として燃料を焚き,熱機関で発電機を駆動して発電し,排熱は温排水あるいは排ガスとして環境中に排出してきた.日本の高度経済成長期にはスケールメリットを追求して,発電所は巨大化の一途をたど図1・8 新エネルギーの分類り,電力や熱の主要な消費地である都市から離れた辺境の地に立地して,消費地まで延々と送電することが定着した.発電所の排熱を利用しようとしても,熱を消費地まで運ぶわけにはいかなかったのである.一方,熱を売ることを業とするガスおよび石油事業者は,単に暖房器具や給湯器を販売してきた.つまり,エネルギー供給のこの二つの流れは,完全に並列かつ独立であったといえる.情報化社会が進展し,暖冷房完備の快適な生活に対する欲求が増大すれば,電力需要の増大は避けられない.電力を作るために投入する年間の一次エネルギー量の,一次エネルギー投入総量に対する比率を電力化率(electrification rate)と呼ぶが,その値は1975年度の27.5%から1998年度には43%まで増大している.電力を作るために投入した一次エネルギー量のうち約60%が,温排水あるいは排ガス損失として環境の中に捨てられるが,電力化率が増大するとこの排熱損失も増加し,国全体の総エネルギー損失の大きな部分を占めるようになり,国全体のエネルギー利用効率を次第に悪化させていく.例えば,先の数字に対応する1975年度の国全体のエネルギー利用効率は約37%であったのに対し,1998年度の値は34%と3ポイントほど悪化しているのである.この間国民が省エネルギーを特段怠っていたわけではないだろう.つまり,並列のエネルギー供給構造を維持する限り,国全体としてのエネルギー利用効率の悪化は避けられない.省エネルギーは本質的・構造的に実現不可能なのである.これを抜本的に改革するためには,発想を転換して直列の構造を取り入れなければならない.直列とはどのようなことを意味するのだろうか.c. 熱力学第二法則に基づく熱の直列利用山の上にダムを作り,水が海面まで流下してくる間に川の流れに沿って高度の高いところから順々に水力発電所を建設し,水の落差を使いつくすことは誰でも知っている.海面まで下りきってしまった水はもはや使いものにならない.「覆水は盆に返らず」である.熱エネルギーの場合には,落差に相当するのが温度差であり,海面に相当するのが地球の常温である.燃料に火をつけて発生させた1500°C以上の高温から,常温つまり15°Cまでの落差を使ってくるのが熱の利用であるが,不思議なことに熱の利用の場合には,水力発電のように温度の高いところから順々に使ってくることがほとんど行われてこなかった.熱を高温から常温まで丁寧に使いつくすためには,燃料に火をつけたらまずエンジンを動かして動力を取出すことを考える.お風呂や暖房の熱が欲しいときに,これまでは風呂釜やストーブに火をつけた.この操作は誤りであったことに気づかなければならない.これからは熱が欲しかったら,まずエンジンやガスタービンを導入することを考える.各家庭でもガスや石油に火をつけたらまずエンジンを回す.高温の熱はエンジンで動力化し,温度を下げてエンジンから排出された熱を風呂や暖房に用いることが,熱の合理的な使い方の第一歩である.高温の熱で動力や電力を作り出し,低温になった熱は熱として用いる直列型のシステムをコージェネレーション(熱電併給,cogeneration)という.日本のガス業界は,ある自動車メーカが製作した排気量163ccの単気筒ガスエンジンで駆動する出力1kW の発電機と,エンジン排熱を75°Cの温水として供給する家庭用コージェネレーションユニットを,世界に先駆けて2003年春に発表した.発電効率20%,排熱を含めた総合エネルギー利用効率85%というこのユニットの外観を図1・9に示す.エンジンは,図中左側の横幅580×奥行380×高さ880mmというパッケージ寸法に収まっている.図1・9右側の大型パッケージは給湯タンクである.コージェネレーションは,排熱を使いきるために熱の需要地に密着して建設されるのが普通であり,分散型とならざるを得ない.究極はこの例のように,熱を必要とする家庭の中にまで発電機が入ってくることになる.熱を高温から低温まで使いつくす原理によって高効率化に成功した例は,コージェネレーションばかりではない.最近の大型コンバインドサイクル発電をはじめ,既存のボイラ・蒸気タービンシステ5―9ムの蒸気タービンをそのまま生かしてボイラを撤去し,ガスタービンと廃熱ボイラに置換えてコンバインドサイクル化するリパワリング,ごみ焼却炉の熱で作る蒸気は,燃焼ガス中の塩化水素でボイラチューブが腐食するため蒸気温度を高くとることができず,発電効率が数%にとどまってしまうので,別途の燃料を焚くガスタービンを付加してその排熱で蒸気温度を高温に過熱し,コンバインドサイクル化するスーパーごみ発電など,いずれも原理は同じである.これらの例のように,既存の要素技術を組合わせシステム化することによって熱利用効率を飛躍的に向上させる技術を,システムエネルギー技術と総称している.d. コージェネレーションの要素技術1996年ごろから各種の展示会で目につき始めたのがマイクロガスタービン(micro gasturbine)である.1996年6月,イギリスのバーミンガムで開かれたASME Turbo Expo ’96で最初に発表したクレイグ(Craig,P.)は,自動車用セラミックガスタービン,航空機用補助発電機(APU : auxiliary power unit)などを手がけてきたガスタービン技術者であるが,それらの経験に基づいて,低コストの小型ガスタービン発電機を大量生産することを目指し,1988年,アメリカのカリフォルニアに会社を設立した.圧力比や温度も,圧縮機やタービンの材料も,静的部品の構造も,すべて確立されたターボ過給機の技術で完成させること,ただし,航空機業界で25年間も使われてきた空気軸受の技術を導入して,メンテナンスフリーを目指したという.当初の開発目標は出力10~100kW,タービン入口温度700°C,定格回転数96000min,再生熱交換器付き熱効率30%,NO9ppm以下(O15%),販売価格550$/kW(年当たり10万台生産の場合)ということであった.アメリカでは単なる発電機として発売したが,日本ではコージェネレーションのための要素機器として用いなければ意味がない.この会社の30kW(大気温度15°Cのとき)機をベースに日本のメーカが蒸気回収式コージェネレーション機にまとめた例を図1・10に示すが,幅790×奥行1850×図1・9家庭用1kW ガスエンジンコージェネレーションユニット図1・10 マイクロガスタービンコージェネレーションユニットの例(3台併結の場合)5―10熱工学高さ1750mm,重量約950kg,総合熱効率75%(低位)を実現している.一方,1997年11月にはドイツの自動車メーカが,2005年までに燃料電池ハイブリッド車の大量生産体制を整えると発表した.その燃料電池技術はカナダの燃料電池メーカの技術をベースにしたもので,固体高分子形と呼ばれており,電解質にフッ素系高分子の0.2mm厚程度の薄膜を用い,膜の両面を白金系の多孔質触媒で被覆している.さらに薄い多孔質の陰・陽電極を介して厚さ1mmほどの黒鉛板で挟んで一組のセルを構成し,燃料としての水素と,酸素の供給源である空気を流す流路となる細い溝が黒鉛板上に刻まれている.燃料の水素分子は触媒に触れて水素イオン(陽子)と電子に分かれ,陽子だけが膜を通り抜けて酸素極側に達し,再び触媒層に触れて供給されてきた酸素分子と,外部回路を経由して到達してきた電子をもらって水の分子を作る.生成された水は黒鉛板の溝に流し,発生した損失分の熱を70°C程度の温水として取出す.一組のユニットセルで0.7V を発生するが,このセルを積層して直列につなぎ所要の出力を出す.およそ大量生産向きの構造である.燃料電池は騒音・振動もなく排気はクリーンである.排熱が70°C程度のお湯で出てくるということは,家庭の給湯や床暖房にちょうど適している.日本の家電各社も,この固体高分子形燃料電池(PEFC: polymer electrolyte fuel cell)を用いた家庭用コージェネレーションユニットを早期に市場導入することを目標に開発を進めている.あるメーカが発表した1.3kW ユニットの外観を図1・11に示す.燃料電池本体は図1・11右下の幅860×奥行320×高さ850mmのパッケージに収められ,発電効率32%,熱利用を含めた総合熱効率72%(高位)という.図1・11左側の貯湯槽の容量は300Lである.以上のように,自動車と家電という大量生産の得意な両業界がリードすれば,価格も急速に低下していくであろう.原子力や大型コンバインドサイクル発電所などのシステムを親亀とすれば,その背中に子亀として,工場などの産業用コージェネレーションが乗る.その背中に孫亀として,マイクロガスタービンなどを用いたホテル,コンビニエンスストア,マンション,病院など業務用コージェネレーションが乗り,最後にひ孫亀として家庭レベルの燃料電池,あるいは小型ガスエンジンコージェネレーションが乗っている.このように,親亀からひ孫亀まで有機的な連系を行い,情報技術を駆使して30分先の電力と熱の需要を予測しながら,中央指令によって各レベルのエネルギー積分量の総和が最小になるように運転を行うことができれば,日本の二酸化炭素排出量は劇的に削減されるだろう.親亀がこければ皆こけるが,このような亀の子エネルギー供給システムこそが真の意味でのエネルギーベストミックスであろう.
