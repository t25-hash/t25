# 1・2・2 エネルギーの有効利用と省エネルギー

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・2 エネルギーシステム

a. 熱エネルギー一次エネルギーは石油、石炭、天然ガス、原子力、水力、地熱、新エネルギーなどである。水力以外はいった図1・11 家庭用1.3kW 燃料電池コージェネレーションユニットの例図1・12 熱のカスケード利用ん熱に変えて仕事に変えるか熱のまま使うので、熱力学の範ちゅうに入る。エネルギーは熱力学の第一法則から保存されるので、それを有効利用するには、むだをなくして所定の用途に使われなければならない。いま、の温度の作動流体があり冷却の温度をとすると、熱効率の理想的な値はカルノー効率(Carnot efficiency)で1-/となる。は通常大気温度または海水の温度をとるので一定である。よって、が高いほど効率が高い。一般の熱機関もが高いほうが熱効率が高くなる。一次エネルギーはできるだけ高い温度で作業流体を使うことと、実際のサイクルをカルノーサイクルに近づけることが望ましい。熱機関を作業流体の温度で見ると、最高温度はディーゼル機関(diesel engine)、ガソリン機関(gasoline engine)が2500°C程度、ガスタービン(gas turbine)が900~1500°C、蒸気タービン(steam turbine)が300~600°C程度になっている。これらの温度の最高値は材料の耐熱性で決まる。ただし、ディーゼル機関とガソリン機関の熱効率は、高位発熱量(HHV : higher heating val-ue)でそれぞれ0.3~0.5,0.25~0.4、ガスタービンと蒸気タービンではそれぞれ0.3~0.4,0.3~0.4である。ガスタービンや蒸気タービンで温度が低いわりに熱効率が高いのは、カルノーサイクルに近いからである。この熱効率は、加える一次エネルギーに対する取出される仕事または電気量である。一次エネルギーを有効に使うには、以下の点が重要である。(1)できるだけ高温で、まず仕事に変える。仕事は電気に変換されるか仕事のまま利用する。(2)そのとき排出される排熱は、温度が高いときは高圧蒸気などに変換し、電気、プロセス蒸気などに使う。(3)さらに低圧蒸気か高温水、低温水の温度レベルであれば、それに適した温度のプロセス温水や給湯に使用する。このような利用をカスケード利用(cascade utilization)と呼び、それを図1・12に示す。カスケード利用は、大きく分けて2種類ある。その一つは、例えば大型の火力発電所の場合で、まず高温のガスタービンで仕事を介して電気を得て、さらに高圧蒸気、低圧蒸気で蒸気タービンを作動させて電気を得る。ただし、排出する熱は捨てられている。これはコンバインドサイクル(combinedcycle)とも呼ばれ、電気を得るために複数のサイクルを組合わせて用いるもので、熱効率は50%(HHV)が得られる。ただし、市街地から離れた位置に設置されるので、電気は送電線で運ばなければならない。他方は、コージェネレーション(cogeneration)と呼ばれる方法である。一次エネルギーを高温のガスにし、熱機関を作動させ、仕事を得た後排出される熱を熱として用いる。従来、火力発電所などでは、電気という一つのエネルギーを取出すのみであったが、この熱エネルギーを有効利用すれば省エネルギーとなる。ただし、目的に合った熱の使い方ができる条件が必要であるため、分散型システム(dispersive system)になる。マイクロガスタービンや内燃機関を使ったマイクロコージェネレーションも分散型システムの例である。図1・13に大型発電所とコージェネレーションの考え方を示す。一般にコージェネレーションの熱効率は、最初の仕事に変換するところでは比較的小型で30%と高くはないが、排出される熱を使うので、総合効率は70~80%となる。また、両者の組合せとして、コンバインドサイクルコージェネレーションが使われつつある。コンバインドサイクルを使って熱を可能な限り高温から低温まで動力化し、その途中で目的に応じた熱を抽出してコージェネレーションシステムを構成するのが熱の合理的な使い方である。熱のカスケード利用にはリパワリング(repowering)の方法がある。これは、既設のボイラ・蒸気タービンのシステムに新たにガスタービンやピストン機関を付加して、熱をより高い温度から使い始めることをいう。すなわち、コンバインドサイクルに変換する方法である。ごみ焼却リパワリングでは、ごみ焼却炉から出る蒸気はボイラチューブが腐食しない程度の温度、すなわち290°C程度以下の温度とし、別に燃料を焚いてガスタービンを駆動し、その排ガスでこの蒸気をスーパーヒートすればコンバインドサイクル発電が可能になり、かなり高効率の発電所となる。同様の発想を原子力発電にも用いることが可能である。原子炉圧力容器の強度で系の最高圧力が抑えられ、その圧力に対応した飽和温度で熱機関としての最高温度が決められてしまうので、320°C前後が蒸気サイクル温度の上限である。そこで天然ガス焚き高温ガスタービンの排ガスを使って、リパワリングを行うことが考えられる。b. 燃料電池(fuel cell)最近、燃料電池が使われつつある。それは、化学エネルギーから直接電気を得るので、熱機関と異なり理論熱効率はで与えられる。ここで、は燃料のギブスの自由エネルギーで、は燃料の発熱量である。理論熱効率は0°Cで80%,1000°Cで65%というように温度とともに単調に下がるが、実際の燃料電池は低温で電気抵抗が高くなり中高の分布となる。燃料電池の種類は、低温型として作動温度が70°C程度の固体高分子形(PEFC: polymer electrolyte fuel cell)と170°C程度のリン酸形(PAFC: phosphoric acid fuel cell)がある。高温型として作動温度が600°C程度の溶融塩形(MCFC: molten carbonatefuel cell)と1000°C程度の固体酸化物形(SOFC: solid oxide fuelcell)とがある。燃料電池の特徴としてNO、ばいじんがほとんど出ないことも挙げられる。(a) 大型発電方式(b) 新エネルギーの利用新エネルギーは、自然エネルギーを利用するものや、これまで使われずに捨てられていたエネルギーを有効に利用するもので、化石燃料の消費量低減やそれに伴うCO排出量の低減などがその効果と考えられる。具体的には、太陽光発電、太陽熱利用、風力発電、廃棄物発電・廃棄物熱利用などや、バイオマス発電・バイオマス熱利用などがある。現時点においては、技術や経済性の面での課題等により、一次エネルギー総供給に占める割合は1%程度にとどまっている。d. 省エネルギー省エネルギーには発電、産業、民生、運輸などの分野があるが、コンバインドコージェネレーションの採用、リパワリングの採用、省電力機器の採用、低燃費自動車の普及、燃料電池の普及などが含まれ、利用者側の意識が重要である。
