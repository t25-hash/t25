# 1・2・3 エネルギーと環境

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・2 エネルギーシステム

a. はじめに―地球環境問題―ひとくちに地球環境問題といってもその内容は多岐にわたっているが、おおむね下記のように定義されている。(1) 被害、影響が、国内にとどまらず、国境を越え、ひいては地球規模にまで広がる環境問題(2) 先進国も含めた取組み[政府開発援助(ODA)等]が必要とされる開発途上国における環境問題これらの範ちゅうに含まれる具体的な問題として挙げられているのは、以下の事柄である。・地球温暖化:CO、メタン、亜酸化窒素、フロン類など・オゾン層破壊:フロン類・酸性雨:SO,NO・熱帯林減少:焼き畑、薪炭材などの過剰採取、農地化、過放牧・砂漠化:過放牧、薪炭材などの過剰採取・海洋汚染:油、浮遊性廃棄物、有害化学物質・有害廃棄物の越境移動:違法など不適正な輸出・野生生物種の減少:生息地の破壊・途上国の公害:工業化、人口集中これらのうち、現在世界的に最も注目を集めているのは、いうまでもなく地球温暖化問題である。この地球温暖化(より一般的には気候変動)については不確定要素を含み、異論のあることも事実であるが、これに先立って指摘された、フロンガスによるオゾン層破壊の可能性への対応の遅れに対する反省もあり、科学的知見に対する完全な合意以前に対策の必要性が広く認識され、予防原則(pre-cautionary principle)に基づいて行動が開始されるという、まれな状況を招いているのがこの問題の特徴である。また、大気中のCO濃度が下記の550ppmという一つの安定化目標を達成したとしても、そのCO濃度そのものが海洋表層中の生態系に影響をもたらす懸念も指摘されている。地球温暖化への寄与率が約60%にすぎないCOが特に重要視されるのは、大気中濃度の増加の主原因が、全世界のエネルギー供給量の90%近くを賄い、現代文明を支えている化石燃料消費の加速的増大にあり、その排出抑制には、人間活動に対する制約を含めた極めて大きな困難が予測されるためである。その一方で、工業化へ向かう潮流はまさに全地球上を覆っており、アジア・アフリカの発展途上国を含めて、化石燃料の消費とそれに伴うCOの排出はますます増加していくものと見込まれる。2001年に発行された、気候変動に関する政府間パネル(IPCC:図1・13 大型発電方式とコージェネレーションシステム5―12熱工学Intergovernmental Panel on Climate Change)の第三次評価報告書第一作業部会報告「科学的根拠」の、「政策決定者向けの要約(Summary for Policymakers)」では、現在までに得られている科学的知見に基づいて、下記のような主張がなされている。・観測成果の蓄積により、世界的な規模での温暖化および気候システムにおけるその他の変化についての全体像が明らかになってきている。・近年になって、最近50年間に観測された温暖化のほとんどは人間活動によるものであるという、より強力な証拠が得られている。・21世紀中を通じて、人間活動が大気組成を変化させ続けると見込まれる。また、同第三作業部会報告「緩和策」の「政策決定者向けの要約」では、下記のような言葉で、気候変動によってもたらされる問題がわれわれにとって無視し得なくなってしまう10年あるいは20年後ではなく、ただちに対策が必要であると述べている。・排出抑制、技術開発や科学的不確実性の低減から成るポートフォリオを考慮した、より迅速な行動が、温室効果ガス(green-house effect gas)の大気中濃度安定化に向けての方向性に柔軟性を増すことになる。・迅速な短期的な行動により、急速な気候変化に伴う環境や人間へのリスクを低減できるだろう。一方、この温室効果ガスによる気候変動問題は、その性質を考えても、短期的・中期的対策のみで解決できるものでなく、今後100年、あるいは200年を見通した長期的観点のもとに、世界規模で取り組む必要のある課題であることは明らかである。しかし、IPCCでの検討でも議論されているように、長期的に大気中のCO濃度を550ppm、すなわち産業革命前の濃度の2倍のレベルに安定化させようとした場合、2050年ごろには現在の全世界の排出量を1990年レベルの半減以下にする必要があることを考えても、対策の困難さは明白であろう。b. 対策技術の概要COの大量排出による気候変動問題は、まさにエネルギー問題と表裏一体としてとらえることができるが、以下では、そのような観点から対策技術について概観する。. エネルギー供給側からのアプローチ2002年のわが国の年間温室効果ガスの排出量約13.3億t(CO換算)のうち、COの排出量が約12.5億t、そのうち90%以上がエネルギー利用に関わっている。また部門別の直接排出量の内訳で見ると、エネルギー転換が30.4%、産業部門が30.1%、民生が13.2%、運輸が約20.4%となっている。したがって、従来からのエネルギー源の多様化や有効利用を目指した技術は、その多くが対策としても有効であり、研究開発と同時に技術の導入促進を図ることが重要である。エネルギー転換部門の中でも、火力発電の高効率化と低炭素燃料への転換は大きなCO排出削減効果を有する。わが国の過去30年間の電力供給実績を見ると、1970年に0.60kg-CO/kWhであったのが、2000年には0.37kg-CO/kWhへと約4割低減しており、使用電力量が約3倍になったにもかかわらず、CO排出量は約2倍にとどまっている。これには、効率向上や送配電損失の低減等に加え、石油危機以降に急速な伸びを見せたLNG(liquefied natu-ral gas)火力と原子力の導入が寄与している。また、再生可能エネルギー(renewable energy、太陽エネルギー等の自然エネルギー)技術はCO排出原単位が極めて小さいという特徴があるが、エネルギー密度の希薄さ、出力の間欠性やそれに伴うコスト高などのため、短中期的に温室効果ガス削減に大きく寄与するだけの規模での導入は困難と見られている。また、同様にCO排出原単位が小さい原子力発電は、安全性への疑問などに起因する立地上の困難など問題を抱えているが、その取扱いは温暖化問題のみならず、今後のエネルギー政策上も重要な課題である。発電効率向上と燃料転換によるCO排出削減効果を試算してみると、比較対象にもよるが、効率向上により数%から20%以上、石炭火力から高効率LNG 火力への転換により50%以上の削減効果を有することがわかる。しかし、既設火力のすべてをLNG 複合発電に置換えたとしても、総排出量の削減は1割程度にとどまる。また、資源制約やアメリカでのLNG 輸入の動きなど世界的な需要増大を考え併せると、現時点で天然ガスへの大規模な転換を前提として考えるのには限度がある。. エネルギー利用側での対応直接的なCO排出抑制策はいわゆる省エネルギーであるが、これには、節エネルギー型ライフスタイルの確立と、技術開発による機器やシステムの効率向上という二つの側面がある。わが国では、石油危機後の技術開発や導入が効果的に行われたこともあり、諸外国と比較しても効率的なエネルギー利用が図られている。しかし、1985年以降のエネルギー原単位[単位GDP(国内総生産)当たりのエネルギー消費量]の値がほぼ横ばいであり、また個別機器の効率は向上しているにもかかわらず、利便性の追求や製品の大型化等により、特に民生部門におけるエネルギー消費が増大している。このように、石油危機以降個々の機器の効率が極限的といわれるまでに向上してきている反面、エネルギー多消費型ライフスタイルにより効果が相殺される場合も多く、一般的に消費者の志向に帰せられるが、そのニーズの大部分は供給側の利益のために創り出されてきていることが多かったのも事実であり、最近の環境志向型の製品供給を売り物にした産業活動のさらなる展開に期待するところが大きい。. システム化によるエネルギー有効利用上記のように、エネルギー供給側と消費側からの個別の対応によるCO排出抑制には限界があり、未利用エネルギーの活用、熱のカスケード利用、コージェネレーションに代表される分散型システムの導入、およびこれらシステムに対するエネルギーマネージメントの高度化にも注目が集められている。これらに関して重要な点は、エネルギー需給構造を把握し、インフラストラクチャの変革に要する期間やコスト、あるいは消費側の利便性をも考慮した評価に基づく最適化を行って初めて実効が上がるということであろう。このようなシステム化を図るためのエネルギー媒体の有力な候補として挙げられているのが、水素である。水素は、化石燃料などの一次エネルギーや、二次エネルギーである電力から容易に変換・製造でき、エネルギー貯蔵媒体、また高効率化の可能性が高い燃料電池用の燃料としても利用できる特性を有する。そのため関連する技術開発が順調に成功した場合には、将来のエネルギーシステムの核として機能しうる可能性を有している。. Carbon Sequestration LeadershipForum)が組織された。このCOの回収隔離技術については、化石燃料からの水素製造プロセスにおける排出物としてのCOにも適用できることから、IPCCやCSLFにおける議論の中でも、将来の再生可能エネルギーや水素を核にしたクリーンなエネルギーシステムが実現するまでの、つなぎの技術として位置づけるようになってきている。c. 終わりに―対策技術の適正な評価―以上、気候変動問題の背景と、対策技術の考え方について述べてきたが、これら技術開発の今後の方向を考える際には、単なる技術的側面のみではなく政策的・制度的な要因など、種々の要素を考え併せた総合的な評価が必要と考えられる。CO対策技術が満たすべき基本的要件は次のようにまとめられよう。(1) ネットでCO低減効果がある。(2) 国・世界規模で寄与できる容量を有する。(3) エネルギーや資源の所要量(損失)が許容範囲内にある。(4) コストや規模が許容範囲内にある。(5) 二次的な容認不可能な環境問題やリスクを伴わない。(6) 所要の時期までに実施可能である。(1)は極めて当然であり、場合によってはライフサイクル評価により有効性を示す必要もあるが、明らかに技術の新規性や研究開発予算獲得だけを目的とし、ネットでの削減効果がほとんど零またはマイナスにしかなり得ない技術提案がいまだに見られることに注意すべきである。(2)も排出量の膨大さを考えれば当然であろう。(3)と(4)はエンジニアリング段階において極めて重要な要素となるが、実際の許容範囲に対する定量的基準はない。特にコストについては、排出権取引や炭素税といった経済的、あるいは政策的オプションにより判断基準が大きく変わる可能性がある。(5)は、例えば大規模な風力発電の導入に伴う社会との摩擦といった事例、あるいはCOの隔離に伴うリスクや環境影響などが該当し、便益とコスト(環境影響やリスク)とのトレードオフに基づいた評価と、社会的合意形成が必要な課題である。(6)も重要な要素であり、温暖化を含む気象変動のメカニズムあるいは予測における不確かさを考えると、明確なタイムリミットを設定することは困難であるが、IPCCで指摘されているような気候変動問題への迅速な対応の必要性などを考えれば、10~20年以内にある程度の規模で実効を上げうるとともに、百年から数百年の期間にわたって効果を持続できる技術が必要となるだろう。ここに挙げたような考え方に沿って、わが国、あるいは世界規模での気候変動対策技術の選択肢について適切な評価に基づいた優先度付けが行われ、迅速にかつ長期的な取組みがなされることが必要である。
