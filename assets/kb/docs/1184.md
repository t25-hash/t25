# 1・2・4 エネルギー・環境関連法規

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・2 エネルギーシステム

環境エネルギー関係の法律は、環境六法によくまとめられている。熱工学の関係者に必要な関連法規は以下のとおりである。a. 環境関係環境問題(environmental problem)に対する国の姿勢は「環境基本法」に示され、そこに、国は「環境基本計画」を策定し発表することが定められている。これに従って策定された環境基本計画には、国の環境保全に関する基本方針が示されている。その全体は膨大な内容であるが、国が対策を実施すべき環境問題の分野は、1地球規模の大気環境の保全:地球温暖化対策、オゾン層保護対策、2そのほかの大気環境保全:酸性雨、光化学オキシダントなど、3水環境、土壌環境、地盤環境、4廃棄物・リサイクル対策等の物質循環、5化学物質対策、6自然環境の保全と自然とのふれあい、の6項目であるとしている。この基本計画に沿って、個々の環境問題に対処するようそれぞれの法律が定められており、また、今後整備されていく。現在の法律として注目すべきものは、全般:環境基本法、環境基本計画、環境影響評価法地球温暖化防止およびオゾン層保護関係:地球温暖化対策の推進に関する法律、特定物質の規制等によるオゾン層の保護に関する法律5―13大気汚染防止関係:大気汚染防止法水質汚濁防止関係:水質汚濁防止法廃棄物および資源関係:廃棄物の処理及び清掃に関する法律、資源の有効な利用の促進に関する法律、ダイオキシン類対策特別措置法、循環型社会形成推進基本法を挙げることができる。b. エネルギー関係エネルギーを扱う事業のための法律として、電気事業法、ガス事業法、石油事業法、熱供給事業法がある。それぞれその事業を行うときは事業の計画を提出し、経済産業大臣の許可を受けることなどが定められている。c. 省エネルギーの法律昨今のエネルギー供給の状況により、新しいエネルギー源の導入が必要であるとの見地から、新エネルギー利用等の促進に関する特別措置法、石油代替エネルギーの開発及び導入の促進に関する特別措置法が定められ、政府がその分野の事業を推進すべきであるとされている。また、わが国はエネルギーの合理的使用すなわち省エネルギーを、産業、民政、運輸の各分野において、産業政策および地球環境保全の両面から強力に推進する必要があるとの判断から、エネルギー使用の合理化に関する法律(1979年6月22日制定、最終2006年4月1日一部改訂)が制定され、国民の省エネルギーへの努力が要請されている。この法律は第二次オイルショックの年に制定され、細かい改訂を加えて現在に至っているもので、その体系を図本法律は燃料資源の有効な利用の確保に資するため、工場、建築物および機械器具についてエネルギーの使用の合理化に関する所要の措置を講じて、国民経済の健全な発展に寄与することを目的としている。1998年度の改訂では、地球温暖化ガスである炭酸ガス削減のために省エネルギーをいっそう強化するため、第二種エネルギー管理指定工場の制度、特定機器の省エネルギー性能向上のためのトップランナ方式(top runner method)の導入、国による指導および助言の強化などが導入された。法律の内容を網羅することはできないので、細かい規定は省略して以下に示す。1. 基本方針経済産業大臣は、国の省エネルギーについて基本方針を示すことになっている。1993年に基本方針が発表されている。全般的に努力するべき分野や目標が示されている。2. 建築物に係る措置(建築主の努力)建築物の建築をしようとする者(建築主)は、基本方針の定めるところに留意して、次に掲げる措置を実施し、エネルギー使用の合理化に資するよう努めなければならない。・建築物の外壁、窓等を通しての熱の損失の防止のための措置・建築物における空気調和設備その他の制令で定める建築設備に係るエネルギーの効率的利用のための措置(建築主の判断の基準となるべき事項)経済産業大臣および国土交通大臣は建築物に係るエネルギーの使用の合理化を図るため、建築主の判断の基準となるべき事項を定めこれを公表するものとする。4. 機械器具にかかる措置(製造事業者等の努力)エネルギーを消費する機械器具の製造または輸入の事業を行うものは、基本方針の定めるところに留意して機械器具のエネルギー消費率向上に努めなければならない。(製造事業者等の判断の基準となるべき事項)特定機器(表1・4)ごとに性能向上に関し製造事業者等の判断の基準となるべき事項を定めこれを公表する。この際、判断基準は当該特定機器のうち性能が最も優れているものの性能、技術開発の将来の見通しその他の事情を勘案して定める。これがいわゆるトップランナ方式である。年間に規定の台数以上製造または販売された機種が該当することになる。(性能の向上に関する勧告および命令)判断基準に満たない場合は改善を勧告できる。従わない場合は公表できるし、手続きを経て改善命令を出すことができる。(表示および表示に関する勧告および命令)特定機器のエネルギー消費効率を表示する方法を示している。業者が表示しない場合は表示するよう勧告し、従わない場合は公表し、さらに手続きを経て表示するよう命じることができる。5. 新エネルギー・産業技術総合開発機構のエネルギー使用の合理化の業務この法律で、NEDO(New Energy Development Organization)に追加した業務として、エネルギー利用合理化のための技術開発、技術導入に必要な資金の補助金、情報の収集および提供、技術に関する指導の業務が規定された。6. 雑則国はエネルギー使用の合理化等を促進するために必要な財政上、金融上および税制上の措置を講じること、科学技術の振興、国民の理解を深めるための措置、地方公共団体の教育活動等における配慮、改善のための資料として業務の状況に関し報告させること、また必要に応じ立入り検査ができる。7. 罰則以上の規定の中で、必ず実施すべき事項に違反した場合には、罰金に処するとして、細かく罰金の規定が定められている。
