# 1・2・5 新たなエネルギーシステムの展開

> **章**: 第1章 エネルギーおよびエネルギーシステム
> **節**: 1・14 に図示する.

熱エネルギーは,種々のエネルギー形態の中で最も有効仕事への変換効率が悪い低品位の形態であるにもかかわらず,現実には最も多く使用され,地球環境問題の要因になっている事実がある.したがって,熱エネルギーから有効仕事を最大限引出し,同時に環境にやさしく,可能な限り化石燃料に依存しない循環型エネルギーシステムが新たな展開条件となる.ここでは,エクセルギー効率(sec-ond law efficiency),LCA(life cycle assessment,ライフサイクルアセスメント)環境負荷評価,脱化石燃料化の新展開を述べる.a. エクセルギー効率による技術展開熱エネルギーシステムに求められる環境性はまず,投入したエネルギーから最大量の有効仕事を引出すことである.このエネルギー変換率は通常,投入したエネルギー量읉윰に対して取出すことができる有効仕事量=읉윰+윱윲읏との比として,次式で定義される熱効率ηで評価している.η=읉윰=1-읉윰윱윲읏(1・1)ηの添字1は熱力学第一法則に基づいているという意味である.これに対し,投入したエネルギーから取出しうる最大有効仕事量を意味するのがエクセルギー(exergy,availability)で,これは熱力学第二法則に基づき,変換プロセスで生じる種々の非可逆現象のため次第に減少する.実際に,温度,圧力の環境と異なる温度と圧力の閉じた系へ環境から熱エネルギーδが流入し,系のエクセルギーがdだけ増加したとするとdはd=δ-d+d(1・2)で与えられる.また,開いた系に対しては次式で与えられる.d=δ-d(1・3)非可逆過程が生じるたびにd>0であるからd<0となり,エクセルギーは保存されずに必ず消費され,したがって,有効仕事量は次第に減少することが証明できる.もちろん,環境と同一状態であればd=0,すなわち有効な仕事は取出せない.なお,式(2・2)と式(2・3)はおのおのヘルムホルツ(Helmholtz)とギブス(Gibbs)の自由エネルギー=-と=-に関連する.系に投入された熱エネルギー읉윰から有効仕事へ変換できる最大値,すなわちエクセルギーの읉윰との比をエクセルギー効率と呼ぶ.η=읉윰(1・4)式(1・4)をd=0の開いた系の式(1・3)に対して適用した場合,ηは動作温度が環境温度より高くなるに従って増加し,有効仕事への変換割合も増加するが,その増加割合は次第に減少する.化石燃料が元来有する化学エネルギーに対応するエクセルギー効率では,燃焼前の化石燃料はη=90~95%で高効率であるが,これを燃焼させると1500K 程度になり,そのエクセルギー効率は60%程度に一気に低下する.すなわち,燃焼させるだけで30~35%も有効仕事への変換能力を失ってしまう.この事実が,熱エネルギーの有効仕事への変換効率を著しく低下させる要因である.したがって,できれば燃やすことなく改質することが省エネルギーの基本である.燃焼させるとしても,系へ投入する熱エネルギーが高温であるほどエクセルギー効率が高いので,燃焼温度ができるだけ高い水素のような物質にして燃焼させるのが理にかなっている.いったん変換すると温度が低下するから,排熱利用の際のエクセルギー効率も悪化する.したがって,高温の水素燃焼でまず金属溶解を,その排熱でガスタービン発電を,次にその排熱で蒸気タービン発電を,さらにその排熱で給湯と冷暖房を,と次々にカスケード利用することが熱エネルギーを最大限活用できる秘訣となる.これが省エネルギー技術のポイントである.またエクセルギーの概念は,環境より寒冷のエネルギーに対しても適用できる点にも留意して,有効利用すべ5―16熱工学図1・15 LCA 環境負荷評価のプロセスフローきである.新たなエネルギーシステムの展開にはエクセルギー効率の概念が不可欠で,これを正しく適用して機器設計に当たる必要がある.b. ライフサイクルアセスメントによる環境調和展開エネルギーシステムは化石燃料枯渇や地球温暖化の環境負荷に直接関与するため,LCA による定量的な環境影響評価を行って,その環境調和性をPR していく必要がある.ISOもいわゆるエコマークのようなラベリング認定にLCA 手法の導入を検討している(ISO 14020s).環境負荷評価で重要なことは,1客観的・普遍的な数値として表す,2原因の異なる環境負荷を同一基準で統合化する,の2点である.LCA は図1・15に示すように,原材料の採掘(ゆりかご)から廃棄(墓場)までの製品一生に関わる全段階の負荷を取扱う.ある特定の段階や原因だけを取上げても意味がなく,すべての比較検討から製品のエコロジー改善に結び付けることが重要である.実際のLCA 環境負荷は,製品に使用される一つ一つの材料の質量や投入エネルギー量から,インベントリーデータベースに基づいて個々の負荷量を算出し,エコロジー改善方策が検討される.上記二つの課題のうち1の数値化は精度の問題が残り,2の統合化は国際的にも同一基準化指標が確立していない.ここでは,多種多様な環境負荷因子を統一的に評価するスキームLCA-NETSを取上げる.NETS は統合化した環境負荷量を表す新単位(numericaleco-load total standard)で,環境負荷を与える側(ローダ)の排出量あるいは消費量が,それを受ける側(レシーバ)の許容量と最大我慢できる限界でバランスを保つという左右均衡則によって同一基準化する.すなわち,環境負荷因子に対してローダの最大消費/排出量욡(kg,kWh,mなど)とレシーバの許容限界負荷量욡(NETS)がバランスを保つと考えると,욡욡=욡(1・5)が得られる.욡[NETS/(kg,kWh,mなど)]は換算係数であるとともに,単位消費/排出量当たりの環境負荷量(統合化基準値と称する)でもある.LCA-NETS では工業製品のLCA 環境負荷評価に不可欠な化石燃料枯渇(fossil fuel depletion),天然資源枯渇,地球温暖化,オゾン層破壊,水質汚染,大気汚染,酸性雨,廃棄物処理,リサイクル効果に関する環境負荷を取扱い,容易なソフトによって算出した各段階や原因別の詳細数値表やグラフが出力できる.また,コストも原単位を入力すれば詳細算出が可能で,LCC(life cycle 3E 同時達成と脱化石燃料への展開エネルギー(energy),環境(environment),経済(economy)の3Eは,トリレンマとして同時解決は困難といわれてきた.しかし,3E同時達成はいわゆる環境ビジネス,静脈産業,グリーン調達,環境税といった創成的コンセプトに基づく,産・官・学・民一体の新価値観の台頭によって現実として可能となりつつある.一方,地球温暖化(global warming)に代表されるグローバルな環境問題の解決には,化石燃料にできる限り依存しない燃料源の探求確保が不可欠であろう.なぜなら,本来静かに地下深く眠っているはずの孤立炭素系(化石燃料)が,まるで水道管が破裂したように地上へ噴出していることに本質的原因があるからである.バイオを中心とした循環炭素系である限り,地球温暖化のような問題は起こらないことを考えると,脱化石燃料への技術展開は環境問題の本質的で唯一の解決策であろう.エネルギーや資源関係の学会でバイオ関係のセッションが超満員となっており,新たな急展開を大いに期待させる.触媒や酵素を駆使した炭酸ガスや木材系化学種の燃料種への改質,酸性排水や酸性雨から金属間ポテンシャルを利用した水素製造と中性化処理,フリーエネルギーのような未開拓エネルギー発電など,脱化石燃料化に向けた具体的な技術展開が急速に高まっており,目を離せない状況が続くものと思われる.文献(1)
