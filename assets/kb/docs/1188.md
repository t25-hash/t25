# 2・1・3 温度

> **章**: 第2章 熱力学
> **節**: 2・1 熱平衡と温度・熱エネルギー

温度(temperature)は,一つの物質の系において,あるいは複数の系の間で熱平衡が実現しているか否かの判定の指標になる.一つの系のある部分の温度が他のすべての部分の温度と等しければ,その系は熱平衡の状態にあり,等しくなければ,高温の部分から低温の部分に熱エネルギーが移動する.一つの系と他の系の間の関係についても同様である.国際単位系SIにおける温度は,熱力学温度(thermodynamictemperature,K)である.SI では,水の三重点(tripletpoint)の熱力学的温度を273.16K(=0.01°C)とし,理想気体の状態式に基づいて,その温度の1/273.16を温度差1K であると定める.国際温度目盛(International Temperature Scale,ITS-90)はSIに依拠してできる限り熱力学的温度に一致する温度値を与え,また,温度が適切に校正できる(トレーサブルである)ように温度を定めるものである.その17の定義定点の一部を表2・1に示す.これらの定義定点の中間あるいは範囲の外については,気体温度計,白金抵抗温度計,ふく射温度計などを用いて内外挿し,温度を目盛付ける.日常生活に用いるセルシウス温度(°C),ファーレンハイト温度(°F)の熱力学温度に対する数値の関係は,次の式で表される.°C=K-273.15(2・1)°C=5°F-32(2・2)
