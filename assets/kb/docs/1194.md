# 2・2・3 項と2・3・2項では,物質としておもに流体を想定し,巨視

> **章**: 第2章 熱力学
> **節**: 2・2 熱力学の状態量

的な運動エネルギーや巨視的な位置エネルギーあるいは電磁気学的なエネルギーなどが,熱力学的なエネルギーに比べて小さく無視できる場合を考える.自動車のエンジンのシリンダ内における流体の圧縮・膨張過程では,流体と外部との間で物質のやりとりはない.このような閉鎖系(closed system)における流体の現象については,その物質の内部エネルギー(internal energy,J)が重要である.内部エネルギーは物質の内部に蓄えられた熱エネルギーであり,微視的には,原子・分子などの粒子の集合の運動エネルギーと位置エネルギー(ポテンシャルエネルギー)から成る.その大きさは,物質のそのときの状態のみによって定まる.すなわち,内部エネルギーは状態量である.単位質量当たりの内部エネルギーを比内部エネルギー(specifc internal energy,J/kg)と呼ぶ.発電所のボイラ・タービン系のような,流体が定常的に流れる流動系(flow system)における状態変化の現象を取扱うときには,=+(2・5)で定義されるエンタルピー(enthalpy,J)に注目するのが有効である.,,が状態量であるので,エンタルピーも状態量である.この量によれば,流れの過程での流体の圧力・比体積の変化を考慮して,流れが運ぶ熱力学的なエネルギーを一括して取扱うことができる.単位質量当たりのエンタルピーを比エンタルピー(specific enthalpy,J/kg)と呼ぶ.=+(2・6)図2・1に示す流路内の一次元定常流れにおいて,質量流量(kg/s)は流路に沿って一定である.単位時間に流路の仮想断面1を通って仮想断面1,2間の領域に流入するエネルギー(J/s=W)は,流入する流体の内部エネルギーと流体が断面の下流側の流体にする絶対仕事(2・3・2項参照)から成り,+=(2・7)である.断面2を通って領域から流出するエネルギーは,同様に,+=(2・8)である.流体が領域内で熱エネルギーを得て,外部に工業仕事욧(2・3・2項参照)をするとき,断面1,2間における流体図2・1 流路内の一次元定常流れ5―19のエネルギー収支は次の式で表される.=-+욧(2・9)流体が流体の外部と,熱エネルギーや工業仕事のやりとりをすることなく流れる場合には,流路に沿って比エンタルピーが保存される.すなわち,流動系における熱力学的なエネルギーの変化は状態量によってよく記述される.ただし,エンタルピーや比エンタルピーは熱力学の状態量であり,流体の流れを前提とする概念ではない.なお,物質の系の状態変化を扱う熱力学の問題では,多くの場合,その比内部エネルギーや比エンタルピーの絶対値よりも,変化に伴うそれらの値の差が重要である.そこで,これらの値については,物性値の表において,例えば水については,三重点(三重線上の飽和水)の状態での比内部エネルギー,比エントロピーを零であると定めて,物性値を目盛付けすることがある.
