# 2・2・3 項と2・3・2項では、物質としておもに流体を想定し、巨視

> **章**: 第2章 熱力学
> **節**: 2・2 熱力学の状態量

的な運動エネルギーや巨視的な位置エネルギーあるいは電磁気学的なエネルギーなどが、熱力学的なエネルギーに比べて小さく無視できる場合を考える。自動車のエンジンのシリンダ内における流体の圧縮・膨張過程では、流体と外部との間で物質のやりとりはない。このような閉鎖系(closed system)における流体の現象については、その物質の内部エネルギー(internal energy,J)が重要である。内部エネルギーは物質の内部に蓄えられた熱エネルギーであり、微視的には、原子・分子などの粒子の集合の運動エネルギーと位置エネルギー(ポテンシャルエネルギー)から成る。その大きさは、物質のそのときの状態のみによって定まる。すなわち、内部エネルギーは状態量である。単位質量当たりの内部エネルギーを比内部エネルギー(specifc internal energy,J/kg)と呼ぶ。この量によれば、流れの過程での流体の圧力・比体積の変化を考慮して、流れが運ぶ熱力学的なエネルギーを一括して取扱うことができる。単位質量当たりのエンタルピーを比エンタルピー(specific enthalpy,J/kg)と呼ぶ。すなわち、流動系における熱力学的なエネルギーの変化は状態量によってよく記述される。ただし、エンタルピーや比エンタルピーは熱力学の状態量であり、流体の流れを前提とする概念ではない。なお、物質の系の状態変化を扱う熱力学の問題では、多くの場合、その比内部エネルギーや比エンタルピーの絶対値よりも、変化に伴うそれらの値の差が重要である。そこで、これらの値については、物性値の表において、例えば水については、三重点(三重線上の飽和水)の状態での比内部エネルギー、比エントロピーを零であると定めて、物性値を目盛付けすることがある。
