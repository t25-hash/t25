# 2・4・2 熱機関とサイクル

> **章**: 第2章 熱力学
> **節**: 2・4 熱力学の第二法則

熱を仕事に変換する機械を熱機関(heat engine)と呼び、熱機関において熱の授受に関与する気体や液体などの媒体を、作動物質(working substance)または作動流体(working fluid)という。熱機関が連続して一定の仕事を行うためには、その系の状態が、ある値から出発して再び元に戻る過程をたどる必要がある。この過程をサイクル(cycle)という。ただし、作動物質が時間的に状態変化することは同じでも、熱機関に注目したときに、ある空間が周期的に変化している場合(例えばレシプロエンジン)と、空間的に異なる部分で異なる過程が同時進行している場合(例えばタービン)がある。また熱機関は、作動物質の循環という視点から、閉じている場合と開いている場合に分けることもできる。例えば2・13節で述べるように、ボイラで発生した蒸気でタービンを動かすランキンサイクルは、水を作動物質とする閉じたループであり、またエアコンディショナの冷凍サイクルは冷媒を作動物質とする閉じたループであるが、一般の内燃機関では、空気・燃料(あるいは燃焼生成物)から成る作動媒体が周囲の環境を含めて循環するとみなして、サイクルを形成すると考える。その熱効率(thermal efficiency)ηは、次式によって定義される。
