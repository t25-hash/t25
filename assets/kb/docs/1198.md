# 9・4・3 スクロール

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ，ノズルなどの内部流れ

外向き旋回流を集めて吐出し管に導くスクロール（scroll ）は，
図9･26のような形状をとるので渦巻室（volute）とも呼ばれる．
巻き始め位置の基準半径における円周上で一様流と仮定して，スク
ロール内で断面平均流速が周方向に一定とする条件をおき，連続の
関係と角運動量保存を基に巻き始めの位置から断面積を線形的に増
加するように，通常，渦巻形状が定められる．式(9･64)にその関係
を示す．
＝2πθ (9･64)
ここで， ：角度θにおけるスクロール断面積， ：巻き終わ
り部の断面積である．スクロールの断面形状として円形，長方形，
台形などが使われる．送風機で使われることの多い平行壁の場合に
は，スクロール外壁は巻き角一定の対数らせんとなる．
への圧力こう配によって壁面に沿う内向きの二次流れが生じ，現実
の流れは複雑である．吐出し量が減少すると羽根車出口流れの周方
向速度成分は増えるが，スクロール各断面における平均流速は流量
図9･27吸出し管内のらせん状渦心
にほぼ比例して低下する．そのため，スクロールに入る流れはその
内部で円周方向の運動量を減らし，それに見合った圧力上昇が出口
に向かって現れる．一方，吐出し量が増える場合には逆の効果で圧
力降下する．このように設計点からはずれた運転点では，周方向に
圧力が変化して羽根車出口流れも円周方向に一様ではなくなり，結
スクロールに羽根車からの流れが直接入る場合には，巻き始めを
形成する舌部の半径位置や形状が流力性能に大きな影響を及ぼすの
水車ランナ下流に置かれる固定の広がり流路を吸出し管（draft
tube）という．円すいディフューザも使用されるが，構造上の都
合から曲り吸出し管が広く使われている．低落差の場合は水車の水
力性能が吸出し管の圧力回復性能に支配されるので，有効度が高く
なるような形状が選ばれる．フランシス水車のような高落差の場合
フランシス水車ランナ出口の流れは定格運転点ではほとんど旋回
をもたないが，負荷に応じて流量を減らしていくとランナ回転方向
の旋回流になる．旋回が強まると流れは外周に片寄る傾向を有する
ので，入口円すい部の管軸に死水域が生じる．それを取巻くように
流れる旋回流との間のせん断層は渦面としての性質から巻上がり，
らせん状渦心（vortex core）となって死水域の周りを回転する．
死水域の挙動によって渦心は種々の形態をとるが，強い旋回で死水
域がほぼ吸出し管入口まで拡大すると，らせん状渦心のピッチは伸
び，その規則的な振れまわりが管内で観測される．図9･27は渦心
（渦ロープとも呼ばれる）の模式図である ．非軸対称となる圧力
場も渦心とともに回転するために，圧力は周方向に脈動する．
通常，らせん状渦心内部の圧力は低いのでキャビテーションが生
じている．この空洞部の体積が増えると吸出し管内流れの固有振動
数は低下するので，渦心の振れまわり周波数と一致して共振状態に
陥り，フット部内水流の管軸方向の大きな動揺を伴う，ドラフトサ
流路の設計では一次元流れの仮定が使用されるものの，半径方向
ージと呼ばれる激しい水圧脈動が発生することがある．
水車流量が増えた過負荷状態では，ランナ回転方向と逆の旋回流
になるが，軸方向速度は管軸付近が大きい分布となるために部分負
荷とは性質の異なる脈動現象が観測される．
なお，図9･27と同種の渦心の振れまわりは，順流と逆流が見ら
れる管内旋回流でしばしば観測される．
