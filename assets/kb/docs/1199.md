# 9・4・4 吸出し管

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ，ノズルなどの内部流れ

ディフューザに入る流れの速度が音速に近づき，さらに音速を超
えるに伴って，ディフューザが正常に作動する流入角の範囲は狭く
なる．特に，超音速流入の場合には羽根直前に通路を横切る衝撃波
が生じて側壁境界層が急激に発達するため，スロートの有効断面積
は狭まる．ある程度の圧力回復が入口半開放部で達成されれば，衝
撃波直前のマッハ数は通路幅にわたって変化する．衝撃波直後の断
には，次に述べる安定性能のほうが重視される．
面では，衝撃波に沿う圧力こう配は衝撃波直前の圧力こう配と逆向
きになるために，側壁境界層内には複雑な二次流れが誘起される．
