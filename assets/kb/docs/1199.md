# 2・4・3 可逆過程と不可逆過程

> **章**: 第2章 熱力学
> **節**: 2・4 熱力学の第二法則

実際のほとんどの現象において,熱力学的変数は時間や空間座標の関数である.しかし熱力学では,時間の概念は陽にはでてこないし,原則として,物質は空間的に均質一様であると考える.このような熱力学の枠組では,理想的な状態変化として準静的過程(quasistatic process),すなわち,近似的に平衡を保ちうるよう極めてゆっくり変化する過程を考える.準静的過程は,次に説明する可逆過程(reversible process)と実質的には等価である.可逆過程は,ある系の状態を最初の状態に戻すとき,周囲(系の外界)も含めて完全に最初の状態に戻すことができる過程と定義される.例えば,一定温度の熱源に接触したピストンを無限にゆっくり往復する準静的過程では,方向の逆転により系および周囲を元の状態に戻すことができるので,可逆過程である.これに対し,有限の速度で膨張と圧縮を繰返すピストンでは,たとえピストン運動に伴う摩擦が全くないとしても,作動物質内に生じる温度・圧力の非一様性のため,熱と仕事の間の完全な相互変換はなされない.このような場合は,ピストンの動きを出発時の状態に向けて逆転したとしても完全には元の状態には戻らないので,不可逆過程(irreversible proc-ess)と呼ぶ.実際の多くの現象は不可逆過程である.不可逆過程となる原因としては,空間的に温度分布があるために生じる伝熱,摩擦による力学エネルギーから熱エネルギーへの変換など,種々挙げられる.
