# 2・4・4 カルノーサイクル

> **章**: 第2章 熱力学
> **節**: 2・4 熱力学の第二法則

高温熱源の温度をとし、低温熱源の温度をとするとき、この温度差の間で作動する最も基礎的な熱機関のサイクルとして、カルノーが考えたカルノーサイクル(Carnot cycle)がある。カルノーサイクルは、気体の準静的な等温(定温)膨張、断熱膨張、準静的な等温(定温)圧縮、断熱圧縮の四つの可逆過程から成る。可逆過程で構成されるカルノーサイクルは、当然のことながら可逆サイクルである。カルノー(Carnot, N.L.S.)は、当時広まっていた熱素の概念に立脚して考察したものの、以下のカルノーの原理(Carnot’s theorem)と呼ばれる正しい結論に到達した。「二つの温度,(>)の熱源の間に働く可逆機関の熱効率は、作動物質や変化過程によらず等しく、したがって熱源の温度のみの関数である。また、任意の不可逆機関の熱効率は、この可逆機関の熱効率よりも必ず小さい」。カルノーサイクルに代表される可逆機関の熱効率をη、また任意の不可逆機関の熱効率をηとすると、2・下のように表される。η=-=,=-=1-, η≦η(2・21)
