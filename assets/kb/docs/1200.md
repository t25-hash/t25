# 9・4・5 円形ノズル

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ，ノズルなどの内部流れ

圧力を降下させて流速を高める通路をノズルというが，円形ノズ
ル（circular nozzle）は反動水車やラジアルタービンで使用され，
高速の旋回流を羽根車に導いて所定の回転力を与えるようにその外
側に配置される．水車では案内羽根（guide vane，wicket gate）
と通常呼ばれる．ノズル内は増速流であるから損失は主として壁面
摩擦であり，境界層は薄い．ただし，羽根形状によっては曲率の過
大な箇所で流れが局所的に加速しすぎとなり，その下流で減速域が
現れて損失の増加につながる．また，静止羽根の後流は消滅しにく
いので，内側に置かれた羽根車との干渉も問題になる場合がある．
