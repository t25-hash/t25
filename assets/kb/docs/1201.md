# 9・4・6 スクロールノズル

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ，ノズルなどの内部流れ

スクロールノズル（scroll nozzle ）は導管からの流れを加速し，
旋回流にして羽根車の外周に一様に分配する，ラジアルタービンの
渦巻室である．反動水車では渦巻ケーシング（spiral casing）とも
呼ばれる．その渦巻形状は図9･26のスクロールと同様な方法で設
計できるものの，式(9･66)に従って をθに比例させる方法
が通常利用される．式(9･66)は基準半径位置の円周上で流れ角を周
方向に一定とする条件に基づくものである．
＝
2πθ (9･66)
ここで， ：流量，＝ ：定数，＝ d： の中
心半径．圧縮性が無視できない場合には，密度比を式(9･66)に考慮
すればよい．スクロールノズルの場合には，内部流れは自由渦とみ
なせるから，羽根車外周の軸対称な流れ状態が流量変化による直接
的な影響を受けることはない．
円形ノズルを介して羽根車に流れを送り込む構造の場合には，流
路断面積を大きくとって壁面摩擦損失を小さくしうる．小型のター
ビンではスクロールノズル内で十分に加速して羽根車に旋回流を直
接導く方式をとることが多い．この場合摩擦損失が大きくなるの
で，壁面摩擦が低減されるような断面形状の選定が望まれる．
半径方向への圧力こう配はやはり大きいので強い二次流れが起き
やすく，流出角が特に小さな場合には，壁面近傍の内向き流量が増
加してノズル出口部の流路中央で逆流状態になることもある．
