# 9・4・7 中間案内通路

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ，ノズルなどの内部流れ

多段遠心圧縮機や多段遠心ポンプで使用される中間案内通路は，
ディフューザからの旋回流を外向きから内向きに変えるU字形通
路と，内向き旋回流を軸方向に向け所定の流れにして次段羽根車に
導く戻り通路（return channel ）から構成される．
U字形通路は子午面内流路の曲率と流れの旋回により外壁面で
圧力が高く内壁面で低くなるために，二次流れが生じて低エネルギ
ー流体が内壁側に集積する性質をもつ．それが成長するとはく離が
生じて損失増につながるので，内壁面の曲率半径はできるだけ大き
くとりたい．U字形通路に入る流れの旋回が小さくなるようなデ
ィフューザ形状にしておくことも望まれる．
内向き旋回流は半径位置が小さくなるほど周方向速度成分が大き
くなる性質を有するから，U字形通路からの流れの旋回をとるた
めに戻り通路内には戻り羽根が置かれる．その設計では，羽根前半
部は緩やかな曲面として羽根に作用する負荷を軽減し，後半部を大
きく曲げて半径方向に流れを導くような形状が採用される．図9･
29はその一例である．流路断面積は内側に向かって狭まるので設
計しやすいといわれるが，二次流れの発達が著しくて所定の流れが
得がたい場合もある．流れの案内が主であるものの，上流に置かれ
たディフューザで十分に減速されていない場合にはこの通路で圧力
回復を図る必要が出てくる．
高性能化のために，ディフューザから戻り流路までを一体に見て
三次元羽根を使って流れを積極的に導く設計も利用されることがあ
る．内部の流れは三次元性が強く，低エネルギー流体が二次流れで
集積し巻上がって通路から流出する場合があるから，次段羽根車へ
の影響に注意を払う必要がある．
