# 2・4・6 エントロピー

> **章**: 第2章 熱力学
> **節**: 4・5 項で述べる熱力学温度,を用いて、カルノーの原理は以

準静的に熱が出入りすることに対応して変化する状態量として、すなわち(示強変数である温度を介して)示量変数である熱と関係付けられる状態量として、エントロピー(entropy,J/K)が、クラウジウス(Clausius)により導入された。以下では、系に流入する熱量を正、系から流出する熱量を負で表す。等号はすべての熱の授受が可逆の場合にのみ成り立つ。式(2・25)をクラウジウスの不等式(Clausius inequal-ity)という。式(2・25)で等号が成立する可逆過程を考える。式(2・27)はが状態量であることを表しており、はエントロピーと呼ばれる。
