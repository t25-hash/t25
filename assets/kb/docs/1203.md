# 9・4・8 その他の固定流路

> **章**: 第9章 流体機械の流れ
> **節**: 9・4 ディフューザ，ノズルなどの内部流れ

被動機では，吸込み管からの流れを羽根車に導く入口流路が置か
れる．加速流れにして目玉部で一様な流れとなるように，先細ノズ
ルからなる流路が単段の場合には使用される．多段になると構造上
の都合から吸込み管が軸と直交するように配置されるので，スクロ
ールノズルと同様の吸込みケーシング（suction casing）が利用さ
れ，流れは半径方向から軸方向に導かれる．予旋回を調節するため
に入口案内羽根（inlet guide vane）が流路内に置かれることもあ
る．この種の通路は基本的には増速流れであるが，流路壁が曲面と
なるために減速流域とそれに伴うはく離が局所的に現れる場合があ
るので，その曲率の選定には注意を要する．
