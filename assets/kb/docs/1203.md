# 2・4・6 エントロピー

> **章**: 第2章 熱力学
> **節**: 4・5 項で述べる熱力学温度,を用いて、カルノーの原理は以

準静的に熱が出入りすることに対応して変化する状態量として、すなわち(示強変数である温度を介して)示量変数である熱と関係付けられる状態量として、エントロピー(entropy,J/K)が、クラウジウス(Clausius)により導入された。以下では、系に流入する熱量を正、系から流出する熱量を負で表す。熱機関の熱効率ηは式(2・18)と式(2・21)からη=+≦η=1-(2・22)となり、ゆえに+≦0(2・23)となる[可逆機関では系の温度が一様で、そのときに接している熱源の温度とも等しいので、式(2・23)で等号が成り立つ].これを多くの熱源がある場合に拡張すると、ある温度の熱機関が温度욢욓の外部熱源と熱욢の授受を行い、一つのサイクルを終えて初期状態に戻るとすると、욢욢욓≦0(2・24)∑と表される。等号はすべての熱の授受が可逆の場合にのみ成り立つ。これを連続的にした極限は、周回積分を用いてd욓≦0(2・25)と表される。式(2・25)をクラウジウスの不等式(Clausius inequal-ity)という。式(2・25)で等号が成立する可逆過程を考える。可逆過程の場合は外部熱源の温度욓と系の温度が等しいので、d≡d(2・26)なる量を定義するとd≡d=0(2・27)と書ける。式(2・27)はが状態量であることを表しており、はエントロピーと呼ばれる。
