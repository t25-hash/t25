# 2・4・7 不可逆過程とエントロピー

> **章**: 第2章 熱力学
> **節**: 4・5 項で述べる熱力学温度,を用いて、カルノーの原理は以

式(2・25)を任意の熱力学サイクルに適用してみる。図2・3に示すように、状態A から状態B まで一般の不可逆変化IR をした後、状態B から状態A まで可逆変化R で戻る場合を考える。エントロピーは状態量であるから、状態A と状態B の間でどのように変化しても、エントロピーの変化量(差)は同一である。このときのサイクルにクラウジウスの不等式を適用する。右辺は外部熱源から出ていったエントロピーであり、換言すれば系に入り込んだエントロピーとみなすことができる。したがって、不可逆変化のとき、系に入り込んだエントロピーと系内で増加したエントロピーとを比べると、後者のほうが大きい。この意味で、不可逆変化に伴って系のエントロピーは増加するという。式(2・30)と式(2・31)は、熱力学の第二法則を定式化したものである。
