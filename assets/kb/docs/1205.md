# 9・6・2 ラビリンスシール内の流れパターン

> **章**: 第9章 流体機械の流れ
> **節**: 9・6 ラビリンスの流れ

ラビリンスにおいて，作動流体は概略以下のように流れている．
まず，絞り片に接近した流れが絞り片先端で縮流し，膨張室（キャ
ビティ）へ壁面噴流として流れ込む．縮流に伴う流速の増加によ
り，絞り片先端付近では圧力が急激に低下する．この流れは，膨張
室上部において拡散（ほとんどの場合，乱流拡散）によって広が
り，同時に膨張室下部に再循環渦を形成する．作動流体が圧縮性の
場合には，拡散による混合と同時に膨張も生じる．このとき，速度
の低下と圧力の回復が生じる．ラビリンスでは絞り片を複数設ける
のが普通であり，したがって，これら一連の過程が絞り片ごとに繰
返されることになる．図9･36に直通型ラビリンスにおける流れパ
ターンと直線壁に沿った圧力分布を示す．圧力損失のほとんどが絞
り片通過時に生じていることがわかる．
ラビリンス流れに対する幾何形状の影響として，(１)絞り片の数
が多いほど，(２)すきまが小さいほど，(３)絞り片先端が鋭くとが
っているほど，(４)絞り片ピッチ/膨張室深さが約0.25のとき，漏
れが減少することが知られている ．
軸回転数が漏れに及ぼす影響は，作動流体が低速（すきまに基づ
くレイノルズ数が10000程度以下）の場合にはほとんど生じない．
一方，作動流体が高速の場合，漏れ量が静止時より10%程度低減
することが知られている．なお，軸振動等のロータダイナミクスに
対しては，作動流体の速度にかかわらず軸回転数の影響は重要であ
る．
