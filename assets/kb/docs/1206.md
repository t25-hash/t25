# 9・6・3 ラビリンスシールの設計

> **章**: 第9章 流体機械の流れ
> **節**: 9・6 ラビリンスの流れ

ラビリンスシールの基本設計は，作動流体が圧縮性の場合には理
想ラビリンス［複合直通型，図9･33(ｃ)］に対する理論解析に基
づいて，非圧縮性の場合には連続する流体抵抗を有する流路として
行われる．いずれも実際の形状に対応させるために，修正係数を導
入して補正を加える半経験的な設計方法となっている．図9･33に
示したような基本的な形態に関しては豊富な実験データが整備され
ており，実用上十分な精度で設計を行うことができる ．ただ
し，回転による弾性変形（遠心力効果）および温度変化に伴う熱膨
張によってすきまが変化するため，これらの効果による変形量を別
途求めておき，稼働時のすきまを用いて設計を行うことが必要であ
る．
最近のラビリンスシールでは，漏れ性能を向上するために絞り片
や段差の形状を複雑化・最適化することが行われている．また，ガ
スタービンのような場合には，内部構造の冷却に漏れ流れが利用さ
れており，漏れを低減しつつ冷却に必要な流量と温度を同時に満足
(ａ)軸流ラビリンス
図9･35回転軸との相対関係による名称
することが求められている．このため，コンピュータシミュレーシ
ョン（数値流体力学）によって多数の形状に対する性能を比較検討
して，その中から最適なものを選択し，最終的に実験を行って性能
を確認するという設計方法がとられるようになりつつある．例え
ば，ロード（Rhode）ら は，ステップラビリンスシールを対象
として標準型 -ε乱流モデルを組込んだプログラムを開発し，絞
(ｂ)半径流ラビリンス
り片高さ，ステップ幅，すきま幅などをパラメータとしたコンピュ
ータシミュレーションを実施し，各パラメータの最適化を図ってい
る．この研究の結果，彼らは，検証実験により流れ場に対する計算
誤差がおおむね7%以内であることを確認し，初期形状に対して約
60%の漏れ流量の低減を実現している．また，ジマーマン（Zim-
merman）ら は，摩耗による絞り片先端形状の変化が直通型ラ
ビリンスの性能に及ぼす影響を評価している．
