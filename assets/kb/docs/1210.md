# 2・6・5 ジュール・トムソン効果

> **章**: 第2章 熱力学
> **節**: 2・6 熱力学の一般関係式

流体の絞り現象において流体の運動エネルギー等が無視できる場合,エンタルピーは一定であるが,圧力の降下に伴い一般に温度の変化が生じる.この現象はジュール・トムソン効果(Joule-Thomson effect)と呼ばれ,ジュール・トムソン係数(Joule-Thomson coefficient)μ=옿옿욠(KPa)によって評価される.μ=옿옿욠=-1욣-옿옿욣=-1욣옿옿(2・71)-옿옿욣=0が成り立つ温度を逆転温度(temperature ofinversion)という.流体が絞り膨張とともに温度が上がるか下がるかは-옿옿욣の正負に対応し,温度が下がるのは逆転温度以下である.各物質の逆転温度の最高値の概略が表2・2に示されるが,Ne,H,Heは最高逆転温度が常温より低い気体である.なお,理想気体ではつねにμ=0である.
