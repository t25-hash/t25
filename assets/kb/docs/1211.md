# 2・7・1 純粋物質の相と関係

> **章**: 第2章 熱力学
> **節**: 2・7 気体・液体・固体・超臨界流体

1種類の分子から構成され一定の化学組成をもつ物質を純粋物質(pure substance)と呼び、二つ以上の純粋物質から成る混合物(mixture)と区別する。このような単一の組成をもつすべての純粋物質は、液体、気体または固体のどの状態においても存在しうるし、固体はいろいろな同素変態のもとで存在しうる。物質の三つの状態は一般に相(phase)と呼ばれ、単一の相から成る物質の状態は均質であると考えられるのに対して、二つ以上の相すなわち共存相から成る系の状態は不均質であるという。純粋物質で認められる共存相の数は最大3相までであり、4相またはそれ以上の共存相が存在する不均質な系は、混合物の場合にのみ認められる。共存しうる相の数と、系を構成する純粋成分物質数との間には、相律と呼ばれる式(2・3)の関係が成り立つ。純粋物質の任意の単一相における熱力学的状態は、式(2・3)からも明らかなように、二つの独立な示強性状態量、例えば圧力と温度とによって決定できることがわかる。純粋物質の圧力・体積(比体積またはその逆数で定義される密度)・温度の関係は関係と呼ばれ、系の熱力学的諸性質を求めるうえで最も重要かつ基本的な関係である。したがって、上述した相律から、,,の状態量のうち任意の二つを独立変数に選べば、残りの状態量は両者の関係として相関される状態式で表現でき、状態曲面(state surface)が定められる。図2・4は通常の純粋物質に対する状態曲面を表したものであり、図2・5には面への状態曲面の投影図を示した。これらの図中には、単独に存在しうる3相、2相が安定に共存しうる共存相のほかに、気・液・固相の3相が共存しうる唯一の状態点である三重点(図2・4,2・5では三重点はA′AB のように三重点の線として現れる)についても示してある。各相間の境界としては、三重点の線のほかに、A′D の融解曲線、AEの凝固曲線、AFCの飽和液線、BGCの飽和蒸気線およびOA′の昇華曲線があり、これらの相境界線を横切る状態変化では、比体積や屈折率などの性質が不連続に変化する。また、これらの相境界線の中間で2相が共存する状態では、示強性状態量とが不変のままであり、相変化の過程で比体積のみが変化する(図2・4、でき、2相共存状態ではまたはのいずれかが与えられれば他の状態量が定まることになる。また、融解・凝固・蒸発・凝縮・昇華といった相変化においては、系のギブス関数が一定不変であることも知られている。図2・6は図2・4の面への投影図であり、融解・凝固曲線は融解曲線tM に、飽和液線・飽和蒸気線は蒸気圧曲線tCにそれぞれ重なり、昇華曲線tS が固・気両相の共存状態を表すことがわかる。また、図2・6中破線tN で示したように、融解曲線が負の傾きをもつ特異な純粋物質(代表例として水、ほかにもビスマス、アンチモン、ガリウムなど)も存在する。図2・4 通常の純粋物質に対する状態曲面5―23
