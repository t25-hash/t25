# 2・9・1 空気および燃焼ガスの性質と状態式

> **章**: 第2章 熱力学
> **節**: 2・9 空気などの気体

空気(air)の密度や比熱、音速などの信頼できる熱力学性質は、高精度な熱力学状態方程式から計算する必要がある。空気(乾燥)および純粋な窒素、酸素、アルゴンの熱力学状態方程式はアメリカ商務省技術開発研究所(NIST)のレモン(Lemmon)らにより開発されている。彼らの報告は55ページの大著であり、状態式の説明のほか計算結果が数表で示されている。彼らの報告によれば、空気の組成はモル組成で窒素78.10%、酸素20.95%、アルゴン0.92%、二酸化炭素0.03%であるが、彼らの状態式では二酸化炭素の影響を省略しており、状態式の空気組成はモル組成で窒素78.12%、酸素20.96%、アルゴン0.92%となる。状態式の有効範囲は、60~2000K および2GPaまでの温度および圧力範囲の気体、液体、超臨界流体状態を含む全流体域である。実測値情報が得られない873K および70MPaを超える高温および高圧域では、窒素の実測値を用いた推算を行っている。また、衝撃波の実測値情報も考慮して開発されていることから、5000K および28GPaまでの推算にも使える状態式であると紹介されている。873K および70MPaまでの流体域において計算される熱力学性質値の不確かさは、密度で0.1%、音速で0.2%、比熱で1%、その範囲を超えた流体域では、2000K,2GPaまでの密度計算値で0.5~1%であるとしている。そして、計算される露点および沸点圧力値の不確かさは1%以内であるとしている。おもな乾燥空気の物理量を表2・6に、そして、空気の露点曲線上の熱力学性質を表2・7に、標準大気圧力下の熱力学性質を表2・8に示す。表2・7にある最高凝縮温度は、図2・7に示すとおり、混合流体としての空気の臨界点付近における沸点曲線、および凝縮曲線の特徴的な状態点である。湿り空気に関しては、ベッカー(Buecker)らが比熱およびエンタルピーについて、200~3300K
