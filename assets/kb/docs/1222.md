# 9・10・1 特徴

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・10 統一型非弾性構成式

一般にひずみ速度が大きいと流動応力も大きくなるが，ひずみ速
度が非常に大きな高速変形や高温でのクリープを含む変形では，速
度依存性を考慮した構成関係のモデリングが不可欠となる．高温で
の変形について，従来は塑性変形とクリープ変形を別々の構成式か
ら導き，それぞれのひずみの和をとってきたが，このような構成式
を重ね合わせ型構成式モデル（superposition model ）という．一
方で，塑性変形とクリープ変形との相互作用が顕著に現れ変形挙動
が複雑になるとき，速度に依存しない塑性ひずみと粘性によるひず
みを区別して取扱うのではなく，粘塑性ひずみ（あるいは非弾性ひ
ずみ）として一括して扱うほうが有利となる場合がある．このよう
な考え方に基づいてモデル化したものを統一型非弾性構成式
（unified inelastic constitutive equation）という．現象論的な粘塑
性変形挙動のモデリングや熱力学的考察はマルバーン やペジュ
ナ らの研究に始まり，内部状態変数を導入することによって，
塑性-クリープ相互作用や繰返し変形に関して精緻なモデリングが
試みられた．現在では，統一型非弾性構成式に関する成書 も
出版されており，複雑な統一型非弾性構成式が実際の構造解析に供
されるようになっている．
統一型非弾性構成式のモデリングに当たっては，静的な応力-ひ
ずみ関係を仮定したうえで速度依存性を表すパラメータと法線法則
に代表される塑性流動則を適用する．すなわち，非弾性ひずみ速度
は一般に
＝Φ, (9･111)
と表される．ここでΦは，応力 と内部状態変数 （スカラまた
はテンソル）による適当な関数である． にはミーゼス型の相当
応力を用いることがほとんどである．内部状態変数は以前の負荷履
歴と変形速度に依存した硬軟化現象を表すためのもので，塑性構成
式での背応力のように応力と同じ単位をもつ場合もあれば，微視的
な変形メカニズムを考慮した転位密度などが選ばれる場合もある．
例えば にべき乗の関数を適用し，塑性構成式での背応力 と
静的な変形での降伏応力 および適当な定数 を導入する と，
＝ － (9･112)
とできる．まず，背応力や降伏応力を無視して考えると，上式はノ

たがって，背応力や降伏応力が一定値を取るとき，ノルトン則に類

応力-ひずみ関係に近づいていく．したがって，モデルに現れるパ
ラメータや内部状態変数を適切に設定することによって，速度依存
性のない狭義の塑性変形からクリープ変形に至る広い範囲のひずみ
速度に適用できる構成式モデルとなることが期待できる．
