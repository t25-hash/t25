能な状態変数（例えば応力や温度）だけで変形挙動を表現すること
はできない．このために，テンソルあるいはスカラ量の内部状態変
数（または内部変数，internal state variable）を導入する．内部
変数は内部構造を表現するものであって，金属材料の非弾性変形に
限れば，転位や欠陥などの微視的な構造変化が巨視的な応力-ひず
