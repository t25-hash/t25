# 2・10・4 その他の実在流体の性質

> **章**: 第2章 熱力学
> **節**: 2・10 相変化を伴う実在流体

その他の実在流体の状態式に関する情報を以下に紹介する。シュパン・ワグナーは、技術開発に有用な多くの流体に熱力学状態式を開発している。水やメタンや二酸化炭素などの国際標準的な基礎状態式のように精密なものではないが、2003年現在で最も高度な状態式開発アルゴリズムの一つによって開発されており、実用に際しては十分な信頼性をもっている。さらに、異なる物質の全流体域を12項の剰余項からなる共通のヘルムホルツ関数で表すことができ、流体熱物性の利用者にとっては大変便利な状態式である。対象としている流体は、メタン、エタン、プロパン、イソブタン、ノルマルブタン、ノルマルペンタン、ノルマルヘキサン、ノルマルヘプタン、ノルマルオクタン、アルゴン、酸素、窒素、エチレン、シクロヘキサン、6フッ化硫黄、R 11,R 12,R 22,R 32,R 113,R 123,R 125,R 134a,R 143a,R 152a、二酸化炭素、そして、アンモニアである。
