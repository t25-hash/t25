# 2・11・3 湿り空気の状態変化

> **章**: 第2章 熱力学
> **節**: 2・11 湿り空気の性質と状態変化

湿り空気-線図(図2・8)はとを斜交軸にとった線図で、空気の状態変化の解析や空調の熱負荷の計算等に用いられる。この関係を利用して空調システム内の空気の状態変化を-線図上に描き、空調熱負荷や機器の容量等を求める。線図によらずコンピュータで解析する方法もある。また、種々の全圧力における湿り空気-線図がある。異種類の分子を混合してできる物質を混合媒体という。2種類の分子から成る混合媒体を2成分混合媒体、3種類の分子のそれを3成分混合媒体という。分子の種類によって分子の大きさ、分子間相互作用が異なるため、混合によって体積・混合エネルギーの増減が発生する。したがって、個々の混合媒体の性質は実験結果に基づいて明らかにする必要がある。混合気体を加圧すると、凝固して固-気間の相転移を生じるが、濃度の異なる2種類の気相(気-気平衡)に分離することもある。2種類以上の純粋物質を混合すると、混合後のエントロピーは、混合前のエントロピーの総和より増大する。したがって、混合気体はそのままでは純粋物質に戻ることはなく不可逆変化である。
