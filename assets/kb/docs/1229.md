# 9・11・5 多結晶材料に対する構成式

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・11 結晶すべりに基づく構成式

多結晶材料の構成式を求める場合は，(１)テーラーの仮定（ひず
み一定) ，(２)ザックスの仮定（応力一定) ，および(３)適
合・平衡条件を満足するセルフコンシステントの仮定（KBWモデ
ル) を採用し，それぞれの結晶に対して最小すべりの原理，等方
硬化則，および最大塑性仕事の原理を採用することで，多結晶体の
構成式を決定することができる．多結晶材料として，つまり結晶集
合体として定義されたマクロなひずみ速度と応力が結晶単位で定義
された分解せん断応力とせん断ひずみ速度が次の関係を満足すると
仮定することで，最終的に応力σを求めることができる．
＝σ＝τ＝τ∑γ
ここで， はテーラー因子，τは降伏せん断応力， は相当塑
性ひずみ速度である ．
