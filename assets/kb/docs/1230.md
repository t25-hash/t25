# 9・12・1 定義

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・12 損傷変数

材料，あるいは負荷条件により材料内部に生じる分布空げきの形
状，分布，力学的効果は異なるが，このような損傷状態を表現する
ためにスカラ，ベクトル，各階のテンソルなどが損傷変数として用
いられる ．
材料の損傷が球形の空洞のように形状に方向性がなく，その分布
も等方である場合，あるいは，分布空げきの個々の形状，局所的な
分布には方向性があっても大域的特性が等方とみなせる場合など，
等方損傷（isotropic damage）に対して損傷変数はスカラで定義
される ．
ここで，スカラで定義される損傷変数を とするとき，その値
は次のように定義される ．
0≦≦1 (9･134)
( ＝0：初期非損傷状態，＝1：最終的破壊状態）
ただし，実際の材料では1より小さな限界値 に達したとき破
壊する場合が多い ．
この損傷変数 は，損傷に伴う分布空げきの存在による有効面
積の減少率と解釈することができる ．図9･32のように丸棒の引
張りを考えるとき，実質的に荷重を負担する丸棒の断面積は，分布
空げきの存在により，丸棒の実際の断面積 よりも小さな有効面
積 となる．ただし，有効面積 には，断面に存在する分布空
げきの総断面積分の面積減少だけではなく，分布空げきの存在によ
る応力集中の効果も含まれる ．このとき損傷変数 は，次式
