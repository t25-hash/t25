# 2・13・1 サイクルと熱効率、成績係数

> **章**: 第2章 熱力学
> **節**: 2・13 各種サイクル

熱を継続的に仕事に変換する熱機関、あるいはその逆に外部から仕事を与えて低温空間から高温空間へ熱を移動させる冷凍機やヒートポンプにおいて、作動媒体は一連の状態変化を経て初めの状態に戻るサイクル(cycle)を構成する。熱機関のサイクル[図2・12(a)]において、熱源から作動媒体が受取る熱量を、周囲へ放出する熱量をとすると、得られる仕事は=-(2・158)となる。したがって、サイクルの熱効率はη==1-(2・159)で与えられる。一方、冷凍機やヒートポンプのサイクル[図2・12(b)]では、式(2・158)のW は外部から与えられる仕事、は高温熱源が受取る熱量、は低温熱源から取り去られる熱量である。冷凍機の場合は高温熱源が周囲であり、ヒートポンプの場合は低温熱源が周囲となる。このようなサイクルの性能は、次式で定義される成績係数により評価される。冷凍機:COP윒=(2・160)ヒートポンプ:COP=(2・161)冷凍機とヒートポンプの成績係数の間には次の関係がある。COP=COP윒+1(2・162)サイクルを評価するには、可逆サイクルを考えると理論的取扱いが容易になる。可逆サイクルは図2・13に示すように-線図上で閉曲線を描く。その閉曲線で囲まれた部分の面積が1サイクルの仕事量となる。熱機関のサイクルの場合、線図上で経路は時計回りに一周し[図2・13(a)]、ヒートポンプや冷凍機のサイクルの経路は反時計回り[図2・13(b)]となる。(b) 冷凍機およびヒートポンプ(a) 熱機関図2・12 二つの熱源の間で動作するサイクル5―36熱工学
