# 2・13・2 カルノーサイクル、スターリングサイクル、エリクソ

> **章**: 第2章 熱力学
> **節**: 2・13 各種サイクル

ンサイクルこれら三つのサイクルは、いずれも等温過程で受熱と冷却を行う。スターリングサイクルとエリクソンサイクルで熱交換器により再生を行った場合は、カルノーサイクルと理論熱効率が等しくなる。a. カルノーサイクル(Carnot cycle)カルノーサイクルはカルノー(Carnot, N.L.S.)により提案されたサイクルであり、図化から成る。熱機関として動作する場合は、a→b:等エントロピー圧縮、b→c:等温膨張(受熱)、c→d:等エントロピー膨張、d→a:等温圧縮(冷却)の各過程を経て、サイクルを完了する。ab間の圧縮仕事とcd間の膨張仕事は大きさが同じであり、1サイクルの間に相殺される。したがって、カルノーサイクルは、bc間の等温過程により受取った熱のすべてをいったん仕事に変換し、その仕事の一部はda間の等温圧縮で熱に逆変換して低温熱源に捨て、残りを外部へ仕事として取出す熱機関であると表現することができる。このサイクルを冷凍機およびヒートポンプに適用すると、成績係数はそれぞれ次式で与えられる。熱機関の場合は、a→b:等積加熱、b→c:等温膨張(受熱)、c→d:等積冷却、d→a:等温圧縮(冷却)、の過程を経て、サイクルを完了する。c. エリクソンサイクル(Ericsson cycle)エリクソンサイクルは、図2・16に示すように二つの可逆等温変化と二つの可逆等圧変化で構成されている。このサイクルの場合もスターリングサイクルと同じように、熱交換器による再生を行えば、作動流体が理想気体の場合カルノーサイクルと同じ熱効率になる。
