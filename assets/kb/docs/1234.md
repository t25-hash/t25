# 2・13・3 内燃機関の理想サイクル

> **章**: 第2章 熱力学
> **節**: 2・14 に示すように二つの可逆等温変化と二つの等エントロピー変

内燃機関の基本サイクルとして代表的なものは、オットーサイクル、ディーゼルサイクル、サバテサイクルの三つが挙げられる。これらのサイクルは往復動機関として実用化されており、吸気および排気を行うが、閉じた系の空気標準サイクルとして取扱うと解析が容易である。最近は吸気後の圧縮を等圧下で行うことにより、膨張仕事のストロークを稼いで熱効率を向上させたミラーサイクルが、自動車用エンジンとして実用化されている。また、サイクルの性能を評価するには、1サイクルで発生する仕事を作動流体の有効体積で割った値、すなわち平均有効圧力(mean effective pressure)が用いられる。ここでは、いずれのサイクルにおいても作動流体1kg 当たりの空気標準サイクルとして考える。図2・17に示すように、a→b:等エントロピー圧縮、b→c:等積加熱、c→d:等エントロピー膨張、d→a:等積冷却の各過程を経て、サイクルを完了する。b. ディーゼルサイクル(Diesel cycle)ディーゼルサイクルは加熱過程が等圧下で行われるので、等圧サイクルとも呼ばれる。図2・19に示すように、a→b:等エントロピー圧縮、b→c:等圧加熱、c→d:等エントロピー膨張、d→a:等積冷却の各過程を経て、サイクルを完了する。熱効率および平均有効圧力は、圧縮比および比熱比が大きくなると、あるいは締切比が小さくなると増加する(図2・18)。c. サバテサイクル(Sabathecycle)サバテサイクルは、図部分を可逆等圧加熱するサイクルで、合成サイクルあるいは二重燃焼サイクル(dual cycle)ともいう。a→b:等エントロピー圧縮、b→c′:等積加熱、c′→c:等圧加熱、c→d:等エントロピー膨張、d→a:等積冷却の各過程を経て、サイクルを完了する。熱効率および平均有効圧力は、ε,φが大きくなると、またσが減少すると増加する(図2・18)。式(2・176)と式(2・177)においてσ=1とすればオットーサイクル、φ=1とすればディーゼルサイクルの式と同じになる。d. ミラーサイクル(Miller cycle)図2・18に示したように熱効率は圧縮比の増加とともに増大するが、実際にはノッキング(異常燃焼)が発生するために上限がある。ミラーサイクルでは圧縮過程の途中まで給気バルブを開けたままにして、流入する混合気の一部を逆流させ、混合気の流入量を減少させる。これにより実質的な圧縮比を小さくし、相対的に膨張比を大きくすることが可能となり、サイクル当たりの発生仕事が増大する。理想化されたミラーサイクルは、図2・21に示すようにa→b:等エントロピー圧縮、b→c:等積加熱、c→d:等エントロピー膨張、d→a:等圧冷却の各過程を経て、サイクルを完了する。
