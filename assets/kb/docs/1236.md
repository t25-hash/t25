# 9・2・4 単結晶および多結晶体の降伏

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

図9･5に示すように，断面積 の単結晶の棒の軸方向に引張応
力σを作用させると，その法線方向が軸よりφだけ傾いたすべり
面上で角度λで示されるすべり方向のせん断応力τは次式で与え
られる．
τ＝σcosφcosλ (9･10)
ここで，τは分解せん断応力（resolved shear stress )，cosφcos
λはシュミット因子（Schmid factor ）と呼ばれる．ある温度です
べりを開始するのに必要な分解せん断応力は材料ごとに決まった値
をもっており，これを臨界分解せん断応力（critical resolved
shear stress ）という．
結晶中で多くのすべり系が可能であれば，引張力を徐々に上げて
いき，いちばん初めに臨界分解せん断応力に達したすべり系が活動
を開始する．これはシュミット因子が最大となるすべり系で，これ
を主すべり系（primary slip system）と呼ぶ．したがって，多結
晶体では分解せん断応力がその臨界値に達した結晶粒から順次塑性
変形が開始することになる．ほとんどすべての結晶粒が降伏し，塑
性変形が巨視的に観察できるようになった時点の応力が材料試験で
決定される多結晶体の降伏強さである．
ところで，結晶粒界は転位の運動に対する障害物となるので，材
料の降伏強さは結晶粒が微細なほど大きくなる．そのことは次のホ
ール・ペッチの式（Hall -Petch relation）で表される．
σ＝σ＋ (9･11)

ある．
