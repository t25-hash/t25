# 9・2・7 変形抵抗の温度およびひずみ速度依存性

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・2 単軸応力下の塑性変形

塑性変形は一般には速度依存性が小さいとされるが，転位の運動
速度は作用する応力に依存するため，室温でもある程度の速度依存
性が見られるのが普通である．転位の運動は本質的に熱活性化過程
（応力作用とともに原子の熱振動エネルギーの助けを借りて，転位
が障害物を乗り越えて進む過程）に支配されるため，変形抵抗は一
般に温度およびひずみ速度に依存する．高温においては粒界すべり
（粘性的な変形挙動を示す）が活発になる場合も多く，ある温度以
上では変形中の回復や再結晶（これらは動的復旧過程といわれる）
も影響するため温度・速度依存性がさらに顕著となる．図9･7 に
は変形抵抗の温度依存性を，図9･8 にはひずみ速度依存性を示す
軟鋼の応力-ひずみ曲線の例を描いてある．
