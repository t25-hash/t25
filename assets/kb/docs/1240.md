# 2・14・1 流れの一般基礎式

> **章**: 第2章 熱力学
> **節**: 2・14 気体の流動

図2・53に示すように断面がゆるやかに変化する管内の気体の一次元定常流れについては、次の諸関係式が成立する。a. 音速、マッハ数理想気体の温度における音速(speedof sound,acoustic velocity)は、ガスの成分が一定のとき次式で表される。b. 全エンタルピー、全温度、全圧力流路のある点の気体の比エンタルピーと運動エネルギー2の和を、全エンタルピー(total enthalpy)またはよどみ点エンタルピー(stagnationenthalpy)という。e. 運動量の式定常流れにおいては、図2・53に示すような検査面を通る気体の運動量の増加は、図2・53内の気体に働く外力の和に等しい。位置エネルギーの変化を無視すると、この流れのエネルギーの式は次式によって表される。式(2・225)(2・226)(2・228)はいずれも定常流れに対するエネルギーの基礎式である。
