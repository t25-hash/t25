# 2・14・2 外部仕事を含まない定常一次元流れ

> **章**: 第2章 熱力学
> **節**: 2・14 気体の流動

これにさらに外部との熱交換がないという条件(d=0)が加えられた流れを、断熱流れ(adiabatic flow)と呼び、ノズル、ディフューザの流れや管内高速流などは、これによって近似される場合が多い。a. 運動量の式外部仕事を含まない流れの運動量の式は、式(2・222)でd=0として求められる。そしてエネルギーの式は次のようになる。摩擦や渦の発生のある断熱流れでは、エネルギーの散逸効果によりエントロピーは増大し、全圧はつねに減少する。等号は摩擦のない流れ(d=0)の場合で、このとき=一定、すなわち等エントロピー(断熱可逆)過程となる。そのような流れを等エントロピー流れという。
