# 2・14・3 等エントロピー流れ(isentropic flow)

> **章**: 第2章 熱力学
> **節**: 2・14 気体の流動

等エントロピー流れでは、全温度と全圧力は場所によらず一定に保たれ、比熱比一定の理想気体では、その状態変化は可逆断熱変化の関係と式(2・216)から次のようになる。すなわち音速流れは、流路の断面積が最小のところでしか実現されない。したがって、先細ノズル(convergent nozzle)では、亜音速流れのとき出口圧力をいくら下げても、あるいは入口圧力をいくら上げても、ノズル内で流速は音速以上にならない。音速流れをさらに加速して超音速流れを得るためには、先細ノズルの先に広がり流路をつないだ末広ノズル(convergent-divergent nozzle)を用いなければならない。同様にして次の諸関係式を得る。
