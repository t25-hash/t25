# 2・14・5 末広ノズルよりの等エントロピー噴流

> **章**: 第2章 熱力学
> **節**: 2・14 気体の流動

ノズル出口で超音速流の噴流を得るためには,図2・57に示すような先細ノズルの先に広がり流路を付けた末広ノズルを用いる必要がある.これはラバルノズル(Laval nozzle)とも呼ばれ,その最小断面積の部分をスロート(throat)という.このようなノズルで理想気体を入口全圧から等エントロピー的に膨張させ,ノズル出口の流速を超音速にしたときの圧力分布は,図2・57の曲線(6)のようになる.すなわち気体はまず先細流路で加速されスロート部で音速に達し,その圧力は臨界圧力욂となる.広がり流路に入った流れはさらに加速し,それに応じて圧力は低下し出口圧力に至る.この場合の出口流速および流量は式(2・266)~(2・271)で与えられる.しかし,スロート部は臨界状態となり流れは閉そくしているので,流量は式(2・275)の断面積をスロート部の断面積욧に変えて,次式により求めることもできる.욂=φ욂욧(2・278)ある形状を与えられた末広ノズルで,入口全体を一定に保ち背圧욄を順次下げていった場合,圧力分布は図2・57に示すように曲線(1)から曲線(7)へと変化する.背圧が高い間はスロート部で圧力最小,速度最大となるが,流れは全域亜音速である[曲線(1),(2)].スロート部が臨界圧力욂に達した後は流れは閉そくし,これ以上背圧を下げても流量は変化しない.スロート部の下流で流れは加速していったん超音速となるが,背圧が曲線(4)の値になるまでは背圧に見合った位置に衝撃波(shock wave)が発生し,圧力は不連続的に上昇した後,流れは再び亜音速に戻る[曲線(3)].背圧の低下とともにこの衝撃波の位置は広がり部を下流に移動し,やがて出口に達する[曲線(4)].一点鎖線は衝激波直後の圧力の衝撃波の移動に対する軌跡を表す.さらに背圧が下がると,ノズル外に衝撃波を伴った流れに変わる[曲線(5)].衝撃波を通る変化は不可逆過程であるので,これを含む流れは等エントロピー流れではない.背圧が曲線(6)の値に達すると,出口圧力と背圧욄とは滑らかにつながる.背圧をこれ以上に下げると流れはノズル内において十分膨張しきれず,ノズルの下流でさらに膨張する[曲線(7)].曲線(5)と曲線(7)の場合,出口圧力と背圧욄とは一致せず,=욄の適正膨張[曲線(6)]を基準として前者(<욄)を過膨張(over-expansion),後者(>욄)を不足膨張(under-expansion)という.
