# 2・14・6 摩擦損失を伴うノズルよりの噴流

> **章**: 第2章 熱力学
> **節**: 2・14 気体の流動

実際ノズル内の流れでは多少の摩擦損失を伴うが、外部との熱交図2・57 末広ノズルと圧力分布図2・56 先細ノズル5―48熱工学換はふつう無視できるので断熱流れの基礎式が適用される。したがって、出口速度および流量は、等エントロピー変化の関係式におけるをポリトロープ指数で置換えれば求まる。よく仕上げたノズルではφ=0.95~0.975(ζ=0.10~0.05)程度である。ここで、は全温度を、はスロート部の断面積を表図2・58 損失を伴うノズル噴流す。また図2・58に示すように、エントロピーの増加により全圧はから′へ低下する。
