# 3・1・1 化学反応

> **章**: 第3章 燃焼
> **節**: 3・1 燃焼の基礎理論

a. 素反応と総括反応化学反応は,反応化学種どうしが衝突によって相互に化学結合の組換えを起こす過程である.燃焼反応においては,何段階もの化学反応が順次進行していくことによって,最終的な燃焼生成物が生じる.この場合に分子,ラジカル,原子などの化学種の間に,実際に生じている個々の化学結合の組換え過程を素反応という.したがって,燃焼反応を精確に表すためには,非常に多数の素反応群を用いなければならないことになる.一方,複雑な化学反応を便宜上,いくつかの代表的な化学種の間の変化であるように表したものを総括反応(overall reaction)という.総括反応は本来,関与している個々の素反応を,その寄与度を考慮したうえで加え合わせることによって得られるが,燃焼においては,実際には形式的に燃料と酸化剤が燃焼生成物に変わるという形で表される.燃料CHO욨の完全燃焼を表す総括反応はCHO욨++4-2 OCO+2HO(3・1)となる.b. 反応式化学反応はνA+νB+... coefficient)である.ν,ν,...はそれぞれ反応系の化学種A,B,...についての反応次数であって,ν=ν+ν+...を全反応次数という.反応(3・2)が素反応である場合には,νは反応系に含まれる分子数に等しくなり,その値が1,2,3となるに従ってその素反応を単分子反応,2分子反応,3分子反応という.以下便宜上反応(3・2)を욇ν욡C욡∑욈ν욢C욢(3・3)∑と表すことにする.c.
