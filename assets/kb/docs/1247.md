# 3・1・1 化学反応

> **章**: 第3章 燃焼
> **節**: 3・1 燃焼の基礎理論

a. 素反応と総括反応化学反応は、反応化学種どうしが衝突によって相互に化学結合の組換えを起こす過程である。燃焼反応においては、何段階もの化学反応が順次進行していくことによって、最終的な燃焼生成物が生じる。この場合に分子、ラジカル、原子などの化学種の間に、実際に生じている個々の化学結合の組換え過程を素反応という。したがって、燃焼反応を精確に表すためには、非常に多数の素反応群を用いなければならないことになる。一方、複雑な化学反応を便宜上、いくつかの代表的な化学種の間の変化であるように表したものを総括反応(overall reaction)という。総括反応は本来、関与している個々の素反応を、その寄与度を考慮したうえで加え合わせることによって得られるが、燃焼においては、実際には形式的に燃料と酸化剤が燃焼生成物に変わるという形で表される。ここにA,B,...を反応物(reactant)、P, Q,...を生成物(product)といい、左辺を反応系、右辺を生成系という。ν,ν,...,ν,ν,...は化学量論係数(stoichiometric coefficient)である。ν,ν,...はそれぞれ反応系の化学種A,B,...についての反応次数であって、ν=ν+ν+...を全反応次数という。反応(3・2)が素反応である場合には、νは反応系に含まれる分子数に等しくなり、その値が1,2,3となるに従ってその素反応を単分子反応、2分子反応、3分子反応という。αは頻度因子の温度依存性を表しているが、一般にはα=0であって、頻度因子は定数であることが多い。反応速度定数の単位は、その反応が何分子反応かによって決まり、1分子反応ではs,2分子反応ではcmmolsなどとなる。個々の素反応の反応速度定数は、文献(1)~(4)などにまとめられている。これに対して反応式(3・2)を正反応(forward reac-tion)といって区別することがある。反応(3・2)が進行していくにつれて、正反応の反応速度は小さくなり、逆反応の反応速度は大きくなっていく。この状態になると反応(3・2)の正味の速度は零となって、実質的に反応は停止してしまう。この状態を化学平衡(chemical equilibrium)という。は濃度単位で表した平衡定数(equilib-rium constant)である。νは化学反応式における生成系と反応系における化学種の化学量論係数の差∑ν-∑νである。また、は標準状態における圧力であって、1atm(または10Pa)である。化学種の標準化学ポテンシャルは、基準物質からそれぞれの化学種1molを標準状態で生成するときに要するギブス自由エネルギー変化として与えられる。燃焼反応に関与する代表的な元素に対する基準物質は、Cについてはグラファイト、H についてはH,OについてはO,N についてはNなどである。
