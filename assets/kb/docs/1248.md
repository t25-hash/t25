# 3・1・2 量論計算

> **章**: 第3章 燃焼
> **節**: 3・1 燃焼の基礎理論

燃焼における量論計算は、主として燃焼条件を表す目的で、燃料と酸化剤との量的関係を求めるために行う。この場合に、実際に生じている化学過程である素反応や化学平衡を取扱うと複雑になるので、通常は総括反応を用いて燃焼反応を表すことによって計算する。さらに量論計算においては、燃料と酸化剤との量が定まっていることが前提となるので、拡散火炎などの場合のように、燃料と酸化剤とが徐々に混合していく火炎を取扱うときには注意を要する。a. 当量比(equivalence ratio)と空気比(air ratio)可燃混合気の燃焼特性の決定要因の一つに、燃料と酸化剤の混合割合がある。可燃混合気の組成を表すのにこの関係を用い、可燃混合気中のOに対する燃料のモル比と、完全燃焼をする場合のOに対する燃料のモル比との比、すなわち当量比で混合割合を表すと便利である。酸化剤が空気である場合でも、Nは燃焼反応に対して不活性であると考える限り、式(3・17)はそのまま成立し、したがって当量比は式(3・18)で与えられる。一方、完全燃焼している可燃混合気について、その組成を燃料1molを完全燃焼させるために必要な酸素量を基準として表すと、空気比が定義できる。空気比は以下のように表される。
