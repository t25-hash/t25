# 3・2・1 火炎の形態

> **章**: 第3章 燃焼
> **節**: 3・2 燃焼現象

燃焼(combustion)は高速の発熱化学反応であり,それが気相で起こりうる場合には火炎を形成し,一般に化学反応による発光を伴う.燃焼現象には,反応に関与する物質がすべて同一の相にある均質燃焼と,その一部が他の相にある不均質燃焼がある.均質燃焼においてはほとんどの場合,気相で反応が起こり火炎が形成される.また不均質燃焼においても,反応が凝縮相表面で起こるものを除けば,凝縮相にある物質が気化した後,気相で反応が起こり火炎が形成される.燃料単独の分解燃焼においても火炎が形成されるが,通常観察される火炎は,燃料と酸化剤の燃焼において形成されるものである.燃料と酸化剤があらかじめ混合してから燃焼する場合に形成される予混合火炎と,それらがそれぞれ別の側から火炎に供給される拡散火炎とでは,性質が大幅に異なる.予混合火炎は予混合気中を伝ぱする燃焼波であり,その速度は化学反応速度と温度伝導率に依存するが,拡散火炎は火炎への燃料と酸化剤の拡散によって維持されており,燃焼量はそれらの拡散速度に依存する.熱や物質の移動速度および化学反応の速度が有限であるので,火炎は流れの状態に強く依存する.火炎付近の流れが層流であれば滑らかな層流火炎が形成され,乱流であれば不規則に変動する乱流火炎が形成される.火炎の形態は,また,火炎の形状や動きが定常であるかどうかにより定常火炎と非定常火炎に,火炎が定在するか伝ぱするかにより定在火炎と伝ぱ火炎に類別される.さらに,予混合気中を伝ぱする発熱化学反応を伴った波には,燃焼波のほかに爆ごう波(デトネーション波)があり,これらの性質は大幅に異なる.
