# 9・4・2 極限解析の上下界定理

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・3 塑性構成式（plastic constitutive equation）

一般に非硬化弾塑性体では，降伏応力が一定であるから，物体が
支えうる荷重には限界があり（極限荷重)，その荷重のもとでは変
形が進行していく（塑性崩壊）状態となる．塑性崩壊状態，極限荷
重を求める解析を極限解析（limit analysis ）という．極限解析は，
弾塑性解析に比べて簡単に最終状態の評価を行うことができ，塑性
加工の解析などに用いられる．
極限解析は上下界定理を用いて行われる．物体 の表面 上
で表面力 が，表面 上で速度 がそれぞれ境界条件として与
