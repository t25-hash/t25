# 3・2・4 予混合火炎

> **章**: 第3章 燃焼

a. 層流予混合火炎(laminar premixed flame)の構造図3・4に層流予混合火炎の構造を模式的に示す。火炎は発熱反応がほとんど起こっていない予熱帯(preheating zone)と、それが盛んに起こっている反応帯(reaction zone)から成り、一般に両者を合わせて火炎帯という。反応帯に流入する混合気は、予熱帯において反応帯からの熱によって予熱された後発熱反応が起こる。発熱反応の大きい素反応が生じている反応帯の厚さは、前者の厚さより1オーダ近く小さい。反応物質濃度、すなわち混合気の燃料と酸化剤の濃度は、いずれも予熱帯において反応帯への拡散によって減少し、反応帯で急速に減少する。生成物質は予熱帯へも拡散していく。最近では、詳細反応機構を用いた数値解析によって、火炎帯の構造をより詳しく知ることができる。図3・5に熱損失がない場合のメタン-空気火炎の火炎構造を示す。発熱速度が最大となる位置は、CH ラジカル濃度が最大となる位置にほぼ一致することが知られている。予熱帯への熱の拡散と同時に、中間生成物質の中でもOH,H など活性化学種が上流へ拡散することは、火炎伝ぱを維持するのに重要な役割を果たしている。一般に模式的構造としては示されないが、OH,H など活性化学種濃度は火炎帯下流側でも比較的高く、下流にいくに従って徐々に化学平衡状態に近づいていく。すなわち、熱損失がない理想的な状況では、反応帯の十分下流における燃焼ガス温度は断熱火炎温度に等しく、生成物質濃度は化学平衡濃度に等しい。予混合火炎の構造を論じる場合には、予熱帯や反応帯での温度、濃度、速度などの分布を考えるが、通常、対象とする火炎全体の空間現象の寸法と比べるとそれらの厚さは非常に薄いので、しばしば火炎帯を薄い火炎面としてとらえる。図3・4 velocity)可燃範囲内にある混合気が燃焼する場合、形成される予混合火炎では一定の速度で混合気が燃焼反応を起こし、燃焼生成物質(燃焼ガス)に変化する。火炎帯のどこかに座標を固定すると、火炎帯に向かって混合気が定常に流れていることになる。火炎帯およびその付近の気体に乱れがない場合、予熱帯前縁位置における火炎面に垂直方向の速度成分を層流燃焼速度といい、混合気固有の特性値となる。ただし、ほとんどの火炎は流れに対して傾いているので、このような場合は図3・6に示すように、主流速度と火炎面とのなす角をθとして、=sinθ(3・53)により層流燃焼速度を表すことができる。層流火炎伝ぱの基本メカニズムは、燃焼化学反応と熱および物質の移動現象であり、層流燃焼速度は各素反応速度、熱流体物性に依存する特性値であるが、層流燃焼速度や火炎帯厚さ(flamethickness)δの性質は、単純化されたモデルでもある程度予測できる。一段総括反応と火炎帯内での熱発生によるエネルギーバランスを考慮すると、総括反応速度(overall reaction rate)をω、平均定圧比熱を욣、平均熱伝導率をλ、未燃混合気の密度をρとして、火炎帯厚さδと層流燃焼速度はδ=λ욣ω(3・54)ωλ욣=ρ(3・55)で表される。火炎帯厚さδは総括反応速度ωの1/2乗に逆比例し、層流燃焼速度は総括反応速度ωの1/2乗に比例することを意味しており、混合気を種々の気体で希釈した結果や混合気を加熱した結果からも確認される。図3・7には、代表的な気体燃料と空気の混合気の濃度(当量比φ)による、層流燃焼速度の変化の様子を示す。燃料と酸化剤の組合せによって、層流燃焼速度の最大値とその際の当量比が異なる。層流燃焼速度は圧力により変化するが、その影響は必ずしも単純でない。例えば、H+OOH+O(3・56)H+O+M HO+M(3・57)の素反応は競合反応であるが、再結合反応である後者は高圧で相対的に反応速度が増大し、温度依存性も前者と異なる。これらや他の素反応との複合効果として層流燃焼速度は決まる。層流燃焼速度に関して、もう一つの注意すべき点は火炎伸長(flame stretch)の影響である。層流燃焼速度を実験的に求める方法は種々あるが、完全な平面火炎を作ることは困難であり、乱れや境界層、速度こう配の影響を慎重に取り除いて火炎を伝ぱさせてもわずかな火炎伸長を受けるので、火炎伸長率を変化させて火炎伸長率ゼロの値を外挿して求める方法も行われている。火炎伸長効果を受ける火炎の伝ぱ速度は、一般に次式で与えられる。-=ε(3・58)ここに、:火炎伸長を受ける火炎の伝ぱ速度、:マークステイン長さ、ε:火炎伸長率[=1/d/d、ただしは火炎面積]である。マークステイン長さ(Markstein length)は混合気組成、ルイス数、温度、圧力等に対する特性値であり、正負に変化する。式(3・58)は乱流燃焼速度を推定する際の基本ともなる。層流燃焼速度は、最近は詳細反応を考慮した数値解析から求めることもできるが、用いられる素反応式の温度や圧力、当量比に対する適用可能範囲が問題である。c. 乱流予混合火炎(turbulent premixed flame)混合気の流れが乱れている場合、火炎の様子は混合気組成だけでなく乱れの性質に依存する。乱れの性質は乱れ強さと種々の乱流特性スケールで表されるが、後者は乱流エネルギースペクトルから理解できる。すなわち、乱流エネルギースペクトルの慣性小領域を挟んで、低波数側の限界は乱れの積分スケール、高波数側の限界はコルモゴロフスケールηで代表され、中間波数領域にテイラーマイクロスケールλがある。乱流予混合火炎の構造はこれら乱流特性値と層流燃焼速度、層流火炎帯厚さ、火炎伸長効果によって変化するといわれており、種々の無次元数を用いた乱流火炎構造ダイアグラムが提案されている。図3・8に、代表としてボルギダイアグラム(Borghidiagram)をベースとした構造ダイアグラムの一例を示す。図乱流カルロビッツ数[=δ/′/λ]、乱流ダムケラー数[=/δ/′]である。乱流予混合火炎の最も単純な構造は、弱い乱れによって層流火炎が緩やかに変動する火炎であり、しわ状層流火炎(wrinkled 火炎の安定化(flame stabilization)予混合火炎は伝ぱ性があるため、静止予混合気中や予混合気流中の火炎は移動する。そこで、燃焼器の特定の位置に火炎を保持するのが予混合火炎の安定化である。予混合円管バーナなどの火炎の安定化機構を論じる際の代表的な考え方に、速度こう配(velocity gradient)の概念がある。バーナ内部には速度境界層により、バーナリム付近で速度こう配が生じている。一方、火炎はバーナリムへの熱損失や活性化学種の失活によって、バーナリムに近づくにつれて火炎伝ぱ速度が低下する。図3・9実験による乱流燃焼速度の一般関係5―57バーナリム付近のどこかで局所燃焼速度と境界層内の速度が釣合う点がある。すなわち、図3・10に示すように速度こう配がある限界点より小さいと逆火が、大きいと吹き飛びが発生するので、図3・11に示す図3・10 バーナリムにおける予混合火炎の安定限界速度こう配(天然ガス-空気混合気)5―58熱工学ように、バーナリムにおいて予混合火炎はある流速範囲でのみ安定化する。安定範囲は混合気組成の影響を受ける。火炎がいったんバーナリムに安定されると、これ以外の火炎部分は流れに対してある傾斜した角度で形成される。バーナリム付近の速度分布はほぼ直線性が保たれ、速度こう配が主流流速に比例するので、速度こう配をパラメータとして用いるのである。速度こう配によって火炎伸長が生じるが、前述の消炎現象を生じるような速度こう配を意味していない。バーナリムのような固体表面への熱損失や活性化学種の失活は、火炎安定の重要な機構であるが、特に幅の狭い間げきや細い穴では、混合気が可燃範囲にあっても燃焼を維持することができず、火炎が伝ぱすることはできない。この限界値を消炎距離(quenchingdistance)や消炎直径(quenching diameter)という。消炎距離は火炎の厚さに比例し、圧力および燃焼速度にほぼ逆比例する。図い。実用燃焼器では、より高速流中に火炎を安定化させる必要がある。このような場合、鈍頭物体やV 形物体後流の再循環域やパイロット火炎、旋回流などを用いる。再循環領域では、物体後流からの高温気体や活性化学種の未燃混合気への供給が、火炎の安定に重要な役割を果たしており、物体後流における反応の特性時間と流れの特性時間の比(ダムケラー数、Damko썦hler number)によって、火炎の安定性が判断できると考えられる。この考え方は、高速空気流中の保炎器や壁上の段差やへこみに保持された、予混合火炎の安定性を論じるのにもしばしば用いられる。
