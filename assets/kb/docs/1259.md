# 3・3・2 燃焼装置内の流れと火炎構造

> **章**: 第3章 燃焼
> **節**: 3・3 燃焼装置内の燃焼

燃焼装置内では、燃料と空気または未燃ガスと高温燃焼ガスとの混合特性が、燃焼状態を支配する。また、燃料と空気の流動は火炎安定に重要な役割を果たす。このため、燃焼装置内の流動様式と火炎形状の関連を把握しておくことが重要である。燃焼装置内への燃料と空気の供給は、円形噴流、環状噴流、または環状噴流と円形噴流を組合わせた同軸噴流とする場合が多い。また、燃料と空気の混合や火炎性状を制御する手段として、空気流に旋回を与える場合が多い。空気供給に末広ノズルを使用した場合の旋回流強さと燃焼装置内流動の関係が調べられており、以下の三つの基本的フローパターンが観測されている。図3・23はその様子を模式的に示したものである。図3・23(a)は旋回強さが零ないし弱い場合の流れの様子を示しており、環状空気流がノズルスロート付近でノズル表面からはく離する傾向をもつ。この結果、旋回を伴わない環状噴流と同様の流れとなり、燃焼させた場合には、バーナ出口の下流域において振動する噴流火炎が形成される。図3・23(b)は旋回強さが中程度から強い場合で、空気流は末広ノズル壁面に安定して付着し、噴流中心に再循環領域が形成される。このような流れの場合、安定の良い火炎が得られ、高負荷燃焼が可能となる。図3・23(c)は半径方向の壁面噴流の様子を示している。このような流れ場では、燃焼室壁面に付着した火炎(ひまわり形の火炎)が形成される。バーナ形状は半径方向壁面噴流の形成に大きく影響し、トランペット形で角度の広い末広ノズルとすることにより、中程度の旋回強さでも壁面噴流を形成できる。このような火炎は、強くかつ均一なふく射加熱を必要とする炉に適している。
