# 9・6・4 残留応力の除去

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・6 残留応力

さらに，塑性不安定状態において材料内での不連続面が発生する場合がある．
この不連続面の法線ベクトルを とし，以下のようなアコースティックテンソル を定義する．
そしてこのアコースティックテンソル に対して，不連続面が発生する場合の不安定条件は以下のように与えられる．
不連続面の法線方向ベクトル の関数であり二次元問題において残留応力は時効割れや応力腐食割れを起こす原因となる．
このため，残留応力を取り除く作業が必要となる場合がある．
熱処理により残留応力を除去する場合は，普通，焼なましが用いられる．
焼なまし条件は材料により異なるが，例えばオーステナイト系ステンレス鋼の場合には，850～900℃で2～4時間保持する方法や，340～430℃で30分～3時間保持する低温応力除去熱処理がある．
また，軽度の引張りやロール成形などの機械的負荷により残留応力を軽減する方法もある．
これは，圧延などの塑性加工を経た単純な形状をもつ部材に対しておもに行われる．
