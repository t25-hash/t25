# 3・3・3 振動燃焼

> **章**: 第3章 燃焼
> **節**: 3・3 燃焼装置内の燃焼

a. 発生機構と原因振動燃焼(oscillating combustion)とは、燃焼装置を構成する気体流路系の音響的固有振動数が燃焼によって熱的に駆動され、燃焼器内の圧力やガス流速が周期的に変動する現象をいう。この振動が自励振動であるためには、燃焼による発熱速度の変動が流速や圧力の変動を駆動し、誘起された音響的固有振動が逆に、燃焼による発熱速度の変動を周期的に駆動するようにフィードバックされる必要がある。図3・24は自励振動が励起される機構を模式的に示したものである。燃焼室圧力が変動すると燃焼室流入部での差圧が変化し、それに応じて燃料および空気流量が変動する。この変動は、供給系の応答時間遅れのために燃焼室圧力変動と完全に逆位相とはならないで、少しのずれが生じる。この流量変動は燃焼時間遅れだけずれて発熱変動となり、燃焼ガスの膨張の効果によってさらなる圧力変動をもたらす。この圧力変動が元の圧力変動と同位相になると、変動は増幅され自励変動に発達する。したがって、振動燃焼が発生するためには、燃料あるいは空気の供給系の応答時間および燃焼時間遅れと固有振動周期との間に、一定の関係が成立している必要がある。(a)(b)(c)図3・23 末広ノズルから噴出する旋回噴流および火炎の流れ模様図3・24 燃焼室圧力変動による自動振動の発生b. 振動燃焼抑制法振動燃焼を抑制するには、まずその振動数や振幅を調べて振動の形態を明らかにし、できる限り振動の原因を明確にすることが望ましい。具体的には次のような振動抑制対策がある。(1) 圧力変動の腹付近に、燃焼室圧力変動とちょうど逆位相になるような圧力変動をもつ共鳴器を取付ける方法。(2) 何らかの方法で振動エネルギーの減衰を図る。具体的には、圧力変動の腹の位置に穴をあけ、そこから振動のエネルギーを外部に放出させたり、燃焼室壁に多孔金属板や吸音材を張り付けて振動エネルギーを吸収する方法。(3) フィードバック機構が明らかになった場合にその機構を弱める方法。具体例としては、トンネルバーナなどの場合に予混合気の燃焼室への噴出速度を速くして、燃焼室と供給管との結合を防ぐ方法が挙げられる。
