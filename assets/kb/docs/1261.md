# 3・4・1 硫黄酸化物(,sulfur oxides)

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

a. 成分と生成量固体燃料や液体燃料中の硫黄分は、燃焼によって大部分が硫黄酸化物(SO)として燃焼ガスとともに排出される。ただし、灰分の多い固体燃料の燃焼では灰中にとどまる硫黄分が増える。SOの排出量は燃料中の燃焼性硫黄の量によってほぼ決まる。例えば質量分率C:0.84,H:0.11,S:0.05の燃料が完全燃焼し、硫黄分がすべてSOになるとすれば、乾き排ガス中のSO濃度は図3・25のようになる。排ガス中のSOの大部分はSOであり、SOは少ない。低温ではSOのSOへの酸化反応の平衡定数が大きくなるが、反応速度も低下するために、低温で気相反応によるSOの生成はあまり起こらない。SOの生成には、高温におけるO原子によるSOの気相酸化や、低温での金属酸化物の触媒作用が寄与している。SOは排ガス中の水蒸気と結合して硫酸の蒸気になるため、SO濃度が高いほど酸露点温度が高くなり、露点より低い温度の壁面上やガス中に、低温腐食の原因となる硫酸ミストを生成しやすくなる。SOの生成を抑制するには、空気比1.05以下の低空気比燃焼が効果的である。燃料過剰条件で燃焼すると、HS,COS,S,S,SO,HSなども生成する。b. 排煙脱硫(flue gas desulfurization)硫黄酸化物の排出を抑制するには、低硫黄分の燃料への転換、燃料中の硫黄分の除去、燃焼過程における脱硫、排ガスからのSOの除去、省エネルギーによる燃料使用量の低減などの方法がある。燃料油中の硫黄分の除去は、有機硫黄化合物の形で存在する硫黄分を、触媒と高圧水素を用いて硫化水素に転換する水素化脱硫法により行われている。石炭中の硫黄分を除去する方法には、物理的プロセス、化学的プロセス、微生物を利用するプロセスがあるが、一般に石炭の脱硫は難しくコストもかかる。石炭中には黄鉄鉱などの無機硫黄と、有機化合物の形の有機硫黄があるが、特に有機硫黄の除去は難しい。そのなかで、微生物による脱硫法は有機硫黄の除去も可能ということで、近年注目を集めている。5―63ガス燃料にもHS や有機硫黄化合物が含まれる。HS は、アルカリ性吸収液や酸化鉄系吸収剤を用いて除去されている。燃焼過程における脱硫は炉内脱硫と呼ばれ、燃焼炉内に石灰石微粉を吹込んだり、流動層燃焼において流動媒体に石灰石などを投入して、SOを吸収除去する。脱硫率、石灰利用率ともにあまり高くはない。石灰石や消石灰を粉炭と混合・加圧成形した石炭ブリケットを燃料として使用すれば、炉内脱硫と同様の効果が得られる。以下、排ガスからのSOの除去、すなわち排煙脱硫について述べる。. 湿式脱硫法湿式脱硫法は、アルカリの水溶液またはスラリーでSOを吸収する石灰(石)石こう法、ソーダ法、水マグ法(水酸化マグネシウム吸収法)などが普及している。石灰(石)石こう法では、石灰石または石灰のスラリーでSOを吸収し石こうに変換して回収する。石こうはセメント原料や石こうボードなどに利用する。アルカリ源として最も安いのが石灰石であり、これを用いる石灰(石)石こう法の大型プラントが火力発電所などに広く普及している。ソーダ法では、NaOH またはNaSOの水溶液でSOを吸収し、NaSOやNaSOを直接回収したり、さらに石灰石などと反応させて石こうにしてから回収したり、加熱してSOを回収したりする。NaOH のコストは石灰石よりはるかに高いので、この方法は、副生物の利用が可能なパルプ、ガラス産業や小規模プラントで使用されている。水マグ法では、海水を消石灰で処理して得られるMg(OH)をSOの吸収に使用する。生成するMgSOは、空気で酸化してMgSOにしてから放流する。ソーダ法と同様に石こうやSOを回収することも可能である。放流法はプロセスが簡単で建設コストが低い、MgSOは溶解度が高く取扱いが容易、ソーダ法より吸収剤が安いなどの長所があり、小規模プラントで普及している。アンモニア、塩基性硫酸アルミニウム、希硫酸等の水溶液でSOを吸収する方法もある。これらの方法ではSOを吸収した後、最終的には石灰石または消石灰を使って、石こうを副生させることが多いが、プロセスの複雑さ、コストなどの面からあまり採用されなくなった。海水中のアルカリ分を吸収剤として使用する海水洗浄法は、脱硫率はやや低いが、最も単純な方法である。. 乾式脱硫法乾式脱硫法は、湿式脱硫法ほど広く普及していない。石炭灰利用法では、石炭灰、消石灰および石こうから得られる脱硫剤を使って排煙中のSOを吸収する。使用済み脱硫剤の一部は、脱硫剤の製造に利用する。アルカリ吸収法では、ソーダ灰、重炭酸ソーダ、消石灰などのアルカリの粉体をダクトへ吹込む。触媒酸化法では、バナジウム触媒を使って排ガス中のSOをSOに酸化後、冷却して硫酸を回収する。銅精錬所などの高濃度のSOを含む排ガスへの適用例がある。活性炭吸着法では、活性炭上でSOとO,HOを反応させて硫酸にして吸着する。吸着した硫酸は、水洗や加熱・還元によって離脱するか、水蒸気を通して離脱する。. 半乾式脱硫法湿式脱硫法は90%以上の高い脱硫率が得られるが、装置が複雑であり、排水処理も必要であるという短所がある。それに対して、脱硫率は高くはないが、簡易に低コストで脱硫する方法として開発されたのが半乾式脱硫法であり、スプレードライ吸収法、石灰ダクト吹込み・水噴霧法、石灰石炉内吹込み・水噴霧法などがある。スプレードライ吸収法では、消石灰スラリーを乾式吸収塔に噴霧し、スラリー粒子の乾燥過程でSOを吸収する。脱硫反応生成物は集じん機で捕集する。その一部は脱硫剤として循環使用し、残りはセメントに混ぜて利用したり、フライアッシュと混合して安定化してから廃棄する。石灰ダクト吹込み・水噴霧法は上記と似たプロセスであるが、スラリーを扱う煩雑さを避けるためにスラリーの代わりに石灰微粉を吹込み、その後で水をスプレーして加湿し、脱硫反応を起こさせる。石灰をダクトに吹込む代わりに、石灰石を炉内に吹込んでから水を噴霧するのが3番めの方法であり、コストの低い石灰石を脱硫剤として使える利点がある。図3・25
