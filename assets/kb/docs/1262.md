# 3・4・2 窒素酸化物(,nitrogen oxides)

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

a. 成因と特性通常の燃焼排ガス中の窒素酸化物の大部分はNOであるが、NOは大気中で酸素によって酸化されてNOになる。両者を合わせてNOと呼ぶ。NOは、燃焼装置内で空気中の窒素や燃料中の窒素分が酸化されて生成するが、生成機構の違いによりサーマルNO、プロンプトNO、フューエルNOと呼ばれている。. サーマルNO(thermal フューエルNO(fuel NO)重油や石炭などの燃料中にはピリジン類、ピロール類などの窒素化合物が含まれており、これが酸化されるとNOが生成する。このようなNOはフューエルNOと呼ばれており、その生成機構は以下のように考えられている。まず、燃料中の窒素化合物の熱分解によりNH(=0~3)やHCN,CN など、窒素を含む中間生成物が生成する。HCN やCNの生成は燃料過濃領域で多くなる。HCN やCN はHCN+OH →HNCO+H,CN+OH →NCO+H などの反応によりC-N 結合の切れやすいHNCO,NCOに変換されてから、NCO+H →NH+CO,HNCO+H →NH+COなどの反応によりNH,NHになる。NH(=0~3)はOH やO,Oによって酸化されてNOを生成したり、NOと反応してNを生成する。窒素分の濃度が高くなるほどNOへの変換率は低くなる。これは、NHがNOと反応してNを生成する割合が高くなるためである。サーマルNOの場合と異なり、フューエルNOの生成に対する温度の影響は小さい。一方、フューエルNOの生成には酸素濃度が大きな影響を与え、空気比が高いほど燃料中の窒素分のNOへの変換率も高くなる。また、燃料と空気の混合も大きな影響を与え、予混合火炎に近づくほどNOへの転換率が高くなる。図3・27は、フェニモアが当量比と温度の異なるエチレン予混合火炎に種々の窒素化合物を添加して、N 生成の抑制上述のNOの成因と特性に基づいて、その生成を抑制することが可能である。サーマルNOのうち拡大ツェルドビッチ機構に基づくものは、温度の影響が大きいので燃焼温度を下げることにより生成を抑制できる。そのほか、高温域での滞留時間の短縮、酸素濃度の低減も効果がある。プロンプトNOの抑制には希薄燃焼、フューエルNOの抑制には窒素分の含有の少ない燃料への転換や、酸素濃度の低減が効果がある。具体的なNO生成抑制技術としては以下のようなものがあり、上記の生成抑制の原理を単独または組合わせて用いている。. 燃料転換一般に燃料を固体から液体、気体に転換すれば、この順にNO排出量は低下する。この順に燃料中の窒素分が少なくなりフューエルNOの生成が低下することがおもな理由であるが、この順に燃焼が良好になって燃え切り時間が短く、高温滞留時間が短くなるために、サーマルNOの生成が低下することもNO低減に寄与している。さらに、良質な燃料ほど以下に記す抑制対策を適用したときの効果も大きい。. 運転条件の改善燃焼用空気の予熱温度の低下、燃焼室負荷の低減は、ピーク温度を低下させるのでサーマルNOを抑制することができる。ただし、空気予熱を排ガスとの熱交換で行っているボイラなどでは、空気予熱温度を低下させると熱効率の低下を招く。低空気比燃焼は酸素濃度の低下によりフューエルNO、サーマルNOの抑制に効果があるが、ばいじんやCOなどの増加に注意する必要がある。窒素分を含まないガス燃料の予混合燃焼では、NO濃度のピークは空気比1.0付近になる。高空気比側でNO濃度が低下するのは、燃焼温度低下の効果のほうが大きいためである。実用燃焼装置は拡散燃焼であり、予混合燃焼の場合のように単純ではない。. 二段燃焼、濃淡燃焼二段燃焼では燃焼用空気を二段に分けて供給する。一段めは空気不足で燃焼し、その後流部へ二次空気を供給して完全燃焼させる。一次燃焼域では酸素濃度が低いためにNOの生成が抑制され、二次燃焼領域では比較的温度の低い速やかな燃焼によりNOの生成が抑制される。これはフューエルNO、サーマルNOの抑制に効果がある。総合空気比を一定にした場合、二段燃焼の効果は一段めの空気比および二次空気の供給位置に依存する。一段めの空気比を下げるほどNOの生成は抑制されるが、ばいじんやCOなども増加するため空気比の低下には限界がある。複数あるバーナの一部を燃料過剰の状態で使用し、その周囲に空気過剰のバーナかあるいは空気のみの送入口を配置して、二段燃焼の場合と同様のNO抑制効果をねらったのが濃淡燃焼である。. 排ガス再循環、水または水蒸気噴射排ガス再循環は、温度の低下した燃焼排ガスの一部を燃焼ゾーンに循環させる。不活性ガスが加わることにより、燃焼ガスの熱容量が増加して火炎のピーク温度が下がり、サーマルNOの抑制に効果があるがフューエルNOの抑制にはほとんど効果がない。燃焼用空気に対する循環排ガスの流量比を大きくするほどNO低減率は大きくなるが、燃焼が不安定になるため限界がある。水または水蒸気の燃焼室内への噴射や、燃料に水を混合したエマルジョン燃料の使用によっても、燃焼温度が低下してNOが減る。ただし、宮内らの実験によれ5―65ば、温度の低下をもたらさない程度の水分の添加によってもNOの減少が起こることから、水分は化学的作用によってもNOを低下させていると考えられる。省エネルギーのために排ガスの熱を積極的に燃焼用空気の予熱に利用するリジェネティブバーナでは、通常の燃焼をさせると高温の予熱空気のためにサーマルNOが増大するので、二段燃焼や炉内ガスの再循環などによってNOを低減している。燃料と高温予熱空気を急速に混合することもNOの抑制に効果がある。それぞれ混合促進型、分割火炎型、自己再循環型、段階的燃焼組込み型と呼ばれている。段階的燃焼組込み型バーナは、二段燃焼と同様にフューエルNOとサーマルNOの両方の抑制に効果がある。. 炉内脱硝炉内脱硝は、一次燃焼の後流部に残存酸素と当量以上の二次燃料を吹込んで、一次燃焼域で生成したNOを還元する。さらにその後流に空気を供給して、未燃分を完全燃焼させる。二段燃焼法と違って、一次燃焼は通常の空気比で行うため燃焼の安定性が良い。c. 排煙脱硝(flue gas denitrification)燃焼排ガス中のNOを煙道の途中で無害化、または除去するのが排煙脱硝技術であり、還元剤を用いてNOを無害な窒素に変換する乾式法と、アルカリなどの水溶液で吸収除去する湿式法に大別される。わが国で固定発生源用に最も普及しているのは、触媒を用いてアンモニアでNOを還元する選択接触還元法であり、処理ガス量で90%以上のシェアを占めている。この脱硝法はアンモニアSCR(selective cata-lytic reduction)法とも呼ばれる。その他、無触媒選択還元法、炭化水素類を用いる選択接触還元法などがある。リーンバーンガソリンエンジン用に実用化されているNO吸蔵還元型三元触媒については、γ4編「内燃機関」1・6・2項および2・5・2項を参照されたい。また、活性炭(活性コークス)上に排ガスを通して乾式で脱硫を行うとともに、アンモニアを添加して脱硝も同時に行う方法、アンモニアを添加した排ガスに電子線を照射してSO,NOを硫酸アンモニウム、硝酸アンモニウムおよびそれらの複塩にして同時除去する方法、酸化チタン光触媒を用いて大気中のNOを除去する方法などもあるが、ここでは触れない。. アンモニアSCR(ammonia SCR)法アンモニアSCR 法は、排ガス中に存在するNOと同量程度のアンモニアを添加して触媒上で反応させ、容易に90%以上の脱硝率が得られる。アンモニアは、過剰の酸素が存在する条件下でもNOを選択的に還元し、窒素と水を生成する(4NO+4NH+O=4N+6HO,NO+NO+2NH=2N+3HO)。触媒としては、アナターゼ型のチタニアにバナジウム、タングステン、モリブデン等の酸化物を担持したものが使用されている。触媒の形状は、ガスが触媒表面に対して平行に流れるような構造(ハニカムや板、格子状でダストスルータイプと呼ばれる)のものが開発され、圧力損失が小さく、ダストの堆積も少ない。脱硝性能は一般に、温度、NH/NO比、空間速度に依存する。高温領域ではアンモニアのNOへの酸化が起こるために、脱硝率が低下する。NH/NO比を大きくするほど脱硝率は高くなるが、未反応アンモニアの排出濃度も高くなる。通常、NH/NOモル比は1.0以下で運転される。触媒の耐久性は長く、石炭燃焼排ガス用でも数年の寿命がある。アンモニア水、尿素水等を用いてもアンモニアと同様にNOを選択還元できる。. 無触媒選択還元法(selective non-catalytic reduction)燃焼炉直後の煙道の高温部にアンモニアを添加して、触媒を用いずに気相でNOを還元する。脱硝率に大きな影響を与えるのは温度、滞留時間、NH/NO比である。低温では脱硝反応が起こら5―66熱工学ず、高温ではアンモニアがNOへ酸化されるため脱硝率が低くなる。脱硝率が高い温度範囲は850~1000°Cである。無触媒選択還元法は選択性が低く、触媒を用いるアンモニアSCR 法より高いNH/NO比を必要とする。実機では反応温度、滞留時間等の制約があるため脱硝率は低く、NH/NOモル比1.2~1.6で30~50%程度である。本法においても、還元剤としてアンモニア水や尿素水が使用可能である。無触媒選択還元法は触媒が不要で、設備も簡単であるという長所があり、ごみ焼却炉への適用例が多い。. 炭化水素類を用いる脱硝法近年、炭化水素類やアルコール類を用いてもNOの選択接触還元が可能なことがわかってきた。この脱硝法はいまだ研究段階のものが多いが、いろいろな触媒と還元剤が試験されている。多くの触媒は水分共存下では活性がかなり低下するが、活性低下の小さい触媒としてFe(II)-ZSM-5,Co-BEA、貴金属系(特にIr)などが発表されている。一方、メタノールやエタノールなどの含酸素化合物を還元剤に用いると、水分共存下で高いNO除去性能が得られる。エタノールとAg-AlO触媒の組合せは、特に性能が高い。Ag-AlOはまた、ゾル-ゲル法で調製した触媒が-オクタンなどの長鎖炭化水素によるNO還元に、水分共存下でも高活性を示す。近年、NO以外の窒素酸化物としてNOが注目されているので、簡単に触れておく。固定燃焼装置の場合、石炭や下水汚泥の流動層燃焼で比較的高濃度のNOが排出されている。NOは地球温暖化ガスの一つであり、その排出は望ましくない。NOはおもに燃料中のN 化合物から生成する。NOの生成を燃焼改善で抑制する方法には、燃焼温度の上昇、酸素濃度の低下、接触粒子との混合促進、高圧化などがある。排ガス中のNOの処理には、分解触媒、選択還元触媒、電気化学的分解などが研究されている。
