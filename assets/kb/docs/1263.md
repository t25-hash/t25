# 3・4・3 粒子状物質

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

窒素酸化物,硫黄酸化物などのガス状物質と並ぶ大気汚染物質として,粒子状物質がある.これは固体,液体状物質の総称であり,発生源,性状により,ばいじん,粉じん,ミストなどさまざまな名称で呼ばれる.粒子状物質のうち,固体粒子は一般にダスト(dust),金属蒸気などが凝縮して生じた1 m以下の粒子はフューム(fume),液体粒子はミスト(mist)と呼ばれる.さらに,燃焼過程から排出される物質のうち,固体状の物がばいじんである.法律では,燃料その他の物の燃焼または熱源としての電気の使用に伴い発生する物質をばいじん(soot and dust)と呼び,物の破砕等の機械処理により発生する粉じん(dust)と区別する.自動車の運行に伴い発生する場合は,粒子状物質(particulate matter)と呼ぶ.ばいじんの成因は,おもに以下の三つである.(1) 燃料中の灰分が微細な固体粒子になる(フライアッシュまたは飛灰).(2) 液体燃料や固体燃料中の炭素分が酸化しきれずに残る[残留炭素または残炭(coke),セノスフェア(cenosphere),チャー(char,固定炭素)].(3) 気体状の炭化水素が火炎中で高分子化,凝集する(すすまたは気相析出型すす).いずれの物質も吸入により健康に影響がある.特に自動車が排出する粒径の小さいSPM(浮遊粒子状物質)は,発がん性物質と考えられる多環芳香族化合物を吸着しており,癌などの疾病を引き起こす可能性が指摘され,花粉症との関連も議論されている.a. フライアッシュ(fly ash)石炭中には2~15%程度の灰分(ash content)が含まれ,重質油もわずかであるが(0.1mass%以下程度)灰分を含む.微粉炭燃焼では灰分のほとんどが気相中に放出されるため,多量のフライアッシュが生成する.わが国での石炭消費量は,発電用に限っても5779万t(2000年度)であり,排出される石炭灰は約774万tにのぼる.一方,一般廃棄物焼却においては,灰分のほかに低沸点金属,塩素などによりフライアッシュが生成される.排出量は石炭灰に比べると少ないが,ダイオキシン類や溶出しやすい金属塩化物を含むため,高度処理が必要である.すなわち,廃棄物焼却炉より排出されたフライアッシュは,重金属溶出とダイオキシン類の含有量基準の両方で,特別管理廃棄物に指定される可能性が高く,指定された場合には薬剤処理,セメント固化,溶融固化など,何らかの有効な処理をしなければ埋立処分できない.フライアッシュの粒径は,微粉炭燃焼炉ではサブミクロンから数百m程度であり,燃料石炭の平均粒径に大きな影響を受ける.石炭灰分の溶融温度は1200~1400°C程度であるので,粒子径数十m以下の微粒子は,燃焼に伴いいったん溶融してから凝固したきれいな球状をしていることが多い.それに対して大きな粒子は,壁面,伝熱面に付着した灰分がはく離したものが主で,不定形をしている.石炭灰の主成分はSiO,AlO,FeOなどの酸化物であり,廃棄物焼却灰にはこれに加えて金属塩化物が多く含まれる.燃料中に含まれていた金属類(Na,V,Pb,Se,As,Ca,Zn,Ga,Mo,Hg など)は燃焼時に蒸発し,燃焼後期以降に微少粒子を核にして凝縮する傾向がある.現在排ガス中の金属は,指定されたばい煙発生施設でCd,Pbが規制されているだけである(捕集された後は重金属溶出量により規制される場合もある)が,今後問題になる可能性がある.ダイオキシン類も同じく燃焼後に凝縮しやすく,特に有機物に吸着しやすい特性があるため,すす粒子に吸着して存在する可能性が高い.フライアッシュの排出抑制は,主として集じん装置により行われる.粒子径が大きければ重力沈降,遠心力(サイクロン),粒子径が小さい場合は電気集じん,ろ過式集じん(バグフィルタ)が用いられる(集じん装置の効率等については化学工学便覧等を参照).そのほかに,高温で溶解した灰を捕集する融灰式燃焼があり,燃焼室壁面を高温に保ち,旋回流により溶融したフライアッシュを高温壁に捕集する.もともと石炭燃焼用に開発された技術であるが,現在は石炭に比べて灰融点の低い下水汚泥で使用されるほか,廃棄物焼却におけるガス化溶融炉にこの燃焼方法が採用されている.b. 残留炭素石炭中の固定炭素分(揮発分でない部分)や,液体燃料中の炭素分が燃焼器内で完全に燃焼しきれずに排出される場合,これを残留炭素と呼ぶ.重油などの液体燃料の燃焼で,燃料液滴が急速に加熱される条件では,揮発分が蒸発した後に多孔質固体粒子が生成する.一方,石炭などの固体燃料の燃焼では,揮発分が放出された後に揮発しにくい固定炭素(チャー)が残りやすい.いずれの場合も,揮発分がほとんどないために燃焼形態は表面燃焼になる.また,粒子径がすすなどに比べて大きいため,これを完全に酸化するためには高温でもある程度の時間を必要とする.セノスフェア,チャーなどの残留炭素は,燃焼効率に大きく影響する.特に石炭燃焼では,条件により数%程度のチャーが残ることがあり,その場合は燃焼効率が低下するとともに,フライアッシュの有効利用の妨げになる.残留炭素の生成は燃料の種類や燃焼条件によって異なり,燃料中の不揮発性炭素分が多いほうが生成しやすい.これを抑制するためには,燃料粒子の初期粒子径をできるだけ小さくして,生成する残留炭素粒子径を小さくすること,酸素の残存する高温域での滞留時間を長くして,酸化を促進することが有効である.c. すす(soot)ガス状の炭化水素(液体または固体燃料から発生した炭化水素を含む)が気相で燃焼する際に,酸素不足で熱分解して生成する不飽和炭化水素または炭化水素ラジカルが,高分子化,凝集を経て炭素質粒子に成長したものがすすである.多数の球状粒子が連なった形をしており,その粒子の粒径は10~200nm程度,多くは10~50nmの間である.炭素物質ではあるが,通常質量で1%程度の水素を含んでおり,炭素との割合からいえばCH程度に相当する.すすは酸素の存在する高温域では急速に酸化するが,一部酸化しきれなかった分が排出される.通常はすすの生成と酸化は同時に起きており,その両方の特性が排出特性に影響する.すすの構造をさらに詳しく見ると,球状粒子群は炭化水素を吸着しており,それもすすの一部を構成している.特にディーゼル排気粒子状物質は,炭素粒子とそれに吸着した有機および無機化合物を合わせたものを意味する.吸着した有機化合物の量とその組成の特徴は燃焼条件により大きく異なる.吸着した有機化合物は,燃料の一部や,すすの前駆物質(precursor)である芳香族化合物,多環芳香族化合物などが多い.一般に,すすの生成には燃料構造の影響が大きいといわれるが,予混合火炎と拡散火炎で燃料構造の影響は異なる.予混合火炎では,ほとんどすべての燃料が同じ構造単位まで分解してそこからすすを生成するため,燃料構造の影響を受けにくく,燃焼温度の影響を強く受ける.すす生成の出発物質になる上記の構造単位は明確ではないが,炭素数2ないし3の二重,三重結合をもつ炭化水素,例えばアセチレンが有力である.一方,拡散火炎でのすすの生成には燃料構造の影響が明確に見られる.一般に,燃料の種類ではパラフィン,オレフィン,アセチレン,芳香族の順に,また,同系列の燃料では分子のサイズが大きい燃料ほどすすが生成しやすい傾向がある.拡散火炎では,火炎近傍の燃料側は温度も高く酸素が非常に少ないので,すすが生成しやすい条件にある.ここで生成したすすは,一部が酸化せずに排出されるため,燃料のもつ熱分解特性の影響を残すことになる.実用的に重要な拡散燃焼でのすすの抑制には,以下の方法が知られている.(1) 燃料と空気の混合を良くする.(2) 排ガス再循環などにより,火炎温度を上げすぎないようにする.(3) 燃焼後期の温度と酸素濃度を維持してすす酸化を促進する.(4) 液体燃料と水をエマルジョンにして噴霧し,燃焼過程で噴霧液滴中の水が沸騰して爆発状態になるミクロ爆発を利用して,微粒化と混合を促進する.(5) 燃料にBa,Mnなどの金属を微量添加する.
