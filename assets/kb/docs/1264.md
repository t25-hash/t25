# 3・4・4 一酸化炭素(carbon monoxide),炭化水素(hydro

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

carbon)-一酸化炭素(CO)は周知のとおり強い有害性をもつため,環境基準が決められている.都市大気中のCOはそのほとんどが自動車排気の影響と考えられ,自動車排気中のCOは,窒素酸化物,炭化水素,ばいじんなどとともに規制されている.一方,炭化水素(HC)は,窒素酸化物とともに光化学オキシダントの原因物質と考えられており,同じく自動車排気について規制されている.通常,空気比が1以上で完全燃焼すれば,燃料中の炭化水素,あるいは炭素や水素は,最終的にCOあるいはHOになり,CO濃度は非常に低くなる.ところが,酸素不足,温度の低下などの条件が存在すると,不完全燃焼により,すすとともにCO,HCが生成する.酸素不足の原因は,過濃燃料の供給や不完全な混合によることが多い.一方,高速で流動する場合や希薄燃焼では,流動に対して相対的に反応速度が不足して,完全な酸化が行えない場合がある.内燃機関などの間欠燃焼では,燃焼後に急速に冷却されるので,燃焼室の冷却した金属壁面での消炎によってCO,HCの排出を完全になくすことはかなり困難である.ガソリン機関のリーンバーン,ガスタービンの希薄予混合燃焼など,いずれも窒素酸化物低減,燃費向上には非常に効果的であるが,燃焼温度が低下しやすいため,一酸化炭素,炭化水素の増加をいかに防ぐかが重要な技術的課題になっている.また,廃棄物焼却などの場合には,塩素など反応性が高く,連鎖反応の中間生成物と急速に結合してしまう成分が含まれるため,高濃度のCOを生成することがある.CO,未燃炭化水素の排出を低減するには,上記のような排出原因をできるだけ避けるように対処する必要がある.また,COやHCは酸素存在下で1200K 程度以上の温度があれば,燃焼処理が可能である.そのため,燃料の発熱量が低くても熱回収により高温5―67を維持できる燃焼器や,触媒燃焼器を用いて処理することがある.ガソリン機関の窒素酸化物処理に用いられる三元触媒は,CO,HC側から見れば酸化触媒なので,窒素酸化物の還元剤として利用することにより,CO,HCも同時に処理できる.
