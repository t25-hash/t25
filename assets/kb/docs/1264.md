# 9・9・2 定常クリープ

> **章**: 第9章 非弾性変形と構成式
> **節**: 9・9 クリープ変形

まず，一定応力下のクリープ曲線が定常クリープを仮定すること
で近似的に表現できる場合を考える．定常クリープ速度（steady-
state creep rate）あるいは最小クリープ速度（minimum creep
rate）εは，温度 と応力σの関数として，
ε＝Ασ (9･100)
のように表現されることが多い ．定常クリープ速度の応力依
存性は，低応力に対して応力のべき関数
σ＝σ(ノルトン) (9･101)
また高応力に対しては応力の指数関数
σ＝expβσ(ルードウィク) (9･102)
を用いて表現できる．低応力から高応力までのすべての応力範囲に
ついては，双曲線関教を用いた定常クリープ速度の表示
σ＝sinhασ(ガロファロ) (9･103)
が有効である．
単軸定常クリープ則に含まれる応力σを相当応力σに置換える
ことによって多軸応力下の定常クリープ速度（定常クリープ速度テ
ンソルの強さ）の応力依存性を記述することができる．式(9･101)
のノルトン則は，取扱いが容易なべき関数型を有することから，ミ
ーゼス型の相当応力を用いて多軸形式に拡張したうえで，構造要素
の定常クリープ解析に広く用いられている．なお，多軸定常クリー
プ則を用いて求めた構造要素の応力場は，類似の形式の非線形弾性
構成式を用いて求めたそれと一致する．このことは，ホフのアナロ
ジーと呼ばれている ．
