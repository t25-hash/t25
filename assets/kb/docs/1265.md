# 3・4・5 ダイオキシン類

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

ダイオキシン(dioxin)は本来,化合物のある構造を示す名称であるが,一般に環境汚染物質としてのダイオキシンとは,ポリ塩化ジベンゾパラジオキシン(PCDDs: polychlorinated dibenzo--dioxin)とポリ塩化ジベンゾフラン(PCDFs: polychlorinateddibenzofuran)の総称である.わが国の法律では,PCB(polych-lorinated biphenyl)の異性体のうち,平面状の構造をもつコプラナPCB(Co-PCBs)を前記のPCDDs,PCDFsに加えて,ダイオキシン類と呼んでいる.ダイオキシン類は,おもに有機塩素系化学製品生産時の不純物として,あるいは廃棄物焼却など燃焼過程での副生物として生成し,製品の使用時あるいは廃棄時に,また燃焼排ガス,排ガス処理に伴う排水に含まれて自然界に放出される.その他,山火事,火災,煙草,自動車排気など,多くの発生源が知られているが,放出量はあまり多くない.表3・8に現在明らかになっている発生源からの放出量を示す.最近では,ダイオキシン類を不純物として含む可能性のある化学製品はほとんど生産されていないので,現在,環境に排出されるダイオキシン類のほとんどは燃焼起源である.ただし,環境中に存在するダイオキシン類については,化学製品の使用に伴って過去に環境中に放出されたものや,トランス油,熱媒体に使用されたPCBの一部が環境中に漏出した影響がいまだに残っているとの有力な学説がある.実際,ダイオキシン類の食品中濃度やヒトの体内濃度は,データの存在するここ30年ほどの間減り続けている.自然界に存在するダイオキシン類の量(毒性換算値)は,河川等の底質で1g 当たり数pg(ピコグラム)から数十pg,大気では1m当たり数pg 以下と極めて低濃度であるが,食物連鎖により濃縮され,おもに食品経由でヒトの体内に取込まれる.非常に強い急性毒性をもつが,環境中濃度が低いため,これが問題になる可能性はほとんどない.しかし,発がん性,催奇性,生殖毒性などの慢性毒性については,必ずしも安全なレベルとはいいがたい状況である.ヒトの平均的なダイオキシン類摂取量は,一生涯とり続けても影響がないと考えられる量(TDI,毒性に関する実験値を精査して補正係数を掛けて決められた)として日本政府が認めた4pg/kg・dの半分程度であるので,食生活の嗜好,地域性などによって,かなりリスクが高い場合も考えられる.a. 生成燃焼に伴うダイオキシン類の生成機構は,まだよくわからない部分が多い.少なくとも,煙道から排出されるダイオキシン類の大部分は,燃焼過程での未燃分,特にすすや多環芳香族化合物を起源とし,酸化分解の過程で一部がダイオキシン類に変化すると考えられている.また,燃焼反応中にベンゼン環が生成され,それが重縮合してダイオキシン類になるとの意見もあるが,直接的な証拠は示されていない.確実にわかっていることは,すす,多環芳香族化合物などをフライアッシュとともに空気中で加熱すると,250~350°C程度の温度域でダイオキシン類,特にPCDFsを生成することである.このとき,時間単位の比較的長い反応時間が必要である.この生成反応はデノボ合成(de novo synthesis)と呼ばれることが多く,すすなどの構造の一部が,廃棄物中に含まれていた金属,特に銅の触媒作用による酸化反応で分離し,PCDFsを生成すると考えられている.焼却炉などの排ガス処理装置,特に電気集じん機の運転温度はこの領域を含むため,処理中にダイオキシン類の濃度が増加することがある.デノボ合成は定義自体が研究者によって異なるため,反応温度域も,下限は同じでも上限は600°C以上と考える場合もある.一方,焼却炉排ガス中の塩化フェノールあるいは塩化ベンゼン濃度と,ダイオキシン類濃度は高い相関をもつことが知られている.5―68熱工学発生源排出量一般廃棄物焼却施設産業廃棄物焼却施設小型廃棄物焼却炉等368~619368~619307~509353~370火葬場2.1~4.62.2~4.82.2~4.92.2~4.9産業系発生源쐍I)大気への排出製鋼用電気炉228.5139.9141.5131.1鉄鋼業焼結工程135.0113.8101.369.8亜鉛回収施設47.425.421.826.5アルミニウム合金製造等施設25.06623.16617.36616.566その他の業種22.764021.971914.030914.6977煙草の煙0.1~0.20.1~0.20.1~0.20.1~0.2自動車排出ガス1.611.611.611.61一般廃棄物焼却施設0.0440.0440.0350.035産業廃棄物焼却施設5.275.275.292.47쐍II)水への排出産業系発生源6.08255.60955.71154.7345下水道終末処理施設1.091.091.091.09共同排水処理施設0.1260.1260.1260.126最終処分場0.0930.0930.0930.056注排出量の単位:g-TEQ/yまた,反応管を用いた実験では,塩化フェノールを触媒とともに加熱することにより,PCDDsおよびPCDFsが容易に生成することも確認されている.さらに塩化ベンゼンもPCB を経由してPCDFsを生成しやすい.これらの事実から,塩化フェノール,塩化ベンゼンを前駆物質(precursor)と呼び,生成の重要な経路と位置づけることが多い.ただし,塩化ベンゼン,塩化フェノールは,ダイオキシン類が生成する条件では急速に分解するので,実際に環境中に排出されるダイオキシン類がどれだけ前駆物質を経由しているかは,現在のところ確認されていない.安定同位体でラベルした炭素をフライアッシュとともに加熱する実験では,PCDDsは単一環まで分解してから縮合して,ジベンゾダイオキシン構造を獲得し,PCDFsは炭素中に存在するベンゼン環2個以上連なった構造から,直接ジベンゾフラン構造が生成したことを示す結果が得られた.これが普遍的な事実であれば,PCDDsとPCDFsの生成経路は異なることになる.また,ダイオキシン類は,塩素が置換していなければ毒性はない.したがって,生成反応における塩素の挙動は重要であるが,これについても情報が不足している.最近の研究では,すすなどから酸化反応によりPCDFs構造が分離するときに,塩素が付加されると見られる結果が示されているが,ダイオキシン構造に対する直接塩素化,脱塩素化も当然起きており,実際に生成反応のどの時点で塩素化が進むのか,言い換えればどの時点で毒性をもつのかは,さらなる研究を待たなければならない.b. 抑制ダイオキシン類抑制技術は,大まかに生成抑制と排出抑制に分かれる.生成抑制では,以下の対策がとられている.(1) 燃料,原料の選別により塩素含有量を減らす.(2) 燃料の粒径,組成を均一にする.(3) 燃焼器構造,運転制御方法を改善して,完全燃焼に近づける努力をする.(4) 高温での滞留時間をできるだけ延ばして,生成したダイオキシン類の分解を促進する.(5) 排ガスを急冷してダイオキシン類生成温度の窓をできるだけ早く通過させる.(6) 排ガス流路内の灰を掃除して,灰の滞留時間をできるだけ短くする.一方,排出抑制のためには,ダイオキシン類の付着しているすす,フライアッシュなどを分離するため,固気分離装置の高性能化表3・8
