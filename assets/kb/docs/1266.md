# 3・4・6 濃度換算,排出係数と毒性換算

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

a. 濃度換算燃焼排ガス中の各種ガスの濃度は,vol.%またはppmで表示されることが多い.ばいじんはg/m(乾きガス)で表される.燃焼後,排ガスが空気と混合する条件では,測定される濃度は低下するため,排ガス中の酸素濃度が等しくなるように換算して表示しないと比較ができない.特に大気汚染防止法などでは,希釈により規制を逃れることがないように,装置ごとに標準的な残存酸素濃度が決められており,それに換算して濃度表示をすることになっている.排ガス中の酸素濃度を욦(%),換算すべき標準酸素濃度を욈(%),測定されたガス濃度またはばいじん濃度を욦(ppmあるいはg/m)とすると,換算濃度(ppmあるいはg/m)は=21-욈21-욦욦で計算できる.욈は,装置の形式,規模によって異なり,大気汚染防止法施行令に定められている.例えば,石炭燃焼ボイラは6%,重油専焼ボイラは4%,産業廃棄物焼却炉は12%,ガスタービンは16%などとなっている.b. 排出係数燃料単位量当たりの大気汚染物質の排出量を排出係数(emission index)といい,g/kg fuel,kg/kL fuelなどが用いられる.これを用いれば異なる燃焼条件や装置の排出特性の比較が可能となるが,こうした使い方はあまりされていない.燃焼装置の使用燃料量を乗じれば大気汚染物質の排出量が算出できるため,COやダイオキシン類など,現在全体排出量が問題となっている汚染物質について使用されることが多い.特定の業種あるいは特定のプロセスからの排出総量を知りたい場合には,調査して代表的な排出係数を決めておき,それに業種全体(あるいはプロセス全体)の燃料消費量を乗じることにより,排出総量を容易に推定することができる.例えば,地球温暖化物質であるCOの排出量を推定する場合には,プロセスごとの測定データから排出係数を決めておき,全産業における当該プロセスの燃料消費量から排出量を推定する.c. 毒性換算ダイオキシン類など残留性有機汚染物質(POPs: persistent organic pollutants)の多くは,毒性の異なる多くの異性体の混合物であるので,毒性の影響を論じる場合は単純に分析値(各異性体ごとに数値が出てくる)を加算しても意味がない.ダイオキシン類の場合は,最も毒性の強い異性体である2,3,7,8-TCDD(2,3,7,8-tetrachlorodibenzo--dioxin)の毒性を1とした場合の各異性体の相対毒性を定めておき,その毒性等価係数(TEF: toxic equivalency factor)を実測濃度に乗じて加算し,2,3,7,8-TCDD 毒性等量(TEQ: toxic equivalence)として表示する.すなわち,測定した試料と同じ毒性を有する2,3,7,8-TCDD の量がTEQである.TEFの値自体は研究の進展に伴い変わってきたが,現在は表3・9に示すWHO-TEF(1998)を用いることになっている.なお,法律やJIS(分析)においてはこの換算を行うことが定められているが,あくまでも毒性換算であるので,物理的な生成,消滅などを評価するには適さないことに留意すべきである.
