# 3・4・7 汚染物質の拡散

> **章**: 第3章 燃焼
> **節**: 3・4 燃焼に伴う環境汚染とその抑制

大気汚染のヒト、生物などへの影響を知り、適切な対策を立てるためには、排出量と環境濃度の定量的な関係を把握することが重要である。大気中での汚染物質の拡散は、風向、風速、風の乱れ、気温の鉛直変化などの気象条件と地形条件により大きく変化する。このような複雑な条件を組込んで拡散結果を予測することは、通常は容易ではない。大気の流れが単純で、汚染物質の大気中での変化がなければ、簡単な拡散方程式を適用して汚染物質濃度を推定することが可能である。例えば、平坦な場所で大気状態が安定しており、汚染物質も安定で変化しにくいときは、煙突から排出された汚染物質は、まず煙突から出た煙の運動量と煙と大気の温度差による浮力で大気中を上昇し、同時に風に流されつつ拡散する。煙はいったん上昇した後、拡散により煙突からの距離に応じて地表での濃度(着地濃度)を増していく。排出された汚染物質の質量と拡散条件が同じであれば、高い煙突から排出された汚染物質の着地濃度は、低い煙突に比べて必ず低くなり、最大着地濃度の位置は遠くなる。しかし、同じような汚染物質の拡散を定量的に考える場合でも、対象となる空間スケールは数mから数千km、時間スケールは数分から数年と幅広いため、拡散過程を統一的に扱うことは通常は困難である。例えば、複雑な地形や構造物の周辺での拡散などは、風洞を用いた実験や、現地でのトレーサ実験、あるいは拡散方程式を数値的に解く方法によらなければ定量的な答えは出てこない。また、地球規模の拡散であれば、成層圏まで含めた大気循環や、森林、海洋などの吸収、放出を考慮しなければならない。さらに、燃焼に伴う大気汚染物質は、多くの場合、拡散に加えて地表への沈着、雨による除去、化学反応による組成の変化などが濃度を変化させるため、目的とする場所での濃度予測には、これらの5―69過程も考慮する必要がある。例えば、燃焼排出源での窒素酸化物は、主としてNO,NOでありNOの割合が高いが、大気中で酸化してNOになり、大半は地面や海面に吸収される。しかし、一部はNOに変わり水分を吸収して硝酸エアロゾルになる。拡散係数は大気中の大小さまざまな渦の不規則な運動に影響を受けるため、これを細かく計算に取り入れることは、理論的にはともかく、ある一定以上の地域を考える場合には困難である。したがって、拡散係数と乱れの統計値との関係を実験的、経験的、理論的に求めておき、これを拡散方程式に取り入れて計算することが多い。さらに、大気の運動としては、気温こう配による対流、地表面摩擦による風速こう配、風向変動、混合層、逆転層などの大気構造、海陸風などを考慮する必要があり、こうした条件をどこまで取り入れるかにより計算の精度と計算量が異なってくる。実際の拡散計算では、条件を単純化した多くの拡散モデル(diffusion model)が提案されており、これらの中から適切なものを選択し、場合によってはいくつかを組合わせて予測することになる。注意すべきは、それぞれの拡散モデルはある前提条件のうえで成立しているので、その前提条件からはずれる場合は当然ながら適用できない。また、モデルに投入する拡散係数、気象条件なども、適切なものを得る努力が必要である。例えば、気象条件は近隣の気象台、消防署などから得られるが、そのほかにも可能な限りのデータを集める必要がある。以下に代表的な拡散モデルについて簡単に説明する。表3・9ダイオキシン類の毒性等価係数(TEF)(a) プルームモデル(plume model)プルームとは連続して流れる煙の形状のことである。汚染物質が定常一様流中で煙として風方向に移動しており、それと直角方向,に拡散する場合を取扱っている。計算は簡単で、主として定常な場合に有効である。煙が移動する必要があるので無風時には使えない。b. パフモデル(puffmodel)パフは一塊りの煙のことである。連続的に流れる煙を、一定間隔で流したパフの行列で表す。微風や無風の場合、非定常な場合にも適用できる。平坦地に適用する。c. ボックスモデル(box model)一つのボックス内では濃度は一定とし、いくつかのボックスをつなげて現象を表す。非定常場での濃度、時々刻々の濃度を求めるのに適している。拡散パラメータはボックスとボックスの境界面で汚染物質の出入りの程度を表す係数として存在する。地球温暖化などまで含めて広域予測に適しており、長期平均、日平均など時系列値を求めることもできる。モデルを構築するときのボックスの設定箇所、設定数がその精度を大きく左右するため、計算結果の精度について十分な検討が必要である。
