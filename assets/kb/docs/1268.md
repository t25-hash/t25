# 10・1・3 カオスへのルートおよび判定法

> **章**: 第10章 分岐現象とカオス
> **節**: 10・1 概説

で一つのアトラクタになる．
このように急激にストレンジアトラクタ（10･3･1項参照）が大きくなることをカオスの危機（crisis ofchaos）（あるいは爆発）という．
この場合のカオスの危機はサドルノード分岐から生じる不安定な3周期解が3バンドアトラクタと接触するとき発生する．
二次元2パラメータ写像として，3階のローレンツ微分方程式を簡単にモデル化したエノン写像（Henon map）がある．
なお，分岐の過程で，どのような不動点がどのような順序で現れるかなど系の挙動を記号列を用いて表す記号力学がある．
動点や周期状態が不安定化し，突然発生するカオスなどがある．
また，カオスの判定法として，振動波形の非周期性，ポアンカレ写像によるストレンジアトラクタの存在，パワースペクトル密度の幅広い周波数分布，リアプノフ指数（10･3･1項参照)，フラクタル次元，不変多様体におけるホモクリニック点の存在などが挙げられる．
