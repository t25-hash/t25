# 4・1・4 熱通過

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・1 伝熱の基本形態

熱交換器などの実用熱機器では、固体壁で隔てられた高低二流体間の熱移動を議論することが多い。この形の熱移動を熱通過(overall heat transfer)と呼び、熱輸送量は次の形で表される。=-(4・5)ここに、:単位時間当たりの総熱移動量、:伝熱面積、:熱通過率(総括熱伝達率)、:高温流体の温度、:低温流体の温度である。熱通過は、固体の両面における熱伝達と固体隔壁内の熱伝導から成っており、熱通過率はその総合的な熱コンダクタンスを表す。なお、円管やフィン付き面のように固体隔壁の両面で面積が異なる場合は、採用する面積によって熱通過率も異なってくるが、通過熱量の値は変わらない。
