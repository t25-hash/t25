# 4・2・1 熱伝導の基本

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・2 熱伝導の基本

物体の温度(°C)の微小面δ(m)を単位時間当たりに通過する熱量(J)は、熱の流れる方向(微小面の法線方向)に沿った温度こう配に比例して、式(4・6)のようになる。比例定数λは熱伝導率[W/(m・K)]と呼び、実験により求められる物質固有の値である。熱伝導とは、分子どうしの衝突により、分子運動エネルギーが拡散し、高温部から低温部へと運動エネルギーが伝わる現象を指す。基本的には、分子、原子の並進、回転、振動エネルギーにより伝わる。したがって、一般的には、分子間平均自由行程が小さい順から、固体、液体、気体の順で熱伝導率の値が減少していく。一般的な物質の熱伝導率の温度依存性(大気圧)を図4・1に示す。ただし、後述するように、良導体の場合は、自由電子によるエネルギー伝ぱの寄与のほうが大きくなる。ただし、は定圧比熱[J/(kg・K)],μは粘性率(Pa・s)、γは比熱比である。希薄気体の場合は、平均自由行程(mean free path)が大きくなり、分子どうしの衝突よりも容器壁面に直接衝突する際のエネルギー吸収のほうが支配的となる。固体の場合は、導電性によりエネルギー伝ぱの支配的な要素が異なる。非金属で結晶体の場合には、結晶格子の多自由度振動によるエネルギー伝達の寄与が大きく、式(4・9)のようになる。非金属で非結晶体の場合は、熱弾性波の散乱が極めて強く、熱伝導率の値は低くなる。純金属から成る良導体の場合には、自由電子によるエネルギー輸送のほうが支配的になり、式(4・10)に示すヴィーデマン・フランツ・ローレンツ(Wiedemann-Franz-Lorenz)の法則がほぼ成立する。低温域では分子間距離図4・1 各種物質の熱伝導率(大気圧)図4・2 座標のとり方5―73が縮まるためいったん熱伝導率は上昇するが、極低温域に入ると非結晶体の格子構造による特徴が現れ、熱伝導率は急激に低下する。電気抵抗が低くなる超伝導(超電導)物質の場合もその例外ではない。液体金属の場合は結晶体でないため、固体金属よりは熱伝導率は低くなる。また、合金など異物質を含む場合には、値は式(4・10)からかなりずれる。通常の断熱材は構造上多孔質体となっており、固体どうしの接触熱抵抗が大きく、狭い空間では気体に流れが生じず熱伝導が支配的となるため、断熱性能が高い。電気回路と同等に扱うことができ、複合物質などの場合に広く用いられる。また、熱抵抗の逆数を単位面積当たりで割った値を熱通過率[W/(m・K)]と呼ぶ。
