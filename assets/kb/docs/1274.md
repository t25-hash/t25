# 4・2・2 熱伝導方程式

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・2 熱伝導の基本

フーリエの法則[式(4・6)]を使って、物体まわりの熱伝導の三次元微分方程式を立てると、式(4・13)のようになる。ρ옿옿=옿옿λ옿옿+옿옿λ옿옿+옿옿λ옿옿(4・13)ただし、は比熱[kJ/(kg・K)]である。対象温度範囲内では物性値に温度依存性がないとすれば、式(4・13)は、옿옿=∇(4・14)となる。ただし、は温度伝導率(=λρ,thermal diffusivity,m/s)、∇は、直交、円筒[図4・2(a)]、極座標[図4・2(b)]それぞれ、∇=옿옿+옿옿+옿옿(4・15)∇=옿옿+1옿옿+1옿옿φ+옿옿(4・16)∇=옿옿+2옿옿+1옿옿φ+cos φsinφ옿옿φ+sinφ옿옿ψ(4・17)である。単位体積、単位時間当たりに一様な発熱(W/m)がある場合は、옿옿=∇+ρ(4・18)(a) 円筒座標(b) 極座標5―74となる。なお、定常問題、一次元問題、二次元問題など上式中不必要な座標が存在する場合は、その座標による微分項を省略すればよい。
