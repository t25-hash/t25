# 4・2・3 温度伝導率

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・2 熱伝導の基本

温度伝導率は,定常状態では考慮する必要はないが,非定常状態では重要な役割を果たす物性値である.式(4・13)から,微小物体の温度変化は熱伝導率λに比例し,その微小物体の熱容量ρに反比例する.したがって,温度伝導率とは温度波の伝ぱ速度に関係する値であって,この値が大きいほど物体の温度変化は顕著になる.ここで,同様な物性値として熱浸透率(thermal effusivity)がある.熱浸透率(=λρ,J・s・m・K)は,非定常熱伝導系における熱の伝わりやすさを示す.つまり,温度伝導率はその値が高いと温度こう配が時間とともに減少してしまい,熱の伝わりやすさは減少する.熱浸透率とは,温度が変化するよりどれだけ速く熱を伝えるかを表している.
