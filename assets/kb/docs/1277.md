# 4・2・5 無次元数(表4・1)

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・2 熱伝導の基本

a. フーリエ数式(4・14)の各変数を式(4・20)のようにおき、式を無次元化すれば、式(4・21)を得る。b. ビオ数固体表面から熱伝達率一定で熱が伝わる系を考えてみる。式(4・22)の各変数を式(4・23)のようにおき、式を無次元化すれば、式(4・24)を得る。ビオ数は固体表面における固体側の熱抵抗と液体側の熱抵抗の比を表す。c. ステファン数式(4・19)の各変数を式(4・25)のようにおき、式を無次元化すれば、式(4・26)を得る。ステファン数は固液界面における顕熱と潜熱の比を表す。
