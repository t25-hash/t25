# 4・4・4 表面のふく射性質

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・4 ふく射伝熱の基礎

a. 灰色体の仮定と完全拡散の仮定物質のふく射性質は,一般にふく射の波長と方向に強く依存する.しかし,伝熱の評価では,特に表面の特性について,これらの点を無視する仮定をすることがある.第1には,放射率(emittance)や反射率が波長に依存しないとする灰色体(gray body)の仮定がある.この仮定をおくと,ふく射性質は全放射率や全反射率などの波長積分量で代表される.第2には,物質はどの方向にも一様な強度のふく射を放射し,さらに,そこに強度の方向分布をもって入射するふく射を,一様な強度のふく射と仮定して半球的に反射するとする完全拡散性表面(perfect-diffuse surface)の仮定がある.この仮定をおくと,ふく射性質は半球放射率εや半球等強度入射半球反射率(c項参照)で代表される.b. 放射率特定の温度と波長λに注目すると,実在の物質が放射するふく射の強度λ,はどの方向についても黒体の場合の強度λ,よりも小さい.この強度の比,ελ,=λ,λ,(4・53)を指向放射率(directional emittance)と呼ぶ.波長積分量についての強度の比,ε=(4・54)を全指向放射率と呼ぶ.これらの放射率は放射されるふく射の方向に依存する.特に,表面に垂直な方向への値を垂直放射率(nor-mal emittance)εと呼ぶ.半球量については,半球放射率(hemispherical emittance)と真空中への全半球放射率をそれぞれ次のように定義する.ελ,=λ,λ,(4・55)ε=σ(4・56)c. 反射率・透過率・吸収率物質に入射するふく射は,多くの場合,物質の表面や内部で非等方的に散乱されるので,反射・透過されたふく射の強度・エネルギーは方向分布をもつ.そのため,反射率(reflectance),透過率(transmittance),吸収率5―77(absorptance)はさまざまに定義されうるが,熱工学のエネルギー収支の評価のために重要であるのは次のものである.すなわち,表面に垂直な方向からふく射が入射するときに,半球方向に反射・透過されるエネルギー割合を垂直入射半球反射率,垂直入射半球透過率,吸収される割合を垂直入射吸収率と呼ぶ.半球方向にわたって強度が一様なふく射が入射するときに,半球方向に反射・透過されるエネルギーの割合を半球等強度入射半球反射率,半球等強度入射半球透過率,吸収される割合を半球等強度入射吸収率と呼ぶ.このような定義の反射率・透過率・吸収率については,次の式が成立する.++=1++=1(4・57)吸収が強く不透明な表面については,次の式が成立する.+=1+=1(4・58)(反射率)+(透過率)+(吸収率)=1という関係は,このように適切に定義された反射率・透過率・吸収率の間でのみ成立する.d. キルヒホッフの法則ふく射の放射と吸収の間には,キルヒホッフの法則(Kirchhoff’s law)が成立する.この法則は,熱平衡状態にある媒質ではふく射の放射と吸収のエネルギーがいずれの波長・方向でも平衡すべきであることを述べる.実用的には,ある方向への(分光指向)放射率はその方向から入射するふく射に対する(分光指向)吸収率に等しいと表現される.吸収が強く不透明な表面については,次の式が成立する.ε==1-ε==1-(4・59)ここで,この法則が前提とする熱平衡条件は一般に熱工学の系では成立しないが,実際上,この法則が成立するとみなして支障はない.しかし,この法則が分光量・指向量を対象にすることは重要である.この法則が全放射率や半球放射率について成立するためには,灰色体,完全拡散性表面などの強い仮定が必要である.e. ふく射性質のデータ多くの物質のふく射性質の実験値がデータ集に集積されている.しかし,その値を用いるときには注意を要する.第1に,放射率や反射率の定義が,上述のように一意的ではないことが多い.特に,反射率については式(4・59)の関係を満たすものであるや否やについて注意を要する.第2に,物質が多様であることである.ふく射性質は,物質の内部の物理的・化学的組成のみならず,表面のあらさや被膜の状態に強く依存する.放射率が表面状態に応じて0.1から0.9に変化することもまれではない.
