# 4・4・5 完全拡散性表面間のふく射伝熱

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・4 ふく射伝熱の基礎

吸収が強く不透明な完全拡散性表面からなる閉空間におけるふく射伝熱を評価するには、次の方法が有効である。a. 形態係数二つの表面が図4・7に示すように対向するとき、表面が放射するふく射エネルギーのうち、表面に達するものの割合は形態係数(configuration factor)と呼ばれ、次の式で表される。5―78熱工学図4・8 閉空間におけるふく射伝熱図4・7二つの表面間のふく射エネルギー交換図4・9半透過性媒質におけるふく射輸送このは自己形態係数と呼ばれ、凹面部をもつ表面では零でない値をとる。式(4・60)の積分は、工学系で重要な多くの幾何関係について計算され、簡単な式の形で表示されている。また、式(4・61)(4・62)の相反則、総和則を用いると、複雑な幾何関係についてより簡単に形態係数が求められることもある。b. 表面間のふく射伝熱図4・8で、個の完全拡散性表面=1,2,...,の温度や放射率はたがいに異なる。この半球的な(反射)+(放射)のふく射流束を表面の射度(radiosity,W/m)と呼ぶ。系は個の表面からなるので、これらの式は計2式ある。
