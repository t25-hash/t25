# 4・5・1 ガス体の放射理論

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・5 熱放射による伝熱

ガス体分子による電磁波の吸収,放射は分子の定常状態間の遷移によって行われるものであり,電磁波の振動数をν(s)とするときν=욊-욆(4・79)なる関係がある.욊,욆は遷移に関与する上の定常状態および下の定常状態に対応するエネルギー準位であり,はプランクの定数である.熱放射で取扱われる赤外線領域の電磁波の振動に関連しているのは,分子内の振動-回転運動である.光を吸収,放射する遷移が起きる条件は,振動量子数,回転量子数に関し,=±1,=±1が基本であり,=0が含まれる場合もある.この結果として理想的な場合には図4・10に示すように中心波数ω=ν(:光速)の両側に一定の間隔をもってスペクトル線が並び,全体としてバンドを形成することになる.一般の場合には左右対称となることはなく,高波数側の広がりのほうが小さい.このような振動-回転遷移によるスペクトル線あるいはバンドの強度は,振動によって分子の双極子モーメントが変化する場合の変化の強さと,各遷移の頻度によって決まる.バンド内の個々のスペクトル線の強度の和をバンド強度(band intensity)あるいは積分吸収係数(integrated absorption coefficient)αと呼んでいる.対象とするガス体の振動が調和振動とみなせる場合には,バンド強度は温度によらず一定であり,室温でその値を計測すれば十分であるが,非調和振動の場合には温度とともに増大するのがふつうであり,高温においては実験的な確認を必要とする.個々の振動-回転スペクトル線の形と幅は,工業的に熱放射を取扱う範囲においては分子間の衝突による圧力広がりによって支配され,その広がりの程度は圧力に比例し,温度の平方根に逆比例するとして取扱われる.また高温になると,エネルギー準位の高い状態にあるガス分子の数が増大するので,図4・10のバンドの中心波数より両側に離れた位置でのスペクトル線強度が極大を示し,また全体のバンド幅(band width)も増大する.吸収バンドの典型的な例としてNO分子の基本バンド(5.3 m)における吸収係数[図た値を示している]の温度依存性を図4・11に示す.各曲線は,圧力広がりによって有限の幅をもつことになった個々のスペクトル線が互いに重なることで作られる包絡線である.上述のように,バンド中心の両側に吸収係数の極大を示す波数があり,温度上昇とともにその極大値はバンド中心から離れ,またバンド幅も大きくなっている.このような多数のスペクトル線から成るバンドによる吸収,放射を取扱うには,1本1本の線について計算を行っていくのは無理で5―79図4・10 剛体回転子モデルに基づくスペクトル線の位置[ω:波数(cm)]図4・11 NO分子基本バンドの吸収係数図4・12 等価バンド幅욁(cm)あり,バンドモデル(band model)と呼ばれる手法を導入して計算を簡略化する.バンドモデルには狭域バンドモデルと広域バンドモデルがある.狭域バンドモデルは吸収バンド内のさらに細かく分割された領域を扱うものであり,スペクトル線が等間隔で並んでいると仮定したエルザッサモデル(Elsasser model)と,ランダムに並んでいるとしたマイヤー・グーディモデル(Mayer-Goodymodel)がある.前者は所与の圧力,温度,組成のガスの単色吸収係数の波長分布を与え,後者は,これらの条件と所与の厚さをもつ気体塊の単色放射率-吸収率の波長分布を与える.これらの狭域バンドモデルは,バンド形状の正確な記述,非均質ガス体の取扱いなどにおいて有力な手段となる.一方,広域バンドモデルは各吸収バンドをそれぞれ一括して扱おうとするものであり,実測データを参考にして図4・12に示す等価バンド幅(equivalent band width)욁욁=윩율윰윫ωdω (cm)すなわち黒体放射相当幅を対象気体の分圧,全圧,光路長,温度の関数として数式化する.この値はバンド吸収(band absorption)とも呼ばれる.その典型的かつ便利なものが指数型バンドモデルで,各気体の各バンドについて計算に要するパラメータが与えられているので,욁を計算することができ,この욁にそのバンドの代表波数ω(cm)における単色黒体放射エネルギー욁욌[kWmsrcm]を掛ければバンド放射エネルギーを計算で5―80熱工学き,さらに全バンド(:バンド番号)を考慮した全放射率εはバンド放射エネルギーの合計よりε=π∑욁욌ω욡,욁욡σ(4・80)で計算される.
