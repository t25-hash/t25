# 4・5・2 ふく射輸送理論と輸送式

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・11 では、4・5・2項で使用している吸収係数(m)を圧力で割っ

a. ふく射輸送熱放射によるエネルギー輸送は、電磁波による遠隔的な伝ぱである。熱伝導(対流をも含む)は、分子などの並進・振動・回転などによるエネルギーの拡散的な伝ぱであり、両者は輸送形態が全く異なっている。ふく射性ガス(radiation participating gas)を含む系のふく射伝熱においては、ふく射により輸送されるエネルギーの釣合い式(ふく射輸送式、radiative transfer equation)を考えなければならない。しかし、ふく射に透明なガス(transparent gas)の系においては、熱ふく射の輸送式を考える必要はなく、4・4・2項に記されている固体表面間のふく射熱交換よりふく射伝熱量を求めればよい。b. ふく射輸送式ふく射性ガスにおけるふく射輸送式は、図る方向のふく射エネルギーの釣合いを考えることにより次式で表される。なお添字λは単色[単位波長幅(m)当たりの値]を意味する。この輸送式の右辺第1項は、ガスによる吸収と散乱による減衰、第2項はこの体積要素中の自己放射による強度増加、第3項は他方向(′)へのふく射が方向に散乱されることによる強度増加を表す。
