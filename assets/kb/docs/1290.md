# 4・5・3 ふく射輸送式の解法

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 に示すガスの体積要素(単位断面積、長さd)を通過してい

a. ふく射熱流束の近似解ここでは2枚の灰色平行平板(温度,、放射率ε,ε)間の均一ガス層中のふく射伝熱を考える。. 光学厚さの薄い場合系を代表する光学厚さτが薄い(τ≪1)場合には、上記輸送式の近似解として、2平板間の単色ふく射熱流束を下記のように求めることができる。これは間のガスがないときの2平板間のふく射伝熱量に等しい。したがって、βは各位置での温度の関数となる。ふく射伝熱は、下式で定義されるふく射熱流束ベクトル(radiantheat flux vector)の発散の形でこの中に入っている。ただし、式(4・88)中に温度が入っているので、この手順は繰返し計算が必要となる。前者の、まず各方向の強度を求める手法には、DO(discreteordinate)法に代表される各種のフラックス法(2流束法、6流束法)があり、c項で後述するように輸送式の微分形を用いて各方向の強度を求める。後者の、各位置でのふく射吸収量を直接求める手法には、モンテカルロ法、ゾーン法、REM法等があり、輸送式の積分形を用いて要素間で伝達されるふく射エネルギー量を直接求める。この2者の中間的な手法のDT(discrete transfer)法、ふく射光線追跡法は、積分形の輸送式を用いてふく射束が各ガス要素を通過するときの強度変化を求め、この変化を各ガス要素ごとに多数のふく射束について集計することにより、各ガス要素のふく射吸収量を求める。以下、現在各種のはん用熱・流体解析コードに組込まれて広く使用されているDO法、モンテカルロ法、DT 法について、その基礎式と原理について述べる。c. DO法(discrete ordinate method)空間の各位置でその周囲の4πの立体角を図4・14のように複数の微小立体角に分割図4・13 ふく射輸送式の座標すると、その各方向に関する輸送式は散乱を無視し、灰色ガスの仮定をおくと、式(4・81)より下記のように表される。これは、この面の自己放射とこの面への入射ふく射の反射の和となる。ここで壁面が灰色とすると、乱射性が仮定できるので、この面から出て行くふく射の方向強度は下式で表される。この結果をエネルギー方程式に入れて、系内の温度分布を求めるためには、式(4・88)のふく射熱流束ベクトルの発散の値を、下式により空間の各位置で求める必要がある。このそれぞれを1個当たり(kJ)のエネルギーを保有する粒子に分割して放射させる。したがって、ガス要素からはすべての方向に微小立体角Ω当たり下記の個数のエネルギー粒子を等確率で放射することになる。吸収係数が一様な系では、これを満足する各粒子の飛距離は下式で確率的に決定することができる。粒子がこの距離飛ぶ前に壁面に当たると、その粒子はその壁の放射率εの確率でその壁に吸収させ、1-εの確率で、壁面の状態により乱反射あるいは鏡面反射させる。反射後の飛距離は、粒子の反射前後の全飛距離が最初に与えられた式(4・99)の値となるまで飛ばせる。このようにして各要素から放射された多数のエネルギー粒子の終点を求め、各要素ごとに受取った粒子数をカウントし、それに粒子1個当たりのエネルギーを掛けると、その要素の吸収したふく射エネルギー量となる。よって、エネルギー方程式に入れるための式(4・88)のふく射熱流束ベクトルの発散の値は、下式により空間の各位置で求めることができる。点Pへの入射ふく射熱流束は、各ふく射束のもたらす熱流束の和をとった下式で表される。また、点Pから出て行くふく射熱流束は、の反射と、この壁面からの自己放射の合計をとった下式で表される。ここで、はこのふく射束上のガス要素番号、δは図4・16に示されるように、この番めのガス要素中でのこのふく射束の光路長である。この要素の正味の全ふく射吸収量は、この要素を通過するすべてのふく射束の正味吸収量の和として求まる。またこれは、エネルギー方程式に入れるための式(4・88)のふく射熱流束ベクトルの発散の値そのものである。
