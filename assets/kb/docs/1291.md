# 4・5・4 火炎等の熱放射

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 に示すガスの体積要素(単位断面積、長さd)を通過してい

不輝炎(nonluminous flame)からの熱放射はHO,CO,COなどのガス体によるものであり、それぞれの気体の放射率ε,ε,ε等は分圧、光路長、温度がわかれば、4・5・1項に述べた手法あるいは文献(9)を用いて算定することができる。液体燃料噴霧燃焼の場合、気体燃焼の過少一次空気燃焼の場合、あるいは微粉炭燃焼の場合などには黄橙色に輝く火炎が見られ、これを輝炎(luminous flame)と呼んでいる。輝炎においては、HO,CO等のガスによるバンド放射と火炎中に析出、浮遊している固体微粒子群による連続スペクトル放射とが共存し、その放射率は不輝炎のそれに比べてかなり高い。気体燃料や液体燃料を燃焼させる場合には主成分が炭素で、ほかに水素、酸素などを含むスート粒子、コーク粒子が浮遊し、微粉炭燃焼の場合には微粉炭粒子が火炎の前半に、灰粒子が火炎の後半に浮遊する。吸収係数は実験的あるいは理論的に定めなければならないが、種々の因子の影響を受け、簡単な式では表せない場合もある。体積濃度も当然のことながら燃焼条件に支配される量であり、結局のところ豊富な実験を基にして放射率を予測する以外にない。液体燃料噴霧燃焼輝炎の放射率に及ぼす燃焼条件の影響としては、空気過剰率が小さく、燃料噴霧推力が小さく、燃料の炭水素比()が大きく、また燃焼室圧力が大きいほうが、いずれも放射率を高める。入口空気温度、燃料温度、噴霧用流体の種類、燃料流量、スタビライザの影響はいずれも小さい。これらはいずれも火炎全体の平均値を示している。燃焼室圧力の影響を計算することも可能である。また、ふく射伝熱の数値解析では、4・5・2項に示されるように火炎の吸収係数が必要となるが、液体燃料噴霧燃焼火炎については吸収係数の実験式が、空気過剰率と燃料比重の関数として提案されている。
