# 4・6・1 対流の種類と熱伝達率

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・6 対流熱伝達の基本

機器の伝熱解析をしようとすると、理論的には伝導、対流、ふく射が共存して、それらを合わせて解析する必要がある。しかし、ふく射が重要な場合は、4・4節で述べたように、それに伴う物体表面間の熱移動を別途追加すればよい。ふく射伝熱は、本節で述べる対流熱伝達に対して、直接的な影響はないと考えて差支えない。対流熱伝達(convective heat transfer)とは、流体(気体、液体など)に蓄えられたエネルギー(内部エネルギー)が、流れによって運ばれることによって生じるエネルギー伝達のことである。したがって、輸送による伝熱量は、流体の速度などによって大きく変化する。ポンプや送風機などで強制的に流れを作ると、伝熱量は当然大きくなる(強制対流という)。人工的に流れを作らなくても、多くの液体や気体は温度が周囲より高くなると密度が小さくなって浮力による上昇流が自然に発生し、その流れとともにエネルギーは輸送される[自然対流または自由対流(free convection)という].また、強制的な流れの速度が小さいとき、あるいは加熱面の温度が非常に高いときには、強制対流と自然対流が共存したような状況が発生する[共存対流または複合対流(mixed convection)という].流体が図4・18のように、物体表面に沿って流れるとき、壁面では流体は動かず流速が零となるために、物体表面近傍の流体は粘性の作用により減速される。しかし、この速度の急激な変化は、物体表面に接する厚さがδの限られた薄い層内で起こることが知られている。この層の外側では、流体はあたかも非粘性流体のような挙動をする。そこで、この薄い層を境界層、その外側を主流(mainflow,freestream)、また厚さδを境界層厚さ(boundary layerthickness)と呼んでいる。厚さδは、流速、流体の粘性の大きさ、物体の形状等で変化する(4・7節参照)。一方、主流の流体温度が、物体表面の温度が욌(>)とすると、物体表面近傍の流体は加熱され、表面に接する厚さδ욧の層内で流体温度は急激に変化する。この層を温度境界層(thermal boundary layer)、またその厚さδ욧を温度境界層厚さという。前述の境界層を、この温図4・18 速度境界層と温度境界層度境界層と区別するために、速度境界層(hydrodynamic bound-ary layer)と呼ぶことも多い。ここで、速度境界層厚さδと温度境界層厚さδ욧は、一致するときもあれば、また全く違う場合がある(4・7節参照)。次に、熱伝達率[Wm・K]の解析的な求め方を示す。熱伝達率の定義式は、一般に次式で与えられる。욌==욌-(Wm)(4・113)ここで、(m)は加熱面(または冷却面)の表面積、(W)はを通して単位時間当たり流れる熱量である(ふく射による分は差引かれている)。욌のことを壁面熱流束(wall heat flux)という。しかるに、図4・18の翼表面のように伝熱面上では流体は静止しているから、流れによって運ばれるエネルギーはない。すなわち、伝熱面と流体間の熱移動は、流体の熱伝導のみに基づいている。したがって、伝熱面における流体の壁垂直方向温度こう配が求まれば、次式のフーリエの法則より熱流束욌が算定できる。욌=-λ옿옿욌(4・114)ここで、添字は壁面上での値を示す。λは流体の熱伝導率、は壁面垂直方向の座標である。式(4・113)(4・114)より熱伝達率が求まる。式(4・114)は壁面での一般式であるが、壁垂直方向温度こう配を求めるためには、温度境界層内の流体の温度分布=,,を求める必要がある(図4・18)。ところが、流体の温度分布は、流れの状態や流体の種類によって変化する。そのため、対流による伝熱量を求めるには、速度場の状態と直結させて温度場の状態を求める必要がある。これらを支配しているのが、4・6・3項に示す連続の式、運動方程式(ナビエ・ストークス方程式、Navier-Stokes equations)およびエネルギー方程式である。
