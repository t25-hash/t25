# 4・6・2 層流と乱流の熱輸送

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・6 対流熱伝達の基本

流れによってエネルギーが輸送される対流は,当然流れの状況に強く依存する.流れは,大きく分けると層流と乱流の2形態がある.レイノルズ(Reynolds)の実験(1883年)で明らかになったように,管内の流速を上げていくと,整然とした流れ(層流)は乱れた流れ(乱流)に遷移する.乱流中にはさまざまなスケールの渦が混在し,この渦によって,層流とは比較にならないほど大きな混合(mixing)と拡散が起こる.すなわち,運動量,熱,物質の輸送速度は乱流では一般に非常に大きい.この性質は工学的に重要で,対流伝熱を促進させるために,壁面に凹凸を付け人工的に大きな乱れを作り出すようなこともしばしばなされる.このように,層流と乱流は流動状況に大きな違いがあるため,対流伝熱は,後述のように層流熱伝達(4・7節)と乱流熱伝達(4・8節)に分けて通常議論する.
