# 4・6・1 項

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・6 対流熱伝達の基本

に述べたように、流体が物体表面に沿って流れるとき(図4・18)、速度と温度の急激な変化は、物体表面に接する厚さがδおよびδ욧の限られた薄い境界層で起こる。境界層の特徴を考慮して、前述の支配方程式(4・131)~(4・134)を以下に簡略化する。最も基本的な仮定は、速度と温度の境界層厚さδおよびδ욧が、解析対象物体の流れ方向の長さに比べて十分に小さいことである。すなわち、δ=δ≪1,δ욧=δ욧≪1が近似式を求める際の出発点である。このとき、速度、温度と座標の変化するオーダはたかだか次の程度である。~1, ~1, ~1~δ(速度場)、 ~δ욧(温度場)すなわち、連続の式(4・131)において옿옿のオーダは1である。式(4・131)が成立するには、옿옿のオーダも1でなければならない。~δであるから、この条件より~δ≪1,~δ≪1(速度場)、~δ욧≪1(温度場)となる。一方、式(4・132)において옿옿~1,옿옿~1δとなり、옿옿の項は、δ≪1のとき無視できる。同様に、式(4・133)のの式は、δのオーダの式となる。の式が1のオーダであるから、δ≪1より、の式は無視できる。これより옿옿0となり、の方向変化は無視できる[すなわち=].以上より、主要項を残した強制対流の場合の運動方程式は、無次元化前の形で書くと、옿옿+옿옿=-1ρdd+옿옿(4・142)となる。これを境界層方程式という。同様に温度境界層については、옿옿+옿옿=옿옿(4・143)となる。これを温度境界層方程式という。境界層方程式は、技術計算で広く用いられている。
