# 4・6・1 項

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・6 対流熱伝達の基本

に述べたように、流体が物体表面に沿って流れるとき(図4・18)、速度と温度の急激な変化は、物体表面に接する厚さがδおよびδの限られた薄い境界層で起こる。境界層の特徴を考慮して、前述の支配方程式(4・131)~(4・134)を以下に簡略化する。最も基本的な仮定は、速度と温度の境界層厚さδおよびδが、解析対象物体の流れ方向の長さに比べて十分に小さいことである。このとき、速度、温度と座標の変化するオーダはたかだか次の程度である。これを境界層方程式という。これを温度境界層方程式という。境界層方程式は、技術計算で広く用いられている。
