# 10・1・4 代表的な材料の静的強度

> **章**: 第10章 材料の強度
> **節**: 10・1 静的強度

代表的な金属材料の降伏応力（0.2%耐力)，引張強さおよび破断
ひずみを表10･2に示す．各種金属・合金の値にはかなりの幅があ
るが，これは個々の合金組成，加工・熱処理等の履歴により，異な
る値を示すことによる．延性材料の場合，前述のように降伏応力が
設計の際の重要な材料特性値となるが，同時に，破断ひずみも延性
の指標としての重要な材料特性値である．鋼の場合，熱処理（例え
ば，焼入れ・焼戻し）の温度・時間等の選択により所定の強度に調
整して使用するのが通常である．したがって，表10･2のような表
示は適切ではなく，各材種について図10･4に示すような，強度特
性に及ぼす熱処理の影響を示す必要がある．一般に，焼戻し温度の
上昇に伴い，引張強さと降伏応力は低下し，一方，破断ひずみは増
加し，靱性が高くなる．また，炭素鋼の機械的性質は炭素含有量に
依存することも知られており，およそ800℃前後から油焼入れした
炭素鋼の機械的性質の炭素含有量依存性は図10･5のようになる．
強度は試験温度および変形速度に依存し，材料によりその程度は
異なるが，一般に温度の低下，変形速度の上昇とともに，降伏応力
は増加する．
