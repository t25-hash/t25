# 4・8・1 乱流境界層および管内乱流

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・8 強制対流乱流熱伝達

固体壁による流体力学的拘束条件を受ける乱流を壁乱流と呼び、乱流境界層(turbulent boundary layer)、円管内あるいは平行平板間の乱流などが該当する。乱流境界層は、図4・26に模式的に示すように、一方で固体壁、他方で自由境界という異なった境界条件を有する点で、円管内や平行平板間などの内部流の乱流と異なるが、両者の壁面近傍の流れ場と温度場の構造は相似であるため、しばしば類似の解析手法が適用される。乱流境界層の壁面近くは、壁面上ですべての速度成分が零となる条件により支配されている。このような壁面の影響の著しい領域を壁領域(wall region)あるいは内層(inner layer)という。一方、境界層厚さの大半を占める外側の領域では、壁面は内層を隔てて間接的に応力を及ぼし、かつ慣性力が流体運動に支配的役割を果たすようになる。この領域を外層(outer layer)と呼ぶが、図4・26に示すように大規模な渦運動が存在して境界層の全般的形状を形成している。まず、壁面のごく近傍では、乱れは強く抑制されるため、せん断応力は主として分子粘性力によって担われる。=(4・180)式(4・180)は、0<<5でよく成立し、この領域を粘性底層(vis-cous sublayer)と呼ぶ。これは普遍速度分布とも呼ばれ、広いレイノルズ数の範囲でその成立が確かめられている。式(4・181)中の定数はフォン・カルマンの普遍定数(von Karman constant)と呼ばれ、実験によると=0.40~0.41である。図4・27に代表的な平板乱流境界層の測定結果を示す。式(4・181)は、30<でかつせん断力がほぼ一定とみなせる範囲で成立し、これを対数速度領域(logarithmic region)と呼ぶ。粘性底層と対数速度領域の間には、緩和層(buffer layer)、5<<30が存在する。以上の壁法則は、発達した円管内乱流などの内部流についてもよく成立することが知られている。定数については、乱流境界層では=4.9~5.1、内部流には=5.4~5.5が多く用いられる値である。上記の対数速度則、式(4・181)は、壁面を基準としてそこからの速度の増加を与えるものであるが、逆に、主流速度を基準とする場合、主流速度からの減少分は以下のように表され、速度欠損法則と呼ばれる。乱流境界層の外層は、図4・26に示したように渦度変動を伴う乱流域と非回転流れの非乱流域がごく薄い粘性表層(viscous super-layer)によって明確に区切られていて、速度を観測すると時間的に乱流、非乱流がある割合で現れる。この乱流の占める時間割合を間欠度(intermittency factor)γと呼び、その分布は図4・26のようになる。このように境界層の外層の構造は自由乱流との類似性が強いため、伴流領域(wake region)とも呼ぶ。管内乱流などでは、管中央でもγ=1でこのような間欠性はないため、代わりに乱流コア領域(turbulent core region)と呼んでいる。外層で、対数速度領域から主流の一様速度に移行する領域について、コールズ(Coles)は境界層流れを壁面乱流成分と伴流成分によって構成されるものと考え、対数速度領域より外側の領域を一括して以下のように表した。加熱された平板上の温度境界層の構造は、流体力学的なパラメータに加えてプラントル数に依存する。が1よりも大きくなると速度境界層に比して温度境界層は薄くなり、が1よりも小さくなると厚くなる傾向がある。温度分布に関しても速度分布と同5―9 0熱工学様の壁法則が成立する。以上の温度分布に関する壁法則も内部流に対してもよく成立する。その後最終的には、小さなスケールの乱れに働く分子粘性によって熱エネルギーに散逸されることがわかる。一方、平均速度こう配は壁面に近づくほど大きくなっている。このような事情から乱れの生成項は、緩和層で最大値をとる。他方、壁面から離れるに従い、乱れの生産は減少し、かつ壁面の拘束条件が弱まるため、各成分は単調に減少して等方的な乱れとなる。この例は円管内の乱流であるので、全せん断力は一定とはならず中心に向かって減少するが、式(4・189)の関係は同様に成立する。レイノルズ数が増大するに従って、レイノルズ応力や乱流熱流束がますます支配的になることがわかる。
