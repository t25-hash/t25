# 4・8・2 乱流場における熱・物質・運動量輸送

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・8 強制対流乱流熱伝達

乱流中に生起する流体塊はそれ自身の速度で運動をするが、この運動に関連して乱流に特有な運動量や熱の輸送が生じる。これが、乱流熱流束であって、乱流渦運動によって生じる熱流束を表している。同様のことを運動量の点で見ると、平均値-ρ′′は乱流渦運動によって生じるせん断応力を表しており、これが4・6・3項のレイノルズ方程式に現れるレイノルズせん断応力である。図4・31では考慮していないが、もし流れ場内に化学種濃度の不均一があれば、やはり乱流渦運動に伴って化学種の物質輸送が生じる。いま、化学種の質量濃度の変動分をω′とすると、この化学種の物質輸送束(mass flux)はρ′ω′である。流れが乱流であっても、分子粘性による応力や分子熱伝導による熱移動は生じる。このことを考慮に入れると、二次元の境界層型の乱流中に生じる平均全せん断応力τと平均全熱流束は次のように表される。
