# 4・8・1 項で述べたように、固体壁面のごく近傍では、式(4・190)

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・8 強制対流乱流熱伝達

(4・191)の第1項は重要であるが、壁から少し離れると第2項が卓越するようになる。乱流境界層の外層には、乱流部の凹凸に伴う大規模な渦運動が存在する。この渦運動は、境界層外の流体を境界層内部に巻き込み、さらに固体壁面近傍に輸送するとともに、固体壁面近傍の流体を境界層の外層部に向かって輸送する。その評価法については、4・8・3項で詳しく取上げるが、それらの量を表記する一つの方法として、それらを平均速度のこう配と平均温度のこう配に関係づけることが古くから行われている。式(4・192)(4・193)を先の式(4・190)(4・191)に代入すると、平均せん断応力と平均熱流束は次のように表される。この領域が粘性底層である。同様に、壁面近傍には温度が直線分布をする領域が存在する。速度や温度が直線的に分布する領域の外側には、式(4・195)ではνとν、式(4・196)ではαとαがほぼ同等に寄与する領域(緩和層)が存在する。
