# 4・8・4 平板に沿う強制対流乱流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・8 強制対流乱流熱伝達

まず、平板がその先端から一様温度で加熱されている場合に注目する。そこで、境界層内の平均流を平板に平行な一方向流れであると近似することができる。この近似のもとでは、運動方程式やエネルギー方程式の対流項を無視することができるため、式(4・195)(4・196)の左辺をそれぞれ平板表面に作用するせん断応力τ(Pa)および壁面熱流束(Ws)に等置することができる。ここで、両者の比をとり、について1回積分すると次式が得られる。これをレイノルズの相似則と呼ぶ。式(4・233)を用いるとについての知見から熱伝達率の大略値が推定できる。より広いプラントル数に適用できる経験式としては、ケーダー(Kader)らによる次式がある。平板の先端部に非加熱区間がある場合には、その下流域で初めて温度境界層が発達を始めるので、温度境界層厚さは非加熱区間がない場合に比して薄くなり、このため熱伝達率は大きくなる。以上は、平板の先端からの位置における局所の熱伝達率であったが、この間の平均熱伝達率を必要とすることも多い。
