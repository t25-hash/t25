# 4・8・6 外部流の強制対流乱流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・8 強制対流乱流熱伝達

流れの中に置かれた円柱のような物体の背後では、レイノルズ数が大きいと流れがはく離し、乱れが生成されるとともに、その影響ははく離点の上流にも現れる。また、大規模渦塊が形成、そして放出されるために、物体まわりの流れは極めて複雑になる。このような複雑な流れ状態の熱伝達率を理論的に求めることは困難であり、主として実験的に求められている。a. 一様流中に置かれた円柱まわりの熱伝達一様流中に置かれた円柱まわりの流れは、レイノルズ数の増加とともに、前面と後面が対称な流れから後面に双子渦を伴う流れ、周期的な渦放出を伴う流れへと大きく変化するので、熱伝達率もこれに伴って変化する。図4・35に示す程度のレイノルズ数では前面と後面のヌセルト数はほぼ同程度である。しかしながら、レイノルズ数が3×10程度を超えるとはく離点が前面から後面に移動する。これは前面で層流はく離した流れが乱流に遷移し、小さなはく離泡を形成しながら壁面に再付着するためといわれている。レイノルズ数が4.26×10では後面における熱伝達率は著しく増加している。d. 管群(tube bank)の熱伝達流れに直交して円柱が規則的に配置されている管群の熱伝達は、種々の熱交換器等と密接に関連し、実用上重要である。規則的な配列形式には図4・36に示す碁盤目配列と千鳥配列がある。管群からの熱伝達では、最前列の円柱の熱伝達率は一様流中に置かれた単一円柱の値にほぼ等しいが、後列に行くに従って前方円柱下流に形成される後流の影響を受けてその熱伝達率は増加し、10列を超えるとほぼ一定となることが知られている。10列以上の管群に対して、管群流入前、流出後の流体の平均温度と円柱表面温度の温度差を用いた管群全体の平均ヌセルト数は次式のように表されている。添字のない物性値は主流温度における値を用いる。ただし、列数が10列未満の場合、表4・8に示す定数を乗じ、列数の効果を補正する必要がある。e. 衝突噴流(impinging jet)による熱伝達二次元ノズルから噴流を平板に直角に衝突させる場合の衝突噴流熱伝達では、ノズルと平板間距離がノズルの幅の10倍以上の場合、平板の局所ヌセルト数は次式で与えられる。なお、式(4・272)のヌセルト数は主流乱れの効果を含んでいる。
