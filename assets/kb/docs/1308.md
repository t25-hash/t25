# 4・9・1 基礎的事項

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

温度の静止流体中に温度の異なる物体(温度욌)を置けば、熱エネルギーの移動を生じ、物体近傍の流体は密度が変化し、浮力による流体運動を引き起こす。この流体の運動は自然対流または自由対流と呼ばれる。温度変化を生じる空間領域には浮力のため対流運動が発生し、さらにこの運動は、温度変化領域の外側に粘性せん断力による流れを誘起する。したがって物体まわりに形成される自然対流の速度境界層は温度境界層よりもつねに厚い。粘性が大きい流体ほど速度境界層は温度境界層に比べて厚くなり、プラントル数が零の極限で速度境界層厚さは温度境界層厚さに一致する。浮力は壁面で最大であるが、粘性のため壁面で速度は零となり、粘性が強いほど壁面の作用が遠くまで波及する。このとき速度境界層内では図4・37のように速度最大位置は壁面から離れ、温度境界層の内側に局大値윯율융をもつ。なお運動の方向は、物体が流体より高温の場合は上昇し、物体が流体より低温の場合は浮力の方向が下向きになる。この自然対流の支配方程式は、4・6・3項の運動方程式、連続の式、エネルギー方程式、および次の密度変化の式(ないし状態式)ρ=ρ욥1-β-욥(4・273)であり、式の数が6あり、速度3成分、圧力、温度および密度の未知数6個が決定される(下から加熱された無限に広い水平流体層など、条件によってはさらに対流パターン選択の極値原理あるいは対流機構選択の原理が必要とされる)。ρ욥は基準温度욥における密度である。温度の静止流体中の物体を加熱あるいは冷却するとき、物体まわりに温度境界層および速度境界層が形成される。速度境界層外では鉛直方向に流れがなく、境界層外の圧力は、静圧こう配として-옿옿=ρ4・274)ここにρは、式(4・273)において温度=における密度である。対流時の圧力を、この静圧と圧力変化(動圧)の和の形に表す。=+(4・275)式(4・274)(4・275)を考慮すれば、対流時の運動方程式における圧力こう配と体積力(重力)の項は-∇+ρ=-∇-ρ-ρ=-∇+ρ욥β-(4・276)図4・37
