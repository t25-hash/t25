# 4・9・1 基礎的事項

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

温度の静止流体中に温度の異なる物体(温度)を置けば、熱エネルギーの移動を生じ、物体近傍の流体は密度が変化し、浮力による流体運動を引き起こす。この流体の運動は自然対流または自由対流と呼ばれる。温度変化を生じる空間領域には浮力のため対流運動が発生し、さらにこの運動は、温度変化領域の外側に粘性せん断力による流れを誘起する。したがって物体まわりに形成される自然対流の速度境界層は温度境界層よりもつねに厚い。粘性が大きい流体ほど速度境界層は温度境界層に比べて厚くなり、プラントル数が零の極限で速度境界層厚さは温度境界層厚さに一致する。浮力は壁面で最大であるが、粘性のため壁面で速度は零となり、粘性が強いほど壁面の作用が遠くまで波及する。このとき速度境界層内では図4・37のように速度最大位置は壁面から離れ、温度境界層の内側に局大値をもつ。なお運動の方向は、物体が流体より高温の場合は上昇し、物体が流体より低温の場合は浮力の方向が下向きになる。温度の静止流体中の物体を加熱あるいは冷却するとき、物体まわりに温度境界層および速度境界層が形成される。流体層内あるいは境界層の干渉がある場合は、この近似は成立しない。ただし、一般に体膨張係数を規定する基準温度は、後に見るように現象の物理的意味を反映してプラントル数などに依存して定められる。また粘性で誘起される流れに対しては、高プラントル数では速度最大位置が温度境界層の外縁に近く、その外側に運動量が拡散することから、速度境界層厚さは次式で規定される。低プラントル数においては、粘性の作用が影響しなくなり、速度最大位置は壁面近傍に移動し、温度境界層の外側に粘性によって誘起される流れ領域がなくなる。このとき、温度境界層は速度境界層と一致する(自然対流では浮力のある領域では必ず流れがあり、速度境界層厚さが温度境界層厚さより薄くなることはない)。この仮定はブシネ近似(Boussinesq approximation)と呼ばれる。一般に物性値は温度によって変化する。したがって、一定物性値を用いて近似するとき、基準温度の選び方が問題となり、種々の方法が提案されている。温度境界層内の流体は、一方で固体伝熱壁と接し、他方で遠方流体に接している。その際プラントル数によって温度境界層内の速度最大位置が変化し、温度境界層内の流体運動に及ぼす粘性摩擦、熱拡散や体膨張係数変化の影響が異なる。したがって平均代表物性値の評価に用いる基準温度はプラントル数に依存して変化すると考えられる。(2)の方法は、物理的意味を反映しつつも簡便であり、かつ他の方法と大差ない。
