# 4・9・2 平板の自然対流熱伝達(natural convection heat

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

transfer)a. 鉛直平板(vertical flat plate)。低レイリー数では境界層厚さが厚くなり、境界層近似が成立しなくなるとともに、実験的には平板幅の有限性による三次元効果が現れ、極限熱伝導の影響として数値0.68が現れたとみなされる。. 乱流への遷移自然対流においては、速度境界層は温度境界層よりつねに厚く、層流域が終わるあたりの可視化による観察では、強制対流におけるトルミエン・シュリヒティング(Toll-mien-Schlichting)波と同様の波が重畳し、波長成長が最大となる周波数をもって境界層の波打ちが生じる。擾乱の発達による温度境界層内外の流体の干渉は、局所的な減速作用をもたらし、低温流体塊の壁面への巻込みを伴う渦ないし渦列を形成する。さらにレイリー数が上昇すると、空間位置の不規則なストリークと呼ばれる筋状運動およびその壁面渦管の巻上がりが生じ、馬てい渦が形成される。このような渦による混合およびこれに伴う粘性誘起流域拡大の相互作用が、温度境界層および速度境界層の急激な拡大をもたらし、相対的に高温で軽くて流速の速い流体の壁近傍からエゼクト、および低温の重くて遅い流体の壁面への接近を伴う大規模の乱流組織構造をもたらす。乱流域では、種々スケールの流体塊の不規則な乱れ挙動を生じ、速度最大位置内側の内層厚さは極めて薄くなり、速度分布のピークの存在は壁近傍の乱流境界層挙動に強制対流の場合と異なる性格を加える。巨視スケールの渦の発達による組織流動構造が、速度最大位置を超えた境界層厚さスケールの広域で観察され、この組織構造が壁面近傍の乱流挙動に影響し、乱流熱伝達率の急激な上昇を支配する。なお境界層外に形成される温度成層は、流体塊上昇の相対的浮力減少の方向に作用し、上昇流への減速ないし復元力として作用し、上昇流の振動運動を増幅し、乱流遷移および乱流挙動に大きく影響する。この式においては長さへの依存性がなく、局所ヌセルト数は長さをに置換える。ただし乱流への遷移は速く、臨界レイリー数は鉛直平板に比べて2けた以上小さく、10の程度で乱流域となる。代表寸法の換算を考慮すれば、式(4・292)は次のように表される。傾斜平板のφ=90°の極限とは異なる特性となる。加熱された傾斜平板上面あるいは冷却された傾斜平板下面に沿う自然対流の乱流域に対しては、重力加速度は傾斜平板の平行方向成分よりはむしろ法線方向成分が支配し、傾斜角にほとんど依存しなくなり重力加速度はのままとし(あるいはをsinφで置換え)、式(4・301)および(4・302)が適用できる。なお傾斜平板においては、乱流遷移に先行する遷移域では流れ方向に軸をもつ渦が形成され、崩壊後の乱流の場合より熱伝達が上昇する領域をもち、傾斜角が大きくなるほど低レイリー数で速く遷移し熱伝達が上昇するが、平均熱伝達率はφ=60°で最大となる。下向き加熱の傾斜平板の乱流域は、鉛直上向き面と同一の式(4・301)が適用できる。この場合の乱流域については重力加速度はcosφで置換えられ、上向き面の場合より熱伝達が低下する。
