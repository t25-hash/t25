# 4・9・3 物体まわりの自然対流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

a. 水平円柱(circular cylinder)静止流体中に十分長い円柱を水平に置いた場合、層流においては、その局所熱伝達率の分布は円柱下端で最大、上端で最小になるが、高レイリー数においては円柱表面に発達する境界層がはく離し、乱流に遷移する。遷移領域の熱伝達は層流時の倍近い局所値となり、円柱下端の局所熱伝達率より大きくなる。直径の水平円柱表面に沿う自然対流は、高さ(=)の鉛直平板に沿う自然対流に近く、平均ヌセルト数は式(4・293)と同形であるが、極限伝導解は一般により小さい。細線の実験によれば、低レイリー数における極限熱伝導が小さくなる傾向を示す。b. 傾斜円柱(inclined cylinder)水平円柱から傾斜させれば、傾斜角φとともに熱伝達は減少し、φ=90°で鉛直円柱の場合に一致する。なお、この関係は、水平に対し角度φだけ傾いた円柱の平均熱伝達率に対して、φが50°~60°くらいまでなら、水平円柱の平均熱伝達率[式(4・315)]にcosφを乗じた修正(すなわち重力をcos φで置換える修正)にほぼ等しい。プラントル数への依存は鉛直平板に対する場合とほぼ等しい。
