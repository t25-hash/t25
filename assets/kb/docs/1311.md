# 4・9・4 管路および熱サイフォンの自然対流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

a. 鉛直流路(vertical channels)自然対流熱伝達壁温一様の鉛直平板(長さ)を静止流体中に間隔で平行に設置した場合、あるいは半径=の円管の場合、低レイリー数では煙突効果による熱伝達が支配し、レイリー数の上昇とともに単独平板の熱伝達特性に漸近する。したがって煙突と単一平板の両極限の熱伝達特性を用い、平行平板および円管の両流路に対して次の関係が適用される。下端に長さの非加熱区間を有する内管の場合には、の増大とともに対流が抑制されて熱伝達率が劣化する。円管流路の場合にもこの修正は有効と考えられる。ただし流路内の自然対流の流速増加は、壁温一定の場合と同様に単一平板より熱伝達を上昇させる。乱流への遷移レイリー数は、単一平板の遷移レイリー数[式(4・297)]に比図4・39水平円柱および球の自然対流熱伝達図4・40 鉛直流路内の自然対流熱伝達5―101べ1けた程度高くなり、乱流に遷移しにくい。b. 熱サイフォン(thermosyphon)熱の形でのエネルギー移動が流体循環を起こす系に熱サイフォンがあり、開放型、密閉型、閉ループ型に大別できる。熱サイフォンはガスタービン羽根、内燃機関の冷却等に応用される。ここでは開放型について述べる(図4・41,4・42)。開放型では下部に半径の円管加熱部(長さ)があり、その下端は閉じ、上端は低温流体の満ちた貯槽に開いており、このつなぎ部はオリフィスと呼ばれる。上昇流が乱流になる場合も下降流との干渉によりむしろ熱伝達の低下が観察される。
