# 4・9・5 密閉空間内における自然対流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

密閉空間内に加熱、冷却熱源がある定常循環流を考える。境界条件に応じて以下のように分類される。a. 水平流体層(horizontal fluid layer)水平流体層を下方から加熱し上方から冷却したとき、軽い流体の上方に重い流体が位置してポテンシャル的に不安定であるが、温度差がある限界値以下であれば、運動量および熱の拡散作用のため運動エネルギーの増幅ができず、流体は静止平衡状態のままに保たれる。しかし、レイリー数욆=β욠-욂νが臨界レイリー数욂以上になると自然対流が発生して、十分に幅の広い流体層においては軽い流体が上昇し重い流体は下降する、ある空間的周期性をもつ対流パターンを形成し、セル対流と呼ばれる運動を生じる。対流発生条件は微小擾乱の安定性で規定され、慣性力の影響が無視できるためプラントル数によらない。臨界レイリー数욂の値は境界条件に大きく依存するが、水平方向に無限に広い流体層に対しては、上下面が固体のとき욂=1708、上面のみ自由表面のとき1101、上下面とも自由表面のとき657.5である。ベナールは、ベナールセル(Benard cell)と呼ばれる、上方から見て六角形のセル対流(hex-agonal cell)を観察した。ただし対流発生時は図4・43にみられるように定常の二次元ロール対流として発生し、比較的広さの狭い流体層においては水平境界(容器形状)の影響を受け、特に矩形容器では短辺に平行する軸をもつロールセルとなる。水平広さが無限に大きいときは、水平境界条件に独立な対流パターンがプラントル数に依存して形成される。またレイリー数の増加により、対流パターンは種々の波長および多モードの不安定を生じ、セル形状の変形を伴い、不連続的に変化して、熱伝達に影響する。図4・43のように=1~10,10≦욆≦10の範囲の領域、レイリー数が3×10以下では定常二次元ロールセル、55000以下では定常三次元流、これ以上では三次元流、三次元振動流を経て、=0.7では욆が10で、=60では2×10で乱流に遷移する。なお振動流の発生は、プラントル数の影響を強く受け、低プラントル数においては二次元ロールの不安定から生じ、レイリー数の上昇により慣性力の影響のため二重周期のセル対流への分岐を生じ(図4・43破線の遷移)、高プラントル数では見られない熱伝達特性の遷移を生じる。なお高プラントル数域においては長距離の運動量拡散が連成し、対流パターン遷移は境界条件ないし対流セル初期形状ないしモードに強く影響される。また流体の粘性係数の温度依存性の正負が、六角形セルの中心部の下降あるいは上昇の流動方向に影響する。上下境界が固体壁の場合には、レイリー数が限界レイリー数を超えないとき(≦욂=1708)、対流が発生せず、伝導による伝熱が욇욆λ
