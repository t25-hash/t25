# 4・9・5 密閉空間内における自然対流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・9 自然対流熱伝達

密閉空間内に加熱、冷却熱源がある定常循環流を考える。境界条件に応じて以下のように分類される。a. 水平流体層(horizontal fluid layer)水平流体層を下方から加熱し上方から冷却したとき、軽い流体の上方に重い流体が位置してポテンシャル的に不安定であるが、温度差がある限界値以下であれば、運動量および熱の拡散作用のため運動エネルギーの増幅ができず、流体は静止平衡状態のままに保たれる。対流発生条件は微小擾乱の安定性で規定され、慣性力の影響が無視できるためプラントル数によらない。ベナールは、ベナールセル(Benard cell)と呼ばれる、上方から見て六角形のセル対流(hex-agonal cell)を観察した。ただし対流発生時は図4・43にみられるように定常の二次元ロール対流として発生し、比較的広さの狭い流体層においては水平境界(容器形状)の影響を受け、特に矩形容器では短辺に平行する軸をもつロールセルとなる。水平広さが無限に大きいときは、水平境界条件に独立な対流パターンがプラントル数に依存して形成される。またレイリー数の増加により、対流パターンは種々の波長および多モードの不安定を生じ、セル形状の変形を伴い、不連続的に変化して、熱伝達に影響する。なお振動流の発生は、プラントル数の影響を強く受け、低プラントル数においては二次元ロールの不安定から生じ、レイリー数の上昇により慣性力の影響のため二重周期のセル対流への分岐を生じ(図4・43破線の遷移)、高プラントル数では見られない熱伝達特性の遷移を生じる。なお高プラントル数域においては長距離の運動量拡散が連成し、対流パターン遷移は境界条件ないし対流セル初期形状ないしモードに強く影響される。また流体の粘性係数の温度依存性の正負が、六角形セルの中心部の下降あるいは上昇の流動方向に影響する。b. 鉛直流体層(vertical fluid layer、図4・45)対向側壁が加熱・冷却され、対向鉛直壁間の距離を、鉛直壁高さをとする。通常、加熱壁に沿って上昇し、冷却壁に沿って下降する二次元流となるがに依存する。それぞれの場合の熱伝達率相関式を列記する。
