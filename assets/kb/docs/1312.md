# 10・4・3 環境疲労

> **章**: 第10章 材料の強度
> **節**: 10・4 環境強度

水溶液中あるいは種々のガス環境下において，乾燥空気あるいは
Arのような不活性ガス環境中と異なる疲労挙動特性を示す場合を
総称して環境疲労（腐食疲労）と呼んでいる．特に海水，湿潤ガス
などの腐食性環境中においては，参照環境中に比べて疲労寿命が短
くなり，またき裂進展速度が上昇することがある．これを環境疲労
の狭義の意味で，腐食疲労（corrosion fatigue）と呼ぶことがあ
る．SCCの場合は，材料と環境がある組合せのときに生じるが，
腐食疲労の場合は必ずしも明確な材料と環境の組合せは存在せず，
どのような材料においても生じることに注意を要する．参照環境と
して大気環境を選ぶこともあるが，大気中の湿気はき裂やすきまに
おいて毛管凝縮を生じるので，高強度鋼や高強度Al合金において
は大気も腐食性環境として無視できない．また，腐食疲労において
は，環境と変動応力が同時に作用するときに害作用が大きく現れ，
前腐食による表面の凹凸に起因する疲労強度の低下よりも大きくな
る．
腐食疲労に及ぼす影響因子としては，力学因子，材料因子，およ

び環境因子があるが，このうち力学因子としては，応力繰返し速
度，応力波形，応力比あるいは最大応力などが挙げられる．参照環
境中においては，応力繰返し速度の影響はほとんど見られないが，
腐食環境中では応力繰返し速度が小さいほど腐食疲労寿命が低下し
たり，疲労き裂進展速度が上昇する．また，参照環境中において
は，応力波形効果はほとんど見られないが，腐食環境中では応力波
形の影響が現れ，応力上昇時の割合が大きいほど腐食の害作用が大
きくなる．平均応力の効果としては，応力振幅との組合せによって
は，保護皮膜を破壊したり，あるいはSCCを誘発して，繰返し応
力腐食割れ，動応力腐食割れを生じて，大きく腐食疲労強度が低下
することがある．材料因子としては，材料と環境との反応性，すな
わち耐食性が第一義的に重要であるが，破面にほとんど腐食が見ら
れない耐食材料においてもき裂進展速度が上昇することがある．

要である．
