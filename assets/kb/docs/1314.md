# 4・10・2 希薄気体の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

低圧気体あるいは微小な系における気体では、気体分子の平均自由行程(mean free path、気体分子が他の分子と衝突することなしに飛行する距離の平均値)が系の代表長に比べて無視できなくなる。このような気体のふるまいは、通常の流体力学のように流速、圧力、温度といった少数の巨視的変数のみによっては表現できず、微視的な記述、すなわち気体分子の速度分布関数,,を用いた記述が必要となる。ここに、は空間座標、は分子速度、は時間であり、の物理的意味は次のとおりである。いま、六次元位相空間,における微小体積をddとすると、時刻にこの微小体積に含まれる気体の質量はddと表される。平均自由行程と系の代表長の比=はクヌッセン数(Knudsen 気体が軽度に希薄な場合この場合には、が小さい場合のボルツマン方程式の系統的漸近解析により、気体の一般的ふるまいに対する流体力学的記述法が確立されている[曾根の漸近理論、文献(186)(187)(189)参照].これによると、気体の定常的ふるまいは、大域的には適切な流体力学的方程式と滑り型の境界条件の組によって記述され、その大域解は境界近傍の厚さ平均自由行程程度の薄い層(クヌッセン層)内でしかるべき補正を受ける。流体力学的方程式系とクヌッセン層補正の型は、考える物理的状況に応じて異なる。例えば、=1(したがって~で温度変化が小さいときには、流体力学的方程式として圧縮性ナビエ・ストークス方程式を、境界条件として通常の滑りの境界条件(slip
