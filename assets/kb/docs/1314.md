# 4・10・2 希薄気体の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

低圧気体あるいは微小な系における気体では、気体分子の平均自由行程(mean free path、気体分子が他の分子と衝突することなしに飛行する距離の平均値)が系の代表長に比べて無視できなくなる。速度分布関数を支配する方程式はボルツマン方程式(Boltz-mann equation)と呼ばれ、次の形をしている。式(4・363)の初期条件としては初期時刻におけるを、境界条件としては境界を離れる分子の速度分布(あるいはそれと境界に入射する分子の速度分布との関係)を指定する。拡散反射条件(反射分子が境界の温度と速度に基づく平衡分布に従い、かつ境界を通過する気体流量が零というもの)あるいはこれと鏡面反射の線形結合が通常用いられる[拡散反射の割合α(0≦α≦1)を適応係数と呼ぶ].非圧縮性・粘性流体の力学では、粘性係数を定数と仮定すると、流れの場と温度場は別々に扱うことができる。希薄気体ではこの限りではなく、これらは密接に結び付いている。例えば、ストークス流に対応する非常に遅い流れにおいても、流れが温度変化を生み、温度場が(外力が働いていなくても)流れを引き起こす。したがって、流れの現象と熱伝達を分けて考えるのは一般にはあまり意味がない。a. 気体が高度に希薄な場合気体が高度に希薄でが無限大とみなせるとき、ボルツマン方程式の右辺は無視できる。この場合を自由分子気体(free-molecular gas)という。例えば定常境界値問題では、境界を離れる分子の速度分布関数を求めることに問題が帰着する。この場合のさまざまな例については文献(186)(190)を参照されたい。特に、任意の形状、配置、温度分布をもつ静止した物体群まわりの気体の挙動を記述する厳密解(無限領域の場合、無限遠で気体の流れはないとする)が曾根によって得られており、これを用いると物体に働く力やエネルギー伝達が解析あるいは簡単な数値解析によって容易に求められる。b. 気体が軽度に希薄な場合この場合には、が小さい場合のボルツマン方程式の系統的漸近解析により、気体の一般的ふるまいに対する流体力学的記述法が確立されている[曾根の漸近理論、文献(186)(187)(189)参照].これによると、気体の定常的ふるまいは、大域的には適切な流体力学的方程式と滑り型の境界条件の組によって記述され、その大域解は境界近傍の厚さ平均自由行程程度の薄い層(クヌッセン層)内でしかるべき補正を受ける。流体力学的方程式系とクヌッセン層補正の型は、考える物理的状況に応じて異なる。この方程式系において連続流極限(continuum limit,0)をとることにより、次の興味ある事実が導かれる。圧力一定で静止した気体中(例えば、静止した密閉容器が任意の壁面温度分布をもつ場合の内部の気体)の温度場は、連続流の極限といえども、ナビエ・ストークス方程式系(このとき、それは定常な熱伝導方程式になる)によっては正しく記述できない。ボルツマン方程式の数値解析法として最も普及しているのは、直接シミュレーションモンテカルロ法(DSMC法)と呼ばれる粒子的・確率的解法である。この方法の詳細については文献(200)、その理論的背景については文献(186)を参照されたい。これは、簡単であると同時にはん用性をもつため、物体まわりの高速希薄気流の解析(エネルギー伝達の解析を含む)などの実用的問題に広く応用されている。これらの例については、国際希薄気体力学シンポジウムの講演論文集[近年のものを文献(201)~(204)に示す]を参照されたい。しかし、この方法はつねに統計的ノイズを伴うため、マイクロスケール流れで重要となる遅い流れや小さな温度変化を記述するには不向きである。この場合には、モデルボルツマン方程式を用いて差分解析を行うのが有効である。また、幾何学的形状が単純な基礎的問題に限れば、ボルツマン方程式の精密な差分解析も可能である。実際、これらの方法により、異なる温度の平行平板間や同心円筒間の気体中の熱伝達、球状物体から静止気体中への熱伝達などの基礎的問題が広い範囲のに対して解析されている。ここではその一例として、平板間の熱伝達を線形化ボルツマン方程式(剛体球分子)によって差分解析した結果を示す。
