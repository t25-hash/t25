# 4・10・3 超臨界圧流体の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

超臨界圧(supercritical pressure)では、もはや液体から気体への相変化は生じないが、臨界圧に近い超臨界圧では、密度、粘性係数、熱伝導率および比エンタルピーが、擬臨界温度と呼ばれる温度を含む狭い温度範囲内で、液体的な値から気体的な値に連続的に急激に変化する。温度がの近傍にある場合、超臨界圧流体の熱伝達は、通常の流体とは異なったものになる。以下に、従来よく研究されている加熱円管内の流れについて示す。図4・47に24.5MPaの水に関するそのような実験例を示す。この場合、の増加につれて、流れを横切る半径方向の温度範囲が広くなりの大きい流体層部分が狭くなるため、のピークは低下する。図4・48は26.5MPaの水に関するそのような実験例で、が大きい場合に熱伝達の低下に伴う壁面温度の上昇が見られる。このような熱伝達の低下は、上昇流の場合に顕著である。また、上昇流で流量が小さい場合、比較的広い範囲の熱伝達低下に加えて、管入口付近での局所的な熱伝達の低下が生じる。これらの熱伝達の低下の原因としては、加熱に伴う密度の減少による加速や浮力の影響が考えられている。
