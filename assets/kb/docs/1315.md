# 4・10・3 超臨界圧流体の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

超臨界圧(supercritical pressure)では,もはや液体から気体への相変化は生じないが,臨界圧に近い超臨界圧では,密度,粘性係数,熱伝導率および比エンタルピーが,擬臨界温度と呼ばれる温度욣욂を含む狭い温度範囲内で,液体的な値から気体的な値に連続的に急激に変化する.定圧比熱욣は,この욣욂で鋭いピークを示す.圧力が高いほど,욣욂は高く욣のピークは低くなる.温度が욣욂の近傍にある場合,超臨界圧流体の熱伝達は,通常の流体とは異なったものになる.以下に,従来よく研究されている加熱円管内の流れについて示す.流量に対して熱流束が比較的小さい場合,流体バルク温度욁が욣욂よりわずかに小さく,욣욂が壁面温度욌と욁の間に挟まれるとき,壁面に近い乱流域での욣の増大,すなわち乱流熱拡散の増大の効果によって,熱伝達係数は大きい値を示す.図4・47に24.5MPaの水に関するそのような実験例を示す.図中욣욂は욣욂における流体エンタルピーである.この場合,の増加につれて,流れを横切る半径方向の温度範囲が広くなり욣の大きい流体層部分が狭くなるため,のピークは低下する.流量に比べてが比較的大きい場合には,욣욂が욌と욁に挟まれる욁の広い範囲で熱伝達の低下が生じるようになる.図4・48は26.5MPaの水に関するそのような実験例で,が大きい場合に熱伝達の低下に伴う壁面温度の上昇が見られる.このような熱伝達の低下は,上昇流の場合に顕著である.また,上昇流で流量が小さい場合,比較的広い範囲の熱伝達低下に加えて,管入口付近での局所的な熱伝達の低下が生じる.これらの熱伝達の低下の原因としては,加熱に伴う密度の減少による加速や浮力の影響が考えられている.次式は,熱伝達の低下がない場合の熱伝達係数の整理式で,水と二酸化炭素の約2000個のデータのほとんどを±25%の誤差内で整表4・9異温度平行平板間の熱伝達0.0330.13771.0460.10.32466.3101.0610.15850.43291.0730.25120.54921.0830.39810.663715.851.0980.50.71701.1040.63100.768225.121.1090.85771.1161.5850.93041.1180.960963.101.1202.5120.98711.1233.9811.030∞1.128図4・47 超臨界圧水の管内流熱伝達(その1)5―106熱工学図4・48超臨界圧水の管内流熱伝達(その2)理する.λ욁=0.0183 μ욁욁ρ욌ρ욁욣욣욁욗(4・365)ここに,:熱伝達係数[W/(m・K)],:管内直径(m),λ:熱伝導率[W/(m・K)],:質量速度[kg/(m・s)],μ:粘性係数(Pa・s),:プラントル数,ρ:密度(kg/m),욣:定圧比熱[J/(kg・K)]であり,添字はバルク温度욁,は壁面温度욌に対する値を示す.また,욣=욌-욁욌-욁で,この式のは比エンタルピー(J/kg)である.指数は次式による.욁<욌≦욣욂または1.2욣욂≦욁<욌のとき,=0.4(4・366)욁≦욣욂<욌のとき,=0.4+0.2욌욣욂-1(4・367)욣욂<욁<1.2욣욂かつ욁<욌のとき,=0.4+0.2욌욣욂-1 1-5욁욣욂-1(4・368)
