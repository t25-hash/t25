# 4・10・4 液体金属の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

液体金属(liquid metal)としては、当初、融点の低い水銀の利用が試みられたが、構造材料との共存性などに難点があるため、高速増殖炉の冷却材として、液体ナトリウムが熱的・核的に優れていることが明らかにされて以来、ナトリウム、カリウムまたはそれらの混合物(ナック)などアルカリ金属の利用が主流となった。最近では、化学的に活性でない鉛-ビスマスなどの重金属の利用も見直されている。液体金属は、非金属流体に比べて熱伝導率が大きく、プラントル数が非常に小さいので、良好な熱伝達特性が期待できるが、対流伝熱にいろいろな特異性が現れ、非金属流体を対象として導かれた熱伝達の推算式が使用できない場合が多い。a. 管内流の熱伝達一般に、非金属流体の発達した乱流場の熱伝達の計算では、乱流コアでの熱移動の評価に当たり、乱流混合による効果に対して熱伝導による効果を無視するが、液体金属は熱伝導率が大きいのでこの省略ができない。マルチネリ(Martinel-li)はこの効果を省略せずにアナロジー計算を行い、熱伝達の理論式を導いたが、その形が複雑であったので、後にライアン(Lyon)はそれをまとめ直し、<0.1,10≦≦10に使用可能な式として次の式を導いた。=7.0+0.25(4・369)ここで、:ヌセルト数、:ペクレ数、:プラントル数で、代表長さは管直径である。式(4・369)は、マルチネリ・ライアンの式(Martinelli-Lyon formula)と呼ばれ、液体金属に関する先駆的な仕事として知られている。しかし、実験結果はこの式により低い値を与えることが多かった。その原因として、初期の温度測定技術の未熟さや汚れた実験装置の使用にあると考えられたが、満足のいく解答が与えられなかった。そうした状況の中でルバースキー・カウフマン(Lubarsky-Kaufman)の整理式=0.625(4・370)が発表された。この整理式は多くの実験結果を整理したもので、式(4・369)より低く、安全側にあるとの考えのもとに、エンリコ・フェルミ(Enrico Fermi)炉の熱設計式として使用された。しかし、その後、学問的には理論値と実験値との差の原因を明らかにすべく研究が続けられ、液体金属の乱流プラントル数욧が通常の流体のそれとかなり違った値を示すことも大きな原因であることがわかり、욧を考慮して、式(4・369)の改良が行われた。最近では、実際の計算に当たっては、等熱流束、等温のいずれの壁面条件に対しても、実測に基づいて、サボチン(Subbotin)らによって提案された次の式=5.0+0.25(4・371)が用いられることが多い。液体金属の対流伝熱におけるもう一つの特徴は、流れの方向の熱伝導に関することである。非金属流体の場合は、流れにより運ばれるエンタルピーに対し、流れ方向へ熱伝導で移動する熱量は小さいので無視するが、液体金属では熱伝導率が大きいために、この効果を無視できないことがある。これは流速の遅い場合の入口領域など、ペクレ数の小さい(≦60)ところで問題となるもので、主として層流場で注意されるべきである。b. 物体まわりの熱伝達非金属流体の場合、境界層近似では、面に平行な方向の熱伝導を省略するのが通例である。液体金属では熱伝導率が大きいので、この省略が必ずしも妥当でない。一方、液体金属のプラントル数が非常に小さいことから、速度境界層厚さは、温度境界層厚さに比べて無視しうるほど小さい。この事実に基づけば、速度場は非粘性流の場と仮定しても大差ないことが推察される。このような仮定を用いると、熱エネルギー式の解析が容易となり、熱伝達の推算が可能となる。管群に直交する液体金属の流れ場に対し、非粘性流の仮定を適用して求めた熱伝達特性が、水銀による実験値とほどよい一致を示したという報告もあるが、単管をよぎるナトリウム流で正確に測定された研究によれば、背面はく離域の形成のため、管まわりに非粘性流れ場を仮定することには無理があり、そのような推算方法には検討の余地のあることが報告されている。c. 沸騰、凝縮の熱伝達沸騰熱伝達に関しては、ナトリウム、カリウムなどのように、伝熱面をぬらす液体金属では、核沸騰域が存在し、水などと同様に、∝욦욀욧욗(:熱流束、욦욀욧:過熱度、すなわち伝熱面温度と液の飽和温度の差、:定数)の関係が成立するが、水銀など伝熱面をぬらさない液体金属では、明確な核沸騰域は存在せず、욦욀욧の増加に伴い、ただちに膜沸騰に移行する。また、気泡の発生が加熱面の表面状態や材質により影響を受けやすく、욦욀욧が異常に変動するような不安定が生じることがある。一般に、液体金属は水に比べて、非沸騰状態から沸騰を起こすために大きな욦욀욧(初期過熱度)が必要である。当初、高温液体金属技術が未熟なため、液体金属に直接通電する加熱方法が用いられたこともあって、初期過熱度が800K に達する実験データもあったが、その後、高熱流束ヒータが開発され、間接加熱方式による実験が行われるようになって、200K 以下に収まっている。また、入口温度を徐々に上昇していくと、温度上昇率(0~0.12K/sの範囲)の増加により初期過熱度が大きくなるという報告もあったが、高速炉の事故を想定した過渡沸騰実験では、より急激な温度上昇率(3~135K/sの範囲)の場合、初期過熱度は大気圧下で190K の範囲で散らばり、温度上昇率の影響は見られなかったと報告されている。凝縮熱伝達に関しては、液体金属は熱伝導率が大きいために、凝縮液膜の熱抵抗を支配的と考えるヌセルトの理論は適用できず、気液界面に生じる熱力学的非平衡状態に起因する温度ギャップがおもな熱抵抗となる。この熱抵抗の大きさに関しては、分子運動論に基づく試算や実測が多数行われているが、いまだ確定的な結果は得られていない。
