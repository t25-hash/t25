# 10・5・3 熱衝撃

> **章**: 第10章 材料の強度
> **節**: 10・5 衝撃強度

物体の表面が急激に加熱または冷却されると，物体内部と表面の力は一定時間で最大値に達し，これが材料によっては限界値を超える現象を熱衝撃（thermal shock）という．
発生する熱応力の大および加熱・冷却条件によって異なる．
熱応力に対する材料の強度壊引張応力の小さい脆性材料で問題となる場合が多い．
材料の熱衝破壊パラメータで評価されている．
度変化は熱衝撃係数と1－ν の積に比例する．
ここで，νはポアソ1－νが大きいもの，βが大のとき1－ν′が大きいものを選ついても同様である．
＝1－ν′ は熱衝撃破壊抵抗係数ともいし，それによって強度低下が問題となるときの熱衝撃破壊抵抗の評以上の係数は材料の破壊に至るまでの耐熱衝撃性を表すが，熱衝クによる材料の崩壊やはく落などの損傷が重要となる場合に用いらはき裂が緩やかに成長するため，繰返し熱衝撃を与えると徐々にき疲労破壊抵抗という．
温度差によって，急激な熱ひずみおよび熱応力が発生する．
この応て破壊することがある．
疲労破壊 繰返し負荷 応力範囲，ひずみ範囲 疲労強度破壊に対する抵抗度を示す係数で，熱衝撃の程度を示すビオーて大きいときは′＝σα で表される．
ここで， ：板厚，γ：板：縦弾性係数，α：線膨張係数である．
熱衝撃に耐えうる最大温ン比である．
したがって，耐熱衝撃用材料としては，βが小のとき定すればよい．
なお，上記は無限平板の場合であるが，無限円柱にわれ，セラミックスなど，材料中の微小き裂が熱衝撃によって成長価に用いられている．
撃損傷抵抗係数は破壊に至ってからの損傷の度合いを表し，クラッれている．
また，熱応力が材料の破壊強度以下でも，環境によって裂が成長し，最後には熱疲労破壊が生じる．
