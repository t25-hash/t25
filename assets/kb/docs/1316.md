# 10・5・3 熱衝撃

> **章**: 第10章 材料の強度
> **節**: 10・5 衝撃強度

物体の表面が急激に加熱または冷却されると，物体内部と表面の
温度差によって，急激な熱ひずみおよび熱応力が発生する．この応
力は一定時間で最大値に達し，これが材料によっては限界値を超え
て破壊することがある．このように急激な非定常の熱応力が作用す
る現象を熱衝撃（thermal shock）という．発生する熱応力の大
き，分布およびその時間変化は，その物体の物理的性質，形状寸法
および加熱・冷却条件によって異なる．熱応力に対する材料の強度
は破壊応力によって決まるので，熱衝撃はセラミックスのような破
壊引張応力の小さい脆性材料で問題となる場合が多い．材料の熱衝
撃強度を一義的に表すのは難しいが，以下のような各種の耐熱衝撃
破壊パラメータで評価されている．

疲労破壊抵抗という．
