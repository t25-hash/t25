# 4・10・5 非ニュートン流体の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

a. 円管内の層流熱伝達流れは定常であり、速度分布が流れ方向に変化しない発達した流れとし、物性は温度により不変、軸方向の熱伝導および内部発熱は無視できるとする。b. 円管内の乱流熱伝達発達した円管内乱流熱伝達に対してヌセルト数はそれぞれ次式より求められる。c. 粘弾性流体の伝熱a,b項で述べた指数法則流体、ビンガム流体などは純粘性の非ニュートン流体に分類されるが、多くの高分子溶液、溶融体などは弾性的性質も併せ持っており、粘弾性流体(viscoelastic fluid)と呼ばれる。粘弾性流体の伝熱は高分子成型加工の分野などで重要となるが、粘弾性流体の構成方程式を用いた流動と伝熱の基礎式を、与えられた境界条件のもとで数値解析して伝熱係数を求める方法が一般的である。粘弾性流体の乱流伝熱に関する一般的な相関式は得られていないが、希薄高分子溶液の限定された条件下では修正ファクタによる相関が可能であると報告されている。
