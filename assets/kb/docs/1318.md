# 4・10・6 流動層における熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

a. 伝熱面と層の間の熱伝達。 気泡流動層(bubbling fluidized bed)の熱伝達(1) 実験式気泡流動層で熱回収を行うときには、伝熱管群を炉内に設置することがよく行われている。設置方式としては水平伝熱管の場合と垂直伝熱管の場合がある。例えば気泡流動層石炭燃焼ボイラでは、直径40~50mm程度の水平伝熱管を水平方向に管直径の数倍程度の間隔をあけて配置する。これまでに流動層内熱伝達に関しては数多くの実験式が提出されており、それらは成書に詳細に述べられている。流動層燃焼装置内水平管群の熱伝達率(伝熱係数ともいう)に対して、代表的な粒子径範囲である300 mから数mmでは、次の式がよく合うとされている。熱伝達率は粒子径が大きくなると低下する。また、温度が上がると熱伝達率は増加するが、これは温度が上がるとガスの熱伝導率が増加するためと、600°C以上になると熱放射による伝熱の寄与が増えるためである。同様に、熱伝導率の高いガスを用いると熱伝達率は増加する。装置を加圧すると熱伝達率が増加する。(2) 熱伝達メカニズムミックリー・フェアバンクス(Mickley-Fairbanks)の粒子群交換モデル(パケット交換モデル、packet-renewal model)に基づけば、粒子群が壁面に到達して、非定常伝熱で壁面と粒子群が熱交換してから粒子群が離れていくことを繰返して熱伝達が起きている。粒子群の交換は気泡によって引き起こされる。500°C以下で放射熱が無視できる場合では、気泡が伝熱表面を覆っているときには、ガスと伝熱面との熱交換であるために熱伝達率は低い。そのような場合、粒子群が付着してから離れるまでの間の非定常伝熱のみを考慮することで、平均熱伝達率は次式で与えられる。ρは粒子群のバルク密度であり、固定層のバルク密度に近い。なお、粒子群が壁面に接触しているごく近傍では、粒子充てん密度が粒子群本体に比べて低くなるために伝熱抵抗が大きくなり、粒子径が大きい場合にはこの伝熱抵抗が無視できなくなる。粒子径が非常に大きい場合には、粒子群交換ではなく伝熱表面に付着した粒子層1層程度との熱交換によるモデルが使われる。600°Cを超えると次第に放射伝熱の影響が現れるので、放射伝熱の項を加える。(3) 熱伝達制御方法通常、装置内の反応量(装置床面積当たり負荷)はガス供給量によって変動するので、反応に伴う発熱(吸熱)量もガス供給量によって変動する。しかし、気泡流動層内の伝熱面では、ガス速度と熱伝達率の間には比例関係はなく、したがって、負荷変化に伴う除熱(あるいは加熱)のための伝熱量を制御するのは容易ではない。負荷が変動する場合に装置内からの熱回収量を制御する方法としては、流動層とその底部分散板(風箱)をいくつかのセルに分けて、流動化させる部分と流動化させない部分の割合を変えるセル分割方式が常圧気泡流動層燃焼で実用化されている。また、装置内部に一部移動層(moving bed)の部分を設け、そこに伝熱管群を設置し、移動層部へ流動層部(反応器本体)から流入する粒子量を制御することで伝熱量を制御する方法もある。加圧気泡流動層燃焼では、粒子溜を流動層の外に設けて粒子を流動層本体とやり取りすることで流動層内の層高を変えて、伝熱管群のうち濃厚層に浸っている割合を変えることで負荷追従をする方法が実用化されている。スプラッシュゾーンの熱伝達率は粒子濃度が高くなると増加することから、ガス流速変動に伴ってスプラッシュゾーンに飛び出す粒子を増減させて、熱伝達を制御することも比較的浅い気泡流動層で行われた。気泡流動層では熱伝達率が高い(数百W/mK)ことが特徴であるが、廃棄物焼却ではその高い熱伝達率がデメリットとなることもある。気泡流動層炉内に投入された廃棄物が急速に昇温を受けると、廃棄物の熱分解で揮発分が突発的に放出されるが、そのとき炉内での燃焼が不完全であると後段でのダイオキシン生成源となりうる。突発的揮発分放出を防ぐために、廃棄物投入時の流動化の停止(低熱伝達率での廃棄物昇温)と流動化(高熱伝達率条件下での反応)を交互に行い、廃棄物への熱伝達を抑制する方式がある。また、流動媒体を密度の低い多孔質にすることで熱伝達率を下げる方法も提唱されている。. 循環流動層(circulating fluidized bed)の熱伝達(1) 実験式循環流動層では高速のガス流で粒子を装置上部まで舞い上がらせて、サイクロンなど粒子分離装置で粒子とガスを分離して、粒子のみを装置底部にリサイクルする。高流速のガスと粒子による伝熱面摩耗を避けるため、気泡流動層で見られるような伝熱管群の配置はできず、装置周囲壁面あるいは面状に整形した伝熱管群を気流に平行に設置した伝熱面で熱回収を行う場合が多い。循環流動層燃焼装置内部壁面では、空間内粒子濃度ρ(kg/m)が高くなると熱伝達率は大きくなり、経験式としては次のものがある。粒子径が小さいほど熱伝達率は大きくなるが、伝熱面を長くすると熱伝達率は粒子径の影響を受けなくなる。加圧しても壁面熱伝達率の増加はわずかである。熱伝達率の測定の際に測定部の高さ方向の長さが短いと熱伝達率を過大評価するので、測定部は1.5~2m以上が必要である。温度が高くなると熱放射による熱伝達の寄与が大きくなるので熱伝達率が大きくなる。なお、循環流動層壁面熱伝達については数多くの研究があり、これまでにも優れた成書、研究レビューがあるので、そちらもあわせて参照されたい。(2) メカニズム循環流動層の壁面熱伝達も気泡流動層と同様に、粒子群(クラスタ)が壁面に到達して熱を壁面に与えては離れていくことの繰返しが起きている。循環流動層では気泡の代わりにクラスタが壁面に付着から離れるまでの時間を用いる。しかし、クラスタの付着時間の推定法はまだ確立していない。高温になれば放射伝熱の寄与が大きくなり、これは式(4・383)と同じ式で与えられる。ただし、壁面近傍では温度境界層が存在するので、壁から見えるクラスタの温度は層の本体温度より低くなり、循環流動層燃焼では壁から3~5cm程度離れた部分が見えているとの説がある。(3) 熱伝達制御方法循環流動層においては、炉内伝熱壁からの熱回収と飛び出した高温粒子をリサイクルするラインからの熱回収の二とおりの熱回収法がある。炉内伝熱量は、先に述べたように粒子濃度によって熱伝達率が変化することを利用して制御ができる。すなわち、炉内の粒子濃度あるいは高さ方向の粒子濃度分布を変化させることが考えられる。炉内の粒子量を制御するには炉外に粒子リザーバを有する必要があるが、循環流動層燃焼装置でこの方法が大規模に実用化された例はない。通常の循環流動層燃焼装置では、炉底部に吹込む一次ガス流量と炉中間から吹込む二次ガス流量のバランスを変えることで、高さ方向の粒子濃度分布を制御する方法が使われる。このとき炉底部が粒子リザーバとなる。一次ガスには原料ガスを用いるほかに、生成ガスをリサイクルする方法もある。リサイクル粒子からの熱回収方法としては、外部に気泡流動層型熱交換器を設けてその中にリサイクルラインから粒子を流入させ、ゲートバルブ等によって熱交換器への粒子供給量を制御することで、熱回収量を制御することが行われている。b. 粒子-流体間の熱伝達流動層内での粒子と流体間の熱伝達は、通常では流動層内にある粒子表面積が極めて大きいために極めて良好であるが、発熱を伴いしかも粒子の融点が操業温度に極めて近いような場合には、粒子熔融トラブルを避けるためには無視できない影響をもつ。また、実験データはこの式の数倍から数分の1の範囲にばらついている。
