# 4・10・7 化学反応を伴う対流熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

a. 特徴反応を伴う伝熱問題は燃焼プロセス、化学工業プロセス、電子材料プロセス、超高速飛行体・ロケットなどで重要な問題である。反応がある場合には、生成熱による発熱・吸熱があること、反応物質による反応エネルギーの輸送があること、反応により化学種の組成が変化すること、反応があると物性値が大きく変化す5―109ることなどの特徴を有する。反応には、流体中で起こっている均一反応と固体壁面で起こっている不均一反応がある。反応が進行するのには時間がかかり、また反応物質が混合するにも時間が必要であり、それぞれ反応時間욂、拡散時間(diffusion time)욃という特性時間を定義し、これらの値の比によって反応形態が分類される[욂욃>1:反応律速(reac-tion controlled rate)、욂욃<1:拡散律速(diffusion controlledrate)].b. 取扱い方実際に遭遇する化学反応は数多くの素反応からなっている複合反応であるが、こうした反応を含む熱流体の数学的な解析は難しく、未解決な問題がほとんどである。しかし、最近の計算機関連技術の進展により、個々の問題に対して複雑な化学反応を伴う対流熱伝達の数値解析が可能となってきた。以下に数値解析の一般的な手法を述べる。図4・49は各種保存式の関係を示す。対象とする系の化学反応(素反応、総括反応、平衡および非平衡など)を考えて反応速度式を定義し、化学種の保存式を組立てる。なお、反応速度式は温度依存性が強く、温度の指数関数として一般に与えられる。運動量保存式については反応が与える特別な効果はないので、ナビエ・ストークス方程式がそのまま用いられる。エネルギー保存式は比熱が一定でないため、温度表示よりもエンタルピー表示が使用される。なお、熱流束には熱伝導項のほかに化学種の分子拡散によるエンタルピー変化などが付加される。これらの支配方程式を解くに当たり必要となる物性値は、粘性係数、密度、比熱、熱伝導率、拡散係数である。反応を伴う場合、これらの物性値は混合物であり、温度、圧力と流体の成分組成によって決まるので、取扱いに注意すべきである。このような化学反応を伴う数値計算においては、いくつかの反応モデルを組合わせて適切な支配方程式系を構築するという問題に加えて、反応時間と拡散時間との間にスケールの違いがあるので、数値計算手法にはいくつかの工夫が必要である。詳細については文献(245)を参照。c. 解析例。 反応境界層(reaction boundary layer)反応を伴う境界層の問題は、燃焼室壁、反応器壁などの冷却壁伝熱、高速物体のアブレーション冷却で重要であり、いくつかの解析がある。気相反応と壁面反応を伴う境界層流れでは局所相似の近似が成り立つ場合が多い。図4・50は空気のよどみ点流れにおける解離、再結図4・49化学反応を伴う伝熱における保存式の依存関係図4・50 壁面熱流束に及ぼす流体反応と壁面反応の影響5―110熱工学合反応に伴う壁面熱流束を気相反応ダムケラー数(Damko썥hlernumber)と壁面反応ダムケラー数욌で表したものであり、この関係は気流温度や壁面温度に関係なく成立する。ここで、ダムケラー数は反応の特性時間と流れの特性時間との比で定義される。ダムケラー数が大きければ、反応に十分な時間が与えられ、反応は平衡状態の욄に近くなり、一方ダムケラー数が小さいときは、反応は凍結状態の욞に近い。が大きいときは、に対する욌の影響は小さいが、が小さくなると気流中での反応が少なくなるため、욌の影響は顕著となる。なお、,욌は境界層の発達によって増加し、は平衡状態の욄に近づくが、,욌の一方が小さくても他方が大きくなればは욄に近づく。. 管内反応管内で反応する流れは化学装置、燃焼装置などで重要であり、多くの研究がなされている。この場合、ヌセルト数(Nusselt number)とレイノルズ数(Reynolds number)の関係は熱伝達率の定義によって異なる。熱伝導および分子拡散による全熱流束に基づくヌセルト数は非反応系の場合より大きくなるが、熱伝導のみの熱流束は非反応系の場合とほぼ同じ値を示す傾向にある。図4・51は非平衡反応乱流場(2NO→2NO+O)における触媒反応壁、および非反応壁管内のヌセルト数とレイノルズ数の関係を示す。管壁からの加熱・冷却いずれの場合においても反応に伴う濃度こう配による拡散効果が熱伝達に寄与していることがわかる。
