# 4・10・8 電磁流体(magnetohydrodynamic flow)の対流熱伝

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・10 その他の重要な対流熱伝達

達a. 特徴電磁流体には、水銀や液体ナトリウムなどの液体金属、気体を高温化して得られるプラズマ(plasma)などがある。電磁流体では、熱伝達が電磁気的な力によって影響されることが大きな特徴である。電磁流体を支配する基礎式は、一般の流体力学における運動方程式にローレンツ力と静電気力を、エネルギー方程式にジュール熱、拡散による電離エネルギー輸送項、および電流による電子のエンタルピー移動項を、成分保存式に電場による移動項と電離反応による生成項をそれぞれ加えた電磁流体力学の式である。さらに、これらに電磁場を支配するマクスウェルの式とオームの法則が加わる。液体金属に関しては、4・10・4項を参照するとして、(a) 全熱流束(b) 熱伝導流束ここでは主として連続流体として扱えるプラズマを対象とする。プラズマ中には電子やイオンが存在しているので、プラズマ中における伝熱現象は通常の伝熱現象とは異なった以下のような特徴を有する。(1) 熱プラズマは高温であるので、伝熱量が非常に大きく、さらに放射による伝熱量が無視できなくなる。(2) プラズマ中に存在する電子やイオンが大きく関与しているので、化学反応を伴う反応性流体と同様に、解離エネルギーの輸送を考慮する必要がある。高温領域で電離して生成した電子とイオンが拡散によって低温領域へ移動し再結合すると、高温領域で得た電離エネルギーを低温領域で放出することになる。つまり、電子とイオンのペアによって電離エネルギーが輸送される。プラズマ伝熱では電離エネルギーによる機構が大きな割合を占める。(3) 電子やイオンは電磁場の影響を受けるので、外部からの電磁場によって伝熱量をプロセスに応じて制御することができる。電磁場がある場合の伝熱現象は、プラズマの流れの場が大きく影響を受けることから複雑になる。(4) 荷電粒子はプラズマの物性値に対して大きな影響を与える。荷電粒子間の衝突であるクーロン衝突は遠い距離でも起こるので、中性粒子間の衝突とは大きく異なる。クーロン衝突は極めて起こりやすいので、粘性係数等の輸送係数は小さくなる。アルゴンの粘性係数は温度に対して上昇するが、電離が起こる温度から減少するのはこのためである。これとは逆に、電子の熱運動速度が大きいために、電子が関与する熱伝導率等の輸送係数は大きくなる。ただし、粘性係数に対しては電子の運動量が小さいのでほとんど寄与しない。b. 電磁場のない場合プラズマにおいてはイオンと電子の電離・再結合反応があることが特徴で、化学反応を伴う熱伝達と類似に扱うことができる。高温領域で電離して生成したイオンと電子は低温領域で再結合し、電離エネルギーを放出するので、通常の流体の伝熱機構に加えて、イオンと電子のペアの両極性拡散によるエネルギー移動がある。電磁場のない場合の伝熱量は、一般の流体と同じようにヌセルト数を用いて推算することができる。しかし、プラズマ中の球の境界層内では物性値が大きく変化するので、一般の流体における熱伝達率(伝熱係数)の推算式をそのまま用いることはできない。さまざまな熱伝達率の推算式が提案されているが、式(4・392)がよく用いられている。添字の∞はプラズマの温度の対応する物性値、は球の表面の温度に対応する物性値である。それ以外の物性値は、プラズマの温度と表面温度の算術平均に相当する物性値を用いる。一般に熱プラズマから粒子への熱伝達率は、通常の気体における熱伝達率より1~2けた程度大きい。これは電離したイオンと電子が固体表面で再結合して、電離エネルギーが粒子へ与えられるからである。被加熱物体の大きさが1~10 m程度の場合には、プラズマの平均自由行程(大気圧熱プラズマの場合はおよそ数m)と被加熱物体の大きさが同程度になるので、プラズマを連続流体として扱うことができない。このときに重要なパラメータとなるのはクヌッセン数(Knudsen number)(平均自由行程/粒子直径)である。温度ジャンプ領域(0.001<<0.1)では、連続流体の仮定が適用できな図4・51管内反応乱流におけるヌセルト数とレイノルズ数の関係いが、補正係数を用いることによって抵抗係数や伝熱量を推算できる。自由分子流領域(10<)では、プラズマを構成している分子それぞれの挙動を考慮して、伝熱量を推算する。このような希薄流体中の熱伝達は4・10・2項を参照する。熱適応係数(thermal accommodation)とは、粒子表面とプラズマ粒子の間の相互作用に関する平均パラメータである。反射される化学種が粒子表面温度と平衡となって反射されると仮定したときにもつエネルギーをとすると、熱適応係数は式(4・395)で与えられる。したがって、熱適応係数が大きいほど伝熱量が大きくなる。c. 電場のある場合電磁流体中に電場があると電流が流れるので、上述のことに加えてプラズマ中のジュール熱による温度上昇があること、被加熱物体が導電性の場合はそこへ電流の流出入があることにより伝熱量が変わる。ただし、は電流密度、σは電気伝導度である。この場合は、イオンが物体表面へ到達すると物体表面から電子を取出して再結合することによるエネルギー放出である。この場合は、電子が物体へ吸収されると仕事関数に相当するエネルギーが物体へ与えられることによる。一般に被加熱物体が正電位になるように電圧を印加すると、電子が多量に流入するために伝熱量が急激に増加する。d. 電磁場のある場合電場に加えて磁場が加わると伝熱現象はさらに複雑になるが、電磁場が存在するプラズマの挙動は電磁流体力学の分野で詳細な研究が行われている。磁場があるとプラズマの物性値が磁場に垂直方向と平行方向とで異なり、非等方性になる。磁場に垂直方向の輸送係数は平行方向に比べて減少する。一般に磁場によって荷電粒子の運動が抑制されるので、伝熱量は減少する。
