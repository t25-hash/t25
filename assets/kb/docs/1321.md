# 4・11・1 沸騰現象

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・11 プール沸騰熱伝達

伝熱面の温度が液体の飽和温度よりある程度高くなると,伝熱面から液体への熱移動に気泡や蒸気塊の発生を伴うようになる.これが沸騰熱伝達であり,その分類は伝熱面から十分離れたバルク液温に注目して,(1)飽和沸騰(saturated boiling,bulk boiling)と,(2)サブクール沸騰(subcooled boiling,surface boiling)に大別される.バルク液温が飽和温度よりわずかに過熱されているのが飽和沸騰で,発生気泡は液中では凝縮しない.バルク液温が飽和温度以下に過冷されていても伝熱面温度が十分高ければ,沸騰が起こる.このサブクール沸騰では,飽和温度とバルク液温の差として定義したサブクール度(degree of subcooling)が大きいと,発生気泡は伝熱面に付着している間,あるいはバルク液中を移動中に凝縮し,正味の蒸気発生はない.バルク液の流れに注目すると,(3)プール沸騰(pool boiling)または自由対流沸騰(free convection boiling)と,(4)流動沸騰(flow boiling)または強制流動沸騰(forced convection boiling)に区別される.容器内の液中に浸積された伝熱面からの沸騰過程がプール沸騰で,発生蒸気による循環力によりバルク液に流れが生じる.ポンプなどにより流れを強制したときの沸騰過程が流動沸騰で,管内沸騰はその典型である.沸騰気泡は伝熱面を飽和温度まで加熱しても発生しない.気泡の発生には伝熱面とその近傍の液体がある程度まで過熱されていなければならない.気泡発生に必要な伝熱面の過熱度욦욀욧(伝熱面温度욌と飽和温度욦욀욧の差)の最小値を沸騰開始過熱度といい,伝熱面の粗さなどの表面状態,液体の物性,流れの状態などで変わる.均一温度の液中に半径욂の気泡が力学的に平衡を保って存在するには,液体は飽和温度より高くなければならず,その過熱度は,-욦욀욧=욦욀욧욞ln 1+2σ욂(4・399)ここに,:系圧力,욦욀욧:に対応する飽和温度,σ:表面張力,욞:蒸発潜熱,:ガス定数,である.沸騰気泡は,伝熱面上のキャビティと呼ばれる微細なくぼみに取込まれた気泡核が活性化されて成長を始める.このような不均一温度場で욂の曲率半径の気泡核が活性化するには,式(4・399)の予測値より高い伝熱面過熱度を要する.
