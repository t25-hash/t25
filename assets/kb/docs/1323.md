# 4・11・3 核沸騰熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・11 プール沸騰熱伝達

核沸騰では,わずかな伝熱面過熱度で高い熱流束が伝達されるのが大きな特徴である.その機構は単純なものではないが,(1)気泡の温度境界層かく乱,(2)気泡による温度境界層のはぎ取り,(3)気液の交換,(4)気泡による潜熱輸送,など多くの説がある.これらが空間的・時間的に総合作用して優れた伝熱特性を達成している.a. 熱伝達の整理式沸騰開始点近くの低熱流束では自然対流の影響が見られるが,中熱流束以上になると自然対流の影響はなくなり,十分発達した核沸騰(fully established nucleate boiling)の領域となり,この領域が核沸騰域の大部分を占める.飽和プール沸騰の十分発達した核沸騰の熱伝達率は次式で与えられる.λ=23λ욦욀욧ρρ욞ρρ-ρ× ρσ(4・401)=0.0206θσρ-ρ(4・402)ここに,λ:液体の熱伝導率,:液体の温度伝導率,θ:接触角(°)である.この整理式は実験データの多重回帰分析から求めたもので,換算圧力が10</욂<0.97の圧力範囲に適用できる.標準の粗さの伝熱面で測定されたデータを対象にしているため,表面粗さの影響は含まない.b. 表面粗さの影響核沸騰気泡が伝熱面上のくぼみから発生することから,伝熱面の表面性状は熱伝達に大きく影響する.表面粗さ,異物の付着,吸着ガスなどの先天的因子と,液中に溶解した不純物の析出や沈着などの後天的因子がある.表面の粗さは最大の影響因子で,中心線深さ욣あるいは10点平均粗さ욨で表面粗さを表すと,∝욣욗∝욨욗(4・403)ここに,指数の値としては=0.133や′=0.20などの一定値がよく用いられる.エメリー紙で磨いて表面粗さを変えて行ったフロン系冷媒の実験によれば,熱伝達に及ぼす粗さの影響は圧力とともに低下し,指数は次のようになる.=0.2 1-욂(4・404)c. 圧力の影響沸騰は高圧になるほど低過熱度で開始するので,沸騰曲線は低過熱度側にシフトし,熱伝達率は高くなる.圧力の影響は整理式(4・401)に含まれる物性値により,暗々裏に表されている.熱力学的相似性から物性値の温度による変化は,換算圧力で表される.この考えに従い,熱伝達率に及ぼす圧力の影響を表すと,욣=욂(4・405)욂욂욂=1-0.99 욂(4・406)これらの式を用いると,一つの圧力の熱伝達率が既知であれば,任意の圧力の熱伝達率は簡単に求まる.d. サブクーリングの影響バルク液が飽和温度より욦욊욁だけサブクールされていても,伝熱面の過熱度욦욀욧を用いるとプール沸騰曲線は飽和核沸騰の曲線にほぼ重なる.したがって,熱伝達率を伝熱面の過熱度を基準に定義すれば,飽和核沸騰の熱伝達整理式(4・401)がそのまま適用できる.また伝熱面温度とバルク液温の差を基準に定義する場合には,飽和核沸騰の熱伝達率に욦욀욧/욦욀욧+욦욊욁を掛ければよい.e. その他の効果伝熱面の姿勢,加速度,電場,液体の流れやかくはん,超音波付与,伝熱面の振動などの影響は,気泡発生が激しい十分発達した核沸騰域では二次的である.沸騰曲線が受ける影響を定性的に図4・53に示す.f. 多成分流体の沸騰異なる単成分液を混合した多成分流体の沸騰曲線は,基本的には単成分液の場合と類似である.熱伝達率は,式(4・401)に多成分流体の物性値を代入して算出した値や単成分の熱伝達率を濃度で内捜して算出した値よりも低下し,低下率は高熱流束ほど大きい.熱伝達率は次式で算出される.=욡욃1+(4・407)욈욅욅(4・408)욡욃=∑요욡욃=욡욃=(4・409)욈욅욅∑요=욁욣욡욃1-0.8exp -10(4・410)ここに,욅,욅:多成分流体と同じ圧力,同じ熱流束における成分の熱伝達率と伝熱面過熱度,욅:バルク液中の成分のモル分率,:成分数,욁욣:ボイリングレンジ(boiling range,バルク液の組成に対応する露点と気泡点の温度差),熱流束の単位はW/mを用いる.
