# 4・11・4 バーンアウト現象

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・11 プール沸騰熱伝達

電気加熱のような熱流束制御型の加熱源で加熱される伝熱装置において、熱流束が限界熱流束を超過すると、伝熱面温度が急激に上昇する。これは多くの場合、装置の重大な損傷や動作不良となるので、正確な限界熱流束の大きさを推定し、限界熱流束以下で動作するように、装置の設計や運転を行うべきである。限界熱流束は、最大熱流束やバーンアウト熱流束と呼ばれることもある。a. 限界熱流束の整理式クタテラゼ(Kutateladze)が次元解析的考察から提案した次式が、飽液のプール沸騰の限界熱流束に対して一般的によく利用されている。定数として、クタテラゼは0.13~0.19を、ズーバ(Zuber)は0.12~0.16を推奨している。式(4・411)は、無限に広い上向き水平面に対するものである。c. 系圧力の効果系の圧力は、限界熱流束を直接支配する因子ではないが、式(4・411)に含まれている物性値を通してその効果が現れる。図4・54より、特定の圧力における限界熱流束から他の圧力における限界熱流束を推定することができる。また、図とも理解できる。なお、図4・54は水の物性値に対して計算されている。d. 形状の効果式(4・411)は無限に広い上向き水平伝熱面に対して得られたものであるが、有限な伝熱面に対するリーンハート(Lienhard)とディール(Dhir)の整理式を表4・13に示す。e. その他の効果など加速度の効果は、一般にであるが、0.15程度まで変化するという報告もある。また、伝熱面熱容量が非常に小さくなると限界熱流束が減少することが報告されている。加熱相当直径を用いて、長方形流路内の限界熱流束に対しても適用できる。
