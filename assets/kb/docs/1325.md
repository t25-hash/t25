# 4・11・5 膜沸騰熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・54 から換算圧力0.3の近くで限界熱流束が最大となるというこ

膜沸騰では液体が伝熱面に直接接触できなくなり、熱伝導率の小さい蒸気膜が伝熱面と沸騰液の間に連続的に存在し、その膜を通しての熱伝達となるために核沸騰熱伝達と比較して熱伝達率がけた違いに小さい。しかし、低温媒体利用技術の普及や原子炉の安全評価などに関連して、膜沸騰熱伝達を予測することも重要となっている。a. 基本的な伝熱特性層流境界層解析と気液界面安定論に基づく熱伝達率の代表的な関係式は次式にまとめられる。西川らは、水平円柱に対して広い範囲で適用できる式を提案している。式(4・417)は、スパロウ(Sparrow)によって妥当性が検証されている。なお、式(4・417)は垂直面や水平円柱に対して適用すべきである。d. 最小熱流束熱流束制御型の加熱源の場合、その熱流束が最小熱流束よりも小さくなると、伝熱面温度は安定な核沸騰領域まで大きく低下する。また、ある熱容量をもった高温物体が非定常的に沸騰冷却される場合、最小熱流束点を通過すると熱伝達率が増大して冷却速度が速くなる。上向き水平面に対する気液界面の安定解析から決定された式を示す。なお、限界熱流束と最小熱流束の比は、次式(の値はほぼ1に等しい)に示されるように気液の密度比の平方根にほぼ等しくなる。
