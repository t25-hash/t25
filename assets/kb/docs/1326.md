# 10・1・6 カー・パリネロ法

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・1 量子分子動力学法

ン演算子の期待値である全エネルギーや，その他一般に演算子の期待値で与えられるさまざまな物理量を，電子密度の汎関数で表す手法である．
その原型はトーマス・フェルミ近似にさかのぼるが，現在用いられているDFTの理論体系は，ホーヘンベルグ・コーン（Hohenberg- Kohn）の定理 および，この定理にエネルギー汎関数の電子密度依存性の局所近似を仮定することで導かれる，コーン・シャム（Kohn-Sham）方程式 に基づいている．
ホーヘンベルグとコーンは基底状態に縮退がない場合，電子系に作用している外場（ポテンシャル） と基底状態の波動関数ψは，電子密度 を与えると一意に決まることを示した．

有限温度で原子が運動している場合は，変化する原子核の座標に対して，瞬時に電子が追随して基底状態を保てると仮定すると，系の全エネルギーはB-O平面の近傍に存在しているはずである．
カー・パリネロ法（Car -Parri-nello method) では電子に仮想的な質量を与え，電子の波動関数に古典力学の自由度を与え，電子密度の時間変化を記述する運動方程式を用い，原子の運動とともに波動関数の時間変化を求める．
