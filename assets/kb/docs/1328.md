# 4・12・3 サブクールおよび低クオリティ域の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・12 強制流動沸騰熱伝達

a. 沸騰開始点伝熱面近傍の液温が飽和温度を超え、核沸騰開始条件を満たしたとき、伝熱面上の微細なキャビティ(沸騰核)から発泡が始まる。伝熱面上に広範囲の大きさの有効キャビティが存在する場合には、核沸騰開始点での熱流束と壁面過熱度욦욀욧=욌-욦욀욧との間には次のような関係がある。=λ욆욆8σ욦욀욧욦욀욧(4・435)ここで、λ욆:液の熱伝導率、욆:蒸発潜熱、σ:表面張力、:飽和蒸気の比容積、욦욀욧:飽和温度(絶対温度)、욌:壁面温度(絶対温度)、添字ONB は核沸騰開始条件における値を意味する。沸騰核として有効に働く最大キャビティの大きさ윯율융と沸騰開始熱流束との関係は、=λ욆윯율융욦욀욧-2σ욦욀욧λ욆욆윯율융(4・436)で求められる。滑らかな加熱面のキャビティ半径は1 mのオーダであるが、実際には、有効なキャビティの大きさが限られるので、上式で計算されるおよび욦욀욧よりも大きい値で沸騰開始する。核沸騰開始条件を与える実験式としては、圧力0.1~14MPaの水に対して、次式がある。=0.01551.8욦욀욧(4・437)ただし単位は、圧力(MPa)、温度욌,욦욀욧(°C、熱流束(MW/m)を用いる。b. 核沸騰熱伝達蒸発管にサブクール液が流入する場合、伝熱面での熱流束と壁面過熱度욦욀욧との間の関係、すなわち沸騰曲線は、図4・56に示すようになる。熱流束を徐々に上昇させた場合、液単相流への強制対流熱伝達(a-b)から始まり、b点で核沸騰を開始し、部分核沸騰域(b-c)を経て、発達した核沸騰域(c-d)に至る。有効なキャビティの大きさが限られる場合、b点で図4・56 強制流動沸騰の沸騰曲線5―116熱工学沸騰開始せず液単相流への熱伝達が続いた後、e点で沸騰開始し、その後、速やかにf点に移動することもある。沸騰曲線は、部分核沸騰の領域までは、質量速度の増加に伴い熱流束の高い側に移るが、発達した核沸騰域ではほとんど変わらない。部分核沸騰域の沸騰曲線は、液単相流による熱流束욂と発達した核沸騰による熱流束욁の和として次式で計算する。=욂+욁, ただし욂=욆욌-욆(4・438)ここで、욆:液単相流への熱伝達率、욆:液温である。発達した核沸騰域の沸騰曲線は、プール沸騰に対する沸騰曲線より求めることもできるが、水の強制流動に対しては、次の実験式がある。욁=exp 46.225욦욀욧(4・439)욁=exp 28.722.7욦욀욧(4・440)ここで、は圧力であり、単位はそれぞれ욁(MW/m )、(MPa)、(°C)である。
