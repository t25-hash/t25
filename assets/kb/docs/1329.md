# 4・12・4 中・高クオリティ域の熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・12 強制流動沸騰熱伝達

a. 強制対流蒸発域の熱伝達強制対流蒸発域の熱伝達は、環状流の液膜内の熱伝導と液膜界面における蒸発が支配する。このため、運動量輸送とのアナロジーより、単相流に対する二相流の熱伝達率の比を、式(4・433)で定義したパラメータの関数として表す。例えば、は液単相乱流に対する次式で求められる。式(4・441)を基にした実験式としては、表4・15に示すようなものがある。b. 核沸騰および強制対流蒸発域にわたる熱伝達表4・15に示した実験定数が零でない実験式は、低クオリティ核沸騰域と強制対流蒸発域の熱伝達を含む式であるが、このほかに低クオリティ核沸騰域と強制対流蒸発域の熱伝達を含む式としてチェンの式がある。ポストドライアウト域の熱伝達の計算では、蒸気と液滴との間に熱平衡を仮定し、蒸気の乱流強制対流熱伝達が支配的とした次式がよく用いられる。このため、熱非平衡を考慮した相関式も提案されている。これらもやはり蒸気の乱流強制対流熱伝達式を基にした式の形をしているが、係数や指数が研究者によって異なり、また、レイノルズ数中のクオリティには熱平衡クオリティではなく実クオリティを用いる。典型的なポストドライアウト域の伝熱過程は、(i)伝熱面から過熱蒸気への対流熱伝達、(ii)過熱蒸気から同伴液滴への対流熱伝達、(iii)伝熱面と液滴との間の熱伝達、および(iv)熱ふく射から成る。これらの伝熱過程についての考察を基に、ドライアウト点から下流の各相の流量変化を計算して、ポストドライアウト域の伝熱面温度を求める方法も提案されている。ポストDNB 域の熱伝達に関しては、逆環状流域や逆スラグ流域における熱伝達相関式やプール膜沸騰熱伝達相関式を基にした熱伝達率相関式が、いくつか提案されている。しかし、ポストDNB 域では、熱伝達が流動条件や流動様式に大きく依存するため、広範囲に適用できる式はいまのところ見当たらない。なお、強制流動沸騰流における沸騰曲線を完成させるには、最小膜沸騰温度あるいはリウエッティング条件を知る必要がある。これについても、いくつかの式が提案されているが、現象自体が熱流動条件によって大きく変わるため、十分な知見が得られているとはいえない。
