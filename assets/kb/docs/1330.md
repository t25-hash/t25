# 4・12・5 強制流動沸騰流における限界熱流束

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・12 強制流動沸騰熱伝達

a. 限界熱流束の機構前述のように沸騰危機の形態には、ドライアウトとDNB とがある。液膜消失によるドライアウト発生機構は、比較的長い加熱長で熱流束があまり高くない条件で、加熱部出口付近で発生する高クオリティでの限界熱流束の特徴をよくとらえている。しかし、加熱長が短く熱流束が高くなると、液膜流が残っている状態で限界熱流束に達する。このような場合、液膜内で相当の沸騰を生じており、液膜の部分的なはく離や破断によりドライアウトに至ると報告されている。一方、DNB はサブクールあるいは低クオリティ域で、流れが気泡流、スラグ流あるいはチャーン流の状態で起こり、さまざまな発生機構[(i)温度境界層への気泡充満による加熱面への液供給阻害、(ii)沸騰気泡下の薄液膜の消滅、(iii)境界層のはく離、(iv)流動不安定など]が考えられている。しかし、一つの発生機構で説明できる実験データの範囲は限定され、圧力や流量、クオリティなどの流動条件や流路の幾何学的条件により、異なる機構が単一で、あるいは複合して働いていると考えられる。いずれにせよ、沸騰危機は液循環系の流量制限や流路の幾何学的条件、流動不安定、加熱面近傍の熱流動現象など、何らかの原因で加熱面への液の供給が阻害されることによって起こると考えられる。b. 限界熱流束相関式加熱部にサブクール液が流入する場合の限界熱流束は、一般に次式で表される。限界熱流束は、その発生機構によって異なる傾向を示すので、それに応じて異なる性質の相関式が提案されている。限界熱流束を入口温度(あるいはサブクール度)一定の条件で質量速度の関数として表せば、一様加熱円管内垂直上昇流の場合、図4・57に示すように、質量速度が小さいときは出口クオリティが1(すなわち完全蒸発)の条件に近い曲線に沿い、質量速度が大きくなると曲線のこう配が小さくなる。甲藤は、前者をL-レジーム、後者をH-レジームと呼び、それぞれに対して次の形の式を提案している。図4・57で、加熱流路上部から重力によって液が流入できる場合は、破線で示すように、下部から流入する質量速度が零であっても限界熱流束は零にならず、フラッディングによる液流入制限により決まる限界熱流束の値をとる。質量速度が大きいときは、低クオリティあるいはサブクール域でDNB 型の限界熱流束が起こり、曲線のこう配は再び大きくなる。ドライアウトによる限界熱流束では、局所の熱流束より加熱部での熱入力が重要であるため、沸騰水型原子炉では、限界熱流束を引き起こす出口クオリティ(限界クオリティ)を核沸騰開始点からドライアウト点までの距離(沸騰長)の関数として表した実験式を用いている。また、近年、液膜ドライアウトの考えに基づき、三流体モデルを用いて液膜、液滴および蒸気の流れ方向の流量変化を計算し、液膜流量ゼロの条件で限界熱流束を予測する手法も用いられている。DNB による限界熱流束は、サブクール域あるいは低クオリティ域において高熱流束の条件で発生する。流れ方向の熱流束分布が中央部にピークをもつ場合には、加熱部出口ではなく、熱流束の高い、やや上流側で発生する。このような非一様加熱流路におけるDNB による限界熱流束は、一様加熱流路の限界熱流束を基に次式で求められる。DNB に対しても、多くの限界熱流束相関式や数表図4・57入口液温一定条件のもとで質量速度の関数として表した垂直上昇強制流動沸騰限界熱流束5―117(ルックアップテーブル)、発生機構に基づく予測モデルが提案されている。しかし、広範囲の条件に適用できる予測法はいまのところなく、対象となる流動条件での発生機構に応じた予測法を使い分ける必要がある。
