# 4・13・2 滴状凝縮熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 凝縮熱伝達

凝縮液が冷却面をぬらさない場合の凝縮様相を観察すると、伝熱面は大きさの異なる無数の半球状の液滴で覆われており、液滴間には実質的に液の存在しない領域が取り残されている。この凝縮形態を滴状凝縮という。液滴はその上への凝縮と、隣接する液滴との合体によって成長する。そしてそのサイズがある限界値に達すると、表面張力による付着力と重力とのバランスが崩れ、下方の伝熱面上に存在する液滴を掃除しながら滑り落ちる。この隣接液滴との合体や液滴の落下に伴って裸の伝熱面が露出し、その部分には新たに微細液滴が形成される。このサイクルの繰返しにより、滴状凝縮では膜状凝縮に比べて1けた高い熱伝達率が得られる。しかし、滴状凝縮の機構はまだ十分解明されておらず、高い熱伝達率が得られる原因が液滴間の実質的に液が存在しない領域にあるのか、または微細な液滴上への凝縮にあるのかについても結論が得られていない。表4・16 凝縮様式均質凝縮空間中における純蒸気の凝縮。クラスタ、微細液滴を形成。空間凝縮浮遊する異物を核とする空間中における凝縮。微細液滴を形成。直接接触凝縮低温液の表面または低温液中における凝縮。滴状凝縮冷却面上の凝縮。凝縮液が液滴を形成。膜状凝縮冷却面上の凝縮。凝縮液が連続した液膜を形成。5―118熱工学滴状凝縮の熱伝達率は液滴径の分布と密接な関係がある。水蒸気が平板伝熱面上に滴状凝縮する際の離脱液滴直径(mm)と熱伝達率[MW/(m・K)]の間には次の経験式が成立する。=0.25=0.12~3.4mm(4・456)滴状凝縮を実現するためには、ぬれにくい伝熱面を作成する必要がある。水蒸気に関しては、銅および銅合金面に対するオレイン酸等の脂肪酸、ベンジルメルカプタン、シリコーン油等の促進剤の添加および吸着法、貴金属薄膜のめっき法、テフロン等の撥水性高分子薄膜による被覆法、およびイオン注入法等が有効である。しかし、工業的に実用化できる長期間有効な方法はまだ開発されていない。
