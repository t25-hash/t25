# 4・13・3 膜状凝縮熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 凝縮熱伝達

凝縮液が冷却面をぬらす場合には、冷却面上に凝縮液の連続した膜が形成される。この凝縮形態を膜状凝縮という。膜状凝縮の温度分布を図4・58に示す。図4・58(a)は単一成分の飽和蒸気が凝縮する場合である。気液界面では、蒸気相から液膜へ向かう正味の質量流束が存在する。σの値は液体金属では1、水および有機物では0.2~0.6とされている。液膜内では伝導と対流による熱伝達が行われ、壁面に近づくほど液温が低下する。壁面熱流束はこの液の過冷却に要する熱量を含んでおり、一般によりも大きい。図4・58(b)は過熱蒸気、多成分蒸気および不凝縮ガスを含む蒸気が凝縮する場合の温度分布を示す。a. 垂直平板上の凝縮垂直平板上の凝縮液膜を観察すると、伝熱面の上端付近では液膜は非常に薄く、流れは層流である。この領域を波状流域(wavy flow regime)という。下流へ行くにつれて波の変形と合体が進み、大きい波と小さい波の二重構造が現れる。>1600の領域は、通常乱流域(turbulent flow regime)と呼ばれている。下向き蒸気速度が存在する場合には液膜厚さが減少し、大きい波の頻度は増加する。. 層流域静止蒸気の層流膜状凝縮についてはヌセルトの理論がある。この理論によれば、平均熱伝達率は次式で与えられる。速度の平行蒸気流が存在する場合の層流膜状凝縮の平均熱伝率は、次式で与えられる。. 波状流域波動を考慮した非定常解析が行われているが、実用的な熱伝達の式は得られていない。静止蒸気について次の整理式がある。 乱流域液膜内の乱れの構造が、単相流の壁近傍と類似の特性をもつと仮定した理論解析がなされている。また、気液界面近傍について表面張力によるダンピング効果を考慮するモデルもある。乱流液膜理論によれば、局所熱伝達率は次式で与えられる。静止蒸気について次の整理式がある。実用的には、式(4・459)(4・461)(4・464)で与えられる平均熱伝達率のうち最大のものを採用すればよい。b. 管内凝縮管内凝縮では、管に流入した蒸気は凝縮の進行に伴って流速を減じ、管出口では完全に液体となる。したがって、管入口付近では蒸気流によるせん断力の影響が支配的であり、管出口付近では重力の影響が支配的である。垂直管内の凝縮現象は基本的には垂直面の場合と同様であり、多くの理論解析がある。水および有機物の管内凝縮一般に適用できる局所熱伝達率(周平均熱伝達率)の整理式として、次式が提案されている。式(4・465)の誤差は多くの場合について15%以内とされている。しかし、冷媒の水平管内凝縮に関する実験データと比較すると、<100kg/m・sの領域ではかなり低めの値を与える。c. 水平管内の凝縮水平管内凝縮では重力と蒸気流によるせん断力の方向が直交するので、管内の気液の流動様式が凝縮の進行に伴って変化する。質量流束が小さい場合には、凝縮液が管底にたまり、成層流となる。この厚い液膜によって管底部の熱伝達が阻害されるため、管の平均熱伝達率は後述の水平円管外面に対する値[式(4・473)で=1とおいた値]よりも低くなる。質量流束が中程度で、重力と蒸気流によるせん断力の影響が共存する場合には、管底部に半月状の厚い液膜が形成される半環状流となる。また、蒸気流によるせん断力が支配的な条件下では、時間平均の液膜厚さが周方向に一様の環状流となる。この全領域に適用できる半経験式として次式がある。
