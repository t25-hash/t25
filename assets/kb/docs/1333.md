# 4・13・3 膜状凝縮熱伝達

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 凝縮熱伝達

凝縮液が冷却面をぬらす場合には,冷却面上に凝縮液の連続した膜が形成される.この凝縮形態を膜状凝縮という.膜状凝縮の温度分布を図4・58に示す.図4・58(a)は単一成分の飽和蒸気が凝縮する場合である.この場合,蒸気相の温度は一様であり,飽和温度욦に等しい.気液界面では,蒸気相から液膜へ向かう正味の質量流束が存在する.気液界面における熱流束욡は욆で与えられる.ここに욆は凝縮の潜熱である.また,気液界面には正味の質量流束に比例した温度飛躍(욦-욆욡)が存在する.分子運動論によれば,界面の熱伝達率욡=욡/욦-욆욡は次式で与えられる.욡=2σ2-σ2π욦욆욦(4・457)ここに,σ:凝縮係数,:分子量,:飽和圧力,:一般ガス定数.σの値は液体金属では1,水および有機物では0.2~0.6とされている.液体金属および低圧蒸気の場合を除けば,気液界面の温度飛躍(욦-욆욡)は液膜内の温度降下(욆욡-욌)に比べて十分小さい.ここに,욌は壁面温度である.液膜内では伝導と対流による熱伝達が行われ,壁面に近づくほど液温が低下する.壁面熱流束はこの液の過冷却に要する熱量を含んでおり,一般に욡よりも大きい.図4・58(b)は過熱蒸気,多成分蒸気および不凝縮ガスを含む蒸気が凝縮する場合の温度分布を示す.これらの物質では気相内にも温度分布が存在し,バルク温度욁は気液界面温度욋욡よりも高い.욋욡の値は,過熱蒸気では욦に等しく,不凝縮ガスを含む蒸気では蒸気分圧に対応する飽和温度に,また多成分蒸気では相平衡温度に等しい.これらの場合にも,液体金属および低圧蒸気を除けば욋-욆욡≪욆욡-욌が成立する.a.
