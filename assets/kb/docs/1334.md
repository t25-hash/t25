# 4・13・4 多成分蒸気の凝縮

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 凝縮熱伝達

多成分蒸気が凝縮する場合には、より高沸点成分の凝縮速度が相対的に速く、低沸点成分の凝縮速度は遅い。そのため、気液界面近傍の蒸気はバルク蒸気に比べて多量の低沸点成分を含むことになる。その結果、蒸気相に濃度境界層が形成され、図4・58(b)に示すような温度分布が生じる。不凝縮気体を含む蒸気の場合の温度分布も同様である。液膜内の熱伝達については純蒸気の凝縮に関する式がそのまま適用できる。a. 凝縮曲線法多成分蒸気の凝縮現象は複雑であり、十分解明されていない。近似解法として、凝縮過程について気液平衡を仮定し、液膜内について完全混合または完全不混合のいずれかを仮定して凝縮曲線(condensation curve)を求める方法がある。凝縮量は熱伝達の関係式より求める。なお、いずれの仮定がより妥当かについては明確な結論は得られていない。b. 膜理論凝縮液膜に接して層流膜があり、蒸気相での熱および物質伝達の抵抗はすべてこの膜の内部で生じるものと仮定する理論がある。膜理論(membrane theory)は成分蒸気間の相互拡散の取扱いによって3種類に分かれる。なお、番めの成分は不凝縮ガスとする。βの値は熱伝達と物質伝達のアナロジーを用い、凝縮条件に対応した熱伝達の整理式から求める。c. 物質伝達率の整理式平板上の二成分蒸気の層流膜状凝縮については、二相境界層理論に基づく解析がある。この解析を基にした蒸気層の物質伝達率の整理式が平板、円管等について提案されている。なお、式(4・471)は空気を含む水蒸気の凝縮に関する式を二成分蒸気に拡張したものである。
