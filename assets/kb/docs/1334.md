# 4・13・4 多成分蒸気の凝縮

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 凝縮熱伝達

多成分蒸気が凝縮する場合には,より高沸点成分の凝縮速度が相対的に速く,低沸点成分の凝縮速度は遅い.そのため,気液界面近傍の蒸気はバルク蒸気に比べて多量の低沸点成分を含むことになる.その結果,蒸気相に濃度境界層が形成され,図4・58(b)に示すような温度分布が生じる.不凝縮気体を含む蒸気の場合の温度分布も同様である.液膜内の熱伝達については純蒸気の凝縮に関する式がそのまま適用できる.a. 凝縮曲線法多成分蒸気の凝縮現象は複雑であり,十分解明されていない.近似解法として,凝縮過程について気液平衡を仮定し,液膜内について完全混合または完全不混合のいずれかを仮定して凝縮曲線(condensation curve)を求める方法がある.凝縮量は熱伝達の関係式より求める.なお,いずれの仮定がより妥当かについては明確な結論は得られていない.b. 膜理論凝縮液膜に接して層流膜があり,蒸気相での熱および物質伝達の抵抗はすべてこの膜の内部で生じるものと仮定する理論がある.膜理論(membrane theory)は成分蒸気間の相互拡散の取扱いによって3種類に分かれる.厳密な取扱いによれば,成分の凝縮モル流束욢は次式で与えられる.욈욪욢욅욅욁-욅욡+욧욢욁(4・469)욢=∑요ここに,:混合蒸気の成分数,욢욅:次元-1×-1の物質伝達率行列の要素,욧:全凝縮流束,:モル分率.添字,は成分蒸気を,また,はそれぞれバルク状態および気液界面を表す.なお,番めの成分は不凝縮ガスとする.は次式で与えられる.=ΦexpΦ-(4・470)ここに,욈욅욁β욢욅욢욢=욢욁β욢욈+∑요요5―119욢욅=-욢욁1β욢욅-1β욢욈욈욅β욢욅Φ욢욢=욢β욢욈+∑요요Φ욢욅=-욢1β욢욅-1β욢욈β:二成分系の物質伝達率,は単位行列.βの値は熱伝達と物質伝達のアナロジーを用い,凝縮条件に対応した熱伝達の整理式から求める.c. 物質伝達率の整理式平板上の二成分蒸気の層流膜状凝縮については,二相境界層理論に基づく解析がある.この解析を基にした蒸気層の物質伝達率の整理式が平板,円管等について提案されている.. 単一円管に直交する蒸気流=0.73(1+0.0028욋욁-욆욡욋욡-욆욡1+욋욁-욆욡욋욡-욆욡×(4・471)ここに,=β/(シャーウッド数,Sherwood number),=/ν욋(蒸気レイノルズ数),:低沸点成分の質量分率,:管外径,:拡散係数,:近寄り蒸気速度.添字は液のバルク状態を,は気液界面を表す.なお,式(4・471)は空気を含む水蒸気の凝縮に関する式を二成分蒸気に拡張したものである.. 水平フィン付き管群に直交する蒸気流=욋욁-욆욡욋욡-욆욡(4・472)ここに,=0.35~0.41.
