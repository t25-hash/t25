# 4・13・5 水平円管群における凝縮

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・13 凝縮熱伝達

水平管群における凝縮は、上方管からの落下凝縮液(イナンデーション)と蒸気流速の影響を受ける。これらの影響は大型管群ほど顕著になる。a. 層流域静止蒸気の水平円管上の層流膜状凝縮についてはヌセルトの理論がある。この理論を本の水平円管の縦管列に拡張すると、その平均熱伝達率は次式で与えられる。カーン(Kern)は小型凝縮器に適用できる式として次式を推奨している。c. 水蒸気の管群凝縮水平および下降蒸気流を伴う碁盤目管群および千鳥管群内の水蒸気の凝縮について、次の整理式がある。d. 冷媒蒸気の管群凝縮下降蒸気流を伴う冷媒の管群内凝縮について、次の整理式がある。
