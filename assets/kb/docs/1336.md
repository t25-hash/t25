# 4・14・1 物質移動の基本

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・14 物質移動

流体内の粘性摩擦、熱伝導、拡散の3現象はそれぞれ、流体内の分子運動の結果として生じる運動量、熱、物質の輸送過程である。なお、乱流の場合には、さらに微小流体塊の乱流運動に伴う輸送作用が加わるが、これらの3現象は物理量が運動量、熱、物質と相違するだけで、輸送機構(transport process)自体は変わらない。しかし、物質移動は多成分系の混合物を扱うため、伝熱より少し複雑になる。成分A,B の分子から成る二成分混合流体の物質移動について述べる。混合流体の濃度は単位容積当たりの質量またはモル数で表し、前者は質量濃度(密度)ρ(kg/m)、後者はモル濃度(kmol/m)と呼ばれる。これらの速度を使って、混合流体の平均速度が定義でき、それぞれ質量基準およびモル基準によって次式で与えられる。物質の移動速度は、ある濃度と速度を有する流体が、ある断面を垂直によぎって流れる場合を考え、濃度と速度の積として定義され、質量速度あるいは物質流束と呼ばれる。ここで、絶対速度に対する質量流束(mass flux)、モル流束(molar flux)は測定可能な量である。
