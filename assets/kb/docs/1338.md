# 4・14・3 物質伝達と熱伝達のアナロジー

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・17 に表す。

a. 物質伝達(mass transfer)壁面と混合流体の間に温度差があれば熱伝達を生じるように、流体が壁面と接触しつつ流動していて、流体と壁面との間に、ある物質の濃度差があれば、物質移動が起こり、これを物質伝達という。移動する物質を成分A とし、その移動の媒質を成分B として物質伝達を考える。物質伝達率も熱伝達率と同様に流れなどの諸因子に依存しており、表4・20に示す無次元数が物質伝達の相関式に用いられている。b. 物質伝達特有の境界条件物質伝達がある場合には、壁面を通して物質の移動が生じているので、混合流体は壁面に垂直な速度成分を必ず有する。すなわち、式(4・481)と式(4・486)より、は次式で与えられる。. 物質伝達相関式の導出i項の条件を満たすとき、既知の熱伝達の相関式(理論式、実験式、層流、乱流を問わず)を利用し、物質伝達の相関式を得る。すなわち、次式に示す物理量を矢印の方向に変換する。
