# 4・15・5 保温、保冷と断熱

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 6・6 節に記されている。

熱伝導率の低い単一物質の典型は、気体である。そこで、図4・69のように、繊維、粒子、多孔性物質、積層板(あるいははく)などにより、すきまに気体を含む構造とすると良好な断熱材となる。表4・21に、断熱材の例を示した。また、例えば相対する平面間の放射伝熱を遮りたい場合には、平面間に放射率εの低い板やはくを熱放射シールドとして介在させることが有効である。両平面の放射率もεである場合に、枚の熱放射シールドを入れると、放射熱流束は熱放射シールドがない場合の1/+1になる。
