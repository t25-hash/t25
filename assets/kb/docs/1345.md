# 4・16・1 熱交換器流路の基本形式

> **章**: 第4章 伝熱(物質移動を含む)
> **節**: 4・16 熱交換器の基礎

高温流体から低温流体へ熱エネルギーを移動させるための機器を、一般に熱交換器(heat exchanger)という。熱交換器の基本的な流路形式は、高温側および低温側流体の相対的な流れ方向により、並流(parallel flow)、向流(counter flow)、直交流(crossflow)などに分類される(図4・70)。これら比較的簡単な流路形式については、いくつかの仮定のもとに設計計算手法が確立されている。直交流において、一方の流体が、他方の流体の流れ方向に混合しない場合を非混合という。例えば、図4・70(d)において、管に直角方向に流れる流体は混合するが、管内を流れる流体は混合しない。
