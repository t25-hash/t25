# 5・1・2 温度測定に関する注意事項

> **章**: 第5章 熱測定法
> **節**: 5・2 のような起電力が生じる.1800°C以上の測定には,イリジウ

表面温度や温度こう配のある場を接触式の温度計で測定する場合,感温部に続く部分をできるだけ物体表面や等温面に沿って配置し,温度計に沿う熱伝導損失に起因する誤差を低減する.微粒子や凝縮性物質が感温部に付着すると熱抵抗層となるので,表面をつねに清浄に保つことが大切である.一方,反応性ガスの温度測定では,感温部の表面触媒作用により真温度を指示しなくなるので,表面を不活性化するために必要最小限の被覆をする.感温部が高温になると,熱放射損失の補正が必要となる.特に800°C以上のガス温度を測定する場合重要となる.補正は,次のエネルギー保存式により行う.-+욂+σε욥-+ddλdd=0(5・1)ここに,,ε,λはそれぞれ熱電対素線の半径,放射率,熱伝導率,,,욥はそれぞれガスの真温度,熱電対感温部温度,周形式原理検出器利用波長領域m/s検出可能最低温度°C検出器の応答速度m/s囲の輝度温度,はガスと熱電対感温部との間の熱伝達率,욂は表面触媒反応による発熱量,σはステファン・ボルツマン定数である.光学的方法,特に発光・吸収により温度計測を行う場合には,光路に沿って温度分布があると,測定温度が空間の平均的温度となったり,低温部分で自己吸収が生じて極端に低い温度として計測されるので,十分注意を要する.NaD 線反転法では,測定箇所に局所的にナトリウム塩を導入したり,NaD 線の中心からやや離れたウイング部分での反転を利用したりして,自己吸収による誤差を防ぐ.そのほか,温度こう配による光線の屈折が問題となることがあり,特に温度が変動する場での測定に注意が必要である.温度を電気的に測定する場合の注意事項については,JISに詳述されている.
