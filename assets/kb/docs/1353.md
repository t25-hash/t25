# 5・4・1 気液二相流

> **章**: 第5章 熱測定法
> **節**: 5・4 二相流測定

a. 気泡流の計測気泡を含む気液二相流の測定対象は、気相、液相の時空間の分布の情報と、各相の流速分布ならびに流量である。. ボイド率の計測気相と液相の時空間の分布は、流動および伝熱の特性を支配する要因である。気泡流、プラグ流、スラグ流や環状噴霧流など、気相と液相の質量流量比によるマッピングがなされている。ボイド率(void fraction)は、ある点での気相と液相の存在時間の割合(気相存在時間率)で表される。この量の計測には図5・6に示されるような、点電極のプローブを用いる。ボイドの測定には電極数は原理的に一つでよく、電極に気相と液相が通過する際に電気伝導度の違いにより、図5・6に示されるような信号が得られ、この時間割合からボイド率を求める。また点電極を2個用いてその信号の時間遅れから気泡の速度を求めることも可能である。これを光学的測定するものが、図5・7に示す光ファイバプローブである。光ファイバの先端をくさび状に削り、気体と接触するとその屈折率の違いにより、ファイバを通過してくるレーザ光の戻りを検出する方法である。受光部に光電変換器(フォトダイオードなど)を用いて、その後の処理は前述の電極プローブと同様な信号処理を行えばよい。面的なボイド率の測定法としては、X 線やγ線などの放射線が測定する管内を透過する際に、気相、液相の違いによる吸収または減衰が異なることを利用して、空間のボイド率を求める。さらに複数の角度からのデータを収録するトモグラフィ法は、空間の分布を再計算する方法である。また最近では、画像処理技術の進5―138熱工学展に伴い、中性子ラジオグラフィによる瞬時のボイド率分布の計測も行われるようになった。電極プローブによるボイド率は存在時間割合で、空間的には局所であるが時間平均値であるのに対し、放射線による方法は流路を横切るビームの線上の平均ボイド率を求める方法である。このほかに、微小な測定空間の静電容量の変化から気液の混合率を計測する方法等がある。最も古典的で基本的な方法は、流路に取付けた2箇所の弁を同時に開閉し、それにより気相と液相の割合を測定する締切法と呼ばれる方法である。定常で均質な流れ場では、局所の時間平均と空間平均はエルゴード性が成り立つならば一致するが、非定常な流れ場では両者は一致しないので注意を要する。. 流速および流量流動様式により流速の測定法は異なるが、気泡流の場合には2点式の電極プローブもしくは光ファイバにより、気泡の通過時間を測定することによってその速度を決定する。液相の測定には熱線(ホットフィルム)プローブを用いる方法がある。基本的には熱線流速計の測定原理に基づくものであるが、気相と液相の出力信号を分離することにより求められる。この方法では速度の校正が必要となる。光学的な流速測定としてはレーザ流速計が用いられる。図5・8に代表的な光学系を示す。レーザ光は等強度に分割され、集光レンズ図5・6 点電極プローブ図5・7 光ファイバプローブの動作原理図5・8 レーザドップラー流速計/位相ドップラー流速計の光学系により1点に集光させる。この交差点が測定体積になり、ここを通過するトレーサ粒子からの散乱光を受光素子で光電変換する。この信号は単発的でドップラーバースト信号と呼ばれ、そのドップラー周波数が流速に比例する。信号処理器において、FFT 法などを用いてこの周波数を検出する。二相流に適応させる場合は、高ボイド率ではレーザ光が入射できずに測定困難になるため、比較的低ボイド率で適用されている。この場合、気泡からのドップラー信号と、液相側に添加されたトレーサ粒子との信号を分離しながら計測する必要がある。近年のディジタル画像技術の進展に伴い、粒子画像流速計(PIV : particle image velocimetry)を利用した気泡および液相の速度測定が行われている。PIV はシリンドリカルレンズ等を用いてレーザ光をシート状にし、流体中のトレーサ粒子の画像をCCD カメラ等を用いて収録する。短い時間間隔で一対の画像を収録し、両者の相互相関を求めることにより粒子の移動距離を求め、速度を算出する手法である。気泡流の場合には液相のトレーサに蛍光粒子を用いることで、入射したレーザ光と異なる波長で画像を収録でき、気泡の径面近傍での詳細な流速測定が可能となっている。気液の各流量の測定には、独立した二つのパラメータを用いなくてはならない。平均ボイド率とオリフィスの差圧、または特性の異なる二組のオリフィスの差圧、または体積流量計とオリフィスの差圧などの組合せにより測定ができる。b. 噴霧流の計測気流中に微細な液滴群を含む流れも気液二相流に分類される。分散相は液滴群で、必要となる情報は液滴の質量流量、液滴径の分布などである。以下に代表的な計測手法を概説する。. サンプリングによる方法噴霧粒径の測定法としては液浸法が古くから用いられている。これは液滴と溶け合わない液体で受け止め、拡大撮影して粒径を測定する。液滴は懸濁状態であるので真球とみなせる。受け止める際の液滴の逃げや、懸濁中の合体の問題に注意が必要である。. 光学的粒子径計測(フランホーファー回折法、位相ドップラー法)レーザ光中の粒子により、入射光の前方に回折光により光のエネルギーが集中する領域が存在する。この回折光のパターンは粒子の断面積に依存するもので、これにより粒径分布を測定する方法がフランホーファー回折法(Fraunhofer diffractionmethod)である。すでにはん用化された標準計測として市販されている。位相ドップラー法(phase Doppler method)はレーザドップラー流速計を拡張し、複数の受光系を配置することによって、飛行してくる液滴の速度と粒径の同時測定を可能としたものである(図信号に位相差が生じる。この位相差は、透過性の粒子の場合、レーザ光の液滴表面の反射と液滴内部での屈折によって決定され、最終的には液滴の粒径に比例することになる。測定限界は液滴が球形を維持していることが条件で、変形するような大きな液滴には適用できない。また最小粒径は入射するレーザ光の波長に依存するが、おおよそ数mまでは測定可能である。球形であれば気泡流に対しての適応も可能である。速度に関しては前述のドップラー信号の周波数を検出することで求められる。. 画像を用いた面的な計測法燃料噴霧など分布計測として蛍光・散乱画像法がある。これは噴霧内にレーザシート光を入射し、粒子が発する散乱光(これはミー散乱と呼ばれ、表面積に比例する)と、燃料に蛍光剤を混入しレーザ光による蛍光発光強度(この強度は液滴の体積に比例する)を同時に画像として収録し、この二つの画像の強度比よりザウター平均径(体積/表面積平均粒径)の空間分布を求める。また近年のディジタル画像技術を応用した方法として、干渉画像法がある。基本原理は位相ドップラー法と同じで、透過性粒子の表面反射と屈折光の干渉を利用したものである。これは結像面より外れた位置で干渉させ、円形の撮影像の中の干渉じまの本数が粒子径に比例していることを利用する。高密度な状況に対して粒子像の重なりを回避するために、光学的な圧縮技法を用いた手法が開発され、粒子径と速度の瞬時の空間分布の計測が可能である。
