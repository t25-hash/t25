# 5・4・2 固気・固液二相流

> **章**: 第5章 熱測定法
> **節**: 5・8 ).それぞれの受光系の配置に応じて,ドップラーバースト

a. 流量固体粒子の搬送が目的の場合,粒子の目づまりのない方法がとられる.ベンチュリとオリフィスを併用し,輸送量の増加により管路の差圧が変化するのを用いて,空気流量と固体粒子の質量流束を算出する方法である.つまり管路断面の変化を用いる方法で,管直径の変化のない直管部の差圧と拡大管部の差圧より求める.b. 等速吸引法による両相速度,粒子濃度の測定この方法は,流れの中にピトー管のようなサンプル管を挿入し,サンプル管直前の静圧を流路断面の静圧と等しくして吸引する方法である.この吸引された二相流(固気)の分離を行うことによって,気相側の流速および分散相側の質量流束を求めることができる.元来この手法は粉じんの測定などに用いられ,日本工業規格(JIS)の粉じん測定法としても規定されている.c. 光学的測定法気流中の微少なトレーサ粒子を測定して速度を求めるレーザ流速計は,固気,固液,気液二相流に適応することができる.固気・固液二相流の場合にも,分散系である固体粒子とトレーサ粒子の信号を見極めることが必要である.これには粒子からの散乱光強度を基に粒子の情報を弁別する.また,球形の粒子に対しては位相ドップラー法も適応可能である.さらに面的な測定では粒子画像流速計(PIV)を適応し,分散相の速度および,トレーサを添加した連続相の速度測定がなされている.いずれの場合に関しても,気泡流と同様に分散相と連続相の粒子画像の分離に注意を要する.d. 非球形粒子の計測工業上現れる固気・固液二相流は,粒子が球形であることは数多くない.そのような対象には回折光強度による等価直径の計測法や,画像処理による非球形粒子の計測法が用いられる.光束中を粒子が通過したときに放つ散乱光は,粒子の光学的な性質や形状,表面性状などに依存する.この回折光の性質に注目し,その強度を検出することによって,粒子形状に依存しない粒径測定を可能とする.従来のLDV や高速な信号処理器と組合わせることにより,不規則な形状をもつ粒子の速度と粒径の同時測定を実現している.粒子の形状を含めた測定の場合には,粒子像を収録しなくてはならない.レーザ流速計の光学系に小型の撮像素子を併設し,粒子の速度と同時に粒子の投影像を収録することによって,粒子の飛行中の姿勢,速度およびその形状の情報も知ることができる.一方,粒子画像流速計(PIV)の機能を拡張し,非球形粒子の形状と速度の同時計測も行われるようになった.2台以上のカメラを用いて,粒子像をステレオで撮影することにより三次元の位置を同定し,その速度の3方向成分を求める.さらに撮影された各方向の粒子像を再構成し,粒子形状を決定する.5―139CPU(central processing unit)の高集積化やMEMS(micro-electro-mechanical systems)技術さらにはナノテクノロジーの発展により,ミクロな領域の温度計測法,熱物性測定法の確立が求められている.光学計測は光の回折限界が空間分解能を決定するが,最近のCPU の線幅が0.1 m以下であることを考慮すると,これまでの光学測定法では対応することが不可能となってきている.ここでは,走査型熱顕微鏡(SThM : scanning thermal micros-copy)を用いたミクロ領域における温度計測法,熱伝導率や温度拡散率などのミクロ熱物性測定法,ナノカロリメトリーについて述べる.
