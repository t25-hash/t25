# 10・3・2 容器内の液体の液面振動，非線形スロッシング

> **章**: 第10章 非定常流れ
> **節**: 10・3 容器内の液体の振動

容器が液体の固有振動数近くの振動数で加振されると，スロッシ
ングの振幅が大きくなり，非線形な旋回振動などが発生し，線形解
析が適用できなくなる．
a.水平加振 長方形容器が水平方向に加振されると，一次の
非対称モードのスロッシングが発生する ．
円筒容器では，一次の非対称モードの旋回スロッシングが発生す
る ．図10･18に，旋回スロッシングが発生する条件（斜線部）
を示す．また，図10･19に，旋回スロッシングの発生を防止するた
め，円筒容器内に容器の中心を通る幅34の仕切板を加振方向に
平行に設けた場合の液体の非線形応答の例を示す ．
b.鉛直加振 円筒容器が振動数 で鉛直方向に加振される
と，＝2の場合にはパラメトリック加振メカニズムにより1/2
分数調波のスロッシングが発生する．
図10･20に，マシュー（Mathieu）方程式
dd＋－2cos2＝0 (10･90)
から得られたスロッシングの発生パターンを示す ．なお，式
(10･90)において， ：定数， ：周期，＝4，＝8π
である．
c .ピッチング加振，ピッチング加振と水平・鉛直加振 円筒
容器がピッチング加振（水平軸まわりの回転加振）を受ける場合，
および直方体容器がピッチング加振と水平・鉛直加振を同時に受
け，対称あるいは逆対称なスロッシングモードが成長する場合，い
ずれもピッチングの回転軸が容器底面の中心軸と交わらない場合に
は，ピッチング加振がパラメトリック振動になる ．
d.流れが存在する場合 流入流出を伴う矩形容器内液のスロ
ッシングでは，液面近傍での水平流速 が増加すると，ハードス
プリング型の非線形性が強くなる ．
円筒および直方体容器底面の中心から噴出された円形および長方
形（二次元）水噴流が水面に衝突すると，スロッシングが発生す
る．

0.05mの流出口を有す）で，ノズル幅0.01mの場合のスロッシ
ングの発生条件の例を，流入速度と水深との関係で示す ．
