# 5・5・1 ミクロ温度計測法

> **章**: 第5章 熱測定法
> **節**: 5・5 ミクロ物性計測法

走査型熱顕微鏡は,原子間力顕微鏡(AFM)のカンチレバーの先端に薄膜型熱電対などの熱計測センサを付与し,カンチレバーのたわみから試料表面形状を計測すると同時に,温度センサで試料表面の熱情報を計測するものである.図5・9にSThM の計測系を示す.カンチレバーのチップ先端への熱センサの作成については,いくつかの方法が試みられている.従来は研究者が自作していたが,最近ではサーミスタ型SThM プローブ(図5・10)も市販されるなど,入手はしやすくなってきている.しかし,熱計測性能はこのセンサの大きさと位置に大きく影響されるなど,いまだ問題点も多い.これらのSThM を用いて大気中で温度の高い試料表面を計測をした場合,熱は試料表面とプローブ先端の間の接点を通してだけでなく,一部は試料とプローブとの間に形成される吸着水ブリッジや,プローブ近傍の空気を伝わってカンチレバーに流入する.このため熱を検知しやすくはなるが,空間分解能はかなり悪くなる.高真空中で計測を行うとこれらの影響を排除することが可能となり,空間分解能も20nm程度まで向上することが可能であるが,カンチレバーに流入する熱量が小さくなる.いずれにしても実際の試料温度を計測することは難しく,得られる信号は何らかの熱情報と理解すべきである.試料実温度を計測するための熱フィードバックを用いたアクティブ型SThM を図5・11に示す.プローブ先端にサーモパイル,ヒータ,熱電対を設け,プローブに流入する熱流束を検出し,ヒータの加熱量を制御することでプローブ温度を試料温度と同じにし,そのときのプローブ温度を計測することにより試料温度を求めるものである.これにより,100nm以下の空図5・9走査型熱顕微鏡図5・10 サーミスタ型SThM プローブ5―140熱工学図5・11 能動型SThM図5・12 炭素繊維含有材料の形状,電気伝導性および熱伝導性計測間分解能で実温度計測が可能となる.
