# 5・5・2 ミクロ熱物性測定法

> **章**: 第5章 熱測定法
> **節**: 5・5 ミクロ物性計測法

加熱した試料表面に低温のSThMプローブを接触させると，試
料からプローブに熱が流入することにより，接触点周辺の試料の温
度がわずかに低下する．この温度低下は試料の温度拡散率に関係す
ることから，ミクロ領域の熱伝導性を調べることができる．図5･
12は炭素繊維をエポキシ樹脂で固め，断面を研磨した試料の表面
形状と電気伝導性，熱伝導性を計測したものである．炭素繊維は熱
伝導が良いため，プローブが接触しても試料内部から熱が伝わるこ
とにより高温を維持しているが，エポキシ樹脂は温度が低下してい
ることがわかる．しかし，この計測においても，試料とプローブの
接触状態が計測結果に及ぼす影響は大きく，定量的計測のためには
今後の改良が必要である．
