# 6・5・1 固体燃料

> **章**: 第6章 熱物性値
> **節**: 6・5 燃料・燃焼生成物の熱物性値

固体燃料（solid fuel）とは固体の状態で使用される燃料であり，
石炭系燃料，バイオマスならびにそれらの加工燃料が主体である．
表6･57 に固体燃料の一般性状を，表6･58 に熱物性値を示す．
石炭はその純炭発熱量，燃料比により，無煙炭，れき青炭，亜れき
青炭，褐炭に分類される．石炭化が進むにつれて炭素含有量が増加
する反面，水素，酸素含有量は減少し，純炭質の比重は増大する．
常温における石炭の定圧比熱は，石炭化が進むに従って減少する傾
向があり，温度の上昇とともに増大する．灰分の定圧比熱は
