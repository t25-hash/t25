# 7・1・2 マクロな力学へ

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・1 分子レベルからの熱流体

熱工学の問題に関しては,原子核の運動がより重要となる場合が多い.量子化学計算で求まる電子状態から,瞬間ごとにポテンシャルを決定して原子核を動かすという発想は極めて素直であり,カー(Car)とパリネロ(Parrinello)が電子状態の計算に密度汎関数法を用い,平面波近似の波動関数を用いてその可能性を示して以来,カー・パリネロ(Car-Parrinello)法として化学反応過程などの計算に用いられ,かつ各種の改良が加えられている.定常シュレディンガー方程式にB-O近似を用いたうえで,原子核の運動に伴う電子状態の時間変化を求めるのは本来自己矛盾であるが,カー・パリネロ法では,仮想的な電子質量を用いて電子密度の変化がニュートンの運動方程式に従うと仮定して解いてしまう.一般にこのカー・パリネロ法を,第一原理分子動力学法や量子分子動力学法と呼ぶ場合が多い.電子状態の計算を大幅に簡略化して,化学の分野で多用されるヒュッケル(Hu썥ckel)法あるいは拡張ヒュッケル法で見積もり,分子動力学法のポテンシャルを計算するタイトバインディング(tight binding)法も一部でかなり利用されている.ただし,ヒュッケル法は波動関数の計算を一定の経験的パラメータを利用して省略するために,ケイ素や炭素などの限られた原子間のみについてしか良い近似とならないのが現状である.さらに近似を進めて,原子間のポテンシャルを原子核の位置のみの関数で表してしまうのが,古典分子動力学法である.原子核の運動は,質点に対するニュートンの運動方程式を解くことから求まり,比較的簡単に導入が可能である.また,この方法によれば,取扱える分子数は場合によって10オーダとすることも可能で,相界面現象,核生成,熱伝導,固体面のぬれなどの問題に対するアプローチが進んでいる.古典分子動力学法からマクロな熱工学の問題への接合は,熱伝導率,粘性係数,表面張力や界面エネルギーなどの物性値の算出を通じて行う場合や,界面熱抵抗,凝縮係数や臨界核半径などの準物性的な量を用いて通常のマクロな保存方程式を用いられる場合が多い.さらに必要に応じて,希薄気体であれば直接シミュレーションモンテカルロ法(DSMC法),固体の熱伝導に関するフォノンのボルツマン方程式,固体の強度であれば有限要素法などに接合することもある.分子熱工学(molecular thermal engineering)は,熱工学で問題となる現象の,根本的なメカニズムとしての分子スケールの現象に関心がもたれるようになってきたことや,微細加工技術・微細計測技術の発達により分子スケールの現象が現実の工学的課題となってきたことなどにより,新たな観点による熱工学として広がりを見せてきた.その中でも分子動力学法(MD 法: moleculardynamics method)は,系を構成するすべての分子の位置および速度について時系列データが得られる利点があり,主要な数値解析法の一つとなっている.古典分子動力学において分子の運動を計算すること自体は,原子・分子を質点あるいは剛体とみなして,古典力学による運動方程式を差分法により時間積分するものであり,原理的にも手法としても困難なものではない.しかし,熱工学の課題を分子動力学シミュレーションで解決する場合には,以下に示す(1)~(5)などが必要となり,これらはしばしば容易ではない.(1) 課題を分解して分子動力学シミュレーションで解決可能な問題を設定する.(2) 適切な計算モデルを境界条件,初期条件とともに設定する.(3) 適切な分子間ポテンシャル(intermolecular potential)を選択する.(4) 選択した分子間ポテンシャルについて,適切な相状態を選択する(温度・密度の設定).(5) シミュレーションの結果から必要な値を得て,適切な熱工学的考察を行う.(1)はおもにスケールの問題として認識されている.分子動力学シミュレーションで解析可能なスケールは,液体・固体の場合にはたかだか空間で数十nm,時間で数nsであるが,熱工学における解析対象はこれよりはるかに大きい場合が多い.このように解析対象の現象をあるがままにシミュレーションすることが不可能な場合は,重要な要素や支配因子を切り分けて,シミュレーションが可能な問題を設定する必要がある.(2)は,標準的な熱平衡状態におけるバルク流体・固体のシミュレーションを,十分な分子数で実施するときには問題とならないが,熱流束や運動量流束が存在する場合,波動の伝ぱなど現象が非定常な場合,界面など対象の物質がバルク状態にない場合などには,特別な系を初期条件・境界条件とともに設定することが必要となり,誤って人為的な結果を得ないようにするためには十分な注意が求められる.(3)の分子間ポテンシャルは,物質普遍の特性について解析する場合には単純なモデルを用いたり,具体的な物質を対象としてその物質固有の現象を観察する場合には,その分子の特定の性質を表現しているモデルを用いるなど,シミュレーションの目的に応じて適切な選択が求められる.また,仮想の分子モデルに対して分子間ポテンシャルを変化させたパラメータスタディも,熱工学におけるエンジニアリング材料(engineering material)を選択あるいは設計する観点からは有効である.いずれにしても,シミュレーションで得られた結果は仮定した分子間ポテンシャルに対する結果であり,実在の物質に対する結果とただちに同一視することはできない.シミュレーションの結果を定性的・定量的に解釈する際には,この点に注意が必要である.(4)はこの点に関連するが,実在の分子を模擬した分子間ポテンシャルモデルは,多くの場合に実在の物質と同じ状態方程式を与えないから,例えばある温度における飽和液の現象を解析する場合に,実在の物質の飽和密度を与えたシミュレーションは,飽和液の現象を再現しない可能性がある.選択した分子間ポテンシャルによる分子系が示す飽和線や臨界点,状態方程式を知って計算条件を設定することが重要である.(5)は,物理化学など他の分野における研究を参考とするだけでは,しばしば解決しない問題である.分子動力学シミュレーションの結果として得られるのは,個々の分子の位置と速度に関する時系列データだけである.これらのデータから温度,圧力,内部エネルギー,運動量流束,熱流束などのマクロ量を求める方法はほぼ確立されているが,こうした方法が適用可能であり,得られたものがマ5―159クロな熱工学におけるこれらの概念と一致するものであるかについては,十分な吟味を必要とする.また,分子熱工学における多くの問題は,分子動力学シミュレーションによりこれらのマクロ量を求めただけでは解決しないので,どのような観点から何を観測するかが工学的課題として重要となる.以下では,分子動力学法に関する基本事項について述べる.最近の分子熱工学における研究とその成果については,文献(7)(8)にまとめられている.
