# 7・2・1 基礎式

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・2 分子動力学法

分子動力学法では、分子の運動を古典力学により解析する。一般には、以下のように系のポテンシャルφの空間こう配で与えられる。多くの場合には、系のポテンシャルφが、二つの分子の間に作用する分子間力(intermolecular force)に起因するポテンシャル(分子間ポテンシャル)φの重ね合わせで表されるとする2体ポテンシャル近似(pair-potential approximation)が、妥当な近似として用いられている。シミュレーションに当たっては、採用したポテンシャルモデルに基づいて式(7・8)により各分子に作用する力を求め、式(7・5)を時間積分することにより、時間とともに変化する分子の速度と位置を求めていく。時間積分のスキームは予測子・修正子法やルンゲ・クッタ法などはん用の方法が適用できるが、物質の構造(分子の配置)の観察を主目的とするシミュレーションではベルレ法(Verletmethod)がよく用いられる。多くの場合fs(フェムト秒、10s)のオーダである。後述のように分子の回転運動(rotational motion)や分子の変形振動を考慮する場合には、一般に並進運動(translational motion)より特性時間が短いため、時間刻みを十分小さく設定する必要がある。ベルレ法では分子の運動方程式に速度が現れない。このため、物質のエネルギー状態を取扱う分子熱工学シミュレーションでは、以下に示すかえる跳び法(leap frog method)が用いられることが多い。分子間ポテンシャルが球対称でない場合には、空間中における分子の配向が問題となる。分子の変形振動が無視できる場合は、分子の剛体運動を質量中心の併進運動と質量中心回りの回転運動に分け、分子に働くトルクをポテンシャルモデルから求めて回転運動を計算する。以下は空間座標系においてオイラーの角φ,θ,で指定される分子の配向を計算するスキームの一つである。分子の変形振動は多くの場合に量子化が著しく、古典力学で扱える範囲にないが、相状態に応じた分子の平均形状の変化を自動的に考慮するためには、これを計算に取込むことは有効な手段であり、特に複雑な高分子体のシミュレーションでは重要な問題である。これを取扱うためには、分子を構成する原子間のポテンシャル(分子内ポテンシャル、intramolecular potential)を用いて原子の運動を記述するか、分子の変形について一部の自由度を凍結して、残りの重要な変形のみを解くことが行われている。前者には単純な粒子の動力学[式(7・5)(7・6)]が、後者にはラグランジュの未定乗数に基づく拘束力学が用いられる。後者のアルゴリズムとしてはSHAKE,RATTLEなどがあり、文献(13)にまとめられている。以上述べた支配方程式は、7・2・4項で後述する周期境界条件(periodic boundary condition)を用いた平衡状態の系に対しては、NVE(分子数・体積・エネルギー)一定のアンサンブルを生成する。また、特殊な境界条件を用いることにより、温度こう配・速度こう配のもとでそれぞれ熱流束・運動量流束が生じている系を模擬することができる。一方、この支配方程式に項を付加して、NVT(分子数・体積・温度)一定あるいはNPT(分子数・圧力・温度)一定のアンサンブルを生成することができるが、内部エネルギーやエンタルピーが人為的に制御された状態にあるこれらの計算結果を熱工学で扱う場合には、その解析の妥当性について十分な注意を必要とする。また、運動方程式に外力項を付加することにより、仮想的に運動量流束や熱流束、質量流束を発生させることができ、粘性係数、熱伝導率、物質拡散係数の測定に用いられる。狭義の非平衡分子動力学(NEMD : non-equilibrium molecular dynamics)はこの方法を指すが、付加した外力項は仮想的なものであり、この項が零になる極限のみが現実と対応していることに留意する必要がある。
