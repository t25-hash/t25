# 7・2・1 項で述べたように、分子系全体が作るポテンシャル場を2

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・2 分子動力学法

分子間のポテンシャルの重ね合わせにより表現する、2体ポテンシャル近似が広く用いられている。このとき分子間ポテンシャルモデルに求められるのは、空間中に2分子のみが存在する場合の厳密なポテンシャルを、分子間距離や配向の関数として表すことではなく、高密度場にある多数の分子対に対して式(7・7)のように分子間ポテンシャルを重ね合わせたときに、結果として場のポテンシャルを表すことである。このことから、成功したポテンシャルモデルは、パラメータの値が経験的にチューニングされたものが多い。単原子分子など分子が球対称とみなせる場合に、最も広く用いられている分子間ポテンシャルモデルは、式(7・19)に示すレナード・ジョーンズ(12-6)ポテンシャル[Lennard-Jones(12-6)poten-tial,LJ ポテンシャル]である。多くの場合は粘性係数や第二ビリアル係数などの実験値を目標に、経験的に決定されている。したがって、その値は同じ分子に対しても決定法により若干の違いがあるが、例えばアルゴン(Ar)についての値の一例はσ=0.341nm,ε=1.654×10Jである。その他単純な多原子分子を含む多くの分子種についてεとσが決定されているが、特に多原子分子の場合は、目標となった実験値以外の特性を再現する保証がないことに注意が必要である。LJ ポテンシャルは、2分子が極めて接近したときに作用する強い斥力や、分子間の引力と斥力が釣合う平衡距離など分子間相互作用の基本的な特性を単純な関数形でよく表しているため、気液の相変化や界面現象など、流体が普遍的に示す熱現象の解析によく用いられる。遠距離では速やかに減衰するので、多くの場合は3-4σの値を切断距離(cut-offdistance)として設定し、これ以上遠距離からの影響を考慮しない。切断距離で残存するわずかな値によるポテンシャルおよび力の不連続とその取扱いについては、文献(21)(22)を参照されたい。先に述べたように、一般的にはあるパラメータ値を与えたLJ ポテンシャルによる流体の状態方程式が、実在流体のそれと一致していることは期待できないので、特に飽和液のシミュレーションを企図した場合などには、計算条件の温度・密度を設定する際にLJ ポテンシャルによる流体の状態方程式、または飽和線図を参照する必要がある。二次元LJ 流体については文献(23)~(27)に、三次元LJ流体については文献(28)~(30)に臨界点等とともに報告されている。実在の物質についてシミュレーションを行う場合には、多くの場合その物質について提案されている分子間ポテンシャルを検索して、適当なものを選択することになる。水を例にとると、その物質としての重要性や分子内の永久双極子などの複雑さから、さまざまなポテンシャルモデルが提案されている。まず、分子の変形を考慮しない剛体分子モデル(rigid molecular model)について述べる。初期には固体および液体水の特異な構造である正四面体構造を直接模擬して、酸素を正四面体の中心として二つの水素と孤立電子対を頂点に配置したモデル(ST 2)が用いられたが、その後は図7・1に示すように3原子上に電荷を置く(SPC,SPC/E等)か、正電荷のみ水素原子上に置き、負電荷は3原子と同じ平面上で角HOH を二等分する線上に置く(MCY,RWK,TIP4P,CC等)モデルがほとんどである。HOH の角度は、正四面体構造を模擬した109.47°(ST 2,SPC,SPC/E等)か、孤立水分子の実測値である104.52°(MCY,RWK,TIP4P,CC等)が用いられる。OH 間の距離は、実測値である0.09572nm(MCY,RWK,TIP4P,CC等)か、0.1nm(ST 2,SPC,SPC/E potential)を重ね合わせるものである。分子の変形を考慮するflexibleなポテンシャルモデルも数多く提案されているが、その多くは上記をはじめとする剛体ポテンシャルを分子間ポテンシャルとして、これに分子の変形に対応したポテンシャルを付加したものである。数少ない例外としては、酸素原子間、水素原子間、酸素-水素原子間に球対称のポテンシャルを定義して質点の動力学に帰着させるものがある。さらに、場のすべての分子の電荷により形成される電場を計算し、この電場により分子内に生じる誘導双極子を求めるpolarizableなポテンシャルモデルも多く存在するが、熱流体シミュレーションに用いられた例は多くない。これらのポテンシャルを用いて物性値や状態方程式を求めた研究5―161や、複数のポテンシャルを比較検討した研究も数多く報告されているので、ポテンシャルを選択する場合にはこれらを参照して検討することが重要である。また、先に状態方程式または飽和線図の重要性について述べたが、これらがすでに報告されているかどうかが、ポテンシャルモデルを選定する際に考慮すべき点の一つである。水については、SPC/Eポテンシャルが単純な関数形であるにもかかわらず優れており、飽和液密度や表面張力、状態方程式や誘電率などに良い再現性が報告されている。ほかにはCCポテンシャルやSPCポテンシャルについて、状態方程式や飽和線が報告されている。これらの決定法については7・2・5項で述べる。水のクーロンポテンシャルのように減衰が遅いものは、LJポテンシャルなどとは異なり、適当な切断距離を設定してその外からの影響を無視することができない。このため、計算負荷はLJ分子系と比較して極めて大きく、分子数の増大に伴って急激に増大する。クーロンポテンシャルの取扱いについては、境界条件に関連して
