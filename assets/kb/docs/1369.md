# 7・2・4 項で後述する周期境界条件(periodic boundary condition)

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・2 分子動力学法

を適用しつつ,計算セル全体で流束を観測できる最も効率的な方法として頻繁に用いられる.この場合は,付加した外力項の値を変化させて得たデータから外挿して,外力項が零の場合における値を求める.次に,平衡状態のシミュレーション(EMD :equilibriummolecular dynamics)において,速度や運動量,エネルギーの散逸過程(dissipation process)を追跡する方法が用いられている.これらの量の自己相関係数と輸送係数との関係は,グリーン・久保(Green-Kubo)の公式により与えられる.最後の方法は広義のNEMD といえるものであり,EMD と同様になんら人為的な項を含まない支配方程式を用いつつ,特殊な境界条件を適用することにより各種流束を発生させるものである.この方法は単に輸送係数を求めるためには効率が悪いが,現実に一定の大きさの流束が発生している状態の観察を可能にする唯一の方法である.
