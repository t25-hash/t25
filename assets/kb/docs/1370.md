# 7・2・4 境界条件

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・2 分子動力学法

まず,初期条件について述べる.熱平衡かつ定常状態にある熱流体のシミュレーションでは,ほとんどの場合個々の分子の位置や速度が問題になることはないので,初期条件の設定とは,温度・密度・圧力など設定したマクロ量をもつ一様な分子系を作成することである.最も一般的なNVE一定のシミュレーションでは,まず計算条件の密度から系の体積と分子数を決定する.シミュレーションが進行して分子が相互干渉を繰返すとともに平衡状態が自然に達成されるから,計算開始時点の分子の位置や速度はどのようなものでも良い.位置は一様乱数で,速度は正規乱数で与えるのが一般的であるが,必須ではない.ただし,この状態から出発して平衡状態に達したときの系のエネルギーは,計算開始時の읜은읜윷윽=읜읚+읚은읜+φが保存されて運動エネルギーとポテンシャルエネルギーに配分された状態にあるので,系は設定しようとする目標温度となっていない.この状態を,目標温度における熱平衡状態に変化させてシミュレーションの初期条件とするには,읜은읜윷윽を増減して計算開始時の値から変化させる必要がある.これには,7・2・1項で触れたNVT 一定のシミュレーションを行うことも有効であるが,以下に述べる速度スケーリング法(velocity scaling method)が最も簡便である.すなわち,現状の系の温度を分子の速度から求めて,その温度を目標温度に近づけるか一致させるように,各分子の速度を一様な倍率で増加あるいは減少させる.これにより,分子の運動エネルギーを通じて系にエネルギーを注入あるいは除去して,읜은읜윷윽を目標温度における値に近づけていることになる.温度を速度スケーリングにより拘束した数千から数万ステップの計算により,읜은읜윷윽が一定値で安定すれば,それがその温度におけるNVE一定の計算の初期条件として利用できる.なお,速度スケーリングにより温度をつねに目標温度に保った状態での計算結果は,一定の精度でNVT 一定のアンサンブルとして妥当だと考えられているが,この状態のデータを熱現象の解析に用いる場合には,この操作が現象に影響を与えていないか注意が必要である.以上述べた初期条件は,一定温度の定常状態で緩和計算を十分行うことにより容易に得られるものであるが,例えば,気泡核生成のシミュレーションで初期条件とする過熱状態など,定常状態にない現実的な状態を作るのは困難な課題である.シミュレーションで観測しようとしている現象の本質と,それに影響を与えない人為的操作の選択について洞察が必要である.以下では空間の境界条件について述べる.特に平衡状態のシミュレーションにおいて,無限に広い領域を限られた数の分子から成る系により表現するために,頻繁に用いられるのが周期境界条件である.7・2・1および7・2・3項で述べた狭義の非平衡分子動力学(NEMD)シミュレーションにおいても,3軸すべての方向にこれが用いられる.また,後に述べる系に温度差を与える場合などの非平衡状態に対しても,温度こう配のない方向に用いられる.まず,図7・3に示すように,実際に分子を配置して計算を行う基本セル(basic cell)を設定する.基本セルは直方体で,多くの場合立方体である.基本セルに対してすべての方向にイメージセル(imagecell)を想定し,すべてのイメージセル中の分子の配置は基本セルと同一とする.基本セルの一つの境界を通って基本セルを出た分子は,同時に対向する境界から基本セルに入ってくることになる.分子には基本セル内の他の分子と,イメージセル中の全分子から,ポテンシャルと分子間力が作用するが,LJ ポテンシャルなど遠距離で減衰が大きく切断距離を設定できるポテンシャルでは,切断距離の2倍よりセルの一辺の長さが大きいことを前提として,分子に作用する分子からの相互作用を計算する際に,基本セル中の分子と基本セルを囲む26個のイメージセル中にある分子のイメージの中で,最も分子に近接したもののみを考慮する(最近接写像法,minimum image convention).一方,分子内の電荷間のクー図7・2分子運動による流体の運動量の伝搬メカニズム図7・3 周期境界条件図7・4系に温度または速度分布を与えるためのNEMD 基本セルロンポテンシャルによる遠距離力については,無限遠まで存在するイメージ分子の寄与をすべて考慮する必要があるが,その座標の周期性を利用してより速く減衰する関数に変換するエバルドの方法(Ewald method)を用いるのが一般的である.分子系に温度こう配や速度こう配を実際に与える場合には,1方向に長い直方体の基本セルを用いて,長手方向に温度・速度のこう配を与える計算系がよく用いられる.温度・速度のこう配を与えるためには,7・2・5項で述べるような固体壁をセル長手方向両端に設置して,その温度・速度を一定に保つか,図7・4に示すように基本セルの長手方向両端にコントロール領域を設定して,その領域内の分子について平均温度または速度を設定する.後者の場合には,流束などを観測する検査体積は,セル両端のコントロール領域から十分離れたセル中央部に設定する.コントロール領域の温度制御は速度スケーリングが適用できる.速度制御はコントロール領域の分子の平均速度と目標速度との差を各分子の速度に一律に加えることで行われる.壁を設置しない場合は,セル中の分子が逸出するのを防ぐため,セル両端において,(1)分子が反射される鏡面の境界条件を設定する,(2)セル端と分子との距離に応じた近距離ポテンシャル(例えばLJポテンシャル)を全分子に適用する,(3)図7・5に示す対称的基本セルに対して周期境界条件を適用する,などの方法が用いられる.その他の2方向には通常の周期境界条件が適用される.液体に速度こう配を与えた場合には粘性加熱により液相の温度が上昇するが,これを自然に任せるか温度条件の変化を避けて液体に温度制御を加えるかは,シミュレーションの目的によって選択されている.
