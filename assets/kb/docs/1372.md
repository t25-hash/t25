# 7・2・2 項で述べたように、特定のポテンシャルモデルを適用した

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・2 分子動力学法

流体が示す状態方程式や飽和線を知ることは、計算条件の温度・密度を設定する場合に必須である。また、適切な温度・密度の設定により飽和状態だけでなく、過熱液・過飽和蒸気など準安定状態を再現することも可能である。状態方程式を得るには、片岡の方法が一般に用いられている。これは、予想される飽和線や超臨界域を含んだ広い領域における、数百点の温度・密度条件について分子動力学シミュレーションを行い、得られた圧力およびポテンシャルエ5―163図7・5 三次元周期境界条件を適用する場合のNEMD 基本セル図7・6 気液共存状態および気液界面のシミュレーションにおける基本セルネルギーに基づいて、最小二乗法により状態方程式を決定するものである。1条件のシミュレーションは数百程度の分子数、数千から数万程度のステップ数により行われることが多い。気液飽和線のみを得る場合には、上記の方法で得た状態方程式から求めるほか、モンテカルロシミュレーションでギブスアンサンブル(Gibbs ensemble)を得て、共存状態の気液の密度を決定する方法や、図7・6のように3方向に周期境界条件を適用したセル内に気液共存状態を作製して、気体・液体の密度を直接計測する方法などがある。表面張力を求めるなど気液界面のシミュレーションにおいても、図7・6の計算系は多く用いられている。LJ 流体などさまざまなモデル流体や水について、気液共存状態の密度や界面張力が求められている。ここでは界面法線方向座標、ρ,ρはそれぞれバルク液相および気相の密度で、は界面領域の厚さの指標として用いられる。右辺の最初の1/2は図7・3の系に界面が二つ存在することによる。なお、上述の小領域ごとに圧力を求めて式(7・32)により局所表面張力の分布を求めることもよく行われている。固液界面あるいは固体表面近傍における液体のシミュレーションは、多孔質体の利用や溶液中の結晶成長などに関連して数多く行われている。この場合、固体壁面と液体分子との相互干渉は、大別して、固体を形成する分子の影響を織り込んで壁からの距離の関数として表現された壁面ポテンシャルを用いる方法か、実際に壁を分子で構成して固体分子-液体分子間のポテンシャルを定義する方法のいずれかで取扱われる。一方、壁面に温度やマクロな速度を与えて固液界面にエネルギーや運動量の流束を生じさせる場合には、後者による壁面を採用する。例えば壁面分子間をハーモニック(ばね)ポテンシャル(harmonic potential)で記述し、壁面分子の温度を速度スケーリングかファントム(phantom)分子法で制御する。ファントム分子法は、一定温度の半無限固体からの干渉を乱数を用いて繰入れるもので、数層の固体分子層の外側に設置して熱浴として利用する。非平衡の弱電離低温プラズマを用いて半導体等機能材料の製造・加工を行うプラズマプロセシング(plasma processing)は、半導体産業の基盤技術である。このようなプラズマは電子・イオン・ラジカル・分子から成るが、数万K の高い電子温度に比較して、重粒子であるイオン・ラジカル・分子の温度は300~1000K 程度であり、粒子間の温度非平衡は著しい。また、ガス圧力がたかだか10Pa程度と低いため粒子間の衝突頻度が十分ではなく、このため粒子の速度分布関数(velocity distribution function)が非マクスウェル分布をしている。したがって、プラズマプロセスに登場する各種粒子の流れは、ボルツマン方程式(Boltzmann equation)に基づいて取扱うことになる。ボルツマン方程式は粒子の速度分布関数の時空間分布を支配する方程式であるが、非線形の微積分方程式であり、そのままでは数値解を求めることさえ容易ではない。そこで速度分布関数の代わりに、この確率密度関数からランダムサンプリングした標本集団の時間発展を計算する方法が新たに登場し発展してきた。粒子が電磁場の影響を受けない場合には、ボルツマン方程式のこのような解法は直接シミュレーションモンテカルロ法(DSMC法: direct simula-tion Monte Carlo method)と呼ばれてきた。これについてはα4編「流体工学」第16章を参照されたい。一方、粒子が電磁場の影響を受ける場合には、セル内粒子モンテカルロ法(PIC/MC法:particle-in-cell Monte Carlo method)と呼ばれてきた。ここでは後者の場合もDSMCと呼び、この場合を中心に述べる。
