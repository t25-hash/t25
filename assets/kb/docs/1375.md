# 7・4・1 マイクロチャネル(micro channel)の熱流動

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・4 マイクロ熱工学

サブミリからマイクロスケールの流路は、インクジェットプリンタ、シリコンチップ上の化学分析装置、電子デバイスの直接冷却など、工業製品への応用の機運とともに1995年ごろから盛んに研究が行われている。これらの流路では、単純なスケール効果による圧力損失・熱伝達率の増大、レイノルズ数が低下することによる流体混合の抑制が生じるが、そのほかに、希薄気体効果、電気二重層の影響など、通常スケールとは異なる現象が現れる。マイクロ流路内の熱流動については、理論解析、計測技術、数値解析のいずれも解決すべき課題が多く残されており、ここでは研究の現状を示す。a. 流路製作法シリコンウェーハ上のマイクロ流路は、水酸化カリウム(KOH)、水酸化テトラメチルアンモニウム(TMAH)による結晶異方性ウエットエッチングにより製作されることが多い。ミラー指数(100)のウェーハを用いた場合、断面形状は底角125.26°の台形となる。KOH はTMAH よりも滑らかなエッチング面が得られ、濃度、温度、添加物について十分に最適化されたプロセスでは表面粗さを数nm程度とすることが可能である。エポキシ系フォトレジスト(SU-8)の厚膜や、シリコーンゴム(PDMS : polydimethylsiloxane)の成形を用いた製作も最近広く行われている。b. 表面粗さマイクロ流路内の流れのレイノルズ数は通常小さいので助走区間は短く、流れのほとんどの領域は発達した層流状態となる。したがって、流れの抵抗、熱伝達について第一次近似としては層流解を用いることができる。しかし、流路の表面粗さは製作方法に強く依存し、2 m程度の粗い面をもつ流路では、粗さが流路高さの数%オーダとなって摩擦抵抗が最大40%増大したとの報告もある。なお、層流から乱流への遷移については、700程度の低いレイノルズ数で発生するとの研究例もあり、極めて限られた条件での伝熱熱伝達相関式も提案されているが、表面粗さが遷移レイノルズ数に与える影響について系統的な検討は行われていない。c. マイクロ流路内の気流気流の代表寸法が分子の自由行程(常温常圧の大気で~0.07 m)に近づくと、希薄気体効果により粘性流れとしての通常の取扱いを修正する必要が生じる。スリップ流れ(slip flow)の範囲では、壁面で有限の速度が存在し、温度分布は不連続となる。σは、壁面に衝突する分子の方向運動量がどの程度保存されるかを表し、σ=0は鏡面反射、σ=1は拡散反射に相当する。σは熱的適応係数であり、本来は並進、回転、振動温度について異なる値をとると考えられるが、通常は単一の値で代表させる。適応係数の値は気体と固体の種類、表面粗さなどに依存し、特定の組合せについては実験値が与えられているが、機械加工した面ではσ=σ~1である。なお、式(7・46)の第2項は熱ほふく流、すなわち温度こう配に沿って流体が移動する効果を表す。圧力こう配で駆動させるマイクロ流路内の流れでは、通常式(7・46)第2項の影響は小さく、第1項のみが用いられることが多い。また、流路長さと流路幅の比が非常に大きいマイクロ流路では、流速が小さいにもかかわらず圧縮性の影響が現れる。したがって、流れ方向圧力こう配は一定でなく、圧力損失と流量の関係が非線形となる。図を示す。流路高さは1.33 マイクロ流路内の液流作動流体が液体の場合は希薄気体効果は存在せず、逆に水などの極性流体に対しては、電気二重層(electric double layer)、界面動電現象による影響が現れ、抵抗が増大する可能性がある。すなわち、壁面の電荷により液体分子が壁面に吸着するため実質的な流路幅が狭くなるとともに、拡散層内に生じる流れ方向電位により流れと逆向きの力が働く。抵抗増大率は、イオン濃度、イオンの価数、液体の電気伝導度などに大きく依存するが、電気二重層の厚さ1 m程度の超純水では、流路高さ14mにおいて約9%となる。e. 熱伝達マイクロ流路内の単相流として表面粗さの影響のみを考慮した場合、通常スケールと同様に、熱伝達よりも摩擦抵抗のほうが相対的には大きくなると予想される。一方、スリップ流れ、電気二重層など分子の効果が顕在化する条件では、熱伝達の計測例は極めて少ない。コルバーンのアナロジーファクタが顕著に増加したとの測定例もあるが、式(7・46)(7・47)を用いた数値解析では、壁面摩擦と同様、熱伝達率も低下する結果が得られている。沸騰を伴うサブミリスケール流路内の流れでは表面張力の影響が支配的となり、流動様式は無重力下と類似の傾向を示す。数mm程度の伝熱管ではクオリティ0.4程度で熱伝達率が最大となり、また管径が小さいほうが熱伝達率が高いが、サブミリスケールではドライアウトが比較的小さなクオリティで発生するため、熱伝達率は必ずしも高くない。したがって、水の単相流の熱伝達率のほうが、フレオン系冷媒の沸騰熱伝達よりも高い場合も生じる。さらに、沸騰開始点が沸点を超える過熱現象が生じることも明らかにされつつあり、マイクロ二相流熱交換器の設計には注意が必要である。
