# 7・4・2 マイクロマシン

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・8 に,横軸にマイクロ流路内の圧力比,縦軸に流量の計測値

マイクロマシン(MEMS : micro-electro-mechanical systems)の応用分野として,光通信機器,工業・医療用センサ,マイクロ生化学分析デバイス,電子デバイスの冷却システム,伝熱・流体制御,マイクロ無線機器などが考えられているが,その中で熱工学とマイクロマシンの接点としては,以下の三つが挙げられる.(1) マイクロマシン周りの伝熱,流れ現象の解析を通じて,熱工学の知識をマイクロデバイスの設計に役立てる.(2) マイクロセンサ,アクチュエータを,既存の伝熱・熱流体機器の診断,熱流動の制御に応用する.(3) 両者の融合により,新たな機械システムを創造する.スケールが小さくなるに従って粘性力,界面張力などの表面力の影響が増大するだけでなく,希薄気体効果,ブラウン運動,電気二重層,温度こう配や電位差による表面張力駆動の対流などが顕在化するため,通常スケールとは質的にも異なったデバイスも提案されつつある.以下では,伝熱関連のマイクロデバイスについての例を示す.a. 熱式センサ熱線流速計と同じ原理を用いた熱式のマイクロセンサは,流速,流量,壁面せん断応力の測定を目的として多く試作されている.ヒータ部を微小にすることによって熱容量が減少するため,高い周波数応答が実現できる.ヒータの材質としては,多結晶シリコン,白金などが用いられる.図7・9に,マイクロせん断応力センサの試作例を示す.一辺200 m,厚さ0.4 mのSiNのダイアフラム上に,幅2 m,長さ40 mの多結晶シリコンのヒータが形成されている.SiN膜の下部は厚さ1.8 mの真空キャビティになっており,基板への熱損失を防ぎ,消費電力の低減と周波数特性の向上に寄与している.熱式センサとしては,こ5―167図7・9マイクロせん断応力センサ図7・10 フラップ型熱変形アクチュエータ図7・11 熱毛細管型光スイッチのほか熱伝導率変化,触媒発熱反応を用いた濃度センサ,赤外線センサなどが試作されている.b. 熱式アクチュエータ通常スケールでは伝熱現象の時間応答は遅いが,スケールが小さくなることによって熱容量が極めて小さくなるため,マイクロアクチュエータの動作原理として伝熱を用いたものも多く提案されている.図7・10に,熱変形を用いたフラップ型アクチュエータの例を示す.ヒンジ部のV 溝にポリイミドを埋め込み,ポリイミドの大きな熱収縮率と熱膨張率を利用することによって,三次元構造の組立,動的な大変形が可能であり,マイクロスケールでの搬送やマイクロロボットの歩行などへの応用が検討されている.図7・11に,表面張力の温度変化を利用した気泡駆動の光スイッチの例を示す.光導波路に基板と屈折率の等しい液滴を極微量封入した微細流路を形成し,マイクロヒータで温度こう配を変化させることにより液滴を移動させ,入射光を透過・反射させる.時間応答は100msであり,全反射を利用するため光の損失が少ない.そのほかに,表面張力の電場依存性を利用したアクチュエータ,液体から気体,あるいは固体から液体への相変化を用いたアクチュエータなどが提案されている.c.
