# 7・4・3 半導体冷却

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・8 に、横軸にマイクロ流路内の圧力比、縦軸に流量の計測値

半導体素子の発熱密度の増大は、機器の性能や信頼性を損なうものとして大きな問題になりつつある。また、機器の小型化、薄型化が加速しており、その結果、2010年で大規模計算機用の半導体の消費電力は200W から600W に上昇することが予想され、その発熱密度は500kW/mに達するとされている。また、チップサイズに制限を加えると、その密度は1000kW/mをも視野に入れる必要がある。よって最新の研究では、130°C以下のレベルで1000kW/mのチップ放熱を指向した研究も報告されている。そこでは、マイクロチャネル(micro channel)を流れる液体の沸騰現象を伴う流熱特性を研究している。もともと、マイクロチャネルを用いた冷却法はタッカーマン(Tuckerman)らによって提案されたもので、半導体チップの裏面にミクロンオーダの溝を切り、そこへ冷媒を流すため、半導体冷却で問題となる冷却部の接触熱抵抗を大幅に軽減でき、冷却性能を向上させることができる。また、浸漬冷却のようにチップ自体を冷媒の中に浸すことがないため、機器の信頼性が向上することも期待される。その後も、半導体冷却指向のマイクロチャネルを用いた熱交換技術に関しては、さまざまな研究がなされている。しかし、それらの研究は基礎的な段階にとどまっており、実用的なレベルのものはまだ少ないが、マイクロチャネルを実際に応用した例としては、高密度レーザダイオードアレイ(LDA : laser diode array)の冷却がある。図7・13は、約1×4mmのLDA が8×10mmの水冷式マイクロチャネルヒートシンクで冷却される例である。LDA の発熱量は500W/cmである。ヒートシンクはシリコン[熱伝導率150W/(m/K)]で作られており、100×400 mの矩形チャネルに水を流している。各流路平均12.5m/sの平均流速、レイノルズ数で約2000である。圧力損失は482kPaである。この冷却法で平均的な熱抵抗は0.049K・cm/W であった。また最近、図7・14のように、実際にシリコンチップにマイクロチャネルを形成して、実機をイメージした冷却装置を作製し、コンパクト性や信頼性などを中心に、実際の電子機器冷却への適用可能性を検討した例がある。図7・15に示すような形状のマイクロチ図7・14 マイクロチャネルを用いた熱交換器のプリント基板搭載イメージ図7・15 マイクロチャネルチップ構造図7・16 マイクロチャネルチップの熱抵抗ャネルチップ(チップ幅×高さ:57×180 mと57×350 mの2種類で、チャネルは100 mピッチで110本作られている)を作製し、チップ加熱面上に形成した薄膜熱電対を用いて、伝熱性能評価や圧力損失評価を行っている。図7・16には、57×180 mのマイクロチャネルチップの熱抵抗を示している。熱抵抗値は、チップ表面温度と冷媒入口温度との差を単位面積熱流束で除した値である。その結果、チャネル内平均流速とチャネル水力直径によってを定義し、=200以上では、入口付近で0.1K・cm/W、出口付近で0.2K・cm/W と小さな値を達成している。しかし同時に、入口および出口付近におけるチャネル出入口間の熱抵抗の差も約0.1K・cm/W であった。これは100W/cmの熱流束を加えた場合、チップ内に約10°Cの温度差がつくことを示している。このことは、平均熱抵抗では他の冷却手法に比べてマイクロチャネル冷却は大きな伝熱性能を示すが、チップ内温度差による熱応力の発生には注意を要することを示している。また、図7・16では、チャネル内を完全な層流と仮定した数値解析との比較を示しているが、=200以上では両者はよく一致している。図7・15のマイクロチャネル熱交換器の設計では、57×350 mのチャネルを使用し、チップ表面の平均熱抵抗が0.1K・cm/W となるようにポンプ仕様が決定された。つまり、圧力損失0.07MPa、全流量3.8cm/s,=220、チャネル内平均流速2m/sである。実験でも平均熱抵抗0.1K・cm/W が達成されている。
