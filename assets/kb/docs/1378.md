# 7・4・4 マイクロ熱交換器(micro-heat exchanger)

> **章**: 第7章 分子・マイクロ熱工学
> **節**: 7・8 に、横軸にマイクロ流路内の圧力比、縦軸に流量の計測値

MEMS,-TAS などマイクロ機器の開発が進展する一方で、その周辺機器である電源、駆動制御系、熱交換器等は依然として大きいままであることが多く、これら周辺機器の小型化の要求は大きい。熱交換器やヒートシンクの場合、小型化により熱伝達率が大きくなること、また熱伝導距離を小さくすることができる可能性があることから、高性能な熱交換器が期待できる。また、マイクロ熱交換器は冷媒等の使用量も少なくて済むことから、万が一、リーク事故が発生したとしても危険性は少ないなどのメリットが考えられる。小さい温度差でできるだけ多くの熱交換を行うという点を考えると、この熱抵抗というパラメータはマイクロ熱交換器の性能の指標としても有効である。マイクロ熱交換器としては、マイクロチャネル型や噴流型がこれまでに製作されており、ここではマイクロチャネル熱交換器について述べる。マイクロチャネル熱交換器の隔壁の厚さは数百m程度であることが多く、一般的には伝導熱抵抗は伝達熱抵抗よりもかなり小さい。容量熱抵抗は作動流体の流量とともに減少するが、大きさとしては伝達熱抵抗と同じ程度である。伝達熱抵抗を小さくするためには、流路直径を小さくすること、熱伝導率の良い流体を用いることが考えられる。容量熱抵抗を減少させるためには、流量を増やすことが挙げられる。以上がマイクロチャネル熱交換器で小さい熱抵抗を達成するための指針であるが、流路直径を小さくすると必要となるポンプ仕事および圧力が大きくなってしまうので、与えられた条件や制約によって、どの程度まで流路直径を小さくするかが決まることになる。図7・17は、水冷マイクロチャネルと他の代表的な冷却方法の適用領域を比較したものであり、小さい熱抵抗が要求される場合にはマイクロチャネル冷却が最も優れていることがわかる。b. 単相流の伝熱・流動特性水を作動流体とし、沸騰を伴わない場合のマイクロチャネルの伝熱特性としては、以下の実験整理式が提案されている(図7・18)。一方、流動特性としては、流量が小さい層流領域においてはマクロな配管やベント部における圧力損失の整理式を用いて予測可能であり、流入部の圧力損失は極めて小さいが、流量が多くなると入口部において流れのはく離が発生することにより、予測値より圧力損失が増大することがある。ヘッダ部の最適形状に関して図7・17 各冷却手法の適用範囲5―169は、圧力損失だけでなく流量配分の均一化の観点からも今後の検討が必要である。c. 沸騰相変化を伴う流れの伝熱・流動特性熱流束が増加した場合にはチャネル内で沸騰が生じ、表面張力の効果が顕著に現れて、スラグ流と呼ばれる流動様式が多くなる。この場合の伝熱および流動特性は単相の場合とは大きく異なり、沸騰により圧力損失の増大が顕著となる。この圧力損失のおもな原因は、沸騰により気体が急激に膨張することによって、液相が急加速されるときの慣性力である。図7・19は沸騰相変化を伴うマイクロチャネル流れの実験から得られた流量、圧力損失および熱流束の関係の一例である。沸騰によって圧力損失は急激に増大するが、伝熱特性はそれほど良くはならないこと、流路途中でドライアウトが生じた場合には、下流における熱交換性能は大きく低下することを考慮すると、マイクロチャネル熱交換器は液単相の条件で用いたほうが有利である。
