# 11・10・3 不規則応答のシミュレーション

> **章**: 第11章 不規則振動
> **節**: 11・10 不規則振動のシミュレーション

11･10･2項の手順によって生成した疑似不規則入力のそれぞれに
対して応答の解析を行って，それらの応答の時刻歴の統計処理を行
うことにより，不規則応答のシミュレーション結果を得ることがで
きる．
このそれぞれの応答の数値解析では，生成された疑似不規則入力
を確定励振として扱うことになるので，通常の確定励振での応答解
析法である，ルンゲ・クッタ法，ルンゲ・クッタ・ギル法，予測子
修正子法などがよく用いられる．近年では，数値計算によってシミ
ュレーションを行うことがほとんどである．
