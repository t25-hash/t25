# 2・2・3 重み付き残差法

> **章**: 第2章 数値計算法の基礎
> **節**: 2・2 数値計算法の基礎

微分方程式に近似関数を代入すると残差(residual)を生じる。重み付き残差法(weighted residual method)は、残差に重み関数を掛けて領域全体で積分したものが零となるように解を定める方法である。以下に具体的手順を示す。重み関数の選び方によって次の6種類の解法が生まれる(図2・5)。a. 部分領域法(subdomain method)領域をいくつもの部分領域(subdomain)に分割する。重み関数をその部分領域の一つの中で1、外では零となるように選ぶ。この方法は有限体積法(finite volume method)と一致する。b. 選点法(collocation method)重み関数としてディラック(Dirac)のデルタ関数を選ぶ。すなわち、残差は選点において零となり厳密解と一致する。特に選点として直交関数を形成する固有点を選ぶと効果的である。d. ガラーキン法(Galerkin method)重み関数を近似関数を構成する形状関数に選ぶと、有限要素法(finite elementmethod)のガラーキン法による定式化となる。e. 境界要素法(boundary element method)の随伴微分演算子の基本解を重み関数に選び、部分積分を2回行って逆形部分領域法(有限体積法)選点法最小二乗法ガラーキン法(有限要素法)境界要素法モーメント法重み付き残差法図2・5重み付き残差法から生じるいろいろな解法最小作用の原理エネルギー原理変分原理重み付き残差法部分積分境界要素法巨大連立一次方程式図2・8 数値解析の体系式を作ると境界要素法となる。f. モーメント法(moment method)モーメント法は重み関数として、1,...を用いる方法である。以上をまとめると図2・6のようになる。すなわち、有限要素法、有限体積法、境界要素法は、重み付き残差法の特殊な場合と考えることができる。有限要素法は重み関数として形状関数を選び、1回部分積分し弱形式で定式化したものである。境界要素法は重み関数として基本解を選び、2回部分積分して逆形式で定式化したものである(図2・7)。有限体積法は重み関数を1として各要素ごとに積分し、各要素の平均値として微分方程式を離散化したものである。
