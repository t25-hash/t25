# 2・3・2 特性理論

> **章**: 第2章 数値計算法の基礎
> **節**: 2・3 特性理論と固有値・固有ベクトル

次に、システム方程式について考察する。a. システム方程式(system equation)一般に高階の偏微分方程式は、1階の連立偏微分方程式系(システム方程式)に書き換えることができる。これをシステム方程式と呼ぶ。固有方程式の根が、互いに相異なる実根λ,λ,...,λをもつならば、システム方程式は全双曲形偏微分方程式といい、逆に一つも実根をもたないならば、だ円形偏微分方程式という。また、全双曲形とだ円形の中間もありうるがここでは必要でなく、全双曲形について話を進める。これはの相似変換(similar transformation)を表している。ここでΛ,は、主対角線上のみが零でない成分をもつ行列である。ここで ̄はによって変換された量を意味し、この式をシステム方程式の標準形という。これをシステム方程式から得られる常微分方程式の第一積分、またはリーマン不変量という。
