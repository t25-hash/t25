# 2・3・3 オイラー方程式と特性曲線

> **章**: 第2章 数値計算
> **節**: 2・3 特性理論と固有値・固有ベクトル

a. オイラー方程式(Euler equation)ここではオイラー方程式に特性理論を適用する。ここで=const.はリーマン不変量で、特性曲線に沿って一定となる。特性曲線法によれば、図2・12の点B での未知ベクトルρ,の値は、3個の特性曲線に沿ったリーマン不変量の連立方程式における解として求まる。よって、点A,A,Aの既知の値から点B の未知の値が求まる。. 断熱変化(adiabatic change)の場合断熱変化の場合、特性理論はさらに簡単化される。一般に圧力が密度だけに依存図2・13 巨大連立一次方程式の解法図2・12 特性曲線とリーマン不変量する流体をバロトロピック(barotropic)流体という。この場合、状態方程式を用いると密度ρが消去できる。この場合変数はとで、2本の特性曲線の交点として点B の未知量が点Aと点Aより求まる(図2・12)。. リーマン不変量の求め方断熱変化の状態方程式はρ=const.で与えられる。積分するとき、音速も圧力の関数となっていることに注意すべきである。計算の高精度化に伴う要素分割の増大により、巨大な連立一次方程式の電子計算機による高速解法が望まれている。すなわち、より巨大な連立一次方程式をいかに精度良く高速に解けるかは、実際に物を設計し作るうえで必要である。その解法は直接法と反復法に分類される。ここで、は下三角行列(lower triangle matrix)、は対角行列(diagonal matrix)、は上三角行列(upper trianglematrix)である。ここで、とは主体格成分がすべて零の、狭義の下三角行列と上三角行列である。以上LDU 分解(LDU decomposition)について説明したが、D がないのがLU 分解である。次に直接法と反復法の代表的な解法を示すと6―13改良コレスキー法:対称行列(LDU 分解)直接法コレスキー法:対称行列(LU 分解)ガウスの消去法:対称行列、非対称行列巨大連立一次方程式の解法クロー法:非対称行列(LDU 分解)SOR 法:対称行列、非対称行列反復法SD 法:最急降下法CG 法:共役こう配法直接法[ガウスの消去法、コレスキー(Choleski)法、クロー(Crout)法、FACR(Fourier analysis cyclicreduction)法]反復法(SOR 法、SD 法、CG 法、モンテカルロ法)となる(図2・13)。これらを基礎としていろいろな改良版がある。
