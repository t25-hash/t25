# 2・4・1 直接法(direct method)

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

直接法の基礎はガウスの消去法である.対称行列,非対称行列を問わず100元以下の行列であれば,ガウスの消去法は強力で高速な解法である.しかし,1000元以上の巨大行列に対しては,より有効な解法が考えられている.対称行列の解法としてはコレスキー法または改良コレスキー法が有効であり,非対称行列の解法としてはクロー法がよい.コレスキー法は行列の対称性を考慮してLU 分解する方法で,ガウスの消去法よりも速い.改良コレスキー法は対称行列をLDU 分解するもので,コレスキー法の改良版になっている.クロー法は非対称行列をLDU 分解して解く方法で,対称行列の改良コレスキー法に対応する.
