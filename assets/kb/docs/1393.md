# 2・4・2 反復法(iterative method)

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

反復法の代表としてはSOR 法、SD 法、CG 法がある。a. SOR 法反復法でいちばんよく用いられるのは、SOR 法(successive over relaxation method)である。SOR 法の基本的な考え方はヤコビ(Jacobi)法にある。SOR 法の変形にはSSOR(symmetric SOR)法、USSOR(unsymmetric SOR)法などがある。b. SD 法SD 法(steepest descent method、最急降下法)は、ある関数の極値探索をする方法である。そのために最大傾斜をたどっていくというのが、SD 法の基本的な考え方である。この方法は一見良さそうであるが、収束が悪く実用的価値が少ない。c. CG 法SD 法を改良するとCG 法(conjugate gradientmethod、共役こう配法)が得られる。SD 法やCG 法は線上探索(linear search)法として統一される。そして方向ベクトルの修正方向を、前回の修正方向に共役な方向に選ぶ。こうすることによって、前回修正した方向の成分を取り除くことができる。すなわち、方向が決定したら最大修正量を満たすようにαを決定する。これは新しい修正方向が前回までのすべての修正方向に直交することを表している。したがって、計算の途中に誤差が入らなければ、必ず回の反復によって解に収束することが保証されている。さらに、収束を加速するため前処理を施した改良版がPCG 法(preconditioned conjugate gradient method)である。前処理の施し方にはいろいろあるが、不完全LU 分解を用いるのがよいとされている(図2・14)。ICCGM はincomplete cholesky CG 法の省略である。CG 系の大規模線形反復ソルバとして、その他BLCG(block CG)、BiCGSTAB(biconjugate gradient stabilizedmethod)、GPBiCG(generalized product-type BiCG)等がある。
