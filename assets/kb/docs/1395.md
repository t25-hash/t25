# 2・4・4 陰解法の緩和法

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

計算流体力学の手法を大別すると,陽解法と陰解法がある.陰解法は,CFL(Courant-Friedrich-Levy)条件による制約をあまり受けず,時間の刻み幅が大きくとれるのが非常に魅力的である.AF(approximation factorization,近似因子化)法は,多次元問題の演算子を近似的に因数分解し,一次元問題に帰着させて行列の反転を効率化する部分解法である.因子化に伴う付加誤差を小さく抑える工夫がいろいろ提案されている.AF法では,この陰解法の手法を用いて,ブロック三重対角行列を解いている.そしてAF法は,陰解法の中でも非常に効率的な手法である.しかし,因数分解による誤差がに比例して増えるので,実際にとりうるはかなり制限される.そこで,AF法を用いずに何らかの反復法によって行列方程式を解き,大きなを用いて収束を加速する手法が考察されている.これを非因子化(unfactored)された陰解法の緩和法と呼ぶ.ここではこの手法について調べることにする.a. 緩和法連立一次方程式=の解を求める.ただし,は正則で対角要素は零ではないものとする.連立一次方程式を,代表的な三つの緩和法であるヤコビ法,ガウス・ザイデル法,SOR 法で解く.. ヤコビ法行列を=++で分解する.ここで=diag,対角行列:狭義の下三角行列:狭義の上三角行列である.ヤコビ法では反復式욖=-+욖+を用いる.ヤコビ法ではベクトル욖の成分を計算するのに,ベクトル욖の成分を記憶する必要がある.. ガウス・ザイデル法ガウス・ザイデル法では,ベクトルの成分욡の新しい推定値욡욖を得たならば,その後の計算ではすべてこの新しい推定値を用いて計算効率を高めている.それには反復式+욖=-욖+を用いる..
