# 2・4・5 スペクトル半径(spectrum radius)

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

代表的な三つの反復法はヤコビ法、ガウス・ザイデル法、SOR 法である。これらの反復法は次のように表せる。ここでは反復行列である。a. スペクトル半径反復行列の絶対値が最大である固有値をスペクトル半径と呼ぶ。そして、連立一次方程式が厳密解に収束する条件は0<スペクトル半径<1、収束条件で与えられる。各手法のスペクトル半径ρ:ヤコビ法のスペクトル半径ρ:ガウス・ザイデル法のスペクトル半径ρ:SOR 法のスペクトル半径で定義する。連立一次方程式の係数行列が、ヤング(Young)6―16計算力学の性質(係数行列が指数2の巡回行列であり、かつ整合順序であること)を満足する場合、各手法のスペクトル半径の間にある関係式が成立する。よってヤコビ法のスペクトル半径が求まれば、ガウス・ザイデル法およびSOR 法のスペクトル半径も求めることができる。特にSOR 法の最適加速係数が、ヤコビ法のスペクトル半径より求まることは注目に値する。b. 最適加速係数(coefficient of optimal acceleration)の推定法不等間隔格子や任意形状要素に対して、上の公式は適用できない。推定手順は以下の3ステップより成る。ここでノルムは、最大値ノルムまたはユークリッドノルムである。
