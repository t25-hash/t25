# 2・4・6 非線形方程式の解法

> **章**: 第2章 数値計算法の基礎
> **節**: 2・4 連立一次方程式の解法

定常ナビエ・ストークスの方程式は非線形方程式である。ここでは非線形方程式の高速解法について調べる。その基礎となるのはニュートン・ラプソン法(Newton-Raphson method)である。ニュートン・ラプソン法は、ゼイド(Zeid)によって固定点反復法に発展した。この二つの方法は、非線形方程式の解法として強力である。a. ニュートン・ラプソン法。この非線形方程式の根を求めるのに、ニュートン・ラプソン法が有効である。ニュートン・ラプソン法は、一次元のニュートン法を多変数へ拡張したものである。この方法を修正ニュートン・ラプソン法という。修正ニュートン・ラプソン法は、1回当たりの収束は遅くなるが計算量が少なくなるため、全体としての計算スピードが向上する可能性がある。. 離散化ニュートン・ラプソン法(discrete Newton-Raph-son method)実際、ヤコビアンの計算は複雑であるので、偏微分を差分商で近似することが考えられる。この方法を離散化ニュートン・ラプソン法という。特に偏微分が実行できないような場合に有効である。b. 固定点反復法固定点反復法(FPI法: fixed point itera-tion method)は、ゼイドによって開発された非線形方程式の解法である(1985年)。その基礎はニュートン・ラプソン法にある。この方法をニュートン・ラプソン法と呼ぶ。図2・16に反復過程が示されている。(a) 変数ヤコビアン(b) 固定ヤコビアン図2・15 修正ニュートン・ラプソン法の原理図図2・16 ニュートン・ラプソン法。この方法を固定点反復法と呼ぶ。ここでαを収束加速因子と呼ぶ。ここでは収束加速行列である。しかし、逆行列を求めなければならない。これを避けるため、としてを対角化した近似行列を用いる。(4) ガウス・ザイデル法の導入と要素レベルでの実行ガウス・ザイデル法を導入して、収束が加速すると同時に、要素レベルでこの計算を実行する。要素レベルで実行するとき、次の三つの利点がある。・全体係数行列を作るための計算過程の省略と同時に、そのCPUタイムが節約できる。・プログラミングおよびコーディングが簡単化される。ニュートン・ラプソン法によりこの非線形方程式を解く。この方程式をマトリックスフリー法に基づくGMRES 法(generalized minimal residual method)で解を求める。このアルゴリズムは図2・18に示すとおりである。計算の反復部分として、内側ループ(inner loop、計算の繰返し部分として、適当な反復回数を繰返す部分)と外側ループ(outer loop、それまでに得られた近似解を、初期近似解として再スタートさせる部分)がある。マトリックスフリー法では、反復法内部における要素ごとの行列とベクトルの積の重ね合わせを行う。そして、このループ内で要素ごとの行列を作成する。これにより、要素ごとの行列を記憶する必要がなくなり、より大規模な問題への適用が可能となる。一般に流れは動的に変化するために、非定常な解析が必要となる。そのためには高精度かつ安定な時間進行法(time marchingmethod)を採用しなければならない。前進オイラー法は高速解法であるが不安定であり、後退オイラー法は陰解法で安定である。これらの手法は時間に関して一次精度である。一般に、二次精度の時間進行として台形公式が用いられる。さらに高精度計算法として、ルンゲ・クッタ(Runge-Kutta)法がある。本節では時間進行法についてまとめておく。
