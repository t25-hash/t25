# 3・1・4 構成式と基礎方程式

> **章**: 第3章 応力解析の基礎
> **節**: 3・1 応力とひずみ

連続体の運動を記述するためには、与えられた境界条件(boundary condition)のもとで、式(3・42)に現れる未知の速度(または変位)と、応力を確定する必要がある。一方、式(3・42)は3個の関係を与えるだけなので、これだけでは未知量を決定できない。式の数を完結するためには、連続体を構成する物質固有の関係(構成式または構成方程式、constitutive equation)が定義されなければならない。式(3・25)によって、ひずみと変位の関係が6個与えられているので、変位(3個)、応力(6個)、ひずみ(6個)の未知数について、変位-ひずみ関係式(3・25)(6個)、運動方程式(3・42)(3個)、および構成式(3・44)(6個)によって、式の数と未知数の数は完結する。変形は微小なので、質量保存則は満足すると考える。さらに、弾塑性体のように非線形な応答を示す材料では、応力速度(増分)とひずみ速度(増分)の関係が与えられ、それらを時間(経路)に従って積分する。ここでは、弾性コンプライアンス(elastic compli-ance)と呼ばれる4階のテンソルである。ここでαは、線膨張係数(thermal expansioncoefficient)と呼ばれる材料定数である。この逆関係が応力-ひずみ関係を与える。熱伝導方程式(1個)と連立させて、未知の変位(3個)、応力(6個)、ひずみ(6個)および温度(1個)を得る。非粘性流体(完全流体、perfect fluid)での応力状態は、式(3・7)で与えられる。上式と連続の式(3・36)で、速度(3個)と応力(圧力、1個)について、式の数と未知数の数は完結する。ここでμとは、材料の粘性(viscosity)を表す物質固有の値である。このとき、式(3・25)に代わり式(3・29)が運動学的関係として用いられる。
