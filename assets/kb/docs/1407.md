# 3・2・1 静的応力解析の境界値問題

> **章**: 第3章 応力解析の基礎
> **節**: 3・2 エネルギー原理

固体の微小変形に対する応力解析を行う場合,解くべき支配方程式は,物体が占める領域において,応力の平衡方程式옿σ욡욢옿욢+ρ욡=0(3・51)変位-ひずみ関係式ε욡욢=1옿욡옿욢+옿욢옿욡(3・52)および物体の変形と応力を関係づける構成方程式(構成式)で与えられる.ここで욡は既知の物体力とする.多くの場合,境界条件として,物体表面の一部(表面力規定境界)において応力ベクトルがσ욡욢욢=욡上(3・53)と規定され(욡:規定表面力,욢:外向き法線ベクトル),残りの表面욊(変位規定境界)において変位ベクトルが욡=욡욊上(3・54)と規定される(욡:規定変位).接触問題やき裂問題などにおいて,境界の一部で変位,応力のある成分が合わせて規定される場合があるが,ここでは領域の境界がと욊に分けられる場合を対象とする.多くの応用において,以上の関係式および境界条件を直接解くよりも,以下に述べる種々の変分原理(variational principle)が数値解析の基礎原理として用いられる.変分原理によって,強形式の定式化である偏微分方程式の境界値問題と等価な,離散的数値解析法により適した弱形式の定式化を実現できることは,特に重要な性質である.変分原理により,境界値問題の多くは,ある汎関数(functional)の極値問題に帰着される場合が多い.特に固体の変形を扱う場合,問題となる汎関数にエネルギーという物理的意味を対応させることができるので,固体力学の変分原理はエネルギー原理(energy principle)とも呼ばれる.
