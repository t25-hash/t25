# 3・2・1 静的応力解析の境界値問題

> **章**: 第3章 応力解析の基礎
> **節**: 3・2 エネルギー原理

多くの応用において、以上の関係式および境界条件を直接解くよりも、以下に述べる種々の変分原理(variational principle)が数値解析の基礎原理として用いられる。変分原理によって、強形式の定式化である偏微分方程式の境界値問題と等価な、離散的数値解析法により適した弱形式の定式化を実現できることは、特に重要な性質である。変分原理により、境界値問題の多くは、ある汎関数(functional)の極値問題に帰着される場合が多い。特に固体の変形を扱う場合、問題となる汎関数にエネルギーという物理的意味を対応させることができるので、固体力学の変分原理はエネルギー原理(energy principle)とも呼ばれる。
