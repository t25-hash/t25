# 3・2・3 補足仮想仕事の原理

> **章**: 第3章 応力解析の基礎
> **節**: 3・2 エネルギー原理

与えられた外力と変位拘束のもとで平衡状態にある物体に対して、平衡方程式(3・51)を満たし上で零となる応力場δσ욡욢(仮想応力)を、静力学的に可容(statically admissible)な応力と呼ぶ。この仮想応力に対して、式(3・52)(3・54)よりε욡욢-1옿욡옿욢+옿욢옿욡δσ욡욢d+욡-욡δσ욡욢욢d=0(3・57)となる。上式にガウスの発散定理を用いると、ε욡욢δσ욡욢d=욡δσ욡욢욢d(3・58)を得る。これは、仮想応力によってなされる補足内部仕事と補足外部仕事が等しいことを表しており、補足仮想仕事の原理(princi-ple of complementary virtual work)と呼ばれる。補足仮想仕事の原理から、逆に変位-ひずみ関係式(3・52)および変位境界条件(3・54)が得られる。3・2・2項の仮想仕事の原理と同様に、補足仮想仕事の原理も特定の構成関係によらず成立する。
