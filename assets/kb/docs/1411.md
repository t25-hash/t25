# 3・2・5 その他の境界値問題に対するエネルギー原理

> **章**: 第3章 応力解析の基礎
> **節**: 3・2 エネルギー原理

以上が微小変形下の静的応力解析の境界値問題に関する変分原理の概要であるが、その他の境界値問題であっても、それに対応した変分原理を導くことができる。その代表的なものについて、以下に挙げておく。剛塑性体の解析では、構成式が応力とひずみ速度の関係として与えられるため、変位、ひずみの代わりに速度、ひずみ速度に基づいた境界値問題が扱われる。この場合には、速度、ひずみ速度に基づく仮想仕事の原理が同様の方法で導かれる。また、応力速度とひずみ速度の関係で記述される弾塑性体では、さらに応力を応力速度に置換えた境界値問題に対して、対応する仮想仕事の原理が成り立つ。これらの非弾性体に対しても、構成式があるポテンシャルから導かれる場合には、3・2・4項で述べた種々の停留原理、最小原理の類似形が導出できる。この場合、ラグランジュ乗数を用いて変分原理に導入するか、または非圧縮性に近い構成関係を仮定して(体積変化に対して十分に高い剛性を仮定して)解析が行われる。さらに、温度場と変形場が連成する熱弾性体、電気場と変形場が連成する圧電体などに関しても、境界値問題から対応する変分原理が導出できる。また、有限変形に関する変分原理の定式化には、用いる応力、ひずみの尺度について適切な選択が必要となる。時間に依存しない塑性変形は、弾性と塑性変形の応力境界曲面である降伏曲面を用いて区別されている。金属の試験片を引張荷重で塑性変形させると、最初の降伏応力から次第に硬化して、より高い応力で塑性変形が継続する。これは降伏曲面が単純に膨張したと考え、これを等方硬化(isotropic hardening)と呼んでいる。金属試験片に加えていた荷重を、その後反転させ、除荷してさらに圧縮荷重を増していくと、引張荷重で硬化したと予測される値より低い応力で再降伏する、いわゆるバウシンガー(Baushinger)効果が見られる。この現象を、降伏曲面が形を変えずに剛体移動したと考えて説明した理論が、移動硬化(kinematic hardening)である。また、降伏曲面の膨張と移動を同時に考慮したものを、複合硬化理論(combined hardening rule)と呼んでいる。単軸だけでなく、構造解析においては多軸応力場における塑性変形を表すことが必要になるが、ミーゼス(von Mises)の降伏条件、関連流れ則、プラガー(Prager)の適合条件を用いて、塑性理論は構築されている。
