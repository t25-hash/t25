# 3・3・5 プラガーの線形移動硬化則

> **章**: 第3章 応力解析の基礎
> **節**: 3・3 材料非線形性

降伏曲面はその大きさを変えず、図3・4(b)に示すように、剛体移動すると考える。偏差応力空間における降伏曲面の中心は背応力(back stress)と呼ばれ、で表す。この場合の後続曲面の降伏条件は、式(3・91)の代わりに次式を用いる。このため、シャボッシュ(Chabo-che)によって提案された背応力の非線形硬化則は、次の形を仮定するものである。また、等方硬化則についても、次の非線形硬化型を仮定し、図3・4(c)に示すような複合硬化則を得ている。このような近似の仕方を非線形硬化理論と呼んでおり、これ以外の近年の塑性構成式と関連がある。
