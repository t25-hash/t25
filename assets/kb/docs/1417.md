# 3・3・6 有限変形時の塑性構成式

> **章**: 第3章 応力解析の基礎
> **節**: 3・3 材料非線形性

有限変形・有限ひずみ時の塑性構成式は,客観性の原理を考慮したものを用いる必要がある.座標軸の回転に対して不変の要請を満足するものとして,通常,応力速度にはヤウマンの応力速度(Jaumann’s stress rate),ひずみ速度にはストレッチングテンソル욡욢が用いられる.等方硬化則を例に用いると,次のようになる.σ°욡욢=2μ욡욢+λδ욡욢욅욅-2μ욡욢욡욢욡욢(3・112)ここで,σ°욡욢=σ욡욢-욡욅σ욅욢+σ욡욅욅욢(3・113)ここで,速度こう配テンソル욡욢=옿옿욢から,ストレッチングテンソル욡욢とスピンテンソル욡욢は得られる.応力はコーシーの真応力が用いられている.
