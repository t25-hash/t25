# 3・4・1 幾何学的非線形性(geometrical nonlinearity)

> **章**: 第3章 応力解析の基礎
> **節**: 3・4 幾何学的非線形性

固体の応力解析の基礎式は、1応力成分の釣合い方程式、2応力-ひずみ関係式、3ひずみ-変位関係式である。これらの基礎式から成る偏微分方程式系(あるいは対応する変分原理)を与えられた4力学的境界条件、5幾何学的境界条件のもとで解くことにより、固体の変形および応力分布などが計算される。このとき、これらの基礎式あるいは境界条件の非線形性に起因して、さまざまな非線形問題が発生する。例えば、相当応力が降伏点を超えて塑性変形を生じると、応力-ひずみ関係式は非線形となり、このことに起因する非線形問題は、材料非線形問題(materially nonlinear problem)と呼ばれている。また、大たわみ問題(より一般的には有限変形問題)、あるいは座屈問題(より一般的には構造安定問題)のように、ひずみ-変位関係式の非線形性に起因して発生する非線形問題を、幾何学的非線形問題(geometrically nonlinear problem)と呼んでいる。さらに、接触問題(contact problem)などのように、境界条件が変形に依存して変化することに起因する非線形問題もある。本節では、幾何学的非線形問題の基礎理論(3・4・2項)と変分原理(3・4・3項)、幾何学的非線形問題の分類(3・4・4,3・4・5項)および具体例(3・4・6項)について述べ、最後に接触問題(3・4・7項)について簡単に説明する。
