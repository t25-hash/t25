# 4・1・1 有限要素法の位置付け

> **章**: 第4章 構造解析のための有限要素法
> **節**: 4・1 弾性問題の定式化と解法

通常私たちは、固体を原子や分子あるいは結晶の集合体として意識することはほとんどない。むしろ空間上に連続的に広がる物体として、その現象論的・平均的挙動に着目して、マクロスコピック(巨視的)に理解するほうが自然である。このようなとらえ方を連続体力学(continuum mechanics)と呼ぶ。連続体力学においては、固体の変形現象は、変位やひずみ、応力という場所と時間の関数に対して成立する偏微分方程式と、境界条件および初期条件によって記述される。偏微分方程式のコンピュータ近似解法としては、差分法(FDM : finite difference method)、有限要素法(FEM : finite element method)、境界要素法(BEM : boundaryelement method)が代表的である。差分法においては、解析空間および時間方向を有限な寸法の格子に分割し、格子点で定義される変数を用いて、これらの式中に現れる微分演算を直接差分で近似する。これに対して、有限要素法や境界要素法では、これらの式に等価な積分方程式を導出し、解析領域内あるいは領域境界面を、小さな多面体あるいは多角形に分割し、その節点で定義される変数を用いて積分方程式を近似的に評価する。これらいずれの方法にも共通するのは、空間を多数の小領域に分割する(離散化、discretization)という過程である。一方、それぞれに次のような得失もある。差分法は座標系に沿う方向に分割するため、任意の形状の問題において境界条件の扱いが難しい。そのため、座標変換によって直交格子を境界形状に合わせるなどの工夫が必要となる。これに対して有限要素法と境界要素法は、任意の形状とそこに付与される境界条件の取扱いが容易である。さらに有限要素法は、非線形問題や時間依存型問題の取扱いにも優れる。このため、有限要素法は固体力学におけるマクロスコピック数値解析法の主流となっている。有限要素法が初めて工学上の問題に用いられたのは、1950年代のアメリカの航空機設計技術者たちによってである。その後、変分法の近似解法、すなわちリッツ(Ritz)法の考え方と結合されるに至って、有限要素法の固体力学・構造力学への応用が大きな飛躍を遂げた。1960年代の後半からはさらに、重み付き残差法を基本とする近似解法、すなわちガラーキン(Galerkin)法を代表とする近似解法と結合されたことにより、その適用範囲は大きな広がりを見せ、電磁気学のマクスウェル(Maxwell)方程式、汎関数(functional)が存在しない流体力学の代表的な支配方程式である、ナビエ・ストークス(Navier-Stokes)方程式へも適用されるようになった。本節では、有限要素法による線形弾性応力解析の定式化と解法の基礎について述べる。
