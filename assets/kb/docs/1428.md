# 4・1・3 変分原理に基づく定式化

> **章**: 第4章 構造解析のための有限要素法
> **節**: 4・1 弾性問題の定式化と解法

原理的には、4・1・2項に示された微分形の支配方程式を、差分法などを用いて直接解くことも可能であるが、有限要素法では本項で用いる変分原理や4・1・4項で述べる重み付き残差法を用いて数学的に等価な積分形の式を導出し、それを解く。ここでは、固体力学において用いられる変分原理について、その要点をまとめる。応力-ひずみ関係が式(4・8)で与えられるものと仮定すると、ひずみエネルギー密度は、次式で定義される。ただし、構造物は静止しているものとする。この状態から、任意の微小な仮想変位δをこの構造物に与えることを考える。ガウスの発散定理と式(4・11)(4・12)を上手に使って、式(4・20)を変形すると、次式を得る。この式が仮想仕事の原理(principle of virtual work)と呼ばれる式であり、応力-ひずみ関係式とは無関係、すなわち弾性問題に対しても弾塑性問題などに対しても成立する。このような関数を用いて、関数の関数、すなわち汎関数Π を次のように定義する。このような変位の任意な許容関数に対して、式(4・24)よりΠ はスカラ量として計算される。式(4・24)の右辺の第1,2,3項はそれぞれ、ひずみエネルギー、物体力および表面力のポテンシャルエネルギーを表している。このように定義されるΠ に関して、が正解のときΠ は最小となること、さらにその解は式(4・23)の解と一致することが数学的に証明される。この原理を最小ポテンシャルエネルギーの原理(principle of minimumpotential energy)と呼ぶ。なお、定義から明らかなように、最小ポテンシャルエネルギーの原理は弾性体についてしか成立しない。
