# 11・3・4 ターボ機械の広帯域騒音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・3 回転音源による騒音

騒音のスペクトル分布において，そのエネルギーが広い周波数帯ように流れの乱れの成分によっているので，広帯域騒音は乱流騒翼面上の境界層内の乱れ（渦）b.離散周波数騒音のダクト内減衰 ファンやコンプレッサから発生する離散周波数騒音の音圧レベル（sound pressure level）の圧力変動が直接音として放射される成分は，一般に無視できるとで翼と干渉して放射される成分は大きい．
これは後縁騒音（trai-ように，主流が翼の凸面側に流入する条件下においては，凸面側のではく離し，凹面側の翼後縁から出発した自由せん断層と対を成しaeolian tone）を発生する．
しかし，翼面に流入する主流速度がハと増加する．
したがって，測定される騒音のスペクトルはこれらの帯域分布を示す．
このカルマン渦による騒音発生が過大流量域での分布の半理論的予測もされている．
が，騒音発生の物理的機構は同じである．
すなわち，いずれも翼にすることによって騒音が発生する．
したがって，前者は乱流騒音となる．
11･3･3･b項で述べたように，流入する流れが偏流していたその例である．
これらのいずれの場合も流れと翼との干渉によって域になだらかに分布するものを広帯域騒音（broadband noise）という．
ブから翼先端に向けて増加するため，その周波数はハブから先端へ積分値であるため，各ピークの包絡線であるなだらかないわゆる広騒音の増加の原因となる．
このような場合の発生騒音のスペクトル主流の迎え角が時間的に大きく変化する場合の二つが考えられる対する迎え角が変化するために，物体まわりの循環が時間的に変化なり，後者は時間的変化が周期的である場合は，離散周波数騒音とり，動翼の上流の物体の後流が流入して発生した離散周波数騒音が音が生じるので，双極子型の音源である．
