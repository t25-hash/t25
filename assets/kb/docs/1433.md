# 11・5・1 カルマン渦励起音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・5 自励音

流体中の二次元物体の後流にカルマン渦列が形成され，このカル
マン渦列の放出に伴って離散周波数音が励起される．カルマン渦列
の発生自体が流体の自励振動現象である．物体から流れ出した渦が
ある距離流れたときに物体周辺に誘起する圧力場は次の渦の流れ出
しに影響を与えているので，周期的に渦が放出されると推論でき
る．渦を周期的に放出することで物体表面に周期的な圧力変動が誘
起されて，物体表面が双極子音源になって音が放射される．なお，
カルマン渦列の発生は渦と物体の干渉によって発生する自励振動現
象である．

も，カルマン渦の柱状体軸方向の相関長さ（correlation length）
を小さくすると放射音が低減する．図11･10に示したように1本の
円柱の直下流に追加の円柱を交差設置して相関長さを短くをする
と，エオルス音が低減する ．円柱よりも角柱のほうが渦放出
音の卓越性が強い現象は，角柱の相関長さのほうがより長いことに
よる．柱状体表面にトリッピングワイヤを巻付けたり，微小突起を
取付けると相関長さが短くなる．表面を一様に粗面にしても相関長
さはあまり変化せず，見掛け状の代表径が太くなることで周波数が
低くなる．一方，柱状体を流れ方向に傾斜させると相関長さが長く
なるが，横切る流速が低下するので発生音周波数が低くなる．円柱
を下流方向へ凸状となるように曲げると，相関長さが極端に短くな
ってエオルス音が消え，はく離騒音だけになる ．渦放出自体を
制御するためには能動的なアクチュエータを用いる ことが必要
であるが，相関長さを制御するためには多様な手法が考案可能であ
る．
