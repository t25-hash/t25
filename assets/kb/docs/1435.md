# 11・5・3 超音速流の自励音

> **章**: 第11章 空力音響および水中音響
> **節**: 11・5 自励音

超音速噴流は擾乱を発生させるような固体物体に衝突しなくて
も，図11･14(ｈ)に示すように衝撃波セルの振動によって自励音が
発生する ．これをスクリーチトーン（screech tone）と呼び，自
励ループは次のような機構である．第三衝撃波セル内を移動する斜
め衝撃波がセル下流の垂直衝撃波にぶつかると音波が発生し，音波
は噴流の外側を音速で伝わって噴出口のせん断流における渦放出に
刺激を与える．噴出口から放出された渦はセル内を移流する斜め衝
撃波を誘起することで閉ループが構成される ．スクリーチトー
ンの周波数は次のようになる．
＝2π1＋

流の場合，衝撃波セル内を移流する斜め衝撃波はらせん状に回転し
て音波を発生する ．
噴流のせん断流が物体に衝突して自励音を生じる現象は超音速噴
流でも存在し，ハルトマン笛 だけではなく，ホールトーン や
平板衝突噴流音 ［図11･14(ｉ)］も確認されている．また噴流で
はなくて物体まわりの膜状超音速流において衝撃波セルが構成され
るとスクリーチトーンが自励する ．一方，超音速噴流の自励音
を低減するためには，噴出口に微小突起（tab）を装着すると効果
的 である．
