# 11・6・3 水中超音波

> **章**: 第11章 空力音響および水中音響
> **節**: 11・6 水中音響

音圧は周波数に比例するので，周波数の高い超音波では，音圧が
大きい．このため水中ではキャビテーションが容易に起こり，発生
した気泡はつぶれるとき，衝撃波を伴った大きな音を発生する．こ
れが壁面近くで起こると壁面を腐食させる．また，発熱や発光現象
である音響ルミネッセンスが生じる．
音波の減衰の大きさは周波数の二乗に比例し，空気中における減
衰は水中のそれに比べて1000倍程度大きい．このため，超音波の
空気中での使用は少なく，液体または固体中での応用がほとんどで
ある．なお，水中に気泡などが存在すると超音波の減衰はさらに大
超音波は波長が短いため，鋭い指向性をもつので幾何光学的扱い
ができることなどの特徴をもつ．このような性質を利用し，音速の
異なる材質で作った音響レンズを用いて超音波を屈折させて集音
し，焦点に強力な音場を作ることができる．このことを超音波加工
や胆石の破壊などに用いている．また，反射を利用してソナーとし
て用い，魚群探知機や測深機に応用している．
