# 4・3・1 要素分割と誤差解析

> **章**: 第4章 構造解析のための有限要素法
> **節**: 4・3 各種問題への応用

a.誤差評価の考え方 有限要素解析で得られる解は，あくま
ように要素形状が著しくゆがんでいる場合にも誤差の要因となる．
許容される自由度の中で，可能な限り誤差を小さくするために
は，誤差を適切に評価することが求められる．誤差の分布が表現で
きれば，誤差の大きな部分について要素を改善することにより，解
析精度を向上することができる．一般に誤差という用語は，厳密解
と解析結果との差異と解釈されるが，有限要素解析では，理論解が
求まっているような特殊な場合を除いては，厳密解が与えられない
のが通常である．したがって，解析で得られる情報だけから，適切
な誤差を表現する量を定義して評価する必要がある．このような手
続きを事後誤差評価と呼ぶことがある．この場合の誤差とは，厳密
解と解析で得られる解との差異という意味での誤差ではないことに
注意を要する．
b.誤差解析 事後解析から評価される誤差指標として，幅広
く利用されているツィエンキヴィッツ（Zienkiewicz）とツー
（Zhu）の提案した簡易誤差評価手法 について紹介する．この指
標は簡単に評価できて，しかも適合再分割の適用に対して良好な結
果が得られることが知られており，幅広い問題に利用されてい
る．また，有限要素解析の過程で使用される マトリック
スや マトリックス，および解析結果とから評価できるので，
既存の有限要素解析システムに大幅に手を加えることなく組込むこ
とができるのも，大きな利点の一つである．
