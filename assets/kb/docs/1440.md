# 4・3・3 高温構造解析

> **章**: 第4章 構造解析のための有限要素法
> **節**: 4・3 各種問題への応用

a. はじめにエネルギー機器など、定常運転時に高温で使用される機器では、起動・停止などの過渡運転時に材料がさらされる雰囲気温度が時間的に変化するが、熱伝導の遅れによって温度の空間的な分布が生じることが多い。これに伴う熱膨張ひずみの非均一分布によって熱応力が発生し、構造健全性の確保のためには、その影響の的確な評価が不可欠である。さらに、運転温度が材料のクリープ変形域に入った場合には、クリープ変形やこれに伴う損傷を考慮して設計や寿命評価を行う必要がある。これらのためには、構造物中に発生する応力やひずみの分布を、精度良く推定することが必要である。6―43これまでの構造設計では、線形弾性体を仮定した応力解析の結果を基に、非弾性変形の影響を簡易的に考慮して応力やひずみの評価を行う、簡易非弾性解析的な手法がおもに用いられてきた。しかし、これらは特定の材料特性や構造形状の仮定のうえに導かれたものが多く、個々の材料や構造に対する適用性が不明な場合が少なくない。一方、信頼できる数値解析法を用いて、構成材料の非弾性変形特性をそのまま考慮した非弾性解析を行えば、人間の判断に依存することなく、自動的に構造の非弾性応答が評価できるため、裕度のばらつきがより小さく、高い信頼性をもった設計が実現できると予想される。このようなことから、材料非線形性を考慮した有限要素解析が適用されることが多くなってきている。本項では、有限要素法を用いて、これらの評価を実施する手法の概要とその適用例を示す。b. 非弾性構成式(inelastic constitutive equation)応力と非弾性ひずみの関係を与える非弾性構成式は、非弾性変形が生じるのを無視できない高温機器に対する非弾性解析を行ううえで不可欠であるが、応力、ひずみなど健全性を評価するための量に対して大きな影響をもつため、使用される材料の挙動をできるだけ忠実に表現できるものを用いることが必要である。非弾性ひずみは便宜的に、時間依存性を考慮しない塑性ひずみと、時間とともに変化するクリープひずみの2種に分けてモデル化されることが多いが、非弾性変形は基本的に時間に依存するため、これらを統一的に表現する統一型粘塑性構成式の開発も進められている。以下、これらの開発現状について概説する。. 塑性構成式(plastic constitutive equation)塑性構成式については、高温機器に対する主要な負荷が熱応力の繰返しであることが多いため、繰返し負荷下の変形挙動に対して十分な表現精度をもつことが必須である。等方性をもった多結晶金属材料については、応力の第2不変量を用いたミーゼス(Mises)の初期降伏条件が仮定されるのがふつうであるが、塑性変形とともに、降伏条件(yield condition)の変化が生じることを考慮することが必要である。ミーゼスの降伏条件は、偏差応力空間上の超球として与えられるが、その半径のみが変化すると考える等方硬化法則や、塑性ひずみに比例して中心の位置のみが変化すると仮定する線形移動硬化法則などのような古典的な塑性構成法則では、繰返し負荷下での挙動に対して十分な表現精度を得ることが困難である。このため、より高度な硬化法則(hardening rule)の開発、提案が多数行われ、近年これらを取り入れた非弾性構成式の実用化が進められている。繰返し負荷下での変形の特徴として、1サイクル内の応力-ひずみ関係の曲線性と、繰返しに伴うその変化が挙げられる。前者に対しては、塑性変形とともに硬化率が変化するモデル化が必要である。少ないパラメータで最も簡便的にこれを表現する手法として、アームストロング・フレデリック(Armstrong-Frederick)の硬化則が挙げられる。さらに、降伏曲面を包含する曲面(限界曲面)を仮定して、硬化速度を規定する2曲面モデルも考案されている。また、多数の曲面を用いて段階的な硬化率の変化を表現する多曲面モデルや、異なった降伏応力をもった複数の弾完全塑性体を並列で結合させた、サブレイヤモデルなども提案されている。そして、異なった硬化速度と飽和量をもった移動硬化成分の重ね合わせによって、非線形硬化特性を表す方法も提案されており、ラチェット変形に対して、多くの他のモデルよりも良好な予測を与えるため、高速増殖炉機器に対する評価指針案などで採用されている。一方、負荷サイクルに伴う変形特性の変化には、大きく分けて変形抵抗が増加する場合と減少する場合があり、それぞれ繰返し硬化、繰返し軟化と呼ばれる。これらによってひずみ分布や定常運転状態の応力レベルが変動し、疲労損傷やクリープ損傷に影響を与えるので、クリープ疲労損傷の評価では特に重要である。これらの挙動は繰返しひずみ範囲に依存するため、精度良く表現するためには、塑性ひずみの変動を代表する状態変数を用いる必要がある。6―44計算力学(a) hardening rule)。ただし、この方法は硬化を累積クリープひずみのみの関数として、等方硬化的に取扱うことを意味しており、負荷応力が反転した場合などでは、クリープ変形でも移動硬化的な挙動が観察されるため、良好な予測を与えない。これに対して、修正ひずみ硬化理論や回復可能な硬化領域モデルなどの解決策が考案されている。さらに、塑性変形によって、クリープ変形特性が変化したり、クリープ変形によって、塑性変形特性が変化することもよく知られており、それぞれの構成式で硬化を表す変数に対して、これらを考慮するための補正項を加えることが重要となる場合も少なくない。逆方向の塑性変形によって、クリープ変形に対する硬化が回復することの考慮は、クリープ疲労状態でのクリープ損傷評価において大きな影響をもつため、特に重要である。. 統一型構成式(unified constitutive equation)統一型構成式は、非弾性ひずみを塑性とクリープに分離せず、統一的に取扱う方法であり、本来的にはその整備が望まれる。これに関してもこれまで多くの研究が行われ、さまざまなモデルが提案されているが、なかでもシャボッシュ(Chaboche)モデルは高速炉などの構造解析に適用されている。なお、弾塑性構成式やクリープ構成式に比較して、定数の決定が難しいのが難点である。また、非弾性変形が小さい条件で片側だけの保持を受けた場合、ヒステリシスループが保持を与えたのと逆方向に移動する現象が、多くの材料で観察される。これによって生じる平均応力はき裂進展速度に影響を及ぼすため、その考慮は重要であるが、通常の粘塑性モデルでは、この現象を的確に表現することが困難であるため、移動硬化則の修正も提案されている。修正移動硬化則のガスタービン用超合金に対する適用例を図4・20に示す。. 弾塑性解析応力とひずみが1対1の関係をもつ線形弾性体と異なり、非弾性変形は負荷履歴によって異なり、構成式も微分形で表現されるのがふつうである。このため、評価する必要があるのが最終的な状態だけであっても、それに至る過程を増分的に解く必要がある。増分は計算精度の点からは細かいほうが望ましいが、計算時間の点からはある程度大きくとるのが望ましく、両者のバランスをとる必要がある。増分形解析では、各点での応力増分とひずみ増分の関係を満足させながら、各点での適合条件式(変位-ひずみ関係)と平衡方程式(応力の釣合い条件)を満足させる解を求めることが必要である。線形弾性ひずみと塑性ひずみの両者を考慮する弾塑性解析では、時間は陽には含まれてこないので、増分は実時間とは無関係に、温度など負荷の変化の程度に応じて決定することができる。変位型有限要素法では、各積分点でひずみ増分から応力増分を求める必要がある。このときに、材料固有の構成式が使用されるが、非弾性構成式は基本的に微分形で表されているため、有限なひずみ増分に対してこれをそのまま前進差分化して用いると、降伏条件からの逸脱のような不具合が生じる。これを避けるため、さまざまな方法が考案されているが、なかでも応力を降伏曲面の中心に向けて縮退させる法線引戻し法(radial return method、ラジアルリターン法)は、簡単でありながら有効性の高いアルゴリズムである。法線引戻し法は、最初は弾完全塑性体に対して提案されたが、近年、非線形な移動硬化則を含む構成式に対しても適用法が開発され、その有効性が確認されている。次に応力増分を用いて更新された応力を、要素内で積分することによって得られる各節点での節点力の釣合いによって、収束性が判定される。このときの不釣合い量(残差節点力)が十分小さければ収束とみなされるが、そうでない場合にはこれを小さくするために、残差節点力を荷重項とした変位増分評価が必要となる。このためには、剛性方程式を再度解く必要があるが、剛性行列作成時に用いる材料の剛性によって、収束性が大きく変化する。弾性関係を用いると、多数の繰返しが必要となるのが一般的であり、弾塑性関係式を用いることが有効である。単純な微分関係を用いる方法は接線剛性法と呼ばれ、古くから用いられているが、単純な微分関係を用いるよりも、応力積分法と整合した関係を用いるほうが収束性が向上する。特に法線引戻し法に対応したコンシスタント接線剛性行列(consistent
