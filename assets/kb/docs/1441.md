# 11・1・3 弾性体と熱の連成

> **章**: 第11章 連成問題
> **節**: 11・1 熱

a.線形熱弾性 温度変化を伴う線形弾性体の変形・応力解析
において考慮すべき方程式は，
･ひずみ-変位関係［式(11･3)］
･運動量保存則［式(11･6)］
･エネルギー保存則［式(11･7)］
･構成式［式(11･8)］
および関連する境界条件である．
式(11･8)は以下のように表すことができる．
σ＝λδε－ε＋2με－ε (11･20)
ε＝α－δ (11･21)
ここで，εは熱ひずみである．式(11･20)は応力σと，全ひずみ
から熱ひずみを差引いたひずみ成分( ε－ε)とが，熱の影響がな
い場合の応力-ひずみ関係に従うことを示している．
したがって，温度変化を伴う線形弾性解析においては，まず式
(11･7)のエネルギー保存則（熱伝導方程式）から弾性体の温度分布
を求め，式(11･21)から熱ひずみを計算する．次に，応力解析を
実施する際には，熱の影響がない場合の応力-ひずみ関係式を用い
る．ただし，熱ひずみに相当する見掛けの物体力を， と同様の
扱いで既知の外力として考慮する．有限要素法における解析手順で
いえば，熱が関係しない通常の弾性解析との違いは，熱ひずみによ
る見掛けの物体力ベクトルが剛性方程式の右辺ベクトルとして，新
たに追加されることである．
b.非弾性問題 高温下においては，熱との連成の結果とし
て，弾塑性，材料物性の温度依存性・ひずみ速度依存性，クリープ
などの非弾性問題が表面化しやすくなる．こうした非弾性問題で
は，荷重もしくは時間を微小な増分ステップに分割し，各ステップ
内では増分形で表された変数について，線形化した支配方程式を扱
うことが多い（増分法）．例えば，各ステップにおいて構成式は増
分表示で次のように表される．
σ＝ε－ε－ε－ε－ε (11･22)
ここで，σ， はそれぞれ，応力増分，弾性応力-ひずみテンソ
ルである．また，ε，ε ，ε ，ε ，εはそれぞれ，全ひ
ずみ，温度依存性によるひずみ増分，塑性ひずみ増分，クリープひ
ずみ増分，熱ひずみ増分である．式(11･22)は応力増分σと，全
ひずみ増分から各種ひずみ増分を差引いた残りのひずみ増分成分
が，熱の影響がない場合の線形の応力-ひずみ関係に従うことを示
している．簡単のために，式(11･22)において，塑性および物性値
の温度依存性がない場合を考える．このとき，
σ＝ε－ε－ε (11･23)
各増分ステップにおいて応力解析を実施する際には，線形熱弾性解
析のときと同様，熱伝導解析によって求められた温度分布を基に，
熱ひずみ増分［式(11･21)］やクリープひずみ増分に相当する見掛
けの物体力を求め，既知の外力として右辺側に考慮する．なお，ク
リープひずみ増分の決定には，硬化則やクリープ流れ則が用いられ
る．
