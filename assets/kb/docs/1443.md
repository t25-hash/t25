工学的問題に頻出する最適化問題は次式の形式をとる。この目的関数と制約条件が機械・構造物の有限要素解析により算出でき、変数,...,が有限要素解析のパラメータである場合の、最適化問題の解法を概説する。a. 線形計画法(linear programming)目的関数と制約条件がすべて、変数,...,に関する1次式となる場合、式(4・169)の最適化問題は線形計画問題(linear programming prob-lem)と呼ばれる。スラック変数(slack variable)を導入したうえで、不等式制約条件を等式制約条件に書き直し、適宜の変換により変数に非負条件を付与して、線形の等式制約条件下で、線形の目的関数を最小化する標準形(standard form)に書き直す。制約条件を満足する実行可能解(feasible solution)の一つを、フェーズI問題(phase I problem)にて求める。フェーズI問題にて実行可能解が求解できない場合は、制約条件を満足する変数がない場合である。その実行可能解より出発して、単体法(simplex method)を用いて順次最適解の候補を更新していき、最終的な最適解を決定する。単体法の手順は表形式(シンプレックスタブロー)で簡潔に示されている。また、ゲーム理論に基づく「ゼロ和二人ゲーム(two-person zero-sum games)」の最適解も、線形計画法により求めることができる。. 有限要素感度解析(finite element sensitivity analy-sis)有限要素法による感度解析は、次の手順による。また、剛性行列と節点力ベクトルの微分値も、有限要素離散化の過程を順次追うことで算出可能である。以上より、節点変位の1階感度が式(4・173)により算出できる。また、固有値問題や動的問題にて解析される諸量についても、支配方程式の違いによる独自の処置は必要となるが、基本的には同様の手法により感度解析を行いうる。感度解析により、有限要素法にて解析できる構造応答の変数に関する感度が算出でき、その結果を用いて目的関数の感度が算出される。. こう配探索法(gradient search method)目的関数の感度、すなわちこう配を用いて目的関数の極値を探索するのが、非線形計画法の基本的手法である。制約条件がない場合に適用できる極値探索手法として、最急降下法(steepest descent method)、共役こう配法(conjugate gradient method)、ニュートン・ラプソン法(Newton-Raphson method)、準ニュートン法(quasi Newtonmethod)がある。最急降下法では、探索方向を最急降下方向、すなわち式(4・174)とする。最急降下法では、最終的な最適解近傍での収束速度が非常に遅いとされている。直前に求めた探索方向と、共役関係を満たす探索方向を設定することで、最適解への収束性を向上させた手法が、共役こう配法である。これを最初に考案したのがフレッチャー(Fletcher)とリーヴズ(Reeves)であり、彼らの方法では式(4・176)により探索方法を決定している。共役こう配法の場合も最急降下法同様に、ステップ幅αについては別途決定する必要がある。目的関数の2階感度を用いて、ステップ幅と探索方向を同時に決定する方法がニュートン・ラプソン法である。2階感度までを用いて目的関数を二次近似し、その二次関数を最小とする,...,をもって、最適解候補とする。そこで、セカント条件を満足する対称行列をヘッセ行列とみなして、ニュートン・ラプソン法を適用する準ニュートン法が考案された。近似的ヘッセ行列を算出する公式として、BFGS(Broyden-Fletcher-Goldfalb-Shannon)公式が広く用いられている。小さなから始めて、最適解候補を求めるごとにを大きくし、制約条件を厳密に満たす最適解を探索する。この場合は、大きなから始めて、最適解候補を求めるごとにを小さくし、不等式制約条件の境界にある最適解を探索する。また、最初の最適解候補は、不等式制約条件を満足するように選択しなくてはならない。c. 確率的探索法数理的方法によらないで、最適解候補を乱数等によりランダムに発生させて、その中から最適解を探索する確率的探索法も、計算機能力の向上と相まって多用されるようになっている。その確率的方法は、多数の最適解候補を同時に発生させる方法と、現在の最適解候補の近傍で一つずつ発生させる方法に大別できる。最適解候補を乱数により多数発生させ、その中から近似的最適解を選択する方法が、モンテカルロ法(Monte Carlo method)である。発生させる最適解の候補が多いほど真の最適解が含まれる確率が高くなるが、計算時間も長大となる。そこで、発生させる最適解候補の数をなるべく減らして、効率良く最適解を探索するため、種々のアルゴリズムが考案されている。その中で最も信頼性が高いとされているのが、遺伝的アルゴリズム(genetic algorithm)である。遺伝的アルゴリズムは、生物の進化を模擬した大域的な最適解の探索手法である。まず、最適解の候補となる多数の個体から成る母集団を形成し、そこからより最適性の高い次世代の母集団をシステマティックに形成する(12・4節に詳述)。多数の最適解候補を発生させるのではなく、現在の最適解候補の近傍でランダムに探索を行い、目的関数が改善されれば、それを次の最適解候補とする方法を局所探索法(local search)と呼ぶ。改善される場合にのみ最適解候補とすると、局所的な最適解に陥る可能性が高いので、改善されない場合でも、ある確率で最適解候補とする方法を、シミュレーテッドアニーリング法(simulat-ed annealing method,SA 法)という。この確率の大小を制御するパラメータを、順次小さくしていき、最終的な最適解を得る。が焼なましを制御する温度と類似であることから、この名で呼ばれるようになった。
