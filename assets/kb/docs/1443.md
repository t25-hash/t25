# 11・1・5 流体・構造・熱の連成

> **章**: 第11章 連成問題
> **節**: 11・1 熱

流体・構造・熱が連成する状況にはさまざまなものが考えられる
が，ここでは，流体領域と構造領域の境界は移動しないか，あるい
は，仮に移動してもその影響は十分小さいという場合を考える．例
えば，
(１)高温の流体と低温の流体が配管やタンク内で混合した結
果，構造壁面において空間的・時間的に変動する熱応力を生じる．
(２)発熱体であり異種材料接合体であるLSI （large scale
integration）パッケージの温度を抑えるために，パッケージに放
熱フィンを設置したり，パッケージ周辺を強制冷却する．
などである．
このような場合，流体領域においては温度場を考慮した熱流体解
析，構造領域においては熱伝導解析と応力解析が実施される．その
際，流体領域と構造領域の境界における，特に温度場に関する境界
条件に注意が必要である．熱流体解析においては，境界条件として
構造表面における温度もしくは熱流束の情報が必要であるが，本来
これらの値は構造側の計算結果に依存する．また，構造領域の熱伝
導解析では，同じく構造表面における境界条件として温度もしくは
熱流束の情報が必要であるが，逆にこれらの値は流体側の計算に依
存する．実際的な計算方法としては，時間進行で流体側と構造側の
解を交互に進めるにつれて，境界での情報を更新しながら境界条件
を整合させていく．
