# 5・1・2 静弾性問題の基本解

> **章**: 第5章 構造解析のための境界要素法
> **節**: 4・1 節で示されているように、ある方向の変位が一定(すなわ

境界要素法の定式化で用いる基本解は、正確には支配微分方程式の随伴形に対する特解である。静弾性問題の場合は、ナビエの方程式が自己随伴(self adjoint、随伴形が元の微分方程式と同じ形)であるため、基本解は次の微分方程式の特解として求められる。μ욡욅욢욢+λ+μ욢욅욢욡+δ욡욅δ-=0(5・20)ここで、とは異なる2点、δ-はディラックのデルタ関数(Dirac’s delta function)である。式(5・20)の解はケルビンの解(Kelvin’s solution)と呼ばれ、次式で与えられる。a. 三次元問題욡욢,=16π1-ν3-4νδ욡욢+욡욢(5・21)ここで、は2点との距離、욡のカンマ付きの添字は点の座標욡による偏微分を意味し、点,の座標욡と욡を用いて、次のように表すことができる。욡=욡-욡(5・22)同様に、二次元問題のケルビンの解は次のようになる。b. 二次元問題욡욢,=8π1-ν3-4νδ욡욢ln 1+욡욢(5・23)ケルビンの解は、無限の広がりをもつ弾性体中の1点の욡軸方向に、単位集中荷重が作用したときの別の点の、욢軸方向の変位という物理的意味をもつ。
