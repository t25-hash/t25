# 5・1・6 境界上の点の応力成分

> **章**: 第5章 構造解析のための境界要素法
> **節**: 4・1 節で示されているように、ある方向の変位が一定(すなわ

境界積分方程式を解けば境界上の表面力成分は得られるが、応力成分は得ることができない。境界の応力成分は、通常は次の三つの式のように、構成方程式、コーシーの公式、変位こう配とその接線微分・法線微分の関係式を連立して解くことにより、計算される。三次元問題の場合は、接線方向が2方向存在するから、式(5・58)に相当する式は、異なる2方向について、二つ用意しなければならないことになる。境界の応力を計算する方法には、内点の応力を計算する積分公式[式(5・51)]の点を境界に移行し、その極限から得られる積分公式を用いる方法もある。この方法を用いると、上記の方法よりも高精度に応力を計算できるといわれているが、超特異積分の数値的評価などの高度な離散化手法が必要となる。
