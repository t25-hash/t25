# 5・1・9 異方性弾性体(anisotropic elastomer)

> **章**: 第5章 構造解析のための境界要素法
> **節**: 4・1 節で示されているように,ある方向の変位が一定(すなわ

横等方性(transverse isotropy),直交異方性(orthotropy),一般の異方性弾性体についても同様に境界要素法の定式化が可能である.異方性弾性体では,構成方程式の弾性定数の個数が等方弾性体と異なるため,基本解も,ケルビンの解とは異なるものとなる.二次元異方性,横等方性,直交異方性弾性体の基本解は,析のための境界要素法章構造解第56―5閉じた形で得られているが,一般の異方性弾性体の基本解は,単位円の円周に沿った積分の形でしか求まらないので,基本解の計算に時間がかかることになる.動弾性問題は,振動外力を受ける機械構造物の振動解析,弾性体内の波動伝ぱ解析,構造物の固有振動解析,衝撃強度評価など,多くの工学上の問題において扱われる.一般に動弾性問題は,調和外力を受ける場合の定常振動解析と,衝撃的な外力が作用する場合の非定常解析に分けられる.本節では,そのおのおのについて境界要素法を適用する方法を,具体的に解説する.
