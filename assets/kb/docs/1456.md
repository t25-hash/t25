# 5・2・2 共振周波数解析

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・2 動弾性問題の定式化と解法

定常振動問題に関連して、弾性体の共振周波数(resonant fre-quency、固有振動数)を求めることは、工学的に重要である。固有振動数は、解くべき連立方程式が自明でない解をもつための条件により求めることができる。連立方程式(5・84)が非自明解をもつためには、次式で表されるように、係数行列の行列式が零であればよい。なぜならば、係数マトリックスが角周波数ωの関数となるからである。すなわち、固有振動数を求めるには、式(5・85)のωを変化させ、その値が零となるωを数値的に探索しなければならない。そこで、以下のように定式化に用いる基本解に、静弾性問題のケルビンの解を用いた定式化が、一般に適用されている。静的な基本解を用いる定式化では、係数マトリックスに関する繰返し計算が不要となるため、有限要素法等と同様に、角周波数ωに関する代数固有値問題(eigenvalue problem)に帰着させて、問題を解くことができる。この定式化の概略を以下に示す。ただし、議論を簡略化するために、物体の境界条件はすべて周辺自由(表面力0)として取扱う。これらの積分方程式を適当な要素を用いて離散化すると、以下のようなマトリックス方程式にまとめることができる。最終的に式(5・88)は以下のような一般固有値問題の形に帰着できるから、固有値λを求めることによって共振周波数が算出できることがわかる。また、既存の高精度な固有値解析ライブラリを容易に適用することができるといった利点を有している。
