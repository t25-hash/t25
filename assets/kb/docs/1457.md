# 5・2・3 非定常問題(unsteady problem)

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・2 動弾性問題の定式化と解法

非定常問題の境界要素法による解析法は,以下の三つの方法に大別することができる.(1) 時間微分を含む基礎式を直接満足する基本解を用いる方法(直接解法).(2) フーリエ変換ないしはラプラス変換(Laplace transform)された基礎式を離散化する方法(変換解法).(3) 時間項に差分近似を適用する方法(時間ステップ解法).これらのうち,対象となる現象が線形である限りにおいては,ラプラス変換を用いる解析法(ラプラス変換解法)が最も有効で,現在までに二次元および三次元弾性体,板・はりの曲げ問題など,種々の問題が境界要素法により解かれている.ラプラス変換解法はさらに,(a)基礎式のラプラス変換を満足する基本解を用いる方法と,(b)静弾性問題と同様の基本解を用いたうえで慣性力の項に対しては領域積分を適用して評価する解法の,二つに大別される.両者を比較すると,基本解を導くことができれば,一般に領域積分が不要となる前者の方法が,解析精度ならびに計算効率の面で有利である.本項では二次元弾性問題を例として,ラプラス変換を用いた非定常問題の解析法(a)について,具体的に解説する.弾性体の基礎式(5・72)を以下の初期条件:욡=0=0옿욡옿욧=0(5・90)すなわち,時刻=0において,物体の初期変位と初速度がともに零である条件のもとにラプラス変換すると,次式を得る.μ욡욢욢+λ+μ욢욢욡=ρ욡(5・91)ここで,はラプラス変換パラメータである.なお, ̄が付された諸量は,すべて元の物理量のラプラス変換を意味する.基礎式のラプラス変換である式(5・91)を解くために,次式を満たす基本解욡욢を導入する.μ욡욢욅욅,+λ+μ욅욢욅욡,+δ욡욢=ρ욡욢,(5・92)上式は,定常問題の基本解に関する基礎式(5・75)におけるフーリエ変換を,ラプラス変換に置換えたものとみなすことができる.両者の違いは,右辺のと-ωだけであるから,定常問題の基本解の定義式(5・78)におけるωを-とおけば求めることができる.基本解욡욢を基礎式(5・91)の重み関数として用い,境界要素法の定式化を適用すれば,以下に示されるような境界積分方程式を,容易に導くことができる.욡욢욢+욡욢,욢dΓ=욡욢,욢dΓ(5・93)各物理量の時間変動解を求めるには,上記の境界積分方程式(5・93)をラプラス変換領域において数値的に解き,各物理量のラプラス変換を求めた後,ラプラス逆変換を施せばよい.なお方程式を解く際には,境界条件もまた,ラプラス変換を施した後に用いる必要がある.得られた像空間解を実時間に戻すには,ラプラス逆変換を行わなければならないが,一般には解析的な逆変換は困難であるから,もっぱら数値的手法が適用される.5・2・4項にて,数値ラプラス逆変換の代表的な方法について触れておく.
