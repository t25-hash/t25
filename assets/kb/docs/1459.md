# 5・2・5 動弾性問題における特異積分の処理

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・2 動弾性問題の定式化と解法

次に、ラプラス変換された非定常問題の境界積分方程式(5・93)を例にとり、動弾性問題における特異性(singularity)の取扱いについて述べる。なお、以下に述べる取扱いは、フーリエ変換された定常問題に対する境界積分方程式にも、全く同様に適用できる。6―62計算力学まず、理解を容易にするために、境界積分方程式(5・82)に含まれる基本解を、特異性を有する静的項(ラプラス変換パラメータを含まない項)と、それ以外の特異性のない動的項に分離して書き改める。このように、非定常問題における境界積分方程式では、基本解の静的項のみに特異性が含まれる点に注意しなければならない。上記の特異性を処理するには、静弾性問題と同様、次式で示される剛体変位条件を用いることができる。式(5・114)と式(5・115)の差をとると、次式を得る。なお、実際の計算では、基本解を静的項と動的項に分離する必要は特になく、両者を再びまとめた次式を離散化すればよい。これらの弱い特異性は、静的問題の場合と同様に、二次元問題の場合は対数型積分公式やべき写像法、三次元問題の場合は三角形要素によるサブエレメント法を用いることにより、評価することができる。また、剛体変位条件のみでなく、一様な応力場(ひずみ場)の条件を用いることにより、上記の特異性を同時に処理することも可能である。以下にその概略を示す。応力、すなわち変位こう配が一様な系について、以下のような積分方程式が成立する。式(5・114)から式(5・118)を差引けば、次式となる。なお実際には、式(5・119)の静的項と動的項を加えて変形された次式を離散化する。以上の手順により、境界積分方程式は、弱い特異性まで含めた形で正則化されるため、通常のガウスの積分公式を用いても、高精度に積分を実行することが可能となる。
