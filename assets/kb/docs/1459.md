# 5・2・5 動弾性問題における特異積分の処理

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・2 動弾性問題の定式化と解法

次に,ラプラス変換された非定常問題の境界積分方程式(5・93)を例にとり,動弾性問題における特異性(singularity)の取扱いについて述べる.なお,以下に述べる取扱いは,フーリエ変換された定常問題に対する境界積分方程式にも,全く同様に適用できる.6―62計算力学まず,理解を容易にするために,境界積分方程式(5・82)に含まれる基本解を,特異性を有する静的項(ラプラス変換パラメータを含まない項)と,それ以外の特異性のない動的項に分離して書き改める.욡욢욢+욡욢,+욡욢,욢dΓ=욡욢,+욡욢,욢dΓ(5・114)ここで,静的項욡욢,욡욢は静弾性問題に対する基本解に一致し,욡욢は可積分の弱い特異性を,욡욢はコーシーの主値オーダの特異性を有する.他方,動的項욡욢,욡욢は,ラプラス変換パラメータを含む項から成る振動成分である.このように,非定常問題における境界積分方程式では,基本解の静的項のみに特異性が含まれる点に注意しなければならない.上記の特異性を処理するには,静弾性問題と同様,次式で示される剛体変位条件を用いることができる.욡욢욢+욡욢,욢dΓ=0(5・115)なお,上式における基本解は,静弾性問題と同様の욡욢となることに注意されたい.式(5・114)と式(5・115)の差をとると,次式を得る.욡욢,욢-욢dΓ+욡욢,욢dΓ=욡욢,+욡욢,욢dΓ(5・116)ここで,上式左辺の욢-욢は,0においての性質を有しているから,基本解욡욢の特異性を1階キャンセルすることができる.なお,実際の計算では,基本解を静的項と動的項に分離する必要は特になく,両者を再びまとめた次式を離散化すればよい.욡욢,욢dΓ-욡욢,욢dΓ=욡욢,욢dΓ∈Γ(5・117)上式を用いることにより,基本解욡욢の特異性は1階だけキャンセルされるが,境界積分方程式には可積分ながら,弱い特異性が残存する.これらの弱い特異性は,静的問題の場合と同様に,二次元問題の場合は対数型積分公式やべき写像法,三次元問題の場合は三角形要素によるサブエレメント法を用いることにより,評価することができる.また,剛体変位条件のみでなく,一様な応力場(ひずみ場)の条件を用いることにより,上記の特異性を同時に処理することも可能である.以下にその概略を示す.まず,領域Ω内で応力が一様にσ욡욢,かつ点Pにおける変位が욢であるような系を考える.応力,すなわち変位こう配が一様な系について,以下のような積分方程式が成立する.욡욢욢+욡욢,욢+욢욅욅,dΓ=욡욢,λ욆욆욢+μ욢욅+욅욢욅dΓ(5・118)ただし,次式の核関数욡욢および욡욢は,式(5・115)と同様に静的問題に対応する基本解である.また,上式の応力は変位こう配욢욅を用いて書き改めてある.式(5・114)から式(5・118)を差引けば,次式となる.욡욢,욢dΓ+욡욢,욢-욢-욢욅욅,dΓ=욡욢,욢dΓ+욡욢,욢-λ욆욆욢-μ욢욅+욅욢욅dΓ(5・119)ここで,特異性を示す核関数욡욢,욡욢の係数部分について,=0の極限を考えれば,욢-욢-욢욅욅,(5・120)욢-λ욆욆욢-μ욢욅+욅욢욅(5・121)となる性質を有していることがわかる.すなわち,基本解욡욢については1階,욡욢については2階,それぞれ特異性が低減されることになり,境界積分方程式における特異性は,すべてキャンセルされることがわかる.なお実際には,式(5・119)の静的項と動的項を加えて変形された次式を離散化する.욡욢,욢dΓ-욡욢,욢+욢욅욅,dΓ=욡욢,욢dΓ-욡욢,λ욆욆욢+μ욢욅+욅욢욅dΓ(5・122)すなわち,動的問題に対する境界積分方程式から,基本解の静的項だけについて成立する,一様な応力場の系の積分方程式の差をとった形の境界積分方程式を離散化すればよい.以上の手順により,境界積分方程式は,弱い特異性まで含めた形で正則化されるため,通常のガウスの積分公式を用いても,高精度に積分を実行することが可能となる.
