# 5・3・1 破壊力学解析

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

a. 破壊力学と境界要素法境界要素法の破壊力学(fracturemechanics)への適用は、弾性解析法がそのまま適用できるため、非常に早い段階で行われた。また、表面力が直接解として得られることや、変位と独立して同程度の内挿が行われることも、破壊力学解析に適していた。しかし、何といっても無限大となる応力場を取扱うため、実用に耐える方法論が確立するまでには多くの研究が必要であり、弾性解析法自体の高精度化、き裂問題としての定式化、無限大となる応力分布の表現、および破壊力学パラメータの精度良い決定法などが精力的に研究された。これらの課題がほぼ解決した現在では、境界要素法は破壊力学解析に関して、体積力法に匹敵する高精度と計算効率、有限要素法に匹敵するはん用性と利便性を兼ね備えた解析法として確立している。b. き裂先端近傍の応力・変位分布き裂先端近傍(0)では応力と変位の各成分は、それぞれ次式のようなモードI、モードII、モードIIIの和で与えられる。0のき裂先端近傍では、応力は第1項がの特異性をもって支配的となり、各モードの応力拡大係数がその大きさの指標となっている。すなわち、き裂先端近傍の応力分布を求めることは、応力拡大係数を求めることに帰着する。c. き裂問題の定式化。 き裂面上では1個の座標点において、上下面の各変位、あるいは片方の変位と変位差という、2個の未知変位ベクトルが存在する。これに対して、通常のようにソース点をき裂の上下面上に設定しても、座標が一致するので独立な積分方程式が得られず、未知数が過剰となる。(2) 領域分割解法き裂問題の特殊性を克服する最も一般的な方法は、き裂面が外境界の一部となるように領域を分割し、結合解法で解く方法(multiregion method)であり、通常の弾性問題と全く同様に、未知数の数と方程式の数が一致して、連立方程式として解けるようになる。リガメント側は表面力の解がそのまま使えるように、き裂面の延長面上で分割するのがよい。分割した領域ごとに式(5・125)を離散化し、結合条件(き裂面上では表面力フリーまたは内圧、リガメント側では変位の連続および表面力の平衡条件)を適用する。この方法の特殊な場合として、対称性を利用した解析法もある。解析条件がき裂面に対して対称である場合、き裂面とその延長面上で領域分割すれば、き裂面とリガメント側も外境界の一部となり、解析上、通常の弾性問題と全く変わるところはなくなる。そして結合解法も不要となる。. discontinuity method)もある。これは、一種の間接積分方程式法と見ることができる。. 特異性を除去する方法他の方法が、特異性をいかにモデル化するかで苦心しているのに対し、この方法は、特異性そのものを除去して解析する方法(subtraction of singularitymethod)である。これには重ね合わせの原理を適用する。すなわち、与えられた問題の変位と表面力を、次式のように特異項とそれ以外の正則項の和で表す。正則成分に境界積分方程式(5・125)を適用し、次式が成立する。すなわち、新たな境界条件、式(5・131)のもとで、積分方程式(5・130)を解く問題に帰着された。この方法では未知数として値が1個増えることになるが、その代わりき裂先端において=0が成立するので、未知数の数は結局同じである。このように、本方法は応力拡大係数が連立方程式の解として直接得られる。この方法は特異性を除去しているので、精度は高いことが期待されるが、き裂先端から遠い境界で境界条件の精度が悪くなる。さらに、領域分割解法は依然として必須である。. グリーン関数を用いる方法(crack Green’s functionmethod)この方法は、二次元無限板中においてき裂面の表面力フリー条件を満たす解を、式(5・126)の中の核関数として用いる。すると、式(5・126)の右辺第2項の積分がすべて零となり、き裂面の離散化が不要となる。したがって、極めて高精度の解が得られるが、連立方程式を解いた後の内点の二次的計算が必要になる。また、三次元き裂ではそのようなグリーン関数は求まっていないので、適用は二次元問題に限られる。d. 応力拡大係数の決定法と特異要素き裂先端近傍は特異応力場のため、実際の計算値が乱れやすい。また、通常の要素を用いて離散化した場合、本来無限大となるべきき裂先端で表面力が有限値に留まり、さらに節点値を多項式で内挿するため、本来の応力分布とは異なる分布として表現される。したがって、応力拡大係数(値)が直接得られる定式化法でない限り、得られた解析結果から値を精度良く決定する方法が必要である。一方、特異応力場を表現できる要素(特異要素)も考えられている。これらについて以下に示す。. 外挿法境界要素法では、解析結果として表面力と変位が得られるので、これらをそのまま利用できる決定法のほうが望ましい。その代表的な決定法として外挿法(extrapolationmethod)がある。このように応力の式を使って外挿する方法を、応力外挿法と呼ぶ。同様のことは他のモードについても成立し、三次元き裂問題にも適用可能である。以上のことは変位を用いても可能であり、次式を用いる。この方法を変位外挿法と呼ぶ。外挿にはある程度の数の外挿点が必要であるが、三次元解析などで十分な要素分割ができないときは、信頼できる外挿ができなくなる。この観点から、変位外挿法と応力外挿法を同時に用いる外挿法も開発されている。変位外挿直線の傾きは、応力外挿の場合の3倍なので、変位の側ののスケールを3分の1に縮小し、応力外挿と変位外挿を接続すれば、両者を通して1本の直線で外挿が可能となる。これを接続外挿法(hybrid extrapolation method)と呼ぶ。接続外挿法は一般的なプログラムで、それなりの精度の解を得る目的のためには最も適している。外挿法ではの高次項が無視されているので、き裂先端付近で要素分割を細かくできないような場合には、直線性が保証されない。. 比例法比例法(proportional method)は、接続外挿法よりも高精度の解が必要な場合に有効である。これはき裂先端近傍の表面力、あるいは変位の解をそのまま利用し、基準解との比をとることにより値を決定する方法であり、比較的簡便な割には高精度の解が得られる。似ている形状の解析が多い場合は特に有力である。ここではモードIを例にとる。言い換えると、二つの異なる問題A,B について、が等しい点でのとの比は近似的に一定であり、次式が成立することを意味する。このような方法を比例応力法と呼ぶ。同様な比例法は変位を用いても導くことができる。実際に比をとるのに用いる節点値としては、比例応力法の場合はき裂先端の位置の表面力、比例変位法の場合は、き裂先端に最も近い位置の節点変位をとるとよい。比例法は混合モードや三次元き裂にも適用可能であるが、第1項のみに立脚しているため、を十分小さくとれないような場合は、高次項の影響で比例関係の成立が悪くなる。. 積分法とエネルギー法ライス(Rice)により提案された、積分と応力拡大係数の関係を利用した計算法も試みられている。積分を計算するには、定義そのままの経路積分を直接行うのと、エネルギー解放率から求めるのと二とおり考えられる。経路積分による方法は、応力やひずみを計算しなくてはならず、内点の二次計算が必要になり、境界要素法の利点が損なわれる面がある。積分は経路独立であるが、数値計算上、き裂先端近傍ではかなり経路によって値がばらつくので、き裂先端から離れた経路をとるべきである。また、三次元き裂問題の場合、経路のとり方がかなり複雑となり、混合モードの場合は積分値を各モードに分離する工夫が必要である。すなわち、同一弾性体において、き裂長を微小に変えた二つの解析を行い、そのときのポテンシャルエネルギーの変化を求めれば、ひいてはを求めることができる。エネルギー差を求める際に誤差がキャンセルされ、精度が比較的良いといわれるが、解析を二度行わねばならない。また、三次元き裂で値が連続的に変化する場合には、局所的な値を求めるのが非常に面倒である。. 特異要素を用いる方法き裂先端近傍の表面力と変位分布を、通常の多項式系の要素による内挿で表現することはできない。そこで、解析の高精度化を図って、その関数変化を表現できる要素がいくつか開発された。この目的の要素群を、特異要素(sin-gular element)と呼ぶ。特異要素は通常、き裂先端を含む箇所にのみ配され、周囲の通常の要素に接合されて用いられる。その多くは、応力拡大係数の決定法のいずれかと併用して効果を挙げている。代表的なものは、二次のアイソパラメトリック要素の中間節点を、要素中点ではなくき裂先端から4分の1の点に移動した、いわゆるクォータポイント要素(quarter-point element)である。これは要素の中点を移動して非線形写像を行う意味があり、通常の多項式の内挿関数で変位の関数形を表現できる。クォータポイント要素は、そのままでは表面力の変化を表現するには適さないが、内挿関数を少し修正することにより、表面力の変化も表現できるようになる。このほか、特殊な内挿関数を工夫した要素もある。この中には、応力拡大係数の決定法を介せずに、直接求まるような要素もある。
