# 5・3・2 非線形問題(nonlinear problem)解析

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

構造解析において非線性が問題となるのは、座屈解析のように構造物の変形が大きくなる場合(幾何学的非線形問題)と、材料の力学的特性が非線形となる場合(材料非線形問題)とに大別される。境界要素法は、線形問題では境界積分方程式を用いて、境界だけを要素分割することで解析できる。この点は、境界要素法がもつ特徴の中で、特に他の数値解析手法に対しての利点となっている。一方、対象とする問題の微分作用素が非線形である場合には、領域積分が残ることになる。微分作用素が、線形部と非線形部に分けられるとすると、次式のような形式の積分方程式となる。このように、積分法方程式は領域積分を含むことになるので、厳密な意味では境界要素法とはいえなくなるが、この式に基づく数値解析手法も、一般に境界要素法と呼ばれている。非線形が強くない場合には、の項を無視した解から出発して、繰返し計算によって逐次的に解の精度を上げていくことが行われる(反復境界要素法)。この解法では、本質的な離散化は境界だけで良いので、境界要素法の利点が保たれる。非線形が強い問題では、反復的解法は効率的でなくなるため、領域積分項に対しても要素分割を導入することが行われる(境界-領域要素法)。以下では、境界要素法の非線形問題への後者の手法の適用例として、弾塑性解析について示す。簡単のために物体力の存在を無視している。塑性ひずみの非圧縮性ε=0を仮定すると、内点におけるひずみ速度は次式で与えられる。ここで、塑性負荷状態ではα=1、その他の状態ではα=0である。以上の3式を離散化すると、以下のような代数方程式が得られる。第1式の右辺第2項が塑性ひずみの寄与分であり、塑性変形を生じる領域に内部要素を設けることで求められる。塑性ひずみの値はあらかじめ既知ではないので、繰返し計算によって求めることになる。続いて、第2式、第3式を用いてを求め、この値が先に仮定した値の許容範囲に入っていれば次の荷重段階へ進み、そうでなければ収束するまで同様の計算を繰返す。弾塑性解析では内部要素を必要とするが、あらかじめ塑性変形が生じると予想される部分にだけ、これを設ければ良いのが境界要素法の特徴である。図5・4に計算例を示す。図5・4は、円孔をもつ板が引張負荷を受ける場合の、塑性変形域の拡大の様子を解析したもので、破線で示された三角形が塑性ひずみの寄与を計算するために設けられた内部セルである。なお、計算の効率化のために、線形弾性体の部分は境界要素法で、塑性変形領域は有限要素法で離散図5・4引張負荷を受ける円孔をもつ平板の塑性域進展の様子図5・5 開領域問題の取扱い方6―65化する結合解法(5・3・3項参照)も提案されている。以上のように、非線形問題では領域積分が残り、これにより他の数値解析手法に対する境界要素法の利点が、ある程度減じられることになる。この点を改善するために、以下のような領域積分を境界積分に変換する数値計算上の工夫が、種々提案されている。(1) 領域積分を解析的に積分する方法(2) フーリエ級数展開を利用する方法(3) ガラーキンテンソルを利用する方法(4) 多重相反法(5) 双対相反法(6) 計算点解析法これらの詳細については、文献(58)~(60)参照。
