# 11・7・2 化学反応機構（kinetic mechanism） ，熱化学定数お

> **章**: 第11章 連成問題
> **節**: 11・7 燃焼流れ

ここで， は混合分率や反応進行度変数の変動強度である．
また，燃料，酸化剤，燃焼ガスの渦塊が崩壊する過程のうち，最も遅いものが反応速度を律速すると考えるマグヌッセンモデル（あるいは渦消散モデル) を用いる場合もある．
そのほか，BML（Bray-Moss - Libby）モデル や局所ダムケラー数モデル，化学種の輸送方程式を解く代わりに化学種の確率密度関数などがあり，解析の対象となる乱流燃焼場の特性によって使い分け速度を決定すると考え，燃料の反応速度を次のように与える．
燃焼反応を記述する反応機構は非常に複雑であり，燃料によっては，数十種以上の化学種と数百組以上の素反応を考慮に入れる必要がある．
メタン等に関しては50種程度の化学種と，300組程度の素反応機構 が用いられている．
燃焼場では，温度が常温から2000K程度まで急激に変化するため，熱化学定数と輸送係数の温度依存性を考慮に入れる必要がある．
一般にはJANAFのテーブル等 や，アメリカ，サンディア国立研究所が開発したChemkinのデータベース を用いて与えることができる．
