# 5・3・4 最適設計

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

構造物や機械要素における最適設計法(design optimizationmethod)には各種のものが提案されているが、設計変数のとり方によって分類すれば、解析対象の形状寸法を設計変数とする形状最適化問題、解析対象の設計パラメータを設計変数とするパラメータ最適化問題、解析対象の構造レイアウトを設計変数とするレイアウト最適化問題等に分けることができる。このうち、以下のような理由から、境界要素法は形状最適化問題に特に適している。形状最適化問題では、解析対象の境界形状が逐次変更して解析されるので、有限要素法などの領域型解析法を用いると、形状変更とともに初期仮定形状について考えた要素メッシュ全体が、大きくゆがんだり重なったりして正確な解析が困難となる。一方、境界要素法では、解析対象の境界だけを要素分割して解析を行うので、形状変形でメッシュがゆがむ可能性は有限要素法などよりも低く、もし境界が大きい変更を受けてメッシュ再構成の必要性が生じても、それは有限要素法に比べてはるかに容易である。また、最適設計問題では、目的関数や制約条件としてしばしば応力を用いるが、最大応力は通常、境界上で生じるので、境界上の応力値を精度良くまた効率良く計算する必要がある。この点でも、境界要素法は有限要素法より有効である。その理由として、境界要素法が境界型解析法であって、応力を評価する節点を境界上に配置できること、有限要素法では変位の内挿関数よりも応力の内挿関数が一次低くなるのに対して、境界要素法ではこのような制約がないことが挙げられる。このように、境界要素法は形状最適化問題に適した数値解析法であるが、特に設計感度解析において、有限要素法よりも数学的な難しさがある。通常、最適化問題は高度に非線形な問題なので、最適解を求めるには目的関数の設計変数に対する変化量、つまり設計感度を利用して解くことが一般的に行われる。境界要素法の場合、定式化に特異性を有する基本解を用いるので、基本解やその導関数のように、高い特異性を有する関数を含む境界積分方程式をどのように処理するかは、非常に大きな問題である。これまでに提案されている設計感度解析手法を整理すると、数値微分法、随伴変数法、直接微分法、その他に分類できる。a. 数値微分法(numerical differentiation)数値微分法では、前進差分や中心差分を用いて設計感度を計算する。この方法はアルゴリズムが簡単で、あらゆる問題に対して容易に適用できるが、計算コストが高いこと、計算精度が差分ステップ幅に依存することなどの欠点がある。6―67そこで、この方法の効率を改善するために、差分荷重法が提案されている。この方法では、実際に設計変数を微小変化させて(つまり、形状を変化させて)解析を行う代わりに、形状変更と等価な外力の変化量を与えて解析を行うことで設計感度を求める。この方法では全体系係数行列を毎回構成する必要がないので、計算効率を改善することができる。b. 随伴変数法(adjoint variable method)随伴変数法では、積分形で表された目的関数に対して随伴方程式を導出し、それを解いて設計感度を計算する。この方法は、数値微分法に比べて設計感度を精度良く、効率良く計算できるが、目的関数の形によって随伴方程式の定式化が困難な場合があること、解析対象や目的関数ごとに定式化する必要があることなどが指摘されている。また、この方法を連続体の感度解析に適用する場合には、随伴問題における随伴荷重が点荷重となって荷重点付近に特異性を生じるため、設計境界に随伴荷重を作用させる場合、設計感度を精度良く計算できないので、これを改善するために、次のような方法が提案されている。第1は、メッシュ改善法や特殊要素を利用する方法である。メッシュ改善法では、随伴荷重作用点付近のメッシュを小さくとる。また特殊要素法では、随伴荷重作用点に隣接する要素に、半非適合要素などの特殊な要素を用いる。これによって設計感度の計算精度を改善することができるが、初期メッシュと同じものを随伴問題に用いることができないため、計算効率を低下させる。第2は、随伴荷重の作用する要素の解を既知の特異解と未知の非特異解の和として定式化し、問題を非特異な問題に変換する方法である。これは、局所特異関数法と呼ばれる。しかし、既知の特異解についての境界積分が必要となり、曲線要素ではこれを解析的に評価することが困難である。第3は、平滑荷重法である。この方法では、随伴荷重を点荷重として与える代わりに、それと等価で滑らかな分布荷重を用いる。滑らかな分布荷重を用いれば特異性を生じないので、精度良く感度を計算できるが、数学的な厳密さを欠くことになる。第4は、特異性除去法である。この方法では、随伴問題を特異性のある問題と特異性のない問題に分ける。そして、これら二つの問題を別々に解いて、随伴問題の解をこれらの解の和として得る。基本的な考えは局所特異関数法と似ているが、局所特異関数法のように定式化の段階で特異関数を導入しないので、それに対する解析的な処理を必要としない利点がある。c. 直接微分法(direct differentiation)直接微分法では、評価関数を含む境界積分方程式を設計変数で直接微分し、得られた方程式を離散化して解くことにより、状態変数の設計感度を求める。この方法は、評価関数を陰に含む境界積分方程式を設計変数で微分し、得られた方程式を離散化して評価関数の設計感度について解く陰解法と、評価関数を陽に含む積分方程式を導出し、それを設計変数で微分して設計感度を求める陽解法に分けることができる。陰解法の定式化理論は次のようになる。境界要素法では、特異性を有する基本解を用いて、支配方程式から境界積分方程式を導出する。陰解法では、この境界積分方程式を設計変数で微分して、評価関数の設計感度を含む積分方程式を導出する。これを離散化して解くことにより、設計感度を計算する。この問題を解決するために、次のような工夫がなされている。第1の方法は、ソース点を微小量だけ境界の外に移動して積分する方法である。このようにすれば特異性の問題は解決されるが、数学的な厳密さを欠くことになる。また、係数マトリックスの微係数を数値微分によって求める半解析的手法も提案されている。通常の定式化では、境界上の観測点における未知関数についての境界積分方程式が導出される。しかし、実際に必要になるのは、未知関数の導関数について設計感度である場合が多い。陽解法では、最初に未知関数の導関数について境界積分方程式を導出し、これを設計変数で直接微分することで、応力の設計感度についての式を導出する。ここで方程式の特異性を下げるために用いられる定式化は、陰解法で同様な目的に用いられる定式化と基本的には同じである。d. その他の方法境界要素法では、定式化に用いる基本解が特異性を有しているために、境界積分方程式に含まれる被積分関数の特異性により、感度解析の定式化が複雑になる。そこで、特異な基本解の代わりに、非特異なT-complete関数を用いて境界型の定式化を行い、導かれた関係式を直接微分することで、設計感度解析を行う方法が提案されている。このような定式化をトレフツ(Trefftz)法という。この方法では、導出された未知関数についての関係式は非特異であるので、これを設計変数で直接微分することが可能であり、設計感度を容易に評価することができる。
