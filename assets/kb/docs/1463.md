# 5・3・4 最適設計

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

構造物や機械要素における最適設計法（design optimization
method）には各種のものが提案されているが，設計変数のとり方
によって分類すれば，解析対象の形状寸法を設計変数とする形状最
適化問題，解析対象の設計パラメータを設計変数とするパラメータ
最適化問題，解析対象の構造レイアウトを設計変数とするレイアウ
ト最適化問題等に分けることができる．このうち，以下のような理
由から，境界要素法は形状最適化問題に特に適している．
形状最適化問題では，解析対象の境界形状が逐次変更して解析さ
れるので，有限要素法などの領域型解析法を用いると，形状変更と
ともに初期仮定形状について考えた要素メッシュ全体が，大きくゆ
がんだり重なったりして正確な解析が困難となる．一方，境界要素
法では，解析対象の境界だけを要素分割して解析を行うので，形状
変形でメッシュがゆがむ可能性は有限要素法などよりも低く，もし
境界が大きい変更を受けてメッシュ再構成の必要性が生じても，そ
れは有限要素法に比べてはるかに容易である．
また，最適設計問題では，目的関数や制約条件としてしばしば応
力を用いるが，最大応力は通常，境界上で生じるので，境界上の応
力値を精度良くまた効率良く計算する必要がある．この点でも，境
界要素法は有限要素法より有効である．その理由として，境界要素
法が境界型解析法であって，応力を評価する節点を境界上に配置で
きること，有限要素法では変位の内挿関数よりも応力の内挿関数が
一次低くなるのに対して，境界要素法ではこのような制約がないこ
とが挙げられる．
このように，境界要素法は形状最適化問題に適した数値解析法で
あるが，特に設計感度解析において，有限要素法よりも数学的な難
しさがある．通常，最適化問題は高度に非線形な問題なので，最適
解を求めるには目的関数の設計変数に対する変化量，つまり設計感
度を利用して解くことが一般的に行われる．境界要素法の場合，定
式化に特異性を有する基本解を用いるので，基本解やその導関数の
ように，高い特異性を有する関数を含む境界積分方程式をどのよう
に処理するかは，非常に大きな問題である．これまでに提案されて
いる設計感度解析手法を整理すると，数値微分法，随伴変数法，直
接微分法，その他に分類できる．
a.数値微分法（numerical diﬀerentiation) 数値微分法で
は，前進差分や中心差分を用いて設計感度を計算する．この方法は
アルゴリズムが簡単で，あらゆる問題に対して容易に適用できる
が，計算コストが高いこと，計算精度が差分ステップ幅に依存する
ことなどの欠点がある．
そこで，この方法の効率を改善するために，差分荷重法が提案さ
れている．この方法では，実際に設計変数を微小変化させて（つま
り，形状を変化させて）解析を行う代わりに，形状変更と等価な外
力の変化量を与えて解析を行うことで設計感度を求める．この方法
では全体系係数行列を毎回構成する必要がないので，計算効率を改
善することができる．
b.随伴変数法（adjoint variable method) 随伴変数法では，
積分形で表された目的関数に対して随伴方程式を導出し，それを解
いて設計感度を計算する．この方法は，数値微分法に比べて設計感
度を精度良く，効率良く計算できるが，目的関数の形によって随伴
方程式の定式化が困難な場合があること，解析対象や目的関数ごと
に定式化する必要があることなどが指摘されている．また，この方
法を連続体の感度解析に適用する場合には，随伴問題における随伴
荷重が点荷重となって荷重点付近に特異性を生じるため，設計境界
に随伴荷重を作用させる場合，設計感度を精度良く計算できないの
で，これを改善するために，次のような方法が提案されている．
第1は，メッシュ改善法や特殊要素を利用する方法である．メッ

b.メッシュ更新法 メッシュ更新法は，h法，p法，r法と名
付けられる方法，その他に大別される．
h法では要素の内挿関数は変化しないままで，要素数が増えるこ
とにより計算精度を改善する．p法では要素分割は一定のままで，
要素の内挿関数の次数を上げることによって近似精度を改善する．
また，r法では要素数や内挿関数の次数は一定のままで，要素配置
を変更することによって数値解の精度を改善する．
h法，r法では，メッシュ更新後に全体系行列を再計算すると計
算コストが高くなるので，計算コストを下げるために，階層型内挿
関数を用いた方法が提案されている．この方法では，実際に要素細
分割を行う代わりに，初期解析に用いた内挿関数に新たに項を追加
する．追加された内挿関数項についてだけ係数行列の再計算を行う
ので，計算コストを下げることができる．
h法，p法，r法の利点と欠点を補完し合う結合解法として，hp
法とhr法などが提案されている．hp法では，特異点に隣接する要
素でh法を，それ以外の要素でp法を用いている．これは，正解
の特異点の近傍を除けば，p法はh法よりも誤差収束特性がよい
（約2倍の収束速度を示す）ことを考慮して提案された．これに対

てから，r法によって再修正する．
