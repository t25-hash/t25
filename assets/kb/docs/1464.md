# 11・7・4 乱流燃焼（turbulent combustion）の数値解析

> **章**: 第11章 連成問題
> **節**: 11・7 燃焼流れ

a.時間平均モデル レイノルズ平均に基づく非反応性乱流とは異なり，乱流燃焼の場合には，ファーブル平均（Favreaverage，質量荷重平均) に基づく基礎方程式を解析する．
レイノルズ応力項や乱流拡散流束項等のモデル化には，非反応性乱流の場合に開発されている - εモデルや乱流プラントル数，乱流シュミット数を用いたこう配拡散型のモデルが用いられる．
乱流燃焼場の場合，これらのモデルに加えて化学反応項のモデルを導入する必要がある．
渦崩壊モデル では，乱流エネルギーの減衰率が反応である．
上述の基礎方程式に加えて，補助方程式として次のような状態方程式を用いる．
噴霧燃焼などの特別な場合を除いて，燃焼流れの数値計算を行う際に解析対象となる基礎方程式は，の質量保存式，の運動量保存式，のエネルギー保存式，およびの化学種保存式との状態方程式である．
