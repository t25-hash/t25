# 5・3・5 適応メッシュ法

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

境界要素解の計算精度は、解析に用いる境界要素分割に依存している。そこで、望みの計算精度を有するようなメッシュ分割を、自動的に決定する方法が検討・開発されている。これを適応メッシュ法(アダプティブメッシュ法、adaptive mesh refinement)と呼ぶ。適応メッシュ法に関する研究は有限要素法で始まり、その成果が後に境界要素法にも利用された(有限要素法については4章を参照していただきたい)。しかし、両者には定式化理論等において根本的に相違があるので、後に境界要素法独自の方法が考案されるに至った。適応メッシュ法のプロセスを図5・8に示す。最初に、形状データや境界条件値などを定義し、初期解析用のメッシュを生成する。続いて境界要素解析を行い、誤差評価(正確には、解はわかっていないので誤差予測)が許容範囲に入っていれば結果を出力する。条件を満足しない場合は、メッシュを更新して境界要素解析に戻る。このように、必要な精度の解が得られるまで反復計算が繰返される。このうち、主要なプロセスは誤差評価法、メッシュ更新、適応基準の三つであるので、以下ではこれらについてさらに説明する。a. 誤差評価法誤差評価法は、残差型、高次特異残差型、内挿誤差型、境界積分方程式型、節点設計感度型、その他に分類できる。残差型などは、有限要素法における研究成果が境界要素法にそのまま適用されたものであるが、有限要素法と境界要素法では定式化理論が異なるために、必ずしも適切とはいえなかった。そこで、後に他の方法が提案された。残差型では、数値解の誤差を境界積分方程式の残差ノルムと関連させる。ガラーキン法では誤差と残差の関係が証明されているが、境界要素法は選点法で定式化される場合が多いので、そのような場合、残差型は必ずしも適切でないが、実用上利用される。高次特異残差型では、通常の境界積分方程式を微分して、未知関数の導関数についての高次特異な積分方程式を得る。そして、これから誤差指標を定義する。内挿誤差型では、初期解析よりも高次の内挿関数を用いて高精度な解を予測し、両者の差で誤差を定義する。この方法は、他の方法に比べて、誤差評価にかかる計算コストが小さいことが特徴である。境界積分方程式型では、数値解が離散化される前の境界積分方程式を満足しないことに基づいて、誤差と残差についての積分関係式を、境界積分方程式から導出する。この関係式を離散化し、残差を何らかの方法で評価してから解くことにより、誤差分布を決定する。節点設計感度型では、境界節点の接線方向への移動に対する関数値の変化量によって誤差を評価する。初期の研究では二次要素を用いて解析を行い、関数値の変化量を数値微分的に求めていたが、後には、高次特異な積分方程式から直接評価する方法が提案されている。上記の分類に含まれない方法としては、解析対象の境界を均等に要素分割して解析を行った後、2倍の要素数で均等要素分割して再び解析を行い、それらの差で誤差を評価する。また、初期境界要素解析を行った後で、誤差評価を行う要素だけを二等分して再解析を行い、初期解析の結果と局所再解析の結果の差を誤差として定義する方法も提案されている。別の方法では、選点を境界上にとる通常の境界要素解析と、選点を解析領域の外側にとる境界要素解析の数値解の差を誤差として定義する。さらに、予測誤差を用いる代わりに、メッシュ変更が解に及ぼす影響の度合いを用いる方法もある。全く別の方法として、モデル問題における誤差の収束挙動を求めておき、それを別の問題にも適用できるものとして扱う提案がある。b. メッシュ更新法メッシュ更新法は、h法、p法、r法と名付けられる方法、その他に大別される。h法では要素の内挿関数は変化しないままで、要素数が増えることにより計算精度を改善する。p法では要素分割は一定のままで、要素の内挿関数の次数を上げることによって近似精度を改善する。また、r法では要素数や内挿関数の次数は一定のままで、要素配置を変更することによって数値解の精度を改善する。h法、r法では、メッシュ更新後に全体系行列を再計算すると計算コストが高くなるので、計算コストを下げるために、階層型内挿関数を用いた方法が提案されている。この方法では、実際に要素細分割を行う代わりに、初期解析に用いた内挿関数に新たに項を追加する。追加された内挿関数項についてだけ係数行列の再計算を行うので、計算コストを下げることができる。h法、p法、r法の利点と欠点を補完し合う結合解法として、hp法とhr法などが提案されている。hp法では、特異点に隣接する要素でh法を、それ以外の要素でp法を用いている。これは、正解の特異点の近傍を除けば、p法はh法よりも誤差収束特性がよい(約2倍の収束速度を示す)ことを考慮して提案された。これに対して、hr法では、最初にh法でおおよその初期メッシュを作成してから、r法によって再修正する。境界要素法では、適合要素を用いて解析を行う場合が多いが、問題の正解が特異性を有する問題では、適合要素では正確に関数分布を表現できないので、適合要素の代わりに特異要素を用いる方法が提案されている。h法、p法、r法とそれらの結合解法については、有限要素法における研究成果を利用しているのだが、階層型内挿関数やその追加方法などについて、境界要素法独自の提案がなされている。また、図5・8 適応メッシュ法のプロセス特異要素の利用は、境界要素法に特徴的な手法といえる。c. 適応基準適応基準の役割は、誤差評価結果を実際のメッシュ更新と関連付けることである。h法に対する適応基準としては、しきい値法、誤差収束特性法、平衡方程式法等がある。しきい値法では、評価基準の関数値についてしきい値を与えておき、関数値がしきい値より大きな要素を細分割する。しきい値としては、全要素での関数値の単純平均値や、最大値に1より小さなパラメータを掛けたものをとることが多い。この方法の応用として、複数のしきい値を用いることで、二等分割だけでなく三等分割などの、より多数の要素へ等分割する方法も考えられている。ガラーキン境界要素法では、数値解の誤差が要素長さの二乗に比例して収束することが示されている。この関係式から、希望の精度を実現する要素数を決定する方法が誤差収束特性法である。平衡方程式法では、弾性体に作用する力の釣合い関係式に基づいてメッシュを細分割する。階層型p法に対する適応基準としては、新たに追加される内挿関数に未知係数を掛けたものが、評価された誤差や積分方程式の残差と等しくなるようにして、未知係数値を決定する方法が用いられる。また、h法と同じく、誤差収束特性法に基づく方法も提案されている。この方法では、誤差が内挿関数の次数の逆数に比例すると仮定して関係式を導出している。r法では全誤差を最小化するように要素分割位置を移動するので、r法に対する適応基準は、最適化問題として定義される場合が多い。このとき、目的関数は系全体の数値誤差であり、要素分割位置が設計変数となる。上記の適応基準では、誤差の大きな要素をメッシュ更新すれば全体精度が改善されるという仮定に基づき、誤差の大きな要素を更新することで全体誤差を小さくしている。しかし、境界要素法の場合定式化に用いる基本解の特性から、各要素の誤差は他の要素の誤差に影響を与えるので、上記の仮定は必ずしも正しくない。そこで、誤差と影響関数の積として拡張誤差指標を定義し、それを誤差の代わりとして適応メッシュ法に用いることが提案されている。
