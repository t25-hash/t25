# 5・3・6 逆問題

> **章**: 第5章 構造解析のための境界要素法
> **節**: 5・3 各種問題への適用

原因を推定するもの、あるいは別の表現で、出力をもとに入力を求めるものである。逆問題解析においても計算の過程において有限要素法(FEM : finite element method)、境界要素法(BEM :boundary element method)、差分法(FDM : finite differencemethod)などの数値解析法が必要となる。数値解析法は取扱う逆問題によって異なるものの、入力を仮定して数値解析によって得られる解(出力)が、実験や計測などによって得られる測定データからの一次情報にできるだけ近づくように入力を繰返し修正したり、一次情報を用いて直接未知入力を計算する方法として利用される。ところで、計測などに基づく一次情報は境界値として得られる場合が多い。また、同定すべき対象が隠れて見えない部分や内部の欠陥のような境界形状であったり、測定できない部分の境界値であったりする場合もある。そこで、逆問題解析で用いる数値解析手法として境界要素法がよく適用される。12・7節でもいくつかの例が示されるが、ここでは境界要素法を逆問題に利用した例を以下に示す。a. 境界値逆問題金属に外部から電流を加え、防食するカソード防食(cathodic protection)での最適な電極配置と、電流の大きさを決定する問題への境界要素法の応用例がある。この問題では、ポテンシャル問題の境界要素法を利用できる。アノードとカソードの部分のポテンシャルと流束がともに未知(不完全規定境界)、他の部分は通常の境界条件が与えられている境界、ほかに内点数点のポテンシャル値を利用して、境界積分方程式と内点のポテンシャル値を計算するグリーン(Green)の公式を、それ6―69ぞれ離散化したものを連立し、不完全境界の未知量を直接求めている。安定な解を得るために、特異値分解(singular value decom-position)を利用して、連立方程式の係数行列の階数低下を行っている。音響問題では、音波の干渉を利用して、騒音源から放射される音を消すために必要な付加音源の振動状態を同定する問題への応用例がある。定常音響問題として定式化し、ヘルムホルツ(Hel-mholtz)方程式に対する境界要素法を利用している。付加音源の振動状態の仮定値を用い、境界要素法で計算された測定点の音圧の平方和が最小になるような振動状態のパラメータを、最適化手法を用いた反復計算により同定している。静弾性問題での例では、領域内部で得られる主応力和から境界の表面力分布を推定する問題に境界要素法が応用されている。主応力和の計算を境界要素法で定式化し、特異値分解による階数低下法で適切化を図っている。b. 境界形状同定問題高炉の寿命を延ばすために必要な、炉床部の耐火物の侵食状況を同定するために、境界要素法を応用した例がある。耐火物外表面の温度の実測値と、境界要素法を用いた定常熱伝導解析による計算値との残差平方和を最小化する形状を、侵食ラインとして同定している。また、構造物中のき裂を同定する問題へ、ポテンシャル問題の境界要素法を適用した例がある。これは電気ポテンシャルCT(computed tomography)法(electric potential CT method)と呼ばれる方法であり、ポテンシャル問題における境界積分方程式を未知数に対して逆に解いてき裂面での境界値を求め、得られた境界値の中で、流束が零となる部分をき裂の存在位置とする方法(未知境界同定法)と、マルチ電流負荷方式により得られた境界上のポテンシャル分布の計算値と測定値の残差を、最小化させる方法(境界残差最小化法)とがある。この問題で、き裂を一意に同定するために必要な電流負荷条件も示されている。ほかに、物体に高周波微小電流を流して表面上での相互インピーダンスの変化を測定し、内部に存在する不均質領域の境界を決定するインピーダンスプレチスモグラフィー(impedance plethysmo-graphy)のような応用例や、半導体素子の空乏層内の電位および電解強度を計算するための、空乏層の境界位置を求める問題への適用例もある。定常動弾性問題に対する適用例として、動弾性問題のひずみ応答を利用して、構造物に内在する欠陥の位置と大きさを同定する方法が示されている。この例では、定常動弾性問題に対する境界要素法が用いられている。境界上のいくつかの点で測定したひずみと、境界要素法の順解析で得られるひずみの残差平方和を最小にするように、欠陥の位置と大きさのパラメータを共役こう配法により計算している。複数の箇所を加振した場合の応答を観測値に利用することや、応答の感度が大きい箇所の加振点としての選定が有効であるとしている。同様の例として、静弾性問題において、弾性体内部の欠陥形状と位置を弾性体境界に作用する集中力による変位応答から同定する問題への適用例がある。解析には境界要素法を用い、欠陥を表現するパラメータ値の探索には拡張カルマンフィルタ(extendedKalman filter)が用いられている。c. 負荷逆問題拡散方程式の時間領域境界要素法を用いた応用例として、時間に依存して変化するソース項同定問題(source identification problem)への適用例がある。ソース項は既知関数の線形結合で近似し、観測値と計算値の残差平方和を最小化することにより、その係数を求めている。また、スカラ波動方程式の時間領域解法に関して、集中負荷の位置と強さおよび作用時間を同定した例がある。音響問題における負荷逆問題の例としては、騒音の低減に関連して、騒音源の位置と大きさを探査する問題がある。定常音響問題とし、ヘルムホルツ方程式に関する境界要素法を用いて、観測さ6―70計算力学れた音響インテンシティと計算値の残差平方和を最小化するように同定している。静弾性問題の負荷逆問題としては、トンネル掘削時の測定変位から地下の初期応力を同定するための適用例がある。内部の初期応力はソース項とし、地上の変形と関係付ける式に境界要素法を利用している。測定値が求めるべき初期応力の成分数より多いときは、最小二乗基準で初期応力を決定している。また、残留応力(residual stress)の同定問題に境界要素法を適用した例では、残留応力を物体力としてモデル化している。物体力の作用領域の大きさと位置を仮定し、静弾性問題の境界要素法で得られる代数方程式を直接解いて物体力の大きさを求め、それを基に残留応力の位置を同定している。弾性平板の静的曲げ問題への適用例では、平板の古典理論に基づく境界要素法を利用して、平板内のひずみ分布から平板の面外圧力分布を同定している。平板内のひずみは、平板内のたわみを関係づける積分公式を微分したものを利用し、面外圧力分布とひずみ分布を連立一次方程式の形に定式化して用いている。問題の適切化には、特異値分解、チホノフの適切化法(Tikhonov’s regulariza-tion method)を用いている。d. 材料特性逆問題材料が不均質な場合、境界要素法をそのまま適用することができないため、材料特性逆問題では、有限要素法などの領域型解析法を用いる場合が多い。しかし、均質な材料の材料特性を求めたい場合には、通常の境界要素法を用いることが可能である。この種の逆問題では例えば、弾性平板の振動応答から曲げ剛性などの材料特性を求めた例がある。境界要素法は、曲げ振動の支配微分方程式を、ラプラス変換した空間で求めた数値解を数値ラプラス逆変換して、時間依存の解を求める方法に基づき、拡張カルマンフィルタを用いて、材料特性パラメータを求めている。文献(1)
