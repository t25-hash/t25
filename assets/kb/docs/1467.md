# 6・1・2 個別要素設計

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・1 個別要素法

粒状体モデルにおける最大の特徴の一つが、粉・粒子が有している寸法・形状に関する分散性を、そのまま考慮できる点にある。初期要素として、図6・2に示すように、一様乱数発生アルゴリズムと実測した粒径・粒形分布を表現する統計分布を組合わせることによって、実測分布を帯びた粒要素を発生させ、それを所定の解析領域に、すでに配置した要素との空間干渉を考慮して配置することで、実際の粉・粒子の状態を再現できる。さらに、剛境界のみを仮定し、重力負荷あるいは振動負荷を与えることで、実際に伴う充てん操作も配慮した粉・粒子配置も実現できる。さらに、実際の粉・粒子の力学特性を測定する材料試験を計算機上で行うこともでき、発生した要素配置の物理的検証をして、実際の粉・粒子の物理特性を転写したモデルを構成できる。粉・粒子の形状設定は、そのミクロ構造を記述するうえで極めて重要である。通常は円形・球形である要素をだ円形・回転だ円体に拡張することは容易であり、また異形粒子に関しても、図6・3に示すように要素間剛性を剛な特性とすることで、アスペクト比の制御、複合粒子の創生が可能である。この要素を登録し、前述の要素発生・配置アルゴリズムを使用することで、粉・粒子の粒形分布を図6・1単一要素としての二次元円盤要素・三次元球形要素図6・2 任意の粒径分布(grain size distribution)の発生方法(a) 球要素の連結による表現(b) 球要素+棒要素による複合表現図6・3 異形粒子の構成図6・4 ソフトシェル-ハードコアモデル図6・5 計算機粘度試験法再現することもできる。粉末射出成形あるいはセラミック粉末プレス成形などでは、流動性・成形性を確保するために、有機系バインダを使用することが多い。この場合には、図6・1のモデルを修正して、バインダ層(粉・粒子の表面に付着し、潤滑効果など本来の粉・粒子の力学挙動を修正している要素)を考慮した粒状体モデルを構成する必要がある。図6・4にそのモデル化の一例として、ソフトシェル-ハードコアモデル(soft shell hard core model)を示す。バインダ層の弾粘性挙動をソフトシェルで、粉・粒子の弾塑性挙動はハードコアで考慮し、その他の粉・粒子に付着しないフリーバインダの流動抵抗も考えて、コンパウンド全体の流動変形挙動を解析できる。
