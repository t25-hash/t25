# 6・1・2 個別要素設計

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・1 個別要素法

粒状体モデルにおける最大の特徴の一つが，粉・粒子が有してい
る寸法・形状に関する分散性を，そのまま考慮できる点にある．初
期要素として，図6･2に示すように，一様乱数発生アルゴリズムと
実測した粒径・粒形分布を表現する統計分布を組合わせることによ
って，実測分布を帯びた粒要素を発生させ，それを所定の解析領域
に，すでに配置した要素との空間干渉を考慮して配置することで，
実際の粉・粒子の状態を再現できる．さらに，剛境界のみを仮定
し，重力負荷あるいは振動負荷を与えることで，実際に伴う充てん
操作も配慮した粉・粒子配置も実現できる．さらに，実際の粉・粒
子の力学特性を測定する材料試験を計算機上で行うこともでき，発
生した要素配置の物理的検証をして，実際の粉・粒子の物理特性を
転写したモデルを構成できる．
粉・粒子の形状設定は，そのミクロ構造を記述するうえで極めて
重要である．通常は円形・球形である要素をだ円形・回転だ円体に
拡張することは容易であり，また異形粒子に関しても，図6･3に示
すように要素間剛性を剛な特性とすることで，アスペクト比の制
御，複合粒子の創生が可能である．この要素を登録し，前述の要素
