# 6・1・3 材料プロセスシミュレーションへの展開

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・1 個別要素法

一例として、図6・5に示すような粘度試験(capillary viscom-eter)の計算機シミュレーションを行った。非ニュートン粘性体でも、チャネル内で発達し、一定のせん断ひずみ速度γに対する面応力(せん断応力)τを測定すれば、見掛け粘度(apparent visco-sity)νをν=τγで求められるというのが、この粘度測定法の原理である。そこで、実際の試験法と同様に、一定のストローク速度で粒状体モデルを圧下し、試験装置モデルのチャネル壁での反力・すべり速度を計算し、反力の単位面積平均をせん断応力として、見掛け粘性を求めた。実験結果と粒状体モデルによる解析値とを比較すると、図6・6において両者は良好に一致し、図6・4に示すモデルが実際の粉末コンパウンドの粘性挙動を、定量的に再現できることがわかった。興味深い点は、図6・4のモデルによって、容器壁でのすべり速度の定量的な予測に加えて、流動速度の増加に伴うチャネル内の速度分布を精度良く記述できることであり、粉体・粒状体のレオロジー(rheology)現象を解明するうえでも、粉末の離散挙6―73図6・6 見掛け粘度のせん断ひずみ速度依存性(c) 均質化モデル(平均距離が大きい場合)(a) 粒状体モデル(b) 均質化モデル(平均距離が小さい場合)図6・7 構造平均化の考え方図6・8 セラミック粉末射出成形シミュレーションへの応用動を記述できる粒状体モデルが極めて有効であることがわかる。実際、種々の粉末射出成形プロセス(powder injection molding)・コロイドプロセスなどにおいて、その有用性が検証されている。
