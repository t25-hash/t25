# 12・1・1 エキスパートシステムの定義

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・1 エキスパートシステム

エキスパートシステム（ES:expert system）は，人工知能の研
究の中で発見，開発されてきた理論と技術を基礎として，現実問題
への適用を指向するシステム開発技術である ．これには，二つの
定義付けが可能である．第1の定義は，専門家の代わりをするとい
うシステムの機能面を強調する．これはシステム利用者の立場に立
つものである．第2の定義は，従来の手続き型システムにはない，
る ．
システム構成上の特徴を強調する．これはシステム開発者の立場に
立つものである．
第1の定義は次のとおりである．「エキスパートシステムとは人
間の知性を用いなければ解けないような特定分野の問題を解決する
のに，専門家の経験的知識（heuristics ）をコンピュータに組込ん
で，人間に代わってあるいは人間を支援して問題解決を遂行するこ
とを目的としたシステムである．」この定義は，専門家の知識が重
ようになる．
要な役割を果たすという，知識工学の考え方に基づいている．
第2の定義は次のようなものである．「エキスパートシステムと
は，問題解決に知識を陽に利用するシステムであり，知識とそれを
使うための仕組みとを，それぞれ知識ベースと推論機構として分離
し，独立性の強い二つの要素からシステムを構成するソフトウェア
システムである．」この意味で，エキスパートシステムを知識ベー
スシステム（knowledge-based system），あるいは知識システム
（knowledge system）と呼ぶ ．
始されたDENDRALである．これは，未知化合物の質量スペクト
ルデータと分子式から，その化合物の構造式を推定する解釈型シス
テムであった．やはり1960年代に開発が始まった数式処理システ
ムMACSYMAは，知識ベースシステムではないが第1の定義に
従うエキスパートシステムであり，実用的なシステムとして有用で
あった．
CASNET，Internist等が続けて発表された．そして，これらのシ
ステムで共通に使われたプロダクションシステム（production sys-
tem）の手法が，鉱物資源探査支援システムPROSPECTORにも
適用可能なことが明確になった時点で，エキスパートシステム技術
を中心とする，AI （artificial intelligence，人工知能）ブームが訪
れた．
現手段として，あるいは知識表現技術として成熟したのが，言語
OPSファミリーやCLIPSなどに代表されるプロダクションシステ
ム技術，フレーム（frame）やスキーマ（schema）に代表される
オブジェクト技術，そして，Prologなどに代表されるロジックプ
ンタフェースの役割は重要である．
