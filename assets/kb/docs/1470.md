# 6・2・1 格子欠陥動力学

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・2 離散転位動力学法

原子/分子が並進対称性をもって、規則正しく並んだ完全結晶(perfect crystal)に対して、幾何学的な格子のずれが存在する場合、この近傍を格子欠陥(lattice defect)という。一般的には、空孔(vacancy)、格子間原子(interstitial atom)と、それらの集合体(complex)、転位(dislocation)、回位(disclination)、積層欠陥(stacking fault)などを指すが、結晶粒界(grain bound-ary)、相境界(phase boundary)、界面(interface)、表面(sur-face)、き裂(crack)などのマクロ欠陥も、広義の格子欠陥とみなすことができる。連続体モデルでは、格子欠陥はその周辺場のゆがみを生むものとしてモデル化され、狭義には、特異点、特異線、または特異面とみなすことができる。これらの格子欠陥を、連続媒体中を運動する実体のある点、線、あるいは面としてモデル化し、格子欠陥を含む変形体の特性を解析する方法を、離散欠陥動力学法(discrete defectdynamics method)と呼ぶ。
