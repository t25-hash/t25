# 6・2・2 離散転位動力学

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・2 離散転位動力学法

転位の運動は、塑性変形の主要メカニズムの一つであり、結晶体の非弾性挙動を考察するうえで、転位の果たす役割は極めて重要である。転位運動を力学問題として取扱う方法には、さまざまなモデル化が考えられ、なかでも原子モデルを用いた直接シミュレーションは、最もミクロな解析として多くの示唆に富む情報をもたらす。しかしながら、計算量の限界から、対象となる問題の時間・空間スケールは限定されたものになる。この困難を克服すべく、種々のハイブリッド法や準連続体モデル(quasi-continuum model)による、大幅な自由度低減のための手法が有望視されているが、多数の欠陥構造を含む系の変形や変形こう配の大きい場を扱う問題では、自由度の増大によってその利点が損なわれる。これに対し、転位心のごく近傍を除いて、力学場・変位場は(線形)弾性論と極めてよく対応するという、多くの原子レベルの結果の支持を受け、転位を特異線としてとらえて、個々の転位の運動を弾性体中の特異線の運動として扱う方法が提案されている。ここでは、このような個別の転位運動を用いる解析手法を総称して、離散転位動力学法(discrete dislocation dynamics method)と呼ぶ。
