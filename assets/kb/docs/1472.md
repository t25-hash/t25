# 12・1・4 エキスパートシステムの主要概念

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・1 エキスパートシステム

エキスパートシステムの開発に使われる諸概念は，基本的には人
工知能のそれと等しい．すなわち，エキスパートシステムにおいて
は，知識を用いて対象となる問題空間を表現し，その空間を探索す
ることで解を導く．通常，エキスパートシステムで使われる知識表
現と推論方式は，ルール（rule ）とフレームの概念に基づくもので
ある．
a.ルールを用いた知識表現と推論方法 専門知識を記述する
手段として，If -Then形式を用いるプロダクションシステムの手法
は，広く利用されてきた．プロダクションシステムの優れたところ
は，ルールの記述性と理解性にある．これをうまく利用したのが帰
多く，非単調推論に直接関連する研究課題となっている．
納推論で得た分類知識をルール群，もしくはそれに相当する決定木
