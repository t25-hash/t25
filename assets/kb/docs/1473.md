# 6・2・4 離散転位動力学法

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・2 離散転位動力学法

a. 種々の方法。 格子法(lattice method)空間を細胞状の多数の要素から成るものとしてモデル化し、その個々の要素が有限個の状態をもち、与えられた局所ルールによってそれらの状態を変化させることのできるものを、セルオートマトン(CA : cellular automa-6―75ton)と呼ぶ。CA では、各要素の状態の更新は離散的な時間の進行とともに、すべての要素で同時に行われ、ある時間ステップでの近傍要素の状態によって、次の時間ステップでの状態が決定される。この方法を利用した転位動力学では、転位線を隣接する要素が作るパターンと考え、要素のとりうる状態を無転位、らせん転位の線素、または刃状転位の線素のいずれかとし、この状態変化でもって、要素間を不連続に移動する転位構造の、時間発展を追跡する方法を基本とする。混合転位を状態に加えたり、さまざまな改良版が存在する。. 転位の近接相互作用のモデル化離散転位動力学では、抵抗則(せん断応力と転位速度の関係)、転位を動かすのに必要なパイエルス力、慣性を考える場合に、転位の質量や、転位心の寸法き裂や表面からの転位発生基準などの、転位固有の未知の情報を必要とする。また、複数の転位の近接相互作用、転位と不純物、転位と粒界との相互作用の記述が必要になる。このようないわゆる構成則は、シミュレーションの結果の良否に密接に関係している。従来、実験結果、あるいは経験的知見を基に導入されてきたが、原子レベルのシミュレーションからの情報を使う方法も検討されている。
