# 6・2・4 離散転位動力学法

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・2 離散転位動力学法

a. 種々の方法。格子法(lattice method)空間を細胞状の多数の要素から成るものとしてモデル化し、その個々の要素が有限個の状態をもち、与えられた局所ルールによってそれらの状態を変化させることのできるものを、セルオートマトン(CA : cellular automa-6―75ton)と呼ぶ。CA では、各要素の状態の更新は離散的な時間の進行とともに、すべての要素で同時に行われ、ある時間ステップでの近傍要素の状態によって、次の時間ステップでの状態が決定される。この方法を利用した転位動力学では、転位線を隣接する要素が作るパターンと考え、要素のとりうる状態を無転位、らせん転位の線素、または刃状転位の線素のいずれかとし、この状態変化でもって、要素間を不連続に移動する転位構造の、時間発展を追跡する方法を基本とする。混合転位を状態に加えたり、さまざまな改良版が存在する。. 剛要素法(rigid segment method)転位線を直線状の線素から成るものとしてモデル化する。個々の線素は、混合転位の性質を有しており、線素の法線方向に剛体的に移動する。この方法は、弾性場に対する情報を必要とせず、線素間の相互作用力が公式(6・7)によって直接計算される。この時間発展を追跡することにより転位線の軌跡を得る。転位線に働く力は、式(6・7)や(6・11)を使うことにより、式(6・12)で評価される。なお、ゴニエム(Ghoniem)らによるパラメトリック曲線モデルによって、効率良く計算する方法が提案されている。このとき、微分応力場は以下のように記述される。式(6・20)を一次元の求積法によって数値積分することにより、応力場が評価される。. フェーズフィールド法(phase field method)母相となる弾性体に任意の、弾性的に均質な系のひずみ場に対する、カチャトゥリャン・シャタロフ(Khachaturyan-Shatalov)の逆格子空間論(reciprocal space theory)に基礎をおく方法(KS 理論ともいう)であり、マルテンサイト変態に用いられるフェーズフィールド法であるPFM(phase field microelasticity)の考えを、転位動力学に応用した方法について述べる。転位の重なりを表現する化学的自由エネルギーを意味し、式中の被積分関数は、周期的なパイエルスポテンシャルを模擬している。最後に、こう配エネルギーを次式で表現する。フェーズフィールド法は、個々の転位線を離散的に区別して取扱う方法ではないが、問題のスケールに応じた適切なパラメータの選択によって、また生成、対消滅などをあらわに考慮することなしに、転位の離散的な描像を獲得することができる。b. 境界値問題(boundary value problem)としての転位動力学簡単のため、二次元の例について示す。速度、ひずみ速度、応力速度場は図6・11に示すように、二つの場の重ね合わせで表現される。三次元問題の場合、転位ループが自由表面によって切断されるような場合などの取扱いに、別途工夫が必要となるが、基礎となる考え方は類似している。c. 転位の近接相互作用のモデル化離散転位動力学では、抵抗則(せん断応力と転位速度の関係)、転位を動かすのに必要なパイエルス力、慣性を考える場合に、転位の質量や、転位心の寸法き裂や表面からの転位発生基準などの、転位固有の未知の情報を必要とする。また、複数の転位の近接相互作用、転位と不純物、転位と粒界との相互作用の記述が必要になる。このようないわゆる構成則は、シミュレーションの結果の良否に密接に関係している。従来、実験結果、あるいは経験的知見を基に導入されてきたが、原子レベルのシミュレーションからの情報を使う方法も検討されている。
