# 6・2・5 補足事項

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・2 離散転位動力学法

離散転位動力学は、メゾスコピック領域を扱う手法として発展途上にあり、現時点で力学理論としてほぼ確立している事項について、ここでは、等方線形弾性体の微小変形論の枠組内での、古典論的取扱いについて概説した。異方性体の場合は、解析解の表現が難しくなる点を除いて原理的に同じであり、また、特異性を示さない解や非局所理論による解も、原理的にはここで述べたシミュレーションに適用可能と考えられる。有限変形については現在のところ簡便な方法は示されていない。また、転位の非古典的運動、例えばトンネル効果による転位運動や相対論的効果については、いくつかの議論があり、特別なケースについて検討された例があるが、いまのところ一般的な方法論は確立されていない。PU(partition ofunity)有限要素法の離散転位動力学法への応用などの新しい手法も提案されており、今後の発展が待たれる。数値計算の主要部に(擬似)乱数を利用する方法を、一般にモンテカルロ法(Monte Carlo method、以下MC法と略記)という。メトロポリス法(Metropolis method)はその一種であり、確率過程の性質を利用して統計力学的な平衡状態を作り出し、対象系の性質(構造、熱力学量など)を調べる方法であり、分子動力学法と並んで、原子・分子シミュレーションの標準的な手法の一つである。分子動力学法とは異なり、系の時間的な性質は原理的に調べることが困難であるが、いろいろな統計集団での平衡状態を容易に作ることができ、自由エネルギー計算などにも優れている。本節では、古図6・11 境界値問題の分解典統計力学におけるメトロポリス法の原理を略述し、簡単な適用例や応用への道筋を示す。なお、量子MC法についても基礎的な考え方は同じであるが、実際上の取扱いがかなり異なるので、本節では触れない。
