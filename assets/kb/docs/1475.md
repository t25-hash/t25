# 6・3・1 統計平均の考え方

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・3 メトロポリス法

統計力学では、対象系を記述する物理量として、原子・分子レベルの詳細な情報(微視的変数)と、それらを何らかの形で統計平均して得られる情報(巨視的変数)とを、区別して考える。各原子の瞬間的な座標や速度は微視的変数であり、内部エネルギー、温度、圧力などの熱力学量は巨視的変数である。物質の構造を表す動径分布関数などの相関関数も、統計平均によって得られるので、後者に分類される。統計平均のとり方には、二とおりある。平衡状態にある一つの系の微視的変数を長い時間にわたって平均する時間平均(time aver-age)と、巨視的に同じ状態にある多くの系についてその微視的変数を平均するアンサンブル平均(ensemble average、集団平均)である。ここで述べるメトロポリス法は、ある統計分布に従って、平衡状態にある系を計算機内に数多く作り出すことによって、集団平均を求めるという手法である。なお、ここでは記述を簡単にするために自由度と書いたが、実際には原子・分子の座標や運動量など、すべての微視的変数を表す。この場合、式(6・37)は一般に、非常に多次元の位相空間での積分となるため、通常の数値積分法を使うことができない。これが、メトロポリス法などの技法を用いる理由である。
