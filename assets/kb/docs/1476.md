# 6・3・2 メトロポリス法の原理

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・3 メトロポリス法

現在の状態から、乱数によって次の状態をある確率で作る。その確率をうまく選ぶことで、平衡状態を作ろうというのがメトロポリス法である。最も単純な例として、状態1と状態2をある確率で行き来する状況を考える。は遷移行列(transition matrix)と呼ばれる。このように、次の状態(+1ステップめ)が現在の状態(ステップめ)だけに依存し、過去にはよらないような確率過程を、マルコフ過程6―77(Markov process)という。ステップ数が大きいときには、どんな初期状態から出発しても定常状態(平衡状態)に近づくことが、経験的に期待できる。これは、についていくつかの仮定を設けることで数学的に証明することもできる。平衡状態を具体的に求めてみる。逆に、平衡状態∞がわかっているとき、式(6・45)となるようにを選べば、任意の初期状態から出発して、目的の平衡状態を得ることができる。式(6・45)からわかるように、との比だけが重要なので、,の選び方は式(6・47)以外にも可能であるが、式(6・47)のように選ぶ方法をメトロポリス法という。以上の原理は、ただちに個の状態がある場合に拡張できる。カノニカルアンサンブルにおけるメトロポリス法の実際の手順は、次のようなものである。(1) 初期状態としては何を選んでもよい。ただし、平衡状態になるべく近いところから出発するほうが収束は早い。(2) いまの状態から、一様乱数(uniform random number)によって次の状態の候補を作る。(4) 式(6・48)に従って遷移させる。が採用されなかった場合は、元の状態のままとする。(5) 十分な回数だけ、手順(2)~(4)を繰返す。このようにして、十分な状態遷移の後に、温度の熱平衡状態が実現されることになる。エネルギーや原子の配置などをモニタし6―78計算力学て、十分に平衡状態(定常状態)に近づいたと判断した後は、それ以降に生成された状態を用いて、式(6・38)に従い集団平均をとることができる。原子・分子系では、粒子の質量中心や分子配向を連続変数として扱うのがふつうである。このように状態が離散的でない場合も、上述の原理や手順は全く同じである。
