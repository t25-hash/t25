# 6・3・4 他の統計集団への拡張

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・3 メトロポリス法

カノニカルアンサンブル(集団)についての手法は、容易に他の統計集団に拡張することができる。よく用いられる例として、グランドカノニカルアンサンブル(大正準集団:温度、体積、化学ポテンシャルμが一定)を作り出す方法を述べる。この場合は粒子数が増減するので、個の粒子が位相空間中で、ある配置をとる確率;を考えることになる。;=1Ξexp -;-μ(6・49)ここで、Ξは規格化因子、すなわち大分配関数である。グランドカノニカルモンテカルロ法(GCMC: grand canonical MonteCarlo method)では、次の三つの操作を行う。(1) 粒子の移動:カノニカルモンテカルロ法の場合と同じである。(2) 粒子の挿入:個の粒子の系に、さらに一つの粒子をランダムな場所に挿入する。このときのポテンシャルエネルギー変化をとすると、式(6・49)より+1=+1Λexp --μ(6・50)となるので、+1の確率で新しい系(+1粒子系)を採用する。ここで、Λ≡2π(:プランク定数、:粒子質量)は、エネルギーのうち運動エネルギー部分の積分に由来するド・ブロイ波長である。(3) 粒子の削除:個の粒子のうち一つをランダムに選んで、系から取り除く。このときのポテンシャルエネルギー変化をとすると、同様に-1=Λexp -+μ(6・51)となり、この確率で新しい系(-1粒子系)を採用する。これら三とおりの操作(試行)を等しい確率で行うことにより、グランドカノニカルアンサンブルを作り出すことができる。ただし、初期条件が平衡から大きくずれている場合などには、粒子の移動はもう少し多い頻度で試みるほうが、収束は早いこともある。同様の考え方により、温度・圧力一定の集団を作ることもできる。この場合は、体積も変数としてランダムに変化させることになる。少し変わった手法として、相平衡状態にある集団を作り出す、ギブスアンサンブルモンテカルロ(GEMC: Gibbs ensemble MonteCarlo)法が開発されている。これは、系を二つ用意して、それらの間で粒子交換と体積の増減を行わせる(粒子数の和と総体積は一定)ことにより、化学ポテンシャルを推定することなく、二相共存状態を作ることのできる便利な方法である。
