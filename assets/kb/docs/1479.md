# 6・3・5 自由エネルギー(free energy)の計算

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・3 メトロポリス法

モンテカルロ法の長所の一つは、他の方法では精度良く求めることの難しい自由エネルギーなどの熱力学量を、比較的手軽に計算できるという点である。内部エネルギーや圧力などに比べて自由エネルギーの計算が難しいのは、自由エネルギーが、式(6・38)のような集団平均の形で表現できないためである。そこで、いくつかの手法が開発された。いずれの手法も、求めたいものが、ある巨視的状態での自由エネルギーの絶対値ではなく、二つの異なる状態間の自由エネルギー差であることに着目したものである。ここでは代表的な手法を三つ紹介する。a. パラメータ積分法(parameter integral method)ハミルトン関数に人工的にパラメータを含ませる。例えば、系内のある原子種を別の原子種で置換したときの自由エネルギー変化を求めるためには、λ=0を元の原子種(状態I)に、λ=1を置換後の原子種(状態II)に対応させる。ハミルトン関数を、両状態の線形結合で表現する。そこで、0≦λ≦1のいくつかの点でMC計算を行い、滑らかに内挿して数値積分することにより、自由エネルギー差を求めることができる。ただし、精度良く自由エネルギーを求めるためには、とがある程度近く、状態の出現確率密度の重なりが大きいことが必要である。この重なりが小さい場合には、いくつかの中間温度を経由すればよい。第2項は、粒子の系に外からランダムに粒子を一つ挿入したときの、ポテンシャルエネルギーの変化のボルツマン因子を統計平均することを表しており、メトロポリス法によって計算することが可能である。実際には、密度の高い凝縮相では挿入した粒子はほとんどの場合に、既存の粒子と重なり合うため、非常に大きなエネルギー増加となり、統計平均の精度が落ちる。こうした場合は、粒子を削除する方法や、パラメータ積分法との組合せなどが試みられている。
