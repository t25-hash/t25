# 6・4・1 分子動力学法の原理

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・4 分子動力学法

分子動力学(MD : molecular dynamics)法は、原子間相互作用力を決定するポテンシャル場のもとで、原子1個1個についてニュートンの運動方程式を数値解析的に解き、多数の粒子系(多体系)の時々刻々と変化する動きを決定論的に計算する手法である。三次元的に周期性をもった結晶構造のみならず、表面や界面、クラスタといった不均質な構造体についても、原子レベルからの熱力学的な安定構造(原子配置)を求めることができる。さらに、環境温度の変化や外部負荷が作用する熱・力学場でのダイナミクス(時間応答)を求めることで、原子レベルからの材料物性や材料強度の評価を行うことも可能である。熱力学な平衡状態における原子の位置や速度を、統計力学的に処理することにより、表6・1のような構造解析、熱力学的あるいは材料的特性値、分光学的性質等が推定される。a. 原子の運動方程式ニュートンの第2法則(運動の法則)は、運動量の時間変化が、その質点に作用している力に等しいという関係を記述したものである。一般に式(6・61)がよく知られた運動方程式であるが、位置と運動量の正準変数を用いた式(6・60)のような運動方程式は、時間に関して1階の連立常微分方程式となり、階数を減じられることで数値計算には適している。また、式(6・60)の第1式および(6・61)の右辺の保存力(conservative force)は、原子間相互作用を表すスカラポテンシャルΦの空間こう配から、次式のように求められる。b. 統計集団(アンサンブル)分子動力学法では、原子1個1個の運動方程式を逐次数値積分するので、その取扱える原子数は計算機の能力に強く依存する。並列計算機の飛躍的な性能向上により、現在約1億個(10個)以上の原子数を用いた解析も行われている。しかしながら、現実の物質は、アボガドロ数(6.02×10)に代表されるオーダの原子数が必要であり、解析では限られた小さな有限サイズの系を取扱うことが問題となってしまう。そこで、現実にある物質の一部を取出し、基本セルと呼ばれる箱に入れる。そして、その箱が3方向に繰返し存在する仮想セルにより空間が埋め尽くされるモデルを考え、無限に広がる物質の構造を再現することになる。そのときの運動方程式は、式(6・60)あるいは(6・61)のニュートンの運動方程式になる。詳細な運動方程式は、多くの分子動力学法に関する書物に記載されているので、参考にするとよい。アンサンブルに応じた運動方程式を数値積分し、平衡状態の長時間平均を求めることで、アンサンブル平均(ensembleaverage)に等しいとするエルゴード性(ergodic hypothesis)により、統計力学に基づく巨視的な諸量が求められる。c. 数値積分法固体系の場合、式(6・60)(6・61)の時間の刻み幅は、対象とする材料の格子振動数(フォノン)を再現できる程度にとられる。したがって、一般にフォノン分散は数THz(テラヘルツ:10Hz)のオーダなので、その周期の10分の1以下程度、すなわちfs(フェムト秒:10s)オーダの時間の刻みで解析される。数値積分法としては、オイラー法、ルンゲ・クッタ法、アダムス法等が古くから提案されているが、分子動力学計算の数値積分法としてはベルレ法、ギア法がよく用いられる。ベルレ(Verlet)法は、アンサンブルの数値積分法としてよく用いられる。したがって、絶対値の小さな項がけた落ちしてその誤差が累積したり、速度の計算精度が悪いという欠点がある。そこで、それを補う方法として、次式のホックニー(Hockney)らによるかえる跳び(leap-frog)法がある。これらの非線形常微分方程式では、次に述べるギア法(Gear method)がよく用いられる。原子の軌跡は時間に関する階微分連続関数として表されるものとする。時刻+における位置、速度、加速度などの諸量は、時刻に関してテイラー展開ができ、次式のようになる。また、以降特に断らない限りは、原子の識別記号αは省略する。式(6・67)に含まれる各諸量から構成される組を、次式のように定義する。したがって、この差を予測子に加えることで、修正された諸量(修正子)が得られる。そこで、式(6・68)の代わりに-表式と呼ばれている次式を用いる方法もある。一般にギア法は、精度が良いが時間ステップを大きくとれない。ベルレ法は加速度や速度を厳密に評価できないが、記憶容量が少なく、時間ステップを大きくとっても安定といわれている。用いる運動方程式の形式や解析条件に応じて選択することになるが、アルゴリズムが比較的簡単で、プログラム化しやすいベルレ法がよく用いられている。d. 初期条件と境界条件
