# 6・4・1 ・a項で示した運動方程式の初

> **節**: 6・4 分子動力学法

初期配置の位置は解析の対象により異なり、周期性をもつ結晶構造体の場合には、基本並進ベクトルが与えられて初期配置が決定される。初期速度は、解析条件としての系の環境温度により決定される。正規乱数の発生には、通常コンピュータで常備されている数値計算ライブラリのサブルーチンを用いればよい。この速度から求まる系の重心の速度は、一般には零ではない。系を静止状態に保つため、を次式のように補正したものを初期速度として設定する。熱平衡状態が実現するまでには、運動エネルギーとポテンシャルエネルギーとの間でエネルギーのやりとりがあり、原子の初期配置によっては、目的の温度を実現させるために温度を調節しなければならない。また、計算誤差に伴い全エネルギーが保存されない場合にも、系の温度が一定に保持できなくなる。次に境界条件について述べる。6・4・1・b項で述べたように、限られた原子数で無限系を取扱う方法として、周期境界条件(periodicboundary condition)がある。周期境界条件では、原子は壁から力を受けることなくそのまま容器の外に出てしまい、同時に対面の壁の相対する位置から、同じ速度で原子が入ってくるとする。この運動を二次元平面的に示したのが図6・15である。中央の直接的な解析対象となる立方体を基本セル(fundamental cell)といい、その前後、左右、上下に積み重ねた基本セル以外のセルを仮想セル(image cell)という。図6・15のように、基本セル内の原子の運動と全く同じ運動が、すべての仮想セルでも繰返される。周期境界条件を用いることにより、限られた有限個を含む系であっても無限系の解析対象とみなすことができ、アンサンブル平均により、統計力学に基づく諸量の推定が可能になる。周期境界条件のもとで運動方程式を数値積分する際には、原子に働く力の計算において基本セル6―82計算力学図6・15 周期境界条件図6・16 粒子登録法と仮想セルの区別なく、力の影響範囲(カットオフ距離)以内のすべての原子からの力を含めなければならない。また、基本セル内の原子数は、つねに一定に保つような処理を施さなければならないことに注意する。粒界や表面といった面欠陥を解析対象にする場合には、粒界が周期的に並んだり、原子の存在しない真空層を設定することで、周期境界条件の取扱いができる。この方法をスーパーセル法という。この際、周期的に並ぶ粒界や表面の相互作用が十分小さくなる程度に、面欠陥の法線方向のセル寸法を大きくとることに注意する必要がある。e. 粒子登録法粒子登録法(bookkeeping method)は、ベルレが最初に導入した計算時間短縮の方法であり、現在多くの分子動力学プログラムで用いられている。したがって、取扱う原子数が大きくなるほど計算時間は短縮されることになる。
