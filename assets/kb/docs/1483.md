# 6・4・2 原子間ポテンシャル

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・4 分子動力学法

原子を時々刻々と変化させる駆動力は、式(6・62)の原子間相互作用力である。したがって、それを決定する原子間ポテンシャルの選定には、最も注意を払わなければならない。分子動力学法で用いられる多くの原子間ポテンシャルは、量子力学に基づく厳密な電子状態から決定されるのではなく、ポテンシャルを微分可能な未定係数を含む簡単な関数形で仮定し、従来の実験的事実(マクロな物理的特性値)や、量子力学に基づく第一原理計算結果に合致するように、その未定係数が決められる経験的ポテンシャル(empirical interatomic potential)である。ポテンシャルが2個の原子間の距離のみを独立変数として定義されるものを、2体間ポテンシャルといい、ほかに3体間、多体間がある。a. 2体間ポテンシャル2体間ポテンシャルの全エネルギー읏윱읏は、次式で与えられる。읏윱읏=12 ∑욏Φ(6・74)ここで、Φはポテンシャルを記述する適当な関数、は原子αとβ間の距離で、次式で表される。=-(6・75)総和∑は、カットオフ距離内にある登録された全原子を対象とし、任意な原子αの位置ベクトルに関する읏윱읏の1階および2階微分は、次式で与えられる。옿읏윱읏옿욡=-∑옿Φ옿욡(6・76)옿읏윱읏옿욡옿욢=-δ욡욢옿Φ옿-욡욢옿Φ옿-1옿Φ옿(6・77)ここで、δ욡욢はクロネッカのデルタ記号である。式(6・74)の関数Φはこれまでに種々提案されており、代表的なものに、金属全般によく用いられているモースポテンシャル(Morse potential)、α鉄によく用いられるジョンソンポテンシャル(Johnson potential)、液相やアモルファス金属等によく用いられる、レナード・ジョーンズポテンシャル(Lennard-Jonespotential)などがある。b. 原子埋込み法による多体ポテンシャル2体間ポテンシャルでは、異方性の弾性定数に関するコーシーの関係=が成立して、3個の独立な弾性定数をもつ、一般的な立方晶の異方性が表現できない。そこで、量子力学に基づいた関数形で定義されるが、そのポテンシャルパラメータは、平衡状態でのマクロな物性値や、あるいは第一原理計算により求められた値に対してフィッティングされる半経験的ポテンシャルが、現在多くの解析に用いられている。一般的には、原子埋込み法(EAM : embedded atommethod)と呼ばれている。一般に、全エネルギーは次式のように表される。읏윱읏=∑ρ+∑∑(6・78)ここで、ρは電子密度ρをもつ位置にある、原子αの埋込みエネルギーを表す。この電子密度ρは、その原子αに隣接する原子群βの電子密度の線形結合として近似される。は原子間の中心斥力を表す。具体的な関数形と各種材料に対してフィッティングされたポテンシャルパラメータは、文献(29)を参照するとよい。EAM と同様な全エネルギーの記述を行うものに、フィニス・シンクレア(FS : Finnis-Sinclair)ポテンシャルがある。FS ポテンシャルは、鉄などの遷移金属と、銅などの貴金属に対するものがあり、フィッティングするための関数形が異なっているが、ポテンシャル構築方法はほぼ同様である。c. 角度依存性を考慮した半経験的ポテンシャル無欠陥な立方晶のように、あまり異方性が顕著でなく、対称性の高い周期構造をもつ金属については、中心間距離のみで表されるポテンシャルが適用できる。しかしながら、共有結合性、イオン結合性の物質などでは、だけの関数表示では正確にその安定構造を得ることができない。シリコン、グラファイトや高分子材料などはその典型的な例であり、その構造を記述するには、3体間の角度依存性を考慮する必要がある。工業的に重要なシリコンの解析に、現在最もよく用いられているのがターソフ(Tersoff)ポテンシャルである。表面再配列に着目してフィッティングされたものと、弾性特性を反映したものとが提案されている。ターソフポテンシャルの拡張系として、シリコン-水素系の結合や炭素-水素結合系を再現するものもある。その他、b項のEAM に電子密度の高次の修正項を導入し、角度依存性を考慮して修正された原子埋込み法(MEAM : modified EAM)が、種々の材料に対して提案されている。
