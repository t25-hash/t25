# 6・4・3 各種物理量の推定

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・4 分子動力学法

初期値問題としてのニュートンの運動方程式を解くことによって、任意な原子における位置と運動量(速度)の時系列データが求められる。その膨大なデータにより、位置の幾何情報からは結晶やアモルファスといった構造解析、統計力学的考察からは比熱や拡散係数といった物理量、そして連続体力学的考察からは弾性定数などの機械的特性値が推定される。以下に代表的な諸量を取上げるが、詳細は文献(20)~(22)(25)および(37)を参考にするとよい。a. 構造解析。 動径分布関数結晶構造の場合、熱平衡状態における原子の存在確率は格子点上で最大になり、それから離れるに従い急速に減少する。ある任意な原子から見た周囲の原子の存在確率を示したのが動径分布関数(radial distribution function)で、分布のピーク値をもつ距離から第一隣接原子や第二隣接原子を同定し、結晶構造の構造解析が行われる。動径分布関数の適用例として、アモルファスの構造解析や融解現象の解析に用いられている。距離の幅は、の第一ピーク近傍の変化がよくわかる程度に小さくとる必要があること、一辺の立方体モデルを対象とした場合には、周期条件のために動径を<2の範囲としなければならないことに注意する。. 自由エネルギーおよびエントロピー熱力学量としては最も基本的な量であるが、ポテンシャルエネルギーや温度等の量に比べて、計算が容易ではない。分子動力学法やモンテカルロ法を用いた自由エネルギーの計算手法としては、大別してアンブレラサンプリング(Umbrella sampling)法と摂動法の二つの方法がこれまでに提案されている。アンブレラサンプリング法は、ポテンシャルエネルギーの動径分布を求めることにより、自由エネルギーの変化を計算する。一方摂動法は、二つの系のエネルギーの差によって定義される関数のアンサンブル平均を求めることにより、自由エネルギーの変化を計算する。MD 法を用いて摂動法による自由エネルギーの計算を行う場合には、一般には比較的小さな摂動の解析に適用される。. フォノン分散固体の格子振動特性を求めるために、時間相関関数(time correlation function)と、その自己時間相関関数がパワースペクトルとフーリエ変換の関係にあることを示した、ウィーナー・ヒンチン(Wiener-Khinchin)の定理を用いる必要がある。解析セル内に位置する各原子の速度を振幅とする平面波を考え、その全原子についての線形結合がセル全体の振動を表すと仮定する。その速度に関する自己相関関数を求め、ウィーナー・ヒンチンの定理に基づき、そのフーリエ変換を行うことによって、フォノンのスペクトル密度が得られる。.
