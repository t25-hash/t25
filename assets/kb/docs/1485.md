# 6・5・1 第一原理計算(first-principles calculation)の基礎

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・5 第一原理解析

a. はじめに固体材料の強度や変形、破壊を支配するのは、究極的には原子間結合の担い手である電子である。第一原理計算とは、量子力学と統計力学の理論のみにより、材料中の電子や原子の挙動を高精度に再現する計算である。分子を扱う量子化学の理論や手法もあるが、結晶や固体材料では、固体物理で発展したバンド計算法が用いられる。本節では、後者の概要を説明する。バンド計算は、結晶のように周期性のある系の電子構造計算法である。電子構造に基づき、全エネルギー(凝集エネルギー)、安定原子配列、弾性定数、原子に働く力、応力等、さまざまな物理量が計算できる。欠陥、表面・界面、アモルファスなども、大きなセル6―84計算力学(スーパーセル)が繰返す構造として扱える。第一原理バンド計算を新しいアルゴリズムにより効率的に実行し、原子核の運動まで扱う方法が第一原理分子動力学法である。1980年代から90年代にかけて、基礎である密度汎関数理論の確立、第一原理擬ポテンシャル法など高効率手法の開発、第一原理分子動力学法の開発など、飛躍的に発展した。90年代末より材料の機械的性質への適用が急増している。b. 基本的近似と密度汎関数理論通常の第一原理計算では、電子は原子核の動きに速やかに追随し、そのときの原子配置に対し瞬時に最安定状態をとると考え、固定した原子配置の電子構造を計算する[断熱近似、ボルン・オッペンハイマー(Born-Oppen-heimer)近似].一方、物質中の電子集団は、静電相互作用とパウリの原理のもとで複雑な多体相互作用により運動しており、単純な解析的取扱いは不可能である。通常、各電子が他電子からの平均的なポテンシャル場の中で独立に運動しているという描像を用いる(平均場近似)。この多電子間相互作用の取扱いが電子構造計算の精度を決定する。バンド計算では密度汎関数理論(DFT : density functionaltheory)を用いる。分子を扱う量子化学では、多電子波動関数の組立を工夫する理論が発達してきたが、密度汎関数理論は、多電子波動関数でなく電子密度分布関数ρに着目する。この理論の結論は、(1)原子核やイオンからの電場(外場)のもとでの多電子集団の基底状態は、ただ一つの電子密度分布関数ρをもつ、(2)基底状態のρは、ρの汎関数(関数の関数の意)である全エネルギー읏윱읏ρを最小にする、ということである。읏윱읏ρは次のように表される。읏윱읏ρ=ρ+ρd+ρρ′-′dd′+욂ρ(6・85)第一項は電子系の運動エネルギー、第二項はポテンシャルエネルギー、第三項は電子間静電エネルギー、第四項が複雑な多電子間相互作用をすべて含む交換相関エネルギーである。全エネルギー汎関数읏윱읏ρを最小にするρを求める方程式がコーン・シャム方程式(Kohn-Sham equation)である。ρが占有された一電子波動関数ψ욡の重ね合わせρ=2∑욡ψ욡で表され(2はスピンの和)、=2∑욡ψ욡-2∇ψ욡dとすると、읏윱읏の最小化条件から、ψ욡=-2∇+윬읆읆ψ욡=ε욡ψ욡(6・86・a)윬읆읆=+ρ′-′d′+μ욂ρ(6・86・b)μ욂ρ=δ욂ρδρ(6・86・c)のようにψ욡を決める方程式が導出される[ラグランジュ(La-grange)の未定乗数法].有効ポテンシャル윬읆읆は、原子核やイオンによる電場、電子による静電ポテンシャル=ρ′-′d′、複雑な多体相互作用を表す交換相関ポテンシャルμ욂の和である。入力の윬읆읆にρを含み、ρは出力のψ욡の重ね合わせなので、入力と出力のρ(または윬읆읆)が等しくなるよう自己無撞着(self-consistent)に解く。通常、入力のρに対するコーン・シャム方程式の固有状態計算とρの更新とを収束するまで交互に繰返す。c. 局所密度近似と密度こう配近似密度汎関数理論の問題点は、ρの汎関数である交換相関エネルギー욂ρやその汎関数微分である交換相関ポテンシャルμ욂ρの具体形がわからないことである。一様な電子ガスについては、理論や数値解析から関数形がわかっている。そこで、ρの空間変動が小さいとしてごとにその地点の電子密度の一様電子ガスの関数形を用いる簡便法が局所密度近似(LDA : local density approximation)である。実空間メッシュ点ごとにρから交換相関ポテンシャルμ욂、交換相関エネルギー密度ε욂を計算して与える。実際にはρの空間変動は小さくない。そこででの密度に加え密度こう配も取り入れて욂やμ욂を表す密度こう配近似(GGA : generalized gradientapproximation)も提案され、精度が向上している。d. 周期的ポテンシャル場における固有値・固有関数結晶やスーパーセルでは格子点ごとに同じ単位胞が並び、ポテンシャル윬읆읆に格子周期性(並進対称性)がある[윬읆읆=윬읆읆+].ここでは任意の格子ベクトルで、基本並進ベクトル,,を用いて=++である(,,は整数)。また、基本逆格子ベクトルは、=2π ××=2π ××=2π ××(6・87)で定義され、逆格子点は=++である(,,は整数)。実空間と同様に逆格子空間の単位胞が考えられ、これがブリルアンゾーン(Brillouin zone)である。,,で組立てられる平行六面体でもよいが、対称性の良い選び方として逆格子点を中心に近接する逆格子点へのベクトルの垂直二等分面で区切られた同じ体積の領域をとる。ブロッホの定理(Bloch’s theorem)から、並進対称性のある系のコーン・シャム方程式の解(固有関数)は、必ず波数ベクトルをもつ波(進行方向、波長2π)であり、はブリルアンゾーン内に限られ、固有関数はψ=expの形[は格子周期関数:=+]に書ける。バンド計算では、ごとに固有値、固有関数ψを求め、これらをについてゾーン内積分してρや욧욉욧を求める。実際にはゾーン内のメッシュ点など離散的なでの重み付きの和となる。積分の観点から効率的な点サンプル法が提案されている。金属の場合、占有部分と非占有部分の区別のため、多数の点が必要になる。限られた点で積分精度を上げる方法として、ぼかした占有率を導入する方法がある。なお、時間反転対称と結晶系のもつ対称操作から、点をサンプルする領域はブリルアンゾーン全域ではなく既約領域に絞ることができる。
