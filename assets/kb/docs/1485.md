# 12・2・4 ポストプロセッサ部への応用

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・2 ファジィ推論

a.概要 大規模計算から得られる膨大な数値データに基づ
き，その解析結果が妥当なものであるか否かを判断することは，解
析の熟練者にとっても容易なことではない．ここでは，多次元熱流
動解析の分野で，実用に供されているポストプロセッサシステムと
して，与えられた解析体系と境界条件のもとで発生する熱流動挙動
を，数値計算を行うことなく推論（ファジィ論理を導入した定性推
論）し，解析結果の妥当性の判断に用いられている機能 につい
て記述する．なお，ここで紹介する機能は，使用するMSFを構造
解析用のそれに入れ替えることにより，構造解析問題においても十
分な威力を発揮するものと考えられる．
b.基本原理 知識工学（knowledge engineering）分野にお
ける一般的な定性推論（qualitative inference，qualitative rea-
soning) は，質的な意味で量子化した集合上で行うものであり，
連続体を評価の対象とした場合には，その滑らかさを表現すること
が難しい．そこで，質的に変化する各ラベルの中間状態（例えば，
固体壁の若干手前等の状況）を，ファジィ論理によって指定可能と
し，この集合上で定性推論を行うものである．定性推論を行う場合
に使用する物理法則（等温流の場合）は，
(１)計算セルに流入する質量流量と流出する質量流量は等しい
(２)上流側の状態量は，下流側のそれに影響を与える
(３)流体は固体壁に沿って流れる
(４)流体は入口から出口に向かって流れる
(５)流れ方向にある固体壁により流速は減衰する
のたかだか5種類であり，また規定すべきMSFは，
(１)流速成分の大小（流速成分は非常に大きいなど）
(２)固体壁までの距離（距離は非常に短いなど）
(３)固体壁での摩擦（摩擦の影響は小さいなど）
で十分である．定性推論処理では，以上の物理法則をコンパイル
し，与えられた解析体系のもとで生じる熱流動挙動を，事前に規定
したMSFを用いて決定していく．なお，複数のMSFが影響し合
う位置では，推論合成を行って評価する．
c .定性推論解の一例とその利用 矩形二次元体系の左面から
流体が一様に流入し，右面から流出する定常問題（総計1000セ
ル）を例題として示す．流路の途中には，主流に直交する方向に障
害板が設置され，障害板の両端で流体のはく離が生じる問題であ
る．図12･12に，定性推論により得られた流速ベクトルと，差分法
