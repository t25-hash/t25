# 6・5・1 第一原理計算(first-principles calculation)の基礎

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・5 第一原理解析

a. はじめに固体材料の強度や変形、破壊を支配するのは、究極的には原子間結合の担い手である電子である。第一原理計算とは、量子力学と統計力学の理論のみにより、材料中の電子や原子の挙動を高精度に再現する計算である。分子を扱う量子化学の理論や手法もあるが、結晶や固体材料では、固体物理で発展したバンド計算法が用いられる。本節では、後者の概要を説明する。バンド計算は、結晶のように周期性のある系の電子構造計算法である。電子構造に基づき、全エネルギー(凝集エネルギー)、安定原子配列、弾性定数、原子に働く力、応力等、さまざまな物理量が計算できる。欠陥、表面・界面、アモルファスなども、大きなセル6―84計算力学(スーパーセル)が繰返す構造として扱える。第一原理バンド計算を新しいアルゴリズムにより効率的に実行し、原子核の運動まで扱う方法が第一原理分子動力学法である。1980年代から90年代にかけて、基礎である密度汎関数理論の確立、第一原理擬ポテンシャル法など高効率手法の開発、第一原理分子動力学法の開発など、飛躍的に発展した。90年代末より材料の機械的性質への適用が急増している。b. 基本的近似と密度汎関数理論通常の第一原理計算では、電子は原子核の動きに速やかに追随し、そのときの原子配置に対し瞬時に最安定状態をとると考え、固定した原子配置の電子構造を計算する[断熱近似、ボルン・オッペンハイマー(Born-Oppen-heimer)近似].一方、物質中の電子集団は、静電相互作用とパウリの原理のもとで複雑な多体相互作用により運動しており、単純な解析的取扱いは不可能である。通常、各電子が他電子からの平均的なポテンシャル場の中で独立に運動しているという描像を用いる(平均場近似)。この多電子間相互作用の取扱いが電子構造計算の精度を決定する。バンド計算では密度汎関数理論(DFT : density functionaltheory)を用いる。分子を扱う量子化学では、多電子波動関数の組立を工夫する理論が発達してきたが、密度汎関数理論は、多電子波動関数でなく電子密度分布関数ρに着目する。通常、入力のρに対するコーン・シャム方程式の固有状態計算とρの更新とを収束するまで交互に繰返す。一様な電子ガスについては、理論や数値解析から関数形がわかっている。そこで、ρの空間変動が小さいとしてごとにその地点の電子密度の一様電子ガスの関数形を用いる簡便法が局所密度近似(LDA : local density approximation)である。実際にはρの空間変動は小さくない。実空間と同様に逆格子空間の単位胞が考えられ、これがブリルアンゾーン(Brillouin zone)である。,で組立てられる平行六面体でもよいが、対称性の良い選び方として逆格子点を中心に近接する逆格子点へのベクトルの垂直二等分面で区切られた同じ体積の領域をとる。実際にはゾーン内のメッシュ点など離散的なでの重み付きの和となる。積分の観点から効率的な点サンプル法が提案されている。金属の場合、占有部分と非占有部分の区別のため、多数の点が必要になる。限られた点で積分精度を上げる方法として、ぼかした占有率を導入する方法がある。なお、時間反転対称と結晶系のもつ対称操作から、点をサンプルする領域はブリルアンゾーン全域ではなく既約領域に絞ることができる。
