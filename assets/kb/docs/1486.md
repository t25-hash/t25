# 6・5・2 第一原理バンド計算法の詳細

> **章**: 第6章 固体の離散系シミュレーション
> **節**: 6・5 第一原理解析

a. 各種の計算法結晶(周期系)のコーン・シャム方程式は、ブロッホの定理を利用して点ごとに解く。最終的に波動関数を基底関数で展開し、展開係数を求める固有値問題になる。この部分を効率的に行うため、基底関数やポテンシャルの扱い方にさまざまな工夫が提案されてきた。これらは、1線形化法[LAPW(linear augmented plane wave)法、FLAPW(full-potential LAPW)法、LMTO(linear muffin-tin orbital)法、FP-LMTO(full-potential LMTO)法],2擬ポテンシャル法[ノルム保存擬ポテンシャル法、ウルトラソフト擬ポテンシャル法、PAW(projector augmented wave)法],3第一原理LCAO(linear combination of atomic orbitals)法等に分類される。扱う電子で分類すると、内殻電子まで扱う全電子法(線形化法など)と内殻は凍結し価電子のみ扱う方法(擬ポテンシャル法など)がある。基底関数では、平面波のような広がった波の場合(擬ポテンシャル法、FLAPW 法など)と原子軌道のような局在波の場合(第一原理LCAO法、FP-LMTO法など)とがある。b. 第一原理擬ポテンシャル法(first-principles pseudo-potential method)の原理第一原理計算として最も広く用いられているのは、価電子のみを扱う第一原理擬ポテンシャル法(ノルム保存擬ポテンシャル法、ウルトラソフト擬ポテンシャル法、PAW 法)である。固体中の価電子の波動関数は、原子間ではスムーズだが、内殻軌道と直交するため原子近傍で激しく振動する。数値計算の立場からは、この振動を扱いたくない。そこで、価電子が感じる原子からのポテンシャルについて、原子近傍部分だけを振動(ノード)を生まないような浅い擬(偽の)ポテンシャルで置換える便法が考えられる。ハマン(Hamann)らは、原子近傍だけを浅くスムーズな形に入れ替えて、かつ価電子に対する散乱の性質を高精度に保持するような人工的ポテンシャルが、原子ごとに構築できることを示した。これをノルム保存擬ポテンシャル(norm-conserving pseudopotential)といい、価電子についてのコーン・シャム方程式のに用いることができる。これは、以下のように組立てられる。まず、孤立原子やイオンの全電子軌道計算(内殻と価電子)から、原子の真の波動関数を求める。動径方程式の反転から、そうした擬原子波動関数を出力する擬ポテンシャルを価電子軌道(軌道角運動量成分)ごとに組立てる。最後に自己無撞着計算用に価電子の擬波動関数による遮へい成分を擬ポテンシャルから取り去る。ここで、一般に、原子やイオンの球対称場の動径方程式の解について、以下の恒等式が成り立つ。左辺の積分がノルムである。擬原子波動関数が上記1~4の条件を満たすとき、式(6・88)の左辺の値や右辺の積の左の項は、真の原子波動関数のものと同じで正しい値になる。対数微分は内殻領域の散乱の性質を表し、そのエネルギー微分が、擬波動関数のものについても真の波動関数のものと同じで正しい値ということは、擬ポテンシャルがエネルギーの付近の一定の幅で散乱の性質を正しく再現することを意味する。したがって、価電子に働く内殻領域からの正確なポテンシャルとして、ノルム保存擬ポテンシャルを結晶やさまざまな系に埋め込んで使用できる。この性質をtransferability(適合性)という。点密度は実空間の単位胞体積Ωに比例し、大きなスーパーセルでは必然的に大きい。擬ポテンシャルは、カットオフ半径が大きいほど浅くスムーズにできるが、内の波動関数は不正確(振動を取り除いている)なので、精度が下がってしまう。そこで、擬ポテンシャル構築の条件1~4のうち、条件1を緩和したウルトラソフト擬ポテンシャル法(ultrasoft pseudopotential method)が開発されている。条件1を緩和すれば、同じについて、より浅くスムーズなポテン6―85シャル形が可能となる。一方、保存されない原子近傍の波動関数成分(ノルム)を別途補えば、式(6・88)のtransferabilityも保障される。凝集エネルギーなど物質の通常の性質は、価電子波動関数の原子の間の領域のふるまいが決定するので、原子近傍の波動関数形状が不正確(ノードがない)であっても問題は生じない。しかし、磁性など、原子近傍の波動関数挙動が鍵を握る性質では問題が生じ、FLAPW 法など計算の負荷の大きな全電子法を用いざるを得ない。そこで、擬ポテンシャル法の効率性と、全電子法の精度を併せ持つ方法として、PAW 法(projector augmented wave method)が開発されている。この方法は、原子近傍で振動のない擬波動関数を扱いながら、全エネルギーの表式では、原子近傍の部分だけをノードをもつ正しい波動関数での展開に置換えた形で計算する。そのため、平面波基底での変分計算は擬ポテンシャル法と同様に少ない基底で効率的に行いながら、結果的に原子近傍で振動をもつ正しい価電子波動関数とそのエネルギーを求めることができる。c.
