# 7・2・1 分子動力学法(molecular dynamics method)

> **章**: 第7章 熱流体方程式の選択と離散化手法
> **節**: 7・2 分子流れの解法

分子動力学法は,計算対象を原子や分子を表すモデル粒子の集合体としてモデル化し,そのモデルにおける各粒子の運動や空間配置の特徴を解析する手法であり,これを用いて構造的物性,熱力学的物性,動力学的物性,輸送現象的物性などが解析される.分子レベルの挙動を可視化して観察することもでき,ミクロな現象の理解に役立つ.分子動力学法は,分子が古典的な物理法則に従って運動するとして,個々の分子の時々刻々の動きを計算する手法である.量子力学的な効果はすべてポテンシャルの中に含まれているとし,各分子の運動量と位置は同時に決定できるものと仮定している.分子運動の支配方程式はニュートン方程式である.ポテンシャルがわかっていれば,分子に働く力がわかり,時々刻々の位置と速度が計算される.分子の運動方程式を解くに当たって,初期条件,境界条件を与える必要がある.初期条件としては分子の位置,速度ベクトルを与えることになる.計算を効率良く行おうとすれば,気相,液相,固相,気液共存相,固気共存相など計算対象に合わせて,適当な初期条件を与える必要がある.空間的な境界条件については,ある大きさの計算空間(基本セル)を考え,その計算空間に周期境界条件を課して,単位セルを周期的に並べ,無限大の大きさの計算領域を考えることが多い.この場合,基本セルから分子が出ていくと,反対側の境界の相対する位置から分子が流入することになる.基本セルの計算空間よりも大きな規模をもつ構造に関する情報は,計算できないことに注意が必要である.場に存在するすべての分子の動きを計算すれば,流れの特性量である密度,速度,圧力,温度など巨視的物理量が計算でき,流れが計算できる.連続体としての密度は,空間内の分子の数の時間平均と分子の質量の積から求められる.流れの対流速度は空間内の分子の平均速度から求められ,温度は分子の速度の平均からの分散として求められる.圧力は,ある空間内に存在する分子に働く外側の分子からの力の平均値として求められる.また,ある状態の分子運動がわかれば,それから自己拡散,運動量輸送,熱エネルギー輸送が計算でき,輸送物性が求まる.すなわち,平衡状態において生じる物理量の揺らぎの減衰から時間相関関数を求め,その値から計算する方法(グリーン・久保の方法),物理量の時間的な変位を求め,その値から計算する方法(アインシュタインの方法)などがある.これらの統計量は,空間の大きさ,平均時間の長さに依存して変動し,連続体として意味のある値を評価するには注意が必要である.なお,分子動力学法のさらに詳しい説明については,6・4節,
