# 7・2・2 直接シミュレーションモンテカルロ法(DSMC法:

> **章**: 第7章 熱流体方程式の選択と離散化手法
> **節**: 10・2 節を参照のこと。

direct simulation Monte Carlo method)分子動力学法によって比較的大きな領域の熱流動現象を解析することは、計算機の記憶容量、計算速度の点で難しい。そのため、このような流れ場を解析する際には、ミクロスケールの情報を合理的に取り込んだメゾスケールの分子衝突(molecular collision)モデルを用いて、DSMC法のような確率統計的手法を用いることになる。ここでは、希薄気体流れの支配方程式であるボルツマン方程式の数値解法として多く用いられるDSMC法の概略を示す。つまり、一つのサンプル分子は個の実分子を代表していることになる。DSMC法ではすべての分子の運動を追跡する代わりに、サンプル分子の運動を追跡することで実分子の運動を解析する。DSMC法では、サンプル分子の運動の時間発展を時間ずつ進めて追跡する。厳密には分子ごとに衝突時間は異なるが、多数のサンプル分子の平均自由時間がτと一致するように衝突確率を決定することによって、サンプル分子の衝突と移動を独立に扱うことが可能となる。また、DSMC法では、空間的に一様な気体に対するボルツマン方程式によって衝突を扱うため、計算領域を平均自由行程程度の多数の微小セルに分割して計算する必要があり、セル一辺の長さとして、平均自由行程の3分の1程度が要求される。このような条件を満たすようにセルを決定すれば、分子間衝突はセル内において、位置に無関係に生じると考えることができる。なお、直接シミュレーションモンテカルロ法のさらに詳しい説明については、10・3節を参照のこと。
