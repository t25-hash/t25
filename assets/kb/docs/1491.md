# 12・4・2 GAの特徴

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GAの特徴は次のとおりである．
(１) 目的関数（適合度）の値そのものを用い，こう配情報を用
いない．このため，微分情報が得られない目的関数や，微分不可能
な目的関数に対しても有効である．
(２) 確率的手法であり，厳密な最適解が得られる保証はない．
しかしながら多くの場合，良好な準最適解を得ることができる．
(３) 複数の探索を同時に行う多点探索である．このため，GA
は大域的最適化手法の一つである．
GAの問題点は次のとおりである．
(１) 多くの場合，設計変数はコード化されて遺伝子となる．こ
のため，用いたコーディングが良くない場合，良好な解が得られな
い．
(２) 交差や突然変異には多くの方法があり，また問題に適した
交差や突然変異を新たに考案することもできるが，その選択や設計
が良くないと，解の探索能力は良好でない．
(３) ふつう，100個体を100世代進化させるとして，目的関数
の評価回数は1万回となり，数理計画法などと比較して著しく多く
なり，評価計算のコストが高い場合には膨大な計算パワーが必要と
なる．
(４) 組立てたGAのスキームが良好でないために適合度の上
昇がなくなったのか，あるいはすでに最適解を見出したのかがわか
りにくく，どこで終了させればいいのか明白でない．
これらの特徴や問題点を十分知り，問題ごとに適切なコーディン
グ，交差，突然変異，そして選択方法を決め，高い計算能力をもつ
コンピュータで予備実験を行いながら，GAオペレータや種々のパ
ラメータをチューニングしなければならない．このため，未知の最
適化問題をGAで効率的に解くにはかなりの経験を必要とする．
