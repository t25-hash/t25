# 7・3・1 NS 方程式に対する離散化手法

> **章**: 第7章 熱流体方程式の選択と離散化手法
> **節**: 7・2 節では、分子流れの解法についてまとめたが、ここでは連続

オイラー座標系で記述された連続体流れは、流体の圧縮性の程度により、非圧縮性流れと圧縮性流れに大まかに分類することができる。圧縮性の程度については、無次元数であるマッハ数(=,:代表速度、:音速)が指針となり、マッハ数が約0.3を超えるあたりから、圧縮性流体としての取扱いが必要となる。マッハ数と並んで重要な無次元数として、レイノルズ数(=ν,:代表長さ、ν:動粘性係数)が挙げられる。レイノルズ数は流れの慣性力と粘性力の比を表し、レイノルズ数が大きくなるに従い、慣性の影響が大きくなり、流れは層流から乱流へと遷移する。乱流では、NS 方程式の移流項が支配的となり、非線形性が強くなるため、一般的には乱流モデルの導入が必要となる。a. 非圧縮性流れ(incompressible flow)乱流モデルを必要としない低レイノルズ数流れの数値解析では、通常、非圧縮性流れとみなすことができ、多くの場合に信頼性の高い結果が得られている。また、非圧縮性流れでは多くの実用的な工学問題に、数値解析が適用されている。したがって、本項では非圧縮性流れの中間レイノルズ数における離散化手法、特に空間離散化手法を中心に取上げる。非圧縮性流れでは、流れの未知量は速度と圧力のみとなる。また、連続の式とNS 方程式とで基本方程式は完結し、エネルギー方程式を解く必要はない。非圧縮性流れでは、運動方程式のみに現れる圧力は密度との関連性がなく、連続の式を満たすように求める。そのため、連続の式とNS 方程式を連成して解く必要があり、数値解析上の困難が生じる。ここで、空間離散化手法を選択するうえで、解析対象に対してどのような離散化格子あるいは要素を用いて、どのように離散化点を配置するかが重要となる。解析に用いられる格子は、構造格子(structured grid)と非構造格子(unstructured grid)に大別される。図7・3に示すように、構造格子は正規座標を基として、格子をサイの目のように配置する。物体境界の形に関係なく、固定した格子を形成する矩形固定型[図7・3(a)]と、境界の形に合わせて格子を形成する境界適合型[図7・3(b)]が考えられる。一方、非構造格子は格子生成の自由度の高さから、通常、境界形状に合わせて格子生成がなされ、二次元の場合には三角形や多角形、三次元の場合には四面体、プリズム要素や六面体などの要素が用いられる。一般に構造格子を用いる手法としては有限差分法、スペクトル法などが挙げられ、非構造格子としては有限要素法などが挙げられる。ここで、離散化について有限差分法では格子、格子点、格子分割といい、有限要素法では要素、節点、要素分割ということが多い。以下、これらの手法について説明する。. スペクトル法(spectral method)スペクトル法は、求める解を三角関数やチェビシェフ多項式などの直交関数系で級数展開する手法である。級数展開により表された解を元の微分方程式に代入し、基底関数の直交性を用いて展開係数の関係を求め、解を求める。この際、用いる基底関数の数は解像する流れのスケールにより決定する。スペクトル法は、計算精度の面では大変優れている。このため、乱流解析において、付加的な数値モデルを用いずに、連続の式とNS 方程式を忠実に解析する直接数値シミュレーション(DNS :direct numerical simulation)で用いられる。一方、形状適応性については、境界条件の与え方に対する制限から簡単な形状にしか適用できず、応用範囲が限られている。この問題を解決するために、スペクトルエレメント法が開発され、任意形状の物体周りの流れなども取扱えるようになってきている。b. 圧縮性流れ(compressible flow)流れ場のマッハ数が、約0.1~0.3を超えると、流れの圧縮性を考慮する必要が出てくる。このような圧縮性流れの数値解法は、一般に非圧縮性流れの数値解法とアルゴリズムが大きく異なる。まずいちばんの違いは、圧力の求め方にある。非圧縮性流れの場合には、圧力はだ円形方程式であるポアソン方程式を解いて求める。すなわち境界値問題の解として、場の圧力分布を求める。これに対して圧縮性流れの場合には、圧力は状態方程式から求められることになる。流体の圧縮性が重要となるのは、空気のような気体の場合が多いため、圧力の決定には、多くの場合に理想気体の状態方程式が用いられる。したがって、状態方程式が用いられる圧縮性流れの場合には、圧力はその格子点における他の物理量(密度、エネルギーなど)から計算されることになり、非圧縮性流れの場合と異なり、周囲の格子点における圧力の値を参照して連立方程式を解くことにより、圧力を求めるプロセスがなくなる。ところで、マッハ数が十分小さい非圧縮に近い流体に対しても、圧縮性流れの解法を用いることは可能である。通常、非圧縮性流れの場合には、圧力のポアソン方程式を解くプロセスが計算時間の8割以上を占めるが、圧縮性流れの解法を用いると、このプロセスがなくなる。しかし、音速により決められる小さな時間刻みを用いる必要があるため、全時間ステップの数が大幅に増え、また、時間ステップ数の増加により、計算誤差の累積が大きくなるなどの問題もある。圧縮性流れのNS 方程式は、双曲形の性質をもつ対流項と、だ円形(時間微分項まで考えると放物形)の性質をもつ粘性項から成っている。圧縮性の影響を考えなければならない状態は、通常、高レイノルズ数流れとなり、高レイノルズ数流れでは固体壁近傍の薄い境界層の領域を除いて粘性項の影響は無視できることが多く、この場合には境界層を除いた大域で双曲形のオイラー方程式に支配される。したがって、圧縮性NS 方程式では、通常、双曲形方程式6―9 2計算力学の解法が用いられる。オイラー方程式を双曲形方程式として解く場合に重要なのは、風上化である。流れのもつ特性に応じて風上方向に差分を行うが、これにはオイラー方程式の局所線形化、流束ヤコビアン行列の対角化などの操作を行い、固有値、固有ベクトルを計算する必要がある。これらのプロセスの詳細については、文献(5)(6)などを参照されたい。以下、圧縮性流れの解法の手順をまとめておく。(1)
