# 12・4・5 GAのパラメータ

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GAには多くのパラメータが存在する．最も重要なパラメータ
は，染色体長，母集団サイズ，交差率，突然変異率，終了世代数で
ある．連続変数を2進符号化した場合，一つの変数を ビットで
コード化し，設計変数の数を とすると，染色体長 は ビッ
トとなる．この場合， は解の精度から決めればよい．
一方，適切な母集団サイズ（全個体数）は，問題やGAオペレ
ータなどに依存しているため不明であり，50～100個体程度の値で
予備実験を行い，進化の推移を観察しながら実験的に決めなければ
ならない．ただし，突然変異率（mutation rate，遺伝子が変化す
る確率）は，従来の研究から0.001～0.01の値を用いるか，あるい
は1/（ ：染色体長）が経験的に良好であることが知られてお
り，この値を中心として予備実験を行えばよい ．また，交差率
（ある世代の中で交差する個体の割合）は，経験的に0.45～0.95の
値，あるいは0.6が良好であることが知られており ，さらに交
差率を変化させても結果への影響は少ないことから，適当な値に決
めても問題は少ない．終了世代数を適切に決めることは難しい．一
般には適合度の改善が見られなければ終了せざるを得ないが，それ
はパラメータの値や用いたGAオペレータに依存して，進化が停
滞しているだけかもしれない．したがって，未知の最適化問題を解
くとき，適切な終了世代を事前に決めることは困難であり，試行錯
誤に頼らざるを得ないが，その場合，解の進化過程を詳細に検討
し，進化の過程で何が生じているのかをできるだけ詳しく観察する
ことが，パラメータのチューニングや適切なGAオペレータの選
択につながる．
