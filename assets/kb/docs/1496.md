# 12・4・7 並列処理

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GAは計算負荷が高いために並列処理が重要となる．特に，解析
部分での時間が比較的長い計算工学分野では，GAの並列処理は必
須であるが，幸いにも，GAの並列処理は容易である．
GAの並列処理には大別して二つのアプローチがある．それら
は，(１)複数個体の評価における複数の解析計算を並列化する，
(２)分割母集団モデルを用いて，各サブ母集団での並列化を行う，
である．このうち，(１)はマスタスレーブ型の並列処理で行うこと
ができる．交差や突然変異，あるいは選択などのGA操作をマス
タが行い，スレーブは子個体の適合度評価だけをマスタから指示さ
れて行う．この並列処理は，適合度の評価計算の時間が比較的長い
場合に有効である．そうでない場合には，マスタがスレーブに計算
を分配している間に待ち時間が生じ，計算効率が悪くなる．
一方，分割したサブ母集団にプロセッサを割当てる方式の並列処
理では，適合度の計算時間に関係なく，高い計算効率が得られ，か
つプロセッサ間通信は移住時のみに発生するため，ネットワークへ
の負荷が少なく，遅いネットワークを用いた安価なPCクラスタで
も高い性能を実現することができる ．
