# 12・4・8 多目的最適化

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

最適化の中でも複数の目的関数をもつものを，多目的最適化
（multi - objective optimization）と呼ぶ．多目的最適化では，単一
の最適解を求めるのではなく，パレート最適解（Pareto- optimal
set ）と呼ばれる最適解の集合を求める．この場合，複数の探索点
をもつGAは多目的最適化に極めて有利であり，現在ではGAは，
多目的最適化の標準的な解法となっている．
多目的GAは大別して三つの手法に分類できる ．
(１)単一目的に変換（スカラ化）し，パラメータを変えてパレー
ト解を得る．
(２)母集団内で異なる目的関数を用いたり，複数の目的関数を順
に適用するなどを行い，母集団全体としてパレート解を得
る．
(３)母集団内の個体を何らかの方法でランク付けし，パレート解
を直接的に得る．
多目的GAの手法としては多くのものがあるが，なかでも
MOGA（multi - objective GA)，NSGA-Ⅱ（non- dominated sort-
ing GA），およびSPEA2（strength Pareto evolutionary algo-
rithm）は代表的な方法である．
MOGAは，1993年にフォンセカ（Fonseca）によって提案され
た手法である ．MOGAでは，個体の評価をパレートランキング
法に基づいて行う．この方法では，パレート解に近い解に高い適合
度を与える．また，非劣解の多様性保持のための手法としてシェア
リングを行う．シェアリング（sharing）とは，母集団内で個体が
偏在して集合したとき，その個体集団の適合度を減少させて個体数
を少なくする操作である ．一般には一つの個体の近傍を調べ，
ある距離内に別の個体が存在すると適合度を下げる．シェアリング
は多目的GAだけでなく，単一目的のGAにも有効な方法である．
NSGA-Ⅱ は，2001年にNSGAの改良アルゴリズムとして，
デブ（Deb）らによって提案された手法である．この手法では，保
存する母集団（親母集団）と，交差・突然変異といった遺伝的操作
を用いた探索を行うための母集団（子母集団）の，二つの独立した
母集団を用いて解探索を行う．また，親母集団と子母集団を合わせ
て非優越ソート（non-dominant sorting）を行い，全個体をラン
ク付けする．またシェアリングも行う．一方，SPEA2 は
SPEAの改良アルゴリズムとして，ツィツラー（Zitzler ）らによ
って2001年に提案された．SPEA2はNSGA-Ⅱと同様に，非劣
解を保存する母集団（アーカイブ母集団）と，交差や突然変異など
の遺伝的操作を行う母集団の，二つの母集団を利用して探索を行う
が，NSGA-Ⅱとは適合度の割当て方法やシェアリングの方法など
に多少の違いがある．また，渡邉，廣安，三木らはSPEA2に近
傍交差の概念を取り入れ，探索性能の向上のみならず，パレート解
集合における一様性の向上を達成した ．
