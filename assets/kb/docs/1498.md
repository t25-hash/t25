# 8・1・3 境界適合座標系

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

図8・8に示されるように、流れ場の幾何形状に曲面が存在するなど、直交座標系で流れ場を正確に格子分割できない場合、一般に流れ場の幾何形状に適合した計算格子を作成する。このように流れ場の境界に沿った計算格子を境界適合格子、その座標系を境界適合座標系(BFC: boundary-fitted coordinate)という。境界適合格子は、隣接する格子間の関係の構造は保たれているので、構造格子(structured grid)に分類される。また、境界適合座標系は不等間隔、非直交であり、いわゆる一般曲線座標系である。境界適合座標系による解析は、直交座標系の解析と比較して、座標変換によって計算量と誤差が増加するが、境界条件の取扱いの点で複雑形状に適用しやすい。一方、非構造格子系の解析と比較すると、複雑形状に対する適用性は若干劣るが、重合格子(oversetgrid,chimera grid)技術を用いれば、かなり複雑な流れ場形状の解析も行える。しかし、計算格子の質(サイズ、格子解像度、直交性など)が、計算の収束性・計算精度に影響を与えるので、境界適合格子の作成においては、対象とする流れ場の知識をある程度必要とする。a. 境界適合座標系の分類境界適合格子は、そのトポロジーによって図8・8のように分類される。O型格子は、格子点のむだが少ないが、翼型のように尖った角がある場合、格子のひずみが大きくなる。C型格子は翼型の計算に適し、後縁付近の格子の質の制御が容易である。H 型、L型格子は、格子形成が比較的簡単であり、マルチグリッド法とともに複雑形状流れ場への適用性に優れている。格子生成技術の向上により、単一の計算格子でもかなり複雑な形状を再現できるようになっている。しかし、もっと複雑な流れ場形状を扱いたい場合や格子生成を容易にしたい場合は、図8・9に示される重合格子、接合格子、複合格子といった複数の格子系を組合わせて、接合部または重合部で情報を交換しながら計算を行う。重合格子は、複数の計算格子を重ね合わせたもので、移動境界に対する適合性にも優れる。接合格子は、重合格子と比較すると格子形成の手間は増えるが、計算コードの修正量は少なくて済む。複合格子(a) 計算格子生成法格子生成法は、代数的方法と偏微分方程式を用いる方法とに分類される。代数的方法は、補間関数やいくつもの写像変換関数等を用いて空間を離散化していく方法であり、格子線の制御が一般的に容易であり、かつ短時間で格子生成できる利点をもつ。一方、偏微分方程式を用いる方法は、用いる方程式の形によってだ円形、双曲形、放物形に分類される。代数的方法に比べてはん用性があり、滑らかな格子線が比較的簡単に得られる。c. 変数配置レイアウト-格子系。 直交速度成分を変数とする方法速度、圧力といった変数の定義点の配置として、レギュラー、スタッガード、コロケート格子系が提案されている(図8・10)。変数をすべて同一点に定義するレギュラー格子系は、速度と圧力のカップリングが悪く、計算が不安定になりやすいためあまり使用されない。スタッガード格子系は、直交格子の計算においては、離散化の際に必要とする補間が少なく、計算解像度が高いためによく使用される。しかし、一般座標系の場合、計算格子が90°回転した場合を考慮すると、直交速度成分がセル界面に沿ってしまい、スタッガード配置の利点が失われ解が不安定になる[図8・10(c)].図8・10(d)のように、直交速度の全成分を各セル界面に定義したスタッガード格子系が提案されているが、空間次元数倍の運動量方程式を解く必要があり、あまり効率的ではない。コロケート格子系は、反変速度成分をセル界面に定義することによって、レギュラー格子系の計算不安定の問題を軽減した格子系である。一般座標系における格子系としてはこのコロケート格子計が最も適合性が高く、一般的によく使用されるが、運動エネルギーの散逸誤差が大きいことが知られ、乱流の数値シミュレーションにおいては注意が必要である。.
