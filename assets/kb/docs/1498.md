# 12・4・9 GAの応用

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

近年，GAは計算工学の分野で広く用いられている．それらは文
献(36)～(40)に詳しく記述されている．ここでは二つの例を挙げ
る．一つは航空機の空力設計に関して，GAを用いて多目的最適化
を行ったものである ．ここで設計変数は翼の形状，目的関数は
遷音速巡航抵抗の最小化，超音速巡航抵抗の最小化，そして超音速
巡航時の翼根の曲げモーメントの最小化である．制約条件は飛行速
度や揚力係数など六つある．この問題を多目的GAで解くとパレ
ート最適解集合が得られ，種々の形状をした解が得られ，設計者は
それぞれの解の特徴を考えながら，最も優れた解を選択できること
がわかった．
また，もう一つの例として，GAをシミュレーテッドアニーリン
グ（SA:simulated annealing）と組合わせた応用を紹介する．そ
れはタンパク質の三次元構造同定問題に，これらの手法を組合わせ
たハイブリッド最適化手法を適用したものである ．ここでは，
タンパク質分子のエネルギーを最小化することで構造を求めてお
り，設計変数は分子内の原子間の二面角である．並列分散SAを基
本とし，定期的に同期をとり，複数の解を基にGAの交差を行い，
より良好な解を見出す．
このように，GAはこれまで解くことが容易でなかった複雑な工
学的最適化問題に次々と適用され，目覚ましい成果を挙げている．
今後は，より広い分野でGAが応用されるとともに，上のような
ハイブリッド手法や，あるいは数理計画法や実験計画法，あるいは
応答曲面法など，従来の最適化手法と組合わせた手法なども開発さ
れ，最適化問題を与えるとその問題の特徴に応じて，最適な最適化
手法を自動的に選ぶシステムも開発されると思われる．
