# 8・1・4 非構造格子による解析

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

構造格子(structured grid)に適用できる空間差分スキームやアルゴリズムが、そのまま非構造格子(unstructured grid)にも適用できるとは限らない。これは、構造格子では格子線の方向と座標の方向が一致しているが、非構造格子ではそうではないからである。高次差分スキームの多くは構造格子を前提としているため、非構造格子では使用できない。なお、境界適合格子は、格子線の方向が計算空間における座標の方向と一致しており、構造格子に分類される。非構造格子は六面体(hexahedron、二次元では四角形)のみをつなげて作成することもできるが、ここでは任意の多面体(二次元では多角形)をつなげて作成した格子であるとする。解析手法としてここでは、有限体積法(FVM : finite volume method)を中心に解説する。非構造格子に適用できる有限体積法は、当然ながら境界適合格子などの構造格子にも適用できる。ただし、有限体積法という用語は、適用対象を非構造格子に限定した狭い意味に用いられることもある。基礎式として、非圧縮性流れにおける質量保存則と運動量保存則を考える。次に、変数配置を考える。式(8・92)(8・93)における変数は、速度ベクトルと圧力である。非構造格子の有限体積法では、これ(a) interpolated pseudo-particlemethod)と同様の手続きで高次差分スキームを導くIDO(inter-polated differential operator)法、流れ方向に一時的かつ局所的な一次元格子を設定する、MAFL(meshless advection usingflow-directional local-grid)法などである。これらは計算点の接続関係に依存しない計算法であり、メッシュレス法(meshlessmethod)としても機能する。式(8・93)の右辺第2項の圧力項では、セル境界の圧力を求める必要がある。これは単純に境界を挟む二つのセルでの圧力の平均とする。これをまずセル中心で計算する。これには圧力項と同じように式(8・100)が使用できる。セル境界でのこう配から、式(8・94)と同様に、粘性項を計算する。そこで、二つのセルの中心間の相対位置ベクトルを計算し、その方向の速度ベクトルのこう配については平均化せず、境界で直接計算することにする。すなわち、図8・12において、境界を挟む二つのセルの中心点をT,R 非構造格子におけるセル(二次元)6―102計算力学散化された式をそのまま組合わせて圧力のポアソン方程式を構成すると、一つとびに接するセルとの関係式が現れる。この場合、圧力場の振動を許すことになる。そこでセル中心にすべての変数を配置する有限体積法では、圧力の計算にライ・チョウの方法を用いる。ここでは非定常問題を想定し、各時間ステップを半陰解法で解くアルゴリズムを考える。速度ベクトルの修正量′は、圧力の修正量′のこう配によって生じるものとする。そのため、このアルゴリズムは半陰解法と呼ばれる。一方、式(8・109)の左辺の′はセル中心の値であり、また同じ式の右辺の′はセル境界の値である。式(8・110)の左辺のは、基本的にはその境界を挟む二つのセル中心の速度ベクトルの平均とする。ただし、圧力項についてはセル境界で再計算する。それと同時に、この圧力項成分をセル中心の速度ベクトルより引く。速度ベクトルの修正量についても、セル中心の圧力と関係させる。これは境界を接するセルの圧力のみで構成されており、圧力の振動は現れない。各セルで、圧力の修正量のポアソン方程式を作り連立させると、連立一次方程式が得られる。この連立一次方程式に現れる係数の行列は、非構造格子上で得られたものであり、一般的には規則的な行列にならない。そのため、ある種の高速解法は適用できない。例えば、ADI法(alternating direction implicit method)は、セルがつねに座標方向に沿ってつながっていることを前提にしているので、非構造格子には適用できない。非構造格子にも適用できる高速解法には、共役こう配法(conjugate gradient method,CG 法)、GMRES 法(generalized minimal residual method)、不完全コレスキー分解付き共役こう配法(incomplete Cholesky decomposi-tion conjugate gradient method,ICCG 法)などがある。さて、圧力の修正量に関する連立一次方程式が解かれ、セル中心における′が得られたら、これを式(8・107)に代入すればが得られる。セル境界での圧力修正量を求めてから式(8・109)に代入すれば、セル中心における′が得られ、これを式(8・106)に代入してが得られる。式(8・112)に代入すれば、セル境界での速度が修正される。このセル境界での速度が対流項、すなわち式(8・99)でも用いられる。圧力のポアソン方程式を解く際に導入したセル境界での速度ベクトルは、離散化された質量保存則、すなわち式(8・108)を満たしており、これをそのまま対流項の計算におけるセル境界での体積流量にも用いる。セル間を結ぶ線とセル境界とがつねに直交するような特殊な格子として、ボロノイ分割がある。すなわち、ボロノイ多角形とデローニー三角形は、互いの辺が直交する関係にある。ボロノイ多角形をセルとすると、デローニー三角形はセル間を結ぶ線を表すことになり、逆にデローニー三角形をセルとすると、ボロノイ多角形はセル間を結ぶ線を表す。セル間を結ぶ線とセル境界とがつねに直交するから、圧力項や粘性項における非直交項が消え、離散化式が簡単になる。ボロノイ多角形とデローニー三角形のどちらでも、有限体積法のセルとして用いることができる。
