# 8・1・5 高精度差分法

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

一般に、時空間で離散的に与えられた数値を用いて多項式近似を行い、それを微分することによって、微係数の差分近似(finitedifference approximation)式を導くことができる。そして、差分近似を行った点を中心として、近似に用いた離散点群(ステンシル)に対してテイラー展開(Taylor expansion)を行うと、真の微係数に対する離散化誤差を見積もることができる。誤差が離散点の幅Δに対してΔのオーダであるとき、その差分近似式は次の精度をもつという。しかし、流れの数値計算では、隣り合う離散点の中間でのこう配の計算のほうがよく使われる。以上に示した差分式は、ステンシルが計算点に対して対称に配置されているので、中心差分(central finite difference)法と呼ばれる。一般に、流れの数値計算における高次精度差分(higher-orderfinite difference)は、三次以上の精度次数を有するものをいう。中心差分法を高次化する一つの方法は、より多くのステンシルを用いて、補間多項式の項数を増やすことである。あるいは、より多くのステンシルに対してテイラー展開し、高次の項まで消去することである。同様にして、より高次の補間式や差分式を求めることができる。精度次数とともに重要な性質は、離散化における整合性(con-sistency)である。このような配慮をしない差分法であっても、精度を高めれば整合性の誤差も低減されると思われる。しかし、整合性を保ちつつ高精度化する方法も提案されており、特に格子分割数の不十分な場合には効果が大きいと考えられる。不等間隔に配置されたステンシルに対しては、実空間での高次の多項式近似は非常に煩雑になるばかりでなく、上記の整合性を満たすことも難しい。ステンシルを拡大する方法による高精度化は、計算領域の境界近傍では、精度を維持することが困難である。すなわち、テイラー展開ではなくパデ展開(Pade expan-sion)を用いるもので、コンパクト差分(compact finitedifference)と呼ばれている。コンパクト差分では、1階および2階の微分は例えば、次のように表現される。明らかに、α,βのいずれかが6―103零でなければ、各点ごとに微係数は直接求まらず、連立方程式となる。そのため、微係数を求めるために計算領域のすべてのデータを参照するという、スペクトル法に類似の特性を有する。また、フーリエ変換で解析すると、細かい変動に対する解像度も、同じ精度次数のテイラー展開に基づく差分よりも高い。しかし、このことは細かい変動に敏感であることを意味し、非線形問題に対しては計算が不安定になりやすい。そのため、フィルタをかけてスムージングするなどの措置がとられているようである。以上述べた中心差分法とは別に、対流項に対して上流差分(up-stream finite difference)法[風上差分(upwind finite difference)法]が用いられることがある。三次精度あるいは五次精度の上流差分法は、(少なくとも二次中心差分よりも精度次数が高いので)高次精度差分に含められることもある。これらは、中心差分と偶数階数の差分による数値粘性に分解できる。この種の方法の問題は、中心差分の部分が前述の整合性を満足していないこと、高階数値粘性の非物理的なふるまいの影響が吟味されていないことである。したがって、精度の次数は高くても、使用に当たっては十分な注意が必要である。時間に対しては、未来側に複数のステンシルを使うことは難しいので、空間差分とは異なる差分法が用いられ、高次精度化の方法も異なる。この方法は、次のようにをテイラー展開することによって導かれる。複数ステップで右辺を評価することによって高精度化するアダムス・バシュフォース法に対して、ステップから+1ステップの間を多段階とするルンゲ・クッタ法(Runge-Kutta method)も、流れの非定常計算ではよく用いられる。ただし、中間段階をすべて記憶する必要のある古典的な方法ではなく、省容量形が好まれる。
