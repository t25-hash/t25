# 8・1・5 高精度差分法

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

一般に、時空間で離散的に与えられた数値を用いて多項式近似を行い、それを微分することによって、微係数の差分近似(finitedifference approximation)式を導くことができる。そして、差分近似を行った点を中心として、近似に用いた離散点群(ステンシル)に対してテイラー展開(Taylor expansion)を行うと、真の微係数に対する離散化誤差を見積もることができる。誤差が離散点の幅Δに対してΔ욗のオーダであるとき、その差分近似式は次の精度をもつという。したがって、精度次数(order of accu-racy)とは、離散点の幅を1倍(分割数を倍)にすれば誤差は1욗倍になると期待されることを意味する指標である。a.
