# 8・1・6 直交格子による任意形状物体の扱い

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

複雑な境界形状を伴う流れ場については、構造格子では一般曲線座標による境界適合格子および非構造格子の方法がすでに解説されているが、ここでは、境界が必ずしも格子点とは一致しない場合の解法について述べる。ここでは固体壁面(ノンスリップ境界条件)の場合を考える。一般に、デカルト座標・円筒座標・球極座標などによる直交格子(orthogonal grid)系の数値計算は、境界適合格子や非構造格子に比べて、1点当たりの計算量や記憶量が少ない利点をもつ。しかし、例えば図8・13(a)のように、座標に沿っていない境界を階段状に扱う場合には、滑らかに表現するために非常に多くの格子点を必要とする。また、高レイノルズ数の粘性流れは、物体の表面近傍で大きな速度こう配をもつので、境界近傍だけで高解像度が要請される場合が多い。境界適合格子や非構造格子の方法は、これらの問題を解決するために発達してきた。しかし、非常に複雑な形状、変6―104計算力学(a)(b)(d)(c)図8・13直交格子による任意形状境界の扱い(説明図)形する物体、多数の物体群、相対運動する物体群などを扱う場合には限界があり、格子を生成するための時間とコストも無視できない。あるいは、全領域で解像度を一様に保ちたい場合もある。これらの背景から、直交格子(主としてデカルト座標)に任意に交わる境界に対して、ある程度の精度を保ちながら効率良く扱う方法が検討されている。以下にいくつかの例を示すが、新たな提案も続いており、方法の名称は必ずしも定着していないので注意されたい。a. 格子の細分化とカットセル法格子点の増加を最小限にとどめて精度を高める最も簡単な方法は、図8・13(b)のような直交格子の局所的な細分化であろう。細かい格子と粗い格子の間で、補間や外挿によって保存則を破綻させないように注意すれば、格子に対して相対的に静止した物体まわりの大まかな流れを知る目的には十分と思われる。ただし、局所的に細分化するだけでは、がたつきを根本的に解消したことにはならない。また、移動境界問題では動的に細分化する必要があり、煩雑となる。差分法の場合には、格子線と物体境界の交点を一つの差分ステンシルとして、図8・13(c)のように境界近傍で不等間隔差分や片側差分をする方法も多く提案されている。しかし、片側差分による精度低下、局所的に格子幅が狭くなることによる不安定化、境界に関与しない差分法との整合性の観点からは問題が多い。一方、有限体積法では、図8・13(d)のようなカットセル法(cut-cell method)が提案されている。二次元であれば境界と格子線の交点を結ぶ直線により、三次元であれば境界と格子点の交点を通る面により、境界を含むセルを分割する。カットセルがクーラン条件などから計算の安定性を損ねるほど小さければ、隣接セルと結合させる。三次元では曲面を扱うために計算時間の増加が無視できないので、カットセルの体積を簡潔かつ迅速に計算するために、図8・13(b)のように細分化して数値積分するなどの方法もとられている。b. 境界埋込み法直交格子で任意形状の境界を扱う代表的な方法として、境界埋込み法(immersed boundary method)がある。この方法の特徴は、ナビエ・ストークスの運動方程式(右辺のすべての項を...と略記)に옿옿=...+(8・131)のような強制力を与え、時間進行の結果を境界条件によって拘束することにある。左辺を옿옿=욗-욗と近似するとき=-...+욞-욗(8・132)とおけば욗=욞となり、時間進行の結果を指定された速度욞に強制できる。類似の考えは、固体境界を取扱う場合だけでなく、重合格子における格子間のデータの伝達法としても提案されている。ペスキン(Peskin)のフィードバック法は、物体を連結された点群で表現し、各点で体積力を求めて近似δ関数により流れ解析格子とカップリングを行うものである。物体を表現する点群の間に弾性体などの応力-ひずみ関係を与えれば、流体と固体の運動の連成問題に応用することもできる。一方、ゴールドシュタイン(Goldstein)らは、減衰振動しながら指定された速度욞に移行することを表すモデル욦,=α욞욦,τdτ+β욞욦,(8・133)を外力として導入した。計算が安定になるようにα,βが調整されるが、時間刻みを非常に小さくする必要があるといわれている。ムード・ユソフ(Mohd-Yusof)が提案した直接強制法(directforcing method)は、フィードバック法に対して計算が簡略であり、安定性もよい。図8・13(c)を使って概略を説明する。まず、流体側にある格子点において、ナビエ・ストークス式で時間進行を行う。空間差分のステンシルが固体内部に入っても、その点の固体速度を使っておけばよい。次に、物体境界に最も近い点(図では中の点)では、境界点(図では下)と流体側の点(図では上)から補間される速度に更新されるよう、強制力を作用させる。その補間の精度を高めることにより、境界近傍における精度の低下を防ぐことも検討されている。
