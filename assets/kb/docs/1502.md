# 8・1・7 移動境界問題(moving boundary problem)

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

物質加工プロセッシング,流体構造連成問題や火炎伝ぱ問題など,相や熱物性値・組成が急峻に変化する内部境界によって特徴付けられる系は,界面周辺の熱流動場や界面不安定性によって境界が移動する.したがって,移動境界を伴う非圧縮性熱流体問題は,界面を挟む流体間の強い非線形相互作用のために解析が難しく,特に移動境界自身が解の一部であることが,問題をより複雑化させている.a. 移動境界問題に対する数値解析手法任意形状の移動境界を追跡する方法はいくつかあるが,大きく二つに分類できる.(1) 界面追跡法(surface tracking)またはラグランジュ法界面形状に一致させるように格子を配置し,陽的に界面を追跡する.(2) 体積追跡法(volume tracking)またはオイラー法固定格子を用い,界面追跡しないが流体の物性値から陰的に界面を再構築する.. 界面の定義ラグランジュ法では界面の不連続性を保ち,次空間については(-1)次界面を陽的に追跡できるので,界面の定義や流れ場に対するモデル化は必要ない.これに対しオイラー法では,界面位置に関する詳細な情報が欲しい場合には,流体率(volume of fluid)に基づく界面位置を求めるための工夫や,二相流領域における物性値に関するモデル化が必要であり,かつ1格子セルの不確かさは避けられない.. 界面境界条件ラグランジュ法では瞬時の界面位置が既知であるため,境界条件を直接適用することができるが,オイラー法では支配方程式中の表面張力項の取扱いに工夫[例えば,CSFモデル(continuum surface force model)]が必要なため,界面幅(数セル)程度ぼかすことになる.. 解析領域の離散化ラグランジュ法では格子が界面と一致しているので,界面が変動し始めた時点で時刻ごとの格子再構成が必要で,このために補助方程式(境界適合座標系など)が用いられることがある.しかし,再構成後の格子のひずみや不均一性が大きい場合には,流れ場の解析精度の低下をもたらす可能性がある.一方,オイラー法は固定格子を用いているので,格子再構成が不要という利点がある.しかし,界面が変形した場合に精度の良い結果を得るためには,より細かい格子を局所的に適用する,解適合格子等を用いる必要があろう.. 界面変形界面が極端に変形したり,分裂・再合一したりする場合には,ラグランジュ法では構造格子が構成できない場合や,非構造格子を用いても界面近傍の流れ場の再分配が必要であるために,界面の位相変化を取扱うことは困難である.一方,オイラー法では流体率の毎時刻での更新により,界面の分裂・合一は自動的に取扱える.. 手法の選択ラグランジュ法かオイラー法かは適用する問題に依存するが,界面についてそれほど詳細に扱わなくともグローバルな流れ場が知りたい場合には,オイラー法が推奨できる.一方,界面の不連続性を保ちつつ界面の挙動を知りたい場合には,ラグランジュ法が便利であるが,界面の位相変形が大きい場合には問題がある.しかし,両方の利点を組合わせることで,よりはん用的で精度の高い手法が構築可能である.b. 境界適合座標系による変換法本手法では,不規則な物理境界は,境界適合座標系(構造格子)により計算空間へ写像変換し,写像空間内で流体方程式を解き,界面を追跡する.界面が多値関数(例えば,オーバハング形状や分裂液滴・気泡発生)となる場合には,単純な写像は困難となるが,界面追跡法としてラグランジュ法を組合わせた手法も開発されている[E-L(Eulerian-Lagran-gian)法,deformed-cell法,MAC法].c. 境界要素法(BEM : boundary element method)本法は問題の次元を下げられるため,フロントトラッキング法としてデンドライト解析やサフマン・テイラー・フィンガー解析など広はんに用いられており,線形場の解析には威力を発揮しているが,対流を伴う非線形問題や,時間依存境界条件をもつ界面変形を伴う物理問題への適用には,いまだ成功していない.d. オイラー法(体積追跡法)a項で述べたように,体積追跡法は界面追跡法と本質的に異なり,陰的に界面を捕獲し,毎時間ステップで界面再構築する.アメリカのロスアラモス研究所で開発された,流体率の輸送方程式をナビエ・ストークス式と連成して解くSOLA-VOF(solution algorithm)は,ソースプログラムが一般公開されている.しかし,VOF法(volume of fluid method)で複雑な界面や気泡を輸送した場合,特別な工夫を施さない限り,例えば円形または球形の気泡は,四角形または直方体状へ変形してしまう.これは,VOF法が気液二相流輸送について,局所移動体積を完全に保存するドナーアクセプタ法という,特殊であるが高精度な離散化手法を用いながら,界面こう配の輸送を無視したことに起因している.界面こう配を考慮しない流体率輸送法はSLIC(simple linear interface calculation),考慮した流体率輸送法はPLIC(piecewise linear interface calculation)と呼ばれ,最近はPLIC系統の手法[TELLURIDE,MARS(multi-interfaceadvection and reconstruction solver)]が発展している.e. 標位関数法(level-set method)およびCIP 法界面の時間発展問題への新しい解析法として,標位関数法が開発されている.この標位関数は界面からの距離を一定に保つ関数であるため,静止している界面の場合には正確な界面位置を与えるが,流体の渦運動に伴って界面が移動する際には距離関数の性質が崩れるため,移動前後の界面で囲まれた領域の面積・体積の保存が保証されない可能性が指摘されている.また,CIP法では,非保存形流体方程式に界面で連続なカラー関数(または密度関数)と,ディジタイザ(tangent変換)の導入によって界面を捕獲している.界面でのカラー関数分布がディジタイザ関数形と整合しない場合や,微小な気泡や液滴が発生した場合には,一部の微小気泡や液滴が消失する可能性があり,CIP法も質量・体積保存の問題が指摘されていたが,最近保存形式の採用により保存性が向上しつつある.f. 非構造境界適合移動格子法本手法は大変形する界面の運動はオイラー法で扱い,界面自身は非構造境界適合移動格子を用いて追跡する方法である.このため界面の移動に伴い,重み付き関6―105数を用いた境界適合格子の界面近傍での再配置が必要であるが,界面が一つのオイラー格子内に複数存在する場合も取扱いできる利点がある.しかし,重み関数の選び方に任意性があり,数値拡散の混入は避けられない.この系統の解析法としては,有限要素法を移動境界問題へ適用した例もある.g. 相分布関数モデル(PFM : phase field model)元来,固体の凝固解析について開発されたモデルである.オイラー格子上で相分布関数を定義(固体内で1,液相で0)し,多成分系のヘルムホルツ自由エネルギー(局所濃度と局所濃度こう配の総和で表され,二重井戸型の関数形)を記述する,カーン・ヒラード(Cahn-Hilliard)方程式を解く方法である.しかし,本法は平衡系に対して構成されており,非平衡系に対しては,二重井戸型モデルやエネルギー最小原理の適用可能性など,検討すべき課題が残っている.
