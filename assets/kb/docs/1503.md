# 8・1・8 CIP 法(cubic interpolated pseudo-particle method)

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・1 有限差分・有限体積法による解析

現代の最先端技術は、異なった相にある物質の間の相互作用に関係することが多くなってきている。例えば、半導体、液晶、光磁気ディスクなどではレーザを用いて超微細加工を行う。固体にレーザを照射すると、加熱により固体が溶けて、応力に支配された固体から粘性に支配される液体へと相転移する。さらに加熱されると今度は蒸発が始まり、このときの密度変化は実に2けたから3けたにもなる。このような変化が、数ミクロンオーダの薄い層で起こるが、明確に定義された相境界がない。いままで固体であったものが液体になり、気体になって飛散していく。相変化を伴わない現象でも、例えば液滴や気泡が微粒化したり合体したりするような過程がある。これは、単純に液体表面の問題だけではなく周囲の空気の流れも大きな影響をもつので、分裂合体する複雑な境界をもつ液体と接する空気まで取扱う必要がある。現在求められているのは、こうした意味の固体・液体・気体を同時に解く数値解法である。問題を解く有効な手段として認識されているCIP法について概説する。a.
