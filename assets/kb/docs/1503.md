# 8・1・8 CIP 法(cubic interpolated pseudo-particle method)

> **章**: 第8章 非圧縮性流れ解析手法

現代の最先端技術は、異なった相にある物質の間の相互作用に関係することが多くなってきている。例えば、半導体、液晶、光磁気ディスクなどではレーザを用いて超微細加工を行う。固体にレーザを照射すると、加熱により固体が溶けて、応力に支配された固体から粘性に支配される液体へと相転移する。さらに加熱されると今度は蒸発が始まり、このときの密度変化は実に2けたから3けたにもなる。このような変化が、数ミクロンオーダの薄い層で起こるが、明確に定義された相境界がない。いままで固体であったものが液体になり、気体になって飛散していく。相変化を伴わない現象でも、例えば液滴や気泡が微粒化したり合体したりするような過程がある。これは、単純に液体表面の問題だけではなく周囲の空気の流れも大きな影響をもつので、分裂合体する複雑な境界をもつ液体と接する空気まで取扱う必要がある。現在求められているのは、こうした意味の固体・液体・気体を同時に解く数値解法である。問題を解く有効な手段として認識されているCIP法について概説する。初期のプロファイル[図8・14(a)の実線]は、連続体では図8・14(a)の破線のように移動する。このとき、次の時刻でのメッシュ点上の値は黒丸となっており、厳密解は破線である。しかし、計算機上のメモリには図8・14(b)の黒丸しか記憶されない。この黒丸を単純につないでしまうと、図8・14(c)のようになる。このように、有限の点(メ(a) 圧力ベース解法による統一解法固体・液体・気体を統一的に一つの手法で解くためには、すべての物質を統一した方程式系を構成しておく必要がある。この三つの物質を記述するために、次のような流体力学の方程式を用いる。CIP法では、流体方程式を式(8・145)のような非保存系で解くために、非圧縮・圧縮性流体を同時に解くことが容易となる。低密度側では∝ρ(理想気体)になっているが、固体(液体)密度付近で急峻である。そこで、状態図を90°回転してみると、いままで急峻な壁であったものが今度は平坦な台地となる(ρが小さい)。この発想の転換により、非常に大きな音速に起因する問題が解決できる。低密度の気体の状態では圧力は密度、温度にほぼ比例しているので、ここで順序を入れ替えても問題は起こらない。そこで、密度や温度が変動するときに、圧力がどのように変動するかを予測する。ρ,についても同様である。このようなスキームは圧力ベース法(pressure-based scheme)と呼ばれており、ハーローらによって提案されたICE(implicitcontinuous Eulerian)が基礎となっている。これを発展させたのがPISO(pressure implicit with splitting of operators)である。一方、非保存系を採用したのが矢部らによるCCUP(CIP-combined unified procedure)法で、移流の処理を分離して解いたために、圧力方程式が簡単に求まり、安定である。これに遅れること約1年、ツィエンキヴィッツ(Zienkiewicz)らは、このような分離解法を有限要素法に適用し、有限要素バージョンのCCUPともいえる方式を提案した。セミ・ラグランジュ法では、移流項に対してCFL(Courant-Friedrich-Lewy、クーラン・フリードリッヒ・ルーイ)条件に縛られない高速解法が存在するので、これを非移流項と同じ手法にしてしまうのはせっかくの特徴が生かせない。これが状態図を90°回転した手順である。また、式(8・149)は大きな密度差がある境界面において、∇ρの連続性が保証されているという点で、非常に良い形をしている。∇ρはすなわち加速度であるから、これが境界面の両側で連続であることは必須で、例えば境界面の両側で3けたも密度が変化した場合、∇ρの分母が3けた変化するために、これをよほど正確に計算しておかないと大変なこととなる。したがって、ICEのような手法では大きな密度差を取扱うことは難しい。この手法を二次元キャビティフローの問題に適用したのが、図隔直交格子を用いた。上面の壁が1の速度で移動し、すべての壁ではノンスリップの境界条件を課している。動粘性係数は1/5000で、したがってレイノルズ数は5000である。ギア(Ghia)らによる256×256の多重格子による結果も示したが、30×30でほぼその変化を再現しており、60×60では詳細なプロファイルまでほぼ完全に再現していることがわかる。以上の多次元手法を追求すれば、ブラソフ・ボルツマン方程式の解に適用できる。これらの方程式は六次元位相空間の移流方程式である。また、粒子数は完全に保存しなければならない。移流速度が一定ならば、CIP法は完全に粒子数を保存する。ブラソフ方程式の移流速度は残念ながら一定ではないが、その空間方向の移流速度はその空間依存性がないというような面白い性質をもっている。これをうまく利用すると、完全に粒子数を保存できる手法を構成できることが中村らによって示された。また、こうした超多次元の場合には、できるだけ項数の少ない補間法が望まれる。
