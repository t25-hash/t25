# 8・2・1 有限要素法の概要

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・2 有限要素法による非圧縮流れの解析

流体解析では,有限差分法,有限体積法あるいは有限要素法(FEM : finite element method)がよく使われる.その中で,有限要素法は非構造格子を用いているため,構造格子を基本としている有限差分法と比較して,複雑な形状に適しているといえる.また,基礎方程式の弱形式の段階で,圧力境界条件などの自然境界条件がフラックス値として組入れられているため,自然境界条件の取扱いが比較的容易である.このようなことから,特に複雑な形状を有する工学的な流体解析の応用問題については,有限要素法が用いられ6―108計算力学ることが多い.有限要素法では,まず与えられた微分方程式系に対して,変分原理あるいは重み付き残差法を適用して,積分形の方程式を導き出す.ナビエ・ストークス運動方程式は非線形方程式で変分原理による汎関数を導くことができないことから,流体解析では重み付き残差法を用いる.一般に有限要素法による離散化の手順は次のようにまとめられる.(1) 有限要素による解析領域の分割任意の形状をもつ有限な大きさの三角形や四角形(三次元では四面体や六面体)で,図8・17に示されているように解析領域Ωを分割する.(2) 弱形式化微分方程式に重み関数を掛けて積分し,重み付き残差方程式を導出する.この際に2階の微分項については,部分積分を用いて1階の微分項と境界積分項に式を変形し,境界条件を考慮して弱形式化の方程式を導き出す.(3) ガラーキン法による近似各有限要素に対して,節点での値を用いて補間する方程式を作成する.その場合,変数と重み関数に同じ補間関数を用いるガラーキン法を用いるのが一般的である.一方,移流項が粘性項に対して卓越した場合,風上化(あるいは上流化)を導入する必要があるが,その際にはペトロフ・ガラーキン法,GLS(Galerkin leastsquare)法などを用いる.(4) 要素ごとの有限要素方程式の作成弱形式化した重み付き残差方程式に補間関数を導入して,各要素の有限要素行列式を作成する.(5) 全体化各有限要素の有限要素方程式を重ね合わせて,解析領域全体の離散化方程式を得る.(6) 大規模な連立一次方程式の計算大規模な連立一次方程式を解いて,解析領域の近似解を求める.(1)から(3)のプロセスは,有限要素法による空間離散化での特有なプロセスである.一方,(4)から(6)のプロセスは,有限差分法を用いて離散化方程式を解く場合と同様である.有限差分法との唯一の違いは,(6)のプロセスで導かれる大規模連立方程式が有限差分法の場合は帯行列になるのに対して,有限要素法の場合は疎密行列となるため,行列解法が異なる点である.また,疎密行列であることから,有限差分法と比較して一般に計算容量・時間が長い.いずれにしても,(1)から(3)のプロセスは有限要素の鍵であることから,以下の項で順を追って詳細に解説する.
