# 8・2・3 有限要素法による空間離散化

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・2 有限要素法による非圧縮流れの解析

有限要素法では、図8・18に示すようなさまざまな要素を使用す三角要素四辺形要素(a) 二次元解析四面体三角柱六面体(b) 三次元解析図8・18 有限要素の種類ることができるため、複雑形状への適合性がよい。各有限要素で、節点値を補間する関数を用いて近似解を求める。この際に、変数と同じ補間関数を重み関数に用いるガラーキン法を用いるのが一般的である。非圧縮性の流体解析では、速度と圧力の補間関数の組合せが不適当であると圧力に振動が発生するため、補間関数の選択には制約がある。この条件は下限上限条件(inf-sup condition)あるいはバブシカ・ブラツィ(Babuska-Brezzi)の条件と呼ばれており、圧力の自由度に比べて流速の補間次数が十分あることを要求される。ここで具体的な例として、三次元の六面要素を用いる場合を取上げる。各要素補間関数は次式で表される。これらの要素補間関数を、弱形式化された式(8・157)と式(8・159)に代入することにより、ガラーキン法による有限要素行列式が導かれる。これらの有限要素行列をマトリックス表示すると、以下のようになる。
