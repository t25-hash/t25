# 8・2・4 連続の式とのカップリングと時間積分法

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・2 有限要素法による非圧縮流れの解析

離散化された連続の式とナビエ・ストークス方程式(8・164)(8・165)を解く手法には、さまざまなものが提案されている。おもな手法としては、二つの基礎方程式を代数的な式操作によってカップリングする手法と、二つの式を直接カップリングさせず連立させて解く手法に大別される。前者はMAC法をはじめとしてSMAC法、フラクショナルステップ法などが提案されている。後者は、有限要素法の特徴である重み付き残差法に特有の解法である。基礎方程式のカップリング手法は、場合によっては時間進行法や計算の安定化手法とも密接に関連している。本項ではカップリング手法の代表として、有限要素法によるMAC法の定式化について説明し、その後に二つの基礎方程式を直接連立させる解法について説明する。a. MAC法による定式化MAC法はmarker and cell法の略称であり、1965年にウェルチ(Welch)らによって提案された、自由表面を有する非圧縮性粘性流れを解く数値解法であった。しかし、今日では自由表面の有無にかかわらず、ナビエ・ストークス方程式と連続の式とのカップリング手法を指して、MAC法と呼んでいる。b. 重み付き残差法による直接解法直接解法では連続の式とナビエ・ストークス方程式の両方を含めた、以下の弱形式化した式を用いる。しかし、この段階では重み関数と変数との積が零であるため、行列を解いて解を得ることは非常に困難である。ここで、τはPSPG(pressure-stabil-ized Petrov-Galerkin)項であり、この手法をPSPG 法という。実際はナビエ・ストークス方程式に含まれる非線形の対流項を線形化するため、式(8・169)の変数をニュートン・ラプソン法に従って増分分解して解く。以上により、時刻における変数の解を得ることが可能になる。方程式中の時間微分項に対しては、あらゆる時間進行法が適用可能である。その中でも特に有限要素法の特徴を生かしたものとして、時間空間有限要素法(space-time finite ele-ment method)が提案されている。時間空間有限要素法はその名前のとおり、時間方向にも有限要素分割を行う手法であり、本項に紹介した方法と同様に、ナビエ・ストークス方程式と連続の式とを直接連立させて解いている。
