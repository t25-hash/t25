# 8・2・4 連続の式とのカップリングと時間積分法

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・2 有限要素法による非圧縮流れの解析

離散化された連続の式とナビエ・ストークス方程式(8・164)(8・165)を解く手法には,さまざまなものが提案されている.おもな手法としては,二つの基礎方程式を代数的な式操作によってカップリングする手法と,二つの式を直接カップリングさせず連立させて解く手法に大別される.前者はMAC法をはじめとしてSMAC法,フラクショナルステップ法などが提案されている.後者は,有限要素法の特徴である重み付き残差法に特有の解法である.基礎方程式のカップリング手法は,場合によっては時間進行法や計算の安定化手法とも密接に関連している.本項ではカップリング手法の代表として,有限要素法によるMAC法の定式化について説明し,その後に二つの基礎方程式を直接連立させる解法について説明する.a.
