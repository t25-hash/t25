# 8・2・5 安定化

> **章**: 第8章 非圧縮性流れ解析手法
> **節**: 8・2 有限要素法による非圧縮流れの解析

流体の数値解析では,移流項が卓越した場合には数値不安定が起こる.そこで,数値不安定を回避するために,さまざまな風上化や安定化手法が提案されている.ここではその中の代表的なものとして,SUPG(streamline-upwind Petrov-Galerkin)法と,GLS(Galerkin least square)法を紹介する.a.
