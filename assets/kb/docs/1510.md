# 9・1・1 保存則と基礎方程式

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・1 圧縮性流体の方程式

圧縮性流体の基礎方程式導出では,検査体積(control vol-ume)を導入して,質量,運動量や全エネルギーなどの保存変数(conservative variables)に対する保存則(conservation law)を考える.a. 保存方程式の導出質量保存則を例にして保存方程式の導出を示す.独立変数として時間と座標,,をとり,従属変数として密度ρと流速,,を考える.図9・1に示された検査体積Ωの表面を옿Ωで表し,簡単のためにΩは静止しているとする.Ωに含まれる全質量はρddd,옿Ωを通って単位時間に流出する正味の質量はρ욈dで与えられる.ただし,욈=++욨であり,옿Ωの外向き単位法線ベクトルを=,,욨とする.Ωに含まれる全質量の時間変化率を,옿Ωから単位時間に流出する正味の質量に関係づけると,積分形式(integral form)の質量保存則옿옿ρddd+ρ욈d=0(9・1)を得る.運動量保存則やエネルギー保存則でも,同様の手順で基礎方程式が導出される.なお,これらの基礎方程式系は,状態方程式(equation of state)を与えることによって閉じる.積分形式で表現される保存則では,Ω内部や옿Ωに,保存変数の不連続面(discontinuous surface)が存在しても構わない.しかし,発散定理(divergence theorem)から表面積分を体積積分に変換して,Ω内で成立する保存則の偏微分方程式を求めるには,保存量の滑らかさが要求される.いまΩ内での滑らかさを仮定すると,옿ρ옿+옿ρ옿+옿ρ옿+옿ρ옿ddd=0を得る.上式が任意のΩで成立するには,被積分関数が至るところで零でなければならない.これより微分形式の質量保存則として,次式を得る.옿ρ옿+옿ρ옿+옿ρ옿+옿ρ옿=0(9・2)b. 微分形式の保存則の拡張と積分形式への対応Ω内の滑らかさを仮定して得られた偏微分方程式で不連続面を取扱うには,偏微分方程式を弱形式(weak form)に拡張して,不連続面を弱解(weak solution)として表現しなければならない.弱形式への拡張では,発散定理を逆にたどるために,流束関数の発散が零になる形式で偏微分方程式が表現されていること,すなわち保存形式(conservative form)で表現されていることが重要である.c. 時間的に変形する検査体積での保存則Ωが時間的に図9・2 時空間における検査体積Ωとその表面옿Ω(空間二次元の例)図9・1検査体積Ωとその表面옿Ω(:流速ベクトル,:単位法線ベクトル,d:面積要素)変形する場合は,図9・2に示されるように,時空間でΩを定義して保存則を構築するとわかりやすい.ALE(arbitrary Lagran-gian-Eulerian)法や移動格子(moving mesh)法など,計算格子が時間的に変形する場合には重要である.d. 曲線座標系への拡張曲線座標系(curvilinear coordi-nate system)への拡張は,反変速度(contravariant velocity)を導入した形式が一般的である.例えば,曲線座標系に変換された保存形式の質量保存則は,옿옿ρ+옿옿ξρ+옿옿ηρ+옿옿ζρ=0(9・3)で与えられる.ただし,曲線座標系への写像をξ=ξ,,,η=η,,,ζ=ζ,,とする.また,反変速度を=ξ+ξ+ξ욨,=η+η+η욨,=ζ+ζ+ζ욨で定義した.なおは写像のヤコビアン(Jacobian)であり,ξ,ξ,ξ욨等は座標変換の計量(metric)を表す.
