# 12・8・3 剛性方程式の導出

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・8 メッシュフリー法

ここでは，線形弾性体におけるEFGの剛性方程式の導出につい
て述べる．基本的な構造はFEMと同様であるので，ここでは異な
る部分について示す．
(１) FEMの場合，要素内の変位は，要素を形成する節点の値
から内挿され，節点位置での値は節点値に一致するのに対して，
EFGの場合は，移動最小二乗法を用いて近似されるため，節点位
置における近似関数の値は，必ずしも節点値に一致しない．
(２) 領域積分は，FEMでは分割された要素単位で行われ，要
素内の積分は要素形状から定められる積分点を用いて行われる．
EFGの場合は，要素というものが存在しないため，積分点は，一
般的には解析領域とは別に定義した積分のためのセルを利用して行
われる．
a.境界条件処理 EFGにおける拘束条件処理については，ラ
グランジュ未定乗数法（Lagrange multiplier method）をはじめ
として種々提案されている が，ここでは，最も簡易で，広く
用いられているペナルティ法（penalty method）について紹介す
る．
EFGでは，12･8･2項で述べたように，離散化された節点値を拘
束するだけでは，節点位置における近似関数の値を拘束することは
できない．したがって，離散化する以前のエネルギー原理の時点
で，その拘束条件を導入する必要がある．そのため，トータルポテ
ンシャルエネルギーは，変位拘束条件を付加して式(12･30)のよう
に表すことができる．

b.領域積分 EFGはガラーキン法に基づいているため，最終
的には，全体領域積分を実施しなければならない．FEMでは，こ
の領域積分は要素単位で行われて，合算される．一方，EFGでは
要素そのものが存在しないために，領域全体をFEMのように効率
良く積分することはできない．そのためベリチコ（Belytschko）
らは，解析対象の領域と積分のための領域を区別し，図12･32に示
すような，バックグラウンドセル（background cell ）と呼ばれる
積分のためのセルを提案した（図12･32) ．領域積分の際には，
それぞれのセルでガウス積分を行う．境界近傍を精度良く求めるに
は，セルを細かくするか，あるいは一つのセルに対して積分点を多
くする．この場合，積分点が領域に含まれているか否かで，数値積
分における重み値を零にすればよい．数値積分の際には，積分点を
中心として影響半径ρの円を描き，影響領域となる円の内側にあ
る節点に対し，移動最小二乗法に基づく形状関数を求める．このと
き，影響領域が境界を越えた場合，境界の外に節点は配置されない
ので，影響領域としては考慮されない．
c .その他 EFGをはじめとして多くのメッシュフリー法で
は，要素というものが存在しないために，比較的容易にき裂のよう
な不連続性を解析モデルの中に考慮することができる．例えば
EFGでは，き裂をまたいで影響領域をとらないようにすることで
不連続場を表現することができ，さらには式(12･22)の基底関数と
して，特異関数を用いることで，より精度良くき裂まわりの解析を
行うことが可能である．
そのほかにも，FEMと比してEFG（メッシュフリー法）の利点
は多々あり，表12･2にそれらを不利な点と併せて示す．
