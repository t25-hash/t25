# 9・2・2 一般座標系における圧縮性ナビエ・ストークス方程式

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・2 構造格子による解析

圧縮性流体の数値計算では、壁面境界層や衝撃波といった計算領域全体のスケールと比べると、非常に狭い領域に物理量変化が集中し、これらの再現性が解の質を大きく左右する。このため、これらの領域に計算格子を集中させ、物理量変化の少ない領域には粗な格子を用いることが、計算効率上必要となる。構造格子によって圧縮性流体を解析する場合には、格子線を境界形状に沿わせた一般座標系(generalized coordinate system)を用いるのが現在の主流となっている。具体的には、デカルト座標系(Cartesian coordinatesystem)での計算領域[これを物理空間(physical space)と呼ぶ]に不等間隔の構造格子を生成し、これを等間隔直交格子で張られた一般座標系の計算領域(計算空間、computational space)に写像して、計算空間において支配方程式を解くことが行われる(図あるが、計算空間での格子が等間隔であるため、高精度の離散化手法を容易に利用できることが利点となっている。デカルト座標系で記述された圧縮性ナビエ・ストークス方程式(compressible Navier-Stokes equation)を、一般座標系での方図9・4 一般座標計算の概念程式に変換するためには、次式で与えられる連鎖法則(chainrule)が用いられる。ξ,η,ζ等は座標変換におけるセル体積比を表しており、メトリクス(metrics)と呼ばれる。連鎖法則を用いることにより、一般座標系でのナビエ・ストークス方程式は、以下のように表される。なお、座標系が時間とともに変化しない場合には、=τ,ξ=η=ζ=0であり、空間微分だけを考慮すればよい。以上の支配方程式を離散化して数値計算を実行する前に、各格子点におけるメトリクスを計算しておく必要がある。式(9・14)の逆変換を考えると、メトリクスには以下のような関係があることが容易に求められる(ただし、煩雑さを避けるため、ここでは格子の時間変化がない場合について示す)。ただし、計算空間への写像は任意であるため、計算空間における格子間隔を1にとって計算負荷を減らすのが通例である(すなわち、ξ=η=ζ=1)。なお、メトリクス計算に二次精度中心差分を用いておけば、通常の流れ場で問題の起こることはほとんどないが、より高精度で厳密な計算に興味のある読者は、文献(20)などを参照していただきたい。
