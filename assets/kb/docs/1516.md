# 9・2・3 時間進行法

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 ).この方法は,極端な複雑形状への適用性に関してはやや難が

時間進行法(time marching method)には,空間微分項を現時刻の既知データだけを用いて評価する陽解法(explicit method)と,空間微分項を次時刻の未知データを用いて評価し,行列の反転を必要とする陰解法(implicit method)とがある.以下では,これらの代表的な手法を紹介する.a. 陽解法陽解法としては,オイラー前進差分法,ルンゲ・クッタ法,有理ルンゲ・クッタ法,マコーマック法,アダムス・バシュフォース法などがおもに用いられている.オイラー前進差分法,ルンゲ・クッタ法,有理ルンゲ・クッタ法については2・5節で紹介されているので,ここではマコーマック法とアダムス・バシュフォース法について解説する.. マコーマック法(MacCormack method)マコーマック法は,1ステップの時間進行に対して,予測子(predic-tor)と修正子(corrector)を順に用いて,二段階に時間進行を行うものである.また,空間微分に対して,予測子段階では前進差分,修正子段階では後退差分を使用する点が特徴となっている.例えば,一次元オイラー方程式옿옿+옿옿=0(9・20)を例にとると,以下のように表される.予測子:욡=욡욗-욡용욗-욡욗(9・21)修正子:욡=욡-욡-욡욪(9・22)욡욗=12 陰解法陰解法としては,オイラー後退差分法,クランク・ニコルソン法,ビーム・ウォーミング法,近似因子化法,LU分解法,アダムス・モウルトン法などが利用されている.オイラー後退差分法,クランク・ニコルソン法,ビーム・ウォーミング法については2・5節で紹介されているので,ここでは近似因子化法,LU 分解法,アダムス・モウルトン法について解説する.. 近似因子化法(approximate
