# 12・9・6 ウェーブレット解析の適用

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・9 ウェーブレット

a.ノイズの除去 信号など一連の時刻歴データが急変動部分
を含む場合，そのフーリエスペクトルは，当然のことながら高周波
成分を多く含む．このようなデータに高周波ノイズが混入した場
合，そのフーリエ変換に基づくノイズの検出は容易でない．一方，
ウェーブレットは実領域（時間域）と周波数域の両者に対して局所
性を有するので，ウェーブレット展開係数の分布より，データ本来
の急変動成分とノイズとの判別が比較的容易に行える．データをウ
ェーブレット展開した後，ノイズに対応するウェーブレット係数の
み切捨て，データを再構成すれば，ノイズを適切に除去することが
可能となる．
b.データの圧縮 12･9･4･ b項のように，ウェーブレット展開
はデータを低解像度の関数近似と，その補正量とに分解する．デー
タの高周波成分は，その急変動部分に対応するので，そのような箇
所がデータ全体にわたり密に分布していない限り，一般にウェーブ
レット展開係数の主要成分は，比較的解像度の低いレベルに分布す
る．その結果，高解像度レベルにおけるウェーブレット展開係数の
うち，絶対値の小さな成分を切捨て，零に置換えることで，データ
の主要な情報を損なうことなく，効率良くデータ圧縮を行うことが
できる．
c .境界要素解析への適用 ウェーブレットによるデータ圧縮
は，境界要素方程式の係数行列の圧縮にも適用できる．境界要素法
の詳細については5章に述べられているので，その詳細には触れな
いが，当該手法の第一の特徴は離散化にある．境界値関数が未知量
であるため，要素分割は対象領域の境界に対してなされる．そのた
め，有限要素法などの領域型解法に比べ，未知数を大幅に削減する
ことができる．しかし，求解方程式の係数行列が一般に密行列とな
るため，係数行列成分の記憶と連立方程式の求解の両面において，
決して効率の良い手法とはいえない．
そこで，境界要素方程式の係数行列にウェーブレット変換を施
し，絶対値の小さな成分を切捨てることで，計算の効率化を図るこ
とができる．ただし境界要素解析では，係数行列を作成する前に圧
縮する必要がある．そのため，境界値関数の近似基底としてウェー
