# 12・2・1 発生条件

> **章**: 第12章 キャビテーション
> **節**: 12・2 キャビテーションの発生

キャビテーションは液体の速度が増加し，流れ場の局所圧力があ
る臨界圧より低下すると，液相が気相に相変化してキャビテーショ
ン気泡が発生する現象である．この点では液体の温度がその圧力の
蒸発温度より高くなったときに発生する沸騰現象とは相異がある．
キャビテーションが発生すると，流体機器の性能を劣化させたり，
材料表面に壊食（erosion）を引き起こしたりするので，機器の設
計や信頼性の観点から考慮すべき重要な課題となる．
一般に，キャビテーションは液中に存在する微小なキャビテーシ
ョン核（cavitation nuclei ）を引き金として初生（inception）し，
その最も活性なものとして直径10～100m程度の空気泡からなる
気泡核の存在が知られている．したがって，主要な発生条件には次
のものがある ．
(１)液中に活性な気泡核が存在すること
(２)局所圧力が蒸気圧より低いこと
(３)気泡核が目に見えるほどの大きさに十分成長できる滞在時間
がとれること
いま，半径 の球状気泡の準静的な安定性を考えると，気泡内外
の圧力が釣合っているとき，気泡内の蒸気および非凝縮ガスの分圧
と表面張力を考慮すると式(12･1)が成り立つ．
