# 9・3・3 検査体積

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・3 非構造格子による解析

有限体積法において、検査体積Ωは流れ全体をすきまなく覆っていなくてはならない。また、検査体積どうしが不規則に重なり合っていてもいけない。これは、質量や運動量を保存するという流体の運動方程式の基本を、有限体積の集まりでの離散近似でも厳密に守らなければならないためであり、逆に保存則を守ることができれば、検査体積の選び方は自由である。したがって、有限体積法は非構造な格子にも適用でき、二次元では三角形要素格子のほかに、四角形や多角形要素が混じったものも可能である。三次元では四面体、六面体、プリズム要素などを含んだ格子が用いられる。代表的な検査体積のとり方を図9・6に示す。図9・6(a)ではセルそのものを検査体積としている(セル中心法、cell-centeredscheme,cell-centered method)。図9・6(b)や(c)の場合は格子頂点に物理量をおく(セル頂点法、cell-vertex scheme)。図9・6(b)では、隣接する節点の検査体積は互いに重なり合っており、検査体積の総和は、二次元三角形格子では全計算領域の3倍になる。また図9・6(c)では、三角形セルの重心と辺の中点を結ぶ線で囲まれた多角形領域を検査体積と定義する[二重格子(dualmesh)検査体積とも呼ばれる].三次元四面体格子での二重格子は、セルの重心、面の中心、辺の中点を結んでできる多面体として定義される。(a) セル中心法(b) セル頂点法(重合検査体積)(c) セル頂点法(二重格子検査体積)図9・6 非構造格子の検査体積有限体積法では流束の演算を検査体積の境界で行うため、検査体積の境界の数で全体の演算量が推定できる。検査体積境界の数は、構造格子では節点の数の2倍(三次元では3倍)となる。したがって、流束演算の数も節点数の2倍だけ行われる。一方、非構造格子では、セル形状により節点数、辺の数、要素の数の関係が大きく変わるため、検査体積のとり方により演算量も大幅に異なってくる。二次元空間を三角形で離散化した場合の節点の数、辺の数、要素の数の関係は、以下のようになる。つまり、同じ節点数で空間を離散化した場合、構造格子(または四角形セル)に比べ三角形セルの非構造格子法では、1.5倍の流束演算を行わなければならないことを意味している。三次元空間を四面体で離散化した場合、節点数、辺の数、面の数、および要素の数の関係は、次のようになる。したがって、セル中心法の演算量はセル頂点法に比べ2倍近く必要である。六面体セルを用いる構造格子に比べると、同じ格子点数で四面体のセル頂点法では3~4倍、セル中心法では6~7倍の演算量が必要となる。ただし、四面体のセル中心法では物理量の定義点も多くなるため、演算量だけでセル中心法とセル頂点法、四面体と六面体の優劣を議論することは難しい。
