# 12・3・2 気泡運動に及ぼす圧縮性，気体拡散，熱拡散の影響

> **章**: 第12章 キャビテーション
> **節**: 12・3 気泡の力学

全溶解時間は気泡初期最大半径 の二乗に比例し，常温水中のなると溶解時間は減少するが，キャビテーション気泡の運動周期にc.熱拡散の影響 崩壊末期の気泡内部，高温液体中の気泡挙の式を解く必要がある．
しかし，液中の温度境界層が薄い条件で無限遠方での液体温度．
を用いて気液界面での蒸発・凝量およびエネルギーの各保存式を直接数値的に解いて気泡運動に及a.圧縮性の影響 気泡表面速度が音速に近づくと圧縮性の影＝10mの気泡は約2.5s で完全に溶解する．
不飽和度が大きく比べれば非常に遅い．
動など熱的影響が著しい場合には，気泡の運動方程式とエネルギーは，次に示す球対称非定常熱伝導式のゼロ次解 が利用できる．
気泡表面マッハ数 が一次のオーダまで厳密な気泡の運動ここに， ：液体の熱拡散率， ：気泡表面での液体温度， ：縮を考慮した圧力波動が検討され，また気液両相の質量，運動ぼす内部現象の影響が明らかにされている．
