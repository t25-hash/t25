# 9・3・5 風上法

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・3 非構造格子による解析

衝撃波を伴った流れ等では風上法が有効であり、その非構造格子上での構築法を考える。式(9・68)の左辺第2項を次のように書き直す。例えばローの数値流束関数を使うと、以下のように書ける。各検査体積ではセル平均の値が与えられており、この値から検査体積境界での値,を評価しなくてはならない。セル平均値そのものをセル境界値として使う場合、つまりは区分的定数値関数を仮定する場合には、空間精度は区分幅の一次の精度となる。この検査体積は、各辺の中点とセルの重心を結んだ線で囲まれる多角形領域である。実際のプログラミングでは、9・3・4項のジェームソンの方法と同様、流束の演算は辺に関するループで効率良く行われる。
