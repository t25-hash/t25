# 9・3・6 二次精度風上法

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・3 非構造格子による解析

空間二次精度にするために、セル平均値を基に、以下の式で区分的一次関数の再構築を行ってセル境界値,を求める。こう配は、グリーン・ガウス公式を用いると次式で求められる。バース・ジェスパーセン(Barth-Jespersen)は、再構築された値がそのセルの値およびそのまわりのすべての隣接セルの値を超えないための、多次元制限関数を提案した。しかし、この制限関数は滑らかな流れの領域でも働き、定常解への計算収束性があまりよくない。この点を改良したベンカタクリシュナン(Venkatakrish-nan)の制限関数がよく用いられる。
