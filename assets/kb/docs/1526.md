# 9・3・7 時間積分法

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・3 非構造格子による解析

非構造格子上での時間積分は、構造格子上で一般的に用いられるADI的な陰的時間積分や、多重格子法(multi-grid method)の適用が困難であるため、多段階時間積分法に代表される陽的な時間積分法が用いられてきた。しかし、高レイノルズ数の粘性流れでは安定性条件が厳しく、多大な時間計算ステップが必要となるため、非構造格子上での多重格子法や陰的時間積分等のさまざまな収束加速法が提案されている。多重格子法は基本格子を基に、それより粗い格子を数段階用意し、基本格子と粗い格子との間を順次計算することで情報伝達を早めるとともに、高周波成分を除去して定常解を素早く得る方法である。非構造格子で多重格子用に粗い格子を生成するには、(1)基本格子とは独立に粗い格子を生成、(2)最初に粗い格子を生成し、それを順次細分化して最終の基本格子を生成、(3)基本格子のセルまたは検査体積セルを融合させて順次粗い格子を生成(agglomera-tion multi-grid)、などが提案されている。これらの中で(3)の方法は、粗い格子の各セルが多角形にはなるものの、(1)や(2)の方法に比べて、多段階の粗さの格子を容易に生成できる点で優れている。陰的時間積分法は、非構造格子上では疎な行列の反転に多大なメモリが要求されるため、ガウス・ザイデル的な繰返し計算法が用いられる。この中でLU-SGS 陰的時間積分法の非構造格子への適用では、大幅なメモリ増加なしに、定常解への収束状況を大幅に改善することが可能である。LU-SGS 陰解法の説明のために、離散式(9・78)の時間積分をより三次元場で一般的にして書くと、욡욗=욡욗-β욡∑욡욢욡욢욗-1-β욡∑욡욢욡욢욗(9・86)∈は検査体積境界を挟んで隣接する節点を意味し、また添字は節点との間の検査体積境界での値を意味し、욡욢はその面積である。욡は検査体積の体積を意味する。また、β=0のとき式(9・86)は陽解法になり、β=1の場合はオイラー陰解法となる。いま、=욗-욗として、非構造格子上でのLU-SGS 法の式を最終形のみ書くと、以下のようになる。前進スイープ:욡=욡-0.5β ∑욡욢욢-ρ욢(9・87・a)後退スイープ:욡=욡-0.5β ∑욡욢욢-ρ욢(9・87・b)ここに、ρはヤコビ行列のスペクトル半径ρ=+を意味し、はセル境界に対する単位法線ベクトル、は速度ベクトル、そしては音速である。また、욡=-∑욡욢욡욢욗=욡+0.5β∑욡욢ρ(9・88)式(9・87)内の∈,∈は、節点に隣接する点∈を、∈と∈の二つのグループに分けることを意味している。非構造格子でLU-SGS 解法を実現するには、式(9・87)のおよび、つまり下三角要素、上三角要素を決める必要がある。最も簡単には、節点に対しては∈[<],は∈[>]とする方法がある。文献(47)(48)では、セル中心法に対してのLU-SGS 法のための並べ替えを提案しており、また文献(50)ではセル頂点法に対する並べ替えを提案している。この並べ替えにより、上下のバランスが改善され、かつベクトル演算が可能となる。ルオ(Luo)らはLU-SGS 陰解法にGMRES を組合わせて、収束特性をさらに改善する方法を提案している。
