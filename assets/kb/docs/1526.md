# 9・3・7 時間積分法

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・3 非構造格子による解析

非構造格子上での時間積分は、構造格子上で一般的に用いられるADI的な陰的時間積分や、多重格子法(multi-grid method)の適用が困難であるため、多段階時間積分法に代表される陽的な時間積分法が用いられてきた。しかし、高レイノルズ数の粘性流れでは安定性条件が厳しく、多大な時間計算ステップが必要となるため、非構造格子上での多重格子法や陰的時間積分等のさまざまな収束加速法が提案されている。多重格子法は基本格子を基に、それより粗い格子を数段階用意し、基本格子と粗い格子との間を順次計算することで情報伝達を早めるとともに、高周波成分を除去して定常解を素早く得る方法である。非構造格子で多重格子用に粗い格子を生成するには、(1)基本格子とは独立に粗い格子を生成、(2)最初に粗い格子を生成し、それを順次細分化して最終の基本格子を生成、(3)基本格子のセルまたは検査体積セルを融合させて順次粗い格子を生成(agglomera-tion multi-grid)、などが提案されている。これらの中で(3)の方法は、粗い格子の各セルが多角形にはなるものの、(1)や(2)の方法に比べて、多段階の粗さの格子を容易に生成できる点で優れている。陰的時間積分法は、非構造格子上では疎な行列の反転に多大なメモリが要求されるため、ガウス・ザイデル的な繰返し計算法が用いられる。この中でLU-SGS 陰的時間積分法の非構造格子への適用では、大幅なメモリ増加なしに、定常解への収束状況を大幅に改善することが可能である。また、β=0のとき式(9・86)は陽解法になり、β=1の場合はオイラー陰解法となる。文献(47)(48)では、セル中心法に対してのLU-SGS 法のための並べ替えを提案しており、また文献(50)ではセル頂点法に対する並べ替えを提案している。この並べ替えにより、上下のバランスが改善され、かつベクトル演算が可能となる。ルオ(Luo)らはLU-SGS 陰解法にGMRES を組合わせて、収束特性をさらに改善する方法を提案している。
