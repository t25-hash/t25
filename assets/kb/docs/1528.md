# 9・4・1 支配方程式と有限要素スキーム

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

ここに上付添字は、転置行列を表している。また、εは内部エネルギーであり、はソース項である。有限要素法による圧縮性流れ解析においては、対流項に起因する数値的不安定性を除去し、安定で高精度な解析を行うために、大別すると三つのスキームが提案されている。一つは、ヒュージ(Hughes)らによるSUPG(streamline-upwind Petrov-Galer-kin)法であり、他の一つは、ドーニア(Donea)らによって提案されたテイラー・ガラーキン(Taylor-Galerkin)法(以下、TG 法と略記)、最後の一つは、レーナー(Lohner)らによる二段階解法である。以下では、それぞれのスキームによる有限要素法の定式化を示し、合わせて解析例を示す。
