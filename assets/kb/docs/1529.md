# 9・4・2 SUPG 法による定式化

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

a. 重み付き残差式(weighted residual equation)SUPG 法は、streamline-upwindという語句の意味するとおり、局所的な流れの方向に、数値的な安定性を確保するために必要な数値粘性を与える手法である。通常の有限要素法[バブノフ(Bubnov)・ガラーキン法]が、未知数の内挿関数と重み関数を同一に選んで定式化を行うのに対して、ペトロフ・ガラーキン法は、未知数の内挿関数と異なる重み関数を用いることにより、差分法の場合の風上差分と同様の上流化を実現する。ここには、領域内で連続な重み関数であり、は要素ごとに導入される重み関数(要素間で不連続)、は要素の総数である。ここには、1節点当たりの未知量の数を表している。非圧縮性流れ解析の場合と異なり、圧縮性流れ解析の場合には、未知量のすべての成分に対して同じ次数の内挿関数が用いられる。圧縮性流れ解析によく用いられる要素を表9・2に示す。式(9・110)(9・111)を式(9・109)に代入して要素ごとの積分を実行すると、最終的に次のような連立一次方程式が得られる。なお、下式の導出に当たっては、簡単のために粘性項は無視している。式(9・116)に示されているように、式(9・114)は非線形方程式であり、同一時刻において反復計算を行う必要がある。その一つの解法である予測子-修正子法による解析手順は、次のとおりである。ステップ(4)~(7)までの計算手順は、修正量が許容値以下になるまで繰返されるが、通常、時間増分は十分小さく、反復をしないで次の時間ステップに進む場合が多い。陽解法(explicit method)陽的な解法は、質量行列を対角化する(非対角要素を対角要素に加え合わせる)ことによって実現される。. マルチパスアルゴリズム(multi-path algorithm)質量行列を対角化することにより、効率の高い解析が可能となるが、解析精度の低下につながるおそれがある。これに対し、質量行列を対角化することにより生じる誤差の混入を避けるために、ドーニアらにより、マルチパスアルゴリズムが提案されている。
