# 12・4・3 各種流体要素のキャビテーション特性

> **章**: 第12章 キャビテーション
> **節**: 12・4 キャビテーション流れの諸特性

a.管路および絞り 管路内流れにおいて，はく離なしの場合
においても，一般の圧力損失，曲り部における遠心効果，絞り部に
おける加速効果などによる圧力低下などによって，局所的にほぼ飽
和蒸気圧（気体的キャビテーションの場合は空気分離圧）に近づい
た点においてキャビテーションが発生する．はく離流れがある場合
においては，はく離泡あるいははく離渦内の圧力まで考慮する必要
がある．その初生キャビテーション数は，さらに液中の気泡核濃度
および空気含有度によって左右される．曲り管，ベンチュリ，オリ
フィスなどのキャビテーション特性について多くの研究がある［例
えば，文献(58)～(60)］．
b.バルブ 基本的には損失を利用した流体機器であり，流路
断面積と流れ方向変化を大きくとる構造となっている．したがっ
て，急激な絞りおよびはく離流れにおいて局所的な圧力低下がもた
らされ，キャビテーションが発生しやすい状況下にある．各種バル
ブについてキャビテーション特性が調べられている ．
c .ノズル ノズルから液中への高速液体噴流は噴流界面に自
由せん断層を形成する．このせん断領域における渦核内圧力が十分
低くなるとせん断層（渦）キャビテーションが誘起され，ジェット
キャビテーション（jet cavitation）状態となる．ノズルからの流
出に伴い，リング渦あるいはスパイラル渦などの組織構造をとるこ
ともあり，キャビテーションクラウド状態となる．固体壁面への衝
突噴流状態の場合，高い衝撃圧を呈しウォータジェット加工に利用
される ．
d.軸受など 軸受，シール，回転ローラなどの狭いすきまの
粘性流れにおいては，くさび状流れや負のスクイーズ作用により負
圧が生じ，気体的あるいは蒸気的キャビテーションが発生すること
がある ．キャビテーションは軸受の安定性などに寄与する面を
もつ．荷重変動が激しいエンジン軸受などを除けば，一般に気体的
キャビテーションが主で壊食の可能性は低い．
