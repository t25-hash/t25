# 9・4・3 1点積分法(one-point integration method)

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

この計算は、解析的に行うことが難しく、ガウスの求積公式を用いて数値的に積分される。このことは、有限要素法による解析が、差分法や検査体積法による解析に比べて、二次元問題で3~4倍、三次元問題で7~8倍の計算時間を要する主要な原因となっている。これを解決するために提案された手法が、ガウス積分を1点のみで行う1点積分法である。この1点積分法を用いて計算された、非粘性完全ガス中を30°の迎え角度で、マッハ数8で飛行する二重だ円形状物体まわりの流れ場を、通常よく用いられる4点積分法による解析結果と比較して、図9・9に示す。1点積分法による計算時間の短縮効果については、二次元問題の場合3.5倍、三次元問題の場合7.5倍と報告されており、差分法などによる解析時間に比肩しうる状態になってきている。
