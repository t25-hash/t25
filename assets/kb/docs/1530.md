# 12・5・1 壊食（erosion）の機構

キャビテーション（cavitation）気泡は，気泡崩壊時にマイクロ
ジェットや衝撃波に起因する衝撃力を生じる ．図12･10にその
概要を示す．一般にキャビテーションは，水中の微細な気泡などの
キャビテーション核が高速・低圧領域において，キャビテーション
気泡として発達する．速度が低下して圧力が回復すると，キャビテ
ーション気泡が収縮してやがて崩壊する．キャビテーション気泡が
固体壁近傍で崩壊する場合には，気泡の固体壁の反対側が固体壁側
よりも変形しやすいためにへこみ，このへこみ部が1500m/s以上
の流速をもつマイクロジェットとして発達して固体壁をたたく．ま
たキャビテーション気泡が，キャビテーション核となる微細な気泡
から発達しているときには，気泡内部の圧力が低いために水中から
気体を取り込むため，気泡崩壊時には気泡内にある気体のために気
泡が収縮した後に崩壊せずに再膨張する．この気泡の収縮・再膨張
がごく短時間に行われるので，再膨張により衝撃波を生じる．この
衝撃波が固体壁に衝撃力を与える．これらのマイクロジェットや衝
撃波による衝撃力が，流体機械の材料表面に作用して壊食を生じ
る．

上記の衝撃力による材料の壊食においては，単発の衝撃力により
材料が質量損失する壊食を生じる場合と，衝撃力が繰返し負荷され
て疲労破壊的に壊食する場合があり，主として疲労により壊食する
場合が多い．図12･12に，疲労破壊的に壊食が発達する場合の質量
損失の経時変化を示す．壊食初期は，衝撃力により塑性変形が繰返
されるだけで質量損失は生じないので，潜伏期と呼ばれる．壊食が
発達すると，質量損失が生じて時間とともに壊食率が加速的に増大
する加速期，壊食が定常的に進行する定常期を経て，壊食率が減少
する減速期に至る．したがって，壊食の過程により壊食率が大幅に
異なるので，材料の耐壊食性を推定する場合には壊食期を把握する
必要がある．壊食率は，計量ごとに求める瞬間壊食率と，全損失量
から求める累積壊食率の二つを定義できるが，ASTM G134：
1995 では累積壊食率を使用して，累積壊食率が極大を示す最大
累積壊食率を材料の壊食性を示す壊食率とすることを推奨してい
る．
