# 12・5・2 壊食に及ぼす諸因子とその予測

> **章**: 第12章 キャビテーション
> **節**: 12・5 キャビテーションによる壊食と騒音

壊食には，おもに以下のような流体力学的因子と材料学的因子が
関わっている．
a.流体力学的因子 流体力学的因子としては，キャビテーシ
ョン流れの支配パラメータであるキャビテーション数，主流の流
速，流体機器の寸法，キャビテーション気泡が崩壊する領域の静圧
などがある．
キャビテーション数が小さくなるほど，キャビテーションが発
達，すなわち高壊食性渦キャビテーションが発達するために，衝撃
力が大きくなり壊食が増加する．ポンプなどの流体機械においては
NPSHがこれに相当する．したがって，キャビテーション数σや
NPSH，あるいはキャビテーションの長さ を指標として壊食
率（erosion rate） を予測する方法が提案されている ．

キャビテーション気泡の崩壊領域の静圧は，大きくなるほど衝撃
力が大きくなるので，静圧が高い領域においてキャビテーションが
発生する場合には，壊食の危険性が大となる．
以上のような流体力学的因子を個々に把握してキャビテーション
の壊食率を予測する方法とは別に，キャビテーションの強さを直接
計測して，壊食率を定量的に評価する方法がある．キャビテーショ
ンの強さの計測には，圧電材料を用いた衝撃力計測センサによる方
法 ，材料表面に生じた塑性変形ピットの深さや大きさによる
方法 などがある．また圧力に応じて発色する感圧紙 や，ペイ
ントにより壊食が発生する領域を推定する方法 もある．
b.材料学的因子 材料学的因子としては，引張強さ，硬さ，
ひずみエネルギーが考えられる．引張強さσが大きくなるほど，
また硬さが大きくなるほど耐壊食性（例えば，壊食率の逆数など）
が大きい傾向がある．また，ひずみエネルギー（1/2×引張強さ×
伸び）が大きくなるほど，耐壊食性が比較的良い相関を示すという
報告 もある．
材料の耐壊食性を定量的に予測するためには，キャビテーション
噴流式壊食試験（erosion test ) を用いて，各材料について壊食
の加速試験を行う必要がある．本法は，従来，振動式試験装置を用
いてキャビテーションを発生させていた方法 に比べて，流体機
械に壊食を生じるキャビテーションと同タイプの渦キャビテーショ
ンにより壊食を行える，強力なキャビテーションを容易に発生でき
るなどの利点がある．
