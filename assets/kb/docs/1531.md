# 9・4・4 TG 法[TG (Taylor-Galerkin)method]

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

TG 法は、移流拡散方程式などに対して、安定で高精度な解析を行うために開発された手法であり、時間に関するテイラー展開を用いて非定常項を評価する。その際に、高次の時間微分項を支配方程式を用いて評価することにより、移流項に起因する数値的不安定を除去することができる。式(9・131)に重み付き残差法を適用し、式(9・110)(9・111)の内挿関数を用いて積分を実行すると、に関する連立一次方程式が得られ、これを解いて圧縮性流れの解を得ることができる。本解法を、ナビエ・ストークス方程式の基本形である一次元移流拡散方程式に適用し、安定性解析を行った結果を表9・4に示す。
