# 9・4・4 TG 法[TG (Taylor-Galerkin)method]

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

TG 法は,移流拡散方程式などに対して,安定で高精度な解析を行うために開発された手法であり,時間に関するテイラー展開を用いて非定常項を評価する.その際に,高次の時間微分項を支配方程式を用いて評価することにより,移流項に起因する数値的不安定を除去することができる.いま,時刻욗における解욗が求められているとき,次の時刻욗における解욗は,時刻욗のまわりのテイラー展開を用いて,욗=욗+욗+(9・126)と書き表すことができる.式(9・126)を,増分形を用いて≡욗-욗=욗+α+(9・127)と書き表し,上式中のは支配方程式(9・94)を用いて変形すると,=욡욡욚-욡욡욑+α욡욡욚-욡욡욑(9・128)の非線形方程式が得られる.上式の右辺第2項は,さらに新たなパラメータβを導入することにより,α욡욡욚-α욡욡욑β욡욡욚-α욡욡욑(9・129)と一般化して書き表すとができる.このようにして得られた式を,(a) 1点積分法(b) 4点積分法図9・9二重だ円形状物体まわりの超音速流れ=86―125욡욑욡욡욚옿욡욚옿+옿욡욚옿욢욢(9・130)の関係式を用いて線形化すると,最終的に,+α욡욡-β옿욡욚옿욢욢욡+옿욡욚옿욡(9・131)が得られる.式(9・131)に重み付き残差法を適用し,式(9・110)(9・111)の内挿関数を用いて積分を実行すると,に関する連立一次方程式が得られ,これを解いて圧縮性流れの解を得ることができる.本解法を,ナビエ・ストークス方程式の基本形である一次元移流拡散方程式に適用し,安定性解析を行った結果を表9・4に示す.
