# 9・4・5 人工散逸項(artificial diffusion term)

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

圧縮性流れ解析の場合,衝撃波を伴う流れ解析のように,未知量が空間的に急激に変化する流れ場においては,解の振動を抑制するために,人工散逸項を付加することが必要になる.ナビエ・ストークス方程式に導入される人工散逸項욡욡は,次式で与えられる.+욡욡욑=욡욡욚+욡욡(9・132)ここに,욡=욡,욢である.人工散逸項としては,さまざまな形が提案されているが,ここではラピダス(Lapidus)の散逸項を紹介する.ラピダスの提案した散逸項を多次元問題に拡張すると,욡=욡ただし=욅욄욡욡(9・133)である.ここに,욅は通常0と1の間の係数であり,욄は要素の面積,욡は軸方向の速度成分である.なお,式(9・133)においては,に関する総和はとらない.
