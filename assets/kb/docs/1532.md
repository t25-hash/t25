# 12・5・3 騒音（noise）の発生と特徴

> **章**: 第12章 キャビテーション
> **節**: 12・5 キャビテーションによる壊食と騒音

キャビテーションが発生すると，キャビテーション気泡の崩壊に
伴う騒音が生じる．キャビテーションによる騒音は，個々のキャビ
テーション気泡の崩壊に伴う高周波ノイズと，12･5･1項で述べた
遷移領域に起因する低周波ノイズがある．
高周波ノイズは，気泡の崩壊・収縮による体積変動により生じる
ので，レイリー・プレセットの式より気泡が崩壊する時間τを求
めると，
τ＝0. 915 ρ
－ (12･23)
ここで， ：キャビテーション気泡の最大半径である．したがっ
て，大気圧下で＝1mmのキャビテーション気泡が崩壊すると
きはτは約0.1msとなり，10kHz程度の騒音を生じることにな
る．実際に流体機械に生じるキャビテーションは，さらに気泡径が
小さい場合が多いので，キャビテーションによる騒音は高周波であ
ることが特徴である．
逆に，高周波ノイズからキャビテーションの発生を検知すること
も可能である．騒音計により大気中で計測したり，水中マイクロホ
ンにより水中を伝ぱするノイズを計測して，キャビテーションの発
生を検知できる．なお，キャビテーション気泡は，騒音源となるば
かりでなく，気液界面で騒音を減少させる効果も併せ持つので，キ
ャビテーションの強さを計測する場合には注意が必要である．ま
た，高周波振動加速度計 やアコースティックエミッション
（AE）センサ を用いてキャビテーションの発生の程度を検出す
ることも可能である．
キャビテーションにより数十Hzの低周波ノイズを生じることも
ある．これは遷移領域において，サブキャビテーション状態とスー
パーキャビテーション状態が，数十Hzの周期で交番的に生じるこ
とに起因する．この遷移領域では，騒音ばかりでなく振動も顕著に
なる特徴がある．
