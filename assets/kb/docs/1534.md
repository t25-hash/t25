# 9・4・7 アダプティブ法(adaptive method)

> **章**: 第9章 圧縮性流体の流れ解析手法
> **節**: 9・4 有限要素法による圧縮性流れ解析

アダプティブ法には、5・3・5項に述べられているように、1節点を最適な位置に移動させるr法、2要素を再分割していくh法、3要素の高次化を行うp法がある。いずれの方法を用いるにせよ、解析領域のどの部分にこれを適用するかを判別する必要がある。これは通常、誤差評価に基づいて行われる。誤差評価は、領域内の密度や圧力、あるいはエントロピーといった、単一の物理量を用いて行われる。この誤差評価に基づいて、要素の修正が行われる。以下では、まずh法の具体例を示し、次にr法の実例を示す。
