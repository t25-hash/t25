# 10・1・2 量子分子動力学法の分類

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・1 量子分子動力学法

量子分子動力学法は、電子構造をどのレベルまで正確に計算するか、原子核の運動を量子力学的に取扱うか質点として考えるかによって、さまざまに分類される。図10・2にその分類の概要を示す。前述のように、非定常のシュレディンガー方程式を用いる必要があるのは、低温の水素分子の振動のように、原子核の運動の量子効果を考える必要がある場合と、光との相互作用などのために、電子の励起状態の時間依存を考える必要がある場合である。図10・2の右端に示すように、原子核の波動性を考える場合にも、分子振動や回転の一定の自由度のみを量子的に取扱うなどの近似を行う場合が多い。また、熱流体工学の分野で時間依存の電子励起状態を計算するのは、光との相互作用がある場合のように、極めて限られた場合となる。多くの量子分子動力学法の計算においては、原子核の運動は質点の運動方程式に従うと近似し、図10・2の左半分に示すように、式(10・4)の定常のシュレディンガー方程式に基づき、B-O近似を用いたうえで、さまざまなレベルでの電子状態の近似計算を用いる。ある原子核の配置に対して、後述の分子軌道法や密度汎関数法によって電子状態を求めると、原子核間には、原子核どうしのクーロン斥力に加えて、10・1・5項で後述するヘルマン・ファインマンの定理より導かれる電子系からのポテンシャル力が決定できる。量子化学計算で求まる電子状態から、瞬間ごとにポテンシャルを決定して原子核を動かすという発想は極めて素直であり、カーとパリネロが電子状態の計算に密度汎関数法を用い、平面波近似の波動関数を用いてその可能性を示して以来、カー・パリネロ法(Car-Parri-nello method)として化学反応過程などの計算に用いられ、かつ各種の改良が加えられている。定常シュレディンガー方程式にB-O近似を用いたうえで、原子核の運動に伴う電子状態の時間変化を求めるのは本来自己矛盾であるが、カー・パリネロ法では、仮想的な電子質量を用いて、電子密度の変化がニュートンの運動方程式に従うと仮定して解いてしまう。一般にこのカー・パリネロ法を、第一原理分子動力学法や量子分子動力学法と呼ぶ場合が多い。電子状態の計算を大幅に簡略化して化学の分野で多用されるヒュッケル法(Hu썥ckel method)、あるいは拡張ヒュッケル法で見積もり、分子動力学法のポテンシャルを計算するタイトバインディング法(tight binding method)も、一部でかなり利用されている。図10・2 量子分子動力学法の分類6―131ただし、ヒュッケル法は波動関数の計算を一定の経験的パラメータを利用して省略するために、シリコンや炭素などの限られた原子間のみについてしか良い近似とならないのが現状である。さらに近似を進めて、原子間のポテンシャルを原子核の位置のみの関数で表してしまうのが古典分子動力学法である。原子核の運動は、質点に対するニュートンの運動方程式を解くことから求まり、比較的簡単に導入が可能である。
