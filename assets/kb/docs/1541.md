# 10・1・4 密度汎関数法

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・1 量子分子動力学法

密度汎関数法(DFT : density functional theory)は,ハミルトン演算子の期待値である全エネルギーや,その他一般に演算子の期待値で与えられるさまざまな物理量を,電子密度の汎関数で表す手法である.その原型はトーマス・フェルミ近似にさかのぼるが,現在用いられているDFT の理論体系は,ホーヘンベルグ・コーン(Hohenberg-Kohn)の定理および,この定理にエネルギー汎関数の電子密度依存性の局所近似を仮定することで導かれる,コーン・シャム(Kohn-Sham)方程式に基づいている.ホーヘンベルグとコーンは基底状態に縮退がない場合,電子系に作用している外場(ポテンシャル)と基底状態の波動関数ψは,電子密度を与えると一意に決まることを示した.さらに以下に示す汎関数욋が,正しいに対して最小になることを示した.욋=ψ++ψd(10・7)ここで,は運動エネルギー,は電子間のクーロン相互作用,は外場によるポテンシャルを表す.これらの定理に基づく電子系の記述は,その基底状態に対する厳密な理論であり,この定理から基底状態の電子密度が満たすべき方程式が導かれる可能性が出てくる.コーンとシャムは,ホーヘンベルグとコーンの理論を,まず仮想的な相互作用しない電子系に対して適用することにより,その基底状態での電子密度の満たすべき方程式を導いた.これに,現実の電子間相互作用によって生じる交換相関エネルギーを考慮し,以下に示すような有効ポテンシャル윬읆읆を感じる一電子方程式を,SCF法で解く問題へと導いた.2∇+윬읆읆φ욡=ε욡φ욡(10・8)윬읆읆=+-′d′+δδ(10・9)φ욡(10・10)=∑ここで,式(10・9)では,交換相関エネルギーの具体形が与えられていない.コーンとシャムは,ある座標における交換相関ポテンシャルが,そこでの電子密度だけで局所的に決まるとする局所密度近似(LDA : local density approximation)を導入することによって,具体的なポテンシャルの形を提案した.=εd(10・11)δδ=ddε=ε+dεd(10・12)
