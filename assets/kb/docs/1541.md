# 10・1・4 密度汎関数法

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・1 量子分子動力学法

密度汎関数法(DFT : density functional theory)は、ハミルトン演算子の期待値である全エネルギーや、その他一般に演算子の期待値で与えられるさまざまな物理量を、電子密度の汎関数で表す手法である。その原型はトーマス・フェルミ近似にさかのぼるが、現在用いられているDFT の理論体系は、ホーヘンベルグ・コーン(Hohenberg-Kohn)の定理および、この定理にエネルギー汎関数の電子密度依存性の局所近似を仮定することで導かれる、コーン・シャム(Kohn-Sham)方程式に基づいている。これらの定理に基づく電子系の記述は、その基底状態に対する厳密な理論であり、この定理から基底状態の電子密度が満たすべき方程式が導かれる可能性が出てくる。コーンとシャムは、ホーヘンベルグとコーンの理論を、まず仮想的な相互作用しない電子系に対して適用することにより、その基底状態での電子密度の満たすべき方程式を導いた。これに、現実の電子間相互作用によって生じる交換相関エネルギーを考慮し、以下に示すような有効ポテンシャルを感じる一電子方程式を、SCF法で解く問題へと導いた。
