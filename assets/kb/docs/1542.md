# 10・1・5 ヘルマン・ファインマンの定理

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・1 量子分子動力学法

10・1・3,10・1・4項で述べたように、分子軌道法や密度汎関数法を用いて、B-O近似により静止している原子核まわりの電子状態を求めることができる。このときのエネルギー固有値욅は、原子核の座標をパラメータとしてもつことになる。すなわち、욅は原子核の運動に対して、ポテンシャルとして作用することになる。このように考えると、原子核が電子系から受ける力욅は以下の形で与えられる。욅=-옿옿욅=-옿옿욅ψψd(10・13)ここで、dはすべての電子に関する積分を表す。ハミルトン演算子のエルミート性と波動関数ψの規格化条件より、式(10・13)は次のように整理できる。욅=-ψ옿옿욅ψd=-ψ옿옿욅ψd(10・14)ここで、はハミルトン演算子のポテンシャル部分を意味する。式(10・14)の後半の変形は、原子核の座標욅がハミルトン演算子の運動エネルギー項には含まれず、ポテンシャル項のみに含まれることから導かれる。式(10・14)をヘルマン・ファインマンの定理という。
