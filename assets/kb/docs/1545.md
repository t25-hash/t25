# 10・2・2 ポテンシャル関数の例

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・2 分子動力学法

分子動力学法シミュレーションを行うに当たり、分子間ポテンシャルの選択あるいは作成は、最も重要な問題である。以下に、よく用いられる具体的なポテンシャルの例を挙げる。a. レナード・ジョーンズ(12-6)ポテンシャル図10・3に示すレナード・ジョーンズ(12-6)ポテンシャル(Lennard-Jones6―133(12-6) potential)は、アルゴンやキセノンなどの希ガスのファンデルワールス力をよく表すポテンシャルとして知られるとともに、気体や液体の統計力学におけるモデルポテンシャルとしてもしばしば用いられる。さらに水や高分子などの複雑なポテンシャルの要素としても、この形が用いられる。代表的な希ガスについて、これらのパラメータを表10・1に示す。レナード・ジョーンズ分子のみを用いた系では、運動方程式(10・15)は、,σ,εを用いて表10・2に示すような無次元化ができる。一定パラメータのレナード・ジョーンズ分子のみを含む計算は、無次元として表現するのが一般的であるが、物理的な解釈のために、アルゴンのパラメータ(σ=0.34nm,ε=1.67×10J,τ=2.2×10s)を用いてアルゴン換算の有次元値として表現することも多い。図10・4には、ニコラスらによる整理式より作成した相図(等圧線)を示す。実際の計算の際には、相互作用の計算量の増大を防ぐために、ポテンシャルをカットオフ半径=2.5σ~5.5σで打切る。b. 水に対する有効二体ポテンシャル現実の分子として最もよく調べられているのが水である。分子内振動は無視して、剛体回転子とした有効ポテンシャルが多数提案されている中で、現在よく使われているSPC/E,TIP4PとCCポテンシャルを紹介する。酸素分子間のレナード・ジョーンズ型のポテンシャルに加えて、これらの電荷間のクーロン力が次式のように加えられている。具体的なパラメータを表10・3に示した。このほか、ヨルゲンソン(Jorgenson)らが開発したTIP4Pポテンシャル(4 point transferable intermolecule potential)では、負の電荷をおく位置が図10・5のように、HOH の角の二等分線上で水素原子寄りの位置となっている。SPC/EやTIP4Pのパラメータは、おおよそ常温・常圧での水の物性値を再現するように、経験的に調整されたものである。TIP4Pの特色は、アルコールなどのさまざまな液体高分子に対して、ヨルゲンソンが開発したOPLS ポテンシャル(optimized potential for liquid simulationspotential)の一部である点である。さらに、第一原理計算による結果に基づいて提案されたCC(カラベッター・クレメンティ)ポテンシャル(Carravetta-Clementipotential)も、よく用いられる。電荷の配置はTIP4Pと同じように、酸素原子からずれた位置に配置しているとともに、電荷以外のポテンシャルを式(10・22)のような単純なレナード・ジョーンズ関数とせずに、それぞれの酸素原子と水素原子間の距離の関数で決めている。実験的に孤立した水分子の永久双極子モーメントは、1.85D 程度であることが知られているが、例えばSPC/Eの2.351D のように、液体中での誘起双極子モーメントの分も加えて、ポテンシャルが作られている。分子内の運動を加えることで、環境に応じて双極子モーメントが変化するフレキシブルモデルも、さまざまに開発されている。c. 炭素とシリコンの多体関数炭素やシリコンなどの共有結合を表現するには有効二体ポテンシャルでは困難で、従来からさまざまな形での多体ポテンシャル(many-body potential)が提案されてきた。シリコンに対するSW(Stillinger-Weber、シュテリンガー・ウェーバー)ポテンシャル、シリコンと炭素に関するターソフ(Tersoff)ポテンシャル、ターソフポテンシャルを炭素に関して発展させたブレナー(Brenner)ポテンシャルなどがよく知られている。
