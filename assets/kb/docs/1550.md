# 10・2・7 動的物性値の予測

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・2 分子動力学法

拡散係数、粘性係数、熱伝導率などの物性値は分子動力学法を用いて、(1)平衡分子動力学のシミュレーションの結果にグリーン・久保の公式を用いたもの、(2)エバンスによる非平衡分子動力学法、(3)境界条件として温度差などをつけた直接非平衡計算の三とおりの方法で計算が可能である。平衡分子動力学では、例えば拡散係数は、グリーン・久保の公式によって、=1욡욡0d(10・32)と速度の自己相関係数から求まる。さらにアインシュタインの関係式によると、一般にの関数に関して、γ=욡욡0d(10・33)で表される自己相関は、十分に大きなに対して、2γ=12-0(10・34)と表される。式(10・32)にアインシュタインの関係式を用いると、十分大きなに対して=16욡-욡0(10・35)によって計算できる。エバンスによる非平衡分子動力学では、温度こう配などが存在する場合の非平衡状態でのハミルトニアンから運動方程式を作る。この仮想の温度こう配に対応する熱流束を求めて、以下のフーリエの式から熱伝導率を計算する。=-λ옿옿(10・36)温度こう配に対応して運動方程式に現れる力を仮想力(ficti-tious force)と呼ぶ。これを変化させて零となる極限での熱伝導率を採用する。直接非平衡計算では、境界条件として温度差を設定し、温度こう配と熱流束を求めて、フーリエの式から熱伝導率を計算する。物理的な解釈は最もわかりやすいが、非現実的なほどに過大な温度こう配をつけてしまうことが多い。
