# 10・2・9 分子動力学法シミュレーションの例2(固液接触)

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・7 には描いていないが、界面と垂直方向の圧力はによ

現実的な分子を用いた固液接触の分子動力学法シミュレーションの例として、図10・8にプラチナ(111)表面への微小水液滴の接触の様子を示す。上図は、350K で平衡に達した状態でのスナップショット、下図は、液滴の重心を通る円筒座標で平均した水の二次元密度分布を表す。水分子間にはSPC/Eポテンシャル、プラチナ原子間は調和ポテンシャル、水分子とプラチナ原子には、ズー・フィルポット(Zhu-Philpott)が拡張ヒュッケル計算に基づいて作成したポテンシャルを用いている。プラチナ表面に一層の水分子の膜が形成され、その膜の上に一定の接触角をもって水の液滴が接触している。水の分子膜と水液滴とは、極めてぬれやすい(界面エネルギーが零)と考えるのがふつうであり、このような構造の存在は大きな驚きである。マクロのヤングの式を考えると、平面的な水の表面で水の液滴が接触角をもつことは考えにくい。さまざまな表面構造や、水とプラチナのポテンシャルを変えた計算などの検討の結果、プラチナ原子表面に強く結合した水分子膜は、水とは大きく異図10・7液体・気体の界面における密度、界面接線方向圧力成分図10・8プラチナ表面の水液滴のスナップショットと二次元密度分布図10・9 DSMC法の計算手順なると考えることで理解できた。すなわち、高密度に平面的に整列した水分子膜は液体の水とは全く異なる性質を示し、この水分子膜と液体の水分子とは十分な水素結合が作れずに、これらの間に大きな界面エネルギーが存在する。実際、分子膜の面密度が増加するほど接触角が大きく(ぬれにくく)なることがわかる。
