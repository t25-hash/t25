# 10・3・1 概要

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・3 直接シミュレーションモンテカルロ法

気体の流れの数値解析は、連続の式やナビエ・ストークス方程式を基礎としたCFD(computational fluid dynamics)が一般的であるが、流れ場のクヌッセン数=λ(λ:平均自由行程、:代表長さ)が大きい場合、その流れ場を連続体として扱うことができなくなる。このような流れ場は一般に希薄気体流れ(rarefiedgas flow)と呼ばれ、分子間衝突の流れ場への影響を考慮した解析が必要となる。これらの流れ場には気体の圧力が小さい系の流れや、狭いすきまの流れなどが該当し、宇宙往還機まわりの流れやターボ分子ポンプ内の流れ、真空チャンバ内の流れや固定ディスクのディスクとチップ間の流れなどに応用されている。本節ではこの希薄気体流れの解析に対して、現在最もよく用いられる手法である直接シミュレーションモンテカルロ法(direct simulation MonteCarlo method、以下、DSMC法と記述)について紹介する。
