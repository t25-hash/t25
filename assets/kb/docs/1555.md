# 10・3・3 壁面反射モデル

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・3 直接シミュレーションモンテカルロ法

希薄気体流れの数値解析における壁面反射モデルとしては、分子の状態が壁面に完全に適応することを仮定したモデルから、分子が壁面に衝突する現象を力学的に解析したモデルまで、さまざまなモデルが提唱されている。ここではその中から代表的なものについて紹介し、解説を加える。また、以下の式では壁面が平面にあるとして、分子は衝突後>0の領域に反射するものとし、衝突後の速度を′と表している。a. 鏡面反射(specular reflection)モデルこのモデルは、分子の速度のうち壁面の接線方向の速度は変化せず、壁面の法線方向の速度が反転するものである。衝突によって分子のエネルギー状態は変化せず、取扱いが最も容易なモデルである。衝突後の分子速度は、以下の式で与えられる。反射角度は余弦則に従い、エネルギー状態は壁面温度におけるマクスウェル分布となる。これは、壁面の凹凸が大きく汚れている場合などの散乱則に対応し、取扱いが容易なだけでなく、工学的にも重要なモデルである。衝突後の分子の速度′は、一様乱数を用いて以下の式で与えられる。c. マクスウェルモデル(Maxwell model)このモデルでは鏡面反射と拡散反射が混在しており、入射分子に対して確率αで拡散反射、1-αで鏡面反射を与える。αはエネルギー適応係数(accommodation coefficient)や運動量適応係数などを用いる。d. チェルチニャーニ・ランピス・ロードモデル(Cercignani-Lampis-Lord model,CLL model)このモデルは実験定数として、接線方向速度と法線方向速度に別々な適応係数を用い、これら二つの速度変化が独立な確率過程に従うと仮定している。このモデルは分子線散乱実験で得られる葉状(lobular)分布をよく再現できる。e. マルチステージモデル(multi-stage model,MS model)このモデルは壁面に衝突する分子の挙動を、分子動力学シミュレーションにより解析し、得られた散乱角度分布や散乱後のエネルギー分布に対する知見から構築されたモデルである。このモデルは、分子動力学シミュレーションの結果を正確に再現できることが示されている。
