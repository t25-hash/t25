# 10・3・5 計算例

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・3 直接シミュレーションモンテカルロ法

近年のコンピュータの発達に伴って,DSMC法による希薄気体流れの計算も大規模なものになってきている.ここでは,このDSMC法を用いて行われている最近の研究を紹介する.6―139a. 一次元垂直衝撃波波面内の温度分布およびエネルギー分布非平衡状態で,どの程度温度やエネルギー分布が平衡状態の値と異なるかを見積もることは,工学的にも非常に興味深いことである.このような非平衡を取扱う問題としては,この一次元垂直衝撃波波面内の温度分布やエネルギー分布は最も基礎的なものである.図윰=12.7の衝撃波波面内の各自由度ごとの温度分布を示す.この計算は二原子分子衝突モデルとしてDMCモデルを用いている.具体的な計算手法は文献(42)に詳しい.図10・10には衝撃波波面内の各自由度ごとの温度分布,および密度分布を示す.図10・10より,衝撃波の進行方向と同方向の温度が著しく上昇していることがわかる.これは읉윰=12.7と強い衝撃波ではまず,流速のエネルギーが衝撃波の進行方向のエネルギーに変換され,これらが徐々に衝撃波の進行方向に垂直方向のエネルギー,回転エネルギーに緩和していく様子を表している.また,図うにDSMC法における計算結果は,非常によく密度分布や回転温度分布の実験結果を再現することができる.b. オリフィス自由噴流の流動構造オリフィスやノズルから真空中へ噴出する自由噴流(free jet)は,薄膜成形における分子レベルの超精密加工に関連してその需要が増大している.宇佐美らは,この自由噴流の二次元的および三次元的DSMC計算を行い,クヌッセン数による流動構造の変化などについて解析を行っている.この計算では対象分子をArとし,分子モデルとしてはVHS モデルを用いている.図10・11に1=2500の密度分布を示す.この計算で用いているサンプル分子の数は80万~120万程度,セル数は約5万である.図10・11からもわかるように,密度分布はいったん落ち込んで最小値をとった後,急激に増加している.これは垂直衝撃波(Mach disk shock)と呼ばれる現象である.また噴流を取り囲むように密度が上昇する部分が見られる.これは樽形衝撃波(barrel shock)と呼ばれ,垂直衝撃波とともに連続域に近い状態での自由噴流で見られる現象である.このようにDSMC計算では,比較的連続域に近い状態での超音速流れについても,非常によく現象を再現することができる.また宇佐美らは自由噴流内の温度分布についても,同様の解析を行っている.図10・10 一次元垂直衝撃波波面内の温度および密度分布図10・11 超音速自由噴流場内の密度分布6―140計算力学格子ボルツマン法(lattice Boltzmann method)は,もともとは格子ガスオートマトン(lattice gas automaton)から導かれた.格子ガスオートマトンの仮想粒子の代わりに,仮想粒子をアンサンブル平均した速度分布関数を用いることにより,格子ガスオートマトンの統計ノイズの減少,計算効率の向上を目指したものである.その後の研究により,格子ボルツマン方程式はボルツマン方程式の離散化形式としても,全く同様に導かれることが示された.後者のほうがより一般的で幅広い考え方をもち,格子選定の自由度が高く,また,格子ボルツマン法を微分方程式を解く緩和法の一種と考える考え方も包含する.ここではこれに従って,ボルツマン方程式の離散化として解説する.
