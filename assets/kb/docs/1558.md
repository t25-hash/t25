# 10・4・1 ボルツマン方程式(Boltzmann equation)

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・4 格子ボルツマン法

a. 分布関数(distribution function)気体分子運動のような極めて多数の粒子の運動を考える。粒子一つ一つの運動はニュートンの運動方程式に従う。多数の粒子を考えるには、運動方程式の一般形であるハミルトンの運動方程式が都合がよい。粒子の力学状態は、粒子の位置と速度により完全に定義できる。位置と速度または位置と運動量を独立変数とする空間の中での粒子の軌跡を考えることにより、粒子系の運動を理解することができる。この位置と速度を座標とする空間を、位相空間(phase space)という。流体運動は流体を構成する粒子、すなわち分子の集団運動であり、粒子の運動を平均化することで流れの特性が求められる。しかし粒子数は膨大であるので、位相空間の中で粒子一つ一つを追いかけていくことは無理である。流れのマクロ特性量は、の速度座標方向の積分により次のように求められる。また粒子質量は1としている。b. 発展方程式分布関数の発展を記述する方程式が、ボルツマン方程式である。粒子系に外力が働かない場合、について次のリュービル方程式(Liouville equation)が成り立つ。dd=0(10・68)ここで、粒子間の相互作用として衝突による粒子の運動量交換を考え、全微分を展開すると、次のボルツマン方程式が得られる。Ωは衝突を含む粒子ダイナミクスの詳細に依存する。一般に極めて複雑な形となり、ボルツマン方程式は非線形の微分積分方程式となる。Ωの満たすべき条件として、粒子衝突によりマクロに保存されるべき質量、運動量、エネルギーが変化しないことが必要であり、衝突不変量である1,をΩに掛けて位相空間全体で積分した量は零にならなければならない。希薄気体に対しては、相関のない二体弾性衝突のみを考えればよい。ボルツマン方程式は、多体粒子系の運動を分布関数という形で平均化したものであるが、粒子衝突という分子間相互作用の概念を保持している。また、積分してナビエ・ストークス方程式が導かれるいわば詳細な方程式であり、ナビエ・ストークス方程式より豊富な物理を包含しているといえる。しかし、流れを求めるのにボルツマン方程式の直接解を求めることは効率的ではない。
