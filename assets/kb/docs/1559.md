# 10・4・2 格子ボルツマン方程式

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・4 格子ボルツマン法

a. 離散化(discretization)格子ボルツマン法は,ボルツマン方程式の位相空間を制限することにより,効率的にボルツマン方程式を解く方法である.計算する体系,つまり物理空間を格子で区切る.分布関数は格子の交点である格子点で定義する.速度空間を制限するため,分布関数の速度方向を離散化する.分布関数は,本来連続的な速度方向と速度の大きさをもつものであるが,格子ボルツマン法では,速度方向はある格子点から隣接格子点へ向かういくつかの速度方向に限定し,速度の大きさも隣接格子点へのベクトルの大きさに相当する少数の離散値とする.この速度空間の大幅な制限により,ボルツマン方程式では速度の連続関数である分布関数が,格子ボルツマン法ではそれぞれの速度に対応する量として,離散的に定義される.これを次のように添字で表し,速度分布関数(velocity distributionfunction)と呼ぶ.욡,=1,...,(10・70)ここで,は離散速度の数である.衝突項も同様に,衝突による方向分布関数の変化を表すものとなり,ボルツマン方程式は次式となる.옿옿욡+욡∇욡=Ω욂욡(10・71)上式の左辺は速度分布関数の時間変化と,空間移動による変化を加えたものであり,時間ステップが離散的に一つ進むごとに,速度分布関数が速度に対応した隣接格子点へ移動するというラグランジュ的な輸送と解釈すれば,次式となる.욡+욡,+1-욡,=Ω욂욡(10・72)これを格子ボルツマン方程式(lattice Boltzmann equation)という.ここで時間ステップ幅は=1としている.流れのマクロ特性量は速度成分についての和で,次のように与えられる.욁욡,(10・73)ρ,=∑욁욡욡,ρ,(10・74),=∑욁-욡,ρ,(10・75),=12∑ここでも粒子質量は1としている.b. 衝突項(collision term)衝突項Ω욂욡は,本来は格子ボルツマン方程式の基礎となっている格子ガスオートマトンの粒子衝突ルールを,アンサンブル平均したものである.しかし,格子ボルツマン法の目的は,ナビエ・ストークス方程式で表されるようなマクロダイナミクスの再現にあり,これを満たす範囲内で簡略化した衝突項を用いることができる.というのは,流れのマクロダイナミクスは,詳細な粒子相互作用の結果というよりは,単に粒子レベルでの保存則の結果が反映されたもので,粒子レベルの保存則が満たされるようなものでありさえすれば,衝突項に簡単なものを用いてもよいからである.ほとんどの格子ボルツマン解析で,次のBGK 近似[BGK(Bhatnagar-Gross-Krook)approximation]が衝突項として用いられる.Ω욂욡=-1τ욡,-욡욓,(10・76)ここでτは定数,욡욓は局所平衡分布関数(local equilibrium dis-tribution function)である.これは単一時間緩和近似(singletime relaxation approximation)とも呼ばれ,衝突により速度分布関数が一定の時定数τで,局所平衡分布に緩和していく形であると解釈できる.これにより,格子ボルツマン法の基本式は次式となる.욡+욡,+1-욡,=-1τ욡,-욡욓,(10・77)
