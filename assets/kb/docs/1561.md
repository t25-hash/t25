# 10・4・4 格子ボルツマン法の拡張

> **章**: 第10章 熱流体解析のための離散系シミュレーション
> **節**: 10・12 に二次元格子と三次元格子の例を示す。二次元格子は、速度

a. 有限差分格子ボルツマン法(finite difference lattice Boltz-mann method)格子ボルツマン法で用いられる格子上での速度6―142計算力学分布関数の並進移動は、格子ガスオートマトンの粒子移動のアンサンブル平均として導入されたもので、一つの時間ステップで速度分布関数は格子点を一つ移動する。しかし、このことはマクロに正しいダイナミクスをリカバーするために必要な条件ではない。格子ボルツマン方程式が、ボルツマン方程式の離散化の一つと考えられることはすでに述べた。このように考えれば、空間の離散化と粒子、または速度分布関数の移動速度を独立に扱ってよいことになる。この考えに基づき、式(10・71)を速度分布関数についての微分方程式として、微係数は通常の差分などで近似して解いてよいことになる。これを有限差分格子ボルツマン法という。衝突項には一般にやはりBGK 近似が用いられる。有限差分格子ボルツマン法では、これまで微分方程式の数値解析で利用されてきたさまざまな離散化を利用することができる。例えば、より高次で高精度のスキームを用いることや、物理量の空間こう配が大きい場合に効率的なTVD(total variation dimini-shing)などのスキームを用いることも可能となる。空間格子にしばられることが必要でなくなるので、非一様な格子や円筒座標などを使用することもできる。また、時間微分項も、速度分布関数の並進とは独立に刻み幅を決められるので、数値不安定性のコントロールや、高レイノルズ数流れへの対応も容易になる。b. 熱解析格子ボルツマン法では、用いる速度の大きさを複数とすれば、それにより内部状態の違いを表現できるので、原理的には熱輸送の問題を扱うことができる。しかし実際には、局所平衡分布をより高次まで展開し、二次元では16速度方向で5スピードの速度分布関数を用いる必要があるため、計算手続きが極めて複雑となる。また、輸送係数が緩和時定数によって定まりプラントル数が一定となること、扱える温度範囲が狭いことなどの問題がある。このため、速度分布関数とは別に、温度に対応する新しい速度分布関数を用意し、両方の速度分布関数を格子ボルツマン法で発展させる方法がある。速度分布関数욡とは別に、新しい速度分布関数욡を導入する。욡は温度と욁욡(10・80)=∑の関係をもち、욡+욡,+1-욡,=-1τ욡,-욡욓,(10・81)の発展方程式に従う。욡と욡を同時に解いて、流れ場と温度場を求めることができる。c. 二相水と油のような液と液や、空気と水のような気液の二相流(two-phase flow)の特徴は、シャープな界面が存在すること、そして、その界面が変形することである。通常の流体数値解析では、これらにより、界面の数値拡散と界面追跡の必要性が課題となる。これに対して格子ボルツマン法では、相の分布は計算の結果として出てくるもので、界面や界面の形状を陽に追跡していく必要がない。格子ボルツマン二相モデルは、粒子間相互作用モデルと二流体モデルの二つに大きく分けられる。実際の物理現象において、粒子間に強い短距離の斥力と弱い長距離の引力が作用すると、低温で第一次の相転移が起こり、気体と液体に分離して気液二相が形成される。粒子間相互作用モデルでは、この粒子間に働く力を速度分布関数に作用する外力として与える。この外力項は、運動量空間での対流項という形でボルツマン方程式に加わるものであるが、ラグランジュ離散化により、結局は衝突項に加わるポテンシャルとして表すことができる。ポテンシャル項を加えた格子ボルツマン方程式の摂動展開からは、非理想気体の状態方程式が導かれ、気液のように密度の異なる二相の共存を表すことができる。図10・14 格子ボルツマン法により求めた二相の分離プロセスの時間発展(時間は左上から右へ進んで下段へ降り、下段左から右へ進む)水と油のような非混和な二相では、水と油の分子間相互作用によって二相に分離し、相間に表面張力が作用する。二流体モデルでは、これを表現するため、例えば赤と青など色で区別する二つの流体成分を用いる。それぞれの相について速度分布関数を用意し、格子ボルツマン方程式をたてる。相間には、局所的な色分布のこう配を反映した相互作用ポテンシャルを衝突項に加え、ポテンシャルの大きさで表面張力をコントロールする。図10・14に格子ボルツマン法により求めた、二相の分離の時間発展の様子を示す。d. その他の流れ以上のほか、多孔質体内の流れ、粘弾性流体、多相多成分流体、高分子流体、液晶、コロイドを含む流体などの流れへの、格子ボルツマン法の適用が研究開発されている。
