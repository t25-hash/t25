# 11・1・1 熱の伝達様式

> **章**: 第11章 連成問題
> **節**: 11・1 熱

物質内部や物質間でやりとりされる熱エネルギーの伝達機構には、熱伝導、対流熱伝達、ふく射熱伝達、相変化を伴う熱伝達などさまざまなものがある。熱の関係する連成問題を考えるとき、熱エネルギーの伝達機構を明らかにしておくことは、対象とする問題の支配方程式や境界条件を選択するうえで重要である。以下においては、熱伝導、対流熱伝達、ふく射熱伝達について簡単に整理する。a. 熱伝導物質を構成する原子・分子の熱振動によるものであり、固体や静止した流体では熱伝導が支配的な熱エネルギーの伝達機構である。b. 対流熱伝達伝熱面(固体)と流体との間、および流体内部での熱エネルギーの伝達であり、強制対流熱伝達と自然対流熱伝達の2種類がある。伝熱面と流体との間では、伝達される熱流束は次式で与えられる。熱伝達率は流体の物性値だけでは決まらず、流況(層流か乱流か、など)、伝熱面の形状や表面状態に依存する。熱伝達率に関するデータは、無次元数のヌセルト数()として整理されることが多い。流体内部での熱エネルギー伝達に着目した場合、強制対流では、熱は流体内を熱伝導(拡散)と移流によって輸送される。すなわち、温度場は流れ場によってパッシブに決定される。一方、自然対流では、温度差で生じた流体の密度差が浮力として流れ場に作用する。すなわち、温度場と流れ場は連成する。c. ふく射熱伝達放射源(温度)から、いま考えている境界面(温度)へふく射により伝達される熱流束は次式で与えられる。修正形態係数は面の放射率や表面積、互いの位置関係などから決定される。
