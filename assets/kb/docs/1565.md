# 11・1・3 弾性体と熱の連成

> **章**: 第11章 連成問題
> **節**: 11・1 熱

a. 線形熱弾性温度変化を伴う線形弾性体の変形・応力解析において考慮すべき方程式は、・ひずみ-変位関係[式(11・3)]・運動量保存則[式(11・6)]・エネルギー保存則[式(11・7)]・構成式[式(11・8)]および関連する境界条件である。式(11・8)は以下のように表すことができる。σ욡욢=λδ욡욢ε욅욅-ε욅욅+2με욡욢-ε욡욢(11・20)ε욡욢=α-δ욡욢(11・21)ここで、ε욡욢は熱ひずみである。式(11・20)は応力σ욡욢と、全ひずみから熱ひずみを差引いたひずみ成分(ε욡욢-ε욡욢)とが、熱の影響がない場合の応力-ひずみ関係に従うことを示している。したがって、温度変化を伴う線形弾性解析においては、まず式(11・7)のエネルギー保存則(熱伝導方程式)から弾性体の温度分布を求め、式(11・21)から熱ひずみを計算する。次に、応力解析を実施する際には、熱の影響がない場合の応力-ひずみ関係式を用いる。ただし、熱ひずみに相当する見掛けの物体力を、욡と同様の扱いで既知の外力として考慮する。有限要素法における解析手順でいえば、熱が関係しない通常の弾性解析との違いは、熱ひずみによる見掛けの物体力ベクトルが剛性方程式の右辺ベクトルとして、新たに追加されることである。b. 非弾性問題高温下においては、熱との連成の結果として、弾塑性、材料物性の温度依存性・ひずみ速度依存性、クリープなどの非弾性問題が表面化しやすくなる。こうした非弾性問題では、荷重もしくは時間を微小な増分ステップに分割し、各ステップ内では増分形で表された変数について、線形化した支配方程式を扱うことが多い(増分法)。例えば、各ステップにおいて構成式は増分表示で次のように表される。σ=욓ε-ε욓-ε-ε욑-ε(11・22)ここで、σ,욓はそれぞれ、応力増分、弾性応力-ひずみテンソルである。また、ε,ε욓,ε,ε욑,εはそれぞれ、全ひずみ、温度依存性によるひずみ増分、塑性ひずみ増分、クリープひずみ増分、熱ひずみ増分である。式(11・22)は応力増分σと、全ひずみ増分から各種ひずみ増分を差引いた残りのひずみ増分成分が、熱の影響がない場合の線形の応力-ひずみ関係に従うことを示している。簡単のために、式(11・22)において、塑性および物性値の温度依存性がない場合を考える。このとき、σ=욓ε-ε욑-ε(11・23)各増分ステップにおいて応力解析を実施する際には、線形熱弾性解析のときと同様、熱伝導解析によって求められた温度分布を基に、熱ひずみ増分[式(11・21)]やクリープひずみ増分に相当する見掛けの物体力を求め、既知の外力として右辺側に考慮する。なお、クリープひずみ増分の決定には、硬化則やクリープ流れ則が用いられる。
