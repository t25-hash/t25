# 11・1・4 ニュートン流体と熱の連成

> **章**: 第11章 連成問題
> **節**: 11・1 熱

ニュートン流体は、偏差応力がひずみ速度の線形関数として表される流体であり、温度変化を伴う場合に考慮すべき方程式は、・ひずみ速度-変位速度関係[式(11・4)]・質量保存則[式(11・5)]・運動量保存則[式(11・6)]・エネルギー保存則[式(11・7)]・構成式[式(11・9)]・状態方程式[式(11・10)]および関連する境界条件である。流体を非圧縮性と仮定し、非均質な温度分布から生じる密度差の効果は浮力の効果としてのみ現れるとすると、式(11・24)は次式のように表すことができる。
