# 11・1・1 項でも述べたように、流体内部での熱エネルギー伝達様

> **章**: 第11章 連成問題
> **節**: 11・1 熱

式に応じて、強制対流と自然対流の区別がなされる。式(11・25)のように、ナビエ・ストークス方程式の外力として浮力を考慮した場合を自然対流と呼び、式(11・25)は式(11・7)のエネルギー保存則(温度の移流拡散方程式)と連成して解く必要がある。一方、浮力を考慮しない場合には、式(11・25)と式(11・7)は別々に扱うことができる。すなわち、式(11・25)からまず流れ場を求め、次にその流れ場のもとで式(11・7)を解けばよい。その場合を強制対流と呼び、熱は流体内を熱伝導(拡散)と移流によってパッシブに輸送される。
