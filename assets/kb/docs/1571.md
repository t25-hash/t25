# 11・2・3 渦電流問題(eddy current problem)の定式化

> **章**: 第11章 連成問題
> **節**: 11・2 電磁場

ここでは三次元渦電流解析の代表的手法-φ法(-φmethod)を紹介する。ここには磁気ベクトルポテンシャルで、φは電気スカラポテンシャルを表す。なお、ここでの渦電流問題は、式(11・28)で電束密度の時間微分(変位電流という)が十分小さく無視できる場合のみを考える。この仮定は一般に、低い周波数領域でのみ許されることを注意しておく(εωσ≪1を仮定している。ここにεは誘電率、ω,σは後述の角周波数と導電率を表す)。式(11・28)右辺第2項の変位電流は長らく無視された後に、マクスウェル自身によって付加された項である。-φ法による渦電流問題の定式化は、マクスウェル方程式(11・26)~(11・29)から出発し、変位電流を無視し、多面体領域Ω全体での磁場の方程式と、導体部分のみの電場の方程式を適切に考慮することにより、次式のように書ける。ここでも式(11・29)に注目し、磁気ベクトルポテンシャルを導入し、クーロンゲージ式(11・46)を考える。ここでもφは電気スカラポテンシャルを表し、磁気ベクトルポテンシャルとともに未知関数である。その際、境界Γの単位法線ベクトルの向きは、どちらか一方にそろえていることに注意する。式(11・49)は渦電流密度(A/m)の法線成分が零であることを意味し、式(11・30)の境界条件と対照的な設定になっていることに注意する。ΩはΩの境界を表し、そこで境界条件式(11・50)を考える。こちらのほうは式(11・31)の言い換えである。また、は境界における外向き単位法線ベクトルを表す。ただし、Γ上での境界条件式(11・47)~(11・49)を考慮するときには、前述の注意が必要である。
