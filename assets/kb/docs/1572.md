# 11・2・4 その他の問題

> **章**: 第11章 連成問題
> **節**: 11・2 電磁場

非線形静磁場だけでも11・2・2項で前述したB-H 特性以外に磁気ヒステリシスを考慮する場合,あるいは永久磁石特性を考慮する場合など,さまざまな問題がある.さらには固有値問題,電磁波の伝ぱ・散乱等,マクスウェル方程式をより完全な形で考慮する多くの問題がある.また,これらの電磁場問題が伝熱工学・流体力学・固体力学等と連成する,工学上のいくつかの重要な課題がある.例えば,マクスウェルの応力テンソルにより,求まった磁場から吸引力を評価する場合がその一例である.取上げる問題に応じて,それにふさわしい未知関数の選択や近似解法があるのは言うまでもない.限られた紙面の中では,これらのすべてを解説することができない.関心のある読者は関連文献等を参照されたい.
