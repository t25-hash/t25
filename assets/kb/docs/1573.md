# 12・8・2 代表的な試験法とその考え方

> **章**: 第12章 複合材料の力学
> **節**: 12・8 試験方法（test method）

ここでは，高強度で異方性が特に問題となるCFRPの試験方法
を中心に，すでに規格化がなされているものについて紹介する．詳
細は，既報の解説 を参照してほしい．
a.繊維の引張試験 繊維の引張試験方法は，繊維1本（フィ
ラメント）を試験する単繊維（filament ）試験と，樹脂を繊維束に
含浸して一種の複合材料にして引っ張り，計算で繊維強度を求める
樹脂含浸ストランド法の2種類が代表的である．両者ともJIS
R7601：1986で採用されているほか，前者はISO11566：1996，
後者はISO10618：2004として国際規格化された．フィラメント
試験法は，図12･58の手順で1本の繊維を紙枠に貼り付けて行う．
この手法は繊維そのものの特性を把握できるが，試験片長依存性が
大きく，ばらつきも大きいため，ワイブル分布を用いた統計的処理
が必要である．また，個々のフィラメントの断面積測定も難しい
（JIS R7601：1986およびISO11567：1995）．
樹脂含浸ストランド法は，3000～24000本の繊維束に比較的伸
びの大きい樹脂を含浸させて複合材料を作製し，これを引っ張り，
強度・弾性定数を求める方法である．この方法のほうがCFRPと
して用いる場合の繊維の特性を反映し ，強度が高くばらつきも
小さいため，繊維メーカのカタログ値では通常この値が用いられ
る．
母材樹脂のみを燃焼または酸で分解して除去し，複合材料中の繊維
の質量を求めるとともに，複合材料の密度，樹脂および繊維の密度
を用いることにより，繊維および空洞の体積含有率を求めることが
でき，JIS K7075：1991に規定されている．燃焼法は，不燃分の
残る樹脂には適用できないが，操作が安全かつ簡便である．一方，
酸を用いる方法は，濃硝酸や濃硫酸を高温に加熱する必要がある
が，樹脂の適用範囲が広くかつ精度の高い方法である．なお，酸に
よる分解はASTM D3171-99（2004)，ガラス繊維に限定した燃焼
法はASTM D2584-02にも規定されている．
一方向強化材では，せん断強度が繊維方向引張強度の1/15～1/30
程度であるため，せん断で荷重を試験片に伝達するのが不利であ
る．一方向強化材の繊維（0 ）方向では，金属材料で用いられる試
験片の幅や厚さを変化させる方法（ドッグボーン，ダンベル型試験
片）では，繊維方向の縦割れやはく離が生じ，荷重伝達が極めて困
難である．一般に図12･59に示すように，短冊型試験片に試験材料
より弾性定数の低いGFRP等のタブを接着する方法が広く用いら
れている（ASTM D3039-00，JIS K7073：1988，ISO527-5：
1997) ．ただし，この場合も応力集中は当然残っている．純粋な
引張破壊であるかどうかの判定は難しく ，真の物理的な値を求

d.圧縮試験（compression test ) 複合材料の圧縮の破壊機構
は主として繊維の局部せん断破壊または座屈である．しかし，圧縮
試験で複合材料中の繊維自体の微視的座屈と，試験片全体の巨視的
座屈を厳密に区別するのは困難で ，理想的な圧縮破壊かどうか
の判定が非常に難しい．したがって，最も根本的な問題が多く未確
立な試験法である．
既存の試験法は，荷重負荷方法から，(１)試験片にせん断で荷重
を負荷する方法，(２)試験片端面に圧縮荷重を負荷する方法に大別
される．(１)はASTM D3410-03，JIS K7076：1991で，(２)は
ASTM D695-02，JIS K7076：1991で採用されている．
せん断で荷重を負荷する方法では，図12･60に概念を示す円筒ス
リーブまたはくさびを使ったジグと鋼製タブ付きの試験片が用いら
れる．基本的な問題点としてはタブ端で応力集中を生じること，座
屈を生じやすいことが挙げられる．
端面に圧縮荷重を負荷する方法は，GFRPのJIS K7056：1995
を発展させたものであり，CFRPタブ付試験片と図12･61のキの
字型の座屈防止ジグを用いる．圧縮負荷を直接試験片にかけるた
め，端部の加工精度が影響する．

状態があるが，微視的な繊維配置を除くとほぼ同様の変形モードで

g.層間破壊靱性試験（interlaminar fracture toughness test ）
積層複合材料ではき裂の進展が繊維によって拘束されるため，モー
ドⅠ以外のモードの検討が必要である．モードⅠ，モードⅡの破壊
靱性試験はJIS K7086：1993としてわが国が世界に先駆けて標準
化しており ，特にその解説に原理，ノウハウ，実測データと
その解釈の仕方が詳細に記述されている．モードⅠに関しては，
ASTM D5528-01も出版されているほか，長年の議論を経て，近
年ISO15024：2001が制定された．これらは，試験片への初期欠
陥導入方法，靱性値の定義が微妙に異なるため，注意が必要であ
る．
引張りでせん断荷重をかけるレールシェア法（JIS K7079：
1991；2レールシェアのみ，ASTM D4255-01），(３)Vノッチの
入った試験片に非対称四点曲げをかけるIosipescu法（ASTM
D5379-98）が中心である．この中で，＋45/－45法が最も簡便であ
るが，近年では比較的均一なせん断応力場ができるIosipescu法も
広く用いられるようになってきている．しかし，いずれの方法も，
厳密にはせん断応力場以外に引っ張り，圧縮応力等を伴う ．
層間せん断試験方法としては，特別に厚い積層板にIosipescu法
を適用する方法，図12･63のように3点曲げ試験でスパンを試験片
厚さに対して比較的短くしてせん断で破壊させるショートビームシ
ェア法（JIS K7078：1991），および図12･64のように試験片の両
面に板厚半分の深さの切欠を入れ圧縮/引張荷重を負荷する目違い
せん断（DNS:double-notched shear ）法（ASTM D3846-02）

ジグを用いる．ショートビームシェア法は簡便なため広く用いられ

ことに注意が必要である．また，高靱性の材料，今後構造材料とし

法を用いないとせん断破壊が発生しないと考えられている．

i .面圧試験（bearing test ) 実際の構造物では接着とともに
ボルトやピンを用いて機械的に結合することが避けられず，継手の
設計データを得るために，面圧強度試験法がJIS K7080：1991，
ASTM D5961-01で制定されている．
j .面外衝撃（多軸衝撃）試験（falling weight impact test ）お
よび衝撃後圧縮試験（compression after impact test ) 積層板
に面外から低速の衝撃荷重が負荷された場合，図12･68に示すよう
に，各層で層間破壊と樹脂割れが発生する．衝撃負荷が小さい場
合，層間はく離だけが生じ外見的な損傷がない場合もあり，注意が
必要である．そのため，円形の枠で積層板の周辺を固定し，数m/ s
の速度の落垂で衝撃荷重を与える多軸衝撃試験がJIS K7085：
1993として，制定されている．
一方，多軸衝撃負荷を与えて層間破壊の発生した積層板に圧縮荷
重負荷を与えると，積層板が局部座屈し強度が著しく低下する．そ
のため，擬似等方積層板に厚さ1mm当たり6.67Jの衝撃負荷を
与えた後に圧縮荷重を負荷して，強度低下を調べる衝撃後圧縮試験
（CAI ）がSACMA規格をもとにJIS K7089：1996として制定さ
れている．CAIは現在航空機等の複合材料構造設計において設計
許容ひずみを決める際に参考にされるほか，材料開発の指標ともさ
れている．しかし，背景となる力学，破壊機構が複雑で完全な理解
には至っていない．
