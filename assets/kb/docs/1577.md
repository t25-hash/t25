# 11・4・2 磁気剛性の解析

> **章**: 第11章 連成問題
> **節**: 11・4 電磁構造連成

磁気剛性の概念をはっきりさせるために、解析的に処理できる簡単な問題を取上げる。図11・4は、無限に長い3本の導体ばりが等間隔に配置された状況を示している。導体ばりには、等しい電流が同じ向きに流れ、上下のはりは完全に拘束されており、中央のはりは長さの部分だけが方向に変形できるものとする。電流の向きが同じなので、3本のはりは引き合っており、配置の対称性より中央のはり1に働く電磁力は釣合っている。電流を増加すると、電流がある臨界値になった所ではり1は座屈する。この現象は電磁弾性座屈(electromagnetoelastic buckling)と呼ばれる。また、この座屈は以下のように、はりの振動問題として説明することができる。磁気エネルギー(magnetic energy)は次式のように与えられる。第3項は中央のはり1のたわみには依存しないので、考慮する必要がない。はり1のたわみに関係する項のみ計算すると、次式が得られる。この磁気エネルギーを考慮して、はりの振動に関するラグランジアンは次式で与えられる。ハミルトンの原理を式(11・118)に適用すると、平衡方程式として次式が得られる。第4項を含めると、電磁力の効果を含めた連成振動方程式となる。式(11・119)を解いて、その解を,ω(固有振動数)、で規格化して示すと、次のようになる。すなわち、この場合はりは座屈を起こしているので、は座屈電流と呼ばれる。軸力を増加させるとオイラー座屈が生じるように、電流を増加させていくと=で座屈することがわかる。通電電流の増加に伴って振動数が減少する現象は、はり1が振動してはり2に近づくときには力のバランスがくずれ、上向きの引張力が大きくなることより理解できる。はり2と3の電流が同方向に、はり1の電流はそれとは逆方向に流れる場合には、振動数が増加し座屈は生じない。これは圧縮力の代わりに引張力が作用したとき、振動数が増加するのと類似している。以上の例では、物体の変形が場の磁気エネルギーを変化させ、変形量に依存した電磁力が物体に働いている。これは、はりの剛性が電磁場との相互作用によって変化していることを意味しており、電流によって変化する剛性を磁気剛性という。通常の導体構造物の場合は電流が小さいため、磁気剛性は機械的剛性に比べて小さくなり、問題にしなくてもよい。核融合炉用の超伝導マグネットのように、電流密度がけた違いに大きい場合や磁場中に置かれた磁性体などの場合には、磁気剛性の効果が重要となる。
