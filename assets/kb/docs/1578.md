# 11・4・3 磁気減衰の解析

> **章**: 第11章 連成問題
> **節**: 11・4 電磁構造連成

磁気減衰効果を考慮した過渡的電磁力の評価手法および振動について説明する。この場合、速度起電力×を考慮した渦電流解析が必要である。渦電流(eddy current)を解析する場合には、磁気ベクトルポテンシャルや電流ベクトルポテンシャルを用いた方法が用いられる。ここでは、薄板に適した電流ベクトルポテンシャル法に基づいた定式化を示す。電流密度ベクトルをとして電流密度連続の条件∇=0より、電流ベクトルポテンシャルを次式によって定義することができる。ここで、σは導体の導電率、は電場である。次に磁束密度を2成分に分けて考える。この場合、上式を有限要素法で離散化すると次のような形の式が得られる。このため、式(11・127)を運動方程式と連成して解く必要がある。薄肉シェル構造の場合には、平面シェル要素や曲面シェル要素を用いた有限要素法により、力とトルクの釣合い方程式は次式で与えられる。このとき、式(11・127)にはクランク・ニコルソン法、式(11・128)にはニューマークのβ法などの時間積分手法を用いることができる。時間ステップを十分に小さくとれば、精度の良い解が得られることが知られている。電気系と機械系を連成させる方法として、速度起電力を×、電磁力を×とした場合には、連成系は完全に線形となるので、電気系と機械系の線形方程式を組合わせた連立線形方程式を作ることもできる。薄板の磁気減衰現象の解析法の検証のためによく用いられている実験として、磁場中で片持支持されたはりの振動問題があり、TEAM ワークショップ(Testing Electromagnetic AnalysisMethod、数値電磁界解析に関する国際ワークショップ)のベンチマーク問題の12番として知られている。さらに、ねじれ振動と磁場との連成現象についても、詳細な実験と解析が報告されている。構造内部に音響入力を有する構造物は、構造外部へ直接空気伝ぱして音になる場合と、構造物の振動と構造内部の音場(soundfield,acoustic field)の音圧との連成(coupling)によって構造物が振動し音となる場合がある。一方、振動入力を有する構造物は、固体伝ぱにより音を生じるが、音響入力の場合と同様に、構造物の振動と構造内部の音場の音圧との連成を生じることがある。連成が生じる場合、構造物と音場を同時に考慮して解析する必要がある。そこで本節では、音場の基礎方程式より構造内部および外部の音場に関して、理論を簡潔にまとめる。
