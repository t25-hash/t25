# 11・5・2 領域境界条件

> **章**: 第11章 連成問題
> **節**: 11・5 音場

領域境界Γにおける音圧の法線微分옿옿について、一般に次式が成立する。옿옿=-ωρ욈(11・137)ただし、:虚数単位、ρ:音場密度、욈:音場粒子速度である。音場粒子速度は音場粒子変位욈の1階時間微分である。a. 完全反射境界完全に音波を反射する剛体壁では、音場粒子速度욈が零、すなわちインピーダンスが無限大となる。욈=0を式(11・137)に代入すると옿옿=0 on Γ(11・138)となる。b. インピーダンス境界(impedance boundary)吸音効果のある境界面では、法線方向インピーダンス욈は、욈=욈で与えられる。욈=욈を式(11・137)に代入すると옿옿=-ωρ욈on Γ(11・139)となる。c. 振動境界振動している境界では、法線方向において構造変位욈が音場粒子変位욈と等しいとおける。욈=욈on Γ(11・140)構造変位욈、音場粒子変位욈が角周波数ωで調和的に振動しているとき、욈=욈,욈=욈と表現できる。式(11・137)(11・140)より、옿옿=-ρ옿욈옿=-ρ옿욈옿=ρω욈(11・141)構造物の振動と音場の音圧との連成を考慮する場合、この式(11・141)を用いる。連成が無視できる場合もあるが、構造物の振動によって音が生じている場合には、必ず必要になる境界条件である。d. 音圧規定境界音圧の値が与えられる境界では、音場粒子変位욈の2階時間微分である音場粒子加速度σ욈、構造外部の音圧として、次式で表される。=ρ욈욈+on Γ(11・142)옿옿=1욈ω-(11・143)e. 開口境界開口境界では音圧を零と考えることもできるが、実際には開口境界補正として、インピーダンス욈を設定する。
