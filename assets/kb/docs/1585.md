# 11・5・5 構造物と構造内部および外部の音場解析

> **章**: 第11章 連成問題
> **節**: 11・5 音場

一般的な音場を考える場合、振動状態や構造内部、あるいは構造外部の音場との相互作用を考慮する必要がある。入力および応答はすべて列ベクトルであり、また、それぞれの入力と応答間の周波数応答関数をマトリックスで表すと、これらの関係は次式となる。a. 構造物の振動と音場の音圧との連成問題自動車室内などのような構造内部の音場において、構造物の振動が音場の音圧に影響し、また逆に音場の音圧が構造物の振動に影響を与えるとき、両者の連成を考慮する必要が生じる。構造内部の振動入力および音響入力と、振動および音圧の関係は、式(11・176)の周波数応答関数マトリックスの3行および3列めを削除した次式となる。しかし、力加振または音響加振による実験からは、各周波数応答関数は容易に測定でき、これらは構造物の振動と音場の音圧の連成が必ず含まれているので、解析との相関から連成度が評価できる。b. 構造物からの放射音問題構造物からの放射音は、固体伝ぱ音(structure-borne sound)と空気伝ぱ音(air-borne sound)に分類でき、振動入力と音響入力に起因して生じることから、式(11・176)の周波数応答関数マトリックスの3行めから次式となる。さらに、固体伝ぱ音に限定すると、式(11・178)の周波数応答関数マトリックスの1列めから次式となる。
