# 11・6・1 気液二相流の連成解析

> **章**: 第11章 連成問題
> **節**: 11・6 混相流

気液二相流の連成解析に用いられる手法を大きく二つに分けると、場を連続体として取扱う手法と非連続体として取扱う手法に分類される。場を連続体として扱う手法は、流体をマクロな視点からとらえた質量保存式・運動量保存式・エネルギー保存式を用いるが、本手法は、さらに空間を格子に分割する手法と、格子を用いない手法に分類できる。格子を用いる手法は比較的歴史が古く、適用する流動挙動に応じて多くの計算方法が提案されている。また、格子を用いない手法では、おもに格子を用いる手法では取扱いが困難とされる、気液二相の界面の大変形を取扱うことが比較的容易であるため、近年注目されている。さらに、場を非連続体として取扱う離散系シミュレーションの手法では、流体をミクロな視点からとらえ、流体の原子・分子、あるいはそれらの集合体の挙動を追跡する手法といえる。この手法はおもに、連続体近似が成立しない流動現象に対して適用されることが多いが、界面近傍の詳細な情報が比較的簡単に得られるため、連続体モデルの高度化にも利用される。本節では、この中から格子を用いた連続体近似シミュレーションとして、VOF法とその改良版であるMARS 法およびレベルセット法、格子を用いない連続体近似シミュレーションとしてMPS 法、離散系シミュレーションとして格子ボルツマン法を取上げる。a. 粒子法による連続体近似シミュレーション多相流解析では、例えば二流体モデルのように、界面の情報を場で平均化した方程式を用いた解析が多く用いられてきたが、コンピュータの処理能力の発達により、有限差分(体積)法を用いて、界面形状を高精度に評価する手法が多く用いられるようになってきた。. VOF法とMARS 法ハート(Hirt)とニコルスによって開発されたVOF法(volume of fluid method)は、計算格子を液体充てん率(0≦≦1)および周囲のセルの状況により気体(=0)、液体(=1)、および液体が部分的に存在するセルに分類し、この液体充てん率の輸送方程式を、ナビエ・ストークス方程式より求めた速度場に沿って移流させる。VOF法では気液二相流の輸送について、局所的な移動体積を保存させるためにドナー・アクセプタ(donor-acceptor)法を用いている。このVOF法を用いて、冨山らによって気泡流の詳細な解析が二次元、三次元で行われているが、ドナー・アクセプタ法では界面のこう配の輸送を考慮していないため、進行方向に斜めの界面が計算を続けるうちに、やがて流れに垂直になってしまうという問題が指摘されている。この問題点を解決する方法として、移動体積流束の計算を行う際に、界面のこう配を考慮したMARS(multi-interface advec-tion and reconstruction solver)法が、功刀らによって提案された。ここで>は空間平均を表す。この定義を用いて、式(11・181)(11・182)の多相流体の連続式と運動方程式を解く。. レベルセット法サスマン(Sussman)らによって提案されたレベルセット法(level set method)では、気液界面からの距離関数(レベルセット関数)φを全計算領域に配置し、その上に気液相の各物性値を配置して、この関数を移流方程式に従って解くことで界面追跡を行う。レベルセット関数の絶対値は、その点から最も近い気液界面までの距離を表しており、∇φ=1(11・183)が成り立つ。さらに次式で与えられるヘビサイド関数を、φの関数として式(11・184)より定義する。レベルセット法では、φが流れに沿って移流するに従い、式(11・183)で示された距離関数としての性質が失われていくため、このままでは式(11・186)(11・187)より求められる単位法線ベクトルや界面曲率が正しく評価できなくなる。このため、ある時間間隔′ごとに、φが距離関数としての性質を回復させるために、式(11・188)で表される再初期化を施す。b. 粒子法による連続体近似シミュレーション二相流の数値シミュレーションには、前述のように有限差分(体積)法等の格子を用いて、界面形状を詳細に追跡する手法が多く用いられ、種々の成功が収められているが、連続体の大変形、分裂、合体等を計算することは難しい。そこで格子を用いないシミュレーション手法として、流れとともに計算点が移動する粒子法が用いられるようになってきた。粒子法の代表例として、カーネル関数の重ね合わせで空間分布を表現するSPH(smoothed particle hydrodynamics)法があり、おもにこれまで天文学の分野の圧縮性流れの数値解法として用いられてきた。SPH 法において、粒子数密度を一定にするような圧力ポアソン方程式を導き、非圧縮性流れの解析ができる粒子法が、MPS 法である。. MPS 法MPS 法(moving particle semi-implicitmethod)では、非圧縮性流れの方程式中に現れる微分演算子(こう配、発散、回転、ラプラシアン)を粒子どうしの相互作用に変換する。ここでは二つの粒子間の距離である。この粒子の移動が、運動量保存則の移流項の計算に相当する。第二段階では式(11・194)の圧力のポアソン方程式を解き、非圧縮性の連続の式を満足するようにする。図11・6はMPS 法に、流れの方向に局所的に一次元座標軸を考えてメッシュを生成する補間法MAFLを組合わせて、サブクール核沸騰における単一気泡の成長と、離脱のシミュレーションを行った例である。離脱した気泡の後流によって、温度境界層が引上げられている様子が再現されている。このように気泡界面の著しい変形とともに、壁面や界面近傍での薄い温度境界層を精度良く再現することに、非格子法は威力を発揮する。c. 離散系シミュレーション近年の計算機の発達により、流れを連続体とみなして巨視的方程式を解く代わりに、原子・分子レベルから流れを再現する手法が注目されている。これらの離散系粒子のダイナミクスに基づく数値計算手法の一つに、格子ガスオートマトン法や格子ボルツマン法(lattice Boltzmann method)がある。格子ガスオートマトン法と格子ボルツマン法は、実在気体の原子・分子の運動の本質的な特徴をできる限り取り込みつつ、非常に簡単化された運動を行う仮想粒子の挙動を追跡して流れを再現する。この特徴から格子ガスオートマトン法は、原子・分子の挙動を直接シミュレートする、分子動力学に代表されるミクロスケールの解析手法と、巨視的力学方程式を解く差分法や有限体積法等のマクロスケールの解析手法の中間に存在するという意味で、メゾスケールの解析手法と呼ぶことができる。格子ガスオートマトン法は統計ノイズが多いことや、動力学方程式から導出される運動量保存式中に非物理的な保存量が現れるため、正しく流体を解くためには低い流速の流れでなければならないなどの制約が指摘されている。この問題を解決するために、格子ガスオートマトン法の改良手法として格子ボルツマン法が提案された。. 格子ボルツマン法格子ボルツマン法においては、格子ガスオートマトン法と同様に、離散化された空間・時間に対する粒子の発展方程式を解く。初期の格子ボルツマン法では衝突関数Ωに、格子ガスオートマトン法の衝突規則と同様のものを用いていたが、ナビエ・ストークス方程式(Navier-Stokes equations)を正しく満たすには不十分で不必要に複雑であることがわかり、一般には次式のBGK(Bhatnagar-Gross-Krook)近似衝突オペレータが用いられる。式(11・197)は局所平衡分布からのずれに比例して、粒子速度分布関数が緩和されることを示している。局所平衡分布関数は、正しく運動量保存が導出できるように解析的に求められる。二相流解析は、一般に一成分モデルと二成分モデルに分類できる。特に一成分モデルの中では、熱力学および気体運動論から導出可能なスイフト(Swift)らのモデルが注目され、気液二相流への適応が研究されている。白い部分が気相を、黒い部分が液相を表している。図11・7(a)(b)では、最初、小さな気泡どうしが合体して徐々に大きくなるが、ほぼ円形を保ちながら上昇し、その後図11・7(c)では、比較的大きな気泡が複雑に変形しながら合体・上昇していく。さらに、図11・7(d)(e)では、一つの大きな砲弾型のスラグ気泡が流路断面を満たすように上昇していることがわかる。
