# 11・6・2 固気二相流

> **章**: 第11章 連成問題
> **節**: 11・7 は稲室らによる上昇気泡の合体の様子である。モートン数

個々の粒子の運動軌跡をラグランジアン的に計算し、それと連成して、気流についてはナビエ・ストークス方程式を解く方法がよく用いられる。粒子濃度が高くなると、粒子どうしの二体衝突が加わり、さらに高粒子濃度になると多体衝突を考慮しなければならない。高粒子濃度の現実の流れ場の粒子を、1個1個計算することは到底不可能である。そこで最終的には連続体モデルが必要となる。そのとき、連続体化した粒子相をラグランジアン的に計算するSPH法が有効であろう。これらについて以下に概説する。a. 高粒子濃度固気混相流の数理モデル高粒子濃度になると、粒子どうしの衝突が無視できなくなり、さらに高濃度になると、衝突・接触が運動を支配する。数理モデルは、(1)ハードスフェア(hard sphere)モデル、(2)ソフトスフェア(soft sphere)モデル、(3)二流体モデルが挙げられる。. ハードスフェアモデルこのモデルは乱流流動層、循環流動層、通常の空気輸送など、比較的低粒子濃度で、二体衝突しか起こらない場合に使われる。衝突後の粒子速度は衝撃方程式から求めることができ、あとは前述した支配方程式(11・206)(11・207)で計算を進めればよい。ただし、すべての粒子を逐一計算するので、計算可能な粒子数に限りがあり、実際の乱流流動層等への適用が困難な場合が多い。そこで気体力学に用いられる直接シミュレーションモンテカルロ法(DSMC法、direct simulation Monte Carlomethod)を近似的に適用し、代表粒子の計算で全体を表すことが行われている。計算結果は実験結果を近似的にではあるがよく表す。ハードスフェアモデルの分子衝突については、10・3・4項に記述されているので参照されたい。. ソフトスフェアモデルさらに粒子濃度が濃くなると、1個の粒子に同時に2個以上の粒子が衝突する。この場合が多体衝突で、衝突に要する時間はもはや零とはみなし得ず、衝突変形過程における力のやり取りを計算しなければならない。このような多体問題を取扱う方法としてソフトスフェアモデルがあるが、その中で離散要素法(DEM : distinct element method)は、簡単でかつ実際の現象を近似的によく表現するので、広く用いられている。分子衝突におけるソフトスフェアモデルの一例は、10・3・4項に記述されている。DEM では衝突粒子の接触点での変形から反力(ばねモデル)を求め、変形によるエネルギー消費から発生する粘着力を変形速度から求め(ダッシュポットモデル)、それらを粒子の運動方程式に考慮して粒子の運動を計算する。実際には粒子相互の重なりから変形量、変形速度を求め、作用力を計算する。粒径が小さくなり付着力が働く場合には、上式にファンデルワールス力等のポテンシャル力を、付着力の働く方向に考慮すればよい。ただしμは粒子の摩擦係数である。これらを、着目粒子に接触しているすべての粒子について加算し、ラグランジアン型運動方程式に代入すれば次式を得る。この粒子数では粗大粒子の場合を除き、現実の流動層を表現することは困難である。最終的には連続体モデル(通常、二流体モデル)に帰着する必要がある。.
