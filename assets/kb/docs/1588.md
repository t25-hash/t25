# 11・7・1 支配方程式

> **章**: 第11章 連成問題
> **節**: 11・7 燃焼流れ

燃焼場では、燃焼反応による発熱のために大きな温度・密度変化を生じ、さらに熱、物質、エネルギーの対流および拡散が複雑に干渉している。このため、燃焼場を支配する基礎方程式は非常に複雑なものとなる。燃焼場の支配方程式は、連続体理論と気体分子運動論から導くことが可能であり、以下のように与えられる。運動量保存式はよく知られたナビエ・ストークス方程式に対応し、次のように与えられる。また、と上付添字は、単位テンソルおよびテンソルの転置を表している。熱流束は、熱伝導、化学種の分子拡散によるエンタルピー変化、濃度こう配によって熱拡散を駆動する効果、いわゆるデュフォ(Dufour)効果および放射熱流束から、次のように与えられる。上述の基礎方程式に加えて、補助方程式として次のような状態方程式を用いる。
