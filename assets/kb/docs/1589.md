# 11・7・2 化学反応機構(kinetic mechanism),熱化学定数お

> **章**: 第11章 連成問題
> **節**: 11・7 燃焼流れ

よび輸送係数燃焼反応を記述する反応機構は非常に複雑であり,燃料によっては,数十種以上の化学種と数百組以上の素反応を考慮に入れる必要がある.メタン等に関しては50種程度の化学種と,300組程度の素反応機構が用いられている.燃焼場では,温度が常温から2000K 程度まで急激に変化するため,熱化学定数と輸送係数の温度依存性を考慮に入れる必要がある.一般にはJANAFのテーブル等や,アメリカ,サンディア国立研究所が開発したChemkinのデータベースを用いて与えることができる.
