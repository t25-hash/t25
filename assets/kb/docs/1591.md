# 11・7・4 乱流燃焼(turbulent combustion)の数値解析

> **章**: 第11章 連成問題
> **節**: 11・7 燃焼流れ

a. 時間平均モデルレイノルズ平均に基づく非反応性乱流とは異なり、乱流燃焼の場合には、ファーブル平均(Favreaverage、質量荷重平均)に基づく基礎方程式を解析する。レイノルズ応力項や乱流拡散流束項等のモデル化には、非反応性乱流の場合に開発されている-εモデルや乱流プラントル数、乱流シュミット数を用いたこう配拡散型のモデルが用いられる。乱流燃焼場の場合、これらのモデルに加えて化学反応項のモデルを導入する必要がある。渦崩壊モデルでは、乱流エネルギーの減衰率が反応速度を決定すると考え、燃料の反応速度を次のように与える。욞욊욄욆=-ρε(11・242)ここで、は混合分率や反応進行度変数の変動強度である。また、燃料、酸化剤、燃焼ガスの渦塊が崩壊する過程のうち、最も遅いものが反応速度を律速すると考えるマグヌッセンモデル(あるいは渦消散モデル)を用いる場合もある。そのほか、BML(Bray-Moss-Libby)モデルや局所ダムケラー数モデル、化学種の輸送方程式を解く代わりに化学種の確率密度関数の輸送を解析するPDF(probability density function)法などがあり、解析の対象となる乱流燃焼場の特性によって使い分けられている。b.
