# 11・8・1 流体関連振動の無次元数

> **章**: 第11章 連成問題
> **節**: 11・8 流体関連振動

流れの中の物体が流体から受ける力は、物体と流体の接触面を含む境界面で囲まれた、領域内の流体についての運動方程式から求まる。図11・11に示すように、流れの中の物体に関して検査体積をとり、その表面は流体に接すると物体に接するで構成されるとする。dを表面上の微小面積、dを微小体積、ρを流体の密度、を流体の速度ベクトル、を外向き単位法線ベクトル、욈を単位面積当たりの表面力、を重力加速度ベクトルとする。単位時間当たりの流体の運動量変化は、検査体積内の流体に作用す6―162計算力学る力に等しいので、次式が成り立つ。옿ρ옿d+ρd流体の運動量変化流入運動量物体からの力(11・253)=ρd+욈d+욈d重力表面力ここで、を代表長さ、を代表流速、μを粘性、を音速、とωを流れ変動の振動数と構造の角振動数とすると、式(11・253)左辺は検査体積内部の流体の運動量変化ρ、検査体積表面から流入する運動量ρであり、右辺各項は重力ρ、表面力(圧力ρと粘性力μ)である。これら各項の比をとると、表11・1に示す流体関連振動に関する無次元数が定義される。流れが振動するときにその振動成分に注目すれば、ωを代表速度とした振動レイノルズ数욅と、振動マッハ数が定義される。次に物体の運動に着目すれば、検査体積内に流入する流体の運動量と、物体の次固有振動数욈を用いて表される運動量変化の比から、換算流速욥が定義される。この量は流体の運動量が物体に伝えられる程度を表している。次に物体と流体がともに運動しているとき、両者の慣性力の比が質量比である。スクルートン数あるいは質量減衰パラメータδは、物体の対数減衰率と質量比の積であり、減衰によるエネルギー散逸と物体のエネルギーの比を表す。
