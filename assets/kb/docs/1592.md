# 11・8・1 流体関連振動の無次元数

> **章**: 第11章 連成問題
> **節**: 11・8 流体関連振動

流れの中の物体が流体から受ける力は、物体と流体の接触面を含む境界面で囲まれた、領域内の流体についての運動方程式から求まる。図11・11に示すように、流れの中の物体に関して検査体積をとり、その表面は流体に接すると物体に接するで構成されるとする。単位時間当たりの流体の運動量変化は、検査体積内の流体に作用す6―162計算力学る力に等しいので、次式が成り立つ。これら各項の比をとると、表11・1に示す流体関連振動に関する無次元数が定義される。流れが振動するときにその振動成分に注目すれば、ωを代表速度とした振動レイノルズ数と、振動マッハ数が定義される。この量は流体の運動量が物体に伝えられる程度を表している。次に物体と流体がともに運動しているとき、両者の慣性力の比が質量比である。スクルートン数あるいは質量減衰パラメータδは、物体の対数減衰率と質量比の積であり、減衰によるエネルギー散逸と物体のエネルギーの比を表す。
