# 11・8・4 計算力学手法の応用

> **章**: 第11章 連成問題
> **節**: 11・8 流体関連振動

流体関連振動(flow-induced vibration)には、直交流あるいは平行流に対する物体の振動、シェルと流体の連成振動、管内流れによる管の振動、流れの中の物体の渦励振、管群の流力弾性振動、水撃などの圧力波伝ぱに伴う物体の振動などがある。実験による観察や、線形化などの単純化に基づく理論的考察により、これら流体関連振動のメカニズムの研究やモデル化がなされている。計算力学手法としては、非比圧縮性流れと円柱の渦励振がよく研究されており、以下に応用事例を示す。円柱構造物の流体関連振動に関しては、大金らがMAC図11・13 cell)法に基づく有限差分法とALE法を用いた。野村や山口らは、ALE有限要素法を適用して高レイノルズ数での解析を行った。有限要素法では、BTD(balancing tensor diffusivity)法やSUPG(streamline-upwindPetrov-Galerkin)法が安定化手法として用いられる。三次元特性に関しては、泉らはSIMPLER(semi-implicit method forpressure-linked equations revised)法と移動境界適合法を用いて、弾性支持円柱の三次元流体構造連成解析を行った。田村らは圧力と流速の定義点を同じとするMAC法と、移流項の差分化に三次精度の河村・桑原スキーム用いて三次元解析を行った。運動する物体の境界を表現するため、時間を含めた一般座標変換を用いて力振動評価指針」の解説で、数値シミュレーションにより流力振動を再現した例として引用されている。Kasseraは、SIM-PLE(semi-implicit kinematics)法を用いて三次元解析を行った。オイラー座標法を用いているが、計算格子を時々刻々切りなおすことにより移動境界を表現している。以上に述べた手法はそれぞれに成熟しつつあり、円柱構造のように簡単な体系のみでなく、熱交換器の管群の流力弾性振動解析などにも適用されている。流体関連振動では、計算格子の変形といった問題をいかに処理するかによって多様な解析方法が考え出されているが、最近は計算格子を使わない方法も開発されており、今後の動向が注目されるところである。流体関連振動の多くは三次元的であり、現象も複雑である。より複雑な体系の三次元解析によって、振動のメカニズムを解明する研究が指向されているといえる。また、より実際的な問題に対して応用が進みつつあること、市販の数値流体力学(CFD
