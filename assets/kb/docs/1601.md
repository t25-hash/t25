# 13・2・4 領域分割（domain decomposition）

> **章**: 第13章 シミュレーション支援法
> **節**: 13・2 ベクトル・パラレル

有限要素法や差分法など，ある領域を対象として解析を行うよう
な場合の並列解析では，領域を部分領域に分割してそれに対応した
データを分散メモリに割当て，各部分領域の計算を並列に実施す
る．ただし，一般に部分領域ごとの計算は完全には独立ではなく，
領域全体の整合性をとるために部分領域間での通信が必要となる．
したがって，領域分割に基づく並列処理が効率的に行われるために
は，部分領域間で通信ができるだけ少なくて済むように，データが
局所化されている必要がある．

なアルゴリズムを適用する場合にはその収束が速い．領域分割のツ
ールとしては，METIS が広く用いられている．
