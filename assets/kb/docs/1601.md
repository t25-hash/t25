# 12・1・6 エキスパートシステムに関連するAI 技術

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・1 エキスパートシステム

a. セマンティックウェブ(semantic web)大規模な知識ベースの構築は極めて困難なはずである。ところが、自然発生的にWWW(world wide web)が大規模知識ベースの性格をもつようになってきた。HTML(hypertext markup language)でタグ付けされた半構造データが、単純なオブジェクトあるいは情報提供ネットワークとして利用できるようになったのである。しかし、HTMLベースのウェブは、人間が読んで理解することに主眼がおかれていた。セマンティックウェブは、情報に意味を与えるタグなどのメタデータを、コンピュータでも自動処理し理解できることを可能としたウェブのことである。今後の大規模なウェブ(web)アプリケーションの高度化を考えるうえで、セマンティックウェブの果たす役割は大きい。b. 制約論理プログラミング(constraint logic programming)Prolog に代表されていたロジックによる知識表現は、第5世代コンピュータプロジェクトで大いに注目を集めた。当初効率の悪かった処理系の性能も著しく改善され、記号処理を中心としたシステムを実現するうえで支障はなくなってきている。ロジックプログラミングでは、単一化の演算により推論処理が行われるが、アプリケーションシステムを開発するためには、変数間の制約処理が手軽に記述できることが望ましい。このような考え方に基づいて進展してきたのが、制約論理プログラミングである。制約論理プログラミングは、計画型エキスパートシステム、特に動的なスケジューリングシステムで許容解を求めるのに、しばしば利用されている。c. データマイニング(data mining)と機械学習(machinelearning)エキスパートシステムを開発するうえで大きなボトルネックになったのが、知識獲得の問題である。これは、専門家が記号を中心とした知識表現で知識を表現できないという問題と、実際にシステムで利用できるような専門知識が意外に少ないことが原因である。そのため、実際に運用されているシステムの性能の多くの部分は、いかに大量かつ有用なデータを利用しているかに大きく左右される。そこで有効なのが、データベースと事例ベース推論・メモリベース推論、あるいは帰納学習の手法である。分類知識を学習するための帰納推論については非常に多くの研究開発がなされており、C4.5ならびにその後継システムがよく使われている。また、相関ルールの学習アルゴリズムなどが定着してきた。これらが、従来の統計手法と統合されて、データマイニング技術の中心となっている。
