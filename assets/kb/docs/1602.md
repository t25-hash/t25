# 12・2・1 はじめに

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・2 ファジィ推論

構造体や熱流体に代表される連続体力学の数値解法として、有限要素法(FEM : finite element method)が広く利用されている。その最大の理由の一つは、任意形状に対する解析が容易に行えることである。しかしながら、解析を行う場合にはその前処理(プリプロセッシング)として、多次元の解析領域を三角形や四面体等の有限要素に分割する作業が必要となり、大規模かつ複雑な形状を解析対象とする場合には、何らかの自動化が必要となる。特に最近のように、スーパーコンピュータや超並列計算機を用いて、数十万から数百万元規模の連立一次方程式の高速求解が可能な環境が整い始めると、一連の有限要素解析作業において、メインプロセッサ負荷に対するプリプロセッシング負荷の割合が急激に増加する。同様にこのような環境整備に伴い、得られる数値解の数も急激に増加するため、数値解の妥当性や現象論的な挙動の把握を自動的に行うための手段(ポストプロセッシング)が必要となる。すなわち、スーパーコンピュータや超並列計算機を用いて効率的な有限要素解析を実現するためには、それぞれの処理段階における自動化が不可欠となる。以上のような観点から、これまでに数多くの手法が研究・開発されてきた。しかし、複雑かつ大規模な系を解析の対象とする場合、計算精度や入力データ作成効率の点から、解析者の要求を満足できるレベルまで到達しているものは数少ない。これは、そもそも解析者が要求する条件の多くがかなり経験的なものであり、必ずしも数学的に定式化できるものではないためである。また、仮にこれが定式化できたとしても、それを複雑な多次元の解析問題に応用できるだけの、数値計算技術が存在しないこともその一因として挙げられる。本節では、有限要素解析等の自動化を実現するためのキーテクノロジーとしてファジィ推論(fuzzy inference)に焦点をあて、プリプロセッサ部、メインプロセッサ部およびポストプロセッサ部への、その応用技法について述べる。
