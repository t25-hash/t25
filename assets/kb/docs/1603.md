# 12・2・2 プリプロセッサ部への応用

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・2 ファジィ推論

a. 概要有限要素分割とは、任意の多次元解析空間を任意の線分、三角形や四角形、あるいは四面体や六面体に分割することである。この場合解析の熟練者は、解析空間全体の中で特に特徴的な部分(例えば、き裂先端、円孔、コーナ部等の応力集中部)に対して、過去の経験や理論的知識に基づくある分割イメージをもっていると考えられる。しかし、この分割作業におけるイメージの組合せでは、それぞれの有効範囲の決定や、それらの滑らかな結合といった作業にあいまいさが含まれる。そこでこのあいまいさを、あらかじめデータベースとして用意した過去の解析経験や数学的理論から、ある程度類推される局所的に最適な要素分割あるいは節点パターンを、ファジィ理論によって表現し、任意解析問題に応用することが有効である。b. 基本原理有限要素分割の一例として、き裂の近傍に円孔が存在し、二つの応力集中部が近接する場合を考える。まず、き裂近傍と円孔周辺のように応力が集中する部位の節点パターンとしては、無限領域にそれらが個別に存在する場合であれば、それぞれ図12・4(a)(b)に示すようなパターンが一般的に望ましいと考えられる。一方、き裂と円孔が近接する場合では、二つの節点パータンを単純に組合わせると図12・4(c)のようになり、要素分割を行うためには両節点パターンが重なっている部分での節点の取り合いを行う必要がある。この取り合い作業における、き裂先端近傍の点か否かといった分類は、本来あいまいなものである。このあいまいな分類を表現するために、メンバシップ関数(MSF:membershipfunction)を利用すると、き裂先端近傍の点や円孔近傍の点といった分類は、図12・5のように表現できる。図12・5ではMSFの値が大きいほど、その点の近傍の度合い(grade)が高いことを表す。(a) 円孔まわりの節点パターン(b) き裂先端まわりの節点パターン(c) 二つの節点パターンの重ね合わせ図12・4 き裂先端近傍および円孔近傍での節点パターンこのような表現方法を用いると、図12・6(a)の節点パターンの重ね合わせは図12・6(b)のように表せ、さらにそれぞれの点において、度合いの高いMSFを採用することにすれば、図12・6(c)のようなMSFの包絡線が規定され、全領域を自動的にき裂先端近傍領域と円孔近傍領域に分離可能である。また一般的には、き裂先端や円孔のような応力集中部に近いほど密な節点配置が要求されることから、MSFを節点密度の情報と関連付けると、図12・6(c)はき裂先端と円孔が近接する場合の節点密度分布を表すことになる。最終的に、この節点密度分布に応じた節点の配置を行うと、図12・6(d)のような細かい要素と粗い要素を、解析熟練者が描くイメージどおりに配置することができる。なお、要素分割の対象となるすべての領域を、円孔やき裂などの特徴的な形状の節点パターンだけで覆うことは困難であることから、領域全体を一様に分割するための節点パターンを併用することが有効である。c. メンバシップ関数の規定方法近傍の度合いを表すMSF(a) 円孔まわりのメンバシップ関数(b) き裂先端まわりのメンバシップ関数図12・5節点パターンに対応したメンバシップ関数(a) 節点パターンの重なり(b) メンバシップ関数の重なり(c) メンバシップ関数の和集合処理(d) 最終節点分布図12・6 メンバシップ関数の重ね合わせ6―169に節点の密度(例えば、隣接する節点間距離の平均値の逆数)を対応させると、MSFの定義が容易になるばかりでなく、異なる節点パターンどうしの結合部で、節点間隔を連続的に表現することができる。MSFは0から1の値をとる関数であるが、個々のMSFの最大値が1となるようにあらかじめ規格化して表現し、それらを重ね合わせる段階で、解析者が与える係数によって調整を行う方法が便利である(図12・7)。
