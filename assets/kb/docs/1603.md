# 13・2・6 アムダールの法則（Amdahl’s law）

> **章**: 第13章 シミュレーション支援法
> **節**: 13・2 ベクトル・パラレル

プログラムのベクトル化率とベクトル化による実効性能向上の関
係，あるいは，プログラムの並列化率と並列化による実効性能向上
の関係は，ともにアムダールの法則と呼ばれる．

トル化率と呼ぶ．このプログラムをベクトルプロセッサで実行する

ベクトル処理もしくは並列処理による演算時間の短縮の度合いを
評価するアムダールの法則においては，計算全体の演算量は一定に

場合，プロセッサ（あるいはノード）の数を増やすにつれて，計算
規模も大きくして実行するということに注意しなければならない．
