# 12・2・3 メインプロセッサ部への応用

> **章**: 第12章 インテリジェントシミュレーション

a. 概要FEM による数値解析では、連続体挙動を支配する偏微分方程式の汎関数微分(functional derivative)を求め、変分原理(variational principle)によって方程式を離散化して、微小領域に対する式を規定し、その解を逐次求めていく操作がその中心となる。また、不規則な外力や衝撃荷重による応答評価等の、いわゆる非線形現象の動的解析においては、時間項に対する高精度な数値積分が必要となる。この時間積分を行う場合、時間刻み幅の選び方によっては、数値計算系が不安定な挙動を示し、最悪な場合にはこの数値計算系を発散に導く。しかし解析の熟練者は、数値計算系の挙動を逐次監視し、不安定な挙動が少し現れたら、時間刻み幅を少し小さくするなど、極めてあいまいな判断に基づいて、数値計算系がつねに安定な状態にあるように制御する。そこで、このあいまいな知識をMSFとして規定し、数値計算系(差分法コード)を自動的に安定な状態に保持する適応制御システムが実用化されている。b. 基本原理図12・8に示すように、解析コードを未知の動特性をもつ制御対象と考え、時間ステップごとに時々刻々計算される各種状態量φ욡から、時間ステップ間でのそれらの変動率[φ욡욗-φ욡욗φ욡욗,:時間レベル]を監視し、数値計算系の状態に応じ図12・7 メンバシップ関数の高さの調整図12・8 時間刻み幅のファジィ適応制御システム6―170計算力学図12・9数値計算系の状態評価図12・10 推論結果の合成て、次時間ステップで使用すべき緩和係数α[クーラン条件(Courant condition)にかかる係数]を制御変数として出力する。この場合の制御戦略としては、If-Then形式で記述されたプロダクションルール(production rule、例えば、変動率が若干大きく系が不安定であれば、緩和係数αを若干小さくして系を安定化させるなど)を使用する。具体的な処理手順は次のとおりである。(1) 計算領域内での最大変動率の読み込み(2) 数値計算系の状態判定とその度合いの算出読み込まれる個の変動率個々について、If~に対応するMSFより数値計算系の状態を判定するとともに、それら個々の度合いを算出する(図12・9)。(3) プロダクションルールからの制御動作の決定プロダクションルールに従い、状態判定結果に対応する制御動作群“Then ~”を導出する。(4) 制御動作に対応するMSFの調整導出された制御動作群にするMSFに対して、(2)で算出された個々の度合いを掛け合わせる。(5) 制御動作群の合成と緩和係数αの規定複数の制御動作群を、単一の制御変数である緩和係数αに縮約するため、推論合成[重ね合わせたMSFに対して重心を求め、緩和係数に対する台集合(base set)上に投影:直接法]を行う(図上記処理により、複数の観測変数に対する判定結果から、単一の制御変数を通じた、ミクロな意味での自動制御を実現することができる。c. メンバシップ関数の自動修正ファジィ制御を行ううえで必要となるMSFの規定では、基本的に解析熟練者のもつ一般的な経験が用いられる。しかしながら、使用すべき経験則は対象とする解析問題に応じて異なるため、事前に規定したMSFが適切でなくなる場合が多い。このため、事前に規定したMSFから出力される制御動作が適切であったか否かを内部で自動的に判断し、各問題について良好な制御動作が規定されるようMSFを修正する機能をもたせることが有効である。その手順は以下のとおりである。(1) 計算領域内での最大変動率の読み込み(2) ファジィ推論による制御動作の規定(b項の基本原理でのシーケンスに従う)(3) 制御動作の内部評価If 制御動作が良好であれば、Go to(4)Else図12・11 メンバシップ関数の内部自動調整制御動作“If ~”に対応するMSFを、その台集合上で単位量だけスライド(図12・11)EndifGo to(2)(4) 修正完了以上の処理によって、評価問題に応じた適切なファジィ制御、いわゆる適応制御が実現できる。
