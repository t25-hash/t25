# 12・2・5 終わりに

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・10 )。

本節では、有限要素法等による解析作業の三つのプロセス、すなわち要素分割などの解析準備を行うプリプロセッサ部、作業の中心である数値計算を行うメインプロセッサ部、および解析結果の評価を行うポストプロセッサ部での負荷を、知識処理手法としてのファジィ推論を利用して、軽減させる方法とその有効性について述べた。昨今の超高性能計算機の出現は、有限要素解析をさらに一般化し、また取扱うことのできる解析規模を大きくすることから、このような解析作業における数値計算処理の高速化はもちろんのこと、前後処理の負荷をも軽減して、解析作業全体の効率化をよりいっそう進めることが求められる。このような課題を解決する手段としてのファジィ推論技術は、比較的簡単なアルゴリズムによって、満足のいく結果を提示してくれる有効な手段の一つであると考えてよい。大規模解析の実施においては、数値計算処理の高速化と同時に、これに付随する周辺の知識処理部を整備して、より効率の良い作業が望まれるであろう。
