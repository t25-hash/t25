# 12・3・1 はじめに

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・3 ニューラルネットワーク

人間の脳には約140億個の神経細胞(ニューロン)があり,それらは並列で情報処理を行っている.このニューロンの働きをまねた人工的な数学モデルをユニットと呼ぶ.ニューラルネットワーク(NN : neural network)はこのユニットを多数,複雑に結合したネットワークであり,人間の脳神経回路網を参考にしているが,現在実現しているニューラルネットワークに含まれるユニットの数は,脳の神経細胞の百万分の一のオーダである.さらに,生物系では異なった機能をもつ神経回路網の階層化によって複雑な動作を可能にしており,現状のニューラルネットワークは,そのごく一部のモデル化にすぎないと考えられる.しかしながら,現在あるニューラルネットワークでも,パターン認識,数式化や定式化が困難な問題の新解法,逆問題解析,組合せが膨大な問題の準最適解を求めるなど,さまざまな分野へ応用することが可能である.階層型ニューラルネットワークは,教師データ(tutor data)を用いての学習が可能なため,その適用範囲は相互型ネットワークよりもかなり広い.ここでは,ニューラルネットワークの詳細な記述は専門書に,その構造工学や材料力学等への応用は文献(13)(14)に譲ることにし,階層型ニューラルネットワークの概要のみを説明する.次に,複合材料の実験データを用いた応用例を紹介6―171図12・13 階層型ニューラルネットワークする.ここではおもに例題として順問題を扱う.逆問題については
