# 12・3・2 ニューラルネットワークとは

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・7 節を参照されたい。

これを順問題とすると、この逆問題は、次式で表せる。しかし、陽な形で表せないか、また表せたとしてもその評価が大変な場合には、階層型ニューラルネットワークが有効となる。一方、逆問題において、が陽な形で表されたとしてもの陽な形が存在しない場合、また解の一意性の存在も保証されない場合に、階層型ニューラルネットワークがその威力を発揮する。ニューラルネットワークによる情報処理は大まかに、次の手順で行われる。(2) ネットワークの学習図12・13に示す階層型ニューラルネットワークの入力層にを与えて、ニューラルネットワークに誤差逆伝搬学習則などを用いて学習させ、出力層からが得られれば、順問題を表す式(12・1)の関係が構築できたことになる。(3) 解析学習を終えたニューラルネットワークを用いて、目的とする解析を行う。
