# 13・3・6 ビジュアリゼーションから解釈へ（効率的な可視化と

> **章**: 第13章 シミュレーション支援法
> **節**: 13・3 ビジュアリゼーション

その理解）
半透明等値面によるスカラ場の可視化と，ベクトル場に対する粒
子追跡法による可視化を中心とした複合的なものが，現在の主流で
ある．一方，計算性能の向上とソフトウェアの整備，さらにプレゼ
ンテーション技術の進展によって，ボリュームレンダリングやLIC
のような画素ベースの可視化も盛んに利用され始めている．
今後は，大規模化する計算データに対して，計算環境を踏まえ
て，できるだけ効率的に（むだな可視化を行わずに），必要な情報
を得るためのビジュアリゼーションへと移行していくだろう．この
際，何がわかるのか，といったビジュアリゼーションの本質が議論
され，分野に特化した可視化手法や，知識工学との融合が模索され
ていくものと考えられる．
