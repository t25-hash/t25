# 12・3・3 実験データによる構築と検証

a. CFRP 積層材の損傷同定(damage identification)CFRP(carbon fiber reinforced plastics,炭素繊維強化プラスチック)積層材を軸方向に10要素で仮想的に分割し,1要素だけ損傷があるとした.損傷を表すために,厚さ方向の任意の位置で,はりの1要素分CFRP材をカットして,その部分をテフロンのフィルムで置換えて成形し,曲げ剛性を低減化させた.厚さ方向の位置を調整することで,健全部の曲げ剛性に対する損傷部の曲げ剛性比を次の三とおり,욃욋=0.57,0.45,0.42になるようにし,先端部の2要素分を除く8箇所の損傷位置を組合わせた24とおりのCFRP積層はりの,一次から三次までの固有振動数をインパルス加振法で求めた.図12・13の入力層に一次から三次までの固有振動数を与え,20個のユニットを有する中間層を2層用意し,出力層では8個のユニットをはりの1から8までの要素に対応させ,損傷のある場合を1,ない場合を0とすることで損傷位置をディジタル表示し,さらに,욃욋の値を加えた9ユニットとした.教師データとして욃욋=0.57,0.42とし,2.5万回の学習を通じて,ニューラルネットワークを構築した.次に,0.45のデータをネットワークの検証用に用いた結果を,図12・14に示す.6―172計算力学図12・14 実験値による損傷同定結果図12・15ニューラルネットワークの推論値と実験結果の比較▲印は욃욋=0.45に対するネットワークの推論値を示しているが,損傷位置については精度良い結果を示した.損傷量の各記号が該当する直線上にあれば,完全な学習と推論をしたことになり,ほぼ満足する結果が得られた.b. 減衰能から曲げ強度の推定CFRP材の曲げ減衰能と曲げ強度の間には,ある関係式があることは予測できるが,具体的な式を得ることは大変困難だといえる.そこで,両者の関係式をニューラルネットワークを用いて構築するため,図12・13の入力ユニットに減衰能と板厚を与え,出力ユニットに曲げ強度を考えた.ネットワークの学習には,板厚2mmと3mmの減衰能と曲げ強度に関する実験データ31組を教師用として用い,二乗誤差総和が最も収束した時点で学習を終了させた.次に,構築したネットワークの有効性を確認するため,学習に用いなかった板厚2.5mmの14組の実験データについて,ニューラルネットワークで推論を行い,実験結果と比較し,その結果を図12・15に示す.実験値(■印)と推論値(□印)の差は最大で20%程度になるが,実験値を最小二乗法で近似した実線と,推論値を近似した点線はほぼ一致している.これにより,比較的少ない実験データを用いて,ニューラルネットワークを学習させることで,内含する他のデータに対して,非破壊的手法で曲げ強度の予測が可能であると考えられる.また,実験による誤差等のノイズを考慮したニューラルネットワークの構築が可能となり,現場での実践的な使用も行えると考えられる.遺伝的アルゴリズム(GA : genetic algorithm)とは,生物の進化を模倣した最適化手法の一つであり,局所最適解を多くもつ複雑な最適化問題に有効な解法である.自然と人工システムにおける適応という問題を深く考察したホランド(Holland)がGA の概念を構築し,彼の学生たち,特にゴールドバーグ(Goldberg)が図12・16 染色体,交差,および突然変異GA を発展させた.十数年前には革新的な最適化アルゴリズムとして,多くの研究者たちが研究に取り組んだが,当時の計算機には計算負荷が大きすぎ,実最適化問題への応用には至らなかった.しかしながら,近年,(1)計算機の速度が極めて速くなり,高い計算負荷でも比較的短時間で解が得られること,および(2)実問題に有効な改良が数多く提案され,これまで有効な解が得られなかった問題に対しても実用的な解が得られるようになってきたことなどから,大きな注目を集めている.特に,GA と高並列PC(personal com-puter)クラスタとの親和性が高いことから,GA の応用は今後ますます拡大すると思われる.ここでは,GA の概略について述べる.
