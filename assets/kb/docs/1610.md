# 12・4・1 基本概念

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GA は生物の進化を模倣した大域的探索手法である.最も重要な概念は,多くの個体(individual,解)から成る母集団(popula-tion,解の集団)が,選択という操作により適合度(fitness)の高い集団に変化し,それらが交差(crossover)という操作によって子個体(offspring)を生み出し,これらが次世代の母集団を形成することである.ここで,適合度とは最適性の度合いを示すものである.一方,生物の多様性を維持しているメカニズムは突然変異(mutation)であり,これは遺伝子をランダムに変化させるもので,多くは無意味なエラーとなるが,まれに最適解に必要な形質を与えてくれる.GA で扱う個体の情報は,表現型(pheno-type)および遺伝子型(geno-type)の二層構造となっている.遺伝子型は,生物の細胞内の染色体(chromosome)に相当するものであり,これがGAにおける交差や突然変異などの,GA オペレータの操作対象となる.一方,表現型は,生物では個体そのものであり,環境との相互作用によって適合度が決まる.工学的な解では,遺伝子型は設計解のパラメータ集合,もしくは数値の2進数化など,ある種のコード変換を行ったものであり,表現型は設計解の実現形であり,シミュレーションなどによってその解の優劣を評価することができるものである.遺伝子(gene)とは,設計解の遺伝子型における一つの情報であり,遺伝子が複数集まって染色体を形成する.また,染色体上の遺伝子の場所を遺伝子座(locus)と呼ぶ.機械構造などの寸法を最適化する問題など,連続変数をもつ最適化問題では,通常,個体の染色体は各設計変数を2進数化したもので表され,図12・16の親や子のようになる.ここでは染色体の長さは11ビットであるが,これは設計変数に応じて適切に決める.この場合の遺伝子は0か1となる.また,複数の設計変数がある場合には,この染色体を長くすればよい.交差は二つの親個体から,それぞれの形質(解の特徴)を受け継ぎ,親と異なる個体を生み出す.交差には1点,2点,および一様交差などがあるが,最も単純な1点交差では,図12・16に示すように,ランダムな交差位置で二つの親染色体の前半部分と後半部分を組換えて,子染色体を生成する.一方,突然変異はランダムに選んだ遺伝子座の遺伝子を,ランダムに変化させる.図12・16では子1の第1遺伝子座が0から1に,子2の第4および第7遺伝子座の遺伝子が変化している.もし,全遺伝子座の1の多いほうが適合度が高いと仮定すると,子2′は最適解となる.交差と突然変異によって得られた子個体は,次に選択(selec-tion)という操作により,適合度の高い個体が高い確率で生き残る.ここで最も重要なことは,確率的に選択されるということであり,これによって局所解(local optimum)ではなく,大域的最適化(global optimum)を得る確率が高くなる.選択の操作には種々の方法があるが,最も基本的な方法は,適合度の高い個体ほど生き残る確率が高いという,ルーレット選択(roulette selection)である.この選択方法では,各個体は自身の適合度の値に応じて確率的に生き残るが,もし各個体の適合度の差が少ない場合には選択圧(淘汰圧,selection pressure)が低すぎるため,種々のスケーリング技法を用いて適合度を変換し,適切な選択圧が実現できるようにする.選択方法としてランキング選択やトーナメント選択もある.前者は適合度の値そのものではなく,適合度の順番で生き残り確率を決めるものであり,後者は選択に関係する個体の数を全母集団ではなく,数個体に制限するものである.トーナメントに関係する個体数をトーナメントサイズと呼ぶが,これを例えば3とすると,ランダムに三つの個体を選び,その中で最高の適合度をもつ個体を次世代に残す.これを繰返し,必要な個体数になるまで行うことで世代が交代する.このような選択方法全体を総称して,世代交代モデルともいう.
