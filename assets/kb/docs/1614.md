# 12・4・5 GA のパラメータ

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GAには多くのパラメータが存在する．最も重要なパラメータ
は，染色体長，母集団サイズ，交差率，突然変異率，終了世代数で
ある．連続変数を2進符号化した場合，一つの変数を ビットで
コード化し，設計変数の数を とすると，染色体長 は ビッ
トとなる．この場合， は解の精度から決めればよい．
一方，適切な母集団サイズ（全個体数）は，問題やGAオペレ
ータなどに依存しているため不明であり，50～100個体程度の値で
予備実験を行い，進化の推移を観察しながら実験的に決めなければ
ならない．ただし，突然変異率（mutation rate，遺伝子が変化す
る確率）は，従来の研究から0.001～0.01の値を用いるか，あるい
は1/（ ：染色体長）が経験的に良好であることが知られてお
り，この値を中心として予備実験を行えばよい．また，交差率
（ある世代の中で交差する個体の割合）は，経験的に0.45～0.95の
値，あるいは0.6が良好であることが知られており，さらに交
差率を変化させても結果への影響は少ないことから，適当な値に決
めても問題は少ない．終了世代数を適切に決めることは難しい．一
般には適合度の改善が見られなければ終了せざるを得ないが，それ
はパラメータの値や用いたGAオペレータに依存して，進化が停
滞しているだけかもしれない．したがって，未知の最適化問題を解
くとき，適切な終了世代を事前に決めることは困難であり，試行錯
誤に頼らざるを得ないが，その場合，解の進化過程を詳細に検討
し，進化の過程で何が生じているのかをできるだけ詳しく観察する
ことが，パラメータのチューニングや適切なGAオペレータの選
択につながる．
12・4・6多様性の維持
生物の進化において最も重要なことは，環境の変化に対して特定
の生物種が絶滅しないように，種内の遺伝的多様性を維持すること
である．同様に，GAでも個体の多様性を維持することが大域的最
適解の発見に役立ち，また変化する動的環境においては，それに対
応して適応する個体の生成に役立つ．
GAにおいて多様性を維持する最も簡単な方法は突然変異率を増
加させることであるが，突然変異は許容解にならない致死遺伝子が
多く，あまり効果的ではない．このため，種々のメカニズムが考え
られているが，最も重要なメカニズムは，母集団を分割する分割母
集団（島）モデル（island model）とMGG（minimal generation
gap）である．
集団遺伝学者のライト（Wright）は，生物の集団は地域的に分
断されていて，各地域での偶然の変化が，好ましい遺伝子を作るう
えで非常に重要であると考えた．この考え方をGAに適用した
のが分割母集団モデルである．このモデルでは母集団を複数のサブ
母集団に分割し，それぞれのサブ母集団内で独立した進化を行い，
それとともにサブ母集団間での解の交換［これを移住（migra-
tion）と呼ぶ］を適当な頻度で行う．これは別名，島モデル
（island model）とも呼ばれる．島モデルでは，サブ母集団におけ
るそれぞれの進化が互いに孤立して進むことから，各サブ母集団で
異なった局所最適解が得られる．このとき，サブ母集団間で個体が
移住する，あるいはサブ母集団間の個体どうしが交差すると遺伝子
の混合が生じ，また異なった局所最適解が生じる．このような状況
こそが遺伝的多様性を保持し，解探索の性能を高める．
上記の地域的孤立性をGAにおける論理的な世代交代モデルと
して考えたものが，MGGである．ここでは適応度を無視して2個
体をランダムに非復元抽出し，これらを親として子個体を数多く生
成し，それらと親個体から成る家族から最良1個体と，ルーレット
選択による1個体を次世代に残す．これにより，全体としては
選択圧を低くし，かつ生存選択の局所化を行い，上で述べた分割母
集団モデルにおいてサブ母集団数を極めて多くし，かつ多数の子個
体を生成して，交差の効果を最大限に活用する仕組みを入れたもの
に近い．分散GAにおいてサブ母集団数を最大まで増加させると，
各サブ母集団内の個体数は2となり，これが2個体分散GAであ
る．このモデルの性能は比較的高いうえに，パラメータチューニン
グの必要がない，並列処理が容易，そして階層的な分散モデルを組
みやすいなどのメリットもある．いずれにしても，これらのメ
カニズムによって遺伝的多様性が維持・増大し，最適解の探索能力
は高くなる．
12・4・7並列処理
GAは計算負荷が高いために並列処理が重要となる．特に，解析
部分での時間が比較的長い計算工学分野では，GAの並列処理は必
須であるが，幸いにも，GAの並列処理は容易である．
GAの並列処理には大別して二つのアプローチがある．それら
は，(１)複数個体の評価における複数の解析計算を並列化する，
(２)分割母集団モデルを用いて，各サブ母集団での並列化を行う，
である．このうち，(１)はマスタスレーブ型の並列処理で行うこと
ができる．交差や突然変異，あるいは選択などのGA操作をマス
タが行い，スレーブは子個体の適合度評価だけをマスタから指示さ
れて行う．この並列処理は，適合度の評価計算の時間が比較的長い
場合に有効である．そうでない場合には，マスタがスレーブに計算
を分配している間に待ち時間が生じ，計算効率が悪くなる．
一方，分割したサブ母集団にプロセッサを割当てる方式の並列処
理では，適合度の計算時間に関係なく，高い計算効率が得られ，か
つプロセッサ間通信は移住時のみに発生するため，ネットワークへ
の負荷が少なく，遅いネットワークを用いた安価なPCクラスタで
も高い性能を実現することができる．
12・4・8多目的最適化
最適化の中でも複数の目的関数をもつものを，多目的最適化
（multi - objective optimization）と呼ぶ．多目的最適化では，単一
の最適解を求めるのではなく，パレート最適解（Pareto- optimal
set）と呼ばれる最適解の集合を求める．この場合，複数の探索点
をもつGAは多目的最適化に極めて有利であり，現在ではGAは，
多目的最適化の標準的な解法となっている．
多目的GAは大別して三つの手法に分類できる．
(１)単一目的に変換（スカラ化）し，パラメータを変えてパレー
ト解を得る．
(２)母集団内で異なる目的関数を用いたり，複数の目的関数を順
に適用するなどを行い，母集団全体としてパレート解を得
る．
(３)母集団内の個体を何らかの方法でランク付けし，パレート解
を直接的に得る．
