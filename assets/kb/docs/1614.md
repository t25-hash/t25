# 12・4・5 GA のパラメータ

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GA には多くのパラメータが存在する。最も重要なパラメータは、染色体長、母集団サイズ、交差率、突然変異率、終了世代数である。この場合、は解の精度から決めればよい。一方、適切な母集団サイズ(全個体数)は、問題やGA オペレータなどに依存しているため不明であり、50~100個体程度の値で予備実験を行い、進化の推移を観察しながら実験的に決めなければならない。ただし、突然変異率(mutation rate、遺伝子が変化する確率)は、従来の研究から0.001~0.01の値を用いるか、あるいは1/(:染色体長)が経験的に良好であることが知られており、この値を中心として予備実験を行えばよい。また、交差率(ある世代の中で交差する個体の割合)は、経験的に0.45~0.95の値、あるいは0.6が良好であることが知られており、さらに交差率を変化させても結果への影響は少ないことから、適当な値に決めても問題は少ない。終了世代数を適切に決めることは難しい。一般には適合度の改善が見られなければ終了せざるを得ないが、それはパラメータの値や用いたGA オペレータに依存して、進化が停滞しているだけかもしれない。したがって、未知の最適化問題を解くとき、適切な終了世代を事前に決めることは困難であり、試行錯誤に頼らざるを得ないが、その場合、解の進化過程を詳細に検討し、進化の過程で何が生じているのかをできるだけ詳しく観察することが、パラメータのチューニングや適切なGA オペレータの選択につながる。
