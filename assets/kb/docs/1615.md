# 12・4・6 多様性の維持

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

生物の進化において最も重要なことは,環境の変化に対して特定の生物種が絶滅しないように,種内の遺伝的多様性を維持することである.同様に,GA でも個体の多様性を維持することが大域的最適解の発見に役立ち,また変化する動的環境においては,それに対応して適応する個体の生成に役立つ.GA において多様性を維持する最も簡単な方法は突然変異率を増加させることであるが,突然変異は許容解にならない致死遺伝子が多く,あまり効果的ではない.このため,種々のメカニズムが考えられているが,最も重要なメカニズムは,母集団を分割する分割母集団(島)モデル(island model)とMGG(minimal generationgap)である.集団遺伝学者のライト(Wright)は,生物の集団は地域的に分断されていて,各地域での偶然の変化が,好ましい遺伝子を作るうえで非常に重要であると考えた.この考え方をGA に適用したのが分割母集団モデルである.このモデルでは母集団を複数のサブ母集団に分割し,それぞれのサブ母集団内で独立した進化を行い,それとともにサブ母集団間での解の交換[これを移住(migra-tion)と呼ぶ]を適当な頻度で行う.これは別名,島モデル(island model)とも呼ばれる.島モデルでは,サブ母集団におけるそれぞれの進化が互いに孤立して進むことから,各サブ母集団で異なった局所最適解が得られる.このとき,サブ母集団間で個体が移住する,あるいはサブ母集団間の個体どうしが交差すると遺伝子の混合が生じ,また異なった局所最適解が生じる.このような状況こそが遺伝的多様性を保持し,解探索の性能を高める.上記の地域的孤立性をGA における論理的な世代交代モデルとして考えたものが,MGG である.ここでは適応度を無視して2個体をランダムに非復元抽出し,これらを親として子個体を数多く生成し,それらと親個体から成る家族から最良1個体と,ルーレット選択による1個体を次世代に残す.これにより,全体としては選択圧を低くし,かつ生存選択の局所化を行い,上で述べた分割母集団モデルにおいてサブ母集団数を極めて多くし,かつ多数の子個体を生成して,交差の効果を最大限に活用する仕組みを入れたものに近い.分散GA においてサブ母集団数を最大まで増加させると,各サブ母集団内の個体数は2となり,これが2個体分散GA である.このモデルの性能は比較的高いうえに,パラメータチューニングの必要がない,並列処理が容易,そして階層的な分散モデルを組みやすいなどのメリットもある.いずれにしても,これらのメカニズムによって遺伝的多様性が維持・増大し,最適解の探索能力は高くなる.
