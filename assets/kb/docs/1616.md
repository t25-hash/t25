# 12・4・7 並列処理

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

GA は計算負荷が高いために並列処理が重要となる。特に、解析部分での時間が比較的長い計算工学分野では、GA の並列処理は必須であるが、幸いにも、GA の並列処理は容易である。GA の並列処理には大別して二つのアプローチがある。それらは、(1)複数個体の評価における複数の解析計算を並列化する、(2)分割母集団モデルを用いて、各サブ母集団での並列化を行う、である。このうち、(1)はマスタスレーブ型の並列処理で行うことができる。交差や突然変異、あるいは選択などのGA 操作をマスタが行い、スレーブは子個体の適合度評価だけをマスタから指示されて行う。この並列処理は、適合度の評価計算の時間が比較的長い場合に有効である。そうでない場合には、マスタがスレーブに計算を分配している間に待ち時間が生じ、計算効率が悪くなる。一方、分割したサブ母集団にプロセッサを割当てる方式の並列処理では、適合度の計算時間に関係なく、高い計算効率が得られ、かつプロセッサ間通信は移住時のみに発生するため、ネットワークへの負荷が少なく、遅いネットワークを用いた安価なPCクラスタでも高い性能を実現することができる。
