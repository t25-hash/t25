# 12・4・8 多目的最適化

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

最適化の中でも複数の目的関数をもつものを、多目的最適化(multi-objective optimization)と呼ぶ。多目的最適化では、単一の最適解を求めるのではなく、パレート最適解(Pareto-optimalset)と呼ばれる最適解の集合を求める。この場合、複数の探索点をもつGA は多目的最適化に極めて有利であり、現在ではGA は、多目的最適化の標準的な解法となっている。多目的GA は大別して三つの手法に分類できる。(1) 単一目的に変換(スカラ化)し、パラメータを変えてパレート解を得る。(2) 母集団内で異なる目的関数を用いたり、複数の目的関数を順に適用するなどを行い、母集団全体としてパレート解を得る。(3) 母集団内の個体を何らかの方法でランク付けし、パレート解を直接的に得る。多目的GA の手法としては多くのものがあるが、なかでもMOGA(multi-objective GA)、NSGA-II(non-dominated sort-ing GA)、およびSPEA 2(strength Pareto evolutionary algo-rithm)は代表的な方法である。MOGA は、1993年にフォンセカ(Fonseca)によって提案された手法である。MOGA では、個体の評価をパレートランキング法に基づいて行う。この方法では、パレート解に近い解に高い適合度を与える。また、非劣解の多様性保持のための手法としてシェアリングを行う。シェアリング(sharing)とは、母集団内で個体が偏在して集合したとき、その個体集団の適合度を減少させて個体数を少なくする操作である。一般には一つの個体の近傍を調べ、ある距離内に別の個体が存在すると適合度を下げる。シェアリングは多目的GA だけでなく、単一目的のGA にも有効な方法である。NSGA-IIは、2001年にNSGA の改良アルゴリズムとして、デブ(Deb)らによって提案された手法である。この手法では、保存する母集団(親母集団)と、交差・突然変異といった遺伝的操作を用いた探索を行うための母集団(子母集団)の、二つの独立した母集団を用いて解探索を行う。また、親母集団と子母集団を合わせて非優越ソート(non-dominant sorting)を行い、全個体をランク付けする。またシェアリングも行う。一方、SPEA 2はSPEA の改良アルゴリズムとして、ツィツラー(Zitzler)らによって2001年に提案された。SPEA 2はNSGA-IIと同様に、非劣解を保存する母集団(アーカイブ母集団)と、交差や突然変異などの遺伝的操作を行う母集団の、二つの母集団を利用して探索を行うが、NSGA-IIとは適合度の割当て方法やシェアリングの方法などに多少の違いがある。また、渡邉、廣安、三木らはSPEA 2に近傍交差の概念を取り入れ、探索性能の向上のみならず、パレート解集合における一様性の向上を達成した。
