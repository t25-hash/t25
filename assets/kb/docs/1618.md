# 12・4・9 GA の応用

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・4 遺伝的アルゴリズム

近年、GA は計算工学の分野で広く用いられている。それらは文献(36)~(40)に詳しく記述されている。ここでは二つの例を挙げる。一つは航空機の空力設計に関して、GA を用いて多目的最適化を行ったものである。ここで設計変数は翼の形状、目的関数は遷音速巡航抵抗の最小化、超音速巡航抵抗の最小化、そして超音速巡航時の翼根の曲げモーメントの最小化である。制約条件は飛行速度や揚力係数など六つある。この問題を多目的GA で解くとパレート最適解集合が得られ、種々の形状をした解が得られ、設計者はそれぞれの解の特徴を考えながら、最も優れた解を選択できることがわかった。また、もう一つの例として、GA をシミュレーテッドアニーリング(SA : simulated annealing)と組合わせた応用を紹介する。それはタンパク質の三次元構造同定問題に、これらの手法を組合わせたハイブリッド最適化手法を適用したものである。ここでは、タンパク質分子のエネルギーを最小化することで構造を求めており、設計変数は分子内の原子間の二面角である。並列分散SA を基本とし、定期的に同期をとり、複数の解を基にGA の交差を行い、より良好な解を見出す。このように、GA はこれまで解くことが容易でなかった複雑な工学的最適化問題に次々と適用され、目覚ましい成果を挙げている。今後は、より広い分野でGA が応用されるとともに、上のようなハイブリッド手法や、あるいは数理計画法や実験計画法、あるいは応答曲面法など、従来の最適化手法と組合わせた手法なども開発され、最適化問題を与えるとその問題の特徴に応じて、最適な最適化手法を自動的に選ぶシステムも開発されると思われる。
