# 12・6・1 最適化問題とその定式化

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・6 最適化

構造設計問題の簡単な例を挙げ、最適化問題の意味とその定式化6―177図12・21 コート掛け問題図12・22 2部材トラス構造の方法を説明する。図12・21に示すような、ある与えられた荷重を、その荷重点から距離離れた壁に伝達するための、具体的な構造を決める問題(コート掛け問題と呼ぶ)を考える。これを二次元トラス構造を用いて実現することにしよう。このような条件を設けても、対象構造を具体的に決めることは難しい。それは、二次元トラス構造でも与えられた問題を解決する構造形態は、その総部材数とそれらがおかれる三次元空間での位置関係(構造全体の位相関係)の選定の仕方によって、無限に考えられるからである。つまりトラスの構造設計には、次のような二つの段階を経る必要がある。その一つは、構造を構成する総部材数と、それらが占める三次元空間での位置関係がまず決められること(位相決定問題)、そしてその後第二段階として、各部材の長さや断面積などの寸法の決定の問題(寸法決定問題)が解かれなければならない。このようなことはトラス構造に限らず、ラーメンや連続体構造についてもいえることであり、さらにより一般的な最適化の問題においても、このように多段階でその問題を分けて考えねばならない場合が多い。先に述べたコート掛け問題を、いま図12・22に示すように壁とθ,θ傾いて配置されている2部材トラスで考えよう。この問題はγを各部材の比重量とすると、次のように定式化できる。また、σは各部材の許容応力で、一種の部材の材料定数といえるものである。さらに式(12・9)(12・10)は、6―178計算力学それら設計変数の自由度を制約するという意味から、通常制約条件(constraint condition)と呼ばれている。ただし同じ制約条件でも、式(12・10)は単に設計空間における変数の動きうる範囲を決めるものであるのに対し、式(12・9)はある種の物理的な意味を有しているので、式(12・9)を挙動制約(behavioral constraint)条件、一方、式(12・10)を側面制約(side constraint)条件と呼んでいる。以上のように式(12・8)~(12・11)で定式化された構造最適設計問題は、結局数理的には次のように示すことができる。そして式(12・12)の具体的なものとして、先の例での式(12・8)があり、これはその目的とするところが最小重量の構造を求めることにあるので、最小重量設計問題(minimum weightdesign problem)と呼ばれる。これに対し、工作機械や精度を要する機器や構造物では、与えられた体積、重量のもとで、それらの剛性を最大にしたいという要求もある。この種の問題は最大剛性設計問題(maximum stiffness design problem)と呼ばれている。また一般の工学上でわれわれが解決しなければならない問題では、設計対象物の価格を最小にすることが多い。このような問題は、最小コスト設計問題(minimum cost design problem)と呼ばれる。
