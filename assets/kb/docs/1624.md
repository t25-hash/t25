# 12・6・3 進化的Lシステムによる最適形態の創生

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・6 最適化

LS は細胞内での状態変化にとらわれずに、細胞の分裂増殖によって生物の形態発生を記述する概念を中心とした、形態発生システムである。その中心をなすものは、発生過程に登場する細胞タイプのセットΣ、出発点の細胞を表すω、細胞の書き換え規則の三つである。これを用いると、どのような複雑な生物の形態も表現可能であるといわれている。このLS の長所を工学的な形態形成問題へ応用する方法として、与えられた問題に対してΣ,ωを、GA を用いて最適に決め、得られたLS より目的とする最適形態を決定する方法[これを進化的LS(ELS : evolutionary Lindenmayer system)と呼んでいる]が提案されている。ここではこの手法を、図12・26に示す一様曲げモーメントを受けるはりの、最適断面形状の決定問題に適用して説明する。問題は=27.4kN・mを受けているはりが、発生する応力を材料の許容応力以下にとどめ、重量を最小とする断面形状を求めるものである。初期断面を図12・26のように正方形とし、問題の対称性よりその1/4領域を10×10要素に分割して、各6―179制約条件の性質制約条件なしの最適化問題制約条件付きの最適化問題一次関数一般の関数は一次関数線形最小二乗法二次計画法[ウォルフ(Wolfe)ほか]変換法(ペナルティ法)GRG傾斜投影法実行可能法今野・山下の方法Murtagh-Sargentの方法ほかKowalik-Osborneの方法Fletcherの方法、ポーウェルの方法Broydenの方法、バーネスの方法変換法(ペナルティ法)切除平面法MAP部分傾斜法GRG実行可能法傾斜投影法Murtagh-Sargentの方法大型の問題:共役傾斜法可変計量法Fletcher-Reevesの方法大型でない問題:Davidon-Fletcher-ポーウェルの方法可変計量法図12・26 解析モデル要素の状態を次の3種の材料から選択することにより、はり断面形状が決まる。したがって、本問題は次のように示される。分割された各要素を細胞とみなし、出発点に設定された細胞から増殖を繰返して、はりの断面形状を創生する。ここでの各細胞は次のものである。・細胞A:材料A(増殖可能細胞)・細胞B:材料B(増殖可能細胞)・細胞a:材料A(不変細胞)・細胞b:材料B(不変細胞)・細胞c:空域6―180計算力学計算では、出発点細胞は断面の中心位置とし、増殖した細胞が10×10要素領域を超えた場合その機能を停止させ、さらに設計領域内の増殖がすべて終了したとき、成長を終了するとした。また、この問題に対しては、図12・27に示すようなテーブルLS を用いる。これは、細胞は初め規則に従って増殖を繰返すが、ある時点(増殖回路が回を超えるとき)で切換えスイッチが働いて、規則に従って活動するとするものである。この染色体の長さは計32ビットで済み、100要素すべてを直接変数とする単純GA の方法では200ビット必要であることから、この方法の解探索の有効性がわかる。図12・28はその結果得られたLS を示す。また、これより得られたはり断面形態を、単純GA で求めた結果と比較して図12・29に示す。これらよりELS で求められた図12・29(a)の断面は、軸から遠い部分に多くの材料を配置することによって、少ない材料数で断面二次モーメントを大きくする、いわゆるI字型断面となっており、むだの少ない妥当なものといえる。一方、GA のみの手法で得られた図12・29(b)の断面は、規則性も見られず、むだの多いものである。計算力学で取扱う解析では、入力を与え、そのときの出力、すなわち応答を求める順問題(direct problem)を解く、順解析(direct analysis)が行われる。しかし、応答を求める順解析そのものが目的ではないことも多い。すなわち、単に応答を求める順解析を行う代わりに、解析結果を実験結果と比較することにより、材料等の特性を同定したり、境界条件を推定したり、物体内に存在するき裂や欠陥を推定したり、あるいは満足すべき仕様を満足するような設計条件を求めることがある。このような、入力を与えて応答を求める順問題とは逆に応答からその基になっている入力を推定するものは、逆問題(inverse problem)と呼ばれており、それを扱うものは逆解析(inverse analysis)と呼ばれる。ここでは、計算力学における逆問題とその解析法について述べる。
