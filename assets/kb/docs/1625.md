# 12・7・1 逆問題の定義と分類

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・7 逆問題

上記のように、順問題が原因から結果を導く、あるいは入力から出力を求めるのに対して、逆問題は結果から原因を求める、あるいは出力より入力を求めようとするものである。計算力学で扱われる逆問題では、支配微分方程式を解く。このとき、5・3・5項でも述べられているように、(1)対象としている領域Ωとその境界Γの位置と形状、(2)物理量φに関する場の支配方程式、(3)φあるいはその微係数に関する境界条件(必要があれば初期条件も含む)、(4)支配方程式に含まれ領域Ωで定義される負荷、(5)支配方程式に含まれる材料特性の分布が必要である(図12・30)。もし、これらの入力のうちのいずれかが欠落していれば、通常の順解析を行うことはできない。逆問題は順問題以外のものと考えると、図12・31のように、順解析に必要な情報のいずれかが未知であるときに、これらを同定しようとするものすべてが逆問題であり、5・3・5項で述べられているように、何が未知であるかに対応して、以下のような逆問題がある。(1) 領域Ωの形状、あるいは物体内にある未知境界を同定する問題(境界/領域逆問題)(例:欠陥の同定、自由表面の推定)(2) φの場を支配している微分方程式を求める問題(支配方程式逆問題)(例:材料の構成則の推定、現象を表すモデルの構築)(3) 境界Γの一部あるいは全部における境界値、あるいは領域Ω内の初期値を求める問題(境界値/初期値逆問題)(例:接触部の面圧分布の推定)(4) 領域Ωの内部で作用している負荷を求める問題(負荷逆問題)(例:領域内の熱負荷や双極子の推定)(a) 順問題(1) 領域Ωと境界Γ(2) 場の支配方程式(3) 境界条件(初期条件)応答(4) 負荷(Γ上の未知境界値)(5) 材料特性(Ω内の物理量φ)(b) 順解析図12・30 順問題と順解析(5) 領域Ωの内部における材料特性を推定する問題(材料特性逆問題)(例:ヤング率分布の推定)このように種々の逆問題があり、発想を転換することにより多くの逆問題が設定できる。
