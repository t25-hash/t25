# 12・7・1 逆問題の定義と分類

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・7 逆問題

上記のように、順問題が原因から結果を導く、あるいは入力から出力を求めるのに対して、逆問題は結果から原因を求める、あるいは出力より入力を求めようとするものである。計算力学で扱われる逆問題では、支配微分方程式を解く。このとき、5・3・5項でも述べられているように、(1)対象としている領域Ωとその境界Γの位置と形状、(2)物理量φに関する場の支配方程式、(3)φあるいはその微係数に関する境界条件(必要があれば初期条件も含む)、(4)支配方程式に含まれ領域Ωで定義される負荷、(5)支配方程式に含まれる材料特性の分布が必要である(図12・30)。もし、これらの入力のうちのいずれかが欠落していれば、通常の順解析を行うことはできない。逆問題は順問題以外のものと考えると、図12・31のように、順解析に必要な情報のいずれかが未知であるときに、これらを同定しようとするものすべてが逆問題であり、5・3・5項で述べられているように、何が未知であるかに対応して、以下のような逆問題がある。
