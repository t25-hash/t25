# 12・7・4 逆解析手法

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・7 逆問題

以下では代表的逆解析手法について述べる。まず、測定値を用いて未知パラメータに関する関係式を構成し、逆問題解を求める一般的な方法の一つに選択法(selectionmethod)あるいは残差最小化法(least residual method)があり、これらの解は準解(quasisolution)と呼ばれる。これらの方法では、同定すべき対象をで表したとき、まずの候補を仮定して対応する応答욑を計算する。この計算値욑と、測定などにより得られる値욖との間で、残差平方和=욑-욖dΓ(12・17)を評価する。ここに、積分は観測値が得られている領域、あるいは境界Γ욇で評価される。が最も小さくなるものとしてが推定される。の最小化には、最適化手法を適用することができる。カルマンフィルタ(Kalman filter)、射影フィルタ(projectionfilter)などのフィルタも適用されている。適切化手法として有名なものに、チホノフの適切化法(Tikhonov’s regularization method)がある。この方法は、式(12・17)のに平滑化項を付けた=욑-욖dΓ+αΛ(12・18)(a) 逆問題(1) 領域Ωと境界Γの一部補助情報(2) 場の支配方程式┆좺(3) 境界条件(初期条件)(A) 未知境界(4) 負荷↑(5) 材料特性応答(Γ上の過剰な境界値)(Ω内の物理量φ)(b) 逆解析の例(境界/領域逆問題)図12・31 逆問題と逆解析6―181を考え、を最小化するを解とする。ここに、αは適切化パラメータ(regularization parameter)である。Λは安定化汎関数であり、Λとしてはのノルムなどが用いられる。境界要素法や有限要素法を適用すると、推定すべき未知量を観測量と関係付ける線形連立方程式が得られることが多い。これを逆に未知量について解くことにより、未知量が推定できる。この計算には一般逆行列(generalized inverse matrix)が利用できる。この方法では、未知数の個数より有効な式の個数のほうが多い場合には、式の残差の二乗和の最小化を基準とした最小二乗解を用いる。有効な式の個数が少ない場合には、解のノルムの最小化に基づいたノルム最小解を用いる。逆問題で現れるマトリックスは、非正則であったり非正則に近かったりするため、解が数値的に不安定になることが多い。このような場合に対して有効な方法に、マトリックスの特異値分解(singu-lar value decomposition)がある。観測値に含まれる誤差が推定結果に及ぼす影響の程度は、条件数(condition number)により与えられ、その値は特異値分解により得られた最大特異値と最小特異値の比に等しい。逆問題では、小さな特異値が含まれているので、小さな特異値を零とおき、ランクを有効ランク(effectiverank)まで落とすランク低下法(rank reduction method)が用いられる。このとき、条件数は小さくなり、推定結果は測定値に含まれる誤差に不敏感となる。問題によっては、特異値分解により得られる解のモードは、数理解析結果と対応づけられる。個々の逆問題に対しては、その特性に対応した逆解析手法が構築されている。適切化手法の中には、チホノフの適切化パラメータαや、マトリックスの有効ランクのような、あらかじめ与えられていない適切化パラメータが含まれているものがある。適切化パラメータの推定法には、食違い量原理(discrepancy principle)、L曲線法(Lcurve method)、情報量基準AIC(Akaike information crite-rion)最小化推定法、最適条件数法、参照問題法などがある。
