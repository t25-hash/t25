# 12・7・4 逆解析手法

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・7 逆問題

以下では代表的逆解析手法について述べる。まず、測定値を用いて未知パラメータに関する関係式を構成し、逆問題解を求める一般的な方法の一つに選択法(selectionmethod)あるいは残差最小化法(least residual method)があり、これらの解は準解(quasisolution)と呼ばれる。ここに、積分は観測値が得られている領域、あるいは境界Γで評価される。の最小化には、最適化手法を適用することができる。カルマンフィルタ(Kalman filter)、射影フィルタ(projectionfilter)などのフィルタも適用されている。適切化手法として有名なものに、チホノフの適切化法(Tikhonov’s regularization method)がある。ここに、αは適切化パラメータ(regularization parameter)である。境界要素法や有限要素法を適用すると、推定すべき未知量を観測量と関係付ける線形連立方程式が得られることが多い。これを逆に未知量について解くことにより、未知量が推定できる。この計算には一般逆行列(generalized inverse matrix)が利用できる。この方法では、未知数の個数より有効な式の個数のほうが多い場合には、式の残差の二乗和の最小化を基準とした最小二乗解を用いる。有効な式の個数が少ない場合には、解のノルムの最小化に基づいたノルム最小解を用いる。逆問題で現れるマトリックスは、非正則であったり非正則に近かったりするため、解が数値的に不安定になることが多い。このような場合に対して有効な方法に、マトリックスの特異値分解(singu-lar value decomposition)がある。観測値に含まれる誤差が推定結果に及ぼす影響の程度は、条件数(condition number)により与えられ、その値は特異値分解により得られた最大特異値と最小特異値の比に等しい。逆問題では、小さな特異値が含まれているので、小さな特異値を零とおき、ランクを有効ランク(effectiverank)まで落とすランク低下法(rank reduction method)が用いられる。このとき、条件数は小さくなり、推定結果は測定値に含まれる誤差に不敏感となる。問題によっては、特異値分解により得られる解のモードは、数理解析結果と対応づけられる。個々の逆問題に対しては、その特性に対応した逆解析手法が構築されている。適切化手法の中には、チホノフの適切化パラメータαや、マトリックスの有効ランクのような、あらかじめ与えられていない適切化パラメータが含まれているものがある。適切化パラメータの推定法には、食違い量原理(discrepancy principle)、L曲線法(Lcurve method)、情報量基準AIC(Akaike information crite-rion)最小化推定法、最適条件数法、参照問題法などがある。
