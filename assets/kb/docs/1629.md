# 12・7・5 逆解析の適用例

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・7 逆問題

境界要素法による逆解析については、5・3・5項で紹介されているので、以下では境界要素法以外を用いたものを中心に、逆解析の適用例を紹介する。欠陥やき裂の非破壊計測は、何らかの物理的応答を基に、き裂や欠陥を同定するものなので、境界/領域逆問題の一つである。き裂の同定法には、物体に通電したときの応答、すなわち電気ポテンシャル分布を用いる電気ポテンシャル法や、インピーダンスプレチスモグラフィーなどがある。電気ポテンシャルCT(computedtomography)法では、能動的に通電したときに物体表面に現れる電気ポテンシャル分布、あるいは物体表面に貼付したピエゾ膜上に受動的に現れる電気ポテンシャル分布の計測結果を、境界要素解析結果や有限要素解析結果と比較することにより、き裂の同定が行われている。渦電流探傷は、欠陥やき裂の定性的検出によく用いられている。これを欠陥やき裂の定量的同定に結びつけるため、電磁場の数値解析結果が用いられている。欠陥を有する物体に熱負荷を与えたときには、その熱的応答に変化が出る。そこで、表面温度分布をサーモグラフィーなどにより求め、これを計算結果と比較することにより、欠陥を定量的に同定する試みがなされており、サーマルトモグラフィー、サーマルウェーブCT などの名称がつけられている。欠陥等が存在すると弾性波の散乱が起こることを利用して、欠陥を同定することが行われており、逆散乱問題、超音波探傷として知られている。定量的同定のため、波動場がヘルムホルツ方程式を用いて解析される。階層型ニューラルネットワークの適用も行われている。境界値逆問題では、不適切性の数理的構造が推定結果に反映される。ガルバニック腐食逆問題では、溶液内部の電気ポテンシ6―182計算力学ャルの測定値を用いて、溶液を通じ電気的に接触した異種金属の表面における電気ポテンシャルと電流密度が推定されている。二つの物体が接触している境界上の接触力と変位を、その状態を乱すことなく、接触部の状態を他の部分で測定される変位と、表面力を用いて推定する方法が考えられている。この方法では、有限要素関係式に測定値を代入することにより得られる関係式が、境界上の接触力と変位に対して解かれている。負荷逆問題に対して、領域内の集中熱負荷や双極子などのソースを、境界観察結果から推定する方法が提案されている。他の負荷逆問題として、多くの点におけるアコースティックエミッションの信号を基に、破壊の発生点とその位置における原波形を推定する原波形解析(source characterization)がある。衝撃力の推定という負荷逆問題の取扱いもなされている。残留応力および残留ひずみを推定する問題も、領域内の負荷に関する一種の逆問題と解釈することができる。これらを、切断時に計測されるひずみ等より求める方法が提案されている。熱弾性応力測定法では、主応力和の分布を求めることが可能であるが、応力の各成分を直接求めることはできない。そこで、逆解析を用いて、物体に作用している応力の成分分離が行われている。また適切化のため、特異値分解を用いたランク低下法などが適用されている。材料の構成関係の同定にも逆解析が適用されており、粘弾性材料の緩和スペクトルなどの同定、粘塑性材料の構成式のパラメータ同定、動的構成式の同定、複合材料の弾性定数同定、セラミックスの破壊進行域の特性評価がなされている。現象に関する支配微分方程式を推定する問題に対しても、検討がなされている。観測値を基に、微係数を数値的に評価し、それを微分方程式に当てはめることによって支配方程式の係数が推定されており、微係数の数値的評価には、モリフィケーション法が適用されている。支配微分方程式の階数の推定も行われている。以上述べてきたことからわかるように、多量の、また種類の異なる観測結果に対して逆解析を適用することにより、観測結果そのままでは困難な定量的推定や同定、評価が可能となる。今後、逆問題的取扱いが拡大するものと期待される。メッシュフリー法とは、図12・32に示すように、いわゆる要素や格子を利用せずに偏微分方程式を解く手法の総称であり、メッシュレス法、エレメントフリー法などとも呼ばれる。偏微分方程式を解く際に用いる変位の、近似関数の種類や解法の種類によってエレメントフリーガラーキン法(elementfree Galerkin method)、リプロデューシングカーネルパーティクル法(reproducing kernel par-ticle method)、有限被覆法(finite cover method)など、さまざまなメッシュフリー法が提案されているが、それらについては文献(97)に詳しいのでそれに譲るものとして、本節ではエレメントフリーガラーキン法(以下、EFG)を中心として、その概要を示す。式(12・24)(12・25)を式(12・23)に代入すれば図12・32 メッシュフリー法の概念図
