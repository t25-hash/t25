# 12・8・2 形状関数の算出

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・32 )。これが移動最小二乗法といわれる、最も多くのメッシ

EFG とFEM はいずれも、ガラーキン法に基づく解法であるから、要素の有無に関して違いはあるものの、いずれも変位を自由度として、領域全体で積分を行う点に関しては同じである。したがって、EFG もその表記をFEM に準じておけば、両者の共通点、異なる点がより明確になり、実際にコンピュータプログラムを作成する際に、FEM の技術が流用できる。ここでは、EFG において形状関数を作成する手順について解説する。なお、FEM の場合は、節点を近似関数が必ず通るため、形状関数のことを内挿関数あるいは補間関数とも呼ぶが、EFG の場合は必ずしも節点上を関数が通らず、内挿と呼ぶのは不適切であるため、ここでは単に形状関数と呼ぶ。代表的な重み関数として四次のスプライン関数によるものを示す。これにより、近似関数の係数ベクトルを決定することができる。φがFEM での形状関数に相当する。このように、EFG においても上記の手順を経て、影響領域内の節点に対する形状関数を定めることができる。ここで、形状関数の観点から、FEM との違いを述べる。(1) FEM の場合は、例えばアイソパラメトリック要素を用いている限り、同節点数の要素の間では形状関数は変化しない。一方、移動最小二乗法より得られる形状関数は、FEM のように陽な形で表現することはできず、評価点ごとに違った形で求められる。(2) EFG では、領域内に含まれる評価点の数だけを計算しなければならない。すなわち、領域積分の際にはすべての積分点において、また節点における値を評価する際にはその節点位置において、を計算する必要がある。(3) は、領域内の節点の数が適切でないと、評価できない場合がある。すなわち領域内の総節点数は、基底関数(あるいは係数ベクトル)の個数以上でなければならない。=の場合は、節点位置における近似関数の値が節点値と一致するように係数が決定される。
