# 12・8・3 剛性方程式の導出

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・32 ).これが移動最小二乗法といわれる,最も多くのメッシ

ここでは,線形弾性体におけるEFG の剛性方程式の導出について述べる.基本的な構造はFEM と同様であるので,ここでは異なる部分について示す.(1) FEM の場合,要素内の変位は,要素を形成する節点の値から内挿され,節点位置での値は節点値に一致するのに対して,EFG の場合は,移動最小二乗法を用いて近似されるため,節点位置における近似関数の値は,必ずしも節点値に一致しない.(2) 領域積分は,FEM では分割された要素単位で行われ,要素内の積分は要素形状から定められる積分点を用いて行われる.EFG の場合は,要素というものが存在しないため,積分点は,一般的には解析領域とは別に定義した積分のためのセルを利用して行われる.a. 境界条件処理EFG における拘束条件処理については,ラグランジュ未定乗数法(Lagrange multiplier method)をはじめとして種々提案されているが,ここでは,最も簡易で,広く用いられているペナルティ法(penalty method)について紹介する.EFG では,12・8・2項で述べたように,離散化された節点値を拘束するだけでは,節点位置における近似関数の値を拘束することはできない.したがって,離散化する以前のエネルギー原理の時点で,その拘束条件を導入する必要がある.そのため,トータルポテンシャルエネルギーは,変位拘束条件を付加して式(12・30)のように表すことができる.π≡2εσ-dΩ-dΓ2α-dΓ(12・30)+ここで,εはひずみ,σは応力,は体積力,は表面力,Ωは解析領域,Γ욧は表面力が働く境界をそれぞれ表しており,αは正の極めて大きいペナルティ数である.ひずみ変位マトリックスとして,応力ひずみマトリックスとして,表面力および体積力を離散化して得られる外力を,まとめてとすれば,最終的には,次の離散化平衡方程式が求められる.6―183+′=+′(12・31)ここで,,は節点番号として=dΩ(12・32)′=-αφφdΓ(12・33)′=-αφdΓ(12・34)=(12・35)であり,,はそれぞれ節点が,方向に拘束されていれば1,そうでなければ0の値をとる.b. 領域積分EFG はガラーキン法に基づいているため,最終的には,全体領域積分を実施しなければならない.FEM では,この領域積分は要素単位で行われて,合算される.一方,EFG では要素そのものが存在しないために,領域全体をFEM のように効率良く積分することはできない.そのためベリチコ(Belytschko)らは,解析対象の領域と積分のための領域を区別し,図12・32に示すような,バックグラウンドセル(background cell)と呼ばれる積分のためのセルを提案した(図12・32).領域積分の際には,それぞれのセルでガウス積分を行う.境界近傍を精度良く求めるには,セルを細かくするか,あるいは一つのセルに対して積分点を多くする.この場合,積分点が領域に含まれているか否かで,数値積分における重み値を零にすればよい.数値積分の際には,積分点を中心として影響半径ρの円を描き,影響領域となる円の内側にある節点に対し,移動最小二乗法に基づく形状関数を求める.このとき,影響領域が境界を越えた場合,境界の外に節点は配置されないので,影響領域としては考慮されない.c. その他EFG をはじめとして多くのメッシュフリー法では,要素というものが存在しないために,比較的容易にき裂のような不連続性を解析モデルの中に考慮することができる.例えばEFG では,き裂をまたいで影響領域をとらないようにすることで不連続場を表現することができ,さらには式(12・22)の基底関数として,特異関数を用いることで,より精度良くき裂まわりの解析を行うことが可能である.そのほかにも,FEM と比してEFG(メッシュフリー法)の利点は多々あり,表12・2にそれらを不利な点と併せて示す.
