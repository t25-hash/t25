# 12・9・2 ウェーブレット

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・9 ウェーブレット

これに加え、次の条件を満たすものを正規直交ウェーブレットという。なお、正規直交ウェーブレットのほかに、半直交ウェーブレットや双直交ウェーブレットなどが提案されている。以下では紙面の制約上、正規直交ウェーブレットを対象に説明する。ハールウェーブレットと同様に、コンパクトサポートをもつ正規直交ウェーブレットとして、ドビッシー(Daubechies)のウェーブレットがある。図12・36は、二次のゼロモーメント性(12・[図12・36(b)]と、それに対応するスケーリング関数φ[図12・36(a)]の分布を示したものである。ドビッシーのウェーブレットは閉じた関数形で定義することができず、図12・36に示すように、複雑で非対称な関数形状を有する。
