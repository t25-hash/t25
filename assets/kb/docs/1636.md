# 12・9・3 ウェーブレットの有する特性

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 9・3 ・a項で説明)を有するドビッシーのウェーブレットψ

a. ゼロモーメント性関数が次の条件を満たすとき、関数は次のゼロモーメント性をもつという。例えばハールウェーブレットは、=1に対して式(12・42)を満たすので一次のゼロモーメント性をもっており、二次のドビッシーのウェーブレットは二次のゼロモーメント性をもっている。b. フィルタ特性一般に、ウェーブレットは低周波域を適切にカットするフィルタ特性を有している。これらの図より、スケーリング関数はローパスフィルタの、またウェーブレットはバンドパスフィルタの役割をもっていることがわかる。ドビッシーのウェーブレットに代表されるように、ウェーブレットは、実領域と周波数域の両方に局在するという特性をもつ。そのため信号や画像などの処理に多用されている。
