# 13・1・2 圧縮試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

圧縮試験（compression test ）は延性に富んだ金属材料にはあま
り適さないが，鋳鉄，木材，石材，コンクリートのようなもろい材
料の強度評価に適した試験法である．また，焼入れした玉軸受の強
度も圧縮試験法で求める．圧縮試験に影響を及ぼす因子としては，
試験片の端面と耐圧盤との摩擦がある．この摩擦が大きいと試験片
がたる型に変形するので，これを防ぐために，端面の仕上げを良く
し，接触面を清浄にし，かつ，テフロンなどの潤滑材を用いること
が望ましい．また，試験片が座屈を起こしやすいので，荷重が中心
軸に正しく作用するように注意することや補助装置を利用すること
が好ましい．圧縮試験に適した圧縮専用試験機がある．
