# 13・1・4 ねじり試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

ねじり試験（torsion test ）は引張試験法ほどは広く普及してい
ないが特色のある試験法である．ねじり試験は二つの目的で行われ
ている．一つは回転軸やドリルのようにねじり応力（せん断応力）
が作用する状態で使われる機械部品の強度を調べるためであり，も
う一つは鍛造材のように強加工して使う材料の機械的性質を調べる
ためである．また，金属材料の機械的性質は引張試験によって通常
求められるが，ひずみ量が50 %を超えるような大変形を問題にす
るような場合には，引張試験ではくびれが生じるため，正確な応
力-ひずみ関係を求めることはできない．しかし，ねじり試験を行
えばくびれは生じず，容易に大変形を扱うことが可能になるため，
金属材料の加工性に関する検討や，塑性変形に関する理論的な研究
の分野ではねじり試験が広く行われている．
ねじり試験機は駆動部とねじりモーメント（トルク）検出部によ
りなっており，試験片は両者の間に取付けられる．ねじり試験の結
果はねじりモーメント-ねじれ角曲線として描かれ，これが引張試
験における荷重-伸び曲線に相当する．ねじり試験の場合，試験片
に生じる応力の算定を容易にするため，円形の断面を有する試験片
が広く用いられている．中実の試験片をねじり試験で用いると，試
験片の半径に依存した応力，ひずみの分布を生じる．例えば，弾性
変形領域における応力は中心部において零で，表面において最大と
なり，その間では半径に比例して直線的に増加するという分布とな
る．薄肉の中空円筒型の試験片を用いれば，応力とひずみの半径方
向への分布が小さくなり，正確なせん断応力-せん断ひずみの曲線

部での応力集中や，荷重線の鉛直性からのわずかなずれにより試験
片が破壊し，正確な強度評価が困難となる．このような場合，三点
曲げ試験（three point bending test ）あるいは四点曲げ試験（four
