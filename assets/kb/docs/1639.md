# 13・1・5 その他の試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

薄板の深絞り性を判定するために行われる試験法の一つとして，
エリクセン試験（Erichsen test ) がある．これはエリクセン試験
機 を用い，直径または幅90mmの円形，正方形または帯状試験
片を図13･8のしわ押さえとダイスの間に置き，ポンチを押込む．
試験片の表裏面に貫通したき裂が生じるまでにポンチの進入した深
さ( mm)の数値をもってエリクセン値を表す．試験中のしわ押さえ
とダイスの間隔には試料の厚さのほかに0.05mmのすきまを設け
て行うのがエリクセン試験A方法であり，別にすきまを設けずに
9800Nの荷重を加え，押さえつけた状態で行うのがエリクセン試
験B方法 である．
e .ナノインデンテーション試験 電子材料の分野における極
薄膜（＜100nm）デバイスの硬度，弾性率測定に用いられる試験
法である．硬度試験時の最大押込み深さは，膜厚の10分の1に制
限され，数十nmの圧痕を観察・解析する．具体的には，走査型プ
ローブ顕微鏡（SPM:scanning probe microscope）と組合わせて
圧痕の形状解析を行い，さらに，押込みにより得られる荷重-変位
曲線から硬度と弾性率を算出する ．
