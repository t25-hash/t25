# 12・9・6 ウェーブレット解析の適用

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 9・3 ・a項で説明)を有するドビッシーのウェーブレットψ

a. ノイズの除去信号など一連の時刻歴データが急変動部分を含む場合、そのフーリエスペクトルは、当然のことながら高周波成分を多く含む。このようなデータに高周波ノイズが混入した場合、そのフーリエ変換に基づくノイズの検出は容易でない。一方、ウェーブレットは実領域(時間域)と周波数域の両者に対して局所性を有するので、ウェーブレット展開係数の分布より、データ本来の急変動成分とノイズとの判別が比較的容易に行える。データをウェーブレット展開した後、ノイズに対応するウェーブレット係数のみ切捨て、データを再構成すれば、ノイズを適切に除去することが可能となる。b. データの圧縮
