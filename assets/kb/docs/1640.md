# 13・1・6 材料試験機

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・1 静的強度試験

a.万能試験機 引張試験のほかに圧縮，曲げなどの試験
にも利用できるものを万能試験機という．一般に引張試験は万能試
験機を用いて行われる．
目盛ざおと送りおもりとをもって荷重を測定する試験機は概して
精度が良好である．単てこ型のものは特に精度が良いため荷重検定
用標準試験機として用いられる．しかし，この種のものは取扱いが
不便であることから多用する向きには適さない．
油圧駆動式の万能試験機では，荷重測定機構として，アムスラー
式振り子力計，すなわち振り子の角変位によって荷重を測る装置を
もつものと，レバーにより縮小された荷重を電気的にとらえる装置
を備えたもの（おもりを移動させゼロ点法で荷重を測る）とが主流
である．測定精度は後者が優れている．これらは荷重-伸び線図を
書かせるにも，降伏点や引張荷重の測定にも好都合であり，万能試
験機として多用されている．
ほかに，直流サーボモータを用いて駆動する試験機（通称インス
トロン型という）も普及している．この試験機は，荷重の変動を忠
実に記録できること，一定速度で変形を与えることが可能であるこ
と，などの長所をもっている．荷重-変位曲線が正確に記録できる
ので，上降伏点とかセレーション変形による荷重の急激な変動も明
りょうに求められる．
b.圧縮試験機 セメント，コンクリート，鋼球，タイヤなど
のようにもっぱら圧縮試験のみ行われる材料に対しては，専用の圧
縮試験機が用いられることが多い．専用の試験機は荷重用のシリン
ダが床上に配置された構造であるため，試験の実施が容易でかつ安
価である．
c .ねじり試験機 ねじり試験には専用の試験機が用いられる．
試験片の一端に歯車装置などを用いてねじりモーメントを与え，他
端に連結した振り子の変位角あるいは抵抗線ひずみ計を用いたロー
ドセル，荷重を電気的にとらえる装置などによってトルクの測定を
行う．高速ねじり変形用に考案されたねじり試験機もある ．
d.精度検査 引張試験機や万能試験機ではひょう量とその
5分の1に相当する荷重間を使用範囲とするので精度検査もこの間
で行われる．ひょう量を変更しうるものについてはひょう量ごとに
行う．使用範囲内における荷重の許容誤差は±1.0 %以内でなけれ
ばならない．これらの精度検査には材料試験機用荷重検定器 を
用いる．
一般に，小荷重の検定にはたわみ型検定器（ループ）を，大荷重
の検定には容積検定器（スタンダータイジングボックス）を用い
る．

と一軸引張りで求めた応力-ひずみ曲線とを比較する場合には，ミ
ーゼス型やトレスカ型の相当応力や相当ひずみを用いれば，両者の
間にはよい一致が見られる．
