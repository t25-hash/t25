# 13・10・1 破面観察，計測技術

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・10 フラクトグラフィー（破面解析）

ィンプル（dimple）と呼ばれる多数のくぼみが作られ，くぼみの側面にはすべり模様が，また底には第二相粒子がしばしば認められc .すべり面分離 結晶材料が塑性変形する際には最もちゅう面に露出するようになる．
このように，すべりによって変形あるい作られる代表的破面様相は，疲労き裂が伝ぱする際に応力サイクル金属の点状破壊もすべり面分離の結果である．
フラクトグラフィーは，破面観察の倍率に応じて3種類に分類で密な原子面ですべりが生じ，材料の内部にあった原子面が新しく表は破壊が生じる機構をすべり面分離という．
