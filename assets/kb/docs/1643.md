はより詳細なものが用いられる．クラスを表す箱とそれらを結ぶ線

.ユースケース図（use case diagram) システム全体の
ふるまいをモデル化するために用いられる．個々のふるまいはユー
スケースとして述語（～を行うという形）で表現される．システム
外の関連するシステムあるいはオブジェクトは，アクターとして定

.シーケンス図（sequence diagram) オブジェクトの動
的関係，すなわちメッセージパッシングによる相互作用を，時間軸
に沿って表したもので，分析および設計の段階で用いられる（図
12･41)．
.状態図（statechart diagram) オブジェクトに注目し
て，その状態の遷移とその条件を示したもので，イベントに対する
反応を記述するのに適している（図12･42)．
b.デザインパターン（design pattern) オブジェクト指向と
いう方法論では，プログラムの再利用性を高めることができるが，
その上のレベルに対しては効果がない．しかし，類似の問題に対し
て，似たような解決法が与えられるのは容易に予想できることであ
り，これを問題解決のパターンとしてカタログ化したのがデザイン
パターン である．デザインパターンの概念自体は非常に広いも
のであるが，実際にはオブジェクト指向設計の枠組の中で，カタロ
グ化されている．ガンマら はデザインパターン利用の目的の一
つとして，システムの変更に対して，再利用性の高いプログラミン
グをすることを挙げている．したがって，本節の最初に挙げた計算
力学分野のソフトウェアに対する要請と合致しており，デザインパ
ターンの利用を検討すべきである．
c.コンポーネント（component) 再利用性を高めるもう一
つの工夫が，コンポーネントの利用である．コンポーネントとはプ
ログラムの部品という意味であるが，インタフェースを統一したオ
ブジェクトを利用することにより，コンポーネント間の相互依存性
が少なくなり，オブジェクト指向アプローチの重要な概念となって
いる．JavaBeans では，入出力インタフェースを統一したオブ
ジェクト（bean）を用いることにより，GUI （graphical user
interface）を用いたアプリケーション開発が可能となっている．
12・11仮想現実感
仮想現実感とは，計算機によって人工的な世界を生成しようとす
るシミュレーション技術である．本節では仮想現実感の概念，要素
技術，応用展開等について概説する．
12・11・1仮想現実感の概念
a.構成要素 仮想現実感という言葉は，virtual reality（バ
ーチャルリアリティ）の訳語であるが，virtualという単語はもと
もと「（表面的または名目上はそうでないが）実質上の，本質の」
といった意味をもつ．つまり，実際には存在しないが，本質におい

想現実感技術ということになる．MIT（マサチューセッツ工科大
学）のゼルツァー（Zeltzer）は，仮想現実感が満たすべき要件と
してプレゼンス（presence），インタラクション（interaction），

b.歴史 仮想現実感技術の起源は，コンピュータグ
ラフィックス，テレロボティクス，アート等いろいろな分野で見る
ことができる．例えば，1960年代にユタ大学のサザランド（Suth-
erland）によって，HMD（head mounted display）の原型が試作
されている．このシステムは，当時の計算機の能力的な限界から線
画による映像ではあったが，三次元映像をいろいろな方向から眺
め，映像とインタラクションを行うという，仮想現実感技術の要素
は含まれていた．また1970年代には，ノースカロライナ大学のブ
ルックス（Brooks）らによって，力覚フィードバックを用いた情
報提示の研究が始められている．GROPEと呼ばれた研究プロジェ
クトでは，分子間の反力をマスタアームを使って提示するという方
法を実現した．またクルーガー（Krueger）により，META-
PLAY等の一連のインタラクティブアートの作品が作られたのも，
このころである．
今日的な意味での仮想現実感技術の展開は，1980年代に見るこ
とができる．アメリカ航空宇宙局（NASA:National Aeronau-
tics and Space Administration），エイムズ（Ames）研究所のフ
ィッシャー（Fisher）らによる仮想環境ワークステーション（vir-
tual interface environment workstation）の研究や，アメリカ空
軍（USAF:United States Air Force）にいたファーネス（Fur-
ness）らによるスーパーコックピット（super cockpit）の研究に
より，仮想現実感技術は大きな注目を集めるようになった．また，
国内における仮想現実感の研究は1990年代に急速な広がりを見せ，
文部省の重点領域研究や日本バーチャルリアリティ学会の設立等に
より研究の基盤が整備され，今日に至っている．
