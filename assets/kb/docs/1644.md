# 13・10・4 数値的破面解析技術

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・10 フラクトグラフィー（破面解析）

破面の特徴的パターンに基づく破面解析は多くの場合かなりの熟
練を要し，ストライエーションが認められる場合を除いて定量的評
価はほとんどの場合困難である．これに対し，三次元SEM，
LSM，SPM等では破面の数値的形状データが得られ，これを解析
することにより定量的評価が可能である．
a.破面粗さ法 破面粗さを計測して粗さパラメータから作用
応力を推定するものである．粗さパラメータ を用いた評価例を
図13･31 に示す．なお， は次式で表される．
＝∑［(粗さレンジペア)×(頻度)］/(計測距離)
b.FRASTA法 FRASTA（fracture surface topography
analysis ）法は，一対の破面の計測結果を用いて破面形状をコンピ
ュータ内で再構築し，破壊過程を再現する手法である ．図13･32
に模式図を示すが，破面を重ね合わせたあと徐々に引き離すことに
よって各段階での横断面図（XSP:cross -sectional plot ）および
耗状態の監視などに利用されているほか，地熱・石油井における水
透視図（FAPP:fractured- area-projection plot ）が得られる．
圧破砕のモニタリングや人工き裂の評価などにも用いられている．
