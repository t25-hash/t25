# 12・11・2 要素技術

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・11 仮想現実感

ここでは、仮想現実感で使われるいくつかの代表的な要素技術を取上げる。a. HMDHMD(head mounted display)は、頭部に搭載するゴーグル型のディスプレイ装置であり、利用者は眼の前に提示される映像をレンズ系を介して見る。この際、左右の眼に対して視差のある映像を提示することで、利用者は立体映像を見ることができる。通常は視線計測用の位置センサと併用され、視点位置に応じた映像を提示することで、利用者は三次元の立体映像をいろいろな方向から見回すことが可能になる。図12・44 イリノイ大学で開発されたCAVEb. IPTHMD に対し、もう一つの代表的な視覚ディスプレイ装置として、没入型立体映像装置(IPT : immersive projectiontechnology)が挙げられる。これは、利用者を取り囲むように配置された大画面スクリーンに、プロジェクタによる立体映像を投影することで、没入感の高い仮想世界を提示する方法である。代表的なIPT としては、イリノイ大学で開発された4面スクリーン構成のCAVEが挙げられる(図12・44)。また、スクリーン構成が改良されたものとして6面構成の完全没入型ディスプレイや、2~3面のスクリーンを用いた簡易没入型ディスプレイ、あるいは円筒や球状にスクリーンを配置した曲面型ディスプレイ等も開発されている。c. 三次元音響ディスプレイ仮想現実感で使用される音響ディスプレイでは、三次元音像定位が重要な技術要素となる。代表的な三次元音像定位技術としては、頭部伝達関数(HRTF: headrelated transfer function)を用いる方法が挙げられる。これは、自由空間において、各音源方向から聴取者の鼓膜に至る音の伝達特性を頭部伝達関数として計測しておき、これを再生時の音源信号に畳み込むことで、三次元音像定位を実現する。再生装置としてはヘッドホンを用いる場合が多いが、クロストークキャンセルを行うことでスピーカによる再生を行うこともできる。d. 力覚ディスプレイ物体に触ったときに受ける反力を提示することで、仮想物体の触覚表現を行う装置が力覚ディスプレイである。代表的な力覚ディスプレイとしては、反力の生成方法によって、糸の張力を用いるもの、多関節マニピュレータを用いるもの、空気圧を用いるものなどが提案されている。また、反力の提示箇所によっても、いくつかの種類に分けることができる。例えば、指先に反力を提示するもの、ペン等をもった把持手に反力を提示するもの、腕に反力を提示するもの、指の各関節あるいは腕の各関節に反力を提示するものなどが挙げられる。e. 触圧覚ディスプレイ触覚情報として反力を提示する力覚ディスプレイに対して、仮想物体に触れたか触れないかという皮膚感覚刺激を提示する装置が、触圧覚ディスプレイである。現在提案されている触圧覚ディスプレイとしては、エアポケット等の空気圧を用いるもの、振動子による振動刺激を用いるもの、マイクロピンによる機械刺激を用いるものなどが挙げられる。また、高解像度の触圧覚ディスプレイでは、物体表面の微小形状等のテクスチャ情報の表現に利用することもできる。f. モーションプラットホーム身体の傾きや加速度を知覚する前庭感覚刺激を提示する装置が、モーションプラットホームである。この装置は、リンク機構を用いたプラットホームの上に利用者を乗せ、並進運動や回転運動の制御を行うことで前庭感覚の提示を行う。通常は構造上の動作限界があるため、プラットホームの傾斜によって生じる重力の加速度を用いて、加速度感覚を提示するなどの擬似的な手法が使われる。広視野の視覚ディスプレイと合わせて、自動車や航空機のシミュレータとして利用されることも多い。g. データグローブ計算機とのインタラクションを行うためには、計算機に対する入力デバイスが必要である。データグローブは、人間の手の動きを入力するために使われる、手袋型の入力装置である。データグローブの指の部分には、光ファイバや導電性インクを用いたセンサが縫い込まれており、これによって各指の曲がり角度が計測される。またエグゾスケルトン型の機構を手に装着し、各関節の曲がり角度を精度良く計測する装置もある。h. モーションキャプチャ計算機に対して全身の動作入力を行うために用いられる装置が、モーションキャプチャシステムである。身体の各部に装着されたマーカの動きから、身体の動作を計測するのが基本的な原理である。使用するマーカによって、磁気センサを用いる方法、赤外線の反射をカメラで撮影する光学式方法、関節に装着されたロータリエンコーダで計測する機械式方法等がある。モーションキャプチャによって計測された身体の動きは、仮想世界における人物の動作表現等にも使用される。
