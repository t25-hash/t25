# 12・11・3 計算力学と仮想現実感

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・11 仮想現実感

計算力学と仮想現実感は，どちらも計算機を利用したシミュレー
ション技術でありながら，これまではかなり異なった研究領域であ
った．しかし，高速ネットワーク環境の整備や，グラフィックスワ
ークステーションの高性能化等を背景に，ここ数年両者の技術は融
合してきている．計算力学と仮想現実感技術の結び付きには，大き
く分けて二つの方向性がある．一つは計算力学の立場から仮想現実
感技術を利用しようという立場であり，サイエンティフィックビジ
ュアライゼーション等の研究がこれに当たる．一方，仮想現実感の
立場から計算力学の成果を利用しようという立場もあり，仮想世界
における法則エンジン等の研究がこれに当たる．
a.サイエンティフィックビジュアライゼーション 数値計算
の結果を可視化するサイエンティフィックビジュアライゼーション
には，いくつかの段階がある．一般に，計算を行った後でデータを
可視化する方法をポストプロセッシング，計算を行いながら経過を
可視化する方法をトラッキング，可視化しながら計算条件を変更す
る方法をステアリングと称する．さらに，仮想現実感を利用したビ
ジュアライゼーションでは，利用者の動作に応じて動的に計算を実
行しながら可視化する，インタラクションとしての可視化が行われ
る．例えば，仮想空間内の利用者が，直接的な動作で荷重条件や境
界条件を与えながら，リアルタイムで構造解析や流体解析計算を実
行し，可視化するといった方法である．このようなシミュレーショ
ンは，仮想実験としての意味ももち，物理現象を直感的に理解する
うえで有効な方法として期待されている．
また，仮想現実感では人間のさまざまな感覚を利用した情報提示
を行うため，サイエンティフィックビジュアライゼーションにおい
ても映像を用いた可視化だけではなく，音や触覚を用いてデータの
表現を行う，可聴化，可触化といった方法も研究されている．
b.仮想世界の法則エンジン 仮想世界の現実感を高めるため
には，映像のリアリティだけではなく，法則に関する再現性が重要
な要素の一つとなる．例えば，仮想世界の中で物体を投げると運動
法則に従って飛んで行ったり，柔らかい物体に力を加えると弾性に
従って変形するといった挙動が，正しく再現されることである．仮
想現実感のシミュレーションでは，このような法則に従った運動を
リアルタイムで表現することが必要であるため，計算負荷との兼合
いで種々のモデルが提案されている．
例えば，物体の変形運動を表現する代表的なモデルとしては，ば
ねモデルが挙げられる．これは，ばねとダンパで構成されたメッシ
ュで，物体を表現する方法である．このモデルでは，ばね定数や減
衰係数の値を適当に調整することで，リアリティの高い運動を表現
することができるが，モデル自体の物理的な根拠は少ない．
一方，物体の挙動を物理法則に従って計算する手法として，有限
要素法や境界要素法等の数値解析手法がある．これらの計算手法
は，精度の高い解を出力することができる反面，一般に多くの計算
時間を要するという問題点がある．そのため，リアルタイム出力が
要求される仮想現実感シミュレーションで利用するためには，大幅
な計算量の短縮化や計算の高速化の工夫が必要となっている．
12・11・4仮想現実感の応用
仮想現実感は，人間とのインタラクションを含むシミュレーショ
ン技術であるため，人間の活動に関するあらゆる分野が応用の対象
になる．そのため，設計，医療等からアミューズメントまで応用分
野は幅広い．
a.設計 設計の分野では，実際に物を作る前に，アイデアを
いろいろな角度から検討することが必要となる．仮想現実感を用い
た設計では，設計モデルを三次元表示するなど，仮想体験を通して
利用者のニーズを製品に反映することができる．設計に関する仮想
現実感の応用研究は，工業製品から住宅，都市設計等の分野まで幅
広い．
b.医療 医療は，そのままでは目に見えない人体の内部状態
を対象とするため，仮想現実感の重要な応用分野の一つである．例
えば，CT（computed tomography）やMRI （magnetic reso-
nance imaging）の三次元映像を用いた可視化や，手術シミュレー
ション，遠隔医療等の応用研究が行われている．また入院患者や障
害者に対する効果的な治療やリハビリテーションを目的とした，仮
想現実感システムの研究等も行われている．
c .テレイグジスタンス 宇宙空間や極限作業空間等で，ロボ
ットや機械の遠隔臨場感制御を用いて，操作者がその場にいるかの
ような作業空間を構築することを目指した研究が，テレイグジスタ
ンスである．特にヒト型ロボットの実用化が近づくにつれ，ロボッ
トを介した人間のコミュニケーション手段としても，テレイグジス
タンスは重要な技術になってきている．
d.アミューズメント テーマパークやゲーム機等のアミュー
ズメントの分野は，ビジネスとして仮想現実感技術の応用が最も成
功している分野である．テーマパークでの体験型シミュレータや家
庭用ゲーム機では，高性能グラフィックスワークステーションに引
けをとらないパフォーマンスを実現している．また，テレビ放送等
の分野でも，三次元ビデオや自由視点映像等の合成技術の研究が行
われている．
e .臨場感通信 仮想現実感はネットワークと結びつくことで，
空間共有型のコミュニケーション技術としての利用が可能になる．
仮想空間の共有に関する研究は，既存のインタネットを使用したサ
イバスペースの構築から，広帯域ネットワークを用いた没入型コミ
ュニケーションの実現まで，さまざまである．この際，共有仮想空
間の中で利用者を表現するアバタの構築方法も，重要な研究課題の
一つである．
