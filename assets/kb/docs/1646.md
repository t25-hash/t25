# 12・11・3 計算力学と仮想現実感

> **章**: 第12章 インテリジェントシミュレーション
> **節**: 12・11 仮想現実感

計算力学と仮想現実感は、どちらも計算機を利用したシミュレーション技術でありながら、これまではかなり異なった研究領域であった。しかし、高速ネットワーク環境の整備や、グラフィックスワークステーションの高性能化等を背景に、ここ数年両者の技術は融合してきている。計算力学と仮想現実感技術の結び付きには、大きく分けて二つの方向性がある。一つは計算力学の立場から仮想現実感技術を利用しようという立場であり、サイエンティフィックビジュアライゼーション等の研究がこれに当たる。一方、仮想現実感の立場から計算力学の成果を利用しようという立場もあり、仮想世界における法則エンジン等の研究がこれに当たる。a. サイエンティフィックビジュアライゼーション数値計算の結果を可視化するサイエンティフィックビジュアライゼーションには、いくつかの段階がある。一般に、計算を行った後でデータを可視化する方法をポストプロセッシング、計算を行いながら経過を可視化する方法をトラッキング、可視化しながら計算条件を変更する方法をステアリングと称する。さらに、仮想現実感を利用したビジュアライゼーションでは、利用者の動作に応じて動的に計算を実行しながら可視化する、インタラクションとしての可視化が行われる。例えば、仮想空間内の利用者が、直接的な動作で荷重条件や境界条件を与えながら、リアルタイムで構造解析や流体解析計算を実行し、可視化するといった方法である。このようなシミュレーションは、仮想実験としての意味ももち、物理現象を直感的に理解するうえで有効な方法として期待されている。また、仮想現実感では人間のさまざまな感覚を利用した情報提示を行うため、サイエンティフィックビジュアライゼーションにおいても映像を用いた可視化だけではなく、音や触覚を用いてデータの表現を行う、可聴化、可触化といった方法も研究されている。b. 仮想世界の法則エンジン仮想世界の現実感を高めるためには、映像のリアリティだけではなく、法則に関する再現性が重要な要素の一つとなる。例えば、仮想世界の中で物体を投げると運動法則に従って飛んで行ったり、柔らかい物体に力を加えると弾性に従って変形するといった挙動が、正しく再現されることである。仮想現実感のシミュレーションでは、このような法則に従った運動をリアルタイムで表現することが必要であるため、計算負荷との兼合いで種々のモデルが提案されている。例えば、物体の変形運動を表現する代表的なモデルとしては、ばねモデルが挙げられる。これは、ばねとダンパで構成されたメッシュで、物体を表現する方法である。このモデルでは、ばね定数や減衰係数の値を適当に調整することで、リアリティの高い運動を表現することができるが、モデル自体の物理的な根拠は少ない。一方、物体の挙動を物理法則に従って計算する手法として、有限要素法や境界要素法等の数値解析手法がある。これらの計算手法は、精度の高い解を出力することができる反面、一般に多くの計算時間を要するという問題点がある。そのため、リアルタイム出力が要求される仮想現実感シミュレーションで利用するためには、大幅な計算量の短縮化や計算の高速化の工夫が必要となっている。
