# 13・11・3 アコースティックエミッション

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

磁束の検出方法により磁粉探傷試験（magnetic particle testing）と漏えい磁束探傷試験（flux leakage testing）に分類される．
被で磁化するコイル法，被検査材に直接電流を流す通電法などがあ磁粉探傷試験は探傷面に強磁性粉末を散布し，漏えい磁束に吸着法と液体に懸濁させた検査液を用いる湿式法がある．
磁化方向と平の場合には磁化方向を90 °変えて探傷を行う必要がある．
磁粉探傷適用法を適切に選択すれば微小な欠陥の検出も可能である．
い，欠陥による漏えい磁束の強さを電気信号に変換して検出する方束の強さからき裂の深さに関する情報も得られる．
アコースティックエミッション（AE:acoustic emission）は，検査材の磁化方法には電磁石を用いる極間法，コイルにより非接触る．
させて磁紛模様を観察する方法である．
乾燥した磁紛を用いる乾式行なき裂に対しては感度が極端に低くなるので，き裂の方向が未知は強磁性材の表面欠陥や溶接部の欠陥検出に広く用いられており，に適している．
圧力容器の保証・保守検査や発電タービン軸受の摩法である．
