# 13・1・1 解析とプリ・ポストプロセッシング

> **章**: 第13章 シミュレーション支援法
> **節**: 13・1 プリ・ポストプロセッシング

近年,有限要素法・差分法・境界要素法などのシミュレーション手法が飛躍的な進歩を遂げてきた.また,計算機の性能向上に伴い,大規模で複雑な問題の解析も可能になってきた.しかしながら,解析技術が進歩する一方で,解析のためのモデル生成(データ作成)に大きな労力が必要になってきた.図13・1は,解析を用いて製品を設計していく過程を簡略化して示したものである.まず,ある設計要求に従って,CAD(computer aided design)等の設計ツールを用いて製品の設計を行う.次に,その設計結果に基づいて解析を行い,要求を満たすかどうかの判断を行う.さらに,この評価を基に再設計(設計の変更),再解析が行われ,設計・解析のループが何度も繰返されることになる.このような一連の解析過程を省力化するためには,単に解析を行う部分を高度化するだけでなく,解析のための形状定義やメッシュ生成等のプリプロセッシング(preprocessing),および,膨大な解析結果の処理や設計・解析過程の支援を行うポストプロセッシング(post-processing)の効率化が重要である.
