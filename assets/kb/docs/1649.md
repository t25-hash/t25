# 13・11・6 マイクロ波，ミリ波による方法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

音波が反射される性質を利用して欠陥を検査する方法である．通
常，周波数帯域1～10MHz程度の超音波を探触子（probe ）によ
り被検査材内に入射し，底面や欠陥による反射波を探触子で受信す
る．なお探触子と被検査材の間には水や油などの接触媒質を介し，
超音波が透過しやすくする必要がある．得られた探傷信号より欠陥
の位置や大きさを知る．代表的なき裂評価法として表面波法（sur-
face wave technique) ，端部エコー法（tip-diﬀraction tech-
nique) などがある．また二つの縦波斜角探触子を用い，試料表
面を直進するラテラル波とき裂端部からの散乱波の到達時間差より
き裂の深さや高さを評価するTOFD（time of flight diﬀraction）
法 も開発されている．
き裂面の接触突起を介して透過し，反射波の強度が低下して検出が
困難になる．これに対し，静的荷重や振動荷重を負荷してき裂を開
かせて探傷する方法や，無負荷状態でき裂寸法とき裂の閉じ具合を
未知量とした逆問題解析により評価する方法が開発されてい
る ．
材料内における塑性変形やき裂の発生・進展に伴って放出される弾
性波であり，これを計測することにより破壊の進行を評価する方法
をAE法という．AEの周波数帯域は可聴域から数MHzであり，
その計測には圧電素子などのAE変換子（AE transducer ）が用い
られる．地震の震源を知るのと同様に，複数の場所で計測したAE
信号を解析することにより，破壊の発生位置や規模などを評価する
ことができる．応力履歴のある材料では，過去に受けた最大応力以
下ではAEは計測されない．この現象はカイザー効果と呼ばれ，材
料の応力履歴を知る指標になる．AE法の特徴は，現在進行中の破
壊をリアルタイムにモニタリングできることであり，稼動中の監視
漏えい磁束探傷試験はコイルやホール素子などの磁気センサを用
超音波探傷試験（ultrasonic testing）は材料内の不連続部で超
マイクロ波（microwave）は周波数300MHz ～300GHz （波長
1m～1mm）の帯域の電磁波である．同帯域の電磁波のうち，周
波数30～300GHzのものはミリ波（millimeter wave）とも呼ばれ
る．マイクロ波を材料に照射したときの反射，透過，減衰などの性
質は，材料の電磁気的特性やその不連続によって異なる．この性質
を欠陥などの検査に用いることができる．マイクロ波は導電材料を
透過できないので，金属材料については表面欠陥のみの検出が可能
である．一方，マイクロ波は誘電材料を容易に透過するので，セラ
ミックス，ポリマ，複合材料等を対象として，はく離の検出など内
部の検査が可能である．また測定結果が材料の誘電特性に関係する
ので，誘電率の計測や吸水率の測定など欠陥以外の評価への応用も
行われている．マイクロ波による非破壊試験では，非接触測定が可
閉じたき裂に対して超音波探傷試験を適用する場合，音波が閉口
能で接触媒質が不要である．また振幅以外に位相や周波数等に関す
る測定も容易であり，得られる情報量が多い．
