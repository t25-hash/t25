# 13・1・4 ポストプロセッシング

> **章**: 第13章 シミュレーション支援法
> **節**: 13・1 プリ・ポストプロセッシング

a. 解析結果のグラフィックス表示解析が大規模になればなるほど,その結果を示す数値データは膨大になり,解析者がそれらを直接見て評価を行うことは不可能となる.そのため,グラフィックス表示による解析結果の可視化が必要不可欠である.図13・17に,二次元問題における簡単な可視化の例を示す.図13・17(a)は,構造解析等で変形状態を評価・確認するための変形図であり,変形後の節点の位置で,メッシュを書くことによって得られる.図13・17(b)は,ある物理量の分布を表すコンタ図と,矢印を用いて物理量の大きさと方向を示す方向図を,同時に示したものである.等高線図やコンタ図は,流体問題での圧力や速度分布表示,構造問題での応力分布の表示等に用いられる.また,方向図は,流体問題における流れの速度表示や,構造問題における主応力の表示などに用いられる.解析者は,このような結果表示を基に工学的な判断を行うことになる.b. 解析結果のフィードバックポストプロセッシングの重要な役割として,解析結果を基に設計変更を合理的に行うためのデータ修正支援機能や,より精度の高い解析を行うための解析支援機能が挙げられる.. アダプティブメッシング(順応型要素再分割)解析を行った後に,その結果を利用してさらに精度の高い解が得られるように,メッシュを修正する手法をアダプティブメッシングと呼ぶ.図13・18に,有限要素法による代表的な手法を示す.通常の解析後に事後誤差解析を行い,その結果誤差の高い要素を小さくする.r法[図13・18(b)]では,節点の位置を移動させることにより,誤差の高いハッチ部の要素を小さくしている.また,h法[図13・18(c)]では,誤差の高い要素をさらに小さい要素へ分割する.(b) コンタ図・方向図(a) 変形図図13・17 解析結果の可視化の例(c) h法(d) p法(b) r法(a) 元のメッシュ図13・19 DOループの例図13・18 順応型要素再分割図13・18(d)のp法では,誤差の高い要素の近似関数の多項式の次数を,高次なものにすることで解析精度を上げる.また,このような局所的なメッシュの修正を行わずに,誤差の大きい要素付近のメッシュが細かくなるように,解析対象全体のメッシュ生成を最初からやり直す場合もある.. ズーミング解析(zooming analysis)大規模な構造物の解析等において,構造詳細部の局所的な応答を得たいとする.しかしながら,詳細部まで表現するような細かいメッシュ生成を行うと解析の自由度が増えてしまうために,ハードウェア上の制約等から解析が不可能な場合がある.このような場合に,ズーミングと呼ばれる多段階の解析が用いられることがある.ズーミングでは,まず粗いメッシュで全体の解析を行う.次に,詳細部付近のみの領域をより細かいメッシュで分割した,詳細部解析用のモデルを作成する.そして,粗い解析で得られた変位,または,応力を詳細解析の対象領域の境界条件として与えることにより,局所的な解析が可能となる.なお,ズーミング解析の境界条件として応力を用いるときは,SPR 法(superconvergent patch recovery)等による応力値の修正が有効である.
