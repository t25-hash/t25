# 13・11・8 渦流探傷試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・11 き裂および欠陥の検査法

渦流探傷試験（eddy current testing）は，電磁誘導現象を利用
して導電性材料内の欠陥を検出する方法である．交流を流したコイ
ルを導体に接近させると導体内には渦電流が誘導される．このとき
コイルには渦電流より生じる二次磁界によって逆起電力が発生す
る．導体表面にき裂などの欠陥が存在すると渦電流の流れが妨げら
れ，コイルに生じる逆起電力が変化する．同変化をコイルインピー
ダンスの変化として検出することにより，欠陥の検査を行うことが
できる．渦流探傷試験は非接触での探傷が可能であり，欠陥信号が
電気信号で得られるので高速・自動化した探傷が可能である．一
方，渦電流は表皮効果により材料表面に集中するため，表面の微小
な欠陥を高感度に検出できる反面，薄肉部材の場合を除き，内部や
裏面にある欠陥の検出は困難である．また欠陥以外にリフトオフや
被検査材の導電率・透磁率などの影響によって信号が変化する．し
たがって多様な用途に応用できるが，欠陥のみを検出する場合には
それ以外の要素の変動に対する対処が必要である．特に加工等によ
り透磁率が不均一になりやすい鉄鋼材料などの強磁性体では /
比が低下する．これに対し，鋼管を対象に数十Hz程度の励磁周波
数を用い，励磁コイルから管径の2～3倍以上離れた位置に検出コ
イルを配置することにより，透磁率の不均一から生じる雑音を受け
ずに管内外面の欠陥を検出できるリモートフィールド（remote
field）渦流探傷試験法が開発されている ．また，き裂信号をも
とに逆問題解析を行い，き裂の断面形状を評価する手法の開発も進
められている ．
