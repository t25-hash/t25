# 13・2・2 アーキテクチャにおける並列処理

> **章**: 第13章 シミュレーション支援法
> **節**: 13・2 ベクトル・パラレル

a. ベクトル処理(vector processing)ベクトル処理はデータレベルでの並列化の一つである.ベクトルデータを同時に処理できるベクトルレジスタと,パイプライン(セグメント)化されたベクトル演算器との組合せによって,高速演算が実現される.図13・20にベクトル処理とスカラ処理の概念図を示す.ベクトルデータをベルトコンベヤに乗って流れる多数のデータ列に例えると,ベルトコンベヤに沿って並べて置かれた装置によって,例えば加算演算の場合,メモリからの読み込み(R),指数部比較(C),指数部合わせ(S),加算(A),正規化(N),メモリへの書き込み(W)といった操作が,データに実行される.ベルトコンベヤがパイプラインに相当する.これらの演算操作は,パイプライン上でオーバラップしてベクトルデータに次々と実行される(パイプライン実行される)ため,並んだ装置の数だけ速度が向上することになる.こうしたベクトル処理に対し,スカラ処理では,一つのデータがベルトコンベヤを流れ終えてはじめて,次のデータをベルトコンベヤに投入することができる.ベクトル型スーパーコンピュータでは,複数個のベクトル演算機を装備することにより,さらに高速化が図られている.b. RISCRISC(reduced instruction set computer,縮小命令セット方式)と呼ばれている方式では,使用頻度が高く実行時間の短い単純な命令セットのみを,メモリよりもアクセス速度の速いキャッシュメモリとのアクセスにより,高速処理している.また,命令実行がパイプライン化され,命令の見掛け上の1サイクル時間が短縮化されている.スーパースカラと呼ばれる計算方式では,複数の命令列のうち可能なものから並列実行される.この方式のプロセッサは,パーソナルコンピュータに広く用いられている.また,並列計算機の要素プロセッサとしても用いられている.c. マルチプロセッサアーキテクチャ面での並列化のレベルをもう一段階上げたものが,いわゆる並列計算機である.複数のプロセッサにメモリをどう割当てるかによって,共有メモリ型,分散メモリ型,共有・分散メモリ型の三つの方式がある(図13・21).共有メモリ型は,全プロセッサから等距離の一つの大容量メモリにアクセスするタイプのもので,SMP(symmetrical multi proces-sor)とも呼ばれる.パーソナルコンピュータでもマルチプロセッサ搭載のSMPが登場している.分散メモリ型は,プロセッサとメモリの組を一つの単位(ノード)とし,それらをさまざまなトポロジーを有するネットワークで接続したものである.共有・分散メモリ型は,共有メモリ型マルチプロセッサを1ノード(SMPノー(a) ベクトル処理(b) スカラ処理図13・20加算演算[C(K)=A(K)+B(K)のDOループ]におけるベクトル処理とスカラ処理の概念(R:データ入力,C:指数部比較,S:指数部合わせ,A:加算,N:正規化,W:データ出力)6―19 7ド)とし,それらをネットワーク接続したアーキテクチャで,いわば共有メモリ型と分散メモリ型を組合わせた方式である.多くのハイエンド並列計算機では,高速なネットワークを用いた共有・分散メモリ型を採用している.コモディティプロセッサを比較的安価で低速なLAN でネットワーク結合したものを,一般にクラスタ計算機あるいはPCクラスタと呼び,並列計算機として多く用いられている.世界のスーパーコンピュータの性能一覧であるTOP500にも,このタイプが多く見られるようになっている.d. グリッド(grid)グリッドあるいはグリッドコンピューティングとは,各地に散らばる計算機やデータベースなどのリソースをインタネットや専用回線で結び,次世代の計算環境ひいては社会基盤にしようというものである.その名称はパワーグリッド(power grid,電力網)とのアナロジーから命名されている.多様なOS,計算機,ネットワーク,データベース等を統一的なツールやインタフェースで取扱うために,さまざまなミドルウェアやプロトコルが開発されている.有名なミドルウェアとしては,Globus,Ninfなどがある.グリッドの利用形態としては,世界中に分散しているスーパーコンピュータをネットワーク接続して超大規模計算を目指すメタコンピューティング,大容量データの分散配置や処理を行うデータインテンシブコンピューティング,遊休PCなどの計算機資源を活用するボランティアコンピューティング,地理的に分散した場所から研究者がバーチャルリアリティなどのツールを援用して協同作業を行うテレイマージョン,ハイエンドのスーパーコンピュータ等に実装されて動作が保証されている高性能アプリケーションを遠隔地から利用するグリッドASPなどがある.
