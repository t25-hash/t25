# 13・3・1 可視化の概要

> **章**: 第13章 シミュレーション支援法
> **節**: 13・3 ビジュアリゼーション

ビジュアリゼーションのプロセスは,可視化対象となるデータの選択の後,可視化手法の選択,可視化結果の解釈を,解析者が繰返すことで進められる.可視化の意味付けや効果は,このサイクルの中で見出されることが多い.可視化対象によっては一連の作業手順が決まっている場合もあるが,解析者の経験に依存する部分が多く,統一的な手順は少ない.このため,複数の可視化手法を組合わせることになる.近年のシミュレーションにおいては,半透明等値面表示をベースにした可視化が多い.また,流体解析であれば,粒子追跡をベースとしていくつかの可視化が組合わされる.図13・28に,屋根付き競技場の火災シミュレーションの可視化例を示す.この場合は,競技場内で発生した火災による対流と外部流れの干渉を調べるという目的に対して,等温面表示,断面上での等温領域表示,建物外部の地面上での等圧線表示と粒子追跡表示を組合わせて6―201いる.建物内部の様子は温度分布によって表現し,外部に対しては,外部流れの把握に有効な圧力分布と粒子追跡を用いている.このような可視化図は1回の試行で得られるものではない.現象の予測,あるいは多くの試行錯誤が必要になる.可視化手法に関していえば,LIC(line integral convolution)などの一部の方法を除き,1980年代に完成している.その後,可視化作業の効率化のために可視化システムが発展することになり,研究領域としてのビジュアリゼーションは,システム構築に移行していった.現在では,計算環境の多様化や可視化結果からの情報抽出支援の必要性から,ビジュアリゼーションの研究は,階層的可視化手法の構築,特徴領域の抽出技術,全情報の表示技術,可視化結果の定量的比較技術,データ圧縮法,知識データベースの構築,可視化に起因する誤差,並列化可視化技術,遠距離可視化技術,表示技術を中心として進められている.研究の最先端を除くと,個々の可視化手法の完成度は高く,効率的な可視化プロセスを目標として多くの可視化システムも提示されており,AVS やFieldViewのように,解析者にとって使いやすい市販の可視化ソフトウェアも利用できるようになっている.このため,ユーザ側で可視化プロセスの個別要素の中身を意識する必要はなくなっている.しかし,可視化の組合せが自動的に選択されるわけではなく,必要とする情報を効率的,かつ正確に手に入れるためには,個々の可視化手法の仕組みを知っておくことは重要である.
