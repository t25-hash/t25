# 13・13・1 変位計

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

ゲージ表面に塗布すればよい．
光ファイバひずみセンサ（fiber optic strain sensor）では光ファイバの各種光学的特性を利用する．
光ファイバの一部（ゲージ部）を被測定物に貼り付けるか埋め込んだ場合，被測定物の変形に追随してゲージ部の長さや屈折率が変化し，伝送光の位相がシフトする．
この位相シフトを光波干渉法によって検出すれば，ゲージ部のひずみが計測できる．
あらかじめ屈曲させたゲージ部の曲率変化に伴う光伝送損失の変化を検出すれば，ゲージ部のひずみ変化がわかる（光ファイバ式ひずみゲージ)．
さらに，ゲージ部コア内に長さ方向の周期的な屈折率変化構造（ファイバグレーティング）を有するひずみセンサでは，ゲージ部でブラッグ（Bragg）の条件を満たす波長（グレーティング周期に比例する波長）をもった光のみが選択的に反射され，変形によってグレーティング周期が変化すると反射波の波長も変化する．
この波長変化を光スペクトルアナライザによって検出すれば，ゲージ部のひずみがわかる（ファイバグレーティングひずみセンサ）．
このほか，光ファイバに生じるブリルアン（Brillouin）後方散乱光のひずみによるスペクトルシフトに着目した方法もある．
引張試験など被測定物形状や負荷形式が比較的簡単な場合には，伸びなどの変形に伴う変位を計測することによって，ひずみが評価できる．
ここでは，各点測定用の変位計に注目する．
変位計には大きく分けて，接触式と非接触式がある．
接触式変位計には，測定子（触針）の機械的変位を，差動変圧器，静電容量変化，ひずみゲージ（13･13･2項）の抵抗変化などを利用して電気量に変換するもの（電気マイクロメータ）や機械的に指針の角度変位量に変換するもの（ダイヤルゲージ）がある．
また，測定子変位を光学式スケールや磁気スケールまたは干渉測長光学系を用いて計数方式で計測するもの（ディジタルマイクロメータ）がある．
一方，非接触式変位計には，圧縮空気流体素子を利用した空気マイクロメータ，測定プローブ用電極と被測定物の間の電気容量変化を検出する静電容量型変位計，測定プローブの一次コイルによって被測定物の導電体に渦電流を発生させ，これにより誘起されるプローブ内二次コイルの電圧を検出する渦電流型変位計がある．
