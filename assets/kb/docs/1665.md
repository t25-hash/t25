# 13・13・12 切抜き，切込みおよび穴あけによる残留応力測定

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

法
残留応力が生じている試料に切込みを入れる，あるいは，その一
部を切抜くと，部分的に残留応力が解放されて試料が変形する．本
手法（sectioning method）は，その切断面に残留応力と大きさが
等しく逆向きの応力を負荷すれば切断による変形が元に戻るという
原理に基づいて，切断に伴う変形量から残留応力を評価するもので
ある．図13･35(ａ)は，切込みによる変形量を直接測定する場合の
例であり，図中の式を使って円管壁面の周方向残留応力が求められ
る．一方，図13･35(ｂ)は，切抜き（穴あけ）前に接着したひずみ
ゲージによって切抜きに伴う板の変形をひずみ量として測定する場
合であり，ひずみ量から図中の式を使って残留応力の主値が求めら
れる．
