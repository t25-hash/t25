# 13・13・13 画像処理法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

高速化，高精度化を図るために，光弾性応力測定法，モアレ法，
光学的干渉法などのひずみや応力の測定法では，測定対象領域全体
の画像をディジタル化してコンピュータに取込み画像処理（image
processing）し，ひずみ分布や応力分布を求めることが一般的にな
ってきた．ディジタル化された画像の画質は，空間フィルタや周波
数帯域フィルタを用いて高周波成分を有する雑音を除去し平滑化さ
れる．平滑化され濃淡のある画像を白と黒に2値化し，得られたし
ま模様を細線化してひずみや応力を求めるためのデータを得る．ホ
ログラフィー干渉法やスペックル干渉法などでは変形前後の画像を
干渉させ，しま模様を得るために画像処理法が用いられる．変形格
子をコンピュータ上で平行移動して重ね合わせ等ひずみ線を表す干
渉じまを得るモアレのシフト法やコンピュータ上で基準格子を作る
走査モアレ法も行われている．
モアレ法，光学的干渉法などしまの間隔がひずみに関係している
ときには，しま模様を波紋ととらえて，周波数解析や位相解析を用
いてひずみを求める方法がある．周波数解析や位相解析にはフーリ
エ変換，ガボール変換，ウェーブレット変換およびアダマール変換
等が用いられる．各場所での変形前後の周波数を求め，周波数の差
を変形後の周波数で割ってひずみを求める．また，変位を表す位相
差を微分することによりひずみ分布を求めることができる．変形格
子のスペクトルと位相解析のみで変形を解析するフーリエ変換モア
レ法も行われている．
