# 13・5・1 デバイス技術

a. CMOS(complementary metal oxide semiconductor)半導体技術これら科学技術計算を支えるスーパーコンピュータの心臓部となるLSI(large scale integration)の性能は,トランジスタや配線の微細化によって向上を続けてきた.2~3年ごとにトランジスタ寸法が0.7倍,トランジスタ密度が2倍という割合での寸法縮小(ムーアの法則と呼ばれる)が,過去30年以上にわたって続いている.表13・3にITRS 2005年版(世界の半導体業界が共同でまとめた半導体のロードマップ)を示す.微細化により,トランジスタ集積度の向上に加え,トランジスタのスイッチ動作の高速化が期待される.トランジスタのスイッチ速度は単純に/と書ける.ここでは総負荷容量,は電源電圧,はトランジスタの電流である.1/倍の微細化により,1個のトランジスタが充放電するはほぼ1/倍に減少する一方で,トランジスタの駆動能力/はほぼ一定に保つことができる.このため倍のスイッチ速度向上が実現できる.以上のように,1/倍の微細化により,理想的には倍の集積度向上と,倍のスイッチ動作高速化を実現することができる.今後も,後述する技術課題が克服されれば,指数関数的な微細化,高密度化が続き,2010年前後には・1チップ当たりのトランジスタ数~10億個・動作周波数~10GHz・浮動小数点演算ピーク性能~100GFLOPS(ギガフロップッス)のプロセッサが開発されるものと予想される.b. 半導体技術課題トランジスタの微細化は順調に進められてきているが,ITRS 2005によれば2005年に32nmのゲート電極幅が,2010年には20nmを切り,ゲート絶縁膜の厚さはすでに2nmを切っている.20nmはシリコン原子70個分,2nmは8個分の間隔にすぎない.このトランジスタ微細化に関する重要課題として,ゲートリーク(gate leakage)とサブスレッショルドリーク(subthreshold leakage)が挙げられる.ゲートリークとは,トランジスタのゲート絶縁膜が薄くなりすぎて,トンネル現象により電流が漏れてしまうことである.これには,従来のゲート絶縁体SiOを代替する,より誘電率が高い絶縁材料(高誘電率ゲート絶縁膜)の開発が進められている.サブスレッショルドリークとは,微細化を進めるとき,同時に電源電圧も1/倍に下げていくことが消費電力を増加させないために必要であるが,このときトランジスタのしきい値電圧(オンとオフの境目となる入力ゲート電圧)も下げないと,トランジスタの駆動能力/が劣化する.しかし,しきい値を下げると,トランジスタがオフのとき流れるリーク電流が増す.これがサブスレッショルドリークである.スーパーコンピュータのような速度優先のLSIでは比較的大きなリーク(待機電力)を許容できるが,それには下限がある.速度向上を制約するもう一つの要因として配線がある.理想的には,1/の微細化により配線の容量も1/に減るはずだが,実際は集積度の増大により,配線長は1/ほどには減らない.また,微細化すると配線抵抗が大きくなるため,配線高を1/ほどには減らせない.これらのため,微細化にはが減りにくく抵抗が増えていく傾向があり,高速化を阻害している.この問題の対策として配線材料がアルミニウムから低抵抗な銅に変更され,また,配線を囲む絶縁体として,低誘電率材料の開発が進められている(従来はSiO).上述のような,トランジスタの高速化を微細化による方法で実現する以外の方法も開発されている.代表的なものとして,以下の二つが挙げられる.(1) SOI(silicon-on-insulator)化これはトランジスタを絶縁体の上に形成することにより,トランジスタ自身の寄生容量を削減するとともに,トランジスタの基板部分が電気的に浮遊状態となることを積極的に利用して,性能を向上させるものである.(2) 高移動度化半導体の移動度自体を向上させることで,キャリヤが走る速度そのものを向上される方法が検討されている.例えば,Siより格子間隔が長いSiGe욪上にSiを薄く成長させると,引張ひずみを受けたSi膜ができる.このようなSi膜はキャリヤ移動度が高く,同じ寸法でも従来トランジスタより数十%,電流を増加させることができると期待されている.c. LSI 技術. 集積化とトランジスタの利用これまでトランジスタ数の増加は演算精度の向上(64ビット化),メモリ空間の増大,パイプライン型浮動小数点演算ユニットの実装に振り向けられてきた.10億トランジスタ時代には,主記憶とCPU(central processingunit)を一つのLSIチップ上に同居させることが可能になり,主記憶からCPU へのデータ供給能力を安価に,かつ格段に高くできる.このようなチップはCIM(chip in memory),また,そのメモリはオンチップメモリ(on-chip memory)と呼ばれている.科学技術用の計算機では,CPU 性能と主記憶量はバランスさせる必要がある.一般的には,GFLOPS の計算機にはGB(ギガバイト)の主記憶が必要とされている.たとえ10億トランジスタをもってしても,従来のSRAM を使う限りオンチップメモリの総量は数百MB(メガバイト)程度であり,100GFLOPS のチップでは必要量の100分の1以下であることに注意を要する.. con-sumption)インテル社ゲルシンガー(Gelsinger)は,1990年代前半までのマイクロプロセッサの電力密度は7W/cm以下であったが,今後トランジスタ集積度が向上し消費電力が増加し続けると,2010年には1000W/cmに達し,ロケットノズルと同じになると警告している.図13・40のLSIの電力密度トレンドを見ると,ゲルシンガーの警告は悲観的であると思われるが,微細化に伴い高性能LSIの冷却が難しくなることは事実であり,低電力化技術が重要となる.LSIの消費電力は,アクティブ電力とスタンバイ電力の和で表現される.アクティブ電流は(配線容量+ゲート容量)×(電源電圧の二乗)×(動作周波数)×(動作率α)に比例する.またスタンバイ電力は(オフ電流係数×総トランジスタ幅×電源電圧×定数β)+(ゲートリーク電流係数×総トランジスタ幅×電源電圧×定数γ)に比例する.ここで,総トランジスタ幅は,一つのトランジスタ幅×チップ内トランジスタ総数を意味する.アクティブ電力を抑えるには,・高並列アーキテクチャ等による動作周波数と電源電圧の抑制・使用しないFF回路へのクロックの供給を行わないようにする・クロックゲーティング技術や専用アーキテクチャによる動作率αの抑制スタンバイ電力の抑制には,高誘電率ゲート絶縁膜を用いるほかに,・高速と低速のトランジスタを最適に組合わせるマルチVth(MVT : multiple threshold voltage)技術の採用・VTCMOS(variable threshold voltage complementary MOS)回路の採用(pMOS,nMOS トランジスタそれぞれの基板電位を回路の電源電位と別々に制御できるようにし,pMOS,nMOSトランジスタともに逆バイアスになるように制御することで,動作時の高速性とスタンバイ時の低リーク電流を実現)・動作していない回路の電源を遮断する電源スイッチの採用などがそれぞれ考えられる.. クロック分配技術とノイズ対策近年のマイクロプロセッサはPLL(phase lock loop)を内蔵し,PLL経由で数万~数十万ビットのFFにクロック信号を分配することで,外部クロック信号がもつジッタ(信号到達時間のあいまいさ)を遮断するようにな6―210計算力学った.しかしPLLで外部クロックのジッタは遮断できたとしても,外部電源電圧変動やノイズによってクロック分配遅延時間が変動することで,LSI内のクロック信号は刻一刻と変動している.このノイズの影響を低減させるためには,以下の手法が有効である.・クロックゲーティング技術等を用いて動作率αを削減しノイズ発生量を低減させる.・ノイズ源の近くに容量を配置したり電源配線のインピーダンスを最適化することで,ノイズが伝ぱしないようにする.・ジッタをキャンセルする耐スキュー回路技術を採用する.d. 相互接続技術(interconnection technology)並列計算機として高い実効性能が要求される場合,計算ノード間を,そのCPU 速度に見合った高速の結合ネットワークで相互接続する必要がある.求められるネットワークのバンド幅は,応用ソフトウェアにも依存するが,はん用性計算機では,計算ノード内主記憶アクセスバンド幅の数十分の1程度が要求される.2010年前後に予想される100GFLOPS のプロセッサ個でノードを構成し,ネットワークのデータスループットを計算ノード内主記憶アクセスバンド幅の1/とすると,ペタフロップスのピーク性能の計算機に必要な結合ネットワークは,・1ノード当たりのバンド幅:100GFLOPS××8B/FLOPS/GB/s・全並列数:10000/になる.ちなみに,=8,=32では,必要バンド幅は,200GB/s,並列数1250である.また,ノード間の距離は数十mに及ぶ可能性がある.このような高バンド幅,長距離接続には光通信が適している.この光相互接続には,光ファイバ等の導波路を用いる光相互接続と,導波路を用いない自由空間型相互接続がある.光通信では,時分割マルチプレックス(TDM : time division multiplexing)に加え波長分割マルチプレックス(WDM : wavelength divisionmultiplexing)が可能になる.さらに,自由空間型相互接続ではスペーシャルマルチプレックスも利用できるため,並列数の大きいシステムには適している.
