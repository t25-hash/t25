# 13・13・7 レーザラマンによる応力測定法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

試料にレーザ光を照射したときの反射光として得られるラマン散
乱光を測定し，そのスペクトルを解析することで，X線法と同様
に非破壊・非接触的に試料表面の応力状態を測定することができる
（laser Raman spectroscopy）．すなわち，レーザ光のフォトンと
結晶格子フォノン間のエネルギー変換（非弾性散乱）により，反射
光には材料固有のスペクトルをもったラマン散乱光が含まれる．結
晶格子にひずみが発生すると，フォノンのエネルギー状態がひずみ
エネルギーに対応して変化するため，ラマン散乱光のスペクトルの
ピーク位置が変化する．また，このピーク位置の変化量と結晶ひず
み（応力）の間には比例関係があることから，この変化量を測定す
ることで応力の測定が可能となる．直径1m程度と極めて微小な
領域の測定が可能であることから，半導体チップ内部における微細
部分の応力評価への適用が試みられている ．
