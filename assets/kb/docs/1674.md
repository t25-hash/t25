# 1・1・3 デバイス技術・実装技術

> **章**: 第1章 情報システム
> **節**: 1・9 )。共有メモリアーキテクチャはCPU がメモリ上のデータに簡

1940年代にトランジスタが発明されてから，半導体技術は急速
な進歩を遂げてきた．半導体チップの集積度は，18箇月で2倍に
なるという，いわゆるムーアの法則（Moore’s law）が有名である
が，科学的な根拠がないにもかかわらず，おおむねこれに従った指
数的な伸びが続いている．
従来はいくつかのチップで構成されていたCPUも集積度が向上
し，一つのチップで構成されるようになり，マイクロプロセッサと
呼ばれるようになった．現在のほとんどのCPUはマイクロプロセ
ッサになっている．チップの面積（ダイサイズという）にもよる
が，インテル社が1971年に開発した世界初のマイクロプロセッサ
である4004ではトランジスタ数は2300なのに対し，2000年に登
場したPentium4では4200万トランジスタが搭載されている．
現在のマイクロプロセッサに用いられているトランジスタは，
CMOS（相補型金属酸化膜半導体，complementary metal oxide
semiconductor）と呼ばれる半導体回路である．これは，電界効果
トランジスタ（MOSFET）をペアで用いた回路で，それまで用い
られていたバイポーラトランジスタに比べて，定常状態では回路に
電流がほとんど流れないため消費電力を抑えられ，発熱が少ないた
め，集積度が高められるというメリットがある．集積度を表す単位
として，半導体製造装置の最小加工寸法を用いるが，具体的には
CMOSトランジスタのゲート長のことを表し，「○ mプロセス」
という言葉を使う．Pentium4に用いられているのは，0.13mプ
ロセスである．
CPUの回路全体はクロック（clock）信号により駆動されるた
め，CPUの速度はクロック周波数が高くなればなるほど速く動作
する．クロックの間隔は1クロックで動作する回路の複雑さにより
決定されるが，基本的にはトランジスタの動作速度とトランジスタ
間の配線の信号の伝達遅延に依存する．チップ上の配線は抵抗と寄
生容量を持っており，そのために遅延を生じる．これに対し，微細
加工により配線を細くすることで抵抗と寄生容量を小さくして，遅
延時間を短くすることができる．したがって，より微細なプロセス
により集積度を上げると同時に回路の遅延時間を小さくし，回路を
高速に動作させることができるようになる．0.13mプロセスを用
いているPentium4では3GHzを超えるクロックで動作するよう
になった．
CMOSトランジスタの動作を速くするためには駆動する電圧を
上げればよい．以前は5Vで駆動される回路が一般的であったが，
ゲート長が小さくなるにつれ，電圧をかけすぎるとゲート間の絶縁
膜が破壊されるという問題点があった．そのために，最近のプロセ
ッサで用いられている0.18mプロセスのCPUでは1.5V付近の
電圧で駆動されている．また，電圧を上げ，クロック周波数を高く
し，多くの電流を流すことにより消費電力も増えるため，回路から
の発熱が問題になる．そのために，トランスメタ社のプロセッサで
は電圧と周波数を動的に変えるDVS（dynamic voltage scaling）
という技術を用いて，発熱の状況に従って自動的にクロック周波数
と電圧を制御している．
一方，集積度が向上すると多くのトランジスタが搭載されるため
に，相対的に配線長が長くなる傾向にある．そのため，多くのプロ
セッサでは配線遅延がクロック速度を決定するようになってきた．
配線の遅延を小さくするために配線の抵抗を下げる技術が急務とな
っている．その一つが，配線にこれまで使われてきたアルミニウム
に替えて銅を使う銅配線技術である．しかし，銅は半導体を作る際
にシリコン上に拡散してしまうため，それを防ぐための技術が開発
されている．
集積度が向上することにより，多くのトランジスタが搭載でき，
チップに多くの機能を組込むことができるようになる．その代表的
