# 13・13・9 応力塗料 ，めっき法 ，再結晶法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・13 ひずみおよび応力の測定

応力解析に用いる特殊な塗料を応力塗料というが，これは塗料樹
脂が硬化後に非常にもろくなる脆性塗料（brittle coating）であ
り，容易にき裂を生じるという性質を利用する．応力塗料を試験体
の表面に塗布して乾燥・硬化させた後に，負荷試験を行うと，ある
応力値を超えた試料表面上の塗料膜にき裂が生じる．このき裂の方
向は試料表面の主応力線の方向と一致し，き裂間隔と応力値の間に
も対応性が見られる．応力塗料は，形状・寸法および材質にかかわ
らず，実物に適用することができ，作用応力の高い領域や，主応力
の方向と大きさを容易に知ることができる．
銅めっきした試験片に繰返し荷重をかけると，めっき層に組織変
化が起こり，めっき層表面に目視で確認できる成長粒子または成長
粒子内のすべり帯（微細な黒褐色のはん点）が生じて変色する．こ
のはん点を利用して繰返し荷重による局所的（20m程度の範囲）
な応力（ひずみ）を測定するものがめっき法または銅めっき法
（copper electroplating method）である．この方法には，ある繰
返し数で，はん点を生じる限界ひずみを利用するものと，繰返され
た最大せん断ひずみ振幅とめっき層の成長結晶粒径との関係を利用
するものとがある．銅めっき法はめっき層の再結晶温度である
80℃以下でしか用いられないが，ニッケルめっきあるいは鉄めっき
を用いると200～250℃近辺まで測定可能となる．機械・構造物の
表面における全視野にわたる応力測定が可能であり，稼働状態にあ
る機械部品の応力測定が行える．
多くの金属材料は再結晶現象（recrystallizing）を示す．この現
象は，再結晶熱処理の温度，時間およびあらかじめ材料に蓄えられ
ている内部エネルギー（転位密度，あるいは塑性ひずみ）に強く依
存し，特にある再結晶処理条件下では，得られる再結晶粒径はあら
かじめ付加した塑性ひずみの大きさに依存する．この再結晶現象を
用いて塑性ひずみを測定する方法が再結晶法である．特に，き裂先
端の鈍化に伴い形成される高ひずみ域のひずみ測定，およびそれに
基づく破壊靱性評価法として用いられている．めっき法が材料表面
の計測に限られるのに対して，再結晶法では材料を薄くスライスす
るなどの方法により内部の状態を評価することができる．
