# 1・1・4 入出力装置と入出力インタフェース

> **章**: 第1章 情報システム
> **節**: 1・9 )。共有メモリアーキテクチャはCPU がメモリ上のデータに簡

ハードディスクドライブ(HDD : hard disk drive)は、二次記憶装置として用いられる外部記憶装置であり、大容量で高速なアクセスが可能である。磁性体を塗った高速の円盤にデータを読み書きするもので、ランダムにアクセスできる。1990年代初頭で数十メガバイトであった容量は現在、数十ギガバイトとなり、年々大容量化している。データを保存する装置としては、従来、フロッピーディスクや磁気テープドライブなどが使われていたが、最近ではCD(compact disk,compact disc)やフラッシュメモリ、DVD(dig-ital versatile disk)などが使われるようになってきた。人間とのインタフェースとなる外部入出力装置としては、キーボードやマウス、ディスプレイがあるが、スキャナやタッチパッド、ジョイスティックなどを含めて多様化している。インタネットの発達に伴い、ネットワーク装置は最も重要な入出力装置の一つになっている。従来はさまざまなネットワークがあったが、現在ではイーサネット(Ethernet)が標準的に使われるようになっている。1990年代に10Mbps(bit per second,1秒間で転送するビット数)だった伝送速度は100Mbpsと高速化し、1Gbpsの転送速度をもつギガビットイーサネットが普及しつつある。CPU と外部の入出力装置でデータのやり取りを行う部分が入出力インタフェースである。CPU 自体は外部とのデータのやり取りにバスを通してデータをやり取りするが、入出力インタフェースは外部装置に合わせてバスの信号を変換する。CPU が入出力インタフェースとの間でデータをやり取りする場合には入出力命令を用いる。入出力装置からメモリにデータを読込む場合、入出力命令を用いてデータをCPU に読込み、これをメモリに書き込むという動作を繰返す。しかし、この場合、入出力のためにCPU が介在するために、データ量が多くなるとCPU が占有されてしまうという問題点がある。そのため、大量のデータを入出力装置とメモリの間でやり取りする場合にはDMA(direct mem-ory access)という方法を用いる。DMA はCPU を介さずに入出力インタフェースとメモリ間でデータ転送を行うもので、これを制御する回路をDMA コントローラという。CPU は入出力命令で7―5DMA コントローラに外部インタフェースと転送先のメモリのアドレス、データ量などをセットすることにより、DMA コントローラに自動的に転送を行わせることができる。外部入出力装置のインタフェースにはさまざまなものがあるが、大きく分けて、シリアルインタフェースとパラレルインタフェースの二つがある。シリアルインタフェースとは、1本の信号線で1ビットずつデータを伝送するものである。パーソナルコンピュータでモデムなどの接続に使われるRS-232Cは最も普及しているシリアルインタフェースで、最大のデータ転送速度は115.2kbpsである。最近では、さらに高速なUSB(universal serial bus)やIEEE1394(FireWire)が使われるようになってきた。当初、キーボードやマウスなどの比較的低速の機器を接続することを想定してUSB 1.1(最大データ転送速度12Mbps)が策定された。USB 1.1が広く普及したため、CD やHDD も接続されることが多くなり、高速データ転送の需要が高まってUSB 2.0(最高データ転送速度480Mbps)が策定された。これにより、二次記憶装置を接続しても十分な性能を発揮できるようになっている。なお、USB 2.0はUSB 1.1の上位互換が確保されており、USB 1.1の機器もそのまま接続できる。IEEE1394は高速なシリアル転送を実現する規格で、最大400Mbpsまでの規格が決められている。機器を数珠つなぎにする、いわゆるディジチェーン接続することが可能で、複数の機器を同時に接続することができる。この規格はディジタルビデオの外部接続に採用されている。このほかに、シリアルインタフェースとしては、赤外線で外部機器と接続するIrDA(Infrared Data Association)がある。パラレルインタフェースは、複数のデータ線で情報を一度に転送する方式である。二次記憶装置として使われるディスク装置など大量のデータの転送に使われる。PCなどで内蔵のディスクドライブの接続に使われているのがIDE(integrated drive electronics)インタフェースである。製造コストが安く、高速であるために広く普及している。IDEを正式に標準化したものがATA 規格である。SCSI(small computer system interface)はPCと周辺機器を接続するためのパラレルインタフェースの一つである。当初のSCSIは、8本の信号線で8ビットデータ転送を行い最大転送速度は5MB(メガバイト)/sであったが、その後、転送速度を向上させてfast SCSIでは10MB/s、さらにultra SCSIでは20MB/sの転送速度が可能になった。同時に、8ビットのデータ幅を16ビット幅(wide SCSIと呼ぶ)にすることによって、さらに高速化を図っている。ultra wide SCSIでは、40MB/sの転送速度が可能である。PCI(peripheral components interconnect)は、インテル社を中心とした団体で策定されたインタフェースの規格で、多くのPCで使われている。データ転送幅は32ビットで、転送周波数は33MHzのため、データ転送速度は133MB/sである。複数の外部HDD がある場合などはデータ転送速度が足りなくなりつつある。そのため、さらにサーバなどに用途に高速化した64ビット、周波数が66MHz のPCI規格では、最大533MB/sの転送速度となる。その次の規格であるPCI-X では、転送周波数を133MHz,64ビット幅では最大1066MB/sのデータ転送速度になる。PCI-X ではデータ転送を速くするだけでなく、複数の周辺機器からのデータ転送が行われたときに効率的にインタフェースが使われるように工夫されている。グラフィック表示のためのチップとCPU の間のインタフェースとして使われているのが、AGP(accelerated graphics port)である。三次元グラフィック処理など高度のグラフィック処理では、メモリからグラフィック装置に高速なデータ転送が必要となるため、専用のインタフェースが開発された。バス幅は32ビットで、転送周波数66MHzで動作するデータ転送速度266MB/sのモードと、転送周波数133MHz で533MB/sの2倍速転送モードの27―6情報・ソフトウェア種類が規定されている。さらに、次世代の入出力インタフェースとしてPCI Expressが発表されている。PCI Expressは、グラフィックスチップの接続に利用されているAGP、従来のPCIなどを置換えるものとなる。特徴としては、シリアル伝送を採用していることで、一方向当たり2本の信号線で構成され、双方向のデータ転送には最小構成(1ビットシリアル転送)で4本の信号線を用いる。この最小構成を1レーンと呼び、必要に応じて複数のレーンを束ねることで、求められる帯域幅の提供を可能にする。1レーン当たり転送周波数は3GHz、データ転送速度は2.5Gbpsが見込まれている。このほか、ノート型PCに装着するPCカードのためのインタフェースの規格としてPCMCIA(Personal Computer MemoryCard International Association)がある。
