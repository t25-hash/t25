# 13・14・1 材料物性評価試験（表13･6）

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・14 微小材料の試験

a.引張試験 微小な変位・荷重を計測・制御することが可能
な試験機を用いて引張試験を実施し，得られた応力-ひずみ曲線か
ら縦弾性係数，ポアソン比ならびに降伏応力，破壊応力を求める．
直接的な機械的特性試験として重要であるが，微小な試験片の壊れ
やすさや，試験機への試験片設置時の精密なアライメント設定など
試験片の取扱いの困難さ，試験片形状特に厚さの正確な計測および
微小押込み試験 弾性係数，硬さ
表13･6 材料物性評価試験
方法 評価対象
引張試験 弾性係数，降伏応力，破壊応力
曲げ試験 弾性係数，降伏応力，破壊応力
共振試験 弾性係数
バルジ試験 弾性係数，内部応力，付着強度
超音波音速測定 弾性係数
微小試験片に生じるひずみ計測の困難さを有している．これらの問
題を解決すべく試験機の駆動系，変位計測系，試験片取付方法につ
いて種々の方法が提案され，改良が続けられている．一方，試験片
のまわりに引張りの機構を作り込んだチップによる試験方法なども
開発されている ．
b.曲げ試験 異方性エッチングや犠牲層を用いたエッチング
により作製した柱状の試験片に曲げ荷重を作用させ，荷重-変位関
係から弾性係数や降伏応力，破壊応力を求める．二つの支点上に試
験片を設置してその中央に荷重をかける三点曲げ試験と，同様に片
持はりを形成し，端部に微小荷重を与える片持はり曲げ試験があ
る．後者は1回のエッチングで多くの試験片を作製可能であるが，
固定部における力学的な境界条件が不明りょうという問題を有す
る．いずれの場合についても，基板上に薄膜を形成し，これを組合
せはりとして試験することにより，薄膜材料の評価を行うことも可
能である．
c .共振試験 片持はりや両端固定はりとした試験片に，静電
引力，レーザ光照射による熱ひずみや空気圧変化を利用して加振を
行い，共振周波数を計測する．はりの共振周波数についての解析解
を利用して弾性係数を求める．組合せはりとして試験し，基板上に
形成した薄膜材料の評価も可能である．静的曲げ試験と比較して，
正確な計測が困難な試験片厚さに依存しにくく，また非接触な測定
が可能である一方，はりの固定部における力学的境界条件が不明り
ょうであり，また雰囲気による振動減衰の影響が大きい場合には真
空中での試験が必要である．
d.バルジ試験（bulge test ) 円形，正方形や長方形の穴のあ
いた基板上の薄膜材料をダイヤフラムとして，その片側に流体によ
り一定の圧力を与え，得られた膜の膨らみ（バルジ高さ）と圧力の
関係から縦弾性係数や内部応力，さらにはポアソン比を求めること
ができる．また，膜が破裂する際の強度や基板とはく離する際の付
着強度を評価することも行われている．
e.超音波音速測定 水晶振動子に独立薄膜の試験片を垂直に
設置し，超音波パルスが試験片中を伝ぱし，端部より反射して戻っ
てくる時間から音速を計測する（パルス反射法）．縦波および横波
の伝ぱ速度から，縦弾性係数およびせん断弾性係数を独立に求める
ことができる．また，表面波を利用して基板上の薄膜の弾性係数を
計測することも行われている．
f .微小押込み試験 微小圧子を従来の硬さ試験よりも小さな
荷重で材料に押込み，負荷時と除荷時の荷重-変位の関係を計測し
て硬さ，弾性係数を求める．特に微小なものとしては，原子間力顕
微鏡をもとにした押込み変位量nm，押込み荷重サブ Nオーダの
ナノインデンテーション（nanoindentation）装置が開発されてい
る ．薄膜材料の評価に広く用いられているが，押込み時の基板
の影響を除くため，押込み深さは膜厚に対して十分小さくなければ
ならない．
