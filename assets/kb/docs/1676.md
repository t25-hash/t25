# 13・14・2 内部応力評価試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・14 微小材料の試験

a.基板の反り測定 基板上に薄膜を形成すると，薄膜の内部
応力により基板に曲げ変形が生じる．このときの基板の曲率を計測
することにより，薄膜材料の内部応力を求める．片持はり上に薄膜
を成膜し，これによる組合せはりの反り変形から評価することも行
われている．
b.ラマン分光法（Raman spectroscopy) 応力作用下での結
晶格子振動の周波数変化を利用した応力測定法である．レーザ照射
によるラマン散乱光の周波数ピークの移動を計測するが，顕微光学
系を用いることによりレーザスポット径を mオーダまで絞ること
ができるため，微小領域の内部応力の評価にしばしば用いられる．
高空間分解能，非接触評価という特徴に加えてin- situ評価が容易
に可能であるが，一方，測定対象が結晶性の材料に限られ，また金
属材料についての計測ができない．
c.X線回折法（X- ray diﬀractometry) 応力作用下の結晶
格子面間隔の変化に起因したX線回折角の変化を利用した応力計
測法である．多結晶薄膜材料に関しては通常の材料と同様に試料表
面に対して傾斜角ψを有する格子面での回折角θを計測し，2θと
sinψの線形関係を得て，この傾きから応力を求める（sinψ法）
が，非常に薄い膜を対象とする場合はX線の入射を低角度とし，
回折角を直接評価する方法も用いられる．非接触での応力評価が可
能である一方，測定対象が結晶材料に限られ，またX線照射径を
絞ることができないためラマン分光法に比し，空間分解能が劣る．
d.基板分離後の薄膜変形計測法 薄膜と基板を分離し，内部
応力の解放後の変形を利用して内部応力を評価するものである．測
定する変形量が小さいため一般に精度は悪い．次の2種類の変形を
計測する方法がある．
.伸縮計測 片持はり，T型やH型に形成された試験片
の基板分離後の長さ変化を計測することにより内部応力を評価す
る．
.座屈計測 犠牲層エッチングにより種々の長さの両端固
定はりを同一の薄膜状材料に形成する．圧縮の内部応力が存在する
場合は特定の長さ以上のはりで座屈が発生するので，この臨界長さ
を求め，座屈発生の臨界条件から内部応力を評価する．
