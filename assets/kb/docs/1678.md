# 13・2・1 疲労試験機

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・2 疲労試験

疲労試験機（fatigue testing machine）は負荷荷重形式，荷重発
生機構の違いにより多種多様の形式のものがある．荷重形式により
大別すると，(ⅰ)引張圧縮疲労試験機，(ⅱ)回転曲げ疲労試験機，
(ⅲ)平面曲げ疲労試験機，(ⅳ)ねじり疲労試験機，(ⅴ)組合せ応力
疲労試験機などとなる．試験片に繰返し荷重を与える駆動方式によ
って分類すると，(ⅰ)クランク機構式または偏心機構式，(ⅱ)試験
片または荷重の回転式，(ⅲ)遠心方式，(ⅳ)電磁式，(ⅴ)油圧式，
(ⅵ)圧縮空気式などがある．また，試験片を含む試験機の振動系の
共振点との関係から分類すると，(ⅰ)非共振型，(ⅱ)準共振型，
(ⅲ)共振型に分類できる．さらに，負荷荷重を任意のプログラムに
従って変動させるプログラム疲労試験機，あるいは荷重を衝撃的に
与える衝撃疲労試験機などがある．
引張圧縮疲労試験は，応力が試験片断面にわたり一様であるの
で，基本的な疲労強度の特性を調べることができるものとして，多
くの種類の試験機が考案され使用されている．しかしながら，試験
片の試験機への取付けにおいて軸合せが悪いと曲げ応力がかかるお
それがあるので，細心の注意が必要である．荷重負荷様式としては
電動機駆動によるクランク式，電磁式および油圧式などが代表的な
ものとして広く用いられているが，最近は適用分野が広く，高荷重
の負荷ができ，かつ精度の高い制御が可能なことより，閉ループ式
の電気油圧サーボ疲労試験機（electro- hydraulic servo fatigue
testing machine）が多く使われている．これは図13･10に示すよ

ユニットにフィードバックして，信号発生器からの基準信号とフィ

loop）を構成する．この制御信号を用いてサーボバルブによりア

パーソナルコンピュータを用いて負荷波形信号の発生からデータの

回転曲げ疲労試験機（rotary bending fatigue testing machine）
は，丸棒に静的な曲げ応力を負荷しながら試験片を電動機で回転さ
せ，繰返し応力を加えるものである．曲げモーメントの加え方とし
ては片持曲げと四点曲げで均一な曲げモーメントを加える方式（図
13･11）がある．前者はベーラー（Wohler ）によって最初に用いら
れたのでベーラー型ともいわれ，構造が簡単であるので現在も広く
用いられている．後者の代表例としてわが国では小野式回転曲げ疲
労試験機が広く用いられている．回転曲げ疲労試験機は構造上から
両振り応力で正弦波の波形しか負荷できないが，おもりで負荷する
ので精度や長期安定性に優れている．標準の試験片寸法は直径
6～12mm程度のものが多く用いられているが，直径250mm程度
の大型試験片による試験も行われている．
平面曲げ疲労試験機（plane bending fatigue testing machine）
は同じ曲げ負荷であるが，曲げモーメントの軸が試験片に対して一
方向に限られており，回転しない場合であって，回転曲げの場合と
異なり平均応力を与えて種々の応力比の試験が実施可能である．通
常は平板状試験片の面外曲げ試験が行われる．試験機駆動方式とし
てはクランク式，電磁式あるいは電気式など各種のものが使われて
いる．図13･12はクランク式（西原式板曲げ疲労試験機）であり，
曲げモーメントの大きさははずみ車の慣性力により求めるが，この
代わりにねじり応力棒によりモーメントを測定するようにしたシェ
ンク式曲げ（ねじり）疲労試験機がある．一般的にいって荷重精度
はあまりよくないので，動荷重の検定は定常的に行う必要がある．
