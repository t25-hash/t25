# 13・2・2 疲労強度試験と疲労限度の決定法

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・2 疲労試験

疲労強度（fatigue strength）は通常10本程度の試験片を用い，
それぞれに異なった大きさの繰返し応力を与え，応力振幅と破断繰
返し数の関係から - 曲線（10･2･2項参照）を描いて求める．繰
返し数10回以上の高サイクル疲労寿命の標準的な試験に関しては
JIS規格 があり，用語，試験片，試験方法，疲労強度の決定方
法などが定められている．また，ISO やASTM にも同様の
規格が設けられている．疲労強度は静的強度と異なり，試験片の形
状寸法，表面性状などによって大きな影響を受けるので，試験片の
作製には十分な注意が必要である．回転曲げおよび平面曲げの小型
の標準試験片については上記JIS規格によって形状寸法が定めら
れている．
JISによる疲労強度（時間強度）の決定法は， - 曲線を試験
結果を表す各点のほぼ中央に引き，指定された繰返し数に対応する
応力振幅を - 曲線上から求め，時間強度と定める（ただし， -
曲線の外挿は許されない）．疲労限度（fatigue limit ，endur-
ance limit ）の決定法は，(１)その応力段階で未破断の試験片が半
数以上となる最大応力，あるいは(２)上記(１)で求められた応力段
階での試験片がすべて未破断の場合は，その応力段階と一つ上の応
力段階の平均の応力とすると規定している．
疲労強度は一般にばらつきを示すものであり，上記JISの規定
は疲労強度をばらつきの中央値（破壊確率の50 %）として定める
ものであるが，疲労強度を統計的にとらえ，破壊確率をパラメータ
とした -- 曲線を描くことも行われる．統計的な性質を考え
た疲労試験方法としては，日本機械学会基準S002統計的疲労試験
方法 がある．時間強度や疲労限度についての平均値や標準偏差
を統計的に求めるには，通常，プロビット法（probit method）や
ステアケース法（staircase method，up and down method）が用
いられるが，前者は多数の試験片本数を必要とするので，疲労限度
の平均値や標準偏差を求めるには後者が用いられることが多い．
プロビット法は疲労限度の平均値の予想値 を中心として一定
階差 のいくつかの試験応力レベルを設定する．各試験応力で複
数個の試験片を疲労試験し，所定の打切り繰返し数までに破断する
確率 を求める．次に，疲労限度が正規分布に従うという前提で，
各試験応力での破壊確率を正規確率紙にプロットして直線を当ては
める．このとき，＝0. 5に対応する応力が疲労限度の平均値
を与える．また，＝0 .841に対応する応力と との差が疲労限
度の標準偏差σを与える．
