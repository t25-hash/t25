# 13・3・1 クリープおよびクリープ破断試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・3 クリープ試験

金属材料のクリープ試験（creep test ）として最も一般的なもの
は，一定温度・一定荷重下の引張クリープ試験（tensile creep
test ）である．この試験の主目的は，クリープひずみを測定し，ク
リープ速度を求めることである．試験機はフレーム構造に，てこと
おもりを用いた荷重装置，抵抗線などによる電気炉，ダイヤルゲー
ジなどを具備した伸び計，それに試験片の温度測定装置と温度制御
装置からなっている．試験結果にばらつきをもたらす試験要因は荷
重精度および温度の許容範囲である．これらは，金属材料のクリー
プおよびクリープ破断試験方法として規格化されている ．その
内容を表13･1に示す．
荷重精度は試験機のてこ比（1：10ないしは1：20）の正確さに
よるが，軸心のずれや試験片ねじ部の片当たりに起因する曲げ応力
の作用に注意を要する．これは特に伸びの小さな材料の低応力試験
や切欠材の試験において問題となる．試験温度の誤差要因としては
温度制御装置および測定装置によるものがあるが，高温，長時間試
験では熱電対の劣化による測定誤差に注意する必要がある．また低
温ではクリープ速度の応力依存性が小さいが，高温になるほど温度
依存性が大きくなるから，試験片の温度分布や温度の経年的変化に
留意しなければならない．
クリープ破断試験（creep rupture test ）についても，最も広く
行われているのは定荷重引張クリープ試験である．金属材料に関す
る試験方法は規格化されており ，前記のクリープ試験方法とほ
とんど相違はないが，中断試験を許すこと，実際の構造部材からの
採取を考慮して4mm径の試験片も含むこと，荷重精度は±1 %以
内であること，伸び測定の規格がないこと，が異なる点である．伸
びを測定する場合にはクリープ試験機によって破断試験を行うが，
特に長時間のクリープ破断時間を経済的に求めるために，試験片が
6連3本づりになっている複式試験機が広く用いられている．
このほか，主として基礎的研究のために行われるクリープおよび
クリープ破断試験として，定応力，変動応力，圧縮，曲げ，ねじ
り，引張り-ねじり組合せ，内圧円筒，き裂進展，真空中あるいは
特殊雰囲気中などの試験がある．
