# 13・3・2 応力リラクゼーション試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・3 クリープ試験

応力リラクゼーション（stress relaxation，応力緩和）とは，初
期の全ひずみを一定に保ったとき，時間とともに応力が減少する現
象をいう．高温下で用いられるボルトの締付力，焼ばめ力，ばね
力，溶接部などの残留応力除去焼なまし，運転開始時に発生する熱
応力の運転中の低下，PCコンクリートにおけるPC鋼材の室温付
近での緊張力の低下などがその代表例である．したがって，リラク
ゼーション試験は時々刻々の応力低下を測定するのが目的である．
最も一般的な試験は引張リラクゼーション試験（tensile relaxa-
tion test ）であり，その試験法はJISに規格化されている ．クリ
ープ試験方法と異なるおもな点は，試験片の標点距離が長いこと，
制御するための伸び測定精度が高いこと，全ひずみ保持の許容範囲
は初期値の±1.5 %以内であること，などである．全ひずみ一定に
保持するために自動制御型の試験機 が用いられるが，通常のク
リープ試験機を用いて応力を段階的に減少させるステップダウン試
験と呼ばれる試験法もある．
