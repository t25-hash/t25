# 13・3・3 クリープ支配下の高温低サイクル疲労試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・3 クリープ試験

クリープ支配下の高温低サイクル疲労試験（low cycle fatigue
test at elevated temperature）における破損繰返し数は，材料の
種類，温度，応力あるいはひずみ波形，ひずみ速度あるいは繰返し
速度によって異なり，それは材料内の局所におけるクリープひずみ
の蓄積および破壊の機構と密接に関係している．同時に，それらに
対して雰囲気が大きな影響を及ぼす場合のあることに関心が寄せら
れており ，真空中の高温低サイクル疲労試験なども実施され
ている．
試験法としては，日本材料学会報告 やJIS規格 がある．電
気炉あるいは高周波誘導加熱装置を備えた電気油圧サーボ疲労試験
機が多く用いられている．このほかに，通常のクリープ試験機を用
いておもりを上下させて片振りあるいは部分片振り応力下の疲労試
験を行ったり，偏心レバーによって繰返し変位を与える機械式疲労
試験機を用いたりする場合もある．
ス法等により，き裂開閉口を計測することが望ましい．
