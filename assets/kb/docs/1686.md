# 1・4・3 無線通信技術

> **章**: 第1章 情報システム
> **節**: 1・4 ネットワーク

初期の無線通信は、広範囲にわたるサービスエリアを少数の基地局で対応するシステムがおもであった。これに対して、多数の基地7―21局を用意し、サービスエリアを複数のセル(cell)と呼ばれるエリアに分割するセルラシステム(cellular system)が登場した。このシステムでは、出力信号のレベルが低く済むため、通信端末が使用する電力は小さい。このため、大型のバッテリが不要となり、通信端末自体の小型化・軽量化が可能となる。本項では、おもにセルラシステムによる無線通信方式について概説する。ところで、無線通信の方式として人工衛星を用いる方式もあるので簡単に触れておく。人工衛星は、地上約36000kmの地点で地球の自転速度と同じ速度で周回する静止衛星と、これよりも低い軌道上で地球の自転速度よりも遅い速度で周回する低軌道衛星に分類される。いずれの場合も、地上の通信端末は信号を人工衛星に送る。人工衛星はこれを増幅した後、地上にいる別の通信端末に転送する。ここで、後者の衛星を用いた通信では、前者のそれと比べて、衛星との通信の信号レベルを低くすることができ、また、信号の伝搬遅延時間が小さくて済む。その反面、一つの衛星がカバーする通信範囲が静止衛星と比べて狭く、さらに、速度が遅いために衛星が地球の陰に隠れることがある。このため、多数の衛星が必要となるほか、通信端末は衛星がどこにいるかをつねに追跡する必要がある。a. セルラシステムセルラシステムによる無線通信方式は、1970年代後半から1990年代前半にかけてアナログ方式がおもに用いられた。これは第一世代と呼ばれ、おもに音声中心の通信を目的に用いられた。1990年代前半以降にディジタル方式が登場し、音声のほか、64kbps以下の低速なデータ通信が行われるようになった。これは第二世代と呼ばれる。また、より高速なデータ通信を行うため、第三世代の通信システムであるIMT-2000(interna-tional mobile telecommunications 2000)が1997年に標準化されている。セルラシステムにおけるセルとは、基地局と通信端末が通信可能な範囲を示す。その形状は基地局を中心とした円となることが多く、図1・27に示すようにセルを配置することで、サービスエリア全体をカバーする。各基地局は交換局と有線で接続されており、この交換局を経由して、異なるセルにいる通信端末や、他のネットワーク網との接続が行われる。以下では、セルラシステムにおける通信技術についていくつか述べる。. チャネル利用方式無線通信方式では、限られた周波数帯域を用いて通信が行われる。このため、この周波数帯域を効率的に使用し、収容可能な通信端末数を高めることが求められる。その方法の一つに多重化方式があり、利用可能な周波数帯域をさらに細かい周波数帯域で分割するFDMA(frequency division multipleaccess)方式と、周波数帯域を短い時間(スロット)単位で区切る時分割多重TDMA(time division multiple access)方式がある。前者は1人のユーザが分割された周波数帯域を占有するのに対し、後者は各ユーザは定期的に利用可能なスロットが割当てられる。また、FDMA で分割された周波数帯域をTDMA でスロットに分割することも可能である。これに対して、すべてのユーザからの通信を広範囲な周波数帯域図1・27 セルラシステム7―22情報・ソフトウェアに拡散し、これらを同一周波数帯域で送信するが、各通信端末ごとに別々な符号を用いることでこれらを区別するCDMA(code divi-sion multiple access)方式があり、近年は、この方式がよく利用されている。. 周波数の再利用セルラシステムでは、低出力で信号を送信することから、信号が届く範囲が限られる。このため、複数のセルで送信に用いる周波数を再利用することが可能である。例えば、図1・27のセルA では、セルB またはセルCと同じ周波数で通信することができる。このような特質から、セルの大きさを小さくすることで、サービスエリア当たりの収容人数を増やすことができる。また、セルの大きさはサービスエリア全体で一定である必要はない。ユーザが集中する地点のセルサイズを小さくして収容人数を増加するとともに、ユーザが少ない地点のセルサイズを大きくすることで、基地局の数を抑制することも可能である。このとき、通信端末は自分が通信している地点のセルの大きさに応じて、出力信号のレベルを変える必要がある。. マルチパスフェージングセル内にビルや木などの遮へい物がある場合、通信端末に基地局から届く本来の信号のほかに、この本来の信号が遮へい物に反射してやや遅れて届く信号を受取る。このように、同じ信号がマルチパス(multipath)で到達すると、これらの信号は互いに干渉する。このため、基地局と通信端末の距離によって、到達する信号レベルが急激に変動するマルチパスフェージング(multipath fading)が生じる。これを改善するために、基地局にアンテナを2本用意し、出力信号の1/4波長分ずらした位置に設置して、良いほうの信号を選択するなどの対策が必要となる。. ローミングとハンドオフ通信中に通信端末が、現在いるセルから別のセルに移動したとする。この動作をローミング(roaming)と呼び、このとき、接続する基地局を切換えたり、出力信号のレベルを変更するなどの処理が必要となる。この処理をハンドオフ(hand off)と呼ぶ。ハンドオフの処理時間が長いと通信が一時的に切断されるため、切換え操作の高速化が要求される。b. セキュリティとプライバシー保護基地局と通信端末間の無線通信を第三者が傍受することは容易に可能である。このため、さまざまなセキュリティ技術を用い、第三者に通信内容を漏えいさせないことで、プライバシーを保護することが重要な課題となる。以下では、無線通信技術のためのセキュリティ技術について述べる。. 周波数ホッピング方式周波数ホッピング(frequencyhopping)方式とは、通信で用いる周波数帯域を送受信間で合意したパターンで切換えながら通信を行う方式である。この切換え周期を十分短くすることで、第三者が通信内容を完全に傍受することが妨げられる。. 直接スペクトラム拡散方式直接スペクトラム拡散(direct sequence spread spectrum)方式では、送信信号に受信者と合意した、ノイズのような性質をもつ信号を加算する。これにより、信号の周波数帯域は拡散され、ノイズのような信号になるため、を知らない第三者はを復元することができない。. 通信データの暗号化通信データ自体を暗号化することでもプライバシーを保護することが可能である。暗号化方式としては、おもに次の2種類がある。まず、送受信者双方で同一の鍵を用いる秘密鍵暗号(private key cryptosystem)方式があり、その代表例としてDES(data encryption standard)がある。一方、暗号化を行う鍵と復号を行う鍵を別々に用意し、前者のみを公開する公開鍵暗号(public key cryptosystem)方式もあり、例としてRSAがある。秘密鍵暗号方式は暗号化・復号の動作が高速である反面、通信相手が多くなるに従い鍵管理が困難になるという問題がある。その逆に、公開鍵暗号方式は鍵管理は比較的容易だが、暗号化・復号に時間がかかる。このため、通信では秘密鍵暗号方式を用いるが、その鍵は通信開始時にランダムに生成し、これを公開鍵暗号方式を用いて送受信間で共有するという手順を取るのが典型的である。
