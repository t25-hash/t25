# 2・1・1 センサ

> **章**: 第2章 メカトロニクス
> **節**: 2・1 ハードウェア

ロボットをはじめ,各種メカトロニクスシステムが自律的に作業を実行するためには,システム自身の状態,もしくは状況によっては作業対象物や周辺の環境の状態を把握する必要がある.そのために用いられるのがセンサである.例えば,モータの回転数を測定するためのタコジェネレータ(角速度センサ)や作業対象物に触れていることを検出するための接触センサ,作業の遂行により対象物が正しくセットされているか確認するためのCCD カメラ(視覚センサ)など,さまざまなセンサが用いられている.システムの機能が高度になるほど,内部状態や外部の環境・対象物の状態を知る必要(a) 完全オープン型メカトロニクスシステム(b) 外界オープン型メカトロニクスシステム(c) 外界フィードバック型メカトロニクスシステム図2・1メカトロニクスシステムとセンサアクチュエータが高まり,多数のセンサを備えることとなる.a. 内界センサと外界センサメカトロニクスシステムにおけるセンサの用途には,大きく二つの種類がある.一つはそのシステム自身の状態を検出し,目標の状態と一致しているか確認するための内界センサである.内界センサには関節の回転角度を検出するロータリエンコーダやポテンショメータ,動作範囲の限界に達したことを検出するリミットスイッチなどがある.もう一つの用途は作業の対象物や周辺の環境などの監視である.このような用途に用いるセンサを外界センサと呼ぶ.外界センサには,対象物や障害物の有無を調べるための赤外線近接覚センサや超音波センサ,作業遂行により対象物が正しくセットされているか確認するための視覚センサ,周囲の温度を監視する温度センサなど,用途に応じさまざまなものが用いられる.特に高度なシステムにおいては外界センサの重要性は高く,多点の情報を一度にとれるものやON/OFF情報でなく細かな分解能でとれるものが要求される場合が多い.b. 接触センサと非接触センサセンサの検出方式として,接触検出型のセンサと非接触検出型のものがある.接触センサとしては,機械的に押されたことを検出するリミットスイッチや,印加されている力・モーメントを検出する力覚センサなどがある.非接触型のものとしては,CCD カメラ,超音波の伝ぱ時間で距離を計測する超音波センサなどがある.例えば同じ動作範囲の限界を検出する用途でも,リミットスイッチのように機械的な接触センサを用いる方法と,光路が遮られることにより物体を検出する光電スイッチのような非接触センサを用いる方法があり,さまざまな条件から適切なセンサを選ぶ必要がある.c. 能動センサと受動センサ同様にセンサの計測原理として,能動型のセンサと受動型のものがある.センサが測定しやすくなるように,光や超音波,赤外線などを照射するものが能動センサである.一方,そのような能動的なものは用いず,センサへの入力のみをもとに計測を行うものが受動センサである.典型的な例としては,距離センサがある.スポットやスリットのレーザ光を照射し,その像をCCD カメラで計測して距離を求める能動型のものもあれば,離れて置かれた2台のCCD カメラで対象をとらえ,対応点を求めて距離を求める受動型のものもある.d. さまざまなセンサの分類ここでメカトロニクスシステムに多く用いられるさまざまなセンサのうちいくつかを紹介する.ここには上述したようなさまざまなタイプのセンサが含まれている.(1) 角度・位置センサモータや関節,ボールねじなどの回転角度,スライダ機構の位置などを測定する角度センサ,位置センサにはさまざまな種類のものがあるが,代表的なものとしては変位に応じて端子間の抵抗値が変化するポテンショメータ(potentiometer)や,回転軸に取付けた円盤にスリットを設け光を遮ることにより回転角度に応じたパルスを生成するロータリエンコーダ(rotary encoder)がある.ロータリエンコーダはディジタル的なため,減速機とうまく組合わせることにより高精度の測定が可能である.これに対し,ポテンショメータは物理的な抵抗値の直線性(リニアリティ)の問題から精度の面で問題となる場合がある.しかし反面,ロータリエンコーダの多くは回転変位をパルスで生成するため,初期角度がわからないと相対的な変位しか求まらない,という問題点を有する.これに対し,ポテンショメータではつねに絶対的な変位が得られる.(2) 角速度センサモータの角速度を測定するセンサとしてはタコジェネレータ(tachogenerator)がある.これはモータを発電機として用いるのと同様に,軸の回転に応じてコイルを貫く磁束を変化させ,そのコイルに発生する起電力が角速度に比例することを利用したセンサである.一般にモータと一体にして組付けておく.機械や物体の姿勢の変化として角速度を計測するセンサとしてはジャイロスコープ(gyroscope)がある.これは,回転もしくは振動している質量に,その回転と別の方向に角速度が与えられるとコリオリ力が発生することを利用している.このコリオリ力による変位を測定することにより角速度を求めている.機械式のものや圧電素子などを用いた振動式のものなどがある.ただし一般にはドリフト(使用しているうちに角速度ゼロの状態を示す電圧がずれてしまうこと)が起きやすく,これを補正する必要がある.(3) 加速度センサ機械や物体の加速度を計測するセンサとしては,加速度によりその内部にある質量に生じる変位を圧電素子などで電圧に変化させるものなどがある.加速度センサもジャイロスコープ同様,ドリフトに対する補正が必要となる場合が多い.(4) 接触センサ接触を検出するセンサにはさまざまな種類のものがある.最も簡単なものはリミットスイッチのように機械的な1点の接触をとるものである.機械や物体がある位置まで動いてきたことを検出するために,このようなリミットスイッチを用いることが多いが,このほかに接触の力を測定したい場合には圧電素子やひずみゲージ(ストレインゲージ),圧力に応じて抵抗値の変化する導電性ゴムを用いた圧力センサを用いることもある.さらにはストレインゲージを組合わせて,三軸もしくは六軸の力・モーメントを測定する力覚センサなども接触センサの一種である.以上はある1点の接触に関するセンサであったが,このほか,ある領域の中でどの点で接触したかを検出するタイプのものや,導電性ファブリックなどの感圧センサを格子状に配置することにより多点の計測が行える分布型のものなどさまざまなタイプの接触センサがある.用途に応じ,最も適切なものを選び使用する必要がある.(5) 近接覚センサ機械の近傍に物体や環境が近づいたことを検出するセンサであり,その多くは非接触型で能動型のものである.赤外線などを周囲に照射し,その反射を計測することにより近距離の計測を行う.その測定距離としては数mmから十数cm程度まである.またこのほか,機械の周囲にバンパ等を取付け,機械的な接触により近接覚センサを構成することもある.(6) 視覚センサ(visual sensor)CCD カメラなどを用いて二次元の濃淡画像が得られる.この情報からシステムに必要な情報,例えば対象物の位置・姿勢や形状,もしくは周辺の障害物の状況など,さまざまな情報が得られる.一般的な視覚センサの情報量としては,1枚の画像当たり縦横256×256~512×512画素で,各画素当たりRGB(赤緑青の光の3原色)各8ビットで計24ビット,それが毎秒30枚得られるので40~180Mbps(mega bit per second)にも及ぶ.このように非常に多くの情報量をもつことから,その処理には高度なものが要求される場合が多い.いくつかの事例をもとにその使用方法を紹介する.まず最初の事例として,作業対象物の位置・姿勢の計測を取上げる.近年はCCD カメラが安価になってきたことから工場の生産ラインでも多く使われるようになっているが,その最も多い用途が,この対象物の発見である.ただし,いかに視覚センサの情報量が多いといっても,二次元的な情報に置き換えられているため,任意の三次元の位置・姿勢の測定は困難である.そのため一般的には,ある決められた面内(多くは水平面)に置かれた対象物をその真正面(真上)に設置されたCCD カメラから測定する.こうすることにより,視野内の対象物は大きさ・形状が一定になり,そのテンプレートを用意し,計測画像内でテンプレートとマッチする場所を探すことにより容易に位置・姿勢の測定が行われる.また,近年のロボット研究では,移動ロボットの障害物検出に視覚を用いる例が多い.このような場合には環境がある程度特定できている場合を除き,三次元の情報を復元する必要が出てくる.そのため,次の(7)に示すように,2台のCCD カメラを用いて立体視7―27を行うなどの手法が必要となってくる.(7) 距離センサロボットマニピュレータが作業対象物をつかみにいく場合や移動ロボットが周辺環境を調べる場合,距離を測定する必要のあることが多い.このような場合に用いられるのが距離センサ(rangefinder)である.距離センサにも,ある一方向のみの距離を測るもの,ある角度の範囲内で最も近い距離を測るもの,ある角度の領域で多数の方向の距離をマップとして求めるものなど,さまざまな種類がある.一方向の距離を測る方法としては,レーザ光を照射してその反射光を測定する方法がある.この手法は数mの測定レンジに対し,数mm程度の高精度の測定が可能である.ある角度の領域内の最短距離を安価に測定する手法として超音波を用いた方法がある.超音波を照射すると,ある程度の角度で広がり,最も近い物体から反射波が返ってくる.この時間に音速を掛けることで距離を求めることができる.ただし,気温により音速が変化するなど,精度の面ではあまり高くない.多数の方向に対する距離のマップを距離画像(range image,depth map)と呼ぶ.距離画像を得る手法としては,能動的手法,受動的手法,ともに存在するが,いずれの手法でも基本的には三角測量の原理に基づいて計測が行われる.三角測量の原理とは,ある2箇所の計測点から同じ地点を観察した際に,それぞれの方向(角度)および2点間の距離から三角形が構成され,その地点までの距離が三角形の高さとして求められる,という原理である.能動的な手法としては,レーザ光をスリットやスポットとして照射し,その像をCCD カメラで観察することにより各スリットやスポットまでの距離を求める方法である.これに対し,受動的な手法では2台のCCD カメラを用いる.2台のカメラで撮影した画像から,同一地点を見ている部分を発見し,その画素位置から三角測量の原理に基づいて距離を計測する.受動的な手法の場合,同一地点を見ているという対応点を発見するのはかなり困難である.特に計測対象が一様な壁など,濃度的に変化がないような場合には対応をとることはできない.以上,さまざまなセンサについて紹介を行った.センサはその用途等に応じてさまざまな原理,実現方法があり,ケースバイケースで最適なものを検討する必要がある.
