# 2・1・3 制御装置

> **章**: 第2章 メカトロニクス
> **節**: 2・1 ハードウェア

メカトロニクスシステムのための制御装置の役割は、時間やセンサからの情報に応じたアクチュエータの動作を管理することである。a. 専用コントローラとはん用コントローラシステムとして制御する対象が特定されている場合には、そのシステム専用のコントローラを用意することがコスト面等で有利な場合が多い。特に単純なシステムであれば、専用コントローラを開発するコストもそれほどかからないためその傾向は強い。しかし、仕様の変更等に応じてコントローラを変更する場合に開発をやり直す必要がでてくる。またIC,LSIの集積能力の向上により冗長なはん用コントローラを製作してもあまりコストがかからず、むしろ大量に安価に作れることから、近年ははん用的なコントローラを用いることが多い。b. オープンアーキテクチャ近年、はん用性をさらに進め、共通の仕様を定めようという試みが、大規模メカトロニクスシステムであるロボットを中心として行われ始めている。センサとアクチュエータ間のやり取りだけでなく、情報のやり取りを含めて規格化し、開発効率の向上、ロボットモジュールの開発・商品化を目指している。c. さまざまなコントローラ単純なものから複雑なものまでさまざまなシステムを対象としたコントローラがある。その中のいくつかを紹介する。(1) シーケンサ(sequencer)ON/OFFだけで動作するスイッチやモータなどを、リレー等を介して接続し、望んだ手順で動作するように構成したもの。専用コントローラとして構成される。(2) プログラマブルコントローラシーケンサをはん用化したもの。多数の入力端子と出力端子をもち、動作対応をプログラムとして登録することにより動作させる。7―29さまざまな種類のものが市販されている。(3) PIC・ワンチップマイコンディジタル回路としてある程度プログラマブルなIC,LSIを用いてコントローラを構成する。組込み機器などに多く用いられる。(4) シングルボードコンピュータはん用コンピュータであるが、機能を絞った小型・安価な、もしくは信頼性を高めたコントローラとして使用される。(5) はん用PC一般に市販されているPC(パーソナルコンピュータ)にディジタルI/OやアナログI/Oボードを搭載し、コントローラを構成する。近年は非常に安価に高性能のものが入手できるため、多く用いられてきている。
