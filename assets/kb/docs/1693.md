# 2・3・1 分散システム

> **章**: 第2章 メカトロニクス
> **節**: 2・3 分散化と協調

a. 固定結合分散システム主としてシステム内に多数のアクチュエータがある構造をもって分散配置され、このアクチュエータ群の協調動作によって革新的な機能の実現を図っている例を示す。以下に述べるように種々のスケールのものがあるが、特にマイクロスケールのシステムについては、半導体プロセスをメカトロニクスに応用したMEMS(micro-electro-mechanical systems)が1990年代に急速に進展し、これによって多数のアクチュエータを配置するシステムが高い実現性をもつようになった。. 大型天文望遠鏡の柔軟主鏡1999年にハワイのマウナケア山頂に建設された日本のすばる天文台(Subaru Observatory)の主鏡は、口径8.2m、厚さ200mmの薄メニスカス鏡であり、裏面に配置された261本のアクチュエータ群によってつねに鏡面の形状が正確な回転双曲面に保たれる。従来型の主鏡では、十分な剛性が実現できるように厚さが設計されてきたが、口径の大型化に図2・6 力制御グラインダ作業ロボット伴いこの方式は限界に至り、主鏡の重量を抑えつつ鏡面の精度を保つために能動支持方式(active support)が考案された。すばる天文台では主鏡の姿勢によって変形の補正に必要となるアクチュエータの発生力を計算しておき、中央の計算機が全体の制御を行う。マウナケア山頂には多数の天文台が設置されているが、近年建設された四つの大口径天文望遠鏡においてはすべて能動支持方式が採用されている。ケック天文台(口径10m,2台併設)は、それぞれ36個の六角形セグメント鏡で主鏡が構成され、隣り合うセグメント鏡間の変位を測定しこれをアクチュエータで一定に保つ。ジェミニ天文台(口径8.1m)は、すばると同様の薄メニスカス鏡方式で120本の油圧アクチュエータによる能動支持方式をとる。. 宇宙用マニピュレータ木村らは宇宙用の合計10個の関節をもつシリアル型のマニピュレータを試作している。各関節は2自由度をもち、手先の位置の制御を適当な2台の関節を駆動することで実現する。冗長構成を生かして、関節が故障した場合でも機能を維持できる構成をとっている。関節ごとに制御部が分散して配置され、動作時にはすべての制御部が駆動する関節の組を選択して関節角を並列的に計算する。計算結果を判断する計算機が置かれ、ここが実際に駆動する関節を選択して指令を出す。. 平面搬送機構平面上の物体の搬送が可能なように、ユニットを二次元配列した機構が開発されている。いずれも各ユニットがその上の物体をあるいは方向に移動させる力を発生し、全体が協調して搬送を実現する。坂尾・冨山らのシステムはユニットを4×4に配列した自動倉庫の機能をもつ機械式のシステムで、それぞれのユニットは等質で判断能力をもつ。ユーザが指定した荷物の情報はユニット間を伝ぱして、各ユニットの自律的な動作により必要な荷物が出口に搬送される。ユニットの故障に対しては自動的に迂回経路を発生させる能力が組込まれている。福田らは生産システムでのワークの搬送を想定して、5×5ユニット配列のシステムを構成した。初期の段階で投入されるパレットにより学習を行い、必要なパレットの流れを生成する戦略ベクトルという場を生成している。藤田らはMEMS により小型の搬送システムを試作した。ユニットにはセンサ、アクチュエータ、通信、論理回路が組込まれており、複数のユニットにまたがって載せられた物体の形状認識、分類などを行いこれらを必要な場所に移送する。アクチュエータとして気流を制御する方式と、バイモルフ型の金属毛の熱変形を利用する絨毛方式を製作している。. DMD(ディジタルマイクロミラーデバイス)テキサス・インスツルメンツ社が開発したMEMS で、微細な可動ミラーが平面上に配置されており、ミラーの角度を2値に制御することで二次元画像表示デバイスを実現している。アルミニウムでコーティングされた数十万個のミラーがCMOS のSRAM 上に集積されており、それぞれがフレキシブルに支持されている。ミラーの角度はメモリセルとの間の静電力により±10°程度に変化させられる。ミラーを支持するヒンジは強靱で10万時間を超える寿命の製品が実用化されている。ミラーのON/OFF駆動時間は10 s程度で、PWM 方式で輝度を表現している。. 乱流制御流体中の翼面や流体が流れる管の内表面に多数のアクチュエータを配置して乱流発生を抑制し、流体抵抗を低減させる研究が行われている。気体の管内流の場合、気体の流速を熱フィルムセンサにより計測し、これによって検出した渦を解消するようにアクチュエータを制御する方式が試みられている。MEMS 技術を用いて500×500 m程度のセンサを製作し、これをアレイ状に配列して渦の検出実験が行われている。アクチュエータとしては直径3mm程度の電磁式のものが作製されており、さらに小型のものが開発されようとしている。流体が液体の場合も研究が行われており、電気動力学的方式で液体を放出するマイクロメカニズムなどが開発されている。実際に渦を解消する技術については7―33まだ研究段階である。b. 可変結合分散システム(reconfigurable distributed system)一つの機械システムを複数のモジュールを結合して構成し、形状や機能を変化させられるような構造としたシステムの例を挙げる。このような構造は状況に応じてさまざまな形態に変化させる(する)ことができるが、まだ研究段階のものがほとんどである。モジュールがすべて同一構造のもの(均質)、異なる構造のもの(非均質)、分離・結合をシステム自身が制御するもの(自律)、外部の操作者が行うもの(非自律)、一次元から三次元システムなどに分かれる。モジュールの物理的な複雑度は大きく2分類される。一つはモジュールが受動的で比較的単純な構造をとるものである。単純な構造を採用する動機は、例えば複雑な構造、材料も元をたどれば分子、原子の組合せに帰着するように、より単純な要素を結合して上位の構造を生成する方法の実現に目標をおく点にある。この種の例は細川、ホワイトサイズ(Whitesides)らの研究が挙げられる。2・3・2項で具体的な手法に触れる。これに対してモジュールを能動的にし、判断機能までを組込む方向もある。まず、結合の自律的な変更を目標としていないものを挙げると、広瀬らのほふく型生物に類似したシステムや小鍜治のフラクタルマシン、サンダーソン(Sanderson)らのテトロボットなどが挙げられる。広瀬の例はモジュールを直列に関節で結合した一次元的な構成である。原理的に長さの拡張・縮小性をもっており、これを変更しても移動の実現が可能な点でユニークである。小鍜治のフラクタルマシン(Fractal Machine)は二次元の可変長リンク構造の均質性、拡張・縮小性を追求したもので、ソフトウェアまで含めて均質なモジュールの結合により移動機能を実現している。サンダーソンは複数のリンクの回転軸をオフセットなしで交差させられる巧妙なピボットを開発し、可変長リンクをこのピボットで結合して多様な運動が可能な三次元構造を構成した。自律的な結合分離までを目標としたものは機構的に独自なものが多い。福田らはユニットが自律的に結合して全体の構造を変更する可能性に注目し、CEBOT(セボット)と呼ぶハードウェアの一連の試作を行い、二次元の結合モジュールを中心に先駆的研究を行った。チリクジャン(Chirikjian)らはリンク機構による六角形のモジュールを開発し、リンクの関節角を制御してモジュール間の結合を変更できることを示した。村田らは二次元のフラクタム(Fractum)、三次元の立方格子を形成するメカニズムなど、自律的に形状を変化させられるシステムの研究を進め、三次元の形状、運動を生成できるMTRAN(エムトラン)を開発した。自律結合・分離型分散システムの例を表2・1に示す。c. 疎結合分散システム(loose connected distributed system)自律型の移動、認識、外部への操作が可能なハードウェア(以下ではロボットと総称する)が複数で協調して動作する場合を考える。自律型のロボットは多種類のものが試作されているが、1台の制御においても各種機能実現のために多くの研究が行われており、複数が動作する状況では相当量の高度な情報処理が実行される必要がある。長田は複数のロボットに期待できる作業を次のように分類している。.
