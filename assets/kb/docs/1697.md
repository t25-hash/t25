# 13・7・4 動的破壊靱性試験

> **章**: 第13章 材料試験および実験応力解析
> **節**: 13・7 破壊靱性試験

動的破壊靱性（dynamic fracture toughness ）としては，(ａ)静
止き裂に動的負荷が加えられた際の進展開始靱性（衝撃破壊靱性，
impact fracture toughness ） ，(ｂ)高速で進展しているき裂の
進展抵抗（動的破壊抵抗） があり，これらは，試験片寸法，試
験温度（特に鋼)，負荷速度［上記(ａ)の場合］，き裂進展速度［上
記(ｂ)の場合］などに依存する．試験に際しては，厳密には動的応
力拡大係数の評価が必要である．ただし，静的な解析で近似する場
合も多い．A533B-1鋼について， のddおよび試験温度
依存性の一例を図13･25に示す ．なお，(ａ)に関しては，これ
までに およびCTODによる実験的な評価もなされている．
ASTM規格のE1820A13 には 試験の負荷速度の上限
dd＝2.75MPamsを超える場合の急速（rapid loading）
試験［ただし，落重や落槌などによる衝撃（impact ）試験や準衝
撃（quasi - impact ）試験とは異なる］の規定がある．また，
ASTM規格のE1221 には短いき裂の進展停止を準静的過程の
近似のもとに で解析する規定がある．ただし，衝撃荷重下での
規格はまだない．
イギリス規格では，BS7448のPart IIIにおいて動的破壊靱性試
験方法の検討が進められている．
