# 13・1・2 励振メカニズムによる分類

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・1 熱・流体と構造系の連成振動の特徴

流れに注目して励振メカニズムを分類 すると，以下のように
なる．
a.外部励振 流れの圧力変動が強制的外乱として振動体を励
振する．ただし振動体の振動が流れに与える影響は無視できる．例
として乱流による構造物の振動，往復動ポンプによる圧力脈動が挙
げられる．
b.不安定励振 流れの不安定性から誘発される交番的流体力
が振動体を励振する．振動体の振動が流れに影響を与えない場合，
すなわち流体力フィードバックがない場合と，流れに影響を与える
場合，すなわち流体力フィードバックがある場合とがある．前者の
例として，流体交番力による構造物や管内流体の強制振動，後者の
例として，渦の放出による流体交番力による構造物の振動（同期共
振）がある．
c.運動励起励振 振動体の振動がフィードバック交番流体力
を誘発し，さらに振動体を励振する．この場合，静的流体力特性に
起因して交番流体力が誘起する場合と，非定常交番流体力が誘起す
る場合の2種類がある．前者の例として，ギャロッピング振動やフ
ラッタがあり，後者の例として，ストールフラッタがある．
