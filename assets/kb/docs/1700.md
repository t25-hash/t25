# 3・1・3 運動生成アルゴリズム

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・1 運動制御

本項では、ロボットアームの先端(ハンド)をある位置・姿勢から指定された位置・姿勢まで移動させたり、指定された経路(目標軌道)に沿って動かす場合のアルゴリズムを考える。ここでは、ハンドは三次元空間内を自由に運動できるものとする。ハンドが他の物体に接触している場合や力(あるいはモーメント)を受ける場合は少し難しくなる。ハンドの軌道を表現する方法として、関節変数を用いる方法とハンドの位置・姿勢変数を用いる方法の二つが考えられる。関節変位を用いる方法は、各アクチュエータの動きはわかりやすいが、実際にハンドがどのような軌道を通るかはわかりにくいことが多い。そのような条件を満たす関数はいろいろ考えられるが、時間の多項式で表すことにすると比較的簡潔な表現ができ、計算量も多くはならない。. 両端で停止の場合条件は次のように初期時刻と終端時刻における位置と速度の4個である。ハンドが、経由点で停止する場合は、式(3・131)(3・132)を繰返し用いて軌道が定まる。また、経由点で停止しないで通過する場合は、経由点での速度がどのように指定されるかで次のように異なる。(2) 経由点での速度が特に指定されていない場合いくつかの定め方が考えられるが、経由点における加速度の連続性あるいは適当な条件を付与し、各区間ごとに三次式で表すことができる。. 三次式以外の定め方(直線と放物線の組合せ)関数の形式として、三次式以外に一次式(直線)が考えられる。ただし、この方法では、各関節変位は直線的に変化するが、ハンドの位置姿勢は直線的に動くとは限らない。また、端点で速度が不連続になる。そこで、滑らかな軌道を生成するために、基本部は直線(一次式)とし、端点の近くは放物線(二次式)とする形が考えられる。このとき、直線部分は定速区間になり、放物線部分は定加速度区間になる。また、速度は、台形速度パターンになる。そして、加速度αの値が大きくなるほど定速区間が長くなる。b. ハンドの位置姿勢変数を用いた軌道生成a項の関節変位ベクトルを用いる方法では、ハンドの軌道が明確ではない。さらに各変数を直線補間してもハンドの姿勢は直線的に動くとは限らないなど、直感的に理解しにくい。そこで、ハンドの位置姿勢ベクトルを用いると、ハンドの動きがわかりやすくなる。ただし、実現不可能な軌道になることもあり、さらに具体的に制御を考える段階では、逆運動学を用いて、対応する関節変数を求める必要がある。変数のとり方はいろいろ考えられるが、a項と同様な方法で、軌道を定められる。位置を表す変数としては、通常、基準の直交座標を用いる。また、姿勢を表す変数としては、オイラーの角、あるいはロール・ピッチ・ヨー角などが用いられるが、ハンドの姿勢は、三軸まわりの回転の合成となり、直感的に理解しにくい。そこで、次の一軸回転法あるいは二軸回転法が用いられる。このとき、図3・12に示すように目標軌道、すなわちハンドは基準座標から見て固定された軸のまわりに回転する。αは、a項で述べた方法で指定できる。
