b.強制外力として物体に作用する例
(１) 容積型あるいはターボ型流体機械で発生する，回転速度や
羽根数などに比例する振動数の励振力，流体の乱れによる不規則圧
力脈動による励振力などで，近似的な取扱いが行われている．
(２) 流出渦列など流体運動の不安定性に起因した圧力変動に伴
う励振力で，カルマン渦による煙突の振動，エッジトーンなどがあ
る．この場合，流体と物体の運動の両者を考慮に入れた数値計算も
一部可能で，本来，流体運動も物体の運動の影響を受ける．しかし
通常は，流体運動の変化は無視できる場合が多い．
c .係数励振力として物体に作用する例 流体管路の場合，
管軸が曲率によって生じる遠心力，管内流の脈動成分による遠心力
の変動などが，管の横振動を励起させることがある．また管の横振
動と管軸方向の流れの相互作用により生じるコリオリ力も管の横振
動の安定性に影響を与える．この場合，流体運動は，一次元流れと
して取扱われ，係数励振領域を求める場合などは，管路の運動に伴
う流体運動の変化は無視してよい．
d.負減衰力として物体に作用する例 失速フラッタや送風機
のサージングのような負抵抗による励振力がある．これらの問題
は，現状では流体励振力を定常な実験結果から評価している．
e.熱および二相流に係わる複雑な励振力として物体に作用する
場合 熱源と気流による管内気柱振動，液面振動の気泡による励
振などがあり，実験的な解明が行われている．
以上に挙げた例の中で，b，c，d，e項の一部は，本来，物体の
運動と流体運動との連成現象が不安定なもので，その固有値の実数
部分が負つまり線形の不安定現象としてとらえるべき問題もある．
さらに振動発生後の状況あるいは外乱の大きさの影響を調べるに
は，非線形安定問題として取扱うべきである．しかし，目下のとこ
ろ解決が困難なため，近似的な取扱いを行っている．
