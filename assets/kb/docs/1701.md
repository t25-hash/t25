# 3・1・4 分散制御(distributed control)

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・1 運動制御

分散制御は、制御の対象が大規模な場合や、それらが地理的に離れている場合、複数個の制御器に制御の機能を分散させる制御方式である。特にロボットの分野では、環境や状況に応じて自身の判断で自律的に行動できる自律ロボット(autonomous robot)の開発が大きな目標であり、複数の自律ロボットが協調・共同してある作業・行動を行うシステムの開発が盛んに行われている。通常、このようなシステムを自律分散制御システムと呼んでおり、個々の自律ロボットは、サブシステムを構成する。また、単体のロボットを一つのシステムと見ると、そのロボットを構成する各要素は一つのサブシステムと考えることができる。システム全体の制御を考える場合、それらを中央集権的に制御するか、各サブシステムにある程度の自由度をもたせて分権的に制御する方法が考えられる。中央集中型制御方式の場合は、統一的な方針を立て、それに基づいて作業計画の配分・相互調整などが容易であり、全体的な効率は高くなるが、個々のロボットの故障が全体の効率の大幅な低下につながり、特に集中管理部分を担当するシステムやコンピュータの故障は、システム全体のダウンに直結する。これに対して、分散型では、個々のロボットに故障が生じた場合でも他のロボットがそれなりに補完的な対応をとることにより、全体の効率に及ぼす影響は比較的少なくて済む。この分散型システムでは、各ロボットがそれぞれ自律的に行動することになるため、その自律性をある程度保証しつつ、全体としての調和のとれた行動をいかにして確保するかが大きな課題となる。図3・11 直線と放物線の組合せによる軌道と速度図3・12 一軸回転法図3・13 二軸回転法7―48情報・ソフトウェアこの自律分散ロボットシステムを構築するためには解決すべき多くの課題があるが、なかでも最も重要な課題は、自律的な行動を行う個々のロボットに協調的な動作を行うシステムをいかにしてもたせるかということと、必要な情報をいかにして伝えるかということである。ロボットは知的な個体であり、自律的で知的な個体は、エージェント(agent)などと呼ばれている。そして、人工知能の分野では、複数のエージェントが統一的な目標をもって協調しながら作業を行うシステムは、マルチエージェントロボットシステム、あるいは、群知能ロボットシステムと呼ばれ、そのときに発揮される全体としての知能は、群知能と呼ばれている。マルチエージェントシステムや群知能システムは、さまざまな分野でさまざまな手法を用いて研究が進められており、これからの発展が期待される。
