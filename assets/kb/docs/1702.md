# 3・1・5 行動計画(motion planning)

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・1 運動制御

現在開発されているロボットには,産業用ロボットとして広く活躍しているマニピュレータ,および各種の環境下での移動ロボットなどさまざまなものがあるが,ロボットの基本は動く機械であり,自律ロボットの開発には,自ら行動計画を作成する機能が重要になる.行動計画問題は,指定された位置(移動ロボットの場合)あるいは状態・姿勢(マニピュレータの場合)から目標の位置あるいは状態までの実現可能な経路を求めることである.途中でロボットやマニピュレータに干渉する物体は,障害物と呼ばれる.一般に,ロボット・マニピュレータは大きさを有するが,これを障害物側に付与することにより,ロボットを点として表現すれば,問題が簡単化される.ただし,複雑な形状や,細長いロボットの場合はこの作業が簡単ではない.すなわち,ロボットのどの部分と障害物のどの位置が干渉するかによって様子が異なることになる.このために,コンフィギュレーション(configuration)の概念が導入されている.ここで,コンフィギュレーションとは,ロボットの位置姿勢を表す変数の集合であり,この集合によってロボットを構成するすべての点が定まる.移動ロボットの場合は,その代表点(例えば質量中心)の位置および姿勢を表すパラメータ(例えば,オイラーの角)から,またマニピュレータの場合は,各関節変位からコンフィギュレーションが構成される.そして,あるロボットのとりうるすべてのコンフィギュレーションから構成される空間をコンフィギュレーション空間(config-uration space)と呼ぶ.この空間の次元は,対象とするロボットのコンフィギュレーションの数に等しく,平面上を移動する一般のロボットでは,位置に2自由度,姿勢に1自由度あるので三次元になり,自由度マニピュレータでは次元になる.さらに,コンフィギュレーション空間内の各点はロボットの特定の位置姿勢に対応しているので,この空間内でロボットは,点で表される.ロボットがある位置姿勢をとったとき,ある障害物と干渉する場合,ロボットはそのコンフィギュレーションをとれない.したがって,コンフィギュレーション空間において,そのような障害物と干渉する点の集合をコンフィギュレーション障害物(configurationobstacle)と呼ぶ.逆に,コンフィギュレーション障害物以外の点は,ロボットがそれに対応する位置姿勢をとることができる領域であり,自由空間(free space)と呼ばれる.図3・14にロボットが図3・14コンフィギュレーション障害物図3・15コンフィギュレーション障害物と可視グラフ図3・16 ボロノイ線図円とみなせる場合について,その代表点を円の中心としたときのコンフィギュレーション障害物の例を示す.ここでは,基本的な行動計画問題として,次の条件を考える.(a) 対象とする空間内で,障害物は固定されている.(b) 空間内に存在するロボットは1個のみである.(c) 空間内の情報は正確に得られている.ロボットの基本的行動計画問題では,初期および目標コンフィギュレーションならびに,障害物を設定し,その空間を構造化,すなわち,グラフ表現する.そして,初期および目標コンフィギュレーションを結ぶ自由空間内の連続的な経路を求めることになる.具体的に経路を決め,行動計画を立てる方法として多くの提案がなされているが,それらは,ロードマップ法,セル分割法,ポテンシャルフィールド法に大別される.a. ロードマップ法(roadmap method)曲線のネットワークにおけるロボットの自由空間の連結性をつないで構成されるものである.ロードマップがひとたび構成されると,問題はそのロードマップにおいて,初期コンフィギュレーションから目標コンフィギュレーションまで点をつなぐ問題に帰着される.このロードマップ法の概念に基づく方法として,可視グラフ法,ボロノイ図法などがある.. 可視グラフ法(visibility graph method)可視グラフ法は,コンフィギュレーション空間が二次元で,コンフィギュレーション障害物が多角形の場合に適用される.可視グラフは,初期および目標コンフィギュレーションとコンフィギュレーション障害物のすべての頂点からなる無向グラフで,二つの頂点を結ぶ線分がコンフィギュレーション障害物の内部に含まれないとき,それらの頂点は辺で結ばれる.可視グラフ法ではもし経路が存在すれば,比較的簡単に最短経路が求まる.図3・15に示す例において,太線で示す経路が最短経路である.. ボロノイ図法(Voronoi diagram method)2個以上のコンフィギュレーション障害物,あるいはコンフィギュレーション空間の境界から等距離にある自由空間の点を連結して描かれる図であり,二次元空間においては,一般に線になる.このボロノイ線上の点は,障害物からの距離が同じであり,通路の真ん中を移動することに相当する.図3・16にボロノイ線図の例を示す.b. セル分割法(cell decomposition method)ロボットの自由空間を,セルと呼ばれる簡単な領域に分割するもので,セル内では任意の2点間の経路が容易に生成できるようにする.そして,セル間の隣接関係を無向グラフで表現する.すなわち,自由空間にある各セルを頂点とし,セルが隣接している場合は対応する頂点が辺で結ばれる.このグラフで,初期から目標コンフィギュレーションまでの経路の探索を行う.セルの分割方法から,厳密法と近似法に大別される.厳密セル分割法は,自由空間に厳密に属する部分をセルとして分割する.また近似セル分割法は,前もって定義された形状(例えば,長方形)のセルを生成する.図3・17にセル分割法の例を示す.各セルは1~10まで番号を付けている.太線で示すように,セルの境界となる線分の中点を結べば,障害物から離れた経路が生成される.c. ポテンシャルフィールド法(potential field method)人工的な障害物等を付与するもので,障害物はロボットに斥力を与図3・17 セル分割法え,接触・干渉が起こらないようにする.また,初期コンフィギュレーションも斥力を発生し,引力を発する目標コンフィギュレーションへ導くことができるとする考え方が基本である.ポテンシャル場の生成方法は数多くの提案がされている.この方法の大きな問題点は,途中で極小値(local minimum)と呼ばれるくぼみに落ち込み,目標コンフィギュレーションに達する前に停止してしまう可能性があることである.この極小値に陥ったときにいかにして脱出し,目標コンフィギュレーションに到達するかが大きな課題である.d. 一般的な行動計画問題基本的行動計画問題を拡張した一般の場合は,解を求めるのが非常に難しくなる.拡張には,(1) 障害物が固定されていなくて,動く場合,または,動かしうる場合(2) ロボットが1台でなくて複数台存在する場合(3) 空間内の情報が,正確でない,または,不明である場合(4) ロボットの空間内の運動に何らかの拘束がある場合など,さまざまなケースが考えられる.それぞれ各種の提案等が行われているが,計算量が莫大になるなどまだ解決すべき課題が多くあり,統一的な解法は得られていない.
