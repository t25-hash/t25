# 3・2・3 クラスタ解析

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・2 パターン処理

標本が得られているが、それらにラベルが付いていない、いわゆる教師なし(unsupervised)状態のとき、それらを分類する手法としてクラスタ解析あるいはクラスタ分析(cluster analysis)と呼ばれる手法が適用できる。以下に挙げる手法はどれも非パラメトリックな手法である。以下の手法のほかにも、ニューラルネットワークの応用であるコホネン(Kohonen)の自己組織化マップ(SOM : self-organizing map)を利用する手法などが挙げられる。a. 平均クラスタリング平均クラスタリング(-meansclustering)は、非階層的クラスタリング手法の一種である。クラスタ数は既知とする。反復的最適化(iterative optimization)の一種であり、山登り手法と等価である。平均アルゴリズムは単純かつ実用的にうまく機能するため、主要なクラスタリング法となっている。また、この平均アルゴリズムの拡張として、どの標本もすべてのクラスタのメンバシップ度をもつよう拡張されたファジィ平均(fuzzy -means)法も提案されている。b. 階層的クラスタリング生物における分類学では、目、属、科、種などと階層的に分けられるように、クラスタ分けを階層的に行うことは人手によっても一般に行われている。階層的クラスタリング(hierarchical clustering)の結果は、樹系図(dendro-gram)で表現される。階層的クラスタリングには、分割的手法(divisive method)と凝集的手法(agglomerative method)がある。分割的手法は、すべての標本を含む一つのクラスタから開始して、連続的にクラスタを分割していくトップダウンアプローチである。一方、凝集的手法は、個の1要素クラスタから開始し、連続的にクラスタを融合して各レベルでのクラスタを形成していくボトムアップアプローチである。ここでは凝集的手法に着目してみる。最近傍クラスタ間距離が、与えられた任意のしきい値を超えた場合に処理を停止するものは、単連結アルゴリズム(single-linkage algorithm)と呼ばれる。ノイズまたはデータ点のわずかな位置変化に対して非常に敏感で、予期しない細長いクラスタを形成することがある。この現象はチェーン効果と呼ばれる。最近傍クラスタ間距離が、与えらえた任意のしきい値を超えた場合に処理を停止するものは、完全結合アルゴリズム(complete linkage algorithm)と呼ばれる。本来のクラスタが密集していてその大きさもほぼ同等である場合に有効である。しかし、細長いクラスタの場合はうまくいかない。比較的効率的とされているが、大きさが小さいクラスタを作りやすいといわれている。どの方法も一長一短あり、一般的なデータセットに対する優劣をつけることはできない。また、これらの距離を、四つのパラメータで統一的に表す式がランス(Lance)とウィリアムス(Williams)によって提案されている。これは距離を再帰的に求めることができる式であり、定義式を直接計算するよりも効率的に計算できる。ただし、各手法でのα,α,β,γの値は表3・1に示す。
