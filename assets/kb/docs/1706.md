# 3・2・4 主成分分析

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・2 パターン処理

主成分分析(PCA : principal component analysis)は、特徴空間の次元数を減らす手段として有効である。ここで具体例を見てみる。いま、二次元データを50点観測したとする。データをそのままプロットしたものを図3・19に示す。データ平均の点(0.087,0.078)を通り、を方向ベクトルとする直線は図3・19中に点線で表されている。また、得られた第一主成分を軸、第二主成分を軸にして各点の主成分をプロットした図を図3・20に示す。次元削減の際、分散の大きい主成分を採用し、分散の小さい主成分を無視することにする。σ=λであるので、先の計算により得られたλの大きさによって、その主成分を採用するか否かを判断することができる。主成分分析は、データ群をひとまとまりとして扱い、分布の散らばりが最大となる射影を探す手法である。一方、サンプル集合が複数のクラスに属するときは、フィッシャーの線形判別法(Fisher’slinear discriminant)や、多重判別分析(MDA : multiple dis-criminant analysis)でデータ群を最も良く判別する射影を探さなければならない。また、信号源が複数ある場合に信号を分離するのに効果的な手法として、独立成分分析(ICA : independent com-ponent analysis)が提案されている。主成分分析は、数学的にはKL展開(Klarhunen-Loeve expansion)とほとんど同じである。ここでは、主成分分析の概略を簡単に述べ、手順の例を示すだけにとどまったが、証明等より細かい議論は文献[例えば文献(16)の3.8.1、文献(19)(20)]を読むとよい。
