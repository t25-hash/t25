# 13・4・6 環状流（annular flow）

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・4 流体構造連成振動（fluid structure coupled vibration）

環状流に起因する流体構造連成問題は大きく分けて環状流路内を
流れる高速流体によって発生するフラッタと，環状すきま流路内で
発生するスロッシングと越流せき（overflow weir ）との連成によ
る自励振動に分けられる．
a.環状すきま流によるフラッタ 狭いすきまを有する環状流
路を挟んで対面する構造物が並進または回転の1自由度振動する場
合，軸方向のすきま形状を自励振動が発生しにくい先細とすること
が，自励振動を止めるのに有効である．これは一次元テーパすきま
流路の場合と同様である．より正確な自励振動発生限界流速の評価
を目指した研究例としては，李らの差分法による数値計算 な
どがある．
内筒がシェルモード振動する場合については，ジェットエンジン
アフタバーナ熱遮へいシェル(図13･20)の自励振動があり，ジアダ
ら はわずかに先細な片持コニカル殻を用いて再現実験を行い，
環状すきま部の流速がある限界を超えると自励振動が発生し，内筒
内部の流速は逆に系を安定化することを明らかにした．また理論解
析研究は，同心二重円筒殻の流動励起振動に関してパイドウシス
ら が精力的に行っているが，メカニズムの検討はほとんど行わ
れていない．
b.環状プレナム内流体のスロッシングと越流せきとの連成によ
るシェル振動（shell vibration） フランスの高速増殖炉スーパ
フェニックスの炉壁冷却系では，越流せきの自励振動が発生した．
図13･21に示すように，これは，下端を支持された円筒せきの外面
に沿って上昇する流体が上端部で越流し，円筒せき内面側のプレナ
ムに流入する系において，プレナム内流体スロッシングと円筒せき
オーバル振動が連成したものである．系の安定性は内側プレナム液
面揺動と内側プレナムへの流入量変動の位相差に依存し，位相差を
支配するのは越流量変動のせき振動に対する一次遅れと越流流体が
プレナムへ落下するまでに要するむだ時間である ．
