# 13・4・8 学会基準（JSME standard）

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・4 流体構造連成振動（fluid structure coupled vibration）

日本機械学会規格は研究開発推進センター標準事業部の監督のも
とに策定されている．そこでは2種類の委員会が活動している．一
つは常置委員会である発電用設備規格委員会である．もう1種類は
会員から規格制定の提案があったとき設置される委員会である．
発電用設備の構造設計に関する規格は，その守備範囲が広く，つ
ねに最新の知見を反映させようとするために常置委員会が不可欠で
ある．一方，流体関連振動に関する指針などは常時見直しが必要と
いうこともない．このため上記のような形態となっている．歴史的
には後者の形態の規格作りが先行している．
流体関連振動の関係では，これまでに二つの指針が制定されてい
る．一つはもんじゅの温度計破損事故を踏まえた「配管内円柱状構
造物の流力振動評価指針」 であり，もう一つは美浜2号機蒸気発
生器伝熱管破断事故とその後のR&Dを踏まえた「蒸気発生器伝
熱管U字管流力弾性振動防止指針」 である．これらの指針に従
い定められた手順どおりに評価を実施していけば，必ずしも振動の
専門家でなくとも振動発生を回避できるように作られている．
