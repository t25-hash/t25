# 13・5・1 流れの中の非円形断面柱自励振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・5 翼と翼列の自励振動

流れと直交する構造物の断面が非円形のときは，抗力のほかに揚迎え角増加とともに揚力が減少する領域で生じる失速フラッタめ生じる曲げねじりフラッタ（classical flutter），翼群におい（cascade flutter），に分けられる．
は単独の1自由度の構造物を超えたところで発生する．
力も作用するため，円柱では生じない不安定現象を生じる．
これは（stall flutter），構造物の曲げ振動とねじり振動が連成するたて翼間の流れの変化による非定常力が発生させる翼列フラッタでも生じ，は2自由度以上の構造物，は翼列にのみ生じる．
実際にはこれらの複合現象も生じる．
