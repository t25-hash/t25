# 3・4・2 形状認識

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・4 視覚情報処理

画像処理における形状認識には，形状をどのように表現するかに
よっていくつかの異なるアプローチが存在する．ここでは特に二次
元画像処理を対象とし，代表的な形状認識の処理方法のいくつかに
ついて説明することとする．
a.テンプレートマッチング 二つの画像が与えられたとき，
二つの画像を重ね合わせてその違いを調べることで，それらが同一
NTSC信号などのアナログ信号をディジタル画像へと変換する
であるかどうかを判断することができる．このことから，いま画像
中で探す対象物体の形状を小さな画像（テンプレート）として表現
すれば，図3･30に示すように，画像中でそのテンプレートを少し
(ａ)画像の走査 (ｂ)画像の標本化と量子化 (ｃ)ディジタル画像
