# 3・4・4 脳内視覚情報処理

> **章**: 第3章 ロボティクス/知能システム
> **節**: 3・4 視覚情報処理

本項では,人間の視覚情報処理について視覚生理と視覚心理の両面から概略を述べる.a. 光信号の感覚器としての眼球視覚により外界を知覚する際に,眼球は光を結像させる光学系としての働きと,光を神経信号へと変換する信号生成系としての働きを併せ持つ.人間の眼球の構造は図3・37のようになっており,外界からの光は角膜(cornea)と水晶体(crystalline lens)を通過し,網膜(retina)上に存在する視細胞において電気信号へと変換される.水晶体はその厚さや表面の曲率を変化させることにより網膜上の像のピントを調節しており,いわばカメラレンズの焦点調節に相当する働きをもつ.一方,虹彩(iris)はカメラの絞りに相当し,入射する光の量に応じてその直径を変化させ,被写界深度と眼球に入射する光量を調節する働きをもつ.網膜上に存在する視細胞には,大きく分けて錐体(cone)と桿体(rod)と呼ばれる形状の異なる二つの種類が存在する(図3・38).円すい状の形状をもつ錐体は,網膜上の中心禍と呼ばれる部分に集中して多数存在し光に対する感度が低い.そのため,錐体は明るい環境で細かい像を得る役割をもっている.また,錐体には異なる光の波長に対する感度の違う3種類が存在し,色の知覚を担っている.一方,棒状の形状をもつ桿体は中心禍を中心に視野角10°ほどの範囲に比較的まばらに存在し,光に対する感度が高い.そのため,桿体は暗い環境において粗い像を得る役割をもっている.錐体と異なり,単一の桿体しか存在しないために,桿体では色の知覚をすることができない.暗い環境では色の見分けが困難になるのはこのためである.人間の網膜には約1億個の視細胞が存在するが,これらの視細胞からの信号は網膜内に約100万個存在する神経節細胞に伝えられ図3・37 眼球の構造図3・38 視細胞の分布図3・36 無限遠点る.これらの神経節細胞からの信号は視神経束として束ねられ,眼球の乳頭と呼ばれる箇所から眼球の外へと出力される.このため,乳頭には視細胞が存在せず光を検出することができないため,人間の視野内には盲点(optic disc)と呼ばれる部分が存在する.眼球における光信号から電気信号への変換では,単に各視細胞からの信号が出力されるだけではなく,視細胞からの出力が神経節細胞へと集約される際に複数の視細胞からの信号がある程度まとまった形で処理され,出力されることが知られている.これにより,ある程度空間的に広がった光に対してのみ出力される信号,視野内のエッジが強調されるような信号,点滅する光にのみ出力される信号などが存在する.b. 視覚信号の伝達眼球から出た視神経束は,視神経交差と呼ばれる部分で交差し,外界の右側視野の信号が大脳左半球へ,また外界の左側視野の信号が大脳右半球へと伝達され処理されることが知られている.両眼からきた信号は外側膝状体と呼ばれる部分を経て大脳視覚領へと送られ知覚される.このようにして,両眼から出力された信号が脳内で取りまとめられ,対象の立体視など両眼からの情報に基づく処理が可能となっている.大脳視覚領における処理の過程の詳細についてはいまだ未解明な部分が多い.両眼視差などの処理を行う神経細胞や特定の形や動きに反応する神経細胞など,特定の機能に特化した神経細胞が存在することが知られている.このような知見に基づき,画像処理システムにおいても脳内情報処理を模したアルゴリズムなどが数多く研究されてきている.c. 視覚心理眼球に入射した光は網膜に存在する視細胞において,電気信号へと変換され大脳へと伝達されたのちに,複雑な処理の結果,人間に視覚情報として知覚される.このとき,眼球に入射する光の波長や放射強度などの物理量と人間によって知覚される明るさや色などの心理量とは単純な形では対応しない.ここでは,人間の視覚の心理的現象のうち明るさと色の知覚についてその一部を紹介する.明るさと色の知覚以外にも,錯視,空間知覚,運動知覚などに関するさまざまな現象が知られている.. 明るさの知覚CCD カメラなどの画像センサであれば,入射光のエネルギーや放射輝度などの物理量と画像の明るさは単純な式で関係付けることができる.しかしながら,人間の目や大脳視覚野における複雑な処理により,人間が感じる明るさという心理量は入射する光の物理量と非常に複雑な関係をもち,光刺激が網膜上で結像する位置や大きさ,周辺環境の明るさ,刺激の時間などさまざまな要因に依存する.そのため,光の輝度が2倍になったとしても,人間は単純に明るさが2倍になったとは感じない.人間が感じる明るさと入射光の輝度との関係に関しては,心理物理的な実験からいくつかのモデルが提案されている.その一つに人間の感覚量は刺激の強さの対数に比例するというウェーバー・フェヒナーの法則(Weber-Fechner’s constancy)と呼ばれる.人間の視覚がもつ明るさの恒常性という性質は,さまざまな照明条件のもとで物体を観察した場合にもそれが同一の物体であると認識するために非常に重要であり,画像処理システムにおいても同様の機能を実現するための手法がいくつか提案されている.. 色の知覚明るさとは別に,色の知覚に関してもいくつかの興味深い性質が知られている.その一つに色順応と呼ばれる現象がある.これは,同じ色を長く凝視しているとその色の彩度が低く感じられるようになり,その後,灰色などの無彩色を凝視すると最初に凝視していた色やその補色に見えるというものである.さらに,凝視した後に別の色を見たときに感じられる色が変化するといった現象も観察される.また,人間の視覚には明るさの恒常性と同様に色に関しても恒常性が存在することが知られている.物体の反射光の分光輝度分布は,物体表面を照らす光の分光輝度分布と物体表面の分光反射率の両方に依存するため,本来であれば異なる分光輝度分布をもった光の下で観察された物体の色は異なって感じられるはずである.しかしながら,例えば,赤みを帯びたタングステンランプの下で観察した場合と,より白色に近い蛍光灯などの下で観察した場合とで,物体が同じ色をもっていると感じられるといったことが起こる.このような傾向は色の恒常性(color constancy)と呼ばれ,明るさの恒常性と同様に,同一の物体を認識するために重要な性質の一つであり,画像処理システムにおいて色の恒常性を実現するためのアルゴリズムが開発されている.文献(1) 伊藤宏司・伊藤正美, 生体とロボットにおける運動制御, (1991),31-32,計測自動制御学会.(2) Denavit, J. and Hartenberg, R.S., J.
