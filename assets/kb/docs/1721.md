# 13・6・3 弾性容器内液面の振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・6 容器内の流体の振動

容器側壁の弾性を考慮した場合には，比較的低振動数で現れる液
面振動が顕著なスロッシング振動のほかに，側壁振動が顕著なバル
ジング（bulging）振動が，振動数の高い領域で生じる．この二つ
の振動の固有振動数は離れている場合が多く，両者の連成は一般的
に小さい ．液面振動が支配的なスロッシングに着目する場合に
は，容器の弾性変形の影響は小さく，容器の弾性変形の影響によ
り，液面振動の固有振動数はわずかに低下する ．バルジング
モードの固有振動数は，液深の増加とともに減少する．振動モード
も液体の量により変化する ．また，弾性容器を高い振動数で励
振するとき低い振動数の液面振動が発達する場合があることが指摘
されている ．
底壁の弾性を考慮した場合にも，液面振動が顕著なスロッシング
モードと，底面振動するバルジングモードが存在し，両者の固有振
動数は離れている．スロッシングモードの固有振動数は液深の小さ
い範囲で底面を剛体と考えた場合の固有振動数より高くなるが，液
深が大きくなるに従い両者は一致する場合があること．一方，バル
ジングモードの固有振動数は，液深の小さい領域で液深の増加とと
もに減少し，さらに液深が大きくなると増加する場合もあることが
ーメント は次式で表される．
指摘されている ．
