# 4・1・4 評価・測定手法

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・1 ヒューマンインタフェースの設計・評価

インタフェースを適切に評価する方法がなければ、計画的な設計もまた不可能となる。評価において最も重要なのは何を測るかを決定すること、言い換えれば、評価すべきインタフェースの性能を定義することである。それが直接評価可能な属性であればよいが、そうでない場合、その性能は別の測定可能な属性で再定義される必要がある。そのように定義された属性が、インタフェースの目的に添ったものでなければ、いくら良い評価が得られても意味がない。一方、設計のために、個々の目的に添った知覚・認知特性が知りたい場合がある。ここではインタフェースの評価手法とともに、しきい値測定などで用いる心理物理学的手法についても簡単に紹介する。a. パフォーマンス評価(performance evaluation)インタフェースの性能を何らかの行動指標で再定義することで、インタフェースを定量的に評価しようとする方法である。評価対象となるインタフェースの特徴をどのような測度で再定義するかを十分に吟味する必要がある。パフォーマンスの測度としては、反応時間、正確さ、習熟までにかかる時間、習熟の保持の度合い、満足度などがよく用いられる。b. 尺度構成法(psychological scaling)尺度には、名義尺度(分類のみ)、順序尺度(順位のみ)、間隔尺度(数値とその間隔に意味がある)、比率尺度(数値とその比に意味がある)がある。これらの尺度を心理学的に構成するのが尺度構成法である。一次元の尺度構成には直説法と間接法がある。直説法として例えば「快-不快」のような5段階から9段階の形容詞対を用いる評定尺度法があり、通常はこの方法でも間隔が保証されているとみなされる。また間接的な間隔尺度構成法として一対比較法がある。これは個の対象の中から一対の刺激を提示して被験者に判断を求め(回の判断を行う)、その後統計的処理によって間隔尺度を構成する方法である。個々の判断は二つの刺激の比較として行われるので、対象の評価に習熟していない被験者でも遂行が容易な利点をもつ一方、判断回数がで急速に増大し、また必ずしも被験者の判断が一次元に収まらないなどの欠点ももつ。心理学的に比率尺度を算出するのは一般に困難であるが、直接法としてマグニチュード推定法がよく用いられる。これは対象の評価を標準刺激に対する比として直接判断させる方法である(標準刺激を用いない方法もある)。心理的データは一般に多次元的な性格を有している。このような多次元的性格を直接尺度化しようとするのが多次元尺度構成法(MDS : multidimensional scaling)である。MDS は一般に個々のデータ間の距離から空間布置を求め、データ構造を明らかにする手法である。c. ユーザビリティテスト(usability test)想定ユーザが実際に評価対象を操作して評価を行う手法である。データは操作を発話させる(プロトコル分析)、操作履歴を記録する、操作後にヒアリングを行うなどして収集する。d. インスペクション法(inspection method)ユーザビリティテストは実際の評価対象やそのプロトタイプを操作して評価できる点で優れているが、開発過程の後半でしか適用できないという欠点をもつ。インスペクション法は実際の評価対象がなくてもその操作を評価するために開発された方法である。操作仕様書を吟味して問題点を明らかにする、あるいは、ユーザが目的に達するためにどのような行動をとるか吟味する(ウォークスルー法)ことが行われる。e. 心理的定数の測定法(心理物理学的測定法、psychophysicalmethod)検出閾や弁別閾、主観的等価点(PSE: point of sub-jective equality)などの心理的定数を測定する際には、一般に心理物理学的測定法と呼ばれる方法が用いられる。心理物理学的測定法には、調整法(被験者自身が刺激を観察しながら刺激を連続的に変化させて測定を行う)、極限法(実験者が刺激を離散的に変化させながら測定を行う)、恒常法(離散的に値が異なるいくつかの刺激をランダムに提示して測定を行う)、上下法[十分に知覚される(あるいはされない)値から実験を開始し、実験者が刺激の値を変化させて知覚されなくなったら(知覚されるようになったら)、値の変化方向を反転させることを繰返して測定を行う]などがある。f. 生理学的指標(physiological measures)評価しようとしているシステムの性能が生体から直接測定できるものとして再定義できれば、人間のさまざまな生理的反応を直接指標とすることも可能である。よく用いられる生理指標として、心拍、脳波(特に事象関連電位)、眼電位、筋電位、皮膚電位などがある。PET(positron emission tomography)やfMRI(functional magneticresonance imaging)など、近年になって進展の著しい非侵襲的な可視化手法も広義の生理学的指標としてとらえることができ、これらを利用してより高次の認知的活動を測定する試みがなされている。
