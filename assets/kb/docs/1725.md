# 4・2・1 マルチメディア情報処理

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・2 マルチメディア

a. マルチメディア(multimedia)メディアはテキスト,動画,音声,画像など情報の表現媒体であると同時に情報を伝送するキャリヤとしての機能ももっている.字義どおりならマルチメディアは複数のメディアの複合であるが,これらを同時に扱える映画やテレビをマルチメディアとは呼ばない.マルチメディアにはコンピュータをはじめとする情報システムによって扱え,ネットワークを通じて電送できるとともに,ユーザが情報を目的に応じてインタラクティブに選択・加工・変換・編集できる性質が備わっている必要がある.つまり,マルチメディアデータはディジタルな電子的データであることを特徴とする.マルチメディアが真価を発揮できるためには,ユーザとシステムの情報交換を担当するヒューマンインタフェース(human inter-face)が自然で,ユーザに認知的負担を強いないことが必要である.この場合のヒューマンインタフェースとして,(i)マルチメディア情報を授受するためのマルチメディアインタフェース(4・2・3・a項参照)と(ii)マルチメディア情報の加工や編集のための操作インタフェースがある.マルチメディアは1990年代に入って広くユーザに注目されるようになったが,基本的な概念と技術はそれ以前より確立されつつあった.インタネット上のWWW におけるハイパーテキスト(hypertext)やハイパーメディア(hypermedia)による情報の構造化,MIT のメディアラボで開発されたメディアルーム(MediaRoom)や町をコンピュータ上で自由に移動できるアスペンムービーマップ(Aspen Movie Map)などはその代表例である.しかし,今では情報の電子的表現媒体がマルチメディアであるのは普通となっていて,単純な電子メールでも音声や画像,映像を送受信,表示できることは周知の事実である.b. マルチメディア情報処理マルチメディアデータを扱う技術を一般にメディア技術(media technology)と呼ぶ.これにはデータ圧縮/復元技術,パターン認識技術,メディア変換技術,情報検索技術,データベース管理技術など非常に多様な技術が含まれる.従来は論理的な情報の処理に重点がおかれてきたが,ユーザに利用容易なメディア技術が確立されるにつれて,マルチメディアで表現される感性情報(kansei information,4・2・2・b項参照)を扱える環境の開発が必要になっている.マルチメディア情報は通常データベースやディジタルアーカイブに蓄積され,そこでのひとまとまりの情報をマルチメディアコンテンツとも呼ぶ.データベースあるいはアーカイブ内の大量のコンテンツにアクセスして上記の技術を駆使した処理を実行するのがマルチメディア情報処理である.音声,画像,映像,動画などは情報量が大きく,これらを蓄積するには音声を対象とするCELPやビデオを対象とするMPEG に代表される適切なデータ圧縮技術が不可欠である.また,ユーザの目的に応じたデータを探し出して提供するためには,内容や意味から検索できることが求められ,含まれているオブジェクトやその特徴を符号化して蓄積したり,検索時にそれらを抽出したりする処理法が提案されている.コンテンツの要約,注釈付け,例示オブジェクトとの類似度演算も内容検索に有用である.メディア変換は情報を維持しつつメディアを変える技術で,言語間の翻訳/通訳,音声や映像とテキストの相互変換などを含む.意味に及ぶ知的情報処理技術は萌芽期を迎えたばかりであり,2000年代のメディア技術の最大の課題である.
