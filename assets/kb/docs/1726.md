# 4・2・2 感性情報処理

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・2 マルチメディア

a. 感性(kansei)感性は,感覚器を通して得られる外界の情報に基づいて感覚,気分,情動,感情,共感,情緒,印象,欲求などを喚起する能動的能力を指し,外界の認知,判断,評価に影響を与える.感性は,論理的思考能力である理性に対するものとして位置付けられる.感性に関連する情報を感性情報,また感性情報を7―73担うメディアを感性メディアと呼ぶ.感性情報は,理性が対象とする論理的情報,知識情報に比べて,(i)主観的であり,個人差が極めて大きい,(ii)多義的,あいまいである,(iii)状況依存性が強いという特徴をもつ.感覚器からの情報は,大脳感覚野を経た後,一部は各種連合野に入って高次の統合処理を受け,他の一部は大脳辺縁系や中脳,間脳の視床下部に送られる.特に右脳連合野では視覚,聴覚,触覚などに関するイメージ処理が行われて,感性的な情報へと統合される.この情報はさらに左脳からの論理的な情報と総合されて外界の判断,評価に関わる.一方,大脳辺縁系,視床下部に送られた情報によって感情,本能が引き起こされるとともに,種々の自律神経反射を生じる.感情はストレスなどおかれた状況からの影響を強く受ける.本能的行動と情動による直接行動は理性により抑制されることが多いが,感情や気分は態度と行動に反映される.b. 感性情報処理感性情報処理は図4・1のように,(i)人間が脳で行う情報処理と,(ii)人工の情報システムでの情報処理に分けて考える必要がある.(i)は人間の感性の特性を明らかにしたり,感性情報処理のメカニズムを解明することが目的であり,商品やデザインの感性評価を行ったり,人間と同じ感性情報処理を行えるシステムを実現するための知見を提供したりするうえで重要である.特に近年,情報表現やヒューマンインタフェースで人間が直感的に理解できたり,操作できたりすることが非常に重視されるようになってきた.このような情報表現,インタフェースをデザインする際には,人間の感性と感性情報処理のメカニズムを理解していることが求められる.(ii)は感性情報処理の自動化が目的であり,特に(i)で明らかになる人間の特性を模擬できれば,感性評価や感性デザインの自動化が可能になる.人間の感性的反応を情報システムでまねることは感性シミュレーションと呼ばれる.(i)(ii)に関する研究を行って,人間の感性にマッチする製品のデザインに応用する分野として感性工学(kansei engineering)がある.データベースから人間の感性に適合する対象を検索する感性検索も感性工学の対象である.c. 感性計測各種のオブジェクトに対して人間が生じる感覚,感情などの感性的反応を計測することが広く行われるようになった.感性計測には図4・1に示すように心理計測と生理計測がある.脳に生じる感覚や感情を計測することは不可能であり,どちらも間接的な計測であることに注意する必要がある.心理計測は被験者の感情などの質と強さを「暖かい」,「なつかしい」などの主観的な感性語と評点で応えさせて,データ化する.「明るい-暗い」のように意味が正反対となる複数の感性語対を用意し,各対ごとに5~9段階で評価させるSD 法(semantic differential method,意味微分法)がよく知られている.さらに評価対象を複数個提示し,特定の評価項目について対象を順序付ける一対比較法(この場合は対象が2個),順位法などもある.生理計測は感情などと自律神経反射が連動していることを利用して,心拍数,皮膚温,脈波,皮膚電気反射などから感情やストレスの強さを知る方法である.感性計測に基づいて製品やデザインを評価することを官能検査(sensory evalu-図4・1 感性情報処理と感性計測,(i)人間の脳における感性情報処理,(ii)情報システムにおける感性情報処理7―74情報・ソフトウェアation),特に心理計測を用いる場合を主観評価(subjective evalu-ation)という.d. 感性情報処理技術感性情報処理を情報システムで実行するための種々の技術が提案されている.それらは以下の4種類に大別できる.(1) 感性認識技術感性情報を含む映像,表情,音声,音楽などから感情や情動を認識,抽出する技術.(2) 感性合成技術感性語を与えて文章,画像や音楽などを合成する技術.(3) 感性検索技術感性語を与えて,それにマッチするコンテンツをデータベースから検索する技術.絵画,画像,楽曲などを対象として行われている.(4) 感性加工技術感性情報を制御して感情を付与したり,強さなどを変えたりする技術.いずれも4・2・1・b項で触れたメディア技術の一部であり,特に(1)と(2)はメディア変換技術の一種といえる.
