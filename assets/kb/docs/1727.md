# 4・2・3 マルチメディア/マルチモーダルインタフェース

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・2 マルチメディア

a. マルチメディアインタフェース(multimedia interface)人間と情報システムとの間でマルチメディア情報をやり取りするために必要なのがマルチメディアインタフェースであり,主としてシステムから人間への情報提示に用いられる.コンピュータにスピーカを組合わせてネットワークに接続すれば,視覚的,聴覚的情報を提示でき,WWW のハイパーメディアやマルチメディアパブリッシング,ビデオオンデマンドなどの応用に対応できる.しかし技術開発の力点は,臨場感,直接感を高めて,人間の感性に直接訴えることのできる環境を実現する方向へ移されている.立体感をもつ音場の提示や,視野の広い三次元視覚ディスプレイ,力覚や触覚,嗅覚のディスプレイを組み合わせることにより,仮想世界を体験したり,実世界と仮想世界を重ねたりできるマルチメディアインタフェースが試作されている.バーチャルリアリティ(virtual real-ity),オーグメンティッドリアリティ(augmented reality),インタラクティブアート(interactive art)はこのようなインタフェースの究極の応用例といえる.b. マルチモーダルインタフェース(multimodal interface)コミュニケーションにおける人間の情報表現は運動器官を通してなされ,音声,身振り,表情,視線などとして表出される.人間どうしの対面コミュニケーションでは髪型や容ぼう,衣服などもメッセージ伝達に用いられる.音声を除く身体による情報表現媒体は身体メディア,ノンバーバルメディア(nonverbal media)などと呼ばれ,キーボード,マウス,スライダなどの接触型インタフェースと異なって,人間にとって自然であり,直感的に扱うことができる.この場合の情報は身体メディアごとに異なるモード[mode,またはモダリティ(modality),表現様式]をもち,人間は同時に多数のモードを用いて情報を表出できる特性をもっている.この特性を人間とシステムの対話に適用して,多様なモードを通して人間がシステムに情報や操作を伝えるためのインタフェースをマルチモーダルインタフェースと呼ぶ.マルチメディアインタフェースが主としてシステムから人間への情報提示に用いられるのに対して,マルチモーダルインタフェースは通常人間からシステムへの情報伝達に使われる.ノンバーバルメディアが人間どうしのコミュニケーションで果たす役割は広範囲に及んでおり,コンピュータをはじめとする機械と人間のコミュニケーションにおいてもマルチモーダルインタフェースを通じてそれらの機能が活用できると期待されている.世界で最初に実現されたマルチモーダルインタフェースは,1970年代末にMIT メディアラボで作られた「それをあそこに(PutThat There)」(図4・2)で,音声と指差しを用いてディスプレイ上の図形を操作するシステムであった.現在は,音声,身振り,姿勢,視線,うなずきの組合せがよく使われ,これにキーボードやマウスが追加されることもある.図4・2 쓕それをあそこに」図4・3 手話を話すアバタヒューマンインタフェース(human interface)の一種に擬人化エージェント(personified agent)がある.これはユーザまたはシステムの代理として機能するソフトウェアを人物に似せて作り,画面上に表示したもので,人間とシステムの対話を容易にする目的をもっている.また物理的,心理的に遠隔の相手との対話を自然にするために,相手の化身であるアバタ(avatar)を画面上に提示する仮想コミュニケーション環境も構築されている.この場合,目前のアバタとの対話はそのまま遠隔の相手とのコミュニケーションとなるわけである.これらエージェントやアバタは人間に似た容姿をもち,またマルチモーダルなふるまいをすることが望ましい.この場合にはシステムから人間への情報提示にもマルチモーダルインタフェースが活用できる.図4・3は手指動作に加えて,表情,まばたき,うなずきを交えた手話表現をするアバタの例で,容ぼう,衣服などへの配慮も要求される.
