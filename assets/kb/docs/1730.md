# 4・4・1 VR システム

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・4 バーチャルリアリティ

バーチャルリアリティ(VR : virtual reality)は仮想現実感または人工現実感とも呼ばれ,物理的にその場に存在しないものをコンピュータによって制御された入出力装置を用いて,人にあたかも存在しているかのような感覚を提示する技術である.例えば,開発中の自動車のCAD データがあったとする.このデータは数値的な情報であるため,三次元形状を確認するには粘土や木型でプロトタイプモデルを作ることになる.しかし,物理的なモデルを作らずに立体的な映像を表示し,かつ自由な位置から眺めることができれば物理的なモデルを作ることと同じように三次元形状の確認が可能である.また,物理的な実体をもたないため,形状の変更等の操作およびその後の再確認が容易であり,実物体以上の柔軟性がある.このように,実物体と同等またはそれ以上の機能をコンピュータによって作り出すことがVR の目的である.a. VR システムの構成VR を実現するための機器の構成は,簡略化すると図4・7に示すように三つのブロックに分けられる.センサ(動作入力)サブシステムは利用者からコンピュータへの入力部分であるが,キーボードのような記号的な入力ではなく,利用者の直接的な動作を計測し,入力としている.ディスプレイ(感覚情報出力)サブシステムでは,利用者の五感に働きかける出力を統合的に提示する.シミュレーションサブシステムは入力に応じてVR空間を更新し,出力を制御する部分であり,VR 空間を矛盾なく管理する.一般にセンササブシステムは計測機器,ディスプレイサブシステムは映像提示装置や音響提示措置,シミュレーションサブシステムはコンピュータという構成をとるが,シミュレーションサブシステムは1台のコンピュータで閉じたものではなく,ネットワークを通して遠隔地のテレビカメラや別の利用者のVR システムとつながったシステム構成をとることもある.また,入出力のサブシステムにおいては,一つのハードウェアで入出力の両方の役割をもつものもある.b. VR システムに求められる機能VR システムが他のコンピュータシステムと異なる点は,利用者に対してコンピュータを操作していると思わせないユーザインタフェースの実現にあり,実時間性(リアルタイム性,real-time),空間性(spatiality),身体性(embodiment)の3点を実現することが必要である.実時間性とは,利用者の動作が即座に出力結果に反映されることであり,工学的にいえば出力結果が安定するまでの時間遅れが少ないシステムということである.先のCAD モデルの例でいえば,視点位置に応じて即座に映像が切換わることに相当する.空間性とは,人間が生活する三次元空間と同様の座標系でVR 空間が管理されていることであり,先の例では,視点が上に移動したら上から見下ろした映像となる.図4・7 VR システムの構成が,車から離れれば遠方から見た映像が提示されることに相当する.身体性とは,利用者の動作が直接的にVR 空間に反映されることであり,これも先の例でいうと,利用者が立っているときは車のボンネットが,かがめばタイヤのあたりが見えるというように,身体動作に応じて提示される映像が変化することに相当する.c. 現実世界を模倣するVRVR 技術の最も一般的な例は,現実の世界と同様の世界をコンピュータの中で構築することである.具体的な機構は次項以降で説明するが,物に触ったかのような感触を指先で感じる,車が目の前を通過したかのような音の移動を聞く,などがわかりやすい例である.そのためには,現実を忠実にシミュレートする必要があるが,忠実なシミュレーションには多くの時間がかかるため実時間性が損なわれるおそれがある.計算機の能力とシミュレーション結果の正しさ,実時間性のバランスを考慮してシミュレーションアルゴリズムを選ぶ必要がある.また,VR 空間のデータ量についても同様のことがいえる.例えば建物のウォークスルーを行うときに,建築用CAD のデータを単純にデータ変換して表示すると,データ量の多さからスムーズなウォークスルーはできない.VR は建築CAD とは異なり,見た目が正しければそれ以上の精度は必要としないため,データの間引きや見た目に応じた精細度の設定(LOD : level of detail)を付けてデータの管理を行う.d. 複合現実感・拡張現実感複合現実感(MR : mixed real-ity)または拡張現実感(AR : augmented reality)と呼ばれる技術は,実空間とVR 空間を重ね合わせて情報を提示する手法である.実空間に重ね合わせたコンピュータ映像を表示するためにはシースルーHMD が用いられ,重ね合わせにあたっては実空間とVR空間の位置ずれを抑えることが問題となる.
