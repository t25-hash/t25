# 13・8・7 蒸気の凝縮による振動

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・8 熱・気泡・相変化に起因する振動

a.静止液中への蒸気吹込みに伴う振動 容器内の液中に蒸気
を直接吹込み，凝縮させる．この技術は原子炉の圧力抑制型格納容
器や簡易凝縮器に利用されている．このとき激しい圧力変動が発生
することがある ．この様子を図13･54に示す．
蒸気流量が非常に小さければ，蒸気は管内で凝縮を完了し，管内
の液面は上下動する ものの大きな圧力変動は生じない．流量が
ある程度大きくなると，管出口の蒸気泡生成，その急激な凝縮と管
内の負圧発生，液の管内流入と圧力回復を間欠的に繰返す．これを
チャギング（chugging）と呼ぶ．蒸気流量が非常に大きくなると，
周期的圧力変動が観測される．これを凝縮振動と呼ぶ．このとき管
外の気液界面は激しく振動している．振動数 ，管径 ，蒸気流速
，サブクールΔ，蒸発潜熱 ，液の比熱 はほぼ次の関係に
ある ．

b.蒸気流中へのサブクール液注入に伴う振動 図13･55に示
すように，蒸気の流れる管内にサブクール液を定常的に注入する
と，蒸気は凝縮し液体プラグを生じる．この液体プラグ位置が振動
し圧力も変動する ．この現象もチャギング（chugging）
と呼ばれる．蒸気流量が小さいときは液体プラグ上流側界面位置は
注入点の上流と下流の間を往復し，周期は比較的長く，圧力振幅も
大きい．蒸気流量の増加とともに界面位置は下流に移動し，注入点
の下流だけで振動するようになる．周期は短く振幅は小さくなり，
ある蒸気流量以上では界面は振動しなくなる．
c .サブクール液中にトラップされた蒸気凝縮による水撃 管
内を流れるサブクール液中に蒸気がトラップされると，ボイド消滅
による水撃現象を発生する ．
