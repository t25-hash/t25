# 4・4・3 動作入力装置

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・4 バーチャルリアリティ

一般的にコンピュータへの入力は、キーボードやマウスのような記号的な入力装置が用いられるが、VR 分野では人間の直接的な動作をコンピュータへの入力としてVR 空間を操作する。特に、人物の全身の動きを計測することをモーションキャプチャといい、一般に関節の位置を計測して骨格モデルに当てはめることで人物の姿勢を再現する。動作入力装置は人間の動作を計測する各種の計測装置であり、必ずしもVR のために開発された特別な装置であるとは限らない。a. 接触型動作入力装置。 グローブ手の動きは人間が外界に対して働きかける動7―81作の中で、最も重要で複雑な動作である。単に手先の位置を計測するのみでなく、手の形状を計測することで、VR 空間における物体の把持や直接的な操作が可能となる。手の形状計測にはグローブ型のデバイスが用いられる。初期のグローブは各指に沿って光ファイバが逆U の字型に縫い込まれており、光ファイバの片側は発光素子に、反対側は受光素子に接続されている。光ファイバは曲げるとファイバから外に漏れ出す光が多くなる性質があるため、指の曲げに応じて光の透過量が減少し、指の曲げ角が測定できる。しかし、もともと光ファイバは曲げに弱く切断しやすい素材であるため、後には曲げによって抵抗値が変化する抵抗膜素子で曲げの検出を行うようになった。簡易なグローブでは各指の第1関節の曲げのみを検出するが、高度なものでは、第1関節に加えて第2関節の曲げと手首の向きを検出するもの、さらには指の間の開き角も検出可能なグローブも存在する。第3関節まで検出するグローブは存在しないが、第3関節は第2関節と連動して動くため、第2関節の曲げ角から推定が可能である。. スーツグローブと同様の原理で、体にフィットするスーツの関節に当たる部分に曲げを検出する素子を縫い込み、体の動きを計測するスーツも存在する。しかし、装着の手間や計測精度、計測素子の耐久性の問題から、最近では光学式のモーションキャプチャシステムに取って代わられている。. 機械式位置・姿勢センサ機械式位置・姿勢センサには多くの種類のものが存在するが、一般にはジョイント部分にロータリーエンコーダを配置した多関節型のアームが用いられる。計測するジョイントの数によって、位置のみを計測する3自由度のものと、位置と姿勢を計測する6自由度のものがある。計測精度は高いが、機械的な制約から広い計測範囲をカバーする装置の作成は困難である。また、これの特殊な例として、人の体の動きを計測するために、人体の関節に対応する位置にエンコーダを配置した、外骨格型のモーションキャプチャ装置も存在する。b. 非接触型動作入力装置。 光学式位置センサ光学式位置センサは動作を計測する人物に取付けた特徴点(マーカ)の位置を、複数のビデオカメラで撮影された画像から三角測量の原理で求めるものである。計測できるのはマーカの位置のみであるが、剛体上に3点のマーカを設置してその位置を計測すれば、剛体の姿勢を計測することも可能である。計測のレートは一般にビデオレート(30Hz)以下であり、マーカの数が多くなると速度は低下する。また、高速化、高精度化のため、撮像素子にラインCCD を用いたものもあり、最大1.2kHzのレートで誤差0.1mmの計測が可能なものもある。光学式センサは近年の画像処理技術の発達によって、高速・高精度で位置計測が可能になったが、オクルージョンの問題が避けられない。オクルージョンとは、マーカがカメラから撮影できなくなる状態になることであり、カメラから見てマーカが人物の陰に入った場合などに発生する。カメラの台数を増やすことでオクルージョンを起こす範囲を少なくすることは可能であるが、完全に回避することは困難である。. 磁気式位置・姿勢センサ磁気式位置・姿勢センサは図で構成され、受信機の位置と姿勢を同時に計測することが可能である。発信機のコイルは一つずつ順に励磁され、受信機のコイルにはその磁界によって誘導電流が発生する。このとき、発信機側の励磁されたコイルと受信機のコイルの位置関係によって、三つのコイルで発生する誘導電流の大きさが異なる。この起電流の大きさの比から受信機の姿勢が求まり、起電流の大きさから位置が求まる。計測範囲は送信機の出力によって異なるが、広いものでも送信機を中心に半径2mの半球内程度であり、同時に32点程度の受信機が利用できる。また、センサは有線のものが一般的であるが、近年無線式のものも開発されている。磁気式センサはオクルージョンが発生し7―82情報・ソフトウェア図4・14 磁気式位置姿勢センサの原理ないという特徴があるが、計測する環境の影響を受けやすく特に金属の影響を大きく受ける。. 超音波式位置センサ超音波式位置センサは、発音体から出された超音波を3点以上の受音点で受信し、音波の到達時間差および位相差から位置を求めるものである。光学式位置センサの場合と同様に、剛体上に3点の発音体を取付けることで剛体の姿勢を計測することも可能である。複数の発音体を用いても、周波数または発音のタイミングによって区別ができる。音波を用いる関係から、発音点と受音点間の距離が離れると計測誤差が大きくなるが、近年では天井に多数の受音点を配置して広範囲の位置計測を行うシステムも開発されている。. ステレオビジョンステレオビジョンシステムは平面上に配置した2台以上のカメラを用いて、カメラから見た奥行き方向の距離を計測するものである。光学式位置センサが点の位置を計測するものであったのに対して、ステレオビジョンでは撮影した画像中の特徴点ごとに奥行き方向の距離を計算するため、面で距離を計測することが可能である。計測対象にマーカなどを取付けることも不要であり、計測用カメラでカラー画像の撮影もできるが、一般に他の計測方法に比べて計測精度は劣る。c. 生体情報計測装置心拍数、血圧、発汗量などの生態情報は人間の心理状態を表す指標として用いることができ、VR システムの入力装置としての利用が可能であると考えられる。しかし、現時点ではVR システムの評価に用いられている程度であり、入力装置としての積極的な利用は見られない。今後、ウェアラブルコンピュータの進歩と相まって、これらのセンサを身に付け、VR システムへの入力として応用することが期待される。
