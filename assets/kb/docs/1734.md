# 13・9・2 パッシブ制御（passive control ）

> **章**: 第13章 熱・流体と構造系の連成振動
> **節**: 13・9 流体を含む系の制振（flow-induced vibration con

流体そのものが振動するシステムを対象とする場合の例として，
容器内で発生するスロッシングを防振するために中央部に隔壁を挿
入して制振する方法 や逆U字管を利用したダイナミックダ
ンパを構成して制振する研究 が行われている．
流体振動現象を利用した構造物用のパッシブ制振装置の代表例は
スロッシングダンパである．長周期構造物用のダイナミックダンパ
として，容器内の液体の同調運動を利用した同調液柱管ダンパ
（TLCD）や同調液体ダンパ（TLD）がある．TLCDは任意形状の
液柱管内の液体の往復運動と液柱管内部に設けたオリフィスの減衰
により制振効果を発生させている ．一方，TLDは容器内のスロ
ッシングを利用したダンパであるが，非線形分布定数系であるため
取扱いが面倒である．このダンパについても，最適な減衰を得るた
めには，TLCDのオリフィスに相当するエネルギー散逸要素が必
要となる．最適減衰を得るための方策は，容器底面の表面粗さを増
す ，作動液体の粘度を増す ，容器の上面に蓋 あるいは反
射板を付けて砕波を起こす ，容器の中に抵抗となるような十字
の断面をもつ棒 やネット を挿入するというものである．
ネットを要素として挿入するタイプのダンパは，浅水波型スロッ
シングダンパ および深水波型スロッシングダンパ の二つの
タイプがあるが，両方について減衰性能を理論的に予測することが
可能である．図13･56には，深水波型スロッシングダンパを斜張橋
頂部に設置した場合の設置図を示す．
磁石を利用したパッシブダンパとしては，図13･57に示す斜張橋
ケーブル制振用ダンパが開発されている．レインバイブレーショ
ン，ウェークギャロッピングの抑制を目的として，磁石と金属板と
の衝突を利用することにより低次モードで発生するケーブル振動を
安定な高次モードへと変換し，発生する空力減衰を負から正に変え
ることで風による自励振動の抑制が可能である ．
