# 4・5・2 タンジブルインタフェース

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・5 実世界指向インタフェース

実世界の道具は物理的な存在であり、手で触れて操作できる。優れた道具には、手にしっくりとなじむ良さがあるものが多い。一方、コンピュータ画面に現れるメタファとしてのGUI部品は、単なる映像であり触れることはできない。そこで、手にもって操作できる実世界の道具の使い心地と、魔法のような機能を発揮するGUIアプリケーションを融合させる試みがタンジブルインタフェース(tangible interface)である。実際にはコンピュータで位置や状況が認識できるブロックや模型によって、コンピュータ情報にアクセスするシステムが作られている。例えば、Trianglesはコンピュータチップ内蔵のパネル型の積み木である(図4・20)。三辺には電気的なコネクタを兼ねた磁石のジョイントがある。積み木を組立てることで、ジョイントが電気的図4・18 機能強化された机。机上に置いた建築模型の影などを表示する[C2000 Tangible Media Group/MIT Media Lab]図4・19機能強化された部屋。机や壁、掛時計、棚に置いた小物などがディジタル情報を表示する[C2000 Tangible Media Group/MIT Media Lab]7―85に接続され、コンピュータチップに電源が供給されると同時に、ジョイントを通して通信が行われ、積み木の接続状況を検出する。この結果、積み木と同じ形の立体をコンピュータ内部に構築したり、積み木の接続によってストーリーが展開するようなインタラクティブな絵本を操作できる。
