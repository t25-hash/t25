# 13・1・2 常流と射流

> **章**: 第13章 自由表面のある流れ
> **節**: 13・1 開水路の流れ

流れに対応する．
等流公式は，河川を対象に歴史的にいろいろ提案されてきたが，現在，次の二つがよく使われている．
シェジー公式（Chezy，1775，フランス）マニング公式（Manning，1889，アイルランド）＞すなわち＞1では微小かく乱波は上流に伝ぱしない．
超音速流と類似である．
1の流れを射流（supercritical flow），＜1の流れを常流（subcritical flow）という．
＝1の流れが限界流で，この水深を限界水深（critical depth）といい，開水路の水面形を分類する基本となる．
