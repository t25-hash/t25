# 4・5・3 ユビキタス/ウェアラブルコンピューティング

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・5 実世界指向インタフェース

ユビキタス(ubiquitous)とはあまねく存在するという意味である.コンピュータが安価で小型になり,特別な道具ではない単なる日用品となり,生活のあらゆる場面で人々の活動を支える目的で利用されることを,ユビキタスコンピューティング(ubiquitouscomputing)と呼ぶ.ユビキタスと同義の用語で,パーベイシブ(pervasive),インビジブル(invisible),ディスアピアリング(disappearing)などが使われることもある.家電製品,家具や調度品,建具,床,ホワイトボード,壁,天井などに組込んだコンピュータや,ノートパッド型や石けん箱大のパッド型の携帯コンピュータ,身に付けるバッジ型のコンピュータなどを利用する.すでにわれわれは何百個もの組込みコンピュータに囲まれて生活しているが,ユビキタスコンピューティングではそれらがネットワーク接続されて連携する世界を目指している.例えば,TRON プロジェクトでは,ネットワーク接続された大量のコンピュータが埋め込まれた電脳住宅を建設して,空調や家電機器を,人や環境の状況に応じて連携してコントロールする実験が行われた.ユビキタスなコンピュータの中で,身に付けるコンピュータをウェアラブルコンピュータ(wearable computer)と呼ぶ.携帯できるコンピュータを使ったモバイルコンピュータがさらに進展した利用形態であり,衣服に組込んだり,身体に装着するコンピュータを指す.ウェアラブルコンピュータは商品化されて(図4・24),倉庫などでの商品管理や機器や設備のメンテナンスの現場で,備蓄状況やメンテナンスマニュアルのデータベースを利用したり,現場の画像データを利用した遠隔コミュニケーションなどの用途で実用化されている.現状ではこのようなニッチな業務用目的に利用されているが,実世界のコンテクストアウェア(状況把握)アプリケーションにより,一般的な日用品として普及していくと考えられている.図4・23 登場キャラクタの声が聴けるインタラクティブな紙の絵本図4・24 ウェアラブルコンピュータ例えばforget-me-notという実験システムでは,オフィスワーカーが常時身に付けるバッジ型の小型コンピュータを使って,誰がいつ,どの場所にいるかの情報を記録し,ネットワークアクセスを提供している.各部屋の天井などに赤外線の送受信機を設置して,バッジ型コンピュータと通信して人の位置を検出する.これを使って,オフィスでの人の位置,出会い,電子的なコミュニケーションなどのコンテクストを利用した情報検索(例えば「先週,A さんと打ち合わせしていたときに見たファイル」を探すなど)を実現している.
