# 4・5・1 項のコンピュータ強化環境では、実世界指向インタフェー

> **章**: 第4章 ヒューマンインタフェース
> **節**: 4・5 実世界指向インタフェース

スにより、従来のGUIでは画面の上に実現していたコンピュータの魔法の世界を、実世界に展開する手法を述べた。環境に埋め込まれたコンピュータを利用すると、環境の機能をコンピュータの魔法により強化できる。一方で、人が常時装着して、常に稼働しているウェアラブルコンピュータで拡張現実システムを動かせば、眼鏡や補聴器と同じように、人の認知能力を強化する道具となるであろう。例えば図4・16のシステムを使って、機械の中の構造が見えるようになったユーザは一種の透視能力を得たのと同様であるし、顔を認識して名前を表示してくれるHMD を身に付ければ、人の名前を覚える記憶力が強化される。
