# 5・1・1 数値処理

> **章**: 第5章 技術情報システム
> **節**: 5・1 情報解析

計算機を用いた数値情報処理は、機械工学をはじめとする今日の科学技術分野において重要な手法となっている。本項では、計算機を用いた数値情報処理の基礎的な事項についてまとめる。a. 数の表現と誤差計算機の内部では、数値は有限のけたで表現される。科学技術計算では、浮動小数点(floating point)という数体系を用いることで連続的な実数を近似し、演算の基礎としている。そのため、演算結果の絶対値が最大数を超えるとオーバフロー(overflow)、最小数より小さくなるとアンダフロー(underflow)を起こし、この数体系では扱えなくなる。そのため浮動小数点を用いた演算では、実数の演算とは異なる現象が生じることに注意する必要がある。例えば、絶対値が大きく異なる2数の加減算では、けたぞろえを行う際に絶対値の小さい数の仮数の末尾が無視されて失われてしまう。これを情報落ちと呼び、これによって生じる誤差を情報落ちによる誤差という。また、絶対値がほぼ等しい2数の減算では、上位の何けたかが失われることで、丸めによる誤差を含む下位のけたが上位けたを占めるようになる。これをけた落ちの誤差と呼ぶ。数値計算では、演算時におけるこれらの誤差の影響により、数学的には等しい式であっても必ずしも等しくならないことがある。この際、すべての標本点を通る関数を求めることで、標本点以外の値を近似推定する方法が補間法である。この次多項式は、+1個の標本点が与えられると一義的に決定され、ラグランジュ補間(Lagrangian interpolation)多項式という。ラグランジュ補間多項式は、全区間を一つの多項式で近似するため、高次になると補間曲線が波打つなどの問題が生じる。そこで、全区間をいくつかの部分区間に分け、各部分区間での補間多項式を滑らかにつなぎ合わせる方法が考えられる。部分区間での補間を三次多項式で表し、接続点で2階の微係数まで連続となる曲線で表現7―9 3したものを三次スプライン(spline)補間という。c. 数値微分と数値積分微分や積分を解析的に行えない場合、あるいは関数が離散的に与えられている場合、数値微分や数値積分が必要になる。d. 連立一次方程式数値計算による元連立一次方程式=の解法は、ガウスの消去法を代表とする直接法とヤコビ法などの反復法に大別できる。ガウスの消去法では以下の手順により計算される。この変換過程を前進消去という。この過程を後退代入といい、最終的に解を求めることができる。反復法は、ある初期値を与え、,...を逐次的に求めていく方法である。が要求される精度に達するまで上式を用いた逐次近似を反復する方法がヤコビ法と呼ばれる方法である。e. 非線形方程式非線形方程式の解法にも逐次近似による反復法が用いられる。反復解法の中で最も基本的で適用範囲の広い方法は、ニュートン法(Newton’s method)と呼ばれる方法である。この反復公式を用いた解法がニュートン法である。常微分方程式の形に定式化される問題は多いが、多くの場合、解析的に解くことが困難で数値解法が必要になる。したがって、初期値問題は上式をできるだけ精度よく計算する積分問題に帰着される。上式を用いた微分方程式の解法をオイラー法(Euler’s method)という。ここでは未知数であるが、これに前述のオイラー法で得られる値を代入して用いる方法を改良オイラー法という。このときはテイラー展開と精度が一致するようにとられ、上述の改良オイラー法は2階のルンゲ・クッタ法に相当する。g. 偏微分方程式偏微分方程式は、常微分方程式の解法を多変数関数に拡張したものと考えることができる。これは初期条件から始まり、を一つずつ増加させながらを順に決定していくことができることを示している。平均の重みは正の値を取るべきことから0<<1の条件が導かれるが、これは解が振動しないための安定条件となる。
