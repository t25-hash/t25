# 13・2・2 水平平板上の液膜流れ

> **章**: 第13章 自由表面のある流れ
> **節**: 13・1 開水路の流れ

管路流に比べて大きくなり，縦渦（二次流セル）が発生しやすい点
である．縦渦が発生すると，最大流速点が降下したり，河床が移動
床のとき縦筋が発生し，流れが三次元性となる．このような三次元
乱流構造に関しては文献(５)を参照していただきたい．
開水路流れの組織乱流構造に関しても，ある程度解明されてい
る ．内層にはバースティング現象が存在し，乱れエネルギーの発
生機構になっている．外層には河床波などの影響で大規模なコルク
渦が発生し，これが水面まで発達し，ボイル渦になる．浮遊土砂は
ボイル渦によって水面まで間欠的に浮上する．河川のさまざまな渦
構造はこのような組織乱流によって構成されている．
