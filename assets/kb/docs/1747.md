# 13・3・2 微小振幅波

> **章**: 第13章 自由表面のある流れ
> **節**: 13・2 液膜流れ

液噴流が回転している円盤の中心に落下すると，放射状に流れる境界層が発達し，境界層が液面に達したあとは液膜全体が一種のねじれ型境界層として流れる．
層流の場合にはねじれ型境界層の速度となる．
4πνωでは液膜は十分に薄くなり流れは準平行流とみなせる．
ここに，ωは円盤の角速度を表す．
このとき，液膜は遠心力のとなる．
よって，波の位相速度 は液膜が形成される．
分布を多項式近似し，運動量積分方程式を用いれば容易に計算することができる．
これについては文献を参照されたい．
波数σ（＝2π， ：周期）はで与えられる．
