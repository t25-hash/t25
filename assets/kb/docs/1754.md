# 1・1・2 定常電流と静磁界

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 )、キャパシタという。実用的には、10F(F)、10F

a. 電流(electric current)電荷の流れを電流という。ある導体の一つの断面積を1(s)に1(C)の電荷が流れるときの電流を1(A=C/s,A:アンペア)という。正電荷の進む向きを電流の向きとして、正電荷の進む向きに電流が流れるという。負電荷が進むときは、負電荷の進む向きと反対向きに電流が流れるという。金属導体においては、原子核の電子のうち、いくつかの電子が自由に移動できる。電界が加わると、電子は負電荷をもつので力が加わって移動し、移動方向と反対向きに電流が流れることになる。金属導体を流れる電流を伝導電流とも呼ぶ。電解液の中においては、正電荷をもつ陽イオンと負電荷をもつ陰イオンが移動する。陽イオンによる電流と陰イオンによる電流を加えた電流が、全体の電流となる。帯電体の移動による電流を対流電流という。伝導電流と対流電流は真の電荷の移動により生じ、真電流という。真電流は電源を用いて外部から制御できる。
