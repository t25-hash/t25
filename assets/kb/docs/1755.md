# 14・1・1 混相流の分類

> **章**: 第14章 混相流
> **節**: 14・1 基礎的概念

物質の3態である気体，液体，固体に対し，姿，形または状態を
意味する言葉，相（phase）を付けると，気相（gas phase)，液相
（liquid phase)，固相（solid phase）となる．混相流（multiphase
flow）とは，気相または液相を基質とし，その中に別の物質が基質
と界面で区別されて混在する物質の流れである．連続相（continu-
ous phase）と呼ばれる基質に対し，固体粒子，液滴，気泡などは
分散相（dispersed phase）と総称される．気体中の液膜流れや界
面で分離される2種類の液体層の流れも混相流とみなされる．混相
流は通常二つの相からなる場合が多いので，二相流（two- phase
flow）という用語がしばしば混相流と同じ意味で用いられる．粒子
が基質内で分散浮遊する場合，そのような混相流体はサスペンジョ
ン（suspension，懸濁液）と呼ばれることもある．混相流に対し，
気体または液体だけの流れは単相流（single- phase flow）と呼ば
れる．混相流は，気液二相流，固液二相流，固気二相流，固気液三
相流，液液二相流に分類される．混相流の実例は，自然界（霧，溶
岩流，砂嵐，漂砂など)，生体（血液など)，工業関連（ボイラ，流
動層，汚泥，固体のパイプライン輸送，固体ロケット，集じん機な
ど）などに見られる．
