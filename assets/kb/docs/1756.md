# 1・1・3 電磁誘導(electromagnetic induction)

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 )、キャパシタという。実用的には、10F(F)、10F

a. ファラデー(Faraday)の電磁誘導の法則図1・8に示す閉回路Cに磁束φが鎖交している。鎖交磁束φを発生するために流す電流の向きを正方向とする。鎖交磁束φが変化するとき、変化を妨げる方向に電流を流そうとして起電力が回路に誘起する。例えば、鎖交磁束φが増加するとき、負方向に起電力が誘起する。これをファラデーの電磁誘導の法則という。この関係を次式に示す。これをレンツの法則という。鎖交磁束が時間的に変化するには、磁束そのものが変化する場合と閉回路Cが速度をもって移動するために変化する場合がある。前者による起電力を変圧器起電力といい、後者による起電力を速度起電力という。この法則は、図1・9に示すように右手の親指の方向を速度、人差し指の方向を磁束方向とするときに、起電力の方向が中指の方向になるので、フレミングの右手の法則という。b. 自己インダクタンス(self-inductance)回路に流れている電流が変化するとき、回路に鎖交する磁束が変化して、それにより回路自身に起電力が誘起する。この現象を自己誘導という。自己誘導により生じる起電力(V)は次式となる。c. 相互インダクタンス(mutual inductance)二つの閉回路CとCがあるとき、Cに流れる電流により発生する磁束のうち、Cに鎖交する磁束をφとする。電流が時間的に変化すれば鎖交磁束φも変化し、Cに誘起起電力を発生する。この定数(H)を相互インダクタンスという。相互インダクタンスにおいては、=であり、相反性が成立する。ほかも同じである。
