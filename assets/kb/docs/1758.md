# 1・2・1 誘電体(dielectrics)

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 物質と電磁界

電気機器や装置において,電圧のかかる部分や電流の流れる部分を支持,冷却したり,あるいは被覆したりする機能をもつ材料として,電気絶縁材料(electrical insulation material)がある.電気絶縁材料を電界の中に置くと誘電分極などの特性を示し,誘電特性に着目する場合に電気絶縁材料は誘電体と呼ばれる.誘電体は,材料の形態から分類すると,気体・液体・固体誘電体とその複合誘電体に分類される.また,材料の組成から見ると,有機誘電体と無機誘電体に分けられる.有機誘電体にはエポキシ,ゴム,高分子フィルム,絶縁油など,無機誘電体にはセラミックス,マイカ,ガラスなどがある.a. 誘電分極(dielectric polarization)電気絶縁材料を電界の中に置くと,材料中に存在している正と負の電荷が電界方向に変位し,材料表面に電荷が現れる.このように電界の作用により材料内に電気双極子を生じる現象を誘電分極(dielectric polariza-tion)といい,分極する物質を誘電体という.誘電分極には,水のような有極性分子が外部電界によって電界方向に回転する,いわゆる双極子モーメントの配向に基づく配向分極(双極子分極ともいう),電界をかけると正電荷を有する原子核が電界方向に,負電荷をもった電子が電界と逆方向に移動する電子分極,さらに正に帯電した原子と負に帯電した原子との相対位置の変化に基づく原子分極,誘電体中に自由イオンが含まれるときイオンの移動によるイオン分極(空間電荷分極ともいう)などがある.分極ベクトルが電界と同じ向きをもち,かつに比例して発生するような材料は常誘電体(paradielectric substance)と呼ばれる.誘電体の中にはこのような通常の常誘電体のほかに,外部電界がなくても自発的に誘電分極を生じるものがあり,強誘電体と呼ばれる.強誘電体にはコンデンサ材料や圧電材料に使用されるチタン酸バリウムやロッシェル塩などがある.b. 誘電体を含む系のガウスの法則(Gauss’law in dielectrics)誘電体がある場合も,電界の保存性やガウスの法則は真空中と同じように成立し,誘電体のある系における電位・電界を求めるのにガウスの法則を適用することができる.しかし真空中と異なるのは,電荷を置くとその電荷の作る電界によって中性であった誘電体が分極し,誘電体中に新たな電荷の発生があることである.ここで,外から制御できる電荷を自由電荷(free charge)あるいは真電荷(true charge)と呼び,分極により新たに発生する電荷を分極電荷(polarization charge)という.したがって,電界に対するガウスの法則は,自由電荷と分極電荷の和の全電荷に対して成立することになる.誘電体の含まれた系に任意の閉曲面を考え,この閉曲面中の自由電荷を,分極電荷をとすると,ガウスの法則は+に対して成立し,=+ε(1・61)となる.ここに,εは真空中の誘電率である.誘電体中の分極ベ8―5電気・電子の基礎クトルから分極電荷を求めると,=-(1・62)となるので,ε+=(1・63)と表すことができる.つまり=ε+(C/m)(1・64)で表される電束密度(dielectric flux density)を用いて,=(1・65)となる.これが,誘電体中におけるガウスの法則である.誘電体中における電位・電界の分布はガウスの法則に基づいた,ラプラスの方程式・ポアソンの方程式を解くことによって求められる.c. 誘電率(dielectric constant,permittivity)分極ベクトルは一般に電界の関数であり,特に等方性の誘電体では分極ベクトルが電界に比例して発生するので,この比例関係を=χ욄ε(1・66)と書くことができる.χ욄は分極率(polarizability)あるいは,電気感受率(electric susceptibility)と呼ばれ誘電体固有の正の無次元定数である.がに比例するともに比例する.=ε+=1+χ욄ε=ε욦ε=ε(C/m)(1・67)この電界に対する比例係数εを,その材料の誘電率(dielectricconstant;
