# 1・2・1 誘電体(dielectrics)

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 物質と電磁界

電気機器や装置において、電圧のかかる部分や電流の流れる部分を支持、冷却したり、あるいは被覆したりする機能をもつ材料として、電気絶縁材料(electrical insulation material)がある。電気絶縁材料を電界の中に置くと誘電分極などの特性を示し、誘電特性に着目する場合に電気絶縁材料は誘電体と呼ばれる。誘電体は、材料の形態から分類すると、気体・液体・固体誘電体とその複合誘電体に分類される。また、材料の組成から見ると、有機誘電体と無機誘電体に分けられる。有機誘電体にはエポキシ、ゴム、高分子フィルム、絶縁油など、無機誘電体にはセラミックス、マイカ、ガラスなどがある。a. 誘電分極(dielectric polarization)電気絶縁材料を電界の中に置くと、材料中に存在している正と負の電荷が電界方向に変位し、材料表面に電荷が現れる。このように電界の作用により材料内に電気双極子を生じる現象を誘電分極(dielectric polariza-tion)といい、分極する物質を誘電体という。誘電分極には、水のような有極性分子が外部電界によって電界方向に回転する、いわゆる双極子モーメントの配向に基づく配向分極(双極子分極ともいう)、電界をかけると正電荷を有する原子核が電界方向に、負電荷をもった電子が電界と逆方向に移動する電子分極、さらに正に帯電した原子と負に帯電した原子との相対位置の変化に基づく原子分極、誘電体中に自由イオンが含まれるときイオンの移動によるイオン分極(空間電荷分極ともいう)などがある。分極ベクトルが電界と同じ向きをもち、かつに比例して発生するような材料は常誘電体(paradielectric substance)と呼ばれる。誘電体の中にはこのような通常の常誘電体のほかに、外部電界がなくても自発的に誘電分極を生じるものがあり、強誘電体と呼ばれる。強誘電体にはコンデンサ材料や圧電材料に使用されるチタン酸バリウムやロッシェル塩などがある。b. 誘電体を含む系のガウスの法則(Gauss’law in dielectrics)誘電体がある場合も、電界の保存性やガウスの法則は真空中と同じように成立し、誘電体のある系における電位・電界を求めるのにガウスの法則を適用することができる。しかし真空中と異なるのは、電荷を置くとその電荷の作る電界によって中性であった誘電体が分極し、誘電体中に新たな電荷の発生があることである。ここで、外から制御できる電荷を自由電荷(free charge)あるいは真電荷(true charge)と呼び、分極により新たに発生する電荷を分極電荷(polarization charge)という。したがって、電界に対するガウスの法則は、自由電荷と分極電荷の和の全電荷に対して成立することになる。ここに、εは真空中の誘電率である。これが、誘電体中におけるガウスの法則である。誘電体中における電位・電界の分布はガウスの法則に基づいた、ラプラスの方程式・ポアソンの方程式を解くことによって求められる。χは分極率(polarizability)あるいは、電気感受率(electric susceptibility)と呼ばれ誘電体固有の正の無次元定数である。またεに対するεの比εを比誘電率という。表1・1におもな誘電体の比誘電率の値を示す。誘電体の比誘電率はつねに1よりも大きく、液体や固体の場合には真空中の誘電率の数倍の値になっていることがわかる。この比誘電率の値は、誘電体が満たされたコンデンサの静電容量(キャパシタンス)を測定することにより求められる。d. 2種の誘電体の境界(dielectric boundary)誘電体の境界面においてガウスの法則を適用すると、以下の二つの条件が成立することがわかる。(1) 二つの誘電体の境界面における電界の接線成分は等しい。(2) 二つの誘電体の境界面上における電束密度の法線成分は等しい。このことから、図1・15に示すように誘電率ε,εの二つの誘電体が電荷のない界面で接しており、点Pで誘電体(1)側の電界がで、その法線方向のなす角度がθであるとき、点Pにおける誘電体(2)側の電界と法線方向のなす角度θとは以下のように求まる。
