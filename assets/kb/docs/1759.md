# 1・2・2 磁性体(magnetics)

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 物質と電磁界

物質のもつ磁気的性質や磁気現象から見るとき、その物質を磁性体と呼び、磁界の影響で磁気双極子モーメントをもつ現象を物質の磁化(magnetization)という。外部から磁界をかけると磁性体の内部の微視的磁気モーメントを強める方向になるものを常磁性体(paramagnetic substance)といい、外部磁界を打消すものを反磁性体(diamagnetic substance)という。常磁性体には白金、アルミニウムなどがあり、反磁性体には銅、銀、水などがある。これに対して、鉄、コバルト、ニッケルなどは大きな磁気モーメントを有しており、強磁性体(ferromagnetic substance)と呼ばれる。これは、磁区と呼ばれる領域内で磁気モーメントのもとになる電子スピンが一方向に整列するためといわれている。外部の磁界がなくても磁化がもとのままで保持され、その方向に強い磁界を発生させるものが、磁石である。磁性体は実用的に、変圧器、モータなどの磁心材料や磁石材料、あるいは磁気記録やセンサなどに応用される特殊磁気応用材料として重要である。a. 磁気に関するクーロンの法則と磁界(Coulomb’s law inmagnetics)磁石の磁極近傍に磁荷というものを考え、磁荷の間に力が働くものとしてこの力を測定することにより、次の磁荷に対するクーロンの法則が導かれた。また、ここでμ=4π×10(H/m)を真空の透磁率という。図1・15 二つの誘電体の境界図1・16 磁気双極子図1・17 棒磁石周辺の磁束と磁力線の分布と表される。このように、電荷に対するクーロンの法則と同じ形の法則が磁荷についても成立するので便利であるが、真電荷に相当する真磁荷が存在しない点に注意を要する。b. 磁気モーメント(magnetic moment)磁気の場合には電荷の場合と違って真磁荷に相当するものがなく、つねに正・負磁荷が対で現れるので、単独磁荷で扱うより双極子で扱うほうが現実的である。多くの物質では外部から磁界を作用させないと磁化は零で、磁界が十分小さいとはに比例する。ここでχは物質に固有の定数であり、その物質の磁化率あるいは磁気感受率(magnetic susceptibility)という。χ>0の物質を常磁性体、χ<0の物質を反磁性体という。また、外部から磁界をかけなくても磁化が自然に発生しているような物質を強磁性体といい、鉄・コバルト・ニッケルなどがこれに相当する。c. 磁束密度(magnetic flux density)と透磁率(permeability)磁性体の場合、電気の電束密度に対応するものとして、磁束密度を次のように定義する。μ=1+χμ(1・81)μは多くの場合μより大きいが、反磁性体ではμ以下の非負の数となる。図1・17に棒磁石内外の、(a)磁力線分布、(b)磁束分布を示す。磁束は閉じた曲線になるため、磁性体内部では磁界と逆向きで磁化と同じ向きになる。d. 強磁性体(ferromagnetics)鉄、コバルト、ニッケルなどの強磁性体は磁性体の中でも最も重要なものであり、外部から磁界をかけなくても磁化が自然に発生している物質である。強磁性体は外部からの磁界に対しては、式(1・79)で示したものとは全く異なった挙動を示す。図1・18に強磁性体に磁界をかけたときの磁化の変化の様子を示す。このように磁界に対して履歴現象を示すことをヒステリシス(hysteresis)現象という。特徴的なことは、磁界に対して磁化が飽和すること、磁界を減らして=0としたときに自発磁化に対応する残留磁化(OB)を有すること、そして磁化を零にするためには、保磁力と呼ばれる逆向きの磁界(OC)をかける(b) 磁束分布(a) 磁力線分布図1・18 強磁性体の磁化ヒステリシス図1・19超伝導現象
