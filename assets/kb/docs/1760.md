# 14・2・1 流動様式（flow pattern）の定義

> **章**: 第14章 混相流
> **節**: 14・2 気液二相流

気液二相流（gas - liquid two- phase flow）では気相と液相が混
在して流れ，いずれもいかようにも変形できるので，その流動様相
は気液それぞれの流量によって複雑に変化する．また圧力損失，ボ
イド率（void fraction）あるいは熱伝達などが流動様式に強く依
存しており，問題にしている流れ場がどの様式であるかをできるだ
け精度良く予測することが必要である．以下では主として従来多く
の研究で対象とされた内径20mm程度の管についての基本的な様
相について述べる．
a.鉛直管内上昇流（図14･1)
気泡流（bubble flow）：連続した液相中に管径より十分小さい気
泡が分散した流れ．
スラグ流（slug flow）：管径より長い砲弾状の大気泡と気泡流状
の液体スラグが交互に流れる脈動性の強い流れ．
フロス流（froth flow）：チャーン流とも呼ばれている．液相と
気相の流量がともに大であると，液体スラグ部は多くの小気泡が含
まれて白濁する．その白濁した液体が大気泡のまわりに厚い液膜と
なって流入するため，全体としても泡立った様相をして高速で流れ
る．脈動性が最も強い流れである．

b.鉛直管内下降流 基本的には上昇流と変わりがない．しか
し，例えば管壁を液膜が下降する液膜流の場合，上昇流のように気

c.水平管内流（図14･2) 気液二相流では流動系の圧力が臨
界圧力に近い，あるいは微小重力の場のような場合を除けば，気液

って，水平管では液相は管の低部に，気相は管頂部に偏って流動
し，そのため鉛直管内の流動様式とは異なる．研究者によって異な
る呼称を用いているが，以下では重力場で見られる一般的なものに
従って説明する．
気泡流（bubble flow）：連続した液相中に管径より十分小さい気
泡がいくぶん管頂部に偏って流れる流れ．
プラグ流（plug flow）：管軸方向に長い大気泡が管頂部に遍在す
る流れ．
層状流（laminate flow）：液相は管底部を，気相は管頂部を層状
に分離して流れる流れ．気液界面は鏡面状に平滑である．
