# 1・2・4 超伝導体(superconducting material)

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・2 物質と電磁界

超伝導とは、多くの金属や合金などの材料が、ある臨界温度以下で直流の電気抵抗が零になる完全導電性と、ある臨界値以下の磁界では超伝導体内から磁束を排除するマイスナー効果で代表される特性である。オランダのカマリン・オンネス(H. KammerlingOnnes)が1911年に金属水銀を冷却し、4.2K で抵抗が零になることを発見し、これを超伝導と名付けた。その後、金属元素、合金、金属間化合物などを中心に数百種類以上の超伝導材料が知られるようになっている。超伝導は図1・19に示すように、臨界温度と呼ばれる材料固有の温度で発現する。一方、超伝導体に外部から磁界を加えると、臨界磁界と呼ばれる磁界の強さで超伝導が破壊される。また、電流密度が一定値以上になると超伝導が破壊され、この電流密度を臨界電流密度と呼ぶ。この状況を図1・20に示す。超伝導体は第I種と第II種に分類され、それぞれ磁界に対して異なった挙動を示す。金属元素の大部分は第I種で、臨界磁界が比較的低く、超伝導状態での抵抗は完全に零とみなされる。これに対し8―7電気・電子の基礎図1・20 超伝導特性と臨界値曲線て、第II種は合金や金属化合物が多く、下部臨界磁界までは完全反磁性を保つが、それ以上の磁界のもとでは超伝導体内に量子化された磁束が侵入し、完全反磁性の状態は失われる。しかし、非常に高い上部臨界磁界まで超伝導状態が保たれるので、エネルギー分野などへの応用に適している。現在20K 以下の低温で実用化されているNb-Ti合金や、NbSnなどの超伝導線材では10A/cm以上の高いが達成されている。一方、銅酸化物超伝導体(高温超伝導体)は、1986年にBednorzとMullerらが、La-Ba-Cu-O系の酸化物が35K で抵抗が下がり始めることを見出し、反磁性を示すことが確認された。その後、1987年にY-Ba-Cu-O系の酸化物が液体窒素温度77K で完全に抵抗零になることが見出された。これまで百種類以上の化合物が発見されている。液体窒素温度での使用が考えられることから、応用技術が飛躍的に進展しつつある。電気回路は、いくつかの素子が接続してできている。電気回路を解くとは、その素子の電圧や電流を求めることである。そのとき、回路素子の電圧電流の関係式、回路の電圧の関係式、電流の関係式を用いて解くこととなる。後者二つの関係式は、素子の接続関係で決まる。さて、素子の定数を集中的に扱うことが可能な場合と分布的に扱わなければならない場合があり、前者を集中定数回路(lumpedconstant circuit)、後者を分布定数回路(distributed constant cir-cuit)と呼ぶ。また、素子の電圧・電流特性が線形である場合と非線形である場合があり、前者を線形回路(linear circuit)、後者を非線形回路(nonlinear circuit)と呼ぶ。また、エネルギーを消費または貯蔵する素子からできている回路を受動回路(passive cir-cuit)、入出力に関してエネルギーを供給する素子を含む回路を能動回路(active circuit)という。まず、線形集中定数受動回路をもとに回路を解くことを考える。その後、分布定数回路を考える。
