# 1・3・2 分布定数回路(distributed constant circuit)

> **章**: 第1章 電気・磁気の基礎
> **節**: 1・26 に示す例題では、直列接続も並列接続もない。この場合はい

集中定数回路において、考えている素子はその大きさを無視している。すなわち素子が1点に集中していると考えている。実際は大きさをもつものであり、回路素子実数が分布していると考えなければならない。このような回路を分布定数回路と呼ぶ。回路で取扱う周波数が低いときは、集中定数回路で簡単に解析すればよいが、高いときは分布定数回路で解析しなければならない。送電線や通信線を分布定数回路としてその解析を考える。図1・34のような伝送線路は右図のように集中回路の集まりで接続されていると近似する(線路に電流が流れると磁界が生じ、それを表すインダクタ。線路に電圧をかけると電荷の分極が生じ、それを表すキャパシタ。各々損失を表す抵抗からなる。精密な解析をする場合は電磁界解析が必要となる)。θ=α+βとおき、αを減衰定数、βを位相定数と呼ぶ。これを無損失線路と呼ぶ。図1・33等価回路図1・32 相互インダクタンス図1・34 分布定数回路の表現8―11電気・電子の基礎例題として図1・35の分布定数回路を含む回路(長さ)を考える。
