# 2・1・1 半導体の基礎

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・1 半導体デバイス

a. 半導体材料半導体材料には、元素半導体と化合物半導体とがある。前者は周期律表の第4属元素からなるSi,Geなどであり、これらの元素が集まった結晶はダイヤモンド構造である。それぞれ隣り合った原子は1個の電子を出し合い電子対を形成し、共有結合で結ばれている。後者はGaAs,InP,InSb,GaN などのように3属元素と5属元素とを交互に配列したもの(III-V属半導体)、CdS,ZnS などのように2属元素と6属元素とを交互に配列したもの(II-VI属半導体)、SiC,SiGeなどのように4属元素どうし交互に配置したものなど多くの種類がある。これらの組合せでは両元素のもつ価電子を互いに共有し、ダイヤモンド構造と同様な構造を作っている。今日の大規模集積回路(LSI,VLSI,ULSI)の主役はSiであり、超高速集積回路ではSi,SiGe,GaAs,InPなどであり、光半導体デバイスではGaAs,InP,GaN などが用いられている。b. 電気的性質不純物が添加されていないものを真性半導体、不純物が添加されているものを外因性半導体という。4価のSi原子の一部をAsなどの5価の原子で置換すると、Siの4本の結合手に対しAsの結合手が1本余る。この過剰電子が室温で伝導電子(自由電子)となり、そしてAsはAsイオンとなる。このような不純物をドナーという。SiではAs,P,Sbなどがドナーである。これと逆に、B などの3価の原子で置換すると、電子が1個不足し正孔ができ、B はBイオンとなる。このような不純物をアクセプタという。SiではB,In,Ga,Alなどがアクセプタである。ドナーを有するものをn型半導体、アクセプタを有するものをp型半導体という。以上述べた電子、正孔をキャリヤといい、数の多いものを多数キャリヤ、少ないものを少数キャリヤという。半導体では不純物原子が100%活性化していると仮定すると電荷中性条件が成り立ち、外因性半導体では+욃=+욀となる。ここで、:電子濃度、:正孔濃度、욃:ドナー濃度、욀:アクセプタ濃度である。真性半導体では=욡が成り立っている。半導体中のキャリヤは電界により加速されるが、熱振動している格子や不純物原子により散乱(前者:フォノン散乱、後者:不純物散乱)されるため、ある定常値욃となる。これをドリフト速度といい、電界が弱い場合は욃はに比例し、욃=-μ욄なる関係が成立する。このμ욄を電子の移動度という。正孔についても同様でありμ욠を正孔移動度という。半導体中を流れる電子電流、正孔電流はそれぞれドリフト電流と拡散電流の和で与えられる。욄=μ욄+욄dd(2・1)욠=μ욠-욠dd(2・2)ここで욄:電子拡散定数、욠:正孔拡散定数、拡散定数と移動度との間には욄=μ욄,욠=μ욠なる関係が成り立ち、これをアインシュタインの関係という。:温度、:電子電荷、:ボルツマン定数である。実際の半導体の各時刻ごとについて見ると、伝導体の電子と価電子帯の正孔との再結合過程が頻繁に生じ、また逆の発生過程も頻繁に生じており、熱平衡状態では両者の割合が釣合っている。一般に半導体中の電流は、この発生、再結合の過程をも考慮した、少数キャリヤに対する連続の方程式を解くことにより求められる。
