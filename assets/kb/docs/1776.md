# 2・2・2 増幅回路(amplifier)

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・10 にこれらの回路の信号を示す。

増幅回路とは、電圧、電流あるいは電力を大きくする回路である。増幅の割合が利得で、電圧利得、電流利得、電力利得がある。トランジスタにはpn接合トランジスタと電界効果トランジスタ(FET)がある。pn接合トランジスタは少数キャリヤと多数キャリヤの両方を用いた構造のバイポーラトランジスタ(BJT)が最もよく使われる。電界効果トランジスタにはJFET(junctionFET)、MOSFET,MESFET がある。シリコンLSIでよく使われるBJT とMOSFET について述べる。a. バイポーラトランジスタ基本増幅回路。 エミッタ接地増幅回路図2・11(a)にエミッタ接地増幅回路を示す。2・1節で説明したようにBJT の動作原理はベース電流の微小な変化を増幅して、コレクタ電流を大きく変化させることである。この増幅回路の電流利得は電流増幅率βにより決まる。入力のベース端子からは順方向のpn接合が見えるので、入力インピーダンスは低い。また、出力のコレクタ電圧の変化によるコレクタ電流変化は少ないので、出力インピーダンスは高い。したがって、大きな負荷抵抗を接続することができ、電圧増幅の機能ももたせることができる。このように電流と電圧の増幅作用をもつので電力利得も大きい。入力と出力の位相は逆相になる。. コレクタ接地増幅回路図2・11(b)にコレクタ接地回路を示す。出力電圧はベース側の抵抗とエミッタの負荷抵抗の抵抗分割で決まり、電圧利得は1より少し小さな値になる。エミッタ電圧がベース電圧に追従して動くので、エミッタフォロワとも呼ばれ、バッファ増幅器としてよく用いられる。. ベース接地増幅回路図2・11(c)にベース接地増幅回路を示す。入力はエミッタに加え、出力はコレクタから取出す。電流増幅率はベース接地電流増幅率αで決まり、ほぼ1である。この回路の電圧利得は入力インピーダンスと負荷インピーダンスの比で決まる。また、入力と出力は同相である。したがって、出力から入力への内部帰還量は小さいので発振しにくく、高周波まで安定に動作する。これらのバイポーラデバイスを用いた増幅器の利得を表2・3にまとめて示す。b. MOS 基本増幅回路。 ソース接地増幅回路ソース接地増幅回路を図2・12(a)に示す。ゲートに加えられる入力によってドレイン電流を変化させるので、駆動n-MOS と呼ばれる。負号は出力信号が逆相になることを表している。しかし、負荷抵抗を大きくすると、電圧降下が大きくなるので電源電圧を高くする必要がある。負荷抵抗の代わりにカレントミラーを用いると図2・12(b)となる。これをアクティブロードと呼ぶ。ここまでの解析は直流あるいは低周波信号に対する特性であるが、高周波特性を解析するには寄生容量を追加する必要がある。. ドレイン接地増幅回路回路を図2・13に示す。ソースフォロワとも呼ばれる。このように、電圧利得はたかだか1であり、増幅作用はないが、が低いのでインピーダンス変換回路あるいは高駆動力の出力回路として使われる。接続されたエミッタは高抵抗を通してグランドに接続される。差動信号に対して各トランジスタの電流は、一方が増加すると他方は減少する。したがって、共通エミッタの電圧は変化しないので、差動信号に対する等価回路は図2・14(b)に示すようなエミッタ接地となる。また、温度や電源電圧の変動によって生じる信号変動は差動回路に対して同一方向である。図2・14(c)に同相信号に対する等価回路を示す。同相信号はできるだけ増幅したくないのでは小さいほうが望ましい。/を同相除去比(CMRR : common mode rejectionratio)と呼ぶ。CMRR は差動増幅回路の重要な性能指数である。d. MOS 差動増幅回路MOS 差動増幅回路を図2・15(a)に示す。CMOS では図2・15(b)に示すアクティブ負荷を用いた差動増幅回路がよく用いられる。この回路は差動信号をシングル信号に変換する機能をもっている。e. 負帰還増幅回路図2・16に示すように出力を入力に戻すことを帰還と呼び、帰還をかけた増幅器を帰還増幅回路と呼ぶ。逆相出力を帰還するのを負帰還、同相出力を帰還するのを正帰還と呼ぶ。負帰還増幅回路では帰還をかける前の利得を、帰還回路の伝達関数をとすると、増幅回路に加わる差信号(入力-出力×)は倍されて出力となる。. 利得の安定化は一般に抵抗などの受動素子の比で決まるので、より大幅に安定となる。. ひずみの低減増幅回路の線形性もほぼの線形性で決まるので、のひずみは低減される。負帰還増幅回路で重要なことは帰還ループの安定性である。帰還前の増幅回路の位相は周波数とともに遅れるので、正帰還になって発振を起こす。これを防止するために位相が回転しないよう位相補償を行う。f. 演算増幅器と演算回路演算増幅器(OPA : operationalamplifier)はアナログ集積回路として最もよく使われ。すなわち、理想OPAは電圧制御電圧源を用いると、図2・17(b)の等価回路で表される。. 演算増幅器を用いた基本回路(1) 逆相増幅回路図2・18(a)の,を抵抗,とした回路を逆相増幅回路という。これは図2・18(a)に示すようなてこのモデルで考えると理解しやすい。てこの支点が仮想接地点であり、てこの長さが抵抗値に対応している。(2) 正相増幅回路図2・18(b)を正相増幅回路という。てこのモデルでは、左端が支点になり、入出力は同相である。(3) 加算回路(adder)図2・19(a)は多入力加算回路である。図2・19(b)は減算回路である。(4) 積分回路(integrator)図2・19(c)に積分回路を示す。. 演算増幅器(operational amplifier)の内部回路図回路、ソース接地回路の二段で構成されている。利得は70dB 程度、0dB 帯域は10~50MHz 程度である。
