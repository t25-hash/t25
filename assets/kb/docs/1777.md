# 14・1・3 連成効果が小さい場合の電磁力と振動

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・1 電磁力による振動の概要

磁場中においても磁場と電流との相互作用によりローレンツ力が生じ構造物を変形させる．
ここで は電流密度（A/ m)， は磁束密度(Ｔ)である．
ここで，μは真空の透磁率（H/m)， は磁化（A/m)， は磁場（A/m）である．
電磁力を利用している発電機やモータなどの回転機や，磁気浮上，電磁力が作用している磁場中で使われる超伝導マグネットなどでは，ローレンツ力および磁気力による変形や振動を考慮して構造物の挙動を評価する必要がある．
磁場中に置かれた構造物や機器の振動現象のうち，変形と磁場との連成，特に磁気剛性，磁気減衰を考慮する必要性のある場合については14･2節で，また，さらに高温超伝導浮上系の場合のように，非線形性が強い場合については14･3節で記述する．
発電機やモータなどの回転機では，磁場中において電流が流れるが，磁気剛性効果や磁気減衰効果を考慮する必要性は少ない．
電磁力解析により得られた電磁力を振動外力として応答解析している例が多い．
回転機は小型化，軽量化の要求のため，構造全体の剛性が低下する傾向にある．
さらにインバータ運転では，電源に高調波成分が多く，周波数が変化することから構造系の固有振動数との共振の可能性がある．
ステータ（固定子）は近似的に円環（リング）の一種と考えられる．
おもに半径方向に作用する電磁力が構造系の円環の固有振動 を励起するため，電磁力に対する共振の回避が設計の課題である点は，発電機，モータについて共通である．
