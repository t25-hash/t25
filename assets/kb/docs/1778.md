# 2・2・4 フィルタ

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・20 に簡単なCMOS 演算増幅器の回路を示す。CMOS 差動増幅

a. フィルタの機能フィルタには低域通過フィルタ(LPF:low-pass filter)、高域通過フィルタ(HPF: high-pass filter)、帯域通過フィルタ(BPF: band-pass filter)、帯域阻止フィルタ(BEF)があり、周波数選択の機能をもつ。また、波形を整形するフィルタも用いられる。b. RCアクティブフィルタRCの積分回路は最も簡単なLPFである。しかし、RC受動回路のQは小さいので、LCフィルタのような急峻な特性は得られない。RCアクティブフィルタ(active filter)の原理はRC受動回路に増幅器を加えて負帰還や正帰還をかけることにより、高いQを実現することである。代表的な回路として、図2・27に示すOPA により正帰還をかけた回路がある。これらの素子を集積化するには大きな面積が必要である。また、,の値の製造偏差をそれぞれ±15%とすると、周波数は±30%の偏差をもつ。図2・25 ∑型A-D 変換器図2・26 A/D 変換LSIの性能c. スイッチトキャパシタフィルタ(SCF: switched capacitorfilter)。 スイッチトキャパシタによる等価抵抗静電容量(C)、アナログスイッチおよびOPA で構成されるので、MOS LSIに適したアナログフィルタである。周期の二相のクロックφ,φで駆動される2個のスイッチS,Sと容量を図2・28(a)のように接続した回路をスイッチトキャパシタ(SC)回路と呼ぶ。この電流はパルス的に流れるが、クロック周波数が信号周波数に比べて十分高いとして時間平均値で考えれば、抵抗を流れる電流と等価である。SC回路のスイッチとクロックドライバは図2・28(b)のCMOS回路で実現され、φ,φにはスイッチオンの期間が重複しない二相クロックが用いられる。重複があると信号電荷が放電されるおそれが生じる。スイッチは小さいMOSFET で実現できるので、抵抗素子に比して1/100程度の小面積ですむ。. 基本回路RC積分器のR をSC回路で置換えると図2・29のSC積分器が得られる。. 一次SCフィルタSC積分器の積分容量に並列にSC回路を接続すれば、一次のLPFあるいはHPFが得られる。これらを縦続接続すれば高次のフィルタが得られる。集積回路の容量素子の絶対値は±10%以上の製造偏差をもつが、比精度は±1%以下で温度変動も小さく高精度である。また、周波数精度の高いクロックは水晶発振器で容易に得られる。したがって、SC積分器は無調整で高精度を実現しうる。またを変えてτを制御することもできる。電源雑音除去比(PSRR)は電源に加わる雑音電圧と出力電圧の比で定義される。電源雑音が信号に漏れる経路を減少させるため、信号線のシールド対策が必要である。差動型の積分回路でSCフィルタを構成すると、外部雑音は同相成分となるので差動出力信号としては現れない。d. -フィルタOTA(operational transconductanceamplifier)は差動電圧を入力とし、電流を出力するアンプである。入力抵抗が無限大、出力は電流源である。OTA の出力にを接続することにより積分回路を実現できる。OTA は差動増幅回路で実現できる。この積分器を単位回路として帰還回路を構成すると、任意の電圧伝達関数を実現できる。OTA は電流出力でありOPA に比べて回路が簡単であるので、消費電力が小さく、広帯域という利点をもつ。100kHz~100MHz程度の高周波のフィルタに適している。e. 各種フィルタの性能図2・31に各種フィルタの周波数との実現領域を示す。数GHz 以上の超高周波のマイクロ波モノリシックIC(MMIC)では分布定数素子による受動フィルタも用いられる。
