# 2・2・5 高周波回路

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・20 に簡単なCMOS 演算増幅器の回路を示す。CMOS 差動増幅

1GHz 以上の周波数の無線通信用回路であり、低雑音増幅器、ミキサ、VCOなどがあり、CMOS 技術の高周波化によって、ディジタル回路を含めて1チップ受信器、トランシーバを目指してシステムオンチップ(SoC: system on a chip)化が進んでいる。標準的な0.25 mCMOS プロセスにおいて、デバイスのは10~20GHz 程度であるので、1~2GHz 帯の狭帯域LNA を実現できる。狭帯域LNA は不要な周波数を抑圧することができる。図2・32に示すようにインダクタをソースに接続した増幅器回路が有効である。この回路では抵抗素子を使用しないでインピーダンス整合がとれるので、高いNFを得ることができる。NFはインダクタのQ によって依存する。インダクタの形状設計は、実験式で初期値を求め、電磁界シミュレータなどで詳細な特性を解析して最適化する。b. ミキサRF信号から中間周波数(IF)へ周波数変換するためにミキサ(down-conversion mixer)が用いられる。集積回路では図2・33に示すギルバートの乗算器を用いたダブルバランスミキサ(DBM)が用いられる。通常のミキサでは後者の成分を利用する。ミキサの特性として入力周波数成分,に対して、±2・が帯域内に入る場合、三次ひずみが問題になる。ほかにローカル信号のリーク特性、ローカル信号と入力図2・32 LNA図2・33 ダブルバランスミキサ(DBM)図2・34 VCOとオンチップインダクタのアイソレーション特性が問題になる。c. 電圧制御発振回路(VCO: voltage controlled oscillator)VCOは無線通信以外にマイクロプロセッサやA-D 変換器のクロック源、ディジタル通信のクロック再生などに広い用途をもっている。LC共振回路を用いたVCOの回路図を図2・34に示す。CMOSで発振周波数1~10GHz、周波数可変範囲100MHz程度が実現されている。発振出力の特性で重要なのはジッタであり、周波数や位相の揺らぎを表している。また、同様な特性の評価として位相雑音(信号周波数成分とそれ以外の雑音周波数成分の電力比)を用いる。発振周波数付近に雑音成分が大きく分布しており、発振周波数1GHz で100kHz 離れた成分が-100dB(電力で10)程度のVCOが開発されている。この位相雑音は、発振振幅、発振周波数、インダクタンスのに依存する。
