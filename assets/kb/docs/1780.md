# 2・2・6 アナ・デジ混載集積回路(hybrid IC)

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・20 に簡単なCMOS 演算増幅器の回路を示す。CMOS 差動増幅

集積規模の増加に伴って、アナ・デジ混載のシステムオンチップが可能になり、開発が活発になっている。通信、マルチメディア、ロボットなどに適用される。アナ・デジ混載LSIの最初の例はCMOS 技術を用いた音声用のA-D,D-A 変換器(PCM-CODEC)である。オーバサンプリングA-D,D-A 変換器を用いている。携帯電話の音声の圧縮処理にA-D,D-A 混載ディジタルシグナルプロセッサが用いられる。また、携帯無線用ベースバンドLSI、中間周波数LSIもアナ・デジ混載集積回路の例である。また、500MHzのサンプリング周波数の高速のA-D 変換器を混載した記憶装置用のリードチャネルLSIも開発されている。このような大規模なアナ・デジ混載集積回路を設計するには、ディジタル回路からアナログ回路に漏れる雑音を抑圧することが重要である。電源とグランド線を介する雑音、集積回路の基板を通した雑音を抑圧する技術が重要である。このために、クロストーク雑音の解析技術、測定技術が開発されている。
