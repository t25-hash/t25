# 14・3・2 高温超電導浮上系の非線形振動

> **章**: 第14章 電磁力と構造系の連成振動
> **節**: 14・3 電磁力が関連する非線形振動

高温超電導浮上系の場合は，上述の常電導浮上系とはかなり異な
る非線形特性を有するので，以下にその特徴を述べる．
a.ピン止め力 超電導体には第Ⅰ種と第Ⅱ種があり，第
Ⅱ種の超電導体を液体窒素で冷却し（77K)，その上に磁石を置く
と軸方向および軸直角方向の2方向に浮上力が発生する．冷却温度
が第Ⅰ種の液体ヘリウム温度に比べて高いので，第Ⅱ種は高温超電
導体と呼ばれている．この浮上力はピン止め力と呼ばれ，第Ⅱ種の
超電導体にその上部，下部臨界磁界の中間の磁界を印加したときに
得られる浮上力である．現在第Ⅱ種の超電導体として，YBCuO
のかなり大きなバルク材の製造が可能となった．
高温超電導体のピン止めの原理は，超電導体を臨界温度以下に冷
却すると，超電導体の部分は超電導状態に相転移する．しかし，超
電導体に格子欠陥などの非超電導部（ピン止め点）があると，その
部分では，磁束が貫通する．磁束をピン止め点から外すためにはピ
ン止め点周囲の超電導状態を壊さなければならないので，力が必要
となる．この力がピン止め力である．
高温超電導体を磁界中で冷却すると，超電導体の欠陥部分に磁束
