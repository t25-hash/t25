# 2・3・4 ディジタル信号処理回路(digital signal processing)

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・3 ディジタル回路(digital circuit)

現実の世界で生じている信号はアナログ信号である。アナログ信号にフィルタリングやフーリエ変換のような処理を加えて利用することは重要である。信号処理の内容は多岐にわたり、アナログ回路だけで実現するのは、困難である。アナログ信号をA-D 変換器を用いて、ディジタル信号に変換し、処理を行うと、処理はすべて加減乗除の算術演算によって行われるため理想的にはいかなる信号処理も行える。処理結果を現実の世界に戻すには、D-A 変換を行えばよい。アナログ信号のディジタル処理はディジタルLSIの高集積化および高性能化とともにより広く使われるようになっている。ディジタル処理は算術演算を必要とするので、通常のCPU を使うことで可能である。しかし、高性能化のためディジタル処理専用のプロセッサが開発された。これをDSP(digital signal proces-sor)という。信号処理の演算の基本形は、次のような積和演算である。욈욪욅욡inout=∑ここに、inはディジタル化された入力信号の時系列であり、個の信号を処理対象とする。outはディジタル信号処理された出力信号の時系列である。係数を変えることで、フィルタリング(filtering)やフーリエ変換(Fourier transform)などの信号処理が可能となる。例えば、離散フーリエ変換(DFT : discrete Four-ier transform)の場合욅욡=욗요となる。この係数はサイクリックに変化し、同じ値をとる場合が多いので、上の式の積和演算の回数を減らすことができる。この計算法を高速フーリエ変換(FFT : fast Fourier transform)と呼ぶ。DSPはこの積和演算を高速で実行するハードウェアを内蔵している。信号処理の内容により演算器の使用方法を変えるので、プログラムが必要である。また、実時間で処理を行うには、DSPは高い周波数で動作しなければならない。通常のCPU にDSP機能を付加したCPU も発売されている。この場合、信号処理の融通性はさらに増す。
