# 2・4・1 マイクロプロセッサ(microprocessor)の動向と概要

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・4 マイクロプロセッサと応用

半導体の微細化に伴い、素子の高速化および数百万ゲートもの多くのトランジスタを集積できるようになった結果、図2・57に示すように、マイクロプロセッサは並列処理などの高性能化技術を駆使したパーソナルコンピュータ、ワークステーションなどに向けた高性能機種と、多くの機能を集積しメカトロニクスなど機器の制御に適した周辺回路などの多機能を内蔵した機種が出現している。前者は8ビットから64ビットのアーキテクチャへの拡大、大容量のキャッシュメモリ搭載、フローティング演算プロセッサ内蔵、スーパスカラと呼ぶ高性能化アーキテクチャの採用、さらにはVLIW(very long instruction word)と呼ぶ並列度の高い命令処理方式の開発により、1000MIPS(million instructions per second)を超え、1~2GHzで動作する高性能機種が次々に開発されている。一方、後者はメカトロニクスなど機器制御に不可欠なシリアル、パラレル・インタフェース、A-D,D-A 変換器、タイマなどの周辺機能や、プログラム、データを収容するメモリの内蔵などシステムの1チップ化を目指すものである。マイクロプロセッサが登場してきて10年程度の間は、マイクロプロセッサや各種周辺LSIが個別のチップで提供されていたが、半導体の集積度が向上するに伴って、周辺機能の多くがマイクロプロセッサと同じチップ上に取込まれてきた。このような方向に進んできたが、周辺機能の集積というだけでなく、この機種のマイクロプロセッサにおいても高性能化は顕著である。図2・58に示したように、特に、マルチメディア機器の登場により、ゲーム機などの家庭用デスクトップ機、携帯情報端末、携帯電話機などの民生分野におけるマイクロプロセッサの進歩は目覚ましいものがあり、この高性能化は産業分野で活用されるマイクロプロセッサにも波及している。民生・産業分野のマイクロプロセッサの性能は、ワークステーションやパーソナルコンピュータ用のそれに匹敵してきており、限られたアプリケーションでは凌駕(りょうが)しているものさえ出現している。図2・57 マイクロプロセッサの動向図2・58 マイクロプロセッサの性能動向図2・59 SoC概要8―31電気・電子の基礎これら制御用のマイクロプロセッサも8ビット機から32ビット機へと進展し、性能も100MIPS を超えるものも出現してきた。1970年代後半に制御用として登場したアメリカDEC社のVAX 11/780という画期的な32ビットコンピュータに対して、現在のものは10~100倍の性能向上を果たしている。この結果、これまでのマイクロプロセッサの性能の低さから活用できなかったような高度な制御アルゴリズムを取入れたり、高性能を生かした新たな応用開発も期待できるようになった。さらに、このようなマイクロプロセッサの性能向上に加えて、最近では個別のユーザの必要とする、すなわちアプリケーションに最適な多くの機能を選択的に1チップに集積するSoC(system-on-chip、システムオンチップ)技術が脚光を浴びており、よりシステムに最適な、高性能なチップを手に入れることができるようになっている。以下にその概要について示す。
