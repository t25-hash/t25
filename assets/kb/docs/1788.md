# 2・5・1 表示デバイスの種類と動向

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・5 表示デバイス

表示デバイスは,人と各種機器とのインタフェース(man-machine interface)の役割を担うデバイスであり,ディスプレイ(display)という呼び名が定着している.代表的なディスプレイの一つであるCRT は,1930年代のテレビ放送開始に合わせて実用化された.以来,技術革新を重ねて完成度が高まり,テレビ受像器やコンピュータ端末のほか,多くの分野で普及している.その一方で,CRT では実現が難しい薄型化,軽量化,大画面化,高精細化などに対応できる新しいディスプレイデバイスの開発も進められてきた.比較的最近になってこれらの技術が大きく進展し,現在では液晶ディスプレイやプラズマディスプレイをはじめとする,表示方式,画面サイズ,表示容量,形状などが異なる多様なディスプレイが市販されるようになっている.これにより,携帯,車載,娯楽などをキーワードとする新たな分野での用途も広がっている.図2・72に,代表的なディスプレイデバイスを表示方式別に分類して示す.併せて,一般的に用いられている略称も同図中に示した.ディスプレイは表示方式により,直視型,投写型,および空間像型に分類することができる.直視型のディスプレイはさらに,それ自身が発光する発光型と,バックライトや周囲光などを利用して表示を行う非発光型に分類される.投写型ディスプレイはスクリーンに画像を拡大投写するタイプであり,直視型に近い感覚で使用できる背面投写型(リアプロジェクタ)と,設置が容易で画面サイズにも自由度のある前面投写型(フロントプロジェクタ)に分類される.その他,立体的な表示についても,各種の手法が提案・開発されている.図2・73には,現在商品化されているディスプレイの表示容量(画素数)と画面サイズを示した.
