# 15・2・2 平面波

> **章**: 第15章 波動・音響
> **節**: 15・2 音響

平面波では，波の進行方向を 軸にとれば波面は 平面に平行ける角周波数と周期の対応関係に一致する．
理論計算では，時刻に（空間）に関しては波長よりも波数がよく用いられる．
－＋αをφ＝Φと表現する．
また，音の強さは進行方向を向き，その大きさは，となる．
ただし，右肩の は共役複素数を表す．
つまり，空間領域における波数と波長の対応関係は，時間領域にお関して周期や周波数よりも角周波数がよく用いられるように，長さ正弦波は，交流電気回路の場合と同様に，複素数を用いて表現すると便利なことが多い．
そこで，正弦波の進行波φ＝cosω振幅 のみならず位相αも含んだ複素量で，複素振幅という．
