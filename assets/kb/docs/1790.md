# 2・5・3 プラズマディスプレイ(PDP: plasma display panel)

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・5 表示デバイス

a. PDP の概要PDPは、放電プラズマに伴う発光現象を表示に利用する発光型ディスプレイである。ネオン(Ne)の放電により発生する赤橙色発光を直接表示に利用する単色PDPと、キセノン(Xe)の放電による紫外光をRGB 蛍光体で可視光に変換するカラーPDPが実用化されている。動作原理的には、画素ごとに8―39電気・電子の基礎(a)(b)図2・83 (a)フルカラーアクティブマトリックスLCD プラズマディスプレイの表示セル構造表示セルと呼ばれる1mm程度以下の極めて小さな放電管(単色PDPの場合)、または蛍光ランプ(カラーPDPの場合)を並べたものと考えることもできる。電圧の印加のしかたによりAC型とDC型に大別され、これらの表示セルの構造は異なる(図2・84)。初めに実用化されたのは単色PDPで、自動券売機、行き先案内板などに使用されている。グラフィック用途でも、ラップトップ型パーソナルコンピュータのディスプレイに一時期使用された。近年はカラー化技術の進展が著しく、対角40~50インチのフルカラーPDPが商品化されるに至っている。PDPの特徴は次のようにまとめられる。まず長所としては、(1)素子構造が簡単で大画面化が容易、(2)フラットパネル型である、(3)特有のメモリ効果を駆動に利用でき、大容量化しても画面輝度が下がらない、(4)高画質のカラー表示が可能、(5)視野角が広い、などが挙げられる。一方短所は、(6)消費電力が大きい、(7)駆動電圧が高い、(8)高精細化が難しい、などである。b. ガス放電の特性あるしきい値(放電開始電圧)以上の電圧をガスに印加すると、原子や分子がイオンと電子に解離してプラズマ状態になる。ガス放電の状態は、電流・電圧等の発生条件によりいくつかの領域に分類できるが、PDPでは放電が安定に持続するグロー放電が利用されている。プラズマ中では、電界により加速された電子が原子やイオンと衝8―40電気・電子の基礎、化学の基礎突し、これらをエネルギーの高い準位に励起する。このような励起状態の粒子が基底状態に戻るとき光を放出する。放電ガスにNeを用いる場合は、570~720nmの波長域に数本の可視発光線が放出され、これらにより赤橙の発光色が得られる。現在のところ、ネオン以外で効率の良い可視発光を示す放電ガスは見つかっていない。一方Xeの放電からは、主として波長147nmの紫外光が放出される。PDPでは、放電開始電圧を低下させるため、Neやヘリウム(He)との混合ガス(Xeの割合は数%)が用いられる。一般に、封入ガス圧は数十kPで、放電開始電圧は200V 前後である。ガス放電の発光効率は放電空間を小さくするほど低下する。そのため、PDPでは表示の高精細化が難しい。PDPでの紫外線発生効率は、蛍光ランプに比べて1けた以上低く、消費電力が高い原因の一つになっている。c. 蛍光体材料カラーPDPでは、蛍光体がXeプラズマから発生する紫外光を吸収し、可視光に変換する。PDP用蛍光体としては、波長200nm以下の紫外線励起に対して発光効率が高いこと、表示用の三原色として十分な色純度をもつことが求められる。また、紫外線の利用効率を上げるために、表示セルの内壁に蛍光体が塗布されることから、放電によるイオン衝撃や温度上昇に対して、特性の劣化や変化が起きないことも重要である。現在主として、(Y,Gd)BO:Eu (赤)、ZnSiO:Mn (緑)、BaMgAlO:Eu(青)などの蛍光体材料が用いられている(2・5節では、蛍光体材料の化学式を、[母体物質]:[発光中心不純物]と表記する)。d. 素子構造と動作特性。 DC型PDP図2・85に、DC型フルカラーPDPの構造を示す。表示セルは、前面板および背面板の2枚のガラス基板に挟まれた100 パルスメモリ駆動方式の動作原理される。DC型PDPの表示には、パルスメモリ駆動方式が採用されている。この駆動法では、放電停止後に表示セル内に残留する準安定粒子などを利用して、表示にメモリ効果をもたせている。図2・86に、動作原理を示す。表示セルには、放電開始電圧以下のパルス状の電圧(維持パルス)が常時印加されている。放電を開始するためには、書込みパルスとして、放電開始電圧以上の電圧を加える。一度放電が始まると、準安定励起種等の存在により、以降は維持パルスのたびに放電が起こるようになる。放電を停止するには、電圧の低いパルス(消去パルス)を印加する。一度放電が停止すると準安定励起種等は減少し、維持パルスでは放電は起こらなくなる。このようなメモリ効果を利用することにより、単純な電極構造でありながら、アクティブマトリックス法と同様の駆動を行うことができる。. AC型PDPAC型PDPの表示セルの特徴は、電極が誘電体層に覆われていることである。AC型ではこの誘電体によりメモリ効果が発現する。電極配置としては、対向するガラス基板にそれぞれ電極を配置する対向型と、一方の基板上に二つの電極を配置する面放電型とがある。面放電型では、もう一方の基板上に形成される蛍光体層をプラズマから空間的に分離でき、イオン衝撃による蛍光体の劣化を抑制することができる。現在量産されているカラーPDPでは、面放電型を発展させた3電極面放電型が採用されている。パネル構造の一例を図2・87に示す。前面板と背面板との間隔は、DC型同様100 m程度である。前面板上には、走査線に相当するストライプ状の透明電極対(維持電極)、誘電体層および保護層が形成される。背面板上には、維持電極対と直交する方向に、アドレス電極、隔壁、蛍光体層が形成される。AC型PDPは、一般にパルス状の交流電圧により駆動される(図2・88)。電極が誘電体で覆われているため、放電で発生した正負の荷電粒子は、それぞれ陰極・陽極方向に移動して誘電体(保護層)表面を帯電させる。この帯電電荷のことを壁電荷と呼び、AC型のメモリ効果の起源となっている。放電は、維持電極間にしきい値電圧以上のパルス(書込みパルス)を印加することにより開始する。放電に伴い壁電荷が蓄積されるが、この電荷は、電場の極性が逆転した直後には、印加された電圧をさらに強めるように作用する。そのため、一度放電が始まると以降は放電開始電圧以下のパルス(維持パルス)で放電が維持される。放電を停止させるには、壁図2・87 AC 型フルカラーPDP の素子構造図2・88 AC 型PDP の動作原理電荷がちょうど消失するだけの短い時間のパルス(消去パルス)を印加する。壁電荷が存在しない状態では、維持パルスで放電は起こらない。以上のようにAC型では、壁電荷の有無と維持パルスにより、放電のオン/オフを制御できる。e. 階調表示法AC型、DC型ともに、メモリ効果を利用した表示では、発光はオン/オフの2状態しかとり得ないため、発光強度を変えて中間調を表示することはできない。代わりに、1フィールド(1画面の表示時間)の中で各画素の点灯時間を変化させて階調表示を行うサブフィールド法が採用されている。図2・89に、サブフィールド法の駆動波形例を示す。この例(256階調表示)では、1フィールドを、点灯時間の比が1,2,4,8,...,128となる八つのサブフィールド(SF1~SF8)により構成し、点灯させるサブフィールドの組合せで人が知覚する明るさを256階調表示で制御する。例えば、すべてのサブフィールドで点灯させると画素の輝度レベルは256となる。また、輝度レベル13で表示する場合は、SF1,SF3、およびSF4を点灯させればよい。このような階調表示法は、PDPのメモリ効果と極めて速い放電応答(~1 s)により実現されている。駆動方法の詳細については、他の文献に譲る。
