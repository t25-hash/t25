# 2・5・4 その他の表示装置:発光型ディスプレイ

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・5 表示デバイス

a. 陰極線管(CRT : cathode ray tube)一般には、発明者の名からブラウン管とも呼ばれる。ガラス製の真空管中で、加熱した陰極から引出した電子を10~30kV 程度に加速し、蛍光体に照射することで得られる発光を表示に利用する。ディスプレイとして最も長い歴史をもち、高輝度で色鮮やかな表示、高い信頼性、低価格などが特徴の完成度の極めて高いデバイスである。その一方で、箱型で場所をとる、重い、消費電力が大きい、画面の大型化・高精細化に限度があるなどの短所がある。図2・90に、一般に使用されているシャドウマスク型カラーCRTの基本的な構造を示す。管内のネック部にはRGB に対応する3本の電子銃が配置され、対向するフェースプレートの内面にはRGB各色の蛍光体が画素パターンに従い塗布されている。電子銃から放出された電子線は、偏向ヨークにより水平および垂直方向に走査されるが、蛍光体面の直前には色選別マスク(シャドウマスク)が配置されており、RGB それぞれの電子ビームが他色の蛍光体に当たることを防いでいる。カラーCRT 用蛍光体としては、主としてYOS:Eu(赤)、ZnS:Cu,Al(緑)、ZnS:Ag(青)が使用されている。一般にCRT 用の蛍光体は高抵抗であるため、陽極として機能する薄いアルミニウム膜を蛍光体層の表面に蒸着して帯電を防いでいる。この膜のことをメタルバックと呼ぶ。電子線は、十分に高いエネルギーをもつため、メタルバックを通して蛍光体を励起図2・89サブフィールド法による階調表示8―41電気・電子の基礎することができる。また、メタルバックは陽極で発生する陰イオンの衝撃から蛍光体を保護する役割ももつ。このような構造において、蛍光面上を電子線で点順次走査しながら、画像信号に応じてビーム電流を変化させることにより、フェースプレートの外側に画像が表示される。b. 蛍光表示管(VFD : vacuum fluorescence display)CRTと同様に、電子線励起による蛍光体発光を利用する方式のディスプレイである。CRT と異なる点は、加速電圧が10~50V 程度の低速電子線が用いられることである。また、CRT のような電子線の走査による表示方式とは異なり、ほぼ面状の電子源から放出される電子線を表示に用いる構造であることから、平面形状のものを作製できる。VFD は、低電圧駆動、平面型、小型・軽量などの特徴をもち、文字、数字、記号、バーグラフなどを表示するディスプレイとして、家電製品、車載機器をはじめとする多くの分野で用いられている。また、単色ではあるがグラフィックディスプレイも商品化されている。基本的な素子構造は3極真空管と同じで、フィラメント(カソード)から熱電子をメッシュ状のグリッドにより引出し、これをアノード上に塗布された蛍光体へ加速・照射する。開発当初は丸形管型であったが、現在生産されているものは、図2・91に示すようなフラットパネル型である。発光は、一般にフロントガラス側からフィラメントおよびグリッドを通して観察される。また、アノードに透明導電膜またはメッシュ状の金属膜を用いて、アノード基板側から発光を取出す透過型のVFD もある。低速電子線は、物質への侵入が非常に浅いため、メタルバックを必要とする高抵抗のCRT 用蛍光体をそのまま使用することができない。そのためVFD では、帯電が生じない程度の導電性をもつ蛍光体、もしくはInOなどの導電性物質を混合した蛍光体が用いられる。代表的なVFD 用蛍光体は導電性のZnO:Znであり、約505nmにピークをもつ青緑色発光を示す。多色表示用についても、青から赤の各色の発光を示す蛍光体が、硫化物を中心に開発されている。c. 電界放出ディスプレイ(FED : field emission display)高い電界を印加して真空中に電子を引出す冷陰極型の電子放出源を利用したフラットパネルディスプレイである。発光の原理はCRT,VFD と同様、電子線励起による蛍光体発光である。視野角が広い、応答速度が速いなどの長所をもつが、電子放出源の構造が複雑なため大面積化が難しいなどの短所がある。対角12インチ程度のフルカラーディスプレイも試作されているが、表示サイズがLCD と競合するため、製造コストの低減も課題である。電子放出源の一例として、図2・92にスピント(Spindt)型冷陰極の電子顕微鏡写真と構造を示す。スピント型電子源は、円すい状の金属冷陰極とゲート電極が一体で形成される。電子は、冷陰極に対しゲートに正の電圧を印加することにより、引出される。電界放出には一般に10V/cm程度の高電界が必要であるが、スピント型では冷陰極の尖った先端に電界が集中することから、実際に印加する電圧は100V 程度である。スピント型のほかにも、大画面表示図2・90 カラーCRT の構造図2・91セグメント表示VFD の素子構造8―42電気・電子の基礎、化学の基礎に適した平面形状の電子源も試作されている。電子放出材としては、Moなどの高融点金属やダイヤモンド状炭素膜などの仕事関数の小さい物質が用いられる。このような電子放出源は、半導体製造プロセスを応用して、ガラスまたはSi基板上に作製される。FEDに用いる場合は、一つの画素に対して数百個の冷陰極をアレイ状に集積する。FED は、アノード電圧が1kV 程度以下の低電圧型と1~10kV程度の高電圧型とに大別される。基本的な素子構造は、電子放出源を形成したガラス基板と蛍光体を塗布したアノード電極基板を、狭いギャップをあけて張合わせた構造である(図2・93)。内部は、10Pa以下の高真空で封止される。発光は、透過型VFD と同様に、アノード基板側から観測される。アノード-カソード間隔は、低電圧型で200~300 m程度である。一方高電圧型では、放電を防ぐために1~5mm程度に離される。そのため、電子線の広がりを抑えるための収束電極が必要になり、素子構造は複雑になる。高電圧型の利点は、発光効率および信頼性の高いCRT 用蛍光体をそのまま使用できることである。一方、低電圧型FED 用蛍光体としてはVFD 用蛍光体の利用が期待されたが、ZnO:Zn以外は励起条件などが異なり特性は十分ではない。現在、フルカラー表示のための新しい蛍光体の開発が進められている。d. 無機エレクトロルミネッセンス(EL)ディスプレイ(in-organic electroluminescence display)蛍光体に電場を印加したときに生じる発光(エレクトロルミネッセンス)を表示に応用したフラットパネルディスプレイである。素子の基本構造は、発光層を透明電極と金属電極で挟んだ積層構造であり、両電極間に交流または直流の電場を印加することにより動作する。特徴としては、高精細表示、広い視野角、速い応答速度、すべて固体から構成され振動・衝撃に対する信頼性が高いことなどが長所である。短所としては、動作電圧が高いことなどが挙げられる。発光層材料としては、ZnS やSrS などの比較的バンドギャップが大きい半導体を母体とし、これらに適当な不純物(発光中心)を添加して可視発光が得られるようにした蛍光体が用いられる。また、10V/cm程度の高電場を印加する必要があることから、キャリヤが少なく、電気抵抗が高いことも要求される。動作原理については不明な点が多いが、基本的には次に挙げる高電界下での各過程により発光が生じると考えられている。1発光層への電子の注入:界面準位、不純物準位などに捕獲されていた電子が、高電界により伝導帯に放出される。2電子の加速:自由になった電子は高電界下図2・92スピント型冷陰極の電子顕微鏡写真と断面構造で加速され、高エネルギー電子(ホットエレクトロン)になる。3発光中心の励起と発光:ホットエレクトロンとの衝突などの過程により発光中心が励起され、基底状態に戻るときに発光する。無機ELは、発光層の構造により薄膜型と分散型に分類される。素子構造の例を図2・94に示す。薄膜型ELでは、一般に、膜厚が0.5~1.0 m程度の発光層を二層の誘電体薄膜で挟んだ二重絶縁構造が採用されている。各層は、真空蒸着やスパッタなどの薄膜堆積法により作製される。駆動は交流電圧により行われ、200V 程度で発光が得られる。最も効率の良い発光を示す素子は、発光層にZnS:Mnを用いた黄橙色素子である。ZnS:MnELは、約585nmにピークをもつスペクトル幅の広い発光を示し、これを用いた単色のグラフィックディスプレイが商品化されている。フルカラー化については、赤色用としてZnS:Mn+赤色フィルタ、緑色用としてZnS:Tb、青色用としてBaAlS: Euなどの材料を中心に研究が進められている。分散型ELでは、樹脂バインダ中に粉末蛍光体を分散させた発光層が用いられる。交流駆動と直流駆動のものがあり、素子構造が異なる。薄膜型に比較すると、発光効率が低く、素子寿命も短いなどの課題があるが、大面積化が容易という特徴を生かし、LCD のバックライトなどに利用されている。e. 有機エレクトロルミネッセンス(EL)ディスプレイ(organic electroluminescence display)厚さ数十nmの有機化合物薄膜から構成されるデバイスで、10V 程度かそれ以下の直流電圧で動作する。高発光効率、高輝度、フルカラー表示、高精細表示、低電圧駆動、広い視野角、速い応答速度、フラットパネル型などの特徴をもち、1980年代の終わりごろから実用化に向けた研究が盛んになった。現在は、使用する材料にもよるが問題とされていた寿命も改善されつつあり、携帯や車載の分野では小型ディスプレイが商品化されている。また、フルカラーディスプレイの開発も進められている。有機ELは、電場印加で発光すること、および面発光タイプの素子であることから、無機ELと同じくELと呼ばれているが、動作原理は電流注入型電界発光であり、発光ダイオード(2・1・4項参照)の原理に近い(無機ELの発光機構は、真性電界発光とも呼ばれる)。有機ELは、用いられる材料系により低分子色素型と高分子型に分類されるが、基本的な動作原理に違いはない。低分子型の一般的な素子構造は、図2・95に示すような、正孔伝導性の高い正孔輸送層と電子伝導性の高い電子輸送層からなる二層構造、もしくは正孔輸送層と電子輸送層の間に発光層を挿入した三層構造である。二層構造の素子では、正孔輸送層または電子輸送層が発光層を兼ね、接合界面の極近傍で発光が生じる。一方、高分子型では単層構造が基本である。これらの有機薄膜は、低分子系では真空蒸着法、高分子系ではスピンコートなどの塗布法により、ITO透明電極(陽極)付ガラス基板上に形成される。背面電極(陰極)としては、仕事関数の小さな金属(Mg/Ag,Li/Al合金など)が蒸着される。このような素子に直流電圧を印加すると、陰極および陽極から電子、正孔がそれぞれ注入される。これらのキャリヤは発光層まで輸送され、再結合して発光を生じる。多色化については、発光層材料の選択のほか、発光層への発光中心分子のドーピング、青色発光層と色図2・94無機EL の素子構造図2・93 FED の素子構造変換層との組合せ、白色発光層とカラーフィルタとの組合せなどが試みられている。
