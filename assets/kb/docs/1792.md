# 2・5・5 その他の表示装置:投写型ディスプレイ

> **章**: 第2章 電子デバイスと電子回路
> **節**: 2・5 表示デバイス

a. 液晶プロジェクタ(liquid crystal projector)光量を制御するためのライトバルブに液晶セルを用いる投写型ディスプレイである。基本的な装置構成は、白色光源、光源からの光をRGB 各色光に分離する光学系、液晶パネル、変調されたRGB 光を合成する光学系、投写レンズからなる。使用される液晶パネルの種類により透過型と反射型に、またライトバルブの構成により3板式と単板式に分類される。液晶パネルとしては、光の利用効率を高めるため、開口率(=[画素部面積]/[パネル面積])の大きいものが要求される。光源としては、ハロゲンランプ、キセノンランプ、メタルハライドランプなどが用いられる。図2・96に、3板式透過型プロジェクタの構造を示す。3板式では、RGB 光のそれぞれにライトバルブが配置される。透過型では、対角1~2インチ前後の高精細p-Si TFT TN 液晶パネルが通常用いられる。一方単板式透過型では、従来直視型LCD と同様のカラーフィルタ方式が採用されていたが、フィルタによる光吸収のため明るい表示を得ることは困難であった。これに対し、分離されたRGB 光を、マイクロレンズアレイを組合わせた1枚の液晶パネルで制御する新しい方式が開発されている。単板式は、3板式に比較すると光の利用効率は低いが、構成が簡素になり装置を小型化できることが利点である。反射型液晶プロジェクタでは、ライトバルブとして画素部に誘電体多層膜や金属の反射鏡を形成した液晶パネルが使用される。透過型パネルとは異なり、光路上に不透明なスイッチング素子や配線がないため、開口率を大きくできる利点をもつ。駆動方法としては、MOS トランジスタなどを用いるアクティブマトリックス型と、CRT などを画像源とする光書込み型がある。これらのパネル構造と光学系を図2・97に示す。光書込み型の液晶パネルに用いられている光導電膜の電気抵抗値は、画像光の強弱により変化する。したがって、パネルに常時一定の電圧を印加しておけば、画像光の強弱に応じて液晶層に分配される電圧が変化し、反射光強度が変調され図2・95 低分子系有機ELの素子構造と材料分子の例8―43電気・電子の基礎る。b. DMD プロジェクタ(DMD projector)ライトバルブにディジタルマイクロミラーデバイス(DMD)を用いた反射型のプロジェクタである。液晶パネルのライトバルブとは異なり偏光を利用しないため、光の利用効率が高いことが特徴である。DMD は、SiのCMOS SRAM チップ上に、メカニカルに動作するAl製の微小ミラーを集積した光変調器である。画素に相当する各ミラーの傾斜角を変化させることにより、反射光の方向を制御する。図2・98,2・99に、現在量産されているDMD の構造と電子顕(a)(b)図2・97(a)反射型液晶プロジェクタ(光書込み型)の基本光学系と(b)パネル構造図2・98 DMD 素子(2ピクセル分)の構造。ミラーは透明に描かれている(テキサス・インスツルメンツ提供)図2・96 3板式透過型液晶プロジェクタの光学系図2・99 DMD 素子表面の電子顕微鏡写真。中央のピクセルはミラーとヨークがはずされている(テキサス・インスツルメンツ提供)8―44電気・電子の基礎、化学の基礎図2・100 単板式DMD プロジェクタシステム(DLP:degital lightprocessing、テキサス・インスツルメンツ提供)微鏡写真を示す。DMD 素子は、SRAM 回路を形成したウェーハ上に、ポリマ層のコーティング、Al薄膜および厚膜のスパッタ蒸着、選択エッチングなどの工程を組合わせて作製され、最終工程でプラズマエッチングによりポリマ層が除去されると、Al部品間に空間が形成される。図2・99に示す素子のミラーは、一辺が16 mの正方形で、17 mピッチで配置されている。ミラーが乗ったヨークは、薄膜Alのヒンジにより支柱に取付けられており、ミラーの対角を軸として回転できる。ミラーは、SRAM に接続されたミラー駆動用電極との間に作用する静電力により駆動され、メモリ出力に応じて水平位置から約+10°または-10°傾斜する。これら2状態間のスイッチング速度は10 s以下である。プロジェクタに応用する場合には、一方のミラー状態(オン)のときに反射光が投写レンズに入るようにし、もう一方(オフ)のときには投写レンズからはずれるようにDMD 素子を配置する。オン状態では画素がスクリーン上に表示され、オフ状態では黒点となる。DMD による光変調はオンとオフの2状態しかとり得ないが、スイッチング速度が速いことを利用した、時分割制御による階調表示法が採用されている。ライトバルブ構成としては、単板、2板および3板式がある。単板および2板式では、回転カラーフィルタを用いて時間的に光源の色分離を行っている。図2・100に、単板式DMD プロジェクタの構成を示す。文献(1) 石原宏、半導体デバイス工学、(1990)、40、コロナ社。(2) 菅野卓雄監修、ULSIデバイス・プロセス技術、(1995)、17、電子情報通信学会。(3) 文献(1)の53-61ページ。(4) Bardeen,J.and Bratten,W.H.,Phys.
