# 3・2・2 電動機の原理

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・2 電動機の種類と特徴

磁石のN 極とS 極を近接すると吸引力が働く。一方、N 極とN極またはS 極とS 極を近接すると反発力が働く。このとき、外輪をなんらかの方法で反時計方向に回転させると、内筒は回転し続けることになる。これが電動機の基本原理である。図定子巻線に三相交流電流を流すと、固定子は静止しているがギャップには回転する4極の磁界ができる。この回転する磁界を回転磁界という。回転子の電磁石の代わりに永久磁石にしたものが磁石同期電動機で、電磁石を単に強磁性体にしたものが同期電動機の一種であるリラクタンスモータである。一方、直流電動機は同期電動機とは逆に、固定子に界磁巻線を、回転子に電機子巻線(巻線電流は交流電流)を設ける。電機子巻線は原理的には図3・25の固定子巻線を回転子外径部に配置した構造となっている。直流電動機に供給する電力は直流であるが、回転部に設けた整流子(commutator、機械式の直流・交流変換装置)により、機械的に回転に同期した交流電流を電機子巻線に供給する。この電機子電流による回転磁界の速度は式(3・14)で表され、回転方向は回転子(速度)とは逆方向である。誘導電動機の回転子には外部励磁の電磁石ではなく、かご形誘導電動機の場合には鉄心両端で短絡される金属棒(二次巻線)が設けてある。この回転磁界速度(滑り速度)と回転子速度の和は、固定子巻線電流によって作られる回転磁界の速度(同期速度)である。固定子と回転子の両回転磁界の相互作用によってトルクを発生する。以上に述べたように、すべての電動機はギャップ磁束を介して、電源からの電気エネルギーを機械エネルギーに変換する機械である。また、外部から機械エネルギーを加えると、ギャップ磁束を介して巻線に電圧を誘導し、機械エネルギーを電気エネルギーに変換する発電機となる。
