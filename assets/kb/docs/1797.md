# 3・2・3 直流機(direct current machine)

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・25 は交流機の最も簡単な三相、4極機の固定子断面図である。固

a. 直流電動機の構造図3・26は直流電動機の構造と原理を示す図である。コイル(電機子巻線)の両端は整流子S,Sに接続され、ブラシB,Bを介して直流電圧が与えられる。コイルは磁界中を運動するので誘導起電力を発生する。図3・26において、コイルが180°回転すると整流子に接触するブラシが機械的に切換わるので、磁界とコイル電流の関係は同一となり、つねに同一方向のトルクを発生する。この整流子の作用を整流という。図3・26は1コイルのみを示しているが、実際の直流機は多くのコイルで構成され、各コイルを直列に接続してブラシから取出している。直流機の主要構成部品は、固定子では主磁極、補極およびブラシ保持器であり、回転子では電機子と整流子である。. 固定子直流機の固定子継鉄は、鋳鉄または鋼板で作られ、磁路を形成すると同時に、機械的構造部分として用いられる。急速な制御性能が要求される直流機では、界磁電流変化に対する磁束変化の追従を良くするため、継鉄部を積層構造にすることがある。主磁極は、1~3.2mm厚の積層鉄心からなる界磁鉄心と界磁巻線を一体に組立てた後、継鉄に取付けられる。小型機では主磁極を永久磁石で構成することが多い。補極は、主磁極の中間に設けられる幅の小さい磁極で、補極巻線には電機子電流を流すことによって整流作用の改善を行っている。また、中型機以上では主磁極の磁極片部に多くのスロットを設け、その中に納めた巻線に電機子電流を流す場合がある。この巻線を補償巻線といい、電機子反作用(電機子電流によりギャップ磁束が変化する作用)を打消してギャップ磁束分布の改善を行う。. 回転子電機子鉄心は界磁とともに磁路を形成するが、界磁鉄心と異なり交流磁束が流れる。したがって、交流機固定子鉄心と同様に、電機子鉄心は0.5mm厚のケイ素鋼板を所要の形に打抜いたものを積層し、これを回転子軸に焼ばめて回転子を作る。電機子鉄心は、積厚40~60mmごとに約10mm幅の冷却ダクトを設けている。現在製作されている産業用の直流機の電機子巻線は、ほとんど耐熱クラスF(耐熱温度155°C)で電機子鉄心に巻線を納めた後に、鉄心と巻線をエポキシ樹脂で一体含浸する全含浸絶縁方式が広く採用されている。整流子は、硬銅材料の整流子片をマイカ板で相互に絶縁し、これを円筒状に組立てる。整流子片と電機子コイルとの接続は、中型機以上ではライザを用い、小型機では直接ろう付けされる。b. 直流電動機の等価回路と電動機基本式直流機の励磁方式は、他励式、分巻式、直巻式および複巻式がある。サイリスタ、トランジスタなどの半導体電力変換装置によって、励磁電源が容易に作れるようになったので、他励式が鉄鋼用駆動機、工作機、製紙機などすべての用途に広く用いられた。図3・27に直流他励電動機の結線と等価回路を示す。誘導起電力は可変電圧で、界磁電流と8―54電気・電子の基礎、化学の基礎回転速度の関数である。鉄損、機械損およびブラシ電圧降下を無視すると、直流電動機の基本式は次式で与えられる。無負荷時の磁束をΦとすると、負荷時の磁束Φは次式となる。ΔΦは電機子反作用による磁束の減少分で、補償巻線付ではΔΦ≒0となる。c. 直流他励電動機の負荷速度特性定格負荷状態における励磁回路を調整することなく、電圧を一定に保ったまま、負荷を漸次変化させた場合の負荷(または電機子電流)と回転速度の関係を負荷速度特性という。ブラシ電圧降下を無視すると、負荷速度特性は式(3・15)より次式で与えられ、図3・28(a)のようになる。式(3・17)の右辺第2項の速度降下Δは、電機子回路抵抗に比例し、磁束Φに反比例する。したがって、直流他励電動機は定速度電動機とみなすことができる。また、電機子反作用によって磁束が減少すると、図3・28(a)の点線で示すようにある負荷以上で回転速度が上昇する場合がある。d. 直流他励電動機の電流トルク特性定格負荷状態における励磁回路を調整することなく、端子電圧を一定に保ったまま、負荷を漸次変化させた場合の電機子電流とトルクとの関係を電流トルク特性といい、図3・28(b)のようになる。鉄損と機械損を無視すると、式(3・15)よりはに比例するが、電機子反作用によって磁束Φが減少する場合には点線のようになる。端子電圧の調整は、現在では半導体電力変換装置で行っている。定格電流で運転すると、式(3・15)よりトルクは回転(a) 負荷速度特性(b) 電流トルク特性図3・28 直流他励電動機の特性図3・29かご形誘導機の概念図速度には無関係に一定になるので、定トルク運転ができる。. 界磁制御法(Φを変える)磁束を減少させると速度が増すが、負荷による速度変動が大きくなる。磁束を増すと速度は低下するが、実際の機械では鉄心飽和の影響があるので界磁電流を変えるわりには速度変化は得られない。速度に反比例させてΦを変化させれば、速度によらずを一定に保つことができる。界磁制御のみで速度制御をすることはなく、通常電圧制御と組合わせて用いられる。
