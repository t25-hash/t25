# 3・3・1 電動アクチュエータ(electromechanical actuator)の

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・3 各種電動アクチュエータ

特徴機械すなわち物を動かすためには動力(力×速度)が必要であり、その動力を発生するのがアクチュエータである。動力を発生するにはその源となるエネルギーが必要であり、エネルギー源をもとにアクチュエータを分類すると、電動、油圧、空気圧アクチュエータに大別できる。電動アクチュエータには、次のような特徴がある。(1) エネルギー源の電源が、油圧源や空気圧源に比較して、一般に容易に得られる。(2) エネルギーの伝達が電線で行えるので、油圧や空気圧の配管と比較して簡便である。(3) パワーエレクトロニクスの進歩により、電気エネルギーの制御が容易になった。(4) コンピュータ等エレクトロニクス制御装置との接続が容易。(5) 耐環境性、保守性、信頼性に優れる。電動アクチュエータは、基本となる動力の発生形態から、回転運動型と直線運動型に分けられる。
