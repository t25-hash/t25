# 3・3・3 直線運動(linear motion)型アクチュエータ

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・3 各種電動アクチュエータ

a. 直線運動型アクチュエータの分類直線運動は回転運動型アクチュエータと、ボールねじなどの直線運動変換機構を用いても実現できるが、さらなる高速・高加減速、高剛性、高精度の要求に応えるため、ダイレクトに直線運動力を発生する直線運動型のアクチュエータを用いたリニア駆動化も産業界で進んでいる。回転型のアクチュエータを使用する場合、直線運動機構に対して、モータは前述のように何種類かのモータとしてそれぞれ商品化されている中から、希望の特性のものを選定して使用することが可能である。リニア駆動の場合は、装置へ直接組込む(ビルトイン構造)ため、専用の設計(カスタムデザイン)となることが多く、モータ部の標準化が難しく多品種少量のためコスト高になりやすいが、リニア駆動の採用は増加傾向にあり、特性の改善とともにコストダウンの努力が続けられている。直線運動型アクチュエータすなわちリニアアクチュエータ(lin-ear actuator)の種類は非常に多く、その分類方法も、発展段階に応じて時代とともに変わってきている。ここでは当初から直線運動のため、独自の電磁的構造で発展してきた狭義のリニアアクチュエータと、回転型モータの電磁的構造を、直線上に展開したと考えられるリニアモータ群を合わせ広義のリニアアクチュエータと考え、次のように分類する。リニアアクチュエータ(広義)―リニアアクチュエータ(狭義)―ソレノイド―ムービングコイルアクチュエータ―リニアモータ―リニアDCモータ―リニアACモータ―リニア同期モータ(LSM)―高力密度型リニア同期モータ(HD-LSM)―リニア誘導モータ(LIM)―リニアパルスモータ(LPM)―永久磁石(PM)型―可変リラクタンス(VR)型b. ソレノイド(solenoid)ソレノイド(正式には電磁ソレノイド、electromagnetic solenoid)は、交流または直流の励磁コイルに通電し、可動鉄心を動かすことにより、電磁エネルギーを機械的直線運動に変換するプランジャ型の電磁石と規定されている。ソレノイドの基本的な構造は、図3・49に示すように電磁石のコイルを励磁して、可動鉄心であるプランジャを吸引動作するもので、消磁時に元に戻す動作は復帰スプリングによって行うことが多い。ストロークと吸引力の関係は反比例で、できるだけ大きなストロークで使用するため、プランジャ先端部の磁極形状をコニカル型とするのが標準的である。短いストロークで大きな保持力が必要な場合は、磁極形状をフラット型とする。ソレノイドを構造と形状から分類すると、コイル部が一部露出した最も簡単な構造の角形オープンフレーム型、コイルの周りを磁性材料で囲んで磁気回路の効率を高めたチューブラ型や、その他各社独自の工夫をした構造のものがある。ソレノイドの動作は、オンオフ動作が基本であるが、コイル電流とスプリングによりプランジャの位置を任意に制御する比例ソレノイドや、内部に永久磁石を組込み、プランジャ吸引後は吸着保持させることにより、吸引後の保持電流を不要にした省エネ型の自己保持型ソレノイドなどもある。c. ムービングコイルアクチュエータ(moving coil actuator)ムービングコイルアクチュエータの代表的なものはVCM(voicecoil motor、ボイスコイルモータ)であり、ダイナミックスピーカのボイスコイル部と同じ原理によって作動するので、ボイスコイルモータと呼ばれる。図3・50に示すように永久磁石によって一様な磁界を円筒状の空げき部に作り、その中に円筒状の可動コイルが挿入されている。コイルに電流を流すと、フレミングの左手の法則によりコイルに推力が発生し、この力は電流に比例する。可動コイルは鉄心がないので可動部の慣性が小さく、電気的時定数も小さいのが特徴である。図3・51にムービングコイルアクチュエータの別の構造例を示す。VCM に比べ構造が簡単なのではん用性がある方式である。ムービングコイルアクチュエータは、ソレノイドと比べ線形性と応答速度に優れており、磁気ディスク装置の磁気ヘッドなどの位置決めが代表的な用途である。d. リニアDCモータ(LDM : linear DC motor)直流のリニアモータ全般が、当初はリニア直流モータと呼ばれていた。例えばc項のムービングコイルアクチュエータや、e項のブラシレスDCモータのリニア型もLDM とする分類方法もあるが、分類の細分化が進んでブラシレス型はLSM(リニア同期モータ)に分類されるようになったので、LDM 自体は回転型のDCモータをリニアに展開したものに限るほうが、わかりやすい分類法と考えられる。これまでにブラシ付のDCモータを、リニアモータ化した例は存在するが、実用的ではないので、現実にはブラシレス化が進み、e項のLSM に置換わってきている。e. LSM(linear synchronous motor、リニア同期モータ)回転型のブラシレスDCモータの電磁的構造を、直線上に展開して直線運動するモータである。永久磁石界磁と電機子コイルの組合せ図3・50 VCM(ボイスコイルモータ)図3・51 ムービングコイルアクチュエータ8―63電気・電子の基礎が最も標準的な構造で、空芯コイルを用いたコアレスと電機子鉄心を用いたコア付に分けられる。リニアモータは回転機と異なり、ストロークの全範囲にわたって推力を発生するギャップが対向しなければならない。すなわち、ブラシレスDCモータの固定子の電機子巻線側か、回転子の永久磁石側のいずれかをストローク全長に配置する必要がある。一般に電機子を可動側にして、永久磁石を固定子全長に配置することが多いが、その場合可動側の電機子に給電する必要がある。可動部への給電をなくすため、永久磁石を可動側とすることもある。電機子鉄心と永久磁石の間には、強力な磁気吸引力(magneticattraction)が発生する。回転型のモータは軸中心の構造から、ギャップ部で吸引力がバランスし外部に出ることはないが、リニアモータはこの吸引力に対応した構造を採る必要がある。. コアレス永久磁石とヨーク間、あるいは永久磁石を両側に配置した磁界中に、渦巻状に巻かれた空芯(コアレス)のコイルを進行方向に並べて配置したもので、可動子となるコイル部は樹脂モールドしたものが一般的である。コア付に比べ、空げき長が長くなるので推力は小さい。磁気吸引力は固定子内で処理すればよく、コアレス構造でコギング力の発生もないので、推力リップルをいかに低減するかということと、推力と直角方向に発生するサイドフォースを小さく抑えることが、この方式のポイントとなる。図。 コア付コア付の可動子と固定子のマグネット間には、最大推力の3~5倍の磁気吸引力が生じる。図3・53に示す片側マグネット方式の場合は、リニアガイドに与圧がかかるという利点はあるものの、あまり強力な力の場合、たわみ防止のため可動子取付部の厚みを増やす必要から、可動部重量の増加やガイドの寿命低下などの問題が、特に大きな推力のモータでは課題となる。これに対して、図3・54に示す吸引力キャンセル構造は、可動子の両側にマグネットを配置して可動子とマグネット間に働く吸引力をキャンセルする。しかし、実際には左右のギャップクリアランスが全く均等ということは難しく、いくらかのアンバランスが生じる図3・52 コアレスLSM の断面図3・53 片側マグネット方式コア付LSM図3・54 吸引力キャンセル方式コア付LSM8―64電気・電子の基礎、化学の基礎図3・55 HD 型(高力密度型)LSM の原理図(a) PM 型(永久磁石型)(b) VR 型(可変リラクタンス型)図3・56 LPM(リニアパルスモータ)の構造表3・1 おもな電力用半導体デバイスの分類素子代表的記号や端子名等出力特性駆動法と波形(抵抗負荷)分類ダイオード整流ダイオードオン制御バルブデバイスサイリスタGTOサイリスタ(逆阻止三端子)サイリスタバイポーラパワートランジスタ(npn型)オンオフ制御バルブデバイストランジスタIGBT(nチャネル型)パワーMOSFET(nチャネル型)注:ショットキダイオードはユニポーラデバイスが、片側マグネット方式の1/20ぐらいに抑えることができるといわれている。. HD-LSM(high-density-type linear synchronousmotor、高力密度型リニア同期モータ)図3・55に示すHD(high density)型と呼ばれるLSM は、電機子内部に永久磁石が組込まれた形で、ストローク全長に磁気回路を構成するための磁性歯があるのみで、ストロークが長くなっても永久磁石の使用量が変わらないのが特徴である。f. LIM(linear induction motor、リニア誘導モータ)回転型の誘導型ACモータの電磁的構造を、直線上に展開して直線運動をするモータである。ACサーボモータの場合と同様に、大型のモータになるとマグネットの体積が増え高価になる。そこで大型大出力の搬送用などの用途では、誘導型が主流になる。g. LPM(linear pulse motor、リニアパルスモータ)回転型のステッピングモータに対応させると、直線運動型の呼び名はリニアステッピングモータとなるが、略称がLSM となりリニア同期モータと同じになるので、ここではリニアパルスモータという呼び方を使用する。回転型のステッピングモータと同様に、LPM というモータ構造が特別にあるわけではなく、モータ構造はLSM と基本的に同じであるが、送りピッチを小さくするため小歯(誘導子)を設けた構造が多い。代表的な方式は、図3・56に示すPM 型(permanent magnettype:永久磁石型)とVR 型(variable reluctance type)である。
