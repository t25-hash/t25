# 3・4・2 電力変換の概要

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・4 半導体電力変換装置

電力変換(power conversion)とはバルブデバイス(電流のスイッチングを行うデバイス)を用いて,電圧・電流・周波数(直流図3・59インテリジェントパワーモジュールの構造例を含む)・位相・相数・波形などの電気的特性のうち,一つ以上を実質的な電力損失なしに変えることと定義されている.ここで,実質的な電力損失なしに変換が行われることが重要な点である.これを実現するために,バルブデバイスとしてパワー半導体デバイスが使われ,その進歩に合わせて,電力変換は理想に近づいてきた.主要な電力変換回路の種類を,その入力と出力が交流か直流かによって分類し,表3・2に示す.以下,電力変換回路の代表例について述べる.
