# 3・4・6 交流変換装置

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・70 に示す。電動機の制御では、エネルギーが負荷から電源に戻

おもな交流変換装置(AC converter)としては、周波数変換装置と交流電力調整装置がある。周波数変換装置には、直接型と間接型がある。直接周波数変換装置はサイクロコンバータ(cycloconverter)と呼ばれ、以前は比較的大容量の交流電動機駆動などに多く用いられた。しかし、最近では順変換器でいったん直流に変換した後にインバータで交流に変換する間接型の周波数変換装置(frequency changer)が用いられることが多い。代表的な回路を図3・71に示す。ここでは、電流の方向によって、変換回路の一方が整流回路、一方がインバータとして動作する。交流電力調整装置は、簡単な回路で交流電圧の調整ができるため、調光装置、電熱装置(ヒータ)や、電動機の簡単な速度制御などに用いられている。交流電力調整装置としては、図3・72に示すように、サイリスタを逆並列接続して用いて電源周期の半サイクルごとにサイリスタを位相制御して電力を調整する方式がある。抵抗負荷の場合の電圧電流波形を図3・72(b)に示す。また、このときの出力電圧の実効値は、サイリスタの制御角(位相遅れ角)をαとすると、次のようになる。パワー半導体デバイスの進歩により、交流電力調整装置においても、PWM 制御方式が用いられるようになった。このような方式では、負荷に誘導分を含む場合にはスイッチがオフになったときの電流のバイパスが必要になることに注意が必要である。基本回路と動作波形を図3・73に示す。なお、図3・72の回路で、サイリスタを位相制御するのでなく、交流の数サイクルオンした後に数サイクルオフする使い方がある。このような使い方は交流スイッチと呼ばれる。
