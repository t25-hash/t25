# 3・4・7 電力変換装置の応用

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・70 に示す。電動機の制御では、エネルギーが負荷から電源に戻

半導体電力変換装置は、電力、産業、交通、家庭、オフィスなど、あらゆるところで応用されている。電動機駆動については、3・5節でまとめて述べる。ここではそれ以外の電力系統や電源としての応用などについて述べる。電力系統(electric power system)では、発電、送変電、配電などにおいて変換装置の応用が進んでいる。大容量直流送電は、送電端の順変換装置と受電端の逆変換装置の間で高電圧大電流の送電を行う。現在は主としてサイリスタ変換装置が用いられているが、オンオフ制御デバイスを用いる方式の開発も進められている。(静止型)無効電力補償装置(SVC: static var compensator)は、従来はコンデンサとリアクトルの並列回路を備えリアクトルをサイリスタで位相制御して無効電力を連続的に調整する方式が主であったが、最近では自励インバータを用いる方式も用いられるようになっている。また、特定の高調波発生源あるいは電力系統の高調波をそれと逆位相の高調波を発生して補償する電力用アクティブフィルタも、自励インバータを用いた方式が主流となり、実用化が進んでいる。太陽電池(solar cell)、燃料電池(fuel cell)などの新エネルギーシステムでは、発生した直流電力を交流に変換して電力系統と連系するのが一般的である。このような場合、制御性のよい自励インバータが用いられることが多い。また、SMES など電力貯蔵システムなどでは、貯蔵時には系統から貯蔵装置にエネルギーを注入する順変換動作、エネルギー放出時には逆変換動作が必要である。このため両方向動作可能な自励PWM 変換装置などが用いられる。(a) 回路(b) 動作波形図3・73 交流電力調整回路(PWM 制御)図3・74 太陽光発電システム太陽光発電(solar photovoltaic power generation)システムの回路例を図3・74に示す。図3・74は住宅用システムとして系統連系する例で、発電電力が住宅内の負荷より小さいときは、系統から不足電力の供給を受け、発電電力のほうが大きいときは、系統に電力を供給する。太陽電池の出力は、日照の変動などの影響が大きいので、つねに最大出力が得られるように、チョッパで太陽電池の動作点を制御する。電気化学などでは大電流の直流電源が必要で、整流装置が用いられる。また、誘導加熱用の電源として、高周波のインバータが用いられる。コンピュータなどの電源として、無停電電源装置(UPS :uninterruptible power supply)が用いられている。これは定電圧定周波数(CVCF)のインバータで、蓄電池などのエネルギー貯蔵装置を備え、電源正常時は順変換器で交流電力をいったん直流に変換した後、インバータで交流に変換して負荷に供給している。交流系統故障時には、蓄電池から電源を供給してインバータで交流に変換して供給するため、停電することなく信頼性を確保している。通信機器、情報機器、AV 機器などの各種の直流電源としてスイッチングレギュレータが用いられている。照明では、調光装置として最近では高周波インバータを用いたものが用いられている。
