# 16・1・3 遷移（中間）流（transition flow）

> **章**: 第16章 希薄気体力学
> **節**: 16・2 ボルツマン方程式

散乱角χが相対速度 と衝突径数 の関数として決まる．
は＝cosχ2となるのでσ＝4となる．
逆べき型ポテンシが大きい場合で，分子衝突頻度は，気体分子・固体表面間衝は実際の分子間力をかなり正確に表している．
ここで，＞0，α≧依存性を無視し， を定数で近似した分子模型を擬似剛体球分子る支配方程式であり，自由分子流から連続流までのすべての流れに求めた粘性係数が実験値と一致するように決める．
滑り流と自由分子流の中間の領域で，分子衝突頻度と分子・固体あるいは が増加するとχは減少する．
この に対してはχはのみの関数となる．
このとき係数 は，剛体球分子α→∞の場合を除きχに依存する．
このボルツマン方程式（Boltzmann equation) は気体の流れに対す＝4とする．
VHSモデルに対してはまたσ＝σ，σ＝23σとなる．
剛体球分子ではσ＝πであ気体 直径 nm 気体 直径 nmる．
