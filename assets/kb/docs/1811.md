# 3・5・1 ドライブシステムと制御系の構成

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・5 各種ドライブシステム

電動機を使い、各種の機器を駆動あるいは操作するのがドライブシステム(drive system)で、産業、交通、家電、情報機器等、幅広い分野で重要な役割を果たしている。ドライブシステムの中には、パルスモータを使って開ループで動作するものや、オンオフ制御によって動作する簡単なものもある。ここでは、閉ループ制御によって電動機のトルク、速度、および駆動対象(control object)の位置等を連続的に制御するものをドライブシステムとして扱う。a. ドライブシステムの基本構成ドライブシステムの一般的な構成は図3・75に示すとおりで、電動機(motor)、電力変換器(power conversion equipment)、制御装置(control device,con-troller)で構成される。電力変換器は電動機を駆動する電源で、商用電源やバッテリ等から所望の電圧や周波数に変換する。電力変換器の構成や性能は使用するパワー半導体デバイスのスイッチング速度や損失に依存する。電動機の電流と速度、駆動対象の位置はセンサ(sensor)によって検出される。制御装置は、指令とセンサで検出された各種制御量を用いて制御アルゴリズムの処理を実行し、電力変換器を制御する。制御装置には一般にマイコンが使われ、センサ信号を取込む図3・75 ドライブシステムの基本構成図3・76 制御系の基本構成A-D 変換器等の入力回路、電力変換器のパワー半導体デバイスを駆動する出力回路と一体で構成される。制御性能はマイコンの演算速度、A-D 変換器の精度や変換時間に依存する。b. 制御系の基本構成ドライブシステムの基本的な制御量は、電動機の電流と速度、駆動対象の位置である。制御系の基本構成は図3・76に示すとおりで、各制御量に対応して閉ループの制御系が構成される。各制御ループの処理はPID(比例、積分、微分)演算が基本であるが、マイコンによるディジタル処理の特徴を生かして、新しい制御手法が開発されており、マイコンの性能向上と相まって次第に高度化している。電動機の制御において、最も重要かつ基本的な制御量はトルクで、その精度と応答性能が問題になる。分巻直流電動機では、励磁を一定に保てば、電機子電流が発生トルクに比例するため、図3・75の電流制御ループによって、トルクを正確に制御することができる。交流電動機においても、後で述べるベクトル制御(vectorcontrol)を適用することにより、電流成分をトルクに比例させることができる。電流制御ループは、速度や位置の制御系においてマイナループとして用いられることが多いが、巻取り機の定張力制御や押出し成形機の定出力制御等ではメジャループで用いられる。速度制御系は、各種機器の運転に幅広く使われるが、運転モードとしては定速運転と可変速運転がある。前者では速度偏差を小さくするために、外乱トルクの抑制特性が、後者では滑らかな加減速のために加減速トルクの応答性能が問題になる。いずれの場合も、電流制御のマイナループの応答が重要である。位置制御は位置サーボとも呼ばれる。工作機械や測定システムの精密位置決め、ロボットの軌跡制御等に適用される。この場合も、高い精度と応答を得るためには、速度制御と電流制御のマイナループの応答が重要である。
