# 3・5・3 ドライブシステムの変遷

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・88 )の中で示す.

ドライブシステムの開発が本格化したのは,最初のパワー半導体デバイスであるサイリスタが実用化された1960年ごろからである.最初は圧延機や鉄道車両の駆動,工作機械やロボットのサーボがけん引し,パワー半導体デバイスとマイコンの進歩によって幅広い分野に普及してきた.その変遷の経緯をけん引役になった半導体技術の進歩,および応用側からの要求と対比して示したのが図3・79である.初期のドライブシステムには直流電動機が使われた.駆動電源の構成が簡単であるうえに,電動機の制御パラメータと駆動特性の対応が明確であるため,比較的簡単な制御で高い性能が得られたためである.制御の面では,それぞれの応用に対して満足すべき性能が得られ,各種の装置やシステムの性能向上に貢献した.しかし,直流電動機の本質的な問題として,機械的な接触部である整流子とブラシの保守の問題が残った.ドライブシステムの発展は,整流子やブラシのない交流電動機を使い,直流電動機と同等以上の性能を出すことに向かって進んできたといってよい.この中で,マイコンとパワー半導体デバイスの進歩が重要な役割を果たした.交流電動機を使ったドライブシステムは二つの方向に向かって進んだ.一つは,直流電動機のブラシと整流子の機能をパワー半導体デバイスで置換えようとする方向である.現実には,直流電動機の整流子の数ほどパワー半導体デバイスを使うことはできないため,三相の同期電動機を使い,磁極位置のセンサによって,各相に電流を供給するタイミングを決める方式が一般的だった.当初はパワー半導体デバイスとしてサイリスタが使われたため,サイリスタモータと呼ばれた.ここでは,電動機の速度を制御するために,別の電源で電圧を制御することが必要であった.しかし,IGBT(insulatedgate bipolar transistor)とMOSFET(metal-oxide semiconduc-tor field effect transistor)の登場でPWM(pulse-width modula-tion,パルス幅変調)制御が可能となり,電圧制御が容易になって,応用が小型電動機まで広がった.電動機のほうも,磁石材料の性能向上もあって,永久磁石型同期電動機が普及した.この方式は,直流電動機と同様に制御は行いやすい反面,トルク脈動が大きいことが欠点である.もう一つは,可変周波数インバータで誘導電動機を駆動する方向である.ゲート信号で電流を遮断できるGTO(gate turn-offthyristor)の登場でPWM 制御が容易になり,周波数と電圧を容易に制御できるようになって普及が進んだ.PWM 制御では,電圧を制御できるだけでなく,電動機電流を正弦波に近づけることもできる.また,IGBT,MOSFET によって小型化が可能になり,応用が小型電動機にまで広がった.この方式の欠点は高応答のトルク制御ができないことである.また,無効電流による損失も発生する.上で述べた交流電動機駆動の欠点を解消するのがベクトル制御である.この狙いは,正弦波駆動のインバータでトルクの脈動をなくしたうえで,トルク電流を励磁電流から分離して制御することで,高応答のトルク制御と効率を直流電動機と同等にすることにある.これ以降の部分では,次の順序でドライブシステムの具体的な内容を説明する.まず,直流電動機によって,ドライブシステムの基本的な動作を述べる.次に,上に述べた2種の交流電動機のドライブについて述べ,この基本的な動作とともに,ベクトル制御の狙いを明らかにする.この後,ベクトル制御の原理を述べ,最後に代表的な応用事例を示す.
