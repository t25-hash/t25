# 3・5・4 直流ドライブ

> **章**: 第3章 電気機器とパワーエレクトロニクス
> **節**: 3・88 )の中で示す.

直流電動機によるドライブシステムを構成するためには,直流電圧を変化させることができる駆動電源が必要である.工場の電源を使う圧延機の駆動では,サイリスタを用いたAC-DCコンバータが図3・79ドライブ技術の変遷使われ,直流架線を使う鉄道車両やバッテリを電源とするものでは,チョッパと呼ばれるDC-DCコンバータが使われる.また,いずれの場合も加減速運転が必要で,発電機運転の領域があるため,電力を電源側に回生する逆変換回路が必要である.ここでは,直流ドライブをけん引する役割を果たした圧延機の駆動を例にして,具体的な構成と動作を説明する.図3・80に圧延機に使われた駆動回路の代表的な構成を示す.サイリスタの点弧位相によって電圧を制御する電力変換器を逆並列に接続している.パワー半導体デバイスを使った電力変換器が使われる前には,直流発電機を駆動電源として用い,この界磁電流で電圧を可逆制御するワードレオナード方式(Ward-Leonard system)が使われていた.これに対応して,電力変換器を使う方式は静止レオナード方式(static Leonard system)と呼ばれた.この構成では,先に述べた弱め界磁の制御を行うために,サイリスタ整流回路で界磁電流を制御する機能も備えている.逆並列に接続された電力変換器は,一方が電動機の電圧を制御するAC-DCコンバータとして働き,一方は電動機が発電機として動作したときに電力を回生するDC-ACコンバータとして働く.両方の役割を切換えることにより,正逆転運転に対応できる.正逆転切換え方式としては,出力電圧を低く抑制した状態で両者の動作をラップさせ,循環電流をリアクトルに流しておいて自動的に切換える方式と,電流が零になったことを検出して切換える方式がある.前者は切換えの無駄時間(dead time)がない反面,循環電流分(circulating-current component)の損失が発生する欠点がある.以下の説明は後者で行う.静止レオナード方式の制御系の構成は図3・81に示すとおりで,電機子電流の制御ループと界磁電流の制御ループを備えている.この制御系によって,図3・78に示したトルク制御と出力制御を具体的に説明する.a. 一定界磁によるトルク制御電流制御をマイナループとする速度制御系が動作する.誘導起電力制御系は動作させず,界磁電流は定格値で一定に保つ.電流制御ループによって制御される電機子電流はトルクに対応する.直流電動機では,わずかな負荷トルクの変動で過大な電流を発生する可能性があるため,速度制御系からの電流指令に飽和特性をもたせ,電流を最大許容値以内に抑える.また,サイリスタ整流回路図3・80 直流ドライブ(静止レオナード方式)の構成8―73電気・電子の基礎で電圧を制御する場合,電流の断続によって,点弧位相と電流の間に非線形性が生じるが,電流制御ループを構成することで,この非線形性が補償され,速度制御系の応答が向上する.b. 弱め界磁による出力制御上記の速度制御系の他に,誘導起電力制御回路が動作する.誘導起電力制御回路において,図3・78の基底速度に対応する誘導起電力を指令値にすると,基底速度を超えた速度指令に対し,界磁電流が自動的に弱め状態になり,速度が最高指令値まで上昇する.電機子電流は電動機の出力に比例する.
