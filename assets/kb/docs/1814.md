vector control）と呼ばれる．正確な位置検出や，低速時のトルク
の制御が難しいという欠点はあるが，ポンプやコンプレッサの駆動
のような応答性能を要求しない速度制御には有効である．
図3･88にセンサレスベクトル制御の基本的な構成を示す．サー
ボ系の例では，電流をインバータの交流側で検出していたが，この
例では，基本構成を簡略化するため直流側で行っている．インバー
タ各相の動作に対応して，直流電流をサンプリングし，A-D変換
器で取込んで合成すれば，交流電流で再現できる．この機能は図の
電流再現回路が対応し，高速のA-D変換器を使うことで実現でき
るが，詳細な説明は省略する．
速度制御系においては，センサからのフィードバックを用いる代
わりに，演算に使っている電流推定値と検出した電流から軸誤差を
計算する．一方，位置指令は -座標系の角度θに換算するこ
とで速度制御系が構成されているが，ここに軸誤差を速度偏差に換
算して加えることにより軸誤差を零にする制御が行われ，電動機は
速度指令に従って回転する．
c .永久磁石型同期電動機の弱め界磁制御 直流電動機の弱め
界磁制御は界磁電流によって行ったが，ベクトル制御を適用すれ
ば，永久磁石型同期電動器においても，等価な制御を行うことがで
きる．この方式は，永久磁石型同期電動機の可変速度制御領域を広
げ，電気自動車や鉄道車両に適用しようとする際に有効である．こ
の構成を図3･89に示す．図3･89で，駆動回路，電動機，座標変換
については図3･86のACサーボと同じである．異なるのは，トル
ク電流成分だけでなく，励磁電流成分を制御する点である．
ここでは，速度がある点を超えると，励磁電流指令を作る関数発
生器が動作し，励磁電流成分をマイナス方向に制御する．これによ
って，永久磁石による磁束の一部が打消され，弱め界磁（field
weakening）の状態になる．この結果，電動機の速度はさらに上
昇する．この動作モードでは，直流電動機の場合と同じように，ト
ルク電流成分を電動機の出力に対応させることができる．ただし，
電動機電流に励磁電流が加わった分だけ損失は増大する．
