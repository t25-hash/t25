# 3・5・6 ベクトル制御の原理

> **章**: 第3章 電気機器とパワーエレクトロニクス

上で述べた交流ドライブの課題を同時に解決し、直流ドライブと同等以上の性能を実現しようとするのがベクトル制御である。その基本は、電動機の三相電流波形から、トルク電流と励磁電流を分離して算出し、それぞれを独立に制御することにある。ベクトル制御の原理はドライブシステムにインバータが使われるようになったころから提案されていた。しかし、正弦波電流による交流電動機駆動が前提になり、かつ複雑な制御演算を高精度で高速図3・85 三相交流から-座標への変換に行う必要があるために実用に至らなかった。ベクトル制御は、パワー半導体デバイスの高速化と低損失化、マイコンの高性能化によって初めて実用化された技術である。まず、ベクトル制御の基本になる座標変換について説明する。図電圧、電流の関係を示すタイムチャートとベクトル図で示す。-座標系とは、回転磁界に同期した回転座標系で、直交する軸は励磁電流成分(field current component)、軸はトルク電流成分(torque current component)に対応する。-座標系では、U 相巻線軸を基準に、反時計周りを正回転方向として、回転子磁束に対応する軸、それより90度進み位相の軸を取り、軸角度をθ욃とする。ここで、電圧、電流の正弦波波高値を大きさとし、位相を角度で表したのがベクトル図である。電動機電圧は、誘導起電力、抵抗とインダクタンスによる電圧降下のベクトル和になる。ここで、電動機電圧を軸成分욃と軸成分욤とすれば、各々は、電動機電流욃と욤による抵抗とインダクタンス降下成分に誘導起電力を加えたものになる。ベクトル制御を適用した制御系においては、まず、三相の電動機電流を-座標系に変換する。この変換は次の行列式で行う。욊욤=cos θ욃sinθ욃욃-13 욊+2욌-sinθ욃cos θ욃ここで得られた욤はトルクに対応するトルク電流成分で、直流電動機の電機子電流と同様に扱って制御ループを構成する。永久磁石型同期電動機の場合、弱め励磁電流成分は、通常零に制御して損失を低減する。これを負にすれば、永久磁石の界磁を打消して、等価的に弱め界磁を実現できる。界磁巻線がある同期電動機の場合、電機子反作用で発生する磁束のずれを補正するために用いることが多い。制御の演算を行った後の욃と욤に対しては、電動機の抵抗とインダクタンスを使った演算を行い、電圧成分の욃と욤に変換する。この後、次の行列式によって三相交流の電圧に変換する。욊-1cos θ욃-sinθ욃욃욋=sinθ욃cos θ욃욤욌-1この三相交流はインバータが発生すべき電圧に対応する。そこで、図3・84に示した方法でPWM 制御の処理を行い、インバータのパワー半導体デバイスを駆動する信号を得る。
