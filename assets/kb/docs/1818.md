# 4・1・1 電圧・抵抗・電流

> **章**: 第4章 電気計測
> **節**: 4・1 電気・磁気量の測定

今日広範に使用されている電気計測器に,ディジタルマルチメータ(digital multimeter)がある.典型的には,1kV までの直流電圧,1GΩまでの直流電気抵抗,1A までの直流電流,および約1MHzまでの周波数帯域の1kV までの交流電圧と1A までの交流電流が測定できる.これらの範囲の電気量の測定をするならまずディジタルマルチメータを使いこなすことである.電源を投入し測定レンジを指定し入力端子に結線すれば,測定結果は一応表示される.しかし,表示される測定結果がどれほど正しいか,精密な測定では,はっきりさせる必要がある.すると,(1)精度仕様書(accuracy specification)を読みこなして実際の測定に的確にあてはめて不確かさを評価すること,(2)トレーサビリティ(traceability)を理解して調整・校正して使用すること,(3)入力段のアナログ回路に気をつけることが,現代では重要な注意事項になっている.その背景と対策を以下に述べる.1970年以前の電気計測器はアナログ回路であった.そして,その回路図と部品の特性を使用者がよく理解することが,電気計測(electric measurement)の基本であった.高精度な測定を行いたい場合は,測定前に電気計測器の目盛りを校正するが,そのためには信頼できる標準器が必要であった.これらの作業には,電気計測の知識・技能・設備が求められるため,大企業では標準室を設けてそこの専門家にゆだねる態勢にあった.それに対して,1980年代に起きた重要な技術革新にマイクロプロセッサ,メモリ,プログラムの搭載がある.これらにより,上で述べた従来は専門家の知識・技能に頼って行われていた作業がある程度,廉価に代替できるようになった.現在でも,電気計測器の入力段の回路とか,電気計測器の心臓部で標準信号を発生する回路や比較回路には高精度なアナログ回路が用いられるので,アナログ回路を理解することは重要であるが,その一方,各レンジのゲイン調整をすることや校正の作業などが,電気計測器に組込まれたマイクロプロセッサとメモリによって比較的簡単に,高精度に実施できるようになった進歩がめざましい.今日の高精度な電気計測器には,自己調整機能が付いたものがあり,使用者はその機能を起動することで,昔なら専門家でなければ行うことができなかった複雑な調整作業を,廉価に行わせることができるようになっている.このように,アナログとディジタルが複合化され,プログラムがメモリに搭載されてマイクロプロセッサにより制御される複雑な計測器の動作の詳細を,使用者が完全に知ることは困難であるばかりでなく時間の使い方から効率的でもない.幸い,高精度電気計測器メーカは,製品マニュアルに詳しい精度仕様書を掲載しているので,使用者は自分がその電気計測器をどのような条件で使用しているかを理解し,精度仕様書の表を読み取ったり不確かさ算出のための式を使って計算することができるかどうかが,重要になっている.すなわち,従来は専門家の管理において行われることが多かった調整・校正ならびに測定不確かさ(昔は類似する概念を誤差と呼んでいた)の見積もりが,測定器の使用者の責任においてなされることが多くなった結果,その意味するところを使用者がよく理解することが求められるようになった.1990年代半ばから広まった現代的な不確かさの評価法を理解し,その基礎のうえで計測器のマニュアルに記述されている精度仕様書の表を読み取ったり不確かさ算出のための式を使って不確かさを正しく見積もることが,高精度測定を行う者に求められようになってきているのである(不確かさの評価法については,ここでは改めて説明しない.別途文献等を参照していただきたい).高精度電気計測器メーカが製品マニュアルに記載している詳しい精度仕様書では,しばしば最後に調整・校正を行ったときから,い8―77ま計測を行うまでに経過した時間が重要な変数とされている.それは次の理由による.通常,計測器の中には基準となる信号を発生させる標準器が内蔵されており,その内蔵標準器の値が正しく知られていることを前提に,入力信号の大きさを換算して表示する.これら内蔵標準器の値は,厳密には時間とともに変化する.そのため,実施しようとしている測定が高精度であればあるほど,内蔵標準器の値の微小な経時変化が無視できないものとなる.計測器がもつ性能を最高の状態で活用するためには,調整・校正用の外部標準器を使用者がみずから所有して頻繁に調整・校正して用いることになるが,その場合は調整・校正用の標準器に信頼できる校正証明書が伴っている必要がある.より一般的な使用方法では,1年に1回程度の頻度で計測器メーカに計測器を送り,計測器メーカにおいて外部標準器を参照した調整・校正を有償で行ってもらって校正証明書を入手する.最近では,この校正作業自体を有償提供する校正事業者も存在する.1990年代から広まっている国際ルールでは,校正に用いる標準器が信頼できることを主張するためには,国家標準(または国際標準)にトレーサブルであることを求められるようになってきている.そこで,ある校正事業者(計測器メーカが校正する場合は,その計測器メーカも校正事業者とみなす)が,はたして国家標準(または国際標準)にトレーサブルな標準器を用いて信頼できる校正を行って校正証明書を発行しているかが問題となる.そこで,校正事業者認定制度という制度が発達してきており,国際的に信用力がある審査機関が申請を受けて校正事業者を審査し,要件を満たしていれば認定している.そのように認定された校正事業者を認定校正事業者と呼ぶ.したがって,一般の使用者は,認定校正事業者による校正を受けた標準器あるいは計測器を用いることによって,信頼できる計測を行っていると第三者を納得させることが容易になる.わが国では,独立行政法人製品評価技術基盤機構が実施しているJCSS(計量法認定事業者制度)が代表的な校正事業者認定制度である.計測器は単体としての動作原理に劣らず,測定結果に信頼性を付与するために社会的に発達してきたインフラストラクチャを活用することが望まれている.以上述べたのは,そのような社会的に発達してきたインフラストラクチャの活用を勧めるものである.では,以下に,計測器単体に注目し,等価的なアナログ回路を用いて入力段で気をつけるべき最も基本的なことに絞って注意点を述べる.a. 電圧(voltage)電圧を測定する場合に注意すべきことに,計測器の入力インピーダンスがある.この点に着目して描いた電圧計(voltage meter)の等価回路が図4・1である.入力端子から計測器の内部を見た回路が,図4・1の左のように理想的な電圧計と抵抗읉윰が並列になったものと等価とみなし,この抵抗읉윰のことを入力インピーダンスと呼ぶ.電圧測定時の入力インピーダンスは,無限大であることが理想であるが,実際には10MΩ~10GΩであることが多い.入力インピーダンスがいくらであるかは,計測器の仕様書に書いてある.入力インピーダンスは,測定電圧レンジに応じて切換わることがあるので,計測器の仕様書を見て,いま測定しているレンジの入力インピーダンスがいくらなのか確認しておくべきである.図4・1の右のような被測定回路を例にして説明を進める.被測定回路に電圧計がつながっていないときには,AB 間の電圧降下は,図4・1 電圧計の等価回路8―78=/+(4・1)であり,これが測りたい電圧であったとしよう.AB 間の電圧降下を測るために,A,B に電圧計のリード線をつなぐ.すると,被測定回路の抵抗と並列に入力インピーダンス읉윰が接続された場合と等価なことが起きることに注意しなければならない.その結果,電圧計をつなげた場合のAB 間の電圧降下は小さくなり,電圧計が表示する測定結果として,本来知りたかった電圧降下よりも小さい値が表示されてしまう.測定対象回路のの抵抗値が大きいほど,また電圧計の入力インピーダンス읉윰が小さいほど,この影響は大きい.影響が無視できるかどうかおおまかに判定するには,읉윰/の比が何けたあるか概算する.自分が測定したい精度のけた数より十分大きくない場合には,無視できない程度の影響があると考えて,等価回路を描いて回路解析し,補正を加える.交流電圧波形は,理想的な正弦波とは限らない.正弦波がゆがんだ場合があるほか,矩形波や三角波(あるいはそれらのゆがんだもの)である場合がある.一般に交流波形の電圧として代表的に用いられる量は,その実効値である.現代のディジタルマルチメータの交流電圧測定レンジは,多くの場合,実効電圧値を被測定量としている.特に断わりがない限り,表示される交流電圧は,実効値である.b. 抵抗(resistance)抵抗を測定する方法に,2端子抵抗測定法と4端子抵抗測定法がある.図4・2に2端子抵抗測定法で測定する場合の等価回路を示す.計測器の中には定電流源があって,外部につないだ抵抗に電流を流すことで電圧降下を生じさせる.その電圧を電圧計が測り,電圧/電流の比を算出して抵抗値として表示する.図4・2の等価回路からすぐわかることだが,測定される抵抗値は知りたい抵抗のほかに,リード線ならびにリード線を結線している接触面の接触抵抗などを合わせた抵抗値+が加算される.したがって,計測して表示される抵抗値は,よりも大きい.その大きさの程度は典型的には数Ωである.この数Ωの差が問題になる場合は,図4・3のような4端子抵抗測定を行うことで,精密な抵抗測定ができる.図4・3のようにすれば,電流を流しているリード線などの+の抵抗分で生じる電圧降下を避けて電圧測定をすることになるので,の抵抗値が正確に測れる.電圧測定用リードにもリード線などの抵抗成分+があるが,電圧測定リードに流れる電流の大きさは無視できるほど小さいので,こちらは抵抗測定結果に差を生じる要因とはならない.最近のディジタルマルチメータには,2端子抵抗測定を行うか4端子抵抗測定かを使用者が選択するようになっているものが多いので,目的にあわせて選択する.半導体などは,理想的な抵抗器と異なって,抵抗値が電圧によって著しく変わることがある.そのような測定対象の場合は,抵抗計(ohmmeter)が流している電流値がいくらであるかが重要な測定パラメータとなることがあるので,意識しておかなければならな図4・2 2端子抵抗測定法の等価回路図4・3 4端子抵抗測定法の等価回路い.抵抗計が流す電流値がいくらかは,計測器の精度仕様書に書いてあるはずである.c. 電流(electric current)電流を測定する場合に注意すべきことに,計測器の入力シャント抵抗がある.この点に着目して描いた電流計(current meter)の等価回路が図4・4である.図4・1の電圧計の場合に似ているが,電圧計の場合には入力インピーダンス읉윰をなるべく大きくするよう設計されているのに対して,電流計の場合はシャント抵抗は小さいことが望ましい.実際には,0.1Ωから数百Ωくらいまでである.シャント抵抗の大きさがいくらであるかは,計測器の仕様書に書いてある.シャント抵抗は,測定電圧レンジに応じて切換わることがあるので,計測器の仕様書を見て,いま測定しているレンジの入力インピーダンスがいくらなのか確認しておくべきである.被測定回路に流れている電流を測るためには,測りたい箇所に電流計を挿入することになる.図4・4に示している例からもわかるように,シャント抵抗の大きさが,被測定回路のインピーダンスと比較して無視できない大きさがある場合は,測定結果が影響される.そのような場合は,等価回路を描いて回路解析し,測定結果に補正を加える.
