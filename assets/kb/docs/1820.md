# 4・1・3 電力(electric power)

> **章**: 第4章 電気計測
> **節**: 4・1 電気・磁気量の測定

電力(パワー)を時間積分したエネルギーのことを電力量(electric energy)という.家庭や工場で使用した電気代を支払う根拠となる測定は,電力量計・無効電力量計・最大需要電力量計とこれに付属する変成器などの電気計器を用いて行われている.電気の取引に使用される電気計器に対しては,計量法に型式承認の制度,公差の限度,検定の有効期間などが定められており,取引のためには計量法令を遵守しない場合は罰則がある.取引にかかわらない研究・開発における計測器の使用は自由である.研究・開発における電力量あるいは電力の測定の目的に,機器の消費電力の測定がある.そのような測定では商用電源と同様な電力供給に対して機器が消費する電力を測定することが多く,そのような目的には,同様な条件での計測を目的に開発・検定されている取引用電気計器を用いてもよい.一級基準電力量計の基準器公差は0.2%なので,この程度の不確かさの測定ができる.電力量計(watthour meter)には電圧入力端子と電流入力端子がある.古来用いられてきた電力量計は,この電圧と電流の積(すなわち電力)に比例する角速度で円盤を回転させる巧妙な原理を用いている.近年では,入力電圧と入力電流をそれぞれアナログ-ディジタル変換し,演算によって有効電力・無効電力を算出するディジタルパワーメータもある.取引用電気計器より小さい不確かさで電力測定ができるほか,電力線の高調波解析機能がついている特徴があるものもある.
