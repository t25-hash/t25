# 4・1・4 磁気計測

> **章**: 第4章 電気計測
> **節**: 4・1 電気・磁気量の測定

鉄やパーマロイなど強磁性体(ferromagnetic material)の比透磁率は数百以上の値で非線形特性を示す。強磁性体以外の物質は、反磁性体、弱磁性体、常磁性体に分類され、それらの比透磁率はほぼ1である。自然環境において計測対象となる磁界は、図4・6に示すようにfT(10T)オーダから数T の範囲である。いわゆる地磁気は30 T 程度であり、モータや発電機など電気機械は1T 前後の磁束密度で使用される。生体磁気は、人間の心臓の拍動によって生じる磁界、すなわち心磁界は数nT 程度であり、脳内の神経活動による磁界、脳磁界は数pT 程度である。磁界測定には電磁誘導の法則をはじめとして各種の物理法則、効果が利用されている。b. 空間の磁界測定一般の空間中の磁界は、ホール効果(Hall effect)による素子を使用して測定される。ホール効果は、1879年、アメリカの物理学者ホールによって発見された。これは、図4・7のように直方体の物体に電流を流し、その電流と垂直方向に磁界を加えると、電流と磁界の方向にそれぞれ垂直方向に電圧が発生する現象である。このとき発生する電圧をホール電圧という。ホール効果は、半導体にはよく現れるが、導体にはあまり現れない。ホール電圧は、以下の式で表される。これは、リング状超伝導体の一部をジョセフソン接合させたもので、リングを通過する磁束量子(2×10Wb)の検出が可能で、極めて高感度な磁界センサとして利用されている。ジョセフソン接合は薄い絶縁膜を介して超伝導体を接合したもので、ジョセフソン接合を流れるトンネル電流は以下の式で表される。すなわち、接合が一つのときの電流は、接合に加わる磁束密度の周期関数になる。SQUID には、一つまたは二つのジョセフソン接合をもつdc-SQUID とrf-SQUID がある。dc-SQUID の分解能は抵抗の熱雑音によって決定される。SQUID は1×10~1×10T の微弱磁界を検出できるため、心磁界や脳磁界を測定、記録することができる。c. ヨーク中の磁界測定モータや発電機において磁束を通過させるための強磁性体を鉄心またはヨークと呼び、ヨークを通過する磁束測定には、図4・8に示すようにサーチコイルと積分器が使用される。サーチコイルは電磁誘導(electromagnetic induction)の法則を利用したものであり、さぐりコイル、検出コイルとも呼ばれる。電磁誘導の法則は以下の式で表される。すなわちコイルに鎖交する磁束が時間変化することによってコイルに電圧が誘起されるので、この電圧を積分することによってサーチコイルを通過する磁束密度を測定することができる。図4・7 ホール効果による磁界測定図4・6 計測対象になる磁界図4・8 サーチコイルによる磁束密度測定8―80電気・電子の基礎、化学の基礎
