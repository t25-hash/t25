# 4・1・5 磁化特性測定

> **章**: 第4章 電気計測
> **節**: 4・1 電気・磁気量の測定

モータや発電機など電気機械の特性は、構成材料として用いる磁石の特性と磁束を通過させるヨーク(例えば電磁鋼)の磁化特性(magnetization characteristics)によって左右される。また、構造用鋼材においても、機械的特性と磁気特性に関連性があることから、それらの磁化特性が測定される。磁化特性の測定法はJIS C 2550,JIS C 2561およびJIS C2501に詳しく規定されている。また、一般に磁性材料は、図4・9のようなヒステリシス曲線を示す。磁石はの値が大きく、ハード磁性材料、それに対して、電磁鋼はの値が小さくソフト磁性材料と呼ばれている。鉄損の測定についてもJIS で詳しく定められている。a. 直流磁化特性測定。ソフト磁性材料の磁化特性測定図4・10に示すように、測定試料(リング状磁性材料)に巻数の一次コイルと巻数の二次コイルを一様に巻く。測定を始める前に、リング状磁性材料の残留磁化を消去する必要がある。. ハード磁性材料の磁化特性測定リング状にできない測定対象や磁石などは、図4・11に示すように、高透磁率の磁性材料で測定対象を挟んだ磁気回路を構成し、さらに、測定対象に生じる反磁界を補償することによって測定対象の磁化特性を測定する。b. 交流磁化特性測定直流の磁化特性測定と同様な構成で、励磁電源を交流にしてB-H 特性を記録する。一般的には磁束密度を正弦波にしたときの磁化特性を測定している。c. 鉄損の測定電気機械における電力損失の多くは、巻線の抵抗によるジュール損と磁心の鉄損で占められ、発熱の原因になっている。そのため、電気機械の効率を評価するうえで、磁性体の鉄損を測定することが重要である。鉄損は、ヒステリシス曲線の囲む面積であるが、図4・12に示すように一次コイルに流れる励磁電流と二次コイルに誘起される電圧の積の平均値から測定する電力計法が用いられる。商用周波数において、JIS 規格で定められているように測定対象の磁性体を短冊状にしたエプスタイン装置で鉄損が測定される。周波数を変えて鉄損を測定することによって、ヒステリシス損と渦電流損を分離することができる。
