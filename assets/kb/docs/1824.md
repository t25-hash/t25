# 4・2・2 熱電対、サーミスタ、白金測温抵抗体

> **章**: 第4章 電気計測
> **節**: 4・16 に示すように、エンコーダのスケールパターンに工夫をして

温度を計測するセンサとして、熱電対(thermo couple)、サーミスタ、白金測温抵抗体などがある。このうち熱電対は、温度変化を電圧変化によって、サーミスタと白金測温抵抗体は抵抗変化によって測定するものである。種々の熱電対や白金測温抵抗体の特性についてはJIS に詳細に記されている。a. ゼーベック効果と熱電対図4・17に示すように2種類の金属I,II(または半導体)の両端を接合して、それぞれの接合端の温度を,として温度差を与えると回路に起電力を生じる。これを特に熱起電力と呼び、この現象をゼーベック効果(Seebeckeffect)と呼ぶ。このとき、発生する熱起電力は、2接点の温度差と、2種類の金属によって決まる。この効果を用いた温度センサが熱電対(thermocouple)である。熱電対を使用するうえでは、以下の二つの法則が重要である。. 中間温度の法則中間温度の法則は温度について熱起電力が加算的であることを示したものである。図4・18に示すように2種類の金属線を使って温度がそれぞれ異なる三つの状態(a)(b)(c)を考える。(a)では、二つの接点の温度をそれぞれ、,として、この温度差によって発生する熱起電力をと表現する。このとき、発生する熱起電力の間には以下の関係が成立する。=+(4・12)この関係を用いると、例えば、を0°Cに設定して熱起電力の特性を測定しておけば、基準接点と任意の温度接点の組合せに対応する起電力を求めることができる。図4・17 ゼーベック効果(a)(b)(c)図4・18 中間温度の法則(c)(b)(a)図4・19中間金属の法則。 中間金属の法則中間金属の法則は、2本の金属線を用いて熱電対を構成し、閉回路を切断した二つの端点から別の金属を用いて電圧計に接続しても、熱起電力はもともとの基準接点と測温接点の温度差のみにより決まり、途中の温度の影響を受けないことを示す。この法則は、図4・19に示すように、熱起電力が金属の材質について加算的であることを示すものでもある。すなわち、図4・19に示すように3種類の金属線A,B,Cを用いて三組の熱電対を作る。この性質を用いると、例えば金属Cを白金としていろいろな金属の熱起電力特性を求めておけば、それから任意の金属の組合せによる熱電対の熱起電力がわかることになる。b. サーミスタサーミスタ(thermister)はニッケル、マンガン、コバルトなどの金属化合物の粉末を測定用の導線とともに焼結し、その表面をガラスなどでコーティングして、図4・20のような形状に仕上げたものである。サーミスタの抵抗変化に関する温度係数は、サーミスタの含有物質の種類や量などにより、正、負どちらの特性をもったサーミスタを作ることも可能であるが、一般には、抵抗値の温度係数が負の特性を示すものが多く用いられる。サーミスタは白金測温抵抗体に比べると、抵抗値、温度係数ともに大きく使いやすいが、使用できる温度範囲は、熱電対や白金測温抵抗体に比べると劣る。c. 白金測温抵抗体金属線の抵抗の値は温度の変化とともに変化するので、金属線の抵抗値を知ることによって温度を知ることができる。一般に用いられている測温抵抗体は白金であり、その構造は細い白金線をガラスや磁器などの絶縁体の棒に無誘電巻きにしたものである。この巻き方は、折返した白金線の平行関係を維持したまま先端まで巻付けるもので、電磁誘導の影響を受けにくくするための工夫である。0~200°Cまでの典型的な白金抵抗線の抵抗と温度の関係は次のとおりある。
