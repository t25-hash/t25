# 4・2・3 ひずみゲージ

> **章**: 第4章 電気計測
> **節**: 4・16 に示すように、エンコーダのスケールパターンに工夫をして

a. ひずみゲージの特性変形または変位を電気抵抗の変化に変換するセンサに、ひずみゲージ(strain gage)がある。ひずみゲージは、図4・21に示すようにプラスチックフィルムなどをベース材としてこれに金属線(フォイルゲージ)を貼り付けたものである。測定対象のひずみによる変形が金属線の軸方向に対応するように貼り付けると、この測定対象にひずみが生じるとひずみゲージが変形し、金属線の伸張または圧縮を起こす。ここで、Δρ/ρは比抵抗の変化率であり、金属の場合には他の項に比べて非常に小さく無視できる。b. ホイートストンブリッジによるひずみ測定ひずみゲージに生じる微小な抵抗変化を精密に測定するためには、ホイートストンブリッジ(Wheatstone bridge)が用いられる。ひずみゲージをブリッジに組む場合ブリッジには直流電圧を印加する方法と、交流電圧を印加する場合があるが、ここでは、直流の場合について説明する。まず、最も簡単な場合として、図4・22に示すようにひずみゲージを1枚だけブリッジに組込み(円で囲んだ抵抗がひずみゲージを示す)、ひずみを生じていない条件下でブリッジの平衡をとった場合について考える。これよりひずみと不平衡電位差の間には比例関係があり、不平衡電位差を測定すればひずみを測定できることがわかる。次に、図4・22の場合をさらに拡張して、図4・23のようにブリッジの各辺にひずみゲージを組込んだ場合について考える。式(4・22)より,の位置のひずみゲージ出力は正で、また、,の出力は負値として不平衡電位に寄与しているので、式(4・22)の性質を利用すると種々の測定ができることがわかる。例えば、のひずみゲージを測定対象に接着し、また、のひずみゲージを測定対象近傍に設置する場合を考える。このブリッジに温度変化を生じると、温度変化による見掛けの抵抗変化は双方のひずみゲージともに等しいので、このようなブリッジ回路を組むと温度変化により生じる抵抗変化は相殺されることがわかる。これは直接ひずみ測定に関係しない(これをダミーゲージと呼ぶ)による温度補償にほかならない。また、板材の表に+のひずみ(伸び)、裏側に-のひずみ(縮み)を生じるときには、,にそれぞれ表裏のひずみゲージを、,にそれぞれに対応するダミーゲージを配置すると、1枚のゲージの場合に比べて2倍の出力電圧が得られると同時に温度補償もされた構成になる。
