# 16・5・2 適応係数

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

気体分子が固体原子と干渉すると，気体分子と固体原子の間でエ
ネルギーと運動量が交換される．エネルギー交換と運動量交換の素
過程の表現として適応係数（accommodation coeﬃcient ）が定義
されている ．例えば，エネルギー適応係数は式(16･20)で与えら
れる．

温度に相当するマクスウェル分布であるときの平均並進運動エネル
ギーである．運動量適応係数も，表面に対する接線方向および法線
方向の運動量について同様に定義される．拡散反射の場合にはα＝
1となる．固体表面温度より低い温度の平衡状態で散乱する場合に
は，反射分布束はcosθに比例するがα≠1となり，これを余弦散
乱と呼ぶ．
