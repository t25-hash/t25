# 4・2・4 超音波マイクロホン

> **章**: 第4章 電気計測
> **節**: 4・16 に示すように、エンコーダのスケールパターンに工夫をして

可聴域以上の音波を超音波(super sonic)と呼び、この信号を扱うセンサが超音波マイクロホンである。現在、23kHz,40kHz,200kHz などのセンサが市販されている。送信・受信別型センサと兼用型があるが、基本的には同様な構造である。また、扱う信号の基本周波数が数十kHz から数百kHz であるので、これらを処理する電子回路の帯域も同程度必要となる。超音波センサは、音波の戻り時間を測定して対象までの距離を計測したり、設定距離内に障害物があるかどうかを判断するために用いられる。測定対象の材質、色などによらず反射波が得られるという利点があるが、多重反射を生じることもあるのでこれに対応する工夫が必要になる。超音波センサの構造は、図4・24に示すように金属板と圧電セラミックスとを貼り合わせた構造のバイモルフ型振動子に、共振子を接合した複合振動体をベースに弾性固定して、これをケースに収めた構造になっている。ここで、圧電セラミックスに電圧を印加すると、加えた電圧の値と加える電圧の周波数に応じた機械的なひずみが発生して圧電セラミックスは振動し、この逆に圧電セラミックスに振動を加えると電荷が発生して電極間に電圧が生じる。圧電セラミックスを2枚か、または圧電セラミックスと金属板をそれぞれ1枚貼り合わせた構造(バイモルフ型振動板)の圧電セラミックスの電極に電気信号を加えると、圧電セラミックスの屈曲振動により超音波が放射される。また、この逆にバイモルフ型振動子に超音波振図4・22 1ゲージ対応するホイートストンブリッジ図4・23 4ゲージに対応するホイートストンブリッジ図4・24 超音波マイクロホンの構造図4・21 ひずみゲージ8―84電気・電子の基礎、化学の基礎動が加わると、電圧が圧電セラミックス内に発生し、圧電セラミックスの電極から電気信号を取出すことができる。
