# 4・2・6 磁気を利用した機械量センサ

> **章**: 第4章 電気計測
> **節**: 4・16 に示すように,エンコーダのスケールパターンに工夫をして

a. 渦電流式変位センサ(eddy current type displacement sen-sor)図4・25に渦電流式変位センサの原理を示す.すなわち,1MHz程度の高周波電流を流したコイルに導体を近づけると,導体中に磁束の変化を妨げるように渦電流が流れる.この電磁誘導作用によって,コイルのインピーダンスが変化する.コイルと測定対象となる金属の間の距離,すなわち,変位が小さいと渦電流が多く流れ,コイルインピーダンスの変化も大きくなる.コイルインピーダンス変化をブリッジ回路で取出して,電圧信号とすることによって,変位にほぼ比例した信号を取出すことができるが,その直線性を改善するためリニアライズ回路が使用されている.一般に,このリニアライズ回路は,測定対象に対応して調整・作成されるので,センサの製作時と異なる金属を測定対象とするときには注意が必要である.渦電流式変位センサには,以下の特徴がある.油,水,ほこりなどの影響は受けにくい.比較的小型のセンサヘッドで高速応答(DC~20kHz),高分解能(0.01 m)である.b. 応力測定用磁気異方性センサ強磁性体に応力が加わると磁わい効果等により,磁気特性が変化するため,磁気特性変化から応力を非接触・非破壊で計測する試みがいろいろ行われている.磁気特性変化を利用した応力測定法は,実構造物の応力測定に適用でき,突合せ溶接した溶接部近傍の残留応力分布測定,鋼板の張力測定,レール軸応力測定,橋りょうの応力測定,ボルトの軸応力測定,高周波焼入れ層の判定などへの応用例が報告されている.磁気異方性センサ(magnetic anisotropy sensor)は,測定対象の磁気異方性を測定するプローブと,励振回路や同期検波回路からなる信号処理部から構成され,測定対象内の応力を被接触で測定することができる.図4・26は,磁気異方性センサの原理を示している.正の磁わい定数をもつ測定対象に,応力を印加すると,引張方向の透磁率が大きくなり,直交方向の透磁率が小さくなる.すなわち方向別の透磁率分布の異方性が生じる.磁気異方性センサを測定対象に近づけ励振コアを励振し,測定対象上でセンサヘッドを回転させると,検出コイルに電圧が誘起される.誘起された電圧を同期検波し直流電圧にし,これを検出電圧とすると以下のように表される.=σ-σcos 2θ V(4・24)ここで,σ,σは測定対象に加わっている応力であり,θはプロー8―85電気・電子の基礎図4・27 磁気異方性センサによる曲げ応力測定図4・28 エリアシングブの方向と主応力σ方向の間の角である.検出電圧は,センサヘッドの回転角πに対して,一周期の信号になる.この信号の最大値と最小値の差をセンサの出力電圧とする.この信号出力から測定対象の磁気異方性の大きさと方向がわかる.これをもとにして,主応力(principal stress)差σ-σとその主応力の方向がわかる.図4・27は,厚さ1mmの低炭素鋼板(SPCC)に曲げ応力を加えたときの磁気異方性センサの出力特性の実験値と計算値を示している.磁束の浸透深さを考慮した計算値は,実験値によく一致している.磁気異方性センサによって,主応力方向θと主応力差σ-σを測定できる.ここで,せん断応力差積分法を適用すると,任意方向の平面応力を測定することができる.この方法は,比較的単純な応力状態の場合,ひずみゲージで応力開放により測定した値との差は約±15MPa,応力こう配がある場合でも約±20MPaの差で測定できることが報告されている.磁気ひずみによる応力測定法は,応力の絶対値測定が可能であり,塗装などを除去するなどの必要がなく,皮膜上から非接触で応力を測定できるという利点を有している.トルクによって,シャフト表面に応力が生じることを利用して,磁気ひずみ方式のトルクセンサが開発研究されている.
