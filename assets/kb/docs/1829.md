# 4・3・1 フィルタ

> **章**: 第4章 電気計測
> **節**: 4・3 電気信号の取込み

フィルタ(filter)は特定の周波数の信号を通過させたり阻止させたりする働きをする。その特性からローパスフィルタ(低域通過ろ波器)、ハイパスフィルタ(高域通過ろ波器)、バンドパスフィルタ(帯域通過ろ波器)、バンドエリミネーションフィルタ(帯域阻止ろ波器)に分類される。また例外としてオールパスフィルタ(全域通過フィルタ)があり、すべての周波数範囲の信号を通過させ、位相だけを変化させるフィルタで制御系の位相余裕改善などの目的に用いられる。フィルタはノイズ除去やエリアシング防止のためなどに用いられている。フィルタにより帯域を制限することにより不要なノイズ成分を除去し目的の信号を際立たせることができる。また、サンプリング定理によりサンプリング周波数の1/2を超える周波数の成分を含む信号をサンプリングすると本来とは異なる周波数の成分が発生する。例えば図4・28に示すように○で示す点で8―86電気・電子の基礎、化学の基礎サンプリングすれば破線で示すとおりの原波形を再現できるが、サンプリング間隔が長くなり□で示す点でサンプリングすると実線で示すような波形として再生される。これをエリアシングと呼び、映画などで車輪が反対向きに回転しているように見えるのと同じ現象である。エリアシングを除去するためにはサンプリングする装置(A-D 変換器など)の前に、サンプリング周波数の1/2を超える周波数の成分を除去する特性のフィルタを挿入すればよい。フィルタの実現の方法により、アナログ回路により実現されるアナログフィルタ、ディジタル演算により実現されるディジタルフィルタに分けられる。またアナログフィルタは抵抗、コンデンサ、コイルなどの受動素子のみから構成されるパッシブフィルタ、アンプ等の能動素子により構成されるアクティブフィルタに分類される。周波数特性からバターワースフィルタ、チェビシェフフィルタ、ベッセルフィルタに分類することができる。バターワースフィルタは通過域では特性が最も平坦で減衰域は6n[dB/oct]で単調に減衰する特性を有する。チェビシェフフィルタは通過域の特性に波状の変動(リップル)があり、その代わりに減衰域での遮断特性を急峻にすることができる。ベッセルフィルタは位相変動が小さく、波形ひずみが小さいのが特徴である。その分遮断特性はいちばん滑らかである。ディジタル制御系の入力部にエリアシング防止のために挿入するフィルタでは位相変動の小さなベッセルフィルタを用いるとよい制御特性が得られることが多い。
