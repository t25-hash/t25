# 16・5・6 分子動力学（MD）法による解析

> **章**: 第16章 希薄気体力学
> **節**: 16・5 気体分子の固体表面との干渉

分子動力学法（molecular dynamics method）は，対象を原子
や分子を表すモデル粒子の集合体としてモデル化し，これらの粒子
の運動をニュートンの運動方程式に従って追跡することで現象を解
析する手法である．分子に働く力やモーメントは分子間ポテンシャ
ルから計算される．
