# 4・3・3 A-D 変換

> **章**: 第4章 電気計測
> **節**: 4・3 電気信号の取込み

A-D 変換器(analog-digital converter)には大きく分けると、フラッシュ型、逐次比較型、オーバサンプリング型がある。フラッシュ型は図4・35に示すように入力電圧を複数のしきい値と同時に比較して入力信号の値をディジタル信号に変換する方法である。フラッシュ型は原理的に高速動作に適しているために、ビデオ信号などの高い周波数の信号、高速で変化する信号の変換に適している。逐次比較型は内部にD-A 変換器を有し、このD-A 変換器の出力と入力信号とを比較し、D-A 変換器の出力を変化させて一致する値を求める方法である。通常の計装用には逐次比較型が広く用いられている。オーバサンプリング型は必要とされるサンプリング周波数よりも図4・33 差動増幅回路図4・34 ローパスフィルタ高い周波数でサンプリングし、間引き処理(デシメーション)をして必要とされるサンプリング周波数の信号を得る方法で、プレフィルタの特性を緩やかにし実現容易にしたり、変換精度を向上したりできる特徴がある。倍オーバサンプリング(サンプリング周波数が倍)の場合、サンプリング定理によりエリアシングが発生するナイキスト周波数も倍となる。すなわち、プレフィルタのカットオフ周波数も倍とすることができ、要求されるフィルタの特性を緩和することができる。ただし、デシメーション後のナイキスト周波数は本来のナイキスト周波数と変わらなくなるが、デシメーションの過程でディジタル演算により実現されるディジタルフィルタにより急峻な特性を容易に実現することができる。オーバサンプリングに使用されるA-D 変換器にはフラッシュ型、逐次比較型などの従来型の変換器と、変換の残差を次回の変換に反映させるデルタシグマ型変換器とがある。A-D 変換器にフラッシュ型、逐次比較型を使用した場合、倍のオーバサンプリングで精度が倍となる。
