# 4・3・4 シールドとグランド

> **章**: 第4章 電気計測
> **節**: 4・3 電気信号の取込み

a. ノイズ混入の原因ノイズ(noise、電気的雑音)防止の基本は、対象となる信号線をノイズ源から離すことである。しかし、ノイズ源から距離をおいてもさまざまな経路でノイズは目的とする信号に混入してくる。このように離れた信号線にノイズが発生することを誘導といい、静電誘導(electrostatic induction)と電磁誘導(electromagnetic induction)とに分けられる。静電誘導は図4・36に示すようにノイズ源と信号線との間に浮遊容量を介した静電結合がある場合に発生する。このとき、信号線にはノイズ源の電圧がノイズ源と信号線、信号線とグランドの間の浮遊容量,によって分圧されるため、信号線に誘起されるノイズ電圧は以下のとおりとなる。b. シールドこれらの誘導ノイズを防ぐためにシールド(shield、遮へい)という対策がとられる。シールドには誘導に対応して静電シールドと電磁シールドとがある。静電シールドは信号線の周りを良導体で囲み、良導体を一定電位に保つ(通常グランドに接続する)ことによりノイズ源と信号線との間の静電結合をなくして静電誘導を防ぐ方法である。電磁シールドは信号線の周りを透磁率の大きな材料、いわゆる磁性材料で囲みノイズ源と信号線との間の電磁結合をなくして電磁誘導を防ぐ方法である。また高周波領域(おおむね100MHz 以上)では先の静電シールドと同じく信号線を良導体で囲むことで表皮効果により電磁シールドの効果も期待できるが、低周波領域ではその効果は低下する。c. ツイステッドペア一方、図4・38に示すように信号源とその信号の受信端の間を結ぶ配線は必ずループを形成する。ループに誘起される電磁誘導電圧はループを貫く磁力線の本数に比例する。ループを貫く磁力線の本数はループの面積は小さいほど少なくなる。この点に着目すると、信号源とその信号の受信端の間を結ぶ配線は、ループ面積が最小となるように配置する必要がある。ループ面積を最小とする意味で、シールド線(shielded cable)は電磁誘導ノイズ防止に効果がある。さらに図4・39のようにより線(ツイステッドペア)とすることにより電磁誘導電圧を相殺させることができる。ツイステッドペアはシールド線と並んでノイズ対策のために広く用いられる方法である。d. グランドループ信号源、受信端が大地にアースされている場合には、図4・40に示すようにツイステッドペアで結ばれていても、大地を介したループが形成され、ループ内を貫く磁力線により誘導電圧が誘起される。このような大地にアースした機器間の電磁誘導ノイズを誘起させるループはグランドループと呼ばれている。グランドループによるノイズの誘起を防ぐために複数箇所でアースせずに1箇所でアースする一点アースと呼ばれる方法が広く用いられている。また安全上の理由などにより、機器を個別に大地にアースしなければならない場合には機器間を結ぶ信号線をトランス、フォトカッ図4・38 ループへの電磁誘導図4・39ツイステッドペア8―88電気・電子の基礎、化学の基礎図4・41 インピーダンスマッチングプラ、絶縁アンプなどで絶縁することが広く行われている。コモンモードチョークで高周波的に絶縁することも行われている。また信号源のグランド側の信号を受信端側のグランドに接続せずに、差動アンプで受ける方法も有効である。
