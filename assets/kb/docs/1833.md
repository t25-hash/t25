# 4・3・5 インピーダンスマッチングとリーク

> **章**: 第4章 電気計測
> **節**: 4・3 電気信号の取込み

伝送する信号の周波数が高く,波長に対して伝送距離が無視できなくなると,反射波の影響が生じる.この反射を防ぐために,図性インピーダンス,そして負荷のインピーダンスを等しくしなければならない.このようにインピーダンスを一致させることをインピーダンスマッチング(impedance matching)と呼ぶ.伝送する信号の周波数が低く,反射波の影響を無視できる場合には特にインピーダンスマッチングに注意を払わなくともよい.ただしに比べてが小さい場合にはによる電圧降下により負荷に印加される電圧は次式に示すように低下する.=+(4・40)ただし,:負荷インピーダンス∞時の信号源の出力.このような出力低下はセンサの出力を計測する場合には計測誤差となり問題となるので,による電圧降下を防ぐために≫とすることが多い.特にが大きい場合には信号入力端子のリーク(漏れ電流)やA-D 変換器ではサンプルホールド回路への充電電流にも注意を払う必要がある.またこのような計測に際しての,信号源にとって負荷となる計測装置の影響を避けるために4端子法などの方法がある.また,高インピーダンスの信号の計測に際して他の信号源,電源からのリークにより誤差の生じないように,ガードリングなどを設けることも広く行われている.詳細は4・1・1・a~4・1・1・c項に記されているので参照されたい.
