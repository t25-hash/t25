# 4・4・2 代表的な測定器

> **章**: 第4章 電気計測
> **節**: 4・4 計測システム・測定器

測定器にはテスタ、ディジタルマルチメータのように機能が単純で操作が簡単なものからオシロスコープのように機能が多岐にわたり操作が複雑なものまである。特にディジタル式オシロスコープなどは機能が多岐にわたるうえ、各機能の設定画面のディレクトリ体系がメーカによりまちまちであることが操作を難しくしている。「習うより慣れろ」の格言どおり、操作に慣れることが最も重要なことであり、格好の測定のチャンスを逃さないための秘訣でもある。ただし、計測器の機能を正しく理解する(習う)ことは正確な、また安全な計測のために不可欠であり、マニュアルを読み理解したうえで操作方法に慣れることが重要であることはいうまでもないことである。a. テスタ、ディジタルマルチメータ最も基本的な測定器がテスタ、ディジタルマルチメータである。電圧、電流、抵抗の測定が可能で、中には静電容量、(トランジスタの電流増幅率)、周波数の測定が可能なものもある。測定レンジの切換えはディジタルマルチメータは自動であるが、テスタは手動による切換えが必要である。測定の際には測定範囲を超えぬように慎重にレンジを選択しなければならない。万一レンジの最大測定値を超えた場合にはメータ本体は保護されるが、ヒューズ、抵抗器などの焼損につながることがある。また、高電圧の測定ではレンジの選択を誤ると操作者の身体、生命に危険が及ぶことがあるので特に注意が必要である。b. クランプメータ測定対象の電路をクランプで挟むことにより、ピックアップコイルと電磁的に結合させ、電流トランスを形成し交流電流を測定するものである。補助的にテスタのようにテストリードにより、直流/交流電圧、抵抗などを計測できるものもある。c. データロガー電圧、温度などのデータを測定、記録する機能をもつ測定器である。記録されたデータはPCに転送することができ、4・4・1項で述べたPC応用計測システムと同様に、表計算ソフトを使用して表形式に整理したり、グラフ化して表示したり、各種統計処理を施すことが容易に可能である。遠隔地での計測が可能なように蓄電池による長時間動作が可能なものも数多くある。d. オシロスコープ信号波形を観測するための測定器で、画面の縦軸を信号の振幅、横軸を時刻として信号波形に同期して横軸方向に掃引させて波形を表示するのが代表的な使用方法である。測定に当たっては振幅(画面上で1目盛当たりの電圧)、掃引(スイープ)速度(1目盛当たりの時間)、掃引タイミングを信号に同期させるためのトリガを適切に選択する必要がある。従来からあったアナログ型のオシロスコープに加えて、最近登場したディジタルスコープは振幅、掃引速度などを自動的に選択する機能も備えていて便利である。しかしサンプリング定理によるエリアシングには注意が必要である。オシロスコープの使用方法として、縦軸、横軸に異なる信号の振幅を入力し、リサージュ図形などを描かせる使用方法もある。e. ユニバーサルカウンタ基準発振器を備え、その発振周期を基準に入力信号の周期、周波数、パルス幅を計測する測定器である。通常数百MHz までの周波数が計測できるが、プリスケーラと呼ばれる周波数を1/N にする(分周する)回路・装置を介すことで測定範囲をGHz 帯まで拡大することができる。f. ファンクションジェネレータ任意の波形を生成する測定器で種々の測定の信号源に用いる。通常5~10MHz までの周波数のサイン波、三角波を出力することができる。そのほかにスロープ、任意階段波形、ノイズ波形、心電図波形、AM,FM などの変調波などを出力できるもの、いったん取込んだアナログ信号を任意のタイミングで再生して出力する機能があるものもある。類似の機能をもつものにシグナルジェネレータ、パルスジェネレータがある。シグナルジェネレータは通信用の信号の発生に重点をおき、特にGHz帯までの各種変調方式による変調波の出力が可能である。パルスジェネレータはおもにディジタル回路を動作させるためのパルス(矩形波)の発生に重点をおき任意の幅のパルスを任意のタイミング(トリガからの遅延)で出力する機能をもつ。g. ロックインアンプ特に雑音に埋もれた周期性のある信号を検出するのに威力を発揮する測定器である。ロックインアンプには測定対象の信号を入力する端子と参照信号を入力する端子とがある。参照信号には抽出すべき信号波形を入力し、ロックインアンプは測定対象の信号と参照信号とを乗じて、その乗算結果の強度を測定するものである。乗算結果の強度を測定する際に不要な信号成分を除去するためにローパスフィルタを通す。以上の信号操作により、ヘテロダイン(同一周波数なので特にホモダインまたはゼロダインともいう)により特定の周波数成分を容易に急峻な選択性により抽出できるだけでなく、波形の相関をとることにより特定の周波数、特定の波形の信号をコヒーレントに抽出することができる。設定した位相成分だけを測定できる1位相型と、直交する位相成分も同時に測定できる2位相型がある。2位相型を用いると、直交する位相成分の強度の比のtanから位相角を測定することができる。h. B-H アナライザモータなどの電気機械を構成する磁性体、すなわちヨークの交流磁化特性(B-H 特性)は、その効率や発熱を知るうえで極めて重要であり、その測定法はJIS で規定されている。また、スイッチング電源やDC-DCコンバータに使用される高周波トランスは使用周波数を高周波化するに伴い損失が増大する。損失の原因は、ヨークにおけるヒステリシス損失や過電流損失等からなり発熱により温度上昇を招く。このためソフト磁性部品の損失(磁心損失:コアロス)を評価し、低損失部品を使用することが重要である。B-H アナライザは、交流B-H 特性を簡便に測定する測定器であり、B-H 特性から、保磁力、残留磁束密度、鉄損(コアロス)等を精度よく測定できるディジタル測定器である。B-H アナライザによって、モータ、アクチュエータ、リレー、電源等のヨーク材に使用されるケイ素鋼、軟鉄、パーマロイ、ステンレス鋼、フェライト等の磁化特性測定に使用される。m. テスラ/ガウスメータテスラメータ(Tesla meter)は、計測用ホール素子を使って空間の磁束密度を測定するための計測器である。計測用ホール素子は、InAnやInGaAsなどでできていて、安定性、温度特性、直線性等の点で優れている。磁束密度の単位は、T(テスラ)であるが、SI単位系以前のCGS 単位系でG(ガウス)が用いられていたためガウスメータと呼ばれることもある。代表的なテスラメータの基本仕様を以下に記す。・測定レンジ:1 T~10T・10測定レンジ:オートレンジ切換え・AC,DC、パルス磁界の測定・周波数応答:DC~10kHz・測定モード:標準、最大、比較、相対、温度・T(テスラ)またはG(ガウス)の測定単位表示の選択・ホール素子の温度補正が自動で可能テスラメータの表示は、普通、ホール素子の単位面積当たりの磁束密度を平均値となっているので、素子サイズによって、同じ場所を測定しても表示が異なることがある。標準的な計測用ホール素子のサイズは1mm×0.8mm程度であるので磁束密度分布がもっと細かく変動している空間の測定、局所的磁束密度部分布の測定には注意が必要である。n. フラックスメータ(fluxmeter)フラックスメータとは、磁束量を測定するもので、サーチコイルと積分器から構成されている。磁束の単位はWb(ウエーバ)である。その基本原理は、サーチコイルを通過する磁束がサーチコイルに電圧を誘起するので、その電圧を積分し演算指示することによって磁束量を測定する。最近のフラックスメータには、演算増幅器を用いた積分V-F変換器とディジタルカウンタの構成で、ノイズやドリフトの影響を低減し、安定した高精度測定ができるものがある。基本仕様の一例を以下に記す。・測定範囲:1mWb~1Wb・分解能:10nWb・判定レンジ:4レンジ・測定モード:バイポーラ、P-P,NP,SP・判定出力:コンパレータ、絶対値、極性別ブザー・精度:1mWb・±(読取り値の2%+レンジの0.1%+50digit)10mWb以上・±(読取り値の2%+レンジの0.1%+10digit)フラックスメータは、着磁されたマグネットの磁束量測定評価、リングマグネットに多極着磁された磁束量測定、ヨークなどの磁性体中を流れる磁束量の測定、モータの磁束、着磁パルス、磁石の試験等に使用される。o.
