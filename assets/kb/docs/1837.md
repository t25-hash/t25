# 16・3・2 クランク軸系のねじり振動モデル

> **章**: 第16章 往復機械の力学
> **節**: 16・3 往復機械の自由振動

ねじり振動（torsional vibration）が支配的な振動モードを対象
とする場合は，図16･8(ａ)に示すような集中質点系等価ねじり振
動モデルが用いられる場合が多い．各気筒のクランクスロー部
（crank throat ）にはコネクティングロッドの等価回転質量
（equivalent mass moment of inertia rotating about crankshaft
axis ）が加算される．船舶用パワートレイン軸系（power train
system）が対象の場合は連れ回る水の質量を加算したプロペラの
慣性モーメント（約25%増）を考慮する．クランクスロー部のね
じり剛性（torsional stiﬀness ）は，その複雑な構造のためクラン
ク形状の実用的等価ねじり剛性を求めるいくつかの実験・経験式が
提案されており，例えば，カーター（Carter )，岩本 などの研
究によれば，クランク軸系の基本設計が特異でなければ，ほぼ満足
できるモデル化が可能である．若林らは，予測技術の向上のため，
クランクスロー部を複数個に分割し集中質点と三次元の質量のない
はり要素（ばね剛性）で表した伝達マトリックス法（transition
matrix method）を適用している ．クランク軸系の減衰要素（機
関減衰，エンジン減衰とも呼ぶ）は，理論的定量化は難しく，実験
式の導入に関する研究 が行われている．ねじりダンパ無装着の
状態での共振応答から粘性減衰（viscous damping）を仮定してエ
ネルギー法や応答曲線カーブフィッティング法（curve fitting
method）によるのが一般的である．精度のよい測定法としてエン
ジン稼働状態からの急激に燃焼を止め自由ねじり振動減衰波形を観
察する手法 も提案されている．粘性ダンパ（viscous damper ），
粘弾性ダンパ（viscoelastic damper ）をクランク軸先端に装着す
る場合は，ダンパの減衰特性が大きく，クランク軸系の減衰は省略
しても実用的には可能である．
一般に，クランク軸系のねじり振動固有モードは，エンジンの使
用回転数領域における上限の加振周波数から一次モードを考慮すれ
ば十分である場合が多い．したがって，ねじり振動ダンパのチュー
ニング（tuning）には，クランク軸系を図16･8(ｂ)のような一次
ねじりモードの等価1自由度質量ばね系に置換えて最適化の検討が
行われる場合が多い．以前は，多自由度の集中質点・ばね系ねじり
振動モデルの解法にホルツァー法が広く用いられたが，最近は，計
算機の計算能力の向上に従い，16･3･3項で述べる各種マトリック
ス法や構造解析によるモードパラメータを算出する手法が用いられ
ている．
