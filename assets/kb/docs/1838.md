# 5・1・3 将来の電気エネルギー

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・1 社会生活と電気エネルギー

21世紀のわが国の電気事業は、表5・1に示すように、高齢化社会、高度情報化社会、ならびに省エネルギー、非化石エネルギーなどの社会動向の中で、電力化率の上昇に対応しながら、安定、高品質な電力を目指して、高機能な電力供給を実現していく必要がある。その前段として、分散型電源(distributed power supply)の普及を含めた規制緩和や地球温暖化(global warming)へ対応しながら、負荷率の向上やコスト低減を図っていくことが求められている。a. 21世紀中葉の電力需要将来の電力需要(electric powerdemand)を考えるに当たって、高齢化社会や高度情報化社会にお図5・9原子量発電の発電容量と設備利用率の推移表5・1 電力需要の想定(億MWh)ける電力化率の上昇と人口動態が重要な要素である。さまざまな社会情勢の変化を想定し、21世紀中葉の電力需要を推定した結果を図5・10に示す。高度情報化の進展と高齢化を念頭に、21世紀中葉の1人当たりの電力需要は約2倍になると想定した。この需要動向と将来の人口動態を掛け合わせることで、将来の電力需要を推定した。人口動態については厚生労働省人口問題研究所の人口推定を参照したが、2010年前後で人口は減少に転じ、2050年には1~1.2億人のレベルと想定されている。21世紀中葉の電力需要は現状の1.2~1.5倍に増加するものと考えられる。この場合の電力需要の伸びは平均的な値で、大都市部への一極集中が今後も進むとすれば、2倍近くに増加することも想定する必要がある。b. 電源設備(electric power installation)の想定発展途上国におけるエネルギー消費の急速な伸びを考えると、CO排出量の抑制は急務であり、化石燃料の枯渇危機も現実のものになってくる。このような地球規模の課題に対応しつつ、上で述べた電力需要に対する発電設備のシナリオを作るのは極めて難しい。21世紀初頭のエネルギー問題については、資源エネルギー庁の総合資源エネルギー調査会において、省エネルギー(energy sav-ing)や新エネルギーを多角的にとらえた審議が行われている。一方、21世紀中葉に対しては、現時点で想定できる範囲内の技術によるシナリオでは、環境と安全の両面で満足できる解は得られない。しかし、間違いないのは、継続的に化石燃料への依存度を減らさなければ、地球上の人類が滅亡に追い込まれることである。省エネルギー技術のいっそうの推進に加え、新エネルギーのさらなる可能性を追求していくことが求められる。電源から需要家に電力を供給する電力ネットワーク[electricpower network、電力系統(electric power system)]は、図5・11の概要図に示すように、電源である発電機から出た電力を昇圧して、大電力を送電可能とするとともに、長距離送電による損失の低減を図っている。一方、受電側では、表5・2に示すように、負荷電力の大きさに従って、特別高圧、高圧、低圧の電圧階級の配電線により需要家に供給している。これらの送電線、配電線をつなぐのが変圧器で、電力用コンデンサ、並列リアクトル、送電線(power transmission line)と配電線の事故に対する保護装置、開閉器などを含めて変電所(powertransforming station)を構成している。この変電所も、電圧階級により、超々高圧、超高圧、高圧(二次)、配電変電所に分類され、配電系統(power distribution system)では配電塔、柱上変圧器により配電線をつなげており、樹技状系統あるいはネットワークを構成している。このほか、電力ネットワークを構成する機器として、直流送電(周波数変換を含む)における交直変換器があるが、直流送電については別途記述(γ5編「エネルギー供給システム」第8章)があるので、ここでは省略する。8―9 3図5・11 電力系統構成の概要図表5・2
