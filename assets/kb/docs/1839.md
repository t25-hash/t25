# 5・2・1 電力ネットワークの発展

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・2 電力ネットワーク

電源の立地が遠隔地化、大容量化することに対しては、送電線を高電圧化して大容量化することで対応してきた。電力ネットワークの発展は図5・12に示すとおりで、発電機容量が伸びるとともに、変電所の容量、送電線や配電線の亘長が増加してきた。また、送電容量の増加に対して、ネットワークの構成機器の高電圧、大容量化、ならびに高機能化が進められてきた。a. 架空送電線架空送電線は大電力を長距離に送電するための基本となる設備で、電気事業が始まって以来、着実に発展を続けている。電力再編以来、戦後復興期における遠距離の水力電源からの送電、経済成長期には需要家近傍の火力電源の連系、さらにオイルショック以降は遠距離立地のLNG、原子力電源からの送電として発展を続けてきた。また、電力の経済的利用、緊急時応援などのための系統間連系にも活用され、現在は沖縄を除き、一部に直流連系、周波数連系を介して全国連系が完成している。架空送電線の大容量化の様相を図5・13に示す。154kV 送電線は、戦後しばらくの間は、水力電源からの送電線8―9 4電気・電子の基礎、化学の基礎として基幹系統に用いられてきたが、275kV 送電線の普及、500kV 送電線の開発に伴い、二次系統として扱われるようになった。この154kV 高圧送電線と275kV を含む超高圧送電線において、絶縁設計の大幅な違いがある。前者では中性線の抵抗接地など有効接地方式を採用し、落雷等による送電線事故において、健全相に過電圧が生じるのを許容して地絡電流を抑制しているのに対し、後者では直接接地方式を採用し、地絡電流が増えることを許容して絶縁設計の合理化を実現している(図5・14)。b. 地中送電線地中送電は、都市部の電力供給方式として、図5・15のような形で発展してきた。高度成長期に154kV,275kV の送電線に採用されてきたが、500kV ケーブルや地下変電所が開発されるに伴い、外輪系統から都市中心部への電力供給系統が構築されるに至った。地中送電線は油で絶縁、冷却を行うOFケーブルから実用化が始まったが、油の供給等の設備を必要としない固体絶縁のCV ケーブルの実用化が進んでいる。500kV 級CV ケーブル(長距離)の普及には、接続部の絶縁厚の低減が重要な課題となっている。今後、都市部への電力供給はいっそう重要となり、大容量化が必要となる。すなわち、送電線用地獲得の困難さから、既存ルートでできる限りの電力を送ることが求められる。このため、新しい方式の地中送電線(underground transmission line)の開発が進められている(図5・16)。その究極は超伝導ケーブルであるが、管路気中送電や交流送電線の直流化などがそれまでの代替と考えられる。c. 変電所変電所は、低圧電力を高圧に昇圧したり、あるいは高圧から低圧に降圧して配分することに加え、それぞれの電圧調整、出入りする送電線、変圧器の保護などの役割をもつ。変電所は図5・17に示すように種々の機器から構成され、それぞれが制御・保護装置で有効に機能する。d. 配電系統配電系統は、一般家庭等の電灯需要家に電力を(b) 直接接地方式(超高圧送電線)(a) 有効接地方式(高圧送電線)図5・14 送電線の接地方式図5・15 地中送電線の推移図5・16 新しい地中送電線の開発供給する系統と、産業などに高品質な電力を供給する系統に分けることができ、前者は樹枝状構成で、後者はネットワーク構成となっている(図5・18)。樹枝状系統では電力が順次分配されるが、幹の系統での電流が大きく、端に行くほど小さくなるため、線路損失による電圧低下に注意する必要がある。また、幹の部分事故が末端の需要家に影響する。このため、配電系統の電圧を一定の範囲内に維持できるように、配電用変圧器の出口電圧の調整に留意する必要があり、事故の影響を軽減するため別ルートからの緊急時供給を可能としている。一方、ネットワーク系統は電圧維持が容易で、系統事故に対しても、供給支障の発生を抑えることが可能である。しかしながら、系統事故時の事故遮断が複雑であるため、負荷の運転状態を十分に考慮した保護方式の開発が重要である。
