# 5・2・2 電力システムの計画と運用・制御

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・2 電力ネットワーク

電力ネットワークに発電機、負荷を含めて電力システム(elec-tric power system)と呼び、それらを一体化した計画、運用・制御が行われている。電源とネットワークを一体にして計画することの重要性については容易に理解できる。一方、運用・制御においても、電源と電力系統が密接に関係していることに注意しなければならない。電力システムの計画と運用・制御に係わるおもな事項を表波数の制御、系統電圧の維持、ならびに系統安定度について以下に述べる。a. 周波数制御電源と需要の過不足により系統周波数は上下する。このことはガス管や水道管でのガス量、水量が変化することに相当し、理解されやすい現象であるが、その影響の出方はかなり異なる。周波数低下が大きく長く続くと、電源発電機のタービンブレードへの影響や、原子力プラントの補機動力の低下によるプラントトリップなどの形で現れ、製紙工場や繊維工場等の需要家においては、製品不良に結びつくことがある。また、一般の負荷機器の入力の減少が機器の効率低下をもたらすことになる。電力システムの周波数は、図5・19に示すように、電源と負荷の差の積分によって決まるが、電源のガバナフリー応答と負荷の周波図5・17 quality)として重要な課題である。雷害による瞬時電圧低下やサージ性過電圧、負荷変動に伴う電圧変動、さらには不平衡電圧、高調波などの対策が求められており、そのベースとなる定常あるいは準定常状態における電圧維持が重要となる。また、高電圧における機器(電力側、需要側共)の安全性、低電圧における機器の運転性能の面からも、電圧維持は重要な課題である。このための方策として、電力系統につながっている発電機が有効に機能するが、電源立地が遠隔地化するに従って、受電側での電圧維持が不可欠になり、電圧・無効電力制御が実施されている。ここで無効電力の制御による電圧維持について述べる前に、無効電力の性質を述べる必要がある。送電線に電圧を印加した場合の送電(有効)電力は、相手端の電圧によって決まる。すなわち、図電圧が低ければ、送電線で伝達される有効電力は小さく、送電端では無効電力のみを供給することになる。これに対し、受電端の電圧が送電端と同程度であれば、無効電力をほとんど供給することなく有効電力を伝達することができる。このためには受電端に無効電力を注入する必要がある。このように、受電側に必要な有効電力を伝達するためには、受電端の電圧維持が重要である。このため、無効電力(reactivepower)を供給する設備として、発電機があれば、それを活用する表5・3電力システムの計画、運用・制御に係わるおもな事項項目おもな事項計画電源計画・長期需要予測に基づく電源計画(地球規模環境問題、エネルギーセキュリティ)・水域、地域開発計画・立地対策・環境対策系統計画・系統構成の最適化・想定事故に対する安定性・立地対策・環境対策運用・制御運用・保守・週間運用、翌日運用計画と当日運用・電源運用計画に応じた保守計画周波数制御・ガバナフリー容量、瞬動予備力、待機予備力・経済負荷配分系統電圧維持・無効電力制御(調相制御、タップ制御)・電圧安定性系統安定度・系統安定化(AVR,PSS、制動抵抗)・緊急時制御(脱調、周波数、電圧)復旧操作・復旧手順の策定・復旧電源の確保図5・20
