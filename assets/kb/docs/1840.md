# 17・2・1 原動機と負荷の特性

> **章**: 第17章 回転機械の力学
> **節**: 17・2 駆動トルクと負荷トルク

回転機械において原動機（prime mover ）の出力軸に現れるトル
ク ［負荷側から見れば駆動トルク（driving torque）］，ならび
に負荷が所定の運動と仕事を行うに必要なトルク ［原動機側か
ら見れば負荷（load）トルク］は一般に軸の回転角速度ωによっ
て異なり，その変化の様子をトルクの回転速度特性（torque-
speed characteristics ），あるいは単にトルク特性という．トルク
特性は機械の種類ごとに特有の形をもっており，その代表例を図
17･1に示す．一般に，
(１) 原動機の特性曲線は原点を通らず，速度零のときにもトル
ク，すなわちストールトルク（stall torque）の発生がある．また，
曲線には必ず右下がりの部分があって，この部分で使用するのが機
械の正常な用い方である．
(２) 負荷の特性曲線は摩擦性のものを除き，すべて原点を通
り，一般に右上がりの曲線である．
回転機械の発生動力または所要動力 はトルク と回転角速度
ωの積＝ωである．したがって図17･1のトルク特性に対応し
た動力特性曲線が描かれ，これより機械が最大動力を発生または消
費する回転速度がわかる．原動機と負荷とでこの速度が異なるとき
は，中間に変速機を用いることが多い．
