# 17・2・2 平衡速度とその安定性（stability）

> **章**: 第17章 回転機械の力学
> **節**: 17・2 駆動トルクと負荷トルク

原動機と負荷のトルク特性曲線の交点は系の平衡速度を与える．
と の曲線の交点では上式の右辺は零だからωは一エネルギーが変化すれば図17･3に示すように両曲線の交点は移動A，Bがあるとき，交点近傍の余剰トルク＝－の正負をか，すなわち平衡速度の安定性がわかる．
図17･4で点Aは安定，れると機械はますます減速の一途をたどり，遂には停止に至る．
トルク，負荷トルクとすると，軸の回転角速度ωに関する運動方定に保たれる．
しかし，駆動側の供給エネルギーまたは負荷の消費し，平衡速度は変動する．
いま，図17･4のように2箇所の交点考えれば，この平衡点がエネルギー変動に対して復元性をもつか否点Bは不安定であり，点Bの近傍で速度がひとたび低いほうへず燃機関のエンジンストール（engine stall）はこの現象の代表例である．
