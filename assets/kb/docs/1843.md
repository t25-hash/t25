# 5・3・3 需要家システムの最近の動向

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・3 節では需要家側の電源システム(power supply system)に

需要家の電源システムも社会の動きの影響を受けて変化している。その変化を社会的ニーズ・動向を踏まえて簡単にまとめたものを表5・6に示す。需要家の電源システムを構築する場合、上に述べた需要家特有の傾向を参照しつつ、表5・6に示した社会的動向も踏まえて計画することにより、目的に沿った無駄のない電源システムを構築することができる。地球規模の環境問題への対応の観点から、太陽光発電や風力発電などの自然エネルギー(natural energy)発電が注目され、各種技術開発とエネルギー法制、補助金による実用化が進んでいる。一方、電力自由化による分散電源の普及は、化石エネルギー(fossilenergy)利用を加速させる面もあるため、エネルギー利用のいっそうの効率化が重要となっている。また、電力と熱の双方を有効活用するコージェネレーション(cogeneration)においては、高効率エネルギー利用や、これまで開発が進められてきた燃料電池など新発電の活用が注目されている。最近では、家庭利用を対象にした固体高分子燃料電池(PEFC)発電やマイクロガスタービン(MGT : micro gas tur-bine)が話題を集めている。家庭における熱利用を含めた高効率なエネルギー利用について検討が必要である。これらの新エネルギー発電や高効率エネルギー利用などの分散型電源の導入が進むにつれて、電力システムにおける電力潮流(electric power flow)が大きく変化することになり、電圧変動、電圧不平衡、高調波など電力品質に関して新たな問題が顕在化してくる。わが国の分散型電源(太陽光発電、風力発電、コージェネレーション)の導入状況は図5・37に示すとおりで、太陽光発電と風力発電は100MW 前後に過ぎないが、コージェネレーションは500万kW を超す勢いである。しかしながら、わが国の発電設備容量約1億8千万kW(内原子力が約30%)と比較すると、3%程度に過ぎない。総合資源エネルギー調査会・新エネルギー部会では、表5・7に示すように、わが国の自然エネルギーやリサイクルエネルギーの賦存量、ならびに将来の導入目標を試算している。この2010年目標値は現状の導入実績の約3倍であるが、将来のエネルギー供給計画(長期エネルギー需給見通し、2010年に6億1600万kL)に対しても、3%弱を占めるに過ぎない。
