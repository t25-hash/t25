# 5・4・3 電力貯蔵(electric power storage)

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・4 分散型電源・電力貯蔵

電力の需給調整を目的とした揚水発電所が活用されてから久しいが、国のムーンライト計画において電池電力貯蔵の技術開発が推進され、電力システムでの電力貯蔵が注目されてきた。電力貯蔵装置としては、表5・11に示すように、化学エネルギーを利用する電池電力貯蔵(BES)、磁気エネルギーを利用する超伝導電力貯蔵(SMES)、圧力による圧縮空気貯蔵(CAES)、慣性エネルギーのフライホイール電力貯蔵などがある。a. 電池電力貯蔵システム電池電力貯蔵は実用化に近い分散型電力貯蔵装置で、国内外において、実証試験や実系統運用の実績がある。これらのBES システムは配電変電所等に設置して、夜間に電力を貯蔵し、昼間のピーク時に利用する日間の負荷平準化が主目的とされているが、系統電圧の変動対策や不平衡補償などの機能が付加されており、周波数調整を目的として導入された例もある。BES システムでは、充電時には高い電圧を維持しながら十分に充電を完了させ、放電時には電力を有効に活用すべく放電させる必要があり、電池の充放電特性に合わせた交直変換器を選ぶことが肝要である。また、BES システムは設置箇所、設備容量が異なるため、導入目的に応じた設計が必要となる。BES システムによるピークカットで負荷平準化(load leveling)を行おうとした場合、配電変電所容量(~数MW)の30%をカバーするためには、BES システムの必要な設備容量は2~3MW と想定できる。このBES システムの構成例は図5・50に示すとおりで、ピークカットに加え並列変圧器の1台停止時の対応、系統電圧の補償などが可能となる。また、配電変電所の増設の繰延べによる電力価格の低減が可能となる。b. フライホイール電力貯蔵システムフライホイール電力貯蔵は、同期機と直結された回転体を加速、減速して、慣性エネルギーの増減を電力貯蔵に活用する。回転体は次の関係式に従って加減速できる。電機子方式では電機子からの入力を上げることで加速され、下げることで減速される。なお、電機子方式では同期機本体と同じ容量の変換器を必要とするのに対し、界磁方式では同期機の容量の10~20%の容量で済むことになる。一方、回転体の制御性能から見ると、電機子方式では同期機の停止状態からの起動や大幅な非同期運転が可能である。図5・54にフライホイール電力貯蔵の実用例を示す。この設備は変動負荷に対する系統周波数の対策のために導入されたもので、数秒~の電力貯蔵を行っている。
