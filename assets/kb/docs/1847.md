# 5・4・4 電力供給システムの高機能化

> **章**: 第5章 電気エネルギーシステム
> **節**: 5・52 に示すトロイダル型のコイルを用いることができ、十分に実

分散型電源・電力貯蔵を電力システムと連系するに当たっては、図5・54 周波数対策用のフライホイール電力貯蔵システムの概要図5・55 高調波による障害の発生状況図5・56 分散型電源・電力貯蔵を含む需要地系統の構成パワーエレクトロニクス技術が重要な役割を果たす。一方、電力自由化に伴い、これらの分散型電源、電力貯蔵、廃棄物発電等のリサイクル発電に加え、従来型の各種発電機が下位系統に導入される。このような場合、分散型電源は、5・4節の最初に述べた3%以下に留まらず、地域によってはさらに比率が高くなる。この場合、電力の流れは従来の上位系統から下位系統だけでなく、下位系統から上位系統へも流れ、送配電設備において利用率が悪くなったり、過負荷でボトルネックが生じることになる。この対策として、基幹系統の潮流制御、系統安定化を目的に技術開発が行われ、一部で実用化されている。a. 電力供給の信頼度わが国の電力システムの信頼度は、機器の高品質化だけでなく、予備回線や予備バンクなど系統構成の多重化等により、年々向上して、世界でも最も停電時間、停電回数の少ない国となっている。今後、電力自由化が進展すると、電力システムの信頼度の確保がますます重要な課題となってくる。電力システムの信頼度は電源のみで決まるのではなく、系統を含めた供給能力、セキュリティが重要である。b. 電力品質配電系統を含めた電力システムでは、テレビをはじめ、インバータ空調機、インバータ照明など半導体機器の普及に伴い、高調波による障害が増大している(図5・55)。一方、半導体工場や液晶工場での精密加工、データセンタなど計算機の普及により、落雷などによる電圧低下を瞬時も許さない状況になってきている。このような電力品質に対しても、個別の対策が必要とされている。一方、需要家ニーズの多様化、電力サービスの向上要求は社会の発展とともに増加するもので、このための電力供給のあり方として、マルチメニュー電力供給も検討されている。このシステムでは、需要家の負担を減らす高品質電力供給、高品質化を需要家に委ねた低コストの電力供給、需要家設備を小型化するための高周波電力供給などを選択できるようにすることを目指す。c. 需要地系統の構築電気事業を取巻く環境変化として、(1)電気事業の規制緩和による電力自由化、(2)新エネルギーへの8―105電気・電子の基礎表5・13 需要地系統における制御・保護の課題対象機器目的備考・配電線のロス低減と不平衡電圧の抑制・SVC,SVG は配電線末端に柱上設置系統電圧制御・SVR,TVR・SVC,SVG・分散型電源・電力貯蔵の変換器潮流制御・分散型電源・電力貯蔵装置・LC(load conditioner)・配電線のロス低減と負荷平準化・LC は需要家に設置、太陽発電の変動調整も行う・停電の局小化・無停電切換え方式保護協調・配電用変電所CB・二次系統、配電線の静止型開閉器関心の高まりによる分散型電源・貯蔵の導入、(3)情報化の進展に伴う多品質電力供給の必要性、(4)配電システムの高機能化のニーズ等が挙げられる。分散型電源の導入に伴う供給信頼度と電力品質の維持を念頭に、次世代に向けた新しい電力エネルギー流通システムとして需要地系統の構築の研究が進められている。その概要は図る。おもな制御・保護の課題を表5・13に示す。文献(1) 日本エネルギー経済研究所、 エネルギー・経済統計要覧、 (2004)、省エネルギーセンター。(2) 資源エネルギー庁長官官房総合政策課、 総合エネルギー統計、(2004)、通商産業研究社。(3) 電気学会編、電気工学ハンドブック第6版、(2001)、23編、電気学会。(4) 文献(3)の25~27編。(5) 文献(3)の24,28~30編。(6) 文献(3)の31編。→ Next Chapter
