# 1・1・1 化学反応の種類

> **章**: 第1章 化学反応の基礎
> **節**: 1・1 化学反応の平衡と速度

a. 単一反応と複合反応単一反応は一つの化学反応式で示されるもの(通常は不可逆反応のみ)で,複合反応はそれらが複数組合わさったものである.複合反応の基本形態には可逆反応,逐次反応,並発反応がある.現実には非常に複雑な反応系もあるが,基本的にはこれらの組合せによる.この分類を明確にしておくことは化学平衡計算を行うための第一段階である.下記に具体例を示す.単一反応:2CO+O→2CO(一酸化炭素の酸化)複合反応(逐次反応):CH+O→CH=CH-CHO+HOCH=CH-CHO+3.5O→3CO+2HO(プロピレンの部分酸化によるアクロレインの生成および逐次的な完全酸化)b. 素反応(elementary step)と反応機構a項の分類は全反応に着目したもので,反応機構に関する情報を包含していない.実際の反応の途中段階では全反応には現れない過程が起こったり,活性中間体が生成することがある.それぞれの反応段階のことを素反応(もしくは素過程)と呼ぶ.素反応の組合せ全体は反応機構に関する情報も含んでおり,反応速度式の誘導に重要である.具体例としてヨウ化カリウムを触媒とした過酸化水素の分解を示す.本反応は単一反応であるが,三つの素反応によって表される反応機構を有している.ここでは,IとIOという活性中間体が現れている.なお,Oは溶存酸素を示す.全反応:2HO→2HO+O素反応1:HO+IHO+IO素反応2:HO+IOHO+I+O素反応3:O→Oc. 均一系反応(homogeneous reaction)と不均一系反応(het-erogeneous reaction)反応が一つの相で起こる場合を均一系反応(または均一反応)と呼び,不均一系反応(または不均一反応)は二つ以上の相が反応に関与する場合である.d項の触媒反応の分類ともに,具体例を表1・1に示した.d. 触媒反応と無触媒反応触媒反応については1・2節で詳説されるので,ここでは詳しく述べない.無触媒反応は触媒の存在なしに進行し,反応物自身が自発的に反応を起こすものである.表挙げた.
