# 1・2・1 触媒反応機構

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

a. 触媒サイクル(catalysis cycle)触媒作用は本質的に複数の素過程からなる複合過程である.その素過程の成り立ちを明らかにするのが反応機構の研究である.触媒作用の仕組みが詳しくわかっている均一系の触媒作用についてまず説明しよう.ヘキスト・ワッカー法(Hochest-Wacker process)として知られるエチレンと酸素からアセトアルデヒドを作るプロセスを例とする.反応は水溶液中で行われ,PdClとCuClが触媒として溶けて存在しており,そこにCHとOが原料として供給されることを想定されたい.触媒反応:CH+12O→CHCHO(1・25)それを構成する素反応CH+PdCl+HO→CHCHO+Pd+HCl(1・26)2CuCl+Pd→CuCl+PdCl(1・27)CuCl+2HCl+12O→2CuCl+HO(1・28)反応(1・25)がわれわれの見ている触媒反応であり,それだけ見ていると触媒PdClおよびCuClは化学式には現れていない.微細に反応機構を調べてみるとそれは式(1・26)~(1・28)の三つの素過程からなっていることがわかる.つまりその素過程に立ち入って見ると,どちらの化学種も反応に直接かかわり,その化学的状態は変化している.ただ,一連の素過程が完結したときには元に戻っている.これが触媒の本質である.固体触媒でも同様である.BiO-MoOを触媒として,プロピレンと酸素からアクロレイン(CHCHCHO)を作るプロセスを考える.この触媒にプロピレンだけを供給するとアクロレインが生成する.そのとき必要となる酸素は触媒格子中の酸素原子である.すなわち,BiO-MoOの固体中の酸素がプロピレンに付加しアクロレインを生成し,触媒それ自身は還元される.もちろんその酸素原子数は限られているので反応は永久に起こり続けるわけではない.プロピレンの供給を停止し,酸素を供給すると触媒は再酸化され元に戻る.もし,二つの反応物を同時に供給すればこのサイクルは永遠に回り続ける.プロピレンと触媒の酸素からアクロレインが生成する過程もいくつかの化学反応からなる複合過程である.この反応はメチル基の水素引抜きを第一段階とし,さらに酸素が付加し水素がとられるという何段階もの過程からなっている.その複合過程の中で最も反応速度が小さい過程をその複合反応の律速段階という.反応の素過程をすべて書き下し,その中で律速段階はどこであるかを明らかにすることが触媒作用解明の目的である.b. 触媒作用についてのサバチエの原理(Sabatier’s principle)このような触媒サイクルを考えると触媒の適格性についてのサバチエの原理を容易に理解できる.触媒反応がある中間体を経て進行すると考えたとき,この原理は,中間体を作りやすく,かつそのできたものが過度に安定でもないような,中庸な物質が触媒として適格8―110電気・電子の基礎,化学の基礎図1・2サバチエの原理:ギ酸の分解反応に対する種々の金属の触媒活性とギ酸塩生成熟との関係縦軸は触媒反応速度がある一定値となるのに必要な反応温度(この値が小さいほど触媒活性は大きい)なものであると表現できる.ギ酸分解(HCOOH →H+COあるいはHO+CO)は原料であるギ酸が触媒金属原子とギ酸塩を形成し,それが分解する機構で進行する.種々の物質においてそのギ酸塩の生成熱と触媒活性をプロットしたのが図1・2である.横軸がギ酸塩生成熱であり触媒元素のギ酸に対する親和力を表現している.親和力の小さいものは中間体を形成する速度が低いゆえに活性が小さく,一方それが大きすぎるものはできたギ酸塩が安定になりすぎるがゆえに触媒活性が低いと説明される.COや炭化水素の触媒的完全酸化反応においても同じことが知られている.アルカリ金属やアルカリ土類金属は酸化物を形成しやすいがその酸素は極めて安定で反応物に受け渡されない.一方金や銀は酸素に対する親和力が小さく,酸素を活性化する能力が低い.結局酸素に対する親和力が中庸である白金やパラジウムが完全酸化用触媒として用いられるのである.c. 吸着(adsorption)固体触媒作用は表面で起こる.つまり表面に存在する化学種だけが反応に関与するのが普通である.一方われわれが測定するもの,そして生成物として得るものは気相にある.したがって,表面相の濃度を気相の濃度と関係付ける必要がある.それが吸着等温式と呼ばれるものである.よく知られたものにラングミュアの吸着式がある.=1+(1・29)ここでは吸着量(mol・g),は表面活性点の量(mol・g),は濃度(mol・m),は吸着平衡定数(m・mol)である.また右辺のを除いた部分は活性点が吸着物質によって占有されている割合θである.多成分1~が共存するときその中のある成分の吸着量は=욡욡1+∑욢욢(1・30)で表される.なお,式(1・30)においてすべての成分は同じ活性点に競争的に吸着するものとしている.2種類の活性点がある場合,吸着に際して解離する場合などいろいろなケースについて吸着等温式が与えられている.詳しくは成書を参照していただきたい.d. ラングミュア型反応速度式の濃度依存性は1次となる.それらの中間では反応次数を求めれば0と1の間の中間的な値となる.つまり,反応次数は吸着の強さをも反映している.. 例題1自動車排気ガス浄化(exhaust gas 速度式に基づき考察せよ.解:式(1・32)をもとに考える.式(1・32)がNOに1次,COに-1次と近似できるのは≫,1となっている場合である.つまり,COの吸着が非常に強くそのため競争するNOがほとんど吸着できない状況である.ラングミュアの吸着式に用いられている仮定は実際の触媒表面を簡略化しすぎている嫌いがある.それにもかかわらず,それより導出された反応速度式は触媒反応において広く見られる特徴をうまく説明しており,特に機構解明のための反応速度論的研究には広く使われている.e. 活性点(active site)触媒の表面がすべて等しく活性であるわけではないということは古くから知られてきた.表面上にある特定の構造だけが活性を担っているというほうが普通なのである.その活性を担う構造のことを活性点という.ある触媒において何が活性点であるかということはいまだに触媒研究者にとって中心的な課題である.例えば,40年前までガソリン製造に使われてきたSiO-AlO触媒の活性点はSiO格子中に同型置換したAl原子が電荷のバランスをとるためにその近傍に保持しているH(プロトン)であることがわかってきた.その数は表面に露出している全原子数の1/100程度にすぎない.また例えばPt-AlO触媒(いろいろな水素化反応に使われる触媒)において活性を発現する主体は白金であり,アルミナはそれを安定に保持する役割を果たしているにすぎない.また露出している白金のすべてが活性点であるとは限らない.表面に露出している特定の結晶面あるいは結晶の稜などの欠陥構造だけが高い活性を示すことなども知られている.活性点の構造がわかってくるとその特定の構造を表面上に作ることにより高活性の触媒を調製することができる[図1・4(a)].例えばSiO-AlOに替わって広く使われているゼオライト(zeolite)は,アルミノケイ酸塩である[図1・4(b)(c)].その対となる陽イオンはNaであるがそれをプロトンとイオン交換すると酸性が発現し,固体酸触媒となる.これは無定形SiO-AlO触(a)(b)(c)図1・4(a)シリカアルミナの活性点構造と(b)ゼオライトZSM-5の結晶構造および(c)その細孔構造(b)において,線分の交点にSiあるいはAl原子が位置し,その間酸素が架橋している.すなわち,一つの線分はSi-O-SiあるいはSi-O-Al結合を意味する.図中に見える二つの10員環はZSM-5特有の結晶構造に基づく細孔である.(c)はその細孔の連結様式を示した図である.媒の活性点構造の解明からその活性点構造を基本とする化合物合成に進んだものとみることもできる.
