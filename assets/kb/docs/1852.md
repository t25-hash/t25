# 1・2・2 触媒の探索・改良への指針

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

触媒作用の研究は結局のところよりよい触媒開発のためのものである.したがって目的とする反応の機構および活性点構造についてできるだけ詳しく研究し,その反応を促進する最も適切な活性点構造を表面に作ることである.また,触媒物質の選択に当たっては前述のサバチエの原理を理解しておくことは非常に役に立つ.ただ,これはある意味では建前である.非常に複雑な触媒反応を設計するときにそのような正面攻撃だけでアプローチするわけでもない.触媒開発者の通常の探索法を紹介しておく.(1) 類似の触媒反応系に学ぶいまブタンの部分酸化によってマレイン酸を作る触媒を開発しようとしているのであれば,プロパンの酸化反応の情報を集め,その反応に使われている触媒を使ってみるのはよい出発点である.化学的な類似性が大きいほど触媒も似ているであろう.(2) 周期律表の利用もし白金を触媒として用いているのであれば周期律表で同じ族に属しているパラジウムを使ってみる,など.(3) 副反応を抑える現状の触媒の問題点が選択性が低いことであれば,その副反応を抑える工夫をする.その副反応自体が触媒反応であるから,その副反応を抑える化学的,工学的工夫をする.副反応が表面の酸点によって促進されるとき,塩基性物質をあらかじめ加えた触媒を作るなどはよく行われる手法である.(4) 触媒の物理構造の改良触媒と反応物との接触をよくすることは触媒反応工学の最も重要な課題である.細孔構造の制御,触媒の大きさ,形状などを最適化する.
