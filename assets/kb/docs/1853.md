# 1・2・3 活性劣化(deactivation)

> **章**: 第1章 化学反応の基礎
> **節**: 1・2 触媒反応

反応中に触媒活性点が変質することにより、活性が劣化することは触媒反応装置にとって最も悩ましい問題である。担体や助触媒の使用、移動層・流動層などの反応器、モノリス型触媒(mono-lithic catalyst)、ガードリアクタの利用など触媒反応プロセス工学の工夫は触媒活性劣化との戦いの中から生まれてきた。その原因を表1・2に示す。この対処法は触媒それ自身の改良、触媒の再生を組込む反応操作法の工夫、劣化をもたらす物質を極小化するようなプロセスシステムとしての工夫、など原因に応じた多種多様な方法がある。詳しくは成書を参考にしてほしい。8―111化学の基礎表1・2 活性劣化の原因1 反応原料から触媒劣化要因がもち込まれる場合a.吸着阻害(inhibition)b.触媒被毒(poisoning)c. 細孔閉塞(pore blocking)a.吸着阻害(inhibition)b.炭素析出(carbon deposition,coking)2 触媒自身の変質による触媒劣化a.シンタリング(sintering)b.相転移(phase transition)c. 相分離(phase segregation)d.固相反応(solid reaction)a.反応流体との反応:還元、酸化、硫化、カーバイド化などb.触媒成分の揮散損失3 触媒成形体の変化による触媒劣化a.機械的な振動による成形体の破壊・一部のはく離b.熱衝撃、熱履歴などによる破壊・一部のはく離c. 固体同伴物、粘着性物質による粒子空げきの閉塞
