# 1・2・4 複合触媒とその機能

> **章**: 第1章 化学反応の基礎
> **節**: 2・2 触媒成分の組成変化を伴う場合

触媒は多くの元素からなる多成分系のものがほとんどである。それらの成分がどのような役割を果たしているかについていまだに不明のことが多い。1担体(support):高価な貴金属を有効に利用するためにはそれを小さな粒子として、それを担体と呼ばれる大きな表面積をもつ安定な物質のうえに担持することがよく行われる。2活性をもつ複合多成分構造を作る:前述のSiO-AlOやゼオライトはそのよい例である。Al元素が酸点を形成するのに必須であるとしてもAlOを用いたのでは狙いとする構造はできない。SiOマトリックス中に固溶させなければならない。3触媒の安定性の向上、シンタリング防止:厳しい条件下で使われる触媒は使用中に触媒粒子が成長し、表面積が低下する。このような変化を抑制するために助触媒成分が添加される。4副反応の防止:触媒反応の選択性を向上させるには競争する副反応を抑制せねばならない。例えばその副反応が酸点によって触媒される場合、その酸機能を抑制するためにアルカリ金属を加えるなどはよく用いられる技術である。自動車触媒を例として、構成成分とそれらの役割を説明する。構造体:熱膨張率が低く耐熱衝撃性が高いコージェライトと呼ばれるセラミックスが用いられる。触媒一体型構造体の骨格を形成する。この骨格は2ミリ程度の穴のあいたハニカム型構造体でその穴の中を排ガスが流れる。担体:コージェライトのモノリス構造体の表面にアルミナがコーティングされる。これは触媒担体としての作用をする。活性金属:Pt,Pd,Rhが主たる触媒作用を担う物質である。担体上に高度に分散され、その粒径は10~数十nmである。Pt,PdはCOや炭化水素の酸化に優れ、RhはNO還元機能が高い。助触媒:CeOは酸素が過剰に存在するときはそれを吸収し、足りないときはそれを放出することにより、反応条件の周期的変動を平滑化する作用をもつ。安定剤:NbOは担体上に存在して主触媒である貴金属の移動を妨げシンターリングを防止する。いろいろな成分がそれぞれの役割を担って総合的に触媒の機能を高めていることがわかるだろう。
