# 1・2・5 表面積、細孔構造(pore structure)

> **章**: 第1章 化学反応の基礎
> **節**: 2・2 触媒成分の組成変化を伴う場合

触媒作用は表面現象である。したがって第一義的に活性は表面積に比例する。表面積とは基本的には一次微粒子の外表面積であるので、その大きさは構成微粒子の径に強く依存する。ちなみに通常の酸化物ではρ=3×10gm程度であるので、200m・gの表面積をもつ酸化物触媒の一次微粒子径は10nm(100オングストローム)程度ということになる。このような一次微粒子が集合してわれわれの見ている粒子を形成しており、その一次微粒子間には空げきが存在する。それが細孔といわれるものである。触媒調製時に可燃性物質を加えて成形し、その後焼成するとその可燃性物質が除去され、細孔として残る。このような手法は細孔径制御法として広く使われるようになっている。ゼオライトの細孔は特殊な結晶構造に固有のものである[図1・4(b)(c)].したがってゼオライトの結晶によりその細孔構造は一意的に決定されている。その細孔の大きさは分子の大きさと同程度であり、触媒機能に大きな影響を与える。例えばトルエンのメチル化によるキシレン合成のとき、適切な細孔径をもつZSM-5触媒を用いると、分子径の大きなオルトキシレンおよびメタキシレンは生成せず、パラキシレンのみが選択的に生成する。これを(分子)形状選択性という。
