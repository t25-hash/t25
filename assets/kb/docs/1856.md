# 1・2・6 細孔構造と活性・選択性

> **章**: 第1章 化学反応の基礎
> **節**: 2・2 触媒成分の組成変化を伴う場合

細孔の中の流体は反応器内の流動状態とは無関係に静止しており、物質は拡散によって移動する。細孔構造が触媒の機能にどのように影響するかは、触媒反応工学の重要な研究課題である。触媒粒子内で反応物は消費され、外部から拡散により補給される。反応速度が大きいとき、物質移動速度が遅いとき、あるいは粒子が大きいとき、触媒粒子内の原料物質濃度は低くなり見掛けの反応速度も低下する。すなわち触媒の有効度[触媒有効係数(effectiveness factor)といい、ηで表す]は低下することとなる。ηは式(1・33)で定義されるチーレ数(Thiele number)の関数として、式(1・34)のように表される。球形触媒粒子に対するチーレ数:=3(1・33)触媒有効係数:η=11tanh3-13(1・34)ここで、は触媒粒子半径(m)、は反応速度定数(s)、は有効拡散係数(m・s)である。またチーレ数の中に現れる3はいろいろな形状の触媒粒子との比較を容易にするために導入された球に対する形状係数である。ηととの関係を図1・5に示す。反応を早くするためには、粒子径を小さくする、有効拡散係数を増すために細孔径を大きくする、触媒有効成分を担体の外表層に担持するなどの対策がとられるが、これらはすべて触媒有効係数を大きくしようとの試みである。触媒反応の選択性とはいくつかの反応の相対的な反応速度によって決まる。したがって、選択性もまたチーレ数によって大きく影響される。詳しくは成書を参照してほしい。
