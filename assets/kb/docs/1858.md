# 1・3・1 重合反応の種類

> **章**: 第1章 化学反応の基礎
> **節**: 1・3 重合反応

モノマを重合し、ポリマを生成する反応形式は以下のようである。a. 付加重合(addition polymerization)モノマ分子中に炭素-炭素二重結合(-C=C-)をもつ化合物(オレフィン、ビニル化合物)や炭素-酸素二重結合(C=O)をもつ化合物(アルデヒド、ケトン化合物)に、開始剤、重合触媒、熱、光、放射線、などが作用して重合が開始される。付加重合には、ラジカル重合(radicalpolymerization)とイオン付加重合(ion polymerization)がある。ラジカル重合は、モノマ内の二重結合が開いて分子末端に不対電子をもったラジカルが形成され、連鎖反応(chain reaction)として進行する。最も多い反応は、CH=CHX で表されるビニルモノマの重合反応、CH=CHX →-CH-CHX-CH-CHX-CH-CHX-、である。ポリスチレン、ポリメタクリル酸メチル、ポリ塩化ビニル、高圧低密度ポリエチレン、アクリル-スチレン共重合体、などがこの重合により生成されている。イオン重合は、共有結合の電子対が完全に分離しないで炭素原子の一方に片寄ってイオンの状態になった活性種により重合連鎖が運ばれて進行する。イオンの正、負によってカチオン重合、アニオン重合に分類される。アニオン重合は、電子受容性モノマ(例えば、メタクリル酸メチル、スチレン)で起こりやすく、各種塩基により開始される。カチオン重合は、電子供与性モノマ(例えば、イソブテンビニルエーテル)で起こりやすく、各種酸で開始される。b. 開環重合(ring-opening polymerization)環状モノマ(例えば、ポリエチレンオキシド、ε-カプロラクタム)の一重結合が開環しながら重合を繰返す反応で、重合生成物は、重縮合反応で生成されるポリマと同様な構造をもつ。例えば、環構成原子として、酸素や窒素などのヘテロ原子を含むとき(例えば、環状エーテル)、化合物は次のように開環重合する。(CH)n→-[(CH)-X]윰-,X=-O-,-NH-,-S-X重合反応は付加的であり、速度論的には付加重合と同じように取扱うことができる。ポリイミン、ポリエステル、ポリアミド、ポリウレタン、ポリ尿素などが環状モノマを原料として生成されている。c. 重縮合(polycondensation polymerization)カルボキシル基と水酸基あるいはアミノ基のように、お互いに反応することができる官能基を、単一化合物に2個もつような場合に、これらの官能基が段階的な繰返し反応をすると同時に、水、塩化水素などのような低分子物質を脱離して共有結合を形成する。代表的な官能基の組合せは次のようである。-NH+-Cl-COOH+-OH-COOH+-NH→-NH-+HCI-COO-+HO-CONH-+HOイミン結合→エステル結合→アミド結合図1・7 フェノールの重合反応8―113化学の基礎この反応の特徴は、生成ポリマの両末端につねに官能基が存在し、段階的に成長し続けることである(段階反応、stepwise reac-tion)。ポリエチレンテレフタレート、ポリヘキサメチレンジアパミド、ポリカーボネートなどがこの重合により生成されている。d. 重付加(polyaddition polymerization)重縮合反応のように、低分子が離脱しながら連結していく反応と違って、単量体の官能基どうしが過不足なく原子を結合し合って連結する付加重合による逐次反応である。具体的には、ジイソシアナートとジオールとの反応によるポリウレタンの生成がある。OCN-R-NCO+HO-R′-OH →O쏂C-O-R′-O)욈-O쏂-(C-NH-R-NH-e. 付加縮合(addition condensation polymerization)付加反応と縮合反応との繰返しが基本的反応で、逐次的に進行する。O쏂R-H+H-C-CR-CH-OH+R-H付加反応→→R-CH-OHR-CH-R+HO熱硬化性樹脂であるメラミン樹脂、フェノール樹脂、尿素樹脂など縮合反応がこの反応により生成されている。ホルムアルデヒドと反応するフェノールなどは、図1・7に示すように、フェノールのベンゼン環の炭素へホルムアルデヒドが付加してメチロール化フェノールとなり、それがさらにフェノールと縮合を起こしてベンゼン環がメチレン基によって橋架けされた形となって連結する。このような反応が、付加、縮合を繰返して高分子化される。付加反応が酸や塩基で触媒されるが、縮合反応は酸触媒される。
