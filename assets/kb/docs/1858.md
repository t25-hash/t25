# 17・7・2 滑り軸受（sliding bearing)

> **章**: 第17章 回転機械の力学
> **節**: 17・7 軸受の動特性

滑り軸受は，油や気体などの流体により形成される潤滑膜に生じ
る圧力により回転体を支持するもので，加圧した流体を外部より供
給することにより潤滑膜に生じる圧力を利用する静圧軸受
（hydrostatic bearing,externally pressurized bearing）と軸と軸
受の相対滑り運動により潤滑膜に生じる圧力を利用する動圧軸受
（hydrodynamic bearing,self -acting bearing）に大別される．ま
た，滑り軸受は半径方向荷重を支持するジャーナル軸受と，軸方向
荷重を支持するスラスト軸受に大別される．
油などの液体で潤滑される滑り軸受は，転がり軸受に比べ減衰能
が高く，さらに液体膜の厚さが薄い場合にも吸着膜が形成され摩擦
を低減させる境界潤滑作用があるといった特徴を有するため多くの
回転機械で用いられている．ここでは，回転機械の力学に直接関連
する動圧ジャーナル軸受の動特性について記述する．
滑り軸受では後に述べるよう高い性能を得るために多くの軸受が
考案されている．よく使われる滑り軸受の構造を図17･24に示す．
なお，滑り軸受の種類・構造・静特性などについては，β4編「機
