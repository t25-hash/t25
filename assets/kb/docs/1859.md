# 1・3・2 重合反応様式

> **章**: 第1章 化学反応の基礎
> **節**: 1・3 重合反応

重合反応の様式は、モノマ、ポリマ、重合開始剤、溶媒などの物理的性質に強く支配されているが、反応装置形式、反応系、反応流体の流れなどを基礎とする重合反応工学的には、塊状、懸濁、乳化、溶液重合に分類できる。a. 塊状重合(bulk polymerization)重合系の相形態により、液相塊状重合法と気相重合法がある。. 液相重合法モノマ液相そのままか、あるいは重合開始剤を溶解したモノマ液相で重合を行う方法であるが、重合の進行に伴う反応混合物の粘度増加を防ぐために、適当な溶媒が添加される場合がある。単純な反応機構や、他の重合法と比べて簡単な重合操作に加えて、重合度の高い純粋なポリマが生成される。しかしながら反面、重合反応混合物の粘度の増加に伴う重合速度の増加や重合熱制御の困難さのために、局所的な温度上昇や分子量分布の広がりが生じることがある。ポリマの品質の低下を招くこのような現象を防ぐために、重合温度を低く保ったり、ラジカル発生速度の遅い重合開始剤を使用したりして、反応系全域にわたり均一な速度で進行するよう工夫がなされている。ポリスチレン、ポリメタクリル酸メチルなどは、ポリマがモノマに溶解するので、均一相系で重合が進行するが、一方、ポリプロピレン、ポリアクリルニトリル、ポリ塩化ビニルなどは、ポリマがモノマに溶解しないで析出する不均一系で重合が進行して粉末状のポリマが生成される。. 気相重合法重合装置として、気流中に触媒粒子を分散8―114電気・電子の基礎、化学の基礎させた気・固系流動反応装置が利用されている。添加されたモノマは、触媒粒子表面でラジカル発生、成長、ポリマ生成というプロセスで重合する。ポリマ物性を均一にするためには、触媒粒子の滞留時間を均一にすること、粒子間凝集を防止すること、安定な流動状態を維持することなど、装置形状や流動状態の最適化が重要となっている。具体的な例として、線状低密度ポリエチレンがこのような重合系で生成されている。この系で使用されている触媒として、MgClからなる金属化合物にTi金属を配位した担持型触媒粒子が使用されている。b. 懸濁重合(suspension polymerization)重合開始剤を溶解したモノマを、このモノマと相溶性のない液体中に分散させた微小滴モノマ内で重合を進行させるものである。すなわち、微小滴モノマは、微小回分反応器としての機能を果たすことになる。スチレンや塩化ビニルのように、モノマが疎水性である場合には水相が連続相となり、アクリル酸ソーダやエチルヘキシルアクリレートのように、モノマが水性である場合には疎水性液体が連続相となる(逆相懸濁重合)。また、ポリスチレンやポリメタクリル酸メチルのように、ポリマがモノマに溶解する場合には透明なポリマ粒子が生成され(pearl polymerization)、ポリ塩化ビニルのように、ポリマがモノマに溶解しない場合には微粉末状のポリマが生成される(powder polymerization)。この重合法は、いかにして分散系を終始安定に維持するかが重要なこととなっている。このために、連続相に水溶性高分子(ポリビニルアルコール、メチルセルロース、ゼラチン)、界面活性剤(ドデシルベンゼンスルホン酸ソーダ、ドデシル硫酸ナトリウム、アルキルベンゼンスルホン酸)、固体微粉末(リン酸三カルシウム、マグネタイト、炭化けい素)、などを添加している。最終的なポリマ粒子径分布や平均粒径は、反応装置形状、かくはん翼形状、かくはん動力や懸濁安定剤種・濃度などに強く依存している。モノマ相に粉末状固体、不溶性液体、気体などを包含させることにより、すなわち、芯物質として存在させることにより、各種の内部分散型複合体粒子(多核カプセル)を調製することができる。例えば、カーボンブラック、マグネタイト、染料・顔料を包含させることにより、乾式電子式複写機現像剤が、硬化剤水溶液・反応触媒、殺虫剤水溶液などを包含させることにより、一液型接着剤や徐放性薬剤が、気体を包含させることにより、マイクロバルーンがそれぞれ調製できる。また、懸濁重合系で出現する液・液界面に付着するような濡れ性を呈する固体粉末種を添加することにより、懸濁安定剤としての効果を発現しつつ、重合終了後にはこの固体微粉末で被覆されたポリマビーズを生成することができる。この際、固体微粉末の液・液界面に対する接触角を種々変化させることにより、複合体微粒子の大きさおよび固体微粉末の浸漬深さをコントロールすることができる。したがって、磁性、熱伝導性、電気伝導性などを呈する固体粉末種の物理化学的特性を付与した被覆型複合体粒子が調製できる。これまでに、マグネタイト、炭化けい素(微粉末、ウィスカ)、二酸化チタンなどで被覆された複合体微粒子が調製されている。このように、懸濁重合法は、各種複合体微粒子やマイクロカプセルを調製するための簡便な方法として大いに期待される。c. 乳化重合(emulsion polymerization)重合開始剤が溶解している水溶液中に、脂肪酸セッケンのような界面活性剤を臨界ミセル生成濃度以上に添加すると、界面活性剤分子の集合体(ミセル)が形成される。このような系にモノマを添加すると、モノマの一部は水溶液中にモノマ滴となって分散し、残りのモノマはミセル中に溶解(モノマの可溶化)するようになる。水溶液中に発生したラジカルは、モノマ液滴より圧倒的に大きな表面積をもつミセル内部へと侵入して重合が開始される。ミセル内での重合の進行とともに、モノマは消費されるが、その後は、モノマ液滴から連続水相中における拡散によりミセルへモノマが供給し続けられるためにやがてモノマ液滴は消滅する。一方、個々のミセル内で生成されたポリマミクロスフェアが重合の進行とともに成長することにより表面積が増加するために、ミセルを維持するに要するだけの界面活性剤濃度が低下することから、やがてミセルは消滅する。一方、ポリマミクロスフェアは成長し続けることになる。重合場がミセル内であることから、速い速度で重合が進行し、かつ、高い分子量のポリマが生成される。しかしながら、ミセル形成が不可欠であるために添加される高濃度の界面活性剤が最終ポリマ製品に対して不純物となり、電気的特性や熱的特性に影響を与えることがある。工業的には、塩化ビニル、スチレン、酢酸ビニルなどが生成されている。乳化重合とは異なり、乳化剤を全く添加しない重合形態として、ソープフリー重合がある。この場合には、モノマ滴から連続水相へ溶解したモノマが、水溶性開始剤のラジカルにより活性化してラジカルモノマとなり、ここに溶解しているモノマが付加していく。そして、連続水相中に溶解しうる臨界鎖長まで成長すると析出して核を形成すると同時に、核どうしの凝集により微粒子として成長し、単分散性のサブミクロンから数ミクロンオーダのミクロスフェアが生成される。成長した微粒子の凝集が頻繁に起こると、単分散性が劣るようになる。このために、開始剤の分裂によって発生するイオン性解離基、例えば過硫酸カリウムの硫酸イオン(SO)による静電気的安定性効果により、凝集を抑制する方法がとられている。また、連続水相中で成長するダイマ、オリゴマ、核などを固体表面に析出させることで、固体表面のポリマによる被覆化や、固体微粒子のポリマによるカプセル化を行うことができる。このようなプロセスが円滑に進行するように、固体微粉末表面に界面活性剤の吸着層を形成させておき、この疎水基部分にモノマ、ダイマ、オリゴマなどを吸収させて反応場として重合を進行させる方法が採用されている。このようなカプセル化法により、顔料、染料、無機固体微粉末などのカプセル化が実施されている。d. 溶液重合(solution polymerization)モノマを適当な溶媒に重合開始剤とともに溶解させて重合反応を行う。このような反応形態を採用することにより、塊状重合で遭遇するような問題点(反応熱の処理、流体混合、ポリマの付着)を克服することが可能となっている。ポリマが溶媒に溶解する場合には、ポリマ溶液として生成されるので均一重合系となる。また、ポリマが溶媒に溶解しない場合にはポリマが析出するので不均一重合系となる。溶媒が使われるために、ポリマの分子量は他の重合系よりも低く、かつ重合速度も遅くなる。したがって、重合操作条件により重合速度を制御することが容易であることから、均一なポリマを生成することが可能である。工業的には、ポリオレフィンやスチレンブタジエンゴム、ブタジエンゴムなどの合成ゴムの製造プロセスに利用されている。
