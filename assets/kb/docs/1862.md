# 1・3・5 重合反応操作形式

> **章**: 第1章 化学反応の基礎
> **節**: 1・3 重合反応

ポリマ製造プロセスの操作形式は、製造するポリマにより異なるが、連続操作(flow operation)、半回分操作(semi-batch opera-tion)、回分操作(batch operation)に大別される。a. 連続操作反応原料(モノマ、触媒、反応媒体、各種添加剤)を一定の割合で供給し続け、全体的なバランスをとりながら反応生成物を取出す。操作の定常状態が成立すると、ポリマの組成や重合率は一定となる。したがって、大量生産に適した操作である。ポリスチレン、ポリエチレン、スチレンブタジエンゴムなどの汎用ポリマは、ほとんど連続操作により製造されている。b. 半回分法(半連続法)所定量のモノマ、溶媒、反応触媒を反応器に仕込み重合を開始しようとする場合に、反応速度の急激な増加に伴う多量の反応熱の発生により、反応系の温度制御が困難になったり、副反応が発生したりすることがある。また、反応混合物の粘度の増加により、反応熱の除去や分散系の安定性が維持できなくなり、結果として、安定操作の継続が不可能になったりすることがある。このような現象の発生を防ぐために、反応開始後の適切な時点に、モノマ、反応触媒、溶媒の適量を断続的に添加する方法が半回分法である。添加する時期や添加量により、ポリマの物性を多様に変化させることが可能であることから、操作条件・反応速度・物性の間の定量的関係を十分把握しておくことが必要である。図1・9反応物濃度の変化ポリエチレン、ポリ酢酸ビニル、乳化重合アクリロニトリル・ブタジエン・スチレン共重合体などの製造に採用されている。c. 回分操作法所定量の反応原料(モノマ、触媒、反応媒体、各種添加剤)を重合開始前に反応器に添加・混合し、必要な前操作(昇温、窒素置換、バブリングなど)を実施してから重合を開始する。反応混合物の組成は、重合時間とともに連続的に変化する。この操作法は、多品種少量生産に適する。スチレンや塩化ビニルの懸濁重合操作はこの方式で実施されている。記号욉:反応物濃度mol/m:反応流体流量速度m/s:開始剤効率욃:開始反応速度定数욞:連鎖移動速度定数욡:熱・触媒開始反応速度定数욣:生長反応速度定数욧:停止反応速度定数욡:開始反応速度욞:連鎖移動反応速度:重合反応速度욣:生長反応速度:反応速度욧:停止反応速度:反応時間s욥욇:かくはん槽型反応器体積m욥욣:押出し流れ反応器体積m:反応転化率θ:平均滞留時間min各手法に共通な基本概念を述べるとともに、合成法、装置工学的な具体例も一部挙げ、概観する。
