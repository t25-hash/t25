# 1・4・1 合成法の分類

> **章**: 第1章 化学反応の基礎
> **節**: 1・4 節では無機系薄膜・微粒子の合成反応・プロセスの分類や、

無機系薄膜や微粒子の合成法は、表1・3に示した気相、液相、固相の3種類に分類される。気相からの薄膜や微粒子など固体の合成は、大きく分けると蒸発・凝縮を基礎とした物理的気相析出(PVD : physical vapor deposition)法と化合物気体の反応を用いた核発生・成長による化学的気相析出(CVD : chemical vapordeposition)法に分けられる。液相での粒子や薄膜の合成法は、水溶液法、水熱法、金属アルコキシドを用いたゾルゲル法などが代表的な手法である。これらの手法は、高純度の製品が得られ、分子、イオンレベルから核生成・成長させてナノレベルの大きさや厚さの微粒子や薄膜を合成するのに適しているが、一般に大きなサイズのものを作るにはコストが高い。固相法は、熱分解や還元反応など高温下での反応や焼結現象を利用する方法で、一般にバルク体を作るのに適しているが、微粒子化するには粉砕等の操作が必要で、純度が犠牲になるが、コスト的には安く製造が可能である。
