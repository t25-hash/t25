# 1・4・3 気相合成反応

> **章**: 第1章 化学反応の基礎
> **節**: 1・4 節では無機系薄膜・微粒子の合成反応・プロセスの分類や、

気相合成法では、一般に高いエネルギーを原料に与え、気化させたうえで、析出、反応を起こさせる。薄膜合成では、前述の基板と析出粒子の濡れ性に加え、高真空中で蒸気等の平均自由行程を長くして壁面や基板に衝突させ不均一核生成を促進させる。一方、粒子合成は10~10Torr程度の比較的低真空で、生成蒸気を気体分子と繰返し衝突させ気相中で均一核生成させる。したがって、気相合成法は、原料となる気体蒸気、イオン等の生成法と真空雰囲気条件によりさまざまな方法に分類される。以下に、各方法について概観する。a. PVD 法物理的気相析出(physical vapor deposition)法は、高いエネルギーをもつ原子、分子を高真空中で基板上に照射し、基板の物質を蒸発し析出、成長により微粒子や特に薄膜を作製するのに適した方法である。おもな方法を使用する雰囲気圧力を目安に表1・4に示した。. 真空蒸着(vacuum coating)法真空中で金属や化合物を加熱蒸発し基板に薄膜を析出させる方法である。加熱方法とし分子線エピタキシー10加熱蒸発で生成した分子線を基板に析出真空蒸着10蒸発蒸気を蒸着イオンプレーティング10グロー放電下で蒸気をイオン化して蒸着スパッタ10プラズマ状態で蒸着8―118電気・電子の基礎、化学の基礎ては抵抗加熱、誘導加熱、電子線加熱などがあるが、蒸発させる物質の融点が高い場合ほど高エネルギー密度で加熱可能な電子線加熱が用いられる。加熱蒸発した原子、分子が基板上に拡散・移動する過程で、真空度が低いと残留した酸素、水、COなどと酸化、炭化、水素化等の化学反応を起こすため超高真空(10~10Torr程度)にする必要がある。. スパッタ(sputtering)法気相中に、陽イオンと電子が共存し電気的中性を保ったプラズマ状態をつくり、陰極にしたターゲット材料に陽イオンを衝突させて気化させ、基板上に析出させて製膜する方法である。プラズマ状態は10Torr程度の真空度で数百V 以上の電圧を印加し、グロー放電することで得られる。電圧の印加法により、直流電界を用いる直流スパッタ、高周波電源を用いるRF(ラジオ周波数)スパッタ、永久磁石をターゲット材料に組込んだマグネトロンスパッタがある。. イオンプレーティング(ion plating)法蒸発源を陽極、基板側を陰極にして数千V の直流電源、あるいはRFをかけグロー放電させたプラズマ内に金属蒸気を通過させイオン化したうえで基板に析出させる方法である。イオン化効率の高いRFイオンブレーティングでは、10~10Torr程度の低い真空度でプラズマの生成が可能で、蒸気の平均自由行程が長く、プラズマ発生部と蒸気部が完全に分離できるので反応や蒸着速度を別々に制御することができる。. 分子線エピタキシー(MBE: molecular beam epitaxy)法超高真空(10~10Torr)中で、物質を加熱、蒸発して生成した分子線を単結晶基板に照射して、その基板上に単結晶を成長させる方法である。b. CVD(chemical vapor deposition)法生成した蒸気等の化学反応により薄膜や微粒子を生成するCVD 法では、純物質の蒸発・気化と析出が必要なPVD 法に比べると、化学反応で目的物質を生成させることができるため、原料には低温で気化しやすい、あるいは常温で気体である化合物を原料に選ぶことが多い。無機化合物原料としては、金属のハロゲン化物、オキシ塩化物、水素化物、有機金属化合物としてM(OR)욈の一般式で表される金属アルコキシド(M:金属元素、R:アルキル基、:金属の価数)等が用いられる。これらの比較的低沸点の化合物蒸気を各種反応性気体と気相中あるいは基板上で反応させて、核生成・成長することで微粒子や薄膜を形成する。反応性気体は、最終目的物質が、酸化物の場合には、O,HO,NOなどが、窒化物ではN,NHなど、炭化物にはCH,CHなどが用いられる。気相からCVD 法により微粒子を合成するには気相中で蒸気を衝突させて核生成を起こさせるため、気体圧は、10~1Torr以上である。工業的に大量生産されているのは、金属塩化物の酸化反応で生成されるSiOやTiOの超微粒子(粒子径10nm程度)が代表例である。この超微粒子合成プロセスの原料は塩化物で、例えばSiOは以下の反応で生成する。SiCl+2HO=SiO+4HCl(1・73)この反応は、酸水素炎(~2000°C)に原料を供給して進行させる。窒化物の合成プロセスは、金属塩化物とNHを1000~1200°C程度の温度でSiN,AlN,TiN などが合成されている。特に、Si(NH)を1000°Cで熱分解して、1400°Cでの熱処理により結晶化させてα-SiN微粒子を得るイミド分解法は最も普及している製法の一つである。一方、薄膜のCVD 法による合成では、気相中で均一核生成を起表1・5 液相合成法の分類と特徴分類合成反応機構おもな方法代表例水溶液法析出沈殿反応加水分解除冷法、蒸発法、凍結乾燥法炭酸塩、シュウ酸塩有機金属加水分解法水熱法水熱合成法、水熱育成法SiO単結晶こさないで基板上で不均一核生成が起こるように、気相反応の飽和度を低く抑制するか、気体の全圧を10Torr程度以下に抑える必要がある。
