# 1・4・4 液相合成反応

> **章**: 第1章 化学反応の基礎
> **節**: 1・4 節では無機系薄膜・微粒子の合成反応・プロセスの分類や、

液相合成も、気相法と同様にさまざまな方法があり、大きく分類して水溶液法、水熱法、融液法の3種類程度に分類される。水溶液法、水熱法での代表的な合成反応機構と工業的手法、代表的な製品例を表1・5にまとめて示した。液相法による粒子合成は、気相法に比べて原料分子の拡散速度が遅く、反応速度も比較的容易に制御できるため、粒度分布や形状、組成等の均一化が比較的容易である。以下に各方法について概観する。a. 水溶液法水溶液法は分類すると、高溶解度の無機塩水溶液の溶解度を下げたり水を取り除くことで溶解度以上の過飽和状態にして粒子や単結晶を析出させる析出法、溶解度の低い炭酸塩やシュウ酸塩を生成させて目的成分の沈殿物を得る沈殿法、金属アルコキシドなどを加水分解・縮重合反応させて酸化物を生成する加水分解法が挙げられる。これらの手法は酸化物を中心とした高純度の粒子や薄膜、単結晶の生成に用いられている。析出法の原理の一つである高溶解度の無機塩飽和水溶液の溶解度を下げる方法で最も容易な方法は、温度を下げる除冷法で、岩塩など不純物を含む原料から高純度のNaCl単結晶を得るプロセスが代表例である。また、溶媒である水を除去して過飽和状態とする析出法では溶液を急速凍結・昇華させる凍結乾燥法がある。沈殿法では、1種類の金属イオンはもちろん、複数の金属イオンがほぼ同条件で沈殿物を生成させることで複数の金属塩を均一に混合した析出物が得られる点に特徴がある。例えばBaTiOを例にすると、最初、BaとTiの塩化物など高い溶解度の等モル水溶液を作っておく。この水溶液にシュウ酸を加え、BaとTiのシュウ酸塩を生成させると、この塩は溶解度が極めて低いため、BaとTiの等モル塩が析出・沈殿物を作る。この沈殿物を加熱・分解することで、目的のBaTiO前駆体を得ることができる。加水分解法の代表例として、有機金属の加水分解・縮重合法がある。代表例にはシリコンのアルコキシド[例えば、TEOS : tetra-ethylorthosilicate, Si(OCH)]の加水分解・縮重合反応を制御することで球形単分散シリカ粒子を合成するストーバー法がある。この方法では、アンモニアなどアルカリを触媒とし、高pH になるほど核生成濃度が抑制されたもとで縮重合反応が進むため粒子径が成長する。核生成は、TEOS の加水分解により生じたOH 基どうしが縮重合反応することによって、モノマが次々と重合し臨界径以上になったときに起こる。均一な粒子を生成するには、TEOS と水、および溶媒であるエタノールのモル比の設定が重要である。化学量論比はTEOS:水=1:4であるがこの比率だと加水分解反応が完了しないため、若干、化学量論比より水を過剰に加え反応させる。粒子径は、アンモニア添加濃度にほぼ比例して制御できる。一例として、TEOS と水のモル比を化学量論である[HO]/[TEOS]=4より水が過剰の10.0に固定し、アンモニア濃度を変化させた場合に得られた粒子をTEM 観察し、画像解析により求めた平均粒子径とアンモニア濃度の比の関係を図1・12に示した。両者は良好な比例関係を示しており、アンモニア濃度によりナノ領域まで粒子径が制御できる。この関係は、限られたHOとTEOSのモル比、アルコール濃度範囲で成立する。また、粒子径を100nm以上に成長させると球形単分散のシリカ粒子が得られる。一例として図1・13に電子顕微鏡観察結果を示した。NaCl単結晶、硫酸鉄粒子BaCO微粒子単分散SiOゾル、BaTiOb. 水熱法密閉容器中で100°C以上に加圧、加熱されている熱水は、常圧100°C以下では水に溶けない化合物を溶解し、化合物間の反応速度を増加させる。水熱法は、こうした熱水の反応性を利用する方法で、代表的な水熱合成(hydrothermal synthesis)法には、750気圧以上の耐圧耐熱反応容器(オートクレーブ)中で行う水晶α-SiOの成長がある。このほか、ゼオライトなど合成されている。
