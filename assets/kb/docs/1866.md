# 17・8・6 曲げ・ねじり連成振動（coupled vibration)

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

歯車を含む動力伝達系において，歯車を支える軸や軸受の剛性が
高くない場合は，負荷運転時に歯車は回転するだけでなく並進運動
する．このとき，歯車のかみあい線方向の接触条件から，歯車の回
転角と並進変位との間に拘束が生じるため，軸系のねじり振動と曲
げ振動（横振動）が関連し合う，いわゆる連成が生じる．連成系の
固有振動数や危険速度は連成がない場合の値と異なるので，設計時
