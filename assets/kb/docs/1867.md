# 17・8・7 ロータの接触振動

> **章**: 第17章 回転機械の力学
> **節**: 17・8 回転軸系に生じるその他の振動

角速度ωで回転しているロータは，通常，不釣合い遠心力によ
ってたわみを生じ，回転方向と同じ方向に角速度ωで円またはだ
円軌跡を描いて旋回（前向き振れまわり）する．しかし，危険速度
通過時や事故などでロータの振れまわりが大きくなりケーシングな
どに接触すると，摩擦力の作用によってロータが回転方向と逆方向
に大きく振れまわる，いわゆる後ろ向き振れまわり（backward
whirling）が発生する場合がある．このとき，振動，騒音，接触力
がともに大きくなり，回転機械にとって極めて危険である．後向き
振れまわりの発生過程は，シミュレーション結果（図17･49）から
次のように説明できる ．前向き振れまわり中は，ロータはケー
シングに接触するごとに摩擦力の作用で反射角δは入射角δよ
り小さくなり，ロータの軌跡は丸いだ円から細長いだ円へ移行す
る．反射角が負になるとロータの振れまわりが後向きになる．この
後，接触のたびにδ＞δとなり，ロータ軌跡は細長いだ円から
丸いだ円となる．反射角が90°を超えるとロータはケーシングに食
込むようになり，接触力や摩擦力が増加し回転速度が低下する．ロ
ータはケーシングの内壁を滑らずに転がり（両者の相対速度が零），
後向き振れまわりの角速度ωは， をロータ半径， を半径すき
まとしてω＝ωで表される．ケーシングを弾性支持した場合
は，その振動速度を含めてロータ・ケーシング間の相対速度が零に
なるようにωが決まる．後向き振れまわりは，摩擦係数，ケーシ
ングの質量および支持ばね定数が大きいほど，ロータ・ケーシング
間の接触剛性が小さいほど発生しやすい．
