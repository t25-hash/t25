# 2・1・1 はじめに

> **章**: 第2章 分離操作
> **節**: 2・1 分離操作の概要

分離(separation)という用語に近い用語に、回収(recov-ery)、濃縮(concentration)、除去(removal)、分画(fractiona-tion)、精製(purification)、そして単離(isolation)がある。それぞれ対応する英単語があるので区別できるのだろう。分離の目的は、有用な成分を回収あるいは濃縮すること、そして有害な成分を除去することである。有用な成分が数種類あってそれを分けるのが分画である。血液、特に血漿中のタンパク質を分画してそれぞれを薬剤にしている。また、目的成分の純度を上げることを精製と呼ぶ。原油を熱分解して全体の分子量を低下させてから、重油、ガソリン、軽油などに分けるのが石油精製である。精製を徹底的に行って不純物をなくし単一成分を取出すのが単離である。例えば、生物の分野では、脳の機能を解明するために脳の組織からホルモンが単離されている。2章「分離操作」には、蒸留(distillation)、ガス吸収(gasabsorption)、抽出(extraction)、晶析(crystallization)、吸着とクロマトグラフィー(adsorption and chromatography)、および膜分離(membrane separation)という六つの操作が説明されている。このほかにも分離操作がある。身の回りを見わたせば、電気掃除機の吸込みがわるくなったときに内蔵の紙フィルタを取出すと、フィルタにごみが引っかかり詰まっているのがわかる。ここでの掃除は集塵という固体(ごみ)と気体(空気)とを分ける分離操作である。電気洗濯機での衣類の脱水に遠心分離が利用されている。ここでの脱水は固体(衣類)と液体(水)とを分ける分離操作である。この2例のような分離操作は、分離対象である固体を変化させることなく機械的に(化学的にではなく)分けるので、機械的分離操作と呼ばれる。これら機械的分離操作の詳細は便覧を参考にしてほしい。2章で取上げた六つの分離操作は、ライバルとして競合するときもあるし、パートナとして連携するときもある。複雑な系は一つの分離操作だけでは望みの分離を達することができないからである。さらに、分離操作だけですむわけではなく、反応操作が分離操作の前後に絡むことが多い。また、分離の効率を高めるために反応を利用する分離操作がある。例えば、気体中の炭酸ガスを除くためには、水に吸わせるよりも、アルカリ液、例えば水酸化ナトリウム水溶液に吸わせるほうが有効である。これは、中和反応によって、吸収された炭酸ガスが炭酸ナトリウムに変化するので、液中の炭酸ガ(a) エタノールの製造(b) 酵素の製造図2・1 製造プロセスの例ス濃度が零になり、高速での吸収を実現できるからである。この反応を利用した分離操作は反応吸収と呼ばれる。蒸留、抽出、晶析などにも反応の名が用語の前につく操作がある。図2・1(a)および(b)に、それぞれ、発酵タンクからのエタノールの製造プロセスおよび酵素の製造プロセスを示す。複数の反応と分離操作の組合せによって製品が作られることを理解してほしい。
