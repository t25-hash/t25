# 2・1・2 分離操作の例

> **章**: 第2章 分離操作
> **節**: 2・1 分離操作の概要

a. 石油精製:蒸留とガス吸収石油化学コンビナートに、中東地域産出の原油を満載したタンカーが接岸する。その原油を原料にして、ガソリンで代表される燃料や、さまざまな化学製品の原料および製品が生産される。羽田空港に着陸する国内線の飛行機の窓から、空港の周辺にある川崎臨海地区のコンビナートのプラントを一望できる。プラント内に立ち並んだ多数の装置を使って、反応と分離操作が繰返され、製品が産み出される。例えば、原油の分子量を低下させるために、粒子状の触媒を装置内部で循環流動させた反応装置に原油を流通させ触媒反応を起こす。その後、原油の分解物中の成分が分子量によって沸点が異なることを利用して、成分を分ける蒸留という分離操作が適用されている。石油化学コンビナートには電力が不可欠なため火力発電所が隣接している。石炭や石油の燃焼反応で生じる熱を利用して水蒸気を作り、タービンを回して電気を作る。石炭や石油中には、その産地によって含有量は異なるものの、窒素や硫黄が含まれている。したがって、燃焼時には窒素酸化物や硫黄酸化物が発生する。これらの有害な酸性ガスを煙突からそのまま排出するわけにはいかない。排出基準が設定されている。そこで、アルカリ液と接触させるガス吸収という分離操作を適用し、これらのガスを除去している。b. 超純水製造:膜分離と吸着とイオン交換集積回路製造工場では、その製造工程でウェーハ表面に残る薬品や微粒子などを洗浄除去するために大量の超純水が使用されている。集積度の高い製品を製造するときに、使用する超純水の水質が高いと歩留りが高まる。言い換えると不良品の数が減る。超純水製造プロセスでは水道水や地下水を原料にして超純水を製造する。微生物や微粒子などの不溶物質、そして有機物や金属イオンなどの溶存物質を徹底的に水から取り去る必要がある。まず、微生物の増殖を抑え、さらに殺菌をするために紫外線を水に照射する。水中の不溶物質の除去には、不溶物質のサイズよりも小さな孔の開いた膜を用いる。膜の形態には平たい膜と中空糸膜(長いマカロニ状の膜)とがあるが、体積当たりの膜面積が大きい中空糸膜が多く用いられる。中空糸膜には貫通した孔がたくさん開いている。化学の基礎中空糸膜の内面から外面へ(逆に、外面から内面へでもよい)、圧力をかけて水を透過させ目的成分(ここでは不溶物質)をろ過除去する分離操作が膜分離である。孔のサイズによって膜には種類があって、除去したい物質のサイズよりずっと小さい孔の開いた膜を使うより、一回り小さい孔の開いた膜を使うほうが液の透過速度が速い。水中の溶存物質のうち、イオンを除去するには、イオン交換樹脂を詰めた筒(カラム)に水を流通させる。この分離操作がイオン交換である。除去したいイオンと反対の符合をもつ化学構造(イオン交換基と呼ぶ)を取付けた、直径0.5mm程度のビーズ状ポリマがイオン交換樹脂である。一方、溶存する有機物質は、活性炭のような内部に大きな表面積をもった吸着材に疎水性相互作用や分子間力に基づいて吸着させ除去する。この分離操作が吸着である。さらに、ウェーハ表面の洗浄に水を用いる直前に、配管類などから発生する不純物(おもに微粒子)を水中から除去するために膜分離を再度施す。こうして製造した超純水のおかげで集積度の高い製品をむだなく生産できるわけである。
