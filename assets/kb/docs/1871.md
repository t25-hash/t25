# 2・1・3 分離操作の分類

> **章**: 第2章 分離操作
> **節**: 2・1 分離操作の概要

a. 温度・圧力の変化と分離材料の利用読者が抱えている分離の課題に取組む前に、分離操作の概略を知っておくことが必要である。そこで分離操作を分類しておく。図2・2に二つの分類法をまとめた。上述の分離操作の例を思い起こしながら読んでほしい。まず、図2・2(a)の分類を説明する。目的成分を分離するときに、そのまま放っておいては何も起こらない。そこで、二つの手段がある。一つの手段は、温度や圧力を変えることである。蒸留は温度を上げていったときに混合物中の成分の沸点に差があることを利用して分離する方法である。エタノールと水の混合液をフラスコに入れて熱をかけると、初めのうちは沸点の低いエタノールの多い蒸気を取出せる。この蒸気を冷却して液化すれば、エタノールが濃縮されることになる。一方、晶析は温度を下げていったときに混合物中の成分の溶解度に差があることを利用して分離する方法である。小学校の理科実験にも登場するように、高い温度で水に砂糖を溶かしておいてから冷却すると、過飽和状態から結晶核の発生が始まりそこから結晶が成長していく。このとき不純物が結晶表面に吸着されないように、あるいは内部に閉じ込められないように工夫すれば、純度の高い結晶が得られる。もう一つの手段は、系に分離材料を加えることである。液体を加える分離操作がガス吸収と液液抽出である。ガスを吸収するために水、アルカリや酸を溶かした液などを使う。ガス中の目的成分は液への溶解度をもつので、その液がその成分で飽和するまで、成分は気相から液相へ移動を続ける。目的成分が水中に溶けているとき、水と溶け合わない油をその水へ混ぜると、水中の目的成分は油への溶解度をもつ(この場合、分配されると呼ぶ)ので、油がその成分で飽和するまで、成分は水(水相と呼ぶ)から油(油相と呼ぶ)へ移動を続ける。このとき、水相と油相との接触面積を増大させるために、系をかくはんしてどちらかの相を滴にすることが有効である。一方、固体を加える分離操作が、吸着、イオン交換、および膜分離である。このとき固体材料は、吸着材、イオン交換材料、および膜と呼ばれる。これらの材質は無機化合物あるいは有機化合物(a) 温度・圧力の変化と分離材料の利用による分類(b) 相変化による分類図2・2 分離操作の分類8―123(多くはポリマ)である。b. 相変化図2・2(b)の分類を時計回りに説明する。6種類の分離操作を相の変化から分類する。まず、気相と液相との間での分離操作には蒸留とガス吸収がある。蒸留は沸点の差を利用する方法であり、ガス吸収は溶解度の差を利用する方法である。ガス吸収という用語は、気相から液相へのガス中の目的成分が移動することを表すが、液相の成分を気相へ移動させるという逆方向の操作もある。この操作を放散と呼ぶ。次に、互いに溶け合わない液の間、すなわち液相と液相との間での分離操作には液液抽出がある。さらに、液相と固相との間での分離操作には吸着とイオン交換、晶析、そして膜分離がある。さらに、固相と気相との間でも同じく吸着とイオン交換、晶析、そして膜分離という分離操作が挙げられる。蒸気から、液体を経ることなく、結晶を作らせる晶析操作を、特に昇華と呼ぶ。
