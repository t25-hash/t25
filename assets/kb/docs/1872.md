# 2・1・4 分離操作の選択法

> **章**: 第2章 分離操作
> **節**: 2・1 分離操作の概要

分離操作の選択法を例示したい.原子力発電の燃料であるウランが将来枯渇することに備えて,海水からのウラン採取が日本を中心に検討されてきた.海水中には1トン中に約3mg のウランが溶けている.全世界の海水の量から,海水中のウランの総量は45億トンと計算される.海水中でウランは三炭酸ウラニルイオンUO(CO)(ウランの錯イオンと名付ける)という形態で溶けている.ここで想定する分離の課題は,海水を原料にして,原子力発電所のウラン燃料の中間製品であるイエローケーキと呼ばれる黄色い結晶粉末を製造することである.海水中には,ウランのほかに,多種類のイオンが広い濃度範囲にわたって溶けている.したがって,イエローケーキを製造するには,海水中のウラン濃度(3mg-U/m)の100万倍程度(kg-U/mの濃度レベル)まで液中にウランを濃縮しつつ,他の成分の濃度を大きく減らす必要がある.海水中のウランを採取する操作を考える.六つの分離操作,すなわち蒸留,ガス吸収,抽出,吸着とイオン交換,晶析,そして膜分離の中から選ぼう.海水の温度を上げてもウランの錯イオンが気相に移ることはないだろうから,蒸留やガス吸収を選択できない.次に,海水中のウラン濃度が低いので,温度を下げたり,沈澱反応を利用したりする晶析は無理であろう.さらに,ウランの錯イオンより一回りサイズの小さい孔の開いた膜を使った膜分離を採用するには,海水中の微粒子のろ過というたいへんな作業が必要である.残る選択肢は,抽出と吸着とイオン交換である.抽出では,油相として,ウランを特異的に捕捉する抽出試薬を有機溶媒に溶かした油を使用する.その油と海水とを接触させると,油が小さな油滴となって海水へ混入したり,油から抽出試薬が海水に溶出したりという欠点がある.そうなると,有機溶媒や抽出試薬のロスのためにコストがかかるだけでなく海水が汚れる.これらの欠点は固体吸着材を使って克服できる.ところが,従来の吸着あるいはイオン交換用の材料ではウランを採取できなかったので,ウランを特異的に捕捉する新しい吸着材が日本で開発されてきた.この吸着材を1箇月間,海水と接触させると,吸着材1kg に1g のウランを吸着できる.それを少量の酸を使って溶出させると約50g-U/mにまでウランを濃縮できる.この後,イオン交換操作を繰返してウランを精製する.こうして得られた高濃度ウラン溶液に沈澱剤を加えるという晶析操作によってイエローケーキをつくることができる.ここでは,海水ウラン採取という分離の課題を想定して分離操作の選択法を述べた.他の分離の課題では,分離対象,条件,そして制約が異なるので,それぞれ最適な分離操作が選ばれる.分離操作の例を読み返してほしい.
