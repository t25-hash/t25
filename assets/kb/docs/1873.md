# 2・1・5 分離装置の設計と運転

> **章**: 第2章 分離操作
> **節**: 2・1 分離操作の概要

読者が抱えている分離の課題では、工業規模での分離が検討されるであろう。そのために、分離操作、あるいは反応と分離操作との組合せを決定したのち、分離装置の設計に入る。装置設計は、平衡論と速度論に基づいて行われる。装置や配管類の材質を選定の8―124うえ、装置設計がすむと、引き続き、装置の最適な運転条件を探索する。こうして工業規模での分離のプランができあがる。混合溶液を加熱すると、揮発しやすい物質(低沸点物質)は蒸気中に、揮発しにくい物質(高沸点物質)は溶液中にそれぞれ濃縮される。この揮発性の違いを利用して混合溶液を濃縮・分離する方法を蒸留(distillation)という。これは種々の混合溶液に適用でき、例えば溶液中に熱分解しやすい物質を含んでいても、低圧下で加熱すれば低い温度で蒸発が起き、その物質を熱分解させずに分離することができる。また、常温常圧で気体であっても、加圧または冷却して液体にすれば分離することができる。このため、蒸留は液体混合物の主要な分離法として化学工業のみならず原子力工業や半導体工業などでも広く用いられている。蒸留操作の特徴をまとめると次のようになる。(1) 原料を連続的に供給しながら所定の濃度の製品を連続的に取出すことができ、大量処理に適している。(2) 熱を加えて分離を行うため、原料と溶け合わない第3物質を加えて分離を行う抽出や吸着操作と比べると分離プロセスが簡単になる。(3) 蒸留は、粗分離から高純度分離に至るまで適用できる。
