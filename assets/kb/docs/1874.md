# 2・2・1 気液平衡(vapor-liquid equilibrium)

> **章**: 第2章 分離操作
> **節**: 2・2 蒸留

a. 純物質の気液平衡物質の液体を圧力一定の条件で加熱すると液温が上がり,それとともに液の蒸発が始まる.しかし,液温がある温度に達すると,加えられた熱はもっぱら液の蒸発にだけ使われ,液温は液体が存在する限り一定に保たれる.このとき蒸気と液は平衡状態にあるといい,このときの圧力を飽和蒸気圧(最初に設定した圧力に等しい)という.純物質の飽和蒸気圧욡は大気圧付近では温度のみの関数となり,次式により相関できることが知られている.ln욡=욡-욡욡+(2・1)これをアントワン式(Antoine equation)といい,その係数욡,욡,욡をアントワン定数という.これらの係数の値は,その物質の蒸気圧データから最小二乗法により得られている.これらの値はハンドブックなどに記載されているが,係数の算出に用いた蒸気圧データとその温度範囲によって少し異なってくることに注意する必要がある.b. 理想溶液(ideal solution)の気液平衡ある濃度の混合溶液を圧力一定の密閉容器に入れて加熱すると,液温の上昇とともに蒸発が起き,液濃度も変化していく.しかし,時間が十分に経過すると,液濃度および液温はいずれも一定になる.このような状態を平衡という.このとき,ラウールの法則(Raoult’s
