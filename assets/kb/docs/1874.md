# 17・2・4 プラズマ流体（plasma flow)

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・2 機能性流体

a.生成法と分類 プラズマは，気体にエネルギーを与えるこ
とにより気体の一部が電離した状態であり，イオン，電子などの荷
電粒子と原子，分子の多成分粒子から構成される．全体としては電
気的中性で，プラズマ構成粒子は，個別的挙動およびデバイの長さ
（debye length）λが関係する集団的挙動を示す．プラズマの性質
は，作動圧力 が重要因子で，プラズマ密度 とプラズマ温度
により大きく変化する．生成法としては，コロナ放電，グロー放
電，アーク放電などの直流（DC）放電と，高周波（RF）放電，マ
イクロ波放電やパルス放電がある．これにより，大気圧程度で用い
られる熱プラズマ（thermal plasma）と，0.1～10Paの低圧下で
電子温度が重粒子温度に比べて高い低温プラズマ（cold plasma）
がある．
b.物理化学的性質と素過程 プラズマ流体の機能性発現に
は，以下の性質が関与する ．解離（dissociation)，励起（ex-
citation)，電離（ionization)，再結合（recombination）などの
素過程による荷電粒子の生成と消滅，粒子間の弾性または非弾性衝
突および電磁場下での荷電粒子の加減速運動やラーモア半径 の
サイクロトロン運動，シース（sheath）に代表されるプラズマと
壁面との干渉，温度と圧力の関数であるプラズマ物性，高温下で大
きな変物性を有する乱流構造，アーク柱の変動や電離不安定性に起
因するプラズマ流の不安定現象，異なる反応速度や低圧下で起こる
温度こう配や濃度こう配の大きな領域での熱非平衡状態（thermal
nonequilibrium condition)，高エネルギー電子や活性ラジカル自
身，または窒素，水素，アルカリ金属などの反応性気体との混合に
よる高機能化，放電形態，流動形式や電磁場による制御などであ
る．
c .基礎方程式 粒子間衝突が頻繁な高圧下での熱プラズマで
は，局所平衡（local thermodynamic equilibrium）を仮定し，連
続流体として流体モデル（fluid model ) を用いる解法がある．
また，低圧下の低密度プラズマでは，個々の粒子軌道を追跡する粒
子モデル（particle model ) による解法がある．

d.種々の流れ 流動場の状態は，高密度プラズマでは連続流
体，低密度プラズマでは希薄気体の取扱いとなる．また，層流およ
び乱流，熱あるいは化学的に平衡または非平衡な状態もある．噴流
としてはプラズマジェット，外部流では，平板上，円柱まわり，ま
た内部流としては，円管内，ダクト内流れ，そのほかに，相変化や
化学反応を伴う粒子や液滴を含んだ複雑プラズマ流，電極間のアー
ク流もある．
e .診断法 電子温度や電子密度のプラズマパラメータや熱
流体力学特性，そして分光特性は，パターン認識，可視化などを統
合した計測診断から明らかにされる．受動的測定として，放電電
流・電圧計測，ラングミュア探針（Langmuir probe）測定，プラ
ズマから放射される電磁波を分光法（spectroscopy）によるスペ
クトルの絶対強度法（absolute spectral intensity method）や相対
強度法（relative spectral intensity method）を用いて測定する方
法などがある．また，レーザによる能動的測定として，プラズマに
電磁波を入射し，その反射，干渉，透過，散乱，屈折を利用する方
法もある．
