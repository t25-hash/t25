# 2・2・1 気液平衡(vapor-liquid equilibrium)

> **章**: 第2章 分離操作
> **節**: 2・2 蒸留

a. 純物質の気液平衡物質の液体を圧力一定の条件で加熱すると液温が上がり、それとともに液の蒸発が始まる。しかし、液温がある温度に達すると、加えられた熱はもっぱら液の蒸発にだけ使われ、液温は液体が存在する限り一定に保たれる。このとき蒸気と液は平衡状態にあるといい、このときの圧力を飽和蒸気圧(最初に設定した圧力に等しい)という。純物質の飽和蒸気圧は大気圧付近では温度のみの関数となり、次式により相関できることが知られている。これらの係数の値は、その物質の蒸気圧データから最小二乗法により得られている。これらの値はハンドブックなどに記載されているが、係数の算出に用いた蒸気圧データとその温度範囲によって少し異なってくることに注意する必要がある。b. 理想溶液(ideal solution)の気液平衡ある濃度の混合溶液を圧力一定の密閉容器に入れて加熱すると、液温の上昇とともに蒸発が起き、液濃度も変化していく。しかし、時間が十分に経過すると、液濃度および液温はいずれも一定になる。このような状態を平衡という。純物質の飽和蒸気圧は温度のみの関数なので、理想溶液の相対揮発度も温度のみの関数となる。したがって、温度一定のときの気液平衡(定温気液平衡)、すなわちある液濃度に対する平衡蒸気濃度は相対揮発度がわかっていれば式(2・3)から計算することができる。それに対して、定圧気液平衡では液濃度が変わると沸点が変化するため相対揮発度も厳密には変化するが、その変化は小さいため、相対揮発度(relative volatility)を近似的に定数とみなして式(2・3)から定圧気液平衡を計算することができる。このとき、相対揮発度には沸点における両成分の相対揮発度の幾何平均値を用いることが多い。なお、液濃度と平衡蒸気濃度の関係を表した図を-線図という。c. 実在溶液の気液平衡実在溶液の中で、ベンゼン-トルエン系のように化学構造の類似した物質の混合溶液は近似的にラウールの法則を満たすが、実在溶液の多くは理想溶液から外れた挙動を示す。一方、実在気体は理想気体と異なる挙動を示すが、大気圧付近では理想気体とみなせることが多い。すなわち、大気圧付近における実在溶液の気液平衡は、溶液に非理想性を、気相に理想性を仮定して、次のように表すことができる。活量係数の推算式は数多く提案されているが、その中で工学的に広く用いられているものにウィルソン式(Wilson equation)、NRTL式、UNIQUAC式、UNIFAC式などがある。一例としてウィルソン式を示す。図2・4は、メタノール-水系を例に、液相のメタノール濃度に対する沸点、および蒸気相のメタノール濃度に対する露点の関係を示したものである。このような図を露点沸点曲線といい、2成図2・3 2成分系理想溶液の気液平衡図2・4 メタノール-水系の沸点・露点曲線図2・5 メタノール-水系の液相活量係数分系気液平衡を表すもう一つの表示法となる。図2・5は、図2・4と同じデータに対する液相活量係数を示したものである。活量係数は=1の近傍で1に近い値をとるが、それ以外では1よりも大きくなり、実在溶液の気液平衡が理想溶液のそれと異なることがわかる。
