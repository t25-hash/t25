# 17・3・2 血液と微小循環

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・3 生体内の流れ

心臓から出た動脈は分岐して細動脈となり，さらに分岐して毛細
血管（最小直径は5m程度）となったのち，合流して細静脈とな
る．これら細動脈から細静脈までの直径が数十 m以下の微小血管
における血液循環を微小循環と呼ぶ．
微小循環における血流は赤血球の変形能の影響を強く受ける．柔
軟な粒子が分散した媒質が管内を流れる際に，粒子には管軸方向に
力が作用することが知られている．血管内の血流においても，血管
壁面近傍には赤血球の存在しない厚さ数 m程度の血しょう層が存
在する．

eﬀect ）と呼ばれており，図17･11に示すように，管の直径が1
