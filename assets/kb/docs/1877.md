# 2・2・4 多成分系連続蒸留

> **章**: 第2章 分離操作
> **節**: 2・2 蒸留

2成分系蒸留で完全分離を仮定すると,塔頂と塔底からそれぞれ低沸点成分と高沸点成分が純物質として得られる.しかし,多成分系では完全分離を仮定したとしても,成分数に応じた蒸留塔が必要となる.このため,多成分系蒸留塔のみならず分離のシーケンスについても考慮する必要がある.a. 成分分離の順序図2・13は,成分A,B,C,D(沸点の低いものから順に並べる)からなる4成分系混合物について,どの成分も塔頂か塔底のいずれかにしか現れないものとしたときの成分の分離パターンを示したものである.すべての成分を分離精製する場合,蒸留塔は3塔必要となり,分離の仕方には5通りある.この中で最適な分離の順序は,5通りの蒸留塔列について蒸留計算を行い,それぞれの塔列で必要となるコスト(設備費と運転経費)を比較して決定される.したがって,4成分系では15塔分の蒸留計算を行うことになる.なお,成分系混合物の分離の仕方は2-1-1通りある.あるいは,2成分系分離にはAB
