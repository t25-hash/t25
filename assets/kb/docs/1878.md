# 2・3・1 ガス吸収の基礎

> **章**: 第2章 分離操作
> **節**: 2・3 ガス吸収

液体中へのガスの溶解度は吸収液の選択、吸収速度の計算などに必要な基本的物性で、気相中の吸収される溶質ガスの分圧により液中の溶質濃度すなわち溶解度が決まる。一般に溶解度は気相中の溶質ガス分圧が高いほど大きく、温度が高いほど小さい。溶解度を分圧の関数として表した曲線を溶解度曲線、または平衡曲線と呼ぶ。一定温度で溶質の溶解度が小さい範囲では溶質ガス分圧はに比例するが、これをヘンリーの法則(Henry’s law)といい、気相、液相中の溶質の濃度表示により、式(2・47)~(2・49)のように書かれ、ヘンリー定数(Henry’s law constant)が異なる。ここで、気相と液相は平衡状態にある。表2・2は種々のガス-水系のヘンリー定数と温度の関係を示したもので、ヘンリー定数が小さいほどガスの溶解度が大きくなる。ガスの溶解度データの所在については化学工学便覧を参照されたい。
