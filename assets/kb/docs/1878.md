# 17・4・1 超流動（superfluidity）ヘリウムの性質

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・4 超流動

ヘリウムは，絶対温度4.2Kで気体から液体に変わり，絶対零ムダ温度2.172K以下で，粘性が消失した超流動 という劇的なウムIIという）．
この現象の本質は，本来ミクロな世界の支配則でールで現れることにある．
液体ヘリウムは量子効果が顕著に現れる呼ばれる．
学の本質は粒子と波動の二重性にある．
粒子と考えられていた原子られていた光が粒子の性質をも有すというものである．
もっとも十十分短く，波動性は事実上現れない．
しかし，ヘリウムのような軽い原子を極低温に冷却すると，その物質波の波長は増大して平均原子間距離と同等またはそれ以上になり，波動性すなわち量子効果が現れるようになる．
超流動は，量子統計力学 により初めて理解できる現象である．
一般に，十分高温ではすべての粒子はマクスウェル・ボルツマン統計に従うが，低温で量子効果が現れると，その波動関数の対称性に依存して，ボース・アインシュタイン統計かフェルミ・ディラック統計かのどちらかに従うようになる．
自然界には2種類の安定なヘリウム原子の同位体HeとHeが存在し，Heはボース統計に従うボース粒子，Heはフェルミ統計に従うフェルミ粒子である．
両度まで冷却しても固体にならない．
この液体ヘリウムは，さらにラ状態に転じる（ラムダ温度より高温側をヘリウムI，低温側をヘリある量子力学的性質が，極低温に冷却することによりマクロなスケ液体という意味で，量子液体または量子流体（quantum fluid）と極低温で量子効果が顕著になる理由は次のとおりである．
量子力や電子が実は波の性質も有し（この波を物質波という），波と考え分温度が高ければ，原子の物質波の波長は平均原子間距離に比べていう特殊な渦が重要な役割を担う．
量子渦は超流体流れが作る渦であり，その循環は を単位として量子化される（ ：プランク定数， ：ヘリウム原子の質量）．
通常流体の渦の循環が任意の連続的な値をとりうることとは異なる．
また量子渦の芯半径は原子サイズと，非常に細いこともわかっている．
ヘリウムIIをある臨界流速以上で流すと，量子渦が高密度で絡まり合った渦タングルが形成され（超流動乱流），それと常流体流れとの間に生じる摩擦により，ヘリウムIIの非粘性は破壊される．
これはヘリウムII管内流の実験と，量子渦の運動の数値シミュレーションにより明らかにされている．
