# 2・3・2 ガス吸収操作

> **章**: 第2章 分離操作
> **節**: 2・3 ガス吸収

ガス吸収にはCOを水に吸収させる場合のように、溶質ガスを液中に物理的に溶解させる物理吸収と、COをNaOH やアミン水溶液に吸収させ、溶質ガスが吸収液と化学反応を起こして溶解する反応(化学)吸収がある。a. 物理吸収(physical absorption)ガス吸収は溶質ガスがガス(気相)本体から気液界面へと移動し、液相に溶解し、界面から液相本体へ移動すると考えられる。ルイス・ホイットマン(Lewis-Whitman)は図2・15に示すように気液界面の両側にガス境膜と液境膜を考え、両境膜内では分子拡散により物質移動が起こり、溶質の濃度分布ができるとした二重境膜説(two-film
