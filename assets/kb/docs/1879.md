# 2・3・2 ガス吸収操作

> **章**: 第2章 分離操作
> **節**: 2・3 ガス吸収

ガス吸収にはCOを水に吸収させる場合のように、溶質ガスを液中に物理的に溶解させる物理吸収と、COをNaOH やアミン水溶液に吸収させ、溶質ガスが吸収液と化学反応を起こして溶解する反応(化学)吸収がある。a. 物理吸収(physical absorption)ガス吸収は溶質ガスがガス(気相)本体から気液界面へと移動し、液相に溶解し、界面から液相本体へ移動すると考えられる。ルイス・ホイットマン(Lewis-Whitman)は図2・15に示すように気液界面の両側にガス境膜と液境膜を考え、両境膜内では分子拡散により物質移動が起こり、溶質の濃度分布ができるとした二重境膜説(two-film the-ory)を提案した。また、気液界面では溶質ガスの溶解は速やかに起こり、気液両相の濃度間には溶解平衡が成立するとした。添字は気液界面での値を示す。しかしながら、気液界面の濃度を用いずに、実測が可能な気液両相本体の濃度を用いて物質移動流束を表すほうが便利な場合がある。総括物質移動係数は気相物質移動係数と液相物質移動係数を用いて次のように表せる。この場合が大きい、すなわち溶解度が小さいガス吸収では液相抵抗が支配的になるので、の大きい吸収装置、すなわち、後述の2・3・3・a項で示すガス分散型装置を選択すれば、液相抵抗を低下させることができる。二重境膜説では境膜内の溶質ガスの物質移動は分子拡散によるので、溶質の濃度が希薄であるとすると、気相物質移動係数と液相物質移動係数は式(2・55・a)(2・55・b)と式(2・56・a)(2・56・b)で書ける。二重境膜説のほかに、液相側の物質移動を表すモデルとしては浸透説、表面更新説などがある。したがって、吸収速度は物理吸収の場合に比べて増大する。その程度を表すのに反応係数βを用いる。ここでは溶質ガスA が反応成分B を含んだ液中に吸収され、式(2・61)に示す不可逆瞬間反応を起こす場合を二重境膜説で考える。図2・17に示すように、成分A,B は液相境膜の両側から拡散してPQ面で瞬間的に反応して、A,B の濃度は零となる。すなわち、βは液中の成分B の濃度が大きくなるほど大となる。本解析はSOのNaOH 水溶液中への反応吸収に適用で図2・17 境膜内濃度分布きる。
