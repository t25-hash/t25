# 2・3・3 ガス吸収装置

> **章**: 第2章 分離操作
> **節**: 2・3 ガス吸収

a. 吸収塔の物質収支一般に分散相内の流体の乱れは連続相内よりも小さく、物質移動係数も小さいので、ガスの溶解度が大きく気相物質移動抵抗が支配的な系では液分散型装置を、液相物質移動抵抗が支配的な系ではガス分散型装置を用いる。ガス分散型吸収装置には、図2・18に示すように充てん塔(packed tower)や気泡塔(bubble column)などの気液両相の濃度が連続的に変化する微分接触装置と泡鐘塔などの段塔のように階段的に変化する階段接触装置に大別できるが、ここではおもに微分接触装置について解説する。充てん塔のような微分接触装置では気相と液相の濃度は、充てん塔の高さ方向に連続的に変化する。ガスは塔底から塔頂へ送られ、逆に吸収液は塔頂から塔底に流れる連続向流充てん塔について、塔全体の溶質成分の物質収支をとる。塔底と塔頂における値に添字1と2を付ける。塔底でこの濃度になるためには無限大の高さの充てん塔が必要となる。通常の操作では最小液ガス比の1.25から2.0倍程度の液ガス比を用いる。また、一定の液モル質量速度に対し、ガスのモル質量速度を増やしていくと、ガスの圧力損失が増大し、やがて液が塔内を流下せず、塔頂に向かって逆流するフラッディング(flooding、いつ汪)が起こり、運転が不可能となる。高さにおける気相および液相中の溶質成分の濃度変化はこの部分での吸収速度に相当するから次式で書ける。すなわち、塔内での溶質成分の移動の起こりにくさを表す量で、これを移動単位数(: number of transfer unit)という。直線PQをタイラインという。さて、平衡線、操作線がほぼ直線とみなしうる場合には、次式より移動単位数を求めうる。
