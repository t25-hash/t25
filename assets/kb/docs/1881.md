# 2・3・4 ガス吸収プロセス

> **章**: 第2章 分離操作

ガス吸収操作は混合ガス中の有効成分の分離,回収や精製,また有害成分の除去などに幅広く利用され,新しい吸収剤の開発に伴い新プロセスが開発されている.ガス吸収プロセスでは吸収塔で吸収液にガスを吸収させた後,放散塔でガスを放散させ吸収液を再生する.ガス吸収プロセスのメカニズムは次の3種類に分類される.(1) 物理吸収プロセス:物理的に溶解するガスの溶解度差を利用して分離する.高圧低温条件でのガス吸収に適する.(2) 反応(化学)吸収プロセス:低圧条件下で濃度推進力が小さい場合に,溶質ガスが液中で反応し吸収される化学反応を利用する.反応吸収では,ガス溶解量が小さい範囲では平衡分圧はあまり増加せず,ある溶解量を超えると急増する.選択分離性に優れている.(3) 物理化学的吸収プロセス:物理吸収と反応吸収を有効に組合わせた中間的なプロセスとして,アミン系吸収液の吸収速度の速さと物理吸収法を組合わせたプロセスがある.a. 物理吸収プロセスは吸収液の再生に減圧,空気・スチームとの直接接触による放散を利用するので,吸収液の開発とともに,プロセスフローの開発が必要となる.水は塩酸などの酸性ガスやアンモニアの吸収液として利用できるが,吸収容量が小さい.しかし,高圧水が得られれば,プロセスは簡単になる.レクチゾール(rectisol)法(ライセンサ:Linde/Lurgi社)では低温・高圧下のメタノールを吸収液に用い,石炭ガス化ガスや合成ガスの精製に用い,硫化水素を0.1ppm以下,炭酸ガスを10ppm以下に除去できる.セレクソール(selexol)法(UCC社)は吸収液としてポリエチレングリコールのジメチルエーテル溶液を用い,天然ガスや合成ガス中の硫化水素や炭酸ガスの除去に用いられる.(2) 反応吸収プロセスは可逆反応による吸収・放散再生を利用するため,再生エネルギー,腐食性が問題となる.主要なガス吸収プロセスとしては1天然ガスの精製,2液化天然ガスの前処理,3アンモニア,メタノールなどの合成ガスの精製,4ガス化発電用ガスの精製,5石油精製の脱硫オフガス,硫黄回収設備テールガスの処理などがあるが,これらは硫化水素,二酸化炭素,有機硫黄化合物(メルカプタン類,サルファイド類など)の分離除去がおもな目的である.商用プロセスとしてはアルカノールアミン法,熱炭酸カリ法などがある.液をノズルで噴霧するためポンプ動力が大固形物によるスプレーの目づまりを生じるガス流速の変動に対する適合が小さいガス処理量が小さいぬれ壁塔ガスの圧力損失が小さい大量のガスあるいは液を処理するとき,多数の垂直管が必要になり,均一分散が難しい塔の高さが高くなる圧力損失が大きいガス処理量が小さい圧力損失が大きいガス処理量が小さい液濃度が均一になるので液向流の利点を発揮できない図2・23代表的な気液接触装置の物質移動特性図2・24 抽出の基本操作1アルカノールアミン法:吸収液としてモノエタノールアミン,ジエタノールアミン,メチルジエタノールアミンの水溶液が広く用いられている.選択性が高く,吸収速度が速い.再生のためのスチーム量が大きい.2熱炭酸カリ法:炭酸カリ水溶液とアミンを用いる.吸収温度を比較的高くできるため,吸収液の冷却や熱交換が不要で,熱収支がよい.3直接酸化還元法:吸収液として鉄キレート化合物の水溶液を用いる.二酸化炭素は吸収しないが,硫化水素含有ガスから直接硫黄を回収できるため,中小規模のプラントに適する.b.
