# 2・4・1 抽出の基礎

> **章**: 第2章 分離操作
> **節**: 2・4 抽出

抽出(extraction)とは、目的成分を溶剤を用いて溶かし出し、溶液二相間における溶解度の差(分配の差)を利用して分離する手法である。目的成分が固体の場合を固液抽出(solid-liquid extrac-tion)、液体の場合を液液抽出(liquid-liquid extraction)という。固液抽出の場合は、目的成分のみをよく溶かし出す溶剤を使用することが重要である。一方液液抽出は、二相間の目的成分の分配比が重要であり、さまざまな工業分野で用いられている。液液抽出操作では、抽出したい目的成分を溶質(あるいは抽質)、それを溶かしている溶媒を希釈剤と呼ぶ。また希釈剤に溶質が溶けた溶液を原料という。この原料中から目的成分である溶質を溶かし出す液体を溶剤(あるいは抽剤)と呼ぶ。例えば、酢酸とベンゼンの混合溶液から酢酸を抽出操作で分離する場合、酢酸をよく溶解する水を溶剤として用いると都合がよい。つまりこの場合、酢酸を溶質、ベンゼンを希釈剤、この混合溶液を原料と考えることができる。原料中に溶剤としての水を加えると、ベンゼン中のほとんどの酢酸は水相中へ移動し、比重の軽い上層のベンゼン相と比重の重い下層の水相に分かれる。このとき、酢酸が8―133化学の基礎図2・25 直角三角座標による3成分組成の表し方抽出された上層のベンゼン相を抽残液、酢酸を抽出した下層の溶剤相を抽出液と呼ぶ。これが抽出の基本操作である(図2・24)。このように抽出操作では、溶質の二相間の溶解度差(分配差)を利用しており、通常、混和しない水と油(有機相)の二相系が用いられる。つまり液液系の抽出操作においては、溶質、希釈剤、溶剤の3成分の液液平衡関係が重要となる。
