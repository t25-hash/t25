# 2・4・2 抽出平衡(extraction equilibrium)の表現法

> **章**: 第2章 分離操作
> **節**: 2・4 抽出

a. 液液平衡関係の表現3成分の液液平衡関係を表す場合は,一般に図2・25に示すような直角三角形の座標を用いる.この場合,慣習的にA に抽質,B に希釈剤,Cに溶剤をとる場合が多い.このとき,点M は縦軸で表されるA 成分すなわち抽質の重量分率を示し,また横軸から読み取れるC成分すなわち溶剤の重量分率を表す.3成分の間にはつねに式(2・88)の関係が成立するので,希釈剤B 成分の重量分率は,式(2・89)で求められる.++=1(2・88)=1--(2・89)この直角三角座標を用いると,組成の異なる二つの3成分溶液を混合した場合の混合後の重量分率を容易に求めることができる.例えば,図2・25中のP点で示す重量分率,,の溶液(kg)とQ点で示される,,の溶液(kg)を混合してできる3成分系の組成は,M 点をPM : QM=: に内分する点にとることによって,点M の重量分率,,で与えられる.液液抽出操作は,二相間の分配比の差に基づく分離操作であるので,二相が形成されることが必要である.このため図2・26に示すような3成分系における溶解度曲線が重要となる.この溶解度曲線は,実験的にはF点で示される組成の原料に純溶媒Cを加えていくことにより濁りの発生する点を求めることによって描くことができる.具体的にF点は溶剤Cを加えることにより直線FC上を移動し,S 点で濁りが生じることになる.F点の重量分率を種々変化8―134電気・電子の基礎,化学の基礎させて同様な実験を行うと,溶解度曲線R′,R,S,E,E′を求めることができる.またこの場合,溶解度曲線は温度に敏感であるので,温度一定の条件下でのデータ取得が必要となる.またR′はB成分に対するC成分の溶解度をE′はC成分に対するB 成分の溶解度を示している.この溶解度曲線より上側では,3成分は均一溶液となるので抽出操作は行えない.溶解度曲線内部の二相領域で抽出操作を行うことができる.溶解度曲線内部の領域では必ず二相を生じる,例えば図中のM点の組成を有する3成分の溶液を調製すると,R 点,E点の組成を有する二相に分かれる.液液平衡関係にあるこのR とEを結ぶ線をタイライン(tie line)と呼ぶ.同様に,溶解度曲線内部のさまざまな液組成についてタイラインを引くことができるが,このタイラインの頂点をプレイトポイントと呼ぶ.実験的には,二つの液相に分かれた上層と下層の成分を分析し,重量分率を決定することによってR 点,E点で示されるタイラインを求めることができる.実際の抽出操作では図2・26(b)で示される分配曲線を用いる場合が多い.例えば酢酸-水-ベンゼン系の場合,上層(ベンゼン)および下層(水)中の酢酸濃度をそれぞれ分析することによって図のような分配曲線を描くことができる.さらに,分配曲線とタイラインは図で示されるように対応している.b. 固液平衡関係の表現固液抽出では,抽質A と抽剤Cの2成分系が相互溶解度に限度のない完全溶解系であるとき,固体原料に抽剤を加えて放置すると,抽質は固体内部に浸透した抽剤中に溶出する.固体内部の溶液組成と固体外部の溶液組成に差があると,抽質は拡散によって固体内部の液相から外部の液相に移動し,最終的には抽質を含まない不溶性固体B と,抽質A と抽剤Cとの均一組成の2成分溶液の固液二相状態となる.この平衡に達した固液二相を相分離するとき,一般に固体相は,溶液の一部を付着して同伴する.すなわち,固液分離して得られる抽出液Eは抽質A と抽剤Cの2成分系溶液であり,抽残物R は不溶性固体(担体)B と抽出液と同組成の溶液との3成分系混合物である.したがって,この場合の固液平衡の関係は,図2・27の三角線図を用いて表現できる.抽残物組成を示す点R は抽出液組成を(a)(b)図2・26 (a) 溶解度曲線とタイライン,(b) 分配曲線図2・27 代表的な3成分系固液平衡状態図与える点Eと固体の状態を表す点B を結ぶ直線上に位置するが,その位置は固液分離操作の条件によって異なる.
