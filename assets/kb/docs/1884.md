# 2・4・3 抽出操作(extraction operation)

> **章**: 第2章 分離操作
> **節**: 2・4 抽出

工業的には図2・28で示すようなミキサセトラ(mixer-settler)型の抽出装置がよく用いられる。ミキサ部に導入された溶剤S と原料Fは、両相間の物質移動を促進するためにかくはん機でよく混合される。ここでミキサ部における物質収支は次のように表せる。これより、得られる抽出液と抽残液の量は次のように表すことができる。溶剤を各段に供給し、単抽出を連続的に多段で行う操作が並流多段抽出操作である。図2・29に示すように、原料が供給されている段を第1段として、順次番号をつけ、段めについて物質収支をとる。並流操作(parallel flow operation)に比べ向流操作(counter flow operation)のほうが効率が良いため、工業的に最もよく使用される抽出操作である。ここで、供給する原料を(kg/h)、これと向流に送入する溶剤を(kg/h)とし、各段での抽出により、取出される抽出液を(kg/h)、抽残液を(kg/h)とする。この場合各段の番号は、図2・30のように定義する。つまり、全物質の収支および溶質成分の収支より次式が得られる。共通点であるD 点は操作点といわれ、F点とE点およびC点とR点とを結んだ二つの直線の交点として与えられる。分離に必要な理論段数はE点から出発し、まずEと溶液平衡にあるR、すなわちタイラインERを求める。次にD 点とR点を直線で結び溶解度曲線上に点Eを与える。この操作を繰返し、最終抽残液Rを超えるまで行う。求める理論段数はタイラインの数で与えられる。
