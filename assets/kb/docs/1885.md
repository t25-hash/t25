# 17・7・1 微小重力下における流体現象

> **章**: 第17章 特殊な環境下の流れ
> **節**: 17・7 無重力流体

図17･ 14 マイクロカプセル（中空液滴）の抵抗係数
a.複雑流体（complex fluids )：コロイド，泡沫流体，磁性流体
コロイドの相変化（液体から固体への相変化）がどのような条件で
生じるかは工学上重要であり，微小重力下のほうがこの現象の観察
が容易である．泡沫流体は物質の分離技術に使われているが，通常
重力下ではその流体力学的特性が時間の経過とともに変化する．微
小重力下では，特性が変化せず，実験時間が長くとれる．磁性流体
中の粒子は，微小重力下では均一に分布するので，そのレオロジー
特性が顕著となり，ショックアブソーバ，クラッチの制御など，応
用範囲が広がる．
b.混相流と熱移動（multiphase flow and heat transfer ) 通
常重力下の沸騰，凝縮等の相変化が生じる現象に重力がどのように
作用しているかを解明することは，地球上のエネルギー変換機器の
性能向上をもたらす．また，宇宙空間でのエネルギー機器の小型化
などにとっても重要である．分子や原子の拡散が重力の影響を受け
ないので，流体内での濃度分布や温度分布が通常重力下とは異な
る．
c .界面現象（interfacial phenomena) 界面（固体-液体，液
体-液体，気体-液体）形状がどのように変化する（運動する）か，
また，形状を維持するかということは，材料プロセッシング，コー
ティングなどの工業技術と関係し重要である．界面の挙動は，加
熱，冷却，化学反応などが界面間の相互作用に関係する．これら
は，液体の界面張力とぬれ性に依存する．通常重力下では，重力の
影響下に隠れていることが多い．しかし，宇宙空間における材料プ
ロセッシング，液体燃料タンクの液面制御，ライフサポートなどに
関係する流体制御では，重要である．
d.力学と安定性（dynamics and stability) 液滴や気泡の運
動やその安定・不安定性は液体の微粒化や材料プロセッシングにお
ける気泡の除去と関係し，重要である．通常重力下では，気泡の除
去は容易であるが，微小重力下で気泡を除去することは困難で，気
泡除去技術の確立が求められる．微小重力下では，液滴や気泡が球
形になるので，表面張力が支配的な現象の解明が期待できる．流体
と電場，磁場の相互干渉の解明では，ジュール熱による対流が生じ
ないので，その基本的現象の解明が期待できる．
