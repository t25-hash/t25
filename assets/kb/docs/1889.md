# 2・5・2 晶析に関する基礎事項

> **章**: 第2章 分離操作
> **節**: 2・5 晶析

a. 結晶系(crystal system)結晶は、原子あるいは分子を構成単位とした周期構造をもっている。周期構造の基本単位(単位格子)は、物質固有であり人為的に自由に変えることはできない。結晶は、単位格子の形によって七つの結晶系に分類される。結晶系を図2・34に示す。例えば、塩化ナトリウム結晶は、ナトリウム原子と塩素原子が構成単位であり、これらがイオン結合し、立方晶系に属する。氷の構成単位は水分子で、水素結合によりこれらが結合し、その結晶系は六方晶系である。このように結晶系は、構成単位(原子あるいは分子)の結合様式によって決まる。b. 結晶形状(crystal shape,crystal habit)上述の結晶系は結晶の内部構造にかかわる原子・分子レベルの問題であり肉眼では見ることができない。われわれが肉眼で見ることができるのは、結晶の外形あるいは形状である。結晶の形状は結晶内部構造を反映したものではあるが、それのみでは決まらない。温度、過飽和度(後述)、不純物の存在、溶媒の種類などの成長条件によって変わる。これは、結晶の外形が各結晶面の成長速度の相対的違いで決まるからである。成長速度の遅い結晶面が大きく発達する。このように速度論的に決まる結晶形態を成長形(growth form)と呼んでいる。図2・35に成長形の説明を示した。形状の変化には2種類あり、結晶面の大小に相対的変化のみが見られる場合を晶癖変化といい、新たな結晶面が出現したりあるいは結晶面が消滅したりする変化を晶相変化といっている。しかし、英語ではこの区別はない。成長形とは異なる機構で決まる形状がある。その一つは、液滴が球になるのと同じ理屈すなわち表面エネルギー最小の条件を満たすべく平衡論的に決まる結晶形状で、平衡形(equilibrium 結晶多形(crystal polymorphism)結晶多形(単に多形ともいう)も、結晶内部構造にかかわる問題である。同一の物質あるいは同一の化合物でありながら結晶構造の異なる現象を結晶多形現象あるいは結晶多形といい、また、多形現象を示す結晶そのものを多形(polymorph)という。例えば、アミノ酸の一種L-グルタミン酸の結晶は多形を示し、αおよびβの多形をもつ。α型はβ型に比較して不安定で水に対する溶解度が高い。一般に、多形結晶には溶解度が高い準安定型と溶解度の低い安定型が存在する。医薬品のような複雑な分子を構成単位とする結晶は、二つ以上の多形をもつ場合が多い。それは、構成単位の分子が複雑な構造をもつと、分子間結合の組合せが多様となり、何とおりもの結晶構造を取りうるからである。図2・36にL-グルタミン酸の溶解度を示した。このように広い温度範囲にわたって一方の多形の溶解度が高い場合を単変系(monotropic system)といい、ある温度を境に溶解度が逆転する場合を互変系(enantiotropic system)という。多形は、医薬品の活性などのような結晶の機能にかかわるため、その制御は工業的に重要である。しかし、制御は難しい。溶媒を変えたり、冷却速度を変化させるなどして制御しているが、多分に経験的といわざるを得ない。また、多形は結晶内部の構造の問題ではあるが、結晶の形状にも影響を与える。なお、多形に準じる現象として、水和物結晶の水和数が変わる現象がある。これを擬多形(現象)といっている。なお、水和物結晶とは、溶媒の水を結晶構成単位として取込んでいる結晶のことである。d.
