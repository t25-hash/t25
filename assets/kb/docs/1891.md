# 2・5・4 結晶粒度分布(crystal size distribution)

> **章**: 第2章 分離操作
> **節**: 2・5 晶析

結晶粒度分布は製品特性として重要であり,また,後に続く工程すなわち遠心分離および乾燥の難易に影響を与える因子としても,重要である.ここでは,連続およびバッチ晶析における粒度分布の形成について,基本的事項を述べる.a. 連続晶析(continuous crystallization)最も簡単な連続晶析装置,MSMPR 型晶析装置(mixed suspension mixed prod-uct removal crystallizer)における結晶粒度分布は,結晶の個数収支(population balance)を適用することにより理論的に得られる.MSMPR 型晶析装置においては,内部懸濁液は完全混合状態であると仮定される.この仮定により装置内のどの部分においても溶液濃度,結晶個数密度,粒度分布がすべて同じとなる.さらに,懸濁液は分級されることなくそのまま排出されるとする.工業的大型装置の場合,完全混合の仮定は必ずしも成立しないが,ここでは粒度分布形成の基本として,完全混合,分級なしの場合について述べる.個数収支を考えるに当たって,「完全混合および分級なし」のほかに,次の仮定をおく.(1) 定常である.(2) 原料液には結晶は含まれない.(3) 結晶の破損,凝集もない.(4) 成長速度の粒径依存性もない.(5) 二次核の粒径は零である.これらの仮定により議論が簡単になる.まず,個数収支式を立てる前に,単位体積当たりの懸濁液に含まれる結晶粒子のフルイ下結晶個数を粒径で微分した関数をポピュレーションデンシティ(population crystallization)バッチ晶析の場合も,原理的には,結晶粒度分布の予測は可能である.しかし,これは非常に難しい.なぜなら,バッチ晶析過程は非定常であり,濃度,温度,過飽和度が時々刻々と変わり,核発生速度,結晶成長速度が複雑に変化するからである.また,これら速度過程が明らかになったとしても,ポピュレーションバランス式のみでは,粒度分布を導くことができない.マスバランス(物質収支)も同時に考慮しなくてはならない.粒度分布は,数値的にはともかく,解析的に求めることは事実上不可能である.バッチ晶析は,非定常過程で複雑ではあるが,一つの特徴がある.それは,基本的には結晶の滞留時間に分布がないということである.したがって,操作のはじめにいっせいに結晶核を発生させるか,あるいは,種結晶を投入し,それらを成長させれば,当然粒度のそろった(粒度分布幅の狭い)結晶が得られる.ただし,操作中に結晶核の発生があってはならない.これに対して先に述べた連続晶析では,滞留時間に分布があるがゆえに製品結晶には必然的に分布が生じてしまう.冷却バッチ晶析において,種晶を添加してそれを成長させる際,操作中の核発生を避ける方法として従来からよく知られている方法は,途中の過飽和度を過大にしないような冷却パターンの採用である.この方法は,マリン・ニヴルト(Mullin and Nyvlt)によって提案され,制御冷却法(programmed cooling method)といわれている.冷却バッチ晶析の場合,冷却速度を上げすぎると,成長による溶液濃度の減少が追いつかず,過飽和度が上がりすぎてしまう.すると,二次核発生が激しく起こり[式(2・124)(2・125)]種晶のみを成長させることが不可能となる.したがって,この制御冷却法では,過飽和度の増加を避けるため,結晶量の少ない初期においては低い結晶析出速度に見合うようにゆっくり冷却し,種晶が十分成長し結晶量が多くなった終期において,急速に冷却する.この方法によれば,バッチ操作時間全般にわたって過飽和度を低く保持できると期待される.これが,マリン・ニヴルトの基本的考え方である.彼らによると,制御冷却曲線は近似的に次式で与えられる.=--욞τ(2・139)ここに,は溶液温度,は初期温度,욞は終点温度,は時間,τはバッチ運転時間である.図2・45に制御冷却曲線を示した.図2・45には,比較のために直線冷却曲線および自然冷却曲線も示した.この自然冷却曲線は,一定温度の冷却水をジャケットに流して溶液を冷却するときの曲線で,温度は指数関数的に低下する.初期に急激に温度が下がるから,制御冷却の考え方からいえば,望ましくない冷却法である.ところが,望ましくないといわれてきた自然冷却法でも,ある臨界量以上に種晶を添加すれば,過飽和度の過大化を避け種晶のみを成長させることが可能となることが,最近明らかになってきた.図8―143化学の基礎ある.製品結晶の粒度分布は,種晶添加量の低い욦=0.051の場合は,二つのピークをもった分布となっている.すなわち,微結晶の発生が著しい.しかしながら,種晶添加量を増加させ욦=0.033とした場合は,結晶の粒度分布はピークが一つである.すなわち添加した種晶がそのまま成長している.このように,望ましくない冷却法とされてきた自然冷却法でも,種晶を十分添加すると微結晶(二次核)の発生は防止できる.なお,ここで種晶添加量욦は,理論結晶収量に対する種晶添加量の比として定義される.図2・47には,カリミョウバン製品結晶の体積平均径と種晶体積平均径の比を種晶添加量に対して示した.図2・47はシードチャート(seed chart)と名付けられている.種晶添加量を増加させると,製品結晶の体積平均径は理想成長曲線に近づく.この挙動は種晶のサイズには依存しない.理想成長曲線は,核発生が全く起こらない(つまり結晶総個数が不変の)場合の,製品結晶の体積平均径を与える曲線で,簡単な物質収支により次式で与えられる.=1+욦욦(2・140)したがって,実測のが理想成長曲線に一致するということは,二次核発生が全く起こらなくなっていることを意味している.このように,従来望ましくないとされてきた自然冷却法においても,種晶を十分添加すれば二次核発生を避けることができることになる.このような条件下では,結晶の個数変化を考える必要はなくなり,晶析過程は物質収支のみで記述でき,装置設計あるいは操作設計もいちだんと簡単になる.
