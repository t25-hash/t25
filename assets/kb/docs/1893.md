# 2・5・6 晶析装置(crystallizer)

> **章**: 第2章 分離操作
> **節**: 2・46 は、自然冷却法によるカリミョウバンのバッチ晶析結果で

ここでは、晶析装置に関する詳しい解説は行わず、典型的な装置を紹介するにとどめる。晶析装置に関する詳細は、文献を参照されたい。晶析装置は、大きく分けて連続装置とバッチ晶析装置に分けられる。バッチ晶析装置は、基本的にはかくはん機を備えたタンク式で、例えば図2・48のような形をしている。かくはん機にはいろいろなタイプがある。冷却によって過飽和状態を作る方式が多い。一方、連続式は、バッチ晶析装置とは違い、いくつかの種類がある。図2・49に示したのは、DTB(draft-tube-baffled)晶析装置である。名前のとおり内部にドラフトチューブとバッフルを備え、ドラフトチューブの下部にかくはん機を備えて、結晶懸濁液を循環させている。溶液は装置下部から供給され、結晶は供給液の上昇流により分級されある程度大きくなったもののみが製品として取出される。DTB 晶析装置のドラフトチューブとかくはん羽根を改良して、結晶の破損を少なくしたダブルプロペラ型装置もある。そのほか、外部に備えた循環ポンプで懸濁液を循環させ、装置内の結晶を流動層状に保って成長させるクリスタルオスロ型の装置もあるが、ここでは装置図は省略する。連続装置の場合は、冷却よりも蒸発濃縮によって過飽和状態を作り出す方式が多い。吸着(adsorption)は流体中の分子の濃度が接触している異なる相の界面において大きくなる現象であり、自然界でも広く観察される。吸着が工業的分離操作として利用されるようになったのは活性炭をはじめとする吸着剤の開発のおかげである。ほとんどの工業的吸着操作は気相-固相(気相吸着)あるいは液相-固相(液相吸着)で固体多孔性吸着剤を充てんした層(固定層fixed-bed、あるいは、充てん層packed-bed)により実施される。最近では超臨表2・5吸着・クロマトグラフィー充てん剤の特徴種類粒子径[m]比表面積[m/g]細孔径[nm]活性炭400~5000900~15001.5~3.0シリカゲル5~5000300~7004~1000合成高分子5~50004~1000天然高分子30~500―界流体相-固相の吸着も行われている。クロマトグラフィー(chromatography)と呼ばれる高度分離および分析操作も同様に固定層を用いる。クロマトグラフィーと固定層吸着は、別々の分野で研究開発されてきたが、設計方法は同一に取扱うことができる。またイオン交換も厳密には吸着現象ではないが、吸着操作と同様に設計されており便覧でも「吸着・イオン交換」と一括して扱っている場合が多い。吸着およびクロマトグラフィー分離操作は、水処理(脱イオン水や超純水製造)、環境問題に対応するための利用法(廃棄物処理)や、医薬品の分離などバルク分離から超精密分離まで幅広い分野で使用されている。これらの操作は(1)不要(不純)成分の除去と(2)有用成分の回収(濃縮、分離、精製)に大別できるが、どちらの操作でも吸着量が多いほうが望ましい。このためには吸着界面積を増やす必要があり、多孔性粒子が吸着剤として使用されている。代表的な吸着剤の比表面積(単位重量あるいは単位体積当たりの吸着面積)や細孔径を表2・5に示す。特にクロマトグラフィー分離用の充てん剤は多種多様な製品が開発されており、例えばシリカゲルでは、粒子径が数mから数mmまでのものが利用できる。実際の吸着剤充てん剤の構造は表2・5下図左のような均質構造のもの(ゲル型)と右のミクロ粒子の集合体とがある。また繊維状や膜状のものも開発されている。合成高分子は細孔径についても多種多様なものが製造可能である。また、細孔にイオン交換基をはじめとする官能基(リガンド)を導入したクロマトグラフィー充てん剤も市販されている。目的物質をできるだけ大量に吸着すると同時に、吸着が選択的であることが望ましい。また目的物質を分離回収するときは高い回収率の脱着も必要となる。クロマトグラフィーでは、選択性すなわち分離性能のために吸着量を犠牲にすることもある。除去が目的でも効率的な再生方法としての脱着は同じく重要である。吸着剤(あるいは充てん剤)はつねに改良・開発されてきており、多種多様な製品が使用できる。しかしながら、分離操作設計に必要な物性値を推算することは実際には難しく実験データが必要となる。ここでは固定層およびクロマトグラフィー操作について、その原理と設計方法について説明する。分離操作としては吸着相互作用の強さが物質ごとに異なることが分離原理となるが、吸着機構の詳細(物理化学的あるいは分子論的な説明)についてはここではふれない。また必ずしも吸着ではないイオン交換などの他の相互作用によるクロマトグラフィーについても全く同様に取扱うことができる。
