# 2・6・4 生産性

> **章**: 第2章 分離操作
> **節**: 2・55 で=0は直線平衡となり、=0.1とほとんど重なる。一

a. 等組成溶出クロマトグラフィーは原理的に不連続回分操作である。等組成溶出では繰返し注入により生産性を増加させることができる。図2・64に示すようにA とB,2成分の分離ではカラム出口での分離が達成されるようなタイミングで注入すればよい。2成分(A,B)分離は一定流速のとき試料負荷量を増加すると分離が悪くなるが分離時間が同じなので生産性が上がる。また、一定試料負荷量では流速を増加すると分離は悪くなるが分離時間は短くなるので生産性が上がる。生産性(カラムベッド体積1サイクル分離時間当たりの回収量)を次式で定義する。等組成ではたいてい分配係数が濃度に依存しない線形であるので、もに比例する。1サイクル分離時間は図2・64に示すA,B 2成分の溶出開始から終了までであり、流速により決定される。流速を増加させて同じ計算をする。このように各流速におけるの最大値をから求め流速に対してプロットすると図2・65のようになる。流速の増加とともに生産性は高くなる。しかしながらの増加率は徐々に低下し、最後には減少に転じる。この点が、このカラムの最大生産性となる。この流速以上では要求された分離が達成されない。図2・64 繰返し注入クロマトグラフィー等組成溶出における2成分分離の模式図ただし低線速度領域では短いカラムのほうが高い生産性となることがある。スケールアップ時には圧力損失が制約因子となる。すなわちカラムが長くなると最大圧力損失で生産性が規定され、必ずしも長いカラムのほうが有利とはならなくなり、最適カラム長さが存在することになる。ここで述べた計算方法は純度と回収率を必要条件としているが、さらに回収生産物の希釈率を加えることもできる。生産物の安定性からサイクル時間を規定した計算が必要なときもある。後述する連続分離にも使用可能できる。また通常操作と連続操作の比較のためにも有用である。b. 段階溶出段階溶出の生産性も式(2・188)で定義される。ただしサイクル時間は図2・51(D)に示すステップの全合計となる。一般には吸着過程の終了点(供給試料濃度の10%がカラム出口に到達した時点)での液量(図2・66)により式(2・188)は以下のようになる。またこの場合も最大操作圧力で最大生産性が決まる場合もある。
