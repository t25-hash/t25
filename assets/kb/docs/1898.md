# 2・6・5 連続クロマトグラフィー分離

> **章**: 第2章 分離操作
> **節**: 2・55 で=0は直線平衡となり,=0.1とほとんど重なる.一

クロマトグラフィーの連続化あるいは擬似連続化がいくつか考案されている.1960年代に回転クロマトグラフィーが考案され多くの論文が報告された.多成分分離が可能であるが原理的に不連続クロマトグラフィーの生産性を超えることはないことも明らかにされている.結局,回転カラムおよび通常カラムのどちらにおいても最大理論ベッド利用率は75%である.回転クロマト装置の利点は原料を一時的に溜めておくタンクが不要になるなどの連続化することに付随するものであると考えられる.一方,2成分分離を目的として開発された擬似連続クロマト分離である擬似移動層は糖の分離に使用されている.通常の不連続クロマトグラフィーのほう図2・65 生産性と流速の関係図2・66破過曲線の形状(左図)と生産性と流速の関係(右図)8―151化学の基礎が得策となる場合もあるので生産性についてはよく検討する必要がある.
