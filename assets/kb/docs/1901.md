c.圧力変換器（pressure transducer ) 圧力変換器は，流体
の圧力を電気量に変換するもので，近年急速に発展し多くの種類が
ある．
電気抵抗型は受圧膜（ダイアフラム）に張り付けた抵抗ひずみゲ
ージの電気抵抗の変化を検出するものである．最近は，シリコンの
単結晶板にダイアフラムとゲージ部を一体化した半導体圧力変換器
が多く用いられている．静電容量型は圧力によるダイアフラムの弾
性的変位を静電容量の変化として検出するもので，感度が高く小型
で固有振動数を高くすることが可能である．圧電型は，水晶などの
圧電素子の圧電効果による電圧を検出するものである．近年開発さ
れた圧電性高分子材料の薄膜（24～110m）は圧電効果が水晶の4
倍もあり，任意の形状に切取って物体表面に張り付けることが可能
で，有用性の高い圧力変換器である．ただ，この種の変換器は変動
圧力の測定に限られる．
電磁誘導型は，圧力により変位する物体の電磁誘導効果を利用す
るもので，微差圧の測定に利用されている．振動型は，弾性体に圧
力が作用するとその固有振動数が変化する現象を利用するもので，
圧力を応力として水晶などの振動ビームに加え，振動ビームの固有
振動数の変化を検出する．
今後の発展が期待される圧力変換器として，光ファイバを利用し
た圧力計がある．その原理の一例を図18･10に示す ．圧力が作用
する受圧膜にレーザ光を投射し，反射光を光検知器で電気信号に変
換するものである．この圧力変換器の受圧部は小型にすることが可
能で，体内に挿入して心臓内の血圧を測定するカテーテルにも利用
されている．固体表面全体の圧力分布を測定できる感圧導電ゴ
ム ，感圧塗料（PSP:pressure sensitive paint ) も開発されて
おり，今後の発展が期待される．
