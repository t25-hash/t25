# 18・4・2 圧力の測定

> **章**: 第18章 流体実験および計測
> **節**: 18・4 圧力計測（pressure measurement )

a.全圧 全圧は，流れ中に図18･11のような全圧管（total
pressure tube）を挿入し，その前面のよどみ点の圧力として計測
される．広く用いられている方法は，流れ方向に軸をもつ円筒形細
管の先端の圧力を測定するものであるが，球や円筒表面に圧力孔を
図18･14円すい静圧管
もつものも利用される．全圧管軸が流れ方向に傾きをもつ場合に
は，その測定値に誤差（圧力降下）が生じるので注意が必要であ
る ．
b.静圧 静圧計測では，流れを乱さないことが必要であり，
静圧は全圧より計測がはるかに困難である．壁面の静圧は圧力孔を
介して圧力計で測定されるが，圧力孔の形状によって測定値に大き
な誤差が生じる．特に，圧力孔径と深さ，圧力孔製作時のバリによ
る誤差に注意を要する ．
流れ中の静圧は静圧管（static pressure tube）により測定される
が，静圧管は図18･12のように流れに平行な細管からなり，その先
端を閉じ後方の管壁に静圧測定用の孔または細溝を設けて圧力を外
部に取出す．最も広く用いられている先端が半球の静圧管では，先
端の影響が圧力孔に及ばないようにするために，先端から圧力孔ま
での距離 は管直径 の10倍以上が必要で，圧力孔径は管径の
10分の1が一応の目安である．ただし， は下流の支柱の圧力上
昇効果を利用して10 以下にすることも可能である ．
静圧管では，流れに対する傾きの影響が全圧管よりも大きく現
れ，管の先端形状が測定誤差（圧力降下）に大きな影響を及ぼす．
図18･13に代表的な静圧管の傾き特性を示す．球形静圧管では，圧
力溝が先端から±104°の範囲に設けられ，測定値が静圧値と一致
するように工夫されており，広い角度範囲で誤差が小さい ．し
かし，この静圧管の適用は二次元流れ場に限られる．先端形状が円
すい形で，管径に対して圧力孔径を比較的大きくした静圧管（図
18･14）は，従来の静圧管と比較して広い角度範囲で圧力降下が小
さい ．
c.多孔ピトー管 流れの全圧と静圧を同時に測定するものを
ピトー静圧管［単にピトー管（Pitot tube）と呼ぶことが多い］と
いい，流速計として広く使用されている．これは，前述の全圧管と
静圧管を組合わせたもので，一例を図18･15に示す．流れ方向も測
定するものとして，図18･16のような多孔ピトー管がある．これら
の多孔ピトー管では流れ方向，全圧，静圧を求めることができる．
また，7孔 ，13孔 のものもある．

は全圧・静圧管の圧力孔の圧力を圧力変換器受圧面に導いて電気信
号として測定される．
