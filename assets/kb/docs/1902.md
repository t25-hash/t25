# 2・6・9 まとめ

> **章**: 第2章 分離操作
> **節**: 2・55 で=0は直線平衡となり、=0.1とほとんど重なる。一

省エネルギー高度分離操作としての固定層吸着・クロマトグラフィー操作は水処理などのバルク分離から精密高度分離へとその用途は広がっている。界面における現象は複雑であり吸着あるいは相互作用の本質はいまだに不明な点が多いが、実用的な見地からは本文で説明したようによく計画された実験を実施することにより装置設計や運転条件の設定が可能である。図2・70は設計の流れ図である。分散を表す因子を総括した変数と、吸着平衡により簡単な設計ができる。移動相が液体以外のときは液の部分を気体あるいは超臨界、亜臨界流体と読換えればよい。表2・7にまとめたモデルによる数値計算も計算機の能力が向上した現在ではそれほど困難ではないが、まず図2・70に従ってプロセスを考えることが重要である。本文中にも説明したように、ほとんどの吸着・クロマトグラフィー操作は粒子内(固定相)拡散に分離性能が支配されている。それゆえクロマトグラフィーにおいては微粒子充てん剤の開発が進み、現在では分析クロマトグラフィーでは数m、分取クロマトグラフィーでは数十mの粒子が使用されている。しかしながら大規模分離プロセスでは、圧力損失の観点から依然として数百mの粒子が主流である。充てん剤メーカも粒子径を小さくするとともに粒子径分布が狭い製品を開発しており、このような充てん剤により装置の小型化と高性能化が可能になると期待される。吸着操作については便覧など有用な文献が多数ある。クロマトグラフィーについても同様であるが、両者を統一的に扱っている参考書は多くないので本文中にも指摘したように共通する現象を異なる用語で説明していることを理解しておく必要がある。実際の装置設計・操作では充てん剤カラム容器構造や充てん方法も重要となる。これらについては、設計実例とともに各種便覧を参考にしてほしい。
