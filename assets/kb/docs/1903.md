# 2・7・1 分離膜の選択性の発現と膜の機能

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

膜を透過する速度が成分ごとに異なる場合,膜に供給される混合物を分離することができる.膜の選択透過性はさまざまなメカニズムにより発現するが,おもに二つの透過機構,(a)ふるい機構,および,(b)溶解拡散機構に分類される.図2・71に示すように,ふるい機構は,膜の細孔よりも大きな粒子や分子は膜を透過できず,小さな分子のみが透過するとするものであり,多孔性膜に適用される.一方,溶解拡散機構では,膜内に透過分子が溶解し,膜中を拡散し,膜透過側で脱離するプロセスを経ることで,膜透過すると考えるモデルであり,高分子あるいは無機材質の無孔性膜に用いられる.ほとんどの分離膜の透過機構は,両モデルあるいはそれらの複合モデルで説明される.分離膜は二つの相を隔てることが可能なため,図2・72に示すように,分離膜へ供給した原料は,膜透過流と非透過流(あるいは未透過)に分かれる.膜透過側には膜を優先的に透過する成分が濃縮しており,一方,非透過流には膜を透過しにくい成分が濃縮されることになる.膜分離の機能は,透過,濃縮,および分画に分けられる.海水脱塩や超純水製造のように,膜を透過しやすい成分である水が目的成分となる場合や,果汁の濃縮などのように非透過成分が目的成分となる場合もある.分画とは,溶質混合物を透過流と非透(a) 分子ふるい(b) 溶解拡散図2・71 膜分離機構の概略相供給透過分離法分離駆動力透過成分/非透過成分応用例液相液相逆浸透圧力差(1~8MPa)水/すべての溶解成分海水脱塩,ジュースの濃縮透析濃度差イオンおよび低分子有機物/高分子有機物血液透析,酸・アルカリの回収電気透析電位差イオン/中性物質,高分子物質海水濃縮,脱塩気相気相ガス分離圧力差(0.1~1MPa)溶解性・拡散性の高い成分/低い成分空気分離,水素回収,天然ガスからの脱二酸化炭素圧力範囲は,目安として示す.8―153化学の基礎過流とに分け,分離を行う操作である.膜分離プロセスは,相変化を伴わず省エネルギーであること,コンパクトな分離装置となること,運転が容易であることなどの特徴を有するため,産業界で徐々に応用されつつあり,便覧や成書に膜分離プロセスがまとめられている.
