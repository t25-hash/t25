# 2・7・2 膜分離プロセスの種類

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

分離を行うためにはエネルギー(分離駆動力)を与え続ける必要がある。その分離駆動力の種類、および膜供給側、および透過側の相に応じて、膜分離プロセスを表2・8に示すように分類することができる。それぞれの分離目的に適した膜や分離手法が、実際の分離プロセスに選定されることになる。膜両側がともに液相の場合、分離駆動力を圧力差とするろ過法においては、分離対象とする溶質の大きさにより、精密ろ過(MF: microfiltration,100nm以上の粒子を阻止)、限外ろ過(UF: ultrafiltration,2~100nm)、ナノろ過(NF: nanofiltration,1~2nm)、逆浸透(RO: reverse os-mosis,<1nm)に分類分けされている。逆浸透法を除き、いずれの透過機構もふるい機構と考えられており、基本的には小さな分子(一般には溶媒)が優先透過する。細孔径が小さくなるに従い付与圧も増大することから、除去あるいは濃縮したい溶質のサイズによって、適切な分離プロセスを選択する必要がある。溶質成分の膜透過は透析(dialysis)と呼ばれ、分離駆動力が溶質の濃度差の場合が透析法であり、電位差により電解質溶質を分離する場合は電気透析法(ED : electrodialysis)と呼ばれる。両相とも気相の場合がガス分離(gas separation)あるいは蒸気透過(VP: vapor permeation)であり、分離駆動力は圧力差(分圧差)になる。室温では液相である成分を分離対象とする場合は蒸気透過法と呼ばれ、一般には減圧あるいは同伴ガス下で操作されることになる。供給原料が液相、透過側が気相である場合が浸透気化(PV : pervaporation)であり、透過側を減圧あるいはスイープガスを流すことで気相状態が保たれる。この場合、膜透過成分はいったん蒸発するため気液平衡が分離特性に寄与することになり、また、相変化を伴うため必ずしも省エネルギー分離とは限らないが、膜がいわゆる第3成分として気液平衡を大きくずらすことが可能となり、その結果として省エネルギー的な分離となりうる。
