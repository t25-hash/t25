# 2・7・3 分離膜の種類

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

分離膜はさまざまな見地から分類可能であり、膜材料からは有機材料と無機材料、細孔の有無から多孔性膜と無孔性膜に分類される。気体分離膜や逆浸透膜は、一般的には無孔膜として分類されている。膜構造の面からは、膜透過方向に均質である対称膜(膜の表から見ても裏から見ても同じ構造)および非対称膜に分類され、細孔径の比較的大きな精密濾過膜では対称膜が実用化されている。し図2・72 膜分離のフローの概略図表2・8膜分離プロセスの分類と応用分野精密ろ過圧力差(10~100kPa)水および溶解成分/懸濁粒子(0.1~10 m)懸濁液の清澄化、バクテリアの除去、浄水処理限外ろ過圧力差(0.1~1MPa)水および塩/コロイド、高分子物質(2~100nm)コロイド溶液の清澄化、バクテリアの除去、タンパク質の濃縮ナノろ過圧力差(0.1~2MPa)水および塩/低分有機物(1~2nm)、NaCl低分子有機物の除去、脱塩(NaCl)しながらの有機物濃縮液相気相浸透気化圧力差(透過側は減圧あるいはスイープガス)溶解性・拡散性の高い成分/低い成分アルコール水溶液など共沸混合物における脱水(水が透過成分)蒸気透過圧力差(透過側は減圧あるいはスイープガス)溶解性・拡散性の高い成分/低い成分アルコール水溶液など共沸混合物の脱水(水が透過成分)、空気中の有機蒸気の分離・回収8―154電気・電子の基礎、化学の基礎かしながら、細孔径が小さくなるにつれて透過抵抗が大きくなるため、分離機能を有する緻密な層を高い強度を有する多孔質支持層の上に形成させた非対称膜が作製されている。図2・73に示す相転換膜は、酢酸セルロースやポリアミドなどの高分子を溶媒に溶かし、厚み50~500 m程度にキャストし、溶媒を蒸発させ表面に緻密な層を形成する。その後、非溶媒中に浸漬することで、高分子相と溶媒相で相分離を起こし、表面緻密層を支える多孔層が形成される。表面の緻密な高分子層が分離機能を有する層であり、活性層とも呼ばれる。異なる素材を用いて製膜した場合は複合膜と呼ばれ、その典型的な構造は図2・74に示すように、不織布の上に限外ろ過膜程度の多孔材料を支持層(中間層)とし、さらにその上の超薄膜層からなる。界面重合法やコーティング法などの手法を用いることで、分離機能性を有する超薄膜層は0.01~1 m程度の厚みで形成される。現在市販されている逆浸透およびナノろ過膜のほとんどは、複合膜である。膜支持体となる限外ろ過膜としては、ポリスルホンが一般に用いられている。
