# 18・5・3 レーザ流速計

> **章**: 第18章 流体実験および計測
> **節**: 18・5 速度計測

可干渉なレーザ光を流れとともに移動する数ミクロンの微小粒子
に照射し，それから散乱された光と入射光との周波数シフトから粒
子の速度を求める流速計がレーザ流速計（laser Doppler veloci-
meter ）である．これはレーザ光源，光学系，散乱光検出器および
信号処理装置からなる．以下に広く用いられている干渉じま方式に
ついて示す．光学系において2本の光線が交差する空間において
は，明暗を示すフリンジパターン（fringe pattern）が形成される．
