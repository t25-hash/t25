# 18・5・4 PTV法（粒子追跡法)

> **章**: 第18章 流体実験および計測
> **節**: 18・5 速度計測

Particle- tracking velocimetryの頭文字をとってPTVと呼ぶ．
図18･22レーザ流速計 図18･24粒子分布相関法の計測原理
この方法は粒子の軌跡をマニュアルで追跡したのが始まりで ，
粒子の移動や粒子配列パターンから流速を求める方法があ
る ．流速計システムは粒子群の可視化を行う可視化光学系と
可視化画像を解析する画像解析システムから構成される．トレーサ
粒子の分布パターンの類似性から流速を計測する粒子画像流速計測
法（PIV:particle image velocimetry）における粒子像の密度が
十分小さい場合に対応する．画像解析システムにとって良好な可視
化画像を得ることが重要であり，また，図18･23に示すように，高
出ダブルパルスYAGレーザによりストロボ的にレーザ光を照明に
使うなど，レーザパルス周波数と高解像度CCDカメラのフレーム
レートによって時間分解能を向上させる方法がある ．粒子の散
乱光強度が観察方向に対して十分なものが望まれる．粒子径が20
m程度の微小粒子を用いる．粒子画像の取得後は，(１)粒子像の
認識，(２)粒子の番号付け，(３)追跡，(４)速度ベクトルの算出を
行う．短い時間間隔で取得された可視化画像において粒子像の移動
を検出する．四時刻追跡法や粒子分布相関法などによって粒子像の
移動先をさがす．四時刻追跡法とは，4枚の画像を用いて，粒子の
位置が最も直線に近い組合せが正しい移動経路である確率が高いと
いう考え方に基づいている．一方，流れの中で粒子群は隣り合う粒
子は類似な動きをするので，粒子群のパターンは大きく変わらない
ことを利用した方法が図18･24に示す分布相関法（二値化相関法）
や粒子群の相対的位置変化を考慮したばねモデル法と呼ばれる方法
がある．また，輝度差累積法に代表されるような可視化画像の濃淡
パターンの相互相関係数から画像の移動先を求め，流速ベクトルを
決める相関法と呼ばれる方法がある．解析対象が可視化画像の輝度
分布であればなんでもよく，煙や染料を流した可視化画像でも解析
が可能である．種々の解析手法に対して，標準的な評価基準である
評価法 が提案されている．その他，画像ノイズ除去後にフィル
タリング処理など，画像改善法が提案されている．一方，三次元流
れ場の計測は，ステレオ写真法を応用した3D- PTV がある．粒
子密度が低いために，一般に速度ベクトルは少なく空間解像度が低
くならざるを得ない．さらに，ホログラムを使って三次元粒子像を
取得して三次元速度場計測に応用するホログラフィックPIV
法 も開発されている．微細な粒子をホログラムに高密度で記
録する．PIVとレーザ誘起蛍光法を用いることによって，流速と
温度や密度の同時計測を行うことができる．流体中の蛍光物質の濃
度を一様に保てば，蛍光強度は温度の関数になり，流体の温度を一
図18･23 PTV
定に保てば蛍光強度は濃度の関数になる．速度計測のために粒子と
蛍光物質を流れ場に混入し，これらを光学フィルタにより分別して
2種類の画像として取得する．
