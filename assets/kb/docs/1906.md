# 2・7・4 分離膜モジュール

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

実際の工業プロセスで分離膜を用いるためには,耐圧容器に膜を組込む必要があり,膜モジュールと呼ばれる.実用化されている平膜を用いる膜モジュールには,平膜を耐圧板の上に重ね上げた平面膜型,のり巻き状に巻いて膜充てん密度を上げたスパイラル型モジュールがある.管状の膜を用いる管状膜(tubular membrane)(内径1cm以上),中空糸膜(hollow fiber membrane)(おおむね外径0.5mm以下),および中間サイズのキャピラリー膜(capil-lary membrane)モジュールがある.いずれのモジュールも,供給流れの混合を促進し,濃度分極現象を小さくする工夫がなされている.一般に,モジュール体積当たりの膜面積は,中空糸(16000~30000mm)>キャピラリー(6000~1200mm),スパイラル(300~1000mm)>平面(100~400mm),>管状(300mm以下)の順番に大きいが,一方,耐汚染性や洗浄性はこの順番に劣る.したがって,供給流体の性状に従って,適正な膜モジュールを選択する必要がある.実際の使用に当たっては,膜モジュールの使用条件(耐圧,温度,pH など)に十分注意する.また,微量成分(例えば,界面活性剤など)が膜性能に大きな影響を与えることもあるので,実プロセスに応用する場合は,実液あるいはガスを用いて事前にテストする必要がある.a. 平面膜モジュール平面膜モジュールの構造は,フィルタプレス型ろ過器やプレート型熱交換器に似ている.円盤状のプレートとフレーム(板)の間に,円形の膜が挟まれたものが多層に重ねられており,プレートアンドフレーム型とも呼ばれる.透単位体積当たりの膜面積は,約100~400mmである.b. スパイラルモジュール(spiral module)2枚の平膜を活性層を外にして,封筒状に接着し,内側に流路材(スペーサ)を入れ,開放された一端を集水管(孔が多数あいた中心パイプ)に接続する.供給水を流すスペーサとともに集水管にのり巻き状に巻き込んでスパイラル型の膜エレメントとし,耐圧容器の中に設置される.図2・75に示す例では,供給水は集水管と平行方向にスペーサ部分を流れる.逆浸透,ナノろ過のほとんどは,このモジュール型式を採用している.c. 管型モジュール内径10~20mmの支持管の内側に活性層を有する管状膜を複数本ヘッダで接続したものをモジュールとする.膜のスポンジ洗浄が可能なことから,濁質の供給水の処理に適する.d. 中空糸モジュール(hollow fiber module)外径40~200m,肉厚10~50 mの中空糸を,数十万本束ねて圧力容器に収めたものである.中空糸モジュールは,膜自体の耐圧性があるため支持体が不要であり,中空糸の径を小さくすることによって膜充てん密度を16000~30000mmまで大きくできるため,極めてコンパクトなモジュールとなる.膜充てん密度が高いが汚れやすさと洗浄性に問題があるため,前処理を十分施した海水淡水化プロセスや気体分離に用いられるケースが多い.e. キャピラリーモジュール外径0.5~5mm程度のチューブ状の膜をキャピラリー膜と呼び,モジュール化したものがキャピ図2・73 相転換膜の構造図2・74 複合膜の構造図2・75 スパイラルモジュールの構造図2・76キャピラリーモジュールの構造ラリーモジュールである.図2・76に示すように,多管式熱交換器のような形状をしており,限外ろ過モジュールで多用される.f. その他のモジュール無機膜に特有のモジュール形式としてモノリス型モジュールがある.レンコン状に成型されており,レンコンの中空部分が原料供給部であり,透過液あるいはガスは多孔性支持体の中を透過する.また,浸漬型膜モジュールと呼ばれ,耐圧容器を用いず膜エレメントを槽内に直接浸漬し,水頭圧や膜透過側を減圧することで,膜間に差圧を付与するモジュール形式が,廃水処理や浄水処理設備において実用化されている.
