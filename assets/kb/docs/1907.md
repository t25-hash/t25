# 2・7・5 逆浸透、ナノろ過、限外ろ過の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

a. 膜移動現象分離膜の性能評価は、膜を透過する流量と選択性が指標となる。逆浸透、ナノろ過、限外ろ過膜の透過基礎式は、非平衡熱力学に基づく現象論方程式による次式が用いられる。は膜供給側と透過側との平均濃度であり、阻止率の小さい場合に有効である。一方、逆浸透膜は式(2・202)(2・203)で溶質反発係数σの大きい場合(σ1)に相当し、次式で表される溶解拡散モデルとなる。実用化されている逆浸透膜は、おおむねこの条件を満たす。水溶液の浸透圧は、便覧などに与えられている。あるいは、ウィルソン式などを用い活量を求め、式(2・207)に代入することで浸透圧を算出することができる。b. 分画分子量(molecular weight cut-off)ナノろ過膜、限外ろ過膜の溶質阻止性能は、分画分子量によって表される。溶質の分子量に対して阻止率をプロットし(分画分子量曲線と呼ばれる)、図2・77に示すように阻止率が90%となる分子量を分画分子量(MWCO: molecular weight cut-off)と呼ぶ。溶質としては、糖類(グルコース、ショ糖、ラフィノースなど)、種々の分子量のポリエチレングリコール、デキストラン、タンパク質などが用いられる。限外ろ過のMWCOは1000~1000000程度、ナノろ過膜はMWCO=200~1000程度である。ただし、溶質の阻止率は操作条件(圧力、流量など)や溶質の種類に依存しているため、分画分子量の値には注意が必要である。物質移動係数の推算式として、流動様式に応じて種々の相関式が与えられている。限外ろ過法で分離対象とする高分子溶質の物質移動係数は小さいため、濃度分極現象が顕著に現れ、膜表面でゲル層の形成や浸透圧により透過流束が純水と比べて大きく低下する。一方、物質移動係数を大きくすることで、見掛けの阻止率は真の阻止率に漸近するため、膜モジュールの供給流路は乱流を促進し、物質移動係数を大きくするように工夫されている。濃度分極を考慮する場合は、式(2・208)を連立させて数値的に解く。モジュールを用い溶液の処理を行うとき、溶媒が膜透過し溶質成分は阻止されるために、供給側(非透過側)の溶質濃度は徐々に増加し、浸透圧が上昇するため体積透過流束は低下することになる。図2・77 分画分子量曲線図2・78 濃度分極の概念図8―156電気・電子の基礎、化学の基礎
