# 18・6・1 差圧式流量計（diﬀerential pressure flow meter ）

> **章**: 第18章 流体実験および計測
> **節**: 18・6 流量計測

差圧式流量計は，大まかに絞り流量計と層流流量計に分類され
る．絞り流量計は，オリフィス，ベンチュリ，ノズルなどに分類さ
れ，ベルヌーイの式を基本にしていて，管路途中の絞り部の上流・
下流の圧力差から流量を算出する流量計である．絞り流量計を使用
する場合，その形状とレイノルズ数に応じた補正係数（流出係数）
が必要である．オリフィスは絞り流量計の中で最も多く使用されて
いて，その流出係数はJISで規定された設置条件や形状を満足し
ていれば，JISで与えられた値を用いて高い精度の計測が可能であ
る ．オリフィスは，プラントなどにおける大流量計測において
主要な流量計であり，さまざまな設置条件や流れ状態などがもたら

す誤差要因，その補正方法などが研究されている ．
ハーゲン・ポアズイユの式を利用して細管の上流・下流の圧力差
から流量を算出する層流流量計は，小流量用に広く使用されてい
る．また，細管でなくて環状のすきまを使用し，二つの高精度絶対
圧力計と温度計を本体に内蔵した質量流量を計測できる層流流量計
が開発された．この流量計は，外部の温度変動に鈍感なように熱容
量の大きい本体をもち，このタイプの流量計の弱点を逃れている．
精度の高い校正をすれば，0.1 %程度の精度と同程度の再現性をも
つ．
