# 2・7・6 精密ろ過(microfiltration)の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

液相系ろ過でも精密ろ過では,分離対象とするのはおもに懸濁粒子のため,膜輸送のモデル式は2・7・5項の逆浸透や限外ろ過法とは異なる.膜(構造や細孔径など)および懸濁粒子(サイズ,濃度など)によるが,粒子は膜表面に堆積層(ケーク層)を形成,あるいは,細孔内を閉塞しながら,ろ過されることとなる.以下の三とおりのろ過方式が用いられる.(1) デッドエンドろ過(全ろ過):膜に供給されたすべての溶液を膜透過させるものであり,図2・72の非透過に相当する流れがないケースに相当する.この場合は懸濁粒子がすべて膜で補足されるために,処理量の増大とともに,ケーク層のろ過抵抗が増大し,図2・79に示すように透過流束は零に漸近する.(2) クロスフローろ過(十字流ろ過):膜内部の閉塞あるいはケーク層形成のために透過流束は徐々に低下するものの,クロスフローでは膜面方向の流れ(非透過流)が存在するためにケーク層をはぎ取ることが可能となり,やがて定常透過流束に達する.(3) 逆洗を伴うろ過:クロスフローろ過において,間欠的に透過側から供給側に透過液あるいは加圧ガスを透過させることで,ケーク層などのファウリング物質をはく離しながらろ過を行うものであり,高い透過流束を維持できる.
