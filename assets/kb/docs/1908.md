# 18・6・2 熱式流量計（thermal type flow meter ）

> **章**: 第18章 流体実験および計測
> **節**: 18・6 流量計測

さまざまな方式が提案されているが ，基本的にはセンサが流
れに直接接しているタイプと管路の外部にコイルを巻いて加熱する
タイプとがある．内部にセンサが置かれたタイプのものは，応答性
は速いが測定対象流体を選ぶ．また，汚れなどによる感度劣化が問
題になる．半導体産業などで使用されている熱式質量流量計は，セ
ンサ管路の外側にコイルを巻いたタイプである ．この流量計は，
センサ管路とバイパス部とから構成されている．センサ管路には内
径0.2～0.3mmの細管のまわりに加熱と温度計測を兼ねた一対の
コイルが巻かれており，流体が流れることによって生じる温度分布
の対称性の崩れからセンサ部を流れる流量を求める．バイパス部に
はセンサ部の流量に比例する形で毛細管等の抵抗体が挿入され，全
体として必要な流量が得られるようになっている．流量 は，セ
ンサ間の温度差 から次の式を用いて算出される．
＝
ここで， はセンサ部分とバイパス部分との分流比率など流量計
に依存した補正係数で，校正によって決定される．気体の定圧比熱
を含むために，計測される質量流量 は気体によって異なる．
そのため，窒素の計測結果を基に任意の気体の流量を推定する変換
係数（CF）が提案されているが，測定精度を必要とする場合は，
測定対象ガスによる直接校正が不可欠である．測定精度は，校正方
法にもよるが2%程度，再現性は±0.5%である．
