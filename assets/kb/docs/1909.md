# 2・7・7 透析法・電気透析の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

a. 透析法(dialysis)溶質の濃度差に基づく溶質の拡散によって分離を行うものであり、圧力などのエネルギーは必要ないが、移動速度が小さいため工業的には汎用されていない。血液透析法は、腎不全患者の尿中に蓄積している尿毒症原因物質、塩、水などを体外に排出する人工腎臓として、広く実用化されている。膜両側に存在する濃度境膜および膜自体が、溶質の膜移動の抵抗となる。血液透析では、一般に中空糸モジュールが用いられており、血図2・79精密ろ過における透過流束の経時変化図2・80電気透析の概念図(A:陰イオン交換膜、C:陽イオン交換膜)液は中空糸内を、透析液は中空糸外を流れる。境膜における移動抵抗は、膜抵抗と比べて無視できないため、モジュール性能の向上のためには、膜の改良のみならず、モジュール内外での流動状態の改良が不可欠である。b. 電気透析法(electrodialysis)イオン交換膜を介して電圧を付与し、電解質を移動させる場合が電気透析と呼ばれる。陽イオン交換膜(カチオン交換膜)はスルホン酸基を有する高分子電解質によって作製され、負荷電を有するため陰イオンは膜に入り込めない。一方、陰イオン交換膜に対しては、陽イオンは膜透過できない。これらのイオン交換膜を交互に並べると、図2・80に示すように濃縮液と希釈液を交互に得ることができる。海水脱塩や製塩における濃縮工程に用いられている。
