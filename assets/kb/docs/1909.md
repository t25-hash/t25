# 18・6・3 渦式流量計（Karman-vortex type flow meter ）

> **章**: 第18章 流体実験および計測
> **節**: 18・6 流量計測

渦式流量計は，管路内に置かれた渦発生体から生じるカルマン渦
のストローハル数を利用するため，レイノルズ数が同じならば流体
の種類によらないという利点をもち，一般的にはストローハル数が
ほぼ一定になる＝2×10以上の範囲で使用される．ストローハ
ル数の定義式から得られた平均流速に管路の断面積を掛けることに
よって，そのときの体積流量が計算できる．
広い流量範囲でストローハル数を一定にするため，外部から強制
的に擾乱を与えて渦の安定を図る など，多くの研究がある．渦
発生体である柱状物体と管路壁の結合部分のすきまが，柱状物体と
管壁近傍に発生する馬てい形渦の挙動に影響し，カルマン渦の安定
性に大きく影響する ．この実験結果を基に規格化された標準渦
発生体をもつ標準渦式流量計がJISで規定されている ．この標
準渦式流量計の校正係数の不確かさは，レイノルズ数が10～
2.5×10の範囲で±0.2 %である．標準渦式流量計の校正係数は管
路直径が決まれば計算から求めることができ，実流校正が不可能な
大流量の流れや水素などの危険な気体の計測などへの適用が期待さ
れる．
