# 2・7・8 ガス分離(gas separation)の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

透過係数の大小によって、膜透過する速度が異なるため、透過側で分離されたガスを得ることが可能となる。膜厚が不明な場合はを一つの係数として取扱い、一般的には透過率と呼ばれるが、透過係数とされる場合もあるので注意が必要である。透過係数は、圧力依存性や混合ガスの場合は濃度依存性を示すケースもあり、必ずしも定数ではない。ポリイミド膜の透過係数の一例を図2・81に示す。温度の上昇とともにいずれの気体も透過係数が増大する活性化拡散の傾向を示し、HCH透過係数比が100~200と非常に高い値を示す。ポリイミド膜は、耐熱性~150°C、耐圧15MPa、良好な耐薬品性を示し、石油化学プラントのオフガスからの水素精製などに使用されて図2・81 ポリイミド膜の気体透過係数の温度依存性図2・82 エタノール/水混合溶液の分離性いる。実用化されている気体分離膜は中空糸膜モジュールであり、外径10~30cm、長さ2~4mである。b. 混合ガスの分離混合ガスを分離膜に供給すると、成分の透過側組成は膜透過した成分の流量比で与えられる。式(2・215)より、全圧比が無限に小さくなる場合(供給側を無限に加圧、あるいは、透過側を減圧)において、混合系でのαは理想分離係数α(透過係数比)に漸近する。
