# 2・7・8 ガス分離(gas separation)の膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

a. 気体の膜移動現象気体透過流束욡mol/m・sの最も簡単な輸送式は,膜供給および透過側における分圧差に比例するとするものであり,気体透過係数욡mol・m/m・s・Paおよび膜厚mを用いて,욡=욡욠욡-욆욡(2・212)と表される.ただし,욠,욡は供給側の全圧およびモル分率を表し,욆,욡は透過側での値を示す.透過係数の大小によって,膜透過する速度が異なるため,透過側で分離されたガスを得ることが可能となる.膜厚が不明な場合は욡を一つの係数として取扱い,一般的には透過率と呼ばれるが,透過係数とされる場合もあるので注意が必要である.透過係数は,圧力依存性や混合ガスの場合は濃度依存性を示すケースもあり,必ずしも定数ではない.しかしながら,そのような依存性が小さい場合は,気体選択性の指標となり透過係数比α=욡욢(厳密には,単成分比ということで,理想分離係数α)と呼ばれる.ポリイミド膜の透過係数の一例を図2・81に示す.温度の上昇とともにいずれの気体も透過係数が増大する活性化拡散の傾向を示し,HCH透過係数比が100~200と非常に高い値を示す.ポリイミド膜は,耐熱性~150°C,耐圧15MPa,良好な耐薬品性を示し,石油化学プラントのオフガスからの水素精製などに使用されて図2・81 ポリイミド膜の気体透過係数の温度依存性図2・82 エタノール/水混合溶液の分離性いる.実用化されている気体分離膜は中空糸膜モジュールであり,外径10~30cm,長さ2~4mである.b. 混合ガスの分離混合ガスを分離膜に供給すると,成分の透過側組成욡は膜透過した成分の流量比で与えられる.욡=욡∑욡(2・213)式(2・212)(2・213)を連立して解くことにより,ある条件での透過側組成욡を決定することができる.2成分系の場合については,욀욁=욀욁욠욀-욆욀욠욁-욆욁=α욀-욥욀욁-욥욁(2・214)となり,透過と供給側の全圧比욥=욆욠,および,透過係数比α=욀욁をパラメータとして,透過組成を求めることができる.また,混合ガスの分離性の指標としては,液相系の場合に用いられる阻止率は通常用いられず,次式で定義される分離係数αα=욀욁욀욁=α1-욥욀욀1-욥욁욁(2成分系の場合)(2・215)が用いられる.式(2・215)より,全圧比욥が無限に小さくなる場合(供給側を無限に加圧,あるいは,透過側を減圧)において,混合系でのαは理想分離係数α(透過係数比)に漸近する.
