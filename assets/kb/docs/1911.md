# 18・6・5 音速ノズル（critical nozzle)

> **章**: 第18章 流体実験および計測
> **節**: 18・6 流量計測

気体の流れが管路に設けられた絞り部で臨界になると，この管を
流れる流量は下流側の流れ場に影響されずに一定になり，そのとき
の流量は上流側条件（圧力，温度）にのみ依存する．この原理を利
用したのが音速ノズル流量計である．ISOではトロイダルスロー
ト（toroidal throat ）とシリンドリカルスロート（cylindrical
throat ）の2種類のノズル形状が規定されている ．質量流量
は，校正で決定される流出係数 を用いて

体定数である． は臨界係数と呼ばれ，理想気体の場合は比熱比
だけの関数である．
境界層が層流で実在気体効果が大きくない場合には，理論的に取
扱われるノズル形状と正確に同じ形状をもつ超精密加工音速ノズル
を使用することによって，理論的に流出係数を±0.05%以下で決定
できる ．レイノルズ数が大きくなり，境界層が遷移状態や乱流
になる場合の理論的取扱いはできていない．気体温度の流出係数へ
の影響 ，ガス種による流出係数の変化 などの研究も行われて
いる．また，0.1mm以下のスロート直径をもつ微小音速ノズルに
よって0.01g/ min以下の気体の流量計測も可能になっている．
