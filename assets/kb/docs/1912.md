# 2・7・10 浸透気化(pervaporation)による膜プロセス

> **章**: 第2章 分離操作
> **節**: 2・7 膜分離(membrane separation)

浸透気化膜としては高分子膜とゼオライト膜が実用化されている。図2・82には、実用化されている高分子膜(PVA 膜)とゼオライト膜(A 型)を用いた場合の、エタノール/水系における分離性能を示す。気液平衡と比べて、非常に高い水選択性を示し、アルコール水溶液の共沸点付近での脱水プロセスで実用化されている。一方、気液平衡以上のエタノール選択性を示す膜としては、シリコンゴム膜やゼオライト膜(シリカライト)が報告されているが、実用化はあまり進んでいない。文献(1) 化学工学会編、化学工学便覧(改訂4版)、(1978)、969-1304、丸善。(2) Belter, P.A., ほか2名、 Bioseparations; Downstream Process-ing for Biotechnology、(1988)、6-8,A Wiley-Interscience Pub.(3) Seko,N.、ほか7名、Nucl.
