# 1・2・1 物理量の次元(dimension)

> **章**: 第1章 単位系と次元
> **節**: 1・2 次元と単位系

任意の物理量は長さ、質量、時間など少数の互いに独立とみなされた物理量を基本として、これらの組合せによって表される。例えば、面積および体積の次元は、長さの次元Lを用いてそれぞれL,Lで表され、速度は長さと時間の次元L,T を用いて、LTとなる。ただし、ζ,α,β,γ,... は定数である。このとき、指数α,β,γ,... は有理数(後述のSIでは、これらは整数になるように選ばれている)であり、これらを物理量の基本量,... に関する次元指数(ディメンジョン指数、dimension exponent)と呼ぶ。基本量の選び方を決めれば任意の物理量の次元は一義的に定まり、同種の物理量に対して次元は同じとなる。しかし同一の次元であっても物理量が同種であるとは限らない。例えば仕事(エネルギー)と力のモーメントは同じ次元であるが異なる意味をもつ物理量である。同一の物理量であっても基本量の選び方が変われば次元は当然異なる。数種の基本量に対してそれぞれ特定の単位を定義し、次元の概念を用いてすべての物理量に対する単位を誘導することによって単位系を構成することができる。SIは基本量として、長さ、質量、時9 ―1間以外に温度、電流、物質量、光度を用い、他の単位はすべてこれらの組合せとして誘導した一貫性のある単位系である。工学単位系は質量の代わりに力を基本量としているため、同一物理量に対してSIとは異なる次元および単位となることが多い。次元の考え方を利用して異なる単位系の間の換算を容易に行うことができる(I-1・数種の物理量の間の相互関係を示す方程式が和の形で表されている場合、各項は同一の物理量であるから同一の次元でなければならない。このことから各項のもつ物理的意味を明確にすることができ、また次元が一致するかどうか点検することによって方程式の誤りを正すことができる。同種の物理量の比をとれば基本量の次元指数はすべて零となり次元は1となる。このような量を無次元量(non-dimensional quan-tity)という。無次元量を用いれば対象の大きさ、実験条件などの具体的な数値とは無関係に、現象を支配する本質的な要因を抽出できることが多い。したがって、現象が複雑すぎて理論的な見通しを得難い場合に適切な無次元量を見出すことは現象の本質を把握するうえで重要である。この目的のために関連する物理量をすべて洗い出し、これらを組合わせて無次元量を見つける手法があり、次元解析(dimensional analysis)という(α4編「流体工学」、β5編「計測工学」、β6編「制御システム」参照)。なお、平面角、立体角、相対密度(比重)などは必ずしもこのような性質をもたないが、次元的には無次元量として扱われる。
