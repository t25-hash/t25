# 1・2・2 一貫性のある単位系(coherent unit-system)

> **章**: 第1章 単位系と次元
> **節**: 2・3 項参照)。

いくつかの物理量を基本量として選んだ単位系において、ある物理量は基本量,,,... を用いて、=ζ...と表現できるが、そのときのは、単位<>と、その単位で表した物理量の大きさを示す数値の積として、数式的に、=<>のように表現できる。このとき、単位<>は基本量に対する基準量(単位)<>,<>,<>,... の関数として、<>=<><><>...で表すことができる。任意の単位<>を基本単位から合成するときに現れる数値因子がつねに1になるように選んだ単位系が一貫性のある(コヒーレントな:coherent)単位系であるという。
