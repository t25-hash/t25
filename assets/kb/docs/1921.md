# 1・3・5 SI による表現のルール(SI-expression rules)

> **章**: 第1章 単位系と次元
> **節**: 1・3 国際単位系

単位と記号と数値の表記に関する基本原則は、第9回CGPM(1948年)で定められたが、その後の細かなルールの設定は、ISO/TC12の役割となっている。1. 単位記号、接頭語記号は立体文字(ローマン体)で表記する。文章の終わりにくる場合を除いて終止符(ピリオド)は付けない。2. 単位記号どうしの積には中点(・)を打つのが望ましい。例えば粘度の単位パスカル秒に対しPa・sとする。3. 単位記号の商が複雑な場合はカッコまたは負の累乗(指数)を用いる。例えば比熱容量の単位ジュール毎キログラム毎ケルビンに対しJ/(kg・K)またはJ・kg・Kとし、J/kg・K としてはいけない。4. 接頭語の記号と単位記号を結合して作った記号の全体は不可分な新しい単位記号(元の単位の十進の倍量または分量単位記号)を形成し、これに正または負の累乗を行ったり、他の単位記号と組合わせて合成単位を作ることができる。例えば1cm=1(cm)=10m,1MK=(10K)=10K,1V/cm=(1V)/(109 ―4α9学単位・物理定数・数編m)=10V/m.5. 接頭語を二つ以上重ねて用いてはいけない。例えば、1Fではなく1pFとする。6. 数値と単位記号の間に適当なすきまを(全角文字幅の1/4程度を目安として)入れる。
