# 1・4・1 CGS 単位系(CGS system of units)

> **章**: 第1章 単位系と次元
> **節**: 1・4 その他の単位系

CGS 単位系は長さの単位としてセンチメートル(cm)、質量の単位としてグラム(g)、時間の単位として秒(s)の三つの単位を基本単位とする一貫性のある単位系である。SIは同種のMKS 単位系が発展したものであり、SIの枠の中には、SI単位の十進の分量単位であるcm,g も含まれるが、この二つと秒(s)を基本としたCGS 単位系はSIとは厳格に区別される。CGS 単位系はおもに物理学の分野で使われてきた。力学現象を記述するにはこの三つでよいが、電気・磁気現象を記述するには、電気または磁気の法則を規定する必要がある。静電気における非有理化で表したクーロンの法則で、誘電率を1として出発した単位系をCGS 静電単位系(CGS esu)、磁気における非有理化で表したクーロンの法則で、透磁率を1として出発した単位系をCGS 電磁単位系(CGS emu)という。この二つの単位系のうちから電気的量に対してCGS esu、磁気的量に対してCGS emuに一致するようにとられた単位系をCGS ガウス単位系という。CGS 単位系で特別な(固有の)名称を付与された単位の例を以下に示す。名称記号SI単位で表される値エルグerg1erg=10Jダインdyn1dyn=10NポアズP1P=1dyn・s/cm=0.1Pa・sストークスSt1St=1cm/s=10m/sガウスG,Gs10T に相当πエルステッドOe(1000/4)A/mに相当マクスウェルMx10Wbに相当スチルブsb1sb=1cd/cm=10cd/mフォトph1ph=10lxガルGal1Gal=1cm/s=10m/s
