# 19・4・1 解析可能なスケール幅

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

乱流は，流れの支配方程式の非線形性に起因し，レイノルズ数を計算することができる．
しかし，乱流は大小さまざまなスケールの渦によって構成されており，そのスケール幅はレイノルズ数とともに拡大する．
例えば，流路や物体に関連して発生する渦の最大スケールを，エネルギーが散逸されるスケールをηとすると，両者の比はレイノルズ数 に対してη∝と見積もられている．
乱流変動は本質的に三次元現象であるから，すべてのスケールにわたって計算するには 程度の格子分割数が必要であることになる．
このことは，格子数1000の大規模計算であっても低レイノルズ数乱流しか扱えないことを意味する．
そこで，乱流の数値計算においては，使用できる計算機の容量に応じて，実際の乱流場の中から解析するスケール幅の切出しを行う必要がある．
計算領域よりも大きなスケールと格子幅よりも小さなスケールにはなんらかの仮定やモデルが必要となる．
（Reynolds number）が高いときに発生する現象である．
したがっオフ波数（計算できる最高の波数であり，等間隔格子では格子幅δに対して＝πδ）を高くする必要がある．
一方，六次以上では高次化の効果は次第に小さくなる．
差分法では，プログラミングの手間も考慮すれば四次精度程度が妥当であろう．
時間発展もこれに対応して二次以上の精度であることが望ましい．
精度次数よりも重要な性質は離散化における整合性 である．
これは，＝＋，＝，＝といった微分で成立する関係が離散化後も保たれる条件であり，保存則の離散化に対する原則的な要請である．
さらに，乱流理論やモデルの検証においては各種の統計量を残差なく算出する条件でもある．
乱流変動の直接計算には，境界条件と同様に初期条件にも乱れの情報が必要である．
非圧縮流れでは連続の式＝0を満たす変動速度場を作成することは困難である．
