# 19・4・2 乱流計算のための境界条件

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

解析スケールの切出しと関連して重要な点は境界条件（bound-に直交する面内に を作成すれば，波数空間における連続の式＝0を満足する．
さらに を調節することにより，速度変動のスペクトルを任意に与えることができる．
また，同様の方法でランダム外力を与えれば，統計的に定常な乱流場を生成することができる．
乱数を初期値としたり，不安定モードを加振する方法があるが，それらの影響がなくなるまで十分に時間発展させる必要がある．
期性を仮定した三方向にフーリエ変換できるので，波数ベクトル速度変動まで計算する場合には，計算領域から流入・流出する時々刻々の乱れも与えなければならない．
流入する乱れは一般には未知であるが，平均流れに適切な近似変動を与え，十分な助走区間を設ける必要がある．
流出条件としては，一般に無反射境界条件が用いられる．
特に圧縮性乱流においては，境界からの非物理的な反射が解析する乱れに影響を及ぼさないようにすることが不可欠である．
