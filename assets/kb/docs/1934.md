# 1・5・7 音響に関する単位(units in acoustics)

> **章**: 第1章 単位系と次元
> **節**: 1・5 単位の説明と換算

1. 音の高さ(sound frequency)音の高さは音波の振動数によって定まる。振動数の大きい音波ほど音が高い。振動数の単位はヘルツ(Hz)。2. 音圧(sound pressure)音波は媒質内における音圧の振動であるが、音圧はこの圧力振動の大気圧からの変動の大きさの実効値をいう。単位はパスカル(Pa)。特定瞬時の音圧をいうときは瞬時音圧という。3. 音圧レベル(sound pressure level)音圧レベル욣は基準音圧(=20 Pa)と音圧(実効値)から、욣=ln=ln10lg (lnはlog윬,lg はlog を意味する)によって決める。単位はベル(B)で、1B は2lg =1のときのに対する音圧レベルである。B の代わりにその1/10である分量単位、デシベル(dB)を通常使用する。20lg =1のときのの音圧レベルは1dB である。4. 音の強さ(sound intensity)音波の伝ぱ方向に直角な面の単位面積に伝達される音響パワーの大きさ。単位はワット毎平方メートル(W/m)。平面進行正弦波の場合、圧力振幅の実効値、媒質の密度ρ、音波の伝ぱ速度を用いると、音の強さはρであるから、同一媒質中では音圧によって音の強さを表現できる。5. 音響パワー(sound pwer)音波として発射、伝達またはo入射されるパワーの大きさ。単位はワット(W)。発射パワーをいうときは音響出力、入射パワーをいうときは音響入力ということがある。6. 音響パワーレベル(音の強さのレベル、sound power level)音響パワーレベルは基準音響パワー(=1pW=10W)と対象とする音響パワーから、=12ln=12ln10lg によって決める。単位はベル(B)で、1B はlg=1のときのに対する音響パワーレベルである。B の代わりにその1/10である分量単位、デシベル(dB)を通常使用する。10lg =1のときのの音響パワーレベルは1dB である。音響出力のレベルに対しては音響出力レベルということがある。7. 音の大きさのレベル(loudness level)音の大きさのレベルは基準音圧(=20 Pa)と対象とする音の音圧の実効値윬읆읆から、=ln윬읆읆윭읐=ln10lg 윬읆읆윭읐によって決める。ここで、윬읆읆は調査対象とする音が、標準化された可聴条件下で、正常な観察者によって1kHz の純音の音圧と同じ大きさであると判定されたときのその純音の音圧の実効値。単位はフォン(phon)。1フォンは2lg 윬읆읆윭읐=0.1のときの音の大きさのレベルである。1kHz の純音については1フォン=^ 1dB.8. 音の大さ(loudness)対象とする音の強さと40フォンきの音の大きさのレベルをもつ標準音の強さとの比の正常な観察者の聴覚による推定値。単位はソーン(sone)。ソーンによる音の大きさとフォンによる音の大きさのレベル間の実用的な標準関係式はのISO131に示されている。9.
