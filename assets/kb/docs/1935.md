# 19・4・6 レイノルズ平均の数値計算（RANS)

> **章**: 第19章 流れの数値解析
> **節**: 19・4 乱流への適用

レイノルズ平均された方程式を解いて時間平均的な流れ場を求め
ようとする計算は，渦粘性モデル（eddy viscosity model ）と応力
方程式モデル（stress equation model ）に大別される．これらは
当然ながらモデルへの依存性が高い．したがって，使用に際して
は，モデルの特性を理解しておくとともに，計算結果を注意深く解
釈する必要がある．
市販の流れ解析ソフトの多くは - εモデルなどの2方程式型渦
粘性モデルを標準的に組込んでいる．渦粘性はε（ ：定
数）で与えられる．これは負の値にはならないため，レイノルズ応
力に対するモデル項は数値計算を安定化させる．このことが，さま
ざまな用途に対応する必要のある市販ソフトに受け入れられている
一因と考えられる．しかし，渦粘性モデルにはさまざまな欠陥が指
摘されている．例えば，流線の曲り，主流方向の加減速あるいは回
転系など，レイノルズ応力成分間の再配分が重要となる流れ場の乱
流を適切に表現するのが困難である．渦粘性モデルが有効な場合
は，ひずみ速度とレイノルズ応力のうち例えば と′′の一
組だけが支配的となる，比較的単純な流れ場である．
これに対して応力方程式モデルは，特定のレイノルズ応力成分に
作用する効果，圧力-ひずみ相関項によって成分間の再配分のいず
れも表現できることから，渦粘性モデルよりも広範囲の乱流場に適
応できると期待される．その結果として解くべき方程式の数が増え
るが，この問題は計算機の発達により克服されるだろう．しかし，
モデルの各部分の吟味はまだ十分ではない．また，多くの乱流統計
量に対して初期条件や境界条件を与える必要がある．さらに，レイ
ノルズ応力方程式は必ずしも拡散的ではなく，数値的に不安定にな
りやすいのでなんらかの工夫が必要となる ．以上のことから，
応力方程式モデルは渦粘性モデルと比べて使いやすいとはいえな
い．

原理的には格子依存性をもたないはずであり，格子を細かくした
り，時間進行の刻みを小さくしたりしても小スケールの乱れが顕在
化することはない．格子の依存性が現れるとすれば，その原因は乱
流渦のスケールと格子幅の関係によるものではなく，例えば壁近傍
の漸近関数に対して格子解像度が不足する場合である．一方，モデ
ルが想定する時間・空間スケールよりも大きな非定常性が現れるこ
とは想定される．これらのことから，格子幅や時間刻みをさらに細
かくしても結果が変わらず，格子依存性がないことが確認された場
合に，その結果は使用したモデル方程式に対する妥当な数値解と考
えてよい．
