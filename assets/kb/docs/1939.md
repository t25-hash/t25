f .構造格子による複雑形状への対応 複雑な形状まわりの流
れ場では単一の境界適合構造格子では困難である．そのため，マル
チブロック法 や重なり格子法 などが用いられる．
.マルチブロック法 リー（Lee）ら によって提案さ
れたマルチブロック法（multi - block method）は，三次元物体を
直方体ブロックの集まりに置換え，そのまわりに格子のブロックを
積み上げて流れ場全体を覆う方法である．個々のブロックは直方体
に近い形であり，ブロック内にH型格子を形成することは容易で
ある．隣接するブロックの境界面では格子点を共有している．流れ
場の計算時には，ブロックごとに計算を行って逐次ブロック境界面
で情報交換をする方法と，直方体で置換えた物体の内部にも格子を
張って全空間を単一のH型格子のようにして取扱う方法とがある．
後者の場合，物体内外を区別するフラッグが必要であるものの，全
場を一度に解くことによりベクトル演算機をもつ計算機には計算効
率の点で有利となる．沢田ら はこの方法を航空機全機まわりに
適用し，その有効性を示した．
.重なり格子法 ステーガーら により提案された重な
り格子法（overset grid method，chimera grid えば飛行機全機まわりの計算の場合，翼や胴体といった部分ごとに
格子を形成して複雑な形状まわり全体の場を表現する方法である．
格子どうしは重なり，その重なった領域で格子間の情報のやりとり
を行いながら計算を進める．マルチブロック法に対して重なり格子
法を用いる有利な点は，格子どうしが時間的に移動することもでき
ることであり，スペースシャトルのブースタが分離する過程のシミ
ュレーション ，ヘリコプタ全機まわりの数値計算 や，トンネ
ル内に突入する高速列車まわりの非定常流れのシミュレーショ
ン も行われている．複数の格子を用いて移動物体を扱うという
アプローチは，非構造格子でも用いられる ．
