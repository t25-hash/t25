# 19・6・3 画像ファイルと動画

> **章**: 第19章 流れの数値解析
> **節**: 19・6 可視化手法

a.画像ファイル 可視化した結果は画像ファイルとして保存
したり，あるいは文書などに張り付けたりする．画像ファイルの形
式としては，EPS，TIFF，BMP，PICT，JPEG，PNG，GIFな
どさまざまなものがあるが，多くものはそれぞれに変換が可能であ
るので問題はない．ただし，これらのいくつかは画質を落としたり
色数を減らしたりしてファイルサイズを小さくするので，希望する
画質によって使い分けることが必要である．
b.動画 動画（アニメーション）を作る方法はおもに二とお
りある．一つは，可視化しているPCやWS（ワークステーショ
ン）に，ディスプレイに表示されている映像の一部をTV信号と
して取出す機能がある場合，可視化ソフトウェアによって動いてい
る画像をそのままビデオに録画するものである．これはハードウェ
ア/ソフトウェアの条件がそろっていること，ディスプレイ上で動
いて見える程度に少ないデータ量の可視化であることが必要とな
る．もう一つは，静止画の画像ファイルを複数枚用意し，これを並
べて動画にする方法である．この場合，1枚の画像を作るのに要す
る時間には制約がないので，どんな複雑な画像でも動画にできる．
動画はディジタルビデオの形式となるが，周辺機器やソフトウェア
を整えれば，TV信号として出力し，ビデオに記録することもまた
可能である．
