# 19・1・3 ブレーキの制動性能

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・1 自動車の振動

自動車の制動力は，前後輪が出しうる路面とタイヤの摩擦力の合
計に等しい．したがって，あらゆる路面において最大の制動力を得
るためには，前後輪が同時にロックするような制動力配分が理想と
なる．制動力がタイヤに作用するとき，慣性力は車体重心に作用す
るので車体に回転モーメントが作用して前輪への荷重移動が起き
る．前後輪の摩擦力で制動力を最大にするためには，前輪と後輪の
荷重変化を考慮した制動力配分が理想となる．この理想制動力配分
は図19･5 に示すような二次曲線となる．この理想制動力配分よ
り下の領域では前輪が先にロックし，上の領域では後輪が先にロッ
クする．後輪ロックは制動力および横力が低下して自動車の安定性
から見て好ましくないので，これを防止するためにブレーキ圧を制
御するPCV（液圧調整弁）が採用されてきた．最近は四輪の制動
力を確保しつつ横力低下による車体の尻振りを回避するための
ABS（四輪ロック防止のアンチロック装置）が多く使用されてい
る．
