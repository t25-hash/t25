# 19・1・4 衝突安全性

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・1 自動車の振動

a.自動車の安全性能 自動車の安全性能は(１)予防安全，
(２)衝突安全，(３)事故後安全の三つに分けられる．予防安全は事
故を未然に防ぐための性能でアクティブセーフティと呼ばれてお
り，ブレーキ性能，操縦性・安定性などの性能に依存する．衝突安
全は衝突時の乗員および歩行者などの被害を軽減するための性能で
あり，おもに車体構造，乗員保護装置，操舵ステアリングなどの衝
突エネルギー吸収性能などに依存する．事故後安全は衝突後の乗員
救出のために容易に開きうるドアシステム，容易にはずせるシート
ベルトをはじめ，火災などの二次災害防止のための燃料タンク構造
や燃料タンク注入口，配管などの配置などが重要である．
b.自動車の衝突 自動車が障害物に衝突すると，車体は数十
～数百ms （ミリ秒）といった短時間で停止するため車体には大き
な減速度が発生する．そのときに，自動車の運動エネルギーを車体
のクラッシュで吸収して，乗員の受ける減速度を低減させて被害を
軽減することは，衝突安全性能において重要な技術課題である．
