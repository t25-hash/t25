# 19・1・7 ステアリング系・ブレーキ系の振動

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・1 自動車の振動

a.ステアリング系の振動 ステアリング系に発生する振動は
次のようなものがある．
(１)シミー（shimmy）またはウォブル（wheel wobble）
タイヤの変形，横滑り特性による操舵輪のキングピン回りの自励
振動．
(２)フラッタ（wheel flutter ）
おもに車輪のアンバランスなどによる操舵輪のキングピン回りの
強制振動．
図19･12 はシミーの振動系の一つのモデルである．タイヤの
コーナリングパワーを とおくと，式(19･5)に示すようにタイヤ
のコーナリングフォースは の関数でタイヤの横剛性のダンパ
として作用する．これがコーナリングフォースの位相遅れをもたら
して，負の減衰として作用することがあるので，シミーの発生に大
きく影響を与えることがわかる．
フラッタの振動数はステアリング系の固有振動数の一つにほぼ一
致する．振動モデルとしては図19･12と同じでありコーナリングフ
ォースによる位相遅れの影響が小さく，車輪の不釣合いによる周期
励振力が加わる場合を考えればよい．フラッタの振動数は＝
12π2で求められる．ここで， は車輪のキングピン
回りの慣性モーメント， は車輪からハンドルまでのキングピン
回りのねじり剛性である．
b.ブレーキジャダー ブレーキをかけるときにもステアリン
グ系または車体が振動することがある．これはブレーキジャダーと
呼ばれる現象で次のような原因によって起こる．
(１)ディスクロータ偏摩耗による厚さ変化の増加，ブレーキド
ラムの真円度不良，ディスクホイール横振れなどが原因でタイヤ回
転一次の振動が発生．
(２)ブレーキドラムのアクスルシャフトへの組付荷重が不均一
でしゅう動部がだ円変形することが原因でタイヤ回転二次の振動が
発生．
(３)ディスクロータ，ブレーキドラム単体の工作精度不良，デ
ィスクロータ表面の発錆による凹凸が主原因でタイヤ回転一次～数
次の振動が発生．
(４)ディスクロータ，ブレーキドラムの温度上昇に伴うしゅう
動部の熱膨張による変形が原因でタイヤ回転二次の振動が発生．
ブレーキジャダーは，懸架系，ステアリング系，車体などの振動
や，それに伴う騒音を発生させるとともにブレーキペダルの脈動を
招く．ブレーキの制動トルク変動の振動周波数は，車速の低下に伴
い減少し，タイヤ，懸架系ばね下上下，ワインドアップ，車体，ス
テアリング上下などの低次の共振周波数（10～40Hz）に一致する
車速でこれらが共振し，はなはだしい場合は車体全体を振動させて
低周波のこもり音となることもある．
