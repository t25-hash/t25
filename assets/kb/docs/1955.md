# 19・2・3 曲線走行時のモデリング

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・2 鉄道

曲線を通過するとき，車輪がレールに与える左右方向の力である
横圧を小さくし，スムーズに車両が旋回していけるためには，曲線
外側車輪が内側車輪よりレール長さの違い分だけ多く進み，車輪が
曲線接線方向に向くことが望ましい．そのためには，等価踏面こう
配を大きくするなど，蛇行動防止上の要求とは相反することが望ま
れる．蛇行動と曲線通過の両性能の両立を図る車両の設計が車両運
動解析の目的である．
a.遠心力 鉄道車両の場合，軌道に拘束されて走行するので
軌道からの車両の変位は小さく，変位が微小との仮定が使えるなど
利点が多いので車両とともに軌道に沿って移動していく移動座標系
に対して運動方程式を導くのが一般的である．絶対座標に対して運
動を記述する場合と違って曲線走行時の遠心力，コリオリ力などは
考慮されない．鉄道車両の場合コリオリ力は小さいので遠心力を付
加してやる必要がある．実際には曲線ではカントと呼ばれる傾きが
ついているのでカントを考慮した超過遠心力を付加する．
b.前後クリープ率 外軌レールの接触点の移動速度は直線の
場合より増加するので，曲線半径を として外軌車輪の左右クリ
ープ率は だけ増加し，内軌車輪では だけ減少すること
の考慮が必要となる．
