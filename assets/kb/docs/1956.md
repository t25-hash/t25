# 19・2・4 通り変位のモデリング

> **章**: 第19章 自動車および鉄道車両の振動
> **節**: 19・2 鉄道

車両の乗り心地を阻害する外乱として代表的なものはレールの変
位である．高低（上下）変位，通り（左右）変位，水準（傾斜）変
位，軌間（間隔）変位の各方向の変位が存在するが，輪軸の左右系
運動方程式に関連する通り変位のモデリングについて述べる．
a.通り変位考慮法 高低変位に対しては車輪とレールは離れ
ることがないので，変位量だけ車輪に上下に強制的な変位が与えら
れる系として扱える．したがって一般の機械構造物の定式化と全く
同じである．
