# 20・3・3 構造物の振動制御

> **章**: 第20章 制振および振動・衝撃の絶縁
> **節**: 20・3 能動的振動制御

a.構造物の制御とその目的 構造物，特に住宅や高層ビルに
焦点を当てるなら，振動制御の目的としては居住環境の快適性，安
全性への要請がおもなものである．居住環境への外乱としては，支
点外乱としての入力と，直接に構造物に力が作用する直接振動とし
ての入力の二つである．支点外乱の要素としては交通振動，地震が
代表的である．直接振動の要因としては風，内部利用者，設備機器
などがある．これら外乱に対して絶縁問題として考える場合と制振
問題として考える場合があるが，ここでは制振問題としての振動制
御に焦点を当て，さらに構造物全体が振動するような低周波の水平
振動モデルに限定するものとする．
能動的制振技術の構造物への代表的な適用方法は，付加的な質量
をアクチュエータで駆動してその反力を制御対象構造物に操作力と
して作用させる方式である．この方式による振動制御装置はアクテ
ィブ制振装置（active damper ）あるいはアクティブマスダンパと
呼ばれている．高層ビルなどの構造物に対して最も有効に制御力を
作用させることができるのは屋上などの高所であり，反力構造を使
って直接に操作力を構造物に作用させることを期待することはでき
ないため，このような付加質量の慣性力が用いられている．
