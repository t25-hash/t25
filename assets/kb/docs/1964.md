# 2・1・1 三角関数の定義

> **章**: 第2章 三角関数および双曲線関数
> **節**: 2・1 三角関数(trigonometric function)

θを原線OA と動径OPとの間の角とし,PからOA に垂線PMを下せば(図II-2・1),正弦(sine):sinθ=MPOP余弦(cosine):cos θ=OMOP正接(tangent):tanθ=MPOM余割(cosecant):cosecθ=OPMP正割(secant):secθ=OPOM余接(cotangent):cotθ=OMMP正矢(versed sine):vers θ=1-cos θ余矢(coversed sine):covers θ=1-sinθ
