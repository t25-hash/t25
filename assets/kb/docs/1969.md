# 21・1・4 応答スペクトル

> **章**: 第21章 耐震設計
> **節**: 21・1 耐震工学の基礎と地震

耐震設計の考え方が他の振動工学と異なる点は応答スペクトル
（response spectrum）の概念であるが，その詳細は21･3･3項へ譲
り，ここでは地震波形の振動数特性を表す一つの方法としての応答
スペクトルの考え方を述べる．これは固有振動数（固有周期）と臨
界減衰比で1質点減衰ばね系を代表させ，その系の地震波入力に対
する応答の最大値（最大変位応答，最大速度応答，最大絶対加速度
応答）をとる．この最大値と系の固有振動数（固有周期）を臨界減
衰比をパラメータとして図示したグラフが応答スペクトルであり，
各々変位応答スペクトル，速度応答スペクトル，加速度応答スペク
トルと呼ぶ．図21･6に1940年のインペリアルバレー（Imperial
Valley）地震のエルセントロ（El Centro）波の代表的な応答スペ
クトルを示す．一般に1～5 %の臨界減衰比に対するものが多く作
成され，この範囲で凹凸の激しい折線グラフを呈する．実用的には
平均，標準偏差分の上積，包絡などの方法で平滑されたもの，さら
に数段の折線で近似されたものなどが設計用として使われる．代表
的な応答スペクトルとしては，ハウスナーの平均応答スペクトル，
土木研究所による平均応答スペクトル，コンビナート保安・防災技
術指針によるスペクトル，大崎スペクトルなどが提案されている．
また，地震動の経時特性のうち最大応答の発生に関するものは応答
スペクトルに含まれているが，継続時間および応答の時間経過に関
するものは含まれていないので，別途規定することが多くの場合必
要である．
