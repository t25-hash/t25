# 21・2・3 設計地震動（design basis earthquake)

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

軽水炉耐震設計の基本方針は原子力安全委員会の発電用原子炉施ごとに想定されるいかなる地震力に対してもそれが大きな事故の誘限界的な地震が発生しても原子炉は冷却材の喪失事故を生じずに安震としては，発電所建設地点を中心とした広い範囲の地域にわたる用最強地震Sと設計用限界地震Sが想定されている．
その近傍に影響を与えたと考えられる地震が再び起こり，敷地およによる地震のうちの最も影響の大きいもの．
回る地震について，過去の地震の発生状況，敷地周辺の活断層の性影響の大きいもの．
で用いられる地震波を用いることが多い．
例えば発電用大型ボイラ計に関しての審査が行われる場合が多い．
そこで，以下にここで用(財)日本建築センターの高層建築物構造評定委員会が作成した構の動的解析用地震波について では，設計用地震動波形として，原則とすると述べている．
地震動の強さは地表面位置での最大速度で定義し，地震活動度が低50kine（cm/ s）以上を要求している．
したがって，建築構造物の大速度をレベル1とレベル2に応じてそれぞれ規準化したものとなら，そのスペクトル特性は当然のことながら観測地震動のそれと同いものであるという前提に立っており，地震動特性が異なる複数のさの問題を克服しようとしたものである．
ここで，からに震波について簡単に紹介しておく．
ニア州南部で発生したインペリアルバレー地震（M7.1）の際に，設置された強震計により記録された．
地盤は，よく締まった洪積地ではP波速度1800m/ sのローム層，厚いれき，砂，粘土層が続いない．
この記録は，1940年当時では世界で数少ない強震記録で，タフトEW波：1952年7月21日にアメリカ，カリフォルニア州離約40kmの位置にあるタフト・リンカーン・スクールに設置さボイラ耐震設計の標準的な設計用地震動波として知られている．
の際に，震央から約180km離れた八戸港の地表に設置された強震弱な沖積層である．

よく用いられているものに通称臨海波と建築センター波があkine，50kineに引伸ばし，慣用的に耐震設計に用いてきた経緯がな問題はないとされてきていたが，堅固な地盤が地中深くにあるよが生じる．
そこで，震源断層を考慮した入力地震動評価により得ら波が開発された．
臨海波は東京の臨海部の工学的基盤である東京れき層や江戸川層を対象に，東京周辺で起こった過去の地震を想定し設に関する耐震設計審査指針 に示されているように，建設地点因とならぬよう設計することであり，現実には起こりえないような全に停止・維持されるようになっている．
などは日本建築センター高層建築物高層評定委員会においてその設いられることの多い設計用地震波について紹介する．

このような考え方は，強震動は元来不規則で予測できな既往の観測波を用いることにより，強震動の不規則性や予測の困難挙げた観測地震動の中から，特によく用いられてきたいくつかの地エルセントロNS波：1940年5月18日にアメリカ，カリフォル断層から約10km離れたエルセントロという場所の建物地下室に盤であり，地表から20mはP波速度360m/ sの粘土層，それ以深ている．
