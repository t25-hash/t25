# 21・2・4 損傷様式（mode of failure)

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

産業施設内の機械構造物は建築物，橋りょうなどに比べ，一般的
にその力学的形状は複雑である．したがって，地震時にどの部位に
おいて，どのように，そしてどの程度損傷あるいは破壊（損傷とは
誤動作も含め機能を失うこと，および破壊することを含む．破壊と
は設備の全体または一部が過度の変形もしくは破断に至ることをい
う）を生じるかを，設計時に正確に把握することが重要となる．そ
のためには理論的考察もさることながら，実際の地震での損傷例を
できるだけ数多く集めて，分類・分析することが重要である．表

系として損傷に至るかを検討する必要がある．このような観点か
ら，実地震の被害調査は極めて重要なものとして位置づけられ，
1995年に生じた兵庫県南部地震の地震被害調査においては地震工
学にかかわる各学会において速報版的な報告書 が発行されると
ともに，地盤工学会，土木学会，日本建築学会，日本地震学会，日

報告編集委員会が設置され，全26巻からなる公式な地震被害調査
報告書 が刊行された．その中で，地震被害のデータベース化の
試みも行われた．さらに，近年になり，1999年のトルココジャエ
リ地震，台湾921地震 ，2001年のインド西部地震と世界的に見
ても，巨大災害地震が頻発していることから，地震被害調査活動を
効率的にかつ正確に実施するための地震被害調査マニュアル作成も
進められている ．表21･2の内容は大別して，(ａ)振動的被害，
(ｂ)強制変形被害，(ｃ)断層被害，(ｄ)地盤液状化被害になるが，

プラインについてのような設計解析はほとんど実施されていない．
(ｄ)になると発生予測の解析は行われても，全体としては地盤改良
等による対策が中心となる．地震被害調査においては，損傷の直接
の誘因が何であるかを判断し，またその損傷過程を現地にて判断す
る必要に迫られることから，高度な専門的知識と判断力が要求され
るため，その体制作りと，若い研究者/技術者のこの種の調査への
損傷の様式が定まると，それに基づき力学モデルを構成し，その
振動特性解析を行う．一般に振動特性は固有振動数，振動モード，
減衰特性から成り立っている．このうち前二者は対象構造物が決定
すれば，それを適当にモデル化することにより計算で精度よく（少
なくとも工学的には）求めることができる．しかしながら，減衰特
性に関しては，例えば制振構造の場合のように特に大きな減衰要素
を人為的に付加した場合を除いては，計算で求めることは一般的に
困難である．したがって，これは通常，振動実験などにより実験的
に求められる．構造物において，減衰は構造部材の結合部や構造物
内可動物に生じる摩擦，がた，衝撃によって，運動エネルギーが消
散するものと，構造物からこれを支持する基礎・地盤や他の結合さ
れた構造物へ逸散していくものがある．地震時における構造部材の
降伏など材料そのものによるエネルギー吸収は，一般的な機械構造
物における通常の設計限度である弾性ないしは弱い弾塑性領域では
大きく寄与しない程度であり，問題ないとされている．また，大型
な構造物で地震時においても弾性変形が支配的となる原子炉建屋な
どでは，地盤に逸散するものが主たるものとなり，計算で比較的精
度よくこの量を推定できる．
構造物の地震応答解析については21･3節において詳細に述べら
れるので，ここでは結果の評価に対する留意点について記載する．
すなわち，より精度の高い解析手法を用いたほうが，設計の信頼性
が向上するものと一般に信じられているが，地震を含む設計条件に
対しては精度は向上するが，自然現象である地震そのものには多く
の不確定な因子が含まれており，実際の地震時における設備の挙動
の推定精度が向上するわけでないことにまず留意する必要があると
いうことである．このような統計的な性質は入力である地震波自体
および出力である応答結果ともに存在する．柴田らによれば，応答
倍率の揺らぎは意外に大きく，特定地点のデータをまとめて解析し
た結果は，変動係数（標準偏差/平均値で と呼ばれる）で
30%程度となり，さらにその地点に固有の地震については，3 σを
超えた応答倍率を示すこともあることが指摘されている ．
