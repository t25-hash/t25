# 21・2・5 振動特性解析・応答解析

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

て，断層モデルを含め，さまざまな方法で地震動を評価し，応答ス
ペクトルのレベルを定めたものである．想定した地震は，関東地
震，安政江戸地震および東海地震であるが，関東地震が他の二つの
地震の評価結果をほぼ満足していることから，関東地震に対して評
価されたレベルをレベル2地震動として定めている．建築センター
波の元となるスペクトルは，臨海波と同じく，東京における工学的
基盤を念頭に，安政江戸地震，関東地震，東海地震の影響を種々の
方法で検討し，それらをほぼ包絡する形で求めたものである．臨海
波との大きな違いは，おもに東海地震による地震動が大きく評価さ
れたため，スペクトルの長周期側が周期10秒まで100kine（cm/
s ）フラットになっている点である．
