# 21・2・6 許容応力

> **章**: 第21章 耐震設計
> **節**: 21・2 耐震設計

鋼構造の許容応力の体系は最大主応力説に基づくものと，最大せん断応力説に基づくものとがあり，その基本的な考え方は，建築・土木のものと機械系のものと分けて考えることもできる．
機械設備e .昇降機の耐震設計度とし，後者を50～100 %大きくとっている．
一方，圧力容器などを中心とした機械系の許容応力は，当初は主応力説により，弾性範d.電力機器の耐震設計を支持する構造物，鉄筋コンクリート，基礎などについては建築系の許容応力度体系の影響を強く受けている．
建築・土木系では，常時荷重に対して長期許容応力度，変動荷重に対しては短期許容応力エレベータ協会．
