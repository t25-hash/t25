# 21・2・7 設計細目（general design criteria)

> **章**: 第21章 耐震設計
> **節**: 21・3 機械構造物の地震応答解析

図21･9にそのMARK-Ⅱ改良型の地震応答解析モデルを示
設計上の細部に関する制限事項で，計算や解析によらず，耐震性
す ．建屋と地盤の連成を考慮している．地震動は建屋の振動
特性によって増幅されて，機械設備の据付位置から機械設備に入力
される．原子炉建屋と圧力容器支持構造物は連成運動を行う系と
し，各部分の剛性は曲げとせん断を考慮して求め，質量を集中質量
とした多質点系モデルとしている．モデルの質点位置は，配管ノズ
ルの取付位置，構造体の板厚変化点，主要応力評価点などが考慮さ
れて定められている．一般の機器は建屋とは非連成でモデル化され
る．
