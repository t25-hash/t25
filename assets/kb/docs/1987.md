# 21・5・3 免震技術と制振技術

> **章**: 第21章 耐震設計
> **節**: 21・5 耐震性向上技術

図21･18に，地震応答低減法としての免震技術および制振技術の
構成を示す．
まず，免震技術については21･2節に述べているが，この技術は
耐震性向上にも大きな役割を果たす．機械系の構造物に対しても積
層ゴムは広く採用されるが，球面滑り支承，各種ベアリングを用い
るものも多い．対象としては，石油タンク，クレーンなどがある
が，高性能コンピュータ，半導体機器，医用設備，放射線設備など
に対しては床免震装置が採用されることが多い．
一例として，クレーンの耐震性向上のために開発された免震技術
を説明する ．1995年1月17日の兵庫県南部地震においては，神
戸港の港湾施設にあるほとんどのコンテナクレーンが震害を受け
た．地盤液状化に伴う，レールからの脱輪や脚部の座屈変形による
被害が多かったが，高重心構造としての横振動によって生じる脚部
のロッキング現象が脱輪や脚部破損を促進した可能性も指摘され
た．その後の研究，実験をもとに大地震時にクレーンの固有周期を
長くして免震機能を作動させる構造が開発された．これには，次の
ような方法がある．
(１)走行装置と脚部間に旋回台ベアリングを使用する．
(２)同じく免震ゴムを使用する．
(３)脚の中間部のフランジに免震支承を使用する．
(４)クレーン構造を柔構造とする．
(１)は，2000年8月に東京大井埠頭のクレーン設置に対して世
界に先駆けて実現された．(１)や(２)の方法は，地震時のみに免震
機能を発揮させるため，作動開始する機構（トリガ機構）と，地震
後に通常の作業状態に戻す機構（復元機構）が必要となるほか，エ
ネルギー吸収能を有するダンパも備えている．(３)や(４)は，免震
機能を発揮させるときの構造強度や付属機器等への影響などの配慮
が重要となる．
