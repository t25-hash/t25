# 22・2・7 伝達マトリックスの特徴

> **章**: 第22章 振動・音響の計算法
> **節**: 22・2 伝達マトリックス法

計算上の複雑さや精度を併せて考慮する必要がある．
またステップごとの収束計算を組入れることにより区分線形系以外の通常の非線形系にも適用できる．
ここではニューマークβ法を例にとり，増分伝達マトリックスを導出する．
ニューマークβ法では時刻＋Δの変位と速度を以下の近似式で表す．
面，複雑な系は直列的な系への変換やマトリックスの乗算回数の増上述のように伝達マトリックスは，要素の自由度とほぼ同程度の自由度で自由振動解析，強制振動解析，時刻歴応答解析，区分線形化された非線形系の解析等が可能で，伝達情報が順次与えられ，工学的に見通しがよい手法である．
したがって構造問題のみならず音響問題 などを含めていろいろな分野に応用されている．
その反加に伴う計算誤差が生じる問題があり，系に応じて個々にその対策が必要である．
