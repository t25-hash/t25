# 23・1・6 加振試験，音響試験 （vibration exciting test,

> **章**: 第23章 計測と信号処理
> **節**: 23・1 振動と騒音の計測

通常の振動や音の計測では解明できない問題も，加振試験や音響
試験により解明が進むことが多い．このような試験についてまとめ
たのが表23･3である．機械や構造物では種々の加振器を取付け，
ロードセル（フォースセンサ）により加振力を測定し，同時に応答
を測定することにより伝達関数を知ることができる．加振器は表に
示す種々のものが用途により適切に使い分けられる．ロードセルは
圧電型のものやひずみゲージ式のものが用いられる．音響試験の場
合は加振器としてスピーカが用いられる．この場合ロードセルの使
用は難しいが，スピーカの特性と入力電流によって入力を推定して
伝達関数を測定することになる．FFTの一般化により両試験とも
インパクト波を用いたテストが多く用いられるようになってきてい
る．
