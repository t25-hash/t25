# 6・2・1 複素関数

> **章**: 第6章 複素関数
> **節**: 6・2 複素数の関数

b. 初等関数。この場合の=0を対数分岐点という。. 分枝切断線多価関数では、点z が分岐点をまわるごとに関数値が他の分枝上の値に移る。分岐点と無限遠点を直線で結び、点z が分岐点をまわるとき、この線を横切れないようにすれば、関数は一つの分枝上の値をとり一価となる。このような線を分枝切断線という(分岐点と分岐点を結んで分枝切断線とすることもできる)。平面が平行に3枚(それぞれを葉という)あるとみなし、三つの平面上にそれぞれ等しい極座標を定義し、分枝切断線(例えば、3葉とも負の実軸を取る)に沿ってのみ3平面を接続させる。各葉上で原点を一まわりして分枝切断線を横切るごとに、第一葉→第二葉→第三葉→第一葉→...に移ると約束すれば、各葉上で=は一価となる。
