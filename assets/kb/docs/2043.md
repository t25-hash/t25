# 7・3・2 高速フーリエ変換(FFT : fast Fourier transform)

> **章**: 第7章 フーリエ変換
> **節**: 7・3 離散フーリエ変換

離散フーリエ変換式(II-7・11)は回の複素数の積であるが、e요욗の周期性を利用して、この計算回数をlogのオーダまで減らすことができる。このアルゴリズムを高速フーリエ変換という。大きいのときの計算効率が顕著によくなり、離散フーリエ変換が実用化された。例えば、=1024のとき、=1048576に対してlog=10240で、計算回数は約1/100になる。FFTは離散フーリエ変換式(II-7・11)の代名詞として使われることもある。
