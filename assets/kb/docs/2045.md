# 1・1・3 切断・結合工程に用いる加工法

> **章**: 第1章 加工学一般
> **節**: 1・1 加工学の体系

切断工程（cut oﬀprocess ）は，主として素形材を熱切断（ガス
切断，アーク切断，レーザ切断，プラズマ切断，火炎加工など），
せん断加工，除去加工などにより所要形状に分断し，仕上加工する
工程である．この工程で完成する場合もあるが，多くはこの後工程
として，結合工程や仕上工程にかけられる．結合工程（joining
process ）は溶接（融接，抵抗溶接など），固相接合，ろう付け，接
着など材料の材料学的結合法のほか，要素締結（リベット，ボル
ト）など，焼ばめ，圧入プレスなどの機械的結合法により，構造部
品や機械構造体を作り上げる工程である．切断・結合工程で製品と
して完成するものもあるが，構造体素材として切断・結合工程後に
さらに除去工程や仕上工程を経て加工・完成されるものも多い．
