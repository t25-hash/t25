# 1・2・1 生産設計と工程設計

> **章**: 第1章 加工学一般
> **節**: 1・2 加工工程の計画法

製品の加工工程の計画プロセスは，生産対象製品の加工コスト，
加工時間，加工精度，加工品質などを決定するうえで，非常に重要
なプロセスである．製品の観点から見た加工工程の計画プロセス
は，以下の二つに大別することができる ．一つは，製品の生産
設計（production design）プロセスであり，ここでは，VA（val-
ue analysis ，価値分析）などの解析手法を用いて，製品の性能，
加工コストおよび加工時間の観点から，適切な形状，寸法，材質，
公差，表面性状などを設計する．もう一つは，製品の工程設計
（process planning）プロセスであり，ここでは，加工対象製品の
形状，寸法，材質，公差，表面性状などの設計情報に基づいて，適
切な加工法，加工設備の設計または選択を行うとともに，加工作業
に必要な作業手順，設備の制御データの作成などを行う．
産設計が終了し，図面が出図された後，生産技術者による工程設計
が行われてきた．しかし，1990年代に入り，製品の設計，解析お
よび加工の分野において，コンピュータ技術を利用したCAD/
図1･2工程設計プロセス
CAE/ CAM（computer aided design/ computer aided engineer-
ing/ computer aided manufacturing）技術が発達，普及してきた
ことにより，製品の設計から生産設計，工程設計における製品デー
タの交換および共有が可能になってきた．このため，三次元形状モ
デル，製品モデル，ディジタルモックアップ，各種の解析およびシ
ミュレーションプログラム，PDM（product data management ，
製品データ管理）技術などを活用して，製品設計の初期段階から生
産設計および工程設計を同時並行的に行うことにより，加工工程の
情報を製品設計，生産設計にフィードバックしながら，適切な製品
とその加工工程の設計を行うことが可能になっている．このような
プロセスはコンカレントエンジニアリング（concurrent engineer-
ing）あるいはサイマルテニアスエンジニアリング（simultaneous
engineering）と呼ばれ，製品の設計から生産に至る期間短縮およ
びコスト削減に大きく寄与している．
例えば，製品設計および生産設計プロセスにおいては，鋳造およ
びプラスチック射出成形における型内部の材料の流動および凝固過
程の解析，プレス加工における板材の変形解析および応力解析，切
削および研削加工における切りくず生成プロセスのシミュレーショ
ン，工作機械の運動により生じる干渉解析，組立プロセスにおける
ロボットの干渉解析，作業者の作業性のチェックなど，多数の解析
プログラムが提供されており，CADで作成した三次元製品モデル
のデータに基づいて，その製品の加工工程で必要な確認および評価
プロセスを，コンピュータ上でシミュレーションすることが可能と
なっている．
