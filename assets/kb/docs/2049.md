# 1・2・2 工程設計の手順

> **章**: 第1章 加工学一般
> **節**: 1・2 加工工程の計画法

工程設計では，加工対象製品の設計データ（製品モデルまたは図
面）に基づいて，その加工工程の設計，すなわち加工部位とその加
工法，加工法のシーケンスを設計する．また，すべての製品の加工
作業を分割あるいはグループ化して，生産システム内の生産設備に
割り当てる．さらに，作業設計を行い，加工作業の実行に必要とな
る詳細な加工データ，ジグおよび工具のデータを作成する．
この工程設計プロセスは，対象とする加工方法，すなわち，鋳
造，鍛造，プレス加工などの成形・造形工程，溶接，組付けなどの
組立工程，切削，研削などの機械加工工程，仕上げおよび調整工程
などで異なるが，いずれも部品または組立品の形状，寸法，材質，
公差，表面性状，組立状態などの設計データから加工すべき部位を
認識し，その適切な加工法，生産設備，ジグ，工具を設計し，生産
設備の制御データ，作業者に対する加工指示書を作成する．以下で
製品の加工工程の計画プロセスでは，設計者が行う製品設計と生
は，NC（numerical control ）制御などの加工プロセスの自動化が
進んでいるため，工程設計技術が最も体系化されている機械加工工
程の設計プロセスを考える．
