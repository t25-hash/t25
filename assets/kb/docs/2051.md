# 8・2・4 直交ウェーブレット(orthogonal wavelet)

> **章**: 第8章 ウェーブレット変換
> **節**: 8・2 離散ウェーブレット変換

ウェーブレット関数はスケーリング関数によって=∑요2욅φ2-(II-8・12)と表される。この式はウェーブレット方程式(wavelet equation)と呼ばれる。ここで、욢욅を式(II-8・5)によって定義し、욢욅욅の張る空間を욢とすれば、욢∈욢は욢=∑요욢욅욢욅と表される。욢は욢における욢욪の補空間となる。つまり욢=욢욪⊕욢욪が成り立ち、解像度2の関数욢∈욢は욢=욢욪+욢욪の形に分解される。ウェーブレットはφと直交になるように、すなわち条件φ,-=욪φ-d=0(II-8・13)を満たすようにできる。このウェーブレットを直交ウェーブレットという。このとき、式(II-8・5)に対して正規直交条件の式(II-8・6)が満たされる。
