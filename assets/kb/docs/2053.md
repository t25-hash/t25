# 8・2・6 モーメント条件

> **章**: 第8章 ウェーブレット変換
> **節**: 8・2 離散ウェーブレット変換

直交条件と総和則とから直交フィルタは一意的に決まらない。普通、ウェーブレットにはさらにモーメント条件式(II-8・4)を課す。>2では一般に直交フィルタはモーメント条件を課しても一意的には決まらない。さらに別の条件をおいてさまざまなフィルタが作られている。ドーブシーフィルタ(Daubechiesfilter;または最小位相フィルタ、daublet)と、最小非対称フィルタ(least asymmetric filter,symlet)がよく知られており、ともに長さが2で係数がすべて実数の直交フィルタである。ま9 ―51た、スケーリング関数もモーメント条件を満たすコワフレット(Coiflet)もある。図II-8・3にハールフィルタによって決まるスケーリング関数とウェーブレットを示す[ここで符号は式(II-8・16)を満たすように決められているが、一般の文献では符号が逆の定義も使われている].また、図II-8・4にフィルタによって決まるスケーリング関数とウェーブレットを示す。この関数は連続であるが至るところで微分可能でない。モーメント条件式(II-8・4)が成り立つ次数が高いほど、スケーリング関数とウェーブレットの正則性がよくなる。
