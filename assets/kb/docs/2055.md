# 8・2・8 信号圧縮(signal compression)

> **章**: 第8章 ウェーブレット変換
> **節**: 8・2 離散ウェーブレット変換

トータルで信号の長さもエネルギーも変えないが、離散ウェーブレット変換によって信号のエネルギーが욇욅,=-1,-2,...に再配分される。その結果、少数個の係数욇욅に信号エネルギーが集中する。したがって、絶対値の小さい욇욅を零と置換えても信号のエネルギーはそれほど失われない。こうして、エネルギーを概ね保持して実質的に信号の長さを減らすこと、すなわち信号圧縮ができる。その割合は信号の特性と使うフィルタの種類によって異なるが、典型的には、98%のエネルギーを保持するのに10%ほどの個数の욇욅を残せばよい。どのような基準で重要な욇욅を選ぶかは、フーリエ変換を利用する信号圧縮と同様で、フーリエ係数の代わりにウェーブレット係数を対象とする。残された係数からは、逆ウェーブレット変換によってもとの信号の近似信号が再構成される。
