# 8・2・10 ウェーブレットのバリエーション

> **章**: 第8章 ウェーブレット変換
> **節**: 8・2 離散ウェーブレット変換

信号処理には波形を変化させることのない線形位相フィルタ(linear phase filter)が有利であるが,直交ウェーブレットのフィルタは対称性がなく,したがって線形位相でない.直交でないウェーブレットで対称性をもつものに,これと双対なウェーブレットを作り,互いに直交にすることができる.こうして作られたウェーブレット系を双直交ウェーブレット(biorthogonal wavelet)という.いくつか典型的なものが知られているが,画像処理などでは多く使われている.また,ウェーブレットの周波数特性をよくするために,いくつかのウェーブレットをつなげて基底関数を構成することができる.これをウェーブレットパケットという.周波数の変化を追跡したり,画像の局所的な細部を分解したりするには有利である.文献(1) Mallat, S., A Wavelet Tour of Signal Processing, 2nd ed.,(1999),5,Academic Press.(2) 榊原進,ウェーヴレットビギナーズガイド,(1995),3,東京電機大学出版局.→ Next Chapter
