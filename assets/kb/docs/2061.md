# 1・11・2 筐体材料（body material ）

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・12 カメラ

どに注意が必要である．ほかにはアクリロニトリル/ブタジエン/ス
モバイル機器である携帯電話には小型軽量化が求められる．筐体
チレン（ABS)，ポリフェニレンサルファイド（PPS），液晶ポリ
マ（LCP）なども使用されている．
ディジタルカメラの構造体は金属板材のプレス，かしめによるシ
ャシ構造が多い．冷間圧延鋼板SPCC，電気亜鉛めっき鋼板
SECC，冷間圧延ステンレス鋼板SUS304，SUS430などが使われ
る．これらは強度だけでなく加工性の良さが材料選択のポイントと
なっている．特にかしめを廃止し，絞り，鍛造加工で複雑な形状を
成形するような設計時には材料の絞り特性が重要なポイントとな
る．

ターンミラー（quick return mirror ），撮像素子などが取付く前板
部分はPCで構成されている．
図1･61はコンパクトカメラのズーム鏡筒（zoom barrel ）の一
例である．鏡筒の内側にはヘリコイド，カムなどの溝が複雑に形成
使用される．
されており，外側にはギヤが切られている．PCの射出成形により
複雑な部品を一体化し，小型化と低コスト化に大きく寄与してい
る．カム，ヘリコイドなどは10m単位の精度が必要であり，従
来はアルミニウムパイプ，ダイカスト材を切削加工するしかなかっ
た．
