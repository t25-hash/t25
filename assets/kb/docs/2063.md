# 1・12・2 機構部材

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・12 カメラ

カメラにおける機構部材はおもにレバー，歯車，ばねである．
a.レバー 1･12･1項の構造部材と同様に金属材料としては冷
間圧延鋼板SPCC，電気亜鉛めっき鋼板SECC，冷間圧延ステンレ
ス鋼板SUS304，SUS430などが使われる．樹脂材料ではPC，
してチタンの薄板の使用も見られる．
ABSがおもに使用されている．強度以外にしゅう動性を必要とす
る場合にはポリアセタール（POM）も使用される．
b.歯車 カメラ用の歯車はモジュール0.2～0.5程度の小型
精密歯車である．平歯車，ウォーム，ラックがおもに用いられる
が，はすば歯車，かさ歯車も用途によっては使用される．射出成形
で安く作れる樹脂製歯車が主流で，強度，精度が必要な部分だけに
かな駆動力で作動することで，カメラの小型化に寄与している．
金属製歯車が使用されている．
金属としては加工性，しゅう動性の良さから黄銅がよく使われて
いる．歯切りによる加工には快削黄銅棒C3602，C3603，C3604
などが，プレス加工には黄銅板C2801，快削黄銅板C3560などが
使われる．二段歯車や他の機構部品と一体化して歯車を使用すると
いる．
きはアルミニウム合金ダイカスト，亜鉛合金ダイカストもよく使わ
れる．
一方，樹脂製歯車は低コスト，軽量化，低騒音の特徴から広く使
われている．特にある程度の強度を保ちながら弾性を有し，摩擦係
数が低く自己潤滑性をもった材料が低騒音の観点から有利である．
POM，ポリアミド（PA）が歯車用樹脂として広く使用されてい
る．しかし，POMは接着が難しい点，PAは吸水率が大きく，高
湿度下では強度が低下する点に注意が必要である．また，高速回転
が必要とされる歯車には，さらに消音特性に優れた材料が求められ
る．ポリウレタン（PUR）は弾性，振動吸収性，耐摩耗性に優れ
ており，消音歯車材料として使われている．ただし，比較的低トル
クの伝達に限られる．
c .ばね 一瞬で立上がり，高速駆動を必要とするカメラのシ
ャッタ，絞り（aperture），クイックリターンミラーなどの駆動源
として，大きなエネルギーを瞬時に放出するばねは重要であった．
モータ駆動が一般的になった現在でも，速写性を重視する一眼レフ
カメラはモータでばねをチャージし，レリーズ時にばねでシャッ
タ，ミラーを駆動するものが多い．
コイルばね材料としては，静的荷重を受けるだけであれば硬鋼線
SW-Cを，耐疲労性を考慮するときはピアノ線SWP-B，ばね用
ステンレス鋼線SUS304- WPBが用いられる．また，導電性，非
磁性体が必要な場合はベリリウム銅線C1720Wが用いられる．
図1･61コンパクトカメラのズーム鏡筒
板ばね材料としてはオーステナイト系のばね用ステンレス鋼帯
SUS304-CSPが，また，耐力，引張強さを増したいときには析出
硬化系のばね用ステンレス鋼帯SUS631- CSPが用いられる．スイ
ッチの接片など導電性を要する板ばねには，ばね用ベリリウム銅板
C1720P，ばね用リン青銅板C5210P，ばね用洋白板C7701Pが
d.その他 カメラ特有の機構部品としてシャッタ，絞りの羽
根がある．羽根は平面性，平滑性，遮光性が要求され，かつ高速駆
動のために剛性と軽量性が求められる．鉄系の材料では板厚0.1
mm以下のみがき特殊帯鋼SK2Mが用いられる．特にみがき特殊
帯鋼の中でも特殊再溶解精練を行い品質向上を図ったESR鋼
（electro slag remelting）が使用される．アルミニウム合金板もシ
ャッタ羽根（shutter blade）に使われるが，熱処理をして鋼材に
匹敵する引張強さ，耐力をもたせたジュラルミンA2017P，超ジ
ュラルミンA2024Pなどが使用される．また，高速シャッタ用と
材料樹脂化の波は羽根にも及び，ポリエチレンテレフタレート
（PET）にカーボンブラック等を練り込んでフィルム状にしたも
の，またはPETフィルムに金属めっきを行い遮光性をもたせたも
のなどがコンパクトカメラ，コンパクトディジタルカメラに使われ
ている．これらは金属羽根より剛性は劣るが，軽量を生かしてわず
逆に超高速シャッタ用としての樹脂羽根も出現した．炭素繊維プ
リプレグ（prepreg）材である．炭素繊維をエポキシ樹脂で充てん
したもので，剛性と軽量を同時に満たす材料として期待されてい
る．この材料はプロフェッショナル用の一眼レフカメラに使われて
図1･62は一眼レフカメラ用シャッタユニットの分解斜視図であ
る．高速駆動される羽根に炭素繊維，超ジュラルミンなどが使われ
ている．
