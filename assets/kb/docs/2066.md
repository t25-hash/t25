# 1・12・5 電気回路材料

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・12 カメラ

カメラの電気回路基板は，大きくリジットプリント板とフレキシ
ブルプリント板（flexible print circuit board）に分けられる．リ
ジットプリント板は他の電子機器と同様に，ガラス繊維の布（織布
または不織布）にエポキシ（EP）を含浸させたベース材に銅はく
を張り付けたガラスエポキシ銅張積層板が多い．銅はくのパターン
層は1～8層程度まで用途に応じて使い分けている．
フレキシブルプリント板はカメラ特有の実装方法として広く用い
られている．狭い空間にむりやり電気回路を実装するため，折曲げ
や曲面が必須となったためである．はんだ付けの耐熱性，屈曲性な
どからポリイミド（PI ）をベース材として使ったポリイミド銅張
板が使われている．銅はくパターンを保護するカバーフィルムもベ
ースと同じPIを使い，厚み方向の対称性をもたせることで屈曲に
対する特性を向上させることができる．フレキシブルプリント板の
用途の中には電気部品を実装せず，単に配線，スイッチのためだけ
のものもある．こういった用途には耐熱性は必要ないため，PET
をベース材としたものもある．導体も銅はくだけでなく，カーボン
や銀をペースト状にしたものを印刷して形成する例もある．しか
し，一般的にフレキシブルプリント板はリジットプリント板に比べ
て高密度な部品実装性に劣る．このため，部品実装部は硬くして実
装性を上げ，それ以外はフレキシブルな両者の利点を兼ね備えたリ
ジットフレックスプリント板も存在する．
