# 10・2・1 座標(coordinates)

> **章**: 第10章 幾何
> **節**: 10・2 平面幾何(plane geometry)

a. 直角座標(orthogonal coordinates)平面上の一点Pの位置を定めるには、平面上に2本の相交わる直線を引き、交点からそれぞれの直線に平行に測ったPまでの距離およびを与えればよい。また2本の直線を座標軸(軸および軸)、交点を座標原点という。普通に用いるのは座標軸が直交する場合で、これを直角座標という。b. 極座標(polar coordinates)直角座標の代わりに極座標,θを用いることもある。は定点O(これを極という)からPまでの距離(動径)、θはOPがOを通る定直線(これを原線という)となす角(偏角)である。直角座標との間には次の関係がある。
