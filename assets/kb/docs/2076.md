# 10・2・4 二次曲線(quadratic curve)

> **章**: 第10章 幾何
> **節**: 10・2 平面幾何(plane geometry)

a. 二次曲線の種類Pから定点Fまでの距離PFと定直線までの距離PH との比が一定(PF:PH=ε)であるような点の軌跡をε<1ならばだ円、ε=1ならば放物線、ε>1ならば双曲線という。また定点Fを焦点、定直線を準線、εを離心率という。これらの曲線を総称して二次曲線または円すい曲線(conic section)という。後の名は同すいを平面で切るときに、その切り口がこれらの曲線のいずれかになることによる。c. だ円(ellipse)および双曲線(hyperbola)以下の式で複号の上はだ円、下は双曲線に用いる。だ円が主軸を切る長さAA=2を長径、BB=2を短径という(図II-10・15,II-10・16)。. 弦と直径曲線上に両端をもつ線分を弦、中心を通る弦を直径という。一つの直径が他の直径に平行なすべての弦を二等分するとき、これらの直径を互いに共役という。この性質はだ円の概略の形を描くのに利用される。双曲線と漸近線に交わる任意の直線UV を引けば、曲線と漸近線の間の線分UPおよびSV は相等しい(図II-10・16)。この性質は漸近線と曲線上の一点を与えて曲線を描くのに用いられる。漸近線上に両端をもつ軸に平行な線分は曲線によって内分され、二つの部分の積はに等しく、同様に軸に平行な線分は曲線によって外分され、二つの部分の積はに等しい。接線は∠FPH を二等分し、接線が軸と交わる点をT とすれば、PT は軸によって二等分される。. 弦と直径点Pを通る軸に平行な直線を直径という。直径はPにおける接線に平行なすべての弦(曲線上に両端をもつ線分)を二等分する(図II-10・17)。
