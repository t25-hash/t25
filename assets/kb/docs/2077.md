# 1・2・2 鉄道車両（rolling stock）

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・2 車両

鉄道車両には機関車（電気機関車，ディーゼル機関車等)，旅客
車（電車，気動車，客車)，貨車等の種類があるが，使用されてい
る材料はほぼ同じである．ここでは，接客設備をもつために材料の
バラエティーが広く，軽量化や高速化のニーズに応えるために新し
い材料の適用が進んでいる電車について概説し，最後にこれらに属
さない磁気浮上式鉄道車両（magnetic levitation railway vehicle）
に触れる．

合金製であっても，なおその比率が高いことがわかる．鉄系材料の
割合が高いことは質量の増加に結び付くが，鉄道が大量輸送機関
で，安全を第一に考えていること，車両のコスト低減が大きな要求
であること，軽量化に対する投資効果が自動車や航空機に比べて小
さいことなどが材料の使用に対して保守的にならざるを得ない理由
と考えられる．よりいっそうの軽量化を目指して新材料（ハニカム
サンドイッチパネル，FRP，CFRP，MMC，新しいアルミニウム
合金等）を車両に適用する研究も進められているが，コスト，保守
性，廃棄方法等の課題も多く，現状では，便・洗面所，水・汚物タ
ンクや内装部品等の一部に使用されている．
法規的には鉄道営業法第1条に定めた鉄道に関する技術上の基準
を定める省令に車両の火災対策に関する規定がある．
a.構体（body structure) 構体は車両の骨格に相当し，薄い
外板を張り巡らせそれを支えるように柱を溶接して組立てたセミモ
ノコック構造の筒状の構造物である．構体材料は長く耐候性鋼が使
われてきた．例として100系新幹線電車の車体構造を図1･19に示
す．近年，アルミニウム合金やステンレス鋼が多く使われるように
なってきた．高速走行のために特に軽量化が要求される新幹線電車
ではアルミニウム合金が多く使用されている．例としてJR西日本
500系，JR東海・西日本700系新幹線電車の構体構造と使用材料
を図1･20 ，1･21 に示す．省エネルギーのための軽量化やメン
テナンスコスト低減のため，ステンレス鋼は通勤電車に多く使われ
ている．例としてJR東日本209系電車の構体構造と使用材料を図
1･22 に示す．
b.台車（bogie) 図1･23 に0系新幹線電車用台車の構造を
模式的に例示する．
台車枠（bogie frame）は台車を構成する主要部材であり，鋼製
（SM400B，SM490B）のプレス溶接構造である．最近の主流は
まくらはりのないボルスタレス台車であるが，図1･23に示すボル
スタ付き台車のまくらはりにはアルミニウム合金製（A7N01）の
ものがある．軽量化のためアルミニウム合金やFRPを用いた台車
枠も試作されているが，強度，保守，コスト等の点から実用化され
ていない．
車輪材料はタイヤ鋼（STY80）であり，圧延後の熱処理により
数種類の区分がある．車輪（wheel ）と組合わされる車軸（axle）
材料は高炭素鍛鋼SFA60（在来線），機械構造用炭素鋼S38C
（新幹線）であり，焼入れ焼戻し，高周波焼入れ等の熱処理により
数種類の区分がある．
軸受（bearing）としては円筒ころ軸受＋スラスト玉軸受の併用
方式が使用されていたが，つば付き円筒ころ軸受や円すいころ軸受
が使用されるようになっている．軸受材料は高炭素クロム軸受鋼と
浸炭軸受鋼が用いられている ．軸箱（axlebox）は軸受を収納し
て軸受と台車枠の間の荷重を伝達する部品で，鋳鋼が用いられてい
るが，近年の新幹線電車では軽量化のためアルミニウム合金7050
が使用されている．
主電動機やディーゼル機関のトルクを輪軸（車輪を車軸に圧入し
たもの）に伝える歯車（gear ）は，大歯車には構造用炭素鋼S40
C等，小歯車にはNi -Cr - Mo鋼SNCM420等が熱処理して使用さ
れている．歯車や潤滑材を収納し，歯車間隔を保持する歯車箱
（gear case）は鋳鋼製であったが，近年は軽量化のため電車用に
Al -Si - Mg系の鋳物AC4Cが使用されている．
車両を安全に停止させるブレーキ方式は，車輪・レール間の粘着
力を利用する粘着ブレーキが主流で，粘着力を利用しない非粘着ブ
レーキ（渦電流，電磁吸着，空力等）は非常用のみに一部が使用さ

c .パンタグラフすり板（pantograph slider ) パンタグラフ
すり板はパンタグラフの最上部に取付けられ，トロリ線としゅう動
することにより，車両に電気を取り入れる部品である．したがって
機械的な強度，電気伝導性，耐摩耗性等に一定の性能が要求され
る ．おもに在来線ではカーボン系すり板，銅系焼結合金，新幹
線では鉄系焼結合金が用いられている．カーボン系すり板は焼成し
たカーボンに金属を溶融含浸して金属と複合化した材料が用いられ
ている．

（A2024系，A7075系）を使用したリベット結合によるものと，
アルミニウム合金（A5083系，A6N01系，A7N01系）を使用

に優れた部材とFSW（摩擦攪拌接合）が多用されている．また，
超電導磁石（SCM:superconducting magnet ）の磁界から客室を
遮へいする磁気シールド材として工業用純鉄（電磁軟鉄板)，電磁
鋼板（ケイ素鋼板）が使われている．
台車はSCMを取付けて浮上力，推進力を車体に伝えるとともに
車体へ伝わる振動の低減を図っている（図1･25) ．台車枠はアル
ミニウム合金（A7N01-T4，A6N01-T5等）を使用した板材，
型材の溶接組立構造または薄板リベット組立構造である．中速域か
らの保安ブレーキにも使用される支持輪のディスクブレーキ材には
C/Cコンポジットが採用されている．
超電導磁石（SCM）は，アルミニウム合金製の外槽の内部に4
個の超電導コイルを収納している．超電導線材はNb- Ti合金であ
り，超電導コイルの支持材料には極低温用FRP，ふく射熱シール
ド板には特殊なピッチ系CFRP等が使用されている．

では電気関係の慣例により「超電導」と表記した．
