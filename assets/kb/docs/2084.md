# 10・3・7 曲面(surface)

> **章**: 第10章 幾何
> **節**: 10・3 立体幾何(solid geometry,space geometry)

b. 柱面(cylindrical surface)任意の空間閉曲線K 上の各点を通り定方向に平行に引いた無限直線(母線、generating line)からなる曲面を柱面といい、K をその底線(base line)という。母線が軸に平行ならば、柱面の方程式はを含まない。c. すい面(conical surface)任意の空間閉曲線K 上の各点を定点V に結ぶ直線(母線)からなる曲面をすい面といい、K をその底線、V を頂点という。
