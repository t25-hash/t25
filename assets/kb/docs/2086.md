# 1・6・1 翼・胴体の主構造

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・6 航空機

運用中におけるき裂の発生・進展をある程度許容した損傷許容設
計（damage tolerant design）概念に適合する材料であることが要
求される．
a.金属材料 図1･38に示すように胴体外板および主翼下面
は高力アルミニウム合金2024- T3（AA記号による）が用いられ，
けた材，縦通材，主翼上面外板には同じく7075が用いられる．
2024はCuを4%程度含み，7075はZnを6%程度，Mgを
2.5%程度含む三元アルミニウム合金である．2024は7075に比べ
て静強度は20 %程度低いが，き裂進展抵抗が高く，したがって引
張りの疲労荷重を受ける構造の材料として優れている．一方，7075
は剛性・圧縮強度が高いため補強材や耐圧縮強度材料として優れて
いる．
なお，従来からよく用いられてきたこれらの材料に対して，比強
度・耐久性に改良が加えられた以下のようなアルミニウム合金の実
用化も進んできている．
2524，7475（エアバスA-340-600の外板部）
7349，7150（エアバスA-340-600の補強材）
また，レーザビーム溶接用のアルミニウム合金として6013- T6お
よび6056-T78が構造部材として使用される．
アルミリチウム合金：2000番系アルミニウム合金にリチウムを
2～2.5 %程度含有させ，比強度，比剛性をそれぞれ10%程度高め
た合金で代表例として2090，8090などがある．
b.複合材料 比強度が金属材料に比べて格段に高く，また疲
労などの繰返し荷重に対して不敏感であるところから複合材料が構
造部材として取り入れられるようになってきている．近年では，二
次構造部材のみならず，フロアビーム，翼けた，トーションボック
スなど一次構造（primary structure）部材（荷重支持構造）への
適用も進んできている．パネル構造材料として用いる場合には軽量
化と高い剛性を得るためにハニカムサンドイッチ構造にして用い
る．
GFRP（ガラス繊維強化プラスチック)：主として二次構造
（secondary structure）部材
CFRP（カーボン繊維強化プラスチック)：主として翼けた一体
構造，トーションボックスなどの一次構造部材
レジンとしては，エポキシ樹脂，ポリイミド（耐熱性）樹脂，
PEEK（ポリエーテルエーテルケトン：熱可塑性樹脂）などが代表
的構造用材料であり，強化繊維としては，ガラス繊維，カーボン繊
維，アラミド繊維，ケブラー繊維などがあり，レジンと繊維の組合
わせでさまざまな複合材料が提案され用いられている．
このほか，カーボン繊維やアラミド繊維層とアルミニウム合金薄
板を積層させて疲労き裂の進展を大幅に抑制したアラールやグレア
といった特殊な複合材も開発されている．グレアは特にエアバス
A-340-600の胴体上部の構造材料等として適用が拡大してきてい
る．
