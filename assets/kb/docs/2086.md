# 11・1・1 ベクトル(vector)の表示

> **章**: 第11章 ベクトルおよびテンソル
> **節**: 11・1 ベクトル代数

速度、加速度、力などのように大きさと方向(向きを含む)をもつ量をベクトルと呼び、これに対して質量、時間などのように大きさだけで定まる量をスカラ(scalar)という。ベクトルは図II-11・1に示すように空間の任意の1点、例えば、Oを通って、その大きさを表す長さをもち、その方向をもった矢印を付けた線分、すなわち有向線分(oriented segment)OA で表される。この有向線分がベクトルを表していることを明示するためOA と表す。これらはベクトルの絶対値(absolute value)と呼ばれる。
