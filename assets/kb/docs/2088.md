# 1・6・3 脚・パイロン構造

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・6 航空機

アルミニウム合金などの軽合金では，強度が不十分なため代替す
ることができず超高強度合金を用いるほかはない構造として，脚構
造およびエンジン懸下構造（pylon，パイロン）や各種主要継手構
造が挙げられるが，これらの構造には高張力鋼が用いられる．ただ
し，高張力鋼は一般に不安定破壊を生じる際の限界欠陥寸法が小さ
く損傷許容設計にはなじまないため，安全寿命設計（safe life
design）構造として健全性を保証する必要がある．
高張力鋼AISI4340，4330M，300Mマレージング鋼などが脚構
造のシリンダ，ピストン，トリクリンク，パイロン他の材料として
用いられている．
その他，客室内装用材料としてはクラッシュ事故などによって生
じがちな火災に対して搭乗者の脱出を助けるために不燃性・難燃
性・自己消火性材料，また，燃焼に伴う有毒ガスを発生しにくい材
料が必要であり，その特性に優れた材料としてポリウレタン樹脂系
材料が用いられる．
