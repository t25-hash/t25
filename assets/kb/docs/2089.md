# 1・7・1 エレベータ（elevator ）

> **章**: 第１章 各種製品に使用されている材料
> **節**: 1・7 エレベータ・エスカレータ

エレベータは縦の交通機関として高層建築ではいうまでもなく，
中低層の建物においても不可欠な設備となっている．
エレベータの駆動方式としてはいろいろなものが製造されている
が，現在一般に使用されているエレベータを大別すると，ロープ式
エレベータと油圧式エレベータに分類される．ここでは，最も一般
的な，ロープ式エレベータについて述べる．
a.全体構造 ロープ式エレベータの全体構造を図1･39に示
す．人（もの）が乗るかごと釣合いおもりが，つるべのごとくロー
プにより連結され，機械室に設置されている巻上機によって駆動さ
れる構造となっている．エレベータのかごと釣合いおもりは，ガイ
ドレールにより案内され上下に移動する．
エレベータの駆動は，モータの回転を減速機を介してシーブ（プ
ーリ）に伝え，シーブとワイヤロープの摩擦力によって行われる．
b.巻上機（traction machine) 高速のエレベータ（速度が毎
分150m以上）では，減速機のない巻上機（ギヤレスマシン）が
使用される．一般的にエレベータに使用されるモータは誘導電動機
で，これをインバータによって制御している．
また最近では，永久磁石を使用した同期電動機式の小型ギヤレス
マシンにより機械室を不要としたマシンルームレスタイプのエレベ
ータも実用化されている．

と配置，心綱の材質（繊維とスチール）が異なる．エレベータ用と
