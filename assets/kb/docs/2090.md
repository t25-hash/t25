> **節**: 1・7 エレベータ・エスカレータ

エレベータとともに縦の交通機関を担うエスカレータは，一定方
向の連続輸送能力に優れており，駅・空港はもとより，デパート・
ショッピングセンタ等の商業施設に至るまで，低層階用の輸送機関
として幅広く用いられている（図1･43）．
また，最近ではエスカレータ全体がカーブしたものや，中間部に
水平移動部分を設けた中間踊り場付きタイプ，さらには拡張変位す
る特殊ステップの搭載により，車いす使用者の利用を可能にした車
いす用ステップ付きタイプも見られるようになってきた．

a.全体構造 エスカレータの全体構造を図1･44に示す．
乗客が乗るステップは前輪（ステップチェーンローラ)・後輪
（ステップローラ）おのおの二輪の計四輪で支持されており，前輪
と後輪は軌条間隔の異なる別々のトラックレール上を走り，つねに
水平を保ちながら移動する．
前輪は上階と下階とを連結するエンドレスのステップチェーンに
取付けられており，ステップチェーンスプロケットのあるメインド
ライブをモータから減速機を介して駆動し，ステップチェーンを回
動することによりステップ全体を運転するものである．
b.ステップ（踏段，step) ステップは図1･45に示すように，
踏板，ライザ，ステップローラ（後輪）で構成されており，ステッ
プローラ以外はアルミニウム合金ダイカスト製やステンレス鋼製が
一般的である．最近では，軽量化やカラフル化を目的として樹脂製
ステップも開発されている．
また，ステップ間およびステップとスカートガードとのすきまに
靴などが挟み込まれないよう，踏板の端部に黄色の樹脂製デマケー
ションラインを設け，足元に注意喚起を促して安全性を高めてい
る．
図1･46立型マシニングセンタに使われている材料
c.ステップチェーン 各ステップを連結するステップチェー
ンは安全上重要な部品であり，建築基準法でその安全率が規定され
ている．また，スプロケットとかみあうときに，騒音が発生しない
ことも重要な要件である．一般的には，機械構造用炭素鋼等が用い
られる．
d.ハンドレール（手すり，handrail ) エスカレータ利用時に
乗客がつかむハンドレールは，ステップと同一方向に同一速度で運
転することが規定されており，スチールコードを心材とする綿帆布
の表面に化粧ゴムが施されているのが一般的である．
最近では，綿帆布の代わりにウレタンゴムを用いたものも見られ
るようになってきた．
e.欄干部（balustrade) エスカレータは欄干部の構造・意匠
により，ステンレスパネルタイプとガラスパネルタイプに分けるこ
とができる．
ステンレスパネルタイプは，インテリアパネルとしてステンレス
鋼板を使用しており，駅・空港等の交通機関向けに耐久性が重視さ
れる．一方ガラスパネルタイプはデパート等の商業施設向けに需要
が多く，ハンドレールの下方に蛍光灯のライン照明を施したものも
一般的である．
ガラスパネルタイプのガラスは，平面強化ガラスが用いられてい
る．
ここでは，より一般的なガラスパネルタイプのエスカレータのお
もな機器・部品について，使用されている材料とその材料に要求さ
れる特性を表1･9に示す．
