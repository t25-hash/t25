# 11・2・2 ベクトルの積分

> **章**: 第11章 ベクトルおよびテンソル
> **節**: 11・2 ベクトル解析

a. ストークスの定理(Stokes’theorem)ベクトルの場の曲面を,その周辺の曲線をとするときd=rotd=∇×d(II-11・35)左辺はの微小部分d方向へのの成分のに沿う線積分(line integral),右辺はの微小部分d(大きさは微小部分の面積に等しく,方向は微小部分の法線の方向)方向へのrotの成分のにおける面積分(surface integral)である.ただし,に沿ってまわる方向との法線方向とは右ねじと同じ関係にあるものとする.b. ガウスの定理(Gauss’theorem)ベクトルの場における体積を,それを囲む閉曲面をとするときd=div d=∇d(II-11・36)左辺はの微小面積d方向へのの成分のにわたる面積分,右辺はdiv のにわたる体積積分(volume integral)である.c. グリーンの定理(Green’s theorem)∇-∇d=-d(II-11・37)gradgradd+∇d=∇∇d+∇d=d(II-11・38)ここで,左辺は領域にわたる体積積分,右辺はを包む閉曲面にわたる面積分,は面の法線の方向である.また面の法線方向の単位ベクトルを用れば,=∇である.
