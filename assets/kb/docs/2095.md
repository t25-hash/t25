# 11・3・1 ベクトルと座標変換

> **章**: 第11章 ベクトルおよびテンソル
> **節**: 11・3 テンソル

ここに、,のようなラテン文字を指標あるいはインデックス(index)といい、1から3までの値をとるものと約束する。また、一つの項の中で同じ指標が二度現れるときは、その指標を1,2,3と変えて加え合わせた総和を表し(アインシュタインの総和規約、Einstein’s summation conven-tion)、そのような指標をダミーインデックス(dummy index)、一つの項に一度だけ現れる指標をフリーインデックス(free図-11・6 座標変換index)という。
