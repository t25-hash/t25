# 1・3・2 状態認識技術

> **章**: 第1章 概論
> **節**: 1・3 生産システムを支える技術

生産システムの知能化，自律化，協調化を実現するためには，生
産システムにおける個々の構成要素のさまざまな状態をインプロセ
ス認識するための状態認識技術を確立することが重要となる．その
ためには，個々のシステム要素に，高性能インプロセス状態認識機
能を搭載することが必要である．そうした計測情報処理系において
加工点とのインタフェースとなるインプロセスセンサ（in-process
sensor ）は，耐環境性，感度，信頼性，コンパクト性など，次世代
生産環境に適合するための基本的な要求仕様を満足する必要があ
る．
したがって，情報の入口に対応するインプロセスセンサには，以
下の特性が要求される．
a.高機能・多機能の具備 熟練技能者の有する経験と勘に基
づく多面的な観察機能，認識対象に適した感覚の自動選択機能，変
換機能，ならびに複合型・能動型センシング機能を備える．
b.高感度・高精度・高S/N比の具備 熟練技能者の有する
高精度な局所観察機能，外乱中の対象信号の自動抽出機能，加工中
の外乱から微小な状態変化を高応答で検出可能である．
c.高信頼性の具備 熟練技能者の有する経験と勘による自動
校正・照合機能，さらに厳しい生産環境で長時間の無人運転に対応
可能である．
d.環境適応性の具備 センサ特性に経時変化がなく，外乱に
強固で加工環境に悪影響を及ぼさないコンパクトな構造である．
e.経済性の具備 低コスト，省エネルギー型で，保守・管
用 されている．
理・点検が容易である．
近年，ナノメートルオーダの超精密加工や超高速加工といった在
来加工とは比較にならない極限加工に適用可能な高性能加工計測セ
ンサの要求が高まっている．そのような厳しい加工要求に対応する
ため，例えば図1･6に示すように認識対象となる加工点近傍におけ
る諸現象の特性を考慮した加工計測（measurement of machin-
ing）について検討する必要がある．超精密加工においては，加工
中に生じるさまざまな物理的挙動の変化が在来加工に比べて格段に
小であることから，既存のセンサでは微弱な状態変化を正確に検出
することが困難である．加工場を乱さずに高精度，高応答で状態認
識を行うためには，計測対象に適したセンサを可能な限り加工点近
傍に設置することが要求される．したがって，超精密加工用インプ
ロセスセンサには，コンパクトな一つのセンサからの信号を最大限
に利用して認識機能を高めることが要求される．例えば，図1･7の
築を行うことが必要である ．
ような複合センサ機能を組込んだセンサデバイスにより，加工点近
傍における力学的特性と熱的特性の経時変化を同時に監視する多機
能センシング やマイクロファブリケーション技術を適用して，
図1･8のような微小なインプロセスマイクロセンサ を加工点近
傍である工具刃先に搭載することが有効な手段である．
