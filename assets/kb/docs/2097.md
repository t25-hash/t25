# 11・3・3 テンソルとベクトル一次関数

> **章**: 第11章 ベクトルおよびテンソル
> **節**: 11・3 テンソル

ベクトル욡の成分がベクトル욡の成分の一次関数として욡=욡욢욢(II-11・48)のように表されるとき、ベクトル욡はベクトル욡の一次関数という。욡욢がテンソルであれば上の関係はつねに成立する。テンソル욡욢の成分に対して욡욢=욢욡が成り立つとき、このテンソルを対称テンソル(symmetric tensor)、また욡욢=-욢욡が成り立つとき、これを反対称テンソル(anti-symmetric tensor)または交代テンソル(skew-symmetric tensor)という。
