# 1・3・6 物と情報の搬送技術

> **章**: 第1章 概論
> **節**: 1・3 生産システムを支える技術

一般的にセル内外の物の搬送機能は，それぞれ内部搬送系と外部
搬送系に大分類される．工具交換装置やパレット交換装置などの内
部搬送系の高速化や高機能化の進展と並行して，外部搬送系につい
ては再構成可能なシステム構築を目的としたモジュラ構成搬送系や
自律分散系などが構築されている．半導体製造産業に代表されるク
リーンな生産環境では，非接触高速搬送の要求が高まり，搬送駆動
系や搬送案内系に磁場，近距離音場，ならびに静電場を利用した搬
送系の構築も行われている．
物の搬送だけではなく，生産システムにおいては，物と情報の流
れを有機的に結合し，それらを一体化して取扱うことによって所要
の製品や部品を高能率に生産できる．そのような観点から最近では
生産制御情報を工具ホルダや工作物パレットなどに搭載可能なコン
パクトで高性能なデータキャリヤ（data carrier ），データタグ，
IDタグが開発されている．例えば，工具管理には，書込みと読取
りの両方を自由にできるコンパクトなデータキャリヤの導入が普及
し，工具ホルダのIDタグに工具長，刃先データ，使用履歴などの
必要な工具情報が格納されている．
一方，流通業界で普及している一次元コードであるバーコードの
特徴を生かしながら，その弱点であるデータ密度やデータ修復能力
を改良した図1･10のようなQRコード（quick response code）に
代表される二次元コードの生産システムへの適用も行われてい
る ．
さらに，光通信技術の進展と大容量化する生産情報を合理的に処
理可能な生産環境を実現する観点から，制御技術，通信技術，建築
設備技術を一体統合化したビルオートメーションネットワーク技術
の確立とその標準化についても検討が進められている．
