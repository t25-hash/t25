# 11・3・8 一般のテンソル

> **章**: 第11章 ベクトルおよびテンソル
> **節**: 11・3 テンソル

共変基底ベクトル욡や反変基底ベクトルに関して욇욈욕,他の共変基底ベクトル욡′や反変基底ベクトル′に関して욢욅′で表される量が욢욅′=욢욖욅욗욆욇욈욕(II-11・68)なる関係で結ばれている場合,욇욈욕は反変1階,共変2階のテンソル(tensor of covariant order 1and contravariant order 2)あるいは混合3階テンソル(mixed third order tensor)と呼ばれる.また,욇욈욕を共変基底ベクトル욡や反変基底ベクトルに関するテンソルの成分という.反変階,共変階のテンソルも同様に定義される.反変階,共変0階のテンソルを反変テンソル(contravariant tensor),反変0階,共変階のテンソルを共変テンソル(covariant tensor)という.反変0階,共変0階のテンソルはスカラ,反変1階のテンソルは反変ベクトル,共変1階のテンソルは共変ベクトルである.
