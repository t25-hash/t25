# 1・4・4 JIT生産システム

> **章**: 第1章 概論
> **節**: 1・4 生産システムの歴史

a.トヨタ生産システム（かんばん生産方式) 多品種生産が
行われるとともに仕掛り品の在庫が注目され始め，生産システムの
基本的な評価が稼働率や生産量の増大から在庫数の削減，生産リー
ドタイムの短縮に関心が向けられ，必要なものを必要なときに，必
要なだけ生産するというJIT（just in time）の考え方が一般化し
ていく．JITの基本的な考え方は，1970年代にトヨタ自動車がか
その後，不良品を作らない品質管理に変化を遂げ，1966年に日
んばん方式として導入し，その後，1990年代においてほとんどの
企業に普及した．半世紀以上用いられてきたフォード生産システム
て大幅に変わっていったのである ．従来の生産方式では，生産
量を事前に決定してその量を最小コストで生産するために経済的な
ロットを算出して生産を行う方法で，生産設備，生産能力を優先的
に考慮して生産形態，受注方式，管理方法を考えるものであった．
一方，かんばん生産方式は，需要が発生した製品を補充していくこ
とを優先して，生産形態，受注方式，管理方式を決定する方法であ
り，生産設備を中心としてきた生産管理の根本を覆させる考え方で
あった．例えば，プレス加工のような段取り時間が大きい部品の場
合，生産ロット量を大きくすることで部品1個当たりのコストを削
減するのが従来の生産方式であるが，かんばん方式では，需要の量
によって生産ロット量が決まり，採算に合わない生産量の場合は，
段取り時間を短縮することで採算に合わせることになる．このよう
な方式で行うと仕掛り在庫が減少し，市場のニーズに敏感に対応す
ることが容易になり，死蔵品が減少し，タイミング良く製品を供給
することができる．
この生産システムを実現するために，部品や製品の引取りを表す
かんばんや段取り時間を短縮するクイック段取り，組立ラインへの
順序付け納入，負荷の平準化などさまざまな方法があり，これらを
総合して，生産システム全体のムリ，ムダ，ムラをなくす努力がな
されている．
b.セル生産システム（cell production system) 製品のライ
フサイクルが短くなってきた1990年代においては，短期間で設備
回収を行い，短期間で生産量が極端に変わることが原因で，従来の
重装備のラインやサイクルタイムを小さくした多人数のラインか
ら，数人あるいは一人で組立を行う組立ラインに変更されてきてい
る．この方式は，セル生産システム，モジュール生産システム，U
字ライン，屋台方式などさまざまな名前が付けられている．
このようなシステムは，基本的に，移動可能な作業台に工具や設
備を用意してステーションを作り，製品によって任意にこのステー
ションを組合わせて，作業セルを形成するもので，生産量の変動
に，セルの数で対応しようとするものである．このシステムの極端
な姿が，一人ですべての工程を担当して組立てるもので，一人生産
あるいは一人屋台などと称されている．このような生産では，一人
の作業者が多工程の作業に習熟しなくてはならないので，作業者の
モチベーションが直接的に生産性に影響してくる．また，作業が短
期間で変更されたりするので，品質上の問題も生じる可能性がある
ことから，この方式を用いるには，誤作業を防止する生産設備，製
品設計，品質を保証する設計などが要求される．この方式は，パー
ソナルコンピュータやFA機器などの組立に用いられている．
