# 12・1・1 標本空間と事象

> **章**: 第12章 確率および統計
> **節**: 12・1 確率

(1) 集合(set)とはものの集まりのことである。要素を一つも含まない集合を空集合と呼び、記号φで表す。(2) 実験、観察、計測、調査などで、結果が一定でなく、いろいろ変動する結果が観測される現象があるとき、起こり得るあらゆる結果をすべて集めた集合を標本空間(sample space)と呼び、これをΩで表す。標本空間の要素は基本事象あるいは標本点とも呼ばれる。われわれは、実験の結果があることがらを満足しているかどうか、つまり実験の結果がある条件を満足しているかどうかに興味がある。このことがらを事象(event)と呼ぶ。事象は標本空間Ωの中の一つの部分集合とみなすことができる。その集合は、その事象の条件を満足するような標本点の集合である。すなわち標本空間Ωは必ず全体集合になり、事象はΩの部分集合になる。(3) 事象を表す集合に対して次のような演算を定義する。これはの起こらない事象を表す。これらの演算に対して、次の公式が成立する。
