# 12・2・1 確率変数と確率分布

> **章**: 第12章 確率および統計
> **節**: 12・2 確率分布

このとき標本空間は実数の集合となり、それらの実数値に対する確率を表したものを、その確率変数の確率分布(probabilitydistribution)という。確率変数のとる値がとびとびの値であるとき、その確率変数は離散型であるといい、連続的な値をとるとき、その確率変数は連続型であるという。一般に確率変数には大文字を使い、それの実現値(sample value,realization)には小文字を使うというように、確率変数と実現値を区別すると便利である。12・2節では、原則としてこの区別をするが、区別をしないときもある。離散型確率変数の分布を離散分布、連続型確率変数の分布を連続分布という。確率分布の一般的な表現法として累積分布関数(cumulative distribution function)がある。
