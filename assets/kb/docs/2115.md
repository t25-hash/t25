# 12・3・2 マルコフ連鎖(Markov chain)

> **章**: 第12章 確率および統計
> **節**: 12・3 確率過程

現象の結果が有限または可付番無限個の状態,,...(これらに実数,,... を対応させる)であり、時間として考える範囲が0,1,2,...であるとする。この場合の確率過程は욧=0,1,2,...と書ける。確率過程욧において、任意のと任意の,,...,,に対し、=욡,=욢,...,욧=욅なる条件のもとで욧용=욆である確率が욅と욆のみに関係するとき、すなわち욧용=욆=욡,=욢,...,욧=욅=욧용=욆욧=욅==욆=욅=욅욆(II-12・30)であるとき、욧は(定常な推移確率욅욆をもつ)マルコフ連鎖という。これを現象としてみると、現時点に立って次の時点の事象の事後確率が、現時点の結果のみに依存し過去の結果には依存しないこと、またこの関係は、現時点がどの時点であるかということに無関係であることを意味している。の従う分布が=욢=α욢,∑α욢=1のとき、α욢を初期分布という。욅욆を욅から욆への推移確率(transition probabil-ity)といい、욅욆を行列の要素とする行列を推移確率行列という。の要素욅욆には욅욆≧0,∑욕욅욆=1という性質がある。定常な推移確率をもつマルコフ連鎖は、初期分布α=α,α,...と推移確率行列によって完全に決まる。ここに、αは列ベクトルαの転置を表す。例えば、1状態욡から出発してステップ後で状態욢に到達する確率は、推移確率行列の積욗の,要素である。2ステップ後の状態確率、すなわち욈の分布はα욗で表される。
