# 12・4・1 統計量の推定

> **章**: 第12章 確率および統計
> **節**: 12・4 統計

a. 統計的推測日常のデータ処理においてデータをとる目的を考えた場合、それは、そのデータが出てきた集団について推論をしたいためであるという場合が多い。つまり、一部分を見て全体を推論したいという形になっている。この場合、推測をしたいものの集まりを母集団(population)と呼び、母集団の推測をするために母集団から取出されるいくつかのもののことを標本(sample)と呼ぶ。標本は母集団からランダムに取出される。標本におけるものの数を標本の大きさもしくはサンプルサイズ(sample size)と呼ぶ。母集団についての推測は、ほとんどの場合、母集団分布の型を仮定し、分布の母数だけが未知であるとして、この未知母数を推定するという形をとることが多い、母集団分布が正規分布であるような母集団は正規母集団と呼ばれる。母集団から1個の標本をとるとき、その標本の値は確率変数とみなされる。そして、この確率変数の分布はこの母集団の母集団分布(population distribution)である。この言葉を用いると、統計的推測(statistical inference)問題は次のようになる。b. 点堆定未知母数θ[一般にはそれの関数θ]を一つの値として推定することを点推定(point estimation)と呼ぶ。標本から適当な統計量を計算し、この値を未知母数θ[またはθ]の推定値とする。ただし、このほかにもいろんな推定量の良さを測る尺度が提案されている。c. 区間推定標本から区間を作り、母数θ[またはθ]の真の値はこの区間の中にあるというような推論を区間推定(inter-val estimation)と呼ぶ。作られる区間は標本から計算されるものであるから、区間は標本ごとに変動し、ある場合には母数の真の値を含まないこともあるであろう。しかし、区間が母数の真の値を含む確率は一定に保証したい。作られた区間が母数の真の値を含む確率を信頼度(reliability)といい、作られた区間を信頼区間(confidence interval)という。信頼度は普通95%もしくは99%にとる(12・4・3項参照)。
