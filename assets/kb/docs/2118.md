# 1・3・3 内分泌（endocrine）系

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

生体を構成するすべての器官，組織，細胞は，化学的手段，すな
わち伝令物質（messenger ，生物学ではシグナルと呼ぶ）を用い
て，相互に情報交換を行っている．そのなかでも，血液やリンパの
流れを用いて遠く離れた臓器に作用する物質の分泌をホルモン分泌
と呼び，分泌する細胞のごく近傍の細胞に作用するものをパラクラ
イン分泌，自分自身に作用するものをオートクライン分泌と呼ぶ．
このようなすべてのシグナル伝達（signal transduction）は，伝令
物質に特異的に結合してその情報を細胞内部に伝える膜タンパク
質-レセプタを経由して行われる．この結果，標的細胞の内部で特
定の酵素の産生あるいは活性の増加が調節され，これが生体の活動
を変化させる．古典的にはいわゆる内分泌器官から分泌される伝令
物質をホルモン（hormone）と呼ぶが，その大部分が中枢神経系
や免疫細胞などによっても産生され生理活性をもつので，神経から
分泌される場合には神経伝達物質と呼び，免疫細胞の場合には，細
胞の種類に応じ，サイトカイン（白血球），リンフォカイン（リン
パ球），モノカイン（単球）などと呼ばれる．
生体は大きな自動制御系とみなせるが，その場合，体性神経系は
最も速い信号系であり，自律神経系はやや遅く，かつ，臓器の動作
モードを調節する．ホルモンなどによる調節を，それが血液の流れ
などによって伝ぱするので，液性の調節と呼ぶ．これは臓器組織の
再構築を含む最も長時間，かつ，非常に時間のレンジの広い範囲の
調節を受持っているといえる．液性調節［humoral control （regu-
lation）］の中枢は，間脳の一部である視床下部にある．視床下部
はその一部が連続している脳下垂体と共同で，末梢の内分泌腺のホ
ルモン産生を調節するので，これを視床下部-下垂体系とも呼ぶ．
視床下部の神経細胞は各種の末梢内分泌臓器の調節ホルモン（リベ
リンあるいはスタチンと総称する）を分泌し，その結果，標的とな
る内分泌臓器から分泌されるホルモンの濃度は視床下部の化学受容
器（レセプタ）で測定され，そのレベルに応じて調節ホルモンのレ
ベルを変化させる多重のネガティブフィードバック系が構成されて
いる．

通過し，細胞内のレセプタに作用するという特徴がある．コルチコ
ステロイドは，電解質コルチコイド（アルドステロン），糖質コル
