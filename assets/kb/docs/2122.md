# 1・3・7 腎・泌尿器系

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

動物は，体内環境（milieu interieur ）と体外環境を皮膚その他
の障壁で区分し，体内環境を一定の状態に維持する機能をもってお
り，これをホメオスタシス（homeostasis ）という．特に陸上に棲
息する動物では，体内に保持する水分（体重の60 %）の量を保ち，
体液の電解質の組成などを維持することは生存に欠かせない．こう
した調節の結果として排出される電解質，なかでもタンパク質代謝
の過程で発生する窒素代謝産物（尿素，urea）などの不要な物質
を血液から排出するための器官系を泌尿器系（urinary system）と
いう．泌尿器系の一部は生殖器（genital organ）と重複しており，

腎臓は，自己の血流を制御対象として血液電解質の組成をコント
ロールし，血圧を調節する内分泌器官でもある．この系をレニン-
アンギオテンシン系と呼び，高血圧症（hypertension）に関連して
重大な意義をもつ．このほか，腎臓は造血機能（hematopoiesis ）
を促進するエリスロポエチン（erythropoietin）も分泌し，赤血球
の量を調整することで血液の組成をコントロールしている．
尿管を通って膀胱に送られた尿がある程度の量になると，尿意を
もよおし，排尿が起こる．膀胱壁とその出口の括約筋は平滑筋であ
り，骨盤腔内に分布する自律神経の支配を受けている．ここは，腹
腔内の手術などの影響で機能が障害されやすく，排尿障害
（micturition disturbance）が起こる．
ネフロン（nephron）は1個の腎臓に約100万個存在するが，損
傷しても再生しないため，慢性化する腎臓の病気は遅かれ早かれ腎
不全（renal failure）を起こし，人工腎臓（artificial kidney）に
よる人工透析（artificial dialysis ）が必要となる．人工腎臓は，特
にカリウムなどの電解質の調節，尿素などの老廃物の除去には有効
な手段であるが，当然のことながら上述した内分泌機能をもたない
ので，長期の人工透析ではこのことが問題となる．
