# 13・3・1 関数近似の型

> **章**: 第13章 数値解析
> **節**: 13・3 関数近似と補間

図II-13・1のように離散点욢=1,2, ...において関数の値욢=욢が与えられているとする.この離散点욢を標本点あるいは選点という.二次元や三次元の計算領域を格子(grid,mesh)に分割するときには욡,욢や욡,욢,욅を格子点という.ここで,連続な関数を別の関数系욅,=0,1,2,...の結合で近似することを考える.욅としては多項式関数系がよく用いられる.そのほかには直交多項式による近似があり,これを用いた計算法はスペクトル法と呼ばれている(13・6節参照).最も基本的な形式は線形(linear)結合++...+욈욈である.非線形結合を含む近似の解析は非常に難しくなるが,有理図-13・1 離散情報を用いた近似関数型(rational)結合++...+욈욈++...+욇욇が適する場合もある.離散情報である욢,욢群を用いて係数욡や욡を決める考え方としては,次のようなものが代表的である.a. 補間法図II-13・1の破線のように,標本点において욢に(丸め誤差を除いて)一致するように係数を決める方法である.このとき近似式は補間式となる.b. 最小二乗法図II-13・1の一点鎖線のように,標本点で与えられた値には必ずしも一致しないが,ある区間,でと近似式の差の二乗を積分したもの(誤差のノルム)を最小にする関数近似法である.
