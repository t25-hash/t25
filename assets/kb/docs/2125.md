# 13・3・2 補間法

> **章**: 第13章 数値解析
> **節**: 13・3 関数近似と補間

数値解析で補間法を議論するおもな理由は次の二つである。この計算は、が標本点の間にあれば補間(内挿、interpola-tion)、外にあれば補外(外挿、extrapolation)と呼ばれる。第二には、補間式が多くの数値計算法を誘導するのに用いられるからである。特に、数値微分や数値積分の多くは補間式から導くことができる。ラグランジュ補間は標本点の数が大きいほど誤差が少ないように見える。しかし、高階微係数が大きくなる(なめらかでない)関数には標本点が多ければよいとは限らない。また、標本点列の両端に近づくほど補間精度は低下する。さらに、標本点が多すぎると計算効率も低下する。参照される標本点をステンシルということがある。これがエルミート補間(Hermite interpolation)公式である。エルミート補間を区分化する。c. スプライン補間区分補間法の一つとして、スプライン補間(spline interpolation)法がある。スプラインは製図で用いられる雲形定規に由来している。スプライン補間では、標本点は節点(node)または結び目(knot)とも呼ばれる。最も一般的に用いられるのは三次のスプラインである。ここでは簡単のため二次元で説明するが、以下に示す方法は三次元以上にも容易に拡張が可能である。この方法の問題点は、精度が低いこととなめらかでないことである。単純に精度を高める目的ならば、図II-13・3のように、より高精度な一次元補間を各方向に順次適用する方法がある。なめらかにする方法としては、双三次補間法がある。これらの微係数は例えば後述の差分法によって近似的に求められる。このほかには双スプライン補間法がある。
