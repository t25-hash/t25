# 1・1・2 人間社会における設計の重要性と技術者倫理

> **章**: 第1章 総論
> **節**: 1・1 設計とは

1･1･1項で述べたように，設計がいままで世界になかったものを
作り出すことであるという事実を理解すれば，設計を含む技術活動
が文明の進歩そのものであることがわかる．人類古来の，何かをし
たいという強い願望がアイデアに結びつき，飛躍を呼んで人類が進
歩してきたわけである．
1･1･1項では，設計の本質は性質からモデルまたはシステムへの
飛躍であるが，その後に続く設計の検証過程や，最初の飛躍によっ
て生まれたモデルを修正しながら望みの性質へ導く過程も設計にと
って非常に重要であると書いた．
この部分も含めて設計を定義すると，「作り始める前に考えるこ
と」となる．最初の飛躍が設計の本質であり，これを誤ると近傍の
最適解に至っても他の解に負けてしまうことがあるのと同様に，技
術においては設計の検証を含む設計の段階が製品の競争力，製品の
もつ性格を決めてしまうところがある．
技術は設計だけではなく，調達，製造，流通，保守，回収，リサ
イクルなど製品のライフサイクル全体を指すが，その中で技術の出
発点である設計のもつ意味は非常に重要なのである．この設計の早
い段階で製品のライフサイクル全体を考えなくてはならないという
のも，設計に対する新しい考え方であり，『機械工学便覧』でもそ
のような視点に立って設計を考えている．
設計が技術活動の最上流であることから，当然，担当する技術者
には技術者倫理（engineering ethics ）が要求される．設計者は自
分が設計したもの（それはつねにいままで世の中になかったもので
ある）が，社会に出ていったときの影響を予測しなくてはならな
い．社会に対する影響のうち，負の側面をも目を背けずに予測する
ことが大切である．
環境に与える影響については3･6節で扱う．しかし社会に対する
影響はそれだけではない．むしろその製品，技術の本来もっている
機能による影響が大きな場合もある．身近なところで携帯電話やパ
ーソナルコンピュータおよびソフトウェアは，そこに新しい機能を
追加するたびに社会を変化させる原動力になっている．
その意味では，地味ではあるが工作機械などの機械類も同様であ
る．いままであった限界を超えて加工が可能になることは，新しい
世界を築くことである．
このような，いままでなかったものを世の中に出すということが
社会に与える変化に対して，技術者は責任をもたなくてはならな
い．これは，製品データでうそをついてはならないということや，
機能を誇大に伝えてはならないというようなことよりも，いっそう
技術の本質に近いところでの技術者の責任である．
このようにいうと，いままで世の中になかったものを社会に出す
といえるほどの設計はしていないと考える人もいるだろうが，従来
のものからのささやかな変更であっても，そこで安易な材料を用い
ることで顧客の健康被害をもたらす可能性もあるし，不用意な設計
のために顧客がけがをするという危険性も発生する．通常はカバー
に覆われているという理由でバリを取らなかっただけで，そのよう
な危険性が生じることもありうる．
すべての設計は，いままで世の中になかったものを世の中に出す
という活動なのである．それだけに設計者には細心の注意が要求さ
れる．当然払うべき注意を怠ることは技術者倫理の問題である．
