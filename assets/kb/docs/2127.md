# 13・3・4 最小二乗法(least square method)

> **章**: 第13章 数値解析
> **節**: 13・3 関数近似と補間

最小二乗法は関数近似にも有用であるとともに、測定値の統計分析や実験式の構築にも用いられる手法である。この性質を利用する方法として、関数や領域に応じてルジャンドル(Legendre)、ラゲール(Laguerre)、エルミート、チェビシェフ(Chebyshev)の各多項式が用いられる。数値微分(numerical differentiation)または数値積分(numer-ical integration)は、工学に現れる微分方程式を数値的に解いたり、面積分や体積積分を求めたりするときに用いられる。原理的には、与えられたデータをもとにして関数の近似式を作り、それを微分または積分すればよい。近似式としては補間多項式が用いられることが多い。
