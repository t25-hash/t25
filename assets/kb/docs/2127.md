# 1・1・3 設計教育の今後

> **章**: 第1章 総論
> **節**: 1・1 設計とは

高等教育機関における設計教育（design education）が変わろう
としている．1999年11月に日本技術者教育認定機構（JABEE:
Japan Accreditation Board for Engineering Education）が発足
し，準備期間を経て2001年から教育プログラムの審査認定を開始
した．この審査認定への申請件数は順調に増加しており，2004年
の審査を終えた時点で総数186プログラム，機械系で33プログラ
ムが認定されている．この審査認定の画期的な点は，工学部，理学
部，農学部などの教育プログラムが申請すれば，技術者教育，つま
り技術者という専門家を育てる教育になっているかどうかを審査認
定することである．その趣旨に従って工学部以外から理学部物理学
科が1プログラム，農学部，水産学部などから8プログラムが認定
されている．ここで従来の工学教育に対して付加された要求は主と
して次の点である．
(１)幅広い視野をもつための教養教育
(２)技術者倫理教育
(３)デザイン教育
(４)コミュニケーション教育
(５)継続的，自主的に学習する習慣づけ
(６)仕事を計画的に進め完遂する能力の付与
このうち，(１)は従来から教養課程あるいは共通教育として行わ
れており，(４)～(６)は主として卒業研究を通じて身につけさせて
いたといえるが，(２)(３)は新しい要求であった．しかし多くの教
育機関では，技術者倫理教育開講の必要性は感じたが，デザイン教
育については，従来から十分に行っているとの見解で特に対応をと
るところは少なかった．
JABEEは2001年に国際的な技術者教育認定制度の相互承認シ
ステムであるワシントンアコードに暫定加盟し，2005年6月には
日本を代表する技術者教育認定機関として正式加盟を果たした．こ
の正式加盟の審査の過程で，JABEEのデザイン能力教育に対する
審査の甘さが指摘された．
それはとりもなおさず，新しいものを作り出すという設計本来の
訓練が行われていないという指摘であった．この指摘を受けて，
JABEEでは2004年からデザイン教育について審査のポイントを
改善し，本来の設計の教育が行われているかどうかを確認すること
になった．これによって今後の高等教育機関における設計教育が変
わっていくと考えられる．
それでは設計の教育はどのように行われるべきであろうか．1章
で述べてきたことからわかるように，設計は新しいものの創造であ
る．創造の教育という意味では，設計の教育は絵画の教育と共通す
るところが多い．
例えば花瓶に入った花を写生することを考えたときに，これが最
もよい絵であるという絶対の正解があるわけではない．設計も同様
で，例えば1kgの質量を1秒以内に10cm動かす装置という設計
課題には絶対の正解はない．とにかく動くという軽量設計もあれ
ば，重量級だが精度のよい設計もある．これは数学や力学の問題と
の明らかな違いであり，このような教育は従来型の教育，特に工学
教育では不得意な分野であると考えられる．
それは工学教育が主として物理教育を基礎においているからであ
る．物理教育では真理を教えるのであるから答えが二つあってはな
らない．数学に関しても同様である．
このあいまいな設計の教育を絵画の教育とのアナロジーでかくと
図1･6のようになる．初めに絵の具の使い方，筆の使い方に相当す
る基本スキルとしての，設計情報を表現するための製図法やCAD
を勉強しなくてはならない．次には課題について設計をしてみると
いう段階に入る．
まず写生のように，設計課題を学生に与える．これに対する解答
を，できれば絵画と同様に個人個人の学生が用意する．もし学生の
数が多かったら，グループで相談して一つの解答とする．教員の仕
事は絵の先生の働きを考えれば類推できる．
