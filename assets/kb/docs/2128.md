# 13・4・1 数値微分

> **章**: 第13章 数値解析
> **節**: 13・4 数値微分および数値積分

ただし、補間と同様に、多くの標本点を用いて高次の多項式を構成することが必ずしも良い結果を与えるとは限らない。そこで、通常は区分化した補間多項式に基づいて数値微分法を導く。b. テイラー展開法近傍の標本点にテイラー展開(Taylorexpansion)を適用して数値微分を導出する方法がある。右辺第2項以降はテイラー展開の高次項を省略したことに起因する打切り誤差である。等間隔に配置された標本点における代表的な差分近似式を示しておく。これらは二次精度であり、標本点の配置と係数の重みがに関して対称であることから二次中心差分と呼ばれる。さらに標本点を増やして低次の誤差項を消去すれば、より高次精度の差分式を導くことができる。一般に、同数の標本点を用いるならば中心差分が最も精度が高い。しかし、標本点の広がりは計算量の増加だけでなく、境界近傍での取扱いを複雑にする。精度次数(order of accuracy)とともに誤差の性質にも注意しなければならない。誤差が奇数階数になるのか偶数階数になるのかは数値計算の安定性に密接に関連する。また、高階の微係数が大きくなる(なめらかでない)関数では、高次精度の方法が良い結果をもたらすとは限らない。c. コンパクト差分比較的狭いステンシルで高精度化を実現する方法として、標本点配置が等間隔の場合には、コンパクト差分(compact finite difference、あるいはパデ法)がある。これは、微係数の近似式が陽に与えられるのではないから、一種の陰的な差分法である。右辺は±1,±2,±3のステンシルを用いる二次精度中心差分法の線形結合になっている。コンパクト差分の特徴として、細かい変動に対する解像度が比較的高いことが挙げられる。d. 座標変換による数値微分標本点が不等間隔に配置されている場合、前述したラグランジュ補間多項式の微分のほかに、座標ξで等間隔となるように写像する方法も考えられる。ddξについては等間隔の差分を適用すればよい。ほとんどの中心差分法は第3式を再現する。しかし、前述の差分法は第1,2式の関係を満たさない。
