# 13・4・2 数値積分

> **章**: 第13章 数値解析
> **節**: 13・4 数値微分および数値積分

a. 台形則とシンプソン則数値積分も補間や数値微分と同様にすべての標本点を通る補間多項式ではなく、区分補間多項式に基づいて計算する場合が多い。これは図II-13・4に示す折れ線(破線)の下の台形の面積である。これはシンプソンの1/3則[Simpson’s (1/3)rule]と呼ばれる。b. 高次精度の数値積分高い次数の積分公式を構成するには、区分ラグランジュ補間多項式の標本点を追加すればよい。これはニュートン・コーツの積分公式[Newton-Cotes (integration)formula]と総称され、=1のとき前出の台形則、=2のときシンプソンの1/3則であり、=3のときはシンプソンの3/8則となる。
