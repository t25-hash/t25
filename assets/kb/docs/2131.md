# 13・5・1 基本的事項

> **章**: 第13章 数値解析
> **節**: 13・5 代数方程式の数値解析

一般に、代数方程式(algebraic equation)に限らず非線形方程式も含めて、方程式=0の根αを数値計算で求める際には、ニュートン法(Newton’s method)で代表されるように、最初に求める根の近似値を適当に与え、それを逐次修正していく方法が広く用いられる。この場合、収束解を確実に得るためには、最初の近似値が解に十分近いことが重要であるため、あらかじめ解の存在範囲や個数を予測することは有益である。したがって、奇数次の実係数代数方程式は、少なくとも一つの実根をもつことになる。
