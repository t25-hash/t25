# 13・6・1 スペクトル法による離散化(spectral digitizing)

> **章**: 第13章 数値解析
> **節**: 6・2 項に解説する。また、非線形の偏微分方程式を数値的に解く場

スペクトル法を用いて、以下の微分方程式を離散化し解を数値的に求める。さらに、スペクトル法では試行関数および重み関数は完全系であり、滑らかな任意の関数について次の関係が保証される。3変数関数のときも同様に拡張できる。多くの完全直交関数系の中でスペクトル法において頻繁に用いられものは、三角関数(フーリエ級数、Fourier series)、チェビシェフの多項式(Chebyshev polynomials)、およびルジャンドルの多項式(Legendre polynomials)である。三角関数を用いたスペクトル法はフーリエ・スペクトル法と呼ばれ、周期境界条件を用いた一様等方性乱流の直接数値シミュレーションに用いられる。これに対して、チェビシェフ多項式、ルジャンドル多項式は、壁面乱流などの非周期的境界条件の乱流の計算に用いられている。
