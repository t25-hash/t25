# 13・6・3 擬スペクトル法

> **章**: 第13章 数値解析
> **節**: 13・25 )に代入すれば、近似解が求まる。

スペクトル法は、比較的少ない未知数(スペクトル)でかなり正確な解を得られるが、非線形項が含まれる場合には、そのスペクトルを求めるのに大変に時間がかかる。高速フーリエ変換(FFT :fast Fourier transform)を用いることで、物理空間で非線形項を、その微分をスペクトル空間で効率的に計算できる。これは、擬スペクトル法と呼ばれる。擬スペクトル法は、非物理的誤差であるエイリアス誤差が発生する。この演算にはオーダの演算回数を要し、計算効率が悪い。この方法によれば、演算回数はlogのオーダである。
