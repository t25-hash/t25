# 1・2・1 測定の尺度

> **章**: 第1章 計測とは
> **節**: 1・2 尺度と次元

何らかの量を計測するということは，その背後に量的概念が存在
し，その概念に基づいてその量に対応する数値などのラベル付けが
なされる．量的概念とラベル付けした数値との間の関係により，こ
の数値の並びである尺度（scale ）のもつ性質は左右される．数値
の同一性，順序性，等質性，加算性などの性質により，尺度は名義
尺度，順序尺度，間隔尺度，比例尺度の大きく四つに分類される．
名義尺度（nominal scale）は，対象を分類し，相互の区別を，
数を利用して行うものである．分類された各群においては，同一群
には同一の数を用い，逆に異なる群には同一の数を用いないように
する．数の大小や順序はなんら量的意味をもたず，数の同一性のみ
が意味をもつ．試料の番号や背番号などはこの例である．
順序尺度（ordinal scale）は，対象をその大小や優劣などの観点
で並べ，その並べられた順に合わせて数を順に割り当てるものであ
る．数の順序性を利用した尺度であり，相互の大小（優劣）関係の
みを表す．モース硬度などがその例であり，相互に引っかき試験を
することによりその順序が定まる．しかし，その順序を表す数は順
序以外の量的意味をもたない．
間隔尺度（interval scale）は，対象に与えた尺度値間の差が等
しければ，その差に対応する対象の量的違いもつねに同一の物理的
意味をもつ尺度であり，数の等質性を利用したものである．セルシ
ウス温度目盛はこの例であり，20℃と40℃は，20Kの差に相当す
るポテンシャルの違いであり，決して40℃は20℃の2倍のポテン
シャルを有しているわけではない．
比例尺度（ratio scale ）は，対象に与えた尺度値の数が，和や比
など数の演算結果によって得られた数値においても量的意味をもつ
ものである．長さ，体積，質量，電流，電圧，時間，絶対温度など
はこの例である．
