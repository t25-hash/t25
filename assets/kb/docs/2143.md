# 1・1・3 設計教育の今後

> **章**: 第1章 総論
> **節**: 1・2 中左端のシステムから当面対象としない部分を除いてモデルを

高等教育機関における設計教育(design education)が変わろうとしている。1999年11月に日本技術者教育認定機構(JABEE:Japan Accreditation Board for Engineering Education)が発足し、準備期間を経て2001年から教育プログラムの審査認定を開始した。この審査認定への申請件数は順調に増加しており、2004年の審査を終えた時点で総数186プログラム、機械系で33プログラムが認定されている。この審査認定の画期的な点は、工学部、理学部、農学部などの教育プログラムが申請すれば、技術者教育、つまり技術者という専門家を育てる教育になっているかどうかを審査認定することである。その趣旨に従って工学部以外から理学部物理学科が1プログラム、農学部、水産学部などから8プログラムが認定されている。ここで従来の工学教育に対して付加された要求は主として次の点である。(1) 幅広い視野をもつための教養教育1―3(2) 技術者倫理教育(3) デザイン教育(4) コミュニケーション教育(5) 継続的、自主的に学習する習慣づけ(6) 仕事を計画的に進め完遂する能力の付与このうち、(1)は従来から教養課程あるいは共通教育として行われており、(4)~(6)は主として卒業研究を通じて身につけさせていたといえるが、(2)(3)は新しい要求であった。しかし多くの教育機関では、技術者倫理教育開講の必要性は感じたが、デザイン教育については、従来から十分に行っているとの見解で特に対応をとるところは少なかった。JABEEは2001年に国際的な技術者教育認定制度の相互承認システムであるワシントンアコードに暫定加盟し、2005年6月には日本を代表する技術者教育認定機関として正式加盟を果たした。この正式加盟の審査の過程で、JABEEのデザイン能力教育に対する審査の甘さが指摘された。それはとりもなおさず、新しいものを作り出すという設計本来の訓練が行われていないという指摘であった。この指摘を受けて、JABEEでは2004年からデザイン教育について審査のポイントを改善し、本来の設計の教育が行われているかどうかを確認することになった。これによって今後の高等教育機関における設計教育が変わっていくと考えられる。それでは設計の教育はどのように行われるべきであろうか。1章で述べてきたことからわかるように、設計は新しいものの創造である。創造の教育という意味では、設計の教育は絵画の教育と共通するところが多い。例えば花瓶に入った花を写生することを考えたときに、これが最もよい絵であるという絶対の正解があるわけではない。設計も同様で、例えば1kg の質量を1秒以内に10cm動かす装置という設計課題には絶対の正解はない。とにかく動くという軽量設計もあれば、重量級だが精度のよい設計もある。これは数学や力学の問題との明らかな違いであり、このような教育は従来型の教育、特に工学教育では不得意な分野であると考えられる。それは工学教育が主として物理教育を基礎においているからである。物理教育では真理を教えるのであるから答えが二つあってはならない。数学に関しても同様である。このあいまいな設計の教育を絵画の教育とのアナロジーでかくと図1・6のようになる。初めに絵の具の使い方、筆の使い方に相当する基本スキルとしての、設計情報を表現するための製図法やCADを勉強しなくてはならない。次には課題について設計をしてみるという段階に入る。まず写生のように、設計課題を学生に与える。これに対する解答を、できれば絵画と同様に個人個人の学生が用意する。もし学生の数が多かったら、グループで相談して一つの解答とする。教員の仕事は絵の先生の働きを考えれば類推できる。教員は一つずつの作品を見て、その作品のよいところを見つけてまずほめる。「この設計は非常に機能的にできているので、このまま設計を進めれば動くものができそうである」などの言葉である。そのうえで改良したほうがよいところを指摘する。例えば、「全体として精度をねらった設計なのに、この部分の精度が出しにくそう基本スキルの習得習作指導者による添削よい作品を見ること制作自己評価図1・6 設計の学習1―4設計工学だから少し考えたほうがよい」、または「この設計のままだとベースの重量が大きくなりそうで、全体設計の考え方の中でバランスが悪い」などのアドバイスである。絵画教育では、初心者に対する絵の教育ができる人は絵のかける人である。設計も同様で、できるだけ多くの設計を手がけた人が教えることが望ましい。教員のアドバイスに応じて設計変更した結果、学生がなるほどと思うような改善が感じられることが大切である。このような経験を何度かすると、学生にもだんだんよい設計というものがわかってくる。これはよい絵を見る目が育つのと同じことであり、図1・6の右半分に移行したことを意味する。そのようになると、学生はよい設計(good design)を見ることによって自ら学ぶことができるようになり、家庭電気製品を見ても、機械装置を見ても、その中によい設計を感じ取ることができるようになる。このあたりまでが高等教育機関に要求される設計の教育であろうと考える。これ以後は実社会において、より複雑なシステムの設計に接し、その中で先人が行った優れた工夫を感じ取ることにより、能力を洗練させていくことが重要である。具体的な設計教育については5章で取り上げることになる。
