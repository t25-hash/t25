# 1・2・3 時間軸による設計の種類

> **章**: 第1章 総論
> **節**: 1・2 設計の構成

製品開発は,時間軸では製品の企画,概念設計,詳細設計,生産設計というような形で進められることが多い.この中で特に企画の方法論を3章で取り上げている.その理由は本章の初めから述べているように,これからの日本の産業を発展させるためには,技術によって何を実現するのかを決めることが最も大切だからである.グローバルな競争を勝ち抜くためには,もはや日本には安さという武器はない.世界に先駆けて魅力的なもの(ハードウェア,ソフトウェア,サービス)を提案する以外にないのである.技術者は今後,世界に向けて何が提案できるかを真剣に考えなくてはならない.これこそが技術の本質である.これが決まれば,概念設計へと進むことはそれほど困難なことではない.目標が明確になるからである.もちろんいままで世界になかったものを提案するのであるから,要素開発を伴うことが多く,その苦労は覚悟しなくてはならない.いままで世界になかったものを世に出すのであるから,自らが道を切り開かなくてはならないのは必然のことである.この概念設計の段階で,その技術が世の中に出たときの総合的な影響を考慮しておく必要がある.個人の権利を侵すようなおそれはないか,差別を感じさせるような内容を含んでいないか,反社会的な用途に使われるおそれはないかなど,開発しているものによってさまざまな注意点が存在しうる.さらに設計を進めるうえでは,使用する材料などに問題はないか,最終的には廃棄するのか,回収して部品のリサイクルや資源への還元を行うのかなど,環境への負荷を十分に考慮する必要がある.このような部分については3・6節で取り上げている.同様に設計を具体化する段階ではDfX が重要である.この手法についても3章で取り上げる.DfX はdesign for something であるが,最も早くから取り上げられていて発展が著しいのはDfM(design for manufacturing)である.設計の一つの重要な側面である,できるだけ早い段階からさまざまなことを考えるために役立つのがこのDfM を含めたDfX である.
