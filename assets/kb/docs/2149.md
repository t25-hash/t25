# 1・2・5 設計の種類

> **章**: 第1章 総論
> **節**: 1・2 設計の構成

設計には少なくとも新規設計と改良設計とがあるといわれる。し1―6設計工学かし新規設計でなく従来の設計を下敷きに進めた改良設計の場合でも、必ず新しい部分が存在し(それでなくては設計業務の必要はない)、そこには前述したようなさまざまなリスクが考えられる。設計者がつねに技術者倫理をわきまえる必要がある理由は、この点にある。このようなリスクをできるだけ排除し、設計の効率を上げる工夫が存在する。それは、多種の製品の設計に共通点を多くもたせることによって設計を効率化する方法である。ねじを共通部品として量産に対応しようとした19世紀の工夫に始まり、現在では複雑な部分も共通な設計で作ることができるように工夫が重ねられている。これについては、4・4節で取り上げている。ここではあるまとまった製品群に共通して利用可能なように、初めから配慮して設計するなどの設計技術が取り上げられている。
