# 1・3・1 設計の役割

> **章**: 第1章 総論
> **節**: 1・3 設計工学の形成と展開

ものづくりや工学における重要な位置を占めることに疑う余地はない.しかしながら,設計を工学的な見地から体系立てて理解したり,その合理性を向上させるために何らかの有用な理論や方法論を構成したりしようとする努力が行われるようになったのは,比較的最近になってからのことである.この点は今回の便覧改訂でβ1編「設計工学」が新たに設けられる背景でもある.本節ではそれについての経緯を概観しておく.繰返しになるが,設計は何らかのもの(製品などの人工物)の実現に向けたさまざまな計画などを立案する過程である.工学のさまざまな知識はそれぞれに何らかの設計解を合成するための体系となっていて,設計の質を向上させるうえで各分野の知識やそれに基づいた方法が果たしている役割は大きい.しかしながら,ものの内容やそれがもたらす物理現象やふるまいが個々の分野で培われた知識を連係させることのみによって決定できるのであれば,設計という行為を改めて論じる必要はない.永年にわたる科学技術上の知識の蓄積は,ものやそれがもたらす効用や弊害の可能性を拡大させており,作り出したものが優れているかどうかについては,前提となる知識や方法の内容のみならず,計画立案を進めていくプロセスに依存する部分も大きくなってきている.すなわち,個別のコンテンツからは離れてそのようなプロセスを科学的に構成し体系的に実施する必要性が,設計工学(design engineering)を成立させている.
