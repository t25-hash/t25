b.心臓の力学（cardiac mechanics ）
.心臓の構造 哺乳類の心臓では，右心系（右心房と右心
室とで肺に血液を送る）と左心系（左心房と左心室で肺以外の身体
全体に血液を送る）から構成され，二つのポンプが直列に接続され
ている（図2･92) ．肺循環の入口の圧力は15～20mmHgであ
るのに対し，体循環の入口は80～120mmHgという高い圧力であ
るため，左心室の壁は厚い筋肉層となっている．右心室と右心房と
の間には，三尖弁をもち，右心室の出口である肺動脈との間には肺
動脈弁をもつ．左心系では，左心室と左心房との間に僧帽弁をも
ち，左心室の出口である大動脈との間には大動脈弁をもつ．三尖弁
と僧帽弁はパラシュート状で，弁の一部は腱索という索状物で心室
内壁に結合され，心室内圧の上昇により血液が房室側に逆流するこ
とを防いでいる．出口側の弁は半月弁とも呼ばれ，心室の弛緩時に
動脈内の血液が逆流しない構造になっている．

.動脈硬化病変の局在性（localization of atherosclerosis )
動脈硬化をはじめとする血管の病変は，おもに太い動脈の特定な部
位に多発的に現れていることが知られている（図2･114) ．特に

いた ．このような動脈硬化症発生の流体力学的因子に関する
さまざまな仮説は，血行力学説と呼ばれている．
動脈硬化は，医学的には粥状（じゅくじょう）動脈硬化，細動脈
硬化，中膜壊死の3種類に分類される ．なかでも粥状動脈硬化

変を生じるので，臨床医学的に関心が集まっている．分子レベルに
よる粥状動脈硬化の進行の様子を説明する有力な説は障害反応仮説

激に対する応答として進行すると考えられている．ここで血流刺激

刺激により，血管内皮細胞（endothelial cell ）に機能変化が生じ
る．すなわち，内皮細胞の表面に接着分子が発現し，単球やリンパ
球が弱く接着して，表面をローリングする．さらに，内皮下に侵入
して分化し，マクロファージとなる．一方，低密度リポタンパク
（LDL）が内皮下に貯留し，それをマクロファージが貧食して泡沫
化し，粥状動脈硬化病変を形成する．泡沫細胞の集合が脂肪線条で
ある．これが，粥状動脈硬化の初期病変といわれるものである．同
時に形質変換を起こした平滑筋細胞が内膜に遊走・増殖する．
壁面せん断応力が内皮細胞の機能変化の原因とする考え方は，

sis ) と相反する高せん断応力説（high shear stress hypothe-
sis ) が有名である．それぞれの立場を裏付ける実験的および理
論的研究が行われ，興味深い論争が展開された．高せん断応力説
は，壁せん断応力が高いと内膜の物質透過性が高まり，アテローム
が生じるという考え方で，フライ（Fry）らによる大動脈内のアル
ブミン透過性の実験や，高脂質食投与のウサギによる実験結果が，
高せん断応力説を支持している．一方，ヒトの動脈硬化病変は，低
せん断応力の領域に多発していることが多く確認され，現在では低

h.微小循環系の血流（blood flow in the microcirculation)
微小循環系とは，体液と組織との間で物質交換の場となる微小血管
とこれに相伴う微小リンパ管，およびこれらを取り巻く間質または
実質組織から構成される循環単位のことを示す ．ここでは特に
毛細血管を中心にした微小血管系の流れについて眺める．
(１) 微小血管系の形態
太い動脈から末梢に向けて血液が流れていくに従って，次第にそ
の径が細くなっていく（p. 69前出，表2･15）．動脈の最末端部分
である細動脈を経て毛細血管になり，その後血管壁を介して物質交
換を行い，静脈の最末端である細静脈に入り静脈を経て心臓に血液
が戻る．一般に径が50mより小さい血管を，細動脈または細静
脈として区別する．毛細血管には平滑筋がないことが特徴である．
微小循環系は臓器や組織の機能に応じて固有な血管網を構築し，
特有な構造を有している．腸間膜でよく見られるパターンは中心通
路型と呼ばれるもので，特定の細動脈から細静脈までを結ぶ主要血
行路に，つねに優先的に血液が流れている血管網で，微小循環系の
基本とされている．皮膚や膀胱などの伸縮性の大きい組織に見られ
るパターンは網状型と呼ばれ，優先的な血行路は特に見られない．
骨格筋や皮膚によく見られるタイプは連結アーケード型と呼ばれ，
細動脈と細静脈が網目状に対をなして分布しているものである．腎
臓の糸球体や膵臓などにも見られるタイプが糸球体型で，細動脈が
分岐して毛細血管球を作り，再び集合して細静脈となる．その他肝
臓に見られる類洞型などのタイプが観察されている．
微小循環系の血管の構造は，細動脈，毛細血管，細静脈の各部位
で異なる．細動脈は，内皮細胞，平滑筋および線維細胞を含む結合
組織の層からなり，特に平滑筋の収縮によって断面の大きさを調節
し，血流の制御を行っている前毛細血管括約筋のある部位を，抵抗
血管と呼ぶ．毛細血管は一層の内皮細胞と基底膜からなり，物質交
換の調節を行っている．毛細血管における物質移動は，臓器の生理
的機能を代表する現象であるが，毛細血管壁の溶媒の透過性能
や蛍光色素でラベルされたタンパク質などの透過性に関する研
究 などが行われている．微視的な観測技術の進歩とともに，微
小循環系の研究が進展しており，細胞レベルのミクロな世界の様子
が次第に明らかにされつつある．
