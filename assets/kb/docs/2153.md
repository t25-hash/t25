# 1・3・4 研究動向から見る設計工学の変ぼう

> **章**: 第1章 総論
> **節**: 1・9 にも示すような、概念設計(conceptual design)・実体設計

ここまでの内容は設計工学が展開してきた経緯を粗く示したものであるが、その様子は学術研究の動向からも読み取ることができる。図1・10は工学関連分野における研究論文データベースCompen-dex において、設計に関係するいくつかのキーワードを指定して各年における論文数の変化を調べたものである。CAD については、1985年ごろから一定数の論文が出ており、‘CAE’(computeraided engineering、コンピュータ援用エンジニアリング)についても同様の傾向が確認できる。1985年という起点はワークステーションやパーソナルコンピュータの出現と呼応している。‘Opti-mal design’(最適設計)については、1980年代以前から継続して論文が出ているが、1993年ごろを境にその数がやや増加している。後者は複合領域設計最適化(multidisciplinary design optimiza-tion)についての動向を反映したものであろう。以上は1・3・3・a項での内容に対応するものである。一方、1・3・3・d項の方面に関しては、欧米における設計工学関連の国際会議での動向を振り返ると、設計対象としての製品(prod-uct)という概念が1990年ごろ以降を特徴づけている。図1・10の灰色の部分は‘product design’(製品設計)あるいは‘productdevelopment’(製品開発)をキーワードとして題目や概要に含む図1・10 研究論文数に見る設計工学の変遷(2004年8月23日調べ)1―8設計工学論文数であり、その数が1983年の前後を境に3.5倍になり、1993年ごろを境にさらに3.5倍になっていることが確認できる。加えて、突き詰めれば、第二の拡大期はproduct designについての論文が急激に増加した後、product developmentについての論文がそれらに入れ替わるがごとく増加していることを確認できる。背景を探るために、1・3・3・d項の内容を踏まえて、‘concurrent engi-neering’(コンカレントエンジニアリング)に加えて‘supplychain’(サプライチェーン)という関連するキーワードについて調べると、おおむね、前者についての動向がproduct designについての動向と呼応しており、後者についての動向がproduct develop-mentについての動向と呼応していることがわかる。つまり、ものごとを単純化してコンカレントエンジニアリング(concurrentengineering)が設計と生産の統合を、サプライチェーンマネジメント(supply chain management)がそれに加えて調達や流通との統合を意味するとすれば、設計についての視点は、いわゆる機能面での設計を起点として、品質やコストの改善、開発期間の短縮や顧客満足度の向上などを課題に取り込みながら拡大していっているのを読み取ることができる。
