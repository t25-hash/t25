# 2・1・1 二次元図面による図形の表し方

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・1 製図

機械の設計過程においては，設計者自身の思考のため，設計結果
の記録・保存のため，設計者から製作者への情報伝達のためなどさ
まざまな目的で，設計した機械の形状・構造をはじめとするさまざ
まな技術情報を表現する必要がある．立体である機械の形状・構造
は本質的に三次元の幾何情報となるが，近代工業化の過程において
利用可能な情報表現の手段・媒体は，二次元である紙を基本として
いた．そのため，二次元の紙面上に三次元の立体情報を含む設計情
報を表現する方法として，図面（technical drawing）という表現
方法が確立された．
紙面上に立体を表現する方法の一つに，等角投影，斜投影，透視
投影などによる作図法があり，これには立体を一つの絵として表現
できることや，立体の形状・構造を直観的に把握しやすいことなど
の特徴がある．しかしこれらの表現方法は，機械の設計・製作に必
要な形状・構造各部の長さや角度など，詳細な情報をあいまいさな
く一意的に規定するのには，必ずしも適していない．
そのため，二次元の紙面上で機械の形状・構造を表現する方法と
しては，立体を各方向から見たときの見え方を正投影（orthog-
onal projection）により一つ以上の図として描き，それらを紙面
上に配置する方法が中心的に用いられている．なお，この方法では
三次元の立体を表現するために，基本的に二次元の投影図三つを組
合わせる必要があることから，この表現方法またはそれにより描か
れた図面を，慣例的に三面図と呼ぶことがある．
