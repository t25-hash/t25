# 2・1・2 JIS 製図の概要

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・1 製図

機械設計に用いられる図面は、立体の基本的な形状・構造だけでなく、ねじ、歯車などの特殊な形状、各部の長さや角度などの寸法およびその公差、はめあい、表面粗さ、材料、加工法、熱処理など、さまざまな技術情報を表現する必要がある。そして、それらの技術情報の表現方法は、描いた者と読む者の間で解釈の相違やあいまいさが入り込む余地のない、一意なものでなければならない。そのため、図面の表現法を規定する必要があり、わが国においてはJIS(Japanese Industrial Standard、日本工業規格)により規格として規定されている。JIS においては、工業分野全般における製図を対象としたJIS Z8300シリーズの規格群がある。そこに規定された事項をできるだけ尊重しつつ、機械の図面を描くためには不十分である部分を補完して、その規格のみを参照すれば機械の図面が描けるよう、機械製図の基本としてJIS B 0001が制定されている。a. 機械製図に関するJIS とISO近代の工業分野では、その活動が国際的な広がりを有している。そのため、設計・生産に関する技術情報の表現媒体である図面についても、国際的に共通に理解可能なものである必要がある。このような観点から、製図に関するJIS 規格はISO(International Organization for Standardiza-tion、国際標準化機構)規格に整合させる方針で、順次改正が行われてきている。JIS Z 8300シリーズの規格群は、JIS の規格とISOの規格が1対1に対応するよう、ISO規格の体系に整合したものとして整理されてきた。一方、JIS B 0001は複数の規格を機械製図を目的として集約した日本独自の規格であり、これに対応するISO規格は存在しない。JIS Z 8300シリーズはISOの体系に整合した規格であるが、ISOに対応してPart(第~部)制をとっているため、それぞれのJIS 規格の適用範囲が狭くなるうえに数が増えており、これらを見比べながら製図を行うのは非常に面倒である。一方、JIS B 0001は設計技術者および教育現場で広く普及しており、製図を行ううえでマニュアル的で重要な規格となっている。そこで、便宜性とISO規格との整合化の観点から、ISOおよびJIS Z 8300シリーズ1―11との整合を図りつつ、対象を狭くして機械分野に絞る規格として、JIS B 0001:1985は存続しJIS B 0001:2000として改正され、現在に至っている。なお、以下に概説する機械製図に関するJIS 規格とISO規格の同等性には、表2・1の3段階がある。b. 機械製図機械製図においては、図形と対象物の大きさが正しく比例関係を保つように描く。ただし読み誤るおそれがない場合は、図の一部または全部について比例関係を保たなくてもよい。線の太さの中心は線を描くべき理論上の位置におく。線間の最小すきまは、平行な場合は最も太い線の太さの2倍とし、線と線とのすきまは0.7mm以上とすることが望ましい。透明な材料で作られる対象物は、投影図では不透明なものとして描く。.
