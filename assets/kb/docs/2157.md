# 2・1・3 二次元CAD による効率化

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・69 )、機能上必要な場合(図2・70)、または寸法指示に必要な場

かつて図面は、鉛筆やペンなどを用いて紙に手で書かれていたが、現在ではCAD(computer aided design)ソフトウェアを用いて、計算機画面上で描かれるようになっている。そこで処理されるのは基本的に二次元図形情報であるので、そのようなCAD を一般図2・76 セクタ歯車(記入例)図2・77 一連の平歯車の簡略図的に二次元CAD と呼ぶ。手で書かれた図面とCAD で作成された図面を区別する場合は、慣例的に手書き図面とCAD 図面という呼び分けをする。二次元CAD では、計算機の画面上で対話的に図面の作成を行い、結果をデータとして保存したり、プリンタやプロッタなどの装置によって、自動的に紙に印刷して出力するなどの操作を行う。二次元CAD により作成される図面は、基本的にディジタルデータであるため、手書き図面に比較して、保存、複製、編集、変更、伝達などの点で非常に優れており、作業を効率化することが可能である。しかし、CAD 図面に表現されている情報の内容自体は、手書き図面と本質的に大きな差はないため、機械の立体形状・構造を計算機上でモデル化して表現、操作する、いわゆる三次元CAD と比較すると、その効果は限定的である。なお、JIS 規格では、主として機械工業におけるCAD に関して用いるおもな用語および定義、CAD によって行う製図などについて規定されている。
