# 2・1・4 CAD 時代における製図法の意義

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・69 )、機能上必要な場合(図2・70)、または寸法指示に必要な場

近年、設計現場において用いられる設計対象の表現方法は、時代とともに次のように変化してきている。(1) 手書き製図(2) (二次元)CAD 製図(3) (二次元)CAD 製図(主)、必要に応じて三次元CAD(従)(4) 三次元CAD(主)、必要に応じてCAD 図面を生成(従)1―27このような状況においても、製図法は依然として、次のような点で意義を有していると考えられる。(1) 三次元CAD の操作に用いられる入力装置の中心であるマウスは、本質的に二次元情報の入力インタフェースである。したがって、三次元CAD において立体形状をモデリングする場合にも、目的とする立体はどのような二次元図形の操作(掃引、回転、交差、切断など)の組合せにより構成できるかが想起できなければ、立体形状データの作成操作が行えない。このような投影図と立体形状の双方向の変換能力を身につけるために、投影法、製図法は有効である。(2) 三次元CAD は、設計対象の形状や構造の構想がある程度固まった段階で、情報を入力し表現するのには適しており、強力なツールである。しかし、前述のように現状の入力インタフェースの機能的制約もあり、漠然としたアイデアの段階で形状、構造を試行錯誤的に表現したり、それをグループで行うなどの場合には、依然として紙の上に略図や概念図を描きながら行うほうが効果的、効率的である場合が多い。その場合に、紙面上に立体の形状、構造を、さまざまな技術情報を含めて表現できる製図法を理解し利用できることが、大きな利点となる。(3) 製図法、製図規格は、長い歴史を経て洗練されてきたこともあり、立体形状の幾何学的な情報だけでなく、寸法指定、公差、表面性状、加工法など、機械設計に必要なさまざまな技術情報を、あいまいさなく簡潔に表現できる、非常に優れた表現体系となっている。三次元CAD においても、設計中の対象物や設計結果はコンピュータディスプレイという二次元的な表現をとることが多く、その場合のさまざまな技術情報の表示方法は、製図法における表記法にならったものとなっている場合が多い。
