# 2・4・2 バーチャルリアリティ

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・134 は自動車のテールランプ部分を、レイトレーシングで作成し

バーチャルリアリティ(virtual reality)は仮想現実感と訳され、実際には存在しない仮想の世界をコンピュータグラフィックスで創造することにより、現実に存在するような模擬体験を行える技術を総称する。例えば体験者は自分の部屋にいながらにして、海外の有名建築物の中を歩いているというような模擬体験ができる。バーチャルリアリティに用いられるシステムの仕組みは、体験者に取付けられた位置センサ等の各種センサからの入力情報から、システムが体験者の位置や視線の方向を検知し、高性能なコンピュータグラフィックス機器を用いて瞬時に体験者が見るべき画像を計算し、大型のスクリーンに立体的な映像をリアルタイムに表示させるものである。ここでバーチャルリアリティが、映画やコマーシャルフィルムで使われるコンピュータグラフィックスとは異なる点について触れておく。バーチャルリアリティでは、リアルタイムに体験者の入力センサからの情報を、映像にフィードバックする必要があるのに対して、通常のコンピュータグラフィックス映像は、膨大な時間を費やして、あらかじめ計算された個々の画像フレームを連続的に映写したものであり、リアルタイム性は要求されていない。その点において、本来のバーチャルリアリティのねらいを商用として実用化しているのはゲーム業界であり、製造業ではプレゼンテーションや、製品の操作性の検討などに部分的に用いられている段階であり、いまだ発展途上の技術である。しかし、ここでは厳密な区分けをすることなく、コンピュータグラフィックスや、製造業で用いられているディジタルモックアップ等も含めて述べる。バーチャルリアリティを実現するためには、表示するオブジェクトの作成、超高性能なグラフィックスシステム、立体表示装置、各種の入出力センサによる体験者とのコミュニケーション機器が必要である。a. 表示オブジェクトの作成三次元オブジェクトの作成には、CAD 等の別システムで作成された形状データをVRMLなどの表示ポリゴンに変換し、必要に応じてテクスチャマッピング等を施すことにより、実在しないオブジェクトを高精度で作成できる。しかしこの作業には多くの労力が必要なことから、バーチャルリアリティの実現だけに形状モデリングを行うことは、経済的に困難である。一方、すでに実物や模型が存在する場合には、三次元測定機による入力や、カメラで撮影した画像から立体合成する方法なども最近発表されている。このような画像からの立体合成では、CADシステムで作成した形状よりも精度は落ちるものの作成コストの面で有利であることから、精度よりも実物感を優先することが多いバーチャルリアリティの入力には、有効な手段である。(1) 高速なコンピュータグラフィックス装置リアルタイムな画像処理を実現するためには、多くの情報量を高速で処理できる強力なコンピュータグラフィックス装置が必要になる。これらは図2・136 テクスチャマッピング事例図2・135 デプスバッファ法1―56設計工学CPU 映像表示装置複数の人数で体験できるバーチャルリアリティの表示装置には、図2・137のように、実物大の自動車が映し出せる超大型表示装置が用いられる。これは横7m×縦2.5mにもなる巨大なスクリーンで、通常は複数のプロジェクタを並べて配置した構造である。また図2・138のように、スクリーンを体験者の周囲を取り囲むように4面に洞窟のように配置することで、体験者が自動車に乗っているような体験をさせる装置も用いられている。この場合には、体験者の位置や視線によって、すべてのスクリーンの映像は変更されるようになっている。また体験者が一人の場合にはより手軽なヘッドマウントディスプレイ(head mount display)も用いられている。また最近では眼鏡を必要としない立体TV や、ホログラフィーなども研究されている。c. 立体視人間に映像を立体として認識させるために、左右の映像の視差を利用する方法が昔から多く用いられている。バーチャルリアリティの場合にも体験者が特殊な眼鏡を装着して、左右別々の画像を見ることにより立体感を出すものが多い。この特殊な眼鏡には左右に90°位相の異なる偏光レンズを用いたものと、液晶シャッタを用いたものがある。液晶シャッタの仕組みは右目で見るべき映像が表示されている場合には、左目のシャッタを閉じることにより左右独立の映像を見せることができる。なお、左右のシャッタはシステムからの同期信号を受けて瞬時に切り換わっている。d. ゲームに使われるバーチャルリアリティ技術一般に、バーチャルリアリティが最もなじみ深いものはゲームである。ゲームセンターの機器の中には仮想の敵と格闘技で対戦したり、戦闘機やカーレースなどを楽しめるものもあるが、これこそバーチャルリアリティ技術そのものである。先に述べたように、バーチャルリアリティのリアルタイム性能を実現するためには、高性能なグラフィックスシステムが必要であるが、家庭用ゲーム機は産業用に比べて安価であるために、その性能については侮られているのではないだろうか。図2・142は最近のゲーム機の描画性能比較を示したものであり、これによると、一般のCAD システムで用いられる機器の性能を大きく上回り、バーチャルリアリティ専用機器にも匹敵する性能であることがわかる。f. ゲームソフトの作成技術人間の動きは複雑なために、自然な動きをシミュレーションで求めることは困難である。そのために高精度の位置センサを実際の人間に取付けて、実際の動きを読み取る図2・143のようなモーションキャプチャが用いられる。位置センサを手や足に取付けた人物が鳥かごの枠内で動くと、手や足の瞬間的な位置を位置センサからの情報で収集でき、これにより図2・144のように、骨格だけの動きとして読み取ることができる。その骨格に肉付けをすることにより、実際のキャラクタの動きが完成する。さらにゲームソフトの中では、さまざまなオブジェクトを合理的に混在させる工夫も多く見られる。例えば、遠くにあるときの建物は二次元のイメージであったのに、近づくにつれて三次元に変化するなど、三次元の自動車や航空機にイメージを重ねて火炎や砂煙を演出している。g.
