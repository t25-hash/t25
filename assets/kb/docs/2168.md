# 2・5・7 生体内物質移動

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・5 生体熱工学

a.生体内における物質移動の形態 物質の移動が起き
るためには駆動力が必要であるが，この駆動力の形態によって，生
体内の物質移動は受動輸送と能動輸送に分けられる．さらに，細胞
膜そのものを変形させて物質の輸送を行う膜動輸送がある．
.受動輸送（passive transport ) 生体内ではほぼ定温，
定圧という環境であるため，生体内のエネルギー状態を表すにはギ
ッブス（Gibbs ）の自由エネルギーが適当である．物質の移動を許
している開いた系において，自由エネルギー の物質移動により
変化する割合 / を電気化学ポテンシャル（electrical
chemical potential ）μと呼ぶ．
(１)自由拡散（free diﬀusion）
二つの溶液の間に電気化学ポテンシャルの差が生じると，そのこ
う配が駆動力となって，こう配を下る方向に物質が移動する．自由
拡散による物質流束 はテオレル（Teorell ）によって次のように
表される．
流束＝移動度×濃度×駆動力
＝－dμ / d (2･346)
理想溶液の場合，よく知られるフィックの拡散の式となる．
(２) 促進拡散（facilitated diﬀusion）
物質を運搬する輸送体（carrier ）が膜内に存在する場合，輸送
体自身がその化学ポテンシャルのこう配に従って移動するため，あ
たかも物質を運ぶ渡し船のような役割を果たしている．このように
受動的に特定の物質を選択的に透過させる物質輸送形態は，促進拡
散（facilitated diﬀusion）と呼ばれ，生体内での特徴的な物質輸
送形態の一つである．
.能動輸送（active transport ) 能動輸送とは，溶質分子
が電気化学ポテンシャルの低いところから高いところに輸送される
ことをいう．ポテンシャルの登り坂を登るためには，ATPで代表
される生体エネルギーを必要とする．
.膜動輸送（cytosis ) 通常は，大きな分子は細胞内に入
らないが，膜自体が変形して，菌体，抗原，タンパク質などの大き
な物質を取込んだり，細胞内で産出された酵素タンパク質，多糖類
を分泌することがある．このように細胞膜が外液の一部を包み込
み，細胞内に取込んで小胞を形成するような，膜の変形を伴う輸送
を膜動輸送（cytosis ）と呼ぶ（図2･180) ．最もよく現れる膜動
輸送は，飲作用（pinocytosis ）と呼ばれるものであり，毛細血管
壁や上皮に小さな小胞を形成して，溶液のみを包む．次に，細菌や
固体粒子のようなものを取込む場合を食作用（phagocytosis ）と
呼び，通常光学顕微鏡で認められるほどの大きさである．第三の取
込み方として，レセプタが関与するエンドサイトーシスがある．細
胞の表面に存在しているレセプタが溶質と結合して取込まれ，内部
に小胞が形成される．リポタンパクが膜動輸送によって細胞内に取
込まれることが確かめられている．
b.血液循環系の物質移動
.血液中の酸素および炭酸ガスの輸送
(１) 血液中の酸素輸送
血液中に含まれる酸素の一部は物理的に溶存した形で，大部分は
赤血球内のヘモグロビンと結合した形で存在する．血液中のヘモグ
ロビンが完全に酸素加されたときの結合酸素量を酸素容量という
が，血液中に含まれている結合酸素量と酸素容量との比を酸素飽和
度といい，通常百分率で表す．平衡状態における酸素飽和度と，酸
素分圧 との関係を表す曲線を酸素解離曲線といい，特異なS字
状の曲線で，この非線形性が血液における酸素輸送の特徴を決め
る．
毛細血管内から組織への酸素輸送の過程を，図2･181 に示す．
赤血球内から吐き出された酸素は，血液中に溶存し対流によって毛
細血管壁に到達し，組織へと運搬される．
(２)血液中の炭酸ガス輸送
血液中に含まれる炭酸ガスの一部は物理的に溶存した形で，大部
分は重炭酸イオンおよびヘモグロビンと結合した，カルバミノ化合
物として存在する．血液中の総炭酸ガス含有量と，炭酸ガス分圧と
の関係を表す曲線を，炭酸ガス解離曲線という．赤血球内に存在す
る炭酸脱水酵素が，溶存炭酸ガスから重炭酸イオンへの解離に寄与
している．
(３)生体組織におけるガス輸送
筋肉や脳組織などの生体組織におけるガス輸送は，古くから生理
学の重要課題であり，クロー（Krogh）による酸素分圧分布の古典
的解析に始まり，多くの実験的理論的研究が行われている．特に微
小血管網による酸素供給のメカニズムが，観測方法の進歩によって
明らかにされ，毛細血管のみならず，細動脈からも酸素が多く供給
されていることが示されている．例えば，図2･182に示されるよう
に，細動脈，毛細血管および細静脈のネットワークの中で，対流と
拡散により組織へ酸素を供給している ．
