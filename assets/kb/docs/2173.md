# 2・2・3 標準物質

> **章**: 第2章 単位と標準
> **節**: 2・2 計量標準

a.標準物質の役割と特性 標準物質は，化学定量分析におけ
るスケールとしての役割をもち，また分析条件（機器，分析者な
ど）の補正用として使用される．すなわち，化学分析の結果はそれ
に用いた標準物質の品質の程度に依存して分析信頼性が決まること
になり，それは，相互承認，貿易，安全対策などの指針に影響する
ことになる．
近年新たに登場した化学物質の種類は多く，また環境問題に対す
る認識の高まりに対応して必要な標準物質の種類も増加し，重要性
も増大している．
2･2･2項にある物理的な計量標準が，国家標準としてその供給が
国家の管理のもとに行われるのに対し，標準物質と目されるものの
中には，民間企業，団体において製造・認証されている物質もあ
る．しかし，これらは国際ルールに基づいて品質保証の確認がされ
ていなかったり，また国家標準物質に対するトレーサビリティに関
係付けられていなかったりする．今後これらの民間団体による標準
物質を国家標準として認証するために，国際ルール（ISOガイド
34）に適合した認証手法を確立することが，標準物質供給のうえか
ら重要である．
b.標準物質の種類と整備状況 標準物質は大別すると図2･1
のように分類される．また，各標準物質は表2･4のように定義され
ている．
わが国は，2010年までに290種類の標準物質を整備することに
している．標準物質の整備が急がれなければならないのは，環境ホ
ルモンなどの極微量でも毒性のある物質（内分泌かく乱物質など）
に社会的な関心が高まっていることが影響している．有害な物質が
排出されたときに，その物質を同定するには標準物質が必要であ
る．表2･5は2010年を目標年度としたわが国の標準物質整備計画
のうち，先端材料標準物質について示している．
