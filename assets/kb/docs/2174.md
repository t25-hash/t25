# 2・2・4 トレーサビリティ

> **章**: 第2章 単位と標準
> **節**: 2・2 計量標準

a.トレーサビリティの意義 計量標準にとって最も重要な概
念はトレーサビリティである．
トレーサビリティとは，一般ユーザまたは企業が所有する計量・
計測機器の値，または標準物質の値がどのくらいの精度で国家計量
標準と校正によるつながりをもっているかというその程度を表す．
そして，トレーサビリティを確保するための一連の手続きの体系を
トレーサビリティ制度（traceability system）という．トレーサビ
リティ制度によって手元にある計量器の値が，対応する国家標準に
対してどの程度の精度をもつかということを客観的に保証すること
ができる．
わが国においてトレーサビリティ制度は1992年制定の計量法第
8章「計量器の校正等」の中に規定されている．ただし，計量法で
はトレーサビリティという用語は用いずに，「指定標準器の指定」，
「特定標準器による校正等」，「証明書の交付等」，「指定標準器によ
る校正等の義務」，「指定の申請」，「指定の基準」，「指定の取消等」
として運用上の規定を定めている．
トレーサビリティは精度がより低い計測器または標準物質からよ
り高い国家標準へ向かう値付け（校正作業）の直線的な連鎖をい
い，逆に国家標準からより精度が低い一般の計測器，または標準物
質のほうへ校正作業により値付けをすることを標準供給ということ
がある．この意味で，計量法は上から下への標準供給の方法につい
て規定しているということができる．
トレーサビリティでは，計量器または標準物質が国家標準に直線
的な連鎖によりつながっていることが重要で，連鎖が途中で分岐し
たりループを作ったりすると，国家標準に対する校正認証が途切れ
ることになる．日常に使用する計測器などがトレーサビリティを満
足しているときは，その計測器を使用して得られた測定値の精度
が，一つ上位の計測器（一般には認定事業者が保有する特定二次標
準器）を用いて規定にのっとった校正により保証されていることを
意味する．したがって，トレーサビリティ制度は，計測器の精度が
国家計量標準に対しどの程度であるかを明確にするものであって，
制度自体が計測器の精度を向上させるものではない．むしろ過度の
精度を追求して経済的な損失を生じるのを防止するものであるとい
える．
b.トレーサビリティの体系 計量法（平成4年法律第51号）
では，トレーサビリティ体系の最高責任者は経済産業大臣であると
定められている．経済産業大臣は，トレーサビリティ体系で使用さ
れる計量標準を現示する計量器や，標準物質を製造するための器
具，機械もしくは装置（この計量器や器具，機械もしくは装置を特
定標準器という）の指定，およびそれらを用いて校正作業を行う校
正作業者［指定校正機関（designated calibration agency）とい
う］の指定を行うこととされている．
さらに経済産業大臣は，一般ユーザと指定校正機関の中間で校正
作業を行う事業者［認定事業者（accredited calibration body）と
いう］の認定を行う．認定事業者による校正作業には特定標準器に
よって校正された計量器を用いる．認定事業者の認定を受けようと
するものは，行う校正業務の範囲や校正作業の実施方法などを記し
た認定申請書を大臣あてに提出する．
指定校正機関，認定事業者は，校正の結果を記した校正証明書を
発行することができる．この証明書には，校正作業が国家計量標準
に対しトレーサビリティを満たしていることを記述すること，およ
びJCSS（Japan Calibration Service System，計量法トレーサビ
リティ制度）のロゴマーク（図2･2）を付すことができる．ロゴマ
ーク付き証明書は，ISO9000シリーズにおける「国際標準との間
に公的に有効な関係を持つ認定された装置を用いて校正し，調整す
る」の項についての証明書としても使用することができる．
c .特定標準器 特定標準器は国家計量標準の最高位に位置す
る計量器であり，計量法134条により経済産業大臣が指定すること
になっている．また，同条第2項には，特定標準器が繰返し使用の
際に不適当であるときに特定標準器に代わりうるものとして，特定
標準器で校正された計量器を指定するとしている．これは通称，特
定副標準器と呼ばれるもので，指定校正機関等で使用される．認定
事業所で校正に使用される計量器は，指定校正機関等で特定標準器
または特定副標準器で校正された計量器であり，特定二次標準器と
呼ばれることがある．
