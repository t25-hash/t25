# 2・3・1 ISO規格の目的

> **章**: 第2章 単位と標準
> **節**: 2・3 工業規格

英語では規格と標準に同じ語（standard）を用いるが，日本語
ではstandardのもつ意味を，規格および標準という語で使い分け
ている．規格はstandardの中の文書を指し，標準は基本単位や物
理定数およびその基準となる量を指す．規格と標準はどちらも人間
の知的活動の一つである標準化（standardization）という作業に
よって得られた成果であり，人類共通の知的基盤である．
標準化とは，ある行動や問題に秩序を生成するために規則を作成
し，これを活用する過程のことである．これだけではわれわれが意
図する工業標準だけではなく多くの分野の活動が包含されることに
なってしまい，意味があいまいになるが，規格や標準の基本的なこ
とを扱う際にはこのような定義は無意味ではない．
規格の機能は，(１)生産と交易における経済性の追求，(２)製品
とサービスの保証による消費者の保護，(３)安全，健康，環境の保
護，(４)関係者間のコミュニケーションの手段であるということが
できる．
これらの機能は，機械製造分野において過去に行われた事例や，
現在われわれが直面している課題を考えると，その意味を容易に理
解することができる．18世紀に機械産業が興ったあと，19世紀中
葉においてイギリスのモーズレイ（Maudslay）やウィットワース
（Whitworth）によってなされたねじの標準化は，その最も重要な
例の一つであり，その後の機械の設計，製造，検証や製品の保証な
どの機械生産活動にとって大きな経済的メリットを生み出すことに
なった．また，規格は取決めをまとめた文書の形式をしており，そ
れはすべての人に理解され，概念を共有させる働きをもっている．
例えば，製図規格は設計者の意図を製造工程や検証工程に伝達する
規則をまとめたもので，設計者の製品についての概念を加工工程や
検証工程の作業者と共有するために，設計情報を伝達する言語の文
法書ということができる．
さらに，規格は供給者と購買者が共通の認識に立って商品の品質
についての情報を伝達する手段として，購買者を保護する観点から
その重要性を増している．規格がもつ性格の一つは，規格は単純化
活動の結果であるということである．上に述べたねじの標準化は，
ねじ径の寸法，ねじ山の形状などを限られた種類に制限することに
より，多様性と複雑性から逃れることを実現した．単純化により製
品の種類や仕様の数は制限されることになるが，それによる経済的
メリットは計り知れない．また，製品の互換性の考え方の可能性を
もたらす．
