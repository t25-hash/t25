# 3・1・2 製品企画と経営戦略

> **章**: 第3章 設計のための個別方法論
> **節**: 3・1 企画のための方法

製品企画は経営戦略の(重要な)一部であり,両者が統合されてこそ,よりその意味を発揮する.ここでは,経営戦略の基礎となる3C,ポートフォリオ分析,およびポーターの競争戦略論についてその概要に触れるとともに,それぞれの製品企画との関連について説明を加える.a. 経営戦略の3 C(3 C of management strategy). 経営戦略の3 Cとは経営戦略の3Cとは,顧客(cus-tomer),競合(competitor),自社(company)のことをいう(図3・3).これらは,経営戦略を策定する際に考慮すべき三つの基本的な視点であり,製品企画で考慮すべき視点とも共通している.ここで,顧客と競合は社外の存在であり,それぞれ自律的に意思決定を行い行動する経済主体であることに注意する必要がある.すなわち,顧客と競合については,どう動くかを完全に予測することはできないし,また自社と違って管理することもできない.このような不確実な条件のもとで行われる経営戦略や製品企画には,そもそも唯一の最適解などは論理的に存在しない.つまり,ここで求めるべきはつねに妥当解であることを意識することが重要である.. 3 Cによる価格決定の視点3Cは単純な思考フレームワークであるが,それだけにその応用範囲は広い.その一例として,製品企画とも関連が深い価格決定の視点への応用がある.(1) 自社の視点まず,最も基本となるのが自社の視点である.ここでは価格は,コストと適正利益の和として表される.公共性が高い製品やサービスなど,その業界ないしは企業を保護・育成すべき社会的な妥当性製品(product)価格(price)マーケティングの4P流通(place)販売促進(promotion)図3・2 マーケティングの4P図3・3 経営戦略の3Cが高い場合に,この視点によって価格が決定される.かつての高度経済成長期においては,この視点が主流の業界が数多く存在したが,社会が豊かになり,成熟の時代を迎えたいまは,この視点が働く余地は徐々に少なくなりつつある.(2) 競合の視点次の視点が競合である.この場合,その製品の競合製品ないしは代替製品との比較によって,価格が決定される.多くの成熟商品では,製品の品質ないし差別化はすでに限界に達している.その結果,価格競争になることが多い.この場合,その製品のコストや適正利益とは関係なく,すなわち自社の論理とは無関係に価格が決定されることに注意が必要である.競争が極端に激化している業界では,すべての企業の製品が赤字であることもありうる.また,これとは反対に,突発的な好景気などにより需給関係が逼迫した場合などは,通常の適正利益以上に価格が高騰することもある.(3) 顧客の視点最後の視点が顧客の視点である.この場合は,その製品が生み出す顧客にとっての価値によって価格が決定される.例えば,業務効率化を目的とした情報システムの場合,毎年削減される費用に,償却年数を乗じた金額が価格の指針となる.以上の三つの視点は,原則としてそれぞれ独立している.現実の価格決定においては,三つの視点が合成されて作用しつつも,そのうちのどれか一つの視点が支配的であることが多い.その製品がおかれた価格決定がどの視点によるものなのかを十分に理解しておくことが,経営戦略策定にとっても,製品企画にとっても,極めて重要である.b. ポートフォリオ分析(portfolio analysis)経営戦略策定や製品企画には,社外の市場や競合が強く関係してくることから,本質的に不確実性が存在することをすでに述べた.このような不確実性を分散させることによって,不確実性を合理的に管理するための考え方が,ポートフォリオ分析である.もともとは,経営戦略を策定する際に複数の事業を分析するために用いられたが,製品企画に用いても考え方は同じである.ポートフォリオ分析では,市場の魅力度と自社製品の競争力の二つの観点から,すべての製品を4種類に分類する.一般的には,市場の魅力度を表す尺度として市場成長率を,また自社製品の競争力の尺度として市場占有率(シェア)を選ぶことが多い(図3・4).. 金のなる木市場成長率が低く,市場占有率が高い製品は,投資金額が小さく,売上金額が大きいので,一般に収益性が最も高い.ここで生み出された利益が,他の製品の開発資金として用いられる.. 負け犬市場成長率と市場占有率の両方が低い製品は,赤字製品となっている場合が多い.今後の成長が期待できないことから黒字化する可能性は低く,撤退を検討すべき製品とされる.. スター市場成長率と市場占有率がともに高い製品は,まさに花形製品であり,スターと呼ばれる.ただし,投資金額も大きいので,製品開発のリスクも大きく,一時的には赤字製品となることもありうる.高問題児スター市場成長率低負け犬金のなる木低高市場占有率(シェア)1―67. 問題児市場成長率は高いものの,市場占有率が低い製品は,まさに問題児である.今後の成長が期待され,投資金額がかさんでいくが,競争力は弱く,一般には赤字となる.製品開発の資金的な流れは,金のなる木で生み出された利益が,スターと問題児に投下される.そして,スターと問題児の製品は,時間の経過とともに,金のなる木か負け犬のどちらかになる.そして,金のなる木の製品が,次のスターと問題児の開発資金を生み出すとともに,負け犬の製品は,市場から撤退へという道を一般的には歩むことになる.c. ポーターの競争戦略(competitive strategy)論経営戦略を語るうえで欠くことのできないのが,M.ポーターの競争戦略論である.その骨子は,業界の競争構造を決定する五つの力と,企業がとるべき三つの基本戦略とからなる.. 五つの力(five forces)競争状態は,五つの力によって決定される.ここで五つの力とは,売り手(供給業者)の交渉力,買い手の交渉力,業者間の敵対関係,新規参入の脅威,代替製品・サービスの脅威のことである(図3・5).つまり,企業の競争は,業界内の競争と業界外との競争とに分けられる.これは,それぞれ企業間の競争と業界間の競争と言い換えてもよい.業界内の競争だけにあまりにも集中していると,例えば,代替製品や新規参入業者によって業界そのものの地位が脅かされることになりかねない.反対に,業界そのものを守ることに対して過度に注力しすぎるのも,結果として業界内の競争があまりにも緩やかになってしまい,企業の発展のためには決して健全とはいえないであろう.企業では,つねにこの2種類の競争を視野に入れて,経営戦略が策定,そして実行されている.これは,製品企画においても同様である.企業間の競争に勝利したとしても,業界間の競争で敗北すると,業界トップ企業といえども,その製品は赤字に転落することがありうる.一方,業界間の競争で勝利を収めれば,業界内では下位企業であっても,安定的な利益を得られる場合もありうる.. 三つの基本戦略企業の基本戦略とは,差別化(differentiation),コスト優位(cost leadership),そして集中(focus)のいずれか一つである(図3・6).差別化戦略では他社にない独自の特徴を訴求し,反対にコスト優位戦略では価格差を追求する.これら二つは決して両立しないというのが,ポーターの重要な主張である.ただし,例外が一つだけあって,それが集中戦略である.集中戦略(ニッチ戦略やフォーカス戦略ともいう)では,すべての市場をねらわず,ある限定された市場をねらい,差別化とコスト優位を両立させようとする.この考え方は,製品企画においても,そのまま適用できる.窮地に陥っている企業は,この三つの基本戦略のいずれをとるかで,迷っていることが多い.すなわち,できるだけ多くの市場をねらいながら,差別化とコスト優位の両方を実現しようとする結果,図3・4 ポートフォリオ分析図3・5 競争戦略論における五つの力1―68すべての取組みの成果が中途半端なものに終わっている.
