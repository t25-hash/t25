.目標仕様（what) 対象分野が決定されたら，次に，そ
の対象分野において具体的にどんな製品を作るのか，その製品が達
成すべき主要な目標仕様（target specification）を決定する必要が
ある．これは，作るべき製品の定義付けにほかならず，目標仕様の
決定こそが，製品企画そのものともいえる最も重要な部分である．
この目標仕様の決定については，顧客と製品の二つの視点がある．
なお，対象分野で利用可能な技術の視点は，こと製品企画における
目標仕様の決定に際してはややあいまいなため，一般には適さない
ことに注意が必要である．
(１)顧客
この視点では，製品の目標仕様を，顧客に提供する便益から決定
する．例えば，東京～大阪間を片道2時間以内，到着時刻の誤差
10分以内，料金1万5000円以内で，人間が安全に移動可能な手段
というのが顧客の視点である．
(２)製品
製品の視点は，顧客の視点を，さらに具体的な製品レベルの目標
仕様へ具体化していくものである．通常，目標仕様は，この視点に
よって定められる．上記の例でいうと，この市場の要求を満足する
具体的な製品としては，飛行機と鉄道が考えられる．ここで，飛行
機では到着時刻の誤差10分以内という要求は実現不可能と判断し
たとする．すると，選択肢としては鉄道が残る．次に鉄道には，新
幹線とリニアモータカーがあるが，後者では1万5000円以内とい
う料金の実現は極めて困難と判断したとする．その場合，結果的に
新幹線という製品を選択することになり，例えば，安定的に走行可
能な時速300km以上という目標仕様が決定される．
この顧客と製品という二つの視点は，それぞれ利点と欠点があ
る．すなわち，顧客の視点は，この製品は何のためのものかという
目的が明確になるという長所があるが，実際の設計を行うにはやや
具体性に欠けるという短所がある．一方，製品の視点は，より具体
的であるという長所があると同時に，ともすれば製品を作りあげる
ことだけに視点が限定され，製品の目的そのものから注意をそらし
がちになるという短所がある．したがって，目標仕様の決定につい
ては，可能な限り，顧客と製品の両面から行うのが望ましい．
c .製品企画の要件（why，how，how much) 対象分野と目
標仕様を決定すれば，製品企画の内容が一応定まったことになる．
しかし，これで製品企画が完了したわけではない．すなわち，製品
企画の内容は，市場性，競争優位性，実現性，収益性の四つの要件
を満足する必要がある．
