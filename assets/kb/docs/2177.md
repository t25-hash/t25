# 2・3・3 ISO規格と測定

> **章**: 第2章 単位と標準
> **節**: 2・3 工業規格

Standardという語が文書としての規格と，計量標準や単位の意
味を併せ持つことは，両者の間に日本語からは見えにくい歴史的関
係があることをうかがわせる．大きくいってその関係は二つの面に
集約される．その一つは，歴史的に計量単位およびその値を定める
標準の歴史は，多様から単純への歴史であること，つまりある単位
系の普及の程度は国家または為政者の統治力の一つの指標となると
考えられることから，統一された計量単位系の整備は国家体制の維
持に重要な意味があったことである．整備された単位系を記述する
文書は国家の強制規則として施行されることになったが，単位系の
使用方法などの標準化は民間レベルに任されている．計量単位系を
定める文書が規格としての性格をもっていること，換言すれば今日
の規格は計量単位系を強制する規則（文書）をそのスタートとして
いるということがいえる．もう一つは，ISO規格などの工業規格
（標準）は，計量単位の裏付けがあって初めてその機能を発揮でき
ることである．例えば，製品の品質を規格で定めているとき，供給
者と購買者間で製品の規格への適合性を評価する際，多くは計量単
位による数値評価が用いられる．SI単位系は，過去の単位の特性
ISO規格そのものは強制力をもつものではないが，必要に応じ国
技術がそれぞれほぼ25 %を占め，エレクトロニクス・通信技術，
運輸・流通技術が続いている．ISO規格はISO会員団体（実質的
会（TC:Technical Committee）において審議され，合意が得ら
会員団体に送付して投票を行い，コメントを求める．投票結果がポ
送付して投票を行う．投票結果がポジティブのとき，ISO規格と
者および購買者双方にとって関心事である．このような判断を行う
価が必要になる．
ISO規格が国家規格として強制文書であるときも，評価主体は民
である多様性を単純化して，世界中で共通に使用されることを目的
に作られたもので，ISO1000の規格番号をもつISO規格である．
家がそれぞれ法律により強制力をもたせている．
