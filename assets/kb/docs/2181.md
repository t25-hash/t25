# 2・1・2 潤滑剤の性能と選定基準

> **章**: 第2章 潤滑剤
> **節**: 2・1 潤滑剤の種類と選択

a.潤滑剤の性能 潤滑剤を正しく使用するためには，摩擦部
分の寸法形状および材質，使用条件（荷重，速度，温度，雰囲気），
潤滑方法などを検討し，これらに適した潤滑性能を有するものを選
定する必要がある．潤滑剤の性能のうちでおもなものとして，(１)
粘度・ちょう度などのレオロジー特性，(２)潤滑性能そのものを示
す潤滑特性，および(３)潤滑剤のメンテナンス間隔に大きな影響を
もつ各種性能の安定性がある．このほかに物理的性質では，P-V-
T関係，熱的性質として，比重・密度，熱膨張係数，体積弾性係
数，圧縮率，比熱・熱伝導率，潜熱，揮発性・蒸発性などがあり，
光学的性質としては，色・蛍光・吸収スペクトル，屈折率などがあ
る．これらの性質からは，性能・品質の変化が読み取りやすく，そ
の測定結果は潤滑剤の品質管理によく利用される．また，絶縁油な
どでは潤滑油の電気的特性として，誘電率・比誘電率，誘電正接，
体積抵抗率，絶縁破壊電圧・絶縁耐力などの性質が重要となる．こ
のほか界面化学特性は，潤滑表面への潤滑添加剤の吸着やぬれ性な
どの良否に深くかかわっており，潤滑特性に密接な関係のある性質
として重要であるが，これらについては参考図書 を参照された
い．

.潤滑性能 潤滑膜の厚さは前述の粘度によって大きく変
は，油性向上剤，摩耗防止剤，極圧剤などと呼ばれる添加剤を必要
.各種性能の安定性 潤滑剤の性能が使用中に変動するこ
とは好ましくないので，使用条件に応じて各種性能の安定性に優れ
潤滑剤の低温における流動性や粘度指数を向上させるために，高
として使用している石けんの網目構造が壊れたり，細かく分散して
いる非石けん系増ちょう剤どうしが集まって固まると軟化する傾向
があるので，機械的安定性に優れていることが必要となる．
潤滑油は使用中に加熱あるいは酸化されると，粘度が増加し，有
機酸や不溶性のスラッジを生成する．粘度の増大は流動抵抗を大き
くし，有機酸の生成は軸受金属などを腐食させる．またスラッジの
生成は，フィルタの目づまりや摩擦面の損傷の原因となる．酸化防
止剤，腐食防止剤，清浄分散剤などを用いることにより潤滑剤の熱
安定性や酸化安定性は改善されるので，使用条件に応じてこれらの
性能のよい潤滑剤を選定する必要がある．
そのほか水と酸素の共存によって起こる鉄さびの発生防止，水が
混入したときの水分離性，空気の混入によるあわ立ち防止などの性
能も潤滑剤の選定に当たっては考慮する必要がある．
.(固形）異物 軸受や油圧機器などでは，潤滑剤中の
（固形）異物が機器の寿命や騒音に大きく影響するので，できるだ
け清浄なものを選定しなければならない．特にグリースでは（固
形）異物はベアリングから発生するノイズの原因にもなっている．
また，異物を除去することも液状の潤滑油に比較して難しいことか
ら，取扱いには十分注意する必要がある．
b.潤滑剤の選定基準 軸受，歯車などの機械要素や各種機械
に適する潤滑剤の選定は，「適油」をキーワードとしてインタネッ
トから情報を得ることができる．一般には，潤滑部分の設計に適合
している粘度・ちょう度の選定が第一となる．また，使用期間，温
度，摩擦面にかかる荷重などから，油膜が十分に形成されるマイル
ドな潤滑条件では，タービン油，作動油などの使用を検討し，厳し
い条件では，摩耗・焼付き防止剤が多く添加されているエンジン
油，ギヤ油などの潤滑剤を選定する．外部から異物が混入するか否
かも選定基準として考慮する重要な条件となる．すすやスラッジの
混入しやすいエンジン油では使用する添加剤の種類や量も多く，こ
れらの異物が入らない工業用潤滑剤では添加剤の使用量も少なくな
っている．また，すでに述べたように使用されている潤滑システム
によって，液状の潤滑剤を選定するか，グリースあるいは固体潤滑
剤が適しているかを判断することになる．これらは機器の設計段階
で詳細に検討されているため，使用機器の潤滑剤選定に当たって
は，機械メーカ，潤滑剤メーカの推奨を考慮し，最適なものを選定
することが必要である．
