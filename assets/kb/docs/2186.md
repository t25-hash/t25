# 2・4・1 固体潤滑剤の種類と特徴

> **章**: 第2章 潤滑剤
> **節**: 2・4 固体潤滑剤

固体潤滑剤は，相対運動する摩擦面の表面損傷や摩擦，摩耗を低
減するために，粉末または薄膜として使用される固体をいい，(１)
潤滑油やグリースへの添加剤，または(２)潤滑油やグリースの代替
の潤滑剤，(３)固体潤滑剤自身の被膜または高分子や金属材料との
複合被膜，および(４)しゅう動部材自身が自己潤滑性を有する固体
潤滑剤含有複合材料として利用される．(１)と(２)は一般の機械や
機器に適用される比較的簡易な適用方法であり，特に(３)と(４)
は，潤滑油が酸化劣化する高温，流動性を失う極低温，蒸発する真
空，放射線下など厳しい環境，食品・薬品工業やLSI製造装置な
ど油やグリースによる汚染を嫌う環境，また酸，アルカリ水溶液や
有機液体などの潤滑油が適用できない環境下での適用を可能にす
る．
