# 3・4・3 故障現象の確率論

> **章**: 第3章 設計のための個別方法論
> **節**: 3・4 信頼性の設計

a. 信頼性の基礎式信頼性とは、アイテムが与えられた条件のもとで、与えられた期間、要求機能を遂行できる能力と定義される。信頼性は能力の解釈によって広い意味で使われるが、信頼度(reliability)は、これを確率で表現したものである。したがって、故障率と信頼度には以下の関係が存在することがわかる。これらは一斉に使用が開始され、故障したPCは修理せずに廃棄されると仮定する。時間の経過に伴い、故障して廃棄されるPCの台数は単調に増加する。一方、λは、各時点で動作中のPCの何%が次の時点までに故障するのかを表したものである。b. MTTFすでに述べたように、アイテムが故障するまでの時間は確率分布としてとらえられるが、その特性を簡単に表すためにMTTF(mean time to failure、故障までの平均時間)が広く用いられている。これは故障までの時間の期待値として定義され、次式で示される。ただし、例えば製品のMTTFが1万時間といっても、1万時間経過した時点ではかなりの割合の製品が故障してしまっていることになる。そこで、製品の耐久性という意味では、図3・64に示す、製品の10%が故障するまでの時間として定義されるB(ビーテンライフという)がしばしば用いられる。c. 故障率に基づく故障の分類確率的な見方においては、時間軸上での故障の発生形態によって故障を特徴付ける。そこで、故障率の時間的変化から、故障を次の三つの基本パターンに分類することが行われている。なお、これらは、故障率の変化パターンを示すもので、図3・61に示した、機能状態を表すパラメータ値の変化とは意味が異なることに注意が必要である。(1) DFR(decreasing failure rate)型故障率が時間とともに減少するタイプである。設計、製造上の欠陥などのために初期段階で故障が多く発生するが、時間とともにこれらの欠陥が取り除かれ、故障率が低下していく場合である。(2) CFR(constant failure rate)型故障率が時間によらず一定のタイプである。アイテムが安定した稼働状態にあり、故障の発生が偶発的な場合である。(3) IFR(increasing failure rate)型故障率が時間とともに増加するタイプである。軸受の摩耗のように、アイテムが時間とともに劣化し、それに伴って故障しやすくな1―9 5図3・65 バスタブ曲線る場合である。対象が単純な部品や簡単なシステムの場合は、故障の発生パターンは上記3パターンのいずれかに分類されることが多いが、複雑なシステムの場合は、そのライフサイクルの中で三つのパターンが順次現れ、図3・65に示すような形態になるといわれている。これは、DFR 型の初期故障期間(early failure period)、CFR 型の偶発故障期間(random failure period)、IFR 型の摩耗故障期間(wear-out failure period)より構成されており、その形態からバスタブ曲線(bathtub curve)と呼ばれる。バスタブ曲線は、人間の一生を、幼年期、青年期、老年期に分けたときの、死亡率の変化パターンに対応していることから、一般に広く受け入れられている概念である。システムがこれらのいずれの期間にあるかを知ることは、信頼性向上のための対策を講じるうえで大切である。ただし、航空機の装備品の故障率を調査した結果では、このようなバスタブ曲線を描くものはごくわずかであったという報告もあるので、個々のシステムへの適用については確認が必要である。d. 代表的な故障分布関数c項では、故障の発生形態を単純に分類したが、信頼性データの解析においては、故障分布関数として種々の確率分布関数を仮定することが行われる。なかでも、指数分布と、ワイブル分布は信頼性の解析において最もよく用いられる。. 指数分布(exponential distribution)初期故障期においてシステムに内在する欠陥が取り除かれた後は、多数の要素から構成されるシステムの故障は多くの場合偶発型になり、故障率はほぼ一定になると考えられる。これは、指数分布と呼ばれる。. ワイブル分布(Weibull distribution)ワイブル分布は、スウェーデンの科学者ワイブル(W. Weibull)が材料の破壊強度の分布を表すものとして提唱したものである。したがって、形状パラメータを変化させることで、ワイブル分布一つで三つの故障パターンを表現することができる。また、3・4・4・d項で後述するように、故障データにワイブル分布を当てはめ形状パラメータを推定することで、発生している故障パターンを分析することもできる。一方、ηは、時間軸上での単位を規定している。ηを変化させることによって、分布を時間軸方向に伸縮させることができる。このように、パラメータを変化させることで、さまざまな故障の形態に対応させることができるため、ワイブル分布は故障分布として広く用いられている。
