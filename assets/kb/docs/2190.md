# 3・4・4 信頼性特性値の推定

> **章**: 第3章 設計のための個別方法論
> **節**: 3・4 信頼性の設計

a. 信頼性データ(reliability data)すでに述べたように、信頼性を数量的に表す特性値としては、信頼度、故障率、MTTFなどがある。これらの値の評価は、試験室やフィールドで得られた信頼性データをもとに行う。いま、個のサンプルで同時に寿命試験を始めて、すべてのサンプルが故障するまで試験を行うとする。この場合は、図3・67(a)に示すような形のデータが得られる。図3・67(a)中×印は、その時点でサンプルが故障したことを示す。これを故障までの時間の大きさの順に並べたのが図3・67(b)で、このようにすべてのサンプルの故障までの時間が与えられているものを完全データという。しかし、完全データを得るために必要な時間の予測は困難である。実際の場面では試験時間に制約があるので、多くの場合、試験を中途で打ち切らなければならない。この場合は、試験を打ち切った時点で、故障に至っていないサンプルがいくつか残る。このようなデータは打切りデータ(censored data)と呼ばれる。中途打切りの方法には、規定時間に達したところで打ち切る方式と、故障が一定個数に達したときに打ち切る方式がある。前者を定時打切り試験方式、後者を定数打切り試験方式と呼ぶ。それぞれの場合に得られるデータの様子を図3・67(c)(d)に示す。○印は、故障に至らなかったサンプルを示している。このほか、サンプルが故障したら新品を補充して常時個のサンプルで試験を行う補充式打切り試験方式がある。この場合も、定時打切り方式と定数打切り方式が考えられる。補充式によって得られるデータを故障までの時間順に並べると、図3・67(e)のように、故障データ(×印)の間に中途打切りデータ(○印)がランダムに挿入された形になる。b.
