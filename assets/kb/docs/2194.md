# 3・5・2 感度解析

> **章**: 第3章 設計のための個別方法論
> **節**: 3・5 最適設計

最適設計やその解法である最適化手法に関連する重要な概念として、感度解析(sensitivity analysis)がある。感度解析は最適設計と関連して考えた場合には、設計変数の変化に対する状態変数、目的関数や制約条件の変化の割合という重要な設計情報を与える。さらに、その情報に基づく最適化手法の主要な手法の一つである傾斜法を形成する。さらに感度解析は次のような、工学上興味深い、しかも重要である諸問題に対して、一つの有力な解析手段を与える。(1) パラメータの重要度の判別(2) パラメータの変動によってあまり影響を受けない性質、すなわちロバスト性の検討(3) 感度(傾斜)に基づく最適化手法に活用することによる最適設計(4) パラメータの変動に伴う挙動等の解を再計算する際の効率のよい再解析(reanalysis)法以下に感度の定義を簡単に述べる。設計変数のいくつかが元の値から微小量変化したとする。ここに式(3・40)の右辺の第1項は、元の関数の値であり、第2項、第3項等は、それぞれテイラー展開の一次の項、二次の項、三次の項等を示す。したがって感度解析とは、一次の感度、二次の感度等の値を計算するための手法といえる。また感度解析と通常呼ばれる場合には単に一次の感度解析を示している。そこで感度解析の多くのコンピュータソフトウェアが用いているように、式(3・40)に現れる感度は多くの場合、差分法によって数値計算的にその近似値を求める方法をとる。なぜならば、個の微係数である感度の計算には、回の関数値の計算をしなければならず、計算時間を要するからである。そこで挙動の感度などを直接的に計算する効率のよい、いわば半解析的方法(semi-analyt-ical method)が種々考えられており、以下にごく簡単に紹介する。半解析的な方法は、基本的には解析の基礎となる基礎式(構造解析の例では静的平衡式、固有方程式、運動方程式など)と基礎式を設計変数で偏微分した式の両式をもとに、各式の特性を考慮して状態変数などの感度を解析的に導く方法である。ここに、半解析とは基礎式そのものが連続系から離散系に近似されたものであり、その後の感度の導出過程は解析的であるという意味で用いられている。構造解析の場合の各種の感度解析の半解析的方法を表3・20にまとめて示す。その導出過程の詳細は文献(143)(144)を参照されたい。また、表3・20に示すほかにいろいろな手法が提案されている。
