# 2・7・1 異常の検出

> **章**: 第2章 潤滑剤
> **節**: 2・7 潤滑管理（lubrication management ）

メンテナンス（maintenance）は機械システムのライフサイクル
（life cycle），すなわち開発から運用・廃棄までの範囲で考えられ
ている．実施されるメンテナンスをライフサイクルに割り振って，
次の三つの階層が想定される ．下層から上層へ①機械システム
の運用段階で実施されるメンテナンス，②運用と開発の両段階にま
たがって実施されるメンテナンス，③開発段階において検討される
メンテナンス．異常の検出は，最下層の①運用段階の保全作業計画
や処置，保全作業評価と並ぶ検査/監視診断に属する．したがって，
異常の検出は，機械システムの運用段階における状態をそれにかか
わる諸情報の観測値と正常値の比較あるいは観測値そのものから評
価することになる．
機械システムの運用中の異常はシステムを構成する機械要素で始
まり，機械要素は主としてトライボロジーにかかわる原因で異常を
生じる．18業種の511事業所のメンテナンス現場における実態調
査 によると，59%の事業所で何らかの機械システムの異常検
出・診断が実施されている．使用されている方法としては，振動法
が38.7 %と最も多く，油分析法が36.3 %で，両者で実施方法の
75%を占めている．温度法（6.3 %)，音響法（4.6%）などがこれ
らに続いている．振動法には振動モード法や変位法が含まれてお
り，油分析法としては油汚染度分析法や油一般性状分析法，フェロ
グラフィー分析法，SOAP法，グリース鉄粉濃度法が採用されて
いる．
振動法 は，機械システムに加速度センサなどを取付け，検出
された信号を時間領域で処理して実効値や波高値，ひずみ度，とが
り度などを求め，それらの値の経時変化あるいは値そのものから正
常あるいは異常の判定を行う．また，信号を周波数領域で解析し，
異常部品の特定を実施する．振動法は機械システムの多様な異常現
象の検出に適している．
油分析（oil analysis ）法は，機械システムを潤滑する油やグリ
ースの性状を測定し，あるいはそれらに含まれる摩耗粒子を分析し
て，システムの状態を評価する．潤滑されるすべての機械要素のト
ライボロジーに関する情報を油が内包しているので，異常の検出・
診断に有効な方法である．先に記したように，油分析法にはいろい
ろな方法があり，潤滑油の性状が異常のおもな原因となる用途では
性状を測定し ，摩耗が発生するところでは摩耗粒子を分析す
る といったように異常検出・診断法の選択が大切である．
前述の調査によると，現状の技術に問題がない，すなわち現状の
技術で異常検出・診断が十分できるとするものが19.6 %に対し，
問題があるとするものが80.4 %となっている．後者には，異常判
定の基準がない，異常判定の基準が適合しない，余寿命判定が難し
いなどの指摘がある．これらの指摘から，(ⅰ)既存の方法の信頼性
向上，(ⅱ)異常現象により強い相関をもつ情報の探索，(ⅲ)複数情
報による診断法の確立などが考えられる．信頼性の向上には，現象
に対応した情報処理法の開発 とデータベースの構築 が不
可欠である．新しい情報の探索ではアコースティックエミッション
