# 2・7・3 潤滑油の劣化と診断

> **章**: 第2章 潤滑剤
> **節**: 2・7 潤滑管理（lubrication management ）

の改善や，基油にはない要求性能付与のための添加剤からなる．し
たがって，潤滑油の性能は，粘度-温度特性や低温流動性などはそ

潤滑油変質・劣化の出発点は油と空気中酸素との化合による酸化
系の銅などの金属は酸化触媒となり，酸化劣化（oxidative degra-
dation）を促進させることがある．潤滑油の酸化劣化は複雑な酸化
反応を経て進み，最終的には劣化酸と呼ばれる腐食性物質や黒色で
粘ちょうな含酸素化合物が生成される．酸化重合体の多くは油に可
溶であり，酸化の進行とともに粘度が増加する．酸化重合が極度に
進むと油不溶性となって油中に析出したり，スラッジとして沈降堆
積し，潤滑トラブルの原因となることがある．潤滑油劣化（lubri-
cant deterioration）にかかわる諸因子と劣化過程での形態変化を
図Ⅱ-2･10に示す．
b.劣化診断（degradation diagnosis ）法 劣化は潤滑油の物
理・化学性状値，油中異物量およびそれらの変化量から診断される
ことが多い．2･2･2項で述べたように用途別各種潤滑油があり，油
種により要求性能や使われ方が異なることから，診断項目や基準は
油種ごとに設定される．
自動車用潤滑油の劣化は通常，エンジン性能と密接にかかわって
いる油の物理・化学性状値の変化から診断され，動粘度，酸価，塩
基価，不溶解分などが一般的項目である．しかし，劣化の基準は，
油交換により性状値や性能が新油レベルに回復可能な変化範囲内に
設定されるべきものであり，通常，各自動車メーカ推奨品質レベル
油を使用した場合は，推奨交換時間内であれば劣化基準内にあると
いえる．建設機械用潤滑油は高負荷使用であることから，上記の一
図 - 2･11 グリースの劣化過程
般性状値のほかに，摩耗金属分や土砂・塵埃などの異物量値や汚染
度測定を診断項目としている場合が多い．舶用ディーゼルエンジン
オイルの劣化診断は，エンジン形式や使用燃料油などを考慮して動
粘度，塩基価，不溶解分，水分などで行われるが，その基準はユー
ザ，エンジンメーカ，オイルメーカ間の協議で決定したほうがよ
い．また，しゅう動部での潤滑状態の追跡や摩耗診断には金属分析
から判断するSOAP法（spectrometric oil analysis program）や
摩耗紛の量と形態から判断するフェログラフィー（ferrography）
法などが用いられている．
工業用潤滑油の種類や使われ方は多種多様であり，潤滑油から機
械の潤滑状態をモニタ・診断する項目として油性状値分析（動粘
度，酸価，水分，不溶解分，色相など)，油中摩耗金属分析
（metal analysis ）［SOAP法，フェログラフィー法，ICP/誘導プ
ラズマ発光分析法（inductively coupled plasma）など］，油中粒
子状物質の計数や重量測定による汚染度管理，添加剤の消耗・変質
量分析（IR/赤外線吸収スペクトル法など）などがある．これらに
よる劣化診断の管理基準は油種や機械により異なるので，潤滑油ご
との変質傾向・特徴を十分に把握し，機械に適合した分析項目と基
準の設定が重要である．
