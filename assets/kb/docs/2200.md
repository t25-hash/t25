# 3・6・1 ライフサイクルエンジニアリング

> **章**: 第3章 設計のための個別方法論
> **節**: 3・6 ライフサイクル設計

ライフサイクルエンジニアリング(life cycle engineering)とは、製品ライフサイクル全体を対象として、その環境調和性、コスト、効率などを最適化、合理化することを目的とした、製品ライフサイクル自体の設計、マネージメント、およびその実現に関わるさまざまな課題を解決するための技術全般を指す言葉である。製品ライフサイクル(product life cycle)とは、素材生産、部品生産、製品設計、製品製造、販売、使用、保全、回収、リサイクル、廃棄などを含む、製品の一生の流れである。従来、ライフサイクルエンジニアリングという言葉は、寿命の長い設備のライフサイクル全般にわたるコスト評価を行うライフサイクルコスティング(LCC:life cycle costing)と類似の概念で用いられたり(すなわち、ライフサイクル全体のコスト最適化)、プラントなどの設備において長期にわたる設備管理、すなわち設備の運用効率の最適化という意味で用いられてきた。また最近は、ますますその重要性が高まっている環境問題(environmental issue)との関係において、ライフサイクルエンジニアリングという言葉が使われ始めている。フサイクルエンジニアリング、特に、製品の一生をおおかた決定するという意味で極めて重要なライフサイクル設計について述べる。ナノテクノロジー、IT 技術に代表されるますます急速に展開するハイテクノロジーや遺伝子操作技術に見られるように、科学技術と社会との関係、特に科学技術が社会に与える悪影響が大きな問題となっている。このため、この問題を意識せずにものづくりを行うことができなくなってきている。冨山は、これをミッション工学と呼び、工学の今後の新しい姿であると述べている。地球環境問題は、その根本的な原因を、大量生産(mass production)・大量消費・大量廃棄型の生産パラダイムに帰着することができる、上記の問題の典型例といえる。そこで3・6節では、地球環境問題解決に向けた今後の工学の姿であるミッション工学の典型例として、ライフサイクルエンジニアリングを取り上げることにする。a. ライフサイクルエンジニアリングの背景現代の環境問題は、物質やエネルギー資源の枯渇、それらの消費に伴う環境汚染や地球温暖化、大量の廃棄物、さらには微量ではあっても人間生存に大きな影響を及ぼす毒性物質など、さまざまな形態で顕在化している。大量の人工物を作り出している製造業に責任の多くがあることは明らかであろう。現代の製造業は、上記のような環境問題に対応した持続可能性を十分に達成しているとはいえない。一方で、産業界における個別の環境対策技術は急速に進歩しており、省エネルギー、廃棄物の分離・分解処理、材料リサイクル、焼1―117却などによる廃棄物処理などに配慮することは、人工物設計・生産技術の常識となりつつある[これを環境配慮設計(ecodesign:environmentally conscious design)と呼ぶ].従来の環境対策技術は、環境汚染物質の除去、省エネルギー技術、廃棄物処理技術に代表されるように、大量生産・大量消費社会を前提としたうえで、その弊害を緩和するための技術が中心であった。しかし、大量生産を前提にしたこれらの緩和技術のみでは環境問題の本質的な解決は困難であり、地球環境にとって許容可能な範囲で、生活の質、雇用、企業の利益などを維持できるようにするために、経済システムや社会システムを変革することが要求されているのである。別の言い方をすれば、現代の環境問題の原因が大量生産にあるとすれば、人工物システムを地球環境保全の観点から適切に管理して、地球環境負荷に対して最適な性能を発揮させることは、製造業の責任である。政治や社会、経済などの各分野の活動は、環境問題の解決に大きな影響があるが、人工物の適切な構成と管理の方法を提示できるものは製造業しかない。この意味において、製造業の責任は重いと同時に、環境問題を根本的に解決できる可能性ももっている。現代の人工物の問題は、それらが人工的に作り出されたものであるにもかかわらず、工場から出てしまうと、利用者の中に埋没してしまい、製造者の目からは見えなくなることである。これらの人工物は、やがて廃棄物となって社会に現れてくる。このように、途中で未管理状態におかれた人工物を、環境に適切に管理し処理していくことは困難である。現代の多くの人工物には、そのライフサイクルにミッシングリンク(missing link)があって、サイクルとはなっておらず、ライフサイクルを適切に管理することが困難になっている。この状況を根本的に改善するためには、人工物の基本概念まで立ち返って、さまざまな製品の関係も含めた統合的な人工物のライフサイクルを議論することが必要であろう。このようなライフサイクルの構築と管理をも製造業の範ちゅうに含めていくことにより、新たな製造業の可能性も開けてくるのではないかと期待される。結果的に、現在われわれに欠けているものは、要素技術の開発よりもむしろ、製品ライフサイクルをシステム化し、ライフサイクル全体をいかに最適化するかという視点であり、技術なのである。b. ポスト大量生産パラダイムとインバースマニュファクチャリング上記に述べたように、地球の有限性のもとでの製造業のあり方として、冨山は、ポスト大量生産パラダイムという考え方を提唱している。その根幹をなす考え方は以下のとおりである。生産活動が富の源泉である以上、持続可能性を満足しながら富の創造を図るメカニズムが必要である。そのために人工物そのものの量的生産高を減らし、製造業における資源・エネルギーの消費量を減らす一方で、ものではなくものが提供するサービスを中心として売るという考え方が必要になる。結果として、製造業は製品ライフサイクル全般を設計、支援、マネージメントするライフサイクル産業へ転換すると、冨山は主張している。以上の議論から、これからの生産は、人工物を大量に、効率よく生産するのではなく、消費資源、廃棄物を最少化しながら、社会が必要とする機能を提供、維持、管理するための活動に転換しなければならないことがわかるであろう。この考え方は、インバースマニュファクチャリング(inverse manufacturing)と呼ばれている。インバースマニュファクチャリングとは、製品価値や生活の質を維持、向上させつつ、製品ライフサイクル全体の視点から、資源・エネルギー消費量、廃棄物量、および環境負荷(CO、有害物など)を最少化するような循環型製品ライフサイクルシステムを実現することを目的としている。これまでの設計生産の基本的な視点は、機能・品質の高いものを設計し、いかに効率よくたくさん作るかであり、廃棄物処理やリサイクルの視点は、排出されたものをいかに安全に大量処理できるかであった。これらは要素技術としてはそれぞれ重要であるが、イン1―118設計工学バースマニュファクチャリングにおける基本的な視点は、これまでの技術を活用しつつ、これらの視点に加えて、要求されるサービスを満足させ利益を確保しつつ、いかにむだに資源・エネルギーを消費しないで済ませるかにある。c. ものの提供からサービス提供によるビジネスへ循環型の製品ライフサイクルを前提にした場合、製造業のビジネス戦略もおのずから変革を迫られる。例えば製品ライフサイクルの管理に適した人工物の供用形態としては、製品販売よりも製品のレンタル・リースが適しているといわれている。さらに、循環型の製品ライフサイクルにおける新たなビジネスチャンスは、製品販売よりもむしろその後工程、すなわち、使用、メンテナンス、回収段階でのユーザ支援や逆工程にあるといえる。すなわち、製品の製造販売を主活動としていた製造業が、製品をサービスの増幅装置としてとらえ、ビジネスの進め方を人工物の提供から、人工物が発現するサービスの提供へ転換することが、大きなビジネスチャンスを生み出す鍵となってくる。(2)
