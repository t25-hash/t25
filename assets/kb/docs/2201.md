# 3・6・2 リサイクルの方法

> **章**: 第3章 設計のための個別方法論
> **節**: 3・6 節では,この流れに沿って,環境調和性という面から,ライ

現在,リサイクル(recycle)という言葉は二つの意味で使われることが多い.一つは,使用済みの製品,材料,容器などを単に捨てるのではなく,再利用して有効に活用するという広い意味に使われる.もう一つは,図3・93に示す循環の中のマテリアルリサイクル,ケミカルリサイクル,サーマルリサイクルの意味で狭義に使われる.専門用語としては後者の狭義の意味で使われることが多い.図3・93は,家電製品,自動車などの大量生産型工業製品のライフサイクルの流れを示している.製品を製造して使用し,使い終わるまでと,使用済み製品をリサイクル,廃棄するまでを対比させて,順工程,逆工程,もしくは人間の血液の流れをまねて動脈工程,静脈工程と呼ぶ.このような人工物の循環にはいくつか方法があり,(1)使用済みの製品を清掃・洗浄し,故障部品を交換して製品として再生する再生産(リマニュファクチャリング,remanufacturing),(2)使用済み製品から部品を取出し,再生した後,別の製品に取付ける部品リユース(reuse),(3)例えば,自動車のボデーを鉄素材に再生するといった,材料として再生するマテリアルリサイクル,(4)ペットボトルからPET(polyethylene terephthalate)樹脂を取出し化学原料とするといったケミカルリサイクル,(5)燃やすなどしてエネルギーを回収するサーマルリサイクル,(6)埋立て処分などの最図3・93製品ライフサイクルの流れ終廃棄に大別できる.さらにリサイクルではないが,後述するライフサイクル設計において重要な方法として,(7)使用中の製品をできるだけ長く使えるように故障部品などを修理するメンテナンス(maintenance),(8)製品の設計段階で,できるだけ資源を使用せずに必要な機能・性能を実現するリデュース(reduce)がある.一般に,できるだけ小さいループで処理する,すなわち,リサイクルよりもリユース,リユースよりもリデュースのほうが全体の環境負荷が小さいといわれている.このため,よく3R,すなわち,リデュース(reduce)・リユース(reuse)・リサイクル(recycling)という言葉が使われる.この場合のリユースとは,上記の再生産と部品リユースを合わせた概念である.図3・93には示していないが,別の分類方法として,複写機のモータを再度複写機で利用するといったように,同一種類の製品に再利用することを閉ループ(クローズドループ)リユース/リサイクルといい,自動車のボデーを建設用鉄骨に再生するというように,別の使い道で再利用することをカスケードリユース/リサイクルという.図3・93の中で,現実の世の中で行われている循環生産は,大まかにいって以下のように分類できる.(1) 家電リサイクル法や容器包装リサイクル法などの種々の法規制や世の中の環境意識の高まりとともに,多くの製品では,カスケード型のマテリアルリサイクル,ケミカルリサイクル,およびサーマルリサイクルが実施されている.これに該当する製品として,自動車,大型家電品,パーソナルコンピュータ,ペットボトルなどが挙げられる.ただし,閉ループ型ではなくカスケード型であることに注意が必要で,多くの場合,コストが余分にかかる,リサイクル材料は新品材料よりも品質が低下する,そのために使用先が見つからない,などの問題点が指摘されている.これらの製品においては,リサイクルを行いやすくする設計(リサイクル設計,design for recycling)が導入されつつある.具体的には,3・3・4項のDfD やDfR が該当する.(2) レンズ付きフィルムや複写機などでは,製品を回収し,製品の製造のために部品をリユースしたり,再生製品を製造するためのリマニュファクチャリングが行われている.これらの製品で適切に循環生産を行うためには,3・6・3項で述べるライフサイクル設計が必要不可欠になる.(3) 自動車に代表されるように中古市場が確立されている製品もある.また,パーソナルコンピュータや家電品の中古市場が盛んになりつつあり,自動車の修理に利用する部品リユースのための中古部品市場も拡大傾向にある.これらも,使用済み製品を検査,清掃,場合によっては修理が行われるので,リマニュファクチャリングや部品リユースに分類される.ただし,実施主体がメーカよりも販売業者などが中心となる点で(2)と質的な違いがある.(4) 建築物,自動車,工作機械,エレベータなど寿命が長く高価な製品においては,古くからメンテナンスが必要不可欠となっている.いずれにせよ,3・6・3項に述べるように,これらの方法を組合わせ,製品ライフサイクル全体の環境負荷を大幅に削減することが最も重要なライフサイクル設計の課題である.
