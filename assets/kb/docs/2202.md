図3・93のようなさまざまな循環生産の実現を考えた場合、リサイクルだけでなく、リデュースやリユースを含めた3R(reduce,reuse,recycling)を適切に組合わせて、インバースマニュファクチャリングを実現する必要がある。このように、製品の一生を適切に設計することをライフサイクル設計(life cycle design)と呼ぶ。ライフサイクル設計が必要な理由は、製品の多くの特性は設計時点で決められてしまうからであり、さらには、製品とその製品がたどるライフサイクルとには不可分な関係があるからである。例えば、金属と多種類のプラスチックが分解困難な形で締結された構造の製品は、リサイクルすることが極めて困難、高コストである。す1―119なわち、ライフサイクル設計は、製品と製品のライフサイクルを統合的に設計しようとするものである。表3・35に従来の設計、3・6・2項の(1)で述べた多くの製品で実施されているリサイクル設計、およびライフサイクル設計の違いをまとめて示す。このようなライフサイクル設計と従来の製品設計を設計者から見た相違は、以下のようにまとめられる。(1) 製品ライフサイクル全体への視野の拡大従来の設計はおもに製品使用時の性能に着目してきたが、ライフサイクル設計においては製造から使用終了後の再利用、廃棄に至るまでの製品のライフサイクル全体を見通して設計を行わなければならない。この問題は技術的には設計と製造の両工程を考えるコンカレントエンジニアリングの延長上にあると考えられるが、設計に要求される視野の広さの点で質的な変化が起きる。(2) 多視点性上記に付随して、設計に要求される、もしくは設計時点で考えなければならない項目、属性が多様かつ複雑になる。極端にいえば、従来の設計の目標はより安い価格でより高性能の製品を作るということであり、ある意味でその目標は明確であったが、環境問題を考えた場合、省資源、省エネルギー、高リサイクル性、有害物質の削減など非常にさまざまな要求項目を満たす設計を行わなければならなくなる。なおかつこれらの要求項目間にはトレードオフ関係があることが多い。このように、設計問題が複雑化、多様化することがライフサイクル設計の第二の特徴である。以上のようなライフサイクル設計においては、図3・94に示すように、対象製品のライフサイクル特性を十分に把握し、ビジネス戦表3・35 従来の設計、リサイクル設計およびライフサイクル設計従来の設計リサイクル設計ライフサイクル設計視点製品の使用製品の廃棄後製品ライフサイクル全体目的使用段階でのコストパフォーンスの最大化リサイクル率の向上ライフサイクル全体での環境効率の向上逆工程考えないごみのリサイクル循環を前提に価値を維持環境問題考えないごみ問題の解決循環させることによる環境負荷の大幅な削減経済性販売までの最適化リサイクルがコストを高める循環させたほうが経済性が向上する使用者便利なものを買う。すぐ買い換えるリサイクルコストを負担循環させることにより、コストが安くなる市場経済ごみ問題家電リサイクル法循環型社会3R拡大製造者責任汚染者負担の原則グリーン購入促進要因の例製品の例従来型の耐久消費財全般最近の自動車、家電レンズ付きフィルム、複写機図3・94ライフサイクル設計のポイント1―120設計工学図3・95 製品シリーズと世代略の策定、ライフサイクルシナリオ(life cycle scenario)の作成(ライフサイクルオプションのベストミックスを選択し、循環方法のシナリオを作成すること)、製品設計、プロセス設計を統合的に行うことである。ここで、ライフサイクルオプション(life cycle option)とは、
