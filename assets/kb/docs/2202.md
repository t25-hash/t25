# 2・1・4 位相進み-遅れ補償（phase lead- lag compensation）

> **章**: 第2章 線形制御
> **節**: 2・1 古典制御

PID制御と並んで現在でも広く用いられている古典的な設計法
として，位相進み-遅れ補償がある．この位相進み-遅れ補償は，こ
れから説明するような位相遅れ要素や位相進み要素をループ中に直
列に順次組込んでいく手法であり，簡単なPID制御に比べて設計
の自由度がある．このため，以下で述べるような開ループ伝達関数
の周波数応答を整形すること（ループ整形）が可能となり，モデル
の不確かさに対処できる設計法となっている．ただし，その設計手
順は必ずしも系統的になるとは限らず，実際には繰返しを必要とす
ることが多い．
