図3･96にライフサイクル設計の全体の流れを示す．3･3･4項で
挙げたDfE（環境調和型設計）のための要素設計技術は，この図
3･96中の中央部分左の製品設計で有効に活用される．逆にいえば，
図3･96に示すように，それ以前にビジネス戦略やライフサイクル
シナリオをきちんと決めておかなければならない．やみくもに分解
性設計を行っても，それは多少のリサイクル性向上に寄与するであ
ろうが，製品循環の実現にはほど遠くなってしまう．製品設計を実
施する前に，環境調和性の観点から，これらビジネス戦略やライフ
サイクルシナリオを決めることがライフサイクル設計の重要な点で
ある．そこで，本項ではこのライフサイクルシナリオの決定法につ
いて概観してみよう．
ライフサイクルシナリオの決定段階で決めるべき項目は，その前
段で決定されるべきビジネス戦略，環境目標，製品コンセプトに基
づいて，製品および構成部品のライフサイクルのシナリオを作成
し，そこから，次工程の製品設計やプロセス設計の目標を決定する
ことである．
この段階がライフサイクル設計の最大の特徴となるわけである
が，その支援手法がいくつか提案されている．例えば，山際
は，製品設計の初期段階において製品の循環戦略を考えるコンセプ
トモデル戦略を策定することの重要性を指摘している．また，表
3･36に示すように，代表的なコンセプトモデル（表3･36の各列）
と製品特性，企業特性，市場特性，社会特性などの種々の特性との
関係を整理している．この表3･36は，ライフサイクルシナリオを
策定する際の一つのガイドラインとなる．
梅田ら は，既存製品が廃棄される原因を分析し，その原因を
取り除くように改良設計を行うことで製品廃棄を軽減させる廃棄要
因分析（disposal cause analysis）表（図3･97）という手法を提案
している．この手法は，廃棄要因を物理寿命である機能消費（飲料
やフィルムなど）と故障・寿命，価値寿命である外観（スポーツカ
ーや衣類など），容量・サイズ（冷蔵庫や子供靴など），および新機
能・機能向上（パーソナルコンピュータやディジタルカメラなど）
の5種類に分類しており，製品使用者の廃棄理由から，その原因と
なっている部品と廃棄要因の組を導出する．この結果から，対象製
品において価値寿命が支配的なのか，物理寿命が支配的なのか，廃
棄の原因となっている部品，廃棄の原因となっておらず，そのため
リユースに適していると考えられる部品などが明らかになる．本手
法は，これらの情報から，対象製品のライフサイクルシナリオの作
成を支援する．
小林ら は，ライフサイクルシナリオの作成段階をライフサイ
クルプランニング（life cycle planning）段階と呼び，ライフサイ
クルプランニング支援手法を提案している．この手法では，品質機
能展開（QFD:quality function deployment）法 を用いて，
