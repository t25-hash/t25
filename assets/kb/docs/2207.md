# 3・6・6 まとめ

> **章**: 第3章 設計のための個別方法論
> **節**: 3・36 に示すように,代表的なコンセプトモデル(表3・36の各列)

環境問題解決へ向けての今後の人工物設計,生産のあり方を議論し,その総合的システム化技術としてライフサイクルエンジニアリングの重要性を述べた.ライフサイクルエンジニアリングにおける重要な課題の一つは,循環を前提とした製品ライフサイクルの設計であり,その基本概念,基本的な流れを整理し,特に製品ライフサイクル設計の初期段階で重要なライフサイクルシナリオの策定方法をいくつか概説した.この分野は成熟しているとはいいがたく,十分に実用化可能なガイドラインは存在しないが,今後の人工物設計を考えるうえでの一助になれば幸いである.
