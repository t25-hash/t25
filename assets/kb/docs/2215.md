# 3・7・8 シミュレーションを設計に生かすために

> **章**: 第3章 設計のための個別方法論
> **節**: 3・111 にフィンの付いたCPU モジュールのシミュレーションモデ

図3・132に設計におけるシミュレーションの位置付けを示す.概念設計,機能設計,配置設計,構造設計,製造設計と設計が進むにつれて,設計情報の詳細度は高まる.しかしながら,設計の下流では多くの設計に関するパラメータは決定されており,設計に自由度は多くはない.また,設計の進捗に伴い,試作等も並行して実施されるため,設計の下流では実体による設計機能の検証が可能となり,実体による機能検証を行うか,シミュレーションによる設計機能検証を行うかは,単に効率の面だけで決定される場合が多い(設計下流でも試作ができない場合はシミュレーションの選択しかない).一方,設計の上流ではリピート製品のように過去の知見が存在する場合を除いて,実体による設計機能の検証は困難である.しかしながら,設計上流では設計情報の詳細度が低いため,いわゆるコンピュータシミュレーションをこの段階で実施することはできない.そこで,設計上流においては設計シミュレーションによる設計機能検証,設計パラメータの決定(最適化等を活用)が重要となる.設計シミュレーションの情報は設計の進捗とともに,シームレスにコンピュータシミュレーションに受け継がれることが理想である.実体による設計機能検証と,シミュレーションによる機能検証はお互いに排他的関係にあるのではなく,補完関係にある.シミュレーション結果を実体で検証し,実体における現象をシミュレーションで確認することにより,シミュレーションというバーチャルな世界から実体というリアルな世界へ効率的にかつ正確に設計情報を反映させることができる.以上述べたように,シミュレーションを設計に生かすための環境は整ってきている.あとは使い手の問題に帰着する.シミュレーションを設計で使いこなすためには,機械工学に関する基礎技術,すなわち,構造力学,機械力学,熱力学,流体力学,制御工学,統計学などについて深く理解していることが前提となる.ラピッドプロトタイピング(rapid prototyping)とは,3D-CAD などの形状データに基づき,自動的,短時間に,さまざまな立体形状を造形する技術の総称である.日本語では高速試作,迅速試作などと表記される.切削加工に代表される従来からの加工技術と比較して,造形可能な形状の制約が少なく,工場のような特殊な環境でなく居室にも設置できるような装置で実現可能であるなどの特徴を有することから,設計やデザインの試作,評価などを中心に活用されている.1―133(a) 3D-CAD モデル(b) 水平平面で切断した断面形状(c) 断面形状を下から順次造形,積層図3・133 積層造形の基本概念
