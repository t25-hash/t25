# 3・8・1 基本概念

> **章**: 第3章 設計のための個別方法論
> **節**: 3・8 ラピッドプロトタイピング

ラピッドプロトタイピングにおいては一般的に,造形可能な形状の制約が少ない積層造形法(layered manufacturing)というアプローチがとられている.図3・133に積層造形の基本概念を示す.3D-CAD で表現された造形対象の立体形状を,一定間隔の水平平面で切断した断面形状の集まりに変換する.その各断面形状を一定の層厚で製作し,それを順次積層していくことにより,元の立体形状が造形される.これにより,例えば切削加工では工具が入り込めず加工ができないような入り組んだ立体形状であっても,容易に造形することができる.積層造形に要する時間は積層の数に大きく影響されるため,長い形状は横長の姿勢で造形することが有利である.原理から明らかなように,造形物の表面には積層による段差が生じ,その方向は造形時の立体形状の姿勢により異なる.積層段差を除く必要がある場合は,適宜切削や研磨などの後加工で仕上げる.積層厚(1層の厚さ)を小さくすると段差は小さくなるが,同じ高さを積層するのに必要な層数が増加し,造形時間も増加する.また,3・8・2項で後述する各造形方式において,安定して形成できる材料の層の薄さにも限界がある.以上のような条件から,市販の積層造形装置においては0.02~0.3mm程度の積層厚が用いられることが多い.造形可能サイズは数十cm立方程度が一般的である.
