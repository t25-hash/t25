# 3・8・2 造形方式

> **章**: 第3章 設計のための個別方法論
> **節**: 3・8 ラピッドプロトタイピング

積層造形法による造形では、実際にどのような物理現象、材料、造形プロセスで積層を行うかにより、さまざまな造形方式が存在する。a. 光造形法光造形法(stereolithography)とは、光が当たると硬化する液体樹脂(光硬化性樹脂)を選択的に硬化する造形方式である。図3・134(a)にそのプロセスを示す。樹脂槽に液体の光硬化性樹脂を満たし、その表面にレーザ光などを走査して硬化したい断面形状の輪郭および内部の塗りつぶしを描くことにより、目的とする断面形状を選択的に硬化する。1層を硬化した後、造形台を1層の厚さ分だけ下に沈めて上部に新たな液体樹脂の層を形成し、次の断面形状を光硬化する。これを繰返すことにより、液体中に目的の立体形状が積層造形されていく。最後に造形台を上昇させ、液中から光硬化した造形物を取り出す。造形後に造形物表面に付着した液体樹脂の洗浄や、紫外線ランプによる後露光などの仕上作業を行う。なお、液中に固体を造形する光造形法においては、目的とする造形形状だけでなくサポート構造(3・8・3項に後述)も合わせて造形する必要があるが、加熱時に低粘度ゾル状態、常温時にゲル状態となる特殊な光硬化性樹脂を用い、固体(ゲル)中に固体(光硬化)を造形するサポート構造不要な光造形法も実用化されている。(b) 規制液面方式図3・134 光造形法樹脂の自由液面で層を硬化する図3・134(a)の方式は、自由液面方式と呼ばれる。この方式においては、樹脂の粘性や表面張力、下層の造形物形状の影響などにより、液面が高精度な平面になりにくかったり時間がかかる場合がある。そのため、板状のリコータ(recoater)で液面の余剰な樹脂液を除去して平滑化するなどの操作を行う。それに対して図3・134(b)に示す透明な窓板を通して樹脂を硬化させる方式においては、硬化する液面が窓板で規制されているため、高精度な平面の液面を形成することができる。これを規制液面方式と呼ぶ。自由液面方式では樹脂槽に一定量の樹脂液を満たしておく必要があるのに対し、規制液面方式においては層厚以上の深さの液量を供給すればよいという利点がある。その一方で、規制液面方式においては次層を形成するために造形台を上昇させたときに、造形物と窓板が固着してはく離せず、造形物が破損する可能性がある点が短所である。材料は光硬化性樹脂に限られるが、硬化後の機械的性質に優れた材料の研究開発も進んでいる。光硬化性樹脂は硬化時に収縮するため造形物に反りやひずみが生じうるが、ウレタンアクリレート系樹脂に比較して反応が緩やかなエポキシ系樹脂は反りが少なく、高精度な造形に適している。光硬化性樹脂は、可視光型、紫外光型など、硬化反応する光の波長によりいくつか種類があるが、可視光で硬化する樹脂は通常の作業環境の中で硬化するなど使用上の制約が生じることもあり、紫外光で硬化する樹脂が用いられることが一般的である。一般的な解像度(ビームのスポット径)は0.05~0.2mm程度である。光造形の特徴は高精度、高解像度な造形が可能な点である。b. 溶融押出法溶融押出法(FDM : fused deposition model-ing)とは、熱可塑性材料を加熱、溶融して微小穴径のノズルから押し出しながらノズルを走査し、目的部分のみに選択的に材料を供給する方式である。図3・135にそのプロセスを示す。材料を加熱、溶融してノズルから押し出しながらノズルを方向に走査して1層の形状の輪郭および内部塗りつぶしを描く。押し出された材料は空冷され硬化する。1層を硬化したらその上に次の層の造形を行う。それを各層について繰返すことにより、空間中に立体が造形される。空間中に造形を行うため、造形時にサポート構造(3・8・3項に後述)が必要となる。材料としては、ABS 樹脂、ポリカーボネート(PC)樹脂、ポリフェニールソルフォン(PPSF)樹脂などの熱可塑性材料を使用できる。造形解像度の主要因はノズル穴径であり、0.2~0.3mmが図3・135 溶融押出法(a) 自由液面方式図3・136 選択焼結法一般的な値である。装置が比較的安価に実現でき、紫外光、溶剤などを使用しないので、通常の居室環境で使用できる点が特徴である。c. 粉末結合法粉末結合法(powder-based method)とは、粉末材料を目的部分のみ選択的に結合して立体を造形する方式であり、選択焼結法(SLS : selective laser sintering)と三次元印刷法(3D printing)の2種類がある。造形物は周囲の粉末に埋まって支えられるため、サポート構造(3・8・3項に後述)は不要である。選択焼結法とは、プラスチックや金属などの粉末材料を選択的に焼結して積層造形する方式である。図3・136にそのプロセスを示す。この方式では、粉末材料をローラなどで平面状にならして薄い層を形成し、その表面を赤外線レーザ光の走査で選択的に加熱して粉末材料を焼結、溶着し、断面形状を造形する。溶着前の粉末密度は50%程度であるが、溶着後は95%以上となる。次に積層厚だけ造形台を降下し、その上に次の層を構成する粉末材料を薄く敷き詰める。それを各層について繰返すことにより、粉体中に埋まって立体形状が造形されていく。最後に造形台を上昇して粉末中から硬化物を取出し、表面に付着した粉末をエアブローなどで除去する。解像度(ビームスポット径)は0.4mm程度、材料粉末の粒径は0.02~0.08mm程度であり、ナイロン、合成ゴム、金属(ステンレスなど)、砂など多くの種類の材料を使用できる点が特徴である。三次元印刷法は、粉末材料を供給し薄い層を形成する点は選択焼結法と同様であるが、その表面にプリンタに使用されているインクジェットの原理で液体バインダ(接着剤)を噴射して粉末を接着する。図3・137にそのプロセスを示す。材料はデンプン、石こうなどであり、粉末を接着剤で固めた状態であるため造形物の強度は小さく、ワックスを浸み込ませるなどの後処理を行う。カラープリンタと同様に着色した複数のバインダを塗り分けることにより、形状だけでなく色彩分布を再現したモデルを造形することが可能である。また、20~30秒程度で1層を造形、1時間で30~50mmの高さを積層できる高速性も特徴である。d. シート積層法シート積層法(lamination method)とは、紙などのシート状材料を積層して造形する方式であり、LOM(laminated object manufacturing)法とも呼ばれる。図3・139にそのプロセスを示す。紙や樹脂などのシート状材料を1枚供給して造形物に接着し、その層の目的断面の輪郭状にカッタやレーザで切込みを入れ、1層分の造形を行う。それを各層について繰返し、立体を積層造形する。造形物は周囲のシート状材料に支えられているため、サポート構造(3・8・3項に後述)は不要であるが、造形終了後に不要部分を取り外す作業が必要となる。そのため、各層においては造形物の断面の輪郭形状だけでなく、不要部分にも格子状の切込みを入れておく。それによって、造形終了後の不要部分をブロック状に除去していく。取外しが困難な不要部分が発生するような複雑な形状や、不要部分の取外し時に破損する可能性のある薄壁や微細形状などの造形には適していないが、装置や造形コストが安価に実現でき、居室環境でも使用できる。f.
