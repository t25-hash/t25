# 3・8・3 サポート構造

> **章**: 第3章 設計のための個別方法論
> **節**: 3・8 ラピッドプロトタイピング

積層造形によって複雑な立体形状が造形可能であるが、サポート構造の必要性を考慮する必要がある。例えば図3・133(c)の飛行機のような形状を下から順に作成していくと、中央の胴体の下部は突然空中に出現する[図3・140(a)].3・8・2項に前述したいくつかの造形法のうち、粉末など固体中に造形を行っていく場合には、このような孤立部分も周囲に支持されるが、空中や液中に造形を行う場合にはこのような孤立部分は安定して存在することができない。この問題を避けるために積層造形においては、造形データをチェックしてこのような孤立形状が発生する部分を事前に検出し、目的形状とは別にサポート構造と呼ばれる足のような構造を同時に造形していく。孤立部分のほか、図3・140(b)のように大きく張り出した形状を安定して造形するためにもサポート構造が必要となる。造形材料は、光が当たって硬化するときに体積が収縮したり、造形時の加熱、冷却によって膨張、収縮するといった性質がある。そのため、大きく張り出した部分ではそのようなひずみが顕著に現れ、形状がたわんだりゆがんだりする現象が発生する。そのような変形を抑制して正確な形状を造形することも、サポート構造を挿入する理由の一つである。サポート構造の挿入は造形用のソフトウェアが自動的に必要箇所を検出してデータを追加するが、造形終了後の取外しは一般的に手作業となるため、サポート構造は取外しが容易なように、造形プロセスにおいて次のような調整がなされる。(1) モデル本体部分に比べサポート部分は露光量を小さくして光硬化反応を弱くし、強度を小さくする。(2) モデル本体部分は材料を密に充てんするのに対し、サポート部分は走査の間隔をあけるなどして材料を低充てんにし、強度を小さくする。(3) モデル本体部分とサポート部分に異なる材料を用いる。例えばモデル本体部分に比べ低融点の材料でサポート部分を造形すれば、加熱によりサポート部分を優先的に溶融できる。また、サポート部分に水溶性材料を用いると、水でサポート部分のみを溶かして除去することができる。同一の立体形状でも、造形時の姿勢によりサポートが付加される面が異なり、モデル表面にサポートを取り外した跡が残る場合もあるため、それも考慮して目的形状の積層方向を決める必要がある。
