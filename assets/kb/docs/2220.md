# 3・8・5 設計検証のためのラピッドプロトタイピング

> **章**: 第3章 設計のための個別方法論
> **節**: 3・8 ラピッドプロトタイピング

現在では3D- CAD，CGの発達により，ある程度までの設計，
デザインの検証は現実感のある画像表示により画面上で行える．し
かし，例えば携帯電話など実際に手にとって操作性を確認する必要
があるような製品の設計，デザインの検討や評価は，実際のものと
して触って行うことが効果的である．ラピッドプロトタイピングで
造形されたモデルや，それに一部実際の部品などを取付けたワーキ
ングモックアップは，そのような目的のために効果的に使用するこ
とができる．また造形物の特性（機械的強度，耐熱温度など）が条
件を満たす場合には，ラピッドプロトタイピングによる造形物は流
体実験など実験用の部品や模型にも用いられている．
設計，デザインとは異なるが，CTなどの医用画像データに基づ
き頭骨などを造形し，外科手術においてどこをどのような手順で削
り，縫合するかなどの手術の術式検討にも使用されている．
