# 3・9・2 新しい工業デザインの方法

> **章**: 第3章 設計のための個別方法論
> **節**: 3・9 工業デザインのための方法

a. 従来の方法との比較1985年以前では、あいまいな企画やスペックのまま、デザインを行うスタイルが多かったようである。このような状況では、徐々にコンセプトを収れんさせる従来の方法は、なんら問題はなかった。しかし、可視化の前に検討しなくてはならない条件が多数ある状況では、このような方法では効率が極めて悪い。それを克服する新しいデザインの方法では、システム設計の考え方を取り入れつつ、コンセプトを徹底的に検討し構築した後、可視化を行う必要がある。その理由は可視化の前に、頑強性の高いコンセプトを構築することにより、試行錯誤の時間をなくし、効率的なプロセスにすることが可能であるためである。この考え方に基づいて考案された工業デザインの方法論の一つとしてヒューマンデザインテクノロジー(HDT : human design technol-ogy)の考え方を工業デザインに適用した方法を以下に紹介する。b. デザイン方法の定義・背景今日の工業デザインは、人間工学面での検討やPL,CS とユーザ優先の考え方(ISO13407)が製品開発の重要な要件となり、従来よく行われていた製品のハード1―137ウェア面の要求事項重視型開発からユーザの要求事項重視型開発にシフトしてきている。ユーザの要求事項重視型開発とは、ユーザニーズを重要視した製品開発方法である。この方法をとる背景は、特に家電製品とか電子機器などの分野で技術的な差がなくなっていることと、ユーザの要求事項を重視せざるを得ない社会環境ができているためである。下記に製品開発上のポイントと問題点を示す。(1) 開発スタンスのポイント・各社の技術力が均等化し、ユーザニーズを具現化できる製品企画力が重要になってきている。(2) 開発組織の問題点・製品開発の上流の各段階(調査、企画、デザイン、評価など)で、専門性が高まり、セクショナリズムが起こり、開発製品をユーザの視点で俯瞰せず、情報の共有が困難になっている。(3) 人間優先の考え方の浸透・商品選択の際、使い勝手や感性が訴求要素、差別化要素として重要視されてきている。・高齢者や障害者に対する配慮などユニバーサルデザインの視点が重要になってきている。以上の状況を踏まえ、HDT は、「マーケティングリサーチ、人間工学(ergonomics)、認知科学(cognitive science)、工業デザイン(industrial design)、ユーザビリティ工学(usability engi-neering)、統計(多変量解析、multivariate analysis)などを統合化し、人間優先および魅力ある商品を実現するデザイン・設計技術である。つまり、人間を科学的に解析して、人間に関する諸情報(生理、心理、認知、行動など)を設計要件にする技術ともいえる」と定義される。従来の直感に依存していた方法から脱却することを目指したものであり、できるだけ分析的で、定量化の視点からの見直しを促進することにより、デザイン・設計要件の漏れのないユーザニーズに立脚した確実な商品作りを達成することをねらって構築された枠組みである。なお、工業デザインを支援するうえで、マーケティングリサーチ、人間工学、認知科学、ユーザビリティ工学、統計(多変量解析)などが背景理論として活用できる場合は一般に多く、HDT もそれらを統合的に用いる構成になっている。c. デザイン開発プロセスここでは、デザインプロセスを以下のように考える(図3・143)。(1) ユーザニーズ収集ステップ:3P(ポイント)タスク分析や直接観察法などによって、ユーザニーズを抽出する。(2) 状況把握ステップ:コレスポンデンス分析により、市場における製品がどのようなイメージで見られているかを把握する。(3) 製品コンセプト構築ステップ:ユーザ要求事項に基づいて構造化された製品のコンセプトを構築する。コンセプト項目はウエイト付けされる。(4) デザイン(総合化)ステップ:構造化コンセプトに基づいて、デザイン案を可視化する。(5) デザイン評価ステップ:可視化されたデザイン案に対して、AHP等で評価を行う。(6) ユーザ使用実態調査:商品を購入し使用したユーザに対し行う調査で、得たデータは製品企画時等に役立てる。
