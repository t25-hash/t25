# 3・9・5 感性デザインの方法

> **章**: 第3章 設計のための個別方法論

感性工学のアルゴリズムは,あるオブジェクトに対するターゲットユーザの感性情報とオブジェクトの物理情報との関係を多変量解析やニューラルネットワーク等により明確にし,最適な感覚のときの物理情報を特定して,デザインや設計に反映させることにある.最も簡単な方法は,オブジェクトの物理情報をカテゴリに分け,SD 法による感性情報との関係を数量化I類で明確にして,SD 法で使われた形容詞のイメージに対応するオブジェクトの特性を決めることである.あるいはバーチャルリアリティを使って,ユーザの感性を抽出してデザインや設計に翻訳する方法も提案されている.ただし,ここで注意しなくてはならないことは,これらのデータは基本的に人間の過去のデータに基づいていることである.新しい感性を生むわけではない.例えば,市場に出ているテレビのデザインをターゲットユーザに見せて,SD 法によりデータをとり,因子分析でそのイメージ構造を特定しても,そのアウトプットはあくまで市販されているテレビのデザインの中での話に終わってしまう可能性が高い.また,デザインの推論のアルゴリズムがいろいろ提案されているが,実用化されているという具体例はあまりない.現状の感性工学はデザインの専門家用というより,おもにデザインの非専門家が営業活動などでユーザのイメージを具体的な形などに翻訳する機能にとどまっている.HDT に基づいた場合,感性デザインの方法は以下のように構成される.HDT では感性設計項目として,(1)デザインイメージ,(2)色彩,(3)フィット性,(4)形態,(5)機能性・利便性,(6)雰囲気,(7)新しい組合せ,(8)質感,(9)意外性の9項目を挙げている.これらの項目は,工業製品から始まって空間,食べ物,風など多岐にわたっている対象物を示して,被験者に自分が感性がよいと感じるオブジェクト(ものにこだわらず何でもよいとした)を自由に提出してもらったものを分析してまとめたデータである.一連の項目の中でも,ものづくりの際には,特に機能性・利便性のよいものは感性を生じさせる要素の一つであることに配慮する必要がある.このことは野村総合研究所の調査データでも同様な結果が出ている.これに限らず,上記の9項目ははん用性の1―143図3・146 UIデザインフォーマットあるキーワードであるが,場合によっては各業界によってその内容が変わるので,カスタマイズする必要がある.前述したように,9項目は3層に分けることができた.第3層の感性の基本構成要素(新しい組合せ,意外性)は,新たな感性を開発する際の手がかりとして位置付けられ,第2層に影響を与える.第2層の感性のインタフェース要素(色彩,フィット性,形態,機能性・利便性,質感)は,人間と機械(システム)との感性面でのインタフェースで影響を与える要素である.第1層の感性そのものを感じさせる要素(デザインイメージ,雰囲気)は,第2,3層から影響を受けて,イメージや雰囲気を醸成させる要素である.HDT では,この9項目は企画者(特にデザイナー)の新しいアイデア,感性を創出させる手がかりとして使うことを基本にしている.特に構造化コンセプトの最下層のキーワードに対し,この9項目を包含した70項目とその活用例を参照しつつ,可視化を行う.キーワードごとに可視化された案を統合すれば,最終デザイン案を絞り込むことができる.しかし,このような定性的な方法でなく,定量的に扱いたい場合は,先に述べた多変量解析やニューラルネットワークを使って,感性の9項目とオブジェクトの属性との関係を明確にすることもできる.このデータに基づきイメージを特定し,このイメージに基づいてやや荒っぽいがデザインを絞り込むことは可能である.
