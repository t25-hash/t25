# 3・10・3 システムの最適性と好適方式

> **章**: 第3章 設計のための個別方法論
> **節**: 3・148 下部のように,個別のサブシステムはより細かなサブシステ

システムを規定する上記の四つの条件のうち,それが固有の目的をもっていて外部からコントロールできることは,システムの設計がその目的を最適に実現することを目指していることも意味している.数理計画法に基盤をおく最適設計の考え方はこの方面における強力な方法論であるが,それ自体は設計対象についての何らかの明確で数値的に操作可能なモデルの存在とその数理的な適切性を前提としている.つまり,数理計画法の有用性は手段としてのものであってむしろ限定的であり,設計本来の立場に立てばシステムの最適性の意味を別途吟味しておくことが不可欠となる.上記のことは,いわゆる好適方式という考え方によって説明されている.図3・150は,それに関連して,システムの(広義の)最適性(optimality)には,実行可能システム(workablesystem,feasible system)・好適システム(suitable system)・最適システム(optimal system)の三つのレベルがあることを示している.実行可能システムとは設計対象の目的や機能を満たす方式を指し,一般にはある種の設計目的に対していくつかの方式が存在する.好適方式とはそれらの方式の中にあって,効率などの評価尺度を想定すれば,他を明らかに優越する特定のものを指す.どの方式が好適なものとなるかについては,設計対象の規模などが支配的となるとされている.最適システムとは,そのような好適方式のもとにあって,特定の要目やサイズ,寸法などをもった具体的なシス図3・149全体設計問題と部分設計問題との関係図3・148 システムの階層性テムを指す.なお,上記の数理計画法に基盤をおく(狭義の)最適設計は,上記の三つの中でも最後の最適システムを定めることを目指したものである.図3・151はシステムの最適性における上記の好適方式の意味を橋梁を事例として示したものであり,橋脚間のスパンに従って望ましい方式が変わってくることを示している.例えば,大きな海峡には,ガータ橋を構築することはできず,つり橋が必然的に用いられる.つまり,大きな橋梁の設計においては,構成部材の詳細な寸法などを決定する以前にこの種の方式の選定が不可欠であることになる.この事例はいささか極端ではあるが,同様の状況はさまざまなシステムの設計において何らかの形で現れる傾向が強い.
