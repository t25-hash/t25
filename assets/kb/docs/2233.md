# 3・10・4 システムにおける方式の転換

> **章**: 第3章 設計のための個別方法論
> **節**: 3・148 下部のように,個別のサブシステムはより細かなサブシステ

個別のシステムを超えて技術の発展を考えると,好適方式のもとにはある種の法則性が隠されていて,それが設計の中でも概念設計におけるある種の指導原理となる場合も多いとされている.前出の図3・151は規模に対応して好適方式が存在することを示したものであるが,そのような関係は技術というものが自然法則に従いつつも経済性などの社会法則によって評価されていることに起因している.石谷らは,ボイラの基本方式が規模(蒸発量)の増大に従って図3・152のように発展していく過程に関して,以下のような法則性を指摘している.すなわち,丸ボイラから水管ボイラへと転換する際には無循環水管ボイラという方式を経由しており,さらに水管ボイラから貫流ボイラへと転換する際には強制循環ボイラを経由していることに着目し,ある基本方式が次の基本方式に移る際には変換点で両方の性質を併せもつ中間形態(ハイブリッド)を経由することを示している.これは,規模が増大しつつあるときに何らかの新しい技術が導入される際には,システムの全体が置換わるのではなく,まずは部分的な置換えが行われるものの,増大が進んでいくと技術以外の要因によってそのような中間形態が破綻し,やがてはシステムの全体に新しい技術が波及することによるものである.このような方式転換の形態はあらゆる技術対象に見られるものである.規模による好適方式の転換をさらに詳細に見れば,規模に応じた機能の分化がそれを支配していることがわかる.図3・152のボイラの場合,丸ボイラは水の循環と蒸気の発生という二つの機能を図3・150 システムの最適性図3・152 ボイラの発展段階図3・151 橋梁における好適方式1―145もっているが,規模の増大に伴って水の循環は水管として独立し,蒸気の発生はドラムとして分化することによって水管ボイラへと移り,さらに,循環機能をポンプによって完全に独立させることによって貫流ボイラへと転換している.また,図3・151の橋梁の場合も,規模(スパン)が小さいガータ橋では,引張りと圧縮の両方を同じはり部材が負担しており機能は未分化であるが,スパンの長いトラス橋では各構造部材が引張りか圧縮のいずれかを分担していて,機能の分化が見られる.しかし,さらにスパンの長いつり橋では,ケーブルとつり索は引張りを,橋柱は圧縮をそれぞれに分担していて,完全な機能分化に至っている.以上のような機能分化も,さまざまな交通機関の高速化に伴う方式転換などを具体例として,あらゆる技術対象について見られるものであり,逆にとらえれば,有効な設計の指導原理となりうるものであるということになる.
