# 4・1・1 コストの構造(cost structure)と低減機会

> **章**: 第4章 設計の管理
> **節**: 4・1 コストの管理

製品価格に占める製造にかかるコスト,すなわち製品原価(product cost)は,製造業の平均で約50~60%である.図4・1に精密電機部品における実際のコストの構成比の例を示す.製品によってその占める割合は異なるが,大きくは材料費,加工費,管理費に区分される.そしてより詳細には図4・1右列に示すように,材料費は,素材費(鉄鋼,銅線など)と部品費(ねじ,半導体,ベアリングなど)に大別され,おもに外注先から調達される.加工費は,直接加工(組立)費と製造間接費に大別される.直接加工費は,作業者の労務費,設備・型の償却費,生産活動の消耗経費などである.製造間接費は,該当製品に関係する設計,生産技術,保全部門などの労務費,その経費である.管理費は,企業全般を管理するための費用で,営業,総務などの労務費や経費および長期・短期の在庫金利である.コスト低減機会という観点からは,直接加工費や製造間接費では,生産開始後の作業習熟など学習効果や改善によるコスト低減がねらえ,また材料費は外注先との間での買い方,連携プレーによるコスト低減を進めることができる.しかしながら,例えば直接加工費の大小は,作り方の工夫と同時に設計仕様に大きく依存する.したがって,仕様決定や設計開発段階でのコストの作り込み,すなわち原価企画が大変重要になる.製品の多品種化から,さらにマスカスタマイズが進行した現在では,設計や技術変更,トラブルシューティング,品種変更に伴う設備の段取替えや工具・金型の変更コスト,そして品種増に伴う管理コスト等の間接費や管理費の占める割合が増加している.一方,加工費の中の直接労務費の占める割合は製造業平均で製品価格の約10%未満であり,製品原価全体の15%未満に低下しつつある.このような状況においては,トータルの間接費や管理費をプールして,その額を生産高や売上高に応じて一定割合で製品別に配賦するこれまでの伝統的な標準原価計算のシステムは,もはや機能しなくなっている.例えばABC(activity based costing,活動基準原価)と呼ばれる方法を用いて,間接費についてもその原価を発生させる本当の要因を明らかにし,各製品の原価を正しく把握する必要がある.でなければ特に多品種化やマスカスタマイズの状況では,製品別の収益性や損益分岐点の判断,そしてコスト削減のターゲットを著しくゆがめることになりかねない.ABCでは,まず製品別に間接費についても資源を消費するアクティビティとその量を把握する.そしてそのアクティビティ(例えば段取り)に対してコストドライバと呼ばれるコストを発生させる要因(例えば手不足状況下であれば時間当たり労務費)が設定される.例えば,ある品種のための段取りというアクティビティの総人時間が100時間であれば,これに時間当たり労務費を乗じることによって段取りに関するコストが把握できる.エンジニアリングや管1―151理のためのコストについても,それを構成するアクティビティと適切なコストドライバが設定できれば製品別の正確なコストを把握することができる.例えば,ある中小企業において,標準原価計算で利益率が3.76~10.89%であった7製品についてABCを適用したところ,実際には7製品中3製品が赤字であった.なかでも10.89%という最高の利益率を生み出していたと考えられていた製品が,実際にはマイナス59.45%という大きな赤字に陥っていたという報告がある.コストの管理という立場からは,上述のようにまずコストを正確に把握することが必要になる.そしてグローバルな競争環境下においてコスト競争力を保つには,安い労働力を求めての海外進出という視点だけでなく,製品の企画構想,設計開発という源流段階でのDfM(design for manufacturing,3・3節参照)や製品VE(valueengineering,4・1・4項参照)といった手法を取込んだ原価企画がより重要となる.さらに個々の顧客によって異なる仕様設計を求められるマスカスタマイズの状況では,その中にVRP(varietyreduction program,4・1・5項参照)で示すような部品の共通化やモジュール化,そしてなるべく後工程で製品をカスタマイズするような製品設計・工程設計の視点も含める必要がある.これに加えて,製品開発そのものの効率化という観点からのコスト削減機会も重要である.
