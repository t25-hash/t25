# 2・4・1 ピン

> **章**: 第2章 締結要素
> **節**: 2・4 ピン，コッタ

ピン（pin）は，種々の部品の組付けや位置決めに用いられる機
械要素である．頻繁な着脱はピンの摩耗を引き起こすが，このよう
な場合はテーパの付いたピンが用いられる．ピンは，通常，せん断
応力を受ける．これに加え，ピンのはめあいが締まりばめのときに
は圧縮応力が，すきまばめのときには曲げ応力が同時に作用する．
JISに規定されているピンには，平行ピン（parallel pin；
B1354：1988)，テーパピン（taper pin；B1352：1988）がある
［図Ⅰ-2･18(ａ)(ｂ)］．これらの呼び径 は，ほとんど共通してお
り，0.6，0.8，1，1.2，1.5，(1.6)，2，2.5，3，4，5，6，8，10，
12，(13)，16，20，5，30，40，50［( )内は平行ピンのみ］が決
められている．
また，脱落防止効果を上げた先割りテーパピン（taper pin with
split ；B1353：1990)，スプリングピン（spring pin；B2808：
2005)，割りピン（split pin；B1351：1987）などがある［図Ⅰ-2･
18(ｃ)～(ｅ)］．先割りテーパピンおよび割りピンは，装着後先を
折り曲げて脱落を防止する．スプリングピンは，ピンのばね効果で
脱落を防止する．
これら以外に，平行ピンにねじ穴を付けためねじ付平行ピン
（parallel pin with internal thread；B1359：1990)，平行ピンに熱
処理を施したダウエルピン（Dowel pin；B1355：1990)，テーパ
ピンにねじ穴やねじを付けたねじ付テーパピン（taper pin with
thread；B1358：1990）などがある．
