# 2・6・1 リベットの種類

> **章**: 第2章 締結要素
> **節**: 2・6 リベット

リベットには，大きく分けて，一般構造用継手に用いられるソリ
ッドリベット（solid rivet )，軸端側に穴や割りが入った比較的軽
負荷で大量自動締結に用いられるチューブラリベット（tubular
rivet ）とスプリットリベット（split rivet )，意匠を考慮し出っ張
りをなくしたコンプレッションリベット（compression rivet )，そ
して一方からだけしかリベット打ち作業ができない場合に用いられ
るブラインドリベット（blind rivet ）の5種類がある（図Ⅰ-2･
24)．このうち，ソリッドリベットは，冷間成形用と熱間成形用に
ついて，JIS B1213：1995およびJIS B1214：1995で，頭部形
状で分類，規定されている（図Ⅰ-2･25)．また，少し古いが，軸端
の穴が浅いセミチューブラリベット（これに対して穴の深いものを
フルチューブラリベットと呼ぶ）の規格JIS B1215：1976も残っ
ている．
