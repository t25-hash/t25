# 2・6・3 リベット継手の設計

> **章**: 第2章 締結要素
> **節**: 2・6 リベット

リベット継手には種々の応力が作用する．
せん断応力である．
これ以外には，リベットに対しては，特に熱間作用すると考えられ，また残留応力も考えられる．
板に対してもリた場合には曲げ応力も作用すると考えられる［図Ⅰ-2･27(ａ)］．
し断応力［この応力による板の破壊状態は図Ⅰ-2･27(ｂ)参照］のみリベット継手の評価は，リベット継手の許容引張力に対する板の重ね継手で0.30～0.50，2列になると0.50～0.70程度の値となる．
(ｆ)ブラインドリベットで打たれる場合には穴とリベットにすきまができるため曲げ応力がベットとの接触部の圧縮応力や重ね継手や重ね板を片側にのみ用いかし，一般には，リベットのせん断応力および板の引張応力とせんを考慮して設計される場合が多い．
それに対する比で行い，リベット継手の効率と呼ぶ．
