# 4・2・5 クリティカルチェーン(critical chain)

> **章**: 第4章 設計の管理
> **節**: 4・6 の問題に対する山積みの様子は、図4・16の下に示すような山

生産スケジューリングや企業のゴールに直結した生産改善で知られるTOC(theory of constraints、制約理論)の分野でも、プロジェクト管理手法に対応したものとしてクリティカルチェーンがある。これは制約工程(ボトルネック工程)に着眼し、その前だけ在庫(バッファ)をもち、その工程の進捗具合と材料投入の同期化を図ることによってスループットを最大化し、かつ最小限に抑えるというDBR(drum buffer rope)の拡張概念である。
