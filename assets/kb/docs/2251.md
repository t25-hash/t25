# 2・8・6 スナップフィット形状設計の要領

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

スナップフィットには多種多様な形状のものがあるが，ここでは
最も一般的なカンチレバーフック（図Ⅰ-2･40）を設計する際の要
領について述べる．
a.基本形状
.基底部でのはりの厚さ 図Ⅰ-2･41(ａ)に示すようにベ
ースプレートに対して垂直にカンチレバーフックを設置する場合は
基底部の厚さ はプレートの厚さ の50～60 %とする．はりの
厚さ が50 %より小さすぎると，射出成形時にうまく充てんされ
ない場合がある．また，はりの厚さ が60 %より厚すぎると，冷
却時に残留応力や気泡などの問題を生じる場合がある．図Ⅰ-2･
41(ｂ)のようにベースプレート上や延長上にカンチレバーフックを
設置する場合は，基底部の厚さ をベースプレートの厚さ と等
しくする．カンチレバーフックのはりの厚さをプレートの厚さより

.挿入面の角度 挿入面の角度は組立力に影響を及ぼす．
角度を大きくするにつれて，組立時に大きな力を要する．理想的に
は組立時に要する力を小さくするために，できるだけ挿入面の角度
を小さくする．実際には図Ⅰ-2･42に示すαとしては25 ° ～30°が用
いられる．組立が困難になるため，挿入面の角度が45 °以上になる
ことは避ける．
.締結保持面の深さ 締結保持面の深さ は図Ⅰ-2･
43(ａ)に示すように組立/分解時に必要なたわみの大きさであり，
アンダカット（undercut ）とも呼ばれる．はりの長さ とはりの
厚さ の比が5：1のときは締結保持面の深さをはりの厚さ より
小さくする．はりの長さ とはりの厚さ の比が10：1に近い場
合は締結保持面の深さをはりの厚さ と同程度にする．
.締結保持面の角度 締結保持面の角度は，図Ⅰ-2･
43(ｂ)に示すように締結力や分解力に影響を及ぼす．締結保持面の
角度は外部からの分解力が結合部に作用しないような解除可能形ロ
ックの場合は35°程度，外部からある程度小さな分解力が作用する
場合は45°程度とし，また，外部から大きな分解力が作用する場
合，解除不可能形ロックとしては80° ～90 °とする．
