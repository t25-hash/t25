# 2・1・2 刑事責任に関する法

# 2・1・2 刑事責任に関する法

> **章**: 第2章 製品に係わる法制度
> **節**: 2・1 製品事故の防止を主眼とする法律

a．はじめに 機械製品を製造したり使用したりする際に，刑
事責任を負うことがあり得る．食品・医薬品の製造過程で有害物が
混入して中毒事故・医療事故を起こした場合や，自動車を運転して
いて交通事故を起こした場合などが典型的な例である．その際によ
く問われる罪名は，刑法第211条前段の業務上過失致死傷であるので，
以下においては，業務上過失致死傷について紹介する㈲．
b．業務上過失致死傷（刑法第211条前段） 刑法においては故
意による犯罪の処罰が原則であるのに対して，業務上過失致死傷に
関する第211条は，過失による犯罪を定めている．
具体的には，前段において「業務上必要な注意を怠り，よって人
を死傷させた者は，5年以下の懲役若しくは響銅又は50万円以下の
罰金に処する」と規定されている．すなわち，人の死傷を招くよう
な危険性のある行為をする者の過失により人を死傷させた場合に，
30万円以下の罰金又は科料に処せられる過失傷害（刑法第209条）や
50万円以下の罰金に処せられる過失致死（刑法第210条）に比べて重
く処罰するというものである．以下，この条文の意味について述べ
る．
i．業務 「業務」とは，人が社会生活上の地位に基づき反復継
続して行う行為をいう．一般的には，人の生命・身体に対して危険
を伴うものが業務とされることが多い．そのような行為が仕事上な
されたものでなく，娯楽目的でなされたような場合でも業務にあた
り得る．
具体的には，例えば，自動車の運転免許を有する者が運転中に事
故を起こした場合，その運転行為は業務である（6）．また，ウレタン
フォーム加工販売会社の工場部門の責任者として，易燃物であるウ
レタンフォームの取扱保管及び火災防止の職務に従事する者が，孫
請会社の従業員の行う工事に立ち会っていたが，火災が発生し7名
が死亡した事例において，人の生命・身体の危険を防止することを
義務内容とするものも業務にあたるとした裁判例がある（最高裁昭
和60年10月21日決定）．
ii．過失 業務上過失致死において「必要な注意を怠」つたと
は，
い．
製品を利用する立場や回収・廃棄する立場からみると，予期され
ていない方法や危険な方法で取り扱わないことが大切である．製品
の取り扱いを第三者（他社など）に依頼する場合には，その方法に
ついてよく協議し，安全なかたちで実行されるような配慮が必要で
ある．
なお，いずれの立場からしても，事故の予防のためには，過去の
事故例を調査することが非常に重要であることを意識して，具体的
な分析・検討を行うべきである．
ii．紛争への対応 製品の設計・製造や利用は究極的には人間
のコントロールのもとで行われるので，どれほど安全性に配慮し合
法的作業をこころがけたとしても，製品事故を完全になくすことは
できない．そこで，むしろ製品事故が起こったことを馨定して，も
（イ）結果を予測すること・結果を回避することが可能であり，
（ロ）結果を予測する義務・結果を回避する義務があったのに，
（ハ） これらの義務を果たさなかったこと
を指す．「過失があった」とも表現される．
例えば，自動車運転中に前方の道路の端で幼児が遊んでいるのを
認めたがそのまま徐行せずに直進したところ，幼児が飛び出してき
て事故を起こしたとする．その場合，前方の道路の端で幼児が遊ん
でいるのを認めた運転者としては，幼児の飛び出しにより事故にな
るかもしれないと予測することができたはずである．つまり結果を
予測することが可能であったといえる．さらに，運転者としては，
徐行したりクラクションを鳴らしたり，一時停止することで事故を
避けることができたはずである．つまり結果を回避することが可能
であったといえる．そして，運転者であれば，このように結果を予
測し，回避するよう行動するべきであったと考えられる．つまり，
結果を予測する義務・結果を回避する義務があったといえる．それ
にもかかわらず，これらの義務を果たさなかったのであるから，運
尋者には過失があったと評価することができる．
なお，過失を考える際に，危険な仕事に従事する者は，相手が幼
児であるなどの特殊な事情がない限り，他人が事故回避のため適切
な行動に出るであろうということを信頼して行動すればよく，他人
がこれと異なる行動をとることまで念頭におかなくてよい，と考え
られている．これを信頼の原則という．
製品の製造過程における過失が問われた事例として，乳児用調整
粉乳の製造工程において原料の牛乳に安定剤として添加された第二
りん酸ソーダの中に多量の亜ひ酸が含まれていたため，これを飲用
した乳幼児多数が死亡又は慢性ひ素中毒の症状を呈することとなっ
た，いわゆるドライミルク事件が挙げられる．この事件においては，
原材料等の購入等を含む製造業務に従事していた製造課長にっき，
業務上過失致死傷が成立するとして，禁銅3年の実刑判決が言い渡
された．すなわち，製造課長としては，発注した第二りん酸ソーダ
の中に第二りん酸ソーダでない物質が紛れ込んでいることがあり得
ることを予測することが可能であったし，添加する前に適切な化学
検査をするなどして確認することにより結果を回避すること，が可能
であった．そして，製造課長には，このように結果を予測し回避す
るという義務があったにもかかわらず，これを果たさなかった，と
いう意味で過失があった，と判断された（高松高裁昭和41年3月31
日判決，その上告審である最高裁昭和44年2月27日判決，差戻し後
の徳島地裁昭和48年11月28日判決）．
また，別の事例として；米ぬか油の製造工程においてPCBを主
成分とする熱媒体カネクロールが混入したため，これを経口摂取し
た消費者891人がPCBによる有機塩素中毒の症状を呈することとな
った，いわゆるライスオイル事件が挙げられる．この事件において
は，米ぬか油精製部門の責任者である工場長につき，業務上過失傷
害が成立するとして，禁鋼1年6月の実刑判決が言い渡された．す
なわち，工場長としては，カネクロールの過熱分解により生成する
塩酸によりステンレス鋼の配管であっても腐食するということや，
カネクロールが人体に経口摂取されると健康状態に不良な影響を及
ぼすということを予測することが可能であったし，カネクロールの
過熱をもたらすような装置の改造や変更を避け，運転前に装置を点
検してカネクロールが漏出しないか確認するとともに，運転中はカ
ネクロールの異常減量の有無を確かめて漏出の事実を発見すること
により結果を回避することが可能であった．そして，工場長には，
このように結果を予測し回避するという義務があったにもかかわら
ず，これを果たさなかった，という意味で過失があった，と判断さ
れた（福岡地裁小倉支部昭和53年3月24日判決，その控訴審である
福岡高裁昭和57年1月25日判決）．
このほか，ディスコの照明の電動昇降装置に使用されていたロー
ラチェーンが疲労破断して照明が落下し，死者3名・負傷者13名を
出した，いわゆるディスコ照明落下事件もある．この事件において
は，電動昇降装置の設計・製造・据付に従事した有限会社の代表取
締役につき，業務上過失致死傷が成立するとして，禁銅2年執行猶
予3年の判決が言い渡された．すなわち，その代表取締役としては，
ローラチェーンに作用する荷重がその疲労限度荷重を著しく超える
かもしれないことや，疲労破断に至るほど繰り返し荷重が加えられ
るかもしれないことを予測することが可能であったし，総重量を正
確に把握したうえで径比を乗じてローラチェーンに作用する荷重を
正確に算出し，起動・停止の際の衝撃も見込んで疲労破断を招かな
い十分な強度を有するローラチェーンを選定して昇降装置を設計・
設置することにより，結果を回避することが可能であった．そして，
その代表取締役には，このように結果を予測し回避するという義務
があったにもかかわらず，これを果たさなかった，という意味で過
失があった，と判断された（東京地裁平成4年2月26日判決）．
また，近時の事例として，いわゆる薬害エイズ裁判のうちの一つ
で，加熱濃縮血液凝固第IX因子製剤の販売を開始した後に，非加
熱製剤の販売中止・回収をしなかったため，病院において患者1名
に対して非加熱製剤が投与され，その患者がHIVに感染しエイズを
発症して死亡したことに関して，製薬会社の歴代社長3名が被告人
となった事件がある．この事件においては，歴代社長3名とも業務
上過失致死が成立するとして，それぞれ三三2年，禁銅1年6か月
及び四三1年4か月の実刑判決が言い渡された．すなわち，歴代社
長らとしては，非加熱製剤を投与された患者らがHIVに感染しエイ
ズを発症するかもしれないことを予測することが可能であったし，
非加熱製剤の販売中止・回収の措置を採ることによって結果を回避
することが可能であった．そして，歴代社長らは，このように結果
を予測し回避するという義務があったにもかかわらず，これを果た
さなかった，という意味で過失があった，と判断された（大阪地裁
平成12年2月24日判決）．
iii．因果関係 「よって人を死傷させた」という条文表現の中
の「よって」とは，その過失ある行為がなければその死傷はなかっ
た，という条件関係を意味すると考えられている．
例えば，自動車を運転中に道路上で歩行者と衝突して転倒させ，
ほどなく追行車や対向車が転倒中の歩行者を礫（ひ）いて死亡させた
場合などには，追行車や対向車の運転手の行為が存在するものの，
はじめに衝突した行為と死亡との間に因果関係を認めた事例が数多
く存在する．
iv．人 人とは生身の人間，すなわち自然人をいう．胎児がい
つ人になるかについてはいろいろな考え方があるが，裁判例におい
ては，胎児の一部が母体外に露出した時であるとされている．すな
わち，その時までは母体の一部を構成するものとして取り扱われる．
いわゆる熊本水俣病の裁判例においても，「胎児に病変を発生させる
ことは，人である母体の一部に対するものとして，人に病変を発生
させることにほかならない」とされた（最高裁昭和63年2月29日決
定）．そのうえで，同決定は，「胎児が出生し人となった後，右病変
に起因して死亡するに至った場合は，結局，人に病変を発生させて
人に死の結果をもたらしたことに帰するから，病変の発生時におい
て客体が人であることを要するとの立場を採ると否とにかかわらず，
同罪［業務上過失致死］が成立する」とした．
v．亮傷 死とは，いわゆる心臓死をさすと考えられてきた．
心臓死にもいろいろな考え方があるが，脈拍・自発呼吸の不可逆的
停止，瞳孔散大，対光反射喪失の三徴候をもって判断するものが有
力であるといわれてきた．ところが，平成9年10月16日に施行され
た「臓器の移植に関する法律」により，脳死が条文に規定された．
死における脳死の割合は高くはないものの，脳死に関する立法化と
も相まって，刑法の解釈の場面において，諸学説により，死とは何
かという議論が活発化している．
傷害の定義についても議論があるが，いわゆるケガのほか，中毒
症状による嘔吐のような病変も傷害にあたると考えられている．
c．機械製品にかかわる技術者等にとって注意すべき点 先に
「2・1・1 被害者救済のための法 d．機械製品にかかわる技術者
等にとって注意すべき点」において，民事責任の一種である不法行
為責任との関係で，機械製品にかかわる技術者等にとって注意すべ
き点について述べたが，技術者等としては，事故が発生した場合な
どにおいては刑事責任に問われることもあり得ることを念頭におい
て，日々の作業を行う必要がある．具体的には，民事の関係法令等
を調査・把握するのと並行して，刑法についても業務上過失致死傷
の規定を中心に概要を把握しておくことが望ましい．また，個別の
製品分野ごとの法律において，例えば電波法第100条に定める高周波
利用設備を無許可で運用した場合には同法第110条の罰則の対象と
なる，というようなことも視野に入れて，関係法令等の調査・把握
を行うことが必要である．
