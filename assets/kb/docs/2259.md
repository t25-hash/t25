# 4・4・2 製品系列の統合化についての概念的意味

> **章**: 第4章 設計の管理
> **節**: 4・4 製品系列の統合化と設計

a. 複雑さの拡大と時間の圧縮共通化・共有化・モジュール化などと称される製品系列の統合化は特異なことではなく,ある種の因果関係のもとで今日的な課題としておのずと表出してきたものである.統合化の本質的な意味を理解するためには,まず,その因果関係を把握しておくことが重要である.突き詰めればものづくりとは何かということになる.物すなわち道具の利用は人類を特徴付けているが,ものづくりとなれば,その生産が組織的な産業を通じて行われていることが本質である.産業における一つの根幹は分業によって労働力や設備,資金や知識などの資源を特定の対象に集中させて効率を向上させることにあり,そのような効率向上が新たな資源の集中を可能にする循環が再帰的に作用することによって,物は高度化し複雑な製品となっていることになる.その背後にあって,人間の欲求は生存欲求,安全欲求,社会的欲求,自我欲求,自己実現欲求へと段階的に発展していくとマズローが指摘しているように,物への要求は可塑的であって,充足とともに高度化し多様化していく傾向がある.つまり,産業としての可能性と市場における必要性が相互に作用しながら,ものづくりの対象である製品の複雑化や多様化,さらにはその価値の短寿命化は増進し続けているということになる.言い換えれば,製品の複雑さと対峙しつつ圧縮された時間の壁を乗り越えていくことは設計の管理における不可避の課題であるということになる.b. 視野の拡大による最適性の拡大a項の内容は,とにかく製品には膨大な内容が関与していることと,それらを短期間で処理することが必要であることを意味している.一方,設計には当然のことながら最適性(optimality)が求められる.この最適性の意味を大まかに概念的に考えてみれば,以下のようになる.(最適性)=(最適性指標)×(操作範囲)×(探索コスト)(4・1)つまり,優れた設計を達成するには適切な指標を用意して,より多くの可能性を調べ尽くすことが必要である.より適切な指標とは物事をより大局的にとらえることを指し,例えば,個別の製品の性能やコストを考えるのでなく,特定のプロジェクトによる利益や企業のある期における利益を考えることを指す.そのような指標が関与する範囲は当然のこととして拡大することから,対応して操作範囲は拡大しなければ,指標に対応した設計解を探し出すことはできない.さらに加えて,指標を置換えて範囲を拡大したことによって生じる膨大な代替案を調べ尽くすには相応のコストが必要となる.つまり,三つの要因がそろって初めて,優れた設計が達成できるということになる.裏を返せば,いずれかが不足すれば,得られる解は部分最適にすぎず,全体最適は達成されないということになる.しかしながら場合によっては,設計に許されるコストがそのレベルに追いつかないこともありうる.製品系列の統合化においては,むしろ,そういう場合のほうが多い.それを回避するための方策は設計問題を解く際の構造そのものを組換えるということにある.すなわち,事前に望ましい最適解を導き出す手順を想定することができれば,そのもとで構成される各部分設計問題のそれぞれは相対的には容易に解決可能であり,個別の設計問題から得られる結果を統合することによって製品全体についての全体最適に準じる設計を達成することが期待できるようになる.なお,実際の設計において,この種の問題解決構造を組換えるには,組織そのものの構造を変革するなり,適切な戦略のもとで製品開発を進めるなりの対応が求められるものと考えられる.
