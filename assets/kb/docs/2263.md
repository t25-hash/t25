# 4・4・6 終わりに

> **章**: 第4章 設計の管理
> **節**: 4・49 の製品表現における階層性を踏まえれば,以下のように定義

製品系列の統合化に関してはさまざまな視点が成立するが,それらのすべてに共通する点は,設計を考えていくうえでの視野を拡大することと,それによって増大する設計内容や拡大する評価基準に対していかに対処すべきかということが課題となっていることである.何らかの問題をより高所から解決しようとすれば,具体的な問題解決を行ううえでの管理の問題や全体問題を部分問題に分割するうえでの戦略に関わる問題など,個別製品についての設計を考える場合とは異なったさまざまな内容にも配慮する必要がある.このため,統合的な設計に向けての対応は容易には進展しにくい.とはいえ,グローバル化などに象徴される製造業の変ぼうや社会の成熟に起因するニーズの多様化や流動化は,いずれも統合化を要請するものであり,長期的かつ俯瞰的に設計を管理していく必要性は時代とともに増大していく性質のものである.文献(1) Anderson,D.M.,Agile Product Development for Mass Custom-ization,(1998),McGraw-Hill.(2) 圓川隆夫・安達俊行,製品開発論,(1997),15-21,日科技連出版社.(3) 藤本隆宏・クラーク,K.B.
