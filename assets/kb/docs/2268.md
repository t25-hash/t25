# 5・1・5 設計学の方向

> **章**: 第5章 設計者のために
> **節**: 5・1 設計学

以上に紹介した三つの設計論(設計方法論)をもとに、設計学の方向は下記のようにまとめられる。設計学においては、機能という概念を重要視している。これは、設計学が、人間の使う人工物を人間が作り出すという、当然ではあるが人間と人工物との関わり方を取扱う学問であり、機能が、まさに人間と人工物の間をとりもつ概念であるからである。すなわち、機能は人工物の物理的特性を抽象化したものであると同時に、人間によって意味が与えられる概念だからである。そして、一般設計学は、機能の構造と、(機能と属性との)写像との関係を議論したものであるといえる。すなわち、どのような構造のときにどのような写像がありうるのか、あるいは、ある写像を実現するためにはどのような機能の構造が必要なのかについての理論であるといってよいだろう。一方で、ポールとバイツの設計方法論およびスーの設計論は、機能の分解操作の方法に特徴があるといえる。双方とも、人工物が全体として発現すべく要求される機能を、いかにしていくつかの部分の機能に分解するかについて、具体的な方法を提案している。共通しているのは、全体の機能を互いに独立したいくつかの部分機能に分解するという考え方である。一般に、自然物にしても人工物にしても、部分の機能と全体の機能との間には、全体-部分の関係は存在しない。すなわち、部分の機能を合成しても全体の機能は一意には定まらないし、ある全体の機能を満足する部分の機能の構造も一意には定まらない。しかし、これらの設計方法論においては、互いに独立した部分の機能に分解することが提案されている。それは、分解することによって、設計や製造のプロセスが、より容易に管理できるからである。すなわち、互いに独立した部分機能に分解できれば、それぞれ独立にそれぞれの部分機能の設計や生産を行うことができる。また、それらを合成した際にも、要求機能以外の想定外の機能が発現する可能性は低くなるので、設計ミスの可能性も低くなる。言い換えると、機能を互いに独立した部分機能に分解することは、設計する側に有効な方法論であるといえよう。一方で、本来であれば多様であるはずの全体と部分の関係を限定してしまうことになるので、設計の対象とする人工物の範囲が狭められる危険性がある。機能とは、利用者と人工物の関係を結びつける概念でもある。機能は設計者が創造するものであり、利用者はそれを利用すればよいという考え方もあるが、利用者が人工物との関わりの中で発見する機能もあるはずである。物理的特性に基づくような機能だけでなく、価値観や意味付けに基づくような機能にも視野を広げると、利用者と人工物を結びつける機能のあり方は多様である。このような利用者の観点から機能をとらえてみると、部分の機能は互いに独立であるほうがよいという考え方には限界があるように思える。設計学において、今後とも機能が重要な位置付けにあると思われる。しかしながら、そのあり方や取扱い方には、まさに、新しい時代の新しい人工物を設計するための、さらなる展開が求められているといえよう。
