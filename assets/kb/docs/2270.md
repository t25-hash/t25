# 2・4・4 断面図（sectional view）の示し方

> **章**: 第2章 製図と図面
> **節**: 2・4 製図における図形の表し方

a.断面表示の原則 隠れた部分をわかりやすく示すために，
切断面（cutting plane）を用いて対象物を仮に切断し，切断面の
手前の部分を取除いて，断面図（sectional view）として図示する
ことができる．切断したために理解を妨げるもの，切断しても意味
がないものは，原則として長手方向に切断しない（図Ⅲ-2･21)．切
断面の切り口（section）を明らかに示す必要があるときには，ハ
ッチングを施してもよい．ハッチング（hatching）は，細い実線
で，主たる中心線に対して45度に施すのがよいが，理解を妨げる
場合には，他の角度を用いる．
切断面の位置は，切断線を用いて指示する．投影方向を示す必要
がある場合には，切断線の両端に投影方向を示す矢印を描く．ま
た，断面を識別する必要がある場合には，その矢印の端にローマ字
の大文字などの記号を上向きに記入し，相当する断面図の直下か直
上に，切断箇所に表示した文字を記入する（図Ⅲ-2･22)．
b.断面図の種類と表し方
.全断面図（full sectional view) 対象物を一平面の切断
面で完全に切断し，得られる断面を省略せずに描いた全断面図は，
対象物の基本的な形状を最もよく表すように切断面を決めて描き，
切断線は記入しない（図Ⅲ-2･23)．必要がある場合には，特定の形
状をよく表すように切断面を決めて描くのがよい．この場合は，切
断線によって切断の位置を示す（図Ⅲ-2･24)．
.片側断面図（half sectional view) 対称形の対象物は，
外形図の半分と全断面図の半分とを組合わせて表すことができる
（図Ⅲ-2･25)．
.部分断面図（local sectional view) 外形図において，
必要とする要所の一部だけを部分断面図として表すことができる．
この場合，破断線によって，その境界を示す（図Ⅲ-2･26)．

て描くか，あるいは図形内の切断箇所に重ねて細い実線を用いて描
くことができる（図Ⅲ-2･29)．
.組合せによる断面図 二つ以上の切断面による断面図を
組合わせて断面図示を行うことができる．なお，必要に応じて断面
を見る方向を示す矢印および文字記号を付ける．対称形またはこれ
に近い形の対象物の場合には，対称の中心線を境として，その片側
を投影面に平行に，ほかの側をある角度をもって切断し，後者の断
面図はその角度だけ投影面のほうに回転して図示する（図Ⅲ-2･
30)．平行な二つ以上の平面で切断した断面図の必要部分だけを合
成して示すことができる．この場合，二つの切断線を任意の位置で
つなぎ，組合せ断面であることを示す（図Ⅲ-2･31)．曲管などの断
面を表す場合には，その曲管の中心線に沿って切断し，そのまま投
影することができる．断面図は，必要に応じて上記の方法を組合わ
せて表してもよい．
