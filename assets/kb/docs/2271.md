# 5・2・3 設計者の継続教育

> **章**: 第5章 設計者のために
> **節**: 5・7 ).また,この時代には,企業,情報処理システムは集中型のツ

設計者の継続教育(continuing education)には,大きく分けて二つの課題がある.設計者がいかに創造性を保持するかと,新しい知識をいかに学習するかである.工学は新しい機能の創造であり,設計は最も重要な創造活動である.そこで設計者の継続教育においては,いかに創造性を保持するかに配慮する必要がある.創造性は,改良ではない.設計者は,創造性を保持し向上するために,広い視点をもつことが必要である.実際,製品実現においては,技術的のみならず,法律的,社会的,経済的な種々の制約をどのように解消できるか,広い見地から異なる視野で検討する必要がある.その意味でPBLは,こうした創造性を育成する一つの有力な教育方法であると期待される.現在のPBL教育は,継続教育を指向していないが,継続教育を指向したPBL教育開発の必要性が指摘されている.これまでは,企業が社内でPBLに相当する教育を,特に幹部候補生に対して実施してきた.しかし,社会の要求の多様化,さらに人材の流動性の増大とともに,このような設計者の継続教育としてのPBLを大学で実施してほしいとの要望が急速に増大している.最近話題となっているMoT(management of technology)も,広い意味での設計者の継続教育であるといえる.アメリカでは,1955年から1965年のアポロ計画を中心にして,さまざまな研究開発の課題をいかに予定に従って実現させるかの研究開発マネージメントが発達した.しかし,社会の多様化,複雑化が急激に進み,限られた人的,物的,経済的資源の制約の中で研究開発における生産性をいかに向上させ,その成果を製品化するかのマネージメントの重要性が強く認識されるようになってきた.従来は,個別の部門,個別のプロジェクト単位で生産性,品質などの向上を目指していたが,5・2・1項で述べたように,企業全体として戦略目標を決定し,一体化して展開することが重要となってきた.すなわち,これまでの個別の活動ではグローバル化,多様化に対応できなくなり,地域,産業を超えた連携と戦略的な展開が重要となってきた.これは,企業におけるデザインポリシーの問題である.この問題は,単に一企業内にとどまらず,国際競争力の向上の視点からは,産学官が一体となって取り組まなければならない課題である.現在,話題の知的財産権の問題や国際規格などの問題も,この問題と密接に関係している.欧米では,MIT(マサチューセッツ工科大学)のテクノロジー& ポリシープログラム,ハーバード大学(Harvard Univer-sity)のケネディスクールなど,技術系と,財務,マーケティングなどのビジネス系が協力して,技術と経営学修士(MBA)を結合した教育に相当するMoT 教育を実施している.特にアメリカのMoT 教育では,企業から派遣されて学習する例が大多数である.アメリカでは,製品開発における設計技術者の役割の重要性から,MoT 教育は製品開発技術者,設計技術者の生涯教育,継続教育として極めて重要な位置を占めるようになってきた.またアメリカでは,経営とは新しい価値の創出であるとの視点から,独創的な方法,システムを生み出す活動の重要性が強く認識され,1987年には連邦法100-107号により国家賞としてマルコムボルドリッジ賞(Malcolm Baldrige Award)が制定された.このように同賞は,アメリカの国際競争力の強化を目指して制定されたものであったが,その後,教育分野にも対象が拡大されている.すなわち,同賞は,国際競争力を向上させるための戦略目標を指向した活動を評価し,産学官で一体となって,製品開発,設計の革新に取り組むことを目指している.日本では,欧米ほど企業が資格を重要視していないために,これまではこのような戦略的な製品企画,設計教育は企業内で実施されてきた.しかし,系列の急速な崩壊,人事流動性の増大,また企業活動のグローバル化に伴うグローバルエンジニアとしての国際資格の必要性などから,日本でもMoT 教育を目指す大学が出始めている.次に,新しい知識の学習については,自分の分野の新しい知識と,自分がそれまでに知らなかった新しい知識の習得について考える必要がある.自分の分野の新しい知識について,継続的に学習しなければならないことは言うまでもない.そのための継続教育は現在も盛んに行われている.最近は遠隔教育が盛んとなり,従来よりも時間,場所によらず自分の分野の新しい知識の習得を図ることが容易となってきた.特にWBT(web based training),すなわちインタネットを用いた遠隔教育はインタラクティブ性があり,それぞれの個人の到達度に対応した学習ができ,個別化した学習プログラムが提供される利点があるため,急速に普及し始めている.例えば,アメリカの大学の中でも遠隔教育が盛んな大学の一つとして著名なスタンフォード大学では,SCPD(Stanford Center forProfessional Development)という組織が大学の中にあり,シリコンバレーの企業向けに遠隔教育を積極的に提供している.そこで提供されるプログラムの中には,設計関連のプログラムも多く含まれている.設計者にとって,自分の分野での新しい知識をつねに吸収し理解する努力が必要であることは言うまでもないが,顧客要求の多様化,社会のグローバル化により,設計者は自分の分野を超えて,より広い分野について知識を拡大する必要がある.問題は,分野によって基本的な考え方が異なることである.機械設計では,連続体の対象物をイメージしてそれを離散化するという,アナログからディジタルへと進む考え方が主流であり,その部品,製品も状況に応じて多様な機能を発揮するが,電気はもともと離散的な機能素子を組合わせてシステムの機能を実現するディジタル思考であり,システムが発揮する機能も明確に規定できる.このような分野による考え方の基本的相違を学ぶことが,例えばメカトロニクス製品などを設計していくうえで重要である.設計者の継続教育においては,このような分野による視点,考え方の相違に注目して,それを統括できるような広い視点をもてるように指導を行っていかなければならない.また,学習者も,そのような点に注意を払いながら継続学習を行うべきである.
