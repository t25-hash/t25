a.寸法線・寸法補助線 寸法線は指示する長さまたは角度を
測定する方向に平行に引き，線の両端には矢印，斜線または黒丸の
端末記号を付ける（図Ⅲ-2･45)．寸法は，普通は寸法補助線を用い
て寸法線を記入し，この上に寸法数値を表示する（図Ⅲ-2･46)．た
だし，寸法補助線を引出すと図が紛らわしくなるときは，これによ
らなくてもよい（図Ⅲ-2･47)．寸法補助線は寸法が記入される形体
に対し垂直に引き，寸法線をわずかに越えるまで延長する．必要な
場合には斜めに引き出すこともできるが，互いに平行にする．寸法
補助線と図形の間をわずかに離してもよい．
b.寸法数値 長さの寸法はmm単位で記入し，単位記号を付
けない．角度は一般に度で表し，必要がある場合には8 °0′52 ″の
ように記入できる．小数点は下の点とし，数字の間を適当にあけ
て，その中間に大きめに書く．また，3桁ごとにコンマで区切るこ
とはしない．
c .寸法記入法 寸法数値の記入位置および向きには，累進寸
法記入法の場合を除き，次のいすれかによる．一般には［方法1］
を用いる．これらの方法は，同一の図面内では混用してはならな
い．
［方法1］寸法数値は，図面の下辺および図面の右辺から読めるよ
うに，寸法線を中断しないで，これに沿って上側にわずかに離して
記入する［図Ⅲ-2･48(ａ)］．斜め方向の寸法線に対してもこれに準
じて記入する［図Ⅲ-2･48(ｂ)，Ⅲ-2･49］．
［方法2］寸法数値は，図面の下辺から読めるように書き，水平方
向以外の寸法線は，寸法数値を挟むために中断する（図Ⅲ-2･50，
Ⅲ-2･51)．
寸法線が短くて上記の位置に寸法数値を記入できないときは，寸
法線を延長してその上側（方法1)，またはその外側（方法2）に記
入してもよい（図Ⅲ-2･52)．寸法指示のために引出線を用いるとき
は，引出す側の端には何も付けない（図Ⅲ-2･53)．
d.寸法の配置をもとにした寸法記入法
.直列寸法記入法（chain dimensioning) 直列に連なる
個々の寸法の公差が累積してもよいときにだけ用いられる（図Ⅲ-
2･54)．
.並列寸法記入法（parallel dimensioning) 並列に記入
された各寸法の公差は，ほかの寸法の公差に影響を与えない（図
Ⅲ-2･55)．
.累進寸法記入法（superimposed running dimensioning)
上記ⅱ項と同様の意味をもち，起点の位置は起点記号で示し，寸法

（図Ⅲ-2･56)，矢印の近くに寸法線の上側にこれに沿って書く．
