# 1・6・3 脚・パイロン構造

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・6 航空機

アルミニウム合金などの軽合金では,強度が不十分なため代替することができず超高強度合金を用いるほかはない構造として,脚構造およびエンジン懸下構造(pylon,パイロン)や各種主要継手構造が挙げられるが,これらの構造には高張力鋼が用いられる.ただし,高張力鋼は一般に不安定破壊を生じる際の限界欠陥寸法が小さ2―17く損傷許容設計にはなじまないため,安全寿命設計(safe lifedesign)構造として健全性を保証する必要がある.高張力鋼AISI 4340,4330M,300M マレージング鋼などが脚構造のシリンダ,ピストン,トリクリンク,パイロン他の材料として用いられている.その他,客室内装用材料としてはクラッシュ事故などによって生じがちな火災に対して搭乗者の脱出を助けるために不燃性・難燃性・自己消火性材料,また,燃焼に伴う有毒ガスを発生しにくい材料が必要であり,その特性に優れた材料としてポリウレタン樹脂系材料が用いられる.
