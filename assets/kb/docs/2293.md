# 2・1・3 二次元CADによる効率化

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・1 製図

かつて図面は，鉛筆やペンなどを用いて紙に手で書かれていた
が，現在ではCAD（computer aided design）ソフトウェアを用い
て，計算機画面上で描かれるようになっている．そこで処理される
のは基本的に二次元図形情報であるので，そのようなCADを一般
的に二次元CADと呼ぶ．手で書かれた図面とCADで作成された
図面を区別する場合は，慣例的に手書き図面とCAD図面という呼
び分けをする．
二次元CADでは，計算機の画面上で対話的に図面の作成を行
い，結果をデータとして保存したり，プリンタやプロッタなどの装
置によって，自動的に紙に印刷して出力するなどの操作を行う．二
次元CADにより作成される図面は，基本的にディジタルデータで
あるため，手書き図面に比較して，保存，複製，編集，変更，伝達
などの点で非常に優れており，作業を効率化することが可能であ
る．しかし，CAD図面に表現されている情報の内容自体は，手書
き図面と本質的に大きな差はないため，機械の立体形状・構造を計
算機上でモデル化して表現，操作する，いわゆる三次元CADと比
較すると，その効果は限定的である．
なお，JIS規格では，主として機械工業におけるCADに関して
用いるおもな用語および定義 ，CADによって行う製図 など
について規定されている．
