# 2・1・4 CAD時代における製図法の意義

> **章**: 第2章 設計情報の表現と伝達
> **節**: 2・1 製図

近年，設計現場において用いられる設計対象の表現方法は，時代
とともに次のように変化してきている．
(１)手書き製図
(２)(二次元）CAD製図
(３)(二次元）CAD製図（主），必要に応じて三次元CAD（従）
(４)三次元CAD（主），必要に応じてCAD図面を生成（従）
このような状況においても，製図法は依然として，次のような点
で意義を有していると考えられる．
図2･76セクタ歯車（記入例）
(１)三次元CADの操作に用いられる入力装置の中心であるマ
ウスは，本質的に二次元情報の入力インタフェースである．したが
って，三次元CADにおいて立体形状をモデリングする場合にも，
目的とする立体はどのような二次元図形の操作（掃引，回転，交
差，切断など）の組合せにより構成できるかが想起できなければ，
立体形状データの作成操作が行えない．このような投影図と立体形
状の双方向の変換能力を身につけるために，投影法，製図法は有効
である．
(２)三次元CADは，設計対象の形状や構造の構想がある程度
固まった段階で，情報を入力し表現するのには適しており，強力な
ツールである．しかし，前述のように現状の入力インタフェースの
機能的制約もあり，漠然としたアイデアの段階で形状，構造を試行
錯誤的に表現したり，それをグループで行うなどの場合には，依然
として紙の上に略図や概念図を描きながら行うほうが効果的，効率
的である場合が多い．その場合に，紙面上に立体の形状，構造を，
さまざまな技術情報を含めて表現できる製図法を理解し利用できる
ことが，大きな利点となる．
(３)製図法，製図規格は，長い歴史を経て洗練されてきたこと
もあり，立体形状の幾何学的な情報だけでなく，寸法指定，公差，
表面性状，加工法など，機械設計に必要なさまざまな技術情報を，
あいまいさなく簡潔に表現できる，非常に優れた表現体系となって
いる．三次元CADにおいても，設計中の対象物や設計結果はコン
ピュータディスプレイという二次元的な表現をとることが多く，そ
の場合のさまざまな技術情報の表示方法は，製図法における表記法
にならったものとなっている場合が多い．
