# 1・11・1 おもな構成

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・11 通信機器(telecommunication equipment):

携帯電話は,おもに筐体構造機構部品,電気機構部品,実装基板関連,電池から構成されており,筐体等の機構部品が質量で30%程度を占める.図1・58および1・59に携帯電話機の模式図を示す.筐体のデザイン上,ストレート型と折り畳み型とが代表的な携帯電話の筐体構造である.音声機では薄型軽量のストレート型に人気があったが,高機能機では大画面化に有利な折り畳み型が若者を中心に人気が高まっている.ストレート型では,本体はフロントケース側とリヤケース側の筐体から構成されている.一方,折り畳み型では,おもにキー部と表示部のそれぞれがフロントケース側とリヤケース側の筐体から構成されている.
