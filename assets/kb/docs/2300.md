# 1・11・2 筐体材料(body material)

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・11 通信機器(telecommunication equipment):

モバイル機器である携帯電話には小型軽量化が求められる.筐体の薄肉化,高剛性化,さらには意匠性,生産性,経済性を考慮し,プラスチックが多用されている.図1・58 ストレート型図1・59折り畳み型a. 筐体材料への要求特性軽量化に対し低比重材料の高剛性化が課題であるうえ,薄肉化に対しては流動性の向上とウェルド強度低下も考慮する必要がある.顔の高さ程度からの落下衝撃強度,またジーンズ等の尻ポケットに入れ座った状態での曲げ強度等も重要である.携帯電話は,屋内外と使用環境が多様であり,筐体への耐熱性,耐候性への要求も厳しい.b. 筐体に用いるプラスチック材料携帯電話機では,ストレート型および折り畳み型の大半に,ポリカーボネート(PC:polycarbonate),エービーエス(ABS : acrylonitrile butadienestyrene),ポリカーボネートとエービーエスのアロイ材(PC/ABSalloy)を使用している.c. その他の適用材料携帯電話機の折り畳み型と一部のストレート型には,筐体材料にマグネシウム合金(magnesium alloy)も適用されている.薄型化かつ大画面化する表示部フロントケース側の筐体の曲げ・ひねり強度の向上,キー部フロントケース側の筐体のヒンジ強度向上等を目的に用いられる場合が多い.同様の目的で,ガラス繊維で強化したポリカーボネートも採用されている.
