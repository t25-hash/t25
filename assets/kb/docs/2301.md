# 1・12・1 構造部材

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・12 カメラ

カメラの骨格をなす構造体は強度,精度を保ちながら軽量化する必要があり,含銅シルミン系アルミニウム合金ダイカストADC10,ADC12が広く用いられてきた.これは機械的性質,被削性に優れているからである.しかし,鋳造時の寸法変化が大きいため,精度を要する部分には切削加工が必要である.鋳造時の寸法変化が比較的少なく,薄肉化も必要な用途には亜鉛ダイカストZDC2も使われている.近年のさらなる軽量化,低コスト要求により,構造体の樹脂材料化が進み,エンジニアリングプラスチックの使用が一般化している.構造部材として最もよく使われているのはポリカーボネート(PC)である.強度,靱性,成形性のバランスに優れ,耐衝撃性は樹脂材料の中では最高である.カメラ本体,レンズ鏡筒など従来金属が使用されてきた部分が置換わっている.また,PCに強化材,充てん材としてガラス,炭素のビーズ,繊維などを混合し,さらに強度を高めることも行われている.炭素の混合は単に強度を高めるだけでなく,樹脂成形品に導電性を与え,静電気除去や電波ノイズの遮断にも利用されている.PCはこのように万能的に利用されているが,長期間の湿度環境中での加水分解,油に侵されやすい点などに注意が必要である.ほかにはアクリロニトリル/ブタジエン/スチレン(ABS),ポリフェニレンサルファイド(PPS),液晶ポリマ(LCP)なども使用されている.ディジタルカメラの構造体は金属板材のプレス,かしめによるシャシ構造が多い.冷間圧延鋼板SPCC,電気亜鉛めっき鋼板SECC,冷間圧延ステンレス鋼板SUS 304,SUS 430などが使われる.これらは強度だけでなく加工性の良さが材料選択のポイントとなっている.特にかしめを廃止し,絞り,鍛造加工で複雑な形状を成形するような設計時には材料の絞り特性が重要なポイントとなる.図1・60 ディジタル一眼レフカメラの構造部材図1・60はディジタル一眼レフカメラの構造を示すが,中央の本体部分はステンレス板のプレス材で構成され,レンズ,クイックリターンミラー(quick return mirror),撮像素子などが取付く前板部分はPCで構成されている.図1・61はコンパクトカメラのズーム鏡筒(zoom barrel)の一例である.鏡筒の内側にはヘリコイド,カムなどの溝が複雑に形成されており,外側にはギヤが切られている.PCの射出成形により複雑な部品を一体化し,小型化と低コスト化に大きく寄与している.カム,ヘリコイドなどは10 m単位の精度が必要であり,従来はアルミニウムパイプ,ダイカスト材を切削加工するしかなかった.
