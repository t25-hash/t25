# 1・12・3 外装部材

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・12 カメラ

a. 金属材料カメラ外装は以前は黄銅板C2680P,C2801Pの絞り加工で上カバー,下カバーを形成し,それ以外を構造部材を兼ねるアルミニウムダイカストADC10,ADC12にシボ皮と呼ばれる合成皮革を接着したものが多かった.その後,低コストと軽量化を図り外装の樹脂化が進み,金属外装は一部の高級カメラのみとなるかに見えた.しかし,デザイン上の嗜好や,外装を薄くできるため小型化に都合が良いことから再び金属が見直され,コンパクトカメラやコンパクトディジタルカメラにも金属外装の機種が増えている.これらは軽量化と加工性の良さからアルミニウムの絞り加工が使われている.特に純アルミニウム板A 1050は絞り特性,アルマイト着色の安定性などに優れているため,複雑な曲面構成,着色の自由度から多くの機種に使用されている.また,さらなる強度向上や小型化のための薄肉化を目的にマグネシウム系アルミニウム合金板A 5000系などが使われることもある.しかし,加工性,着色の安定性などはA 1050に比較して劣る.これらアルミニウム系の2―30材料学・工業材料外装材料は板厚0.6~0.8mmのものが使われることが多い.小型化のためにさらに薄い外装材料を求めた結果,ステンレス板の絞り加工を使う例もある.アルミニウムに比べると重くなるが板厚は0.4~0.5mmで済むこと,アルマイトなどの表面処理工程が省けるなどの特徴がある.一般的なオーステナイト系冷間圧延ステンレス鋼板SUS 304を使うこともあるが,汗や海水など実使用時の耐食性の向上を目的にSUS 316Lを使う例もある.耐食性,強度,硬度,軽量化のバランスのとれた絞り加工用板材としてチタンがある.しかし,加工性の悪さと材料費が高価なため一部の高級カメラに使われるにとどまっている.この用途には純チタンの圧延材が使われている.金属外装の軽量化を進めるとマグネシウムに行き着く.比重が約1.8であるため,樹脂と同等の軽量化が可能であるうえに強度,耐電波ノイズ性,リサイクル性に優れている.しかし,前記のような圧延材の絞り技術はまだ発展途上であり,ダイカスト加工が主体である.このため,表面の巣などの欠陥を埋めたうえで塗装をする必要があり,コストが高くなるのが欠点である.この用途ではマグネシウム合金ダイカスト1種D(MD 1D)が,高級カメラの外装に使われている.図1・63はディジタル一眼レフカメラのマグネシウム外装の一例である.カメラの外装すべてを薄肉のマグネシウム合金ダイカストで形成し,強度アップと軽量化に大きく寄与している.b. 樹脂材料外装用樹脂は強度,特に耐衝撃性が要求されるためPCの使用例が多い.PCは成形性も比較的良好なため塗装をせずにそのまま外装として使うことも多い.しかし,偏肉部があると艶むらが発生するため外観としての品位を損なうおそれがある.よって,急激な肉厚変化に注意し,強度の点からも肉厚1~1.5mm程度で使われることが多い.ABS も外装材料としてよく使われる.耐衝撃性ではPCに劣るが成形性で勝る.特に表面に金属めっきを行う場合には耐熱性に優れためっきグレードABS を使用する.めっきは単にデザイン上の理由だけではなく,カメラの電子回路が発する,または外部から受ける電波ノイズを遮断するという目的で処理されることもある.外装材としてPCとABS を混合したアロイ材が使われることもあり,両材料の中間の特徴を示す.ファインダ,各種センサ,液晶表示器などの窓部材は透明度と平面性が要求される.これらの窓部材はポリメタクリル酸メチル(PMMA)が用いられる.PMMA は無色透明であるが,赤外線オートフォーカス用に可視光カット特性をもった顔料を混ぜてフィルタとして使うこともある.カメラ保持時の手掛かり,滑り止めとしてグリップゴムが取付けてある製品もある.カメラは長期間使用する製品なので,ゴムには高い耐候性が要求される.この観点からシリコーンゴムが使われる.しかし,シリコーンは接着性が悪いため,ブチルゴムもよく使われている.表1・22はカメラでよく使われる外装材料の特性の代表値である.必要とされる強度,加工限界から肉厚を決め,デザイン自由度,コストなどを考慮して外装材料を決定する.
