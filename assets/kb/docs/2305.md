# 1・12・4 光学部材

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・12 カメラ

レンズ,プリズム,鏡,フィルタなどの光学部品は以前はすべてガラス製であった.光学性能は屈折率,分散,分光透過率の異なったガラスを組合わせて設計するため,主成分のケイ素に各種元素を混ぜて目的に合った光学ガラスを作る.レンズやプリズムは溶解した光学ガラスを型で適当な大きさ,形に成形し,概略の形を作る.その後,レンズ研磨機で精度が出るまで磨きをかけて作り上げている.一般的にガラスレンズは研磨加工が多いが,最終的な精度,形状まで型で成形してしまうガラスモールドレンズ(glass moldinglens)もある.収差補正を効率的に行い,レンズの小型化,少枚数化を行うための非球面レンズの量産はこのガラスモールドの技術に負うところが大きい.ガラスモールドは比較的低温で軟化し,型の転写性や型離れ性の良い特殊な材料が必要とされる.ディジタルカメラは撮像素子の直前にローパスフィルタ(low-pass filter)を配置し,偽色やモアレじまなどを軽減する場合がある.このローパスフィルタには水晶板が使われる.光学材料にも樹脂化の波は押し寄せている.レンズやプリズムを成形で安く作るために透明なPMMA やポリスチレン(PS)が使われる.しかし,樹脂材料は温度変化による膨張,収縮が比較的大きいため,あまり精度を要求される用途には向いていない.特にPMMA は吸湿性も大きく,使用には注意が必要である.また,樹脂材料の中では比較的屈折率が大きいため,透明なPCでレンズを作ることもある.しかし,PCは紫外線により黄色に色付く性質をもっていることを知って使用する必要がある.
