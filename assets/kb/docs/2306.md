# 1・12・5 電気回路材料

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・12 カメラ

カメラの電気回路基板は,大きくリジットプリント板とフレキシブルプリント板(flexible print circuit board)に分けられる.リジットプリント板は他の電子機器と同様に,ガラス繊維の布(織布または不織布)にエポキシ(EP)を含浸させたベース材に銅はくを張り付けたガラスエポキシ銅張積層板が多い.銅はくのパターン層は1~8層程度まで用途に応じて使い分けている.フレキシブルプリント板はカメラ特有の実装方法として広く用いられている.狭い空間にむりやり電気回路を実装するため,折曲げや曲面が必須となったためである.はんだ付けの耐熱性,屈曲性などからポリイミド(PI)をベース材として使ったポリイミド銅張板が使われている.銅はくパターンを保護するカバーフィルムもベースと同じPIを使い,厚み方向の対称性をもたせることで屈曲に対する特性を向上させることができる.フレキシブルプリント板の用途の中には電気部品を実装せず,単に配線,スイッチのためだけのものもある.こういった用途には耐熱性は必要ないため,PETをベース材としたものもある.導体も銅はくだけでなく,カーボンや銀をペースト状にしたものを印刷して形成する例もある.しかし,一般的にフレキシブルプリント板はリジットプリント板に比べて高密度な部品実装性に劣る.このため,部品実装部は硬くして実装性を上げ,それ以外はフレキシブルな両者の利点を兼ね備えたリジットフレックスプリント板も存在する.表1・22 各種外装材料の特性比較材質密度g/mm引張強さN/mm耐力N/mm純アルミニウム(A 1060-O)2.71アルミニウム合金(A 5052-O)2.68SUS 3167.98純チタン(JIS 1種)4.70アルミニウムダイカスト(ADC 12)2.70マグネシウムダイカスト1.81ポリカーボネート1.20―ポリカーボネートG 30%1.43―ABS1.05―図1・63 ディジタル一眼レフカメラのマグネシウム外装図1・64 コンパクトカメラのフレキシブルプリント板図1・64はコンパクトカメラで使われているフレキシブルプリント板の一例である.ポリイミド製のプリント板上に各種電気素子がはんだ付けされている.この写真は展開状態であるが,実機では組立時に複雑に折曲げられ,カメラ内に実装される.他の配線材料としては塩化ビニル被覆銅線も使われているが,外径が0.4~2mm程度の細いものである.
