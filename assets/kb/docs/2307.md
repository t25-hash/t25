# 1・12・6 カメラ用材料の今後

> **章**: 第1章 各種製品に使用されている材料
> **節**: 1・12 カメラ

カメラ用材料は金属から樹脂への大きな流れを経験してきた.これは携帯機器全般の小型軽量化の中で当然なことであった.新規な材料の採用が進んだが環境問題への対応などで,もう一度材料を見直す機運が高まっている.生物,人体にとって害を及ぼす危険のある有害物質を極力使用しない方向で材料の改良が進んでいる.鉛,カドミウム,水銀,六価クロムなどが廃絶対象物質であるが,これらは意外に身近で使われている.鉛ははんだ,光学ガラスに多量に含まれているが,性能を保ったまま鉛を他の物質に代えた,鉛フリーはんだ,鉛フリーガラス(non-lead glass)が実用化されている.また,金属の切削性を上げるために鋼,アルミニウム,黄銅に鉛を含ませた快削合金があるが,これも含有率を下げる努力が行われている.六価クロムは亜鉛めっきクロメート鋼板やダイカストの防錆処理に使われているが,六価クロムフリー化が進行している.樹脂材料は製品の安全性向上(発火事故対策)のために難燃性が求められるが,臭素系,塩素系の難燃材は削減対象物質であることが多い.これらもより安全な難燃材への切換えが進みつつある.また,部品のリサイクルを進めるために樹脂部品には材質表示を行うこと,異種材料を分解しやすく設計することなどの工夫も求められている.
