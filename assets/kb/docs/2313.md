# 2・1・4 生産システムの評価項目

> **章**: 第2章 設計・評価技術
> **節**: 2・1 生産システム設計・評価技術の体系

生産システムの評価は，各種の観点から行うことが必要になる．
その中で，代表的な評価項目を挙げると以下のようになる．
a.投資効果 生産システムの計画，設計，開発，運用にかか
る投資コストとそれから得られる利益との比較において評価するも
のである．投資効果を評価する手法としては多数の算定方法があ
る．例えば，最も単純な算定方法は，以下のようになる．
総投資利益額＝総投資収入金額－総投資額
総投資収入金額＝総キャッシュフロー＋残存価格
総投資額＝設備および流動資産への投資額 (2･1)
b.生産コスト 製品の生産に要するコストであり，式(2･1)
の設備および流動資産への投資額に含まれる．生産に関わるコスト
は多岐にわたっており，すべての項目を検討することは困難である
が，一例を挙げると以下のようになる．すなわち，調達する材料コ
スト，生産システム内外の物流コスト，生産システム内外での在庫
コスト，加工および組立などの生産プロセスの実行に関わるコス
ト，生産設備の保守，修復などに関わるコストなどである．
c.生産管理コスト 生産システムにおけるプロセスの管理を
行うために必要なコストであり，式(2･1)の設備および流動資産へ
の投資額に含まれる．具体的には，生産システムにおける生産計
画，資材調達計画，在庫管理計画などの策定に必要な設備費および
人件費である．
d.信頼性 生産システムを構成する生産設備の信頼性および
品質である．信頼性は，生産設備の保守および修復などのコストに
直接影響する．生産システムの設計段階において設備の故障や異常
などを事前に解析する手法としては，FMEA（failure mode and
eﬀect analysis )，FTA（fault tree analysis )，ETA（event
tree analysis ) などの手法が挙げられる．また，生産プロセスの
運用段階においては，設備の平均修理時間（MTTR:mean time
to repair ），平均故障時間間隔（MTBF:mean time between fail-
ure ）などを挙げることができる．
e .フレキシビリティ 生産システム全体および生産設備が，
生産対象製品の品種および生産量の変更に対してどの程度柔軟に対
処できるかを示す．すなわち，当初に設定した生産品種および生産
量の変動に対して，生産ラインおよび生産設備のセットアップを変
更するコストおよび時間で評価する．また，最近は，当初設定した
生産品種以外の製品の生産に適用する場合のフレキシビリティも重
ーションシステムが利用されている．
要視されている．すなわち，新たな生産品種の導入に対応して，生
産システムの全体構成および生産設備の生産機能および生産能力を
拡張あるいは再構築するためのコストおよび時間も重要視されるよ
うになってきている．このため，ハードウェアおよびソフトウェア
のモジュール性，相互接続インタフェースの標準化およびオープン
化が求められている．
