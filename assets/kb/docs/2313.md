# 1・15・3 金属:その安定した特性

> **章**: 第1章 各種製品に使用されている材料

アルミニウムバット、ゴルフクラブヘッド、スキーのエッジ、シューズのスパイク(鋲)、アイゼン等各種登山用具、テントの支柱、野球のキャッチャマスク、剣道の面金など数えればきりがないほど、多くのスポーツ用具にさまざまな金属が使用されている。鉄系金属はその歴史が長く、安定した特性(等方性、強度、弾性率、硬度など)で品質の信頼度が高い。また非鉄金属はタングステン、銅系金属の高比重、アルミニウム、チタン等の低比重など、それぞれの特徴を生かした使用がなされている。a. ゴルフクラブヘッド用チタン合金ウッドヘッドには、柿材(パーシモン)が強度、打球音、比重の点でバランスがとれており19世紀以降長く使われてきた。ところが1970年代後半から、それまでの比較的軟らかい糸巻き構造のボールに代わって硬いソリッドボールがより大きな飛距離を求めて誕生した。パーシモンヘッドで硬いソリッドボールを打球するとヘッドがへこんだり割れたりするため、より硬く強度があるステンレス鋼製メタルウッドが1980年代に登場した。当初のステンレス鋼製メタルヘッドは、ヘッドの体積(大きさ)が150~160cm程度とパーシモンヘッドの180~190cmに比べ小さかったが、ヘッド内部が中空で重量がヘッドの外郭にあるため、ヘッドの重心周りの慣性モーメントは中実のパーシモンヘッドよりも大きく、スイートスポットをはずして打球したときのヘッドのぶれ(回転)が小さくなり、打球が曲がりにくく方向性に優れる。そのメタルヘッドの体積をもっと大きくすることでヘッドの慣性モーメントが大きくなり、打球の方向性、飛距離はいちだんと向上する。ヘッド体積を大きくするためには、打球の衝撃に耐えられる強度を保ったうえでヘッドの肉厚を薄くするか素材の比重を小さくすると可能になる。そこで製造方法の改良とともに素材の研究が進められ、1991年にチタン合金のウッドヘッドが開発され、以降チタ表1・27 金属バット用アルミニウム合金バットには、木、金属(アルミニウム合金)、FRPの3種類の素材が使用されている。近年では、外観は同じに見えるバットでも、異素材を組合わせたものや、特殊な構造をしたものなども開発されている。金属製バットが初めて登場したのは1970年で、1974年には高校野球で使用されるようになり、現在では硬式用、軟式用、ソフトボール用いずれにおいても金属製バットが主流となっている。金属製バットの素材は、強度、比重(密度)、加工性などからアルミニウム合金が使われている。ゴルフクラブで人気のあるチタン合金は比重、強度の面で非常に優れた素材であるが、反発が良すぎるという理由でルール上バットへの使用が禁止されている。硬式用バットには、アルミニウム合金の中でも特に強度に優れた連結デバイス(φ3×2mm)探傷デバイス(5×5×1.5mm)永久磁石希土類磁石(SmCo)センサ筐体アルミニウムモータ軸ステンレス鋼端子ニッケルモータ電極板銅コアティースケイ素鋼基板アルミナセンサ底板ステンレス鋼コアリングケイ素鋼磁石ケースアルミニウムセンサ電極板アルミニウム雄型接合素子銅、ニッケル、希土類硫石(NaFeB)ジンバルばねステンレス鋼探傷センサ銅、二酸化シリコン減速・走行デバイス(片側5×1.5×5mm)雌型接合素子はんだ、ニッケル、希土類硫石(NaFeB)ボビンステンレス鋼駆動デバイス(φ1.6×2mm)内側側板ステンレス鋼コネクタコイル銅固定子支持部黄銅初段大歯車超硬合金固定子コイル銅、二酸化シリコン備考本資料は経済産業省産技プロジェクトの一環として、NEDOから委託を受けた(財)マイクロマシンセンターの再委託業務として、三菱電機(株)、松下電器産業(株)、住友電気工業(株)が実施した成果より転載したものである。図1・74マイクロマシン構成部品とその素材2―35超々ジュラルミン(7000系)が使用され、軟式用、ソフトボール用バットには硬式用より強度は劣るが加工性に優れたアルミニウム合金(6000系)が一般的に用いられている。
