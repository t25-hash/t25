# 2・1・2 鋼の熱処理

> **章**: 第2章 鉄鋼材料
> **節**: 2・1 鉄鋼材料の基礎

a. 熱処理の目的熱処理(heat treatment)は加熱と冷却の組合せによって材料に所要の性質を付与する操作である。熱処理によって変態、固溶、析出、再結晶、拡散などの現象が制御され、機械的性質、耐食性、被削性などが改善される。熱処理では加熱、冷却の方法や条件が最も重要であるが、さらに、雰囲気の管理、変形や残留応力、熱処理前後の洗浄処理、作業管理、安全対策なども重要である。熱処理炉は熱源については燃料(重油、灯油、LPG、都市ガスなど)と電気(抵抗加熱、誘導など)に大別され、ガス炉、重油炉、電気炉、誘導加熱炉などがある。雰囲気について見ると、雰囲気炉、真空炉、塩基炉などがある。加熱・冷却では加熱温度、冷却剤、冷却方法などが重要となる。これらの条件を適切に選定することによって、目的の性質が得られ、酸化、脱炭、割れ、ひずみなどの欠陥を避けることができる。加熱温度は材料の種類および熱処理の目的に応じて決める。JIS などに適切な温度範囲が示されている。加熱時間は焼入れなどのオーステナイト化加熱に際しての現場的な標準として、厚さ25mmにつき低合金鋼では30分、高合金鋼では1時間が目安とされている。品物の中心の温度が目的温度に達した後の保持時間は品物の大きさに関係なく一定でよい。構造用鋼の焼入れでは保持時間は零(目的温度に到達後ただちに冷却)としてよいし、工具鋼では炭化物の固溶の難易によって10~40分程度がよいとされている。冷却速度も熱処理の目的に応じて変化させる。例えば、完全焼なましでは粗大なフェライト/パーライト組織が得られるように小さくし、焼入れではマルテンサイト組織が得られるように大きくする。オーステナイト化温度から冷却したときに起こる相変態の状況を温度と時間の関係によって表した変態図は、冷却速度、冷却温度などの選定に有力な情報を提供す2―39る。変態図には一定温度に保持した場合の等温変態曲線(TTT図、TTT 曲線)と連続的に温度を低下させた場合の連続冷却変態曲線(CCT 図、CCT 曲線)とがあり、熱処理の方法に応じて使い分けられる。冷却速度は冷却剤の冷却能と材料の大きさによって変化する。冷却剤としては水、油、空気が一般的であり、この順に冷却能が小さくなる。これらの中間的なものとして、水溶性焼入れ液、溶融塩、水または油の噴霧、衝風などがある。冷却効果は材料の形状によっても異なり、球:丸棒:板=4:3:2と考えてよい。加熱あるいは冷却は一般には連続的に行われるが、急熱、急冷は材料に応力を発生させ、割れ、変形、残留応力の原因となる。そこで、加熱では中間温度での予熱、冷却では等温保持や段階的な冷却が用いられることがある。b. 焼なまし焼なまし(annealing)は鉄鋼材料を適当な温度に加熱し、その温度に保持した後に徐冷する操作であり、硬さの低下、残留応力の除去、金属組織の調整などによって、被削性、冷間加工性、耐食性、機械的性質などを改善することを目的とする。鉄鋼のおもな焼なましを示すと、完全焼なましは軟化による被削性などの改善、鋳造や鍛造後の組織の均質化、残留応力の除去などを目的とする。等温焼なましは完全焼なましと同一の目的をもち、パーライト変態が急速に進行する温度で等温変態させることによって処理時間の短縮を図る。球状化焼なましは完全焼なましでも十分に軟化しない鋼の軟化に適用される。さらに、合金工具鋼、軸受鋼などの過共析鋼の靱性向上のために焼入れの前処理としても用いられる。応力除去焼なましは冷間加工、機械加工、溶接、鋳造、鍛造などによって生じた残留応力の除去が目的であり、応力は加熱温度が高いほど短時間の保持で減少する。拡散焼なましは大型鋼塊や鋳鋼などの偏析の軽減が目的である。c. 焼ならし焼ならし(normalizing)は鉄鋼材料をAc(加熱時にフェライトがオーステナイトへの変態を完了する温度)またはAc(加熱時に過共析鋼中のセメンタイトが完全に溶解する温度)以上の温度に加熱した後に大気中で冷却する操作をいう。その目的は組織の均質化と微細化であり、鍛造、圧延、鋳造後に多く用いられる。焼なましに比べて、均質化の効果は同様であるが、微細化の効果は冷却速度が速いことによって大きく、機械的性質の改善効果が大きい。d. 焼入れ、溶体化処理鉄鋼材料の焼入れ(quenching)はオーステナイト化温度から急冷してマルテンサイト組織を得ることによって硬化する操作である。マルテンサイトの硬さは図2・6に示すように、炭素量の増加とともに増加し約0.6%で飽和する。工具などではマルテンサイトの高い硬さをそのまま利用するが、機械部品などでは十分な靱性を与えるために高温の焼戻しを行って硬さを低下させる。マルテンサイト組織にするには臨界冷却速度以上の速度で冷却することが必要である。焼入れ材の大きさが小さい場合には中心部までマルテンサイト組織になり硬化するが、大きさが大きくなるにつれてまず中心部が硬化しなくなり、次いで表面部も硬化しなくなる。この現象を質量効果(mass effect)という。一方、焼入れ材の大きさが同一の場合には、硬化する深さは鋼種によって異なる。一定冷却条件の焼入れにおいて硬化する深さを示す性質を図2・5 共析鋼のCCT 図図2・6 鋼の焼入れ硬さに及ぼす炭素量および焼入れ組織の影響2―40材料学・工業材料焼入れ性(hardenability)という。焼入れ性のよい(大きい)鋼は質量効果が小さく、大きい品物まで焼きが入ることになる。焼入れ性の測定方法としてJIS では一端焼入れ方法(ジョミニー試験)が規定されている。焼入れ性は炭素およびCr,Mo,Mnなどの合金元素によって増大する。機械構造用鋼のJIS に熱処理後の機械的性質が示されているが、これは直径25mmの標準供試材で得られる値である。焼入れ材の直径が大きくなると質量効果が現れてこの値が得られなくなる。この限度が有効直径として示されている。焼入れ組織が完全にマルテンサイトにならない場合には図同一の硬さにした場合に、降伏点、疲労強度、靱性が低いなどの問題が起こる。ステンレス鋼、高マンガン鋼などで、高温に加熱して合金元素を十分に固溶させた後に、急冷して析出を阻止し過飽和固溶体を作る処理を固溶化熱処理(solution treatment)あるいは溶体化処理という。固溶化熱処理を施すことによってオーステナイト系ステンレス鋼は本来の優れた耐食性を発揮し、高マンガン鋼では優れた強靱性が得られる。e. 焼戻し、時効処理焼戻し(tempering)は焼入れによって得られた組織を安定な組織に近づけ、所要の性質および状態を得るために、適当な温度に加熱、冷却する操作である。加熱温度はAc(加熱時にオーステナイトが生成し始める温度)以下である。焼入れされたままのマルテンサイトは不安定であり、残留応力とともに仕上加工や使用中における変形や割れの原因となる。370~470K(97~197°C)の焼戻しでこれらの危険がかなり除かれる。しかも硬さはほとんど変化せず、高炭素の鋼ではε炭化物の析出によってむしろ増加する。また、降伏点、引張強さも上昇する(図2・7)。靱性を重視する機械部品では820~920K の高温焼戻しが行われる。弾性限や疲労強度を重視するばねでは靱性をやや犠牲にして720~820K の焼戻し温度が用いられる。合金元素の添加はセメンタイトの凝集を遅らせ、焼戻しによる軟化抵抗を大きくする。さらに、Mo,V,W,Crのような炭化物形成元素は添加量が多くなると、770~870K の加熱で合金炭化物を生成し硬さが増加する。この現象は二次硬化(secondary hardening)と呼ばれ、高速度工具鋼などの焼戻しで利用されている。靱性は一般に焼戻し温度の上昇とともに向上するが、570K 付近および770K 付近の焼戻しでかえって低下することがある。この現象を焼戻し脆性(temper embrittle-ment)といい、それぞれ低温焼戻し脆性および高温焼戻し脆性と呼ぶ。靱性を重視する場合にはこの温度付近での焼戻しを避けるこa. 固溶強化鉄原子を主体とする母相に合金元素を固溶させ図2・7焼入れした炭素鋼(0.41%C,0.72%Mn,0.15%Si)の焼戻しによる機械的性質の変化とが望ましい。高温焼戻し脆性はCr,Mn,Niを含有する合金鋼で現れやすく、Moを含有すると軽減される。高温焼戻し脆性を避けるために、構造用合金鋼などの焼戻しでは約820K 以上に加熱した後、急冷することが必要とされている。しかし、脆化する冷却速度は一般にかなり小さいので、大型部品では強制冷却が必要となるが、小型部品では空冷で十分である。焼戻しによる性質、例えば硬さの変化は式(2・2)のような温度と時間の関数として表される。式(2・2)は焼戻しパラメータ(tempering parameter)と呼ばれ、の値が一定になるような温度と時間の組合せにおいて焼戻し硬さは一定の値になる。溶体化処理によって得られた過飽和固溶体を室温に保持するかあるいは適当な温度に加熱すると、金属間化合物などが析出し硬化する。この操作を時効処理(aging)または析出硬化熱処理(precipita-tion heat treatment)という。析出硬化系ステンレス鋼やアルミニウム合金などの硬化に適用される。f. 特殊熱処理鉄鋼では特殊な熱処理方法が用いられることがある。オーステンパ(austempering)はオーステナイト化温度からベイナイト変態域の温度の塩浴に急冷し、この温度で等温変態させてベイナイト組織にする処理であり、焼入れに伴う割れやひずみの危険が少ない。さらに焼戻しが不要であり、処理温度が適切であれば同一硬さに焼入れ、焼戻ししたものよりも靱性が高い。マルテンパ(martempering)はいったんMs付近の温度の塩浴あるいは油浴に焼入れし、品物の内外の温度がほぼ均一になったのちに取出して空冷する処理であり、マルクエンチ(marquenching)ともいう。焼割れや変形の危険を防止するのが目的である。時間焼入れ(time quenching)は水あるいは油中に焼入れし、短時間の保持後、引上げて空冷する処理であり、マルテンパと同様な効果がある。冷却液に保持する時間は品物の直径3mmについて水では1秒、油では3秒程度が適切である。サブゼロ処理(sub-zero treat-ing)は焼入れした鋼を273K(0°C)以下の温度に冷却する操作であり、深冷処理(deep freezing)ともいう。この処理によって残留オーステナイトがマルテンサイトに変態するので、使用中の経年変形が減少する。また、硬さも上昇する。ゲージや精密軸受などに利用される。加工熱処理(thermo-mechanical treatment)は鉄鋼材料の機械的性質を向上させるために塑性加工と熱処理を組合わせた操作である。塑性加工と相変態の順序や相変態の種類によっていろいろなものがある。鍛造焼入れ(direct quenching from forging temper-ature)は熱間加工後の余熱を利用してそのまま焼入れする処理である。オーステナイト結晶粒の粗大化によって焼入れ性が向上し、焼きが入りやすくなるので焼入れ性の低い鋼種への転換が可能となる。また、再加熱が不要であるので省エネルギー効果が大きい。自動車部品などへ広く利用されている。オースフォーム(ausform-ing)はオーステナイト化した鋼を等温変態図における変態の遅い比較的低温の温度範囲で加工したのちに焼入れしてマルテンサイト組織にする操作である。通常の焼入れに比べて靱性が低下することなしに強度が上昇することが大きな特徴であるが、加工が困難なことなどの難点がある。制御圧延(controlled rolling)は1170K 以下の温度まで圧延したのちに冷却してフェライト組織にする処理である。低温での強圧延によって微細なフェライト結晶粒が得られるので強度と靱性の両者が増加し、非調質高張力鋼の製造に広く用いられている。
