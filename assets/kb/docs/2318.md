# 2・1・3 強化機構

> **章**: 第2章 鉄鋼材料
> **節**: 2・6 に示したように焼入れ硬さが低下する.そして焼戻しによって

鉄鋼材料の強化の素機構には,固溶強化,加工硬化(転位強化),析出(分散)強化および粒界強化がある.各強化機構の詳細は文献(13)~(15)を参照されたい.以下ではそれらの概要を述べる.ることによる強化機構であり,そのおもな素機構には以下のようなものがある.. 弾性ひずみ母相の鉄原子と原子半径の異なる合金元素を固溶させることにより,母相の結晶格子には弾性ひずみが生じる.このような格子ひずみが存在する結晶格子面は,格子ひずみの存在しない格子面に比べて,転位の運動に対してより多くのエネルギーを必要とする.そのため,転位の運動に対する抵抗が増大することとなり,強度が向上する.この効果を固溶強化(solid solu-tion strengthening)という.α-Feの引張強さに及ぼす固溶元素の影響を図2・8に示す.母相の鉄原子と溶質原子の大きさの違いが大きく,固溶量が多いほど格子ひずみの大きさは増大するため,固溶強化量は増大する.. コットレル効果母相の鉄原子に比べて,原子半径が±15%以内の合金元素は結晶格子を構成する鉄原子の位置を置換して固溶する.一方,鉄原子に比べて十分に小さな炭素や窒素などの原子は,結晶格子を構成する鉄原子のすきまに固溶する.前者を置換型固溶元素と呼び,後者を侵入型固溶元素と呼ぶ.転位近傍の格子が拡張した場所には侵入型固溶原子が偏析しやすく,偏析により転位近傍の弾性ひずみエネルギーが軽減される.このように転位近傍に固溶原子が固着されることにより転位はエネルギー的に安定な状態になり,転位の運動に対する抵抗となるため強度は向上する.この機構をコットレル効果という.. I-S 効果350°C以上になると,鉄中における炭素や窒素原子の拡散速度が大きくなるため,転位がこれらの侵入型固溶原子を引きずるための抵抗は小さくなる.ここで,炭素や窒素原子との化学的結合力の大きなCrやMoなどの合金元素を複合添加すると,炭素や窒素の侵入型固溶原子とCrやMoなどの置換型固溶原子が化学的に引き合い,I-S 原子対(Iは侵入型固溶原子,S は置換型固溶原子)を形成する.そのため転位は炭素や窒素原子だけでなく,I-S 原子対を引きずることとなる.置換型固溶原子の拡散速度は侵入型固溶原子に比べて小さいため,I-S 原子対を引きずることは転位の運動に対する抵抗となり強度は向上する.この強化機構をI-S 効果という.固溶強化は高温強度に関しても重要な強化因子であり,その詳細は文献(17)(18)を参照されたい.合金元素の固溶量には限界(固溶限)があるため,固溶強化は後述する析出(分散)強化に比べると,本質的に大きな効果は期待できない.b. 加工硬化(転位強化)加工により塑性変形させた場合,塑性ひずみの増加に伴う転位密度の増加により,塑性変形を起こすのに必要な応力が増大し,強度が増加する.この現象を加工硬化図2・8フェライトの引張強さに及ぼす固溶元素の影響.( )内の数字はFeとの原子半径差(Å)図2・9オロワンのバイパス機構2―41(work hardening)という.構造用圧延鋼材のように,冷間加工の状態で使用される鉄鋼材料では,塑性変形による加工硬化を利用して強度の増加が図られている.塑性変形により転位密度が増加すると,転位間の相互作用が働くことによって,転位が互いに動きにくくなるため,強度が増加する.塑性変形を起こすのに必要な応力σは,転位密度ρが大きいほど大きく,式(2・3)で表されることが実験的に明らかにされている.σ=αρ(2・3)ここで,αは結晶構造に依存する材料固有の定数,は剛性率,はバーガースベクトル(Burgers vector:滑り面上の転位の単位滑り距離)であり,強度は転位密度の1/2乗に比例する.式(2・3)から,理論上は転位密度の増加に伴い強度は著しく増加するはずである.しかし,実際に得られる最高転位密度は1×10/cm程度であり,それが加工硬化による強化の限界である.c. 析出(分散)強化(precipitation strengthening)母相中に炭化物や金属間化合物などの析出相や酸化物粒子などの分散相が存在することにより,それらが転位の滑り運動に対する障害となり,強度が増加する現象を析出(分散)強化という.四つの強化機構の中で,強化能力が最も大きいのは析出(分散)強化である.その素機構および転位と粒子との相互作用には以下のようなものがある.. オロワン機構析出物や分散粒子などの第二相粒子は転位の滑り運動の障害となる.応力が十分に高い場合には転位が第二相粒子を通過することも可能であるが,転位が第二相粒子を通過する代わりに,粒子と粒子の間に張り出し,粒子の周りに転位ループを残して通り抜ける.これをオロワン(Orowan)のバイパス機構と呼び,粒子の周りに残された転位ループをオロワンループと呼ぶ.オロワンのバイパス機構により転位が粒子間を通り抜ける様子を図2・9に示す.オロワンのバイパス機構により転位が運動するときに必要とする応力σ(オロワン応力)と粒子間距離λとの関係は式(2・4)で表される.σ=αλ(2・4)ここで,αは材料固有の定数,は剛性率,はバーガースベクトルであり,強度は粒子間距離に反比例する.したがって,第二相粒子を微細かつ多量に分散させ,粒子間距離を短くすることにより,強度は増加する.. 粒子の整合ひずみ場Al合金におけるG-Pゾーンやスピノーダル分解によって粒子の周りには整合ひずみ(弾性ひずみ)が形成される.このひずみ場は,固溶強化における弾性ひずみと同様に転位の運動を妨げ,変形応力を増加させる.. 剛性率効果転位の弾性エネルギーは剛性率に比例するため,第二相粒子が母相よりも大きな剛性率を有する場合には,転位が第二相粒子に近づくと弾性エネルギーが増加して,反発力を受ける.逆に,第二相粒子の剛性率が母相よりも小さい場合には,転位と粒子の間には引力が働く.. 新界面の形成母相と整合な粒子を転位が通過すると,滑り面に沿って粒子の前後に新しい母相-粒子界面が形成される.このため,整合粒子を通過する転位は,新しい界面を形成するために必要なエネルギーを余計に必要とするので,抵抗力を受けることになる.これは,Al-CuやAl-Ag 合金のG-Pゾーンによる硬化に対して寄与していると考えられている.. 不規則境界の形成析出粒子が規則構造を有する場合,転位の通過により,滑り面に沿って不規則境界が形成されることになる.そのため,転位の滑り運動に対して大きな抵抗となる.これは,γ′相(NiAl)を含むNi基超合金において重要な素機構である.2―42材料学・工業材料d. 粒界強化鉄鋼材料は通常多結晶体であり,転位の滑り面は結晶粒界で不連続となる.また,常温においては結晶粒界は転位の滑り運動に対する障害となる.そのため,結晶粒を微細化し,結晶粒界面積を増加させることにより強度は増加する.降伏強さσと結晶粒径との関係はホール・ペッチ(Hall-Petch)の関係としてよく知られる式(2・5)で表される.σ=σ+(2・5)ここで,σは転位が結晶粒内を動くのに要する応力であり,温度,ひずみ速度および金属組織学的因子に依存し,は結晶粒界の諸条件によって決定する定数である.したがって,強度は結晶粒径の-1/2乗に比例し,結晶粒径を微細化することにより強度は著しく増加する.また,結晶粒の微細化は強度だけでなく,靱性の向上にも有効である.実用的な鉄鋼材料で実際に得られている最も微細な結晶粒径は数mであるが,実験室的には1 m以下の超微細粒組織が得られており,相変態・再結晶を利用した加工熱処理により超微細粒を得るための研究が精力的に行われている.一方,拡散が活発に生じる約1/2윯以上の高温においては,変形に対して拡散が重要な役割を果たすようになるとともに,結晶粒界は転位の生成・消滅場所となる.そのため,高温強度,特にひずみ速度の小さなクリープ強度に対しては,粒界の存在は強度を低下させ,結晶粒の大きいほうが有利である.
