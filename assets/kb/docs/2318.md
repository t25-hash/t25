える．
c .ライフサイクルコスト（LCC:life cycle cost ) 製品，生
産ライン，設備の開発・設計・導入・操業・保全・支援・廃却の一
生涯の中で発生する総コストをライフサイクルコストという．
LCCの内容は，設備（システム）や資材の設計，製作，導入を
要するコスト，すなわちイニシャルコスト（AC，取得コスト）と，
運転，保全，廃棄に要するコスト，すなわちランニングコスト
（OC，維持コスト），および解体，運搬，保管等の廃棄に関するコ
スト，すなわちディスポーザルコスト（DC，廃棄コスト）の総和
で把握される ．
LCC＝AC＋OC＋DC
設備（システム）や資材の経済性を高めるためにLCCを最小化
することが行われる．そのためには，設備（システム）の開発や取
得の段階でLCCを評価し，ACとOCおよびDCのトレードオフ
の中で最適な設備（システム）を追求することが必要になる．
LCCを最小にする設備の設計案を立案していくうえで鍵を握る
のは，設備（システム）のユーザ（製造部門）の現場における経験
を設備の開発・設計にいかに生かすことができるか，という点であ
る．設備の信頼性，安全性，保全性，良品製造条件などに関する設
備設計に織り込むべき内容の情報（MP情報）を，設備（システ
ム）のユーザ（製造部門）から組織の壁を乗り越えて，メーカある
いは設備の開発・設計部門へ上手にフィードバックすることが要に
なる．
設備（システム）が一生涯の中で生み出す利益をライフサイクル
プロフィット（LCP:life cycle profit ）と呼ぶ．LCPを最大にす
る設備（システム）を目指す考え方もこれからは必要になってく
る．
LCPを追求するということは，設備が作り出す製品需要の量変
動を考慮して，売り逃がしや過剰能力を避けることができる，量変
動適応型の設備・システムを追求することである．量変動適応型の
設備・システムの基本的な考え方は，固定費型と変動費型の手段を
ミックスして対応することに求められる．
