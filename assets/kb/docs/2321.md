# 2・2・2 炭素鋼の成分

> **章**: 第2章 鉄鋼材料
> **節**: 2・2 炭素鋼

炭素鋼はマンガン,ケイ素,リン,硫黄などを不純物として含有し,製鋼に使用する原料によっては,銅,ニッケル,クロム,アルミニウム,ヒ素などを含む.また,酸素,水素,窒素などのガスも含まれている.マンガンは普通1.6%以下であるが,MnS として存在する場合は,硫黄の赤熱脆性を防止するのに効果がある.しかしフェライトに固溶して冷間加工性を害するから,深絞りに使用される低炭素鋼には0.5%以下であることが望ましい.また低炭素鋼として電磁気材料に使用される場合もMn%の低いほうがよい.ケイ素はキルド鋼(killed steel)には0.2~0.4%程度含まれ,リムド鋼(rimmed steel)では0.1%以下である.この範囲のケイ素は一部ケイ酸として介在し,残りはフェライトに固溶しているが,機械的性質に影響を及ぼすことは少ない.しかし低炭素鋼では透磁率を低下し,鍛接性および冷間加工性を害するので,0.2%以下であることが望ましい.銅は普通0.3%以下で,おもにフェライトに固溶している.0.4%以上になると,赤熱脆性および冷間脆性を起こし,鍛接性を害することがある.したがって強い加工を施す管,板材などでは,0.18%以下であることが必要であるが,機械的性質を改善し,耐食性を向上する効果もある.図2・11に銅含有量と耐食性の関係を示した.ニッケル,クロムがまれに0.1%程度含まれることがある.ニッケルは性質を害することが少なく,かえって粘り強さを改善して好ましいが,クロムはその量が0.5%付近になれば高温加工後に硬化して切削加工を困難とし,また熱処理において焼ならし後の硬さを増し,焼入れ性を改善する.しかしその含有量が普通一定しないため,取扱いが複雑になるから,これらの不純物は少ないほうが望ましい.リンは純酸素転炉または電気炉で溶製されたものは,0.05%以下である.フェライトに固溶して存在するが,偏析して冷間脆性を起こし,衝撃値を低下する.しかし切削性を改善する効果もある.硫2―44材料学・工業材料黄は0.05%以下であることを普通とするがFeS として介在することがあり,マンガンが十分共存すればMnS として存在する.FeSはFeOを伴って赤熱脆性の原因となり,MnS として存在すれば高温でもろさを示すことがない.MnS は切削性を改善する.アルミニウムはAlOおよびAlN として介在し,結晶粒の微細化に有効である.窒素はフェライトに固溶して存在し,析出硬化を起こし,この現象は低炭素鋼ほど顕著で時効硬化の原因となる.酸素はフェライトに固溶し,また酸化物として介在する.低炭素鋼では析出硬化を呈し,あるいは赤熱脆性の原因となる.普通0.005%程度含まれる.水素もフェライトに溶解するが,析出して白点の原因になることがある.
