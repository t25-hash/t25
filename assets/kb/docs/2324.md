# 2・2・5 機械構造用炭素鋼鋼材

> **章**: 第2章 鉄鋼材料
> **節**: 2・2 炭素鋼

a. 種類と用途機械構造用炭素鋼鋼材(S-C材)は特殊鋼の中の合金鋼に分類されているが、炭素、ケイ素、マンガン、リン、硫黄しか含まないため、2・2・5項で解説する。JIS に規定されている機械構造用炭素鋼(carbon steel for machine structure use)章鉄鋼材料2―49第2は、S 09CK からS 58Cまでの23種ある。炭素量の範囲は0.07~0.61wt.%まである。表2・18に機械構造用炭素鋼の各鋼種ごとの化学成分を示す。ただし、S 09CK,S 15CK,S 20CK はリン、硫黄および銅、クロム、ニッケルを他の鋼種よりも低く抑え、特に肌焼鋼として規定されている。これらの鋼種は熱間圧延、熱間鍛造などの熱間加工によって、棒鋼、形鋼、鋼板などにされたのち、さらに、鍛造、切削、熱処理などが加えられる。b. 炭素鋼の熱処理機械構造用炭素鋼鋼材(S-C材)に適用される熱処理は、焼ならし、焼なまし、焼入れ焼戻し、高周波焼入れ、浸炭などがある。. 焼ならし(normalizing,JIS 記号:HNR)熱間圧延材の組織の均一化、結晶粒の微細化による機械的性質の改善を目的とした熱処理である。鋼材をAc点以上約50K に加熱し、均一なオーステナイトにしたのち、大気中に放冷するものである。焼なましとの違いは冷却速度で、大気中に放冷することがポイントである。. 焼なまし(annealing,JIS 記号:HA)焼なましには目的に応じて、完全焼なまし(HAF)、球状化焼なまし(HAS)、応力除去焼なまし(HAR)などがある。完全焼なましは鋼材をAc点以上に加熱したのち、炉冷または徐冷し、塑性加工を容易にするものである。球状化焼なましは、鋼中のセメンタイト(FeC)を1 m程度の均一な球状にすることによって、冷間での塑性加工性、切削性を向上させるために行われる。A点よりもわずかに高い温度の1023~1033K に長時間加熱したのち、徐冷する。応力除去焼なましは、冷間加工や溶接によって生じた残留応力の除去を目的として行う。加熱温度は773~793K で、完全焼なましより低温で行う。図2・15に焼ならし、または焼なまし後の機械的性質を示す。. 焼入れ焼戻し(quench temper,JIS 記号:HQ,HT)⬅字間ツ焼入れ焼戻しは、適切な強度と靱性を付与する熱処理方法で、Ac点以上に加熱し、完全にオーステナイト化したのち、急速冷却するるためメ改行を入のが焼入れ(HQ)で、そのあとA点以下の適当な温度に加熱しれてますて徐冷するのが焼戻し(HT)である。焼入れ温度は表2・19に示す範囲で定める。炭素鋼は質量効果が大きいため、有効熱処理径は約20mm以下で、これ以上の直径または対辺距離をもつ鋼材では完全な焼入れ組織を得ることは不可能で、オーステナイト結晶粒の境界、または粒内へき開面に沿ってフェライトが析出する。このフェライトの析出は粘り強さを低下させるが、合金鋼の場合に比べてその程度は軽微である。焼割れを防止するには、マルテンパ、またはオーステンパを行うか、あるいは階段焼入れを行う。マルテンパでは473~573K まで急冷して、鋼材が一様にこの温度になるまで保持したのち徐々に、しかも全断面同時にマルテンサイト変態を完全に終了せしめる。オーステンパではS 曲線で、フェライトおよびパーライトへの変態とマルテンサイト変態との中間の約623K に保持した熱浴あるいは恒温加熱炉中に急冷、その温度に十分保持してベイナイト変態を完了せしめたのち徐冷する。階段焼入れは焼入れ液中に焼入れしたのち約573~673K に冷却したとき焼入れ液より引上げ、別に用意した約473K の焼戻し炉中に装入するか、そのまま空気中に放置して徐冷する方法である。しかしこれらの方法は直径または対辺距離約10mm以下の小物においてのみ有効な焼入れが可能で、それより大型の場合は質量効果のため十分な焼入れ効果は望めない。焼戻しには、453~473K で行う低温焼戻し、673~923K で行う高温焼戻しがある。刃物等の硬度の高いものや耐摩耗性を要求されるものに対しては、低温焼戻しを行う。高周波焼入れ品、浸炭焼入れ品に対しても低温焼戻しを行う。高温焼戻しは、ある程度の強度と靱性を求められる場合に行う。なお、573K 付近での焼戻しは、脆性が現れるため避けなければならない。図2・16にS 30C鋼の焼戻し温度による機械的性質の変化を示す。一般に、823~923K
