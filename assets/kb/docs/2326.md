# 2・3・2 構造用合金鋼の規格

> **章**: 第2章 鉄鋼材料
> **節**: 2・3 構造用合金鋼

a.JIS 表2･29～2･31に，構造用合金鋼のJISを，素材と素
形材に大別して，主要合金成分，機械的性質，特記すべき用途によ
って示す．
b.規格と仕様 規格における規定値は，本来最小必要値を提
示するものであるが，これに加えて，いわゆる機械的諸特性値は，
直径25～50mm相当の製品サイズに対応するもので，より大型の
多様な製品を完全には規定できないのが通念とされている．したが
って設計と材料購入に当たっての個別的仕様は，原規格以上のもの
でなければならないと考えるべきである．このため構造用合金鋼の
選択に当たっては，既存のデータに基づく類似の部材からの内挿あ
るいは外挿，実物と同等の部材による実証試験などによって，原規
格に対応しつつ個々の材料仕様を決める．
