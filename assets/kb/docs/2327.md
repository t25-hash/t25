# 2・3・3 構造用合金鋼選択の指針

> **章**: 第2章 鉄鋼材料
> **節**: 2・3 構造用合金鋼

炭素鋼の発展型材料としての合金鋼を,設計あるいは使用の目的に適合させるための選択は,次のような考慮に基づくべきである.a. 構造用鋼の実用的強度限界構造用鋼は,機械装置の高性能化,大型化,軽量化などによって,その強度はつねに増加を求め2―53C%Si%Mn%P%S%Ni%Cr%Cu%0.550.230.760.0180.0120.010.090.01焼戻し温度K823(550°C)873(600°C)923(650°C)られる.鋼の強度はその延性と相反する関係にあるから,実用上限界がある.図2・19に示すように,軟鋼の引張強さ400MPa,伸び30%を出発して,合金元素の添加と熱処理によって引張強さは増すが,設計,運転などの観点から伸びの要求値を10%以上と仮定すれば,約1200MPa が実用上引張強さの上限(applicablestrength limit)となる.これ以上の強度と伸びの組合せが必要とされるときは,特殊用途鋼を選択しなければならない.b. 熱処理と質量効果鋼材の焼入れにおいては,鋼の変態特性と冷却速度の組合せによって,大型鋼材の焼入れ効果が減少し,鋼材内部の機械的性質は表面付近のそれよりも低減する.この質量効果(mass effect)は,構造用合金鋼の選択において,強度と靱性,溶接性,コストなどとの兼合いで重視される(2・1節参照).c. 機械的性質の経時的変化常温での機械的性質は,鋼材が機械装置を構成して稼働の経過とともに,一般には劣化する.劣化の現象としては,疲労,クリープ,腐食(応力腐食割れを含む),中性子照射劣化,内部欠陥の拡大などである.これらの大部分は常温の機械的性質に正の相関をもつが,例外としては,応力腐食割れのように,引張強さと逆の相関を示すものもあるから,鋼材の選択においては設計,用途(使用環境)などを十分に考慮して,引張強さ,靱性を決定しなければならない.d. 鍛造成形性精密型鍛造では正確な形状と良好な肌は重要な要求品質であるが,最近の構造用合金鋼では不純成分は著しく減少して,材料の変形能に由来する難鍛造性はない.要点は材料と金型の界面にあるが,材料の無酸化加熱,周辺機器の整備と高速化によるスケールの確実な除去,金型と材料の体積的なマッチングなどによって,全体としての鍛造成形性は格段に向上した.e. 被削性構造用合金鋼の範囲では,被削性(machina-bility)は中位の性能と見られる.最近,材料の靱性が著しく増加して,機械装置の信頼性はいっそう高められたが,その反面,被削性が低下して,ある種の製品の最終仕上げを困難にしている.靱性と被削性は,互いに相いれない特性と考えられるが,まず,靱性を可能な限り高めて,次に切削加工の総合的な効率化を考えるべきである.快削鋼(free machining steel,S やPb入りなど)に途を譲る場合は限られている.f. 溶接性材質と金属組織(熱処理の効果),設計応力とその変動パターン,母材の肉厚,開先形状(拘束の程度),機器の運転環境など,構造用合金鋼の溶接性に影響する要素は多い.溶接性(weldability)の各種評価法については専門書に譲るが,溶接構造用合金鋼の選択に当たっては,次の諸点に留意する.製品分類JIS
