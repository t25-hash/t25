# 2・4・6 関連法規

> **章**: 第2章 設計・評価技術
> **節**: 2・4 工場計画手法

建築物にはさまざまな法規制が加えられている．特に工場に対し
ては廃棄物，騒音，危険物の使用等一般の建物とは違い，環境に影
響を与える要因が多種多様にわたるためさまざまな法規が関連す
る．これら関連法規を図2･47に示す（環境関連を除く）．
これら関連法規を計画段階で十分に考慮するとともに早い段階で
の官公庁，近隣との折衝を行い，工場稼働予定どおりに進むように
する必要がある．これら関連法規には立地地域により県条例や地域
独特のもの（瀬戸内海環境保全特別措置法など）が存在するため，
立地選定時や計画段階で十分に考慮し選定基準の評価にする一方，
工場誘致による緩和規定や助成制度等についてもよく調べておく必
要がある．
こうした関連法規の中で，特に工場建設に関連する三つの法規に
ついて述べる．とりわけ環境問題は地域性が高く，国の法規制に地
方公共団体の条例による規制が加わるため注意が必要である．
a.工場立地法（factory location law) 工場立地法の対象工
場は敷地面積9000m以上，または建築総面積3000m以上が対
象となる．当然敷地の増加や増築により，上記条件を超えた場合も
届け出が必要となる．新設や変更の実行は，この届け出受理後90
日をみておく必要がある．また，地区により緑地面積はまちまちと
なる．
b.建築基準法（building standard law) 工場施設に対して
の建築規制の主たるもので，建築主は工事着工前にその建築計画が
法令に適合しているかの確認を行うため，地方公共団体の建築主事
に提出し承認を受ける必要がある．このとき提出するものとして，
･開発行為または建築についての許可に関する証明書
･建物の設計図書（図面および書類)
が必要で，当然ながらこのとき消防法にも適合するものでなくては
ならない．申請から許可まで約1.5～2箇月程度みておく必要があ
る．当然この許可が下りるまでは着工はできない．
c.環境基本法（basic environment law) 1970年代ごろから
当時の環境保全活動の高まりとともに，各種の環境基準や公害に対
する法の整備がなされてきた．これらの関連法規を図2･48に示す．
特に工場はさまざまな環境破壊物質を排出してきた歴史的経緯があ
り，地域住民や社会の厳しい目にさらされているといってよい．こ
のため，これらの法規を遵守することは当然のことであるが，生産
活動という事業を行うためには，効率の良い投資でこれらへの対策
を行う必要がある．対策方法は，生産システムだけではなく工場計
