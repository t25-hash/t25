# 2・3・5 鋼材の品質

> **章**: 第2章 鉄鋼材料
> **節**: 2・3 構造用合金鋼

鋼材の内部欠陥には種々あるが、製鋼、鍛造、熱処理法の進歩によって、最近の構造用合金鋼の欠陥(defects in structural alloysteel)は次のようなものに限定されている。a. 欠陥の種類(1) 砂きず(非金属介在物のクラスタ)。(2) 水素割れ。b. 欠陥の検出可能寸法(1) 表面欠陥0.5mmのもの。(2) 内部欠陥超音波探傷試験において波長からその2分の1程度までの相当直径のもの(2MHz ならば、約1.5~3mm径)。このような欠陥の現実を機械的性質とあわせて考慮して、材料の表2・31 構造用合金鋼の規格:クラッド材(海外規格との対比も表示)引張試験おもな用途類似規格・記号引張強さ伸びMPa%合せ材:各種ステンレス鋼母材:各種炭素鋼、合金鋼圧力容器、ボイラ、貯槽ASTM A263-79母材または合せ材の規格のうち、いずれか低いほうの規格値以上ASTM A264-79BR 1,BR 2合せ材:JIS H 4600,TP28,TP35母材:各種炭素鋼、合金鋼〃B 1,B 2合せ材:銅、銅合金各種母材:各種炭素鋼、合金鋼圧力容器、貯槽、水処理装置ASTM B 432-76a仕様を決める。
