# 2・3・6 構造用合金鋼の用途とその性能

> **章**: 第2章 鉄鋼材料
> **節**: 2・3 構造用合金鋼

a. 機械装置,構造物の部品全般構造用合金鋼の素形材は機械工業を支える主要構造材料の一つでその用途は多岐にわたるが,前述の諸規格,材料選択と合金設計の指針に照らして決定することが望ましい.まず機械工業の全般にわたる代表的な中・小型の製品を挙げる.これらの多くは鍛造素形材である.(1) 産業機械関係軸類,スピンドル,ピストン棒,ロータ軸,ディスク,フランジ,シリンダ,歯車類,ケーシング.(2) 自動車,車両,舶用機器,航空機関係各種エンジン部材,伝達系部材,クランク軸,ピストン棒,接合棒,シリンダカバー,ピストンクラウン,クロスヘッド,タービンロータ軸,歯車類,プロペラ軸,中間軸,推力軸,ラダーストック.(3) 鉄鋼機械関係連鋳機用部材,スピンドル,カップリング,ピニオン,歯車類,リム,テーブルローラ,ラム,シリンダ,コラム,アンビル.(4) 電力原子力関係タービン軸,ディスク,水車軸,発電機軸,主塞止弁,圧力容器胴,フランジ,ノズル,チューブシート.(5) ボイラ関係フランジ,T-Y ジョイント,水室,フィッティング,ボイラ胴,ボイラ水管,ヘッダ.(6) 化工機関係圧力容器胴,ノズル,チャネル,カバー,フランジ,チューブシート.次に,これらの中から,大型,高品質の代表的なものを挙げて,構造用合金鋼の用途と技術の先端を示すこととする.b. タービン,タービン発電機軸材タービン軸材(turbineshaft material),発電機軸材(generator shaft 圧延用ロール金属材料の塑性加工において圧延は最も大きな量的比率を占める.無機材料(ガラスなど),有機材料(プラスチックなど)でも同様であるが,鉄で鉄を加工する鉄鋼圧延においてロールに要求される性能は最も厳しく,分塊から孔型圧延,さらには管圧延に至るまでの複雑な成形過程での,摩耗,肌荒れ,クラック発生,折損などの過酷な損耗に耐えて,圧延トン数を最大にすることに要点がある.このためロール(roll)の材質は,表2・34に示すように,まず硬さを目安として高炭素側のものが選ばれる.構造用合金鋼鍛造品としては,鋼板圧延における補強ロール(back up roll)とワークロール(work roll)が主である(表2・35).これらの材質は材料規格としては工具鋼寄りのものであり,圧延におけるいっそうの高負荷操業指向に応えて,絶えず最適条件が探求されている.e. 圧力容器用鋼圧力容器(pressure vessel)は機械工業のあらゆる分野で,小型から超大型まで,また極低温から高温まで使用され,圧力容器用鋼(pressure vessel
