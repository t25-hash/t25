# 2・5・2 シミュレーション（simulation）とは

> **章**: 第2章 設計・評価技術
> **節**: 2・5 生産システムのモデル化とシミュレーション

自然界や人間社会におけるさまざまな現象やシステムの振舞いを
解明するためには，現実の世界における振舞いを直接観察するばか
りでなく，図2･50に示すように実世界を抽象化，簡略化した仮想
世界を構築し，さまざまな解析を可能とすることが考えられてい
る．仮想世界は解析対象に焦点を絞って実世界と同等な振舞いを再
現できるようにしたものであり，モデル（model ）と呼ばれる．理
論的な数式モデル，風洞モデルなどのような実体モデル，あるいは
コンピュータシミュレーションモデルとして実現される．
シミュレーションは，対象とする現象やシステムの空間的，時間
的な変化を逐次的に観察できる仕組みを仮想世界，すなわちシミュ
レーションモデルとして構築し，変化のありさまを追跡し，対象の
特性を解析できるようにするものである．したがって，シミュレー
ションモデルの構成やパラメータをさまざまに変更することによ
り，モデルの特性がどのように変化するかを明らかにすることがで
きる．
さまざまな解析を容易に行えるようにするためには，できるだけ
モデルを簡略化することが望ましいが，対象とする現象の特性が的
確に把握できる程度に詳細にわたるものでなければならない．シス
テムが大規模，複雑化し，さらにその詳細な特性の解明が必要にな
るにつれて，その必要度に応じた適切なモデルを比較的容易に構築
することができるコンピュータシミュレーションの有用性が広く認
められてきている．
