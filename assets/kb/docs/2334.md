# 2・5・6 生産システムシミュレーションの今後

> **章**: 第2章 設計・評価技術
> **節**: 2・5 生産システムのモデル化とシミュレーション

2･5節では生産システムのシミュレーションについて，離散シミ
ュレーションを中心にその仕組みと利用法について述べた．また，
近年の情報化の進展とともに生産システムのシミュレーションにお
いて必要性が高まると考えられる分散シミュレーションについても
触れた．離散シミュレーションは従来から広く用いられているが，
依然として，計画・設計の段階で用いられるにとどまり，運用段階
での利用は限られている．今後，サプライチェーンマネジメントが
普遍化し，生産の現場をプロフィットセンタ（profit center ）とし
て管理する重要性が高まるとともに，シミュレーションを評価に用
いるだけでなく，予測に用いることにより運用に不可欠な意思決定
（decision making）支援ツールとなることが期待される．このた
めには，生産システムの効率を従来の時間，数量，比率などを指標
として評価するばかりでなく，経営的意思決定指標であるコストを
指標として取り入れることが必要となると考えられる．
