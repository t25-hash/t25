# 2・4・5 析出硬化型ステンレス鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・4 ステンレス鋼およびステンレス合金

析出硬化型(PH : precipitation-hardening)ステンレス鋼は、ステンレス鋼としての耐食性を保持し、析出硬化を利用して強度の向上を図ったものである。基地の組織により、マルテンサイト系、セミオーステナイト系、オーステナイト・フェライト系、オーステナイト系に分けられる。析出硬化元素として、Al,Cu,Mo,Nb,Tiなどが添加される。17-4PH(SUS 630)および17-7PH(SUS 631)が最も代表的な鋼種であり、生産量もこの種のステンレス鋼中では最も多い。前者はマルテンサイト系で17Cr-4Ni-4Cu-0.06C-0.25Nb、後者はセミオーステナイト系で17Cr-7Ni-1.2Al-0.07Cの組成である。析出物は前者ではCuに富む析出相、後者ではNiAl相であり、これらが微細に分散析出して硬化に寄与する。これら両鋼種の耐食性は18Crのフェライト系ステンレス鋼よりは優れている。17-4PH は歯車、軸、送酸ポンプのライナ、バルブ、コックなど耐食性と耐摩耗性、高強度を要する部品に、17-7PH は航空機外殻、ジェットエンジン部品、スプリングなどに用いられている。オーステナイト・フェライト系は、多量のCr,Si,Mo,Beな図2・34 ステンレス鋼の系統図どを含み、高温からの冷却で約50%のフェライトを含む二相鋼である。析出硬化はフェライト地で起こる。Cr,Si,Moを多く含むため耐食性は著しく改善されている。耐摩耗性、耐食性を必要とするポンプ部品、化学装置部品などに使用されている。オーステナイト系は、析出硬化のためにC,P,N,Ti,Al,Nb,V などを添加し、オーステナイト地に、炭化物、窒化物、リン化物、Ti,Alなどの金属間化合物として析出させ硬化させている。SUH 660(A 286)が代表鋼種である。この系の鋼種は耐食鋼としてよりも耐熱鋼としての用途が多く、高温用送風機、高温用ボルト、高温用ばねなどとして使われている。おもなステンレス鋼を系統的にまとめて図2・34に示す。
