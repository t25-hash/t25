# 2・6・1 高炭素クロム軸受鋼および肌焼軸受鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・6 軸受鋼

玉軸受,ころ軸受のように接触面圧が1000~1300MPa,場合によっては3000~4000MPaにも達する軸受には,炭素含有量の多い高炭素クロム軸受鋼や表面を浸炭する肌焼鋼が用いられる.JIS 規格には高炭素クロム軸受鋼として5種類が規定されており,1%C-1~1.5%Crを主成分としてMnやMoによって焼入れ性に変化をもたせている.表2・44にこれらの用途に用いられる鋼の化学成分を示す.この中でSUJ 2は軸受鋼の基本鋼であり,使用される軸受鋼の大部分が当鋼種である.SUJ 1は焼入れ性が低く使用可能範囲が狭いため,現在ではほとんど使われていない.SUJ 3は,SUJ 1の焼入れ性をSiとMnで高めた鋼種であり,SUJ 2よりも大型の軸受に使用されている.SUJ 4は,SUJ 2の焼入れ性をMoで高めた鋼であり,SUJ 2とSUJ 3の中間の焼入れ性を示す.SUJ 4は新幹線用軸受などに使用されるが,その使用量は非常に少ない.SUJ 5は,焼入れ性が高くSUJ 3よりもさらに大型の軸受用として使用される.ただし,SUJ 4,5が該当する外国規格はない.軸受としてはHRC58以上が必要とされており,これらの鋼は優れた軸受性能を得るために,通常,1050~1120Kでオーステナイト化後焼入れして約7vol.%の未固溶の球状炭化物がマルテンサイト基地中に均一に分散した組織としたのち,これを420~470K の低温で焼戻しして使用される.ただし,寸法安定性が特に問題となる場合には焼入れ後にサブゼロ処理あるいは高温焼戻し処理が採用されることもある.また,硬さが高いため,地きず,非金属介在物の少ない清浄な鋼が望まれる.転がり疲れ寿命は,非金属介在物,特に酸化物系非金属介在物の影響を強く受けるため,軸受鋼における非金属介在物の低減は極めて重要である.鋼中の酸素は全量が酸化物系非金属介在物になっていると考えてよく,最近優れた溶解・鋳造技術によって酸素量を5ppmに低減した極低酸素鋼は以前の非脱ガス鋼と比較して数百倍の転がり疲労寿命を示すことが報告されている.量産軸受鋼における製造行程では,現在,電気炉または高炉-転炉→とりべ精錬→真空脱ガス(RH 脱ガス)→連続鋳造が主流となっており,極低酸素の高清浄鋼が得られている.清浄度の測定法としてはJIS 点算法やASTM 法に加えて,最近では極値統計を用いた方法などが注目されており,製鋼プロセスと非金属介在物の大きさ,転がり疲れ寿命の関係をより明確にし軸受材料のさらなる長寿命化・高信頼化が図られている.肌焼軸受鋼は,肌焼鋼を浸炭あるいは浸炭窒化処理して作られる表2・44 JIS で規格化されている高炭素クロム軸受鋼の化学成分(wt.%)化学成分wt.%CSiMnCrMoSUJ1 0.95~1.10 0.15~0.350.50以下0.90~1.20―SUJ2 0.95~1.10 0.15~0.350.50以下1.30~1.60―SUJ3 0.95~1.10 0.40~0.70 0.90~1.15 0.90~1.20―SUJ4 0.95~1.10 0.15~0.350.50以下1.30~1.60 0.10~0.25SUJ5 0.95~1.10 0.40~0.70 0.90~1.15 0.90~1.20 0.10~0.25ので高炭素クロム軸受鋼と比較して熱処理コストが高くなるが,高炭素クロム軸受鋼と同様の高い硬さの表層部に加えて強靱性な心部を有しており,特に衝撃荷重を受ける用途に適している.また,表層部には圧縮残留応力が残存するので耐割れ性に優れること,硬さが低いため冷鍛性に優れているなどの長所がある.浸炭条件の調節は容易であり,鋼種選択に当たっては軸受の肉厚に応じた焼入れ性をもつ鋼種を選んで利用する.肌焼軸受に用いられている鋼としてはJIS SCr 420,SCM 415,SCM 420,SNCM 220,SNCM 420などがある.浸炭処理は,浸炭したのち冷却してから再加熱・焼入れをする方法と,浸炭したのち直接焼入れする方法とがあるが,大型軸受の場合は前者が多い.浸炭後の焼戻しは,通常420~470K で行い,心部の硬さはHRC35~40に調整される.用途例としては,おもに製鉄用圧延機,土木建築用重車両,鉄道車両など大型機械の回転部分に使用される.特殊な例としてニードルローラ軸受の外輪に使われることもある.
