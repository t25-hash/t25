# 2・6・3 ステンレス軸受鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・6 軸受鋼

ステンレス軸受鋼は酸に対する耐食性に優れ、またCr含有量の増加に伴って耐酸化性も向上する。低摩擦を必要とする計器用軸受、腐食環境の厳しい船舶用軸受、腐食性雰囲気で使用される化学工業用軸受などに用いられる。代表的な鋼種はJIS SUS 440Cである。SUS 440Cは17%Cr-1%Cを主成分とする高炭素マルテンサイト系ステンレス鋼であり、すべてのステンレス鋼の中で最高の硬さを有する。ただし、焼なまし硬さも高いので高炭素クロム軸受鋼より被削性は劣る。また440系鋼種は、C,Cr量が多く、寿命劣化の原因となる巨大炭化物が生成しやすいので製造方法に注意を要する。
