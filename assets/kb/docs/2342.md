# 2・6・4 高温軸受鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・6 軸受鋼

使用温度が390K を超える環境下で用いられる軸受には、SUJ 2のような低温焼戻しタイプの鋼は組織変化を起こし軟化や寸法変化を生じるので使用できない。したがって、高温軸受用には焼戻し軟化抵抗が高く、さらに二次硬化を起こすような鋼が用いられる。この種の軸受鋼では、使用温度における硬さと寸法安定性が最も要求される。実際に生産されている高温軸受のほとんどは航空機用である。精錬法として真空誘導炉溶解-真空アーク再溶解のダブルメルト法などの特殊溶解が適用され、不純物レベルは低く抑えられている。鋼種としてはM 50,T 1,JIS SKH 51,SKH 2などがある。M 50(0.8%C-4%Cr-4.3%Mo-1%V)はジェットエンジン用高温軸受として最も使用量が多く、使用温度の上限は700K とされている。M 50に次いで使用量の多いT 1(0.7%C-4%Cr-18%W-1%V)はSKH 2に相当するタングステン系高速度鋼である。これらの高炭素高合金鋼では、製造行程において転がり疲れ寿命を劣化させる巨大炭化物の生成をいかに抑えるかが重要である。通常の溶解・鍛造のプロセスでは添加元素量に限界があるため、さらに高温をねらった材料では粉末冶金法も検討されている。ばね鋼(spring steel)は、高い弾性限を利用して繰返し応力下でのエネルギー吸収を図った、いわゆるばね性を高めた材料であり、疲労特性と耐へたり性に優れることが要求される。設計に際しては引張強さまたは硬さに応じて増加する弾性限が重要である。非金属介在物や表面きず、脱炭層などの表面欠陥は疲労特性に強い影響を与えるため、製造行程においてこれらの欠陥をできるだけ低減することが必要である。熱間圧延で特に変形しにくい酸化物系非金属介在物は疲労起点になりやすく、低酸素化によりその総量を低減する方法や介在物を低融点組成に制御して軟質化する方法がとられている。耐へたり性の向上にはばね素材の高強度化と鋼の焼戻し軟化抵抗の増加が必要である。へたりに対する予防策としてはホットセッティング、クリープテンパなどがある。ばね素材の全体ではなく表層部のみを強化する方法として、ショットピーニング(shot peening)、窒化処理などの表面改質技術が採用されている。また、高強度ばねでは腐食疲労による材質の劣化が大きな問題となっている。ばね用材としては高弾性限のものほどその特性は優れているが、実用に当たっては弾性係数、形状、寸法精度に十分配慮する必要がある。また、製造コスト原価が低く、熱処理などの操作が容易なことが望ましい。ばね鋼は、製造行程の違いから熱処理ばねと加工ばねに大別できる。
