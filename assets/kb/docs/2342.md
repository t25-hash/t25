# 2・6・4 高温軸受鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・6 軸受鋼

使用温度が390Kを超える環境下で用いられる軸受には，SUJ2
のような低温焼戻しタイプの鋼は組織変化を起こし軟化や寸法変化
を生じるので使用できない．したがって，高温軸受用には焼戻し軟
化抵抗が高く，さらに二次硬化を起こすような鋼が用いられる．こ
の種の軸受鋼では，使用温度における硬さと寸法安定性が最も要求
される．実際に生産されている高温軸受のほとんどは航空機用であ
る．精錬法として真空誘導炉溶解-真空アーク再溶解のダブルメル
ト法などの特殊溶解が適用され，不純物レベルは低く抑えられてい
る．鋼種としてはM50，T1，JIS SKH51，SKH2などがある．
M50（0.8%C-4%Cr -4.3%Mo-1 %V）はジェットエンジン用高温
軸受として最も使用量が多く，使用温度の上限は700Kとされて
いる．M50に次いで使用量の多いT1（0.7%C-4%Cr -18%W-
1 %V）はSKH2に相当するタングステン系高速度鋼である．これ
らの高炭素高合金鋼では，製造行程において転がり疲れ寿命を劣化
させる巨大炭化物の生成をいかに抑えるかが重要である．通常の溶
解・鍛造のプロセスでは添加元素量に限界があるため，さらに高温
をねらった材料では粉末冶金法も検討されている．
ばね鋼（spring steel）は，高い弾性限を利用して繰返し応力下
でのエネルギー吸収を図った，いわゆるばね性を高めた材料であ
り，疲労特性と耐へたり性に優れることが要求される．設計に際し
ては引張強さまたは硬さに応じて増加する弾性限が重要である．非
金属介在物や表面きず，脱炭層などの表面欠陥は疲労特性に強い影
響を与えるため，製造行程においてこれらの欠陥をできるだけ低減
することが必要である．熱間圧延で特に変形しにくい酸化物系非金
属介在物は疲労起点になりやすく，低酸素化によりその総量を低減
