# 2・10・1 構造用鋳鋼（steel casting for structural purposes )

> **章**: 第２章 鉄鋼材料
> **節**: 2・10 鋳鋼

a.炭素鋼鋳鋼（carbon steel casting) 炭素鋼鋳鋼品は，Si
を約0.5mass%以下，Mnを約0.8mass%以下含むほか，合金成
分を含まないものをいう．含有するC mass%（以下massを略す）
によって以下の3種類に分類される．
低炭素鋼鋳鋼品 C＜0.2%
中炭素鋼鋳鋼品 C＝0.2～0.5%
高炭素鋼鋳鋼品 C＞0.5%
この中で，需要の多い低炭素鋼および中炭素鋼鋳鋼品は引張強さ
によって表2･53に示すようにJIS規格に規定されている．JIS規
格では炭素鋼鋳鋼品（JIS G5101），溶接構造用鋳鋼品（JIS G
5102）である．
から明らかなように，C%の増加とともに強度は増し，靱性は低下
するため，使用目的に応じて適当な鋼種を選ぶべきである．一般に
は構造用材料としての需要が多いため，SC410，SC450，SC480
などが多く使用される．また寒冷地用材料として使用するときには
C%を低くし，Mn%を上げて低温靱性を向上させた低炭素鋼鋳鋼
品を用いるとよい．表2･54に炭素鋼鋳鋼品の用途例を示す．
がある．JlS G5102には溶接性を保証した炭素鋼鋳鋼として，溶
接構造用鋳鋼が4種類規定されており，材質記号にSCW410のよ
うにWの記号を付けて一般の炭素鋼鋳鋼と区別している．また溶
接性を保証するためにC%，炭素当量（carbon equivalent ）など
化学成分の上限を規定している．

b.低合金鋼鋳鋼（low alloy steel casting) 低合金鋼鋳鋼の
JIS規格は構造用高張力炭素鋼および低合金鋼鋳鋼品（JIS G
5111）であり，比較的高い強度［引張強さ490～1170N/mm （50
～120kgf / mm )］と優れた靱性，焼入れ性（hardenability）をも
たせるためにMn，Cr ，Moなど種々の合金元素を添加したもの
で，目的，用途に応じて焼入れ，焼戻し，焼ならし後焼戻しを施し
て使用する．合金元素添加の最大の目的は焼入れ性の改善である．
図2･44は鋼の焼入れ性に及ぼす合金元素添加の効果 を示す．
焼入れ硬化能に対する合金元素の効果はMnが最も大きく，Mo，
Cr ，Si ，Niの順に小さくなる．表2･55はJISに規定されたおも
な低合金鋼鋳鋼品の機械的性質とおもな化学成分を示す．同一合金
系でもC%によって数種類に分けられ，C%の低いものは溶接性を
重視した構造用材料として用いられることが多く，C%の高いもの
は表面を高周波焼入れなどにより高硬度にして耐摩耗部品として使
われることが多い．低合金鋳鋼の各合金系の特徴は以下のようであ
る．

.シリコンマンガン鋼鋳鋼（silicon manganese steel cast-
ing，SCSiMn) 低マンガン鋼鋳鋼よりMnを少なくしてSiを
多くすることにより溶鋼の流動性を改良した鋼種で，比較的薄肉で
複雑形状の鋳鋼品に適する．
.マンガンクロム鋼鋳鋼（manganese chronium steel cast-
ing，SCMnCr ) 低マンガン鋼鋳鋼にCrを添加して焼入れ性を
改良し，高い硬さと引張強さをもたせた鋼種で，車輪，ショベルの
刃先などの耐摩耗部品に用いられるが，鋳造時や溶接時に割れが発
生しやすい欠点がある．
.マンガンモリブデン鋼鋳鋼（manganese molybdenum
steel casting，SCMnM) Crの代わりにMoを添加し，焼戻し
軟化抵抗を改良した鋼種で，鋳造時や溶接時の割れも少なく，かつ
製品内部まで高い強度を有するため，大きな負荷を受ける大型部品
に使われる．
.ニッケルクロムモリブデン鋼鋳鋼（nickel chromium
molybdenum steel casting，SCNCrM) 高強度，高靱性で焼戻
し軟化抵抗も大きく構造用鋳鋼品中最も優れた鋼種であり，高荷重
で高面圧を受ける掘削刃や歯車などに使用される．上記の低合金鋳
鋼のほか，JIS規格にはないが高価なNi ，Moの添加をやめ，安
価で焼入れ性に優れた徴量のBを添加したボロン鋳鋼や，鋼材の
ハイテンと同様に少量のTi ，Nb，Vなどを含んだ高張力鋳鋼もか
なり使用されている．
