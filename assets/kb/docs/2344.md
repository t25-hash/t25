# 2・10・3 耐熱（heat resistance）鋼鋳鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・10 鋳鋼

耐熱鋼および耐熱合金鋳造品（heat resisting cast steels and
alloys for general applications ）の中の耐熱鋼は高Cr鋼，高Cr -
Ni鋼が用いられ，約770K以上の高温に耐える性能をもってい
る．2･10･2項のステンレス鋼鋳鋼と同程度のCrやNiを含むが，
さらにC%，Si%をそれぞれ0.2～0.5 %，1.5～3.0%と高くし，
高温クリープ強度の向上と耐酸化性（oxidation resistance）の改
善を図っている．耐熱鋼および耐熱合金鋳造品はSCHI ～47まで
ある．その中の規格はJIS G5122に示され，SCH1～24まで，32
種の鋼種が耐熱鋼として規定されている．おもな鋼種は次のようで
ある．
(１)高Cr系：13%Cr ，28 %Cr
(２) 高Cr -Ni系：19%Cr-9%Ni ，25%Cr-12%Ni ，25%Cr-
20%Ni
CrやNiが増すほど高温強度は向上し耐用温度も上昇する．一
必要がある．
