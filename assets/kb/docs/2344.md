# 2・7・2 加工ばね鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・7 ばね鋼

加工ばね鋼は,冷間加工および熱処理によってばね性を高めた鋼であり,ばね成形が冷間で行われるため,線ばね,薄板ばね,小物ばねなどとして使用される.高炭素鋼線(ピアノ線,piano wire,high carbon steel wire)は,パテンティング処理で得られる微細パーライト組織に大きな減面率の伸線加工を施してばねの形状とし,これを低温で焼戻してばね性を付与した材料で,一般ばね用や弁ばね用に適用される.硬鋼線は,強度面ではピアノ線に劣るが応力の繰返し数が少なく衝撃荷重がかからない用途で用いられる.伸線加工された鋼線に焼入れ,焼戻し処理を施して得られるオイルテンパ線(oil tempered wire)は,弾性限,降伏比が高く耐熱性も期待できることから,現在,弁ばね,懸架ばねなどに用いられる高応力ばね鋼線の主流となっている.2100~2200MPaの強度を有するオイルテンパ線も実用化されている.ステンレス鋼を素材として冷間でばね性を付与したばね用ステンレス鋼線は,耐熱性,耐食性,非磁性が要求される用途に用いられる.析出硬化型ステンレス鋼のSUS 631(17-7PH)は,冷間で強加工を加えたのちばねに成形し,750~840K の範囲で時効処理するとNiAlなどの金属間化合物の微細析出により1370~1820MPaの引張強さが得られるので,耐食性が要求される高強度ばねとして利用される.ばね用冷間圧延鋼帯は冷間圧延によりばね性を付与したもので,寸法精度,肌が良好などの特徴を有し,薄板ばねとして用いられる.焼入れ焼戻し鋼帯は薄板ばねやぜんまいとして用いられる.焼入れ焼戻し鋼帯には,ばね成形後焼入れ焼戻しするものと,焼入れ焼戻し後成形するものがある.工具鋼(tool steel)は,金属や非金属の切削,塑性加工用などの各種ジグ・工具として用いられる特殊鋼で,用途が広く,要求性能は多岐にわたる.化学成分および性能を考慮して,JIS 規格では,現在,11種の炭素工具鋼(carbon tool steel),32種の合金工具鋼(alloy tool steel),15種の高速度工具鋼(high speedtool steel)が規格化されている.実用的にはJIS 規格以外にも各特殊鋼メーカで独自の鋼種がブランド化されている.加えて,新鋼種の開発だけでなく,主要成分元素の高精度コントロールや非金属介在物の低減およびO,P,S などの不純物元素の極低減化によって工具鋼の品質向上も図られている.工具鋼はおもな用途に従って切削工具用,耐衝撃工具用,冷間金型工具用,熱間金型工具用に大別される.表2・46に代表的な工具鋼の化学成分およびおもな用途例を示す.炭素工具鋼は0.6~1.5%のCを含有し,特別に合金元素を添加しない最も基本的な工具鋼である.価格も廉価なので各種の工具鋼として用いられており,その生産量は工具鋼全体の過半数を占めている.このような高炭素鋼は,通常,1020~1050K 付近でセメンタイトの球状化焼なまし処理を行ったのち,オーステナイト化後焼入れして球状の未固溶セメンタイトがマルテンサイト基地中に分散した組織とし,これを焼戻しして用いられる.鋼の炭素含有量が多いほどセメンタイト量が増し耐摩耗性が向上する.一方,靱性は炭素含有量が少なく,炭化物が微細で均一に分散しているほど向上し,また基地の結晶粒度が細かいものほど高い靱性を示す.高い硬さと耐摩耗性を要する切削工具,組やすりなどには含有量の多い鋼が使用され,靱性を必要とするぜんまい,刻印,プレス型などには炭素含有量の少ない鋼が使用される.炭素工具鋼は,元来,焼入れ性が低いため,水焼入れをする必要がある.その際,寸法のやや大2―70材料学・工業材料表2・46 系とMo系に大別される.おもな用途は切削工具や金型などで,特に高速切削に適し,摩擦熱による温度上昇によく耐える.また,粉末冶金法で得られる粉末冶金工程高速度工具鋼は,炭化物が極めて微細で均一に分散した組織を有し,高硬度と高耐摩耗性を兼ね備えており,従来の溶製材よりも高合金化が可能である.高速度工具鋼の焼戻しは,通常,830~900K の高温域で行い,残留オーステナイトを十分に分解するために焼戻しを2~3回繰返す.
