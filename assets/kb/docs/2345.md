# 2・8・1 切削工具鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・8 工具鋼

切削用としては、切削性の観点から高い硬さが要求されると同時に、切削時の衝撃によって刃こぼれが生じないような適度な靱性も要求される。したがって、これらの兼合いから用途に適合した炭素含有量の鋼が選ばれる。かみそり、やすり、バイトなどには、通常、炭素含有量が1.1~1.5%の鋼が多く用いられている。炭素工具鋼ではSK 140(SK 1)、SK 120(SK 2)などがある。さらに3~5%のW の添加により耐摩耗性、焼戻し軟化抵抗を高めたSKS 11がある。ハクソー、ドリル、ねじ切りダイスなどのやや靱性を要する用途には炭素含有量が1.0~1.2%の鋼が用いられる。SK 105(SK 3)CSiMnCrMoWVその他用途例(参考)にCr,W などを添加したSKS 2およびSKS 21は、特殊炭化物の形成によって耐摩耗性を向上させた鋼で、タップ、ドリルなどとして多く使われている。W を2~2.5%とやや多く含むSKS 7は耐摩耗性に優れているのでハクソーとして用いられている。木工用丸のこや帯のこなどには炭素含有量が0.7~0.9%のSK 85(SK 5)、SK 75(SK 6)などが用いられる。SKS 5およびSKS 51はNi,Crなどを添加してさらに靱性を向上させた鋼である。高速切削では工具に大きな機械的摩耗を受けるうえ、摩擦熱によって刃先温度が著しく上昇するので、高温での硬さが必要となる。このような用途には高速度工具鋼が用いられる。JIS SKH 55などのCoを含む鋼は焼戻し軟化抵抗と高温硬さが高いので高速切削に適する。
