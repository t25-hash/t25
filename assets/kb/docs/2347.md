# 2・8・3 冷間金型用工具鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・8 工具鋼

冷間金型用としては硬くて耐摩耗性に優れることが要求されるので、比較的炭素含有量が多く、Cr,Mo,W,V などの炭化物形成能の高い合金元素を多く含む高合金鋼がよく用いられる。通常、冷間金型用工具鋼は焼なまし状態で機械加工を施し、その後の熱処理によって硬化させたのちは研磨仕上げなどのわずかな仕上加工のみで使用するため、熱処理時の変形が少ないことも要求される。炭素工具鋼では炭素含有量が0.9~1.1%のSK 105(SK 3)、SK 95(SK 4)などが抜型として使用されるが、水焼入れを要するので焼入れひずみが大きく、また寸法の大きな型では焼入れ硬化が十分でない。Mn,Crを添加して焼入れ性を改善した鋼としてSKS 93~SKS 95がある。最近のプレス金型には油冷でも高硬度が得られるSKS 93が広く採用されている。さらにW の添加により耐摩耗性と焼入れ性を向上させた鋼としてSKS 3およびSKS 31がある。1.4~2.2%のC,11~15%のCrを含有したSKD 1,SKD 11は、炭化物量が多く耐摩耗性に優れており、プレス型などをはじめとして多目的に使用される。1%のC,5%のCrを含むSKD 12は多少耐摩耗性を犠牲にしても靱性を要する場合に適している。これらの鋼種はいずれも焼入れ温度からの空冷によって硬化できるので、焼入れひずみはSKS 3よりもさらに小さい。冷間鍛造用型鋼としてSKH 51などの高速度工具鋼も使用される。冷間金型用の硬さは一般にHRC58以上を必要とし、二次硬化を示さないSKS などの低合金鋼では450K 付近の低温焼戻しが施されるのに対し、SKD 11などの高合金鋼では低温焼戻しと770K以上の高温焼戻しが用途に応じて使い分けられる。低温焼戻しと比較して、工具表面の温度上昇に耐えるには高温焼戻しが有利であり、しかも熱負荷に対する相安定性と焼入れ残留応力の低減を図ることができる。最近では、SKD 11に対し、晶出炭化物を微細に分散して靱性を向上させるとともに、高温焼戻しでHRC62が得られる8Cr系鋼の利用が定着化している。大型の抜型、曲型などで全体を焼入れ、焼戻しすることが困難なものでは炎焼入れ(flame hardening)などによって局所的に所要部分を硬化することが行われる。また、冷間金型の用途で性能を上げる表面改質としては硬質被膜処理が挙げられ、CVD(化学的蒸着)、PVD(物理的蒸着)、TRD(熱反応析出・拡散)処理などが用いられている。
