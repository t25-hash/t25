# 2・8・4 熱間金型用工具鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・8 工具鋼

熱間金型用としては，高温強さ，靱性，耐摩耗性，耐高温酸化性
などに優れていることに加えて，急熱急冷による熱応力の繰返しで
表面にき裂を発生するいわゆるヒートチェック（heat checking）
に対する抵抗性が高いことが要求される．特に耐ヒートチェック性
を配慮して炭素含有量の少ない鋼が選ばれる．冷間金型と比較して
熱間金型は寸法が大きく，形状も複雑なものが多いので，焼入れの
際には割れや変形を生じさせないように配慮するとともに，十分な
性能を得るための冷却条件を選択する必要がある．また熱間鍛造時
には金型表面が被鍛造材からの熱により870～970Kまで上昇する
ことが知られており，耐熱性を保持させるために，通常，770～870
K前後の高温焼戻しが採用される．
SKD4，5はW量が高く，高温強さ，耐摩耗性は大であるが，
ヒートチェックには弱い点がある．SKD6，SKD61はヒートチェ
ックを改善した鋼種で，熱間プレス型，アルミダイカスト型，押出
工具などに用いられている．SKD62は，耐摩耗性を改善した鋼で
あり，プレス型や押出型などに広く利用されている．4%のCoを

あり熱処理による変形が少なく，用途に応じてHRC43～50程度
の硬さに熱処理して用いられる．
Ni -Cr -Mo系のSKT3およびSKT4鋼は高温硬さや焼戻し軟
化抵抗は前述のSKD系鋼に比べて劣るが，靱性が高いので衝撃荷
重のかかる鍛造型や大型で割れの危険性のあるプレス型などに用い
られる．SKT系鋼はHS56前後の硬さで用いられるので，通常焼
入れ，焼戻し後に型彫りを行って用いられる．
金型の表面処理は，窒化処理や浸硫窒化処理が主体であり，温
間，熱間プレス型やアルミダイカスト型，アルミ押出工具などに適
用されている．
熱間金型用鋼は冷間金型用鋼に比べてその使用環境が複雑で厳し
いことから，より多くの研究と新種開発が行われている．
2・8・5ゲージ用鋼
ゲージとして最も必要な性質は使用中に寸法，形状の変化を生じ
ないことである．そのためにゲージとしては経年変化，耐摩耗性，
耐さび性，熱膨張係数などを考慮して使用材料が選択される．また
形状の複雑なゲージやブロックゲージでは熱処理変形が小さいこ
と，研削やラップ仕上げなどの加工性がよいことも要求される．
合金工具鋼SKS3およびSKS31は焼入れ性や耐摩耗性がよく，
ゲージ用鋼として最も多く用いられる．炭素工具鋼SK2～SK5は
水焼入れを必要とするため熱処理変形を生じやすいので，すきまゲ
ージのように形状が簡単で小寸法の場合のみに用いられる．使用頻
度の高いゲージやブロックゲージとしては耐摩耗性の優れた
SKD1，SKD11などの高合金工具鋼が用いられる．
ゲージ鋼は普通，焼入れ，焼戻しによってHRC60～65の硬さ
に調整して用いられる．ゲージ鋼の熱処理に当たっては熱処理変形
に留意するほか，時間経過に伴う寸法変化の原因となる残留オース
テナイトの分解，不安定なマルテンサイトの安定化をねらってサブ
ゼロ処理や繰返し焼戻しが行われる．
2・8・6磨き特殊帯鋼
磨き特殊帯鋼は熱間圧延した特殊鋼の帯鋼に冷間圧延と焼なまし
を加えて製造した極薄，長尺の帯鋼で，表面が美しく寸法精度が高
いのが特徴である．JIS規格には炭素鋼，低合金機械構造用鋼，ば

まい，ゲージ，バンドソー，ハクソー，チェーン部品などに用いら
れている．これらのほかに高速度工具鋼，軸受鋼，快削鋼などの磨
き帯鋼も製造されている．磨き特殊帯鋼は打抜き，曲げ，絞りなど
のプレス加工を受けることが多いので，これらの加工に適するよう
