# 2・8・4 熱間金型用工具鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・8 工具鋼

熱間金型用としては,高温強さ,靱性,耐摩耗性,耐高温酸化性などに優れていることに加えて,急熱急冷による熱応力の繰返しで表面にき裂を発生するいわゆるヒートチェック(heat checking)に対する抵抗性が高いことが要求される.特に耐ヒートチェック性を配慮して炭素含有量の少ない鋼が選ばれる.冷間金型と比較して熱間金型は寸法が大きく,形状も複雑なものが多いので,焼入れの際には割れや変形を生じさせないように配慮するとともに,十分な性能を得るための冷却条件を選択する必要がある.また熱間鍛造時には金型表面が被鍛造材からの熱により870~970K まで上昇することが知られており,耐熱性を保持させるために,通常,770~870K 前後の高温焼戻しが採用される.SKD 4,5はW 量が高く,高温強さ,耐摩耗性は大であるが,ヒートチェックには弱い点がある.SKD 6,SKD 61はヒートチェックを改善した鋼種で,熱間プレス型,アルミダイカスト型,押出工具などに用いられている.SKD 62は,耐摩耗性を改善した鋼であり,プレス型や押出型などに広く利用されている.4%のCoを含有したSKD 8は銅の押出型やダイカスト型に用いられる.これらのSKD 系鋼は,焼入れ温度からの空冷によっても硬化が可能であり熱処理による変形が少なく,用途に応じてHRC43~50程度の硬さに熱処理して用いられる.Ni-Cr-Mo系のSKT 3およびSKT 4鋼は高温硬さや焼戻し軟化抵抗は前述のSKD 系鋼に比べて劣るが,靱性が高いので衝撃荷重のかかる鍛造型や大型で割れの危険性のあるプレス型などに用いられる.SKT 系鋼はHS 56前後の硬さで用いられるので,通常焼入れ,焼戻し後に型彫りを行って用いられる.金型の表面処理は,窒化処理や浸硫窒化処理が主体であり,温間,熱間プレス型やアルミダイカスト型,アルミ押出工具などに適用されている.熱間金型用鋼は冷間金型用鋼に比べてその使用環境が複雑で厳しいことから,より多くの研究と新種開発が行われている.2―71
