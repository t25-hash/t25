# 2・8・5 ゲージ用鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・8 工具鋼

ゲージとして最も必要な性質は使用中に寸法,形状の変化を生じないことである.そのためにゲージとしては経年変化,耐摩耗性,耐さび性,熱膨張係数などを考慮して使用材料が選択される.また形状の複雑なゲージやブロックゲージでは熱処理変形が小さいこと,研削やラップ仕上げなどの加工性がよいことも要求される.合金工具鋼SKS 3およびSKS 31は焼入れ性や耐摩耗性がよく,ゲージ用鋼として最も多く用いられる.炭素工具鋼SK 2~SK 5は水焼入れを必要とするため熱処理変形を生じやすいので,すきまゲージのように形状が簡単で小寸法の場合のみに用いられる.使用頻度の高いゲージやブロックゲージとしては耐摩耗性の優れたSKD 1,SKD 11などの高合金工具鋼が用いられる.ゲージ鋼は普通,焼入れ,焼戻しによってHRC60~65の硬さに調整して用いられる.ゲージ鋼の熱処理に当たっては熱処理変形に留意するほか,時間経過に伴う寸法変化の原因となる残留オーステナイトの分解,不安定なマルテンサイトの安定化をねらってサブゼロ処理や繰返し焼戻しが行われる.
