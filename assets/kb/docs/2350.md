# 2・8・6 磨き特殊帯鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・8 工具鋼

磨き特殊帯鋼は熱間圧延した特殊鋼の帯鋼に冷間圧延と焼なましを加えて製造した極薄,長尺の帯鋼で,表面が美しく寸法精度が高いのが特徴である.JIS 規格には炭素鋼,低合金機械構造用鋼,ばね鋼などから製造された磨き帯鋼が定められ,かみそり替刃,ぜんまい,ゲージ,バンドソー,ハクソー,チェーン部品などに用いられている.これらのほかに高速度工具鋼,軸受鋼,快削鋼などの磨き帯鋼も製造されている.磨き特殊帯鋼は打抜き,曲げ,絞りなどのプレス加工を受けることが多いので,これらの加工に適するように冷間圧延のものと焼なましのものが供給される.磨き特殊帯鋼は成形加工後,一般には焼入れ,焼戻しにより用途に応じた強靱性が付与される.特殊な場合にはマルクエンチ(marquenching)やオーステンパ(austempering)が行われる.
