# 2・9・1 鋳鉄(cast iron)の概要

> **章**: 第2章 鉄鋼材料
> **節**: 2・9 鋳鉄

a. 鋳鉄の歴史鋳鉄鋳物は,西暦540年ごろ仏教とともに日本に伝来したといわれている.その後,鋳物生産の集落を形成しつつ,日本各地に広がった.強度が問題になるようになったのは産業革命以後であり,日本では1900年ごろから工業製品が作られ始めている.b. 鋳鉄の分類と名称鋳鉄材料の特性は黒鉛の形状に大きく依存するので,黒鉛形状から次のように分類できる.ねずみ鋳鉄(gray iron),球状黒鉛鋳鉄(spheroidal graphite iron),C/V 黒鉛鋳鉄(compacted graphite iron),可鍛鋳鉄(malleable iron)である.また,合金成分を変化させ,特殊な機能をもたせたものを合金鋳鉄と呼び,この中には各種黒鉛形状のものが含まれる.その他,冷却速度を大きくして炭化物を積極的に析出させ,硬度をもたせるチルド鋳鉄もある.
