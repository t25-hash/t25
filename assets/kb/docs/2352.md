# 2・9・2 ねずみ鋳鉄(gray cast iron)

> **章**: 第2章 鉄鋼材料
> **節**: 2・9 鋳鉄

a. 化学組成一般的な金属材料と異なり、鋳鉄は特殊なものを除き組成が規格化されていない。主成分はCとSiであり、一般にC2.0~4.0%,Si1.0~3.0%程度のものが多く使われる。その他Mnは0.1~1.0%程度、またP0.03~0.4%,S 0.01~0.15%などを不純物として含有している。組成については、受渡し当事者間で決定してよいことになっている。b. 規格JIS 機械的性質、物理的性質ねずみ鋳鉄は成分範囲が広く、また種々の組織を形成し、これに伴って諸性質が変化する。鋳鉄の機械的・物理的諸性質のうち特性とされているものを挙げれば次の諸点である。(1) 圧縮強さは引張強さの3~4倍と見てよい。(2) 弾性係数は鋼より低いが、アルミニウム合金より高い。(3) 熱伝導率は鋼よりも高く、特に強度の低い級になるほど増加する。(4) 耐摩耗性一般に良好であって軸受、歯車、シリンダ、ピストンリング、工作機械ヘッド、ブレーキシューなど耐摩耗部品としての用途が広い。これは(1)黒鉛が潤滑剤的な役割をすること、(2)適度の硬さであること、(3)弾性係数があまり高くないこと、(4)熱伝導がよいので摩擦熱を速やかに逃がし、焼付きを防ぐことなどの特性をもつことによるものと考えられている。シリンダライナ類ではCr,Cu,Moを単独あるいは組合わせて少量添加し、基地を硬くすることが有効である。大型のディーゼル機関用ライナではC,Siを低めにして遊離セメンタイトを少し晶出させたり、V,B を少量添加し硬いステダイトを均等に分散させたりして耐摩耗性を向上させるようにしている。Pの含有量を高めると硬いステダイトが網目状に分散し、高荷重によく耐え、工作機械部品に適する。過酷な摩擦を受けるブレーキ類は、C,Siを多めにして黒鉛を大きく晶出させ熱伝導性を高めるようにしたものが好まれる。g. 減衰能振動を受けるとそのエネルギーを速やかに吸収する。つまり減衰能(damping
