# 2・2・5 機械構造用炭素鋼鋼材

> **章**: 第２章 鉄鋼材料
> **節**: 2・2 炭素鋼

a.種類と用途 機械構造用炭素鋼鋼材（S-C材）は特殊鋼の
中の合金鋼に分類されているが，炭素，ケイ素，マンガン，リン，
硫黄しか含まないため，2･2･5項で解説する．JISに規定されてい
る機械構造用炭素鋼（carbon steel for machine structure use）
.焼なまし（annealing，JIS記号：HA) 焼なましには
目的に応じて，完全焼なまし（HAF），球状化焼なまし（HAS），
応力除去焼なまし（HAR）などがある．完全焼なましは鋼材を
Ac点以上に加熱したのち，炉冷または徐冷し，塑性加工を容易に
するものである．球状化焼なましは，鋼中のセメンタイト
（FeC）を1m程度の均一な球状にすることによって，冷間での
塑性加工性，切削性を向上させるために行われる．A点よりもわ
ずかに高い温度の1023～1033Kに長時間加熱したのち，徐冷す
る．応力除去焼なましは，冷間加工や溶接によって生じた残留応力
の除去を目的として行う．加熱温度は773～793Kで，完全焼なま
しより低温で行う．図2･15 に焼ならし，または焼なまし後の機
械的性質を示す．
.焼入れ焼戻し（quench temper，JIS記号：HQ，HT)
焼入れ焼戻しは，適切な強度と靱性を付与する熱処理方法で，Ac
点以上に加熱し，完全にオーステナイト化したのち，急速冷却する
のが焼入れ（HQ）で，そのあとA点以下の適当な温度に加熱し
て徐冷するのが焼戻し（HT）である．焼入れ温度は表2･19に示
す範囲で定める．炭素鋼は質量効果が大きいため，有効熱処理径は
約20mm以下で，これ以上の直径または対辺距離をもつ鋼材では
完全な焼入れ組織を得ることは不可能で，オーステナイト結晶粒の
境界，または粒内へき開面に沿ってフェライトが析出する．このフ
ェライトの析出は粘り強さを低下させるが，合金鋼の場合に比べて
その程度は軽微である．
焼割れを防止するには，マルテンパ，またはオーステンパを行う
か，あるいは階段焼入れを行う．マルテンパでは473～573Kまで
急冷して，鋼材が一様にこの温度になるまで保持したのち徐々に，
しかも全断面同時にマルテンサイト変態を完全に終了せしめる．オ
ーステンパではS曲線で，フェライトおよびパーライトへの変態
とマルテンサイト変態との中間の約623Kに保持した熱浴あるい
は恒温加熱炉中に急冷，その温度に十分保持してベイナイト変態を
完了せしめたのち徐冷する．階段焼入れは焼入れ液中に焼入れした
のち約573～673Kに冷却したとき焼入れ液より引上げ，別に用意
した約473Kの焼戻し炉中に装入するか，そのまま空気中に放置
して徐冷する方法である．しかしこれらの方法は直径または対辺距
離約10mm以下の小物においてのみ有効な焼入れが可能で，それ
より大型の場合は質量効果のため十分な焼入れ効果は望めない．
焼戻しには，453～473Kで行う低温焼戻し，673～923Kで行う
高温焼戻しがある．刃物等の硬度の高いものや耐摩耗性を要求され
るものに対しては，低温焼戻しを行う．高周波焼入れ品，浸炭焼入
れ品に対しても低温焼戻しを行う．高温焼戻しは，ある程度の強度
と靱性を求められる場合に行う．なお，573K付近での焼戻しは，
脆性が現れるため避けなければならない．図2･16にS30C鋼の焼
戻し温度による機械的性質の変化を示す．一般に，823～923Kに
焼戻すと，強靱な状態が得られる．

性は低炭素鋼より悪く，難しくなるが，焼入れ焼戻しを行うことに

鋼の回転曲げ疲労限度が試験温度によってどう変化するのかを
0.2%耐力，引張強さとともに示した．
被削性（machinability）を良好にするためには，球状化焼なま
しがよい．硬さはHB200以下が普通であることから焼なまし状態
で荒削りを施しておき，その後焼入れ焼戻しを行って仕上加工を行
う．これは焼入れ時の焼割れを防ぐためにも有効である．炭素鋼は
質量効果も大であることから，荒削りを行ったほうがよい．
.高炭素鋼（C＞0.5%) 炭素量が0.45%を超すと次第に
もろさを増し，焼割れを生じる可能性が著しく増大する．そのため
用途は，表面硬さ（surface hardness ）が要求されるキー，ピン，
軸類などに限定される．粘り強さに対するよりも，炭素量を高める
ことによって強さを増し，焼入れ，焼戻しによりさらに高い硬さを
