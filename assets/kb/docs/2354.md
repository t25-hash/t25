# 2・9・4 C/V 黒鉛鋳鉄(compacted vermicular graphite cast

> **章**: 第2章 鉄鋼材料
> **節**: 2・9 鋳鉄

iron)a. 概要球状黒鉛鋳鉄が発明されたころは,黒鉛が球形であることが重要視され,形の崩れた黒鉛は敬遠されていた.エバン(Evan),ドーソン(Dowson)らは,球状化阻害元素のTiを含有したMg 系球状化剤を用い,黒鉛の球状化率を低下させた鋳鉄を作製し特性を調べた.球状黒鉛鋳鉄に比較して伸びは低下するが,強度はあまり変わらず,熱伝導性はよく,鋳造性もよい材料が得られた.これが,C/V 黒鉛鋳鉄であり,自動車部品,鋳鋼の鋳型などに用いられている.C/V とは,compacted vermicularの略であり,それぞれ,ぎっしり詰まった,芋虫状の,という意味である.b. 製造法C/V 化処理法としては,C/V 化剤を置注ぎ法を主体とした方法で添加するのが一般的である.C/V 化剤は基本的には,次の3種存在する.Mg を減らした球状化剤,球状化剤に球状化阻害元素(Tiなど)を加えたもの,球状化剤のMg を弱い球状化作用のある元素(Ce,Caなど)で置換えたものである.c. 機械的性質引張強さは400MPaほどであり伸びは数%である.フェライト化しやすいので,400MPa前後の場合フェライト地であり,Cuなどの添加によりパーライト地にすると500MPa以上となる.図2・42はねずみ鋳鉄,球状黒鉛鋳鉄とC/V 黒鉛鋳鉄の伸びと強度バランスの分布を示したものである.球状や片状とは明らかに異なる特性をもつことがわかる.球状化率の変化により片状黒鉛鋳鉄に近い性質から球状黒鉛鋳鉄に近い性質に変化させることができる.
