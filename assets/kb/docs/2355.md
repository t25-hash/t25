# 2・3・1 合金鋼の範囲

> **章**: 第２章 鉄鋼材料
> **節**: 2・3 構造用合金鋼

鋼（あるいは炭素鋼）は，合金元素の適切な添加と熱処理の組合
せによって，各種の材料特性が著しく向上する．2･3節では，構造
用合金鋼（structural alloy steel ）として，特殊用途鋼（special
purpose steel ，2･4～2･8節参照）と鋳造品（2･9，2･10節参照）
を除く合金鋼の鍛造材と圧延材（素材および素形材，2･1節参照）
を対象とする．
まず化学組成に基づいて，日本工業規格（JIS）を中心とする鋼
種を掲げ，次に構造用合金鋼の選択の考え方を述べた後，さらに代
表的な用途別の製品によって，材質的特性と使用時の特性を述べ
る．なお便宜上，構造用合金鋼の中に，各種クラッド鋼も含める．
