# 2・3・4 構造用合金鋼の合金設計

> **章**: 第２章 鉄鋼材料
> **節**: 2・3 構造用合金鋼

構造用合金鋼の合金組成の決定には，関連する合金元素（alloy
element ）あるいは不純成分元素（impurity element ）の数（20以
上），塑性加工と熱処理の履歴，製品の肉厚，溶接性などの関数と
して，多面的な検討を要し，次のように進める．
a.JISに基づく材料選定 JISの各号（表2･29～2･31）にお
ける規定は，製品の肉厚（あるいは直径）が大きくない場合には，
原則として技術的な基準として採用できる．
ただし，まず所要の機械的性質をJISの規定内でさらに範囲を
絞り（例えば上限側，下限側など），次に化学組成の範囲および成
分元素間の組合せを次に述べるb項のように絞って材料仕様を決
定する．

b.合金元素の効果 構造用合金鋼の性能は合金元素の複合的
な影響（combined eﬀect of alloy elements ）を受ける．その詳細
は専門書に譲り，以下にその大要を述べる．
(１)C C量の増加とともに引張強さを増す．またCr ，
Mo，V，(Fe）と炭化物を作り強度を増す．しかし，C量の増加は
溶接性を損ねる．
(２)Si 鋼の脱酸剤として有力であり，またフェライト相を
強くする．Siは鋼塊の偏析と熱間加工材の方向性を増す．Si ＜
0.1 %の溶鋼を真空炭素脱酸（VCD:vacuum carbon deoxyda-
tion）すると，鋼の靱性と等方性が著しく改善される．
(３)Mn 鋼の脱酸剤として有力であり，また焼入れ性を増
す．廉価であるので有用な元素．
(４)P，S 靱性を害し，また偏析を起こしやすい．しかし，
精錬法の進歩（炉内での2回スラグ法，炉外でのとりべ精錬法）に
より，これらを0.01%以下にすることは容易である．Pは中性子
照射脆化を増大する．
(５) Ni 焼入れ性を増し，低温靱性を向上させる．
(６) Cr 焼入れ性，焼戻し抵抗性を大きくし，耐食性を増
す．しかしNi - Cr鋼は焼戻し脆化が著しい．
(７)Mo 焼入れ性を増し，またCrと安定な複炭化物を作
って焼戻し抵抗性を増大する．Ni -Cr鋼の焼戻し脆化を防ぐ．構
造用鋼にとって必須の元素であるが，高価である．
(８) V 0.3%までの添加では焼入れ性と焼戻し抵抗性を増
し，結晶粒を微細にする．Vは溶接性を損ねる．
(９)Cu 強度を増すが，熱間加工性，溶接性を損ねるので，
構造用鋼では積極的には添加しない．Cuは中性子照射脆化を増大
する．
