# 2・10・4 耐摩耗(wear resistance)用鋳鋼

> **章**: 第2章 鉄鋼材料
> **節**: 2・10 節ではa,bの分類に沿って示す。

耐摩耗用鋳鋼の規格はJIS などでは少ないため、各メーカが独自の規格を採用していることが多い。耐摩耗用鋳鋼を分類すると、マルテンサイト系(2・10・1・b項の低合金系)とオーステナイト系(高マンガン系)とに分けられる。両者の耐摩耗鋳鋼の用途を表2・56に示す。オーステナイト系耐摩耗用鋳鋼である高マンガン鋼鋳鋼(highmanganese steel casting)のJIS 規格は高マンガン鋼鋳鋼品(JISG 5131)でSCMnH 1~SCMnH 21まで5種がある。この鋳鋼は11~14%Mn,0.90~1.3%Coを含む鋼種で、他の鋼種には見られない極めて大きな靱性と著しい加工硬化能をもっている。この特徴を発揮させるため、通常鋳物は1320K に加熱して炭化物をオーステナイト中へ固溶させ、均一オーステナイト組織にして使用する。この場合の焼入れは、通常の鋼のように硬化を目的としたものではなく、靱性の付与を目的とした焼入れであるため、水靱処理と呼ばれている。この状態では硬さがHB 200程度と軟らかいが、表面に衝撃的荷重を受けるとHB 550程度の硬い加工硬化層が形成され、優れた耐摩耗性を発揮する。しかし砂地における引っかき摩耗のように加工硬化を生じない場合には耐摩耗性を発揮しない。したがって、つねに表面に打撃を受けるような摩耗部品、例えばレールクロッシングや各種粉砕機などに使用されている。また高マンガン鋳鋼は加工硬化性を有する高靱性な材料であるため、機械加工はかなり土木機械破砕機その他マルテンサイト系(低合金系)トラックリング、スプロケット、爪ライナ、打撃子、ハンマ[しゅんせつ(浚渫)]ポンプ、インペラ、カッタオーステナイト系(高マンガン系)トラックリンク、タンブラ、爪マントル、コンケーブ、歯板、ハンマ、コークカッタ、ロール(鉄道)マンガンクロッシング、床板(その他)ドラグチェーン、エプロン、非磁性耐摩耗部品表2・57 低温高圧力用鋳鋼品(JIS G 5152:1991)種類記号鋼種最低使用温度1種SCPL1炭素鋼228K(-45°C)11種SCPL11 0.5%モリブデン鋼213K(-60°C)21種SCPL21 2.5%ニッケル鋼198K(-75°C)31種SCPL31 3.5%ニッケル鋼173K(-100°C)図2・45 焼結品密度と抗折力の関係の例困難であり、超硬工具や砥石を用いて適正な加工条件を選んで行う必要がある。
