# 2・4・2 マルテンサイト系ステンレス鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・4 ステンレス鋼およびステンレス合金

12～13%Crを含むSUS410（0.1C），SUS420（0.2C）が代表
的なマルテンサイト系ステンレス鋼（martensitic stainless steel ）
である．本鋼は空気中で不動態化する最低限のCrを含み，空気中
ではさびにくいが，水溶液中の耐食性は十分ではない．しかし，焼
入れ性は良好で，焼入れ時にマルテンサイト組織を得，焼戻しによ
り靱性をもたせることで優れた機械的性質を得ることができる．焼
入れは1220～1270Kにてオーステナイト化後油冷，焼戻しは920
～970Kで行うのが普通である．このような熱処理により，
SUS410は引張強さ540MPa以上，伸び25 %以上，SUS420では
引張強さ640MPa以上，伸び20%以上が得られる．本鋼は刃物と
して用いられるが，刃物のように硬さと耐摩耗性が要求される場合
には423～473K程度の低温で焼戻す．また，SUS440系は0.60～
1.20 %のCを含む17Crの鋼で，ステンレス鋼中最も硬く，刃物，
外科用器具，ゲージ類，軸受などに用いられる．SUS410はバル
ブシート，ポンプシャフトなど機械部品，刃物，SUS420はばね，
ボルト，プラスチックの型，刃物などに用いられる．
