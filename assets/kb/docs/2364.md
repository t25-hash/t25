# 2・11・2 金属粉末および合金粉末

> **章**: 第2章 鉄鋼材料
> **節**: 2・11 焼結合金

粉末冶金で用いられる原料粉末には、固体の状態で機械的に粉砕されたもの、溶湯よりガス噴霧(gas atomization)、水噴霧(water atomization)、遠心噴霧(centrifugal atomization)によって粉化されたもの、還元、熱分解、湿式電解などの化学的な方法によって製造されたものなどがある。機械的に粉砕する場合には大概の原材料が対象となる。もろい原材料から粉化されたものは角ばった粉となり、軟らかい材料あるいは強靱な材料から作られた粉は片状または皿状となるものが多い。溶湯からガス噴霧で作られる粉末は凝固温度区間の大きい合金では球状、凝固温度区間の狭いものでは丸味を帯びた不規則形状である。フィルタ用の原料粉末として用いられる。水噴霧で作られた粉末は表面に微細粒が付着したような不規則形状となるが、圧粉体としたときにこの凹凸がからみ合うことで圧粉密度が低くとも圧粉体の強度を維持できる。ガス噴霧で作られる粉末には青銅、鉛青銅、ケルメット、高速度鋼、超合金などがある。水噴霧のものは鉄、各種合金鋼、ステンレス鋼、ステライト、銅、銅合金などがある。酸化物のガス還元によって作られる粉末にはFe,Ni,Co,Cu,W,Moなどがあり、Fe粉は多孔質で成形性が良好である。化合物の熱分解によるものは鉄カーボニル、ニッケルカーボニルから作られる純鉄、純ニッケル粉がよく知られており、Feは球状、Niは金平糖状でいずれも磁性材料用として使用される。水素化物の熱分解によって製造される粉末としてはTi,Th,Zr,U,Taなどがある。この方法はチタン合金粉などの製造にも用いられる。加熱分解した焼結ケーキは機械的に粉砕されるので粉末は不規則形状のものとなる。水溶液電解において陰極板上に析出した金属から得られる粉末には二とおりのものがある。その一つは、もろい板状に析出した金属を採取して機械的に粉砕したもので粉砕電解粉と称し、Fe,Cr粉などがその例である。他は陰極板上に直接粉末として析出させるもので電解粉と称し、Cu,Ag 粉などがその例である。直接粉末として析出された粒子は樹枝状を呈している。水溶液電解によるものでは上記のほかNi,Mn,Pb,Zn,Snなどがある。電解で得られる鉄粉は純度が高く圧縮性に優れているので、高密度の焼結部品や磁性材料用に適するが高価である。溶融塩電解によって得られる粉末にはTa,Nb,V,Ti,Zr,U,Thがある。a. distribution)が表せる。粒子の形態とともに粒度および粒度分布は混合、圧粉、焼結の各過程に影響を与える。微粉末は一般に見掛け密度が低いうえに流動性が悪いので生産性、作業性が低下するが、焼結性が優れているという特性がある。実用に当たっては成形に適した粒度分布となるよう粒度調整が行われている。見掛け密度(apparent density)は378±5K で乾燥した試料粉末を漏斗を通して付属のコップ中に自然落下の状態で充てんし、充てんされた粉末質量をコップの容積で割って求める。見掛け密度が低い粉末は粒子形状が不規則であり、圧粉時の圧縮比が大となって高密度の圧粉体が得にくくなる。流動度(flow rate)は一定量(50g)の粉末が細孔から流出するのに要する時間で表される。この値の小さいものほど流動性がよく押型内への充てん能率が向上するほか、圧粉時に粒子の滑りが起きやすく、均一かつ高密度の圧粉体が得られる。また、この値の小さいものほど見掛け密度は大きくなる傾向がある。圧縮性(compressibility)のよい粉末とは見掛け密度が大きくかつ可塑性の大きいもので、これを用いると高密度の圧粉体が得られる。成形性(compactibility)は圧粉体の強度と関連があり、粒子形状が不規則なものほど成形性がよくなる傾向がある。焼結性は焼結時の寸法変化と焼結体の強度の双方で評価される。一般に粒度が細かく粒子表面が清浄で凹凸の大きい粒子は焼結性がよいといわれる。
