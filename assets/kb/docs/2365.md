# 2・4・3 オーステナイト系ステンレス鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・4 ステンレス鋼およびステンレス合金

オーステナイト系ステンレス鋼（austenitic stainless steel ）は
Cr添加による不動態化促進効果と，Ni添加による鉄の活性域にお
ける腐食速度減少効果を巧みに組合わせた耐食合金鋼であり，多く
の腐食環境において優れた耐食性を示す．オーステナイト系ステン
レス鋼として，最も典型的な鋼種はSUS304（18Cr -8Ni -0.06
C）である．
SUS304鋼のシリーズとしてSUS316（18Cr -12Ni -2.5Mo），
SUS321（18Cr -10Ni -Ti ），SUS347（18Cr -10Ni -Nb）がある．
SUS316はMoを添加して耐孔食性，耐酸化性を向上させ，
SUS321およびSUS347はそれぞれTiおよびNbをC量に対応
（Ti / C≧5，Nb/C≧10）して添加し，鋼中のCrを主成分とする炭
化物MCの代わりにTiC，NbCを生成させ，熱処理や溶接後の
冷却過程で起こるMCの粒界析出による粒界隣接部でのCr欠乏
層形成に起因する粒界腐食（intergranular corrosion）を防止する
よう合金設計したステンレス鋼である．粒界腐食に対してはCを
減少することも効果があるのでSUS304のCを0.03%以下にした
304Lも実用されている．この場合，Cの減少に伴う強度低下を補
うため，Nを少量（0.05 %程度）添加することがある．
オーステナイト系ステンレス鋼は，耐食性とともに加工性，成形
性，溶接性にも優れ，高温強度も高く，低温でも靱性が良好なた
め，各種の化学装置に広く利用され，また建築，食品製造設備，熱
交換器，厨房用品，自動車部品，車両，原子炉関係，発電関係など
広範囲に使用されている．
この系の鋼は耐力は低いが加工硬化率が大きく，加工硬化により
高強度を得ることができる．SUS301はNi量が6～8%で304と
比較してやや低いが，Cを0.1%前後の高めにし，加工硬化を利用
して強度を出し，電車の車体などに用いられている．Seを0.15 %
以上添加して切削性を改良した鋼種（SUS303Se ）も実用されて
いる．また，Niを4～5%にして節約し，代わりにMnを6～
9%，Nを0.2%程度添加した高Mnオーステナイト系ステンレス
鋼（200シリーズ）もあるが，耐食性は300シリーズの鋼種より劣
る．高Mn鋼は，非磁性鋼（non-magnetic steel ）として，核融合
炉部材，磁気浮上高速車用部材などの候補材として注目され，実
際，使用されているものもある ．
オーステナイト系ステンレス鋼の最大の欠点は応力腐食割れを起
こしやすいことである．応力腐食割れを起こす環境としては，塩化
物環境，硫化物（ポリチオン酸）環境，高温高圧水，カ性アルカリ
などがおもなものであるが，これまでに環境因子と材料因子との関
連がかなり解明され，高Ni材，高Si材（SUSXM15J1系 )，
不純物制御材など目的，環境に応じて優れた鋼材が開発されてい
る．
ステンレス鋼の耐酸化性，耐孔食性を300シリーズの鋼より大幅
に改善した特殊なステンレス鋼が，その用途に応じ各国，各社にお
いて種々製造されている．これらの鋼は，Mo，Cuを同時に含有
し，Ni量を高めたものが多い．アメリカで20合金（Alloy20）と
呼ばれている20Cr -30Ni -2Mo-3Cuはその代表的なものである．
20合金ではCを0.02%以下とし，またNbを添加して粒界腐食を
抑制している．
