# 3・1・1 アルミニウムの物理的および化学的性質

> **章**: 第3章 非鉄金属材料
> **節**: 3・1 アルミニウムおよびその合金

アルミニウム(aluminum,化学記号Al)はアルミナを氷晶石とフッ化アルミニウムとともに溶融塩電解して還元したもので,電解地金の純度は99.5~99.9%である.これをさらに三層電解法,あるいは部分凝固法(偏析法)による固相と液相で濃度差を利用する方法を用いて精製すると99.99~99.999%の高純度アルミニウムが得られる.おもな不純物はFe,Siで,その他にCu,Zn,Ti,Mn,V などが含まれる.アルミニウムのおもな物理的性質を表3・1に示す.アルミニウムの物理的性質は不純物(Fe,Si,Ti,Cu,Mn,V など)の含有量および加工状態によって異なる.アルミニウムはCuに次いで高い導電率および熱伝導率を示すが,0.1%以下の微量のCr,Mn,V,Zr,Tiの含有によって著しく低下する.さらに,加工によっても低下する.また,反射率が優れており,比重はFeの3分の1である.アルミニウムは活性な金属で,酸素の存在により表面に緻密な酸化被膜(AlO)を形成する.この酸化被膜の形成により内部酸化が抑制され,耐食性が良い.この酸化被膜は内部が緻密なバリア被膜と,その上に水和されたポーラス酸化層からなり,硬く,電気的に絶縁性である.人工的にアルミナ被膜を製品表面に作り,耐食性を与える陽極酸化被膜処理(anodic oxidation treatment,アルマイト処理)が広く行われている.これはアルミニウムを陽極としてシュウ酸,硫酸またはクロム酸溶液中で電解して酸化被膜を作り(この被膜は多孔質である),さらに高圧の加熱水蒸気中で封孔処理するもので,電気絶縁性が高く,耐摩耗性が高く,耐食性に優れた被膜が形成される.アルミニウムは,無機酸の塩酸,硫酸,リン酸,希硝酸にはかなりの速度で侵され,水銀および塩類にも侵される.一方,濃硝酸,クロム酸には侵されず,硫黄や硫化物には極めて安定である.ほとんどの有機酸には室温では安定であるが,ギ酸とシュウ酸にはかなり侵される.アルカリ溶液はアルミナ被膜を溶解するので激しく侵食する.そのため水とCa(OH)を含むコンクリートや石灰モルタルと接触していると侵食されるので注意を要する.pH 4~9の中性水溶液においては,塩素イオンを含まない低濃度溶液ではアルミナ水和物の被膜が成長し,塩素イオンを数ppm以上含むと孔食が生じる.塩化物溶液にクロム酸を添加すると反応が抑えられる.アル表3・1 アルミニウムの物理的性質性質高純度(99.996%)普通純度(99.5%)原子番号―原子量26.98―結晶構造面心立方格子面心立方格子格子定数(nm,293K)0.404940.404密度(Mgm,293K)2.6982.71融点K933.2~923沸点K―溶解潜熱(10Jkg,10Jmol)396840比熱[JkgK,293K]凝固収縮vol.%―6.6線膨張係数(10K,293~373K)24.623.5(10K,293~373K)25.425.6熱伝導率[WmK,293~673K]導電率(% IACS,標準銅に対して)64.959(軟質)57(硬質)電気抵抗率(10Ωm,293K)2.692.92(軟質)3.02(硬質)抵抗の温度係数10K4.24.0体積磁化率10Hm9.909.88反射率(%,λ=250;500;2000nm)―87;90;972―85ミニウムの腐食はすきまにおいて起こりやすく,孔食が発生するので設計上注意を要する.多くの金属はアルミニウムの孔食電位よりも高い電位を示すため,水溶液中でそれらと接触すると孔食が発生する.その促進度合は電位差のみでは決まらない.促進度が大きいものはHg,Cu,Fe(Clが多い場合),中程度のものはNi,Sn,Ti,Fe(Clが少ない場合),促進しないものはZn,Cdである.アルミニウムより低い電位で孔食を促進しない金属によって保護する陰極防食法(cathodic protection)がある.
