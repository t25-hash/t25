# 3・1・1 アルミニウムの物理的および化学的性質

> **章**: 第3章 非鉄金属材料
> **節**: 3・1 アルミニウムおよびその合金

アルミニウム（aluminum，化学記号Al）はアルミナを氷晶石
とフッ化アルミニウムとともに溶融塩電解して還元したもので，電
解地金の純度は99.5～99.9 %である．これをさらに三層電解法，
あるいは部分凝固法（偏析法）による固相と液相で濃度差を利用す
る方法を用いて精製すると99.99～99.999 %の高純度アルミニウム
が得られる．おもな不純物はFe，Siで，その他にCu，Zn，Ti，
Mn，Vなどが含まれる．アルミニウムのおもな物理的性質を表3･
1 に示す．アルミニウムの物理的性質は不純物（Fe，Si，Ti，
Cu，Mn，Vなど）の含有量および加工状態によって異なる．アル
ミニウムはCuに次いで高い導電率および熱伝導率を示すが，
0.1%以下の微量のCr，Mn，V，Zr，Tiの含有によって著しく低
下する．さらに，加工によっても低下する．また，反射率が優れて
おり，比重はFeの3分の1である．
アルミニウムは活性な金属で，酸素の存在により表面に緻密な酸
化被膜（AlO）を形成する．この酸化被膜の形成により内部酸化
が抑制され，耐食性が良い．この酸化被膜は内部が緻密なバリア被
膜と，その上に水和されたポーラス酸化層からなり，硬く，電気的
に絶縁性である．人工的にアルミナ被膜を製品表面に作り，耐食性
を与える陽極酸化被膜処理（anodic oxidation treatment，アルマ
イト処理）が広く行われている．これはアルミニウムを陽極として
シュウ酸，硫酸またはクロム酸溶液中で電解して酸化被膜を作り
（この被膜は多孔質である)，さらに高圧の加熱水蒸気中で封孔処理
するもので，電気絶縁性が高く，耐摩耗性が高く，耐食性に優れた
被膜が形成される．
アルミニウムは，無機酸の塩酸，硫酸，リン酸，希硝酸にはかな
りの速度で侵され，水銀および塩類にも侵される．一方，濃硝酸，
クロム酸には侵されず，硫黄や硫化物には極めて安定である．ほと
んどの有機酸には室温では安定であるが，ギ酸とシュウ酸にはかな
り侵される．アルカリ溶液はアルミナ被膜を溶解するので激しく侵
食する．そのため水とCa(OH)を含むコンクリートや石灰モルタ
ルと接触していると侵食されるので注意を要する．pH4～9の中性
水溶液においては，塩素イオンを含まない低濃度溶液ではアルミナ
水和物の被膜が成長し，塩素イオンを数ppm以上含むと孔食が生
じる．塩化物溶液にクロム酸を添加すると反応が抑えられる．アル

（cathodic protection）がある．
3・1・2アルミニウムおよびその合金の分類
工業的に用いられているアルミニウムとその合金には，展伸用合
金と鋳物・ダイカスト用合金があり，さらにその中で非熱処理型合
金（non- heat - treatable alloys）と熱処理型合金（heat alloys）に大別される．ここでいう熱処理とは，時効硬化を伴う処
理をいう．各合金は添加される合金成分から表3･2 のように分類
される．これらの合金の基本二元系は，Al -Cu，Al - Si，Al よびAl - Znであり，実用材料はこれらを組合わせた三元，四元系
ならびにその他の元素を少量加えた多元系からなる．
相の分散によって強度や硬さを向上させており，展伸材では圧延，
押出し，引抜き，鍛造などの塑性加工による加工硬化によって強度

い，一般に焼入れという．焼入れした材料を，冷間加工して強度を
