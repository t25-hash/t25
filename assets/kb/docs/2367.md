# 3・1・2 アルミニウムおよびその合金の分類

> **章**: 第3章 非鉄金属材料
> **節**: 3・1 アルミニウムおよびその合金

工業的に用いられているアルミニウムとその合金には、展伸用合金と鋳物・ダイカスト用合金があり、さらにその中で非熱処理型合金(non-heat-treatable alloys)と熱処理型合金(heat-treatablealloys)に大別される。ここでいう熱処理とは、時効硬化を伴う処理をいう。各合金は添加される合金成分から表3・2のように分類される。これらの合金の基本二元系は、Al-Cu,Al-Si,Al-Mg およびAl-Znであり、実用材料はこれらを組合わせた三元、四元系ならびにその他の元素を少量加えた多元系からなる。非熱処理型合金は、鋳造材では固溶強化および結晶微細化と第二相の分散によって強度や硬さを向上させており、展伸材では圧延、押出し、引抜き、鍛造などの塑性加工による加工硬化によって強度の向上が図られる。一方、熱処理型合金は時効析出によって所定の強度を得るもので、溶体化、焼入れ、焼戻しなどの熱処理が施される。また、熱処理と加工が組合わされる場合もある。このように、加工や熱処理によって所定の性質を得ることを調質といい、その種類を質別という。アルミニウムおよびその合金は、合金組成と質別の組合せにより広い範囲の強度、靱性、その他の特性をもつ。材料の質別記号(temper designation)とその内容は次のとおりである。非熱処理型合金は、製造のまま、すなわち圧延、押出しや鋳造のままの状態(質別記号F)あるいは圧延、抽伸などの冷間加工による加工硬化によって所定の強度を得た状態で使われる(質別記号H)。図3・1に示すように、加工率の増加とともに引張強さは上昇し、伸びは減少する。これを適度に加熱して焼なますと強度は減少し、伸びは上昇する。このように加工のみによる強度の調整の場合、H 4、焼入れ後比較的低い温度で加熱して人工的に析出強化させる焼戻し(人工時効、artificial aging)を行って最高強度を得るものをT 6、過時効によって安定化したものをT 7、製造時に急冷されてある程度過飽和状態が得られ、焼戻しのみで析出強化を図るものをT 5処理という。また、焼入れ後人工時効前に冷間加工するT 8や人工時効後冷間加工を施すT
