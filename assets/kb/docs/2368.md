# 3・1・3 アルミニウムおよびその合金の性質

> **章**: 第3章 非鉄金属材料
> **節**: 3・1 アルミニウムおよびその合金

アルミニウムの一般的特徴として次のことが挙げられる。密度が2.70Mg/mでマグネシウムに次いで軽く、Fe,Cuのおよそ3分の1で実用構造用金属中で最も軽い金属に属する。種々の合金元素を添加し、加工あるいは熱処理を施して大きな比強度をもつ合金が得られる。表面に薄い緻密な酸化被膜を生じて保護し、耐食性が良好である。展延性に富み、加工性、成形性に優れ、特に押出し加工によって複雑な断面の形状が容易に得られる。ろう付けやガス、アルゴンおよび抵抗溶接が容易にできる。低温における脆性破壊や靱性低下がなく、温度の低下とともに強度、伸びともに上昇する。Cuに次いで電気および熱の伝導率が高く、白色光線およびふく射熱の反射が大きい。また、ほとんど磁性を帯びない。a. 量を減じ、Mn,Crを添加した合金で、海水、工業地帯の環境下に強く、溶接構造用として船舶、車両、化学プラントに用いられる。5000系の合金は特に低温における静的強度、疲労強度が高く、5083は低温構造材として使用される。(6) Al-Mg-Si系合金(Al-Mg-Si alloys,6000系)Al-Mg-Si系合金は、0.5~1.0%程度のMg
