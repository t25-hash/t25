# 2・4・6 ステンレス合金

> **章**: 第２章 鉄鋼材料
> **節**: 2・4 ステンレス鋼およびステンレス合金

ステンレス鋼は，塩酸やアルカリに対する耐食性が十分でない．
これを改善するために高Cr ，Mo，Ni ，Cuとした高合金をステン
レス合金という．ステンレス合金として，最もよく実用に供されて
いる合金はNi量を増加したNi基の耐食合金である．Ni自体は酸
化性化学薬品に対しては耐食性はあまりないが，中性およびアルカ
リ性環境では耐食性がよく，カ性アルカリ，食品などを扱う装置材
料として多く使われている．
Ni基耐食合金はNi -Cu，Ni - Cr ，Ni -Cr - Fe，Ni - Cr - Mo合金
に大別される ．モネル合金（monel alloy）はNi約66 %，Cu
約30%の合金で，その耐食性はNi ，Cuよりよい．特に海水に対
して非常に耐食性があり，かつキャビテーションに耐えるので，プ
ロペラ，プロペラシャフト，ポンプシャフトなどに広く使用されて
いる．Ni -Cr合金は高温腐食の一種であるバナジウムアタックに
対し耐える材料として開発された合金で，35Cr -65Ni ，50Cr -50
Ni ，60Cr -40Niの3種類がある．Ni - Cr -Fe合金の代表的なもの
はインコネル600（Inconel600）と呼ばれる0.05C-16Cr -8Feを
主成分とするNi合金である．NCF-1としてJISに規格化されて
いる．同合金はCrを含み，Ni量も多いので，酸化性，還元性の
両環境において優れた耐食性を有している．高温でSを含む雰囲
気では硫化されるが，窒化に対しては優れた抵抗性を示す．また軽
水炉（原子炉）の各種配管，シュラウド（shroud）などに用いら
れている．インコネル600は耐熱合金としての用途も多い．Ni -
Cr -Mo合金としてはハステロイ（Hastelloy）と呼ばれる一連の合
金が開発されており，耐食合金としてはハステロイC（0.06C-2.5
Co-15Cr -16Mo-4W-6Fe-Ni ）が代表的なものである．同合金は
耐酸合金特に塩素，塩化物を含む環境，ギ酸，酢酸等に対して優れ
た耐食性を示す．またハステロイCのCおよびSiを低減（C：
0.02%，Si ：0.05 %）し，さらに耐食性を高めたハステロイC-276
がある．この合金は，最近，油井の深井戸化に伴い，高温の強度と
表2･40 JIS（G4311：1991）による代表的な耐熱鋼とその主要用途
