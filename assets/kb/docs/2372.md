# 3・3・2 工業用純ニッケルと合金の種類

> **章**: 第3章 非鉄金属材料
> **節**: 3・3 ニッケルおよびその合金

a. 工業用ニッケル工業用ニッケルは塑性加工性に富む。耐食性が良く、非酸化性の環境下や海水によく耐え、酸性およびアルカリ性塩類水溶液に対しても耐食性が良い。特に、カ性ソーダに対して耐食性を示すため、カ性ソーダ濃縮材として使用される。実用的には、電子管用ニッケルがJIS のH 4501,H 4502,H 4511,H 4522に規定されている。b. ニッケル合金の種類ニッケルは合金科学的見地から二つの特徴を有する。一つは第一次固溶体を形成する合金元素の種類が数多く、それによって形成される固溶体が高濃度域まで広がっている場合が多いことである。固溶体領域が広いことは、物理的性質や機械的性質を連続的に幅広く変化させることを可能にさせている。そして、各種合金元素の添加による流動応力の変化が実測されており、Ta,Nb,W,Moなどで固溶強化率が高いとされている。もう一つの特徴は、各種金属間化合物(intermetallic compounds)表3・11 純ニッケルの物理的性質比重Mg/m8.902融点K磁気変態点K導電率%25.2固有抵抗Ωcm6.84線膨張係数K13.3×10熱伝導度W/(mK)格子定数Å3.5167引張強さ0.2%耐力伸び絞りα-β型合金Ti-6Al-4VSTA1070MPa1000MPa13%34%Ti-6Al-6V-2SnSTATi-6Al-2Sn-4Zr-6MoSTATi-10V-2Fe-3AlSTA2―9 2材料学・工業材料を形成することである。後述のように、NiAl,NiTi,NiNbなどは耐熱合金中の析出強化相として極めて重要な役割を果たしている。これらの金属間化合物は温度の上昇とともに強度が上昇するという、通常の金属・合金には見られない特徴を有しており、これが耐熱超合金(heat resistant superalloys)の強度を担う基本強化機構となっている。そして、これら二つの特徴が、多種多様なニッケル基耐熱合金の開発を促している。ニッケル基合金における各添加元素の役割を総括して表3・12に示す。これらの役割を組合わせながら新しい合金を開発していく場合、合金元素の種類と量を増やすとσ相と呼ばれるTCP相(topologically closed packedphase)が高温・長時間使用中に出現し、機械的性質に悪影響を及ぼすことが多いことに注意を要する。最近では、これらの要因を整理し、目的に応じた合金を開発しようとする合金設計の手法がとられている。ニッケル合金は合金成分の差異によって、Ni-Cu系、Ni-Mo系、Ni-Fe系、および、Ni-Cr系合金などがあり、いずれも、耐熱・耐食性に優れた合金群である。いくつかの代表例を機械的性質とともに表3・13に示した。Ni-Cu系合金は全率固溶体を形成し、合金の諸性質は組成の変化によって徐々に変わる。60~70%のニッケルを含むNi-Cu合金はモネルメタルと呼ばれており(表3・13)、耐食性に優れていることから、化学工業用ポンプや蒸気弁などに用いられている。ニッケルにモリブデン(Mo)を添加すると、塩酸に対して極めて耐食性の良い合金となり、Cr,Fe,Cuなどを添加したハステロイ系統の合金として実用化されている(表3・13)。ニッケルにクロム(Cr)、鉄(Fe)、アルミニウム(Al)などを添加すると耐酸化・耐熱性が向上することから、航空機用・産業用ガスタービンの超高温材料としてなくてはならない材料となっている(表3・14)。ほかに強磁性合金のパーマロイ(Ni-Fe系)がある。
