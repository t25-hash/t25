# 2・8・1 切削工具鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・8 工具鋼

切削用としては，切削性の観点から高い硬さが要求されると同時
に，切削時の衝撃によって刃こぼれが生じないような適度な靱性も
要求される．したがって，これらの兼合いから用途に適合した炭素
含有量の鋼が選ばれる．
かみそり，やすり，バイトなどには，通常，炭素含有量が1.1～
1.5 %の鋼が多く用いられている．炭素工具鋼ではSK140
（SK1），SK120（SK2）などがある．さらに3～5%のWの添加
により耐摩耗性，焼戻し軟化抵抗を高めたSKS11がある．
ハクソー，ドリル，ねじ切りダイスなどのやや靱性を要する用途
には炭素含有量が1.0～1.2 %の鋼が用いられる．SK105（SK3）
にCr ，Wなどを添加したSKS2およびSKS21は，特殊炭化物の
形成によって耐摩耗性を向上させた鋼で，タップ，ドリルなどとし
て多く使われている．Wを2～2.5%とやや多く含むSKS7は耐摩
耗性に優れているのでハクソーとして用いられている．
木工用丸のこや帯のこなどには炭素含有量が0.7～0.9%の
SK85（SK5），SK75（SK6）などが用いられる．SKS5および
SKS51はNi ，Crなどを添加してさらに靱性を向上させた鋼であ
る．
高速切削では工具に大きな機械的摩耗を受けるうえ，摩擦熱によ
って刃先温度が著しく上昇するので，高温での硬さが必要となる．
このような用途には高速度工具鋼が用いられる．JIS SKH55など
のCoを含む鋼は焼戻し軟化抵抗と高温硬さが高いので高速切削に
適する．
