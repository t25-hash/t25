# 3・5・1 マグネシウムの特徴

> **章**: 第3章 非鉄金属材料
> **節**: 3・5 マグネシウムおよびその合金

マグネシウム(magnesium、化学記号Mg)は、1808年、ダービィ(H. Davy)により発見された銀白色の金属であり、比重が1.74と鉄の約4.5分の1、アルミニウムの約3分の2で、実用金属材料中では最も軽い。マグネシウム合金は、その軽量性、比強度(specific strength)、放熱性、振動減衰能、切削性、耐くぼみ性、寸法安定性など優れた特性を有し、近年高純度合金の製造やSFガス雰囲気による溶解技術の確立から、マグネシウム合金のもつ腐食に弱いという問題は解決され、ディジタルビデオカメラ、ノートパソコン、ミニディスクプレーヤー、携帯電話など電子機器、携帯機器などの筐体、スポーツ用品などの生活分野に用途が広がりつつある。さらに、マグネシウムは、地球環境保全の立場から、プラスチックあるいは鉄、アルミニウムに代わる軽量、かつリサイクル性に優れた、環境に優しい材料として注目され、欧米においては自動車部品への利用が活発化している。遅ればせながら、今後、国内でも自動車への応用が予測されている。a. 資源マグネシウムの地殻構成クラーク数は8番であり、その地殻表層部での存在比重量は1.93mass%である。マグネシウムの原鉱石はドロマイト(MgCO・CaCO)、マグネサイト(MgCO)などであり、製錬法には、これらの鉱石を高温で焼成してマグネシア(MgO)とした後、これを塩化物(MgCl)にして溶融塩電解する電解法と、マグネシアを高温でおもにケイ素を用いて還元する熱還元法がある。また、マグネシウムは、海水中に、金属元素としてナトリウムに次いで多く含有され、その量は約1300mg/Lである。このようにマグネシウムは、資源的には無尽蔵な金属材料であるといえる。b. 物理的性質マグネシウムは、原子番号12、原子量24.32、ちゅう密六方晶(=0.32095nm,=0.52107nm)で、密度は、1.74Mgmである。沸点は1378 K、融点923K で、融点における蒸気圧は367Paと高い。熱伝導率は155WmK、導電率は39%IACS で、アルミニウムの約3分の2である。溶融潜熱は、372×10Jkg、比熱1013JkgK(293K)、凝固収縮率4.2vol.%である。したがって、少ない熱量で溶解することができ、また、凝固潜熱も小さく早く固まるという特徴がある。線膨張係数は26.0×10K(293~373K)で、鉄の2倍以上で大きい。構造材料としては最も軽く、縦弾性係数は、44.7GPa、剛性率は17.3GPaである。ポアソン比は0.291である。マグネシウムは、結晶構造が図3・11に示すようにちゅう密六方晶であるため、室温での塑性加工がやや難しいという欠点がある。滑り系を図3・12に示す。滑り系は、底面滑り系、柱面滑り系およびすい面滑り系があるが、室温では底面滑りは容易に活動するものの、他の滑り系は高温にならないと活動しにくいうえ、加工硬化率も大きくなるので、マグネシウムの冷間加工は、実用的には図3・11 化学的、電気的性質マグネシウムは一般に、耐食性に劣る金属といわれているが、大気中では鉄より優れている。マグネシウム合金は塩素イオン、酸、塩類の存在する雰囲気では、耐食性は良くないが、大部分のアルカリおよび多くの有機化合物に耐える。マグネシウムは電気化学的に非常に卑な金属であり、実用金属の中では、表3・22に示すように低い標準電極電位を示す。したがって、マグネシウムは他の金属を接触させておくと、マグネシウムが先に腐食するので、マグネシウム合金を異種金属と接触して使用する場合は注意が必要である。他方、マグネシウムが先に腐食するという特徴を利用して、鋼鉄製船舶、土中に埋設する石油パイプラインなどにあらかじめマグネシウムを付属し、船舶、パイプなど本体の腐食を防止する、いわゆる防食用アノードとして使われる。また、マグネシウムは活性な金属であることから、Ti,Zr,V,Be,B などの金属製錬において還元材として利用されている。特に、Ti製錬のクロール法においてチタン塩化物の還元に多量のマグネシウムが使用されている。最近では、半導体や太陽電池用の高純度シリコンの製造において、モノシランの製造にケイ化マグネシウムが用いられている。d. その他の特性(1) 減衰能(damping capacity)マグネシウムは、振動吸収性が高いため、繰返し運動、断続運動する部品に使用すると、それらの振動を吸収し、機械寿命を長くする。図3・13に種々の金属材料の減衰能と強度の関係を示す。マグネシウムは軽量でかつ振動吸収性の優れた材料であることから、チェーンソーなど携帯工具のハウジングとして多く用いられている。(2)
