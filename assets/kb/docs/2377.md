# 2・8・3 冷間金型用工具鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・8 工具鋼

冷間金型用としては硬くて耐摩耗性に優れることが要求されるの
で，比較的炭素含有量が多く，Cr ，Mo，W，Vなどの炭化物形成
能の高い合金元素を多く含む高合金鋼がよく用いられる．通常，冷
間金型用工具鋼は焼なまし状態で機械加工を施し，その後の熱処理
によって硬化させたのちは研磨仕上げなどのわずかな仕上加工のみ
で使用するため，熱処理時の変形が少ないことも要求される．
炭素工具鋼では炭素含有量が0.9～1.1%のSK105（SK3），
SK95（SK4）などが抜型として使用されるが，水焼入れを要する
ので焼入れひずみが大きく，また寸法の大きな型では焼入れ硬化が
十分でない．Mn，Crを添加して焼入れ性を改善した鋼として
SKS93～SKS95がある．最近のプレス金型には油冷でも高硬度が
得られるSKS93が広く採用されている．さらにWの添加により
耐摩耗性と焼入れ性を向上させた鋼としてSKS3およびSKS31
がある．
1.4～2.2 %のC，11～15%のCrを含有したSKD1，SKD11は，
炭化物量が多く耐摩耗性に優れており，プレス型などをはじめとし
て多目的に使用される．1%のC，5 %のCrを含むSKD12は多少
耐摩耗性を犠牲にしても靱性を要する場合に適している．これらの
鋼種はいずれも焼入れ温度からの空冷によって硬化できるので，焼
入れひずみはSKS3よりもさらに小さい．冷間鍛造用型鋼として
SKH51などの高速度工具鋼も使用される．
冷間金型用の硬さは一般にHRC58以上を必要とし，二次硬化
を示さないSKSなどの低合金鋼では450K付近の低温焼戻しが施
されるのに対し，SKD11などの高合金鋼では低温焼戻しと770K
以上の高温焼戻しが用途に応じて使い分けられる．低温焼戻しと比
較して，工具表面の温度上昇に耐えるには高温焼戻しが有利であ
り，しかも熱負荷に対する相安定性と焼入れ残留応力の低減を図る
ことができる．最近では，SKD11に対し，晶出炭化物を微細に分
散して靱性を向上させるとともに，高温焼戻しでHRC62が得ら
れる8Cr系鋼の利用が定着化している ．
大型の抜型，曲型などで全体を焼入れ，焼戻しすることが困難な
ものでは炎焼入れ（flame hardening）などによって局所的に所要
部分を硬化することが行われる．また，冷間金型の用途で性能を上
げる表面改質としては硬質被膜処理が挙げられ，CVD（化学的蒸
着），PVD（物理的蒸着），TRD（熱反応析出・拡散）処理などが
用いられている．
