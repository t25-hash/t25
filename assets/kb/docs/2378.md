# 3・5・2 マグネシウム合金の種類と性質

> **章**: 第3章 非鉄金属材料
> **節**: 3・23 に示す。マグネシウムの所用切削動力は軟鋼の約6分の1,

わが国では、マグネシウムおよびマグネシウム合金(magne-sium alloys)が鋳物、板、棒などの形で構造材料として実際に使用されている量はアルミニウム合金に比べ極めて少なく、しかもマグネシウムの全消費量の10%以下にすぎない。国内で、マグネシウムの需要で最も多いのは、アルミニウム合金への合金元素としての添加である。なお、世界的には欧米をはじめとして、自動車部品へのマグネシウムダイカスト材の応用が進み、全需要量の約3分の1から4分の1に達している。構造材用マグネシウム合金は、それぞれの用途に応じて、Al,Znなど添加元素を加えて使用する。構造材として使用されるマグネシウム合金は結晶微細化方法の違いにより、Mg-Al系およびMg-Zr系の2種類に大別される。マグネシウム合金は、アルミニウム合金と同様に、鋳造(砂型、金型、ダイカスト)、押出し、圧延、鍛造などの成形が可能である。a. 鋳造用マグネシウム合金(magnesium casting alloys)代表的な鋳造用マグネシウム合金の記号、標準化学組成、機械的性質の例を表3・24に示す。マグネシウム合金の呼称は、その単純さと組成により速やかに理解できるということで、市場ではASTM 規格による合金名がよく用いられている。これは、最初の1文字あるいは2文字が主要合金元素を示し、その後にそれらの合金元素の質量パーセント(mass%)が続くというものである。例えば、AZ 91D はアルミニウムを9%、亜鉛を1%含むことを表し、最後のD は測定された順番(アルファベット順、D は4番めを示す)標準化学組成mass%質別引張強さMPa耐力MPa伸び%―FT 4T 5T 6FT 4T 5T 6―――FT 4T 5T 6注:はダイカスト用合金である。機械的性質は鋳物材では最低値、ダイカスト材では標準的な値を示している。RE は希土類元素である。AM 60B,AM 50A,AZ 91D,AZ 91E,AS 41B,WE 43A,WE54A 合金は、耐食性の向上を目的として不純物Fe,Ni,Cuを極力少なくした高純度合金である。2―102材料学・工業材料表3・25 展伸用マグネシウム合金の化学組成と機械的性質区分種類ASTMJISAZ 31CMP 1Al3.0,Zn1.0,Mn0.15OH 14圧延板材―MP 4Zn1.2,Zr0.6H 112―MP 5Zn3.3,Zr0.6H 112―MP 7Al2.0,Zn1.0,Mn0.05OAZ 31C MT 1Al3.0,Zn1.0,Mn0.15H 112継目なし管AZ 61AMT 2Al6.4,Zn1.0,Mn0.28H 112―MT 4Zn1.2,Zr0.6H 112AZ 31CMB 1Al3.0,Zn1.0,Mn0.15H 112AZ 61AMB 2Al6.4,Zn1.0,Mn0.28H 112AZ 80AMB 3Al8.4,Zn0.6,Mn0.25H 112押出し棒材―MB 4Zn1.2,Zr0.6H 112―MB 5Zn3.3,Zr0.6H 112ZK 60AMB 6Zn5.5,Zr0.6H 112T 5AZ 31CMS 1Al3.0,Zn1.0,Mn0.15H 112AZ 61AMS 2Al6.4,Zn1.0,Mn0.28H 112AZ 80AMS 3Al8.4,Zn0.6,Mn0.25H 112押出し型材―MS 4Zn1.2,Zr0.6H 112―MS 5Zn3.3,Zr0.6H 112ZK 60AMS 6Zn5.5,Zr0.6H 112T 5を示している。実用マグネシウム合金中の合金元素を示す文字は以下のとおりである。A:Al,C:Cu,E:希土類元素(RE)、H:Th,K:Zr,L:Li,M:Mn,Q:Ag,S:Si,W:Y,Z:Zn.鋳造用合金の添加元素としては、強度と鋳造性を得るための基本元素Al,Zn、結晶微細化のためのZr、耐熱性をもたせる希土類元素がある。. Mg-Al系Mg-Al系合金はマグネシウム合金の基礎となる合金で、一般に機械的性質は良いが、耐食性は悪い。Mg-Al系では、850~900°Cに10~30min保持し、注湯温度まで速く冷却し、鋳造するという過熱処理(super heating)を施し、結晶微細化を図る。(1) Mg-Al系合金(M g-Al alloys,AM 100A など)Mg-Al系は、α-Mg 固溶体とβ-MgAl化合物の共晶系で、最大固溶度12.7mass%(710K)である。Mg-10%Al合金であるAM 100A はα単相組成であるが、非平衡β相が多量に晶出するので十分な溶体化を必要とする。近年、エアバック装着用ステアリングホイール心材に靱性を高めたAl含有量の少ないAM 50A,AM 60B ダイカスト材が多く使用されるようになってきた。(2) Mg-Al-Zn系合金(Mg-Al-Zn alloys,AZ 91D など)Alを9%,Znを1%含むAZ 91系は機械的性質や鋳造性などバランスの取れた代表的なマグネシウム合金でダイカスト合金として最も多く使用されている。特に、AZ 91D 合金は、高純度耐食性合金として、自動車、コンピュータ、携帯電話、各種ハウジング類、スポーツ用品など多岐にわたって使用されている。さらに、最近注目されているチクソモールディング(プラスチック成形機である射出成形機を用いてマグネシウム合金を成形加工するプロセス)でも現在のところ、このAZ 91D 合金がおもに用いられている。Mg-Al-Zn系では、AlとZnの含有量によって異なるが、α固溶体とβ-MgAl化合物が共晶として晶出する。Zn含有量が多くなると、さらにMg(Al, Zn)化合物も晶出するようになる。溶体化処理・時効により、粒内への均一析出に加え、粒界反応析出が起こり、粒界における偏析が著しい。.
