# 3・6・1 亜鉛

> **章**: 第3章 非鉄金属材料
> **節**: 3・6 亜鉛およびその合金

亜鉛(zinc、化学記号Zn)は電解法と蒸留法によって作られる。日本での生産分類は、電気亜鉛67%(電解法)、蒸留亜鉛27%(蒸留法)、精留亜鉛6%(蒸留法)である。純度と成分を表3・26に示す。主要鉱石は閃亜鉛鉱(ZnS)、菱亜鉛鉱(ZnCO)である。亜鉛は電極電位が卑であるため主用途は鉄鋼の防食(protec-tion)すなわち亜鉛めっき鋼板用(50%)であり、また銅合金添加金属(黄銅など)(15%)、亜鉛ダイカスト合金(10%)、亜鉛板(2%)等が挙げられる。亜鉛めっきには、電気亜鉛めっきと溶融亜鉛めっきがある。溶融亜鉛めっきでは蒸留亜鉛およびこれに特殊元素を添加したものが用いられる。また、電極電位が卑であることを利用した用途としてはほかに電池用材料、電気防食用陽極、亜鉛溶表3・26 亜鉛地金規格(JIS H 2107)種類化学成分(wt.%)ZnPbFeCdSn最純亜鉛地金>99.995 <0.003<0.002<0.002<0.001特種亜鉛地金>99.99<0.007<0.005<0.004―普通亜鉛地金>99.97<0.002<0.01<0.005―蒸留亜鉛地金特種>99.6<0.3<0.02<0.1―蒸留亜鉛地金1種>98.5<1.3<0.025<0.4―蒸留亜鉛地金2種>98.0<1.8<0.1<0.5―注:防食用亜鉛は最純亜鉛地金でもFe<0.001を用いる。表3・27 亜鉛の物性値物性低融点金属(元素記号)亜鉛(Zn)原子番号(原子量)30(65.38)結晶構造六方ちゅう密格子格子定数nm=0.26649=0.49468原子半径nm0.133,0.148密度kg/m7.13融点K692.65沸点K融解潜熱kJ/mol7.12比熱kJ/(kgK)0.394熱膨張係数10/K軸15.0軸61.5熱伝導率W/(mK)119.5比抵抗10ΩK5.92ヤング率kN/mm104.5せん断弾性係数kN/mm41.9引張強さN/mm98~127破断伸び%50~65硬さHv合金種類記号亜鉛合金ダイカスト1種ZDC13.9~4.30.75~1.25 0.03~0.06<0.003<0.075<0.002<0.001残部亜鉛合金ダイカスト2種ZDC23.9~4.3<0.250.03~0.06<0.003<0.075<0.002<0.001残部金型用合金3種―3.9~4.32.85~3.35 0.03~0.06<0.003<0.02<0.001<0.001残部2―103射の亜鉛線などがある。また、亜鉛粉が電池、および塗料の顔料、製錬・染料工業の還元剤として使用される。亜鉛めっき鋼板の亜鉛、亜鉛加工品はリサイクルされ、溶融めっきやダイカストで発生するドロスは金属分を絞ったあと酸化亜鉛の原料となる。亜鉛のおもな物性値を表3・27に示す。亜鉛は原子番号30、原子量65.39、周期律表IIb族である。六方ちゅう密格子で格子定数は=0.26649nm,=0.49468nmで/=1.856で軸方向に伸びた結晶となっている。そのため亜鉛の物性は結晶方位により大きく異なり、例えば熱膨張係数は軸方向15.0×10Kに対し軸方向61.5×10Kと4倍の差がある。ちゅう密六方格子であるために室温での滑り面が少なく圧延など塑性加工性に劣る。しかし、再結晶温度は346K であり熱間での塑性加工は容易でかつ鋳造組織を再結晶組織に変えることができる。また、273K 以下では衝撃強度が低下する。融点は692.6K、沸点は1179K で溶解潜熱は7.12kJ/molで金属としては比較的低い温度で大気中で容易に溶解できる。
