# 2・8・5 ゲージ用鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・8 工具鋼

ゲージとして最も必要な性質は使用中に寸法，形状の変化を生じ
ないことである．そのためにゲージとしては経年変化，耐摩耗性，
耐さび性，熱膨張係数などを考慮して使用材料が選択される．また
形状の複雑なゲージやブロックゲージでは熱処理変形が小さいこ
と，研削やラップ仕上げなどの加工性がよいことも要求される．
合金工具鋼SKS3およびSKS31は焼入れ性や耐摩耗性がよく，
ゲージ用鋼として最も多く用いられる．炭素工具鋼SK2～SK5は
水焼入れを必要とするため熱処理変形を生じやすいので，すきまゲ
ージのように形状が簡単で小寸法の場合のみに用いられる．使用頻
度の高いゲージやブロックゲージとしては耐摩耗性の優れた
SKD1，SKD11などの高合金工具鋼が用いられる．
ゲージ鋼は普通，焼入れ，焼戻しによってHRC60～65の硬さ
に調整して用いられる．ゲージ鋼の熱処理に当たっては熱処理変形
に留意するほか，時間経過に伴う寸法変化の原因となる残留オース
テナイトの分解，不安定なマルテンサイトの安定化をねらってサブ
ゼロ処理や繰返し焼戻しが行われる．
