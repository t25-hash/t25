# 3・6・2 亜鉛合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・6 亜鉛およびその合金

亜鉛は化学的に活性で多くの元素と化合物を生成するため、合金元素として最大固溶量1at.%以上のものは金・銀・銅・水銀・アルミニウム・カドミウム等に限定される。その中で重要なのは亜鉛-アルミニウム系である。アルミニウムは、亜鉛合金の融点を下げるとともに強度を付与する。また、耐食性を向上する。a. ダイカスト用亜鉛合金ダイカスト用亜鉛合金(zincalloys)の成分を表3・28に示す。アルミニウム4wt.%添加を基本に、マグネシウムが0.04wt.%添加される。亜鉛-アルミニウム系合金では、亜鉛製錬時の微量不純物元素でもある鉛・スズ・カドミウムの存在により粒間腐食と呼ばれる特有の腐食が発生する。マグネシウムは、この粒間腐食を抑制するために添加される。以上の亜鉛-アルミニウム-マグネシウム系合金に加えて、さらに高強度化・耐摩耗性等を向上させるために銅が1wt.%程度添加された合金も広く使用されている。本系合金で留意しなければならないのは経時寸法変化である。アルミニウムの添加により短時間に寸法収縮が生じる。マグネシウムは、この収縮反応を遅らせる。一方、1wt.%を超える銅の添加は長時間において寸法膨張が生じる。また、日本では上述の2合金が使い分けられているが、欧米では銅0.7wt.%系の合金が一般的である。亜鉛ダイカスト品は、鋳肌が美しく型転写性に優れている。また、肉厚1mm以下の薄肉鋳造も可能である。表3・29に亜鉛ダイカスト品の物性値を示す。また、バラエティーに富んだ表面処理が可能で装飾品用途も多い。おもな用途とし表3・29亜鉛合金ダイカストの物性値物性合金種類亜鉛合金ダイカスト1種亜鉛合金ダイカスト2種融点K653.8654.1密度kg/m6.76.6比熱kJ/(kgK)0.4190.419熱膨張係数10/K27.427.4熱伝導率W/(mK)108.9113.1引張強さN/mm破断伸び%硬さHv95~10585~95表3・28亜鉛合金ダイカスト地金規格(JIS 型用亜鉛合金亜鉛合金はダイカストのほかに砂型鋳造にて試作用プレス型や試作および少量量産用樹脂成形型として用いられる。合金成分としては、アルミニウム4wt.%-銅3wt.%-マグネシウム0.04wt.%系が多く用いられる。プレス型用としてはおもに自動車のボデー部品で鋳物重量としては数tから十数tに及ぶ。樹脂成形型用も同様に自動車バンパやラジエータグリルなど大物が多い。最近、小型のブロックにして切削で樹脂成形用型を製作することも行われるようになってきた。ここで用いられる合金はさらに強度を向上させたアルミニウム7wt.%-銅4.5wt.%-マグネシウム0.02wt.%が用いられる。c. その他の合金電気防食の陽極材としては純亜鉛に微量のアルミニウムを添加したものが用いられる。亜鉛の加工材としては亜鉛板がある。亜鉛板は連続鋳造・圧延で製造されるが、古くは印刷用、また乾電池用、ヒューズ用、メダル用、屋根板用などがある。乾電池用としては鉛を微量添加したものが一般的であるが、鉛規制から鉛フリー合金(lead-free alloys)が研究されている。新しい用途としてはアルミニウム22wt.%系の超塑性合金板がある。本合金は超塑性成形、例えばブロー成形が可能であり、少量生産の医療用機器カバー等に用いられる。その他、電気防食用亜鉛合金を箔にしてテープにしたもの、粉からペーストにしたもの、電池用としてアトマイズ粉にしたものがある。
