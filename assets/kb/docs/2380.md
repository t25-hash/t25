# 2・8・6 磨き特殊帯鋼

> **章**: 第２章 鉄鋼材料
> **節**: 2・8 工具鋼

磨き特殊帯鋼は熱間圧延した特殊鋼の帯鋼に冷間圧延と焼なまし
を加えて製造した極薄，長尺の帯鋼で，表面が美しく寸法精度が高
いのが特徴である．JIS規格には炭素鋼，低合金機械構造用鋼，ば
ね鋼などから製造された磨き帯鋼が定められ，かみそり替刃，ぜん
まい，ゲージ，バンドソー，ハクソー，チェーン部品などに用いら
れている．これらのほかに高速度工具鋼，軸受鋼，快削鋼などの磨
き帯鋼も製造されている．磨き特殊帯鋼は打抜き，曲げ，絞りなど
のプレス加工を受けることが多いので，これらの加工に適するよう
