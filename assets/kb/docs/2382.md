# 3・8・1 鉛およびその合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

a. 鉛鉛(lead、化学記号Pb)は、溶鉱炉製錬にて粗鉛が製造され、電解(湿式)または溶離(乾式)で精製される。主要鉱石は方鉛鉱PbS で亜鉛鉱とともに算出する。純度と成分を表3・43に示す。鉛は比重が大きく、軟らかく展延性に優れる。また、放射線遮へい力が高く耐食性にも優れる。したがって建築物耐震性のための基礎マットレスや、遮音および放射線遮へい板、またヨーロッパでは屋根板、古くは水道管に使用されてきた。また、容器の内張りとしての用途もあり、一部鉛めっきも行われている。化学的特性を利用して蓄電池電極として、また酸化物など化合物は塗料に用いられる。鉛の主用途は、70%が蓄電池用、14%が無機薬品、3%が鉛管および板である。一方、鉛は経口摂取・吸引により体内に蓄積し発育不全・精神不安定などの原因になると指摘され、世界的に鉛の使用規制が進められている。そうした流れの中で蓄電池の鉛については全量回収しリサイクルするシステムが確立している。また、鉛合金であるはんだ合金の鉛フリー化の開発が進められている(3・8・2・c項参照)。鉛のおもな物性値を表3・44に示す。鉛は原子番号82、原子量207.2、周期律表IVb族である。面心立方格子で格子定数は0.49505nm、原子半径0.176nm、比重11.34、融点600.5K で塑性変形性に極めて優れる。b. 鉛合金鉛合金(lead alloys)は、Pb-Sb合金およびPb-Sn合金が一般的である。Pb-Sb合金は硬鉛と呼ばれ、Sb1wt.%合金は古くは水道用鉛管に用いられ、化学工業用装置のライニング・ポンプ・バルブ・コック・継手などにも用いられてきた。しかし、上述したように鉛使用の規制のためほとんど用いられなくなってきている。本系合金の用途としては蓄電池の陽極グリットに用いられる。Sb2~8wt.%の合金が一般的であるが、電池特性改善のためSn,Se,AsあるいはCaが添加されることもある。Pb-Sb-Sn合金は軸受合金(ホワイトメタル、white metal)に用いられる。また、活字合金にも用いられていた。軸受合金としてはSb12~17wt.%,Sn量は1~70wt.%である。活字合金は、鋳造性から三元共晶(Sb11wt.%-Sn4wt.%)が標準であるが、鉛の凝固収縮の小さいことを利用した用途である。Pb-Sn合金は、はんだ合金として用いられる。
