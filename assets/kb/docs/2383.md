# 3・8・2 スズおよびその合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

a.スズ スズ（stannum，化学記号Sn）は，二段還元溶錬法
にて荒スズが製造され，電解（湿式）または溶離（乾式）で精製さ
れる．主要鉱石はスズ石（SnO）であるが産地は局在する．純度
と成分を表3･45に示す．スズは，白色の金属で，曲げると音を立
てる（スズ鳴り）．これは結晶系の滑りに起因した現象である．鉛
と同じく融点が低く，柔軟で耐食性も優れている．また，自然環境
下では安定で人体には無害とされている．そのため鉄板へスズをめ
っきして食料品・食器関係に広く用いられる．電気部品へのめっき
はろう付け性を増す（消費量の40%）．めっき法としてはどぶ漬け
と電気めっきがある．さらに，スズにはスズペストと呼ばれる低温
での崩壊現象がある．これは，低温（286.2K以下）で共有結合性
の強いダイヤモンド構造に変態［α-Sn（灰スズ）］するからであ
る．寒冷な地域では，このα-Snがコロニー状に生成し，その際の
変態応力にて崩壊するのである．この変態は，Bi，Sb，Pbなどの
不純物により抑制される．
スズのおもな物性値を表3･44に示す．スズは，原子番号50，原
子量118.7，周期律表Ⅳb族である．正方晶で格子定数は ＝
0.5831， ＝0.3182，比重7.3，融点505.1Kである．上述のとお
り，低温でダイヤモンド構造となる．また，表面にSnOを生成し
耐食性に優れるが，酸やアルカリには侵食される．
はんだ合金の近年の課題としては二つ挙げられる．一つは，電子
機器の高密度実装に伴い，基板からの放熱の増大，温度上昇が著し
くなっていることである．それによりはんだ合金にも使用温度条件
下での強度特性や機器のon/ oﬀに伴う熱疲労への強度特性が求め
られるようになっている．これらの要望に対応する合金としては，
Sn-3.5wt.%Ag系やSn-5wt.%Sb系が挙げられる．二つめは，
はんだ合金に鉛が含まれ，電子機器の廃棄場所からの鉛流出という
環境汚染が指摘されるに至っていることである．これに対して，鉛
フリーはんだ合金としてSn- Zn系，Sn-Ag系，および，これらへ
のCu，In，Bi等の添加が検討されている．現状，Sn-3.5wt.%
Ag-0.7wt.%Cu系やSn-3.5wt.%Ag-0.5wt.%Cu-8wt.%In系が
広く実用されるようになっている（表3･46）．
