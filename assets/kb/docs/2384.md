# 3・8・3 タングステン

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

タングステン(tungsten,化学記号W)は,体心立方構造であり,融点3660K(金属中最高),密度19.3×10kg/m,ヤング率345GPa,熱伝導率178W/(mK),熱膨張率4.5×10Kである.表3・47,3・48には電気抵抗や諸性質を示した.タングステンワイヤは,耐熱性が必要な電球(electric lamp),蛍光灯のフィラメント,電気炉ヒータ(heater),高強度が必要なつり線,精密機械器具などに用いられる.酸化トリウムを添加したトリエーテッドタングステンや希土類元素添加タングステンは,高エミッション性(電子放射特性)と耐熱性が必要な放電灯や水銀灯の電極(electrode),溶接用電極などに,タングステン重合金(W-Fe-Ni-Cuなど)はおもり,飛行機のバランスウェート,放射線遮へい(radiation shield)用部品などに,W-Cu合金は耐アーク性が必要な電気接点,高熱伝導性と低熱膨張率が必要な半導体パッケージのヒートシンクなどにそれぞれ用いられる.タングステンの炭化物を利用した超硬合金(cemented car-bide)は,硬くてしかも高強度である.WC-Co合金は,切削工具,鉱山用工具,ダイス,プラグ,鍛造用工具,切断工具などの耐摩耗部品に,WC-TiC-TaC-Coは切削工具に,WC(-Cr)-Niは,非強磁性が必要な磁性粉末プレス用金型や耐食性が必要なシールリングなどとしてそれぞれ用いられる.また,タングステンは,高速度工具鋼,ステライトなどの硬質,耐熱合金の炭化物形成元素としても重要である.
