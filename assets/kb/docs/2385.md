# 3・8・4 モリブデン

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

モリブデン(molybdenum,化学記号Mo)は特殊鋼および超合金などの添加元素として多く使用されている.その他,触媒,潤滑剤および顔料などの化学製品や線棒材,板材および機械加工品などの形の金属製品に使用されている.金属製品としてのモリブデンは,高融点で耐熱性に富み,熱・電気伝導性に優れる特徴を有し,タングステンに比べて軽量で塑性加工性の良い耐熱材料として位置付けられ,ランプや電子管,ミサイルや航空機などの構成部品,高温炉の発熱体,熱遮へい板,敷板などの耐熱部材,恒温鍛造やダイカスト金型などの温間加工工具に使用されている.また,熱膨張係数がSiに近いことや,半導体デバイスの放熱部品や溶融ガラスと反応しにくいことからガラス溶融用の電極や容器などに使用されて表3・47タングステンの電気抵抗(線引きのまま)温度KnΩ×m温度KnΩ×m56.5590.5130.7904.0213.51085.0302.6表3・48 タングステンの諸性質化学的性質常温では安定であるが,赤熱では低級酸化物を経てWOとなる.また水蒸気と反応してWOを作る.硫酸・塩酸などと室温では反応せず,高温でわずかに侵され,硝酸とフッ酸の混合液には高温で急激に侵される.引張強さ(焼なまし) MPa引張強さ(線引きのまま) MPa1715~4116伸び(焼なまし) %伸び(線引きのまま) %1~4硬さHv550(線引きのまま)いるほか,核融合炉の第1壁材料としても期待されている.モリブデンは大気中で酸化しやすいため,ほとんどの用途において還元性・不活性ガス中あるいは真空中で使用する必要がある.モリブデンの化学的性質は塩酸(HCl)には耐食性を示すが,硝酸(HNO)やフッ酸(HF),硫酸(HSO)などには腐食される.低融点の溶融金属には優れた耐食性を示す.モリブデンは粉末冶金法あるいは溶解法で製造される.粉末冶金法では,粉末を機械プレスあるいは静水圧で成形したのち水素気流中または真空中で焼結し,鍛造,スエージング,線引き,圧延などの塑性加工を熱間と冷間で行って線棒材,板材にする.焼結は1800~2200°Cで,熱間の塑性加工は800~1400°Cで行われる.溶解法では真空アーク溶解や電子ビーム溶解が用いられている.モリブデンの結晶粒界は本質的に脆弱であり,室温近傍以下で粒界脆性破壊を示す低温脆性が現れる.この低温脆性は合金元素やO,N などの侵入型不純物の影響も強く受けるが,再結晶温度以上の高温環境や中性子照射環境での使用でさらにもろくなる.1000°Cを超える高温では再結晶により高温強度も低下する.結晶粒組織を制御して低温脆性や高温強度の改善を図った合金に,炭化物分散強化合金としてMo-Ti-Zr-C合金,Mo-Hf-C合金,固溶強化合金としてMo-Re合金,Mo-W 合金,組織制御合金としてMo-LaO合金,Mo-K-Si合金などがある.特に高いクリープ強度が必要な場合は二次再結晶法を利用して製造した単結晶材が用いられる.また,最近では,メカニカルアロイング法や内部窒化法(nitriding)を利用して強靱化の研究が進められているが,メカニカルアロイングによって微細な炭化物粒子で粒界を強化し結晶粒微細化したモリブデン合金は,照射によって低温靱性が著しく向上した報告がある.
