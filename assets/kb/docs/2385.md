c .パーライト可鍛鋳鉄（pearlitic malleable cast iron) Mn
の含有量を0.6～0.8 %に高め，第二段の焼なまし時間を短縮し，
冷却を早め基地にパーライトを残したまま製品としたものである．
他の可鍛鋳鉄品より強度が高く，耐摩耗性に優れており，自動車用
部品として多く使われる．黒心可鍛鋳鉄およびパーライト可鍛鋳鉄
のJIS規格を表2･51に示す．FCMPと書かれた記号はパーライト
可鍛鋳鉄を表す．ここでは，将来統合されるであろう種類は省略し
ている．
d.可鍛鋳鉄の諸性質 可鍛鋳鉄は，その成分や熱処理方法に
ついて特に規定はなく，主要寸法に見合った引張強さと伸びが，規
格を満たしていればよい．前記3種の可鍛鋳鉄の機械的性質は表
2･50，2･51に示すとおり，靱性が優れていることが特徴であるが，
球状黒鉛鋳鉄が開発されてからは，その生産量は減少している．
