# 2・9・6 合金鋳鉄（alloy cast iron)

> **章**: 第２章 鉄鋼材料
> **節**: 2・9 鋳鉄

a.概要 鋳鉄を高機能化することにより，その利用範囲を拡
大する試みがなされている．普通鋳鉄では対応できない環境でも使
用できるように，さまざまな元素を添加し，機械的性質をはじめ，
耐熱性，耐食性，耐摩耗性などの性質を改善したものが合金鋳鉄で
ある．表2･52に比較的使用される合金鋳鉄の例を示す．
b.低クロムねずみ鋳鉄（low chromium gray cast iron) Cr
は強い炭化物生成元素であり，鋳鉄の組織を大きく変える．低クロ
ム鋳鉄は，1.5%以下のCrを添加したもので，パーライトを安定
化させかつ微細化させるので，機械的性質，耐熱・耐摩耗および耐
食性を向上させている．Crが多いと白銑化しやすいのでNiやCu
などの黒鉛化促進元素を加えることが多い．
c .高クロム鋳鉄（high chromium cast iron) 高Cr鋳鉄は
10～35%Crを添加したもので，炭化物が多量に存在し耐摩耗性に
優れている．また，耐熱・耐食性もステンレス鋼に匹敵する．
d.低ニッケル鋳鉄（low nickel cast iron) Ni4 %以下の低
Ni鋳鉄では，Ni単体ではなくCrやMoを併用し機械的性質の向
上を図っている．Ni -Cr鋳鉄あるいはNi -Cr -Mo鋳鉄と呼ばれる．
e .マルテンサイト鋳鉄（martensitic cast iron) Ni4～8%
となると基地組織がマルテンサイトあるいはベイナイトとなり耐摩
耗性が非常に優れている．これをニハード鋳鉄と呼んでいる．Cや
Crが多いほど耐摩耗性は良好となる．
f .オーステナイト鋳鉄（austenitic cast iron) Niを12%以
上含有すると鋳鉄は鋳造のままで黒鉛とオーステナイト基地からな
るオーステナイト鋳鉄となり，耐熱・耐食性に優れた材料となる．
ニレジスト鋳鉄が最も著名である．その他，ニモル，モネル，ニク
ロシラル，ミノーバ，ノーマグといった名称で呼ばれるものがある．
