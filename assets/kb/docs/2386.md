# 3・8・5 ベリリウム

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

ベリリウム(beryllium,化学記号Be)は,比重が小さく,融点が高く,比熱,弾性係数が大きいので,航空機,宇宙船,ロケットの熱遮へいや構造材料として使用される.熱中性子吸収断面積が極めて小さく,散乱断面積が大きいので,研究炉の燃料被覆管,減速材,反射材としても用いられる.またγ線を吸収すると中性子(neutron)を発生するので,中性子源に用いられる.さらにX 線の吸収が少ないことから,X 線管球窓などに利用される.また各種合金の添加元素としても重要である.例えば,銅合金やニッケルに添加されると,優れた耐摩耗性,腐食抵抗性,高い電気伝導度,非磁性,ノンスパーク特性,非弾性挙動を付与する.またアルミニウムやマグネシウムの合金鋳物に対しては,細粒化や酸化抵抗性を向上させる.ベリリウムは,フッ化物のマグネシウム還元か塩化物の溶融塩電解で得られる.粉末焼結品として熱間圧延・押出しを行うことで,棒,板,チューブになる.ベリリウム合金(beryllium alloys)は,応力除去焼なましを行えば,曲げ,3ロール加工,深絞り,穴あけ加工などが可能になる.ベリリウムの接合は,酸素との親和力が大きいので,不活性ガスや真空中で行う.ベリリウムは,空気中で表面に薄く透明緻密なBeO被膜を形成し保護性があるが,750°Cを超すと酸化被膜のはく離が生じる.窒素に対しては,900°C以上で反応する.アンモニアとは,さらに低温で反応する.炭酸ガスに対しては,乾燥状態で700°Cまで,湿った状態で600°Cが限界となる.水素とは,いかなる温度でも反応しない.純水に対しては,ベリリウム中に炭化物,塩化物が混在していなく,細粒であれば,300°Cまで耐食性がある.塩酸,フッ酸,希硫酸,希硝酸に対しては,室温で反応する.濃硫酸とは反応が緩慢,濃硝酸とは反応しない.アルカリとは,水素を発生して激しく反応する.溶融塩に対しては,酸素,窒素を含まない場合に耐食性が良い.なお,ベリリウムは特定化学物質に指定されており,ベリリウム化合物を含め,微粉末,ちり,蒸気,溶解物などの可溶性化合物には毒性があり,呼吸器症状,アレルギー症状,皮膚炎などの障害を表3・49ジルコニウムの化学組成規格(ASTM B 349-93)Al:≦75B:≦0.5Cd:≦0.5C:≦250Cl:≦1300Cr:≦200Co:≦20Cu:≦30Fe:≦1500Hf:≦100Mn:≦50Mo:≦50Nb:≦1400 Ni:≦70Si:≦120Ti:≦50W:≦50U:≦3.0(単位:wt.ppm)表3・50 原子力炉用ジルコニウム合金の組成と機械的性質合金名ジルカロイ-2ジルカロイ-4Zr-2.5Nb合金元素wt.%Sn1.451.45―Fe0.140.21―Cr0.100.10―Ni0.06――Nb――2.5室温機械的性質耐力MPa引張強さMPa伸び%備考BWR 被覆管PWR 被覆管ATR 圧力管BWR 燃料チャネル引き起こすので,安全衛生管理が必要となる.
