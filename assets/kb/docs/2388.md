# 3・8・7 タンタル

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

タンタル(tantalum,化学記号Ta)はニオブ(niobium)と同族(Va)の,高融点(3270K)で非常に大きな比重(16.6)を有する金属である.鉱石(タンタライト,スズ鉱滓)から溶媒抽出法等によって精製した後,アルカリ金属による還元法あるいは電解法によってタンタル粉末に精錬される.造塊にはチタンやジルコニウムに採用されているアーク溶解法や電子ビーム溶解法に加えて粉末焼結法も使用される.タンタルは白金族に次いで耐食性に優れており,タンタルを侵すのはフッ酸,発煙硫酸や高濃度アルカリなどわずかである.酸素との親和力が強く,酸化性雰囲気中では570K以上の温度で容易に酸化する.また,常温でも水素を吸収する傾向があり,水素脆化には留意する必要がある.加工性は非常に良好で,常温において細線や箔への加工が可能である.2―113表3・51 タンタル(棒,ワイヤ)の成分規格例規格名上限濃度wt.%ASTM B 365(EB,VAR 材)C:0.010,O:0.015,N:0.010,H:0.0015,Nb:0.10,Fe:0.010,Ti:0.010,W:0.050,Mo:0.020,Si:0.005,Ni:0.010,Ta:残ASTM B 365(焼結材)C:0.010,O:0.03,N:0.010,H:0.0015,Nb:0.10,Fe:0.010,Ti:0.010,W:0.050,Mo:0.020,Si:0.005,Ni:0.010,Ta:残JIS H 4701(展伸材)C:0.03,O:0.03,N:0.01,H:0.0015,Nb:0.10,Fe:0.02,Ti:0.01,W :0.03,Si:0.02,Ni:0.02,Mo:0.02,Ta:残表3・52 タンタル(棒,ワイヤ)の室温機械的性質規格例規格名耐力引張強さ伸びASTM B 365(EB,焼結材)MPaMPa%直径:3.18~63.5mm≧103≧172≧25直径:0.635~3.14mm―≧207≧20JIS H 4701(直径:3mm以上)MPaMPa%軟質(TaB-O)―≧275≧251/4硬質(TaB-1/4H)―≧343≧20硬質(TaB-H)―≧490≧1タンタルの主用途は化学的に安定で誘電率の大きい酸化物の特性を利用した電解コンデンサで,粉末,線および箔として使用される.タングステン,ハフニウムやニオブ等との合金は耐熱,耐食材料としても使用されている.また,炭化タンタルや酸化タンタル等の化合物は,それぞれ超硬工具鋼や光学レンズの性能向上に利用されている.タンタルの成分規格および室温機械的性質規格の例を表3・51,
