# 3・8・9 金および金合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・52 に示す。

金（gold，化学記号Au）は面心立方構造をもち，柔軟で塑性加
工性，切削加工性に優れている．例えば，箔にすると厚さ0.1
m，線には直径8mまで加工できるが，機械的強さが弱い．耐
硫化，耐酸化性に優れ硝酸，硫酸，塩酸などには侵されない．王
水，青酸，塩素，臭素，ヨウ化カリウムに反応し銀同様に水銀とア
マルガム（amalgam）を作る．
導電性や耐食性に優れるため，電子工業では微量添加物により機
械的特性を改善した99.99 %Au以上の純度で，直径10～35mの
Au極細線がLSIのボンディングワイヤに使用されている．接触の
信頼性を得るためと耐食性向上に電気接点，コネクタ，その他金属
の表面装飾にめっきやクラッドが使われている．接合材料としては
18～25%Niおよび25～35%Pd合金がロケットのノズル，スカー
トおよびその他の高温ろう付けに用いられ，80%Au-Sn
（mp553.15K)，96.85 %Au-Si （mp643.15K）の共晶合金はセラ
ミックパッケージの封止用はんだに使用されている．また，波長
550nmから超遠赤外領域の電磁波を高い反射率で反射するた
め，熱線の反射膜として宇宙装置や機器，電気炉に利用され
る．超微粒子はCO低減触媒として活用され，生体材料として
歯科合金やカテーテルのマーカ，超微粒子コロイドはDNA制御用
に用いられている．
装飾用材料（ornament material）は審美性，機能性，加工性，
耐食性を考慮しAg，Cu，Znの配合比を調整したカラット金や
Ni，Pd，Ag，Cu，Zn（NiはEUで使用を規制）を合金した白色
系のホワイトゴールド（white golds）が多種類ある．純金のコイ
ン，記念メダル，金地金は資産や投資対象商品として定着してい
る．Au合金のおもな用途例を表3･53に示す．
線，管をはじめ微細加工に向き，切削性も良好であるが機械的強度
が低く，加工硬化しても常温で再結晶し軟化する．硫化水素，亜硫
酸ガスによって常温で変色する．この硫化を防ぐにはAuまたは
Pdを50at.%以上含有させる必要がある．硝酸や熱硫酸には容易
に溶け，塩素イオンと化合し塩化銀を形成する．セレン酸，次亜塩
素酸に反応するが，酢酸，フッ酸，リン酸，カ性ソーダとは反応し
ない．多湿下で電圧印加があるとマイグレーション（migration）
を起こす．

ペーストにしてハイブリッドICや抵抗器，マルチレイヤキャパシ
タ電極に使用されている．銀塩の感光特性がフィルム，印画紙とし
て銀需要の3割以上であったが，ディジタル化により激減してい
る．銀器，コイン，装飾品には約30%使われている．可視光線の

応により還元し反射板や鏡に利用されている．触媒にはホルムアル
デヒド製造にAgの網や細線が使われるほか，抗菌材料
（antibacterial material）としても広く利用されている．酸素を透
過することおよびピンニング効果から酸化物超伝導用のシース材と
しても使われている．Ag合金のおもな用途例を表3･54に示す．
3・8・11白金およびおもな白金族とその合金
白金（platinum，化学記号Pt），パラジウム（palladium，化学
記号Pd），ロジウム（rhodium，化学記号Rh），イリジウム（irid-
ium，化学記号Ir），ルテニウム（ruthenium，化学記号Ru），オ
スミウム（osmium，化学記号Os）の6元素が白金族である．面
心立方のPt，Pdは常温でも加工性が良い．同じ面心立方でもRh
とIrの常温加工は難しいが，熱間加工は可能である．ちゅう密六
方のRu，Osは研削，研磨加工はできるが常温，熱間の塑性加工は
困難である．銀，金およびおもな白金族の物理的性質，おもな貴金
属二元合金の機械的性質，白金族および白金族合金とそのおもな用
途を表3･55～3･57に示す．
