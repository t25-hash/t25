# 2・1・4 鋳造における諸現象

> **章**: 第2章 鋳造
> **節**: 2・1 鋳造の基礎

鋳造加工が他の加工法と大きく異なる点は製品形状の自由度であ
ろう．この特徴は2･2･3項で示される中子の使用にある．中子を用
いることでエンジンブロックのような複雑形状品を一体で成形でき
る点に最大の特徴がある．しかし，溶融金属を出発点にするため，
寸法の安定性は他の加工法に比べて劣るといわざるを得ない．ま
た，一般の溶融金属は凝固時に体積収縮を伴う．この結果として製
品にひけ巣欠陥や，ピンホール，ガスホールと称するガス欠陥がで
きやすい．以下にこれらの点について簡単に記述する．詳細は専門
書 を参照いただきたい．
a.寸法変化 凝固の終了から室温までの間の冷却により，そ
の金属の熱膨張係数に見合った寸法変化（収縮）をする．このた
め，鋳物の模型はこの寸法変化を見込んで大きめに作製しなければ
ならない．これに使用する定規を鋳物尺［shrinkage rule，または
伸尺（contraction rule)］といっており，製品の寸法より0.8～
2.5%大きめに作製されている．また，冷却が均一に行われないと，
熱応力による変形が生じる．これらの結果として鋳造品の寸法精度
は他の加工法に比べて劣る ．
b.ひけ巣（shrinkage，shrinkage cavity) 凝固時の収縮は，
鋳物にひけ巣と称する内部欠陥をもたらす．初期の鋳造技術におけ
る多くのシミュレーションはひけ巣防止の対策のために行われた
といっても過言ではない．これらのシミュレーションは凝固解析
から，湯流れ解析，そして凝固組織の予測へと進展してい
る ．
ひけ巣を防止するために押湯を用いることは2･1･1･d項に述べ
た．原理的には，押湯は製品よりも後で凝固が終了することで，押
湯から製品に不足分の溶融金属が補給される．
c.ガス欠陥（gas defects ) 溶解時に溶融金属に過剰に含有
されたガス，あるいはとりべ内や鋳型内の流れの途中で吸収された
ガスは，凝固時にガス欠陥を生じる．このためには種々の脱ガス法
が開発されてきた．これらの代表的な例が，鋼の場合には脱酸であ
り，アルミニウム合金の場合には脱水素である ．
また，溶解した溶融金属ではガス含有量が少なくとも，とりべな
どの運搬中，あるいは鋳型内でのガス吸収や，ガスの巻込みにより
欠陥が発生する．これらはピンホールや，吹かれなどのガス欠陥を
生じる．さらには，ダイカストなどでは，巻き込んだガスが高圧で
押しつぶしており，熱処理を行うとこのガスが膨張してブリスタ
（brister ，またははがれ）といわれる欠陥を生じる ．
