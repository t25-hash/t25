# 3・8・9 金および金合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・52 に示す。

金(gold、化学記号Au)は面心立方構造をもち、柔軟で塑性加工性、切削加工性に優れている。例えば、箔にすると厚さ0.1m、線には直径8 mまで加工できるが、機械的強さが弱い。耐硫化、耐酸化性に優れ硝酸、硫酸、塩酸などには侵されない。王水、青酸、塩素、臭素、ヨウ化カリウムに反応し銀同様に水銀とアマルガム(amalgam)を作る。導電性や耐食性に優れるため、電子工業では微量添加物により機械的特性を改善した99.99%Au以上の純度で、直径10~35 mのAu極細線がLSIのボンディングワイヤに使用されている。接触の信頼性を得るためと耐食性向上に電気接点、コネクタ、その他金属の表面装飾にめっきやクラッドが使われている。接合材料としては18~25%Niおよび25~35%Pd合金がロケットのノズル、スカートおよびその他の高温ろう付けに用いられ、80%Au-Sn(mp553.15K)、96.85%Au-Si(mp643.15K)の共晶合金はセラミックパッケージの封止用はんだに使用されている。また、波長550nmから超遠赤外領域の電磁波を高い反射率で反射するため、熱線の反射膜として宇宙装置や機器、電気炉に利用される。超微粒子はCO低減触媒として活用され、生体材料として歯科合金やカテーテルのマーカ、超微粒子コロイドはDNA 制御用に用いられている。装飾用材料(ornament material)は審美性、機能性、加工性、耐食性を考慮しAg,Cu,Znの配合比を調整したカラット金やNi,Pd,Ag,Cu,Zn(NiはEU で使用を規制)を合金した白色系のホワイトゴールド(white golds)が多種類ある。純金のコイン、記念メダル、金地金は資産や投資対象商品として定着している。Au合金のおもな用途例を表3・53に示す。
