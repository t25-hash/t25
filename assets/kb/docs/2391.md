# 3・8・10 銀およびその合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・52 に示す.

銀(silver,化学記号Ag)は金同様に展延性に富み,箔,板,線,管をはじめ微細加工に向き,切削性も良好であるが機械的強度が低く,加工硬化しても常温で再結晶し軟化する.硫化水素,亜硫酸ガスによって常温で変色する.この硫化を防ぐにはAuまたはPdを50at.%以上含有させる必要がある.硝酸や熱硫酸には容易に溶け,塩素イオンと化合し塩化銀を形成する.セレン酸,次亜塩素酸に反応するが,酢酸,フッ酸,リン酸,カ性ソーダとは反応しない.多湿下で電圧印加があるとマイグレーション(migration)を起こす.金属の中では電気伝導性,熱伝導性が最も良く,電気接点,各種表3・53 金および金合金とそのおもな用途代表的な金,金合金用途例99.99%up(純金),硬化金金貨,記念メダル,装飾品,めっき材,電気接点,反射膜99.99%up(微量添加物),99Au-Pdボンディングワイヤ93~74Au-Ag-Cu,63~58Au-Ag-Cu-Pd-Zn歯科用材料82Au-Ni,80Au-Cu,75Au-Cu-Ag,70Au-Pd金ろう96.85Au-Si,80Au-Sn,79Au-Sn金はんだ91.7Au-Ag-Cu(K 22),83.5Au-Ag-Cu(K 20)装飾・工芸品,食器類75Au-Ag -Cu(K 18),58.5Au-Ag-Cu(K 14)装飾・工芸品,時計,万年筆75Au-Pd,75Au-Ni-Pd,75Au-Cu-Pd-Ni(K 18)装飾・工芸品(ホワイトゴールド)導電材料や酸化銀電池に使われている.Ni,Mg を0.3%添加し内部酸化により電気伝導率を維持し再結晶温度を高めたもの,Cuを合金し電気伝導率をあまり下げずに硬くしたもの,PdやAuを合金し耐食性を高め接触の安定化を図ったものがリレーやスイッチの電気接点に使われている.また内部酸化法や粉末冶金法でCdO,SnO,ZnOを分散させて耐溶着,耐摩耗性を改善して開閉器などの電気接点に使用している.接合材料としては72%Ag-Cu共晶合金およびそれにZn,Cdなどを含むAg ろうや含銀はんだおよび導電接着剤(electro conductive adhesives)がある.電子工業ではペーストにしてハイブリッドICや抵抗器,マルチレイヤキャパシタ電極に使用されている.銀塩の感光特性がフィルム,印画紙として銀需要の3割以上であったが,ディジタル化により激減している.銀器,コイン,装飾品には約30%使われている.可視光線の95%以上を反射することから,硝酸銀を還元性有機物で銀鏡反応により還元し反射板や鏡に利用されている.触媒にはホルムアルデヒド製造にAg の網や細線が使われるほか,抗菌材料(antibacterial material)としても広く利用されている.酸素を透過することおよびピンニング効果から酸化物超伝導用のシース材としても使われている.Ag 合金のおもな用途例を表3・54に示す.
