# 3・8・11 白金およびおもな白金族とその合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・52 に示す.

白金(platinum,化学記号Pt),パラジウム(palladium,化学記号Pd),ロジウム(rhodium,化学記号Rh),イリジウム(irid-ium,化学記号Ir),ルテニウム(ruthenium,化学記号Ru),オスミウム(osmium,化学記号Os)の6元素が白金族である.面心立方のPt,Pdは常温でも加工性が良い.同じ面心立方でもRhとIrの常温加工は難しいが,熱間加工は可能である.ちゅう密六方のRu,Osは研削,研磨加工はできるが常温,熱間の塑性加工は困難である.銀,金およびおもな白金族の物理的性質,おもな貴金属二元合金の機械的性質,白金族および白金族合金とそのおもな用途を表3・55~3・57に示す.a. ロジウムRhは熱間で塑性加工ができ,常温で切削加工2―116材料学・工業材料が可能である.結晶粒を小さく制御すれば細線の引抜加工もできる.加熱した王水に溶け,熱硫酸,臭化水素酸,次亜塩素酸ナトリウム,食塩の飽和溶液,ヨウ素に侵される.耐食性と色仕上げのため装飾品やリードリレーの電気接点,自動車排ガス浄化の還元触媒,酢酸製造などの触媒としても使用される.Ptに5~40%合金させると機械的強さや耐熱性を高める役割も果たしている.文献(1)
