# 4・1・1 電線・導体材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・1 導電材料

a. 線材および裸電線導電線材料の必須の特性に次のものが挙げられる。(1)電気抵抗が小さい。(2)接続が容易。(3)耐食性、機械的特性が優れる。特性と価格などから電線・導体材料は、銅、アルミニウムとそれらの合金が中心になる。表4・1におもな導電線材料の特性を示す。銅線(copper wire)の原料は電解精錬された99.99%純度のものである。これを溶融し要求に応じてタフピッチ銅(tough pitchcopper,JIS H 2123:1963)、リン脱酸銅(phosphorous deoxid-ized copper,JIS H 2124:1963)、無酸素銅(oxygen free cop-per,JIS H 2125:1969)の組成に溶湯処理し、荒引線まで連続的に加工される。荒引線は伸線加工され、そのまま硬銅線(JIS C3101:1950)になるか、焼なましされて軟銅線になる。JIS には電気機器巻線用軟銅線(JIS C 3103:1950)、平角銅線(JIS C3104:1953)、硬銅より線(JIS C 3105:1953)、スズめっき銅線(JIS C 3151,3152:1965)、整流子片(JIS C 2801:1952)、硬銅トロリ線(JIS E 2101:1951)、銅ブスバー(JIS H 3140:1977)などが規格されている。銅線の電気抵抗値をできる限り保持しながら強度などの機械的性質、耐熱性を改良するために合金化が行われる。強度の向上を目的にクロムジルコニウム銅、クロム銅、ジルコニウム銅(C 15000)、ベリリウム銅の熱処理合金が電子機器、航空機の配線に、非熱処理合金ではCu-Mg-Zn合金、Cu-Sn-In合金が電気毛布用線に、C2200(9/1丹銅)がアーマーロッドに用いられる。ばね性を目的に黄銅(JIS H 3260:1977)、洋白(JIS H 3270:1977)、リン青銅(JIS H 3270:1977)、キュプロニッケル、ベリリウム銅(JIS H3270:1977)が、通信機などのスプリングに用いられる。耐熱性を目的に銀入銅、スズ入銅、クロム銅、クロムジルコニウム銅が電気毛布用線、トロリ線、高温銅線、整流子片に用いられる。スズ入銅表4・1 市内電話線加入者線と中継線に分かれ、発泡ポリエチレン絶縁ラミネートシースケーブル、ポリエチレン絶縁ポリエチレンシースケーブルまたはビニルシースケーブル、着色ポリエチレン絶縁ラミネートシースケーブルが用いられる。さらに誘導障害防止用の遮へいケーブル、架空用の自己支持型ケーブル、コルゲートシースケーブル、直埋用の帯外装ケーブル、鳥獣害防止用の防ぎ(蟻)、防そ(鼠)ケーブルなどがある。. 市外電話線市外電話線は多重化した信号の伝送を可能にするため市内ケーブルより良好な特性を要求される。近い都市間には発泡ポリエチレン絶縁市外PEFケーブル、搬送ケーブルが用いられる。長距離都市間には超多重伝送が可能な9.5mm同軸ケーブル(coaxial cable)、4.4mm同軸ケーブルが、海洋横断には海底同軸ケーブルが用いられる。これらのケーブルは、下記に述べるような光ケーブルにとってかわられつつある。. 電話局内その他配線局内、加入者への引込み、局内または構内配線、接続器から電話機に使われる各種電線である。電話局および構内ではビニル絶縁ビニルシースケーブル、ポリエチレン絶縁ビニルシースケーブル、プラント局内ケーブル、ポリエチレン絶縁ビニルシース構内ケーブルが用いられる。加入者への引込みはSD ワイヤ、鋼心入り屋外線、屋内配線にはビニル絶縁電線が、電話機から接続器までの部分には機ひもが用いられる。最近では環境への配慮から、ビニルから難燃ポリエチレン化が進んでいる。電話線以外の通信用電線に次のものが挙げられる。光ファイバ(optical fiber)は石英などの透明体にレーザ光を通すことによって情報やエネルギーを伝送する伝送媒体である。このケーブルは細く、軽く、低損失しかも広帯域で漏話や誘導のない理想的なケーブルとして広範囲にわたり使用されている。光電力複合ケーブルや光ファイバ複合架空地線の複合ケーブルも含まれる。テレビジョン放送の普及発達、CATV の普及に伴い、これらの用途に適した電線が使用される。給電線用同軸ケーブル、テレビカメラ用ケーブル、テレビ受信用同軸ケーブル、CATV 用同軸ケーブルなどがある。また鉄道および自動車無線用に漏えい同軸ケーブルが用いられ、宇宙開発の進歩につれて宇宙通信用の電線、電子機器の進展につれてこれらの信号伝送に多様な電線が使われる。
