# 2・2・3 中子と幅木

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

中子（core）とは，鋳物の孔部，空げきおよび突出部あるいはく
ぼみ部等の鋳型の一部を別個に作製し，鋳型の組立時に取付けるも
ので，作業性，経済性の観点からは好ましくない造型法である．し
たがって中子の使用は，鋳物の形状の点から中子が絶対に必要なと
き，あるいは中子を用いることで鋳物の製造工程が簡単になるとき
に限られる．中子は鋳物の内部形状を形成する鋳型として使用する
ため，その特性は凝固過程で発生するガスの放出しやすい通気性，
流入する溶湯の衝突に対して十分耐えられる強い表面強度と結合
力，鋳型より高い耐火性あるいは非焼付き性，また凝固後の型ばら
し（shake-out ）性などが求められる．そして中子の位置決めおよ
び固定するために，幅木（core print ）をあらかじめ模型の一部と
して設計しておく．
幅木の大きさは溶湯から受ける浮力とその支持力から決められ，
支持力≧浮力でなければならない．中子の支持方法（鋳込方法）に
よって異なるが，中子の大きさを直径 と長さ として浮力と支
持力より算出すると，乾燥鋳型の場合の横両幅木（double core
print ）では表2･3 に示すような大きさ となる．しかし，これ
らの値はおおよその目安であって，中子の支持力不足や強度不足は
そのまま不良品となるので，設計段階で製品の内面形状を改善する
必要がある．図2･22 に，中子の薄い部分をなくし，幅木による
支持力を高めるとともに，ガスの抜けが良好になるように改善した
例を示す．
