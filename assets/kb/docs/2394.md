# 4・1・2 超伝導材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・1 導電材料

一般的に金属材料は冷却によって電気抵抗が低下するが有限の値を示す。一方、ある種の材料は極低温に冷却すると臨界の温度以下で電気抵抗が消失し零になる。このような特性を示す物質を超伝導物質(superconductor)と呼ぶ。超伝導物質は数千種が知られている。超伝導物質を導電材料としたとき、エネルギーロスなしで電流を流すことができるため、エネルギー、エレクトロニクスなどの幅広い分野で応用、開発がなされている。a. effect)と呼ぶ]、外部磁界が臨界磁界욂に達すると超伝導体が磁場の侵入を許し、突然常伝導状態に戻る超伝導体を第一種超伝導体と呼ぶ。第一種超伝導体としてはHg,Pb,Snなどが挙げられるが、臨界磁界が数十mT と小さいためジョセフソン素子などの超伝導エレクトロニクスに用いられる。一方、第二種超伝導体では、ある磁場までは第一種超伝導体と同様に磁場の侵入を許さないが、下部臨界磁場욂1を超えると超伝導相と非超伝導相の混合状態になり、さらに上部臨界磁場욂2を超えると完全に常伝導状態になる。第二種超伝導体にはNbTi合金、NbSn,NbAl化合物、BiSrCaCuO酸化物など、욂,욂,욂が高い材料が多く、導電材料としたとき大容量の直流電流を流すことができる。マグネット用巻線とした場合、強力な磁場を発生できるため高磁界マグネットなどの分野で実用化が進んでいる。表4・2におもな超伝導体とその特性を、表4・3に超伝導線構成材の物性を示す。超伝導体を超伝導マグネット(superconducting 応用分野超伝導の特徴はエネルギーロスなしに大きな磁場空間が得られることである。その特徴を生かして、医療用画像診断装置(MRI)、分析用核磁気共鳴装置(NMR)、粒子加速器、粒子検出器、物性研究用高磁界マグネットなどが実用化されている。さらに大規模な研究開発が進められているものとして、核融合、MHD 発電、電力貯蔵(SMES)、発電機、送電ケーブル、磁気浮上リニアモータカー、電磁推進船、磁気分離装置、磁気軸受など幅広い分野への応用が検討されている。また、量子特性を利用してジョセフソン素子、SQUID などエレクトロニクス素子への応用が進んでいる。
