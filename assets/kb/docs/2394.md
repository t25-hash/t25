# 2・2・4 抜けこう配

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

造型作業において鋳型から模型を引き抜く際に，長い平行部をも
つ形状は引き抜くことが困難で，砂型鋳造では鋳型を壊すおそれが
ある．また，金型鋳造やダイカスト鋳造であっても，鋳型から鋳物
を取出しにくい．このような部分には抜けこう配（pattern
draft ）をつけることが望ましく，JIS規格では表2･4や表2･5に
示すような抜けこう配の許容値が定められている．また，ダイカス
ト鋳物においても，図2･23 に示すように，合金の種類に適した
抜けこう配を設ける．さらに，鋳物設計では，図2･24 に示す例
のように，どの部分を分割面とするかをあらかじめ確認し，その寸
法差があっても差し支えないか検討しておく必要がある．あるいは
寸法差が許されない場合は，図2･25 のように，形状そのものを
変更しなければならない．
