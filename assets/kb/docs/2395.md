# 4・1・3 接点材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・1 導電材料

接点(electrical contacts)は電気回路を開閉する部分に使われ、電力用の遮断機や開閉器、各種制御装置の機械接点などで幅広く用いられている。接点には接触抵抗が低く安定している、溶着が生じにくい、放電消耗や摩耗が少ないなどの特性が要求される。このため材料には、導電率や熱伝導率が高い、耐酸化性が良い、融点や沸点が高い、硬度が高い、耐食性が良いなどの特性が望まれ、単金属や合金のほかに焼結材料が用いられている。表4・4に代表的な接点材料とその特徴を示す。Ag は導電率や熱伝導率が高く常温では酸化されにくいため、軽負荷・中負荷用の接点材として最も広い範囲で使用されている。しかし、融点が低い、硬度が低いため接触圧力を大きくできない、Sが存在する環境では容易に硫化物の被膜やウィスカが発生して接触障害を起こすなどの欠点をもっている。Ag を主体とする合金はそれらの欠点を改善するために種々の金属や酸化物などを添加したもので、耐摩耗性や硬度が改善されている。Auは耐食性に優れており微弱電流に対する用途で用いられているが、融点や硬度が低いため溶着しやすく耐摩耗性に欠点がある。Auを主体とする合金としてはAu-Ag-Pt合金が代表的であり、通信用継電器においてPt代用接点材として広く用いられている。材料特徴おもな用途Ag代表的接点材料、耐硫化性が劣る継電器、マイクロリレー、コネクタAu耐食性が良い、粘着しやすいコネクタ、音響用接点Ag-Cu耐摩耗性が良い、耐酸化性が劣るマイクロモータ整流子Ag-PdAg の耐硫化性を改善継電器、交換機、サーモスタットAu-Ag-Pt耐食性が良い通信機用継電器、制御機器Pt-Ir耐食性に優れる、高強度、耐アーク性が良いガバナ接点、警報器、制御機器Ag-CdO耐溶着性、耐摩耗性が良い継電器、制御器、ブレーカW耐アーク性、耐溶着性が良い、接触抵抗が高い電圧調整器、バイブレータAg-C耐溶着性が良い、耐摩耗性が劣る信号継電器、電圧調整器Ptは高融点で耐食性も高く溶着を起こしにくいが、導電率や硬度が低い欠点がある。よって通信用継電器などの接触圧力が低く確実性を要求される用途に用いられている。この合金としては、Pt-Ir合金がマイクロモータのガバナ接点などの極微弱電流に対する用途で使用されている。W およびその合金は粉末焼結法によって作られる接点材料である。W は融点が高く硬度や熱伝導性にも優れているため、接点として用いたとき摩耗やアーク放電による消耗が少ない特徴がある。一方、接触抵抗が高い、加工性が悪い、酸化されやすいなどの欠点ももっている。合金材としてはAg-W やCu-W の焼結合金が使用されている。接点材料に関する規格としてはJIS C 2509:1965に通信機器用接点材料についての規定があるが、国内、海外を通じて十分整備されている状況とはいえない。
