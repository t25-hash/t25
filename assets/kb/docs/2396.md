# 4・1・4 抵抗材料(電熱材料、サーミスタ、バリスタ)

> **章**: 第4章 電磁気・電子材料
> **節**: 4・1 導電材料

抵抗材料(electric resistance material)は物質のもつ電気抵抗およびその変化を利用する材料であり、標準抵抗や電位差計などの計測器に用いる精密抵抗材料、電子機器の回路制御や電車用電動機のような電流調整に用いる加減抵抗材料、電気炉の発熱源として用いる電熱材料などに分類することができる。抵抗材料は金属性のものから炭素や酸化物のような非金属性のものまで種類が多く、それぞれのもつ固有の特性によって用途別に選択されている。抵抗材料には一般的に、抵抗率が適当に大きく経時変化が小さい、抵抗の温度係数が小さい、耐食性が良い、機械的強度が高いなどの特性が要求される。a. 精密抵抗材料(standard resistance material)標準抵抗器や電気計測器など精度を要求される抵抗材料としてはCu-Mn合金やCu-Ni合金などが用いられている。JIS では、JIS C 2522:1953にCu-Mn合金、JIS C 2521:1953にCu-Ni合金についての規定がある。表4・5および4・6にJIS に規定された特性を示す。マンガニンの名前で知られているCu-Mn合金は、代表的な精密抵抗材として最も広く用いられている。抵抗温度係数や対銅熱起電力が極めて低い特徴があるが、耐食性や耐酸化性に乏しい欠点ももつ。Cu-Ni合金はすべての配合割合で固溶体を形成し、安定した抵抗材料として使用できる。このうち約45%のNiを含む合金はコンスタンタンなどの名前で知られ、抵抗率が大きく、かつ温度係数が小さいため精密抵抗材料として広く使用されている。Cu-Mn合金化学成分%体積抵抗率Ωm等級A material)一般的な電流調整回路に用いられる抵抗材料の性能には精密抵抗材ほどの精度は要求されず、むしろ経済性を重視して安価な材料が求められる。JIS ではJIS C2532:1976において表4・7に示す7種の合金系の材料が一般電気抵抗用として規定されているが、なかでもCu-Ni系やNi-Cr系の合金が広く使用されている。また、大電流の調節にはFe-C,Fe-Cr-Al,Ni-Cr-Feなどの合金が用いられている。これらは耐熱性が高く安価であり、電車の速度調節用などに用いられている。c. 電熱材料(electric heating material)電気加熱の発熱源として用いられる抵抗材料には、高温での耐酸化性、耐食性、高温強度に優れる、冷熱サイクルに強い、加工性が良いなどの特性が求められ、金属のほかに非金属の材料も用いられている。ニクロム線として知られるNi-Cr合金、およびカンタル線として知られるFe-Cr-Al合金は古くからこの用途に使用されており、JIS C 2520:1953において規格化されている。表4・8にその特性を示す。これらの合金の使用温度限界を超える高温での使用を必要とする場合には、Pt,Mo,W,Taなどの単金属も用いられる。Ptは高価であるが耐酸化性が高く、溶融点に近い1873K 程度まで大気雰囲気で使用できる。一方、Mo,W,Taは2273K 以上の高温で使用できるが、酸化しやすいために雰囲気を真空や不活性ガスにする必要がある。非金属の電熱材料は電気抵抗が大きいため合金系の電熱材料より高温で使用できる。SiC粉末にC粉末やバインダを混合して焼結成形したSiC発熱体は、エレマ、シリコニットなどの名前で広く使用されており、大気中において1873K 程度までの高温で使用できる。そのほかにもMoSi,ZrO、黒鉛などが発熱体として用いられている。d. 測温抵抗体(resistance bulb)一般に金属の電気抵抗は温度の上昇とともに増加する。ここで、温度変化による電気抵抗の変化が一定で単純な関係にある材料を使えば、電気抵抗を測ることによって温度を求めることができる。このような温度測定に用いる抵抗材料を測温抵抗体といい、73~873K サーミスタ(thermistor)半導体の電気抵抗が示す特徴を利用したものとしてサーミスタとバリスタがある。サーミスタは半導体の電気抵抗の温度係数が負の大きな値を示す性質、すなわち温度が高くなると電気抵抗が大きく減少する性質を利用した素子である。サーミスタは、Mn,Co,Ni,Cu,Feなどの金属の酸化物を混合焼結したり、SiCと金属酸化物の混合焼結によって作られる。用途としては温度-抵抗特性を利用して、温度測定、温度補償、定電圧装置などに用いられる。f.
