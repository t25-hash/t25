# 4・2・3 項でも述べるトランジスタのような微細構造を原子レベルで

> **章**: 第4章 電磁気・電子材料
> **節**: 4・2 半導体デバイス材料

規則的に再現性良く製造するには、原子が集まって作る基本構造が三次元空間で周期的に繰返されている物質、すなわち単結晶(sin-gle crystal)が適しているためである。単結晶の成長方法には、浮遊帯域(FZ : floating zone)法と引上げ(CZ : czochralski)法がある。FZ 法は多結晶棒を周囲から高周波コイルで加熱溶融し、種結晶を接触させた後コイルを移動させることで単結晶を成長させる方法である。一方、CZ 法は多結晶の塊を石英るつぼに入れて加熱溶融し、種結晶を溶融部に浸して回転させながら引上げることで丸棒状の単結晶を成長させる方法である。CZ 法は比較的低抵抗率の単結晶を、FZ 法は高抵抗率の単結晶をそれぞれ得るのに適している。一般のトランジスタやダイオードの製造に使用される単結晶には、CZ 法で製造したものが使用される。これは、CZ 法で成長させた単結晶には石英るつぼから溶け込んだ酸素が約10個/cmとFZ 法で成長させた結晶より約2けた多く存在し、この酸素が熱応力で発生する転位に析出し、半導体としての電気特性を低下させる転位の成長を阻止してくれるためである。これら半導体材料のエネルギー帯構造を示したのが図4・6で、価電子が充満した価電子帯と伝導電子(自由電子)が存在する伝導帯からなっている。両者のエネルギー差は禁制帯(band gap,)と呼ばれ、一般的には約1eV(1.6×10J)である。室温近傍(例えば27°C)ではボルツマン定数(=1.38×10J/K)と絶対温度の積(kT 値)は約4×10J であるから、ボルツマン因子[exp-]は有限の値となり、価電子は伝導帯に励起され伝導電子となりうる。このとき、価電子帯には正孔が発生する。したがって、通常の半導体では電子密度と正孔密度は等しくなる。しかし、半導体材料中に不純物が混入すると、このバランスがくずれる。例えばIV族のSi結晶中にV族の元素PがSi元素の格子位置に混入すると、価電子が一つ過剰になり、この過剰な価電子はP元素の束縛を離れ伝導電子となる。一方、III族の元素B がSiの格子位置に混入すると、価電子が一つ少ないために価電子帯に正孔が発生する。この状況を図4・6のエネルギー帯で示すと、不純物の形成する中間準位からの電子あるいは正孔の励起に相当する。伝導電子を増加させる不純物をドナー(donor)、正孔を増加させる不純物をアクセプタ(acceptor)と呼ぶ。不純物を含まない半導体は図4・6 半導体のエネルギー帯構造図4・7 というエネルギーは、波長約1.24 mの近赤外光のもつエネルギーと等価である。したがって、半導体に禁制帯幅以上のエネルギーをもった光が入射すると、図4・7に示したように、光エネルギーは吸収され電子と正孔対が発生し導電率が増加する。この現象を光電効果(photoelectric effect)という。禁制帯幅が大きくなると吸収される波長は短くなり、可視光(波長380~770nm)が吸収される禁制帯幅は1.61~3.26eV となる。この光電効果を応用したものが太陽電池である。逆に、過剰な電子や正孔が存在すると互いに結合して光を放出する。このとき放出される光の波長は禁制帯幅で決定される。したがって、禁制帯幅を制御することで特定の波長の光を放出させることが可能となる。この原理を応用したものが発光ダイオードや半導体レーザである。半導体材料に応力が作用すると結晶にひずみが発生し、エネルギー帯もひずむことから電子あるいは正孔濃度が変化し、結果として導電率(抵抗率)も変化する。この現象をピエゾ抵抗効果(piezoresistance effect)という。このピエゾ抵抗効果を応用して半導体ひずみゲージが加速度センサ等に使用されている。このように半導体材料の電気・光・機械特性はエネルギー帯の構造で大きく変化することから、このエネルギー帯構造を積極的制御することでさまざまな製品分野に活用されている。おもな半導体材料とその用途を表4・9にまとめて示す。
