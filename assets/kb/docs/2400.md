# 4・2・3 トランジスタ、集積回路材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・2 半導体デバイス材料

半導体材料を使用してトランジスタやダイオードおよびこれらを用いた集積回路(IC: integrated circuit)を製造するには、n型半導体とp型半導体を接続させたpn接合構造を形成する。n型半導体には余剰電子が存在し、p型半導体には余剰の正孔が存在するため、pn接合を形成するとn型領域の電子はp型領域に、p型領域の正孔はn型領域にそれぞれ拡散し、残されたドナー、アクセプタの空間電荷による電界と釣合って平衡状態となり図4・8のような(a) 平衡状態(b) 順方向(c) 逆方向図4・8 pn接合のエネルギー帯構造エネルギー帯を形成する。接合に生じた電位差(拡散電位)はそれぞれ電子と正孔の流れを抑制する障壁となる。このpn接合に、p型領域が正となるように電圧を印加する(順方向)と、この障壁が減少し、n型領域からp型領域へ電子が、p型領域からn型領域へ正孔がそれぞれ流れ込み電流が生じる。一方、p型領域が負となるように電圧を印加する(逆方向)と障壁が高くなり、電流は流れない。したがって、pn接合には整流特性がある。このpn接合を二つもったnpnあるいはpnp構造を作り、一方のpn接合を順方向、他方を逆方向にバイアスして動作させるのがバイポーラトランジスタ(bipolar transistor、図4・9)である。順方向にバイアスするpn接合をエミッタ、逆方向にバイアスするpn接合をコレクタ、中間領域をベースと呼ぶ。エミッタから流れ出した電子あるいは正孔は大部分がコレクタに流れ込むが、一部はベースから流れ出す。ベース電流のわずかな変化によってコレクタ電流が変化するため電流増幅が生じる。金属と半導体を接続して形成した接合もpn接合と同様に図4・10のような障壁を形成し、整流特性を示す。この金属と半導体の間に絶縁物層を挟んだ構造をMIS(metal insulator semiconductor)構造(図4・11)と呼ぶ。絶縁物層が中間に存在するので、半導体に向かって電流が流れ込むことはない。金属側に負の電圧を印加すれば半導体表面(絶縁物との界面)には正孔が増加し、逆に正の電圧を印加すれば電子が増加する。したがって図4・11において紙面に垂直方向に電界を印加すると電流が流れることになる。この原理を応用したのが図4・12に示したMOS(metal oxide semiconduc-tor)型電界効果トランジスタ(FET : field effect transistor)である。ゲート電圧の変化によって流れる電流も大きく変化することから増幅作用が生じる。半導体材料の一部にp型領域あるいはn型領域を形成すれば抵抗を形成することが可能となる。抵抗値はそれぞれの領域にドーピングする不純物濃度で制御できる。また上述したMOS 構造はコンデンサとしても動作させることができる。したがって、半導体材料の所望の位置にトランジスタや抵抗あるいはコンデンサを形成し、電気配線で接続すれば集積回路を形成することができる。このような集積回路は、ウェーハと呼ばれる円盤状に加工された単結晶基板表面に形成される。ウェーハの直径は最大で300mm、厚さは約1mmである。現在ではトランジスタや薄膜配線の最小加工寸法は0.1 mにまで微細化されており、1cmの領域に約10億個のトランジスタ(大規模集積回路、LSI: large scale integrat-ed circuit)を作りこむことが可能となっている。
