# 4・2・4 はんだ接続材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・2 半導体デバイス材料

こうして製造された集積回路はウェーハから個片に切出され、LSIチップ(chip)となる。このLSIチップと外部との電気的接続を容易に行うため、はんだ(solder)材料が使用される。はんだ材料に望まれる性質としては、材料費が安く、数百°C以下の温度で容易に多数点を同時に接続できる経済性と、その接続が-100~150°C程度の温度範囲で安定に保たれる信頼性等がある。また、電気的な接続が不良となった場合には、部品の取替えを容易に行える再溶融性も求められる場合がある。さらに、降伏強度が低く、容易に塑性変形することで、融点から室温近傍までの冷却過程でLSIチップに過度な熱応力を発生させないという性質も重要である。適度な粘性、流動性をもち、低融点であることが望まれるため、一般には共晶合金が使用される。代表的な材料を表4・10に示す。組成別に分類すれば、Sn-Pb系、Sn系、Pb系、Au系、In系などがある。このほかに特にアルミニウムとの接続用にZn系、Sn-Zn系、Cd-Zn系材料などが使用される場合もある。これらの材料の中で最も広く使用されてきたのがSn-Pb系はんだである。共晶組成は61.9%
