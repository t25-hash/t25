# 4・3・1 磁器コンデンサ

> **章**: 第4章 電磁気・電子材料
> **節**: 4・3 強誘電体材料

磁器コンデンサ(ceramic condenser)は、多種多様な電子機器に数多く使用され、その用途によって用いられる誘電体材料が異なる。携帯電話機器やノート型パソコン用の小型チップとして図4・15に示す積層磁器コンデンサが利用されている。薄膜な磁器誘電体層と内部電極層が交互に積み重ねられ、しかも内部電極層は1枚おきに左右の外部電極に接続されている。積層磁器コンデンサの静電容量は式(4・1)で示され、は静電容量(F:ファラッド)、は内部電極の面積(m)、は磁器誘電体の厚み(m)、は内部電極の枚数、εは電気的誘電率、εは真空の誘電率(F/m,8.854×10)である。=ε×ε×-1×(4・1)なお、静電容量の単位は実用上F,pFが用いられる。小型で大容量のコンデンサを得るために、電気的誘電率が1000~10000と高い誘電体材料を用いたり、磁器誘電体層の厚みを数mから数十mと薄膜化したり、かつ積層枚数を20枚から数百枚と多層化している。現在は、積層磁器コンデンサの大きさが0603形で表現されるように0.6×0.3mmの小型化がなされている。
