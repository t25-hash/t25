# 4・3・2 圧電素子

> **章**: 第4章 電磁気・電子材料
> **節**: 4・3 強誘電体材料

圧電効果(piezoelectric effects)は、強誘電性結晶が対称中心をもたない構造で特定の方向に張力を加えると分極を生じる現象をいう。同様に圧電体結晶に分極を施し分極方向に圧縮応力を加えると圧電体結晶の両端に式(4・2)に示す起電力が生じる。=×(4・2)は電荷、は圧電(は定数)、は荷重を示す。この性質を利用して液晶バックライト用圧電インバータに用いられようとしている。代表的な圧電トランスの構造を図4・16に示す。圧電トランスは巻線トランスに比べて、構造が簡単であり、しかも薄型化が可能などの特徴をもっており、高圧発生用として適している。また、外部電界によって誘起されたひずみを利用した圧電素子(piezoelectric elements)がある。例えばカラーインクジェットプリンタのインクジェットヘッドにも圧電素子が用いられている。インクジェットヘッド用圧電素子の形状は種々あるが、積層圧電素子の縦振動モードや屈曲モードを利用したものが使われている。他の応用例としては、圧電アクチュエータによる変位検出や微小位置調図4・15 積層磁器コンデンサ図4・16 圧電トランス整、広帯域特性を有する圧電スピーカ、振動ジャイロによる角速度の計測・制御など多くの用途がある。そのほかに、携帯電話機器や自動車電話の移動体通信システムにも誘電体材料が用いられ、マイクロ波帯での使用のためにマイクロ波誘電体材料と位置付けられている。マイクロ波誘電体は、温度係数(τ)が零であり、高周波領域において誘電損失(tanδ)が低く、しかも電気的誘電率が高い材料が要求されている。代表例としてSnO-TiO,BaTiO、(Mg, Ca)TiO,BaO-TiO、(Zr, Sn)O-TiOなどの材料が用いられる。
