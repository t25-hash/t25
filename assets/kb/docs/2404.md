# 4・4・1 気体・液体絶縁材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・4 絶縁材料

気体の絶縁材料(insulating material)には六フッ化硫黄(SF)ガスが多く使用されている.SFガスは不燃性,無色,無臭で,熱安定性も優れ,500°Cまで安定である.六フッ化硫黄ガスの用途は,ガス絶縁変圧器,ガス絶縁開閉装置,ガス遮断器,ならびに電力送電の管路の絶縁媒体や消弧媒体で,使用時の圧力は0.1~6MPaである.なお,六フッ化硫黄ガスは地球温暖化係数が大きいため,使用に当たってはガスの大気中への放出は避け,回収が必要である.また,この問題を解決するため,六フッ化硫黄と窒素との混合ガスの使用検討が行われている.表4・11に各気体の特性を示す.液体絶縁材料は炭化水素系,シリコーン系,塩素化炭化水素系ならびにフッ化炭化水素系に分けられる.炭化水素系油は引火点が低い点に注意が必要である.シリコーン系油は引火点が高く難燃性で,また電気的諸特性が優れているが,動粘度が高い短所がある.塩素系炭化水素油は不燃性で比較的沸点が高い長所はあるが,毒性の面で注意を要するものがある.フッ化炭化水素系油は優れた電気絶縁性や適当な沸点範囲をもつことから,変圧器のほかに電力変換素子の沸騰冷却装置などにも用いられている.フロンR 112を30%,R 113を5%およびパークロロエチレン65%の混合油の例を表4・12の右から3列目に示す.
