砂型は最も基本的な鋳型である．砂型はすべての金属の鋳造に適
用され，鋳鉄，鋳鋼，銅合金鋳物ではほとんど砂型鋳造法である．
鋳型原料の鋳物砂は，ケイ砂を主とした砂粒に粘土鉱物，粘結剤，
添加剤に水を混合したもので，成形性をもつ．鋳型は次の条件を満
たさなければならない．
(１)適度な強さ，硬さ，通気度をもつ
(２)適当な熱伝導性をもつ
(３)溶融金属に対して十分な耐熱性をもち焼き付かない
(４)鋳造・凝固後の型ばらしの段階では良好な崩壊性をもつ
(５)反復使用が可能で安価である
(６)作業環境を悪くせず公害源とならない

b.無機自硬性鋳型（inorganic self - curing mold) 自硬性鋳
型は造型後常温に置くだけで粘結剤の化学反応により硬化する鋳型
で，水ガラス系自硬性鋳型（炭酸ガス型，CO型）は，ケイ砂に
ケイ酸ナトリウムを粘結剤として3～6%添加し型込めした後，
COガスを砂型内へ吹き込んでゲル化させて硬化させる．CO型
は崩壊性が悪いため，硬化剤としてカルシウム金属酸化物，ケイフ
ッ化ナトリウム，リン酸塩，無水硫酸ばん土なども利用されてい
る．セメント系自硬性鋳型は，粘結剤としてポルトランドセメン
ト，超速硬セメントが用いられ，ケイ砂に混合し無水鉱物の水和反
応で硬化させる鋳型である．
c .有機自硬性鋳型（organic self - curing mold) 有機粘結剤
を含むケイ砂を用いて造型し，常温で有機粘結剤と酸性あるいは塩
基性触媒の硬化剤または常温硬化樹脂と化学反応を起こさせて自然
硬化させる．鋳型砂には流動性があり，突固めが不要である．有機
粘結剤として，常温硬化性のフラン樹脂，ポリウレタン樹脂，フェ
ノール樹脂，常温ガス硬化性のポリイソシアネート樹脂，フルフリ
砂型は粘結剤の種類，造型方法によって分類される．砂型の種類
ルアルコール樹脂が用いられる．
