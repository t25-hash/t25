# 4・4・2 無機固体絶縁材料[雲母(mica)、ガラス(glass)、

> **章**: 第4章 電磁気・電子材料
> **節**: 4・4 絶縁材料

セラミックス(ceramics)]無機の固体絶縁材料は1MV 級の超高圧変電所用の長さ12mに及ぶ大型の碍管からエレクトロニクス部品の放熱基板など幅広く用いられている。碍子・碍管にはシリカ(SiO)が、成形性が良好で曲げ強度も120~140MPaと高いことから多く用いられている。冷却と絶縁の両立が必要なエンジンの点火プラグにはアルミナ(AlO)が使用されている。高温のヒータ支持体などには雲母(マイカ)が使用されている。半導体素子の冷却を兼ねた絶縁基板材料として、アルミナ、窒化アルミ(AlN)が用いられている。また、信号の高速伝送のために低誘電率が必要な用途では、ムライト(3AlO・SiO)が使用されている。表4・13におもなセラミックス材料の特性を示す。
