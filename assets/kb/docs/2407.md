# 4・5・1 磁性材料の種類

> **章**: 第4章 電磁気・電子材料
> **節**: 4・5 磁性材料

磁性材料(magnetic materials)はその性質を直流の磁化曲線(DC hysteresis loop)を測定することによって得られた特性値で表現することができる。磁性材料をその保磁力()の大きさで分類すると図4・23のように整理できる。磁性材料は軟質磁性材料(soft magnetic materials)と硬質磁性材料(hard magnetic materials)に大きく分けられる。また、軟質と硬質の境界は4kA/m(50Oe)程度といわれる。軟質磁性材料は保磁力が低く、ヒステリシス損の小さい材料で回転機、変圧器の磁心、磁気ヘッドなどに用いられる。両者の中間の保磁力()を有する材料を半硬質材料(semi-hard magnetic mate-rials)と呼び、ヒステリシスモータ、ラッチングリレーなどに用いられている。このほかに磁気テープ、磁気ディスクに用いられる磁気記録材料、書換え可能な光磁気ディスク装置に用いられる光磁気記録材料、FDD,HDD に用いられる磁気ヘッド材料、超音波発生に用いられる磁わい材料、インバなど多くの磁性材料が工業材料として使用されている。工業材料として使用される形態によっても磁性材料の分類が可能である。形態としてはバルク、薄板、薄帯、薄膜がある。バルクは溶解材と焼結材があり、薄板は圧延によって、薄帯は超急冷技術によって作製される。薄膜はスパッタ、蒸着等種々の方法によって作図4・23 保磁力による磁性材料の分類製される。材料的には金属間化合物を含む金属系とフェライト等の酸化物系に分かれる。磁性材料はFe,Co,Niを含有することが多い。
