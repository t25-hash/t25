# 4・5・2 磁心材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・5 磁性材料

磁心材料(core materials)は電流や永久磁石で発生した磁束を流すための磁気回路を構成する磁性材料である。そのために要求される磁気特性は残留磁束密度(욥)が大きく、透磁率(μ)が大きいこと、鉄損(core loss)が小さいこと、磁わい(magneto-striction)が小さいことが挙げられる。高周波用途で薄板、薄帯が用いられる場合は、機械的性質として占積率、平坦度、打抜き加工性の良いことが要求される。a. 電磁軟鉄(棒状および板)電磁軟鉄(soft magneticiron)は棒状(バルク)または板状で、電気機器の鉄心や磁気回路構成用ヨークとして用いられる。比較的低周波駆動でエディーカレントロスの小さい場合に多用される。飽和磁化が大きく、保磁力が小さく、また、小さい外部磁界で飽和する磁気特性が求められる。JIS C 2555:2000により規定された材料である。b. 電磁鋼板電磁鋼板(magnetic steel sheet,Si steelsheet)はSi添加による磁気特性の改善以来80年以上の歴史をもつFe-Si系軟磁性材料である。パワーマグネティックス機器の高能率化と小型化に必要な商用周波数における低損失および低励磁での高磁束密度が得られ、コスト的にも大変安価な工業材料である。積層して発電機、モータ、変圧器の鉄心として広く使用されている。大別して方向性ケイ素鋼板と無方向性ケイ素鋼板があり、回転機のように磁束が鉄心のあらゆる方向に通る場合は無方向性ケイ素鋼板が使用され、タービン発電機では方向性ケイ素鋼板が使用される。これらの電磁鋼板の種類、磁気特性および用途を表4・21に示す。無方向性ケイ素鋼板(non-oriented magnetic steel sheet)は表面性状、寸法精度、占積率に優れているため、打抜き製法による積層鉄心として回転機用に適している。方向性ケイ素鋼板(grain-oriented magnetic steel sheet)は粒径が数mm以上の多結晶組織で、各結晶粒は(110)씗001> 方位が圧延方向から約7°以内のずれ角で分布している(JIS G alloyのことで、35~80Ni-Fe合金の総称である。パーマロイの低保磁力0.8~0.008A/mはFeNi規則格子合金の異方性定数と磁わい定数λがともにほとんど零になることに起因する。λ~0の約80wt.%Ni-Fe合金を急冷処理して、~0にする。パーマロイはNb,Mo,Cr,Cu等の添加物によって特性が変化し、種々の材料が開発され、工業材料として用いられている。応用は磁気ヘッド用コア、シールド材、トロイダルコア等がある。表4・22に各種パーマロイの組成、磁気特性および用途を示す。Fe-3~8wt.%Al-5~11wt.%Si合金をセンダストという。Fe-Al-Si3元合金系では零およびλ零を示すラインが交わるが、本交点近傍の組成で優れた軟磁気特性を示す。センダストは規則合金FeSiやFeAlに近い金属間化合物的な性質をもち、硬くてもろい。したがって、微細組織を有する鋳造材を加工したり、センダスト粉を樹脂で結合するか、スパッタ薄膜として使用する。応用は磁気ヘッド用コア等がある。d.
