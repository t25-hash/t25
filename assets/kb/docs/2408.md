# 2・4・3 低圧鋳造法（low pressure casting process ）

> **章**: 第2章 鋳造
> **節**: 2・4 鋳造法各論

低圧鋳造法はほとんどアルミニウム合金鋳物に適用されており，
重力金型鋳造とほぼ同程度の利用度で，ダイカストを含むアルミニ
ウム合金全鋳造品生産量のうち18～20 %が低圧鋳造法による．最
近マグネシウム合金への適用が図られている．図2･44 に示す密
閉された溶湯保持炉内の溶湯表面に0.01～0.06MPaの空気圧をか
けて，溶湯中のストーク（給湯管）を通して溶湯を押し上げて上部
の金型内に鋳造し低圧力下で凝固させ，湯口部の凝固が完了した時
点で加圧を止め，ストーク内の溶湯をるつぼに戻す鋳造方式であ
る．原則的には押湯がないので鋳造品の鋳造歩留りが高く，重力金
型鋳造の40～70 %に対して，低圧鋳造では90 %以上である．砂中
子を用いた薄肉の複雑形状の鋳造ができ，内部欠陥が少なく，寸法
精度，鋳肌が優れている．加圧は3段階で，溶湯がストーク中を上
昇し湯口にまで到達する段階を第一段加圧，さらに溶湯が金型キャ
ビティ内を流動する段階を第二段加圧，溶湯の充てん完了後も加圧
して凝固収縮に対する押湯効果をもつ段階が第三段加圧である．各
段階で昇圧速度と加圧力の制御が重要である．ストークは一般には
鋳鉄製が多く，溶湯侵食防止のため塗型されるが，溶湯の鉄汚染の
問題で最近ではセラミックス製が使われている．ストークを通して
溶湯供給が押湯を兼ねるため，湯口の位置や数が制約され，重力鋳
造に比べて自由度が少ない．また，指向性凝固や湯回りを完全にす
るために金型温度が高くなる傾向があり，凝固完了までの時間がか
かり，鋳造サイクルが長くなる．そのため複数個取りや鋳造機の自
動化が進められている．近年，冷却速度を上昇させ鋳造サイクルを
上げる方法として，不活性ガスで保持炉と金型内を満たした不活性
雰囲気低圧鋳造法 や，鋳型内を減圧して溶湯をストークから吸
引して充てんする差圧吸引鋳造法 が開発されている．
