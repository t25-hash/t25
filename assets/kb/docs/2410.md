# 2・4・5 ダイカスト法（die casting process ）

> **章**: 第2章 鋳造
> **節**: 2・4 鋳造法各論

a.普通ダイカスト法 ダイカストは，精密な金型に溶湯を高
圧力を用いて高速で射出注入して薄肉部位にも溶湯を充てんさせる
大量生産の鋳造方式である．ダイカストマシンにはコールドチャン
バダイカストマシン（cold chamber die casting machine）とホッ
トチャンバダイカストマシン（hot chamber die casting machine）
がある（図2･46) ．コールドチャンバ式は，スリーブ内に注湯さ
れた溶湯をプランジャで加圧射出して金型内へ鋳造する．アルミニ
ウム合金のダイカストはすべてこの方式であり，マグネシウム合
金，黄銅にも用いられている．ホットチャンバ式は，スリーブとプ
ランジャが溶湯中に浸漬されていて，溶湯はグースネックを通して
加圧されて金型内へ鋳造される．亜鉛合金ダイカストのほとんどが
この方式であり，マグネシウム合金にも用いられる．スリーブおよ
びプランジャの溶湯による侵食のためアルミニウム合金への適用は
難しく，現在小型機が開発されている．
コールドチャンバダイカストマシンは，型締力が最小0.5MN
から最大40MN，射出力が型締力の1/20～1/40である．ダイカス
トの1サイクルは，まず金型の過熱防止のため内部が水冷され，焼
付き防止のため離型剤が塗布される．次に可動型を固定型に合わせ
て金型を閉じる．ラドルで溶湯をスリーブ内へ注湯する．注湯後た
だちにプランジャが作動し，溶湯を金型の湯道，ゲートを通して鋳
造する．ゲートは湯道より狭いので，溶湯は金型内へ高速射出され
る．溶湯の凝固後金型を開き，プランジャと可動中子を元の位置に
戻し，押出しピンを作動させて製品を金型から押出す．アルミニウ
ム合金ダイカストの場合，鋳造圧力（射出力）は10～200MPa，
溶湯のゲート通過速度は約20～100m/sで，溶湯充てん時間は約
0.01～0.7s ，金型内の溶湯の冷却速度は約30～300℃/sである．
スリーブの移動速度は，スリーブ内が溶湯で充満されるまでは
0.2～0.5m/sの低速で射出し，溶湯が金型キャビティ内に入りだ
した時点で1.5～2.5m/ sの高移動速度にする2段階方式が多くと
られている．ダイカストの1サイクルは小さい鋳造品で数秒，大き
いものでも数分であり，高速生産性をもつ鋳造法で，自動給湯装
置，自動プランジャ潤滑装置，自動離型剤スプレー装置，自動製品
取出し装置など自動化が進んでいる．中小マシンでは完全無人化も
行われている．複雑形状の鋳造品を一工程で量産が可能であり，湯
回りがよく薄肉部品の鋳造が可能，寸法精度が高い，鋳肌がきれ
い，急冷により組織が微細などの特徴が挙げられる．一方，高速射
出のためスリーブおよび金型内の空気やガスを溶湯中に巻き込んだ
まま凝固しやすく，鋳巣対策が必要である．巻き込まれた空気・ガ
スのため溶体化加熱や溶接が不可能であり，金型に急速拘束される
ので鋳造割れの対策を要する．
ホットチャンバダイカストマシンは，射出部のプランジャおよび
スリーブを含むグースネックが溶湯中に絶えず浸漬されるので，グ
ースネック材質が溶湯に侵食されない亜鉛合金，スズ合金，マグネ
シウム合金などの鋳造に用いられる．射出部が溶湯内にあり1サイ
クルごとに注湯する必要はなく，完全自動化が行われている．射出
力は7～25MPaの低圧力で済み，鋳造サイクルが速い．型締力は
0.05～10kNのマシンが使用されている．
b.新しいダイカスト法 普通ダイカスト法では高速射出充て
んに伴う空気とガスの巻込み，厚肉部のひけ巣は避けにくく，充て
ん性，気密性，機械的性質の向上，熱処理や溶接を可能にするため
の鋳造技術が考案されている．
(１)真空ダイカスト法（減圧ダイカスト法，vacuum die cast-
ing process ）
金属キャビティ内の空気やガスを溶湯射出直前に真空ポンプで吸
引し減圧状態でダイカストする方法である．GF（gas free）法，
マスベント法，カットオフピン法，VACURAL法などがある．真
空度の保持と溶湯射出のタイミングの制御が難しい．日本ではGF
法（図2･47) が多く用いられており，金型キャビティ内の空気
を短時間に大量に排気して溶湯が外部に飛び出ないエアベントバル
ブが用いられる．最近，スリーブ内も減圧して注湯も吸引すること
で減圧時間を長くしたVACURAL法 が，高品質ダイカスト鋳
造法として強度部品の製造にわが国でも実用されている．溶体化熱
処理，溶接が可能である．
(２)PFダイカスト法（無孔性ダイカスト法，pore free die
casting process )
金型およびスリーブ内空気を酸素ガスに置換して，およそ80m/
s以上の高速射出により溶湯を金型内へ噴霧状に鋳造して酸素をす
べてAlOとし鋳造品内に微細分散させ，空気，ガスの巻込みに
よる鋳巣をなくす方法で，無孔性（PF:pore free ）ダイカスト法
と呼ばれる．ガス含有量が普通，重力金型鋳造法と同程度になり，
溶体化熱処理，溶接が可能である．
