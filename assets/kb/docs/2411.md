# 4・5・5 非磁性鋼

> **章**: 第4章 電磁気・電子材料
> **節**: 4・5 磁性材料

非磁性鋼(non-magnetic steel)はオーステナイト系ステンレス(SUS)とハットフィールド鋼系Mn鋼に大別され、オーステナイト組織を有することが特徴である。電子部品などの高強度が要求されない場合は非磁性SUS が用いられ、モータ用シャフト、ブラウン管、撮像管などの電子銃のグリッド材に使用される。高Mn系非磁性鋼は磁気浮上鉄道ガイドウェイ、核融合実験炉やMHD 発電などの電磁石保持部材、大型発電機のリテイニングリングに用いられる。これらの応用では、磁気特性として低温度まで非磁性で、透磁率が低いこと、機械特性としては高強度、低温脆化のないこと、低熱膨張係数、切削性や溶接性などの加工性が良いことが要求される。表4・24に主たる非磁性材料の成分、機械特性、応用を示す。
