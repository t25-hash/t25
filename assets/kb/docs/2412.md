# 4・6・1 熱電効果(thermoelectric effect)材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・6 特殊機能材料

a. 熱電対2種の異なる金属(合金)を接合して閉回路を作り、両接点を異なる温度に置くと両接点間に温度差に応じた電位差が生じる現象を、ゼーベック効果(Seebeck effect)と呼ぶ。熱電対(thermocouples)はこの現象を利用して、一方の接点を高温に、他方を室温(または氷点)に保って温度を測定する素子である。表4・25にJIS C 1602:1995に規定される熱電対の種類、測定温度範囲を示す。貴金属系のB,R,S 熱電対はおもに1000°C以上の高温で、卑金属系のK,E,J,T 熱電対はおもに1000°C以下の低温で用いられる。このほかに2600°Cの高温測定用のW-WRe熱電対や、極低温測定用のAuFe-クロメル熱電対などがある。熱電対を用いて温度計測を行う場合には、電位差を測定するための導線として一般に補償導線(compensating lead wire)を用いる。補償導線は、組合わせて使用する熱電対の種類に応じて種類分けされている(JIS C 1610:1995)。b. 熱電材料ゼーベック効果を利用して工場、廃棄物焼却炉、発電所などからの大規模な排熱を電力に変換する熱電発電システムが、エネルギー有効利用の観点から期待されている。すでに、腕(体温)と外気温の温度差により発電された電力を駆動源とする熱電時計が、実用化されている。一方ゼーベック効果と逆に、異種の導体または半導体の接点に電流を流したときに接点でジュール熱以外に熱の発生または吸収が起きる現象を、ペルチェ効果(Peltier effect)と呼ぶ。ペルチェ効果を利用した冷却素子は、小表4・25 熱電対の種類と測定温度範囲(JIS C 1602:1995)構成材料の記号カッコ内は旧記号構成材料測定温度K階級級+脚-脚BRh30%を含むPt,Rh合金Rh6%を含むPt,Rh合金873以上1973未満0.5RRh13%を含むPt,Rh合金Pt273以上1873未満0.25SRh10%を含むPt,Rh合金Pt273以上1273未満0.4K(CA)NiおよびCrを主とした合金Niを主とした合金273以上1473未満0.7573以上273未満1.5273以上1073未満0.4E(CRC)NiおよびCrを主とした合金CuおよびNiを主とした合金273以上1273未満0.7573以上273未満1.5273以上1023未満0.4J(IC)FeCuおよびNiを主とした合金73以上1023未満0.75273以上523未満0.4T(CC)CuCuおよびNiを主とした合金273以上523未満0.7573以上273未満1.5型冷蔵庫や可搬クーラーボックスなどの民生用製品、理化学機器の温度調整装置、半導体モジュールの冷却素子、通信用半導体レーザの温度調整用素子などに広く使われている。これらの目的で用いられる熱電素子の性能は、熱電性能指数(thermoelectric figure of merit)=ασ/1/Kで表される。ここでαはゼーベック係数、σは導電率、は熱伝導率である。室温付近で最大の性能指数をもつ熱電材料はBi-Te系材料であり、性能指数が約3.0×10/K のものが工業的に用いられている。p型の(BiTe)(SbTe)系、n型の(BiTe)(BiSe)系、(BiTe)(SbTe)(SbSe)系などがある。ほかには性能指数がこれより約1けた低いFeSi系複合セラミックス材料、高温領域用のSiGe系材料、もみがらの灰から作製する多孔質SiCセラミックス材料、スクッテルダイト(skutterudite)と呼ばれる構造をもつMX系化合物(M=Co,Rh,Ir; X=P,As,Sb)などが検討されている。
