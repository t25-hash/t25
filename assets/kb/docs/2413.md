# 4・6・2 光電変換材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・6 特殊機能材料

近紫外から赤外部までの光のエネルギーは0.3~4eV 程度で、半導体中の禁止帯幅エネルギーと同等である。したがって半導体中のキャリヤ(荷電担体)がもつエネルギーが光に変換されたり、逆に光のエネルギーが半導体中のキャリヤを励起し、光電変換(photoelectric conversion)が行われる。a. 発光ダイオード半導体のpn接合に電流を流すと、価電子帯から伝導帯に励起された電子と、それと同数の価電子帯に残された正孔が生じる(注入励起法)。励起された電子は価電子帯の正孔と再結合し、このとき放出光が発生する。発光ダイオード(LED : light emitting diode)はこの現象を利用した発光素子で、光通信用の光源や家電機器、計測機器用の表示素子、屋外ディスプレイ等に用いられる。発光波長は、用いる化合物半導体の系や組成を変えることによって変わり、AlGaAs系で0.7~0.9 m,InGaAsP系で1.05~1.65mとなる。最近、ZnS,ZnSe系材料やGaN,AlGaN などの禁止帯幅が大きい材料を用いて、青色や青紫色の発光ダイオードが実現された。b. 半導体レーザ発光ダイオードと同様の構造で、自然放出で生じた光を閉じ込める光共振器をさらに備え、自然放出光が共振器内を往復する間に誘導放出を生じて強い光を放射する構造としたものが、半導体レーザである。共振器内では特定の波長が強められ、鋭い波長スペクトルとなるとともに、位相がそろったコヒーレントな光が放出される。半導体レーザは、共振器の構造によってファブリ・ペロー型(FP: Fabry-Perot)、分布帰還型(DFB : dis-tributed feedback)などに分けられる。また半導体レーザの性能を向上させる方法として、半導体層を電子の量子力学的波長程度まで薄くしてキャリヤを閉じ込めた、量子井戸レーザが用いられるようになっている。コンパクトディスク用の光源としてGaAs基板上のAlGaAs材料による0.78~0.88 m帯レーザが使われ、DVD 用にはGaAs基板上のInGaAlP材料による0.63~0.68 m帯レーザが使われる。また光通信用の光源としてInP基板上のGaInAsP材料による1.1~1.6 m帯レーザが用いられ、光ファイバ増幅器の励起用にGaAs基板上のGaInAsP材料による0.98 mレーザが用いられる。最近、データ記録用に波長の短い緑色あるいは青色レーザが開発され、注目を浴びている。用いられる材料は、ZnSe系やGaN系化合物半導体である。c. 太陽電池半導体に禁止帯幅より大きなエネルギーの光が照射されると、価電子帯の電子が伝導帯に励起され、伝導帯に正孔が生成される。こうして生じたキャリヤにより電気伝導度が増大する現象を、光伝導効果(photoconductive effect)と呼ぶ。半導体中の内部電界が存在する場所に光が照射されて電子-正孔対が生成されると、電子と正孔は互いに逆の方向に動いて分離され、半導体の両側には電位差が生じる。この現象を光起電力効果(photovoltaic effect)と呼ぶ。太陽電池は、pn接合による接触電位差を内部電界として用い、太陽光エネルギーを光起電力効果により直接電気エネルギーに変換する。太陽電池によって入射光エネルギーが電気エネルギーに変換される割合を、エネルギー変換効率と呼ぶ。理論限界効率を考えると、禁止帯幅が狭いほうが太陽光をより多く吸収してキャリヤを形成でき、逆に禁止帯幅が広いほうが内部電位を大きくできるため、最適な禁止帯幅エネルギーが存在する。最適禁止帯幅エネルギー近傍の半導体材料は、結晶系Si(1.10eV)、CuIn(Ga)Se(1.10~1.35eV)、InP(1.35eV)、GaAs(1.43eV)、CdTe(1.52eV)、アモルファスSi(~1.7eV)などである。地上用平面型太陽電池として最も普及している単結晶および多結晶Si太陽電池では、それぞれ、ほぼ24%および19%の変換効率が達成されている。GaAs太陽電池では理論限界の28.5%に迫る変換効率が達成されているが、材料が高価なため、小面積の集光型や耐放射線性が必要な宇宙用として使われている。太陽光スペクトルの広い範囲を利用するために、禁止帯幅の異なる太陽電池を複数個接続したタンデム構造が用いられる。低コスト太陽電池が作製可能なアモルファスSiとアモルファスSiGeとを組合わせたタンデム型太陽電池により、15%の変換効率が得られている。d. フォトダイオードフォトダイオードは半導体pn接合における光起電力効果を利用して光信号を電気信号に変える受光デバイスである。感度が高い、応答速度が速い、雑音が小さくS/N 比が高いなどの特徴をもち、広く利用されている。Si,Ge,InSb,InAsなどの半導体材料がフォトダイオードに用いられる。
