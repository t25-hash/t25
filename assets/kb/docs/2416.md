# 4・6・5 調節、保安材料

> **章**: 第4章 電磁気・電子材料
> **節**: 4・6 特殊機能材料

a. バイメタル材料バイメタル(bimetal)は熱膨張係数の異なる金属板を張り合わせたもので、温度が変化したときに両者の伸縮の度合いの違いによりたわみが生じるので、温度を検出して回路を遮断または接続する素子として用いられる。バイメタル材料として用いられるのは、36Fe-Ni系合金のインバなどの低熱膨張材料と、MnまたはCrを添加したFe-Ni系合金やCu-Zn系合金などの高熱膨張材料である。b. ヒューズ材料ヒューズ(fuse)は過電流によりエレメントが自己発熱して溶断し、回路を遮断する過電流保護装置である。エレメントは遮断電流定格により線エレメント、または切欠や穴を加工した薄い帯状のリボンエレメントが用いられる。エレメント材料には、銀、銅、亜鉛、スズ、鉛などの純金属やそれらの合金が用いられる。c. 避雷器材料避雷器は発変電所や送電系統、配電系統において電路と大地の間に設置され、落雷や電路の開閉により系統に発生する過電圧を抑制して、設備の絶縁破壊を防止する保護装置である。通常は絶縁状態にあり、系統に過電圧が発生したときのみ地絡状態となってサージ電流を大地に流し、過電圧消滅後は元の絶縁状態に復帰する。このような動作をさせるために、特性要素と呼ばれる内部素子には電圧によって抵抗値が大きく変化する非直線抵抗体2―137(varistor、バリスタ)が用いられる。特性要素にはかつては炭化ケイ素(SiC)が用いられていたが、1970年代からは酸化亜鉛(ZnO)素子が主流になった。図4・24にZnO素子の電圧電流特性の例を示す。動作時には電圧が例えば1.6倍変化することにより電流が1000万倍流れるようになる。このような特性は、ZnOにBi,Co,Mn,Sb等の酸化物を少量添加したセラミック焼結体によって得られる。ZnOの結晶は半導体であるが、この焼結体は多数の結晶粒子からなり、その粒界領域に添加物に起因して伝導電子に対するポテンシャル障壁が形成され、これが高い非直線性をもたらす。文献(1) 岡崎清、セラミックス誘電体工学、(1992)、8、学献社。(2) 川崎雅司・ほか2名編、 セラミックス薄膜の原子・分子レベル制御技術とデバイス、(2000)、89、ティー・アイ・シー。(3) 平井平八郎・ほか3名編、電気・電子材料、(1997)、7、オーム社。(4) 塩嵜忠・ほか3名編、強誘電体薄膜メモリ、(1995)、11、サイエンスフォーラム。(5) 山本博孝、セラミックス電子部品・材料の技術開発、(2000)、9、シーエムシー。(6) 文献(3)の8ページ。(7) 文献(5)の65ページ。(8) 塩嵜忠・ほか2名、 注目の誘電体セラミックス材料、 (1999)、 328、ティー・アイ・シー。(9) 文献(5)の68ページ。(10) 文献(8)の393ページ。(11) 福長脩、機能性セラミックス、(1989)、88、日本ファインセラミックス協会。(12) 電気学会編、新版電気工学ハンドブック、(1988)、229、電気学会。(13) 岩波理化学辞典、第3版、(1972)、834、岩波書店。(14) 理科年表、(1967)、物29、丸善。(15) 石油学会編、電気絶縁油ハンドブック、(1987)、26、講談社。(16) 宗宮重行編、アドバンストセラミックス、(1984)、73、講談社。(17) 文献(16)の22ページ。(18) 電気学会技術報告(II部)、342(1990)、29、電気学会。(19) 由井浩、複合プラスチックの材料設計、(1992)、78、プラスチックスエージ。(20) Nishimura,A.、ほか2名、Proc.
