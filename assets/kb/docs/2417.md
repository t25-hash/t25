# 5・1・1 プラスチックの種類

> **章**: 第5章 高分子材料ほか
> **節**: 5・1 プラスチック

プラスチックを分類すると、それを構成する高分子の性質から熱可塑性プラスチック(thermoplastics)と、熱硬化性プラスチック(thermosetting plastics)とに大別される。前者は温度が上がると軟化溶融して流動性を示し、冷えると固化する(可逆的変化)。後者は液状ないし粉末状の主材が加熱ないし硬化剤(架橋剤)との反応によって、三次元網状構造を形成するもので、不可逆的変化を呈する。後出の表5・2(140~141ページ)に、成形品として実用化されている主要なプラスチックを一覧するが、表5・2の前半(セルロース系まで)が熱可塑性プラスチックであり、後半(フェノール樹脂以降)が熱硬化性プラスチックである。プラスチックはまた結晶性(crystalline)と非晶性(amor-phous)に分けられる。結晶性プラスチックの例としてポリエチレンとポリプロピレンがある。高分子における結晶構造は、高分子鎖が厚み方向に向いて並び、表面に達すると折り畳まれてまた内部に入っていくという、折り畳み構造である。結晶性プラスチックが金属やセラミックスの結晶と異なる点は、完全には結晶ではなく、必ず非晶部を伴うことである。全体積に占める結晶部の割合を結晶化度といい、この値はプラスチックの種類や合成法によって大きく異なる。ポリエチレンの場合で45~90%である。ポリエチレンのように分子構造が規則的なものは結晶化しやすく、大きな側鎖をもつようなものは結晶化しにくい。非晶性プラスチックとは分子が全く不規則に並んだもので、ポリスチレンやポリメチルメタアクリレートなどが代表的なものである。熱硬化性樹脂はすべて非晶性プラスチックである。プラスチックのもう一つの分類法は、実用上から分けるもので、はん用プラスチック、エンジニアリングプラスチック(engineer-ing plastics)、特殊な高級プラスチックなどに分類される。はん用プラスチックとは、ポリエチレン、塩化ビニル樹脂など材料価格が低く、成形しやすく、一般用途に多量に使用されるものを指し、プラスチックの全生産量の約9割を占めている。エンジニアリングプラスチックとは、100°C以上でも室温での寸法安定性と大部分の機械的性質を保持するような熱可塑性プラスチックをいう。ポリアセタール、ナイロン(ポリアミド)、ポリカーボネート、ポリブチレンテレフタレート(PBT 樹脂)などのように、強靱で剛性が大きく力を受ける用途にも使いうるプラスチックがそれに当たる。実用化されているエンジニアリングプラスチックを列挙すると、上記した4材種のほかにPPO樹脂、ポリサルホン、ポリエーテルサルホン、ポリエチレンテレフタレート(テトロン樹脂)、PPS 樹脂、フェノールアラルキル、芳香族ポリエステル(エコノール)、ポリフェニレンエーテルなどがあるが、定義を拡大して、高密度ポリエチレン、ポリプロピレン、硬質塩化ビニル樹脂、メタクリル樹脂(アクリル樹脂)、ABS 樹脂、フェノール樹脂、不飽和ポリエステル樹脂、エポキシ樹脂、ジアリルフタレート樹脂、シリコーン樹脂、熱可塑性ポリウレタンやそれらのガラス繊維強化グレードを含めることもある。特殊な高級プラスチックとしては、フッ素樹脂、シリコーン樹脂、ポリイミドなどに代表される、高コストながら耐熱性などの特性が卓越している材料が含まれる。以上示したように、プラスチックの材種は非常に多く、工業化されているものだけで約40種に及んでいる。しかもそれぞれの材種について、平均分子量の差異、結晶化度の差異、充てん物の種類と含有率の違いなどにより、数多くのグレードがある。一例を挙げると、ポリカーボネートの場合、成形材料の全銘柄は国産および国内で販売している輸入品を含めて160種を超えている。さらに数量がまとまれば、要求に応じた特注グレードも製造される。プラスチックはいわゆる成形品のほかにも、フィルム、シート、パイプ、ロッド、フォーム、積層品、ライニング、コーティングなどいろいろな形態で利用され、プラスチック系複合材料の母材としても有用である。
