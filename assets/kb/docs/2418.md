# 5・1・2 プラスチックの成形方法

> **章**: 第5章 高分子材料ほか
> **節**: 5・1 プラスチック

プラスチック材料の利点の一つは、概して量産成形に適していることである。そして微小精密部品から、バスタブなどの大型製品まで寸法は広範囲に適合し、瞬間的に成形できる射出成形法など10種類以上の成形方法がある。ここでは代表的なものについて概説する。a. 圧縮成形法(compression molding)プラスチックのすべての材種に適用できるもので、最も古くから用いられ、熱硬化性プラスチック等については現在でもしばしば適用されている成形法である。プレスと金型を用いて、加圧、賦形、加熱によって成形する。熱硬化性プラスチックの場合には硬化反応を金型内で生じさせるため、熱可塑性プラスチックの射出成形法などに比べて成形サイクルが長くなる。この方法の適用される分野は、フェノール樹脂ほかの熱硬化性プラスチックのすべてをはじめ、熱可塑性プラスチックを用いるスタンピング成形、プリント配線基板の積層、また複合材料に属するBMC,SMC(ともに不飽和ポリエステル系)などである。成形機としては、通常のプレスのほか、複動式のものや全自動式、ロータリ式などの進歩が目立ち、成形の高速化、自動化、精密化を指向している。b. トランスファ成形法(transfer molding)圧縮成形法と射出成形法との中間的な方法であり、熱硬化性樹脂用の成形法で、最近は特に低圧、液体成形に適用され、電子部品のパッケージングなどが代表例である。プランジャとポットが金型とは別に装備され、加熱軟化した成形材料ないし液体材料をノズルから金型内へ押込み、金型内で硬化させて成形する。圧縮成形法よりサイクルが速く、寸法精度もより優れている。移送成形法ともいう。c. 射出成形法(injection molding)熱可塑性プラスチックの代表的な成形法であり、成形サイクルが極めて短く、寸法精度も高いため、量産に適する。ペレット(粒)状などの成形材料をホッパから連続的に供給し、射出用シリンダ内で加熱溶融し、スクリューないしプランジャによって固く閉じた金型の中にノズルを通して高速で圧入(射出)し、冷却固化して割り型を離すことによって成形品を取出すという方式のものである(図5・1)。熱硬化性プラスチック用の射出成形では、金型内での冷却固化の代わりに、加熱して硬化反応を起こさせるため、通常、熱可塑性プラスチックに比べて成形時間が長くなる。最も量産向きの成形法であり普及しているが、成形条件の管理が重要であり、特に射出圧力、シリンダ温度(樹脂温度)、金型温度、冷却時間、射出速度、スクリューの回転数と背圧、型締力などの適正な設定が必要である。したがって、コンピュータによる成形条件の最適設計、自動制御が急速に進歩し、成形機の自動化、省エネルギー化、省力化、精密成形化などの進歩も目覚しい。高温で加水分解を生じる傾向をもつ材種(ポリアミド、ポリカーボネート、ポリエステルなど)の場合には、成形直前の成形材料の十分な乾燥が不可欠である。ポリカーボネートの場合、成形条件の不適正、特に樹脂の乾燥不十分が、疲労強度やソルベントストレスクラッキングに与える悪影響が顕著であることが報告されているので注意を要する。また、結晶性プラスチック(ポリアミド、ポリアセタール、ポリエチレン、ポリプロピレンなど)の場合には、金型温度や冷却時間が結晶化に影響を与えるので管理が重視される。d. 押出成形法(extrusion molding)量産成形法の一つで、断面が一様な長尺製品に適する方法であり、熱可塑性プラスチックに限られる。押出機(extruder)を用いて、加熱溶融した樹脂をダイ(die)から押出すことによって成形する。棒(ロッド)、管(パイプ)、型材、シート、フィルム(T ダイ使用)、被覆電線、ライニング、樹脂被膜鋼管など広く適用されている。断面は円形や長方形など対称形が多いが、異形断面のものも生産されている。いずれも数量がまとまらないと、価格上有利とならないが、射出成形ほど樹脂の流動性は必要としない。e. ブロー成形法(blow molding)加熱した熱可塑性プラスチックを押出機からチューブ状(パリソン)に押出し、熱間で割り型の金型内にパリソンを封じ込め、上下両端をシールして空気吹込みを行い、パリソンを金型の内面に向けて膨張させ、中空成形品を得る成形法である。パリソンを射出機を用いて作る方法や、2枚のシートを連続的に押出してブロー成形用の金型に挟み込む方法などがある。f. 注型法(casting)液状樹脂など流動状の材料を、開放金型の中や面の上に流して固化する方法のことで、エポキシ樹脂、不飽和ポリエステル樹脂など熱硬化性樹脂や、ポリウレタンプレポリマ、カプロラクタムなどのモノマ、メタクリル樹脂、アクリル樹脂などのエマルジョンが対象となる。注型の際に他の固体を封入することもできる(封入注型と呼ぶ)が、電子回路のときは特にポッティング(potting)ともいう。メタクリル樹脂の場合、押出成形板と比べ注型板は美観に優れるため高級用途に適用される。固化させるには、溶剤の蒸発と硬化反応によるものとがあり、常温による場合と加熱を要する場合とがある。g. 発泡成形法(expansion molding)狭義の発泡成形法は、金型内に発泡性ペレットや発泡剤を混入した材料を充てんし、加熱して金型内で発泡させて成形する方法を指す。広義にはプラスチックフォームの発泡方法のすべてを包含する。気体混入法(PVCや各種ゴム)、分解型発泡剤法(架橋PE,PVC,PP,PS,ABS、各種ゴム)、蒸発型発泡剤法(PE,PP,PS など)、化学反応法(軟質、硬質のポリウレタン樹脂)その他がある。これらの中には、ストラクチュラルフォームを作る射出発泡法も含まれ、PE,PP,PS,ABS などに適用され、準構造材などに応用されている。発泡の倍率により、低発泡品(3倍以下)と高発泡品(10倍以上)およびその中間発泡品とに大別される。RIM 成形(reaction injection molding)とは、2液を噴射混合し、瞬間的に金型内に吐出する一種の射出成形であり、噴射混合に図5・1スクリュープランジャ射出成形機の断面図2―139よる硬化反応を伴う。代表例はポリウレタンの低発泡品であり、自動車のバンパの一体成形などが実用化されている。ガラス繊維を混入する方式(R-RIM)も進歩している。
