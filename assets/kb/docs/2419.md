# 5・1・3 性質

> **章**: 第5章 高分子材料ほか
> **節**: 5・1 プラスチック

プラスチックの性質は原料高分子の製造-成形-後処理に至る各プロセス、さらに使用環境によって著しく影響を受ける。したがって、ここでは代表的な性質について具体的な特性値を示してあるが、それらはあくまでも一測定例として参考に供される性格のものである。a. 比熱(specific heat)、線膨張率(thermal expansioncoefficient)、熱伝導率(thermal conductivity)表5・1に代表的プラスチックの熱特性を示す。一般に金属に比べて比熱、熱線膨張率は大きく、熱伝導率は低い。また、環境温度、分子配向、結晶化度などの影響を受ける。これらはプロセスの最適化や無機物などの異種材料との複合化で重要となる。b. 比重(specific gravity)、静的強度(static strength)表ASTM 規格に基づいた測定値を示す。透明性は参考として挙げたが、結晶の大きさや結晶化度、充てん材の配合により透明性は異なる。また一般に、使用中の劣化や表面きずにより低下していく。比重も結晶相の状態や充てん材の種類・量によって変化する。引張強さは原則として破断応力で示し、一部は降伏応力で示してある。なお、ポリアミドでdryとあるのは絶乾状態(水分0%)での測定値である。この樹脂は親水性で、測定には状態調節や調湿処理が必要である。伸び率は破断時の標線間変位から求めた値であり、縦弾性率はASTM-D 638に基づいた測定値である。圧縮強さ、曲げ強さはおのおのASTM-D 695,D 790によっている。硬さは原則としてASTM-D 785によるロックウェル硬さで示してあり、R,M,Eはスケールの種別を表す。一部はショア硬さ(表5・2中◯S印、ASTM-D 2240)やバーコル硬さ(◯B印、ASTM-D クリープ特性(creep property)プラスチックのクリープは低負荷でも応力緩和とともに静的粘弾性の一特性として高温になるほど大きく現れ、特に熱可塑性プラスチックで著しい。荷重の増大とともに遅延弾性から粘性流動に支配機構が変わるとされ、クレーズやクラックが関与することもある。図5・4に数種類のプラスチックのクリープ量(20°C,100時間後)を示す。クリープにおいても温度-時間換算性(temperature-time super-position)が見られ、負荷応力の対数を縦軸に、寿命の対数を横軸にとって異なる温度での応力-寿命曲線を水平移動するとクリープマスタカーブが描ける。これは金属材料のラーソン・ミラー法に対応し、ある温度でのクリープ寿命推定に有用である。f. 疲労特性(fatigue property)図5・5に数種類のプラスチック平滑試料で得られたS-N 曲線(S-N property)プラスチックは一般に電気絶縁性および誘電特性に優れ、電波透過性をもつが、耐電圧は低い。また、帯電性があることは加工作業の妨害やほこりの吸着などの障害の原因になる。絶縁性の尺度である体積抵抗率は通常10Ω・cm以上であり、ポリエチレンのように10Ω・cm以上の高い値を示すものもある。絶縁破壊強度は20kV/mmのものが多いが、それ以上を示す樹脂(ポリ塩化ビニル、ポリアセタール、架橋ポリエチレン)もある。誘電率は2~10程度であり、誘電損失は0.01以下のものが多いが、ポリ塩化ビニルやポリアミドでは大きく、オレフィン系樹脂やフッ素樹脂では0.005以下と極めて小さい。プラスチックに炭素粉末などを充てんすると導電性を付与することができる。さらには、ポリアセチレン、ポリピロール、ポリ(-フェニレン)などに微量のハロゲン(Br,Iなど)、ルイス酸(FeCl,SOなど)をドープすることにより原料高分子自体に導電性をもたせたプラスチックが開発され、大きな発展が期待されている。h. 化学的特性(chemical property)高温の水蒸気、アンモニア、塩素、オゾンなどは短時間の作用でも多くのプラスチックを変質させ、高温空気中では酸化反応による劣化も起きる。液体では水が重要で、その影響はポリアミドのような親水性樹脂で大きいが、あまり吸水しないメタクリル樹脂などでも水の浸入により寸法や物性が変化する(量に応じて可塑化や脆化が起きる)。さらに、高温水中や酸あるいはアルカリの介在下ではポリカーボネート、ポリエステル、ポリウレタンなどは加水分解により劣化する。ただし、一般には金属に比べて酸やアルカリに対する抵抗性は大きい。一方、有機液体(蒸気)には弱く、膨潤や溶解が起きたり、環境応力き裂(ESC: environmental stress cracking)を生じたりする。ESCは無負荷では何の変化もないのに、引張応力の印加や残留により液体(蒸気)と接触している表面にクレーズやき裂が生じる一種の遅れ破壊現象で、成形品のトラブル原因の約30%を占めるといわれる。このような環境下での使用では事前評価が必要であり、プラスチックと溶解度パラメータが近い液体(有機溶剤)や分子容積が小さく拡散しやすい液体、あるいは界面活性剤を含む環境では特に注意が必要である。なお、固体環境でも銅との接触でポリプロピレンの劣化が進んだ例があるので、配慮は必要であろう。他の化学的劣化因子としては光(紫外線)があり、化学結合の切断による変退色、失透、脆化などを呈する傾向がある。この光劣化に対してはカーボンブラックなどの配合が有効とされている。屋外使用では光のほかに水分や温度が複合的に作用するので、このような気象条件に対する耐久性(耐候性という)を事前に評価しておく必要がある。参考に高分子の劣化に関する文献(14)を挙げておく。i. 燃焼性(burning property)プラスチックは不燃ではないが、難着火性や自己消火性をもつ材種はある。また、表面処理や難燃剤(無機化合物、ハロゲン化合物、リン化合物)の配合による燃焼性の改善も行われている。プラスチックの燃焼性の評価にはUL-94やASTM-D 635のほか酸素指数法(ASTM-2863-70)など用途に応じた種々の規格がある。UL-94ではHB,V-2,V-1,V-0の順により高い難燃性となり、ポリアセタール、ポリエチレンはHB、難燃剤入りのポリプロピレンはV-1、ポリ塩化ビニルはV-0である。ULの認定は特にアメリカ向け電気製品の使用材料などで重視される。
