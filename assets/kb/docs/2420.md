# 2・6・4 各種部品と使用材質例

> **章**: 第2章 鋳造
> **節**: 2・6 鋳造品の種類

JIS規格では百数十に及ぶ鋳造材料が規定され，膨大な数の鋳造
部材・鋳造部品が製造されている．鋳造品の使用材料は用途により
要求される形状・寸法，被削性，強度，使用環境，コストなどとと
もに経験的事例を参考にして選択される．このため，同一用途部品
であっても異なる材料が選択されることも少なくない．表2･16に
各種機械などに使用される鋳造品の用途と適用材料の一例を示す
が，同様の部品に対してさまざまな材料が使用されていることには
留意されたい．
