# 5・1・5 その他のプラスチック

> **章**: 第5章 高分子材料ほか
> **節**: 5・2 に成形品として実用されている主要なプラスチックについて

a. ポリマアロイ(polymer alloy)ポリマアロイとは多成分高分子の総称で,アロイ化技術で大別すると,(1)相溶系の均一混合,(2)半相溶系の熱処理(スピノーダル分解),(3)ブロック・グラフト共重合体,(4)非相溶系の機械的混練,(5)相互浸入網目(IPN)化により形成される.これらのうちおもに(3)と(4)によるものをポリマブレンド(polymer blend)と呼ぶ.これらでは各成分高分子が異なる相(領域)を作ってミクロ相分離することで海-島構造(sea-island structure)を形成しており,通常それら成2―145分の共重合体を相溶化剤(コンパティビライザ,compatibilizer)として添加することにより各相を連結させる.代表的なものにスチレンとポリブタジエンからなる耐衝撃性ポリスチレン(HIPS)やスチレン-アクリロニトリル混合物とポリブタジエンからなるABS樹脂があり,ポリブタジエン相は内部にも他成分相が分散したサラミ構造となって直径数m以下の微粒子が散在している.これらが引張応力を受けるとポリブタジエン相の赤道界面から周囲に向かってクレーズが形成され,高靱性・耐衝撃性に寄与する.他のブレンド材でせん断流動が高靱化機構となるものもある.また,相分離構造では粘弾性の温度分散も各成分に応じた温度で別々に現れるので,低温(-85°C付近)でもポリブタジエン相のエネルギー散逸により耐衝撃性が発現する.ただし,疲労においてはポリブタジエン相界面からのクレーズ形成はむしろ寿命低下をもたらすので注意が必要である.b. 粒子分散プラスチック系複合材料(particle dispersed poly-mer composites)エポキシなどの熱硬化性樹脂に直径数m程度のゴムの微粒子を分散させることにより靱性を高めることができる.HIPS などのクレーズ形成機構とは異なり,ゴム微粒子自体のキャビテーションや肥大化による周囲樹脂の塑性変形,ゴム微粒子によるき裂の捕捉が主要機構といわれる.エポキシ樹脂に対するゴムとしてはCTBN(カルボキシル基末端ブタジエンアクリロニトリル),ATBN(アミン基末端ブタジエンアクリロニトリル),ブチルアクリレートなどが使われ,20wt.%程度のゴム濃度が破壊靱性を最も高めるとされている.熱可塑性プラスチックにガラス微粒子を分散させても,強度は期待できないが,弾性率(剛性),硬さ,耐摩耗性を高めることができる.この場合のクレーズ形成は粒子の極点付近から起きる.c. 生分解性プラスチック(biodegradable polymer)微生物により分解され,自然的副産物(炭酸ガス,メタン,水など)のみを生じるプラスチックを生分解性プラスチック(グリーンプラスチック)という.廃棄における問題が少ないことから使い捨て容器のほか,外科手術用縫合糸,薬物および農薬の徐放担体,農業用敷きわら代替物などへの適用が期待されている.その構造上の特徴は高分子鎖が加水分解容易な結合を骨格にもち,しかも十分に長く発達していることで,ポリエステル(特にポリオキシ酸),ポリエステルをベースとしたポリウレタン,ε-アミノヘキサン酸のオリゴマーなどが知られている.その他に炭素骨格を有するプラスチックでも容易に酸化される官能基をもつポリビニルアルコール,ポリ酢酸ビニル,ポリアクリル酸エステルには生分解性がある.生分解性試験としては,土壌中への埋込み試験,培養微生物あるいは酵素標示による分解試験,血清中での分解試験,生体内埋植試験がある.
