# 2・7・2 溶解設備

> **章**: 第2章 鋳造
> **節**: 2・7 鋳造工場設備

金属の溶解設備には連続式とバッチ式があり，前者にはコークス
（乾留石炭）を用いるキュポラがある．後者には電気炉やガス炉が
ある．後者の場合も溶湯の一部使用により，半連続溶解がなされて
いる．
a.キュポラ（cupola) 鋳鉄溶解用の円筒形の炉で，小型の
溶鉱炉ともいえる．上方からコークス・地金・石灰石を投入し，下
方の羽口から空気を吹き込んでコークスを燃焼させ地金を溶解し，
溶湯は出湯口から取出す．鋳鉄は1200℃程度で溶解するが，注湯
時の温度低下を考え，また高温溶解が品質のよい溶湯を得ることか
ら1450～1500℃の溶湯を得るように，コークス量や風量を選ぶ．
キュポラの上方から排出されるガスの温度は400～800℃くらい
でCOガスを10 %以上含有しているので，このガスを燃焼させ熱
交換器によってキュポラに送る空気を予熱する方法をとることで
（熱風キュポラ)，コークス量の低下，出湯温度の上昇を図ることが
できる．しかし，熱風を用いると炉内温度が高くなり炉壁の寿命が
低下するので，外壁に水を流して冷却する方法（熱風・水冷キュポ
ラ）がとられる．
一方，排ガス中には粉じん・ばい煙が大量に含まれていて，これ
を除去するために集じん装置を設ける．排ガスの温度は高いのでバ
ッグフィルタに直接入れることを避けて途中に冷却塔を設け，温度
を下げて処理する．
b.低周波誘導電気炉（low frequency induction furnace)
炉体のまわりにコイルが巻いてあり，それに電流を流すことによっ
て炉内の溶解金属中に誘導電流を生じさせ，ジュール熱によって溶
解するものである．るつぼ型炉と溝型炉とがあり，商用周波数，3
倍周波数が用いられる．
るつぼ型の炉体のまわりには水冷銅管があり，その外側に磁束の
漏えい防止と耐火物の膨張を抑えるための継鉄があり，これらが型
鋼のフレームに収納されている．溶解の始めには地金を塊にしたス
ターティングブロックを炉内に入れてコイルに通電する．誘導電流
により地金の温度が上昇し溶解するので，通常の地金を挿入して溶
解量を増加させる．溶湯を所定の温度に上げてとりべにとり鋳型に
注湯する．連続して溶解する場合には溶湯の3分の1から2分の1
を残し，次の地金を挿入して溶かすようにする．このほうが電気量
が少なくて済み，省エネルギーになる．炉の容量は1回に保持しう
る量で示す．大型では30t ，40tのものがある．溝型炉は溝状の湯
だまり部のまわりにコイルおよび鉄心がある．コイルに電流を流す
と溝内の溶湯がジュール熱によって加熱され炉内を循環する．るつ
ぼ型炉より構造が複雑で溝が狭いため地金を挿入するには適さな
い．したがって，溶湯を最初から入れ，保温・昇温するのに適して
いる．熱効率はこのほうがよい．非鉄金属・鋳鉄の溶解・保温に用
いられる．
c .高周波誘導電気炉（high frequency induction furnace)
原理は低周波誘導電気炉と同様であるが，500Hz以上の周波数を
使用する．高温にすることが可能で，鋼の溶解も容易にできる．溶
解温度に短時間（30～150min）で達し，溶湯のかくはんが自然に
行われるため他の元素を加える場合，特にかくはんする必要がな
い．しかしスラグを浮かしたい場合に溶湯を静止することができ
ず，不都合な場合もある．通常容量は2～3t程度である．
d.アーク炉（arc furnace) 電気アーク熱によって金属を溶
解する炉で，鋼や銅合金の溶解に用いられる．エルー式電気アーク
炉は銅の溶解に多く用いられる．耐火物がマグネシアの塩基性アー
ク炉，ケイ砂の酸性アーク炉がある．塩基性炉では鋼中のリンや硫
黄の除去が可能であるが酸性炉では不可能であるため，後者は高級
スクラップの場合に用いられる．わが国では塩基性炉が多い．
e .るつぼ炉（crucible furnace) るつぼ炉はるつぼの中に地
金を入れ，重油・ガスバーナなどで溶解する炉である．るつぼが炉
に固定されているもののほか，るつぼを燃焼室内に入れて溶解を行
い取出して用いるものもある．銅合金や軽合金の溶解に用い，るつ
いる．
ぼのふたをかぶせて燃焼ガスが溶湯に触れないようにする．るつぼ
には鉄に耐火ライニングを施したものや黒鉛製のものを用いる．鋳
鉄用には横型の回転式のものも用いられる．
f .注湯設備 通常は溶解炉からのとりべに小分けして鋳型に
く，型の締まりもよい．
注湯するが，高熱作業であるため，自動化されている．大容量のも
のでは加熱保温装置を備えたものもある．
