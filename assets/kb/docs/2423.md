# 5・2・1 ゴム(rubber)材料一般

> **章**: 第5章 高分子材料ほか
> **節**: 5・2 ゴム

ゴム材料は,柔軟で破断伸びが大きく圧縮永久ひずみが小さいことが特徴である.繰返し大変形を受けても破断せず,応力を取り除けば原形状に回復することが要求される製品部位に用いられ,その物性は表5・4に示すように,金属その他の一般材料の弾性率と著しく異なり,ポアソン比が約0.5,弾性率は1~10MPaの極めて小さい値を示す.加硫ゴム(vulcanized rubber)は,融点が低く室温で液状である長鎖状の高分子を,ところどころで架橋し,三次元網状構造とすることにより,変形に耐えうる柔軟性と流動を抑える形状保持性をもたせたものである.ゴム弾性理論によれば,加硫ゴムの架橋密度と弾性率の関係は式(5・1)で与えられ,ゴムの架橋密度が高いほど高弾性である.=3ν(5・1)ここで,は引張弾性率,νは架橋密度,はボルツマン定数,は絶対温度である.一般の固体の弾性が分子間隔の変化に対する内部エネルギーによるものであるのに対し,ゴムの弾性は分子の熱運動によるエントロピー弾性であり,温度の上昇に伴い増加する特徴がある.通常のゴム材料では,補強剤や樹脂等によるエンロトピー弾性以外の補強効果が加わっているため,必ずしも温度上昇に伴う弾性率の増加が見られるとは限らないが,輪ゴムのようにほとんど天然ゴムからなる製品では,弾性率が絶対温度に比例する現象が観察される.一般の工業用製品として使用されるゴム材料は,原料ゴムのままで使用されることはなく,使用目的に応じた物性とするため加硫剤や粉体補強剤等の添加剤が加えられ,それを加圧加熱して化学的に架橋された加硫ゴムが使用される.添加剤の選択は使用目的によるが,基本的には,表5・5に示すようなものが用いられる.原料ゴムはかつて天然ゴム(natural rubber)のみであったが,現在は多くの品種の合成ゴム(synthetic rubber)が開発されている.化学構造と架橋構造によりゴムの物理的,化学的性質が大きく異なり,はん用ゴム(general purpose rubber)とそれ以外の特殊ゴム(special purpose rubber)に大別される.前者は,強度と加工性に優れ生産量が多く安価な,天然ゴム(NR),シスポリイソ表5・4 各種材料のヤング率,引張強さ,ポアソン比ヤング率GPa引張強さMPaポアソン比軟鉄0.28アルミニウム0.33ガラス0.23ポリスチレン3.40.33ポリエチレン0.240.38ゴム0.0020.492―148材料学・工業材料プレンゴム(IR),ポリブタジエンゴム(BR),スチレンブタジエンゴム(SBR),ブチルゴム(IIR),エチレンプロピレンゴム(EPDM)を指し,タイヤを中心として使用される.後者はそれ以外の原料ゴムの総称で,耐油性に優れたニトリルゴム(NBR),耐候性,耐オゾン性に優れたクロロプレンゴム(CR)のようにはん用ゴムにない性質を重視する目的で使用され,アクリルゴム(ACM,ANM),ウレタンゴム(U),シリコーンゴム(MQ),フッ素ゴム(FKM)等がある.なお,IIR とEPDM は特殊ゴムに分類されることがある.これらの概略特性を表5・6に示す.原料ゴムのブレンドや添加剤の選択により,より優れた特性を発揮させることが可能であり,材料設計に当たり,用途に適した組合せを考えることが必要である.
