# 5・3・1 木材(wood)

> **章**: 第5章 高分子材料ほか
> **節**: 5・3 木質材料(wood-based materials)

a. 木材の構造木材の大部分は樹軸とほぼ平行に配列する繊維状の細胞で構成され、針葉樹では約90%が仮道管、広葉樹では70~90%が木部繊維(真正木繊維、繊維状仮道管など)と道管要素で占められる。樹軸と平行に配列する他の細胞には、軸方向柔細胞などがあるが量的に少ない。一方、樹軸と直交する方向に配列する細胞には放射柔細胞などがあるが、木材全体に占める割合は一般に10%以下である。針葉樹の仮道管の細胞径(約40~50 m)に対する長さは(4~5mm)約100倍、広葉樹の木部繊維は約20~50倍と両者ともに極めて細長い形であり、これらがほぼ規則正しく配列するので、木材は方向によって性質の異なる異方性を示す。樹軸と直交する横断面を木口面、樹心を通る年輪と直交する縦断面をまさ目面、年輪に沿った縦断面を板目面といい、これらは性質を比較するときの実用的な基準断面となる。また、樹軸に沿った方向を繊維方向、まさ目面に沿った方向を半径または放射方向、板目面に沿った方向を接線方向という。樹木の木部では、形成層付近の細胞および辺材の柔細胞を除いては、生理活動を行っていない死細胞の残骸である細胞壁だけが残っている。さらに、心材ではすべての細胞が死細胞で、生理活動は行われていない。したがって、辺材と心材とでは性質がやや異なり、辺材では、一般に含水率が高く、デンプン等を多く含むので腐朽菌などの攻撃を受けやすく耐久性にやや難があるが、心材では含水率が低く、デンプン等が少なく、心材化による化学物質の付加によって樹種固有の心材色が現れ、かつ耐久性が高くなっている。温帯の樹木では、1年の間で、早春に直径が大きく壁の薄い早材、初夏に直径が小さく壁の厚い晩材を形成するので、1年ごとにリング状の構造が認められ、それを年輪という。しかし、広葉樹と針葉樹とでは、細胞構成が異なることなどから年輪の見え方も異な樹種密度g/cm平均収縮率%全乾気乾接線方向半径方向カヤ0.41~0.590.45~0.630.18~0.310.10~0.200.1101.8220~230380~390トドマツ0.30~0.450.32~0.480.29~0.420.10~0.180.0771.4241~260―カラマツ0.37~0.560.40~0.600.19~0.350.10~0.240.0911.5264~278―エゾマツ0.32~0.480.35~0.520.25~0.360.11~0.200.0811.4251~270470~480アカマツ0.39~0.580.42~0.620.26~0.310.14~0.200.0951.6253~272480~490ツガ0.42~0.560.45~0.600.18~0.380.10~0.250.0911.5245~260450~470スギ0.27~0.410.30~0.450.21~0.300.05~0.210.0751.3232~252430~440ヒノキ0.31~0.490.34~0.540.14~0.270.07~0.150.0821.4247~258420~440ヒバ0.34~0.510.37~0.550.15~0.340.08~0.240.0841.5247~267―マカンバ0.46~0.740.50~0.780.21~0.400.10~0.240.1202.0――ブナ0.47~0.730.50~0.750.25~0.550.10~0.250.1171.8265~278460~480ミズナラ0.41~0.880.45~0.900.21~0.510.10~0.300.1221.9263~267440~460アカガシ0.77~1.040.80~1.050.36~0.570.17~0.330.1602.2―420~460ケヤキ0.43~0.790.47~0.840.17~0.380.09~0.230.1231.7257~270450~480クス0.39~0.670.41~0.690.23~0.470.12~0.310.0951.4―440~460イスノキ0.71~1.000.75~1.020.32~0.540.15~0.310.1662.0295~305430~440イタヤカエデ0.54~0.730.58~0.770.25~0.400.11~0.230.1161.8――トチノキ0.37~0.590.40~0.630.19~0.370.09~0.220.1001.8254~269シナノキ0.34~0.590.37~0.610.21~0.420.12~0.290.0911.4――ヤチダモ0.40~0.710.43~0.740.22~0.460.11~0.280.0991.6―アオダモ0.57~0.790.62~0.840.23~0.380.10~0.210.1402.1235~245375~385キリ0.17~0.370.19~0.400.13~0.340.05~0.140.0631.2262~276420~4302―151る場合が多い。また、熱帯産材では年輪は一般に見られない。針葉樹材で、しま模様のように硬い部分と柔らかい部分が繰返しで見られるのは、おもには年輪における早材および晩材の存在のためである。木材の実質は木材細胞の二次壁であり、成分はおもにセルロー
