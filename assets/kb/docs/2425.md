c .静圧造型機 模型に金枠をかぶせて自然落下で砂を入れ，
枠上部をふさいで密閉状態にして圧縮空気を0.3s程度砂上部に導
き，砂を加圧する．空気は模型のベント穴から逃がす．騒音が低
d.中子造型機 ブロー造型機の一種だが，型に吹き込んだ砂
は原則加圧成形しない．サンドリザーバ内の砂は，シューティング
バルブを開けると上方および側面から吹き出る圧縮空気によって中
子取り型に吹き込まれる．空気は空気逃げ穴（ベント）から排出さ
れ砂だけが中子取り型内に残り，中子ができる．砂はできるだけ流
e.自硬性造型 非量産の鋳物を製造する装置で，小物の場合
には連続移動装置を使用するが，大物の場合にはバッチ固定式であ
a.サンドミル（sand mill ) 鋳型砂の調整上最も重要な装置
である．砂には粘土・水その他を配合し均一に混練する必要があ
る．場合によっては油や合成樹脂などの粘結剤を混合することもあ
る．最も普通に用いられるローラ型のサンドミルは2個のローラが
容器内で回転し，混合物を容器の底板に押し付けて強力に練るよう
な形で作動する．鋳型砂の特性によってはかくはんを主体とするア
ジテータ型のミルも使用される．
b.砂再生装置（sand reclamation machine) 型ばらし後の
鋳型砂の表面に付着している不純物や古くなった粘結剤を取り除い
て，再び使用できる砂にする装置である．乾式では高速空気や回転
羽根で砂粒をターゲットに衝突させ再生させる方法と，ローラによ
り機械的にすり合わせる方法とがある．砂の再生に当たって砂中の
不純物を燃焼させ再生砂を肌砂として使用する流動ばい焼炉が用い
られている．
