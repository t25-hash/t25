# 2・7・5 仕上設備

> **章**: 第2章 鋳造
> **節**: 2・7 鋳造工場設備

b.清掃装置 鋳造品に付着している砂等を取り去る装置で，る．
ものではさらに高圧のものもあり，焼付き部の清掃などに用いられ面をきれいにするものである．
去・仕上げには，研削，ガス切断，パウダ切断，チッピングを用いスイング型が，表面仕上げにはハンドグラインダが用いられる．
鋳り取るバリ取り装置も開発されていて，自動車エンジンのクランクられる．
これらの作業は粉じん・騒音の点で労働衛生面に問題があa.型ばらし装置 鋳造後鋳型から鋳造品を能率よく取出すために型ばらし装置を用いる．
金枠を用いている場合には，まず鋳型と製品を取出し，その後，鋳型を製品から分離する方式（解枠）と，製品を鋳型から引き出す方式がある．
ショットブラスト，ハイドロブラスト，サンドブラストなどがあリッドを，羽根車で鋳造品に投射するものである．
0.2～0.5mmのものを用いる．
軽合金の場合には粉じん爆発が起こらない材質を選ぶ必要がある．
ノックアウトと称している．
ショットは砂粒といっしょに回収し，分離して再利用する．
ハイドロブラストは高圧水をノズルから噴射して砂落としおよび清掃を行うもので，高圧水を使用する．
サンドブラストは砂を圧縮空気でノズルから噴射し，品物の表c .バリ取り装置 清掃では除去できない鋳ばりや焼付きの除る．
小物のバリ取りや湯口切断にはフロアスタンド型が，大物にはばりの発生形状に合わせたカッタを油圧で往復動させて鋳ばりを折ケース内面鋳ばりの除去に用いられている．
ガス切断は鋳鋼に用いるため，自動バリ取り装置やロボットの利用が進んでいる．
