# 5・3・2 加工木材

> **章**: 第5章 高分子材料ほか
> **節**: 5・3 木質材料(wood-based materials)

a. 合板(plywood)通常,ロータリレースで丸太から0.5mmから数mmの単板をカツラむきし,1枚ごとに繊維の方向を直交させて接着剤で張り合わせたもので,普通3,5,7枚等の奇数合わせとする.製品品質はJAS(日本農林規格)で決められていて,普通合板,コンクリート型枠用合板,特殊合板,構造用合板,難燃合板,足場板合板,防災合板およびパレット用合板の規格がある.構造用合板(表5・9,5・10)には,板面の品質によって1級と2級があり,接着剤の耐久性によって特類と1類がある.実際的には1級の構造用合板はラワン合板で高度な構造的利用,2級は針葉樹合板で壁,床,屋根の下地板などに用いられる.特類の構造用合板は常時湿潤状態における接着耐久性を保証するもので,試験片を72時間連続煮沸等の処理後の接着力試験の値が表5・11の値以上であること,1類では通常状態での耐久性を保証し,煮沸繰返し試験等の処理後の接着力が表5・11の値以上であることが決められている.表5・10構造用合板(2級)の単板構成と曲げ性能の適合基準(JAS)(a)単板構成(b)適合基準合板の厚さmmプライ数単板の厚さmm構成比率合板厚さmm曲げヤング係数kg/cm5.06.07.59.012.065×101.5以上5.5以下15.0未満3以上表面単板と同じ繊維方向の単板の合計厚さが合板厚さの40%以上,70%以下15.0以上18.0未満4以上15.018.0以上24.0未満5以上18.021.024.0以上7以上24.0表5・11 wood)集成材とは,ひき板または小角等をその繊維方向を互いにほぼ平行にして,厚さ,幅および長さの方向に集成接着した材である.一般には,厚さ20mm以上,幅50mm以上で長さは1m以上のひき板を欠点を除いた後にフィンガジョインタ等で縦継ぎし,ラミナに接着剤を塗布してそれらの繊維方向を互いにほぼ平行にして数層に積層し,圧締め硬化後,四面を切削仕上げする.JAS に定める集成材の種類は,造作用,化粧ばり造作用,構造用および化粧ばり構造用柱がある.構造用途の集成材については,JAS の強度等級区分に応じた基準強度が国土交通省告示で表5・12のように定められている.造作用では一般にユリア樹脂系接着剤または同等の接着性能を有する接着剤,構造用集成材ではレゾルシノール樹脂系接着剤またはそれと同等の性能を有する接着剤が使われる.c. 木質ボード類. パーティクルボード(particle board)木材の小片をおもな原料として接着剤によって成形熱圧した板で,小片とはチップ,フレーク,ウエファー,ストランドなどをいい,例えばフレークは長さ10~30mmの長方形の小片である.JIS では,ウエファーボードおよびOSB(oriented strand boards)もパーティクルボードに含めている(JIS A 5908:2003).接着剤は,ユリア樹脂やフェノール樹脂が使われる.. 繊維板(fiberboard,ファイバボード)解繊した木材繊維を湿式法または乾式法で成形した板状の製品(表5・13)で,密度によってインシュレーションボード(IB,0.35g/cm未満),ミディアムデンシティファイバボード(MDF,0.35~0.8g/cm未満)およびハードボード(HB,0.8g/cm以上)に分類されている.性能規格は,JIS A 5905:2003に定められている.. 木毛セメント板幅3.5mm,長さ450mm以下,厚さ0.3~0.5mmの木毛とセメントを圧縮成形した板で,防火性が高く,断熱および吸音性も備えている.性能規格は,JIS A 5404:2001に定められている.. 木片セメント板長さ50mm以下の木片とセメントを圧縮成形した板で,木片の形状および木質/セメント比を変えることにより,物性が変わる.普通木片セメント板と硬質木片セメント板とがあり,防火性,断熱性および吸音性に優れる.性能規格は,表5・12
