# 2・7・6 環境衛生設備

> **章**: 第2章 鋳造
> **節**: 2・7 鋳造工場設備

鋳造工場は砂を多用するので比較的粉じんが出やすいため，工場
流気を考慮することが望ましく，発生箇所での局所対策が必要であ
る．
a.集じん装置（dust collector ) 鋳造工場のキュポラ，電気
炉，砂処理装置，注湯場などから出るばい煙や粉じんの対策に，乾
式や湿式の集じん装置が用いられる．通常，鉱物性粉じんの管理濃
度 ( mg/m ）は粉じん中の遊離ケイ酸含有率 （%）の関数，
＝2.9/ 0. 22＋1
として決められており，作業場のほとんど（95%以上）が基準以下
（第1管理区分)，平均は基準以下だが改善余地がある（第2管理区
分)，基準以上で適切でない（第3管理区分）に分けて，むらなく
環境が向上するように管理する．粉じんは，鋳物工場内の発生箇所
により性状が異なり，溶解ヒューム系，生砂系，中子（シェル砂）
系，仕上げ系に分けて管理される（図2･68)．作業場の粉じんを集
めるにはフードを設け，ダクトを通して排出する．フードは粉じん
発生箇所をできるだけ包むように設け，ダクトはできるだけ曲がり
部などがないようにする．粉じんをフードに吸引するために必要な
風速は粉じんの発生条件によって異なるが，注湯作業場，砂混練作
業場，型ばらし場の順に大きくする必要がある．ダクトの抵抗損失
などを考慮してフード入口でこの値になるよう排風機の容量を選定
する．フードはシェイクアウト（型ばらし），コンベヤの落とし口，
スクリーンなどに設置されている．シェイクアウト部からの蒸気含
じんガスの吸引については，ガス中の水分を考慮して吸引ガス温度
が露点に達しないよう高温排ガスを引き入れる．バグフィルタは円
筒または扁平な袋で，ろ布に付着したほこりの層によって粉じんを
ろ過する．ポリエチレン製の織布で，微細な粉じんまで取ることが
でき，しかも圧力損失も比較的少ないので，特に高い集じん効率を
必要とする場合に用いる．ろ布についた粉じんは一定時間ごとに揺
動，逆圧，圧縮空気噴射により払い落とす．
b.気中有害物質（ガス）と臭気 鋳造工場のにおいは炭素系
物質の不完全燃焼による焦げ臭さによるものである．成分濃度は低
く健康上問題はないが，嫌悪感が大きい．対策としては脱臭装置が
用いられ，薬液洗浄・吸着方式から（触媒）燃焼方式が用いられ
る．
c.騒音対策 一般的には，騒音発生源を覆って騒音の低減を
図ることが多く実施されている．目標値として騒音レベルを工場内
80dB以下，工場境界線で50dB以下としている．集じん機の送風
機とモータについては防音室へ格納し，消音器設置，ラギング処理
を行う．シェイクアウトマシンは砂落とし時に100dB（A）を超
えるので防音室へ格納し，製品の出入口も扉で遮断するなどの考慮
を払う．建屋壁を防音壁としたり，工場境界線に遮音塀を設けるこ
とも行われている．
d.振動 加速度の大きいジョルトを使用しなくなってきてい
るので，振動は低減されてきている．大型のシェイクアウトマシン
などの振動の大きいものは，装置の固有振動数を考慮した基礎工事
を行う必要がある．
e.産業廃棄物 廃棄物は溶解材の種類によって異なるが，鋳
図2･69 集じんダスト分散システム
物廃砂，スラグ，ダスト（溶解およびその他)，れんがくず等があ
る．金属の再資源化率は高く，問題となる廃砂やダストは分別処理
され（図2･69)，埋立処分から，骨材，セメント等への有効利用が
進められ，ゼロエミッションへの試みが進められている．
