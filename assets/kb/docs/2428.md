# 3・1・1 交通事故とデータベース

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・1 交通事故の調査

交通事故数を低減し，事故にあったときに受ける被害を軽減する
ことが社会的に強く求められ，そのため交通事故の実態を把握し，
事故発生要因と傷害発生要因とを調査し，さらにそれらデータの共
有化を図るべく，交通事故データのデータベース化が行われてい
る．
データベースという意味では収集されるデータの規模と性質によ
って二つのとらえ方がある．交通事故統計データ（statistical data
of traﬃc accident ）と事故例データ（in-depth investigation data
of traﬃc accident ）であり，前者は通常事故1件当たりのデータ
量は少ないが国あるいは地域全体で発生するすべての交通事故を対
象とするもので，後者は調査対象事故や調査地域を限定するが1件
の事故について詳細な内容を調査するものである．交通事故統計デ
ータは，対象事故すべてがそろっているので，交通事故の実態を把
握することが可能である．マクロデータ（macro-scopic ）あるい
は全国データと呼ばれることもある．一方，事故例データは事故1
件ごとに事故の発生原因あるいは傷害の発生起因を明らかにしよう
とする場合に利用され，現場調査（on-the-scene,on site investi-
gation）あるいはミクロデータ（micro-scopic ）と呼ばれる場合
がある．
また，データベースである場合については，その利用目的および
アクセス方法が重要である．交通事故データは，社会的に広い分野
で利用され，かつデータに含まれるプライバシーを保護する必要が
あることなどから，交通事故データの収集やデータベース化につい
ては国策として取り組まれる場合が多い．利用者の要求によりデー
タを提供する場合があるが，非公開としている場合もある．
