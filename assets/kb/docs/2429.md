# 5・4・1 紙,不織布,合成紙

> **章**: 第5章 高分子材料ほか
> **節**: 5・4 紙および繊維製品

a. 紙,板紙. 定義と製法紙は広義には繊維を水中に均一に分散した後,脱水,乾燥してシート状にしたものと定義できる.このように紙は水を使って繊維をシート化するのが特徴で,繊維としてほとんどの紙は木材繊維を使用しているが,そのほか麻,綿,わら等の非木材繊維や,ガラス繊維,合成繊維を使用したものも生産されている.木材構成成分の大半は繊維で,この繊維分を木材から取出したものをパルプと称し,これが紙の主原料となる.木材から繊維を取出すのに,化学的に薬品で蒸煮して取出すものを化学パルプ(chem-ical pulp),機械力を使ってすりおろすものを機械パルプ(mechanical pulp)といい,一度使用した紙(古紙)をリサイクルしたパルプを古紙パルプという.現在日本では紙の原料として古紙の使用比率が高くなり,2004年現在で60%と過半数を占めている.木材から取出したままのパルプ繊維では水中でのシート形成には絡み合い不足なので,叩解工程(beating process)で繊維をほぐして絡みやすくする.これをワイヤ上に流し出して脱水,湿紙を形成した後,さらにロール間でプレスして脱水し,これを高圧蒸気を通したシリンダで乾燥して紙となる.この機械を抄紙機という.紙の厚さはワイヤ上に流し出した繊維の量で決まり,量が多いほど厚くなる.しかし量が多くなるとワイヤの脱水能力が落ちるので,厚い紙は複数枚のワイヤを有する抄紙機で抄いた紙を重ね合わせたうえで,脱水,乾燥する.これを抄き合わせといい,この方法で作った厚紙が板紙である.. 紙・板紙の種類紙は人間生活の中で広範囲に使用されているが,数量的には新聞紙や印刷紙,コピー等の印刷情報用紙が大半である.板紙は包装用のダンボール原紙,紙器用板紙が大半を占め,その他建材原紙,紙管原紙等に分類される.紙,板紙の品種別分類表を表5・15に示す.. 紙,板紙の性質と特徴紙は水中で紙層を形成し,乾燥により水素結合を生じて強度を保つので,紙をぬらすと結合が消滅して強度を失う.これが紙の耐水性に劣るという欠点であるが,逆にこれが古紙の水への分散を容易にし,リサイクルを可能とする利点ともなる.また,この特性は紙の吸湿性と伸縮(dimensionalstability)に関連し,吸湿すれば伸び,乾燥すれば縮む.したがって紙の印刷に際しては印刷の環境と紙の水分が平衡していることが重要で,印刷時に紙が伸縮すると多色印刷で重ね刷りをした際に,伸縮により見当が合わなくなるおそれがある.紙の抄造に際し,ワイヤ上に流出した繊維はワイヤの進行方向に並ぶ傾向があり,紙に縦横を生じる原因となる.すなわち,ワイヤの進行方向を紙の縦と呼び,縦方向の引張強度は横方向の引張強度よりも強い.また,紙を引裂くときの強度を引裂強度といい,これは引張強度とは逆に横のほうが強い.紙の強度は湿度とも関連しており,図5・22に湿度と各種強度との関係を示す.また,紙は織物やフィルムと比べて腰が強いのが特徴で,このため製本したときにしっかりとした書物となり,厚い板紙を使用すると段ボールやカートンボックスとなって包装用に有用な特性を発揮する.紙は繊維の重なり合った構造のため,繊維間の空げきがあり,通気性と吸収性を有する.この通気性がフィルタとしての特性を示し,吸収性がティッシュのような衛生用紙としての特性を発揮する.さらに紙はその電気絶縁性と誘電性を生かし,ケーブルやトランス等の絶縁材料やコンデンサの誘電体としても使われていたが,高電圧用を除き,順次,合成高分子にその座を譲りつつある.b. nonwoven)に大別される.水を使用する湿式不織布は紙と全く同じ工程で,抄紙機が使用されるが,紙と異なって繊維間の接着工程は必要となる.乾式でも繊維間の接着工程は必須で,不織布の製造工程はマット化方式と接着方式の組合せで種々の生産方式が実用化されている.マット化方式で最も古くから行われていたのは紡績用のカード機を使って繊維をそろえる方法と,ランダムウェバを利用して空気流で繊維を分散する方法である.カード方式は比較的長繊維に適し,ウェバ方式は短繊維に適している.その後開発されたのがスパンボンド(spunbond)方式で,紡糸ノズルから吐出した糸をそのまま金網上に集積し,シート化してしまうので,マット化と接着を同時に行う大量生産型という特徴がある.メルトブロー方式はスパンボンドと同じく紡糸ノズルから吐出した糸を,ノズルの両側から噴射した高温エアで引伸して繊維を細くしたもので,紡績工程では扱えないほど細い繊維をシート化したところに特徴がある.また,最近ではスパンボンドとメルトブローの不織布を積層して,両方の特徴を生かしたSM 型とSMS 型の不織布が伸びてきている.接着方式で最も多いのがニードルパンチ(needle punch)方式である.これは繊維マットにとげのあるニードル(針)を突き刺して上下し,繊維を機械的に絡み合わせてシートとする.次に多いのが,ケミカルボンド方式で,繊維マットにバインダ溶液をスプレーするか含浸した後,乾燥して接着する.サーマルボンド方式は熱接着性繊維を混合してマット化し,加熱溶融して繊維間を接着する.また,水流交絡(スパンレース)方式は繊維マットに高圧の細い水流を噴射して繊維を絡み合わせるもので,しなやかで強度のある不織布とすることができる.最近生産量が伸びてきているエアレイド方式は,木材パルプ繊維を使った乾式不織布に当たり,おむつや失禁用のパッド,医療用等に使用されている.最近の日本における不織布の生産量を生産方式別に分類したものを表5・16に示す.c. 合成紙(synthetic paper)合成紙は繊維を使用せず,フィルムをベースとした非繊維のシート状体である.多くのプラスチックフィルムは,添加物を含有しなければ透明なものが多く,表面は平滑で通気性がない.透明であることは片面に印刷をすると透き通って両面印刷は不可能であり,印刷もインキの浸透が必要なオフセット印刷よりも,浸透の不要なグラビア印刷が適している.そこでフィルムに紙と同じ不透明性を付与して両面印刷を可能にし,紙と類似の印刷性を付与して,紙の用途に使用しながらも紙にない特性を発揮せしめたものが合成紙である.紙の特徴の一つにその腰の強さがあるが,プラスチックフィルムは概して柔軟なものが多く,結晶性の高い原料樹脂を選択し,さらに延伸を行うと剛性が向上する.そこで現在実用化されている合成紙の原料はほとんどポリプロピレンかポリエステルが多い.表5・16
