# 5・4・2 繊維製品(textile goods)

> **章**: 第5章 高分子材料ほか
> **節**: 5・4 紙および繊維製品

a. 繊維(fiber)繊維とは、イギリスのThe Textile Insti-tuteの定義によれば、しなやかさと細いこと、幅に対する長さの比が大きいことにより特徴付けられる物体の一つの単位、である。また、JIS L 0204-3:1998「繊維用語」によれば、糸、織物などの構成単位で、大きさに比して十分の長さをもつ、細くてたわみやすいもの、となっている。繊維には、天然に存在するもの、人間が合成したものなど種々のものがある。繊維の種類を表5・17に示す。. 天然繊維(natural fiber)天然繊維とは、天然に存在する繊維状物質を人が利用したものである。天然繊維は、植物系、動物系、無機系に分けられる。植物系繊維には、綿や麻(亜麻、苧麻など)などが含まれるが、いずれも天然高分子であるセルロースを主成分としている。動物系繊維には、羊毛や絹などが含まれるが、その主成分はアミノ酸である。. 化学繊維(chemical fiber)化学繊維は、人間が化学的に作り出した繊維である。化学繊維は、再生繊維と合成繊維およびその中間の半合成繊維に分けられる。再生繊維は、天然にある高分図5・23 machine)化学繊維紡糸装置とは、原料の高分子を繊維化する装置である。化学繊維紡糸装置には、湿式紡糸、乾式紡糸、溶融紡糸の三つの方式がある。これらの方式の概念図を図5・24に示す。. 溶融紡糸機(melt spinning machine)溶融紡糸機とは原料高分子を熱で溶融させた状態で、細い孔が無数にあいた紡糸口金から押出して繊維状にした後、冷却して固化させる化学繊維紡糸機械である。ポリエステル、ナイロン、ポリプロピレンなどは本法による。. 乾式紡糸機(dry spinning machine)乾式紡糸機とは、原料高分子を熱で気化する溶剤に溶かした後、熱雰囲気中で紡糸口金から押出した後、溶剤を気化させて繊維状にする化学繊維紡糸機械である。アセテート、アクリル、ビニロンなどは本法による。. 湿式紡糸機(wet spinning machine)湿式紡糸機とは、原料高分子を溶剤に溶かした後、凝固浴と呼ばれる溶液中で紡糸口金から押出し化学反応で原料高分子を再生させながら繊維状にする化学繊維紡糸機械である。レーヨン、アクリル、ビニロンなどは本法による。c. 繊維製品(textile
