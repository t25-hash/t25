# 5・5・1 塗料の構成

> **章**: 第5章 高分子材料ほか
> **節**: 5・5 塗料

塗料には、数多くの種類があるが、機能的に、塗膜形成主要素と塗膜形成副要素、塗膜形成助要素から構成される。これらの構成要素からなる透明媒質がビヒクル(vehicle)で、その不揮発性成分をビヒクルソリッド(vehicle solid)と呼ぶ。また、塗料の構成において顔料を含まない塗料を透明塗料(クリヤ、clear lacquer)、着色顔料を含む塗料を着色塗料(エナメル、enamel)と呼ぶ。これら塗料の構成を図5・26に示す。塗膜形成主要素は、塗膜を形成する主成分であり、乾性油、天然樹脂、合成樹脂、セルロース誘導体などが用いられる、塗料および塗膜の性能を決定する重要な部分である。塗膜形成副要素は、塗膜や塗料の性能を向上させる目的で加えられる成分であり、可塑剤、硬化剤、乾燥剤、増粘剤、界面活性剤、皮張り防止剤、殺菌剤などがある。塗膜形成助要素は、塗料を液状に保ち、塗料を塗装するときに流動性を調整する目的で使用され、乾燥過程で蒸発し、形成塗膜中には残らない成分である。主要な塗料用有機溶剤および水の性能を表5・22に示す。一般には、各種の有機溶剤が用いられるが、溶剤成分はすべての塗料に不可欠の成分ではない。近年は、溶剤を含まないか少量しか含まない塗料として無溶剤型塗料例えば粉体塗料や紫外線(UV)硬化型塗料、ハイソリッド塗料などが、また、水を溶剤成分として用いる水系塗料などが広く使用されている。顔料(pigment)は、塗膜に色彩や隠ぺい力を与え、塗膜の物性や耐久性を補強したり、防錆効果を付与するために用いられる。顔料には、無機着色顔料、有機着色顔料および体質顔料、防錆顔料がある。無機着色顔料は一般に隠ぺい力、耐候性、耐熱性および耐溶剤性に優れるが着色力に劣る。有機着色顔料は一般に着色力、鮮明度に優れるが隠ぺい力や耐候性に劣る傾向をもつ。しかし、近年、高い耐候性をもつ多くの有機着色顔料が開発されており、目的とする用途に応じて有機着色顔料と無機着色顔料を組合わせて用いられ溶剤区分溶剤分子量沸点°C密度粘度cP/20°C相対蒸発速度-キシレン106.2139.10.860.579/25°C0.768.808.650.51.5炭化水素系エチルベンゼン106.2136.20.8678.808.700.30.7エチルアルコール46.178.30.7891.171.5412.927.734.39.5アルコール系-ブチルアルコール74.1117.70.8102.950.4711.307.812.87.7酢酸エチル88.177.10.9010.4494.29.107.442.64.5-4エステル系酢酸-ブチル116.2126.10.8810.7341.08.467.671.83.1エチレングリコールモノブチルエーテル118.2171.00.903.2/25°C0.0810.257.7633.15.9エーテル/アルコール系プロピレングリコールモノブチルエーテル132.2170.00.9メチルエチルケトン72.179.60.805 0.423/15°C3.79.277.774.42.5-6ケトン系メチルイソブチルケトン114.2116.50.801 0.542/25°C1.68.577.493.02.0その他水18.0100.01.0001.0020.3823.56.015.316.7る。また、無機着色顔料には、アルミニウムやマイカなどのフレーク顔料をメタリック塗料として用いる場合や、特殊な表面処理をされた顔料(例えば、パール顔料をチタンコーティングするなど)を光輝性塗料として用いる場合も多い。各色相での代表的着色顔料の特性を表5・23に示す。体質顔料は、塗膜の補強や増量のために使用され、タルク(3MgO・4SiO・HO)、天然産の白色粘土の粉砕品であるクレー(AlO・SiO・2HO)などがある。防錆顔料は、鉛丹(PbO)、鉛シアナミド[Pb(CN)]、鉛酸カルシウム(CaPbO)、鉛粉(Pb)などの鉛系顔料、および、亜鉛黄(ZnCrO)、クロム酸バリウム(BaCrO)、亜鉛末(Zn)などが使用されるが、近年の環境保護および安全衛生の観点から、鉛系顔料が使用されなくなりつつある。
