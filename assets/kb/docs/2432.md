# 5・5・2 塗料の分類

> **章**: 第5章 高分子材料ほか
> **節**: 5・5 塗料

塗料は,品目別(塗膜形成要素別),需要区分別,塗料形態別などさまざまな分類がされている.例えば,塗膜形成要素別では,油性塗料,溶剤系合成樹脂塗料,水系塗料,無溶剤系塗料,無機質塗料などに分類され,需要区分別では,建物用塗料,道路車両用塗料,船舶用塗料,金属製品用塗料,家庭用塗料などに分類されている.ここでは,合成樹脂塗料について,近年の社会的ニーズにマッチした環境対応型塗料を中心に形態別の分類で示す.合成樹脂塗料とは,各種合成樹脂を塗膜形成要素として使用し,現在のはん用および工業用塗料の主流をなしている塗料である.合成樹脂はその合成法から,アクリル樹脂,ビニル樹脂などの付加重合物と,アルキド樹脂,ポリエステル樹脂,アミノ樹脂などの縮合重合物とに大別され,性能と価格面から使い分けられている.合成樹脂塗料の特性を表5・24に示す.合成樹脂塗料はまた,形状により,粉体塗料,水系塗料(水溶性型,エマルジョン型,水性ディスパージョン型)および有機溶剤系塗料(溶剤型,溶液型,非水ディスパージョン型)に分類され,また乾燥形式により,常温乾燥塗料,焼付乾燥塗料,2液反応硬化塗料,電子線・紫外線硬化塗料などに,塗装法からは,電着用塗料などに分類される.a. 粉体塗料(powder coating)粉体塗料は,有機溶剤や水などの揮発性成分を含まず,数mから数十mに微粉砕した固形樹脂が用いられ,塗装後,熱により溶融させ,堅牢な膜を形成する2―163表5・22一般的な塗料用溶剤について溶解性パラメータ引火点°Cδ分散力成分δ욃極性成分δ욣水素結合成分δ욠トルエン92.1110.40.8670.5872.08.918.820.71.0ヘキサン86.268.70.559 0.307/25°C7.27.247.24-25.5メチルアルコール32.064.70.7910.5951.914.287.426.010.9ジアセトンアルコール116.2168.10.9392.90.1510.187.654.05.3塗料である.無溶剤であること,回収・再利用が可能な点で環境適応度が高い塗料といわれており,今後大いに期待される塗料である.その需要量は着実に伸びているが,まだ,一部の限られた分野での使用にとどまっている.薄膜での仕上り性(平滑性)や色替え時の作業性で改良が望まれている.b. 水系塗料(waterborne coating)水系塗料は,有機溶剤の大半を水に置換した塗料であり,揮発性有機化合物(VOC)低減対策,悪臭防止,危険物対策などに期待がかかり早くから使用されている.粉体塗料に次いで環境対応度が高いともいわれている.この水系塗料は,水溶液型と水分散型に分けられ,水分散型にはエマルジョン型とディスパージョン型がある.表5・25に水系塗料の特徴を示した.c. ハイソリッド塗料(high solids coating)従来の有機溶剤に樹脂を溶解して用いる最も一般的な溶剤型塗料の延長線上に位置する塗料である.粉体塗料や水系塗料のようにVOC規制に対して飛躍的な効果は期待できないまでも,近年,VOC低減を目的として,塗料状態での加熱残分が約75%以上が望ましいといわれている,溶剤量の少ない,ハイソリッド型塗料が多く用いられるようになってきた.ハイソリッド化技術は,使用する樹脂を低粘度化すること,すなわち低分子量化の方向にもっていくことであり,これに伴う,性能の維持が望まれている.
