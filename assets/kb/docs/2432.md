# 3・10・1 マルチボデーモデル

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・10 人体の数値モデル

インパクトバイオメカニクスにおける人体数値モデルには，衝突
ダミーをモデル化する衝突ダミーモデル（crash dummy model ）
と，人体を直接モデル化する実人体モデル（real human body
model ）がある．モデル化の手法としてマルチボデー解析と有限要
素解析が一般に用いられる．マルチボデー（multibody）のこの分
野への適用は1963年マクヘンリー（McHenry）による人体，拘束
装置，車両を表す二次元7自由度モデル が最初とされている．
衝撃時の人体の挙動では関節が自由度をもち，他の部位は剛体と
近似できることが多いため，マルチボデー解析が適用可能なことが
多い．マルチボデーではボデー（剛体またはフレキシブルボデー）
が図3･51のような多様なジョイントにより結合され，衝撃時の各
ボデーの過渡応答が求められる．ボテーに働く外力は，加速度場，
ばね-ダンパ要素，拘束装置，接触等の力の相互作用がある．衝撃
時の人体の全身挙動は大きくなるため，これを効率よく計算し，保
護装置の最適化を行うためには有限要素よりもマルチボデー解析が
適していることが多い．
衝突解析のマルチボデーシミュレーションプログラムでは，
TNOによって開発されたMADYMO が用いられることが多い．
特に，MADYMOには衝突解析用にシートベルトモデル，傷害値
の出力などが準備されており，さらに多様な衝突ダミーやインパク
タのデータライブラリをもっている．マルチボデーと有限要素法と
の結合も可能となっており，例えば有限要素でモデル化されたエア
バッグとマルチボデーでモデル化された衝突ダミーの連成解析が可
能である．この場合，変形は有限要素のみとなり，マルチボデーは
変形しないものとして計算される．図3･52に示す衝突ダミー
（Hybrid-Ⅲ）による乗員シミュレーションは，拘束装置の最適化
に用いられる．
これまでのマルチボデーモデルは人体そのものよりも，衝突ダミ
ーに対して開発されたものが多い．この理由としては，衝突ダミー
ではデータが求めやすく検証が容易であること，および認証試験で
ダミーが用いられるため，これに対する設計の効率化が挙げられ
る．
衝突ダミーのマルチボデーによるモデル化では，運動学を考慮し
てダミーがセグメントに分割される．相対運動がないダミー部品が
一つのセグメントとして選択され，セグメントどうしは互いの自由
度を考慮したジョイントによって結合される．セグメントの幾何学
的パラメータ，慣性特性，ジョイント位置・方向は，図面や計測に
よって決定される．ダミーの外形形状はだ円体のほかに，任意形状
を表すファセットが用いられる．ダミーの荷重変形特性は衝突時に
予測される変形を考慮して静的および動的測定が行われ，ジョイン
トについても静的，動的な特性が計測される．衝突ダミーモデルは
部位ごと，あるいは全身についてインパクト試験，スレッド試験に
よる検証が行われる．図3･53にEuroSID-1側突ダミーモデルと
して，だ円体，ファセット面によるマルチボデーモデルと有限要素
モデルを示す．
