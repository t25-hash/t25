# 5・5・3 塗料の製造

> **章**: 第5章 高分子材料ほか
> **節**: 5・5 塗料

塗料の構成成分は、使用用途に従って適正な配合比で混合、均一化されて塗料となる。塗料になるまでには、大まかに(1)溶解工程、(2)顔料分散と調色工程、(3)製品の仕上工程、(4)製品の缶詰工程に分類され、その工程あるいは操作に関する化学工学が塗料の製造で重要である。一般的に、透明塗料の場合には、ディゾルバ(高速かき混ぜ機)でビヒクルと溶剤をかくはん混合し、溶解均一化して製造されるが、顔料着色塗料では、サンドグラインドミルやボールミル、アトライタなどの各種分散機を使用して、顔料をビヒクルに均一に混和・分散させる顔料分散(pigment dispersion)工程を必要とする。そして、得られた原色エナメルを必要に応じて他の原色エナメルと調合して所定の色調に合わせる調色工程、さらに、塗料をカートリッジろ過機やふるいろ過機などでろ過精製するろ過工程を経て、容器に詰めて製品とする缶詰工程から成り立っている。2―164材料学・工業材料色相顔料名化学構造C.I.No.(カラーインデクッスナンバー)吸油量N/100N比重備考キナクリドン(有機顔料)PV-191.4~1.5青銅フタロシアニンブルー(有機顔料)PB-1530~581.5~1.7 α,β,ε型がある。*ASTM D 281:1989
