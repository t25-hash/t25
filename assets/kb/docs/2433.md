# 3・10・2 FEMモデル

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・10 人体の数値モデル

近年，有限要素法（finite element method，以下，FEM）を用
各要素の応力やひずみを計算により求めることができるという利点
がある．その反面，モデルの作成に多くの時間を要することや，要
素の分割数が増加すれば計算時間も増加するという欠点がある．こ
のFEMを3･9節で述べているダミーや人体そのものに適用すれ
ば，複雑な形状や構造が再現でき，リアリスティックな解析が可能
になってくる．自動車の衝突現象等を解析するためには，FEMの
中でも動的陽解法有限要素法が用いられる．
図3･57はHybrid-ⅢダミーのFEMモデルを示している．計算
機能力の飛躍的な向上により，形状を忠実に再現した車両モデルと
ダミーモデルを組合わせた衝突解析が行われ，衝突時のダミーの応
答を計算機上で試行実験することが可能になってきている．しか
し，機械要素で構成された擬似人体ダミーモデルでは，生身の人間
に発生する複雑な骨折や関節部の靱帯破断・内臓臓器の破裂といっ
た軟組織の傷害等の発生メカニズムに対する，よりきめ細かでかつ
直接的な解析を行うには限界がある．
そこで，近年，人体そのものを解剖学的な構造ならびに力学特性
に忠実に計算機上でモデル化し，傷害を再現・予測しようとする試
みがなされている．この人体FEMモデルは人体の解剖学的構造を
有限要素でモデル化し，組織を構成する各要素に実験で得られたそ
の組織の材料特性（ヤング率，ポアソン比，降伏応力等）を与える
ことにより作られている．モデル作成のために必要となる解剖学的
構造や形状の三次元座標データには，医療診断等に使用されるCT
やMRI画像から得られる輪郭形状や，市販の人体形状に関するポ
リゴンデータが用いられている．材料特性としては，人体や動物の
組織を用いた荷重試験データ などが用いられる．
図3･58は衝突安全（crash safety）分野で下肢傷害解析のため
に開発されたFEMモデルの一例を示している ．形状を忠実に
再現した骨・靱帯・腱・筋肉を含む軟組織により構成され，材料特
性には文献から得られる実験データが使用されている．衝突安全分
野では，このような人体各部位のモデルが多く開発されており，そ
の妥当性（validity）は献体実験データと比較することにより評価
されている．献体実験は主として人体耐性値の取得，ダミーの妥当
性検証のため，欧米で実施されているが，実験により得られるデー
タは人体FEMモデルの妥当性を検証するためにも有用である．献
体実験で計測された結果を，献体実験と同じ条件を与え計算した結
果と比較することで，モデルの妥当性を検証することができる．図
3･59は胸部単体モデルを用いて，インパクタ質量23.4kg，初速度
5.8m/ sという献体実験と同じ条件で，側面衝撃計算を行ったとき
の胸部の変形状態を示している ．ブーケ（Bouquet ）らが報告
した献体実験の応答範囲 との比較を図3･60に示すが，モデル
の応答は実験データとよく一致しており，モデルの妥当性を示して
いる．この人体FEMモデルを用いて傷害評価を行うためには，
FEMで計算される応力やひずみを用いることができる．例えば要
素の応力やひずみが，あらかじめ定められた応力やひずみの限界値
を超えた場合，人体のその部分に破壊すなわち傷害が発生したと判
定する手法である．このような手法を用いて傷害発生を予測し，献
体実験における傷害部位や程度と比較することで，モデルの傷害予
測（injury prediction）精度を検証することが可能である．
最近になって，計算機能力の向上に伴い，人体各部位を詳細にモ
デル化した全身のFEMモデルが登場してきた．その代表例を図
3･61および3･62に示す ．これらのモデルは，衝突時の全身
の挙動や傷害を再現・予測するために利用されている．図3･63に
全身モデルTHUMSを用いて，後面衝撃時の乗員の全身挙動を解
析した例を示す．計算により得られた全身挙動を献体実験やボラン
ティア試験により計測された全身挙動と比較し，モデルの妥当性の
評価が行われている．このような全身モデルを用いれば，衝突事故
時に起こりうる多重傷害を再現・予測できると期待され，注目され
ている．しかし，現状のモデルでは，あらゆる方向や速度に対し
て，モデルの応答特性と傷害を再現するのはいまだ困難である．今
後は，生体組織の特徴であるひずみ速度依存性や異方性を考慮した
モデル化も必要になってくる．また，現状の人体モデルは献体の応
答特性に合うようにモデル化されているが，今後は生体の応答特性
の再現へと進化していくと考えられる．そのためには，アクティブ
な筋肉特性や脳機能・神経伝達系のモデルも付加していく必要があ
