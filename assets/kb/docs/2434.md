# 5・5・4 塗装

> **章**: 第5章 高分子材料ほか
> **節**: 5・5 塗料

塗装とは，被塗物の表面に塗料を塗布するために適当な状態に調
整し，適当な方法で塗料を塗り付けて被塗物の表面に塗料の薄層を
形成させ，乾燥工程によって塗料を塗膜に変化させ，所期の効果を
発揮できるようにすることである．被塗物としては，鉄鋼その他の
金属，木材，プラスチックなど種々あり，それぞれに合った塗装方
法が提案されている．
被塗物の表面に塗料を塗布するために適当な状態に調整すること
を塗装前処理という．金属の前処理は，対象とする金属の種類，施
工場所，方法によって異なる．建築，船舶，橋りょうなどの屋外構
造物では，金属表面の油，ごみ，さびなどの付着物を除去して塗装
される（素地調整処理）．一方，工場で生産される部材に対しては
表面処理が行われる．この方法は，化学的手段によって金属表面に
