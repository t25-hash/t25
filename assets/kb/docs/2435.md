# 5・5・5 塗料の乾燥(drying)

> **章**: 第5章 高分子材料ほか
> **節**: 5・5 塗料

塗料は被塗物の表面に塗装後,乾燥工程により連続被膜を形成する.塗料の乾燥方法には,自然の条件下で乾燥させる常温乾燥(自然乾燥)と,熱風あるいは赤外線を用いて乾燥させる強制乾燥,あるいは焼付乾燥がある.これらの乾燥方法以外に特殊ではあるが,電子線照射や高周波加熱などが挙げられる.理想的な乾燥状態の塗膜にするには,素材特性(熱容量など)や被塗物の形状および塗料の樹脂組成,乾燥機構,溶剤特性(溶解力,蒸発速度,溶剤離れなど),助剤効果などを知ることと,乾燥炉の特性(乾燥方式,能力,昇温カーブなど)を併せて考えることが重要である.乾燥機構のその1として,塗膜中の溶剤が蒸発することにより乾燥する揮発乾燥型がある.ニトロセルロースラッカや塩化ビニル樹表5・28 塗料の乾燥機構乾燥形式乾燥機構塗料の例酸化乾燥空気中の酸素を吸収して酸化し,これに伴って重合が起こり塗膜が乾燥する.アルキド樹脂塗料合成樹脂調合ペイント塗料中の溶剤や水が蒸発して塗膜が乾燥する.ラッカ塩化ビニル樹脂塗料エマルジョン塗料揮発乾燥(融着乾燥)溶融冷却乾燥塗料を加熱溶融して塗布したのち,放冷して塗膜を乾燥させる.ゾル塗料熱可塑性粉体塗料加熱架橋乾燥加熱によって樹脂が架橋を行い,塗膜を乾燥させる.アミノアルキド樹脂塗料熱硬化性アクリル樹脂塗料触媒,架橋剤の作用によって樹脂が重合し架橋して塗膜が乾燥する.不飽和ポリエステル塗料エポキシ樹脂塗料ポリウレタン樹脂塗料架触媒架橋乾燥橋電子線加速器を用いて電子線を照射し,イオン・ラジカル重合で塗膜を乾燥させる.電子線硬化型塗料乾燥電子線架橋乾燥光架橋乾燥有効波長の光線を照射して重合乾燥させる.紫外線硬化型塗料表5・29塗料用樹脂のおもな架橋反応反応名化学式内部不飽和結合の酸化重合アクリロイル基の光ラジカル重合フマロイル基またはマレオイル基のスチレンとのラジカル共重合水酸基とN-アルコキシメチル基またはN-メチロール基との縮合アルコキシシリル基の水による自己縮合カルボニル基とヒドラジド基との縮合エポキシ基とアミノ基との付加エポキシ基とカルボキシル基との付加水酸基とイソシアネート基との付加脂塗料などがそのタイプで,乾燥しても,また溶剤がかかると溶けてしまう性質をもっている.その2として,樹脂が空気中の酸素を吸収して,酸化することにより重合が起こり,網目を形成する酸化重合型がある.油性調合ペイントや合成樹脂調合ペイント,フタル酸樹脂塗料などがそのタイプで,できあがった塗膜は,網目が粗いため性能が少し劣り,また網目ができるまでに時間がかかる.その3として,常温下あるいは加熱乾燥することにより樹脂の反応(化学的橋架けや架橋反応,重合ともいう)が起こり,塗膜は緻密な網目(三次元網目構造ともいう)を形成する熱重合型がある.塗膜性能は優れており完全に乾燥すると溶剤にも溶けなくなる.エポキシ樹脂塗料,ポリウレタン樹脂塗料など2液重合型やアミノアルキド樹脂塗料やアクリル樹脂塗料などがそのタイプである.各種塗料を乾燥機構で分類したのが表5・28であり,塗料用樹脂のおもな架橋反応(crosslinking)を表5・29に示す.
