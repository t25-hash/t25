# 5・6・1 接着剤

> **章**: 第5章 高分子材料ほか
> **節**: 5・6 接着剤およびシーリング材

a. 接着とは物と物をつなぎ合わせることを接合というが、接合方法には、機械的接合法(ボルト締め、リベット締結、ねじおよびくぎ止めなど)、冶金的接合法(圧接、溶接、融接、ろうおよ2―167びはんだ付けなど)、その他(縫い合わせ、はめ込みなど)多くの方法がある。接着剤でつなぎ合わせることを接着といい、接着も接合方法の一つの方法である。接着剤によって接着される材料を被着材と呼んでいる。ISO(International Organization for Standardization、国際標準化機構)では、「接着とは、接着剤を媒介とし、化学的もしくは物理的な力またはその両者によって二つの面が結合した状態であり、接着剤とは、接着によって2個以上の材料を一体化することができる物質」と定義している。b. 接着接合の長所と短所接着剤は、航空機、自動車、車両、土木、建築、電気電子などの各産業において幅広い用途に利用されている。ものを接合する方法はいろいろあるが、安定した接着強度を得るためにはできるだけ広い面積を接合することが好ましい。リベットは点接合、溶接が線接合であるのに対して接着は面接合となる。このように点、線、面と比較してみると、接着剤による接合はすべての接合方法の基本であり、異種材料の接着などにおいてリベットや溶接などにはない多くの長所がある反面、短所もある。接着で失敗しないためには長所と短所をよく理解しておくことが大切である。接着接合の長所と短所については表5・30に示す。2―168材料学・工業材料接着を成功させるためには、接着の長所および短所に関する十分な理解と、さらに張り合わせる被着材についての知識、接着剤の選択、表面処理(surface treatment)、接着工程管理などの条件が適切でなければならない。c. 接着の理論。 接着の機構接着機構には、(a)機械的結合、(b)物理的相互作用、(c)化学的相互作用の三つがある。機械的結合とはアンカ効果(投びょう効果)やジッパ効果ともいわれ、材料表面の孔や谷間に液状接着剤が入り込んで、そこで固まることによって接着が成り立つという考え方があり、木材や繊維、皮などの吸込みのある材料の接着を説明するのに有効な理論である。物理的相互作用とは分子間(引)力(intermolecular force)といわれるもので、分子の間の引き合う力(ファン・デル・ワールス力)、双極子間力、ロンドン分散力のことである。また、二次結合力ともいって接着剤の基本的な原理とされている。三つめの化学的相互作用とは、一次結合力といって最も強い接着力が期待される共有結合、イオン結合、配位結合や水素結合をいう。すなわち接着とは、機械的な引っ掛かりや分子間力、原子間力によって成り立っており、そのどれかに原因を絞り込むことができない複雑さをもっている。図5・27に接着の理論・説を示す。. 接着のプロセス接着剤が十分に機能するために必要な三つの要件がある。第一には接着剤が液体であること、第二には接着剤が被着体の表面をよくぬらすこと、第三はなんらかの方法によって接着剤が硬化(固化、solidification)することである。図5・28に液体である接着剤が固化することにより接着が完了する接着のモデル図を示す。接着の原理からいえば接着剤と被着材はその分子間力の及ぶ範囲に接近していなければならない。ここでぬれということが大事となる。油の上に水を落としても水は広がらない。接着剤は被着材の表面をぬらして広がったのち、固化により初めて界面結合力と凝集力が発生して接着が完了する。. 接触角(contact angle)ぬれを定量的に示すのが接触角である。被着材の上に接着剤を1滴垂らすと、図5・28のようになる。被着材と接着剤の角度θ(シータ)が接触角である。液で被着材が完全にぬれるときにはθ=0°であり、全くぬれないときにはθ=180°になる。実際には0°と180°との間の角度をとる。接触角θとぬれとの関係を図5・29に示す。. 臨界表面張力(critical surface tension)臨界表面張力γとは、cos θ=1,θ=0°、完全にぬれている状態のときの表面張力をいい、被着材に固有の値である。γの小さい被着材は表面張力の小さい液体でなければぬれないし、γの大きい被着材は表面張力の大きい液体でも、表面張力の小さい液体でもともによくぬれる。液体である水は76dyn/cm(0.076N・m)であり、アセトンは26dyn/cm(0.026N・m)である。したがって、水はポリエチレン表面ではぬれないが、アセトンはぬれる。また、ポリテトラフルオロエチレンまたはポリフッ化ビニリデンのようなγの小さいフッ素系プラスチックは通常の液体ではぬれないので、γを大きくするためにエッチング液(金属ナトリウム/ナフタリン/THF系処理液に浸漬)で処理を行ってから接着する。. 溶解度パラメータ(SP値: solubility parameter)蒸発エネルギー[(Cal)]を分子容積(CC)で割った値の平方根である。接着剤はSP値がよく似た溶剤に溶解するし、被着材のSP値とよく似た接着剤を使用するとよく接着する。例外もあるが、一応の目安となる。分子構造からSP値を計算できる簡易式がある。d. 接着の手順接着の対象となる被着材としては金属、石材、コンクリート、木材、プラスチック、ゴム、ガラス、紙、皮革、繊維などであるが、同じ材料どうしを接着する場合は接着剤の選定も比較的容易であるが、異種材料どうしを接着する場合、接着剤の選定はより難しくなる。一つの万能型接着剤でなんでも接着できることが好ましいが、作業性、接着性能および耐久性を考慮した場合、各種の接着剤を使い分ける必要がある。接着剤を選定するポイントは、使用の目的と用途をはっきりさせることである。. 接着剤の種類と分類接着剤は、無機系接着剤と有機系接着剤に大別される。無機系接着剤にはセメント、ケイ酸ソーダなどがあり、有機系接着剤には、天然系接着剤と合成系接着剤がある。天然系接着剤には膠(にかわ)、正麩(しょうふ)、続飯(そくい)、デンプン、松やになどがあるが、工業的にあまり使われていない。合成系接着剤の分類方法には、主成分による分類、固化および硬化(cure,curing)方法(1室温硬化型、2熱硬化型、3熱溶融型、4感圧型、5再湿型)による分類、形態(1水溶液、2溶剤溶液型、3水分散型、4無溶剤型、5固型、6テープ状)による分類、接着強さ(1構造用、2準構造用、3非構造用)による分類、機能(1電気的、2力学的、3熱的、4界面、5放射線、6光学的、7生体、8その他)による分類などがある。ここでは、最も一般的な主成分による合成系接着剤の分類について表5・32に示す。g. 粘着接着剤粘着剤ではないが初期硬化過程の中に長い粘着領域を導入した新しい概念の先進対応型多用途接着剤である。空気中の湿気により反応して粘着(コンタクト)性が発現し、粘着接着が可能となる。無溶剤、1液、初期接着性、常温速硬化、弾性、幅広い接着性などの特徴を有する、夢の接着剤に一歩近づいた理想的な接着剤である。. 弾性(elasticity)接着剤特殊変成シリコーン樹脂を主成分とする接着剤で、周囲からのさまざまな外力に対して変形し、元に戻る復元力が強い弾性機能を付与された接着剤である。高強度、高剛性のエポキシ系構造用接着剤と異なり、硬化後ゴム状弾性体となる。はがれない接着、環境に対して安定な接着、より信頼性の高い接着のコンセプトのもとに開発され、エポキシ樹脂系接着剤のように強度は高くないが従来の接着剤にはないさまざまな特徴をもっている。. 低臭気2液アクリル(SGA : second generation acrylicadhesive、第2世代アクリル系接着剤)系構造用接着剤低臭気2液アクリル接着剤は混合比、混合状態がばらついても硬化後の物性があまり変わらない低臭気接着剤で、短時間に硬化し、接着強度も大きいので、構造用途に適用できる強靱な強度を有している。.
