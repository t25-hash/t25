# 6・1・1 ケイ酸塩ガラスの性質

> **章**: 第6章 無機材料
> **節**: 6・1 ガラス

ガラスは高温の融液を結晶化させることなく冷却固化させた非晶質固体であり、実用ガラスの大部分は無水ケイ酸(SiO)を主体としたケイ酸塩ガラス(silicate glass)である。無水ケイ酸のみを溶融して得られる石英ガラス(シリカガラス、silica glass)は優れた化学的、熱的、機械的性質を有し、光通信や半導体産業には欠かせない材料である。石英ガラスでは、ケイ素を4個の酸素で囲んだSiO四面体が、非常に強い共有結合性のSi-O-Si結合を通じて三次元的かつランダムにつながっている。石英ガラスにNaOなどのアルカリ金属酸化物やCaOなどのアルカリ土類酸化物を加えると、強いSi-O-Si結合が切断され、非架橋酸素が生成して粘度が低下し溶融や成形加工が容易になる。そのようなガラスの総称がケイ酸塩ガラスである。したがって、ケイ酸塩ガラスの性質はSiOに加えられた酸化物の種類と量によって大きく変化する。ケイ酸塩ガラスの種類および製品は日常的なビンガラスから最先端の光ファイバ関連ガラスまで多種多様にわたっている。代表的な実用ガラスの組成を表6・1に、それらのガラスの諸性質を表6・2に示す。a. 粘度ガラスはガラス転移温度(glass transition temper-ature)以上の温度域では液体であるが、その粘度は温度によって大きく変化する。粘度の温度依存性を知ることは溶融、成形(加工)、徐冷の温度設計をするうえで必要不可欠であり、特に、作業点、軟化点、徐冷点、ひずみ点の粘度に対応する温度が重要である。溶融ガラスのある形への成形(作業点)は粘度が10~10Pa・sとなる温度範囲で行われる。成形後急激に冷却するとガラスにはひずみが残り、自然に割れてしまうので、10Pa・sの徐冷点から10Pa・sのひずみ点までの温度領域をゆっくり冷却する必要がある。SiOガラスの粘度は式(6・1)で表されるアレニウス型の温度依存性[エンジェル(Angell)の定義に従えばストロングなガラス]を示し、融液の粘度は温度上昇に対しゆるやかに低下する。SiOにNaO,CaO,PbO,AlO,BOなどを添加して作製されたケイ酸塩ガラスは粘度が低下するだけではなく、融液の粘度はガラス転移温度以上で比較的大きく変化し、式(6・2)のフルチャー(Ful-cher)式で表される温度依存性(フラジャイルなガラス)を示すようになる。ケイ酸塩ガラス以外のほとんどのガラス系もフルチャー型の温度依存性を示す。アレニウス型:η=exp(6・1)フルチャー型:η=exp-(6・2)ここで、ηは粘度、は粘性の活性化エネルギー、は気体定数、は温度、,,は定数である。b. 機械的・弾性的性質ガラスは脆性固体であり、ほとんどの場合引張応力により破壊する。ガラスは結晶と異なり、粒界や転位などの強度劣化をもたらす要因はもともと存在しないため、ガラスそのものの本質的強度は5~20GPaと極めて大きいと推定されている。しかし、実用ガラスの強度はその百分の1程度の50~100MPaにすぎない。この大きな違いは、ガラス表面に微細なきずが存在し、きずの先端に応力集中が起こることによる。成形直後の清浄な表面をただちに保護膜で覆い、きずの発生を防いだガラス繊維では、3~4GPaの高い強度も得られている。ガラスの弾性係数は組成によって大きく変わるが、実用ガラスのほとんどは50~100GPaのヤング率、0.16~0.28のポアソン比を有する。イオン半径が小さく電荷の大きいイオン、例えばAlOやLaOを含むガラスではヤング率も大きい。ガラスにビッカース圧子を押込むと弾性変形と塑性変形が起こる。荷重を除去した後の弾性回復率は50%程度である。ケイ酸塩ガラスのビッカース硬さは4~8GPa、破壊靱性は0.5~1MPa・m程度である。表6・3には、ケイ酸塩ガラスおよび光機能性ガラスとして注目されているBiO系やTeO系ガラスなどの特性を示す。これらのシリカを含まないガラスの破壊靱性値は非常に小さい。e. 光学的性質ガラスの透明性はガラスの有用性という点では最も重要である。ガラスに光を当てた場合の損失の要因は、表面での反射(空気とガラスの屈折率差や表面状態)とガラス内部での吸収と散乱である。吸収損失には、ケイ酸塩ガラスを構成する成分の固有吸収帯(価電子帯から伝導帯への電子遷移)に対応する紫外域での強い吸収とガラスの格子振動による振動エネルギー準位間の遷移に対応する赤外域での強い吸収がある。不純物を含まないケイ酸塩ガラスでは、約160~4500nmの波長の光に対して透明である。通常の板ガラスに見られる薄い緑色は原料中に不純物として混入している鉄イオンによる吸収によって生じたものである。ビンガラスや各種フィルタ、工芸ガラス、信号機用ガラスなどに見られる色は微量の遷移金属イオンやセレン、銅、金、銀などを添加して得られる。振動数の高い光の領域では、媒質の分極は主として電子分極による。媒質の屈折率と電子分極率の関係は式(6・3)のローレンツ・ローレンツの式(Lorentz-Lorenz optical glass)の開発が進められている。ナノサイズの半導体(CdS,CdSe)や金属(Au,Cu)の超微粒子を分散させたガラス(χ=10~10esu、非線形応答はサブピコ秒)や屈折率の高いBiO系ガラス(χ=10~10esu、非線形応答はフェムト秒)の実用化が検討されている。シリカを主成分とするケイ酸塩ガラス自体の三次の非線形光学効果はχ=10esu程度で小さい。f. 電気的性質一般のケイ酸塩ガラスは室温付近で10~10Ωm,300°C付近でも10~10Ωm程度の抵抗を有する絶縁体である。ガラス内での電荷担体はほとんどの場合がLi,Naなどのアルカリイオンであり、PbO,BaO,SrOのような修飾酸化物が存在すると、アルカリイオンの易導度が低下するため抵抗が増す。また、2種類のアルカリイオンを含むガラスは単一アルカリの場合より数けた高い抵抗を有する(混合アルカリ効果、mixedalkali effect)。ディスプレイ用基板ガラスでは信頼性の点からアルカリ金属を含まない無アルカリのケイ酸塩ガラスが使用されている。また、ガラス表面に透明導電膜をコートする技術が開発されている。
