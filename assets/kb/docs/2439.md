# 6・1・2 特殊な製法・処理によるガラス

> **章**: 第6章 無機材料
> **節**: 6・1 ガラス

a. 石英ガラスSiOは融点が高く通常のタンク窯による溶融成形が不可能なため,原料および製造方法は特殊である.石英ガラス(silica glass)は,1天然水晶やケイ石精製粉を(i)カーボン材を発熱体およびるつぼとして電気的に溶融する,(ii)ベルヌーイ法により酸水素ガス中で溶融する,2四塩化ケイ素(SiCl)ガスを,(i)酸水素火炎中で加水分解する,(ii)CVD 法(化学気相析出:chemical vapor deposition)により熱加水分解して多孔質母材を作製する,3ケイ素のアルコキシドSi(OCH)の加水分解ならびに重合反応を利用したゾル-ゲル法などの方法で作られる.石英ガラスの特性は不純物の種類と量によって大きく異なる.電気溶融した石英ガラスはOH の含有量は少ないが,AlO,NaOなどの不純物が含まれる.酸水素により火炎加水分解したものは,Al,Naは少ないがOH が多い.SiClをCVD 法で作製したガラスは,OH は少ないがClが残存している.ゾル-ゲル法は,低温合成法である,コーティング膜やファイバ化が容易である,有機物とのハイブリッド化が可能であるなどの特徴があり,無反射や紫外線遮へいコーティング膜などの作製に実用化されている.b. 光通信用石英ガラスファイバ光通信用ファイバ(opticalfiber)は信号光(レーザ光)をガラス内部に閉じ込めるために屈2―176折率の高いコア(中心部)と屈折率がわずかに低いクラッド(周辺部)の二重構造になっている.コアとクラッドの屈折率が階段上に変化しているステップ型とコア部の屈折率が放物線状に徐々に変化しているグレーデッド型がある.超低損失で長距離用に使用されているのは石英系ファイバである.短距離用には,多成分系ガラスファイバが使用されており,二重るつぼ法で作製される.低損失石英系ファイバでは,まずファイバの多孔質母材が四塩化ケイ素(SiCl)と四塩化ゲルマニウム(GeCl)の火炎加水分解反応を利用したVAD 法(気相軸付け法:vapor phase axial deposition)で作製され,その後透明化,線引きして直径125 mにファイバ化される.ファイバ中の光の伝送損失は1.55 mがいちばん低いので,この波長のレーザ光が使用されている.なお,光ファイバでの伝送損失は,1km当たりの光の強度の減少(デシベル,dB)で表され,レーザの入射強度を욡,出力強度を욉,伝送距離を(km)とすると,式(6・6)で求められる.=10log욡욉(6・6)例えば,伝送損失が2dB/kmであれば,レーザ光の強度は1km先では約60%になる.0.1dB/kmであれば,1km先でも98%のレーザ光の強度が保持される.現在の光ファイバは0.10~0.2dB/kmという伝送損失が達成されている.通信情報量の飛躍的な増加に対応するため1本のファイバに一つの波長のレーザ光ではなく,1.55 mを中心として複数の波長のレーザ光を通す技術(波長分割多重伝送)や材料の開発が進められている.また,ファイバ伝送中に弱くなったレーザ光を電気信号に変えて増幅するのではなく,希土類イオンの蛍光を利用した光で光を直接増幅する光増幅ファイバも開発されている.現在,1本の光ファイバでは1Tbps以上の伝送速度が実現されている.c. バイコールガラス(Vycor glass)96~98%のSiOを含有するシリカガラスは石英ガラスに非常に類似した特性を示す高ケイ酸塩ガラスであり,ガラスの相分離を利用して作られる.まず,SiO70~75%,BO20~30%,NaO 5~10%のホウケイ酸塩ガラスを1400~1450°Cで通常の方法で溶融,成形した後,500~600°Cで加熱してNaO-BO相とSiO相に分相させる.酸(HCl)でNaO-BO相を溶出し,SiO相に富む多孔体を得た後,約1200°Cまで加熱して細孔を消滅させると透明なガラスが得られる.コーニング社の製品ではバイコールガラスと呼ばれている.なお,多孔質シリカガラスは固定化酵素の担体として利用される.理化学用に使われているパイレックスガラスは約82%程度のSiOを含むNaO-BO-SiO系ガラスである.d. 強化ガラスガラスは引張応力により破壊するのであらかじめガラスの表面に圧縮ひずみ層を形成しておくと破壊強度が増す.強化ガラス(tempered glass)の製法には,1所定形状のガラスに軟化点付近まで加熱しておいて圧縮空気を吹付け急冷する物理強化法(physical tempering),2ガラスを徐冷点以下の低い温度域で,イオン半径の大きいアルカリイオンを含む溶融塩中に浸漬して,ガラス中のアルカリイオンとの間でイオン交換を行う化学強化法(chemical tempering)がある.物理強化(風冷強化とも呼ばれる)では車両用窓ガラスなどに,化学強化は軽量ビン,眼鏡レンズなどに応用されている.f. 結晶化ガラス結晶化ガラス(crystallized glass)とは,前駆体ガラスを制御された条件のもとで熱処理を行い,所望の結晶を析出させた結晶集合体である.試料全体が結晶になる必要はなく,ガラス相が残存していても構わない.ガラスセラミックス(glass ceramics)とも呼ばれる.ゼロ膨張高強度,生体適合性,低誘電率,快削性などの特性を有する製品が作られており,耐熱食器,高級建材,人工骨,IC用多層基板などに応用されている.最近はCdS(cadmium sulfide)などの半導体超微粒子,Erなどの希土類イオンドープ,第二高調波発生を示す光非線形性などの光機能を顕著に示す結晶化ガラスが非常に注目されている.また,通常図6・1 フォトクロミックガラスフォトクロミックガラス(photochromic glass)はNaO-BO-AlO-SiO系ガラスの原料に微量のハロゲン化銀(AgBr,AgCl)とCuOなどを添加して溶解し,冷却の際に5~20nm程度のハロゲン化銀結晶(AgX)を内部に析出させたガラスである.紫外線を照射するとAg とX に分解することにより暗化し,照射を止めるとAg とX が再結合して再び無色透明になる.Cuは増感剤として働く.フォトクロミックガラスは眼鏡レンズとして実用化されている.最も重要な特性は暗化度,暗化速度,退色速度であり,退色速度が最高のものでも退色時間は数十秒であるので,光速スイッチには使用できないが,低速の表示装置への応用などが考えられている.
