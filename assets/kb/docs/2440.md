# 6・1・3 非ケイ酸塩ガラス

> **章**: 第6章 無機材料
> **節**: 6・1 ガラス

a. ホウ酸塩ガラスBOはSiOとともに重要なガラス形成酸化物であり,多種多様なガラスを作製するうえで必要不可欠である.BO単一成分でできたガラスではB 原子は3配位(BO)であるが,アルカリ酸化物を導入すると一部のB 原子は4配位(BO)に変化する.ホウ酸塩ガラス(borate glass)は,1熱中性子の吸収率が高い,2X 線透過率が高い,3アルカリ金属蒸気(Na,Cs)の作用に耐える,4電気絶縁性が高い,5低融点である,6希土類酸化物を多量に含有しうるなどの特徴がある.低融点の特徴は,接着用ガラス(BO-ZnO-PbO),複合系ハンダガラス(BO-PbO+β-ユークリプタイトなどの低熱膨張結晶粉末)などに利用されている.接着用ガラス(solder glass)は,金属とガラスの封着やセラミックスとガラス,ガラスとガラスの接合などに用いる低融点ガラスで,各種電子部品の組立に重要である.最近ではPbOの入っていない低融点のホウ酸塩ガラスの開発が急がれている.ランタンクラウン(BO-CaO-LaO-ZrO),ランタンフリントと呼ばれる高屈折率低分散光学ガラスはLaO,GdOなどの希土類酸化物を多量に含むことが必要であり,BOの特徴が生かされている.b. リン酸塩ガラスPOガラスでは,P原子は4配位を取るが5価であるため,3個のP-O-P結合と1個の二重結合P=Oからなっている.NaOのような網目修飾酸化物を加えていくと鎖状構造になる.POはSiOに比較して結晶化しやすい,耐薬品性が低い,揮発性が高い,液相温度付近での粘度の温度による変化が大きいなどの欠点があるが,1イオン力場が強く修飾酸化物を多量にガラス中に分散させることができる,2耐フッ酸性が高い,3紫外線を透過し赤外線を吸収しやすい,4高原子量の元素を多量に含有しうる,5低融点である,などの特徴を有する.これらの特性を利用したリン酸塩ガラス(phosphate glass)には,耐フッ酸性ガラス(PO-AlO-ZnO),紫外線透過ガラス(PO-AlO-CaO),熱線吸収ガラス(PO-BO-AlO-MgO-FeO),放射線線量計ガラスなどがある.c. ゲルマン酸塩ガラス(germanate glass)GeOはSiOと同様にガラス中では四面体(GeO)構造をもつことから,多くの点でケイ酸塩ガラスと類似している.GeOは非常に高価な酸化物であり,耐薬品性が低いことから,工業製品にはわずかに赤外透過性ガラス(GeO-AlO-CaO)がある程度である.SiO系光ファイバでは,屈折率の高いコア部分を作製するのにGeOが少量ではあるが必要不可欠である.光機能性ガラスの分野では注目され始めている.d. 亜テルル酸塩ガラスTeOを主成分とするガラスで,一般にテルライト系ガラス(tellurite glass)と呼ばれている.1屈折率が高い(1.8~2),2誘電率が高い(10~30),3融点が低い(800°C程度),4赤外線をよく通す(5 m程度)などの特徴を有する.単独ではガラスになりにくいが,NaO,BaO,WOなどを少量加えると容易にガラスになる.フォノンエネルギーが小さい2―177(最大の格子振動エネルギー:700~800cm)ために,Erなどの希土類イオンの蛍光特性が優れており,波長分割多重用(広帯域)の光増幅ファイバとして活発に研究されている.また,大きな三次の非線形光感受率を示すことから,非線形光学ガラスとしても期待されている.e. ビスマス酸塩ガラスBiOを主成分とするガラスで,一般にビスマス系ガラス(bismuthate glass)と呼ばれている.TeO系ガラスと同様に,高屈折率,高誘電率,低融点,高赤外透過性の特徴を有する.BiO-ZnO-BaO-LiO系ガラスはガラス転移温度が300°C程度であり,結晶化に対しても比較的安定であることから大きなバルク状試料が得られている.PbOを含まない新しい封着用の低融点ガラスとして研究が進められている.最近では,三次の非線形光学ガラスや増幅用光ファイバとしても注目されている.f. バナジン酸塩ガラスVOは単独ではガラス化しないが,少量のBaO,PbOなどが存在するとガラス化する.通常,バナジウムイオンはガラス中ではVとVで存在し,ホッピング機構により電子伝導性を示す.さらにLiやAgに対して優れたインターカレーション特性を示すことから,充放電可能な二次電池用カソード材料として注目されている.g. フッ化物ガラスBeF,ZrF,HfFなどを主成分とするフッ化物系は酸化物ガラスと同様に融液の急冷で無色透明なガラスになる.特に,ZrF-BaF-LaF-AlFを基本系とするZrF系フッ化物ガラス(fluoride glass)が有名である.フォノンエネルギーが小さい(500~600cm)ために,Erなどの希土類イオンをドープしたガラスは波長分割多重用の光増幅ファイバとして期待されている.8~10 m程度までの赤外光を通す.リン酸とAlFなどのフッ化物からなるフツリン酸塩ガラスは非線形屈折率が非常に小さいため(レーザの自己収束が避けられる),核融合用の巨大出力用ガラスレーザ母体として実用化されている.h. カルコゲン化物ガラスカルコゲン元素S,Se,Teを主成分とするガラスで酸素は含まれていない.カルコゲン化物ガラス(charcogenide glass)は,高赤外透過性(~16 m),電子伝導性(室温での比抵抗10~10Ωm),特異な光誘起現象(光伝導性,構造変化)などの特徴を有する.赤外透過性ガラス(As-S,Te-Ge-Si),電子写真や撮像間用の光伝導ガラス(Se,Se-Te),非線形電流電圧特性を利用したスイッチ素子ガラス(Te-As-Ge-Si),ガラス-結晶間の相変化を利用した光メモリ(Ge-Te,Te-Ge-Sn),フォトドーピング材料など幅広い機能を発現する.i.
