# 6・4・1 耐火物の種類と性質

> **章**: 第6章 無機材料

a. 耐火物の概要耐火物(refractory)とは、高温で溶融し難い非金属材料の総称であり、JIS R 2001:1995では1500°C以上の定形耐火物、および最高使用温度が800°C以上の不定形耐火物、耐火モルタルならびに耐火断熱れんが、と定義している。高温で用いられる構造材料である耐火物は、その目的に応じて安全に使用されるために、前もってその特性を正しく把握(評価)しておく必要がある。そこで、評価すべき性質を表6・13に示した。また、その中で、特に重要ないくつかの諸特性について、その意義等を次に示す。. PCE)、ISOコーン(ISOcone)がある。酸化物系耐火物の耐熱性を示す目安として重要だが、近年増加している炭素、炭化物、窒化物などの非酸化物を含有する耐火物の場合は、必ずしも判断基準となり得ない点に注意を要する。. 圧縮強さJIS R 2206:1996では、一辺約60mmの立方体試験片2個を用いて測定するよう規定されている。室温(冷間)もしくは熱間で測定される。耐火物は圧縮強さが引張強さに比べて数倍大きい。それゆえ、引張応力の生じない状態で用いるのが望ましく、圧縮強さは設計上重要な性質である。. 荷重軟化(refractoriness under load)およびクリープ(creep)i項の耐火度は無荷重下の変形だが、自重を含めて、荷重下では耐火度よりもはるかに低い温度で変形が始まる。各種窯炉など、構造体の高温でのひずみを防止するために必要な特性である。高温での変形を追跡するため、荷重軟化点(JIS R 2209:1991)と圧縮クリープ(JIS R 2658:1995)試験法が規格化されている。荷重軟化点は、直径50mm、高さ50mmの試験片を一定荷重(0.2MPa)下で昇温する。このとき、試験片の高さ方向の寸法変化を追跡し、軟化変形(収縮)を開始する温度を点、および2%,20%変形する温度を(°C)、(°C)点として表す。クリープ試験は、一定の荷重と温度のもとで、変形量を時間的変化として測定する。. 熱膨張(thermal 耐火物の種類と特性耐火物は、品質や形態が多岐にわたっているため、画一的な分類が困難であるが、いくつかの分類例を以下に示す。. 化学的性質による分類化学的な性質の酸とアルカリという見方から、耐火物は酸性(acidic: RO)、中性(neutral: RO)、塩基性(basic: RO)に区分できる(図6・9)。この分類は、高温における異種耐火物どうしの相互反応や耐火物と溶融スラグなどとの反応性を理解するうえで重要な意義がある。例えば、ケイ石れんが(SiO:酸性)とマグネシアれんが(MgO:塩基性)を接触させて使用すると、おのおのの融点(もしくは、耐火度)よりはるかに低い温度で相互反応を起こし溶融してしまうため、留意する必要がある。表6・15に各種耐火物の成分による分類例を示した。これまで、雰囲気炉をはじめ、各種の溶解炉(容器)でも酸化物系耐火物が多かった。しかし最近では、特に、溶融スラグや溶融金属容器用には炭素類を含有する複合耐火物の生産量が増大している。.
