# 6・5・1 ニューカーボンの種類

> **章**: 第6章 無機材料
> **節**: 6・5 ニューカーボン

定義が緩やかであるため、ニューカーボンの種類は非常に多岐にわたっている。年代順にいうと、1960年代に開発された炭素繊維は代表的なニューカーボンといえよう。炭素繊維の出現は、複合材料に新しい局面を切り開いたといっても過言ではない。熱分解蒸着炭素は気相熱分解という新しい製造方法によって得られ、炭素材料の基本構造である六角網面の強い異方性をもつ材料として具体化したものである。ガラス状炭素材料は非晶質に近い構造でガス不透過性、高い硬度など色が黒いことを除けば、性質はガラスに極めて近いものでその開発はそれまでの炭素材料に対するイメージを一新した。高密度等方性炭素材料は、一般炭素材料の製造プロセスに静水圧プレス法(CIP法:cold isostatic pressing)を採用することによって初めて得られた、均質で、異方性の少ない材料である。このほか、黒鉛層間化合物を出発原料とするシート状黒鉛は、脆性材料である従来の炭素材料には全く期待できなかった可撓性を示している。このほか粉体形状のものでは、1表面修飾活性炭、2分子ふるい炭、3メソカーボン、4フッ化炭素、5リチウムイオン二次電池負極材などがあり、炭素繊維についても原料の違いによってアクリル系、PAN 系、レーヨン系、フェノール樹脂系などがあり、気相成長炭素繊維もある。さらに気相成長技術を発展させた気相析出硬質炭素、気相合成ダイヤモンドがある。最近の話題として、1996年度ノーベル化学賞を与えられたフラーレンの発見があり、炭素原子60個がサッカーボール状に結び付いたCのほか、これに続くC、さらに大きな高次フラーレン類、そして微細な中空体であり、高温における電子デバイスへの応用が期待されるカーボンナノチューブもまたニューカーボンの仲間である。このように、ニューカーボンについては枚挙にいとまがない。そこでここでは、ソリッドとしてほとんどそのまま機械用に利用できる材料について限定して述べる。
