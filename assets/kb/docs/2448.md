# 6・5・3 ニューカーボン適用製品

> **章**: 第6章 無機材料
> **節**: 6・22 に示す。

a. 等方性高密度炭素材料当初、機械的特性が優れているためホットプレス用型材として、また精密加工が可能でかつ電極消耗に方向依存性がないことから放電加工用、電極用に開発され、使用された。さらに全灰分を100ppm以下まで純化できることからSi単結晶引上げ用ヒータ、るつぼあるいは封着ジグなど半導体用材料としても多量に用いられている。このほか原子力関係では多目的高温ガス炉用減速材、反射材などへ利用されている。b. 熱分解蒸着炭素材料熱分解蒸着炭素材料は高純度で核的性質に優れ、かつ不通気性で耐熱性が高いので核燃料の被覆材として好適である。熱的異方性および比強度の大きい点ではロケットノズル、スロート、ノーズコーンなどの宇宙航空機材、電気的異方性を有する半金属的性質からエレクトロニクスおよびマイクロ波の分野、不通気性と塑性を利用しての高温および腐食性流体に対するガスケット、熱膨張の異方性を利用した高温計、その他熱遮へい材、ヒータ、ジグ用るつぼ、ボート、分析用電極などに応用されている。c. ガラス状炭素材料ガラス状炭素材料は高純度であるうえに炭素粉の発生が少なく、吸着ガス量も少ない耐熱性の高強度材料表6・23 シート状炭素材料の特性品種特性かさ比重1.0かさ比重1.5引張強さMPa3.98.7面方向熱伝導率W/(mK)厚さ方向10%変形荷重(室温) MPa圧縮率1t/cm%復元率1t/cm%最低有効締付圧MPa6.813.6ガスケット係数-200~3000~450~600(不活性気中)(空気中)(空気中、ガスケットとして使用の場合)適用温度範囲°C面方向3×10熱膨張係数1/K厚さ方向1×10応力緩和20MPa,100°C,22h%2.5厚さ増加率%耐油性重量増加率%厚さ増加率%耐水性重量増加率%厚み増加率%1.2強熱減量重量増加率%であることから、腐食性の強いGaAs,Siなどの半導体用単結晶引上げ用のるつぼ、サセプタ、ボートなどに使われる。また耐熱性、耐リン酸性、耐酸化性、気体不透過性、小さい熱膨張係数、高い熱および電気伝導度など非常に多くの要求項目のあるリン酸型燃料電池用サセプタとしても採用されている。そして最近は平滑度を高めたものはテープレコーダ、VTR やフロッピーディスクの垂直記録磁気ヘッド用としても使われている。さらに導電性と低反応性を利用した発光分光用電極、ポーラロ用電極、電位差滴定用電極などがあり、金属に対する安定性から溶融塩電解用るつぼへの利用、発じんが少ないことから歯根材、各種関節等の生体用材料への展開もされている。d. シート状炭素材料グランドパッキンとして多量に使用されるが極低温から高温まで広い温度範囲にわたって使用でき、その間でほとんど性能の変化がない。また低い締付圧力によって十分にシール効果を示し、増し締めの必要が少ないなどの利点があるため各種バルブ、回転ポンプ、かくはん機、スートブロアほかのシャフト部のシール材として使われる。ガスケットとしては配管用、バルブ用、流量計用、ポンプ用、コンプレッサ用、熱交換器用などのほかシリンダガスケット、マニホルドガスケットなど自動車用としても多く使用されている。このほか高温炉用の断熱材、炉壁保護用内張材、クッション材などへの利用もある。文献(1) 境野照雄・ほか9名編、ニューガラスハンドブック、(1991)、丸善。(2) 速水諒三・ほか76名編、光機能材料マニュアル、(1997)、オプトロニクス。(3) 山根正之・ほか6名編、ガラス工学ハンドブック、 (1999)、 朝倉書店。(4) Komatsu,T.、ほか3名、J.Am.Ceram.Soc.,84-10(2001)、2401.(5) Komatsu,T.、ほか5名、Opt.
