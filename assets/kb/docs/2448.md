# 3・3・4 傷害基準と衝撃耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・3 頭部傷害のバイオメカニクス

衝撃と対側の脳実質に脳挫傷が発生する対側損傷（contre coup
injury）がしばしば起こっている．図3･9(ｃ)は転落により後頭部
を打撃し，対側の前頭葉に脳挫傷が発生した例である．
(ｄ) びまん性軸索損傷：CTやMRIによる画像診断で小規模
の出血を脳梁，脳幹部やその周辺のクモ膜下腔などで確認できる．
図3･9(ｄ)は，自動車事故で意識障害が数週間継続し，左右の大脳
を連結する脳梁に出血による高密度領域が生じた例である．びまん
性軸索損傷は交通事故などで発生している．
