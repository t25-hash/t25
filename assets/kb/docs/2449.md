# 3・4・1 頸部の解剖学（anatomy)

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・4 頸部傷害（neck injury）のバイオメカニクス

頸部の傷害として，頭蓋-頸椎接合部の骨・血管損傷，椎間板お
よび頸椎椎体椎弓損傷，脱臼，靱帯・筋損傷があるが，神経などの
軟部組織損傷の実体については明らかでない．ここでは頸部の傷害
発生について，その理解に必要な知識としての解剖学を大まかに頸
椎，椎間板，椎間関節，靱帯，ならびに筋肉に分類 して示す．
a.頸椎（cervical vertebrae) 頸部は，頭部と体幹（torso）
を連結し，神経中枢が集中する重要な部位である．神経器官を保護
するように，第1頸椎（C1）から第7頸椎（C7）までの七つの頸
椎が椎間板，靱帯，筋肉組織により連結されている．第1頸椎（環
椎：atlas ），第2頸椎（軸椎：axis ），第7頸椎（隆椎）の形状は
他の頸椎とは異なっているが，C3～C6頸椎はほぼ同じような形
状となっている．頸椎の後方には椎弓（図3･15中の1）が位置し，
その先端には棘突起（図3･15中の2）がある．椎弓は中央に椎孔
（図3･15中の3）を形成しており，その中を脊髄が走っている．そ
の外側は横突起（図3･15中の4）につながっている．横突起には
神経の通る脊髄神経溝（図3･15中の5）が存在しており，横突起
の後方には上下関節突起（図3･16中の6，7）が突出し，その先端
には関節面が存在し椎間関節を形成している．
環椎（図3･17，3･18）には椎体が存在せず，前弓（図3･17，3･
18中の8）と後弓（図3･17，3･18中の9）からなる環状の椎骨で
ある．大きな椎孔の外側には上関節窩（図3･17中の10）と下関節
窩（図3･18中の11）が存在する．後頭骨との間には，環椎後頭関
節を，軸椎（図3･19）との間には軸椎の上関節面と外側環軸関節
を形成している．軸椎は，上面に歯突起（図3･19中の12）をも
ち，他の頸椎と形状が異なっている．歯突起の前後面には関節面が
あり，前面は正中環軸関節（図3･19中の13）が形成されている．
b.椎間板（intervertebral disk) 椎間板（図3･20）は，外
周を取り巻く線維輪（anulus fibrosus ，図3･20中の14）と柔らか
い髄核（nucleus pulposus ，図3･20中の15）とからなっている．
髄核は，70～90%が水分で，椎間板の30～50%を髄核が占めてい
る．その特性は粘弾性を示し，脊柱に加わる圧力を弾力的に受け止
めるクッションとして働き，軸方向，せん断方向，回転方向の自由
度をもつ．
c.椎間関節（facet joint ) 頸部の椎間関節は胸部や腰部と比
較して柔軟で，関節包は広く，関節間に滑膜ひだ（図3･21中の
16）が存在している．これにより，運動の自由度が促進され，頸部
全体として複雑な動きが可能となる．
d.靱帯（ligament )
.前縦靱帯（ALL:anterior longitudinal ligament ，図3･
22中の1) 前縦靱帯は強固な線維質で，環椎から仙骨までの椎
体の前表面を結合している．前縦靱帯は椎体には強固に付着してい
るが，椎間板を形成する線維輪との結合はなく，独立して存在して
いる．
.後縦靱帯（PLL:posterior longitudinal ligament ，図3･
23中の2) 後縦靱帯は歯突起や環椎横靱帯を保護する蓋膜から
発生し，仙骨までの椎体後面に付着し，椎体の上面と下面で結合し
ている．後縦靱帯は椎間板と一体化して結合しており，椎間板を保
護する役割ももっている．
.黄色靱帯（LF:ligamenta flava，図3･24中の3) 黄
色靱帯は軸椎から仙骨の椎弓間に張っている．脊柱直立の静止時に
もこの靱帯は緊張している．黄色靱帯は弾性線維でできているのが
特徴であるが，年齢が増すにつれ弾性が低下するといわれている．

e.筋肉組織
を屈曲させる役割をもつ椎前筋に属し，上斜線維（図3･27中の
2），下斜線維（図3･27中の3），垂直線維（図3･27中の4）の3種
類の線維群に分かれている．垂直線維は最も内側に位置しC5～
C7から起こり，C2～C4に付着している．上斜線維は，C3～C5
の横突起から起こり環椎の前方（前結節）につながっている．下斜
線維は，第1胸椎（T1)～第3胸椎（T3）から起こりC5～C7横
突起に付着している．両側の頸長筋が作用すると頸部は屈曲し，片
側のみ作用すると側屈を引き起こす．
様に椎前筋に属し，C3～C6の横突起が起点となっており，後頭
骨の底部に終点をもつ．その働きも頸長筋と同じように屈曲と側屈
を引き起こす．

筋に含まれることがある．前斜角筋はC4～C6の横突起から起こ

.伸展を促す筋群（extensor muscles ）
(１)頭最長筋（longissimus capitis ，図3･28中の1）と頸最
長筋（longissimus cervicis ，図3･28中の2) 頭最長筋の始点
は，T2～T6の横突起から起こり，後頭骨の乳様突起に付着する．
頸最長筋の始点はT1～T6の横突起にあり，C2～C5の横突起の
後結節で終わる．両者が作用すると，脊柱は反り，肋骨を引き上げ
る．
(２)頭板状筋（splenius capitis ，図3･28中の3）と頸板状筋
（splenius cervicis ，図3･28中の4) 頭板状筋は，T1～T3の
上位3胸椎とC4～C7の下位3頸椎の棘突起から始まり，後頭骨
の乳様突起に付着する．頸板状筋はT4～T6の棘突起から始ま
り，C1～C2の横突起に連結する．これらの筋が両側で作用する
と，頭部および頸部を伸展させる．
(３)頭半棘筋（semispinalis capitis ，図3･29中の1）と頸半
棘筋（semispinalis cervicis ，図3･29中の2) 頭半棘筋の始点
は，T4～T7の横突起，C3～C7の関節突起から始まり，個々が
5個以上の椎骨を飛び越して，最終的に後頭骨の上項線と下項線の
間に連結する．両側が働くと頭部および頸部を伸展させる強力な筋
である．
(４)多裂筋（multifidus ，図3･29中の3) 多裂筋は多数の
小さな筋束からなっており，仙骨からC2まで達している．頸部で
は，胸椎の横突起，C4～C7の関節突起から始まり，2～4個の椎
骨を飛び越してそれぞれより上にある椎骨の棘突起でつながってい
る．その作用は，両側が働くと伸展作用を引き起こし，長い筋束ほ
ど強く働く．
.その他の筋群
(１)僧帽筋（trapezius ，図3･30中の1) 背部の筋肉で，
後頭骨の上項線，外後頭隆起，項靱帯，第7頸椎以下の全胸椎の棘
突起から始まり，鎖骨の外側に終わる．静力学的には肩甲骨を保持
し，上腕を固定する役割をもち，能動的には肩甲骨と鎖骨を脊柱側
へ向かって引き寄せる．頸椎から起こる僧帽筋の下行部におけるこ
の作用が，頸部を伸展させる．
(２)胸鎖乳突筋（sternocleidomastoideus ，図3･30中の2)
胸鎖乳突筋は胸骨と鎖骨から始まり，後頭骨の乳様突起と上項線で
終わり，そこでは僧帽筋との腱性の結合が見られる．両側の胸鎖乳
突筋が収縮すると，頭部は首をすくめたように後方に傾く．
