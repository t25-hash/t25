# 7・1・1 ファインセラミックスの機能別分類と特徴

> **章**: 第7章 セラミックス
> **節**: 7・1 ファインセラミックス

先進的なセラミックス(ceramics)は、家電、自動車、電子・情報、航空・宇宙などの新しい科学技術と産業を支えるキー材料として、1950年代に登場し急速に進歩して今日に至っている。わが国で、このようなセラミックス群をファインセラミックス(fineceramics)と呼ぶようになったのは1980年頃であり、ファインセラミックスの分類に関する議論も盛んに行われて表7・1に示す機能別分類ができあがった。この分類によれば、ファインセラミックスは電気・電子的機能、機械的(力学的)機能、光学的機能、熱的機能、生化学的機能に大別される。現在では、ファインセラミックスという用語はISO/TC206としても認知されているが、歴史の経過とともにファインセラミックスは応用分野から構造材料と機能材料に分類されるようになった。日本ファインセラミックス協会がまとめたわが国のファインセラミックス産業の生産高は、2002年の時点で約2兆円であり、これまでほぼ右上がりに成長してきた。そのうち7割が機能性セラミックス、残りが構造用セラミックスとなっており、前者の伸びが著しい。産業規模としては決して大きいものではないが、わが国が最も強い分野であり、その波及効果は約50兆円ともいわれている。しかし、材料側から見れば、機能用と構造用の区分は難しい。例えば、アルミナ(AlO)や炭化ケイ素(SiC)のように、同じ物質が多くの機能を備えて両分野に貢献しているものがほとんどである。元来、セラミックスは多機能性を特徴とする材料群であるといってもよいことから、このような分類は適切ではないのかもしれない。ここでは、機械工学と関連深い機械材料としてのセラミックスに力点をおいて解説する。
