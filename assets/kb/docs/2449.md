# 7・1・1 ファインセラミックスの機能別分類と特徴

> **章**: 第7章 セラミックス
> **節**: 7・1 ファインセラミックス

先進的なセラミックス(ceramics)は,家電,自動車,電子・情報,航空・宇宙などの新しい科学技術と産業を支えるキー材料として,1950年代に登場し急速に進歩して今日に至っている.わが国で,このようなセラミックス群をファインセラミックス(fineceramics)と呼ぶようになったのは1980年頃であり,ファインセラミックスの分類に関する議論も盛んに行われて表7・1に示す機能別分類ができあがった.この分類によれば,ファインセラミックスは電気・電子的機能,機械的(力学的)機能,光学的機能,熱的機能,生化学的機能に大別される.現在では,ファインセラミックスという用語はISO/TC206としても認知されているが,歴史の経過とともにファインセラミックスは応用分野から構造材料と機能材料に分類されるようになった.日本ファインセラミックス協会がまとめたわが国のファインセラミックス産業の生産高は,2002年の時点で約2兆円であり,これまでほぼ右上がりに成長してきた.そのうち7割が機能性セラミックス,残りが構造用セラミックスとなっており,前者の伸びが著しい.産業規模としては決して大きいものではないが,わが国が最も強い分野であり,その波及効果は約50兆円ともいわれている.しかし,材料側から見れば,機能用と構造用の区分は難しい.例えば,アルミナ(AlO)や炭化ケイ素(SiC)のように,同じ物質が多くの機能を備えて両分野に貢献しているものがほとんどである.元来,セラミックスは多機能性を特徴とする材料群であるといってもよいことから,このような分類は適切ではないのかもしれない.このことは,先進的な材料として注表7・1 セラミックスの機能-材料-応用用途分類表機能大分類酸化物セラミックス機能材料応用電気・電子的機能絶縁性AlO,BeO基板絶縁性C,SiC,AlN基板誘電性BaTiO,TiOキャパシタ導電性SiC,MoSi発熱体圧電性Pb(ZrTi욪)O,ZnO,SiO発振子,着火素子,表面弾性波遅延素子半導体電子放射性SiCLaBバリスタ,避雷器電子銃用熱陰極磁性Zn욪MnFeO記憶・演算素子,磁心半導性SnO,ZnO-BiO,BaTiOMn-Co-Ni系酸化物ガスセンサ,バリスタ,抵抗素子イオン導電性β-AlO,安定化ZrONaS 電池,酸素センサ力学的機能耐摩耗性切削性AlO,ZrO研磨材,砥石,切削工具耐摩耗性,切削性BC,ダイヤモンド,c-BN,TiC,WC,TiN耐摩耗材,砥石,切削工具強度機能SiN,SiC,サイアロンエンジン,耐熱・耐食材料,離型材潤滑機能C,MoS,h-BN光学的機能蛍光性YOS :Eu蛍光性透光性AlON,N 含有ガラス窓材透光性AlOナトリウムランプの外とう管光反射性TiN集光材偏光性PLZT光学偏光素子導光性SiO,多成分系ガラス光通信ファイバ熱的機能耐熱性AlO断熱構造材耐熱性SiC,SiN,サイアロン,h-BN各種耐熱材原子力関連機能原子炉材UOBeO核燃料減速材原子炉材UCC,SiCC BC生化学的歯骨材AlO人工歯骨耐食性h-BN,TiB蒸着容器機能Ca(F,Cl)POSiN,サイアロンポンプ材,ほか各種担体性SiO,AlO触媒担体C,SiC耐食部材Si-Al-O-N目されるプリセラミックス,シナジーセラミックス,ナノセラミックス,その他の新しいセラミック複合材料などにおいては,機能の融合が目標となっていることから見ても明らかであろう.ここでは,機械工学と関連深い機械材料としてのセラミックスに力点をおいて解説する.
