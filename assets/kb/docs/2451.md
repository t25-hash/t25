# 7・1・3 機械材料としてのセラミックス

> **章**: 第7章 セラミックス
> **節**: 7・1 ファインセラミックス

a. 機械用セラミックスの概要セラミックスは金属材料と比較して、(1)耐熱性、(2)耐食性、(3)耐摩耗性に優れた性質をもっている。しかし、一方でもろいという機械材料としては致命的な欠点をもつため、これまでは主として低応力あるいは静的な環境下で使用されてきた。しかし、セラミックスがもっている耐熱、耐食、耐摩耗などの特性は、金属材料を凌駕するものが多いことから、これを機械材料として利用するための研究開発が積極的に進められてきた。このような機械用セラミックスをエンジニアリングセラミックス(engineering ceramics)と呼んでいる。その課題は第一義的には脆性の克服に帰着するが、個々のセラミックスがもつ魅力的な機能を引出すことが基本であり、これを最大限に発現させることが最も重視される。エンジニアリングセラミックスは表7・3のように酸化物と非酸化物に大別される。おもなものとしては、酸化物ではアルミナ、ジルコニア(ZrO)、ムライト(3AlO・2SiO)とコーディエライト(MgAlSiOあるいは2MgO・2AlO・5SiO)やチタン酸アルミニウム(AlTiO)に代表される低熱膨張材料などが挙げられる。また、非酸化物材料としては炭素材料、窒化ケイ素(SiN)、サイアロン(SiAlON)、窒化ホウ素(BN)、炭化ケイ素(SiC)、炭化タングステン(WC)、炭化チタン(TiC)、ホウ化物などを挙げることができるが、機能はそれぞれの構成物質と微構造で決まることから、構造制御が重要な課題となって多くの材料が開発され実用化されてきた。最近では、セラミック微細粒子、ウィスカあるいは長繊維で強化した複合材料も研究されており、セラミックスをマトリックスとした複合材料は将来の高靱性セラミックスとして注目されている。酸化物と非酸化物セラミックスについて概略を紹介する。b. アルミナ(alumina)AlOには多くの多形が存在する。その中でも最も安定なα-アルミナは、六方晶系のいわゆるコランダム型の構造をもつ化合物である。製造の過程でγ,θ,δなど種々の多形があるが、これらはいずれも準安定相であるので、一般にアルミナセラミックスといえばα-アルミナをいう。非酸化物に比べるとイオン結合性が高い化合物で、単結晶は無色透明である。融点が2050°C、比重が3.99、電気的には優れた絶縁性を示す。原料粉末は、ボーキサイト[主成分がAlO(OH)、AlO・HOあるいはAl(OH)、AlO・3HO]中のアルミナ成分をオートクレーブ中170°CでNaOH 水溶液に溶解させ、ろ過した飽和溶液からAlO・3HOを析出させ、これを1100°C前後に焼成して作られる。900°Cで焼成するとγ-アルミナが得られるが、1000°C以上では安定型のα-アルミナ結晶に転移する。このような工程を経て、高純度のアルミナ粉末が比較的容易に製造されている。また、アルミニウムアルコキシドの加水分解によって合成する方法も実施されており、多彩な粉末が工業的に供給されるようになっている。アルミナは焼結性に富む物質であるので、緻密化は容易であるが、粒成長も同時に起こるため少量の気孔が残存する。そのため、通常は粒成長抑制剤として微量のマグネシアを添加して、品質の優れたアルミナ焼結体が製造されている。表7・4にアルミナセラミックスの特性を他の酸化物と比較して示す。強度は総じて低いが高い硬度をもち、耐熱性、耐摩耗性、化学的安定性などに優れる。また取扱いやすく、経済的にも有利であるため応用範囲は極めて多岐にわたる。炉心管や耐熱ジグとして広く使われており、耐熱性と透光性を生かしたナトリウムランプの発光管への応用はアルミナのもつ特性を効果的に活用したものの一例である。そのほか、耐摩耗材料として繊維用の糸道やガイド、各種ノズル、しゅう動部品、伸線用ノズル、ポンプのメカニカルシール、切削工具などに利用されている。しかし、靱性が劣り、高温での強度低下や熱衝撃に弱いなどがこの材料の弱点となっている。そのため、特性改善を目的として、複合系の研究がなされ、強度・靱性がかなり改善された。AlO-TiC系とAlO-SiC(ウィスカ)系やAlO-ZrO(粒子)系などでその効果が確認されている。アルミナは溶融によっても作られる。原料をアーク炉で溶融した後、固化したものを種々の粒度に紛砕する。純粋なアルミナを溶融したものは白色を呈するため、ホワイトアランダム(WA 砥粒)と呼ばれる。アルミナに酸化チタンTiOを数%含むものを溶融すると、茶褐色で靱性に富むアルミナが得られる。これをアランダムと呼んでいる。これらは研磨材に利用されている。そのほか、溶融したアルミナを固化する過程で育成された単結晶は、軸受、レーザ材料や宝石などに用いられている。. ジルコニアジルコニア(zirconia)は温度によって転移し、単斜晶系(~1170°C)、正方晶系(1170~2370°C)、立方晶系(2370~2680°C)の3種類の構造をとる。これを防止するために、YO,CaO,MgO等の酸化物をジルコニアに固溶させることにより、結晶格子内に酸素欠陥を生じさせて立方晶系として安定な構造に変換させている。これを安定化ジルコニア(stabilized zir-conia)と呼ぶ。安定化ジルコニアは常温から高温域まで安定であるが、強度が低いため構造用ではおもに耐火物として用いられている。しかし、酸素イオンの拡散速度が大きいため、イオン導電性を利用した酸素センサとしては多量に生産されている。近年、ジルコニア微粒子を原料として用い、安定化剤の量を少なくすると強度や破壊靱性が飛躍的に向上することが見いだされた。この場合は正方晶系が準安定相として存在し、一部立方晶系で安定化されているため、部分安定化ジルコニア(PSZ : partial stabilized zirconia)と呼ばれている。最近では、全域を正方晶系にした正方晶ジルコニア(TZP: tetragonal zirconia polycrystalline)も開発されている。代表的な高靱性ジルコニアの特性を表7・4に示した。表7・4から明らかなように、常温近辺での破壊靱性値や破壊強度が酸化物セラミックス中で最も高く、耐摩耗性に優れることが特徴となっている。また、熱膨張係数が大きく、耐熱合金とほぼ同等の値を示すこと、断熱性に優れること、比重が6前後であるなどもこの材料が他材料と区別されるところである。しかし、正方晶系が準安定相であるため、現在のところ使用温度は室温近辺に限定される。以上のような性質により、高靱性ジルコニアは、その特徴を利用して、エンジン部材、各種刃物、工具類、さらには粉砕用のボールなどに利用されている。. ムライトアルミナとシリカから構成される唯一の安定な化合物で、3AlO・2SiOで表される。粘土鉱物を焼成することによって生成される化合物であり、耐火物として広く利用されてきた。ムライトは融点が1850°Cと高く、熱膨張係数が小さい。さらに弾性率が比較的低いことなどから、アルミナより安価で耐熱衝撃性に優れる材料として炉心管、理化学製品などに用いられている。これに対して、高純度ムライトが開発された。これはゾル-ゲル法によって合成された高純度で活性の高い微粉末を使って、粒界に不純物やガラス質などの低融点物質を残さないで焼結が可能になったことによるものである。この新しく開発された材料は、靱性は低いが耐食性に優れ、常温から1500°Cまで強度低下が少ないという特徴をもっている。代表的な材料特性を表7・4に示した。c. 非酸化物セラミックス(non-oxide ceramics)表7・3に示す非酸化物セラミックスの中には、エンジニアリングセラミックスとして興味深い性質をもつものが多い。それらを単体および化合物別に区分して、炭素、窒化物、炭化物、ホウ化物、ケイ化物などに分けることができる。古くから使われている炭素材料にはダイヤモンドと黒鉛があり、前者は工具材料として、後者は耐熱・耐食材料、電気材料として使われている。炭素繊維は複合材料用として広く利用されている。最近ではフラーレンやカーボンナノチューブも発見され、ニューカーボンとして新しい分野が開かれつつある。炭化タングステン、炭化チタン、窒化チタン、ホウ化チタンなどの遷移元素の炭化物、窒化物、ホウ化物などは非化学量論物質と呼ばれ、硬くて融点が高く、特殊な条件下で耐食性が優れるなどの特徴をもっている。特に、切削工具用には欠かせない材料となっている。窒化ホウ素は炭素とよく似ており、黒鉛によく似た六方晶系窒化ホウ素は耐熱材料、耐食材料、潤滑材料として用いられる。一方、立方晶系窒化ホウ素はダイヤモンドと類似の構造をもち、硬質材料として利用されている。しかし、エンジニアリングセラミックスとして最も重要な材料は、優れた耐熱性と耐食性、高い機械強度などで特徴づけられる窒化ケイ素と炭化ケイ素であろう。以下、窒化ケイ素(含サイアロン)と炭化ケイ素を中心に、おもな非酸化物系材料について述べる。.
