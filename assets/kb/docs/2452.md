# 3・5・1 胸部の解剖学

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・5 胸部傷害（thoracic injury）のバイオメカニクス

胸部は肋骨郭とその内部の胸腔にある軟組織器官で構成されてい
る．胸部と腹部の境界は横隔膜である．横隔膜は薄い筋肉でできた
ドーム状の膜である．
a.肋骨郭 肋骨郭は12個の胸椎，12対の肋骨および胸骨か
らなる．後部では肋骨が胸椎と結合されている．前部では上部7対
の肋骨が胸骨に結合されており，第8～10肋骨は軟骨を介して第7
肋骨につながっている．第11，12肋骨は遊走肋骨と呼ばれ，前面
では他の骨格と結合していない．第7肋骨から第12肋骨は横隔膜
よりも下方に位置しており，肝臓，胃，脾臓，膵臓，腎臓など腹部
臓器の一部を囲んでいる．
b.肺と縦隔洞 胸腔内は三つの領域に分けられる．左右の肺
を包んでいる領域と，それらの間にある縦隔洞である．肺はその形
状に適合した肺胸膜により覆われている．肺胸膜の外側および胸壁
の内面は壁側胸膜で覆われている．肺胸膜と壁側胸膜の間には空気
層があり，胸膜腔と呼ぶ．縦隔洞には心臓，心臓から出入りする大
血管，胸腺，食道，気管などが包含されている．縦隔洞は前端を胸
骨で，後端を胸椎で遮られている．
c .心臓と大血管 心臓は中空筋肉臓器で縦隔洞の胸腔下部に
ある．大ざっぱにいえばその大きさは人の握りこぶし大で，成人男
性では300g，成人女性では250gである．心臓は左右の心房と左
右の心室からなる4個の部屋に分かれている．右心房へは肺以外の
全身から脱酸素血液が上大静脈，下大静脈を通って入ってくる．右
心室は脱酸素血液を，肺動脈を通って肺に送るポンプの役割をして
いる．肺で酸素を供給された血液は左心房へ4個の肺静脈により戻
される．酸素を供給された血液は厚肉の左心室から大動脈を通っ
て，肺以外の全身へ供給される．
