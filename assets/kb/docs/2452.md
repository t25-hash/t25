# 7・1・4 機能性セラミックス

> **章**: 第7章 セラミックス
> **節**: 7・1 ファインセラミックス

力学的機能という表現からもわかるように機械材料も機能性材料の一つであるが,ここでは,一般に機能性セラミックスとして知られている,電気・電子的機能材料,磁気的機能材料,光学的機能材料,生体機能材料について概略を紹介する.詳細は参考図書などを参照されたい.a. 電気・電子的機能材料物質の電気・電子的機能としては,導電性,半導性,絶縁性,誘電性,圧電性,焦電性等が知られている.一般に使われている電子材料は,これらの機能のいずれかを含む電気・電子部品の総称をいう.物質中に電流が流れるのは,電気を帯びた電子やイオンが電界の方向に移動するからである.物質には金属のように電流が流れやすいものからアルミナやダイヤモンドのように流れにくいものがある.金属は自由電子を多く含んでいるので電圧を印加すると電流が容易に流れる.ダイヤモンドのような共有結合性物質では,電子は電子対として結合に使われているので絶縁性を示す.しかし,シリコンやゲルマニウム等では,温度の上昇や不純物などによって電子が移動できるので半導体となる.半導体セラミックスを利用した電子部品としてサーミスタやガスセンサなどが知られている.アルミナは高絶縁性セラミックスであり,集積回路の基板やパッケージに利用される.安定化ジルコニアはカルシアやイットリアを添加して作られることから,酸素イオンの空孔が形成されイオン導電性を示すようになるので酸素センサとして利用されている.絶縁体に電圧を加えると,正電荷の中心や負電荷の中心が離れて分極する.このような性質をもつものを誘電体と呼び,コンデンサとして広範囲に使われている.誘電性のある物質は,電圧を印加すると伸びたり縮んだりする性質をもっている.また,力を加えると電圧を生じるものがある.このような性質を圧電性と呼び,ガスレンジの圧電着火素子や超音波素子などに使用されている.さらに,圧電性を示すものの中には,瞬間的な熱エネルギー変化によって,自発分表7・8電気・電子的機能セラミックスの機能と用途機能材料おもな応用素子・用途伝導性SiC,M 磁気的機能材料天然のマグネタイトが刀や鉄器に吸い付けられたり,南北の方向を指す性質をもっていることは古くから知られている.現在はこれを発展させた多くの磁性体が開発され,磁気テープや磁気ディスクなどの情報分野,発電機・変圧器などの電力分野,電話・ラジオ・テレビなどの通信分野などに広く利用されている.磁気的機能セラミックスの代表的なものにフェライトや焼結永久磁石がある.磁性体には,磁気履歴曲線[磁場の強さ()と磁束密度()の関係を示す曲線]によって評価され,弱い磁場がかかると容易に磁化するが,磁場がなくなると元の状態に戻る軟質磁性(ソフト)材料と,磁化するためには強い磁場が必要であるが,いったん磁化させるとその状態を保ち続ける硬質磁性(ハード)材料がある.ソフトフェライトは軟質磁性材料の代表であり,各種の高周波用磁心に用いられる.化学式は,MFeO(MO・FeO,M:Mn,Ni,Zn)で示される.図7・8にソフトフェライトの用途と特性を示しておく.セラミックスの硬質磁性材料である永久磁石はハードフェライトあるいはフェライト磁石と呼ばれ,化学式はMFeO(MO・6FeO,M:Sr,Ba)で示される.軟質磁性材料と硬質磁性材料の中間的な性質を示すものが,半硬質磁性材料であり,針状のγ-FeO微粒子は磁気記録媒体として広く使われている.c. 生体機能材料生体機能材料としては,1生体環境での親和性に優れ毒性を示さない,2強度・靱性など優れた機械的性質を示す,3埋入材料として周辺組織と適切な相互関係を示す,4埋入後も補修などの技術的な措置が容易であるなどの条件を具備することが必要である.生体材料のおもな用途は人工歯根と人工骨であり,対象となるセラミックスには,多孔質セラミックス,炭素材料,アルミナ,リン酸カルシウム,バイオガラスなどがある.多孔質セラミックスは生体内において自らがもつ気孔内に生体組織を増殖させることによって固定化させる.そのためには,生体親和性が極めて重要である.生体親和性に優れ安定なセラミックスとして,アルミナ,TCP(tri calcium phosphate),アパタイトなどを挙げることができる.
