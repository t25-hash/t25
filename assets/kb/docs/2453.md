# 7・2・1 多孔体の形態と分類

> **章**: 第7章 セラミックス
> **節**: 7・2 多孔質セラミックス(porous ceramics)

多孔体はすきまや孔が多いため,外形寸法から計算した表面積より真の表面積は一般に非常に大きい.また,孔の形状や大きさも材料の種類により,また同一材質での調製法によりさまざまな孔が存在する.図7・9に代表的な孔の種類を示す.孔の種類には大きく分けて外部から物質が孔に侵入できる開気孔(open pore)と,外界と遮断された閉気孔(closed pore)に分類される.さらに,開気孔には基材を貫通し,そこを物質が通過できる貫通孔(through pore)と,単なるくぼみである非貫通孔の2種類に分類される.フィルタや分離膜などに適用する場合には,貫通孔か非貫通孔かの孔の形態が大きく影響してくる.入口部が狭く,内部が広い孔形態,図7・9(c)に示したインクびん型の孔を有する多孔体は吸着に使用される代表的な例である.開気孔は吸着,分離,触媒などの機能を果たす場所になり,閉気孔は断熱,吸音,材料の軽量化に重要な役割を果たす.細孔は,その大きさによりIUPAC(International Union ofPure and Applied Chemistry)で分類が提唱されており,20Å以下をミクロポア,20~500Åの細孔をメソポア,500Å以上をマクロポアと分類する方法が多く採用されている.表7・10に細孔の大きさによる分類を示した.セラミック多孔体の細孔径範囲と応用分野とのおおよその関係を図7・10に示した.孔径の範囲も極めて広く,特に数Åから数十Åのミクロポア領域での細孔構造を有する多孔材料を多種類作ることができるのは大変貴重である.そのため,物理的機能だけでなく,触媒そのものあるいは触媒担体として,反応工学や生物工学分野にも用途が拡大されている.
