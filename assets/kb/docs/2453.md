# 3・5・2 傷害のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・5 胸部傷害（thoracic injury）のバイオメカニクス

自動車衝突時の胸部傷害はおもにステアリングホイール，インス
トルメントパネル，その他の内装構造に衝突したときに発生する．
また，激しい衝突ではシートベルトやエアバッグなどの乗員拘束装
置による傷害も含まれる．胸部傷害は，一般的に肋骨の傷害，肺の
傷害および心臓・血管の傷害に分けられる．
a.肋骨骨折（rib fracture） 胸郭への鈍衝撃（blunt
impact ）により肋骨骨折とフレイルチェスト（flail chest ）が発生
する．肋骨骨折はそれほど重い傷害ではなく，1箇所の骨折は軽傷
（AIS1），2～3箇所の骨折は中等傷（AIS2）である．肋骨骨折は
肋骨に曲げ荷重が作用し，骨が引張り側で破断すると考えられてい
る．フレイルチェストとは，胸壁動揺とも呼ばれ，肋骨多発骨折に
より，吸気時に肋骨郭の一部が持ち上がらない呼吸困難な状態にな
る重い傷害である．片側のフレイルチェストは重傷（AIS4）であ
り，両側にフレイルチェストが発生した場合は重篤（AIS5）とさ
れている．フレイルチェストの判定は，連続する三つ以上の肋骨
に，各2箇所以上の骨折がある場合とされている．
スタルネーカー（Stalnaker ）とモーハン（Mohan) およびメ
ルビン（Melvin）ら は，肋骨骨折を生じさせるファクタは最大
胸たわみ（圧縮変位）であると結論付けた．彼らは過去の死体実験
結果をレビューし，肋骨骨折は胸たわみが7.6cm以上になるとよ
く発生し，5.8cm以下では発生しないといっている．しかしなが
ら，肋骨骨折を生じさせる荷重は，胸部軟組織の粘性特性により，
荷重速度に依存する．したがって，荷重速度が一定の場合には，荷
重が肋骨骨折数と関係すると考えられる．
b.肺挫傷 高速衝撃では，圧縮変位と圧力波が胸壁を通して
肺組織へ伝達され，肺細胞の毛細血管床へ傷害を与える．肺組織の
裂傷もまた肋骨骨折側に生じる．肺挫傷はたわみ速度依存性がある
と考えられている．
c .血胸と気胸 胸膜腔に血液がたまる傷害を血胸と呼び，空
気がたまる場合を気胸と呼ぶ．血胸，気胸は肺の健全な呼吸機能を
妨げる．血胸，気胸はしばしば骨折した肋骨が胸腔内の血管や肺に
裂傷を与えたときに生じる．そのように肋骨骨折のみならず，血
胸，気胸も最大胸たわみに依存すると考えられている．
d.心臓と大血管傷害 高速胸部衝撃の場合，心臓は挫傷およ
び裂傷を受ける．前面衝突ではエネルギー吸収ステアリングハブと
コラムにより，心臓脈管傷害が減ることが報告されている ．心
臓挫傷は胸たわみと胸たわみ速度により生じる．一方，心臓裂傷は
胸骨に大きな胸たわみが作用したときに生じる．
大動脈裂傷は自動車事故でよく見られ，しばしば致命傷となる．
動脈裂傷は一般に血管の軸方向でなく，周方向に発生する．その理
由は，大動脈の引裂き強度は周方向よりも軸方向のほうが強いから
である．一般に大動脈裂傷は血管のねじり，引裂き，高い血管内圧
およびそれらの組合せにより生じると考えられている．
