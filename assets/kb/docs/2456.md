# 7・3・2 強度評価法

> **章**: 第7章 セラミックス
> **節**: 7・3 複合セラミックス

セラミックスの破壊強度は引張試験ではなく曲げ試験で評価されることが多い。その理由として(1) セラミックスの破断ひずみが他の材料に比べて極端に小さいため、正確な引張強度の測定にはわずかな引張軸のずれも許容できず、非常に高い試験精度が要求される。(2) ダンベル型やドッグボーン型等の典型的な引張試験片の加工が容易ではない。(3) 引張試験片のゲージ部ではなくゲージ部端や引張支持部等の応力集中箇所での破壊が生じやすい。などが挙げられる。同じ理由により、複合セラミックスの破壊強度も一般的に曲げ試験で評価される。粒子またはウィスカ分散型複合セラミックスの場合は単相セラミックスと同じ条件の曲げ試験が採用されてもほぼ問題とならない。しかし、連続繊維で強化された複合セラミックスではせん断強度が低いことが多いため、(曲げスパン/試験片高さ)比の大きな曲げ試験でないと中立面近傍でのせん断破壊が引張破壊に先行する事態となりやすく(図7・23)、異種複合セラミックス間での破壊強度比較の際に混乱が生じてしまう。さらに、連続繊維強化複合セラミックスの曲げ試験では荷重点および支点での集中応力による圧壊が生じることも多いため、ナイフエッジの曲率半径や材質の選択に留意する必要がある。表7・11 セラミックスの材料設計において考慮すべき諸特性物性等の種類熱物性機械物性電磁気物性光学物性化学的性質適合性その他比熱容量熱伝導率熱膨張率耐熱性熱分解温度融点ふく射キュリー温度相変態温度デバイ温度弾性強弾性非弾性粘弾性音波物性硬さ強度破壊靱性降伏応力疲労強度摩擦係数耐摩耗性クリープ電気抵抗絶縁性誘電性強誘電性圧電性焦電性半導性イオン導電性超伝導性磁性強磁性反磁性電子放射諸特性2―201セラミックス(ceramics)の材料設計において設計指標(スペック)となる諸特性の一例を表7・11に示す。材料特性は、熱物性、機械物性、電磁気物性、光学物性、化学的性質、生体および環境適合性、その他の物理特性に大別され、これらはさらに多数の特性に分類される。各特性は、相互に影響し合う場合が多く、単独の材料特性のみを向上させることは一般に困難である。またこれらの特性は表7・12に示した各種因子(組成、組織、幾何学、目標物性以外の物性、外場、環境の各因子)により強く影響される。各因子に影響される度合いは特性により異なり、強度、耐熱衝撃性、熱伝導率、透光性などは多くの微構造因子に強く影響される(構造敏感性)。それに対し、比熱容量、熱膨張率、弾性率、電気抵抗、誘電率などは組成や微構造の平均値のみに依存する性質(構造鈍感性)がある。したがって、要求されるスペックを満足させるためには、これらの因子や製造コストを考慮したうえで、材料設計しなければならない。以下では、セラミックスの基礎物性に対する支配因子、セラミックスの特性に及ぼす微構造因子と製造条件との関係についてその概要を述べる。
