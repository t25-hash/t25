# 7・4・2 セラミックスの諸特性に及ぼす微構造因子と製造条件

> **章**: 第7章 セラミックス
> **節**: 7・4 セラミックスの材料設計

との関係いわゆる微構造因子は、表7・12に記載されている組成因子、組織因子、および幾何学因子を総称したものと解釈される。すなわち、化学組成、結晶相、非晶質相粒子径、粒界、気孔、偏析、第二相、第三相、未焼結部、界面反応層、点欠陥(ホール)、転位、積層欠陥、双晶、ドメイン構造、およびそれらに関連した幾何学因子などである。各特性は、これら微構造因子により大きな影響を受ける。セラミックスの強度はその典型例で、例えば粒子径の影響を強く受ける。図7・25は酸化チタンの強度に及ぼす粒子径の影響を調べた結果である。強度は粒子径の小さい間は、粒子径の-1/2乗に対して線形関係にあるが、臨界粒子径である20 m以上になると、その強度は急減する。セラミック製品の製造法には、大きく分けて下記の3種類の方法がある。粉体合成→混合→成形→乾燥/脱脂→焼結→機械加工・接合の各工程が、製品の微構造や欠陥、表面性状に影響を及ぼし、結果としてセラミック製品の諸特性を左右することが知られている。成形は、粉体に目的に合った形状を付与する工程であり、その善し悪しは製品の特性に直接大きな影響を与える。成形工程においては、成形体に密度むらやき裂が生じないように工夫する必要がある。成形工程で水や揮発物質を用いる場合は乾燥工程が必要となる。乾燥時の問題点は、大型品になるほどひび割れが生じやすいことである。また、成形助剤であるパラフィンや高分子類を除去するために脱脂工程が必要であるが、乾燥工程と同様にひび割れしないように注意が必要である。乾燥や脱脂が終わった成形体は所定の温度プログラムに従って焼成される。焼成は酸化雰囲気、アルゴンなどの不活性雰囲気、水素雰囲気(還元雰囲気)、真空雰囲気などの条件下で行われ、緻密化される。通常、粒成長を抑制しながら焼結を促進させる目的で各種焼結助剤が使用される。焼結助剤の種類によって特性が大きく変化することがあるので、使用目的に応じて最適な助剤系を選択する必要がある。焼結品は多くの場合機械加工(研削、切断、研磨)が必要であるが、セラミックスは極めて脆性的であるため、加工傷が導入されやすい。そのため機械加工を能率よく行うことができず、加工コストも非常に高い。以上で述べたように、高品質のセラミックス部材を作製するためには、緻密に練られた材料設計を行うことはいうまでもなく、全製造工程に対して高度に制御された技術を適用する必要がある。
