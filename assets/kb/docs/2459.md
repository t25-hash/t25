# 7・5・1 材料強度・寿命特性の評価法

> **章**: 第7章 セラミックス
> **節**: 7・5 セラミックスの構造強度設計

a. 構造用セラミックスの機械的性質セラミックス材料を金属材料と比べて特徴づけている最も大きなものは,セラミックスが脆性体であることである.強度試験を行うとほとんど塑性変形をせずに破壊する.粉体を焼き固めて作るセラミックスは,空孔等の種々の内部欠陥を含み,脆性体であるがゆえにその影響が強度に敏感に反映され,その結果として強度が大きくばらつくことになる.強度のばらつきを調べた例を図7・26に示す.図7・26は種々の大きさのSiN試験片の引張りおよび曲げ強度の分布をワイブル確率紙にプロットしたもので,図上で直線関係が得られれば強度σの累積確率分布が2母数ワイブル分布(Weibull -σ-γα(7・2)で整理することが行われる.多くの場合,より安全側の評価となり,かつ数学的な取扱いが簡単な2母数ワイブル分布[式(7・1)]を用いてデータが整理される.図7・26に示す例も式(7・1)がよく当てはまることがわかる.式(7・1)中のはワイブル係数(Weibullmodulus)と呼ばれ,ワイブル確率紙上でデータを直線近似したときの直線のこう配を表しており,これが大きいほどデータのばらつきが小さいことを表す.ワイブル係数の値は材料に対して決まる定数と考えられており,これの大きな材料ほど,強度のばらつきが小さく信頼性が大きい材料ということができる.図7・26では試験片のサイズによらず=15の値が得られている.現在のよく調製された構造用モノリシックセラミックスのワイブル係数の値は=15~20程度である.また,図7・26では体積の大きいものほど強度が小さいことが認められるが,これは次のように理解されている.すなわち,セラミックスの強度は内部に含まれる最も有害な欠陥の強度を反映しているものであるから,体積の大きいものほど有害な欠陥に遭遇する確率が大きく,したがって平均的な強度が小さくなる.このように強度が体積依存性[volume effect,寸法依存性(size effect)]を示すこともセラミック材の一つの特徴である.また,セラミックスには,負荷をかけた瞬間に破断しなくても,そのまま負荷を続けると,ある有限な時間の後に破断する現象がある.この現象を,静的な負荷のもとでの材料の疲労という意味で静的疲労(static fatigue)と呼んでいる.これは,材料中の欠陥が応力作用下で次第に大きくなり(スロークラックグロウス,SCG :slow crack growth),ついに破壊条件を満足したときに破壊が生じると解釈されている.また,セラミックスのSCG に水分が大きな影響をもち,水分の存在下で寿命が短くなることが知られている.したがって,セラミックスの静的疲労は金属材料の応力腐食割れに類似の現象ということができる.また,一定応力下である時間後に破断する現象は,金属のクリープ破断にも似ているが,金属のクリープ破断に際してはクリープ変形を伴うのに対し,セラミックスの静的疲労では顕著な変形を伴わないのが普通である.負荷応力が繰返されるとき,セラミックスの種類によっては,静的疲労に比べて寿命が低下することが知られている.これを静的疲労と区別する意味で繰返し疲労(cyclic fatigue)と呼んでいる.金属の場合,疲労といえば繰返し疲労を指すが,セラミックスでは2―204材料学・工業材料このような使い分けをしている.セラミックスでは,静的疲労が時間依存破壊の主メカニズムで,繰返し疲労は負荷の繰返しによるSCG の加速現象であると考えられている.ガラスなどのシャープなき裂とは異なり,多結晶セラミック材のき裂先端には破壊のプロセスゾーンが形成され,このプロセスゾーンはき裂先端での応力拡大係数を低下させる効果をもつ(応力遮へい効果).応力の繰返しは応力遮へいに寄与するプロセスゾーン内の微視構造を劣化させ,き裂進展を加速させるものと考えられている.セラミックスには,疲労に対しこのような繰返し効果を示すものと示さないものがある.一般に,粒界破壊をする材料は疲労に対する繰返し効果を示す.セラミックスは高温強度の大きい材料とされ,高温でその性能を発揮することが期待されているが,材料によって定まるある一定温度を超えると粒界相の軟化が始まり,粘性流動をするようになる.また,それに伴い,結晶粒の回転や粒界においてボイドの発生,成長,合体が生じ,材料の非弾性変形や破壊をもたらす.これをセラミックスのクリープ(creep)という.したがって,セラミックスのクリープ開始温度は粒界相の軟化温度であるということができる.一定荷重下での寿命評価実験を高温で行う場合には,静的疲労とクリープを区別するのが難しいが,破面を観察することによりある程度可能である.静的疲労の場合は,一つの潜在欠陥を起点とする比較的平坦な破面を呈するのに対し,クリープの場合は,粒界相物質に覆われた凹凸の激しいき裂進展面が観察される.これは,クリープの場合は,き裂がその起点から作用応力によって定まる一定の方向に進展したと考えるより,粒界相物質の粘性流動によりところどころの結晶粒界にボイドが形成され,それらが合体し,ついに破断したと考えることで,破面の凹凸や,破面が粒界相物質に覆われていることなどが説明できる.また,前述のように,通常の温度領域では静的疲労寿命のほうが繰返し疲労寿命より大きいのが一般的であるが,クリープ温度領域ではこの関係が逆転する例がしばしば報告されている.これはクリープ温度領域における欠陥の自己治ゆ現象と関係があると考えられている.一般的に,セラミックスは金属材料に比べると化学的活性が低く,耐環境性が大きい.しかし,耐環境性が大きいゆえに,より過酷な環境で使用されることが多いので,このような場合には,セラミックスといえども腐食かつ劣化する.おもなものでは,(1)高温酸化,(2)溶融塩腐食,(3)金属腐食が挙げられ,ガスタービン環境中で使用するような場合には,高温・高圧燃焼ガス流に乗って混入飛来する固体粒子による(4)エロージョン摩耗,(5)異物衝突による損傷なども生じ,これらに対する評価が不可欠である.セラミックスの環境依存性については,ジルコニアが熱水中で分解してしまうという例も報告されており,まだまだ未知の事象も多いと予想される.今後も,使用が想定される環境のもとでのセラミックスの応答を可能な限り明らかにしていく努力が必要である.b.
