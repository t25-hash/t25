# 3・8・3 傷害基準と傷害耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・8 四肢傷害のバイオメカニクス

四肢傷害では骨折および靱帯損傷が問題となることから，骨に負
荷される荷重やモーメント，関節部の変位や回転角などが指標とさ
れている．以下に，法規や安全アセスメント試験で参照されている
ものや，提案中ながらも有力な指標を含めて解説する（表3･12）．
a.上肢傷害基準と耐性 上肢傷害に関してはエアバッグとの
干渉という観点からおもに曲げモーメントに対する耐性が論じられ
てきた．成人男性（mid- size male）を想定した上腕骨の骨折耐性
としては217～230N･ mという値が，前腕骨については橈骨と尺
骨を合わせて91N･mという値が提唱されている ．ただし，
これらの耐性値は現在のところ車両の衝突安全性評価試験における
b.下肢傷害基準と耐性 大腿部（femur ）の耐性を求めるた
め，死体の膝頭をインパクタで強打して骨折時の荷重を測定する実
験が行われてきた ．大腿部の骨折荷重は時間依存型であり，
衝撃力の継続時間が20ms以上の場合は8.9kNだが，短くなるに
つれ23.14kNに漸近すると報告されている ．3･8･2･ b項で述べ
たように，大腿部の傷害は負荷入力形態により受傷部位が膝関節・
大腿骨・股関節に分かれる．過去に実施されたさまざまな実験デー
タを統計的に分析した研究から ，成人男性における大腿部の傷
害耐性としては10kNという値が広く参照されている．膝関節の
靱帯傷害では大腿骨顆部と脛骨上端の相対変位が問題となる．死体
下肢に対する衝撃実験結果から，膝変位（knee sliding）14.4mm
で局所的な靱帯損傷が始まり，22.6mmで全損傷に至ると報告さ
れている ．これらの結果を踏まえ，成人男性における膝関節の
傷害耐性としては15mmという値が広く参照されている．下腿に
おいては荷重負担の大きい脛骨の耐性が研究されており，曲げモー
