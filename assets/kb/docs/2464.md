# 8・2・1 ガラス繊維

> **章**: 第8章 複合材料
> **節**: 8・2 強化繊維

a. ガラス繊維の物性ガラス繊維(glass fiber)は1930年代に開発され、その後、生産技術および利用技術の発達とともに、複合材料の強化繊維あるいは充てん、断熱材などの資材として大量に使用されるようになった。ガラス繊維は、ケイ酸(SiO)を主成分として、ナトリウム、カリウムなどの一価の酸化物、バリウムのような二価のアルカリ土類金属、三価のアルミナなどの酸化物に、それらの副産物として発生する炭酸を適量混合・溶融し、これらが結晶化しない程度に急冷・凝固された無定形物であるとされている。複合材料の強化繊維としてのガラス繊維は、その形態から連続繊維と短繊維の2種類がある。いずれのガラス繊維でも、溶融したガラス素地が満たされた溶融炉から直接紡糸するダイレクトメルト法にて成形される。紡糸炉の底面に設置されたノズルから押出した後、集束材(サイジング剤)を噴霧してストランドとし、高速で引き伸ばしながらドラムに巻取る。単繊維の直径は3~23 m、ストランドの集束本数は約50~2000本程度である。ガラス繊維の強度、弾性率、破断ひずみなどの力学的特性は、原材料の構成、製造時の条件などによって決定される。これまでにいくつかのガラス繊維が用途別に開発されている。全般的なガラス繊維の特徴を列挙すると、1生産性に優れ安価である、2取扱いが容易で、加工性に優れる、3材料特性のばらつきが小さい、4電気絶縁性、断熱性に優れる、5母材との接着技術が確立している、などである。逆に短所としては、1弾性率が他の繊維に比べて低い、2比重が大きい点などが挙げられる。現在、よく用いられている代表的なFRP用のガラス繊維を以下に概説する。(1) Eガラス繊維最もよく使われている繊維で、電気絶縁2―210性に優れるが、耐酸性に劣る。別名、無アルカリ繊維と称されている。プラスチック複合材料として、プリント基板、絶縁材料用スペーサ、はん用FRP構造物などに用いられる。Eはelectricalを意味する。(2) Cガラス繊維耐酸性に優れた特性を発揮するように開発された繊維で、おもに耐酸容器用FRP構造物、下水道配管の補強材などに使用される。Cはchemicalあるいはcorrosionを意味する。(3) S(T)ガラス繊維ガラス繊維の強度・剛性を高めるために、Eガラスに比べてシリカ(SiO)とアルミナ(AlO)の成分を高め、高温度で紡糸した繊維で、強度は36%、弾性率は16~20%以上大きい。特に、高温度環境下でも優れた特性を有している。主として、航空・宇宙、スポーツ用品分野などの用途が多い。S はsilicaを意味する。(4) AR ガラス繊維耐アルカリ性を目的に開発された繊維である。EガラスやCガラス繊維では耐久性が保証されない環境でよく使用される。以上のガラス繊維の組成を表8・2に示す。また、よく使用されているEガラスとS(T)ガラスの物性値を表8・3に示した。b. ガラス繊維の形態と種類ガラス繊維を工業製品として使用する場合、1プラスチックの剛性・強度の向上を目的とした強化繊維としての利用、2ガラス繊維の有する断熱性・電気絶縁性・耐食性を目的としたガラス繊維単独での利用、3両者の利用目的を同時に取込んだ強化繊維としての利用形態がある。1は熱可塑性樹脂を母材にしたガラス短繊維強化プラスチック(GFRTP)、2はグラスウール、3は熱硬化性あるいは熱可塑性樹脂を母材にしたガラス連続繊維強化プラスチック(GFRP)などが、それぞれ代表的な利用形態である。ガラス繊維製品は、用途別に種々のFRP用の強化材料として加工されている。それぞれ、ヤーン、クロス、テープ、ロービング、チョップドストランド、ロービングクロス、チョップドストランドマットとして製品化されている。表8・4に、ガラス繊維の代表的な製品形態別の特徴と規格を示した。また、クロス材は平織り、綾織り等の形態があり、成形法の多様化とドレーパビリティ向上のために種々の織り方の形態がある。マット材はロービング繊維を短繊維に切断してマット状にしたもので、等方性材としての特徴がある。積層材料の中間層に使われることも多い。
