# 8・2・2 カーボン繊維(carbon fiber)

> **章**: 第8章 複合材料
> **節**: 8・2 強化繊維

カーボン繊維は,ポリアクリルニトリル(PAN),メゾフェーズピッチ,ピッチ,レーヨンなどの有機繊維を原料とし,無反応ガス雰囲気中で加熱し,炭化または黒鉛化することによって得られる.図8・2に(a)PAN 系,(b)メゾフェーズピッチ系,(c)ピッチ系カーボン繊維の製造工程を示す.製造工程では,まず,原料繊維を製造し,これを200~300°Cの空気中で加熱する.これは,PAN繊維の場合は前処理あるいは耐炎化処理,メゾフェーズピッチ繊維,ピッチ繊維の場合は不融化処理と呼ばれる,空気による軽い酸化処理である.この空気酸化処理は,部分脱水素あるいは酸素付加,分子間架橋などによって繊維に耐熱性を与えて,焼成の際の溶融または炭素原子の逸出を抑制するためのものである.その後,焼成処理に入るが,焼成工程には炭化および黒鉛化処理あるいは高温熱処理の2段階の工程がある.炭化工程では,焼成温度の上昇に伴って原料繊維を構成している有機高分子の炭素原子以外の原子がガス化して逸出し,炭素原子が残留して単体炭素に凝集する.この過程で生成した炭素の繊維は六角網平面状の多環芳香族分子状断片(フラグメント)の集合体であるが,黒鉛化工程では,これらの芳香族フラグメントがより成長する.焼成を終えたカーボン繊維には複合材料として使用する際にマトリックスとの接着性を良くするた表8・4ガラス長繊維の加工製品と規格製品の名称製品の概要用途FRP treatment)を施す.複合材料の強化材としてカーボン繊維を利用するうえでの立場には,繊維としての優れた強度,弾性率などの力学的性質を利用する立場と,炭素としての耐熱性,導電性,耐薬品性などの性質を利用する立場がある.航空機部品やスポーツ用品に用いられているカーボン繊維強化プラスチック複合材料(CFRP)は高強度,高弾性率といった力学的性質を利用した構造材料で前者の立場での利用例であり,ロケット噴射孔やシリコンウェーハ製造用るつぼに用いられるカーボン繊維強化カーボン複合材料(C/C)は耐熱性を利用した後者の立場としての利用例である.カーボン繊維を強化材に用いた複合材料の中で生産量が多いのはプラスチックをマトリックスとしたCFRPである.CFRPとして構造材料に使用されるカーボン繊維は,強度特性に優れることが求められ,強度と弾性率のバランスが良いPAN 系カーボン繊維が多用されている.PAN 系カーボン繊維は,引張りおよび圧縮の強度,繊維と樹脂の接着強度がピッチ系カーボン繊維に比較して優れているため,荷重を負担する構造材料に適しており,航空機,船舶,自動車などへの使用,建築・土木分野への応用が進んでいる.強度より弾性率が優先される用途では,ピッチ系カーボン繊維が適しており,宇宙機器,長尺ロール,搬送ロボットのアーム,スポーツ用具の剛性向上に使用されている.ピッチ系カーボン繊維の場合,その特徴は高い引張弾性率であるが,圧縮強度が低く,表面を酸化処理しても樹脂との接着強度が低いという欠点もある.カーボン繊維の強度,弾性率,破断伸びといった力学的性質は,原料の有機繊維の種類,構造および焼成工程の温度,延伸などの条件によって決まる.表8・5にカーボン繊維の諸性質を示す.PAN
