# 8・2・4 セラミック繊維(ceramic fiber)

> **章**: 第8章 複合材料
> **節**: 8・2 強化繊維

繊維で強化したセラミック基複合材料では,強化繊維にセラミック繊維が用いられることが多い.これらの複合材料の成形は高温でなされることがほとんどであるため,強化に用いられる繊維には高温における長時間安定性が重要な因子となる.これらの点から,現状で用いられている繊維としては,アルミナ(AlO)系の繊維,炭化ケイ素(SiC)系の繊維,炭素系の繊維,窒化ケイ素(SiN)系の繊維などがある.これらの繊維の直径は,太いものでも数十mである.そして,これら繊維は,物理的な長さの観点から長繊維系とウィスカ系に分類される(表8・7).セラミック系繊維の大きな特徴は,強度および弾性係数が高温域まで低下しないこと,および,熱膨張係数が金属系の繊維に比べて小さいことにある.一方,破壊の観点から見たとき,延性が乏しいことや靱性が低いという欠点も有する.代表的なセラミック繊維単体の物理的,機械的性質を表8・7に示した.表8・7 セラミック繊維の物理的性質繊維名組成密度強度弾性率熱膨張係数×10/Kg/cmGPaGPa炭素繊維PAN 系(高強度型)1.7510.6炭素繊維PAN 系(高弾性型)1.82.2炭素繊維ピッチ系2.12.1AlO系繊維99αAlO3.95 1.4~2.18.3AlO系繊維86(α+β)AlO,4SiO3.4≒1.0240~3208.7AlO系繊維85AlO,15SiO3.21.8~2.6 2.1~2.58.8AlO系繊維70AlO,20SiO,2BO3.05 2.1~2.3 190~224―AlO系繊維AlO,15ZrO4.22.1~2.5―SiC系繊維59Si,15Si,31C,10O2.55 2.5~3.3 182~2103.1SiC系繊維48Si,28C,18O,4Ti2.33.1SiC系繊維47Si,30C,15N,8O2.3 1.75~2.1175~210―SiC系繊維59Si,10C,28N,3O2.35 2.1~2.45140~175―SiC系繊維69Si,30C,1O2.61.1~1.4 175~210―SiC系繊維70Si,30C(CVD)2.553.92.6SiN系繊維59Si,41N,0.4C,2O2.21.5―SiN系繊維56Si,28N,14C,2O2.53.17―SiN系繊維60Si,39N,0.5C,1O2.52.5―BN 繊維BN1.49.5SiCウィスカβSiC3.93.8SiCウィスカβSiC3.183.8SiCウィスカαSiC+βSiC3.186.9―SiCウィスカαSiC+βSiC3.181.8―SiNウィスカ(α+β)SiN3.184.82.5AlOウィスカAlOウィスカ3.8827.57.7グラファイトウィスカグラファイト構造―19.6―これらの繊維を複合材として用いる場合,繊維単体の機械的特性等がそのまま複合体中の繊維強度には必ずしもなっていないことに留意すべきである.また,繊維1本の強度と束となったときの強度(束強度と呼ばれる)が異なることや,それらの相違が繊維がもつ統計的ばらつきの程度に依存していることにも配慮すべきである.後者の統計的ばらつきはワイブル係数を用いて表されることが多い.
