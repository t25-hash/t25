# 8・3・1 プラスチックの種類による分類

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

プラスチック基複合材料(polymer matrix composites)は8・2節で説明された強化材を用い、マトリックスにプラスチックを用いた複合材料である。プラスチックを大別すると、熱硬化性プラスチックと熱可塑性プラスチックになるが、両者とも複合材料用マトリックスとして用いることができる。構造用複合材料ではエポキシ樹脂やポリエステル樹脂といった熱硬化性プラスチックがよく用いられている。熱可塑性プラスチックはおもに短繊維と組合わされて用いられる。a. 熱硬化性プラスチック(thermosetting plastics)分子量が数百から数千のポリマで、重合反応する官能基を多数もつ。一般に、成形時の材料粘度が小さいので強化材へのぬれがよく、長繊維もしくは連続繊維を強化材に使用しやすく、付加重合タイプの材料(不飽和ポリエステル樹脂、エポキシ樹脂など)では大きな成形圧力を必要としないので大型の成形品を比較的容易に作ることができるという特徴がある。また、硬化したマトリックスの分子構造が三次元網目であるので硬く、強度、耐熱性に優れている反面、もろいという欠点もある。以下、複合材料に多用される熱硬化性プラスチックについて概説する。. 不飽和ポリエステル樹脂不飽和基を有するポリエステルプリポリマにスチレンモノマを硬化剤として使用する。ラジカル重合開始剤を加え、強化繊維に含浸させた後に、架橋反応を進行させ成形する。常圧での成形が可能なので、現場施工での大型製品の成形にも適している。エポキシ樹脂に比較すると、繊維との接着性、耐熱性などに劣るが、硬化時間が短い、取扱い性が良い、安価であるといった特徴がある。. エポキシ樹脂1分子中に活性水素と反応するグリシジル基を2個以上有し、開環付加反応で硬化する樹脂である。分子中に水酸基があるので接着性が良く、また、ベンゼン環(芳香環)があるので耐熱性を有しており、したがって、複合材料用マトリックスとしてバランスのとれた性質を示す。硬化剤として脂肪族アミンを用いると常温硬化タイプになり、また、芳香族アミンを用いると加熱硬化して耐熱性を向上させることができる。耐薬品性や耐熱性に優れている。. ビニルエステル樹脂基本構造として、エポキシ当量の大きなエポキシ樹脂にアクリル酸(メタクリル酸)を反応させて二重結合を導入した形になっている。分子中に水酸基を含むので接着性が良く、また、メタクリル酸単位を含むので、耐候性に優れる。. フェノール樹脂フェノールとホルムアルデヒドとからなる樹脂で、硬化は縮合反応によって起きる。構造としてはフェノール環がメチレンで結合されたものである。そのために剛直で、耐熱性に優れ、難燃性で、耐水性、耐酸性に優れている。また、フェノール性水酸基を有しているので接着性も良い。一方、欠点はもろいことである。. ビスマレイミド系樹脂ビスマレイミドの二重結合に芳香族ジアミンのアミノ基が付加することによって架橋体となるポリアミノビスマレイミドと、シアン酸エステルを加えてトリアジン環を形成させるビスマレイミドトリアジン樹脂とがある。いずれも樹脂原料の粘度が低いので、強化材への含浸が容易である。耐熱性に優れている。b. 熱可塑性プラスチック(thermoplastics)熱硬化性プラスチックとは異なり、反応する官能基をもたない。成形は溶融温度以上に加熱して液体状にした材料を金型に入れ、冷却固化することで行われる。分子量が大きいので分子どうしの絡み合いが多数あり、一般的に靱性が大きい。また、上述のような成形過程のため、リサイクルが容易である。この材料には、結晶性プラスチックと非晶性(無定形)プラスチックがある。炭素と水素から構成されるものにポリエチレン(PE)、ポリプロピレン(PP)、ポリスチレン(PS)がある。PEおよびPPは、結晶性で、密度が1より小さく、ポリオレフィンと呼ばれ、その性質は親油性または疎水性である。耐薬品性には優れるが、接着性に劣っている。酸素や窒素、ハロゲンを構成元素とするものは親水性で、接着性に優れるが吸湿性がある。代表的な材料として、ポリエ表8・8
