# 8・3・2 力学的特性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

構造材料として用いられる複合材料は,連続繊維を一方向に配向した薄い層(単層板あるいはラミナ)がいろいろな角度に積層された多方向積層板として用いられることが多い.実際にはプリプレグ(prepregs)と呼ばれる0.1~0.2mmの厚さの中間基材を重ね合わせ,オートクレーブやホットプレスで成形する.まず,複合材料の力学的特性を知るうえで重要である,一方向単層板(ラミナ,lamina)の力学的特性を考える.表8・8に各種プラスチック基複合材料の力学的特性を示す.繊維方向弾性率は高強度炭素繊維を強化材とする場合には,130~170GPaと鉄鋼より若干小さいが,高弾性率炭素繊維を用いたときには,それを超える値になる.一方,繊維直角方向弾性率は6~9GPaと低く,大きな異方性を有する.この異方性の度合いは,ガラス繊維複合材料よりカーボン繊維複合材料のほうが大きい.また,繊維方向の強度は従来の工業材料と比較して非常に大きい値になっている.異方性の度合いは弾性率よりさらに大きい.繊維方向の破断ひずみは,繊維の破断ひずみでほぼ決まるので金属材料に比較して低い.繊維方向の熱膨張係数が小さいので,寸法安定性の要求される用途に使用される.構造物を設計する際には,強度,弾性率そのものではなく,強度を比重量(単位体積にかかる重力)で除した比強度,弾性率(剛性)を比重量で除した比弾性率または比剛性が重要になる.表8・9に各種材料の値を示す.従来の材料では,比強度を4×10m,比剛性で4×10mを超えることが困難であった.しかし,一方向強化複合材料ではこれらを大きく上回っている.ただし,その優位性は,擬似等方性積層板(quasi-isotropic laminates)や織物材では低下する.表8・10にカーボン繊維/エポキシおよびケブラー繊維/エポキシ一方向積層板を繊維方向とある角度θ方向に引張ったときの弾性率および強度の変化を示す.弾性率,強度ともに繊維方向から角度が増えると急激に低下することがわかる.一方向強化CFRPの引張りの応力-ひずみ曲線は,繊維方向(0°)では,カーボン繊維の表8・9各種材料の比強度および比剛性材料比強度(10m)比弾性率(10m)軟鋼0.42.7構造用鋼0.62.7クロムモリブデン鋼1.32.72017ジュラルミン1.02.67075超々ジュラルミン1.82.7カーボン/エポキシ一方向強化材繊維方向18.510.3カーボン/エポキシ擬似等方積層板5.34.0表8・10 一方向強化複合材料の特性の引張方向依存性材料カーボン/エポキシケブラー/エポキシ角度θ弾性率GPa強度MPa弾性率GPa強度MPa136.561.1――49.488.929.4――17.923.27.082.7――4.648.3――4.141.4――4.334.59.84.127.62―214材料学・工業材料非線形性を反映して,若干下に凸の非線形性を示す.一方,繊維方向に対し斜め方向に引張ったときは,樹脂の非線形特性が反映され,上に凸の関係となる.実際には,プラスチック基複合材料は前述までの一方向強化材をいろいろな角度に重ね合わせて成形した積層板として用いられる.表8・11にはカーボン/エポキシ積層板の種々の積層構成のものについて0°方向に引張荷重を与えた場合の弾性率とポアソン比を示す.積層構成の表記は各層の繊維方向の負荷方向に対する角度で示している.下付添字sはその面で対称な角度で積層されていることを示す.例えば,(0/±60)읎は,0°,60°,-60°,-60°,60°,0°の順に積層,成形されている積層板であることを示す.表8・11より,同じ材料でも積層構成を変化させることにより,さまざまな特性のものが得られることがわかる.これが,複合材料が設計可能な材料といわれる理由である.
