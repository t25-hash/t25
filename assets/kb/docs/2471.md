# 8・3・4 クリープ特性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

プラスチック基複合材料の強化材としてよく用いられるガラス繊維やカーボン繊維は、FRPの通常の使用温度下ではクリープを示さない。一方、マトリックスである高分子材料は、たとえ常温下で表8・11 カーボン/エポキシ積層板の特性積層構成弾性率GPaポアソン比(0/±60)읎52.30.30(0/±45/90)읎52.30.30(0/90)읎71.80.04(±30)읎50.51.24(±45)읎19.60.74(0)136.50.24(90)10.80.02の弾性限度内の小さな荷重によってもクリープを示し、温度の上昇とともに顕著となる。このことから、FRPは通常の使用温度下でもクリープを示すことが考えられるが、実際、CFRP一方向材の繊維直角方向でのクリープ試験において、時間の経過および温度の上昇に伴いクリープコンプライアンスが著しく増大すること、クリープコンプライアンスを対数時間軸に沿って平行移動することによりマスタ曲線が得られ、クリープコンプライアンスの時間依存性と温度依存性の間に時間-温度換算則が成立することが報告されている。
