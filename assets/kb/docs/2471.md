# 8・3・4 クリープ特性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

プラスチック基複合材料の強化材としてよく用いられるガラス繊維やカーボン繊維は、FRPの通常の使用温度下ではクリープを示さない。このことから、FRPは通常の使用温度下でもクリープを示すことが考えられるが、実際、CFRP一方向材の繊維直角方向でのクリープ試験において、時間の経過および温度の上昇に伴いクリープコンプライアンスが著しく増大すること、クリープコンプライアンスを対数時間軸に沿って平行移動することによりマスタ曲線が得られ、クリープコンプライアンスの時間依存性と温度依存性の間に時間-温度換算則が成立することが報告されている。
