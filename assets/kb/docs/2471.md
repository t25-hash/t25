# 3・2・3 じん肺法

> **章**: 第3章 工場・施設内部に係わる法制度 β9－77
> **節**: 3・2 労働者の福祉の向上を図るための法律

a．じん肺法制定の目的（第1条） じん肺法は，じん肺に関し
適正な予防および健康管理などの必要な措置を講じ，労働者の健康
保持その他福祉の増進に寄与することを目的として，昭和35年3月
31日に制定された．じん肺発生防止のためには，作業環境対策に加
えて，定期的健康診断によりその病状を早期に的確に把握し，適切
な管理区分を設定し，管理区分に応じて，粉じんばく露の減少また
はばく露機会をなくすことを基本とする的確な鯉康管理上の措置を
行い，じん肺の発生とその進展を防止することが不可欠である．こ
のため，昭和52年7月1日，じん肺の定義，粉じん作業，じん肺管
理区分，じん肺健康診断，健康管理のための措置などを整備充実し，
じん肺と密接な関連ある合併症の療養を含む健康管理について改正
が行われている．
b．じん肺法の構成 じん肺法は，第1章 総則（第1条一第
6条），第2章 健康管理，第1節じん肺健康診断の実施（第7条一
第11条），第2節じん肺管理区分の決定等（第12条一第20条），第3
節健康管理のための措置（第20条の2一第23条），第3章 じん肺審
