# 8・3・5 疲労特性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

金属材料の疲労破壊メカニズムは,繰返し荷重とともに微小クラックが発生,成長し最終的な破壊に至る.疲労初期に,複数のクラックが生じたとしても,そのうちの一つのクラックの成長が問題となる.複合材料の場合,疲労過程においてさまざまな形態の損傷が材料内部に発生するため疲労破壊メカニズムは金属材料とは大きく異なる.複合材料中に発生する損傷としては,ラミナのマトリックス樹脂中のクラック,繊維/マトリックス界面のはく離,繊維破断や積層板レベルでのマトリックスラックや層間はく離がある.このようなさまざまな損傷がお互いに影響を及ぼし合いながら発生,進展していき,最終的な複合材料の破壊に至るので,疲労寿命を予測する統一的な手法は存在しない.複合材料の疲労寿命に影響を及ぼす因子としては,荷重条件や環境など材料に起因しない要因を除くと,1構成材料(繊維と樹脂とその組合せ),2強化材の形態(一方向強化材,織物,短繊維強化材),3強化材の含有率,4各層の荷重方向との角度,積層構成など数多くある.一般に複合材料の繊維方向の-曲線は金属材料に比較して平坦であり,良好な疲労強度を有することが知られている.
