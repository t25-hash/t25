# 8・3・6 衝撃特性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

衝撃的な負荷を受ける構造において複合材料が使用される場合には,構造設計のための基礎資料として衝撃下における複合材料の力学的特性の評価データが必要である.複合材料の衝撃下での力学的特性を調べるためにさまざまな試験がなされている.測定される力学的特性も試験法により異なり,単なる衝撃吸収エネルギーを求めるものから高ひずみ速度下での応力-ひずみ関係を求めるものまでさまざまである.プラスチックやFRPのアイゾット衝撃試験およびシャルピー衝撃試験法は,JIS ならびにASTM で規格化されている.表8・12に各種一方向繊維強化FRPのアイゾット衝撃値を0°方向の引張りおよび圧縮強度とともに示す.カーボン繊維強化複合材料は優れた引張強度を有するものの衝撃値は低い.アラミド繊維強化複合材料は高い引張強度と衝撃値を示すが,圧縮強度が低い.表8・13に計装化シャルピー衝撃試験より得られた各種一方向強化エポキシ樹脂の縦と横方向の衝撃特性を示す.윯율융は最大荷重,욡は最大荷重までに吸収されたエネルギー,욧は全吸収エネルギー,(욣/욡,욣は最大荷重後に吸収されたエネルギー)は延性指標である.それぞれのパラメータにおいて材料により優位さが異なっており,耐衝撃性を評価するにはさまざまな指標を用いて評価する必要性があることが示唆される.複合材料の衝撃下での応力-ひずみ関係,強度,破断ひずみなどの衝撃特性を求めるために,スプリットホプキンソン棒(SHPB :split Hopkinson pressure bar)式衝撃試験機,one-bar method,油圧サーボ式高速試験機などが用いられている.特に,SHPB 式表8・12 一方向積層板のアイゾット衝撃値強化繊維引張り/圧縮強度MPa アイゾット衝撃値kJ/mアラミド(ケブラー49)1568/225アラミド(テクノーラ)1617/225カーボン(PAN 系)1421/745ガラス(Eタイプ)951/608表8・13一方向強化複合材料の縦と横方向の衝撃特性(計装化シャルピー試験)複合材料のタイプ負荷方向に対する繊維方向윯율융kN욡kJ/m욧kJ/m高強度カーボン/エポキシ垂直平行12.90.80.671.56.5S ガラス/エポキシ垂直平行13.40.70.5510.910.118.8ケブラー49/エポキシ垂直平行7.90.41.1315.810.5Nomex ナイロン/エポキシ垂直平行5.60.40.785.56.1衝撃試験機は応用範囲が広く,圧縮,引張り,せん断および曲げなど各種変形状態に対応するよう改良され使用されている.
