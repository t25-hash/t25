# 8・3・7 複合材料の新技術

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

a. テキスタイルコンポジット(textile composites)最も基本的な複合材料の形態は、一方向材であり、通常多方向に積層され使用される。その積層工程は自動化が困難なため、高価で時間のかかる作業である。また、基本的に面外に強化繊維が配向されていないため、面外方向には強度が低い。このような問題を解決するため繊維を多方向に配向した複合材料を作製することが必要である。このような観点からさまざまなテキスタイルコンポジットが考えられてきた。テキスタイルコンポジットの強化形態には、織物、編物および組物がある。それらは、昔からよく用いられてきた伝統技術であるが、日本ではあまり明確な区別はされず、混同して用いられることがしばしばある。英語では、それぞれ、interlacing, interlooping,intertwining として区別されている。織物は、一般には繊維束を縦方向と横方向に直角に交差するように織り込んだもの、編物は縦方向あるいは横方向にループを連結させたもの、組物は3本以上の繊維束が長手方向に対して斜めに配向し、切断されることなく連続しているもの、と定義されている。b. ラジエーション成形光(紫外線など)や放射線(電子線など)を照射(ラジエート)することによって、硬化する樹脂(光硬化樹脂、電子線硬化樹脂)は塗装や印刷などの分野では、広く用いられている。このような光硬化/電子線硬化樹脂と繊維からなる積層体に電子線などを照射して硬化させ複合材料を得る技術の研究が進められている。複合材の硬化に用いる場合、紫外線などの光は透過力が弱く、材料内部まで硬化させることが難しい。電子線の場合、十分なエネルギーがあれば材料内部まで透過させることができ、硬化速度も速いので、複合材料の硬化に適している。光硬化/電子線硬化樹脂は、その硬化のメカニズムから、ラジカル重合系とカチオン重合系に大別される。電子線硬化複合材料の初期の研究では主としてラジカル重合系が用いられていたが、近年航空宇宙用複合材料のマトリックス樹脂への適用を目的とした研究ではカチオン重合系が主流となりつつある。電子線硬化複合材料は、オートクレーブなどを用いる熱硬化複合材料に比較して、極めて短時間で硬化可能であること、基本的に常温硬化なので熱応力が小さい、成形コストが低い、などの点から注目されている。c. 耐熱および極低温複合材料プラスチック系複合材料は、当初、旅客機などの音速以下で飛ぶ航空機部品のアルミ合金の代替として使用することが開発のきっかけだったので、使用温度は90~-55°C程度の常温から大きくはずれない環境で耐えれば十分であった。しかし、複合材料の技術が進み、実用化が進んでくると、より過酷な環境への使用が構想されるようになった。まず、戦闘機・宇宙往還機への使用を目指して、高温に耐える耐熱樹脂とそれをマトリックスにする複合材料の開発が盛んになった。そこで開発されたのがポリイミド系樹脂PMR-15である。続2―215いて、コンコルドの後継となる超音速輸送機(SST、マッハ2.4級)の開発に関連して、熱可塑性ポリイミド系樹脂PIXA が開発された。アメリカでは同様の動機から、PMR-15の先駆となったLarcシリーズ、耐熱熱可塑性樹脂の先鞭を付けたデュポン社のAvimid K-IIIなどが開発され、なかでもNASA で開発されたPETI-5が注目されている。d. 知的複合材料(smart composites)材料・構造を知能化(スマート化)しようとする試みが最近活発になってきた。一つのアプローチとしてセンサやアクチュエータを材料に組込むことが考えられている。プラスチック基複合材料の、素材から構造体への一体成形の特徴を考えると、知的材料・構造への有力な候補であることは明らかである。近年、光ファイバを用いたヘルスモニタリング(health monitoring)の研究、形状記憶合金を用いた損傷抑制材料、自己修復FRPの開発、スマート成形モニタリング技術の研究などがなされている。
