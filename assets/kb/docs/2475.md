# 8・4・1 金属基複合材料の構成と組織

> **章**: 第8章 複合材料
> **節**: 8・4 金属基複合材料

金属基複合材料(metal matrix composite)は主としてセラミック質の繊維や粒子を金属の母材(マトリックス,matrix)に埋込んで複合させ,優れた機械的性質を発揮させるように人工的に作った材料である.マトリックス金属に比べて,高比強度,高比弾性であることを特徴とする.1959年にアメリカ空軍研究所(AFML)でタングステンの心線にボロンを化学蒸着させて作った,いわゆるボロン繊維が開発されて以来,1970年代にかけて炭素繊維やシリコンカーバイド繊維などのセラミック繊維(ceramicfiber)が次々と開発され,工業的規模で生産されるようになった.金属基複合材料はセラミック繊維の開発とともに発達した.今日,表8・14に示すような繊維や粒子が金属マトリックスの強化用として使用される.表中の金属繊維も一部用いられることがある.強化材(reinforcement)の形態としては,円形断面形状をした細長い連続繊維,長さ数mmの短繊維,連続繊維をチョップした短繊維,ウィスカ(whisker)に代表される針状結晶および粒子がある.これらがマトリックス金属に埋込まれて,光学顕微鏡的には図8・3のような組織を呈する.図8・3(a)は連続繊維で一方向に強化したもので,最も大きな強度が得られるが,性質に大きな異方性がある.図8・3(b)は短繊維またはウィスカが三次元ランダム方向に配向された強化材である.繊維強化効果は低いが,ほぼ等方均質的な性質を示す.これを圧縮成形するに従い,繊維は二次元ランダム方向に配向し,ある面内で等方性を示す.また強い引抜きあるいは押出し加工をすると繊維が一方向に配向し,一方向短繊維強化材となる.図8・3(c)はいわゆる粒子強化材であり,機械的,物理的性質は均質等方的であり,強度はマトリックスの性質を強く反映する.複合材中の強化材の存在量は,通常体積含有率で表す.繊維体積率と配向性は所望の性質を発揮させるためにあらかじめ材料設計されるが,体積率は製造上,実用上の制約によって,通常5~50%である.金属基複合材料のマトリックス金属としては,軽金属のアルミニウム,マグネシウム,チタニウムとその合金が多用される.300°Cまでの用途には,鋳造用アルミニウム合金AC1C,AC4C,AC8A,AC8B,展伸用アルミニウム合金6061,2024,5056,2024,7075などがある.600°Cまでの用途にはチタン合金Ti-6%V-4%Al,Ti-15%Mo-5%Zr-3%Al,Ti-5%Al-2.5%Snなどがある.このほか,さらに高温特性を改善するため,金属間化合物(intermetallic compounds)TiAl,NiAl,MoSiなどや超合金Fe-Cr-Alなどをマトリックスとする超高温用金属基複合材料が研究途上である.
