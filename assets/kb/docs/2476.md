# 3・2・2 ステンレス鋼

> **章**: 第3章 機械材料の標準形状と素材例
> **節**: 3・2 鉄鋼材料

オーステナイト系，マルテンサイト系，析出硬化系に大別される．
品まで広い用途をもつ．
加工性，成形性，溶接性にも優れ，各種の化学装置に自動車部品，車両，原子炉関係，発電関係など広く用いられる．
典マルテンサイト系ステンレスは，耐食性は十分ではないが，焼入の12～13 %Crを含むSUS410（0.1C）はバルブシート，ポンプト，プラスチックの型，刃物などに用いられている．
（代表例にSUS630）とオーステナイト系（SUS631）があり，主級ばねなど強さと耐食性，耐熱性を必要とするものに用いられる．
れ，焼もどしにより優れた機械的性質を得ることができる．
