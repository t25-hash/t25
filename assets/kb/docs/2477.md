# 8・4・3 機械的性質

> **章**: 第8章 複合材料
> **節**: 8・4 金属基複合材料

金属基複合材料は開発が始まってからまだ日が浅いために、材料としての規格や標準試験法は確立していない。金属基複合材料の特徴は、(1)軽量で高強度である、(2)耐摩耗性がある、(3)耐特殊環境性(原子状酸素、放射線、高真空など)がある、(4)電気や熱の良伝導体である、などがある。これらの特徴を生かして、機械や装置の軽量化ならびに輸送機械の省エネルギー化に新しい解決法を提供する。a. 比強度(specific strength)金属基複合材料の比強度と比弾性(specific modulus)を従来材に比べて示すと、おおむね図8・4に示した範囲の値となる。通常の金属材料では比強度で4×10m、比弾性で3×10mを超える材料はないが、金属基複合材料ではそれらの値をはるかに超え、軽量、高強度材であることを示す。繊維の体積含有量を増加させることによって、比弾性-比強度のグ図8・4金属基複合材料およびマトリックス金属の比強度、比弾性の比較図8・5金属基複合材料とマトリックス金属の比強度の温度依存性ラフは右上がりになる。図中において三次元ランダム配向の短繊維強化材や粒子強化材にあっては、比強度・比弾性はさほど大きくないことを示している。図8・5は各種金属基複合材料について研究報告された強度の中から高強度のものを選んで、その比強度を温度とともにプロットしたものである。焼成法によって製造されたシリコンカーバイド繊維(SiC읋윪읎と記す、商品名ニカロン)および炭素繊維を用いてアルミニウム合金を強化した複合材は、約450°Cまで強度の低下がなく、従来材にない優れた比強度をもつ耐熱材料である。これは1981年から8年間にわたって実施された次世代産業基盤技術研究開発成果の一つである。しかし、試験片ベースの材料としては良い特性を生み出したが、実製品ベースでの材料強度の実証はされていない。シリコンカーバイドウィスカ(SiC윴と記す)やシリコンナイトライドウィスカ(SiN윴と記す)を用いてアルミニウム合金を強化した複合材の比強度は、250°Cの温度までマトリックスのアルミニウム合金に比べてかなり高いが、300°Cになると強度の絶対値そのものが著しく低下し、もはやこの温度では強度部材としては実用に耐えない。耐熱性を上げるためにはマトリックス金属をチタニウム合金、チタンアルミの金属間化合物(TiAl)および超合金(Fe-Cr-Alなど)へと替え、マトリックスに適合する繊維を採用して複合化すれば使用温度の上昇が可能になる。この場合、600°Cにおいても1000°Cにおいても従来材より金属基複合材料の比強度が格段に優れている。しかし、マトリックスそのものの比重が増加するため、大きな比強度は次第に望めなくなる。2―217表8・16 composites)一方向連続繊維強化材の特徴は繊維方向の強度と弾性率が大きいことである。表8・16にボロン繊維、化学蒸着によって作られたシリコンカーバイド繊維(SiCと記す)および炭素繊維で強化したアルミニウム合金ならびにチタニウム合金の機械的性質を示す。ホットプレス法で作られたものは概して複合則で予測される値に近い強度(1~2GPa)と弾性率を発揮する。破断伸びは強化繊維の破断伸びにほぼ等しい値(約2%)をとる。繊維と直行する方向の強度はマトリックス金属の強度と同程度である。一方向連続繊維強化材の引張試験をすると最大荷重点からほぼ非累積的破壊(non-cumulative fracture)が観察される。最大荷重点に達する前には弱い繊維の先行破壊とマトリックスの塑性変形を伴うことが金属基複合材料の特徴である。この先行破壊によって繊維の負担していた荷重がマトリックスに伝ぱし、伝わった応力の集中は金属マトリックスの塑性変形を伴い、応力が緩和される機構が働き、複合材の強度の信頼性(strength reliability)は繊維強度の信頼性より改善される。ボロン繊維/6061アルミニウム合金製の管材は、軽量高強度を生かして、いち早くスペースシャトルのトラス構造材に使用された。図8・6のようにスペースシャトルの胴体部に243本取付けられ、145kg
