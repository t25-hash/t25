# 8・5・1 サンドイッチ板(sandwich plate)

> **章**: 第8章 複合材料
> **節**: 8・5 サンドイッチ板・多層シート板

サンドイッチ構造は複合構造の一種で、薄くて耐荷能力のある高強度の上下表面材を比較的厚い軽量心材で結合して構造効率を高めた形態であり、軽量化が最大級に要求される航空機、宇宙飛翔体に広く応用されている。最近では複合材料をはじめとする種々の新材料の開発により、サンドイッチ板の表面板および心材の材質、寸法についても多様な選択が可能となり、建築、船舶、車両などの一般構造の分野にも広く用いられているので、ここでは表面板にFRP材を用いた一般的な異方性サンドイッチ板について述べる。異方性サンドイッチ板の閉じた形の一般的な解は与えられていない。しかし、種々の負荷条件のもとで各種の境界条件を有する一般的な直交異方性のサンドイッチ板の近似解を求めるのには、以下に示すエネルギー法が有効である。a.
