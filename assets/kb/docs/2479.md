# 8・5・1 サンドイッチ板(sandwich plate)

> **章**: 第8章 複合材料
> **節**: 8・5 サンドイッチ板・多層シート板

サンドイッチ構造は複合構造の一種で、薄くて耐荷能力のある高強度の上下表面材を比較的厚い軽量心材で結合して構造効率を高めた形態であり、軽量化が最大級に要求される航空機、宇宙飛翔体に広く応用されている。最近では複合材料をはじめとする種々の新材料の開発により、サンドイッチ板の表面板および心材の材質、寸法についても多様な選択が可能となり、建築、船舶、車両などの一般構造の分野にも広く用いられているので、ここでは表面板にFRP材を用いた一般的な異方性サンドイッチ板について述べる。異方性サンドイッチ板の閉じた形の一般的な解は与えられていない。しかし、種々の負荷条件のもとで各種の境界条件を有する一般的な直交異方性のサンドイッチ板の近似解を求めるのには、以下に示すエネルギー法が有効である。a. エネルギー法図8・15に示すように、サンドイッチ板の中央面に対して対称にそれぞれFRP材が積層された上下表面板と、心材も異方性材からなる一般的な異方性サンドイッチ板を考える。表面板は心材に比べて十分薄いと考え、表面板自身の曲げ剛性は省略し、心材は厚く、その横方向のせん断剛性のみを考える。その行列の固有値から座屈荷重が得られる。等方性材料や直交異方性材料を表面板に用いた場合、図8・18は=0の軸に対して対称となる。A~Fの順にたわみが小さくなり、逆に固有振動数は大きくなる。A~D の解析結果は曲げと加振の両実験結果と良い一致をみている。厚さ比が小さい範囲で配向角が大きいほど、カップリング効果が大きいことを示している。
