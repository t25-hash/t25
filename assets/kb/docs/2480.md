# 3・3・2 銅と銅合金

> **章**: 第3章 機械材料の標準形状と素材例
> **節**: 3・3 非鉄金属

伸銅品と鋳物に大別され，それぞれ主要添加元素による合金系
統，合金種類より細分化されている．
a.Cu・高Cu系合金 無酸素銅（C1020)，タフピッチ銅
（C1100），りん脱酸銅（C1201など）などがある．電気，熱の伝
導性，展延性，絞り加工性に優れ，電気用，化学工業用，建築用，
ガスケットなどに使われる．
b.Cu-Zn系合金 丹銅（C2100～C2400)，7/3黄銅
（C2600）は絞り加工，建築，装具品などに，6/4黄銅（C2801）
は強度が高く，板，棒などの加工材として用いられる．
c.Cu- Zn-Pb系合金 快削黄銅（C3560など）は快削性に優
れ，打抜き性もよく，機械部品などに用いられる．鍛造用黄銅
（C3771など）は熱間鍛造性がよく，バルブ，機械部品などに用い
られる．
d.Cu-Zn-Sn系合金 ネーバル黄銅（C4621）は耐海水性に
優れ，船舶用部品に用いられる．
e.Cu-Sn系合金 代表的な素材は青銅（bronze ）で，鋳造
性，耐食性がよく，青銅鋳物（CAC401～407）は滑り軸受，バル
ブ，電動機器部品，一般機械部品に用いられる．りん青銅
（C5101～5212）は展延性，耐疲労性，耐食性がよく，機械部品，
ばね，軸受などに用いられる．特に冷間加工硬化材は高弾性ばね材
料として用いられる．
f.Cu-Al系，Cu-Si系合金 アルミニウム青銅（C6161な
ど）は強度が高く，耐摩耗性，耐食性がよい．船舶部品，化学工業
用品，歯車，軸受などに用いられる．
g.Cu-Ni系合金 白銅（C7060，C7150）はキュプロ-ニッ
ケルとも呼ばれ，耐食性，特に耐海水性がよく，比較的高温の使用
に適する．熱交換器用管板などに用いられる．洋箔
（C7351～7541）はばね材料として用いられるほか，家庭用器具や
食器類，装飾品にも用いられる．
h.Cu-Be系合金 ベリリウム青銅と呼ばれ，高強度，高ばね
特性を有し，一般ばね材，ダイヤフラム，ベローや電気接点などに
用いられる．
