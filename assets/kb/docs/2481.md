# 8・5・3 多層シート板(multi-layered sheet)

> **章**: 第8章 2―
> **節**: 8・5 サンドイッチ板・多層シート板

プラスチックの発泡倍率を厚さ方向に変化させることで表面材と心材を同時に、かつ連続的に変えて成形したものが多層シートであり、その結果、表面材は硬い層を構成している。ABS 3層シートがすでに市販されている。この多層シートは耐衝撃性に優れてい図8・23 ABS 3層シートと他の構造材料と剛性比較て、軽量構造材料として自動車用ボデーの一部や各種コンテナなどに使用されている。ABS 3層シートの曲げ剛性は上下表面材の厚さとヤング率、ポアソン比を욞,욞,ν욞、心材のそれらを욂,욂,ν욂とおき、サンドイッチ板の場合と違って心材の剛性を考慮した次式=욂욂121-ν욂+2욞욞1-ν욞욞12+욞+욂(8・32)で計算できる。図8・23は各種板材の曲げ剛性と単位面積当たりの質量の関係を示したが、同じABS 単板に比べ3層シートは軽量でかつ優れた曲げ剛性を有することが示されている。また、ABS3層シートはアルミ合金板、FRP板、鋼板などに比べ優れた軽量性を有し、板厚面に沿った発泡倍率の変化によってさまざまな構造のものが成形されている。構造物を組立てるためには複数の部材を互いに接合する必要がある。その接合法には溶接や、ボルト、リベットなどを用いた機械的接合法がある。複合材料の場合、金属部品と接合しなければならない例が多い。例えば、風力発電用の回転翼やヘリコプタ用のロータブレードにも軽くて強いFRP (fiber reinforced plastics)が用いられているが、駆動軸とつなぐ部分では金属フランジと接合しなければならない。大きな動力を伝える自動車用プロペラシャフトにも軽くて高速回転が可能なCFRP/GFRP製薄肉パイプが用いられようとしているが、等速ボールジョイントとの接合が開発のポイントとなっている。ボルトやリベットを用いた機械接合法では部材の一部に孔をあけるため、複合材料の強化繊維を切断することになる。また、母材に樹脂を用いた繊維強化複合材料(PMC: polymer matrix compos-ites)では厚さ方向の応力緩和が顕著なため、接合面の摩擦力に頼る高力ボルトによる接合は期待できない。一方、鋼やアルミ合金のように溶接を用いることもできない。GFRFPやCFRPに代表されるPMCは、パイプや平板(曲面板を含む)であれ一般に二つの寸法(幅・長さ)に比べて厚さが薄い。そのため、構造組立に当たって機械的接合法ではPMCの軽くて強い特性を十分生かすことができない。PMCの強度特性を大きく損なうことなくPMC部材の組立を行うには、二つの部材間に働く力を面で伝えることのできる接着接合(adhesive bonding)が欠かせない。
