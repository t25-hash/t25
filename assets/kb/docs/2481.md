# 3・3・3 アルミニウムとアルミニウム合金

> **章**: 第3章 機械材料の標準形状と素材例
> **節**: 3・3 非鉄金属

g.7 000系：Al- Zn- Mg，Al- 展伸材，鋳物およびダイカストに大別され，それぞれ主要添加元系統により，以下の1000系～7000系およびそのほかに分類される．
金のなかでは最も強度が低いが成形性，耐食性，陽極酸化処理（アルマイト）性に優れる．
家庭用品，電線，はく，アルマイト処理建素による合金系統，合金種類より細分化されている．
展伸材は合金ジュラルミンと呼ばれる．
a.1 000系：純アルミニウム（非熱処理型) アルミニウム合分類される．
