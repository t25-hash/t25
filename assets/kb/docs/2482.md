# 8・6・1 接着形態と接着強度

> **章**: 第8章 2―
> **節**: 8・6 複合材料の接着・接合

一般に、接着剤(adhesives)はせん断に比べて、はく離や引張りに弱い。そのため、二つの部材を接着によって組立てようとするときには、接着面積を大きくするとともに、図8・24に示すようにできるだけ接着剤層に平行に荷重が作用するように設計する。ここで、図8・24(a)は突合せ継手(butt joint)、図8・24(b)(c)はそ2―225(a) 突合せ(悪い)(b) 突合せ重ね合わせ(良い)(c) 突合せ二重重ね合わせ(優れている)(d) T 型突合せ(優れている)図8・24 種々の接着継手形態(a) 二重重ね合わせ継手(b) 単純重ね合わせ継手図8・25 二重重ね合わせ継手と単純重ね合わせ継手れぞれ突合せ重ね合わせ継手(lapped butt joint)および突合せ二重重ね合わせ継手(double lapped butt joint)と呼ばれる。やむを得ず突合せ継手形態となるときは、図8・24(d)のように、引張応力の作用する接着面積を増やすなどの工夫が必要である。二つの部材を接着接合する場合、設計に当たって接着剤の接着せん断強度に対するなにがしかの目安が必要となる。また、接着剤の選択に当たっても、適当な指標が必要となる。JIS 規格では接着剤のせん断接着強さを求めるため、金属-金属接着について、薄板の単純重ね合わせ継手(single lap joint、図8・25)を用いた引張せん断接着強さ試験法(JIS K 6850:1999)および圧縮せん断強さ試験法(JIS K 6852:1994)が規定されている。そのほかにも、試験片として二重重ね合わせ継手(double lap joint)を用いた引張せん断試験も慣用的に広く行われている。
