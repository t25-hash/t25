# 8・6・2 重ね合わせ継手の接着剤層中の応力

> **章**: 第8章 複合材料
> **節**: 8・6 複合材料の接着・接合

重ね合わせ接着継手[図8・24(b)(c)で突合せ部分の接着剤が取り除かれた継手形態である]に引張荷重が作用した場合,接着面にはせん断応力が作用する.しかし,その応力は一定でない.継手両端でせん断応力が集中するばかりでなく,はく離応力(peelingstress)も発生する.この応力集中により,接着面積を増しても,接着継手(adhesively bonded joint)の強度は線形的には増加しない.比較的低い荷重で接着端面ではく離が生じ,疲労破壊するおそれもある.8・6・2項では,引張せん断荷重を受ける重ね合わせ継手の接着剤層(adhesive layer)中での応力分析について考える.ここでは,被着体を金属のような等方性材料として扱っている.2―226材料学・工業材料a.
