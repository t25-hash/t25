# 3・2・1 直接測定と間接測定

> **章**: 第3章 測定概論
> **節**: 3・2 測定方式の分類

電位差計では，標準電池による基準電圧をポテンショメータで分割し，可変な基準電圧を発生させ，その値と測定すべき電圧との差を高感度のガルバノメータで読み取り，差が零になるように基準電圧を調整している．
この種の調整操作により測定系は一つの閉じた系を構成するが，測定者が手動で調整を行う場合もあれば，調整が自動化されている場合もある．
測定の平衡が実現されているかどうかの確認は，測定量と基準量との差の極性を検出する変換器によって行っており，一方，測定系の精度を決めるものは，フィードバック側に入れた要素の特性であり，差を検出する変換器の感度が多少変動してもほとんど測定誤差に影響を与えない．
また，零位法において，測定に要するエネルギーは外部から供給されており，測定系からとるエネルギーはわずかである．
零位法と偏位法とを組合わせ，測定量からそれにほぼ等しい既知の量を引き去り，残りの量を偏位法的に測定する方法が補償法（compensation method）である．
例えば，てんびんで質量を測る場合がよい例であるが，この場合，測定される物体と分銅の質量が正確に一致することはまれであり，不一致分に対するさおの傾きを指針の振れから読み取ってその値を加えた測定結果を出すことになる．
方法であり，例えば質量が既知の分銅とてんびんを利用して試料の質量を測定する場合や，標準尺を用いて長さを測る場合などに代表される．
一方，間接測定とは，測定すべき量と一定の関係にある他の量，または測定すべき量を組立てているいくつかの量を直接測定して，それらの測定値と測定すべき量とを結びつける一定の関係に基づいて，測定量の大きさを導き出す方法であって，測定量と基準量との比較が間接的であるところに特徴がある．
球体の直径の測定に基づく体積の決定，自動車が一定距離を移動するのに要する時間の測定からその速度を求める場合などが代表的な間接測定に分類される．
以上のように，直接測定であるか間接測定であるかは，測定する量によってではなく，用いる測定方法によって決まるのであり，同じ一つの量を測定する場合でも，直接測定による場合と間接測定による場合とが考えられる．
一方，量によっては，固体の密度や熱膨張係数のように，直接測定が不可能で間接測定のみによるものも存在する．
