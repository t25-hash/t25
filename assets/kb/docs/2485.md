# 8・6・3 複合材料接着継手の破壊

> **章**: 第8章 複合材料
> **節**: 8・6 複合材料の接着・接合

(単純)重ね合わせ継手の代表的な破壊様式は以下の5種類に分けられる(図8・30).(1) 接着破壊a. 界面破壊(接着剤層と被着体の接着界面での破壊)b. 凝集破壊(接着剤層内破壊)c. 混合破壊(aとbの破壊が混じった破壊)(2) 被着体破壊a. 被着体の引張破壊b. 被着体の局部破壊界面接着強度より接着剤の強度が低い場合,接着剤の凝集破壊[(1)-a]が起こる.一方,界面接着強度が接着剤の強度より低い場合,接着界面が露出する界面破壊[(1)-b]が起こる.被着体図8・28引張せん断荷重を受ける単純重ね合わせ継手の接着剤層のせん断応力および垂直(はく離)応力分布図8・29二重重ね合わせ継手の曲げ変形2―227表面の処理は金属被着体の場合,サンドブラストあるいはサンドペーパなどで研磨した後,溶剤で脱脂するのが一般的である.高い信頼性の接着強度を得るためには,適当な薬液に浸すなどの表面処理が行われる.一方,FRPなどの高分子系複合材料では,サンドペーパ(あるいはサンドブラスト)により表面の樹脂リッチ層を取り除いて表面を荒らすとともに,溶剤で脱脂するのが主たる表面処理法である.この場合,強化繊維を露出するほどに研磨することも多い(しかし,露出した繊維の表面からかつて塗布された表面処理剤が取り除かれるため,樹脂との接着性は劣り,必ずしも接着強度を高めるとはいえない).この処理が不十分であったり,汚れていた場合,界面破壊はいっそう起こりやすい.したがって,凝集破壊が発生すれば,被着体の接着面の表面処理は適切であったとみなされる.しかし,接着剤の強度が低い場合にはたとえ凝集破壊が起こっても接着継手の強度が高いことは意味しない.逆に,界面破壊が起こったとしても接着継手の強度が低いわけではない.被着体の種類と厚さ,接着剤と表面処理によっては混合破壊[(1)-c]が起こる.(2)の被着体破壊は被着体の厚さが厚い,あるいは強いと起こらない.これは紙の接着を考えれば理解できる.薄い紙では,壁などに強度の低い接着剤で接着しても引きちぎれることがある.また,厚いボール紙などでは,接着剤とともに紙が壁に残り,取り外すのに苦労する.鋼やアルミ合金ではよほど薄くないと(1)-aの破壊は起こらない.鋼やアルミ合金の接着を対象として,(1)-bの破壊を起こすような接着剤は現在のところ見当たらない.一方,PMC重ね合わせ継手では,荷重の多くは強化繊維により支えられており,厚さ方向におおむね均等に分布しており,接着面へは各層の層間を通して伝えられることになる.そのためPMC被着体のラップ付近では,接着面に近い層間で層に沿うせん断応力とともに大きなはく離応力が生じる.特に,単純重ね合わせ継手では二つの被着体間の軸心が一致しないため,層間はく離応力の集中は顕著である.PMCの層間はく離およびせん断強度はあまり高くないため(2)-bの破壊は起こりうる.被着体の1層め,あるいは2層めから破壊が生じることも少なくない.
