# 8・6・4 さまざまな複合材料接着継手

> **章**: 第8章 複合材料
> **節**: 8・6 複合材料の接着・接合

金属を被着体とする接着継手では、界面破壊であれ、凝集破壊であれ基本的に接着部分で破壊が生じる。一方、上述のようにPMC接着継手では、被着体破壊を防止することも重要である。これらを背景に、PMC接着継手の強度(継手効率)を高めるさまざまな工夫が行われている。a. テーパ付き重ね合わせ継手(tapered lap joints)図8・28に示されるように、引張せん断荷重を重ね合わせ継手ではラップ端でせん断応力、はく離応力とも集中する。継手を一体材料と考えたとき、この応力集中は接着端でいわば部材厚さが急変するからである。そこで、厚さの急激な変化をなくせば接着剤層の応力集中は減少すると考えられる。また、PMC被着体の層間応力の集中も減じる。そこで、図8・31のように被着体の一端にテーパを付与すればよい。また、テーパ状に加工したPMC面の保護とテーパ端でのはく離を抑えるため、新たにマットあるいは平織布状の強化材を1層(1) 被着体の引張破壊(1) 界面破壊(2) 凝集破壊(2) 被着体の局部破壊(3) 混合破壊(a) 接着剤層破壊(b) 被着体破壊図8・30 複合材料接着継手の破壊形態2―228材料学・工業材料図8・31 接着面にマットインしたテーパ付き接着(a) 被着体を曲げて接着する(b) スカーフ継手(c) ステップドラップ継手(d) 二つの被着体の中心面を同一に保つための接着法図8・32 種々の継手形態積層する(オーバレイ:over-lay)こともある。さらに、二つの被着体が同種のPMCであれば、接着表面の凹凸を吸収し、二つの被着体の接合を密にするため、接着面にマットを挿入して接着する場合もある。接着面に挿入されるマットには粘度の低い接着剤を保持する役目もある。b. オーバレイ接着図8・24のように、PMC平板どうしの接合では、両者の中心が中立面を一致させる必要が多い。そのままで図8・24(a)のように突合せ継手となり、高い継手強度は望めない。そこで、一方の被着体の接着部分を変形させた継手(一方の被着体をあらかじめこのような形に成形してある)が考えられる[図8・32(a)].ただし、本質的には単純重ね合わせ継手形態なので、継手効率は高くない。図8・24(b)の例では、あらかじめ成形されたPMC添え板を接着するのではなく、多くの場合、その場で積層成形するオーバレイ接着が用いられる。例えば、パワーショベルやブルドーザなどの建設機械、大型コンバインの乗員室の天井のように、被着体が曲面板の場合も同様に接着できる。一方の外面、あるいは両面から作業できない場合、図8・24(b)のように片面のみオーバレイ接着される。しかし、より大きな継手強度が必要な場合、図8・24(c)のように両面からオーバレイ接着される。その継手強度は図8・24(b)の形式に比べて2倍以上になる。c. スカーフ継手(scarf joint)・ステップドラップ継手(step-ped lap joint)図8・24(b)(c)の形態では元のPMC平板に対して接着部が盛り上がってしまう。しかし、図8・24(a)の突合せ継手では強度が小さい。そこで、図8・32(b)のスカーフ継手が考えられた。スカーフ角θを小さくすれば理論的には継手強度は増すが、被着体の精度の高い加工が困難になるとともに、接着面が広がることから精度良く接着する(二つの被着体の中心を合わせて接着する)のが困難となる。そこで、スカーフ継手と同様な特性を有する図8・32(c)のステップドラップ継手も使われる。この継手形態は木材の接合でも見られる。その他、図8・32(d)など、単に強度だけではなく、概観の滑らかさなど要求される機能に合わせ、オーバレイ接着とスカーフ、テーパ継手形態の利点を取入れた継手がある。複合材料の成形方法は母材により大きく異なる。金属系およびセラミック系の複合材料は母材となる素材の成形方法をそのまま応用することが多いが確立されたものは少ない。ここでは、実用化が進む高分子系材料の成形方法について紹介する。高分子材料の代表はプラスチック材料であり、熱硬化性プラスチックと熱可塑性プラスチックがある。これら素材は、充てんした材料の特性を生かすため、特有な成形方法が開発されている。
