# 8・7・1 熱硬化性プラスチック系FRP の成形方法

> **章**: 第8章 複合材料
> **節**: 8・7 複合材料の成形法

熱硬化性プラスチック(thermo setting plastics)は,硬化剤を混入した液状の樹脂にさらに充てん材や強化材を混合し,型に入れ形状を決めた後,加熱・硬化できることから広く実用化されている.この方法の多くが長繊維を使用できることから高強度,高弾性率が得られ,機械・電気機器の構成材料として,工業製品,スポーツ用品,建築用品に使用されている.代表的な成形方法を以下に示す.a. フィラメントワインディング法(FW : filament winding)樹脂を含浸した強化繊維を回転するマンドルに巻付けた後,加熱硬化する方法である.マンドルは樹脂の硬化・冷却後に除去する.この方法は,細い管や太い円筒状物体の成形に適している.この成形法による製品の直径は,数cmのものから5~6mに至る.繊維の巻き角度は,マンドルの回転速度と軸方向への繊維の移動速度を調整することにより自由に変えられる.巻き角度によって製品の強度を任意に調整することが可能である.繊維の巻き角度が管の軸方向に近いものほど軸方向引張強度が強く,また曲げ変形に強い素材となる.逆に,繊維の巻き角度が管の軸方向に対して直角に近くなるに従い軸方向の引張強度が低下するものの,内圧に強い構造体となる.この構造体は,高圧容器に向いている.径の小さい管状のものは,ゴルフのシャフトやテニスのラケットに,大口径パイプではロケット,魚雷の胴体に使用されている.現在プロパンガスなどの高圧容器へのFW 製FRPの応用が試みられている.FW 法と同じロール状に巻く成形様式の,テープレイイング(tape laying)法が開発されている.この手法は,フィラメントの代わりに一方向性プリプレグ強化シート(テープ状)をマンドルに巻付け,その後加熱硬化させて製品を作製する.この成形法はFW成形法より装置が簡便で済む.b. ハンドレイアップ法(hand-lay-up)代表的接触圧成形法の一種で,手積み法ともいわれる.この方法は,強化材のシートやマットを型の上に置きこれに樹脂を含浸させ,手積みにより積層し,これを加熱して硬化する方法である.船体等大型の構造物の作製に使用されている.この単層をラミナ(lamina)と呼ぶ.一般には,単層では異方性が出すぎるため,一般には多層を積層して硬化したものを用いる.これを積層材(laminate)と呼ぶ.この方法では,層間に気泡が残留しやすいため,物性を安定させることが難しく,訓練を必要とする.c. 圧縮成形法(compression molding)コールドプレス成形法(cold-press)とホットプレス成形(hot-press)があるがホットプレス法が多く使用されている.これはプリフォームマット材を型に入れ,高温のもとで加圧成形する方法であり大型の浴槽などの大量成形に使用されている.代表的な材料は,SMC(シートモールディングコンパウンド)で,あらかじめ繊維と樹脂に硬化剤を混ぜ,強化材,炭酸カルシウム等充てん材を混入したものをフィルムに挟み,シート状にして巻取っておく.このSMCのロールは,低温下では数週間保存が可能であり,熱硬化樹脂であるにかかわらず生産の時間的調整が容易となる利点を有している.成形はロールより適量を切取り,これをホットプレス機に設置した金型に入れ,加圧・加熱して硬化する方法である.この方法は,大量生産化が容易な方法である.繊維長は成形時に切断されることはないが,強化材の場所的むら,充てん物の混合むら,繊維配向のむら,ストランドの崩壊状態のむらなどが原因で,引張強度や疲労寿命に比較的幅広いばらつきが出やすいことが欠点である.d. オートクレーブ成形法(autoclave molding)成形用の型にプリプレグを充てんした後減圧による脱泡行程を行いその後,加圧による緻密化ができる装置を用いる成形方法である.大型でかつ高精度の寸法安定性を有するものの成形に用いられている.製品には,直径数mで長さ10mを超す送水用圧力管や大型容器,航空機の尾翼,部品等の高強度で品質管理が必要な構造物の作製に使用されている.大型の装置が必要となることが弱点となっている.e. R-RIM 成形法(reinforced reaction injection molding)熱硬化性プラスチック系樹脂にミルド繊維や充てん物を混合したものを,射出成形機を用いて金型中に注入し成形する方法.金型中で化学反応が進行し液状材料から固体樹脂へと変わる.強化繊維を混入したウレタン系樹脂を使用した自動車部品が数多く作製されている.バルクモールディングコンパウンド(BMC)材料は,強化材と充てん材を不飽和ポリエステル樹脂に混合した材料に硬化剤を入れ射出成形した材料である.精密機械部品の作製に使用されている.f. RTM 成形法(resin transfer molding)金型中に繊維を配置した中に,2液混合型液状樹脂を射出成形した後,加熱・硬化して成形する方法である.射出成形の利点をもっている.射出成形法では,繊維を樹脂と同時に射出するために高圧を必要とするため,繊維は切断されてしまう.これに対して,この方法では長繊維による強化が可能となり,さらに繊維の方向性を調整することが可能であり,金型も簡便なものを使用することが可能である.g. 引抜き成形法(pultrusion molding)強化繊維に2液混合型液状樹脂を含浸した後,必要とする断面形状をもつ金型(ダイ)を通過させ,加熱硬化させながら引出す方法である.一方向性の長尺物FRPを作製可能であり,この成形方法は,高強度・高弾性率の棒や板の構造部材を作製するのに向いている.
