# 8・7・2 熱可塑性プラスチック系FRP

> **章**: 第8章 複合材料
> **節**: 8・7 複合材料の成形法

熱可塑性プラスチック系FRPはFRTP(fiber reinforcedthermoplastics)と呼ばれており,この成形には熱可塑性プラスチックの成形が使用されている.代表的な方法は,射出成形方法と,ホットプレス法である.a. 射出成形法(injection molding)溶融した樹脂を金型中に射出した後,冷却して固化する方法である.材料には,強化繊維を含む熱可塑性樹脂ペレットを使用する.射出成形は,各種ペレットを調合して希望する量の繊維を含有するペレットを用意し,これを射出機内で溶解し,スクリューにより金型に射出して成形する.この方法により作られているFRPは,全FRPの生産量の90%以上に達しており,現状では大量生産が最も可能な方法である.はん用の機械部品,特に吸振性,耐熱性と強度を向上する必要のあるプラスチック部品,例えば電子機器内部に使用されている小型の変速ギヤはこの方法で作製されている.航空機や自動車の小型構造用部品,スポーツ用品,日常用電気機器のほとんどがこの方法で成形された製品が使用されている.この方法は,大量生産に向いているが,強化材の充てん量に比較して,強度・弾性率は上がらない.これは,ペレットの段階では,繊維のアスペクト比(繊維長/繊維直径)は100~200程度あったものが,射出成形後には切断され10~50程度に低下し繊維の充てん効果が著しく下がるためである.さらに,射出成形時に樹脂の流れにより繊維の配向方向の調整が困難なため,荷重の伝達効果すなわち充てん効果が期待できない直交配向の繊維が増加するためである.b. ホットプレス成形法(hot-press)熱硬化性プラスチックの場合と同じ方法であるが,強化繊維の一方向に配列し熱可塑性樹脂で固めたプリプレグシートまたは二次元配列の布に熱可塑性樹脂を含浸させたラミナを使用する.このプレグシートを積層して,ホットプレス機に入れ,加熱して溶融した後,加圧して接合する.こ2―229の成形法の製品は,連続繊維を使用できることと熱可塑性樹脂の機械的・化学的特徴をもつことから,広く使用され始めている.比強度は,熱硬化性系FRPに匹敵する.母材として使用される熱可塑性プラスチックは,ナイロン樹脂,ポリカーボネート樹脂などが使用されている.文献(1) 帝人(株),テクノーラ技術資料,(1987).(2) 強化プラスチック協会編,FRP 設計便覧,(1988),3-8.(3) 日東紡技術資料,(2001).(4) 島村昭治編,カーボンファイバ,(1984),13-31,オーム社.(5) 日本複合材料学会編,複合材料活用辞典,(2001),328-331,産業調査会.(6) 強化プラスチック協会編, プラスチック系先端複合材料, (1989),19-24.(7) 香川豊・八田博志,セラミックス基複合材料,第2章,(1990),アグネ承風社.(8) Coleman,B.D.,J.
