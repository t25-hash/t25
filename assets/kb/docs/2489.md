また，次の用語の意味も付記しておく．
(１) 系統誤差（systematic error ）
何らかの明確な要因によって，測定値全体にあるかたよりをもた
らす誤差のことをいう．
(２) 偶然誤差（accidental error ）
測定系に影響を与える要因には多種多様なものがあるが，一般に
はそのすべての因果関係が明確に把握されているわけではない．正
体不明の，原因がいくつか積み重なって測定値に不規則なばらつき
が現れることとなる．このような誤差を偶然誤差と呼んでいる．
(３) 過失誤差（mistake error ）
測定者の不注意や知識不足に起因すると考えられる誤差をいう．
次に，誤差に関連する用語について，それぞれの定義を確認して
おく必要がある．近年では3･3･2項で述べる不確かさの概念が導入
されているが，まだ日本工業規格（JIS:Japanese Industrial
Standards ）においても，誤差と不確かさの概念が混在しているの
で，現状でのそれらの定義を示しておく．
(１) 繰返し性（repeatability）
次の条件で行われた，同じ測定量の一連の測定における結果間の
一致の程度．
同一測定手順，同一測定者，同一条件下で用いられる同一測定
装置，同一場所，短時間での繰返し
(２)再現性（reproducibility)
次の諸条件を変えて測定を行ったときの，同じ測定量の測定結果
間の一致の程度．
測定の原理または方法，測定者，測定装置，場所，使用条件，
時期が異なる
(３)認証値の不確かさ（uncertainty of a certified value）
ある量の認証値につけられた推定値で，真の値が，記述された信
頼水準で，確実にその中に入るといえる値の範囲を指定したもの．
(４)精度（precision）
規定された条件で得られる独立した試験値間の一致の度合い
( JIS Z8103「計測用語」では，precisionを精密さと訳してい
る）．
(５)真度（trueness ）
一連の多数の試験値から得られる平均値と採択された参照値との
間の一致の度合い（真度の尺度は，通常かたよりを用いて表す）．
(６)精確さ（accuracy）
精度と真度を総合的に表す（JIS Z8103「計測用語」では，
accuracyを精度と訳している）．
