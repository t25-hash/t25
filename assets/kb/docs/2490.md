# 9・1・2 ショットピーニング(shot peening)等の表面改質法

> **章**: 第9章 表面処理
> **節**: 9・1 耐食性向上表面処理

(surface modification)ブラスティング,ピーニング,レーザ・電子ビーム溶融法,イオン窒化,イオン注入,拡散浸透処理,化成処理などの各種の表面改質法が各種の機械,構造物に適用されている.本法は表面の改質により耐食性を付加するもので,めっきや溶射等の被覆法に比べ,皮膜がはく離するおそれが少ない.基本的には改質層が腐食や溶失に耐えて表面層に存在していれば母材の耐食性は改善されると考えてよい.図9・4は自動車リーフスプリング用高強度鋼平板試験片の3%NaCl水溶液環境中における三点曲げ腐食疲労試験結果である.圧縮残留応力が存在するショットピーニング層により同鋼の腐食疲労強度が改善されていることが明らかである.ショットピーニングは耐SCC性の改善にも効果的である.拡散浸透処理法ではAl(カロライジング)による高温設備の耐酸化防止,Cr(クロマイジング)による高温腐食防止等ボイラ,ガスタービン等の高温機器部品に適用されている.
