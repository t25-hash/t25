# 3・3・2 不確かさ

> **章**: 第3章 測定概論
> **節**: 3・3 測定の誤差と不確かさ

とが先決であるが，いわゆる4W1Hの，いつ，どこで，誰が，何を，いかに測定したかの状況そのものが不確かさを生じる根本的かを見積もることが不確かさの評価につながる．
従来の偶然誤差と系統誤差の分け方と，新しいAタイプ，Bタイプの分け方を直接関係づけることは困難であるが，不確かさとすること，およびこれらを何らかの形で合成して総合的な評価を(ａ)従来の精度・誤差評価：EAするという思想は一致している．
したがって，不確かさの成分ごとの判断（ここに専門家としての知恵が要求される）に依存した形で整理することが可能となる．
測定結果の性能を客観的に評価する方法として，従来からの誤差な原因として考えられ，これらの要因がどのように結果に寄与する管理における特性要因図の考え方と一致する．
評価方法とここで述べた新しい評価方法を対比して示したものであ（あるいは誤差）の成分を分類してそれぞれの大きさを推定しよう測定における不確かさの定義は次のように説明されている．
「測(ｂ) 不確かさ表現の概念：UAに分類した元の形にさかのぼれば，その後の合成方法は評価する者的特性， ：従属特性， ：要因数， ：分散）．
