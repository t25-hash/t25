# 9・2・1 耐摩耗表面設計

> **章**: 第9章 表面処理
> **節**: 9・2 耐摩耗性向上表面処理

部品，素材などの耐摩耗性（wear resistance）を向上させる表
面処理（surface treatment）方法は多岐にわたるが，大きく分け
ると，高周波焼入れなどの熱処理法，浸炭，窒化などの拡散処理
法，肉盛溶接，溶射，めっきなどの表面被覆法，CVD，PVD，イ
オンプレーティングなどによる薄膜被覆法がある．部材の表面を硬
化させ，耐摩耗性を向上させる表面硬化（hard facing）法が一般
的であるが，潤滑性を付与し耐摩耗性を向上させる軟質皮膜の形成
法もある．表面処理法の表面硬化層厚みと硬さの関係を図9･5に示
す．
耐摩耗対策は，相手材との接触により起こる界面での摩耗機構に
対処した表面設計が必要である．摩耗損傷原因の多くは，凝着摩
耗，アブレシブ摩耗，疲労摩耗である．摩耗機構に対処した表面皮
膜の組成の一般的な概念を図9･6に示す．雰囲気，潤滑条件，荷重
などにより同じ表面皮膜でも摩耗特性は変わるので，実際の使用環
境と表面に要求される特性を考慮して最適な表面処理方法を選定す
る必要がある．
9・2・2熱処理（heat treatment)
熱処理による表面硬化法としては，高周波焼入れ（induction
hardening)，火炎焼入れ（flame hardening)，レーザ焼入れ
（laser hardening）などがある．高周波焼入れは，火炎焼入れに比
べ，短時間で局部加熱，焼入れが可能であり，生産性が高く硬さ特
性を制御しやすい．レーザ焼入れは，必要な部分を急速加熱，自己
急冷でき，緻密で深い硬化層を得ることができる．焼入れされる材
料は，炭素量0.3～0.5 %を含む炭素鋼が多いが，高炭素鋼や鋳鉄
でも可能である．中炭素鋼では，浸炭焼入れほどの硬さは得られな
いが，高炭素鋼や軸受鋼では十分な硬さが得られる．疲労強度向上
を主目的として使用されることが多いが，耐摩耗性も向上する．
9・2・3拡散処理
拡散処理は，母材とは異なる元素を表面から拡散させ，表面に耐
摩耗性を付与する方法である．拡散させる元素により，浸炭，窒
化，浸炭窒化，浸硫，浸硫窒化，ホウ化処理などがある．
a.浸炭（carburizing) 低炭素鋼を，炭素を供給する物質中
で加熱し，表面層の炭素量を増加させ焼入れ硬化させる表面硬化処
理方法で，浸炭焼入れとも呼ばれる．浸炭処理法には固体，液体，
ガス，真空浸炭法があるが，生産性と制御性からおもにガス浸炭法
が用いられている．低炭素鋼でも，HV0.8GPa程度の表面硬化層
が得られ，内部は柔軟な組織のままであることから，耐摩耗性と靱
性を要求される部品に使用される．
b.窒化，浸炭窒化（nitriding，carbonitriding) 窒素を内部
拡散させ，表面に窒化物を形成させ，Al，Crを含む鋼では
HV0.8～1GPa以上の硬化層を得る表面硬化法である．窒化法に
は，ガス法，塩浴法，イオン法がある．処理可能材は，ガス法で
は，Al，Crを含む鋼に限定されるが，塩浴法，イオン法では，多
くの構造用鋼，工具鋼に適用でき，処理時間も短い．窒化した鋼
は，浸炭と同様に耐摩耗性と靱性に優れる．鋼のA点以下の温度
で処理するため，焼入れひずみなどの変形が少なく，寸法精度に優
れるが，硬化層の厚みは0.1～0.3mmと，浸炭に比べ小さい．工
業的には，浸炭と窒化を同時に行う浸炭窒化処理が実用化されてい
る．図9･7に，浸炭窒化処理したS45Cの摩耗特性の例を示す．
