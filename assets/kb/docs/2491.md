# 9・2・1 耐摩耗表面設計

> **章**: 第9章 表面処理
> **節**: 9・2 耐摩耗性向上表面処理

部品、素材などの耐摩耗性(wear resistance)を向上させる表面処理(surface treatment)方法は多岐にわたるが、大きく分けると、高周波焼入れなどの熱処理法、浸炭、窒化などの拡散処理法、肉盛溶接、溶射、めっきなどの表面被覆法、CVD,PVD、イオンプレーティングなどによる薄膜被覆法がある。部材の表面を硬化させ、耐摩耗性を向上させる表面硬化(hard facing)法が一般的であるが、潤滑性を付与し耐摩耗性を向上させる軟質皮膜の形成法もある。表面処理法の表面硬化層厚みと硬さの関係を図9・5に示す。耐摩耗対策は、相手材との接触により起こる界面での摩耗機構に対処した表面設計が必要である。摩耗損傷原因の多くは、凝着摩耗、アブレシブ摩耗、疲労摩耗である。摩耗機構に対処した表面皮膜の組成の一般的な概念を図9・6に示す。雰囲気、潤滑条件、荷重などにより同じ表面皮膜でも摩耗特性は変わるので、実際の使用環境と表面に要求される特性を考慮して最適な表面処理方法を選定する必要がある。図9・5 表面処理法の表面硬化層厚みと硬さ図9・6 摩耗機構に対処した表面皮膜の組成
