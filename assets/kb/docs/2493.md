# 9・2・3 拡散処理

> **章**: 第9章 表面処理
> **節**: 9・2 耐摩耗性向上表面処理

拡散処理は,母材とは異なる元素を表面から拡散させ,表面に耐摩耗性を付与する方法である.拡散させる元素により,浸炭,窒化,浸炭窒化,浸硫,浸硫窒化,ホウ化処理などがある.a. 浸炭(carburizing)低炭素鋼を,炭素を供給する物質中で加熱し,表面層の炭素量を増加させ焼入れ硬化させる表面硬化処理方法で,浸炭焼入れとも呼ばれる.浸炭処理法には固体,液体,ガス,真空浸炭法があるが,生産性と制御性からおもにガス浸炭法が用いられている.低炭素鋼でも,HV 0.8GPa程度の表面硬化層が得られ,内部は柔軟な組織のままであることから,耐摩耗性と靱性を要求される部品に使用される.b. 窒化,浸炭窒化(nitriding,carbonitriding)窒素を内部拡散させ,表面に窒化物を形成させ,Al,Crを含む鋼ではHV 0.8~1GPa以上の硬化層を得る表面硬化法である.窒化法には,ガス法,塩浴法,イオン法がある.処理可能材は,ガス法では,Al,Crを含む鋼に限定されるが,塩浴法,イオン法では,多くの構造用鋼,工具鋼に適用でき,処理時間も短い.窒化した鋼は,浸炭と同様に耐摩耗性と靱性に優れる.鋼のA点以下の温度で処理するため,焼入れひずみなどの変形が少なく,寸法精度に優れるが,硬化層の厚みは0.1~0.3mmと,浸炭に比べ小さい.工業的には,浸炭と窒化を同時に行う浸炭窒化処理が実用化されている.図9・7に,浸炭窒化処理したS 45Cの摩耗特性の例を示す.c. 浸硫,浸硫窒化(sulfurizing,sulfonitriding)浸炭や窒化のように表面を硬化させるのではなく,表面に多孔質の硫化鉄層または硫化鉄を含む層を形成し,耐焼付き性や耐摩耗性の向上を図る処理である.浸硫窒化は,浸硫と窒化を同時に行う処理方法であり,硬い窒化物層の表面に,イオウを含む多孔質層が形成され,高面圧条件下での耐焼付き性や耐摩耗性に優れる.
