# 9・2・5 薄膜被覆

> **章**: 第9章 表面処理
> **節**: 9・10 に,無電解ニッケル-リンめっきとニッケル-リン-炭化ケイ素

0.1~10 mの薄膜を成膜する方法として,CVD 法(chemicalvapour deposition method),PVD 法(physical vapour deposi-tion method)がある.形成される薄膜には,TiN,TiC,CrN などの硬質皮膜と,潤滑性を付与するAu,Ag,Pbなどの軟質皮膜がある.薄膜被覆以外に,表面の物性を変えることにより耐摩耗性を付与するイオン注入法,イオンビームミキシング法がある.CVD 法は,薄膜を基材表面に気相化学合成(化学蒸着)する方法で,反応に必要なエネルギーを供給する方法により,熱CVD法,プラズマCVD 法,光CVD 法がある.処理温度がPVD 法に比べ高いため,密着強度が大きい耐摩耗性に優れた皮膜ができるが,母材が熱影響を受け強度が低下する欠点がある.CVD 法では,反応ガス種の制御により,TiC,WCなどの炭化物膜,TiN,SiNなどの窒化物膜,TiBなどのホウ化物膜,ダイヤモンドに近い特性を示すダイヤモンドライクカーボン(DLC)膜など,各種の耐摩耗膜が形成できる.PVD 法は,通常10~10Paの真空中で加熱やプラズマ照射図9・10 無電解Ni-PめっきとSiC複合めっきの摩耗特性いては最適条件が存在し,過大であっても疲労強度が低下する(図図9・11軟鋼板の引抜加工における各種表面処理材の見掛けの摩擦係数および摩耗深さなどの手段で固体源から構成原子を気相状態にし,基板上に移送・凝集させて製膜する方法で,真空蒸着,スパッタリング,イオンプレーティング法などがある.イオンプレーティング法は,蒸発させた物質をイオン化し,基板に照射,堆積しながら膜を形成する方法で,膜の密着性が高く,高速成膜が可能である.反応ガスを導入することにより,TiC,TiCN,TiN,CrN,c・BN,AlOなど多くの耐摩耗膜の成膜が,イオンプレーティング法により可能である.CVD 法およびPVD 法により成膜される硬質皮膜の特徴は,硬さがHV 2GPa以上と,溶射,めっきなど他の表面処理法に比べ著しく高いことで,母材との密着性にも優れていることから,低摩擦係数と優れた耐摩耗性を示す薄膜が多い.図9・11に各種表面処理材の摩擦および摩耗特性の例を示す.他の処理材に比べ,CVD,PVD 処理材が,摩擦係数は低く耐摩耗性に優れていることがわかる.薄膜は,摩耗により膜が消滅すると,一般に摩擦係数が増加することから,耐摩耗性を,低摩擦係数を維持する滑り距離などで評価することも多い.平滑な部材においては外力が一般に表面で最大になることが多く,また表面には切欠,切削傷など疲労破壊の起点となる応力集中部が存在しやすいなどの理由によって,機械部品の疲労破壊は一般に表面から発生することが多い.したがって特に部材表面を改質できれば疲労強度(fatigue strength)の改善が期待できる.このような疲労強度の改善を目的とした表面処理には大別して以下の二つがある.・塑性加工による方法(ショットピーニング,ロール加工など)・熱処理による方法(高周波焼入れ,火炎焼入れ,浸炭,窒化など)これらの表面処理による基本的な疲労強度向上機構は,(1)表面層の硬さが上昇すること,(2)表面層に圧縮残留応力が形成されることの2点によるものであるが,処理方法によっては微視組織やひずみ時効の影響もあるほか,処理条件や部材形状によって各因子の影響度合いは異なり,場合によっては逆効果になる場合もあるので注意が必要である.以下に各処理法の概略と疲労強度に及ぼす影響について概説する.
