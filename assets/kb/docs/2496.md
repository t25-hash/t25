# 9・3・1 ショットピーニング(shot peening)

> **章**: 第9章 表面処理
> **節**: 9・3 疲労強度向上表面処理

ショットピーニングは平均直径1mm前後の鋼球を圧縮空気や遠心力によって高速度で部材の表面に打ちつけ、表層部に発生する圧縮残留応力と加工硬化によって部材の疲労強度を向上する表面加工処理である。本処理は、ばね、浸炭部品、溶接止端部の疲労強度向上策として工業的に広く用いられており、材料についても鋼だけでなく、アルミニウム合金等の非鉄材料にも適用される。ショットピーニングにより得られる硬化層および圧縮の残留応力(residual stress)層はせいぜい表面から200~300 m程度であり、後述の表面ロール加工や熱処理による硬化方法と異なりかなり浅い。そのため処理材の引張強さにはほとんど影響を及ぼさないが、疲労強度には顕著な効果を及ぼす。図9・12に示すように表面が黒皮のままであったり脱炭層がある場合などに特に効果が大きい。ばねについては工業的に最も広くショットピーニングが適用されており、図9・13に示すようにショットピーニングによる疲労強度改善効果が著しいことがわかる。また平滑部よりも切欠部に対して、さらには大型の部材よりも小物部品に対してその効果が顕著に現れる。ただし、切欠部材の場合、切欠底の応力集中部に確実にショットが当たることが条件である。ショットピーニング効果は被加工材やショットの材質、ショット径、速度、パス回数などの条件によって変化する。ショット径については小さいほど効果があるが、ショット速度、パス回数、アークハイト(平板にショットを与えたときのわん曲の度合い)につ裂が発生するため疲労強度が低下すると考えられる。炭素鋼の場合には含有炭度量が多いものほど、また強度が高い材料ほど形成される圧縮残留応力値が大きくなり、疲労強度の上昇が大きい。軟質の材料では圧縮残留応力が負荷応力の繰返しとともに減衰しやすく、残留応力よりも加工硬化による改善効果のほうが大きいようである。図9・12 ショットピーニングその他の表面処理を施したSUP5板材の曲げ疲労強度図9・13ばね(ピアノ線)の疲労強度に及ぼすショットピーニングの効果図9・14ショットピーニング条件と疲労強度増加率(=ショットピーニング処理後の疲労強度/無処理の疲労強度)図9・15 表面ローラ加工材(0.38%C鋼、ローラ径50.8mmφ、ローラ荷重1t)の回転曲げ疲労強度とその後の加工層研磨除去の影響2―235浸炭焼入れ材へのショットピーニングの適用も効果的である。浸炭ままでは表面粒界酸化層や不完全焼入れ層[残留オーステナイト(retained austenite)が多い異常層]が存在する場合があり、ショットピーニングでこの残留オーステナイトをマルテンサイト(martensite)に変態させることにより圧縮の残留応力を形成し、表面硬度を増大させることができるからである。アルミニウム合金にショットピーニングを行った場合には、その効果は鋼に比べて小さく、表面粗さの増大等のために逆に疲労強度を低下させる場合があるが、腐食環境下ではかなりの効果があるとされる。アルミニウム合金の場合、鋼と異なり加工硬化は疲労強度増大にはあまり寄与せず、圧縮残留応力によるき裂進展抵抗の増大が疲労強度向上の主因子であるとされる。なお、前記のとおりショットピーニングによる加工層の深さは大きくないため処理後に切削などの加工を行うことは適切でなく、また腐食などによって硬化層が失われてしまうと疲労強度向上効果は消失してしまう。さらに加熱を行うと表面の残留応力などのピーニング効果が失われ、表面にショットが残した表面あれの影響だけが残るためかえって逆効果になってしまう。また、前述のとおり疲労強度向上率は大型部材になるほど小さくなるので、小型試験片の疲労試験結果から実部品の疲労強度を推定する場合には注意すべきである。
