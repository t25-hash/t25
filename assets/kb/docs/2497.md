# 9・3・2 表面ロール加工(rolling)

> **章**: 第9章 表面処理
> **節**: 9・14 )。過度にピーニングを施すと、表面粗さが増したり微細なき

適当な荷重でローラなどを部材に押付け、部材表面層に圧縮塑性変形を生じさせると、その際の加工硬化とひずみ時効、および表面層に形成される圧縮残留応力によって疲労強度が向上する。部材全体に圧延、プレス、引張りなどで加工を与えた場合にも加工硬化とひずみ時効によって疲労強度の向上が期待できるが、形成される残留応力が一般に小さく、表面層にのみ加工を与える場合に比べて疲労強度向上効果は小さい。表面ロール加工は車軸等の圧入軸(press-fitted shaft)のフレッチング疲労(fretting fatigue)強度向上に古くから適用されており、近年ではクランクのフィレット部の疲労強度向上にも利用されている。ごく最近では12Cr鋼製の発電ロータに低合金鋼を肉盛溶接した軸受部に圧縮残留応力を付与する目的でロール加工が適用されている。表面ロール加工を施すと平滑材では5~30%程度の疲労強度上昇が見込めるが、図9・15に示すように段付棒のフィレット部やV 溝の底部のような切欠部に適用した場合、条件によっては50%以上の疲労強度向上が期待できる。例えばクランクのピンまたはジャーナルとアームの付け根部分(フィレット部)の疲労強度向上に2―236材料学・工業材料適用されており、SC44鋳鋼製クランクの場合、表面層の加工硬化(表面でHV 205、内部でHV 146)と圧縮残留応力(表面で-413~-264MPa)によって、実体疲労強度で40~50%程度向上することが示されている(図9・16)。負荷するローラ荷重としては表層部の加工硬化が飽和する程度でよいとされ、あまり過大な荷重を加えるとかえって疲労強度を害するようになる(図9・17)。そのほかローラ径やローラの総回転数も疲労強度に影響する。一般に、低強度材では加工硬化が、高強度材では圧縮残留応力が疲労強度向上に大きく寄与すると考えられている。鋳鉄、高力アルミニウム合金、黄銅でも鋼の場合と同様の特徴を示し、最適なローラ圧延荷重が存在するようである。ただし鋳鉄の場合にはほとんど塑性変形しないので、形成される残留応力はわずかであり、強度上昇は主として加工硬化によるものであると考えられる。ロール加工後時効処理を行ったり、温間でロール加工を行うと時効硬化によってさらに疲労強度やフレッチング疲労強度が向上し、ロール加工のままの材料に比べて寸法効果による強度低下が現れにくいため大型の部材についても効果的である(図9・18)。なお、ロール加工を施した部分と未加工の境界部には引張残留応力が発生するため、境界部が負荷による高応力部位とならないよう注意すべきである。(a) フィレット部の圧縮残留応力(b) 疲労強度の比較図9・16 ロール加工した実体クランクの残留応力と疲労強度
