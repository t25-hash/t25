# 9・3・3 熱処理による表面処理

> **章**: 第9章 表面処理
> **節**: 9・14 )。過度にピーニングを施すと、表面粗さが増したり微細なき

a. 高周波焼入れ(induction hardening)鉄鋼にコイルを通して1~500kHz 程度の高周波電流を通じ、その誘導電流によって表面層のみを急速に加熱させ、焼入れ温度に達した後、適当な方法で急冷して硬化させる方法を高周波焼入れという。急速加熱冷却に基づく表面層の材質改善と、それに伴って発生する圧縮残留応力が疲労強度向上のメカニズムである。高周波加熱は(1)一般の焼入れに比べて熱効率が良く、また浸炭焼入れ、窒化などの表面硬化法に比べて短時間に熱処理操作が行える、(2)その結果、酸化や脱炭を生じず微細な組織が得られる、(3)局部加熱が可能で硬化深さの制御が容易である、(4)変形が小さいため後の加工に有利である、(5)被焼入れ面の形状によって適当な加熱コイルを用いれば一定の条件で多量の製品を処理しうるなどの利点がある。そのため高周波焼入れは各種軸類、歯車、コイルばね、クランク軸、ピン、ブッシュなど機械部品に広く適用されている。電気炉焼入れ等の通常熱処理材と同様、高周波焼入れ材についても、硬さの上昇とともに平滑材の疲労強度も上昇するが、同じ硬さでも通常熱処理材に比べて高い疲労強度が得られる[図9・19(a)].これは高周波焼入れ材では、オーステナイト化時間が通常の炉加熱焼入れに比べて極めて短いため、微細な結晶粒と組織が得られるほか、高い表面圧縮残留応力が得られることによるものである。また高周波焼入れ材では硬さの高い領域でばらつきが大きいが、硬さが同じでも材質や処理条件によって組織や残留応力が異なるためである。切欠材においては、通常熱処理では焼戻し温度を下げて硬さを上げると疲労強度はむしろ若干低下するが、高周波焼入れの場合は硬さの上昇とともに疲労強度は比例的に向上し、ずぶ焼入れ(全体焼入れ)に比べて表面焼入れの場合にはいっそう効果が大きい[図9・19(b)].高周波焼入れによって直径8~10mmの平滑材の疲労限度は2~3.5倍に増加するが、切欠試験片の場合は3~5倍に増加する例が多く、9倍に達する例もある。通常焼入れままの材料では切欠係数βは応力集中係数αにほぼ等しいが、高周波焼入れした材料では高い圧縮残留応力のためαを増加させてもβはあまり増加しない(図9・20)。平滑材を高周波焼入れ後、種々の温度で焼戻しても疲労限度はさほど低下しないが、切欠材の場合には焼戻し温度の上昇とともに圧縮残留応力の消失により疲労限度は急激に減少し、電気炉焼入れしたものに接近していくため(図9・21)、高周波焼入れ材の焼戻しは低温で実施される。硬化深度については、浅いとフィッシュアイを形成して内部破壊を生じ本来の疲労強度が得られないことがあり、逆に深すぎると圧縮残留応力の低下と過熱組織の発生からかえって疲労強度が低下する(図9・22)。また、部材の寸法に応じて適当な硬化深度があるようである。なお、部分焼入れにおける焼き境部には引張残留応力を生じるた図9・17 疲労強度に及ぼすローラ加工圧力の影響図9・18 圧入軸の疲労に対する温間加工および冷間加工後時効の影響め疲労限度が低下する場合がある。そのため応力集中部に焼き境がこないように配慮することが望ましい。b. 火炎焼入れ(flame hardening)硬化能を有する鉄鋼の表面をガス炎で加熱して焼入れ硬化する方法であって、ガスとしては天然ガス、都市ガス、プロパンガス、アセチレンガスなどが用いられる。この方法は装置が簡単低廉であるため非常に手軽に利用することができ、高周波焼入れと並んで歯車、鋳物などの機械部品に古くから行われている。火炎焼入れ材の疲労強度に関する系統的な公表データは他の表面硬化処理に比べて少ないが、高周波焼入れと同様、表面の急熱急冷処理であるので、適用材料に対する注意点や、焼入れ後の性質は高周波焼入れの場合とほぼ同様に考えてよいであろう。ただし、一般に薄い硬化層を得ることは高周波焼入れよりも難しく、硬さ分布や残留応力のこう配は比較的緩やかとなる。また、公表された例は単純な軸対称形状部品に対して適用されたものであり、実際の機械部品は必ずしも単純形状ばかりではないので、危険部位に引張残留応力が発生したり、その他の原因で疲労強度が低下するおそれもあり、適用には十分注意を要する。c. 窒化(nitriding)窒化は、500~600°Cの温度域において鋼材の表面に窒素を拡散浸透させ硬い窒化層を形成する処理である。窒化による硬化機構は浸炭法のような焼入れによる硬化ではなく、浸透した窒素が鋼中のAl,Cr,V,Moなどと結合し微細窒化物を形成し、そのひずみ硬化によるものである。処理温度が低いため変形が極めて小さい。窒化による硬化層深さは浸炭によるそれよりも小さいが、材料によっては極めて高い表面硬さ(HV 1000~1300)が得られる。この表面層は耐摩耗性と耐食性に優れた特徴をもっているが、同時に疲労強度も向上する。代表的な窒化法として以下の方法がある。(1) ガス窒化法約770K のNH雰囲気中で加熱する。(2) 塩浴窒化法(タフトライド処理、tuftride)KCN やNaCN の塩浴中に浸漬する。(3) ガス軟窒化アンモニアとプロパンの混合ガス中、または変成炉ガス(RX ガス)中で処理する。(4) イオン窒化法容器内を数百Paの低真空にした後、NあるいはNHなどを含む雰囲気中でグロー放電を起こさせて処理する。ガス窒化法では表面にFeN のもろい化合物が形成されるので、疲労強度改善を目的とする場合には塩浴窒化、ガス軟窒化またはイ図9・25 浸炭ままと浸炭後研削した材料のS-N 線図の比較図9・26
