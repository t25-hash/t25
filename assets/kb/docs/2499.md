# 10・1・1 金属材料成分の分析法および鑑別法

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・1 金属材料の成分と組織検査

a. 化学分析法湿式法とも呼ばれ、化学反応により材料の化学成分を定量する標準的方法である。各元素ごとに定量法は異なり、JIS でも数多くの分析法が規定されている。b. 機器分析法発光分光分析法は試料をアークなどで発光させ、その波長を測定して各元素の定量を行う。蛍光X 線分析法は一次X 線を照射したときに発生する蛍光X 線から同様に分析を行う。工場の品質管理の分析法としての主流である。大がかりな設備を必要とするが、大量処理、迅速化、自動化が可能であり、必要試料も少量で済む。また、原子吸光分析やICP分光分析法など化学分析の機器化も進んでいる。c. 簡易鑑別法定性的または半定性的に化学成分を簡便に判定する方法である。. グラインダ火花試験(spark test)試料にグラインダを当てると火花が発生する。含有する化学成分の種類、量によって火花の形態、すなわち流線の色、形、長さおよび破裂(花)の形状、数、色彩などが異なる。既知の材料との比較からかなりの鑑別が可能であるが経験を要する。鋼についての観測法は、JIS G0566:1980に規定されている。. 粉末火花試験試料の粉末をブンゼンバーナの酸化炎で燃焼させ、発生する火花で判別する方法で、銅合金、軽合金にも利用できる。
