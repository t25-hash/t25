# 10・1・3 結晶粒度試験(grain size test)

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・1 金属材料の成分と組織検査

a. 鋼のオーステナイト結晶粒度試験通常,鋼(オーステナイト系の鋼を除く)は,高温でオーステナイト組織であり,その粒度は,その後の冷却時の焼入れ性,鋼材の材質に大きな影響をもつ.試験法は,JIS G 0551:1998に規定されており浸炭粒度試験方法と熱処理粒度試験方法がある.前者は925°Cで浸炭を行い,後者は実際作業に近い温度に加熱後,研磨,腐食し結晶粒の大きさを測定し粒度番号で表す.粒度番号は付図の標準図と比較して決定する.b. 鋼のフェライト結晶粒度試験使用状態の組織であるフェライト組織の結晶粒度を求める.炭素含有量0.2%以下の鋼に適用される.JIS G 0552:1998に規定されている.
