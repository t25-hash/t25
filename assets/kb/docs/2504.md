# 10・2・1 圧延材の欠陥

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・2 金属材料の欠陥

a. ラミネーション(lamination)鋼塊中には、ある程度の非金属介在物の存在は避けられない。多くは熱間圧延時に圧着されるが、酸化物系介在物が多いと鋼中水素の内圧も加わり層状の割れとなる。b. サルファバンド(sulfur band)硫化物が層状に偏析したもので、割れではなく、また、ラミネーションとは無関係である。溶接部の近くに存在すると、サルファクラックやラメラテアの原因になることがある。c. 表面欠陥(surface defect)鋳造原因の欠陥には、へげきず、二重肌、縦割れ、横割れなどがあり注入時の温度や速度の改善が必要である。圧延原因の欠陥には前段ロールでできた耳をたたみ込んだ折込み、スケールが付着したまま圧延したスケールきず、ロールの不良によるロールきずなどがある。
