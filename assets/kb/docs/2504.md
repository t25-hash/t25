# 3・5・5 各種の補償法

> **章**: 第3章 測定概論
> **節**: 3・5 補償法

補償には外乱変数と目的変数の関係から差動構造と除算構造があ
るが，その演算に供する二つの出力のとり方にも，同時にとるもの
と繰返しの計測でとるもの，空間的に異なる点の情報を得るものと
同一点の情報を得るものとがある．
a.反転法 目的変数と外乱変数との位相関係が逆になるよう
な状態を生じさせるために，変換器に反転（reversal ）という操作
をしてその前後の2回の測定結果の和（または差）をとる方法を反
転法（reversal method）と呼ぶ．
図3･35のように，床などの平面の水平からの狂いを計測する道
具に水準器（level ）がある．この水準器の基準を示す標線が必ず
しも正しい水平位置を示しているとは限らない．この水準器の有す
る器差を取り除くために水準器を水平面内で180°反転して測定す
る．水準器の入力である対象面の傾斜と器差の符号の関係が反転に
よって逆転する．2回の読みの和をとると傾斜が，差をとると器差
が明らかになる．
類似の補償構造に，てんびんの左右の不釣合いによる器差を取り
除くための二重秤量法（double weighing，図3･36）や置換法があ
る．
図3･37に微小な電位を計測する電位差計を示す．電位差計にお
いて，導線の接続部で生じる熱起電力の影響を消去するための反転
法が用いられる．図3･37(ｂ)のように，電源の向きを逆にしても
う一度測定し，2回の結果の平均値をとる．
b.形状測定における反転法 図3･38に，真円度
（circularity，あるいは，軸の回転運動誤差）の測定に用いられる
反転法の模式図を示す．回転する円筒に向けた変位センサで円筒表
面までの距離の変動を測る，いわゆる半径法による真円度測定で
は，センサの出力には被測定円の真円からのずれ（真円形状）と，
回転軸のセンサ感度方向の振れ（半径方向回転運動誤差，ラジアル
モーション）が含まれる．図3･38のように，回転軸に対して被測
定円筒とセンサを180°反転した配置にしてもう一度測定を繰返す
と，反転前後で回転運動誤差と真円形状の振れの位相が180°ずれ
る（符号が逆転する）ことになり，2回の測定結果の平均値から真
円形状が，2回の測定の差から回転誤差が得られる．もちろん，反
転前後の2回の測定での回転運動誤差に十分高い繰返し性が必要で
ある．同様の考え方で，真直形状を測るための反転法も用いられ
る．
図3･39に示す改良型反転法 は，反転前後の運動誤差の繰返し
性を必要としない．被測定円筒上に置かれた補助基準円筒に関して
相殺される．
反転法が適用され，被測定円筒の真円形状は反転のない2回の繰返
し測定になる．この方法は，軸方向に測定空間を広げたために軸の
回転中の傾斜運動（アンギュラモーション）の影響を受けることに
なる．
なお，この改良型反転法は真直度（straightness ）の測定にも応
用される ．また，反転操作を含む他の方法として，直定規
（straight edge）を3個組合わせて測定する3直定規法がある ．
よい．
その場合は図3･40のように，対向させた直定規に関してそれらの
間隔を測る形で二つの変位センサで走査測定する．これを3本の三
つの組合せについて行うと，運動誤差の取り除かれた真直形状が求
められる．定盤の製作時に平面度の検査に用いられている3面合せ
法（3-flat method) も同じ原理に基づいていて，厳密には平面
