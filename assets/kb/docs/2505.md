# 10・2・2 鍛造品の欠陥

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・2 金属材料の欠陥

a. ゴースト(ghost)大型鍛鋼品の切断面に見られる。P,Sなどの偏析で圧延、鍛伸方向に沿って変形している。割れやき裂は伴わない。b. 白点(flake)Ni,Cr,Moなどを含む自硬性の高い鋼材を熱間鍛造後、冷却したとき、内部に発生する割れで、その形状によって毛割れ(hair crack)、銀点(fish eye)と呼ばれる。鋼中の水素、残留応力(変態応力、熱応力)、偏析、介在物が複合して発生原因となっている。造塊前に溶鋼の脱ガス処理を行うと防止効果が大きい。白点は200°C以下で発生するので、鍛造後徐冷や再加熱処理により、脱水素、応力低減、軟化を図ると防止効果が大きい。c. 置割れ(season crack)鍛造後または熱処理後、長時間経過してから生じる割れで、過大な残留応力が主因である。気温の激変時に生じやすく、大きな音を発することがある。d. もみ割れ(forging crack)鍛造条件が不適切なため中心部に生じたき裂である。高合金鋼に多く見られる。e. 砂きず(sand mark)脱酸生成物、溶鋼に混入した微細なスラグや耐火物が圧延、鍛造により変形したもので、肉眼、磁粉、浸透探傷法などの検査で確認されたとき、砂きずという。密集度が低い場合には強度面への影響は少ない。その他、圧延材と同様、へげきず、かぶさり、スケールきず、きっ甲割れ、焼けすぎなどの表面きずがある。これらの防止には適切な加熱・鍛造条件の選択が必要である。
