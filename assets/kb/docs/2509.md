# 10・3・1 種類と適用

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・3 非破壊検査

非破壊検査(NDI: nondestructive inspection)は,検査対象物の形状,機能を損なうことなしに検査し,部材中の欠陥(きず)を検出して品質分類,補修の要否,供用中の部材に対する残存寿命の推定などの判定を行う.欠陥の許容度は,使用目的,使用条件,材質により異なる.溶接鋼構造物では,この決定に破壊力学的考え方が大きく取り入れられている.非破壊検査は間接的手法によるため,ある程度の不確実さを残しているが,あらかじめ存在する欠陥の状態を予測し,最も適切な手法を選んで行えばかなり高度の判断が行えるようになってきている.表10・1におもな試験方法の種類とその適用の概要を示す.
