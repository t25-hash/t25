# 10・3・2 放射線透過試験(RT : radiographic test)

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・3 非破壊検査

X 線,γ線(以下放射線)は物体の中を透過するので,一様な強さの放射線を照射すれば欠陥により放射線の吸収に差が生じ,写真フィルムに撮影することができる.普通の工業用X 線装置は,管電圧が10~400kVpで,鋼板の場合,100mm近くまでの検査に適用される.電子を加速した高エネルギーのX 線装置によれば,100mm以上の鋼板の検査が可能である.公称エネルギー12MeVのライナック,25MeV のベータトロンでは最高500mmまでの厚さの鋼板の検査ができる.γ線は小型で高エネルギー放射線が得られ,複雑な形状や,厚物の鋼材に適用される.Coは鋼板25~140mm,Irは10~60mmの検査に適している.図10・1に撮影配置の例を示す(JIS Z 3104:1995).透過度計は,識別度を見るもので,識別可能な針金の最小径を試験体の厚さで除した値で表し,通常は2%が要求される.階調計は板厚が20mm以下の場合に使用して写真像の質の確認を行う.小さな欠陥を見落とさない,すなわち像質を改善するためには線源の寸法を小さくするのが有効であることから,マイクロフォーカスX 線装置も使用されてきている.検査対象物に応じて,欠陥を,割れ,ブローホール,スラグの巻込みなどに分類し,欠陥の大きさ,数により等級分類を行う.JISにも多くの規格があり,JIS Z 3104,JIS Z 3105:1993,JIS Z3106:2001,JIS Z 3107:1993などは材質別の撮影方法,等級分類,判定基準を包含している.
