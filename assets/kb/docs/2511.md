# 3・10・1 原理と特徴

> **章**: 第3章 溶接・接合および切断
> **節**: 3・10 接着接合

接着（adhesive bonding）とは，材料との化学的結合性が高い
接着剤（adhesive）を，液相状態で接合界面に供給した後に固化
させて接合する方法である．通常，常温で行われるが，固化を促進
するために加熱することもある．接着させるには，接着剤が被接着
体の表面をぬらし，次いで両者の間に何らかの結合が起こる必要が
ある．
接着剤と被接着体の界面において，結合力が発現する要因には諸
説あるが，現在では分子レベルでのミクロな結合力に，アンカー
（投錨）効果等のマクロな結合が寄与しているとされている ．ミ
クロな結合力としては，通常，水素結合やファン・デル・ワールス
力が考えられるが，一部が化学結合（共有，イオン結合など）によ
る場合もある．マクロな結合とは，接着剤が被接着体の凹部に流入
固化して結合するというアンカー（投錨）効果による機械的結合で
あり，接着強度に大きな役割を及ぼす．
接着の利点は，(１)被接合体の種類を問わない，(２)極端な場合
を除いて接合面の形状を問わない，(３)せん孔による応力集中がな
く残留応力が生じない，(４)比較的低コストである，などが挙げら
れる．
