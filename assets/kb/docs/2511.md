# 10・3・3 超音波探傷試験(UT : ultrasonic test)

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・3 非破壊検査

超音波は波長が短く、直進性があり、固体と液体・気体の境界面で反射されやすいので金属内の欠陥検出に適している。おもに用いられているのは、周波数が500kHz~10MHz のパルス反射方式である。パルス発振器で発生させた電圧を振動子に加え超音波パルスを発生させる。振動子は水晶、チタン酸バリウム(略称チタバリ)、表10・1 非破壊試験方法の種類と適用放射線透過試験内部鋳造材(パイプ、ひけ巣)、溶接部(スラグ巻込み、ブローホール) 密着した割れ超音波探傷試験内部圧延材(ラミネーション)、鍛造材・溶接部(割れ)鋳鉄、オーステナイト鋼磁気探傷試験表面・表層強磁性鋼全般、鋳鍛鋼材、鋼溶接部非磁性材料浸透探傷試験表面開口金属・非金属共通多孔質材料電磁誘導探傷試験表面・表層管・線など単純な形状のもの形状が複雑なものアコースティックエミッション(割れの発生、成長) 圧力容器の耐圧試験時、その他溶接割れなどすでに存在している欠陥2―243ジルコンチタン酸鉛(ジルナマ)、硫酸リチウムなどの電圧材料を切り出したもので、ダンパなどと合わせて取扱いやすいよう一つのケースに組込んだものが探触子(probe)である。振動子は、送信・受信用を兼ねることが多い。また、最近では広い周波数帯域を有する振動子として、ニオブ酸鉛、高分子材料(ポリマ)や圧電体と高分子を複合化したコンポジット(複合圧電体)が使用されている。固体中を伝わる超音波には、空気中、水中を伝わる疎密波(縦波)のほか、横波、表面波、板波がありいずれも探傷に使われる。超音波が境界面に垂直に当たったとき、一部は反射し残部は通過するが、その割合は境界面で接している二つの物質の音速と密度で決まる。空気と鋼ではほぼ100%反射するが、水と鋼では音圧通過率が12%である。したがって、探触子と試験材の間は水、油、グリセリンなどの液体で満たして通過しやすくする。垂直探傷法(nor-mal beam method)では、垂直探触子で発生させた縦波を利用する。図10・2にパルス反射法による垂直探傷の原理を示す。図ほか、発振器から直接の高周波電圧も受信器に入る(送信パルスT)。したがって、図10・3に示すように、欠陥位置、大きさを図形から読み取ることができる。超音波が境界面に斜めに当たったときは、境界で反射と屈折が起こる。探触子と鋼のように固体どうしの場合には、反射波、屈折波ともに縦波、横波が存在する。斜角探触子では、屈折縦波を臨界角以上にして試験材中に横波だけ入るようにしている。通常、通過率を良くするため、屈折角は35°~80°の範囲で用いられる。探傷感度を決める方法には標準試験片の人工欠陥によるエコー高さを基準にする方法と、底面エコーを基準にする方法がある。探傷感度や測定範囲の調整、探触子の性能点検に種々の標準試験片、対比試験片が用いられる(例JIS Z 2345:2000)。欠陥の分類、判定は、エコー高さ、欠陥指示長さ、個数(頻度)から総合的に決められる。超音波探傷は平板状の欠陥であればどんなに薄くても大きな欠陥試験方法対象欠陥位置おもな適用分野適用が困難なもの図10・1 撮影配置の例図10・2 パルス反射法の原理図10・3 探傷図形と見方2―244材料学・工業材料エコーを得ることができるが、球状の欠陥(例えばブローホール)の検出能力は低い。また、試験材の金属組織が微細であれば、超音波はかなり遠くまで到達するので、直径が数mの大型鍛鋼品の内部の探傷も可能である。しかし金属組織が粗いとわずかな反射・散乱が重なり、林状エコーや減衰が著しくなる。18-8ステンレス鋼の鋳物と溶接部、大型鋳鋼品などでは、これらが原因で探傷が困難になる。今日の電子技術、映像化技術の進歩により、屈折角や音波の絞りを連続的に変化させることができるフェーズドアレイ探傷法(phased array method)や電気・磁気的作用を応用した電磁超音波探傷法が使用されるようになってきた。また、最近では、破壊力学的な手法による残存寿命の推定、補修の要否を決定するうえで欠陥寸法、特に欠陥の深さを知る必要があることから、欠陥先端の回折波の伝ぱ時間を取扱う端部エコー法(flaw tip echo method)、TOFD 法(time of flight diffraction method)が使用されている。
