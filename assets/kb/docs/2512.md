# 3・10・2 接着剤

> **章**: 第3章 溶接・接合および切断
> **節**: 3・10 接着接合

接着においては，被接着材料や使用環境に適した接着剤の選択が
重要となる ．
a.縮合系接着剤 尿素樹脂，フェノール樹脂など熱硬化性樹
脂をベースとする接着剤で，樹脂成形と同様の方法により硬化，接
着する．主たる用途は合板の製造である．
b.溶剤型接着剤 合成樹脂またはゴムを主成分とし，溶剤に
溶かして接着剤としたものである．多くの分野で使用されている
が，溶剤の毒性や溶剤揮発による大気汚染があり，脱溶剤が進みつ
つある．
