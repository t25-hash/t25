# 10・3・4 磁気探傷試験(MT : magnetic test)

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・2 のような場合(一探触子法)、欠陥エコーF、底面エコーB の

強磁性体を磁化したとき、表面近くに欠陥が存在すると表面に漏えい磁場が発生する。これを磁粉、あるいはホール素子やマグネチックダイオードなどの感磁素子で検出して、欠陥の位置、形状、大きさを知る。表面欠陥の検出法としては、感度が良く、肉眼による直接観察ができるので、最も有力な方法となっている。なかでも磁粉探傷法は、古くから広く使われている。磁粉探傷の磁化方法には、電磁石による磁場を用いる方法(極間法など)、電流による磁界を用いる方法(プロッド法、直角通電法は、試験体に直接電流を流す)がある。いずれも磁場は欠陥に直角に加える必要がある。交流、直流ともに用いられるが、表面に近い内部欠陥の検出には直流のほうが適している。必要な磁化の程度、磁化電流は磁化方法、磁粉、試験体の磁化特性によって変わる。極間法、プロッド法などではJIS G 0565:1992に定めたA 型標準試験片で表面有効磁場の強さ、方向を確認する。磁粉はおもに微細な鉄粉が使用される。乾式用は空気に分散させて用い、10~60 mの粒子径のものが多く、湿式用は、1~10 mの磁粉を液体(白灯油または水)に分散懸濁させ検査液として用いる。検査液の磁粉濃度は、通常、溶媒1L中に含まれる磁粉の量で示す。白色、黒色に着色した非蛍光磁粉と、暗所で紫外線により黄緑色などの蛍光を発する蛍光磁粉がある。磁粉または検査液をふりかけることを磁粉の適用という。連続法では試験体に磁場が加わっている状態で磁粉を適用する。残留法は保磁力の強い材料で磁化したあと磁粉を適用する。材質変化の境界、断面寸法の急変部は欠陥がなくても磁粉模様が現れることがある(疑似模様)。欠陥の判定には、それまでの履歴なども含めた総合判断が必要である。JIS G 0565では線状欠陥と円形状欠陥について等級分類をしている。磁気探傷試験はオーステナイト系ステンレス鋼のような非磁性材料には適用できない。
