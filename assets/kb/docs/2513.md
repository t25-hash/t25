# 3・10・3 種々の接着法

> **章**: 第3章 溶接・接合および切断
> **節**: 3・10 接着接合

構造を改善するか，被接着体と接着剤との界面相互作用を大きくするような方策，あるいは相互作用を阻害する要因（例えば表面汚染など）を除去することが有効である．
したがって，接着性を向上させるためには被接着体の表面構造あるいは表面層の強化を行う必要がある．
以上のように，接着部で破壊が生じた場合には，その破壊面の詳細な観察を行うことが重要であり，それぞれの破壊に適した対策を講じることで接着性は向上する．
接着接合の継手強度は，継手の種類や被接着体厚さによって変化する．
図3･102は，これらの因子の継手強度に及ぼす影響およびその際に発生するおもな破壊形態を示したものである．
図3･102より，被接着体厚さが増加すると継手強度が増加するものの，ある板厚以上では強度は飽和する傾向がある．
また，スカーフ継手やステップドラップ継手といった荷重の偏心や応力流れが改良された継手の強度が高いことがわかる．
しかし，これらの継手は複雑な機械加工を必要とするため，高強度を要する場合には有効であるが，低強度でも済む場合には加工性を考慮すると必ずしも有効な継手とはいえない．
したがって，要求強度と機械加工性を考慮して継手を決定する必要がある．
a.ウェルドボンド法（weld- bonding) 接着継手とスポット溶接のような溶接継手の互いの欠点を補いつつ，より継手強度が高く信頼性の高い接合部を得る手法として，接着と接合を組合わせたウェルドボンド法が1960年代初期に旧ソ連で開発され，自動車や航空・宇宙関連分野で広く用いられている．
接着部に対しては，はく離強度や耐クリープ性などが改善でき，スポット溶接部に対しては，面接合の効果により応力集中が小さくなり，疲労特性が向上する．
ウェルドボンド法の目的には，接着特性の改善：接着継手の欠点であるはく離強度，衝撃強度，高温強度の改善，点溶接特性の改善：アルミニウム合金や薄鋼板の点溶接強度の低さの改善，および点溶接の疲労特性の改善，接着工程の合理化：点溶接を接着硬化するまでのジグ代わりに用いるなどの合理化などがある．
b.溶着法 熱可塑性プラスチック材を溶かし，プラスチック相互または他の材料と接着する方法である．
一般に接着には圧力が必要である．
接着剤を溶かして硬化させる方法もこれに属する．
溶着には外部加熱による方法として熱風溶着，熱板溶着，内部加熱による方法として，高周波誘導溶着，摩擦溶着，超音波溶着などがある．
