# 10・3・6 電磁誘導探傷試験(ET : eddy current test)

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・2 のような場合(一探触子法),欠陥エコーF,底面エコーB の

交流を流したコイルを試験体に近接させ,コイルのインピーダンスが,試験体の欠陥の存在により変化する様子を検知する.コイルの形状には,1貫通型コイル(線,棒,管など断面が円形の試験体に適用),2プローブ型コイル(板,インゴットなどの表面の欠陥検出に適用),3内挿型コイル(管,穴などに入れて内面の試験に適用)がある.金属のほか,黒鉛など導電性のある材料に適用できる.表面欠陥の検出に適しており,試験結果が電気的出力として得られるので自動化できる.また,欠陥の検出のほか,金属の種類・成分の変化,寸法,塗膜,腐食状況の測定に適用できる.
