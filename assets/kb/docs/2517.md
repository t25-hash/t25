金鋼など 継手 JIS化された試験方法が多い
したがって，溶接性試験（weldability test ）には，目的とする
特性によって，数多くの試験方法が考案され実用に供されている．
表3･41は各種溶接性試験法を基本特性別に分類し，その特徴を示
したものである ．材料および溶材の変態特性，硬化性，溶接割
れ感受性（crack susceptibility），熱影響部の脆化などを調べる試
験と，溶接継手の適合性や母材の溶接構造物として使用する場合の
適性を調べる試験とがある．
JIS Z3040：1995に規定されているように，一般に溶接施工方
法確認の際には，溶接継手の引張り・曲げ試験のほかに継手（溶接
金属，熱影響部）のシャルピー衝撃試験，マクロ組織観察および硬
さ試験などが行われる．構造物の使用条件によっては，破壊靱性
（ ）試験，CTOD（crack tip opening displacement ）試験や大
型脆性破壊試験が行われることもある．
溶接部の硬さを調べるには，一般にビッカース硬さが用いられる
が，実物について現場で調べるときにはブリネル硬さがよく用いら
れる．母材の硬化性の評価には，試験板に溶接ビードを置く最高硬
さ試験やテーパ硬さ試験がJISの規定に従って行われる．
表3･41に示すように溶接割れ試験には，材料の種類，母材と溶
材，低温割れと高温割れ，溶接金属割れと熱影響部割れなどを対象
として多数の試験方法があり，その一部はJISにより規定されて
いる試験もある．目的に応じて試験方法を適切に選定する必要があ
る．
化学成分と組織は母材の溶接性に大きく影響を及ぼす．簡便に熱
影響部の硬さや低温割れ感受性を予測するため，母材の化学成分値
を用いた実験式として炭素当量式Cや溶接低温割れ感受性組成式
P などが使われている．ただし，炭素当量や溶接割れ感受性に関
する実験式を適用するときは，適用条件と適用範囲に注意し，か
つ，これらの計算値は目安であって厳密なものではないことを承知
しておく必要がある．
