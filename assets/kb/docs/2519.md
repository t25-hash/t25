# 3・2・10 ガス溶接（gas welding）

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

各種のガス炎を用いて，ガスの燃焼時に生じる高温を利用して，
金属を溶接する方法を総称してガス溶接という．ガス炎に酸素アセ
チレン炎（oxyacetylene flame）を使用することが最も多く，一般
にはガス溶接といえば酸素アセチレン溶接（oxyacetylene weld-
ing）を指すものと考えてよい．これは，ガス溶接に要求される炎
の性質，すなわち，火炎温度が高い，燃焼速度が大きい，熱量が大
きい，溶融金属に化学的影響を与えないことを満足する炎が得られ
る燃料ガスは，実用的にアセチレンガスだけであるからである．本
法は電源設備が不要で，装置の機動性に富み，場所を選ばず容易に
ガス溶接を行ううえで重要なことは，母材表面の酸化皮膜を完全
に除去することと，溶接中に溶接部を酸化させないことである．そ
のため，母材の金属に適したフラックスが使用される．しかし，軟
鋼のガス溶接では，鉄の酸化物がフラックスとしての作用をするこ
とから，フラックスを用いない場合もある．
ガス溶接施工法には，前進溶接法と後退溶接法と称される二つの
基本的な施工法がある．前者は，母材に対する入熱の調整が行いや
すく，かつ溶接操作が容易であるので，通常，3.2mm程度までの
薄板の溶接に使われる．また，後者は，溶込みを確認しながら溶接
できるので厚板の溶接に適しており，均一で良好な外観の裏波溶接
を行うことができる．
