# 3・2・11 テルミット溶接（thermite welding）

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

テルミット溶接は，金属酸化物や硫化物を金属アルミニウムで還
元するときの発熱反応（テルミット反応，thermite reaction）を
利用する溶接方法で，金属酸化物として酸化鉄を用いた場合，溶融
鉄を得ることができる．テルミット溶接法を大別すると，反応で生
じた溶融金属を接合または肉盛に利用する溶融テルミット法と，反
応熱および反応生成物の有する熱エネルギーを加熱源とする加圧テ
ルミット法に分類される．
鋼用の溶融テルミット法では，酸化鉄とアルミニウムを主成分と
し，溶接金属の品質調整のための合金元素や脱酸剤を配合した粉末
（テルミット溶剤）をるつぼ内で反応させ，得られた溶鋼を，あら
かじめ接合する継手の周囲に設置したモールド（鋳型）に注入して
開先部分を溶接する．なお，溶接継手部分は溶鋼を注入する前に
600～900℃程度に予熱しておく．
鋼用テルミット溶剤の反応は強烈で，生成する溶鋼の温度は
2000℃以上となるが，反応自体は爆発的ではなく，しかも反応開
始に約1200℃以上を必要とするため，その取扱いは比較的安全で
ある．
テルミット溶接は，大断面積部材を容易に比較的短時間に溶接で
き，かつ使用器具が軽量で，大きな電源や加圧力が不要で機動性に
優れるといった特徴がある．日本では鉄道用レールのロングレール
（continuous welded rails ）化のための溶接方法（図3･54) とし
て広く用いられている．また，酸化銅とアルミニウムを主原料とし
た銅用テルミット溶接も実用に供されている ．
