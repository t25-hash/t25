# 10・6・2 摩耗試験(wear test)

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・6 その他の試験

摩耗を接触の仕方で分けると、2金属間の滑り摩耗、転がり摩耗(軸受など)、土砂、岩石との間の土砂摩耗があるが、実際にはこれらが混在することが多い。耐摩耗性は、一般的には硬くて靱性に富んだ材料が優れているが、摩耗の機構によって変化する。したがって、摩耗試験は実際の使用状態をできるだけ再現したほうがよい。摩耗試験機の構造には、1回転する円板の平面部に試験片を押付ける方法(ピンオンディスク)、2回転する円板外周面を平板に押付ける方法、3円筒外周面と小試験片を摩擦する方法、4往復運動する平面に試験片を押付ける方法、5回転する円筒どうしを押付ける方法、6土砂などの中で試験片を回転させ摩耗量を見る方法などがある。
