# 3・2・8 電子ビーム溶接（electron beam welding）

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

電子ビーム溶接は，高エネルギー密度の電子ビームを材料表面に
照射し，溶融・溶接する方法である．図3･49にその原理を示す．
真空チャンバ内で，熱陰極から放出した熱電子を陽極との間に印加
した高電圧により加速し，集束レンズ内の磁場により材料表面付近
に収束する．収束された電子ビームのパワー密度は10～10W/
cmにも達し，アークと比べてけた違いに大きい．このため，電子
ビームが照射した位置では瞬時に材料の溶融と蒸発が起こり，蒸発
時の反跳力により，図3･50に示すようなキーホール（keyhole，
ビーム孔）と呼ばれる細長い空洞が形成される．キーホールは電子
ビーム溶接にとって非常に重要な存在であり，この内面に電子ビー
ムが当たることにより，材料の内部まで熱源が投入され，図3･
50(ｂ)に示すように幅が狭く非常に深い溶込みが得られる．これが
電子ビーム溶接の最大の特徴であり，すでに100mmを超える鋼
板の1パス溶接が実用化されている．
溶融幅が狭いため，熱影響部が狭く，かつ熱変形の小さい高精度
な溶接が可能になる．このため，溶接後製品の再加工を行う工程が
省け，開発当初から自動車用トランスミッションギヤ等の高精度溶
接に本溶接法が適用されてきた．また集中した熱源を用いるため，
アーク溶接では困難な高融点材料や高熱伝導率をもった材料の溶接
も可能である．さらに特筆すべき点は，本溶接法が非常に制御性に
優れていることである．電子ビームは磁場により簡単に軌道が修正
できるため，図3･49に示した偏向コイルに流す電流を変化するこ
とにより，電子ビームの照射位置を高速で自在に制御することがで
きる．また，ビーム電流や焦点位置も同様に高速で制御が可能であ
る．
一方，大気中では電子ビームが散乱するため，本溶接法は真空雰
囲気で行う必要がある．これは，ガスと反応しやすい活性金属等の
溶接には適しているが，通常の材料では，真空チャンバのサイズに
より非溶接物の大きさが限定されたり，生産性の低下を招くため，
本溶接法の最大の欠点となっている．このため，初期においては小
型部品の精密溶接にもっぱら電子ビーム溶接が使われてきたが，
1970年代後半から巨大チャンバを備えた大出力溶接機が開発され，
極厚板の溶接が行われるようになった．厚板溶接における問題点
は，図3･50に示したようなポロシティ（porosity），凝固割れ
（solidification cracking），スパイク（spike）等の欠陥が発生しや
すくなることである．これらは，キーホールおよび溶融金属の挙動
と密接に関連して発生することが知られており，ビームオシレーシ
