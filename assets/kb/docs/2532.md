# 3・3・3 突合せ抵抗溶接

> **章**: 第3章 溶接・接合および切断
> **節**: 3・3 抵抗溶接法

a.アプセット溶接 アプセット溶接は直径10mm以下程度
の比較的小断面の棒材，線材，チェーンなどに適用される典型的突
合せ抵抗溶接法である．被溶接材を電極でクランプし，その端面ど
うしを加圧下で密着させ（または一定間隔を隔てた後，ある速度で
密着させ），その後通電し，端面が所定の温度に達した時点に電流
を遮断し（時には加圧力を急激に増して加圧し），溶接する（図3･
66)．
溶接法によって通電・遮断および加圧のシーケンスが異なる．標
準的なアプセット溶接は，突合せ面をあらかじめ一定の加圧力（鋼
材の場合は約40～60MPaを加えた状態）で電流（鋼材の場合は約
20～70A/mm ）を流し，接合部が適温に達したとき（あらかじめ
設定した所定の時間後）遮断し，加圧はさらに一定の冷却時間をお
いた後開放，終了する．加熱温度は鋼では1300℃以上を必要とし，
温度が低いと接合部に沿って空孔や酸化物が残り，溶接強さを低下
させる．加圧力は溶接部に変形を与え，接合面の空孔を消失させる
とともに接合部，熱影響部の結晶粒を微細化し，溶接部の性質を改
善するためのものである．この溶接では軸方向の定常時の温度分布
は放物線または余弦曲線のような分布となる．このため突合せ近傍
を溶融させることはその不安定さのため困難で，接合面での塑性変
形もさほど大きくない［図3･67(ａ)］．したがって，この標準的な
アプセット溶接は，溶接装置も簡明で操作が簡単という利点がある
が，接合面に酸化介在物が残存しやすく溶接継手の機械的性質が劣
化しやすいので，継手強さをあまり必要としない場合に用いられ
る．
接合面の温度分布を接合面近傍のみ急激に上昇させて溶接部に大
きな塑性流動を起こさせると，接合面での酸化介在物が押出され，

溶接継手の機械的性質は向上する．これを得る一つの方法として
は，図3･67(ｂ)のように被溶接材の突合せ端面に開先を設けて溶
接する．開先を設けることによってその部分の電流密度が高くな
り，局部的な温度上昇が得られる．他の方法としては，溶接開始時
には突合せ面に適切な初期間隔をもたせて電圧を印加しておき，そ
の後，突合せ面を移動，衝突させて溶接する．この方法では衝突時
に突合せ面での接触抵抗（集中抵抗）を利用するため，大電気入力
を必要とし，また溶接装置には高追従性を必要とする．
b.フラッシュ溶接 被接合材の一方を可動電極，他方を固定
電極にクランプし，電圧を加えながら両部材を接触させると，接合
面で局部的な短絡とアークが交互に発生する．この短絡とアークに
よって接触部が加熱され溶融飛散する，いわゆるフラッシュが発生
する．可動側の部材を前進させていく過程でフラッシュが至るとこ
ろで発生し，やがて接合面全体がほぼ均一な溶融層を形成する．こ
の状態で急速に移動し，アプセット加圧して接合部を加圧変形させ
ることにより接合する（図3･68)．溶接装置はアプセット溶接（図
3･66）に類似しているが，移動速度制御が加わる．溶接工程での加
圧力，移動量，電流波形の時間経過は図3･69に示す．予熱工程は
フラッシュの発生を容易にするもので，小断面の溶接では省略され
る場合もあるが，通常，突合せ面を加圧・接触させ連続または断続
通電するか，場合によってはこの時点で移動台を前・後進させ短
絡-フラッシュを繰返し交互に行って予熱する．フラッシュ工程は，
端面温度が比較的低い初期の段階では大粒の溶融金属が飛散する
が，次第に温度が上昇するとともに細かいフラッシュとなり溶融速
度が増すので移動速度も速くなる．この工程では端面表面に均一な
溶融層を形成させることが重要なので，二次無負荷電圧，電源回路
