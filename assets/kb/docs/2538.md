# 12・1・4 特殊用途鋼

> **章**: 第12章 工業材料とJIS 規格、外国の規格
> **節**: 12・1 鉄鋼材料

JIS に規定されている特殊用途鋼は12・1・3項のステンレス鋼および耐熱鋼を除くと工具鋼(tool steel)、ばね鋼(spring steel)、快削鋼(free cutting steel)、軸受鋼(bearing steel)の4鋼種である。工具鋼はSK×,SKH××,SKS××,SKD××と表記される。最初のS はsteelを、次のK は工具を示している。H,S,D はそれぞれhigh speed,special,dieを、最後の××は種類を示している。ばね鋼、快削鋼および軸受鋼はSUP××,SUM××およびSUJ×と表記され、最初のS はsteelを、次のU はuse(特殊用途)を、続くP,M およびJ はそれぞれspring,machin-ability,journalを、最後の××ないし×は種類を示している。これらJIS の抜粋を表12・31~12・34に示す。外国規格との比較はあまりにも煩雑であり省略する。展伸用アルミニウム合金(wrought aluminum alloys)は、主要添加元素によって、以下のように1000系から8000系まで分類される。JIS に登録されている展伸用アルミニウム合金の化学成分を表12・35、代表的な実用合金の標準的引張性質を表12・36に示す。1000系合金は純アルミニウムであり、微量のSiとFeを含有し、国内で生産される展伸材の約20%を占める。強度は最も低いが、耐食性、加工性、溶接性、導電性、熱伝導性などに優れるため、反射板、装飾品、導電材、器物、印刷版などに使用されている。代表的な合金は1050(純度99.5%以上)、1100(純度99.0%以上)である。2000系合金は、Cuを主要元素として含有する。切削性に優れ、静的強度、疲労強度および高温強度にも優れるが、耐食性は悪い。そのため、耐食性が要求される用途では、耐食性に優れた純アルミニウムやアルミニウム合金で被覆して用いる場合がある。代表的な合金は2014,2017(ジュラルミン)、2024(超ジュラルミン)、2219合金である。2014,2017,2024合金は航空機などの構造材として使用されるほか、鍛造材にも適用される。また、2219合金は表12・6
