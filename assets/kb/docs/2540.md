# 12・3・2 JIS 規格一覧

> **章**: 第12章 工業材料とJIS 規格,外国の規格
> **節**: 12・3 銅・銅合金

おもな銅・銅合金材料のJIS 規格として,伸銅品,ニッケル銅合金,銅・銅合金鋳物について,その一覧を表12・41に示した.a. 伸銅品種類番号伸銅品の種類は68合金種があり,その種類記号は合金番号に形状記号が付与されて示される.合金番号(alloy number)は,ASTM でも採用しているCDA(copperdevelopment association)の合金番号を参考とし,JIS では記号C(copper)と4けたの数字で以下のとおり表示する.1位2位3位4位5位××××C合金別種別を表す2位,3位,4位の数字はCDA 合金番号と同じ番号が使用される.5位の数字はCDA と同一組成の場合は0で表示し,同じ系統の合金であるが組成が若干異なる合金,すなわち改良合金(modified alloy)の場合は制定順に1~9の番号が付けられている.合金の系統は第2位の数字,すなわち主要添加元素によって表12・42のように7分類される.参考までに,各合金系統における合金名称を表12・42に合わせて示した.形状・等級を示す記号を表12・41に合わせて示したが,いずれも各製品形状の英字表記の頭文字である.b. 伸銅品質別記号(temper)伸銅品の機械的性質の違いは,JIS ではそれぞれの規格の中で軟質(記号O,オーと読む.以下同様),1/4硬質(1/4H,H : hardの略,以下同様),1/2硬質(1/2H),硬質(H),さらに特殊なものとして,製造のまま(F: fabri-cationの略),特硬質(EH : extra hard),ばね質(SH : springhard),特ばね質(ESH : extra spring hard)などと規定され,種類記号-の後に記号で示す.JIS 規格に使用されるおもな質別とその特徴を概説すると次のようである.(1) 軟質引張強さが最も低く,冷間加工した材料を再結晶温度以上で焼なましして得る.深絞り加工や冷間鍛造を行う場合に適している.なお,管のみに指定される軽軟質(OL:O-light)材は軟質(O)材と同等の引張強さであり,同材に軽い加工を施して得る.(2) 1/4硬質軟質の材料を軽く冷間加工して硬くしたもので,光沢があり,曲げたり,軽く絞り加工をするのに適している.(3) 1/2硬質1/4硬質をさらに強く加工したもので,打抜き性が良く,曲げ加工などをした後,ある程度強度を必要とするものなどに適する.(4) 硬質・特硬質・ばね質・特ばね質強い冷間加工を行って硬くしたもので,高い強度を必要とする場合に用いられる.特に,ばねなどのように弾性を必要とするものには,特硬質,ばね2―285表12・44 伸銅品JIS 規格と関連外国規格の比較例(JIS H 3100の一部)関連外国規格アメリカイギリスドイツ国際規格CDAASTMBSDINDINISO規格番号H 3100:2000―――1787:197317670-1:19831337:1980規格名称銅及び銅合金の板及び条――銅,銅合金板銅銅および銅合金の板および条,機械的性質展伸材用銅C1020P,-RC10200C10200(B 152:97)C103OF-CuOF-CuCu-OF材料記号C1100P,-R,-PPC11000C11000(B 152:97)C101C102Ecu58,Ecu57Ecu58,Ecu57Cu-ETPC1201P,-RC12000C12000(B 152:97)C106SW-CuSW-CuCu-DLP質,特ばね質が適している.c. ニッケル銅合金ニッケル銅合金は63~70%Ni,少量以下のFe,Mn,C,S,Si,残部Cuからなる合金であり,JIS 規格ではニッケルおよびニッケル合金として包括された.d. 銅・銅合金鋳物銅・銅合金鋳物の記号は以下のようにCACと3けたの数字で表し,銅合金連続鋳造鋳物はその末尾にCを付けて表す.銅・銅合金鋳物のJIS 記号と種類,合金系を表12・43に取りまとめた.記号1位2位3位×××C
