# 12・3・3 JIS 規格と関連外国規格

> **章**: 第12章 工業材料とJIS 規格、外国の規格
> **節**: 12・3 銅・銅合金

銅・銅合金に関わる外国規格のおもなものとして，イギリス国家
規格BS（British Standard），ドイツ国家規格DIN（Duetsche In-
dustrie- Normen），アメリカ材料試験協会（ASTM:American
Society for Testing and Materials）規格，銅センター（CDA:
Copper Developmet Association）規格，国際（ISO:Interna-
tional Organization for Standardization）規格がある．これら外
国規格とJIS規格との関連の一例として，伸銅品の銅および銅合
金の板および条の一部について表12･44に示した．全ぼうについて
は，日本規格協会から出版されている『JISハンドブック非鉄編』
を参照されたい．なお，JIS規格とこれに対応する諸外国規格とに
おいては，細部において，例えば化学成分範囲，引張強さ範囲，硬
さ規定等，わずかに異なる場合があるので，製品を外国規格に基づ
いて製造する場合には，該当する外国規格を細部にわたって確認す
ることが好ましい．
