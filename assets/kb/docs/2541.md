# 12・3・3 JIS 規格と関連外国規格

> **章**: 第12章 工業材料とJIS 規格,外国の規格
> **節**: 12・3 銅・銅合金

CAC銅・銅合金に関わる外国規格のおもなものとして,イギリス国家規格BS(British Standard),ドイツ国家規格DIN(Duetsche In-dustrie-Normen),アメリカ材料試験協会(ASTM : AmericanSociety for Testing and Materials)規格,銅センター(CDA :Copper Developmet Association)規格,国際(ISO: Interna-tional Organization for Standardization)規格がある.これら外国規格とJIS 規格との関連の一例として,伸銅品の銅および銅合金の板および条の一部について表12・44に示した.全ぼうについては,日本規格協会から出版されている『JIS ハンドブック非鉄編』を参照されたい.なお,JIS 規格とこれに対応する諸外国規格とにおいては,細部において,例えば化学成分範囲,引張強さ範囲,硬さ規定等,わずかに異なる場合があるので,製品を外国規格に基づいて製造する場合には,該当する外国規格を細部にわたって確認することが好ましい.文献(1) 日本規格協会編,JIS ハンドブック,鉄鋼I,(2004),1460,日本規格協会.(2) 日本規格協会編,金属材料データブック(改訂5版),(2000),日本規格協会.(3) 文献(1)の1444ページ.(4) Deutsche Normen,DIN 17006:1999および17007:1999.(5) ISO TR 4949:1989,Steel names based on letter symbols.(6) 文献(1)の843-1111ページ.(7) 日本規格協会編,JIS ハンドブック,鉄鋼II,(2005),21,34,36,日本規格協会.(8) 文献(7)の886,887,895ページ.(9) 文献(7)の401,402,407,408,431,432,883ページ.(10) Annual Book of ASTM Standards, Practice for NumberingMetals and Alloys(UNS),E 527-83(1997),754,ASTM.(11) 軽金属協会編, アルミニウムハンドブック(第5版), (1994), 30,軽金属協会.(12) 軽金属学会編,アルミニウムの組織と性質,(1991),518,軽金属学会.
