# 3・5・2 ろう付け設計

> **章**: 第3章 溶接・接合および切断
> **節**: 3・5 ろう付け

ろう付け設計は，ろうの特徴を最大限に生かすように工夫されな
ければならない．すなわち，(１)ろうのぬれによる毛管現象を利用
し十分な継手強度を確保する，(２)機械的強度および信頼性を保
つ，(３)耐食性その他必要な特性を具備できる，などの条件を考慮
して継手形状（joint shape）が選択される．このためには使用母
材とろうの選定が最も重要であるが，通常の生産工程では母材は決
まっているので，それに合致したろうと継手形状を設計することに
なる．ろう付け継手（brazed joint ）は基本的には重ね継手と突合
せ継手に分類できる（図3･77) ．通常，ろう付けする母材（base
metal ）に比べてろうの強度が低く，また伸びが不足するので，突
合せ継手の強度信頼性は低い．そのため，重ねしろをもつ継手形状
が一般的に使用され，単純な突合せやT継手のように，ろう単独
で継手強度を確保しようとする設計はほとんど行われない．このた
め，突合せ的な継手の場合は図3･77に示すように，スカーフ継手
や突合せ重ね継手などを用いる工夫がなされる．
