# 3・5・3 はんだ付け

> **章**: 第3章 溶接・接合および切断
> **節**: 3・5 ろう付け

はんだ付け（soldering）は，電気・電子機器実装に使用される
プリント基板上の高密度多点一括微細接合技術と，その他の産業で
使用されるはんだ付けの2種類に大別して考えなければならない．
前者では継手形状が年々微細微小化しており，材料とプロセスの革
新が続いている．図3･80にパッケージの小型微小化の推移を模式
的に示す ．IC（integrated circuit ）パッケージの小型化に伴い，
リード間隔とはんだ付けピッチが狭くなるとともに，QFP（quad
flat package）からBGA（ball grid array）やCSP（chip size
package）への変更は，従来のはんだ付けからはんだボールを用い
たバンプ実装（bump assembly）への変更となるなど，はんだ付
けそのものの形態とプロセスが大きく変化してきている．
後者で最も大きかった用途としての自動車用ラジエータなど銅製
品へのはんだ付けが，材料のアルミニウム化の進展とともにほとん
ど皆無となってきているので，本項では産業規模としてはんだ使用
のほとんどを占める電気・電子機器のはんだ付け（マイクロソルダ
リング）について記述する．
はんだ（solder ）とろうは材料の融点で区分され，液相線温度
450℃以下の溶加材をはんだという ．主たるはんだ合金系として
はZn基（Zn-Al ，Zn-Al -Cu，融点約380℃)，Au基，Pb基，Sn
基，In基，Bi基などがあり，主要組成と融点範囲を図3･81に示
す ．プリント基板実装の主要はんだ付け工法は，こてを利用す
るマニュアルはんだ付け，溶融はんだを用いるフローはんだ付け，
ソルダペーストを印刷塗布して接合するリフローはんだ付けなどで
ある．
はんだ付けの要点は均一温度上昇である．プリント基板実装のよ
うに，熱容量の異なる部品が混載している基板では，はんだ付け継
手部の温度が均一に上昇しないため，熱容量の大きい部品近傍では
温度が低く，微小部品電極では温度上昇が速くなる．最も温度が上
昇しにくい部分では未接合が生じ，逆に，温度上昇の速い部分では
高温長時間加熱となり，メタライズ層の過剰溶解，金属間化合物の
過剰成長など継手信頼性を低下させる現象が起こる．このため，一
般的な適正操業条件範囲は図3･82となる ．
これまでに，プリント基板実装に多用されてきたはんだのほとん
どはSn-Pb共晶系（Pbを36～40%含有，Sn-Pb共晶温度183℃）
であったが，環境問題への対応から有害物質Pbを使用しない鉛フ
リーはんだ付けに移行している．図3･81から融点的にSn-Pb共晶
代替となりうるのはSn基はんだとなる．表3･17に，これまで検
討されてきてほぼ実用化レベルと判断されているいくつかの鉛フリ
ーはんだ（lead- free solder ）を融点別に定義分類した ．鉛フリ
ーはんだの基本はSn-3.5Agをベースとした合金であり，Sn-
3～4Ag-0.5～1Cu系がはん用として利用されている．Sn-Pb共
晶並の融点はSn- Zn- BiおよびSn-Zn- 性が極めて劣ることと，銅との拡散反応性が高く界面に金属間化合
物が厚く成長するなど解明すべき問題は多い．Sn-58Bi系は融点
が低すぎることと機械的信頼性が乏しいので特殊用途である．
Sn- Ag- Cu系鉛フリーはんだとそのはんだ付けのおもな特徴を
まとめると，以下のようである．(１)ぬれ性がSn- Pb共晶よりは
劣る，(２)融点が高いために操業温度が高くなる，(３)鉛フリーは
んだの各種金属との反応性が高いために溶融はんだ浴槽等機器の損
傷に注意を要する，(４)鉛フリーはんだ自体は熱・機械的および電
