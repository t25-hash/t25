# 1・1・1 加工工程一般

> **章**: 第1章 加工学一般
> **節**: 1・1 加工学の体系

加工学を大分類すると,鋳造,溶接・接合および切断,塑性加工,切削加工,砥粒加工,特殊加工,表面処理,三次元造型技術,超精密・超微細加工技術などの各種加工法ならびに加工機械,加工測定,生産システムなどの技術から成り立っている.他方,機械の製造工程から加工を分類すると,成形・造型工程,切断・結合工程,除去工程,仕上工程などに分けられる.これらの工程を経て機械部品が加工され,組立工程を経て機械製品が完成する.一般的な機械製造工程を示すと図1・1のようになる.加工学においては,競争力の強化,持続可能な社会の実現といった観点から,加工学に課せられた課題である加工精度・品質の向上,加工能率・生産性の向上,省エネルギーの達成,環境に及ぼす負荷の低減さらには加工コストの削減を目指して,加工方法,加工機器・機械と工具および測定法の新たな開発と改良が行われる.製品の開発・設計,生産設計,生産技術の立場で加工学を応用するという観点からは,加工学の基本を理解し,各種加工法の特徴,利点と欠点を考慮したうえで,機械製造のために必要な加工工程を総合し計画する工程設計(process planning)を行うとともに,必要な製造設備,生産システムの計画や開発が行われる.
