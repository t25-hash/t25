# 1・1・2 成形工程に用いる加工法

> **章**: 第1章 加工学一般
> **節**: 1・1 加工学の体系

機械製造の立場からの成形工程には、鋳造、塑性加工、粉末成形、プラスチック成形などが含まれる。ほとんどが金型、モールドなどの型工具を用いた形状変化による成形であり、総称して素形材加工と呼ぶ。これらの成形加工では形状を作るだけでなく、品質変化を目的とすることもある。専門工場で成形加工された素形材が機械工作工場に供給され、切断・結合工程、除去工程、仕上工程などを経て、部品として完成されることが多い。最近では工程短縮のため異なる加工法を組合わせた工程、専門工場と機械工場を組合わせた工場形態も増えている。製品設計、工程計画の立場からは、この素形材の材料品質、寸法・形状許容差、熱処理特性、残留応力、繊維流れも含めた材料諸特性、材料歩留り率、型製作期間を含めた供給時期などが、工程選択の基準となる。成形工程を大別すると、板、棒、線などの素材を製造する一次成形工程(primary forming process)と、それらを用いて最終形状(ネットシェイプ)やそれに近い形(ニアネットシェイプ)にする二次成形工程(secondary forming process)に分けられる。一次成形工程は、鉄鋼や非鉄金属の製造工場において比較的大規模な生産設備により連続生産されるもので、インゴットの鋳造、連続鋳造、圧延、鍛造、押出し、引抜き、ロール成形などの加工法がある。二次成形工程の鋳造方法としては、各種砂型鋳造、ダイカスト、チクソモールディングなどがある。塑性加工方法としては、せん断、曲げ、紋り、深絞り、ロール成形、スピニング、熱間・冷間・図1・1 一般的な機械製造工程の流れ3―1温間鍛造、転造などの加工法がある。プラスチックの成形方法には、鋳造方法に似た射出成形や塑性加工法に似たブロー成形などがある。二次加工は一次成形に比べて複雑な製品を高精度に製造することが特徴であり、最近では切削などの仕上加工を省くネットシェイプ加工が目標となっている。
