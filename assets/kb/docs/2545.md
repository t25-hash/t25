# 1・1・4 除去工程に用いる加工法

> **章**: 第1章 加工学一般
> **節**: 1・1 加工学の体系

除去加工(removal process)とは、成形・造型工程、切断・結合工程を経た素材に対して、切削加工、砥粒加工(研削加工、遊離砥粒加工)、特殊加工、超精密加工などによって、設計で指定された機械部品の最終仕上がり寸法、形状、表面粗さを得るために、荒取り除去、仕上除去の加工を行うもので、二次成形工程のプレス加工も含めて、一般に機械加工と呼ばれるものである。切削加工(cutting process)を分類すると、単刃加工(旋削、形削り、立て削り、平削り)、穴あけ・中ぐり加工(ドリル加工、中ぐり加工、ガンドリル加工、リーマ加工)、フライス加工(正面フライス加工、エンドミル加工)、特殊形状創成加工(ブローチ加工、歯切り加工、シェービング加工、ねじ切り)、その他となる。研削加工(grinding process)を分類すると、円筒研削、内面研削、心なし研削、平面研削、歯車研削、ねじ研削、工具研削、複合形状研削、ホーニング、超仕上げ、研磨布紙加工などになる。遊離砥粒加工(abrasive process)を分類すると、ラッピング、ポリシング、バレル加工、バフ加工、超音波加工、複合研磨加工などになる。特殊加工(nontraditional machining)を分類すると、光・物理加工(レーザ加工、イオンビーム加工、プラズマ加工、電子ビーム加工、反応性イオンエッチング加工)、電気加工(放電加工、電解加工)、化学加工(フォトリソグラフィー、電鋳・エレクトロフォーミング、化学研磨加工)、噴射加工(砥粒噴射加工、液体ジェット加工)、その他となる。超精密加工(ultraprecision machin-ing process)を分類すると、超精密切削加工、超精密研削加工、超精密研磨加工、その他となる。工程計画の観点からは、加工すべき部品の製作数量(ロットサイズ)、全体寸法・形状、加工面寸法・形状、材質、材料硬度、削りしろ、仕上がり寸法精度、形状精度、表面粗さなどに応じて、使用する加工法、工作機械・工具、加工条件が決定される。多くの成形工程、一部の切断・結合工程などを担当する専門工場や、除去工程でも少品種多量生産を行う工場では、それぞれの基本工程のうちから最適な加工法を選んで一連の工程を計画し、固定された工程によって連続的に生産を行う。他方、機械加工の大半を占める多品種少量生産工場や部品加工専門工場では、あらかじめ除去工程のうちから必要と考えられる各種工程とそれらに必要な工作機械・工具を標準的に選び出して用意し、部品や製品の製造計画に応じてそのつど与えられた条件下で最適と考えられる加工法を選び、組合わせて工程計画を立てる。その工場の製造計画にとって特殊な加工法、工程と考えられるもの、あるいは保有する技術や生産コストの面で不利と考えられるものについては、他の専門工場に依頼加工することになる。したがって一般3―2の機械加工工場では、標準となる工作機械、工具、ジグ、取付具、測定機器などのハードウェア、標準加工時間、機械・工具管理などのソフトウェア技術、必要な作業者・組織を確立して初めて適正な加工計画が立てられることになる。
