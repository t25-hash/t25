c .ガス切断条件と切断面品質 切断板厚は，特殊な切断法を

ガス切断面の品質について日本溶接協会規格WES2801：1980
では，上縁の溶けなど7項目について，表3･22に示すような等級
評価基準を与えている．
d.酸素および燃料ガス（fuel gas ) 酸素は圧縮気体または液
体として市販されており，その純度は99.5 %以上と規定されてい
るが，ガス切断の速度および切断面の品質は切断酸素の純度に大き
く影響される．
燃料ガスは，酸素と混合して予熱炎を作る．燃料ガスにはアセチ
レン，LPG（liquefied petroleum gas ），天然ガス，水素などが用
いられている．

手動吹管，自動吹管がある．

f .自動切断機 切断能率，精度向上のために各種の自動切断
機（thermal cutting machine）が用いられる．自動化の最も進ん
だものとしてNC（numerical control ）切断機，ロボット切断機
などが挙げられる．
g.パウダ切断 切断反応部に外部から微細鉄粉を連続的に供
給し，その酸化反応熱およびフラックス作用により，切断材を溶融
除去して行うガス切断をパウダ切断（powder cutting）と呼び，
鋳鉄，高合金鋼，非鉄金属など通常のガス切断が困難な材料に適用
される．
h.水中切断 ガス切断は切断吹管の構造を一部変えることに
より水中切断（underwater cutting）にも適用（燃料ガス：水素）
できる．図3･87は水中切断に使用される切断火口の構造を示した
ものである．
しかし，現在ではこの切断法が適用されることは少なく，アーク
切断法に置換えられている．
