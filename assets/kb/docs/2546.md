# 3・6・3 プラズマ切断（plasma arc cutting）

> **章**: 第3章 溶接・接合および切断
> **節**: 3・6 熱切断および火炎加工

アーク熱で金属を局部的に溶融して切断する方法をアーク切断あるが，現在多く使われているのはプラズマ切断である．
周囲から冷却されるとサーマルピンチ効果により細く絞られ，また水冷ノズルとガス流（作動ガス）によるピンチ効果で得られる高するもので，金属はもちろん非金属にも適用できる．

複雑な形状の手動切断も容易に行えるため薄板切断に適用されえ設備費も比較的安価で，ステンレス鋼やアルミニウム合金も切断（arc cutting）と呼び，表3･23に示したようにいくつかの方法がa.プラズマ切断 高温で電離状態のアークプラズマは，そのその温度が大きく上昇する．
