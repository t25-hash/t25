# 1・1・5 仕上工程に用いる加工法

> **章**: 第1章 加工学一般
> **節**: 1・1 加工学の体系

一般に仕上工程は、部品の寸法・形状精度や表面粗さを設計の許容範囲に収めることを目的とした部品精度達成のための加工と、部品表面の耐食性、耐摩耗性、外観などを維持する目的の表面加工に大別することができる。前者の部品精度達成のための加工には、機械加工、電気化学的加工、手仕上げによる加工がある。機械加工は、切削加工によるものと研削加工、研磨加工などの砥粒加工によるものがある。近年の切削加工精度は著しく向上しており、従来の研削・研磨加工を必要とせずに、切削加工プロセスのみにより高能率に粗加工から仕上工程までを実施することが多い。また定盤や精密案内面に対しては、熟練者の手仕上げによるきさげ加工(scraping)が行われる。後者の表面加工においては、対象とする加工部品に要求される機能に応じて電気めっき、溶融めっき、拡散めっき、蒸着めっき、溶射などの金属被膜処理、セラミックコーティングやDLC(dia-mond like coating)などの非金属被膜処理、さらには浸炭、窒化処理、高周波焼入れ、炎焼入れなどの表面硬化処理が施される。このほか、部品の外観上の美観の維持や耐食性維持の目的で、各種の塗装が行われる場合がある。
