# 3・6・3 プラズマ切断（plasma arc cutting）

> **章**: 第3章 溶接・接合および切断
> **節**: 3・6 熱切断および火炎加工

アーク熱で金属を局部的に溶融して切断する方法をアーク切断
（arc cutting）と呼び，表3･23に示したようにいくつかの方法が
あるが，現在多く使われているのはプラズマ切断である．
a.プラズマ切断 高温で電離状態のアークプラズマは，その
周囲から冷却されるとサーマルピンチ効果により細く絞られ，また
その温度が大きく上昇する．プラズマ切断は図3･88に示すように，
水冷ノズルとガス流（作動ガス）によるピンチ効果で得られる高
温，高速（30000K，l000m/ s程度）のプラズマ気流により切断
するもので，金属はもちろん非金属にも適用できる．
図3･88において(ａ)をプラズマアーク式（あるいはトランスフ
ァアーク式），(ｂ)をプラズマジェット式と呼び，前者は金属の切
断一般に，後者は薄い金属・非金属の切断に用いられる．作動ガス
の種類は対象金属によって選ばれるが，表3･24にその例を，図3･
89 にはプラズマ切断とガス切断およびレーザ切断の比較を示
す．複雑な形状の手動切断も容易に行えるため薄板切断に適用され
るほか，数mm以下の板厚ではガス切断より高速切断が可能なう
え設備費も比較的安価で，ステンレス鋼やアルミニウム合金も切断
可能であるため，薄板のみならず中厚板でも適用が広がっている．
