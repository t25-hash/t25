# 1・2・1 生産設計と工程設計

> **章**: 第1章 加工学一般
> **節**: 1・2 加工工程の計画法

製品の加工工程の計画プロセスは,生産対象製品の加工コスト,加工時間,加工精度,加工品質などを決定するうえで,非常に重要なプロセスである.製品の観点から見た加工工程の計画プロセスは,以下の二つに大別することができる.一つは,製品の生産設計(production design)プロセスであり,ここでは,VA(val-ue analysis,価値分析)などの解析手法を用いて,製品の性能,加工コストおよび加工時間の観点から,適切な形状,寸法,材質,公差,表面性状などを設計する.もう一つは,製品の工程設計(process planning)プロセスであり,ここでは,加工対象製品の形状,寸法,材質,公差,表面性状などの設計情報に基づいて,適切な加工法,加工設備の設計または選択を行うとともに,加工作業に必要な作業手順,設備の制御データの作成などを行う.製品の加工工程の計画プロセスでは,設計者が行う製品設計と生産設計が終了し,図面が出図された後,生産技術者による工程設計が行われてきた.しかし,1990年代に入り,製品の設計,解析および加工の分野において,コンピュータ技術を利用したCAD/図1・2 工程設計プロセスCAE/CAM(computer aided design/computer aided engineer-ing/computer aided manufacturing)技術が発達,普及してきたことにより,製品の設計から生産設計,工程設計における製品データの交換および共有が可能になってきた.このため,三次元形状モデル,製品モデル,ディジタルモックアップ,各種の解析およびシミュレーションプログラム,PDM(product data management,製品データ管理)技術などを活用して,製品設計の初期段階から生産設計および工程設計を同時並行的に行うことにより,加工工程の情報を製品設計,生産設計にフィードバックしながら,適切な製品とその加工工程の設計を行うことが可能になっている.このようなプロセスはコンカレントエンジニアリング(concurrent engineer-ing)あるいはサイマルテニアスエンジニアリング(simultaneousengineering)と呼ばれ,製品の設計から生産に至る期間短縮およびコスト削減に大きく寄与している.例えば,製品設計および生産設計プロセスにおいては,鋳造およびプラスチック射出成形における型内部の材料の流動および凝固過程の解析,プレス加工における板材の変形解析および応力解析,切削および研削加工における切りくず生成プロセスのシミュレーション,工作機械の運動により生じる干渉解析,組立プロセスにおけるロボットの干渉解析,作業者の作業性のチェックなど,多数の解析プログラムが提供されており,CAD で作成した三次元製品モデルのデータに基づいて,その製品の加工工程で必要な確認および評価プロセスを,コンピュータ上でシミュレーションすることが可能となっている.
