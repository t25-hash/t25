# 3・6・4 特殊切断

> **章**: 第3章 溶接・接合および切断
> **節**: 3・6 熱切断および火炎加工

b.レーザ切断（laser cutting) 炭酸ガスレーザ，YAGレーう．
レーザはガス炎やプラズマアークに比べてさらにエネルギー密であり，切断による熱変形も少ない．
このため，鋼構造物業界では3･90 に炭素鋼で良好な切断速度領域を示す．
またYAGレーザわせた切断が行われる．
金属では薄いはく状のものから数mm程ス，布，皮，木材などの非金属に対してもせん孔切断することがで方法を電子ビーム切断（electron beam cutting）と呼ぶが，真空密切断など特殊な用途に限定される．
ザなどのレーザビーム熱を用いて切断する方法をレーザ切断とい度が高く，切断幅は数十 mから数百 mと細く，高速切断が可能大出力が得やすい炭酸ガスレーザ切断機が広く導入されている．
図はその波長が短くファイバ導光が可能なため，ロボットなどと組合度のものまで，精度の高いせん孔・切断が行え，宝石，セラミックき，極めて将来性の高い熱切断方法である．
