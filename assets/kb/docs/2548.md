# 3・6・5 火炎加工

> **章**: 第3章 溶接・接合および切断
> **節**: 3・6 熱切断および火炎加工

a.スカーフィング ガス切断の原理を応用して，鋼塊，鋼片
などの表面欠陥を欠陥の度合いに応じ，深さ数mm程度で切削的
に除去する方法をスカーフィング（scarfing）またはデシーミング
と呼ぶ．
b.ガウジング（gouging) ガス切断を応用したフレームガウ
ジング（oxygen gouging）とアークエアガウジング（arc-air
gouging）がある．後者は炭素アークで金属を溶融し，圧縮空気で
溶融金属を除去して行う切断方法であり，溶接部の裏はつり，仮付
け溶接の除去など厚板鋼構造物製作時に広く利用される．
c .線状加熱 ガス炎で鋼板表面を線状に加熱し，その周辺を
シャワー状の水で冷却するときに生じる微小な角変形を重畳するこ
とにより，鋼板を任意の形状にわん曲成形する方法を線状加熱と呼
び，造船所などで広く使われている．溶接によって生じた薄板の凹
凸状の座屈変形やすみ肉溶接部の折れ曲がり変形の矯正等にも広く
用いられている．近年，成形自動化に際し，投入熱量を一定に保つ
ため電磁誘導加熱も行われている．
d.その他 そのほかの火炎加工には，低温応力緩和法，炎焼
入れ，炎軟化，炎強化，炎さび取り，炎ひずみ取りなど，数多くの
加熱加工処理の方法がある．
