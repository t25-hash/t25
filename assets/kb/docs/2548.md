# 1・2・2 工程設計の手順

> **章**: 第1章 加工学一般
> **節**: 1・2 加工工程の計画法

工程設計では、加工対象製品の設計データ(製品モデルまたは図面)に基づいて、その加工工程の設計、すなわち加工部位とその加工法、加工法のシーケンスを設計する。また、すべての製品の加工作業を分割あるいはグループ化して、生産システム内の生産設備に割り当てる。さらに、作業設計を行い、加工作業の実行に必要となる詳細な加工データ、ジグおよび工具のデータを作成する。この工程設計プロセスは、対象とする加工方法、すなわち、鋳造、鍛造、プレス加工などの成形・造形工程、溶接、組付けなどの組立工程、切削、研削などの機械加工工程、仕上げおよび調整工程などで異なるが、いずれも部品または組立品の形状、寸法、材質、公差、表面性状、組立状態などの設計データから加工すべき部位を認識し、その適切な加工法、生産設備、ジグ、工具を設計し、生産設備の制御データ、作業者に対する加工指示書を作成する。以下では、NC(numerical control)制御などの加工プロセスの自動化が進んでいるため、工程設計技術が最も体系化されている機械加工工程の設計プロセスを考える。機械加工の工程設計プロセスをまとめると、図1・2のようになる。図1・2は、IDEF0(ICAM Definition 0)に基づいて記述したものであり、各ボックスが設計のアクティビティを表す。また、左側からの入力が右側の出力に変換される。上からの矢印は参照情報、評価基準などのコントロール情報、下からの矢印はリソースの情報を表す。第1段階では、加工対象製品すべての特徴を認識し、分析する。製品の特徴は、以下の情報を含む。(1) 製品全体の概略形状および寸法概略形状および寸法は、使用する加工設備の容量および加工設備に取付ける際のジグの選択に重要である。また、薄肉形状、板状形状のように剛性が低い形状の場合には、加工法、ジグおよび加工条件などについて詳細な検討が必要になる。(2) 製品に含まれる加工部位の形状および寸法素形材を製品に変換するための加工部位(加工すべき局所的な形状および寸法)は、加工に必要な工具および加工設備の運動軸構成を設計するうえで必要になる。(3) 製品全体および加工部位の要求精度製品全体および加工部位の要求精度、すなわち仕上面粗さ、幾何公差、寸法公差、はめあいなどは、加工法、加工を実行する順序、必要な加工設備、ジグおよび工具の精度などを決定するうえで必要になる。機械加工工程の場合、(2)の加工部位は比較的単純な形状をしており、これらは加工特徴(加工フィーチャ、machining feature)と呼ばれている。加工特徴は、フライス加工、旋削加工、ドリル加工などを行う際に単位となる形状であり、その代表例を図1・3に示す。第2段階では、それぞれの加工部位の加工法を選択し、加工する順序を設定する。この部分を、狭義には工程設計と呼ぶ。また、加工部位に対する加工作業を実行することができる加工設備の候補を選択する。このとき、自動車や家電製品の部品のような少品種多量生産方式の場合と、工作機械や建設機械の部品のような多品種少量生産方式の場合では、選択の基準が異なる。すなわち、少品種多量生産の場合は、製品の加工工程に基づいて加工を行う生産システムの設備配置を計画、設計することが多いため、ここでは加工設備、ジグ、工具を計画あるいは設計するための要求仕様を決定することが必要となり、この要求仕様に基づいて、生産システムの計画および設計が行われる。これに対して、多品種少量生産の場合は、既存の生産システムで生産されることが多いため、生産システムに設置されている加工設備、ジグ、工具の中から選択することが必要である。すなわち、現状の生産システムのもつ加工能力、加工設備、人的資源の制約のもとで、適切な加工設備を選択することが求められる。第3段階の設備割当てでは、すべての製品の加工作業を分割またはグループ化し、適当な加工設備に割り当てる。このとき、加工法だけではなく、要求される加工精度、製品の概略の寸法および形状図1・3 加工特徴の代表例3―3なども考慮して割り当てることが必要になる。ここでも、生産方式が異なると、加工作業の分割および加工設備への割当ての基準が異なってくる。第4段階では、各加工設備に割り当てられた加工部位の加工を実行するための詳細な加工データを設計する。この部分を、狭義には作業設計と呼ぶ。ここでは、ジグおよび工具の設計または選択を行い、加工条件などの設計を行う。また、これらのデータに基づいて、作業者に対する作業指示書の作成、NC工作機械の制御データの作成を行う。
