# 1・2・3 生産方式と工程設計

> **章**: 第1章 加工学一般
> **節**: 1・2 加工工程の計画法

工程設計は,生産システムにおける生産方式によりその設計基準および評価基準が異なる.生産方式は,少品種多量生産を対象とするライン生産方式と,中品種中量生産あるいは多品種少量生産を対象とするジョブショップ生産方式に大別することができる.a. ライン生産方式の場合ライン生産方式は,加工工程が同一または類似の製品品種をライン状に配列された加工設備を用いて加工する生産システムであり,加工対象製品はコンベヤなどのライン形式の搬送システムで搬送される.そのため,原則として,すべての製品品種が一連の同一の加工設備により加工される.自動車,家電製品の部品またはコンポーネントを加工するトランスファライン(TL: transfer line)あるいはフレキシブルトランスファライン(FTL: flexible transfer line)がこの方式の代表例になる.この生産方式の場合,生産ラインにおける単位期間当たりの製品の生産量から定まるサイクルタイム(cycle time),すなわち1個の製品がラインから作り出される時間が基準となる.このため,各加工設備における加工作業時間を,サイクルタイム以下で,なるべくサイクルタイムに近い時間に設定することが要求される.これを,ラインバランシング(line balancing)と呼ぶ.そのため,図および作業者の加工能力,加工機能および台数(人数)を適切に設計することが必要になる.また,第3段階においては,加工作業時間が,サイクルタイムの制約を満足するとともに,なるべく一定になるように,加工作業を分割し,各加工設備に割り当てることが必要になる.b. ジョブショップ生産方式の場合ジョブショップ生産方式は,加工工程が異なる製品品種を,適切に配置された加工設備を用いて加工する生産システムである.この方式では,加工対象製品の品種により使用する加工設備の種類およびその使用順序が異なるため,加工対象製品はAGV(automated guided vehicle,自動搬送装置),フォークリフトなどの間欠的でフレキシブルな搬送システムにより搬送される.工作機械や建設機械の部品を加工するFMS(flexible manufacturing system,フレキシブル生産システム)およびFMC(flexible manufacturing cell,フレキシブル生産セル),電機製品のフレキシブルな組立を行うセル型生産システムが,ジョブショップ生産方式の代表例である.ジョブショップ生産方式の場合には,加工設備の間の搬送および加工設備,ジグ,工具などの段取り作業が問題となる.そのため,図1・2における第2および第3段階において,なるべく多数の加工部位を同一の段取りで加工することができるように加工設備を選択するとともに,加工作業を加工設備に割り当てることが必要になる.すなわち,工程の集約が必要になる.また,加工設備の工具交換装置(ATC: automatic tool changer)に装着することができる工具の本数,あるいは工作物交換装置(APC: automatic palletchanger)の容量に制約があるため,これらの制約を考慮することが必要になる.
