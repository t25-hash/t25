# 1・2・4 生産コストと工程設計

> **章**: 第1章 加工学一般
> **節**: 1・2 における第2段階においては,生産ラインを構成する加工設備

生産システムの目的が,よい製品を安価にタイムリーに顧客に提供することであるため,工程設計プロセスでは,生産対象製品の機能,性能,品質を満足する製品の加工工程を設計するとともに,その生産時間(production time)および生産コスト(production3―4加工学・加工機器cost)の削減を考慮することが必要である.製品1個を生産するための生産時間の構成をまとめると図1・4のようになる.すなわち,生産時間は,加工時間,加工準備・後始末時間,およびその他の付加的時間からなる.加工時間は,正味加工時間とそのための段取り時間からなる.加工準備・後始末時間は材料,ジグ,工具などの生産資源の調達時間,加工後の後始末時間などを,付加的時間は製品の搬送・保管時間,作業の余裕時間および作業の習熟に必要な時間などを含んでいる.図1・4に示す生産時間の各要素は,生産を実施するまでに標準作業時間として標準化することが必要である.この標準時間に基づいて,製品の生産時間(工数)を見積もることができる.この標準時間のうち作業者が関与する時間については,同一または類似の加工作業を行っていく過程で時間の短縮を図ることができる.これを習熟効果と呼び,一般的には,次式に示すように,作業の繰返し回数とともに指数関数的に短縮されていくと考えられている.=+-=+-욘(1・1)ここで,,:個め,あるいは生産進行期間のときの加工時間,:1個め,あるいは生産開始時の加工時間,:最終的に習熟した際の加工時間,α:低減指数,τ:習熟時間定数である.製品1個を生産するための生産コストは,図1・5に示すように,基本的には生産時間と同じような構成をとる.ただし,生産コストの場合は,材料費が含まれる.また,加工費用,加工準備・後始末費用,付加的費用の中には,製品の設計,生産,管理に直接的または間接的に関わる人件費,生産システムの建物および設備の償却費,必要な消耗品の費用などが含まれる.生産コストを体系的に評価する手法の一つとして,ABC分析(ABC analysis )に基づく手法がある.この手法は,製品の生産作業にかかる直接経費および間接経費を必要な活動(アクティビティ)に基づいて分析,積算するものであり,加工作業に必要な標準時間と各作業に必要な生産資源(設備,人,消耗品など)の単位時間当たりにかかる費用に基づいて,生産コストを体系的に評価することができる.
