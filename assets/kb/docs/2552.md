# 2・1・1 溶融金属の性質

> **章**: 第2章 鋳造
> **節**: 2・1 鋳造の基礎

鋳造は高温で金属材料を溶解しなければならず，鍛造に比べてそ
の歴史が新しいと思われがちである．しかし，鍛造の素材も通常は
溶解されて作られることより，鋼を除いてはその歴史はほぼ同等で
ある．鋳造による成形技術が誕生したのは5000年以上前といわれ
ている．
鋳造とは，溶融（液体）金属を鋳型に流し込んで形を作る加工法
である．それには，まずは溶融金属の性質を理解することが重要で
ある．代表的な溶融金属の物性値を表2･1と表2･2に示す．これら
の値は文献(２)～(８)に基づいてまとめた値であるが，溶融金属の
性質は合金の組成だけでなく，溶解方法や雰囲気によりガス成分の
吸収などでかなり異なることがあるので注意を要する．これらの物
性値は大中の著書 によく取りまとめられているので参照された
い．
a.液相線温度（liquidus temperature) 純金属の場合にはそ
れぞれの特定の温度で固体から液体に変化するが，通常の合金の場
合には液体から，固体と液体が共存する温度域を経て，固体へと変
化する．この固体と液体が共存する領域を固液共存域といい，凝固
温度範囲（freezing range）ともいう．この凝固温度範囲の広さが
鋳造性，おもに湯流れ（溶湯の流動性）やひけ巣の発生に影響す
る．この点に関しては2･1･1･ d，2･1･2･a，2･1･4･b項で後述する．
b.密度 温度上昇とともに溶融金属の密度は減少する．鋳造
用合金の密度の正確な値の報告は少ないが，溶融合金の密度（den-
sity of molten alloys）は原子%による加成性が比較的よく成立す
ることが知られている ので，おおよその値は計算で求めること
ができる．
c .熱物性値 鋳造は凝固によって形を得る加工法であり，こ
れを理解するには熱物性値も重要である．熱物性値としては熱伝導
率，比熱，凝固潜熱，ふく射率などが問題になる．真の熱伝導率は
固体のときよりも低下する．しかし，液体の場合には熱対流または
溶質対流などの混合を伴うので物性値よりも大きくなり，見掛けの
値と真の値は単純には一致しないことが多い．

熱は通常，固体の比熱よりもいくぶん大きくなる．また凝固
黒鉛が晶出するかセメンタイトが晶出するかで，その値が大きく変

g.粘度 溶融金属の粘度は温度の低下とともに増大する（図
2･3)．一般には溶融金属の粘度は水よりも大きいと思われている
が，溶融金属は水に比べて密度が大きく，動粘度で比較すると水よ
りもはるかに小さくなる．このため，鋳造時の溶融金属の流れは乱
流であり，粘度の影響は少ない．しかし凝固が開始すると，固体と
液体の混合流れになり，粘度は急激に上昇する．この現象により溶
融金属の流れが途中で停止し，湯回り不良（misrun）を発生させ
h.ガスの溶解度 溶融金属は活性であり，ガスを吸収しやす
い．例えば，溶融鉄合金では酸素と水素，窒素が吸収されやすく，
特に問題になるガスは酸素である．このためにアルミニウムやシリ
コンなどの脱酸元素が添加される．脱酸とは鉄に溶解した酸素をア
ルミナやシリカのような固体の酸化物にすることで，ガス欠陥
（gas defects）を防止する手段である．窒素にも同様の手法はある
が，通常は水素や窒素を除くには真空脱ガスなどの各種の脱ガス処
理（degassing treatment）が実用化されている．
これに対してアルミニウム合金や銅合金ではおもに水素が問題に
