# 2・1・1 溶融金属の性質

> **章**: 第2章 鋳造
> **節**: 2・1 鋳造の基礎

鋳造は高温で金属材料を溶解しなければならず、鍛造に比べてその歴史が新しいと思われがちである。しかし、鍛造の素材も通常は溶解されて作られることより、鋼を除いてはその歴史はほぼ同等である。鋳造による成形技術が誕生したのは5000年以上前といわれている。鋳造とは、溶融(液体)金属を鋳型に流し込んで形を作る加工法である。それには、まずは溶融金属の性質を理解することが重要である。代表的な溶融金属の物性値を表2・1と表2・2に示す。これらの値は文献(2)~(8)に基づいてまとめた値であるが、溶融金属の性質は合金の組成だけでなく、溶解方法や雰囲気によりガス成分の吸収などでかなり異なることがあるので注意を要する。これらの物性値は大中の著書によく取りまとめられているので参照されたい。a. 液相線温度(liquidus temperature)純金属の場合にはそれぞれの特定の温度で固体から液体に変化するが、通常の合金の場合には液体から、固体と液体が共存する温度域を経て、固体へと変化する。この固体と液体が共存する領域を固液共存域といい、凝固温度範囲(freezing range)ともいう。この凝固温度範囲の広さが鋳造性、おもに湯流れ(溶湯の流動性)やひけ巣の発生に影響する。この点に関しては2・1・1・d,2・1・2・a,2・1・4・b項で後述する。b. 密度温度上昇とともに溶融金属の密度は減少する。c. 溶融金属のぬれ鋳物の製造には現在でも主として砂型が最も多く用いられている。それは安価なことに加えて、溶融金属に耐える耐熱性と、溶融金属にぬれないという特性を具備しているためである。多孔質の固体が(例えば砂型)液体にぬれると、吸取り紙に水がしみ込むように、毛細管現象(capillarity)により溶融金属は砂型に浸入してしまう。これでは鋳物ができないので、両者がぬれないことが砂型の条件になる。砂は多くの溶融金属にぬれないという特性を備えており、鋳型に使われてきたことがわかる。両者がぬれると、砂型の中に溶融金属が浸透し鋳造欠陥を作る。砂型鋳物ではこれを焼付き(burn on,sand burning,pene-tration)欠陥という。一般的に固体の金属と液体の金属はぬれやすい。しかし、金型で鋳物が製造できるのは、溶融金属と金型の間の温度差がぬれを妨げた結果である。金型の温度が上がると、ぬれて両者は接合してしまう。これも焼付き(seizure)という。g. 粘度溶融金属の粘度は温度の低下とともに増大する(図が、溶融金属は水に比べて密度が大きく、動粘度で比較すると水よりもはるかに小さくなる。このため、鋳造時の溶融金属の流れは乱流であり、粘度の影響は少ない。しかし凝固が開始すると、固体と液体の混合流れになり、粘度は急激に上昇する。この現象により溶融金属の流れが途中で停止し、湯回り不良(misrun)を発生させることがある。h. ガスの溶解度溶融金属は活性であり、ガスを吸収しやすい。例えば、溶融鉄合金では酸素と水素、窒素が吸収されやすく、特に問題になるガスは酸素である。このためにアルミニウムやシリコンなどの脱酸元素が添加される。脱酸とは鉄に溶解した酸素をアルミナやシリカのような固体の酸化物にすることで、ガス欠陥(gas defects)を防止する手段である。窒素にも同様の手法はあるが、通常は水素や窒素を除くには真空脱ガスなどの各種の脱ガス処理(degassing treatment)が実用化されている。これに対してアルミニウム合金や銅合金ではおもに水素が問題になる。図2・4に各種純金属の水素溶解度の温度依存性を示す。図中の矢印は凝固温度を示す。図2・4よりアルミニウムや銅では凝固により水素の溶解度が著しく減少することがわかる。これが鋳物に水素ガス欠陥が生成されやすい主原因である。i.
