図3･94セラミックスどうしおよびセラミックス-金属の接合方法
付け法は，非常に簡便で接合継手特性が最も優れた接合方法の一つ
であり，現在，セラミックスの接合技術の主流をなしている．しか
しながら，接合継手の耐熱性や接合欠陥（割れやボイド）の発生に
は問題があり，より厳しい環境下への適用や信頼性確保には留意す
る必要がある．
c .固相接合 拡散接合（diﬀusion welding）はセラミックス
どうしあるいはセラミックスと金属を直接接触させたまま加圧下で
加熱し，構成元素を拡散させて接合する方法である．その場焼結接
合（in-situ sintering bonding）法はセラミックスもしくはセラミ
ックスと金属の混合粉末を接合界面に挿入して，加圧力を加えなが
ら加熱し，その場で焼結させて接合する方法である．この方法に
は，ホットプレスや熱間等方加圧（HIP:hot isostatic pressing）
技術が適用される が，最近，パルス通電加圧焼結 を用いる
新たな接合技術が試みられている．
一方，摩擦圧接（friction bonding，friction welding）法は，高
速度で回転している被接合体を接触させ，その際に発生する摩擦熱
を利用してセラミックスと金属を接合する方法である．大気中で加
熱・加圧が比較的容易に行え，特に，接合表面を清浄化しつつ接合
界面を局部的に加熱することができる特徴を有する．界面でのセラミ
ックスと金属の反応を利用して接合が行われる．固相接合方法は，い
ずれも精密接合が可能で優れた接合継手特性を示すが，一般に接合
装置が大がかりで接合条件や前処理を高度に制御する必要がある．
d.融接 セラミックスの溶融接合については，電子ビームや
レーザビームなどの高エネルギー密度熱源を用いて試みられてい
る．AlOやZrOの融接では，通常，1223～1673K程度の予熱
が必要である ．また，非酸化物系セラミックスのように解
離・蒸発しやすいセラミックスには適用が困難である．
