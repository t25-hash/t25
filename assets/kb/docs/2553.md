# 2・1・2 鋳造における湯流れと凝固

> **章**: 第2章 鋳造

a.湯流れ（flowability) ここでは溶融金属を鋳造現場で用
いられる用語の湯と表現して話を進める．鋳型内での湯流れは，流
動中に湯から鋳型への熱伝導により温度が低下し，ついには凝固が
起こり流動を停止することがある．これを湯回り不良（misrun）
という．また，先の化学反応(2･2)(2･3)や流動中のガスの巻込みに
よりピンホール（pin hole）や，吹かれ（blow hole，gas hole）な
どのガス欠陥を生じることがある．これらの分野の学問に湯口方案
（gating design，gating system，gating）がある．詳細は専門
書 を参照されたい．
溶融金属の流動停止の機構は図2･5のようなモデルで説明されて
いる．図2･5(ａ)の純金属（凝固温度範囲は零）では流入口付近
の凝固で閉じることで，図2･5(ｃ)の凝固温度範囲の広い合金では
流動先端に固相が生成することで粘度が増大し，流動が停止すると
いわれている．これに対して，図2･5(ｂ)の凝固範囲の狭い合金で

いう］は鋳込温度，正確には過熱度（鋳込温度と液相線温度 の

b.凝固時間 鋳物の凝固は伝熱問題である．凝固終了までの
全放熱量は鋳物の体積（mm）に比例し，放熱速度は鋳物の表面
積（mm）と鋳型の熱拡散率に比例する．そこで単純に考えると，
鋳物の凝固時間はその体積を表面積で除した値（これをモジュラス
［modulus( mm)］という）に支配される．したがって，単純には
凝固時間は鋳物のモジュラス（形状）に比例することがわか
る．少し正確に記述すると，これに鋳型の熱拡散率がきい
てくる．各種の鋳物の凝固時間とモジュラスの関係を図2･8に示
す．図2･8は鋳物のおおよその凝固時間の推定には極めて有効であ
る．

る．しかし，通常われわれは1atm下で生活しており，圧力を一
定として温度と組成を変数に存在する相を示す．代表的なFe-C二
元合金状態図を図2･11に示す．ここで，炭素含有量が2mass%以
下を鋼，それ以上を鋳鉄という．

2・1・4鋳造における諸現象
鋳造加工が他の加工法と大きく異なる点は製品形状の自由度であ
ろう．この特徴は2･2･3項で示される中子の使用にある．中子を用
いることでエンジンブロックのような複雑形状品を一体で成形でき
る点に最大の特徴がある．しかし，溶融金属を出発点にするため，
寸法の安定性は他の加工法に比べて劣るといわざるを得ない．ま
た，一般の溶融金属は凝固時に体積収縮を伴う．この結果として製
品にひけ巣欠陥や，ピンホール，ガスホールと称するガス欠陥がで
きやすい．以下にこれらの点について簡単に記述する．詳細は専門
書 を参照いただきたい．
a.寸法変化 凝固の終了から室温までの間の冷却により，そ
の金属の熱膨張係数に見合った寸法変化（収縮）をする．このた
め，鋳物の模型はこの寸法変化を見込んで大きめに作製しなければ
ならない．これに使用する定規を鋳物尺［shrinkage rule，または
伸尺（contraction rule)］といっており，製品の寸法より0.8～
2.5%大きめに作製されている．また，冷却が均一に行われないと，
熱応力による変形が生じる．これらの結果として鋳造品の寸法精度
は他の加工法に比べて劣る．
b.ひけ巣（shrinkage，shrinkage cavity) 凝固時の収縮は，
鋳物にひけ巣と称する内部欠陥をもたらす．初期の鋳造技術におけ
