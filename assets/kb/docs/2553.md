# 2・1・2 鋳造における湯流れと凝固

> **章**: 第2章 鋳造
> **節**: 2・3 )。一般には溶融金属の粘度は水よりも大きいと思われている

a. 湯流れ(flowability)ここでは溶融金属を鋳造現場で用いられる用語の湯と表現して話を進める。鋳型内での湯流れは、流動中に湯から鋳型への熱伝導により温度が低下し、ついには凝固が起こり流動を停止することがある。これを湯回り不良(misrun)という。また、先の化学反応(2・2)(2・3)や流動中のガスの巻込みによりピンホール(pin hole)や、吹かれ(blow hole,gas hole)などのガス欠陥を生じることがある。これらの分野の学問に湯口方案(gating design,gating system,gating)がある。詳細は専門書を参照されたい。溶融金属の流動停止の機構は図2・5のようなモデルで説明されている。図2・5(a)の純金属(凝固温度範囲は零)では流入口付近の凝固で閉じることで、図2・5(c)の凝固温度範囲の広い合金では流動先端に固相が生成することで粘度が増大し、流動が停止するといわれている。これに対して、図2・5(b)の凝固範囲の狭い合金では両者の中間の機構で流動が停止するとされている。いずれの金属材料でも流動の停止は凝固が鍵であることには変わりはない。したがって、通常の流動長さ[流動性(fluidity)ともいう]は鋳込温度、正確には過熱度(鋳込温度と液相線温度の差)に依存する。鋳込温度以外にも、流動性は図2・5のように凝固3―7図2・6 Pb-Sn合金の流動性の組成、温度依存図2・7 湯口系における湯流れのX 線観察の例(デイビスら)機構の影響を強く受けるので、一般的には純金属や共晶組成の合金の流動性がよい(図2・6)。また、凝固温度の低い合金ほど伝熱の問題で流動性はよくなる。鋳型内の流動の研究は、コンピュータを用いたシミュレーション(simulation)の発達により近年著しい進歩を遂げた。その詳細は専門書や、最近の解説記事を参照されたい。これらのシミュレーションの基礎となった技術に湯流れの直接観察がある。図b. 凝固時間鋳物の凝固は伝熱問題である。凝固終了までの全放熱量は鋳物の体積(mm)に比例し、放熱速度は鋳物の表面積(mm)と鋳型の熱拡散率に比例する。そこで単純に考えると、鋳物の凝固時間はその体積を表面積で除した値(これをモジュラス[modulus (mm)]という)に支配される。したがって、単純には凝固時間は鋳物のモジュラス(形状)に比例することがわかる。少し正確に記述すると、これに鋳型の熱拡散率がきいてくる。各種の鋳物の凝固時間とモジュラスの関係を図2・8に示す。図2・8は鋳物のおおよその凝固時間の推定には極めて有効である。
