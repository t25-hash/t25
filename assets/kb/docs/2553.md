# 3・8・1 溶接施工管理一般

> **章**: 第3章 溶接・接合および切断
> **節**: 3・8 溶接施工管理

a.溶接施工管理の基本 溶接施工においては，構造物・製品
の用途および使用条件に応じて作成された設計図書（仕様書，図面
など）と適用基準に基づいて，所要の製造品質を，決められた納期
で能率よく，かつ安全に実現することが要求される．このために必
要なのが施工管理である．施工管理においては，品質管理が最も重
要であることはいうまでもないが，資材，設備，日程，配員，コス
ト，作業環境などの管理もそれに劣らず重要である．したがって，
施工管理に際しては，溶接工程のみならず，その前後の工程との関
連性の理解，工程全体における作業の流れの把握が不可欠である．
b.製造品質管理 溶接の製造品質管理の基本は，最終品質に
影響を与える種々の因子を明確にし，適確な計画を作成するととも
にそれらを施工時に十分に管理することといえる．図3･95は，溶
接施工において，溶接品質を保証するために管理しなければならな

である ．
.溶接施工要領の作成 溶接施工管理の計画において最も
重要なものが，最適な溶接施工要領の作成である．溶接施工要領
は，設計品質と工程能力を勘案しながら，図3･95に示した特性要
因図における各種要因，すなわち，
(１)溶接技術者の管理能力，溶接作業者の資格，技能レベルなど
の人的要因
(２)溶接機，ジグなどの設備，作業場所の環境
(３)使用する鋼材，溶接材料の種類，特性
(４)溶接方法，溶接条件などの溶接施工時の管理項目
(５)確認，記録などの作業管理項目
(６)溶接結果を評価する各種試験検査
などの要因を考慮して決定する必要がある．
溶接施工要領を決定するまでの過程と各種要因の相互関係を図
3･96に示す．要求される溶接構造物の品質に対して，設計品質と
しての設計図，仕様書などから鋼材を選定し，その鋼材に適した溶
接材料を選ぶ．溶接材料の選定に際しては，製造品質としての溶接
方法，溶接設備なども考慮すべきであることはいうまでもない．ま
た，品質管理方針を定めるとともに，製造品質に関係する溶接設備，
溶接要員，施工・作業管理に基づいて，溶接施工要領を決定する．
このとき，経済性，納期，安全性などの検討も忘れてはならない ．

.設備管理 設備計画の中で最も重要なものは，溶接機を
代表とする溶接設備である．溶接作業場が屋内なのか，屋外なの
か，固定設備として使われるのか，可搬設備として使われるのか，
る．また，使用する溶接機に応じて必要電源設備容量も変わってく
るので注意を要する．溶接機の結線においては，溶接用ケーブル，
アースケーブルの長さ，サイズによる電圧降下に留意しなければな
らない．
それぞれの場合により，同じ溶接量でも必要溶接機器の数量は異な
屋外作業の場合には，溶接機器に防雨対策が必要であるし，施工
時の防風設備も準備する必要がある．屋内作業，狭あい部作業では
溶接ヒュームなどの換気設備も考慮すべきである．
設備管理には，省力化のための溶接自動化，ロボット化設備，溶
接機器専用の運搬装置（門型移動架構，ビームなど）などの計画，
導入も含まれる．また，作業内容に適したポジショナやワーク反転
ジグなどの各種溶接用ジグの準備，適用も行うべきである．
c .溶接工程・コスト管理 一般的には，継手1m当たりの溶
接に要する時間（標準時間）を求め，それに継手長さを掛けて溶接
作業時間を見積もり，作業工程の計画に用いられている．標準時間
は継手板厚，脚長サイズ，溶接姿勢，溶接方法，アークタイム率な
どによって変わるので，それぞれの場合の標準時間をデータとして
もつか，基本の標準時間に係数を掛けて対応する方法がとられてい
る．
溶接にかかる費用は，表3･36のような構成で成り立っており，
コストは一般的に，それぞれの材料単価にその材料の使用量を掛け
たものと各工費単価に作業時間を掛けたものとの合計として求めら
れる．溶接工程・コストは，基本的には上記のようにして得た見積
り数値をもとに計画され，実施工において管理される ．
d.生産性の管理・向上 生産性を管理し，その向上を推進す
ることも，広い意味では施工管理の役割の一つといえる．溶接の生
産性向上を図るには，(１)溶接構造と継手設計，(２)溶接法・施工
方法などの固有生産技術としての溶接技術，(３)生産システムと生
産管理技術のそれぞれの面から検討する必要がある．基本的には，
溶着速度の増大，アークタイム率，稼働率の向上，継手長・開先断
面積の削減などの作業量の減少が生産性の向上に寄与する．図3･
98は溶接構造物の生産性向上基本概念の一例を示したものであ
る ．
溶接の生産性を定量的に把握する指標には，表3･37に示される
ようなものがあり，生産性の管理とその向上のための手段として用
いられている ．
