# 2・2・1 凝固と収縮

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

ほとんどの金属・合金は凝固時に収縮する．その収縮は，溶融段
階で液体収縮（liquid contraction）し，鋳込み後に鋳型内で凝固
収縮（solidification shrinkage），最後に鋳物が室温まで冷却する
間に固体収縮（solid shrinkage）する．図2･12 は鉄および各種
炭素鋼の溶融状態から室温までの容積変化を示す．これらの炭素鋼
の液体収縮は約1.6%/100℃で，収縮率の値は金属・合金の種類
によってあまり相違はないが，凝固収縮率は種類によって異なる．

a.湯口の設計 図2･14に鋳型各部の名称を示す．湯口系は，
受け口（pouring cup）で注湯された溶湯が湯口（downsprue），
湯道（runner），せき（ingate）を経て鋳型内へ流れ込むのを直接
規制するものであって，各所の設計は流れ学的な因子を考慮すると
ともに，滓やガスの巻込み，鋳型のすくわれ（scab），絞られ
（buckle）などの現象も考慮する．湯口系を設計する際にあらかじ
め以下のようなことを考慮しておく．
(１)鋳物は上部より下部のほうから組織が緻密となり，鋳巣や巻
込み滓などの欠陥が少ないので，仕上面や強度を必要とする
重要な面は下面とする．
(２)溶湯は下方から満たされる．密閉型部では，空気を圧縮する
ので鋳型の破損やピンホール生成の原因となる．したがっ
て，溶湯が最後に満たされるところにはガス抜き（gas
vent）を付ける．
(３)凝固時に発生するガス抜きはなるべく上向きに付ける．
(４)溶湯は型内を短時間で満たすようにし，溶湯の流れがなるべ
く衝突しないようにする．
(５)湯口の高さ，湯道，滓だめ（runner extension）などを工夫
して溶湯が静かに，しかも最後まで威勢よく流れるようにす

b.押湯の設計 押湯（riser，feeder）は鋳型内に流入した溶
融金属が凝固収縮して不足する分を補足するとともに鋳型内の溶湯
に圧力を加え，さらにこれによって溶湯の流れを促進する．さらに，
溶湯内に停滞しているガスや滓を排除する効果や凝固収縮に伴う溶
湯金属を補給する目的もある．押湯の設置は製品とは全く別な部分
であるから，効率のよい大きさや設置位置を考慮する必要がある．
押湯方法には，製品部に直接設置する(１)直下押湯（top riser），
側面から補給する(２)側面押湯（side riser），押湯の一部が製品部
にかかる(３)コンノール型（connor top gate）またはちょい掛け
押湯（lap gate），湯口を兼ねた(４)湯口兼用押湯および製品上部
の全面を押湯にする全面押湯がある．図2･16に各種押湯法を示す．
押湯の大きさは凝固収縮量によって定まるので，製品の体積とその
