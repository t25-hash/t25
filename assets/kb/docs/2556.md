# 2・2・1 凝固と収縮

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

ほとんどの金属・合金は凝固時に収縮する。その収縮は、溶融段階で液体収縮(liquid contraction)し、鋳込み後に鋳型内で凝固収縮(solidification shrinkage)、最後に鋳物が室温まで冷却する間に固体収縮(solid shrinkage)する。図2・12は鉄および各種炭素鋼の溶融状態から室温までの容積変化を示す。これらの炭素鋼の液体収縮は約1.6%/100°Cで、収縮率の値は金属・合金の種類によってあまり相違はないが、凝固収縮率は種類によって異なる。したがって凝固収縮の大きい材料では、鋳物体積を十分満たす注湯量でないと、鋳物欠陥の発生要因となる。例えば、図2・13(a)に示すように、鋳物体積と溶湯量が等量で、溶湯が補給されないとす図2・12各種炭素鋼の冷却過程における収縮量変化(a)(b)図2・13凝固収縮によるひけ巣や収縮空孔の発生3―9ると、鋳型壁より熱放出されて、凝固は等温線(isothermal line)に沿って進行するので、上部表面に大きな収縮管またはひけ巣が形成される。また、鋳物の最終凝固部では金属不足となり、凝固収縮により内部に二次収縮空孔(shrinkage void)が生じる。実際の鋳物製品には凝固収縮が一様な場所と厚肉部や交差部など凝固速度の異なる場所があるので、凝固の遅いところにさまざまな形状、大きさの収縮空孔が形成する。図2・13(b)のように、凝固の遅いところに冷し金(chill)を埋め込むと、凝固速度が一様となり、凝固収縮による欠陥発生を防止できる。あるいは薄肉部のような冷却速度が高い場所には保温性の高い鋳型材料を用いたり、高温の溶湯によって燃焼反応する自己発熱材料を鋳型に用いる方策がとられるが、凝固までの時間が浪費されたり、組織の粗大化が生じるので得策ではない。一方、凝固が急速であると発生するガスが放出されず、内部にピンホールとなって残留する。大きな収縮空孔は凝固速度を一様にすることで防止できるが、凝固末期においてデンドライト組織の樹枝間では液相が不足する場所で微細な空孔(micropo-rosity)が発生する。アルミニウム合金鋳物などでは局所的な凝固収縮によって表面にへこみが生じる外ひけ(depression)が起こることがある。鋳物の設計段階でこのような欠陥発生の危険を防止するには、十分な溶湯を補給する湯口系(gating system)を設けるとともに、凝固時間が遅い部分に外圧を付加しながら溶湯を補給する押湯を設けるなどの総合的な鋳造方案(casting plan)が必要となる。a. 湯口の設計図2・14に鋳型各部の名称を示す。湯口系は、受け口(pouring cup)で注湯された溶湯が湯口(downsprue)、湯道(runner)、せき(ingate)を経て鋳型内へ流れ込むのを直接規制するものであって、各所の設計は流れ学的な因子を考慮するとともに、滓やガスの巻込み、鋳型のすくわれ(scab)、絞られ(buckle)などの現象も考慮する。湯口系を設計する際にあらかじめ以下のようなことを考慮しておく。(1) 鋳物は上部より下部のほうから組織が緻密となり、鋳巣や巻込み滓などの欠陥が少ないので、仕上面や強度を必要とする重要な面は下面とする。(2) 溶湯は下方から満たされる。密閉型部では、空気を圧縮するので鋳型の破損やピンホール生成の原因となる。したがって、溶湯が最後に満たされるところにはガス抜き(gasvent)を付ける。(3) 凝固時に発生するガス抜きはなるべく上向きに付ける。(4) 溶湯は型内を短時間で満たすようにし、溶湯の流れがなるべく衝突しないようにする。(5) 湯口の高さ、湯道、滓だめ(runner extension)などを工夫して溶湯が静かに、しかも最後まで威勢よく流れるようにする。鋳型への湯流れは各せきからの入り方によって変わる。受け口か図2・14 押湯の設計押湯(riser,feeder)は鋳型内に流入した溶融金属が凝固収縮して不足する分を補足するとともに鋳型内の溶湯に圧力を加え、さらにこれによって溶湯の流れを促進する。さらに、溶湯内に停滞しているガスや滓を排除する効果や凝固収縮に伴う溶湯金属を補給する目的もある。押湯の設置は製品とは全く別な部分であるから、効率のよい大きさや設置位置を考慮する必要がある。押湯方法には、製品部に直接設置する(1)直下押湯(top gate)、湯口を兼ねた(4)湯口兼用押湯および製品上部の全面を押湯にする全面押湯がある。図2・16に各種押湯法を示す。押湯の大きさは凝固収縮量によって定まるので、製品の体積とその表面積から求める方法が提案されている。図2・17に、高抗張力鋳鉄について乾燥鋳型を用いた場合の押湯の大きさを決定する線図を示す。これらの押湯は鋳物全体の凝固収縮を補給するために設けられるが、製品形状によっては部分的に溶湯不足となり、ひけ巣が生じる場合がある。例えば、等肉厚であってもL字部や十字部では熱放出が遅れるので、直線方向の凝固収縮によってひけ巣が生じ図2・15
