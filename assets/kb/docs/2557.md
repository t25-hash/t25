# 2・2・2 分割面、天地

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

鋳物を作製するとき模型(pattern)が必要で、正確で品質の高い鋳物とするには模型が単なる形状のみでなく、鋳型造型に対し、種々の要素を考慮した設計をしなければならない。そのためには、機械設計図面を解読し、適正な鋳造方案を設定してから模型設計することが順当である。複雑な形状製品の模型はいくつかの部品に分割して作製し、組立てて一つの模型にして造型する。鋳型の造型作業では模型を取出すため、鋳型を上型と下型の二つに分割するみきり面(線)(partingplane,parting line)を定め、模型の分割面もこれに一致させる。みきり面はなるべく平面とし、模型の分割面もできる限り少ないほうが造型作業は容易となる。図2・19はスライドバルブを形状変更し、横込めすることによって分割面を減らした改善例である。これによって分割面に生じるバリが減少するので、バリ取り作業も減って生産能率が向上する。また、図2・20に示すような製品形状の場合でも、分割面を平面にすることで型合わせの際の食違いが少なくなるため、発生するバリが薄く、バリ取りも容易となる。ただし、分割面を減らす目的で縦型を横型にする場合、仕上面や強度を図2・17 乾燥鋳型における押湯の大きさの決定(a) /=53/40=1.75(b) /=60/40=2.25(c) /=72/40=3.24(d) /=77/40=3.70図2・18 溶湯流れの交差部における冷却速度の比較例(a) 改善前(b) 改善後図2・19分割面の数を減らす方法例(a) 改善前(b) 改善後図2・20 分割面を平面にする方法例(b) ガス抜き優先(a) 作業性優先図2・21作業性よりガス抜きを優先した天地を決める方法例必要とする重要な面は下面であるから、どちらを下型にするか天地を考慮する必要がある。また、中子を用いる場合、中子が安定するような天地を選ぶことや、中子からガスが容易に上へ抜けるようにすることも考慮する。図2・21に、造型作業性よりも孔内面でのガス欠陥の発生を防止することを優先した天地の例を示す。また、機械加工部の欠陥防止を優先するとガス抜きが不十分となるような形状では、鋳物品全体にピンホールができる場合があるので、天地の決め方に注意する。
