# 2・2・4 抜けこう配

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

造型作業において鋳型から模型を引き抜く際に、長い平行部をもつ形状は引き抜くことが困難で、砂型鋳造では鋳型を壊すおそれがある。また、金型鋳造やダイカスト鋳造であっても、鋳型から鋳物を取出しにくい。このような部分には抜けこう配(patterndraft)をつけることが望ましく、JIS 規格では表2・4や表2・5に示すような抜けこう配の許容値が定められている。また、ダイカスト鋳物においても、図2・23に示すように、合金の種類に適した抜けこう配を設ける。さらに、鋳物設計では、図2・24に示す例のように、どの部分を分割面とするかをあらかじめ確認し、その寸法差があっても差し支えないか検討しておく必要がある。あるいは寸法差が許されない場合は、図2・25のように、形状そのものを変更しなければならない。
