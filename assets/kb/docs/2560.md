# 2・2・5 縮みしろ、仕上げしろ

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

凝固収縮は鋳型の設計にも考慮しなければならない。すなわち、鋳型の寸法は製品寸法より収縮量だけ縮みしろ(shrinkage allow-ance)として大きく作る必要がある。凝固収縮量は鋳物の材質によって決まるが、縮みしろは形状や肉厚、鋳型材料などを考慮して定める。表2・6に各種鋳物の収縮量を示す。模型を作製するときに、この収縮量を補正値として目盛にした定規が鋳物尺(shrinkage rule)で、表2・7に示すような基準で作られている。機械部品としての鋳造品は、その主要部を機械加工する。したがって、加工部分には仕上げしろ(machining allowance,finishingallowance)を設けておく。仕上げしろは凝固収縮量のほかに、鋳物表面に硬いチル層が形成することを考慮に入れ若干大きくとるが、必要以上に大きくすると加工費が高くなるので、適正な値を設けておく必要がある。表2・8に各種合金の仕上げしろの基準値を示す。また鋳鉄の場合は、鋳造後の変形が予想されるので、形状・寸法により適正な仕上げしろとする。
