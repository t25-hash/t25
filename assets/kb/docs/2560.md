# 3・9・4 リベット接合の強度

> **章**: 第3章 溶接・接合および切断
> **節**: 3・9 リベット接合

リベットは一般にせん断荷重を伝達するよう設計する．1本当た
りの許容荷重は材質やサイズにより大きく異なり，結合する板厚の
面圧強度とバランスをとるよう，サイズを決める．おもなリベット
のせん断強さを表3･38 に示す．リベットは複数本を用いるのが
通常であり，リベットパターンによる分担荷重の不均一や不適切な
リベットピッチの適用は，疲労強度（fatigue strength）低下を招
くことがあるので，十分注意しなければならない．また，リベット
1本当たりの許容荷重だけでなく，結合される板部の強度について
も考慮が必要であり，びょう間座屈や板端部からの縁距離の確保な
どの検討も行う．
さらに，張力場として設計されている薄板外板の周囲をリベット
接合する場合は，リベットにせん断だけでなく引張荷重が加わるの
で，これに対する配慮も欠かせない．
金属構造に対する一般的なリベット接合部では，以上の強度要件
を満たす経験値として，下記が目安とされる（リベット径 ，孔
中心から外板端までの距離 ，外板板厚 とする）．
リベット径と板厚： /＝3
縁距離： ≧2
リベットピッチ： ≧4
≦8または24の小さいほう
千鳥間隔： ≧3
なお，複合材構造の場合には，一般により大きな値を必要とす
る．
