# 2・2・6 角、隅部

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

鋳物製品の形状で角・隅部や断面積に急変部では、押湯等による溶湯の補給がなされないとひけ巣が生じやすく、あるいは凝固収縮による鋳造応力(casting stress)が作用して、凝固後に残留応力となって存在する。さらに、この応力が大きくなるような形状ではき裂の発生する危険性がある。したがって、このような部分には丸み(曲率半径、)やこう配をつけることにより、鋳造応力の軽減を図る。図2・26にその基準を示す。こう配は図2・26中に示す計算式で求める。図2・27や図2・28に示すように、L字形やV 字形あるいはT 字形などの交差部も同様な形状にする。十字交差部はできるだけ避け、交差位置をずらしてT 字交差するようにする。
