# 2・2・7 最小肉厚

> **章**: 第2章 鋳造
> **節**: 2・2 鋳造品の設計

鋳型内を流れる溶湯は粘性が高く,流動損失が大きい(2・1・2項参照).薄肉部では湯回り不良や湯境いなどの欠陥が生じやすい.(a) 内側および外側に丸みをつける場合(b) 外側を尖らせる場合図2・26 肉厚変化部の形状(JIS B 0703:1962)(a) 薄肉部に厚肉部が交差する場合(b) 厚肉部に薄肉部が交差する場合図2・27 L字交差部の形状(JIS B 0703:1962)(a) 肉厚比が1.5以下の場合(b) 肉厚比が1.5を超え3以下の場合図2・28 T 字交差部の形状(JIS B 0703:1962)3―13したがって,薄肉部の厚さは,溶湯金属の種類や鋳型材料により異なるが,基準値以上の厚さとしなければならない.表2・9に各種金属・合金の最小肉厚(minimum wall thickness)の基準値を示す.構造設計上,さらに薄い肉厚を必要とする場合は,鋳型に自己発熱性材料を使用するか部分加熱するなど,鋳造方案を工夫しなければならない.
