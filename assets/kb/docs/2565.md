# 2・3・1 はじめに

> **章**: 第2章 鋳造
> **節**: 2・3 鋳造用模型

鋳造品の鋳型を造型するとき、使用する型を模型(pattern)という。古くは模型を木材で製作することが多かったので、模型は木型(wood pattern)と呼ばれてきたが、現在は金属や樹脂が多く用いられるようになり模型と呼ぶのが正しい。鋳物は模型をもとにして作られるので、模型の役割は大きい。模型を作るとき、その材質や種類は、一つの模型から作られる鋳物の製作個数、形状などとその鋳物工場における造型法によって決まる。模型は木材、金属などの材質により価格が大きく変わる。また、鋳造方案を十分考慮した加工精度の高いものほど模型の価格は高くなるが、このような模型を用いると鋳型の造型が容易なので造型費が低減される。したがって、鋳物1個当たりの価格について考えるべきで、鋳物の製作個数が多数であれば、模型に手をかけても造型費が低下するので、製品1個当たりの模型費はごくわずかとなり、1個当たりの鋳物の価格は安価となる。模型を作るときは、抜けこう配(pattern draft)、加工時の仕上げしろ、縮みしろ、分割位置および天地などを考慮しなければならない。これらについては2・2節を参照されたい。また、設計図を見て収縮量を加算して寸法を出すのは面倒であるから、あらかじめ寸法を何%か大きく作った物差しを用いる。この物差しを伸尺、または鋳物尺という。鋳物の材質・寸法・場所などによって違った伸尺を用いる。伸尺は鋳鉄用が8/1000~10/1000、アルミニウム合金用が10/1000~12/1000である。分割位置は、模型が鋳型から抜きやすいように模型を分割する位置で、みきり部となる鋳物外周の線をみきり線という。また、鋳型の分割面の外周もみきり線という。天地は、模型を上下に分けたとき、上を天、下を地と呼び、加工面など重要な面を下型に配置したり、中子の安定と中子のガスが上に抜けるよう模型の上、下型を決めることである。鋳物の角や隅部が鋭角であると材質的に欠陥が生じるので、この部分に丸みをつける。これを面取り(fillet)という。鋳物は冷却過程において、その冷却速度の遅速と鋳物形状により、あとから冷却するほうに引張応力が生じ、反りが発生する。その反り量を予想して、あらかじめ模型を逆に反らして作る場合がある。これを逆反りしろという。図2・32のように肉厚が異なる形状のものは、A 部の肉厚が厚いので、B 部よりA 部のほうがあとから冷却するため、破線のように反りが発生する。反りは鋳物の形状、肉厚、鋳造法などにより異なるので、経験に基づいて決められるが、一般には1mについて2~3mmが適当とされている。図2・33のような形状のものは、A 部の外周側より内周側のほうが遅れて冷却するので、破線のように変形する。この変形を防ぐため、模型にB 部のように捨て骨を取付けることがある。この捨て骨は製品に関係ないもので、鋳造後は切り取られる。鋳物の変形を防ぐため、あるいは湯回りをよくするため余分に骨を付けることを捨て骨という。
