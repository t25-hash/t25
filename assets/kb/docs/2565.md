# 3・1・4 管理システムにおける諸管理活動の役割とそれらの関

> **章**: 第3章 管理システム
> **節**: 3・1 生産システム管理の体系

連性
生産管理システムを効果的に運用するためには，さまざまな管理
活動の実施が必要である．ここでは，管理活動として
･生産計画の立案
･作業管理
･工程管理
･購買・外注
･在庫管理
･品質管理
･設備管理
･原価管理
を取上げ，生産管理システムにおけるそれぞれの役割を述べるとと
もに，それらがいかなる部門の活動として，また複数の部門がどの
ように連携して実施されるかを説明する．連携の仕方は企業によっ
て，言い換えればその企業が用いる管理システムによって異なる．
以下においては，今後製造業において一般的になると考えられる生
産スケジューリング中心の管理システムの利用を前提として述べる
ことにする．
a.生産計画の立案 基本的には顧客の注文に基づいて生産計
画が生産管理部門によって立案される．顧客は海外の販売代理店の
ような大口の発注を行うものから特殊な仕様の製品を少量発注する
小口のユーザまでさまざまなものがある．注文の内容は，製品の仕
様，数量，価格，納期がおもなものである．仕様によっては設計が
必要であり，この場合は販売部門と設計部門が顧客と折衝しなが
ら，仕様の概要と納期が決定される．
生産計画の主要な内容は設備や作業員の稼働計画つまり生産スケ
ジュールであり，これに基づいて顧客の要求する納期の実現性が検
討され，必要に応じて販売部門を通して納期や生産量の調整が顧客
との間で行われる．生産スケジュールの細部については製造部門の
要求を考慮して決定する．資材の調達に時間を要する場合には，生
産スケジュールの作成と並行して資材調達の可能性が検討され，生
産スケジュールが修正されることがある．つまり，その際には購買
部門が生産スケジュールの作成に係わる．
b.作業管理 生産計画，特に生産スケジュールの作成に当た
って，加工時間，段取時間のデータが必要であり，機械加工の場合
は，マシニングタイム，組立作業の場合は標準時間が用いられる．
これらのデータの作成，メンテナンスは，通常，生産技術部門によ
って行われる．BOMやM-BOMを新たに開発する場合は，設計
部門と生産技術部門が連携してそれに当たる必要が生じる．作業の
差立てや組立ラインの編成は製造現場つまり製造部門によって行わ
れ，残業の計画やシフト数増減は製造部門，生産管理部門，工場の
総務部門がその決定に関与する．
c .工程管理 生産ラインの設計を含むことがあるが，本稿で
は生産システムは所与としているため，工程管理の主要な活動とし
ては，生産スケジュールが予定どおり行われるように監督し，遅延
が生じた場合に必要な対策を講じる進捗管理が挙げられる．また，
ジグや工具が複数の工程で共通して利用される場合に，生産スケジ
ュールが支障なく実行されるように必要なジグ・工具をタイミング
良く各工程に提供することも工程管理の重要な活動である．通常，
工程管理は製造部門によって行われる．
d.購買・外注管理 生産スケジューリング中心の生産管理で
はスケジュールの作成は，生産管理部門が中心になって行われる
が，購買部門が加わって購買あるいは外注先に発注する資材の必要
量・時期が決められる．したがって，購買・外注管理の主要な仕事
は必要な種類の資材を，必要な量だけ，必要な時期に調達できるよ
うに，発注先の進捗状況を監督し，必要であれば督促を行うという
ものになる．このような監督・督促の業務は，通常，購買部門によ
って行われる．外注管理の中に外注先に対する生産技術や品質に関
する指導が含まれることがあるが，この種の業務は生産技術部門や
品質管理部門によって行われる．
e .在庫管理 製品が完成後ただちに出荷される場合，工場に
おいては製品の在庫管理は行われない．ただし，ここでいう在庫管
理は在庫水準に基づいて発注量や発注時期を決定する狭義の在庫管
理を指している．製品の在庫管理はむしろ流通経路中のものが中心
になり，製造業が管理する場合には生産管理部門が係わる必要があ
る．近年，サードパーティロジスティクスが普及し，流通経路中に
ある製品の現物管理は外部の業者に委託することが多くなってい
る．
したがって，製造業が行う前述したような狭義の意味での在庫管
理は，原材料の一部や常備品がその主要対象となっており，他方，
多くの資材は生産管理システムが保有するメカニズム（例えば，か
んばん方式）によって発注量や発注時期が決定される．通常，狭義
の在庫管理は購買部門の活動の一部として行われる．
f .品質管理 生産スケジューリング中心の生産管理システム
では，特定の顧客オーダに対して要求量どおりの生産が行われるた
め，不良品の発生を極限まで抑えねばならない．資材の受入検査を
受入工場側で実施する場合は，通常，品質管理部門が行う．工程間
検査や製品検査は作業員が行うことが一般的であり，工程に異常が
認められた場合，品質管理部門が中心になって，生産技術部門，製
造部門が協力して原因を明らかにし，その対策が講じられる．
g.設備管理 生産スケジューリング中心の生産管理システム
では，生産設備の使用日程はその稼働可能性を前提としたものであ
るため，設備故障による生産の遅延は許されない．設備管理は，通
常，設備の更新と保全の活動を意味しており，近年，生産管理にお
いては日常的な設備保全の重要性が増大している．特に，予防保全
が事前に定められた日程に従って計画的に行われ，保全日程が生産
日程の妨げにならないように，生産管理部門と設備管理担当部門と
の間で調整が行われる必要がある．
h.原価管理 現代の生産管理では全体最適の観点に立った収
益性の極大化が望まれる．そのためには，製造原価が的確かつスピ
ーディに把握でき，製造工程の収益に対する貢献の程度を示した
日々の付加価値の算定ができる必要がある．これは，原価管理の方
式の問題にとどまらず，生産管理のあり方に係わった問題であると
いえる．つまり，生産管理の仕組み自体が，購入資材の総額＝消費
資材の総額，出荷高＝販売高の関係を指向するものであることが必
要であり，それをより良く実現しうる生産管理システムのもとにす
べての管理活動が統合され，原価管理活動はすべての活動の成果を
適切かつ不偏的な財務的尺度によって評価する役割を担うものでな
ければならない．さらに，これらの評価値の変動が生産管理にフィ
ードバックされることが重要であり，そのためには生産管理部門，
原価管理部門を中心とした各部門の密接な連携と協調が望まれる．
