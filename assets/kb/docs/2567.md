# 3・2・2 生産計画の階層

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

図3･2に示すように，生産計画は一般に計画期間の長さに従っ
て，長期計画，中期計画，短期計画に分けられる．
長期計画は基本的には生産能力/設備計画と労働力計画に分けら
れ，次に続く中期計画や短期計画の制約条件としての役割をもつ．
中期計画は総合計画（aggregate planning）とも呼ばれ，雇用/
解雇，在庫量，ユーティリティ，設備変更，原材料調達等について
図3･2階層生産計画のフレームワーク
の計画にかかわる．
短期計画の主たる機能は，最終製品やエンドアイテムに対するマ
スタプロダクションスケジュール（MPS:master production
schedule，基準生産計画）を作成することである．この結果は，
種々のタイプの生産管理システム（ポンドドレンニングシステム，
プッシュシステム，プルシステム，ボトルネック中心システム）へ
の主要なインプット情報として用いられ，部品や組立品に対する短
期生産スケジュール，購入部品スケジュール，ショップフロアスケ
ジュール，労働力スケジュールが作成され，直接，生産管理のため
の計画として用いられる．
