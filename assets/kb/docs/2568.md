# 3・2・3 生産予測

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

すべての生産計画はまず生産予測（production forecasting）か
ら始まる．それは，これから将来に対する生産計画にかかわる決定
のほとんどが予測を基になされるからである．特に，製品に対して
将来の需要を予測することを需要予測（demand forecasting）と
呼び，生産計画の予測としては最も重要なものである．
予測は将来に対するものであるから，当然のことながら不確定要
素が介在するのは避けられない．したがって，この不確定要素をい
かにして精度良く予測するかが予測手法の有効性を決める鍵とな
る．
予測手法は大きく次の二つに分類される．
･定性的予測法（qualitative forecasting method）
･定量的予測法（quantitative forecasting method）
まず，定性的予測法は，数学モデルを用いる方法とは異なり，人
間がもつ専門的知識に基づいて将来のシナリオを展開する方法で，
代表的なものにデルファイ（Delphi ）法がある．
デルファイ法の例として，ある新しい技術が創生される時期を予
測する場合を考えてみよう．まず，専門家に新技術の時期を予測し
てもらう．これは通常，文書の形でなされるが，口頭でもよい．そ
して，回答を集めてそれを表の形に表し，もう一度専門家に返され
る．このプロセスを数回繰返して，一定のコンセンサスやある値に
収束したときに終了する．
デルファイ法は不確定性が非常に高い事柄の予測に対して有効で
あるといわれている．
これに対して，定量的予測法は，将来の現象がある数学モデルを
用いることによって過去の現象データから予測できるという前提の
もとになされる方法である．
定量的予測法は次の二つの基本モデルに分かれる．
(１)因果関係モデル 例えば，将来の製品需要量を種々のパ
ラメータ（例えば，金利，GNP成長率など）の関数から予測す
る．これには回帰分析から単純な一次式モデルを誘導し，この式か
ら製品需要量を推定する方法がある．
(２)時系列モデル 例えば，将来の製品需要量を過去の需要
量から予測する．これには多くの方法があり，移動平均法，指数平
滑法などは代表的な予測モデルである．
