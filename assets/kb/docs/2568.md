# 2・3・4 消失模型(lost pattern)

> **章**: 第2章 鋳造
> **節**: 2・3 鋳造用模型

模型を鋳型から抜き取らずに鋳造する方法がある。鋳型を模型から抜き取らないので鋳型を分割する必要がない。したがって、バリのない鋳物が得られる。これに使用する模型が消失模型で、再使用することはできない。a. 発泡ポリスチレン樹脂発泡ポリスチレン樹脂は消失模型鋳造法(evaporative pattern casting)の模型に使用されている。消失模型鋳造法をロストフォーム法(lost foam process)、フルモールド法(full mold process)ともいう。消失模型鋳造法は中子を有する鋳物に適用範囲が広がっており、エンジンのシリンダヘッド、マニホルドなどの模型が本法で製作されている。単品大物鋳物を製作するとき、発泡ポリスチレン樹脂ブロックを切削し、必要に応じてこれを接着して組立て、模型が作られている。近年、模型を三次元数値データに変換し、これをNC工作機に送り、発泡ポリスチレン樹脂ブロックを迅速に加工して消失模型鋳造法用の模型が作られている。これにより模型置き場をなくし、鋳造工場の合理化に貢献した例も見られる。一方、量産小物品の場合には金型内でポリスチレン樹脂を発泡して作る。発泡倍率によっても多少異なるが、成形後、経時変化を示し、模型の寸法は成形後何日経過しているかにより異なる。成形直後は約2/1000,7日後は約6/1000収縮があり、その後は緩やかに収縮するので、在庫管理、履歴管理が必要となる。発泡ポリスチレンは、ベンゼンとエチレンとの合成反応によって作ったポリスチレンに発泡剤・添加剤を配合し、ポリスチレンビーズを製造し、これを成形発泡させて作られる。この成形工程は大きく分けて2段階を経る。予備発泡工程で原料ビーズを目的の倍率、密度まで蒸気で膨らませる。次にこの予備発泡した粒子を、金型へ充てんし、同じく蒸気を利用し、ビーズどうしを融着させ、目的の形状に成形する工程となる。発泡倍率は鋳鉄用が50~60倍、アルミニウム合金用が40~50倍である。発泡ポリスチレン樹脂は、低い応力であっても変形してしまうので、模型表面に塗型をして使用している。この塗型が鋳型であり、砂はバックアップにすぎないといえる。したがって、塗型には強度、通気性などが要求される。最近、ポリスチレン樹脂以外の材料、例えばポリエチレン、ポリプロピレン樹脂を使った発泡体が研究されている。b. ろう模型(wax pattern)ろう模型は、インベストメント鋳造法(investment casting)、ロストワックス法(lost wax proc-ess)に用いられている。パラフィンワックス、カルナウバワックス、ビーズワックス(みつろう)にロジン(松脂)などの添加物を加え、軟化点を高めた成形性のよいものが、ろう模型として用いられている。溶融ろうを金型に射出してせきの付いた模型を作り、これを加熱したへらなどで湯口棒に複数つけ、ツリー状に組立て一つの鋳型で多数の製品が得られるようにしている。ろう模型としては、(1) 軟化点が高いこと3―18加工学・加工機器(2) 収縮や変形が少ないこと(3) 型離れがよいこと(4) 灰分が少ないことなどが必要である。c. 水溶性模型水溶性ユリア樹脂は、ロストワックス鋳造法で使用しているろう模型の代わりに使用できる。水溶性ユリア樹脂模型は粉末の樹脂を溶融点以下(393K)で金型に射出するため、収縮量はほとんど零に近く、したがって金型からの離型が容易で、模型の寸法精度がよく、変形を生じることがない。射出サイクルはろう模型の半分以下と速いので、生産性を高めることができる。ろう模型と比べると、強度が高いので運搬・取扱い・貯蔵も容易である。ロストワックス鋳造法では、セラミックシェル鋳型を形成した後、加熱してろう模型を溶出させるとき、ろうの膨張のためシェル鋳型を破壊する場合があるので、これを防ぐ方法が必要となる。しかし、水溶性ユリア樹脂模型は単に室温の水につけておくだけで樹脂が水に溶出し、シェルを傷めることなく模型を消失できる。
