# 3・2・4 生産能力/ 設備計画（capacity planning）

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

この計画は生産能力の大きさと種類を決定するためのものであ
る．これには当然，部品やコンポーネントや最終製品を作る機械が
中心であるが，これらの機械を支援するために必要な装置類，例え
ば，電力設備，空気/水/薬品供給装置，スペア在庫，マテリアルハ
ンドリングシステムなども計画される．
生産能力/設備計画の作成に当たって考慮すべき要素に次のもの
がある ．
a.製品のライフタイム 設置すべき設備のタイプや大きさ
は，生産する製品のライフタイムに依存する．近年，製品のライフ
タイムは年々短くなるばかりで，設備のライフタイムが製品のライ
フタイムよりも長い場合が多く，その場合，製品の生産中止によっ
て残存設備能力がむだになることがしばしばである．したがって，
設備は将来的に他製品への転用が可能なフレキシブル性を有するこ
とが望ましい．
b.部品の内作/ 購入 設備計画は部品を内作するか，購入
（外注も含む）するかによって大きく変わる．このことからも，部
品の内作/購入の決定は重要である．そのため，以下のような考慮
が必要になる．
(１)内作か購入かは単にコストだけでは決められない．アウト
ソーシングによって，これまで培われてきた生産技術や技能が失わ
れるだけでなく，使っていた設備がむだになってしまう．当然，こ
のような損失も考慮されなければならない．
(２)また，アウトソーシングを考える場合は，長期的展望に立
った判断がなされなければならない．生産を放棄して流通/サービ
スに移行することは必ずしも悪いことではないが，アウトソーシン
グ市場が将来どのように変化するかを長期的展望のもとに把握して
おくことが重要である．
(３)内作か購入かは，その製品全体を対象とする決定であれば
明解であるが，その製品を構成する一部の部品だけをアウトソーシ
ングする場合は面倒になる．それは，内作か購入かの決定の際に，
工場のワークロード（負荷）を考慮しなければならないからであ
る．どの製品のどの部品をアウトソーシングするかによって工場負
荷が変わり，工場負荷の平準化に変化が生じることになる．
c .価格 一般に，製品価格は生産能力/設備計画においてはあ
まり問題とされない．それは工場計画に対する影響が直接的でない
からである．しかし，生産能力を決定する場合には価格予測ないし
は経済分析が欠かせない．なぜならば，ある特定の設備構成が経済
的に妥当であるか否かは，その製品の売上高から求められる利益で
評価されるからである．
d.資金の時間的価値 設備投資や設備改善は，通常，資金の
時間的変化を考慮してなされる．したがって，利率や減価償却期間
は設備選択に大きな影響を与える．
e .アベイラビリティ いうまでもなく，生産能力は次式で与
えられるアベイラビリティ によって決まる．
＝＋
ただし， ＝平均故障間隔， ＝平均修理時間．
また，アベイラビリティは設備のある一定の信頼性を確保するた
めにも重要な尺度となる．
f .ボトルネック OPT（後述）の考え方にもあるように，ボ
トルネックになっている生産設備を増やせばスループット（生産
量）は増すが，非ボトルネックの生産設備を増やしてもスループッ
トは増えない．この理論からいえば，高速高性能の機械をボトルネ
ック工程に投入することは非常に魅力的であるが，現実にはボトル
ネック工程がそれほど明確に識別できるわけでもないし，また，生
産品種や需要状況によってもボトルネック工程は変化するので，問
題はそれほど単純ではない．
g.変動（ばらつき) この影響はボトルネックほど注目され
ていないが，機械の変動，すなわちばらつき，例えば，機械故障に
よる生産停止は，スループットにかなりの影響を与える．特に，ボ
トルネック工程の変動は，そのスループットに与える影響が大であ
る．
