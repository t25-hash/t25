# 2・4・2 金型鋳造法[metal mold casting process、重力金型

> **章**: 第2章 鋳造
> **節**: 2・4 鋳造法各論

鋳造法(gravity die casting process)]砂型と同様に、製品形状に相当する空げきをもつ金型内に溶湯を鋳造する鋳造方式である。鋳型に金型を用いるのはすべて金型鋳造であるが、一般に金型内に溶湯自体の重力を利用して鋳造する場合を重力金型鋳造という。金型鋳造には中子や型の一部に砂型を利用できる。同一金型で繰返し鋳造することができ、自動化、機械化がしやすく、生産性が高い。砂中子の使用により、複雑な形状でアンダカットの鋳物が製造できる。砂型に比べて寸法精度が高く鋳肌が滑らかで、また溶湯の冷却速度が速いため組織が微細となり、ピンホールが生成しにくく、機械的性質が優れている。鋳造方案は、基本的には砂型と同じであるが自由度が制限され、湯口、湯道、押湯などの位置は型の分割面に制約される。鋳物の形状、鋳造方案によっては指向性凝固させることが難しい場合が多く、金型を部分的に空冷あるいは水冷し、また塗型剤の材質や膜厚を変えることや断熱材の利用や砂型との組合せにより、冷却速度と温度こう配を調節する。金型鋳造はアルミニウム合金、鋳鉄、銅合金などに適用され、アルミニウム合金においては主鋳造法の一つである。鋳鉄や銅合金では金型寿命が短いためほとんど用いられない。また銅合金では収縮応力による鋳造割れが生じやすく、簡単な形状の黄銅およびアルミニウム青銅に限られている。金型は溶湯による熱衝撃と侵食に耐えられる材料で、鋳物の数、材料費、加工費などにより選ばれる。生産量の少ないものはFC200~FC300の普通鋳鉄、ミーハナイト鋳鉄、生産量の多いものは低合金鋳鉄、炭素鋼、ダイス鋼などが用いられる。アルミニウム合金鋳造における金型の寿命は、鋳鉄で2~4万ショット、SKD 6熱処理材で3~10万ショットである。金(a)(d)(b)(e)(c)図2・43 V プロセスの造型工程3―19型鋳造機は左右開きの簡単なものから、前後、左右、上下を組合わせた複雑なものまである。また、生産上、単に鋳造機を並列に設置したもの、ターンテーブル式にテーブル上に数機の金型を配置し、一周で鋳造、冷却、凝固、鋳造品取出し、塗型のサイクルとなるものがある。
