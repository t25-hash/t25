# 2・4・3 低圧鋳造法(low pressure casting process)

> **章**: 第2章 鋳造
> **節**: 2・4 鋳造法各論

低圧鋳造法はほとんどアルミニウム合金鋳物に適用されており,重力金型鋳造とほぼ同程度の利用度で,ダイカストを含むアルミニウム合金全鋳造品生産量のうち18~20%が低圧鋳造法による.最近マグネシウム合金への適用が図られている.図2・44に示す密閉された溶湯保持炉内の溶湯表面に0.01~0.06MPaの空気圧をかけて,溶湯中のストーク(給湯管)を通して溶湯を押し上げて上部の金型内に鋳造し低圧力下で凝固させ,湯口部の凝固が完了した時点で加圧を止め,ストーク内の溶湯をるつぼに戻す鋳造方式である.原則的には押湯がないので鋳造品の鋳造歩留りが高く,重力金型鋳造の40~70%に対して,低圧鋳造では90%以上である.砂中子を用いた薄肉の複雑形状の鋳造ができ,内部欠陥が少なく,寸法精度,鋳肌が優れている.加圧は3段階で,溶湯がストーク中を上昇し湯口にまで到達する段階を第一段加圧,さらに溶湯が金型キャビティ内を流動する段階を第二段加圧,溶湯の充てん完了後も加圧して凝固収縮に対する押湯効果をもつ段階が第三段加圧である.各段階で昇圧速度と加圧力の制御が重要である.ストークは一般には鋳鉄製が多く,溶湯侵食防止のため塗型されるが,溶湯の鉄汚染の問題で最近ではセラミックス製が使われている.ストークを通して溶湯供給が押湯を兼ねるため,湯口の位置や数が制約され,重力鋳造に比べて自由度が少ない.また,指向性凝固や湯回りを完全にするために金型温度が高くなる傾向があり,凝固完了までの時間がかかり,鋳造サイクルが長くなる.そのため複数個取りや鋳造機の自動化が進められている.近年,冷却速度を上昇させ鋳造サイクルを上げる方法として,不活性ガスで保持炉と金型内を満たした不活性雰囲気低圧鋳造法や,鋳型内を減圧して溶湯をストークから吸引して充てんする差圧吸引鋳造法が開発されている.
