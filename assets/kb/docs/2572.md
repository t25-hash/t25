# 2・4・4 高圧鋳造法[high pressure casting process,スクイー

> **章**: 第2章 鋳造
> **節**: 2・4 鋳造法各論

ズキャスティング法(squeeze casting process)]金型キャビティ内へ溶湯を0.05~0.1m/sの低速でゲートを通して充てんし,金型内の溶湯に機械的外力により40~200MPaの高圧力を凝固終了まで作用させる鋳造法で(図2・45),一般にはアルミニウム合金鋳物の鋳造に用いられる.ゲート断面積を大きくした低速充てんであるため,後述(2・4・5項)のダイカスト法に比べて金型内の空気やガスの巻込みが非常に少なく,また凝固過程中の加圧により凝固収縮やガスによるポロシティが少ない.高圧力により溶湯の金型への熱伝達が重力金型鋳造に比べて約4倍大きく,冷却速度が速くなり凝固組織が微細になる.機械的性質が非常に優れた鋳物の製造が可能である.ゲートから離れた厚肉部位には凝固終了時に高圧力が作用しきれない場合があり,2・4・5・b項で後述する局部加圧操作を行うことがある.高圧力を溶湯に作用させるた図2・44 低圧鋳造法3―20加工学・加工機器(a) とりべから湯をスリーブに移す(c) スリーブ支持台が上昇してスリーブと金型が合わさる(b) メインフレームが垂直位置まで移動図2・45 スクイーズキャスティング法型開き・製品押出し射出型締め・注湯(a) コールドチャンバダイカストマシンの作動状況射出型開き・製品押出し型締め(b) ホットチャンバダイカストマシンの作動状況図2・46 ダイカスト鋳造法め,金型の精度が厳しく,金型費と設備費が高い.
