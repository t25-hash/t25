# 2・4・9 遠心鋳造法(centrifugal casting process)

> **章**: 第2章 鋳造
> **節**: 2・4 鋳造法各論

回転している鋳型に溶湯を鋳造し、溶湯に遠心力を作用させて品質の高い鋳造品を安価に製造する鋳造法である。鋳造機の回転軸の方向により横型と縦型遠心鋳造法に大別され、遠心力の利用方法により、3種類に分類される(図2・51)。a. 真の遠心鋳造法溶湯を金型あるいは砂型製円筒鋳型に鋳造し、垂直軸あるいは水平軸の回りに回転すると、溶湯は鋳型の内壁に押し付けられて中空となる。水道管、シリンダライナなどの管状や環状の製品を中子を用いずに鋳造する方法である。肉厚は溶湯の鋳造量により調節される。遠心力により溶湯と鋳型との密着がよく、冷却速度が速く、ブローホールやひけ巣などの欠陥が極めて少ない。図2・49消失模型鋳造法の工程図2・50 消耗電極式アークスカル炉分類鋳造方法A横型遠心鋳造BC縦型遠心鋳造DE図2・53 気体加圧複合化鋳造法図2・51 遠心鋳造法(A,B,C:真の遠心鋳造法、D:半遠心鋳造、E:遠心加圧鋳造)b. 半遠心鋳造円板状の歯車、車輪、プーリなどを、その対称軸を回転軸として鋳型を回転しながら中央の湯口から鋳造する。溶湯は遠心力により中央の湯口から外周へ押出されて鋳型を満たす。製品全体が鋳型によって造られる。製品肉厚が薄くても押湯効果が働き、品質が高く、鋳造歩留りが高い。c. 遠心加圧鋳造不規則な形状の製品を中央の湯口から放射状の湯道をつけて配置し、湯口を回転軸として垂直軸の回りに鋳型を回転させ、鋳造圧力に遠心力を利用する鋳造方式である。半遠心鋳造法とは鋳物の軸を回転軸としない点が異なる。鋳物の形状は限定されず、回転時にバランスがとれ、振動が発生しないよう注意する。適正な回転数は振動、応力、溶湯の滴下、偏析を考慮して決定される。溶湯にかかる重力は、一般に遠心力による加速度を重力による加速度の倍数で表したナンバー()で表される。値は、中空チルドロールで175~200、大型シリンダライナで50~70、鋳鋼管は50~65、鋳鉄管は砂型で65~75、金型で30~50である。横型遠心鋳造では、鋳造された溶湯が十分な加速度状態に到達しないと雨降り状に落下するレイニング現象が起こる。溶湯への遠心力の作用は密度差の影響を生じ、密度の大きい構成成分は外側に、密度が小さい成分、ガス、非金属介在物は内側に偏析する傾向がある。ガスや介在物の内側への偏析は鋳物の清浄化に効果がある。
