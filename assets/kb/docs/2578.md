# 3・3・5 作業管理の評価指標

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理の評価指標にP，Q，C，D，S，Mがあることは前述し
た．また近年これらの評価項目に加えて，
CS:customer satisfaction（顧客満足度）
ES:employee satisfaction（従業員満足度）
SS:society satisfaction（社会的満足度）
ES:environment satisfaction(環境満足度）
という観点も重要となっている．
作業管理の目的は生産性，利益の追求であるが，決して，人間
性，安全性そして経済性という観点を忘れてはならない．
ここでは，主たる評価指標である生産性概念について示す．
生産性＝アウトプット/インプット ×100%
＝生産数量/投入資源
材料 設備・方法
人 エネルギー
土地・建物 時間
それぞれの投入資源に対して次のような生産性指標がある．作業
管理を行ううえで，対象職場・システムの特色を十分に把握し，ど
の指標を用いるかの判断が重要となる．
材料：歩留り，良品率
設備・方法：設備生産性
人：労働生産性
エネルギー：原単位
土地・建物：m当たり生産性
時間：時間当たり生産性
