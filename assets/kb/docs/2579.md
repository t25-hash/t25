# 3・3・6 IE手法

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理に活用することのできるIE手法の代表として，材料か
ら製品への時系列変化をとらえる工程分析，レイアウト図面上に
人・物の動線を描き運搬およびレイアウト改善の手掛かりを得る流
れ線図，D・I （distance×intensity）分析，人・機械の稼働状況
を把握するための連続稼働分析，ワークサンプリング，複式活動分
析，標準時間設定のための時間研究，人の動作に着目するサーブリ
ック，PTS（predetermined time standard）法等がある．いずれ
も，
稼働：付加価値を生じる作業
準稼働：稼働をサポートするが本来の作業ではない
非稼働：付加価値を生じない作業
という観点より分析を行い，非稼働は排除，準稼働はシステムの変
更で改善するという方策をとる．
