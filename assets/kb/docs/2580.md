# 3・3・7 作業管理の業務範囲の拡大

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理の対象である作業は，当初は有形な物の生産活動に直接
かかわる人の作業がおもな範囲であったが，生産工程の自動化，ロ
ボットの導入などによりマンマシンのインタフェースの管理，また
情報との一元化など，その範囲は広がっている．
生産の場は，顧客ニーズに対応した多品種少量・変種変量生産へ
の移行を余儀なくされており，それに伴う生産管理手法，在庫管理
手法も変化してきている．これらも作業管理のための要素となろ
う．

nique)，CPM（critical path method），品質の管理に各種管理図，
統計手法，FMEAなどが，また，製品設計にはVE（value engi-
neering）が用いられている．これらも作業管理の有効なツールと
なっている．
