# 2・5・1 鋳造品の仕上げ(cleaning)

> **章**: 第2章 鋳造
> **節**: 2・5 鋳造品の仕上げと補修

鋳物を鋳造後製品とする工程では,まず,鋳型から鋳物を取出す型ばらしを行い,型砂および中子砂を取り除く.ついで,湯口系,押湯,揚がりを切り取って鋳肌を整えるとともに,鋳ばりを取り,湯道またはせき跡を除去するための研磨を行う.そして,これらの作業後に最終的な検査を行い製品となる.また,製品に対しては,必要に応じて塗装・表面処理が施される.これらの作業を能率的に行うには,さまざまな設備,機械器具を利用する.以下に,これらの工程の具体的な内容について,簡単に述べる.a. strip)鋳物に付着している鋳型砂を除去する工程である.中子砂または鋳肌に接した砂,鋳物の隅部の砂は大部分が取れずに残るため,これらの砂を完全に取る必要がある.このために3mm以下の鋼粒または鋼線を短くしたワイヤなどを高速回転する羽根により製品に投射して砂を落とすショットブラスト法が一般的に使用される.鋼粒など投射するときの製品の挿入方法によって図2・57~2・59に示した方法に分けられる.図2・60は連続式のショットブラストの例で,前工程から砂落とし工程まで連続化・無人化できる設備である.このほかにも,新しいショットブラスト装置は開発されている.なお,大型製品を対象とするブラスト法としては,高圧水を鋳物表面にブラストパイプから噴射する,衛生的なハイドロブラスト法がある.この方法は,水力による清掃を行うため,圧力水の圧力が高いほど好都合とされている.c. 湯口切り(sprue shearing)・せき折り型ばらしした鋳物には,製品には不必要な湯口系,押湯,揚がりのようなものが付属していて,砂落とし前に取り除く場合と砂落とし後に取り除く場合がある.比較的大きな湯口やせきをつけた製品では,次の三つの方図2・59ハンガ式ショットブラスト図2・60 連続式ショットブラスト図2・61 破断道具の例法で取り除かれる.(1) 破断による方法(a) ハンマの衝撃によって破断する一般的な方法(b) プレスによる方法(c) 製品との間に工具を差し込み,これを油圧で押し広げて破断する方法(図2・61),また図2・62のように,ロボットを使用して破断する方法も行われている.(2) 切断による方法板状の砥石を付けたグラインダを人手,もしくはロボット,マニピュレータに持たせて切断する方法,あるいは図2・63に示すようなロボットに製品を持たせ,製品をグラインダに押し付けて切断する方法がある.(3) 溶断による方法簡単に破断できない場合,ガス切断で行う.ガス切断で最も一般的な酸素-アセチレンガスによる方法,アークと酸素を組合わせてアークを飛ばしながら酸素を流す方法,あるいは火炎に鉄粉を添加し,鉄粉の酸化熱を利用する方法がある.溶断による方法の中で,鋳鋼品のガス切断工程は手作業からロボットに置換えられて自動化されている.d. バリ取り(fettling,deburring)この工程は製品に付随3―25図2・62 ロボットによる破断作業の例図2・63 ロボットによる切断作業の例する不要な部分の除去と,外観上問題になるような部分の手直しであり,これらを総称してバリ取りという.この工程に使用される器具は砥石を電気あるいは圧縮空気の力で回転させるグラインダが一般的である.この使い方は大別して,グラインダを固定しておき製品を動かす方法と,逆に製品を固定してグラインダを移動させる方法がある.このほか,プレスの圧力を利用してバリを取るプレス式バリ取り機も小さくて数の多い製品に適用される.一方,量産品を対象として,テーブルに送り込まれた製品がシステム内で自動的に工具や反転機などを経て搬出コンベヤから完成されて出てくる完全自動化した専用機が使用されている.また,押湯切断,切断面仕上げ,およびバリ取りをするロボットも開発されている.e. 塗装・表面処理砂落としや不要物が除去された後,必要に応じて塗装が行われる.この塗装には,次の機械加工工程前などに一時的にさびの発生を避けるための塗装と,本格的な塗装がある.本格的な塗装の表面処理としてエポキシ樹脂等による樹脂塗装,亜鉛やクロムなどによる金属めっき,化学的な表面処理,ほうろう掛けなどがある.樹脂塗装は下水道管や化粧マンホールに行われている.また伝統工芸品の中に黒色あるいは茶色などを主体にした漆によって塗装されているものも多い.鋳鋼品,鋳鉄,アルミニウム,ダイカストなどの生産工程が全自動化される中で,仕上工程は量産工場では一部の工程を除いてロボットが導入され自動化されている.
