# 2・6・1 鉄鋳物

> **章**: 第2章 鋳造
> **節**: 2・6 鋳造品の種類

鉄鋳物は鋳鉄鋳物と鋳鋼鋳物に大別され，鋳鉄はオーステナイト
中の炭素の固溶限2.06mass%以上の炭素と黒鉛化のための1～3
mass%ケイ素を含むもの，鋳鋼は固溶限以下の炭素と他の合金元
素を含むものとされている．表2･13に鉄鋳物のJIS規格とその機
械的性質を示す．
鋳鉄は，晶出黒鉛の大きさ，形状，基地組織の調整によりさまざ
まな性質を得ることができる．ねずみ鋳鉄（gray cast iron，
FC100～FC350）は，片状黒鉛とパーライト基地をもち，減衰能，
耐摩耗性，被削性，耐熱衝撃性，耐食性に優れている．ねずみ鋳鉄
のうち引張強さが300N/ mm以上のものは強靱鋳鉄，金型などに
急冷し，表面部にセメンタイトの晶出した白鋳鉄とし，内部に黒鉛
晶出させた鋳鉄はチル鋳物と呼ばれる．球状黒鉛鋳鉄（spheroidal
graphite cast iron，FCD350-22～FCD800-2）はMg添加により
黒鉛を球状化するとともに，焼なまし処理，あるいは焼ならし・低
温焼戻し処理により基地組織をフェライトとパーライトの共存組
織，あるいは微細パーライトまたは焼戻しマルテンサイト組織とす
ることにより強度を高めたものである．さらに，基地を恒温変態に
よりベイナイト化し，高強度化したものがオーステンパ球状黒鉛鋳
鉄（austempered spheroidal graphite cast iron，FCDA900-
4～FCDA1400-1）である．

金元素を添加し，耐食性，耐熱性，低熱膨張性などを高めたものを
