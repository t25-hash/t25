# 2・6・1 鉄鋳物

> **章**: 第2章 鋳造
> **節**: 2・6 鋳造品の種類

鉄鋳物は鋳鉄鋳物と鋳鋼鋳物に大別され、鋳鉄はオーステナイト中の炭素の固溶限2.06mass%以上の炭素と黒鉛化のための1~3mass%ケイ素を含むもの、鋳鋼は固溶限以下の炭素と他の合金元素を含むものとされている。表2・13に鉄鋳物のJIS 規格とその機械的性質を示す。鋳鉄は、晶出黒鉛の大きさ、形状、基地組織の調整によりさまざまな性質を得ることができる。ねずみ鋳鉄(gray cast iron,FC100~FC350)は、片状黒鉛とパーライト基地をもち、減衰能、耐摩耗性、被削性、耐熱衝撃性、耐食性に優れている。ねずみ鋳鉄のうち引張強さが300N/mm以上のものは強靱鋳鉄、金型などに急冷し、表面部にセメンタイトの晶出した白鋳鉄とし、内部に黒鉛晶出させた鋳鉄はチル鋳物と呼ばれる。球状黒鉛鋳鉄(spheroidalgraphite cast iron,FCD 350-22~FCD 800-2)はMg 添加により黒鉛を球状化するとともに、焼なまし処理、あるいは焼ならし・低温焼戻し処理により基地組織をフェライトとパーライトの共存組織、あるいは微細パーライトまたは焼戻しマルテンサイト組織とすることにより強度を高めたものである。さらに、基地を恒温変態によりベイナイト化し、高強度化したものがオーステンパ球状黒鉛鋳鉄(austempered spheroidal graphite cast iron,FCDA 900-4~FCDA 1400-1)である。鋳鉄に5%以下の合金元素を加えたものを低合金鋳鉄、さらに合金元素を添加し、耐食性、耐熱性、低熱膨張性などを高めたものを高合金鋳鉄と呼ぶ。低合金鋳鉄にはパーライト系基地で耐摩耗性や一般機械用輸送機械用電気機械用その他用管継手用鋳鋼品23238855.427.44.512.7―銅・銅合金鋳物8676467.919.21.411.5―表2・13 鋳造品の特性と材質例特性部品例材質例耐滑り摩耗機械工作滑り面ねずみ鋳鉄、パーライト鋳鉄耐転がり摩耗、耐熱き裂ロールアダマイト鋳鋼、チル鋳鉄耐摩耗性耐滑り摩耗、耐熱ブレーキディスク低合金鋳鉄、高リン鋳鉄耐熱、潤滑性シリンダライナ低合金鋳鉄、高リン鋳鉄耐アブレシブ摩耗ボールミル、スプロケット高クロム鋳鉄淡水ポンプケーシング、バルブ、インペラねずみ鋳鉄、青銅海水ポンプケーシング、バルブ、インペラステンレス鋼鋳鋼、ねずみ鋳鉄耐食性土砂水ポンプケーシング、バルブ、インペラ低マンガン鋼鋳鋼、炭素鋼酸ポンプケーシング、バルブ、インペラステンレス鋼鋳鋼、ねずみ鋳鉄燃焼ガスバルブ、反応容器ステンレス鋼鋳鋼0~-45°Cバルブ炭素鋼鋳鋼低温靱性~-70°Cバルブ低温高圧用鋳鋼~-100°Cバルブ低温高圧用鋳鋼~-200°Cバルブオーステナイト系ステンレス鋼鋳鋼耐熱性の高いクロム系鋳鉄、マルテンサイト(ベイナイト)系基地で強度、耐摩耗性の高いニッケル系鋳鉄(ニハード鋳鉄、アシキュラ鋳鉄)などがある。高合金鋳鉄にレーデブライト系基地で耐熱性、耐摩耗性に優れた高クロム鋳鉄、Niを12mass%以上含みオーステナイト基地で耐熱性や耐食性に優れたオーステナイト鋳鉄(austenitic cast iron,FCA-NiMn137~FCA-Ni35,FCDA-NiMn137~FCDA-NiSiCr3552)、Siの多量添加によるフェライト基地をもち、耐食性、耐酸化性を著しく高めたフェライト系鋳鉄などがある。白鋳鉄を熱処理によって脱炭あるいは黒鉛化させ、延性を向上させた鋳鉄は可鍛鋳鉄(malleable cast iron)と呼ばれる。脱炭により破面が白い白心可鍛鋳鉄(FCMW 35-04~FCMW 45-07)、黒鉛化により破面が黒い黒心可鍛鋳鉄(FCMB 27-05~FCMB 35-10)、耐摩耗性と強度に優れたパーライト可鍛鋳鉄(FCMP45-06~FCMP70-02)がJIS 規格に規定されている。鋳鋼(steel castings)鋳物は前述のように用途も多様であり、耐摩耗性、耐食性、耐熱性、低温靱性などの要求特性に応じたさまざまな鋼種の鋳造品が製造されている。鋳鋼は、大きく炭素鋼鋳鋼、低合金鋳鋼、ステンレス鋳鋼、耐熱鋳鋼、高マンガン鋳鋼などの高合金鋳鋼に分類されているが、その合金設計や組織、特性は圧延材などと共通するものであり、ここでは主として鉄鋳物に要求される特性と材質例を表2・14に示すにとどめる。
