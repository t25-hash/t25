# 3・4・3 差異の検出と処置方法

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

a.差異の検出 生産計画と実績の差異は，注文生産において
は進度の遅れ，見込生産においては生産数量の不足という現象で表
面化するから，それぞれの進度あるいは生産数量を把握することに
よって差異を検出するのが普通である．差異を検出するに当たっ
て，どの位置で，どのくらいの間隔で，いかなる手段を用いて差異
を検出するかを設計しなければならない．検出する位置を多くし，
検出間隔を短くすると差異が生じてから早い段階で処置を執ること
ができるが，検出のための費用が増大する．逆に検出位置が少な
置・間隔・手段は，注文生産と見込生産との違い，工程の性質（工
程の数，合流型・分岐型工程），製造所要日数などによって異なり，
一概には決められないが，生産指示する工程，生産指示間隔および
生産指示段階に対応した形で差異の検出を行うのが普通である．
例えば，作業伝票を用いて生産指示する場合には，指示を受けた
工程あるいは職場は，作業伝票で指示された内容と生産実施状況と
を対比して差異の検出を行うことになる．また，作業伝票の代わり
に生産現場にコンピュータの端末器を配置し，端末器を通して生産
指示をする場合には，作業完了後，作業指示番号・作業者名・工程
名・完成数量などの情報を端末器に入力してセンタで迅速に差異を
検出することになる．
見込生産において，各工程が同期化している工程系列において
は，物の流れが速く仕掛期間も短いから，工程系列の最初の工程に
作業指示を行い，最後の工程で生産量を把握するだけでよい．その
ため，同期化している工程系列ごとに1箇所検出位置を設け，検出
間隔も必要に応じて1日複数回，生産状況を把握することも可能で
あり，差異の処置も迅速にできる．最近では，加工する物にバーコ
ードを付けて，検出する工程を通過するごとにバーレコーダで即座
に数量を数え上げ，予定した計画数量と比較して進捗を常時表示す
るシステムを用いている工場もある．他方，注文生産においては，
各工程が同期化しておらず，物の流れも錯綜しているから迅速に差
異を検出しようとすれば各工程ごとに検出位置を設けなければなら
ないが，経済性の面から検出する箇所は制約される．一般には，合
流工程，分岐工程，能力面での隘路工程などを選んで差異を検出す
る場合が多い．
b.差異の処置方法 生産計画と生産実績との差異を検出した
ときの処置は，差異の大きさや採算工程の性質にもよるが，次のい
ずれかの方法で処置することになる．
① 次の計画で差異を修正する．
② 差異を検出した時点で当該機の作業を再分配する．
①の方法は差異が僅少であるか生産能力の変更，あるいは材料や
部品の補給が困難な場合に行われ，一般に見込注文の場合に多く用
いられる．
②の方法は生産能力変更に柔軟性がある注文生産の場合に多く用
いられる．
具体的には次のような方法がとられる．
(１) 負荷が能力より大きい場合
･残業，休日出勤，外注利用
･余裕のある職場からの応援
･作業順序の入替え
･ロットサイズの変更（一般に小ロットに分割して，一部を先行，
一部を後回しにする場合が多い）
(２) 負荷が能力を下回る場合
･予定の繰上げ
･余裕のない職場への応援
･ロットサイズの変更（予定を繰上げてロットサイズを大きくする
場合が多い）
