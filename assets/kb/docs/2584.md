# 3・4・4 トヨタ生産方式（Toyota production system）

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

トヨタ生産方式は，自動車の最終組立ライン（後工程）が必要な
部品を必要なときに，必要な量だけ前工程から受取り，前工程は，
後工程に引取られた分の部品を作る，という基本的な考え方を実施
しようとするものである．したがって，トヨタ生産方式のねらいは
このようなジャストインタイム（JIT）生産によって，職場にひそ
むあらゆるむだを排除して生産性を向上することにある．JITを確
実に実施するには，工程の流れ化・同期化が必要であり，このため
の手段としてプルシステム，かんばんおよび生産の平準化を用いる．
a.プルシステム（pull system) 製造工程がいくつかの工程
からなっているとき，各工程に基準在庫水準を設け，後工程が前工
程（にある基準工程）から部品を引取る（引張る）と前工程は後工
程で引取られた量だけさらに一つ前の工程から部品や材料を引取っ
て生産し，もとの基準在庫量に戻す．このようにして，各製造工程
はさらにその前工程から必要な量だけ引取る形で順次前へ前へとさ
かのぼっていく．このような生産指示方式をプルシステムという．
このような方式のもとでは，生産計画を全工程に同時に提示する必
要はなく，最終組立ラインにだけ伝わればよい．
b.かんばん（Kanban) かんばんは通常1枚のカードで長方
形のビニール袋に入れてあり，大別すると引取りかんばんと生産指
示かんばんの二つに分けることができる．引取りかんばんは後工程
が前工程へ部品を引取りに行くときの請求書であり，品名・品番，
コンテナの収容数，前工程の置き場所，後工程の受入れ場所などが
明示されている．生産指示かんばんは品名・品番，工程名，生産指
示量，完成品の置き場所が明示されている．かんばんはむだを排除
するためのいろいろな機能をもっている．第一はかんばんと物とが
そろっているときは生産を行わない．かんばんだけのときだけ生産
を行うから余分なものを作らないことになる．第二はかんばんの枚
数やかんばんに明示されている収容数を少なくすることによって仕
掛在庫量を下げることになる．第三はかんばんなしの生産・運搬は
ないし，かんばんのない在庫はないから，かんばんは目で見る管理
の手段になりうる．
