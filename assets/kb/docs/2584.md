# 2・6・3 その他

> **章**: 第2章 鋳造
> **節**: 2・6 鋳造品の種類

JIS 規格に規定されていないが、工業的にも重要な鋳造品としてニッケル基耐熱超合金鋳物が挙げられる。ニッケル基超合金(nickel base superalloys)は、Cr,Co,Mo,W,Taなどの添加元素の固溶したニッケル固溶体(γ相)中にNi3Al相(γ′相)を析出させたものであり、優れたクリープ強度、熱疲労強度、耐高温ガス腐食性などを有する。γ′相中には添加元素が固溶するとともにγ′相中のAlはTi,Taなどで置換される。これらの超合金は鋳造後の溶体化処理、時効熱処理により体積率にして約60%のγ′相を立方体状に析出するが、このγ′相が超合金の耐熱性を担っている。ジェットエンジンやガスタービンの動翼、静翼はいずれも鋳造法により製造されるが、要求使用温度上昇につれ、普通鋳造用から一方向凝固用、単結晶用と合金開発が進んできた。これらニッケル基耐熱超合金には開発会社による各種のものがあり、普通鋳造用合金、一方向凝固合金、単結晶合金が製造されている。
