# 2・6・4 各種部品と使用材質例

> **章**: 第2章 鋳造
> **節**: 2・6 鋳造品の種類

JIS 規格では百数十に及ぶ鋳造材料が規定され、膨大な数の鋳造部材・鋳造部品が製造されている。鋳造品の使用材料は用途により要求される形状・寸法、被削性、強度、使用環境、コストなどとともに経験的事例を参考にして選択される。このため、同一用途部品3―29用途部材・部品等適用材料例一般機器用油圧シリンダ、ピストンCAC 503,CAC 702ポンプ、油圧・空気圧用バルブCAC 402~CAC 406,CAC 407ライナ、軸受、ギヤCAC 301~CAC 304,CAC 700製鉄機械用高炉羽口、転炉用ランスノズルCAC 101~CAC 103圧延機ロールハウジングSC360~SC480産業機械圧延機ウォームホイールCAC 304,CAC 502,CAC 503連続鋳造機用モールドCAC 103土木建設機械用キャタピラSCMn4粉砕機用ハンマSCMnCr2~SCMnCr4船舶用船外機プロペラADC 6船舶用プロペラCAC 301,CAC 701クランクケース、シリンダハウジングFCD 450鉄道用連結器、ブレーキシューSC,SCMnH 1~SCMnH 21電気機器用ターミナル、ケーブルコネクタCAC 102,CAC 201~CAC 203化学・バルブ、ポンプケーシングSCS 1~SCS 36石油工業用ケーシング、バルブ、フランジSCPH 1-CF~SCPH 32-CF美術・建築用門扉、エクステリアAC 3A装飾用金具、建築用金具CAC 201,CAC 203,ZDC2美術鋳物、景観鋳物CAC 406であっても異なる材料が選択されることも少なくない。表2・16に各種機械などに使用される鋳造品の用途と適用材料の一例を示すが、同様の部品に対してさまざまな材料が使用されていることには留意されたい。
