# 3・5・11 海外生産（overseas production）と世界最適調達

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

貿易の自由化と世界各国の規制緩和の流れの中で，製造企業の海
表3･ 4トヨタの世界最適調達に関する基本方針
･購買取引は，国籍，企業規模，新規・継続を問わず，開放的・公平・
無差別・対等かつ透明に行う．また，新規取引を希望する国内外の
取引先に対して，公平・公正な参入機会を提供するよう心掛ける．
･取引先の選定にあたっては，品質，技術，価格，および数量・納期
の確実性・安定性などの経済合理性に加え，経営の安定性，技術開
発力なども総合的に勘案する．
･長期的なビジョンの中で，相互繁栄を図ることができる取引関係の
確立をめざす．その基礎となる相互の信頼関係を築くため，取引先
との密接なコミュニケーションの促進を図る．
･自動車産業の国際化が進展する中で，海外での車の需要に応えて現
地生産を積極的に進める．この現地生産にあたっては，地元の企業
として地域社会に貢献できるよう，現地のサプライヤーからの調達
を積極的に推進する．
ープンドアポリシーのもとで，国籍，企業規模，取引実績の有無を
問わず，品質，原価，技術，納期と継続的改善に取組む経営姿勢，
体制などを総合的に勘案して取引先（サプライヤ）を選定する．表
3･4にトヨタ自動車の世界最適調達に関する基本方針を示す．
このように，海外調達（overseas purchasing）の重要性が増す
ことから，国際的視野に立った政治，社会，経済，技術，法律など
に関する幅広くかつ深い知識をもつ購買担当者を育成する必要があ
る．また，インタネットを利用した国際入札方式も普及しているの
で，購買担当者は情報・通信技術を身につける必要がある．
海外調達には，海外現地企業の現地における調達と，海外から国
内への調達（輸入）があるが，国内で海外企業から購買する場合，
国内企業からの購買と違って，為替レート，支払い貨幣（円建てか
ドル建てか），支払方法，国境を越えた遠距離輸送，関税等の新た
な問題を考慮する必要がある．
