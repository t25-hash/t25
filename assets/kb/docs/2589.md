# 3・5・2 購買管理の役割

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

製造企業における購買活動は，生産活動に必要な原材料，部品，
資材，構成品，機器，工具，消耗品などを調達する活動である．製
造企業が，より良い製品をより安く生産するためには，経営上必要
なものを，信頼できる取引先から，適正な価格で，必要なときに，
必要な量だけ確実に購入できる仕組みを整え，維持する必要があ
る．経営上必要なものを合理的に調達することを計画し実施・統制
する行動体系を購買管理（purchasing control ）という．特に，自
動車製造業のように多数の構成品・部品を組立てて製品を製造する
加工組立型企業においては，使用する構成品・部品の多くを他の企
業から買うかまたは他の企業に委託して作ってもらうので，製造原
価に占める原材料費，買入部品費，外注加工品費など，購買・外注
に伴う費用のウエイトが大きく，購買・外注管理の巧拙が製造原価
に大きな影響を及ぼす．また，購入された原材料，部品，資材，構
成品の在庫管理（inventory control ）の巧拙が在庫投資に大きな
影響を及ぼす．このように，製造企業においては原材料，部品，資
材，構成品など生産に必要な資材の購買が特に重要であるから，以
下では，これら生産用資材を購買管理の対象として考える．購買管
理の主要な役割には，以下のことが挙げられる．
a.用途に最も適したものを購入すること 製品のもつべき機
能から，その製品に必要な構成品・部品がもつべき機能を明らかに
し，最小のコストで必要な機能を得る努力をすることである．これ
は，購買品，外注品の価値分析（VA:value analysis ）や市場調
査（marketing research），品質管理（QC）により実現される．
b.信頼できる取引先から購入すること 購買活動は自社の製
品に利用するために他の企業が製造した構成品や部品を購入するこ
とであり，継続的に生産活動を行い，自社の製品の市場における競
争力を向上させるためには，継続的に取引可能な信頼できる優れた
取引先を選定しなければならない．そのためには，広い範囲から取
引可能な供給源の情報を集め，潜在的な供給源をリストアップして
おくことが必要である．
c .適正な価格で購入すること 製品の製造原価を引下げ，市
場での競争力を高めるためには購買品，外注品の適正な購入価格を
設定するとともに，継続的に購入価格を引下げる努力をする必要が
ある．ただし，購入価格を引下げることと必要以上に安く買う（買
いたたく）こととは全く異なる概念であることに注意しなければな
らない．前者は価値分析をとおして最小のコストで必要な機能を得
る努力をすることを意味し，後者は場当たり的に不当に安い価格で
購入することで，信頼できる継続的な取引は不可能となり，結果と
して高い買い物になるおそれがある．
d.納期に間に合うように購入すること 購買品，外注品の納
期（due date）を確保することは製品の製造原価を引下げるために
特に重要である．納期の確保とは，必要なものを必要なときに必要
なだけ入手することで，必要なときとは，生産計画であらかじめ定
められた時期をいう．この時期に遅延することはもとより，早すぎ
てもよくない．
e .適正な在庫を維持すること 購買したものを，使用する前
に一時的に保管する現象を在庫（inventory）という．需要量の変
動に柔軟に対応しながら生産活動を円滑に遂行するためには，購買
品，外注品をある程度在庫しておく必要がある．しかし，過剰な在
庫は製造原価の増大を招くので，購買品，外注品の在庫を適正量に
維持することが製造原価を低減するうえで重要である．
f .むだのない輸送方法で購入すること 必要なものを必要な
ときに必要なだけ購入することは，多頻度小口輸送をすることとな
り，輸送回数が増加し，結果として輸送コストが増大する．特に，
自動車製造業のように多数の構成品・部品を購買・外注する加工組
立型企業の場合，その影響が大きい．したがって，効率の良い輸送
方法を利用する必要がある．
以上のように，購買管理の基本的な役割は，生産活動を円滑に遂
行し，製造原価の低減を図るために，必要な資材を，必要なとき
に，必要な量，適正な価格で獲得し，必要としている生産工程に効
率良く供給することにある．したがって，購買に関する計画（購買
計画，purchasing planning）は，販売計画，生産計画（produc-
tion planning）に依存する．
