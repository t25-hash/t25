# 2・7・3 造型設備(molding plant、図2・65)

> **章**: 第2章 鋳造
> **節**: 2・64 )。

量産用には連続再生の容易なベントナイトをバインダとする生砂型造型が用いられ、主として鋳型全体(主型)を作る。鋳物の特徴である複雑な内部構造を作るためには、強度が高く崩壊性のよい中子が併用される。大型や非量産の部品では、造型機を使用しないでも高い鋳型強度が得やすい自硬性鋳型が使われる。a. ジョルト・スクイーズ造型機模型を設置した金枠に自由落下で鋳型砂を充てんし、ジョルト(衝撃加圧)およびスクイーズ(静的加圧)を行わせる造型機で、ジョルトピストンの外側にスクイーズピストンがあり、ジョルト後スクイーズが行えるようになっている。実際に行うジョルト回数は一つの鋳型を作るのに5~30回である。高圧造型機にはスクイーズヘッドが分割されていて油圧で操作するものがある。圧縮空気によりジョルトを加え、油圧によりスクイーズする。通常は造型後も鋳枠を付けたまま注湯ラインヘ移送するが、造型後鋳枠を抜き取る造型機もある。鋳枠は重量も重く、数も多くなるので、これをなくせば経済的であり、かつ管理に手数を要しない。これを抜枠造型と称している。振動騒音が大きく騒動負荷が高いので、最近では用いられない。ジョルトの代わりにさまざまな工夫をしたスクイーズ造型機に転換している。砂入れにエアレーションを適用してスクイーズ効果を高めたり、スクイーズを両側から行うタイプの造型機に変わりつつある。b. ブロースクイーズ造型機(図2・66)砂を鋳枠内に空気を用いて吹き込み、スクイーズして固める抜枠造型機である。この機械の特徴は両側に半分ずつの模型を付けたモールディングチャンバに砂を吹き込み、スクイーズして固め、主型を一度に作ることである。縦型の場合には押出して鋳型とする鋳型の両面に模型によるくローラコンベヤ式上下枠同時造型モールドコンベヤ式金枠式(枠付き)並列配置上下枠専用造型水平割り造型ライン定盤台車式上下枠交互造型抜枠式上下枠同時造型ローラコンベヤ式パレットコンベヤ式垂直割り造型ライン無枠式(抜枠)型左右同時造型ウォーキングビームベルトコンベヤ式図2・65 造型設備レイアウト自由落下方式(重力式)サイドブロー方式トップアンダブロー方式水平割り方式ブロー方式垂直割り方式チャンバ移動式トップブローチャンバ固定式3―31ぼみができるので、鋳型を前の鋳型に次々に組合わせて一組の鋳型とする。しかし、この方法では縦鋳込みとなり落差が大きいため、鋳造品の形状によっては好ましくない場合もある。そのような場合には模型と鋳枠を水平に置いた形式のブロースクイーズ造型機を用いる。c. 静圧造型機模型に金枠をかぶせて自然落下で砂を入れ、枠上部をふさいで密閉状態にして圧縮空気を0.3s程度砂上部に導き、砂を加圧する。空気は模型のベント穴から逃がす。騒音が低く、型の締まりもよい。d. 中子造型機ブロー造型機の一種だが、型に吹き込んだ砂は原則加圧成形しない。サンドリザーバ内の砂は、シューティングバルブを開けると上方および側面から吹き出る圧縮空気によって中子取り型に吹き込まれる。空気は空気逃げ穴(ベント)から排出され砂だけが中子取り型内に残り、中子ができる。砂はできるだけ流動性のよいものがよい。e. 自硬性造型非量産の鋳物を製造する装置で、小物の場合には連続移動装置を使用するが、大物の場合にはバッチ固定式である。
