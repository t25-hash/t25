# 2・7・4 砂処理設備(sand control plant、図2・67)

> **章**: 第2章 鋳造
> **節**: 2・64 )。

a. サンドミル(sand mill)鋳型砂の調整上最も重要な装置である。砂には粘土・水その他を配合し均一に混練する必要がある。場合によっては油や合成樹脂などの粘結剤を混合することもある。最も普通に用いられるローラ型のサンドミルは2個のローラが容器内で回転し、混合物を容器の底板に押し付けて強力に練るような形で作動する。鋳型砂の特性によってはかくはんを主体とするアジテータ型のミルも使用される。b. 砂再生装置(sand reclamation machine)型ばらし後の鋳型砂の表面に付着している不純物や古くなった粘結剤を取り除いて、再び使用できる砂にする装置である。乾式では高速空気や回転羽根で砂粒をターゲットに衝突させ再生させる方法と、ローラにより機械的にすり合わせる方法とがある。砂の再生に当たって砂中の不純物を燃焼させ再生砂を肌砂として使用する流動ばい焼炉が用いられている。c. 砂処理プラント回収砂はサンドクーラで冷却され、ブレーカスクリーンに入り、砂塊は粉砕され、ふるいでふるわれ、新砂とともにサンドミルに入る。混練された砂はサンドホッパに蓄えられ、必要に応じてベルトコンベヤで造型場に送られる。空気輸送装置を用いる場合もある。必要に応じてブレーカスクリーンのところに砂再生装置を配置する。効率上の問題から縦型に配置することが多い。
