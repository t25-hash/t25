# 3・5・3 外注管理の役割

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

外注は購買活動の一部である．
一般に，製品が多様化して複雑にて自社内で製造するには膨大な設備と資本を要するので，自社内で連産業や他の企業に委託することが多い．
は主要な構成品・部品を生産し，その他の構成品・部品の生産を関親会社という．
狭義の購買と違って，親会社は，図面，諸工具や材指導や融資援助などを行うのが普通である．
外注の意義は，①垂直②社内製作に必要とされる生産設備や要員の固定費を削減できるこ重構造を利用して加工費の節減を利用できること，⑤外注先企業がげられる．
自己の定める規格の構成品・部品などを，他の企業に委理（subcontract control）という．
外注管理の主要な役割は，基設計で他の企業に生産を委託することから，外注先企業の善し悪し次のことが挙げられる．
協力企業の中には経営規模が小さく，経営管理能力や技術水準の低り，外注品の品質，コスト，納期，数量を確保する必要がある．
ま協力関係を強めることが重要である．
