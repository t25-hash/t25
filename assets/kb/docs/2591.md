# 3・5・4 内外作区分

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

どの構成品・部品を自社で作り（内作），どの構成品・部品を外
注にするかという品目の決定を内外作区分という．内外作の決定
は，自社の生産技術や生産設備を規定するものであるから，経営計
画の一部として十分検討されなければならない．内外作区分に関し
て企業の取りうる選択は自社で作るか外注にするかのいずれかであ
り，考えられる方策は，①内作にする，②外注にする，③一部を内
作し一部を外注にする，のいずれかになる．内作にするおもな理由
は，コストや品質面で，外注にするよりも自社で生産するほうが優
れているからであり，逆に，外注にするおもな理由は，自社で生産
するよりも外注するほうがコストや品質面で優れているからであ
る．内外作の決定において，自社の現有の生産技術や生産能力をも
とに，長期的な展望に立って，新しい生産技術や生産設備をもつか
もたないかを決める生産構造的な決定を戦略的内外作決定といい，
優れた生産技術をもっている場合，その技術を用いて生産する構成
戦略的決定であり，保有技術による生産能力をどの水準に維持する
現在すでに所有している生産技術や生産設備をもとに，需要の変動
等に対処するために一部を内作し一部を外注にする生産調整的な内
外作の決定を戦術的内外作決定という．戦略的内外作決定は，経営
方針の一環として，経営政策や経営戦略的観点から経営幹部の責任
において決定されるべき問題である．内外作区分は，自社の有する
生産技術の水準により影響される．
a.自社が優れた生産技術をもっている場合 自社が他社より
品・部品は内作するべきである．このような技術を基幹技術と呼
ぶ．
b.他社が優れた生産技術をもっている場合 他社が自社より
も優れた生産技術をもっているか，自社にその技術がない場合，そ
の技術を用いて生産する構成品・部品は，当面，外注せざるを得な
い．このような技術を外部依存技術と呼ぶ．
c .自社も他社も同レベルの生産技術をもっている場合 自社
も他社も同レベルの生産技術を有する場合，一部を内作にし，一部
を外注にすることができる．このような技術を保有技術と呼ぶ．
基幹技術は，自社がもっている特許技術のような特に優れた技術
であり，その優位性が自社で作る構成品・部品の特徴となるような
技術である．構成品・部品の生産を自社で継続することにより，そ
の技術を維持し，発展させる必要がある．
外部依存技術は，他社がもっている優れた技術であり，自社で生
産するよりも外注したほうが品質，原価の面で有利である．外部依
存技術が特許技術のような特に優れた技術の場合，その企業に外注
せざるを得ない．
保有技術は，基幹技術ほどの秘密性や重要性はなく，他社に依存
することも可能な技術である．このような技術で生産する構成品・
部品は，需要の多いときに，自社で生産できない部分を外注するこ
とができる．
外部依存技術を保有技術あるいは基幹技術にするか否かの決定は
かの決定は戦術的決定である．
