# 2・7・6 環境衛生設備

> **章**: 第2章 鋳造
> **節**: 2・64 )。

鋳造工場は砂を多用するので比較的粉じんが出やすいため、工場流気を考慮することが望ましく、発生箇所での局所対策が必要である。a. 集じん装置(dust collector)鋳造工場のキュポラ、電気炉、砂処理装置、注湯場などから出るばい煙や粉じんの対策に、乾式や湿式の集じん装置が用いられる。粉じんは、鋳物工場内の発生箇所により性状が異なり、溶解ヒューム系、生砂系、中子(シェル砂)系、仕上げ系に分けて管理される(図2・68)。作業場の粉じんを集めるにはフードを設け、ダクトを通して排出する。フードは粉じん発生箇所をできるだけ包むように設け、ダクトはできるだけ曲がり部などがないようにする。粉じんをフードに吸引するために必要な風速は粉じんの発生条件によって異なるが、注湯作業場、砂混練作業場、型ばらし場の順に大きくする必要がある。ダクトの抵抗損失などを考慮してフード入口でこの値になるよう排風機の容量を選定する。フードはシェイクアウト(型ばらし)、コンベヤの落とし口、スクリーンなどに設置されている。シェイクアウト部からの蒸気含じんガスの吸引については、ガス中の水分を考慮して吸引ガス温度が露点に達しないよう高温排ガスを引き入れる。バグフィルタは円筒または扁平な袋で、ろ布に付着したほこりの層によって粉じんをろ過する。ポリエチレン製の織布で、微細な粉じんまで取ることができ、しかも圧力損失も比較的少ないので、特に高い集じん効率を必要とする場合に用いる。ろ布についた粉じんは一定時間ごとに揺動、逆圧、圧縮空気噴射により払い落とす。b. 気中有害物質(ガス)と臭気鋳造工場のにおいは炭素系物質の不完全燃焼による焦げ臭さによるものである。成分濃度は低く健康上問題はないが、嫌悪感が大きい。対策としては脱臭装置が用いられ、薬液洗浄・吸着方式から(触媒)燃焼方式が用いられる。c. 騒音対策一般的には、騒音発生源を覆って騒音の低減を図ることが多く実施されている。目標値として騒音レベルを工場内80dB 以下、工場境界線で50dB 以下としている。集じん機の送風機とモータについては防音室へ格納し、消音器設置、ラギング処理を行う。シェイクアウトマシンは砂落とし時に100dB(A)を超えるので防音室へ格納し、製品の出入口も扉で遮断するなどの考慮を払う。建屋壁を防音壁としたり、工場境界線に遮音塀を設けることも行われている。d. 振動加速度の大きいジョルトを使用しなくなってきているので、振動は低減されてきている。大型のシェイクアウトマシンなどの振動の大きいものは、装置の固有振動数を考慮した基礎工事を行う必要がある。e. 産業廃棄物廃棄物は溶解材の種類によって異なるが、鋳3―33図2・69集じんダスト分散システム物廃砂、スラグ、ダスト(溶解およびその他)、れんがくず等がある。金属の再資源化率は高く、問題となる廃砂やダストは分別処理され(図2・69)、埋立処分から、骨材、セメント等への有効利用が進められ、ゼロエミッションへの試みが進められている。文献(1) 石野亨、鋳物五千年の足跡、(1994)、日本鋳物工業新聞社。(2) 日本鉄鋼協会編、溶鉄・溶滓の物性値便覧、(1972)、日本鉄鋼協会。(3) Brandes,E.A.and Brook,C.B.,eds.,Smithells Metals Refer-ence Book、(1998)、Butterworth-Heinemann.(4) 日本鋳物協会編、鋳物便覧、改訂4版、(1986)、丸善。(5) 日本金属学会編、金属データブック、(1993)、丸善。(6) 大中逸雄、コンピュータ伝熱・凝固解析入門、(1985)、326、丸善。(7) Carwley,A.F.,Int.
