# 3・5・6 購買業務（purchasing operation）

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

購買活動に必要な業務には，購買要求の受付，発注先の選定と交
生産技術部門または設計部門で購買品の仕様，規格，品質などが
表3･3 経団連の購買取引行動指針（抜粋)
・購買取引は国内外に開放され，公平かつ透明とし，理解しやすい簡
素な手続によって行われる．
･購買組織は，購買取引となることを希望する企業等に対して，公平
かつ公正な参入機会を与える．
･購買組織は，購買取引先となることを希望する企業等からの申し入
れに対して真摯に対応する．
･新規に購買取引先となることを希望する企業等の資格審査は，当該
企業等の実績，信頼性，技術力等を基準とし，その評価に基づいて
行う．
･購買取引先および見込取引先の情報は，契約に基づき，購買活動を
行う上で必要なもののみを入手する．又，業務上知り得た情報は厳
格に管理し，機密の保持に努める．
･購買組織に属するもの（購買担当者）は，購買取引先および見込み
取引先との個人的な利害関係を持ってはならない．
･購買担当者は，購買取引先および見込み取引先から謝礼などいかな
る個人的な利益も得てはならない．また，社会的常識の範囲を超え
決められ，生産管理部門で数量と納期が計画されると，購買業務が
次のように進められる．
a.購買要求の受付と見積り依頼 生産管理部門から購買部門
に購買要求書が発行される．購買部門は，購買要求書をもとに，い
くつかの取引先へ見積りを依頼する．
b.発注先の選定と交渉 購買部門は，生産技術部門または設
計部門へ購買品の材料の所要量や加工時間などを問合わせる．取引
先からの見積りと生産技術部門または設計部門からの回答を比較検
討する．
c .発注契約 発注先と品質，納期，価格に関する条件を決定
すると正式に所定の手続きを経て注文書を発行し，契約を行う．条
件を満たす取引先と品質，納期，価格に関する交渉を行う．
d.納期管理 契約がなされ，発注業務が終了すると，購買品
が納期どおりに納品されるかどうか管理する．
e .検収 実際に納入される際に，納期より早い場合は返品し，
遅れた場合はペナルティを課す．また，品質検査を行い，受入の可
否を決定する．
f .発注先への決済 納品が済めば，経理部門により契約した
支払条件で決済する．
