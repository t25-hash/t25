# 3・1・1 溶接・接合の原理

> **章**: 第3章 溶接・接合および切断
> **節**: 3・1 溶接・接合の基礎

溶接・接合は,電子機器から大型構造物に至るものづくりの基盤技術として,産業立国としてのわが国の発展に重要な役割を果たしている.溶接・接合技術は,いくつかの基礎工学の上に発展を遂げており,近年の科学・技術の著しい進歩は,基礎理論をはじめとし,応用技術に大きく影響を与えている.この流れの中で,新しい学問的体系化と高度な先進技術の有機的な結合による溶接・接合技術が進展し,それらの正しい理解と適用が必要となっている.a. 材料の性質物質を材料としてものづくりが行われている.材料の接合は,以下の二つに分けられる.(1) 化学的結合による原子間引力,すなわち共有結合,イオン結合,金属結合などの強い結合力が基本的なメカニズムとなっている溶接・接合(溶融接合,液相-固相接合,固相-固相接合,気相-固相接合)(2) ファン・デル・ワールス(van der Waals)結合や水素結合のような弱い結合力が基本となっている接着同種,あるいは異種材料どうしが接合するかしないかは,物質を構成する元素の組合せによって決まっているものであり,量子論的化学結合や熱力学的に説明できる.すなわち,材料の組合せにより,接合するものとしないものは材料固有の性質として決まっているということである.空気中の溶接・接合においては,材料固有の性質のままにふるまうことを妨げる要因がある.金属表面には酸化膜,ガスや水分,有機物などの吸着層が存在する.接合界面の原子のオーダでの凹凸も接合効率を低下させる.これらの阻害要因を種々のプロセスで除去し,材料間の強い化学結合を効率よく達成させる方法が溶接・接合法である.b. welding)や,セルフピアシングリベット(self piercing rivet)などへの転換も実現化されつつあり,新しい,より効率的な接合法が日夜開発されている.c.
