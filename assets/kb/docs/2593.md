# 3・1・1 溶接・接合の原理

> **章**: 第3章 溶接・接合および切断
> **節**: 3・1 溶接・接合の基礎

溶接・接合は，電子機器から大型構造物に至るものづくりの基盤
技術として，産業立国としてのわが国の発展に重要な役割を果たし
ている．溶接・接合技術は，いくつかの基礎工学の上に発展を遂げ
ており，近年の科学・技術の著しい進歩は，基礎理論をはじめと
し，応用技術に大きく影響を与えている．この流れの中で，新しい
学問的体系化と高度な先進技術の有機的な結合による溶接・接合技
術が進展し，それらの正しい理解と適用が必要となっている．
a.材料の性質 物質を材料としてものづくりが行われてい
る．材料の接合は，以下の二つに分けられる．
(１)化学的結合による原子間引力，すなわち共有結合，イオン結
合，金属結合などの強い結合力が基本的なメカニズムとなっ
ている溶接・接合（溶融接合，液相-固相接合，固相-固相接
合，気相-固相接合)
(２)ファン・デル・ワールス（van der Waals）結合や水素結合
のような弱い結合力が基本となっている接着
同種，あるいは異種材料どうしが接合するかしないかは，物質を
構成する元素の組合せによって決まっているものであり，量子論的
化学結合や熱力学的に説明できる．すなわち，材料の組合せによ
り，接合するものとしないものは材料固有の性質として決まってい
るということである．
空気中の溶接・接合においては，材料固有の性質のままにふるま
うことを妨げる要因がある．金属表面には酸化膜，ガスや水分，有
機物などの吸着層が存在する．接合界面の原子のオーダでの凹凸も
接合効率を低下させる．これらの阻害要因を種々のプロセスで除去
し，材料間の強い化学結合を効率よく達成させる方法が溶接・接合
法である．
b.接合法の分類 溶接・接合法は，接合のメカニズムで見る
と，図3･1に示すように，原子間の化学結合によるものと，リベッ

る．溶接・接合法の原理を，図3･2に示した．これらの原理は，
いずれも接合を阻害する要因，特に接合面の酸化膜や凹凸を取るた
めの手法である．
融接では熱源から高エネルギーを付加し，接合面を溶かして阻害
する要因を取り除くことにより接合する．高エネルギーを付加する
ということは，高度に調質された材料組織を溶かすことや，固体の
ままでも高温に加熱し組織の粗大化を起こさせることとなり，材料
の劣化につながる．このような材質の劣化の見返りとして，接合を
実現する方法である．
