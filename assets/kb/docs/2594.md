# 3・1・2 溶接熱伝導

> **章**: 第3章 溶接・接合および切断
> **節**: 3・1 溶接・接合の基礎

ここでは、アーク溶接(arc welding)の場合を中心に、溶接の熱伝導論的取扱いについて簡単に述べる。a. アーク溶接の熱効率アーク放電への電気的な供給エネルギーはアーク電流と電圧の積で与えられる。アーク溶接では、このエネルギーのすべてが母材へ流入し、溶融に寄与するわけではなく、その一部は溶接トーチや大気中へ放散される。母材へ流入するエネルギーとアークへの電気的供給エネルギーの比を熱効率と呼んでいる。すなわち、アークにより母材へ供給されるエネルギーは、次式で与えられる。熱効率は、溶接法、溶接条件により異なる。通常のアーク溶接における熱効率は、50%以上になる。b. 準定常熱伝導方程式とその解溶接においては移動熱源による温度場が問題となることが多い。移動熱源による定常的温度場は、次の準定常熱伝導方程式により表現される。ただし、座標系は、熱源とともに速度で軸方向に移動するものとしている。物性定数が温度に依存しない場合には、準定常熱伝導方程式に対する解析解が得られているので、それらを利用すれば溶接時の温度場を近似的に知ることができる。例えば、半無限母材の表面を軸方向に一定速度で移動する点熱源(図3・4)による温度場は式(3・3)で、また、厚さの無限平板上を物体表面と垂直な線状の熱源が、物体の表面に沿って一定速度で移動する場合の温度場は式(3・4)で与えられる。図3・5中のθおよびは無次元化された温度と入熱であり、それぞれ次式で定義される。図3・5のように、溶接部の温度場は熱源の移動速度の影響を強く反映したものになる。図3・6は、アークを点状の集中熱源とみなし、式(3・3)による計算結果と比較したものである。計算値(図3・6中の曲線)は実験結果を定性的に説明しており、熱伝導による解析がアークによる母材の溶融現象の推定に有効であることがわかる。d. 溶接部の冷却速度溶接部の材料組織ならびに特性は、冷却速度または冷却時間によって著しく影響される。上述の解を利用すれば、熱源後方における溶接線(軸)上の移動点熱源による冷却速度として、次式が得られる。なお、厚板溶接の冷却速度は式(3・6)で、薄板溶接の冷却速度は式(3・8)で近似的に表現される。
