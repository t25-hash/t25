# 3・1・3 溶接部における材料の変質と溶接欠陥

> **章**: 第3章 溶接・接合および切断
> **節**: 3・1 溶接・接合の基礎

アーク溶接のような溶融溶接では、溶加材と母材が溶融した溶接金属(weld metal)と、融点以下ではあるが溶接熱により組織(microstructure)が変化した熱影響部(HAZ : heat affectedzone)が図3・7のように溶接部に形成される。a. 溶接金属とガスおよびスラグの反応アーク溶接では、健全な溶接金属を得るためにフラックスやシールドガスなどで溶融金属を大気から保護する。しかし、遮断がつねに完全とはいかず、高温のアークにさらされた酸素、窒素、水素などのガス(gas)の一部は原子状に解離して溶融金属に吸収されやすくなる。通常の金属は、溶融状態では固体状態の場合に比べてガス原子の溶解度が高いため、溶融した金属は極めて短時間で多量のガスを吸収しやすい。実際、鋼アーク溶接金属のガス成分量は普通の鋼材に比べてかなり高いことが多い。吸収されたガスは有害な化合物を作るばかりでなく、溶接金属の凝固に際してガスの溶解度が激減するので、溶けきれなくなったガスが直接または間接的に、気孔、割れおよび脆化の原因になり、溶接部の特性に悪影響を及ぼす。酸素は、酸化物や炭酸ガスとして鋼溶接金属に残留すると、介在物や気孔として伸びおよび衝撃値を減少させ、酸化鉄(FeO)が多くなると鋼は赤熱脆性を示す。しかし、適量の酸素は、合金元素との組合せで適度の微細酸化物を分散させて微細フェライト(アシキュラフェライト)の核となり、高靱性を得るのに有効と考えられている。窒素は、フェライト鉄中では窒化鉄(FeN)として針状に析出する。一般に、鋼溶接金属の窒素量が増加すると引張強さは増加す3―37(a) 溶融池形状と柱状晶の関係るが、伸びと衝撃値が低下する。また、焼戻し脆性、青熱脆性の原因とも考えられており、気孔の原因にもなる。水素は非常に拡散速度が速いため、室温においても時間とともに放出されて減少し、加熱するとさらに急速に減少するため、予熱後熱処理は水素量の低減に効果がある。一般に過飽和に含まれた水素は鋼をもろくし、溶接金属中で気孔や割れを生じさせるばかりでなく、拡散性が高いため熱影響部にまで達してビード下割れを生じさせたりする。オーステナイト系溶着鋼の使用がビード下割れ抑制に効果があるのは、水素の溶解度がフェライトに比べてオーステナイト中のほうが高いので、溶接金属のオーステナイト中に多量の水素がとどまり、熱影響部に水素が拡散するのを防ぐためと考えられる。被覆アークやサブマージアーク溶接ではスラグ(slag)が形成される。スラグを構成する酸化物は塩基性、中性および酸性に分けられる。一般に、スラグの塩基度が大きくなると溶接金属の酸素量は減少する。また、酸素との親和力がFeより大きい元素を添加すると酸素量の減少(脱酸)が期待される。おもな元素の脱酸力は、大きいものから順にAl,Ti,Si,Mn,V,Cr,W,Mo,Niとなり、実際に脱酸可能なものはCr以上である。溶接過程では、被覆剤やフラックス中の脱酸剤や金属元素が大気からの酸素と反応し、同時に被覆剤やフラックス中の酸化物の一部も還元されて溶接金属中に金属元素が入る。b. 溶接金属の凝固と偏析溶接金属は、アークが移動して遠ざかるに従って冷却し、溶融境界部から凝固し始める。凝固(solidification)は溶融境界で隣接する母材の結晶粒と一体化して、母材の結晶粒と同じ結晶方向をとってエピタキシャル成長する。結晶は特定の方向に速く成長しやすい性質をもち、鉄のように立方晶構造をもつ結晶は100> 方向への成長速度が大きく、溶接金属結晶の容易成長方向が温度こう配の最大方向と一致すると、成長が持続しやすい。成長方向に長く伸びた結晶は、その形状から柱状晶と呼ばれる。図3・8に溶融池形状と柱状晶の関係を示す。温度こう配が最大となる方向は溶融境界部と直角方向であるから、結晶の成長の状態は溶融池の形状に大きく依存する。溶接速度が大きくなると溶融池は図3・8(a)のように涙滴状となり、温度こう配が最大となる方向は、溶接金属の中心部近傍までほとんど変化しないので、柱状晶が溶融境界部からまっすぐ溶接方向に垂直に伸びて溶接金属中央でぶつかるような凝固組織となる。一方、溶接速度が小さくなると図3・8(b)のようなだ円状の溶融池となり、温度こう配が最大となる方向は周辺部から中心部に向かって連続的に変化し、柱状晶もそれに応じて徐々に方向を変える。そして、容易成長方向と温度こう配最大方向が一致しなくなると、新たな結晶の成長が始まる。一般に溶接金属には、凝固時の固相と液相の溶質濃度の差を反映した偏析(segregation)が生じる。マクロ偏析としては、溶接金3―38加工学・加工機器属の溶融境界から中央部にかけて次第に変化していく偏析が、クレータ部やスポット溶接部に見られることがある。ミクロ偏析としては、結晶の粒内と粒界での成分変化などで、非常に多く見られる。鋼溶接金属では、P,S が偏析しやすく、P,S を含む液相の融点が低いこともあって高温割れを起こしやすい。c. 溶接熱影響部の組織と硬さ溶接金属の周辺の母材は、溶接熱影響によって金属組織が変化する。溶接熱により、母材は室温から高温へ急速に加熱された後高温から室温へ冷却されるが、これを溶接熱サイクルという。図3・9に鉄-炭素系状態図(phase dia-gram)とアーク溶接時の熱サイクルの例を示すが、1~6の熱サイクルは鋼溶接部の断面図の場所に対応する。点1の融点付近まで加熱されたところでは、結晶粒が粗大化した後冷却過程でGS 線(A)と交差すると、オーステナイト粒界からフェライトが析出する。点3は一度オーステナイトに変態した後結晶粒が粗大化しないうちに冷却するため、フェライトとパーライトに変態して微細化した領域である。点4はAとAの中間まで加熱されたので、パーライトのみがオーステナイト化しようとしてパーライト中のセメンタイトが一部球状化した領域である。点5はA直下までしか加熱されていないので、マクロ的な組織変化は生じていないが、ミクロ的には炭化物や窒化物の析出が生じ材質劣化を伴うことがある。溶接熱影響部の組織をより正確に予測するには、連続冷却変態図(CCT 図)が有用である。溶接用CCT 図から溶接方法や条件に応じた冷却速度に対して組織と硬さ(hardness)が予測できる。鋼は、オーステナイト状態からある程度以上の冷却速度で冷やされるとマルテンサイトという硬い相ができる。マルテンサイトの硬さは、オーステナイトに比べて著しく高く、炭素量の増加とともに急激に増加する。一般に硬化した組織ほど、塑性変形しにくく延性を欠き、割れやすくもろい傾向にある。溶接熱サイクルの冷却時に生じるいろいろな変態も、結晶粒の微細化、多量の固溶炭素、炭化物の析出、転位や双晶の導入などを伴うため、鋼の溶接熱影響部を硬化させる。硬化の度合いはフェライト、パーライト、ベイナイト、マルテンサイトの順に大きくなり、同じ順序で割れやすくなる。したがって、鋼の硬化性は、溶接熱影響部の低温割れ感受性や靱性に密接に関係することから、溶接性の指標になる。すなわち、硬くなりやすい鋼は割れやすくて溶接しにくい。鋼の熱影響部における硬さは場所によって異なる。高張力鋼溶接部では熱影響部の中でも溶接金属に近い粗粒域がマルテンサイト組織となるため、著しく硬化する。最高硬さは、鋼の化学組成と溶接時の冷却速度によって変化する。硬化の度合いを鋼の化学組成から予測するため炭素当量(carbon 溶接割れ溶接部の割れは、図3・10に例示するように、(a)溶接金属および(b)熱影響部にいろいろな形態で発生する。溶接金属の割れは、高温割れ(hot cracking)と低温割れ(coldcracking)に大別され、溶接金属がまだすべて固まりきらない高い温度(凝固温度範囲)またはその直下で発生する割れを高温割れ、約300°C以下の低い温度で生じる割れを低温割れと呼ぶ。溶接金属は凝固時に急激に収縮しようとするが、まわりの母材によって拘束を受けて引っ張られるため、溶接金属に割れが入る現象が高温割れである。高温割れには元素の偏析が大きく関与する。通常、溶接金属の凝固は溶融境界から始まり、結晶が中央方向に伸びながら進行する。その間、早く固まった部分から液体のほうに向かって不純物元素が吐き出されるように凝固偏析が生じるため、伸びてきた結晶どうしが中央でぶつかるようになると、そのすきまに液体の薄い膜ができる。この薄い膜は、不純物濃度が高いため融点がかなり低くなる場合があり、このような液体の薄い膜が存在している状態で、すでに固まった部分は収縮しようとするので、溶接金属全体としては引張応力を受け、液体膜は強度が低いのでその部分から割れることになる。この液体膜の部分は結晶粒界に対応し、もし液体膜が固まって粒界を形成した状態で力がかかったとしても、粒界には一般に不純物が偏析しているので結晶粒内に比べて強度が弱く、粒界割れとなる。したがって、高温割れは、元素の偏析が生じやすいすみ肉溶接やクレータに現れやすく、溶接金属形状や拘束を考えて(a) 溶接金属の割れ(b) 溶接熱影響部に起こるいろいろな割れ図3・10 溶接部の割れ応力を小さくするよう工夫したり凝固偏析を抑えることが、高温割れ防止に有効である。鋼の場合、母材のS 成分が多いとFeS を生じて粒界に低融点の共晶を作って高温割れが発生しやすくなる。これを減少させるのにMn量の増加が効果がある。MnはS と反応して硫化物MnS を作り、有害なFeS の偏析が層状に圧延方向に平行に存在するサルファバンドを含むような鋼材では、強いサルファバンドから延長するような形で溶接金属内の粒界に沿って割れが入ることがあり、サルファクラック(sulfurcrack、硫黄割れ)と呼ばれる。一方、溶接金属の低温割れは、溶接金属中に侵入した水素(hydrogen)と室温付近で溶接金属のまわりに存在する収縮応力や切欠部の応力集中、溶接金属の延性低下などに起因すると考えられる。突合せやすみ肉溶接のルート部の応力集中によって生じるルート割れ(root cracking)、水素に起因すると考えられるミクロ割れなどがその代表である。溶接熱影響部の割れも溶接金属の割れと同様、発生温度によって高温割れと低温割れに大きく分けられる。熱影響部の高温割れには液化割れと延性低下割れがあり、液化割れは不純物元素の偏析や低融点共晶溶融による粒界液化に起因すると考えられている。一方、低温割れは、一般に温度がマルテンサイト変態が始まる温度以下または300°C以下の温度で発生し、粒界、粒内のいずれにも生じる。溶接後かなり時間を経過した後に発生する遅れ割れは低温割れの代表的なものであり、熱影響部の硬化、水素、引張応力が原因といわれている。つまり、熱影響部の内部ひずみが高い硬化組織に水素が集まり、もろい状態になっているところに引張応力がかかると割れやすくなる。割れが遅れるのは、水素がある程度の量集まるのに時間がかかるためと考えられている。ミクロ組織としては、マルテンサイトが最も割れやすく、ベイナイト、フェライトとパーライトの混合組織の順に割れにくくなる。.
