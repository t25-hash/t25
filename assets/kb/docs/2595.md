# 3・5・8 外注利用の目的

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

製造企業が外注を利用する目的は，他社の資本，生産設備，生産
技術，労働力を有効に活用することによって，よりいっそう良質廉
価な製品を生産することにあり，一般に次のような場合に利用され
る．
a.技術能力が不足した場合 自社にその構成品・部品を生産
する技術と設備がない場合，これを補うために外注を利用する．
b.外部に技術的に優れた企業がある場合 自社でも生産可能
であるが，外注したほうが低原価で構成品・部品を手に入れること
ができる．
c .生産能力が不足した場合 需要は変動するので，好況時に
合わせて人員・設備を用意すると不況時に遊休を生じ経済性を損な
う．このような危険を避けるために，安全な範囲の生産設備と要員
を保有し，この能力以上になった分を外注を利用して補う．これ
は，一時的な能力不足を補う場合にも当てはまる．
d.資金が不足した場合 自社にその構成品・部品を生産する
ための技術や設備を導入する資金が不足している場合，あるいは，
仮に資金を調達できても，投資に見合う需要が見込めないような場
合，自社で生産する代わりに外注する．また，技術進歩が早い場合，
膨大な設備投資を繰返すことは危険を伴うので外注を利用する．
部品が外部からは手に入らない場合，②サプライヤの信頼性が低い
場合，③コストや品質面で外部よりも自社のほうが優れている場
合，④自社の生産力が余っている場合，には内作する．また，⑤自
社の生産工程の秘密を守りたい場合にも内作する．
ちに生産工程に入るという点と，自社が定める仕様や規格の特殊部
品（規格外部品）が対象であるという点で，特に納期管理と品質管
理が重要になる．このため，外注先に立入った管理・指導が求めら
れる点が購買の場合と異なる．また，外注先に提供する図面（指定
図）を作ったり，外注先が提供する図面（承認図）を承認したりす
る技術を自社がもっているというのが前提である．特殊部品を生産
する外注先には，その専門的な生産技術や品質管理技術が要求され
る．外注品がただちに自社の製品の品質に反映するため，外注先の
決定には十分な慎重さが要求される．外注業務の要点は，外注先の
選定と外注品の円滑な生産進行である．外注先の選定とは，外注品
の品質，コスト，数量，納期を満足する外注先企業を選定し，契約
発注することであり，この適切な選定が外注管理の最も重要な点で
ある．一般に，その選定基準として，①技術，管理能力，②負荷状
況，③外注単価のレベル，④過去の品質，納期などの確保程度，⑤
総合評価，が挙げられる．外注先企業との折衝，打合せにより契約

捗状況を把握することが必要である．
