# 3・1・4 溶接による変形と残留応力

> **章**: 第3章 溶接・接合および切断
> **節**: 3・1 溶接・接合の基礎

溶接されると溶接部近傍は急激な温度上昇およびその後の冷却により収縮する能力(固有ひずみの積分値)を与えられる.固有ひずみにより発生した応力を溶接残留応力(welding residualstress)と呼び,収縮やひずみを総称して溶接変形(weldingdeformation)と呼ぶ.固有ひずみは固有応力(残留応力や熱応力など)の発生原因であり,残留塑性ひずみにほぼ等しい.しかし,溶融金属の液体状態では塑性ひずみの定義が使用できないから,溶接分野では固有ひずみが一般に使用される.a. 固有ひずみ(inherent deformation)溶接構造物に生じる溶接変形は構造物の大きさ,部材配置,溶接施工順序や溶接法によって異なり,非常に複雑である.溶接変形は図3・16に示すように分類される.横収縮は図3・12(c)に示す方向の固有ひずみの積分値で,溶接入熱윰윬읏による温度上昇過程の膨張量α윰윬읏/ρで与えられる.縦収縮,縦曲り変形,座屈変形は固有ひずみにより生じる.回転変形は溶接過程での熱膨張に起因している.横曲り変形(angular distortion)は板厚表裏面での固有ひずみの相違で生じる.溶接変形で問題となる横収縮,横曲り変形に及ぼす溶接入熱,板寸法などの諸因子の影響は入熱のパラメータ윰윬읏/で説明されている.平板の上に直線ビードを置くビードオンプレート溶接で得られた横収縮を図3・17に示す.溶接入熱が大きいか,板厚が小さくなって윰윬읏/が大きくなると,板厚裏面が溶融するようになり,1層1パスの突合せ溶接の横収縮(図3・17中の原点を通る直線)と同じになる.ビードオンプレート溶接で得られた横曲り変形を図3・18に示す.横曲り変形が윰윬읏/の値の中間値で最大になる理由は次の2点にある.(1) 横曲りを生じる能力は板厚表面側と裏面側の横収縮の差に起因しており,윰윬읏/が大きくなると表裏面の横収縮の差が小さくなり,横曲りを生じる能力が小さくなる.(2) 윰윬읏/が小さくなると,横曲り変形の抵抗を示す曲げ剛性が大きくなる.多層溶接での横収縮と横曲り変形は残留応力と異なり加算的であり,横収縮は全溶着金属が増えると増加し,横曲り変形は全パス数が多いほど大きくなる.d. 溶接残留応力の緩和と溶接変形の低減溶接残留応力は継手の使用性能や加工精度に影響を与えるため,除去を目的とした手段として,応力除去焼なまし(stress relief annealing),低温応力緩和法,機械的処理法,ピーニングなどがある.いずれも原理的には溶接固有ひずみ発生部に引張りの塑性ひずみを与えて,固有ひずみの大きさを小さくする手段である.なお,応力除去焼なましは鋼の変態点か焼戻し温度以下の適当な温度に加熱保持する方法で,材質の改善も図られるので溶接後熱処理(PWHT : post weldheat treatment)とも呼ばれる.溶接変形の軽減には,(1)収縮を見込んで部材寸法を調整する延ばし,横曲り変形を見込んだ逆ひずみ法,(2)高エネルギー密度の溶接方法の採用による溶接入熱量の低減,(3)開先形状の工夫による溶着金属の減少や板厚内のバランスによる変形の低減,(4)抑えジグによる変形の拘束,などがある.許容値を超えた変形に対しては,(1)線状加熱や点加熱による矯正,(2)プレス加工などの機械加工による矯正,などの軽減法がある.
