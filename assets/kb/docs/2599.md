# 3・6・3 代表的発注システム

> **章**: 第3章 管理システム
> **節**: 3・6 在庫管理

基礎理論に基づき，インプットをコントロールするための代表的
発注システムについて記す．
a.発注点方式［reordering point system（JIS Z8141），reor-
dering point method，order - point system（JIS Z8141）］ 在庫
を連続的に調査し（連続在庫調査方式)，在庫量が一定の水準（発
注点 ）まで減少したとき，一定の数量を発注する方式である．
発注点 は調達期間中の推定払出量と安全在庫の和として次式
で求める（3･6･2･b項参照)．
＝μ＋ασ
ここで，μ：単位期間の注文量の平均，σ：単位期間の注文量の
標準偏差，α ：安全係数，α：品切れ率， ：調達期間．
発注量としては，経済的発注量EOQを用いることが多い．
この方式では，発注量は一定，発注間隔は変動することになる．
この方式は，その都度予測をする必要がないため，比較的簡単な方
法であり，自動化しやすい方法であるが，在庫量は多くなる傾向が
ある．したがって，払出しが比較的安定しており，単価が安い標準
品や共通品に用いられる．
b.定期発注方式［periodic ordering system（JIS Z8141)，
図3･24,政策
periodic ordering method，fixed order period system］ 発注間
隔をあらかじめ定めておき（定間隔在庫調査方式)，現在在庫量，
払出量の予測値などに応じてそのつど発注量を求め，発注する方式
である．この方式の発注量は次式で示される．
発注量＝(調達期間＋発注間隔）中の予測需要量
－発注残－手持ち在庫量＋安全在庫
発注間隔とは，発注から次の発注が行われるまでの期間をいい，
発注残とは，発注したもののまだ入荷していないものをいう．
発注間隔は，生産計画のサイクルや取引先との関係，経済性等を
考慮して決める．経済的発注量の逆数として求めることもある．こ
の間隔を短くすれば，きめの細かい管理ができ，在庫も減少する
が，手間がかかる．安全在庫は，（調達期間＋発注間隔）中の需要
量の予測誤差に対処するために保有する（3･6･2･b項参照)．
この方式では，発注間隔は一定，発注量はそのつど変動すること
になる．また，予測のための手間がかかるが，正確な発注を行い在
庫を極力削減したい品目，すなわち単価が高い引当品や特殊品に用
いられる．
c.定期発注点・補充点方式 在庫調査間隔 ，発注点 ，最
大在庫量（補充点） をあらかじめ定めておき，定期的に在庫調
査して（定間隔在庫調査方式)，有効在庫量が発注点を下回ってい
るときに（補充点－有効在庫量）分を発注する在庫管理方式で，
-- 方式ともいう．この方式は，つねに在庫を監視しなくて
もよい定期発注方式と，発注のつど予測しなくてよい発注点方式の
利点を活用したものである．調査間隔が一定である点と，発注量が
一定量ではなく（－有効在庫量）であるところが一般の発注点法
と異なる．
発注点 は，在庫調査時点の直後に有効在庫量が発注点を下回
る場合を考慮して，調達期間に在庫調査間隔を加えた期間の払出要
求量に対処することが可能になるように決める．例えば，
＝μ×＋＋α＋σ
：発注点， ：調達期間， ：在庫調査間隔
μ：単位期間当たりの平均払出量
σ：単位期間当たりの払出量の標準偏差
α ：安全係数，α：品切れ確率
として求める．補充点は，払出要求量の分布，購入単価，期首（あ
るいは期末）に保有している在庫に対する保管費用，品切れ費用，
発注費用等を考慮して，統計的方法やシミュレーションによって定
めるとよい．
d.2ビン方式［double bin system，two bin system（JIS
Z8141）］ 同容量の在庫が入った二つのビン（箱，容器）を用
意しておき，一方のビンが空になり，他方のビンを使用し始めたと
きに一つのビンの容量を発注する方法」（JIS Z8141）である．発
注点と発注量を同じ1ビン分とする発注点方式であり，ダブルビン
方式，ツービン法，二棚法，2（ツー）ボックスシステムなどとも
呼ばれる．
この方式は2箱分の保管場所が必要になるので，単価が安い小物
部品などに適しており，体積が大きい品目にはあまり適さない．
