a.保管場所の管理 在庫品の保管場所を決める方法は大きく
次の二つがある．
.固定ロケーション（fixed location）方式 品目ごとに
それぞれ固定した保管場所を決めておく方式のこと．管理が容易で
有効な方法であるが，品目数が多く，在庫量の変動が大きい場合に
はスペース効率が悪くなる．
.フリーロケーション（free location）方式，（ランダムロ
ケーション方式) 入庫のつど適切な空きスペースに保管する方
式のこと．扱う品目が多い場合，特に製品ライフサイクルが短い品
目の場合に保管スペースを有効活用できる．保管場所の決め方や保
管場所と数量の記憶が必要になり，コンピュータの支援が必要にな
ることが多い．
b.在庫の数量把握：棚卸（inventory stocktaking) ある時
点における在庫保有高を品目ごとに調査し，台帳の残高を確定する
手続きを在庫棚卸という．在庫品は，保管中に目減りや破損，陳腐
化・劣化等の減耗が生じるため，棚卸によって現在の手持ち数量を
把握し，これらを金額的に評価して正しく記録しなければならな
い．棚卸の実施方法には，現品調査を行わずに帳簿上の在庫数量を
検討する帳簿棚卸と，実際に現品の数量・価格・品質等を調査する
実地棚卸があり，また，実地棚卸の実施方法には次のものがある．
.一斉棚卸 期末など，特定の日に全品目の在庫をいっせ
いに調査し，在高を確定する方法であり，この方法が最も望ましい
が，調査に多くの手間と時間がかかる．
.分散棚卸 品目をいくつかに分け，対象期間中に分散し
て棚卸する方法で，棚卸作業を平準化できる．循環棚卸法はその代
表的なものである．
c .在庫の評価 在庫は棚卸資産として評価される．同じ材
料・部品であっても取得価格に相違があり，保管や運搬中の目減り
やき損，陳腐化・劣化等による減耗をも考慮して価値評価する必要
がある．おもな評価方法には次のものがある．
.先入れ先出し法（FIFO:first - in first -out method)
先に受入れしたものから順次払い出したものと仮定してその価値を
評価する方法．この方法によると，在庫在高が現時点の時価に近い
価値として評価されることになる．
.後入れ先出し法（LIFO:last -in first - out method) 先
入れ先出し法とは逆に，最も新しく受入れたものから順に払出した
ものと仮定して評価する方法．この方法によると，消費した材料や
部品の価値を現時点の時価に近いもので評価することになるが，在
庫在高は古い受入れ価格で評価することになる．インフレ時や価格
変動が大きいものの単価計算に向いている．
.移動平均法 単価の異なる物を受入れるごとに，受入れ
額とその時点の在庫額の和を受入れ数量と在庫数量との和で除し
て，平均単価を求める方法．この方法は，物を受入れるごとに平均
単価を計算する必要があり手間がかかる．また，割切れない場合に
は，若干の誤差が生じることがある．
.総平均法 単価の異なる物を受入れた場合に，月初在庫
額と月間受入額の和を月初在庫数量と月間受入数量との和で除して
総平均を求め，これを単価とするもの．月単位の移動平均法ともい
える．この方法は，受入れ，払出しを月でまとめて計算するので事
務の簡素化につながる．この方法も，割切れない場合には，若干の
誤差が生じることがある．
.予定価格法 過去の受入れ価格に現在の受入れ価格や将
来の予想価格を考慮して，予定の単価を決めて用いる方法．予定価
格は，通常1期間は据置くが，時価との差異が大きくなると改訂す
ることが必要である．この方法によると，予定価格法で求めた単価
から得られた期末在庫高と，実際の購入単価で求めた期末在庫高と
の間に差異が生じるため，その処理が必要となる問題はあるが，払
出単価の据置や1箇月分の受入れ，払出しを一括計算することによ
り事務の簡素化につながるメリットがある．
.その他の方法 ほかにも，個別法（実価法)，単純平均
法（算術平均法)，最終仕入原価法等がある．

g.流動数曲線（図3･28) 生産工程において，仕掛品の
滞留状況を知るための分析であり，生産工程への日々のインプット
数量と，アウトプット数量の累積グラフを描き，この差を見ること
によって仕掛品の滞留状況を知ろうとするものである．
横軸に日付，縦軸に累計数をとり，日々の投入数量の累計線と
日々の完成数量の累計線を描く．こうすると，
(１)二つの累計線の縦の差から仕掛数量が読み取れる（投入数
量の累計と完成数量の累計の差が，工程内に存在する仕掛数量であ
る)．
(２)二つの累計線の横の差から生産リードタイムが読み取れる
（投入されたものの完了日は，投入数量の累積と完了数量の累積が
等しくなる日を見ることによってわかる)．
この流動数曲線によって複数の工程にわたる進度状況を見ること
ができ，工程の進度管理，仕掛品管理，リードタイムの管理ができ
る．
