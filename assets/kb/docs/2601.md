# 3・2・3 サブマージアーク溶接(SAW : submerged arc weld

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

ing)-a. 原理図3・37に示すように溶接部にあらかじめ粒状フラックスを散布しておき、その中に電極ワイヤを送り込み、母材とワイヤとの間でアークを発生させ、ワイヤの溶ける速さに合わせてワイヤを送給して溶接する自動溶接法で、溶接中にアークが見えないので潜弧溶接とも呼ばれている。アークの周囲のフラックスは溶融スラグとなって溶融金属を保護する。また、溶融金属との化学反応によって溶接金属の品質を向上させると同時に、安定で滑らかなビード面を形成する。残ったフラックスは吸引器で回収し再利用される。電源には交流または直流(定電流または定電圧電源)が用いられる。サブマージアーク溶接の利点は、(1)大電流が使用できるので溶着速度が大きい(図3・38)、(2)溶込みが深い、(3)ヒューム、有害光線がなく作業環境がよい、(4)ビード外観が美しい、などである。一方、(1)溶接姿勢は下向き、横向きに限定され、かつ複雑な溶接線には適用できない、(2)開先精度が要求される、(3)設備費が高い、といった欠点も有しており、これらの特性をよく踏まえて適用することが必要である。b. ワイヤ径1.2~6.4mmのソリッドワイヤがほとんどで、チップとの通電性および防錆の観点から表面に銅めっきを施しているものが一般的である。JIS Z 3351:1999に炭素鋼および低合金鋼用サブマージアーク溶接ソリッドワイヤの化学成分が規定されている。c. フラックス被覆アーク溶接棒の被覆と同様に、アークの安定化、アーク周辺のシールド、脱酸精錬作用、合金添加作用などの機能がある。フラックスはその製造方法により、溶融フラックス(fused flux)とボンドフラックス(bonded flux)に大別される。JIS Z 3352:1988に炭素鋼および低合金鋼用サブマージアーク溶接フラックスの化学成分が規定されている。溶融フラックスは原料鉱石を所定の配合比で混合し、アーク炉中で約1300°C以上で溶解、凝固後粉砕して適当な粒度に整えたフラックスで、ガラス状のものと軽石状のものがある。成分は種々の金図3・37 サブマージアーク溶接略図属酸化物やフッ化物からなっており、冶金的にはほぼ中性である。このフラックスの特徴は、(1)耐吸湿性が良好である、(2)低電流域から中電流域(700~1000A 程度以下)での作業性がよい、(3)フラックス粒度の調整により広範囲の電流が利用できる、などであり、わが国では幅広い産業分野で最も一般的に用いられている。ボンドフラックスは原料鉱石粉や合金粉に水ガラスを粘結剤として加え、原料が変質しないよう比較的低温(400~550°C)で焼き固めたものである。このフラックスの利点は、(1)脱酸剤や合金元素の添加が可能で、溶接金属の化学成分の調整が容易である、(2)1000A 以上の高電流でも作業がしやすい、(3)溶融フラックスに比べフラックス消費量が少ない、(4)アークの安定性がよく、狭開先内での溶接作業性もよい、などである。その一方、耐吸湿性が劣るため材料の保管管理に注意を要すること、また溶接条件により溶接金属の成分が変動しやすいなどの特性があり、使用に際しての配慮が必要である。このフラックスの適用範囲は広いが、代表的な適用例としては造船の板継片面溶接、建築ボックス柱の角継手溶接などがあり、溶接能率の向上に貢献している。なお、これ以外に、ボンドフラックスと同様な工程で製造され、700~1000°Cの高温で焼成が行われる焼結フラックス(sinteredflux)があるが、わが国ではほとんど使用されておらず、かつJISZ 3352でも規定されていないため解説は割愛した。d. ワイヤとフラックスの組合せ適切なワイヤとフラックスを組合わせることにより、所定の性能を備えた溶接金属や溶接特性が得られる。組合せの選定に当たっては一般に次の諸点を考慮する必要がある。(1) 母材の種類と溶接継手形状・寸法。(2) 適用溶接施工法と溶接条件。(3) 溶接部に求められる性能。JIS Z 3183:1993にワイヤとフラックスの組合せにより得られる炭素鋼および低合金鋼用サブマージアーク溶着金属の化学成分、機械的性質が規定されているので参考にされたい。e. 溶接施工上の注意アーク点が見えないサブマージアーク溶接で、安定した品質の溶接部を得るためには高精度の開先管理が求められる。また、通常開先加工は自動ガス切断または機械切削で行われるが、ガス切断時のドロスや機械切削時の油脂類ほかミルスケール、赤さびなど溶接欠陥の原因となる異物は、溶接前にきれいに除去しておくことが必要である。フラックスが吸湿していると、割れ・気孔などの欠陥につながる。したがって、溶接前に溶融フラックスは150~350°C、ボンドフラックスは200~350°Cで1時間程度の乾燥を施す必要がある。電流・電圧・速度といった溶接条件のみならず、開先形状・ワイヤや母材の傾斜・フラックス散布高さといった施工条件もビード断面形状をはじめ、ビード外観、溶接金属の化学成分、機械的性質に影響するので、事前テストなどで適正条件を把握したうえで適用することが望ましい。f. 高能率溶接法(high efficient welding method)より高溶着を目指した各種高能率溶接施工法が提案実用化されている。(1) 片面自動溶接法(automatic one side welding):厚板の突合せ溶接を両面溶接とすると構造物の反転のために大形設備が必要となり、かつ労力・時間も要するが、これを合理化したのが片面自動溶接法である。この方法では図3・39に示すように裏面に裏当て材を当て、表面から溶接するだけで裏面にも適度な裏波を形成させるため、裏面からの溶接は必要としない。裏当ての方式としては、大型の装置によるものと簡易な裏当て材を利用するものとがあり、種々の裏当て材やフラックスなどが開発実用化されている。(2) 多電極溶接法(multiple wire welding method):これは2本以上の電極を溶接線方向に一列に配し、それぞれ独立の電源に結線して溶接する方法で、高い溶着速度が得られる。上述の片面自動溶接では4電極による高能率溶接施工も実用化されている。図3・38 各種溶接法の溶着速度の比較図3・39片面自動溶接法の一例図3・40ガスシールド消耗電極式アーク溶接法(炭酸ガスおよびミグ溶接)(3) 帯状電極溶接法(band arc welding method):ワイヤの代わりに帯状電極を用いて溶接する方法で、溶着速度が大きく能率的である。幅が広く扁平なビードが得られ、溶込みが浅くて希釈率が少ないなどの特徴があるため、主としてステンレス鋼の肉盛溶接に利用されている。(4) 狭開先溶接法(narrow gap welding method):特に厚板溶接の際、狭開先化による開先断面積の減少により高能率化を図ろうとする方法である。本施工では狭開先内でのスラグのはく離性が重要で、はく離性のよいフラックスも開発されており、極厚板の1層1パス狭開先溶接が容易に実施されるようになった。以上のほか、開先内にあらかじめ鉄粉や粒状充てん材を散布する方法や、フラックス中に鉄粉を含有させ高溶着速度を達成する方法などが提案実用化されている。
