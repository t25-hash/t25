# 3・6・6 在庫管理の将来

> **章**: 第3章 管理システム
> **節**: 3・6 在庫管理

情報技術の発展によって大量データを迅速に処理することや，情
報ネットワークの普及によって必要な情報をリアルタイムに伝達す
ることが可能になったことが在庫管理のあり方を大きく変えてい
て，さまざまな経営管理システムや技法を生み出してきた．それが
る．部品展開を中心として，生産指示，進度管理，納期管理までを
結果として日本の品質管理の水準を押上げることになっている．
行うMRP（material requirements planning，資材所要量計画）
