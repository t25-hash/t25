# 3・2・4 ガスシールドアーク溶接

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

これにはアルゴン、ヘリウムなどの不活性ガスまたはこれらに少量の活性ガス(炭酸ガス、酸素など)を加えたガス雰囲気中で行うイナートガスアーク溶接(innert gas shielded arc welding)と、主として炭酸ガス雰囲気中で行う炭酸ガスアーク溶接(COgasshielded arc welding)とが含まれる。そして溶接中に電極が溶けて溶接金属になる方式のものを消耗電極式、そうでないものを非消耗電極式といっている。a. ガスシールド消耗電極式アーク溶接法。 原理図3・40に示すようにアルゴンや炭酸ガスなどのシールドガス中で、ワイヤと母材との間にアークを発生させ、ワイヤが溶融するにつれてワイヤを送給し、溶接していく方法である。シールドガスとしてアルゴンなどの不活性ガス(数%程度のCO,Oを混合したものを含む)を用いる場合がミグ(MIG)溶接であり、炭酸ガスを用いるのが炭酸ガスアーク溶接である。そして両方のガスを適当な割合で混合したガスをシールドガスとするものをマグ(MAG)溶接と呼ぶ。なお、ガスシールド消耗電極式アーク溶接はGMA(gas metal arc)溶接ともいわれている。. ミグ溶接[MIG(metal inert gas)welding]不活性ガスは温度が上昇しても他の元素とは全く反応しない。使用される電極ワイヤはソリッドワイヤが大部分であり、直流でワイヤをプラスにすれば溶滴の小さいスプレー移行となり、アークの安定性がよい。アルミニウム合金においてはシールドガスに純アルゴンを用いるが、アークによるクリーニング作用(cleaning action)によって母材表面の酸化皮膜を除去しながら溶接が進められる。鋼の溶接時3―53(a) パルスアーク方式(b) 短絡アーク方式図3・41 溶滴の強制移行にはアークが不安定となりやすいので、酸素または炭酸ガスを数%程度混合してシールドガスとする。ビード形状など作業性を改善すべく混合量を10~20%と多くすればマグ溶接[MAG(metalactive gas)welding]になる。ミグ溶接はステンレス鋼、耐熱合金、銅合金(厚さ約6mm以上)、アルミニウム合金などに利用され、高品質の溶接部が得られる。溶込みは深い。高電流密度でのミグ溶接の作業性は優れているが、電流を下げると溶滴の移行形態がグロビュール状となり、薄板や上向き姿勢の溶接などは不可能になる。それを克服すべく開発されたのがパルスアーク方式と短絡アーク方式である。図3・41(a)がパルスアーク溶接(pulsed arc welding)であるが、低入熱(小さい平均電流)でスプレー移行させるようにパルス電流を加える。パルスが加わったときのピーク電流は臨界電流(それ以上の電流値でスプレー移行をする境界電流)よりも十分に高くなっている。このようにして強制的にスプレー移行をさせれば低電流であっても作業はしやすい。アルミニウム合金や特殊鋼薄板などの溶接に適用されている。短絡アーク溶接(short circuiting arc welding)の要点を図3・41(b)に示す。薄板などの溶接時に、あるワイヤ径に対して電流を臨界電流以下にすると溶接できなくなるので、図3・41のように、短絡移行、アーク発生という現象を繰返し、ピンチ効果と溶融池の表面張力を利用してワイヤの溶滴を移行させるのが短絡アーク溶接である。この方法によれば入熱が下げられ、薄板の溶接が可能になる。なお、この方法は炭酸ガスアーク溶接でも適用できる。短絡回数は溶接条件、シールドガス組成、電源特性などに左右される。狭開先溶接は厚板を効率的に溶接する方法である。母材にI形の狭い開先を準備し、特殊なワイヤガイドを取付けてミグ溶接をする。使用するのは0.9~1.6mm径のソリッドワイヤであり、シールドガスには20~50%の炭酸ガスを混合したガスを用い、電流値は200~300A とする。ただし、より線ワイヤのような場合には500A を通電する。1層1パスで細いビードを重ねていくので、溶接部の衝撃値など機械的性質は優れており、全姿勢溶接もできる。軟鋼のみならず低合金鋼にも適用されている。. 炭酸ガスアーク溶接シールドガスとして用いる炭酸ガスは、高温で分解して酸素を発生するので多少の酸化性を示す。したがって、アーク中での酸化を防止するため、ワイヤにはシリコ3―54加工学・加工機器ン、マンガンなどの脱酸剤が含まれている。炭酸ガスアーク溶接は被覆アーク溶接に比べて、(1)電流密度が大きいので能率的である、(2)溶着効率がよい、(3)水素量が少ないので溶接性がよい、(4)経済的である、などの利点があるが、風速2m/s以上のときにはシールド性が悪くなる、梨形ビードになりやすい、行動範囲が狭い、溶接機が高価である、などの欠点もある。しかし、利点のほうが大きいので用途は次第に拡大されている。ソリッドワイヤは多少スパッタが多いので、それを改善するためフラックス入りワイヤが実用化されている。薄板には短絡アーク溶接が行われている。b. ガスシールド非消耗電極式アーク溶接法ここではティグ溶接[TIG(tungsten inert gas)welding]についてのみ述べる。ティグ溶接は図3・42に示すごとく、不活性ガス雰囲気中でタングステン電極と母材との間にアークを発生させて溶接する方法である。溶加材(溶接棒)は手動または自動的にアーク内に挿入する。入熱と溶着量とは独立に制御できる。電極プラスの場合には優れたクリーニング作用を示す。ただし、溶込みは浅い。電極マイナスのときには溶込みが深くなる。ミグ溶接に比較すれば溶接能率は劣るが、溶接継手としては極めて優れている。ステンレス鋼、耐熱合金、銅合金(厚さ約6mm以下)、アルミニウム合金、60kg 級以上の高張力鋼などの溶接に利用されている。
