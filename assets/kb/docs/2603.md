# 3・2・5 プラズマアーク溶接(plasma arc welding)

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

a. 原理と特徴プラズマアーク溶接はアークの熱的ピンチ効果(アーク柱を周囲から拘束・冷却すると、アーク柱が緊縮すると同時にプラズマ温度が上昇する現象)を利用し、1万~2万°C以上の集中高温プラズマ流を形成させ、融接する方法である。アーク発生は通常、ティグアーク方式が採用され、水冷銅ノズルとアルゴンガスとによってアーク柱を絞り、高温プラズマを定常的に噴出させる。プラズマ流発生方式には図3・43に示す3方式がある。プラズマジェット方式では電極と拘束ノズル(銅製)との間に発生させた高温のプラズマアークをノズルから噴出させる。このプラズマアーク熱を利用して金属材料の溶接に用いられる。プラズマアーク方式では母材が一方の電極となっており、アーク図3・42 ティグ溶接法(a) プラズマジェット方式(b) プラズマアーク方式は拘束ノズルで絞られる。入熱密度が大きく、熱効率も高いので、極めて板厚が薄い場合を除いて金属材料の溶接に利用されている。中間型は上記2方式を併用したものであり、小電流域での細径アークが安定に維持できる。この方式であれば1A 以下の小電流を通じるので板厚0.1mm程度の薄板でも溶接できる。b. 応用プラズマアーク溶接はティグ溶接に比較して溶込みが深く、その割にビード幅は狭い。厚板の溶接に適しており、タングステン電極の消耗は少ない。厚板溶接時にはキーホール技法が採用されている。この技法は溶融池の前方に、板の表面から裏面まで貫通した鍵穴状の穴をあけながら溶接する方法である。この穴は溶接の進行につれて埋められていく。穴のあくような溶接条件にすれば完全に裏まで溶け込むので、条件の選定は容易である。溶接に先立ち、裏面に適当な裏当て材、水冷銅、ガスシールドを施しておけば良好な裏ビードが形成される(バッキング法)。アルミニウム合金を電極プラスでキーホール溶接すれば、アーク電流がキーホールを通って裏面にも流れるので、裏面にもクリーニング帯が生じる。厚板の突合せ溶接には、バッキング法以外にティグ溶接と同様な施工も行われる。第2層め以降はキーホール技法が実施できないので、大きなノズル径を用いて溶融池のあれを防いだり、溶加材を加えたりする。溶加材は溶滴移行を円滑にさせる。溶接能率向上には溶加ワイヤに通電した形で使用されることもある。小電流プラズマアーク溶接は中間型の方式のものを用い、シールドガスには、通常、アルゴンに水素かヘリウムを混合したものを用いる。プラズマアークとミグ法とを併用したプラズマ・ミグ溶接ではワイヤの溶融速度が大きく、スパッタも少ない。
