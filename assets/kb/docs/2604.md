# 3・7・2 TQMの方法論

> **章**: 第3章 管理システム
> **節**: 3・6 在庫管理

システムは，資材在庫管理から生産管理・工程管理を統合的に行うものであり，また，生産を中心に，受注から出荷納入までをコンピfacturing）も展開されている．
さらに，軍事用資材の補給活動を円滑に行うための支援システムとして開発されたCALS（com-設計・資材調達・製造など企業活動全体を取込む概念（continu-達から供給までのコンピュータ化の重要な概念であり，現在ではcommerce at light speedへと変ぼうしている．
この進展には商品の受発注や取引を電子的に行うための電子データ交換：EDI （elec-tric data interchange）の進展が大きく寄与している．
さらに，顧客ニーズの多様化，短納期化といった社会状況の変化進展しつつある．
これによって，異なる企業間相互にまで連携が進み，調達から供給までの全過程にわたって相互協力する最適ロジスティックシステムの構築が要求されている．
そのためには，情報の共有化が不可欠であり，流通や保管に関係する情報の標準化も進められ，必要に応じて情報の書込みや読出しができる電子ラベル（ICタグ，RFID）の開発・応用に関する研究も進められている．
この中心的役割を果たすのが在庫管理である．
つから構成される．
以下でこれらの解説をする．
QC手法については3･7･3項で述べる．
a.TQMのフィロソフィー フィロソフィーとは，品質管理を進めるうえでの根底にある考え方で，品質の意味の浸透，全員参加，改善，後工程はお客様，管理のサイクル（PDCAサイクル)，プロセス管理，事実に基づく管理，人間性尊重などである．
品質という言葉は，製品品質だけを指すのではなく，仕事の質，経営の質をも意味することが浸透しており，そのためにあらゆる業務の改善に品質管理が貢献するとの理解が得られてきた．
また，経験や勘だけに頼るのではなく，科学的分析に基づいて問題解決を図ること，すなわち事実に基づく管理が強調されている．
そのために，QC七つ道具，実験計画法，回帰分析，多変量解析などの統計的方法や抜取検査，サンプリングなど統計理論に基づいたさまざまな技法が多用される．
このことを強調するために，戦後発展した品質管理を統b.コアマネジメントシステム コアマネジメントシステムは，品質を重視した経営を行い，先のフィロソフィーを具現化するために活用される経営管理の仕組みである．
特に，後述する日常管理，方針管理，機能別管理は，その根幹をなす3本柱である．
また，品質を達成するための中核の仕組みとして品質保証システムがある．
これには，品質を保証するための体系，組織とともに，品質改善活動，重要品質問題管理システム，品質情報の活用などの具体的な活動が含まれる．
さらに，狭義の品質（製品・サービスの品質）のみならず，原価，量，納期，安全など製品・サービスに関わな役割を担うであろう．
