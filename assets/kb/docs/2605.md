# 3・2・7 エレクトロスラグ溶接(electroslag welding)

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

エレクトロスラグ溶接法は、図3・47に示すように溶接ワイヤと母材間を、溶融スラグを介して通電させ、この部分でのジュール発熱により電極と母材を溶融させる立向溶接法である。これは溶融スラグが高温になると電気伝導性がよくなることを利用している。溶接の進行に応じて上昇する水冷銅当て金で溶融金属とスラグの流れ落ちを防ぐ。厚板の場合、複数のワイヤを供給する場合もある。スラグは溶接金属表面に一定の厚さで付着するため、溶融池内のスラグ量を一定に保つべく、ホッパに蓄えたフラックスを一定速度で供給する。溶接電源としては、垂下特性もしくは定電圧特性に近い交流電源が用いられる。標準の交流、直流アーク溶接電源が使用可能である。消耗ノズル式エレクトロスラグ溶接はエレクトロスラグ溶接装置を簡易化したもので、ワイヤ送給ノズルを溶接長に相当する長さの3―56加工学・加工機器図3・47 エレクトロスラグ溶接法図3・48アーク溶接とエレクトロスラグ溶接の熱サイクル(板厚100mm)鋼管で溶接の進行に伴い消耗する固定ノズルを用いる。水冷銅当て金も全溶接長にわたって固定しておく。この溶接法は炭素鋼をはじめ、低合金鋼ステンレス鋼など各種の鋼材に適用可能である。溶接ワイヤやフラックスについてのJIS規格はないが、一般的にはワイヤはサブマージアーク溶接用のワイヤが用いられ、フラックスは、アークが発生しにくく、電気伝導度がよく適度の流動性を必要とし、サブマージアーク溶接用のフラックスに近い組成のものが使用されている。この溶接方法の特徴は、厚板(約25mm以上)の溶接を1パスの溶接で仕上げることが可能であり、高能率なことが挙げられる。また、開先形状も単純なI形で、開先面もガス切断のままでよく、このことも厚板の溶接に適している。溶接による角変形はほとんどない。溶接部の特性は、溶接金属への大気ガス成分の吸収はほとんどなく、溶接金属の静的な機械的性能は良好である。しかしながら、被覆アーク溶接や炭酸ガスアーク溶接法による溶接部に比べ、衝撃性能が悪い傾向にある。これは入熱が大きく図3・48に示すように冷却速度が遅く、溶接金属や熱影響部の結晶粒が粗大化するためといわれている。これに対し、母材、溶接金属とも結晶粒の微細化を図る改良がなされている。
