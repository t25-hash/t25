# 3・2・8 電子ビーム溶接(electron beam welding)

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

電子ビーム溶接は、高エネルギー密度の電子ビームを材料表面に照射し、溶融・溶接する方法である。図3・49にその原理を示す。真空チャンバ内で、熱陰極から放出した熱電子を陽極との間に印加した高電圧により加速し、集束レンズ内の磁場により材料表面付近に収束する。収束された電子ビームのパワー密度は10~10W/cmにも達し、アークと比べてけた違いに大きい。このため、電子ビームが照射した位置では瞬時に材料の溶融と蒸発が起こり、蒸発時の反跳力により、図3・50に示すようなキーホール(keyhole、図3・49電子ビーム溶接法の原理(a) 電子ビーム溶接現象(b) 溶込み形状図3・50 電子ビーム溶接現象と溶込み形状ビーム孔)と呼ばれる細長い空洞が形成される。キーホールは電子ビーム溶接にとって非常に重要な存在であり、この内面に電子ビームが当たることにより、材料の内部まで熱源が投入され、図3・50(b)に示すように幅が狭く非常に深い溶込みが得られる。これが電子ビーム溶接の最大の特徴であり、すでに100mmを超える鋼板の1パス溶接が実用化されている。溶融幅が狭いため、熱影響部が狭く、かつ熱変形の小さい高精度な溶接が可能になる。このため、溶接後製品の再加工を行う工程が省け、開発当初から自動車用トランスミッションギヤ等の高精度溶接に本溶接法が適用されてきた。また集中した熱源を用いるため、アーク溶接では困難な高融点材料や高熱伝導率をもった材料の溶接も可能である。さらに特筆すべき点は、本溶接法が非常に制御性に優れていることである。電子ビームは磁場により簡単に軌道が修正できるため、図3・49に示した偏向コイルに流す電流を変化することにより、電子ビームの照射位置を高速で自在に制御することができる。また、ビーム電流や焦点位置も同様に高速で制御が可能である。一方、大気中では電子ビームが散乱するため、本溶接法は真空雰囲気で行う必要がある。これは、ガスと反応しやすい活性金属等の溶接には適しているが、通常の材料では、真空チャンバのサイズにより非溶接物の大きさが限定されたり、生産性の低下を招くため、本溶接法の最大の欠点となっている。このため、初期においては小型部品の精密溶接にもっぱら電子ビーム溶接が使われてきたが、1970年代後半から巨大チャンバを備えた大出力溶接機が開発され、極厚板の溶接が行われるようになった。厚板溶接における問題点は、図3・50に示したようなポロシティ(porosity)、凝固割れ(solidification cracking)、スパイク(spike)等の欠陥が発生しやすくなることである。これらは、キーホールおよび溶融金属の挙動と密接に関連して発生することが知られており、ビームオシレーシ表3・13 レーザの種類と特徴レーザの種類波長m発振効率%長所短所気体レーザCOレーザ10.610~20高出力化が容易ビーム品質が高い比較的安価ランプ励起YAG レーザ(ロッドタイプ)1.062~4ファイバ導光が可能発振効率が低いCOレーザよりビーム品質が劣るLD 励起YAG レーザ(ロッドタイプ)1.0610~20ファイバ導光が可能ビーム品質はランプ励起YAG レーザより高いが、COレーザより劣る固体レーザディスクレーザ1.0付近レーザ媒質に依存20~30ファイバレーザ半導体レーザ0.8~0.9530~50ョン等を用いてこれらを制御することにより抑制することができる。もう一つの欠点は、装置が高価になることである。このため、高い品質と信頼性が要求される高付加価値製品の溶接や量産が可能な製品に多く用いられる。量産品では、予備排気室を設けた連続生産可能な専用機を使うことにより、排気時間のロスを最小限にすることができ、生産性の高い溶接が可能となる。
