# 3・2・9 レーザ溶接(laser welding)

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

レーザは、指向性が高く位相のそろった単色光であり、集光性が極めて高い。このため、電子ビームと同等のパワー密度を得ることができ、次のような特徴をもった溶接が可能となる。(1) キーホール溶接(2) 幅が狭く、深い溶込み形状(3) ひずみの小さい高精度溶接(4) 狭い熱影響部(5) 高融点材料の溶接が可能レーザ溶接が電子ビーム溶接と異なる最も大きな特徴は、大気中で溶接ができる点にあり、適用範囲が圧倒的に広くなる。一方、レーザは金属表面で反射しやすく、電子ビーム溶接と比べて熱効率が悪い。しかし、キーホール溶接を行った場合には、キーホール内部でレーザ光が繰返し反射し、この間に吸収が大きくなるため、70~80%程度まで熱効率を上げることができる。溶接に用いられるレーザとしては、COレーザ(COlaser、気体レーザ)、各種固体レーザ、半導体レーザ(laser diode)がある。固体レーザで最も一般的なのがNd:YAG レーザ(Nd:YAG laser、ロッドタイプ)であり、これにはランプ励起とLD(半導体レーザ)励起の2種類がある。また、ロッドタイプレーザの欠点であるビーム品質の低下を改善した固体レーザとして、ディスクレーザ(disk laser)やファイバレーザ(fiber laser)がある。それぞれの特徴を表3・13に示す。COレーザはCO,He,Nをレーザ媒体とした気体レーザであり、比較的低コストで高出力化が容易にできる。また、ビーム品質(ビーム集光性)が高いこともCOレーザの大きな特徴である。このため、最も早くから溶接に用いられてきており、現在では50kW の高出力発振器が開発されている。反面、波長が10.6 ファイバレーザの原理源に半導体レーザを用いているため、発振効率が高い。またレーザ媒質の熱ひずみを非常に小さくできるため、ビーム品質の高いレーザを発振することができる。これと同様に、レーザ媒質を細いファイバ状(図3・53内部コア)にし、この中に励起用の半導体レーザを導光することにより発振するファイバレーザ(図3・53)も2000年代になって高出力化が進んだ。ディスクレーザと同様に発振効率が高く、ビーム品質に優れており、いずれも新しい固体レーザとして、広く溶接に用いられることが予想される。唯一の欠点は、半導体レーザを励起源として用いているため、装置が高価になる点であるが、半導体レーザの出力当たりの単価が減少するとともに、この問題も解消されると予想される。一方、小型で発振効率の高い半導体レーザも1990年代後半に高出力化が進み、溶接熱源として用いられている。非常に小型のため、ロボットに直接取付けて三次元形状の加工ができる。また、発振効率が最も高く、価格も最も廉価である。反面、ビーム集光性が他のレーザと比べて劣るため、ろう付けや表面改質、薄板の熱伝導溶接におもに用いられている。最近ではコア径の小さなファイバにも導光できるものが開発されており、ビーム品質の向上がさらに進めば、より用途が広がると期待される。レーザ溶接は、現状ではおもに板厚が数mm程度の薄板の高速高精度溶接に用いられている。溶接変形と熱影響の少ないことが最も大きな特徴であり、特に自動車産業において実用化が進んでいる。一方、最近では各種レーザ発振器の出力が飛躍的に高くなってきており、中・厚板の溶接にも適用範囲が拡大しつつある。事実、ヨーロッパでは1990年代後半から造船分野でレーザ溶接が実用化されている。厚板溶接における問題点は、ポロシティや凝固割れ等の溶接欠陥の発生にある。特にレーザ溶接は大気中での溶接であるため、シールドガスを巻き込みやすく、ポロシティが顕著に発生する。これらもキーホールの挙動と密接に関連して発生することが指摘されており、基礎的な研究成果を踏まえて防止法を確立していくことが必要である。厚板の溶接に対しては、レーザとアークを組合わせたハイブリッド溶接(hybrid welding)が有望である。レーザ照射部から発生する金属蒸気がアークの安定性と集中性を高め、高速で深い溶込みを得ることができる。アーク側からフィラーワイヤを添加することができるため、レーザ単独では難しい開先ギャップが開いた部分の溶接も可能になる。また、フィラーワイヤの成分を調整することにより、溶接金属の特性を改善することができる。比較的廉価で厚板の溶接ができる方法として期待されている。
