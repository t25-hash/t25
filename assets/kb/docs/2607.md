# 3・8・1 設備ライフサイクルとその管理

> **章**: 第3章 管理システム
> **節**: 3・8 設備管理

設備管理の目的は，設備に求められる機能が最大限に発揮される
設備のライフサイクルとは，図3･29に示すような，設備の計画
から，設計・製作，据付・試運転，運用，更新・廃棄までの設備の
一生を指している．計画段階では，経営，製品，生産管理，工程な
どに関するデータを基に，機能，経済性，環境影響などの面での評
価を行いながら，設備の基本設計を行う．この際，設備の一生すな
わちライフサイクルを考慮することが重要である．すなわち，ライ
フサイクルコスティング（life cycle costing) ，ライフサイクル
アセスメント（LCA:life cycle assessment ) などの手法を用
い，設備ライフサイクル全体の評価に基づいた計画立案が求められ
る．
設計・製作および据付・試運転段階では，問題を起こしにくい設
備設計と，迅速な設備立上げが課題となる．設備設計においては，
設備が要求された機能を達成するようにするだけでなく，製造性，
信頼性，保全性，廃棄性など，設備ライフサイクルの各局面での効
率向上を目指した設計を行う必要がある．このためにMP設計
（MP design）の推進が重要である．MPとはmaintenance pre-
vention（保全予防）の略称で，1950年代にGEが提唱した生産保
全の中で示された概念である．過去の設備運用経験から得られた情
報を設計に生かすことで，不良や不具合の出にくい設備を作る活動
である．また，近年の製品寿命の短期化とビジネススピードの向上
に対応するために，設備立上げまでの期間短縮が強く要求されるよ
うになっている．このためには，事前の設計審査（DR:design
review）において，十分不具合を取り除いておくことが基本とな
るが，さらに，初期流動期間（設備の据付・試運転から，安定稼働
に至るまでの期間）において，開発・設計部門と運転・保全部門の
協調体制のもとで，効率的なデバッグを行うことが必要である．ま
た，この段階で，運転，保全に関わる基準類の整備，教育・訓練を
行うことも必要である．
設備の運用段階では，安全の確保，環境負荷の低減といった条件
を満たしながら，設備の能力を最大限に引出すように，保全管理を
行う必要がある．そのためには，設備を構成する要素ごとに，そこ
で生じる可能性のある劣化・故障の特性とその影響の評価を行い，
その結果に基づいて適切な保全方式を定め，計画的に保全活動を進
めていくことが必要となる．さらに，設備の更新・廃棄段階では，
設備の運用履歴に基づく余寿命評価や再使用性の評価を行い，可能
な限り廃棄量を少なくできるような更新・廃棄計画を立案する．
以上述べたように，設備管理が扱う内容は広範にわたるが，以下
では，運用段階における保全管理を中心にその考え方，手法などを
述べる．
