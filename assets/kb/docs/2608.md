# 3・2・10 ガス溶接(gas welding)

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

各種のガス炎を用いて、ガスの燃焼時に生じる高温を利用して、金属を溶接する方法を総称してガス溶接という。ガス炎に酸素アセチレン炎(oxyacetylene flame)を使用することが最も多く、一般にはガス溶接といえば酸素アセチレン溶接(oxyacetylene weld-ing)を指すものと考えてよい。これは、ガス溶接に要求される炎の性質、すなわち、火炎温度が高い、燃焼速度が大きい、熱量が大きい、溶融金属に化学的影響を与えないことを満足する炎が得られる燃料ガスは、実用的にアセチレンガスだけであるからである。本法は電源設備が不要で、装置の機動性に富み、場所を選ばず容易に溶接作業を行うことができるので、比較的小規模の溶接に多用されている。酸素アセチレン炎の理論的な炎温度は3300~3500°Cとなり、各種ガス炎中最高の温度である。ガス炎のエネルギー密度はその他の溶接熱源に比べて小さいため、溶接能率がよくない、溶接ひずみが大きいなどにより、他の溶接方法の発達につれて、その適用範囲が縮小した。しかし、薄板、小径管の溶接等には適しており、また各種の修理溶接には広く使用されている。酸素アセチレン溶接トーチ(吹管)には、使用するアセチレン圧力により、低圧式トーチ(圧力0.7MPa以下)と中圧式トーチ(圧力0.7~1.3MPa)とがある。日本では、取扱いが容易で安全性に優れている低圧式トーチが広く使用されており、ガス溶接トーチの大半を占めている。ガス溶接では板厚が小さい場合には溶接棒を用いないこともあるが、一般には溶接棒を用いる。軟鋼用溶接棒としてはJIS Z3201:2001が規定されている。軟鋼以外の金属に対しては母材と同じ共金を用いることが多いが、銅および銅合金、あるいはブレーズ溶接用の規格としてJIS Z 3202:1999がある。溶接棒は母材の種類あるいは必要性能によって適切な材質を選ぶことが必要であるが、例えば流動性のような作業性も考慮して選択しなければならない。ガス溶接を行ううえで重要なことは、母材表面の酸化皮膜を完全に除去することと、溶接中に溶接部を酸化させないことである。そのため、母材の金属に適したフラックスが使用される。しかし、軟鋼のガス溶接では、鉄の酸化物がフラックスとしての作用をすることから、フラックスを用いない場合もある。ガス溶接施工法には、前進溶接法と後退溶接法と称される二つの基本的な施工法がある。前者は、母材に対する入熱の調整が行いやすく、かつ溶接操作が容易であるので、通常、3.2mm程度までの薄板の溶接に使われる。また、後者は、溶込みを確認しながら溶接できるので厚板の溶接に適しており、均一で良好な外観の裏波溶接を行うことができる。
