# 3・2・11 テルミット溶接(thermite welding)

> **章**: 第3章 溶接・接合および切断
> **節**: 3・2 融接法

テルミット溶接は,金属酸化物や硫化物を金属アルミニウムで還元するときの発熱反応(テルミット反応,thermite reaction)を利用する溶接方法で,金属酸化物として酸化鉄を用いた場合,溶融鉄を得ることができる.テルミット溶接法を大別すると,反応で生じた溶融金属を接合または肉盛に利用する溶融テルミット法と,反応熱および反応生成物の有する熱エネルギーを加熱源とする加圧テルミット法に分類される.鋼用の溶融テルミット法では,酸化鉄とアルミニウムを主成分とし,溶接金属の品質調整のための合金元素や脱酸剤を配合した粉末(テルミット溶剤)をるつぼ内で反応させ,得られた溶鋼を,あらかじめ接合する継手の周囲に設置したモールド(鋳型)に注入して開先部分を溶接する.なお,溶接継手部分は溶鋼を注入する前に600~900°C程度に予熱しておく.鋼用テルミット溶剤の反応は強烈で,生成する溶鋼の温度は2000°C以上となるが,反応自体は爆発的ではなく,しかも反応開始に約1200°C以上を必要とするため,その取扱いは比較的安全である.テルミット溶接は,大断面積部材を容易に比較的短時間に溶接でき,かつ使用器具が軽量で,大きな電源や加圧力が不要で機動性に優れるといった特徴がある.日本では鉄道用レールのロングレール(continuous welded rails)化のための溶接方法(図3・54)として広く用いられている.また,酸化銅とアルミニウムを主原料とした銅用テルミット溶接も実用に供されている.
