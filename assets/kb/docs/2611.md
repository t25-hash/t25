# 3・3・1 抵抗溶接法の分類

> **章**: 第3章 溶接・接合および切断
> **節**: 3・3 抵抗溶接法

抵抗溶接法(resistance welding method)は被溶接材料を対向する電極によって加圧した状態で、大電流の短時間通電による抵抗発熱を利用して加熱し、通電遮断後も加圧したまま冷却・凝固して接合する方法である。この溶接法の長所としては一般的に次のことが挙げられる。(1) 溶接速度が速く生産性がよい。(2) 溶接変形が少なく品質が安定している。(3) 接合部の板合わせに高加工精度を必要としない。(4) 溶接パラメータが制御しやすく自動化に適している。(5) 溶接棒のような消耗資材を必要とせず低コストである。反面、大入力を必要とするため電源設備が大がかりになること、大加圧力を得るための加圧機構を必要とするため溶接機自体が複雑であるなど不利な点もある。抵抗溶接法の分類は溶接の種類による分類、溶接装置から見た電気的制御方式による分類と加圧方式による分類の仕方がある。これらは制御技術の進展が旺盛であり明確な分類はできないが、おおよそ表3・15のとおりである。抵抗溶接は、板と板あるいは部品と板を重ねて接合する重ね抵抗溶接(lap resistance welding)と、板どうしあるいは棒どうしの端面を突き合わせて接合する突合せ抵抗溶接(butt resistancewelding)に大別される。重ね抵抗溶接にはスポット溶接(spotwelding)とプロジェクション溶接(projection welding)、およびシーム溶接(seam welding)とがある。スポット溶接は電極先端の形状を通常、ほぼ円形(直径数mm)の必要最小限の寸法に加工し、溶接部に大電流を集中して溶接を行う。プロジェクション溶接は被溶接材料の溶接部に突起を打ち出すなどの加工を施し、その部分に溶接電流を集中させて溶接する。シーム溶接は一対の円板状の電極で重ね合わせた被溶接材料を、連続的に加圧・回転して材料を移動しつつ通電・溶接して、被溶接材料を帯状に溶接する方法である。シーム溶接で円板電極からの電流を断続通電することによって不連続で適当なピッチのスポット溶接ができるが、これはロールスポット溶接という。突合せ抵抗溶接にはアプセット溶接(upsetwelding)とフラッシュ溶接(flash welding)がある。アプセット溶接は一対の被溶接材料の溶接面を突き合せ、加圧しながら電流を流し、抵抗発熱により過熱、接合する。フラッシュ溶接は突き合わ表3・15 抵抗溶接法の分類(a) 溶接の種類による分類重ね抵抗溶接スポット溶接プロジェクション溶接シーム溶接突合せ抵抗溶接アプセット溶接フラッシュ溶接(b) 電気的制御方式による分類交流式単相交流式三相低周波式インバータ交流式直流式単相直流式三相直流式インバータ直流式蓄勢式コンデンサ式インバータ式(c) 加圧方式による分類人力加圧式手加圧式足踏加圧式流体加圧式空気加圧式油圧加圧式電動加圧式(サーボモータ駆動)せた一対の溶接面に、溶接開始時点では、電圧を与えながら溶接面を軽く接触させてフラッシュ(火花)を発生させて加熱し、その後大加圧・大電流にてアプセットして接合する。突合せ溶接にはこのほかにバットシーム溶接があるが、これは突合せ溶接とシーム溶接の複合型であり、円板による給電、被溶接材料の移動方法はシーム溶接と同じ方法であり、加圧は突合せ溶接と同様の方法で行う。電気的制御方式は、入力をいかにして効率よく出力し、溶接入熱として使用するかという観点から種々の方式が採用されている。交流出力で溶接する交流式には、単相でそのまま出力する単相交流式、大電流出力を目的とする三相低周波式、相数および周波数を変換するインバータ交流式がある。直流出力で溶接する直流式には単相入力の単相直流式、三相入力の三相直流式、主として三相入力のインバータ直流式がある。蓄勢式は交流入力を整流していったんコンデンサに蓄電するが、そのまま放電して溶接するコンデンサ式と、電流をインバータ制御するインバータ式がある。コンデンサ式は溶接電流を短時間大電流化する目的もあり、インバータ式は溶接電流波形の制御の精密化をも目的としている。加圧方式は手加圧または足踏みによってテコを介してのばね力を利用した人力加圧式と、空気圧、油圧を利用した流体加圧式がある。流体加圧式はバルブにより動作させるものが大半であるが、フラッシュ溶接には電気-油圧サーボ式が一部採用されている。スポット溶接では流体加圧式が最も多く使用されているが、制御技術が進み、最近ではサーボモータ駆動の電動加圧式のいわゆるサーボガンが、ロボットによるスポット溶接に多く適用されてきている。
