･保全部門の計画保全体制づくり
･運転・保全のスキルアップ
･設備初期管理体制づくり
･品質保全体制づくり
･管理間接部門の効率化体制づくり
･安全・衛生と環境の管理体制づくり
活動体制としては，図3･36に示すような職制と一体化した重複
小集団組織と称する全社的な体制をとるところに特徴がある．これ
により，設備の専門家に閉じることなく，全社的な全員参加の活動
を展開する．
b.ロスの把握 TPMでは，生産活動の中に存在するロスを
発見して，それを徹底的に排除することを活動の主眼としている．

c .改善の考え方 TPMでは，主として，生産効率阻害の原
因系である設備に着目して改善を行う．その基本的考え方を示した
のが，図3･39である．設備を使いこなし，使い切ることが目標で
あり，そのためには，設備のあるべき姿を明確にし，現状がそれか
ら外れている場合には復元をすることが基本となる．設備のあるべ
き姿とは，工学的な原理・原則から考えたときの望ましい姿であ
り，ユニットや部品の一つ一つについての形状，寸法，材質などの
属性，および使用条件，環境条件などに関する基準として示される
必要がある．復元とは，現状をあるべき姿に戻すことであり，それ
だけでは故障につながらないような微欠陥も排除することが重要で
ある．これは，微欠陥の集積がチョコ停（choko-tei ，部品交換や
修理などを要しない一時的な設備の停止または空転）を引き起こし
て慢性的なロスを生むからである．このような，復元やあるべき姿
の追求のためには，現象を物理的に解析し，そのメカニズムを明ら
かにする必要がある．このための分析手法をTPMではPM分析
（PM analysis ）と呼んで体系化している ．
あるべき姿が明確になり，また，不具合現象が解明されると，復
元は試行錯誤に基づく調整ではなく，機械的に行える調節によって
達成できるようになる．これにより，設備状態の維持が容易にな
り，高い生産効率を保つことが可能となる．
