# 3・3・2 重ね抵抗溶接

> **章**: 第3章 溶接・接合および切断
> **節**: 3・3 抵抗溶接法

a. スポット溶接抵抗溶接の中で最も多く使用される溶接であり、自動車をはじめ鉄道車両、家庭用電気製品など薄板のプレス成形部品の組付けに適する。スポット溶接を適用する材料は自動車では0.7~2.6mm程度の軟鋼薄鋼板が主体である。しかし、近年は車体の軽量化や防錆能力の向上を目的とし、引張強さ440~780MPaの高張力鋼板やZnめっきなど表面被覆鋼板の使用割合が増している。鉄道車両、航空機ではステンレス鋼やアルミニウム合金にも適用される。スポット溶接は電極で加圧ができ、そこに通電できることが前提となるが、通電の形式により、図3・58のようにダイレクト溶接、インダイレクト溶接とシリーズ溶接に大別される。溶接箇所を個々に加圧、通電するダイレクト溶接が一般的であるが、インダイレクト法では裏面側の板表面に通電痕を残したくない(a) condition)は、電極先端径を決定すると加圧力、溶接電流、通電時間の3条件が基本となる。ナゲット径の大きさは溶接部の強さに直接関係し、ナゲット径が大きいほど強さは増加する。最適なナゲット径の大きさは溶接品質、生産性、作業性、経済性などの点を総合的に考慮して決められるが、ナゲット径が小さすぎるとナゲットの安定性、作業性が悪くなり、大きすぎると溶接設備が大きくなり設備の経済性からは好ましくない。電極チップ先端形状にはフラット型のものとラジアス型のものとがある。いずれもスポット溶接の打点の経過とともに徐々に変化するので一概にいえないが、フラット型のものでは所定のナゲット径の0.9~1.1倍程度の先端径、ラジアス型のものでは50~200R の先端形状のものが多い。電極先端径は、適正な溶接が行われる溶接条件ではナゲット径=0.9~1.1(:電極先端径)の範囲で、ほぼナゲット径と同様とみなしてよい。ナゲット径および電極先端形状・寸法が決まると、溶接条件は電極加圧力、溶接電流および通電時間の組合せで選択可能である。電極加圧力は、適正な値よりも大きすぎると溶接部での電流通路が広がり、電流密度が低下するので発熱しにくくなり、ナゲット径が減少して強さが低下し、小さすぎるとその逆で、通電中に過熱し溶接部より散りが発生する。加圧力は板厚の厚い材料、高張力鋼などの3―61図3・59溶接電流とナゲット径の関係図3・60 ウェルドローブ硬い材料、酸化皮膜などのように電気的に抵抗が大きい表面のもの、精度のよくない部品の溶接の場合は高くする必要がある。溶接電流は過度に大きくすると通電中の過熱により散りが発生する。多少の散りの発生は溶接部の強さには影響しないので問題はないが、過度の散りは強さの減少にもなり、また作業環境的にも好ましくない。それ以上に電流を上げると、材料に接している電極をも加熱して電極と被接合材料が溶着し、溶接作業が中断するトラブルとなる。図3・59中の○印は、=0.8mmの軟鋼板において溶接電流とナゲット径の関係を他の条件を一定にして求めたものである。図3・59中に、それ以上の電流で基準のナゲット径(この場合4.5mm)が得られるナゲット形成限界、溶融金属が飛散する散り限界、接触部が過熱し電極と被接合材が溶着し溶接困難となる溶着限界をそれぞれ矢印で示している。溶接条件はめっきなど表面処理の有無や材料の違いにより影響を受ける。図3・59中の△印はZnめっき層の存在がナゲット形成に与える影響を示すもので、めっきの溶融により通電路が拡大し、めっきのない場合に比べて高い電流を必要とする。また、銅合金を素材とする電極がめっきと容易に軟質の合金を形成し消耗が激しいため、ナゲットを一定水準に保つには厳重な管理が必要である。図3・60は、電極加圧力を一定にして、通電時間を種々変えてこの電流-ナゲット径曲線を求め、ナゲット限界、散り限界、溶着限界を電流と通電時間の関係として求めている。図3・60はウェルドローブ(weld lobe)と呼ばれ、溶接条件の範囲を端的に示しているので現場的な条件設定や異種材料の溶接性比較に有効である。ウェルドローブは加圧力を高くすると高電流側に、加圧力を低くすると低電流側に移動するとともに、それが囲む範囲も変化する。プレス成形精度の影響で板の合いがよくない場合は加圧力を高くする。溶接電流は、散り発生もなくナゲットの生成が安定している散り限界直前が理想的とされている。しかし、量産現場では多数の溶接機の同時通電による電源電圧の降下、電極の消耗による電流密度の低下、板の合いのばらつきなどさまざまな条件変動を補償するため、散り限界を超える電流設定をすることが多い。溶接部の強さは引張せん断強さと十字引張強さにより評価される(試験方法はJIS Z 3136:1999「抵抗スポット及びプロジェクション溶接継手のせん断試験に対する試験片寸法及び試験方法」、JIS3―62加工学・加工機器Z 3137:1999「抵抗スポット及びプロジェクション溶接継手の十字引張試験に対する試験片寸法及び試験方法」を参照)。継手強さは母材の強さに依存するが、鋼板においては、一般に引張せん断強さは母材の引張強さに比例して増加するが十字引張強さはさほど増加しないという特性があり、高強度材にスポット溶接を適用する場合に注意を要する。スポット溶接の継手構造は、図3・61に示すようにラップ形、フランジ形、ヘミング形を基本とし、デザイン、強度、作業性などの要求に応じさまざまな形状がある。構造設計に当たってはガン、電極との干渉、フランジ幅、打点ピッチ(狭すぎると強度、作業性が劣る)、重ね枚数(3~4枚以下)などの制約条件を十分に考慮する必要がある。b. プロジェクション溶接プロジェクション溶接は、被接合物の一方あるいは両方に突起(プロジェクション)を設けるか、または角部を利用して電流および加圧力の集中を図り抵抗溶接する方法である(図3・62)。この方法は突起の形状次第で小さいナゲットの溶接ができ、狭い場所でも多点を同時にあるいは線状・環状に溶接することができる。また板厚に差がある場合でも、厚板側にプロジェクションを出すことにより熱バランスをとりながら接合界面上にナゲットを形成させることができる。電極先端は平面でよいため消耗が少なく管理しやすい。また、電極形状が自由に選べるなど有利な点がある。反面、溶接装置には高精度が要求され、可動電極を突起の急激な変形に追随させるために加圧機構の剛性と即応性を要し、上下電極の平行度の確保も必要であるなど設備の管理が重要となる。被溶接材では、突起の寸法精度が溶接品質に大きく影響するので、その精度管理が重要である。溶接条件は突起の形状・寸法と接合面積に依存するが、基本的にスポット溶接と同様、電極加圧力、電流、通電時間の組合せである。プロジェクション溶接は、通電中に突起が徐々に加熱変形する図3・61
