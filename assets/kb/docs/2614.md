# 3・4・1 固相接合一般

> **章**: 第3章 溶接・接合および切断
> **節**: 3・4 固相接合法

われわれを取り巻く構造体において、特に強度部材として用いられるものの多くは、金属、セラミックス、高分子などの固体からなっている。固相接合(solid state bonding,solid state welding)は厳密にいえば、このような固体材料を全く溶融することなく接合する方法である。しかし、われわれの日常的経験では、金属でもセラミックスでも、単に重ね合わせるだけでは接合はできない。むしろ焼付き現象などのように、接合すれば困ることのほうが多い。しかしこのことは、条件が整いさえすれば固体どうしの接合が可能なことを示しており、実際に摩擦圧接は、焼付き現象からヒントを得て発想されたものといわれている。溶接・接合とは、表面・界面現象論的に見れば、一対の固相表面(接合表面)が消滅し、その痕跡を残さなくなるか、あるいは接合界面を形成することによって固体が合体する現象ということができる。融接法においては、接合表面近傍が母材の一部も含めて溶融状態となり、溶接金属と液体状態で混合されるため、通常は接合界面は形成されない。一方、固相接合においては、加熱や加圧によって接合表面の性状は変化はするものの、表面の汚染層や凹凸が残存しやすく、さらには組織の不連続性も解消されにくいため、接合界面が形成される場合が多い。材料学的には、固相界面は、以下の2種類に大別される。(1) 界面の両側が同じ相で、結晶構造も組成も同一であるが、結晶方位が異なる場合、すなわち結晶粒界。(2) 界面の両側が異なる相からなる場合、すなわち異相界面。これらの界面と比べてみると、同種材料間の固相接合界面は結晶粒界に、異種材料間の接合界面は異相界面にそれぞれ近いと考えられる。熱力学的にいえば、ある現象が進行しうるか否かは、それに伴うエネルギー変化の正負によって判断されるから、表面と界面のエネルギーを比較すると、一般に、上記(1)(2)の界面のエネルギーは表面エネルギーと比べて、いずれも3分の1程度と評価されている。したがって、固相接合界面がこれらの(1)(2)と大差なければ、エネルギー変化から考えて、固相状態でも接合はおのずから達成されるはずである。一方、電子論的な計算によれば、一対の金属表面を互いに近づけて行った場合、両者の間に強い引力が作用するのは、両者の間隔が原子の大きさと同程度、すなわち0.1nm台以内になってからといわれている。しかし、通常の切削加工仕上げを施した金属やセラミックスの表面には、少なくとも0.1 m台の高さの微視的な凹凸、また酸化皮膜(酸化物系セラミックスや貴金属の一部は除く)や吸着層あるいは油脂などの汚染層が存在し、互いの表面が強い引力の作用する距離まで接近するのを妨げている。したがって、固相状態で金属やセラミックスを接合するためには、これらの接合妨害因子を克服する必要がある。さらに、表面汚染層や凹凸に起因する空げき(未密着部)が接合界面に残存すれば、接合欠陥として作用し、接合強度を低下させる。本書で取り上げる固相接合法の(1)常温圧接、(2)拡散接合、(3)爆発圧接、(4)摩擦圧接、(5)熱間圧接のいずれもが、何らかの形で接合部の加熱と加圧を必要とするのは、これらの表面汚染層や微視的凹凸に起因する接合妨害因子を排除するためである。すなわち加熱は、吸着層の脱着、酸化皮膜の熱的分解の促進、さらには固相金属を軟化させて、密着化に必要な凹凸の変形を容易にする効果などが考えられる。また加圧については、密着面積の増加、および下地金属の塑性変形に伴う汚染層の破壊・分散の効果が考えられる。なお、表面が非常に清浄であれば、固相でも無加熱、かつ軽い加圧力で接合できることは、超高真空中での接合実験において、多くの金属の組合せで観察されている。
