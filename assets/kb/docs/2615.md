# 3・4・2 常温圧接

> **章**: 第3章 溶接・接合および切断
> **節**: 3・4 固相接合法

常温圧接(cold welding)は冷間圧接とも呼び、接合部を加熱することなく、単に加圧することによって大きな塑性変形を与えて接合する方法である。すなわち、下地金属の顕著な塑性変形によって、酸化皮膜等の表面汚染層が破壊・分散されて清浄表面が露出され、さらに加圧力によってこれらが互いに十分に接近(本節冒頭に述べたように)することによって、高い接合強度が達成されるといわれている。このため、下地金属に比べて酸化皮膜がもろいほど、すなわち酸化皮膜の下地金属に対する硬度比が高い金属ほど、接合に要する塑性変形量が少なくて済むといわれている。したがって、互いに接合される材料の少なくとも一方が、高い延性をもつことが必要で、アルミニウム、銅、およびこれらの合金への適用例が多い。また、異種金属間の接合にも適用可能で、特にアルミニウムと鉄鋼や、アルミニウムと銅などのように、反応によって脆弱な金属間化合物を形成する組合せの場合は無加熱であるため、化合物の形成に必要な拡散・反応を抑制することができる。継手形状は、一般に突合せ、および重ねで、図3・70にその一例を示す。接合強さは変形量によって支配されるといわれており、アルミニウムや銅の場合、重ね継手では厚さの減少率にして50~60%以上、突合せ継手では断面積の増加率にして約2倍近くが必要といわれている。また、表面処理も重要で、水、油脂、じんあいなどの汚れがないことは当然として、化学的処理や研磨などに比べて、ワイヤブラシのほうが優れた結果が得られるとされている。突合せ継手の場合は、突合せ部近傍に浅い溝を切って、その部分を圧接直前に捻(ねん)断することによって清浄面を得る工法も用いられている。常温圧接部は、強度の冷間加工を受けているため加工硬化しており、また熱影響部も形成されない。これらの材質的な面だけでなく、他の接合法と比べて装置、操作が単純で、再現性にも富んでおり、電線工業や電気工業の分野で数多く適用されている。なお近年、超高真空中で表面を清浄化し、そのまま大気にさらすことなく互いに接触させることによって、常温でも接合継手が得ら(a) 突合せ継手(b) 重ね継手図3・70 常温圧接継手例れることが、多くの金属の組合せで見出されている。またこの方法では、アルミニウムなどの活性金属とセラミックスとの接合も可能であるといわれている。表面の清浄化法としては、イオン衝撃が最も一般的であるが、超高真空中でのやすり研磨や摩擦も有効であると報告されている。また、異種金属のはく(箔)を交互に積層し、常温で繰返し圧延を行うことによって一体化し、複合材料を製造する試みも報告されている。
