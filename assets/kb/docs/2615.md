c .原価分岐点（cost turning point ) 損益分岐点分析の考え
方である総原価を変動費と固定費に分ける方法を用いると，生産量
の多寡を踏まえた現状の製造方法とその改善案について原価の比較
評価ができる．すなわち，図3･42に示したように，ある生産量以
上あるいは以下であれば，どちらのほうが原価的に有利であるの
か，またその程度を明らかにすることができる．
d.変動費と固定費の管理 変動費と固定費をいかに管理し，
低減するかということについて，変動費はすでに述べたように生産
量や売上高に従って変化する原価であるから，生産量や売上高の単
位当たり変動費をとらえて管理することになる．そしてこれは日々
の生産活動を行うことによって生じる費用であるので，部門責任者
には絶えず管理努力を傾注すべき費用ということになる．例えば変
動費として代表的な直接材料費や直接労務費は，材料のむだ遣いは
ないか，作業者の作業能率が適正かなどをチェックしながら管理し
なければならない費用である．
これに対して固定費は，時間の経過とともに発生する費用である
ので，原価計算期間単位で費用を管理することが必要である．ま
た，固定費は機械や設備などの長期的な視野に立った経営者の意思
決定に基づいて発生する場合が多いので，現場管理者にとっては管
理不能な費用であることが多い．
e.IE（industrial engineering）アプローチ IEは工業経営
とも経営工学とも訳されているが，1911年のテーラーの科学的管
理法から始まったといわれ，製造現場を中心とした企業経営の合理
化，改善，生産性向上を目指す管理技法である．IEは経営改善を
行うのが目的であり，そのために具体的な手法が開発されているの
で，これを用いれば原価の低減につながる．したがって，どのIE
の手法も原価低減という見地から必ず原価管理と結びつくという関
係にある．例えば，日程計画によって作業時間の改善が図られて労
務費が減少する．在庫管理によって在庫が減少し，在庫費用は圧縮
される．作業管理によって作業が改善され，作業時間の短縮，労務
費の低減が図られる．また，品質管理によって不良が減少し，材料
費，加工費が節約できる．また，特に原価維持に関わる標準原価に
よる管理においても，標準原価を設定する際に必要となる標準時間
は，作業管理での標準時間資料法や時間研究，PTS法などによっ
て決められる．また，標準原価と実際原価の差異をなくすための改
善は，動作経済の原則などIEの手法によって提供される．
