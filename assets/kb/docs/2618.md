# 3・4・5 摩擦圧接

> **章**: 第3章 溶接・接合および切断
> **節**: 3・4 固相接合法

摩擦圧接(friction welding,friction bonding)は、接合部材を突き合わせて相対的に回転運動をさせ、接触面に生じる摩擦熱を利用して加圧接合する方法である。接触面が摩擦熱によって軟化し、溶融温度近くに達した後、相対運動を停止させる。実用的な摩擦圧接は1950年代に旧ソ連で開発された。主要な接合方式として、(1)一定回転数で一方の材料を回転させながら他の材料に加圧接触させ、所定の摩擦時間後に回転を急停止させる制動式(ブレーキ式)と、(2)はずみ車に蓄積した回転エネルギーを一方の材料に与えて回転させ、他の材料に加圧接触させて摩擦によって自然停止させる蓄勢式(イナーシャ式)の、大きく分けて2種類がある。日本では前者の使用が多く、後者はアメリカで多い。摩擦圧接では、金属材料表面の酸化皮膜や薄い汚染層が接合中に分解すること、あるいは同種材の接合においては汚染層がバリとなって排出されることで接合が行われる。鉄鋼系の同種材の接合では、接合部の衝撃強さ(靱性)は母材のそれに比較して若干低い傾向があるものの、静的引張強さは母材以上の値が得られ、また回転曲げ疲労強度も高いなどの優れた継手特性が得られる。摩擦圧接を用いると、圧接面における両材料の反応や拡散が短時間で終わるために異相の発生と成長が少なく、異種金属材料間の接合にも有効な手段となっている。例として電気部品用のアルミニウムと銅の接合などがあるが、材料の組合せによっては接合困難な場合がある。また金属とセラミックスとの接合にも一部適用されているものの、接合後の残留応力とひずみの軽減のためにインサート材が必要な場合が多い。異種金属材料間の接合においては接合現象に不明な点があり、現在も多くの研究が続けられている。摩擦圧接は、装置が比較的大がかりになること、少なくとも一方の材料を回転させる必要があるため適用が軸材やパイプ材に限定されることが多いなどの課題がある。しかし、接合が秒単位の短時間で行えること、自動化が可能でランニングコストが低いこと、さらに接合部の品質が安定していることから、大量生産に適する。工業的には、ボルトやスプロケットなどの一般機械部品をはじめとし、自動車や航空機用の軸形状部品、ドリルビットやエンドミルなどの工具類、そのほか数多くの製品に適用され、軸材や中空材の重要な接合法の一つとなっている。
