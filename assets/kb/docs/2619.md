# 3・1・3 疲労損傷（fatigue failure)

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

繰返ししゅう動する摩擦面において，ある運転時間経過後に微小
き裂が発生し進展して表面層がはく離する現象である．一般に摩耗
よりも大きな粒子が脱落し，表面には穴状の損傷が点在して，機械
要素の振動を発生させたり運転精度の低下を招く．
疲労損傷は，機械要素の種類や損傷の形態によってさまざまな用
語で表現される．特に転がり接触面においては疲労損傷が機械要素
図 -3･1摩耗における雰囲気圧力の影響
第Ⅱ部 トライボロジー 第3章 表面損傷
3･2はおもな金属の相互溶解性を示したものである ．相互溶解度
の高い組合せは摩耗の程度を大きくする．同一部材どうしの摩擦は
ともがねと呼ばれ，接触界面で強い凝着が生じ，凝着摩耗の程度は
大きくなる．焼付きは，凝着が非常に強くなり摩擦抵抗が駆動力よ
りも大きくなった状態である．
凝着摩耗のアーチャードのモデル（Archard’s wear model ）を
図Ⅱ-3･3に示す ．摩擦面は半径 の接触点 個からなり，距離
2滑ると別の位置に新たな接触点を形成する（Ⅱ-1･3･4参照）．
