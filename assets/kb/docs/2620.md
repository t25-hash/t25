# 3・1・4 キャビテーションエロージョン（cavitation erosion)

> **章**: 第3章 表面損傷
> **節**: 3・1 損傷の種類

相変化や化学反応を伴う変化，光の照射などによる原子レベルの変
液体中に発生・成長した気泡が圧力上昇に伴って崩壊する際の衝
化までが含まれる．3章では狭義の表面損傷として，滑り軸受や転
撃圧で，近傍の固体表面に塑性変形や疲労によるはく離を生じる現
がり軸受，歯車，カム，シールなどの機械要素の摩擦面に生じる損
象であり，液滴の衝突による衝撃摩耗と区別される．
傷について解説する．
