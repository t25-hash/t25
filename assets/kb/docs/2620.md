# 3・4・7 その他の方法

> **章**: 第3章 溶接・接合および切断
> **節**: 3・4 固相接合法

a. 超音波圧接(ultrasonic welding)2枚の溶接物を重ね合わせて溶接チップと下部台の間に挟み、静加圧をかけて超音波振動をさせて接合する方法である。超音波振動によるエネルギーが物体に相対摩擦運動を発生させ、金属の場合は酸化皮膜を破壊し、発生した熱によって接合される。鉄鋼系の接合は困難を伴うが、厚さ数mm以下のアルミニウム系薄板の接合や、プラスチック薄板の接合に有効である。b. 摩擦かくはん接合(FSW : friction stir welding)イギリス溶接研究所(TWI)が1991年に発表し、その後国際特許を取得した比較的新しい接合法であり、近年非常に脚光を浴びている固相接合法の一つである。図3・75に示すように、一対の板材の突合せ目に沿って、かくはんツール(stir rod)と呼ばれる回転工具の先端のピン(pin)の部分を材料の間に挿入し、肩部(shoulder)と呼ばれる部分で合せ目を押さえながら回転移動させることによって接合部材に摩擦熱と塑性熱を発生させ、その熱で材料の塑性流動を生じさせて接合する。接合装置は機械加工のフライス盤とほぼ同じ構造を有する。材料は融点以下で接合され、溶融溶接法に比較して残留応力およびひずみが少ないために薄板の接合に適している。工3―67業的には、アルミニウムおよびその合金を中心として適用が進み、電車の外板などの大型板材を製作することが行われている。アルミニウム系材料で厚さ数十mmまでの板材の接合が行われている。課題として、装置が大がかりになること、鉄鋼など融点が高く、かつ強度の高い材料や異材継手の接合は困難であることが挙げられる。現在、アルミニウム系以外の材料、マグネシウムやその合金、鉄鋼材料、ならびに異材接合への適用に対する研究開発が進められている。
