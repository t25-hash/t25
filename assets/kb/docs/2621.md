# 3・5・1 ろう付けの分類と特徴

> **章**: 第3章 溶接・接合および切断
> **節**: 3・5 ろう付け

ろう付け(brazing)は、金属の接合法としては最も古くから用いられてきた技術である。これは接合部に母材より低い融点の金属をあらかじめ挿入して加熱、溶融させるか、接合部の外側から溶けた低融点金属を添加し、ぬれと毛管現象によって接合部すきまに浸透させ接合するという、最も単純な物理的現象を利用しているためである。しかし、狭いすきまにろうを流れ込ませるためには、母材をろうの融点より高い温度に加熱しておくことが重要であり、ろうが母材にぬれるための必須条件でもある。ろう付けおよびはんだ付けを表す総称的な言葉としてろう接(brazing and soldering)が使われており、母材の融点より低い温度で溶ける溶加材(ろう、はんだ)を接合部に溶融添加して母材にぬれさせ、接合する方法と定義されている。ろう付けとはんだ付けは用いる溶加材の融点によって区別され、728K(450°C)以上の場合をろう付けといい、728K(450°C)以下の場合をはんだ付けという。ろう付けを比較的整理しやすい方法で分類してみると、図3・76のようになる。これは施工時の熱源、熱の供給方式、装置または雰囲気で分類したもので、このほかに、銀ろう付け、リン銅ろう付け、貴金属ろう付けなどのようにろうの種類による分類、耐熱ろう付け、異材ろう付けなど継手の特色による分類、さらに、真空用ろう付け、歯科用ろう付け、装飾用ろう付けなど用途による分類法もある。ろう付けの最大の特徴は、母材がほとんど溶けないので寸法精度がよく、ひずみが少ないなど、薄板の接合や精密な接合ができることである。また、ぬれによってろうが母材の狭いすきまに入るので、複雑な形状で、接合部が多数箇所あるものの接合が容易で、自動化や大量生産に適している。一方、母材どうしや、ろうと母材がことさら溶け合う必要がないので、ろうおよびろう付け条件を適切に選択すれば、特殊な材料や異種材料の接合が融接に比べて容易である。さらに、母材とろうの融点が違うので、再加熱によって接合部を離すことができるし、施行についても、見えないところやトーチ、溶接棒が届かないところの接合、気密性の高い接合や広い面積の接合も容易にできるなどの特徴がある。
