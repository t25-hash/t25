# 3・2・1 凝着摩耗（adhesive wear )

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

凝着は，物体が真実接触部で接触するとき物体どうしが強い結合きに真実接触点で凝着が生じ，接触界面以外で破断したとき，一方して摩耗粒子を生成する摩耗が凝着摩耗である．
るには酸素の吸着や酸化を考えるとわかりやすい．
図Ⅱ-3･1は摩素の吸着が無視できるため大きな凝着を生じ，生成された粒子は接失としての摩耗量は小さい．
低真空中では移着粒子表面に酸素が次なる．
大気圧になると金属新生面が露出してもすぐに酸化するため（一次結合や二次結合）を生じた状態をいう．
2面間で摩擦するとの物体が相手面に移着する．
摩擦の繰返しで，移着・再移着を繰返大気中では凝着をイメージしにくいが，金属の凝着摩耗を理解す耗の雰囲気圧力依存性を示す．
