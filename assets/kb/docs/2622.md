# 3・2・1 凝着摩耗（adhesive wear )

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

凝着は，物体が真実接触部で接触するとき物体どうしが強い結合
（一次結合や二次結合）を生じた状態をいう．2面間で摩擦すると
きに真実接触点で凝着が生じ，接触界面以外で破断したとき，一方
の物体が相手面に移着する．摩擦の繰返しで，移着・再移着を繰返
して摩耗粒子を生成する摩耗が凝着摩耗である ．
大気中では凝着をイメージしにくいが，金属の凝着摩耗を理解す
るには酸素の吸着や酸化を考えるとわかりやすい．図Ⅱ-3･1は摩
耗の雰囲気圧力依存性を示す ．高真空中（10Pa以下）では酸
素の吸着が無視できるため大きな凝着を生じ，生成された粒子は接
触面で移着・再移着を繰返し，摩耗粒子の形成は抑制され，体積損
失としての摩耗量は小さい．低真空中では移着粒子表面に酸素が次
第に吸着し再移着を妨げ，大きな摩耗粒子が生じて摩耗量は大きく
