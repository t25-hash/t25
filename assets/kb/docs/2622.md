# 3・5・2 ろう付け設計

> **章**: 第3章 溶接・接合および切断
> **節**: 3・5 ろう付け

ろう付け設計は、ろうの特徴を最大限に生かすように工夫されなければならない。すなわち、(1)ろうのぬれによる毛管現象を利用し十分な継手強度を確保する、(2)機械的強度および信頼性を保つ、(3)耐食性その他必要な特性を具備できる、などの条件を考慮して継手形状(joint shape)が選択される。このためには使用母材とろうの選定が最も重要であるが、通常の生産工程では母材は決まっているので、それに合致したろうと継手形状を設計することになる。ろう付け継手(brazed joint)は基本的には重ね継手と突合せ継手に分類できる(図3・77)。通常、ろう付けする母材(basemetal)に比べてろうの強度が低く、また伸びが不足するので、突合せ継手の強度信頼性は低い。そのため、重ねしろをもつ継手形状が一般的に使用され、単純な突合せやT 継手のように、ろう単独で継手強度を確保しようとする設計はほとんど行われない。このため、突合せ的な継手の場合は図3・77に示すように、スカーフ継手や突合せ重ね継手などを用いる工夫がなされる。図3・75 摩擦かくはん接合の原理3―68加工学・加工機器火炎(ガス、重油など)ろう付け電気光線図3・76 ろう付け方法の分類継手板管突合せ重ね突合せ重ねスカーフ図3・77 ろう付け継手の基本形(a) 板材がブレージングシートでフィン材がベア材(b) チューブ材がベア材でフィン材がブレージングシート図3・78ブレーシングシートを用いたろう付けなお、アルミニウムろう付けにおいては線状のろうを供給する手法よりも、ろうをクラッドしたブレージングシート(brazingsheet)の使用が一般的であり、その継手はフィン-チューブあるいはフィン-板の組合せとなり、その模式図を図3・78に示す。アルミニウムろうはAl-Si共晶系であり、その引張強さは母材(3003,1050など)に比べて高いので、フィレット形状がある程度大きければ、継手のろうで壊れることはなくフィン切れを起熱源熱の供給方式雰囲気・装置トーチろう付け手ろう付け自動ろう付け炉中ろう付けフラックスろう付け雰囲気ろう付け抵抗加熱ろう付けフラックスろう付け雰囲気ろう付け真空ろう付け直接通電加熱ろう付けフラックスろう付け雰囲気ろう付け真空ろう付け誘導加熱ろう付けフラックスろう付け雰囲気ろう付け真空ろう付け赤外線加熱ろう付けフラックスろう付け雰囲気ろう付け真空ろう付けレーザろう付けフラックスろう付け雰囲気ろう付け真空ろう付け図3・79引張強さとろう厚さの関係表3・16 いくつかの代表的なろうの物性値BAlBCuZnBAgBAuはんだ組成mass%Al-10Si Cu-40Zn Ag-28Cu Au-20Cu Sn-40Pb温度K表面張力γ N/m0.890.971.01.20.38密度ρ kg/m140002γ/ρmm8.65.34.74.23.1γ:表面張力、ρ:密度、:重力こすことになる。アルミニウムろう付けについては文献(86)を参照願いたい。継手の強度と信頼性に影響を及ぼす設計因子としては、すきま(clearance,gap)がある。図3・79は突合せ継手の引張強さに及ぼす継手すきまの影響を示す。一般的に母材強度がろうの強度よりも大きいので、すきまが狭くなると母材に挟まれたろうは母材からの拘束により継手強度が高くなる。図3・79ではおよそ50 mのすきまで強度が最大となる。この傾向はろうよりも強度の低いはんだにおいても現れる。ろう付け継手の推奨すきまは、ろう付け相手母材、フラックス、ろう付け雰囲気などによって、ろう付け作業時のぬれが異なるので、一様には決定できないが、およそ100 m程度が種々の組合せに適している。ぬれや毛管侵入には毛管定数[capillary con-stant、表面張力と密度に関係した値、表3・16に参考値を示す。2γ/ρは無限垂直板に接触角0°でぬれた場合のろうの最大ぬれ上がり高さ(フィレット高さ)を示す]が関係しており、これらの値を理解しておくこともろう付け継手設計に重要である。ろう付け継手の強度および信頼性にはろう付け欠陥が影響を与える。外観検査でチェックできるぬれ不良、ひけ巣、割れ、ろう切れ等のほか、内部にはボイド、フラックスの巻込みなどがあり、継手強度と疲労特性などの低下を招く。重ね継手では重ねしろを板厚(母材の肉厚)の3~5倍程度とるが、すきまはろうの流入速度、ボイドなどの内部欠陥発生に関係し、最外部のフィレットの大きさは継手強度に関係するので、フィレットができる部分の形状設計や使用するろうの量の規定も重要である。
