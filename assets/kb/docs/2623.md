# 3・2・2 アブレシブ摩耗（abrasive wear )

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

粗さをもつ硬い面が軟らかい面を摩擦することで，軟らかい面が
著しく摩耗する状態をアブレシブ摩耗という．アブレシブ痕が生じ
るには，硬い物体の硬さ（ビッカース硬さ）が軟らかいほうの1.3
倍以上必要である．アブレシブ摩耗には二元摩耗（two-body
abrasion）と三元摩耗（three- body abrasion）がある．前者は硬
さの異なる摩擦する二つの接触面間で生じる場合である．一方，後
者は接触面間に硬い粒子などが侵入し，それが摩耗を引き起こす場
合である．このアブレシブ摩耗は材料の摩耗損傷として最も大きな
ダメージを与え（図Ⅱ-3･13参照)，また摩耗損傷の約50 %がアブ
レシブ摩耗に起因するといわれている．
図Ⅱ-3･4はアブレシブ摩耗のモデルであり，1個の硬い円すい突
起が軟らかい面に食い込み，摩耗を引き起こす（Ⅱ-1･3･4参照）．
アブレシブ摩耗は荷重に比例し，材料の硬さに反比例する．図Ⅱ-
3･5はコランダム研磨布に対する軟らかいホワイトメタルを基準に
した耐摩耗性（摩耗量の逆数）の結果であり，アブレシブ摩耗が硬
さに反比例することを示している ．
しかし，アブレシブ摩耗では図Ⅱ-3･4のモデルのように突起の
形状のまま摩耗することはまれである．図Ⅱ-3･6はアブレシブ摩
耗形態図（abrasive wear map）を示している ．図の縦軸は突起
の押込み深さ（荷重変化)，横軸は接触面の無次元せん断強さ＝
τ ，すなわち潤滑状態の変化を示している．ここでは摩耗痕の両
側に塑性変形により盛上りが生じ，実質的な摩耗損傷がない場合を
