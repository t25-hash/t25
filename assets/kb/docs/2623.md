# 3・5・3 はんだ付け

> **章**: 第3章 溶接・接合および切断
> **節**: 3・5 ろう付け

はんだ付け(soldering)は、電気・電子機器実装に使用されるプリント基板上の高密度多点一括微細接合技術と、その他の産業で使用されるはんだ付けの2種類に大別して考えなければならない。前者では継手形状が年々微細微小化しており、材料とプロセスの革新が続いている。図3・80にパッケージの小型微小化の推移を模式的に示す。IC(integrated circuit)パッケージの小型化に伴い、リード間隔とはんだ付けピッチが狭くなるとともに、QFP(quadflat package)からBGA(ball grid array)やCSP(chip sizepackage)への変更は、従来のはんだ付けからはんだボールを用いたバンプ実装(bump assembly)への変更となるなど、はんだ付けそのものの形態とプロセスが大きく変化してきている。後者で最も大きかった用途としての自動車用ラジエータなど銅製品へのはんだ付けが、材料のアルミニウム化の進展とともにほとんど皆無となってきているので、本項では産業規模としてはんだ使用のほとんどを占める電気・電子機器のはんだ付け(マイクロソルダリング)について記述する。はんだ(solder)とろうは材料の融点で区分され、液相線温度450°C以下の溶加材をはんだという。主たるはんだ合金系としてはZn基(Zn-Al,Zn-Al-Cu、融点約380°C)、Au基、Pb基、Sn基、In基、Bi基などがあり、主要組成と融点範囲を図3・81に示す。プリント基板実装の主要はんだ付け工法は、こてを利用するマニュアルはんだ付け、溶融はんだを用いるフローはんだ付け、ソルダペーストを印刷塗布して接合するリフローはんだ付けなどである。はんだ付けの要点は均一温度上昇である。プリント基板実装のように、熱容量の異なる部品が混載している基板では、はんだ付け継手部の温度が均一に上昇しないため、熱容量の大きい部品近傍では温度が低く、微小部品電極では温度上昇が速くなる。最も温度が上昇しにくい部分では未接合が生じ、逆に、温度上昇の速い部分では高温長時間加熱となり、メタライズ層の過剰溶解、金属間化合物の過剰成長など継手信頼性を低下させる現象が起こる。このため、一般的な適正操業条件範囲は図3・82となる。これまでに、プリント基板実装に多用されてきたはんだのほとんどはSn-Pb共晶系(Pbを36~40%含有、Sn-Pb共晶温度183°C)3―69であったが、環境問題への対応から有害物質Pbを使用しない鉛フリーはんだ付けに移行している。図3・81から融点的にSn-Pb共晶代替となりうるのはSn基はんだとなる。表3・17に、これまで検討されてきてほぼ実用化レベルと判断されているいくつかの鉛フリーはんだ(lead-free
