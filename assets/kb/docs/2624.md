# 3・2・3 腐食摩耗（corrosive wear )

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

腐食摩耗は腐食環境中において摩耗部材が環境因子と化学反応
し，結果として摩耗が影響を受ける摩耗である．酸化摩耗
（oxidative wear ）は摩耗面や摩耗粒子が酸化し，その生成物であ
る酸化物が以後の摩耗過程に影響を与えるが，これは乾式の腐食摩
耗とみなされる．

腐食の影響には摩耗を促進させる場合と，抑制する場合とがあ
る．腐食作用として材料損失を促せば摩耗量は増加する．この場合
の摩耗モデルを図Ⅱ-3･7に示す．相手面の突起と突起の接触間隔
の間で摩擦面において反応が生じ，突起は反応生成物を除去する作
用をし，したがって反応生成量が摩耗量となる ．図Ⅱ-3･7では
反応速度が放物線則に従うと仮定している．一方，腐食生成物と表
面の結合が強いと反応生成物は接触界面にとどまり，相手面との直
接接触を抑え，固体潤滑剤の作用により摩耗を抑制する．その例
は，境界潤滑における極圧添加剤や，高温大気中の緻密化された酸
面き裂を起因とした疲労現象である．
