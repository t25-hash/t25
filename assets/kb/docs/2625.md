# 3・2・4 フレッチング（fretting)

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

フレッチングは，軸のはめあい部など，外的な振動により接触面
で微小な相対滑り振動が起き，摩耗が生じる現象である．通常，微
小な相対滑り振幅のおおよその値は全振幅で100m以下とみなさ
れている．
フレッチングは，時にフレッチングコロージョン（fretting cor-
rosion）と呼ばれるように，酸化摩耗が顕著である．これは滑りが
小さいために，大気中において形成された摩耗粒子が接触界面から
排出されにくく，金属粒子が次第に酸化され，摩耗痕内部および周
辺に酸化物粒子が著しく堆積するからである．しかし酸化が抑えら
れる状況でもフレッチングが生じる．図Ⅱ-3･11は滑り振幅と摩耗
量の関係 であり，ある振幅以下では摩耗量はほぼ零であるが，
振幅とともに摩耗は増加する．
