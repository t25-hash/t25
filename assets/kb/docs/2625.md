# 3・6・1 熱切断の分類

> **章**: 第3章 溶接・接合および切断
> **節**: 3・6 熱切断および火炎加工

熱エネルギーにより、素材を局部的に溶融除去して切断する方法を熱切断(thermal cutting)と呼ぶ。熱切断は、加工時に材料に力が加わらないので、材料を固定したり、クランプしたりする必要がないため、特殊な場合を除いて装置を小型化することができる。また、熱源をいわゆる線熱源として取扱えるので、曲線を含む自由形状の切断が行え、特に板状素材の切断に適している。一方、材料への入熱、溶融の影響を避けることができないので、材質変化、熱ひずみ、熱変形などの発生についての配慮が必要である。図3・84に各種熱切断法を示す。図3・84は、エネルギー源によって分類したものであるが、酸化エネルギーを利用したガス切断、電気エネルギーを利用したプラズマ切断、光エネルギーを利用したレーザ切断は、それぞれの分類の中でも最もよく利用されている熱切断法である。
