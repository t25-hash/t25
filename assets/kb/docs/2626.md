# 3・2・5 摩耗の評価方法および摩耗遷移

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

a.摩耗パラメータ 摩耗（wear ）は材料損失あるいは材料損
傷である．摩耗量（wear volume）の評価方法は体積損失や質量損
失で表示するが，質量で評価しても材料の密度を用いて一般には体
積で表す．
摩耗は時間（滑り距離）に対して逐次減量現象であり，その変化
を摩耗進行曲線（wear curve）と呼ぶ．図Ⅱ-3･12はその代表的な
例であるが，摩耗の進行は材料や摩擦条件に依存する．摩耗進行曲
線の傾き / を摩耗率（wear rate）と呼ぶが，摩耗率が滑り
距離に対して図Ⅱ-3･12の曲線①のように一定の場合もあれば，曲
線②や③のように変化する場合もある．曲線①ではアブレシブ摩耗
や接触面の変化が無視できる摩耗で見られる．曲線②において摩耗
率の大きい領域を初期摩耗（initial wear )，後半の傾きが一定とな
った領域を定常摩耗（steady-state wear ）と呼ぶ．初期摩耗は接
触面がなじむ段階であり，比較的大きな金属粒子の形成と表面のあ
れが生じる．摩耗機構としては凝着摩耗が支配的である．その後，
摩耗粒子の微細化と酸化が生じ酸化摩耗に推移する．これが定常摩
耗である．曲線③は疲労摩耗やエロージョンなどで潜伏期が存在す
る場合である．
