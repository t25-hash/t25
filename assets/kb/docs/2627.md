# 3・2・6 油潤滑下の摩耗

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

多くの機器は乾燥状態よりも潤滑油を用いて運転される．それは
摩擦を低減し，摩耗を抑制するためである．しかし接触面の潤滑状
態により摩耗が生じる．油潤滑下では特別な摩耗機構は存在せず，
摩擦条件に依存して3･2･1～3･2･4項の摩耗機構や3･4節の疲労摩
耗が起こる．固体接触が生じない流体潤滑では摩耗を考慮する必要
はないが，確率的には摩耗が生じ，また実際の機械では運転の開
始・停止時に油膜の形成は期待できずに直接接触をしている．また
境界潤滑では確率的に直接接触が起きており，摩耗が生じる．図
Ⅱ-3･9のように潤滑油中の極圧添加剤による固体表面との反応は
摩擦を低減する効果をもつが，摩耗という立場からは腐食摩耗の概
念と同じである．その摩耗の程度は図Ⅱ-3･13に示すように，摩耗
係数が10～10 と大きく変化する．図Ⅱ-3･16は油潤滑下の摩耗
遷移を示しており ，荷重の増加により摩耗機構が変化し，比摩
耗量が増加する．しかし乾燥摩擦に比べて，遷移はなだらかである．
