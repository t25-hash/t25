# 3・3・2 臨界温度条件

> **章**: 第3章 表面損傷
> **節**: 3・3 焼付き

摩擦面材料と潤滑油によって決まるある臨界温度まで表面温度が
上昇したときに焼付きが発生するという説 で，現時点での焼付
き条件の主流を成している．臨界温度としては吸着膜の配向喪失温
度 ，脱離開始温度 ，すなわち転移温度や，表面膜の軟化温度
が採用されることが多い．
臨界温度の基準になる摩擦面温度としては，バルク（本体）温度
θと接触部の閃光（せんこう）温度θの和，すなわち摩擦面最高
温度上昇θ＝θ＋θが採用されることが多い．θは計算により
推定することが一般的であるが，その値としては，真実接触部での
実際の最高温度上昇ではなく，摩擦熱が見掛けの接触域全体に平均
的に分布していると仮定して求めた平均温度上昇を採用することが
多い．また，摩擦面間の接触状態，作動条件により焼付き発生温度
の基準温度が異なるとことも指摘されている ．例えば，作動
条件が過酷であれば接触部で発生した局所的凝着がそのまま焼付き
にまで進展しがちであるのでθが基準温度となり，転がり滑り接
触におけるように摩擦面の接触時間が非接触時間に比べて極めて短
い場合では，接触時に生じた局所的損傷が非接触時間に回復するこ
とが考えられるので，摩擦面全体が臨界温度に達することが焼付き
発生には必要となり，θが基準温度となることが多い．
