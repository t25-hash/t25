# 3・7・1 溶接部の材料挙動

> **章**: 第3章 溶接・接合および切断
> **節**: 3・7 各種材料の溶接

溶接部は短時間で溶融から凝固,および室温に至る急峻な熱サイクルを受けるため,その組織は母材のそれとはかなり異なったものとなる.溶接部の組織は継手の特性ならびに割れ,脆化,耐食性劣化など溶接に伴う諸問題と密接に関連する重要な因子である.したがって,溶接部の欠陥制御や特性予測を行うためには溶接部の材料挙動とその結果である材料組織を理解する必要がある.溶接部には溶接過程で溶融凝固する溶接金属と,その周囲の高温に加熱され組織や材質が変化する溶接熱影響部が含まれる.溶接部における材料挙動の特徴は以下の点である.(1) 温度が室温→最高加熱温度→室温と時間とともに短時間で急激に変化する溶接熱サイクル過程で生じる.(2) 場所によって温度が異なる温度場が存在する過程で生じる.(3) 熱サイクルと同時に応力-ひずみサイクル下で生じる.特に,温度が急激に変化する溶接熱サイクル過程で現象が生起することによる影響が大きく,これを理解することが溶接部の材料挙動を知るうえでの重要なポイントとなる.溶接部の材料挙動を決定する支配因子はそれぞれの箇所の最高加熱温度,加熱速度および冷却速度である.図3・91に溶接部に生じる材料の組織変化ならびにそれに関連する溶接欠陥および特性の変化を模式的に示す.溶接部の中で材料の融点以上に加熱される領域は溶融した後凝固し,溶接金属を形成する.その過程で,溶接金属は母材と異なる凝固組織を有する組織となる.また,溶接金属内では凝固過程で固-液間の元素分配に起因した元素の不均一分布,すなわち凝固偏析(solidification segregation)が生じる.この現象に関連し,最終凝固領域で材料の融点を下げる不純物元素が濃化した場合,低融点の液相が形成し凝固割れ(solidification cracking)を生じる場合がある.また,溶質元素の偏析は耐食性や靱性の劣化原因となる.溶接金属に近接したボンド部近傍の熱影響部では材料の融点直下の高温まで数秒の時間で加熱後,数十秒から数分の後に室温付近の温度まで冷却される.この領域では溶接熱サイクル過程での最高加熱温度付近で結晶粒の粗大化が生じる.同時に熱サイクルの加熱過程では析出物の固溶や高温相への変態,冷却過程ではその逆変態や析出物の再析出が生じる.その結果,変態硬化する鉄鋼ではボンド部近傍の熱影響部は硬化し,析出硬化型合金では熱影響部で溶体化軟化や過時効軟化が生じる.加工硬化により強化された材料では,再結晶温度以上に加熱された領域が回復(recovery)や再結晶(recrystallization)により加工ひずみの解放がなされ,材料が軟化する.また,これらの材料挙動に関連して靱性や耐食性などの特性劣化や延性低下割れや低温割れ(cold cracking)が生じる場合がある.溶接後,溶接部の組織改善や残留応力低減を目的とした溶接後熱処理(PWHT : post weld heat treatment)が行われる場合がある.この熱処理過程では析出による二次硬化や不純物の粒界偏析などが生じる場合があり,これにより再熱割れ(reheat cracking)が発生する場合もある.以上のように,溶接部の材料挙動は特性の変化や溶接欠陥の発生と密接に関連しており,その制御のためには溶接部における材料挙動の正確な理解が不可欠である.特に,溶接部における諸現象は短時間の溶接熱サイクル過程で生じることから,その変化が平衡状態図3・91 溶接過程における材料挙動とそれに関連する溶接欠陥に至らず中間段階にとどまる場合が大部分であるため,温度が変化する非等温過程で,しかも現象が平衡に至るまでの途中段階での変化過程の組織変化を定量的に知ることが必要になる.このため,溶接部の材料挙動を定量的に取扱う速度論的な観点からのアプローチが特に重要となる.これらの結果得られる情報に基づき,欠陥防止や特性制御のための溶接パラメータの適正選択が可能になるといえる.
