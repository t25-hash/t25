# 3・7・4 非金属材料の溶接

> **章**: 第3章 溶接・接合および切断
> **節**: 3・7 各種材料の溶接

非金属材料の溶接で最も重要なものは、セラミックス(ceramics)と金属の接合である。セラミックスと金属を接合するに際して考慮すべき重要な問題点としては、セラミックス-金属間の接合界面反応、セラミックスと金属の熱膨張差に起因した接合継手の熱応力の発生の二つに集約される。セラミックスと金属の界面における反応形態の多くは、接合界面で明らかに反応が生じて拡散層や反応層を形成する場合である。セラミックスと金属の接合界面反応では、反応金属の非金属化合物(酸化物、窒化物、炭化物など)や反応金属とセラミックス構成金属の金属間化合物が生成する。セラミックスと金属の界面反応を適正に、かつ高度に制御することが良好な接合を達成させるための重要なポイントである。一方、セラミックスと金属間の熱膨張差のため、高温で接合後冷却すると熱応力が発生する。実際の接合体中の熱応力分布は非常に複雑であるが、セラミックスが金属に比べて熱膨張係数が小さい場合、セラミックス中の端部付近に大きな熱応力(引張応力)が生じる。熱応力は接合継手の割れ発生や接合継手特性の低下を引き起こすため、表3・35に示すような熱応力緩和策が必要不可欠である。セラミックスどうしあるいはセラミックスと金属の接合方法を図および機械的結合法に大別できる。以下に、これらの接合方法の代表的なものについて特徴や長所・短所などを述べる。なお、それぞれの接合方法の詳細は文献(129)~(131)を参照されたい。a. 無機ろう付けセラミックフリット法は主として酸化物の混合物をろう材として、大気中、不活性ガス中もしくは真空中で加熱処理してろう付けする方法である。酸化第一銅法はCuO粉末をろう材とするおもに酸化物系セラミックスに対する接合方法である。また、硫化銅、炭酸銀やフッ化物を溶融もしくは焼成させて接合する方法もある。これらの接合方法は、接合温度を比較的広く選定でき、耐熱性に優れる利点はあるが、一般に接合継手特性や金属との接合性に問題がある場合が多い。b. 金属ろう付け金属ろうを用いる方法は、メタライジング法と直接ろう付け法に大別される。メタライジング(metalizing)はセラミックスの表面改質法の一つであり、セラミックス表面に金属層や膜を被覆・形成させる処理である。メタライジング法には、Mo-Mn法に代表される高融点金属あるいは酸化銅、硫化銅などの金属化合物を塗布し、加熱分解により金属を析出させセラミックスと反応させる方法、化学反応により気相から金属を析出させる方法(CVD : chemical vapor deposition、化学的蒸着法)、および蒸着やスパッタリングなどの物理的方法で金属をめっきする方法(PVD : physical vapor deposition、物理的蒸着法)などがある。これらの処理後、Niろうや銀ろうなどを用いて金属などとろう付けする。一方、直接ろう付け法としては、TiHやZrHなどを用いる水素化物法およびAg やPtを用いる貴金属法などもあるが、TiやZrなどのIV族元素(活性金属)とNi,Cu,Ag などの合金をろう材に用いて真空ろう付けする活性金属法(active metal brazing)が最も広く用いられる。また、熱応力低減効果をさらに増加させることを目的とし、V族およびVI族元素を数%程度含むCu基合金をろう材とした弱活性金属法と呼ばれる接合方法も開発されている。さらに、セラミックスとの反応性と塑性変形能が良好なアルミニウム合金をろう材に用いた接合方法もよく用いられる。金属ろう表3・35 welding)はセラミックスどうしあるいはセラミックスと金属を直接接触させたまま加圧下で加熱し、構成元素を拡散させて接合する方法である。その場焼結接合(in-situ sintering bonding)法はセラミックスもしくはセラミックスと金属の混合粉末を接合界面に挿入して、加圧力を加えながら加熱し、その場で焼結させて接合する方法である。この方法には、ホットプレスや熱間等方加圧(HIP: hot isostatic pressing)技術が適用されるが、最近、パルス通電加圧焼結を用いる新たな接合技術が試みられている。一方、摩擦圧接(friction bonding,friction welding)法は、高速度で回転している被接合体を接触させ、その際に発生する摩擦熱を利用してセラミックスと金属を接合する方法である。大気中で加熱・加圧が比較的容易に行え、特に、接合表面を清浄化しつつ接合界面を局部的に加熱することができる特徴を有する。界面でのセラミックスと金属の反応を利用して接合が行われる。固相接合方法は、いずれも精密接合が可能で優れた接合継手特性を示すが、一般に接合装置が大がかりで接合条件や前処理を高度に制御する必要がある。d. 程度の予熱が必要である。また、非酸化物系セラミックスのように解離・蒸発しやすいセラミックスには適用が困難である。溶接による接合では、他の機械的接合方法の場合と異なり金属の溶融を伴うため、熱による材料の組織変化、溶接後の残留応力や変形の発生などの問題が生じる。施工条件により溶接欠陥も発生しやすいという特徴がある。これら以外にも溶接には接合方法として種々の際立った特色がある。したがって溶接の施工には、他の接合方法の場合にはない管理方法、管理項目が求められる。施工技術者はもとより、設計技術者においても、溶接構造物・製品の製造に際しては、溶接施工管理の特徴、重要性をよく理解し、それらの知3―83酸化物ソルダ法酸化第一銅法硫化銅法/炭酸銀法その他無機ろう付けメタライズろう付け活性金属法/弱活性金属法Alろう付け貴金属法水素化物法その他圧着金属ろう付け直接ろう付け拡散接合その場焼結接合法HIP
