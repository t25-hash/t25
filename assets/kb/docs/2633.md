# 3・4・2 転がり接触（rolling contact ）における疲れ

> **章**: 第3章 表面損傷

a.転がり疲れ（rolling contact fatigue) 転がり接触は，転
がり軸受，カム，歯車などのトライボ要素の可動部の相対運動面に
おける運動形態である．その接触領域は一般に小さく，外力の作用
により大きな集中荷重が生じる．接触面の一点に着目するとその集
中荷重が繰返し負荷されることになる．転がり接触により機能を発
揮するトライボ要素の性能を検討する際には，この運動形態および
力の作用の状態を十分に把握しておく必要がある．
一般に，材料は方向の変動する繰返し応力のもとでは，静的強度
よりも低い応力（弾性限以下の場合もある）によって破壊する．こ
のような現象を材料の疲労，あるいは疲労破壊という．転がり接触
により機能を発揮するトライボ要素では，表面あるいは表面下に繰
返し応力による局部的な特殊な疲労現象が生じる．破壊が生じる
と，その後の継続使用が不可能になる場合が多いこと，言い換えれ
ば，トライボ要素が全体形状を保持してはいるが，材料の疲労によ
る局部的な破壊によって本来の機能を喪失して使用に耐えられなく
なる場合が少なくない．転がり軸受のフレーキング，歯車のピッチ
ングなどの歯面損傷，カムなどの表面損傷の類がこれらの典型であ
る．表面近傍に集中して作用する応力の繰返しにより，損傷の外観
と名称はトライボ要素のそれぞれで違うが，総称して転がり疲れと
呼んでいる．これらの損傷では，接触の繰返し数がある値を超える
と，通常の摩耗粉の大きさに比べて大きな物質が脱落していき，比
較的大きな穴や欠けが生じるのが共通した特徴である．これは軽微
な段階にとどまっていても，トライボ要素の回転精度や作動性能の
低下をきたすが，引き続く接触の繰返しによって振動あるいは発熱
を伴い，やがて損傷を拡大してトライボ要素の作動を不可能にさせ
る．
材料の一般疲れでは局部に発生したき裂が，その後の応力の繰返
しによって拡大進展して全体的な破断に至るため，転がり疲れでも
通常，き裂の発生と進展過程に区分して取扱われる．このき裂の発
生と伝ぱの過程を明確に区別して論じられるようになったのは古く
はない．しかしながら，両者の境界がどこにあるかを定めることは
実際には困難である．
転がり疲れの破損の機構が確率論的に体系的に論じられたのは，
20世紀の半ばにルンドベルグ・パルムグレン（Lundberg&Palm-
gren) によってワイブル分布を基礎において転がり軸受を対象に
提案されたものが初めてとされている．ルンドベルグ・パルムグレ
ンらの理論は，材料強度が構造（組織）敏感性を有していることを
前提にしていること，また外力により材料内部に発生する相対的に
最大となる応力の繰返し振幅の大きさを考慮し，しかもその応力の
材料内部における影響範囲を同時に考慮するという材料破壊の原理
原則に立脚している点が今日までその有効性を保ってきた理由であ
る．

c .転がり軸受の転がり疲れ 転がり軸受は，内外輪，転動体，
保持器から構成されているが，転動体と内外輪の組合せによる転が

生すれば一般に継続使用が不可能になる．作動状態，雰囲気，取扱

を対象にして，一定の約束のもとに規定した荷重により，軸受の負

的小さな穴があく現象であるが，潤滑油膜を十分に形成させるか異

によってその発生を抑えることができる．フレーキング
（flaking）は，うろこ状または片状の形態をとって表面からはく離
