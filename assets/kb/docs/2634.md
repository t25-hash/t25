# 3・8・1 溶接施工管理一般

> **章**: 第3章 溶接・接合および切断
> **節**: 3・8 溶接施工管理

a. 溶接施工管理の基本溶接施工においては、構造物・製品の用途および使用条件に応じて作成された設計図書(仕様書、図面など)と適用基準に基づいて、所要の製造品質を、決められた納期で能率よく、かつ安全に実現することが要求される。このために必要なのが施工管理である。施工管理においては、品質管理が最も重要であることはいうまでもないが、資材、設備、日程、配員、コスト、作業環境などの管理もそれに劣らず重要である。したがって、施工管理に際しては、溶接工程のみならず、その前後の工程との関連性の理解、工程全体における作業の流れの把握が不可欠である。b. 製造品質管理溶接の製造品質管理の基本は、最終品質に影響を与える種々の因子を明確にし、適確な計画を作成するとともにそれらを施工時に十分に管理することといえる。図3・95は、溶接施工において、溶接品質を保証するために管理しなければならない因子を、鋼構造物の場合を例にして特性要因図として示したものである。. 溶接施工要領の作成溶接施工管理の計画において最も重要なものが、最適な溶接施工要領の作成である。溶接施工要領は、設計品質と工程能力を勘案しながら、図3・95に示した特性要因図における各種要因、すなわち、(1) 溶接技術者の管理能力、溶接作業者の資格、技能レベルなどの人的要因(2) 溶接機、ジグなどの設備、作業場所の環境(3) 使用する鋼材、溶接材料の種類、特性(4) 溶接方法、溶接条件などの溶接施工時の管理項目(5) 確認、記録などの作業管理項目(6) 溶接結果を評価する各種試験検査などの要因を考慮して決定する必要がある。溶接施工要領を決定するまでの過程と各種要因の相互関係を図しての設計図、仕様書などから鋼材を選定し、その鋼材に適した溶接材料を選ぶ。溶接材料の選定に際しては、製造品質としての溶接方法、溶接設備なども考慮すべきであることはいうまでもない。また、品質管理方針を定めるとともに、製造品質に関係する溶接設備、溶接要員、施工・作業管理に基づいて、溶接施工要領を決定する。このとき、経済性、納期、安全性などの検討も忘れてはならない。3―84加工学・加工機器設計品質設計図仕様書等溶接構造物品質製造品質工程能力決定した溶接施工要領を書類化したものが、溶接施工要領書(WPS 技術・技能要員管理専門知識と経験のある溶接技術者の存在は重要である。溶接施工要領書の作成、その確認試験、実工事への溶接施工要領の適用、施工管理、コスト管理など溶接技術者の取り組むべき役割は多い。したがって、溶接技術者には高度の知識、経験、能力が要求される。溶接施工において、溶接技能者の重要性は溶接技術者のそれに劣らない。適用する溶接方法とその難易度に応じた技量と資格を有した技能者を確保し、適材適所に配員し管理することも施工管理の重要項目の一つである。このための教育計画、資格取得計画の立案、実施も広い意味での施工管理である。. 溶接工程・コスト管理一般的には、継手1m当たりの溶接に要する時間(標準時間)を求め、それに継手長さを掛けて溶接作業時間を見積もり、作業工程の計画に用いられている。標準時間は継手板厚、脚長サイズ、溶接姿勢、溶接方法、アークタイム率などによって変わるので、それぞれの場合の標準時間をデータとしてもつか、基本の標準時間に係数を掛けて対応する方法がとられている。溶接にかかる費用は、表3・36のような構成で成り立っており、コストは一般的に、それぞれの材料単価にその材料の使用量を掛けたものと各工費単価に作業時間を掛けたものとの合計として求められる。溶接工程・コストは、基本的には上記のようにして得た見積り数値をもとに計画され、実施工において管理される。d.
