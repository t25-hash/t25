溶接の品質管理（人，設備，技術管理）については，JIS Z
3400：1999（溶接の品質要求事項）に規定されており，溶接は特殊
工程と呼ばれ，認定された工程，設備，要員により管理を行う必要
があるとされている．さらに，品質確認項目についても細かく記さ
れている．このJISは，ISO（国際標準化機構）の品質保証システ
ム（9000シリーズ）に整合させ，国際的商取引に通用するもので
ある．したがって，溶接の品質管理は，これに従うのが望ましいこ
とはいうまでもないが，すべての製品に対してこのような定められ
た品質管理が要求されるものでもない．あくまでも，使用者と製造
者の取決めによるものである．以下に，そのJISの要点と溶接管
理一般について記す．
a.溶接管理技術者 溶接の全体を管理する溶接管理技術者の
任務および責任については，JIS Z3410：1999に規定されており，
これに当たる人の資格も記されている．この資格は，JISがISO
規格と整合性をとり，国際規格として認知され，日本では現在，日
本溶接協会がその認証業務を行っている．資格は職務経験を前提と
したうえで，以下の三つに分類されており，これらの資格保持者が
少なくとも1名，全体の溶接管理に当たるよう定められている．も
ちろん，このような定められた規定が適用されない製品もあり，こ
の場合は使用者と製造者の契約条件が優先する．
(１)包括的技術知識をもつ溶接管理技術者
( IIW資格，IWE:international welding engineer）
(２)特定技術知識をもつ溶接管理技術者
( IIW資格，IWT:international welding technologist）
(３)基礎技術知識をもつ溶接管理技術者
( IIW資格，IWS:international welding specialist）
b.溶接技能者 すべての溶接技能者および溶接オペレータ
は，適切な規格に従い，資格を取得した者が行うように規定されて
いる．適切な規格としては，各事業の団体が定める，例えば船舶の
溶接工技量試験，ボイラ溶接士試験等とJISで規定する資格があ
る．特別な指定がない場合は，JISの資格を取得し，作業に従事さ
せることが望ましい．
c.溶接材料 溶接材料の管理に伴う責任および手順は，製造
者が定める．製造者は，吸湿，酸化，損傷などを避けるための溶接
材料の保管，取扱いおよび使用に関する手順を作成，実施し，その
手順は，供給者（溶接材料メーカ等）の推奨に従って実施すること
と定められている．溶接材料の保管状態および使用方法が悪いと，
溶接欠陥の発生につながるので，十分な管理が必要である．また，
溶接材料はJISで識別（色）処理が規定されているが，誤使用の
ないように十分な注意が必要である．
d.溶接機器 溶接機器を安全かつ能率よく稼働させるために
は，定期的な保守，点検を行うことが重要である．規定においても
設備の保守計画を文書化し，確実に実施しなければならないと定め
ている．保守，点検は，日常的に行うものと定期的（例えば，6箇
月ごと）に行う項目とに分けて，計画，実施するのが一般的であ
る．
e.溶接条件 溶接条件は，承認された溶接施工要領書に基づ
き決定されるが，さらに細かく，溶接中の溶接電流，アーク電圧，
溶接速度，予熱パス間温度，溶接順序などを，適切な間隔または連
続するパラメータによって点検することと定められている．溶接条
件は，材質が同じでも板厚や開先形状が異なると適正溶接条件が変
わるので，個々の部位に対する適正溶接条件を作業標準的な形で現
場に指示徹底し，管理することが望ましい．
f .欠陥の防止 溶接された継手部には，使用目的や設計条件
に適合しない有害な欠陥があってはならない．発生した場合は，溶
接施工要領書，作業標準などを見直して，その原因を究明し，再発
防止に反映させなければならない．溶接欠陥の発生原因は，溶接士
の技量のみならず，材料の溶接性等にも起因するので，それらに適
した溶接作業標準（手順）などを作成し，現場へ指示徹底すること
が必要である．万一，欠陥が発生した場合は，十分な改善策を講
じ，その条件により補修溶接を行うのが通例である．これらの問題
の発生原因の追求および品質保証の面から，実際に施工した条件の
記録を残すことも重要である．
g.記録と保管 品質記録は，契約要求事項に従って，必要に
応じて次の項目を含めることが望ましいと定められている．その項
目のおもなものには，溶接材料証明書，溶接施工要領書およびその
承認試験結果の記録，溶接技能者または溶接オペレータの認証書，
非破壊検査要領書およびその検査結果の記録等があり，その記録の
保管は最低5年保持すると定められている．
3・8・4安全および衛生・環境
アーク溶接作業においては，母材および溶接材料をその融点以上
に加熱して溶融させる必要があるため，その加熱のための熱源（ア
ーク）はさらに高温を必要とする．このような高温作業をごく近く
で行うのが溶接作業であり，安全・衛生上の諸問題もここに起因し
ている．
最近は，自動化，機械化により，作業者を熱源から遠ざけて，作
業環境をよくすることも進められているが，新たにロボットの暴走
事故等の問題が発生するようになってきた．また，新たな熱源の開
発により，レーザビームによる眼障害や，電子ビームによるX線
障害等の問題も発生している．このような種々の危険から人体を保
護するために，管理者および溶接作業者は適切な防止対策をとり，
溶接作業者だけでなく，近くで作業を行う人々の安全を確保しなけ
ればならない．
法令においても，「労働安全衛生法」を根幹として，種々の障害
の防止策を講じるとともに，安全衛生教育，定期健康診断等が義務
類，遮光眼鏡，手袋等の溶接用保護具を着用し，目や皮膚を露出さ
状の代表的なものとしては，金属熱と呼ばれる症状で，作業中から

a.溶接による熱 アーク溶接では高温のアーク（例えば，ミ
グ溶接のアーク周辺部で約8000K）から多量のふく射熱が発生す

に母材を約373～473Kに加熱することがあり，この場合，さらに
溶接作業者は高温の環境にさらされる．
このような環境では，溶接作業者の体温が上昇し，発汗過多によ
る熱中症になることがあるので，溶接作業場の風通しをよくする，
あるいは冷房服を着用するなど，作業環境の改善が必要である．
b.有害光線 アーク溶接やレーザ溶接では強烈な可視光のほ
かに，目や皮膚に有害な紫外線および赤外線を多量に放出するた

熱傷などを引き起こす．アーク光から目，皮膚を保護するには，衣
せないことが必要である．また，溶接作業者だけではなく，周辺で
作業をしている人もこのアーク光により目や皮膚を侵される場合も
あるので，遮光カーテンや衝立てにより遮へいすることが望まし
い．
