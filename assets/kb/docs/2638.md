# 3・9・1 概要

> **章**: 第3章 溶接・接合および切断
> **節**: 3・9 リベット接合

リベット(びょう、鋲、rivet)は薄板構造部材を永久結合する場合に用いられる機械接合方式の一種である。溶接や接着等の接合方法に比べ、簡単・確実に接合できる方式として、特に航空機構造の外板接合方式としては代表的なものである。航空機構造の主要材料であるアルミニウム合金は熱を加えることにより特性が劣化してしまい、溶接に適さないこともあるが、豊富な継手としての機械的強度データと使用実績に基づく信頼性によるところが大きい。航空機の図面に規定されるリベットの規格は、アメリカ規格(NASM,NAS 等)がそのまま国内でも使用されている。リベット接合は、結合孔に差し込んだリベットを、孔から抜けないよう反対側の軸部をつぶして結合するものである。通常のファスナに比べて小型であり、呼び径3/32in(2.4mm)~3/16in(4.8mm)が多用される。代表的なリベット接合部と各部の名称を図
