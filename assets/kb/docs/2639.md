# 3・9・2 リベットの種類

> **章**: 第3章 溶接・接合および切断
> **節**: 3・100 に示す。

出来頭の形状により、丸頭(protruding head)、皿頭(flushhead)の2種類がある。皿頭は結合部表面が出っ張らないような形状のものであり、空気抵抗低減をねらった航空機への使用を目的に開発されたものである。皿頭リベットを取付ける場合は、出来頭に合わせた皿取りを行う。装着する皿頭リベットの皿頭の高さより薄い板厚の場合は皿取りを行わず、図3・100(c)のような皿出しを行う。なお、より薄い板厚の場合でも皿取りを可能とした、頭部が小さいシアヘッドリベット(NAS 1097等)もある。疲労強度標定部位においては、皿取りに必要な板厚は一般に皿頭高さの1.5倍以上である。機能の面から見ると、構造の裏側にアクセスできない袋構造への適用を可能にしたブラインドリベット、インテグラルタンク部など取付け部の気密性が要求される部位に使用されるスラグリベットなどがある。これらは通常のリベットと異なり、専用のツールを用いて打びょうする必要があり、必要最小限の使用にとどめる。リベットの材質は、アルミニウム合金(2017,2024等)、モネル、耐食鋼などがあり、用途に応じて選定する。*2:せん断許容荷重は、上記リベット孔の径に対応する値である。を必要とする部分は2024アルミニウム合金製のDD びょう、耐食・耐熱性が要求される部位にはモネルびょうが使用される。
