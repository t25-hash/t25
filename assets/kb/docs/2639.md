# 3・7・2 非破壊検査（nondestructive inspection）

> **章**: 第3章 表面損傷
> **節**: 3・7 損傷の検出と診断

材料の内部き裂の発見や探傷に超音波やX線が使われることが
多いが，トライボロジーの分野では，関連するき裂や損傷の広がり
が10ミクロンオーダと極めて小さいので，一般に用いられる探傷
機器は必ずしも有効ではない．近年AE（acoustic emission）を用
いて転がり接触面下の微小き裂を検出し，転がり軸受の残存余命を
推定する手法も考案されている．これについては3･7･3項で述べ
る．
トライボロジーの分野では，トライボ要素が多くの場合，潤滑下
で使用されることから，潤滑油を分析することによって損傷に関す
る種々の情報を入手することができる．すなわち，潤滑油は使用中
にさまざまな外部要因で変質して劣化するが，同時に機械設備など
から排出される金属摩耗粉などのごみにより汚染される．潤滑油の
劣化と汚染を調べる方法には，図Ⅱ-3･29に示す，性状分析，組成
分析，金属分析，の三つがある．
密度，色，粘度などを調べる性状分析は比較的手軽であるため，
現場の業務として行われる場合が多い．組成分析ではクロマトグラ
フィーなどの機器が使われる．

なお，油中の摩耗粒子を磁気こう配で分離し，大きさの順に配列す
るフェログラフィーは金属分析の一種である．

断 ，製鉄プラントの油定期診断 などがある．
