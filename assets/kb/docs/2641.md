# 3・1・1 製品企画とは

> **章**: 第3章 設計のための個別方法論
> **節**: 3・1 企画のための方法

これらの製品，価格，流通，販売促進の四つを，その英語の頭文字をとって，マーケティングミックスの（あるいは，単にマーケティングの）4Pという．
このうち，製品企画と最も深く関わってくるのは，言うまでもなく製品（product）である．
次に，製品のコスト設計を通じて価格（price）が関わり，また製品が対象とする顧客セグメントを通じて流通（place）および販売促進（promotion）が関わってくる．
これらのマーケティングとの関わりを無視して製品企画を行うと，売れない製品を作ってしまうことになりかねない．
a.製品企画（product planning）の意義と位置付け 技術の視点からはいかによい製品であっても，その製品が顧客に受け入れられ，かつ競合する製品に打ち勝って利益を生むことができなければ，企業にとっては意味がない．
また，この利益が十分なものでなければ，次期製品への開発投資を削減せざるを得なくなってしまい，その結果，製品開発活動そのものが停滞するという悪循環に陥ってしまう．
