# 3・9・4 リベット接合の強度

> **章**: 第3章 溶接・接合および切断
> **節**: 3・100 に示す.

リベットは一般にせん断荷重を伝達するよう設計する.1本当たりの許容荷重は材質やサイズにより大きく異なり,結合する板厚の面圧強度とバランスをとるよう,サイズを決める.おもなリベットのせん断強さを表3・38に示す.リベットは複数本を用いるのが通常であり,リベットパターンによる分担荷重の不均一や不適切なリベットピッチの適用は,疲労強度(fatigue strength)低下を招くことがあるので,十分注意しなければならない.また,リベット1本当たりの許容荷重だけでなく,結合される板部の強度についても考慮が必要であり,びょう間座屈や板端部からの縁距離の確保などの検討も行う.さらに,張力場として設計されている薄板外板の周囲をリベット接合する場合は,リベットにせん断だけでなく引張荷重が加わるので,これに対する配慮も欠かせない.金属構造に対する一般的なリベット接合部では,以上の強度要件を満たす経験値として,下記が目安とされる(リベット径,孔中心から外板端までの距離,外板板厚とする).リベット径と板厚:/=3縁距離:≧2リベットピッチ:≧4≦8または24の小さいほう千鳥間隔:≧3なお,複合材構造の場合には,一般により大きな値を必要とする.
