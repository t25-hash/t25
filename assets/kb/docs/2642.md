# 3・9・5 加工方法

> **章**: 第3章 溶接・接合および切断
> **節**: 3・100 に示す.

手作業によるリベット接合は,一人が孔に挿入したリベットの出来頭にリベットガンをあてがい,もう一人が当て板で裏側から押さえてリベット軸をつぶし,結合するというものである.結合すべき構造物が大きくなると,互いの連絡が取りにくくなり効率が落ち3―89表3・38おもなリベットのせん断強度3/321/85/323/160.0960.12850.1590.1912.443.264.044.85せん断許容荷重NA 286NAS 1198NAS 120011450モネルMS 20615MMS 20427Mる.したがって,胴体外板などの広い部分を結合する場合は,自動リベッタが使用される.これはNC加工情報に基づき,下孔あけから皿取り,リベット差込み,打びょう,製品の送りまでをこなす加工ロボットであり,高速かつ安定した成形品質が得られる.なお,結合した部品の修理や交換のためリベット接合部を取り外す場合は,リベットの頭部を切断して取り外すことになる.再取付けする場合は,修理後の部品のリベット孔部を1サイズ大きな孔に加工し,1サイズ大きなリベットで再打びょうするのが一般的である.
