# 3・10・2 接着剤

> **章**: 第3章 溶接・接合および切断
> **節**: 3・10 接着接合

接着においては、被接着材料や使用環境に適した接着剤の選択が重要となる。a. 縮合系接着剤尿素樹脂、フェノール樹脂など熱硬化性樹脂をベースとする接着剤で、樹脂成形と同様の方法により硬化、接着する。主たる用途は合板の製造である。b. 溶剤型接着剤合成樹脂またはゴムを主成分とし、溶剤に溶かして接着剤としたものである。多くの分野で使用されているが、溶剤の毒性や溶剤揮発による大気汚染があり、脱溶剤が進みつつある。c. 水性型接着剤合成樹脂乳化物や合成ゴム乳化物を接着剤3―9 0加工学・加工機器としたものである。家具・木工や繊維、紙工関連で消費されている。d. ホットメルト型接着剤加熱溶融した接着剤を塗布し張り合わせるタイプの接着剤である。エチレン-酢酸ビニル重合体をベースとするものが歴史的に古く、現在でも大量に使用されている。e. 反応系接着剤低分子量であるモノマあるいはオリゴマが化学反応により固化し接着するタイプである。主剤と硬化剤からなる2液型と1液型がある。
