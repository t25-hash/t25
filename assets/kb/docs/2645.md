# 3・10・3 種々の接着法

> **章**: 第3章 溶接・接合および切断
> **節**: 3・10 接着接合

a. ウェルドボンド法(weld-bonding)接着継手とスポット溶接のような溶接継手の互いの欠点を補いつつ、より継手強度が高く信頼性の高い接合部を得る手法として、接着と接合を組合わせたウェルドボンド法が1960年代初期に旧ソ連で開発され、自動車や航空・宇宙関連分野で広く用いられている。接着部に対しては、はく離強度や耐クリープ性などが改善でき、スポット溶接部に対しては、面接合の効果により応力集中が小さくなり、疲労特性が向上する。ウェルドボンド法の目的には、(1)接着特性の改善:接着継手の欠点であるはく離強度、衝撃強度、高温強度の改善、(2)点溶接特性の改善:アルミニウム合金や薄鋼板の点溶接強度の低さの改善、および点溶接の疲労特性の改善、(3)接着工程の合理化:点溶接を接着硬化するまでのジグ代わりに用いるなどの合理化などがある。b. 溶着法熱可塑性プラスチック材を溶かし、プラスチック相互または他の材料と接着する方法である。一般に接着には圧力が必要である。接着剤を溶かして硬化させる方法もこれに属する。溶着には外部加熱による方法として熱風溶着、熱板溶着、内部加熱による方法として、高周波誘導溶着、摩擦溶着、超音波溶着などがある。(c) 被接着体の表面層における破壊(b) 接着剤と被接着体の界面破壊(a) 接着剤間での破壊図3・101 接着接合にて発生する破壊形態図3・102 接着継手強度に対する被接着体厚さおよび継手種類の影響
