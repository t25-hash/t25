# 3・10・4 接着接合での破壊形態

> **章**: 第3章 溶接・接合および切断
> **節**: 3・10 接着接合

接着接合において破壊が生じる界面は,大別すると図3・101に示す3種類となる.図3・101(a)は凝集破壊である.このような場合に接着力向上のために表面あるいは界面状態を改善して強度向上を考えることはむだであり,接着剤自体の物性または接着硬化条件等を改良する必要がある.図3・101(b)の破壊形態の場合,接着性向上に適するように表面構造を改善するか,被接着体と接着剤との界面相互作用を大きくするような方策,あるいは相互作用を阻害する要因(例えば表面汚染など)を除去することが有効である.図3・101(c)の破壊形態の場合,破壊は被接着体の内部で発生している.したがって,接着性を向上させるためには被接着体の表面構造あるいは表面層の強化を行う必要がある.以上のように,接着部で破壊が生じた場合には,その破壊面の詳細な観察を行うことが重要であり,それぞれの破壊に適した対策を講じることで接着性は向上する.接着接合の継手強度は,継手の種類や被接着体厚さによって変化する.図3・102は,これらの因子の継手強度に及ぼす影響およびその際に発生するおもな破壊形態を示したものである.図3・102より,被接着体厚さが増加すると継手強度が増加するものの,ある板厚以上では強度は飽和する傾向がある.また,スカーフ継手やステップドラップ継手といった荷重の偏心や応力流れが改良された継手の強度が高いことがわかる.しかし,これらの継手は複雑な機械加工を必要とするため,高強度を要する場合には有効であるが,低強度でも済む場合には加工性を考慮すると必ずしも有効な継手とはいえない.したがって,要求強度と機械加工性を考慮して継手を決定する必要がある.
