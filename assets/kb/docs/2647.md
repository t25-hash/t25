# 3・11・1 試験および検査一般

> **章**: 第3章 溶接・接合および切断
> **節**: 3・11 試験および検査

溶接方法として種々のものが利用されているが、その多くは各種熱源を用いて材料を溶融して接合を行う溶融溶接が適用されている。この場合には、材質の変化が生じるとともに、溶接残留応力あるいは変形、ならびに各種の欠陥が発生することもある。溶接部には、荷重に耐える強度や延性のほか、使用条件に応じて脆性破壊、疲労、腐食や応力腐食割れなどに対する性能が要求される。これらの性能には、材料特性とその溶接による変化、溶接部の形状と欠陥、ならびに残留応力などが影響を及ぼす。表3・39に、溶接部の性能とその試験方法を示す。溶接部の品質を保証するためには、溶接前、溶接中、溶接後、そして溶接物の稼働中の各段階で適切な非破壊試験を適用して、幾何学的な欠陥を主とした欠陥の有無を調べ、欠陥が検出された場合には、その製品の適用規格に照らし合わせて不具合なものは欠陥の除表3・39溶接部の性能とその試験方法溶接金属突合せ継手すみ肉継手母材硬さ試験疲労試験分析試験腐食試験溶接部の性能引張り引張り衝撃引張り曲げ衝撃引張り曲げ引張強さ○○○○降伏点(耐力)○○△延性○○△○○硬さ○△破壊靱性○○疲労強さ△○化学成分○耐食性○○去および補修をする必要がある。欠陥が溶接物の性能に及ぼす影響については、あらかじめ各種機械試験(破壊試験、destructivetest)を用いて把握しておく必要がある。3・11・2項に述べるように、溶接部に対しては種々の非破壊試験方法が適用されるが、これらの試験結果と適用規格や基準に基づいてその対象部の合否を判定することを検査という。
