# 3・11・2 非破壊試験

> **章**: 第3章 溶接・接合および切断
> **節**: 3・11 試験および検査

a. 非破壊試験の適用非破壊試験(nondestructive test)においては,溶接部を破壊せずに評価するため,溶接部の全長について検査することも可能である.表3・40に,溶接部の各種欠陥とその検出に用いられる非破壊試験方法を示す.非破壊試験方法には万能なものはなく,それぞれが長所と短所を有しているため,長所を生かすためにいくつかの試験方法を組合わせて適用している.非破壊試験方法は,表面および表面付近の欠陥の検出を得意とする方法と,内部欠陥の検出を得意とする方法に大別できる.同種・同寸法の場合であれば,内部欠陥に比べると,表面あるいは表面付近の欠陥のほうが溶接継手の機械的性能に及ぼす影響が大きいため,非破壊試験方法の適用に際しては,まず表面あるいは表面付近における欠陥の有無を調べることが重要である.また,内部欠陥の中では,大まかに,割れ,溶込み不良,融合不良,スラグ巻込み,そしてブローホールの順に影響は小さくなるといえる.母材の材料特性,溶接部の特性および欠陥発生の傾向,欠陥の許容限度,溶接物の形状・寸法,ならびに継手形式などを考慮して,適切な非破壊試験方法を適切な頻度(全数検査,抜取り検査,検査密度など)で用いる必要がある.また,非破壊試験施工要領書の作成,溶接検査技術者・非破壊検査技術者の活用など,試験と検査の計画・実施・管理について,溶接施工管理と同様な考え方の採用が望まれる.また,欠陥の検出だけでなく,溶接継手の劣化診断などの材料評価にも種々の非破壊試験方法が適用されている.b. test)は,目視では発見できない表面または表面直下の欠陥を調べる目的で,割れや微小なきずの発生のおそれのある溶接部の検査,ジグ溶接取外し後の検表3・40 1査,ベベル面のラミネーションの検査などに用いられる.表面近傍で表面に開口していない欠陥も検出できるが,強磁性体にしか適用できない.磁化方法としては極間法とプロッド法が用いられている.プロッド法では,電極を通して大電流を流すため,高張力鋼,合金鋼など硬化性・低温割れ感受性の高い材料に適用する場合には,表面を損傷しないように注意が必要である.浸透探傷試験(PT : penetrant test)は非磁性体にも適用できるが,表面に開口していない欠陥は検出できない.表面欠陥を検出する際の信頼度は,試験方法,探傷条件,探傷手順および表面の状態などによって大きく変化する.試験の目的と欠陥の許容条件に応じて,適切な方法・手順(清浄を含む)を定め,必要な場合には溶接部表面の仕上げを行うべきである.c. 内部欠陥の検出表面に開口していない割れ,ブローホール,スラグ巻込み,融合不良,溶込み不良などの欠陥の検出には,放射線透過試験(RT : radiographic test)または超音波探傷試験(UT : ultrasonic test)が用いられる.放射線透過試験は,原理上,三次元欠陥の検出に適し,割れのような欠陥は,すきまの量や放射線方向と割れの方向との関係によっては検出困難となる.重要な突合せ溶接部の内部欠陥の検査に広く用いられており,適切な試験条件を用いて良好な透過写真を得ることが重要である.X 線フィルムを用いないディジタルラジオグラフィーも広く利用されてきている.超音波探傷試験も基本的には内部欠陥の検査に用いられる(表面波を用いた方法では表面欠陥の検出も可能である).設備は軽量で,かつ,片面から探傷できる.平面状の欠陥の検出性能が優れているとともに,探傷結果の画像化が格段に進歩してきたため,従来の放射線透過試験に代わって適用され始めている.通常,斜角探傷法(angle beam inspection method)を用いるが,鋼板のラミネーションの検出や特殊な条件では垂直探傷法(normal beam inspec-tion method)も用いられる.TOFD(time of flight diffraction)法やフェーズドアレイ法など新しい方法が開発され,威力を発揮している.d. その他の非破壊試験方法渦電流探傷試験(ET : eddycurrent test)の溶接部への適用が進められており,また,アコースティックエミッション試験(AET : acoustic emission test)やサーモグラフィー試験(thermography test)なども溶接部の非破壊試験方法として適用されている.
