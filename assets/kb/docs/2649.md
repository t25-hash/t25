# 3・11・3 溶接性試験

> **章**: 第3章 溶接・接合および切断
> **節**: 3・11 試験および検査

溶接性(weldability)は、溶接される材料の溶接の難易度を表す用語として用いられており、(1)欠陥のない健全な溶接ができるかという、溶接加工上の溶接性(狭義の溶接性)、(2)溶接後の継手性能が構造物の使用目的を満足させることができるかという、使用性能に関する溶接性(広義の溶接性)、との二つの意味で使われている。加工上の溶接性として考慮すべきものには、溶接金属および熱影響部の高温割れ、低温割れ、気孔、スラグ巻込みといった溶接欠陥と溶接部の形状がある。例えば、ビード形状の不良が応力集中を引き起こして疲労破壊や使用中脆化につながる例もあるためである。一方、使用性能上の溶接性には母材および溶接部の機械的性質、すなわち強度、延性、切欠靱性、破壊靱性がある。そのほか使用目的に応じ、疲れ強さ、高温強度、クリープ特性、耐食性、耐応力腐食割れ性なども重要な項目である。鋼材の溶接においては、溶接中の急熱、急冷熱サイクルにより熱影響部は母材より硬くなり、これが溶接部における特性劣化の要因となる場合がある。また、熱影響部の低温割れ感受性も高くなる。すなわち、熱影響部が硬くなりやすい鋼材はそれだけ溶接性が悪いといえる。構造物全体の信頼性はさまざまな観点から評価しなければならないが、溶接部の特性によりほぼ決定されてしまう場合が多いため、特に熱影響部の硬化性と低温割れ感受性は、鋼材の溶接性を考えるうえで重要な項目である。3―9
