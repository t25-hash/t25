# 4・1・1 総論

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

機械部品の製造法には，大きく分けて除去加工（切削加工，砥粒
加工，放電加工，レーザ加工，エッチングなど），成形加工（鋳造，
鍛造，プレス成形，粉末冶金，プラスチック成形など），および付
加加工（溶接，溶射，めっき，CVDなど）の三つに分けることが
できる．塑性加工は成形加工に含まれ，型を用いて素材に弾性限度
以上の応力を加えて変形させ，材料の塑性を利用して所望の形状寸
法に成形する加工法である．
塑性加工は板，線，棒，管，形材などの素材の製造に利用され，
これらの素材は，さらに鍛造，転造，押出し，プレス加工などの各
種の塑性加工を受けてさまざまな製品となる．素材の製造におい
て，塑性加工は材料の組織と機械的性質の改善に役立っている．型
やロールなどの工具を用いて製造する塑性加工の特徴は，高速で高
度の再現性をもって同一寸法形状品を効率よく大量に生産する加工
法として適していることと，切りくずを出さないで成形できる省資
源・省エネルギー型の加工法であることである．
圧延，鍛造，押出しなどの圧縮応力を主体とする加工は，鋳造材
の空洞を圧着し，粗大結晶粒を微細化し，材質を均質化する鍛錬作
業としての意義が大きい．また，再結晶温度以下の温度域で塑性加
工を施すと，塑性ひずみに伴い加工硬化が生じ，材料の強度は増加
する．塑性加工と熱処理を組合わせた加工熱処理は，結晶粒を微細
化して強度と靱性の向上に役立つ．圧延および押出しなどによる異
種材料の圧着は，複合材料の製造に応用されている．また，塑性加
工による圧縮残留応力は疲れ強度の向上に役立つ．
塑性加工に対する省資源，省エネルギー，省力，高品質，低コス
トの要求がいちだんと強まり，多種少量生産への対応が迫られてい
る．多種少量生産に対しては型の設計・製作に時間を要し，高価で
あることが問題となる．この問題の解決のためにCAD/CAMシス
テム，マシニングセンタや放電加工機などのNC工作機械や高速
切削に対応できるcBN工具および表面処理法などが開発されてい
る．また，塑性加工のFMS化もなされている．
日本における金属成形加工品の生産額（総額3兆9294億円）の
割合は，2000年の『素形材年鑑』 によると，鋳造品が48.7 %，粉
末冶金品が7.8 %であり，塑性加工品は43.5 %（うち鍛造品は
16.7%，金属プレス製品は26.8%）であった．塑性加工品の用途
は多岐にわたっているが，量的に見ると自動車等の車両用が圧倒的
に多く60～70 %に達するものと見られている．自動車，電気・電
子機械，産業機械，精密機械は日本の代表的な工業製品であり，こ
れらの機械は多くの成形加工品から成っている．これらの工業製品
の生産量（重量）は減少傾向にある．
