# 4・1・1 総論

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

機械部品の製造法には、大きく分けて除去加工(切削加工、砥粒加工、放電加工、レーザ加工、エッチングなど)、成形加工(鋳造、鍛造、プレス成形、粉末冶金、プラスチック成形など)、および付加加工(溶接、溶射、めっき、CVD など)の三つに分けることができる。塑性加工は成形加工に含まれ、型を用いて素材に弾性限度以上の応力を加えて変形させ、材料の塑性を利用して所望の形状寸法に成形する加工法である。塑性加工は板、線、棒、管、形材などの素材の製造に利用され、これらの素材は、さらに鍛造、転造、押出し、プレス加工などの各種の塑性加工を受けてさまざまな製品となる。素材の製造において、塑性加工は材料の組織と機械的性質の改善に役立っている。型やロールなどの工具を用いて製造する塑性加工の特徴は、高速で高度の再現性をもって同一寸法形状品を効率よく大量に生産する加工法として適していることと、切りくずを出さないで成形できる省資源・省エネルギー型の加工法であることである。圧延、鍛造、押出しなどの圧縮応力を主体とする加工は、鋳造材の空洞を圧着し、粗大結晶粒を微細化し、材質を均質化する鍛錬作業としての意義が大きい。また、再結晶温度以下の温度域で塑性加工を施すと、塑性ひずみに伴い加工硬化が生じ、材料の強度は増加する。塑性加工と熱処理を組合わせた加工熱処理は、結晶粒を微細化して強度と靱性の向上に役立つ。圧延および押出しなどによる異種材料の圧着は、複合材料の製造に応用されている。また、塑性加工による圧縮残留応力は疲れ強度の向上に役立つ。塑性加工に対する省資源、省エネルギー、省力、高品質、低コストの要求がいちだんと強まり、多種少量生産への対応が迫られている。多種少量生産に対しては型の設計・製作に時間を要し、高価であることが問題となる。この問題の解決のためにCAD/CAM システム、マシニングセンタや放電加工機などのNC工作機械や高速切削に対応できるcBN 工具および表面処理法などが開発されている。また、塑性加工のFMS 化もなされている。日本における金属成形加工品の生産額(総額3兆9294億円)の割合は、2000年の『素形材年鑑』によると、鋳造品が48.7%、粉末冶金品が7.8%であり、塑性加工品は43.5%(うち鍛造品は16.7%、金属プレス製品は26.8%)であった。塑性加工品の用途は多岐にわたっているが、量的に見ると自動車等の車両用が圧倒的に多く60~70%に達するものと見られている。自動車、電気・電子機械、産業機械、精密機械は日本の代表的な工業製品であり、これらの機械は多くの成形加工品から成っている。これらの工業製品(例:50mmに対する公差/mm)0.0110.0160.0250.0390.0620.1000.1600.250.390.621.6ISO等級加工方法熱間型鍛造温間型鍛造冷間型鍛造サイジングしごきせん断精密せん断旋削円筒研削3―9 5はいずれも代表的な輸出品であり、品質的に優れた成形加工品が国内で安価に供給されることにより、国際競争力が高いと見られていた。この分野における日本の技術は世界のトップレベルにあるが、最近は賃金の安い国において成形品が製造されるようになり、国内の生産量(重量)は減少傾向にある。
