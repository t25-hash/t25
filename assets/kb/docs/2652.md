# 4・1・3 塑性加工製品の精度

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

多くの部品製造法では最終製品形状に近いもの[ニアネットシェイプ(near net shape)品]が得られるが,完成部品の精度は不十分とされてきた.熱間鍛造品はこのイメージに合致するものの,冷間鍛造品や板金プレス加工品はすでにネットシェイプ(netshape)の域にあり,後加工なしで使われているものが多い.後加工を省略できる高精度化技術を確立することにより,生産性を高めて生産コストを大幅に低下させることができる.多くの成形加工では,従来に比べよりネットシェイプ化や高精度化のための努力が続けられている.塑性加工の分野では鍛造における閉そく鍛造,温間鍛造や冷間鍛造,板金プレス成形でのしごき加工や精密打抜きなどが普及している.図4・1に各種加工法により達成できる寸法公差(dimensionaltolerance)の比較を示す.冷間型鍛造,サイジング(コイニング),しごき,精密せん断は旋削並みの加工精度が得られる.塑性加工品の精度は,加工機械の動的精度と金型精度によって決まる.塑性加工では加工力が大きいため,機械および金型の弾性変形が大きく,切削加工や研削加工ほどの高い加工精度は期待できない.しかし,冷間プレス加工品の精度は著しく向上している.これは,電子部品を中心として,製品に高い精度が強く要求されていることと,金型加工技術の向上によるものである.特に薄板の打抜き品や曲げ加工品では,切削加工に匹敵する精度が得られている.塑性加工品のJIS 寸法許容差は,プレス加工品では最高±0.1mmを規定している.熱間鍛造においては弾性変形のほかに熱の影響も加わるため,寸法許容差は1mm以上になっている.表4・2は実用的なプレス加工品の精度基準と加工限界を示したもので,実際の受注・発注の際の寸法許容基準はおおむねこの基準に沿っている.
