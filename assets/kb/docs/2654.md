# 4・1・5 各種金属の塑性変形特性

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

塑性加工に利用する金属材料は、図4・2に示すFCC(面心立方)、BCC(体心立方)およびHCP(最密六方)の3種類の結晶構造のいずれかの構造をもつ金属原子が規則的に配列している。具体的には、面心立方金属としてはアルミニウム、銅、体心立方金属としては鉄、最密六方金属にはマグネシウム、チタニウムなどがある。それらの単結晶の金属材料に力を加えると、それぞれの結晶構造の最密の原子配列をしている一定のすべり面で、せん断力の作用によるすべりによって原子が一定のすべり方向に移動し、塑性変形を生じることになる。このように、すべり系の多い結晶構造をもつ金属が塑性変形を起こしやすいことから、すべり系が多いFCC金属やBCC金属が塑性加工用金属材料として多く用いられている。他方、すべり系の少ない最密立方金属を室温で塑性変形させることは非常に困難である。実際の金属材料は多結晶で存在しており、結晶粒の大きさが変形抵抗などに影響し、結晶粒径が小さくなれば変形抵抗(flowstress)が増加することが知られ、ホール・ペッチ(Hall-Petch)の式として与えられている。また、金属材料に塑性ひずみを与えると、金属材料の変形抵抗は加工硬化により大きくなる。これらの特性は、いずれの結晶構造をもった金属材料においても現れる。これらの塑性加工金属材料の塑性変形のしやすさを示す尺度を、加工性または成形性などと呼んでいる。その加工性または成形性を示す定量的な値として、材料が破断するまでの変形量を定義し、それを材料の変形能(ductility)と呼んでいる。金属材料の変形能は、引張試験においては、破断伸びや絞りにより評価される。ここで、は初期断面積およびは破断後のくびれ部の断面積である。圧縮変形においては、据込み試験(upsettability test)における欠陥または破壊発生までのひずみによって評価されている。その据込み試験は、高さと直径比が1.5の円柱試験片を用い、図4・3に示す同心円溝付き耐圧板を用いて軸方向に据込み、試験片側面中央部に長さの5~1.0mmの割れを発生させる。これらの試験により得られた変形能の尺度から、各種金属材料の変形能は次のように整理することができる。変形能の優れた金属材料としては、結晶の対称性に優れた面心立方金属、体心立方金属、すべり系の多い遷移温度以上の体心立方金属、および遷移元素の最密六方金属、ならびに緻密な組織をもつ超塑性材料などがある。一方変形能に問題のある金属材料としては、結晶の対称性の少ないりょう(菱)面体金属および金属間化合物、ならびにすべり系が少ない遷移温度以下での体心立方金属、および遷移元素以外の最密六方金属などがある。
