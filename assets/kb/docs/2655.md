# 4・1・6 材料の変形抵抗

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

材料の力学的特性を評価するために最も広く用いられる材料試験は、引張試験法である。引張試験法はJIS で規定され、丸棒あるいは帯板の試験片を一方向に引張変形させ、荷重と伸びを記録させる。荷重-伸び曲線より、公称応力-公称ひずみ曲線を求め、比例限度、弾性限度、降状点または耐力、引張強さ、一方、一様伸び、局部伸びを求める。変形が大きい場合は、公称ひずみに対して対数ひずみ(真ひずみ)を用い、公称応力に対して真応力で表した真応力-真ひずみ曲線を用いる。この応力-ひずみ曲線は変形抵抗曲線(flow stresscurve)と呼ばれる。塑性加工において用いる変形抵抗曲線はσ=+ε욗[ルードウィク(Ludwick)の式](4・2)σ=+ε욗[スイフト(Swift)の式](4・3)などの形で近似された式が用いられる。簡単にはルードウィクの式において=0とした式が用いられることが多い。ここで、は塑性係数、は加工硬化係数と呼ばれる。変形抵抗は冷間、熱間に限らず、ひずみε、ひずみ速度ε、温度に依存し、次式のように表現される。σ=ε,ε,(4・4)一般に、冷間において変形抵抗は加工硬化により影響を受けるので、ひずみに大きく依存し、ひずみ速度にはあまり依存しない。一方、熱間において変形抵抗は再結晶により影響を受けるので、ひずみにはあまり依存しないで、ひずみ速度に大きく依存する。一般に金属の冷間変形抵抗としては、式(4・4)を次式σ=σε+σε,ε,(4・5)として記述することがある。ここで、σεはひずみ速度と温度に依存しない変形抵抗の成分、σε,ε,はひずみ速度と温度に依存する変形抵抗の成分である。低炭素鋼、17%Crステンレスの鋼ならびにアルミニウムおよびアルミニウム合金の冷間変形抵抗式としては、次の五弓らの式がある。低炭素鋼の変形抵抗はσ=σε+9.8εεε요(MPa)(4・6)17%Crステンレスの変形抵抗はσ=σε+9.8ε요(MPa)(4・7)アルミニウムおよびアルミニウム合金の変形抵抗はσ=σε1+0.023
