# 4・1・6 材料の変形抵抗

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

材料の力学的特性を評価するために最も広く用いられる材料試験は、引張試験法である。引張試験法はJIS で規定され、丸棒あるいは帯板の試験片を一方向に引張変形させ、荷重と伸びを記録させる。荷重-伸び曲線より、公称応力-公称ひずみ曲線を求め、比例限度、弾性限度、降状点または耐力、引張強さ、一方、一様伸び、局部伸びを求める。変形が大きい場合は、公称ひずみに対して対数ひずみ(真ひずみ)を用い、公称応力に対して真応力で表した真応力-真ひずみ曲線を用いる。この応力-ひずみ曲線は変形抵抗曲線(flow stresscurve)と呼ばれる。簡単にはルードウィクの式において=0とした式が用いられることが多い。ここで、は塑性係数、は加工硬化係数と呼ばれる。変形抵抗は冷間、熱間に限らず、ひずみε、ひずみ速度ε、温度に依存し、次式のように表現される。一方、熱間において変形抵抗は再結晶により影響を受けるので、ひずみにはあまり依存しないで、ひずみ速度に大きく依存する。低炭素鋼、17%Crステンレスの鋼ならびにアルミニウムおよびアルミニウム合金の冷間変形抵抗式としては、次の五弓らの式がある。次に、炭素鋼の熱間平均変形抵抗σの式としては、美坂らの式と志田の式がある。志田の式は、以下で与えられる。式(4・9)の有効範囲は<1.2%,ε<0.5,ε=30~200s,=973~1673K であるが、変態点以下では計算値の的中率が低く、補正係数を乗じて使用する必要がある。冷間から温間領域における炭素鋼変形抵抗式としては、次の小坂田らの式がある。
