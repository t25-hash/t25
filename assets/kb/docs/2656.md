# 4・1・7 塑性加工における摩擦・潤滑

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

a. 特徴圧延、引抜き、押出し、鍛造などの塑性加工界面でのトライボロジー特性の特徴を以下に挙げる。(1) 塑性加工面の潤滑(lubrication)は、一般に流体潤滑よりも境界潤滑に近い状態にある。多くの塑性加工の界面の摩擦係数(coefficient of friction)は、ほぼ0.02~0.15の間にあり、μ<0.01の機械要素界面の摩擦係数よりもかなり高い値になっており、このような摩擦係数の値においては境界潤滑と流体潤滑とが混在した混合潤滑状態にあることがいえる。このことから、流体潤滑が主流である軸受などの摩擦(fric-tion)に比べて、塑性加工界面においては境界潤滑特性をかなり考慮しなければならない。(2) 塑性加工界面が材料の弾性限をはるかに超えた高い面圧下にあり、しかもその面積が極めて広い。塑性加工面の荷重が大きくなると、その境界膜の破断を引き起こす可能性が考えられる。その境界膜の破断の臨界荷重は、温度や速度により影響を受けているが、実験より2000~3000MPaであることが示されている。この値は、一般の多くの塑性加工の界面にかかる面圧と比較するとかなり高い値であり、塑性加工面にかかる面圧が機械要素どうしの接触に比べ油膜破断の大きな原因になるとは考えにくい。一方、潤滑油の破断は面圧よりも界面の滑り距離に大きく影響を受ける。塑性加工面のように接触面積が大きい場合には滑り距離が大きくなる。その増加によって界面の摩擦温度は高くなり、接触時間も長くなるため、機械要素面に比べ塑性加工面においては油膜の動的破断が起こりやすい。(3) 塑性加工面の温度が高い。塑性加工面温度は、荷重、速度、滑り距離や摩擦係数の影響を受け、冷間圧延加工の場合には、材料の加工熱も考慮すると200°C以上にもなることがある。一方、潤滑油膜は特定の温度(転移温度)以上になると、潤滑能力が著しく低下することが知られている。この転移温度は潤滑油によって少し異なるが、200°C前後であるので、油膜破断の大きな原因となる。この式(4・11)(4・12)を使って塑性加工界面の界面温度を求めることができる。(4) 塑性加工界面内で滑り速度が材料の塑性変形により部分的に変化し、しかもその方向も変化することがある。軸受などの機械要素の接触面とは異なり、材料の塑性変形により滑り方向に速度が変化することになる。これは、流体膜が形成しているときには負荷の原因となる。特に、圧延の場合には中立点で滑り速度が零となり、方向を逆転する。(5) 塑性加工界面で加工中に新生面が露出する。塑性加工面でのトライボロジー特性を特徴づける最も大切な因子として、新生面の露出と材料の表面構造の変化が挙げられる。塑性変形により接触面が増大した場合、外部からの潤滑油の供給がなければ、その増加分につれて油膜は減少する。特に、表面拡大率の大きな鍛造加工や押出し加工においては、新生面が露出する部分への潤滑剤が薄くなると、油膜破断を引き起こす原因となる。一方、界面での材料の表面構造の変化により多量の潤滑油が接触面に介在すると、表面品位の劣化を引き起こす原因となる。このような塑性加工における工具と材料間の界面において、(1)加工力の低減、(2)冷却、(3)製品表面品質の改善、(4)焼付きや表面損傷の防止、および(5)工具摩耗の抑制などのために潤滑油が使用される。しかし実際には、塑性変形した突起が真実接触領域において垂直応力とせん断応力τの間で降伏条件を満足しなければならない。ここで、αは定数、は式(4・16)を満足しているときの真実接触面積である。これは、ジャンクション成長(junction growth)と呼ばれている。図4・4のγは突起角、(a)~(d)は圧下の進行を示す。一方、実際の多くの塑性加工の場合(例えば圧延、鍛造、押出し、引抜き加工など)には、バルクが塑性変形する場合、材料表面の平坦化に対してバルク変形の影響を考慮する必要がある。牧野内らは無限個の突起モデルに弾塑性有限要素法を用いて変形解析を行い、バルク変形がある場合には、接触率は無次元化垂直圧力が1の近くでほぼ100%となり、容易に材料表面が平滑化されることを示している。ここで、τは境界潤滑領域でのせん断応力である。しかし、塑性加工界面を工具-潤滑油-材料の一つの系の中でトライボロジー(tribology)特性を考えることが試みられている。まず、工具と材料が接触(塑性変形)を開始する点では、工具と材料との速度による潤滑油の動水圧効果のくさび効果による導入量、あるいはスクイーズ効果による捕捉量が各種加工条件により定量的に決められる。また、工具と材料の凹部では潤滑油を界面に持ち込む効果として作用する。塑性変形域の界面においては、材料が工具に微視的に接触する境界潤滑領域、材料-工具界面の凹部において静水圧を生じている静圧流体潤滑領域、および静水圧を生じていない流体潤滑領域などが混在共存している混合潤滑を呈している。加工条件が厳しくなると、境界潤滑領域が増加して局部的に金属接触が発生し、固体摩擦域が出現することになる。この塑性加工界面の接触モデルが軸受など機械要素に生じる弾性体どうしの接触と異なる点の一つは、図4・6に示すような混合潤滑域において、界面に持ち込まれた材料凹部の潤滑油、または材料の塑性変形により生じたポケットに閉じ込められた潤滑油に静水圧が発生することである。この静水圧が真実接触域に作用する圧力に近づき、凹部またはポケット内の潤滑油にも動水圧作用が発生すると考えると、図4・7に示すような凹部に封じ込め図4・6 静水圧が作用した潤滑状態図4・7 ミクロ塑性流体潤滑のモデル3―100加工学・加工機器られた潤滑油が凹部周辺の接触領域に浸出して、流体潤滑作用をするミクロ塑性流体潤滑の可能性が示唆された。このような塑性加工界面におけるトライボロジー特性を理解して、機械要素の場合と同様、導入(捕捉)油膜厚さと工具と材料の表面粗さの和との関係から、塑性加工の場合も次のように潤滑メカニズムを分類することができる。特に作業環境の改善としては、熱間鍛造における黒鉛系潤滑剤に代わる潤滑剤の開発や、冷間鍛造に用いられるリン酸被膜処理に代わる潤滑剤の開発が行われている。一方、塑性加工における加工条件が高生産性化、難加工材の使用などの観点から過酷化しており、工具の面から耐摩耗性、耐焼付き性の向上した工具の開発に力が注がれてきた。特に鍛造およびプレス加工用の工具の耐摩耗性の観点から見ると、炭素系工具鋼、低合金工具鋼、ダイス鋼、高速度鋼、超硬合金の順に耐摩耗性の要求度の高い順に使い分けられている。表面処理材は、比較的安価で強靱な工具材料の表面を硬くかつ材料と凝着を起きにくい物質によって覆うという考え方が発展し、現在では鍛造プレス加工用に多く使用されている。現在使用されている表面処理工具は、硬質クロムめっきおよびCVD,PVD,TRD プロセスによるチタン、クロムなどに代表される炭化物や窒化物などを被膜したものである。この種のCVD,PVD,TRD の表面処理工具は、他の表面処理剤や工具鋼に比べて耐摩耗性、耐焼付き性に優れている。
