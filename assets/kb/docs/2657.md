# 4・1・8 塑性加工の解析と解析法

> **章**: 第4章 塑性加工
> **節**: 4・1 塑性加工の基礎

a. 塑性加工の解析塑性加工の工程を設計するために、加工プロセスが解析され、各種の情報が計算される。加工荷重は加工機械の容量、素材の変形形状は製品形状、素材内部の応力・ひずみ・温度分布は割れなどの欠陥発生、工具の弾性変形は製品の寸法精度、工具面圧は金型の焼付き・損傷の予測にそれぞれ利用できる。塑性加工は大きな塑性変形を伴い、しかも塑性変形は非線形であるため、塑性加工の解析は容易ではない。塑性加工の解析では、力の釣合い式、ひずみの適合条件式、降伏条件式、応力-ひずみ速度関係式、体積一定条件、境界条件を満足する解を求めるが、それらをすべて厳密に満足させる解を得るのは、実際の加工においては不可能である。このため、いくつかの近似および仮定が用いられており、それらを基にして解析法が構築されている。塑性加工の解析法は、スラブ法、すべり線場、上界法などの解析的方法と、有限要素法、差分法などの数値解析法に大別される。b. スラブ法(slab method、初等解析法)スラブ法は、素材の変形領域内の薄い板状要素に対して力の釣合いの微分方程式を導き、降伏条件を用いて微分方程式を積分し応力分布を計算する方法であり、その応力分布から加工荷重も得られる。一般に、釣合い式は一次元、素材は剛完全塑性体を仮定するため、釣合いの微分方程式を解析的に積分でき、単純な式が得られる。図4・8は、平面ひずみ圧縮におけるスラブ法の力の釣合いを示したものである。一次元の変形を考え、方向を主応力方向と仮定すると、板状の要素に対する方向の釣合い式は次のように表される。ミーゼスの降伏条件を用いて式(4・19)を積分すると、工具面圧が得られる。式(4・19)は常微分方程式になるため、解析的に積分できる。工具面圧を接触面で積分すると加工荷重が得られる。計算では一次元の変形を仮定しているため、実際の加工で生じるたる型変形は無視されており、スラブ法では複雑な金型形状および自由表面の非定常変形などへの適用は困難である。スラブ法は、鍛造、押出し、引抜き、圧延、深絞りなどに適用されており、特に圧延加工では実ラインの制御に用いられている。スラブ法は塑性加工の解析法として最も初期に開発された解析法であり、解析解が得られて使用しやすいが、単純な変形しか取扱えない。c. 上界法(upper bound method)、エネルギー法(energymethod)塑性変形を受けた剛塑性体素材のエネルギー(仕事)を考える方法が、上界法およびエネルギー法であり、得られた変形エネルギーを工具に作用する力によるエネルギーと等しいとすることによって、加工荷重が計算できる。エネルギー法は単純な変形を仮定するが、上界法では動的(運動学的)可容速度場を考えて、それぞれ変形エネルギーを求める。動的可容速度場は変形している素材内での流れを表しており、動的可容速度場を用いる上界法は、材料の流れを考える方法である。図4・9は平面ひずみ押出しにおける速度場を示しており、速度不連続面においてせん断変形を受けて流れの方向が変化している。速度不連続面では、面に垂直な速度が面を通過する直前と直後で等しく、それ以外の領域では剛体変形しているため、体積一定の条件が素材内部全域で満足されている。速度不連続面におけるせん断変形からエネルギー消費率が計算でき、押出し荷重が求まる。動的可容速度場から得られる全エネルギー消費率は、正解のものよりも大きいことが保証されているため、上界法ではエネルギー消費率を小さくする速度場を求めることが一般的に行われており、上図4・8 平面ひずみ圧縮におけるスラブ法の力の釣合い図4・9平面ひずみ押出しにおける上界法の速度場界接近法とも呼ばれている。例えば、図4・9の速度不連続面の角度αを変化させると流れが変わり、得られるエネルギー消費率も変化し、エネルギー消費率を最小にするαを求めると、正解により近い解が得られることになる。上界法は体積一定条件および速度の境界条件だけを満足する解を求めており、近似解析法である。上界法に対して下界法(lowerbound method)があり、力の釣合い式、降伏条件式、外力の境界条件を満足する静的可容応力場から得られるエネルギー消費率は、正解よりも小さくなることが下界定理で保証されており、下界法は静的可容応力場を使って近似解を求める方法である。しかし静的可容応力場を求めるのは困難であり、下界法はほとんど使用されていない。d. すべり線場法(slip line field method)等方性剛完全塑性体の平面ひずみ変形において、接線方向が最大せん断応力の方向と一致する曲線群をすべり線と呼び、この曲線に沿って応力および速度を計算する方法がすべり線場法である。力の釣合い式、降伏条件式から、次のヘンキーの式が得られる。最大せん断応力は直交する2方向であり、αすべり線、βすべり線は最大主応力方向から時計回りおよび反時計回りにそれぞれ45°回転した線であり、φは軸からαすべり線へ反時計回りに測った角度である。ヘンキーの式から応力の境界条件を使って、すべり線に沿った応力が計算できる。また、応力-ひずみ速度関係式からすべり線が伸び縮みしないことが求まり、次のガイリンガーの式が得られる。すべり線場法で得られた解は、力の釣合い式、降伏条件式、応力-ひずみ速度関係式、ひずみの適合条件、境界条件を満足する正解である。しかしながら、一般にすべり線を描くことは容易でなく、剛完全塑性体の平面ひずみ変形と限定されているため、実加工にはほとんど応用されていない。e. 塑性加工の有限要素法有限要素法(FEM : finite ele-ment method)は、物体を多くの要素に分割して微分方程式を解く方法であり、工学の広い分野に応用されている。塑性加工分野においても、計算機のハードウェアだけでなく、ソルバ、プリポストプロセッサなどのソフトウェアの能力が向上して、有限要素法は塑性加工分野で最も優れた方法になってきた。塑性加工への応用における利点は次のとおりである。(1) 素材が変形する形状を精度よくモデル化できる。(2) 工具・材料の形状および材料特性を変化させるのが容易である。(3) 材料内部の材料流動、応力・ひずみ分布が求まる。(4) 自由度が大きく、精度の高い解が得られる。(5) 試行錯誤実験を減らすことができ、コストが低減できる。(6) 設計、開発の時間が短縮できる。有限要素法で塑性加工を計算する最も大きな特徴は、(1)の変形形状を精度よくモデル化できることである。塑性加工では素材形状が刻々と変化するが、スラブ法、上界法などの解析的方法ではそれを精度よく表現するのが難しい。有限要素法では、次のような材料特性を取扱うことができる。(1) 変形抵抗:加工硬化、ひずみ速度依存性、異方性、温度依存性(2) 摩擦:摩擦係数、摩擦せん断係数(3) 加工による組織変化:組織予測式(4) 粉末成形における圧縮性:粉末鍛造(5) 慣性力:高速鍛造塑性加工に必要な材料特性は、有限要素法でほぼ取扱うことができる。3―101f. 弾塑性有限要素法有限要素法は1960年代の後半に航空機の設計に対して開発された。この有限要素法は弾性変形を対象としていたが、1960年代の後半に弾塑性構成式を用いた拡張が行われ、その後1970年代の前半に大変形理論に基づいて定式化が行われ、弾塑性有限要素法が開発された。弾塑性有限要素法は素材を弾塑性体としてモデル化する方法であり、負荷時の塑性変形だけでなく除荷後の弾性変形も計算できる。しかしながら、この方法では変形段階ごとに応力の増分を求めて加え合わせているため、1回の変形量を非常に小さくする必要があり、計算時間が長くなる。また、厳しい変形における要素の再分割が問題になる。現在市販の塑性変形解析用はん用ソフトウェアは、弾塑性有限要素法を基礎としたものが大半である。弾塑性有限要素法では、アップデーテッドラグランジュ形式の定式化が多く用いられており、公称応力速度を用いた仮想仕事の原理は次のように表される。式(4・23)において、客観性のある応力速度を用いて構成式と関係づけて剛性方程式が導かれている。g. 剛塑性有限要素法剛塑性有限要素法(rigid-plasticfinite element method)は、素材の弾性変形を無視して素材を剛塑性体として取扱う方法である。剛塑性有限要素法も1970年代の前半に定式化がなされ、上界法をはん用化させるものとして、塑性加工の研究者によって開発された。塑性加工のような大きな塑性変形を伴う加工では、素材のわずかな弾性変形を無視してもよい場合が多い。剛塑性有限要素法では、変形段階ごとに応力が直接計算されるため、1回の変形量を比較的大きくすることができ、大きな変形後の要素の再分割も容易である。また、一般に微小変形理論に基づいて定式化されるため、プログラミングも簡単である。このように剛塑性有限要素法は実用的な方法であり、塑性加工の数値シミュレーションによく用いられている。剛塑性有限要素法では、体積一定条件の取扱い法として、三つの方法がある。ラグランジュ乗数法(Lagrange multipliermethod)では、ラグランジュ乗数法を用いて体積一定条件を取扱い、次の汎関数を停留させる解を求める。式(4・24)の第1項めは塑性変形、第2項めは体積一定の拘束条件、第3項めは摩擦損失、第4項めは外力に関するものである。汎関数の停留化条件は、汎関数を変数によって偏微分した式をそれぞれ零にすることであり、ラグランジュ乗数で偏微分すると体積一定条件になり、体積一定条件が満足される。汎関数の最小化において、体積一定条件を近似的に満足する方法としてペナルティ法(penalty method)があり、次の関数を最小にする。αが大きな値をもつため、汎関数が最小になるには、体積ひずみ速度の値は零に近づき、体積一定条件がほぼ満足される。この方法では、仮想仕事の原理から近似的な応力も得られる。圧縮性材料では、体積一定条件が満足する必要がないため、近似的にわずかの圧縮性を導入した方法が圧縮特性法(slightly com-pressible material method)であり、次の汎関数を最小にする。この方法はペナルティ法と似たものであり、変数が節点速度だけであり、ラグランジュ乗数法のように変数の数が増加しない。h. 動的陽解法動的陽解法は自動車の衝突問題のような衝撃解析に対して開発されたが、塑性加工にも適用されるようになってきた。この方法は弾塑性有限要素法の一種であるが、通常の有限要素法と違って連立方程式を解かないため、大規模な三次元問題のシミュレーションに適しており、板材成形を中心とした実加工への適用が盛んになりつつある。動的陽解法では、仮想仕事の原理に慣性力の項が含まれる。陽解法では、現在の時刻で応力が既知とし、式(4・27)の慣性力を表す左辺第1項めだけを考えて質量を節点に集中させ、加速度を差分近似して節点変位で表す。得られた連立方程式は対角成分だけに係数が現れ、単純な計算で解が求まる。動的陽解法では連立方程式を解くことなしに解が求まるため、要素数が非常に多くなっても計算時間が急激に増大することはなく、大規模な三次元問題の解析に適している。しかしながら、実加工の成形時間は衝撃問題のように極端に短くないため、実際の加工速度で計算すると時間が長くなる。そこで、実際の10~100倍大きな速度で計算が行われている。
