# 3・3・4 DfXの代表例としてのDfE

> **章**: 第3章 設計のための個別方法論
> **節**: 3・3 DfX

a.背景 地球温暖化のような地球環境問題から，廃棄物処理
のような地域環境問題を含む，いわゆる環境問題が深刻化してい
る．環境問題が人類にとって重要課題であることは，1972年の国
連人間環境会議における人間環境宣言で初めて共通認識された．ま
た，同じ1972年のローマクラブによる「成長の限界」レポートは，
環境汚染が人類の危機を招く可能性を示し話題となった．1987年
には環境と開発に関する世界委員会（WCED）が持続可能な発展
という概念を提示し，持続可能性が環境問題を考えるうえでのキー
ワードとなった．そして1992年には環境と開発に関する国連会議
（地球サミット）が開催され，地球環境保全のための行動計画「ア
ジェンダ21」が策定された．さらに，1997年には気候変動枠組条
約第3回締約国会議（COP3）で，先進国の温室効果ガスの削減目
標を定めた京都議定書が合意されている．
いまや，環境問題は企業にとって必ず考慮しなければならない課
題の一つである．企業の環境に対する責任の歴史は，1972年の経
済協力開発機構（OECD）環境指針原則勧告による汚染者負担の原
則（PPP:polluter pays principle）の導入で本格化した．そして，
2001年には，製品生産者の視点を製品ライフサイクルの全体に向
けるべきであるという方向性を明確に示したOECD拡大生産者責
任（ERP:extended producer responsibility）ガイダンスマニュ
アルが発行されている．OECDによると，EPRは「製品のライフ
サイクルにおける消費者より後の段階にまで生産者の物理的または
経済的な責任を拡大する環境政策の手法」と定義されている．
国際標準化機構（ISO）でも，製品のライフサイクルを通した環
境負荷・影響を評価するライフサイクルアセスメント（LCA:life
cycle assessment ）規格を標準化し（ISO14040-43），また，環境
負荷の小さな製品に張り付ける環境ラベルに関する規格
（ISO14020-25）も制定している．今後，グリーンコンシューマと
呼ばれる消費者が増えてくると，環境に配慮した製品に対する要求
がいっそう強くなっていくであろう．環境に配慮しないものづくり
をしていては，持続可能な企業活動はできないのである．DfE
( design for environment ）の標準規格についても，ISO/ TR
14062：2002として制定されている．
近年は生産パラダイムも変化してきた．環境問題の原因の一端が
大量生産，大量消費，大量廃棄という生産パラダイムにあったとい
う反省から，循環型社会の構築によって環境負荷・影響を低減しよ
うという考え方が広まってきた ．こうした考え方は，1994年に
国連大学から提案されたゼロエミッション や，1995年に東京大
学を中心として提案されたインバース・マニュファクチャリン
グ にも見出すことができる．国内の法規制整備という観点では，
資源循環の考え方を前面に出した循環型社会形成推進基本法（循環
型社会基本法）が2001年に公布された．本法においては，3R
( reduce，reuse ，recycle ）の実行によって資源循環を実現すると
いう方向性が示されている．また，2001年から施行されている特
定家庭用機器再商品化法（家電リサイクル法）も，企業が資源循環
を考慮したDfEに注目する契機となった．
b.目的と動向 DfEの目的は，製品を通して発生する環境負
荷を低減することにある．そして，DfEによって設計された環境
負荷の小さい製品を環境調和型製品（環境配慮製品，eco- prod-
uct:environmentally conscious product ）と呼ぶ．図3･51に
DfE関連手法の概観を示す．
当初，DfEは有害物質を含まない材料の選択や，製品回収後の
有害物質処理を容易にするための分解性設計（DfD:design for
disassembly）を中心に研究，開発されてきた．しかし，現在で
は，適用対象を材料採掘から製品廃棄に至る製品ライフサイクル全
体に拡大し，製品ライフサイクルを通して発生する環境負荷を低減
するための設計手法をDfE，エコデザイン，環境配慮設計
（ecodesign:environmentally conscious design），環境調和型設
計などと呼んでいる ．このように，製品ライフサイクルの全体
を視野に入れた思考様式は，ライフサイクルアプローチと呼ばれて
いる．ライフサイクルアプローチに基づくDfEの特色は，設計対
象が製品ではなく製品ライフサイクルであることにある．図3･51
には，DfM（3･3･1項参照）やDfA（3･3･3項参照）も含まれてい
るが，これらは，DfEとは別の目的をもって発展してきており，
DfEと区別してとらえられている．
DfEやエコデザインが環境調和型製品の設計を指すことが多い
のに対し，ライフサイクルデザイン（3･6節参照) はより広い概
念としてとらえられている．すなわち，ライフサイクルデザイン
は，循環型社会におけるものづくりのあり方を指し，製品を通した
環境負荷削減だけでなく，企業戦略やビジネスモデル，サービスの
設計，製品使用中のライフサイクル管理などの課題も対象範囲とし
ている．
以下ではおもなDfE関連手法，ツールについて簡単に紹介する．
