鍛造加工鍛造加工では,素材が金型によって圧縮されて所定の形状に成形される.鍛造加工では,素材の形状は加工中に大きく変化するが,スラブ法,上界法などの解析的な方法はこのような非定常変形(non-steady state deformation)を取扱うのは難しい.有限要素法では素材が多数の要素に分割されているため,非定常変形を精度よくシミュレーションできる.高い素材を鍛造加工すると,塑性座屈(plastic buckling)を生じて素材が曲がって欠陥になる.図4・10はわずかに傾いた円柱のヘッディング加工の計算結果であり,塑性座屈が発生している.塑性座屈の発生は,加工硬化および工具面摩擦,工具および素材の傾きによって大きく影響され,有限要素法はそれらの影響を考慮することができる.c. 押出し,引抜き加工押出し,引抜き加工は,金属の棒・線の断面を連続的に減少させて伸ばす加工法である.これらの加工法では,変形が時間に依存しない定常変形(steady state deforma-tion)が現れるため,剛塑性有限要素法では定常解析法が利用できる.定常解析法では,変形形状と変形抵抗分布を仮定して,それらを繰返し計算で収束させる.入口境界から速度分布をたどることによって流線を求め,その流線に沿って相当ひずみ速度を積分することによって相当ひずみ分布を計算し,変形抵抗曲線を用いて変形抵抗分布を求める.通常5~10回程度の繰返しで,変形形状,相当ひずみ分布が収束する.図4・11は定常解析法によって計算された線材の引抜き加工中の静水圧応力分布である.中心部で引張応力が発生しており,中心割れの原因になる.得られた応力とひずみ分布から,延性破壊(duc-tile 圧延加工圧延加工では,ロール面の摩擦によって素材がロールギャップ中に引き込まれて,素材の板厚が減少させられる.ロールとの接触面では,素材とロールの速度が等しい中立面が現れ,中立面を境にして摩擦力の方向が逆転し,圧延加工では中立面の取扱いが問題になる.剛塑性有限要素法では,摩擦力を相対すべり速度の関数に近似して中立面を取扱っている.六面体三次元要素を用いた完全三次元解析法が開発され,棒材,形材,管材の孔型圧延における定常および非定常三次元変形がシミュレーションされている.図4・13はH 形鋼の孔型圧延における三次元変形のシミュレーション例である.e. 板材成形板材はおもに引張変形で加工されるが,板厚方向に変形が集中する局部くびれ(local neck)を生じると,破断に至る.局部くびれの発生は,素材の加工硬化指数,異方性パラメータ値,工具形状,工具面摩擦などによって影響される.深絞り加工において素板径を大きくすると,図4・14に示すようにある素板径以上ではくびれを生じて破断に至る.絞り加工,管の曲げ加工などではしわ(wrinkle)が発生し,成形限界となる.しわは一種の座屈現象である.図4・15は,有限要素法でシミュレーションされたしわ押えなし再絞り加工におけるフランジしわである.しわ押えなし再絞り加工では,ダイスがテーパ状になっており,角度を適正に設定することによってしわの発生が抑制される.シミュレーションでは動的陽解法を用いており,計算から生じたわずかな不均一性を起点としてしわが発生する.f. 粉末成形金属およびセラミックスの粉末は圧粉成形して所定の形状に成形し,その後焼結することによって製品になる.圧粉成形および焼結では,素材が空げきを含んでいるため,見掛けの体積が変わり,通常の金属材料の塑性加工における体積一定条件は成立しない.空げきを含んだ素材を多孔質体として連続体でモデル化し,多孔質体の塑性力学に基づいた有限要素法が粉末成形の解析法として開発されている.図4・16は軸対称二段圧粉成形における相対密度分布である.多段成形によって,歯車などの複雑な部品が粉末材から成形されている.多段圧粉成形では,分割した複数個のパンチで成形され,各部の圧下率を等しくして均一な圧粉体を得るようにしているが,粉末の流動および工具の弾性変形によって少し密度が分布する.図4・17は各種鋼材の製造工程を示したものである.連続鋳造機または分塊圧延機からの鋼塊は各種の鋼材に加工されるが,ほとんどが圧延加工である.図4・18に示すようにロールの数とその配置により異なる.図4・18(a)は二段圧延機で粗成形を行う分塊圧延や形鋼,線材を製造する孔型圧延に用い,図4・18(b)は四段圧延機で薄板や厚板の圧延に広く用いる.図4・18(c)は六段圧延機図4・17
