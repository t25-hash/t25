# 3・4・1 信頼性の設計の考え方

> **章**: 第3章 設計のための個別方法論
> **節**: 3・4 信頼性の設計

信頼性は製品が満足すべき最も重要な特性の一つである．製品の
ライフサイクルを通じて，ユーザの要求する機能を満足できるよう
にするためには，種々の方策が必要となるが，それらは以下の3種
に大別することができる．
(１)故障が発生しないようにする．
(２)故障が発生しても，なるべく機能を保てるようにする．
(３)故障が発生しても，ただちに機能回復のための修復ができる
ようにする．
また，これらの方策を講じる時期としては，設計・製作段階と，
運転・保全段階がある．前者では信頼性を作り込む活動が行われ，
後者では信頼性を発現させるための活動が行われる．ここでは，前
者の信頼性の作り込みについて考える．この観点から，上記三つの
カテゴリに該当する方策例を示したのが表3･12である．
以下では，おもに(１)と(２)のカテゴリの方策に関しての基礎を
与える信頼性工学について概説する．3･4･2項においては，まず故
障をとらえる視点を整理し，3･4･3項で説明する確率論に立脚した
信頼性理論の意味を明確にする．また，3･4･4項では，信頼性デー
タに基づく信頼性特性値の推定について，3･4･5項では，特に上記
(２)に関連する，要素とシステムの信頼度の関係および冗長化につ
いて述べる．さらに，3･4･6項では，必ずしも確率論にとらわれな
い広義の信頼性工学手法として，特に設計段階で対象システムの弱
点を抽出するのに活用されているFMEA（failure mode and
eﬀect analysis ）とFTA（fault tree analysis ）について述べる．
信頼性工学の範囲は非常に広いが，ここでは，製品の信頼性を考
えるうえで必要な基本的視点と，信頼性に関する手法の中でも特に
広く利用されている，累積ハザード法，FMEA，FTAの三つの手
法の理解を主眼に解説する．このため，故障物理，信頼性試験など
の信頼性工学を構成する重要な分野をカバーしていない．これらに
ついては多くの優れた教科書が出ているので，それらを参照された
い ．
