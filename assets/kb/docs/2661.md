c.故障率に基づく故障の分類 確率的な見方においては，時
間軸上での故障の発生形態によって故障を特徴付ける．そこで，故
障率の時間的変化から，故障を次の三つの基本パターンに分類する
ことが行われている．なお，これらは，故障率の変化パターンを示
すもので，図3･61に示した，機能状態を表すパラメータ値の変化
とは意味が異なることに注意が必要である．
(１) DFR（decreasing failure rate）型
故障率が時間とともに減少するタイプである．設計，製造上の欠
陥などのために初期段階で故障が多く発生するが，時間とともにこ
れらの欠陥が取り除かれ，故障率が低下していく場合である．
(２) CFR（constant failure rate）型
故障率が時間によらず一定のタイプである．アイテムが安定した
稼働状態にあり，故障の発生が偶発的な場合である．
(３) IFR（increasing failure rate）型
故障率が時間とともに増加するタイプである．軸受の摩耗のよう
に，アイテムが時間とともに劣化し，それに伴って故障しやすくな
る場合である．
対象が単純な部品や簡単なシステムの場合は，故障の発生パター
ンは上記3パターンのいずれかに分類されることが多いが，複雑な
システムの場合は，そのライフサイクルの中で三つのパターンが順
次現れ，図3･65に示すような形態になるといわれている．これは，
DFR型の初期故障期間（early failure period），CFR型の偶発故
障期間（random failure period），IFR型の摩耗故障期間（wear -
out failure period）より構成されており，その形態からバスタブ
曲線（bathtub curve）と呼ばれる．バスタブ曲線は，人間の一生
を，幼年期，青年期，老年期に分けたときの，死亡率の変化パター
ンに対応していることから，一般に広く受け入れられている概念で
ある．システムがこれらのいずれの期間にあるかを知ることは，信
頼性向上のための対策を講じるうえで大切である．ただし，航空機
の装備品の故障率を調査した結果では，このようなバスタブ曲線を
描くものはごくわずかであったという報告もあるので，個々のシス
テムへの適用については確認が必要である．
