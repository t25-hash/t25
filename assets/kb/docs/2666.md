# 4・3・3 板、管の切断

> **章**: 第4章 塑性加工
> **節**: 4・3 素材の予備加工

塑性加工によって製造される長尺素材の板は、予備加工として最も生産性の高いせん断加工によって切断され、次工程に供されることが多い。加工対象材は金属から非金属に至るほとんどすべての工業材にわたる。管については棒材と同様、切削切断と比べれば、せん断による切断は精度は劣るが抜群の生産性ゆえ、精度のさほど要求されない予備加工としてはやはり多用される。そのほか、プレス機にセットした型によるせん断加工のみで最終製品を作ることも頻繁に行われる。a. せん断加工における変形機構直線刃により材料をせん断する場合を想定し、ダイは固定されていてパンチが移動すると仮定する。パンチが下降して材料表面に接触するにつれて、材料のパンチ、ダイとの接触表面は次第に凹凸がならされる。さらにパンチが(b) クランプ方式の拘束せん断(a) 軸圧縮せん断図4・50 拘束せん断法図4・51せん断力とパンチストローク線図の例(c) フライングシヤー(b) スリッティング(a) シヤーリング図4・52 板材のせん断加工(b) 走行式二重突切り法の原理(a) 二重突切り法の原理図3―115下降すると、パンチとダイの刃先付近の材料が降伏し、パンチとダイは材料内に次第に深く食い込んでいき、材料はせん断変形を受ける。ただし、加工により変形を受ける部分は、クリアランス部分とそのごく近傍に限られている。パンチ、ダイがさらに材料に食い込むと材料は加工硬化するので、図4・51に示すようにパンチ、ダイに作用するせん断力は増大する。材料は次第にその変形能が尽き、刃先からわずか側面部分に入ったところでき裂が生じ始める。それ以降き裂は、応力集中効果のためより低いによって成長を続け、最後にパンチ、ダイからそれぞれ発生したき裂が会合し、材料の分離が起こり、せん断が完了する。b. 予備加工としての板材のせん断加工これらには、大きく分けて図4・52に示すように、せん断荷重を低下させるために傾斜角を有する刃を用いたせん断方法であるシヤーリング(shearing)加工、コイル材のような幅広長尺材を回転する刃物を用いて複数個の幅狭材にせん断し、また個々にコイル材として巻取るスリッティング(slitting)加工、同じくコイル材を長手方向に回転刃物によりせん断的に分断して定尺板材を作っていくフライングシヤー加工がある。c. 予備加工としての管のせん断加工薄い切断くずの出る突切り法も広義のせん断加工の範ちゅうに入れて考える。せん断加工が用いられるのはほとんどが薄肉(例えば3mm以下)材を対象としている。管の加工の形態としては、長尺材を短尺材へと切断分離する場合と、いったん切断された短尺材を端面加工する場合に大別される。用いる装置には切断型を装着したプレス機と走行切断(フライング切断)のための専用機とがある。図4・53(a)に示す二重突切り法は長尺管の切断に用いられる方法で、第1工程で水平方向に動く切り刃で管上部に切れ目を切削的に入れ、第2工程で前記切れ目内にV 字状の突切り刃を押し込むことにより、一工程のみの突切り法では管のへこみ(変形)が大きいという欠点を軽減している。このようなプレス切断の方法は、図4・53(b)に示すように、専用機を用いることで管が走行している状況でも実施できる。すなわち、ボールスライド上を管の走行速度と同期して走行するキャリッジ上に装着された第1回転刃がまず管の表面を切欠き、続いて第2回転刃が管の完全切断を行う。かくして、管の最大造管速度が140m/minの場合のカット数は150カット/minという高速切断が実現されている。せん断加工のみによる管の切断方法の図4・54 連結心金を用いたパイプせん断法図4・53 管の二重突切り法3―116加工学・加工機器例を図4・54に示す。これは心金型によるプレス切断の方法であり、前述の突切り型によるものに比べて走行切断方式に対応させることがより困難なため、薄肉管の切断という観点からは一般的ではないが、比較的高精度の切断が可能である。特徴は固定心金と可動心金が連結されている点である。軟鋼の管(電縫管)のように比較的内径精度がよく、曲がりの少ない管に対しては、管は可動刃側から供給され、切断された製品管は、リメインダ側管を装入時と反対側に突き出すことにより取出される、いわばシャトル作業が行われる。切断刃の動きは上下、左右または偏心状である。
