# 4・4・3 管の成形

> **章**: 第4章 塑性加工
> **節**: 4・63 に示した基本的な加工法が単独で用いられることは少なくな

管の成形(チューブフォーミング、tube forming)は、管を素材として行う二次成形の総称であり、成形、接合、分離に分けられる(図4・64)。曲げおよびねじり加工のほか、管端部および管胴部の各種拡管・縮管加工、異形加工、変肉厚加工、据込み加工、つぶし加工等を含んでいる。それらの成形には、プレスを用いる成形、液圧による成形、回転工具による成形など、多岐にわたる加工様式が適用されている。チューブフォーミングには、他の加工法には見られない特徴として、管自体の剛性が高く軽量効果が大きい、型が省略できる、一体成形による大幅な部品点数の減少、工数削減、材料歩留り向上、などが挙げられる。その反面、管は中空で断面が閉じていることから内部を拘束したり外力を作用させることが難しく、加工上の面で多少取扱いにくく、独自の加工法や成形技術が必要となる点が短所となっている。a. 曲げ加工(bending)管に曲げ変形を与えると、横断面は中空のため扁平化して偏肉が生じ、さらに曲げると屈服を起こすなど、管特有の変形が現れる。また、曲げ内側には圧縮応力によってしわが、外側には引張応力によって破断が生じる。このほか、形材では断面のゆがみ変形や倒れ込みの不整変形が生じる。曲げ加工限界はこのような形状精度も含めて破断、しわおよび屈服で規定される。実曲げ加工では、加工限界や形状・寸法精度を向上させるために、ダイの形状を工夫したり、管内部に種々の心金(砂、氷、低融点合金を含む)を入れて管内面から変位を拘束したり、応力状態を変えるため、張力や圧縮力を種々の形で作用させるなどの工夫がなされている。それらの加工条件と型設計、さらに使用する材料特性によって、曲げ加工品の最小曲げ半径とスプリングバックも含めた加工精度が支配される(9・7・7項「曲げ加工機」を参照)。b項に述べる液圧を利用して、内圧負荷連続せん断変形で曲げ半径が零に近い曲がり管を製造したり、複雑な曲げ成形品を得る方法がある。3―120加工学・加工機器チューブフォーミングねじり加工曲口広げ加工口絞り加工カーリング・反転加工つぶし加工据込み加工異変肉厚加工拡管成形(バルジ加工含む)縮管成形(スエージング含む)げ形加加工工管端部成形管胴部成形横断面が閉じたアルミニウム押出し形材の曲げには、三次元曲げができるCNC押通し曲げ加工法が多く利用されている。b. バルジ加工(bulge forming)とハイドロフォーミング(hydro-forming)バルジ加工は管の胴部を膨らませ、所望の形状に成形する加工法である。金型を用いてその形状を転写する型バルジと、金型を用いない自由バルジがある。管内側から加える圧力媒体には油や水のほか、ゴム(ゴムバルジという)、低融合金、粉体、気体などが用いられている。管に内圧だけでなく軸方向に圧縮力を負荷できることが大きな特徴であり、それによって管の応力状態を変化させたり、材料の供給ができるので、肉厚減少の制御ができたり、大きな張出し変形が実現できる。その内圧と軸圧縮力(あるいは軸押込み量)を組合わせる負荷経路が加工の成否を決定する重要な加工因子となっている。加工管材料としては一般に、破断伸び、値および値が大きいほど加工性がよく、肉厚減少が抑えられる。近年、軽量成形加工技術として注目されているハイドロフォーミングは、液圧によるバルジ加工を中心にして、管のもつ特性を巧みに利用し、より複雑な形状の成形を可能にするため、曲げ、つぶし、穴あけ加工などを複合した加工法の総称である。現在では図これは前工程で曲げ加工を行い、それをプリフォーム、そのあとハイドロフォーム、穴あけ、さらに端末切断が行われたものである。ハイドロフォーミングの利点は、(1)複雑断面形状をもつ中空部品を一体成形、(2)構造性能の向上とそれによる重量軽減、(3)機械的結合に必要なフランジ部の撤去による材料歩留りの向上、部品点数の削減、(4)溶接箇所の減少による信頼性向上と工程数削減、(5)高圧負荷による寸法精度向上とスプリングバックの低減、である。これらを通してコスト削減が達成できる点も大きな魅力になっている。一方、欠点としては、(1)サイクルタイムが長く生産性が低い、(2)加工の実績やノウハウの蓄積がなく試行錯誤に頼っている、(3)加工設備が大型で高価である、(4)素材費が高い、(5)加工工程が複雑で高度な加工技術を要する、などが挙げられる。負荷する加工内圧のレベルによって、高圧法(>150MPa)と低圧法(<150MPa)に分類され、加工目的によって使い分けられる。高圧法では高圧力のため大変形して加工硬化による強度向上と製品精度向上が図れる反面、大型加工装置となって大きなスペースが必要で、さらに内圧の上昇・下降に時間がかかり、製品肉厚分布も一様でないという短所をもっている。一方、低圧法では、複雑断面形状の成形が難しく拡管量も大きくとれないという欠点を有する成形接合分離塑性除去加工変形法接着法塑性溶接法変形法切輪郭・穴あけ加工圧その他の接合法バーリングかダブルシーミング法ビード加工エキスパンド法断し接加め工法法図4・64チューブフォーミングの分類図4・65 エンジンクレードルが、加工装置がコンパクトになり、製品も比較的一様な肉厚が得られ、素管からの肉厚減少も小さくサイクルタイムが短い。ハイドロフォーミングは本質的に多工程になるため、その加工限界や製品精度は前加工での加工の種類とひずみの程度、さらにその履歴により大きく異なる。また、管と型との摩擦の影響を大きく受けるため、管の軸押しによる材料の流動制御が大きな加工パラメータとなる。内圧負荷を利用した管の穴あけを行うハイドロピアシングは、ハイドロフォーミングでの合理的で重要な加工法の一つとして位置付けられている。c. 管端部加工(口絞り、口広げ)管の端末部を細く絞る加工が口絞り(tube nosing)で、逆に広げる加工が口広げ(tubeflaring)である。プレスを用いた軸圧縮による口絞りでは成形部は軸方向にも円周方向にも圧縮応力が作用し、種々の座屈不良で加工限界が支配される。同じく口広げでは円周方向には押し広げられ、引張応力と軸方向の圧縮応力場となる。加工限界は口辺部の破断(割れ)と円筒部の座屈で決定される。成形形状はいろいろ可能であり、これらの加工製品のおもな用途は、管あるいは部材との結合・締結、継手や配管部品の結合要素、および管の支持部材の位置決め用や流路のノズル、さらにカーリングの場合には製品取扱い上の安全と、端部の補強用に使用されている。加工手段としては、プレスによる方法以外に、回転加工、電磁成形、逐次成形、スエージングが用いられる。加工限界や製品精度の向上には適切な材料選択と型設計のほかに、工具と管との摩擦力制御と工具による変形拘束、さらに多工程化が効果的である。なお、チューブフォーミング全般の参考書として、文献(96)を参照していただきたい。
