# 4・5・2 自由鍛造

> **章**: 第4章 塑性加工
> **節**: 4・5 塊状品の塑性加工

自由鍛造(open die forging)は金敷と呼ばれる比較的単純な形状の工具を用いて、材料の再結晶温度以上の高温に加熱された素材を液圧プレスや空気圧ハンマで成形加工する方法である。一般に鍛造製品は鋼片や鋼塊を購入して鍛造する鍛工品と、製鋼・造塊から鍛造まで一貫して製造される鍛鋼品に分類されている。自由鍛造はおもに後者の鍛鋼品を製造する目的で実施される鍛造法で、発電機用軸材、原子炉圧力容器、石油化学プラント用反応容器、圧延機用補強ロールなどの、重量が1~200tにも及ぶ大型の製品が多い。自由鍛造の基本的な作業形態は、図4・70に示すように鍛伸鍛造、展伸鍛造、据込み鍛造、中空鍛造(心金鍛造)、穴広げ鍛造が主たる形態であり、背切り、曲げ、ねじり、せん孔(穴あけ)などの作業も必要に応じて加えられる。実際の工程では、これらの基本作業が組合わされて、プレスによる圧下工程と再加熱が順次繰返される。いずれの場合でも比較的単純な形状の工具を用いるため、成形の自由度が大きく、それに比例して所要荷重も低くなるが、成形精度に限界があり多品種少量生産向きの加工法である。このような基本作業以外に、特殊な形状や超大型の製品も金型や付帯装置を工夫して鍛造されている。例えば、直径が7mを超すような大きなシェル材は、図示するようなプレス機内での穴広げ鍛造が不可能であることから、プレスの圧下方向を鉛直から水平方向に変換する特殊なアタッチメントにより、機外での穴広げ鍛造が実施される。このような方法を用いると最大直径が10mもの超大型シェル材の鍛造が可能である。鍛造作業で使用される金敷と呼ばれる工具には、圧下用として平金敷、薬研(やげん)金敷、タップ金敷などがあり、そのほかに心金、据込み臼、台金、背切たがねなどが多用される。これらの金敷類は製品の重量や形状により選択されるが、プレスの最大圧下力や与える鍛造効果を勘案して適切な条件に設定することが肝要である。鍛造前の素材の加熱温度の選択も重要である。一般に自由鍛造で3―123(a) 鍛伸鍛造(b) 展伸鍛造(c) 据込み鍛造(d) 中空鍛造(e) 穴広げ鍛造(g) 曲げ(f) 背切り(h) ねじり(i) せん孔(j) 搾出(k) 口絞り図4・70 自由鍛造の基本的作業形態は繰返し加熱回数が多く、適正な加熱条件の設定が工期やコストに大きな影響を及ぼす。鉄鋼材料では1000~1250°Cが選ばれることが多い。設定温度が低すぎると、鍛造時間が短く再加熱が多くなるばかりか、冷却による低温割れなどを誘発することもある。逆に高すぎると結晶粒の粗大化による内部性状の劣化や、さらに過加熱状態では局部的な溶融であるバーニングを起こし、廃却となることもある。自由鍛造における目的は、まず製品の最終形状に近い形を成形することにあるが、素材の内部性状を改善するための鍛錬効果を付与することも極めて重要な役割である。すなわち、据込みや鍛伸の繰返しにより累積的に大きな加工ひずみを付与することで、鋼塊の粗大な鋳造組織を破壊するとともに、特に大型鋼塊の凝固過程で、軸心部に不可避的に発生する微小空げきを閉鎖し圧着することが必要である。このような自由鍛造における鍛錬効果を定量的に表現するため、基本作業形態に応じて鍛造比がJISにより制定されている。この鍛造比は最大主ひずみに対応する変形比を表す値であり、この値が大きいほど素材内部には大きなひずみが付与され、鋳造組織破壊により鍛流線を伴う緻密な鍛造組織が得られる。しかし、この値は外形的な形状変化から計算される平均的な変形比であり、素材内部の真のひずみ分布は異なっている。実際には、金敷による摩擦拘束の影響からひずみの小さいデッドメタル域の発生が避けられないため、極めて不均一なひずみ分布となり、組織改善効果の予測には十分に考慮しなければならない。また、素材の内部空げきの圧着を意図する場合には、単にひずみの付与だけでは十分ではなく、圧縮静水圧応力が重要な役割をなすことにも留意する必要がある。このことから、特に大型鍛鋼品の鍛造工程では、軸心部まで十分なひずみと圧縮静水圧応力が作用するように、幅の大きな金敷で圧下しなければならないが、所要荷重の急激な増大を避けるために、鋼塊の表層部を故意に冷却して鍛造する方法や、上下非対称な金敷を用いる方法など効率的な空げき圧着鍛造法が実用化されている。3―124加工学・加工機器
