# 3・6・6 まとめ

> **章**: 第3章 設計のための個別方法論
> **節**: 3・6 ライフサイクル設計

環境問題解決へ向けての今後の人工物設計，生産のあり方を議論
し，その総合的システム化技術としてライフサイクルエンジニアリ
ングの重要性を述べた．ライフサイクルエンジニアリングにおける
重要な課題の一つは，循環を前提とした製品ライフサイクルの設計
であり，その基本概念，基本的な流れを整理し，特に製品ライフサ
イクル設計の初期段階で重要なライフサイクルシナリオの策定方法
をいくつか概説した．この分野は成熟しているとはいいがたく，十
分に実用化可能なガイドラインは存在しないが，今後の人工物設計
を考えるうえでの一助になれば幸いである．
