# 4・6・1 板成形と鍛造の複合

> **章**: 第4章 塑性加工
> **節**: 4・6 複合塑性加工

板成形は本来、元の板厚を積極的に変更する成形法ではないが、最近は、板成形時に積極的に板厚を増肉したり、あるいは多段の板成形工程内に、据込み鍛造工程を組込むなどの複合成形法(com-plex forming)が実用化されている。これらの複合成形法を分類すると、図4・87に示すように、(a)シャープなエッジ形状の製品、(b)差厚寸法および段差形状の製品、(c)局部成形の製品等に分類される。これらの複合成形では、せん断(打抜き)、深絞り、曲げ、バーリング等の板成形の後、据込み、しごき加工、押出し等の鍛造によって所望の差厚製品を得ている。カップ状製品の積極的な増肉成形法(forming with increasingthickness)の一例として、図4・88に示すように、厚板を浅く絞った円筒容器の側壁を高さ方向に加圧しながら外周に歯形成形を行う工程がある。すなわち、円筒絞りの後、しごき加工により外周に凹凸形状を成形し、この厚肉容器を外周に歯形を有する型内へ入れ、高さ方向に加圧する。この加圧により、カップ底角隅部の形3―129(a) エッジ形状(b) 差厚寸法および段差形状(c) 局部形状図4・87 板成形と鍛造の複合成形図4・88 厚板の絞り・しごき・圧縮成形従来方式本方式増肉加工前増肉加工途中増肉加工完了図4・89厚板の増肉工程の比較状が整えられ、増肉しながら外周に歯形が成形される。この例は、厚板の浅い容器を用いる特別の場合で、一般的には、図4・89左側の図のように座屈(buckling)が生じるため増肉は困難であるが、図4・89右側の図のようにテーパ状の側壁を上下の型で挟んで、座3―130加工学・加工機器屈を防ぎながら高さ方向に加圧する新たな増肉成形法が開発されている。深絞りの成形限界向上をねらって、素材外周に半径方向あるいは軸方向に圧縮応力を負荷する方法が提案されている。図4・90に示すように、復動作動によるプレスを用いて第1軸で軸方向に圧縮荷重を負荷し、素材を押し込みながら圧縮絞りを行う。その後、第5軸を下降させながらミドルパンチで二段めの深絞りを行う。さらに、図4・91に示すように、その後の工程でフランジ部のコイニングとカップ底の孔抜きを行う。これにより、フランジ部の増肉工程と、大幅な工程短縮を実現している。これらの一連の加工工程は、板成形、鍛造、せん断を複合した成形で、鍛造の金型設計を必要とする。
