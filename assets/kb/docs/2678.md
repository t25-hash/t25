# 4・6・2 鋳造と鍛造の複合

> **章**: 第4章 塑性加工
> **節**: 4・6 複合塑性加工

鋳造では十分な製品強度が得られない場合に、鍛造等の塑性加工による高強度化を図ることが試みられる。しかし、複雑形状製品では鍛造の工数がかかりすぎ、生産コストが高くなる場合がある。そのような場合に、製品に近い鋳造素形材(casted work metal)を鍛造することにより、鍛造品より安いコストで鋳造品より高強度の製品を得ることができる。このような鋳造・鍛造複合プロセスの一例として、アルミニウム合金の自動車用ホイールの製造工程を、従来の鍛造だけによるプロセスと比較して図4・92に示す。
