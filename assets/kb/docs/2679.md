# 4・6・3 切削と鍛造の複合

> **章**: 第4章 塑性加工
> **節**: 4・6 複合塑性加工

歯車の歯形や等速ジョイントの転動面のように、複雑形状の面を切削で仕上げると時間とコストがかかりすぎる。これらの製品を大量生産する場合に、精密鍛造によりこれらの複雑形状面を完全に仕上げることが可能になれば、生産速度とコストの面で極めて有利になる。それ以外の単純な仕上面は切削により行うほうが、完全なネ2工程3工程図4・91 ピストンの圧縮絞り工程サンプルットシェイプ成形を行うより有利な場合が多い。その際、最重要部分を冷間鍛造で仕上げ、その面を基準として、その後の切削を行うことが必要になる。冷間鍛造に要求される素材材質と切削加工に適する材質では相反する場合が多い。そこで両者の生産性を考慮して、適切な組織に調質することが必要になる。例えば、浸炭焼入れ材はフェライト/パーライト組織で冷間鍛造を行うが、高周波焼入れ材は球状化焼なまし材を用い、切削加工の条件、および焼入れの内容に合わせて球状化の度合いを制限するなどの工夫が必要になる。
