# 4・6・4 接合と塑性加工の複合

> **章**: 第4章 塑性加工
> **節**: 4・6 複合塑性加工

塑性変形を利用する変形・流動接合法(pressure welding withplastic forming)として、リベット結合、かしめ、圧入、シーミング等の部分的な塑性変形に基づく構造締結法が多く用いられるが、材料全体に塑性変形を生じさせながら接合する方法も広く用いられている。板成形におけるヘミング、シーミング、かしめ等は、構造締結の典型的な例であり、プレス成形工程の中に組込んで、金型を用いて行われるため生産性がよく、また異種材の締結も可能である。また、図4・93に示すように、部材A の内側にA よりも軟らかい部材B を加圧して結合する方法も、接合と塑性加工の複合化として採用されている。冷間圧接(cold welding)と拡散接合(diffusion welding)は、固相接合として、一貫した接合メカニズムによって説明が可能である。図4・94は各種金属の圧接可能な下限表面積拡大比を示したものである。いずれの金属でも接合温度と融点の比/が高まり、図4・93塑性流動結合法1工程図4・90 復動作動による圧縮絞り図4・94円柱端面の圧接試験における各種金属材料の圧接の下限表面積拡大比に及ぼす温度の影響図4・92 従来の鍛造プロセスと鋳造・鍛造複合プロセスの比較0.5以上になると下限表面積拡大比は1に近づくことがわかる。すなわち、その金属の再結晶温度に近づくに従い、拡散接合が支配的になる。異種材料の冷間圧接では、変形抵抗の差異により接合面での相対滑りが生じ、同種材料の圧接よりかえって接合性がよくなる場合がある。一方、再結晶温度以上での接合では、図4・95に示すように界面にもろい合金層が形成されて接合強度が極端に低下する場合があるので注意を要する。直接通電加熱装置(DC pulse resistance heating apparatus)を用いて各種金属組合せどうしの接合を行った場合、超硬合金以外の同種金属組合せでは、800~1000°Cで継手効率約100%が達成されている。また、鉄系合金どうしの異種金属の組合せでは、800~1000°Cで継手効率70%以上が達成されている。鍛造のように素材に大きな塑性変形を与えることができる場合には、図4・96に示すように、異種材料を重ね合わせただけの素材を冷間で後方せん孔押出し、あるいは前方押出しすることにより、接図4・95異種金属板の圧延接合における接合後の加熱保持時間に対する接合強さの変化(b) 前方押出し(a) 後方押出し図4・96 異種材複合ブランクの鍛造・接合法表4・15種々の金属組合せの重ね合わせ圧延によるクラッド板材の製品例接合材の組合せ用途アルミニウム/銅台所用品、屋根、壁板、熱交換器、電圧用ケーブルの結合器、分配器、電熱器などの反射板、自動車用消音器(アルミニウムは耐食材)アルミニウム/ステンレス自動車部品(ステンレスは強度材、良い体裁、Alは耐電食)アルミニウム/亜鉛高速輪転機のプリントプート銅/ステンレス整流子、電線の外装、台所用品銅/鉄銃弾のジャケット銀/銅/銀、ニッケル/銅/ニッケル、ニッケル/鉄/ニッケル貨幣(熱間が多い)ニッケル/銅石油化学機器(Ni/耐食材)銅/銀、銅/金、銅/白金精密電気機器のばね接点、コンピュータ、自動車用定電圧接点アルミニウム/アルミニウム(部分接合)定電圧器の接点、熱交換器銅/銅(部分接合)太陽熱パネルの集熱器、熱交換器3―131合と成形を同時に行うことが可能になっている。種々の金属組合わせの重ね合せ圧延によるクラッド板材(cladsheet)は、表4・15に示すように、種々の用途に用いられている。
