# 4・7・2 粉末の射出成形

> **章**: 第4章 塑性加工
> **節**: 4・7 粉末およびプラスチック加工

プラスチック製品の大半が射出成形(injection molding)法により製造されていることは広く知られている。これは熱可塑性の物質を流動可能な温度まで加熱し、型内へ押出し冷却することによって複雑形状部品を生産するものである。粉末射出成形(powderinjection molding)プロセスでは、図4・99に示すように金属やセラミックスの粉末(60~70vol.%)と有機系バインダ(30~40vol.%)を加熱混練することで、プラスチックに近い流動性をもたせて成形を行うもので、その後脱バインダおよび焼結を施すという(a) 湿式法(b) 乾式法のが基本工程である。特に金属粉末を対象とした射出成形(MIM :metal injection molding)においては、従来の粉末冶金(P/M :powder metallurgy)での平均粒径100 mのものに比べ、約10m前後の1けた細かい粉末を用いることから、焼結の促進に伴う高密度材(真密度に近い)が得られ、機械的性質のみならず各種機能性も著しく向上するのが特徴である。また、旧来のP/M(おもに金型プレス成形)では成形が難しい硬質金属材料、あるいはこれまでの成形技術では困難であった低熱膨張合金や軟質磁性材料などの難加工性機能材料にも適用できるため、用途に応じた材料の選択自由度が大きいことも特徴である。したがって、対象材質としてもFeをはじめ、Fe-Ni,Si,Co合金、ステンレス鋼、高速度鋼、Ti合金、Ni基やCo基の超合金、W 系重合金、超硬合金、サーメット、繊維強化型合金など広範囲な種類のものが挙げられている。MIM の用途は特に限定されておらず、最近では0.003g の超小型部品から17kg という大物品もあるが、一般には100g 以下の複雑形状の小物品がおもな対象で、時計などの精密機器部品、医薬機器部品、光通信・電子・磁気部品、OA 関連ならびに自動車や一般産業機械の部品など、多岐にわたる分野で使用されている。a. 原料粉末(powder)まず原料粉末の特性に関して、粉末の大きさは2~8 mで、球形よりやや異形(アスペクト比で1.2~1.5)のものが好ましく、かつ粒度分布は狭いか広いものがMIMには適しているようである。安価で品質が安定した微細粉末供給のための粉末製造技術(例えば超高圧水アトマイズ法など)の革新が望まれている。b. バインダ(binder)成形用材料であるバインダは、脱バインダや焼結工程で消失することから、あくまでも補助的な役割を果たすにすぎないが、MIM プロセスにとっては必須の成分であり、その組成によって成形や脱バインダがうまく行えるか否かが決まる。すなわち、バインダには成形性、離型性、保形性、脱バインダ性などの各種特性が要求されるが、すべての要求を満たすようなバインダは今のところ開発されておらず、後述する各種脱バインダ法に合った多種多様なバインダ系が存在する。簡便なバインダ系としては、ワックス系のものが最も多く使用されている。それは基本的には、流動性をもたせるため(低粘性)のワックスと、粉末に対する固着力を増強させるため(保形性)の高分子材料(ポリエチレンやポリプロピレンなど)、そして粉末とのぬれ性や離型性向上の図4・98 等方圧成形(CIP)図4・99金属粉末射出成形の工程概略図ための潤滑剤(ステアリン酸など)からなる。c. 射出成形成形に関しては、基本的にはプラスチックの射出成形技術と同じであるが、被成形物が高粘度でかつ摩耗作用をもつことから、やはりMIM 専用の成形機が望ましく、機能性に富んだ機種も登場しつつある。成形体の品質を左右する成形条件(射出温度、速度、圧力や金型温度など)については、被成形物によっても異なることから、多くは経験に基づいた試行錯誤的な設定で行われている。CAE(computer aided engineering)導入による金型内充てん過程での流動解析によれば、成形品のデザイン、金型構造、成形条件などの最適化、あるいは成形欠陥の発生防止や精度および品質の向上にとっても有効な情報が得られるようで、CAEの活用が今後の健全な成形品の量産にとっては必須となるであろう。d. 脱バインダ(debinding)バインダ系と関連する脱バインダ方法には、表4・16に示すようにこれまで多種多様のプロセスが開発されているが、工業的には熱分解や溶媒抽出がおもに採用されている。前者では脱バインダに長時間を要したり、変形が生じやすいなどの問題が残されている。また後者では、それらの欠点がかなり克服されるものの、溶媒にはアセトンやエチレン、四塩化炭素などのような人体に害を及ぼすものや環境汚染につながるものが多いため、その取扱いが問題とされており、現在、エタノールや水溶性の新しいバインダも開発されつつある。いずれにしても、脱バインダに長時間を要することは生産的には不利であり、このことがMIM 製品の許容肉厚を大きく制限していたが、最近では脱バインダ技術も進歩して25mm程度の肉厚までは可能となっている。e. 焼結(sintering)焼結は従来のP/M 法と同様で、不活性あるいは還元性のガス雰囲気、または真空雰囲気で行われる。ただ、脱バインダ後の成形品はかなり脆弱でハンドリング性に劣ることから、従来のバッチ式から脱バインダと焼結を連続的に行う方法も考えられている。なお焼結部品の寸法精度は、焼結による寸法収縮(15~20%)が著しいにもかかわらず、現在のところ±0.3~±0.1%まで向上している。これは精密鋳造品やダイカスト製品の精度に比べると優れているが、全般的にはP/M 製品の精度(±0.1%)レベルまでには達していない。二、三の例として、±0.05%という数字も挙げられているが、このレベルまで精度を向上させることができれば、精密切削加工品と拮抗し、MIM の飛躍的な進展が期待される。
