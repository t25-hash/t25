# 5・1・1 切削加工理論

> **章**: 第5章 切削加工
> **節**: 5・1 切削加工の基礎

a. 実際の切削加工最も単純な切削加工は,図5・1(a)に示すように切削方向と工具切れ刃が垂直な二次元切削(orthog-onal cutting)である.二次元切削では切削幅が切込み量に対して十分に大きく,同じ変形状態が工具の切れ刃方向に続く平面ひずみ状態を仮定する.本様式は基礎的な切削機構(cuttingmechanics)を理解するうえで大変に有用であるが,実際の切削加工の大部分は,図5・1(b)(c)に示すように切れ刃が切削方向に対して傾斜していたり,切れ刃が一つの直線ではない三次元切削(three dimensional cutting)である.被削材の削りやすさは被削性(machinability)と呼ばれ,次の4項目によって評価される.すなわち,仕上面性状と加工精度,切削抵抗と切削動力,工具寿命,切りくず処理性である.これらの項目は互いに複雑に影響しており,例えば切削抵抗が増大すると工具が損耗しやすくなり,仕上面粗さが増大する原因となる.切りくず(chip)は,除去される不要部分であるが,その形態は切削状態をよく表し,またその処理性は実用的にも重要である.図型,周期的に部分的破断を起こすせん断型,完全な破断によって小片に分離するき裂型の三つに大別される.流れ型切りくずは,延性材料を切削する場合に生じることが多い.この場合には切削状態が安定しているため,良好な仕上面が得られる.せん断型切りくずは比較的もろい材料を,き裂型切りくずは非常にもろい材料を切削する場合に生成しやすい.これらの場合には,切削力が変動したりき裂が残留するため,仕上面性状が劣化するうえに工具損耗を生じやすい.他方,切りくずの処理性に関しては流れ型より良好となる.鋼やアルミニウム合金などの延性材料を比較的低速(例えば50(a) 二次元切削(a) 流れ型(b) 傾斜切削(b) せん断型(c) 普通の切削(c) き裂型図5・1切削様式図5・2 切りくず形態の分類3―137m/min)で加工する場合,刃先部分に被削材の一部が凝着して擬似的な刃先(構成刃先,built-up edge)を構成することがある.これが発生すると,実質的に刃先が鋭くなって切削抵抗が減少したり,工具刃先が保護されて摩耗が減少する利点がある.その反面,構成刃先は不安定であって成長と脱落を繰返すため,仕上面性状が劣化したり,工具欠損を引き起こしやすい欠点がある.旋削加工における切削抵抗の合力は,図5・3に示すように三つの分力に分解される.切削運動を与える主軸の動力は,切削方向の主分力(principal force)욣と切削速度の積となり,送り動力は送り分力(feed force)욞と送り速度の積によって決まる.一方,切込み方向の背分力(thrust force)욧は,工具や工作物の変形によって加工精度劣化の原因となる.低剛性の工作物や工具,あるいは工作機械を使用したり,大きな切削幅で加工をする場合に,激しい振動を伴うことがある.これをびびり振動(chatter vibration)と呼ぶ.これには,加工プロセス内に振動を増幅する不安定性をもつ自励びびり(self-excitedchatter)と,断続切削における切削力などが機械の固有モードを加振して生じる強制びびり(forced chatter)がある.いずれのびびり振動が生じても,仕上面粗さが著しく増大し,工具欠損を生じやすい.最悪の場合には加工機械の故障を引き起こすこともあるため,避けなければならない現象である.近年,工作機械の高速化に伴って機械および切削プロセスに起因する減衰性が低下し,びびり振動が問題になることが多くなったため,この機構を理解することの重要性が増している.b.
