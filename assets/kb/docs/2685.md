# 5・1・2 切削工具

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように、切りくず形態は、切りくずの厚さが一様な流れ

a. 切削工具材料への要求特性切削は工作物を破壊させる加工法であることから、切削工具には強さが求められるとともに、加工時の熱に対する強さも必要とされ、切削工具が具備すべき条件として高温硬度(hardness)と靱性(toughness)が重要な因子となる。高温硬度は耐摩耗性ということもでき、靱性は粘り強さであり、過大な切削力となる断続的な切削の場合には特に重要である。しかし、図5・13に示すように一般的には硬度と靱性は相反する性質で、硬度が高ければもろいという性質があり、両特性を兼ね合わせることが工具材料に要求される大きな課題である。また、最近の高付加価値製品に求められる超精密加工は、多くの場合非常に小さな切込みで実現され、微小切込みが可能な刃先の鋭利さ(sharp-ness)、つまり工具先端の刃先丸みが小さいことが求められる。そのほか、自動化の進展に対する工具寿命の安定性、多様な用途への工具形状に対する成形性などが求められる。b. 切削工具材料現在使用されている切削工具材料としては、図5・13に示すように、高速度工具鋼(ハイス、HSS)、超硬合金(sintered cemented carbide)、コーテッド工具(coatedtool)、サーメット(cermet)、セラミックス(ceramics)、cBN(cubic boron nitride、立方晶窒化ホウ素)、ダイヤモンド(dia-mond)がある。それぞれには製法、組成、組織などにより多くの種類があるが、硬度と靱性の特性としては定性的には図5・13のような領域で示すことが可能である。高温硬度が高いほど高速切削が可能であり、靱性が高いものは断続的な切削に適用される。超硬合金、コーテッド工具、セラミックス、cBN は微粉末素材の超高圧高温焼結により製造される。高速度工具鋼は炭素鋼にW,Cr,V,Mo,Coを添加し、熱処理により硬度を改善させた材料であり、また、ダイヤモンドは単結晶が超精密加工に用いられているが、いずれも焼結製造される材種もある。超硬合金は主成分としてWCの粉末をCoで焼結したもので、主要材料である鋼との溶着性(耐摩耗性)からTiC,TaCが加えられた材種もある。WCの代わりに溶着性の小さいTiC,TiN を主成分とした材種がサーメットであるが、欧米ではhard metalとして超硬合金に含まれた呼び方をする。セラミックスはAlO,AlOにTiCやZrOを分散させた黒色セラミックス、さらに、SiN,SiCなど多くのものが開発され、焼結時に添加される助剤、粒度などにより特性は異なる。ダイヤモンドが鉄と反応するのに対し、反応を示さずダイヤモンドに次ぐ硬さのものとして開発されたのがcBN である。以上の工具材料が硬さの増加とともに靱性が減少するのに対し、靱性を維持しつつ切りくずと接触する表面硬度を改善しようと開発されたのがコーテッド工具であり、TiCN,TiN,TiAlN、さらにはDLC(ダイヤモンドライクカーボン)など多くのものが実用化され、現在の主要な工具材料になっている。さらに、耐摩耗性を向上させるべく、CVD(chemical vapor de-position、化学蒸着)、PVD(physical vapor deposition、物理蒸着)などのコーティング法も含めた開発検討がなされている。c. 切削工具と構造切削工具には、加工製品形状に対応して非常に多くの種類がある。一つの分類として、単刃工具(singlepoint tool)、多刃工具(multi point tool)に分けられる。旋削、中ぐり、形削り、平削りなどではバイトと呼ばれる単刃工具が用いられ、外径、内径、溝入れ、突切り、ねじ切りなどに使用される。多刃工具としてはフライス、エンドミル、ドリル、ブローチなど、用途に応じ種々の名称・形状・形態がとられる。個々の名称、形状については5・2~5・5節で述べられる。単刃工具、多刃工具のいずれにおいても、直接的切削加工に関与する部分は局部的で、刃部、切れ刃と呼ばれる。図5・14に示すように、刃部を含め工具全体を工具材料で一体成形したものと、そのほかに焼結成形される工具では単純な形状の刃部を鋼製のシャンクまたはボデーにろう付け、あるいは機械的に固定した構造のものがある。工具材料で一体成形したものはむくバイトと呼ばれ、エンドミル、ドリル、フライス加工における総形工具、さらに小型の工具などで用いられる。一方、後者は、工具摩耗などによる工具交換の必要性から機械的に固定したクランプ方式が多く用いられ、刃部には切りくず処理性の改善を目的としてチップブレーカ(chipbreaker)が取付けられる。この場合、刃部はスローアウェイチップと呼ばれ、摩耗後は廃棄され資源の浪費となるが、現在のところ経済性の観点から、再生ではなく廃棄が選択されている。さらに、工具シャンクを工作機械に直接取付ける場合と、図5・15に示すように、ツールホルダを介して取付けられる場合、シャ(a)
