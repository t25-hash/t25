# 5・1・3 加工誤差と仕上面の特性

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように、切りくず形態は、切りくずの厚さが一様な流れ

工業製品の高品質化に伴って、寸法、形状ならびに表面粗さに対する要求がいちだんと厳しくなってきている。特に近年の超精密加工や電子デバイスの高集積化がこれに拍車をかけている。加工仕上面の特性については寸法はもとより、うねり(waviness)などの形状誤差(form error)、表面粗さ(surface roughness)、さらにはすじ目、きず、光沢、色、コロージョン、表面の微細構造などの表面性状が含まれる。加工精度は、周辺環境まで含めた工作機械-工具-工作物系に存在する数多くの因子の影響を受ける。近年工作機械の力学的構造の強化、熱的構造の改良が進められ、静・動剛性はかなり高められてきている。しかし、現在のような加工形態が続く限り、加工誤差(machining error)は小さくできてもなくすことはできない。図5・16に加工誤差の一般的な要因を示す。実際はこれらの要因が相互に複雑に関連し合い、工作物の精度に影響を及ぼす。したがって精度向上のためには、これらの要因の影響をできるだけ小さく抑えるか除去すればよいわけである。例えば図5・16の主軸回転誤差は、おもに工作物の表面粗さやうねりなどの表面性状に関係する。つまり基本的にはいかなる作業環境、加工状態のもとでも、つねに工作物と工具の相対位置を一定に保つことが必要である。これを旋削加工の場合について簡単に見てみよう。図5・17に示すモデルについて旋削加工では、切削点における断面での工具刃先A と(b) 付刃バイト加工プロセス幾可学誤差力学的変形熱的変形主軸回転誤差静的変形工作機械力学的変形熱的変形工具損傷静的変形工具+加工誤差寸法誤差形状誤差など表面粗さ力学的変形熱的変形静的変形工作物周囲の温度変化外的な力、地盤の変形周囲環境図5・16 おもな加工誤差要因図5・17 加工モデル3―142加工学・加工機器機械の回転中心Oおよび実際の工作物中心O′の3点が求まれば、工作物の寸法と形状は一義的に決定される。すなわち図5・17に示すように、工具刃先と回転中心Oとの距離により、その断面における工作物の径と真円度が定まり、工作物中心と回転中心および工具刃先の位置関係によって、その断面での偏心量と方向が定まる。各断面でのこれらの形状を連ねていったものが仕上面となる。したがって、これら3点の位置関係が目標値よりずれた場合に、工作物に寸法や形状誤差、ひいては表面性状の劣化をもたらすわけであるから、加工誤差の原因とは、これらの位置関係を乱す原因のことを指す。寸法精度(dimensional accuracy)は、加工図面で例えば100φのように寸法の上限などを示して規定する精度である。円筒加工部品の直径などを1箇所で表示するので、テーパやうねりなどの加工誤差をもつ場合にはこの尺度では不十分で、次の形状精度が求められる。表5・1は形状偏差についてJIS 規格に説明されているものである。
