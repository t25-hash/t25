# 5・1・4 切削油剤とその効果

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように、切りくず形態は、切りくずの厚さが一様な流れ

切削加工では工具刃先に油剤を供給しながら加工を行う場合が多い。その切削油剤(cutting fluids)には表5・2に示す働きがある。潤滑作用とは切削工具すくい面と切りくずの間、および切削工具逃げ面と工作物仕上面の間の摩擦を小さくする作用である。工具すくい面と切りくずの接触圧力は極めて大きく(材料の降伏応力の3倍程度)、容易には油剤が入り込めないが、切りくずがカールしてすくい面から離れるところの近くでは油剤の潤滑性が作用して摩擦力を低減する。これが切りくず生成部のせん断角を増加させ、切削抵抗の減少を導き工具寿命の延長が図れる。また工具逃げ面と工作物仕上面との弾性的接触部の潤滑も、逃げ面摩耗の減少と仕上面品位の向上に貢献する。冷却作用とは、加工熱による工具および工作物の温度上昇を抑制する働きである。温度依存型の工具摩耗では、温度上昇を少なくすることで工具寿命の延長が図れるが、セラミック工具による高速切削などでは急激な冷却が熱衝撃となり工具刃先にき裂が発生し、逆に工具寿命を短くすることがあるので注意を要する。工作物の温度上昇は熱膨張による製品精度の低下につながるので、仕上加工では切削油剤の冷却作用は重要となる。抗溶着作用とは、切りくずが加工面や工具表面に溶着するのを防ぐ作用である。切りくず表面および加工面は新生面であり、酸化膜や汚れ膜のない活性の高い表面であるため、両者は容易に溶着する。穴加工では切りくずが加工穴内面に強く押し付けられ溶着し、工具欠損の原因ともなる。切削油剤は速やかに新生面を油剤分子で覆い、溶着を妨げる。また特に低速切削ではせん断面に発生したき裂の再溶着を防ぎ、切削抵抗を低減することが知られている。切りくず排出作用とは、切削部で発生した切りくずを洗い流す作用である。穴加工では穴の内部から外へ切りくずを運び出す役割を果たし、特に深穴では重要となる。また、フライス加工でも切りくずが工作物やパレット上に堆積することは作業性、安全性からも有害であるので、加工液を使って切りくず搬送コンベヤまで洗い流している。切削油剤としては鉱油、動・植物油などを主成分とする不水溶性油剤と、エマルジョン(乳化油)などの水溶性油剤がある(表5・3)。前者は潤滑効果に優れているが、無人生産では火災の危険性がある。後者は冷却性に優れているが、近年では腐敗等による廃液処理が大きな負担となっている。表5・2 切削油剤のおもな働きと効果作用効果潤滑作用切削抵抗減少工具寿命延長仕上面品位向上冷却作用工具寿命延長加工精度向上抗溶着作用仕上面粗さ向上切削抵抗減少切りくず排出作用作業性向上表5・3 切削油剤の種類不水溶性油剤鉱油、動・植物油、混成油、合成油水溶性油剤エマルジョン、ソリューション、ソリューブル快削添加剤Pb,S,Ca,Bi,Te等冷却剤冷風、液体窒素、ドライアイス重切削の高温度においても潤滑性を保つために、Cl,S,P系の極圧剤を添加することがあるが、Cl系は環境負担の観点から使われなくなった。材料内部に介在させる快削添加剤は、切削時に工具表面に出現して潤滑効果をもたらす。鉛快削鋼のPbはその代表的な添加物であるが、これも環境負担軽減(レッドフリー化)のために使用が制限されてきた。加工点冷却の手段としては冷風、液体窒素、ドライアイス等も試みられている。
