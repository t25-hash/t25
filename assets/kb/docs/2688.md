# 5・1・5 切りくず処理

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように、切りくず形態は、切りくずの厚さが一様な流れ

切りくず処理は大まかに見ると次の三つのステージがある．(１)
切削点における切りくずの制御，(２)機械から工場外への切りくず
搬出，および(３)作業環境改善や地球環境保全，である．これらの
解決が(１)は工具メーカに，(２)は工作機械メーカあるいは切りく
ず処理機器メーカにゆだねられる部分が多い．これに対して(３)
は，ユーザである各製造現場自身の課題である．
切りくず処理では，工具や製品にダメージを与えることなく，切
りくずを機外に排除することがまず第一である．そのためには適当
な長さに分断した切りくずを作ることが必要であり，旋削では多様
なチップブレーカから選択する技術が重要である．チップブレーカ
の開発・改良が進み，工具選択をきちんとすれば，鋼の場合，通常
の切削条件では十分対応できる．表5･4にチップブレーカの分類
を，図5･18に適用領域の考え方を示す．最近は素材の鍛造技術が
向上し，切削での取りしろが少なくなり，仕上げだけの一発加工も
少なくない．切りくずの発生量が少ないという点では好ましいが，
切りくず制御という点では最も難しい領域であり，仕上切削用ブレ
ーカの選択が重要になる．アルミニウム合金の切削ではブレーカ付
きダイヤモンド工具の開発，耐熱合金の切削では低切削抵抗型二段
ブレーカなどのチップの開発が進んできた．穴加工，特に深穴加工
では油穴付き工具は欠かせない．ドリル以外にも油穴付き工具の開
発が進み，機械，ツールホルダ，供給ポンプの高圧化などと相まっ
て，高速・高能率化が図られている．
発生した切りくずの後片付け自体を無人化，システム化しようと
するものが第2ステージである．工作機械のテーブル上に堆積した
切りくずを排除する方法をまとめたものを表5･5に示す．どれがよ
いかは，切削形態，切りくずの材質や大きさ，切削油剤の有無など
によって異なる．水溶性切削油剤の大量供給は工作物の熱変形防止
にもなり効果的である．しかし，潤滑油への混入防止対策，機械か
らの飛まつ防止対策，さらには廃液処理なども必要になってくる．
