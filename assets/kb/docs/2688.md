# 5・1・5 切りくず処理

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように,切りくず形態は,切りくずの厚さが一様な流れ

切りくず処理は大まかに見ると次の三つのステージがある.(1)切削点における切りくずの制御,(2)機械から工場外への切りくず搬出,および(3)作業環境改善や地球環境保全,である.これらの解決が(1)は工具メーカに,(2)は工作機械メーカあるいは切りくず処理機器メーカにゆだねられる部分が多い.これに対して(3)は,ユーザである各製造現場自身の課題である.切りくず処理では,工具や製品にダメージを与えることなく,切りくずを機外に排除することがまず第一である.そのためには適当な長さに分断した切りくずを作ることが必要であり,旋削では多様なチップブレーカから選択する技術が重要である.チップブレーカの開発・改良が進み,工具選択をきちんとすれば,鋼の場合,通常の切削条件では十分対応できる.表5・4にチップブレーカの分類を,図5・18に適用領域の考え方を示す.最近は素材の鍛造技術が向上し,切削での取りしろが少なくなり,仕上げだけの一発加工も少なくない.切りくずの発生量が少ないという点では好ましいが,切りくず制御という点では最も難しい領域であり,仕上切削用ブレーカの選択が重要になる.アルミニウム合金の切削ではブレーカ付きダイヤモンド工具の開発,耐熱合金の切削では低切削抵抗型二段ブレーカなどのチップの開発が進んできた.穴加工,特に深穴加工では油穴付き工具は欠かせない.ドリル以外にも油穴付き工具の開発が進み,機械,ツールホルダ,供給ポンプの高圧化などと相まって,高速・高能率化が図られている.発生した切りくずの後片付け自体を無人化,システム化しようとするものが第2ステージである.工作機械のテーブル上に堆積した切りくずを排除する方法をまとめたものを表5・5に示す.どれがよいかは,切削形態,切りくずの材質や大きさ,切削油剤の有無などによって異なる.水溶性切削油剤の大量供給は工作物の熱変形防止にもなり効果的である.しかし,潤滑油への混入防止対策,機械からの飛まつ防止対策,さらには廃液処理なども必要になってくる.切りくずを工場外へ搬送する手段としては,機械的な方法,マグネットの利用,空気(吸引)や切削油剤の利用などがあるが,問題はその選定である.切りくずの材質と形,切りくずの時間当たりの発生量,切削油剤の可否(ドライ,水溶性,不水溶性),搬送距離と表5・4 チップブレーカの分類形式用途クランプオン型溝がつけられないチップ溝研ぎ込み型精密(G 級)仕上げ用全周溝型中・重切削用三次元型はん用(M 級)図5・18 チップブレーカの適用領域3―143搬送経路,メンテナンス(機構が単純か,消耗品の価格)などを考慮する必要がある.もう一つは環境対応である.高能率化の武器であった切削油剤の減量化,さらにはドライ化が求められている.また切りくず処理性の良好な鉛快削鋼の利用が難しくなってきている.これらの解決が目下の課題である.
