# 5・1・7 高速・高能率加工

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように、切りくず形態は、切りくずの厚さが一様な流れ

工作機械による加工では、切削速度、主軸速度、送り速度の三つの速度が存在し、そのどれかが高い場合に高速加工と呼ばれるが、ここでは最も基本的な高切削速度による加工に限定する。高速切削(high speed cutting)が常用切削速度を超える速度での加工であり、常用切削速度とは最もよく使用される超硬工具あるいはコーテッド工具で比較的安定した寿命が得られる速度であるとすれば、切削速度が高いか低いかは基本的には被削材種に依存する。例えば普通炭素鋼S 45Cであれば、100m/minの切削速度は高速とはいえないが、代表的Ti合金であるTi-6Al-4V では、100m/minは高速切削に相当する。工具寿命(工具摩耗)は切削温度と密接に関連し、切削温度と工具摩耗との間には図5・21に示すように、切削温度(切削速度)にあまり左右されず切削距離(摩擦距離)に依存する機械的なすり減り摩耗(abrasive wear)の領域と、切削温度に依存する熱的摩耗(thermal wear)の領域とに分けられる。超硬工具では、この境界温度は図5・21に示すようにおおよそ1100~1200K 付近であり、これを超えると熱的摩耗が顕著(図5・21の縦軸は対数目盛であることに注意)になることから、高速切削は切削温度で800~900°C以上に相当する切削速度による加工とみなすことができる。図5・22に旋削加工時の切削速度と切削温度の関係(通常両者は両対数グラフ上で直線となる)を示す。これから明らかなように切図5・23 三次元表示されたキャビティ(左)とコア(右)の例[日本ユニシス(株)提供]図5・22 各種被削材の切削温度(工具-被削材熱電対法による測定)切込み:1.0mm、送り:0.2mm/rev削条件が同じであっても切削温度は被削材により大きく異なり、したがって、高速切削の範囲も被削材により異なることが理解できる。ただし、注意すべき点は切削温度が高くなると工具の摩耗速度は確実に増加することから、高速切削を行うためには工具摩耗を許容できる値以下に抑える工夫が必要となる。このためには、熱的摩耗の起こりにくい工具、例えばセラミック工具などの使用や、切削速度は高くても切削温度を抑えた加工、例えば切込みや送りを低く設定した加工などが高速切削を行ううえで検討すべき点となる。高能率加工(high cutting efficiency)は、通常の加工能率(単位時間当たりの除去量)よりも大きな加工能率をもつ加工であると定義できる。時間当たりの除去量は、切削速度×切込み×送りで決まるため、このいずれを増やしても除去量は増加するが、切削速度の増加は切削温度の上昇を招き、工具寿命を低下させる。一方、切込みや送りの増加は切削速度ほどではないにせよ切削温度を上昇させ、また切削抵抗の増加を招き、加工精度や仕上面粗さに悪影響を及ぼす。安定した加工の原則は、ある程度の工具寿命と加工精度・仕上面粗さを確保することと考えると、高能率加工では対象とする被削材に対し、加工能率、加工精度、仕上面粗さのバランス、言い換えれば、切削速度×切込み×送りのバランスを見極める必要がある。
