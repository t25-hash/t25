# 5・1・8 金型加工

> **章**: 第5章 切削加工
> **節**: 5・2 に示すように、切りくず形態は、切りくずの厚さが一様な流れ

量産部品を生産するための金型(die and mold)は八つの種類に区分けされる。すなわち、射出成形用のプラスチック金型、打抜きや曲げ用のプレス金型、ダイカスト金型、鋳造金型、ゴム金型、ガラス用金型、鍛造金型、粉末冶金金型であり、それは使用する材料によって名前をつけられている。金型は型構造といわゆる金型部に分かれ、金型部に製品形状が設けられる。金型には簡単な形状のものもあるが、多くは三次元形状の複雑な形状であり、特に製品形状を創成するキャビティ(cav-ity)とコア(core)の加工が重要となる。図5・23にはキャビティとコアの三次元CAD 表示を示すが、キャビティ側が製品の表になるため、加工品位が求められる。金型の材料は使用目的ごとに異なり、鋼だけでなく鋳鉄やアルミニウム、合金鋼、高速度鋼、さらには超硬合金まで多様である。その加工は、従来、放電加工機を主体としていたため電極加工が中心となっていたが、最近では深いキャビティやフィンを除いて切削加工が主力になっている。加工工程は荒加工、中仕上加工、仕上加工、磨き加工からなるが、磨きは人手に頼ることが多く、時間のかかる磨き工程を省略できる高精度の仕上加工法が注目を集めている。ことに、複雑な三次元曲面の仕上加工にはボールエンドミルが多用される。仕上面精度を高めるには径の大きな工具を用いて小さいピックフィードで加工することが望ましいが、仕上工具径は金型の曲面の最小曲率半径に3―145図5・24 マシニングセンタによるコア型の加工[(株)牧野フライス製作所提供]よって規定されるために、工具径は拘束を受ける。小径工具を使用して小さいピックフィードで加工すると時間がかかり、加工能率の低下を引き起こす。最近の金型の仕上加工は、高速マシニングセンタを用いた高速・高送り加工法が一般的になっている。工具の種類や径を変えながら加工してきたものを、型の加工面の最小曲率半径と同じか少し小さい径のボールエンドミルだけで、工具を高速回転させながら小切込みで高速で送りながら加工するもので、これにより加工時間の短縮だけでなく、工具本数の削減と工具寿命の延長、さらには精度向上を図っている。図5・24に、マシニングセンタでコア型を加工している様子を示す。このような高速高送り加工が可能になったのは、設計法の進歩による工作機械の高剛性化と高速対応できるようになったこと、適切な工具経路を生成する三次元CAD/CAM の発展があったこと、さらに工具の性能が上がったことが挙げられる。なかでもコーティングによる超硬工具の性能向上が大きく貢献している。
