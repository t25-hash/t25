c .デザイン開発プロセス ここでは，デザインプロセスを以
(１)ユーザニーズ収集ステップ：3P（ポイント）タスク分析や
直接観察法などによって，ユーザニーズを抽出する．
(２)状況把握ステップ：コレスポンデンス分析により，市場にお
ける製品がどのようなイメージで見られているかを把握す
る．
(３)製品コンセプト構築ステップ：ユーザ要求事項に基づいて構
造化された製品のコンセプトを構築する．コンセプト項目は
ウエイト付けされる．
(４)デザイン（総合化）ステップ：構造化コンセプトに基づい
て，デザイン案を可視化する．
(５)デザイン評価ステップ：可視化されたデザイン案に対して，
AHP等で評価を行う．
(６)ユーザ使用実態調査：商品を購入し使用したユーザに対し行
う調査で，得たデータは製品企画時等に役立てる．
