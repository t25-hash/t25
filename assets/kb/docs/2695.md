# 5・2・2 形削り・立て削り・平削り

> **章**: 第5章 切削加工
> **節**: 5・2 単刃加工

a. 形削り工具に直線往復運動と1ストロークごとに工作物に送りを与えて削る方法を形削り(shaping)という。おもに平面や直線溝などの加工に用いられる。形削りを行う機械を形削り盤(図5・32)と呼ぶ。b. 立て削り形削りにおける工具の運動を垂直にして、おもにキー溝などの溝加工を行う方法を立て削り(slotting)という。この作業を行う機械が立て削り盤(図5・33)である。c. 平削り形削りと異なり、テーブルの水平往復運動と工具図5・33 立て削り盤図5・34 平削り盤をテーブルの運動方向と直角方向に送り、主として大きな平面を加工する方法を平削り(planing)という。平削りを行う機械を平削り盤(図5・34)と呼ぶ。ドリル(drill)を用いて穴のない工作物に穴をあけることを穴あけ(drilling)、またはむく穴あけ(solid drilling)といい、1枚刃の切削工具を用いて、鋳抜き穴やすでに穴があいている工作物の穴の直径を拡大すること、または穴内面を仕上げることを中ぐり(boring)という。また1枚刃から数枚刃のリーマ(reamer)を用いて、すでにあけられている穴の内面を切削とバニシング(burni-shing)によって仕上げることをリーマ加工(reaming)という。中ぐりは比較的大径の穴の仕上げに、リーマ加工は比較的小径の穴の仕上げに用いる。これらの加工は、ボール盤、中ぐり盤、フライス盤、マシニングセンタまたは旋盤などで行うことができる。
