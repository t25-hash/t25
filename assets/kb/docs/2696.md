# 5・3・1 ドリル加工

> **章**: 第5章 切削加工
> **節**: 5・3 穴あけ・中ぐり加工

ドリルとしては、被削材、直径および深さなどによってさまざまの材質、構造、形状のものが用いられている。a. ドリルの材質材質は、一般に高速度鋼(モリブデン系、溶解型と粉末型)または超硬合金である。高速度鋼ドリルは、表面に酸化処理あるいは窒化処理したもの、または硬質材(窒化物、炭化物、酸化物)をコーティングしたものを用いる。超硬合金ドリルは、研削状態のものと、その表面に硬質材(TiAlN,TiN,TiCN,AlO、ダイヤモンド)をコーティングしたものを用いる。近年、超硬合金工具の抗折力およびき裂伝ぱ抵抗の向上と信頼性向上により、超硬合金ドリルの使用量が増加している。TiAlN コーテッドドリルは高靱性超硬合金にPVD コーティングしたものが多く、焼入れ鋼(HRC60程度)への穴あけも可能である。そのほか、ダイヤモンド焼結体を超硬シャンクにろう付けしたドリルを、アルミニウム合金への穴あけ(φ5~12mm)や単結晶シリコンへの小径穴あけ(φ0.4~0.6mm)に適用している例もある。また、超硬合金またはその表面にダイヤモンドをコーティングした極小径ドリル(φ0.1~0.3mm)は、プリント基板(ガラスエポキシ樹脂)、グラファイト、マシナブルセラミックスおよびセラミック仮焼結体(グリーン)などへの穴あけに適用している。b. ドリルの構造構造としては、ドリルのボデーとシャンクが一体であるむくドリル、切れ刃をボデーにろう付けした付刃ドリル、およびスローアウェイチップをボデーに機械的に取付けたスローアウェイドリル(図5・35、約φ15mm以上)などがある。ドリルのボデーとシャンクに通り穴をあけ、その穴を通して切削油剤を切れ刃に供給する油穴付きドリルもある。これは特に、難削材(チタン、チタン合金、ニッケル基合金など)への穴あけにおいて、刃先の熱の除去、切りくずの排出促進および切削抵抗の低下に有効である。また切削油剤の高圧化によりアスペクト比(穴深さと穴直径の比)の高い穴(5程度)の加工も可能である。c. ドリルの形状最も一般的に用いられているドリルはツイストドリル(twist drill、または、ねじれ刃ドリル)である(図5・36)。ボデーにねじれを与えることによって、すくい角を増加させ切削抵抗を減少させるとともに、切りくずを排出しやすくする。通常のツイストドリルの断面形状(図5・36下)に比べ、ウェブ厚を大きくし曲げ剛性を高めた平溝ドリル[図5・37(a)]もある。このドリルは、高強度の被削材の深穴あけに使い、切りくずが長ピッチ型になるのが特徴である。加工硬化性の強い被削材に対しては、一般に送りを大きくし、せん断ひずみを小さくして切削抵抗を減らすために、強ねじれ(大すくい角)のドリルを用いるが、高硬度鋼図5・35 形シンニングなどである。また、チゼルエッジ部の切削性能改善や切りくず排出性向上の目的で、ドリル中心部に近い切れ刃のすくい角を大きくしかつ円弧状にしたドリルや、中心部の心を残す方式のドリルなどもある。e. ドリル加工の切削条件表5・7に超硬ドリルの標準切削条件を示す。f. 切削油剤ドリル加工では、被削材内部の切れ刃部の潤滑(c) 半月ドリル(b) フラットドリル(a) 平溝ドリル型図5・37 マイクロボーリングツールとボーリングバーと冷却が難しく穴壁からの締付けも大きいので、これまで極圧添加剤(EPA)入りの不水溶性切削油剤を使ってきた。しかし、法律により塩素系のEPA が禁止されたため、現在は硫黄系またはカルシウム系のEPA 入りの不水溶性・水溶性切削油剤を適用している。また環境保護の観点から、内部給油(油穴付きドリル)または外部給油でミスト加工、MQL加工、ドライ穴加工などを試み、適用可能となっている例もある。
