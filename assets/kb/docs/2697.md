# 5・3・2 中ぐり加工

> **章**: 第5章 切削加工
> **節**: 5・3 穴あけ・中ぐり加工

a.中ぐりの方法 中ぐりの方法にはスタブボーリングとライ
長くなるために剛性の低下が避けられず，びびり振動やたわみが発
生し，表面粗さの劣化や寸法精度の低下が起こりやすい．ラインボ
ーリングは，両持ちのボーリングバーに中ぐりバイトを取付けて行
う方法で，ディーゼルエンジンのシリンダライナのような深穴の中
ぐりに適する．
b.ボーリングツールとボーリングバー 中ぐりは，ボーリン
グバー（鋼製または超硬製）に中ぐりバイトを取付けた工具，また
は穴ぐりバイトで行う．この加工では，バイトの刃先の位置を微妙
に調整して穴径を精密に仕上げることが必要であるため，しばしば
図5･41に示すような刃先位置調整のできるアジャスタブルツール
をボーリングバーに取付けて行うことがある．一方，ボーリングバ
ーは剛性を高めるとともに，加工中の振動の減衰性向上のため，質
量ダンパや粘性ダンパを組込むなど種々の工夫をしているものがあ
る．
