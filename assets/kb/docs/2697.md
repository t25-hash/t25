# 5・3・2 中ぐり加工

> **章**: 第5章 切削加工
> **節**: 5・3 穴あけ・中ぐり加工

a. 中ぐりの方法中ぐりの方法にはスタブボーリングとラインボーリング(図5・40)がある。スタブボーリングは中ぐり工具を片持ちの状態で行う加工法で、穴が深くなるとボーリングバーが長くなるために剛性の低下が避けられず、びびり振動やたわみが発生し、表面粗さの劣化や寸法精度の低下が起こりやすい。ラインボーリングは、両持ちのボーリングバーに中ぐりバイトを取付けて行う方法で、ディーゼルエンジンのシリンダライナのような深穴の中ぐりに適する。b. ボーリングツールとボーリングバー中ぐりは、ボーリングバー(鋼製または超硬製)に中ぐりバイトを取付けた工具、または穴ぐりバイトで行う。この加工では、バイトの刃先の位置を微妙に調整して穴径を精密に仕上げることが必要であるため、しばしば図5・41に示すような刃先位置調整のできるアジャスタブルツールをボーリングバーに取付けて行うことがある。一方、ボーリングバーは剛性を高めるとともに、加工中の振動の減衰性向上のため、質量ダンパや粘性ダンパを組込むなど種々の工夫をしているものがある。
