# 5・3・3 ガンドリル加工

> **章**: 第5章 切削加工
> **節**: 5・3 穴あけ・中ぐり加工

深穴とは，通常，直径の約5倍以上の深さの穴をいう．深穴加工
は，能率を無視すれば5･3･1･ c項のツイストドリルをステップ送り
することによっても可能であるが，一般に深穴専用のドリルを用い
る．それはおもに，ガンドリル［gun drill，図5･42(ａ)］とBTA
方式ドリル［boring&trepanning association system drill，図5･
42(ｂ)］である．いずれのドリルも先端部に切れ刃とガイドパッド
をもっており，軸直角面内の切削抵抗をガイドパッドで受け穴壁を
バニシするとともに，高圧の切削油剤を切れ刃部に供給することに
よって切れ刃部の潤滑と冷却，ならびに切りくずの排出を行うもの
である．ガンドリルの場合は，シャンク内部を通して油を刃先まで
送り，シャンク外側の扇形の部分を通して切りくずと油を排出す
る．最近，マシニングセンタにおいてブシュを使わずにまず通常の
ドリルにより浅穴をあけ，その穴をガイドブシュとしてガンドリル
加工を行う方法が提案され，深穴加工の高能率化に貢献している．
BTA方式ドリルでは，ドリル外周部と穴壁のすきまを通して高圧
油を切れ刃部まで送り込み，ボーリングバー内部を通して切りくず
と油を排出する（図5･43）．ガンドリルは直径30mm以下の深穴
加工に用いることが多く，BTA方式ドリルは直径20mm以上の
深穴加工に用いることが多い．BTA方式ドリルの系列には，大径
用として，エゼクタドリル（ejector drill）や穴の心を残すトレパ
ニングドリル（trepanning drill）などがある．
5・3・4リーマ加工
リーマ加工は，ドリル加工などで得られた穴の精度向上のために
行う穴の仕上加工である．通常は，4～8枚の刃をもつ多刃リーマ
［図5･44(ａ)，マシンリーマ，ハンドリーマなど］を多用するが，
1枚刃とパッドをもつガンリーマやマパールリーマ［図5･44(ｂ)］
なども用いる．刃の材質は高速度鋼か超硬であり，後者は高速切削
用である．多刃リーマの場合，工具の振れまわり振動抑制の目的で
刃の円周方向ピッチを不等にすることがある．
