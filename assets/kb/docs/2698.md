# 5・3・3 ガンドリル加工

> **章**: 第5章 切削加工
> **節**: 5・3 穴あけ・中ぐり加工

深穴とは,通常,直径の約5倍以上の深さの穴をいう.深穴加工は,能率を無視すれば5・3・1・c項のツイストドリルをステップ送りすることによっても可能であるが,一般に深穴専用のドリルを用いる.それはおもに,ガンドリル[gun drill,図5・42(a)]とBTA方式ドリル[boring & trepanning association system drill,図5・42(b)]である.いずれのドリルも先端部に切れ刃とガイドパッドをもっており,軸直角面内の切削抵抗をガイドパッドで受け穴壁をバニシするとともに,高圧の切削油剤を切れ刃部に供給することによって切れ刃部の潤滑と冷却,ならびに切りくずの排出を行うものである.ガンドリルの場合は,シャンク内部を通して油を刃先まで送り,シャンク外側の扇形の部分を通して切りくずと油を排出する.最近,マシニングセンタにおいてブシュを使わずにまず通常のドリルにより浅穴をあけ,その穴をガイドブシュとしてガンドリル加工を行う方法が提案され,深穴加工の高能率化に貢献している.(a) ガンドリル(b) BTA 方式ドリル図5・42 深穴加工用ドリル図5・43 BTA 方式深穴加工の状況(a) マシンリーマ(b) マパールリーマ図5・44 リーマの例BTA 方式ドリルでは,ドリル外周部と穴壁のすきまを通して高圧油を切れ刃部まで送り込み,ボーリングバー内部を通して切りくずと油を排出する(図5・43).ガンドリルは直径30mm以下の深穴加工に用いることが多く,BTA 方式ドリルは直径20mm以上の深穴加工に用いることが多い.BTA 方式ドリルの系列には,大径用として,エゼクタドリル(ejector drill)や穴の心を残すトレパニングドリル(trepanning drill)などがある.
