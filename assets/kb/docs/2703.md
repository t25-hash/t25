b.工作機械などの軸 工作機械の刃物軸，スピンドル（spin-
dle），そのほか機械内部の軸類は，強さとともに軸の使用目的に応
じて特に変形量の少ないことや，寸法の精度および耐摩耗性が要求
される．この場合には変形の許容量が一般の伝動軸に比べてはるか
に小さいので，まず剛性の観点から設計し，その後に強さの検討を
行う．
許容変形量は機械の種類や使用目的によって異なるが，例えば工
作機械の歯車伝動軸では歯車装着点のたわみを3/（mm）［ ：歯
幅(mm)］に抑える．ただし特に歯車の精密なかみ合いを要求され
るときは，それよりはるかにたわみを小さく制限する必要がある．
両端が滑り軸受で支持された機械軸のたわみは1.5mm ［：軸
受中心より負荷点までの距離(m)］以下にとる．軸のねじり剛性に
ついては，普通静荷重用の軸で0.33° /m，変動荷重を受ける軸で
0.25° / m（＝0.00436rad/m），急激な繰返し荷重または長い送り
軸では0.17 ° /m以下にとる．工作機械の主軸のねじり剛性は面板
の周の動きを0.4mm以下に，またフライス盤の主軸では先端の
工具のねじれ角を1°以下にとる．
c.車軸（axle) 車両の車軸には自動車の前車軸のように回
転せずに軸端の軸受で車輪を支えるものと，鉄道車両の車軸のよう
に車輪と一体で回転するものとがある．いずれも車体の重量を支
え，主として垂直荷重による曲げモーメントを受けるが，車輪の駆
動軸にはねじりモーメント（トルク）も作用する．そのほか，車両
の旋回運動時に車体に働く遠心力，または車輪の接地摩擦力による
横荷重も作用し，それらから付加的曲げモーメントやねじりモーメ
ントも受けるのでそれらを考慮して設計を行う．
d.プロペラ軸（propeller shaft ) 船舶あるいは航空機のプロ
ペラを取付ける軸は，プロペラによって生じる圧縮力または引張
力，そしてプロペラの自重および軸力の偏心による曲げモーメント
を受けるとともに，駆動トルク（ねじりモーメント）も伝達する．
軸径の計算は一般的には表Ⅰ-3･2の設計式を用いればよいが，軸
力の偏心による曲げモーメントは特異なものである．例えば，船体
または機体が移動することによって，プロペラが置かれている部分
の水流または気流は不均一になり，この中で回転するプロペラから
生じる軸力は軸の中心線と一致せず曲げモーメントを発生するの
で，設計に当たってはこの値も考慮しなければならない．
e .たわみ軸（flexible shaft ) たわみ軸は，ねじり剛性はあ
るが曲げ剛性が著しく小さいたわみ性の軸であり，単なる回転また
は小動力を方向を自由に変えつつ伝動する場合に用いられる．それ
には自身のたわみ性を利用する多層より線形のもの（図Ⅰ-3･5）
と，短い軸を小形の自在継手などで多数連結した形式のもの（図
Ⅰ-3･6）とがある．
より線形たわみ軸は数層（普通4～10層）の密巻きコイル線から
なり，内層ほど細い素線を用い，中央に心線を入れたものもある．
各層は1ピッチに数本ずつの多重巻きとし，かつ隣の層は順次逆巻
きとする．トルクはおもに外層で伝えられるので，普通は最外層の
最も太い素線が引張力を受ける向きのトルクに用い，逆向きのトル
クに対しては正トルクの15 %程度の伝達能力となる．しかし最外
層の一つ内側の層の素線にも太い線を用いて丈夫にしておけば，
70～80%の逆トルクに耐えるものもできる．多くの場合軸の周囲に
たわみやすい案内筒を付け，その間の摩擦による摩耗を防ぐために
経年変化しない潤滑油を塗り，両端には継手を溶接，ろう接または
プレス締めして用いる．素線の材質は0.6～0.7%Cの高炭素鋼が
普通である．
