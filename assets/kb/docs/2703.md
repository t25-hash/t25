# 5・5・2 歯切り加工・シェービング加工

> **章**: 第5章 切削加工
> **節**: 5・5 形状創成加工

a. ホブ切りホブ切り(hobbing)は、ホブと歯車材との相対運動によって歯車を削り出すことができる創成歯切り法の一つである。高精度の歯車を高能率で加工することができるので、歯切り加工中、最も多く用いられている。ホブは、ウォームに数条の縦溝を入れて切れ刃を作り、ウォームのねじすじに沿って二番をとった一種のフライスである。これを軸の回りに回転させることによって並進するラック歯形を創り出すことができる。ホブ切りでは、このホブと歯切りしようとする歯車材とを図5・59(a)のように取付け、ホブを矢印の方向に回転させると同時に、それによって得られるラック歯形と理想的にかみ合うような回転を歯車材に与える。これにより、図5・59(b)に示すホブが最も深く切り込む面(創成仕上面)内において、ホブの多数の切れ刃により歯車が創成される。歯車の一つの歯形が創成される様子を図5・60に示す。さらに、ホブを歯車材軸に沿って送ることによって、歯幅全体にわたって歯車を削り出すことができる。(歯車材の回転数)×(削り出す歯車の歯数)=(ホブの回転数)×(ホブの条数)という関係が成り立つから、1条ホブのときには、ホブ1回転につき歯車材は1ピッチの割合で回る。ホブ盤(gear hobbing ピニオンカッタによる歯切りピニオンカッタ(piniontype cutter)は歯車の形をした工具で、歯すじに二番をとり、一端に切れ刃を付けたものである。これを歯車形削り盤(gear shap-ing machine)に取付けて軸方向に往復切削運動を与えると同時に、歯車材とともに理想的にかみ合うような回転送りおよび歯車材との相対的な切込みを与えて、このとき干渉する部分を削り取り、歯車を創成する。平歯車(spur gear)、はすば歯車(helical gear)はもちろん、ホブ切りでは不可能な段付き歯車、内歯車をも容易に歯切りすることができるが、生産性ではホブ切りに劣る。カッタの良否とその取付精度が削り出される歯形およびピッチに直接影響を及ぼすから、精度の高いカッタを正しく取付けることが大切である。カッタの材質としては、以前はCoを含まないMo系の溶解高速度工具鋼であるJIS 規格のSKH 51材あるいはSKH 51,SKH 53相当材の粉末高速度工具鋼が使用されていたが、量産歯車用としては高速・重切削が可能なCo,V を多く含んだ靱性の高い粉末高速3―156加工学・加工機器度工具鋼の適用が一般的である。ホブと同様に、イオンプレーティングカッタがほとんどの歯車加工用として広く採用されている。さらに最近ではホブ切りと同様に最新のカッタ、ギヤシェーパ(gear shaper)によってドライ切削による歯切りも実用化されている。適用される切削速度は50~90m/min、さらには100m/min以上の高速切削歯切りが、最新の表面処理のカッタと動剛性に優れたギヤシェーパにより実現されている。最近では、はすば歯車をピニオンカッタで歯切りする場合に必須であった、ヘリカルガイドと呼ばれる加工リードを与える部品を必要としない画期的なギヤシェーパも開発され、広く採用されつつある。c. ラックカッタによる歯切りラックカッタ(rack type cut-ter)は、ピニオンカッタのピッチ円直径を無限大にしたものであり、形状が単純であるため製作が容易である。これを歯車形削り盤に取付け、軸方向の往復切削運動を与えると同時に、歯車材とともにかみ合い運動を行わせて歯車を創成する。送り運動の方式によって、マーク式とサンダランド式がある。ラックカッタによる歯切りは現在、ホブ切り、あるいはピニオンカッタによる歯切りに替わり、ほとんどの歯車工場では加工法自体も、非常に低い生産性のために適用されていないが、大型歯車、大モジュール歯車の歯切り加工法として一部の大型歯車メーカにおいて適用されている。d. かさ歯車の歯切りすぐばかさ歯車(straight bevel gear)は、ピッチ円すい(pitch cone、かさ歯車のピッチ面をなす円すい)の母線に沿って、頂点に向かう歯すじのまっすぐな歯を付けた歯車で、交わる二軸間に力を伝えることができる。すぐばかさ歯車の歯切り法としては、バイトを用いて創成歯切りを行う方法と、セグメント状の刃を円周上に配した形状のレバーサイクルカッタを用いて成形歯切りを行うグリーソン社の方法がある。まがりばかさ歯車(spiral bevel gear)およびハイポイド歯車は、歯すじがねじれた曲線のかさ歯車である。これらの歯車の歯切り法としては、環状カッタを用いて創成歯切りを行うグリーソン社の方法と、円すいホブを用いて創成歯切りを行う方法とがある。e. シェービング加工シェービング加工(gear shaving)は、はすば歯車の歯面に切れ刃溝を付けた形状のシェービングカッタと、ホブ盤や歯車形削り盤等によって荒加工した歯車とをかみ合わせて、一方を回転駆動させ、他方を自由に回転させることにより歯車歯面を仕上げる方法である。回転中、工具と歯面とは軸を交差させたねじ歯車としてかみ合っているために、歯面上で両者の間に仕上げられる歯車の歯すじ方向の滑りが生じる。この滑りによって切削が行われ、歯車は滑らかに仕上げられる。シェービング加工にはカッタを相対的に歯車軸方向に送るコンベンショナル加工、歯車軸に対して直角方向に送るアンダパス、歯車軸に対して斜め方向に送るダイヤゴナル加工、および切込み方向のみに送るプランジ加工がある。プランジ加工は高能率な加工が可能であるため、量産加工において広く用いられている。シェービング加工では、歯車の凸凹やピッチのむらを除去したり、加工後の熱処理時の歯形の変形を考慮して歯形を修正したりすることが可能である。また、クラウニングも容易に行える。ただし、加工精度を安定に保ち、また、工具寿命を延ばすためには、前加工で精度を管理し、シェービング加工の取りしろを一定量に保つ必要がある。また従来はカッタ軸を回転させて歯車を連れ回り加工する方法が主流であったが、同期制御技術の進歩により、カッタ軸と歯車軸を同期回転させて加工する同期シェービング加工が可能となった。同期シェービングでは加工時間の短縮だけでなく歯車のピッチ精度を改善することが可能である。
