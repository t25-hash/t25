# 3・2・1 滑り軸受（plain bearing）の種類と選定

> **章**: 第3章 軸・軸受要素
> **節**: 3・2 滑り軸受

動特性も含まれる．
滑り軸受に要求される荷重の支持方向，スペース，運転条件（負
荷の大きさ，速度，環境温度），性能，寿命などの仕様によって，
採用すべき軸受の種類と詳細設計が定まる．その際，転がり軸受も
選択肢の一つに入り，両者の性能比較も同時に検討される．滑り軸
受は，荷重の方向によりジャーナル軸受とスラスト軸受に大別さ
れ，また，荷重支持原理により流体潤滑軸受と磁気軸受，流体潤滑
軸受はさらに動圧軸受と静圧軸受に，動圧軸受にはさらにくさび膜
static load）
作用による一般的なタイプとスクイーズ膜作用による特殊なタイプ
がある．また，使用潤滑剤により液体潤滑軸受，気体軸受，無潤滑
軸受（自己潤滑軸受あるいはドライベアリング）に，軸受材料によ
り金属軸受，非金属（セラミックス，プラスチック，ゴムなど）軸
受に分類される．そのほか，荷重（静，動），運動の種類（回転，
直動），速度（高，低），温度（高，低）などの作動条件，用途，精
度などによる分類もある．

体の負荷能力が生じる．滑り軸受を設計するには，潤滑面形状を含
む軸受の形式をまず選び，次に軸受材料と潤滑剤を定め，要求仕様
を満足する軸受特性が得られるまで設計変数を変えて軸受特性計算
を行う．その軸受特性にはゾンマーフェルト数，最小油膜厚さ，摩
擦係数，必要給油量，軸受温度などの静特性だけでなく，その軸受
で支持される回転体の振動応答振幅特性，自励振動の安定性などの
やむを得ず混合潤滑・境界潤滑状態，あるいは乾燥状態で使用す
る場合は，軸受材料，潤滑油とその添加剤，あるいは固体潤滑剤の
選定が特に重要になる．流体潤滑状態で使用される軸受でも起動，
停止時には混合潤滑・境界潤滑状態になるので，軸受材料および潤
滑剤の選定にはその配慮が必要になる．
