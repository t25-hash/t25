# 5・5・3 ねじ切り

> **章**: 第5章 切削加工
> **節**: 5・5 形状創成加工

めねじを形成する際の切りくずを排出するための溝が、軸線に平行なものがハンドタップで、一般的に用いられる。溝が軸線に対して右にねじれているものがスパイラルタップで、切りくずをシャンク側に排出しねじ部の損傷を軽減する機構になっている。溝が軸線に対して左ねじれになっているタップが左スパイラルタップで、切りくずを進行方向に押し出すので通り穴のねじ立てに適している。食付き部の切れ刃側の溝を左ねじれと同一方向に数山削り取ったスパイラルポイントタップも通り穴用のタップで、ガンタップとも呼ばれる。軟鋼や非鉄合金など被削材が展延性に富んでいる場合には、塑性加工によりめねじを形成する溝なしタップもしばしば用いられる。このほかにねじ部の径が微調整可能なエキスパンジョンタップやアジャスタブルタップ、タップの逆転時間を短縮する刃部引込み式のコラプシブルタップなどがある。b. ダイスダイス(die)は、切りくずを逃がす溝をもち、おねじを切るのに用いるめねじ形の工具で、固定ダイス・調整ダイスの2種類がある。一般には、割り部を設けて径の調節ができる調整丸ダイスがよく用いられる(図5・62)。ねじ山形切れ刃をもつチェーザをダイヘッドに取付けてねじを加工するダイヘッドチェーザには、ラジアル形、接線形、丸形の3種類の形式がある。また、これには刃が動かない固定型と、ねじ加工後には刃が自動的に開く開閉型とがある(図5・63)。このほかに、多量生産の分野では転造によるおねじ加工法が広く用いられている。文献(1) 中山一雄、切削加工論、(1978)、8,9,16,111,127、コロナ社。(2) 杉田忠彰・ほか2名、基礎切削加工学、(1984)、66,102、共立出版。(3) Krystof, J., Berichte uber Betriebswissenschaftliche Arbeiten,Bd.,12、(1939)、1-19,VDI Verlag.(4) Merchant,M.E.,J.
