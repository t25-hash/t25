# 6・1・2 研削工具と砥粒

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

研削工具（grinding tool）は研削砥石であり，気孔（pore）を
包含しながら砥粒（grain）を結合剤（bond）で固めたものであ
る．構成要素である砥粒，結合剤，気孔の三者を研削砥石の3要素
と呼ぶ．これらを定性的・定量的に規定して砥石の材質としての性
図6･4穴と熱電対付近における温度分布
質を決定する五つの要因すなわち，研削材の種類（下の例では
WA，以下同様），砥粒の粒度（80)，結合度（H)，組織（６)，結
合剤の種類（V)，を研削砥石の5因子と呼ぶ．これらの前に砥石
の幾何学的性質を規定する，結合研削材砥石の呼び（平形砥石），
規格番号（JIS R6211-1），形状（１)，縁形（L)，外径（205），
厚さ（18)，孔径（50.8）を，後ろに最高使用周速度をつけて例え
ば次のように表す．
平形砥石JIS R6211-11L-205×18 ×50.8WA80H6V-30m/s
砥粒の材質のうち人造研削材 は，A：褐色アルミナ（AlO
不純），WA：白色アルミナ（AlO純），PA：淡紅色アルミナ
（AlO ＋CrO)，HA：解砕型アルミナ（AlO単一結晶），AE：
人造エメリー（コランダム結晶とムライト結晶），AZ：アルミナジ
ルコニア（コランダム結晶とアルミナジルコニアとの共晶），C：
黒色炭化ケイ素（SiC不純），GC：緑色炭化ケイ素（SiC純）であ
る．超砥粒と呼ばれるのはダイヤモンドとcBN（立方晶窒化ホウ
素) であり，D：天然ダイヤモンド，SD：合成ダイヤモンド，
SDC：金属被覆合成ダイヤモンド，BN：立方晶窒化ホウ素，
BNC：金属被覆立方晶窒化ホウ素である．なおこれら超砥粒の場
合は先の研削砥石と区別してホイールといい，主として金属製台金
の周辺または端面に砥粒層をもつ．
砥粒の粒度 は三つに区分されそれぞれの試験方法 が異な
り，粗粒（F4～F220）はふるい分け試験方法，一般研磨用微粉
（F230～F1200）は光透過沈降法と沈降試験方法，精密研磨用微粉
（#240～#8000）は沈降試験方法と電気抵抗試験方法が用いられる．
なお上の例のように，粗粒についてはFを省略して表示すること
ができる．ふるい分け試験方法による粗粒の粒度は，例えば60
～70%がとどまるふるいの目開きは，F4：4.00～4.75mm，
F16：1.00～1.18mm，F36：425～500 m，F60：212～250
m，F80：150～180m，F220：45～53mである．また，一般
研磨用微粉と精密研磨用微粉では同じ粒度の粒子径が異なり，例え
ば，F240：48mと#240：60m，F360：27mと#360：40m，
F800：11mと#800：18m，F1200：7.8mと#1200：13m
（沈降試験方法による -50%粒子径) である．超砥粒の粒度に
はナローレンジとワイドレンジの区分があり，それぞれにA，B，
2方式の表示が規定されている．
結合度 はAが最も軟らかくZが最も硬い．結合材の量が多い
と硬くなるが他の要因もからむ．超硬合金等のマイナスドライバに
似たビットを一定加重のもとでねじ込んだ際の食込み深さによって
表され，結合度に対する食込み深さが粒度ごとに定められてい
る．
組織（0～14）は砥石中の砥粒の体積率を表し，0：62%から
14：34%まで2 %刻みに定められている．超砥粒ホイールにおい
てはコンセントレーションと呼び，砥粒層の砥粒体積率を表し，
200：50.00%，175：43.75%，150：37.50%，125：31.25%，
100：25.00%，75：12.50%，50：12.50%，25：6.25%であ
る．
結合剤（ボンド）の種類のうち最も一般的なのはV：ビトリフ
ァイド であり，長石，陶石，耐火粘土および媒溶剤などを水で
練り，乾燥させて焼いたものである．R：ゴム は主として切断
用であり，天然ゴムまたは合成ゴムを加硫したものである．RF：
繊維補強付きゴムは繊維で補強したゴムである．B：レジノイド
（合成樹脂) は天然ゴム，合成ゴム以外の合成樹脂であり，切断
用にも用いられる．BF：補強付きレジノイド（合成樹脂）はこれ
をガラスクロス等で補強したものである．E：セラックは樹木に寄
生する虫が分泌する天然の熱硬化性樹脂である．Mg：マグネシア
はマグネシアセメント（MgOとMgCl）を用いる．S：シリケ
ートは水ガラスに代表されるケイ酸塩を用いている．M：メタルボ
ンドおよびP：電着メタルボンドは超砥粒ホイールに用いられ
る．これら結合剤の得失による変遷もある．

通常の条件下での研削作業における各種被削材料に対する砥石の
選択標準が定められている．例えば，高張力の被削材にはA系
砥粒，低張力の被削材にはC系砥粒を用い，硬くてもろい被削材
には細かい粒度・低い結合度・密な組織を用い，軟らかく強靱な被
削材には粗い粒度・高い結合度・粗な組織を用いる．
845℃に達すると再焼入れされる とされる．
