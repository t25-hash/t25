# 6・1・3 研削加工面の幾何学的創成過程

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

先の6・1・1・a項に述べられているように,通常の研削作業によって得られる研削加工面は平面か円筒面である.円筒研削において工作物の回転に同期して砥石の位置を変化させるとカム等の研削ができる.総形研削は砥石輪郭にもたせた目的形状を転写して研削加工面を形成する.ねじ研削もほぼこれによっているが,ねじ溝はねじれているため,砥石の干渉が起きて砥石輪郭は正確には転写されない.創成研削(generation grinding)は歯車研削等に用いられる.歯車の創成研削では歯車工作物を回転させ,そのピッチ円上の周速度に一致させて砥石を直線運動させることで,平面である砥石の端面によって正確なインボリュート歯面を創成する.総形研削によるねじ・ボールねじの研削では砥石の干渉が起きることと,砥石を溝形状に整形することに手間がかかることから,普通形状砥石あるいは不整形な砥石による創成法が試みられている.以上の幾何学的目的形状を得るために砥石切込み深さを設定して加工を行った際に,工作機械や工具・工作物の変形等によって切り取られる工作物の深さは設定値より小さくなる.図6・6はプランジ研削におけるこの過程を示した例であり,設定切込み(砥石台移動量)に対して工作物半径は遅れて減少していることがわかる.また,工作物の熱変形による切込み深さの変化も存在する.
