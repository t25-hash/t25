# 6・1・4 研削加工面の性状

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

研削加工面は砥石表面に露出した多数の砥粒切れ刃が高速で微細な切削を行うことにより形成されるため、滑らかな面ではなく粗さが存在する。粗さは確率的に求められるが、研削方向に垂直な断面における算術平均粗さ(旧中心線平均粗さ)は、例えば次式で与えられる。また、研削焼けの目安は加工面の反射率がアルミニウム蒸着面に対して16%に低下する点であり、研削点の最高温度が約500°Cに達すると一義的に発生するともいわれている。表面内部の組織変化も観察され、例えば焼入れ後硬度HC64まで焼戻ししたAISI52100鋼は最高温度約345°Cに達するとさらにHC63まで焼戻しされ、最高温度約845°Cに達すると再焼入れされるとされる。研削割れについてもおもな原因が調べられている。それによると、高速度鋼では焼入れ組織中に焼戻しされずに残っているマルテンサイトが原因で、ダブルテンパリングで防止できる。浸炭鋼では浸炭層の過剰炭素が原因で、表層の炭素量が0.9%以下ではあまり発生しない。球軸受鋼などでは残留オーステナイトが原因であるとされている。これら熱損傷は悪影響であるが、一方ではこれを積極的に利用して、新しい熱処理法として利用する試みもなされている。
