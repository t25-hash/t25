# 3・2・5 気体軸受（gas bearing，Ⅱ-1･4･5参照）

> **章**: 第3章 軸・軸受要素
> **節**: 3・2 滑り軸受

気体軸受は気体膜圧力により物体を非接触に支持し，位置決めす
る機構要素である．油潤滑軸受と同様に，軸受面の相対運動によっ
て生じる圧力を利用する動圧軸受と外部から供給される加圧気体の
圧力を利用する静圧軸受がある．気体軸受の潤滑剤は普通空気で，
大気環境で使用される動圧型では潤滑剤が自然に供給されることが
最大の利点である．油に比べて気体の粘性係数は約1/1000なの
で，摩擦力が小さいという利点があるが，同時に負荷容量や軸受剛
性も小さいので，軸受すきまを数 m以下に小さくし，小形軽量な
高速回転体の支持に用いられる．またターボ膨張機などでは作動気
体を潤滑剤に用いることができる．静圧型では圧縮気体が必要であ
るが，低速でも動圧型よりも負荷容量・剛性が高くでき，低摩擦・
清浄なので精密加工機械，計測・試験機器などの軸受に利用され
る．
