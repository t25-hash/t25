# 3・2・6 磁気軸受（magnetic bearing）

> **章**: 第3章 軸・軸受要素
> **節**: 3・2 滑り軸受

磁気軸受の最大の特徴は磁気力を利用してロータを非接触で支持
することである．この特徴を生かしてほかの軸受では不適切な分野
にも磁気軸受が使われている．ラジアル荷重を支えるラジアル磁気
軸受とスラスト荷重を支えるスラスト磁気軸受がある．また利用す
る電磁力から(１)磁気吸引力だけを利用するもの，(２)磁気反発力
と磁気吸引力を併せて利用するもの，(３)電磁誘導反発力を利用す
るもの，(４)超電導を利用するものがある．さらに，磁気力を発生
する磁石の種類としては永久磁石，常電導電磁石，超電導磁石があ
る．
a.軸受の構成 最も一般的な磁気吸引力だけを利用する磁気
軸受は図Ⅰ-3･49に示すように，磁気吸引力を発生する電磁石，軸
と軸受間のすきまを計測する非接触変位センサ，軸受すきまを一定
に保つよう電磁石に流す電流値を指示する補償回路，補償回路から
の指示に従って電磁石に電流を流す電力増幅器から構成される．さ
らに固体潤滑材を使用した非常用転がり軸受，電力増幅器へ供給す
る直流電流を商用交流電源から変換するインバータDC電源，商用
電源やDC電源の故障に対処するバックアップバッテリー，運転状
態を監視する状態監視モニタも装備している．
b.軸受負荷能力 磁気軸受の負荷能力は電磁石鉄心に使用す
る磁性材料の磁気飽和特性で決められる．電磁石鉄心には，けい素
鉄，磁性ステンレス綱が使われている．これらの磁性材料の飽和磁
束密度は1.6T（磁性軟鉄や3%けい素鉄）から1.4T（磁性ステ
ンレス）程度である．したがって上限変動荷重を考慮して，静的最
大磁束密度が1.2T程度で支えられる静加重を設計することが無
難である．鉄心は，鉄心内を通過する変動磁束による渦電流の発生

c .軸受すきま 磁気軸受のすきまを非常用軸受すきまより大
きく取ることにより，磁気軸受の磁気吸引力が失われても磁気軸受
のロータとステータが接触することを防止している．軸受すきまを
大きく取ると磁気回路の磁気抵抗がそれに比例して大きくなり，コ
イルを励磁する電源電圧を高くしなければならなくなるので，軸受
すきまをむやみに大きくはできない．しかし，滑り軸受のすきまと
比べれば格段に大きくできるので，ロータが大きく振れまわる回転
機械の軸受に適している．
d.制御法 磁気軸受の制御には，(１)アナログPID制御，
(２)ロータの高次固有振動発振対策やターボ機械での不安定振動を
励起する作動流体からの流体力に対する補償のためディジタルロバ
スト制御，(３)ノッチフィルタ挿入，(４)回転体の質量分布の不均
一性による不釣合いによる振動を補償して軸振動をなくす制御（ア
ンバランス相殺制御：unbalance counteracting control ），(５)回
転軸の振れまわりは許容するが，不釣合い慣性力による影響が静止
構造物に伝わらない制御（アンバランス力遮断制御：unbalance
force rejection control ）などの手法により制御される．さらに磁
気軸受の励磁電流と磁気吸引力は非線形な関係となるので，非線形
制御も研究されている．
e.信頼性 磁気軸受を用いてロータをフィールドで加振して，
ロータ磁気軸受システムの開ループ伝達関数を計測し，感度関数を
求めることができる．感度関数の逆数が古典制御のゲイン余裕や位
相余裕に対応しているので，ロータ磁気軸受システムの安定性をフ
ィールドで評価しながら運転できる点がこの軸受の優れた特徴であ
る．電気部品点数が多いので，信頼性を高めるため，システムの多
重化を採用することもある．
f.DN値 ロータの外径（単位mm）と回転数（単位rpm）
の積をDN値と呼ぶ．許容DN値は，ラジアル軸受ではロータ鉄
心材料，アキシアル軸受ではアキシアルディスクの鉄心材料の強度
によるがおおむねラジアル軸受で4×10，アキシアル軸受では1×
10である．

ヒステリシス損は高速回転では渦電流損に比べて小さい．ステータ
