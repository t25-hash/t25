# 6・1・5 砥石摩耗と砥石寿命

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

砥石摩耗(grinding wheel wear)には砥粒切れ刃の摩耗による切れ味の低下を意味する場合と、砥石の幾何学的形状・寸法の変化を意味する場合がある。切れ刃の摩耗による切れ味の低下は目づまり(loading)と呼ばれ、目直しによって回復させる。先の6・1・1・a項に述べられている切れ刃の自生作用は、切れ刃の摩耗が砥粒の切削抵抗の増加につながり、それによって砥粒がへき開して鋭利な砥粒切れ刃が再び現れて目づまりが生じない状態を意味している。一方、切れ刃に切りくずが付着する状態を目つぶれ(glazing,dulling)と呼び、これも目直しによって回復させる。砥石寿命(grinding wheel life)も切れ味の低下により目直しが必要となるまでの間隔を意味する場合と、寸法の減少により砥石が使用不能となるまでを意味する場合がある。砥粒切れ刃の摩耗は逃げ面の面積の増加となって現れて寿命の目安となるほか、図6・7のような研削抵抗の増加は実際の寿命を支配する。ただし、α(0.1~0.5mm)は砥粒摩耗定数、はほぼ50 mの定数である。砥石の幾何学的な摩耗はツルーイング(truing、形直し)によって修正する。目直しと形直しとは、その目的は異なるが、ドレッサを用いて行う作業の内容は同じである。砥石の幾何学的な摩耗は、砥粒の摩耗と切れ刃の自生作用(self-sharpening)によって徐々に生じるほか、砥粒の脱落[目こぼれ(shedding,breaking)と呼ぶ]によって大きく生じる。長手送り円筒研削においては必ずしも砥石の端面側が摩耗しやすいとは限らず、工作物1回転当たりの長手送り量によって摩耗の箇所が異なることに注意を要する。摩耗は砥石の熱膨張と補完関係にあるが、その進行特性が異なるので、プランジ研削においても仕上精度に与える影響は無視できない。これら砥石摩耗に関連する研削状態は、目こぼれ型、正常型、目づまり型、目つぶれ型と分類され、砥石や研削条件によって図6・8のように変化する。
