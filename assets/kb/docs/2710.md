# 6・1・6 研削油剤

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

研削油剤(grinding fluid)は砥石の摩耗低減と仕上精度の向上などに有効であることが求められているが,具体的には冷却作用と潤滑作用および洗浄作用である.大きく分けて鉱物油または脂肪油を主成分とした不水溶性(N 1~N 4種)と水溶性(A 1~A 3種)があるが,研削加工においては引火の危険性と冷却作用の効果のために水溶性が主流である.A 1種はエマルジョン(emulsion)と呼ばれ,鉱物油や脂肪油に界面活性剤を加えたもので,水で希釈すると乳白色になる.A 2種はソリュブル(soluble)と呼ばれ,界面活性剤単独またはこれと鉱物油や脂肪油などからなり,水で希釈すると半透明ないし透明になる.A 3種はソリューション(solu-tion)と呼ばれ,無機塩類を主成分とし水で希釈すると透明になる.従来研削油剤の高性能化に威力を発揮した塩素系極圧添加剤は環境保全のために用いられず,塩素に代えて硫黄系添加剤が用いられ,その全硫黄分も5%以下に抑えられている.水溶性における亜硝酸塩も発がん性のため用いられなくなり,その特性を他の物質で補っている.塩素,亜硝酸塩のほかベンゼンなどの有害性有機溶剤も使われず安全性は高くなり,脱脂作用やアルカリ作用および防腐剤・殺菌剤などによる刺激は洗濯用洗剤と同程度であるが,皮膚に付着する機会が多いことやミストが呼吸器粘膜に刺激を与えることには注意を要する.環境保全からは研削油剤の使用量は少ないことが望ましく,極微量の油剤による加工の研究もなされつつある.一方で加工の高速化が進むにつれて加工油剤の役割は将来ともなくならないとし,その供給,運用管理,洗浄,廃棄処理を一貫して扱うサービスシステムも開発されている.
