# 6・1・7 研削条件と研削コスト

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

円筒研削作業における製品1個の研削コストは加工費、動力費、砥石消耗費の三つの経費要素の和とし、以下のように求められている。動力費は研削容積と単位動力当たりの費用に、研削動力とドレッシング動力を加えた総動力を乗じて次式で表されている。砥石消耗費は単位砥石消耗深さ当たりの費用に総ドレッシング深さを乗じて次式で表されている。これらのに対する変化は図6・9に示したとおりであり、研削コストには工作物速度と砥石切込み深さに対する最適値の存在することがわかる。また、その最適値の位置は加工費の増加によって右に移動するから、人件費の高騰に対処するには工作物速度と砥石切込み深さを増加すべきであることを物語っている。
