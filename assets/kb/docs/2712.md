# 6・1・8 倣い研削

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

平面研削や円筒研削による単純な平面や円筒面の創成ではなく、研削によって複雑な輪郭(profile,contour)を創成する方法の一つがプロファイル研削(profile grinding)で、コンタリング研削(contouring grinding)、倣い研削(copy grinding)とも呼ばれる。もう一つの方法である総形研削は、創成したい形状の断面を砥石作業面の幅方向に成形し、この形状をプランジ研削で工作物に転写するものであり、倣い研削はこれとは異なる。倣いとはコピー(模倣)のことである。コピーするためには見本が必要であり、その見本に倣って工作物または砥石を移動しながら見本どおりの輪郭形状を創成する。見本は実体のあるもの[拡大、等倍、あるいは縮小の実物、すなわちテンプレート(template)]であったり、図面であったり、数値データであったりする。図6・10は自動車用カムシャフトのカムプロファイル研削の模式図である。カムは、円筒研削盤をベースとする専用研削盤において、砥石台と工作物との間に相対運動を与えて加工されるが、実物のカム(テンプレート)に倣って相対運動を与えるものから、実物なしの数値制御(数値データ)によって相対運動を与えるものに変わってきている。図6・11は単純形状に成形された砥石を移動させて輪郭を創成す図6・9研削作業における経費図6・11 プロファイル(コンタリング、倣い)研削図6・12 通常研削とクリープフィード研削る場合で、テンプレートに沿って移動するスタイラスの動きを砥石に伝えるテンプレート式があったが、現在では、図面(工作物の仕上り形状を拡大して描いた半透明の樹脂フィルムの原図)の輪郭形状に沿って砥石を移動させる光学式倣い研削が一般的である。一般的にはプロファイル研削盤と呼ばれる光学式倣い研削盤には、工作物と砥石を映し出す光学式投影機が付いている。このスクリーンには原図が張り付けられており、この原図に工作物像と砥石像を重ねることによって、原図に描かれたけがき線に倣って砥石を移動させて加工を行い、加工後の形状がけがき線と一致するまで加工する。代表的な工作物として、打抜き型用パンチ(典型はICリードフレームパンチ)やダイス、焼結型用パンチ、総形バイトなどがある。
