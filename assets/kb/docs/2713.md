# 6・1・9 クリープフィード研削

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

研削法の呼び方には,創成する形状からの分類(平面研削,円筒外面研削,円筒内面研削,総形研削,プロファイル研削など),研削条件からの分類(重研削,クリープフィード研削,スピードストローク研削,高速研削など),そして加工物からの分類(工具研削,ジグ研削,カム研削,クランクシャフト研削など)などがある.通常とは大きく異なる研削条件であることから命名されたクリープフィード研削(creep feed grinding)は,航空・宇宙産業において,ジェットエンジンなどに用いられる耐熱合金(ニッケル基合金)製のタービンブレード(turbine blade)などを高能率に加工することを目指して開発された研削法である.例えばタービンブレードの場合には,精密鋳造あるいは鍛造されたブレード素形材に対して,そのルート(root)部やシュラウド(shroud)部を所定の寸法・形状に仕上げる必要があり,取りしろは3mm程度ある.初めは,この取りしろをフライス加工やブローチ加工によって除去していたが,あまりにも非能率的,また材料によっては不可能なことから,次に,これらに代わって通常のテーブルが往復する平面研削方式が適用された.しかし,通常の研削法では,砥石寿命が短い,成形された砥石の形状保持性が悪い,加工能率が低い,といった問題があった.そこでクリープフィード研削法が開発され,さらにこれに連続ドレッシング装置が付加されて発展してきたが,航空・宇宙産業のあまり盛んではない日本では,当初それほど注目されず,ヨーロッパが中心であった.現在では,クリープフィード研削は,自動車,ポンプ,ダイス,金型,工作機械,切削工具などの製造分野にも適用されており,特に,硬い材料や靱性の高い材料の加工に効果を上げている.さて,クリープフィード研削は平面研削様式で行われることが多いが,この研削法の最大の特徴は,図6・12に示すように,大きな取りしろを1パスから数パスで除去するところにあり,砥石切込み深さは1~10mmあるいはそれ以上,工作物速度は10~1000mm/minであり,多くの場合,所定の形状に成形された砥石を用いて,全取りしろをワンパスで除去する.研削中の砥粒への負荷を最大砥粒切込み深さで代表させれば,研削の幾何学より,それは工作物速度に比例し,砥石切込み深さに関しては0.5乗に比例する.したがって,クリープフィード研削における砥石切込み深さを通常の1000倍とすれば,研削中の砥粒への3―163加工学・加工機器(b) 砥石切込み深さと工作物速度が切りくずのサイズや研削抵抗に及ぼす定性的な影響負荷を通常の研削と同じにするためには,工作物速度を通常の約30分の1にすればよい.これによって,単位時間・単位研削幅当たりの研削量である金属除去率(=砥石切込み深さ×工作物速度)は通常の約30倍になることになる.しかし,接触弧長さが通常の約30倍,すなわち同時作用砥粒数も通常の約30倍となることから,研削抵抗は通常の約30倍となることになる.このままでは,研削盤の主軸やテーブルの駆動動力は通常よりも約30倍大きくする必要があり,研削盤の剛性も不足となる.したがって,一般的には,金属除去率を通常の研削と同程度にする.すなわち,砥石切込み深さを通常の1000倍とすれば,工作物速度は1000分の1である.これがクリープフィード(ゆっくりとした送り速度)の名の由来である.図6・13は,金属除去率が一定の場合において,工作物速度と砥石切込み深さが研削パラメータに及ぼす影響を定性的に示したものである.クリープフィード研削においては,以下のことがいえる.(1) 幾何学的な切りくず形状は非常に薄くて長い.(2) 砥粒に作用する研削抵抗は小さいが,総研削抵抗は大きい(比研削抵抗あるいは比研削エネルギーも大きい).したがって,通常の研削と同じ仕様の普通砥石を用いる場合には,砥粒切込み深さが小さくなりすぎて切れ刃の上滑りが多くなることから,砥石周速度は低いほうが望ましい.クリープフィード研削に適した砥石としては,組織が粗で連続切れ刃間隔が大きいことが必要である.このような砥石は回転破壊強度が低いから,必然的に低周速度となる.クリープフィード研削の長所・短所を通常の研削と比較して列挙すれば,以下のとおりである.(1) 総合的な研削能率が高い.(2) 研削抵抗および比研削エネルギーが大きい.(3) 工作物温度が低い.(4) 砥石輪郭形状の保持性が高い.(5) 仕上面粗さが小さい.(6) 工作物には圧縮の応力が残留しやすい.3―164(2)と(3)は矛盾するように思われるが,これには適切な砥石の選択と研削領域への確実かつ十分な冷却が前提となっている.すなわち,研削領域における余分な発熱を抑えかつ十分な冷却が行えるかが,クリープフィード研削の成否の鍵となり,換言すれば,研削焼けなどの熱的問題が加工限界となるであろうことが容易に予測できる.
