# 6・1・10 スピードストローク研削

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

スピードストローク研削(speed stroke grinding)は,平面研削加工における工作物テーブルの単位時間当たりの反転回数を大きくするとともに,砥石の切込みを連続的に与えて,高能率加工を行う方法である.工作物の材質と大きさにより異なるが,最大で反転回数は500~600回/min,砥石の切込み速度は数mm/minで加工することができる.この結果,通常研削と比較し実質的な砥石半径切込み量Δは表6・2に示すように小さくなる反面,工作物速度は大きくなる.その結果,個々の砥粒切込み深さは通常研削よりわずかに大きくなるため,砥粒切れ刃の自生作用が活発になり,砥石の切れ味が持続する.一方,砥石と工作物の接触弧長さ(砥粒切削長さ)は逆に短くなるため,研削油剤の冷却効果が良好になり,研削熱の発生も少なくなる.したがって,熱損傷を受けやすい材料や研削油剤の供給が難しくなる深溝加工に効果を発揮する.スピードストローク研削盤の高速反転テーブルには大きな加速度が作用するため,剛性の高い移動機構と重心移動に伴う振動を防止するためのバランス機構が必要不可欠である.図6・14は代表的なスピードストローク研削盤用の高速反転テーブルの構造を示したものである.スライド面は超精密油静圧案内で,往復運動はクランク機構で与えるようになっている.また,工作物固定用のテーブルと同質量のカウンタバランスを逆位相で反転運動させ,研削盤全体の重心移動をなくし,振動発生を極力防いでいる.この機構により,非常にスムーズな高速反転動作が可能となった.スピードストローク研削における研削加工率′は通常研削とほとんど同じである.しかし,テーブルの高速化とテーブル反転に伴うオーバランの減少により実加工時間を短縮することができる.そのうえ砥石の連続切込みが可能であるため,実質的な研削能率は大幅に向上する.しかし,高速で反転動作を繰返す必要からストロークを大きくとることが難しく,数十mm程度に限定される.した表6・2各種研削方式の砥石半径切込み量Δ,工作物速度,砥石周速度,研削加工率′の比較通常研削クリープフィード研削スピードストローク研削超高速研削Δ mm小:0.001~0.05大:0.1~30 小:0.0005~0.01大:0.1~30m/min大:1~20小:0.05~0.5大:1~50大:0.5~10m/s小:20~60小:10~60小:20~60大:100~500′mm/(mm・s)小:0.1~10 小:0.1~10 小:0.1~15大:50~図6・14 高速反転テーブルの構造がって,研削長さが短い工作物に適用されることが多い.図6・15(a)は超硬製ICリードフレーム用パンチのプロファイル加工例,(b)は超硬板材に40 mピッチで幅20 m,深さ0.2mmの細溝加工をした例である.スピードストローク研削は,後者のように研削面のたおれとうねりが問題になるような場合にも,良好な加工精度が得られるのが特徴である.さらに研削面が鏡面となるため,種々の超精密切断加工にも適用されている.
