# 3・4・2 転がり案内（rolling guide way）

> **章**: 第3章 軸・軸受要素
> **節**: 3・4 案内

機械のベッド上を直線運動するテーブルなどのように，直線運動
を行う移動物体の案内システムとして運動部分に玉やころの転動体
を組込み，転がり運動によって案内機能を得る転がり案内が，工作
機械，半導体製造装置，工業用ロボット，各種搬送装置などに幅広
く使用されている．
a.転がり案内の構造 代表的な転がり案内の構造は，図Ⅰ-
3･69 に示すように直線運動が必要とするストロークに応じて，
所定の長さをもつ断面がほぼ矩形のレールと，レールにまたがるよ
うに組合わされるキャリッジから構成されている．キャリッジは，
レールに設けられた転動面上を移動するために，キャリッジ内部に
は玉あるいはころのような転動体が複数列（2条列，4条列，時に
は6条列のものもある）組込まれており，キャリッジの移動に伴っ
て，荷重を負荷しながら転がり運動を行う．転動体はキャリッジの
両端に設けられたエンドキャップにより方向転換を行って，再び負
荷部へ移動する循環式をとるものが大半である．転動体として玉を
使用する場合，キャリッジとレールの転動面には図Ⅰ-3･70(ａ)サ
ーキュラアーク溝形状（単一円弧溝状）や図Ⅰ-3･70(ｂ)ゴシック
アーク溝形状（2円弧溝状）がある ．循環式のものは移動物体の
移動ストロークに制限はないが，非循環式のものは移動ストローク
に限界があるほか，理論上は移動物体の移動量の2分の1だけ転動
体の中心が移動するため，荷重条件によっては転動体列の片当たり
状態を生じるので注意が必要である ．
荷重が加わると玉と溝間では接触面積が広がり，玉と平面が接触
する場合に比べ，溝の曲率にもよるが，5～13倍の荷重を負荷する
ことができ，また，許容弾性変形量もかなり増大する ．転動面
は，高周波焼入れまたは浸炭焼入によりHRC58～64程度の硬さ
となり，成形研削により精度を得る．
精度は，レールに設けられた底面基準面および側面基準面と組合
わされるキャリッジの上面基準面および側面基準面により，走行平
行度として規定されている ．
また，今までの転がり案内は，ほとんど転動体どうしが直接接触
しながら循環する総転動体式であったが，最近の傾向として，玉と
玉あるいはころところの間に間座を設けて，直接の接触を防止する
ことにより，摩擦変動の低減，長寿命，静音などの性能を向上させ
たものが開発されてきている ．
