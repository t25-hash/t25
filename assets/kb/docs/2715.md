# 6・1・11 超高速研削

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

超高速研削(ultra-high speed grinding)は、クリープフィード研削(creep feed grinding)と同じような大きな砥石半径切込み量Δと、通常研削と同じような大きな工作物速度を与えて研削する方法である。この状態では最大砥粒切込み深さは異常に大きくなってしまい、研削不可能である。そこで砥石周速度を大きくし、最大砥粒切込み深さの値を通常研削と同程度にすることにより、研削を可能としている。実際は、=200~400m/sの領域まで砥石周速度を引き上げて研削を行う。この超高速研削の最大の特徴は、最初に研究が開始されたドイツにおいてHEDG(high efficiencydeep grinding)と称されていることからわかるように、非常に高い研削加工率が得られることである。例えば、ヴェルナー(W.Werner)らは砥石周速度180m/sで研削加工率3000mm/sという驚異的な値を実現している。表6・2に示すように、通常研削、クリープフィード研削の二つに比べ、超高速研削がいかに大きな研削加工率′となっているかがわかる。砥石の周速度を上げることにより研削加工率を向上させようとする試みは古くから行われていた。しかし砥石の回転強度不足のために、砥石の実用周速度は一部の研削切断を除けば、ほとんど60m/sの域を超えることができなかった。しかし在来砥粒に比べて耐摩耗性の極めて高いcBN 砥粒が開発され、それを用いた金属コア砥石が使用可能となって、砥石の回転強度が飛躍的に向上した。そしてヴェルナーらは周速度180m/sの超高速下で初めて研削を実現し、ケーニッヒ(W. Konig)ら、庄司らは300m/s以上とさらにその記録を書き換えている。砥石の回転破壊強度を高めるために、砥石断面形状や材質に工夫がなされている。図6・16は砥石断面と破壊強度の関係を示したものである。高速化のために、砥石センタ穴をなくし、砥石厚さを外周にいくほど薄くした形(a) 超硬製IC リードフレーム用パンチ(b) くし歯溝図6・15 スピードストローク研削加工例[提供:(株)ナガセインテグレックス]図6・16 超高速研削用砥石の断面状としている。また、材質は鋼、チタン、アルミニウム合金、CFRP(炭素繊維強化プラスチック)いずれも破壊強度には大差ないが、砥石軸に装着した場合の固有振動との関係で軽量であるほうがよい。アルミニウム、CFRP、あるいはそれらのハイブリッド型コアが開発されている。超高速研削は単に高能率研削を実現するだけではない。切削速度が高速化することにより、図6・17に示すような研削抵抗の減少や、目づまりや砥石摩耗の減少等の効果が報告されている。また、軟鋼研削において問題になっていた砥石の異常摩耗や研削表面への切りくずの溶着等も、超高速研削を適用することにより全く問題なく加工できるようになった。現在は超高速研削用の砥石のみならず、超高速回転に耐える高出力主軸、高剛性研削盤、高圧研削油剤供給システム、高速砥石バランス装置、超高速ツルーイング、ドレッシング装置等の開発が進み、超高速研削が可能な平面研削盤、円筒研削盤が市販されている。特に円筒研削への適用が先行しており、すでに砥石周速度200m/sでの研削が行われている。
