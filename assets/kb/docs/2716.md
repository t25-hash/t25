# 6・1・12 難削材の研削

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

難削材(difficult-to-grind material)のとらえ方は加工条件、要求精度により大きく異なるが、一般に次に示す現象が現れて問題となることが多い。(1) 研削抵抗が大きい。(2) 砥石の摩耗が大きい。(3) 研削焼けや研削割れが生じる。(4) 寸法精度や表面粗さが悪い。(5) 材料の脱粒やチッピングを起こす。例えば、ダイヤモンド、セラミックス、純金属、耐熱合金、チタン合金、オーステナイト系合金鋼などは代表的な難削材である。このような難削材を加工する場合、研削方式、研削盤の剛性や砥石駆動モータの出力、砥石、研削油剤、研削条件などのさまざまな要因を総合的に考慮する必要がある。しかし、基本的には難削性を示す原因を解明し、それに対する対策を行うことにより、加工可能となることが多い。難削性の原因としては以下の要因が考えられる。(1) 硬度が大きい。(2) 高硬度の粒子や繊維を含む。(3) 延性が大きい。(4) 熱伝導率が小さい。(5) 加工硬化しやすい。(6) 靱性値が小さく欠けやすい。(7) 活性度が高く砥粒や結合剤と反応、凝着しやすい。実際には、これらの原因のいくつかが複合して難削性を示している材料が多く、これらを加工する場合には、それぞれの原因に対する対応策を試み、その中から最も効率のよい方法を選べばよい。合成ダイヤモンドおよびcBN(cubic boron nitride、立方晶窒化ホウ素)に代表される超砥粒砥石の出現と、これら超砥粒砥石の性能を十分に発揮させるためのツルーイング、ドレッシング技術の進歩により、従来難削材といわれていた材料の研削が可能となって図6・17 砥石周速度と研削抵抗の関係3―165加工学・加工機器きた。特に上に記した原因の(1)や(2)のように硬度の点で難削性を示していた材料は、易削材へと変化した。セラミックスや超硬合金がその典型的な例で、ダイヤモンド砥石を使用すれば極端に硬い一部の材種を除いて容易に研削できるようになった。また、(4)のような熱的性質が原因の場合は、研削熱の発生を抑えるための対策と、発生した研削熱を効率よく除去するための対策をとることにより研削可能となる場合がある。前者の観点からは砥石の自生作用が活発になるような研削方式と研削条件を選定すればよい。例えば、6・1・10項のスピードストローク研削を採用することにより、難削材の一つであるインコネルのコンタリング研削を行った例がある。また、同じインコネルの研削において、砥石切れ味の劣化を連続ドレッシングという方法で回避しながら、クリープフィード研削を行っている例もある。この場合、超砥粒砥石ではなく通常砥石で高能率研削を行う点が特徴的であり、ジェットエンジンブレード基部の型研削に適用されている。この方法ではつねに切れ味のよい切れ刃が出てくることになるが、砥石の損耗量が著しく、それを随時補正する機構が必要となる。また、この場合に、発生研削熱量を小さくするために低砥石周速度を採用することが多い。後者の観点からは、研削油剤の種類の検討や高圧供給方法の採用、図6・18に示すような溝付き砥石の使用が効果的である。例えば、難削材の一つであるサーメットのクリープフィード研削に通液溝付き砥石を用いることにより、冷却効果が向上し、研削割れ、研削焼けが防止できた例も報告されている。一方、硬度は小さいが砥石の摩耗が大きいために難削性を示す材料もある。例えば、低炭素鋼の生材などがその例で、ビトリファイドボンドcBN 砥石で研削した場合、研削抵抗が小さいにもかかわらず砥石が異常摩耗を起こすことが知られている。これは、気孔に堆積した切りくずが工作物新生面に再溶着し、その後方の砥粒がなだれのように脱落するためである。異常摩耗の対策としては、砥石気孔内に目づまりしてしまった切りくずを研削油剤の高圧供給と高圧洗浄で除去する方法が効果的である場合がある。図6・19の写真は、軟鋼を通常研削および超高速研削により研削したときの表面である。通常研削による研削面上には切りくずや砥粒切れ刃の一部が付着しているのが確認され、良好な研削状態とはいえない。それに対し、砥粒切込み深さが同じ条件で比較した場合、超高速研削では良好な研削面と加工能率が得られている。これは砥石の高速回転による大きな遠心力で切りくずが飛散され、目づまりが激減したためである。このほかにも難削材の研削加工に際し、他の加工作用を重畳させる複合研削法も効果的である場合がある。例えば、超音波研削によるセラミックスの加工や放電電解研削による耐熱合金の加工が挙げられる。以上のように難削材加工の対策は多様である。しかし、難削材を研削する場合に重要なことは、難削性を示している原因を考察し、その対策を施した研削方法を採用することである。研削加工は、研削砥石の強制切込みにより、その作業面形状を砥石の回転運動と研削盤の送り運動(形状創成運動)を通じて工作物に転写する加工法である。このため、形状創成運動と回転運動が正確であることがまず要求される。これらの正確さは研削盤固有のものであるが、加工中における工作物と研削盤の熱変形や弾性変形などによる形状創成運動の劣化は、作業者の配慮(切込みや送りの減少、適切な研削砥石の選択と目立て作業、研削油剤の適切な供給など)によって抑制できる。また、研削砥石の回転運動精度は、研削砥石のツルーイング(truing、形直し)および研削砥石の動バランスの適切な調整によって向上させることができる。研削加工には、形状精度ばかりでなく高い品質の表面創成が求められる。そのため、研削砥石、ツルーイング・ドレッシング条件および研削条件の適切な選択、研削油剤のろ過と十分な供給などが必要である。特に、鏡面を仕上げる場合には、動バランスの確実な調整(振動振幅0.02 m以下)、微粒砥石の選択、鋭利な単石ドレッサまたはインプリドレッサによる精密ドレッシング、0.1 m以下の微小切込みと微細なトラバース送り、十分なスパークアウト研削などが欠かせない。一方、研削加工にはその能率を最大限高めることが求められることから、砥石回転数の高速化、テーブル速度や反転回数の増加、エアカット時間の低減、自動定寸、工作物や研削砥石の自動交換など、加工の高能率化、自動化に関わる多くの取組みがなされている。
