# 6・2・1 円筒研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

円筒研削(cylindrical grinding)は,円筒外面研削(externalcylindrical grinding)と円筒内面研削(internal cylindrical grind-ing)に分けられるが,前者を単に円筒研削と呼ぶことが多い.円筒外面研削は,図6・20に示すように,砥石の外周または側面を使って円筒状工作物の外周を研削するもので,以下の方式がある.(1) トラバース研削[図6・20(a)~(c)(f)](traverse grinding)(2) プランジ研削直角切込み方式(plunge-cut grinding)[図6・20(d)(e)]傾斜切込み方式[図6・20(g)]さらに,円筒外周ばかりでなく,段付き部分の端面の加工[図6・20(c)(g)]や工作物台を旋回させることでテーパ面の加工[図6・20(b)(f)]を,また研削砥石を目的の形状にあらかじめ成形することによって各種軸対称形状の総形加工[図6・20(e)(g)]を行うことができる.工作物の保持は両センタ支持とするのが加工精度上望ましく,工作物に回転力を与える回し金(lathe dog)が用いられる.両センタ支持ができない場合にはチャックによって固定する.トラバース研削においては,テーブルが反転するたびに切込みを与えるが,工作物端で振り切ってしまうと研削抵抗の変動により形状精度が低下する.このため,反転時に砥石幅の3分の2程度は接触させておく(b) テーパ面のトラバース研削(c) 段付き部分の端面加工を含むトラバース研削(a) 円筒面のトラバース研削(g) 総形アンギュラ研削(f) 円すい面のトラバース研削(e) 軸対称形状の総形プランジ研削(d) 円筒面のプランジ研削図6・20 円筒外面研削作業の方式(a) アンギュラ砥石(b) ストレート砥石図6・21 端面部の研削方法のがよい.円筒研削では,研削砥石と工作物の接触長さが短くなるため,研削熱による影響を受けにくく,また鏡面を比較的容易に創成できるが,細長い工作物は工作物の支持剛性が低くなり,円筒度に問題を生じやすい.このため,振れ止めを用いることがある.円柱状工作物の端面を研削するには,図6・21に示すように(a)アンギュラ砥石(angular wheel)による方法と(b)ストレート砥石(straight wheel)による方法があり,前者は,砥石台の送り方向によって二つに分けられる.一つは砥石の傾いた方向に送りをかけるアンギュラスライド型で,他方は砥石だけが傾いて,砥石台の送り方向は工作物に直角であるアンギュラヘッド型である.アンギュラスライド型では円筒部と端面部を同時に研削できる.なお,ストレート型砥石によって端面の研削を行う場合,砥石は必ずへこみ型とし,砥石のエッジで研削することが義務付けられている.
