# 6・2・2 内面研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

穴の内面を研削する作業を内面研削(internal grinding)という.内面研削盤には,工作物が回転する方式と工作物は回転せず砥石が遊星運動するもの(プラネタリ型)の2種類があり,前者が一般的であるが,精度を重視するジグ研削ではおもに後者が用いられる.図6・22は,工作物を(a)ロールや(b)シューで保持して研削する,心なし方式を示す.これらはおもに,ベアリング内,外輪のように,同心度が重視される場合に適用される.内面研削にもトラバース研削方式とプランジ研削方式があり,トラバース研削では深穴やテーパ穴の加工ができる.プランジ研削では種々の軸断面形状の穴を加工できる.内面研削では,砥石軸が細長くなる場合が多いため,軸のたわみにより円筒度が低下しやすい.このため,低切込み,高速回転で研削を行い,研削抵抗を低下させるのが望ましく,毎分数十万回転に及ぶものがある.この場合,砥石軸が振動を起こしやすいことから,共振点を避ける配慮が必要である.内面研削では砥石と工作物の接触弧長さが長くなり,切れ刃当たりの切込みが小さくなること(a) ロール使用の場合(b) シュー使用の場合図6・22 心なし方式による内面研削図6・23 心なし研削から,比較的粗い粒度(#46程度)の軟らかい(I~J 程度)砥石が常用される.ただし,小径の穴を高精度加工する場合には,粒度の細かい,結合度の高い砥石を用いる.最近では,砥石の損耗を避けるため,耐摩耗性に優れたcBN ホイールが用いられている.また,鉄系ボンドの超砥粒ホイールを用い,電解インプロセスドレッシング(ELID)研削を適用することによって,高精度,高能率の内面研削を可能にした例もある.
