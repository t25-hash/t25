# 6・2・3 心なし研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

心なし研削(centerless grinding)は,図6・23のように研削砥石と調整砥石(regulating wheel)の間に工作物を導入し,受け板(work rest)で支持しつつ,その外周を研削する方法である.工作物が砥石全幅にわたって支持されるため,細長い工作物の加工が可能である.また,工作物の脱着が容易で,自動化しやすく,加工能率も高い.調整砥石の軸心は,図6・24のように研削砥石の軸心に対し傾いており,工作物を一方向に押し付けたり送ったりするための力を発生する.工作物外周と調整砥石外周を工作物全長にわたって接触させるため,調整砥石は鼓形に成形しておく必要がある.心なし研削には,図6・25に示す(a)通し送り法,(b)送り込み法,(c)端送り法の3方式がある.図6・25(a)では,砥石幅以上の長さの工作物が連続的に加工できるので,長尺棒材などの外面研削に適用される.図6・25(b)では,段付き工作物の総形研削やテーパを有する工作物の加工が可能であるが,工作物の直径の異なる部分で調整砥石との間に滑りを生じる.図6・25(c)では,図に示すようなつばのある工作物の加工が可能である.3―167加工学・加工機器図6・24 調整砥石の形状(a) 通し送り法(c) 端送り法(b) 送り込み法図6・25 工作物の送り込み方式(a-1) 横軸テーブル往復型(a-2) 横軸テーブル回転型(b-2) 立て軸テーブル回転型(b-1) 立て軸テーブル往復型(c-1) 横軸テーブル往復型(c-2) 横軸両頭型図6・26 平面研削作業の方式心なし研削では研削抵抗に起因する工作物のたわみを生じにくいので比較的高精度な加工が可能であるが,円筒面の一部が欠けているなど,母線の長さが変化するような工作物の場合には真円度が低下しやすい.
