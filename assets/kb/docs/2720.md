# 6・2・4 平面研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

砥石車の外周面あるいは側面を使用し平面を仕上げる加工法を、一般に平面研削(surface grinding)と呼び、(1)砥石の外周面で加工を行うもの、(2)カップ形砥石(straight cup wheel)の端面で加工を行うものおよび(3)ディスク形砥石等の側面で加工を行うもの、の3種に分類できる。また、形状創成運動の形態によって、(1)往復運動をするものと(2)回転運動をするものに分けられる。図6・26はその概略図である。図6・26(a)ははん用的に使用されるが、研削砥石の作用面積が他に比べて小さいため加工能率は低い。能率を上げるために切込みを大きくすると、両者の干渉領域に研削熱が集中し、研削焼けが発3―168生しやすい。一方、円筒外面研削と比べると、砥石と工作物の接触長さが長くなり、切りくず厚さは薄くなる。このため、やや結合度の低い研削砥石を選ぶとともに、砥石と工作物の干渉領域を十分冷却することが重要になる。図6・26(b)では、カップ形砥石の端面を使用するので、砥石の作用面積が広く高能率な加工が可能で、大量生産に向いている。また、剛性の大きい機械構造とすることにより、高精度な仕上面を得ることができる。しかし、砥粒切れ刃の切削長さが大きくなり、研削油剤の加工域への浸入が難しくなることから、砥石中心部からの通液などによる研削油剤の十分な供給が必要である。図6・26(c)では、ディスク形砥石等の側面を使用するので、(b)と同様、加工能率に優れている。このうち向かい合った2枚の平形砥石の間に工作物を通過させる方法(c-2)では、1回の通し研削で寸法精度と形状精度の良い工作物を大量に生産できることから、転がり軸受のうち、外輪両側面の仕上げなどに利用される。さらに本方式は、チャッキングが不要で、加工ひずみや前加工面形状の影響を受けにくいことから、薄板の高精度な平面加工に有利であり、大型シリコンウェーハの加工などにも適用される。平面研削にも、プランジ研削とトラバース研削がある。図6・26(a-1)の場合、研削砥石の作業面をあらかじめ成形しておけば、総形加工が可能である。最近のcNC横軸角テーブル型平面研削盤の場合、研削砥石のドレッシング、研削手順、定寸などのサイクルが自動化されたものが多い。また、高速反転が可能な研削盤も開発されており、その性能を生かすために研削砥石の回転数も高くなる傾向がある。この場合、高速回転に適した、超砥粒ホイールを用い、研削砥石のバランスを高精度に調整するとともに、研削点を十分冷却することが肝要である。さらに最近では、非軸対象非球面ミラーなど加工面形状の複雑化、高精度化に伴い、超精密CNC三次元研削盤を用いた自由曲面加工、加工面形状の機上計測とこれに基づく補正加工などが試みられている。
