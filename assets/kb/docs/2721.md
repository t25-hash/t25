# 6・2・5 歯車研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

歯車研削(gear grinding)の基本方式としては、図6・27に示すように、総形成形した砥石により研削する方式と、皿形砥石(dishwheel)によって成形研削する方式がある。前者には、図6・27(a)のように成形した砥石によって歯溝を一つずつ研削する方法と、図6・27(b)のようにウォーム状に成形した砥石と工作物を高精度に同期回転させながら連続研削するホブ盤に類似した方式がある。いずれの場合も研削砥石の成形に困難を伴うが、前者は機械構成が簡単で、非インボリュート歯形をもつ歯車の研削に使用されることが多い。また、内歯車の研削にも使用される。後者は、非常に高能率で、特に小型歯車では素材から直接仕上げることが可能である。図6・27(c)に示す方法は、2枚の皿形砥石の研削面を仮想のラック面として、これに工作物を転動させながら接触させることで、歯車を創成するもので、高精度な歯形の創成が可能である。この方法で、はすば歯車を創成することも可能であるが、生産性において図6・27(b)の方式に劣る。なお、機械の低騒音化、高速化などに対応するため、焼入れした素材の高品位加工が必要となる場合がある。このようなときには、あらかじめ高精度に成形した電着cBN ホイールを用いるのが有利(a) 歯車:緩やかな転動図6・27 歯車研削の方式である。これは、砥石の摩耗が少ないことと、cBN の熱伝導率が一般砥粒砥石よりも大きいことから工作物への熱流入が少なく、かつ研削面に圧縮残留応力が生じやすいことによる。
