# 3・6・10 フック形自在軸継手（Hooke’s universal joint ）

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

2軸が同一平面上にあり，かつその中心線がある角度で交わる場
合の伝動装置で，カルダン形軸継手（Cardan’s joint ）とも呼ばれ
る．図Ⅰ-3･98(ａ)に示すように原動軸および従動軸の回転角をθ
およびφとし，その角速度をθおよびφとすれば，これらの関係
は次式によって与えられる．
ここで，αは2軸の交角である．したがって軸が1/4回転する間に
速比はcosαより1/ cosαに変動する不等速自在継手であって，伝
達されるトルクも速比に反比例して変化する．
間軸を用い，その両側に軸継手を取付けて用いる．
金具からなり，この間にニードル軸受または軸受ブシュを挿入し，
これをグリース潤滑するようになっている．軸の交角はα＝45ま
でとれるが，動力伝達用では回転速度が≒33s のとき，α≦20
とする．一般に回転速度が大きいときは交角を小さく，回転速度が
小さいときは交角を大きくとることができる．図Ⅰ-3･99のような
構造の自在軸継手は比較的大きなトルクを伝えることができる．
