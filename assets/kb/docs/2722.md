# 3・6・11 こま形自在軸継手（universal ball joint ）

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

工作機械や産業機械によく使われる不等速形軸継手で，構造はフ
ック形自在軸継手と本質的には変わらない．JIS B1454：1988に
構造，寸法が定められており，図Ⅰ-3･100に示すように，本体と
つめ，さやおよびこまからなるA形と，本体とこまおよびピンか
らなるB形，およびそれらを複式にしたAA形，BB形がある．
動力伝達用としては2軸の交角は普通30ぐらいまでであって，例
えば45で使用するような場合には伝達トルクを十分に小さくと
り，また回転数も低くする．表Ⅰ-3･39はJIS B1454に規定され
たA形，AA形の主要寸法と性能である．
なお，こま形自在軸継手と同じ形状で，ヨークで直接ボールを抱
く構造をしたボール形の不等速形軸継手もある．
