# 6・2・6 ねじ研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

ねじは切削によって創成されるものがほとんどであるが,高精度の送りねじやねじ状工具などは研削によって仕上げられる.ねじ研削(thread grinding)には,一山ねじ研削砥石または多山ねじ研削砥石が使用される.前者では,工作物のねじ山形状に一致させた砥石をねじのリード角と同じ角度傾け,回転するねじの進みに応じた送りでトラバース研削を行う.ねじの精度には,研削盤の運動精度,砥石の成形精度と加工中における摩耗などが影響する.このため,cBN ホイールの使用が効果的である.リード角が5°程度より大きいねじの場合は,砥石の干渉によりねじ山の先端と基部にアンダカットを起こすので,砥石の輪郭を修正する必要がある.一方,多山ねじ研削では,ダイヤモンドロールなどで成形した多山形状の砥石を用い,高能率にねじ山を創成することが可能である.しかし,成形された砥石のねじ山形状の誤差が,そのまま工作物に転写されるため,ロールには高い形状精度が求められる.
