# 6・2・7 工具研削(tool grinding)

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

バイト,ドリル,エンドミル,ホブなどの工具にはいずれも高硬度材料が用いられることから,その創成は研削に頼らざるを得ない.さらに,工具の形状は複雑かつ多様であることから,使用される工具研削盤も多様で,専用機も多い.例えば,複雑な形状をしたエンドミルなどの研削加工のために,図6・28に示すような多軸の万能工具研削盤(universal tool grinder)が使用されている.CNC装置を有する工具研削盤では,工具形状の創成運動をシミュレーションすることが可能で,創成後の工具断面形状を簡単にチェックできる.しかし,形状の創成運動と砥石形状の組合せの最適化については,今後のシミュレーション技術の進展を待たなければならない.工具研削においては,所望の形状を高精度,高能率に創成すると同時に,工具材料本来の強度を失わせない配慮が重要である.鉄系図6・28 万能工具研削盤(b) 創成運動と割出しを同時に行う(c) 歯車:速い転動をしながら緩やかに軸方向に移動合金工具鋼の加工にはcBN ホイールを,非鉄系および非金属系工具の加工には,ダイヤモンドホイールを使用するのがよい.いずれも熱伝導率が高く,工作物への熱流入を抑制する効果がある.工具の刃先が鋭利になるに従って刃先近傍で研削焼けが発生しやすくなり,硬度の低下,引張残留応力の発生など,工具寿命を低下させる原因となる.さらに,脆性材料の場合には,切れ刃稜に欠けを発生させやすい.このため,微粒の超砥粒ホイールを用い,極力振動や衝撃を抑えた状態で,微小切込みの研削を繰返す必要がある.なお,導電性工具材料の場合には,電解加工を併用した電解研削(ECG : electro-chemical grinding)が適用できる.この場合,難研削材の加工能率を格段に向上させることができ,工具へのダメージも抑制できる.
