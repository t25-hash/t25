# 6・2・8 その他の研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

研削盤(工作機械)では工作物-砥石(工具)間の相対運動の軌跡により,研削加工される形状が定まる.そして,回転運動と直進運動が工作機械の基本運動であることから,平形砥石による円筒の内・外周,および平面が研削における主たる加工対象となる形状といえる.このことは,既述された円筒および平面加工用の各種研削盤が標準的な研削盤機種として,はん用されている理由である.一方,円筒,平面以外の複雑な形体を研削加工する目的で設計されている研削盤の機種も多彩である.その代表的なものを以下に示しているが,これらはどちらかというと専用,あるいは特殊な研削盤と位置付けられる.歯車研削盤,ねじ研削盤,工具研削盤,センタ穴研削盤,ジグ研削盤,倣い(プロファイル)研削盤,カム研削盤,スライドウェイ研削盤,クランクシャフト研削盤,グラインディングセンタ,など上に列挙する中で歯車研削盤,ねじ研削盤および工具研削盤の3機種については6・2・7項までにすでに紹介済みである.そこで本項ではそれ以外のいくつかを取り上げて,説明を加えることにする.本項で取り上げる研削盤の機種はいずれも,円筒,平面以外の曲面や複雑な形体の研削を目的としたものであり,その機能は次のような各種の方式によって実現されている.(1) 砥石台の運動制御による方式メカニカル方式:クランクシャフト研削盤テンプレートやマスタによる倣い方式:カム研削盤,倣い研削盤NC方式:ジグ研削盤,グラインディングセンタ(2) 砥石形状を転写する総形研削方式センタ穴研削盤,スライドウェイ研削盤内燃機関用のクランクシャフトやカムを研削する専用機がクランクシャフト研削盤(crank shaft grinder)とカム研削盤(camgrinder)である.クランクシャフトやカムシャフトの回転に連動して砥石台が加工面に追従し,研削加工を行う機構となっている.例えば,カム研削盤ではカムシャフトと同軸のマスタカムにフォロアが接触追従することによって砥石台が移動し,カムプロファイルを創成研削する(図6・29).3―169加工学・加工機器センタ穴研削盤は,特に高精度の要求される円筒研削用工作物のセンタ穴を,その穴形状に成形された軸付き砥石によって,精密仕上加工するものである.近年,NC制御装置の機能が大幅に向上し,加えてcBN 砥石やダイヤモンド砥石の進歩,普及によって,CNC研削盤の利点を十二分に発揮できる環境が整いつつある.NC研削盤の運動精度も0.1 m/pulseが容易に実現できるようになってきており,例えば,ジグ研削盤(jig grinder)は金型などの精密研削にこのNC研削の特徴をフルに発揮している.さらに,このようなNC技術の進歩はメカニカル方式や倣い方式のNC化への転換を促進させ,クランクシャフト研削盤やカム研削盤なども今日では,メカニカル方式や倣い方式をNC方式が代替するのが一般化している.研削盤のNC化を象徴するのが,切削におけるマシニングセンタに対応する,砥石によるグラインディングセンタ(grinding center)の発展である(図6・30).このように特殊および専用研削盤の世界でも今日ではNC制御方式が主流となり,これに伴いそのNCソフトウェアの作成が重要な技術課題となっている.
