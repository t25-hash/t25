# 6・2・9 ホーニング(honing)

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

ホーニングとは、スティック状(角形棒状)のホーニング砥石を設定圧力で加工面に押し付け、回転運動と軸方向への往復しゅう動を連動させ、精密中ぐり、研削などによって前加工された円筒内外面、時には平面を高い能率で仕上げる固定砥粒による仕上研磨法である。その加工原理の要点は以下のとおりである。a. 圧力切込み研削方式とホーニング盤一般の研削盤は設定(強制)切込みである。そこでは、研削盤の運動精度が寸法・幾何的加工精度として工作物に転写される、いわゆる母性原則に支配されることを特徴としている。これに対し、ホーニング、超仕上げ(マイクロホーニング)、ハンド工具を利用したロボットによる自動磨き装置などで採用されているのが圧力切込み(定圧切込み)方式である。圧力切込みとは、砥石をばね、油・空圧などにより適切な荷重で比較的広い研削面に押し付けて、両者の間に回転運動と往復しゅう動運動を与えながら湿式で研削する加工方式である。ここでは、工作物と砥石の接触面において、実接触する加工面の凸部から順次、優先して除去加工が進行する、いわゆる選択原理が支配している。圧力支持機構(floating 機構など)が砥石に加わる衝撃を吸収し、研削点の過剰負荷から逃れることができるため、角形棒状(スティック)砥石のような広い研削作用面をもつ砥石の適用を可能としている。通常の研削に比べるとはるかに低速な研削速度(10~100m/min)のもとでの加工面へのテクスチャ創成、自由曲面の磨きなど仕上研磨加工にとって、この圧力切込み方式は有効である。しかし反面、取りしろの除去が圧力による間接的な管理となることから、前加工形状に倣ったり、うねりが形成されやすいなど、形状・寸法精度の確保にとって有利であるとは決していいがたい。b. ホーニング盤の諸形式とホーニング砥石ホーニング盤とは、ホーニングヘッド(ストーンとも呼ばれるスティック状のホー図6・29カムプロファイルの研削図6・30 グラインディングセンタ3―170ニング砥石3~24本程度をヘッド外周に配置し、ばね加圧機構あるいは油・空圧拡張機構、ねじ拡張機構などを介して所定の圧力で加工面に押し付ける研磨機構部、ホーンとも呼ばれる)に円筒面に倣っての回転運動(スティックの回転周速度욣:数m/minから数十m/min)を与えると同時に往復しゅう動運動[速度: (1/3~1/2)욣]を連動させ、主として穴の内面を平滑に仕上げる研磨機械である。円筒の外周面や平面の加工にも適用できる。機種は立て型(図6・31)と横型に分類される。加工寸法精度は、各種の自動定寸方式によって管理できる。この定寸精度をより確実に保証するため、砥石の拡張量によって切込み量を設定する強制切込み方式が採用されることもある。ホーニング特性を支配する主要なホーニング条件には、ホーニング圧力とクロスハッチ交差角2θ(θ=tan/욣)が挙げられる(図6・32)。ホーニング圧力としては50~300N/cm(5~30×10Pa)あたりが目安であり、傾斜角θは20°~80°である。研磨圧力と交差角を大きく選ぶほど加工能率が上がる反面、仕上面粗さは粗くなる(粗仕上げ:θ=40°~60°)。したがって、仕上工程に近づくほど、ホーニング圧力と交差角を小さくする(精密仕上げ:θ=20°~40°)。目づまり防止の観点から、比較的大きな気孔率(50%前後)の砥石が賞用される。また、研磨液としては冷却性と洗浄性を重視して、ケロシンや鉱油を基油と図6・31 ホーニング盤図6・32 ホーニング運動とクロスハッチテクスチャし、各種の油や添加剤が加えられたものが工夫されている。c. 加工精度と加工面性状ホーニング砥石と工作物は面接触状態にあり、圧力切込み方式なので工作物の前加工面に多少の凹凸があっても無理なくなじみ、研削が進行する。加工による発生熱が少なく、平滑な粗さ(粗仕上げ:数十m~精密仕上げ:数m)に能率よく仕上げることができるが、前加工面の形状やうねりになじみやすいため、それらの修正作用に難点がある。したがって、形状精度(真円度、円筒度、真直度など)の確保を容易にするためには、その前加工にかなり高い加工精度が望まれ、研削や精密中ぐり工程などが適用される。ホーニングヘッドの回転と軸方向の往復しゅう動運動の連成によって形成されるクロスハッチ(交差)パターンは、オイルポケットの機能を果たすため、ホーニング仕上面は機械しゅう動面として好適である。鉛やバビットメタルのような軟質な軸受メタルからガラスのように硬脆な無機材料の仕上研磨まで、広く適用されている。d. 平面ホーニングによる固定砥粒ラッピングへの展開ラッププレートを固定砥粒砥石(ディスク形砥石)によって代替することによってラップ剤を水に代えることを可能とした、固定砥粒ラッピングと呼ばれているこの研磨法(ラッププレート平面の回転運動と工作物の往復あるいは回転運動の連成による平面ホーニング加工)の実用化が、長年繰返し試みられてきた歴史がある。この研磨技術の魅力は、研磨工程の自動化、作業環境のクリーン化であり、特に半導体の加工現場からのニーズが高い。光ファイバフェルール端面の仕上研磨など、ある種の加工物に対して実用されている。しかし、遊離砥粒による仕上ラッピングのレベルに比肩できるまで、固定砥粒ラッピングの加工面品質が至っていないのが現状である。この加工における最大の障壁は、スクラッチの発生である。この打開への鍵は、優れた固定砥粒ラッピング専用砥石の開発が握っている。
