# 3・6・4 オールダム軸継手（Oldham’s coupling）

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

できる軸継手で，図Ⅰ-3･92に示すように回転中に自由にずれるこ
とができるフローティングカムを中間板として挿入した構造になっ
ている．カムの突出し部は回転中に本体の溝の中を圧力を受けなが
ら滑るので，潤滑に注意し，許容面圧は20MPa以下で設計され
る．
示すように，外筒にある内歯車と内筒にある外歯車をかみあわせて
トルクの伝達を行う．その歯形はインボリュートで圧力角20 °が多
い．このような構造で軸心の狂いを許すことができるのは，外歯の
歯先と歯面にクラウニングを施してあるためで，外歯は内歯に対し
