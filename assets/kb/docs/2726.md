# 6・2・10 超仕上げ(superfinishing)

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

ホーニングに比べ往復運動の小さな振幅(1~4mm程度)と高いサイクル数(200~3000c/min)のもとで、結合度の軟らかい細粒度(粒度280~1500番)砥石を低圧(0.5~3×10Pa)で押し付け、高能率(数分のサイクルタイム)に円筒状工作物の外周の鏡面(sub- mの表面粗さ)生成によって耐摩耗性・疲労強度などの改善を目指す量産仕上研磨法が超仕上げである。円筒外面の超仕上加工の模式図を図6・33に示している。工作物表面に描く砥粒の運動軌跡は正弦波となり、その最大傾斜角θが重要な加工のパラメータとなる。この傾斜角は普通30°~60°である。超仕上加工は、超仕上ユニットを旋盤の刃物台などに取付けて使用できるユニット型のもので手軽に行えるが、もちろん形状(円筒外面:クランク軸やクランクピン、穴内面:転がり軸受の走路面、平面:顕微鏡用試料)に応じて各種はん用機、専用機も製造されている。一般に超仕上砥石の振動は、モータによって等速回転される偏心ダボなどメカ図6・33 超仕上加工の原理ニカルな機構によって与えられる。砥石圧力や研磨速度はホーニングにほぼ準じている。超仕上げには数m以下の前加工面粗さが望ましいとされている。その研磨機構の本質はホーニングのそれと同じであることから、マイクロホーニングと呼ばれる。砥粒の自生サイクルを管理するためには、砥石のボンド強度と砥粒の破壊限界圧力(臨界圧力)を踏まえて加工条件を設定することが肝要である。例えば、意識的に目づまりを生じさせて効率よく鏡面仕上げを達成するため、一般にはビトリファイド砥石が用いられるが、小物部品や軟質金属には当たりが柔らかいことから、レジノイドボンドやラバーボンドなどの細粒度砥石が賞用されている。ホーニングと同様、工作物の前加工形状になじみやすいことから、形状精度の修正、改善能力はあまり期待できないが、表面の磨き加工能率に優れている。また、その仕上面粗さには方向性が見られないこと、加工変質層が軽微であることなども、この加工法の特徴といえる。
