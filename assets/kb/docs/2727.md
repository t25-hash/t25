# 6・2・11 研磨布紙加工

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

a. ベルト研削(belt grinding)技術の特徴固定砥粒タイプの研磨工具は,bonded abrasivesとcoated abrasivesに大別される.この両者の代表として前者には砥石があり,後者には研磨布紙がある.研磨布紙の中では,サンドペーパが昔からなじみ深いが,1960年代ごろからクロス(綿布)基材のcoated abrasivesをベルト状に裁断・加工した研磨ベルトを回転駆動するベルト研削加工技術が発展してきた.今日では,coated abrasivesのタイプも多彩になり,基材にポリエステルなどのフィルム素材を用いた研磨フィルムなどは研磨布紙をしのぐ勢いである.したがって,広義には,研磨布紙は各種材質の基材に砥粒を塗布した研磨工具(coated abra-sives)すべてを包含するものとして取扱われている.ベルト研削にはおのずと,その工具である研磨布紙(coatedabrasives)の以下の特徴が強く反映される.(1) 柔軟性に富む.(2) 過負荷や破断に際して,機械の破損や作業者のけがに対する安全度が高い.(3) 工具寸法や形状(シート,ディスク,ベルト,ロールほか)が多彩で,その選択の自由度が大きい.(4) 各種金属,ステンレス,アルミニウムなどの難削材などはもとより,木材,プラスチックなど多様な材質に対しても適用できる.(5) 切れ刃の再生(ドレッシング)を行わないスローアウェイ型の砥粒工具である.(6) 研磨特性の経時変化が大きい.(7) 加工コストに占める工具コストが砥石に比べ高い.(8) ベルトに接合部をもち,かつ弾性工具であることから,形状精度や仕上面精度には限界がある.今日,ベルト研削やテープ研磨は,(1) 重研削:バリ取り,圧延工程のきず取りやディスケーリング(2) 曲面研削:クランクピン研削,タービンブレードの研磨(3) 自動研削加工:磁気ヘッド研磨(4) 表面テクスチャの付与:プリント基板の荒らし研削,建築金物の仕上研磨などの分野で,重用されている.b.
