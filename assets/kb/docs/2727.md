# 3・6・5 歯車形軸継手（gear coupling）

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

いわゆるギヤカップリングと呼ばれているもので，図Ⅰ-3･93に
て傾きを与えることができる．外筒の中心線に対して内筒の中心線
の傾き角を傾斜角といい，その角度は一般に1.5 °以下のものが多
いが，特殊な要求によっては8 °程度まで傾くことができるものも
ある．したがってこの軸継手では歯面の潤滑のために良質なグリー
スあるいは潤滑油を封入しておく必要があり，油漏れに注意し，密
封装置は確実でなければならない．継手外径の周速度40m/s以
下，回転速度66.67s以下のものについてはJIS B1453：1988
に構造，寸法，材質などが決められており，表Ⅰ-3･35は規格に定
められた寸法の歯車形軸継手の性能の一部である．
