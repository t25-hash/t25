# 3・6・6 ローラチェーン軸継手（double roller chain coupling）

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

チェーンカップリングとも呼ばれ，図Ⅰ-3･94に示すように，ス
プロケットの付いた継手本体を二列ローラチェーンで結合し回転力
を伝える．この軸継手はスプロケットの歯とローラ間の遊び，ロー
ラとブシュまたはブシュとピンのすきまおよびチェーンのたわみな
どによって，軸心のわずかな狂いを吸収することができる．性能は
フランジ形たわみ軸継手よりやや劣っているが，外径が比較的小さ
いという特長がある．しかし，歯面とローラ，またはローラ，ブシ
ュおよびピン間の摩耗を避けるために潤滑しなければならない．低
速のときはグリースを塗布すればよいが，高速になると飛散するの
