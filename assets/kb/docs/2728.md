# 6・3・1 ラッピングとポリシング

> **章**: 第6章 砥粒加工
> **節**: 6・3 遊離砥粒加工

遊離砥粒加工(研磨)には、粗面加工と鏡面加工があり、前者をラッピング、後者をポリシングと称する。さらに、通常のポリシング以上の鏡面加工を施す超精密加工は、これらの中でも最も優れた精度や品質の仕上げが可能な加工法に位置付けられている。実際に超精密部品を製作する場合、最初から超精密加工を適用することはなく、例えば、前加工のラッピングを施してからポリシングもしくは、超精密ポリシングを経る加工プロセスが採用される。ラッピングやポリシングでは、双方とも研磨剤を介して加工物と工具をすり合わせることにより加工が進む。表6・4にラッピングとポリシングの相異を整理して示す。a. ラッピング(lapping)ラッピングでは、砥粒の機械的作用で加工物やラップ(一般的には鋳鉄)に、凹凸や割れが形成されており、砥粒はその状態の表面に対して転動や引っかきの挙動をとる。図6・36は、硬脆材料(図中の下面)ならびに金属材料(図中の上面)の加工モデルである。硬脆材料では、砥粒の転動や引っかきなどで生じる割れの交差が切りくず生成を伴う。砥粒の供給量が不足なく均一にラップ工具上に供給されていれば、単位時間当たりの砥粒の作用数を増大させるために加工物とラップ工具の相対速度を大きくし、加工圧力を大きくすることなどによって、加工能率を増大することができる。小さな表面粗さに仕上げる場合には、細かい砥粒を使用することが基本であるが、前述の相対速度と加工圧力などを抑える加工条件を選択するのが有効となる。この式中の比例定数は、加表6・4ラッピングとポリシングの相異砥粒の寸法大(数十mから数m)微小(数m以下)研磨の工具硬質ラップ(鋳鉄)軟質ポリシャ砥粒の保持法転動ポリシャによる弾塑性的保持加工面状態なしじ面鏡面主目的形状精度の確保平滑鏡面化工物の性質、砥粒の種類・大きさ、ラップ工具の性質などに関連する。他方、金属のような割れを伴わない延性材料では、転動している砥粒の押付けで弾性変形や塑性変形が生じる。換言すれば、破砕を伴わないということになり、切削加工のように砥粒による微小引っかき作用が生じ、切りくずを生成する。この点、ポリシングのメカニズムに類似するともいえる。したがって、金属材料のラッピングでは、前述の硬脆材料と異なって、切りくず生成に結びつく破砕がないことが加工量に大きな影響を及ぼす。すなわち、硬脆材料の場合に加工量や表面粗さが砥粒の大きさにほぼ比例して増加するのに対し、延性材料になると砥粒の大きさが同様に表面粗さに影響するが、加工量に対してはほとんど影響しない。表層部分に生じる加工変質層については、硬脆材料や金属材料をラップした場合、概して表面粗さの7~15倍であり、また使用した砥粒の平均粒径に近い値の深さになるといわれている。表6・5は、ラッピングにおける表面性状、加工能率、加工精度、経済性などに対する加工要因である。b. ポリシング(polishing)機能性材料のポリシングにおける特徴・目的は、材料をいかに加工変質のない表面に創成するか、いかに能率よく表面加工するか、ということに帰着する。一方、レンズ、プリズムをはじめとする光学部品のように、所定の形状精度を確保することを第1目的にする場合もある。一般的にポリシングは、1 m以下の微細な砥粒とピッチ、ワックス、合成樹脂、人工皮革(artificial leather)などの軟質なポリシャ(polisher)が使用され、平滑な鏡面仕上げを得ようとする。図6・36 ラッピングの加工モデルラッピングポリシング表6・5 ラッピングの加工要因・条件の一例種類金属酸化物・炭化物・ホウ化物・ダイヤモンド等砥粒性質形状、硬さ、靱性粒径1~30 m種類鋳鉄、工具鋼、ガラス、セラミックス等性質硬さ、不純物ラップ(工具) ポリシングの加工モデル微細な砥粒がポリシャに弾塑性的に支持されて、被加工物表面を微小引っかきする加工メカニズムが基本であるので、その作用はラッピングに比べてはるかに軽微となり、脆性材料でも割れを生じない鏡面になるのである。ポリシングにおける材料除去は、ラッピングに比べると微細な砥粒が軟質なポリシャ[研磨布(polishing pad)、パッドともいう]とともに用いられるので、図6・37の加工モデルのように、砥粒は加工物のごく表層の小規模な領域に作用する。例えば、ガラスのポリシングでは、0.1 m前後の酸化セリウム(cerium oxide,CeO)、ベンガラ(red rouge,FeO)、あるいはシリカ(silica,SiO)など比較的軟質・微細な金属酸化物砥粒とピッチ、ワックス、合成樹脂、人工皮革などの軟質工具のポリシャが使用される。砥粒はポリシャに弾塑性的に抱え込まれ、極めて軽微な引っかきを行う。ただし、例えばガラスの研磨における切りくずの実態は、湿式条件であるので水和現象により確認できないのが普通である。ポリシングのメカニズムについては、古くから、前加工のラッピングで形成された凹凸がポリシングの進行に伴い平滑な鏡面になること、その鏡面部分がエッチングで再び前加工のラップした面の凹凸状態に戻ること、などさまざまな実験事実があり、(1)砥粒による機械的な材料除去、(2)切りくずなどの凹凸部への付着、(3)表面流動、(4)化学溶去、などの諸説が議論されてきた経緯がある。現在では、これらのすべてが共存した複合作用によると考えられている。もちろん、これらの配分は加工物、砥粒、ポリシャ、加工液などの組合せによって異なり、時にはエッチング効果のある溶液を加工液に選んでケミカルメカニカルポリシング(CMP: chemicalmechanical polishing)を可能にする場合もある。ポリシングによる加工変質層(damaged layer)の発生原因には、まず、切れ刃工具に相当する砥粒による機械的作用(微小切削作用)と、ポリシャによる摩擦作用の影響がある。また、加工室の環境あるいはポリシ剤中に混入したじんあい・異物などによる機械的作用に起因する加工欠陥が原因となることもよく経験されている。したがって最近の加工室は清浄度のよく管理されたものが一般的になっている。加工変質層については、その極小化を目指してポリシング技術の3―173加工学・加工機器高度化が進められている。最近では、加工の単位を小さくするポリシング方法・条件を導入し、対象材料によって異なるものの、今や加工変質層深さの測定における検出感度以下に近い表面が得られるまでに至っている。c. 複合研磨加工超精密ポリシングは多くの方法が提案され、いろいろな作用・複合化させた方法が導入されてきた。従来からガラスの高精度研磨では、加工条件の改善により、精度のみでなく無擾乱化も可能となってきている。それとともに、機械的作用に化学的作用やその他の作用を複合化させた種々の加工法が実用化され、対象とする材料の要求条件に応じて適用されている。半導体結晶の湿式メカノケミカルポリシング(MCP: mechanochemical polish-ing)はその典型的な例である。ここでは、加工変質層のない無ひずみの無擾乱鏡面を得る、各種の超精密複合加工法について概述する。超精密ポリシングでは、表面の結晶格子の配列を乱すことなく原子オーダの平滑さに鏡面仕上げを行うことが基本である。必然的に加工の単位を原子もしくは分子オーダにし、被加工物表面の極数原子層を除去することになる。特に、物理的に除去する場合、ひずみが伝ぱする領域を極小化し取り去る原子もしくは分子近傍内に限定しなければ、無ひずみ無擾乱鏡面の創成は困難となる。被加工物表面の除去加工の方法には、力学的/機械的エネルギー、熱エネルギー、電気エネルギー等の物理的作用、ならびに化学的作用を利用するものがある。しかし、このような作用を単独で用いるよりも、条件をうまく設定することができれば、物理的作用と化学的作用を複合化させたほうが加工能率・加工面品位などの点で有利となる。図6・38は現在の物理的(機械的)・化学的複合加工法の位置付けを図式化したものであるが、複合ポリシング、特に湿式メカノケミカルポリシングMCP/ケミカルメカニカルポリシングCMP(ここではCMPとして同義に扱う)に注目する。ポリシングにおける材料除去は、究極的には機械的作用と化学的作用によって加工が進むことになる。前者の機械的作用には除去作用と摩擦作用があり、他方の化学的作用には溶去作用と反応生成物・被膜形成作用が考えられる。これらの作用を複合化させると、さまざまな特徴ある加工条件が設定できたり、新奇な超精密ポリシング法を見いだすことも可能となる。表6・6は、代表的な超精密ポリシング法/複合研磨法とその加工メカニズムを考慮したおもな加工要因などをまとめたものである。まず、砥粒による機械的除去作用(機械的ポリシング)に、砥粒もしくは加工液による化学作用を付加させた化学的複合ポリシング、すなわちケミカルメカニカルポリシング(CMP)/メカノケミカルポリシングについて整理する。メカノケミストリーに関する考え方を参考にすると、メカノケミカル現象(mechanochemicaleffect)としては次のように定義することができる。(1) 物質に加えた機械的エネルギー(例えば衝撃、粉砕、圧延、引張り、加圧、塑性変形など)によって、固体表面が物理化学的性質の変化をきたす。(2) 固体物質の周囲に存在する気体や液体物質に化学的変化をもたらす。もしくは、(3)
