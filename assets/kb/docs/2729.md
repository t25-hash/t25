# 6・3・2 バレル加工

> **章**: 第6章 砥粒加工
> **節**: 6・3 遊離砥粒加工

バレル加工(barrel finishing)は,図6・40のようにバレル(barrel)と呼ばれる加工槽内に工作物,遊離研磨材としてのメディア(media)および必要とあれば加工液(コンパウンド水溶液)を挿入し,バレル全体に回転運動や振動を与えることによって工作物とメディアの混合物(マス)に流動を生じさせ,両者の相対運動によって誘起される研磨作用を利用して工作物を加工する方法である.バレル加工は,マスフィニッシング(mass finishing)と呼ばれるように量産加工に適したもので,次のような特徴がある.(1) スケール除去,表面仕上げ,バリ取り,エッジ仕上げ,光沢仕上げ等の複数の加工工程が一度にでき,省人化が図れる.(2) 多量の工作物をバッチ処理することができ,生産性に優れる.(3) 工作物の寸法・形状や材質を選ばず,ばらつきのない均一な仕上面品質が得られる.(4) 作業者に熟練度を必要としない.a. バレル加工方式バレル加工方式は,開発初期の共ずり法(self tumbling)と呼ばれた回転バレルが主であったが,図6・41図6・39化学的機械的複合加工における一般的加工特性(*:全加工作用=化学的作用+機械的作用)図6・40 バレル加工の概念図3―175加工学・加工機器に示すように加工目的に適した種々の加工方式が開発されている.回転バレルは構造が単純で操作やメンテナンスが容易で価格が安いが,加工時間が長い.振動バレルは設置面積が小さく,加工中に工作物のチェックが可能であるが,振動音が問題になる.遠心バレルは多品種少量の小型工作物に適しており,加工時間が短いが,工作物表面に打痕や変形を生じやすい.流動バレルは振動バレルと遠心バレルの中間的な特徴をもつが,ライニングの消耗が大きい.ジャイロバレルおよびレシプロバレルは工作物をジグに固定するために打痕の発生はないが,多量生産には適さない.磁気バレルはバリの大きさに応じて磁界強さをコントロールすることができ,洗浄工程と同時に実施できる.b. バレルメディアバレルメディア(barrel media)は,媒介物・中間の意味をもつmediumの複数形であり,文字どおり工作物間の衝突を防止するとともに研磨作用を与えるものである.バレルメディアには,成形メディア,金属メディア,有機質メディア,プラスチックメディア,乾式研磨用メディアがある.成形メディアは,焼成メディアと焼結メディアに分けられ,前者はアルミナ砥粒などをビトリファイド結合剤によって焼成したもので研磨作用に優れ,後者はアルミナ微粉末のみを高温焼結したもので良好な仕上面が得られる.金属メディアはその硬さによって加工目的が異なり,工作物より硬質のメディアはバニシング作用による光沢仕上げに,工作物より軟質のメディアは微粉研磨材とともに研磨仕上げに用いられる.有機質メディアは,竹,クルミ,木,コーンなどを粉砕した破片に油脂とともにアルミナや酸化クロムをコーティングしたものであり,樹脂製品などの光沢仕上げに使用される.プラスチックメディアは合成樹脂を結合剤として砥粒とともに成形したもので,軟質金属の研磨仕上げに用いられ,乾式研磨用メディアもこの種類に属するが,研磨能力に優れるためにあらゆる金属材料の乾式バレル研磨に用いられる.
