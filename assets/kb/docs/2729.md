# 3・6・7 ゴム軸継手（rubber shaft coupling）

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

ゴムを多量に使用し，ゴムの圧縮やせん断などの弾性効果を利用
した軸継手をいう．形状は図Ⅰ-3･95(ａ)の星形ゴム軸継手や(ｂ)
のタイヤ形ゴム軸継手など種々の形状のものがあり，潤滑が不要な
こと，運転中騒音を発しないこと，2軸の電気絶縁が可能なことな
どの利点もあるが，構造上外径寸法が大きくなること，同一軸穴直
径を有するフランジ形たわみ軸継手に比較して許容伝達トルクや許
容回転数が約2分の1に低下するなどの欠点もある．比較的大きい
軸心の狂いがある場合や振動を吸収するなどの場合に用いられてい
る．JIS B1455：1988ではゴム軸継手を軸心の狂いの許容値によ
ってE1，E2，E4の3種類に，またはねじりばね定数の許容値に
よってK2，K4，K8，K16の4種類に分けている．表Ⅰ-3･37
はその主要寸法と性能の例である．なおここで，許容トルクは一方
向に定常速度で長時間にわたって伝達することができるトルクのこ
とであって，短時間であれば最大伝達トルクは許容伝達トルクの約
2倍の大きさまでとれる．
