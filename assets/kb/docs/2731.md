# 6・3・4 超音波加工

> **章**: 第6章 砥粒加工
> **節**: 6・3 遊離砥粒加工

超音波加工(ultrasonic machining)は,一般的に図6・42に示すように,10~150 mの全振幅で15~30kHz の周波数で一方向にのみ振動する工具を,砥粒を混入分散させたスラリー状加工液を介して,工作物に一定の圧力で押し付けて穴あけや切断などを行う加工法である.超音波加工は,砥粒を介した工具の打撃作用,回転バレル水平式回転バレル可傾式回転バレル振動バレルサークル型振動バレルボックス型振動バレル遠心バレル通常型遠心バレルディスク型遠心バレルバレル加工流動バレル半自動型流動バレル全自動型流動バレルジャイロバレルレシプロバレル磁気バレル図6・41 バレル加工方式3―176表6・7 バフ車の構成材料と適用研磨工程材質バフ構造・形状適用研磨工程あや織綿布鉄バフ粗研磨サイザル麻布オープンサイザルバフ粗研磨・中研磨レーヨン布バイアスバフ中研磨平織綿布バイアスバフ中研磨・仕上研磨キャラコばらバフ,縫いバフブロードばらバフ,縫いバフネルバイアスバフ,ばらバフ仕上研磨フェルトバフ,軸付きバフ牛皮バフ図6・42 超音波加工の概念図加工液のキャビテーションによる浸食作用およびその気泡によって加速された砥粒の衝突作用が相乗して進行する.工具には軟鋼,焼入れ鋼,ピアノ線などの金属材料が用いられ,適用される工作物としてはガラス,セラミックス,半導体,ダイヤモンド,カーボンなどの脆性材料,超硬合金や表面硬化処理鋼などの硬質金属材料,ステンレス鋼などの難削材料がある.加工液に混入される砥粒としては,炭化ケイ素,アルミナ,炭化ホウ素,ダイヤモンドが用いられ,一般的に水を用いる加工液への混合割合は重量比で0.1~0.2がよいとされている.超音波を付加することにより,加工能率は振動振幅および周波数に比例して向上するが,仕上面粗さは逆に粗くなる.遊離砥粒加工以外の加工法に対しても超音波振動が広く利用され,超音波切削,超音波研削,超音波研磨,超音波塑性加工,超音波成形,超音波めっき,超音波洗浄など多くの例が見られる.文献(1) 臼井英治, 切削・研削加工学(下), 研削加工, (1979), 22, 共立出版.(2) 大越諄・ほか2名,精密機械,25-6(1959),266.(3) 臼井英治・井上誠,精密機械,44-8(1978),988.(5) 小野浩二,研削仕上―理論とその応用―,(1970),4-11,槇書店.(6) 文献(5)の86ページ.(7) 佐藤健児,研削加工(上),(1958),19,日刊工業新聞社.(8) 文献(1)の34-36ページ.(9) 吉川弘之,精密機械,28-5(1962),286.(10) Outwater,J.O.and Shaw,M.C.,Trans.
