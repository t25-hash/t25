# 7・2・2 電気加工

> **章**: 第7章 特殊加工
> **節**: 7・2 特殊加工法各論

a. 放電加工. 加工法の概要(1) 加工の原理放電加工(EDM : electrical discharge machining)では,油などの絶縁体や脱イオン水などの導電性を低下させた加工液中で,数十mの極間距離を挟んで陽極と陰極の間にパルス状にアーク放電を生じさせる.一つのパルスごとに絶縁破壊が生じる場所はただ1箇所であり,そこに放電電流が集中してアーク柱を形成する.10W/mという高パワー密度のもとで両電極は溶融・蒸発し,微小な放電痕が形成される.そして,加工くずは加工液に接して凝固し,加工液とともに極間から排出される.パルスは前のパルスの放電点の温度が十分低下し,絶縁が回復したころに印加される.このとき前のパルスと同じ位置に放電が発生する確率は小さく,それ以外の極間の狭い箇所に放電点は移動する.こうして放電点が電極面上に分散し,工具電極形状が工作物に転写される.(a) 形彫り放電加工(b) ワイヤ放電加工図7・6 放電加工の2形態(2) 形彫り放電加工とワイヤ放電加工図7・6の形彫り放電加工(sinking EDM)は,切削などで成形した銅あるいはグラファイトの工具電極の形状を工作物に転写するか,単純な形状の工具電極をNC (numerical control)により三次元的に動かし,その包絡面により所望の形状を得る.ワイヤ放電加工(wire EDM)は直径0.03~0.3mmの黄銅(細径の場合はタングステン)などのワイヤを電極として,糸のこ盤のように複雑な形状を切り抜く加工である.ワイヤの傾きを制御し,テーパカットや上下異形状加工も行える.ワイヤ放電加工では,イオン交換樹脂を通して導電率を低く調整した水を加工液として用いる.形彫り放電加工においても環境や火災の危険性を考えると水が望ましいが,工具電極消耗や仕上面粗さなどの問題から油を使用している.(3) 用途・特徴熱的加工である放電加工の特徴は,導電性をもつ材料ならば硬さによらず加工でき,加工可能な形状に制限が少ないことである.例えば,小径コーナや深溝,深穴の加工など工具の剛性が十分でない場合や,工作物自身が薄物であったり,長軸物である場合に有利である.さらに,機械加工と違って工具の回転を前提としないので,コーナの小さい角穴や異形状を容易に加工できる.しかも,加工精度は研削加工と同じ程度によい.一方で,最大の欠点は切削などに比べて加工速度が小さいことである.したがって,用途としては付加価値の高い加工,特に金型の製作に広く使用される.また,近年ではマイクロ加工への応用が注目されている.. 加工装置(1) 放電回路加工装置にはトランジスタ放電回路が用いられる.電流制限抵抗とトランジスタの直列回路が直流電源と極間の間に並列に多数つながれている.したがって,放電電流は一度にオンするトランジスタの数で決まる.トランジスタがオンになり放電が生じたとすると,極間の放電電圧욄は約20V であるので,例えば直流電源の開放電圧が100V,抵抗が20Ωの場合は一つの回路で4A の放電電流が得られる.図7・7は極間の電圧波形と電流波形である.トランジスタがオンして開放電圧が印加されてもすぐには放電は生じず,放電遅れ時間욃を経て絶縁破壊が生じる.極間の状態(極間距離,極間の加工くず濃度)は絶縁耐力に影響を与えるので,放電遅れ時間はパルスごとにばらつき,その平均値は極間の状態の指標である.絶縁破壊が生じると,トランジスタの数で決まる電流値(放電電流욄)に達する.設定された放電持続時間욄が経過するとトランジスタはオフとなり放電が終了する.この制御により放電ごとの放電痕の大きさは一定となり,均一な仕上面が得られる.そして,設定された休止時間後に,再びトランジスタはオンされる.1回の放電のエネルギーは욄と욄と욄の積である.したがって,粗加工か仕上加工かによって放電電流や放電持続時間は作業者が設定する.また,休止時間は放電の安定のために不可欠である図7・7 極間電圧波形と電流波形が,短いほうが加工速度にとって有利なので,必要最小値に設定される.(2) サーボ送り制御放電加工では工具電極の送りは一定速度で行われるのではなく,一定の放電頻度が保たれるように加工の進展に合わせて行われる.加工の進展に対して極間距離が広がると,放電遅れ時間が長い電圧波形が続き,極間電圧の平均値が高く検出される.逆に,極間距離が小さいと,放電遅れ時間が短い短絡ぎみの電圧波形が続き,平均極間電圧は低下する.そこで設定された基準サーボ電圧と平均極間電圧を刻々と比較し,検出された平均電圧が基準サーボ電圧より大きい場合は工具電極を送り,小さい場合は後退を指令する.. 極性と工具電極材料工具電極の消耗率は,時間当たりの工具電極消耗体積を同じ時間の工作物除去体積で除した値であり,加工精度に直接影響を及ぼす.この消耗率は,加工条件により1%以下とすることも可能である.消耗率を決定するおもな因子は極性と電極材料の物性値である.まず,極性の影響を調べるために陽極,陰極ともに銅を使用して,放電1回当たりの除去量を測定すると,本来陽極へのエネルギー配分のほうが陰極への配分より大きいにもかかわらず,放電持続時間が20 s程度より長いときは陽極の除去量が陰極より小さい.この理由は,加工油の中の炭素が分解しておもに陽極表面に付着し,陽極を消耗から防護するからである.したがって,放電持続時間が比較的長い加工条件下では工具電極を陽極にする.逆に,放電持続時間が短い仕上加工やワイヤ放電加工,マイクロ加工では炭素の付着が少ないので,エネルギー配分どおり工具電極を陰極にする.次に,銅の工具電極を用いて鋼を加工する実際の加工を考える.アーク柱からの熱流束が鋼と銅の表面に入る場合,表面温度の上昇は銅に比べて鋼は大きい.その理由は銅の温度伝ぱ率が鋼の約5倍だからである.したがって,銅は鋼よりも融点や沸点は低いにもかかわらず工具電極材料に適している.また,温度伝ぱ率が等しくても,融点や沸点の高い材料は除去されにくい.その点グラファイトは昇華温度が約3900K であり工具電極材料として有利である.黄銅は温度伝ぱ率が銅の3分の1以下で融点も銅より低い.しかし,ワイヤ放電加工では新しいワイヤがつねに繰出されるので消耗の問題はない.また,切断面の真直度を向上させるために張力をかけてワイヤ振動を抑制するが,黄銅は銅に比べ引張強さが大きいので断線しにくい.したがって,ワイヤ電極には黄銅が使用される.b. 電解加工. 電解加工(ECM : grinding)工具に砥石を用い,超硬合金など難研削材の高能率加工を目的とする電解加工である.加工量のほとんどを電解に依存し,電解しにくい物質の除去と表面の活性化が研削の役割である.メタルボンド,電着ダイヤモンド砥石などの導電性砥石を電極工具とする複合電極法と,非導電性砥石を陰極から離して配置し,電解と研削を別の場所で行う中性極法がある.電解に伴う精度低下を補正するため,通常は最終段階において電源を切り,機械研削のみで仕上げる方法がとられる.. 電解研磨(electro-chemical polishing)電解液中に浸漬した工作物表面に電解に伴って生じる不働態酸化皮膜を,液中の強い酸あるいはアルカリ性化学成分により溶解する加工法である.表面のミクロ凸部の溶解速度が凹部よりも幾分大きいことから平滑化が行われ,表面粗さが改善される.工作物材質により,適正な電解液,電圧,電流条件が大きく異なり,電流密度は10~10A/mオーダになる.多数の工作物を同時加工できるのが利点である.極性を逆にすれば,陽極上に液中の金属イオンが析出するめっき現象が起きる.. 電解砥粒研磨(electro-abrasive polishing)硝酸ナトリウム水溶液中における電解に伴って生じる不働態皮膜を,砥粒の擦過により除去する加工法であり,電解複合研磨(あるいは加工)とも呼ばれる.ミクロ凹部では電解溶出がほとんど起こらず,凸部が選択的に除去されるため表面粗さが急速に改善される(図7・9).電流密度は電解研磨と同程度であるが,加工時間は1けた少なくて済み,仕上面粗さも1けた小さい.取りしろが表面粗さと同じオーダに止まることから形状精度はほぼ前加工面レベルに維持される.手送り,自動機による平面研磨のほか,小径管内面への適用もある.
