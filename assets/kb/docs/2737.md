# 8・2・1 電気めっき

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

電気めっきは,めっきしようとする金属イオンを含む電解質溶液に2本の電極を浸し,直流を通電させ,カソードに金属を被覆する方法である.めっき層に要求される条件は,(1)付着性,(2)耐摩耗性,(3)無孔質等が主である.めっき層の特性に電流密度,電解質温度,電解質のpH,電解質の均質化,添加剤の種類等がある.めっきの重要特性の一つは付着強度であり,下地によく付いているかがポイントとなる.電気めっきは機能という面から分類して以下のようになる.(a) 装飾用:貴金属またはその合金のめっきなど(b) 耐食用:亜鉛,カドミウム,スズめっきなど(c) 装飾兼耐食用:銅,ニッケル,クロムめっきなど(d) 耐摩耗用:硬質クロムめっきなど(e) 潤滑用:インジウムめっきなど(f) 電子工業用:金,銀,銅,ニッケルめっきなど一般に,電気めっきの作業は,前処理,電気めっき,後処理よりなっている.前処理は素材表面の研磨,脱脂洗浄,酸洗いなどからなる.前処理の適否は電気めっき製品の最終仕上がりに大きく影響する.後処理は,めっき製品の洗浄,乾燥,さらに必要な場合には仕上げ研磨,化学処理などによる改質を指す.電気めっき製品の性能は電着層の厚さ,多孔性,耐食性などめっき層の物理的,化学的特性,およびめっき層の下地金属との密着性などによって決まる.(1) 銅めっき銅めっきは,ニッケル,クロムめっきの下地めっきなどに広く使われる.めっき浴としては,硫酸銅浴,ホウフッ化銅浴,シアン銅浴,ヒロリン酸銅浴などがある.(2) ニッケルめっきニッケルめっきは,装飾用,耐食用,クロムめっきの下地めっきなどに広く使われる.めっき浴としてはワット浴が代表的であるが,スルファミン酸浴もある.(3) クロムめっき装飾用クロムめっきは,美しい色調光沢を利用するため薄くめっきする.工業用クロムめっきは硬くて摩擦係数が小さいため耐摩擦性に優れているので,多くの機械部品や工具類に適用される.めっCVD(化学蒸着)法熱CVD 法プラズマCVD 法直流プラズマ法高周波プラズマ法マイクロ波プラズマ法光(レーザ)CVD 法真空蒸着法スパッタリング法イオンビームスパッタリング法マグネトロンスパッタリング法アークイオンプレーティング法多陰極熱電子照射法中空陰極法熱陰極法イオンプレーティング法図8・1 表面処理技術の分類き浴はいずれもサージェント浴,これにフッ化物を加えた浴などが用いられる.(4) 亜鉛めっきおよびカドミウムめっき亜鉛めっきおよびカドミウムめっきは,主として鉄鋼の防食用でいずれもシアン化浴が用いられてきたが,非シアン化も進んでいる.(5) その他スズめっきは食品中の各種有機酸に耐えるので缶材料に,金めっきはその耐食性,電気・熱伝導性などを利用して,装飾,電子工学,航空機用に,銀めっきは金めっきに比較して安価なことから,装飾,航空機,電子機器部品などに,白金めっきは耐熱,耐酸化性を利用して電子機器部品,特に高温における接点など,ガラス工業,電極などに,ロジウムめっきは耐食,耐摩擦性,高い鏡面反射率などを利用して,電子機器部品,光学機器部品などに適用される.合金めっきは単一金属めっきでは満たすことのできない品質特性を満足させるため開発され,単一亜鉛めっきより耐食性の優れたニッケル-亜鉛合金めっき,コバルト系磁性合金めっき,金の代用となるスズ-ニッケル合金めっきなどが注目されている.複合めっきは金属の折出と同時に分散粒子の吸着が起こるもので,耐摩耗性,耐食性の優れたニッケル-SiC膜の開発などが進められている.また,重金属,特にシアン化物,カドミニウム,クロム酸などの排出量は厳しく規制されている.対策としては排水処理の完全化と同時に,一方ではめっき浴の非シアン化,無水クロム酸の低濃度化などが図られている.
