# 8・2・2 溶融めっき

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

溶融めっき(hot dipping)とは、溶融金属中に処理しようとする素材を浸漬し、引き上げて表面に付着した溶融金属を凝固させ、被覆層を形成させる方法である。素材は主として鉄鋼で、めっき金属は亜鉛、スズ、アルミニウム、鉛など比較的融点の低いものに限られる。この方法では比較的厚いめっき層が得られ、その用途は主として耐食、耐酸化用である。(1) 溶融亜鉛めっき溶融めっきのうち最も広く実用化されているのは鉄鋼材料への亜鉛めっきで、これは特にガルバナイジング(galvanizing)ともいわれる。めっきは浴温度約450°Cで行われ、合金層の発達を抑制するためアルミニウムが0.1~0.2%添加される。亜鉛めっき鋼の優れた耐食性は、亜鉛自身の優れた耐食性および亜鉛のカソード防食効果によると考えられる。亜鉛鉄板は大気中におけるその優れた耐食性のため、建築材料や家庭電気機器などに広く用いられているが、これらの用途はさらに化成処理、塗装して使うことが多い。(2) 溶融スズめっき溶融スズめっき鋼板はブリキ(tinplate steel)と呼ばれ、食缶類に使われてきたが、最近はスズの節約のため電気めっきに替わられ、さらにTFS(tin free steel)用クロムめっきに替わられつつある。15~25%スズを含む鉛浴によるめっき鋼板をターンプレート(terne plate)といい、自動車の燃料タンクなどに使用される。(3) 溶融アルミニウムめっき溶融アルミニウムめっき(aluminizing)製品は耐熱用と耐食用に分けられ、前者にはケイ素を5~10%含むアルミニウム、ケイ素合金が、後者には純アルミニウム浴が使われる。アルミニウムめっき製品の用途は主として燃焼機器や自動車の排ガス部分などである。
