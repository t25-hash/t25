# 8・2・3 拡散めっき

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

金属素材の表面層に他の金属を拡散浸透させて被覆層を作る方法を拡散めっき(diffusion coating)という.通常行われているのは粉末パック法(pack cementation)で,これはめっきをしようと思う金属の粉末,例えばクロムの拡散めっきのときには金属クロム粉末,活性剤としてのアンモニウム,焼結防止剤としてのアルミナ粉を混合し,その中に素材を埋め込んで,不活性ガスまたは還元性ガス中で加熱する方法である.このとき金属クロム粉末は塩化クロムとなって蒸発し,素材表面では逆に分解して金属クロムを析出する.この析出したクロムは素材中へ拡散浸透し,合金層を形成する.
