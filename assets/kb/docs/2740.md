# 8・2・4 溶射

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

溶射(thermal spraying)は、溶融状態に加熱した溶射材料粉末あるいは粒子を、素材表面に高速度で吹付けて層を作り、皮膜とする表面処理法である。溶射材料の加熱源の種類によって、ガス式と電気式とに大別され、それらには下記の種類がある。(1) ガス式溶射フレーム溶射(flame spraying)は燃料ガス(アセチレン、プロパンなど)と酸素の燃焼の反応熱を溶射材料の加熱に用いるもので、溶線式、溶棒式および粉末式がある。前二者は、ワイヤ状あるいは棒状の溶射材料を燃焼炎によって溶融し、溶融部を空気ジェットで微粒子として素材表面に吹付けて皮膜とする方法である。粉末式では溶射材料粉末をトーチ燃焼炎部へ送給する。本法では、溶射材料は燃焼炎の温度の関係から融点が約2700°C以下のものとなる。本法の特色は操作が簡易で、設備費、運転経費が低廉なことである。(2) 電気式溶射アーク溶射(arc spraying)は、2本の溶射材料ワイヤの先端間に直流ジェットで微粒子として素材面へ吹付けて皮膜とする方法である。本法は、溶射材料が金属に限られるが、出力を大きくして溶射能率を著しく高くでき、しかも運転経費が安いなどの長所を有する。しかし、溶射材料を加熱し、そのため過度の酸化を生じやすく、合金元素の損失が大きいことなどの短所をもっている。プラズマ溶射(plasma spraying)は陰極と陽極ノズルの間に発生させた直流アークで作動ガス(アルゴン、窒素あるいはアルゴンと窒素、水素またはヘリウムの混合ガス)を超高温に熱し、プラズマジェットとしてノズルから噴出させ、この中へ溶射材料の粉末を送り込み、加熱、加速して素材表面に吹付けて皮膜とするものである。本法は、セラミックスなどの高融点材料から各種の金属、合金あるいはプラスチックまで簡単に溶射でき、皮膜の素材への密着性が良好で、高密度であること、不活性あるいは減圧雰囲気での溶射が行えて、皮膜の性質を改善できることなどの長所をもつが、作業中に騒音と強い光を発すること、設備費ならびに運転経費が高いことなどの短所をもっている。溶射法のおもな特徴は次のようである。長所として、(1)素材として金属材料をはじめ、セラミックス、有機材料など、ほとんどの材料を対象とできる、(2)溶射材料の種類が広範囲で、しかも、異種材料を混合した皮膜を簡単に形成できる、(3)表面積の広い素材の限定された部分に皮膜を形成できる、(4)溶射による素材の変形が小さいなどが挙げられる。一方、短所として、(1)一般的に、皮膜と素材との密着強さが比較的に低い、(2)素材表面の前処理に用いられるプラスト作業ならびに溶射作業そのものの雰囲気が人体に有害なので、適切な対策が必要であるなどが挙げられる。
