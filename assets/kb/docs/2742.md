# 8・2・6 物理蒸着(PVD : physical vapor deposition)

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

(1) スパッタリング法真空容器内に導入した不活性ガスをイオン化し,原料ターゲット表面に衝突させ,表面近傍の原子をはじき出し,これらを堆積させて薄膜とする方法である.プラズマを直接用いる方法とプラズマをイオン源としてイオンビームを用いる方式に大別される.工業的には,前者のプラズマを直接用いる方法が多く使われている.直流を図8・2 アークイオンプレーティング法の原理図利用したスパッタリング法に関し,ターゲット材料が導電体の場合は問題ないが,絶縁体の場合は高周波スパッタリングを用いる.(2) イオンプレーティング法イオンまたは励起粒子を利用した真空蒸着法の発展型装置で,プラズマ技術を取り入れた点が特徴である.現在,多くのコーティング分野で使用されており,数nmの極めて薄い膜から数十mの厚膜まで形成可能である.真空蒸着で作製された膜に比べ,一般に密着性が高く,結晶性に優れている.したがって,機能性材料や新材料の開発に適しており,近年はホローカソード放電蒸発式やアーク放電方式等(図8・2),より洗練されたタイプのイオンプレーティング法が設計製作されている.一般にPVD 法はCVD 法に比較し,非平衡度が高く,熱平衡下では作製不可能な物質を創製することが可能である.PVD 法の中でもアークイオンプレーティング法は導電性のターゲットをアーク放電させ,プラズマ中で窒化または炭化して硬質膜を作製する技術である.ターゲットに炭素を選べば種々の条件下で硬質炭素膜が成膜でき,またメタンのプラズマを通せば,水素を含んだ炭素膜となる.
