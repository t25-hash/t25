# 8・2・7 化学蒸着(CVD : chemical vapor deposition)

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

コーティングしたい材料の構成元素を含む単体または化合物の1種類以上の原料ガスを基板上に供給し,気相または基板表面で化学反応により膜を作製する方法の総称である.CVD では広範囲にわたる材料のコーティングが可能であり,また原料ガスの組合せにより,全く新しい構造や組成の膜を作製することもできる.化学反応を生じさせるエネルギーの与え方により,熱CVD,プラズマCVD,レーザCVD に大きく分類できる.熱CVD 法では,一般に1000°C近くの高温を必要とするが,プラズマCVD やレーザCVDでは室温付近または比較的低温で良質の膜が作製可能である.薄膜全体の売上げ高の点で,PVD と比較し,装置がシンプルであることや大量生産に適していることから圧倒的に多い.(1) 低温CVD 技術機械部品の高精度加工に伴い,材料の変形や変質防止のために,より低温での成膜技術が望まれている.したがって,従来の熱CVD 技術に代わり,プラズマやレーザを用いた低温CVD 技術開発が注目されている.(2) 高平滑膜作製基板を平滑化する目的や精密機械のしゅう動特性への要求から,ダイヤモンド状炭素(DLC: diamond-like carbon)が大きく脚光を浴びている.密着性の問題もかなり改善され,高品質かつ高密着強度のDLC膜の開発が注目されている.
