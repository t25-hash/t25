# 8・2・8 各成膜法の長所と短所

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

CVD 法とPVD 法を選択するうえで重要なのは、処理温度の違いである。CVD 法では、約1000°C前後での処理となるため、超硬合金材を除いては、再度焼入れ、焼戻しの処理が必要となり、金型等の精密機器の精度を上げるのが困難となる。一般的には3/100mm以上の許容公差のない金型にCVD 処理を行うのは精度の面でリスクが大きい。また密着性が良いこと、内面への付きまわりが良いなどの利点があり、自動車向けの金型など大型のものに多く適用3―187されている。PVD 法は、上記の煩わしいプロセスを省くことが可能である。PVD 法では、処理温度が500°C以下のものが多く、コーティング後の熱処理は不要である。したがって、ファインブランキングの金型をはじめ、精密・小径のものに使用されている。ただし、内面への付きまわりはCVD に比べ悪く、深絞りのダイ等には適していない。
