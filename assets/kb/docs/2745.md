# 8・3・1 化成処理(chemical conversion treatment)

> **章**: 第8章 表面処理
> **節**: 8・3 化成処理,陽極酸化処理

化学的あるいは電気化学的な処理によって金属表面に安定な化合物を生成させる方法で,その代表的なものにクロム系化成皮膜とリン酸塩処理がある.a. クロム系化成皮膜(chromium conversion coating)被処理物を六価クロムまたは三価クロム化合物とその他の塩を含む処理液に浸漬することによって得られる化成皮膜のことで,次の2種がある.. 六価クロム系化成皮膜一般的にはクロメート処理と称されている.クロム酸または重クロム酸塩を主成分とする溶液中に被処理物を浸漬してその表面に防食皮膜を生成させる方法で,耐食性の向上や塗装の下地処理を目的として行われている.処理できる金属は亜鉛,アルミニウム,マグネシウムおよびその合金などであるが,銅,銀などに処理すると変色防止の効果がある.皮膜はクロム酸クロムを主成分とした非晶質で,生成直後は水を含む膠状であるが,これを乾燥し,脱水させることにより強固な皮膜が形成される.皮膜種類には光沢,黄色,黒色,緑色などがあり,電気亜鉛めっきをした自動車や家電の部品,建築用材などにはこの処理を行うことが必須条件となっている.中性塩水噴霧試験による耐食性試験を行った場合,亜鉛めっきのみでは短時間で白色腐食生成物(塩基性炭酸亜鉛など)が発生するが,例えば黄色皮膜を施したときには腐食生成物が発生してはならない最低時間は72時間と規定されている(JIS H 8625:1993).処理方法には浸漬方法以外にスプレーや電解方法もある.. 三価クロム系化成皮膜上記の六価クロム系化成皮膜の代替として開発されたもので,安全とされている三価クロムが主成分となっている.耐食性など六価クロム系と比較しても遜色がないとされ試験,評価も原則的には同じ方法で行われる.作業性やコストなどに問題はあるが処理装置も大きな変更を加えることなく転用可能なので,環境対応に厳しい自動車や家電業界を中心として採用部品は増えつつある.b. リン酸塩処理(phosphating,phosphate treatment)リン酸塩を含む水溶液を用いて化学的に皮膜を形成させる方法で浸漬あるいはスプレー,はけ塗り方法などにより処理される.おもに鉄鋼材や亜鉛めっき鋼板を対象とした塗装の下地処理方法で素地金属と塗膜との密着性向上を目的として行われる.処理液にはリン酸-亜鉛,リン酸-亜鉛-カルシウム,リン酸-鉄系などの多種類が市販されており,素材や塗料の種類,塗装方法などの違いによって選択される.一般的に皮膜の付着量(g/m)が大になるほど耐食性は良好となるが,反面,ある領域を超えると塗料との密着性が劣ってくる傾向にある.近年,塗料品質の向上に伴って皮膜付着量の少ない薄膜型が多く採用されている.塗装下地を目的とする以外にもリン酸塩皮膜の特性を生かした使用法もある.例えばプレス時における延展性向上やきず防止,あるいはリン酸-マンガン系皮膜をしゅう動面などに処理しておくと初期なじみ性効果が期待できるとされる.また,リン酸-マンガン系皮膜を処理した後に防錆油を塗布すると防錆効果が長期に保てる.
