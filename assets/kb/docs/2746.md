# 8・3・2 陽極酸化処理(anodizing,anodic treatment)

> **章**: 第8章 表面処理
> **節**: 8・3 化成処理、陽極酸化処理

陽極酸化処理とは、アルミニウム、マグネシウム、亜鉛のような金属の表面層を防食、装飾、あるいは機能的性質をもった皮膜(普3―188加工学・加工機器通は酸化物)に変えるための電解処理のことをいう。おもにアルミニウムを対象とした処理方法で、電解溶液中で被処理物を陽極にして通電することにより表面に酸化皮膜層(AlO)を生成させるもので、通称アルマイト処理といわれている。電解液は一般的には硫酸(10~20vol.%)が使用されるが、例えば航空機部品などのように耐食性とともに柔軟性が要求される場合にはクロム酸系が、食器などゴールド色が好まれる場合にはシュウ酸系などと処理浴の使い分けがされている。膜厚は用途により異なるが一般的には10~30mで、これにより硬度、耐摩耗性、耐食性などが向上する。皮膜にはナノメートルレベルの微細な孔があいているので、処理後、沸騰水や加圧水蒸気、化学薬品などにより封孔処理が行われる。また、微細孔には染料などを吸着しやすい性質があるので多彩な色調に染色ができる。一般的な工程は次のようになる。陽極酸化→(染色) →封孔処理陽極酸化皮膜は硬く耐摩耗性が良好なのでしゅう動部品などにも広く採用されているが、摩擦係数が高いのが欠点とされていた。しかし、微細孔中にフッ素化合物や二酸化モリブデンなどを吸着させることにより潤滑性が向上し摩擦係数を低下させることができる。硬質皮膜(アルマイト)は電解液を10°C以下程度の低温に維持しながら処理するもので、皮膜がより緻密となり硬度がさらに向上するので機械部品や半導体製造装置などに用いられる。アルミニウム合金の場合、合金金属もともに陽極酸化されて発色する。処理条件によっても異なるが、例えば銅合金系では赤茶色に、ケイ素系ではグレーに発色する。これらを自然(合金)発色法という。電解着色法とは上法で酸化皮膜層を形成させた後、ニッケルなどの金属塩を添加した溶液中で二次電解することにより、微細孔中に金属を析出させて発色させる方法である。前記の合金法とともに日光堅牢度に優れているので建材などに採用されている。
