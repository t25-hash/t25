# 8・4・2 高周波焼入れ(induction hardening)

> **章**: 第8章 表面処理
> **節**: 8・4 表面硬化法

高周波焼入れとは、電磁誘導現象を利用して電気エネルギーを金属内で直接熱エネルギーに変えて発熱させる方法である。すなわち、磁界内に金属片を置くと電磁誘導によって起電力が発生し、これによる電流はジュール熱となり金属材料の表面部のみで発熱する。そのときの電磁誘導の深さδ(cm)は、次式で定まる。δ=5.03×10ρμρ:被加工物の抵抗率(Ω・cm)μ:被加工物の比透磁率:周波数(Hz)高周波加熱法では局部加熱、硬化層深さの選定が自由であるうえ、運転、標準化、自動化も容易である。加熱後、電源を切れば、ただちに水などにより焼入れをすることができる。はん用されている高周波加熱法のほかに、300~400kHz の高周波を処理品に直接接触させて通電させ、同時に電流は処理品の表面に近接した誘導子を通って返るようにする高周波近接加熱焼入れ法がある。この方法は、処理品の表面層が一部直接通電加熱されるのと同時に、近接誘導子によって処理品の表面が誘導加熱されることになるため、加熱時間が通常の2分の1であり、また加熱部分が局限されるのでそのまま電源を切れば自己焼入れできる。新しい方法として、超高周波(27.12MHz)を用いて短時間にごく表面層のみを加熱し、焼入れする方法も開発されている。
