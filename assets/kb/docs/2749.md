# 8・4・3 レーザ焼入れ(laser quenching)および電子ビーム焼

> **章**: 第8章 表面処理
> **節**: 8・4 表面硬化法

入れ(electron beam quenching)レーザおよび電子ビームを用いた表面硬化処理法としては、焼入れ硬化、合金化、グレージングなどがある。局部的な加熱であるため、処理品はひずまないほか、電源を切るとただちに周辺部に熱が吸収されるため焼入れた状態(self quenching)となり、冷却剤が不要である。省エネルギー性にも優れ、クリーン熱処理として有望である。また加熱源であるレーザや電子ビームの制御は数値コントロールが容易であり、加工ラインとの一体化を図ることもできる。ただし、欠点として加熱場所が狭小で焼入れ幅が狭く、硬化層の深さも他の方法と比べると浅い。広範囲な表面を焼入れるためにはスキャニングをしながら加熱することが必要であり、一般には狭い領域の焼入れや特殊部品に限定されて利用されている。処理設備としても現在では標準機は少ない。
