# 3・2・1 チタンの物理的および化学的性質

> **章**: 第3章 非鉄金属材料
> **節**: 3・2 チタンおよびその合金

a.物理的性質 チタンの特徴の一つは軽量であり，純チタン
の密度は4.51g/cmと鋼の約60%である．チタン合金ではAlが
主体であるα，α- β合金の密度では純チタンと同等またはさらに軽
量になる．Mo，Vなどが10 %以上入るβ合金では密度は5g/cm
まで上がる．ほぼ非磁性であり，透磁率は1.0003である．電気伝
導度は小さく，純銅の3.1 %である．放射化されても半減期が秒か
ら分単位と極めて短く，放射化されない金属の一つとして原子力用
などに有用である．
b.化学的性質 チタンは酸化性酸（硝酸系）に極めて大きい
耐食性を有し，還元性酸（塩酸系，硫酸系）に対しても有効であ
る．塩化物のある環境では白金に匹敵する耐食性を有する．
