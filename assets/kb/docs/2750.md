# 8・4・4 ショットピーニング(shot peening)法

> **章**: 第8章 表面処理
> **節**: 8・4 表面硬化法

部品の表面部に圧縮応力を残留させることは、疲労強さをはじめ応力腐食割れなどの多くの物理的、化学的性質の向上に効果がある。ショットピーニング法は他の化学的な硬化処理法とは異なり機械的に表面硬化をさせる特殊な方法である。具体的には、硬い球状微粒物を高速で金属部品の表面に投射することにより、部品表面を硬化するとともに残留圧縮応力を付与し、おもに金属部品の疲れ強さを向上させる目的で用いられている。すでに航空機部品や自動車部品などに多く用いられているが、特に、ばねや歯車など非常に硬い材料にも適用できる点で、工業上重要な表面硬化処理技術の一つとなっている。最近は数m程度の微粒ショットによる効果の研究も行われている。
