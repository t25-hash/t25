# 3・2・2 チタンおよびその合金の種類

> **章**: 第3章 非鉄金属材料
> **節**: 3・2 チタンおよびその合金

チタン材料は純チタンとチタン合金に分けられ，前者には耐食性
チタン合金も含めている．
a.純チタンの種類 純チタンはおもに酸素によって強度調整
され，JISには1種から4種まで4種類が規定されている．表3･9
に化学組成および機械的性質を示す．国内ではASTM規格も多用
される．ASTM規格でもJIS規格のようにGr. 1からGr.4まであ
るが，JIS規格とは組成ならびに強度規定も異なるので注意が必要
である．純チタン1種は最も軟らかく，三次元的に複雑な成形性が
要求される用途に多用されている．2種も同様に成形性が良く，国
内では最も使用量が多い．
純チタンはそれ自体耐食性に優れた材料であるが，特に優れてい
るのは硝酸に代表される酸化性酸の環境下である．例えば65 %硝
酸では沸騰環境下でも使用可能である．一方，塩酸や硫酸などの還
元性酸の環境下では常温では耐えられるが，沸騰環境下では腐食を
起こす．しかし，酸化剤の役割をする3価の鉄イオンなどがプロセ
ス液中に数ppm存在するだけで腐食は抑えられる．
純チタンでも耐食性の点で問題がある場合は，耐食チタン合金が
使用される．チタンの耐食性を改善する合金組成としてはPt ，Pd
などの貴金属系と，Mo，Niなどのβ安定化元素系がある．前者
の場合はTi -0.15Pd合金が規格化（JIS H4655）されており，同
じ腐食環境下では純チタンに対して腐食量は2オーダ減少する．さ
らに，これらの耐食チタン合金ではすきま腐食の点でも改善され，
すきま腐食発生領域も高濃度，高温側に移行する．しかし，Pdは
貴金属のため高コストになり，Pdをより安価なRuなどに置換え
た耐すきま腐食合金も開発され，実用に供されている．また，β安
定化元素系ではASTMでTi -0.3Mo-0.8Ni合金（Gr.12）が規
格化されている．
b.チタン合金の種類 チタン合金には合金組成によりα型，
α-β型，β型の3種類がある．α型チタン合金にはα安定化元素
であるAlやα，β両相に固溶するSn，Zrが加えられている．代
表的なα型チタン合金としてTi -5Al -2.5Snがある．α型チタン
合金はβ相を含まないため，低温でも脆化することがなく液体He
温度まで使用可能である．また単相であるため，高温まで相変化が
少なく，溶接性（weldability）も良く，高温用途にも向いている．
α-β型チタン合金にはα相を安定化させるAlのほかにβ相を
安定化させるV，Mo，Cr ，Feなどが入れられており，チタン合
金の中で最も種類が多い．β相の少ないα- β型チタン合金はnear
α合金とも呼ばれ，α型チタン合金と同様，高温用として使用され
る．また，α- β型チタン合金は二相合金のため熱処理性を有し，
溶体化・時効により高強度化が図れる．この熱処理性はβ相が多
い合金ほど良好となる．また，溶体化処理条件のコントロールによ
り破壊靱性や疲労き裂伝ぱ特性も改善される．
β型チタン合金にはβ相を安定化させるV，Mo，Cr ，Feなど
がおもに加えられており，合金組成だけで20～30wt.%になるもの
も珍しくない．基本的にはTi -V系，および，Ti -Mo系の系統
あり，前者にはTi -15V-3Cr -3Sn-3Al ，Ti -13V-11Cr -3Al合
金などが，後者にはTi -15Mo-5Zr -3Alがある．両方の組成から
成るものとしてTi -3V-8Cr -6Mo-4Zr -4Snもある．溶体化によ
ってβ単相状態にすると，冷間加工が可能となり，板材として箔
までにすることができる．また，溶体化・時効処理によってα- β
型チタン合金よりも高強度化することも可能で，冷間加工から直接
時効を行った場合には1600MPa以上の強度も達成できる．
これらのチタン合金の機械的性質をまとめて表3･10に示す．
c .その他のチタン合金 金属間化合物系のチタン合金が各方
面で実用化されている．
Ti - Ni系合金は一方向または二方向形状記憶能（shape mem-
ory）を有しており，衣類の形状保持，弱電機器の熱スイッチなど
に利用されている．また，最近ではTi - Nb-Sn系，Ti -Mo- Ga系
などの新しい形状記憶能合金も開発されている．
Ti - Nb系合金は極低温下で超伝導（superconductivity）特性を
有していることから，核磁気共鳴（MRI ）の強磁場発生装置や電
気保存用のコイルなどに採用されている．
Ti - Fe- Mn系合金は水素吸蔵特性があり，プロトタイプの燃料
電池車の水素発生源としてすでに実用化されており，今後の期待も
大きい．
Ti -Al系合金は高温ほど強度が上がる強度の逆温度依存性を示
すことから，将来の軽量耐熱材料の候補となっている．一部はすで
にジェットエンジンの排気系ノズルとして採用されている．
