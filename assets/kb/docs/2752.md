# 3・3・1 ニッケルの物理的および化学的性質

> **章**: 第3章 非鉄金属材料
> **節**: 3・3 ニッケルおよびその合金

ニッケル（nickel ，化学記号Ni ）は周期表でⅧに属する金属
で，面心立方格子の結晶構造を有し，また，353℃で磁気変態点を
有する．合金元素として加えられることの多いV，Cr Ti ，Mo，Mn，Zn，Sn，Cuなどはこの温度を下げ，Co，Feは上
げる．物理的性質を表3･11に示す．
ニッケル地金の純度はJIS H2104，および，ISO規格6283で規
定されている．前者の規格によれば，工業用ニッケルの純度はそれ
に含まれる微量のCoを含めており，Ni ＋Co量で98.0，99.85，
99.95%としている．
