# 8・5・1 プラスチックライニング

> **章**: 第8章 表面処理
> **節**: 8・5 非金属皮膜処理

プラスチックライニング(plastic lining)は,金属素材の耐食性や耐摩耗性を改善させるために表面にプラスチックを被覆することである.一般にプラスチックは薬品に対して優れた耐食性を示すので,これで金属をライニングした部材は化学装置に多く使われ,3―189化学装置の腐食を防止するのみならず,装置内の化学薬品が容器から溶出する金属イオンで汚染されることも防止する.ライニングの施行法は,ライニング材の状態に対応してほぼ次のように分類される.(1) 液体塗布法:プラスチックを溶剤に溶かしたものを塗布後,溶剤を蒸発させる方法.液状プラスチックを塗布後,固化させる方法.溶剤に溶けないプラスチック微粉末を溶剤中に分散させた液を塗布後,乾燥,焼付ける方法など.(2) 粉末焼付け法,静電塗装法,溶射法(8・2・4項参照):プラスチック粉末の流動床の中へ予熱した素材を浸漬し,焼付けライニングする方法.管を加熱しながら回転し,中にプラスチック粉末を流す方法など.(3) シートライニング:素材にシートを張り付ける方法.プラスチックテープを管外面に巻き付けてから融着する方法など.ライニング材にはポリエチレン,塩化ビニル,フッ素樹脂,FRP(fiber reinforced plastic)など多くのプラスチック材料が使用されるが,そこで要求される特性としては,素材金属との密着性,薬品の浸透阻止性,耐食性,耐熱性,耐摩耗性などがある.さらにはライニングのしやすさ,補修のしやすさも考慮しなければならない.被覆材にゴムを用いる場合を特にゴムライニング(rub-ber lining)という.スラリーや汚泥を取扱うような各種ポンプやバルブなどに用いられる.天然ゴムのほか,クロロプレンやブチルゴムなどの合成ゴムも用いられる.
