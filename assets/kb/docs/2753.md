# 8・5・2 セラミックコーティング

> **章**: 第8章 表面処理
> **節**: 8・5 非金属皮膜処理

セラミックコーティング(ceramic coating)は、金属等の素材の摩擦係数を低減させたり、耐摩耗性、耐熱性や電気絶縁性を改善させるために表面にセラミックスを被覆することである。セラミックコーティングはその施行法により次のように分類される。(1) 焼付け法:ほうろう(porcelain enamelting)、ガラスライニング(glass lining、耐酸ほうろう)、耐熱ほうろう(2) 蒸着法:PVD(8・2・5項参照)、CVD(8・2・6項参照)(3) 溶射法(8・2・4項参照)(4) タイルまたはれんがライニングa. ほうろうフリット(ガラス質セラミックスの粉末)を適当な組成になるように湿式混合・粉砕したスリップ(うわぐすり)に薬品と水を加え、素材にはけ塗りまたはスプレーし、乾燥後、焼成炉で焼付ける。素材には炭素鋼や鋳鉄が用いられることが多い。家庭用の浴槽には炭素鋼にほうろうがけした部材が用いられている。また、ガラスライニングは主としてシリカ質からなる被覆層を作るもので、耐薬品性を要求される反応容器の内側に用いられる。耐熱ほうろうは、軟化温度の高いフリットを使って耐酸化性の良い皮膜を作る。耐熱鋼に耐熱ほうろうがけした部材は、ジェットエンジンの燃焼室外筒など耐熱部品に利用される。b. 蒸着法・溶射法によるセラミックコーティング金属等の図8・3 セラミックコーティング膜の硬さ3―19 0加工学・加工機器素材上にPVD,CVD により数nmから100 mの薄いセラミック膜を蒸着したり、溶射法により数mmの厚いセラミック膜を堆積することにより素材の耐摩耗性や耐熱性を改善させる。切削工具および機械部品のしゅう動用には、おもにTi系の薄膜(TiN,TiC,TiAlN など)やCr系の薄膜(CrN など)およびダイヤモンド状炭素(DLC: diamond-like carbon)が用いられる。また情報機器においてもハードディスク表面は数nmから数十nmの薄いDLC膜で被覆されている。耐摩耗性の観点から膜の硬さは重要な特性で、各種コーティング膜について図8・3のように示される。縦方向の広がりは組成・組織や混入不純物の差異による硬さの分布を意味し、化合物の場合は原子量の大きい原子で代表している。耐摩耗性コーティングに必要な膜特性としては、ほかに素材との付着力、相手攻撃性の低さ、低摩擦係数、耐衝撃性、耐薬品性、耐熱性、平坦性がある。さらには蒸着後の後処理の有無や使用環境をも考慮しなければならない。また、ガスタービン翼など高温部材の耐熱性コーティング(thermal barrier coating)にはおもに溶射法によるセラミックコーティングが用いられる。材料にはイットリア安定化ジルコニア(YSZ : yttria stabilized zirconia)が表面層に用いられ、素材の耐熱合金と表面層との間にNi基合金のボンド層を形成することが多い。→ Next Chapter
