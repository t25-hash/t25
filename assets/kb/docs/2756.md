# 9・2・3 ボール盤・ドリリングセンタ

> **章**: 第9章 加工機械
> **節**: 9・2 工作機械の種類

ボール盤(drilling machine)はドリルと称する工具を回転させて切削運動とし,軸方向送りによって穴を加工するための工作機械であり,,軸方向に送りを与えて,溝加工や段などを加工する機能を有しないものをいう.しかし,ドリリングセンタ(drilling center)においては,少なくともタップ加工によってねじを加工する機能を有し,場合によってはごく小径のエンドミルを用いた溝加工,座ぐり加工の機能を有しているのがふつうである.後者のためには,主軸軸受にモーメント負荷がかかるため,それに耐えられる軸受構造にする必要がある.ただし,これを大規模に行えば,通常の立て形マシニングセンタとなってしまうので,どの程度にすべきかは判断が難しい.電気機器においては,それを構成する部材の大半はアルミニウム合金製であり,小径の穴加工,ねじ切りと座ぐり加工程度で十分なことが大半であり,それにはドリリングセンタ,タッピングセンタ(tapping center)と称する小形の専用機的な加工機(図9・14)が適しているので,かなりの台数が製作されている.図9・12 チルティングテーブル[5軸機,ヤマザキマザック(株)提供]図9・13 チルティングヘッド[5軸機,ヤマザキマザック(株)提供]図9・14 タッピングセンタ[ブラザー工業(株)提供]3―19 3
