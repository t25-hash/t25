# 9・2・4 研削盤・グラインディングセンタ

> **章**: 第9章 加工機械
> **節**: 9・2 工作機械の種類

工具として研削砥石を用いる工作機械は研削盤(grindingmachine)と呼ばれ,非常に多くの種類がある.代表的なものに円筒研削盤と平面研削盤がある.円筒研削盤の大部分はNCサーボ化され,数値制御円筒研削盤となっている(図9・15).これは,旋盤と数値制御旋盤との関係に非常によく似ている.平面研削盤はいまでも数値制御化されていないものが大半である.図9・16に示すグラインディングセンタ(grinding center)は,この数値制御平面研削盤から発達してきたというよりも,切削加工を行うマシニングセンタに研削加工用の機能(砥石のツルーイング,ドレッシング機能など)を追加した形で発達してきた.
