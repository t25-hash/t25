# 9・2・5 放電加工機

> **章**: 第9章 加工機械
> **節**: 9・2 工作機械の種類

放電加工機(electric discharge machine)は、干渉用の工具として放電電極またはワイヤを用いて工作物との間に放電を行わせ、図9・15 数値制御円筒研削盤[三菱重工業(株)提供]図9・16 グラインディングセンタと砥石のツルーイング・ドレッシング装置[大阪機工(株)提供]図9・17 放電加工機[三菱電機(株)提供]3―19 4加工学・加工機器図9・18 ワイヤ放電加工機[三菱電機(株)提供]図9・19数値制御ホブ盤[三菱重工業(株)提供]その際に生じる電極反応によって少しずつ工作物を除去する。したがって、その他の加工法と異なり、回転運動は使用されない。送り運動も放電電極は総形を用いるので、軸方向の緩やかな運動に限られる(図9・17)。ワイヤを使う場合は,2軸の同時制御を行って輪郭形状を切り抜く(図9・18)。この場合は加工能率を向上させることと仕上面をきれいにするために、ビートと呼ばれる微小振動をワイヤに加える。
