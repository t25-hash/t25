# 9・3・1 工作機械の形状創成運動

> **章**: 第9章 加工機械
> **節**: 9・3 工作機械の選択

a. 形状創成運動の構成工作機械による加工は、工具と工作物を構造によって堅固に保持し、かつ両者に相対運動を与え、工具と工作物の干渉部分を切りくずとして除去することによって行われる。その結果、工作物には工具形状と工具・工作物間の相対運動が転写され、所要の形状が得られる。したがって、高精度で高能率の加工を実現するためには、工作機械の幾何学的精度だけではなく、高い静剛性、動剛性、熱変形特性、ならびに運動精度を具備することが要求される。工作機械による代表的な加工方法を図9・22に示す。旋削、中ぐり、穴あけ、フライス削り、研削加工などの機械加工により、円筒面や平面などのさまざまな幾何学的部品形状が、所要の工具および工作物の形状創成運動により得られる。表9・1には、工具と工作物の相対運動に着目し、工具と工作物に異なる形態の運動を与えることにより実現される加工方法を示す。最近、工作機械の一構造形態(structural configuration)として空間6自由度を有するパラレルメカニズム(parallel mecha-nism)を応用した工作機械の開発が国内外で進められている。b. 形状創成運動の記述産業革命により近代工作機械が出現して以来、さまざまな形状創成運動機能(form-shaping function)を有する100種類ともいわれる機種の工作機械が栄枯盛衰を経て、現在でも図9・23のように分類される工作機械が存在しているが、それらの統一的な機能および構造の分類体系は確立されていない。わずかに、工作機械の構造設計方法論(design methodol-ogy)を体系化することを目的として、工作機械の形状創成運動の記述方法がブラゴフ(Vragov)、ザリエ(Salje썝)ら、ならびに伊東らにより提案されている。それらは、座標系の設定や配列工具:単刃工具(バイト)(b) 中ぐり(boring)工具:単刃工具(バイト)(a) 外丸削り(turning)工具:多刃工具(ドリル)工具:単刃工具(バイト)(d) 穴あけ(drilling)(c) 平削り(planing)工具:多刃工具(正面フライス)工具:多刃工具(側フライス)(f) 正面フライス削り(face milling)(e) 側フライス削り(side milling)工具:固定砥粒(研削砥石)工具:固定砥粒(研削砥石)(h) 円筒研削(externalcylindrical grinding)(g) 平面研削(surface grinding)図9・22 代表的な加工方法の模式図表9・1工作機械の形状創成運動と加工方法3―19 5旋盤多軸自動盤大形立て旋盤特殊旋盤形彫りフライス盤歯車ラップ盤歯車加工機バフ盤ホーニング盤超仕上盤ベッド案内面研削盤歯車研削盤工具研削盤研削盤在来形工作機械中ぐり盤ジグ中ぐり盤その他ブローチ盤システム適合形工作機械NC 旋盤ターニングセンタマシニングセンタプラノミラーGC歯車加工センタ工NC工作機械システム対応形マシニングセンタ/ターニングセンタ複合加工機複合形ターニングセンタ作機械円筒研削盤内面研削盤平面研削盤心なし研削盤ロール研削盤ねじ研削盤〔例〕カム研削盤特殊研削盤在来形NC工作機械研削盤図9・23 実用に供されているおもな工作機械の分類図9・24 マシニングセンタの運動記述[(株)森精機製作所による]方法に若干の違いはあるものの、基本的には、JIS B 6310:2003に規定された直交座標系により、工作機械の基本的に可能な運動軸とその組合せで与えられる形状創成運動を記述するものである。例えば図9・24は、5軸マシニングセンタの運動軸を工作物側から構造体を経由し、工具に至る力の流れ(flow of force)に沿って記述したものである。図9・24では工場床に設置するベッドやベースのような固定ユニットをで表示している。なお、形状創成運動精度と加工精度(machining accuracy)の関係を解析的に取扱う観点から、同次座標変換行列(homogene-ous transformation matrix)を用いた工作機械の誤差解析を行う方法が定式化されている。
