# 3・6・1 亜鉛

> **章**: 第3章 非鉄金属材料
> **節**: 3・6 亜鉛およびその合金

亜鉛（zinc，化学記号Zn）は電解法と蒸留法によって作られる．
日本での生産分類は，電気亜鉛67%（電解法），蒸留亜鉛27%（蒸
留法），精留亜鉛6 %（蒸留法）である．純度と成分を表3･26に示
す．主要鉱石は閃亜鉛鉱（ZnS），菱亜鉛鉱（ZnCO ）である．
亜鉛は電極電位が卑であるため主用途は鉄鋼の防食（protec-
tion）すなわち亜鉛めっき鋼板用（50%）であり，また銅合金添加
金属（黄銅など）（15%），亜鉛ダイカスト合金（10 %），亜鉛板
（2 %）等が挙げられる．亜鉛めっきには，電気亜鉛めっきと溶融亜
鉛めっきがある．溶融亜鉛めっきでは蒸留亜鉛およびこれに特殊元
素を添加したものが用いられる．また，電極電位が卑であることを
利用した用途としてはほかに電池用材料，電気防食用陽極，亜鉛溶
