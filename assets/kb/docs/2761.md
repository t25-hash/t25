# 9・3・2 各種加工方法と工作機械の関連

> **章**: 第9章 加工機械
> **節**: 9・3 工作機械の選択

加工要求(machining requirement)に基づいて工作機械を選択するためには,まず素形材をどのように所要の加工精度を有する最終部品形状に仕上げるかが重要な問題となる.その際には,各種加工方法の精度,コスト,時間,生産量などを考慮して最適な加工方法を選択する必要がある.多くの場合,同時に治工具・取付具の選定,熱処理方法,組立方法,ならびに検査方法を考慮に入れて加工手順と工作機械を設定する必要がある.すなわち,多種多様な加工要求に対していかなる加工方法を用いて形状創成を実現するか,すなわち加工要求と形状創成運動の関係を明確にする必要がある.表9・2には,加工方法を選択する際に考慮すべき制約条件を集約3―19 6加工学・加工機器表9・2 加工方法を選択する際の制約条件制約条件備考1. 材質2. 所要加工精度3. 加工形状・寸法4. 他の生産工程・生産設備との適合性5. 形状創成の信頼性加工材料の熱的・力学的特性公差,はめあい最大外形寸法,自由曲面形状熱処理工程,ライン生産,工程の集約・分割加工の再現性技術的要因6. 加工能力1ロット当たりの生産量,一定期間の生産量単独工程におけるコスト,総コスト切りくず処理,クーラント所要加工時間工具,ジグ,取付具の準備経済的要因7. 加工コスト8. メンテナンス性9. 生産性,所要加工能率10. ジグ,取付具等の付属品11. 現有設備12. 人的資源13. 安全性14. 環境適合性15. 法規制,標準規格利用可能な設備,空間熟練度,過去の経験発火性,有害物質の生成処理困難な廃棄物の生成騒音・振動,社内規格生産環境要因する.表9・2から,加工対象となる部品の幾何学的形状は加工方法を選択する際の制約条件の一つでしかなく,幾何学的形状が与えられれば,ただちに加工方法が決定されるわけではないことがわかる.実際の生産環境において,所要の加工精度および加工能率を実現可能な加工方法が与えられた際に,それらの要求に最適な工作機械を合理的に選択することが必要になるが,このような部品形状と加工方法の間には一対一の対応関係が成立しないことから,一義的に加工要求から工作機械を選択することは,工程設計(process plan-ning)の自動化に関する研究が数多くなされてきた現在でも困難である.ある程度の対応関係が存在する場合においても,工作機械レベルで考えるか,あるいは付属品レベルで考えるかによっても,両者の対応関係は大きく異なり,例えば旋盤で軸を加工する方法は,種々の形状創成方法と使用工具形状の組合せが存在する.このように,部品形状と加工工程(machining process)の一義性や,要求される加工精度とそれを実現可能な工作機械の選定は,古くから解決が困難な問題として残されており,工程設計の自動化を阻害する一大要因となっているが,依然として経験や勘に依存しているのが現状である.熟練工程設計者は,1枚の図面を見ながら一瞬のうちに工程設計を行うことができる.このような熟練された設計者の高度な問題解決能力は,経験や勘に裏付けられた知識の蓄積によることは確かであるが,すべての工程設計プロセスにおける意思決定(decisionmaking)を経験的知識(empirical knowledge)だけで解決しているわけではなく,その過程では科学的根拠や技術的根拠に基づいた問題解決も重要である.
