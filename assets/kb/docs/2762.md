# 9・3・3 加工要求と工作機械の選択

> **章**: 第9章 加工機械
> **節**: 9・3 工作機械の選択

加工要求に適合した工作機械を合理的に選択するためには，まず
工作機械の設計原理である母性原則（copying principle）の具備
の達成が必要不可欠であり，そのためには，以下の条件を満足しな
ければならない．
(１)所要の形状創成運動の正確な実現
(２)変位基準の設計による構造の実現
(３)全体構造による所要の加工空間（machining space）の実現
これらを考慮した工作機械の選択プロセスを模式化した結果を図
9･25に示す．加工要求に基づいて所要の加工空間内で形状創成運
動を実現することのできる構造形態を具備した工作機械を，一連の
プロセスを経て選択することが可能となる．その際に，加工精度や
加工能率は，最適な工作機械を選択するために必要な選択評価基準
となる．
工部品群から最適な工作機械の構造形態を決定しようとする試
み が行われているが，対象とする問題を限定，あるいは単純化
する以外に系統的な処理が困難である．工作機械の構造設計におい
ては，構造形態の創成 と創成された候補案からの最適な構造形
態の選択を定量的な評価を経て段階的に行う ことが現実的であ
る．
9・4工作機械の性能評価
工作機械（machine tool）の送り軸，主軸およびそれらを支え
る本体構造の特性が切削能率や仕上性能に大きく影響を及ぼすこと
はよく知られている．そのために，古くから工作機械の性能評価方
法（performance evaluation method）の体系化 が行われ，ISO
（International Organization for Standardization，国際標準化機
構）規格やJIS（Japan Industrial Standard，日本工業規格）とし
て整備されてきた．
特に製品の形状精度を決定する幾何精度（geometric accu-
racy）の評価方法は，最初にISO規格としても整備された．数値
制御（NC）工作機械の普及とともに，寸法精度を決定する位置決
め精度（positioning accuracy）の評価方法が標準化され，その後
に輪郭精度（contouring accuracy）に影響を及ぼす同時2軸制御
運動精度が新たな規格として標準化された．最近では，精度の評価
だけでなく，環境温度特性，騒音特性，振動特性などの評価方法も
規格として整備され始めた．しかし，工作機械で重要な特性，例え
ば静・動剛性やびびり振動特性の評価方法の標準化は進んでいな
い．
ところで，工作機械の構造設計に関する研究では，想定される加
ISOでは，どのような工作機械にも共通して適用できる評価方
法を試験方法通則（表9･3) として整備している．これらの通
則では，用語の定義，測定方法，測定器，測定値の求め方，測定結
果の表示方法などが詳細に規定されている．試験結果の評価基準
は，機種別に規定された検査規格の中で，検査項目ごとに許容値
（tolerance）として規定されている．これらのISO規格は逐次日
本語に翻訳されてJISになっている．
