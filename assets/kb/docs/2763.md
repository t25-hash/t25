# 9・4・1 静的精度

> **章**: 第9章 加工機械
> **節**: 9・4 工作機械の性能評価

JIS では,無負荷または仕上げ条件下の運転状態を含めて,静止した状態における工作機械の構成要素(例えば,テーブルや主軸など)の運動の幾何学的な軌跡の精度を,幾何精度とはいわずに静的精度(static accuracy)と称している.この静的精度は,個々表9・3 ISO 230シリーズ「工作機械の試験方法通則」と対応JISISO番号内容対応JIS 番号ISO 230-1無負荷または仕上げ条件下における静的精度の評価方法B 6291ISO 230-2数値制御による位置決め精度の評価方法B 6292ISO 230-3熱影響の評価方法B 6293ISO 230-4数値制御による円運動の評価方法B 6294ISO 230-5騒音放射の評価方法B 6295ISO 230-6加工空間における対角位置決めの精度の評価方法B 6296ISO 230-7主軸の回転精度の評価方法(FDIS)B 6297(予定)ISO 230-8振動の評価方法(DTR)B 6298(予定)注FDIS :Final Draft International Standard,DTR :Draft Techni-cal Reportの工作機械のもっている幾何学的な精度がそのまま工作物に転写されるために,工作機械の評価の中で最も基本的な性能になっている.静的精度は,仕上げられた工作物の幾何精度(工作精度)に関係する工作物または工具を保持する構成部品の動きを,直定規,直角定規,円筒スコヤ,テストバー,水準器などの単純な物理的基準を使用して測定し,評価することを基本としている.ISO 230-1:1996(JIS B 6191:1999)は,おもに(1)工作機械を構成する要素単体の形体精度,(2)工作機械を構成する要素の運動の精度,および(3)二つの運動要素の相対的な姿勢と位置の精度を規定している.これら幾何学的な精度の定義は,ISO/DIS 1101:1996(JISB 0021:1998)に規定されている形状公差(真直度,平面度,真円度,円筒度),姿勢公差(平行度,直角度),位置公差(位置度,同心度,同軸度)および振れ公差(円周振れ,全振れ)と同じであるが,幾何公差表示方式で使用している用語である公差(tol-erance)ではなく精度(accuracy)を使用している.静的精度の測定に使用する測定器としてダイヤルゲージや電気マイクロメータのほかに,同等以上の精度をもつ測定器,例えばレーザ干渉測長器やオートコリメータなどを目的に合わせて使用してもよいことになっている.なお,静的精度とは別に主軸の回転精度(rotational accu-racy)の評価方法(ISO/FDIS 230-7:2006)の規格化が進められている.そのおもな内容はANSI規格に準拠している.
