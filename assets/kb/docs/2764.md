# 9・4・2 位置決め精度・空間精度・円運動精度

> **章**: 第9章 加工機械
> **節**: 9・4 工作機械の性能評価

a. 位置決め精度位置決め精度(positioning accuracy)の評価は、目標とする位置に正または負の方向から繰返して位置決めしたときの目標位置と実際の位置との差の標準不確かさを考慮して、移動の全長にわたって起こりうる狂いを評価し、その最大値を測定値として求める。ISO 230-2:2006に規定する評価方法は、直進軸の移動範囲2000mm以下の場合に適用され、それよりも範囲の大きな大形機の場合には、よく使用すると想定される2000mmの範囲だけに適用される。位置決めをする目標位置は、測定長さ1000mmについて少なくとも5位置、1000mm未満の場合には全移動範囲で少なくとも5位置を定めて往復5回の測定を行う。この測定によって得られた目標位置と実際の停止位置との差を位置決め偏差として定義し、各目標位置において正の向きに近づけた場合と負の向きに近づけた場合との位置決め偏差の平均値および標準不確かさを求める。その後に表9・4に示す項目について評価する。位置決め精度の測定は、レーザ干渉計を使用して行うが、測定結果にはさまざまな要因による不確かさが含まれる。また、機械の暖機運転条件によっては測定値が安定せずに一定の傾向を示すような変化を起こすために、その条件の設定にも細心の注意が必要である。なお、この測定はロータリテーブルなどの回転軸にも適用される。b. 空間精度加工空間のひずみの程度を調べる一つの方法として、空間精度(volumetric accuracy)試験が規格化された。ISO規格では、対角位置決め試験(diagonal displacement test)とも呼んでいる。加工空間を直方体であると考え、その直方体の一つの長方形の対角線に沿って位置決め精度の測定を行う平面対角位置決め(plane diagonal positioning)、直方体の対角線に沿って行う立体対角位置決め(body diagonal positioning)とがある(図方法は、直交2軸または3軸をもつ工作機械の評価に適している。c. 円運動精度円運動試験(circular motion test)は、二つの直進軸を制御して円弧補間運動させたときの軌跡と理想円との差を比較することによって、NC工作機械のさまざまな狂いを総合的に評価することのできる試験方法である。この試験を実施することによって、運動の真直度や直角度などの幾何学的な精度だけでなく、バックラッシ量、半径縮小、位置ループゲイン調整の適否などを評価でき、特に輪郭制御をすることの多いマシニングセンタの検査には欠かせない測定方法になっている。この測定のためにISO 230-1:1996には3種類の測定器を規定しているが、二つの球とその間に変位センサを内蔵した伸縮バーを備えたボールバー(ball bar)が最も広く使用されている(図9・27)。これ以外の同等以上の性能をもつ測定器として、交差格子線を用いたグリッドエンコーダと呼ばれる二次元形状の測定に適した測定器が開発されている。これを使用すれば、任意の二次元形状の軌跡を切削することなく評価できる。円運動の測定方法を拡張し、工作物側の1点と工具側の1点との距離を一定に保って、同時軸(=3~5)制御運動させて求めた軌跡の狂いから機械の組立精度を評価する方法が開発されてい3―19 8加工学・加工機器図9・27 円運動試験に使用されるボールバー装置図9・28 立て形マシニングセンタにおける環境温度の影響および主軸の回転による構造の熱変形を測定する装置の例る。この方法はまた、パラレルメカニズム形加工機の機構誤差の補正にも使用されている。
