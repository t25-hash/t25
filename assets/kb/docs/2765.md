# 9・4・3 熱変形評価

> **章**: 第9章 加工機械
> **節**: 9・26 ).測定は,位置決め精度試験と同じ方法で行う.この試験

さまざまな熱源が原因で発生する工作機械構造の変形や,テーブルなどの位置決め系に発生する熱影響(thermal effect)が工作物の形状と寸法に関係することから,熱変形の評価は工具と工作物との相対距離変化を測定することによって行う.そのために,まず(1)工作機械を設置した環境の温度変化の影響を調べ,次に(2)主軸を回転させたときの影響および(3)テーブルを繰返して移動させたときの影響をそれぞれ評価する.一例として測定器の設置状態を図9・28に示す.規格で定めている測定条件は,機械が室温の状態から4時間運転し,運転を停止した後少なくとも1時間は測定することが条件となっている.熱変形評価の特徴は,測定に長い時間がかかることである.それは,機械が十分に冷えた状態から測定を開始することを条件としているために,一つの測定を終わった後にすぐに続けて次の測定ができないからである.
