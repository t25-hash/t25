# 9・4・5 工作精度および加工能力の評価

> **章**: 第9章 加工機械
> **節**: 9・26 ).測定は,位置決め精度試験と同じ方法で行う.この試験

a. 工作精度の評価工作精度試験(machining test)は,ISO規格やJIS の機種別規格に詳細に規定されている.その一例として図9・29に示すマシニングセンタの工作精度試験方法がある.この試験では真円度,真直度,直角度,同軸度などを評価する.この規格のほかに5軸制御マシニングセンタでは,古くからアメリカ航空宇宙規格(National Aerospace Standard)NAS979:1969に準拠して同時5軸制御による円すい台の仕上削りが行われてきた.これらの工作精度試験は,静的精度や補間精度(interpolation accuracy)が工作物に転写されることを確認するために行うことを主眼としている.b. 加工能力の評価びびりの発生限界を調べるためのさまざまな方法が提案されてきたが,工具,工作物およびその取付具の影響が含まれてしまうことはよく知られている.そのため,例えば,どのような旋盤にでも適用できるはん用性のある方法は確立されていない.しかし,多くのメーカでは独自の試験方法を採用し,評価している.その評価結果は,自社の他の機械と比べてどの程度能力が向上したかを知るために使用されていて,他社の機械と比較してより多く削れるというような結果を出すためには使用されていない.図9・30は横中ぐりフライス盤の基本構造で,工作機械に共通な基本構成要素を示している.その基本構成要素としては,機械の図9・30 工作機械に共通な基本構成要素基本形態を決定付けている主要構造要素(major structural ele-ment)と,それらを結合する結合部(joint),主要運動要素に所要の工作物形状を創成するための運動を与える駆動機構(drivingsystem)などが挙げられる.主要構造要素は,構造本体と主要運動要素から構成され,構造本体としてはコラム,ベッドなど,主要運動要素としてはテーブル,主軸頭,主軸などが挙げられる.結合部は,これら主要構造要素を結合するために設けられており,図ら両者をしっかり固定する固定結合部と,サドルとベッド間のようにサドルを案内し,高精度な直進運動を実現する直進案内結合部と,主軸と主軸頭間のように主軸を案内し,高精度な回転運動を実現する回転案内結合部(軸受部)などがある.駆動機構には,テーブルのような直進送り運動を実現するための直進駆動機構と,主軸のような回転運動を実現するための回転駆動機構がある.工作機械は,以上のような基本構成要素だけでは所要の機能を発揮することはできない.このほかに,上述の運動を高精度に実現するための数値制御装置が必要である.また,必要に応じて,工具を自動的に交換するための自動工具交換装置(ATC),工作物を自動的に交換するための自動工作物交換装置(APC,AWC)や切りくず処理装置などが装備されている.これらに加え,加工用の刃具とその保持具や工作物の取付具といったツーリングシステムと,それらの利用技術が必要となる.
