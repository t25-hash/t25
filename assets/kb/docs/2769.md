# 3・8・5 ベリリウム

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

ベリリウム（beryllium，化学記号Be）は，比重が小さく，融点
が高く，比熱，弾性係数が大きいので，航空機，宇宙船，ロケット
の熱遮へいや構造材料として使用される．熱中性子吸収断面積が極
めて小さく，散乱断面積が大きいので，研究炉の燃料被覆管，減速
材，反射材としても用いられる．またγ線を吸収すると中性子
（neutron）を発生するので，中性子源に用いられる．さらにX線
の吸収が少ないことから，X線管球窓などに利用される．また各
種合金の添加元素としても重要である．例えば，銅合金やニッケル
に添加されると，優れた耐摩耗性，腐食抵抗性，高い電気伝導度，
非磁性，ノンスパーク特性，非弾性挙動を付与する．またアルミニ
ウムやマグネシウムの合金鋳物に対しては，細粒化や酸化抵抗性を
向上させる．
ベリリウムは，フッ化物のマグネシウム還元か塩化物の溶融塩電
解で得られる．粉末焼結品として熱間圧延・押出しを行うことで，
棒，板，チューブになる．ベリリウム合金（beryllium alloys ）
は，応力除去焼なましを行えば，曲げ，3ロール加工，深絞り，穴
あけ加工などが可能になる．ベリリウムの接合は，酸素との親和力
が大きいので，不活性ガスや真空中で行う．
ベリリウムは，空気中で表面に薄く透明緻密なBeO被膜を形成
し保護性があるが，750℃を超すと酸化被膜のはく離が生じる．窒
素に対しては，900℃以上で反応する．アンモニアとは，さらに低
温で反応する．炭酸ガスに対しては，乾燥状態で700℃まで，湿っ
た状態で600℃が限界となる．水素とは，いかなる温度でも反応し
ない．
純水に対しては，ベリリウム中に炭化物，塩化物が混在していな
く，細粒であれば，300℃まで耐食性がある．塩酸，フッ酸，希硫
酸，希硝酸に対しては，室温で反応する．濃硫酸とは反応が緩慢，
濃硝酸とは反応しない．アルカリとは，水素を発生して激しく反応
する．溶融塩に対しては，酸素，窒素を含まない場合に耐食性が良い．
