# 3・8・6 ジルコニウム

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

ジルコニウム（zirconium，化学記号Zr ）はチタン（titanium）
やハフニウム（hafnium）と同じⅣa族に属する金属であり，ジル
コニウム鉱石（ジルコンサンド等）からクロール法あるいはヨード
法によって精錬される．一般工業用ジルコニウム中にはハフニウム
が約2%程度含まれているが，原子炉材料としては中性子吸収断面
積が大きいハフニウムを100ppm以下に分離除去した原子炉級ジ
ルコニウムが使用される．現在流通している金属ジルコニウムはほ
とんどが後者である．
クロール法で精錬された原子炉級金属ジルコニウム（スポンジジ
ルコニウム）の化学組成規格例を表3･49に示す．ジルコニウムの
造塊・熱間/冷間加工は，その冶金的特性の類似性からチタンと同
一の方法・設備で行われることが多い．ジルコニウムは酸，アルカ
リやその塩に対する耐食性が優れており，王水，フッ酸，塩化第二
銅，塩化第二鉄以外にはほとんど侵されないので化学工業用耐食材
料として使用されている．また，熱中性子吸収断面積が小さいので
軽水炉炉心材料として重要である．例えば，高温水および水蒸気中
の耐食性改善を図ったSn，Fe，Cr ，NiあるいはNb等との合金
が開発され，燃料被覆管，燃料チャネル，燃料部材あるいは圧力管
に使用されている．最近では，さらに性能向上を目指した合金の開
発も進められている．
原子力用各種ジルコニウム合金の代表的合金組成および室温機械
的性質を表3･50に示す．ジルコニウムおよびジルコニウム合金の
規格にはASTM規格が一般的に参照されるが，ジルコニウム合金
管についてはJIS規格（JIS H4751：1990）が制定されている．
