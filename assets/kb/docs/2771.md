# 9・5・4 周辺機器

> **章**: 第9章 加工機械
> **節**: 9・41 に示す。このような主軸駆動系は、基本的には主軸、主軸

図9・46は、マシニングセンタ本体とそれに付属している周辺機器の一例を示している。前述のように工作機械は、図9・46に示すような各種周辺機器なしにはその機能を発揮することはできない。周辺機器のうち、数値制御(NC)装置については9・6節で述3―204加工学・加工機器べるので、ここではそれ以外の周辺装置の概要を述べる。マシニングセンタでは自動工具交換装置(ATC: automatictool changer)は必須のもので、周辺装置というよりは機械本体構成要素の一つといってもよい。切りくず処理装置(chip disposalsystem)もほとんどの機械に付属している装置といえ、ATCと並んで工作機械を中核とする生産システムの無人化には必須の装置となっている。さらなる無人化を進めるには自動工作物交換装置が必要となり、長時間無人化を実現するためには、10台以上のパレットで構成される自動パレット交換装置(APC: automatic palletchanger)も使われ、その形式にも各種のものが存在している。
