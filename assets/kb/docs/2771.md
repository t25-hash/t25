# 3・8・7 タンタル

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

タンタル（tantalum，化学記号Ta）はニオブ（niobium）と同
族（Ⅴa）の，高融点（3270K）で非常に大きな比重（16.6）を
有する金属である．鉱石（タンタライト，スズ鉱滓）から溶媒抽出
法等によって精製した後，アルカリ金属による還元法あるいは電解
法によってタンタル粉末に精錬される．造塊にはチタンやジルコニ
ウムに採用されているアーク溶解法や電子ビーム溶解法に加えて粉
末焼結法も使用される．タンタルは白金族に次いで耐食性に優れて
おり，タンタルを侵すのはフッ酸，発煙硫酸や高濃度アルカリなど
わずかである．酸素との親和力が強く，酸化性雰囲気中では570K
以上の温度で容易に酸化する．また，常温でも水素を吸収する傾向
があり，水素脆化には留意する必要がある．加工性は非常に良好
で，常温において細線や箔への加工が可能である．
タンタルの主用途は化学的に安定で誘電率の大きい酸化物の特性
を利用した電解コンデンサで，粉末，線および箔として使用され
る．タングステン，ハフニウムやニオブ等との合金は耐熱，耐食材
料としても使用されている．また，炭化タンタルや酸化タンタル等
の化合物は，それぞれ超硬工具鋼や光学レンズの性能向上に利用さ
れている．
タンタルの成分規格および室温機械的性質規格の例を表3･51，
3･52に示す．
