# 9・6・1 数値制御工作機械一般

> **章**: 第9章 加工機械
> **節**: 9・6 工作機械の数値制御および適応制御

数値制御(NC:numerical control)工作機械は,手動の工作機械に主軸制御用と送り制御用のモータや割出しテーブル,自動工具交換装置(ATC)などを取付け,それらを数値制御データによって運転するものである.主軸系は工作物や工具を把持し,主軸モータによってそれらに必要な加工速度を与える.送り機構は,サーボモータの回転をボールねじによって直進運動に変換し,案内面をもつテーブルを移動させ,その上に固定された工具や工作物に直進の送り運動を与える.送り機構のことを送り軸制御ともいい,複数の送り機構を同時に作動させて工具やテーブルを空間の任意の場所に位置決めできる.直進運動だけでなく,回転運動も制御軸数に含める.その総数は自由度に相当する.任意の位置に移動するには面内で2自由度,空間では3自由度が必要になる.工具や工作物の姿勢を指定するには回転運動を用いるので,さらに制御軸数が増加する.NC装置は,加工プログラムの指令どおりに工作機械を制御するもので,主軸制御モータと送り制御サーボモータをコントロールするNC機能と,操作盤やATCの操作,クーラントのON/OFF,搬送ロボットなど周辺装置とのやりとりなどを行うシーケンス(PLCあるいはPMC)機能を有する.図9・47に示すように,最近のNC装置は高性能のマイクロプロセッサを内蔵し,入出力インタフェースを装備しているので,接続されたコンピュータやネットワーク経由でNC情報を入出力できる.キーボードからMDI(manual data input)で直接入れることも可能である.ディスプレイには実行中のNCデータや工具の現在位置,各種パラメータの設定値が表示される.装置内にストアされているNCデータは順次実行され,一定時間ごとに計算された主軸回転数と各軸の移動量は,主軸制御部とサーボ制御部へ転送される.同時に,シーケンス機能によってNCデータ中で指定されたM,S,T の補助機能を実行する.工作機械には運動制御に必要な制御軸数が用意される.工具をある位置から次の位置に決まった速度で移動させるよう,NC機能は各サーボモータへの指令信号を相互に関係付ける.この機能を補間機能(interpolation function)と呼ぶ.直線に沿って位置と速度を制御する直線補間,円弧に沿って指令された速度で移動する円弧補間が一般に使われる.NCデータで指定された工具経路をたどるよう,補間機能は各制御軸に送る信号を分配する.分配された信号がサーボ制御部に入ると,サーボ制御ソフトウェアはサーボモータの位置,速度,電流を制御し,算出された電流指令値はサーボアンプに送られ,サーボモータを駆動させる.
