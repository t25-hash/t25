# 3・8・9 金および金合金

> **章**: 第3章 非鉄金属材料
> **節**: 3・8 その他

金（gold，化学記号Au）は面心立方構造をもち，柔軟で塑性加m，線には直径8mまで加工できるが，機械的強さが弱い．
耐水，青酸，塩素，臭素，ヨウ化カリウムに反応し銀同様に水銀とア導電性や耐食性に優れるため，電子工業では微量添加物により機Au極細線がLSIのボンディングワイヤに使用されている．
接触のの表面装飾にめっきやクラッドが使われている．
接合材料としてはトおよびその他の高温ろう付けに用いられ，80%Au-Snミックパッケージの封止用はんだに使用されている．
酸ガスによって常温で変色する．
この硫化を防ぐにはAuまたはに溶け，塩素イオンと化合し塩化銀を形成する．
セレン酸，次亜塩ない．
95%以上を反射する ことから，硝酸銀を還元性有機物で銀鏡反デヒド製造にAgの網や細線が使われるほか，抗菌材料過することおよびピンニング効果から酸化物超伝導用のシース材と工性，切削加工性に優れている．
例えば，箔にすると厚さ0.1硫化，耐酸化性に優れ硝酸，硫酸，塩酸などには侵されない．
王マルガム（amalgam）を作る．
械的特性を改善した99.99 %Au以上の純度で，直径10～35mの信頼性を得るためと耐食性向上に電気接点，コネクタ，その他金属18～25%Niおよび25～35%Pd合金がロケットのノズル，スカー550nmから超遠赤外領域の電磁波を高い反射率で反射するたる．
超微粒子はCO低減触媒として活用され，生体材料としてに用いられている．
耐食性を考慮しAg，Cu，Znの配合比を調整したカラット金や系のホワイトゴールド（white golds）が多種類ある．
Au合金のおもな用途例を表3･53に示す．
が低く，加工硬化しても常温で再結晶し軟化する．
硫化水素，亜硫Pdを50at.%以上含有させる必要がある．
硝酸や熱硫酸には容易素酸に反応するが，酢酸，フッ酸，リン酸，カ性ソーダとは反応しを起こす．
導電材料や酸化銀電池に使われている．
Ni，Mgを0.3%添加し内合金し電気伝導率をあまり下げずに硬くしたもの，PdやAuを合電気接点に使われている．
また内部酸化法や粉末冶金法でCdO，の電気接点に使用している．
接合材料としては72 %Ag-Cu共晶合タ電極に使用されている．
銀塩の感光特性がフィルム，印画紙としる．
銀器，コイン，装飾品には約30%使われている．
可視光線の応により還元し反射板や鏡に利用されている．
触媒にはホルムアル（antibacterial material）としても広く利用されている．
酸素を透しても使われている．
