# 1・1・3 固体接触（solid contact ）

> **章**: 第4章 で述べるように，表面は各種金属のほか，セラミックス，
> **節**:  

詳細を1･3･1～1･3･3項に述べるような，以下の二つの固体表面
の接触問題がある．マクロに見ると，幾何学的に点接触（球面どう
しあるいは球面と平面など）あるいは線接触（円筒面どうしあるい
は円筒面と平面など）となるような平滑面の接触状態は集中接触状
態にあり，ヘルツ接触モデルで表され，接触領域の応力は平滑な平
面どうしが接触する場合に比較してはるかに大きく，局所的な弾性
変形が生じる．平滑な平面どうしの接触ではマクロな見掛けの応力
はずっと小さいが，ミクロに見れば，固体表面が法線荷重の作用で
互いに接近すると表面粗さの突起どうしの直接接触が始まり，接触
部およびその周辺に弾性変形や塑性変形が生じる．硬脆材料の場合
ではマクロな変形をほとんど伴わないで，き裂が発生，進展する．
応力が大きいといずれも破壊が始まる．破壊により露出した新生面
は化学的に活性度が高く，このため酸化膜などで覆われた表面とは
異なるトライボロジー的挙動を呈する．また破壊により，接触面か
ら材料の微細片が脱落，飛散する発塵現象はクリーン環境の汚染源
となることもある．したがって，各種荷重下での接触と接触力，変
形と破壊の挙動は接触面に要求される機能と寿命の維持に大きな影
響を与えるので，表面近傍の接触現象のモデル化，応力分布の評
価，き裂の発生と進展の評価，破壊のモデル化などの研究が重要と
なる．
