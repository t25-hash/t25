# 9・6・3 数値制御工作機械用機器

> **章**: 第9章 加工機械
> **節**: 9・6 工作機械の数値制御および適応制御

数値制御工作機械は,前述のようにNC装置から送られる指令に基づいて,工具をある位置から次の位置に決まった速度で移動させながら,与えられた形状と寸法の部品や製品を作り出すものである.数値制御工作機械を構成する機器には,複数の送り制御用サーボモータとアンプ,サーボモータの回転運動を直進運動に変換するボールねじ,ナットを介して結合された送りテーブルとその案内面,加工速度を出すための主軸制御用モータとアンプ,それらを保持して空間の位置を確定するための構造体などがある.サーボモータには位置,速度などを検出するリニアスケールやエンコーダが取付けられ,フルクローズドループ制御やセミクローズドループ制御でフィードバック制御される.回転運動から直進運動に変換するのに使用されるボールねじは,摩擦係数が0.002~0.005と非常に小さく,台形ねじに代わっていまでは大半の送り制御系に採用されている.高精度のうえに互換性もあり,さらにナットを2個組合わせて与圧を与え,バックラッシを取り除くことも可能である.案内面は,送りテーブルをある方向に移動できるようガイドするもので,その方式によって転がり,滑り,静圧案内に分けられる.転がり案内にはボールとローラタイプのものがあり,高速送りや高加速度での移動に適している.滑り案内は,案内面とスライド面が接触しながら移動できる構造であり,移動体の質量の重い図9・51 適応制御の基本構成ものの案内に用いられる.潤滑油を間欠的に供給する方法に加え,転がりを組合わせたハイブリッド式と,つねに空気や潤滑油を供給する半浮上式がある.静圧案内は,加圧された油や空気を作動流体としてスライド面のすきまへ強制的に供給し,流体潤滑状態で送りテーブルなどの移動体を支えるものである.接触面がないので抵抗が小さく,スティックスリップのない高精度な送りが可能になる.工作機械には油静圧案内が使われるが,空気静圧案内は電子機器や超精密加工機や超精密測定機に採用されることが多い.
