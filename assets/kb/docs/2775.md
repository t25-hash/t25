# 1・1・4 摩擦と表面損傷（surface damage）

> **章**: 第4章 で述べるように，表面は各種金属のほか，セラミックス，
> **節**:  

その次に問題となるのが相対運動による摩擦と表面損傷の発生で
あり，詳細は1･3･4～1･3･6項，第3章で述べる．接触面は相対運
動せずに互いに静止していることもあるが，多くは面に沿う滑り運
動や転がり運動，あるいは面に垂直な衝突運動を相互に行い，その
過程で法線力や接線力を互いに及ぼし合う．このため，接触面で生
じた直接接触部がこれらの力の作用により凝着し，さらに相対運動
によってせん断破壊が生じる．あるいは硬いほうの突起が軟らかい
ほうの面に食い込んで掘り起こしながら移動することもある．この
ような現象により，相対運動の方向と逆向きの抵抗力，すなわち摩
擦力が発生し，大気環境下の乾燥した表面では，摩擦係数が
0.3～1程度の値になることが知られている．さらに摩擦によって
表面が損傷し，せん断された凝着部が脱落して摩耗粉が発生する．
相対運動の速度が大きい場合は，摩擦や表面損傷による表面の温度
上昇が大きくなり，酸化物の生成が促進されたり，表面の材料強度
が低下して過大な変形が生じる．また結晶レベルでの変質が生じ
て，表面形状が変化したり，表面の物理化学的性質が当初の設定値
から外れてしまうこともある．温度がさらに上昇して材料の融点を
越えると，局所的あるいは全面的な融解が生じて焼付きが生じる場
合がある．接触面に高い圧力が繰返して作用する場合には，疲労破
壊が生じて表面近傍部分がはく離，脱落する．このような状態にな
ると，接触面はもはや本来の機能を果たせなくなる．トライボ機械
要素の設計は，表面の機能と寿命の維持が目標であるから，この摩
擦抵抗と接触面の表面損傷をいかに制御するかが設計の要点とな
る．
所定の運動を妨げる向きに作用する摩擦抵抗力はエネルギーをむ
だに消費したり，位置決め精度を狂わせるので，多くの機械装置で
は摩擦抵抗ができるだけ小さくなるように設計したい．二つの対向
する固体壁面が互いに直接接触した状態で滑ったり転がったりする
ことが原因で摩擦抵抗が発生するのであれば，2面間に摩擦抵抗の
小さい第三の物質を介在させ，固体表面どうしができるだけ直接接
触しないようにすればよい．これが潤滑作用であり，この第三の物
質を潤滑剤と呼ぶ．潤滑剤の詳細については第2章で述べる．
潤滑には1･4，1･5節で述べるようにさまざまな形態（モード）
