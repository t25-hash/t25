# 9・7・1 塑性加工機械一般

> **章**: 第9章 加工機械
> **節**: 9・7 塑性加工機械

塑性加工とは材料に塑性変形を与えて所望の形状・寸法の製品に
加工する方法であり，塑性変形を与える機械を塑性加工機械とい
う．塑性加工機械は一般に，ロール，ポンチ（パンチ），ダイ（ダ
イス），マンドレル，金型等の工具により，あるいはゴム，液体等
を介して必要な加工力を与え，被加工材を変形させる機械である．
温度，雰囲気，静水圧，摩擦条件等さまざまな加工条件のもとで必
要な変形が与えられ，被加工材を塑性変形させるので，通常，塑性
加工機械は高剛性である．被加工材に大きな荷重を加えることによ
り，塑性加工機械自身の弾性変形が被加工材の製品精度に影響を与
えることがある．
塑性加工機械には，一般に，(１)回転運動するロールまたは工具
（金型）により被加工材に変形を与える形式，(２)工具（金型）に
往復運動を与え，工具間で材料の一部あるいは全体に変形を与える
形式，(３)そのいずれにも属さないか，あるいは二つ以上の機能を
合成した形式，に大別できる．
各種の圧延機やロール成形（roll forming），スピニング（spin-
ning），スリッティング（slitting），レべリング（leveling）等の加
工機や回転鍛造（rotary forging）機が(１)の形式に属している．
一般に工具（ロール）は一軸のまわりを回転するが，回転軸もまた
揺動回転（歳差運動）する揺動鍛造（orbital forging）機もある．
工具は並行運動のみであるが，被加工材が回転し成形されるタイプ
の転造（form rolling）も回転加工に分類 される．したがって，
機械の運動による分類と加工法の分類とは一致しないことがある．
(２)の形式を代表するのはいわゆるプレス（press）機械であり，
板成形や鍛造に用いられる．多くの形式があり，加工力はクランク
等を利用して機械的に，あるいは液圧によるシリンダにより，ある
いはモータとねじ等により与えられる．
(３)の形式に属するのは液圧やゴムを利用したバルジ加工機，電
磁力や放電エネルギーを利用する高速度加工や，レーザを利用し切
断や接合を行うレーザ加工機もあり，レーザは熱応力/残留応力を
利用して成形にも用いることができる．最近，液圧を利用したチュ
ーブハイドロフォーミング（tube hydro-forming）機 が自動車
産業を中心に注目されている．
塑性加工機械を素材への加工機械と二次加工機械に分けることも
できる．素材への加工は，通常，溶けた金属を固化したインゴット
等を被加工材とし，二次加工用素材を製造する．圧延機，押出プレ
ス機等が該当する．一般に大きな変形を与えるので，熱間で行われ
ることも多い．加工機械も大形であることが多く，周辺機器を含ん
だ大規模機械システムとなることが多い．
圧延機は被加工材を板，棒，管，形材，線材に加工する．押出プ
レスは棒，管，形材を製造する．引抜き機械は中間素材の製造に使
われることがある．また，板・管・棒・形材・線材の圧延品や押出
形材・管が二次加工を受けず，そのまま使用されることもある．
圧延や押出しで製造された素材をさらに塑性加工して素形材を製
造する二次加工機の典型は，プレス機械である．板成形プレス，鍛
造プレスはその代表であり，ほかにシヤーリング，ブランキング，
クロッピング等のせん断加工機，コイニング，ホビング等の押込成
形用プレス，曲げ加工用のプレスブレーキ等，その種類も多岐にわ
たる．プレス機械は往復運動により加工を行うが，スピニング，回
転鍛造，転造等回転により二次加工を行う機械もある．対向液圧プ
レスのように(２)と(３)の中間の形式のプレスも存在する．
これらの分類に分けがたい加工方法・加工機械も増えつつある．

（mushy- state forming)，金属粒子をバインダとともに射出成形
