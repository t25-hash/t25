# 9・7・1 塑性加工機械一般

> **章**: 第9章 加工機械
> **節**: 9・7 塑性加工機械

塑性加工とは材料に塑性変形を与えて所望の形状・寸法の製品に加工する方法であり,塑性変形を与える機械を塑性加工機械という.塑性加工機械は一般に,ロール,ポンチ(パンチ),ダイ(ダイス),マンドレル,金型等の工具により,あるいはゴム,液体等を介して必要な加工力を与え,被加工材を変形させる機械である.温度,雰囲気,静水圧,摩擦条件等さまざまな加工条件のもとで必要な変形が与えられ,被加工材を塑性変形させるので,通常,塑性加工機械は高剛性である.被加工材に大きな荷重を加えることにより,塑性加工機械自身の弾性変形が被加工材の製品精度に影響を与えることがある.塑性加工機械には,一般に,(1)回転運動するロールまたは工具(金型)により被加工材に変形を与える形式,(2)工具(金型)に往復運動を与え,工具間で材料の一部あるいは全体に変形を与える形式,(3)そのいずれにも属さないか,あるいは二つ以上の機能を合成した形式,に大別できる.各種の圧延機やロール成形(roll forming),スピニング(spin-ning),スリッティング(slitting),レべリング(leveling)等の加工機や回転鍛造(rotary forging)機が(1)の形式に属している.一般に工具(ロール)は一軸のまわりを回転するが,回転軸もまた揺動回転(歳差運動)する揺動鍛造(orbital forging)機もある.工具は並行運動のみであるが,被加工材が回転し成形されるタイプの転造(form rolling)も回転加工に分類される.したがって,機械の運動による分類と加工法の分類とは一致しないことがある.(2)の形式を代表するのはいわゆるプレス(press)機械であり,板成形や鍛造に用いられる.多くの形式があり,加工力はクランク等を利用して機械的に,あるいは液圧によるシリンダにより,あるいはモータとねじ等により与えられる.(3)の形式に属するのは液圧やゴムを利用したバルジ加工機,電磁力や放電エネルギーを利用する高速度加工や,レーザを利用し切断や接合を行うレーザ加工機もあり,レーザは熱応力/残留応力を利用して成形にも用いることができる.最近,液圧を利用したチューブハイドロフォーミング(tube hydro-forming)機が自動車産業を中心に注目されている.塑性加工機械を素材への加工機械と二次加工機械に分けることもできる.素材への加工は,通常,溶けた金属を固化したインゴット等を被加工材とし,二次加工用素材を製造する.圧延機,押出プレス機等が該当する.一般に大きな変形を与えるので,熱間で行われることも多い.加工機械も大形であることが多く,周辺機器を含んだ大規模機械システムとなることが多い.圧延機は被加工材を板,棒,管,形材,線材に加工する.押出プレスは棒,管,形材を製造する.引抜き機械は中間素材の製造に使われることがある.また,板・管・棒・形材・線材の圧延品や押出形材・管が二次加工を受けず,そのまま使用されることもある.圧延や押出しで製造された素材をさらに塑性加工して素形材を製造する二次加工機の典型は,プレス機械である.板成形プレス,鍛造プレスはその代表であり,ほかにシヤーリング,ブランキング,クロッピング等のせん断加工機,コイニング,ホビング等の押込成形用プレス,曲げ加工用のプレスブレーキ等,その種類も多岐にわたる.プレス機械は往復運動により加工を行うが,スピニング,回転鍛造,転造等回転により二次加工を行う機械もある.対向液圧プレスのように(2)と(3)の中間の形式のプレスも存在する.これらの分類に分けがたい加工方法・加工機械も増えつつある.溶湯から固化の工程を経ず半凝固の状態で成形する半溶融加工(mushy-state forming),金属粒子をバインダとともに射出成形3―207する方法等が開発され,塑性加工と鋳造や射出成形の間の垣根が除かれつつある.塑性加工機械は従来,金型を用い,金型形状を素材に短時間に転写することで大量生産に利用されてきた.最近,はん用工具を用い,素材に局所的に塑性変形を与え,順次成形を行うインクリメンタル成形(incremental forming)が注目され,専用の機械が市販された.これは昔から行われていたハンマリングや鎚起(ついき,ハンマによる打出し加工の一種で銅製品の加工に使われることが多い)の技能を,コンピュータで制御する知能化機械として現代によみがえらせたものといえる.多品種少量生産に向いた加工方法・加工機械であるが,加工時間は長く,製品精度は金型を用いる場合に比べて悪くなるのがふつうである.塑性加工製品にも高付加価値化が求められており,製品の高精度化,複合化が必要とされている.これに伴い,塑性加工機械の高精度化,多機能化,複合化が要求されている.切削機能を有するプレスシステムもすでに市販されている.また,コンピュータを核とするIT 技術の利用は避けて通れない.シミュレーションをはじめ,CAD/CAM/CAEといった塑性加工全般にわたるコンピュータ利用技術はすでに実用のレベルにあり,塑性加工機械やシステムの最適化が必要とされる.さらに,環境負荷の低減も重要な課題である.
