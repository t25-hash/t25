# 1・2・2 設計項目と設計ツール

> **章**: 第4章 で述べるように，表面は各種金属のほか，セラミックス，
> **節**: 1・2 トライボ設計

設計項目は三つに大別され，表面のマクロ・ミクロな形状，寸法
および相対運動，表面を構成する材料，潤滑剤と潤滑法となる．
これらの各項目について，要求される機能，性能，寿命を満足す
る解を得るための設計ツールとして，要求項目と設計変数を関係づ
けることのできる確かなモデルが必要である．流体潤滑モードの場
合には，各種のレイノルズ方程式が実験的検証に耐えた理論モデル
として利用可能であり，設計変数と要求性能を精度よく関係づける
ことができる．しかしながら，混合潤滑，境界潤滑，固体潤滑のモ
ードについての理論モデルは，現象を定性的に説明することはでき
ても，設計に使用可能なほど精度のよい定量的予測は現状では達成
できていない．すなわち，性能を精度よく予測する潤滑理論のモデ
ルのみならず，要求寿命を満足する設計解を得るための損傷理論の
モデルもまだ確立されたものが提示されていない．
これら理論モデルの不足を補うものとして実際のトライボ設計に
使用されているのは，実機における使用実績や実験的評価手法によ
り集積したデータベースやマップであり，材料，摩擦，表面損傷形
態，摩耗形態と摩耗量，焼付き限界，潤滑油の機能・性能に与える
各種添加剤の効果，潤滑油寿命などが各所で作成されている．
