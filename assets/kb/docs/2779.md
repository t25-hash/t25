# 9・7・3 プレス

> **章**: 第9章 加工機械
> **節**: 9・7 塑性加工機械

プレス(press)はプレス機械(presswork machinery)と同意語で使われ、プレス機械を鍛圧機械ともいう。プレス機械を使った加工をプレス加工といい、加工される材料の塑性変形能を利用して造形する塑性加工機械である。プレス加工の対象となる製品は、半導体や光ファイバコネクタの金属部品のように小さく、加工精度がミクロン単位のものから、たたみ3畳を超える大板まで可能である。また、最近ではクリーンルーム内でも使えるプレス機械がある(図9・53)。a. プレス機械の種類プレス機械は成形に必要な大きな加圧力を出す機構をもつだけで、プレス機械自体には造形能力がなく、いろいろな金型が取付けられるはん用的な機械である。一つのプレス加工部品を作るために必要な金型数は加工形状の複雑さによって決まり、一つの金型のみで出来上がるものは少なく、一般的に一つの部品をプレス加工するために、3~20種の金型が必要である。金型構造は型製作と型メンテナンスを考慮して単純化される。したがって、1台のプレス機械に一つの金型を取付けて生産する方法、数型を取付ける方法、全加工工程のすべての金型を取付ける方法がある。取付ける型数が多くなるとプレス機械は大形化・大能力化しなければならないので、設置面積やプレス価格を考慮して、一般的には数台のプレス機械を使って生産する。プレス機械ははん用性があり、金型の交換によっていろいろなプ進んでいる。図9・53 プレス機械の構造3―209レス加工が行える。はん用性があるといっても加工内容が複雑であったり、成形精度がミクロン単位などのように精密になると、金型だけでは正しいプレス加工を行うことは難しくなる。また、高速運転に適した設計も必要となるので、実際には加工の内容に適した機能や装置をもつプレス機械が作られている。日本国内で使われているプレス機械台数はおよそ30万台弱といわれ、その90%以上が機械式駆動のプレス機械であり、機械プレスの70%以上がプレス加工内容を特定しないはん用用途のプレスである。以下では機械プレスに限定して述べる。b. プレス機械の設計基本プレス機械の設計はプレス機械の定格荷重(公称能力)、公称能力を出せる下死点からの距離(トルク能力)、成形に使えるエネルギー量(エネルギー能力)の3能力を基準に行われる。公称能力は荷重のかかる部品、例えばフレームやスライドの設計に使われ、荷重を受ける構造部材は応力値、たわみ量を重点に計算する。トルク能力はクラッチやクランク軸および歯車などの設計に使われ、歯車類は精度と耐久性に重点をおいて設計するが、低騒音化も必要である。エネルギー能力は電動機やフライホイールの設計に使われ、保有エネルギー量は抜き加工プレスでは少なく、絞りや冷間鍛造を主とするプレス機械では多くし、いずれも速度減少率を考慮しながら設計する。速度減少率は最大で30%くらいである。また、省エネルギーや電源容量なども考慮する。エネルギー供給源である電動機は、出力容量が大きくなるほどkW 間隔が広がって急激に価格が高くなり入手も困難になるため、エネルギー余裕も最小限にしている。c. プレス機械の駆動方法と駆動装置プレス加工には大きな成形荷重と大量の成形エネルギーが必要である。したがって、プレス機械は大きな加圧力を発生できる機構、その荷重に耐える構造体、大量の成形エネルギーを蓄えかつ短時間に放出できる機構をもつ。加圧力とエネルギーを出す駆動源には油圧式と機械式があり、機械式駆動プレスの中で最も多いのがクランク機構である。クランク機構における滑り子(スライドという)の行程速度は正弦曲線で変化し、必ずしも成形に適しているとはいえない。そこでクランク機構よりも成形に適した速度が得られるとして、1970年ごろからリンク機構が登場し、現在では多くのプレス機械に利用されている。構造が複雑になる分、プレス機械価格が高くなる。油圧駆動のプレスを油圧プレス(hydraulic press)といい、機械プレス(mechanical press)に比較して使用台数は少ないが、100万kN を超える加圧ができるなど油圧の利点が活用されている。1990年ごろから油圧プレスもCNCサーボ化されるようになり、機能が高まっている。ACサーボモータで駆動するサーボプレスも1995年ごろから作られるようになり、プレス加工もIT 化に向かい始めている。このほかにわずかであるが、空圧プレスや人力プレスがある。d. プレス機械の制御装置プレス機械は切削機械より極めて作動速度が速い。さらに、加圧力が大きいためプレス機械の誤操作や誤作動は人や機械に大きな損傷を発生し、時には死亡事故につながる場合もある。また、高速化には自動化装置などの周辺装置と同期運転や同調運転が必要となるなど、制御性能が重要な要件となっている。쓕労働安全衛生法」でプレス機械および関連装置を正しく設計製造するための「動力プレス機械構造規格」および「安全装置設計指針」等が定められており、法律に基づいて検定を受け、設置後の定期的な特定自主検査実施も法で定めている。違反は法で罰せられる。2003年ごろには国際的にも安全規格が厳しくなり、ISO化が3―210加工学・加工機器制御は1980年ごろからそれまでのリレー回路がCPU 内蔵のシーケンサ回路に替わり、現在ではCNCが一部であるが使われ始めている。e. プレス機械のクラッチとブレーキ機械プレスでは加工に必要なエネルギーをフライホイール(flywheel)の回転エネルギーとして蓄えるので、フライホイールは常時回転させる。プレス機械の起動・停止はフライホイール回転に関係なく任意に行うため、フライホイールとクランク軸の間にクラッチを設ける。クラッチは、単板または複板の摩擦板を空圧または油圧シリンダで締付けて回転を伝えるフリクション式クラッチが多く使われる。摩擦板締付け開放はばねで行う。ブレーキも単板または複板のフリクション式である。ブレーキは非常停止にも使うので、高い信頼性と短時間急停止性能が求められる。ブレーキ作動はクラッチと逆でばね締付け停止、空圧シリンダによる締付け開放である。ブレーキ開放後クラッチ入り、クラッチ開放後ブレーキ入りとなるよう機械的に調整され、固定される。クラッチもブレーキも電気的にインタロックされ制御的に監視し、空圧低下は非常停止となる。摩擦板摩耗は作業者を危険にさらすので毎日の点検が必要である。f. プレス機械の付属装置(ダイクッション装置)最も典型的な付属装置は絞り加工に使われるダイクッション装置(die cush-ion units)で、円筒絞りでは次のように使われる。図9・54に絞り加工と金型を示す。プレス加工は一つの加工工程で終わることはなく一般的に3~20工程が必要で、リチウムイオン電池ケースなどは絞り加工だけでも10工程以上を必要とする。円筒絞りは、素材円板の外径が縮小しながら円筒立体に成形される。縮径時に板厚方向に座屈すると、板が波形に変形して成形不良となる。この波形変形をしわといい、しわ発生を防止する板押さえ装置が必要となる。ステンレス鋼の絞りでは絞り力に近い板押さえ力を必要とするなど、板押さえ装置は大きな力を発生させる機構で、自動車ドアなど自動車外板の絞りでは3000kN を超える力が必要である。板押さえ力はプレス加工内容によって大きく変わるので、圧力調節しやすい空圧シリンダ式が多く利用され、大形プレス機械では油圧式、油圧サーボ式などが、小形プレス機械ではゴム式やばね式が使われる。g. プレス機械の付属装置(過負荷防止装置、overload protec-tor)自動化したプレス加工では、送り装置や金型の不具合によって金型内に2個以上の材料を入れたまま加圧したり、金型交換時に必要な調節を間違えたりしてプレス機械に過負荷を発生させてしまうことがある。いずれも発生頻度は少ないが、過負荷はプレス機械や金型を破損する。過負荷からプレス機械や金型を保護する装置が過負荷防止装置である。図9・55の装置は過負荷を機械的または図9・54円筒絞り金型の構造電気的に検出して、油圧室内の高圧油を瞬時に開放して過負荷から離脱する。同時にプレス機械は非常停止する。h. プレス機械の付属装置(金型交換装置、die changer)金型重量は1型当たり小形で100kg、大形では20tを超える。金型交換と交換に伴う締付け・調節作業を安全に、かつ短時間で行う交換装置が用いられる。手動で交換する装置からコンピュータでプログラム制御される自動交換システムもある。交換装置を使うことによって作業が数分で終わる。金型締付けは、油圧やばねなどを使って安全確実に行えるダイクランパが使われる。i. プレス機械の付属装置(材料送り装置、strip feed units)プレス機械はプレス加工の安全化、成形品質の向上、生産性の向上、高速化などを目的として自動化される。送り装置はプレス機械の動きに同期して、材料の挿入と成形品の取出しを自動的に行う。金型で加工中は材料を固定し、加工が終わると同時に送るという間欠送りを行う。金型から金型へ送る自動化では送りの正確さと同時に位置決め精度も重要であり、板厚0.2mmの薄板を毎分2000回以上の高速で送るためには、送り精度±20 mが要求される。駆動装置には機械式とサーボ電動機式があり、多関節ロボットも使われている。送り対象物は板材から成形中の立体形状品まであり、かつ大きさも大小さまざまで、搬送重量が200kg を超える場合もある。図9・56は薄板を高速・高精度で送るのに適したカムインデックス駆動式送り装置を示す。カム式は毎分2000回以上の送りが可能だが、送り長さ変更が任意にできないので、最近は送り長さ変更が自在なACサーボ式送り装置が作られている。j. これからのプレス機械プレス加工も少量少品種から多品種少量生産に変わり、そのうえ日本国内のプレス加工空洞化が激しく、プレス機械へもIT 化が押し寄せている。成形シミュレーション解析やCAD がものづくり現場に直接入り込む時代になりつつあり、プレス機械の知能化や複合機能化が求められている。さらに、後継者不足から熟練者の技能が伝承しにくくなってもいるので、低コストの知能機械が求められる。省資源工法としてのプレス加工はこれからも必要であるから、新世代のプレス機械実現が望まれる。
