# 1・2・4 そのほかの潤滑モードにおけるトライボ設計

> **章**: 第4章 で述べるように，表面は各種金属のほか，セラミックス，
> **節**: 1・2 トライボ設計

流体潤滑モード以外のモードでは，形成される潤滑膜が薄かった
り，固体表面の直接接触が大なり小なり存在するので，流体潤滑モ
ードに比較して大きな摩擦係数と表面損傷の発生確率が高くなり，
これらを極力低減する設計に重点がおかれる．材料の設計では，材
料表面のミクロ形状を工夫して摩擦面間に潤滑剤をわずかでも滞留
させることが可能な自己潤滑性の材料を使用する，各種の表面改質
を施して摩擦を下げたり耐損傷性を高める，などがある．境界潤滑
モードの潤滑剤の設計では，材料表面に強固な吸着膜を形成できる
ように吸着活性の高い極性基を有する物質を添加する，摩擦面温度
が上昇して焼付きの危険性があるときに低せん断性の成分が発現す
るような極圧添加剤を加える，などがある．
