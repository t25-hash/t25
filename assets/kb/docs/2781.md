# 9・7・5 せん断加工機

> **章**: 第9章 加工機械
> **節**: 9・7 塑性加工機械

a. せん断加工機の概要材料の上・下に刃物を設け、刃物の相対運動により材料をせん断(shear)加工するための加工機を、一般的にせん断加工機と呼ぶ。b. せん断加工機の種類せん断加工機(shearing machine)の種類は非常に多く、せん断加工機を性格づける要素により種々の名称がある。バーシヤー、ビレットシヤー(材料を止めることなく走間中にせん断を行うもの)のように加工材の運動状態を表したもの、ダウンカットシヤー(下刃固定、上刃を下方に動かしてせん断を行うもの)、アップカットシヤー(上刃固定、下刃を上方に動かしてせん断を行うもの)のように刃の運動方向を表したもの、エアシヤー、油圧シヤー、電動クランクシヤーのように刃の駆動方法を表したもの、ギロチンシヤー、ロータリシヤー、アリゲータシヤー(下刃固定、上刃が挟み運動を行うもの)のように刃の運動様式を表したものなどがあるが、一般的には、刃の形状およびせん断の種類により、(1)直刃せん断加工機、(2)回転刃せん断加工機、(3)回転ドラムせん断加工機、(4)その他のせん断加工機に大別できる。c. せん断加工機の詳細。 直刃せん断加工機板材を直線せん断するもので、静止した板材をせん断するものと、走間中の板材を止めることなく所定の寸法にせん断するものとがある(図9・58)。前者のうちシート状の板材をせん断するには、フレームにギャップのあるギャップシヤー、フレームにギャップのないスケヤシヤーが用いられ、コイル状の板材を巻戻してせん断するには、種々の駆動方式のダウンカットシヤーやアップカットシヤーが用いられる。これらのシヤーは厚板から薄板まで幅広く使用できる。走間中の板材を止めることなくせん断するせん断加工機をフライングシヤー(flying shear)といい、種々の形式のものがある。熱間厚板の走間せん断には4リンク形、ペンジュラム形のフライングシヤーが用いられ、中・薄板用にはダイセット形、揺動クランク形のフライングシヤーが用いられる(図り口はきれいであるが、高速せん断には適していない。材料の最大送り速度は揺動クランク形で180m/min、ダイセット形で60m/min程度である。刃物にはせん断荷重を小さくするためレーキ3―212加工学・加工機器図9・59揺動フライングカットシヤー構造例図9・60 回転刃せん断法を付けるが、レーキが大きいと材料にシヤーボウが発生し、切り口の直角度も悪くなる。. 回転刃せん断加工機ロータリシヤー(rotary shear)ともいい、円形の上下刃を回転させて板材をせん断加工する(図ん断するスリッタなどがある。この形のロータリシヤーは中・薄板のせん断に適しており、刃物の軸を駆動してせん断するパワーカット方式と、刃物を材料の送りに従動させてせん断するプルカット方式がある。また、円形上刃と直線下刃を用い、直線刃と回転刃せん断方式を組合わせたローリングカットシヤーがある。せん断材の反り、ねじれ、あるいは横曲がりの少ない切断が可能である。円形刃の転がり運動を発生させるためのガイド機構を単純化し、さらに逆押さえを用いることによって、より精密せん断が可能なローリングカットシヤーも考案されている(図9・61)。. 回転ドラムせん断加工機ドラムシヤー(drumshear)ともいい、クロップシヤー、チョッパ、デバイディングシヤー、ドラム形フライングシヤーなどがある(図9・62)。ドラム形フライングシヤーは刃物の運動軌跡が円形であるため、板厚が厚くなると切り口が悪く、かえりが大きくなるが、構造が簡単で保守が容易である。また高速せん断に適し、ドラム形フライングシヤーの最大送り速度は、400m/minに達する。. その他のせん断加工機鋼、線材、山形鋼、円管、平鋼のせん断には、刃物をその材料の形状にしたギロチン形のせん断加工機が使用される。また板材の打抜き、穴あけ、縁取りなどにもそれぞれの用途に適した形のせん断加工機が広く使用されている(図
