# 9・7・6 回転加工機

> **章**: 第9章 加工機械
> **節**: 9・63 )。

棒材、板材、管材などの素材または工具を回転させて塑性加工を行う機械を総称して回転加工機(rotary forming machine)と呼び、その代表的なものを以下に挙げる。. ねじ転造盤(thread rolling machine)ねじ転造盤には2個の平ダイスによる平ダイス転造盤、2個または3個の丸ダイスによる丸ダイス転造盤、セグメントダイスと丸ダイスを用いるプラネタリ転造盤がある。平ダイス転造盤はM 24以下、プラネタリ転造盤はM 12以下の小径ねじの大量生産に適しており、一方、2個の丸ダイスによる丸ダイス転造盤は広範囲のねじ寸法と材質に適用できるので、精密ねじの多種生産に適している。このほかに、旋盤に転造ヘッド(転造アタッチメント)を取付けてねじ転造を行うこともある。. 歯車転造盤(gear rolling machine)歯車の創成転造法としてはラックダイス方式(Roto-Flo法)、ローラダイス方式、内歯ダイス方式(WPM 法)などがあり、成形転造法としてはGrob法がある。ラックダイス方式では比較的小径、小モジュールのインボリュートスプライン軸、インボリュートセレーション軸、インボリュートはすば歯車などが冷間転造されている。ローラダイス方式ではインボリュート平歯車、はすば歯車の熱間転造が行われ、またシェービングに代わる歯車の冷間仕上転造も行われている。WPM 法では内歯をもつ一対のセグメントダイスが平行を保ったまま同期して円運動することによって、インボリュートスプライン軸などの冷間転造が行われている。Grob法では、同期して遊星回転する一対または二対の総形ローラによって歯形を成形転造し、スプライン軸の冷間転造法として長い実用実績があるが、薄肉のタイミングベルトプーリなどの加工にも利用されている。. クロスローリング機(cross rolling machine)クロスローリング機は、対向する一対のくさび状の突起を有する平ダイスまたはロールによって段付き軸状部品を熱間転造する加工機であるが、小物精密部品の冷間クロスローリングも行われている。また、スルーフィード転造を行うクロスヘリカルローリング機では大形ねじ、フィンチューブ、球、ころ状部品などの連続加工が可能である。. 回転鍛造機(rotary forging machine)回転鍛造機には上型揺動回転方式と下型回転駆動方式がある。二重偏心スリーブで上型に種々の運動軌跡をさせる場合には、上型揺動回転方式が必須であるが、連続した据込みを行う場合には、下型回転駆動方式のほうが装置の駆動系が比較的簡単になる。b. スピニング機(spinning machine)スピニング機は、板材、管材、鍛造材、鋳造材などの素材をマンドレル(成形型)に取付けて回転させ、1個または複数個のローラを押し付けてマンドレルと同じ形状に成形する加工機であり、薄肉製品用の横形加工機の場合は旋盤と類似の主軸台、心押し台、ロール台および各種補助工具などで構成される。しごきスピニングを除けば多サイクルパスの成形となり、以前はテンプレートを利用した電気-油圧サーボならい制御でロールを油圧駆動していたが、現在ではNC制御が主流となっている。厚板成形用の加工機では主軸台と心押し台の推力が数百kN の高剛性で、板状素材から曲げ、絞り-しごきスピニングを行って1チャックで内歯付きの部品などが加工されている。また、2個のタレットヘッドに6個ずつのローラを取付けて複合加工を行う加工機もあるが、これらの高剛性の加工機ではローラの軸方向位置は固定で、数百kN の推力でクランプされた素材が、主軸台ならびに心押し台とともに回転しながら軸方向に往復することによって成形するものが多い。1台のはん用加工機で、板状素材から多サイクルパスの複合加工を1チャックで行うもの、複数の専用機械をライン化してマルチステーションで複合加工を行うものなどさまざまである。c. スエージング機(swaging machine)スエージング機は、相対向する複数組のダイスに半径方向の往復運動を与えて、棒材や管材を外周から圧縮して軸方向に絞り出す加工機であり、ダイスを素材軸のまわりに回転する方式と、ダイスは固定で素材を回転する方式のものがある。前者はロータリスエージング機(rotary swag-ing machine)と呼ばれ、外径120mm以下の細物に適し、各種段付き軸やテーパ部品の加工に用いられる。後者はダイスと中空軸固定でアウタレースを回転させるもの、あるいは偏心軸によりダイスを往復運動させるものなどがあり、大形製品に適している。d. その他リング状素材を圧延成形するリングローリング機があり、ベアリングレースや車輪のタイヤ、そのほか異形断面のリング状製品を熱間で高能率に加工できる。またディスクローリング機ではサイドロール、ウェブロール、リムロールの位置制御によって、種々の形状のディスクやディスクリングが加工できる。
