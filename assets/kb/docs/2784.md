# 9・7・8 矯正加工機

> **章**: 第9章 加工機械
> **節**: 9・63 ).

a. 板材矯正機金属帯板の矯正機には,矯正プレス,ストレッチャレベラ,ローラレベラ,テンションレベラなどがある.矯正プレス(gag press)は主として油圧による加圧方式であり,加圧力を容易に調整できる.さらに,ラムに矯正ジグを設け,ラム位置を矯正位置まで移動させる機構を有する.ストレッチャレベラ(stretcher leveler)は,グリップマークや糸巻き状の幅縮みのため,四周をトリミングする必要があるが,処理材の残留応力が低く強度的な信頼性があり,アルミニウム板等で特殊用途のものに利用される.ローラレベラ(roller leveler)は多数のワークロールを上下千鳥状に配設し,板材に繰返し曲げを与える.ワークロールは大きな矯正反力を支持するため,図9・67に示すごとくバックアップロールで支えられる.これを四段式と呼ぶが,簡単な二段式が曲がり矯正の目的で使用されている.ワークロール径により矯正可能な材料仕様限界が決定されるが,その例を図9・68に示す.所要ロール本数は経験的に薄板で11~21本,厚板で5~11本が採用される.薄板のローラレベラでは,バックアップロール位置を調整してワークロールをたわませるクラウニング装置を装備し,部分伸びの矯正能力の向上を図るものが多い.テンションレベラ(tension leveler)は連続的に処理材料に塑性伸びを与えるもので,張力を付加するブライドルロールと,繰返し曲げ変形を与えるレベリングロールユニットより構成される.レベリングロール前後のブライドルロール群をそれぞれ厳密に速度制御することにより,矯正に最適の伸び率を設定できる機構が多く採用されている.処理材料は最大板厚で6.0mm程度であり,これを超えると設備費が増大し経済的でなく,ローラレベラの使用される領域となる.矯正速度は1000m/minを超える高能率機もある.特に硬質,極薄板には曲率半径の小さい矯正工具により,流体を介して材料に強い曲げを与えて矯正する方法も一部行われている.b. 管材・丸棒矯正機管材,丸棒の矯正にはロータリストレートナ(rotary straightener)が用いられる.これはローラの駆動に伴って材料を回転させながら繰返し曲げを与えるもので,ローラは上下とも鼓形が多く,ロールの配置は図9・69(a)の対向式,図9・69(b)の千鳥式およびこれらの複合形がある.対向式は材料が上下ロールで拘束されており,安定した大曲がり矯正ができる.丸棒の矯正には管材と同じものが使用されるが,そのほか簡単な図9・図9・67 四段式ローラレベラロール配置図69(c)の2ロール式もあり,矯正速度は最高120m/minで,圧延棒材,磨き棒材に広く使用されている.c. 形材矯正機山形鋼,平鋼,レール等の矯正には一般にカリバのついたロールをもつ形鋼矯正機(shape straightener)が使用される.材料はロールによって垂直方向に曲げられ,水平・垂直両方向の曲がりが同時に矯正される.ロールは片持式が多く,材料の寸法範囲が広いものはロールピッチ可変式のものも用いられる.一般に駆動は上下いずれかのロール群とし,他方は非駆動となっている.最高矯正速度は200m/minを超えるものがある.抽出法により製造される複雑な断面の形材には,ストレッチャツイスタ(stretcher twister)が使用される.これは曲がりのほかにねじれを除去するため,形材を引っ張った状態でねじり,まっすぐに矯正する機械である.
