# 10・1・1 長さの実用標準

> **章**: 第10章 加工測定
> **節**: 10・1 長さの測定

現在,光の波長が長さの基準として用いられている.1983年の第17回国際度量衡委員会において,長さの基準であるメートルの定義が「メートルは1秒の299792458分の1の時間に光が真空中を伝わる行程の長さである」と改訂された.これまではメートル原器という基準器が用いられ,そしてクリプトン86原子が放射する単色光の波長が用いられてきた.現在はヨウ素吸収安定化レーザが長さの基準発生装置として用いられている.そして現場用レーザ干渉測長器に直接長さを移すことが行われている.レーザ干渉測長器が普及した現在,現場においても極めて長さの定義に近い精度をもつ道具が使用されるようになった.現場で使用されるHe-Neレーザ干渉測長器の安定度は10の-8~9乗程度,すなわち1mで0.01 m(10nm)~0.001 m(1nm)の安定性をもっている.現場でのレーザ干渉による測定は空気中で行われる.したがって,空気の屈折率の変化がその波長に影響を与える.その主たる要因は気温,気圧,水蒸気圧,炭酸ガス含有量であるとされている.およそ1°Cの気温変化,または4hPaの気圧変化で屈折率は1×10変化する.その結果波長も同程度変わる.こうした状況を考えると,例えば鋼の長さの測定時,10の安定性を得るには以下のようになる.鋼は1mにつき1°Cの温度変化で11 m変化することから,0.01°Cで0.11 mとなり,1/1000°Cで11nmである.したがって,極めて高い温度安定性とその推定精度が要求される.このことから,実用上レーザ干渉測長器の波長の安定性に関しては無視しても差し支えないことがわかる.言い換えれば,今や測定の不確かさを小さくする最大の要因は温度推定精度であるともいえる.長さの実用標準としては現在,光の波長を基準とするレーザ干渉測長器,目盛の線間隔を基準長さとする線度器(line standard),物体の両端の長さを基準長さとする端度器(end standard)の3種が用いられている.a. レーザ干渉測長器(laser interferometer)レーザ干渉測長器は図10・1に示す構成で用いられる.大きく分けて光学系,発振器ヘッド,波長補正ユニット,演算表示部であるが,近年はパーソナルコンピュータ(PC)の普及・機能高度化により演算表示部をPCで置換えるものが増えている.光学系はコーナキューブを用いるはん用のもの,光路長を2倍とし,実際の変位量の2倍が光路長変化とするダブルパス用プレーンミラーなどがある.また,光学系を変更することにより,単に長さだけでなくピッチ,ヨーなどの角度変化の測定,真直度の測定なども可能となっている.角度変化の測定では2本のビーム長さの変化より計算することから光路中の空気の揺らぎに敏感であり,不確かさを小さくすることが困難である.最大測定距離は使用する光学系にもよるが,おおよそ30~40図10・1 代表的レーザ干渉測長器の構成例3―217mである.分解能は10nmあるいは1nmが一般的であるが,1nm台では光学系,内挿系の非線形の影響がでてくる可能性がある.b. ブロックゲージ(gauge block)ブロックゲージは現場において寸法の基準を作り出す重要なゲージである.両端面を基準とすることから端度器と呼ばれる.基準に用いられる両端面が互いに平行かつ平面に仕上げられている.両端面の長さは,呼び寸法と呼ばれるゲージに記載された数値に極めて近い(例えば50nmの不確かさ)値で製作されている.平面は他のブロックと互いに密着(リンギング)できるよう平滑にラップ仕上げされている.ブロックゲージは複数のゲージのリンギングにより任意の寸法が作り出せるよう,適当な寸法のものの組合せセットで販売されることが多い.一般に材質は鋼種で作られるが,最近はセラミックス,溶融石英などのものも現れている.材種が異なると線膨張係数も異なることから,利用に際しては温度と材質の関係に十分注意を払う必要がある.また,熱処理を行い経年変化に十分に注意して製作されているが,それでも寸法変化が認められている.したがって,使用による摩耗と経年変化の把握のために定期検査が必要となっている.一般には100mm以下のブロックゲージを組合わせるセットが多いが,なかには1mの長さのブロックゲージもある.ほかに,同様の端度器として棒状のバーゲージ(bar gauge)がある.また,工作機械の位置決め検査などに用いられる段差ブロックゲージ(ステップゲージ,step gauge,図10・2)では,このブロックゲージを組合わせて製作しているものもある.限界ゲージ(limit gauge)は加工物の寸法が定められた交差内に入っているかどうかを検査するゲージである.一般的なものとしてプラグゲージ,挟みゲージ,リングゲージなどがある(図10・3).これらのゲ図10・2 段差ブロックゲージ(ステップゲージ)の例[(株)ツガミ](a) 限界プラグゲージ(b) 限界平プラグゲージ(c) 限界挟みゲージ(d) 限界リングゲージ図10・3 限界ゲージの形状3―218ージでは最大許容寸法を与えるものと,最小許容寸法を与えるものがセットで用いられる.穴の径を検査する場合,最大許容寸法ではプラグゲージが入らず,最小許容ゲージが入ることになる.限界ゲージはその性質上摩耗が激しく,定期的な検査・校正が必要とされる.c. ディジタルスケール(digital linear scale)従来の標準尺に取って代わる線度器としてディジタルスケールがある.ガラスあるいは金属の板の上に光学格子を付けたものをスケールとし,スライダに取付けた光電素子を組込んだ検出ヘッドにより移動量を読み取る.図10・4はその内部構造例である.格子はあまり細かいピッチでは作れないことから2本の格子線の間を分割する内挿が電気的に行われる.格子ピッチ8 mのスケールであると,スケール信号の位相差を利用してまず4分割,そして電気的に20~200分割して最終的に0.1~0.01 mの分解能を得ている.最も普及しているのが光学式であるが,ほかに静電容量式,磁気式などがあり,おのおの一長一短の特徴をもつ.ガラスをスケール材としているものは大体8.8×10の線膨張係数をもち,鋼よりわずかに小さい.精度の高い測定の場合には温度補正が必要となる.また零膨張係数ガラスを母材としたスケールもあり,1~10nm台の分解能をもつスケールに多い.ディジタルスケールでは目盛の作成時のばらつき,非線形性と目盛分割の誤差の2種が,その主たる目盛の不確かさの要因であるが,反面空気中をビームが飛ぶような構造ではないため,再現性は極めて高い.したがって,総合誤差を考えると,短い長さではレーザとほぼ同レベルの不確かさを期待できる.機械加工現場で用いられるものはほとんどが2m以下であるが,長いものではテープスケールがある.この場合は,数十mまでの長さのスケールを作ることができる.短いものではディジタルダイヤルゲージなどに組込まれている20mm程度のものもある.
