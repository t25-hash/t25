# 10・1・2 加工現場における長さの測定機器

> **章**: 第10章 加工測定
> **節**: 10・1 長さの測定

加工現場において使用される測定機器には,長さおよび寸法測定・管理のためのもの,形状測定・管理のためのもの,制御系の一部として組込まれるものなどがある.また.特に用途が規定されず多様に用いられる変位センサ類がある.a. 長さおよび寸法測定・管理用測定器ノギス(verniercaliper),マイクロメータ(micrometer),(ディジタル)ダイヤルゲージ(dial gauge),工具顕微鏡,三次元座標測定機,投影機,レーザ干渉測長器,空気マイクロ,電気マイクロ,穴径測定器.加工現場に現在最も普及しているものはノギス,マイクロメータ,ダイヤルゲージであり,これらはよく測定用工具と呼ばれる.これら測定工具もディジタル化が進み,従来より1けた精度,分解能が向上している.またディジタル化の結果,読み間違えによるトラブルの減少,高齢者の目の負担軽減,測定結果の直接収集などが実現している.特にマイクロメータにおいてはその目盛の構造上,0.5mmを間違えるトラブルがあったが,直読によりこの問題も解決された.残る問題は電池の寿命であるが,静電式の開発,改良により年単位の電池寿命も達成されている.図10・5(a)~(c)は従来型のノギス,マイクロメータ,ダイヤルゲージの基本構造である.また図10・6(a),(b),(c)はディジタル化されたノギス,マイクロメータ,ダイヤルゲージの内部構造例である.ディジタル化により大きく内部が変化したのがダイヤルゲージで,ラック図10・4 mの分解能も得られるなど利点が多い.反面,応答速度が遅い,管理,制御のための電気信号を得るためには圧力-電気変換器を必要とするなどの弱点もある.b. 形状測定・管理用測定器光波干渉(干渉じま)測定器(平面度,形状など),一部の形状測定(粗さ,真円度,真直度など).形状測定用の測定器も小寸法の長さ測定器として応用されることが多い.高さ,厚みの測定で,100 m以下の範囲では光波干渉測定機器(例えば,レーザ平面度干渉計など)により直接段差あるいは厚みの変化を測定することができる.同様に表面粗さ測定器により,やはり段差,厚みの寸法測定が可能である.その際,測定器のもつダイナミックレンジには十分留意する必要がある.場合によっては,近い寸法の段差をブロックゲージで作り,比較測定する必要もある.軸受,球の直径の測定に真円度測定器を応用し,比較測定した例もある.あらかじめ径のわかっているリングゲージを用いて真円度測定器の測定子の位置を校正し,そのままの状態で対象物の径を測定する.チャート上に現れた輪郭線の直径方向の位置の差が直径差を表す.c. てこ型を組合わせて使用する場合も多い.特に生産プロセスの過程の記録を要求されることが多くなった現在,多種の寸法情報が収集されている.三次元座標測定機に用いられるタッチプローブも単体でセンサとして用いられることが多い.特にNC(numerical control)工作機械にあっては,タッチプローブ信号により工作物の取付け座標,アライメントなどが測定できる機能をもっているものが増えている.センサとして比較的長い変位の測定にはディジタルリニアスケール,あるいはそれを組込んだプランジャ型変位センサが,短い変位(1mm以下)には差動トランス型,静電容量式,光電式などの変位センサが適宜用いられている.表10・1にはこれら変位センサの中の代表的なものの特徴を示す.
