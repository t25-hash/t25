# 10・2・1 一般事項

> **章**: 第10章 加工測定
> **節**: 10・2 加工部品の測定

a. 加工部品の測定加工部品の測定は、その部品が設計者が意図した機能を満たしているかどうかを評価する機能測定が最終的な目的となる。実際には、設計者は機能を満たすような部品の各要素の寸法や形状を設計するため、加工部品の測定では、加工された部品が設計された図面に示されている寸法および形状を満たしていることを評価することになる。寸法には寸法公差、形状には形状公差が示されているので、その公差によって部品の合否を判定するこ図面指示加工物加工物の表現測得当てはめ(a)(b)(c)(d)A:図示外殻形体、B:図示誘導形体、C:実形体、D:測得外殻形体、E:測得誘導形体、F:当てはめ外殻形体、G:当てはめ誘導形体図10・14 形体の定義(JIS B 0672-1:2002)とが目的になるが、試作や開発の段階、非常に高精度な加工の場合などでは、形状を測定することで生産方法を改良や変更したり、再加工を行ったりする場合もある。今後のディジタル化された生産システムの中で、加工部品の形状測定における動向を考えた場合、以下に示す三つの項目をきちんと理解する必要がある。・製品(加工部品)の幾何特性(形状)を明確に定義するための標準化体系・形体を中心としたデータ処理の考え方・測定の不確かさを考慮した判定加工部品の形状測定する方法には、10・1節で示した長さ測定、および10・2・2項で示す三次元座標測定機を用いた測定、10・2・3項で示す形状測定機、および10・2・4項で示す表面粗さ測定機による測定がある。b. 製品の幾何特性仕様加工部品を測定する場合のデータの流れを考えた場合、生産システムのディジタル化に対応して、製品の幾何学的な特性をきちんと定義し、あいまいさをなくした新しい考え方が重要となっている。この考え方は、製品の幾何特性仕様(GPS : geometrical product specification)として、ISOを中心に国際標準化が進んでいる。この分野の新しいJIS 規格の多くは、GPS という名前の規格体系の一部となっている。この規格体系は、従来の機械部品の規格のあいまいさをなくし、測定の不確かさの概念を中心とした規格の整合化を目指しているISO/TC 213(ISOのGPS 規格に関連する技術委員会)を中心に整備が行われている。この概要については、TR B 0007:1998「製品の幾何特性仕様(GPS)―マスタープラン」を参照すること。加工部品の形状の定義に関しては、形体を中心とした考え方を理解する必要がある。形体(JIS B 0672-1:2002)とは、点、線または面のことで、外殻形体と誘導形体を含んでいる。外殻形体は実体のある形体で、誘導形体は外郭形体から導かれた中心点、中心線または中心面である。形体の定義の相互関係を図10・14に示す。図面上の理想的な幾何形状(図示形体)に対して、加工中の工作機械の変動などにより加工物の形体(実形体)には形の偏差が生じる。測定による加工物の表現では、測定で得られたデータによる表現(測得形体)に対して、最小二乗法などにより当てはめを行った表現(当てはめ形体)を計算し、最終的な測定結果を得る。c. 測定の不確かさ(measurement uncertainty)とトレーサビリティ(traceability)測定項目に対応した測定機器を選択する場合および測定結果の合否を判定する場合には、測定の不確かさを考慮する必要がある。測定の不確かさは、測定した結果がもつ精度の評価である。ISOから出ている『計測における不確かさの表現ガイド(GUM : Guide to the expression of uncertainty in mea-surement)』では測定の不確かさは「測定の結果に付随した、合理的に測定量に結び付けられる得る値のばらつきを特徴付けるパラメータ」と定義されているが、非常に高精度な測定以外の一般的な場合には、「真の測定値(実際にはこれは不明である)が存在すると考えられる範囲」と思ってもよい。例えば、長さの測定結果が10mmでその拡張不確かさが0.1mmという場合には、真の測定値は10±0.1mmの間に95%の確率で存在する。測定の不確かさは、測定機器の精度だけで決まらないので、測定の不確かさを正しく推定することは必ずしも簡単ではない。加工部品の測定に限って考えれば、測定機器の測定精度、測定機器の校正方法による不確かさ、測定環境による不確かさ、測定戦略による不確かさを考えれば十分な場合が多い。測定の不確かさを含めた測定結果の合否判定については、JIS B0641-1:2001「製品の幾何特性仕様(GPS)―製品及び測定装置の測定による検査―第1部:仕様に対する合否判定基準」に従う必要がある。基本的な考え方としては、図10・15のように、適合範囲に対して不確かさを内側に取って合否を判定する。このため、測定の不確かさの大きい場合には、適合と見なされる範囲が小さくなるので、注意をする必要がある。さらに、測定の不確かさの推定には、トレーサビリティを考慮することが不可欠である。測定結果がトレーサビリティをもつためには、測定機器をトレーサビリティが保つ方法で校正する必要がある。長さの測定においては、ヨウ素安定化レーザ、ブロックゲージ、標準尺などがトレーサビリティ体系の中で供給されているので、それらを使った校正やそれらと関連付けられた校正を行うことで、トレーサビリティ体系を保つことができる。トレーサビリティを保つこと、測定の不確かさを考慮した測定を行うことは、生産システム全体のグローバル化に必要不可欠な要素となっている。
