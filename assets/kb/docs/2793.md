# 10・2・4 表面性状(surface texture)の測定

> **章**: 第10章 加工測定
> **節**: 10・22 は,座標測定機の寸法測定における最大許容誤差の表現の例

a. 一般事項加工物の表面において表面粗さは,滑り,摩耗などの機能に対して重要であるため,その測定方法や指示方法が定義されていた.しかし,加工物の多様化により表面のうねりや加工による筋なども重要な要素となってきた.そこで,表面全体を表面性状(JIS B 0601:2001「製品の幾何特性仕様(GPS)―表面性状:輪郭曲線方式―用語,定義及び表面性状パラメータ」)として扱うようになった.断面曲線(primary profile)は,測定された加工物の実表面を,ある断面で測定した測定断面曲線(図10・31)にカットオフ値λ욦の低域フィルタを適用して得られた曲線であり,うねり成分および粗さ成分を含んでいる.図10・32のように,この断面曲線に対してカットオフ値λ욂の高域フィルタを適用して粗さ曲線(roughnessprofile)を,カットオフ値λ욞の高域およびλ욂の低域フィルタを順次かけることによりうねり曲線(waviness profile)が得られる(図10・33).表10・4に表面性状パラメータを示す.JIS B 0601の1994年版に対して,2001年版は大きく変わっている.特に,各パラメータに対して,粗さ曲線,うねり曲線,断面曲線に対するパラメータが定義されている.例えば,算術平均高さは粗さ曲線に対しては,うねり曲線に対しては,断面曲線に対してはと表示される.b. 0651:2001「製品の幾何特性仕様(GPS)―表面性状:輪郭曲線方式―触針式表面粗さ測定機の特性」には,表面粗さおよびうねりを測定するための触針式表面粗さ測定機の一般的な構造について規定されている.図10・38は,触針式表面粗さ測定3―228加工学・加工機器機の構成例である.測定機の測定ループの中に入っている各要素(触針,プローブ,送り装置)が測定結果に影響を及ぼす.触針はダイヤモンド製で,球状先端をもつ円すい形状で先端の半径が2m,5 mまたは10 mでテーパ角度が60°または90°である.触針を含むプローブ(ピックアップ)は送り機構がもつ基準案内(基準線)によって正確な軌跡に沿って案内される.プローブは,触針先端の変位を変位伝達機構によりトランスデューサへ伝播し,トランスデューサにより信号へ変換され,増幅器,A-D 変換器により基準線に対する測定断面曲線をディジタル量へ変換する(図10・39).プローブの測定範囲を拡大することで,うねりや断面形状に対応した触針式輪郭測定機もある.このようにして得られた測定断面曲線は,表面性状の定義に従って,輪郭曲線のフィルタ処理および評価が行われる.粗さ曲線のフィルタのカットオフ値λ욂としては,0.08,0.25,0.8,2.5,8mmが使われ,λ욦としては2.5,8,25 mが使われる.うねり曲線については,現在規格化が進められている.最近の触針式表面粗さ測定機は,ディジタル化され多くのパラメータを計算できるものが主流となり,複数の走査を行うことで三次元的な表面性状に対応しているものもある.図10・40に触針式表面粗さ測定機の例を示す.図10・41に触針式輪郭測定機の例を示す.
