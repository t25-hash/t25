# 10・3・1 一般事項

> **章**: 第10章 加工測定
> **節**: 10・3 インプロセス・オンライン計測

高度の自動化、生産性の向上、高度の品質管理等の要求が厳しくなるにつれ、加工中の状態や機械の状態を監視・診断しつつ、適宜修正、補正等を行う技術への期待はますます高まりつつある(図でもなく、インプロセス計測法(in-process measurement)あるいはオンライン計測法(on-line measurement)として、これまで長い歴史をもっている。つまり、加工中に測定を行い、その結果を用いて監視、診断、制御を行うためのオンライン計測法である。加工プロセスを正常に保ち、最適な状態に維持するためには、センサを通して加工状態に関する情報を監視システムや制御システムに提供する必要があり、また異常を診断するためにも診断システムに必要なセンサ情報を送る必要がある。この計測法の最大の問題点は、センサ自身あるいはセンシング法としていかに信頼性と安定性をもたせるかにある。加工中の切りくず、加工液、発熱、振動などの劣悪な加工環境の中にあって、それらに耐えられるようなセンサや計測法の開発が強く求められてきた。加工を一つのシステムと考えると、図10・46に示すように計測対象は工具、工作物、加工プロセス、それに工作機械に分けられ、それぞれに対して計測項目が挙げられる。これらの計測項目を専用に検出できるセンサが特にあるわけではなく、最も適していると思われるセンサを選択して計測しているのが実情である。したがって、まずセンサの取付け位置を最適に選んだうえで、センサからの出力信号をいかに信号処理するかが成功の鍵といえる。センサに要求される基本的な性能としては、精度、信頼性、安定性、適応性そして経済性が挙げられる。それと同時に、以下のような計測事象側の状態についても考慮し、センサを選定しなければならない。(1) 必要精度、感度、分解能、測定レンジ、さらにはセンサの取付け位置およびスペース(近接計測-遠隔計測、接触式-非接触式、直接式-間接式の選定)。特にセンサの位置が離れている場合、その情報により何らかの修正制御を施す際は、制御遅れ(不安定)3―229訂正動作センシング最適化制御または適応制御システム加工プロセス工作物センシング監視システム工具判断、対策、命令工作機械センシング管理システム停止などの命令図10・45 機械加工における診断、監視、制御システム図10・46 計測内容特徴抽出工作機械複数センサ信号処理決定過程・ニューラルネットワーク・GMDH・Fuzzy必要な情報。.....図10・47 センサフュージョン法の構成に注意を要する。(2) 計測事象の速さ(センサの応答性や周波数特性に関係。予測計測法の必要性)。例えば旋削加工で、表面粗さを計測する場合と寸法変化を計測する場合とでは、その変化の速さは大きく異なる。(3) 作動環境(外乱対策の可能性)。例えばレーザを用いた光学式計測では、エアジェット注入による切りくず、加工液、じんあい等の除去が有効となる。このような状況にあって、切削条件等の変更に対しロバスト性も有する点から、図10・47のセンサフュージョン(sensor fusion)法は一つの有望な計測法である。これは、複数のセンサから得られる情報を統合的に処理することにより、単一のセンサからは得られない新たな機能を作り出そうとする考え方である。単に新しい情報の抽出にとどまることなく、同じ情報でも精度、速度、測定レンジ、柔軟性、さらには信頼性の向上をねらった計測法である。図め決定過程(decision making)に送られ、最終的に必要とする情報が求められる。この方式のキーポイントは、必要とする情報に結びつく有用な信号処理を複数個施し、かつ有効な決定過程を用いて判断することである。それによって検出精度や信頼性等が決まってくる。決定過程としては、ニューラルネットワーク(neural net-work)、GMDH(group method of data handling、発見的自己組3―230加工学・加工機器光学的方法接触抵抗方式アイソトープ方式流体的方法直接的測定法工具-工作物間の接近量検出切削抵抗方式切削動力方式温度測定方式電気抵抗方式振動測定方式音波測定方式AE検出方式表面粗さ測定方式複数センサ方式工具損耗の測定法間接的測定法図10・48 工具損耗の測定法分類図10・49複数センサによる切削状態の認識織化法)、Fuzzyなどがあるが、柔軟性という点でニューラルネットワークが多用されている。以下図10・46に従い、代表的な計測技術について切削および研削加工を中心に概説する。
