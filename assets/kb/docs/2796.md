# 10・3・2 切削加工における計測技術

> **章**: 第10章 加工測定
> **節**: 10・47 で、適宜信号処理された各センサからの出力は特徴抽出のた

機械加工の出力である工作物が最終的に工具を介して制御される。そのときの工具-工作物間の相対的位置関係が加工精度として直接工作物に転写されることから、工具と工作物についての計測技術の開発が積極的に行われてきた。a. 工具系加工の進行とともに工具の刃先形状は摩耗によって変化し、工作物の寸法や表面粗さ等に影響を及ぼす。また工具の破損や欠損は多くの場合突発的に生じ、その検出は可能であっても予測は極めて難しい。損耗の計測法を分類すると図10・48のようにまとめられる。これらには、信号処理の工夫によって破損や欠損の検出が可能な方式もいくつかある。直接的検出法としては、工具の摩耗面に光を照射し、摩耗量と反射率や反射パターンを関係づけて検出する光学式が従来から研究されてきた。しかし、光を用いる方法が環境に弱いことから、必ずしも実用化レベルでは用いられていない。これに代わり摩耗に伴って変化する切削抵抗、切削動力、切削温度など切削プロセスの変化と摩耗量を関係づけた間接的検出法が従来から提案されている。なかでも複数のセンサを用いるセンサフュージョン法が、切削条件の変更や被削材種に対し、ロバスト性のある検出法として興味深い。その例を図10・49に示す。AE(acoustic 加工プロセス系図10・46に示したように加工プロセスに関しても種々の計測項目がある。なかでも加工状態の監視あるいは最適(最適化)制御の目的から、加工中の力を計測する技術の開発が活発に行われている。従来から今日まで開発されてきた計測法を分類すると、おおよそ表10・9のようにまとめられる。最近の例として、工具にひずみゲージを組込み3分力を検出する点では従来方式と変わらないが、図10・52のように工具ホルダの後部に信号増幅および伝送の回路を埋め込んで、ノイズにできるだけ3―232加工学・加工機器影響されないような工夫をしている。そして、ひずみゲージからの信号を赤外線伝送によりリモート検出を行っている。このほか、びびり振動も品質管理の観点から重要な計測項目であり、光学式、音波検出式、振動検出式、変位検出式などがある。d. 工作機械系軸受部、回転部などの過負荷や潤滑不良による異常検出、駆動部分の異常動作、欠陥検出、締結部の異常状態の検出、マシン全体の異常温度や振動の検出など、工作機械の異常状態の検出に関する研究開発技術が比較的多い。インプロセス計測法の理想は、新たにセンサを付加するのではなく、すでに内蔵されている機器や使われている信号を利用し、必要な情報を得ることである。その点で工作機械については動力としてモータが用いられており、モータからの情報(電流、電圧、回転速度など)を適宜処理して異常状態を総合的に診断できる。一方、工作機械の送り駆動系に対する位置決め精度の要求は、マイクロメータからナノメータへと移行しつつある。レーザ干渉計などを用いたオンライン位置計測により、クローズドループ方式による位置制御が実施されている。一例として、図10・53に非球面加工機の場合を示す。直進案内および主軸をそれぞれ静圧油軸受で構成し、DC(direct current)サーボモータと精密ボールねじで駆動される。位置センサにはHe-Neレーザ光を用い、テーブル上の平面反射鏡との間で2往復させ、テーブル移動によるドップラー効果と干渉により0.01 mの分解能を得ている。そのフィードバック制御で0.02 mの位置精度を確保している。
