# 11・1・1 背景(ものづくりの変化:コンカレント、ソリューシ

> **章**: 第11章 三次元造形技術
> **節**: 11・1 概説

ョン)三次元造形は、1980年代以降に盛んになった三次元CAD(com-puter aided design)やCAE(computer aided engineering)解析、医療用のCT(computed tomography)・MRI(magneticresonance imaging)データからそのまま複雑な加工工程なしに、もの、つまりオブジェクトを作り出す技術として生まれた。基本は試作品、つまりプロトタイプ製作であり、簡単かつ迅速(高速)に実施できることからラピッドプロトタイピング(RP: rapid proto-typing)と呼ばれた。
