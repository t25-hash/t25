# 11・1・2 三次元造形とは

> **章**: 第11章 三次元造形技術
> **節**: 11・1 概説

本来の三次元造形は,ホログラムのように三次元オブジェクトを一体で製作する方法(図11・1)と考えられるが,現状の加工技術では困難な内部構造を一体で作り込むことが要求される.こうした加工を実現するキーテクノロジーが,積層造形法(layered manufacturing)である.つまり,三次元オブジェクトを平面で薄くスライスし,これを平面加工技術で成形しながら積重ね・接合・一体化する方法である.薄く切り開くことで閉空間が開放され,既存の加工技術でも加工できるようになる(図11・2).実際の加工は三次元CAD のデータをそのまま取込むのではなく,形状データのみをRP造形法のために定義した三角パッチ[三角形の張り合わせ:STL(standard triangulation language)データという]で表現し直して装置内に取込み,スライスすると同時に成形を行うための種々の処理を行う(図11・3).基本となる加工単位は,11・2・1項で後述する光造形の場合にもレーザの強度分布はガウス分布となるため,固化物は紡すい形状を示して異方性を示し(図11・4),積層時の段差が生じる.光造形や粉焼結の場合には非接触加工であり,レーザによる固化のみで材料図11・1 RP三次元造形システム図11・2 RPシステムの基礎的な移動は伴わないが,溶融物堆積の場合には下地を部分的に溶融して接合するため,堆積物の変形を伴う.軸方向の積重ねによって段差が生じるため,必要に応じて仕上げを行う(図11・5).使用する材料や工法の選択によって,多くの組合せが考えられるが,大きく四つの方法に分かれる.これらの方法を模式図(図11・6)および表11・1に示す.日本国内では微細加工が可能で寸法精度の高い光造形法が主体であったが,肉厚ものの成形に効率的なシート(紙)積層装置が伸びてきた.はん用樹脂であるABS(acrylonitrile butadiene styrene)の成形が可能な溶融樹脂押出法が次に伸び,ナイロンや金属の成形が可能な粉末焼結法や造形速度の大きい粉末固着法が近年伸びてきている(図11・7).ユーザとしては用途別装置の使い分け,同一方法でも材料の種類による装置の多数台使用の傾向がある.
