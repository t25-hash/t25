# 11・1・4 RP の効果および影響

> **章**: 第11章 三次元造形技術
> **節**: 11・1 概説

RPの適用は生産工程へ大きな影響をもたらした。3―237図11・8 RP 造形の用途図11・9コラボレーションによる工程短縮その一つはデータに対する認識の変化である。三次元データであっても健全性が保たれなければ容易に使用できないこと、逆に完全であればデータの使いまわしができることが認識され、CAD/CAM(computer aided manufacturing)/CAE/CAT(computeraided testing)化が進むとともにデータの中身が充実していった。それとともに不具合の前倒し検討や同時設計が可能となり、工程は短縮され、コラボレーションによる前倒し加工などソリューション革命を進めた(図11・9)。最近では既存の加工方式も、積層の概念が普及することも相まって高速化が進んでいるため、RPの概念も広がってきている。
