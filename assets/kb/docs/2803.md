# 11・2・1 光造形法(stereolithography)

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

a. 加工の原理光造形法は、光源(紫外線を発生するためのAr-イオンレーザ、He-Cdレーザ、固体レーザ、紫外線ランプなど)、感光性樹脂、樹脂を選択硬化させるための光学駆動系と積層作業を実現するための移動ステージが構成要素となる。光学駆動系には、光スポットを移動させるための回転反射ミラーによるスキャナ機構、XY プロッタ機構、ポリゴンミラーなどが使われる。造形精度は樹脂特性や光学駆動系の制御精度などに依存するが、2006年度現在で商用システムでは水平方向1 m、積層ピッチ5 m程度の分解能が実現され、研究段階では、数百nmでの構造物も実現されている(図11・10)。光造形法の基本的アイデアの実現は、1981年に小玉秀男が紫外線ランプとフォトマスクを用いて感光性樹脂を硬化積層し、立体模型を作成した研究が最も古いとされている。同時期に世界中で独立に研究が進められ、アメリカでハル(Hall, C.W.)が1984年8月8日に、日本では丸谷洋二が1984年5月23日に、またフランスではアンドレ(Andre썝,J.C.)が1984年にそれぞれ特許出願を行っている。3―238図11・10 微細加工の例図11・11樹脂開発の変遷[提供:シーメット(株)]1987年にはアメリカの3D システムズ社により世界初の製品化(SLA-1)が行われて以降、ラピッドプロトタイピング技術として光造形法が広く認知されるに至った。現在三次元CAD からのデータフォーマットとして普及しているSTL(standard triangulationlanguage,stereolithography language)ファイル形式は、このSLA 装置に入力可能なデータ形式として指定されたことに由来している。b. そり変形、サポート光造形法は、どんな複雑な形状でも造形可能ではあるものの形状データによっては前処理が必要な場合がある。光造形法は液体中に立体を成形するために、成形中に断面形状が突然現れてくるような特徴、すなわちオーバハングした箇所などは造形途中に落下分離してしまい、正しい造形ができなくなる。その対策として、サポート(support)と呼ばれる支柱をその特徴の下部に付加して成形を行う。このようなサポートは、変形しやすい片持ばり等の形状に対して拘束を与える役割も果たすことから、そり変形低減のための効果的な対策の一つとしても用いられている。c. 材料特性光造形法には、光を照射した部分が形状として残るネガ型と呼ばれる感光性樹脂(photopolymer)が使われる。光が照射された領域では、光開始剤が光子を吸収して活性化し、モノマを攻撃するとモノマどうしの連鎖的な反応すなわち重合反応が起きる。重合中に架橋反応と呼ぶ網目構造を形成する反応が起きることで、三次元的な硬化物が形成される。樹脂材料は反応形態の違いにより、開始剤が光子を吸収してカチオンを発生し重合を開始するカチオン重合に基づく樹脂と、開始剤がフリーラジカルを発生して重合を開始するラジカル重合に基づく樹脂とに分類される。一般に、精度の高い部品成形には、カチオン重合に基づくエポキシ系樹脂が用いられ、強度や靱性の必要な部品には、ラジカル重合に基づくウレタンアクリレート系の樹脂がおもに使われる。最近では、前者の重合を用いるオキセタン系樹脂や、後者の重合を用いるイミド系樹脂も開発されている。光造形法の樹脂材料の開発では、成形時の精度の向上と、実製品に使う樹脂の強度、靱性、耐熱性等の機械的特性に近づけることが目標とされている。このことに加えて、現在では多様な樹脂材料への対応としてエラストマ様樹脂や耐水性樹脂等の開発も進んでいる(図11・11)。図11・12 医療用途の例図11・13 耐水性樹脂による部品の例図11・14 粉末積層法(焼結法)原理図d. 用途例自動車用プラスチック部品の試作や、小型化の進む情報機器、家電製品などの筐体、コネクタ等の試作には欠かせない役割を担っている。医療分野では、患者のCT 画像から頭蓋骨等の三次元形状データを再構築して光造形モデルを作成し、手術計画等に利用されることが行われている(図11・12)。試作のみでなく光造形法で作成されたものを実製品へ組込むことも行われ始めている。耐水性のあるイミド系感光性樹脂が開発されたことによるもので、複雑な流路を内部構造としてもつ部品の製造例を図11・13に示す。
