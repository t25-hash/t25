c .逆問題（inverse problem）解析 科学および工学において
は，入力を与えてそれに対する応答を求める順方向の問題が多い．

である．これらの逆問題の解析を行う操作に対する呼び方として

calculation）など問題により種々である ．ここでは走運動に例
