# 11・2・2 粉末積層法

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

a. 焼結法(SLS : selective laser sintering)造形原理は樹脂やワックスなど熱可塑性の粉末表面に、遠赤外線レーザを走査して溶融・焼結させることで所定の形状の固化層を成形し、さらにこの上に粉末層を載せ、次の断面形状の固化を行うことを繰返して三次元モデルを作成する(図11・14)。この手法は1986年からテキサス大学で研究が着手されており、その後、さまざまな樹脂材料や砂材料・金属材料が開発・販売されるようになった。図11・15に材料と造形温度の関係の概略を示す。樹脂材料については、ワックス粉末、ポリカーボネートやナイロン系・ゴム質材料など使用用途により多岐にわたって造形可能であるが、強度等の面からナイロン系の材料がよく使用されている。フェノール樹脂をコーティングした砂材料のレーザによる焼結によって、鋳型を直接成形することができる。自動車部品等で使用する鋳型は大型のものが多いので、焼結用レーザを2台使用することにより造形時間の短縮をねらった専用装置も開発・販売されている。金図11・15 焼結法における材料と造形温度の関係図11・16 金属モデル造形法(樹脂焼結法)属材料については、樹脂バインダをコーティングしたステンレス材料や、樹脂バインダを使用していないブロンズ-ニッケル系材料、スチール系材料が開発されている。以下にそれぞれの金属材料を使用した場合のモデル造形手順と特徴を述べる。. 樹脂焼結法(図11・16)ステンレス粉末に樹脂バインダをコーティングした材料粉末にレーザを走査することで樹脂バインダを溶融固化させてモデル(グリーンパーツ、green parts)を成形する。その後、真空炉中で脱脂焼結させたモデル(ブラウンパーツ、brown parts)にブロンズを溶浸させることで密度・強度を高め、100%金属のモデルを得る。このモデルはおもに射出成形用金型に使用されており、放電加工などすべての機械加工が可能である。造形法の利点は、低出力レーザによりモデル造形が可能であるため、樹脂材料と共通の機械での造形が可能なことである。欠点としては、グリーンパーツ造形後の炉中での後処理(溶浸など)に長い時間とノウハウを必要とすることが挙げられる。. 液相焼結法(liquid phase sintering)材料に樹脂バインダを使用していないブロンズ-ニッケルを主材料とした混合粉を使用し、樹脂焼結より高出力のレーザを走査することで融点の低いブロンズを溶融させ、液相焼結の原理により100%金属のモデルを造形する。この材料の特徴としては、溶融による体積の減少を低く抑えているため、造形後のモデルの密度が低い。そこで密度・強度を高めるため熱硬化性のエポキシ樹脂を含浸させモデルの密度・強度を高める。この造形法の利点としては、造形直後のモデルの強度が高いため取扱いやすいことと、造形後の後処理が簡易で短時間で行うことができることである。欠点としては、最終モデルが100%金属ではなくエポキシ樹脂を含んでいるため金属に比べて熱伝導が悪く、金型として使用した場合の寿命を縮めている場合がある。また、スチール系材料の開発・販売もされ始めた。この材料では、レーザ走査速度をさらに遅くすることで多くの熱を与えることにより、モデルの造形を行う。このモデルは溶融による体積収縮を許しているため、造形後のモデルの密度も高く、必要に応じたショットブラスト等の表面処理のみで高密度・高強度なモデルを得ることが3―239図11・17 粉末積層法(接着法)原理図図11・18 押出し法の原理と事例(ストラタシス社)できる。b. 接着法(3D printing)造形原理は、砂やセラミックス粉末等にインクジェットプリンタの原理でバインダ液(接着剤など)を噴射・塗布することで、粉末どうしを接着して順次硬化・積層させる方法である(図11・17)。この手法はマサチューセッツ工科大学で研究され始め、1992年にセラミックス材料粉末についてDSPC(direct shell production casting)技術として発表されたが、実質的な装置の販売は行われなかった。1996年には同手法ででんぷん粉末に水滴を吹付けてモデルを造形する装置が発表・販売され始めた。この装置で造形したモデルは、精度は低いが安価で早くモデルを造形することができ、材料の廃棄も簡単であるため、意匠検討用モデル造形装置として普及している。応用の一つとして鉄粉を使った焼結法も開発された。
