# 11・2・3 溶融物堆積法(FDM : fused deposition molding)

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

下地(サブストレート)に新たに造形材料を盛り上げていく方法で、基本的には溶融材料(熱可塑性樹脂もしくは金属など)を積み重ねていく方法である。最近では光造形用樹脂や水を冷凍させる方式もある。類似の方式にCVD(chemical vapor deposition、化学蒸着)用化合物ガスをレーザで分解沈積させる方式もある。a. 樹脂押出し法(図11・18)溶融樹脂を押出して成形する方式で、加熱されたはん用樹脂、ABS(acrylonitrile butadiene sty-rene)、最近では強度の大きなポリカーボネートを押出して積層する方式である。この方式は前述の光造形(11・2・1項)や粉末焼結方式(11・2・2項)とは異なり、バルク材に非接触のレーザが入力されるのではなく、溶けた材料が押出されていくため、平面状で交差することができない。そのため、形状外形を描画した後、内部を一筆書きで埋めつくすソフトウェアが必要である。この場合も形状維持を保つサポート材は必要であり、若干配合を変化させたり、特殊な構造をもたせたりして、容易に引きはがせる構造にする3―240加工学・加工機器必要があり、ヘッドが二つあるのが標準である。方向の精度は押出しノズル平面で出す方式である。ABS 樹脂については強アルカリ水に可溶な材料が開発されたので、より複雑形状の成形が可能である。材料がABS 等の実用材であるため、自動車内装品や日用品などの一般プラスチック材料の実用モデル品として用いられることが多い。b. インクジェット法(ink jet printing、図11・19)材料にピエゾ素子などを用いて、下地から離れた地点で噴射された液滴を連続堆積して線状固化物を形成し、積層体を形成する。シングルノズルでベクタ(輪郭)描画するタイプと多数ノズルでラスタ(ライン)描画するタイプがある。基本的に低融点材料が用いられるため、ワックス状の材料が用いられる。精度を要求する目的には、ベクタでサポート材を造形済材料の周囲を三次元的に覆うように噴射し、また造形層の厚さ(軸)制御のためにカッタで切込んでいく方式がある。サポート材を使う二色成形では溶剤可溶性の材料を用いるため、造形品を溶剤に浸漬して膨潤・溶解させることで困難なサポート除去作業が簡略化されている。モデル材料はワックスと同様に扱えるので、精密鋳造のモデルとして使用されることが多い。特に、宝石を支えるための爪の造形で複雑な造形を要求される宝飾関係には欠かせない方法である。スピードが要求される場合には、多数個ノズルを一方向に動かして一挙に造形する方法が使用される。この場合には造形物輪郭形状はベクタ造形とは異なり非連続となる。また、ノズルの使い分けもできないので、サポート材も造形材と同一材料を用いるため、ソーメン状の微細形状となる。そのため、この部分は仕上げを必要とするが、後工程でもともと仕上げの必要な補綴関係の造形では、低コストであり一品生産用に適している。
