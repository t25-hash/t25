# 11・2・5 その他の造形法

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

a. インクリメンタル成形(incremental forming)薄板成形品は、シート成形法で作った紙積層(含浸)品や鉄板積層模型を用いて、通常のプレス成形あるいは対抗液圧成形することも行われているが、一枚の板から直接NC加工で打出す三次元造形法がある。いわゆる、ハンマによる打出し成形を自動化したものである。通常は最終形状を眺めて、全体を順次に変形させていくが、積層造形の加工法、つまり等高線加工でパンチを制御して型なしで板金部品を作る方法である。複雑形状で極端に個数の少ない製品、特に大物成形には向いている。b. 高速切削加工やはり等高線加工を切削に適用して実現した方法である。最近、高速回転スピンドルと高テーブル送りが実現したため、可能になった。当然、曲面加工ではスピードの加減速があるため、直線部分の多い大物加工に向いている。c. 粉末レーザ溶融法基本的には粉末積層法だが、焼結法ではなく敷き詰めた金属粉末を完全に溶融しながら積層する方法である。焼結とは異なり溶融時の収縮による材料移動を伴うため、変形を防止する造形パターンを使用する(図11・22)。d. レーザ溶射法(laser cladding,laser generating)敷き詰めた粉末をレーザ溶融する方法では自由度が低いので、エネルギー源としてレーザを用い、金属粉末をガスで吹込む造形法である。3―241図11・22 骨(右)のスキャンデータからのチタンの造形通常の方法では造形しにくいチタンの大物構造物の加工用プリファームの造形に使用されている。途中で材料を換えることが可能なので、冷却促進用にステンレス金型の冷却配管近傍を熱伝導の大きい銅で造形することもなされている。
