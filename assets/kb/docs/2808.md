# 11・2・6 応用、新展開

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

三次元造形に関する期待は、ものづくりの状況が大きく変ぼうしようとしている21世紀のキーテクノロジーとなっている。おもなポイントは次の三つである。(1) 適用範囲の拡大三次元造形に関しては、日本は世界と大きくその志向が異なっている。日本には下請制度に根差したこだわりが大きく、寸法や仕上げ精度に関する要求が大きすぎる傾向がある。そのため三次元処理への移行の遅れが大きい。その一方で、多品種少量生産への要求に応えられる三次元造形の適用は着実に拡大すると思われる。(2) バーチャル造形ものづくりのベースとなる対象、オブジェクトに対する意味付けが大きく変化してきているので、今後は形状寸法だけでなく構造特性まで含めた定義と実使用環境をシミュレートできるComputerAsisted Manufacuturing を実現するようになる。二次元加工に関する過去の遺産を引きずっている日本とそれ以外の世界での競争は激しくなり、過去のデータの使いまわしができる範ちゅうでは試作は減少するが、全く新しいものづくりにおいては実感の占める割合はさらに増加し、試作そのものは増加すると考えられる。(3) 3D プリンタ三次元CAD 用のプリンタ装置として、小型の装置が伸びてきている。文献(1) 中川威雄・ほか5名、積層造形システム、(1996)、工業調査会。(2) 今中瞑・ほか2名、積層造形技術資料集、(2002)、232-243、オプトロニクス社。(3) Kodama,H.,Rev.
