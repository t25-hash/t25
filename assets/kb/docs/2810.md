# 4・1・1 トライボ材料の選定基準

> **章**: 第4章 トライボ材料
> **節**: 4・1 トライボ材料の種類と選定

空気中・高温環境で使用されるものには，耐熱性・耐食性のあるセラミックスであるBNなどがある．
低温環境中では，PTFEにガラス繊維のような強化材を充てんした高分子系複合材料がよく用いられる．
酸・アルカリなどの腐食性環境中で使用されるものには，ステンレス鋼にMoSをコーティングした材料などがある．
トライボ材料（tribo-material）とは，トライボ要素の機能を発揮し，かつ目的に応じた低摩擦，耐摩耗，耐焼付き，耐疲労等の特性をもつ材料の総称である．
トライボ材料には，上記の特性が潤滑油・グリースの存在下で得られるものと，それらの有無によらず得られるものとがある．
それらをおもな特性により分類すると，表Ⅱ-4･1，Ⅱ-4･2のようになる．
