# 4・1・3 使用環境による選定

> **章**: 第4章 トライボ材料
> **節**: 4・1 トライボ材料の種類と選定

トライボ材料をその使用環境との関係から選定する際には，材料
の劣化が起きず，摩擦・摩耗特性が良好であることを基準に選定す
ることが必要である．
a.材料の劣化防止 材料の選定に当たってその使用可能温度
範囲を確かめることが必要である．特に，トライボ材料としてよく
使われる軸受鋼（SUJ）は，通常の鉄鋼材料と違い低温焼戻しによ
って硬度を上げているので，100℃程度のあまり高温といえない温
度でも寸法変化などの不具合を生じる．したがって，使用温度によ
っては高温での焼戻しをするか，高温用の軸受鋼（M50）を使用
する必要がある．
また，トライボ材料の使われ方として異種の材料との接触部分を
もつ場合がある．このような場合には，海水などの電解質中では自
然電位の差によって局部的に電池を作り，腐食を起こすことがあ
る．これについては，絶縁などの腐食対策と材料選定を総合的に考
える必要がある．
b.摩擦・摩耗特性 部材が損傷なく滑るためには，次のいず
れかの条件が満たされている必要がある．
① 部材間にその材料よりせん断強度の小さい第三の物質が存在
する．
② 材料表面のせん断強度が変化して内部のせん断強度より小さ
くなる．
③ 材料の内部に滑りやすいせん断強度の小さい層を多数もって
いる．
①の条件の例には，油やグリースなどの潤滑剤を使用して材料の
潤滑をする場合が相当する．その場合，使用環境と使用する潤滑剤
の適合性が問題になるので，第Ⅱ部第2章「潤滑剤」を参照して適
切な潤滑剤選定を行う必要がある．一般に，極低温や高温など油や
グリースなどによる潤滑が難しい環境の場合には，油に代えて固体
潤滑剤を第三の物質として使用したり，②③の条件を満たす材料を
選定することになる．
表面のせん断強度が小さい例として，金属の表面に酸化物ができ
て金属どうしの凝着を妨げる場合が相当する．この場合には雰囲気
とトライボ材料の反応を考慮に入れた材料選定が必要である．一見
無潤滑の状態であっても，通常の大気中では材料の表面に酸素が吸
着することによって潤滑された状態となり，材料のトライボロジー
特性は雰囲気によって大きな影響を受ける．金属材料の場合，接触
面圧と速度が十分小さい条件では微細な酸化摩耗粉を生じるマイル
ド摩耗であっても，同条件で雰囲気圧力が低下していくと大型の金
属光沢を有する摩耗粉を生じ，摩擦面もあれるシビヤ摩耗に変化す
る．したがって，高真空環境のもとでは固体潤滑剤の使用が必要に
なる．
一方，環境によっては雰囲気と材料の反応によって摩擦特性が向
上することもある．水中でSiC（炭化けい素）を使用した場合に摩
擦表面に水和物ができて低摩擦になる例や，液体ナトリウム中で
SUS304どうしの摩擦面にナトリウムとクロムの酸化物が生成し
て低摩擦になる例などが報告されている．
滑りやすい層をもつ材料の例として，黒鉛のように層状の組織を
もち，層間での分子間結合が弱く滑りやすいものがある．こうした
層状組織を有するトライボ材料としては，黒鉛，BN（窒化ホウ
素)，二硫化モリブデンなどがある．いずれもほかの金属や樹脂材
料などと複合化されてトライボ材料として使用されたり，固体潤滑
剤として表面へコーティングなどして使用されるが，黒鉛とBN
は単独で部材として使用される場合がある．
黒鉛とBNの場合，雰囲気気体が層間の結合を弱めて滑りやす
くなるために自己潤滑特性が高まることが知られている．したがっ
て，真空中などの雰囲気圧力が小さい環境のもとでは黒鉛やBN
の潤滑特性はあまり良くない．特に，黒鉛の場合には雰囲気中の水
分の影響が大きく，雰囲気大気の露点温度が低い場合には大きな摩
耗を生じることがある．反対に，二硫化モリブデンや二硫化タング
ステンでは雰囲気気体が層間の滑りを阻害するように働くと考えら
れており，真空雰囲気のほうが優れた潤滑特性を示す．
金属でも，結晶構造が六方晶系のものは体心立方や面心立方のも
のに比較するとc軸の距離が長くその面で滑りやすいために摩擦が
小さい傾向にある．六方晶系のコバルトをベースにした合金はトラ
イボ材料としてよく使われ，炭化物を析出させて硬度を高くしたス
テライトなどの合金が無潤滑の滑り面に使われることが多い．
