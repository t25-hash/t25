# 4・2・1 金属材料（metallic material ）

> **章**: 第4章 トライボ材料
> **節**: 4・2 硬質材料（hard material ）

金属材料の耐摩耗性は一般に硬さと靱性に依存するが，これらの
性質は必ずしも両立しないため，用途に応じてバランスをとってい
る．硬質金属トライボ材料を使用する代表的な機械要素には，転が
り軸受ならびに歯車があり，それぞれJISで規定された合金鋼を
一般的に使用している．
転がり軸受は軌道面と転動体との小さな接触面で大きな荷重を受
けながら高い精度を保って回転する必要があるため，硬さ，耐摩耗
性，ならびに転がり疲れ強さが求められている．1 %C-1.5%Crの
高炭素クロム軸受鋼（SUJ2）が最も多く使用されており，完全硬
化処理が施されている．高炭素クロム軸受鋼の成分は古くから変わ
っていないが，製鋼技術の発展とともに鋼中の不純物量は飛躍的に
減少し，転がり軸受の寿命はこの30年間で30倍以上に向上した．
耐衝撃性が必要な転がり軸受では，表面から適当な深さまで浸炭し
て高い硬さとし，比較的硬さの低い芯部を形成した，硬さと靱性を
兼ね備えたニッケル-クロム-モリブデン鋼（SNCM815）などの肌
焼鋼が使用されている．耐食性が求められる場合には，マルテンサ
イト系ステンレス鋼（SUS440C）などが使用されている．200℃
以上の高温で使用される転がり軸受では，高速度鋼（X75
WCrV18-4-1）や耐熱浸炭鋼（81MoCrV42-16）が使用されてい
る．表面のみを硬化させる方法として高周波焼入れが利用されてお
り，大形軸受や自動車のホイールユニット軸受などでは機械構造用
炭素鋼（S55C）が用いられている．
歯車には歯面の耐摩耗性および転がり疲労強度とともに歯車がか
み合う力を許容できる歯元強度が必要である．一般的な炭素鋼を調
質して用いる場合や，焼入れして硬さを高くして使用する場合があ
る．焼入れ鋼としては，各種クロム鋼（Cr鋼，Cr - Mo鋼，Ni -
Cr -Mo鋼）を使用している．
ブレーキも重要な機械要素であるが，自動車や在来線の鉄道車両
のブレーキのロータやディスクには片状黒鉛鋳鉄（FC250，
FC280）が，新幹線では耐熱亀裂性に優れた鍛鋼ディスクが使用
されている．
粉末ハイス（SKH10）は熱間静水圧焼結（HIP）技術を利用し
て製造されるが，W，Mo，V，Crなどを含むため耐熱疲労性や耐
摩耗性に優れる．焼結金属の応用例として，材料自体の摩擦係数が
低く耐摩耗性を有する含油焼結合金や固体潤滑剤複合焼結合金があ
る．含油焼結合金にはFe系やCu系があり，気孔部に潤滑油を含浸
させる．真空中や高温雰囲気などの潤滑油が使用できない場合は，
焼結合金にMoSあるいはWSが固体潤滑剤として配合される．
表面のトライボロジー特性を改良する技術の一つに金属溶射処理
がある．溶射は溶融，微細化された金属を母材に高速で吹付けて被
覆層を形成させる技術である．耐摩耗性ならびに耐焼付き性の改善
を目的に，Ni -Al ，Mo，自溶性合金（Ni - Cr -B-Si -C）などが使
用されている．
