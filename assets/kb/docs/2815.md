# 2・1・1 ねじの用途

> **章**: 第2章 締結要素
> **節**: 2・1 ねじ

ねじには多くの用途があるが,一般には次のものが挙げられる.1機械または構造物の部分と部分とを結合したり,それら相互間の動きを止めるねじ.例えば六角ボルト,六角ナット,小ねじ,止めねじなどがあり,締付けによる軸力の発生により目的を達する場合が多いので締結用ねじ(threaded fas-tener)と総称される.2管と管,または機械と管とを継ぐねじ.例えば管継手などに用いられる管用ねじ(pipe thread).3機械または構造物の部分を本体に対して直線的に移動させるねじで,一般に送りねじ(feed screw)といわれるが,工作機械などで特に送り精度を必要とする場合に親ねじ(leadscrew)という.4微細な位置調節.例えばデバイダやコンパスの脚の開きを調節するねじ.5微小寸法を拡大指示するねじ.例えばマイクロメータのねじ.6大きな力の発生.例えばプレス,万力のねじ.7大きな力の発生と位置調節を兼ねたねじ.例えばジャッキや弁開閉用のねじ.8張線の張力の加減.例えばターンバックルのねじ.9互いに,はめあわされたおねじとめねじの一方を直線的に移動させることにより他方を回転させる機能をもったねじ,など.
