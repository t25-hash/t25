# 2・1・4 トルク法によるねじの締付け

> **章**: 第2章 締結要素
> **節**: 2・6 に示す。図中に細円筒=1とあるのは、ボルトのねじ

a. 締付け軸力の最大値例えばボルト・ナットによるねじ締結体があって、ナットにトルクを加えることによってボルトの軸力を増加していった場合、ボルト軸部には軸力のほかねじりトルクが作用し引張応力とねじりせん断応力の組合せ応力状態となるので、降伏締付け軸力は図I-2・7に示すようにねじの単軸引張の降伏軸力より20%程度(ねじ面の摩擦係数により異なる)小さい軸力で降伏する。図I-2・7はM 10、強度区分10.9、締付け長さ45mmのボルト・ナットをボルトの塑性域(伸び1mm)まで締付けた後、トルクレンチを離し軸方向外力を作用させた場合のボルトの伸びと軸力の関係を示す。この図からわかるように塑性域まで締付けても外力の作用に対して弾性的な追加伸びが一般に期待されるので、トルク法による締付け軸力の最大値は式(I-2・12)で与えられる。は締付け係数(tightening factor)といわれ、普通の表面状態をもつ鋼製のボルト・ナットおよび被締付け物表面に潤滑油を塗布して締付けを行った場合の概略値は、VDI2230に与えられた表から抜粋して表I-2・2に示す。この予張力を得るために加えるべき締付けトルク(tighteningtorque)の目標値は、式(I-2・2)によるトルクにナット座面における摩擦トルクを加算すればよい。メートルねじでは、α′≒α=30°,μtanβcos α′≪1であるから、式(I-2・21)は次のように簡単化される。すなわち、全トルクのうち約40%がねじ面の摩擦、約10%がねじのリード、約50%がナット座面の摩擦に起因していることがわかる。なお、ナットを緩める方向に回転する際のトルクは式(I-2・21)および式(I-2・22)においてβの符号をマイナスにした式で与えられる。したがって、式(I-2・23)が成り立つ場合、緩めトルクはの値の約80%となる。
