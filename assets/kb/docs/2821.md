# 2・1・7 ねじ締結体の強度設計

> **章**: 第2章 締結要素
> **節**: 2・6 に示す。図中に細円筒=1とあるのは、ボルトのねじ

ねじ締結体に静的または動的な外力が作用した際に、締結体に要求される機能(被締付け部材どうしが接合面で分離したり、接合面内でずれたり、接合面から液体が漏れたりしないことなど)を満足する締付け軸力を発生できるねじの寸法と強度を決定することが、ねじ締結体の強度設計である。a. 外力の推定機械構造物の強度設計において、作用する外力を正確に見積もることは、大抵の場合非常に困難な仕事の一つである。推定した外力が実際より大きすぎると設計製作された製品は強度上安全であるが、経済性が損なわれる。また逆に小さすぎると機能を満足しないで、破壊に至ることにもなる。使用中に作用する外力を正確に見積もることは、設計者にとって最大のノウハウである。設計の際、作用する外力の推定が困難な場合には、設計に先立って簡単なモデルにより、ひずみゲージや荷重計を用いて実験を行い、外力の大きさを測定しなければならない。ねじ締結体の設計においても締結体に作用する外力の推定が最初の重要課題である。これらの値を決めるための解析は大抵の場合大変困難であるので、ここではこれらの値は既知とする。b. 計算による検討事項組立時にナットまたはボルト頭部を締付け工具により締付けることによりボルト軸部は引張応力とせん断応力の組合せ応力状態になっている。さらに使用中に外力を受けることにより応力を追加される。ボルトはいくつかの切欠部があるので、その応力集中も考えて応力を見積もることは難しいので、通常ねじの有効断面積または軸部の最小断面積(伸びボルトの場合)を用いて公称応力で設計を行う。4―13第I部機械要素必要なボルトの寸法を計算により求める際には、外力の状態を既知として、以下の検討を行う。1ボルトの強度区分。2接合面のへたりにより予張力の低下が生じること。6繰返し荷重の際の疲労強度。7被締結部材の座面強度。上記のファクタは、ねじの寸法を決定するための基本公式の構成要素であり、基本公式は式(I-2・18)から次式のようになる。使用中に振動負荷が作用する場合には繰返し応力が問題になる。以上の式(I-2・39)~(I-2・43)を満足するようにねじの寸法と強4―14機械要素・トライボロジー度区分を選択する。安全率をどの程度にするかは設計技術者に任される。設計手順の詳細については専門書[文献(4)(8)(9)(20)など]を参照されたい。c. 遅れ破壊について所定の予張力のもとに締付けられたボルトがある時間(短いもので1時間、長いもので10年の例がある)経過後、突然破断する現象を遅れ破壊(delayed fracture of screwthread)という。静的引張破断のような塑性伸びを伴わず、常温でも破壊するのが特徴である。ボルト内に侵入した水素による鋼の水素脆性が大きな原因であるとされ、ボルト内にもともと水素が含まれていた場合と、ボルト締付け後外部から水素が侵入した場合とがある。引張強さσ=1200Nmm以上の材料によるボルトだけに発生しているので、それより低い強度のボルトを用いることで防止できる。
