# 2・1・7 ねじ締結体の強度設計

> **章**: 第2章 締結要素
> **節**: 2・6 に示す。図中に細円筒=1とあるのは、ボルトのねじ

ねじ締結体に静的または動的な外力が作用した際に，締結体に要
求される機能（被締付け部材どうしが接合面で分離したり，接合面
内でずれたり，接合面から液体が漏れたりしないことなど）を満足
する締付け軸力を発生できるねじの寸法と強度を決定することが，
ねじ締結体の強度設計である．
a.外力の推定 機械構造物の強度設計において，作用する外
力を正確に見積もることは，大抵の場合非常に困難な仕事の一つで
ある．推定した外力が実際より大きすぎると設計製作された製品は
強度上安全であるが，経済性が損なわれる．また逆に小さすぎると
機能を満足しないで，破壊に至ることにもなる．使用中に作用する
外力を正確に見積もることは，設計者にとって最大のノウハウであ
る．設計の際，作用する外力の推定が困難な場合には，設計に先立
って簡単なモデルにより，ひずみゲージや荷重計を用いて実験を行
い，外力の大きさを測定しなければならない．
ねじ締結体の設計においても締結体に作用する外力 の推定が
最初の重要課題である．この外力は締結体の構造により，それぞれ
のねじ締結箇所に軸方向外力，軸直角方向外力，曲げモー
メント，負荷トルク などを作用させる．これらの値を決め
るための解析は大抵の場合大変困難であるので，ここではこれらの
値は既知とする．
b.計算による検討事項 組立時にナットまたはボルト頭部を
締付け工具により締付けることによりボルト軸部は引張応力とせん
断応力の組合せ応力状態になっている．さらに使用中に外力を受け
ることにより応力を追加される．ボルトはいくつかの切欠部がある
ので，その応力集中も考えて応力を見積もることは難しいので，通
常ねじの有効断面積または軸部の最小断面積（伸びボルトの場合）
を用いて公称応力で設計を行う．
必要なボルトの寸法を計算により求める際には，外力の状態を既
知として，以下の検討を行う．
① ボルトの強度区分．
② 接合面のへたりにより予張力の低下 が生じること．
③ 組立時，予張力 が外力の分力＝1－Φによって
減らされること．
④ 密封機能，接合面における片側分離の防止，緩みの防止など
のためには最小締付け力 が必要であること．
⑤ 組立時の締付け方法によって，組立予張力 のばらつき範
囲が異なること（締付け係数＝)，を考慮する
（図Ⅰ-2･10)．
⑥ 繰返し荷重の際の疲労強度．
⑦ 被締結部材の座面強度．
