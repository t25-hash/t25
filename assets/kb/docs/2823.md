# 2・2・1 キー

> **章**: 第2章 締結要素
> **節**: 2・2 キー,スプライン

キー(key)は,歯車,プーリなどの回転体や種々の軸継手を軸に固定し,軸とそれらの機械要素との間のトルク伝達を可能にするための機械要素の一つである.通常,軸および固定される要素のハブに設けられたキー溝(key way)にキーをはめ込むことにより両者を固定する.キーの分類は,その形状によるものと軸との組付け状態によるものがある.形状による分類には,JIS B 1301:1996で規定されている,平行キー(square key),こう配キー(taper key),および半月キー(Woodruffkey)がある(表I-2・9~I-2・11).このJIS B 1301は鋼製のキーを規定したもので,素材の引張強さは600N/mm以上が要求されている.表I-2・9~I-2・11に示したように,平行キーにはねじ用穴のあるものとないもの,こう配キーには頭(gibhead)のあるものとないもの,そして半月キーには丸底と平底がそれぞれ規定されている.さらに,平行キーには,その端部形状が異なる三つの種類;両丸形,両角形,および片丸形がある.なお,こう配キーに付けられた頭はキーを抜くときの利便性を考慮したものであるが,その突起部は危険を伴うものであるので使用に際しては十分な注意が必要である.JIS B 1301では,同時にキー溝についても規定されており,キーとキー溝のはめあいを適当に選定することにより,キーと軸・ハブの結合状態を滑動形,普通形,あるいは締込み形に設定できる.滑動形とは,軸とハブとが相対的に軸方向に滑動できる結合で,適用されるキーは平行キー[このようなキーを特に滑りキー(slid-ing key)と呼ぶ]である.普通形とは,軸に固定されたキーにハブをはめ込む結合で,適用されるキーは平行キーと半月キーである.締込み形は,軸に固定されたキーにハブを締込む結合,あるいは組付けられた軸とハブにキーを打込む結合で,三つのキーのいずれも適用できる.しかし,キーを打込むことにより,軸と回転体の同心度が必ず悪化するので,ごく低速のものにしか使用できない.なお,平行キーを軸に固定する場合,はめあいによらず,ねじ止めする場合もある.軸との組付け状態による分類では,上述のように軸にキー溝を付けて組付ける沈みキー(sunk
