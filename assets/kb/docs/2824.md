# 2・2・2 スプライン

> **章**: 第2章 締結要素
> **節**: 2・2 キー,スプライン

スプライン(spline)は,多数のキーが軸と一体化したものと考えられ,軸の外周に等間隔に設けた歯を穴に設けた歯溝にかみあわせることによりトルクを伝達する.軸直角断面の両歯形が,平行直線のものを角形スプライン(straight-sided spline),歯形がインボリュート曲線のものをインボリュートスプライン(involutespline)と呼ぶ.角形スプラインは,JIS B 1601:1996で規定されている.表I-2・12に基準寸法,また表I-2・13に軸と穴のはめあいを示す.インボリュートスプラインは,元来,基準圧力角20°の自動車用スプラインとして運用されてきたが,ISO準拠により,現在はJISB 1603:1995で規定されており,基準圧力角は,30°,37.5°,および45°の3種類とされている.規格に挙げられているモジュールは,0.25,0.5,0.75,1,1.25,1.5,1.75,2,2.5,3,4,5,6,8,および10の15種類で,基準圧力角30°および37.5°の場合はこのうち0.5~10,45°の場合は0.25~2.5を規定している.いずれも相対する歯面の隅肉曲線が個別に歯底円につながっている平底と相対する歯面が一つの隅肉曲線でつながっている丸底が規定され表-2・13角形スプラインの穴および軸の寸法公差(JIS spline)と呼び,JIS B 1193:2004で規定されている(図I-2・14).また,ピッチを細かくすることにより,より細かな軸と穴の位相合せを可能としたものをセレーション(serrations)と呼ぶ.スプライン結合部では,両軸の同心度が保たれないことが多く,その場合には,スプラインが片当たりして一方の端部のみで負荷を受けることになる.このため,面圧の上昇,角当たり等に起因する損傷を引き起こすこともある.これを避けるため,スプラインの歯厚を長手方向にクラウニング修整することも行われている.止め輪(retaining ring,snap ring)は,軸あるいは穴に種々の部品を組付ける場合,その部品を軸方向に固定したり位置決めするために用いられる機械要素である.JIS B 2804:2001では,溝にはめ込んで用いるもので形状が“C”に似ているC形止め輪(偏心形および同心形)と形状が“E”に似ているE形止め輪,そして溝を介さないグリップ止め輪を規定している.このうち,E形止め輪とグリップ止め輪は軸用のみで,C形止め輪のみ軸用に加えて穴用も規定されている.表I-2・14~I-2・19にこれらの基準寸法を示す.止め輪はその弾性変形を利用してはめあわされる.したがって,ばねとして十分な強度が要求されるため,硬さ40ないし53HRCのばね鋼や帯鋼などを用いるように規定されている.また,その形状は,ペンチなどで広げて着脱が容易になるようなものになっている.(a)
