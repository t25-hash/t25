# 2・4・1 ピン

> **章**: 第2章 締結要素
> **節**: 2・4 ピン、コッタ

ピン(pin)は、種々の部品の組付けや位置決めに用いられる機械要素である。頻繁な着脱はピンの摩耗を引き起こすが、このような場合はテーパの付いたピンが用いられる。ピンは、通常、せん断応力を受ける。これに加え、ピンのはめあいが締まりばめのときには圧縮応力が、すきまばめのときには曲げ応力が同時に作用する。また、脱落防止効果を上げた先割りテーパピン(taper pin withsplit;B 1353:1990)、スプリングピン(spring pin;B 2808:2005)、割りピン(split pin;B 1351:1987)などがある[図I-2・18(c)~(e)].先割りテーパピンおよび割りピンは、装着後先を折り曲げて脱落を防止する。スプリングピンは、ピンのばね効果で脱落を防止する。これら以外に、平行ピンにねじ穴を付けためねじ付平行ピン(parallel pin with internal thread;B 1359:1990)、平行ピンに熱処理を施したダウエルピン(Dowel pin;B 1355:1990)、テーパピンにねじ穴やねじを付けたねじ付テーパピン(taper pin withthread;B 1358:1990)などがある。止め輪呼び基準寸法許容差基準寸法許容差約1.9+0.05-0.080.62.851.42.17.444.13±0.043.8+0.054.75±0.082.22.91.311.076.49
