# 2・4・2 コッタ

> **章**: 第2章 締結要素
> **節**: 2・4 ピン、コッタ

コッタ(cotter)とは、板状のくさびで、片側こう配のものと両側こう配のものがある。一般に、軸とボスを固定するために、図I-2・19のように用いられる。板状にすることで、ピンよりも大きな軸荷重に耐えることができる。また、テーパがあるので、繰返し着脱することもできる。コッタの強度は、締まりばめとなるので、せん断応力に加え、圧縮応力を考慮して評価しなければならない。ねじ、リベットなどの機械要素を用いた締結は、いわば点による締結ということができる。それに対して、溶接は線による締結、接着は面による締結である。
