b.溶接継手の種類 基本的には，図Ⅰ-2･20(ａ)に示した，
突合せ継手（butt joint)，重ね継手（lap joint)，T継手（T
joint)，角継手（corner joint)，およびへり継手（edge joint）の5
種類に分けることができる．その派生形として，十字継手（cruci-
form joint)，せぎり継手（joggled lap joint)，当て金継手
（strapped joint）などがある［図Ⅰ-2･20(ｂ)］．
c .溶接部の種類 溶接部の種類には，溶接する母材間に開先
（groove）と呼ばれる溝をつけて，その開先部に溶接金属を溶かし
接合する開先溶接（groove welding)，ほぼ直行する二つの面を溶
接する際，溶接金属をほぼ三角形断面になるように接合するすみ肉
溶接（fillet welding)，重ね合わせた母材の一方に穴をあけ，それ
を溶接金属で満たして接合するプラグ溶接（plug welding)，そし
て重ね合わせた母材の一方に溝を施し，その溝に隅肉溶接を行うス
ロット溶接（slot welding）の四つがある（図Ⅰ-2･21)．プラグ溶
接およびスロット溶接は，通常，一方の面からの溶接が困難な場合
に用いられる．なお，図Ⅰ-2･22に種々の開先を示す．
d.溶接継手の選定 溶接継手の選定に際しては，(１)構造物
の種類（重要度）と溶接継手の形状（切欠・応力集中)，荷重の種
類と大きさ，および継手の特性，(２)溶接方法，板厚，材質，(３)
溶接変形・残留応力，材質の熱的変質，溶接割れ，および溶接の難
易度，(４)継手の準備および実際の溶接に要する費用などを考慮す
る必要がある．
2・5・2接着継手
接着（adhesive bonding）とは，接着剤を媒介とし，化学的も
しくは物理的な力，あるいはその両者によって二つの面が結合した
状態であり，接着剤（adhesive）とはそのような結合を実現でき
る物質をいう．この接着の本来の意味どおりの接着を構造接着と呼
び，長期間破壊することなく，その最大破壊荷重に近い応力を加え
ることのできる接着と定義されている．

a.接着剤 接着剤をその主成分で分類すると以下のようにな
る．まず，大きく有機質系と無機質系に分けられる．有機質系はさ
らに天然系と合成系に分けることができる．天然系には，デンプン
系，タンパク質系などがある．一方，合成系には，樹脂系，エラス
トマー系，混合系がある．この樹脂系には熱硬化性のものと，熱可
塑性のものがある．
一方，硬化方法で分類すると，室温硬化性，熱硬化性，ホットメ
ルト形（加熱して用い，冷えると接着できる)，感圧性，再湿性の
ものに分けられる．室温硬化性のものは，さらに，溶剤乾燥形，触
接着強さの大きな構造接着用接着剤には，主成分で見ると，熱硬
化性樹脂のエポキシ樹脂やフェノール樹脂，エラストマのポリアミ
ドやポリサルファイドなどが用いられている．また，硬化方法で見
ると，従来は熱硬化性のものが多かったが，最近は室温硬化性のア
クリル系やポリウレタン系の接着剤なども出てきている．
b.接着工法 接着の手順は，被着剤の表面処理，接着剤の塗
布，および接着剤の乾燥・硬化に分けることができる．被着剤の表
面処理には，洗浄，研磨，化学的処理，物理的処理，およびプライ
マ処理があるが，被着剤の種類やその組合せ，要求される接着強さ
などによってこれらを使い分け，あるいは組合わせて行う．接着剤
の塗布方法には，ヘラ，棒，油差し，ローラなどによる手作業によ
る方法，スプレーによる方法，ホイルコータ，エアナイフコータ，
ブラシコータなどの塗布用装置を用いる方法がある．接着剤の乾
燥・硬化は，スチームシリンダや熱風乾燥装置などを用いる熱によ
る方法，マイクロ波，赤外線，紫外線などを用いる電磁波による方
法，および電子線による方法がある．
