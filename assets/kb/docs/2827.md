# 2・5・1 溶接継手

> **章**: 第2章 締結要素
> **節**: 2・5 溶接継手、接着継手

その選択肢はかなり広く、溶接の種類、継手の種類、および溶接部の種類を選定し、これらの組合せで決定される。a. 溶接の種類溶接中に機械的圧力を加えず、母材あるいは溶加材(filler metal)を種々の熱源によって溶融させ接合する融接(溶融溶接の略;fusion welding)の代表例であるアーク溶接(arc welding)が最も一般的な溶接方法である。これは、文字どおり、アークの発熱を利用した溶接で、シールドガス(shieldinggas)や被覆剤(coating flux)により大気から溶接部を遮断し、大気混入の防止や酸化防止を図った被覆アーク溶接(shieldedmetal arc welding)、炭酸ガスアーク溶接(COarc welding)などがある。大電流により発生する熱を利用し、同時に加圧しながら溶接する方法が抵抗溶接(resistance welding)である。その代表例に、重ね合わせた母材を電極先端で挟み、比較的小さな部分に電流を集中して局部的に加熱し、同時に加圧して溶接するスポット溶接(spotwelding)がある。このほかに、ローラ電極を用いて加圧および通電しながら電極を回転させ、継手を連続的に接合するシーム溶接(seam welding)などがある。ろう(450°C以上の高い融点をもつ溶加材;brazing fillermetal)、あるいは、はんだ(450°C以下の低い融点をもつ溶加材;solder)を用いて、母材をできるだけ溶融しないで接合する方法をろう付け(brazing)あるいははんだ付け(soldering)と呼ぶ。例えば、加熱源により、アークろう付けやトーチろう付けなどと、あるいは工程により、ディップはんだ付けやリフローはんだ付けなどと区別される。これら以外に、接合面を押付けながら相対運動させ摩擦熱を発生させて接合する摩擦圧接(friction welding)、母材を密着させその融点以下の温度下で、塑性変形が生じない程度に加圧し、接合面間の原子拡散を利用して接合する拡散接合(diffusion welding)などがある。b. 溶接継手の種類基本的には、図I-2・20(a)に示した、突合せ継手(butt joint)、重ね継手(lap joint)、T 継手(Tjoint)、角継手(corner joint)、およびへり継手(edge joint)の5種類に分けることができる。その派生形として、十字継手(cruci-form joint)、せぎり継手(joggled lap joint)、当て金継手(strapped joint)などがある[図I-2・20(b)].c. 溶接部の種類溶接部の種類には、溶接する母材間に開先(groove)と呼ばれる溝をつけて、その開先部に溶接金属を溶かし接合する開先溶接(groove welding)、ほぼ直行する二つの面を溶接する際、溶接金属をほぼ三角形断面になるように接合するすみ肉溶接(fillet welding)、重ね合わせた母材の一方に穴をあけ、それを溶接金属で満たして接合するプラグ溶接(plug welding)、そして重ね合わせた母材の一方に溝を施し、その溝に隅肉溶接を行うスロット溶接(slot welding)の四つがある(図I-2・21)。プラグ溶接およびスロット溶接は、通常、一方の面からの溶接が困難な場合に用いられる。なお、図I-2・22に種々の開先を示す。d. 溶接継手の選定溶接継手の選定に際しては、(1)構造物の種類(重要度)と溶接継手の形状(切欠・応力集中)、荷重の種類と大きさ、および継手の特性、(2)溶接方法、板厚、材質、(3)溶接変形・残留応力、材質の熱的変質、溶接割れ、および溶接の難易度、(4)継手の準備および実際の溶接に要する費用などを考慮する必要がある。
