# 2・6・1 リベットの種類

> **章**: 第2章 締結要素
> **節**: 2・6 リベット

リベットには、大きく分けて、一般構造用継手に用いられるソリッドリベット(solid rivet)、軸端側に穴や割りが入った比較的軽負荷で大量自動締結に用いられるチューブラリベット(tubularrivet)とスプリットリベット(split rivet)、意匠を考慮し出っ張りをなくしたコンプレッションリベット(compression rivet)、そして一方からだけしかリベット打ち作業ができない場合に用いられるブラインドリベット(blind rivet)の5種類がある(図I-2・4―23第I部機械要素24)。このうち、ソリッドリベットは、冷間成形用と熱間成形用について、JIS B 1213:1995およびJIS B 1214:1995で、頭部形状で分類、規定されている(図I-2・25)。また、少し古いが、軸端の穴が浅いセミチューブラリベット(これに対して穴の深いものをフルチューブラリベットと呼ぶ)の規格JIS B 1215:1976も残っている。
