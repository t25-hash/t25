# 4・2・5 内接式遊星歯車減速機

> **章**: 第4章 伝動要素
> **節**: 4・2 歯車伝動装置

平歯車を利用した差動減速機構で，その歯形曲線にサイクロイド
系曲線を使用していることからサイクロ減速機（Cyclo drive）と
命名されたが，この名称は登録商標となっている．
この減速機は原理的には，歯数差1枚の内接式遊星歯車機構と出
力取出用内歯車機構との二つの機構を組合わせたものである（現在
では歯数差2枚以上の多数歯数差のものもある)．
図Ⅰ-4･49のような簡単な遊星歯車装置では，遊星歯車および固
定太陽内歯車の歯数をそれぞれ および ，偏心軸および遊星歯
車の回転速度をω （公転）およびω （自転）とすれば，一般に，
ωω＝1－＝－－ (Ⅰ-4･44)
となる．上式において，固定太陽内歯車の歯数と遊星歯車の歯数の
差を1枚すなわち－＝1とすれば最も大きな減速比が得られ
る．そこで－＝1を式(Ⅰ-4･44)に代入すれば
ωω＝－1 (Ⅰ-4･45)
となり，減速比は遊星歯車の歯数に等しく，符号よりわかるように
その回転方向は偏心軸回転方向と逆になる．このような機構を内接
式遊星歯車機構と呼ぶ．
いま，固定内歯車の歯形に円弧歯形を使用すれば図Ⅰ-4･50のよ
うに遊星歯車の歯形はエピトロコイドとなる．これらを歯形曲線に
使用すればインボリュート曲線を使用する場合に生じるような歯先
干渉なしに歯数差1枚の内歯車が得られ，しかもすべての歯は同時
に接触し，理論的に同時かみ合い歯数は内歯車の歯数の2分の1と
図 -4･51 出力取出用内歯車機構
いう大きなものとなり，極めて有利である．
遊星歯車は自転と公転とを同時に行っており，これから減速され
た自転のみを取出す機構が出力取出用内歯車機構である．すなわち
図Ⅰ-4･51のように偏心軸中心と同心円上にある等ピッチのピン，
および遊星歯車と同心円上にある等ピッチでしかも相手のピン径よ
り中心距離（偏心量）の2倍だけ大きい直径の円孔とをかみあわ
せ，遊星歯車の自転のみをピンにより取出すものである．図Ⅰ-4･
50と図Ⅰ-4･51を組合わせた内接式遊星歯車減速機の機構・構造の
一例を図Ⅰ-4･52に示す．
この装置の利点として，以下のことが挙げられる．(１)１段で大
きな減速比が得られる．現在1段で最大180分の1程度までのもの
が生産されている．また，多段形に組合わせた場合は1日1回転
（数百万分の1）という大きな減速比も容易に得られる．(２)機構
上動力伝達部分に滑り接触がないので効率がよく，運転が円滑で振
動騒音が少ない．(３)極めて小形で，また軸が同心であるので取付
け取外しが容易であり，据付に場所を要しない．(４)同時かみ合い
歯数が大きく，また歯の折損がない滑らかな歯形をしているのでオ
ーバロードやショックロードに強い．(５)分解・組立は非常に簡単
で保守も容易である．(６)ノーバックラッシ機構を簡単に構成でき
る．
一方，欠点として，以下のことが挙げられる．(１)正確なかみ合
いを実現するために，各部品を精度よく加工する専用設備が必要で
ある．(２)機構上，かみ合い部の周速が大きく，高速運転には注意
を要する．
