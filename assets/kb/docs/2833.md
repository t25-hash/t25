# 2・7・2 締結体の強度

> **章**: 第2章 締結要素
> **節**: 2・28 に、面圧、ボス側の周方向応力σおよび径方向応力σの

a.疲労強度 はめ合わされた軸に繰返し負荷が与えられる
と，その接触端部での面圧および接線力の著しい集中によりフレッ
チングによる微小き裂が発生し，疲労強度の低下を引き起こすた
め，強度設計上注意を払う必要がある．このはめあい軸疲労限の，
平滑軸疲労限に対する低下率は，ねじり負荷の場合は1.3程
度 であるが，ねじり負荷の場合は2～3と大きな値となる．
この値は厳密には接触端の形状，材質，摩擦係数，面圧，平均応力
などの影響を考慮して予測される．
b.強度向上策 はめあい軸のフレッチング（fretting）強度
の向上策としては，まずは，接触端部の軸側あるいはボス側の形状
を工夫して，面圧および接線力の集中を防ぐ方法があり鉄道用車
軸，産業機器駆動軸等に利用され効果が報告されている，それらの
例を図Ⅰ-2･30に示す．そのほか，軸表面に焼入れ処理ある
いはショットピーニング処理を施し，表面に圧縮の残留応力を与
え，フレッチング強度を向上する方法があり，その効果が報告され
ている．

2・8・1スナップフィット
スナップフィット（snap-fit) は，位置決めと固定を行うこと
で部品の取付けを可能にする機械的な締結部品である．スナップフ
ィットは，図Ⅰ-2･31に示すように，位置決めのためのロケータ
（locator），部品どうしを固定し締結するロック（lock），部品どう
しの締結強度を向上させる補強材（enhancement）から構成され
る．ロケータと補強材は，高剛性と位置決め精度が要求され，ロッ
クは一部が弾性的にたわむことで挿入を可能にし，元の形に戻るこ
とで締結するため，柔軟性が要求される．
2・8・2スナップフィットの利点
スナップフィットには以下の利点がある．
① スナップフィット締結を用いると，部品点数を少なくするこ
とができ，軽量化が図れる．
② スナップフィットは単純な形状をしており，安価である．
③ スナップフィットは形状設計に自由度があり，さらにさまざ
まな材料の組合せにも対応可能である．
④ 組立に特殊な工具を必要とせず，組立作業性に優れている．
⑤ 分解作業も容易にでき，保守，修理，リサイクルには優れて
いる．
⑥ 接着剤や潤滑剤などが不要なため，クリーンな環境下で使用
可能である．
