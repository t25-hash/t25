# 2・7・2 締結体の強度

> **章**: 第2章 締結要素
> **節**: 2・28 に、面圧、ボス側の周方向応力σおよび径方向応力σの

a. 疲労強度はめ合わされた軸に繰返し負荷が与えられると、その接触端部での面圧および接線力の著しい集中によりフレッチングによる微小き裂が発生し、疲労強度の低下を引き起こすため、強度設計上注意を払う必要がある。このはめあい軸疲労限の、平滑軸疲労限に対する低下率は、ねじり負荷の場合は1.3程度であるが、ねじり負荷の場合は2~3と大きな値となる。この値は厳密には接触端の形状、材質、摩擦係数、面圧、平均応力などの影響を考慮して予測される。b. 強度向上策はめあい軸のフレッチング(fretting)強度の向上策としては、まずは、接触端部の軸側あるいはボス側の形状を工夫して、面圧および接線力の集中を防ぐ方法があり鉄道用車軸、産業機器駆動軸等に利用され効果が報告されている、それらの例を図I-2・30に示す。そのほか、軸表面に焼入れ処理あるいはショットピーニング処理を施し、表面に圧縮の残留応力を与え、フレッチング強度を向上する方法があり、その効果が報告されている。図-2・31 スナップフィット(c) ウェッジ(b) タブ(e) ピン(d) コーン(f) キャッチ図-2・32 凸形ロケータ
