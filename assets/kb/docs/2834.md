# 4・2・7 歯車装置の潤滑（第Ⅱ部2章参照）

> **章**: 第4章 伝動要素
> **節**: 4・2 歯車伝動装置

a.歯車潤滑の目的 歯車のかみ合い歯面は高い接触応力状態
で転がり滑り接触しており，適正な潤滑を行わなければ円滑な作動
が維持できなくなる．歯車装置潤滑の目的を下記に示すが，いずれ
を優先させるかによって潤滑方法が異なり，複数の目的を両立させ
るためには工夫が必要である．
① 摩擦低減 かみ合い面の摩擦を低減し，歯車装置の効率を
向上させる．
② 摩耗低減 歯面の摩耗を低減し，歯車の寿命を長くする．
③ 冷却 焼付きを防止し，装置を小形にする．歯面のピッチ
ング強さを向上させる効果も大きい．
④ 異物排除 侵入異物や摩耗粉を洗い流し，異物かみ込みに
よる損傷を防ぐ．
⑤ 防錆 歯車や歯車箱の腐食を防止する．
⑥ 装置部品の潤滑 軸受やしゅう動部品などを潤滑し，装置
寿命を長くする．

油浴式（oil bath）では歯面摩擦損失，潤滑油撹拌仕事，軸受損
失が潤滑油温度上昇につながるため，温度上昇の絶対値を潤滑油の
劣化と歯面焼付きの両面から設定しなければならない．潤滑油の寿
命は鉱油の場合，油温が10℃上昇するごとに半減する．劣化の目
安として鉱油で90℃程度が推奨されてはいるが，限界温度は歯面
状況や潤滑油劣化交換を考慮して設計者が決めることであり，限界
温度以下が維持できなければ循環式給油に変更せざるを得ない．
.循環式給油 油浴式の速度限界を超える周速を採用の目
安にする．給排油系を備えた強制潤滑（forced feed lubrication）
であり，潤滑装置を必要とするので，設備の設置スペースや価格面
での検討が必要である．
(１)給油方式 給油方式として噴射式と重力給油方式があ
る．噴射式は潤滑油ポンプを用いて潤滑部位に直接潤滑油を圧送す
る．歯車箱内の圧力や回転部位の遠心力に打ち勝って潤滑油を確実
に供給したい場合や冷却を促進したい場合に適用する．重力給油は
油の自重で滴下あるいは流下させる滴下式と，循環給油として排油
を油ポンプで油タンクに回収し，タンクから重力で潤滑部位に給油
する方式がある．

(２) 給油量 給油量は供給した潤滑油の温度上昇許容値が保
たれるように決める．温度上昇概略値としては，歯車装置の動力損
失（4･2･1･ d項参照）が油温上昇に寄与するとして求める．温度上
昇が適当な範囲に収まるよう，また潤滑油交換頻度，歯面の損傷状
況などを見て，給油量を決める必要がある．
(３) 給油ノズル 給油ノズルの配置としてかみ込み側からと
かみ外れ側からの選択が重要である．一般には比較的低周速（50
m/ s以下が目安）で潤滑効果を確実にしたい場合はかみ込み側給
油，50m/ sを超える高周速で冷却効果を期待する場合はかみ外れ
側を主にしてかみ込み側を補助とする給油が望ましい．歯幅が100
mmを超えるような場合は，給油ノズルの本数を複数設置し，歯
幅方向の給油むらを防止する．また，歯車周辺の空気の流れを考慮
して，給油位置をずらせる配慮も必要である．
c .開放式歯車の潤滑 歯車が雰囲気に露出している場合を開
放式と呼ぶ．基本的には歯車周速が小さく，潤滑油の飛散による周
囲の汚れがない場合や歯車が大きすぎて密閉歯車箱を作るのが困難
な場合に適用する．
.刷毛塗り，手差し給油 周速が低く（4m/ s以下）軽負
荷で冷却が必要ない場合に用いる．刷毛やへらでちょう度（NLGI
consistency number ）が大きいグリースやギヤコンパウンド，あ
るいはタール状の高粘度油を歯面に塗布する．
.滴下給油 滴下給油機で圧力をもたせずに潤滑部位に給
油する．所定の部位に確実に給油したい場合に利用する．やや高速
でも適用できるが，歯車装置周辺に潤滑油の飛散がないことに注意
を要する．
.オイルパン式 歯車の下端を浅い皿に付ける油浴潤滑方
式である．低周速大形歯車に適用される．オイルパンに潤滑油加熱
ヒータを設置しておけば，寒冷時の潤滑油粘度増加による潤滑不良
を防止できる．
.噴霧式 希釈した潤滑剤をノズルから圧縮空気とともに
歯面に吹付ける．タイマ設定または手動で一定時間ごとに必要油量
を確実に給油することができる．圧力給油するので潤滑装置が必要
である．

e .潤滑油の選定 歯車装置用としての潤滑油の選定は下記項
目に留意する必要がある．潤滑油種の選定に当たっては各種のギヤ
油規格（JIS K2219：1993など）を参照し，使用に当たっては潤
滑油メーカと性状や添加剤の内容を確認する．粘度だけで潤滑油を
選定すると，スラッジ，さび，水分離などのトラブルを誘発する場
合があるので注意を要する（Ⅱ-2章「潤滑剤」参照）．
(１)粘度，粘度指数 潤滑油は粘度グレードで分類されてお
り，グレードは潤滑油膜形成能に密接に関係する．運転温度や歯面
接触応力に応じた油膜形成を期待して，4･2･7･d項で述べた考え方
で粘度を選定する．粘度に影響するものとして圧力と温度がある．
圧力による変化は粘度の圧力係数αで示され，弾性流体潤滑油膜
厚計算に必要な値である．温度による粘度変化は粘度指数（vis-
cosity index）で示され，油膜計算には大気圧粘度を用いる．例え
ば鉱油系潤滑油ではパラフィン系のほうがナフテン系よりも温度に
よる粘度変化が小さく，粘度指数が大きい．使用温度範囲が広い場
合には，粘度指数の大きな潤滑油を選定する．また油膜厚さを計算
する際に必要となる粘度の圧力係数も潤滑油選定の際に考慮する．
(２)添加剤への配慮 ウォームギヤやハイポイドギヤなど歯
面の発熱が大きく焼付きが懸念される場合には，極圧添加剤［EP
剤（extreme pressure agent ）］入りの潤滑剤が多用される．EP剤
は高温，特に120℃以上で急速に分解し，油の劣化が進行するの
で，注意を要する．また潤滑油の酸化安定性，防錆性，消泡性，水
分離性などを用途に応じて考慮する．
(３)歯車潤滑用合成油 高周速歯車などで高温にならざるを
得ない場合，耐熱合成油が用いられる．油種としてはポリ- α-オレ
フィン（蒸発開始温度200℃），ポリオールエステル（220℃），ポ
リフェニルエーテル（250℃）などが用いられる．
