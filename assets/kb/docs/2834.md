# 2・8・1 スナップフィット

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

スナップフィット(snap-fit)は、位置決めと固定を行うことで部品の取付けを可能にする機械的な締結部品である。スナップフィットは、図I-2・31に示すように、位置決めのためのロケータ(locator)、部品どうしを固定し締結するロック(lock)、部品どうしの締結強度を向上させる補強材(enhancement)から構成される。ロケータと補強材は、高剛性と位置決め精度が要求され、ロックは一部が弾性的にたわむことで挿入を可能にし、元の形に戻ることで締結するため、柔軟性が要求される。
