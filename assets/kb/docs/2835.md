# 2・8・2 スナップフィットの利点

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

スナップフィットには以下の利点がある。1スナップフィット締結を用いると、部品点数を少なくすることができ、軽量化が図れる。2スナップフィットは単純な形状をしており、安価である。3スナップフィットは形状設計に自由度があり、さらにさまざまな材料の組合せにも対応可能である。4組立に特殊な工具を必要とせず、組立作業性に優れている。5分解作業も容易にでき、保守、修理、リサイクルには優れている。6接着剤や潤滑剤などが不要なため、クリーンな環境下で使用可能である。一方、スナップフィットには以下の注意点がある。1成形精度に限界がある。2使用できるプラスチック材料には、多少の制限がある。3プラスチック材料を繰返し使用する場合には、疲労破壊を考慮する必要がある。4高温および低温環境下での使用には注意が必要である。
