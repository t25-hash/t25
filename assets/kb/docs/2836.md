# 2・8・3 スナップフィットの材質

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

プラスチックを大きく分けると、熱硬化性プラスチックと熱可塑性プラスチックがある。熱硬化性プラスチックは網状高分子または三次元高分子で加熱などによって一度架橋・重合して高分子となり固化すると再び加熱しても流動状態とならない。熱可塑性プラスチックは線状高分子または一次元高分子であり、加熱と冷却によって流動状態と固体状態とが可逆的に変化することが可能である。そのため、リサイクルの観点から、スナップフィットなどには熱可塑性プラスチックが用いられ、その中でもポリスチレンが多用されている。このポリスチレンは外観が美しく、成形性、電気的絶縁性に優れている。
