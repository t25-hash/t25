# 4・3・3 歯付ベルト伝動

> **章**: 第4章 伝動要素
> **節**: 4・3 ベルト伝動装置

歯付ベルト（synchronous belt ，toothed belt ）は，1940年代半
ばに米国で実用化された比較的新しいベルトで，(１)同期伝動が可
能，(２)機構が単純で比較的効率が高い，(３)高速伝動が可能，
(４)平ベルトやVベルトに比べ初張力が小さくてよい，などの特
徴から近年，自動車OHCエンジンカム軸駆動用のみならず，自動
化・省力化機器，工作機械，ロボット等産業機械，OA機器，
ATM，CDなど銀行用端末機，アミューズメント機器などに数多
く使用され，駆動，搬送，位置決めなど多種多様な使われ方をして
いる．ベルトの歯部および背面部はクロロプレンゴムでつくられ，
歯面はゴムを塗布したナイロン帆布で覆われているものが一般的で
あるが，ポリウレタンで作られたものもある．心線は，グラスファ
イバが一般的であるが，アラミド繊維コード，スチールワイヤ，ポ
リエステルコードなども使われる．歯付ベルトの歯形形状は台形歯
形からスタートしたが，①低騒音，②高寿命，③高伝達トルク，④
高伝達精度などの理由により，近年は円弧歯形歯付ベルトが主流と
なってきた．円弧歯形歯付ベルトにはHTD，STPD（STS)，PX
など，メーカによって異なる数種類の歯形があり，それらの間でプ
ーリの互換性はないが，いずれも台形歯形ベルトより歯ピッチに対
する歯厚と歯丈が大きく，プーリとのかみ合いもスムーズで伝動容
量が大きい．またベルト歯先とプーリ歯底を適当に干渉させるなど
