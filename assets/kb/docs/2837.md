# 2・8・4 スナップフィットの形状

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

プラスチック製スナップフィットなどの射出成形品は,力学的な設計のほかに形状設計が重要であり,(1)面取り,(2)肉厚の均等化,(3)型抜き,(4)温度特性などの検討が必要である.寸法および外観の点から,成形品設計の大きな原則は,(1)肉厚は均一に,段差を付けず,(2)肉厚よりもリブで強さをもたせ,(3)内側外側を問わず角は丸く,(4)適切な肉ぬすみをする,(5)寸法収縮による変形を予測した形状にする,(6)精度を要する寸法は金型によって直接決まるような形状にする,(7)クリープ,応力緩和,疲労破壊などに強い形状にすること,である.
