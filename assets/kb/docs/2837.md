# 2・8・4 スナップフィットの形状

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

プラスチック製スナップフィットなどの射出成形品は，力学的な
設計のほかに形状設計が重要であり，(１)面取り，(２)肉厚の均等
化，(３)型抜き，(４)温度特性などの検討が必要である．
寸法および外観の点から，成形品設計の大きな原則は，(１)肉厚
(ｃ)エッジ
は均一に，段差を付けず，(２)肉厚よりもリブで強さをもたせ，
図 - 2･33平面形ロケータ
(３)内側外側を問わず角は丸く，(４)適切な肉ぬすみをする，(５)
寸法収縮による変形を予測した形状にする，(６)精度を要する寸法
は金型によって直接決まるような形状にする，(７)クリープ，応力
緩和，疲労破壊などに強い形状にすること，である．
