# 4・3・4 そのほかのベルトによる伝動

> **章**: 第4章 伝動要素
> **節**: 4・3 ベルト伝動装置

a.Vリブドベルト伝動 Vリブドベルト（V-ribbed belt ）
は，平ベルトの柔軟性と，Vベルトの高伝動能力の利点を併せも
つベルトとして開発された．ベルト厚さが薄いので，使用できる最
小プーリ径はVベルトより小さく，平ベルトに近い．また，ベル
トの背面でも平ベルトのように動力伝動を行うことができるので，
多軸伝動が可能である．単位幅当たりの質量もVベルトより小さ
いので，高速度での使用が可能である．このような利点から，近
年，自動車エンジンの補機駆動用としてよく使用されるようになっ
てきた．VリブドベルトのJISは定められていないが，ISO
9981：1990，ISO9982：1991や自動車規格JASO E109：1994，
E111：1993に規格化されている．Vリブドベルトには，H，J，
K，L，M形の5種類があり，リブゴムがはまり合うV溝をもっ
た，Vリブドプーリ（V-ribbed pulley）とともに使用される．
ISO規格やJASO規格には，プーリ溝部の詳細な寸法と公差が定
められている．
b.丸ベルト 丸ベルト（round belt ）は直径が2～12mmの
円形断面のベルトで，搬送用の回転ローラ駆動や軽負荷伝動に使用
される．また，断面形状に方向性がないので，伝動面が三次元でも
伝動が可能である．
