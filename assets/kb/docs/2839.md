# 2・8・6 スナップフィット形状設計の要領

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

スナップフィットには多種多様な形状のものがあるが、ここでは最も一般的なカンチレバーフック(図I-2・40)を設計する際の要領について述べる。a. 基本形状。基底部でのはりの厚さ図I-2・41(a)に示すようにベースプレートに対して垂直にカンチレバーフックを設置する場合は基底部の厚さはプレートの厚さの50~60%とする。はりの厚さが50%より小さすぎると、射出成形時にうまく充てんされない場合がある。また、はりの厚さが60%より厚すぎると、冷却時に残留応力や気泡などの問題を生じる場合がある。図I-2・41(b)のようにベースプレート上や延長上にカンチレバーフックを設置する場合は、基底部の厚さをベースプレートの厚さと等しくする。カンチレバーフックのはりの厚さをプレートの厚さより図-2・40 カンチレバーフックの形状図-2・44 カンチレバーフック(a)(b)図-2・41 はりの厚さ図-2・42 挿入面の角度(b)(a)図-2・43 締結保持面の角度および深さ薄くしたいときは基底部に生じる応力集中や射出成形時の充てんの問題を避けるため、先端に向けて徐々に細くなるようなテーパ形のカンチレバーフックとする。. はりの長さはりの長さははりの厚さの少なくとも5倍でなければならず、10倍が推奨されている。しかし、はりの厚さに対してあまりに長すぎると射出成形の充てん時に問題を生じる場合がある。はりの厚さに対して短い場合は基底部の曲げだけでなく、せん断による影響も考慮する必要がある。. 挿入面の角度挿入面の角度は組立力に影響を及ぼす。角度を大きくするにつれて、組立時に大きな力を要する。理想的には組立時に要する力を小さくするために、できるだけ挿入面の角度を小さくする。実際には図I-2・42に示すαとしては25°~30°が用いられる。組立が困難になるため、挿入面の角度が45°以上になることは避ける。. 締結保持面の深さ締結保持面の深さは図I-2・43(a)に示すように組立/分解時に必要なたわみの大きさであり、アンダカット(undercut)とも呼ばれる。はりの長さとはりの厚さの比が5:1のときは締結保持面の深さをはりの厚さより小さくする。はりの長さとはりの厚さの比が10:1に近い場合は締結保持面の深さをはりの厚さと同程度にする。. 締結保持面の角度締結保持面の角度は、図I-2・43(b)に示すように締結力や分解力に影響を及ぼす。締結保持面の角度は外部からの分解力が結合部に作用しないような解除可能形ロックの場合は35°程度、外部からある程度小さな分解力が作用する場合は45°程度とし、また、外部から大きな分解力が作用する場合、解除不可能形ロックとしては80°~90°とする。. 先端でのはりの厚さ図I-2・44(a)のように多くの場4―29第I部機械要素(a) テーパなし(b) テーパあり合は基底部でのはりの厚さとはり先端での厚さは等しい。しかし、基底部でのひずみが大きい場合には図I-2・44(b)のように、応力集中を避けるために先端に向けて徐々に細くなるようなテーパをもつフックを使用する場合がある。. はりの幅はりの幅ははりの長さよりも小さくする。はりの幅がはりの長さの2分の1以上になると、はりとしてよりも平板のような振舞いをする。. 基底部の丸め半径基底部の応力集中を低下させるため、ベースプレートの厚さの2分の1以上のを付ける。
