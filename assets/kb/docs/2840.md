# 2・8・7 スナップフィットの形状設計

> **章**: 第2章 締結要素
> **節**: 2・8 スナップフィット

図I-2・44(a)のようにまっすぐなはりのカンチレバーフックと図I-2・44(b)のようなはり部にテーパをもつカンチレバーフックの2種類について、レナウド・カール(Renaud-Kar)の簡易式を用いて組立時および分解時に要する力を算出する。スナップフィットの変形解析は非線形となるがレナウドとカールは形状を細かく制限し、材料力学の基本式を用いることで、締結保持面の許容深さ、分解時に必要な力、組立時に必要な力を求めている。まず、テーパがないものとテーパがあるカンチレバーフックについて、締結保持面の深さをレナウド・カールの簡易式より求めると次式のようになる。次に、断面が一定(はりの幅と厚みが一定)であるカンチレバーフックにおいて分解時に要する力は、縦弾性係数をとすると、次式のようになる。断面aに引張応力が作用したときには、aとb を入れ替える。式(I-2・54)を用いて、式(I-2・52)(I-2・53)の許容ひずみεを消去して、分解時に要する力を求めると次式のように表される。これらにより、カンチレバーフック形スナップフィットの組立時および分解時に必要となる力が求まり、強度および形状設計が行える。文献(1) 辻裕一・丸山一男、精密工学会誌、59-4(1993)、589.(2) 山本晃、ねじ締結の原理と設計、(1970)、57、養賢堂。(3) 丸山一男・ほか2名、VDI 2230(1986)、高強度ねじ締結の体系的計算法―円筒状一本ボルト締結―、第3版、(1989)、17、日本ねじ研究協会。(4) 辻裕一・ほか3名、精密工学会誌、58-9(1992)、125.(5) 丸山一男・中川正人、精密工学会誌、53-3(1987)、416.(6) 文献(3)の96ページ。(7) 文献(2)の66ページ。(8) 文献(3)の55ページ。(10) 文献(3)の97ページ。(11) 吉本勇編、 ねじ締結体設計のポイント(JIS 使い方シリーズ)、(1992)、198、日本規格協会。(14) 山本晃・賀勢晋司、精密機械、43-4(1977)、470.(15) 佐藤進・ほか3名、神奈川大学工学部研究報告、19(1981)、31.(1972)、2666.(17) 山本晃、ねじ締結の理論と計算、(1970)、29,31、養賢堂。(18) Rothbert, H.A., Mech.
