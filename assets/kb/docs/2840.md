# 4・4・2 サイレントチェーン伝動

> **章**: 第4章 伝動要素
> **節**: 4・4 チェーン伝動装置

サイレントチェーン（inverted tooth chain，silent chain）は，
歯を設けたプレートどうしをピンまたは転がりを取入れた連結要素
などで組立てたチェーンである．図Ⅰ-4･64にサイレントチェーン
とスプロケットの歯形を示す．横ずれを防止するガイドをチェーン
両側に設けてスプロケットをまたぐ形式のサイドガイド形と，中央
部に設けるセンターガイド形（スプロケット中央部に溝を切る）が
ある．プレートの形状，ピンの形式，ピン端の止め方，継手リンク
の形式等は製造者に任されており，ANSI B29.2M：1982では，
これらとかみあう共通歯形，チェーンの幅寸法，ガイド形式，伝動
いる．
能力を規定している．チェーンピッチは4.76，9.52，12.70～50.8
mmの9種類あるが，中間的なピッチのチェーンも用いられる．
表Ⅰ-4･30にチェーンの寸法，破断強さ，質量を示す．
ユニット
部品すべてが張力を分担するためむだがなく，同一幅であればロ
ーラチェーンに比べ伝動能力は高い．サイレントと称する理由は図
Ⅰ-4･65のようにプレートがスプロケットに対し斜めにかみあい，
相対衝突速度が低下して騒音レベルが低減するためである．特殊な
用途では，内股部をスプロケット歯に干渉させつつ外股部を着座さ
せたり，内股のみでスプロケット歯とかみあわせるなど騒音・振動
低減の工夫をこらしているものもある．
