# 4・5・2 チェーンテンションドライブ（chain tension drive)

> **章**: 第4章 伝動要素
> **節**: 4・5 機械式無段変速機

湿式のメタルブロックをチェーンで連結し，図Ⅰ-4･67に示すよ
うにVプーリで挟み，接触部の回転半径を変えることにより可変
速するもので，力はチェーンの張力で伝えられる．トルクを伝える
プーリに挟まれたブロックは間欠的に配列されているため，エラス
トマVベルトのような連続的な力の伝達とはならない．騒音振動
問題が課題ではあるが，伝達効率が高く，大容量化が行いやすい．
