# 3・1・4 軸の設計式

> **章**: 第3章 軸・軸受要素

丸軸、切欠付き軸(段付き軸、油穴付き軸など)、圧入軸などの設計は以下の方法で行う。なお以下に示す軸の設計式においては特別な指示のないかぎり、左右両辺の量記号の単位は統一して数値を代入するよう注意しなければならない。a. 許容応力による設計。許容せん断応力による設計軸の材料が延性材料である場合には、軸に働く曲げモーメントとねじりモーメントの比に無関係に、破損は最大せん断応力説に従うものとし、許容せん断応力τを用いて軸径を計算する。許容応力および設計式は表I-3・2中の(a)に示される。. 許容垂直応力による設計軸の材料に脆性材料を使用する場合(そのような場合は一般には少ないが)、および軸方向荷重として圧縮力を受ける場合には、軸の破損は最大主応力説に従うものとして許容垂直応力σを用いて軸径を定める。許容応力および設計式は表I-3・2中の(b)に示される。なお、表I-3・2の設計式の欄に書かれた、(a)および(b)の計算に必要なモーメントの動的効果の係数,の目安は表I-3・3に示される。ウォーム歯車軸や船用プロペラ軸のように軸方向の圧縮力を受け、座屈(buckling)の危険性のある軸に対しては表I-3・4のような座屈効果の係数ηを考えて表I-3・2の設計式を適用する。なおηを決めるためには軸の直径が必要であるが、この場合まず初めに、既存の軸を参考にを仮定してηを求め、このおよびηを設計式の右辺に代入しての第一次近似解を求める。次にそのを用いてηの値を修正し、および修正されたηの値を再び設計式の右辺に代入しての第二次近似値を求める。このような逐次近似計算を数回繰返すことにより、求めるべきの値を決定できる。. limit)を基準にして許容応力を定め、表I-3・2中の(c)の設計式を用いて計算することができる。次に、図I-限度線との交点Pτ,τを決定し、さらに安全率を考慮して許容応力をP点より内側に求める。したがって、許容応力および設計式は表I-3・2中の(c)(1)のようになる。なお、平均応力の値が確実であれば=1としてよい。. 曲げ疲労限度を基準とする簡易設計式多くの材料では繰返しねじりの疲労限度は回転曲げ疲労限度の約2分の1であることから、軸の疲労破損を最大せん断応力説に従うものとみなし、かつ疲労限度線図において両振り疲労限度(曲げ疲労限度)と引張り降伏点を直線で結ぶ修正グッドマン線(α3編10・2・5項参照)で基準強さが与えられるものとし、さらに切欠効果、寸法効果、表面効果などは両振り疲労限度に影響を及ぼすが、静的強さには影響しないものとみなして安全率を応力振幅と平均応力の両方に共通の値とすることにより表I-3・2中の(d)の簡易設計式が得られる。(1) 軸の切欠係数[(fatigue)notch factor]β軸には構造上の必要から、段、環状溝、ねじ、車輪ボス玉軸受の座、油穴、キー溝など種々の切欠が付く。これらの部分には応力集中を生じ、疲労強度が低下する。同一径で切欠がない場合とある場合の疲労強度の比を表す切欠係数βは、切欠の形状、荷重の種類、材質、軸径の大小などによって変わる。表I-3・5は普通の軸材料を用い、普通の軸に用いられる程度の切欠の場合のβの概略値である(α3編10・2・5項参照)。一般にβの値は材料の引張強さにほぼ比例的に増加するので、強い材料を用いてもそれほど利益にならないこともあり、設計上注意を要する。なお、形状係数(応力集中係数)αの値が2以下の比較的鈍い切欠の場合にはβαとなるので、αの値(α3編7・1・2項参照)からβを推定することもできる。(2) 軸の寸法効果係数(dimension factor)ξ曲げまたはねじりの繰返しを受ける軸では疲労限度に寸法効果が第I部機械要素現れる(α3編10・2・5項参照)。したがって小形標準試験片の疲労限度から大形軸の疲労限度およびその許容応力を推定するときは寸法効果の係数ξを考慮しなければならない。多くの実験結果から、統計的に推定された寸法効果の係数が求められている。一般の大形軸ではξ=0.86~0.89 程度であるが、Si-Mn鋼、Si-Mn-Cr鋼などの軸では熱処理における質量効果や鍛造の影響で、寸法効果が著しく現れ、ξ=0.5~0.7に低下するという実験値の報告もある。図I-3・2は0.2~0.5%Cの各種炭素鋼圧入軸の回転曲げ疲労限度と軸径の関係を示したもので、直径100mm以上の軸ではξ=0.3~0.4に低下する。圧入軸ではフレッチングコロージョンの影響で寸法効果はさらに著しい。圧入軸のこのような強度低下を防ぐには圧入部の表面にあらかじめロール加工や高周波焼入れを行うと効果がある。一般の切欠軸にも寸法効果は顕著に現れる。(3) 以下に研磨紙により仕上げを行った場合の疲労限度の比を表面効果の係数と呼ぶ。図I-3・3は各種表面仕上げの鋼製軸の回転曲げにおける表面効果係数ξと材料の引張強さとの関係を示している。繰返しねじりの場合のξの値はこれよりも少し大きい。腐食作用のある場合は腐食疲労のために疲労限度の低下が著しく、鋼材では材料の引張強さにはほぼ無関係に、淡水腐食で100~200MPa、塩水腐食で130~140MPaとなり、ξの値はかなり小さい。図-3・2
