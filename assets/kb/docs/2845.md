# 3・1・5 キー溝付き軸の設計

> **章**: 第3章 軸・軸受要素
> **節**: 3・1 の原点Oより/=ττ=一定、の直線を引き、疲労

軸にキー溝を付けると溝底のすみ付近に応力集中を生じ軸の強さが減少するので、その応力集中係数(α3編2・7節および7・1・2項参照)、切欠係数(α3編10・2・5項参照)あるいは表I-3・2などを参照して許容応力を低く決め、直径を求める。次の式はキー溝付きの軸と同径でキー溝なしの軸の強さの比およびねじれ角の比を与える簡便式である。したがってキー溝付き軸の許容応力をキー溝なしの場合の倍に減じてとればキー溝付き軸の径を決定できる(表I-3・2)。またキー溝付き軸の溝底の径をもつ丸軸と同等とみなす簡便法もある。ねじれ角はキー溝のある部分では増すが、はめあわされるキーおよびボスが軸のねじり剛性の減少を妨げる性質があるので、滑りキーなどに用いられる特に長いキー溝の場合以外はキー溝によるねじれ角の増加は無視してよい。なお、表I-3・6はキー溝付きの切欠係数βの実験値である。
