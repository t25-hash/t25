# 4・6・2 ウェッジローラ減速機（wedge roller reducer ）

> **章**: 第4章 伝動要素
> **節**: 4・6 トラクションドライブ式変速機

トルクに応じたローディング力を得る方法として，サン軸とリン
グ軸の中心をわずかにずらし，これらに挟まれる遊星ローラの中の
一部をスイングできるようにして，接触したときにくさび角ができ
る変速機の例を図Ⅰ-4･86に示す ．図の例では三つのローラのう
図 - 4･87 ボールねじの外観
図 -4･ 86 ウェッジローラ減速機（トルク比例ローディング）
ち，二つのローラの回転軸は固定されており，一つのみ可動する構
造（これをウェッジローラと呼ぶ）になっている．例えば，入力軸
となるサン軸が反時計方向に回ると，ウェッジローラはサンとリン
グの間にかみ込みリングに回転力を伝える．一方，サン軸が時計方
向に回るとウェッジローラは挟まれた状態から解放され，リングに
動力を伝達しない．このようにウェッジローラ減速機は一方向クラ
ッチの機能も併せ持つことになる．
