# 3・2・1 滑り軸受(plain bearing)の種類と選定

> **章**: 第3章 軸・軸受要素
> **節**: 3・2 滑り軸受

滑り軸受に要求される荷重の支持方向,スペース,運転条件(負荷の大きさ,速度,環境温度),性能,寿命などの仕様によって,採用すべき軸受の種類と詳細設計が定まる.その際,転がり軸受も選択肢の一つに入り,両者の性能比較も同時に検討される.滑り軸受は,荷重の方向によりジャーナル軸受とスラスト軸受に大別され,また,荷重支持原理により流体潤滑軸受と磁気軸受,流体潤滑軸受はさらに動圧軸受と静圧軸受に,動圧軸受にはさらにくさび膜作用による一般的なタイプとスクイーズ膜作用による特殊なタイプがある.また,使用潤滑剤により液体潤滑軸受,気体軸受,無潤滑軸受(自己潤滑軸受あるいはドライベアリング)に,軸受材料により金属軸受,非金属(セラミックス,プラスチック,ゴムなど)軸受に分類される.そのほか,荷重(静,動),運動の種類(回転,直動),速度(高,低),温度(高,低)などの作動条件,用途,精度などによる分類もある.潤滑下の滑り面の潤滑状態は,第II部1・1・5項で示されるように,流体潤滑,混合潤滑,境界潤滑の三つに大別できる.流体潤滑状態では,軸受すきま内に形成された流体膜中に発生する圧力が負荷荷重に対抗して相対運動をする固体壁面どうしの直接接触を阻むので,摩耗がなく摩擦係数も小さい.したがって,できるだけこの状態が実現されるように設計する.軸受性能は,形成される流体潤滑膜の形状によっておもに支配される.図I-3・7~I-3・9は,この流体潤滑膜形状を基本的に規定する代表的ないくつかの潤滑面断面形状を示す.図I-3・8(a)の平行平面軸受は,原理的にはくさび(a) 真円軸受(b) 部分軸受(c) 圧力ダム軸受(d) 二円弧軸受(e) 三円弧軸受(f) 四円弧軸受(g) オフセット軸受(h) ティルティングパッド軸受(i) 浮動ブシュ軸受(j) スパイラル溝軸受(k) フォイル軸受図-3・7動圧ジャーナル軸受(hydrodynamic journal bearing)の各種潤滑面形状作用が生じないが,実際には工作誤差,荷重変形,熱変形などによりくさび形状が形成され,また溝の端部での微小なくさびにより流体の負荷能力が生じる.滑り軸受を設計するには,潤滑面形状を含む軸受の形式をまず選び,次に軸受材料と潤滑剤を定め,要求仕様を満足する軸受特性が得られるまで設計変数を変えて軸受特性計算を行う.その軸受特性にはゾンマーフェルト数,最小油膜厚さ,摩擦係数,必要給油量,軸受温度などの静特性だけでなく,その軸受で支持される回転体の振動応答振幅特性,自励振動の安定性などの動特性も含まれる.やむを得ず混合潤滑・境界潤滑状態,あるいは乾燥状態で使用する場合は,軸受材料,潤滑油とその添加剤,あるいは固体潤滑剤の選定が特に重要になる.流体潤滑状態で使用される軸受でも起動,停止時には混合潤滑・境界潤滑状態になるので,軸受材料および潤滑剤の選定にはその配慮が必要になる.
