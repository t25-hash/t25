# 4・7・1 送りねじの一般的特徴

> **章**: 第4章 伝動要素
> **節**: 4・7 ねじ伝動装置

ねじ伝動装置は，ねじ軸とナットを組合わせて，相対的な回転運
動を軸方向の直線運動に変換する装置をいい，以下に一般的な送り
ねじ（feed screw）という言葉を使う．
a.機能 送りねじを機能的に分類すると次のようになる．
(１) ねじ軸にかかるトルクを軸方向の推力（軸力）に変換す
る．この例としてジャッキのねじがある．
(２) ねじ軸の回転角を直線変位に変換して，正確な位置決めを
行う．この例としてマイクロメータのねじがある．
(３) 上記(１)と(２)の両方を行う．工作機械の親ねじはこの例
に属する．
b.ほかの要素との比較 送りねじ以外の回転運動を直線運動
に変換する要素として，ラックピニオン，摩擦駆動（friction
drive，キャップスタンドライブと呼ぶことがある），ベルト送り
（ワイヤ送り）などがあるが，送りねじと比較すると次のようにな
る ．
(１) ラックピニオン：非常に簡単な機構なのでよく用いられ
る．基本的にバックラッシが存在し，位置決め精度はそれほどよく
ない．
(２) 摩擦駆動 ：高速性は優れているが，摩擦力を駆動力に
変換するため発生する推力はあまり大きくできない．
(３) ベルト送り（ワイヤ送り)：何より機構が簡単である．送
り方向の剛性は小さい．
また，各種送りねじのうち特にボールねじについては，このとこ
ろ高速化手段としてリニアモータとの比較議論 が行われている．
超ロングストロークの送りや，短ストロークの揺動はボールねじの
苦手とするところなので，今後住み分けがはっきりしていくであろ
う ．
