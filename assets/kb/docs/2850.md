# 4・8・1 クラッチの種類

> **章**: 第4章 伝動要素
> **節**: 4・8 クラッチ

クラッチ（clutch）は同心軸上にある駆動側から被駆動側へ動力
の断続を行う機械要素で，連結の形式によって表Ⅰ-4･32に見られ
るように，かみあいクラッチ，摩擦クラッチ，自動クラッチに分類
され，その作動方式によって表Ⅰ-4･33のように機械式，油圧式，
空気圧式，電磁式のものがある．

カム，レバー，リンク機構などにより力の拡大を図るが，作
動力は人の手や足によるので，その大きさ・断続の回数・連
結時間が限定される．いわゆる効き具合を感じることができ
る．最も安価である．
作動圧を大きくとることができるので，単位容積当たりの伝
達トルクは最大で，連結時間も短い．油圧ポンプ・配管・制
御弁などが必要である．
本体は構造が簡単で安価である．伝達トルクの大きいものも
ある．空気による摩擦面の冷却ができ，緩衝連結もできる．
圧縮空気（または空気圧縮機）・配管・制御弁などが必要であ
るが戻り回路は不要．
遠隔操作に最も有利である．連結時間は最小で，断続回数も
多くとることができる．比較的高価であり，爆発の危険のあ
る雰囲気では使用できない．
