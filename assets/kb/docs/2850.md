軸受の使用箇所と負荷条件図I-3・33は、往復動エンジンに使用される各種の滑り軸受を示す。エンジン組立上の制約から主軸受(main bearing)、コンロッド大端部軸受(connectingrod big end bearing)は薄肉円筒を二つ割り(半割りタイプ)にして使用するのに対し、ピストンピン軸受(piston pin bearing)は全周一体のまま(ブシュタイプ)使用する。またクランクワッシャ(crank washer)は、軸方向の位置決めをするスラスト軸受である。そのほか、カムシャフト軸受、バランスシャフト軸受(半割りタイプまたはブシュタイプ)やロッカアームブシュなどがある。4―42機械要素・トライボロジー図I-3・34の荷重極線図(polar load diagram)に見られるように、主軸受、コンロッド大端部軸受にかかる荷重は大きく、負荷条件は厳しい。荷重は、シリンダ内で発生する爆発力、ピストンおよびコンロッドなどの往復運動と振れまわりによる慣性力が主である。近年、ディーゼルエンジンの燃焼圧力は年々高くなり、最近では15MPaを超えるものも現れ、最大軸受面圧(軸受にかかる最大荷重を軸受投影面積で割った値)が100MPaに達する場合も出てきた。一方、スポーツ仕様のガソリンエンジンでは、最高出力時で8000rpmを超すエンジンもあり、軸受部の温度が180°Cに達する場合もある。b.
