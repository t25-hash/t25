# 4・8・2 かみあいクラッチ（positive clutch)

> **章**: 第4章 伝動要素
> **節**: 4・8 クラッチ

駆動側と被動側とを，機械的かみあいによって連結するクラッチ
で，滑りがなく伝達が確実なので確動クラッチ（ポジティブクラッ
チ）ともいわれる．円筒端面に設けた角形，台形，のこ歯形，スパ
イラルなどの凹凸によって連結するジョークラッチ（図Ⅰ-4･91)，
円筒端面に設けた多数の山形の凹凸によって連結するツースクラッ
チおよび外歯車と内歯車とによって連結する歯車クラッチ（インボ
リュートスプラインもかみあいクラッチとして用いられる）があ
る．
ジョークラッチは接続的に衝撃を伴うため，停止中またはごく低
速回転のときにしか連結できない．また，ツースクラッチは比較的
低速ならば回転中でも連結できるが，負荷能力はジョークラッチよ
り小さい．ツースクラッチの山形の一部を特殊形状にして，1回転
中に1箇所でしかかみあわないようにした定位置かみあいクラッチ
は，位置決め・同期などの目的に用いられる．
