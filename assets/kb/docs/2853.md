# 3・2・5 気体軸受(gas bearing,II-1・4・5参照)

> **章**: 第3章 軸・軸受要素
> **節**: 3・42 に示すように,球面や円すい面を利用して一つの軸受面で半

気体軸受は気体膜圧力により物体を非接触に支持し,位置決めする機構要素である.油潤滑軸受と同様に,軸受面の相対運動によって生じる圧力を利用する動圧軸受と外部から供給される加圧気体の圧力を利用する静圧軸受がある.気体軸受の潤滑剤は普通空気で,大気環境で使用される動圧型では潤滑剤が自然に供給されることが最大の利点である.油に比べて気体の粘性係数は約1/1000なので,摩擦力が小さいという利点があるが,同時に負荷容量や軸受剛性も小さいので,軸受すきまを数m以下に小さくし,小形軽量な高速回転体の支持に用いられる.またターボ膨張機などでは作動気体を潤滑剤に用いることができる.静圧型では圧縮気体が必要であるが,低速でも動圧型よりも負荷容量・剛性が高くでき,低摩擦・清浄なので精密加工機械,計測・試験機器などの軸受に利用される.図I-3・46におもな動圧ジャーナル軸受の形式を示す.真円軸受は最もシンプルであるが,偏心方向と直角方向の軸受反力が大きく,垂直軸では低速域から不安定な振れまわりが生じるので用いら図-3・45 おもな静圧ジャーナル軸受の構造の概略4―45第I部機械要素またはスラスト軸受面に一列または二列の微小な給気孔を設ける構造で,>4(:給気孔径,:給気孔直下のすきま)のとき最小絞り断面積はπとなりこれを自成絞りといい,<4のとき最小絞り面積はπ4になりオリフィス絞りという.オリフィス絞りのほうが軸受剛性を50%近く高くできる.高速ロータ支持には軸受すきまが大きく給気流量も大きい給気孔絞り型が用いられる.給気孔・溝絞り型は,軸受端近傍の給気孔を微小な幅と適当な深さの円周溝で導通させることにより,少ない給気孔で軸受内の圧力発生効率を高め,最適すきまを小さくできるので,軸受剛性を高くできる.表面絞り型は軸受面の溝に絞り機能を設けるもので,軸受効率は小さいが剛性を最大にする最適軸受すきまを無制限に小さくできるので剛性が高くできる.上記の静圧軸受はSUS 系の材料で製作される場合が多い.多孔質絞り型は金属粉末焼結材またはグラファイト材により軸受面を作り,微小な給気孔を軸受面全体に分散させたものである.給気部の微小外乱がなく,表面目詰まりを最適に設計すると比較的減衰も高くでき,またグラファイト材は接触時の焼付きがないので,近年精密機械用の静圧気体軸受として多く使用されている.
