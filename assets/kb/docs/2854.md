# 3・2・6 磁気軸受(magnetic bearing)

> **章**: 第3章 軸・軸受要素
> **節**: 3・42 に示すように,球面や円すい面を利用して一つの軸受面で半

磁気軸受の最大の特徴は磁気力を利用してロータを非接触で支持することである.この特徴を生かしてほかの軸受では不適切な分野にも磁気軸受が使われている.ラジアル荷重を支えるラジアル磁気軸受とスラスト荷重を支えるスラスト磁気軸受がある.また利用する電磁力から(1)磁気吸引力だけを利用するもの,(2)磁気反発力と磁気吸引力を併せて利用するもの,(3)電磁誘導反発力を利用するもの,(4)超電導を利用するものがある.さらに,磁気力を発生する磁石の種類としては永久磁石,常電導電磁石,超電導磁石がある.a. 軸受の構成最も一般的な磁気吸引力だけを利用する磁気軸受は図I-3・49に示すように,磁気吸引力を発生する電磁石,軸と軸受間のすきまを計測する非接触変位センサ,軸受すきまを一定に保つよう電磁石に流す電流値を指示する補償回路,補償回路からの指示に従って電磁石に電流を流す電力増幅器から構成される.さらに固体潤滑材を使用した非常用転がり軸受,電力増幅器へ供給する直流電流を商用交流電源から変換するインバータDC電源,商用電源やDC電源の故障に対処するバックアップバッテリー,運転状態を監視する状態監視モニタも装備している.b. 軸受負荷能力磁気軸受の負荷能力は電磁石鉄心に使用する磁性材料の磁気飽和特性で決められる.電磁石鉄心には,けい素鉄,磁性ステンレス綱が使われている.これらの磁性材料の飽和磁束密度は1.6T(磁性軟鉄や3%けい素鉄)から1.4T(磁性ステンレス)程度である.したがって上限変動荷重を考慮して,静的最大磁束密度が1.2T 程度で支えられる静加重を設計することが無難である.鉄心は,鉄心内を通過する変動磁束による渦電流の発生を抑制するため,薄い磁性材料の綱鈑を積層して製作する.図I-(m),(m)とおくと,静的負荷能は0.2(MN),図I-3・51に示す円板型アキシアル磁気軸受では軸受対向面積を욀(m)と図-3・49磁気軸受の基本構成4―46機械要素・トライボロジー図-3・51 アキシアル磁気軸受おくと,静的負荷能は,0.6욀(MN)である.c. 軸受すきま磁気軸受のすきまを非常用軸受すきまより大きく取ることにより,磁気軸受の磁気吸引力が失われても磁気軸受のロータとステータが接触することを防止している.軸受すきまを大きく取ると磁気回路の磁気抵抗がそれに比例して大きくなり,コイルを励磁する電源電圧を高くしなければならなくなるので,軸受すきまをむやみに大きくはできない.しかし,滑り軸受のすきまと比べれば格段に大きくできるので,ロータが大きく振れまわる回転機械の軸受に適している.d. 制御法磁気軸受の制御には,(1)アナログPID 制御,(2)ロータの高次固有振動発振対策やターボ機械での不安定振動を励起する作動流体からの流体力に対する補償のためディジタルロバスト制御,(3)ノッチフィルタ挿入,(4)回転体の質量分布の不均一性による不釣合いによる振動を補償して軸振動をなくす制御(アンバランス相殺制御:unbalance counteracting control),(5)回転軸の振れまわりは許容するが,不釣合い慣性力による影響が静止構造物に伝わらない制御(アンバランス力遮断制御:unbalanceforce rejection control)などの手法により制御される.さらに磁気軸受の励磁電流と磁気吸引力は非線形な関係となるので,非線形制御も研究されている.e. 信頼性磁気軸受を用いてロータをフィールドで加振して,ロータ磁気軸受システムの開ループ伝達関数を計測し,感度関数を求めることができる.感度関数の逆数が古典制御のゲイン余裕や位相余裕に対応しているので,ロータ磁気軸受システムの安定性をフィールドで評価しながら運転できる点がこの軸受の優れた特徴である.電気部品点数が多いので,信頼性を高めるため,システムの多重化を採用することもある.f. DN 値ロータの外径(単位mm)と回転数(単位rpm)の積をDN 値と呼ぶ.許容DN 値は,ラジアル軸受ではロータ鉄図-3・52 焼結含油軸受の適用例図-3・50 ラジアル磁気軸受心材料,アキシアル軸受ではアキシアルディスクの鉄心材料の強度によるがおおむねラジアル軸受で4×10,アキシアル軸受では1×10である.g. 軸受損失(bearing loss)軸受ロータに発生するパワー損失は風損と鉄損(渦電流損,ヒステリシス損)である.風損は軸受すきま内流れをクエット流れとして求めることができる.渦電流損は욄=π윯율융욄욞욞욄욞욞6ρとなる.ここで,,윯율융,욄욞욞,욄욞욞,ρは積層板の厚さ,ロータ鉄心の最大磁束密度,有効周波数,ロータ鉄心の有効体積,ロータ鉄心の電気抵抗率である.ヒステリシス損は高速回転では渦電流損に比べて小さい.ステータでは荷重を支える励磁電流と磁気軸受の特性を線形化するバイアス電流が励磁コイル直流抵抗を流れることに起因する銅損が発生する.電力増幅器ではPWM アンプのon-offの動作でのパワー損失やリニアアンプでのコレクタ損失が発生する.
