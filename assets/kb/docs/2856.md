# 3・3・1 転がり軸受(rolling bearing)の種類と選択

> **章**: 第3章 軸・軸受要素
> **節**: 3・3 転がり軸受

a. 転がり軸受の種類転がり軸受は、回転している部位を支持する回転用転がり軸受と直線運動している部位を支持する直動用転がり軸受に大別される。しかし、一般には転がり軸受は回転用転がり軸受を指し、直動用転がり軸受は直動軸受(リニア軸受)と呼ぶことで区別されている。回転用転がり軸受および直動軸受は、国際的にはISO(国際標準化機構)のTC4(第4技術委員会転がり軸受)において標準化の作業を行っている。日本でも日本工業規格JIS B 1511:1993転がり軸受総則」の付表1および2(軸受に関する規格)に示すような諸規格が制定されており、さらにJIS を補足するために日本ベアリング工業会規格が制定されていて、ISOとの整合性が図られている。これらの諸規格では、使用上関係のある輪郭寸法、精度(内部すきまを含む)、定格荷重などの計算方法などについて規定しているが、軸受内部設計については、技術的進歩を阻害しないために規定しておらず、軸受メーカの自由に任されている。したがって、動定格荷重、静定格荷重の値などは軸受メーカのカタログを参照する必要がある。回転用転がり軸受は、転動体の種類や列数や接触角などにより例えば図I-3・56のように分類される。そのほか、分離型や非分離型の軸受、フランジ付き軸受、軸受内径を12分の1(または30分の1)のテーパ穴にしたテーパ穴軸受、グリースを封入したシールまたはシールド軸受などがある。深溝玉軸受(deep groove ball bearing)や自動調心ころ軸受(self-aligning roller bearing)などは1個の軸受でラジアル荷重と両方向のアキシアル荷重を受けることができ、アンギュラ玉軸受および円すいころ軸受はラジアル荷重と一方向のアキシアル荷重を受けることができる。ここで、接触角(contact angle)の大きい形式の軸受ほど大きなアキシアル荷重に耐えることができる。また、一般に玉軸受のほうがころ軸受より高速回転ができる。したがって、高速回転でのアキシアル荷重用としてはアンギュラ玉軸受が使用される。またラジアルすきまの大きい深溝玉軸受も接触角は増大し、比較的大きなアキシアル荷重に耐えられるので、3点接触または4点接触玉軸受とともに高速のアキシアル荷重用として用いられる。円筒ころ軸受は、ラジアル荷重を受ける形式の軸受であるが、内輪および外輪につばのある軸受やL形つば輪付き軸受を用いれば、ある程度の大きさのアキシアル荷重を支持することができる。直動軸受は急速に発展してきたこともあり、標準化はこれからの段階である。ISOではISO 10285、「玉循環式スリーブ形リニア軸受の寸法及び精度」およびISO 13012、「スリーブ形リニア軸受の付属品」の規格があるのみで、国内ではISO 10285、「玉循環式スリーブ形リニア軸受の寸法及び精度」に対応する日本ベアリング工業会規格があるだけである。図-3・56 転がり軸受の分類例直動軸受の代表的なものを図I-3・57に示す。直動軸受は軌道の形式、転動体の種類、転動体循環機構の有無などにより分類される。b. 転がり軸受の選択軸受の選択は、3・2節で述べられている滑り軸受と3・3節で述べる転がり軸受の特徴や性能を比較して決定することになる。転がり軸受は、a項に示したように国際的な規格に基づいているため互換性に優れ、また大量生産による経済性に優れていることにより用途が広がっている。
