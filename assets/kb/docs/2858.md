# 3・3・3 直動玉軸受

> **章**: 第3章 軸・軸受要素
> **節**: 3・59 のように時間の間は荷重および回転速度が一定であ

直動玉軸受(linear motion ball bearing,ball bushing)は、1945年にアメリカで発明され、わが国には1960年代に導入された機械要素であり、軽荷重用で取付けや保守等簡便な丸軸の案内作用を行う往復直線運動用軸受としての機能を果たす。その形式を大別すると、鋼球が循環運動するものと循環運動しないものとに分けられる。前者は往復直線運動のストロークに制限がなく、軸周りに回転させることができない。一方、後者はストロークが限定されるが、軸周りに回転・揺動運動を行わせることができる。呼び寸法は軸径で与えられ、現在=φ3~φ100mmの軸に適応する軸受が製造されている。潤滑は、初期グリースを封入したままで大部分が軸受寿命に至るまで無交換で使用される。a. 直動玉軸受の構造と種類。 循環式直動玉軸受(circulation type of linear motionball bearing)図I-3・60(a)(b)に示すように循環式直動玉軸受は、軸受取付けのため外筒外周に止め輪溝や丸フランジまたは角フランジを設け、鋼球とプラスチック製保持器、保持器止め輪およびシールから構成され、外筒の内側に保持器を固定して取付け、その保持器の軌道溝内を玉が循環運動する構造になっている。外筒は軸受鋼または浸炭鋼を材料とする円筒の数箇所(3~6箇所)に玉の転動する軌道面が長手方向に設けられている。このような軌道面は管材または精密鍛造したフランジ付き円管状材料を旋削加工後内面をブローチ加工して削り出されるか、あるいは管材内面を多角形に引抜き後、旋削加工して作られるのが一般的である。また、図示しなかったが、負荷容量を増大するため外筒長さを2倍強にして軸方向に複列の保持器を設けたものや、欧米ではプラスチック製外筒を成形加工し、各玉列位置に研削加工を施したサーキュラアーク溝付き鋼製コマを外筒として配置した構造のものもある。プラスチック製保持器はその両端を外部からシールするとともに保持器止め輪によって外筒に固定され、位置決めされている。保持器には、玉が循環する2本の直線部分と両端の曲面部分とをもつだ円形状の軌道溝が成形加工されている。軸側の直線溝部分には玉の一部が露出し、しかも玉が脱落しないように玉径よりやや幅の狭い切欠窓が設けられ、この部分にある鋼球が軸表面と接触しながら、保持器の軌道溝内で循環運動をして軸を支持することになる。保持器内に充てんされる玉数は、保持器両端における軌道溝の曲面部で玉が円滑に方向を転換できるように、溝中心の循環全長を玉径で除した玉数より1個分程度少なく若干のゆとりが設けられている。このような保持器は回転用転がり軸受の保持器とは異なって、一群の玉全体を保持するものであるから各玉間の相互接触を防止できるような構造にはなっていない。また、軸と接触しない側の保持器溝と外筒内面とで形成する空間内における玉の循環は、玉どうしの押合い作用で循環運動が行われる構造になる。また、防じん対策として使用されるシールの材料は標準品としてゴム材が、指定すればフェルト材が用意されている。フランジ取付け型以外の軸受は、次の3形式に分類され、それぞれに特徴を備えているから使用目的に応じて最も合理的な形式を選定するとよい。(a) 標準型:図I-3・60(a)およびI-3・61(a)に示すような最も代表的な直動玉軸受であり、種々の直線運動機構の案内用として広範囲に使用される。この軸受は外筒加工が容易なため、精度上から上級、精密級、超精密級と精度等級が区分され、製造されている。この形式における軸受予圧は、外筒内面と軸表面間の寸法に対して数mから十数m程度のオーバサイズ球を組込むことによって付与される。(2) 調整型:図I-3・61(b)に示すように外筒の一箇所で長手方向にスリットが設けられ、外筒はある程度まで直径を調整できる構造になっている。したがって、この軸受を穴径の調節可能な軸受箱に取付けると、軸に対して全くすきまのない状態や適当なラジアル方向の予圧(preload)を付与した状態で軸受を運転することが可能となる。(3) 開放型:図I-3・61(c)に示すように外筒の一部分を一玉列分だけ長手方向に切欠いたもので、一列上に何箇所か支点を設けて曲げ剛性を高めた案内軸にして使用することができる。また、調整型と同じように若干の直径調節も可能であるから、予圧を付与した状態で軸受を運転することもできる。用途は、おもに軽荷重用の1軸または2軸テーブルが多い。. 非循環式直動玉軸受(non-circulation type of linearmotion ball bearing)図I-3・62に示すように、外筒、玉、保持器および止め輪からなり、軸または外筒ストロークに対して保持器は軸受内で/2だけ直線運動を行う。この軸受は、構造上限定された直線運動と限定されない回転運動および両者の組合せ運動に対する支持機能をもっている。また、図I-3・62(a)に負荷分布状態を示したが、図I-3・62(b)に示したように玉が奇数列と偶数列に対して交互にできるだけ数多く配置される構造のため、軸受の静定格荷重および動定格荷重は同一呼び径の循環式直動玉軸受に比べて大きいのが一般的である。また、軽荷重用に対して中荷重用はストロークがほぼ2分の1で、負荷容量は2倍程度であり、外形寸法は同一である。b. 特徴直動玉軸受の大きな特徴の一つは、摩擦抵抗が小さいことである。循環式軸受の起動摩擦係数は作用荷重が小さい場合で0.005程度、作用荷重が基本動定格荷重の2分の1程度になると0.002程度に低下する。動摩擦係数は、普通0.001~0.005で同一荷重条件ならば玉列数の多いものほど値は低下する。また、非循環式軸受の摩擦係数は循環式のおよそ2分の1程度の値である。なお、静定格荷重に対応するヘルツ応力は最大玉荷重が働く軸側軌道面で5300MPaがとられ、このときヘルツ変形に加えて、玉径の1/10000の塑性変形が接触部に生起する。また、寿命を問題にするときの外部荷重、すなわちラジアル荷重は<2に制限されるのが普通である。なお、非循環式の直動玉軸受の場合、玉列数が多いため荷重状態(A)(B)の区別はしない。c. 定格寿命一般の回転用転がり軸受と同様に90%定格寿命は、単位走行寿命を50kmまたは100kmとして次式で計算される。温度係数を表I-3・21に示す。軌道面温度が100°Cを超える場合に使用する。もちろん、このような場合にはシールや保持器も高温仕様にする必要がある。表I-3・22に接触係数を示す。外筒の長い直動玉軸受や複数の軸受を密着させて使用するときのモーメント荷重や軸変形の影響等を考慮する場合に使用する。荷重係数を表I-適宜使用する。なお、直動システムを設計する場合、基本動定格荷重は単位走行寿命が50kmの場合には、100kmの場合に対して1.26倍の大きさになっていること、一つの直動玉軸受にモーメント荷重が作用した場合には、時として接触係数で定格荷重の補正を行った計算走行寿命以下に寿命がかなり低減するので、注意しなければならない。この場合の直動システムとしての寿命計算は、各軸受がいずれも90%の信頼度で残存しなければならないという確率乗法定理を用いて、次式で求められる。
