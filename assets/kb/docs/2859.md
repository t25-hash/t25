# 3・4・1 滑り案内(sliding guide way)

> **章**: 第3章 軸・軸受要素
> **節**: 3・4 案内

滑り案内には、直線運動と回転運動を与えるものがあり、動圧型、静圧型および両者の中間型の半浮上式案内面がある。テーブルやサドルなどしゅう動体の運動は往復動形式となる。動圧案内は主として工作機械の案内面として使われてきたが、切削力のような大きな外力が作用しない測定器、光学機器などの送り案内やXY テーブルにも使われる。超精密切削加工機や半導体製造関連装置など、特に高い精度が要求される案内には静圧案内や転がり案内が常用される。滑りと転がりを組合わせたハイブリッド型の案内もある。3・4・1項でははん用の案内である動圧型の直線滑り案内について述べる。a. 案内(guide)に要求される条件滑り案内は主として以下の条件を満足しなければならない。1しゅう動体がスムーズに正確に移動できること(運動精度)。2しゅう動体を正確に位置決めできること(位置決め精度)。3荷重や切削反力に打ち勝つ剛性があること(幾何精度)。b. 案内面の形状と構造直線案内は一般に図I-3・64に示すような形式をもつ。しゅう動体は送りねじによって案内方向に往復移動できる。しゅう動体の運動は図I-3・64のように6自由度をもつが、a項の3条件を満たすには、案内方向以外の運動や変位はないことが望ましい。案内面断面形状は、図I-3・65のV(Vee)、平・平面(flatsurface)、ダブテール・あり・ばち(dovetail)、円・バーガイド(cylinder)の4要素を、図I-3・66~I-3・68に示す例のよう図-3・65 案内面形状図-3・64 滑り案内図-3・67 平-平組合せ図-3・66 V-平組合せ図-3・68 ダブテールに、単独あるいは2個以上平行に組合わせて使用する。図I-3・65の4要素のうち前2者の組合せがよく使われ、最も多いのは図I-で山形とも呼ばれ、摩耗粉排出に有利な使い方である。そのほか、平-平(図I-3・67)、平-V-平、V-V などの組合せを用途に応じて使い分ける。ダブテールはあり溝と一対で稼働するもので(図I-ある。c. 滑り案内の設計原理案内面としゅう動体に案内方向以外の不必要な押付力や回転・曲げモーメントが極力作用しないように、案内の機構を設計する必要がある。これらは種々の案内精度の低下を招くだけでなく、場所による接触圧力(面圧)の違いを招くため、案内面の不均一な摩耗や不安定な摩擦特性の原因となる。そのため案内面の設計には以下の考慮を要する。案内面の中には運動の基準となる基準案内面が必要である。しゅう動体は送りねじやラック・ピニオン方式などで送られるが、これらの駆動力の作用線は、可能な限り基準面に近づけ、案内面間に不要なモーメントがかからないように工夫する(アライメントの原理)。図I-3・66では逆V の山の両斜面が基準面である。図I-3・67は図I-3・64を案内方向に対して直角に切った断面図で、この平-平組合せの水平移動の基準面は左側ガイドの左側面である。この面に忠実にしゅう動体が移動するように、右側面にすきま調整用ギブ(くさび)を入れてガイドを挟み、両案内面を拘束する。同様な拘束は、右側のガイドを使っても可能であるが、この場合しゅう動体の回転モーメントやヨーイングが大きくなるので好ましくない。このように水平面内において直線運動を拘束する2案内面の間隔を可能な限り狭く設計する方式をナローガイドの原理(narrow guideprinciple)と呼ぶ。ギブは図I-3・68のようにあり溝にも入れる。1案内方向で基準面は平行に2面必要である。垂直方向の案内精度も重要である。潤滑油が過多で移動速度が大きい場合、しゅう動体が浮上する可能性があるので、図I-3・64や図I-3・67に示すバックプレートを配置し、例えば図I-3・67の面を使って上下運動を拘束する。負荷容量であるしゅう動面圧は一般に0.1~0.2MPa、精密案内面で0.03~0.05MPa、重切削機械で0.3~0.4MPaを考える。d. 案内機能向上対策案内面の機能向上のための一般的な対策を表I-3・24に概説する。表-3・24 案内機能向上対策内容必要な性質構造簡単、高剛性、姿勢誤差小、組立・調整容易材料加工しやすい、高精度加工可、熱的に安定、経時変化小潤滑適切な油溝、適切な潤滑油、適切な給油法摩擦摩擦力が変動しない、スティックスリップがない摩耗低摩耗率、偏摩耗しない、自己修正機能あり防じん切りくずやごみが堆積しにくく除去しやすい図-3・69転がり案内の構造4―55第I部機械要素
