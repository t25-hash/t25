# 3・4・2 転がり案内(rolling guide way)

> **章**: 第3章 軸・軸受要素
> **節**: 3・68 ),形削り盤のラムのガイドは図I-3・68の上下反転使用例で

機械のベッド上を直線運動するテーブルなどのように,直線運動を行う移動物体の案内システムとして運動部分に玉やころの転動体を組込み,転がり運動によって案内機能を得る転がり案内が,工作機械,半導体製造装置,工業用ロボット,各種搬送装置などに幅広く使用されている.a. 転がり案内の構造代表的な転がり案内の構造は,図I-所定の長さをもつ断面がほぼ矩形のレールと,レールにまたがるように組合わされるキャリッジから構成されている.キャリッジは,レールに設けられた転動面上を移動するために,キャリッジ内部には玉あるいはころのような転動体が複数列(2条列,4条列,時には6条列のものもある)組込まれており,キャリッジの移動に伴って,荷重を負荷しながら転がり運動を行う.転動体はキャリッジの両端に設けられたエンドキャップにより方向転換を行って,再び負荷部へ移動する循環式をとるものが大半である.転動体として玉を使用する場合,キャリッジとレールの転動面には図I-3・70(a)サーキュラアーク溝形状(単一円弧溝状)や図I-3・70(b)ゴシックアーク溝形状(2円弧溝状)がある.循環式のものは移動物体の移動ストロークに制限はないが,非循環式のものは移動ストロークに限界があるほか,理論上は移動物体の移動量の2分の1だけ転動体の中心が移動するため,荷重条件によっては転動体列の片当たり状態を生じるので注意が必要である.荷重が加わると玉と溝間では接触面積が広がり,玉と平面が接触する場合に比べ,溝の曲率にもよるが,5~13倍の荷重を負荷することができ,また,許容弾性変形量もかなり増大する.転動面は,高周波焼入れまたは浸炭焼入によりHRC58~64程度の硬さとなり,成形研削により精度を得る.精度は,レールに設けられた底面基準面および側面基準面と組合わされるキャリッジの上面基準面および側面基準面により,走行平行度として規定されている.また,今までの転がり案内は,ほとんど転動体どうしが直接接触しながら循環する総転動体式であったが,最近の傾向として,玉と玉あるいはころところの間に間座を設けて,直接の接触を防止することにより,摩擦変動の低減,長寿命,静音などの性能を向上させたものが開発されてきている.b. 転がり案内の特性. 予圧転がり案内はa項に示したように,レールとキャリッジ間を複数の条列で案内する構造をもっているため,内部すきまをあらかじめ設定することができる.内部すきまをマイナスとして,予圧(preload)を加える場合が多い.転がり案内に予圧を加えることで1がたのない直線運動2剛性の向上3軸方向摩擦の増大による減衰性の向上などの特性を得る.予圧は,キャリッジ,レール転動面間の寸法より数mプラスした寸法の転動体を挿入することにより,内部で転動体および案内面が弾性変形することにより加えられる.予圧を加えすぎると,1転がり寿命を低下させる2発熱が増え,潤滑寿命を低下させる3摩擦および摩擦変動が増加するなどの機能低下をもたらすので,機械に応じて最適値を求める必要がある.予圧は上記の特性と問題点を考慮し,定格荷重のほぼ10%以下に調整されることが多い.予圧は特に,剛性に与える影響が大きく,玉を使用した場合の予圧が零(零すきま)の場合と10%の場合の剛性の一例を図I-3・72に示す.0%と10%を与えた場合では剛性に2倍の違いが得られる..
