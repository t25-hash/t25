# 3・5・1 シール(seal)の種類と選択

> **章**: 第3章 軸・軸受要素
> **節**: 3・5 シール

a. シールの種類シール(密封装置)は軸受・歯車と同様に、機器において不可欠なトライボ機械要素として位置付けられている。シールの役割は、機器で使用される流体などの外部漏えい防止と異物などの機内侵入防止である。近年、機器の小型軽量化、高性能・高信頼性、環境対策の観点からシールの機器に果たす役割はますます重要となり、新形状・新材料を付与したシール開発が進められている。図I-3・76にシールの種類を示す。ここで、シールは相対運動の存在する面間に用いられる運動用シール(dynamic seal)と、相対運動の存在しない面間に用いられる固定用シールに分類される。さらに、運動用シールは相対運動する2面間を適正面圧にて押付け、見掛け上のすきまをなくした状態とした接触式シール(con-tact seal)と、一定のすきまを確保した非接触式シール(non-contact seal)に分類される。また、これら多くの形式に加え、合成ゴム、樹脂、金属、セラミックス、カーボンなどを母材とする複合材料を使用することにより、さまざまな用途、使用条件に対応可能なシールが設計されている。b. シールの選択シールの選定においては、各種密封装置の使用限界を充分理解しておく必要がある。特に接触式シールの使用限界は、密封流体の種類、シール構造と使用材料、運転条件、シール周りの環境などによって、大きく左右される。また、シール材料として合成ゴムが広く用いられるが、選定に当たっては不具合防止のためにも、シール材料の耐熱・耐寒・耐油・耐液・耐燃料性や潤滑添加剤との適合性、さらには耐オゾン・耐摩耗・圧縮永久ひずみ特性などを充分理解しておく必要がある。なお、接触式、非接触式シールの選択においては、図I-3・77に示すフローチャートが参考になる。
