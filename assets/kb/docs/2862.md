# 3・5・2 静止シール(static seal)

> **章**: 第3章 軸・軸受要素
> **節**: 3・5 シール

固定用シールとも呼ばれる静止シール[ガスケット(gasket)]は、相対運動せずに固体接触する2面間に用いられて、接触面のす図-3・76 シールの種類図-3・77 シール選定4―57第I部機械要素きまを通って内部の流体が漏出したり外部から異物が侵入するのを防止する。フランジや管継手などの対向する二つの壁面(平面、円筒面)の間に挟み込まれたガスケットは、ボルトやねじなどの締結力が加えられると内部に圧力が発生し、これにより固体壁面への密着度と面圧を高めるとともにガスケット自体への流体の浸透を抑止して、流体の漏れを封止する。表I-3・25に示すように、取付け部の形状や密封対象の流体(以下、密封流体という)、使用する温度、圧力に対応して種々のガスケットがある。以下、表I-3・25に添って説明する。ガスケットの材質は、面圧を維持できるようクリープや応力緩和、熱劣化による経時変化が小さいものを、また接触面状態を維持できるよう化学的安定性(耐膨潤、耐腐食性)の高いものを選定する。代表的なゴムガスケットのOリング(O-ring)は、安価で入手しやすく、取付けに必要なスペースが小さいことから最も広く使用される。Oリングは、多くの場合、壁面上に加工された幅の狭い取付け溝内に装着して使用される。流体の圧力が増加すると、変形が増大して壁面との接触面積が大きくなるとともに接触圧も高くなるので、漏れがおのずと防止されるセルフシール機能をもつ。しかし、線径の細いOリングは、装着時あるいは締結時にねじれて変形する場合があり、密封性能が低下する。このような場合、代わりにD リングやT リング(表I-3・26参照)を使用するが、溝内部での倒れや座屈が生じないよう溝は充分注意して設計する必要がある。ゴムガスケットは取付面の粗さにしっかり密着し、圧力変動や振動があっても面の密封効果を維持できるが、密封流体の種類によってはガスケット内部に流体が浸透・透過して漏れる場合がある。使用圧力がゴムガスケットの耐圧を超える場合は、ゴムやPTFEをバインダとして種々の充てん材を配合したジョイントシートやPTFE複合ガスケットなどの非金属ガスケットを使用する。膨張黒鉛シートから所定形状に裁断して使用される黒鉛ガスケットは、化学特性に優れ、極低温域でも軟らかく優れた低温特性を示す。このほかの非金属ガスケットとしては、高分子材料を溶剤で溶解し充てん材などの配合材を加えた液状ガスケットあるいはシーラントがあり、簡易なガスケットとして密封面に塗布して使用される。準金属ガスケットは、非金属材料の芯材の周りを金属で被覆して、芯材の圧力緩衝作用と金属のばね作用により流体を密封する。圧力緩衝作用を有するので、シール面への追従性をもち金属ガスケットより粗さが大きな密封面に使用することができる。金属ガスケットは、非金属ガスケットおよび準金属ガスケットが使用できない高温・高圧・極低温・高真空など過酷な条件で使用される。金属の反発弾性を利用してシール面に面圧を発生させることでシール機能をもたせた平形・波形ガスケット、密封流体の圧力を利用してシール面の圧力を高めるセルフシール型のリングジョイントガスケット、ガス封入により弾性に富みシール面への追従性を高めたメタル中空Oリングがある。金属ガスケットは剛性が大きいので、固体壁面の粗さに密着するように変形することが非金属あるいは準金属ガスケットに比して難しい。このため取付面の粗さが当初から大きかったり、あるいは異物侵入や腐食による取付面のあれが増大するとガスケットと壁面の密着度が低下する。その結果、すきま漏れを生じたり(振動に対する追従性の不足があると漏れが増大する)、圧力変動や熱収縮による流体の掻き出しが生じることがある。したがって、壁面粗さを小さくしたり、メタル中空Oリング以外は締結力を大きくする必要がある。
