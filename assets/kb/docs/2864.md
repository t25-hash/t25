# 3・5・4 非接触式シール(non-contact seal)

> **章**: 第3章 軸・軸受要素
> **節**: 3・5 シール

a. オイルフィルムシール(oil film seal)オイルフィルムシールは,化学プラントやパイプライン用の高速・高圧ガス圧縮機によく使用される.機内ガスは可燃性や毒性がある場合が多く,外部漏えいを零にするために通常図I-3・83のようなシールシステムとして構成される.ガス側と大気側の一対のフローティングリングシール(floating ring seal)を軸貫通部にもち,両者の間に機内ガス圧より高い(0.02~0.1MPa)圧力のシール用潤滑油を供給し,軸とシールリング間に油膜を形成して機内ガスを密封するものである.フローティングリングとする理由は,運転時の軸位置の変動や軸振動に対しできるだけ追随して同心のシール油膜を形成するためである.リングには連れ回りを防止する目的で回り止めを設ける.圧力で20MPa,周速で150m/s程度までの範囲で多くの使用実績がある.シールリング使用材料は,潤滑油が使用されるので,通常の油潤滑の軸受と同一材料(内周部にホワイトメタル,裏金部に炭素鋼)が使用される.腐食性のガスの場合,耐食性の観点からホワイトメタルの添加元素の成分を変更することがある.シールリング部での潤滑油流量,摩擦損失は,流れが層流の場合,次式で求められる.=πΔ1+1.5ε/12μ(I-3・39)=πμ2-3ε/2+ε/21-ε(I-3・40)ここで,:軸直径m,:シールリングの半径すきまm,4―66機械要素・トライボロジー:シールリングの幅m,Δ:シール部差圧Pa,:軸回転速度1/s,μ:潤滑油粘性係数Pas,ε:シールリング偏心率,π:円周率である.なお,停止時は,シールリングは軸と接触しているので偏心率ε=1を,高速回転時は油膜で浮上するので,近似的にε=0を用いてよい.ただしリング自重やシール差圧が大きい場合は,リング自重とリングに作用する流体力による静止部との接触部摩擦力の合力と,第II部1・4・2項の軸受理論から求まるシール油膜反力の平衡から偏心率を求め,油膜が確保される設計となっているかを確認する必要がある.また,シール部での油温上昇Δは,潤滑油が摩擦損失による発熱をすべて吸収すると仮定すると,式(I-3・39)の,を用いて次式で求められる.Δ=/γ(I-3・41)ここで,:熱の仕事当量Nm/J),γ:潤滑油の密度kg/m,:潤滑油の比熱[J/kg°C]である.なお,機内側のシールでは差圧が小さく流量も少ないので,摩擦損失発熱をシール部の漏えい油量ではほとんど吸収できないことが多いので,シールリング部外周部での冷却で吸収する設計とする必要がある.また,差圧の大きい機外側シールリングは圧力による変形が無視できない場合が多く,この変形量を計算しすきまの減少を考慮して製作時すきまを設定することが必要である.b. 装置,スパッタリング装置など,また分析機器である電子顕微鏡,XPS などに使用されている.利便性から多軸構造になっているものや,ベローズと組合わせて回転運動のみならず多少の往復運動も可能なシールも開発されている.特に半導体製造装置用としては,クリーン度が製品の価値を左右することから磁性流体の蒸気圧を7×10Pa(293K)にまで抑えたものもある.また磁性流体,グリースの耐熱性に限界があるので,高温下における使用に際しては冷却する必要が生じてくる場合がある.
