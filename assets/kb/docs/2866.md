# 3・6・2 フランジ形固定軸継手(rigid flanged shaft coupling)

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

図I-3・89に示すように,継手本体のフランジをボルトによって締付けて結合するもので,一般に広く使用されている固定軸継手である.この継手の構造,寸法および材質などはJIS B 1451:1991「フランジ形固定軸継手」によって規定されており,継手本体にはFC200,SC410,SF440A あるいはS 25Cが用いられている.また継手のボルトには4~10本のリーマボルト(SS 400)を用い,ばね座金を緩み止めに使用する.継手の選択に当たっては次式で計算される許容伝達トルク(N・m)をもったものをカタログなどから選ぶ必要がある.=159 (I-3・45)図-3・89フランジ形固定軸継手図-3・88 磁性流体真空シール4―68機械要素・トライボロジーここで,:伝達動力(kW),:毎秒回転数(s)で,は負荷状態や速度によって決まる安全係数で,一様な負荷の場合で=1~2,変動負荷に対して=1.5~3,衝撃負荷に対して=2~4を用い,回転速度が速い場合(20s以上)にはは大きいほうの値を用いる.図I-3・90は軸の直径と軸継手の性能との関係の概略を示したものである.
