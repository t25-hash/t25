# 3・6・5 歯車形軸継手(gear coupling)

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

いわゆるギヤカップリングと呼ばれているもので、図I-3・93に示すように、外筒にある内歯車と内筒にある外歯車をかみあわせてトルクの伝達を行う。その歯形はインボリュートで圧力角20°が多い。このような構造で軸心の狂いを許すことができるのは、外歯の歯先と歯面にクラウニングを施してあるためで、外歯は内歯に対し図-3・90 フランジ形固定軸継手の性能図-3・91 フランジ形たわみ軸継手て傾きを与えることができる。外筒の中心線に対して内筒の中心線の傾き角を傾斜角といい、その角度は一般に1.5°以下のものが多いが、特殊な要求によっては8°程度まで傾くことができるものもある。したがってこの軸継手では歯面の潤滑のために良質なグリースあるいは潤滑油を封入しておく必要があり、油漏れに注意し、密封装置は確実でなければならない。継手外径の周速度40m/s以下、回転速度66.67s以下のものについてはJIS B 1453:1988に構造、寸法、材質などが決められており、表I-3・35は規格に定められた寸法の歯車形軸継手の性能の一部である。
