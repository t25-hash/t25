# 3・6・6 ローラチェーン軸継手(double roller chain coupling)

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

チェーンカップリングとも呼ばれ、図I-3・94に示すように、スプロケットの付いた継手本体を二列ローラチェーンで結合し回転力を伝える。この軸継手はスプロケットの歯とローラ間の遊び、ローラとブシュまたはブシュとピンのすきまおよびチェーンのたわみなどによって、軸心のわずかな狂いを吸収することができる。性能はフランジ形たわみ軸継手よりやや劣っているが、外径が比較的小さいという特長がある。しかし、歯面とローラ、またはローラ、ブシュおよびピン間の摩耗を避けるために潤滑しなければならない。JIS B 1456:1989には最大軸穴直径=22~200mmのローラチェーン軸継手の構造、寸法、材質などが規定されている。表I-3・36にはその主要寸法および性能の一部を示した。なお、表中の許容伝達トルクは軸心の狂いが小さく、回転速度も比較的遅い場合の値であって、軸心の狂いが大きく、回転速度が速くなるとその値は表I-3・36の値より小さくなる。
