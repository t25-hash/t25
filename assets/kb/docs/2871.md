# 3・6・7 ゴム軸継手(rubber shaft coupling)

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

ゴムを多量に使用し、ゴムの圧縮やせん断などの弾性効果を利用した軸継手をいう。形状は図I-3・95(a)の星形ゴム軸継手や(b)のタイヤ形ゴム軸継手など種々の形状のものがあり、潤滑が不要なこと、運転中騒音を発しないこと、2軸の電気絶縁が可能なことなどの利点もあるが、構造上外径寸法が大きくなること、同一軸穴直径を有するフランジ形たわみ軸継手に比較して許容伝達トルクや許容回転数が約2分の1に低下するなどの欠点もある。図-3・96金属ばね軸継手る。JIS B 1455:1988ではゴム軸継手を軸心の狂いの許容値によってE1,E2,E4の3種類に、またはねじりばね定数の許容値によってK 2,K 4,K 8,K 16の4種類に分けている。表I-3・37はその主要寸法と性能の例である。なおここで、許容トルクは一方向に定常速度で長時間にわたって伝達することができるトルクのことであって、短時間であれば最大伝達トルクは許容伝達トルクの約2倍の大きさまでとれる。
