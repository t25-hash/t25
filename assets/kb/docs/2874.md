# 3・6・10 フック形自在軸継手(Hooke’s universal joint)

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

2軸が同一平面上にあり、かつその中心線がある角度で交わる場合の伝動装置で、カルダン形軸継手(Cardan’s joint)とも呼ばれ4―70機械要素・トライボロジーる。図I-3・98(a)に示すように原動軸および従動軸の回転角をθおよびφとし、その角速度をθおよびφとすれば、これらの関係は次式によって与えられる。したがって軸が1/4回転する間に速比はcos αより1/cos αに変動する不等速自在継手であって、伝達されるトルクも速比に反比例して変化する。そこで回転を滑らかに伝えるためには図I-3・98(b)のように中間軸を用い、その両側に軸継手を取付けて用いる。構造は図I-3・99に示すように、ヨークをもった継手本体と十字金具からなり、この間にニードル軸受または軸受ブシュを挿入し、これをグリース潤滑するようになっている。軸の交角はα=45までとれるが、動力伝達用では回転速度が≒33sのとき、α≦20とする。一般に回転速度が大きいときは交角を小さく、回転速度が小さいときは交角を大きくとることができる。図I-3・99のような構造の自在軸継手は比較的大きなトルクを伝えることができる。
