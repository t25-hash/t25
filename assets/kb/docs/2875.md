# 3・6・11 こま形自在軸継手(universal ball joint)

> **章**: 第3章 軸・軸受要素
> **節**: 3・6 軸継手

工作機械や産業機械によく使われる不等速形軸継手で、構造はフック形自在軸継手と本質的には変わらない。JIS B 1454:1988に構造、寸法が定められており、図I-3・100に示すように、本体とつめ、さやおよびこまからなるA 形と、本体とこまおよびピンからなるB 形、およびそれらを複式にしたAA 形、BB 形がある。動力伝達用としては2軸の交角は普通30ぐらいまでであって、例えば45で使用するような場合には伝達トルクを十分に小さくとり、また回転数も低くする。表I-3・39はJIS B 1454に規定されたA 形、AA 形の主要寸法と性能である。なお、こま形自在軸継手と同じ形状で、ヨークで直接ボールを抱く構造をしたボール形の不等速形軸継手もある。
