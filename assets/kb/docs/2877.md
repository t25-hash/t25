# 4・5・1 半溶融（チクソ）加工

> **章**: 第4章 塑性加工
> **節**: 4・5 塊状品の塑性加工

a.定義 半溶融（チクソ）加工のチクソとは，1970年代，ア
メリカ，MIT（マサチューセッツ工科大学）のフレミングス（M.
C.Flemings ）らの研究によるチクソキャスティング法（thixocast-
ing process ）から取り入れられた語で，半溶融金属の特性の一つ
であるチキソトロピー（thixotropy）に由来している．木内
は，半溶融加工について，「素材内部に固相成分と液相成分が定常
的または非定常的に共存する半溶融の状態にある素材に，荷重また
は圧力を加え，変形・流動・撹拌・混合・圧入・接合・分離，など
の加工を加え，所要の形状・寸法・構造・特性を有する製品を得よ
うとするもの」と述べている．現在では，半溶融（チクソ）加工
は，半溶融処理または半凝固処理された素材を再加熱し，半溶融状
態で加工する，といった意味に受け取られている．なお，半溶融
（状態）とは，固体金属を加熱して得られる状態を表し，一般に結
晶粒界から溶融が始まり，結晶レベルで固相成分と液相成分が分布
した状態をいい，半凝固（状態）とは，溶融金属を冷却して得られ
る状態で，一般にデンドライト間に液相成分が存在する状態をい
う．
b.半溶融・半凝固処理材の製造法 半溶融（チクソ）加
工では，均一微細な等軸結晶組織の素材を半溶融加工する前に作製
する必要がある．これには，均一（単純）加熱法，SIMA法，電
磁かくはん法，傾斜冷却法，せん断冷却ロール法等が提案されてい
る．

.均一加熱法（homogeneous heating method) 金属材
料をできうる限り均一に加熱し，その内部に平衡な半溶融状態を作
り出す方法．
.SIMA法（strain induced melt activated method) 金
属材料に冷間加工によってひずみを付与し，その後再加熱して半溶
融処理する方法．
.レオキャスト法（rheocast method) モールド内にか
くはん子を設置し，モールドを冷却してかくはん子を高速で回転さ
せつつ上部から溶湯を注入，モールド壁面から成長する凝固層をか
くはん子で破砕して溶湯中に分散させ，溶湯と初晶粒子との混合ス
ラリーを得る方法．
.電磁かくはん法（electromagnetic stirring method)
溶湯が温度制御されたモールド内を通過中に，モールド壁面から成
長する凝固層を電磁力によってかくはん・破砕し溶湯中に分散さ
せ，溶湯と初晶粒子との混合スラリーを得る方法．
.傾斜冷却法（cooling slope method) 傾斜した冷却板
上に溶湯を流下させ，予熱した鋳型に鋳造することによって，初晶
の組織形態が粒状化したビレット/スラリーを作製する方法．
.せん断冷却ロール法（shearing/ cooling roll method)
回転するロールと固定冷却シューの間に溶湯を注入し，固定冷却シ
ューと回転ロールとの間で溶湯にせん断を加えつつ冷却し，半凝固
化したスラリーを得る方法．
c .加工の形態
.半溶融押出し（mushy extrusion) 半溶融・半凝固処
理材を再加熱してコンテナに充てんし，ラム（パンチ）を介して加
工力を与え，温度管理されたダイス内を通過させつつ加工成形し，
一様断面の長尺の製品を得る方法．
.半溶融鍛造（mushy forging) 半溶融・半凝固処理材
を再加熱して金型に挿入し，おもに密閉型鍛造により所望の製品を
得る方法．
.半溶融圧延（mushy rolling) 半溶融・半凝固処理材を
再加熱し，これを適当な搬送方式によりロール間に挿入して所望の
製品を得る方法．
.半溶融ダイカスト（mushy die-casting) 半溶融・半
凝固処理材を再加熱してこれをコンテナに充てんし，通常のダイカ
ストと同様な手法により，金型内に圧入して成形する方法．
.半溶融接合（mushy joining) 半溶融・半凝固処理材
の一方または両方を再加熱し，圧力を加えて両者を接合させる方
法．

d.加工法の特徴
半溶融加工法の特徴を以下にまとめた．
(１)被加工材の変形抵抗の低下に伴う加工力の低下により，加工
設備の小型化が図れる．
(２)被加工材の流動性の増大により，高加工率での加工が可能と
なり，工程数の短縮が達成できる．
(３)難加工材の加工への適用が図れる．
(４)金属と金属，または金属とセラミックスなど，複合材料の製
造と加工に適する．
(５)半凝固処理後，ただちに加工を行うことで，溶湯からの直接
製品化が可能となる．
(６)被加工材ならびに工具の温度管理を厳しく行う必要がある．
