b.鍛造材料と温度 再結晶温度以上に加熱された金属材料

熱間型鍛造品は複雑形状を対象とするので，単工程で成形できる
場合は少ない．被加工材料の準備から始まり，予備成形（pre- for-
ming)，荒打ち（blocking，rough forging)，仕上打ち（finish
forging)，バリ抜き（trimming）など，多工程で成形される．後
工程として熱処理，酸化膜除去，冷間仕上げや仕上除去加工が組合
わされ，最終鍛造製品となる．
予備成形は，据込み，曲げ，背切り，密閉据込みや押出しなどの
基本成形加工（4･5･2項参照）を利用して，最終形状の材料配分に
近づける工程である．この良否は荒地型や仕上型の成形荷重，およ
び上記(１)～(５)の要求項目の質を支配するので，工程設計では重
要になる．材料配分が複雑な場合には，予備成形にロール鍛造
（roll forging）および他の回転鍛造（rotary forging）も利用され
る（4･5･5項参照)．
荒地型および仕上型の設計では，型充満（鍛造品欠陥)，鍛流線，
製品の離型，型寿命，型製作コストを考慮して，上下型分割位置で
あるパーティングライン（parting line)，バリ部形状，エゼクトピ
ン位置および型剛性（壁や底厚さ）が決められる．パーティングラ
インは，スラスト力が小さく単純で段差や彫込み深さが少なくなる
ように設計される．必要に応じて，型の分割構造，案内（いんろ
う，counter lock）なども採用される．
成形が難しくなる条件は，深い金型で小さなフィレットアール
（彫込みの浅い型では2.5mm以上，深い型では5mm以上)，小
さな抜きこう配，薄くて広いウェブ，高くて薄いリブやボス（通
常，高さ/幅＜4～5)，ウェブ（厚さ ）からリムボス（幅 ）への
厚さ変化の大きい分岐部（ / ＜2）である．これらは，被加工材
料の冷却および摩擦による荷重増加，座屈による欠陥発生の危険を
高める．これは型の寿命に影響する．工程設計では上記条件をでき
る限り分散緩和できるように工程分割が行われる．

なる．また，型の修正使用や局所的に耐摩耗性や耐熱性を改善する
ため，耐熱鋼の肉盛溶接や高温強度の高い工具鋼などを入れ子型と
して利用する場合もある．
g.潤滑および冷却 ハンマ鍛造では加圧接触時間が数ms以
下と短いため，離型作用が主となる．塗布容易でガス発生作用のあ
る油性黒鉛系の潤滑剤やおがくずが用いられる．プレス鍛造の場合
は，接触時間が長く摩擦滑りの大きくなりやすい押出し型では，油
溶性（黒鉛および非黒鉛系）がなお用いられている．熱負荷の大き
な場合には，型内外における水冷却が行われる．表面積拡大が小さ
く潤滑剤の閉込めが容易な場合にもっぱら用いられている潤滑剤
は，型冷却効果の大きな水溶性黒鉛および白色系の潤滑剤である．
型寿命を考慮しつつ油溶性から水溶性へ，黒鉛系から環境に優し
い非黒鉛系に置換える努力がなされる．水溶性潤滑剤はスプレー塗
布されるが，均質に必要量の膜を形成させるには，型温度
（150～300℃）の選定，ノズル形状と数の選択，乾燥時間の確保な
どの考慮が必要になる．
