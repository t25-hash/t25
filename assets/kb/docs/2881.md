# 4・1・5 その他の歯車

> **章**: 第4章 伝動要素
> **節**: 4・1 歯車

a. ねじ歯車ねじ歯車(skew gear,crossed helical gear)は、一対のはすば歯車から構成される食違い軸歯車である[図I-4・1(j)].表-4・9ウォーム歯形の種類ISOJISウォーム歯形形状A 形1形軸平面直線歯形ウォームI形4形インボリュートねじ面ウォームN 形2形軸直角断面直線歯形ウォームK 形3形フライス削りあるいは円すい面研削ウォームC形―中凹歯形ウォーム*ISO/TR 10828:1997,**JIS B 1723:19774―81第I部機械要素ねじ歯車の組立諸元は、歯車1と歯車2の軸角Σおよびかみ合い中心距離である。しかし歯車1と2の材料の組合せ、潤滑油および極圧添加剤の最適な選択、点接触を線接触に近づけるための凹形歯すじ[ホローリード(hollow lead)]の採用などにより、その負荷能力をかなり大きくできる。歯切りおよび歯面研削は容易である。b. コニカルギヤ平行軸歯車としてのコニカルギヤ(conicalgear)を図I-4・1(f)に示す。インボリュートはすば歯車を創成するラックを考え、次にこのラックの中心面を歯車軸に対して角δだけ傾ける。コニカルギヤは、この傾けられたラックによって創成される歯車である。コニカルギヤの基本諸元は、ラックの歯直角モジュール、ラックの歯直角圧力角α、ラック中心面でのねじれ角ψ、創成円すい角の頂角δ、および歯数である。なおこの歯車の基準円筒ねじれ角βは、角ψとは大きく異なる。コニカルギヤの外見はベベルギヤに似ているが、その歯面はインボリュートヘリコイドである。特例として、δ=0ならはすば歯車、δ=0,ψ=0なら平歯車となる。すなわちコニカルギヤは、インボリュート歯車の最も一般的な形といえる。このため、コニカルギヤどうしだけでなく、平歯車、はすば歯車、ラックおよびウォームギヤともかみ合う。軸方向に歯車をずらしてかみ合わせることにより、バックラッシを調整できるので、工作機械や自動車のステアリングセクタギヤなどに用いられることがある。また、小軸角の交差軸歯車・食違い軸歯車として、船舶用トランスミッション等に使用され、駆動系の小型化を実現している。歯車の諸元と加工法の組合せで歯車生産上の制限の出てくる割合は通常のインボリュート円筒歯車より大きい。c. WN 歯車軸直角平面あるいは歯直角平面で、基準ラック歯形の主要部分が円弧で構成されるような円弧歯形はすば歯車を、一般にWN(ヴィルトハーバー・ノビコフ、Wildhaber-Novi-kov)歯車と呼び、一方は凸、他方は凹の歯形の対でかみ合わせる図-4・21 WN 歯車の一例(第1種ノビコフ歯車)図-4・20 鼓形ウォームギヤ4―82機械要素・トライボロジー図-4・22 フェースギヤの歯の形状(図I-4・21)。歯形が円弧であるので、歯形のみのかみ合いで等速運動を伝達することは不可能である。等速回転を伝えるためには重なりかみ合い率が1以上のはすば歯車にする必要がある。WN 歯車の歯形としては、対称、非対称形の種々の歯形(ノビコフ、シンマーク、サークアーク歯車など)が提案されている。インボリュート系歯車に比べて接触歯面の相対曲率半径が大きくなることから歯面強さは大きいが、歯の曲げ強さに対する利点は少ない。中心距離の微小な変動によっても曲げ・歯面強さや振動・騒音が大きく影響を受けるという欠点をもっている。高速運転には不向きで、また組付け後のならし運転が重要である。研削仕上げ加工が難しく、また表面硬化歯車に使われることは少ない。d. フェースギヤフェースギヤ(face gear)は円筒歯車とかみ合うかさ歯車またはハイポイドギヤである。円筒歯車をピニオンとすることにより、フェースギヤには、通常のかさ歯車やハイポイドギヤと違って、ピニオンの軸方向の移動がかみ合い性能に影響を及ぼさないという利点がある。インボリュート歯車をピニオンとすると、実績のある歯面形状修整技術があるため、歯当たりの調整が容易になる利点もある。ヘリコプタ用のトルクスプリット形トランスミッションにフェースギヤを用いたアレンジが提案されて以来、フェースギヤに関する解析手法の開発や強度試験が広く行われるようになった。しかしながら、これまで、魚釣り用のリールなどに用いられていた程度で、その設計、製作、検査などに関する技術は確立されていない。図I-4・22はインボリュート歯車をピニオンとするフェースギヤの一歯の形状の一例を示したものであるが、内端側に近づくに従ってフィレットが大きくなり、逆に外端に近づくに従って圧力角が極めて大きくなり歯頂部幅が小さくなる。このように、歯が存在できる歯幅に限界がある。歯が存在できる最大歯幅は、歯数比が大きくなると大きくなる。一方、かみ合い率は、接触線が、図I-4・22に示したように傾いているため、歯幅が小さくなると下がる。これらを考え合わせると、歯数比が大きい場合、歯幅を大きくとることができるのは有利であるが、重量や製作精度などの制約から歯幅が限定されるとかみ合い率が下がり十分な強度を得るのが難しくなる。逆に、歯数比が小さいとかみ合い率は大きくなるが、歯が存在できる歯幅が小さくなり、やはり十分な強度が得られなくなる場合も多い。したがって、フェースギヤに適した歯数比範囲が存在する。
