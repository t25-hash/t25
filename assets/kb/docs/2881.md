# 4・5・5 転造・回転加工

> **章**: 第4章 塑性加工
> **節**: 4・5 塊状品の塑性加工

a.転造（form rolling) 図4･83に示すように，ダイスとの
摩擦力を利用して素材を回転させつつ，素材の半径方向に局部的な
圧縮加工を連続的に加えて，ダイスの凹凸に対応した形状を創成す
る．平滑な加工面が得られるほか，熱間加工でのダイス熱負荷が小
さく生産性も高い．ローラ軸を傾斜させれば，素材を軸方向に送り
ながら転造ができる．また，1ローラダイス押込み方式もある．
多量生産のおねじ類はおもに図4･83(ａ)(ｂ)あるいは(ｃ)の方
式により，スプライン軸や小径のウォームが図4･83(ａ)(ｃ)の方
式で，また，長尺のフィンチューブやボールねじ軸が図4･83(ｃ)
(ｄ)のローラ軸傾斜方式でそれぞれ冷間転造されている．さらに，
図4･83(ａ)(ｃ)の方式あるいは1ローラダイス方式による歯車の
転造技術（冷間，熱間）も開発されている．
図4･84に示すクロスロール加工では，傾斜面付きの一対のくさ
び形ダイスにより，材料を軸方向に流動させて段付き軸部品を熱間
成形する．この加工は，自動車変速機用の段付き軸素形材などの成
形に用いられており，図4･83(ｃ)の2ローラ方式でも可能である．

のダイスにより半径方向に順次圧縮加工する．棒材やパイプの鍛伸
加工や段付き軸部品の成形，中空軸状部品の内面プロフィル成形
（心金を使用）などの高精度冷間加工が可能である．
回転鍛造あるいは揺動鍛造では，図4･85(ｂ)のように，傾斜軸
をもつ円すい形ダイスに回転もしくは揺動運動を与えて，素材軸方
向の局部的圧縮加工を連続的に行う．全面圧縮の鍛造法に比べて加
工力が5分の1以下で済み，据込み加工や下ダイス側でのプロフィ
ル成形による薄肉部品の鍛造に適している．
c .リングローリング（ring rolling) 図4･86に示すように，
素材をマンドレルロールと主ロール（駆動）の間で圧延し，所望の
断面形状・直径のリング製品を成形する．単純な矩形断面のほか異
形断面（凹凸，段差，テーパ等）の小直径から大直径のリング製品
が，熱間もしくは冷間で成形できる．鉄道車両用タイヤ，リング歯
車，ベアリングレース等の素材のほか，ガスタービンのリング部品
などの素材成形に広く適用されている．
