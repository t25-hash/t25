(ａ) 湿式法
図4･98等方圧成形（CIP) 図4･99 金属粉末射出成形の工程概略図
のが基本工程である．特に金属粉末を対象とした射出成形（MIM:
metal injection molding）においては，従来の粉末冶金（P/M:
powder metallurgy）での平均粒径100mのものに比べ，約10
m前後の1けた細かい粉末を用いることから，焼結の促進に伴う
高密度材（真密度に近い）が得られ，機械的性質のみならず各種機
能性も著しく向上するのが特徴である．また，旧来のP/M（おも
に金型プレス成形）では成形が難しい硬質金属材料，あるいはこれ
までの成形技術では困難であった低熱膨張合金や軟質磁性材料など
の難加工性機能材料にも適用できるため，用途に応じた材料の選択
自由度が大きいことも特徴である．したがって，対象材質としても
Feをはじめ，Fe-Ni ，Si ，Co合金，ステンレス鋼，高速度鋼，Ti
合金，Ni基やCo基の超合金，W系重合金，超硬合金，サーメッ
ト，繊維強化型合金など広範囲な種類のものが挙げられている．
MIMの用途は特に限定されておらず，最近では0.003gの超小型
部品から17kgという大物品もあるが，一般には100g以下の複雑
形状の小物品がおもな対象で，時計などの精密機器部品，医薬機器
部品，光通信・電子・磁気部品，OA関連ならびに自動車や一般産
業機械の部品など，多岐にわたる分野で使用されている．
a.原料粉末（powder ) まず原料粉末の特性に関して，粉末
の大きさは2～8mで，球形よりやや異形（アスペクト比で1.2～
1.5）のものが好ましく，かつ粒度分布は狭いか広いものがMIM
には適しているようである．安価で品質が安定した微細粉末供給の
ための粉末製造技術（例えば超高圧水アトマイズ法など）の革新が
望まれている．
b.バインダ（binder ) 成形用材料であるバインダは，脱バ
インダや焼結工程で消失することから，あくまでも補助的な役割を
果たすにすぎないが，MIMプロセスにとっては必須の成分であ
り，その組成によって成形や脱バインダがうまく行えるか否かが決
まる．すなわち，バインダには成形性，離型性，保形性，脱バイン
ダ性などの各種特性が要求されるが，すべての要求を満たすような
バインダは今のところ開発されておらず，後述する各種脱バインダ
法に合った多種多様なバインダ系が存在する．簡便なバインダ系と
しては，ワックス系のものが最も多く使用されている．それは基本
的には，流動性をもたせるため（低粘性）のワックスと，粉末に対
する固着力を増強させるため（保形性）の高分子材料（ポリエチレ
ンやポリプロピレンなど)，そして粉末とのぬれ性や離型性向上の
ための潤滑剤（ステアリン酸など）からなる．
c .射出成形 成形に関しては，基本的にはプラスチックの射
出成形技術と同じであるが，被成形物が高粘度でかつ摩耗作用をも
つことから，やはりMIM専用の成形機が望ましく，機能性に富ん
だ機種も登場しつつある．成形体の品質を左右する成形条件（射出
温度，速度，圧力や金型温度など）については，被成形物によって
も異なることから，多くは経験に基づいた試行錯誤的な設定で行わ
れている．CAE（computer aided engineering）導入による金型
内充てん過程での流動解析によれば，成形品のデザイン，金型構
造，成形条件などの最適化，あるいは成形欠陥の発生防止や精度お
よび品質の向上にとっても有効な情報が得られるようで，CAEの
活用が今後の健全な成形品の量産にとっては必須となるであろう．
d.脱バインダ（debinding) バインダ系と関連する脱バイン
ダ方法には，表4･16 に示すようにこれまで多種多様のプロセス
が開発されているが，工業的には熱分解や溶媒抽出がおもに採用さ
れている．前者では脱バインダに長時間を要したり，変形が生じや
すいなどの問題が残されている．また後者では，それらの欠点がか
なり克服されるものの，溶媒にはアセトンやエチレン，四塩化炭素
などのような人体に害を及ぼすものや環境汚染につながるものが多
いため，その取扱いが問題とされており，現在，エタノールや水溶
性の新しいバインダも開発されつつある．いずれにしても，脱バイ
ンダに長時間を要することは生産的には不利であり，このことが
MIM製品の許容肉厚を大きく制限していたが，最近では脱バイン
ダ技術も進歩して25mm程度の肉厚までは可能となっている．
e .焼結（sintering) 焼結は従来のP/M法と同様で，不活性
あるいは還元性のガス雰囲気，または真空雰囲気で行われる．た
だ，脱バインダ後の成形品はかなり脆弱でハンドリング性に劣るこ
とから，従来のバッチ式から脱バインダと焼結を連続的に行う方法
も考えられている．なお焼結部品の寸法精度は，焼結による寸法収
縮（15～20 %）が著しいにもかかわらず，現在のところ±0.3
～±0.1%まで向上している．これは精密鋳造品やダイカスト製品
の精度に比べると優れているが，全般的にはP/M製品の精度
（±0.1 %）レベルまでには達していない．二，三の例として，
±0.05%という数字も挙げられているが，このレベルまで精度を向
上させることができれば，精密切削加工品と拮抗し，MIMの飛躍
的な進展が期待される．
