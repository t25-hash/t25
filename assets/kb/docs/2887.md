# 4・2・6 波動歯車装置

> **章**: 第4章 伝動要素
> **節**: 4・47 )などが挙げられる。かさ歯車は組立誤差のかみ合い精度に及

基本的な波動歯車装置は図I-4・53に示すウェーブジェネレータ(波動発生器)、フレクスプライン(柔歯車)、サーキュラスプライン(剛歯車)の3個の要素から構造されているもので、ハーモニックドライブ(Harmonic drive)と呼ばれるが、これは登録商標である。波動発生器はだ円状のカムとその外周にはめられた玉軸受からなる。柔歯車は外歯車をもった薄肉円管でだ円状に自在にたわむことができる。剛歯車は厚肉の内歯車環で柔歯車より複数枚歯数が多い。この機構は3要素の一つを拘束することによってほかの2要素がそれぞれ入出力軸となり、増減速装置が構成される。また拘束をはずせば差動装置として利用できる。例えば剛歯車を固定すれば、波動発生器と柔歯車の間の動力伝達が可能となる。このとき波動発生器を入力軸とすれば、柔歯車が出力軸となる減速装置ができる。歯車部のかみ合い機構の特性からノーバックラッシが可能であり、かつ少ない構成部品で一段で大きな速比の増・減速装置が得られる。図I-4・54のように組合わせ、波動発生器を回転させると、柔歯車はたわみ運動を行いながら歯のかみ合い位置が移動していく。剛歯車を固定して波動発生器を1回転させると、柔歯車は剛歯車との歯数差分だけ反対方向に回転する。柔歯車はK-H-V 型遊星歯車機構(4・2・2項参照)の遊星歯車に相当し、その自転運動を取出す方法によって図I-4・55に示すような種類の波動歯車装置の構造が可能となる。速度伝達比は、剛歯車、柔歯車の歯数をそれぞれ욂,욞、剛歯車を固定した場合、=욞욂-욞となる。
