# 4・3・1 平ベルト伝動

> **章**: 第4章 伝動要素
> **節**: 4・3 ベルト伝動装置

平ベルト(flat belt)は,伝動ベルトの中で最も歴史が古く,動力伝動や回転伝達,搬送用などに幅広く用いられているが,構造や機能が多岐にわたっている.かつてはJIS に規格化されていたが,現在は廃止されているので,ベルトメーカのカタログに基づき使用目的にかなったものを選択する必要がある.表I-4・15に代表的な平ベルトの種類と構造ならびにおもな用途を示す.このほか,高分子材料の薄いフィルムでつくられたフィルムベルト,厚さ数十mの鋼板でつくられたスチールベルトなどがある.一般に平ベルトは,ベルト厚さが薄いので曲げ剛性が小さく,小径プーリを使用することができ,速度比は1:15程度まで可能である.また単位長さ当たりの質量が小さいので,周速60m/s程度の高速運転が可能である.平ベルトは走行中プーリから逸脱するのを防止するため,プーリ外周の中央を中高にするクラウン(crown)と呼ばれる加工を施すことが多い.クラウンについては,JIS B 1852:1980に規定されているが,使用条件やベルトの特性によって変わるので,ベルトメーカのカタログなどで推奨する値を参考にしたほうがよい.a.
