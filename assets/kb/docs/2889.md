# 4・3・1 平ベルト伝動

> **章**: 第4章 伝動要素
> **節**: 4・3 ベルト伝動装置

平ベルト(flat belt)は、伝動ベルトの中で最も歴史が古く、動力伝動や回転伝達、搬送用などに幅広く用いられているが、構造や機能が多岐にわたっている。かつてはJIS に規格化されていたが、現在は廃止されているので、ベルトメーカのカタログに基づき使用目的にかなったものを選択する必要がある。表I-4・15に代表的な平ベルトの種類と構造ならびにおもな用途を示す。このほか、高分子材料の薄いフィルムでつくられたフィルムベルト、厚さ数十mの鋼板でつくられたスチールベルトなどがある。一般に平ベルトは、ベルト厚さが薄いので曲げ剛性が小さく、小径プーリを使用することができ、速度比は1:15程度まで可能である。また単位長さ当たりの質量が小さいので、周速60m/s程度の高速運転が可能である。平ベルトは走行中プーリから逸脱するのを防止するため、プーリ外周の中央を中高にするクラウン(crown)と呼ばれる加工を施すことが多い。クラウンについては、JIS B 1852:1980に規定されているが、使用条件やベルトの特性によって変わるので、ベルトメーカのカタログなどで推奨する値を参考にしたほうがよい。伝動中の滑りには、ベルトの張り側と緩み側との張力差により生じるプーリ上のベルトのクリープに基づく弾性滑り(elastic slip)と、負荷が増大したときに生じる移動滑り(sliding slip)とがあり、弾性滑り表-4・15 平ベルトの種類No種類構造主な用途フイルムコア平ベルト繊維機械のスピンドル駆動、抄紙機の各パート駆動コード平ベルト工作機械の主軸駆動、自動改札機の切符搬送積層式平ベルト一般動力伝動、ATMなどの紙幣搬送単体式平ベルトフロッピディスクのディスク駆動、オーディオなどの軽負荷伝動は、張り側と緩み側のベルトのひずみをε,εとすれば、近似的に次式となる。≒ε-ε(I-4・49)負荷が増加し、弾性滑りが1%前後に達すると、ベルトとプーリとの間の摩擦力によって伝達し得るトルクは限界に近づいて、移動滑りが発生し始め、ついには伝動不能となる。伝動時に移動滑りがあるとベルトの温度は急上昇し、寿命は著しく低下する。b. ベルトの長さ図I-4・56に、平行な2軸での代表的なベルトの掛け方を示す。図I-4・56(a)はプーリを同一方向に回転させるオープン掛け(open drive)で、図I-4・56(b)は従動プーリを原動プーリと逆回転させるクロス掛け(cross drive)である。ベルトの長さは、軸間距離をとすると式(I-4・50)により求めることができる。しかし、設定された伝動条件の最大負荷より小さい負荷のときは、ベルトがプーリに巻付いてもただちに張力は変化せず、またはのままある角度回転したのち張力は変化す(a) オープン掛け(b) クロス掛け図-4・56 ベルトの掛け方図-4・57 動力伝達時の張力分布る。この張力の変化は巻付き終わり部から発生する。摩擦係数μは、ゴムと鋳鉄の場合、接触面の状態や伝動条件により変化するが、0.3~0.5程度で、0.5以上となる場合もある。負荷が増加すると有効巻付き角ψも増大し、巻付き角ψと等しくなったときが伝達しうる動力の限界である。このときの張力と巻付き角との関係は、式(I-4・52)(I-4・53)中のψの代わりにψを代入すれば求められる。プーリに巻付けられたベルトが滑ることなく所定の動力を伝達するのに必要な静止時における最小限の張力を初張力(initialtension)といい、その大きさは伝達する負荷と、遠心張力ならびにベルトの剛性を考慮して式(I-4・56)で与えられる。アイドラは曲げによる屈曲疲労を増加させるので、なるべく使用しないのが望ましいが、使用するときはベルトの緩み側の大プーリに近いところを内側から用いるのがよい。実際にベルトを張るときは、ベルトの運転中の伸びを考慮して、上式で求められる値より大きくすることが多く、これを取付張力(installation tension)と呼ぶ。
