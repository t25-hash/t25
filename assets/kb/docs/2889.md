# 4・1・2 石油コンビナート等災害防止法 石油コンビナート等

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**:  

災害防止法はコンビナート地区にあって石油等（引火性の危険物，
高圧ガス等）の貯蔵，取扱が一定以上となる特定事業所について，
新設等の届出，防災組織，各施設地区（製造，貯蔵，用役，入出荷，
事務管理，その他施設地区）の配置・面積，特定通路等の配置・幅
員，特定施設等（流出油等防止堤，屋外給水施設，非常通報設備）
の設置，必要とされる防災資機材（化学消防車，オイルフェンス等）
が規定されている．
特定事業所は次の定義により第1種又は第2種事業所と定義され

a．新設等あ届出 石油及び高圧ガスを共に扱う第1種事業所
（レイアウト規制事業所）は石油の貯蔵・取扱量又は高圧ガスの処
理量を増加する工事，新たに石油の取扱い又は高圧ガスの処理をす
る工事を行う場合は工事開始の45日前に以下の事項を含む新設に関
する計画を主務大臣（総務，経済産業両大臣）に届出る必要がある．
又，（1）事業所敷地内を用途に応じ製造施設地区，貯蔵施設地区，
溶液施設地区，事務管理施設地区，その他施設地区に区分し，各施
設地区の面積と配置．（2）特別防災区域内の事業所間連絡導管及び
連絡通路の配置．（3）敷地面積の一部変更を行う場合も同様に45日
前に変更に関する計画を主務大臣に届出る必要がある．いずれの場
合も不指示決定通知を受けてからでないと工事着工は出来ない．又，
工事完了日から10日以内に新設等完了届出を主務大臣に提出し，確
認検査を受けた後でないと設備を稼働させることはできない（試運
転も含む）．

等防止堤，屋外給水施設，非常通報設備）の設置，並びに定期点検

村長等への届出が必要とされ，定期点検は外観点検3ケ月ごと，機
能点検3ケ月ごと，総合点検1年ごとが求められる．
