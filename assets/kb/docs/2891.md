# 4・3・3 歯付ベルト伝動

> **章**: 第4章 伝動要素
> **節**: 4・3 ベルト伝動装置

ベルトの歯部および背面部はクロロプレンゴムでつくられ、歯面はゴムを塗布したナイロン帆布で覆われているものが一般的であるが、ポリウレタンで作られたものもある。心線は、グラスファイバが一般的であるが、アラミド繊維コード、スチールワイヤ、ポリエステルコードなども使われる。歯付ベルトの歯形形状は台形歯形からスタートしたが、1低騒音、2高寿命、3高伝達トルク、4高伝達精度などの理由により、近年は円弧歯形歯付ベルトが主流となってきた。円弧歯形歯付ベルトにはHTD,STPD(STS)、PXなど、メーカによって異なる数種類の歯形があり、それらの間でプーリの互換性はないが、いずれも台形歯形ベルトより歯ピッチに対する歯厚と歯丈が大きく、プーリとのかみ合いもスムーズで伝動容量が大きい。引張試験伸び率%4.0以下4.0以下4.0以下4.0以下4.0以下伸び率測定時荷重(kN/25.4mm)1.572.165.397.458.63の工夫をし、騒音低減が計られている。円弧歯形歯付ベルトのピッチは一般にミリシステムであり、ピッチ8mmと14mmの円弧歯形歯付ベルトについてはプーリとともにJIS B 1857-1:2001「一般用円弧歯形歯付ベルト伝動」ならびにJIS B 1857-2:2001として規格化されている。従来から一般機械類に多く使用される比較的大きなピッチの台形歯形歯付ベルト、ならびに主としてOA 機器などに使用される小ピッチの歯付ベルトはそれぞれJIS K 6372:1995「一般用歯付ベルト」、JIS K 6373:1995「軽負荷用歯付ベルト」に規定されている。ピッチはインチ系列で、表I-4・22に一般用歯付ベルトの形状と種類を示す。自動車に用いる歯付ベルトは、使用環境が厳しいので、高品質の自動車用歯付ベルト(自動車技術会規格JASO E105-99,E106-99)が使用されている。このほか、多軸伝動用としてベルトの背面にも歯の付いた両面歯付ベルト、騒音低減を目的としたはすば歯付ベルト、規格化はされていないが、古くからT 歯形、AT 歯形などのベルトがあり、これらに使用するプーリの歯形もベルトの歯形に対応した形状となっている。歯付プーリは、ベルトがプーリから外れないようにフランジがプーリの両側についているもの、片側だけのもの、フランジのないものなどがある。台形歯形歯付プーリについてはJIS B 1856:1993「歯付プーリ」に規格化されている。歯付ベルト伝動における回転比は、原動、従動プーリの歯数比であり、ベルト長さは規格で決められているから、使用に当たっては式(I-4・50)または(I-4・51)で概略の長さを求め、それに最も近いベルトを選定し、その長さに基づいて軸間距離を決めるか、アイドラを用いる。すなわち、設計動力を式(I-4・62)より求め、これと小プーリの回転数とから図I-4・60によりベルトの種類を決定する。アイドラを使用する場合および増速の場合には、負荷補正係数にさらに係数を加える必要がある。次に幅25.4mm、かみあい歯数6以上における基準伝動容量種類(N)を次式により求める。求めた基準伝動容量(kW)と設計動力(kW)とから、式(I-4・68)によりベルト幅を決定する。歯付ベルトの初張力は摩擦伝動ベルトより小さくてよいが、伝動時に必要な初張力の最大値と推奨値とを表I-4・26に示す。
