# 4・3・4 そのほかのベルトによる伝動

> **章**: 第4章 伝動要素
> **節**: 4・3 ベルト伝動装置

a.Vリブドベルト伝動 Vリブドベルト（V-ribbed belt）
は，平ベルトの柔軟性と，Vベルトの高伝動能力の利点を併せも
つベルトとして開発された．ベルト厚さが薄いので，使用できる最
小プーリ径はVベルトより小さく，平ベルトに近い．また，ベル
トの背面でも平ベルトのように動力伝動を行うことができるので，
多軸伝動が可能である．単位幅当たりの質量もVベルトより小さ
いので，高速度での使用が可能である．このような利点から，近
年，自動車エンジンの補機駆動用としてよく使用されるようになっ
てきた．VリブドベルトのJISは定められていないが，ISO
9981：1990，ISO9982：1991や自動車規格JASO E109：1994，
E111：1993に規格化されている．Vリブドベルトには，H，J，
K，L，M形の5種類があり，リブゴムがはまり合うV溝をもっ
た，Vリブドプーリ（V-ribbed pulley）とともに使用される．
ISO規格やJASO規格には，プーリ溝部の詳細な寸法と公差が定
められている．
b.丸ベルト 丸ベルト（round belt）は直径が2～12mmの
円形断面のベルトで，搬送用の回転ローラ駆動や軽負荷伝動に使用
される．また，断面形状に方向性がないので，伝動面が三次元でも
伝動が可能である．
4・4チェーン伝動装置
チェーン伝動（chain drive，chain transmission）では，チェー
ンとスプロケット歯のかみあいによって運動または力を伝達する．
特長は，滑りがないため同期運転ができ，初期張力を小さくできる
ので軸受に余分な力を加えず，比較的長い軸間距離（好ましいスプ
ロケットの軸間距離はチェーンピッチの30～50倍）に使用でき，
一つの駆動軸から多軸伝動ができ，コンパクトな断面積で大張力を
伝達でき，伝動効率は95 %以上と高いことである．また，ローラ
チェーンでは表裏両面を利用した伝動ができ，世界的な標準化が行
われており，切継ぎによる長さ調整が容易であるなどの利点があ
る．欠点は，スプロケットが多角形であるため運動伝達に速度変動
があること，潤滑が必要，摩耗して伸びること，スプロケット歯と
のかみあいで騒音を発生すること，横荷重に弱く据付時に心出しが
必要，水・ちりなどの腐食性・摩耗性雰囲気を避けるべきことなど
が挙げられる．これらの欠点を補うチェーンには，自己潤滑性のあ
るプラスチックチェーン，焼結含油ブシュを用いたオイルレスチェ
ーン，ピン～ブシュのしゅう動部をゴムシールでカバーしたシール
チェーン，耐環境チェーンなどのシリーズがある．
一般的に3m/s以下の低速大荷重の伝動に使われるが，使用方
法を工夫すれば乗用車エンジンのカム軸駆動で30m/s （ローラチ
ェーン，サイレントチェーン)，競技用モータサイクルの後輪駆動
で25m/s （ローラチェーン）といった高速伝動にも使用できる．
伝動用に用いられるチェーンは，ローラチェーンとサイレントチェ
ーンの2種類であるが，量的にはローラチェーンが多い．ローラチ
ェーンはISO606：2004で日本，アメリカでおもに用いられるA
系（ピッチ6.35～76.2mm)，ヨーロッパでおもに用いられるB
系（ピッチ8～114.3mm）の両仕様が制定されている．JIS B
1801：1997にもA系，B系が制定されている．一部を除きA系，
B系は寸法，伝動能力に差異がある．規格に制定されているよりも
小さい，または大きいチェーンもある．A系の小サイズチェーン
（ピッチ6.35と9.525mm）は，ローラのないブシュチェーンであ
る．外リンク＋内リンクで一組となるため，通常偶数リンク数で使
用されるが，どうしても長さを奇数リンク数としたいときはオフセ
ットリンクを用いる．1ピッチ型オフセットリンクや一部の継手リ
ンクは本体部分に比べ伝動能力が劣る．サイレントチェーンは
ANSI B29.2Mで規定されているがJISにはない．サイレントチ
ェーンは通常片側にのみ歯を有するため，チェーンの裏側を利用し
た伝動はできない．また，オフセットリンクはない．
