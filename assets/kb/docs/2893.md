# 4・4・1 ローラチェーン伝動

> **章**: 第4章 伝動要素
> **節**: 4・4 チェーン伝動装置

a. ローラチェーンローラチェーン(roller chain)は、2枚の外プレート(outer plate)に2本のピン(pin)を圧入した外リンク(outer link)と、2個のブシュ(bush)を2枚の内プレート(inner plate)に圧入し、ブシュの外側にローラ(roller)を自由に回転できるようにはめた内リンク(inner link)を交互に組立て4―102機械要素・トライボロジーる。ピンは両端をかしめるか、割りピン、クリップなどを用いてとめる。大荷重を伝動するときや小ピッチチェーンを選定したいときは、2列以上の多列チェーン(multiple strand chain)が用いられる(図I-4・61)。A 系ローラチェーンの主要寸法と引張強さ、疲れ限度を表I-4・27に示す。b. スプロケットスプロケット(sprocket)は鋼、鋳鉄などを用いて製作され、小歯数のものは歯部を表面硬化することが多い。JIS B 1801:1997ではS 歯形、U 歯形とISO歯形の3種を規定しているが、S 歯形が一般的に用いられる。スプロケットの種類を図I-4・6に、基本寸法計算式を表I-4・28に示す。歯数は17~114の間で選定する。チェーンの巻付け角度は120°以上を確保する。一般用途では高速度での使用や衝撃荷重の加わる場合には、駆動スプロケットの歯数は25以上とし、歯面は硬化する。c. ローラチェーンの選定ローラチェーンの選定はJIS B1810:1999(ISO 10823)「伝動用ローラチェーンの選定指針」で示されている。平行かつ水平な二つの軸に正しく取付けられたスプロケットによる1列チェーン伝動(オフセットリンクを含まない)で、駆動スプロケット歯数25の場合、選定は下記の手順で行う。・伝達動力を求める伝達動力は伝達トルク(N・m)と駆動スプロケット回転速度(r/min)より次式で求める。なお、伝動能力図はチェーン製造者によって異なるので、選定に当たっては製造者に確認することが望ましい。4―104機械要素・トライボロジー
