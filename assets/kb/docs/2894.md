# 4・4・2 サイレントチェーン伝動

> **章**: 第4章 伝動要素
> **節**: 4・4 チェーン伝動装置

サイレントチェーン(inverted tooth chain,silent chain)は,歯を設けたプレートどうしをピンまたは転がりを取入れた連結要素などで組立てたチェーンである.図I-4・64にサイレントチェーンとスプロケットの歯形を示す.横ずれを防止するガイドをチェーン両側に設けてスプロケットをまたぐ形式のサイドガイド形と,中央部に設けるセンターガイド形(スプロケット中央部に溝を切る)がある.プレートの形状,ピンの形式,ピン端の止め方,継手リンクの形式等は製造者に任されており,ANSI B 29.2M:1982では,これらとかみあう共通歯形,チェーンの幅寸法,ガイド形式,伝動能力を規定している.チェーンピッチは4.76,9.52,12.70~50.8mmの9種類あるが,中間的なピッチのチェーンも用いられる.表I-4・30にチェーンの寸法,破断強さ,質量を示す.図-4・64 サイレントチェーンとスプロケケットの歯形(直線歯)表-4・30 サイレントチェーンの寸法,破断強さ,質量破断強さ概略質量チェーン高さチェーン幅mmピッチmmkg/mkNmm0.971.281.932.573.8525.033.350.066.6100.019.0525.438.150.876.29.52510.871.712.573.425.136.8444.566.688.9133.3177.825.438.150.876.2101.612.7014.483.875.127.7410.312.8100.1133.4200.1266.9333.638.150.876.2101.6127.019.0520.886.8510.313.717.120.5177.8266.9355.8444.8533.750.876.2101.6127.0152.425.4027.8415.520.525.730.8400.3533.7667.2800.776.2101.6127.0152.438.1041.7820.527.434.241.1533.7711.6889.61067.576.2101.6127.0152.450.8055.70図-4・65 サイレントチェーンとスプロケットのかみあい部品すべてが張力を分担するためむだがなく,同一幅であればローラチェーンに比べ伝動能力は高い.サイレントと称する理由は図I-4・65のようにプレートがスプロケットに対し斜めにかみあい,相対衝突速度が低下して騒音レベルが低減するためである.特殊な用途では,内股部をスプロケット歯に干渉させつつ外股部を着座させたり,内股のみでスプロケット歯とかみあわせるなど騒音・振動低減の工夫をこらしているものもある.サイレントチェーン用のスプロケットについては,ANSI規格で直線歯を規定しているが,インボリュート歯形も多く実用化されている.
