# 4・5・1 エラストマベルトテンションドライブ(elastomer

> **章**: 第4章 伝動要素
> **節**: 4・5 機械式無段変速機

belt tension drive)機械式無段変速機(mechanical continuously variable powertransmission)の中でも親しみやすい乾式の可変速装置の一つで,標準的なゴム製のV ベルトとプーリを組合わせたものである.変速比[speed ratio;ここでは入力軸回転速度읉윰に対する出力軸回転速度윱윲읏の比욦=윱윲읏/읉윰で表すことにする]は図I-4・66において,ばねの押付け力とベルト張力による軸向力とが平衡するプーリ位置で決まる.力の伝達はベルトとプーリ間の摩擦力(オイラーの式に従う)とベルト張力により伝えられる.
