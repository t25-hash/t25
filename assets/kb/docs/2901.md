# 4・2・7 土壌汚染・地盤沈下

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 10・15 M HC 平成13年 0．17（0．08）

a．土壌汚染 土壌は，大気，水とともに環境の主要な構成要
素であり，人をはじめ植物，小動物，微生物等の生育の場として，
また雨水を貯留し地下水を酒養する場としての重要な役割を担って
いる．しかし，土壌は大気や水と比べてその組成が複雑で有害物質
に対する反応も多様であり，また，いったん汚染されるとその影響
が長期にわたり持続する蓄積性の汚染となる等の特徴も有している．
土壌汚染の歴史は，足尾鉱山から流れ出た鉱毒水により広い範囲
の農用地が汚染されたことに遡ることができる．この土壌汚染は農
作物の生育阻害を生じさせるものであったが，人の健康への影響の
おそれがある土壌汚染問題の顕在化は，神通川流域のイタイイタイ
病の原因となったカドミウムによる環境汚染が周辺の農用地の土壌
にも及んでいることが判明してからである。これを契機にカドミウ
ムによる農用地の汚染ボ全国各地で顕在化したため，土壌汚染防止
対策が緊急な課題としてとりあげられ，昭和45年末に開催された第
64臨時国会（いわゆる公害国会）における公害対策基本法の一部改
正により，典型公害の一種として新たに土壌汚染が追加された．ま
た，土壌の汚染に係る環境基準を設定することとされるとともに，
その実施法として，農用地の土壌の汚染防止等に関する法律が制定
された．この法律に基づき農用地の土壌汚染の防止および除去等の
対策が推進されており，対策の必要がある地域の8割以上ですでに
・対策が完了している．ただし，土壌中の有害物質の量と農作物の生
育および農作物中の有害物質の量との因果関係がきわめて複雑であ
る等から土壌の汚染に係る環境基準は，平成3年まで制定されなか
った．
一方，昭和50年代になって，東京都における六町クロム野津問題
や国立研究所の移転に伴う土壌汚染の発覚など，工業活動や廃棄物
の埋め立て等が原因となる土壌汚染問題による社会問題が発生した．
また，土壌汚染が原因となった地下水汚染が全国的に拡がっている
ことが判明したことなどから，これら農用地以外の土壌汚染，いわ
ゆる市街地における土壌汚染問題への取り組みが必要であることが
認識されるようになった．そこで，平成3年8月に公害対策基本法
に基づく 「土壌の汚染に係る環境基準」が制定され，また，平成11
年1月には土壌汚染調査対策の進め方を示した「土壌・地下水汚染
に係る調査・対策指針および運用基準」が策定された．そして，平
成14年5月，我が国における土壌汚染対策の基本的なルールを示し
た，「土壌汚染対策法」が制定された．
