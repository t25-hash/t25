# 4・6・1 遊星ローラ変速機(planetary roller transmission)

> **章**: 第4章 伝動要素
> **節**: 4・6 トラクションドライブ式変速機

遊星ローラ機構を用いて滑らかで静粛な増速や減速のできる変速機で,リング固定,キャリヤ入力욂,サン出力욦とすると増速機となる.図I-4・84でリング内径を욥,サン外径を욦とすると,変速比(増速比)욦は욦=욦/욂=1+욥/욦となる.増速比は8程度までとれ,出力回転数50000rpm,トルク0.5N・m,出4―107第I部機械要素図-4・80 ダブルキャビティCVT 욦=/=0.44~2.27,욞=/=0.741,윯율융=340N・m,윯율융=7000rpm,=40mm,2욧=130mm,θ=62.5°図-4・84 遊星ローラ増速機(キャリヤ入力,サン出力)図-4・85 遊星ローラ減速機(planetary roller reducer,弾性リングローディング型)力2.6kW のマシニングセンタ用増速スピンドルなどが実用されている.一方,図I-4・85は,遊星ローラと接触するリングのローディングに弾性変形するU 形リングを用い,これに与圧を与え遊星ローラを入力軸に押付け,サン入力,キャリヤ出力とした減速機の例である.1段減速で1:12.5,2段減速で1:100,3段減速で1:1000と高い減速比が得られ,サーボモータ駆動,出力回転数フィードバックにより,バックラッシのほとんどない精密駆動ができ,製版機・印刷機の送り機構に実用されている.
