# 4・6・1 遊星ローラ変速機(planetary roller transmission)

> **章**: 第4章 伝動要素
> **節**: 4・6 トラクションドライブ式変速機

一方、図I-4・85は、遊星ローラと接触するリングのローディングに弾性変形するU 形リングを用い、これに与圧を与え遊星ローラを入力軸に押付け、サン入力、キャリヤ出力とした減速機の例である。1段減速で1:12.5,2段減速で1:100,3段減速で1:1000と高い減速比が得られ、サーボモータ駆動、出力回転数フィードバックにより、バックラッシのほとんどない精密駆動ができ、製版機・印刷機の送り機構に実用されている。
