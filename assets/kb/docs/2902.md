# 4・6・2 ウェッジローラ減速機(wedge roller reducer)

> **章**: 第4章 伝動要素
> **節**: 4・6 トラクションドライブ式変速機

トルクに応じたローディング力を得る方法として、サン軸とリング軸の中心をわずかにずらし、これらに挟まれる遊星ローラの中の一部をスイングできるようにして、接触したときにくさび角ができる変速機の例を図I-4・86に示す。図の例では三つのローラのう4―108機械要素・トライボロジー図-4・86 ウェッジローラ減速機(トルク比例ローディング)ち、二つのローラの回転軸は固定されており、一つのみ可動する構造(これをウェッジローラと呼ぶ)になっている。例えば、入力軸となるサン軸が反時計方向に回ると、ウェッジローラはサンとリングの間にかみ込みリングに回転力を伝える。一方、サン軸が時計方向に回るとウェッジローラは挟まれた状態から解放され、リングに動力を伝達しない。このようにウェッジローラ減速機は一方向クラッチの機能も併せ持つことになる。ねじ伝動装置はモータと組合わせて、非常に多くの位置決め装置に使われている。(社)精密工学会の超精密位置決め専門委員会による1998年の国内アンケート調査を見ると、各種位置決め装置の51.4%に送りねじが用いられ、ボールねじはそのうちの86%に達して、年々増加している。
