# 4・8・1 クラッチの種類

> **章**: 第4章 伝動要素
> **節**: 4・8 クラッチ

クラッチ(clutch)は同心軸上にある駆動側から被駆動側へ動力の断続を行う機械要素で、連結の形式によって表I-4・32に見られるように、かみあいクラッチ、摩擦クラッチ、自動クラッチに分類され、その作動方式によって表I-4・33のように機械式、油圧式、空気圧式、電磁式のものがある。表-4・32 連結の形式によるクラッチの分類かみあいクラッチジョークラッチ、ドッグクラッチツースクラッチ(歯形クラッチ)歯車クラッチまたはスプラインクラッチ摩擦クラッチディスククラッチ(円板クラッチ)円すいクラッチドラムクラッチ(リムクラッチ)スリップクラッチ(トルクリミッタ)一方向クラッチオーバランニングクラッチフリーホイールクラッチ遠心クラッチ自動クラッチ表-4・33 作動の方式によるクラッチの分類とその特徴名称特徴カム、レバー、リンク機構などにより力の拡大を図るが、作動力は人の手や足によるので、その大きさ・断続の回数・連結時間が限定される。いわゆる効き具合を感じることができる。最も安価である。機械クラッチ作動圧を大きくとることができるので、単位容積当たりの伝達トルクは最大で、連結時間も短い。油圧ポンプ・配管・制御弁などが必要である。油圧クラッチ本体は構造が簡単で安価である。伝達トルクの大きいものもある。空気による摩擦面の冷却ができ、緩衝連結もできる。圧縮空気(または空気圧縮機)・配管・制御弁などが必要であるが戻り回路は不要。空気圧クラッチ遠隔操作に最も有利である。連結時間は最小で、断続回数も多くとることができる。比較的高価であり、爆発の危険のある雰囲気では使用できない。電磁クラッチ図-4・91 ジョークラッチ(スパイラル形)
