# 4・8・2 かみあいクラッチ(positive clutch)

> **章**: 第4章 伝動要素
> **節**: 4・8 クラッチ

駆動側と被動側とを,機械的かみあいによって連結するクラッチで,滑りがなく伝達が確実なので確動クラッチ(ポジティブクラッチ)ともいわれる.円筒端面に設けた角形,台形,のこ歯形,スパイラルなどの凹凸によって連結するジョークラッチ(図I-4・91),円筒端面に設けた多数の山形の凹凸によって連結するツースクラッチおよび外歯車と内歯車とによって連結する歯車クラッチ(インボリュートスプラインもかみあいクラッチとして用いられる)がある.ジョークラッチは接続的に衝撃を伴うため,停止中またはごく低速回転のときにしか連結できない.また,ツースクラッチは比較的低速ならば回転中でも連結できるが,負荷能力はジョークラッチより小さい.ツースクラッチの山形の一部を特殊形状にして,1回転中に1箇所でしかかみあわないようにした定位置かみあいクラッチは,位置決め・同期などの目的に用いられる.
