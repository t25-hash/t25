# 4・8・3 摩擦クラッチ(friction clutch)

> **章**: 第4章 伝動要素
> **節**: 4・8 クラッチ

clutch)湿式多板クラッチは摩擦面を潤滑状態で作動させるクラッチであり、潤滑油に含有される添加剤の効果により係合が滑らかで、潤滑油の冷却効果により摩耗が少なく、寿命が長い。この特徴により、土木、建設機械、農業機械、船舶、車両に数多く使用されており、図I-4・96に見られるように自動車用自動変速機では変速段ごとに多くの多板クラッチ(図I-4・97)が用いられている。特に高速回転する湿式多板クラッチの加圧ピストンには、回転する油の質量による遠心油圧が働くため、これを補償する必要がある。その一つの方法として、図I-4・98に示すように、加圧ピストン室の反対側に油の保持室を設け、遠心油圧を補償する方法がとられる。クラッチの伝達トルクは、押付け力、摩擦材(フェーシング)の平均摩擦半径욇、摩擦材の外周半径、摩擦材の内周半径、摩擦面数、摩擦係数μとすると=μ욇=μ2-3-で表される。湿式多板クラッチの評価法は、世界的なATFの統一規格(ILSAC 湿式多板クラッチの構造擦係数はクラッチ伝達トルクから算出されるが、油温、面圧などの条件、潤滑油、摩擦材、摩擦相手材などの材質、表面粗さを代表とする表面状態に左右されるため、評価の際には十分な注意が必要である。特に摩擦材と潤滑油(ATF: automatic transmissionfluid)との適合性については、油種との組合せによって摩擦係数が異なるため、静的および、動的クラッチ伝達トルク容量を設計する際の油種選定は、非常に重要である。また油種によっては、使用環境が摩擦材のはく離強度を低下させるため、圧縮疲労寿命に関する評価方法がJASO自動変速機油摩擦材適合性試験方法(JASO M 357:2001)に規格化されている。湿式クラッチに多用されるペーパ摩擦材(paper-based frictionmaterial)は図I-4・99に示す構成成分を配合したペーパに熱硬化性レジンを含浸させ、乾燥、硬化後、製品形状に加工する。ペーパ摩擦材は弾性に富み、無数の孔(気孔)が内部の微小孔とつながり合い、網状組織を形成し潤滑油を保持する細胞のようになっている多孔質弾性体である(図I-4・100)。この特徴的な構造がクラッチ係合時において摩擦材を圧縮変形させ、内部の潤滑油を排出し、高温となった摩擦熱を急速に冷却する。開放時は、摩擦材が復元することによって内部に潤滑油が浸透するサイクルを繰返し、摩耗を少なく寿命を長くする。これをスポンジ効果と呼んでいる。c.
