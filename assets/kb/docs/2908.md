# 4・8・4 自動クラッチ

> **章**: 第4章 伝動要素
> **節**: 4・8 クラッチ

a. スリップクラッチ(slip clutch)定トルククラッチ,安全クラッチ,トルクリミッタ(torque limiter)などともいわれ,あらかじめ定めたトルクで自動的に,かみあいがはずれるかまたは摩擦面でスリップする構造のクラッチである.一般にトルクが規定トルク以下になれば,再びトルクの伝達が行われる.機械的なものでは,ツース式(図I-4・103),ボールまたはローラとポケット式,摩擦板式などがある.シャーピンを用いた破壊式のものもあるが,これは自動的にリセットされず,またトルクの調整はピンの交換によらなければならない.負荷条件の不明な場合には,伝達装置の1箇所にスリップクラッチを設けることにより合理的な設計ができる.b. 一方向クラッチ(one way clutch)一方向にだけ動力を伝達するクラッチで,球,ローラ,カム,まゆ形こまなどのスプラグ(sprag)を駆動側と被駆動側の間に一方向だけに食い込むようにしたものと,ラチェットを用いたもの,カムを用いたもの,軸にコイルばねを巻付けてその巻締め力を利用するものがある.図I-図-4・101 円すいクラッチ図-4・102 ドラムクラッチ(空気圧式内張り型)4―113第I部機械要素ボンスプリングで空転時にも押付け力を与え(図I-4・105),空転から,かみあいに移るときのバックラッシを小さくしている.スプラグ形一方向形クラッチのトルク伝達機構を図I-4・106で説明する.スプラグのカム面は通常偏心した位置욡,욉にある二つの円弧からなる.スプラグが内輪・外輪と接する点をそれぞれ,とすると直線をストラト(strut)と呼び,これと内・外軌道径の中心から引かれる直線,とのなす角α,βをストラト角といい,α>βの関係にある.スプラグが滑りを起こさないためには,ストラト角αを小さくし,tanαは鋼の摩擦係数μより小さくなければならない.tanα<μスプラグがトルクを受けると,ストラト上に力を生じる接触点における垂直分力,接線分力の関係は,=tanαとなり,ワンウェイクラッチの伝達トルクは=욡tanαとなる.ここに,:スプラグ数,욡:内輪軌道径の2分の1である.c. 遠心クラッチ(centrifugal clutch)遠心力を利用して駆動側と被動側を連結するクラッチで,あらかじめ定めた一定速度範囲を超えると遠心力の作用によりクラッチが入り始め,伝達トルクは回転速度の二乗に比例して大きくなり一定値に達する.低速度になればクラッチははずれる(図I-4・107).図I-4・108は,パウダクラッチと呼ばれているもので,ケーシングが回転するとパウダ(金属粒子)が遠心力で外方に飛ばされ,図-4・103 スリップクラッチ(ツース型)図-4・104 スプラグ形一方向クラッチ図-4・106 スプラグクラッチ図-4・105 空転時のばね押付け力(sprag clutch)のと摩擦力トルク伝達機構4―114機械要素・トライボロジー図-4・108 パウダクラッチ構造図-4・107 遠心クラッチ図-4・109電磁パウダクラッチ(コイル静止形)の構造ロータ外周の波面とケーシング内面との間にくさび状に入り込みその摩擦力によってロータを回転させる.遠心クラッチは伝達負荷容量に関係なく無負荷起動ができるから,高慣性の負荷に徐々に連結させるような場合や,始動時の衝撃をきらう場合などに用いられる.また,スリップクラッチと同じく過負荷に対してはスリップして駆動側を保護する.d. 電磁パウダクラッチ(electromagnetic powder clutch)図I-4・109に電磁パウダクラッチの構造の概要を示す.ロータとシリンダとの間のギャップに透磁率が高く残留磁気の小さい,耐摩耗性・耐熱性に優れたパウダ(磁性粉体)を入れ,これに磁束を通して磁化させ,パウダ間の連結力およびシリンダロータとパウダ間の摩擦力によってトルクの伝達を行う.これはパウダを媒介としたドラム摩擦クラッチといえるが,遠心クラッチと同様な特性をもっており,また静摩擦係数と動摩擦係数にほとんど差がないこと,励磁電流を変化させることによって自由にトルクを制御できることなどの利点をもっている.
