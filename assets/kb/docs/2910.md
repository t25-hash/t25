# 4・9・2 摩擦ブレーキ(friction brake)

> **章**: 第4章 伝動要素
> **節**: 4・9 ブレーキ

a. 帯ブレーキ帯ブレーキ(band brake)は産業機械や自動車の自動変速機、駐車ブレーキなどに用いられる。ブレーキドラムの周囲に鋼の帯を巻付け、帯を締めるもので(図I-4・110)、効きのよい反面、摩擦係数μの変化に大きく影響される(図I-4・111)。帯の角度は180°~270°位が多いが、大出力を要するときは帯を2回巻いたりする。設計によっては振動を生じやすいので注意を要する。b. ブロックブレーキブロックブレーキ(block brake)はウインチ、クレーンや鉄道車両などに用いられる。ブロックを2個図-4・110 帯ブレーキの原理図-4・111 帯ブレーキの効き(相対比較)図-4・112 ブロックブレーキの原理図-4・113 鉄道車両に用いたブロックブレーキ(c) デュオサーボ(b) リーディング(a) ツーリーディトレーリングング図-4・114 ドラムブレーキの形式用いたものは複ブロックブレーキと呼ばれる。鉄道車両に用いた例を図I-4・113に示す。c. ドラムブレーキドラムブレーキ(drum brake)は自動車、鉄道、産業機械などに広く用いられる。図I-4・114のような形式があり、入力は液圧シリンダまたはカム、ウェッジなどの機械的方法で行う。図I-4・115に見られるように、効きは自己サーボ性の高いデュオサーボがいちばんよいが、同時に摩擦係数の変化に対して大きく影響される。シューを遠心力で働かせるものは、ウインチなどで巻下げ時に速度が過大になったときの自動的な抑速などに用いられる(図I-4・116)。d. ディスクブレーキディスクブレーキ(disk brake)は平らな円板(ディスクロータ)の片側または両面の一部または全面に図-4・115 ドラムブレーキの効き(相対比較)図-4・117 片側押付けディ図-4・116 遠心力ブレーキスクブレーキ図-4・118 ねじブレーキ図-4・119円すいブレーキ摩擦材を押付けるようにしたものである。交通機関、産業用などに広く使われる傾向にある。長所としては摩擦係数の変化に対する効きの変化が直線的で(図I-4・115)、表面についたごみや水分の除去が速やかに行われ、熱放散もよく、整備も容易なこと。欠点としては入力に対し出力が少ないため、倍力装置などを必要とすることが挙げられる。片側に押付けるタイプは図I-4・117のようなものがあり、押付けにねじを利用したものはねじブレーキ(図I-4・118)、摩擦材および相手材を円すい形にしたものは円すいブレーキと呼ばれる(図I-4・119)。両側全面に働くものは航空機、産業機械などに用いられ、トルクが不足するときはロータの枚数を増加させる(図I-4・120)。ディスクを挟む形式には図I-4・121のようなものがあり、一般には図I-4・121(b)(c)が用いられる。(c)はキャリバと称するシリンダを含む入力部分がしゅう動しなければならないが、構造簡単、安価で冷却性もよいところから広く自動車用などに採用されている。
