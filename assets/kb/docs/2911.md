# 4・9・3 そのほかの制動装置

> **章**: 第4章 伝動要素
> **節**: 4・9 ブレーキ

a. レール用ブレーキ(car retarder)鉄道などにおいて、レールに摩擦材を押付けまたは挟んで制動するものである。ケーブルカーやエレベータでは、ケーブル切断時にレールを自動的に挟んで制動するもの、貨車用には突き放された車の車輪を内外の桁で挟んで減速させるものが用いられる(図I-4・122)。浮上鉄道でもレール状のものを挟む形式が試みられている。電磁力でレールに吸着させる形もある(図I-4・123)。b. 空力ブレーキ空力ブレーキ(aerodynamic brake)は主として航空機、競争用自動車などで、じゃま板を出して空気抵抗を増大し制動するものである。航空機ではスポイラやダイブブレーキ4―115第I部機械要素図-4・120 多板式ディスクブレーキ(航空機用)(c) キャリパ固定(b) キャリパ浮動(a) ディスク浮動(オポーズド(シングルシシリンダ)リンダ)図-4・121 ディスクブレーキの形式図-4・122 レール用ブレーキ図-4・123 電磁式レールブレーキなどとして使われる(図I-4・124)。自動車では減速のほかに接地力を増加し、車輪ブレーキの効きを上げる効果もある。c. リターダリターダ(retarder)は自動車などに用いられるもので、降坂時に制動エネルギーの一部を吸収して基礎ブレーキの負担を軽減する。エンジンの弁時期を変え、これを空気ポンプとして制動作用を行わせるもの、排気管を閉鎖してエンジンブレーキ効果を高める排気ブレーキ、ストールしたトルクコンバータなどにより流体抵抗を生じさせるもの(図I-4・125)、進推軸に取付けた円板を磁場内で回し、生じた渦電流により制動作用を行わせるものなどがある(図I-4・126)。電磁式のものは温度上昇とともにブレーキ効果は減少する。d. 電磁ブレーキ(electromagnetic brake)鉄道において、電磁石によりレール内に誘発される渦電流を利用するものである。e. エネルギー回生ブレーキ(regenerative brake)ブレーキは多く本体の運動エネルギーを熱の形で放散するが、エネルギー節4―116機械要素・トライボロジー約の見地から、このエネルギーの一部を回収し、駆動に利用しようとするものである。電気車(電車、電気機関車、電気自動車など)では電動機の種類によっては制動時にこれを発電機として用い、蓄電池やキャパシタに戻したり、架線を通じて電源やほかの車に供給したりできる。機械式のものでは、駆動系にフライホイール(flywheel)を装備し、制動時にこれにエネルギーを蓄える。重量の関係からはずみ車は高速で回すことが望ましく、また車速と一致しないので無段変速機を必要とする(図I-4・127)。ガス圧の形でエネルギーを蓄えることも試みられている。通常油圧ポンプを用い、リザーバ内のガス圧を流体を介して上昇させ、駆動時には油圧モータを利用する。(a)(b)図-4・124 空気抵抗ブレーキ図-4・125 流体式リターダ図-4・126 電磁式リターダ図-4・127 フライホイール(電気自動車)
