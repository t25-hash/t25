a.レール用ブレーキ（car retarder) 鉄道などにおいて，レ

カーやエレベータでは，ケーブル切断時にレールを自動的に挟んで
制動するもの，貨車用には突き放された車の車輪を内外の桁で挟ん

b.空力ブレーキ 空力ブレーキ（aerodynamic brake）は主
として航空機，競争用自動車などで，じゃま板を出して空気抵抗を
増大し制動するものである．航空機ではスポイラやダイブブレーキ
などとして使われる（図Ⅰ-4･124）．自動車では減速のほかに接地
力を増加し，車輪ブレーキの効きを上げる効果もある．
c .リターダ リターダ（retarder）は自動車などに用いられ
るもので，降坂時に制動エネルギーの一部を吸収して基礎ブレーキ
の負担を軽減する．エンジンの弁時期を変え，これを空気ポンプと
して制動作用を行わせるもの，排気管を閉鎖してエンジンブレーキ
効果を高める排気ブレーキ，ストールしたトルクコンバータなどに
より流体抵抗を生じさせるもの（図Ⅰ-4･125），進推軸に取付けた
円板を磁場内で回し，生じた渦電流により制動作用を行わせるもの
などがある（図Ⅰ-4･126）．電磁式のものは温度上昇とともにブレ
ーキ効果は減少する．
d.電磁ブレーキ（electromagnetic brake) 鉄道において，
電磁石によりレール内に誘発される渦電流を利用するものである．
