c .ウェーブレット変換を用いた信号解析 ウェーブレット変
換を用いた信号解析であるウェーブレット解析には，おもに以下の
四つの応用分野がある．
(１)異常信号の検出（不連続波形の検出）
(２)相似相関の検出（ウェーブレットと相似な信号の検出）
(３)データ操作（係数の操作による雑音除去など）
(４)システム同定（制御系のパラメータ同定）
(１)では，振動計測結果に混入しているパルス加振力などを検出
できる．(２)については，a項で示した図4･24の周波数変調の抽
出が代表的なものである．(４)については，振動計測結果から構造
物の剛性や減衰を同定する例 が報告されている．
ここでは，(３)のウェーブレット解析による雑音除去の例を示そ
う．矩形パルス信号に白色ガウス雑音（Gaussian white noise）が
重畳している信号について，正規直交ウェーブレットであるドブシ
ーのウェーブレット（Daubechies wavelet ) を用いて，ウェー
ブレット変換してウェーブレット展開係数を求め，その絶対値があ
るしきい値より小さいものを零に置換えて再構成を行う，ウェーブ
レット縮退（wavelet shrinkage) を実行した例を図4･25に示
す．図4･25(ａ)の雑音を含む原信号から雑音が除去された図4･
25(ｂ)の再構成信号が得られている．比較のために，原信号を
FFTにかけ，高周波成分を零に置換えてIFFTにより再構成した
信号が図4･25(ｃ)である．ウェーブレット縮退の場合と異なり，
原信号の特徴が損なわれており，ウェーブレット縮退の有効性がわ
