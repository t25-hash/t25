# 5・1・1 リンク機構(linkage)の構成

> **章**: 第5章 運動変換要素
> **節**: 5・1 リンク機構

機械を複数の物体の運動系とみると、機械は複数の物体が互いに接触し、拘束を受ける状態で組立てられたものとモデル化できる。このモデルを機構という。このとき物体を節(link)、二つの節のそれぞれの接触部分を対偶素(pairing element)と呼び、二つの物体の対偶素の組合せを対偶(kinematic pair)という。対偶には二つの物体を点接触や線接触で拘束する高次対偶と、面接触で拘束する低次対偶がある。節を対偶により連結した運動系を連鎖(kinematic chain)といい、連鎖の一つの節を基準とし、それに対してほかの節が相対運動を行うとき、その連鎖を機構(mecha-nism)という。機構は用いられている対偶の種類によりいろいろな名称が付けられているが、一般に低次対偶から構成される機構をリンク機構(linkage)という。高次対偶からなる機構にはカム機構、歯車機構などがある。二つの節が対偶で拘束され、それらの間の運動の自由度がであるとき、この対偶の自由度をという。対偶には多くの種類があるが、多く用いられるのは1自由度低次対偶で、(1)回り対偶(revolute pair):軸回りの回転運動、(2)進み対偶(prismaticpair):軸に沿っての並進運動、(3)ねじ対偶(helical pair):軸に対するねじ運動がある。連鎖は、節が閉回路を成しているか否かにより、閉ループ連鎖と開ループ連鎖に分類される。開ループ連鎖の中で、節が分岐せずに直列に結合させたものを特にシリアル連鎖という。従来の機構は閉ループ連鎖で構成されたものが中心であったが、最近はロボット機構を中心としてシリアル連鎖が用いられる場合も多くなっている。閉ループ機構は、そのループの中で取り得る運動の種類によって、ねじ機構、平面並進機構、平面機構、球面機構、空間並進機構、空間機構などに分類できる。対偶は二つの物体間に運動の拘束を加えるものであり、自由度の対偶は6-の拘束を与える。したがって、機構の自由度は、機構を構成する静止節を除く節がおのおのほかの節と関係なく自由に運動するときの自由度の総和から、対偶による拘束を引いたものであり、機構を構成する節の数を、対偶の数をとすれば、機構の自由度は욣-욡=∑욣욡-α(I-5・1)=-1-∑となる。ここには上記平面運動、空間運動などの種類によって定まる定数であり、平面機構の場合は3、空間機構の場合は6である。またαは機構のもつループの数である。式(I-5・1)はこれは対偶のもつ自由度から、ループでの拘束であるを、ループ数分引いたものと考えることもできる。
