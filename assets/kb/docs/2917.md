# 4・1・3 生産方式

> **章**: 第4章 自動化システム
> **節**: 4・1 生産システム自動化の体系

20世紀を代表する生産システムはフォード生産システム（Ford
production system）とも呼ばれるベルトコンベヤによる流れ作業
システムである．流れ作業は，製造ラインにて実行される複雑な作
業を単純な要素作業の並びに分解し，その作業を連続的に実現する
方式であり，フォード生産システムは上述のようにこの連続作業を
ベルトコンベヤで実現する．流れ作業は組立ラインのみならず加工
システムにおいても一般化した．自動車の部品製造におけるプレス
機械を複数台並べたり，多重ヘッドの穴あけ加工機械を連結したり
することで，トランスファマシンと呼ばれる専用加工システムが作
られた．組立工程においても，インデックスマシンと呼ばれるカム
駆動の割出機構で複数の組立装置が連動する専用組立機が使用され
た．トランスファマシンは製品ごとに設計される専用システムであ
る．しかし，組立作業の多くは機械化ができていなかったので，手
作業のままに残された．例えば一見単純に見えるねじ締め作業も，
(１)ねじ位置が散らばっていること，(２)その位置にばらつきがあ
ること，(３)ねじ締め作業の成否の判断が難しいこと，などの理由
から自動化は困難であった（現在でも，自動車のぎ装ラインでのね
じ締め作業の自動化率は必ずしも高くない)．
フォード生産方式により製品価格は低下し，その結果，いっそう
大量販売が可能となり，大量生産・大量消費が可能となった．しか
し，生産ラインの設置コストは高く，かつ，生産ラインが長くなっ
て仕掛品が多数ライン内に存在するなどの問題点が明らかになって
いった．
この問題に対応したのがトヨタ生産方式（Toyota production
system，徹底したむだの排除で注文によって生産し，切れ目のな
い流れ生産方式を用いた多品種少量生産方式）である．1980年代
に大野耐一によって広められたこの生産方式は生産改善活動という
側面も強いが，同時に多くの生産方法の概念を提示した．この方式
では，生産に取り掛かってからできあがるまでのプロセスに着目
し，むだを徹底的に排除することを目標とした．例えば，仕掛品を
少なくすることで，生産ライン内に発生したトラブルはすぐ下流へ
と影響を与える．そのため，ライン全体の生産性を向上させるに
は，トラブル発生箇所の担当者のみならず，他の作業員も問題解決
に協力しなければならない．このように，安定的な生産システムの
CIM
構築にはライン担当者全員で取組むことが必要となるとの発想をも
っている．このシステムのもう一つの特徴は，プルシステム（pull
system，下流側の要求によって，上流側が生産物を供給する方式）
にある．これによって，いわゆる見込み生産を廃し，作りすぎのむ
だを排除する．このプルシステムでの物の流れを単純な方法で制御
しているのがかんばんシステムである．かんばんとは1回に納入す
べき部品の種類と量を表示した指示伝票であり，納入先（受取り
手）のところに部品がなくなると，注文伝票として部品メーカへそ
のかんばんが渡され，かんばんの情報に従って，次の部品が納入さ
れる．このかんばんシステムは部品の流れと情報の流れを上手に同
期させるために作られたシステムである．その結果，各工程は部品
が必要となったときにのみ部品が供給されることとなり，ライン内
の部品在庫が減少し，製品の仕掛品も減少する．この考え方はジャ
ストインタイム（just in time）と呼ばれ，必要なときに必要な部
品供給がされることを目標としたもので，かんばんシステムと組ん
で実現される．なお，トヨタ生産方式も生産ラインの構造としては
流れ作業を基本としており，フォード生産システムとの共通性は高
い．
1990年代に入ると，インタネットの一般化により情報の流通に
おいても実時間性と大量データ操作とが容易になった．CADデー
タやCAMデータをインタネットで送ることで，設計地から遠く離
れた海外での生産が可能となった．円高から多くの製造業が東南ア
ジア・中国などへ移り，日本や工業化国では製造業の空洞化が進ん
だ．そのため，国内ではいままで以上に低コストでの生産システム
の構築が必要となっている．2004年現在，日本における生産シス
テム構築の考え方は二つに分化している．一つは高い人件費を圧縮
することを目的として自動化を積極的に推進する方法論であり，い
ま一つは人を活用することで設備コストならびに設備設計コストを
徹底的に圧縮する方法論である．
後者の典型はセル生産システム（cell production system）と呼
ばれ，少数（できれば単独の）多能工により製品の組立と品質保証
のすべてを担当させる方式である．これはベルトコンベヤシステム
の欠点である生産機器据付設置にかかる時間や手間をなくし，市場
変化に迅速に対応できるという特徴をもつ．ただし，このセル生産
方式は基本的に部品搬送が人手ででき，かつ，人手作業が容易に適
用できる生産対象で強みをもつ．
