# 5・1・2 剛体の運動の表現

> **章**: 第5章 運動変換要素
> **節**: 5・1 リンク機構

三次元ユークリッド空間内の剛体の運動は、基準座標系に対する剛体に固定された直交座標系の位置と姿勢で表される。この座標系を剛体座標系という。剛体の位置・姿勢が時間の関数のとき、これを剛体の運動という。ここに上付きのは行列の転置を示す。機構の形状は、節における対偶素の位置関係と、対偶を構成するしました。3/5再校で、全文差し替え4―119対偶素間の変位(対偶変位)で定まる。いま1自由度対偶からなるシリアル連鎖の機構において、静止節を1とし、静止節側から節に2,3,...と番号を付ける。また対偶にも静止節側から1,2,...と番号を付ける。この座標系を節の座標系とする。これら四つのパラメータで節との位置関係を表すことができる。これらはDH(Denavit-Hartenberg)パラメータと呼ばれる。回り対偶からなる平面機構や球面機構ではこのパラメータのうちのいくつかが常に零の値をもつため、より少ないパラメータ数で機構を表現できる。開ループ機構において、対偶変位が与えられたときの節の運動は、静止節から順次先端の節に向かって式(I-5・4)を計算すればよい。閉ループ機構の変位解析では、式(I-5・4)をループ一巡に適用すると最初の節の座標系に戻るということから関係式を導くことにより対偶変位間の関係が求まる。しかし行列の要素比較により得られる式には従属な式が多く含まれる。そこで閉ループ機構の変位解析では、一般に平面三角法や球面三角法を用いる(5・1・4・a,d項参照)。
