# 5・1・2 剛体の運動の表現

> **章**: 第5章 運動変換要素
> **節**: 5・1 リンク機構

三次元ユークリッド空間内の剛体の運動は、基準座標系に対する剛体に固定された直交座標系の位置と姿勢で表される。この座標系を剛体座標系という。基準座標系に対する剛体座標系の原点の位置と座標軸の方向余弦ベクトルからなる直交行列とにより与えられる行列=(I-5・2)により、剛体の位置・姿勢が表される。剛体の位置・姿勢が時間の関数のとき、これを剛体の運動という。剛体座標系で1 で与えられる剛体上の点の、基準座標系の位置1 は=(I-5・3)により計算することができる。ここに上付きのは行列の転置を示す。機構の形状は、節における対偶素の位置関係と、対偶を構成するしました。3/5再校で、全文差し替え4―119対偶素間の変位(対偶変位)で定まる。いま1自由度対偶からなるシリアル連鎖の機構において、静止節を1とし、静止節側から節に2,3,...と番号を付ける。また対偶にも静止節側から1,2,...と番号を付ける。連続する対偶および=+1の座標系を図I-5・1に示すように定める。対偶軸ととの共通垂線と対偶軸との交点を節上の対偶素の座標系の原点とし、対偶の軸を욢軸とする。また共通垂線上に、対偶からの方向に욢軸を定め、右手系に욢軸を定める。この座標系を節の座標系とする。このように定めると、節と節の座標系の関係は、節の座標系原点から共通垂線までの距離욡,욡軸と욢軸のなす角θ욡、共通垂線の長さ욡욢、および対偶軸とのなす角α욡욢、で定まる。これら四つのパラメータで節との位置関係を表すことができる。これらはDH(Denavit-Hartenberg)パラメータと呼ばれる。回り対偶からなる平面機構や球面機構ではこのパラメータのうちのいくつかが常に零の値をもつため、より少ないパラメータ数で機構を表現できる。回り対偶からなる平面機構では욡욢,θ욢、球面機構ではα욡욢,θ욢のおのおの二つのパラメータで表される。静止節に対する節≧2の運動욢は、욢=...욡욢(I-5・4)で与えられる。ここに욡욢は節とをつなぐ対偶により生じる運動で、節からみた節の運動を表す行列である。対偶が回り対偶および進み対偶の場合の욡욢は、sinθ욡sinα욡욢-sinθ욡cos α욡욢cos θ욡욡욢cos θ욡욡욢=-cos θ욡sinα욡욢cos θ욡cos α욡욢sinθ욡욡욢sinθ욡cosα욡욢sinα욡욢욡(I-5・5)と、DH パラメータを用いて表される。回り対偶のときはθ욡、進み対偶のときは욡が対偶変位である。開ループ機構において、対偶変位が与えられたときの節の運動は、静止節から順次先端の節に向かって式(I-5・4)を計算すればよい。閉ループ機構の変位解析では、式(I-5・4)をループ一巡に適用すると最初の節の座標系に戻るということから関係式を導くことにより対偶変位間の関係が求まる。しかし行列の要素比較により得られる式には従属な式が多く含まれる。そこで閉ループ機構の変位解析では、一般に平面三角法や球面三角法を用いる(5・1・4・a,d項参照)。
