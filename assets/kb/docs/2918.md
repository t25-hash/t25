# 5・1・3 剛体の速度と加速度

> **章**: 第5章 運動変換要素
> **節**: 5・1 リンク機構

ここに[×]はの外積に等価な反対称行列を表す。図I-5・2のように、剛体2が静止している剛体1の座標系に対しで与えられる運動をしているとし、また剛体3は剛体2の座標系に対しての運動をしているとする。式(I-5・9)右辺の第2項は剛体3の剛体2に対する相対速度を基準座標系で表したものである。この相対速度をで表すと、式(I-5・9)右辺の第1項は剛体2の剛体1(基準座標系)に対する相対速度であるから、次式の関係を得る。=+(I-5・10)これは、剛体3の速度は剛体1と2,2と3のそれぞれの相対速度の和であることを示している。すなわち運動している剛体上の点の速度はその点の基準座標系でみた同次座標に剛体の速度行列をかければ得られる。このとき、右辺第1項のベクトルは回転運動を表し、第2項は回転軸に沿っての並進運動を表す。このように剛体の速度は一つの軸回りの回転速度とその軸に沿っての並進速度の和と表すことができる。すなわち、剛体は瞬間的に一つの軸回りのらせん運動(ねじ運動)を行っており、この運動を剛体の瞬間らせん運動(instantaneous screw motion)といい、この軸を瞬間らせん軸という。平面運動の場合、はに直交するから、剛体は回転運動を行い、式(I-5・13)で与えられる点は平面上のその回転軸が通る点を表す。これを平面運動の瞬間中心または速度極という。ここには平面に垂直な単位ベクトルである。この関係から、平面運動をする三つの剛体の間に存在する三つの瞬間中心は一直線上にあることがわかる。これをケネディーの定理(Kennedy’s theorem)という。この関係が剛体上の点の加速度を与える。
