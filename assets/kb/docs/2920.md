# 5・1・5 機構の総合

> **章**: 第5章 運動変換要素
> **節**: 5・1 リンク機構

機構の総合(synthesis)とは与えられた条件を満たす機構を作り上げることである.これには機構の節数,対偶数を決定する数の総合(number synthesis)と,節の寸法を決める量の総合(dimensional synthesis)がある.a. 数の総合機械を設計するには,まずそのモデルである機構が行う運動の種類と機構の自由度を定める.これらは機械の行うべき作業,運動により定まるものである.このとき,機構の節の数と対偶の数の関係は式(I-5・1)で与えられる.これをもとに機構を作るための節の数と対偶の数を決定することが数の総合である.一つの節が個の節と隣接している場合,その節には個の対偶素があることから,対偶素節という.対偶素節の数を욇とすると,節数,対偶数との間に=∑욡(I-5・41)=12∑욡(I-5・42)の関係が成り立つ.式(I-5・1)(I-5・41)(I-5・42)をもとに節数,対偶数を決定する.解は多数存在するが,すべての対偶の自由度が1のとき,=+α(I-5・43)=+1+α-1(I-5・44)で与えられる.また,,,...,욇の解が一組求まったなら,その中の욡と욢≧3,≧2の値を1減らし,욡욪と욢용の値を1増やしても,式(I-5・41)の値は変化しない.この性質を利用し,残りの解を求めて行くことができる.b. 関数創成機構機械装置内に,リンク機構の入力角θと出力角φの関数関係φ=θ(I-5・45)を利用することがある.このように,入力変位と出力変位が所望の関係を満たす機構を関数創成機構という.平面4節機構を関数創成機構として総合するために用いることのできるパラメータは,節の長さ,入力角と出力角の基準位置である.一つの節の長さを基準長さとし無次元化することにより,節の長さは三つのパラメータで表されるから,独立なパラメータの数は全部で5となる.入出力変位曲線上のいくつかの点での入力角とそのときの出力角の数値を与え,その条件を満たすように機構のパラメータを決定する手法を厳正点による総合という.このとき,厳正点の数は,パラメータの数を超えることはできないから,最大で5である.厳正点法では,機構の入出力関係を,パラメータをすべて含んだ形で定式化し,それに厳正点の数値データを代入し,パラメータを未知数とする連立方程式を導く.この方程式は非線形の方程式である.この方程式を数値的に解き,機構のパラメータを決定する.入力角と出力角の基準位置を固定して,三つの厳正点を与えて,リンク長さを計算することはフロイデンシュタインの式(Freudenstein’sequation)を用いて容易にできるが,基準点を変化させると,その処理は複雑となる.また,五つの厳正点を与えると,機構のパラメータがすべて決定されるため,そのパラメータが機構として好ましくないものであっても,その修正ができない.厳正点の数がパラメータの数より小さいときは,その差分だけ,パラメータを修正する自由度があることになる.リンク機構では連続関数の全域にわたって関数値を厳密に満たすことは不可能である.ただし,ある誤差範囲内で,定められた関数を満たすような機構を総合することは不可能ではない.このときは,まず,厳正点法で,候補とするパラメータ値を決定し,これをもとに,パラメータを修正し,最適なパラメータを探索する手法がとられる.c. 経路創成機構の総合中間節が平面内に定められた所定の位置を通過する機構,あるいは中間節上の1点が所定の経路(中間節曲線という)を描くような機構を経路創成機構という.前者の総合では,静止節と中間節の相対位置関係として,静止節に対し中間節上に定められた線分の位置を複数箇所与える.機構の運動にしたがって,定められた線分の位置を中間節が通過するように機構を総合する.図I-5・8に示すように,中間節上の線分が指定された線分に一致するよう,複数の中間節の位置(図では4)を定め,そのときに中間節上の点で,一つの円周上にあるものを二つ探すものである.円周上にある点と,その円周の中心とを節で結合すると,4節機構が定まる.中間節の位置として,平面上に三つの線分の位置を与えると,中間節上の任意の点は一つの円周上にある.したがって,この場合は容易に機構が得られる.四つの線分の位置を与えたとき,中間節上で,円周上にある点の集合は中間節上の三次曲線(円周点曲線という)となる.また逆に中間節に対し静止節の位置が与えられているとみると,同様にして静止節上に三次曲線(中心点曲線という)が定まる.円周点曲線上の一つの点に対し,その中心となる点が中心点曲線上に定まる.これら二つの三次曲線を合わせてブルメスタ曲線(Burmester curves)という.円周点曲線上に二つの点を定め,それらの点に対応する中心点曲線上の点とを節で結合することにより,4節機構が得られる.中間節上の1点が所望の曲線を描くような機構の総合には剛体の瞬間運動を利用した総合手法がある.4節機構の中間節上の点が描く軌跡の曲率,曲率の停留点の作る曲線が機構の姿勢から定まる.曲率が零である点の集合は円となり,これを変曲点円(inflection図-5・8 中間節の四つの位置に対する円周点と中心点図-5・9近似L字線を描く機構(A,B:中心点,A,B,A′,B′:円周点)circle)という.曲率の変化率が零の点の集合は三次曲線となり,これを曲率停留点曲線(cubic of stationary curvature)という.曲率停留点曲線と変曲点円との交点をボール点(Ball’s point)といい,この点は曲率が零でその変化率が零という,直線に近い軌跡となる点である.以上のような剛体の平面運動の瞬間運動の性質を利用して,機構を総合することができる.図I-5・9に機構の総合例を示す.これは部分的にL字形をもつ軌道を創成し,L字経路の一つの端点Tは直線に近い経路をもつ機構である.点T はボール点上にあり,点T と点T′での軌跡の接線が直交するような機構を,変曲点円と曲率停留点を表す三次曲線(例では速度極が無限遠であるため,曲率停留点:,軸,変曲点円:軸)の位置関係から求めたものである.
