5・2・2カムの種類と用途
カム機構は表Ⅰ-5･1に示すように，カムの形状が平面的である
か立体的であるか，カムの運動が直進か回転か，従動節の運動が直
動か揺動か間欠回転か，従動接触子の形状が尖端従動子（knife-
edge follower）であるか円端従動子（roller follower）であるか
平端従動子（flat surface follower）であるかによって分類される．
表Ⅰ-5･2 はカムの形状を示す．
カム機構は高速の自動機械においてさまざまな運動（motion）
を作る目的に用いられる．表Ⅰ-5･3 は従動節が間欠回転運動を

またカム装置は内燃機関の弁の開閉などにも用いられ，速い繰返し
の運動を確実に制御することができる．

c .停留と連続条件 カムは回っているが従動節は静止してい
るような状態を停留（dwell）という．従動節の往復運動の内の片
道の動きに対し，両端に停留のある運動を両停留運動（dwell -
rise- dwell motion，two-dwell motion），一端だけに停留のある
運動を片停留運動（dwell - rise-return- dwell motion，one-dwell
motion），両端ともに停留のない運動を無停留運動（rise-return-
rise motion，no- dwell motion）という．
高速用に用いられるカム曲線は変位，速度，加速度が連続でなけ
ればならない，この条件は上のそれぞれの場合で異なるため，それ
ぞれに対して連続条件を満たす曲線が考案されている．
