# 4・2・2 離散系生産システムの構成要素

> **章**: 第4章 自動化システム
> **節**: 4・2 離散系生産システム

図4･7は2台のマシニングセンタを無人搬送台車（AGV）で結
合したFMSの例で，システムの構成要素は図中に示すように5種
類に大きく分類できる．
a.工作機械（生産機械) 生産システムの中核をなし，工作
物を加工するために工具を有し，創成運動を行う機械である．工具
図4･ 7 MCとAGVによるFMS（カーネイアンドトレッカ社）
交換装置（ATC）と一体化し，CNC制御されたマシニングセンタ
などが利用される．
b.加工エリア（作業領域) マシンエリア，ワーキングエリ
アあるいはワーキングステーションと称され，工作物を加工する場
所を意味する．従来は工作機械のテーブル部分が対応していたが，
自動パレット交換装置（APC:automatic pallet changer ）との組
合せ，インデックステーブルの利用による多面加工，などのよう
に，工作機械のコラム側と加工エリアを分離した形態が増えてお
り，a項の工作機械とb項の加工エリアを分けて扱う必要性が生じ
ている．
c.着脱エリア（着脱ステーション) ロード/アンロードステ
ーションで，物の取付け取り外しを行う機能と場所を示す．工作物
の着脱，工具マガジンへの工具の装てんなど，物流と結び付いた働
きをするところである．図4･7では工作機械の着脱エリアはパレッ
ト交換部に相当するので，b項の加工エリアと区別が難しい例もあ
る．
d.搬送装置（搬送システム) 工作物や工具など，物を移動
する装置，システムを意味する．搬送方法と搬送経路が物流の特性
を決めることになり，工作機械の配置方法とも密接に関連する．
e .保存エリア（ストックエリア) 物を蓄える場所である．パ
レットプールや自動倉庫などと直結したFMSが開発されているこ
とから，重要性が増している．図4･7では工作機械の前に設置され
たパレットをプールする場所が保存エリアに対応する．また，
AGVへの着脱を行う場所は，パレット上の工作物やマガジン内の
工具を一時保存する場所ともみなせることから，c項の着脱エリア
とe項の保存エリアを兼ねる場合もある．
生産工程は加工以外にも組立工程，検査工程があるので，組立機
械，測定機などがシステム内に共存する場合があり，工作機械と同
様に重視される．個々の生産機器はNC化され，集中管理方式で
はなく，自律分散で制御される傾向にあり，相互に協調してシステ
ム全体を制御する必要性が生じている．離散系生産システムでは稼
働率，運用効率を種々の状況でシミュレーションしながら最適化す
るためのシステムコントローラが重要な役割を果たしている．
