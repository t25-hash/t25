# 4・2・3 工作機械の配置と物流経路

> **章**: 第4章 自動化システム
> **節**: 4・2 離散系生産システム

工作物の搬送経路は工作機械の配置方法に依存し，加工順序と関
連して生産性に大きく影響することが知られている．工作機械の代
表的な配置方法は次の3種類に大別でき，工作物の種類やロットサ
イズなど生産形態により物流が異なる．
a.機種別配置（functional layout ) 工作機械の種類ごとに
まとめて配置する方法で，工作物の移動距離が長く，生産性，品質
管理の点では劣るが，代替できる工作機械が隣接しているため，工
作機械の故障に対処しやすく，保守管理が容易であることから，多
品種少量生産に適している．
図4･8工作機械，生産システム，配置方法の生産性，融通性
b.グループ配置（group layout ) 異なる複数の工作機械を
グループ（セル）としてまとめて配置する方法で，工作物の搬送距
離が短く品質管理が行いやすい利点をもつ．グループ内での変更が
可能で，全体を把握できることから，作業者に適した方法といわれ
ている．類似部品をファミリーとして集めて加工を行うことで量産
効果を得る方法であるGT（group technology）と対応してお
り ，セル生産方式，GTセル生産とも称される．
c .直列（直線）配置（line layout ) 工作物の加工順序に合わ
せて工作機械を直列（直線）に配置する方法で，流れ型生産システ
ムと対応し，搬送経路を短くすることができ，高い生産性が得られ
る．工作物の種類ごとにラインを構成する必要があり，1台の工作
機械が故障するとライン全体が停止してしまうことから，融通性に
は乏しいとされている．
図4･8は工作機械の配置方法が有する特徴を生産性，融通性で表
した例で，図中には工作機械や生産システムについても概略的な位
置関係を示している．前出の図4･6と比較するとわかるように，生
産性と融通性は相反関係にあり，工作機械の配置方法は生産形態と
も密接に関連している．図4･8では単体の工作機械を□で，複数の
機械で構成されたシステムを○で示している．
FMSはグループ配置のものが多く，生産性と融通性の向上を目
標に種々のシステムが開発されている．トランスファマシン
（TM:transfer machine）は搬送ラインに加工エリアを直結させ
た直列配置に対応しており高い生産性を上げている．これに対して
フレキシブルトランスファライン（FTL:flexible transfer line）
は搬送ラインと工作機械を多機能化し，TMより融通性を向上さ
せたものである．一方，個別生産やバッチ処理を対象とした従来の
ジョブショップでは，機種別配置が一般的で搬送経路が複雑で自動
化が難しい．
