# 5・2・6 動特性を考慮したカム機構の設計

> **章**: 第5章 運動変換要素
> **節**: 5・24 に示す。このほかにこれらを組合わせたものがある。各種の

速度が変化する運動を実現する装置には、昔からリンク機構やカム機構が用いられており、近年ではサーボ機構が広く用いられるようになった。しかし、運動が固定している場合における高速駆動では、ロータリ形表面実装機のようにカム機構に優る機構はない。高速駆動カム機構の設計においては、機構の動特性を考慮した設計を行うことが大切である。a. カム曲線の残留振動に関する評価高速でスタート・ストップを繰返すカム機構では、残留振動が生じ、これは機構の位置決め精度に影響する。インデックス装置などの位置決め装置の残留振動特性の観点からカム曲線を選択する場合には、次のことを考慮する。1カム曲線の非対称性は残留振動の抑止に効果がある。2減衰係数が比較的大きい場合は、位置決め開始時の加速度が不連続なカム曲線を使ってもよい。しかし高速駆動では、加速度の不連続性の影響が十分減衰する前に残留域に達するので、加速度の不連続なカム曲線は使うべきでない。3代数多項式系と正弦関数系のカム曲線の残留振動への影響は、それぞれのカム曲線によって異なる。4位置決め完了直前のジャークの大きさは残留振動に影響する。b. トルク補償カム機構産業機械における位置決めの高速化においては、高速になるほど激しい交番負荷が作用するため、高速・高精度位置決めには限界がある。この限界を打破する技術として、トルク補償カム(TCC: Torque Compensation Cam)機構が開発された。この機構の原理は本体であるカム装置による変動トルクを、カム装置に取付けたTCC装置から発生する変動トルクで相殺・平滑化するものである。この原理は、カム装置とTCC装置を含む系全体で図I-5・26に示すように、加減速にかかわらず駆動エネルギーを一定にしようとするものである。すなわち、カム装置が加速時に必要なエネルギーをTCC装置からもらい、減速時にはこのエネルギーをTCC装置に戻すものである。TCCには慣性エネルギー方式とひずみエネルギー方式があり、後者は機械ばね方式と空気ばね方式に分けられる。図I-5・27はばね方式TCC装置の例である。このようなTCC装置を取付けると、過渡振動、残留振動が減少し、結果として位置決め整定時間を短縮することができ、高速化が可能になる。またモータの必要定格容量が減少するので、省エネルギー効果も高い。TCC装置の実施例の図-5・26 トルク補償原理図-5・27 ばね方式のTTC装置の例4―129第I部機械要素顕著なものはマイクロチップの超高速表面実装機であり、最高速機の半分以上がTCC装置を搭載している。TCC装置はカム装置を対象に開発されたが、リンク機構など、高速、固定サイクルの装置には適用可能であり、応用範囲は広い。
