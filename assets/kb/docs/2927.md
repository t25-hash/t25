c .生産・製造 生産（production）は製品を生み出す全工程
を意味し，製品を加工するという意味では製造（manufacturing）
ということばが使われる．最近では，製造工程の多くが自動化され
ている．しかし，その設備の完成度を高めていくには，現場で働く
人々の経験と知識で改善できる余地を残しておく必要がある ．
自動化工程はコンピュータの役目であって人間とは無関係であると
いった考えは誤りである．自動化工程ではむしろ人間のものづくり
に関するノウハウや創造力が不可欠なのである．
他方，組立工程においてはほとんどが人の手に依存しているのが
ステム内の部分的な変更，組替えを可能にすることで，対象製品，
生産形態の大幅な変更にも即応できる再構築可能な生産システム
（reconfigurable manufacturing system) の実用化が検討されて
いる．
図4･13はモジュラ構成と生産システムの再構成，再構築性を概
念的に示したものである．工作機械を部分的な要素（モジュール，
ユニット）に分解し，これらを組合わせて工作機械を製造する考え
方は，積木式構成法（building block system）やモジュラ構成法
（modular construction）として古くから提案されている ．標準
化，規格化した主軸モジュール，コラムモジュール，テーブルモジ
ュールなどを数種類ずつ準備しておき，それらを組合わせて，工作
機械を短期間で設計・生産する方法である．同一機種だけでなく，
旋盤系とフライス系など異機種へのモジュラ構成の適用が試みられ
ている．
図4･13に示すように，モジュラ構成は単体工作機械の構築を目
的としているが，モジュールやユニットに搬送装置やロボットなど
を含めることにより，セルレベルのFMCの構成，さらにシステム
レベルのFTLやFMSの設計・生産にも適用できる．既存のシス
テムに対しても，余分なモジュールを取り除き，必要なモジュール
を追加することにより，簡単に新たなFMSやFTLに組替え可能
となる．異種モジュール間の結合性，互換性を実現するとともにシ
ステム制御，運用ソフトウェアと総合することで，システムレベル
での変更を行ってもすぐに全体の立上げ稼働が可能となり，離散系
生産システムとしての再構成・再構築性（reconfigurability）の利
点が生かされる．
生産のグローバル化や要求仕様の多様化に従い，大量生産時代を
支えた大規模な生産システムは変ぼうし，最近では身軽（lean
manufacturing）で機敏（agile manufacturing）に対応できるシ
ステムへの要望が高まり，再構築可能な生産システムの実現が今後
の重要な課題となっている．
