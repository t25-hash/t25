# 5・3・1 間欠運動の概要

> **章**: 第5章 運動変換要素
> **節**: 5・3 間欠運動機構

一定速度で運動している原動節から従動節に対して間欠的な運動を伝達する機構を間欠運動機構(intermittent motion mecha-nism)といい,これを実体化した装置を間欠運動装置という.従動節に,停止―加減速―停止を繰返し行わせる手段としては機械式の間欠運動装置とともに,いわゆるメカトロニクス技術を用いたシステム機器(メカトロニクスシステム)がある.メカトロニクスシステムでは,ハードウェアの制御のためにソフトウェアが重要な役割を果たし,ソフトウェアの論理により,アクチュエータを制御して,間欠運動を創成する.しかしながら,コンピュータとハードウェア間の情報の入出力に費やす時間,位置決めのむだ時間などにより所要の速度条件を満足しないなどの理由により,機械式の間欠運動装置が用いられる.この間欠運動装置の場合,各機械要素の形状寸法に所要の運動を実現するためのソフトウェアの代替内容が埋込まれているとみなすことができる.したがって,間欠運動装置では,メカトロニクスシステムで行われるセンサ情報に基づくアクチュエータへの信号送信による運動実現の代わりに,アクチュエータを等速運動させるだけで,高精度の形状寸法を有する機械要素間の相対運動に基づいて所要の間欠運動が実現される.このため,メカトロニクスシステムを用いる場合以上の高速作動が可能となることが多い.このほかに,間欠運動装置は,メカトロニクスシステムのような停止区間や加減速区間の時間間隔を自由に設定可能なフレキシビリティに劣る反面,高負荷作業,多数の動作端の同期性,高低温,塵埃,振動などの環境下における信頼性,さらに耐久力,保守・点検,製作・運転費などに優れている.間欠運動装置は,原動節の運動中の特定な時間および区間を限って従動節の運動が停止するような自動機械に適用されるから,その設計は,物体を所定の静止位置から搬送して停止させるための停止位置,搬送物体の姿勢,搬送径路,物体の速度,加速度などの適正な条件を考慮して行われなければならない.間欠運動装置のうち,機械式なものとしてはゼネバ機構,間欠歯車装置,カム装置,つめ車,リンク装置,リンクに歯車やチェーンを組込んだ装置などがある.
