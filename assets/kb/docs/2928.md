# 4・3・1 人間と生産活動

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

ものづくりの能力は人間が太古から備えた本能であり，生産活動
は何千年もの間継続されてきた人間特有の知的活動であるといえ
る．生産の自動化が進展した現在においても，自動化技術を支える
のはやはり人間のもつ創造力やノウハウであり，また，最終的な意
思決定を行うのもやはり人間である．
a.製品開発・企画 製品開発の初期段階において，何を，ど
のように作るか，どのような構造，機能にするか，社会のニーズに
マッチするかなどといった思索は，開発者の経験や知識に基づき発
見的（heuristic）な方法で展開される．生産活動の第一に行われ
るこういった創造的活動は，自動化が困難であり，この段階におけ
る人間の役割は大きい．
1品生産方式を採用する企業が現れた．これはセル生産方式あるい
は屋台方式 と呼ばれる．このような方式が結果的に製品完成ま
での生産期間であるリードタイム（lead time）を短縮できた例も
ある．また，このような生産方式では巨大な設備投資が不要である
こと，熟練者を養成できること，需要の変動や作業者の高齢化とい
った環境変化にも柔軟に対応できることなどの利点がある．
