# 5・3・3 間欠歯車装置(intermittent gear drive)

> **章**: 第5章 運動変換要素
> **節**: 5・8 は=1の場合に対するゼネバ機構の運動特性を示す。

図I-5・30の間欠歯車装置は原動歯車bの歯の一部分を取り除いて円弧に置き換えることによって、この部分では従動歯車dに運動が伝達されないから間欠運動が可能となる。しかしながらこの間欠歯車装置では高速度回転の場合慣性力によって停止時の振動を誘発するおそれがある。図I-5・31に示すピンを用いた間欠歯車のほうが滑らかな運動となり衝撃をやわらげる効果がある。図-5・28 ゼネバ機構とその諸元図-5・30 間欠歯車装置(c) 円筒カム(b) 両面カム(a) 二枚の板カム図-5・33 カムによる間欠運動装置図-5・34 つめ車
