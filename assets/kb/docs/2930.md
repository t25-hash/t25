# 4・3・2 人間中心生産システム

> **章**: 第4章 自動化システム
> **節**: 4・2 離散系生産システム

a.無人化工場の矛盾 上述のように，人間が生産活動に密接
に関わっていることは明らかであるが，人間は疲労しやすく，不注
意なミス（ポカミス）を犯しやすいといった弱点をもっており，産
業革命以来，利益追求型の生産工場にとって好まれる存在ではなか
った．そのため，大規模なFA，FMS，ロボットなどによる自動
化・無人化を促進し，人間をできるだけ排除しようという流れがで
きてしまった．
しかし近年，自動化・無人化において以下のような矛盾が指摘さ
れるようになった ．
(１)システムが複雑なため設備投資が増大し，損益分岐点が高
くなる．
(２)システムが硬直し，需要の変動や作業者の高齢化といった
環境変化に柔軟に対応できない．
(３)技術の伝承が行われないため，次世代の生産システムの構
築が困難となる．
(４)ものづくりの喜びが感じられず，作業者の就業意欲を低下
させる．
(５)付加価値の高いものができない．
b.人間中心生産システムの誕生 このような背景のもとに，
人間中心生産システム（human-centered manufacturing sys-
tem) が誕生した．極端な例では，コンベヤラインを導入せ
ず，極めて少人数で製品の1台を組立てる方式が採用されている．
このような生産方式をいち早く取り入れたのが，ボルボ社のカルマ
ー工場 であることから，カルマー方式と呼ばれたこともある．
