# 5・3・4 カムによる間欠運動装置(cam-type intermittent

> **章**: 第5章 運動変換要素
> **節**: 5・8 は=1の場合に対するゼネバ機構の運動特性を示す。

motion mechanism)所要の間欠運動を得るためにはカムに一時休止をもつ輪郭曲線を与えればよいが、従動節先端と一つのカム面との接触によって運動を伝達する場合には、高速度の作動に対して従動節が飛び上がってカム面より離れてしまい確実な間欠運動が望めなくなる。そこで図I-5・32のようにカムbに溝を付けて二つの面bおよびbによって従動節dの先端のころCをはさみ込む必要がある。この場合カム溝のCCの区間がカムの回転中心A を中心とする円周上にあればカムの回転角θの期間だけ従動節dの揺動角φは一定となり、従動節は停止して間欠運動を行うことになる。ところが溝を工作する場合にエンドミルを使用すれば、カムの等速回転に対してカッタ中心の送り速度に遅速が生じ、切残し量が変化し、溝幅L,M およびN の精度にむらができ、カム面ところの間に遊びができて、発熱、摩耗、騒音を発生し、二つの面で押さえた拘束性がなくなるおそれがある。溝幅の不同をなくすためには、溝の輪郭曲線に沿って等速度でカッタを送るようにフライス削りをしなければならない。図I-5・33に示すカム装置においても精度の高い間欠運動を得るためには従動節先端のころが接触する二つのカムまたは溝面の加工方法に十分配慮する必要がある。
