人間工学的技法とは，機械あるいは作業と人間の適合性を検討す
るための手法であるが，それは人間工学独自の手法ではなく，医
学，生理学，心理学，生体工学などから関連のある手法を抽出し，
複合化したものであるといえる．以下は生産システムにおける作業
特性を検討するうえでおもに用いられる人間工学的技法である．

c .フリッカ検査（flicker test ) 人間の中枢性疲労，意識水準
や覚醒水準などを評価する検査である．検査の値は眼や身体の疲労
の代表値として用いられる．検査にはフリッカ値測定器を用いる．
LEDによる点滅光を点滅周波数の高い状態（60Hz ）から徐々に下
げていくと，ある周期のところで点滅を知覚するようになる．この
周期を下降法によるフリッカ値（CFF:critical flicker frequency）
といい，単位はHzで表す．通常30～40Hzの値をとるが，大脳
皮質の活動水準や年齢などによって変化する．
d.筋電図 筋電図（EMG:electromyogram）は作業姿勢や
動作における筋負担を評価する一つの指標である．筋肉は収縮する
と微弱な筋放電が起こる．これを皮膚表面から電極を通して測定し
増幅して，正負に振動する電圧波形が得られる．筋に大きな負荷が
かかると，筋収縮が増大し，筋電図の振幅は増大する．したがっ
て，筋電図の振幅から筋負担の大小が推定できる．筋電図を時間積
分して単位時間当たりの総放電量を求めたり，一定時間ごとの総放
電量をプロットする積分筋電図（I -EMG）を求めたりして評価す
る．また，筋疲労に伴って筋電図の高周波成分が除波化される傾向
にあるため，周波数成分を求めたり，平均周波数を求めたりして筋
身体寸法計測（anthropometry）は，機械的な測定器による従来
疲労を評価する試みも見られる．最近の筋電図測定器には，皮膚の
表面電極からの信号を無線で受信できるものがあり，それを用いる
ことによって被測定者は配線を気にしなくてよくなり，行動範囲も
広がった．
