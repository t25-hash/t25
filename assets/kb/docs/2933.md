# 6・2・1 緩衝器とダンパの機能

> **章**: 第6章 緩衝・制振要素
> **節**: 6・2 緩衝器およびダンパ

緩衝器(shock absorber)とダンパ(damper)は,いずれも物体の運動エネルギーの吸収を目的とした装置であり,多くの場合,熱エネルギーの形で吸収する.ダンパは,計測器,鉄道車両,自動車,航空機,一般機械,構造物などの振動系に付加し,自由振動の減衰や強制振動の共振抑制のために,比較的小さいエネルギーを長時間にわたって吸収する場合が多い.その抵抗力は,流体の動圧抵抗,粘性抵抗,固体摩擦,電磁力などによって与えられる.これに対して緩衝器は,運動を1行程で停止させたり,行程中の最大加速度や最大変位を抑制したりするので,比較的大きいエネルギーを短時間に吸収する場合が多い.緩衝器の性能要件は,許容される最大荷重と吸収できる最大エネルギーが十分であること,緩衝効率と消費効率が高いことである(図I-6・8).その抵抗力は,流体の動圧抵抗,固体摩擦,金属,ゴムおよび空気の弾性などによって与えられる.
