# 6・2・3 摩擦緩衝器

> **章**: 第6章 緩衝・制振要素
> **節**: 6・10 )。

a. 構造しゅう動部の摩擦によって運動を抑制するもので、くさびを利用するものが多い。くさび摩擦緩衝器は図I-6・11のような機構を有し、コイルばねと組合わせたものが多い。輪ばね緩衝器は図I-6・12のように外輪と内輪との摩擦を利用しており、コイルばねが寸法調整と小さい衝撃の吸収のために取付けられている。b. 特徴比較的簡単な機構により大きな抵抗力(例えば変位50mmで500kN)が得られ、消散効率も高い(例えば75%)。緩衝効率は通常のばねと同様で、たかだか50%である。摩擦が生じたり、摩擦係数が変動するという欠点もあるが、鉄道車両連結部には広く使用されている。図-6・9ダシュポット図-6・10 油圧緩衝器図-6・11 くさび摩擦緩衝器
