# 6・2・4 ばね緩衝器

> **章**: 第6章 緩衝・制振要素
> **節**: 6・10 ).

金属ばね,空気ばねは小容量の緩衝器として使われるが,緩衝効率は50%であり,消散効率は零に近い.ゴム緩衝器は図I-6・13に示されるような構造を有し,ゴムの圧縮またはせん断弾性を利用している.その抵抗力は剛性漸増形の非線形特性をもつため,緩衝効率は50%より小さいが,構造が簡単で経済的であり,パッドの組合せ枚数を変えていろいろな特性をもたせることができるため,鉄道車両連結部,バンパ,ストッパなどに広く使用されている.このほか,広義のばね緩衝器としては,液体の圧縮性や固体の塑性変形を利用したものがある.
