# 6・2・5 油圧ダンパ

> **章**: 第6章 緩衝・制振要素
> **節**: 6・10 ).

a. 構造流体動圧ダンパ,オイルダンパ,ショックアブソーバとも呼ばれており,その原理は図I-6・9のダシュポットと同じであるが,抵抗力を運動速度の関数として定めるため,オリフィスの部分には表I-6・4に示されるような調圧弁を設けている.オリフィスの面積はばねで押さえられた調圧弁のリフトによって定まり,そのリフトはシリンダ内圧に左右されるから,式(I-6・12)により弁のリフトの釣合い位置がピストン速度によって定まり,抵抗力は速度の関数として決定される.抵抗力が一方向のみに働く片効きのものと往復方向に働く両効きのものとがある.b. 特徴吸収すべきエネルギー量に対して小形に設計でき,しかも抵抗力が広い振動数にわたって安定しているため,自動車,鉄道車両の懸架装置をはじめ,広い分野に使用されており,その種類と特徴を挙げると表I-6・5のようになる.図I-6・14,I-6・15およびI-6・16には軸揺動形と直動形の例を示す.なお通常の標準設計品では,振幅が5mm以下では抵抗力が低下し,温度が20°C以下では増加する.
