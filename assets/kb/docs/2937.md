# 4・3・9 生産システムにおける人間工学の応用事例

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

実際の工場における作業姿勢（work posture）の改善を目的とあるいは無人化といったものは，できるところだけを自動化し，自つ．
したがって，人間が担当する作業は重筋作業であったり，悪環産システムは，働く喜びは保証されるものの，身体の疾病やストレ産性をいかに両立させるかは今後も課題として残る．
以上人口比率は2005年までに世界一高くなり，2010年までにそのる．
さらに2020年には4人に1人が，2050年には3人に1人が65間違いない．
年齢とともに低下する心身機能をいかにサポートし，齢者（elderly people）を弱者としてとらえるのではなく，高齢者場を指向する視点が必要である．
生産の無人化指向に加え，生産拠点の海外移転や高齢在，ディジタルマイスタープロジェクトが国家レベルで進められて承方法の確立を早期に達成する必要があろう．
動化が技術的に困難な部分を人間が担当するといったケースが目立境下の作業であったりする．
また，昨今の人間を中心とするセル生スの増大は否めない．
そのような状況のもとで作業者の人間性と生b.勤労者の高齢化 日本において，高齢化指標である65歳値は21 %を超え，世界で最も早く超高齢社会を迎える見通しであ歳以上となる．
生産に携わる人々もそれに準じて高齢化が進むのは現在の生産性をいかに維持するかが大きな問題となる．
そこでは高が積極的に参加できる場があって元気に豊かに働けるような生産工c .技能伝承 各企業において技能（スキル）の伝承が危ぶま者の増加と若年者の減少などがその問題に拍車をかけている．
