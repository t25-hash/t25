# 4・3・9 生産システムにおける人間工学の応用事例

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

実際の工場における作業姿勢（work posture）の改善を目的と
図4･17中高年齢者適作業条件基準の例
あるいは無人化といったものは，できるところだけを自動化し，自
動化が技術的に困難な部分を人間が担当するといったケースが目立
つ．したがって，人間が担当する作業は重筋作業であったり，悪環
境下の作業であったりする．また，昨今の人間を中心とするセル生
産システムは，働く喜びは保証されるものの，身体の疾病やストレ
スの増大は否めない．そのような状況のもとで作業者の人間性と生
産性をいかに両立させるかは今後も課題として残る．
b.勤労者の高齢化 日本において，高齢化指標である65歳
以上人口比率は2005年までに世界一高くなり，2010年までにその
値は21 %を超え，世界で最も早く超高齢社会を迎える見通しであ
る．さらに2020年には4人に1人が，2050年には3人に1人が65
歳以上となる．生産に携わる人々もそれに準じて高齢化が進むのは
間違いない．年齢とともに低下する心身機能をいかにサポートし，
現在の生産性をいかに維持するかが大きな問題となる．そこでは高
齢者（elderly people）を弱者としてとらえるのではなく，高齢者
が積極的に参加できる場があって元気に豊かに働けるような生産工
場を指向する視点が必要である．
c .技能伝承 各企業において技能（スキル）の伝承が危ぶま
れて久しい．生産の無人化指向に加え，生産拠点の海外移転や高齢
者の増加と若年者の減少などがその問題に拍車をかけている．現
在，ディジタルマイスタープロジェクトが国家レベルで進められて
いるが，科学的な観点から熟練技能者のもつスキルの解明やその伝
承方法の確立を早期に達成する必要があろう．
