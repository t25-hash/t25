# 4・4・5 その他の新しい概念

> **章**: 第4章 自動化システム
> **節**: 4・4 新しい生産システム

これまで述べてきたもののほかに，下に示すように，新しい生産
システムの概念がいくつか提案されている．それらの多くは研究途
上にあるが，将来の生産システムのあり方として大変興味深いもの
である．
(１)バーチャルエンタプライズ
(２)インタラクティブ生産
(３)マルチエージェント生産システム
(４)トータルライフサイクル生産システム
(５)サプライチェーンネットワーク
(６)人工物ネットワーク
図4･23 は，4･4･4項で述べた生物指向型生産システムを社会
システムまで拡張したものであり，DNA型情報とBN型情報を埋
込んだ人工物で構成されるネットワークの概念図である．このよう
な人工物ネットワークでは，人工物と人間が局所的に相互作用しな
がら，人工物の設計・生産・運用・リサイクル・廃棄の過程に，い
つでもどこでも，誰でも参入・関与するシステムである．これは，
物質とエネルギーの循環システムに情報の循環という性質を加える
ことを意味することになる．将来においては，人工物の生産活動を
人間社会や経済システム，さらには自然循環システムと切り離した
孤立系として考えることは，意味をなさなくなり，図4･23のよう
なとらえ方がより重要になるであろう．ユビキタスコンピューティ
ングの技術開発と相まって実現されると期待される．
