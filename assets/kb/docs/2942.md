# 7・1・2 鋼管の外径寸法と肉厚

> **章**: 第7章 配管要素
> **節**: 7・1 管と配管

配管用鋼管の厚さの寸法体系は，アメリカ規格協会（ANSI）で
管の厚さを標準化したスケジュール方式におおむね従っている．こ
の方式が考え出された背景は，より多くの種類の厚さ系列の必要
性，その系列は使用条件にあった合理的なものであること，安全規
則の計算式を適用したものであること，および頻度の少ない寸法は
除けるようにしたこと，である．ANSIでは，管内圧力を，使
用温度における材料の許容応力を とし， /×1000をパラメ
ータとして厚さ系列とした．1000は整数化したもので， /
は，材料，温度，圧力の三つの要素を一つのパラメータで表せる
（バーロの式： /＝2 /；ここに ：厚さ， は外径）ためであ
る．厚さの許容差－12.5 %を考慮して2×1－0.125＝1. 75とし，
さらに腐れしろ0.1(in)として / ＝1 . 75－0.1 / としている．
ここで， ：最大使用圧力（psi）， ：使用温度における許容応力
（psi）， ：管の公称厚さ（in）．
7・1・3配管
配管（piping）は主として流体の輸送を目的として，家庭用の上
下水，ガス，電気配線から産業設備まで多用される．構成要素に
は，管（pipe)，管継手（pipe fitting)，バルブ（valve)，管フラン
ジ（pipe flanges ；ガスケットも含む），配管支持部材などがある．
これらを配管の計画図，設計図，系統図に示すための基本的な図示
記号はJIS Z8205で規定している．また，施設，装置などの配管
系の流体とその状況を明確にするため，流体の種類に応じた色分け
による識別（JIS Z9102）や，名札，バンドを用いて安全性確保
と取扱い作業誤りのないようにしている．
配管の接合面には流体の漏れ止めを目的とするガスケットが使用
される．ガスケットには，小さな力で大きな変形が得られ，接合面
になじみやすく，外力が除かれれば原形にもどりやすく，かつ保守
に伴う取替えがあるので，接合面からのはがれやすさも要求され
る．一般的には，ゴム，四フッ化エチレン樹脂，金属などが用いら
れる（3･5･2項参照）．

7・3弁およびコック
7・3・1弁の種類
弁は流体の流路を絞ることにより流れの開閉や，方向，圧力，流
量を制御するものである．弁（valve）およびコック（cock）はそ
の応用分野により，水圧用，油圧用，空気圧用（これらについて
は，機械工学便覧γ2編「流体機械」第5部 フルードパワーシス
テム参照），蒸気用などあり，その構造や材質，形状は異なる．こ
こでは代表的な水，油用の弁類として用いられる基本的な形式の弁
について示す．
a.玉形弁（globe valve) 玉形弁の構造を図Ⅰ-7･11(ａ)に示
す．弁体を弁棒のねじによって弁座に押付けて流路の開閉を行うも
ので，その特徴は流れの向きが弁の通過前後でかわるので，流体抵
抗が大きいが，小口径のものは廉価で，弁開度の小さな領域におい
て流量調節が可能である．図Ⅰ-7･11(ｂ)は流れの向きを直角に変
えるアングル弁の構造例である．
b.仕切弁（gate valve) 仕切弁は弁棒により，くさび状の弁
体を弁座の間に押込んで流路の開閉を行うもので，流れはまっすぐ
であるので流体抵抗が小さく，大口径になっても弁の操作力がそれ
ほど大きくならない利点がある．
c .コック（cock) コックは貫通した穴のあいた栓を軸の回
りに90°回転して流路の開閉を行うもので，弁本体の流出口を三
つ，四つ設けることにより，栓の開孔位置を変えることにより流路
を切替ることができる．高圧用のものでは，プラグ弁（plug
valve）と呼ばれ，外部から強制的にグリースを注入し気密性と潤
滑を行っている．図Ⅰ-7･12にコックの一例を示す．

7・4超高圧用配管と弁（high pressure piping and valve）
超高圧流体の利用分野には，材料加工（等方加工），機械加工
（静水圧押出し，ジェット加工，バルジ加工），多相平衡（気体水和
物，圧力晶析法），化学反応（気相反応），生物工学（生体高分子の
変性，酵素反応，滅菌），燃料噴射などがある．例えば，エチレン
重合では300MPa，機械加工では1GPa，ディーゼル噴霧形成で
は160MPaなどが実用されている．入手できるチューブの材質は
ステンレス鋼で，210～420MPa対応では304SS，316SSのもの
が市販されている．配管法は，図Ⅰ-7･17に示すように，専用のね
じと59°の円すい状の端末をもつポペット形状のチューブと，60°
の円すい状の座面をもつ受側本体の組合せからなり，チューブを本
体に取付けるためには，専用のカラーとグランド（保持器）でねじ
締めされる．グランドと本体のねじ部には専用のグリースを塗る．
配管は直線状に行い，継手により目的の方向に配管する．やむを得
ず曲げる場合には，曲率半径をチューブ外径の4倍以上とし，チュ

高圧流体の開閉には，図Ⅰ-7･18に見られるようにハンドルを回
してもボール支持機構によりポペット弁を回転せずシート部に押付
けられる構造のものが用いられる．
