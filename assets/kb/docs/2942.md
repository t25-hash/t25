# 7・1・2 鋼管の外径寸法と肉厚

> **章**: 第7章 配管要素
> **節**: 7・1 管と配管

配管用鋼管の厚さの寸法体系は、アメリカ規格協会(ANSI)で管の厚さを標準化したスケジュール方式におおむね従っている。この方式が考え出された背景は、より多くの種類の厚さ系列の必要性、その系列は使用条件にあった合理的なものであること、安全規則の計算式を適用したものであること、および頻度の少ない寸法は除けるようにしたこと、である。ANSIでは、管内圧力を、使用温度における材料の許容応力をとし、/×1000をパラメータとして厚さ系列とした。1000は整数化したもので、/は、材料、温度、圧力の三つの要素を一つのパラメータで表せる(バーロの式:/=2/;ここに:厚さ、は外径)ためである。厚さの許容差-12.5%を考慮して2×1-0.125=1.75とし、さらに腐れしろ0.1(in)として/=1.75-0.1/としている。ここで、:最大使用圧力(psi)、:使用温度における許容応力(psi)、:管の公称厚さ(in)。
