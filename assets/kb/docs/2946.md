# 7・2・3 メカニカル式管継手(くい込み式,パッキン式)

> **章**: 第7章 配管要素
> **節**: 7・2 管継手

各種一般配管のほか,油圧,空気圧に用いられる.くい込みスリーブによるメタルシール形のくい込み式管継手(bite type pipefitting)の構造例と面圧の概念図を図I-7・8に示す.スリーブはナットを締上げることにより管にくい込みメタルシールするもので4―140図-7・7 可動式管支え,引張り棒式管支えの例図-7・8 くい込み式管継手と面圧の概念図図-7・9管フランジ(面圧のモデル図参照),最近のものではスリーブを長くし,ナットの外側まで引出すことにより管の振動に強く耐久性のあるものが市販されている.
