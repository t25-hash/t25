# 4・5・4 家電の生産システム

> **章**: 第4章 自動化システム
> **節**: 4・5 事例（自動車，家電，半導体等）

a.生産システムの変遷 戦後の1960年代から現在に至るま
で，家庭電化製品の生産システムを以下のように大別することがで
きる．
(１)創生期（1960～1970年代）
人々の生活を豊かにする商品を大量に生産するために，1人1作
業での流れ作業によるベルトコンベヤ方式が主流となる．
(２)成長期（1980年代）
生産性の追求から，人手に替わって，ロボットが組立・検査を行
う自動化生産システムが急速に普及する．末期ごろより，多品種生
産への対応が要求され始める．
(３)成熟期（1990年代）
人々の生活が豊かになり，価値の多様化と商品の短命化が進む
中，本格的な多品種少量生産の時代に移行する．複数のモジュール
の組合せで効率的なもの作りを実現する，フレキシブル生産システ
ムが普及する．
(４)個別最適期（2000年代）
中国・韓国・台湾メーカの台頭やマーケットイン志向の商品展開
など，競争が激化する中，グローバルに市場が要求する高機能商品
を，高品質・低価格でタイムリーに供給できる生産システムが必要
となっている．
b.ものづくりの最適化
.セット（完成品）のものづくり 家庭電化製品は，大型
のテレビ・冷蔵庫，中型のDVDレコーダ・ステレオコンポ，小型
のムービー・ディジタルカメラなど多岐にわたっている．それぞれ
の，①生産数量，②商品サイズ・重量，③構成部品点数，④機種展
開数，⑤作業者能力などにより，最適な生産システムの追求がなさ
れている．図4･35～4･37にその一例を示す．
図4･35はセル生産方式の例（ワンマンセル生産方式-屋台方式）

置いて商品を組立てる．複雑な形状の部品やゴム系の柔軟な部品の
組立作業・調整作業，電気配線処理などの自動化が困難な作業にも

合もある．
