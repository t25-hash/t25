# 7・2・4 フランジ式管継手

> **章**: 第7章 配管要素
> **節**: 7・2 管継手

管フランジ(pipe flange)は図I-7・9に示すように,ガスケットをフランジにはさみ,ボルト・ナットで締付けて接続する管継手である.フランジの管への取付け方によって図I-7・10のように分類される.図I-7・10で(a)は弁や機器と一体に作られるもの,図I-7・10(c)には,ろう付けによるものも含まれる.管フランジの規格化されているものは図I-7・10(b)と図I-7・10(c)だけであるが,互換性に影響する寸法は呼び径・呼び圧力ごとに基準寸法として規格化されている.また管フランジは用途によって,一般用(JIS B 2210,2220,2238,2239,2240),水道用(JIS G 5527),油圧用(JIS B 2291),冷凍用(JIS B 8602),真空用(JIS B2290),舶用(JIS F 7804,7805)に分類することもできる.JIS の一般用管フランジの呼び圧力に対する流体の状態と最高使図-7・6可動式管支え,可動管支えおよび管つりの例(c) 差し込み溶接フランジ(b) 突合せ溶接フランジ(a) 一体フランジ(e) 遊動フランジ(d) ねじ込みフランジ図-7・10 管フランジの種類用圧力との関係(pressure-temperature rating)に関して,鉄鋼製管フランジについてはJIS B 2238,2239,銅合金製についてはJIS B 2240に規定されている.
