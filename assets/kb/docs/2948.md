# 7・3・1 弁の種類

> **章**: 第7章 配管要素
> **節**: 7・3 弁およびコック

弁は流体の流路を絞ることにより流れの開閉や,方向,圧力,流量を制御するものである.弁(valve)およびコック(cock)はその応用分野により,水圧用,油圧用,空気圧用(これらについては,機械工学便覧γ2編「流体機械」第5部フルードパワーシステム参照),蒸気用などあり,その構造や材質,形状は異なる.ここでは代表的な水,油用の弁類として用いられる基本的な形式の弁について示す.a. 玉形弁(globe valve)玉形弁の構造を図I-7・11(a)に示す.弁体を弁棒のねじによって弁座に押付けて流路の開閉を行うもので,その特徴は流れの向きが弁の通過前後でかわるので,流体抵抗が大きいが,小口径のものは廉価で,弁開度の小さな領域において流量調節が可能である.図I-7・11(b)は流れの向きを直角に変えるアングル弁の構造例である.b. 仕切弁(gate valve)仕切弁は弁棒により,くさび状の弁体を弁座の間に押込んで流路の開閉を行うもので,流れはまっすぐであるので流体抵抗が小さく,大口径になっても弁の操作力がそれほど大きくならない利点がある.c. コック(cock)コックは貫通した穴のあいた栓を軸の回りに90°回転して流路の開閉を行うもので,弁本体の流出口を三つ,四つ設けることにより,栓の開孔位置を変えることにより流路を切替ることができる.高圧用のものでは,プラグ弁(plugvalve)と呼ばれ,外部から強制的にグリースを注入し気密性と潤滑を行っている.図I-7・12にコックの一例を示す.d. ボール弁(ball valve)原理はコックと同じで球状の弁体に貫通孔を設け流路の開閉を行うもので,弁座を設けてグリースを不要としている.コンパクトで流体抵抗が少なく,水や油,エアコン用冷媒などの流路切換えに多用される.図I-7・13に構造例を示す.e. バタフライ弁(butterfly valve)円板状の弁体を90°回転(a)(b)図-7・11 玉形弁(a)とアングル弁(b)図-7・12 コックの一例図-7・13 ボール弁図-7・16 遠隔操作弁4―141第I部機械要素することにより,流路の開閉を行うもので,図I-7・14に構造図を示す.f. スイング逆止め弁(swing check valve)一方向の流れのみを許す弁で,弁体をちょうつがい運動する機構で支持し,その機能をはたしている.構造を図I-7・15に示す.g. 遠隔操作弁(remote control valve)上記の弁を遠隔操作するもので,図I-7・16(a)はモータ駆動の仕切弁,図I-7・16(b)は空気圧揺動アクチュエータ駆動のボール弁,図I-7・16(c)はソレノイド駆動による2ポート形電磁弁の例である.
