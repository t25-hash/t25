設計では，4･1･4項で取り上げるVE手法や，VR（variety
reduction，バラエティリダクション）と呼ばれる方法論，品種に
よる共通部分と変動部分を区分したモジュール設計，一方向組立や
作りやすい設計を織り込むDfM（製造容易性）設計や生産設計と
呼ばれる方法，また特殊材を安価な一般材で済ませるための材料開
作り方では，設計とリンクして加工費をダウンさせるための人と
機械の役割分担を含めた工程設計，また段取替えを容易にするよう
な金型・ジグ開発，さらに過去の故障履歴やその対策の情報に基づ
き設備の保全性・信頼性を設計時に作り込むMP（maintenance
prevention，メンテナンスプリベンション）設計と呼ばれる設備開
買い方では，部品メーカと連携したVE活動，さらには最近では
世界最適調達という旗印のもとに，インタネットを介した安価な資
総原価を半減するといったことが求められる今日では，もはや製
造段階だけのコストダウンには限界があり，製品開発の原価企画が
製品の競争力を決める．そのためには，設計，生産技術，購買，品
質保証等の部門横断的活動，そして部品メーカを含めた情報の共有
なお，原価企画の対象となるコストのおもな対象は製品原価であ
るが，これに加えて，顧客の立場からの製品ライフサイクルコスト
（LCC:life cycle cost ）も考慮した設計が重要である．LCCとは，
製品の取得コスト（イニシャルコスト）に，製品を使用する際の電
気代等のランニングコスト，保守のためのメンテナンスコスト，さ
らに寿命がきたときの廃棄コスト等のトータルのコストである．特
に産業財の場合は，LCCの尺度は重要な製品の品質を表す客観的
同時に環境への負荷という面からLCA（life cycle assessment ）
評価も，金額による評価は困難としても，インベントリーデータに
基づくCO等排出量等の尺度による評価とその軽減のための設計
が不可欠である（3･6節参照）．またリサイクルしやすい設計のた
めのDfE（design for environment ）を考慮する必要性があること
はいうまでもない（3･3節参照）．
