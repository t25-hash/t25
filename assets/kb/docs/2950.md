# 1・1・1 接触面(contact surface)の機能

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・1 接触面の機能と発生する事象

トライボ機械要素は、互いに力を及ぼし合いながら相対運動(静止も含む)を行う接触面を利用して、次のように四つに大別される要求機能を果たすことが求められる。(1) 所定の自由度の運動を滑らかに実現するよう案内する。機械装置に回転運動や直線運動など目的の運動をさせる場合、それに対する抵抗力(摩擦力)はできるだけ小さいことが望ましい。滑り軸受の軸と軸受面、転がり軸受の玉やころと軌道輪の接触面、工作機械の滑り案内面などがこれに該当する。(2) 所定の自由度以外の運動あるいは所定の可動範囲を越えた運動を確実に抑制するように拘束する。例えば回転軸が外力を受けて半径方向に振動的に変位することがあれば、回転機械の運動部分が静止部と接触して損傷が発生するおそれがあるので、軸受面はそのような軸の動きを適切に抑制する力を軸に伝達し、その反作用をしっかり受け止めることが求められる。(3) 必要な力を伝達する。歯車の歯面、クラッチ面、トラクションドライブ機構の摩擦面は、小型でできるだけ大きな力を確実に伝達することが求められる。伝達する力は、歯車の場合は主として表面に垂直な法線力、そのほかは表面に沿う接線力となる。(4) 接触面間のすきまを通過して固体、液体、気体が内部から漏えい、あるいは外部から侵入しようとするのを防止し、あるいはその量を制御する。コンプレッサ、ポンプで昇圧された気体や液体、あるいはガスタービン、蒸気タービンに導入された高圧の燃焼ガスや蒸気が、所定の流路ではない回転部と静止部の狭いすきまを通って漏出しないように、各種のシールが設けられる。また、自動車用往復動エンジンでは、燃焼行程に応じて燃焼室を密閉したり、混合気や排気ガスの吸入、排出を制御するバルブとバルブシートの接触面がある。さらに、ピストンとシリンダの円環状のすきまを通って高圧の燃焼ガスが漏出しないように設けたピストンリングとシリンダライナの接触面がある。そのほか、しゅう動面に外部から異物が入らないようにするダストシール、しゅう動面から潤滑油、グリースなどが外部へ漏出しないようにするオイルシールなどがある。
