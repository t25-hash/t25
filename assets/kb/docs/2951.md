# 4・1・5 VRP（variety reduction program）

> **章**: 第4章 設計の管理
> **節**: 4・1 コストの管理

顧客や得意先の要求により，そのままでは製品やそれを構成する
部品の数は増加の一途をたどる．そうすると製品原価の中で，特に
取扱う部品や材料の点数に伴って発生する間接業務（設計，発注や
品質管理）や多工程化に関するコストが飛躍的に増加する．このよ
うな多品種・多部品・多工程に着眼したコスト削減技法として
VRP がある．
VRPは，表4･3に示すような視点に基づき，顧客の要求を満た
しながら部品数や製品・設備（工程）数を削減し，それに関わる製
品コストを削減するものである．また流用設計を促進することにも
なり，設計の効率化や開発リードタイム短縮にもつながる．現状の
多品種・多部品・多工程の問題を解決するとともに，製品開発時に
おける原価を作り込むチェックリストとして必須事項であるといっ
ても過言ではない．

要である．
