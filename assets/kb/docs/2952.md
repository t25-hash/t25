# 1・1・1 項に述べた機能を実現し,性能と寿命を達成するように接

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・1 接触面の機能と発生する事象

触面を合理的に設計するには,接触面そのものの性質とそこにおいて発生するいろいろな事象の実態と発生のメカニズム,影響する因子を明らかにして,精度の高い予測モデルを確立しておくことが必要であり,ここにトライボロジー研究の意味がある.高分子材料など,目的に応じて物理的,化学的性質の異なる多様な材料で構成する.しかし,表面に存在するのは母材の材料組成の物質そのものではなく,大気環境下では酸化物の被膜が形成されていたり,酸素などの気体分子,水などの液体分子,その他の物質が付着しているのが常態である.また,性能,寿命の改善のため,積極的に各種流体の潤滑膜や硬質あるいは軟質の固体被膜を表面に形成させる場合もある.形状も,マクロ的には平面,球面,円筒面,円すい面など多様であるが,いずれも平滑な面を目指して加工されることが多い.しかし,加工精度の限界によりミクロ的には表面にある程度の粗さやうねりが残留している.一方で,トライボロジー性能,寿命の向上を目的として,マクロレベル,ミクロレベルのさまざまな表面形状が創成される.これら表面のトポグラフィーと性状の同定はトライボロジー解析の出発点である.4―143
