# 1・1・3 固体接触(solid contact)

> **章**: 第4章 で述べるように,表面は各種金属のほか,セラミックス,
> **節**: 1・1 接触面の機能と発生する事象

詳細を1・3・1~1・3・3項に述べるような,以下の二つの固体表面の接触問題がある.マクロに見ると,幾何学的に点接触(球面どうしあるいは球面と平面など)あるいは線接触(円筒面どうしあるいは円筒面と平面など)となるような平滑面の接触状態は集中接触状態にあり,ヘルツ接触モデルで表され,接触領域の応力は平滑な平面どうしが接触する場合に比較してはるかに大きく,局所的な弾性変形が生じる.平滑な平面どうしの接触ではマクロな見掛けの応力はずっと小さいが,ミクロに見れば,固体表面が法線荷重の作用で互いに接近すると表面粗さの突起どうしの直接接触が始まり,接触部およびその周辺に弾性変形や塑性変形が生じる.硬脆材料の場合ではマクロな変形をほとんど伴わないで,き裂が発生,進展する.応力が大きいといずれも破壊が始まる.破壊により露出した新生面は化学的に活性度が高く,このため酸化膜などで覆われた表面とは異なるトライボロジー的挙動を呈する.また破壊により,接触面から材料の微細片が脱落,飛散する発塵現象はクリーン環境の汚染源となることもある.したがって,各種荷重下での接触と接触力,変形と破壊の挙動は接触面に要求される機能と寿命の維持に大きな影響を与えるので,表面近傍の接触現象のモデル化,応力分布の評価,き裂の発生と進展の評価,破壊のモデル化などの研究が重要となる.
