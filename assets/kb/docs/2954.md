# 1・1・4 摩擦と表面損傷(surface damage)

> **章**: 第4章 で述べるように,表面は各種金属のほか,セラミックス,
> **節**: 1・1 接触面の機能と発生する事象

その次に問題となるのが相対運動による摩擦と表面損傷の発生であり,詳細は1・3・4~1・3・6項,第3章で述べる.接触面は相対運動せずに互いに静止していることもあるが,多くは面に沿う滑り運動や転がり運動,あるいは面に垂直な衝突運動を相互に行い,その過程で法線力や接線力を互いに及ぼし合う.このため,接触面で生じた直接接触部がこれらの力の作用により凝着し,さらに相対運動によってせん断破壊が生じる.あるいは硬いほうの突起が軟らかいほうの面に食い込んで掘り起こしながら移動することもある.このような現象により,相対運動の方向と逆向きの抵抗力,すなわち摩擦力が発生し,大気環境下の乾燥した表面では,摩擦係数が0.3~1程度の値になることが知られている.さらに摩擦によって表面が損傷し,せん断された凝着部が脱落して摩耗粉が発生する.相対運動の速度が大きい場合は,摩擦や表面損傷による表面の温度上昇が大きくなり,酸化物の生成が促進されたり,表面の材料強度が低下して過大な変形が生じる.また結晶レベルでの変質が生じて,表面形状が変化したり,表面の物理化学的性質が当初の設定値から外れてしまうこともある.温度がさらに上昇して材料の融点を越えると,局所的あるいは全面的な融解が生じて焼付きが生じる場合がある.接触面に高い圧力が繰返して作用する場合には,疲労破壊が生じて表面近傍部分がはく離,脱落する.このような状態になると,接触面はもはや本来の機能を果たせなくなる.トライボ機械要素の設計は,表面の機能と寿命の維持が目標であるから,この摩擦抵抗と接触面の表面損傷をいかに制御するかが設計の要点となる.
