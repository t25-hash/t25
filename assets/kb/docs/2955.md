# 1・1・5 潤滑と潤滑モード(lubrication mode)

> **章**: 第4章 で述べるように、表面は各種金属のほか、セラミックス、
> **節**: 1・1 接触面の機能と発生する事象

所定の運動を妨げる向きに作用する摩擦抵抗力はエネルギーをむだに消費したり、位置決め精度を狂わせるので、多くの機械装置では摩擦抵抗ができるだけ小さくなるように設計したい。二つの対向する固体壁面が互いに直接接触した状態で滑ったり転がったりすることが原因で摩擦抵抗が発生するのであれば、2面間に摩擦抵抗の小さい第三の物質を介在させ、固体表面どうしができるだけ直接接触しないようにすればよい。これが潤滑作用であり、この第三の物質を潤滑剤と呼ぶ。潤滑剤の詳細については第2章で述べる。潤滑には1・4,1・5節で述べるようにさまざまな形態(モード)があり、それに応じて潤滑剤の役割も異なる。固体表面の粗さに比して十分な厚さの流体の潤滑膜が形成される流体潤滑モードでは、レイノルズ方程式によって規定される流体の圧力が潤滑膜内に発生し、これが固体表面を接触させようとする法線荷重に対向して面どうしの直接接触を阻むところに特徴がある。摩擦抵抗は厚い流体潤滑膜のせん断抵抗となるので、固体どうしが直接接触する摩擦に比較して2桁あるいは3桁小さい摩擦係数を実現することが可能となる。さまざまな流体の性質の中で流体潤滑モードで支配的となる因子は粘性係数である。この粘性係数は潤滑剤の構成分子によって大きく異なり、また温度や圧力によっても変動する。荷重や速度の条件が厳しくなると、流体潤滑膜は全体として保持できているものの、一部固体接触が存在する混合潤滑モードに移行し、固体どうしの接触、摩擦の要素を加味しなくてはならない。条件がさらに厳しくなって潤滑膜がさらに薄くなり、表面間に存在する潤滑膜が単分子あるいは数分子程度の厚さになると境界潤滑モードとなる。このモードでは潤滑剤の粘性係数は支配的な性質ではなくなって、それ以外の物理化学的な性質の影響が強くなる。すなわち、せいぜい数分子層の極薄の潤滑膜が法線力や接線力に耐えて潤滑膜を維持できるか否かは、潤滑剤分子が固体の表面に付着する強さと潤滑剤分子どうしの凝集力の強さによって決まる。この強さは潤滑剤分子の構成と分子の長さ、潤滑剤分子と固体材料との相性にも左右され、また環境温度が高くなると強さが低下することが知られている。法線荷重が一定の場合、この極薄潤滑膜の強さが大きいほど潤滑膜が負担する荷重の割合が大きく、摩擦面全体に占めるその面積割合も大きくなる。その分、固体表面が直接接触する面積割合も小さくなる。境界潤滑モードでの摩擦抵抗は、極薄潤滑膜が保持される部分では分子の長さによって左右され、固体の直接接触により荷重を支える面積では乾燥時の固体摩擦と同様になる。高温環境、高真空環境では流体の潤滑剤が使用できず、せん断摩擦抵抗の小さいさまざまな固体の潤滑剤あるいは自己潤滑性のある表面材料や被膜が使用され、固体潤滑モードとなる。潤滑の作用は摩擦抵抗の低減だけでなく、表面の損傷も軽減する。流体潤滑モードでは、厚い潤滑膜が固体の直接接触を阻んで損傷の主原因を取り除き、また潤滑面が広い場合は荷重の圧力を油膜を介して広く分散させることができるので、高い応力が狭い領域に集中せず材料破壊の確率を低減させる。さらに流体摩擦抵抗は小さいので発生する摩擦熱が小さくなるだけでなく、潤滑剤自身が摩擦損失熱の大部分を吸収して摩擦面外へ運び出すので、面の温度上昇が抑制され、材料強度が低下しにくい。固体接触が一部生じる混合潤滑、境界潤滑のモードでは、発生した摩耗粉の表面を潤滑剤が覆ってその表面エネルギーを下げ、摩耗粉が合体、成長してさらに表面の損傷を拡大することを阻むことができる。しかし、潤滑剤に含まれる添加物の成分と材料が反応して材料表面が徐々に腐食される腐食摩耗、潤滑膜内の負圧によって発生するキャビティの崩壊が原因となるキャビテーションエロージョンなど、潤滑剤の存在が原因となって発生する損傷もある。
