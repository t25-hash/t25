# 1・2・1 トライボ設計(tribo-design)と潤滑モード

> **章**: 第4章 で述べるように,表面は各種金属のほか,セラミックス,
> **節**: 1・2 トライボ設計

トライボ設計とは,要求されるトライボ機能(1・1・1項参照)を実現し,さらに要求性能と要求寿命を達成できる設計解を決定することである.また,近年では使用中のメンテナンスの容易さ,あるいはメンテナンスフリーの達成も設計に際して考慮される.設計には,今までにないものを創り出す新設計,従来の設計解の中から最も適合するものを選んで使用する選定,従来の設計解に幾分かの修正を施す改良設計の三つがある.また,特定の性能を最大にすることを目的とする最適設計がある.トライボ設計解の探索は,接触面の基本的形状,相対運動の形態と速度,接触面が受ける法線荷重や接線荷重,最大寸法,使用環境などが制約条件として与えられたもとで行われるのが通例である.これらの与条件によって接触面に可能な潤滑モード(1・1・5項参照)が規定されると,トライボ設計の大略が方向づけられる.
