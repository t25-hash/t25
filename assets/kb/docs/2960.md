# 1・3・1 表面形状モデル(surface profile model)

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・3 固体接触論(solid contact theory)

固体表面の接触を議論する場合、平面や円筒面、球面などのマクロな表面形状の上に存在するミクロな表面凹凸を扱う必要がある。機械加工などによって作られるこれら微細な形状は、その生成過程に応じて周期的様相とランダムな様相を呈する。ダイヤモンド旋削面などを除けば、周期性を有する面であっても完全な周期性を満足することは少なく、これらは通常、統計的な扱いによって特性が表現される。実際に加工された面を測定し、その特性を求める立場からは、高さパラメータ、傾斜や異方性パラメータ、頂点や山の間隔パラメータ、頂点や谷底パラメータ、高さ分布の特性パラメータなどが断面曲線より算出され、トライボロジー特性の評価に際して表面の特徴を表現する手段として使われる。また、近年計測技術や解析手段の進歩に支えられ、面領域のパラメータ、特に面領域の頂点や窪底、鞍点、稜線、およびそれらの分布などが議論されている。さらに、断面曲線から求める傾斜パラメータと面領域データから求めるそれとの間には、傾斜のとらえ方の違いから、特に等方的表面において大きな隔たりが存在することは注意する必要がある。また、測定手段の違いが、同じ面を測定しても、異なった結果を生む問題点も指摘されている。現状では、安定性と原理の単純さから、引続き触針法が主流であるが、面領域への展開やそれに伴う測定時間の短縮、非接触測定の要求などから光学式測定法が種々提案・使用されている。さらに、原子レベルの測定や横分解能の飛躍的向上を目指して原子間力顕微鏡やそれを応用した種々の機能測定装置が開発されている。一方、トライボロジーの解析的研究などにおいては、表面の微細形状を数学的にモデル化し、接触などのトライボロジー特性解析を行う場合がある。これらにおいては、表面微細形状を生成するいくつかのモデルが提案されている。固体表面の形状は、歴史的には確率過程論や自己回帰・移動4―145平均モデルなどが提案され、実形状と比較・使用されてきた。これらのモデルは、表面の機能と結び付けられる傾斜や先端曲率などをモデルのパラメータで導出できる点が魅力的であった。しかしながら、実形状とそれより同定されたパラメータを用いたモデルとの乖離が、特に突起頂点の分布やその近傍の曲率などにおいて無視できず、数学的整合性とは裏腹に、機能と表面モデルとの関連の研究は十分完成されたものといえない現状である。近年、この延長上の研究として、図II-1・1に示すように高さ分布の非対称性や表面形状の方向性をも考慮できる三次元モデルの提案がなされており、モデルパラメータを増やすことによって、要求の一端に対応しようとしている。一方、1980年代から、フラクタル幾何学が表面形状の評価・推定モデルとして適用されてきた。自己類似フラクタルの適用は、表面形状を科学的に取扱う視点を導入し、フラクタル次元が表面の濡れ性などと関連づけられ、その評価指標として有効性が示されている。また、逆にフラクタルブラウン運動を用いて表面の形状モデルを創成することも可能であり、任意の特性をもった形状モデルの作成が可能となっている。機能表面、特にトライボロジーが関与する表面においては、表面形状の一部が消失されたなじみ摩耗面やプラトーホーニング面のような複層構造表面が重要となっている。このような表面では、2種類以上のプロセスに基づく情報が混在しており、その評価・解析には高さの累積分布(負荷曲線)とその統計処理が用いられる。
