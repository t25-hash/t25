# 1・3・3 粗面の接触モデル

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・3 固体接触論(solid contact theory)

図II-1・6に示すように、粗さをもつランダムな粗面と真平面とが干渉すると、突起に変形が生じる。そのような突起の塑性変図-1・5 接触形態4―147第II部トライボロジー形のしやすさの指標として用いられるのが塑性指数ψであり、次式で与えられる。1ψ<0.6のときは、ほとんどの突起が弾性接触である。2ψ>3のときは、ほとんどの突起が塑性接触である。その中間のψ=0.6~3の範囲では、弾性接触をする突起と塑性接触をする突起の混在や、を越える突起に対する弾塑性変形の考慮が必要になる。この場合の全真実接触面積と押付け荷重は、弾性接触ならびに弾塑性接触を考慮して次のようになる。傾きは、表面粗さ曲線上の各点での、高さと傾きの二乗平均平方根の実測値(σ=,Δ=)から、式(II-1・24)を満足する値として数値的に求めることができる。図II-1・9は、押付け荷重と2面の平均すきまの関係である。ところで、以上の解析の基礎となる表面粗さ曲線は、測定機器の分解能によって異なり、突起の傾斜や先端半径にはこの影響が現れる。この問題点の解決のために表面のフラクタル性を考慮した解析を行い、フラクタル次元による真実接触面積の違いを明らかにした例もある。
