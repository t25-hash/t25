# 1・3・5 摩耗理論(wear theory)

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・3 固体接触論(solid contact theory)

摩耗、すなわち固体の表面を摩擦することによってその一部が逐次減量していく現象は、自然現象の中でも理論的に数式で表すことが難しいものの一つである。摩耗が物理的に確かな根拠のうえに定義された現象でなく、物体表面の経時変化を漠然と表す概念にすぎないのがその原因である。それでも、摩耗の予測を得るために、摩耗機構の比較的明らかな二つの摩耗、凝着摩耗(adhesive wear)とアブレシブ摩耗(abrasive wear)について、理論的に摩耗量を求める基本式が提案されている。摩耗の理論式として、最初に提案されたのはホルム(Holm)の摩耗式である。ホルムの摩耗式で重要なことは、真実接触部で真に付着する現象を基とした凝着摩耗が起こること、そして、摩耗量が荷重、摩擦距離、あるいは材料の硬さに影響されるという摩耗式の原型を得た点にある。本来、凝着摩耗に重要なはずの表面の相互作用や表面化学作用は、すべて確率という因子に押込められている。一方、摩耗がホルムのいうような原子の捕獲ととらえるのは現実的でないという考えから、真実接触部の破断によってジャンクションに相当する大きさの粒子が取り去られるとして理論式を提案したのがアーチャード(Archard)あるいはラビノヴィッツ(Rabinowicz)である。ここで、はジャンクションの破断によって小片が取り去られる確率であり、ホルムの式で、=3としたものと一致する。摩耗量を理論的に導くには、真実接触部の破断によって生じる小片が摩擦面から脱落するまでの過程をも含めた摩耗式が必要となる。これについては笹田がモデル(移着成長過程)を提案しているが、理論式までには至っていない。この場合、移着した粒子の脱落を決めるのは固体表面と雰囲気物質との物理的、化学的作用である。潤滑下の凝着摩耗では、界面に介在する潤滑剤が真実接触面積を変化させ、同時に、それらのもつ表面化学作用が摩耗に影響を及ぼす。摩耗式では、これらの作用がすべて比摩耗量に集約された形になっている。式(II-1・32)(II-1・33)以降、新たな摩耗式は提案されていない。この間、摩耗理論としてはの中身の解明が進められてきたといえる。凝着摩耗のがとる値の範囲は10~10mm/N と広く、実に10に達する。例えば潤滑状態では10~10mm/N、無潤滑では10~10mm/N である。摩耗の経時変化として、ある摩擦距離で比摩耗量が突然変化するシビア・マイルド遷移など複雑な要素の解析が現在なされている。また、2面間で凝着を生じないほど潤滑状態がよい場合では、表面が疲労はく離する現象がある。クラゲルスキー(Kraghelsky)が疲労はく離を起こす繰返し数を求め、スー(Suh)はデラミネーション理論を提案しているが、これもまた、摩擦面の破壊機構の一つのモデルであり、限られた摩擦条件下の摩耗の原因でもある。摩擦している表面の突起あるいは摩擦面に介在している硬い砥粒が表面を削り取ることによって起こる摩耗がアブレシブ摩耗であり、摩耗機構の説明ではいちばん古い考えである。まとめられた理論の中ではラビノヴィッツの押込みモデルの摩耗式がよく知られている。頂角2θの円すい状の剛体突起が荷重で硬さの表面に押込まれ荷重を支え得るまで塑性変形する。ところで、この摩耗式の摩耗量に注目してみると、は剛体突起によって押しのけられた体積を意味し、摩擦系外に排出される摩耗体積を表していない。つまり削ることによって摩耗する機構とは一線を画している。ラビノヴィッツより数年早く、削るというモデルからア4―150機械要素・トライボロジーブレシブ摩耗の摩耗式を立てたのが笹田である。セドリック(Cedric)もほぼ同様の考えを提案している。潤滑などによってすくい面の摩擦係数が減少すると、切込み量が増えて摩耗は増大する。その比例定数である比摩耗量にそれぞれの摩耗機構を特徴づける値が入る。凝着摩耗では表面の相互作用や摩耗粉の脱落に影響する複雑な作用を考えなければならないのに対し、アブレシブ摩耗では、表面突起あるいは砥粒の寸法や先端半径、切込み量などおもに幾何学的形状にかかわる因子が含まれる。一般に、アブレシブ摩耗の比摩耗量は=10~10mm/N の範囲の値をとる。この値は、式(II-1・34)(II-1・35)の幾何学的因子から計算した値とほぼ等しい。この意味でも、アブレシブ摩耗は凝着摩耗に比べてその機構がより明快である。
