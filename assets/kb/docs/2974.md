# 1・4・9 表面粗さ(surface roughness)を考慮した流体潤滑理

論潤滑面間のすきまが表面粗さの程度になったときは、両面の粗さの中心線を基準にとったすきま(公称すきま)および粗さによって生じる圧力変動の期待値(平均値)で表した統計的潤滑方程式が必要になる。この式は、解析的に求められた粗さの修正係数の式に表面粗さの高さの統計分布(確率密度関数)を代入すればよいので最も簡便な式であるが、粗さの方向が運動面の方向、もしくはそれと直角方向の、一方向だけに凹凸のある筋状の粗さ(一次元粗さ)に限定される。また粗面が静止面、運動面が真平面の組合せでなくてはならない。その考え方を一般的な三次元表面形状(二次元粗さ)に拡張し、さらに両面に粗さがある場合には面の相対滑り運動によって局所すき間厚さが時間的に変動するのでスクイーズ膜作用の影響をも考慮すると、一般性のあるパティア・チェンの修正レイノルズ方程式が次式のように得られ、この理論は平均流モデル(average flow model)と呼ばれる。一方、トリップの修正係数は、一次元粗さに適用されたエルロッドの摂動法を二次元粗さにまで拡張することによって求められ、次式のようにいっそう簡便な形で表される。この摂動法は粗さの振幅が小さい場合にしか適用できないが、より大きな粗さ振幅の場合にも適用できるように摂動パラメータを変え、グリーン関数の代わりに数値解析を用いて半解析的手法により求めたブッシュとヒューズの修正係数は、φについてはトリップの解に近く、また、Φについては、<3でパティアとチェンの解に近い。しかし、この修正係数は式あるいは数表の形で与えられていない。パティアとチェンの修正係数は、一次元粗さ、すなわち、γ=0(直交粗さ)とγ=∞(平行粗さ)の場合はクリステンセンの結果と一致し、表II-1・3のようになる。また文献(99)には、なじみ面に見られる頂部が平坦になった粗さを、単純な台形モデルで表した場合の修正係数が示されている。磁気ディスクの浮動ヘッドなどのように、サブミクロンのすきまで空気潤滑される場合は、流体の圧縮性と第I部3・2・5項に示されるように気体分子の平均自由行程の影響、すなわち固体壁面でのスリップ流れを考慮する必要がある。その場合の修正係数は、粗さ突起の局所スケールでは気体の密度変化が小さいと仮定して、希薄気体潤滑の修正レイノルズ方程式と粗さに関するトリップの摂動解法を用いることにより、次の形で求められている。
