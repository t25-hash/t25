# 1・5・1 潤滑モード

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・5 混合潤滑、境界潤滑

そのとき、固体接触して凝着した部分のせん断応力は流体膜のそれに比べて桁違いに大きいので、全体の摩擦係数が急激に上昇する。軸受特性数がさらに減少すると固体接触部の負荷分担ならびに摩擦力が支配的になり、摩擦係数は飽和値に達する。このような摩擦係数と軸受特性数の関係を図示すると図II-1・40のようになり、その曲線はストライベック曲線(Stribeck curve)と呼ばれる。図中の摩擦係数が飽和値に達した領域を境界潤滑(boundary lubrication)、大きく変化する領域を混合潤滑(mixed lubrication)と呼び、右側の低い摩擦係数の領域が流体潤滑となる。このように潤滑状態には三つのモードがある。
