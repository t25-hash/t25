# 1・5・2 接触モデル

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・5 混合潤滑、境界潤滑

図II-1・41は、境界潤滑状態におけるバウデンとテーバーの接触図-1・41 バウデンとテーバーの境界潤滑モデルモデルである。このモデルは、単分子膜が一様に吸着した真平面どうしが対向するハーディのモデルと異なり、荷重を実際にささえる真実接触部の面積は、単分子吸着膜の一部が突起先端部により破断して固体壁面どうしが直接接触している部分αと吸着膜に被覆された(1-α)とから成る。この2面が相対運動するとき、粗さの谷間の流体膜内に発生する動力学的な圧力による負荷分担率が、全体の摩擦係数を低減するほど大きくなる潤滑モードが混合潤滑であり、このような影響が小さい潤滑モードが境界潤滑である。
