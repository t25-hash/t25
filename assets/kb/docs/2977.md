# 1・5・3 境界膜

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・5 混合潤滑、境界潤滑

が小さくなるためには、まず式(II-1・127)で示したように強度の弱い層(境界膜)をある程度厚くして被覆率を増やすことであるが、被覆率が100%になった後はその膜が厚すぎると真実接触面積が増加するのでその膜が破断しない程度の厚さを有し、かつ接触圧力降伏領域(静水圧の死水域半球の外側)の深さに比べて十分薄いことが必要である。アルコール、脂肪酸、アミン、アミド、エステルなどの極性化合物は低濃度で溶解していても単分子吸着膜を形成し、炭素数が14以上であればその吸着膜の膜厚はナノメートルオーダであり、上記の条件を満たす。潤滑基油の粘度には影響しない程度の低濃度で溶解している分子が固体表面に吸着することによって摩擦係数を下げる性質を油性(oiliness)といい、油性をもつ物質を油性向上剤(oilinessagent)という。固体表面(金属の場合は一般に酸化膜)への有機分子の吸着には、分子間力(ファンデルワールス力)による物理吸着(physical adsorption,physisorption)と電子の授受または共有の化学結合による化学吸着(chemical adsorption,chemisorp-tion)がある。物理吸着の吸着熱は、吸着する固体材料の種類によらず2×10J/mol程度と小さく、吸着する固体材料の種類によらない。一方、化学吸着の吸着熱は4×10J/mol以上と大きいが、相手材料の種類に依存して異なる。いずれにしても、吸着は発熱反応なので温度が上昇すると表面から吸着した分子の離脱が起こり、潤滑性を失う。この離脱温度を転移温度(transition tempera-ture)と呼ぶ。一方、摩擦による発熱と高圧によって化合物が分解し、その一部が固体表面と化学反応して表面に硫化物、塩化物、りん酸塩などの反応膜を形成し、その厚さが上述の適度の薄さであれば摩擦係数あるいは摩耗率を低減させる。そのような反応膜を極圧膜(extremepressure film)といい、極圧膜を形成する物質を極圧添加剤(extreme pressure agent)という。このように、摩擦係数を減少させ、場合によっては摩耗も低減させる吸着分子膜や反応膜を境界膜(boundary film)と呼ぶ。酸化膜やわずかなさびなども境界膜の一種と考えることができる。摩耗防止剤兼酸化防止剤としてエンジンオイルなどに広く用いられているZDTP(ジアルキルジチオりん酸亜鉛)は極圧剤と油性向上剤の両方の効果、すなわち無機の反応膜(FeS,ZnO,ZnS)の上にガラス状のりん酸重合体層(摩擦に起因する反応によって生成される重合体をフリクションポリマと呼ぶ)、さらにその上にアルキルりん酸の吸着膜を形成することが知られている。
