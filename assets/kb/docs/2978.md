# 1・5・4 有機吸着分子膜のレオロジー特性

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・5 混合潤滑、境界潤滑

固体表面に吸着した鎖状の分子の濃度が飽和すると分子は配向し、隣り合った鎖状分子間に凝集力が働き、固体的な変形特性をもつ。しかし、吸着分子の層が厚くなると、固体表面から遠い分子層ほど配向は弱まり、液体的変形特性をもつようになるが、吸着膜に加わる圧力が増すことによって固体的になり、せん断強度は増加する。ステアリン酸塩のせん断強度τと圧力の関係がτ=τ+となることは実験的に示されている(τ,は常数)。摩擦係数はτであるから=τ+(II-1・128)となり、圧力がτに比べて十分高ければ摩擦係数は一定であるが、低くなると増加する。温度が低ければ吸着膜は固体的であり、温度が上昇するにつれてアモルファスから液体的に相変化する。いずれの状態でも吸着膜は粘弾性体と考えられるので、液体的であってもひずみ速度が増加するにつれてアモルファス状態、固体的へと変化する。図II-1・42では、この変形特性が摩擦力と滑り速度によってマッピングされている。また文献(108)では、アイリングの反応速度論に基づいて、吸着膜のせん断強度と温度、圧力、速度の関係式が提示されている。
