# 1・5・6 混合潤滑理論(mixed lubrication theory)

> **章**: 第1章 トライボロジーの基礎
> **節**: 1・5 混合潤滑、境界潤滑

真実接触面積は両面の距離すなわち油膜厚さが決まれば表面の幾何学的形状と機械的性質から決定されるので、結局、混合潤滑理論は、(1) すきまと表面粗さ形状と機械的性質(弾性係数、ポアソン比、硬さ)から真実接触面積および接触荷重を決める1・3・3項の接触理論式と、(2) 混合潤滑状態で流体膜圧力による分担荷重と油膜厚さを決める1・4・9項の表面粗さを考慮した流体潤滑理論式に、(3) 流体膜圧力による負荷能力と固体接触部圧力の分担荷重の合計が負荷に等しいという荷重の釣合い式、を加えた連立方程式の形で表される。なお、混合潤滑の摩擦係数を算出するためには、固体接触部の境界摩擦係数を仮定する必要がある。文献(1)ISO TC 213が制定する一連のGeometrical ProductSpecifications (GPS)規格群。(2) 例えば、 笹島和幸・ほか3名、 1999年度精密工学会秋季大会学術講演会講演論文集、K 06(1999)、429.(3) 例えば、Nayak,P.R.,Trans.
