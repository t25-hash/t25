# 2・1・1 潤滑剤の種類

> **章**: 第2章 潤滑剤
> **節**: 2・1 潤滑剤の種類と選択

潤滑剤(lubricant)は液状の潤滑剤(潤滑油)、半固体状のグリース、および固体潤滑剤に大別される。これらの詳細については、それぞれ2・2~2・4節を参照されたい。一般には液状の潤滑剤が広く用いられているが、潤滑方法にスペースがとれない、メンテナンスが難しい、水や塵埃(じんあい)からのシール性が要求される、など潤滑油の使用が制約される場合には、グリースが用いられる。グリースは密封使用が容易なことから、転がり軸受に広く使用されている。また真空中や高温環境など、潤滑油・グリースの使用が難しい場合には、二硫化モリブデンやグラファイトで代表される固体潤滑剤が使用される。液状の潤滑剤には、使用されている基材の種類により鉱油系潤滑油、合成系潤滑油、および水系潤滑剤がある。これらのうち、鉱油系および合成系潤滑油には、使用される基油や添加剤の配合方法により、2・2・2項に述べるように、自動車用潤滑油、工業用潤滑油など使用目的別種類がある。一方、難燃性に優れた水溶性潤滑剤やエマルションなどの潤滑剤が水系潤滑剤と呼ばれ、火災に対する配慮が必要な環境での潤滑や、優れた冷却性能が必要とされる潤滑など、目的に応じて使用されている。半固体状の潤滑剤であるグリースも、配合される増ちょう剤、基油、添加剤の組合せで、自動車用、電機・情報機器用、産業用電動機用など多くの種類がある。これらグリースの種類については2・固体潤滑剤としては、二硫化モリブデンなどの無機化合物系、金や銀などの金属系、およびポリテトラフルオロエチレンなどの有機化合物系に分類される。これら固体潤滑剤の種類については2・4・1項を参照されたい。
