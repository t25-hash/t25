# 2・1・1 潤滑剤の種類

> **章**: 第2章 潤滑剤
> **節**: 2・1 潤滑剤の種類と選択

潤滑剤（lubricant）は液状の潤滑剤（潤滑油），半固体状のグリ
ース，および固体潤滑剤に大別される．これらの詳細については，
それぞれ2･2～2･4節を参照されたい．一般には液状の潤滑剤が広
く用いられているが，潤滑方法にスペースがとれない，メンテナン
スが難しい，水や塵埃（じんあい）からのシール性が要求される，
など潤滑油の使用が制約される場合には，グリースが用いられる．
グリースは密封使用が容易なことから，転がり軸受に広く使用され
ている．また真空中や高温環境など，潤滑油・グリースの使用が難
しい場合には，二硫化モリブデンやグラファイトで代表される固体
潤滑剤が使用される．
液状の潤滑剤には，使用されている基材の種類により鉱油系潤滑
油，合成系潤滑油，および水系潤滑剤がある．これらのうち，鉱油
系および合成系潤滑油には，使用される基油や添加剤の配合方法に
より，2･2･2項に述べるように，自動車用潤滑油，工業用潤滑油な
ど使用目的別種類がある．一方，難燃性に優れた水溶性潤滑剤やエ
マルションなどの潤滑剤が水系潤滑剤と呼ばれ，火災に対する配慮
が必要な環境での潤滑や，優れた冷却性能が必要とされる潤滑な
ど，目的に応じて使用されている．
半固体状の潤滑剤であるグリースも，配合される増ちょう剤，基
油，添加剤の組合せで，自動車用，電機・情報機器用，産業用電動
機用など多くの種類がある．これらグリースの種類については2･
3･2項で述べられている．
固体潤滑剤としては，二硫化モリブデンなどの無機化合物系，金
や銀などの金属系，およびポリテトラフルオロエチレンなどの有機
化合物系に分類される．これら固体潤滑剤の種類については2･4･1
項を参照されたい．
2・1・2潤滑剤の性能と選定基準
a.潤滑剤の性能 潤滑剤を正しく使用するためには，摩擦部
分の寸法形状および材質，使用条件（荷重，速度，温度，雰囲気），
潤滑方法などを検討し，これらに適した潤滑性能を有するものを選
定する必要がある．潤滑剤の性能のうちでおもなものとして，(１)
粘度・ちょう度などのレオロジー特性，(２)潤滑性能そのものを示
す潤滑特性，および(３)潤滑剤のメンテナンス間隔に大きな影響を
もつ各種性能の安定性がある．このほかに物理的性質では，P-V-
T関係，熱的性質として，比重・密度，熱膨張係数，体積弾性係
数，圧縮率，比熱・熱伝導率，潜熱，揮発性・蒸発性などがあり，
光学的性質としては，色・蛍光・吸収スペクトル，屈折率などがあ
る．これらの性質からは，性能・品質の変化が読み取りやすく，そ
の測定結果は潤滑剤の品質管理によく利用される．また，絶縁油な
どでは潤滑油の電気的特性として，誘電率・比誘電率，誘電正接，
体積抵抗率，絶縁破壊電圧・絶縁耐力などの性質が重要となる．こ
のほか界面化学特性は，潤滑表面への潤滑添加剤の吸着やぬれ性な
どの良否に深くかかわっており，潤滑特性に密接な関係のある性質
として重要であるが，これらについては参考図書 を参照された
い．
.粘度・ちょう度 粘度（viscosity）の単位はパスカル
秒［Pa･ s，絶対粘度（absolute viscosity)］である．絶対粘度η
と密度ρとの比ν＝ηρが動粘度（kinematic viscosity）であり，
単位は平方メートル毎秒（m /s）である．潤滑油の動粘度にはこ
のほかに粘度，動粘度のCGS単位系としてポアズ（P），センチポ
アズ（cP），ストークス（St），センチストークス（cSt）が使用さ
れていた．また，工業用粘度（commercial viscosity）としてセイ
ボルト秒（SUS），レッドウッド秒（RWS），エングラー度（°E）
などがよく使用されていた．これらを動粘度に換算するために換算
式および換算表 がある．
