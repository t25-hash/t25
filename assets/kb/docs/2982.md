# 2・1・2 潤滑剤の性能と選定基準

> **章**: 第2章 潤滑剤
> **節**: 3・2 項で述べられている。

a. 潤滑剤の性能潤滑剤を正しく使用するためには、摩擦部分の寸法形状および材質、使用条件(荷重、速度、温度、雰囲気)、潤滑方法などを検討し、これらに適した潤滑性能を有するものを選定する必要がある。潤滑剤の性能のうちでおもなものとして、(1)粘度・ちょう度などのレオロジー特性、(2)潤滑性能そのものを示す潤滑特性、および(3)潤滑剤のメンテナンス間隔に大きな影響をもつ各種性能の安定性がある。このほかに物理的性質では、P-V-T 関係、熱的性質として、比重・密度、熱膨張係数、体積弾性係数、圧縮率、比熱・熱伝導率、潜熱、揮発性・蒸発性などがあり、光学的性質としては、色・蛍光・吸収スペクトル、屈折率などがある。これらの性質からは、性能・品質の変化が読み取りやすく、その測定結果は潤滑剤の品質管理によく利用される。また、絶縁油などでは潤滑油の電気的特性として、誘電率・比誘電率、誘電正接、体積抵抗率、絶縁破壊電圧・絶縁耐力などの性質が重要となる。このほか界面化学特性は、潤滑表面への潤滑添加剤の吸着やぬれ性などの良否に深くかかわっており、潤滑特性に密接な関係のある性質として重要であるが、これらについては参考図書を参照されたい。. 粘度・ちょう度粘度(viscosity)の単位はパスカル秒[Pa・s、絶対粘度(absolute viscosity)]である。絶対粘度ηと密度ρとの比ν=ηρが動粘度(kinematic viscosity)であり、単位は平方メートル毎秒(m/s)である。潤滑油の動粘度にはこのほかに粘度、動粘度のCGS 単位系としてポアズ(P)、センチポアズ(cP)、ストークス(St)、センチストークス(cSt)が使用されていた。また、工業用粘度(commercial viscosity)としてセイボルト秒(SUS)、レッドウッド秒(RWS)、エングラー度(°E)などがよく使用されていた。これらを動粘度に換算するために換算式および換算表がある。4―165潤滑油の動粘度は温度により大きく変化するが、その関係は次式(II-2・1)で示される。式(II-2・1)より異なる2点の温度での動粘度を測定することにより定数A およびB 潤滑性能潤滑膜の厚さは前述の粘度によって大きく変化するが、粘度が小さくなった場合や摩擦面の受ける荷重が大きくなった場合には、摩擦面どうしが接触する境界潤滑領域になる。境界潤滑領域において摩擦を低下させ、摩耗や焼付きを防ぐためには、油性向上剤、摩耗防止剤、極圧剤などと呼ばれる添加剤を必要に応じて使用しなければならない。これらの添加剤は、摩擦面に吸着膜を作ったり、摩擦面で分解して金属との反応皮膜や付着膜を生成して金属どうしの直接接触を防ぐ作用がある。また固体潤滑剤も摩擦面に付着して滑りやすい層を作る作用がある。. 各種性能の安定性潤滑剤の性能が使用中に変動することは好ましくないので、使用条件に応じて各種性能の安定性に優れたものを選定する必要がある。潤滑剤の低温における流動性や粘度指数を向上させるために、高分子化合物が添加される。この種の潤滑油では、せん断速度が大きい条件で使用されると、高分子化合物の分子が切れて見掛け粘度が低下する傾向がある。したがって、このような場合は使用中の粘度低下の小さいもの、すなわちせん断安定性のよいものを選ばなくてはならない。またグリースでは、機械的混和によって、増ちょう剤として使用している石けんの網目構造が壊れたり、細かく分散している非石けん系増ちょう剤どうしが集まって固まると軟化する傾向があるので、機械的安定性に優れていることが必要となる。潤滑油は使用中に加熱あるいは酸化されると、粘度が増加し、有機酸や不溶性のスラッジを生成する。粘度の増大は流動抵抗を大きくし、有機酸の生成は軸受金属などを腐食させる。またスラッジの生成は、フィルタの目づまりや摩擦面の損傷の原因となる。酸化防止剤、腐食防止剤、清浄分散剤などを用いることにより潤滑剤の熱安定性や酸化安定性は改善されるので、使用条件に応じてこれらの性能のよい潤滑剤を選定する必要がある。そのほか水と酸素の共存によって起こる鉄さびの発生防止、水が混入したときの水分離性、空気の混入によるあわ立ち防止などの性能も潤滑剤の選定に当たっては考慮する必要がある。. (固形)異物軸受や油圧機器などでは、潤滑剤中の(固形)異物が機器の寿命や騒音に大きく影響するので、できるだけ清浄なものを選定しなければならない。特にグリースでは(固形)異物はベアリングから発生するノイズの原因にもなっている。また、異物を除去することも液状の潤滑油に比較して難しいことから、取扱いには十分注意する必要がある。b. 潤滑剤の選定基準軸受、歯車などの機械要素や各種機械に適する潤滑剤の選定は、「適油」をキーワードとしてインタネットから情報を得ることができる。一般には、潤滑部分の設計に適合している粘度・ちょう度の選定が第一となる。また、使用期間、温度、摩擦面にかかる荷重などから、油膜が十分に形成されるマイルドな潤滑条件では、タービン油、作動油などの使用を検討し、厳しい条件では、摩耗・焼付き防止剤が多く添加されているエンジン油、ギヤ油などの潤滑剤を選定する。外部から異物が混入するか否かも選定基準として考慮する重要な条件となる。すすやスラッジの混入しやすいエンジン油では使用する添加剤の種類や量も多く、これらの異物が入らない工業用潤滑剤では添加剤の使用量も少なくなっている。また、すでに述べたように使用されている潤滑システムによって、液状の潤滑剤を選定するか、グリースあるいは固体潤滑剤が適しているかを判断することになる。これらは機器の設計段階で詳細に検討されているため、使用機器の潤滑剤選定に当たっては、機械メーカ、潤滑剤メーカの推奨を考慮し、最適なものを選定することが必要である。
