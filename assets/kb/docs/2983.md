# 2・2・1 種類と特徴

> **章**: 第2章 潤滑剤
> **節**: 2・2 潤滑油

a. 種類と特徴潤滑油(lubricating oil)はこれまで基油の精製度合い(粘度指数)によって高級潤滑油7種と並級潤滑油5種に分類されていたが、この分類が実状と合わなくなったため、2000年1月からまとめて9種類の用途別に分類されるようになった。9種類とは、1ガソリンエンジン油、2ディーゼルエンジン油、3その他車両用潤滑油、4船舶用潤滑油、5機械油、6金属加工油、7電気絶縁油、8その他特定用途向潤滑油、9その他潤滑油である。エンジン油をはじめとする車両用潤滑油は用途とグレードにより比較的わかりやすく分類されているが、機械油と金属加工油は使用される機械装置によって要求される性能が大きく異なるため、粘度グレードも合計すると約100にも及ぶ種類になる。したがって、使用条件および環境に適した潤滑油を正しく選定して使用することが重要となる。一般に、潤滑油は下記に述べる基油と添加剤から構成されている。b. 基油(base oil)基油には鉱油系基油(mineral oil)、動植物油(animal or vegetable oil)、合成系基油(synthetic oil)、水がある。基油の中では鉱油系基油が圧倒的に多く使用されている。これは鉱油系基油が安価であり、潤滑性に優れ各種の粘度を製造可能という長所を有しているからである。鉱油系基油は原油から常圧蒸留および減圧蒸留によって分取された後、水素化分解法または溶剤精製法によって精製され、さらに脱ろう工程、必要に応じ水素化精製工程を経て製造される。かつて使用されていた硫酸洗浄法、白土処理法は廃棄物の処分が問題であり、最近では一部の油種の精製に用いられているのみとなっている。鉱油系基油は原油のタイプによってパラフィン系油(paraffinbase oil)とナフテン系油(naphthene base oil)に分かれる。現在では各種潤滑油の基油としてはパラフィン系が主流である。最近になり水素化分解を特殊な条件で行うことにより、粘度指数120以上の合成油に近い粘度指数を有する基油が実用化されるに至っている。ナフテン系は原油ソースが限られており、使用量は少ないが、流動点およびアニリン点が低いという特徴を有している。このため、低温用作動油、ショックアブソーバー油、絶縁油、冷凍機油などに使用されている。また、界面活性剤の溶解性に優れるためエマルション系油にも用いられる。動物油、植物油の中では、価格的な理由から菜種油(rape seedoil)および大豆油(soybean oil)が主として使用されている。これらは酸化安定性において鉱油系基油に大きく劣るため、ほとんど実用的には使用されてこなかったが、生分解性が高いことから、最近になり自然界に流出する可能性の高い油圧作動油などの一部に使用され始めている。合成系基油として用いられている化合物と用途を表II-2・1に示す。主として価格の問題から、実際に使用されている量は鉱油系基油に比較するとはるかに少ない。合成系基油が使用される場合は、極めて高い温度で使用される場合、難燃性(nonflammability)が必要な場合、生分解性(biodegradability)が必要な場合、ワックス分を極端に嫌う場合、冷媒と溶け合う必要がある場合などである。水が基油として用いられている油種としては、水-グリコール系油圧作動液、エマルション系切削液などが主流であり、おもに冷却性または耐火性が要求される用途に使用されている。エマルションには水中油(O/W)型、油中水(W/O)型がある。c. 添加剤(additive)潤滑油の性能は主として添加剤の配合によって決まる。添加剤の種類は大きく分けると、酸化防止剤(antioxidant)、耐荷重添加剤(load carrying additive)、清浄分散剤(detergent dispersant)、腐食防止剤(corrosion in-hibitor)、粘度指数向上剤(viscosity index improver,viscositymodifier)、流動点降下剤(pour point depressant)、乳化剤(emulsifier)、消泡剤(antifoaming agent)がある。酸化防止剤にはフェノール系、アミン系、および耐荷重添加剤を兼ねるZDTP(ジアルキルジチオりん酸亜鉛)系がある。耐荷重添加剤は機能から油性剤(摩擦低減剤、oiliness agent)と極圧剤(extreme pressure agent)に分かれ、さらに極圧剤は摩耗防止剤(antiwear agent)と焼付き防止剤(antiseizure agent)の2種に分類される。油性剤は摩擦を低減する添加剤であって、金属表面に物理的または化学的に吸着することによって機能を発揮する。脂肪酸、脂肪族アミン、エステルなどがある。極圧剤は添加剤がしゅう動熱で分解し、次いで分解した活性種が金属表面と反応することによって摩耗や焼付きを防止する。りんおよび硫黄を含む化合物が代表的である。ただし、一つの添加剤を摩耗防止剤か焼付き防止剤かに区別することは難しい。なぜなら、焼付きを防止することによって摩耗を低減する場合があり、摩耗防止剤と焼付き防止剤を明確には区別しにくいからである。一般的にいうなら、摩耗防止剤は比較的低荷重で摩耗を減らすことができる添加剤であり、高荷重の焼付き防止性には劣る。りん系化合物はこのタイプが多い。焼付き防止剤は高荷重条件において焼付きを防止するが、低荷重領域での摩耗を低減する機能は低い。硫黄化合物は焼付き防止剤的な働きをするものが多い。硫黄系化合物は低荷重では摩耗を増加させることがあり、これは化学摩耗と呼ばれる。したがって、通常摩耗防止剤と焼付き防止剤を併用することによって、焼付き荷重が高く耐摩耗性がよい潤滑油が製造されている。第II部トライボロジー表-2・1 automatic transmissionfluid,CVTF: continuously variable transmission fluid)は摩擦係数がある程度高いことが要求されており、清浄分散剤が有効に利用されている。腐食防止剤は金属に対する腐食性を抑えるために添加される。鉄系金属のさびを防止するさび止め剤も腐食防止剤の一種である。粘度指数向上剤は油溶性高分子であり、高温での粘度低下を抑制する。より高分子のほうが粘度指数向上効果は大きいが、使用中に機械的せん断を受け低分子化しやすいのが欠点である。流動点降下剤としてはポリメタクリレートがおもに使用される。ポリメタクリレートは一般に粘度指数向上剤としての機能も有する。乳化剤はエマルション系の潤滑剤に添加される。抗乳化剤も界面活性剤であるが、油中に混入した水分の分離性を促進するため添加される。消泡剤は潤滑油の表面泡の消泡を目的として添加される。シリコーン油またはポリアクリレートが用いられる。
