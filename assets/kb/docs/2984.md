# 2・2・2 用途別潤滑油

> **章**: 第2章 潤滑剤
> **節**: 2・2 潤滑油

a. 自動車用潤滑油(automotive lubricating oil)陸用内燃機関用潤滑油、自動車用ギヤ油、自動変速機油などがある。内燃機関用潤滑油(internal combustion engine oil)は燃焼室からの混入物質の影響を強く受けるため、エンジンの構造や燃料の違いに応じて最適な添加剤が配合される(表II-2・2)。また潤滑油の規格決定に携わる機関は油種ごとに異なり、排出ガス規制対応などの自動車(エンジン)技術の変化に合わせて潤滑油規格を見直している。粘度グレードは、SAE粘度分類に準じて表示される(表II-2・3)。自動車用ギヤ油(automotive gear oil)は、ハイポイドギヤなどの終減速機を潤滑するものと、シンクロ機構の付いた手動変速機を潤滑するものがある。終減速機にはSAE粘度分類(表II-2・4)の80W-90や90,APIサービス分類(表II-2・5)のGL-5に対応する潤滑油が主として用いられるのに対し、手動変速機には75W-80から75W-90のGL-3級またはGL-4級の潤滑油が用いられる。終減速機用には耐荷重能を高くするために硫黄/りん系4―167低温流動性、冷媒との相溶性、生分解性ブレーキ液、ギヤ油、冷凍機油、グリースなどの添加剤が用いられるのに対し、手動変速機用にはシンクロ摩擦係数を増加させ、酸化安定性を向上させるため清浄分散剤、ZDTPといった添加剤も用いられる。なお、機械式差動制限機構(LSD)の付いた終減速機には適正な摩擦係数をもつLSD 油が用いられる。自動変速機油(ATF: automatic transmission fluid)は、自動変速機のトルクコンバータ、油圧系の作動油性能と歯車、軸受、湿式クラッチの潤滑性能をあわせもつ。特に、湿式クラッチ性能は乗り心地を左右するとともに、省燃費性能に優れるスリップ制御式トルクコンバータでは、湿式クラッチ性能が車両の振動防止に大きな役割を果たしている。GM 社のDEXRON-III、フォード社のMERCON-V などの規格が有名である。なお、金属ベルト式無段変速機は専用油の場合もあるがATFを使用する場合もある。b. 舶用潤滑油船舶で用いられる潤滑油としては舶用エンジン油(marine engine
