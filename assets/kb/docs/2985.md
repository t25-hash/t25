# 2・3・1 グリースの組成と性能

> **章**: 第2章 潤滑剤
> **節**: 2・3 グリース

グリースの組成は増ちょう剤(thickener)、基油(base oil)および添加剤からなっており、グリースの性能はこれらの組合せおよび製造方法により決定される。半固体状を特徴とするグリースの硬さは、おもに増ちょう剤量の多少により決まり、表II-2・12のとおり半流動状から極めて硬い状態まである。a. 増ちょう剤(thickener)グリースの半固体状を維持する役割を果たしており、増ちょう剤の種類によりグリースの耐熱性、耐水性、機械的安定性などの性能が主として決定される。一般的なグリースの増ちょう剤として脂肪酸の金属塩、すなわち金属石けんが使われ、石けん系増ちょう剤と呼ばれる。石けん系に用いられる金属として、リチウム、カルシウム、アルミニウム、ナトリウムなどがあり、現在単純な石けん系としては、欠点が少なく万能グリースとして使われるリチウム石けんグリースが主流になっている。リチウム石けんグリースは最高使用可能温度130~150°Cで、耐水性、機械的安定性は良好である。石けん系増ちょう剤の耐熱性を改良するため、脂肪酸とほかの有機酸との混合石けんを使ったものがコンプレックス石けんである。アルミニウムコンプレックス、リチウムコンプレックス、カルシウムコンプレックスグリースなどがあり、最高使用可能温度150~180°Cとなる。アルミニウムコンプレックスグリースは耐水性、機械的安定性に優れる。さらに石けん系の耐熱限界を超えるため、石けん以外の増ちょう剤として非石けん系が用いられる。有機系ではポリウレアやフッ素樹脂系があり、無機系にはベントナイトおよびシリカゲルがある。非石けん系の中ではポリウレアを用いたグリースの最高使用可能温度が180~200°Cであり、高温で長寿命であることや低摩擦性であることなどから最近市場での伸びが著しい。b. 基油(base oil)基油はグリースの主成分であり、おもに潤滑の役割を担っている。基油の種類によりグリースの潤滑性、使用温度範囲、酸化安定性、ゴム・樹脂材との適合性などの性能がお4―172機械要素・トライボロジー表-2・13 JIS K 2220によるグリースの分類種類用途別種別ちょう度番号適用温度範囲備考一般用グリース自動車用シャシグリース1種00号、0号、1号、2号-10°C~60°C鉱油とカルシウム石けんからなるシャシグリース。集中給油用グリース*この番号は、粘度範囲により分類したものである。もに決定される。経済性、潤滑性が良好なことから基油として鉱油が広く用いられている。鉱油としては低粘度から高粘度まで、潤滑油として使われるものはほとんど使用されるが、一般に高粘度のほうが潤滑性、使用温度上限、酸化安定性の面で有利である。鉱油の限界を超える使用温度範囲に対しては、合成潤滑油が基油として使用される。低温用にはジエステル、低粘度ポリ-α-オレフィン、低温から高温までの広温度範囲または高温用にはポリオールエステル、ポリ-α-オレフィン、シリコン、フェニルエーテル系などが適している。これらのうち、エステル、ポリ-α-オレフィンなどを用いた合成潤滑油グリースが多く用いられている。パーフルオロポリエーテルは耐熱耐酸化安定性に非常に優れるグリースを提供するが高価である。c. 添加剤(additive)添加剤はグリースの特殊な性能を高めるために使用される。グリースの極圧性、摩擦特性、さび止め性、酸化安定性などの性能がおもに添加剤の使用により向上する。潤滑油にも使われるような硫化油脂、りん酸エステル、有機金属酸塩などの極圧添加剤、スルホネートなどのさび止め添加剤、アミン系やフェノール系などの酸化防止剤、ベンゾトリアゾールなどの銅不活性化剤、ポリブテンなどの粘着剤などが同様の目的で使われる。また油と違ってグリースでは、二硫化モリブデン、グラファイトなどの固体潤滑剤が沈降することなく配合でき、境界潤滑下での潤滑性能が向上する。
