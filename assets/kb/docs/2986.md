# 2・3・2 グリースの種類と用途

> **章**: 第2章 潤滑剤
> **節**: 2・3 グリース

通常用いられるグリースの分類としては、成分にかかわるもの、用途にかかわるものおよびちょう度(硬さ)にかかわるものなどがある。a. グリース成分による分類と用途グリース成分に基づく分類として、増ちょう剤の場合は石けん系グリースやリチウムグリースなどの増ちょう剤名称、基油の場合は鉱油系グリース、あるいはシリコングリース、添加剤の場合は極圧グリース、あるいは二硫化モリブデングリースがその例である。1種1号、2号、3号、4号-10°C~60°C鉱油とカルシウム石けんからなり、耐熱性はないが耐水性がある。旧カップグリース。2種2号、3号-10°C~100°C鉱油とナトリウム石けんからなり、耐水性はないがやや耐熱性がある。旧ファイバーグリース。1種1号、2号、3号-20°C~100°C鉱油とリチウム石けんからなるはん用形で、自動車用に適用されることもある。転がり軸受用グリース2種0号、1号、2号-40°C~80°C鉱油あるいは合成油とリチウム石けんからなる低温用グリース。3種1号、2号、3号-30°C~130°C鉱油あるいは合成油とリチウム石けん、あるいは非石けん増ちょう剤からなる長寿命、広温度範囲用。自動車用ホイールベアリンググリース1種2号、3号-20°C~120°C鉱油と主としてリチウム石けんからなる耐熱性のよいホイールベアリング用。1種00号、0号、1号-10°C~60°C主として鉱油とカルシウム石けんからなる集中給油式中荷重軸受用。2種0号、1号、2号-10°C~100°C主として鉱油とリチウム石けんからなる集中給油式中荷重軸受用。3種0号、1号、2号-10°C~60°C主として鉱油とカルシウム石けんおよび極圧添加剤からなる集中給油式高荷重軸受用。4種0号、1号、2号-10°C~100°C主として鉱油とリチウム石けんおよび極圧添加剤からなる集中給油式高荷重軸受用。高荷重用グリース1種0号、1号、2号、3号-10°C~100°C鉱油と増ちょう剤および固体潤滑剤からなり衝撃を伴う高荷重軸受、しゅう動面用。ギヤコンパウンド1種1号、2号、3号-10°C~100°C鉱油とアスファルトからなる粘ちょうな潤滑油でオープンギヤおよびワイヤロープ用。鉱油系リチウムグリースおよび鉱油系リチウム極圧グリースが広範な用途に使用されている。鉱油系ウレアグリースは鉄鋼設備の連続鋳造機、自動車の等速ジョイント、ホイールベアリングユニットハブなどでの使用が多い。合成潤滑油系ウレアグリースは自動車の電装品、補機などで使用されている。エステル系リチウムグリースは家電機器、情報機器などで使用されている。b. JIS 分類と用途JIS K 2220:2003では表II-2・13に示すような分類になっており、7種類の用途、種別(成分および性能)とちょう度で細分されている。転がり軸受用、自動車用ホイールベアリング、集中給脂用などの用途別に品質が規定されているが、グリースは成分の組合せで種々特色のある製品が可能であり、例えば同じ転がり軸受用であっても、特定の用途・運転条件に適した品質のグリースが選定される場合も多い。
