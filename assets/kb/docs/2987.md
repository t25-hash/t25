# 2・4・1 固体潤滑剤の種類と特徴

> **章**: 第2章 潤滑剤
> **節**: 2・4 固体潤滑剤

固体潤滑剤は、相対運動する摩擦面の表面損傷や摩擦、摩耗を低減するために、粉末または薄膜として使用される固体をいい、(1)潤滑油やグリースへの添加剤、または(2)潤滑油やグリースの代替の潤滑剤、(3)固体潤滑剤自身の被膜または高分子や金属材料との複合被膜、および(4)しゅう動部材自身が自己潤滑性を有する固体潤滑剤含有複合材料として利用される。(1)と(2)は一般の機械や機器に適用される比較的簡易な適用方法であり、特に(3)と(4)は、潤滑油が酸化劣化する高温、流動性を失う極低温、蒸発する真空、放射線下など厳しい環境、食品・薬品工業やLSI製造装置など油やグリースによる汚染を嫌う環境、また酸、アルカリ水溶液や有機液体などの潤滑油が適用できない環境下での適用を可能にする。固体潤滑剤としては表II-2・14に示すものがあり、固体潤滑剤の一般的な特徴は固体自身が低摩擦を示す自己潤滑性を有し、潤滑油に比較して熱・酸化安定性に優れ、揮発性がないものが多い。これ表-2・14 固体潤滑剤の種類と分類種類固体潤滑剤無機化合物系層状構造化合物二硫化モリブデン(MoS)、二硫化タングステン(WS)、グラファイト、窒化ホウ素(hBN)、雲母、インタカレーション化合物(フッ化グラファイト、遷移金属ジカルコゲナイド)、複合酸化物系セラミックス(SrCaCuO)非層状構造化合物CaF、(PbO)、SiO、フラーレンC、クラスタダイヤモンド、カーボンナノホーン、カーボンナノチューブ金属系軟質金属Au,Ag,Sn,Cu、(Pb)有機化合物系固形物質脂肪(牛脂)、ワックス(パラフィン、蜜ろう)、金属石けん(ステアリン酸石けん)ポリマポリテトラフルオロエチレン(PTFE)、ポリイミド、ポリアセタール展性物質フタロシアニンその他メラミンシアヌレート、アミノ化合物固体潤滑剤結晶構造比重硬さ(モース)融点°C熱安定性大気中熱安定性真空中そのほかトライボロジーに関する性質二硫化モリブデン六方晶系(層状構造)4.8~5.01~21800<摩擦係数大気中:0.06~0.15真空中:0.01~0.2極圧性、複合材および被膜化二硫タングステン六方晶系(層状構造)7.4~7.51~1.5(不活性ガス中)摩擦係数大気中:0.05~0.28真空中:0.01~0.2合成系、極圧性、複合材および被膜化グラファイト六方晶系(層状構造)2.2~2.31~23500<―摩擦係数大気中:0.05~0.3真空中:0.40~1.0極圧性、複合材および被膜化ポリテトラフルオロエチレン(PTFE)窒化ホウ素(hBN)六方晶系(層状構造)2.26(昇華)摩擦係数大気中:0.2真空中:0.8合成系、白色、耐衝撃性、化学安定性金(Au)面心立方19.32.5~3――摩擦係数大気中:0.1~0.5化学安定性、耐腐食性、薄膜化メラミンシアヌレート(MCA)蒸気タービン、発電プラントギヤカップリング、バルブ初期なじみ、耐熱、耐摩耗油圧機器ベーン、側板、ロータ初期なじみ空圧機器シリンダ耐荷重、耐食工作機械ギヤ、滑り面、チャック耐荷重、耐食、耐スティックスリップ高圧機器バルブ、ボンベ、継手耐荷重、耐酸素、耐食自動機、ロボット、制御機器ソレノイド、カム、リンク耐摩耗、低摩擦係数、耐食原子炉、核融合炉シールリング、可動リミッタ、歯車耐荷重、耐放射線、ガス無放出半導体製造装置歯車、ラックピニオン、真空ポンプ耐熱、耐食、真空電子顕微鏡、真空機器歯車、滑り軸受高真空、耐熱、ガス無放出航空宇宙用ロケットランチャ滑り軸受耐荷重性、初期なじみ、休止効果人工衛星、宇宙ステーションベアリング、歯車、滑り機構耐真空、耐摩耗、耐食、一定の摩擦係数石油、天然ガス、パイプラインバルブ、ボール、ゲート、継手耐荷重、耐摩耗、耐食4―173第II部トライボロジーらのうちMoSやWSなどの遷移金属ジカルコゲナイドやグラファイトは異方性の、結晶面間の結合力の小さい層状構造を有するために摩擦により容易にせん断され、低摩擦を示す。またAuやAgなどの軟質金属やワックスなどの軟質物質、PTFEやポリアセタールなどのポリマもせん断されやすく、固体潤滑剤となる。表II-示す。
