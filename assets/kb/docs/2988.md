# 4・1・3 接点材料

> **章**: 第４章 電磁気・電子材料
> **節**:  

接点（electrical contacts ）は電気回路を開閉する部分に使わ
れ，電力用の遮断機や開閉器，各種制御装置の機械接点などで幅広
く用いられている．
接点には接触抵抗が低く安定している，溶着が生じにくい，放電
消耗や摩耗が少ないなどの特性が要求される．このため材料には，
導電率や熱伝導率が高い，耐酸化性が良い，融点や沸点が高い，硬
度が高い，耐食性が良いなどの特性が望まれ，単金属や合金のほか
に焼結材料が用いられている．表4･4に代表的な接点材料とその特
徴を示す．
Agは導電率や熱伝導率が高く常温では酸化されにくいため，軽
負荷・中負荷用の接点材として最も広い範囲で使用されている．し
かし，融点が低い，硬度が低いため接触圧力を大きくできない，S
が存在する環境では容易に硫化物の被膜やウィスカが発生して接触
障害を起こすなどの欠点をもっている．Agを主体とする合金はそ
れらの欠点を改善するために種々の金属や酸化物などを添加したも
ので，耐摩耗性や硬度が改善されている．
Auは耐食性に優れており微弱電流に対する用途で用いられてい
るが，融点や硬度が低いため溶着しやすく耐摩耗性に欠点がある．
Auを主体とする合金としてはAu- Ag- Pt合金が代表的であり，通
信用継電器においてPt代用接点材として広く用いられている．
