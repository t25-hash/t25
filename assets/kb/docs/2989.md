# 4・1・4 抵抗材料（電熱材料，サーミスタ，バリスタ)

> **章**: 第４章 電磁気・電子材料
> **節**:  

抵抗材料（electric resistance material ）は物質のもつ電気抵抗
およびその変化を利用する材料であり，標準抵抗や電位差計などの
められ，金属のほかに非金属の材料も用いられている．
計測器に用いる精密抵抗材料，電子機器の回路制御や電車用電動機
のような電流調整に用いる加減抵抗材料，電気炉の発熱源として用
て知られるFe- Cr いる電熱材料などに分類することができる．抵抗材料は金属性のも
のから炭素や酸化物のような非金属性のものまで種類が多く，それ
ぞれのもつ固有の特性によって用途別に選択されている．
抵抗材料には一般的に，抵抗率が適当に大きく経時変化が小さ
い，抵抗の温度係数が小さい，耐食性が良い，機械的強度が高いな
どの特性が要求される．
a.精密抵抗材料（standard resistance material ) 標準抵抗
器や電気計測器など精度を要求される抵抗材料としてはCu- Mn合
金やCu-Ni合金などが用いられている．JISでは，JIS C2522：
1953にCu- Mn合金，JIS C2521：1953にCu-Ni合金についての
規定がある．表4･5および4･6にJISに規定された特性を示す．
マンガニンの名前で知られているCu-Mn合金は，代表的な精密
抵抗材として最も広く用いられている．抵抗温度係数や対銅熱起電
力が極めて低い特徴があるが，耐食性や耐酸化性に乏しい欠点もも
つ．
Cu- Ni合金はすべての配合割合で固溶体を形成し，安定した抵
抗材料として使用できる．このうち約45%のNiを含む合金はコン
スタンタンなどの名前で知られ，抵抗率が大きく，かつ温度係数が
小さいため精密抵抗材料として広く使用されている．Cu-Mn合金

e.サーミスタ（thermistor ) 半導体の電気抵抗が示す特徴を
利用したものとしてサーミスタとバリスタがある．サーミスタは半
導体の電気抵抗の温度係数が負の大きな値を示す性質，すなわち温
度が高くなると電気抵抗が大きく減少する性質を利用した素子であ
る．サーミスタは，Mn，Co，Ni ，Cu，Feなどの金属の酸化物を
混合焼結したり，SiCと金属酸化物の混合焼結によって作られる．
用途としては温度-抵抗特性を利用して，温度測定，温度補償，定
電圧装置などに用いられる．
f .バリスタ（varistor ) バリスタは電圧-電流特性がオーム
の法則に従わずに非直線性を示す抵抗素子である．このような非直
線性はサーミスタにも見られるが，サーミスタの場合，通過電流に
よる発熱で温度が上昇して抵抗値が減少することによる．一方，バ
リスタは本質的に電圧-電流特性が非直線性を示すもので，一定以
下の電圧ではわずかな電流しか通さないが，ある程度以上の電圧が
印加すると抵抗値が著しく減少して通過電流が急激に増加する．こ
の特性を利用して高電圧大電流に耐えるものは避雷器に用いられる
ほか，過電圧に対する回路保護や電話機の雑音防止などに使用され
ている．バリスタの材料としては通常SiCの焼結体などが用いら
れる．
