# 2・5・1 潤滑の目的と潤滑法

> **章**: 第2章 潤滑剤
> **節**: 2・5 潤滑法

潤滑の目的として次のような作用が要求されている.1減摩作用(摩擦,摩耗,発生熱の軽減)2冷却作用(潤滑部の温度上昇の抑制)3密封作用(シリンダとピストンなどの気密保持)4腐食防止作用(さび,腐食の抑制)5防塵(ぼうじん)作用(異物の混入防止)6清浄分散作用(内燃機関などの異物除去,スラッジの溶解)7応力分散作用(歯車,軸受などの集中応力分散)この目的を達成するために次の3要素を考慮しなければならない.1摩擦条件(軸受,歯車,しゅう動面といった摩擦面の種類と回転速度,荷重,温度などの運転条件)2潤滑剤(種類と性能)3潤滑法(全損式,循環式などの種類と給油条件)潤滑剤としては,油,グリース,または固体潤滑剤がある.これらの選定は摩擦条件および潤滑系と連携して選定されるべきである.これら3要素がうまくかみあったときに潤滑の目的が達成できる.設備の信頼性を表II-2・18に示す.固有の信頼性で特に重要なものが設計技術であり,設備の信頼性の30~40%程度は設計技術にかかっている.使用の信頼性は環境因子のほかに,使用方法や保全方法,運転員や保全員の技術,技能などの人的要素によって大きく左右される.したがって,設備の信頼性を上げ,最終的にコストの低減を達成するためには,装置に適した潤滑系であること,使用環境に配慮していること,運転員や保全員の人的要素を考慮に入れること,公害対策,防災対策を満足させること,なるべく標準形の装置,器具を採用することが必要である.集中潤滑装置を採用すれば,装置全体の価格は上昇するが,運転費の低減が大きく十分ペイアウトすることが多い.また,多数の潤滑箇所を有する装置では自動潤滑装置にすれば,給油ミスを防ぐことができ,信頼性,作業性,安全性が向上できる.こうした理由のため,現在ではほとんどの機械装置に自動集中潤滑装置や噴霧給油装置,また大形装置では循環給油装置が採用されるようになってきている.
