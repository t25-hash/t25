# 2・5・3 グリース潤滑と潤滑系

> **章**: 第2章 潤滑剤
> **節**: 2・5 潤滑法

グリース潤滑法を用いる場合の利点は次の諸点である.1ハウジングの密封構造を単純化できる.2保守が比較的容易である.3外部からの異物の侵入がグリースの特性により比較的容易に防止できる.4漏えいが少なく機械の清浄性を向上させやすい.グリース潤滑系の種類を表II-2・20に示す.グリース潤滑系は全損式で回収循環されることはなく,また冷却能力はないが,使いや表-2・19油循環系の種類手差し給油口を設けて,手差し給油器により油の切れ方を見て適時給油する.軽荷重低速間欠運転滴下視滴型,ビン型給油器により,ほぼ一定の量を細孔を通して常時給油する.軽中荷重周速4~5m/s以下灯心オイルカップの油つぼより灯心の毛管作用と細孔のサイホン作用で給油する.軽中荷重周速4~5m/s以下全損式機械のカム,斜板などで駆動されるプランジャポンプで35MPaまでの圧力で給油される.機械式回収循環式軸にかけたオイルリングの回転により油だめから油をかき上げて軸受上部に給油する.低速ではチェーンを使用する.軽中荷重10~12m/s重荷重6~7m/sリング,チェーン回転体で油だめの油をたたいたり,かき混ぜたりして油の粒子をはねかける.高速歯車シリンダはねかけ(飛沫)強制循環式強制タンク,ポンプ,ろ過器,冷却器,配管系をもつ強制循環式で,油は絶えず循環している.はん用で特に大形機械用
