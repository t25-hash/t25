# 4・2・4 はんだ接続材料

> **章**: 第４章 電磁気・電子材料
> **節**: 4・2 半導体デバイス材料

こうして製造された集積回路はウェーハから個片に切出され，
LSIチップ（chip）となる．このLSIチップと外部との電気的接
続を容易に行うため，はんだ（solder ）材料が使用される．はんだ
材料に望まれる性質としては，材料費が安く，数百℃以下の温度で
容易に多数点を同時に接続できる経済性と，その接続が
－100～150℃程度の温度範囲で安定に保たれる信頼性等がある．
また，電気的な接続が不良となった場合には，部品の取替えを容易
に行える再溶融性も求められる場合がある．さらに，降伏強度が低
く，容易に塑性変形することで，融点から室温近傍までの冷却過程
でLSIチップに過度な熱応力を発生させないという性質も重要で
ある．
適度な粘性，流動性をもち，低融点であることが望まれるため，
一般には共晶合金が使用される．代表的な材料を表4･10に示す．
組成別に分類すれば，Sn- Pb系，Sn系，Pb系，Au系，In系など
がある．このほかに特にアルミニウムとの接続用にZn系，Sn- Zn
系，Cd- Zn系材料などが使用される場合もある．
これらの材料の中で最も広く使用されてきたのがSn-Pb系はん
だである．共晶組成は61.9 %Sn-38.1 %Pb（以下，組成表示は

mass%）で液相線温度（融点）は183℃であるが，一般には63%
Sn-37%Pbをスズ鉛共晶はんだと呼んでいる．このSn-Pb系はん
だは，ほぼ全組成範囲で使用することができる．融点が高い材料と
してはPb-5Sn（液相線温度：314℃）やPb-10Sn（液相線温度：
302℃）が用いられる．逆に融点を下げたい場合には，BiやAg，
Cu元素などを数%添加することが有効である．特にCuの添加は
強度やクリープ特性の改善に有効である．高価なSnの代替として
高い接合強度や熱伝導性が要求される場合には，Au系はんだが
使用される．代表的な材料としてはAu-3.15Si （液相線温度：
363℃)，Au-20Sn（液相線温度：280℃)，Au-12Ge（液相線温
度：356℃）などがある．いずれの材料も引張強度が室温近傍で
耐熱性の低い部品の接合にはIn系はんだが使用される場合があ

今後環境保護という観点からPbフリーの代替材料の開発が望ま
れている．融点や強度特性がスズ鉛共晶はんだに近い材料として
Sn-Ag系はんだ（共晶組成：Sn-3.5Ag，液相線温度：221℃）が
ある．液相線温度を下げるためには数%のBiやCuを添加するこ
とが多い．Biの添加は液相線の低温化には有効であるが，含有量
が多くなるともろくなる場合がある．液相線温度はほとんど変化し
ないが，強度特性が安定している材料としては，Sn-3Ag-0.5Cu
などが検討されている．低融点はんだ材料としてはSn-58Bi （液
相線温度：139℃）が使用されることもある．信頼性向上の観点か
らは材料単体の特性だけでなく，接合する材料との界面近傍に形成
される合金層の制御も重要な課題となる．
