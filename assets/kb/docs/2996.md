# 2・5・4 固体潤滑と潤滑系

> **章**: 第2章 潤滑剤
> **節**: 2・5 潤滑法

固体潤滑剤の実用化に際して重要なことは、この特異性のある潤滑剤の機能を十分に発揮させるための適用法を誤らないようにしなくてはならない。4―176機械要素・トライボロジー適用法を誤れば、固体潤滑剤の性能を発揮させるどころか、かえって潤滑上で悪影響をもたらす場合がある。この特色のある潤滑剤が適用法を誤ったために失敗し、本来ならば産業界に貢献できたであろう場合に逆に失敗し、不評を買ってしまった事例が枚挙にいとまがないくらいある。このような事態を避けるためには、最初に固体潤滑剤の長所および短所を習得熟知し、適用法を誤らないように注意する必要がある。a. 固体潤滑(solid lubrication)。 粉末のまま使用する方法固体潤滑剤の粉末を手作業またはタンブリングなどの方法により母材の表面に擦り込む方法、または高速気流中に固体潤滑剤の粉末を浮遊させて目的とする母材表面に衝撃的に打込む方法などがある。. 油、グリースに混入する方法固体潤滑剤の微粒子を油、グリース混入してその潤滑特性を向上させるもので、摩擦特性の向上は別として、物理的性状は一般の油、グリースに類似しているので使用法もそれに準じればよく比較的利用しやすい利点がある。ただし、この場合には、固体滑剤含有、油、グリースならではの潤滑方法もあるので注意は必要である。. インサイチュ(in-situ)法金属表面に直接、固体潤滑剤として有効な物質を形成させる方法である。例えば、母材の金属と、硫黄または硫黄化合物と反応させ金属硫化物を表面に形成させる方法である。. 金属マトリックス法複合めっきともいわれ、母材表面に、化学めっきまたは電気めっきなどの方法でめっきをする場合に同時に固体潤滑剤の微粒子を析出させる方法である。この方法に類似した方法に、母材がアルミニウムまたはアルミ合金の場合は、陽極酸化を行い、その層にフッ素樹脂または二硫化モリブデンなどを含浸させる方法である。. 複合材固体潤滑剤とほかの物質を配合し潤滑性のある複合材を形成させる方法である。この場合、一部の固体潤滑剤は単に物理的に混合されるだけでなく化学反応を伴うことがあり飛躍的な性能向上が得られる場合がある。. ドライプロセス法ドライプロセスとは真空を利用して真空中で母材表面に固体潤滑被膜を形成させるものである。真空蒸着、スパッタリング、イオンプレーティング、イオンプランテーションなどの方法がある。ドライプロセス法の長所は、すべての工程が真空中で行われるため清浄化された母材表面は異物に汚染されることなく被膜形成が行われるので被膜の付着強度が高い。短所としては、厚膜の形成が困難で真空装置を用いるため、大きな品物に対するコーティングが困難であり、形成された被膜は酸化などの影響を受けやすいので取扱いに注意が必要なことである。. 埋込み法固体潤滑剤を圧縮成型したもの、またはほかの物質と配合固形化した複合材を母材の摩擦面に埋め込み、摩擦により母材が摩耗するに従い埋め込まれた固体潤滑剤が摩擦面に供給されるようにする。. 結合(ボンディング)法固体潤滑剤の粉末などに結合材(バインダー)を配合して塗料状にして母材表面にコーティングして潤滑被膜を形成させる方法で乾燥被膜潤滑ともいわれ産業界では幅広く大量に使用されている。b. 潤滑系固体潤滑剤の潤滑系としては、油、グリースに混入して使用する場合は油、グリースの潤滑系に準じるが、固体被膜潤滑剤として使用する場合は次の系がある。(1) 終身潤滑系固体潤滑剤の寿命が機構自体の寿命より長い場合はメンテナンスフリーで使用できる利点がある。(2) 定期補修系定期補修を行う系で、その期間中は潤滑性能を維持できる場合、例えば鉄道車両、舶用エンジンなどで固体潤滑剤の補修が目的でなく機構全体の定期補修の際に合わせて再処理を行う。(3) 初期なじみ油、グリース潤滑で過酷な条件でスタートする場合優れた耐荷重性を生かし、なじみ運転用として使用される。通常の場合、そのまま運転状態に入る。
