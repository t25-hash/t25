# 4・4・2 無機固体絶縁材料［雲母（mica） ，ガラス（glass ） ，

> **章**: 第４章 電磁気・電子材料
> **節**: 4・4 絶縁材料

セラミックス（ceramics ） ］
無機の固体絶縁材料は1MV級の超高圧変電所用の長さ12mに
及ぶ大型の碍管からエレクトロニクス部品の放熱基板など幅広く用
いられている．
碍子・碍管にはシリカ（SiO ）が，成形性が良好で曲げ強度も
120～140MPa と高いことから多く用いられている．冷却と絶縁
の両立が必要なエンジンの点火プラグにはアルミナ（AlO ）が使
用されている．高温のヒータ支持体などには雲母（マイカ）が使用
されている．
半導体素子の冷却を兼ねた絶縁基板材料として，アルミナ，窒化
アルミ（AlN）が用いられている．また，信号の高速伝送のため
に低誘電率が必要な用途では，ムライト（3AlO ･ SiO ）が使用さ
れている．かつて，熱伝導率の高いベリリア（BeO）が使用された
