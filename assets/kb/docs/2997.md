# 2・6・1 集中潤滑装置(centralized lubrication system)

> **章**: 第2章 潤滑剤
> **節**: 2・6 潤滑装置

1台のポンプにより多数の潤滑箇所に潤滑剤を供給する装置であり,油,グリースの共用形が多い.これにはポンプから配管を通じて直接潤滑箇所に潤滑剤を圧送する直接供給型(direct feed)と配管内に分配弁を設けて,これを通して間接的に圧送する間接供給型(indirect feed)がある.機械式給油は直接供給型であり,代表的な機械式給油器を図II-2・2に示す.この場合供給量の調整はプランジャポンプの行程の調整か,回転速度の変更によって行う.配管系に分配弁を組込む間接供給型にも供給量の調整のやり方により,抵抗型分配弁式と容積型分配弁式がある.抵抗型分配弁は油専用で,弁内の油路にすきま,または絞り機構を設けて,油の流れに抵抗を生じさせ,各潤滑箇所に相対的に給油量を変えて配分給油するもので,油量の絶対量は規定できない.図II-2・3に抵抗型分配弁(proportioning device)を示す.容積型分配弁は油,グリ一ス両用で,潤滑剤の圧力によって,分配弁内のピストンを作動させて計量配分するものである.図II-2・4は容積型分配弁(distributor)の一例を示す.間接供給型の集中潤滑装置には分配弁の作動方式によって3方式がある.a. 抵抗型分配方式(resistance types)油専用で,供給主管は単管で間欠または連続運転ができる.表II-2・21にこの例を示す.この装置は機構,配管とも簡単で小形の給油装置に適しており,工作機械,織機などの少量給油に使用されている.分配弁には作動の保証機構がなく,また油温が変化すると油粘度の変化により給油量が変動する欠点がある.b. 直列作動方式(series types)[進行作動方式(progressivetypes)]油,グリース両用で分配弁が供給主管に直列的に入っており,個々の分配弁は潤滑剤がその中を通って次の分配弁に入って行く前に動作が完了していなければならない.もし1個の分配弁図-2・2 機械式給油器図-2・3 抵抗型分配弁が作動しなくなると全システムが停止する.すなわち動作保証が得られることになる.分配弁の構造は複雑であるが,制御装置は簡単である.直列型分配弁はピストンの作動行程は一定で,したがって供給量も一定で調整できないので,計画時に適当な容量の分配弁を選定することが必要である.供給主管は単管が多いが,小形のものでは複管もあり,回路型(loop type)と行き止まり型(end-of-line type,エンド型)がある.表II-2・22にこの例を示す.c. 並列作動式(parallel types)個々の分配弁が単独に供給表-2・21 抵抗型分配弁方式の主要な集中潤滑装置圧送圧力MPa形式配置図使用潤滑剤抵抗型分配弁並列作動単管式行き止まり型間欠運転潤滑油0.2~0.75抵抗型分配弁並列作動単管式行き止まり型連続運転潤滑油~1.5制限なし圧送圧力MPa主管長さm形式配置図使用潤滑剤容積型分配弁直列作動単管式回路型間欠運転潤滑油グリース~21潤滑油グリース容積型分配弁直列作動単管式分岐型連続運転グリース(100)潤滑油容積型分配弁直列作動複管式回路型間欠運転グリース(50)潤滑油4―177第II部トライボロジー主管に連結され,潤滑箇所に潤滑剤を供給するために必要な圧力に達したとき,個々の分配弁が独自に作動し,全部の分配弁が作動完了し圧力がさらに上昇し一定圧力に達したときにサイクルが完成する.したがって一つの分配弁が故障してもほかの分配弁の作動には影響がない,すなわち動作保証は得られないことになる.複管式は配管に場所をとり,設備費も高いが分配弁の構造が簡単で,作動も確実性があるため最も広く使用されている.表II-2・23にこの例を示す.図II-2・5に大形機械で広く用いられている集中潤滑装置を示す.以上の3方式の集中潤滑装置には,分配弁の故障,配管のつぶれや破損,空気の侵入,ごみや固形物の侵入などによる故障が発生する.このため表II-2・24に示すような制御装置が設置されている.
