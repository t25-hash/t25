# 2・6・3 噴霧給油装置(aerosol lubrication system)

> **章**: 第2章 潤滑剤
> **節**: 2・6 潤滑装置

噴霧給油はオイルミスト(oil mist)と呼ばれる霧状の油粒子を含んだ空気を給油箇所に送り、薄い油膜を形成させ、同時に冷却用に空気流を利用したもので次の利点がある。1油のかくはんがなく、馬力損失、温度上昇が小さい。2冷却作用が比較的大きい。3内部圧力を高く保つことで、ごみの侵入が防止できる。図II-2・8に噴霧潤滑用オイルミスト発生装置の一例を示す。ろ過され減圧された清浄な空気が発生装置のベンチュリを通過する。油だめの潤滑油はベンチュリ部の圧力低下により、ドームに吸上げられる。ドームは油の通過を見るためのもので、通過量は空気の油4―180機械要素・トライボロジー図-2・8 噴霧潤滑用オイルミスト発生装置図-2・9油粒子の付着性能表-2・27 オイルミストの配分比率配分比率オイルミスト発生量リクラシファイヤにてウェットミスト状態になった量配管途中においてウェットミスト状態になった量空中に飛散したミスト量油種g/h%g%g%g無添加油40.864.526.315.26.220.38.3ミストオイル30.672.822.320.96.46.31.9だめへのバイパス回路に設けられたニードル弁により調整される。空気はベンチュリ部の上部から入り、ドームから滴下した油と混合しミストを発生し、さらに下方の4個のノズルから吹込まれた空気の二次ジェット噴流により粉砕される。こうしてできたオイルミストは付着しにくい性質の乾いた霧状の油粒子と付着しやすい性質の湿った霧状の油粒子が混合したものである。このオイルミストは油だめの上部空間を浮遊しながら滞留し、90%程度の油粒子は油中に落下し、10%程度の2 m以下の乾いた油粒子だけが空気出口から、空気流の中に送り出される。油粒子の大きさと、壁に衝突したときに壁に付着するために必要な衝突速度との間には、図II-2・9に示すような関係がある。油粒子は細かいほど付着しにくく、速度は遅いほど付着しにくい。すなわち乾き領域にあれば付着しにくい。このため配管内の流速は最小限度にし、また各給油点に所定の給油量が得られるように一定の静圧を維持するようにする。乾き領域の油粒子を湿り領域の油粒子に転換させるために、給油点にリクラシファイヤ(reclassifier fitting)あるいはノズルといわれる器具を設けて管路を絞り、ここで乱流を発生させ、油粒子を互いに衝突させて大きな粒子に成長させ、同時に流速も上昇させて潤滑面に付着しやすいようにする。噴霧給油ではどうしてもある程度のオイルミストが潤滑部位から空気中に飛散し、作業環境の雰囲気を悪化させる。この飛散ミストを少なくするために潤滑油の選定が重要であり、細粒化しつつ過度に微細な粒径を抑制する。表II-2・27に普通の潤滑油と噴霧潤滑用ミストオイルとの配分比率を示す。ミストオイルでは大気中に飛散するミストの量が77%減少しており、雰囲気を著しく改善させている。この噴霧潤滑は、シールが完全で連続給油ができる所では、極めて有利な方法であるが、水分や固形物の侵入の多い所では不適当であり、シールの改造か、給油方法の変更が必要である。
