# 3・1・1 摩耗(wear)

> **章**: 第3章 表面損傷
> **節**: 3・1 損傷の種類

摩擦面から微小な粒子,すなわち摩耗粉が次々に離脱し,固体の体積が漸進的に減少する現象を摩耗という.広義には,体積減少を実質的に伴わない表面損傷,例えば塑性変形による形状変化や相手面への材料の移着のみが起こる場合も摩耗に含めることがある.摩耗量は摩擦面材料の固有の性質によって決まるものではなく,相手面材料やしゅう動条件に依存する.摩耗において作用する機構は多様かつ複雑であり,一対の摩擦面で同時に複数の損傷機構が働くことも多いため,摩耗機構の解釈や分類,用語は必ずしも統一されていない.最も一般的な摩耗の分類は,現象の支配的因子に基づくものであり,摩擦面間に生じる凝着力に起因する凝着摩耗(adhesivewear),硬い相手面ないしは2面間に介在する硬質粒子の切削作用に起因するアブレシブ摩耗(abrasive wear),雰囲気や潤滑剤との化学反応の影響が支配的である腐食摩耗(corrosive wear),繰返ししゅう動に伴う摩擦面材料の疲労に起因する疲労摩耗(fatigue wear)に大別されることが多い.このほか,重摩耗(シビヤ摩耗),軽摩耗(マイルド摩耗),正常摩耗,異常摩耗など損傷の程度による表現や,滑り摩耗,転がり摩耗,往復動摩耗,潤滑摩耗などのように摩擦面の運動形態やしゅう動条件を付した表現もある.特別な場合として,はめあい部などにおいて面が微小振幅で往復運動する場合に生じる摩耗はフレッチング摩耗(frettingwear)ないしはフレッチングコロージョン(fretting corro-sion),固体粒子を含む流体の連続的な衝突によって起こる摩耗はエロージョン(erosion,erosive wear)や衝撃摩耗(impactwear)と呼んで区別する.
