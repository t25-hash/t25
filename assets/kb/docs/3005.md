# 3・1・2 焼付き(seizure)

> **章**: 第3章 表面損傷
> **節**: 3・1 損傷の種類

潤滑された摩擦面において,何らかの原因で潤滑状態の不安定な遷移が急速に起こり,固体どうしの直接接触が生じて摩擦が急に増大し,温度上昇とともに巨視的な凝着や溶着を生じて表面が激しくあれ,場合によっては2面が完全に固着して相対運動が停止する現象を総称して焼付き(やきつき)と呼ぶ.焼付きは,それが生じる機械要素の種類や損傷の形態によってさまざまな用語で表現される.歯車,カムタペット,ピストンシリンダ,バルブ案内面などの滑り接触面に生じる焼付きやそれに伴う摩耗をスカッフィング(scuffing)ないしスコーリング(scoring),転がり軸受の転動面の焼付き全般をスミアリング(smearing),スコーリングのうち特に表面のあれが激しいものをゴーリング(galling),冷間圧延でのロールとストリップの間に生じるものをヒートスクラッチ(heat scratch)などと呼ぶ.
