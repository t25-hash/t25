# 3・1・3 疲労損傷(fatigue failure)

> **章**: 第3章 表面損傷
> **節**: 3・1 損傷の種類

繰返ししゅう動する摩擦面において、ある運転時間経過後に微小き裂が発生し進展して表面層がはく離する現象である。一般に摩耗よりも大きな粒子が脱落し、表面には穴状の損傷が点在して、機械要素の振動を発生させたり運転精度の低下を招く。疲労損傷は、機械要素の種類や損傷の形態によってさまざまな用語で表現される。特に転がり接触面においては疲労損傷が機械要素の寿命を左右するため、転がり軸受の軌道面や転動体の表面に生じるものをフレーキング(flaking)、歯車の歯面に生じるものをピッチング(pitting)、表面硬化された歯面などで比較的大きな穴が生じるものをスポーリング(spalling)、歯面の硬化層と下地との境界にき裂が進展してはく離を生じるものをケースクラッシング(case crushing)などと呼んで区別する。
