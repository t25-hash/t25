# 3・2・1 凝着摩耗(adhesive wear)

> **章**: 第3章 表面損傷
> **節**: 3・2 摩耗

凝着は、物体が真実接触部で接触するとき物体どうしが強い結合(一次結合や二次結合)を生じた状態をいう。2面間で摩擦するときに真実接触点で凝着が生じ、接触界面以外で破断したとき、一方の物体が相手面に移着する。摩擦の繰返しで、移着・再移着を繰返して摩耗粒子を生成する摩耗が凝着摩耗である。大気中では凝着をイメージしにくいが、金属の凝着摩耗を理解するには酸素の吸着や酸化を考えるとわかりやすい。図II-3・1は摩耗の雰囲気圧力依存性を示す。高真空中(10Pa以下)では酸素の吸着が無視できるため大きな凝着を生じ、生成された粒子は接触面で移着・再移着を繰返し、摩耗粒子の形成は抑制され、体積損失としての摩耗量は小さい。低真空中では移着粒子表面に酸素が次第に吸着し再移着を妨げ、大きな摩耗粒子が生じて摩耗量は大きくなる。大気圧になると金属新生面が露出してもすぐに酸化するために凝着は抑えられ、大きな粒子が生じても酸化が進み、細かい酸化物粒子となるために摩耗量も小さい。このことから凝着摩耗は金属摩耗粒子の形成と表面への適度な酸素の吸着や酸化が必要となる。また、凝着摩耗は接触面間の凝着のしやすさに依存する。図II-図-3・1 摩耗における雰囲気圧力の影響第II部トライボロジーの高い組合せは摩耗の程度を大きくする。同一部材どうしの摩擦はともがねと呼ばれ、接触界面で強い凝着が生じ、凝着摩耗の程度は大きくなる。焼付きは、凝着が非常に強くなり摩擦抵抗が駆動力よりも大きくなった状態である。凝着摩耗のアーチャードのモデル(Archard’s wear model)を図II-3・3に示す。摩擦面は半径の接触点個からなり、距離2滑ると別の位置に新たな接触点を形成する(II-1・3・4参照)。
