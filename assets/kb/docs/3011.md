# 3・2・2 アブレシブ摩耗(abrasive wear)

> **章**: 第3章 表面損傷
> **節**: 3・2 はおもな金属の相互溶解性を示したものである。相互溶解度

粗さをもつ硬い面が軟らかい面を摩擦することで、軟らかい面が著しく摩耗する状態をアブレシブ摩耗という。アブレシブ痕が生じるには、硬い物体の硬さ(ビッカース硬さ)が軟らかいほうの1.3倍以上必要である。アブレシブ摩耗には二元摩耗(two-bodyabrasion)と三元摩耗(three-body abrasion)がある。前者は硬さの異なる摩擦する二つの接触面間で生じる場合である。一方、後者は接触面間に硬い粒子などが侵入し、それが摩耗を引き起こす場合である。このアブレシブ摩耗は材料の摩耗損傷として最も大きなダメージを与え(図II-3・13参照)、また摩耗損傷の約50%がアブレシブ摩耗に起因するといわれている。図II-3・4はアブレシブ摩耗のモデルであり、1個の硬い円すい突起が軟らかい面に食い込み、摩耗を引き起こす(II-1・3・4参照)。アブレシブ摩耗は荷重に比例し、材料の硬さに反比例する。図II-した耐摩耗性(摩耗量の逆数)の結果であり、アブレシブ摩耗が硬さに反比例することを示している。しかし、アブレシブ摩耗では図II-3・4のモデルのように突起の形状のまま摩耗することはまれである。図II-3・6はアブレシブ摩耗形態図(abrasive wear map)を示している。ここでは摩耗痕の両側に塑性変形により盛上りが生じ、実質的な摩耗損傷がない場合を図-3・2 金属の相互溶解度図-3・3 アーチャードの凝着摩耗モデル4―185掘り起こし(ploughing)型、突起の先端に摩耗粒子が堆積した状態をウェッジ形成(wedge formation)型、切くずのような摩耗粒子が形成される切削(cutting)型と呼んでいる。したがってアブレシブ摩耗は突起の形状や表面状態により変化することを示している。
