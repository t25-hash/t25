# 3・2・3 腐食摩耗(corrosive wear)

> **章**: 第3章 表面損傷
> **節**: 3・5 はコランダム研磨布に対する軟らかいホワイトメタルを基準に

腐食摩耗は腐食環境中において摩耗部材が環境因子と化学反応し、結果として摩耗が影響を受ける摩耗である。酸化摩耗(oxidative wear)は摩耗面や摩耗粒子が酸化し、その生成物である酸化物が以後の摩耗過程に影響を与えるが、これは乾式の腐食摩耗とみなされる。図-3・4 円すい突起によるアブレシブ摩耗モデル図-3・5 アブレシブ摩耗に対する硬さの影響図-3・6 アブレシブ摩耗形態図4―186腐食の影響には摩耗を促進させる場合と、抑制する場合とがある。腐食作用として材料損失を促せば摩耗量は増加する。この場合の摩耗モデルを図II-3・7に示す。相手面の突起と突起の接触間隔の間で摩擦面において反応が生じ、突起は反応生成物を除去する作用をし、したがって反応生成量が摩耗量となる。図II-3・7では反応速度が放物線則に従うと仮定している。一方、腐食生成物と表面の結合が強いと反応生成物は接触界面にとどまり、相手面との直接接触を抑え、固体潤滑剤の作用により摩耗を抑制する。その例は、境界潤滑における極圧添加剤や、高温大気中の緻密化された酸化物粒子層[高温ではグレーズ酸化物(glaze oxide)と呼ばれる]である。図II-3・8は鋼の摩耗の温度依存性であり、高温ほど酸化物の緻密化により摩耗が減少する。酸化物層の摩耗機構としてはその層の厚さが臨界厚さに達するとはく離するというプロセスが考えられている。図II-3・9は摩耗量と極圧添加剤の溶液濃度の影響であり、ある濃度までは反応性が進み摩耗を抑制するが、さらに反応が進むと逆に摩耗を促進する。積極的に化学反応を利用して摩耗を抑制する場合には腐食摩耗を化学摩耗(chemical wear)と呼ぶこともある。また、腐食摩耗を電気化学的側面から見ると、腐食摩耗は機械的摩耗量と腐食溶解量の和と考えられ、摩擦は新生面をつねに露出させる作用をする。摩擦面は活性となるために一般にはアノードとして作用し、周りの面がカソードとなる。摩擦面は通常の酸化膜で覆われた表面より腐食溶解速度が大きい。SUS 304鋼は静的腐食環境中では耐食鋼であるが、腐食摩耗中では摩擦により不動態被膜が破壊され、その特性が損なわれる。図II-3・10はSUS 304鋼の電位と摩耗の関係であり、電位とともに摩耗が増加するのは電気化学的に金属が溶解する量が大きくなるからである。一方、電位が-800mV(vs. SCE)以下では摩擦面もカソードとなり溶解は生じず、そのときの摩耗量は機械的摩耗量のみとみなせ、カソード防食の状態にある。
