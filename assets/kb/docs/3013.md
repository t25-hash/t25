# 3・2・4 フレッチング(fretting)

> **章**: 第3章 表面損傷
> **節**: 3・5 はコランダム研磨布に対する軟らかいホワイトメタルを基準に

フレッチングは、軸のはめあい部など、外的な振動により接触面で微小な相対滑り振動が起き、摩耗が生じる現象である。通常、微小な相対滑り振幅のおおよその値は全振幅で100 m以下とみなされている。フレッチングは、時にフレッチングコロージョン(fretting cor-rosion)と呼ばれるように、酸化摩耗が顕著である。これは滑りが小さいために、大気中において形成された摩耗粒子が接触界面から排出されにくく、金属粒子が次第に酸化され、摩耗痕内部および周辺に酸化物粒子が著しく堆積するからである。しかし酸化が抑えられる状況でもフレッチングが生じる。図II-3・11は滑り振幅と摩耗量の関係であり、ある振幅以下では摩耗量はほぼ零であるが、振幅とともに摩耗は増加する。フレッチングでは、損傷形態から滑り振幅は三つに分類され図-3・7 腐食摩耗のモデルと摩耗進行曲線る。ミンドリン(Mindlin)の理論により、全体の接線力が摩擦力より小さくても局部的には滑りを生じ、球と平面の接触ではリング状の摩耗痕となる。これをマイクロ滑り域、あるいは固着滑り域と呼ぶ。振幅が大きくなると接触面全域に損傷が広がるが、大きな塑性変形やき裂の発生を伴う混合滑り域になり、さらに振幅が大きくなり著しい摩耗が生じるとグロス滑り域となる。フレッチングの対策はこの損傷形態の違いを考慮しなければいけない。フレッチング疲労(fretting fatigue)は混合滑り域のフレッチングによる表面き裂を起因とした疲労現象である。図-3・8 摩耗の温度依存性図-3・9極圧剤の添加量と摩耗図-3・10 海水中におけるSUS 304鋼の摩耗と電位の関係
