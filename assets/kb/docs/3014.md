# 3・2・5 摩耗の評価方法および摩耗遷移

> **章**: 第3章 表面損傷
> **節**: 3・5 はコランダム研磨布に対する軟らかいホワイトメタルを基準に

a. 摩耗パラメータ摩耗(wear)は材料損失あるいは材料損傷である。摩耗量(wear volume)の評価方法は体積損失や質量損失で表示するが、質量で評価しても材料の密度を用いて一般には体積で表す。摩耗は時間(滑り距離)に対して逐次減量現象であり、その変化を摩耗進行曲線(wear curve)と呼ぶ。図II-3・12はその代表的な例であるが、摩耗の進行は材料や摩擦条件に依存する。摩耗進行曲線の傾き/を摩耗率(wear rate)と呼ぶが、摩耗率が滑り距離に対して図II-3・12の曲線1のように一定の場合もあれば、曲線2や3のように変化する場合もある。曲線1ではアブレシブ摩耗や接触面の変化が無視できる摩耗で見られる。曲線2において摩耗率の大きい領域を初期摩耗(initial wear)、後半の傾きが一定となった領域を定常摩耗(steady-state wear)と呼ぶ。初期摩耗は接触面がなじむ段階であり、比較的大きな金属粒子の形成と表面のあれが生じる。摩耗機構としては凝着摩耗が支配的である。その後、摩耗粒子の微細化と酸化が生じ酸化摩耗に推移する。これが定常摩耗である。曲線3は疲労摩耗やエロージョンなどで潜伏期が存在する場合である。摩耗量は条件に依存するために、摩耗特性を比較するときには摩耗率や比摩耗量(specific wear rate)、摩耗係数(wearcoefficient)を用いる。ここでは滑り距離(m)、は荷重(N)、は摩擦対の軟らかいほうのビッカース硬さ(Pa)である。摩耗進行曲線において傾きが変化する場合に、上式により摩耗率や比摩耗量を算出するときには滑り距離の選択に注意を要する。比摩耗量や摩耗係数は材料や機器システムの摩耗特性を比較するときのパラメータになる。図II-3・13は摩耗係数のおおよその範囲を示す。凝着摩耗(無潤滑下のシビヤ摩耗)では10~10の広い範囲の値をとり、二元アブレシブ摩耗が最も大きい摩耗を示す。比摩耗量が10mm/Nm以下が耐摩耗性材料の目安となる。図-3・11 フレッチングの振幅依存性図-3・12 摩耗の進行曲線4―187第II部トライボロジーb. 摩耗遷移摩耗機構が同じであれば、摩耗量は荷重や硬さの関数となるが、条件によっては摩耗機構が変化する。図II-3・14は比摩耗量の荷重依存性を示す。荷重が小さく比摩耗量の値が小さい領域をマイルド摩耗(mild wear)と呼び、荷重の増加に伴い比摩耗量が急激に増加した後の摩耗状態をシビヤ摩耗(severewear)と呼ぶ。大気中ではマイルド摩耗は酸化摩耗が主であり、シビヤ摩耗は凝着摩耗に起因する。この現象をマイルド摩耗-シビヤ摩耗遷移という。酸化現象から見るとシビヤ摩耗はその摩擦条件で酸化速度や酸化物の量、その機械的強度が不十分なため、摩擦界面で激しい凝着が起こることによる。図II-3・11に示した摩耗進行曲線においても初期摩耗から定常摩耗への変化も摩耗遷移現象の一つである。また図II-3・15は無潤滑の鋼の摩耗形態図(wear図-3・13 摩耗係数のおおよその範囲図-3・14 荷重による摩耗遷移図-3・15 鋼の摩耗形態図4―188機械要素・トライボロジー図-3・16 油潤滑下における鋼の摩耗遷移map)であり、六つの摩耗状態に分類している。摩耗形態図は図II-3・6や図II-3・15以外にもセラミックスやプラスチックなどの各種材料や潤滑下やフレッチングなどの摩擦条件に対して作られている。
