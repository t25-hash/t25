# 3・2・6 油潤滑下の摩耗

> **章**: 第3章 表面損傷
> **節**: 3・5 はコランダム研磨布に対する軟らかいホワイトメタルを基準に

多くの機器は乾燥状態よりも潤滑油を用いて運転される。それは摩擦を低減し、摩耗を抑制するためである。しかし接触面の潤滑状態により摩耗が生じる。油潤滑下では特別な摩耗機構は存在せず、摩擦条件に依存して3・2・1~3・2・4項の摩耗機構や3・4節の疲労摩耗が起こる。固体接触が生じない流体潤滑では摩耗を考慮する必要はないが、確率的には摩耗が生じ、また実際の機械では運転の開始・停止時に油膜の形成は期待できずに直接接触をしている。また境界潤滑では確率的に直接接触が起きており、摩耗が生じる。図II-3・9のように潤滑油中の極圧添加剤による固体表面との反応は摩擦を低減する効果をもつが、摩耗という立場からは腐食摩耗の概念と同じである。その摩耗の程度は図II-3・13に示すように、摩耗係数が10~10と大きく変化する。図II-3・16は油潤滑下の摩耗遷移を示しており、荷重の増加により摩耗機構が変化し、比摩耗量が増加する。しかし乾燥摩擦に比べて、遷移はなだらかである。摩擦面が何らかのきっかけ、例えば温度上昇に伴う油膜破断などにより良好な潤滑状態が維持できなくなり、厳しい溶着が生じ、表面が激しくあれたり、場合によっては摩擦面どうしが固着してしまう現象を焼付き(seizure)という。焼付きは流体潤滑膜破断に伴う混合潤滑への遷移や、境界潤滑膜や表面反応膜の破断に伴う固体摩擦への遷移など、摩擦が急に増大する潤滑状態の遷移現象と考えられている。図II-3・17に焼付き発生に至る過程と潤滑状態の遷移の関連を示す。焼付き発生には摩擦面の設計、作動条件をはじめ多くの要因が関係しており、偶発性も存在するため、普遍性のある焼付き発生条件または焼付き防止条件を示すことは本質的に不可能である。一方、現実には焼付き発生防止のための基準が必要であり、次に示す4条件がおもな焼付き発生条件として設計等に採用されている。
