# 3・3・2 臨界温度条件

> **章**: 第3章 表面損傷
> **節**: 3・3 焼付き

摩擦面材料と潤滑油によって決まるある臨界温度まで表面温度が上昇したときに焼付きが発生するという説で、現時点での焼付き条件の主流を成している。臨界温度としては吸着膜の配向喪失温度、脱離開始温度、すなわち転移温度や、表面膜の軟化温度が採用されることが多い。θは計算により推定することが一般的であるが、その値としては、真実接触部での実際の最高温度上昇ではなく、摩擦熱が見掛けの接触域全体に平均的に分布していると仮定して求めた平均温度上昇を採用することが多い。また、摩擦面間の接触状態、作動条件により焼付き発生温度の基準温度が異なるとことも指摘されている。例えば、作動条件が過酷であれば接触部で発生した局所的凝着がそのまま焼付きにまで進展しがちであるのでθが基準温度となり、転がり滑り接触におけるように摩擦面の接触時間が非接触時間に比べて極めて短い場合では、接触時に生じた局所的損傷が非接触時間に回復することが考えられるので、摩擦面全体が臨界温度に達することが焼付き発生には必要となり、θが基準温度となることが多い。
