# 5・1・1 材料の生体適合性（biocompatibility）

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

生体内で人工物を安全に使用するには，人工物と生体との適合性
について考慮を払う必要がある．人工物の適合性は多くの要因に支
配されるが，考慮すべきおもな事項には次のようなものがある．
a.材料に対する生体の反応 材料が生体内に埋植された場合
には，まず材料表面にタンパク質などが速やかに吸着するととも
に，組織からの出血により材料が凝血で覆われる．材料周囲には好
中球やマクロファージなど白血球系の細胞が現れ，傷害された組織
や細菌の分解，貪食を開始する．次いでリンパ球系の細胞も現れ，
免疫応答の初期段階が開始される．材料の周囲では，活性化したマ
クロファージの刺激により，間葉系細胞が増殖しコラーゲンなどの
細胞外マトリックスの産生が開始されるとともに，周囲組織より新
生血管が侵入し，肉芽組織が形成される．その後マクロファージな
ど炎症性の細胞は姿を消し，間葉系細胞は筋線維芽細胞に分化して
肉芽組織を収縮させるとともに細胞数が減少して，肉芽組織は線維
組織として再構築されていく．最終的には，材料は線維組織のカプ
セルにより包まれることになる．
ただし材料からのイオン溶出が多い場合や材料が抗原性をもつ場
合などには，材料周囲での炎症反応が継続するため，材料はカプセ
ル化されずに異物として排除される．またアパタイトなどの材料が
骨組織内に埋植された場合には，カプセル化せずに，アパタイト材
料と骨の細胞外マトリックス成分が線維組織の介在なしで連続する
場合もある．
b.材料の生物学的安全性（biological safety) 無機，金属，
有機材料などでは，材料から低分子量の物質が溶出して周囲の組織
に拡散していき細胞に為害作用を及ぼす．無機，金属材料では金属
イオンの溶出が問題であり，特にNi ，Cr ，Cu，Hg，Coなど重金
属のイオンの毒性はよく知られている．これは金属イオンが細胞内
外のタンパク質に結合し，酵素の不活性化や細胞膜の変化を招くこ
とによるものと考えられている．有機材料ではモノマや不純物の溶
出が問題になる．有機モノマ類は程度に差があってもすべて毒性を
もつと考えるべきである．例えばセメント使用型の人工股関節の手
術時に急激な血圧低下を生じて死亡する例が発生し問題となってい
るが，これは骨セメントが重合する前のメチルメタクリレートモノ
マの毒性によると考えられている．しかし，手術中に髄腔内の内容
物が循環系に流出し肺塞栓を起こすことや，骨セメント重合時の発
熱などを原因とする意見もあり最終的な結論は出ていない．またモ
ノマについては内分泌かく乱化学物質としての作用が報告されてい
るものもあり，極微量でも問題となる可能性があるので注意が必要
である．生体由来材料では材料の抗原性が問題になり，例えばⅠ型
コラーゲンであればコラーゲン分子の三重らせん構造の両端にある
抗原性の強い部分を酵素処理で除いたアテロコラーゲンとして利用
される．生体分解性材料では材料の分解産物が問題になっている
が，例えばポリ乳酸では材料の分解により生じる乳酸により周囲組
織が酸性となり，炎症を生じる可能性が指摘されている．
の生物学的な安全性を確認するために必要な生物学的試験につい
て，厚生労働省によるガイドラインが定められている．その内容に
は，細胞毒性試験，感作性試験，刺激性試験，急性全身毒性反応試
験，亜急性毒性試験，遺伝毒性試験，埋植試験，血液適合性試験，
および慢性毒性試験，発がん性試験，生殖/発生毒性試験，生分解
性試験などがあり，培養細胞や実験動物を使用する試験方法が規定
され，定量的な評価を行うことができるようになっている．

が，このような方法では滅菌ができない病原体が存在することは事
実であり，材料供給源であるドナーの選択が重要となる．このため
ヒトまたは動物由来，特にウシ由来の物質の取扱いについては厚生

金属などの人工材料では材料自体の感染性が問題になることはな
いが，表面に付着する病原体を死滅させるための滅菌処理は必須で
ある．現在利用されている滅菌法には，高圧水蒸気を用いたオート
クレーブ滅菌，高温を利用した乾熱滅菌，エチレンオキサイドガス
を利用したEOGガス滅菌，放射線を利用したγ線滅菌および電子
線滅菌などがあり，材料特性に合わせて選択利用されている．これ
らの滅菌操作により材料は必ず劣化するので，滅菌の影響はつねに
留意する必要がある．またEOG滅菌では材料に残留するEOGが
問題になることがある．生体分解性材料では，表面のみならず材料
内部までの無菌性が要求される．医療用具の無菌性を保証するため
の検証方法については，厚生労働省の基準が設けられている．
d.術後感染（postoperative infection) 健全な組織に細菌単
独で感染を生じさせるには極めて大量の細菌が必要であり，皮下組
織であれば組織1g当たり10万個以上を要するといわれている．
しかし人工材料があると，その1/500～1/5000の数の細菌で感染
が生じると考えられている．生体内に埋植された後でも，血行を介
した細菌感染が発生する可能性が排除できず，生体組織内に人工材
料が存在すること自体が感染に対する危険性を高めている．さらに
生体に挿入されたカテーテルやペースメーカなどの周辺で感染が成
立した場合には，いくら抗菌剤を投与しても挿入物を抜去しない限
り感染は治癒しないといわれている．人工関節手術などにおいては
クリーンルーム内で手術を行うなど，感染の防止に最大限の配慮が
なされているが，それでも1%ほどの術後感染が生じ，その治療に
は極めて難渋する．
生体分解性材料の場合には，最終的に材料がなくなった時点では
材料に起因する感染の危険性はなくなる．しかし分解に長期間を要
する場合では，分解が生じるまでは非分解性材料と同様の危険性を
もつ．
材料の周囲で感染が生じやすい原因としては，手術による侵襲の
影響や材料周囲で血行が障害され感染しやすくなっている可能性も
考えられるが，材料自身を原因とすることが大きい．材料が感染を
招く一つの機構は，材料表面に凹凸があり死腔ができる場合であ
る．これは組織からの浸出液で満たされ，血管はもちろん周囲組織
が入り込まない空間であるが，このような死腔は細菌の増殖に好適
となり，しかも細菌が生体の防御機構から逃れる環境となってしま
う．多孔性の材料では材料の内部にも死腔を生じる可能性があるの
で，特に注意を払う必要がある．材料の存在が感染を招きやすくす
るもう一つの機構は，バイオフィルム（biofilm）である．材料表
面に細菌が付着すると，細菌は細胞外に多糖やタンパク質を分泌し
ながら増殖していく．このため細菌のマイクロコロニーが材料表面
わが国においては，医療用具の製造（輸入）承認申請に際し，そ
を覆う分泌物の層の中に埋め込まれた状態で形成され，バイオフィ
ルムと呼ばれている．バイオフィルム中の細菌は，好中球やマクロ
ファージなどの食細胞から守られ，さらに抗菌剤に対しても抵抗性
を増すことから，いったん感染が成立すると難治性となる．カテー
テル，カニューレ，ドレナージチューブ，心臓ペースメーカ，人工
関節などで細菌によるバイオフィルム形成が生じると報告されてい
る．

c.感染性（infectiveness ) 生体に適用する人工物が感染源と
響を与えるわけではない．材料の形状（塊，多孔体，膜，メッシ
ュ，繊維）や大きさにより材料の生体為害性に差異が生じる場合が
燥ヒト脳硬膜によるクロイツフェルト・ヤコブ病の感染が発生し大
まず材料の大きさが細胞に貪食されるほど小さいかどうかが問題
になる．非分解性の材料で本来はカプセル化され生体内で安定的に
存在できる材料であっても，大きさが小さくなると貪食され，しか
も細胞内で分解されないので炎症反応が継続してしまう．よく知ら
れた例にアスベストがある．これはケイ酸マグネシウム塩を主成分
とする長さ10m，直径がその1000分の1程度の繊維状結晶であ
る．肺に吸入されるとアスベストはマクロファージに貪食されるが
分解はされず，アスベスト繊維を包み込んだ線維化巣を形成して肺
線維症を引き起こす．そして慢性的な炎症を生じて活性酸素を産生
させ，DNAを傷害して発癌に至るとされている．動物実験ではア
スベスト繊維の長さにより組織に及ぼす影響が異なるので，形状に
より体内での動態や刺激性が異なると考えられている．また人工関
節では，しゅう動部に使用されるポリエチレンが摩耗して発生する
粉が骨吸収を引き起こすことが知られているが，膝関節と股関節で
は摩耗粉の大きさと形状が異なり，生体への影響も異なるといわれ
ている．
材料が大きい場合には，その大きさ自体が問題になる可能性があ
る．例えばラットなどの小型の実験動物に比較的大きな材料を埋植
することにより，発癌する場合が知られている．これは移植物が周
囲組織に対して機械的な刺激を与えたり，あるいは循環障害をもた
らすなどの機構により生じるものと考えられている．ただしヒトに
おいては，このような機構による発癌は確認されていない．
f .材料の物理的生体適合性 埋植した材料と生体組織の力学
特性が適合しないことにより，問題を生じることがある．例えば骨
折治療用の金属プレートを使用した場合，骨の癒合が生じた後もプ
レートを除去しないでおくと，プレート近傍の骨量が減少すること
がある．これは骨への荷重がより剛性の大きいプレートを通じて伝
達されることから，プレート近傍の骨は免荷され，荷重の減少に適
応して骨量が減少することで生じるものと考えられている．このよ
うな現象はストレスシールディングと呼ばれており，人工股関節大
腿側コンポーネントの金属ステムや膝前十字靱帯の人工靱帯による
再建などでも観察されている．
g.生体内での材料の破綻 生体内に埋植された人工物は，移
植後しばらくは意図した機能を発揮することができるが，人工物の
劣化は進行していく．生体内で材料がカプセル化された後は，ほと
んど生体外での液中浸漬状態と同じはずであるが，金属材料の場合
は生体内での酸素分圧が低いため不動態膜の形成が悪いことから，
腐食疲労が発生することがある．特に人工物どうしが接触する箇所
においては，フレッチング腐食疲労が問題になる．また異種の金属
が近接して置かれる場合には電池を形成し，電気化学的な腐食を生
じることもある．無機材料では脆性が問題となる．強度的に十分な
余裕をもたせたつもりでも，生体内における荷重状況は十分に明ら
かではなく，予期しない破壊を経験することがある．有機材料で
は，荷重負荷による疲労はもちろん，低分子物質の溶出や脂質の吸
着による脆化，吸水による加水分解，結晶界面での酸化などさまざ
まな原因で劣化する．
材料の劣化が進むと，イオンの溶出や分解産物の発生などによ
り，材料のカプセル化という生体内で材料が安定して存在するため
の前提条件が崩れる場合がある．いったんこのような状態に至る
と，マクロファージの集合など生体側の反応が開始するが，そうな
るとマクロファージによる活性酸素の放出などにより材料の劣化も
ますます加速される．こうなると，時間をおいても改善する可能性
はほとんどなく，材料を除去せざるを得ない．
このように生体内で材料は，力学的因子，化学的因子，生物学的
因子の複合的な影響を受け，耐久性の評価は極めて難しい．現在の
ところ，ISO規格やASTM規格において，一部のインプラント製
品の疲労強度試験などが規定されているが，試験方法を定めたもの
が多く，製品が満足すべき耐久性についての標準化はまだ十分でな
い．
