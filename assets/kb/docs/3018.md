# 5・1・2 金属材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

金属材料は適度な強度と弾性があり，加工性に優れていることか
ら，古くから歯科治療や骨折の固定用に用いられてきた．口腔内で
使用される歯科材料と，生体内に直接埋植されるインプラント用材
料とでは，使用方法や安全性に違いがあるため，ここでは分けて取
上げることにする．
a.歯科材料（dental materials ) 歯科に用いられる金属材料
は，金合金（gold alloys ），銀合金（silver alloys ），コバルトクロ
ム合金（cobalt -chromium alloys ）に大別することができる．さ
らに1980年代からチタン合金や超弾性チタンニッケル合金
（super - elastic titanium- nickel alloys ）が使用されるようになっ
てきた．歯科においては，歯列矯正の場合はワイヤとブラケットと
呼ばれる金具など加工材を用いるが，その他は大部分精密鋳造によ
り作製されるため，鋳造用合金が多い．
金合金は歯科用金属としては最も歴史があり，現在でも重要な役
割を果たしている．金それ自体は軟らかいため，Ag，Cu，Pt ，Pd
などが合金元素として添加されている．金合金に関しては，JIS
T6116：2000（ISO1562：2004）に歯科鋳造用金合金の規格があ
り，力学的性質により使用目的別に四つのタイプに分けられてい
る．合金組成については規定がないが，表5･1には市販の歯科用貴
金属合金の代表的な組成と硬さについて示した．金合金は伝統的に
カラットで呼ばれることが多いが，金100%が24カラットに相当
する．白金加金というのは，白金を少量加えると優れた弾性を発揮
することから，強度と弾性が必要な部分用として白金加金と名付け
て市販されている．また陶材焼付用合金は，陶材を焼き付けるため
十分な強度が必要である．
銀合金は，JIS T6108に規格があり，金を含まない銀含有量60
mass%以上のものと規定されている．1種と2種に分類されてお
り，1種はインジウム（In）含有量5mass%未満のもの，2種はIn
含有量5mass%以上で白金族元素10mass%以下ということにな
っている．銀合金は金合金と比較すると低価格で，融点が低いため
手軽に使用できるという特徴があるが，耐食性があまりよくないう
えに，硫化により黒化しやすいために変色をいかに防ぐかが重要で
ある．
金銀パラジウム合金は，金12mass%，Pd20mass%，Ag40
mass%以上と，銀を主成分とする合金であるが，習慣的にこのよ
うに呼ばれ，国内では広範囲に用いられている．
