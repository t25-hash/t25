# 5・1・3 無機材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

a.概説 生体材料に分類されている数種類の無機材料（バイ
オセラミックス，bioceramics ）は比類のない生体親和性を有し，
インプラント材の開発に不可欠である．これらは2群に大別され
る．一つはハイドロキシアパタイトを中心とするリン酸カルシウム
類である．リン酸カルシウム類は脊椎動物の骨や歯などの硬組織に
見られる無機物質とほぼ同一の組成や構造を有し，顕著に生体組織
と反応する生体活性（バイオアクティブ）材料群である．他の一つ
はリン酸カルシウム類以外の生体親和材料群である．アルミナや部
分安定化ジルコニア，あるいはシリカ（SiO ）やリン酸分を主成
分とするガラスなどがこれに分類される．これらは従来は生体不活
性（バイオイナート）材料として分類されてきたが，長期間の動物
体内での埋入実験により生体組織との干渉作用があることが指摘さ
れてきているので，生体不活性という表現は必ずしも適切ではな
い．特にバイオガラスと呼ばれるNaやCaを含むリンケイ酸
ガラスは骨と結合することが報告されている．
無機材料が医用に供されるのは人工歯根や人工骨のような生体硬
組織の代替材料，あるいは骨補てん材のように骨修復材料が主であ
る．リン酸カルシウム類の水和硬化反応を利用して歯科用セメント
が実用化されている．また，セラミックスの機械的短所を補う目的
で金属やポリマと複合化したバイオマテリアルやハイドロキシアパ
タイトをコーティングしたバイオマテリアルも開発されている．
歯科用には長石や石英，陶土（AlO ・2SiO ・2HO）からな
る陶材が歯冠（ポーセレンジャケットクラウン）として供される．
陶材の特徴は優れた審美性で，天然歯に酷似した色調や光沢を有す
る．陶材の着色にはZnO，TiO ，FeOおよびFeO ，CoO，InO
などの酸化物が使用される．
b.リン酸カルシウム類
.ハイドロキシアパタイト（hydroxyapatite) ハイドロ
キシアパタイトの組成はCa (PO )(OH)(HAp）で表される．
結晶構造は一般的には空間群6 /mをもつ六方晶（＝0.9432
nm，＝0.6881nm）に分類される．六方晶アパタイト構造は
OHの格子位置にフッ化物イオンが配位した場合に最も安定化さ
れる．水酸化物イオンがフッ化物イオンよりも大きいために， 軸
に沿った副格子中ではOHはCaを含む面からややずれた位置
に配位している．空間群2 /をもつ単斜晶HAp構造（＝
0.9421nm，＝2，＝0.6881nm）の存在も報告されている．
HApは生体親和性以外にも触媒能やタンパク質の吸着性，センサ
応答性など重要な性質をもつが，これら諸性質の発現に格子水酸化
物イオンが重要な役割を果たす．また，特にバイオマテリアルとし
ては焼結プロセスを経て作製されるセラミックスが多く用いられる
ため，格子水酸化物イオンの熱的安定性を十分に理解しておく必要
がある．水和物や水酸化物は通常は100～300℃程度で脱水が起こ
り，分解するが，HApは上述のように高温で焼結できるので他の
水和物や水酸化物と比較すると非常に熱的に安定で，構造中に水酸
化物イオンを強固に保持している．従来は大気中の水蒸気分圧が零
でも1300℃以下ではHAp構造は分解しないと考えられていた．
しかしながら複素インピーダンス法による最近の高精度測定解析に
より，HApは500～700℃程度の温度域で徐々に部分脱水を起こす
ことが明らかにされている．脱水が進行し，分解が起こる場合は以
下のように表せる．
HAp→3Ca ( PO )＋CaO＋HO（↑）
あるいは
HAp→2Ca ( PO )＋Ca ( PO ) O＋HO（↑）
したがってHApセラミックスの焼成雰囲気に水蒸気を導入する
ことは製品の性質に影響を及ぼすことを念頭においておく必要があ
る．
.その他のリン酸カルシウム（calcium phosphate)
CaO- PO系にはHAp以外にも多種類のリン酸カルシウム化合物
が生体材料として用いられる．
リン酸水素カルシウムは，無水塩（モネタイト，DCPA）と二水
塩（ブルッシャイト，DCPD）として存在する．また，両者の中間
的な水和物も存在することが知られている．両者とも歯磨剤中の研
磨剤として使用される．
リン酸三カルシウム（TCP:tricalcium phosphate ）は現在まで
にα，β，γの3種類の多形の結晶構造が決定されている．α型は
高温相で1125～1430℃で安定であるが，室温でも準安定で容易に
保持できる．β型は低温相で1125℃以下で安定であり，天然に存
在する鉱物，ウィットロカイト［Ca (Mg,Fe )H (PO )］と類
似した結晶構造をもつ．γ型は高温高圧相で，約10GPa，1000℃
で得られBa ( PO )型構造をもつ．このうち生体材料として用い
られているのは，α相（α- TCP）とβ相（β- TCP）である．α-
TCPは水に対する溶解度が高く，また加水分解反応によりHAp
になり，高濃度であれば硬化体となることから，生体用セメントと
して用いられている．β- TCPも水に対する溶解度が比較的高く，
生体親和性に優れていることから，生体吸収性のインプラント材と
して，骨充てん材などとして用いられている．
リン酸四カルシウム（TeCP）はスラグ中で発見された物質で，
ヒルゲンシュトッカイトとも呼ばれる．TeCPはHApより塩基性
で触媒活性が高い．また水中でも室温では安定であるが，80℃以上
では加水分解する．バイオマテリアルとしては，酸性のTeCPと
組合わせて生体用セメントとして使用される．TeCPはモネタイト
と炭酸カルシウムの固相反応で合成される．固相反応は低水蒸気分
性がある．
圧下で行い，急冷しなければならない．徐冷すると，HApと酸化
カルシウムに分解する．
リン酸八カルシウム（OCP）は，Ca (PO )・0.5HO組成のア
パタイト構造に類似した層とCaHPO ・2HO組成のブルッシャ
イトに類似した水和層の積層構造よりなる．層間の結合はイオン性
であり層状粘土鉱物と比べるとはるかに強固である．OCPは板状
の大型の結晶が比較的簡単に水溶液系から得られることから，生体
用セメントのフィラーとして期待されている．
.生体用セメント リン酸カルシウム類の多くは水溶液中
では安定なHApへ加水分解し，石こうやセメントのような硬化を
伴いHAp多孔体となる．生体中での硬化体は生体骨に近い結晶性
を有する．これを利用して生体用セメントが開発された．これらの
セメントは生体中のカルシウムイオンやリン酸イオン種を組み入れ
適合性がよいと推定されている．
再石灰化するため，歯骨に類似の生体セラミックスとして歯科用あ
るいは医用のセメントや骨補てん材，再石灰化促進剤等へ応用され
る．また，自己硬化性を利用し骨形成タンパクの賦形材としても応
用される．
α- TCPは単独で転化反応を起こし硬化する．α-TCP粉末と硬
化液として水，生理食塩水，ポリアクリル酸，および生体内に存在
する有機酸の水溶液を用いたセメントが開発されている．TeCPと
るものと考えられている．
