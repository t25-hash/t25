# 3・3・4 熱的不安定条件

> **章**: 第3章 表面損傷
> **節**: 3・3 焼付き

発生した摩擦熱が吸収できなくなり、摩擦面での熱的平衡が破れると焼付きが発生するという説である。焼付きは摩擦面間の激しい凝着の結果として発生するわけであるから、焼付き防止のためには摩擦表面に保護膜を形成することにより摩擦面間の直接接触を防ぐことが肝要である。保護膜としては流体潤滑膜、境界潤滑膜、極圧添加剤などによる表面反応膜、酸化膜、さらにりん酸塩処理などの表面処理による表面被膜など、表面改質膜、などがある。しかし、これらの保護膜は熱的擾乱(軟化、変質、分解など)、機械的擾乱(破壊、摩耗など)により破断するが、保護膜の破断がすぐに焼付き発生に結びつくわけではなく、保護膜の局所的破断により生じた凝着部が摩擦過程により成長していく場合に焼付きが発生する。したがって、作動条件が過酷になり、保護膜の破断と再生・補修の間の平衡状態が破れ、保護膜の十分な補修ができなくなると焼付くことになる。図II-3・18に作動条件の過酷度と焼付き発生および摩耗率の関係を示す。表面温度、接触圧力、滑り速度などが増加することにより作動条件が過酷になると、焼付きが発生する。さらに作動条件が過酷になると相対滑りが不可能になってしまうほど激しい焼付き状態となる。一般に流体潤滑膜、表面反応膜をはじめとする保護膜は、乾燥摩擦状態に比べて接触状態を温和にし、軽摩耗域、すなわち焼付き発生に対する安全作動範囲を著しく拡大する。流体潤滑域における摩擦面間の直接接触の程度は、膜厚比Λで推定できる。Λ>3で摩擦面は流体膜によりほぼ完全に分離されているが、Λ<1になると荷重はもはや流体膜ではほとんど支持することができず、荷重の大部分は直接接触部で支持されるようになる。境界膜(吸着膜)は純機械的には容易にはく離しないのが普通であり、はく離の主たる原因は温度上昇である。境界膜、表面膜の破断には、それらを支えている下地金属の変形に大きく支配され、下地が大きな塑性変形を受けると、その変形に吸着層や表面膜が追随できなくなり破断が促進され、焼付き発生の危険性が増大する。表面の粗さ突起間の接触が弾性的か塑性的かは塑性指数Ψ により推定できる。図II-3・19に塑性指数と焼付きとの関係を示す。Ψ が小さく突起間の接触が弾性的であれば、焼付き発生温度は150~200°Cと高いが、Ψ が大きく接触が塑性的になると50°C程度の低い温度で焼付きが発生する。ただし、焼付きが生じるためには、塑性指数で考慮されている表面突起部での塑性変形ではなく、表面層全体にわたって塑性変形が進行する必要があるとの指摘がある。また、表面硬度が高いほうがΨ が小さくなり、凝着が生じてもその損傷範囲が硬度の低いものに比べて狭く、内部にまで及びにくいために損傷の程度は軽くなる傾向がある。しかし、片当たりなどにより応力集中が生じるとその緩和が起こりにくいため、かえって焼付きやすくなる場合がある。摩擦面材料としては、化学的に類似性の少なく、互いに固溶しにくい材料の組合せや、凝着部の成長を妨げやすい鋳鉄のような不均質な成分構成を有する材料が焼付き防止には有利である。また、図II-3・19より、表面を極めて滑らかに仕上げると、境界潤滑域ではかえって焼付きを発生しやすくなることがわかる。粗面では表面粗さの谷部が潤滑油の供給源の役目をすること、局所的な凝着部が滑りに伴い非接触状態にもなりうるが、平滑面では直接接図-3・18
