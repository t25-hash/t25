# 3・4・1 滑り接触における疲れ

> **章**: 第3章 表面損傷
> **節**: 3・4 疲労損傷

降伏応力以下の作用力の繰返しによって材料が破壊する現象を疲れ破壊(疲労破壊)という.滑り接触する固体表面層においても,軽負荷の繰返しによって疲労破壊損傷が現れる.この損傷には,大別して次の二つの形態がある.(1)疲労損傷(fatigue failure)が表面近傍で比較的小規模に継続して起こり,表面から次々に材料が離脱する.この損傷を摩耗機構の一つである疲労摩耗という.(2)表面の摩耗損傷は軽微であるが,疲労き裂が内層奥深くまで,さらには下地との界面まで進展して,材料が大きくはく離する.これらの起こりやすさは次に述べるように潤滑状態で異なっている.a. 無潤滑あるいは枯渇潤滑下における疲れこの条件下でのおもな疲労損傷の形態は(1)である.疲労き裂の発生点は表面もしくは内部にあり,摩耗粉の形状は微小な薄片状となりやすい.内部き裂の発生による疲労摩耗機構の説明としては,デラミネーション理論が知られている.一方,(2)の形態の疲労損傷は,無潤滑や枯渇潤滑下の滑り接触では摩耗あるいは焼付きなどの損傷が生じやすいため,ほとんど観察されない.b. 潤滑下における疲れ流体潤滑下における疲労損傷の多くは(2)に該当する.これは,静荷重下よりも動荷重下のほうが格段に起こりやすい.いずれも,負荷の強く働く箇所,すなわち最大負荷面や片当たりの大きい局部で起こる.これらの箇所では,一部油膜が破断して固体接触が生じ,いわゆる混合潤滑状態になりやすい.したがって,流体潤滑軸受の疲労対策では,最大油膜圧力のほかに混合潤滑状態になりうる油膜の薄い領域にも注意し,全体の当たり状態を考慮しなければならない.以下に裏金付きバイメタル軸受の疲労破壊の進展と形態,ならびにその機構について述べる.c. 疲れ破壊の進展と形態初期の疲労き裂は微小である.それは,軸受ライニング表面で滑り方向に直角,すなわち回転軸方向に発生し,ライニング層中を表面から内部に向かって進展する.時間経過とともに,き裂はその数と大きさを増す.内部に進行したき裂は裏金との界面に到達すると,ほぼその界面に沿って二次元的に伝ぱする.各所から伝ぱしたこれらのき裂が結合すると軸受ライニング層の部分的はく離に至る.き裂のライニング層内部への進展形態は,荷重点の移動方向によって異なる.軸心が上下に往復する動荷重では,表面に発生したき裂は面にほぼ直角に進展する.回転慣性荷重のように荷重点の方向が回転する場合には,荷重点の移動方向に沿い表面に対して斜めにき裂が伝ぱする.これらの実例を,それぞれ,図II-3・20(a)図-3・19焼付きに及ぼす塑性指数の影響4―19 疲れ破壊の機構軸受ライニングが疲労によってき裂を生じるのは,おもに機械的な応力の繰返しと熱の影響である.ここでいう機械的応力としては,衝撃的にかかる軸受荷重のみならず摩擦力が大きな意味をもつ.これは,動荷重を繰返し加えるに当たり,軸を静止させたままの場合と回転させた場合とで微視的き裂の発生までの繰返し数を比較すると,後者のほうが桁違いに小さいことから理解できる.すなわち,混合潤滑状態の負荷面で真実の接触機構を考えると,接触部には大きな摩擦せん断応力と熱応力が繰返されており,これらの応力が軸受ライニングに疲労破壊を起こさせるのである.熱の影響としては,軸受ライニング強度の温度特性,不均一温度分布による熱応力,ライニング組織内部の熱膨張率の違いや異方性などが挙げられる.微視的き裂が発生した後は,摩擦力の繰返しにより一定の方向性をもったき裂が微視的き裂を起源として発達する.ここで回転荷重のように荷重点が移動する場合には,転がり接触面の潤滑下における疲労はく離の進展と同様な機構が考えられる.すなわち,微小き裂内に侵入した潤滑油が荷重点の移動方向に沿ってき裂内に流体力学的な高圧を発生させ,それがき裂を斜めに進展させるというもので,このき裂内の高圧の発生とき裂の進展は荷重点の移動方向がき裂の進展方向に一致した場合に著しいと考えられる.
