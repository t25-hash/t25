# 3・5・1 軸受におけるキャビテーション

> **章**: 第3章 表面損傷
> **節**: 3・5 キャビテーションエロージョン

液体の圧力が蒸気圧以下に低下するとキャビテーション気泡が発
生するが，圧力が再び蒸気圧以上に回復すると気泡は崩壊する．そ
のとき発生する衝撃圧力や，気泡の分割に伴うマイクロジェット流
の衝撃圧力などが，材料表面の非常に小さい面積に高速で作用す
る．そのため，材料表面に局所的な塑性変形やピット，あるいは疲
労破壊による損傷が生じる現象を，キャビテーションエロージョン
（cavitation erosion）という．
滑り軸受に発生したキャビテーションエロージョンの例として，
中/低速ディーゼルエンジンやガソリンエンジンの軸受，発電用タ
ービン装置の高圧圧縮機軸受などが報告されている．流体潤
滑状態にあるジャーナル軸受やスラスト軸受の負圧部にキャビテー
ションが発生し，油膜圧力の回復する位置で気泡が崩壊しエロージ
ョンが生じる．また，内燃機関の軸受のように変動荷重を受ける軸
受では，滑り軸受の負荷側にもエロージョンが発生することがあ
る．このほか，給油経路で生じたキャビテーション気泡が，軸受の
4―192 β4編 機械要素・トライボロジー
（密度×音速）との関係
図 - 3･26 キャビテーションエロージョンによる質量減と液体のρ
油穴の周囲で崩壊してエロージョンを発生させた例もある．
高速転がり軸受では，潤滑油の流れに乱れが生じる場所でのキャ
ビテーション気泡の発生・崩壊によって，エロージョンが発生する
ことがある．
キャビテーション気泡の発生と崩壊に及ぼす液体の物性値として
は，液体の蒸気圧，表面張力，粘度，密度，音速などがある．図
Ⅱ-3･26は，種々の液中でのエロージョン量が液体の音響インピー
ダンス（密度と音速の積）と密接な関係にあることを示す．ま
た，液体あるいは溶質成分の蒸気圧が比較的高い場合には容易にキ
ャビテーションを発生するが，その被害は比較的小さく，蒸気圧が
低い場合にはこれと逆の結果となる．液体やその溶質成分が腐食作
用をもつ場合，例えば軸受材料に対して腐食性がある添加剤を含む
潤滑油中では，エロージョンが著しく進行する．
キャビテーションエロージョンに対する材料の強さは，引張り強
さや硬さなどの単独の機械的性質よりも，H / Eやσ /2E（ulti-
mate resilience）との間によい対応関係が得られる．また，軸受材
料の耐エロージョン性は，マクロ的な機械的性質だけでなく，成分
や組織に著しく左右される．
