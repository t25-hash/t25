# 3・6・1 軸受における電食

> **章**: 第3章 表面損傷
> **節**: 3・6 電食

電食は広い意味で電気化学的腐食または電解腐食の略として用いられる。また、電気鉄道では、レールからの漏れ電流などによる迷走電流が原因となって沿線のパイプなどに生じる腐食を電食と呼ぶことが多い。一方、転がり軸受では内部を電流が流れると、内・外輪の軌道面と転動体の間の薄い潤滑油膜を通してスパークやアークが発生し、その結果として、表面が局所的に溶融する現象を特に電食と称し、電気鉄道における車両用軸受では重要な損傷要因である。この現象は、損傷の程度は小さいものの蒸気タービンなどの滑り軸受にも発生することがある。電食痕は、噴火口状(クレータ状)のピットおよびその集合であり(図II-3・27)、転がり軸受の場合には洗濯板状のしま模様(リッジマーク、図II-3・28)が認められることもある。電食が発生すると機械的摩耗が促進され、発生した鉄分がグリースや潤滑油に混入して劣化を早め、発熱や保持器の摩耗・損傷につながることがある。滑り軸受では、軸受メタル面に多数のピットが発生し光沢のない梨地状の面が生じる例もあり、その場合、硬さが大きい相手のジャーナル面にも、程度は小さいが対応してピットが発生している。電食痕を発生させる接触面間の電圧は、荷重、回転数、軸受温度や潤滑剤などによって変化するが、1V 以下の低電圧でも電食痕を生じるといわれ、損傷を発生させない最低電圧として、瞬間的には0.5V、時間が長い場合には0.3V 程度が限界とされる。電食の大きさには、電流値を接触面積で除した電流密度が関係する。許容電流密度は、1.4A/mm以下という説もあるが、損傷程度には時間の要素も加味され、上記は500時間以内という条件のもとでの値で、さらに長時間の使用には0.7A/mmが限界ともいわれる。リッジマークに関しては、弱い電流が比較的長時間流れることによって、内・外輪の軌道面やころ表面に酸化被膜が生成し、転動で接触荷重が周期的に加わり、押しつぶされたり、はがれたりすることによってかなり規則的な凹凸が形成される。ただし、凹凸の高さは数m程度と小さい。このようなリッジマークを生じさせない限界電流密度は、経験上0.1A/mm以下とされる。軸受に電流が流れる電圧発生要因として、電動機軸受では鉄芯の磁気回路の不平衡によるもの、電動機巻線と鉄芯や磁気枠との間のコンデンサ作用および磁束の不平衡によるもの、電源回路の接地不良を原因とするものなどがあり、さらに電気鉄道の場合、変電所への帰還電流の通路であるレールの長手方向の電圧降下により前後車輪間で電圧が発生し、軸受に一部電流が流れることなどが挙げられる。電食を防止するためには、第一に電流が軸受を通過しないように接地回路を設けることであり、例えば鉄道車両の車軸の場合、軸端にカーボンがしゅう動する接地装置を設けることが多い。さらに、軸箱絶縁の方法を追加するケースもある。また、電動機軸受では絶縁軸受が用いられ、鉄道車両では、外輪外形部にセラミックス(ア図-3・27 玉軸受の外輪に発生した電食痕(ピット)る。図-3・28 円筒ころ軸受の内輪における電食(リッジマーク)4―19 3第II部トライボロジールミナ)を溶射してワニスを含浸したもの、または、樹脂を外形部に成型したものが適用され、さらに、転動体をセラミックスとした軸受の適用の可能性もある。
