# 3・7・1 フェログラフィー

> **章**: 第3章 表面損傷
> **節**: 3・7 損傷の検出と診断

ここでは,フェログラフィー(ferrography)の原理とその特徴について述べる.フェログラフィーとは,潤滑油中の摩耗粒子を分析し,摩擦部分で生じている異常を知るためにアメリカで開発されたシステムである.1970年ごろ発表されているが,1972年にはフェログラフィーに関する最初の論文が発表されている.その論文においては,1940年ごろから異常診断法として用いられてきた分光分析法(SOAP)の問題点を指摘し,フェログラフィーの有効性を説いている.フェログラフィーの原理は,強い磁力によって潤滑油中の摩耗粒子をガラス板上あるいはガラスチューブ内面に分離固着させ,その状態から摩擦部分の異常を診断しようというものである.固着した粒子の分析法によって,分析フェログラフィーと定量フェログラフィーと呼ばれる二つの方法に分けられる.分析フェログラフィーは,摩耗粒子の形状と大きさから摩擦面の異常を診断しようというものである.筒状の磁石の一部が長手方向に切り取られ,断面がU 字型に近い磁石が用意されている.その溝の上に少し傾けて薄いガラス板を置く.その上に試料油を流す.試料油中の粒子には磁力のほか,流体によって押し流される力と重力が作用する.これらの力がバランスしたところで粒子は沈着する.結果として,数十m以上の大型粒子は上流に,ナノメートルオーダの微細粒子が下流に沈着する.粒子が沈着したガラス板をフェログラムと呼ぶ.このフェログラムを2色顕微鏡と呼ばれる特殊な顕微鏡で観察する.粒子は約15種類に分類されており,得られた粒子がどの粒子に当てはまるかを判定することによって生じている摩耗の形態を類推する.定量フェログラフィーはDR(direct reading ferrography)とも呼ばれているが,その原理は以下に述べるとおりである.強い磁場の中に置いた細いガラス管内底面に粒子を沈着させる.粒子が沈着したガラス管の入口部分と少し下流部分をよぎるように光を当て,粒子の量に対応して変化する透過光を光電素子によって測定する.上部の読みを,下部の読みをとし,=-の値を求める.これを危険指数と読んで,摩耗の過酷さの目安としている.機械が正常に運転されているときの値を知っておき,その値から大幅にずれたときを異常の発生と判定するのである.フェログラフィーは潤滑油中の摩耗粒子を調べるという点においては,最初に述べた分光分析(SOAP)と同じであるが,SOAPは粒子を高温で燃やしてしまうため,粒子の形状がわからず,摩耗機構の判定が難しいという問題点がある.ただしフェログラフィーとSOAPを比べた場合,分析可能な粒子のサイズに関して両者の守備範囲が異なっている.SOAPがmかそれ以下の微細粒子を得意としているのに対し,フェログラフィーは数m以上から200mくらいまでの大形粒子を得意としている.このことから,もし両者ともに使うことができるならば,より的確な判定が可能とな4―19 4機械要素・トライボロジー
