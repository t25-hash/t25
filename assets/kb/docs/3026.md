# 5・2・1 ティッシュエンジニアリングによる医療のパラダイム

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・2 ティッシュエンジニアリング

人体の機能低下または機能喪失を補うためにこれまで金属，セライスを作製し，生体機能代替の医療用具として用いてきた．
急激な一方，代替物は体内の厳しい環境下でその機能を維持することは医療費の爆発的高騰のため，今後わが国では医療福祉レベルを維例えば，臓器移植は医療技術として広がりつつあるにもかかわらいという欠点がある．
このように細胞ソースは利点と欠点を併せ持っており，ケースによって適切なソースを選び取っていく判断が求められる．
細胞をどこから採取するという観点では，組織ごとに存在し分化しきった成熟細胞を用いるのが一般的であると考えられる．
しかしながら一方で，幹細胞をティッシュエンジニアリングへ応用しようとする方向性が強く打ち出されている．
幹細胞とは，自己複製能をもち，かつ多分化能をもつ細胞として定義される．
幹細胞の中でも体細胞分化の頂点にあり，いわゆる全能性をもつとされる胚性幹細胞（ES細胞）が注目されている．
胚性幹細胞は，分化を適切にコントロールすることができればいかなる細胞へも分化できる潜在能力をもつが，クローン胚が出現しない限り基本的に他家細胞であり，また受精卵に関わる倫理的問題を切り離すことはできない．
一方，ES細胞のような全能性はないが，比較的多くの細胞に分化することのできる能力をもつ骨髄性幹細胞（mesenchymal stemcell）が注目されている．
骨髄性幹細胞は骨髄に含まれており，自家細胞としての道も開けている．
骨髄の中には，血球に分化することのできる血液幹細胞のほか，中胚葉由来の細胞（例えば骨芽細胞，軟骨細胞，筋芽細胞，腱・靱帯細胞，脂肪細胞など）に分化することの可能な幹細胞が含まれていると考えられている．
さらに，組織中にあり，組織特異的な幹細胞（例えば神経幹細胞，表皮幹細胞，肝臓幹細胞，小腸上皮幹細胞，毛根幹細胞など）の存在も示唆されており，一部はすでに見いだされている．
これらの幹細胞はES細胞を除けば自家細胞として用いることは可能であり，また成熟細胞とは異なり組織形成能が高いと考えられており，ティッシュエンジニアリングにおいて有望な細胞ソースである．
生体組織を生体外で再構築する研究において最も進んでいる分野の一つが再生皮膚（正確には再生表皮）である．
皮膚はそもそも再生能が高く，日々新しい組織に置換わっていることや，再生皮膚を骨細胞の体外培養によって損傷部位をカセット的に置換することはする組織形成ができていないからである．
えば，体外での培養・再生不可能の代表と思われていた肝細胞につの生体外培養が可能になった．
さらに肝細胞の組織培養の研究が急スフェロイド状態での培養が行われている．
肝細胞だけでなく多種礎研究の進展によって可能となりつつある．
また，遺伝子導入，組向けて着実な進歩を遂げつつある．

この場合，本来体内で最も長い細胞周期をもつ細殖を誘起させているため，増殖後の細胞の軟骨細胞としての考える．
このように軟骨細胞そのものをソースとして用いる研究のミックス，ポリマなどの人工物や生体物質を加工した代用品でデバされることが必要条件となる．

リカなど諸外国への一方的な依存もこの先できないものと考えられる．
この問題を根本的に解決するためには臓器再生技術（organregeneration）の開発が必須である．
アメリカにおいては，代替臓器技術は世界規模での新産業領域であると認識され，きたるべき世紀における重要産業技術としてとらえ，戦略的取組みがなされている．
わが国においてもバイオテクノロジーおよび医療福祉産業の国際的な競争に対応するための方策が緊急に必要である．
これまで血管，皮膚などの単純な組織についてはプラスチックなどを用いた代替材料が開発されてきたが，性能および耐久性に問題がある．
さらに肝臓など高度な機能の臓器の代替は人工材料だけでは不可能である．
したがって臓器疾患の究極的な治療法として，ヒト細胞を用いた人工的な代替臓器の開発が必要である．
多種類の細胞を三次元的に組織化して培養し，モジュールを構成する技術を開発し，代替となりうる組織を創出するティッシュエンジニアリング（tissue engineering）の確立が求められている．
臓器移植は供給量の不足が懸念され，動物の臓器あるいはその臓器細胞の利用が当面の技術として考えられている．
しかしながら，未知の病原体の問題など未解決要素があり，根本的な治療としては自分の細胞を用いた臓器の再生以外は考えられず，その基盤的技術の整備が急がれる．
代替臓器の研究開発には近年大きな進歩が見られる．
例えば大口径血管（5mm以上）であれば人工材料で形成した血管を長期に利用できるようになった．
骨，歯なども人工材料を長期安定に利用できる．
また，培養細胞の利用についても皮膚の上皮細胞は口腔粘膜の組織培養により代替が可能である．
しかしながら，上記の成果はごく一部の成功でしかない．
人工血管も細いものは血栓が生じて使えず，心臓冠状動脈のように重要な部位の治療に使えない．
また，重度の熱傷などでは表皮だけでなく真皮層まで取り替えなければならない．
また，皮膚機能としても完全ではない．
軟骨細胞の移植はアメリカで企業化されているが，軟まだできていない．
軟骨細胞の培養自体には成功しても形態を再現歩は劇的な進歩を遂げており，上記の問題解決の可能性は高い．
例いても急速な進歩があった．
肝細胞の増殖因子が発見され，肝細胞速に進み，単に浮遊培養だけでなく実質細胞が100個程度集まったの細胞から増殖因子が見つかっており，動物細胞の生体外培養は基織培養技術も進みつつあり，ティッシュエンジニアリングの確立に工材料を用いて機能を代替しようとする試みと異なり，細胞を用い胞が基本となる．
ティッシュエンジニアリングで用いられる細胞の他人の細胞である同種または他家細胞（allogeneic cell），そしてられる．
自家細胞は，拒絶反応がない，ウイルスなどの問題がない場合が多い，などの欠点をもつ．
一方，同種細胞は，細胞のアクテが，拒絶反応を引き起こす，ケースによっては倫理的問題を引き起タのように遺伝子改変を施すことで移植用臓器または細胞としての界的に研究が積極的に展開されている．
したがって，ティッシュエは他家細胞移植を前提にして研究が進められている．
この場合，他植した再生皮膚は治癒とともに脱落することが期待されている．
一方，軟骨組織は再生能に乏しく，また体の比較的深部にあるこを用いる研究が主流である．
