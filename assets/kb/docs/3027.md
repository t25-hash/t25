# 3・7・2 非破壊検査(nondestructive inspection)

> **章**: 第3章 表面損傷
> **節**: 3・7 損傷の検出と診断

材料の内部き裂の発見や探傷に超音波やX 線が使われることが多いが,トライボロジーの分野では,関連するき裂や損傷の広がりが10ミクロンオーダと極めて小さいので,一般に用いられる探傷機器は必ずしも有効ではない.近年AE(acoustic emission)を用いて転がり接触面下の微小き裂を検出し,転がり軸受の残存余命を推定する手法も考案されている.これについては3・7・3項で述べる.トライボロジーの分野では,トライボ要素が多くの場合,潤滑下で使用されることから,潤滑油を分析することによって損傷に関する種々の情報を入手することができる.すなわち,潤滑油は使用中にさまざまな外部要因で変質して劣化するが,同時に機械設備などから排出される金属摩耗粉などのごみにより汚染される.潤滑油の劣化と汚染を調べる方法には,図II-3・29に示す,性状分析,組成分析,金属分析,の三つがある.密度,色,粘度などを調べる性状分析は比較的手軽であるため,現場の業務として行われる場合が多い.組成分析ではクロマトグラフィーなどの機器が使われる.金属分析では油の劣化と汚染を同時に検査するSOAP法(spectrometric oil analysis program)があり,原子スペクトル分析(発光分光分析,原子吸光分析,蛍光X 線分析)が採用される.なお,油中の摩耗粒子を磁気こう配で分離し,大きさの順に配列するフェログラフィーは金属分析の一種である.発光分光分析の一種ICP[誘導結合高周波プラズマ分光分析(inductively coupled plasma spectrometry)]とは高温で潤滑油を原子状に解離・励起して,放射スペクトルを得る手法であり,定量分析用に利用される.図II-3・30にICP法により分析できる元素を示す.潤滑箇所の材質と正常運転時のデータが明らかであれば,異常箇所の特定,原因の推定ができる.実例には船舶用潤滑油診断,製鉄プラントの油定期診断などがある.原子吸光分析は金属固有の波長の光吸収量を測定する方法である.ほかの分析法に比べ,極めて高感度な定量分析ができる.しかし,一元素ずつ測定するため,ICP法よりは分析時間が長くなり,試料の前処理も必要になる.蛍光X 線分析では試料に一次X 線を照射して発生する二次X 線(蛍光X 線)の分光で元素の種類と量を調べる.重元素の検出限界はppmの桁であるが,軽元素の検出感度は低い.実例として,油圧サーボポンプの油潤滑診断などがある.これらの潤滑油検査法では評価したい箇所の計測が可能であり,精度の高い情報が得られる.しかし,欠点として,サンプリングするための孔を必要な箇所に設けなければならない場合が多い.
