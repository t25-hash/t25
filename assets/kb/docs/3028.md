# 3・7・3 故障予知技術(failure prediction technique)

> **章**: 第3章 表面損傷
> **節**: 3・7 損傷の検出と診断

機械システムの故障はシステムを構成している機械要素に起因して損傷を生じ、その損傷がシステムの機能限界を超えるとき故障と判定される。故障予知技術は、損傷が機械システムの機能限界を超える時点を予測する技術である。機械システムの安全性確保ならびに生産効率の維持・向上の視点から、故障の発生を防止するかあるいは軽微にとどめ、メンテナンス計画にのっとって交換・修理を実行するうえで故障予知技術への期待が大きくなっている。故障予知技術における予測はデータベースに基づいて行われる。したがって、データベースの質が予測の信頼性を左右する。データベースは機械要素の損傷モードと使用条件、検出情報の種類と検出条件などに深くかかわるので、一般性に優れたデータベースを構築することが故障予知技術のキーテクノロジーとなる。言い換えると、使用状態における機械システムの損傷の進行と検出情報の経時的な対応関係がデータベースに要求される。故障予知技術において不可欠なデータベースは、公開されることが極めて少ない。公表されたデータベースの例を以下に記す。文献(65)は、国内の潤滑油分析(おもにフェログラフィー法)に関するデータをまとめたものである。軸受、歯車などの各種損傷モ図-3・29油分析法図-3・30 ICP により分析できる元素ードについて、損傷要素が組込まれていた機械、使用条件、収集されたフェログラムならびに診断結果が示された貴重なデータベースである。文献(66)(67)は、接触面圧を大きくした転がり疲れの加速実験で、AEと振動を観測して疲れクラックの発生から疲れ損傷の発現までの時間を測定したものである。疲れクラックの進展速度は、同一条件でも大きくばらつくうえ、荷重条件や潤滑条件などの影響を受けるので、分布として取扱われる。このデータベースは、疲れクラックの発生を検知して疲れ損傷の発現を予知する場合に有効である。文献(68)は、上記と同様な実験で、疲れ損傷の発現後それが進展して所定の振動値を超えるまでの損傷進展速度を測定したものである。このデータベースは、損傷の発現を許容するが損傷の進展で機械システムの機能限界を超えることが問題となる用途で役に立つ。クラック進展速度と組合わせたものも破損予知のうえで意味がある。上記のようなデータベースがほかの検出情報についても構築・公開されることが破損予知技術の今後の発展にとって肝要である。文献(1) Sasada,T.,Proc.
