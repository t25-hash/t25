# 4・1・1 トライボ材料の選定基準

> **章**: 第4章 トライボ材料
> **節**: 4・1 トライボ材料の種類と選定

トライボ材料(tribo-material)とは、トライボ要素の機能を発揮し、かつ目的に応じた低摩擦、耐摩耗、耐焼付き、耐疲労等の特性をもつ材料の総称である。トライボ材料には、上記の特性が潤滑油・グリースの存在下で得られるものと、それらの有無によらず得られるものとがある。それらをおもな特性により分類すると、表II-4・1,II-4・2のようになる。すべての特性に優れたトライボ材料は存在しないので、主として必要とされる特性が何かにより、目的とする機械要素に適したトライボ材料を選定することが必要である。潤滑油やグリースにより潤滑が可能な機械要素には、耐摩耗性や耐焼付き性、耐疲労性などの機能を付与するために硬化などの表面処理をした鉄系材料が一般に用いられる。アブレシブ性の硬質粒子を含む環境で使用するような特殊な場合には、SiCなどのセラミックスも用いられる。また、滑り軸受やメカニカルシールなどの特定の機械要素には、それらの用途に適するように作成された材料が用いられる。真空、高温などの、潤滑油やグリースが使用できない環境で用いられる機器用のトライボ材料としては、以下のような材料が用いら表-4・1潤滑油・グリースによる潤滑下で使用するトライボ材料(1) 低摩擦・耐摩耗・耐焼付き・耐疲労(a) 単体:鋳鉄、軸受鋼、ステンレス鋼、りん青銅、黄銅、SiC、ほか(b) 表面改質材:高周波焼入れ鋼、浸炭焼入れ鋼、窒化処理鋼、浸硫処理鋼、ほか(c) 複層材:Cu-Pb合金、ホワイトメタル、Al合金、ほか(d) 複合材:Fe-Cu-炭素など(含油軸受材)、炭素-フェノール樹脂(シール材)、ほか(e) コーティング材:Crめっき、TiN,CrN、ほか(2) 高摩擦・耐摩耗(a) 複合材:パルプ繊維-シリカ-炭素など(クラッチ用ペーパ摩擦材)、ほか表-4・2 無潤滑下で使用できるトライボ材料(1) 低摩擦(a) 単体またはコーティング材:グラファイト、MoS,PTFE、ナイロン、ほか(2) 耐摩耗(a) 単体またはコーティング材:ダイヤモンド、ポリイミド、ほか(3) 低摩擦・耐摩耗・耐焼付き(a) 金属系(イ)複合材:Cu-グラファイトまたはMoS,Fe-グラファイト、Fe-Cu(含油)、ほか(ロ)コーティング材:Au,Ag,MoS、ほか(b) 高分子系(イ)単体:ナイロン、フェノール樹脂、ほか(ロ)複合材:PTFE-炭素繊維、ポリイミド-グラファイト、フェノール樹脂-PTFE、ポリアセタール-PTFE-炭素繊維、ほか(ハ)コーティング材:PTFE-青銅、ほか(c) 無機系(イ)複合材:AlO-MoS、ほか(ロ)コーティング材:ダイヤモンドライクカーボン、ほか(4) 低摩擦・耐摩耗・耐アーク(a) 金属系複合材:Fe-Pbなど(パンタグラフ集電材)、ほか(b) 無機系複合材:炭素-Cuなど(ブラシ材)、ほか(5) 高摩擦・耐摩耗(a) 金属系複合材:Cu-ムライト-炭素など(ブレーキ材)、ほか(b) 高分子系複合材:Fe-フェノール-カシューダストなど(ブレーキ材)、ほかれる。真空環境では、金属と真空中で低摩擦を示すMoSとの複合材料や、金属基材にMoSをコーティングした材料がよく用いられる。空気中・高温環境で使用されるものには、耐熱性・耐食性のあるセラミックスであるBN などがある。低温環境中では、PTFEにガラス繊維のような強化材を充てんした高分子系複合材料がよく用いられる。酸・アルカリなどの腐食性環境中で使用されるものには、ステンレス鋼にMoSをコーティングした材料などがある。
