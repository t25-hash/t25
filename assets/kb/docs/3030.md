# 4・1・2 接触条件による選定

> **章**: 第4章 トライボ材料
> **節**: 4・1 トライボ材料の種類と選定

トライボ要素は,滑り運動,転がり運動または両者の混在する運動などの相対運動を行う可動エレメントによりその機能が発揮される.したがって,特に可動エレメントに対する外的条件である運動条件,荷重条件,内的条件である材質および微小形状を含む表面の幾何学的形状などの因子が相互に関連し合って可動エレメントの質的および動的挙動に影響を与え,結果としてトライボ要素自体の機能精度などを含む動的特性や耐久性に大きな影響を与える.トライボ要素の可動エレメントの幾何学的形状は,トライボ要素の種類によって設計上決まるので選択の余地はなく,トライボ要素での接触条件とは,狭義には可動エレメントの相対運動面における速度などの運動条件,相対運動面に作用する荷重条件という限られた因子を対象とすると考えてよい.しかし,ここでは,接触条件からトライボ要素の材料選択を検討するという立場から,接触条件を広く解釈することとする.このような観点から,接触条件としては,(1)幾何学的形状―相対運動に関与する可動エレメントの外形輪郭としての表面形状,および表面微小形状,(2)しゅう動条件―相対速度の種類および運動条件,(3)荷重条件―しゅう動面に作用する垂直力および接線力,(4)雰囲気(4・1・3項参照)―温度,湿度,大気状態,真空度,潤滑油の介在などの因子を考えることが必要である.可動エレメントには,接触条件にかかわる上記因子とそれらの大きさによって生じる具体現象,すなわち,弾性変形,塑性変形,粘弾性的変形,各種の損傷(破断,形状の永久変化,軟化・溶融,摩耗などの表面損傷),熱発生などが要素の性能に大きな影響を及ぼす.これらの現象は,物理的な因果関係により生じるもののほか,可動エレメントの材質と雰囲気との接触による化学的因果関係に起因して生じるものもある.これらの中で,例えばなじみなどのようにトライボ要素の良好な作動をもたらす現象や,適度な弾性変形などのように応力の過度の集中を回避する現象などもある.多くの場合,接触条件の厳しさにより生じるこれらの現象がトライボ要素に機能低下だけでなく機能喪失,耐久寿命の短縮をもたらす.材料選択の際には,相対運動面のトライボロジー性能が第一義的に検討されるが,接触条件によっては,固体材料の有する機械的性質として,(1)静的強度(引張強度,圧縮強度,曲げ強度,ねじり強度),(2)疲労強度,(3)クリープ,(4)破壊靱性,(5)硬さなどにも配慮することが重要である.トライボロジー性能に対するこれらの因子の影響に関しては,それぞれのトライボ要素の関連する項を参照されたい.一般に材料では力と変形(厳密には応力と変位)の間にはある種の関係があり,一方が決まれば他方は従属して決定される性質をもっている.トライボ要素の性能を検討する実用の場では,力(外力)と変形との間には複数の要素・因子が関与していることが多いので,その因果関係を数式で表示することは容易ではない.したがって荷重条件では可動エレメントの機能に対して力として影響を与えるのか,変形として影響を与えるのか,をそれぞれ個別に扱って材料選定の基準に使用しなければならない.例えば,滑り軸受や転がり軸受では外力に対する軸心の偏心量が重要となる場合があるが,外力を与えても軸心の偏心量は油膜挙動,固体の弾性変形,軸受を支えるケーシングの剛性などによって大きく影響されて一義的には決まらない.このような場合には直接偏心量の大きさに着目して材料選択は周囲条件などを考慮して総合的な観点から検討されなければならない.トライボ要素の機能が損なわれなければ,例えば弾性変形や時によって塑性変形さえ許容される場合がある.しかし,工作機械の主軸を支える転がり軸受などでは,弾性変形でさえ許容範囲を超えると回転精度を阻害して加工精度を低下させるので,避けなければならない.トライボロジーに関する性能と外力に対する強度を両立させるという観点から,主として表面のみにトライボロジー性能の発揮を期待し,強度などに関しては母材の機械的性質に依存するという設計方針から複合材または積層材を採用して目的とする性能を確保する場合も多い.
