# 4・2・1 金属材料(metallic material)

> **章**: 第4章 トライボ材料
> **節**: 4・2 硬質材料(hard material)

金属材料の耐摩耗性は一般に硬さと靱性に依存するが、これらの性質は必ずしも両立しないため、用途に応じてバランスをとっている。硬質金属トライボ材料を使用する代表的な機械要素には、転がり軸受ならびに歯車があり、それぞれJIS で規定された合金鋼を一般的に使用している。転がり軸受は軌道面と転動体との小さな接触面で大きな荷重を受けながら高い精度を保って回転する必要があるため、硬さ、耐摩耗性、ならびに転がり疲れ強さが求められている。1%C-1.5% Crの高炭素クロム軸受鋼(SUJ 2)が最も多く使用されており、完全硬化処理が施されている。高炭素クロム軸受鋼の成分は古くから変わっていないが、製鋼技術の発展とともに鋼中の不純物量は飛躍的に減少し、転がり軸受の寿命はこの30年間で30倍以上に向上した。耐衝撃性が必要な転がり軸受では、表面から適当な深さまで浸炭して高い硬さとし、比較的硬さの低い芯部を形成した、硬さと靱性を兼ね備えたニッケル-クロム-モリブデン鋼(SNCM 815)などの肌焼鋼が使用されている。耐食性が求められる場合には、マルテンサイト系ステンレス鋼(SUS 440C)などが使用されている。200°C以上の高温で使用される転がり軸受では、高速度鋼(X 75WCrV 18-4-1)や耐熱浸炭鋼(81MoCrV 42-16)が使用されている。表面のみを硬化させる方法として高周波焼入れが利用されており、大形軸受や自動車のホイールユニット軸受などでは機械構造用炭素鋼(S 55C)が用いられている。歯車には歯面の耐摩耗性および転がり疲労強度とともに歯車がかみ合う力を許容できる歯元強度が必要である。一般的な炭素鋼を調質して用いる場合や、焼入れして硬さを高くして使用する場合がある。焼入れ鋼としては、各種クロム鋼(Cr鋼、Cr-Mo鋼、Ni-Cr-Mo鋼)を使用している。ブレーキも重要な機械要素であるが、自動車や在来線の鉄道車両のブレーキのロータやディスクには片状黒鉛鋳鉄(FC250,FC280)が、新幹線では耐熱亀裂性に優れた鍛鋼ディスクが使用されている。粉末ハイス(SKH 10)は熱間静水圧焼結(HIP)技術を利用して製造されるが、W,Mo,V,Crなどを含むため耐熱疲労性や耐摩耗性に優れる。焼結金属の応用例として、材料自体の摩擦係数が低く耐摩耗性を有する含油焼結合金や固体潤滑剤複合焼結合金がある。含油焼結合金にはFe系やCu系があり、気孔部に潤滑油を含浸させる。真空中や高温雰囲気などの潤滑油が使用できない場合は、焼結合金にMoSあるいはWSが固体潤滑剤として配合される。4―19 8表面のトライボロジー特性を改良する技術の一つに金属溶射処理がある。溶射は溶融、微細化された金属を母材に高速で吹付けて被覆層を形成させる技術である。耐摩耗性ならびに耐焼付き性の改善を目的に、Ni-Al,Mo、自溶性合金(Ni-Cr-B-Si-C)などが使用されている。
