# 4・2・2 非金属材料(non-metallic material)

> **章**: 第4章 トライボ材料
> **節**: 4・2 硬質材料(hard material)

トライボ材料として用いられる硬質非金属材料としては、窒化けい素(silicon nitride)、炭化けい素(silicon carbide)、ジルコニア(zirconia)、アルミナ(alumina)などのセラミックスや、硬質炭素材料が挙げられる。大気中や真空中においては、セラミックスの摩擦係数は1.0を超えることはほとんどないが、特別な低摩擦材料ではない。一方、水中においては、ある滑り速度以上では、窒化けい素や炭化けい素の摩擦係数は0.01以下の極めて低い値を示す。このように水中で低摩擦を示すことから、炭化けい素は水ポンプ用のメカニカルシールなどに使用されている。セラミックスは、硬質という特徴から耐摩耗しゅう動材料として期待され実際にも用いられているが、脆性に起因した激しい摩耗がしばしば問題となる。耐摩耗しゅう動材料としてセラミックスを用いるために、脆性破壊型摩耗の発生領域を表した摩耗形態図が提案されている。代表的な硬質炭素材料としては、炭素繊維強化炭素材料(C-Cコンポジット)が挙げられる。C-Cコンポジットは、高速での摩擦・摩耗特性に優れることから、航空機や自動車のブレーキシューなどに利用されている。また、RB セラミックス(RB ceramics)という硬質多孔性炭素材料も開発されている。RB セラミックスは、無潤滑下でも摩擦係数が0.13,0.17と低摩擦を示すこと、耐摩耗性に極めて優れることなどから、無潤滑滑り軸受などのしゅう動材料として用いられている。セラミックス被膜やダイヤモンドライクカーボン(DLC)膜などの硬質被膜材料もトライボ材料として用いられている。
