# 4・3・1 金属材料(metallic material)

> **章**: 第4章 トライボ材料
> **節**: 4・3 軟質材料(soft material)

金属系滑り軸受には、滑り合う鋼製の相手軸に対して一般的にはより軟らかい非鉄合金がおもに適用される。焼付きにくさやなじみやすさなどの特性とともに荷重を支える強さ、硬さを必要とし、軸受合金は硬質相と軟質相とが混在した組織で構成されたり、鋼裏金付きの複層構造とされることが多い。それらは、薄肉でコンパクトな設計ができるため内燃機関用軸受や各種円筒ブシュとして潤滑条件下にて適用される。一般産業機械などでは特殊な用途や形状に対して軟質材料のソリッド軸受(裏金なし)も適用され、また各種金属中に黒鉛などの固体潤滑剤を多量に含有した自己潤滑軸受もあ表-4・3代表的な滑り軸受用金属材料の組成軸受材料化学組成wt.%る。表II-4・3に代表的な滑り軸受用金属材料の組成を示す。(1) ホワイトメタル:バビットメタルとも呼ばれ、すず基と鉛基の合金がある。軟らかく順応性に優れ、大形の舶用内燃機関やタービン、コンプレッサなどの回転機械用軸受として使用される。(2) 銅合金:ケルメット(Cu-Pb)や鉛青銅(Cu-Pb-Sn)は疲労強度が高く、ディーゼルエンジンなど高荷重の内燃機関用軸受に適用される。焼入れした硬軸が推奨され、軸受表面には軟質で薄いオーバレイを設けて使用される。また鉛青銅は境界潤滑下における各種ブシュとしても多用されている。さらに強度や耐摩耗性、耐食性などが必要な用途については青銅、りん青銅、アルミ青銅などの高強度な銅合金が適用される例もある。(3) アルミ合金:アルミ合金軸受は銅合金軸受に比べて軟らかいことや耐食性が良いことから、通常オーバレイを設けず使用され、耐焼付き性や耐摩耗性の優れるAl-Sn-Si軸受が乗用車用ガソリンエンジンや小形ディーゼルエンジンなどの内燃機関に適用される。一方舶用などの大形内燃機関の軸受でホワイトメタルの負荷能力に限界がある場合には、40%すず入りの軟らかいAl-Sn軸受が使われている。(4) オーバレイ(表面層):オーバレイはおもに銅合金軸受の表面に10~30 mの厚さに電着法にて施され、軸受のなじみ性、異物埋収性や耐焼付き性を向上させる。成分としてはPb-Sn-Cuが代表的であり、高荷重、高速条件においてはPb-Sn-Inを使用する例も多い。
