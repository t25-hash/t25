# 5・3・6 生物を利用した計測

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・3 生体計測

生体は，それ自体を計測システムとみなすことができる．マクロ
なレベルでは，個体が周囲環境に応じて行動したり，個体集団が環
境変化に対応して行動したりする．ミクロなレベルでは，生体高分
子が周囲の環境条件に応じて立体構造を変化させたり，細胞が周囲
の特定の物質や環境に反応して変形・移動したり ，物質を分泌
したりする．
前者に関しては，地震の予知にナマズや鳥類の行動を役立たせよ
うとする研究がなされる．後者に関しては，研究成果がバイオセン
サ開発などの応用分野へと発展している．
バイオセンサは，生体を計測システムに組込むものの一つである
（図5･44）．生体には，外界から情報を識別して選択的に受取り，
その情報を変換し，内部に蓄積し，外界へ情報を発信するなどの働
きがある．例えば，細胞のレベルで考えると，生体膜には，タンパ
ク質や糖質の高分子が配列され，接触する物質・周囲の流れ・変形
などの入力に対するセンサの役割を担っている．出力としては，電
気信号・化学物質の分泌・形態変化・増殖・細胞どうしの凝集など
がある．
さまざまな信号は，電気信号に変換されれば，電気回路の技術に
よって計測システムを楽に実現できる．電子回路基盤上に，生体高
分子，酵素，微生物などを固定化したセンサが作られるのはそのた
めである ．固定化においては，いかに生体高分子などの立体構
造を有効に作用できるような状態に保つかが課題となる．抗原抗体
反応に代表される特異的な反応のためには，その鍵となる構造が生
かされた形で固定される必要がある．適切な高分子を間に挟んで固
定化することによって，電気化学的平衡を利用して，反応基の位置
や向きを制御するなどの工夫が考えられる．
温度や水溶液中の水素イオン濃度をはじめとする周囲の物質の状
態などの環境条件もセンサの反応に影響を与えるので，周囲に余計
な物質が近づかないように，多孔質膜などを配置する．これは，多
孔質膜が選択的に分子などをふるい分ける効果をねらったものであ
る．
生体機能の測定法が進歩すれば，死の判定方法の問題点が改善さ
れる可能性がある．現在の脳死判定は，以下のように記述されてい
る ．(１)深昏睡，(２)自発呼吸の消失，(３)左右瞳孔の直径が4
mm以上，(４)脳幹反射（対光反射，角膜反射など）の消失，
(５)平坦脳波，(６)6時間経過後に(１)～(５)の状態が再度確認さ
れる．なお，以上の基準は，6歳未満の小児，急性薬物中毒，低体
温，代謝・内分泌障害の場合には適用されない．判定の困難さは，
バイオセンサ（biosensor ）は，生体反応を利用するので，糖，
アミノ酸などの生体由来物質の検出に向いている．微量物質の検出
にも有効であるが，反応時間が長いことも多く，計測時間の短縮が
課題となる．
赤血球をマイクロカプセルと見れば，流れの中でのせん断応力セ
ンサとしての応用が考えられる．また，血管内皮細胞は，血管壁付
近の流れに応じて物質を分泌している．これを利用すれば，せん断
流センサとしての血管内皮細胞の応用が考えられる．感覚器細胞を
利用した計測システムも開発される可能性がある．胎児幹細胞や体
性幹細胞からの分化制御，細胞増殖の研究の進展によって細胞の応
用範囲が広がると思われる．
