# 4・4・1 物理的表面処理(physical surface treatment)

> **章**: 第4章 トライボ材料
> **節**: 4・4 表面処理(surface treatment)

トライボロジー特性を改善するための物理的表面処理としては、焼入れ、溶射、物理蒸着が代表的なものである。一般に耐摩耗性や疲れ強さが要求される部材には、その表面に浸炭焼入れや窒化、高周波焼入れなどの表面硬化処理が行われる。しかし、浸炭焼入れ、窒化は加熱領域が部材の全域に及ぶ。一方、高周波焼入れは高周波誘導電流のもつ性質を利用して鋼の表面をジュール熱により急速加熱後急冷し、表面硬化を行う。炎焼入れによる表面硬化は同様に表面を急速加熱急冷する方法であり、廉価で手軽に処理できる。また、レーザや電子ビームなどの高パワー密度エネルギービームによる表面焼入れも熱処理ひずみが小さいこと、局部的な焼入れが可能なことなどの利点を活用し実用化が進められている。溶射は、溶融あるいはそれに近い状態にある金属、セラミックスなどの粒子を連続して表面に吹付け被膜を形成する方法である。比較的厚い膜を形成できる。被膜は多孔質であり、緻密性に欠ける。プラズマ溶射が代表的な方法であるが、その性質は溶射材料の粒子の基材面への衝突速度で特徴付けられる。一般にこの速度が大きいほど形成された被膜は緻密になり、付着強度も高くなる。被膜と基材の境界では、衝突した粒子や粉末は基材表面の凹凸部にかみ込み、おもに機械的結合を介して密着している。このため一般的に付着力が低い。各種金属、セラミックスなどの材料を形成可能であ4―19 9第II部トライボロジーり、その熱影響が少なく、下地の変形も小さい。このため溶射被膜は耐衝撃性に劣るが、耐摩耗性を著しく向上できる場合が多い。真空もしくは所定のガス雰囲気中で薄膜材料を蒸発させて成膜する気相法を総称して物理蒸着(PVD : physical vapor deposition)と呼んでおり、多くのトライボロジー用途で実用されている。蒸発方式、プラズマ発生方式、基板処理方式などの違いから、真空蒸着、イオンプレーティング、スパッタリングの三つの基本的成膜法に分けられる。さらにこれらの方法を基にそれぞれの特徴を生かし、各種の成膜法が開発されている。例えば単に物理蒸着のみではなく、反応ガスを導入して反応生成物として化合物や合金を形成する反応性PVD プロセスが開発され、反応性真空蒸着、反応性イオンプレーティング、反応性スパッタリングと呼ばれている。物理蒸着法では一般に粒子の入射エネルギーは、表面処理層の深さに関係し、基材との付着力に影響を与える。形成される層の組成なども、処理法によって変化する。また、新しい表面形成法の開発および改良が進められており、それらの技術を適用することによりトライボロジー特性が大幅に改善できる可能性も大きい。
