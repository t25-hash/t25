# 4・4・3 そのほかの表面改質(surface modification)

> **章**: 第4章 トライボ材料
> **節**: 4・4 表面処理(surface treatment)

しゅう動面の摩擦や摩耗を低減するための工夫が行われている.これらは改質による品質が前述の物理的あるいは化学的表面改質より劣るが,おおむね低コストである.また成分や組成の自由度が高いので,改質したい表面への施工条件を実験的に求めていくことにより,十分実用効果が得られる場合がある.a. 固体潤滑剤を表面に付着させる改質. 塗布固体潤滑剤粉末を溶剤に懸濁させ,刷毛(はけ)でしゅう動面に塗布し乾燥させて使用する.スプレーやディップ(浸漬)する方法も簡易である.MoSなどの固体潤滑剤が用いられ,乾燥後にしゅう動面に固体潤滑剤が残る.ペースト状にしてしゅう動面に指や箆(へら)で刷り込む方法もある.また耐高温用としてのBN や複酸化物系粉末も水に懸濁させてしゅう動面に塗布する方法を用いる.. 焼成刷毛(はけ)やスプレーでの塗布膜を,炉中で焼結して作成する.焼成のための結合剤として,無機系(けい酸ソーダなど),有機系(高分子など)材料が用いられる.. インピンジメント高速で固体潤滑剤を衝突させて表面にめり込ませる方法がある.ほとんど寸法変化はない程度の微量の付着であるが,塗布などの施工が困難な場所やはめあい面で,改質による寸法変化を嫌う場所に適している.また耐粉体エロージョン対策として,あらかじめ衝突する粉体や異物を表面にめり込ませる方法もある.めり込みやすくするために耐エロージョン材として硬さが小さい高分子材料を用いる場合が多い.また固体潤滑剤や金属などの粉体の中で対象物を揺すり表面に反応付着させるタンブリングもあり,球へのコーティングやメカニカルアロイングに利用される.b. 潤滑性を向上させるための表面の改質. 塑性加工表面に塑性変形を生じさせる程度の大きさの面圧を発生させ,材料の加工硬化能で表面硬さを大きくする.表面粗さの小さい型に押付ける方法,ローラにより低速で表面を転造する方法,微小な鋼球を吹付ける(ショットピーニング)方法,などがある.表面改質層の硬さ上昇や圧縮残留応力の増加により,しゅう動面の耐摩耗性が向上する.ただ過剰に行うとしゅう動面形状が崩れ,しゅう動部品としての精度が低下するので注意を要する.. なじみしゅう動面のなじみは,表面粗さレベルの摩耗と塑性変形で得られる.実際のしゅう動条件と異なる条件で,初期に表面状態を改善していく.機械部品を良好に作動させるための一種の表面改質である.なじみ過程での表面あれを防ぐために,なじみ期間だけ固体潤滑剤をしゅう動面に施したり,微量の摩耗を促進させるための潤滑油(なじみ油)を供給する場合もある.. 微細溝境界潤滑面は表面粗さオーダの微小な溝やディンプルを施すことによって,耐摩耗性や耐焼付き性が向上する場合がある.微小な表面プロファイルをコントロールするためには,機械加工が困難な場合も多く,フォトエッチングなども活用する.. 超仕上げ表面粗さを鏡面にする加工法で一般には砥石を軽く押し付けてしゅう動する.しゅう動面間に微細砥粒を介在させるラッピング,軟らかい物質で擦りあげることにより表面の微小突起を除去するバフ研磨の方法もある.また鋳物砂などの硬い粉体の中に対象物を入れ,一緒に揺することによって不要な突起物をとるタンブリングもある.いずれの方法も微小ではあるが初期の形状を崩すことになるので,形状管理に注意を要する.なおSi系セラミックスを水などの環境でしゅう動させるトライボケミカル反応の利用も鏡面を得る方法として有効である.文献(1) Tomizawa,H.and Fischer,T.E.,ASLE Trans.,30-1(1985),41.(2) 佐々木信也,潤滑,33-8(1988),620.(3) Kato,K., Wear,136 (1990),117.(4) Hokkirigawa,K., Wear,151 (1991),219.(5) Hokkirigawa,K.,Proc.
