a.人工心臓（artificial heart )
.種類と機能 人工心臓は，自然心臓の機能の部分的ある
いは完全代行を，一時的あるいは半永久的に行う装置と定義されて
いる．人工心臓には，補助人工心臓と全置換型人工心臓がある．補
助人工心臓は自然心臓を残したまま左心または右心，あるいは両心
を補助し，全置換型人工心臓は自然心臓を取り去り，完全に置換す
る．

また人工心臓はその流れの形態から拍動流ポンプと定常流ポンプ
に分類される．拍動流ポンプは自然心臓に近く生理的である．一方
定常流ポンプは小型にできる特徴がある．
.構造 全置換型人工心臓および補助人工心臓システムの
一般的な構成を図5･50に示す．システムは，体外と体内間のエネ
ルギーや情報の伝送を行う経皮的エネルギー・情報伝送システム，
人工心臓（血液ポンプとアクチュエータ），制御装置，アクチュエ
ータ内の空気容積変化を補償する容積補償器，電池からなる．
(１) 完全埋込み型拍動流血液ポンプ（pulsatile blood pump）
アクチュエータとして多くは直流ブラシレスモータが使用され
る．モータの出力パワーはモータ速度に対して放物線となる．した
がって，モータ出力が与えられると，その出力を与える二つのモー
タ速度（低速と高速）が得られる．ところで，モータの効率は，モ
ータ速度が高速になるほど大きくなるから，モータは高速で使用す
る必要がある．したがって高速回転するモータと低速往復運動する
ポンプ（拍動流血液ポンプ）とを結合するのに，減速機構と運動変
換機構（回転-直線運動変換）の二つの機構が必要となる．この機
構の違いにより，多くの種類の人工心臓が開発されている．
流体を使用するものは，モータの高速回転にて定常流ポンプを駆
動し，低速一方向の流体の流れを作り，これにて血液ポンプを駆動
する．モータを正逆転して左右の血液ポンプ間を液体が往復運動す
る方式（摩擦ポンプを使用) や，モータは一方向回転で他の機
構により流れの方向を変える方式（流れの方向変換にモータ弁を使
用) がある．任意の構造の血液ポンプと組合わせる，作動流体
をベアリング等の潤滑剤としても使用できる，などの特徴がある．
ボールねじやローラねじを用いる方式では，モータの高速回転を
ねじ軸の低速往復運動に変換し，血液ポンプを駆動する．一つの部
品で減速と運動変換の二つの変換が行える特徴がある ．また
低速度一方向回転モータにてカムを駆動し，血液ポンプを駆動して
いるシステムもある ．さらに，電磁石とばねでプッシャプレー
トを直線往復運動させ，血液ポンプを駆動しているシステムもあ
る ．
波動型血液ポンプでは ，高速一方向に回転するモータ軸に円
板を斜めに取付けて円板の揺動運動を作り，新しい型の定常流ポン
プを作っている．ポンプの出口には人工弁を取付けている．モータ
を間欠的に駆動し，拍動流を得ている．
(２) 定常流血液ポンプ（rotary blood pump）
モータにより羽根車を回転させて一方向の流れを得るポンプで，
小型で構造が比較的簡単である特徴がある．しかし，羽根車を高速
で回転するため溶血の問題や，羽根車とモータとの結合部における
血栓形成の問題があり，結合法により各種ポンプが開発されてい
る．遠心性ポンプでは，永久磁石を埋め込んだ羽根車を磁気浮上さ
せシールの問題を解決したものや ，羽根車との磁気結合を用い
てシールの問題を解決し，羽根車をピボット軸受で支持するものが
ある ．羽根車とモータ軸を直結する型の遠心ポンプでは，リ
ップシールとシール部への少量の流体流れにより軸シール機能を保
持し，シール部での発熱，血栓構成を防いでいるものがある ．
軸流ポンプでは，磁性流体シールを用いてモータと血液をシール
するもの ，磁気結合によりシールをなくし，ピボット軸受を使
用し，永久磁石を埋め込んだ羽根車を保持・回転するものがあ
る ．羽根車を磁気浮上させるポンプも開発されている．
振動流ポンプは磁気力による管の振動により流体を振動的に移動
させるもので，シールを必要とせず，任意の高周波振動流を出せる
特徴がある ．
b.人工心肺（heart -lung machine )
.適応 正常に拍動している心臓を切開すると大出血をき
たすので，この状態で心臓の内部に手術操作を加えることは不可能
である．そのため，開心術を行うためには心臓と肺の血流を遮断し
て心臓腔内を完全に空虚な状態にすることが必要となる．人工心肺
装置とは心肺血流遮断中の心臓と肺の働きを一時的に代行するため
の体外循環を行う装置である．
.機能と構成 構成を図5･51に示す．心臓と肺の機能を
代行するために以下の機能を有する．
(１) 脱血：心臓に還流してくる静脈血を体外に導く．
(２) 貯血：送血量を維持するための予備の血液を貯留しておく．
(３) ガス交換：人工肺を用いて静脈血に酸素を添加し二酸化炭素
ガスを除去する．
(４) 送血：酸化された血液をポンプを用いて動脈に送る．
(５) 血液温の調節：血液を熱交換器の中を通すことにより加温す
る．また低体温を必要とする手術では血液を冷却する．
(６) 出血した血液の回収と再利用：体外循環中に出血した血液は
吸引して貯血槽に貯留し再利用する．
(７) 異物および気泡の除去：血液フィルタと気泡除去器を通すこ
とにより，血液に混入した気泡や異物を除去することができ
る．
.血液ポンプ（blood pump) ローラポンプと遠心ポンプ
がある．
(１) ローラポンプ（図5･52)：ローラで回転させながら弾力性
のあるチューブを圧閉し，しごくことにより血液を送るポンプであ
る．ローラは血液の充満したチューブをほとんど逆流がない程度に
圧迫しつつ回転し，血液を前方に押し進める．ローラの後方ではチ
ューブが復元力で元の円形に戻るため再び血液を充満させる．1回
転当たりの駆出血液量はチューブの内径とローラの回転半径によっ
て決定され，これに回転数をかけると1分間当たりの送血量を求め
ることができる．また，構造が簡単で故障が少ないことも利点の一
つである．
(２) 遠心ポンプ（図5･53)：血液を回転させてその遠心力で血
液を移動させるポンプである．羽根車を回転させて渦流を起こすポ
ンプと，ポンプ内のコーン（円すい状の突起）を高速で回転させる
ことにより，血液の粘性を利用して渦流を起こすポンプがある．送
血量は血液の粘性，送血回路内の抵抗と末梢血管抵抗の影響を受け
て変化するものの，血球の破壊が極めて少ない利点がある．流量計
によって送血量をモニタしつつ回転数を調節する必要がある．

d.ペースメーカ（cardiac pace maker )
.適応 心臓は，右心房にある洞結節からの電気パルスが
刺激伝導路を通って心筋に伝えられ，規則正しく拍動している．
種々の原因（洞機能不全症候群，房室ブロック，両脚ブロック）で
心臓が除脈（心拍数の減少）になると循環不全になるため，ペース
メーカ植込みの適応となる．ペースメーカは電気パルスで心筋を直
接刺激し，心臓を拍動させる．
.構造 心臓ペースメーカは，パルス発生器，1本ないし
2本のリード電極（刺激電極），およびプログラマからなる．ペー
スメーカは，上部胸壁の皮下に植え込まれ，心筋内に植え込まれる
リード電極と接続される（図5･55）．
パルス発生器は心臓ペースメーカの中枢部であり，ハイブリッド
回路（マイクロコンピュータ）と電池からなる．ハイブリッド回路
には，ペーシング回路とセンシング回路がある．ペーシング回路で
は刺激パルスのレート，電圧，パルス幅などを設定する．センシン
グ回路では心電図のP波やR波を感知し，種々のデマンド機構や，
心房同期刺激を起こさせる．電池には，ヨウ化リチウム電池が使用
されており，その寿命は10～12年である．パルス発生器は，溶接
密閉されたチタン製ケースの中に入れられている．典型的なパルス
発生器は，幅50mm×50mm，厚さ7mm以下で，重さも30g以
下となっている．
リード電極は，先端に電極をもつ絶縁されたフレキシブルなワイ
ヤである．先端は心臓壁内に挿入され，パルス発生からの刺激パル
スを伝え，心臓が拍動するよう刺激する．
プログラマを使用することにより，医師はペースメーカの動作特
性の評価や刺激パラメータの変更を，非侵襲的にすることができ
る．
c .人工弁（artificial heart valve）
.機能 ヒトの左右の心臓はそれぞれ逆流を防ぐため二つ
の弁をもっている．右心房と右心室間にある弁を三尖弁，右心室と
肺動脈間にある弁を肺動脈弁という．左心房と左心室間にある弁を
僧帽弁，左心室と大動脈間にある弁を大動脈弁という．もし心臓弁
が先天的に欠損していたり，後天的に病的変化したときには，心臓
は血液を効率的に駆出することができない．このため心臓は肥大し
たり，機能不全になったりする．このような心臓弁が外科的に修復
できないときは，残された治療法は欠損弁を人工弁で置換すること
である．人工弁には，機械弁と生体弁がある．
.機械弁（mechanical heart valve ，図5･54) 1952年に
初めて臨床に使用されて以来，多くの種類の機械弁が開発されてい
る ．新しい機械弁の開発に伴い，耐久性，流体特性が向上し，
血液損傷（溶血）が減少した．機械弁の利点は耐久性に優れている
ことであり，おもな欠点は血栓形成の危険性があることである．
初期の機械弁はケージ内のボールが圧力差により往復運動し，弁
口を開閉する構造であった．弁高が高いため，次にはケージ内でデ
ィスク往復運動する構造の弁が開発され，弁高が低くなった．しか
し，これらの弁は，流速の速い中心流部分にボールやディスクの障
害物があるため，圧損失が大きい欠点があった．そこでディスクが
傾く傾斜型ディスク弁が開発された．これによりほぼ中心流が得ら
れた．その後さらに完全な中心流が得られる二葉弁が開発された．
現在，二葉弁が臨床で最も多く使用されている．
二葉弁では弁輪で二つの弁葉を支持している．弁葉は圧力差によ
り開閉する．弁輪と弁葉はパイロライトカーボンをコーティングし
たグラファイトでできている．弁輪は縫合用のポリエステルカフを
もっている．
.生体弁（tissue valve ，図5･54) 生体弁はウシの心嚢
膜やブタの大動脈弁から作られる．心嚢膜や大動脈弁は免疫反応を
抑制するため，化学的に処理される．生体弁材料は通常，金属や高
分子のリングに取付けられる．リングには，縫合のためのポリエス
テル製カフが取付けられる．生体弁は機械弁に比べ，血液適合性に
優れているが，耐久性，石灰化の問題がある．
.動作 ペースメーカは症状に応じて各種様式で駆動され
ている．ペーシング様式を表記するため，ICHDの3文字コード表
記法が使用されている．おもなペーシング様式は次のようである．
(１)VVI ：心室でペーシングおよびセンシング，デマンド型
ペーシングで，臨床的に用いられる最も基本的なペーシング法であ
る．心室で心臓の拍動をセンスし，一定時間内に拍動がないときに
は，ペースメーカにより心臓を刺激する．
(２)DDD：心房および心室でペーシングとセンシングが行わ
れる．生理的な刺激法である．心室でR波が感知されると心室お
よび心房のペーシング刺激が抑制される．心房でP波が感知され
ると心房のペーシング刺激は抑制され，P波に同期して，設定され
た房室間隔時間の後に心室ペーシング刺激を行う．
