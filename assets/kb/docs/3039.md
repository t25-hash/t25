# 5・2・1 表面状態解析の必要性

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・2 極表面の物理・化学的同定

ここではマイクロ/ナノトライボロジー特性に強く影響する固体の最表面の物理的および化学的状態を分析によって同定することの必要性について述べる(詳細なデータは文献(1)(4)(5)を参照)。ある大きさの物体が水平な固体表面上を動くとき、摩擦力に寄与する法線荷重として、通常は物体の自重のみを考えればよく、表面に作用するいわゆる表面力は自重に比べると無視できるほど小さい。しかし、物体が微小化するにつれて、表面力が自重に対して無視できないほどの大きさになる。これは、物体が相似的に微小化すると体積は長さの3乗に比例して減少するので、自重もその割合で小さくなるが、表面力が作用する表面積は長さの2乗で、接触部の周長は1乗で減少し、このため、表面力/自重の比は、微小化の1乗あるいは2乗に逆比例して増大し、表面力がトライボロジー現象の支配的な要因となる。一般に、表面力は固体表面の状態に強く依存することから、マイクロ/ナノトライボロジー現象を理解し、超微細機構の開発に応用するためには、固体表面の状態を正確にとらえなければならない。トライボロジー特性に影響する表面状態は、物理的な表面状態(形状、構造欠陥、硬度など)と化学的な表面状態(元素・組成、結合状態など)に大別できる。状態が表面で二次元的に分布する様子(面情報)だけでなく、その面情報が厚さ方向に変化する様子(深さ情報)も重要で、後者については、イオンエッチングによって最表面から所定の深さだけ掘って、そこの表面分析を行うことを段階的に繰返す方法がよく用いられる。摩擦力は摩擦界面における相互作用力に支配されるので、極表面に存在する元素の特定だけでなく、その化学結合状態を知らなければならない。例えば、金属試験片の表面にフッ素(F)が検出された場合、その化学構造が有機フッ素化合物(C-F結合)か金属フッ化物(M-F結合)かで機械的性質や表面エネルギーが異なるため、表面の化学構造に依存したマイクロ/ナノトライボロジー特性が観察される。すなわち、表面の元素、組成だけでなく化学構造に関する情報、言い換えれば化学的分解能の高い分析が必要である。また、トライボロジーでは摩擦により界面構造が動的に変化するため、時間分解能を上げた表面状態の同定も必要となる。特に、トライボロジー試験を行いつつ同時に観察も行うその場観察はトライボロジー現象を生きたままとらえることができるので、試験終了後に表面を観察、分析するのに比較して、得られる情報の質と量が優れている。図II-5・1は、空間分解能を横軸に、化学的分解能を縦軸にして各種の表面分析装置(手法)の座標を示す。原子間力顕微鏡(AFM)をはじめとする走査型プローブ顕微鏡(SPM)は、原子スケールの空間分解能をもつので極微小領域での摩擦現象を観察図-5・1 分解能から見た表面分析装置の特徴4―202するのに適するが、表面に存在する物質の化学結合状態に関する情報は得られない。また、X 線光電子分光法(XPS : X-rayphotoelectron spectroscopy)は化学結合状態を知るには好都合であるが、空間分解能が10 m程度である。したがって、表面分析装置は目的にあわせて選択することが重要である。マイクロ/ナノトライボロジー特性に影響するような最表面の分析をする場合、試料の扱いには十分注意しなければならない。試料の表面状態は、大気中の酸素、水分などの吸着や表面反応により時々刻々変化するだけでなく、試料の取扱い時に有機物などによって表面が汚染され、分析結果に大きな影響を与えることがある。
