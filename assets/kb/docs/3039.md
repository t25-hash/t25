れている．中空糸型透析器のコストパフォーマンスはよい．
b.人工肺（blood oxygenator ) 停止した心臓の
手術（直視下開心術）を行う間の肺の機能を代行するのが人工肺で
あり，人工腎臓と並んで多く臨床使用されている，膜を用いた人工
臓器である．人工肺は，体外循環血液に酸素を吸収させ，血液から
炭酸ガスを放散させるモジュールである．また急性の重症呼吸不全
患者の肺機能を助けるため，膜型人工肺がECMO( extracorporeal
membrane oxygenation）として短期間（数日から数週間）用いら
れ，また心不全時の心肺バイパスにも適用されている．
.装置（apparatus ) 図5･58 に人工心肺装置の概念
図を示す．開心術の間だけ心臓と肺の両方をバイパスし，その代わ
りに人工心肺装置が用いられる．右心房に入る大静脈から抜かれた
静脈血は，手術時に心腔内にたまる若干の心内血とともに，静脈血
貯槽に導かれ，そこからポンプで人工肺に送入される．その前で，
温度調節された水によって，熱交換器で血液は適当な温度まで加熱
あるいは冷却される．人工肺で酸素加された動脈血は下大動脈に送
入される．ポンプの数と位置は人工肺の形式と機器によって異な
る．安全操作のために多くの計測機器が必要である．血液ポンプと
してはローラポンプ，拍動型ポンプが使われているが，最近は小型
の無拍動型遠心ポンプが盛んに使われている．熱交換器には，多管
型，コイル型，平板型がある．血液-水熱交換器の性能指標には，
熱交換効率が用いられている．熱交換効率は，血液の温度降下を血
液入口における血液と水との温度差で割った値で，装置効率であ
る．この熱交換効率は，総括伝熱係数と伝熱面積が増すほど大きく
なり，血液流量が増すほど小さくなる．
