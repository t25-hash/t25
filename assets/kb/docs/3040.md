リッド人工膵臓
a.ハイブリッド人工肝臓（hybrid artificial liver ) 現在，わ
が国の肝疾患患者は60万人以上といわれ，年間5万人以上が死亡
している．重篤な肝疾患の根本的治療法は肝移植治療であるが，世
界的にドナー不足は避けられない．一方で血漿交換等の血液浄化療
法が確立されているが，依然として救命率が低い．そこで，肝臓の
機能を担う肝細胞を利用して，肝臓の旺盛な再生力を促進させ自己
回復を促す，あるいは肝移植までの橋渡しをするハイブリッド人工
肝臓が注目されている．このようなハイブリッド人工肝臓の基本概
念は，肝細胞を充てんした人工肝臓モジュール（バイオリアクタ）
や体外循環システムから構成される体外設置型の治療システムであ
る（図5･63）．ハイブリッド人工肝臓の当面の目標は急性肝不全治
療を実現する2週間程度の機能維持であるが，将来的には慢性肝不
全治療への波及効果の拡大や肝移植までの長期的な橋渡しを実現で
きる数箇月の機能維持が必要である．
.肝細胞培養法 肝細胞培養法は，機能発現レベルやその
維持期間と密接に関係している．従来より知られる単層培養法で
は，数日のうちに肝細胞の機能が劇的に低下あるいは消失する．一
方，近年，分散状態の肝細胞が凝集して形成された細胞組織体（オ
ルガノイド）が高機能発現の長期維持を達成できることが明らかと
なっている．オルガノイドは生体肝臓の組織様構造を生体外で再構
築した状態にあると考えられ，例えばオルガノイド培養法の一種で
ある球状組織体（スフェロイド）では少なくとも2週間以上の機能
維持，中空糸を利用した円柱状組織体（シリンドロイド）やlotus
root （レンコン）状オルガノイドでは数箇月もの機能維持が実現で
きている．
.人工肝臓モジュール 人工肝臓モジュールには，生体肝
の10～30%の肝細胞量，すなわち70kgの成人の場合150～500g
程度の細胞量が必要と考えられている．患者のベッドサイドで使用
するためには，容積数L程度のコンパクトなモジュールが必要で
あり，次のような技術的ポイントが求められる．(１)モジュールの
コンパクト化のために1×10cells /cm -module以上の高密度培養
の達成，(２)良好な機能発現のために肝細胞と患者血漿（あるいは
培養培地）の間で良好な物質交換の達成，(３)液流動による高いせ
ん断応力からの細胞の保護，(４)良好な操作性と容易なスケールア
ップのためにシンプルなデザイン，が必要である．
現在，中空糸タイプ，マイクロカプセルタイプ，多層平板型タイ
プ，不織布タイプ，多孔質タイプ等の種々の人工肝臓モジュールが
開発されている．中空糸やマイクロカプセルタイプは細胞の固定化
やせん断応力からの細胞の保護の面で優れるが，膜やゲルによる物
質交換能の低下，不均一な細胞分布といった課題がある．多層平板
タイプでは均一な細胞分布の面で優れるが，細胞が付着する比表面
積が小さいために高密度化とスケールアップが難しいという課題が
ある．不織布や多孔質タイプでは物質交換やスケールアップの面で
優れるが，細胞の離脱や不均一な細胞分布を改善することが必要で
ある．
.体外循環システム ハイブリッド人工肝臓用の体外循環
システムには次のような技術的ポイントが求められる．(１)患者と
人工肝臓モジュール間での良好な物質交換の達成，(２)体外循環負
荷の軽減，(３)モジュールの良好な機能発現のために溶存酸素
（6.8ppm付近）およびpH（7.2付近）の制御，(４)安全監視装置
の導入や容易な操作性，などである．現在，血液浄化療法用に開発
されてきた体外循環システムがハイブリッド人工肝臓用に転用され
ている．

b.ハイブリッド人工膵臓（hybrid artificial pancreas ) わが
国におけるインスリン依存型糖尿病患者は，約5万人といわれる．
現在，インスリン注射による血糖値のコントロールが行われている
が，代謝および内分泌動態の異常や細小血管合併症の発生等は避け
られない．そこで，より生理的な状態での治療法として，膵臓内で
血糖調節ホルモンを分泌しているランゲルハンス島（直径約200
mの組織．以下ラ島）あるいはラ島中のインスリンを分泌するβ
細胞を生体適合性担体に固定化し，患者に移植するハイブリッド人
工膵臓の研究が試みられている．
.ハイブリッド人工膵臓の技術的ポイント ハイブリッド
人工膵臓には次のような技術的ポイントが求められる．
(１) インスリン分泌細胞の大量確保：ヒト成人治療に必要なラ
島は35～70万個，β細胞にすると10～20億個が必要であり，大量
採取が可能でヒト型に近いインスリンを分泌するブタ細胞の利用が
有力である．しかし，ブタからのラ島の分離は難しく，現在，大量
調製法や大量凍結保存技術の確立，インスリン産生不死化細胞の作
出等が研究されている．
(２) 生体適合性の培養装置の開発：利用する担体が異物反応を
受けない，さらには生体の免疫機構から移植したラ島を保護する免
疫隔離が必要である．血糖値制御のためのグルコースやインスリン
（分子量6kD），移植細胞の生存のための酸素や栄養素は通過する
が，免疫担当細胞，抗体，補体等の免疫因子（分子量が最も小さい
抗体IgG，160kD）は通さない免疫隔離性が要求される．また，
大量の細胞を移植させるための高密度培養（装置のコンパクト化），
移植した細胞の生存と良好な機能発現のために移植部位や方法を考
慮した装置形状の工夫等も重要である．

る．このタイプは移植ラ島の固定化や取扱いやすさの面で優れてお
がよい．
り，生体適合性や選択透過性，物質移動能の向上が今後の改良点と
考えられる．マイクロカプセル型は個々のラ島を薄いゲル被膜で覆

の外腔部分にラ島を付着させ，内腔に患者血液を流すタイプであ
る．移植ラ島と患者血液間の応答性には優れるが，膜の生体適合
性，細胞の固定化や移植時の取扱いが難しいという課題がある．

り有効性は実証されているが，イヌやブタ等の大動物を用いた研究
では大きな成果が得られていないのが現状である．今後，実用化に
向けてヒト臨床応用を想定した大動物実験による治療効果および長
期的な免疫反応の評価が必要である．
