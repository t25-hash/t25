# 5・3・2 原子間力顕微鏡のシミュレーション

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

近年の原子間力顕微鏡（AFM）や表面力測定装置の開発によっ
て，摩擦や潤滑現象を原子・分子レベルで観察することが可能にな
ってきた．その一方で，AFMでは実際に何を観察しているのかと
いう議論も活発に行われている．そこで，AFMの探針と測定試料
との間の凝着過程に関するシミュレーションが行われている．
例えば，Au基板上にNi原子で構成されたティップを接触させ
るとき，ティップを基板から離す過程においてティップと基板間に
働く力はヒステリシスを有することが分子動力学シミュレーション
により予測されている．これは，ティップを基板から離す過程
において，表面のAu原子の一部がNiティップに凝着すること
で，基板とティップを接続するネックを形成することが原因である
と結論され，実験的にもこの現象が確認されている．
また，AFM測定におけるスティックスリップ現象やAFM像の
理論的な理解も進められている．
