# 5・3・3 ダイヤモンド表面の摩擦現象のシミュレーション

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

分子動力学法を用いることで,水素原子,メチル基,エチル基,あるいはn-プロピル基が化学結合により全表面を覆った二つのダイヤモンド(111)面を摩擦させたときの摩擦係数が計算されている.水素原子に覆われたダイヤモンド表面は,負荷の増加に伴い摩擦係数が大きくなる.一方,メチル基に覆われた表面では,低荷重での摩擦係数は水素原子に覆われた場合に比べて大きいが,高荷重では水素原子に覆われた場合に比べて小さくなり,最終的には一定値をとる.さらに,メチル基からエチル基へとサイズを大きくすると,高荷重での摩擦係数がより小さくなる.これは,サイズが大きくなるにつれて固体表面間の距離が遠くなること,さらにこれらアルキル基の屈曲により,アルキル基どうしの反発力の小さい経路を通って2枚のダイヤモンドが摩擦できるようになることがその要因であると結論づけられている.また,摩耗過程を調べるために,33GPaの圧力下での摩擦シミュレーションが,エチル基に覆われたダイヤモンド(111)面について行われ,H ラジカル,CHCHラジカル,エチレン分子の生成など摩耗現象に伴う化学反応が再現されるとともに,摩擦方向によって摩耗率が異なることも明らかにされている.
