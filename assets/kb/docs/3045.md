# 5・3・4 スティックスリップ現象のシミュレーション

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

分子動力学法を用いることで，境界潤滑におけるスティックスリ
ップ現象の検討が行われている．fcc構造をもつ2枚の基板間に
レナード・ジョーンズ（Lennard-Jones，LJ）流体を挟み込み，
片方の基板にばねを取付けて，ばねの他端を一定の速度で動かして
引っ張る．しかし，基板はLJ流体膜によって固着されているの
で，始めのうちは静止したままである．ばねが次第に伸ばされて，
ばね力がLJ流体のせん断降伏力以上になると，基板が引きずられ
て動き出すので摩擦が起こる．基板の変位によりばねの伸び量が小
さくなってばね力が低下すると基板はいったん静止するが，その
後，ばねの伸びが次第に増してばね力が再び降伏力以上になると基
板がまた動き出す．このような連続的なスティックスリップ現象が
再現されている．このとき，LJ流体が周期的なせん断融解と再結
晶化を繰返すという興味深い理論的知見も得られている．さらに，
ばね他端の移動速度を大きくするとスティックスリップが発現せず
に連続的に滑るようになり，これはせん断融解したLJ流体膜が再
結晶化する十分な時間がなくなることに起因すると結論されてい
る．

5・3・5固体間に挟まれた液体分子のパッキング構造
固体間に挟まれた液体分子がどのような秩序あるいは無秩序な配
置・配列をとるかというパッキング構造は，潤滑，摩擦，凝着など
のトライボロジー現象に顕著な影響を与えることが知られている．
そこで，グランドカノニカル分子動力学法を用いることで，Au
(111)表面に挟まれたn-ヘキサデカンとスクアランのパッキング構
造が計算されている．グランドカノニカル分子動力学法とは，
通常の分子動力学法とは異なり，与えられた圧力を維持するよう
に，シミュレーション中に分子数を増減させることで，平衡構造を
求めることができる手法である．その結果，両液体ともAu(111)
表面間で層状構造をとるが，枝分かれのあるスクアランは直鎖の
n-ヘキサデカンに比べて形成される層の数が少なく，層内の秩序
性も低くなることが理解された．さらに，液体分子が固体表面に及
ぼす力を層間距離に対してプロットすると，n-ヘキサデカンでは
層間距離の連続的変化に対して，正と負の力が交互に現れる興味深
い現象が得られている．
5・3・6せん断場における潤滑剤のシミュレーション
分子動力学法を用いることで，せん断場における潤滑剤のダイナ
ミックスが検討されている．例として，2枚の鉄基板にシク
ロヘキサンおよびジシクロヘキシルを挟み込んだ後に，片方の鉄基
板を一定方向に動かすことで，せん断場におけるダイナミックスが
調べられた結果を図Ⅱ-5･4に示す．シクロヘキサンはせん断の進
行に伴い分子が層状構造を形成するのに対し，ジシクロヘキシルで
は層状構造の形成は確認されず，分子種の違いによる明らかなせん
断挙動の違いが見られた．さらに，この計算により得られたシクロ
ヘキサンの摩擦係数はジシクロヘキシルに比べ小さく，潤滑剤のパ
ッキング構造と摩擦係数の相関が理論的に示された．
