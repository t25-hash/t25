# 5・3・4 スティックスリップ現象のシミュレーション

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

分子動力学法を用いることで、境界潤滑におけるスティックスリップ現象の検討が行われている。fcc構造をもつ2枚の基板間にレナード・ジョーンズ(Lennard-Jones,LJ)流体を挟み込み、片方の基板にばねを取付けて、ばねの他端を一定の速度で動かして引っ張る。しかし、基板はLJ 流体膜によって固着されているので、始めのうちは静止したままである。ばねが次第に伸ばされて、ばね力がLJ流体のせん断降伏力以上になると、基板が引きずられて動き出すので摩擦が起こる。基板の変位によりばねの伸び量が小さくなってばね力が低下すると基板はいったん静止するが、その後、ばねの伸びが次第に増してばね力が再び降伏力以上になると基板がまた動き出す。このような連続的なスティックスリップ現象が再現されている。このとき、LJ 流体が周期的なせん断融解と再結晶化を繰返すという興味深い理論的知見も得られている。さらに、ばね他端の移動速度を大きくするとスティックスリップが発現せずに連続的に滑るようになり、これはせん断融解したLJ流体膜が再結晶化する十分な時間がなくなることに起因すると結論されている。
