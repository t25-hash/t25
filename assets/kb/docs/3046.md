# 5・3・5 固体間に挟まれた液体分子のパッキング構造

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

固体間に挟まれた液体分子がどのような秩序あるいは無秩序な配4―203第II部トライボロジー(a)(b)図-5・4 (a)シクロヘキサンおよび(b)ジシクロヘキシル分子のせん断後の構造置・配列をとるかというパッキング構造は、潤滑、摩擦、凝着などのトライボロジー現象に顕著な影響を与えることが知られている。そこで、グランドカノニカル分子動力学法を用いることで、Au(111)表面に挟まれたn-ヘキサデカンとスクアランのパッキング構造が計算されている。グランドカノニカル分子動力学法とは、通常の分子動力学法とは異なり、与えられた圧力を維持するように、シミュレーション中に分子数を増減させることで、平衡構造を求めることができる手法である。その結果、両液体ともAu(111)表面間で層状構造をとるが、枝分かれのあるスクアランは直鎖のn-ヘキサデカンに比べて形成される層の数が少なく、層内の秩序性も低くなることが理解された。さらに、液体分子が固体表面に及ぼす力を層間距離に対してプロットすると、n-ヘキサデカンでは層間距離の連続的変化に対して、正と負の力が交互に現れる興味深い現象が得られている。
