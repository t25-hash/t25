# 5・3・6 せん断場における潤滑剤のシミュレーション

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

分子動力学法を用いることで、せん断場における潤滑剤のダイナミックスが検討されている。例として、2枚の鉄基板にシクロヘキサンおよびジシクロヘキシルを挟み込んだ後に、片方の鉄基板を一定方向に動かすことで、せん断場におけるダイナミックスが調べられた結果を図II-5・4に示す。シクロヘキサンはせん断の進行に伴い分子が層状構造を形成するのに対し、ジシクロヘキシルでは層状構造の形成は確認されず、分子種の違いによる明らかなせん断挙動の違いが見られた。さらに、この計算により得られたシクロヘキサンの摩擦係数はジシクロヘキシルに比べ小さく、潤滑剤のパッキング構造と摩擦係数の相関が理論的に示された。文献(1) Bhushan, B., Handbook of Micro/Nanotribology, (1995)、 3,CRC Press.(2) Singer,I.L.,J.
