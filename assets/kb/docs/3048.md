# 5・2・1 表面状態解析の必要性

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・2 極表面の物理・化学的同定

ここではマイクロ/ナノトライボロジー特性に強く影響する固体
の最表面の物理的および化学的状態を分析によって同定することの
必要性について述べる（詳細なデータは文献(１)(４)(５)を参照）．
ある大きさの物体が水平な固体表面上を動くとき，摩擦力に寄与
する法線荷重として，通常は物体の自重のみを考えればよく，表面
に作用するいわゆる表面力は自重に比べると無視できるほど小さ
い．しかし，物体が微小化するにつれて，表面力が自重に対して無
視できないほどの大きさになる．これは，物体が相似的に微小化す
ると体積は長さの3乗に比例して減少するので，自重もその割合で
小さくなるが，表面力が作用する表面積は長さの2乗で，接触部の
周長は1乗で減少し，このため，表面力/自重の比は，微小化の1
乗あるいは2乗に逆比例して増大し，表面力がトライボロジー現象
の支配的な要因となる．一般に，表面力は固体表面の状態に強く依
存することから，マイクロ/ナノトライボロジー現象を理解し，超
微細機構の開発に応用するためには，固体表面の状態を正確にとら
えなければならない．
トライボロジー特性に影響する表面状態は，物理的な表面状態
（形状，構造欠陥，硬度など）と化学的な表面状態（元素・組成，
結合状態など）に大別できる．状態が表面で二次元的に分布する様
