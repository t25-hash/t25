# 2・1・1 製図の目的

> **章**: 第2章 製図と図面
> **節**: 2・1 製図の目的と基本条件

製図(technical drawing,drawing)とは,工業の各分野で使用する図面を作成することをいう.図面(technical drawing)とは,三次元の対象物を二次元の製図用紙の上に図示したもので,設計者・製作者の間,発注者・受注者の間などで,必要とする技術的な情報を伝えるためのものをいう.製図の目的は,(1) 情報の伝達,(2) 情報の保存・検索・利用,(3) 情報作成時の思考の手段である.図面を作成する目的は,図面作成者の意図を,図面使用者に確実かつ容易に伝達することにある.その図面に示された情報は保存され,部品補給時などに利用されるだけでなく,ほかの新しい図面を作成するときにも,検索して,最も信頼できる技術情報として再利用される.したがって,図面は保存・検索・利用しやすいことが必要である.製図は,図面作成者(主として設計者)が,頭の中の抽象的な概念を具体化し総合する過程での有効な思考手段である.したがって,製図能力がないということは,技術者にとって有効な思考手段の一つを欠くことになる.
