# 5・3・1 分子動力学法（molecular dynamics method）

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

マイクロトライボロジーの分野においては，原子・分子の挙動が
直接的に材料の特性や物性に影響を与えるため，コンピュータシミ
ュレーションを用いた原子レベルでの摩擦・摩耗現象の解明，さら
にはシミュレーションを基礎にした摩擦材料，磁気記録材料，潤滑
剤，添加剤などの分子設計が重要な学問分野となりつつある．原子
レベルのシミュレーション手法としては，分子動力学法，モンテカ
ルロ法，量子化学などが一般的であるが，摩擦，摩耗，凝着，潤滑
といったダイナミックな現象の解明が要求されるこの分野では，お
もに分子動力学法が活用されている．
分子動力学法とは，ニュートンの運動方程式を解くことにより，
有限温度下における個々の原子の動的挙動をシミュレーションでき
る手法である．本計算手法では，第一原理計算などで決定された原
子間ポテンシャルを用いて，原子・分子間の相互作用が表現され
る．分子動力学法は，潤滑剤の摩擦係数，トラクション係数，粘
度，密度などの物性値の推算，さらには摩擦材料に挟まれた潤滑剤
のせん断場における動的な摩擦挙動や構造変化を解明することがで
きるため，近年その応用例が加速度的に増加している．
