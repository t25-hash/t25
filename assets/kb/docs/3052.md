# 2・3・1 製図用紙の大きさと様式

> **章**: 第2章 製図と図面
> **節**: 2・3 製図に用いる用紙,尺度,線および文字

図面は,対象物が必要とする明瞭さを保ち,適切な大きさを保つことができる最小の用紙を用いる.用紙のサイズはA 列サイズ(表III-2・2)から選ぶ.対象物の大きさそのほかの理由により,やむを得ない場合には,長手方向に延長してもよい.図面は,長辺を横方向に用いるが,A 4については縦方向で用いてもよい.図面には,輪郭,表題欄(title block,title panel),中心マークを設けなければならない.また,比較目盛,格子参照方式を設けることが望ましい.さらに,用紙の向きを示す方向マーク,複写図の裁断のための裁断マークを付けてもよい.表題欄は図面の右下隅に設け,図面番号,図名,企業(団体)名,責任者の署名,図面作成年月日,尺度,投影法などを記入する.B 0001機械製図128,129,406,3040,5455第III部機械要素設計の基礎と製図
