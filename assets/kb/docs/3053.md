# 2・3・2 製図に用いる尺度

> **章**: 第2章 製図と図面
> **節**: 2・3 製図に用いる用紙、尺度、線および文字

図面を作成する場合に用いる尺度(scale)は、図面上の対象物の対応する長さをA、対象物の実際の長さをB として、A:B の比の形で表す。対象物を表現する目的および複雑さなどを考慮し、図の明瞭さを保つように、表III-2・3に示す推奨する尺度から選ぶ。表中に適当な尺度がない場合は。表中の尺度に10の整数乗をかけた値とする。推奨する尺度を図面に適用できない場合には中間の尺度を選んでもよい(JIS Z 8314:1998の附属書1に規定する尺度を選ぶことが望ましい)。図形の大きさと対象物の大きさとの間には、尺度により正しい比例関係をもたせて描く。ただし、製作図のうち、単純な図形があらかじめ図面用紙に印刷されてあり、その都度、寸法などを記入している場合や説明図では、比例関係を保たなくてもよい。
