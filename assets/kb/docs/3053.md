# 5・3・3 ダイヤモンド表面の摩擦現象のシミュレーション

> **章**: 第5章 マイクロトライボロジー
> **節**: 5・3 コンピュータシミュレーション

分子動力学法を用いることで，水素原子，メチル基，エチル基，
あるいはn-プロピル基が化学結合により全表面を覆った二つのダ
イヤモンド(111)面を摩擦させたときの摩擦係数が計算されてい
る ．水素原子に覆われたダイヤモンド表面は，負荷の増加に
伴い摩擦係数が大きくなる．一方，メチル基に覆われた表面では，
低荷重での摩擦係数は水素原子に覆われた場合に比べて大きいが，
高荷重では水素原子に覆われた場合に比べて小さくなり，最終的に
は一定値をとる．さらに，メチル基からエチル基へとサイズを大き
くすると，高荷重での摩擦係数がより小さくなる．これは，サイズ
が大きくなるにつれて固体表面間の距離が遠くなること，さらにこ
れらアルキル基の屈曲により，アルキル基どうしの反発力の小さい
経路を通って2枚のダイヤモンドが摩擦できるようになることがそ
の要因であると結論づけられている．また，摩耗過程を調べるため
に，33GPaの圧力下での摩擦シミュレーションが，エチル基に覆
われたダイヤモンド(111)面について行われ，Hラジカル，
CHCHラジカル，エチレン分子の生成など摩耗現象に伴う化学
反応が再現されるとともに，摩擦方向によって摩耗率が異なること
も明らかにされている．
