# 2・3・4 製図に用いる文字

> **章**: 第2章 製図と図面
> **節**: 2・3 製図に用いる用紙、尺度、線および文字

文字の種類は、ローマ字、数字、記号、漢字および仮名とし、平仮名と片仮名は一連の図面においては混用しない。ローマ字、数字、記号の文字の大きさは大文字(頭文字)の外側輪郭の高さを呼びとし、書体はA 形書体またはB 形書体のどちらかとし、直立体または右へ15度傾けた斜体を用いる。漢字および仮名は文字表-2・5 製図に用いる文字種類文字の大きさ呼び(単位mm)文字の線の太さローマ字、数字、記号2.5,3.5,5,7,10A 形書体(1/14)B 形書体(1/10)漢字3.5,5,7,10(1/14)仮名2.5,3.5,5,7,10(1/10)*ある種の複写方法ではこの大きさは適さない。寸法を記入するのに用いる。a)図形の中心を表すのに用いる。b)中心が移動する中心軌跡を表すのに用いる。a)隣接部分を参考に表すのに用いる。b)工具、ジグなどの位置を参考に示すのに用いる。c)可動部分を、移動中の特定の位置または移動の限界の位置で表すのに用いる。d)加工前または加工後の形状を表すのに用いる。e)図示された断面の手前にある部分を表すのに用いる。対象物の一部を破った境界、または一部を取り去った境界を表すのに用いる。断面図を描く場合、その断面位置を対応する図に表すのに用いる。ハッチング細い実線で、規則的に並べたもの図形の限定された特定の部分をほかの部分と区別するのに用いる。例えば、断面図の切り口を示す。4―210機械要素・トライボロジーの外側輪郭が収まる基準枠の高さを呼びとする(表III-2・5)。図面中の一連の記述に用いる文字の大きさの比率は、(漢字):(仮名):(ローマ字、数字および文字)=1.4:1.0:1.0、「ゃ、ゅ、ょ、っ」など小書きにする仮名は0.7とするのが望ましい。文字は、一字一字が正確に読めるように、明瞭に書き、鉛筆書きの文字は、図形を表した線と同じ濃さにそろえて書く。
