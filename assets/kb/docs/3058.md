# 2・4・3 図形の省略

> **章**: 第2章 製図と図面
> **節**: 2・4 製図における図形の表し方

a. 簡明な図示図示を必要とする部分をわかりやすくするために,次のように示すのがよい.図-2・6 部分投影図図-2・7 局部投影図図-2・8 部分拡大図1かくれ線は,理解を妨げない場合には省略する.2補足の投影図に見える部分を全部描くと,図がかえってわかりにくくなる場合には,部分投影図(図III-2・10)または補助投影図(図III-2・11)として表す.3切断面の先方に見える線は,理解を妨げない場合には省略するのがよい(図III-2・12).4一部に特定の形をもつものは,なるべくその部分が図の上側に現れるように描くのがよい(図III-2・13).5ピッチ円上に配置する穴などは,側面の投影図(断面図を含む)においては,ピッチ円が作る円筒を表す細い一点鎖線と,その片側だけに1個の穴を図示(投影関係にかかわりなく)し,ほかの穴の図示を省略することができる(図III-2・14).b. 対称図形図形が対称形式の場合には,対称中心線の片側を省略することができる.この場合,対称中心線の両端部に短い2本の平行細線[対称図示記号(symmetry symbol)]をつけるか(図III-2・15),片側の図形を対称中心線を少し越えた部分まで描く(図III-2・16).この場合には,対称図示記号を省略することができる.図-2・9回転投影図(a)(b)図-2・10 部分投影図による簡明な図示図-2・11 補助投影図による簡明な図示図-2・12 図示の省略(c)(b)(a)図-2・13 特定の形をもつものの図示4―211第III部機械要素設計の基礎と製図c. 繰返し図形同種同形のものが多数並ぶ場合には,次によって図形を省略することができる.1実形の代わりに図記号をピッチ線と中心線の交点に記入し,記号の意味を記述する(図III-2・17).2両端部(一端は1ピッチ分)または要点だけを実形または図記号によって示し,ほかの図形を省略して,ピッチ線と中心線の交点によって示す(図III-2・18).ただし,寸法記入によって交点の位置が明らかなときは,ピッチ線に交わる中心線を省略してもよい(図III-2・19).d. 中間部分の省略による図形の短縮同一断面形の部分,同じ形が規則正しく並んでいる部分,または長いテーパなどの部分は,中間部分を切取って,その肝要な部分だけを近づけて図示することができる(図III-2・20).切取った端部は,破断線で示す.図-2・14 ピッチ円上に配置する穴などの側面投影図図-2・16 対称図形の省略図-2・15 対称図形の省略(対称中心線を少し越えた描画)(対称図示記号)図-2・17 繰返し図形の省略(図記号を用いた例)図-2・19繰返し図形の省略(ピッ図-2・18 繰返し図形の省略チ線に交わる中心線を省(ピッチ線と中心線略した例)の交点で示した例)図-2・20 中間部分の省略4―212機械要素・トライボロジー図-2・21 長手方向に切断しないもの図-2・22 断面の表示図-2・23全断面図図-2・24 全断面図(切断線による切断位置の図示)
