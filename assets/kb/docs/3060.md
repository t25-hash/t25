# 5・1・3 加工誤差と仕上面の特性

> **章**: 第5章 切削加工
> **節**: 5・1 切削加工の基礎

工業製品の高品質化に伴って，寸法，形状ならびに表面粗さに対
する要求がいちだんと厳しくなってきている．特に近年の超精密加
工や電子デバイスの高集積化がこれに拍車をかけている．加工仕上
面の特性については寸法はもとより，うねり（waviness ）などの
形状誤差（form error ），表面粗さ（surface roughness ），さらに
はすじ目，きず，光沢，色，コロージョン，表面の微細構造などの
表面性状が含まれる．加工精度は，周辺環境まで含めた工作機械-
工具-工作物系に存在する数多くの因子の影響を受ける．近年工作
機械の力学的構造の強化，熱的構造の改良が進められ，静・動剛性
はかなり高められてきている．しかし，現在のような加工形態が続
く限り，加工誤差（machining error ）は小さくできてもなくすこ
とはできない．
