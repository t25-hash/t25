# 2・5・3 特別な形体の寸法記入方法

> **章**: 第2章 製図と図面

特殊な形態の寸法は,寸法補助記号(表III-2・7)と寸法数値の組合せで指示することができる.a. 円の直径と円弧の半径円形断面の対象部分が円形であることを円を描かないで示すには,直径の記号φを寸法数値の前に記入して表す(図III-2・58).円形の図に直径の寸法を記入するときはφを記入しない(図III-2・59).直径の異なる円筒が連続していて,その寸法数値を記入する余地がないときには,片側に描くべき寸法線の延長線と矢印を描き,直径の記号φおよび寸法数値を記入する(図III-2・60).半径の寸法は,半径の記号を寸法数値の前に記入して表す(図III-2・61).ただし,半径を示す寸法線を円弧の中心まで引く場合には,記号を省略してもよい.矢印や寸法数値を記入する余地がないときには図III-2・62の例による.半径の大きさがほかの寸法から導かれる場合には,半径を表す矢印と数値なしの記号()によって指示する(図III-2・63).b. 球の直径または半径球の直径の記号φ,半径の記号表-2・7 寸法補助記号項目記号呼び方直径φまる半径あーる球の直径φえすまる球の半径えすあーる正方形の辺□かく円弧の長さえんこ板の厚さてぃー45°の面取りしー注:,はISO 129:2004では規定されていないため,国際的には必ずしも通用しない.図-2・59円の直径図-2・58 円の直径(円形の図に記入するとき)(円形であることを示す場合)図-2・61 円弧の半径図-2・62 円弧の半径図-2・60 円の直径(狭いとき)(記入余地がないとき)図-2・63 円弧の半径(ほかの寸法から導かれるとき)4―216機械要素・トライボロジーを寸法数値の前に記入して表す(図III-2・64).c. 正方形の辺正方形断面の対象部分が正方形であることを,正方形を描かないで示すには,正方形の記号□を辺の長さを表す寸法数値の前に記入して表す(図III-2・65).d. 板の厚さ板の主投影図の周辺または図の中の見やすい位置に,厚さの記号を厚さを表す寸法数値の前に記入して表す(図III-2・66).e. 円弧の長さ円弧であることを明示するために,円弧の長さを表す寸法数値の上に円弧の記号を付ける(図III-2・67).f. 面取り(chamfer)一般の面取りは通常の寸法記入法によるが,45°面取りの場合には,面取り寸法×45°または面取りの記号を面取り寸法の前に付けて表す(図III-2・68).g. 曲線曲線の寸法は,円弧の半径とその中心(図III-2・69)または任意の点の座標寸法(図III-2・70)で表す.h. 穴きり穴,打抜き穴,鋳抜き穴などの加工法による区別図-2・64球の直径と半径図-2・65 正方形の辺の長さ図-2・66板の厚さ図-2・67 円弧の長さ図-2・68 面取り図-2・69曲線の寸法記入(半径と中心)図-2・70 曲線の寸法記入(座標寸法)を示す必要があるときには,原則として工具の呼び寸法または基準寸法を示し,そのあとに加工方法を指示する(表III-2・8,図III-2・71).一群の同一寸法の穴の寸法は,その総数の次に×を挟んで記入する(図III-2・72).座ぐりの表し方は,座ぐりの直径を表す寸法の次に「座ぐり」と書く(図III-2・73).「座ぐり」は,平仮名「ざぐり」,片仮名「ザグリ」,漢字と片仮名の混用「座グリ」でもよい.深座ぐりは,深座ぐりの直径を表す寸法の次に「深座ぐり」と書き,次に「深さ」と書いてその数値を記入する(図III-2・74).長穴は図III-2・75の例による.i. テーパ・こう配対象の形体の近くに,参照線とテーパの向きを示す図記号(図III-2・76),こう配の向きを示す図記号(図III-2・77)を用いて指示する.表-2・8 加工方法の簡略指示加工方法簡略指示鋳放しイヌキプレス抜き打ヌキきりもみキリリーマ仕上げリーマ図-2・71 穴の寸法と加工方法図-2・72 同一寸法の穴の寸法図-2・73 座ぐり図-2・74 深座ぐり図-2・75 長穴の寸法記入j. 鋼構造物構造物の構造線図で,格子間の寸法を表す場合は,部材を示す線に沿ってその寸法を直接記入する(図III-2・78).形鋼などの寸法は,表III-2・9の表示により,それぞれの図形に沿って記入することができる.
