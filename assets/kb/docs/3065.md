# 2・6・1 形体とデータム

> **章**: 第2章 製図と図面
> **節**: 2・6 幾何公差

対象物の形状、姿勢、および位置は、寸法を与えるだけでは決まらない。これらを決めるために、幾何公差(geometrical toler-ance)という概念が導入される。幾何公差の対象となる点、線、軸線、面および中心面を、対象物の形体(feature)と呼ぶ。形体間の姿勢、位置を定めるための理論的に正確な幾何学的基準を、データム(datum)と呼ぶ。幾何公差を決めるためにデータムを必要とするものを関連形体(related feature)、必要としないものを単独形体(single feature)と呼ぶ。
