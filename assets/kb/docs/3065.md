# 2・6・1 形体とデータム

> **章**: 第2章 製図と図面
> **節**: 2・6 幾何公差

対象物の形状，姿勢，および位置は，寸法を与えるだけでは決ま
らない．これらを決めるために，幾何公差（geometrical toler-
ance）という概念が導入される．幾何公差の対象となる点，線，
軸線，面および中心面を，対象物の形体（feature）と呼ぶ．形体
間の姿勢，位置を定めるための理論的に正確な幾何学的基準を，デ
ータム（datum）と呼ぶ．幾何公差を決めるためにデータムを必要
とするものを関連形体（related feature），必要としないものを単
独形体（single feature）と呼ぶ．
2・6・2幾何公差の種類とその記号
幾何公差の種類とその記号を表Ⅲ-2･10に示す．公差は，その形
体が幾何学的に正しい形状，姿勢または位置からの，狂うことが許
される領域である公差域（tolerance zone）を指定することによっ
て与えられる．
