# 5・2・2 形削り・立て削り・平削り

> **章**: 第5章 切削加工
> **節**: 5・2 単刃加工

a.形削り 工具に直線往復運動と1ストロークごとに工作物
に送りを与えて削る方法を形削り（shaping）という．おもに平面
や直線溝などの加工に用いられる．形削りを行う機械を形削り盤
（図5･32) と呼ぶ．
b.立て削り 形削りにおける工具の運動を垂直にして，おも
にキー溝などの溝加工を行う方法を立て削り（slotting）という．
この作業を行う機械が立て削り盤（図5･33) である．
c.平削り 形削りと異なり，テーブルの水平往復運動と工具
をテーブルの運動方向と直角方向に送り，主として大きな平面を加
工する方法を平削り（planing）という．平削りを行う機械を平削
り盤（図5･34) と呼ぶ．
