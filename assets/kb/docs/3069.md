# 2・6・5 幾何公差の適用を限定する図示方法

> **章**: 第2章 製図と図面
> **節**: 2・95 )。形体の軸直線または中心平面をデータムと指定するときに

公差を形体の限定した部分だけに適用する場合には、この限定した部分を太い一点鎖線で示し、それに寸法を記入する(図III-2・105)。形体の任意の位置における限定した長さに同じ特性の公差を適用する場合には、この限定した長さの数値を公差値の後に斜線を引いて記入する。この指示は、形体全体に対する公差記入枠の下側の区画に直接記入する(図III-2・106)。公差域内での形体の性質を特に指示したいときには、公差記入枠の近くに要求事項を記入するか、これを引出し線で結ぶ(図III-2・107)。
