# 5・3・2 中ぐり加工

> **章**: 第5章 切削加工
> **節**: 5・3 穴あけ・中ぐり加工

a.中ぐりの方法 中ぐりの方法にはスタブボーリングとライ生し，表面粗さの劣化や寸法精度の低下が起こりやすい．
ラインボう方法で，ディーゼルエンジンのシリンダライナのような深穴の中b.ボーリングツールとボーリングバー 中ぐりは，ボーリンは穴ぐりバイトで行う．
ーは剛性を高めるとともに，加工中の振動の減衰性向上のため，質る．
長くなるために剛性の低下が避けられず，びびり振動やたわみが発ーリングは，両持ちのボーリングバーに中ぐりバイトを取付けて行ぐりに適する．
グバー（鋼製または超硬製）に中ぐりバイトを取付けた工具，またに調整して穴径を精密に仕上げることが必要であるため，しばしばをボーリングバーに取付けて行うことがある．
