# 2・6・7 寸法と幾何特性の相互依存性

> **章**: 第2章 製図と図面
> **節**: 2・95 )。形体の軸直線または中心平面をデータムと指定するときに

図面上に個々に指定した寸法および幾何特性に対する要求事項は、それらの間に特別な関係が指定されない限り、独立に適用する。それゆえ何も関係が指定されていない場合には、幾何公差は形体の寸法に無関係に適用し、幾何公差と寸法公差は関係ないものとして扱う。したがって、寸法と形状、姿勢または位置との間に特別な関係が要求される場合には、そのことを図面上に指定する。特別な関係には、包絡の条件(envelope requirement)、突出公差域図-2・104 形体グループのデータム指定図-2・106 限定した長さ当たりの公差の指定図-2・105 限定した部分の公差の図示図-2・107 形体の性質の指定図-2・108 理論的に正確な寸法図-2・109包絡の条件の記入法(projected tolerance zone)、最大実体公差方式(maximum mate-rial requirement)、および最小実体公差方式(least materialrequirement)がある。a. 包絡の条件の指示円筒面や平行二平面よりなる単独形体の実体が、機能上の要求から、最大実体寸法をもつ完全体の包絡面を超えてはならないことを指定する必要がある場合には、寸法公差の後に記号◯E を付けて表す。図III-2・109の図示例では、円筒形体の表面は、最大実体寸法φ150における完全形状の包絡面を超えてはならず、かついかなる実寸法もφ149.96より小さくてはならない。b. 突出公差域の指示公差域をその形体自体の内部ではなく、外部に指定したい場合には、その突出した部分を細い二点鎖線で表し、その寸法数字の前および公差の値の後に記号◯P を記入する(図III-2・110)。c. 最大実体公差方式の適用の指示最大実体公差方式は主と(a) 交差付き形体に適用(b) データム形体に適用(c) 両者に適用図-2・111 最大実体公差方図-2・110 突出公差域の指示式の適用の指示して二つの形体を単に組付ける必要があるときに、それぞれの形体について寸法公差と姿勢公差または位置公差との間の相互依存性を考慮し、寸法公差の余裕分を姿勢または位置公差に付加することができる場合に適用する。最大実体公差方式は、軸線または中心面をもつ関連形体に適用し、平面または平面上の線には適用できない。この公差方式が指定されたとき、図面に指定された姿勢または位置公差の公差値は、公差付き形体が最大実体状態のときに対して定められたものである。形体が許容限界内でこの最大実体寸法から離れて仕上がったときには、その分だけを姿勢または位置公差に付加することが許される。この付加公差は、実効寸法[外側形体では(最大許容寸法)+(姿勢または位置公差)、内側形体では(最小許容寸法)-(姿勢または位置公差)]を超えない範囲で与えられる。寸法公差をもつデータムに関連した形体に姿勢または位置公差が適用される場合には、データム形体にも最大実体公差方式を適用することができる。この場合には、データム形体がその最大実体寸法から離れている値だけ、データム軸直線やデータム中心平面が浮動することができる。なお、この場合に公差付き形体の公差が増すことにはならない。最大実体公差方式の適用を指示するには、この方式を公差付き形体、データム形体またはその両者に適用するかによって、記号◯Mを用いてそれぞれ図III-2・111(a)~(c)のように表す(図III-2・104参照)。d. 最小実体公差方式の適用の指示最小実体公差方式は、最大実体公差方式の基準となる最大実体状態を最小実体状態に変えて公差の相互依存関係を要求するものである。完全形状の形体がその最小実体寸法から離れたときには、その離れた寸法分だけ幾何公差を増加させることができ、最小厚さの管理、破断防止などに有効である。実体公差方式の指示は、適用する幾何公差の公差値のすぐ後に記号◯L を記入する。
