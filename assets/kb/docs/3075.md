# 2・8・2 歯車製図

> **章**: 第2章 製図と図面
> **節**: 2・127 に示すような略図で表す.谷底および不完全ねじ部を表す線

歯車の部品図は,要目表(tabular)および図を併用する.a. 図図には,主として歯車素材を製作するのに必要な寸法を記入する.歯先円は太い実線,基準円(基準ピッチ円)は細い一点鎖線で表す.歯底円は細い実線で表すが,軸に直角な方向から見た図(以下,主投影図)を断面図で図示するときは太い実線で表す(図III-2・132).歯底円の記入は省略してもよく,かさ歯車およびウォームホイールの軸方向から見た図では原則として省略する.歯すじ方向は,外形図の場合は,通常3本の細い実線で表す.主投影図を断面で示すとき,外はすば歯車の歯すじ方向は紙面から手前の歯の歯すじ方向を3本の細い二点鎖線で表す[図III-2・132(b)].b. 要目表要目表には,原則として歯切り,組立,検査などに必要な事項(歯車歯形,基準ラックの歯形,モジュール,圧力角,歯数,基準円(基準ピッチ円)直径,歯厚の測定方法と基準寸法・寸法許容差,仕上げ方法,精度,備考など)を記入する.材料,熱処理,硬さに関する事項は必要に応じ表の備考欄または図中に記入する.c. かみ合う歯車の図示かみ合う一対の歯車のかみ合い部分の歯先円は,ともに太い実線で表す.主投影図の断面図示では,かみ合い部の一方の歯先円を示す線は細い破線または太い破線で表す(図III-2・133).簡略図は図III-2・134の図例による.
