# 2・8・4 転がり軸受製図

> **章**: 第2章 製図と図面
> **節**: 2・127 に示すような略図で表す.谷底および不完全ねじ部を表す線

転がり軸受はそのほとんどが軸受製造業者により製造されるので,一般には転がり軸受を製作するための図面を作成することはない.転がり軸受の正確な形状および詳細を示す必要のない組立図中では,略図による図示が適用できる.a. 軸受の輪郭と内部構造との概略を図示する場合比例寸法による作図方法により,転がり軸受の形式が了解できる程度に輪郭と内部構造を図示する(図III-2・141).図例以外の形式の転がり軸受の作図方法は文献(29)を参照のこと.b. 簡略図示方法により図示する場合簡略図示方法には,図示する対象の種類,製図の尺度および書類作成の目的によって,転がり軸受を四角形および四角形の中央に直立した十字で表す基本簡略図示方法と,要素(表III-2・16)により列数または調心など転がり軸受をより詳細に示した個別簡略図示方法のいずれかを用いる.図III-2・142に転がり軸受の個別簡略図示方法の例を示す.
