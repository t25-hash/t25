# 3・2・2 ステンレス鋼

> **章**: 第3章 機械材料の標準形状と素材例
> **節**: 3・2 鉄鋼材料

ステンレス鋼(例SUS:steel,use,stainless)はその成分からCr系およびCr-Ni系に、またその金属組織より、フェライト系、オーステナイト系、マルテンサイト系、析出硬化系に大別される。フェライト系ステンレスは16~18%のCrを含み、通常Niは含まない。代表例のSUS 430(16~18%Cr,C<12%)はおもに耐食・耐熱性を必要とする化学工業用装置、器具そのほか一般家庭用品まで広い用途をもつ。オーステナイト系ステンレスは多くの腐食環境において優れた耐食性を示す。加工性、成形性、溶接性にも優れ、各種の化学装置に広く利用され、また、建築、食品製造設備、熱交換器、厨房用品、自動車部品、車両、原子炉関係、発電関係など広く用いられる。典型的な鋼種はSUS 304(18Cr-8Ni0.06C)で、そのシリーズとしてSUS 316(18Cr-12Ni-2.5Mo)、SUS 321(18Cr-10Ni-Ti)、SUS 347(18Cr-10Ni-Nb)がある。マルテンサイト系ステンレスは、耐食性は十分ではないが、焼入れ、焼もどしにより優れた機械的性質を得ることができる。代表例の12~13%Crを含むSUS 410(0.1C)はバルブシート、ポンプシャフトなどの機械部品、刃物、SUS 420(0.2C)はばね、ボルト、プラスチックの型、刃物などに用いられている。4―229析出効果系ステンレスには、固相組織によりマルテンサイト系(代表例にSUS 630)とオーステナイト系(SUS 631)があり、主としてジェットエンジン部品、航空機やロケットなどの構造用、高級ばねなど強さと耐食性、耐熱性を必要とするものに用いられる。
