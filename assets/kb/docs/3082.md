# 5・1・5 温度（temperature)

> **章**: 第5章 各種変量の測定
> **節**: 5・1 基本量の測定

a.熱力学温度（thermodynamic temperature) 自然科学や
工学の分野で扱う温度は，通常は熱力学温度（またはそれから派生
するセルシウス温度）である．熱力学温度は，特定の物質の性質に
依存せずに，熱力学あるいは統計力学の中で定義される温度であ
り，理論の出発点をどこにおくかに応じて互いに等価な種々の定義
法がある．
測定方法に比較的関連付けやすい熱力学温度の定義法は，二つの
熱浴の間で働く可逆カルノーサイクルを行う熱機関が熱浴との間で
授受する熱量の比に基づくものである．ある任意の方法で表した温
度θを仮定し，それぞれの温度がθおよびθ （ただし，θ≠θと
する）である二つの熱浴RおよびRの間で可逆過程のカルノー
サイクルを行う熱機関を考える．この熱機関の作業物質が1回のサ
イクルの間に熱浴RおよびRから受け取る熱量をそれぞれ お
よび とすると， と は符号が逆で，作業物質の種類に依存
せずに任意のθとθの組合せに対して＝θθを満
足する関数 が存在する．そこで，＝θにより改めて定義さ
れる温度 を熱力学温度と呼ぶ．熱浴RおよびRの熱力学温度
をそれぞれ および とすると，＝が成り立つ．
b.熱力学温度の単位 上述のように熱力学温度の比は熱
量の比により表せるが，熱力学温度を数値で表す場合には定数倍の
任意性が残る．この任意性を取り除くためには，ある特定の熱平衡
状態の熱力学温度値を定義することにより熱力学温度の単位を定義
することが一つの解決法となる．
国際単位系では，「熱力学温度の単位，ケルビン（kelvin）は，
水の三重点の熱力学温度の1/273.16である」と定義されている．
つまり水の三重点の熱力学温度は273.16ケルビンと定義されてい
る．ケルビンの記号はKである．
c .セルシウス温度 国際単位系では，セルシウス温度
は，熱力学温度 の熱力学温度＝273.15Kからの差，＝
－として定義される． は氷点（ice point ）と呼ばれる．セ
ルシウス温度（Celsius temperature）の単位はセルシウス度
（degree Celsius ），単位記号は℃であり，その大きさはケルビンと
等しいと定義されている．したがって， ℃＝K－273. 15が成
り立つ．
熱力学温度およびセルシウス温度を表すときは単位としてそれぞ
れケルビンおよびセルシウス度を用いるが，温度差を表すときはど
ちらの単位も使用できる．
d.熱力学温度の測定，一次温度計（primary thermometer )
熱力学温度を測定するためには，熱力学温度との関係が理論的に知
られている測定可能な物理量に置換えて測定しなければならない．
この目的に用いる温度計は一次温度計と呼ばれる．
a項で示したように可逆カルノーサイクルを実行して授受される
熱量の比を測定すれば，熱力学温度の測定が可能である．しかし，
この方法は高精度の測定が難しく，実行されることはまれである．
比較的よく用いられる代表的な方法は，気体の状態方程式に基づく
気体温度計（gas thermometer ），音速温度計（acoustic gas ther-
mometer ），電気抵抗の熱雑音電圧と熱力学温度の関係（ナイキス
トの定理）に基づく雑音温度計（noise thermometer ），熱放射と
熱力学温度の関係に基づく放射温度計（radiation thermometer ）
などである．
e.二次温度計（secondary thermometer ) 一方，温度計の
出力となる物理量と熱力学温度の関係が理論的に高精度には知られ
ていない温度計も多数存在する．そのような温度計でも，その出力
量と別の一次温度計により得られた熱力学温度値との関係をあらか
じめ決定しておくこと（校正）により，熱力学温度の測定に使用で
きる．このような温度計は二次温度計と呼ばれる．ガラス製温度計
（liquid- in-glass thermometer ），熱電対（thermocouple），抵抗
温度計（resistance thermometer ）など実用に広く用いられている
温度計の多くはこの範ちゅうに含まれる．
一般に一次温度計は，副次的な効果による理論からのずれの低減
と補正項の評価などが必要なため，測定手順が複雑で測定時間が長
く装置が大掛かりになる傾向がある．また，温度測定の精密さ・分
解能・安定度・再現性のみに着目すれば，一次温度計より優れた二
次温度計も多い．しかし熱力学温度測定の正確さに関しては，二次
温度計がそれらの校正のよりどころとなった一次温度計を超えるこ
とはもちろんない．
なお，温度測定の専門家の間では，熱力学温度（の）測定という
場合は，一次温度計による熱力学温度の測定を指し，二次温度計に
よる温度測定は含まないことが多い．一方，単に温度（の）測定と
いう場合には，両者が含まれることが多い．本項も以下ではこの用
語の使い分けの慣習に従う．
f .温度標準（temperature standards ) 温度標準が，熱力学
温度の単位を定義する水の三重点を唯一の起点として一次温度計に
より直接設定されれば，温度標準の構造は単純になるであろう．し
かし，現在の温度標準はそのようになっていない．これはe項で述
べたように，現在のところ，二次温度計の中に温度測定の再現性・
安定度・分解能・精密さなどの点で一次温度計よりも優れ，また同
種の温度計間の温度特性の個体差も小さいものがあることに起因す
る．そのような二次温度計の出力量と熱力学温度の関係式を一次温
度計によりあらかじめ決定し，その関係によって温度標準を定義す
るという方法が現在は採用されている．
この定義方法は，温度標準を一次温度計により定義するよりも，
再現性と分解能の点で優れた温度標準を設定できる．それと同時に
その温度標準による温度値は，同種温度計間の個体差および一次温
度計による熱力学温度測定の不確かさ以内で熱力学温度値に一致す
ることになる．
g.温度定点（fixed point of temperature) f項で述べたよう
に，熱力学温度の単位は水の三重点（triple point ）により定義さ
れている．一般に，温度の単位の定義，温度目盛の定義および温度
計の校正などのために温度の基準として用いられる温度再現性のよ
い熱平衡状態は，温度（の）定点と呼ばれる．
温度定点としてはおもに純物質の相転移点が用いられる．なかで
も，三重点，特定の圧力のもとでの凝固点や融解点，あるいは特定
の蒸気圧での液相-気相の熱平衡状態などが多く用いられている．
以下に述べるように，現在の温度標準の定義にも多くの温度定点が
利用されている．
h.国際温度目盛 国際的に合意が得られ現在通用している温
度標準の設定方法は，1990年国際温度目盛（ITS-90:The Inter-
national Temperature Scale of1990）および0.9mK-1K暫定低
温度目盛（PLTS-2000:The Provisional Low Temperature
Scale from0.9mK to1K）により定義されている．これらの温度
目盛は，温度標準の設定に用いる温度計の種類・仕様を特定し，そ
の出力量と熱力学温度の関係式を規定している．その関係式は，一
次温度計による測定の結果に基づいて，それぞれの温度目盛を採択
する時点において最も確からしいとして国際的に合意が得られた熱
力学温度値に一致するように決められている．
科学技術の進歩に伴い，温度測定の手段にも改良が加わり，一次
温度計による熱力学温度測定の温度範囲拡大や精度向上，二次温度
計の使用温度範囲拡大や再現性・出力量測定精度の向上，および温
度定点の温度幅低減や再現性向上などが実現してきた．これに伴い
温度標準の定義も，熱力学温度値により近づけるため，定義温度範
囲を拡大するため，および再現性を向上するために改訂が行われて
きた．
最初に国際温度目盛と呼ばれた温度標準は1927年国際温度目盛
（ITS-27）であるが，その後，1948年（ITS-48），1960年
（IPTS-60），1968年（IPTS-68），1975年［IPTS-68(75)］と改
訂された．なお，1960年から1989年までは国際実用温度目盛
（IPTS:International Practical Temperature Scale）と呼ばれ
た．IPTS-68(75)の下限温度は13.81Kであったが，1976年0.5
K-30K暫定温度目盛（EPT-76:Echelle Provisoire de Tempera-
ture de1976）が採択され，その温度範囲は1990年にITS-90の中
にほぼ取り入れられた．ITS-90の下限温度は0.65Kであるが，
2000年に0.9mKを下限温度とするPLTS-2000が採択された．
