# 3・2・4 鋳鉄

> **章**: 第3章 機械材料の標準形状と素材例
> **節**: 3・2 鉄鋼材料

ねずみ鋳鉄品(例FC250:ferrum,casting,引張強さ250MPa)は一般鋳物品として広く用いられる.一般に引張強さ250MPa以下を普通鋳鉄,以上を高級鋳鉄と称している.可鍛鋳鉄品(JIS G 5705)は普通鋳鉄と比較して大きな引張強さと伸びを有し,黒心可鍛鋳鉄品(例FCMB 270-05:FC,malleable,black,引張強さ270MPa,伸び5%)などがある.球状黒鉛鋳鉄品(例FCD 400-18:FC,ductile,引張強さ400MPa,伸び18%,JISG 5502)は,鋼に近い引張強さと伸びを有するため,鋳鋼品に代わり構造部材に広く利用されている.
