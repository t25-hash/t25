# 5・2・2 角度

> **章**: 第5章 各種変量の測定
> **節**: 5・2 幾何学量の測定

角度（angle）とは，1点より発する二つの半直線のなす図形の
ことである．角度の大きさは円弧の長さと半径の比で表すことがで
きる．角度の単位として，SI補助単位であるラジアンは，円周の
上でその半径の長さに等しい円弧に対応する中心角で定義されてい
る．また，SI単位ではないが，実用上では度（°），分（′ ），秒
（″）もよく用いられている．1°は円周を360等分した円弧に対す
る中心角で定義されている．1°の60分の1が1′ ，1′の60分の1
が1″である．なお，1ラジアンは360° /2 π（約57° 17′ 44.6″ ）であ
る．
角度の測定方法には次のようなものがある．

table) 図5･56に示すように，上部の回転台と下部の固定台

繰返し性が0.02″のものが市販されている．

合わせることによって，精密角度インデックスを自律校正すること
ができる ．

.重力を利用した測定
(１)水準器（level )
ガラス管にアルコールまたはエーテルをごくわずかな空気を残し
て封入した気泡管を用いて，水平面または鉛直面に対する傾きを測
定する．本体の形状によって，四辺が測定面となっている精密角形
水準器［図5･57(ａ)］と，底面だけが測定面となっている精密平
形水準器［図5･57(ｂ)］がある．これらはそれぞれ，気泡管の気
泡の長さが調節できるA級と調節できないB級に分けられてい
る．水準器の感度は，気泡管の気泡を1目盛（2mm）変位させる
のに必要な傾斜を，底辺1mに対する高さまたは角度秒で表す．
角形および平形とも底面の長さ は150，200，250，300の4種
で，感度は第1種0.02mm/m（約4″ ），第2種0.05mm/m（約
10″ ），第3種0.1mm/ m（約20″ ）となっている．JIS B7510：
1993にはないが，感度が0.01mm/ m（約2″ ）のものも市販され
ている．
また，マイクロメータを組込み，傾斜角測定に用いるものもあ

(６)ロータリエンコーダ（rotary encoder )
ロータリエンコーダは回転角を符号化しパルス信号として出力す
るディジタル角度検出器で，一般には360°以上の回転角を対象と
している．回転運動のサーボセンサとして用いられる場合が多い．
回転軸に取付けた目盛円板の円周上に光学的あるいは磁気的に作っ
た周期的なパターン（光学スケールあるいは磁気スケール）を目盛
として読み取ることによって回転角を測定するが，分解能が高いと
いう特徴をもつ光学式エンコーダ（photoelectric rotary en-
coder ）が主流となっている．1回転の目盛数が数百から数十万に
なっており，さらに目盛間を電気的に数十から1000等分に内挿し
て分解能を高めている．パルス数が多いものでは，光干渉読取り方
