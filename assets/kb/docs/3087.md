# 3・3・4 鉛と鉛合金

> **章**: 第3章 機械材料の標準形状と素材例
> **節**: 3・3 非鉄金属

鉛は鉛板(JIS H 4301)として化学工業用容器の他、鉛バッテリー、放射線遮へい材などに用いられる。鉛管はJIS H 4311に規定されている。合金としては活字合金、軸受合金、はんだなどに用いられている。最近は鉛のもつ毒性の問題から、鉛フリー化が進められている。鉛またはすずを主成分とする軸受合金をホワイトメタル(記号WJ)と呼び、1~5種(WJ 1~5)がすず基合金、6~10種(WJ 6~10)が鉛基合金である。軸になじみやすく摩擦係数が小さいなどの特徴をもつ。文献(1) JIS ハンドブック2005, 1鉄鋼I, 参考1. 鉄鋼記号の見方、(2005)、1461-1462、日本規格協会。(2) JIS ハンドブック2005, 3非鉄、 参考非鉄金属記号の表し方、(2005)、1257-1259、日本規格協会。
