# 5・2・4 表面性状（surface texture）

> **章**: 第5章 各種変量の測定
> **節**: 5・2 幾何学量の測定

a.定義と算法 表面性状とは，製品の幾何特性仕様の一つで
あり，表面を構成するすべての波長成分からなる表面の特性をい
う．実際には機械加工面を扱うことが多く，加工表面は波長成分に
よって粗さ，うねり，幾何偏差（現行の粗さの規格 では二次元
断面を扱っているので，すべての成分を含むものとして断面曲線と
している）に分けて考えられており，それぞれ測定法，パラメータ
の定義，図面表示法等が定められている．従来，これらは定義や測
定法が独立で異なっていたが，信号処理の観点から統一的に扱うよ
う変更された．
図5･88は，その概念をプロセスの流れとして示した図である．
表面は極力忠実にディジタル情報に変換され，フィルタ処理によっ
て定義された各波長成分に分離される．最も短い波長域の粗さ曲線
（roughness profile，Rパラメータと呼ぶ），中間域のうねり曲線
（waviness profile，Wパラメータ），全波長成分を含む断面曲線
（primary profile，Pパラメータ）は，それぞれ同じ考え方で各成
分の高さ方向，幅方向，複合，分布などのパラメータが定義され，
体系化されている．
.粗さ 粗さの定義は，国際的にはISO4287：1997に定
められているが，わが国ではJIS B0601-2001 に従う．近年，
表面性状を三次元的に扱うことも可能になっているが，二次元断面
曲線のパラメータは，表5･22のようになっている．粗さは，形成
されるプロセスにもよるが，ランダムな場合も多いことから，統計
的に安定なパラメータである による指定が推奨されている．
粗さを求めるためのフィルタとして，ガウシアンディジタルフィ
ルタを離散データにかけることが二次元の輪郭曲線で推奨されてい
る．しかし，ガウシアンフィルタ（Gaussian filter ）においては，
評価に必要なデータの前後にカットオフ波長の半分ずつフィルタの
ために原データが必要となること（端効果）から，測定時間や測定
コストを意識する場合や測定できる領域が狭い場合に問題となる．
三次元の面領域のデータにおいては，畳み込み演算に時間がかかる
こと，端効果のために二次元より格段に多くの測定をしなければな
らないことなどから，原データの測定ならびにそのフィルタ処理に
多くの時間を要する．
表5･23 は，粗さ，うねりの波長帯域を振幅の大きさとともに
示した推奨値であり，この表から粗さの波長の範囲が約300倍であ
ることがわかる．

.ガウシアンフィルタによらない粗さ曲線，うねり曲線の算
法 i項で述べたように，表面性状は通常ガウシアンディジタル
フィルタで各成分に分離されるが，端効果を考えると測定長がとれ

い形状においてガウシアンフィルタが形状をゆがめるような場合に
は，いくつかの別の分離方法が提案されている．
(１) モチーフ法（motif method)
モチーフ法とは，断面曲線を図5･89に示すような左右の高い山
とその間にある谷を一つの特徴的な要素（モチーフ）とし，一定の
ルールのもとで断面曲線をこの要素に区分し，抽出されたそれぞれ
のモチーフの両端の高さとモチーフの幅に関するパラメータ（モチ
ーフパラメータ）で粗さの特徴を表すものである．さらに，モチー
フの左右の高い山の頂点どうしを直線でつないだものを粗さを除い
た断面曲線とし，粗さモチーフより広い幅のうねりモチーフを適用
して，同様にモチーフ解析することによってうねりモチーフのパラ
メータを求めることができる．
この場合，粗さ曲線は粗さモチーフの左右の高い山を結ぶ直線が
うねり曲線（すなわち直線）となるため，左右の高い山を高さ零と
するように変換される．これは，各モチーフ内に油をどの程度蓄え
ることができるかを表す指標として，トライボロジーの分野で有効
なものとなる．
(２) モルフォロジカルフィルタ（morphological filter )
画像認識の分野で用いられるモルフォロジカルフィルタを断面曲
線に適用する．一般的なガウシアンフィルタでは大きなくぼみの入
口や出口付近でフィルタ曲線の落込みを生じ，フィルタ曲線との差
である粗さ曲線はくぼみの入口と出口で逆に持ち上げられてしまう
特性を有する．図5･90はトランケーション面の断面形状と，その
フィルタ出力の例である．このことが好ましくない接触などへの応
用においては，モルフォロジカルフィルタや前述のモチーフ法が有
効に作用し，粗さ曲線の持上りを生じない．
b.測定法 測定法としては，触針式スタイラスがその安定
性，SN比の高さ，規格が社会に浸透していることなどから標準測
定法となっている．しかし，三次元測定への要求やnmオーダの超
精密加工面への測定要求にこたえるべく，顕微光学系において対物
レンズあるいは測定物をピエゾ素子などを用いて高精度に光軸方向
に駆動する技術の開発と相まって，干渉じまスキャンや共焦点光学
系を応用した非接触測定も普及している．より微細な測定分野にお
いては，特に横方向の分解能の高いSTM（scanning tunneling
microscope，走査型トンネル顕微鏡）やAFM（atomic force
microscope，原子間力顕微鏡）が用いられるが，駆動素子の制約
から150m程度以下の測定範囲しかとれない点において，粗さの
測定とは一線を隔している．
.触針式粗さ測定法 最も一般的であり，かつ標準的な表
面微細形状測定法である．ダイヤモンドの触針（stylus ）先端を
2～10mRとして，接触荷重0.7～4mNで被測定面上を走査し，
触針の上下変動を差動トランスで検出する．触針が被測定面をなぞ
った際の軌跡であることから，光を用いた方法などと違って安定し
ており，測定結果がそのまま幾何学的断面形状と理解されている．
触針法では，軽荷重であっても先端半径は小さいため，接触応力
が大きい．したがって，軟質金属などでは測定面にきずがつくばか
りでなく，表面を耕しながら進んだ軌跡が出力される．また，極力
軽荷重としている一方で検出系の慣性が大きいため，測定時の走査
速度が速すぎると，触針が被測定面からジャンプしてしまう．ジャ
ンプしていてもその箇所は出力波形の中で区別できないため，使用
する触針の仕様に合わせ，あるいは被測定面の凹凸形状の大きさな
どに応じて走査速度を選択する必要がある．さらに，長期使用にお
ける触針先端の摩耗など触針先端形状の管理や，被測定形状が微細
になった結果として，触針による包絡軌跡が原形状を大きくゆがめ
たものとなる影響なども，形状を忠実に把握するうえで問題となり
つつある．
軟質材などの測定や接触させたくない対象の測定では，光触針が
使用される．原理には，フォーカスエラー，臨界角，非点収差など
を利用し，中立位置との差を変位として直接出力する方式と，合焦
させる方式がある．比較的大きな粗さや断面形状の測定において
は，三角測量方式の光学式変位計などが用いられる場合もある．
.微細な表面性状の測定技術 表面性状の高さ方向の変動
が10nm程度までは，触針式粗さ計などで測定可能であるが，そ
れ以下の変動に対しては，より高分解能の測定機が必要となる．ま
た，近年の三次元測定の要求や形状の三次元表示機能の要求への対
応もあって，AFMに代表されるSPM（scanning probe micro-
scope）や，共焦点光学系の特徴や顕微干渉光学系を応用した表面
形状測定技術などが，測定データの表示機能やパラメータ演算機能
を装備したソフトウェアとともに使用される．
(１)SPM
探針を可能な限り尖らせ，被測定表面との間に作用するトンネル
電流や原子間力，エバネッセント光，その他の物理的現象を一定に
保つように表面と探針との距離をピエゾ素子を利用して制御し，そ
の探針を制御した高さを表面形状とする一連の測定技術である．非
導電物質の測定，バイオ関係などの軟らかい物質の測定等について
は，AFMを振動モードで使用する測定法が低測定力，安定性等の
点で普及している．これらの測定法の最大の特徴は，高い高さ分解
能もあるが，特に高い横分解能にある．一般的な粗さ計の触針や，
顕微光学系では横分解能は0.1～0.5m程度であるのに対し，表
5･24に比較するようにサブナノメートル程度と，約3けたほどの
違いがある．開発は原子像の観察という究極の測定の成功から華々
しくスタートしたが，近年では工学的応用の観点から，より広い視
できる．
野への適用に力が注がれ，高分解能レーザ干渉測長技術などを併用
してワイドレンジ化が図られ，また真空中や空気中だけでなく培養
液中や各種環境下での測定も可能となっている．
測定データは，高さを擬似カラー化して鳥瞰図や等高線で表した
り，任意断面の二次元形状や元の三次元形状データに基づく規格化
されているパラメータや，その他の多くのパラメータを算出できる
ようになっている．
(２) 顕微干渉光学系（interferometric microscopy）とその垂
直走査による高精度測定法
ミラウタイプやマイケルソンタイプの干渉光学系を対物レンズに
ておくことが重要である．
組込み，白色光源によって参照鏡との絶対等距離による干渉じまを
観察し，対物レンズを光軸方向に微動させながら測定を繰返し，複
るものなどいくつかの方法が提案されている．
数の干渉じま画像とそのときの対物レンズの高さ情報から，撮像素
