# 1・2・1 測定の尺度

> **章**: 第1章 計測とは
> **節**: 1・2 尺度と次元

何らかの量を計測するということは,その背後に量的概念が存在し,その概念に基づいてその量に対応する数値などのラベル付けがなされる.量的概念とラベル付けした数値との間の関係により,この数値の並びである尺度(scale)のもつ性質は左右される.数値の同一性,順序性,等質性,加算性などの性質により,尺度は名義尺度,順序尺度,間隔尺度,比例尺度の大きく四つに分類される.名義尺度(nominal scale)は,対象を分類し,相互の区別を,数を利用して行うものである.分類された各群においては,同一群には同一の数を用い,逆に異なる群には同一の数を用いないようにする.数の大小や順序はなんら量的意味をもたず,数の同一性のみが意味をもつ.試料の番号や背番号などはこの例である.順序尺度(ordinal scale)は,対象をその大小や優劣などの観点で並べ,その並べられた順に合わせて数を順に割り当てるものである.数の順序性を利用した尺度であり,相互の大小(優劣)関係のみを表す.モース硬度などがその例であり,相互に引っかき試験をすることによりその順序が定まる.しかし,その順序を表す数は順序以外の量的意味をもたない.間隔尺度(interval scale)は,対象に与えた尺度値間の差が等しければ,その差に対応する対象の量的違いもつねに同一の物理的意味をもつ尺度であり,数の等質性を利用したものである.セルシウス温度目盛はこの例であり,20°Cと40°Cは,20K の差に相当するポテンシャルの違いであり,決して40°Cは20°Cの2倍のポテンシャルを有しているわけではない.比例尺度(ratio scale)は,対象に与えた尺度値の数が,和や比など数の演算結果によって得られた数値においても量的意味をもつものである.長さ,体積,質量,電流,電圧,時間,絶対温度などはこの例である.5―1
