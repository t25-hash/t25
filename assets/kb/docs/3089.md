# 1・2・2 測定の次元

> **章**: 第1章 計測とは
> **節**: 1・2 尺度と次元

測定は、その対象と物理的性質が等しい基準とを比較し、その量の大小を決定する。この比較を両者の比で表し、一方を一定の基準量に選ぶことによってその比の値は無次元の上記比例尺度になる。この基準量は、その物理的性質に応じて、長さや質量、時間など、互いに独立とみなせる物理量の組合せとして表せる。この組合せは=욏욐욑...のように基本となる物理量の組合せで表せる。ここで、は対象となる測定量、,,はそれぞれ基本となる独立な物理量、,,は整数であり、は定数である。一般にの次元をと表す。上式を次元の式で表すと、=욏욐욑...となる。次元と単位系に関しては、『機械工学便覧』α9編「単位・物理定数・数学」に詳述しているので、物理量の次元や次元式についてはそちらを参照されたい。쓕器」と「機」測定器と測定機はどう違う?どう使い分ける?といった疑問があり、本編でもその点は特に明確にせず、各執筆者に任せている。何となくわかっているようで、正しく定義しようとすると明確に答えられないところである。以下に、編集子が理解している範囲でその違いを述べる。国語辞典では、「機械」=物体を組合わせ、動力によって一定の運動を起こし、有用な仕事をするものであり、原動機構・伝導機構・作業機構(・制御機構)からなり、その一機構を欠けば「器械」とある。一方、「器械」=道具、器物となっている。従来から、モータなどの原動機構があるものは「機械」といわれてきたゆえんである。測定機器においては、ノギスやマイクロメータ、フラスコやはかりなど、可動メカニズムは内包していても、被測定物を載せるなどの動作の結果、受動的に動く機構であるものは「器械」または「器具」等であり、「測定器」と呼ぶべきであろう。一方、従来は測長器や粗さ測定器など「器」を使って呼んでいたものは、最近ではモータを内蔵した自動測定システムとなっており、三次元座標測定機に代表されるように「測定機」が使われている。英語標記では、「測定機」はmeasuring machineであり、「測定器」はmeasuring instrumentやmeasuring apparatus(あるいはequipment,device)である。測定の標準となる実量器は、上述の考えからも「器」のままでよい。文献(1) JIS Z 8103:2000.→ Next Chapter
