c.流速の測定法と標準
.単位 流速（flow velocity，flow speed）とは，空間の
中のある1点を通過する流体が単位時間当たりに移動する距離であ
り，通常使われる単位はm/sである．本来，流速は速さと向きを
もつ三次元のベクトル量であるが，工学的には速さのみのスカラ量
を指すことが多い．
.流速の標準 従来，流速の基準としては標準ピトー静圧
管が用いられてきたが，近年の主要国の国家標準としては回転円盤
を利用したレーザドップラー流速計校正装置が採用されている．日
本の国家標準として使用されている校正装置 を図5･130に示
す．一定速度で回転する円盤の外周に細線を取付けると，この細線
の回転直径と角速度から細線の速度を正確に算出することができる
ので，この細線をレーザドップラー流速計（LDV）の測定部を通
過させることで発生するレーザ散乱光のドップラーシフトと細線の
速度を比較してLDVを校正する．校正範囲は1.3～40m/ sで，校
正の拡張不確かさ＝2は約0.06～0.5 %である．この装置によ
って校正されたLDVを基準として他の風速計を校正することで，
連鎖的なトレーサビリティ体系を築くことができる．

.レーザドップラー流速計（LDV:laser Doppler
velocimeter ，laser Doppler anemometer ) LDVは流れ場にレ
ーザビームを照射し，流れに含まれる微粒子によって散乱される光
のドップラーシフトから流速を求める．光源としては，測定対象が
液体の場合はHe - Neレーザ，気体の場合はArイオンレーザが一
般的である．
非接触で測定できることが最大の特徴で，火炎の内部の流れや回
転するプロペラ近傍の流れなど，流れの中にセンサを挿入すること
ができない場合には唯一の測定手段となる．また，ⅱ項で前述した
LDV校正装置で校正することにより，高精度の基準流速計として
使用できる．しかし，高価で取扱いに熟練を要することと，流れの
中に散乱粒子が必要なことが欠点であり，工業的に使用されている
例はまだ少ない．最近の技術の進展により，半導体レーザを光源と
してパーソナルコンピュータによる信号処理を行うなど低価格化の
努力も払われているので，今後流量計などに応用される可能性もあ
る．
.熱式風速計（thermal anemometer ) センサを加熱し，
流れによる強制熱伝達の増加から流速を求める方式である．実験室
で乱流計測に用いられる熱線風速計と一般用の熱式風速計の2種類
がある．乱流計測用の熱線風速計は，センサは直径5m程度の白
金線あるいはタングステン線であり，極めて高い空間分解能と時間
応答性をもつ．しかし，専用のブリッジや信号処理回路に接続して
用い，測定に先立ち校正と回路の調整が必要であるため，適切な使
用には熟練を要する．
一方，一般用の熱式風速計には，耐久性を増し経年変化を少なく
てきた．
するためにコイル状に巻いた金属線またはサーミスタがセンサとし
