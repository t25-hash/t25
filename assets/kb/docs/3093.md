# 2・2・2 項にある物理的な計量標準が、国家標準としてその供給が

> **章**: 第2章 単位と標準
> **節**: 2・2 計量標準

国家の管理のもとに行われるのに対し、標準物質と目されるものの中には、民間企業、団体において製造・認証されている物質もある。しかし、これらは国際ルールに基づいて品質保証の確認がされていなかったり、また国家標準物質に対するトレーサビリティに関係付けられていなかったりする。今後これらの民間団体による標準物質を国家標準として認証するために、国際ルール(ISOガイド34)に適合した認証手法を確立することが、標準物質供給のうえから重要である。b. 標準物質の種類と整備状況標準物質は大別すると図2・1のように分類される。また、各標準物質は表2・4のように定義されている。わが国は、2010年までに290種類の標準物質を整備することにしている。標準物質の整備が急がれなければならないのは、環境ホルモンなどの極微量でも毒性のある物質(内分泌かく乱物質など)に社会的な関心が高まっていることが影響している。有害な物質が排出されたときに、その物質を同定するには標準物質が必要である。表2・5は2010年を目標年度としたわが国の標準物質整備計画のうち、先端材料標準物質について示している。5―4表2・4 標準物質の定義標準物質物質、材料の濃度やその他の物性値等である特性値を決定するための基準となる物質。ISOガイド30にある定義は、測定装置の校正、測定方法の評価や材料に値を付けることに用いる特性値の一つ以上が均一で、適切に確定している材料や物質化学分析測定用標準物質濃度あるいは組成についての特性値をもつ標準物質純物質系標準物質純物質または水、酸または有機溶媒のマトリックスに溶けている標準物質無機系標準物質金属や無機イオンのような無機化合物の標準物質有機系標準物質有機化合物の標準物質組成標準物質マトリックス中の微量成分または組成を特性値とする標準物質で、鉄鋼、海水、岩石、血清などの標準物質を含む物性・工業量標準物質組成や濃度以外を特性値とする標準物質で、温度、熱特性、密度、硬さ、粘度の物理的特性に関する標準物質表2・5標準物質の整備計画の一例(先端材料標準物質)標準物質供給済み(2006年)2010年までGaAs/AlAs超格子(各層25nm,4層)◎SiO多層膜層(各層20nm,5層)◎各層10nm以下のGaAs/AlAs多層膜◎Si基板上のSiO3~10nmの薄膜◎1目盛25~100nmの面内宝庫スケール◎Si基板上の2nm以下の金薄膜◎イオン注入ケイ素系物質◎酸化ケイ素薄膜中の空孔標準◎◎高分子中の超微細空孔標準◎多分散高分子標準物質◎◎超微粒子標準物質(<50nm)◎
