# 2・3・3 ISO規格と測定

> **章**: 第2章 単位と標準
> **節**: 2・3 工業規格

Standardという語が文書としての規格と,計量標準や単位の意味を併せ持つことは,両者の間に日本語からは見えにくい歴史的関係があることをうかがわせる.大きくいってその関係は二つの面に集約される.その一つは,歴史的に計量単位およびその値を定める標準の歴史は,多様から単純への歴史であること,つまりある単位系の普及の程度は国家または為政者の統治力の一つの指標となると考えられることから,統一された計量単位系の整備は国家体制の維持に重要な意味があったことである.整備された単位系を記述する文書は国家の強制規則として施行されることになったが,単位系の使用方法などの標準化は民間レベルに任されている.計量単位系を定める文書が規格としての性格をもっていること,換言すれば今日の規格は計量単位系を強制する規則(文書)をそのスタートとしているということがいえる.もう一つは,ISO規格などの工業規格(標準)は,計量単位の裏付けがあって初めてその機能を発揮できることである.例えば,製品の品質を規格で定めているとき,供給者と購買者間で製品の規格への適合性を評価する際,多くは計量単位による数値評価が用いられる.SI単位系は,過去の単位の特性である多様性を単純化して,世界中で共通に使用されることを目的に作られたもので,ISO 1000の規格番号をもつISO規格である.ISO規格そのものは強制力をもつものではないが,必要に応じ国家がそれぞれ法律により強制力をもたせている.
