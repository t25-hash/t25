# 2・3・4 ISO規格制定のプロセス

> **章**: 第2章 単位と標準
> **節**: 2・3 工業規格

ISO規格は2007年現在16500規格以上が制定されており,毎年1250規格が制定されている.分野別の規格数では材料技術,機械技術がそれぞれほぼ25%を占め,エレクトロニクス・通信技術,運輸・流通技術が続いている.ISO規格はISO会員団体(実質的に国を代表する)から選ばれたエキスパートで構成される技術委員会(TC: Technical Committee)において審議され,合意が得られるとその文書(DIS : Draft International Standard)を各国の会員団体に送付して投票を行い,コメントを求める.投票結果がポジティブのときは,コメントをしんしゃくした最終文書(FDIS :Final Draft International Standard)を作成し,再度会員団体に送付して投票を行う.投票結果がポジティブのとき,ISO規格として発行される.最近は,インタネットを利用した審議が増加している.
