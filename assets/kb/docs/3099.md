音響パワーレベルを厳密に決定するには，音源を囲む面でのイン
テンシティの法線方向成分を積分することになる．実際の計測では
外部からの音波による成分も含めて計測されるが，この成分は閉曲
面への出入りが打消され，音源からのエネルギーのみが求められ
る．しかし詳細な積分は現実的ではない．そこで，ISO3741：
1999～ISO3746：1995，ISO9614-1：1993～ISO9614-3：2002
には有限の測定点数からこれを推定する方法が何種類か示されてい
る．通常は無響室（anechoic chamber ），半無響室で音圧のみを計
測する方法，残響室で計測する方法，そしてv項で述べたインテ
ンシティ計測による方法がある．
無響室あるいは半無響室では，音源からある程度離れた場所で音
波はほとんど平面波としてふるまうので，式(5･63)に基づき，音圧
のみを測ればよい．計測点数のガイドラインはJIS（またはISO）
に規定されている．残響室では，エネルギーの発生と散逸が釣り合
うことから求める．
.その他の音響計測法 騒音を計測するに際して，その音
源構造を明らかにしたいことも多い．その場合，いろいろな音源の
可視化法が用いられる．一次元アレイによる超指向性マイクロホン
や，二次元アレイによるビームフォーミングでの音源方位の探査，
音響ホログラフィーによる音源の可視化などが行われるが，設備が
大掛かりになったり，演算処理が複雑になることもある．対象をよ
く観察して問題点を整理し，要求に対して適切な計測方法を選択す
ることが重要である．
d.信号の解析 信号の解析には，フーリエ解析を用いた周波
数分析が一般的に用いられるが，これは主として時間平均をとるこ
とが多い．これに対して時間変化を含めて信号を解析する手法とし
て，短時間フーリエ解析，ウィグナー分布（Wigner - Ville distri-
bution)，ウェーブレット変換（wavelet analysis )，瞬間周波数
（instantaneous frequency)，声紋分析（sound spectrogram）と
いった手法やキーワードも存在する．
また回転機械については，回転次数分析（order analysis ）と称
して，横軸を回転周波数の整数倍として分析することもある．いず
れも，適材適所での活用が重要であって，どれかが万能な手段とい
う認識は不適切である．
信号の解析に一般的に用いるFFTは，昨今では道具として容易
に用いることができるため，その中身については忘れがちである．
そこでは離散フーリエ変換（DFT:discrete Fourier transform）
を行うが，その高速演算処理方法である高速フーリエ変換（FFT:
fast Fourier transform）が用いられている．このプロセスの詳細
は専門の解説書に譲るが，活用のための留意事項を以下に挙げる．
(１)ウインドウ関数
解析する信号の長さがこれに含まれる調和成分の周期の整数倍に
なっていないとき，いわゆるリーケージ（leakage）が発生し，真
の周波数を中心として，その周囲の周波数成分が存在し，スペクト
ルに広がりがあるように観測される．この影響を減じるためにウイ
ンドウ関数を用いる．しかし，回転機械のように明白な周期性のあ
る信号を分析したいときには，その回転周期とデータの長さが一致
するように調整することと，その周期に同期をかけて分析すること
でウインドウ関数を用いない（矩形ウインドウと同値）ことも肝要
となる．またこのような測り方では，位相も意味をもつ．
(２)平均化
定常的な振動を解析するといっても，現実には外乱などの不確定
な要因が存在するので，平均処理が必須である．しかし，対象とす
る振動に周期性がある場合とない場合とで平均化の仕方にも配慮が
必要で，上述のような回転機械では回転トリガを用いたり，用いな
いという選択肢がある．
(３)周波数分解能
定常な振動現象には，周期的な現象と不規則（ランダム）な現象
が存在する．これらが混在する信号を周波数分析するとき，前者は
周波数分解能を変えてもその振幅の値は変わらない．一方，不規則
な成分（例えば乱流騒音）を分析するとき，周波数分解能を高くし
た場合，個々のバンド幅が2倍になるとレベルが3dB上昇する．
DFTはその分解能のバンド幅に成分をもつ信号のrms値を求めて
いると考えればよい．
(４)定比分析と定幅分析
オクターブバンド（octave band）分析や1/3オクターブバンド
分析と呼ばれる定比（バンド幅）分析と，DFT（FFT）に代表さ
れる定バンド幅分析がある．前者は，それぞれの分析中心周波数に
対して分析するバンド幅が比例しているので，聴感との対比をしな
がらうるささを評価するには便利である．一方，物理現象として，
例えば共振現象などを精密にとらえるなら，狭帯域分析とも呼ばれ
る定幅分析が役に立つ．
(５)波形とスペクトルの関係
フーリエ変換と元の波形との関係を知っておくことも重要であ
る．フーリエ変換は，時間軸上で分散している情報を周波数軸上の
1点に集約する．逆に，周波数軸で分散している情報は，時間軸の
1点に集約されるという関係にある．
