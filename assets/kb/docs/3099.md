# 2・3・5 適合性評価について

> **章**: 第2章 単位と標準
> **節**: 2・3 工業規格

一般に製品、サービスまたはプロセスがある規格を満足しているかどうかを判断することは、製品、サービスまたはプロセスの供給者および購買者双方にとって関心事である。このような判断を行うことを適合性評価(conformity assessment)という。ある機械部品の軸径が指示された寸法公差内に入っているかどうか、すなわちその部品が合格品であるか不合格品であるかは、わかりやすい例である。この場合、軸径を測定すればそれで決着がつくはずであるが、測定の不確かさのために、公差の限界値付近の軸径をもつ部品では合否の判断が揺れるおそれがある。このようなとき、適合性評価が必要になる。適合性評価方法はISOではISOガイドの形で発行されており、ISO規格が国家規格として強制文書であるときも、評価主体は民5―6計測工学間機関が行うのが通例である。このことは、計量標準は計量法に基づいて国家が管理しているが、一般の測定機が国家標準にトレーサブルであることを認証するのは、民間の校正(認証)機関が担うのに似ている。適合性評価の認証(certification)は、供給者、購買者と異なる第三者機関が行うことになるので、それが行う認証の公正さが問題になる。このため認証の公正さを保証する必要があるが、これを認定(accreditation)と呼んでいる。一般に認証作業は当該分野についての高度の専門的知識が要求されるために、認証を行う専門家を養成するための教育・研修制度の整備が必要である。ISOでは、規格の作成作業時に規格に関連して適合性評価、つまり認証制度についても審議されることがある。文献(1) 通商産業省機械情報産業局計量行政室編、 新計量法の概要、(1994)、178、第一法規。→ Next Chapter
