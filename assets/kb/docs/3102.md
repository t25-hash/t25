# 3・1・1 項において述べ

> **章**: 第3章 測定概論
> **節**: 3・1 測定の計画と設計

た測定計画を具体化するに当たって大切なことは、必要にしてかつ十分な判断が立てられるように目的とする因子を割り付けることである。例えば、検討すべき因子がA,B,Cと3種類あったとする。それぞれについて、A,A,A,B,B,B,C,C,Cの3水準を選定する。因子の水準とその間隔をどのようにとるかということが、目的に適した結果を得るためには大切な点であり、専門的な知識、経験に依存するところが大きい。一般的にいえば水準数は3で十分であり、水準の間隔は目的に合わせて選ぶ。A,B,Cの全部の組合せで実験を行う場合、3×3×3=27とおりの実験を行うことになる。このようにすればA,A,Aについての比較はB とCのすべての組合せについての比較であるので、判断に普遍性が生まれる。このように完全な配置を考えた全数実験が理想であるが、それは時間的にも経済的にも大きな投資となる。もしも、A,A,Aの比較をB,Cの条件に固定して行えば、実験は3とおりで済むが、B とCの因子の水準が変わった場合の結果は不明となる。27とおりでは実験規模が大きすぎると考える場合は、実験規模を小さくする方法として、直交配列表(orthogonal array)が利用される。A,B,Cについてならば、表3・1に示すようなL(3)の直交配列表が利用される。下付きの9は9組の実験を行うことを示し、3は因子の水準数である。上付きの4は因子が4種類まで割り付けられることを示している。表3・1中の1,2,3は水準を表している。直交配列表とはJIS Z
