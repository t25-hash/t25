# 3・2・1 直接測定と間接測定

> **章**: 第3章 測定概論
> **節**: 3・2 測定方式の分類

測定における基本は、測定すべき量(測定量)と基準となる量(基準量)との比較を行うことにあるが、測定量と基準量との量的な性格の違いに基づいて分類すれば、直接測定(direct measure-ment)と間接測定(indirect measurement)に分けられる。直接測定とは、測定すべき量をそれと同じ種類の基準と比較する方法であり、例えば質量が既知の分銅とてんびんを利用して試料の質量を測定する場合や、標準尺を用いて長さを測る場合などに代表される。一方、間接測定とは、測定すべき量と一定の関係にある他の量、または測定すべき量を組立てているいくつかの量を直接測定して、それらの測定値と測定すべき量とを結びつける一定の関係に基づいて、測定量の大きさを導き出す方法であって、測定量と基準量との比較が間接的であるところに特徴がある。球体の直径の測定に基づく体積の決定、自動車が一定距離を移動するのに要する時間の測定からその速度を求める場合などが代表的な間接測定に分類される。以上のように、直接測定であるか間接測定であるかは、測定する量によってではなく、用いる測定方法によって決まるのであり、同じ一つの量を測定する場合でも、直接測定による場合と間接測定による場合とが考えられる。一方、量によっては、固体の密度や熱膨張係数のように、直接測定が不可能で間接測定のみによるものも存在する。
