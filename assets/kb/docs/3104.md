# 3・2・2 絶対測定と比較測定

> **章**: 第3章 測定概論
> **節**: 3・2 測定方式の分類

測定方式を分類する第二の立場は、測定すべき量と基準となる量との比較の結果をどのように与えるかに基づくもので、絶対測定と比較測定に分類する概念である。比較の結果、量の大きさについてその絶対値を与える場合が絶対測定(absolute measurement)であるのに対し、比較測定(rela-tive measurement)とは、ある定められた量を基準として、相対的にそれからの差の大きさを表示する方式である。絶対測定の場合、測定はその量の定義に従って実行されることになるが、得られた測定値の正しさを最終的に保証するものは基本単位の原理的な正しさとなる。光の速度を決定する場合、光の波長と振動数の測定といった基本量のみの測定に基づき、組立量としての光速度の定義に従って導出しているわけであり、これは絶対測定の代表例である。他方、比較測定の場合、基準となる量の大きさは明示されないが、その値自体は一定であることが前提であり、何らかの原因で変化する可能性があるならば、適当な別の方法でそれを確認することが必要となる。なお、比較測定に用いられる基準量について、その値の大きさが正しいことを保証するには、絶対測定によって正しい値が得られている量との比較が必要となる。
