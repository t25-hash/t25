# 3・2・3 各種の測定方式

> **章**: 第3章 測定概論
> **節**: 3・2 測定方式の分類

これまでに述べた二つの分類の視点とは別に,測定方式を分類する考え方に,測定すべき量と基準となる量との比較を行うときのプロセス(操作の手順)に着目する観点による分類がある.この観点からの分類によれば,測定方式には,偏位法,零位法,補償法,置換法,合致法,差動法などの方式が存在する.まず,偏位法と零位法は3・2・1項に記した直接測定における代表例であり,次の特徴を有している.偏位法(deflection method)は,測定量が直接的な原因となり,その結果,測定量の大きさに応じて生じる指針などの振れの大きさから測定量を知る方法で,可動線輪型電圧計,ばねばかり,ブルドン管圧力計,水銀温度計などがその典型であり,多くの古典的計器が偏位法を活用している.この偏位法の欠点を閉じた系により補うのが零位法(zeromethodまたはnull method)である.すなわち,大きさ可変の基準量を測定量とは独立に用意し,これを直接測定量と比較させつつ平衡させて,平衡したときの基準量の値から測定量を知るものであり,ポテンショメータを用いる電位差計が代表例である.電位差計では,標準電池による基準電圧をポテンショメータで分割し,可変な基準電圧を発生させ,その値と測定すべき電圧との差を高感度のガルバノメータで読み取り,差が零になるように基準電圧を調整している.この種の調整操作により測定系は一つの閉じた系を構成するが,測定者が手動で調整を行う場合もあれば,調整が自動化されている場合もある.測定の平衡が実現されているかどうかの確認は,測定量と基準量との差の極性を検出する変換器によって行っており,一方,測定系の精度を決めるものは,フィードバック側に入れた要素の特性であり,差を検出する変換器の感度が多少変動してもほとんど測定誤差に影響を与えない.また,零位法において,測定に要するエネルギーは外部から供給されており,測定系からとるエネルギーはわずかである.零位法と偏位法とを組合わせ,測定量からそれにほぼ等しい既知の量を引き去り,残りの量を偏位法的に測定する方法が補償法(compensation method)である.例えば,てんびんで質量を測る場合がよい例であるが,この場合,測定される物体と分銅の質量が正確に一致することはまれであり,不一致分に対するさおの傾きを指針の振れから読み取ってその値を加えた測定結果を出すことになる.また,測定器に対する基準量と測定量の作用のさせ方を互いに置換えた前後の2回の測定結果から測定量を知る方法が置換法(sub-stitution method)であり,測定器に入る系統誤差成分を低減するのに有効である.てんびんによる質量測定において,左右の腕の長さの相違などによるかたより誤差を除くために,物体と分銅を置換した前後2回の測定結果の平均値として質量を定めるのがこれに相当する.一方,合致法(coincidence method)は,ノギスの副尺などに見られるように,間隔の異なる二つの目盛線などの合致の仕方を規定する基準量と測定量の間の一定の関係を利用して,測定量を読み取る方法であり,1目盛以下の端数値を知るのに有効である.さらに,差動法(differential method)は,測定量による影響の受け方が互いに逆特性をなす同一種類の検出系を二つ複合的に配置し,それぞれの出力の差を利用して測定する方法で,単一の検出系に存在する偶数次の非線形性を低減し,検出系に入る同相雑音を除去するのに有効である.その典型例が差動変圧器による変位の測定に見られる.
