# 3・2・4 アナログ方式とディジタル方式

> **章**: 第3章 測定概論
> **節**: 3・2 測定方式の分類

連続量に対するアナログ検出・記録と離散的量に対するディジタル検出・記録は,それぞれの特徴を生かした活用が肝要である.現象を連続的に記録したり,その傾向を目で見て確かめるためには,アナログ量が便利であるが,読取りの手間やそれに伴う誤差を回避するためにはディジタル記録が優れている.しかしながら,腕時計や家庭用時計のように,高性能な人工の水晶振動子が量産されるようになって,一時はアナログ表示の時計が消えかけたように見えたが,ディジタルの味気なさや前後関係の類推が困難なことなどが原因となり,最近ではアナログ表示が巻き返しているように思える.いずれにしても,最近ではアナログからディジタルへ,また,その逆の変換回路や変換器が普及してきており,用途に応じて使い分けることが大切である.
