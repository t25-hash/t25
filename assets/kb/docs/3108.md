# 5・2・1 化学物質の管理と法規制 我々の身の回りには多くの

> **章**: 第5章 放射性物質，有害物質に関する法制度
> **節**: 5・2 化学物質の総合管理

化学物質があり，我々の生活を豊かにしているが，他方ではその化
学物質の取り扱いを誤ると，火災・爆発が起こり，人や環境に対し
て有害となる場合がある．そこで，危険有害な性状を有する化学物
質については，その危険有害性から人，社会，環境：を守ることを目
的として，化学物質を規制する各種の法，規則が定められてきた。
しかし，化学物質に関する管理を十分に行うためには，規制を体系
づけて把握し，それぞれ法規制の特徴ど限界を知り，化学物質のラ
イフサイクル全般にわたって管理する必要がある．
法規制の全体を体系づけて把握するために，「危険有害性からの見
た法体系」，「用途から見た法体系」「防護する対象から見た法体系」
の三つの観点から説明する．
体系中の個々の法令については，大部分は本編の各章で述べられ
ているので，本章では，他章で触れられていない「化学物質の輸送
関係の法」，「新規化学物質に関する法」，「自主管理と情報公開」に
ついて詳しく述べる．
