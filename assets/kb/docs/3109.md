# 3・3・3 不確かさの原因

> **章**: 第3章 測定概論
> **節**: 3・3 測定の誤差と不確かさ

測定された結果の信頼性(信用性)をいかに評価し、表現するかは、測定される状況を詳細に検討することから始まる。一般的な計測において、どのような不確かさの要因が存在するかを推定することが先決であるが、いわゆる4W 1H の、いつ、どこで、誰が、何を、いかに測定したかの状況そのものが不確かさを生じる根本的な原因として考えられ、これらの要因がどのように結果に寄与するかを見積もることが不確かさの評価につながる。この概念は、工程管理における特性要因図の考え方と一致する。図3・2は、不確かさについての概念を、3・3・1項で述べた従来の評価方法とここで述べた新しい評価方法を対比して示したものである。従来の偶然誤差と系統誤差の分け方と、新しいA タイプ、Bタイプの分け方を直接関係づけることは困難であるが、不確かさ(あるいは誤差)の成分を分類してそれぞれの大きさを推定しようとすること、およびこれらを何らかの形で合成して総合的な評価を(a) 従来の精度・誤差評価:EA(error approach)(b) 不確かさ表現の概念:UA(uncertainty approach)図3・2 測定における不確かさ表現の概念(EA →UA)するという思想は一致している。したがって、不確かさの成分ごとに分類した元の形にさかのぼれば、その後の合成方法は評価する者の判断(ここに専門家としての知恵が要求される)に依存した形で整理することが可能となる。
