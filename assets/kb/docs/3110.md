# 3・3・4 不確かさの評価(解析の手順)

> **章**: 第3章 測定概論
> **節**: 3・3 測定の誤差と不確かさ

国際的な文書により新しく推奨されている不確かさの解析・評価の流れを具体的に順序立てて示すと、次のようなステップに整理することができる。(1) 準備段階:測定・校正の手順(手順を簡潔に記述する)原理、原則や測定方法、測定装置・機器などを簡潔に記述する。(2) ステップ1:数学モデルの構築(数式の形を書くか、おもな要因を挙げる)(a) 数式による不確かさ表現が可能であれば、それを記述する。(b) 数式の形で表現できない場合は、不確かさの要因を列挙し、加算形式で合成する。(c) 実験計画法に基づく実験と要因分析から有意差検定を行い、要因別の不確かさを見積もる。(3) ステップ2:補正の実施の有無(補正の項目とその方法を記述する)補正ができれば、不確かさの推定は補正後のデータに対して行う。その後に最良推定値を求める。(4) ステップ3:不確かさ成分の分析と見積り(A,B タイプの区別を含む)不確かさの成分を列挙して分類し、成分ごとの標準偏差(またはそれに相当するもの)を次のように推定する。(a) 標準のもつ不確かさ(標準不確かさで記述)(b) 標準との比較における不確かさ校正装置、校正環境、校正周期、被測定物などに起因する不確かさ(標準不確かさで記述。その求め方、根拠を示す。以下同じ)(5) ステップ4:合成標準不確かさの計算(二乗和の平方根、見掛け上はA,B タイプの区別なし)(6) ステップ5:拡張不確かさの計算(7) ステップ6:最終的な記述(表現)±さらに詳細を解析するには、図3・3における数学モデルの構築(ステップ1)を場合分けして、図3・4に示す、理論式、経験則、実験的要因分析による方法を適宜採用することになる。すなわち、図3・4 不確かさ推定のための数学(統計)モデル図3・3不確かさ解析・評価の流れ図(ステップごとの内容)5―13総合的な不確かさ(拡張不確かさ)を推定する場合には、次の三つの方法の中から一つの方法を選定することが効果的である(:目的特性、:従属特性、:要因数、:分散)。(1) 理論式からの推定=욡=1,2,...,(2) 経験則からの推定=++...+욈(3) 実験的要因分析による推定=++...+욈(ここでは、交互作用効果を考慮する)
