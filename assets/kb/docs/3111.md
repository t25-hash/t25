# 3・3・5 不確かさの表示(不確かさの総合的見積り)

> **章**: 第3章 測定概論
> **節**: 3・3 測定の誤差と不確かさ

従来は、測定や分析結果を表現する際に、誤差、精度、かたより、ばらつきなどの用語を用いていたが、分野によってその意味するところが必ずしも一致していなかったために多くの混乱を生じていた。そこで、最近になって国際的な統一を図ろうとする動きが出て、ISOより不確かさの表現に関する指針が刊行されたことは前述したとおりである。これらの考え方は、標準物質に関するISOガイド30の中にも採用されている。新しく出されたISOの国際文書では、3・3・2項で述べたように不確かさの成分を統計的に求めることのできるもの(A タイプ)と、それ以外の方法で求めるもの(B タイプ)に分けている。これら要因ごとの指標を標準偏差としており、最終的には、これら要因ごとの標準偏差(これを標準不確かさと呼ぶ)を二乗和して平方根を求めて、合成標準不確かさとしている。さらに、これに信頼率を示す包含係数を掛けて拡張不確かさとしている。手順として再整理すると次のようになる。(1) 測定値を求めるための関係式を明確にして、不確かさの要因を挙げる(関係式が得られない場合でも、考えられる要因は列挙しておく)。(2) 不確かさの要因を分類して整理する。(3) 要因ごとの成分の大きさを求める(標準不確かさ)。(4) (3)で求めた成分を合成する(合成標準不確かさ)。このとき、考慮すべきすべての不確かさ要因とその程度を不確かさの見積表(budget table)として定量的に表示しておくことが効果的である。(5) 包含係数(=2が一般的である)を決めて、拡張不確かさを求める(これを総合的な不確かさと考えることができる)。
