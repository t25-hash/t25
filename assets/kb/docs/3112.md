# 3・3・6 不確かさの性質

> **章**: 第3章 測定概論
> **節**: 3・3 測定の誤差と不確かさ

不確かさ評価の議論において、個々の不確かさを標準不確かさとして表現し、それらを総合的に合成する合成標準不確かさ、さらには、一定の信頼率のもとでの総合的な不確かさを拡張不確かさとして表現する一連の概念の根底には、従来の誤差論から出発し、不確かさの伝ぱを誤差の伝ぱと同等と考える、統計論的な分布の概念が導入されている。そこで、図3・5のような種々の分布の相互関係を知っておくことが重要である。また、不確かさ評価におけるA タイプとB タイプの分類においては、特にB タイプに関する分布として矩形(一様)分布と三角形分布を付け加えておく。A タイプの不確かさは、あらゆる統計的な手法によって求めることが可能であり、それぞれの要因による繰返しの成分として与え5―14計測工学図3・5 統計量の分布系と相互関係られる。一方、B タイプの不確かさとしては、次のような内容が考えられる。・校正証明書の内容・ハンドブック等文献データ・過去の管理データ・メーカの仕様書
