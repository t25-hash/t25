# 5・2・6 新規化学物質に関する法 日本で初めての化学物

> **章**: 第5章 放射性物質，有害物質に関する法制度
> **節**: 5・2 のようになる．現在の法体系では，法規制が最も充実している

である．すなわち「化審法」及び労働安全衛生法における「新規化

a．化審法 化審法は難分解性の性状があり，かつ，人の健康
を損なうおそれがある化学物質による環境の汚染を防止するため，
新規の化学物質の製造，輸入に際し，事前にその化学物質が難分解
性などの性状を有するかどうかを審査する制度を設け，その性状に
応じて，化学物質の製造，輸入，使用などについて規制を行うこと
を目的としている．対象は，医薬品，農薬等を除く化学物質，一般
に化学工業品などと呼ばれている化学物質を対象としている．
i．事前評価 我が国で初めて化学物質を製造又は輸入する事
業者は，製造・輸入前にその化学物質が環境経由で人の健康に与え
るリスクを評価すること（事前評価）が要求されている．また，こ
のような新規化学物質にかかわる事前評価の実施と並行して，国に
より，既に日本国内で製造または輸入されていた化学物質（既存化
学物質）について同様のリスクの評価が行われている．
ii．新規化学物質の審査 新規化学物質を製造または輸入する
際に，厚生労働省，経済産業省，環境省は事業者からの届出に基づ
き，その化学物質の下記三つの性状に着目して規制すべき化学物質
を特定するように定められている．
イ）自然的作用による化学的変化を生じにくい．（難分解性）
ロ）生物の体内に蓄積しやすい．（高蓄積性）
ハ）継続的に摂取される場合には，人の健康を損なう恐れがあ
る．（長期毒性）
審査結果に基づいて，当該新規化学物質が第一種特定化学物質，
第二種特定化学物質ジ指定化学物質，そのいずれにも該当しない物
質のいずれであるかを判定し，必要に応じて適切な規制を行うこと
を定めている．
ただし，製造量及び輸入量が全国で1t以下の化学物質（少量新
規化学物質）については，事業者からの申し出に基づいて確認を行
っている．
iii．特定化学物質等の規制 化学物質の審査の結果，その判定
により，．以下のような製造等の規制が実施されている．
（1） 第一種特定化学物質（PCBなど9物質） 難分解性，高蓄
積性及び長期毒性を有する化学物質を政令で定めると共に，製造・
輸入を実質的に禁止している．
（2） 第二種特定化学物質（トリクロロエチレンなど23物質）
高蓄積性ではないが，難分解性であると共に長期毒性を有する化
学物質を政令で定め，製造・輸入の予定及び実績数量の届出義務を
課すると共に，環境の汚染を防止するためにとるべき措置について
技術上の指針を公表し，必要に応じ指導を行うことにより環境中へ
の残留量の低減に努めることを定めている．また，環境の汚染によ
り人の健康に係る被害が生じることを防止するため，製造・輸入予
定数量の変更を命令することができる．
（3）指定化学物質（クロロホルムなど342物質） 高蓄積性では
ないが，難分解性であり長期毒性の疑いのある化学物質を告示し，
製造・輸入量の実績などの届出義務を課することで，環境汚染状況
を監視している．製造，輸入，使用等の状況からみて当該化学物質
による環境の汚染により，人の健康に係る被害が生じる恐れがある
と見込まれる場合には，製造，輸入業者に対し有害性の調査を指示
でき，その結果長期毒性を有すると判断された場合には第二種特定
化学物質に指定される．
b．労働安全衛生法による有害性調査制度 労働安全衛生法の
第57条の3，第57条の4，第58条に，労働者の職業性疾病，特に職
業がん等の発生防止を目的として有害性調査制度が規定されている．
この制度では，新規化学物質の製造叉は輸入に際し，事業者が事前
にその化学物質にがん原性の疑いがあるかどうかを調査し，その結
果と取扱方法の概要を届け出ることを義務づけている．がん原性の
疑いがあるか否かは甲州ムス試験という変異原性試験を行うことで
判断する．
届出対象物質は，化審法より範囲が広いことに留意する必要があ
る．すなわち，医薬品や農薬も含むすべての新規化学物質が対象と
なる．また，その新規化学物質を製造するための中間体や廃棄物も
届出対象となる．
