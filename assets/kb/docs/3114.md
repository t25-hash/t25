# 3・4・2 安定性

> **章**: 第3章 測定概論
> **節**: 3・4 測定の精度と校正

測定器の指示値や感度などの特性が,時間の経過または影響量の変化によって変わる程度の少なさを安定性(stability)という.a. 影響特性測定対象量以外で,測定結果に影響を与える量を影響量(influence quantity)といい,その量の変化による指示の変動の特性を影響特性という.影響量としては,室温,大気圧,振動など測定システムの外部の量と,電源電圧などの内部量とがある.影響量の効果を原因に対する結果の比率で表す量を影響係数と呼ぶ.また,測定器によっては,許容される影響量の範囲を定め,その範囲での影響の効果量を影響変動量として表すことがある.例えば,温度に関しては,温度係数:0.03 m/°C対温度変動量:3mV(0~50°C)のように表示する.b. 時間安定性時間の経過に対する安定性を時間安定性といい,単に安定性という場合はこの時間安定性を指す.安定性は一般に,測定器を使用状態にセットした状態での変動を指し,測定のたびに電源を入れるなどの再設定をするときの測定器の指示のばらつき(繰返し性,反復性)とは区別する.安定性のうち,測定器の指示が時間の経過とともに緩やかに変化する傾向をドリフト(drift)という.ドリフトは一定時間当たりの変化(変化率)で表す.また,計算機を介した自動計測がなされる場合は,ある時間にわたって連続的に出力を記録しておけば,適当な幅での移動平均を掛けるなどして,ドリフトの様子が種々の周波数域で判定できる.一つの測定対象に関する測定所要時間,与えられた全数の測定所要時間,継続的な変化を測定するときの持続時間等によって,安定性を評価する時間も意味も変わるべきものである.また,計測に要求される精度が高まるにつれて,測定対象量そのものの時間的安定性,空間的安定性等が測定器の安定性と分離できなくなる可能性が高まる.
