# 5・2・7 化学物質の安全管理に関する国際的な動き

> **章**: 第5章 放射性物質，有害物質に関する法制度
> **節**: 5・2 のようになる．現在の法体系では，法規制が最も充実している

国際的にも化学物質の安全管理の関心がますます高まり，経済開
発協力機構（OECD）や国連の各専門機関（UNEP， IPCS， ILO，
WHOなど）において多くの活動が行われている．このような背景
の中で，1992年6月に開催された国連環境開発会議（UNCED）で
決定された「アジェンダ21」は，その第19章「有害化学物質の環境
的に健全な管理」において，これまでの国際協議を総合して21世紀
に向けての行動計画を設定した．
その計画では，
a．各国の協力で高生産量物質（HPV）の安全性試験を行い，リ
スク削減対策を講ずること．
b．化学物質の人および環境への影響と各国の規制等に関する情
報を収集・蓄積するための「国際有害化学物質登録制度
（IRPTC）」を進めること。
c．規制化学物質の貿易時の情報交換手続きを定めた「改正ロンド
ンガイドライン」で輸出国への事前通達・承認通報・承認制度
（PIC）を推進すること．
d．世界保健機関（WHO）および国際労働機関（ILO）と共同で
「国際化学物質安全性計画（IPCS）」を実施し，優先度の高
い化学物質のリスク評価等を行った結果を「環境保健クライテ
リア」として刊行すること．
OECDでは，従来から国際的に整合性のある化学物質管理手法を
確立するため，「テストガイドラインの開発・整備」や「評価情報の
相互受け入れ」制度を進めている．
さらに，OECDでは，1996年2月号各国政府のための「環境汚染
物質排出・移動登録（PoUutant Release and Transfbr Registers，
PRTR）」のガイダンスマニュアルをとりまとめた． OECDの定義
によれば，PRTRとはさまざまな排出源から排出または移動される
潜在的に有害な汚染物質め目録もしくは登録簿である．この制度で
は，多くの対象化学物質について工場・事業場から環境への排出量
や廃棄物等として移動量を企業等が自ら把握し，その結果を行政な
どに報告し，行政などはそれを何らかの形で公表するという手順か
ら構成されている．
国際標準化機構（ISO）では，環境マネジメントシステムと環境
監査の国際規格ISO 14000シリーズを定め，1996年に発行させた．
環境マネジメントシステムは企業の環境保全活動が適切に行われて
いるか否かを管理するための体制，計画，活動，責任，実務，手続
き，過程および資源を含むシステムである．環境監査は，体系的な
環境管理がなされていることを第三者から認証してもらう制度であ
る．日本の企業でもこの環境管理・監査企画の認証を受け始めてい
る．
