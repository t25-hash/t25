# 3・4・3 直線性とヒステリシス

> **章**: 第3章 測定概論
> **節**: 3・4 測定の精度と校正

測定器の入力(測定対象量)と出力(指示値)の関係は、もし表示に誤差がなければ=である。一般に、誤差(かたより)は量の大きさによって異なる値になるので、実際の測定器に関しては、=+εと表せる。ここで、εはばらつきの成分である。決められたの範囲内でが正しく求められれば、その範囲内で誤差のない測定ができる。は校正曲線といい、普通は一次式(直線)で表される。そして、この直線からのずれの小さい程度を直線性、または線形性(linearity)といい、直線からのずれを線形誤差と呼ぶ(図3・6)。測定器によっては、測定値がそれまでの入力の変化の履歴によって影響を受け、同一測定量に対して異なる指示値を示すことがある。バックラッシや摩擦の影響を受けるダイヤルゲージ(戻り誤差)、圧電素子による追従型の測定をする走査型顕微鏡などにその典型が見られるこの現象をヒステリシス(hysteresis)という。図3・7のように測定量(入力)を増加させながら求めた,の関係と、減少させながら求めた,の関係の差、ε욠=욃-욀(3・1)をヒステリシス差(hysteresis error)という。また、ε욠=욃-욀(3・2)を入力換算ヒステリシス差という。なお、ヒステリシス曲線は途中で入力の増減の方向を変えると、,の関係はその転換点を起点とした別の軌跡をたどることになり、図3・7のループ内のすべての点をとる可能性がある。入力の増減方向の変化があらかじめ決まっていない場合には、このヒステリシスによる線形誤差を補償することは極めて難しい問題になっている。そこで例えば、ねじ駆動で移動体を目標に近づける際に、つねに同じ回転方向から近づけるような走査を義務付けることで、バックラッシや摩擦の影響を回避する策がとられる。
