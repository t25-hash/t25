# 6・1・1 現行の労働安全衛生管理

> **章**: 第5章 放射性物質，有害物質に関する法制度

さらに，これらの法規類に対しては，制定後も重大な労働災害の
発生するたびに，法改正または詳細な規制基準などの追加・改正が
なされてきている．
本来，制定された法令は，自主的に遵守されることが法の理想と
するところであるが，現実には遵守されない場合がある．そのとき
は，法的な制度として刑罰などの制裁が課されるか，行政機関によ
って営業停止命令，使用禁止命令，改善命令などの行政処分が発せ
られる．すなわち，労働安全衛生に係わる現在の法システムは，伝
統的な命令・監督型の規制手法を中心としているといえる．
b．労働安全衛生法に基づく労働安全衛生管理 我が国におけ
る現行の労働安全衛生管理においては，a項に記したように法シス
テムを遵守することが基本となっている．ここでは，労働安全衛生
法（昭和47年6月8日 法律第7号，以下「安衛法」という）に基
づき，労働安全衛生管理がどのように行われているかを示す．
i．安衛法の目的 安衛法の目的は，その第1条に次のように
規定されている．
（1）職場における労働者の安全と健康を確保すること
（2） 快適な作業環境の形成を促進すること
そして，この目的を達成するために，国，労働災害防止団体およ
び事業者等には，その責務として，労働災害防止に関する総合的か
つ計画的な対策を推進することが求められている．具体的な実施事
項は次の3点である．
（i） 労働災害防止のための危害防止基準を確立すること
伍）労働災害防止のための責任体制を明確化すること
（皿） 労働災害防止のための自主的活動を促進する措置を講ず
ること
