# 3・4・4 校正とその目的

> **章**: 第3章 測定概論
> **節**: 3・4 測定の精度と校正

JIS Z 8103「計測用語」によれば,校正(calibration)とは,「標準器・標準試料などを用いて計測器の表す値とその真の値との関係を求めること」とある.したがって,測定器,センサが作られたときの目盛付け,値付け,あるいはすでに目盛や値がつけられているものに対しての補正データを求めることが含まれる.また測定器,センサをISOの基本単位へのトレーサビリティ体系に組込むことも意味する.これは,アナログセンサでいえば平均感度(校正直線の傾斜)を,ディジタル計測器でいえば平均目盛間隔を国際標準単位に関連付けて決めることになる.国家計量標準研究所,計測器メーカ,ユーザ等の立場の違いによって校正の目的も異なる.国家計量標準研究所では,トレーサビリティの維持,国内外の測定基準の管理,レベル維持が主要な校正目的になり,計測器メーカ等では,製品の品質保証,精度の明示・誇示が目的になる.これに対して,ユーザではメーカ等から与えられた校正データの必要に応じた確認のほかに,校正データを使って測定結果の詳細な補正を行うためにも校正が行われる.計測を行ううえで,誤差が小さいほどよいが,当然,技術的,経済的な限界がある.したがって,校正に関しても,目的,必要精度,経済効果を勘案してその精度,データの密度等が決められる.a. 校正の方法と校正の種類校正には定められた基準点を選んで行う基準点校正がある.この基準点が測定器の零点のとき,これを零点校正という.零点校正にはマイクロメータのように零を定める付属の基準ゲージを使う場合と,被測定量として何も与えない状態を使う場合がある.b. 校正直線と平均感度校正範囲の両端の点(図3・6中のOとM)を結ぶ線の傾きが,その範囲での平均感度(mean sensitiv-ity)になる.測定範囲内のいくつかの点についての校正データを使って,最小二乗法で直線を当てはめて校正直線を作ることもできる.これらの直線の傾きが,その測定器を用いた計測結果の尺度図3・7 ヒステリシス曲線図3・9校正結果5―15(スケール)を与えることになる.c. 校正曲腺と線形誤差校正データは,一次曲線(直線)での表現が多用されるが,計算機を組込んだ計測システムが一般になりつつある現在,高次曲線,折れ線などの校正曲線(calibrationcurve)による表現で校正データ(calibration data)を使うことはそれほど難しくない.校正データから校正直線を差引いたものを線形誤差(linearity error)というが,線形誤差を詳細に求めて測定値の補正に用いると,測定器の分解能や安定性の限界近くまで,測定値の確からしさを高めることができる.
