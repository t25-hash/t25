# 3・4・5 幾何学量の計測と各種の校正法

> **章**: 第3章 測定概論
> **節**: 3・4 測定の精度と校正

精密な計測においては,測定器の分解能や安定性の限界近くまでの精度が要求され,詳細な校正曲線による線形誤差の補正が必要になる.特に,真円度,真直度,段差などの測定では,多くの場合センサ入力の変化範囲に比べて目的の測定量の変化が極めて小さく,計測器の線形誤差の影響が強くなる.詳細な校正データが必要な場合には,同一の測定対象量を,基準計測器と被校正計測器の両方で同時に測定して比較校正を行う.a. 比較校正図3・8は,変位センサの信号処理後の電圧出力と入力変位の関係を校正するときの様子を示す.校正される変位センサと,それより信頼性の高い基準変位計を一直線上に並べ,変位センサの入力となる目標物の変位を基準変位計で測定して被校正センサの入力とし,そのときの電圧出力との関係を図3・9のようなグラフにプロットする.ナノメートルの領域までの精度を要求される変位センサの校正に用いられる最高精度の比較基準として,X 線干渉システムがある.これは図3・10のように2枚の結晶格子ブレードで回折させたX
