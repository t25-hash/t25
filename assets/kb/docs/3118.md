# 3・5・1 補償の考え方

> **章**: 第3章 測定概論
> **節**: 3・5 補償法

計測システム、計測器は広義の変換器であり、出力が計測の目的量である入力変数と1対1の関係にあり、出力から入力が正しく定まることが理想である。しかし、一般には計測シス5―18計測工学テムにおける出力には目的以外の変数が影響し、また、目的変数との関係にも未知のヒステリシスや非線形成分が含まれ、計測の誤差をもたらすことになる。この誤差の要因を取り除き、入出力関係を理想状態に近づけるための工夫が、広い意味での計測システムの補償(compensation)である。計測システムのヒステリシスや非線形性の影響は主として3・4節の校正の結果を用いて補償される。3・2・3項の測定法の分類で偏位法と零位法の中間に位置する補償法も、さらにフィルタの利用も、この工夫の特別な場合と考えてよい。また、計算機を援用する計測システムでは、複雑な計算によってこの目的を達成することも可能になっている。ここでは、フィルタや計算機によるデータ処理の項では取り上げられない、比較的単純な工夫に限って述べる。補償法の構造は、目的変数と外乱変数の関係(モデルの関数関係)によって異なる。例えば、両者の影響が加算的な場合は差動構造の補償法がとられ、乗算的な場合は除算構造の補償法がとられる。そのほか、ランダムな外乱の影響を低減する加算的な構造もある。
