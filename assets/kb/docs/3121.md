# 5・2・2 組立性の評価

> **章**: 第5章 生産設備
> **節**: 5・2 組立装置

人間が組立てることが困難な半導体，回路基板等の微細部品の組
立を除くと，組立作業の多くは人手による組立と自動組立との間の
経済的効果により，その形態が決定される．このような状況で，自
動組立により経済的効果を出すには，安価な組立装置と，そのよう
な組立装置で作業が可能な組立性の良い製品設計が必須技術であ
る．
一般に，機械にとって容易な組立とは，組付けられる部品を上か
ら下に，重力の方向に下ろし，軸や穴などに装着することである．
機械にとってはこの動作がいちばん安価で，確実性のあるものであ
る．横方向や斜め方向の装着は，重力の影響が運動の精度を落とす
要因となり，また機械の運動機構も複雑なものとなってくる．そこ
で，部品の設計で，その組立の大部分を上から下方向への動作によ
り組付けが行えるかが，その自動化が効果の出るものかどうかの判
定基準となる．そこから，設計された機械部品の組立動作を分析
し，それが組立てやすいかどうかを評価する組立性評価法が開発さ
れ，これによる評価を設計にフィードバックすることにより，組立
が容易な部品の開発につながっている．
が容易であり，組立性評価は機械部品設計にとって非常に重要な項
目となっている．
