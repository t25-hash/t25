･多数の中から一つずつ自動的に姿勢をそろえて送り出すフィーダ

･マガジン内に複数の部品をマトリックス状に並べ，位置決めし，
供給するマガジン方式
･加工機から直接一つずつ送り出す移送方式
･テープを用い，部品をそれに貼り付けて供給するテープ方式
などがある．
フィーダ方式は，おもに小物部品で，形状が単純なものに用いら
れる．供給のための部品送りには振動を用いることが多い．また部
品の重心位置や，上下，左右部での形状の違い等を利用し，姿勢を
そろえ，整列して供給する手法がとられる．小さなねじや，形状の
単純なプラスチック部品等の供給に用いられている．姿勢をそろえ
て供給するには，部品に応じて専用の姿勢整列部を設ける必要があ
るため，特定の部品向けの専用装置となる場合が多い．
マガジン方式は，一つのマガジンの中に複数の部品を精度よく位
置決めするための細工をし，部品をマトリックス状に並べるもので
ある．一般に，部品の加工現場において，加工後の部品をマガジン
内に整列させ，収納し，マガジンごと組立ステーションに供給する
ものである．半導体チップの小さなものから，比較的大きな機械加
工部品やモルード部品に用いられる．運搬にかさばる，組立ステー
ションに広い面積を必要とする等の欠点があるが，多くの部品に同
じ手法で対応できるという利点がある．
加工機から直接送り出す移載装置の例としては，例えば小型モー
タのシャフトを加工後，そのままシュータや直線的間欠送りにより
組立ステーションに送るものがある．加工現場と組立現場が同居で
きる小物の製品の組立に適している．また，材料の打抜きにより作
られる小物部品を，組立ステーションで打抜きながら組立を行うよ
うな，加工・組立の一体型のシステムも存在する．
テープを用いて部品を供給する手法は，おもに抵抗やコンデンサ
チップ等の電気，電子部品に用いられている．チップ部品を粘着テ
ープ上に付け，カバーを掛けたまま（部品メーカから納入された状
態で）組付装置に装着され，組付けられる．また同軸の抵抗は，軸
の両端をテープで固定した状態で納入され，組付装置に装着され
る．このようなテープを用いた部品供給では，その多くが規格化され
ており，多くのメーカから同じ仕様のものを購入することができる．
組立装置において，部品供給部は部品の形状種類の影響を最も受
けやすい部分であり，適切な部品供給方式が組立装置全体の簡素化
につながり，またその逆の場合も起こりうる．したがって，部品供
給の導入には慎重な検討が必要である．
c .組付装置 組付装置は部品供給装置から組付けるための部
品を取上げ，被組付部品に組付ける．組付動作が上から下への単純
動作であるとしたときの，組付装置に要求される代表的な運動は以
下のものとなる．
･円筒運動：垂直並進運動と垂直軸回りの回転（2自由度）
･2並進運動：垂直並進運動と水平1軸の並進運動（2自由度）
･円筒運動1回転運動：円筒運動と垂直軸回りの回転（3自由度）
･2並進運動1回転運動：上記2並進運動と垂直軸回りの回転（3
自由度）
･3並進運動：垂直並進運動と水平2軸の並進運動（3自由度）
･3並進運動1回転運動：水平平面運動と垂直並進運動（4自由度）
･3並進運動3回転運動：空間6自由度運動（6自由度）
2自由度の組付装置は，部品供給装置でつねに部品が定位置に供
給され，組付装置がこれを取上げ，被組付部品の所定の位置に組付
けるものである．円筒運動は一つの回り対偶と，その軸に平行な進
み対偶からなる装置であり，2並進運動は直交する軸に沿っての二
つの進み対偶からなるものである．2自由度ということは，基本的
には部品の位置を変えるだけの機能をもつものと考えてよく，部品
の垂直軸回りの姿勢を変化させるには自由度不足である．なお2並
進運動は姿勢を一定に保つという性質を有する．
円筒運動1回転運動と2並進運動1回転運動機構は，2点間の部
品の搬送を行うとともに，部品の垂直軸回りの姿勢の位置決めを行
うものであり，2自由度機構に姿勢の制御機能を付加したものであ
る．ただし，回転軸は部品の把持中心を通る垂直軸でなければなら
ない．
3並進運動機構は互いに軸が直交する三つの進み対偶からなる機
構である3自由度にすることにより，2自由度に比べ，平面内の位
置を自由に選べることになる．すなわちマガジン供給のように，組
付部品の位置が複数箇所である場合に対応することができる．部品
の姿勢の調整機能はないが，姿勢は一定に保たれるという性質を有
する．なお同様の機能を，二つの同軸でない垂直軸回りの回転と垂
直軸に沿っての並進運動で実現できるが，このときには部品の姿勢
の保持機能はない．
3並進運動1回転運動は，平面内の任意の位置にある部品を，別
の平面内の任意の位置に搬送し，かつ姿勢を調整する機能を有す
る．すなわち，垂直の並進運動で部品を組付ける作業に対しては十
分な自由度を有していることになる．この機構は
(１)三つの直交軸に沿っての並進運動と一つの垂直軸回りの回転
(２)二つの直交軸（垂直軸を含む）に沿っての並進運動と二つの
同軸でない垂直軸回りの回転
(３)垂直軸に沿っての並進運動と三つの同軸でない垂直軸回りの
回転
の形で構成することができる．(１)を直交座標型，(３)をスカラロ
ボット（SCARA robot ）という．このタイプではスカラ型が広く
用いられている．その形状を図5･12 に示す．

d.ツール類 組立には，
(１)部品を保持するための，部品形状に応じた保持装置
(２)ねじ締め固定のためのねじ締めツール
(３)軸を穴に挿入するためのはめあいツール
(４)工具交換装置（ATC:automatic tool changer )
などのツールが用いられる．
部品を保持し，もち上げ，搬送し，下降して組付けるのが組立装
置の基本動作であり，保持装置は不可欠な装置である．部品を保持
する方法としては，吸着，把持がある．吸着では真空吸着が主であ
り，まれに磁力による吸着が用いられる．吸着の場合は，保持した
ときの部品の位置精度が問題となる場合があり，このようなときは
別途位置保持をするための手段を考慮する必要がある．把持は複数
の指で挟み込む方法が主である．加工された部分を把持することに
より，把持精度を保つことができる．把持装置は空圧アクチュエー
タや電磁ソレノイドで駆動される場合が多い．これらの保持装置に
は，部品を確実に保持したかどうかを確認するためのセンサを取付
けるのが普通であり，これにより組込みミスをチェックする．
自動ねじ締め装置はよく用いられる機器であり，多くの装置が市
販されている．ねじを部品供給装置で装置本体に供給し，装置は下
降運動とねじの回転運動によりねじ締め，固定を行う．複数のねじ
を締めるときは，ねじ締めヘッドを上下機構に複数個取付け，同時
に行うものも多い．装置の動作の信頼性を決める要因として，ねじ
の長さがある．ねじの長さがねじヘッドの大きさと同等，あるいは
それより短い場合，重心位置の関係からねじの供給時に姿勢がうま
く出ない場合が生じ，作業の信頼性を落とす．生産性を保つために
は，ヘッドの径よりも少し長めのねじを用いる必要があり，これは
設計時に考慮しておくべき問題である．
回転部分を有する部品の組立にははめあい作業が伴う．かつて
は，はめあいの自動化は困難な作業とされていたが，現在では径の
小さい軸の穴への挿入，例えばビデオやカセットテープのレコーダ
の駆動部の組立では，軸に適切な面取りを行うことにより，簡単な
垂直運動で挿入を行うことができるようになっている．これに対
し，径が大きくなると，単に組付装置の運動だけでは困難となり，
はめあい用のツールを用いて挿入を行う．ここで用いられるツール
は図5･13に示すRCC（remote center compliance）と呼ばれるコ
ンプライアンス機構であり，コンプライアンスによる瞬間運動が，
軸を把持したときのその軸の先端で交わる3方向の回転運動となる
ようにしたものである．これにより，挿入するべき軸の姿勢が，穴
に対して誤差があった場合でもRCCが自動的に姿勢を修正し，は
めあいを可能とする．このツールは1970年代後半，アメリカのド
レーパー研究所で考案されたものであり，現在では国内でも製品化

ATCはAHC（automatic hand changer ）ともいわれ，組付装
置が複数の部品の組付作業を行うとき，部品に応じて保持機構を交
換するためのものである．保持機構を駆動するためには電力や高圧
空気の供給が必要であり，また部品を保持したかどうかの信号を保
持機構から，組付装置に送る必要がある場合が多い．そこで，
ATCでは，保持機構を付けたとき，電力，電気信号，高圧空気が
接続できるようになっている．複数個の保持機構を組付装置の近辺
にセットしておき，組付装置は保持機構を取り替えながら複数の作
業を行うというように用いられる．
e.組立装置の課題 近年，一つの製品の寿命は非常に短く，
他社に先駆けての新製品の短期での開発，発売が重要となってい
る．そのため，生産ラインでは，新製品の生産を素早く立ち上げる
ことができるように，新しい製品に合わせたラインを早急に立ち上
げるためのシステムの再構築性（リコンフィギュラブル）と，多品
種の生産に対応できるフレキシビリティが要求され，また技術の革
新を即座に取り入れられる機能も要求されている．21世紀の生産
システムといわれているバーチャルエンタプライズによるアジャイ
ルマニュファクチャリングでは，情報ネットワークを活用し，新製
品開発とコンカレントに生産計画を行い，製造に入る必要がある．
このためコンピュータを介した設計と製造の情報の共有が不可欠で
あり，またコンピュータ支援による生産計画，生産管理が必要であ
る．
このように，アジャイルマニュファクチャリングにおける組立装
置には，多品種生産に対応できるフレキシビリティ，新製品開発サ
イクルの短周期化に伴う頻繁な生産工程変更を可能とする再構築
性，新たな技術を即座に取り入れることができる移植性，他の自動
機や作業者とデータをやりとりし，共同で作業を実行する協調性が
要求される．
しかし，現在はいまだ端境期であり，アジャイルマニュファクチ
ャリングとしてどのようなシステム構成が良いのかということにつ
いての道筋は見えていない．そのため，当面は生産工程を人件費の
安い所に移し，価格競争で生き残りをかけているというのが実情で
ある．これからの生産システムはどのようにあるべきかの明確な概
念をうち立て，それに従ってシステムの概念設計を行い，製造シス
テム，製造装置がどうあるべきかを見据え，それに向かっての技術
開発が必要である．組立装置についてもそのような新しい概念のも
とでの機能の見直しが必須となると考えられる．
