# 4・1・1 静的データとは

> **章**: 第4章 測定データの処理
> **節**: 4・1 静的データの処理

一般に、測定データは、時間(ないし、それと等価な速度、距離などの変量)とともに変化する量を連続的に計測することによって得られる動的データ(dynamic data)と、測定点を決めて対象とする測定変量を個別(離散的)に計測して得られる静的データ(static data)に分けられる。静的データの構成要素(最小単位)は、単一の数値または複数個の数値の組であり、前者から成る測定データを一次元(1変量)データ、後者を二次元(2変量)または多次元(多変量)データと呼ぶ。一次元データには、長さ、質量、時間、密度、濃度、温度、電流、電圧、硬さ、表面粗さ、強度などの計量値(連続量)のほか、故障率、頻度、個数、順序などの計数値(離散量)があり、多次元データでは、それら任意の変量間の関係を問題にする。データ処理とは、測定対象に内在する問題を顕在化するために、データ全体のもつ特性をいくつかの数値(要約値)で表したり、データ全体を視覚的に表現する行為であり、前者はデータを要約すること、後者はデータ全体を見やすい形に近似またはグラフ化することである。通常、計測データは、測定対象の全体ではなく、その母集団から抜き取られたサンプルである場合がほとんどであり、得られたデータそのものの性質だけでなく、元の母集団の性質を知ることも重要である。そのためには、得られたデータを利用して、母集団を推定したり、母集団のもつ諸性質に対して立てられた仮説の正しさを検定するための解析も必要であり、これらも併せてデータ処理と呼ぶ。
