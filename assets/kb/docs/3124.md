# 5・3・2 真円度測定機

> **章**: 第5章 生産設備
> **節**: 5・3 検査設備

エンジンのシリンダブロック，ピストン，モータ軸，軸受などの
円筒，球などの工作物について，真円度，円筒度，同軸度，真直度
などの形状精度を評価するのに使用される．回転機構をもち，接触
式の変位計測器により測定対象の円周方向の半径の変化，円周方向
および軸方向の半径の変化を測定する．測定装置の構造は，図5･
21に示すように，測定用のスタイラスが回転する形式と，測定対
象の工作物を搭載したテーブルが回転する形式とに分かれる．
回転部分の軸受は高い回転精度が要求されるので，静圧空気軸受
が用いられ，回転精度は一般には0.05m，高精度のもので，0.02
mに達する．また，スタイラスの回転を止め， 方向（高さ方
向）にスタイラスを送ることにより，工作物の真直度を測定でき
る．測定に先立ち，測定対象の中心とテーブルの回転中心を一致さ
せるため，あるいは，測定対象の工作物の中心軸とテーブル上面と
の直角度を調整する作業が必要であるので，自動的に心出し・傾斜
調整を行う機構とソフトウェアを装備するものが多い．
