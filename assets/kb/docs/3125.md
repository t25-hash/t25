# 5・3・3 インプロセス測定機器

> **章**: 第5章 生産設備
> **節**: 5・3 検査設備

自動生産のためのオンライン計測機器，センサのもつべき機能に
はさまざまなものが要求されるが，部品の寸法精度の計測，切削
力，振動，工具損傷などの検出が重要な項目となる．インプロセス
で使用する測定機器は種類が非常に多い．導入に当たっては，セン
サ，コントローラ，計測ソフトウェア等をシステム化し，ハードウ
ェアの選定から，ソフトウェアの構成まで，測定作業ニーズに併せ
てメーカが関与する場合が多い．
a.自動定寸装置 部品加工において内径，外径などを測定す
る必要性は，工作機械の稼働率を向上するために必要であり，自動
定寸装置の開発が行われてきた．各種測定装置により加工機上にて
寸法測定を行うことは，インプロセス，ポストプロセス測定を問わ
ず，工作物を工作機械より取り外すことなく測定ができる点で，精
度上，能率上のメリットが大きい．自動測定が求められる理由とし
ては，
(１)加工後に工作機械を停止し，通常の測定器具により測定す
ることは非常に時間のかかる作業であり，工作機械の稼働率を低下
させる．
(２)熱変形の影響や工具の摩耗により，工具位置の補正を行わ
ないと指定の寸法に仕上げることができない場合．
などがある．
図5･22に示す定寸装置 は，インプロセスで工作物の寸法を測
定し，砥石の切込み速度を変化させたり，加工機を制御する機能を
有するものである．検出器を砥石と対向させたテーブル上に取付
け，接触子を工作物に2点で接触させ，工作物の外径変化を差動ト
ランスの出力として求めるものである．定寸装置を用いた研削サイ
クル制御例を図5･23に示す．工作物の直径の変化を研削中に連続
的に測定し，所定の寸法になったところで砥石の切込み速度切換え
指令，切込み停止指令，砥石後退指令を出力する．また，定寸装置
の測定部位への接近の方法，測定箇所などの例を表5･2に示す．
一方，旋盤用などのインプロセスゲージは切りくずの排出が障害
となり，加工中におけるインプロセス測定が困難であり加工直後に
測定して，工具補正信号を出す検出器が使用されている ．検出
器は基本的には研削盤用のものと同様ではあるが，測定寸法に合わ
せて専用に設計されたガイドを基準として，内径寸法測定を電気マ
イクロメータを用いて行うガイド付内径測定ボアゲージ等がある．
b.工具計測器 工具に関する測定項目としては，長さ，径，
回転時の振れ，損傷，熱変形が考えられる．工具に関する計測は，
折損検知だけでは十分ではなくなってきている．工具プリセッタな
どの機外計測機器が進歩しており，チップに書き込まれたデータを
加工機械に転送することが可能である．しかし，機械ごとに差があ
るために，工具を機外で測定するだけでは十分ではない．工具が回
転しているときには，熱変形によるスピンドルの伸び，工具の振
れ，ATCでの工具の引込み量が変化するなど，静止状態では現れ
ない現象がある．したがって，切削時と同じ回転状態での工具の測
定が最も望ましい．また，工具の摩耗，欠損の検出も行う必要があ
る．
図5･24にレーザ方式の工具径・長さ測定装置の例を示す．工具
の刃先管理，スピンドルの熱変位，回転による位置変化，工具摩耗
やチッピングを検知し，予備工具に変える管理などに適用できる．
c.AEセンサ（アコースティックエミッションセンサ：acous-
tic emission sensor ) 加工中に工具に損傷が発生するとAE信
号が出力されるので，これを測定することにより，工具の欠損，摩
耗などを検知することができる．また，ベアリングの回転の異常検
出，研削砥石と工作物との接触時点の検出砥石のツルーイング用セ
ンサなど，加工におけるモニタリング用センサとして重要な位置を
占めている．
図5･25は，AEセンサを組込んだ研削スピンドルの例である．
AEセンサは砥石の近くに取付けられ，信号は高周波導管で接続さ
れたトランスミッタより，空げきを介してレシーバに伝達される．
加工時や砥石のドレッシング時の信号などを高精度に受信できる．
また，エアカット時の信号により，ベアリングのノイズを測定し
て，スピンドルの異常診断に利用できる．
