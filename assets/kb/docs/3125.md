# 4・1・1 項で述べたように,測定値には連続量と離散量があるが,

> **章**: 第4章 測定データの処理
> **節**: 4・1 静的データの処理

いずれの場合でも,測定環境や測定時の演算処理等によって,意味のない下のけたまで数値が求められてしまうことがある.そのようなデータは,意味がないばかりでなく,取扱いに不便であるため,結果を丸めて示すのがよい.数値の丸め方には,切上げ,切捨て,四捨五入などの方法があるが,安全や機能上の要求がない限り,電子計算機等での処理を含めて,四捨五入が一般的であろう.JIS Z 8401:1999「数値の丸め方」では,このような場合を含めて,ある一定の丸めの幅(間隔)をもつ数列を作り,丸める前の数値を,数列内の二つの隣り合ういずれかの数値に置換える作業と定義している.丸めの幅は,10요(0.1,1,10など)だけでなく,例えば,0.5や0.2などでもよい.さらに,通常の四捨五入では,ちょうど中央の5がつねに大きいほうの数値に置換えられるため,丸めによる偏りをなくしたい場合には,隣り合う二つの数値のうち,丸めの幅の偶数倍となるほうの数値に置換える方法が用いられる.例えば,丸めの幅が1であれば,数値12.5は13ではなく12に丸められ,13.5では14となる.ただし,これはあくまでも等しく近いことが条件であり,操作は1段階で行わなければならない.すなわち,12.51はより近い13に丸めるべきであり,13.49を2段階の操作で14とすることも誤差を大きくする原因となるので避けなければいけない.
