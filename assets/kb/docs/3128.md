a.仕分けコンベヤ 生産システムにおける物流は材料から製
スタッカクレーン方式は，荷を棚に搬入・排出する二次元移動を
品まで物理的に一つの流れで実現されることはまずない．したがっ
て，物の流れに分岐・合流を行う仕分け作業が発生する．小型部品
の仕分けや低頻度物流の仕分けでは，人手を主体とした流動ラック
や回転ラックによる小規模な仕分け作業が行われるが，大物部品，
高頻度の仕分け作業では比較的大規模な仕分け作業が自動化機器を
用いられて行われることが多い．仕分け作業では，行き先情報に応
じて所定の方向に荷を分岐させる仕分けコンベヤが用いられる．
機械的な仕分け方法によって押出し式，送り出し式（図5･37)，
斜行式，傾倒式などがある．

c .包装・梱包機器 製品の搬送・保管に当たり，品質保持の
ために適切な材料，容器等で保護を行うことを目的とする機器で，
包装機と梱包機がある．
d.データキャリヤ（data carrier ) 搬送・保管作業において
は荷の流れを管理・制御するために個々の荷に固有の情報を付与
し，それを生産システムの随所で読み取ることによって荷の保管状
況を管理し，物流制御に用いることは不可欠となっている．そのた
め，情報を認識するための媒体およびその読み書きを行う設備が必
要となる．バーコードは最もはん用的に用いられており，一次元に
配置されたバーの組合せによって情報を付与し，それを読み取るこ
とによって物の流れを管理するものであるが，取扱える情報量に制
約があるため，最近では二次元上にバーを配置した二次元コードが
増え始めている．ISOなどでは複数のコード体系が規定されてお
り，これらのコードはその特徴に応じて使い分けられている．図
5･39にその一例を示す．これらのコードは，プリンタやレーザ加
工などによって情報が物に同期して搬送される認識媒体に書き込ま
れるか物に直接印字され，光学的な認識装置によって読み取られ
る．さらに認識媒体の情報を適宜書き換え，双方向で情報伝達を行
えるRFID（radio frequency identification）なども用いられる．
磁界，電波，光などによって情報を認識媒体に伝達し，その情報を
認識媒体内で書換え可能かつ大容量の情報として保管することがで
きる．荷の管理を自立的に行うことが可能となるので非常に柔軟な
生産システムを構築できるが，認識媒体自体のコストが現時点では
まだ高価である．
