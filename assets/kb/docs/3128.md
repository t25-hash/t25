# 4・1・3 項に述べた要約値は、データ

> **章**: 第4章 測定データの処理
> **節**: 4・1 静的データの処理

のもつ性質を定量的に表すものであるが、全体の傾向を把握するためには、データの度数分布を表示するのがよい。以下では、その代表例であるヒストグラムおよび累積度数図について説明する。. ヒストグラム図4・1に示すように、データの範囲全体を、幅の区間に区切り、それぞれの区間に入るデータの度数を、長方形の柱の高さとして図示したものをヒストグラム(histo-gram)という。JIS Z 9401-1:1999では、区間の幅は、測定値の範囲から、区間の数が5から20程度になるように決め、級の境界を最小測定単位の2図4・1 ボルトの引張破断荷重の分布分の1とすることとしている。一方、データ数または区間の数が少なすぎれば、分布の形を正しく表すことはできない。そこで、データ数と区間の数の関係を決める目安として、式(4・7)で示すスタージェスの公式または式(4・8)を利用することがある。通常、各区間の上側境界値に対応する累積相対度数(cumulative relative frequency)をプロットし、それらを直線で結んで表示する。b. データの近似(分布関数による表現)度数分布などで表されるデータ群を単純な(きれいな)関数形で近似し、その数学的な性質を利用したい場合には、確率プロット(probability plot)を利用する。なお、一般にデータは、ある母集団から抜き取られたサンプルである場合が多く、その場合には、母集団の分布形を推定することが重要であるが(4・1・7・a項参照)、両者の違いは、単に母集団(サンプルを無限に集めたときの度数分布の極限値)の存在を想定するか否かという点だけであり、当てはめの行為自体には特別な違いはない。データ群に当てはめた直線は正規分布0.182,0.011であり、測定データをこの正規分布でよく近似できることがわかる。. ワイブル分布の当てはめ図4・3は、ねじ転造丸ダイスの寿命(破損までの生産数量)のヒストグラムであるが、右にゆがんでいることから、=log などの変数変換を施さなければ、正規分布に当てはまりそうにない。しかし、二つのパラメータ(およびη)によって分布の形状を変化させることができる次式のワイブル分布は、このようなデータに対しても適用できる。プロットが直線とならない場合には、=+γの変数変換(分布の平行移動)を行うこともある。5―27図4・5 成形後の円周振れと動不釣合い量の散布図
