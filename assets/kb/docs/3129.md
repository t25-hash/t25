# 4・1・5 二次元データの要約

> **章**: 第4章 測定データの処理
> **節**: 4・1 静的データの処理

二次元データを二つの一次元データと考えたならば,4・1・3項で示したように,二つの平均値,標準偏差,ヒストグラムなどによってデータを要約・記述するだけでよいが,二次元データでは,二つの変量間の関係という情報が重要である.ここではその中で,2変量間の直線関係に対する当てはまりの良さと,直線関係の強さを表す要約値について述べる.a. 直線関係の当てはまりの良さを表す要約値(相関係数)二つの変量,の組のデータ,,...,욡,욡,...,욈,욈を,座標にプロットしたとき,個のプロットがだ円状の広がりをもったとする.直観的には,このときのだ円のふくらみの程度が,2変量間の直線関係の当てはまりの良さを表していると考えられる.次式のように,だ円のふくらみ具合を,変量の単位に依存しない指標として定量化したものが相関係数(correlationcoefficient)である.욈욡-욡-=∑욈욡-(4・12)욈욡-∑∑相関係数は-1から1の間の値をとる.相関係数が負の場合は,傾きが負の直線の当てはまり度合いを,相関係数が正の場合は,傾きが正の直線の当てはまり度合いを表す.相関係数が零のときを,二つの変数は無相関であるという.変数との相関係数が大きいからといって,必ずしもとに因果関係(原因と結果の関係)があるとは限らない.とが共通の原因をもつとき,とには相関が生じ,これを擬似相関と呼ぶ.b. 直線関係の強さを表す要約値(回帰係数)変量との関係を,直線関係=+(4・13)で近似したとき,式(4・13)のを回帰係数(regression coeffi-cient)と呼び,2変量の直線関係の強さを表す要約値となる.このとき,式(4・13)の右辺の変数を説明変数,独立変数などと呼び,左辺のを目的変数,従属変数などと呼ぶ.直線を当てはめる方法(回帰直線の求め方)については,4・1・6・c項で述べる.
