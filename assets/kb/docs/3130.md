# 5・5・2 画像処理装置

> **章**: 第5章 生産設備
> **節**: 5・5 制御装置

視覚認識技術は宇宙，気象，医用から産業，家庭に至る幅広い分
野で応用展開が進んできている．視覚からの入力情報が人の五感全
体で得る情報量の80 %強を占めるといわれており，人間の判断基
準に重要な役割をもっていると考えられる．
a.画像処理技術の歴史 1950年代末にコンピュータ技術を応
用した文字認識の研究を始点として画像処理技術が本格展開を始め
た．1960年代には，三次元の認識を目指したコンピュータビジョ
ンの研究が盛んになり，当初は平面で構成された簡単な積み木の世
界を対象に，人工知能の一つの分野として機械の目の研究が行われ
た．この時期に画像の特徴量抽出アルゴリズムの基本が確立され
た．1970年代には，機械部品や室内風景など複雑な実世界を扱う
ようになり，これらの技術を使って実用化研究が行われたが，カメ
ラで取込んだ画像から物体を認識することは困難で，生産ラインで
の導入においては大きな制約（カメラの方向，照明の仕方，物体の
置き方など）があった．
1980年代には，マイクロプロセッサの出現と進展とともに，生
産ラインでは外観検査，位置決めを行う視覚センサが急速に普及
し，機械の目の本格実用期に入った．同時に画像処理技術の高速
化，小型化，インテリジェント化が加速し，実用可能なアプリケー
ションが多数実現した．1990年代に入ると，LSI技術やITの進展
により，小型化，高速化，高精度化およびネットワーク化が一層進
展した．
b.画像処理装置のニーズ 生産ラインにおいては，検査工程
の自動化の高まりは著しく，人手不足・労働環境対策の要因もある
が，検査の高精度化・高品質化・高速化および制御の高度化などが
挙げられる．おもなニーズを表5･6 に示す．
一方，検査機器装置は世の中の変化に対応するため早くから実用
化研究がなされてきた．さまざまな視覚認識装置が開発されなが
ら，価格・精度・処理速度などに一長一短があり，本格普及までに
は至らなかったが，高精度，高感度の撮像素子の実用化・画像処理
技術の進歩およびマイクロプロセッサおよびITの進展により，精
度・処理速度の点で，人間の目をはるかにしのぐ機器装置が現実と
なってきている．

(１)撮像装置
CCD素子を利用したCCDカメラが主流であり，画素数の増加に
伴い，高分解能化，低価格化が進んでいる．今後，撮像装置として
画像の取込みだけでなく，画像処理機能を保有したインテリジェン
トセンサが出現すると考えられる．
(２)画像メモリ
画像メモリは半導体技術の進展とともに，高速・大容量・ローコ
スト化が実現してきている．今後も高速化，大容量化，ローコスト
化は進むが，加えて低消費電力化が求められ，低電圧化が進むと考
えられる．
(３)前処理
前処理はリアルタイム処理が要求されるため，画像処理用LSI
で構成されることが多い．1970年後半に，ディジタル信号処理の
演算をソフトウェアで高速実行するディジタル信号処理LSI
（DSP:digital signal processor ）が出現し，広く使用されるよう
になっている．

.画像処理装置ソフトウェア 一般的な画像処理装置
のソフトウェア構成図を図5･48に示す．
(１) システム管理
画像処理のOS部分である．画像処理プログラムのプログラミン
グや処理の全体管理を行う．
(２) 画像データ管理
画像データが主記憶装置/補助記憶装置のどこに格納されている
のかの管理や外部記憶装置へ出力する場合のフォーマット管理など
を行う．
(３) 画像入出力機器管理
A-D変換された撮像装置からの画像入力やD-A変換してモニタ
テレビへの出力などの入出力管理を行う．接続するためには，各機
器専用のソフトウェアが必要になることが多い．
(４) ユーザインタフェース
メニューを用いてユーザとの対話や画像データ表示を行う．
(５) 画像処理アルゴリズム
画像処理の基本機能を行う．有効活用しやすいようにモジュール
化，ライブラリ化されている（表5･8) ．
新たな画像処理アルゴリズムとして，
(ⅰ)ニューラルネットワーク 人間の脳はニューロンと呼ば
れる神経細胞が組合わさった構造（神経回路網）で構成されてお
り，この構造をまねることで，人間が得意とするパターン認識に適
用しようとするものである．文字認識などに使われ始めている．
(ⅱ)ファジィ推論 人間が得意とするあいまいさをコンピュ
ータに扱わせる手法で，IF～THEN～という形式でいくつかのル
ールを用意し，そのルールに基づいて行うもの．特定の色成分の抽
出や文字認識などでの検討がされている．
(ⅲ)遺伝的アルゴリズム 生物の進化の過程をまねることで
ソフトウェアの最適化を図る手法で，遺伝と自然淘汰を繰返すこと
によってより優秀なアルゴリズムを導き出そうとするものである．
選択，交差，突然変異の三つのプロセスを行う．
･選択：集団の中から優秀なものを選び出す．
･交差：選び出された集団の中でランダムに遺伝子の一部を交換す
る．
･突然変異：低い確率で起こり，遺伝子情報の一部をランダムで書
き換える．
このような方法で最適解をランダムで最速に探索する手法であり，
最適照明やカメラの最適観測点などの研究がなされている．
