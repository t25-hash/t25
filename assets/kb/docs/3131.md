# 5・6・1 マルチメディア（multi - media）技術の導入

> **章**: 第5章 生産設備
> **節**: 5・6 ヒューマンインタフェース

工作機械のヒューマンインタフェース（human interface）は，
従来の機械では文字だけを表示するコンソールとキーボードやスイ
ッチによる入力装置を備えるものであった．コンピュータの発達と
ともにマルチウインドウやグラフィックスによる表示，入力装置と
してのタッチパネルが用いられるようになってきた．さらに，五感
情報を提示可能なマルチメディア技術も進んでおり，静止画ばかり
でなく動画や音声情報も活用されるようになってきた．
