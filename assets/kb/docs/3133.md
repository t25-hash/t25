# 4・1・7 検定と推定

> **章**: 第4章 測定データの処理
> **節**: 4・1 静的データの処理

このとき、欲しい情報は個の測定データの要約値ではなく、測定対象の真の値μに関する情報である。式(4・18)のεは誤差である。仮想的に、測定データを無数に得ることができると考えたとき、上記の個の測定値は、数値が記入された無数のチップが入った袋から無作為に抽出(ランダムサンプリングという)されたもの(実現値)である。この袋に相当する概念を母集団(population)と呼ぶ。無作為抽出の行為によって選ばれた母集団の構成要素は、それがどのような値をとるかがある確率法則によっているため、確率変数(random variable)という。確率変数は確率が定義できる分布(確率分布)によって記述される。確率変数には連続型と離散型があり、連続型の確率変数に対する確率分布(確率密度関数と呼ぶ)としては正規分布、離散型の確率変数に対する確率分布としては二項分布が代表的である。以下、正規分布が仮定できる母集団(正規母集団と呼ぶ)を対象とする。式(4・19)は二つのパラメータ(母数)μとσによって決まるので、正規分布をμ,σと表記することが多い。このことから、正規分布は、ばらつき(母分散)一定のもとで位置(母平均)だけを移動させることも、位置一定のもとでばらつきだけを変化させることもできるという、一般の分布にはない性質を有している。b. 正規分布の標本分布確率変数(=1,...,)の関数を統計量といい、統計量の分布を標本分布という。. 分布次式で計算される統計量χは、自由度φ=-1のχ分布に従う。4・1・3・b項でも述べたように、を不偏分散と呼び、σの推定量として用いる。これらは、統計数値表や、表計算アプリケーションソフトウェアにアドインされている関数から求めることができる。c. 検定の基本的考え方と数値例検定とは、母集団からサンプリングして得たデータをもとに母集団に関する命題(仮説)の検証を行うことである。具体的な数値例を用いて検定の考え方を説明する。表4・2に示したねじ締付け工程での軸力の目標値は40kN である。実験結果をもとに、締付けトルクの設定は現状でよいかどうかを検討したい。検証したい仮説は、表4・2のデータの母平均μが40kN であるということであるが、は40.43kN である。0.43kN のずれをどのように判断すればよいであろうか。仮説検定の基本は、適切な統計量(検定統計量という)を選び、仮説が正しいとしたとき、検定統計量が従う確率分布を考えることである。この場合の母集団はねじ締付け工程であり、その母集団分布として正規分布を仮定する。次に、仮説が正しいとしたときまれにしか起こらない値の領域(棄却域という)を設定する。しかし、まれにしか起こらないという表現は主観的であり、客観性に乏しい。そこでこれを確率で表し、前もって宣言しておく。この確率を有意水準という。有意水準(significance level)はαと表記され、通常、α=0.05に設定することが多い。の実現値が、まれにしか起こらない領域に含まれたとき仮説が成立していれば、まれにしか起こらないと考えられる現象が起きたとして仮説(帰無仮説)を棄却し、この結果を統計的に有意であると表現する。しかし、仮説が正しいとき、確率0.05で生起するの領域は無限に存在する。そこで、仮説を棄却したときに採択するもう一方の仮説(対立仮説)を前もって決めておき、それが正しいとき、そのことを最も効率良く検出できるように棄却域を設けておく。有意水準を決めるための帰無仮説をH、対立仮説をHと表記する。もし、対立仮説が正しいとき、それを最も効率的に検出するためには、分布の両すそに棄却域を設定するのが合理的である。よって、まれな現象ではない(有意ではない)と判断し、目標値40kN からずれているとはいえないという結論を得る。もし仮説がH: μ=40.0kNH: μ<40.0kNであるならば、分布の確率が0.05となる左すその部分に設定するのが最も合理的である。前者を両側検定、後者を片側検定という。d. 推定の基本的な考え方と数値例再度、表4・2の数値例を取り上げる。検定の役割はアクションへの動機付けであった。これに加えて、アクションに対する具体的な情報として母平均と目標値とのずれに関する情報を得たい。これを推定という。検定は母数に関する命題の検証であり、推定は母数の値を求めることである。このように、母数を一つの数値として求めるとき、点推定(point estimation)と呼ぶ。推定には点推定のほかに区間推定(interval estimation)がある。このとき、1-αを信頼係数といい、通常95%が用いられる。数値例では、締付け軸力の母平均μの点推定値は、表4・2からμ==40.43kN である。自由度9の分布の2.5パーセント点は2.262であるのでPr-2.262≦≦2.262=0.95である。検定では、μ=40.0kN の仮説を棄却しなかった。区間推定の結果、[38.78kN,42.08kN]を得た。検定の結果、このデータではμ=40.0kN を否定できない。一方、区間推定は[38.78kN,42.08kN]を示している。推定精度(信頼区間の幅)が満足できるものかを判断したうえで、必要であればさらにデータをとり、検討を行うことを考える。e.
