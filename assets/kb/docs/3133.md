# 5・6・3 ネットワーク化された生産システムのヒューマンイン

> **章**: 第5章 生産設備
> **節**: 5・6 ヒューマンインタフェース

タフェース
はん用OSを搭載したCNC，あるいは，パーソナルコンピュー
タNCが普及したことにより，CNCを直接ネットワークに接続す
ることが容易になっている．また，パーソナルコンピュータ機能を
もたないCNCについてはNCにネットワーク接続用のボードを入
れることによってネットワークに接続が可能となっている．これを
利用して電話回線やインタネット上で遠隔地の工作機械を監視した
り，遠隔地からリモート操作（remote operation）したりするシ
ステムが工作機械・産業機械の分野でも増えてきている ．遠
隔監視ソフトウェアについては，CNCメーカが自社CNCとパー
ソナルコンピュータ間で通信を行うためのAPIを供給することで，
工作機械メーカが独自にネットワークソフトウェアを開発すること
ができる環境が整備されてきている．遠隔監視の機能については，
多くのメーカがプログラム転送，リモート監視，アラームの携帯電
話への通知の機能を実現している．機械一覧画面では多くの場合，
機械が運転中であれば一覧画面に表示されている各機械の表示が
青，または緑で表示され，機械が停止中ないし待機中のときは黄
色，アラーム発生中は赤で表示される．このように機械の稼働状況
を色で区別して表示し，アラームを発生している機械が容易にわか
るようになっている．一覧画面上で機械を選択すると指定された機
械の詳細情報が表示されるようになっていることが多い．遠隔シス
テムで実現されている機能としては，リモートメンテナンス
（remote maintenance），稼働状況監視，マニュアル表示，データ
アップロード/ダウンロード，E- mail送信，ソフトウェアアップ
デート，オプション追加があり，各社が多少異なった用語で呼んで
いる．また，稼働状況監視画面の用語として，稼働状況監視一覧画
面，同詳細画面，実績監視画面，NC画面モニタ，アラーム履歴，
生産進捗などがある．通常のCNCに関しては歴史が長く，開発し
ているメーカも多くはないため標準化にはあまり議論の余地はない
が，遠隔監視，遠隔メンテナンスなどに関しては歴史も浅く，導入
するメリットをユーザが見いだすことができるよう議論の余地があ
る．
