# 4・2・1 動特性の表示

> **章**: 第4章 測定データの処理
> **節**: 4・2 動的データ処理

計測すべき信号が時間とともに変化する場合は、計測系の動特性を考慮しなければならない。ここでは、線形同次微分方程式で記述される計測系について考える。そのような計測系の動特性は入出力信号間の伝達関数(transfer function)で取扱う。図4・8は振動を生じる応答の例である。この場合、以下のような特性値により応答性を表示する。(1) 立上り時間(rise time):応答が最終出力値の10%から90%に達するまでの時間であり、応答の速さを表す。(2) 行きすぎ量(overshoot):応答の最初のピーク値と最終出力値との差を最終出力値で除したもので、応答の安定性を表す。(3) 整定時間(settling time):応答が最終出力値から許容範囲内(例えば±5%以内)に収まるまでの時間であり、応答の速さと安定性の両方の特性を表す。b. 周波数特性計測系に正弦波入力を与えて定常状態に達した場合には、出力信号も正弦波信号となる。それらの入出力正弦波信号の振幅比と位相を入力正弦波の周波数の関数として与えるのが周波数応答(frequency response)である。つまり高周波数域では、入力周波数が10倍高くなると検出感度が20dB 減少することを示す。また、周波数が高くなるほど出力信号は遅れ、-90°までの位図4・9一次遅れ系のボード線図図4・10 二次遅れ系のボード線図相差がつくことがわかる。ここで、ζは減衰比、ωは系の固有角周波数である。また、周波数が高くなるほど出力信号は遅れ、-180°までの位相差がつくことがわかる。そこで、系の固有角周波数ωをできるだけ高くとり、ゲインのピーク値や位相遅れも勘案して減衰比ζを定めるのがよい。
