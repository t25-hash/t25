# 5・1・1 生物にならう制御

> **章**: 第5章 知的制御
> **節**: 5・1 知的制御概論

知的制御は生物（特に動物）のもつ能力をならおうとしているこ御の主目的である．
生物的人工物の実現人間の計測・制御行動のコンピュータ化なわち人間のもつ能力をうまく利用することにより，従来の制御理の機能を工学的に見れば，図5･1(ａ)のようになる．
他方，図5･1(ｂ)は人工システムの制御機能の階層構造を示したもので，図5･とはいうまでもないが，それを制御の観点から分類すると次のようになる．
補装具や人工臓器など生体機能の一部代替5･1節での知的制御は上のに関するところが中心である．
す論の欠点を補完する．
ここで，情動などの機能を別にして，ヒト脳る．
