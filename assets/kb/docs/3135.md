# 4・2・2 フィルタリング

> **章**: 第4章 測定データの処理
> **節**: 4・2 動的データ処理

計測される信号から望ましくない成分（雑音）を除去し，必要な
成分のみを取出す操作がフィルタリング（filtering）である．例え
ば，センサケーブルに電磁誘導により混入する高周波雑音の除去
や，振動加速度計測における低周波のうねり成分の除去など，ここ
では，周波数帯域に応じて雑音成分を除去する周波数領域フィルタ
について取扱う．
