# 4・2・2 フィルタリング

> **章**: 第4章 測定データの処理
> **節**: 4・2 動的データ処理

計測される信号から望ましくない成分(雑音)を除去し、必要な成分のみを取出す操作がフィルタリング(filtering)である。例えば、センサケーブルに電磁誘導により混入する高周波雑音の除去や、振動加速度計測における低周波のうねり成分の除去など、ここでは、周波数帯域に応じて雑音成分を除去する周波数領域フィルタについて取扱う。a. フィルタの分類フィルタ(filter)は、信号の周波数の通過域または阻止域により以下の4種類に分類され、それぞれの理想的な周波数伝達関数を図4・11に示す。(1) 低域通過フィルタ(LPF: low-pass filter)5―31(a) 低域通過フィルタ(b) 高域通過フィルタ(c) 帯域通過フィルタ(d) 帯域除去フィルタ図4・11 各種フィルタの理想的な周波数特性(2) 高域通過フィルタ(HPF: high-pass filter)(3) 帯域通過フィルタ(BPF: band-pass filter)(4) 帯域除去フィルタ(BEF: band-elimination filter)図4・11中の通過・阻止の境界の角周波数を遮断角周波数(cut-offangular frequency)と呼ぶ。しかしながら、現実には図4・11のような急峻な周波数特性は得られず、通過域と阻止域の中間に遷移域をもち、また通過域においてもゲインが一定とはならず、入力周波数により変動する。b. フィルタ特性よく用いられるフィルタ特性について、まず低域通過フィルタの特性を以下に示す。. バタワース特性(Butterworth characteristics)次式のように、伝達関数がラプラス演算子の多項式の逆数で与えられるフィルタを多項式フィルタという。バタワース型低域通過原形フィルタは次数を大きくしていくと、高域での減衰が大きくなり、理想的な低域通過フィルタに近くなる。減衰特性は6(dB/octave)である。また低域ではゲインにリップルがなく、最大平坦特性をもつ。. ベッセル特性(Bessel characteristics)次式の伝達関数をもつフィルタをベッセル型低域通過原形フィルタと呼ぶ。. チェビシェフ特性(Chebyshev characteristics)次式の漸化式で求められるチェビシェフ多項式を用いて得られるのがチェビシェフ型低域通過原形フィルタである。遮断角周波数以降の遮断特性が最も急峻であり、理想的な遮断特性といえ、通過域のゲインにリップルを生じる。振幅比を拡大表示すれば図4・15となる。このリップルが大きいほど急峻な遮断特性が得られるが、ステップ応答にはリンキングが生じる。先に述べた通過域、阻止域で分類したすべての具体的なフィルタは低域通過原形フィルタから表4・3に示す周波数変換を用いて求めることができる。例として、五次のチェビシェフ型低域通過原形フィルタから求められる各種フィルタのゲインの周波数特性を図4・16に示す。c. フィルタ回路の構成各種フィルタはアナログ電子回路で実現できる。最も簡単なものは図4・17の抵抗とコンデンサ(a) 低域通過フィルタ(b) 高域通過フィルタ(c) 帯域通過フィルタ(d) 帯域除去フィルタ図4・16チェビシェフ型低域通過原形フィルタから導出される各種フィルタ図4・17 RC低域通過フィルタで構成されるRC低域通過フィルタである。受動素子のみで構成されるためパッシブフィルタ(passivefilter)と呼ぶ。高次のフィルタを得るためにはRCフィルタ回路を多段接続することもできるが、演算増幅器(OPアンプ、opera-tional amplifier)を用いたアクティブフィルタ(active filter)が有用である。例えば図4・18は、正帰還型二次低域通過フィルタ回路の例である。この回路の素子定数を変えつつ多段接続すれば、各種高次低域通過フィルタが得られる。また、回路のRC-CR 変換を行えば高域通過フィルタも得られる。d. ディジタルフィルタアナログ電子回路ではなく、計測用コンピュータまたはディジタルシグナルプロセッサ(DSP: digi-tal signal processor)によるディジタル演算処理でフィルタリングを行うのがディジタルフィルタ(digital filter)である。このデータを用いて、4・2・2・b項のバタワース特性やチェビシェフ特性などの周波数特性をもつディジタルフィルタをコンピュータプログラムとして構成することができる。最も簡単なディジタルフィルタは、移動平均(moving average)法である。この処理によりデータの平滑化が行える。
