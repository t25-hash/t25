# 4・2・3 周波数解析

> **章**: 第4章 測定データの処理
> **節**: 4・2 動的データ処理

計測された信号にどのような周波数成分が含まれているかを調べることを周波数解析(frequency analysis)またはスペクトル解析(spectrum analysis)といい、動的データ処理の基本となるべきものである。無限に続く信号を取扱うために、次式のフーリエ変換(Fourier trans-form)を定義する。振幅、位相スペクトルを用いて、次式のフーリエ逆変換(inverse Fourier transform)により時間信号を再生できる。離散フーリエ変換および離散フーリエ逆変換(IDFT : inversediscrete Fourier transform)はそれぞれ次式で定義される。離散フーリエ変換は標本数の二乗()の積和演算が必要であるため、標本数が多くなると計算時間が膨大となる。この問題を解決するのがクーリー(Cooley)とテューキー(Tukey)によって提案された高速フーリエ変換(FFT : fast Fourier transform)である。このため、周波数解析には一般にFFT を用いる。b. 標本化定理とエリアシング誤差周波数解析を行うためには、まずアナログ信号の計測データのA-D 変換を行って標本化する。この際、計測データに含まれる上限の周波数をとすれば、標本化周波数(sampling frequency)との間に、次式を満足しなければならない。標本化定理が成立していなければ、周波数解析結果から原信号の計測データを再生できない。逆にいえば、周波数解析結果に本来存在しない成分が生じる。これをエリアシング誤差(aliasing error)という。周波数スペクトルは最大値に対して正規化してデシベル表示している。これに対応して図(図4・20中の黒丸で表す)が得られているが、本来存在しないその他の成分(図4・20中の白丸で表す)は、高次成分に対して標本化定理が成立していないために生じたエリアシング誤差である。このような誤差を防ぐためには、一般にデータの標本化(A-D 変換)の前段階でアナログ低域通過フィルタによる高周波成分の除去を行う。このようなフィルタをアンチエリアシングフィルタ(anti-aliasing filter)という。c. 窓関数を用いた周波数解析離散フーリエ変換は有限の区間を基本周期としてその整数倍の周波数スペクトルを求めるよう定義されている。したがって、その解析区間に周期性がない場合、すなわち区間の始点と終点のデータ値が異なっている場合、誤差を生じる。この重みを窓関数(window function)と呼ぶ。周波数解析によく用いられる窓関数を表4・4に示す。矩形(rec-tangular)窓のみ、振幅が同じで2.8Hz と3.2Hz の成分を分離検出できており、周波数分解能が高いことがわかる。このように、目的に合わせて窓関数を選択する。一般的にはハニング窓を使用するとよい。
