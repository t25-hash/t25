# 5・2・2 ファジィ制御の構成

> **章**: 第5章 知的制御
> **節**: 5・2 ファジィ制御（fuzzy control ）

ファジィ制御の明確な定義は存在しないが，ファジィ推論を利用
した制御と考えるのが一般的である．この制御は，ザデー教授のフ
ァジィ理論の意思決定問題への適用を提案する論文 を契機とし
て，1974年にスチームエンジンの実験装置の制御への適用が試み
られた ．日本では，1979年ごろより産業界での応用研究が開始
され ，列車の自動運転 や浄水場の薬品注入制御 が最初の例
である．1987年に仙台市地下鉄にてファジィ制御列車自動運転シ
ステムが稼働し ，実用的な技術であることが実証されるととも
に，鉄鋼，非鉄金属，化学，食品，建設，土木，空調などの産業分
野へ適用が一気に進み，1989年には，洗濯機やエアコンなどの家
電製品に適用が拡大した ．
従来の制御理論は，対象システムの線形モデルを必要としてい
る．しかし現実のシステムは，作成したモデルの適用範囲が狭い，
モデルパラメータの同定が難しい，適切な観測データが得られない
ため，制御系の設計を行うことが難しい対象が多く存在する．この
ような対象でも，熟練オペレータはうまく運転・操作を行っている．
ファジィ制御は，この熟練オペレータの制御に対する知的活動
を，(１)その経験則に含まれる言葉（命題）の意味をファジィ集合
によって定量化し，(２)その構造を制御知識（If -thenルール）と
して記述することにより計算機化することができる．すなわち，熟
練者の制御に関する知識を備え，対象システムからの時々刻々の問
い合わせ（システム状態の観測）に対する回答（制御指令）の知的
活動をリアルタイムに実行するエキスパートシステムとして考える
ことができる．

レーキの踏み方を例にとり説明する（図5･11)．このブレーキ操作

では，前の車にぶつからないように車間距離と走行速度からブレー
キをかけることを考えている．したがって，運転知識は，「もし，
車間距離が小さく走行速度が大きいならば，ブレーキを強く踏む」
「もし，車間距離が大きく走行速度が小さいならば，ブレーキを弱
く踏む」のように記述できる．この「車間距離が小さい」「走行速
度が大きい」などをファジィ集合で定義し，各ルールを用いてファ
ジィ推論を行い，現時点で適切なブレーキ力を決定する．
このような熟練者の制御に関する知的活動を組込む方式として以
下のようにファジィ制御方式が適用されている．
(１) 状態評価ファジィ制御（state evaluate fuzzy control )
対象システムの状態を，運転知識に基づき総合的に判断し，制御指
令を決定する方式である．目標変更や原料変更などに伴う外乱要因
を用いてフィードフォワード制御などを協調させた制御も可能であ
る．特に，対象システムの構造が不明であり制御系の設計が困難な
場合にも，熟練者の運転知識に基づき適用が容易である（5･2･3項
参照)．
(２) 予見ファジィ制御（predictive fuzzy control ) 現在
のシステム状態で△△の操作（複数）をしたならば，◇◇のように
動くから制御目的□□を満足した制御ができそうだ」という運転知
識に基づき，制御指令を決定する方式である．対象システムの動き
の予測・予見にモデルを用いているが，簡易モデルや特性パラメー
タの変動を組込むことが容易であり，非線形な対象でも環境の変動
に頑強な制御系の構築が可能である（5･2･4項参照)．
(３) ファジィモデルに基づく制御 制御対象の動特性をファ
ジィモデルにより記述し，それを踏まえて各部分の制御器を制御理
論の手法に基づき設計し統合する方式（5･2･5項参照)．
(４) ファジィ適応制御 制御対象システムの特性変動に適応
させて，制御アルゴリズムを変化させる方式（5･2･6項参照)．
さらに，ファジィ推論により操作器への入力を直接求めるのでは
なく，現場のオペレータが制御対象の状況を見ながら従来の制御装
置をそのまま用いて，目標値を高めに設定したり比例制御のゲイン
を調整したりしている操作を目標値設定則やパラメータ設定則とし
て記述し，既存の制御系に上乗せして制御性能の向上，自動化を図
る統括ファジィ制御（図5･12）の手法が既存制御システムの性能
向上に有効であり適用例も多い．
ファジィ制御の一般化と普及に伴い，専用のマイクロプロセッサ
やコントローラハード，ソフト，および制御系設計支援システムの
提供がなされている．
