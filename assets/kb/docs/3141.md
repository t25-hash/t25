# 5・2・4 予見ファジィ制御（predictive fuzzy control )

> **章**: 第5章 知的制御
> **節**: 5・2 ファジィ制御（fuzzy control ）

予見ファジィ制御は，制御状態の観測値を基に，制御則に仮定さ
れた制御指令の代替案について制御対象の動きを対象システムモデ
ルを用いて予測計算を行い，その予見・予測した制御目的となる被
評価量の結果についてファジィ多目的評価をして，最適な制御指令
を決定する方式である（図5･18)．
具体的には，対象システムの特性を把握したオペレータの経験則
から「もし（If ）制御指令uをCとした場合，評価指標xはA
であり，評価指標yはBであるならば，（then）制御指令uをC
とする」といった制御規則に基づき，現時点で実行可能な制御指令
の中から，最も望ましい状況に制御できる制御指令を選択してい
く．この予見ファジィ制御の制御則を一般的に示すと，
If ( u is C →x is Aand y is B )then u is C.
…
If (u is C →x is Aand y is B )then u is C.
となる．
図5･19に，この制御方式の推論過程を示す．対象システムの計
測値を用いて現在実行中の制御指令を保持した場合，増加した場
合，減少させた場合など，すべての制御指令候補について対象モデ
ルを用いて対象の動きの予見・予測計算を行う．その予見値を，制
御目的である「許容範囲にある」「正確である」「大きい」「小さい」
などのファジィ集合で評価し，ファジィ多目的評価を行う．その結
果より最適な制御指令を決定する．すなわち，その時点で実行可能
な制御指令の代替案に対する実行結果を予見・予測し，制御則に記
述した知識に従い，最も望ましい状況に制御できる制御指令を選択
する方式である．
この制御方式は，列車自動運転システムへ実用化されており，仙
台市地下鉄にて，乗客の乗り心地を直接考慮しながら，正確にホー
ムに停止する運転を実現している ．また，壁への近さ，障害物
の動きなどを考慮した，自動車のハンドル操作などに適用が図られ
ている．
