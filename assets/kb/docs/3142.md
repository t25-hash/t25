# 5・2・5 ファジィモデルに基づく制御

> **章**: 第5章 知的制御
> **節**: 5・2 ファジィ制御（fuzzy control ）

ファジィモデル（fuzzy model ）に基づく制御は，制御系の安定
性や他の制御性能を満足するファジィ制御器をシステマティックに
設計することを目的としている．このアプローチでは，非線形シス
テムをファジィモデルで表現し，ファジィモデルに対応した制御器
を構成し，安定性や他の制御性能を満足するフィードバックゲイン
を求めるものである．
ファジィ制御系の安定性（stability）や他の制御性能を保証する
条件の多くは線形行列不等式（LMI:linear matrix inequality）
条件で記述されている．このため，ファジィ制御系の多くの設計問
題は数値的に解ける設計問題となっている．線形行列不等式で記述
されている制御系設計条件の一例として，decay rate，入出力制
限，外乱除去， 制御，二次形式評価関数の上限値を最小化する
条件，ロバスト安定条件，動的出力フィードバック，オブザーバ設
計，非線形モデル追従制御問題などがある ．
ファジィ制御系の設計問題では，メンバシップ関数値のpermu-
tationの操作 を用いることで，安定条件を含めた多くの制御性
能条件を緩めることができる．この点がポリトピックシステムの安
定解析よりも優れている点である．メンバシップ関数の特性や具体
的な関数型を考慮することにより，さらにシャープな条件を得るこ
とができる．
