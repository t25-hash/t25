# 5・1・7 光度(luminous intensity)

> **章**: 第5章 各種変量の測定
> **節**: 5・7 の温度の欄に記された温度範囲に入っていなければならない。

a. はじめに光度の単位カンデラ(candela,cd)は、国際単位系(SI)の基本単位に含まれているが、他の六つの基本単位が純物理量であるのに対し、カンデラは視感度という生体効果の係数を含んだ心理物理量の単位である。このように、光度を含む測光量は純物理量でないことに注意しなければならない。1979年の第16回CGPM で採択されたカンデラの定義は、旧定義を抜本的に改訂したもので、「カンデラは、周波数540×10Hzの単色放射を放出し、所定の方向におけるその放射強度が1/683W/srである光源の、その方向における光度である」となった。旧定義は、白金凝固点の温度にある黒体からの放射を標準にしてカンデラの単位を定義していたが、新定義は標準光源、標準受光器といった標準器に頼らず、周波数540×10Hz(波長555nm)の単色放射に対して他のSI基本単位から導かれた物理量を用いて定義されているので、厳密に計測できるわけである。とはいっても、他の波長や広い波長分布をもつ光源(一般にはこのケースがほとんどである)の光度を求めるには、人間の眼の分光感度を標準化・規格化した比視感度関数λを併用しなければならないことになった。このλを放射の視感効果度(luminous efficacy ofradiation)という。したがって測光はλというフィルタをかけた放射エネルギーの測定と考えてよい。光束の単位はルーメン(lumen,lm)を用いる。λは黄色い光が赤や青より明るく感じられることから、可視部の中央部に極大を有する曲線と考えられるが、この曲線は人によって、また観測条件などによって異なっている。したがってこのままでは測光というものに普遍性がなくなる。そこで、ある一定条件下における多くの人の分光視感効率の適当な平均値をとり、このような分光視感効率をもつ人(標準観測者)が観測した値が測光値であると約束する。明所視における標準観測者の分光視感効率λ(比視感度曲線)としてわれわれが採用している値は、1924年国際照明委員会(CIE: Commission Internationale de l’Eclairage)が同意し、補間と補外を行って完全なものとしたのち、1972年にCIPM が勧告した値である。暗所視における分光視感効率′λに対しては、CIEが1951年に若い観測者についての値を採用し、CIPM は1976年にその値を確認した。ここで、明所視とは少なくとも数カンデラ毎平方メートル以上の輝度レベルに順応したときの正常眼による視覚をいい、暗所視は百分の数カンデラ毎平方メートル以下の輝度レベルに順応したときの正常眼による視覚をいう。次に、光度、輝度、照度と光束との関係を考える。光度(lumi-nous intensity)は光源の本来の明るさを与える量で、単位立体角当たりの光束の量で表す。光度の単位名はカンデラ(cd)で、lm・srで表される。概念としては、発光面を頂点として頂角がψである円すい形を考え、この円すい内の光束の量dを考える。立体角を決めれば、光源からの距離が変わってもその中の光束の量は一定であるから、光源自身の光の強さを定めるのに適している。ここで、測光の量と単位の組立方に相違があるので注意しなければならない。量としては光束が基本であるが、単位としては光度の単位カンデラが基本となっている。したがって、光束の単位ルーメンは、すべての方向に1cdの光度をもつ光源が立体角1sr内に放出する光束として定義されている。光を受け取る側(受光面)の測光量として照度(il-luminance)がある。表面の特定の点での照度は、その点を含む単位面積に入射する光束の量で表す。単位はルクス(lux,lx)で、lm・mで表される。すなわち、照度1lx というのは、面積1m当たりに光束1lmの割合で光が入射しているときのことを示している。ある位置から見たときの光源の、一般には物体の面の輝きの強さを表す量が輝度(luminance)である。光源が有限の面積を有し、われわれが光源の各部分を見つめることができる場合、眼が感じる明るさは視線に直角の面上での単位面積当たりに直した光度に比例する。輝度の単位はcd・mで表され、特に固有の名称による単位は与えられていない。一様な発光面または反射面が完全拡散面(面の法線となす角がθの方向での光度がcos θに比例する)の場合には、この面の輝度は式(5・15)よりどの方向に対しても一定である。c. 広義の測光量測光を広くとらえると、可視域に限らず放射の強さに関係する量を測ることといえる。そのような量として放射束、放射強度、放射照度、放射輝度などの量があり、これらの量は式(5・12)においてλの代わりに波長によらず1とおき、放射のパワーをワットの単位で測った放射束を基本量として表されるものである。各波長ごとの量として分光放射輝度、分光放射照度、分光応答度などがある。図5・27 でカンデラの新しい定義が採択されるまでの間、「カンデラは、白金の凝固点にある黒体の60万分の1mの平らな表面の垂直方向の光度である」(1967年の修正では、101325Paの圧力のもとでという内容の指定が加わった)と定められていた。1979年のカンデラの新しい定義は=683lmWと規定しているので、わが国の測定結果と全く同じであり、新定義になってもわが国のカンデラの大きさが変わることはない。近年、各国の標準研究所は新しく開発された極低温電力置換型絶対放射計を導入し、これをベースに光度単位の設定が行われている。従来の室温放射計はSN 比が悪く、測定精度が上がらないものであった。受光器を液体ヘリウム温度まで冷却することにより、熱雑音の低減と熱伝導の等価性が向上し、SN 比は飛躍的に改善される。この手法によれば、測光量の測定精度は従来(自己校正法)よりも大きく改善されている。.
