# 5・2・1 寸法(size),変位(displacement)

> **章**: 第5章 各種変量の測定
> **節**: 5・2 幾何学量の測定

a. 長さの実用標準長さの実用標準は大別すると,長さ標準に直結した実用安定化レーザ光源(波長)と線度器(line stan-dard),端度器(end standard)に分けられる.実用安定化レーザ光源はその発振波長を長さ基準とし,光波干渉計,干渉測長器等の光源として用いられ,標準器の値付け,高精度な測定に用いられている.線度器は目盛の線と線との間隔を基準長さとするもので,標準尺,金属製直尺,巻尺などがこれに属する.一方,二つの面間の距離が一定の基準寸法となるものを端度器と呼び,ブロックゲージ,標準棒ゲージ,限界ゲージ,すきまゲージ,標準鋼球などがこれに属する.. 実用安定化レーザ光源一般に波長633nmのHe-Neレーザの発振波長(周波数)を安定化した光源を実用安定化ヘリウムネオンレーザ(commercial stabilized He-Ne laser)と呼んでおり,その発振波長の安定度は10~10程度である.代表的な安定化法として,ゼーマン法,2モード法などがある.この発振波長は通常,上位の長さの基準であるヨウ素安定化ヘリウムネオンレーザ(iodine stabilized He-Ne laser)光源で値付けして使用する.この実用安定化レーザ光源を大気中で使用する場合,レーザビーム周囲の空気屈折率により発振波長が変化することに注意する必要がある.真空中のレーザ波長をλ욋,空気屈折率をとすると,空気中の波長λ욈はλ욈=λ욋となる.通常,レーザビーム周辺の環境(温度,気圧,湿度等)を測定し,その結果を用いて空気屈折率を演算により算出[エドレン(Edle썝n)の算出式等]し,発振波長を補正して用いる.この補正の精度は環境測定精度に左右されるが,おおむね10~10程度である.最近では633nm以外の波長,あるいはHe-Neレーザ以外のレーザを用いた実用安定化レーザ光源も使われている.. m程度で,ガラス基板上にフォトリソグラフィーに代表される微細加工法により製作される.最近ではピッチ0.5
