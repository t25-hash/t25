# 5・4・2 遺伝的アルゴリズムによる直接的最適化による制御

> **章**: 第5章 知的制御
> **節**: 5・4 遺伝的アルゴリズムによる制御

GAによる最適化は，問題を個体としてコーディングすること
と，目的関数（適応関数）を定義することから始まる．直接的最適
化は，問題そのものを個体として取扱いできる場合であり，おもに
記号表現される最適化問題や数値を扱う場合でも組合せ最適化問題
である．つまり，静的な最適化問題やオフラインでの最適化問題を
扱うことができる．例えば，トラベリングセールスマン問題，ナッ
プサック問題，ジョブショップスケジューリング問題などが直接的
最適化問題といえる ．また工学的な応用として，エレベータの
最適配置問題，ICチップの最適配置問題，原子炉停止最適制御問
題，倒立振り子の制御問題などに適用されている ．
一方，コントローラや制御対象に対する評価が複雑な場合には，
評価関数の記述や同定が困難な場合がある．特に，人間の感性によ
って評価されるような場合には，その評価を関数として記述するこ
とが容易ではない．ファジィを用いて，人間の経験的な評価を関数
化したファジィ評価を用い，それをGAによって最適化する方法
も提案され，有効性が示されている ．これは，非線形な評価
関数の問題に対してGAが有効であることを示すとともに，人間
と制御対象とのインタフェースとしてファジィを用いることが有効
であることを示している．さらに，制御対象に対する評価のデータ
から非線形回帰モデルによって評価関数を同定し，それをGAに
よって最適化する方法も提案されている．例えば，ロボットアーム
の軌道計画問題において，熟練教示作業者のスキルを非線形回帰モ
デルによって評価関数を構築し，GAにより最適な軌道を計画する
方法の有効性が示されている ．
