# 5・4・3 遺伝的アルゴリズムによる間接的最適化による制御

> **章**: 第5章 知的制御
> **節**: 5・4 遺伝的アルゴリズムによる制御

GAの特徴は，最適化の過程で最小二乗法などのように，最適化
したい関数のこう配情報を必要としないことである．この特徴を生
かして，個体のコーディングと最適化したい問題とを直接的に対応
させずに最適化することができ，これを間接的最適化と呼ぶ．GA
は，最適化に計算時間がかかるため，オンラインのパラメータチュ
ーニングには向いていない．また数値でのコーディングは探索空間
が膨大になるため，大まかにしか最適化できない場合が多い．そこ
で，オンラインの適応的な制御では，コントローラ自身に適応側を
与えることが必要である．このような場合に，ニューラルネットワ
ークやファジィを用いて数値表現される問題を取扱うようにす
る ．コントローラの内部では，ニューラルネットワークやフ
ァジィによって適応的に制御を行う．このとき，GAは，制御結果
を適応度にして，コントローラに使用されているニューラルネット
ワークやファジィの構造とパラメータを最適化する．このように，
最適化したい評価関数と直接的に関連する必要がないため，直接的
な教師信号がなくてもニューラルネットワークやファジィルールを
学習させることができる．この特徴を生かして，GAを用いて強化
学習をさせることが可能である ．
