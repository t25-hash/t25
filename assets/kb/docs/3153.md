a.一般設計学における設計過程のモデル 一般設計学は，理
想的知識におけるプロセスとそうでない知識（現実的知識と呼ばれ
ている）におけるプロセスに分けて考えることから始まる ．
まず，理想的知識においては，仕様の詳細化のプロセスに対応し
て実体概念の（元の）順序列が存在し，それは一点に収束すること
が知られている．したがって，その場合には設計プロセスはこの収
束のプロセスととらえられる．
ところが，現実の設計者の設計プロセスは，理想性のない条件下
で行われる．このような現実的設計プロセスを考えるには，二つの
方法があるとされている．その第一は，まず理想的プロセスにおけ
る収束政策のモデルを確定し，次にその近似として現実的プロセス
のモデルをたてるものである．第二は，理想的プロセスにおける収
束政策とは独立に，現実的知識（real knowledge）のもつ特徴を
理想的知識の静的構造との対比においてとらえ，その特徴から直
接，現実的知識における設計プロセス，すなわち収束政策を導こう
とするものである．第二の方法については，さらに具体的に考察が
進められている．まず，現実的知識の欠陥については次のようにま
とめられている ．
(１)実体記憶に関する不完全性：有限個の実体記憶しかないの
で，実体概念を含まない抽象概念による部分集合が生じる．
(２)実体分類の不完全性：位相の欠陥である．異なる実体概念が
うまく分離できないこと，あるいは全く近傍をもたない実体
概念，すなわち全く抽象概念とは独立なものが存在する可能
性がある．また，間違った抽象概念によって分類されている
実体概念もある．
(３)操作上の不完全性：詳細化によって不可能な設計仕様を作っ
てしまったり，収束しなかったりする．また，仮に縮小列と
なっても，収束に無限時間がかかることもある．
(４)分析の不完全性：たとえ属性位相における部分集合を指定し
得ても，近傍に関する記憶が不完全であるために設計解が正
確に記述できない．
このように現実的知識に理想的知識からのずれがあるとすると，
具体的には，現実的知識の欠陥を補うという意味において設計プロ
セスが必要になるはずである．前述の各知識の欠陥を補うために下
記のプロセスが必要となる．
(１)実体記憶に関する不完全性を補償するためには，既存の実体
を収集することや，いままでは存在が想定されていなかった
新たな実体について，その存在可能性を予測し評価する必要
がある．それを行うプロセスにおいて，過去の設計物を参照
するプロセスや，新たな人工物を創案しその製造可能性を評
価するプロセスが発生する．
(２)実体分類の不完全性を補償するための方法としては，既知の
機能概念や物理法則を適用すること，新たな機能概念を創出
すること，さらには物理法則を発見することなどが考えられ
る．そこには，既存の人工物の仕様やメカニズムを理解する
プロセスや，新たな仕様やメカニズムを想定するプロセスが
発生する．
(３)操作上の不完全性を補償するためには，実現可能な操作手順
を案出する必要がある．そのためには，過去に行われて成功
した設計プロセスを参考にしながら実現可能な操作プロセス
を考案することなどが考えられる．
(４)分析の不完全性を補償するためには，形状や製造に関する情
報を追加する必要がある．したがって，既知の製造知識を収
集することや，新たな製造方法を開発することなどが考えら
れる．
