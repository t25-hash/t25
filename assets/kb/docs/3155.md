# 5・4・1 熱的諸量

> **章**: 第5章 各種変量の測定
> **節**: 5・4 熱的諸量、湿度の測定

a.温度計 熱的諸量の測定は温度計で代表され，工業用温度
計では接触式温度計の中でも熱電対温度計と抵抗温度計が大勢を占
めている．熱電対，測温抵抗体は20世紀初めに実用化されて以来，
多くの研究成果を得て技術的には成熟し，その集大成として規格で
標準化されている．温度センサ関連規格は，国内はJIS（日本工業
規格），国際的にはIEC（International Electrotechnical Commis-
sion，国際電気標準会議）によって規定されているが，現行は両規
格の整合化が図られている．
規格の世界的統一への動きは，1980年に規格・基準認証制度に
完全一致が図られた．
関する規律が国際的に初めて合意されたことに始まる．さらに，
1995年1月に発効したWTO/TBT協定（貿易の技術的障害に関
記号 構成材料

.熱電対 熱電対は，二つの異種金属を組合わせて構成す
る温度センサなので，構造が簡単なこと，素材の選択で極低温度域
から高温度域まで広い温度範囲で使用できるなどの特徴があり，産
業界では永年にわたり最も多く使用されている．
熱電対がわが国に導入されたのは1918年の国産化に始まる．温
度センサ関係JISは1954年にJIS C1602が補償導線を含めて新
しく制定された．その後，適時に技術水準に則したものに改正が繰
返されて，現行JISの1995年版は1990年国際温度目盛に対応し，
併せて国際整合化が強く求められている状況に鑑みIEC規格との
JIS C1602：1995に規定された種類の記号，構成材料，許容差
