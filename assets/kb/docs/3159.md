# 5・5・3 弾性係数

> **章**: 第5章 各種変量の測定
> **節**: 5・5 各種物性値の測定

a. 弾性係数外力を受けた弾性体の応力とひずみは、弾性の範囲内ではフックの法則に従い正比例する。このときの比例定数が弾性係数であり、材料が受けた引張り・圧縮・せん断・曲げなどの応力を、材料のひずみで除した値で定義される。したがって、弾性係数の大きい材料ほど荷重に対する変形が小さい。身近な例として、ばねに加えた力と伸びの関係を結びつけるばね定数を思い浮かべると理解しやすい。このように、弾性係数は固体の応力-ひずみ関係を記述する基本的な力学特性であり、構造材料の利用、設計に不可欠なパラメータとなっている。固体材料では最大21の独立な弾性係数が考えられるが、実際には材料の対称性により数は少なくなり、力学的に等方体とみなせるガラスや多結晶金属の場合、独立な弾性係数は二つである。単結晶材料を取扱う場合を除いて、多くの工業材料は等方体として近似され、その変形挙動は二つの弾性係数により記述されることが多い。通常、材料利用の形態に応じて、ヤング率(Young’s modulus)・ポアソン比(Poisson’s ratio)・剛性率・体積弾性率の四つから適宜選択して材料の特性を示す。ヤング率は、例えば軸方向だけの単軸応力状態で、応力σとそれによって生じるひずみεの比で示される。このとき、軸と垂直な,軸方向にはε=ε=-νε(5・40)図5・146 引張り、曲げによる測定法引張り・曲げによる測定では、試料に荷重を加え、それによる試験片ひずみや変位を測定してヤング率やポアソン比を測定する。この方法は比較的簡単であるため材料試験機を用いて広く利用され、荷重測定にロードセル、また伸び計には抵抗線ひずみ計や差動トランス等が使用される。零荷重近傍でのロードセルや試料保持の不安定さを避けるため、初めに初荷重をかけ、次いで荷重を増加させて伸びを測定する。曲げによる測定では、荷重による試験片のたわみ変位、あるいはたわみ角からヤング率を求める。三点曲げ試験を利用する場合、試験片を両端で支え、中央の1箇所に集中荷重を加えて試験片を曲げる。このとき、試験片中央部のひずみやたわみ変位、試験片両端のたわみ角を測定してヤング率を求める。三点曲げは簡易であるが、中央荷重の接触点で曲げ応力のほか、接触応力やせん断応力が複雑な三次元分布となることに留意すべきである。四点曲げ試験では、左右両支点から等距離のところに同じ大きさの二つの集中荷重を加えるため、二つの荷重点より内側に標点をとれば、標点距離上では一様な曲げモーメントを得ることができ、測定精度が高くなる。引張りや曲げによってヤング率やポアソン比を求める方法は、簡単な応力-ひず表5・33
