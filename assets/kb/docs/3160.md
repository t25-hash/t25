a. 熱物性値の単位と標準熱はエネルギーの移動形態の一つであり、これを制御する伝熱、断熱、蓄熱、熱交換などの技術やエネルギー変換技術は幅広い学術産業分野で利用される基盤技術である。これらの技術を駆使する熱工学システムの効率的な設計や安全性を評価するためのシミュレーションに欠かせないのが、熱物性値である。本項では熱物性量として比熱容量、熱伝導率、熱拡散率、熱膨張率、放射率を取上げる。熱量の単位J(ジュール)には、電気エネルギーや力学的な仕事との等価性から、1J=1W・s=1N・m(5・47)の基本的な関係がある。熱物性値の単位の多くは、温度(K)、長さ(m)、エネルギー(J)、時間(s)、質量(kg)などの複数の基本単位から構成される組立単位であり、比熱容量[J/(kg・K)]、熱伝導率[W/(m・K)]、熱拡散率(m/s)、熱膨張率(K)、放射率(無次元)となる。熱物性量の標準では、不確かさの小さな絶対測定装置で評価した標準物質(certified reference materials)をユーザの使用する実用測定装置の校正用に供給することが一般的である。代表的な固体の熱物性標準物質を表5・34にまとめる。b. 比熱容量の測定法物質の温度を単位温度(1K または1°C)だけ上昇させるのに要する熱量が熱容量(heat capac-ity)であり、これを単位量(1kg method)は最も標準的な測定法であり、多くの測定がこの方法で行われている。一方、組込黒体法(integral blackbody method)では、等温条件を簡便に確保するために、参照黒体を試料内部に組込むことを特徴とする。図5・154(b)の例では試料標的のすぐ近くに小さな穴を掘り、これを参照黒体として利用している。他の放射率測定法としては反射鏡法や試料移動法などがある。熱放射が微弱な常温やそれ以下の温度で有効な手法が反射率測定法(reflectivity method)であり、物体の反射率の測定からキルヒホッフの法則を用いて放射率を決定する。半球-方向反射率または方向-半球放射率を測定し、不透明体の場合には1から上記の反射率を引いて放射率が求まる。図5・154(c)に加熱空洞法による半球-方向反射率測定を模式的に示す。空間的に等方的な黒体放射が充満する加熱空洞(1000°C程度)に置かれた試料と標準試料がそれぞれに反射する放射輝度を比較測定する。試料自身の熱放射の影響を低減するためには、試料を裏から冷却するなどの工夫が必要である。他の反射測定法には積分鏡法や積分球法がある。熱量測定法(calorimetric
