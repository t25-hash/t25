# 5・5・4 熱物性値

> **章**: 第5章 各種変量の測定
> **節**: 5・5 各種物性値の測定

a.熱物性値の単位と標準 熱はエネルギーの移動形態の一つ
であり，これを制御する伝熱，断熱，蓄熱，熱交換などの技術やエ
ネルギー変換技術は幅広い学術産業分野で利用される基盤技術であ
る．これらの技術を駆使する熱工学システムの効率的な設計や安全
性を評価するためのシミュレーションに欠かせないのが，熱物性値
である．本項では熱物性量として比熱容量，熱伝導率，熱拡散率，
熱膨張率，放射率を取上げる．

b.比熱容量の測定法 物質の温度を単位温度（1Kま
たは1℃）だけ上昇させるのに要する熱量が熱容量（heat capac-
ity）であり，これを単位量（1kgまたは1mol）当たりに換算し
たのが比熱容量である．一般に純物質であれば，物質に固有の値を
示すために比熱容量（specific heat capacity）と呼ばれる．ただし
測定条件により値は異なり，例えば圧力一定で測定した定圧比熱容
量 と，体積一定で測定した定積比熱容量 を区別する必要が
ある．
