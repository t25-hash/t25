# 5・6・3 環境関連

> **章**: 第5章 各種変量の測定

環境の測定は、環境保全上の支障をきたす大気の汚染、水質の汚濁、土壌の汚染、悪臭などについて、人の健康を守るうえでそれぞれの汚染物質を監視するために必要となっている。環境の測定は、一般に極微量の汚染成分を検出・測定するため、高感度な機能を有するものである。建築物内の空気環境については、事務所ビルや住宅内で起こるシックビル症候群、シックハウス症候群あるいは化学物質過敏症などの人体に影響を及ぼす汚染物質や、半導体・液晶を製造するクリーンルーム内で製品の信頼性悪化を引き起こす分子状汚染物質(AMC: airborne molecular contaminant)についてもさらに低い濃度で制御することが重要となり、その環境計測にも高い信頼性が必要となってきている。a. 水質の測定方法水質に関する環境基準には、水道水・環境・地下水・排水に関する基準がある。それぞれの基準成分と基準値、測定方法について表5・36に示す。水道水質基準は、人の飲用に適する水の水質について、病原生物の汚染と化学物質による汚濁、色、臭気、味や濁りに関する項目を定めたもので、表5・36のほかに細菌、pH 値、味、臭気、色度、濁度などがある。環境基準は、河川や海域の水質について、人の健康を保護し生活環境を守るうえで維持されることが望ましい項目を示している。その中で、生活環境の保全に関する環境基準として、河川および海域についても水素イオン濃度(pH)、生物化学的酸素要求量(BOD : biochemi-cal oxygen demand)、浮遊物質量(SS : suspended solid)、溶存酸素量(DO: dissolved oxygen)、大腸菌群数、-ヘキサン抽出物質(油分等)が基準項目として含まれている。地下水基準は、地下水の水質汚濁に関わる環境上の条件について、人の健康を保護するうえで維持することが望ましい項目について示している。排水基準は、特定施設を設置する工場または事業場から公共用水域に排出される汚水または廃液等の規制および地下浸透の規制を行い、公共用水域および地下水の水質汚濁を防止する目的で定められた項目を示したものである。水を汚濁する成分の種類は非常に多く、JIS K 0102:1998、「工場排水試験方法」およびJIS K 0125:1995、「用水・排水中の揮発性有機化合物試験方法」に試験項目が記載されている。多成分を高感度に、また同時に分析を行うためには、前処理等を必要とする分析機器を用いた分析となる。金属成分の分析には多元素同時分析が可能なICP(inductively coupled plasma、誘導結合プラズマ)発光分析法(ICP-AES : ICP-atomic emission spectrometry)、ICP質量分析法(ICP-MS : ICP-mass spectrometry)を用い、有機成分の分析にも同様に多成分を同時に分析可能なガスクロマトグラフ質量分析法(GC/MS : gas chromatograph/mass spectrom-etry)、またはガスクロマトグラフ(GC)に検出器としての電子捕獲検出器(ECD : electron capture detector)をもった分析法が用いられる。また前処理として、パージ・トラップ(P& T : purge andtrap)法とヘッドスペース(HS : head space)法がある。P& T法は、試料水中に不活性ガスを導入(パージ)し、揮発成分を気相中に移行させ、その空気を分析機で測定するものである。また、HS 法は静的な気液平衡を利用したもので、試料水を入れた密閉容器を加熱することで水中の揮発成分を気相中に移行させ、その空気を測定するものである。そのほかに、環境水中の内分泌かく乱化学物質(環境ホルモン)が近年問題となっている。環境ホルモンの実態調査は水質、水道、下水道に限らず、底質、土壌、大気、水生生物、野生生物、器具・容器包装、食品、生体など多岐にわたって行われている。測定方法は、GC/MS による同時一斉分析が基本であり、検出限界は河川水の場合10pptとされるため、前処理として1000倍程度の濃縮処理を行わなければならないなど、高度な前処理技術が必要とされる。b. 大気の測定方法大気汚染物質(atmospheric
