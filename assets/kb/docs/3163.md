ずしも一致するとは限らないためのみならず，人間がデータを追加

に高速な完全探索を実現したり，最良優先探索で近似的に十分な解
を得たりする方法が提案されている．また対象グラフも連結グラフ
とは限らず，飛び地があるような非連結グラフにまで一般化した研
究や，逆に閉路を含まない木構造グラフの部分構造導出の研究も行
われ，実用レベルの性能が得られている．構造データマイニング
は，元データを前処理で近似的に表形式に押し込む必要がないた
め，結果の解釈容易性が高く，知識発見に向いていると考えられる．
験的に得られるデータに潜み，かつわれわれにとって何らかの意味
を有する規則性の中で，いくつかの厳しい条件を満たすものを法則
という．法則式発見で重要な点は，対象分野やその観測過程に関す
る先験的知識に基づいて，対象を支配する原理が観測数量間に課す
制約と同型ないしは準同型である関係式を探索することである．こ
沿って，さまざまな試みがなされている．
れは，従来のシステム同定や統計的手法が観測数量間の漸近式を見
出そうとすることと異なる点である．現状において最も一般性をも
つ法則発見手法の一つが，各観測数量の尺度に関する知識を用いる
ものである ．大半の観測数量は比例尺度と間隔尺度に分類され
る．比例尺度は質量や距離などのように，絶対的原点をもち，その
比に意味がある数量である．一方，間隔尺度は温度の摂氏や華氏な
どのように，原点が人為的に決められ，値の差の比にしか意味のな
い数量である．これら2種類の尺度を含む法則式形式には，一定の
数学的許容性が存在する．例えば，測定数量 と が両方とも比
例尺度であり，仮にその関係が対数関数＝logであるとする．
にある正数 を掛け単位を変更すると，＝log＋logとな
り，log分だけ の原点が移動してしまう．これは が絶対的原
点をもつ比例尺度であることと矛盾してしまう．したがって， と
の関数関係は対数であってはならない．詳細は省略するが，二つ
の比例尺度間に許容される式は＝しかない．これは一辺の長
さが の体積 が＝であることや，距離 の二つの天体間に
働く万有引力 が＝ （ただし は両天体の質量に依存）で
あることに反映されている．このような制約を一般化すると，比例
尺度や間隔尺度で構成される測定データ間に許容される関数系が得
られる．それをもとに先見的に関数系を限定して，漸近式ではない
法則式の候補をデータから導出することは可能である．観測数量に
課されている制約に従う式は，その背後の物理的原理と矛盾しない
可能性が高く，予測精度を多少犠牲にしても人間の物理的解釈が得
やすい．科学的法則発見は知識発見の典型例である．
また，紙数の都合上詳述は避けるが，最適予測問題の解法とし
て，複数の分類学習器を並列に組合わせて単一の分類学習器よりも
高い予測精度を得ようとするbaggingやboostingなどのアンサン
ブル学習（ensemble learning) ，実属性空間のデータを関数に
よって高次元ないしは無限次元のヒルベルト空間に写像し，そのう
えで1枚の判別超平面で2値クラス予測を行うサポートベクタマシ
ン（SVM:support vector machine ) などの研究も盛んになっ
ている．さらに関連して，高次元空間を張る属性どうしにラティス
や木などの幾何学的構造関係を与えて判別距離を計算する，カーネ
ル関数法の研究も進みつつある．次元間に先見的構造関係を導入す
ることで，分類基準の説明能力を高めることができれば，知識発見
にもつながる可能性があり，さまざまな制約を取り入れたカーネル
関数の開発が期待される．
d.設計知識の高度処理への適用可能性 設計分野へのデータ
マイニングや知識発見の適用は，インタネットのウェブサービス設
計や医薬品設計において進められている．ウェブは多数のホームペ
ージ（URL:uniform resource locator ）が互いをリンクによって
グラフ構造のマイニングについても，たくさんの構造とその構造
指し示すネットワーク構造をしている．ウェブページの閲覧では，
このネットワーク上を閲覧者が逐次移動する．ある閲覧者の閲覧履
歴は，ウェブサーバに記録された一つのIP（internet protocol ）
アドレスからのウェブネットワークアクセス履歴で近似できるの
で，ウェブサーバログからそのサイトを訪れた多数の閲覧者の閲覧
履歴グラフを取り出すことができる．このデータに前述した構造デ
ータマイニング手法を適用して，ある内容のウェブページをある順
序で見る閲覧者は，ほかにどのような内容のページをどのような順
序で頻繁に見るか，という知識を発掘することができる ．この
ような知識を閲覧者の興味に応じた広告バナー配置やショートカッ
トリンクの付加などの形で，ウェブサービスの設計に反映させるこ
とが可能である．一方，化学分子構造中のいかなる部分構造や物性
が，人体にどのような影響を及ぼすかに関する知識が利用可能であ
.科学的法則式発見（scientific equation discovery) 経
れば，それらの構造や物性を適切に組合わせた分子を有する医薬品
を設計することが可能になる．そのために，さまざまな生理活性を
有する化学分子構造やその物性に関するデータを集め，それに構造
データマイニングや科学的法則式発見の技術を適用して，必要な設
計知識を得ることが考えられる ．現在，このような考え方に
現状では，機械工学分野の設計へのデータマイニングや知識発見
の適用事例はあまり見られない．この一つの理由は，機械工学分野
においては，設計者が膨大で複雑な生データをもとにして作業を進
めなければならない場面が，比較的少ないためと考えられる．多く
の場合には，設計に必要な機構に関する知識は設計者が有してお
り，また材料や部品に関してはデータベースとして整理されている
ことが多い．このような状況では，非均質で膨大なデータを比較的
均質なデータのセグメントに分割する機能とセグメントごとの最適
予測か知識発見を行う手法であるデータマイニングは，あまり適用
される機会がないと思われる．機械工学分野の設計にデータマイニ
ングが広く使われるようになるとすれば，それは膨大で詳細な設計
事例データベース（design example data base ）が整備されるよう
な状況であると思われる．現状においても，特許データベース等で
の設計事例検索は利用されているが，機構や部品，材料などの細か
い粒度の生情報が事例間で比較可能な詳細さで提供されているわけ
ではない．このような詳細な設計事例データベースが提供されれ
ば，設計目的や部分的機能が共通な設計事例群をそれらの内容によ
ってセグメント化し，各セグメントがもつ共通な特徴を設計知識と
して抽出する目的でデータマイニングを用いることが可能になると
期待される．機械工学分野での設計知識高度化へのデータマイニン
グや知識発見の適用方法に関する研究が待たれるところである．
