# 5・1・1 リンク機構（linkage）の構成

> **章**: 第5章 運動変換要素
> **節**: 5・1 リンク機構

機械を複数の物体の運動系とみると，機械は複数の物体が互いに
接触し，拘束を受ける状態で組立てられたものとモデル化できる．
このモデルを機構という．このとき物体を節（link），二つの節の
それぞれの接触部分を対偶素（pairing element ）と呼び，二つの
物体の対偶素の組合せを対偶（kinematic pair ）という．対偶には
二つの物体を点接触や線接触で拘束する高次対偶と，面接触で拘束
する低次対偶がある．節を対偶により連結した運動系を連鎖
（kinematic chain）といい，連鎖の一つの節を基準とし，それに対
してほかの節が相対運動を行うとき，その連鎖を機構（mecha-
nism）という．機構は用いられている対偶の種類によりいろいろ
な名称が付けられているが，一般に低次対偶から構成される機構を
リンク機構（linkage）という．高次対偶からなる機構にはカム機
構，歯車機構などがある．

連鎖は，節が閉回路を成しているか否かにより，閉ループ連鎖と
開ループ連鎖に分類される．開ループ連鎖の中で，節が分岐せずに

ループ連鎖で構成されたものが中心であったが，最近はロボット機
