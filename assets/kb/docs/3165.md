# 5・6・5 感応量(psychophysical response value)

> **章**: 第5章 各種変量の測定

a. はじめに感覚量(psychophysical sensory intensity)の計測には基本的に二つの方向がある。一つは人間や生物の感覚の生5―114計測工学成量(脳への入力量、狭義の感覚量、心理量)を客観的、数量的に表すこと、もう一つは人間や生物の感覚と同様な生成量を人工的に作り出すメカニズムをセンサ(sensor)として実現することである。前者においてまず必要なのは、感覚生成の背景にある基本的な法則あるいは数理的なモデルを見いだすことであり、これがある程度確立すると、次の段階としてそのモデルに従った心理実験の方法を考案し、数多くの実験によってモデルの妥当性を検証するとともに、実験の結果を具体的な感覚尺度として構成することである。このような方向の初期の段階では、得られるものは感覚の生成量の外面的なモデルにとどまるが、背景となるメカニズムの解明が進むにつれて人間や生物の感覚の実質的なモデルに成長することになる。ここまで到達すれば感応量の人工的なセンサといってもよい。後者の方向は基本的にボトムアップ的であり、より直接的に、より工学的に、感応量を代替できる出力をもつセンサの開発と応用を目指す。理工学的アプローチの特色として、その物理的メカニズムの観察から数理的なモデルを推察し、仮定されたモデルに従い既存の計測技術上で成功を収めた方法論を大きなよりどころとして、当面の需要を満たすデバイスを生み出そうとする。このような方向の初期の段階の成果は、構成も単純で適用範囲も限定的であるが、背景となるメカニズムの解明が進むにつれて大変優れたセンサに成長し、同時に工学技術として一般化されて他分野のセンサの開発にも大きな波及効果をもつことになる。b. 感応量の基礎と基本法則感覚器に物理的な刺激を与えたときに、人間はそれに気づき何らかの反応を示す。その反応の強さは物理的な刺激を強めると当然のように強まる。物理的な刺激をさらに強めると、感覚としての生成は飽和したり、失われたり、感覚器の損傷を生じたりする。最小の物理的な刺激の強さを刺激閾値(stimulus value)、最大の物理的な刺激の強さを刺激頂(terminalstimulus)と呼ぶ。これらの間の刺激の強さでは、その大小の比較能力の把握とモデル化が主要課題である。二つの刺激を提示したときに、それらが異なる強さとして知覚される限界のことを弁別閾値(just noticeable difference)と呼ぶ。これ以下の差異は人間には知覚されないという意味で、二つは等しい。これを主観的等号(subjective equality)と呼ぶ。物理的には全く異なる刺激が主観的等号で結ばれることがある。この背景には感応量生成の特定のメカニズムがあるため、これらを解明するための大変重要な手段に関係する概念である。物理的刺激と人間の感応量との間には、一般的に成り立つ法則がいくつか知られている。ウェーバーの法則(Weber’s law)は、弁別閾を与える刺激の強さを変えて測定したときの一般的な傾向を表したもので、弁別閾と刺激の強さには比例関係があることを主張する。例えば、触覚の同時2点刺激の距離の判別では、2点間の距離が離れるほど距離の違いが判別しにくくなる。フェヒナーの法則(Fechner’s law)はこれを数式で表現したもので、感応量は刺激強度の対数に比例することを主張する。また、感応量はこのときの比例係数をべきとする刺激量のべき乗に比例するともいえる。これをスティーブンスの法則(Stevens’law)と呼ぶ。しかし、これらは必ずしも広範囲および一般的に成り立つわけではない。実際上、このような刺激強度と感応量との関係を求めることを感覚尺度の構成と呼ぶ。感覚尺度は、物理的刺激を感応量に写像する非線形関数と座標軸を与え、その座標軸の尺度はその強さの刺激での人間の弁別閾に等しくなるようにとられる。よって、このような座標系にとられた二つの刺激は、人間に等しい大きさの違いとして知覚される。c. sensor)ここからは、感覚のメカニズムを模倣したセンサについて述べる。センサの開発のために生物を模倣することは、生物の感覚器や感覚情報処理系の中に、センサを囲む物理場の構造、フロントエンドの回路構造、情報選択や統合のための信号処理構造などの独特な仕組みを見いだすことである。それらを数理的にモデル化し、工学的手段で実現可能な形に再設計することにより、従来にない高性能なデバイスや回路、アルゴリズムの実現が可能になる。.
