# 6・1・1 はじめに

> **章**: 第6章 各種応用計測
> **節**: 6・1 電気計測器

電気計測器(measuring instrument)は産業のマザーツールとして、数多くの機種が多方面で活躍している。本章ではすべてを網羅することができないので、機械工学分野において使用されている代表的な機種として次の製品を紹介する。(1) 波形測定器(2) 電力計測器(3) データ収集装置(4) 現場用計測器最近の多くの機械は電子回路によって制御されているので、機械を評価するときには必ず電子信号を見る必要がある。電子制御された機械には、動作を確認するための各種センサと機械の状態を変化させるためのアクチュエータが取付けられている。多くのセンサは直接もしくは間接的に電気信号に変換されて、電子回路で構成された制御装置に入力される。また制御回路には一般にマイクロプロセッサが搭載されており、あらかじめ組込まれたプログラムに従って制御信号や表示情報などを出力する。制御情報は電気信号であり、何らかの仕組みを介して機械の状態を変えるアクチュエータに伝えられる。一般に電子化された制御装置を備える機械は複雑な動作が可能で、機械のもつ本来の特性を最大限に引出すことができる。電子部品だけで構成された機器、例えばテレビや携帯電話などはつねに安定した動作を長期にわたって保証されており、シミュレーションを行った結果と実際の回路はほぼ同じ動作をする。したがって大規模な集積回路であっても、シミュレーションを行うだけで十分な品質が確保できる。一方、機械の位置決め誤差を測定する要因は非常に多くあり、例えば長期間機械を使ったときは摩耗、腐食などが発生し、機械の動作に変化が生じる。したがって、いろいろな環境で長期にわたって機械の安定した動作を保証するために、機械に対し多岐にわたるケースを想定した評価が必要になってくる。機械を評価するための電子測定器ではさまざまな物理量を測定する必要がある。おもなものを挙げると、温度、振動、音、圧力、ひずみ、変位、時間、電圧、電流、電力など広範囲にわたる。最近の電子化された自動車を見れば、多くのセンサが各所に取り付けられて複雑な制御を行い、最適な動作を保証している。機械は電子回路に比較すると一般にゆっくりした動きをする。機械そのものを評価するにはマイクロ秒の単位があれば十分であるが、機械を制御している電子回路は年々マイクロプロセッサが高速化してナノ秒単位の時間で動作している。すなわち機械を取り巻く状態をすべて評価するためには、時間や分の単位からナノ秒の単位まで時間を取扱う必要がある。また機械に取付けられたセンサからはマイクロボルト単位の微小な信号が出力されることがある。一方、機械を駆動するためには数百A の電流が必要な場合もある。このように、機械を取り巻く電気信号にも大きな幅がある。電子化された機械を評価するにはこのような特徴をもっていることを理解する必要がある。また、日本は自動車、白物家庭用電化製品やOA(office automation)機器など電子化された機械を得意とする国であるため、日本の電子測定器メーカには電子化された機械の評価を得意としているところが多い。
