# 6・2・1 光干渉計(optical interferometer)による変位計測

> **章**: 第6章 各種応用計測
> **節**: 6・2 光応用計測

(displacement measurement)a. 単一波長光を用いた変位計測干渉計図6・23のようなレーザ光を用いたマイケルソン型光干渉計によって,光の波長オーダで変位の計測ができる.波長λの単色レーザビームを半透鏡上の点Oで強度の等しい二つのビームに分割し,一方を固定鏡上の点R に,他方を可動鏡上の点M に照射し,両者からの反射光を再び半透鏡で重ね合わせ,光検出器でその強度を測る.可動鏡が変位する前は距離OR とOM が等しい(光路差なし)とし,そこから可動鏡が光軸に垂直に距離だけ変位したとき,光路差は2となるから,光検出器で検出される光強度は∝cos2πλである.を整数として,=λ2のときは明るく,=+12λ2のときは暗くなる.変位の間,明暗のしまが何回繰返されるかをカウントすれば変位の距離が求まる.最近,小型・堅牢で高出力(数百mW),しかも可干渉距離が長くて,可視光(緑色)を連続発振する半導体励起固体レーザが市販され,普及するようになった.実際にこのレーザを用いて実験室でマイケルソン型干渉計を組み,可動鏡を数m変位させてもコントラストの良い干渉じまが肉眼で観測できる.このように,高精度で測定範囲が広い変位計測が実験室で容易に実現できるようになっている.b. 二波長光(two-wavelength light)を用いた変位計測干渉計上記のマイケルソン型干渉計において,周波数のわずかに異なった二つの波長の光源(ゼーマン効果によって周波数差約1.8MHzの互いに直交偏光のHe-Neレーザ光)に替えると,ヘテロダイン干渉が起こり,光のビート信号が観測される.この状態から可動鏡(回転誤差の影響を受けないように実際にはコーナキューブが用いられる)が速度で移動すると,ドップラー効果(Dopplereffect)により周波数変位=2が観測される.ここで,は光の速度である.これから可動鏡の速度がわかり,要した時間について積分すれば変位距離が求まる.この方法はビート強度の交流成分のみを用いるため外乱の影響を受けにくいとされているが,可動鏡の最高速度が制限される.本システムは高精度測長システムとして計測器メーカによって市販されている.c. 位相共役(phase conjugate)光を用いた変位計測干渉計上記のようなマイケルソン型干渉計は,干渉計の二つの光路が空間的に分離しているため,空気の揺らぎの影響を受けやすいが,位相共役を利用した干渉計によれば,空気の揺らぎや光学部品の不均一さによる位相ひずみを補うことができる.図6・23におけるマイケルソン型干渉計の固定鏡および可動鏡を,それぞれチタン酸バリウム単結晶などの自己ポンプ型位相共役結晶で置換える.すると,位相共役効果によって二つの光路の空気揺らぎによる屈折率分布が補正される.しかしながら光路差の変化は検出されるので,マイケ図6・23 マイケルソン型干渉計ルソン型干渉計と同じ原理で変位量は計測される.
