# 6・2・3 光干渉計による形状計測

> **章**: 第6章 各種応用計測
> **節**: 6・2 光応用計測

a. 位相シフト(phase shift)高精度フィゾー型干渉計現在、わが国やアメリカで製造され、市販されている高精度干渉計システムの大部分がフィゾー型であり、おもに光学部品の表面形状誤差や収差の分布などが計測される。図6・26に曲面(凹面)鏡および平面鏡の形状誤差を計測する場合の光学系を示す。干渉計本体の光学系(図6・26の左ブロック)では、単色レーザビームをビームエキスパンダとコリメートレンズによって平行光とし、裏面が部分反射する基準面をもった参照鏡に照射する。参照鏡を透過した波面は、曲面鏡計測の場合(図6・26の右上ブロック)はナル(null)レンズを通って測定対象である曲面鏡にて反射され(測定波面)、元来た光路を戻ってビームスプリッタで反射され、測定面にフォーカスした観測レンズを通してCCD(charge coupled device)などの受光素子に到達する。平面鏡計測の場合(図6・26の右下ブロック)は、そのまま測定平面鏡から反射された波面が測定波面となる。一方、参照鏡の裏面によって反射され、測定鏡面を経ずに元来た光路を戻ってきた波面(参照波面)は、同じくビームスプリッタで反射され、測定波面と干渉して受光素子に到達する。フィゾー型干渉計が市販され、普及しているおもな理由は、基準面と測定鏡面が一つの光軸上にあり、マイケルソン型に比べてコンパクトな光学系になっているためである。高精度干渉計システムでは干渉じまの解析精度を上げるため、位相シフト(しま走査)法が導入されている。位相シフト法は通常、参照鏡に装着されたピエゾ素子によって基準面の位置を光軸に垂直方向にわずかにずらしながら、干渉じま画像を複数取込んで処理することによって行われる。それによってさまざまな誤差要因を排除できる。平面鏡計測の場合に多重反射によって生じる干渉じま強度の正弦波からのひずみや、CCD 受光素子感度の非線形性を補正するアルゴリズム、位相シフトを行う位相変調器における非線形性による変調誤差を補正するアルゴリズム、干渉計の光路長が2Hz以下程度の機械的振動を受ける場合の補正アルゴリズム、そのほか、さまざまな誤差要因を補正するアルゴリズムが5―127提案されており、精度の向上が図られている。b. シヤー干渉計(shear interferometer)一般にマイケルソン型のように測定光波面と参照光波面が空間的に別光路をとる干渉計は、空気の揺らぎや外部振動などの外乱によって精度が低下する、光学系が大きくかさばる、光学系の調整が面倒であるなどの問題点がある。それに対して平面鏡測定の場合のフィゾー型干渉計やここで述べるシヤー干渉計は、測定光波面と参照光波面がほぼ共通の光路となるため、外乱に強い、光学系がコンパクトなどの利点がある(共通光路干渉計)。共通光路干渉計の一つであるシヤー干渉計は、光波面そのものの形状が測定できる(通常のマイケルソン型干渉計では光波面形状は直接検出できない)。ただし、得られる干渉じまパターンは光波面の差分(もしくは微分)を表し、マイケルソン型やフィゾー型のように直接光波面の等高マップを表さないので、目視での光波面誤差の定量的評価はしにくいが、外乱に強いと同時に光学系の調整が不要であるという大きな利点がある。図形状を計測する方法を示している。光源からの光をコリメートレンズで平行光とし、大きな面積をもった計測面からの反射波面を、光軸に対して45°の角度に置かれた平行平面板で反射させる。このとき、平行平面板の表面と裏面から反射された二つの光波面は互いに横方向にずれて干渉する。光波面の形状(すなわち測定面の形状)が平面の場合は干渉じまが生じないが、形状が曲面の場合にはその差分が干渉じまとして観測され、しまを解析することによって形状が求まる。大きな面積の計測の場合は、測定面(またはシヤー干渉計)を直進ステージ等によって横に移動させて逐次計測していく。この場合、直進にわずかな真直度誤差(並進および回転角度誤差)があっても、測定精度に影響しないという大きな利点がある。c.
