# 6・2・4 ホログラフィーおよびスペックル干渉による変形計測

> **章**: 第6章 各種応用計測
> **節**: 6・27 はシヤー干渉計の一種である横シヤー干渉計で、大きな曲面

ホログラフィー干渉法(holographic interferometry)やスペックル干渉法(speckle interferometry)によると、粗面(通常の干渉計は鏡面のみを対象)の変形分布や振幅分布を干渉計測できる。従来はホログラフィー干渉がよく用いられたが、最近はCCD カメラと画像メモリを用いて電子的に干渉計測のできるスペックル干渉法[ESPI(electronic speckle pattern interferometry)またはテレビホログラフィーと呼ばれる]が開発され、装置も市販されている。ここではESPIを用いて大きな変形を計測する方法を紹介する。大変形を求める手順は以下のとおりである。まず、図6・28のESPI計測システムによって変形前から変形後までのスペックル画像(,...,)を時系列で次々と画像メモリに蓄える。最後にそれらを加算することによって大変形を求める。実験では、光源はHe-Neレーザ(λ=632.8nm)であり、計測対象はマイクロメータヘッドで変形を与えた片持ばりとした。これは、ESPIによってのみ可能な方法で、ホログラムを作成する時間が有限である実時間ホログラフィー干渉法では不可能である。
