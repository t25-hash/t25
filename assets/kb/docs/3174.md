# 6・2・4 ホログラフィーおよびスペックル干渉による変形計測

> **章**: 第6章 各種応用計測
> **節**: 6・27 はシヤー干渉計の一種である横シヤー干渉計で,大きな曲面

ホログラフィー干渉法(holographic interferometry)やスペックル干渉法(speckle interferometry)によると,粗面(通常の干渉計は鏡面のみを対象)の変形分布や振幅分布を干渉計測できる.従来はホログラフィー干渉がよく用いられたが,最近はCCD カメラと画像メモリを用いて電子的に干渉計測のできるスペックル干渉法[ESPI(electronic speckle pattern interferometry)またはテレビホログラフィーと呼ばれる]が開発され,装置も市販されている.ここではESPIを用いて大きな変形を計測する方法を紹介する.大変形を求める手順は以下のとおりである.まず,図6・28のESPI計測システムによって変形前から変形後までのスペックル画像(,...,욈)を時系列で次々と画像メモリに蓄える.次に,それらの適当な二つの差욡욢=욢-욡をとってカウント可能なしま画像を出現させ,それを解析して部分的変形욡욢を求める.最後にそれらを加算することによって大変形を求める.実験では,光源はHe-Neレーザ(λ=632.8nm)であり,計測対象はマイクロメータヘッドで変形を与えた片持ばりとした.1フレーム当たり256×240×8ビットの画像を変形前から変形後まで10フレーム蓄積した.変形全体をとった場合の干渉じま=-は,図6・29のように片持ばりの先端付近の変形量が大きすぎて干渉じまが解析できない.そこで図6・30のように=-,=-,=-の解析可能なしまを出現させ,部分変形,,を求めてそれらを加え合わせれば,全変形が図6・31のように求まる(ここでは簡単のため片持ばりの中心を通って長手方向に沿った変形のみを求めた).これは,ESPIによってのみ可能な方法で,ホログラムを作成する時間が有限である実時間ホログラフィー干渉法では不可能である.
