# 5・3・3 間欠歯車装置（intermittent gear drive）

> **章**: 第5章 運動変換要素
> **節**: 5・3 間欠運動機構

図Ⅰ-5･30の間欠歯車装置は原動歯車bの歯の一部分を取り除い
て円弧に置き換えることによって，この部分では従動歯車dに運
動が伝達されないから間欠運動が可能となる．しかしながらこの間
欠歯車装置では高速度回転の場合慣性力によって停止時の振動を誘
発するおそれがある．図Ⅰ-5･31に示すピンを用いた間欠歯車のほ
うが滑らかな運動となり衝撃をやわらげる効果がある．
