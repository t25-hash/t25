# 6・3・2 工業用X 線CT

> **章**: 第6章 各種応用計測
> **節**: 6・3 放射線応用計測

a. 概要工業用X 線CT[X-ray CT (computed tomogra-phy)]は,物体内部の正確な三次元形状(three-dimensionalshape)を計算機内に取込み解析・表示する唯一の手段で,その利用価値は極めて高い.しかし計測対象がX 線を透過しにくい鉄などの高密度材料で,しかも高い空間分解能が必要なことなど技術的に難しく,その普及は医療用のCT スキャナに比較して20年以上も遅れている.近年,透過力の強い高エネルギーX 線を発生する電子線型加速器(electron linear accelerator,LINAC: linearaccelerator)をX 線源とし,その強い透過力をもつX 線を高感度で検出する小型のシリコン半導体検出器(Si-SSD : silicon solidstate detector)をX 線検出器とする工業用X 線CT が実用化したことで,その利用が広がり始めた.工業用X 線CT は医療用CT スキャナと原理は全く同じなので,本節ではその説明は省き,工業用X 線CT 固有の構造と性能についてのみを記す.b. 工業用X 線CTの構成図6・37に工業用X 線CT の構成を示す.X 線源と一次元のX 線検出器アレイを対向して配置し,その間に試料を走査するスキャナを設置する.試料を回転走査5―130計測工学図6・38 高エネルギーX 線対応シリコン半導体検出器アレイしながら透過減衰したX 線強度を測定する.c. X 線源例えば鉄に対するX 線の透過力は約9MeV で最大になる.したがって,厚い高密度材料が試料の場合はこのエネルギー領域のX 線が有効である.このレベルの高エネルギーX 線はLINACにより加速した電子線の制動ふく射によって発生させる.LINACによるX 線はエネルギーが高いばかりでなく強度も強いので,短い撮影時間で高い分解能の断層像が得られる.ただし鉄で数十mm,アルミニウムで200~300mmまでの試料であればLINACの代わりに数百kV のX 線管でもCT 撮影は可能である.d. X 線検出器アレイ状に配列したX 線検出器と検出器単体を図6・38に示す.焦点から出射して試料を透過したX 線は短冊形のシリコン半導体検出器の小口から長手方向に入射する.短冊形の検出器はすべてがX 線の焦点を向くように配列してある.この構成により,約40mmのシリコンを透過する間に数MeV の高エネルギーX 線光子の20%ないし30%が検出器と相互作用する.この相互作用の確率はX 線フィルムのおよそ3×10倍である.各検出器は互いにタングステン板で遮へいして,検出器相互間のクロストーク(ある検出器に入射したX 線が散乱して隣接する検出器で検出される現象で,画像劣化の原因になる)を防止する.e. 工業用X 線CTの性能工業用X 線CT の性能を決める重要な因子は,(1)空間分解能,(2)撮影時間,(3)鉄換算厚さの三つである.鉄換算厚さとは,どの程度まで厚く密度の高い試料の撮影ができるかを鉄の厚さで代表して示すものである.これらの性能因子には,試料透過後に検出されるX 線光子数が深く関わる.それはX 線光子数の統計的ばらつき(揺らぎ)が画像上の雑音として現れるためである.高密度で厚い試料ほど,高い分解能ほど,また撮影時間が短いほど,画像上の雑音は増加する.つまり三つの性能因子のいずれを向上させようとしても,画像上の雑音は増加する方向にある.この統計的ばらつきを低減するには,透過後に検出されるX 線光子数を増す以外に手段はない.単位時間当たりの検出光子数を増加させる手段には,X 線源の強度を増すこととX 線検出器の検出効率を増すことの二つがある.X 線CT 用の線源としてのLINACは,12MeV 程度を上限と考えるべきである.これ以上にエネルギーを高くしても透過力は頭打ちになるし,中性子発生による遮へいコストが増大するという問題も起きる.また検出器についても,その感度はあと数倍で100%に近づくことになり,それ以上の改善はあり得ない.したがって12MeV のLINACを使う場合,表6・4に掲げた性能の組合せでの撮影が高エネルギーX 線CT の技術的限界である.表6・4に「―」で示してある箇所は撮影時間が非現実的に長くなり,実用上撮影が不可能なことを示す.工業用X 線CT の主たる利用目的が物体内部の三次元形状計測であれば,一試料に対して数百枚から数千枚の断層撮影が必要であり,1枚当たりの撮影時間を短縮することは切実な問題である.鉄換算厚さが100mmの場合でも0.1mmの分解能での撮影は光子数が不足し,20s程度の撮影時間が必要である.とはいえ広い条件範囲で一断面当たりの撮影時間が10sで済み,また分解能をある程度犠牲にすれば500mmまでの鉄厚(アルミニウムならば1500mm程度まで)の撮影が短時間で可能である.なお,鉄厚が数mmまでの試料であれば,特殊なX 線管を使うことで数mの分解能で表6・4 工業用X 線CT の撮影時間(s/断面) と鉄換算厚さ・分解能の関係鉄換算厚さmm0.1――――0.2――0.5―分解能mm1.02.05.0(a) 実物(b) 透明レンダリング(c) サーフェスレンダリング(d) 切断レンダリング図6・39工業用X 線CT による水車模型の撮影例撮影できるマイクロフォーカスX 線CT も実用化している.f. 工業用X 線CTの応用工業用X 線CT は,物体の外表面ばかりでなく内部の複雑な形状を計測・表示できる三次元計測器として,3D-CAD(3D-computer aided design)図面とそれに基づく製品・試作品との照合や,製品から3D-CAD 図面の自動生成(リバースエンジニアリング),さらにはラピッドプロトタイピング(RP: rapid prototyping)の利用による現物からレプリカ作成など,機械の設計・製造プロセスを変えるほどのポテンシャルをもっている.図6・39は金属RPで作成した水車の模型を工業用X 線CT で撮影し,その外観形状・内部形状を表示した例である.これらの形状データと3D-CAD による設計図を重ね合わせて製作誤差を算出することも可能である.この技術は正確な立体形状を保存・表示する機能から古生物学,考古学,文化財保存学などへの利用も進みつつある.今後は計測された精密な立体像をうまく利用するためのソフトウェアの開発が重要になる.
