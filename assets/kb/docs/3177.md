# 5・3・4 カムによる間欠運動装置（cam-type intermittent

> **章**: 第5章 運動変換要素
> **節**: 5・3 間欠運動機構

所要の間欠運動を得るためにはカムに一時休止をもつ輪郭曲線をを伝達する場合には，高速度の作動に対して従動節が飛び上がってⅠ-5･32のようにカムbに溝を付けて二つの面bおよびbによっム溝のCCの区間がカムの回転中心Aを中心とする円周上にあ従動節は停止して間欠運動を行うことになる．
第Ⅰ部 機械要素 第5章 運動変換要素ても精度の高い間欠運動を得るためには従動節先端のころが接触す与えればよいが，従動節先端と一つのカム面との接触によって運動カム面より離れてしまい確実な間欠運動が望めなくなる．
そこで図て従動節dの先端のころCをはさみ込む必要がある．
この場合カればカムの回転角θの期間だけ従動節dの揺動角ϕは一定となり，る場合にエンドミルを使用すれば，カムの等速回転に対してカッタよびNの精度にむらができ，カム面ところの間に遊びができて，おそれがある．
溝幅の不同をなくすためには，溝の輪郭曲線に沿っい．
外接ゼネバ機構 内接ゼネバ機構することは容易なことではない．
