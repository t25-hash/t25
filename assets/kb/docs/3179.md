# 6・4・2 使用する周波数範囲

> **章**: 第6章 各種応用計測
> **節**: 6・4 超音波応用計測

図6・40のように,100Hz 程度からGHz 領域までの幅広い周波数が用途に応じて使い分けられている.海洋の温度分布や流速分布などを地球規模で測定する海洋音響トモグラフィーと呼ばれる技術では,1000km程度伝搬させるために,減衰の少ない数十Hz から200Hzの音波が使用される.また,魚群探知,ソナー(sonar)などには,分解能と減衰量とのかねあいにより数kHzから100kHz くらいが用いられている.一方,遺跡など土中の埋設物探査のために,数百Hz の横波音響波の利用が試みられており,地中レーダを使いにくい水分を含む土質に有効であると考えられている.また,医用診断装置では数MHz から40MHzが診断部位によって使い分けられる.鉄道の車軸,レール等や,原子力発電所の配管など金属の非破壊検査では数MHz が,超音波顕微鏡では数百MHz から1GHz 程度が利用される.また,超音波を用いた物性研究においては低い周波数から数GHz 以上の領域まで利用されている.一方,光ファイバ中の光散乱の一つにブリルアン散乱があり,この現象が分布型ひずみセンサとして実用に供されるようになっている.ブリルアン散乱は光ファイバ中に発生する音響波によって周波数シフトした光が後方散乱されるものであり,通信用光ファイバではその周波数シフト量は約11GHz である.
