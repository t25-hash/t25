# 6・4・3 伝搬する弾性波動の種類

> **章**: 第6章 各種応用計測
> **節**: 6・4 超音波応用計測

超音波は媒質の慣性力と弾性力より伝搬する弾性波動であるが,気体中・液体中では媒質粒子の振動方向と伝搬方向が同じである縦波としてのみ伝搬する.電磁波の電界と磁界に対応する波動場の基本量は,スカラ量である音圧とベクトル量である粒子速度である.音圧は圧力の変動分(Pa)であり,粒子速度は媒質粒子の振動速度(m/s)である.一方,ずり弾性も有する固体中では,縦波に加えて,振動方向が伝搬方向と直交する横波も存在する.この場合,応力テンソルと粒子速度ベクトルが波動場を表す基本量になる.固体中では,存在可能な伝搬モードが境界条件によって変化するが,振動方向が境界面と平行な横波をSH 波,垂直な横波をSV 波と呼ぶことがある.境界のある固体中では,縦波は単独では存在せず,弾性体のポアソン比によって横波と結合している.縦波をP波というので,これをP+SV 波と呼んでいる.表面近傍のみにエネルギーが集中したのが弾性表面波(SAW : surface acousticwave)であり,フィルタ応用がよく知られているが,計測応用上も重要である.
