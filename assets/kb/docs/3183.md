# 6・4・6 距離測定

> **章**: 第6章 各種応用計測
> **節**: 6・4 超音波応用計測

距離は伝搬時間を測定して既知の音速から求めるのが基本であるが,パルス波ないしバースト波を用いる方法と連続波を用いる方法とに大別される.a. パルス波,バースト波を用いる方法パルス波やバースト波を送波して,反射波が戻るのに要する時間を測定する方法であり,音速をとして,反射物体までの距離は,=(6・3)と求まる.実際の時間測定には電気系の遅れなども含まれるので,それを除外した正味の伝搬時間を求める必要がある.しきい値を設ける単純な受波パルス検出法によると信号強度などによるばらつきが生じるので,図6・46のように,受波波形の最初のピーク以降のゼロクロス点を受波した瞬間とする方法などが用いられている.図図6・46ゼロクロスによる受波時間の求め方図6・47 FM-CW 法による距離測定を開き,それ以降の最初のゼロクロスを検出し,その瞬間を受波時間としている.また,送波波形と受波波形をメモリに記憶し,計算機上で片方の波形を時間軸方向に移動しながら相関が最も強くなる移動時間を見つけ,これを伝搬時間とすることも行われる.すなわち,送波波形を,受波波形をとし,욀욐-τdτ(6・4)の値が最も大きくなるτを伝搬時間とする.長距離測定などでは自己相関関数がデルタ関数となるM 系列信号を送波する方式などもある.一方,受波信号により次の送波パルスをトリガしてさらに送波することを繰返し,送波の繰返し周波数を測定する方法はシングアラウンド法と呼ばれる.b. 連続波を用いる方法連続波を用いる方法には,送信波と反射波の干渉により生じる共鳴を利用する方法がある.送波周波数をゆっくり掃引すると,受波強度は周期的に変化する.反射体との距離が共振長となるとき強度は極値を示すので,その周期と音速から距離が求まる.これは厚みなどを求めるのに利用される.また,図6・47のように,送波周波数をのこぎり波掃引した場合,送波信号と受波信号の間には伝搬時間に比例したつねに一定の周波数差が生じるので,この周波数差から伝搬距離を求める方法がある.これはFM-CW(frequency modulation-continuouswave)法と呼ばれている.のこぎり波の繰返し周波数を욇,周波数掃引幅を욃とすると,=욇욃(6・5)のように伝搬距離が求まる.反射を利用していれば,反射物体までの距離はの半分である.この式から明らかなように,距離分解能は周波数掃引幅욃が大きいほど高くなる.
