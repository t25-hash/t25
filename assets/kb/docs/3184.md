# 6・4・7 速度測定

> **章**: 第6章 各種応用計測
> **節**: 6・46 中では、適当なしきい値を設け、これを超えたときにゲート

物体の移動速度は反射波のドップラー効果(Doppler effect)から測定される。移動速度、音速、送波周波数からドップラーシフト量は、≪として近似的に、=2(6・6)と与えられるが、温度変化や風による音速変動に留意する必要がある。血流を可視化する診断装置は超音波のドップラー効果を応用した好例である。一方、伝搬距離がわかっていれば、伝搬時間か5―133図6・48 超音波顕微鏡ら媒質の移動速度を測定することができる。これは風速計、流速計などに応用されている。
