# 6・4・8 物性測定・材料評価

> **章**: 第6章 各種応用計測
> **節**: 6・46 中では、適当なしきい値を設け、これを超えたときにゲート

超音波による材料物性測定では一般に音速と減衰(吸収)を測定する。音速は密度と弾性定数で決まるので、密度を他の方法で求めておけば弾性定数がわかる。伝搬距離が既知の場合に6・4・6項で示した距離測定の原理を用いれば音速がわかる。例えば共鳴法を用いれば、その共鳴周波数から音速が、共鳴の値から吸収が求まる。数十MHz以上の高周波では、音場による光の回折現象やブリルアン散乱なども音速測定に利用される。また、集束超音波を試料に照射してその反射波や透過波の強度の分布を映像化する顕微鏡が機械走査型超音波顕微鏡(ultrasonicmicroscope)である。密度と音速の積である音響インピーダンスに関わる像が得られるので、弾性的性質など光学顕微鏡とは異なった情報を取得できる。また100MHz以上の高周波数が使用されるので、高い空間分解能が得られる。超音波の集束にはサファイアレンズなどが用いられ、試料との間に純水などのカップラを介在させる。図6・48のように、集束点を試料表面よりわずかに深い位置に設定すると、表面からの直接反射波と、いったん表面波として表面を伝搬した後に再放射する音波(漏洩弾性表面波)との干渉が生じる。さらに、線状に集束する音響レンズを用いれば表面波の伝搬方向を特定できるので、結晶など異方性材料の各方位ごとの音速を求めることができる。また、断続したレーザ光を試料に照射し、その熱現象で超音波を非接触で励起して伝搬音や放射音を検出するのが光音響顕微鏡(photoacoustic microscope)である。この方法は固体でも、セルに閉じ込めた気体や液体にも適用できる。放射音は、マイクロホン、圧電素子、光学的手法などで検出する。発生する超音波強度は一般に高くはないので、ロックイン検出を行うのが普通である。超音波領域では非線形現象が発現しやすいが、生体、医用分野では非線形現象を利用した可視化や、媒質の非線形特性を測定することが試みられている。金属材料中のきずを非破壊で検査するには、適当な周波数、振動モードの超音波を送波し、これまで述べてきた手法を用いてきずからの反射波よりその位置と性質を判定する。一方、特に超音波を送波することなく、きずが生じたり成長したりするときに発生する弾性波を待ち受けて検出する非破壊検査方法はAE(acoustic emis-sion)法と呼ばれている。
