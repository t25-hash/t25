# 6・4・9 超音波を用いた計測、センサデバイス

> **章**: 第6章 各種応用計測
> **節**: 6・46 中では、適当なしきい値を設け、これを超えたときにゲート

超音波を利用したセンサ素子にはさまざまなものがあるが、その例を紹介する。液体中では横波は伝搬しないが、粘性は滑り振動する振動子の負荷となる。これを振動子の電気インピーダンスや共振の値から読み取るのが粘度計である。また、振動子に微小物体が付加すると、その質量効果によって共振周波数が低下する。この現象を利用したのが水晶式膜厚計で、蒸着装置などでの成膜管理に利用される。あらかじめ振動子上に堆積する膜厚と共振周波数の変化量を校5―134計測工学図6・49振動ジャイロの原理正しておく。また、特別な吸着膜を水晶振動子に形成しておくと、特定のガスに反応するガスセンサが構成できる。質量や粘性の効果は弾性表面波素子にも有効で、各種の弾性表面波センサが提案されている。これは圧電基板を用いているので、液体の電気抵抗や誘電率などによる境界条件の変化にも敏感で、電気的特性も反映したセンシングの実現が可能である。振動子の出力端に力を加えると共振周波数が変化することを利用した力センサも提案されている。図6・49は圧電振動ジャイロ(vibratory gyroscope)の原理図であるが、角柱形振動子にたわみ振動を励振している。いま、回転角速度が加わると、コリオリの力によって励振した振動と直交する振動面を有するたわみ振動が発生する。その振動速度は回転角速度に比例するので、その振動を圧電的に電気信号として取出す。これはビデオカメラの手ぶれセンサやカーナビゲーション用のセンサ、ロボット用センサなどとして広く実用化されており、最近では微細加工技術を利用した超小型で高精度な振動ジャイロ素子も実現されている。このように、超音波ないし超音波振動を利用して、さまざまな物理量、化学量に関するセンサが開発されている。画像応用計測には、計測の高速化を図ることができる、瞬時に全体の情報を得ることができる、直観的に理解しやすいといった特徴がある。赤外光や紫外光を用いることも可能であって、光学的にパターンを形成したり、不可視情報を可視化することも行われている。画像による計測を行う場合に関連する光の性質を表6・5に示す。
