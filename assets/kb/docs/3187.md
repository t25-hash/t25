# 6・5・1 幾何光学的原理による方法

> **章**: 第6章 各種応用計測
> **節**: 6・5 画像応用計測

光には直進するという性質があるとともに，その結像作用を利用
できるという特徴があり，幾何光学的な性質を利用して変位や形状
をとらえることができる．
a.二次元形状計測 スリット光のように空間に光の帯（シー
ト）を作り，これを物体に投影すると，あたかも光で物体を切断し
たかのように物体の断面が観察できる．この断面は輝度が高いため
に，画像としての取込みと処理を行いやすい．スリット（マルチス
リットでもよい）を通して光を投げかける方法，レーザビームを回
転ミラーなどで走査する方法（図6･50），レーザビームを円柱レン
ズを通して帯状に広げる方法，干渉じまを投影する方法などが行わ
れている．図6･51では車の模型を光切断（optical section）によ
り等高線表示した例を示す．
実用例としては，鉄道レールやトロリ線の摩耗検査に応用された
り，溶接ロボットの視覚センサとしても利用されるなどしている．
また形状のディジタイザという概念で使用している例もある．自動
車産業では，ホワイトボデーやパイプ部品の検査に使われている．
投光されるシート状ビームが角度によっては，受光側に届かずに死
角が生じないように投光部を回転させたり，物体を移動したりす
る，さらには2方向からの投光や観察を行うなどの工夫が施されて
いる．
