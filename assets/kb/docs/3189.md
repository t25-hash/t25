# 6・6・1 知識応用計測の範囲

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

知識応用計測(intelligence applied measurement)をどのように定義するかは、特に定まっていない。ここでは、センサから得られる信号を、増幅したりフィルタリングしたり、非線形変換するなどして所望の情報が得られるものに対して、そのようなデータに何らかの知識に基づく計算が施されて初めて有意な情報が出現する計測を、知識応用計測とする。5―139知識応用計測には一般化しているものもあるが、引き続き最先端の計測を支えているものもある。ここではそれらの中から、センサ情報と計算の考え方によりいくつかに分類して示す。
