# 5・3・2 加工木材

> **章**: 第５章 高分子材料ほか
> **節**: 5・3 木質材料（wood-based materials ）

a.合板（plywood) 通常，ロータリレースで丸太から0.5
mmから数mmの単板をカツラむきし，1枚ごとに繊維の方向を
直交させて接着剤で張り合わせたもので，普通3，5，7枚等の奇数
合わせとする．製品品質はJAS（日本農林規格）で決められてい
て，普通合板，コンクリート型枠用合板，特殊合板，構造用合板，
難燃合板，足場板合板，防災合板およびパレット用合板の規格があ
る．
構造用合板（表5･9，5･10）には，板面の品質によって1級と2
級があり，接着剤の耐久性によって特類と1類がある．実際的には
1級の構造用合板はラワン合板で高度な構造的利用，2級は針葉樹
合板で壁，床，屋根の下地板などに用いられる．特類の構造用合板
は常時湿潤状態における接着耐久性を保証するもので，試験片を
72時間連続煮沸等の処理後の接着力試験の値が表5･11の値以上で
あること，1類では通常状態での耐久性を保証し，煮沸繰返し試験
等の処理後の接着力が表5･11の値以上であることが決められてい
る．
b.集成材（laminated wood) 集成材とは，ひき板または小
角等をその繊維方向を互いにほぼ平行にして，厚さ，幅および長さ
の方向に集成接着した材である．一般には，厚さ20mm以上，幅
ガジョインタ等で縦継ぎし，ラミナに接着剤を塗布してそれらの繊
維方向を互いにほぼ平行にして数層に積層し，圧締め硬化後，四面
を切削仕上げする．JASに定める集成材の種類は，造作用，化粧
ばり造作用，構造用および化粧ばり構造用柱がある．構造用途の集
成材については，JASの強度等級区分に応じた基準強度が国土交
通省告示で表5･12のように定められている．造作用では一般にユ
リア樹脂系接着剤または同等の接着性能を有する接着剤，構造用集
成材ではレゾルシノール樹脂系接着剤またはそれと同等の性能を有
する接着剤が使われる．
