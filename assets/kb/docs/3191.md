# 6・6・3 反転法、マルチステップ法

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

高精度測定においては、測定の基準となるものの形状誤差等の大きさが測定対象の評価値と同程度となり、計測値から両者を分離する方法が必要となる。その一つは反転法である。反転法(reversalmethod)は、古くからある手法であるが、現在でも被計測物が達成可能な参照基準や運動精度と同程度の形状偏差等で仕上がっている場合には、有効な手段として利用される。マルチステップ法(multi step method)も同様である。反転法は、測定対象と基準との配置を反転(したがってセンサの検出方向も反転される)させ、計測値に含まれる基準の形状誤差の寄与を逆向きにして、反転前後の2回の測定値から両者を分離する方法である。直線形体の真直度(straightness)や円筒形体の真円度(circularity)などの評価で用いられる。図6・68,6・69にその原理図を示す。それぞれ、式(6・16)(6・18)のように観測方程式が立てられ、反転に伴う傾斜成分や偏心成分であるα,βを取り除くことにより、それぞれ式(6・17)(6・18)のように形状偏差と基準の誤差を分離することができる。=+=-′+α+β(6・16)=++=--(6・17)ただし、の項は5―140計測工学図6・68 反転法による真直度測定図6・69反転法による真円度測定=-′θ=θ+θθ=θ-′θ+αθ+β(6・18)θ=θ+θ+θθ=θ-θ-θ(6・19)真円度の評価においては調和解析(harmonic analysis)が可能であることから、上記反転法では反転位置の精度が直接結果に影響するため、測定物と基準を2πずつ円周方向の位相を変えて測定した結果から、2πずつ位相が進む測定物の形状偏差と、位相変化のない基準の系統誤差を分離するマルチステップ法が用いられることがある。
