# 6・6・4 ステレオ法

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

三次元立体視の獲得においては、奥行きを計測する必要があり、位置の異なる複数の画像から、三角測量に相当する幾何学的配置関係と、使用する光学系の関係から画像の中の特徴点の奥行きを計算し、三次元立体認識を行う。このいわゆるステレオ画像の作成方法には、同一像面に複数の撮像素子を距離を離して配置する標準的なステレオ撮影法から、一つの撮影素子を移動させて撮影する方法や、自動車などに搭載した1台のカメラの動画像から、カメラの移動距離や姿勢変化をも考慮して構築する方法など、さまざまな方法が提案されている。図6・70は、基本的なステレオ法(stereo method)の原理図である。
