# 6・6・5 CT

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

コンピュータトモグラフィー(CT : computed tomography)には、人体や工業製品用のX 線CTや、光を用いたCTなど多図6・70 ステレオ法の原理くの種類がある。このCT の原理は、一種の逆問題解析の一つととらえることができる。X 線CT の場合、被検物を構成している各要素が、その材質(厳密には構成元素)ごとにX 線吸収係数が異なることから、各方向からのX 線照射と吸収係数を積分した透過X 線分布を計測し、被検物内のX 線吸収係数分布に関する連立積分方程式を解くことにより、X 線吸収係数分布、すなわち材質ごとの空間分布を計算することができる。計算の原理からはスライスCT が理にかなっているが、計測の速度、すなわち計測時間の短縮のため、スパイラル撮影が行われることが多い。胸部撮影などで呼吸を止めなければならない場合には有効な方法となっている。X 線CT と同じような人体等の非侵襲検査として、核磁気共鳴によるMRI(magnetic resonance imaging)がある。MRIでは、強磁場中で水素原子核の共鳴(NMR : nuclear magnetic res-onance)スピンによる微弱な電波を検出している。MRIでは、共鳴原子の位置を制御(エンコード)するため、,各方向に傾斜磁場をかけ、各位置での共鳴周波数に差をもたせている。したがって、検出周波数をスキャンすることにより、被検体の空間位置をスキャンしたことになり、患部等の信号を断層像の集積として三次元的に得ることができる。
