# 5・4・1 紙，不織布，合成紙

> **章**: 第５章 高分子材料ほか
> **節**: 5・4 紙および繊維製品

a.紙，板紙
.定義と製法 紙は広義には繊維を水中に均一に分散した
後，脱水，乾燥してシート状にしたものと定義できる．このように
紙は水を使って繊維をシート化するのが特徴で，繊維としてほとん
どの紙は木材繊維を使用しているが，そのほか麻，綿，わら等の非
木材繊維や，ガラス繊維，合成繊維を使用したものも生産されてい
る．
木材構成成分の大半は繊維で，この繊維分を木材から取出したも
のをパルプと称し，これが紙の主原料となる．木材から繊維を取出
すのに，化学的に薬品で蒸煮して取出すものを化学パルプ（chem-
ical pulp），機械力を使ってすりおろすものを機械パルプ
（mechanical pulp）といい，一度使用した紙（古紙）をリサイク
ルしたパルプを古紙パルプという．現在日本では紙の原料として古
紙の使用比率が高くなり，2004年現在で60 %と過半数を占めてい
る．
木材から取出したままのパルプ繊維では水中でのシート形成には
絡み合い不足なので，叩解工程（beating process ）で繊維をほぐ
して絡みやすくする．これをワイヤ上に流し出して脱水，湿紙を形
成した後，さらにロール間でプレスして脱水し，これを高圧蒸気を
通したシリンダで乾燥して紙となる．この機械を抄紙機という．
紙の厚さはワイヤ上に流し出した繊維の量で決まり，量が多いほ
ど厚くなる．しかし量が多くなるとワイヤの脱水能力が落ちるの
で，厚い紙は複数枚のワイヤを有する抄紙機で抄いた紙を重ね合わ
せたうえで，脱水，乾燥する．これを抄き合わせといい，この方法
で作った厚紙が板紙である．
.紙・板紙の種類 紙は人間生活の中で広範囲に使用され
ているが，数量的には新聞紙や印刷紙，コピー等の印刷情報用紙が
大半である．板紙は包装用のダンボール原紙，紙器用板紙が大半を
占め，その他建材原紙，紙管原紙等に分類される．紙，板紙の品種
別分類表を表5･15に示す．
.紙，板紙の性質と特徴 紙は水中で紙層を形成し，乾燥
により水素結合を生じて強度を保つので，紙をぬらすと結合が消滅
して強度を失う．これが紙の耐水性に劣るという欠点であるが，逆
にこれが古紙の水への分散を容易にし，リサイクルを可能とする利
点ともなる．また，この特性は紙の吸湿性と伸縮（dimensional
stability）に関連し，吸湿すれば伸び，乾燥すれば縮む．したがっ
て紙の印刷に際しては印刷の環境と紙の水分が平衡していることが
重要で，印刷時に紙が伸縮すると多色印刷で重ね刷りをした際に，
伸縮により見当が合わなくなるおそれがある．
紙の抄造に際し，ワイヤ上に流出した繊維はワイヤの進行方向に
並ぶ傾向があり，紙に縦横を生じる原因となる．すなわち，ワイヤ
の進行方向を紙の縦と呼び，縦方向の引張強度は横方向の引張強度
よりも強い．また，紙を引裂くときの強度を引裂強度といい，これ
は引張強度とは逆に横のほうが強い．紙の強度は湿度とも関連して
おり，図5･22に湿度と各種強度との関係を示す．また，紙は織物
やフィルムと比べて腰が強いのが特徴で，このため製本したときに
しっかりとした書物となり，厚い板紙を使用すると段ボールやカー
トンボックスとなって包装用に有用な特性を発揮する．
繊維まで広範な繊維を使用した製品が生産されている．
紙は繊維の重なり合った構造のため，繊維間の空げきがあり，通
気性と吸収性を有する．この通気性がフィルタとしての特性を示
し，吸収性がティッシュのような衛生用紙としての特性を発揮す
る．さらに紙はその電気絶縁性と誘電性を生かし，ケーブルやトラ
ンス等の絶縁材料やコンデンサの誘電体としても使われていたが，
高電圧用を除き，順次，合成高分子にその座を譲りつつある．
