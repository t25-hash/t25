# 6・6・5 CT

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

コンピュータトモグラフィー（CT:computed tomography）に
は，人体や工業製品用のX線CT や，光を用いたCT など多
くの種類がある．このCTの原理は，一種の逆問題解析の一つとと
らえることができる．X線CTの場合，被検物を構成している各
要素が，その材質（厳密には構成元素）ごとにX線吸収係数が異
なることから，各方向からのX線照射と吸収係数を積分した透過
X線分布を計測し，被検物内のX線吸収係数分布に関する連立積
分方程式を解くことにより，X線吸収係数分布，すなわち材質ご
との空間分布を計算することができる．計算の原理からはスライス
CTが理にかなっているが，計測の速度，すなわち計測時間の短縮
のため，スパイラル撮影が行われることが多い．胸部撮影などで呼
吸を止めなければならない場合には有効な方法となっている．
X線CTと同じような人体等の非侵襲検査として，核磁気共鳴
によるMRI （magnetic resonance imaging）がある．MRIで
は，強磁場中で水素原子核の共鳴（NMR:nuclear magnetic res-
onance）スピンによる微弱な電波を検出している．MRIでは，共
鳴原子の位置を制御（エンコード）するため，，， 各方向に
傾斜磁場をかけ，各位置での共鳴周波数に差をもたせている．した
がって，検出周波数をスキャンすることにより，被検体の空間位置
をスキャンしたことになり，患部等の信号を断層像の集積として三
次元的に得ることができる．
6・6・6ニューラルネットワーク，GA
ニューラルネットワーク（neural network）の計測への応用
は，複数の入力の非線形かつ複雑な結びつきによる出力が必要な場
合に有効である．センサフュージョンの一部にはその応用ととらえ
ることができるものもあり，種類の異なる入力から，例えば人
間の快適性を判断するような枠組で適用の可能性がある．実際の応
用においては，入出力のノード数，層数，各要素における複数入力
それぞれの影響係数である重み，発火のしきい値などを対象モデル
に対してどう決めるかが重要である．理想的には物理的意味付けが
なされることであろうが，実際には，ニューラルネットワークを適
用する目的の一つにブラックボックス化，あるいは入力相互関連構
造そのものの探索等があるため，学習や数理計画的実験などが必要
となる．これは一種の最適化問題として扱うことができる．例え
ば，自動車室内の空気調和を考えるとき，温度だけでなく，湿度や
外部からの太陽光の差込み等，またエアコンの吹出しの風速や車室
内の位置など，複数の情報を総合してその座席の空気調和を制御し
ようとする際，これらの複数の種類の異なったセンサの融合によ
り，吹き出す冷気の温度や風量を決定することができる．
