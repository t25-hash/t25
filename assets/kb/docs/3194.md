# 6・6・6 ニューラルネットワーク,GA

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

ニューラルネットワーク(neural network)の計測への応用は,複数の入力の非線形かつ複雑な結びつきによる出力が必要な場合に有効である.センサフュージョンの一部にはその応用ととらえることができるものもあり,種類の異なる入力から,例えば人間の快適性を判断するような枠組で適用の可能性がある.実際の応用においては,入出力のノード数,層数,各要素における複数入力それぞれの影響係数である重み,発火のしきい値などを対象モデルに対してどう決めるかが重要である.理想的には物理的意味付けがなされることであろうが,実際には,ニューラルネットワークを適用する目的の一つにブラックボックス化,あるいは入力相互関連構造そのものの探索等があるため,学習や数理計画的実験などが必要となる.これは一種の最適化問題として扱うことができる.例えば,自動車室内の空気調和を考えるとき,温度だけでなく,湿度や外部からの太陽光の差込み等,またエアコンの吹出しの風速や車室内の位置など,複数の情報を総合してその座席の空気調和を制御しようとする際,これらの複数の種類の異なったセンサの融合により,吹き出す冷気の温度や風量を決定することができる.遺伝的アルゴリズム(GA : genetic algorithm)も,対象とするシステムを理解する方法の一つである.生物の世代更新における遺伝子の継承になぞらえて,システムにおいて制御されるパラメータ等を遺伝子とし,適当な混合や突然変異を設定して世代更新を繰返す.その結果として現れる分布が現実に対象としているものに合っているかどうかでこのGA モデルを評価し,修正と再計算を繰返すことで,対象としているシステムにおける各制御パラメータ等の作用を理解しようとするものである.
