# 5・5・1 塗料の構成

> **章**: 第５章 高分子材料ほか
> **節**: 5・5 塗料

塗料には，数多くの種類があるが，機能的に，塗膜形成主要素と
塗膜形成副要素，塗膜形成助要素から構成される．これらの構成要
素からなる透明媒質がビヒクル（vehicle）で，その不揮発性成分
をビヒクルソリッド（vehicle solid）と呼ぶ．また，塗料の構成に
おいて顔料を含まない塗料を透明塗料（クリヤ，clear lacquer ），
着色顔料を含む塗料を着色塗料（エナメル，enamel ）と呼ぶ．こ
れら塗料の構成を図5･26に示す．
塗膜形成主要素は，塗膜を形成する主成分であり，乾性油，天然
樹脂，合成樹脂，セルロース誘導体などが用いられる，塗料および
塗膜の性能を決定する重要な部分である．塗膜形成副要素は，塗膜
や塗料の性能を向上させる目的で加えられる成分であり，可塑剤，
硬化剤，乾燥剤，増粘剤，界面活性剤，皮張り防止剤，殺菌剤など
がある．塗膜形成助要素は，塗料を液状に保ち，塗料を塗装すると
きに流動性を調整する目的で使用され，乾燥過程で蒸発し，形成塗
膜中には残らない成分である．主要な塗料用有機溶剤および水の性
能を表5･22 に示す．一般には，各種の有機溶剤が用いられ
るが，溶剤成分はすべての塗料に不可欠の成分ではない．近年は，
溶剤を含まないか少量しか含まない塗料として無溶剤型塗料例えば
粉体塗料や紫外線（UV）硬化型塗料，ハイソリッド塗料などが，
また，水を溶剤成分として用いる水系塗料などが広く使用されてい
る ．
顔料（pigment ）は，塗膜に色彩や隠ぺい力を与え，塗膜の物性
や耐久性を補強したり，防錆効果を付与するために用いられる．顔
料には，無機着色顔料，有機着色顔料および体質顔料，防錆顔料が
ある．無機着色顔料は一般に隠ぺい力，耐候性，耐熱性および耐溶
剤性に優れるが着色力に劣る．有機着色顔料は一般に着色力，鮮明
度に優れるが隠ぺい力や耐候性に劣る傾向をもつ．しかし，近年，
高い耐候性をもつ多くの有機着色顔料が開発されており，目的とす
る用途に応じて有機着色顔料と無機着色顔料を組合わせて用いられ
