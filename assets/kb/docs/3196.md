# 5・5・2 塗料の分類

> **章**: 第５章 高分子材料ほか
> **節**: 5・5 塗料

塗料は，品目別（塗膜形成要素別），需要区分別，塗料形態別な
どさまざまな分類がされている．例えば，塗膜形成要素別では，油
性塗料，溶剤系合成樹脂塗料，水系塗料，無溶剤系塗料，無機質塗
料などに分類され，需要区分別では，建物用塗料，道路車両用塗
料，船舶用塗料，金属製品用塗料，家庭用塗料などに分類されてい
る．
ここでは，合成樹脂塗料について，近年の社会的ニーズにマッチ
した環境対応型塗料を中心に形態別の分類で示す．
合成樹脂塗料とは，各種合成樹脂を塗膜形成要素として使用し，
現在のはん用および工業用塗料の主流をなしている塗料である．合
成樹脂はその合成法から，アクリル樹脂，ビニル樹脂などの付加重
合物と，アルキド樹脂，ポリエステル樹脂，アミノ樹脂などの縮合
重合物とに大別され，性能と価格面から使い分けられている．合成
樹脂塗料の特性を表5･24に示す．
合成樹脂塗料はまた，形状により，粉体塗料，水系塗料（水溶性
型，エマルジョン型，水性ディスパージョン型）および有機溶剤系
塗料（溶剤型，溶液型，非水ディスパージョン型）に分類され，ま
た乾燥形式により，常温乾燥塗料，焼付乾燥塗料，2液反応硬化塗
料，電子線・紫外線硬化塗料などに，塗装法からは，電着用塗料な
どに分類される ．
a.粉体塗料（powder coating) 粉体塗料は，有機溶剤や水
などの揮発性成分を含まず，数 mから数十 mに微粉砕した固形
樹脂が用いられ，塗装後，熱により溶融させ，堅牢な膜を形成する
塗料である．無溶剤であること，回収・再利用が可能な点で環境適
応度が高い塗料といわれており，今後大いに期待される塗料であ
る．その需要量は着実に伸びているが，まだ，一部の限られた分野
での使用にとどまっている．薄膜での仕上り性（平滑性）や色替え
時の作業性で改良が望まれている ．
b.水系塗料（waterborne coating) 水系塗料は，有機溶剤
の大半を水に置換した塗料であり，揮発性有機化合物（VOC）低
減対策，悪臭防止，危険物対策などに期待がかかり早くから使用さ
れている．粉体塗料に次いで環境対応度が高いともいわれている．
この水系塗料は，水溶液型と水分散型に分けられ，水分散型にはエ
マルジョン型とディスパージョン型がある．表5･25に水系塗料の
特徴を示した ．
c .ハイソリッド塗料（high solids coating) 従来の有機溶剤
に樹脂を溶解して用いる最も一般的な溶剤型塗料の延長線上に位置
する塗料である．粉体塗料や水系塗料のようにVOC規制に対して
飛躍的な効果は期待できないまでも，近年，VOC低減を目的とし
て，塗料状態での加熱残分が約75 %以上が望ましいといわれてい
る，溶剤量の少ない，ハイソリッド型塗料が多く用いられるように
なってきた．ハイソリッド化技術は，使用する樹脂を低粘度化する
こと，すなわち低分子量化の方向にもっていくことであり，これに
伴う，性能の維持が望まれている．
