# 5・5・3 塗料の製造

> **章**: 第５章 高分子材料ほか
> **節**: 5・5 塗料

塗料の構成成分は，使用用途に従って適正な配合比で混合，均一
化されて塗料となる．塗料になるまでには，大まかに(１)溶解工
程，(２)顔料分散と調色工程，(３)製品の仕上工程，(４)製品の缶
詰工程に分類され，その工程あるいは操作に関する化学工学が塗料
の製造で重要である．一般的に，透明塗料の場合には，ディゾルバ
（高速かき混ぜ機）でビヒクルと溶剤をかくはん混合し，溶解均一
化して製造されるが，顔料着色塗料では，サンドグラインドミルや
ボールミル，アトライタなどの各種分散機を使用して，顔料をビヒ
クルに均一に混和・分散させる顔料分散（pigment dispersion）工
程を必要とする．そして，得られた原色エナメルを必要に応じて他
の原色エナメルと調合して所定の色調に合わせる調色工程，さら
に，塗料をカートリッジろ過機やふるいろ過機などでろ過精製する
ろ過工程を経て，容器に詰めて製品とする缶詰工程から成り立って
いる ．
