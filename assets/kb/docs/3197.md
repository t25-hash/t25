# 6・7・1 官能検査と官能量計測

> **章**: 第6章 各種応用計測
> **節**: 6・7 官能検査

官能検査(sensory test)とは,「人間の感覚を用いて品質特性を評価し,判定基準と照合して判定を下す検査」と定義されている.それは,てんびんやものさしなどによる理化学的検査に対し,以下のように特徴付けられる.すなわち,目・口・鼻などの感覚器官の働きである官能によって行う官能検査は,人間を一種の計測機器と考え,人間の感覚(視覚・聴覚・味覚・嗅覚・触覚の五感)を総動員して行う品質検査のことである.人間が感じるうまさやまずさを,実際ににおいを嗅いだり,飲んで味わったりして判断する.たとえ分析値が正常であっても,ほんのわずかな味のずれが官能検査により指摘されることもあり,さまざまな味や香りの組合せによる香味バランスの検査には,機器よりも人間のほうが適しているとされている.官能検査は,前述したように判定基準と照合する絶対的検査となっており,味覚検査において,あらかじめ基準の味覚を確認するなどの方法が取られている.一方,FPD(flat panel display)や紙幣の検査など,欠陥の存在を検査する場合などにあっては,基準との比較を必要としない場合もある.また,官能検査とは若干異なるが,消費者モニタなどによる調査では,嗜好の調査も含まれており,基準との比較という枠組とは考え方が異なっている.官能検査の種類は,五官による五感と,それらの組合せによるものがあり,一概に感覚器官の数だけでは納まらない.
