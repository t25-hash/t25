# 6・7・2 官能検査方法

> **章**: 第6章 各種応用計測
> **節**: 6・7 官能検査

官能検査の試験方法には、一般的に三つの方法がある。すなわち、(1) 二つの試料に差があるかどうかを決定するために用いる識別試験法(2) 差の順序もしくは大きさを評価するために尺度を用いる試験法(カテゴリー分類試験法を含む)(3) 試料特有の官能特性を確かめるために用いる定量的分析型試試料間のわずかな差を検出する場合評価者が限られる場合試料数の増加には不適評価者の選抜や訓練に使う感覚疲労が大きい差を強調してしまう基準試料と比較すべき試料に差があるかどうかを決定する場合刺激が持続する性質には向いていない少数の評価者の場合効率良く両者の差を確かめたい場合3点試験法と同様である感覚疲労、記憶効果の影響がいっそう強いA 非A 試験法評価したい試料の属性を評価者に理解させるため、試料A か非A かを判断させる外観に違いがある試料や刺激が長く持続する試料評価厳密に似ている試料を多く用意できない場合に有効5―142計測工学験法である。このうち、(1)の識別試験法が比較的多く用いられ、それはさらに次の5種類の試験方法に分類できる。(a) 2点試験法(b) 3点試験法(c) 1対2点試験法(d) 2対5点試験法(e) A 非A 試験法これらの特徴は表6・6に示すとおりである。製造プロセスや実験では、2点試験法(1対比較法)やA 非A 試験法が使われることが多い。すなわち、正常なもの(基準)と被験物を比較し、欠陥品など基準と異なるものを検出したり、訓練された検査作業者が製品の中から不良品を抜き取るなどの方法が採られる。
