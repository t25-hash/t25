# 6・7・3 適用例

> **章**: 第6章 各種応用計測
> **節**: 6・7 官能検査

官能検査が適用されている例を以下に示す.a. 味つくりインスタント食品のスープなどの味つくりにおいては,化学分析では判断できない,総合的な評価を官能検査によって行う.また,ワインや日本酒,ウイスキー,しょうゆや味噌など,発酵をコントロールしたり,ブレンドしたりする場合にも,官能検査が活躍する.これは,味覚は大きく,甘い,辛い,酸っぱい,苦いの4種類(最近ではこれらにうまみを加えて5種類とすることもある)に分類することができ,これらとにおい,色など,複数の官能量を総合して評価することが必要なためであり,かなり熟練した検査員が求められる.b. 欠陥検査(defect inspection)ブラウン管のシャドウマスクのエッチングによる微細パターン加工や,紙幣の印刷など,局所的にごみが影響したりインクがのらなかったりする現象は,まだまだ検査の自動化が困難な領域である.これらにおいては,A 非A試験法に分類できる検査方法が採られている.人は,規則性の中の不規則を発見する効率において,卓越した能力を発揮する.c. 加工面や塗装面品質ヘアライン加工と称される,ベルト研削等によって作られる加工面の粗さの特徴は,周期的虹面より,白色と評価される面のほうが高品位に見える.時計や装飾品のバンドはその良い例であるが,同じ加工法であっても,微妙な加工条件の違いで作られる異なった品質は,その製品の芸術的価値を大きく左右する.工作機械の案内面などでは,高精度を長期間維持するため,きさげ加工が用いられる場合があるが,単位面積当たりの接触点の数や分布,パターン等が求められ,その善し悪しの判定も官能検査によっているといえる.自動車ボデーの塗装面の評価も同様であるが,こちらはむしろ脱脂不足等による塗料ののりの悪い箇所の判定などに用いられている.
