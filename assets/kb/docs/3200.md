# 6・7・4 官能検査の不確かさ

> **章**: 第6章 各種応用計測
> **節**: 6・7 官能検査

官能検査は,個人の官能器と,個人の中枢神経網における判断によるため,個人差が存在する.各個人の試料間の差の方向は,訓練などによってある程度平均的な評価に近づけ,調整することができるが,このように評価者個人の特性に起因する不確かさを含むため,官能検査では,基本的に複数の評価者による結果を統計的に扱う.実際の処理は,比較する両者に差がないという帰無仮説に対し,対立仮説として,両者に差があるとする両側検定か,一方が他方より好ましい,あるいは好ましくないとする片側検定によって行う.したがって,ある程度の評価者の数,組合せ試料の数が必要であり,5%程度の有意水準によって検定が行われる.
