# 6・7・5 感性と官能検査

> **章**: 第6章 各種応用計測
> **節**: 6・7 官能検査

官能検査が,評価者の嗜好を極力排除し,意識させないように行われるのに対し,感性(kansei,sensibility)は,評価者の好みなど,嗜好を積極的に加味した評価を扱う.消費者モニタリングなどはこの範ちゅうである.この場合は,より好ましいものを選択する必要から片側検定となり,試験法も限定される.また,嗜好の分布は年齢,性別,生活環境など,多くの要因によって変動するため,より多くの標本量と,調査の目的によっては標本の選定も必要となる.文献(1) ECCJ,http://www.eccj.or.jp/catalog/2006w-h/index.html(2) ECCJ,http://www.eccj.or.jp/labeling/index.html(3) ECCJ,http://www.eccj.or.jp/ene-star/prog/index.html(4) 石井行弘・高橋毅,光技術コンタクト,40-2(2002),87.(5) Heer,C.V.,Bull.
