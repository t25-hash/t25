# 5・6・2 シーリング材（sealants )

> **章**: 第５章 高分子材料ほか
> **節**: 5・6 接着剤およびシーリング材

シーリング材は，一般的には弾性体である．部材の間げきや継目
などに充てんされ気密性，水密性を保つことが最も重要な機能であ
る．具体的な用途は，輸送機関連（自動車，車両，航空機），情報
機器関連（電気，電子，ディスプレイ，光通信），建築・土木関連
など広い分野となっている．これらの用途の中には，気密性，水密
性のほかに断熱性，防振性，防音性，耐腐食性，絶縁性などの機能
がさらに要求される場合が多い．
シーリング材の中には，特にガラスパテ，油性，れき青質系のよ
うな非弾性型の材料があり，これはコーキン材といわれており弾性
型シーリング材と区別している．
なお，シーリング材と似た表現にシール材（ガスケット，パッキ
ン類）があるが，この表現は，おもに装置産業分野で用いられてい
る ．以下にシーリング材のおもな使用例を示す ．
a.自動車
イレクトグレージング法により窓ガラスがボデーに取付けられる．
具体的には，ボデーの塗装鋼鈑と窓ガラスの双方にプライマを塗布
したのち，窓ガラス面に湿気硬化型1液ポリウレタン系シーリング
材が塗布され取付けが行われる．水密性，気密性のほかに，硬化後
のゴム弾性（熱応力の吸収，ボデー変形への追従性），高強度（ボ
デーの高剛性化），さらには耐熱・耐寒性，耐水性，耐候性などの
耐久性が要求される（日本自動車規格，JASO M338-89参照)．
プラスチゾルが用いられている．一般的には，電着塗装面に塗布し
次工程で加熱されて弾性体となる．最近では，ポスト塩化ビニル材
料としてポリウレタン系材料等が検討されている．
b.鉄道車両
が用いられている．耐久性等が要求される．
ファイド系シーリング材が用いられている．防水性，耐候性等が必
要．
リング材が用いられている．電気絶縁性，耐候性等が必要．
c .航空機
グ材が用いられている．耐油性，耐振動性，防食性，接着性等が重
要．その他の部分として，機体構造物接合部（機体内外の気圧差を
保つため），窓ガラス，与圧室の気密性保持などにも用いられてい
る．これらの航空機用シーリング材は，MIL規格や航空機メーカ
の規格をクリアする認定品が用いられる．
d.電気，電子，ディスプレイ
硬化ゴム等が用いられている．耐熱・耐寒性，絶縁性等が必要．ま
た，電子基板にウレタン系樹脂をポッティングし高寿命化を図った
例もある．

晶の注入孔のシールには実績があるが，セルのシールへの適用も検
討されている．
