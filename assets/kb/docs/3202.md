# 6・1・2 自主対応型アプローチによる労働安全衛生管理

> **章**: 第6章 マネジメントと法制度
> **節**:  

a．法規準拠型アプローチの問題点 労働安全衛生管理とは，
「事業の運営に伴う労働災害の根絶を目的として，経営者が行う合
理的，組織的な一連の施策」をいう．これらの施策を具体化するに
あたり，従来からの労働安全衛生管理は，我が国はもとより欧米に
おいても，労働安全衛生に係わる法令を遵守するという，いわゆる
法規準拠型のアプローチを中心に進められてきた．しかし，この方
法には限界があり，問題点が指摘されている．我が国の安衛法が制
定されたちょうど同じ1972年に英国において公表された「労働にお
ける安全と健康」，いわゆるローペンス報告のなかで，法規準拠型の
アプローチの問題点として，次の4点があげられているω．
（1）法律があまりにも多く詳細すぎるため，全体の理解が困難と
なり，現場において改善に取り組む上で逆効果である．また，
法はすべてをカバーできず，本質的に不満足なものとなってい
る。
（2） 限られた数の監督官で，すべての事業場に法規を徹底させる
ことは物理的に不可能である．
（3） 規制的要素が強すぎ，外部からの働きかけで改善されるもの
だという受動的な態度が職場に広がり，自主性，自発的努力が
薄らぎ，現場に則した具体的な改善が進みにくい．
（4） 行政管轄が細分化され過ぎ，多様な法規定に支配される事業
場がある一一方，全く範囲外となる職場が存在する。また，既存
の法規定と施行制度の多くのものが時代遅れになっている．
また最近では，法規準拠型のアプローチは，「不確実性を伴う先端
的な分野の規制においては，規制内容の不確実性等からの基準の欠
陥と，規制基準を執行する段階での人的リソースの不足等からの執
行の欠陥，という二大欠陥を伴いうる」ωとの指摘もなされている．
b．自主対応型アプローチへの移行 ローペンス報告では，a項
に記載したような問題点を解決するための施策を提言として取りま
とめている．それは基本的に次の3点に要約される（3）。
（1）細分化され二二している安全衛生行政を一元化し，統一的な
執行体制を確立すること．
（2）複雑化している安全衛生関連法規を体系化し，明確化するこ
と．
（3） 時代の進歩に柔軟に対応できるように，規制や監督中心から，
自主基準を活用した自主対応型の安全衛生活動へ変換すること．
イギリスではこの提言を全面的に取り入れ，真974年には自主的対
応を重んじる法制度が成立している．
最近の世界的な動向としても，企業の自己責任と労使の協力によ
る自主対応努力およびそれによる現場改善が重視されてきている．
そして行政側としては，従来は，個々の設備・作業に対する技術基
準の追加設定などの，より細分化された法規制の整備を主として推
進してきたが，最近では，職場自主改善システムづくりの支援や，
そのために必要なサービスの推進を行うことに重点を移してきてい
る．より技術的な基準は，対策指針集という形などで整理し情報提
供され，企業における自主対応努力を支援するものと位置付けられ
ている．
すなわち，従来進めてきた法規準拠型のアプローチを転換し，今
後は，企業の自己責任と労使の協力による自主対応型のアプローチ
で進めていく必要があるとの認識が世界的に広まってきている．表
