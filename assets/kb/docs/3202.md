# 7・1・1 計測機器管理の必要性

> **章**: 第7章 計測における管理と教育
> **節**: 7・1 計測機器の管理

製品の品質が,規定されている要求事項に適合しているか否かは,測定によって実証されることが必要である.品質は測ることによって維持・向上を図れ,品質管理(quality control)は計測管理(measurement control)があって初めて実行することができる.品質管理においては,測定値は正しく得られたものとして取扱われているのである.確かな計測の実現のために,計測管理は重要な事項である.現在,多くの企業に課せられている品質システム(quality system)の構築は,ISO 9000(品質マネジメントシステム,quality management system),JIS Q 9000シリーズによっている.その中で,監視,測定,検査および試験が一連の重要事項となっているが,測定機器の管理なくして,適切な測定,検査,および試験はできない.
