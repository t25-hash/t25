# 7・1・2 管理対象機器の選定

> **章**: 第7章 計測における管理と教育
> **節**: 7・1 計測機器の管理

管理対象としての監視および測定機器の選定は,製品の仕様への適合のいかんを実証するために,監視および測定をする対象量,手段・方法と必要な測定精度,測定範囲,信頼性などを考慮して行う.ある機器を使用して対象量を測定したとき,その測定結果が製品の品質に影響を与えるような測定装置は,精度などの性能の管理対象となる.例えば,ある長さを直尺で測定するとき,その直尺が品質を決定する測定器であれば管理対象とするが,後工程でより正確な測定が別に設けられた装置で行われるものであれば,前工程での直尺の測定は目安として使用していることになり,管理の必要はない.生産設備に付いている圧力計,温度計,電気計器などの多くは製品の品質を定めるものでなく,目安として使用していることが多いので管理対象外であるが,その直後に,品質を直接測定できない熱処理,エージング,接着などの工程があるような設備に付いている計器は,管理対象として取扱わなければならない.精密測定室における制御系の温度計,湿度計などは,品質を確認するための測定が別にできないならば,管理対象として取扱う必要がある.そのとき,別に用意した温度計,湿度計,気圧計による値を用いることができるなら,前者は目安とし管理対象外に,後者を管理対象とすることになる.同じような測定機器がいくつかある場合は,品質に関係する測定量であれば抜取りで行うのでなく,すべて管理対象として取扱わなければならない.使用していない機器やバックアップ用の測定機器は,管理対象外であることを明記して,誤って使用しないようにしておくことが必要である.必要とする測定項目,必要精度などを明確にしておき,それに必要な監視,測定,試験装置の選定理由など,第三者がわかるような記述をしておくことが望まれる.
