# 7・1・2 管理対象機器の選定

> **章**: 第7章 計測における管理と教育
> **節**: 7・1 計測機器の管理

校正周期は規定として定めておくことが必要である．推奨値を示
管理対象としての監視および測定機器の選定は，製品の仕様への
しているものがいくつかあるが，表7･1に長さ測定機器の校正周
期の一例を示す．ブロックゲージなどは材料の性質上，経年変化が
あるので，特に寸法の長いものは注意が必要である．計量標準供給
システムで認定されている校正事業者が，表7･1に示す長さ測定機
ある機器を使用して対象量を測定したとき，その測定結果が製品
器などの標準器としてもっているものが特定二次標準器である．各
種の量について，この特定二次標準器等の校正周期の一例を示すと
表7･2となる．企業においては，ISO9000，JIS Q9000等に対応
な測定が別に設けられた装置で行われるものであれば，前工程での
直尺の測定は目安として使用していることになり，管理の必要はな
い．
生産設備に付いている圧力計，温度計，電気計器などの多くは製
品の品質を定めるものでなく，目安として使用していることが多い
ので管理対象外であるが，その直後に，品質を直接測定できない熱
処理，エージング，接着などの工程があるような設備に付いている
計器は，管理対象として取扱わなければならない．精密測定室にお
ける制御系の温度計，湿度計などは，品質を確認するための測定が
別にできないならば，管理対象として取扱う必要がある．そのと
き，別に用意した温度計，湿度計，気圧計による値を用いることが
できるなら，前者は目安とし管理対象外に，後者を管理対象とする
ことになる．同じような測定機器がいくつかある場合は，品質に関
係する測定量であれば抜取りで行うのでなく，すべて管理対象とし
て取扱わなければならない．
使用していない機器やバックアップ用の測定機器は，管理対象外
であることを明記して，誤って使用しないようにしておくことが必
要である．必要とする測定項目，必要精度などを明確にしておき，
それに必要な監視，測定，試験装置の選定理由など，第三者がわか
るような記述をしておくことが望まれる．
