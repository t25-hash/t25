# 7・1・3 計測機器の校正とその周期

> **章**: 第7章 計測における管理と教育
> **節**: 7・1 計測機器の管理

品質に影響を与える測定,検査および試験のための装置は,設備する際,およびその後定期的に,あるいは使用する前に校正すべきである.使用される標準器は校正の不確かさがわかっている適正な校正方法で行い,その校正の上位の標準器を順次たどったとき,源が国の標準器に結びつくことが求められる.これを全うしていることを計測機器のトレーサビリティ(traceability)があるという.測定値の正確さは,測定機器または標準器のトレーサビリティ体系に従って得られている.測定機器,標準器の種類によってトレーサビリティ体系が完備されていない測定量もあり,その場合,その基準についての十分な記録が望まれる.わが国では,計量法にJCSS(Japan Calibration Service Sys-tem)制度として物理量の標準器の校正体系に従って標準の値,また化学量を与える標準物質の供給がなされて,測定量,試験量の正しさが保たれている.国際的にもこのようなことが望ましく,各国のシステム間で相互承認などが行われつつある.図7・1は,JCSS制度による計量標準供給システム(measurement standard supplyservice system)と,それによる標準供給認定事業者の大要を示したものである.ここで,測定のトレーサビリティの考えに基づいて,ユーザの測定機器は個々の企業独自に処理することが求められる.すなわちユーザからの矢印を上に向いた矢印として考え,企業測定標準の管理をしていくことが重要である.標準器や測定機器の示す値は,使用の際に,それにまつわるいろいろな原因によって変化するため,校正して用いることが必要である.校正周期(calibration period)は,その測定機器の要求精度,使用頻度,使用環境などの使用条件やその機器の校正実績に応じて決める.世間でいわれている期間は参考にしてもよいが,それに従う必要はない.後述する定期点検の状況をもとに校正を考え,結果を校正周期に反映させる.校正周期は規定として定めておくことが必要である.推奨値を示しているものがいくつかあるが,表7・1に長さ測定機器の校正周期の一例を示す.ブロックゲージなどは材料の性質上,経年変化があるので,特に寸法の長いものは注意が必要である.計量標準供給システムで認定されている校正事業者が,表7・1に示す長さ測定機器などの標準器としてもっているものが特定二次標準器である.各種の量について,この特定二次標準器等の校正周期の一例を示すと表7・2となる.企業においては,ISO 規格に基づいて,使用条件を加味して定める.使用頻度の高い部分や最大,最小,中間点などの校正,また基準点,使用する範囲の主要な3点の校正ということも考えられる.いずれにしろ校正事項と校正測定基準とを明文化しておかなければならない.校正手順は校正装置の構造形式,環境条件,配置場所等を考慮して定める.その測定機器が校正状態にあることを適切に表示し,校正中の測定機器について間違った使用がなされないように明示しておくことが大切である.校正結果が不満足な場合はその処置方法を明文化して,取扱いを誤まらないようにしておく.さらに製品の品質への影響として,製品の手直しや廃棄も考慮することが必要であり,また測定機器の修理・調整を行う基準などを決めておくことも必要である.校正は社内で行う場合と外注で行う場合がある.要求精度や校正テスト,校正機器の数量などによって外注か内部で行うかを定める.外注の場合,選んだ機関の概要と選定の理由を記述し,その後の対応に備えておくことが必要である.校正した測定機器にはその有効期間を示し,使用者の利用の便を図る必要がある.
