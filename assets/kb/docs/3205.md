i．Iso 14001規格 Iso 14001規格の構成は，序文に始まって，
この規格のなかで用いられる重要な用語の定義および17項目の要求
事項が述べられた後に，附属書が付けられている、序文および要求
事項についてはそれぞれb項および。項で詳述するので，ここでは
ISO 14001の規格としての性格と付属書の内容について述べる．
（1） 規格の性格 ISO 14001規格には，「環境マネジメントシス
テム：仕様及び利用の手引」とタイトルが付けられている．タイトル
に示されるようにこの規格は仕様であるから，マネジメントシステ
ムの適合性評価の基準である．いわば自主的に参加できる法律のよ
うなもので，この規格で要求されている事項をすべて守れば適合の
評価が与えられるし，自己宣言も成立する．したがって，この規格
の条文（要求事項）にはすべて「Shall… しなければならない」
の記述が用いられている．ISO 14004の記述がすべて「Should， can，
may… するとよい，することができる，してもよい」となって
いるのとは対照的である．
この規格を作る際には，ここで要求する事項は適合の可否を客観
的に容易に評価できることを条件として記述することが申し合わさ
れた．完成した規格の要求事項の大半はこの条件を満たすものとな
ったが，一部には客観的評価の若干困難なものが見られる．
適合性評価をめぐる解釈上の議論が生じた場合には，ISO／TC
207の環境マネジメントシステム分科会がこの問題に対して責任を
持つが，原則として規格の解釈は各国の規格制定機関（我が国では
「工業標準調査会」）に任されている．国の規格制定機関が解釈に困
った場合には，前記分科会に解釈案の提示を求めることができるよ
うになっている．
（2） 規格の附属書 附属書Aは利用の手引，BはISO 14001
とISO9001の関連を記述したもの， Cは参考文献を記載したもので
ISO 14000およびISO9000の両シリーズのうちシステムに関連する
もの9件があげられている．附属書Aは，ISO 14001規格に書かれ
た要求事項を解説的，例示的に説明したもので，規格の要求にした
がってシステムを構築したり，適合性を評価したりする場合の手引
を与えている．
ii．Iso 14004規格 Iso 14004規格は，「環境マネジメントシス
テム：原則，システム及び支援技法の一般指針」と名付けられたガイ
ドラインである．ISO 14001の要求事項を基準として，五つの原則（約
束及び方針，計画，実施，測定及び評価，見直し及び改善）ごとに，
環境マネジメントシステムの計画，運用に対する各種の示唆を与え
ている．ただしこのガイドラインには，ISO 14001の要求事項を満た
すための解説に加えて，持続可能な開発に貢献するためのより高度
の内容まで含まれている．したがって，マネジメントシステムの適
合性評価の観点で，この規格を引用することには無理があることを
理解しておく必要がある．
なお，附属書のA1には1992年リオサミットにおいて採択された
リオ宣言の27原則が収録されており，附属書A2には1994年に決議
された国際商業会議所のビジネス憲章（ICC Charter）が収録され
ている．
b．ISO 14001規格の序文 ISO 14001規格の冒頭には3頁にわ
たって序文が書かれている．序文には，この規格が必要とされた背
景から，規格の理念，原則的な事項および活用の仕方などが述べら
れている．
この規格を利用する人にとって，最も重要なことは規格の要求事
項を正しく理解することであるが，それと同時にこの規格作成の意
図を理解することも重要である．序文にはその意図が理念あるいは
原則の形で書かれているので，重要な部分を抽出して解説しておく．
b序文は12の小文節に分かれて書かれているので，以下の説明は文
節ごとに進めていく。
まず，第2文節では，「システムアプローチの重要性」が説かれて
いる．規格の発行以前にも環境監査などの手法が用いられていた事
例もあったが，従来の環境監査はもっぱら環境パフォーマンス監査
の形で行われていた．すなわち監査の時点での環境パフォーマンス，
例えば廃棄物管理の状況あるいはエネルギー使用の状況などを評価
していた．これに対してこの規格は，廃棄物あるいはエネルギーに
関する管理などを含めた「システム」を提供するもので，このシス
テムを構築して，これを適切に運用していれば，環境パフォーマン
スが良好に保たれる，というのがこの規格の基本的考え方である．
次に，第8文節では，「継続的改善の重要性」が説かれている．こ
のシステムは組織の環境方針を実現するためのものである．方針に
は組織の目指すパフォーマンスレベルに関する意図が表明されてい
る．この規格では，パフォーマンスレベルの高低にかかわらず，そ
の到達点に向かって継続的に改善が進んでいくことが重要であると
の認識から，システム自体のなかに「継続的改善」が組み込まれ（義
β9－142 β9編 法 工 学

とこの「継続的改善」の2点がこの規格の最も重要な原則といえる．
さらに，第4文節には，「トップのリーダシップ」および「全員参
加」の原則が示されている．規格によれば，いかなるパフォーマン
スレベルを目指すかは環境方針に表明されるべきで，その環境方針

はトップのリーダシップがきわめて重要であることになる．またこ
の文節では同時に，組織全員の参画の重要性が記されている．トッ

第5文節では，この規格の適合性評価には，第三者機関による審

的なアプローチによって実現しようとする一つのモデルである．そ

善，法規遵守および汚染の予防の3点は必ず約束をしなければなら
ない．また方針は，目的，目標’ ﾝ定するための枠組（拠りどころ
あるいはガイド）を与えるものでなければならないし，その組織に
とって適切なものでなければならない．組織の種類，規模によって
活動の内容，環境に与える影響には大きな差を生ずるものであるが，
こうした点を環境方針は必ず反映しなければならない．そして方針
は文書化された上，システムを構成するすべての人に伝えられ，す
べての人がこの方針に沿って活動していかなければならない．なお，
ISO 14001では環境方針に限って外部に公表することが求められて
いる。

（1）環境側面 このシステムによって管理の対象となるのは，
その組織の持つあらゆる環境側面である．組織のなかであるプロセ
スを動かせば，エネルギーが必要となり，製品や廃棄物を産む．こ
のプロセスに与えられるインプットおよびこのプロセスから生ずる
アウトプットは，すべてこの組織の持つ環境側面である．また，産
み出した製品がエネルギーを使用するとすればこれも環境側面であ

料や部品を製造する部分）でも，それ相応の管理が求められる．

すなわち，どのような可能性に基づいて環境的な緊急事態が発生す
るかをあらかじめ予測して，そのような可能性をなくすること，お
よびそれでも緊急事態が発生してしまった場合に備え，それによっ

いる．
評価の結果から，影響の大きいものを著しい環境側面として重点
的な管理の対象とする．これを著しい「環境側面」と呼ぶ．
（2） 法的及びその他の要求事項 この段階では組織に適用さ
れる環境法規を特定する必要がある．それと同時に，環境側面や環
境法規は組織の事業活動の変化に応じて次々と変化していくはずで
あるので，それに合わせて絶えず最新化を図る必要がある．
（3） 目的及び目標 環境側面に対する影響評価の結果，影響の
大きいものから順次対策を練って，影響を削減していかなければな
らない．削減のための到達点が目的なり目標として設定ざれる．目

vi．経営層による見直し サイクルの最後の段階はAすなわち

これを行うのは方針を設定した経営トップである．，トップはこの
操作のなかで，システムの「妥当性，適切性および有効性」を評価
して，次のサイクルを有効にまわしていくための改善を加えなけれ
ばならない。この場で改善を加えることによってはじめて，継続的
改善が有効に実施されることになる．この見直しのためには，シス
テム運用に関する内外の情報が十分に，適切に集められていなけれ
ばならない．
さて，ここで「妥当性，適切性，有効性」の意味について若干触
れておきたい．ISO 14001規格は適合性評価の基準であり，適合性の
評価は通常第三者機関が実施する（自己宣言の場合には組織自身が
行う）．適合性の評価とはISO 14001規格の要求事項が満たされてい
るか否かだけの評価である．したがって，ISO 14001規格に適合した
システムがすべて有効なパフォーマンスを生むかというとそれは保
証の限りではない．そのシステムの要素の構成あるいは目標とする
パフォーマンスレベルの値，運用上の厳しさなどによってシステム
の有効性は大いに異なる．そこで，トップはこの見直しの段階で，
自らのシステムの「有効性」を評価して，次のサイグルに進む責任，
チャンスを与えられているのである．言い換えれば，適合性の評価
は審査機関に与えられているが，システムの有効性の評価はトップ
に委ねられているのである．
なお，「妥当性」とは，目的・目標のレベルあるいは方針の目指す
パフォーマンスレベルなどが妥当であるか否かを指し，また「適切
性」とは，そのシステムの構成あるいは適用の対象がその組織にと
って適切なものであるか否かを指している．これらも，「有効性」と
同様に経営トップの責任で評価されなければならない．
