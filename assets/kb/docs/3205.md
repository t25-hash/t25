# 7・1・4 日常点検

> **章**: 第7章 計測における管理と教育
> **節**: 7・1 計測機器の管理

日常点検（daily inspection，daily check）によって計測機器を
良好な状態に保つことは重要で，時には早期の調整・校正の必要性
の発見にもつながる．表7･3は長さ測定機器の日常点検項目および
定期点検（periodical inspection）項目の一例である．点検はす
べての測定機器について，かつ多くの性能項目について行うことか
ら始め，その状態によって定期点検項目を減じていく方向に変えて
いくようにする．この点検については，工程の重要性に従って記録

測定に適性をもつ人が必ずしも測定に従事しているとは限らな
い．適性いかんは人間の評価につながるので，つねに問題として残
されているといえる．また，測定技術は生産全工程と密接な関係に
あり，測定技術が単に計測だけを考えていればよいとはいえないこ
とを認識しなければならない．測定者（measurement operator）
または測定技能者（measurement technician）としての適性の把
握のためには次の事項 を条件として考えておくべきであり，参
考として挙げておく．
(１)一般作業者に比べて適性が高いと考えられる測定者の性
格，気質としては，次のようなものが挙げられる．
