# 7・2・1 計測技術者としての条件の把握

> **章**: 第7章 計測における管理と教育
> **節**: 7・2 計測技術者の教育・訓練

計測技術者には計測システムを完成させて優れた結果を導く能力,そして計測に関する多くの知識をもつことが要求される.計測技術者(metrologist,measurement engineer)の役割についてはいろいろな事項があるが,おもなものを挙げるとおよそ次のとおりである.(1) 技術的問題(a) 計測の計画(b) 測定における制約,制約条件の把握(c) 測定機器の選定:試料の形状,測定場所,必要精度,測定方法等(d) 測定手順,方法の組立:測定箇所,測定順序,データ個数,ジグ等の決定(e) 測定条件の決定:温度等環境条件の管理・決定,振動の有無の確認(f) 測定器の管理・校正:測定室の管理および状態,測定機器の管理,校正周期および校正方法の決定,補正方法の決定および活用(g) データのまとめ方(h) データの分析の吟味(2) 技術技能力にかかわる問題(a) 測定物の取扱いおよびセットの方法(b) 測定機器の取扱い(c) 測定の手順,手ぎわ(d) 測定者自身の心身の管理(3) 判断能力(a) 測定対象量,対象点選定の裏付け(b) 測定の不確かさの算出および決定(c) 測定結果とその問題点の検討(d) 結果に基づいての処置測定に適性をもつ人が必ずしも測定に従事しているとは限らない.適性いかんは人間の評価につながるので,つねに問題として残されているといえる.また,測定技術は生産全工程と密接な関係にあり,測定技術が単に計測だけを考えていればよいとはいえないことを認識しなければならない.測定者(measurement operator)または測定技能者(measurement technician)としての適性の把握のためには次の事項を条件として考えておくべきであり,参考として挙げておく.(1) 一般作業者に比べて適性が高いと考えられる測定者の性格,気質としては,次のようなものが挙げられる.(a) 注意深く,慎重で細かい点によく気づく(b) 正直で規則を守る(c) 責任感が強く几帳面(d) 忍耐強く真面目(e) 内向的(f) 精神的に安定(2)
