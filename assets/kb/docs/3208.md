# 6・2・5 環境ラベル

> **章**: 第6章 マネジメントと法制度
> **節**: 6・2 環境マネジメント

a．概要 ISO 14000シリーーズ規格において，環境ラベルに関す
る規格は，図6・3に示したように，ISO 14020，14021，14024，14025
の4件である．
i．環境ラベルの狙い 自由競争の市場社会において，各種の
製品は工夫をこらしてその特徴を発揮しようとする．従来の製品の
コンセプトには，機能，コストなどは含まれていたが，環境への視
点が盛り込まれたものは少なかった．製品の環境に与える負荷は，
原料・材料の生産，製品製造，販売・流通，使用および廃棄のそれ
ぞれの段階で発生し，製品によってどの段階での負荷が高いかは異
なる．環境ラベルの考え方は，これら製品のライフサイクルを通し
て環境影響に配慮されたものを評価し，差別化しようとするもので
ある．
環境ラベルは，14001規格とは直i接的には関係ないが，製品の環境
側面を改善していく上では有効なツールである．
ii．環境ラベルの三つのタイプ 環境ラベルは世界の各国でさ
まざまな形で実用化されているが，ISO／TC 207ではこれらを三つ
のタイプに分類して，それぞれのガイドラインをまとめ上げた．そ
の一つ目は，マネジメントシステムの認証と同じように，第三者機
関がある基準に基づいて評価した上で与えるもので，タイプ1と呼
ばれている．二つ目は，その製品を生み出す組織自身が評価を行っ
た上で自発的な宣言の形で表現するもので，このなかには省エネル
ギー型，再生可能など12のクライテリアがある．これをタイプHと
呼ぶ．三つ目は，製品の製造に要する資源，エネルギー，あるいは
化学物質等の使用量などの環境情報をラベル化して製品に表示する
もので，タイプ皿と呼ばれている．
iii．環境ラベルの一・般原則 ラベル表示が適切でなかったり，
評価が不当である場合には，自由競争の原則を破壊しかねないし，
特に貿易障壁の原因となる懸念がある．こうした点に配慮して，ISO
14020の「環境ラベルの一般原則」のなかでは，すべてのタイプのラ
ベルに共通する下記9原則を重視すべきであると記されている．
① ラベル・宣言は正確，検証可能，適切で誤解を生じないこと．
②ラベルの表示手続要求事項は貿易障壁の誘因となってはな
らない．
③ラベルによる主張の裏付けは正確かつ科学的でなければなら
ない．
④ ラベルの証認判定の根拠となる情報は利害関係者が入手可能
なこと。
⑤ラベル・宣言の評価基準には製品のライフサイクルにおける
環境側面すべてが配慮されていること．
⑥ ラベル・宣言は環境パフォニマンスの維持・改善および技術
革新を抑制してはならない．
⑦ ラベル運用上の要求事項は適用判断基準および評価基準の確
立に必要なものに限定される．
⑧ ラベル・宣言の評価基準開発プロセスには情報の一般公開お
よび一般のコンセンサスを得る努力が強く望まれる．
⑨ ラベル・宣言の対象は製品の持つ環境側面に関する情報の入
手が購入者にとって可能でなければならない．
b．各環境ラベルの特徴
i．第三者機関認証型ラベル…タイプ1 このラベルは，製
品・サービスを提供する企業や購買者から独立した第三者機関が，
決められた製品カテゴリと判定基準によって審査を行い，合格した
製品にラベルの使用を認めるものである．ドイツ，北欧を中心にし
て世界中に25以上の実例があり，日本国内では日本環境協会による
エコマークが68品目の製品を対象に実施されている．
すでに実施されているこれらのラベルの運用を統一するρは難し
いので，ISO 14024規格は個々の製品分野や判定基準を規定すること
はやめて，新しいラベルの開発や運用に関するガイドラインとなっ
ている．ISO 14024規格には，自主参加の原則に始まって，法規遵守，
ライフサイクル的思考，差別化，環境基準，製品機能，検証期限，
透明性，科学性などを含めて18の原則が列記されている．この規格
は，環境面で優れた製品に対して，他の製品との差別化を行った上，
有利な条件を与えようとするものであるから，途上国への差別，非
関税貿易障壁の発生など種々の問題を産みかねない．これを避ける
目的で，環境基準選定のマトリックスが表6・3のように示されてい
る．内容は空欄のままであるが，ここには資源の利用あるいは汚染
に関する指標が記述されることになる．

ii．自己宣言型ラベル…タイプ皿 このラベルは，製品を提供
する企業が自己宣言によって行う環境主張である．したがって必ず
しも製品に貼付するとは限らず，種々の媒体を通して主張すること
が可能である．
ISO 14021規格には，主張項目，主張を適用するにあたっての要求
事項，評価・検証のための要求事項および主張項目ごとの特別要求
事項などが規定されている．自己宣言による主張であるが，製品の
製造者がこの規格に則って自己宣言を実施するためには，ここに規
定された要求事項は守らなければならない．
自己宣言によって主張できるのは次の12項目である．
①コンポスト化可能②分解可能③解体可能設計
④長寿命製品⑤回収エネルギー⑥リサイクル可能
⑦リサイクル材含有率 ⑧省エネルギー消費 ⑨省資源型
⑩節水型⑪再利用・再充填型⑫廃棄物削減型
iii．情報開示型ラベル∴・タイプ皿 このラベルは，製品単位当
りの環境関連情報として，資源・エネルギーの使用量，汚染物質の
排出量，廃棄物発生量，化学物質あるいは有害物質の使用量などを
数値データとして，一覧表もしくは棒グラフなどの形で製品に表示
するものである．通常，薬品，食品には成分表が付けられているが，
これに加えてエネルギーや廃棄物のデータを付け加えた形を想像す
れば良い．タイプ皿ラベルでは生のデータがそえられるが，単に表
示するのみでなく，第三者の評価が可能なように表示する必要があ
る．
タイプ皿ラベルの実用例としては現在，アメリカのSCS（Scien・
ti丘。 Certi丘cation Systems）やカナダのテラチョイス社のEPDS
（Environmental Profile Data Sheet）などがある．
タイプ皿の標準化にあたっては時期尚早との意見も多く，とりあ
えず，ISO 14025は関連技術資料を盛り込んだ技術レポート（TR）
として発行されている．
