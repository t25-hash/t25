# 7・2・3 教育・訓練の方法

> **章**: 第7章 計測における管理と教育
> **節**: 7・2 計測技術者の教育・訓練

教育・訓練は企業の人づくりの目的に基づき行うものであるが,各種の制限が加えられる.制限としては以下のものが挙げられる.(1) 教育費用に見合う効果を上げる.コストを下げる.(2) 教育時間の短縮を図る.この制限のもとで,教育・訓練の原則は,次の事項を守ることである.(1) 教育・訓練を受ける人を受身でなく,自らの行動とさせる.(2) 教育・訓練を受ける人に訓練中行ったことが正しかったかどうかをなるべく早くフィードバックさせる.(3) 教育・訓練を受ける人がたどるべきコースの概要を明らかにし,そのコース上で細かいステップで目標に到達できるようにする.(4) 能力に応じたスピードで行い,能力以上のことを強制しない.しかし,能力をわずかに上回る程度に組立てることも考慮する.計測に関する教育・訓練の内容は,およそ次のようなものである.技術管理者(technical manager)のためのものと測定技術者(measurement engineer),測定技能者(measurement techni-cian)のためのものとの間の違いがあるのは当然で,受講対象者によって内容を分けて行うようにする.(1) 計測における基礎(物理学)の理解(2) 測定機器の原理・構造の理解(3) 測定機器の取扱方法の把握(4) 測定法に対する熟練(5) 測定機器の保守・管理の方法これを個々の測定機器について編成することによって,教育・訓練を行う.例えば,マイクロメータを見ると次のようになる.(1) マイクロメータの歴史(2) マイクロメータの原理(3) マイクロメータの規格(4) マイクロメータの構造と用途(a) 外側マイクロメータ(標準,バーニヤ付き,ミクロン,直進式,ダイヤルゲージ付きなど)(b) 内側マイクロメータ(棒型,キャリパ型,換えロッド型,つぎたし型,3点式など)(c) デプスマイクロメータ(d) ハイトマイクロメータ(e) マイクロメータヘッド(f) 指示マイクロメータ(g) ディジタルマイクロメータ(5) マイクロメータの測定誤差(a) 測定において構造上生じる誤差(b) 使用において生じる誤差(6) マイクロメータ使用上の注意(7) マイクロメータの検査(a) 概観,機能(口元がた,ねじ遊び,はめあい,ラチェットストップ,フリクションストップ,目盛,きず,さび,クランプ,外筒の硬さ,シンブル位置,外観)(b) 性能[測定面の平面度および平行度,器差(指示の誤差),測定力](8) マイクロメータの保守・管理(a) マイクロメータの管理(b) 定期検査と補修(c) 保管法(9) マイクロメータの将来5―148計測工学内容を見れば知識の習得が多いといえるが,これらを座学だけで行ったとしても教育・訓練にはならない.まず,(4)において,実物に触れて学ばせる.これをもとに,測定の目的と構造から見て,特に(5)~(7)の実習を行わせることを計画する.次に,この(5)および(7)を中心として,マイクロメータを用いて実際の測定物,あるいは測定物に近い模擬品を数回測定させてみることである.それによって測定値にばらつきがあることを知り,取扱いによっても値が異なってくることも知るだろう.測定は正しい測定を行うことが基本であるから,性能との関連で取扱いを学んでいく.数人いるなら,その者たちの間で回し測定をすれば,目盛の分割読みの仕方,視差等の原因に基づく読みの違いなど人による特徴があることをはじめ,個人差の存在を知ることができる.測定物の持ち方や,保持時間の長さなどの影響も現れるかもしれない.そのうえで,(7)の検査に対応する性能を維持するためにその検査の重要性と具体的な対処によって,性能維持法を認識することになる.終わりに測定結果をまとめさせる.実習の与え方によって,かなり異なった習熟度となるので,計画を十分練ることが大事である.このようなことをまとめて,教育・訓練の方法としての基本的な方法を示すと,以下のようになる.(1) 講義または説明,質疑応答や意見交換を行う.(2) 実演して見せ,それをもとに,実習,演習を行う.(3) さらに参加者間での話し合いを行う.(4) 結果と測定演習を参加者にまとめさせる.(5) 最後に教育指導者が意見を述べてまとめる.教育・訓練と成績評価においては,事前に対象物の測定あるいは試験,検査を行っておき,被教育・訓練者に知らせずに演習をさせ,その値を分析するのがよい.その結果により対応を変える.測定機器の実習によって,触れる楽しみとともに体験的に正しい取扱いを知ることが可能になり,さらに有効かつ迅速な測定を自らのものにしていくことができる.机に向かっての勉強に偏らないよう,実際に手に触れての測定実習(measurement practice)を取り入れることが教育・訓練の基本である.以上のようなことを基本とするが,教育・訓練には,職場訓練,事例研究,役割演技,模擬訓練,交差訓練,巡回訓練などがあるほかに外部機関への委託なども考えられる.測定担当者の教育・訓練に当たっては,製造工程における現実の計測問題に対応させて行うべきである.したがって,このような職場訓練(shop instruction,shop training)はOJT(on-the-jobtraining)によることになる.OJT は,その職場の熟練者によって行うことが多い.このような場合,その熟練者が必ずしも教育・訓練を実行するための適性をもっているか否かわからず,適していない場合には,問題を残すことになる.
