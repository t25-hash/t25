# 7・2・4 教育・訓練の効果

> **章**: 第7章 計測における管理と教育
> **節**: 7・2 計測技術者の教育・訓練

教育・訓練の効果は、その終了直後に以下の効果を計ることで見るようにする。(1) 教育・訓練を受けた者に生じた効果(2) 教育・訓練を担当した者に生じた効果(3) 教育・訓練を受けた部門全体に与えた効果効果の判定は教育・訓練を行う前と後との比較、教育・訓練を受けた者と受けない者との比較などによって、測定項目を定めて、それに従って行うことが多い。効果がただちに現れればよいが、測定技術の内容によって、また目的によって効果の現れ方に違いがあるので、時間を見なければならない場合がある。受けた本人に現れるほか、周囲に影響を与えることもあり、担当グループ全体での効果も見る必要がある。そのため、計測のあり方に対する企業、または部門における扱い方の変化、測定能率の向上、測定誤差の減少、クレームの減少などの変化、費用の軽減、測定関連仕事に対するモラルの向上など、仕事上にどう現れるかを調べるべきである。これらの調査は追跡調査やアンケート、時にはテストや報告書などを用いて行うことが普通である。この結果を教育・訓練の修正および再計画の資料とするため、データとして記録保存することが望ましい。
