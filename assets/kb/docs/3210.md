# 7・2・5 測定データへの人間の影響と教育

> **章**: 第7章 計測における管理と教育
> **節**: 7・2 計測技術者の教育・訓練

測定機器を取扱った教育・訓練をもとに、人間の作用の効果(human effect)および教育について示す。a. 先入観人間には多かれ少なかれ先入観(preconceivedopinions)がついて回る。例えば、マイクロメータで機械部品の寸法を測定したり、位置の読取りを行うとき、測定者は数回繰返してラチェットを回してから読取りを行い、さらにもう1回測定を行うことが多いが、そのとき、測定者は第1回めに読み取った値にかなり左右されているということができる。第2回めの値が第1回めと違う場合には、その値は記録しないで、第3回めの測定に入り、第1回めの読みと大差ないと判断されたとき、記録を行うことが多い。しかしその場合、第2回めの値のほうが正しかったかもしれないのである。工場の検査において、一般に熟練者(skilledworker)がいて、その人の値に合わせるように教育・訓練は行われるため、訓練後もその熟練者の値に合った測定値となることが多いのはすでによく知られたことである。企業が異なり、測定者が違うと、測定値は異なり、企業の中央値が真と思われる多くの機関の平均値と違っている場合があることを、プラグゲージ等の国内持回り測定値の結果が示している。OJT は良いのであるが、7・2・3項で述べたことはこのような問題も含んでいる。図7・2は教育・訓練の実験例の一つの構成である。測定者は望遠鏡をのぞきながら、ひし形の角隅部分、および正方形稜(りょう)線の一つに望遠鏡の標線を、シンクロサーボ発振器のつまみでマイクロメータを回し、合致させている。この作業を測定機器のもつ測定顕微鏡で行うこともある。測定者は前もって、それぞれの合致の状態を見本で見て、合わせ方を教わった後に操作に入る。その測定結果が図7・3である。繰返し測定約10回、日を変えて再び繰返し10回が、図7・3の左寄りに示された訓練前の反復1、反復2である。その後、正しい位置を教わった後の測定操作の結果が訓練中のデータである。図7・3から正しい値を教え、測定結果がフィードバックされたときの誤差は小さくなっていることがわかる。これは測定機器の操作の教育・訓練の必要性、および正しい値を教えることの重要性を示す一例である。図7・2 権威とデータ光速度は物理定数として、長さの単位メートルの標準の定義に用いられるようになった。その値が1983年に国際的に取り決められるまでの長い間、多くの研究者が光速度の測定を試み、約55回の研究報告がなされている。表7・4は1906年の第17回から1950年の第29回までの報告と、物理定数として固定された値とを示している。この中で第21回から第25回、および第27回の値は、その前後の測定値と違うとともに、1983年に定義された値とも異なる値となっている。これは、第21回の値がマイケルソン(Michelson)の値だったからである。マイケルソンは、メートル原器の長さとカドミウム赤色光の波長とを最初に比べた研究者で、ノーベル賞を受賞し、また、いわゆるマイケルソン光波干渉計の構成者として、その道で知られた偉大な研究者である。この威光に影響されて、第22回から第27回までの光速度の値はマイケルソンの値に引きずられた値となったのである。このことは、科学的客観的データとされた測定値が、測定者の測定結果に対する自信のなさや状況判断に左右されて、あるべき値になっていないことを示している。企業において、社長や社内的に地位の高い人の発言に測定値が左右されやすいことも、知られている。c. 社会的信頼性のあるデータと教育a,b項から、人間の測定データへの関わりは、測定者・個人差というものだけに留まらな5―149いことがわかる。計測を計画する人の能力に左右され、測定の自動化によっても誤りを犯すことがある。測定の目的や許容される誤差がトップに立つ人の考え方に左右されるなど、測定値は測定者によって任意に変えることができる状態にある。初めから自分たちに不都合なデータが出ない測定システムを構成して測定を行うことも多い。安全に関係する試験検査でさえ、他部署から束縛されることもある。人間の故意の操作(intentional act,intentional operation)による問題をいくつか示す。日本の中規模の製鉄メーカで、JIS 指定商品のデータを改変した事実がわかり、社会的に問題になったことが、1989年11月9日の日刊工業新聞に掲載された。ノーベル生理学・医学受賞者であるアメリカ、マサチューセッツ工科大学のD.ボルチモアが、ある論文のもととなるデータの一部を、後から改変して挿入したという鑑識結果によりアメリカ下院エネルギー商業委員会委員長J.ディンゲルから告発されたという報道が、日本経済新聞1990年5月21日(11版、17面)に、またアメリカ、ベル研究所のH.シェーンの論文中にデータの偽造があり、アメリカの科学誌サイエンス中の論文8件が撤回されたという報道が、2002年11月1日読売新聞夕刊(4版)にあった。人間は、不都合なデータを除いて有利なデータをそろえ、極端な場合にはデータをねつぞうするに至る可能性をもっている。だがこれは心の問題であり、計測担当者にとって、絶対に行ってはならないことである。計測は科学的に行われて数値が示されると理解されており、そのために信頼されている。にもかかわらず、上述のようなことが行われてしまうことは、極めて残念ではあるが、計測が社会的側面をもっていることを示している。ただし、偽造データ(forgeddata)は、その場ではそれがなし得たように見えても、やがて科学的に、また技術的に不正が見つけられてしまうのがつねである。以上のように、計測においても人間そのものが問題で、その人となりや、人間の心が重要であるということができる。これは計測の教育以前の問題となるものであるが、そのうえで、正しい計測の実行には、適切な教育が大切である。上記の悪い事例などをよく示したりすることによって、測定とその測定値の意味を十分に修得させるようなしっかりした教育が、極めて大切になる。計測に与えられる外的制約の中には法的問題もあり、計測に当たる者が合理的に考えて、解決しなければならないのである。文献(1)
