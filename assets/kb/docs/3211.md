# 1・1・1 伝達関数

> **章**: 第1章 システム解析
> **節**: 1・1 システム表現

ここで、式(1・1)における≧は、物理法則に従った関係であること、いわゆる因果律を満たしていることを意味している。このように、線形微分方程式で表されるシステムの伝達関数は、分母も分子も複素数に関する実係数多項式の分数関数(有理関数)になる。入力や出力が複数である場合、伝達関数は行列形式で表現されるが、これについては1・1・2項で述べる。b. ブロック線図とその結合制御系の構造を表現するのに、ブロック線図(block diagram)が用いられる。ブロック線図は、制御系における信号や情報の流れの方向を表す矢印、構成要素のシ6―1ステムを表すブロック、信号の伝わり方を表す加え合わせ点と引出し点からなり、それぞれ図1・2に示した記号が用いられる。ブロックは制御対象や制御器などの制御系構成要素の入出力特性を表し、入力と出力の矢印にはそれぞれの信号の記号が、箱にはふつう伝達関数が記入される。加え合わせ点は、複数の信号の和を出力するシステムで、正で加算するか負で加算するかによって、入力の矢印の先に+または-の符号が付けられる。引出し点は信号の分岐を表すシステムで、その出力はすべて入力と同じ信号である。図1・3は、フィードバック制御系の典型的な構成をブロック線図で表現したものである。このように、複数のブロックが結合しているとき、その全体の伝達関数を計算することができる。図1・4に、代表的な結合とそれと等価な伝達関数を示したが、ここで、負のフィードバック結合の場合について、その計算法を述べておこう。結合されたブロックの伝達関数を組織的に計算する方法にメーソン(Mason)の公式があるが、よほど複雑なものでなければ図1・4の利用や上記の計算法で十分計算できる。図1・2 ブロック線図の基本要素図1・1 システム図1・3 フィードバック制御系のブロック線図6―2図1・4ブロックの結合と等価な伝達関数c. 周波数伝達関数(frequency transfer function)伝達関数は対象システムが不安定であっても定義できるシステム表現であるが、安定なシステムの場合は周波数伝達関数でも表現することができる。周波数伝達関数は、システムに正弦波入力を加えたときの入出力関係と密接な関係がある。この角周波数ωに対するゲインと位相の関係を周波数応答(frequency response)という。周波数応答は、微分方程式が立てられないシステムの伝達関数を実験的に求めるのにしばしば利用される。また、周波数応答の図的表現にボード線図やゲイン位相線図などがある。d. パルス伝達関数(pulse transfer function)サンプル値制御では、サンプリング周期ごとの信号値に注目して、システムの入出力関係を表すことがある。このように、時間に関して離散的な信号で特徴付けられるシステムは、伝達関数表現されるシステムが連続時間システムと呼ばれるのに対し、離散時間システムと呼ばれる。パルス伝達関数についても、分母多項式=0、分子多項式=0を満たす複素数をそれぞれ極、零点という。ここで、変換について補足しておこう。上でパルス伝達関数を導くのに用いたのは、この性質である。変換についての詳細は、例えば文献(1)を参照されたい。さて、パルス伝達関数の物理的意味を考えてみよう。このインパルス信号は数学的にはクロネッカのデルタで与えられており、連続時間信号の単位インパルスがディラックのデルタ関数であるのとは異なることに注意してほしい。ところで、パルス伝達関数はもともとサンプル値制御系の解析のために定義されたものであり、システム表現としての本質を把握するためには、サンプル値制御系の理解が必要になる。詳しくは、文献(2)を参照されたい。
