# 6・3・2 環境影響評価の手順 ここでは，環境影響評価法に

> **章**: 第6章 マネジメントと法制度
> **節**: 6・3 環境アセスメント

基づく環境影響評価の手順について，図6・10のフロー図にしたがっ
て述べる．
（1） 事業の概要 事業者は，事業の概要を取りまとめて，当該
事業が第一種事業か第二種事業かの判定を行う．第一種，第二種の
事業規模は事業の種類ごとに表6・4に示すように決められている．
一定規模以上の事業（第一種事業）を行うには環境影響評価手続を
必ず実施し，これに準ずる規模の事業（第二種事業）については主
務大臣が知事意見を聞いて環境影響評価を行うかどうかを決めると
定められている．以上の手順はスクリーニングと呼ばれている．
（2） 方法書の作成 事業者は，環境影響評価方法書（環境影響
評価計画書と呼ぶ場合もある）を作成し，主務官庁または自治体へ
提出する．方法書は，事業区域およびその周辺における自然的状況
と社会的状況を踏まえて，選定した環境影響評価項目ならびに調査，

価書の原案にあたる環境影響評価準備書を作成し，主務官庁または
自治体へ提出する．これは公示・縦覧され，住民や知事が意見を述
べることができる．
（5） 評価書の作成 事業者は，これらの意見を踏まえて準備書
を訂正し，環境影響評価書を作成し，主務官庁または自治体へ提出
する．評価書は，各環境影響評価項目に対し，現況調査結果，予測
結果および評価結果を詳細に取りまとめ，事業の環境に及ぼす影響
の評価の結論を提示したものである（詳細は6・3・5項b参照）．こ
の評価書に対し，環境大臣は意見を述べることができ，主務大臣は
意見を考慮して評価書の修正や補強を事業者に申し出ることができ
る．事業者は最終的な修正を：行う．
（6） 公示，縦覧 事業者は，環境影響評価書を公示，縦覧しな
ければならない．公示縦覧後，主務大臣は事業者に事業の許可（免
許など）を与え，事業が開始される．

が望ましい．過去に，地方自治体の環境影響評価条例に基づいて，
事業者が「事後調査」を求められた例もあるが，環境影響評価法では，
「事後調査」を義務付けてはいない．
