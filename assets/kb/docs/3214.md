# 1・1・4 ビヘイビアアプローチ

> **章**: 第1章 システム解析
> **節**: 1・1 システム表現

a. システムの記述ビヘイビアアプローチ(behavioralapproach)では時間領域における信号の軌跡の集合をシステムとして取扱う。その部分集合であるビヘイビアがまさにシステムの定義であり、その内包的記述がシステムのモデルである。モデルの記述には微分方程式や不等式などが用いられるが、一般にその記述形式を表現(representation)と呼びシステムとは区別する。すなわち、システムが同じであるということはビヘイビアが同じということであり、一つのシステムに対して無数の表現が存在する。システムの有限次元性は数学的にはの点収束の意味での完備性と関連し、工学的には予測可能性と密接な関連がある。これは多少の準備を必要とするのでここでは省略する。文献(7)を参照されたい。この場合、外部変数からなるはマニフェスト信号(manifest)空間と呼ばれる。6―6制御システムこのような定義を基本として線形システム理論の諸概念の体系付けが可能である。この定義は、状態変数∈を用いるとシステムの挙動を過去、現在、未来に分離可能であることを意味している。これは軌道設計に十分な自由度があることを意味している。すなわち、不可制御であるということはシステムのふるまいがある軌道面に拘束されており、設計の自由度がない状況を意味している。これを像空間表現(image representation)と呼ぶ。ここで、:新たに導入されたレイタント変数。導出方法から明らかなように、は列フルランクである。これは、からが一意に定まることを意味している。像空間表現されたシステム式(1・62)ではマニフェスト変数からレイタント変数が観測可能である。g. 因果性(causality)マニフェスト変数を=×のように二つの信号の直積とする。上記のような因果性が成り立つとき一般にを入力、を出力と呼ぶ。a~f項の議論において必ずしも因果性を必要としなかったことに注意すべきである。すなわち、ここまでの議論は1・1・1~1・1・3項までの議論を因果性を前提としない場合に拡張したことになっている。これを多変数系に拡張すると左規約分解表現が得られる。線形システム論は、線形系の表現(線形状態方程式など)の数学的性質から組立てられる理論体系である。本節の内容について詳しくは、文献(15)~(18)などを参照いただきたい。
