# 1・2・2 可制御性、可観測性、最小実現

> **章**: 第1章 システム解析
> **節**: 1・2 線形システム論

このとき入力はどれほど自由に状態を動かす能力をもっているのか、また出力はどれほど状態に関する情報を提供しているのか、これらの基本的な問いに答えるために、可制御性、可観測性という概念が必要になる。入出力関係が同じになる状態方程式は無数にあるが、それらの中で状態ベクトルの次元が最小になるものを考えることができる。これが最小実現である。a. 可制御性と可観測性の定義線形状態方程式(1・72)(1・73)の可制御性、可観測性を次のように定義する。[定義1]状態方程式(1・72)を考える。[定義2]状態方程式(1・72)(1・73)を考える。この観測と矛盾しない状態が唯一に定まるとき、式(1・72)(1・73)は可観測であるという。b. 可制御性、可観測性条件定義1、定義2で定めた可制御性、可観測性は、実は係数行列,のみに関する代数的な必要十分条件を与えることができる。モニック多項式は最高次数の係数が1である多項式のことである。[定理3]式(1・72)が可制御であるためには、以下の等価な条件のいずれかが成り立つことが必要十分である。定理3の(1)(2)より、定義1の可制御性の定義において、時刻<に関しては何の割約もないことがわかる。つまり、可制御条件が成り立つならば、状態を移動するための時間間隔は、任意にとることができる。これは可制御性の定義には入力の大きさに制約がないためであり、現実の制御では時間間隔をいくらでも小さくできるものではない。条件(1)の行列を可制御性行列という。条件(2)をポポフ・ベレヴィッチ・ハウタスの判定法という。この条件は次のようにも解釈できる。もしλがの固有値でないならば、行列λ-は正則となるので、条件にある行列の階数を落とす可能性のある複素数λはの固有値である。つまり条件(2)は、の固有ベクトルξ≠0でξ=0を満たすものはない、と言い換えることができる。条件(3)の被積分関数は、準正定行列であるので、もし積分がある区間で正定となれば、それを含む区間でも正定となる(実際にはどのような区間でも正定となる)。条件(4)は、状態フィードバックによる極配置可能性を示している。[定理4]式(1・72)(1・73)が可観測であるためには、以下の等価な条件のいずれかが成り立つことが必要十分である。定理4の(1)(2)より、定義2の可観測性の定義において、時刻<に関しては何の制約もないことがわかる。これは一種の双対を示しており、線形システム論ではしばしばこの事実を用いる。定理4の条件(1)の行列を可観測性行列という。条件(2)をポポフ・ベレヴィッチ・ハウタスの判定法という。これはの固有ベクトルξ≠0 可制御性と可観測性の意義状態方程式(1・72)(1・73)が可制御または可観測であることは、定義1、定義2において、制御の観点から定義されたが、定理3、定理4において、代数的に等価な条件として特徴付けられた。これらを制御の観点から意義付けをすると、状態フィードバックによる極配置可能性、オブザーバという状態観測機構の構成可能性と関連していることがわかる。オブザーバについては、2・2節で述べることとする。これは、定理1からもわかるように、その大まかなふるまいは+の固有値によって規定される。安定性については、1・3節を参照されたい。このようにして、状態フィードバック系が任意に設定する固有値をもつようにすることができる。d.
