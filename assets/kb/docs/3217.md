# 1・2・3 平衡実現

> **章**: 第1章 システム解析
> **節**: 1・2 線形システム論

a. 可制御性グラミアン(controllability Gramian)、可観測性グラミアン(observability Gramian)状態方程式(1・72)(1・73)で記述される可制御かつ可観測な系を考える。ただし、行列の固有値の実部はすべて負であるとする。行列を可制御性グラミアン、行列を可観測性グラミアンという。可制御(または可観測)であるかないかは定理3、定理4からもわかるように入力と状態(または状態と出力)の間の二値的な性質である。ところが、以下に示すように可制御性グラミアン、可観測性グラミアンを考えると、入力から状態が制御しやすいか、出力から状態が観測しやすいかといった二値的でない情報を与える。まず状態方程式(1・72)で初期条件を=0とし、初期時刻は十分小さいと考えて、-∞の極限を考える。このとき以下の定理が成り立つ。[定理9]状態方程式(1・72)(1・73)が可制御(または可観測)であるためには、可制御性グラミアン(または可観測性グラミアン)が正定であることが必要十分である。また、初期条件がの大きな固有値に対する固有ベクトルの方向に向いているほど、式(1・95)の値は大きい。このような状況のとき、状態の初期条件は出力により大きく影響を与える。また、終端条件がの小さな固有値に対する固有ベクトルの方向に向いているほど、式(1・94)の値は大きい。このような状況のとき、入力は状態に影響を与えにくくなっている。つまりグラミアンは(相似変換ではなく)合同変換を受けるので、それらの固有値は正則行列に依存する。これは座標変換によって、入力と状態の結び付き、状態と出力の結び付きが変化することを示している。グラミアンの積の固有値の平方根をハンケル特異値(Hankel singular value)という。同値変換を用いて、可制御性グラミアン、可観測性グラミアンの双方を同一の対角行列に変換することを考える。グラミアンが式(1・96)を満たすような状態方程式による実現を平衡実現という。定理9から平衡実現の状態ベクトルは、入力と状態の結び付き、状態と出力の結び付きがバランスしているといえる。平衡実現を与える正則行列による同値変換が存在することは、1・2・3・c項で計算方法を与えることにより示す。平衡実現は、ディジタルフィルタにおける有限語長の問題(四捨五入による丸め誤差の問題)に対処するために文献(23)で提案された。c. 平衡実現の計算方法状態方程式(1・72)(1・73)は、可制御かつ可観測であり、行列の固有値の実部はすべて負であるとする。可制御性グラミアンおよび可観測性グラミアンを式(1・89)および式(1・90)によって定める。d. 平衡打切り近似伝達関数が与えられたときに、次数のより小さな伝達関数による近似が必要な場面は多い。例えば、制御系設計のためのモデル化では、高次のモデルはその補償器設計が難しくなるので、低次のモデルで近似したいことが多い。また、補償器についても低次の補償器のほうが実装しやすい。平衡打切りは、系の近似方法の一手段であり、文献(24)で提案されている。式(1・72)(1・73)は平衡実現であるとする。この項では簡単のため、係数行列にバーを付けないことにする。可制御性グラミアン、可観測性グラミアンは式(1・89)(1・90)で定義され、式(1・91)(1・92)を満たしており、==Σと式(1・96)の対角行列と一致する。式(1・91)(1・92)が==Σによって満たされている。リアプノフ方程式(1・101)(1・102)の性質より、の固有値の実部は零以下になる。さらにやや込み入った議論をすると、それらが負であることを示すことができるので、は安定行列である。このため、ステップ応答の定常値は近似したあとで一般には保存されない。一方、=0での級数展開を基に近似するパデー近似では、安定な伝達関数の近似が不安定になることもありうる。平衡打切りでは定理10に示したように近似系も安定な伝達関数をもつ[このことは文献(25)で示された].また平衡打切りでは、周波数領域での誤差の上限が式(1・100)によって保証できることに注意したい。このためハンケル特異値が十分小さい部分を切り捨てても、誤差は切り捨てた特異値の2倍以内でおさまる。また、定理9からハンケル特異値は状態を介入した入出力関係の大きさを計っているので、平衡打切りは、入出力関係に寄与の小さな状態を切り捨てる近似方法だといえる。
