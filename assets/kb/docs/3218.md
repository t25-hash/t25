# 1・2・4 幾何学的方法

> **章**: 第1章 システム解析
> **節**: 1・2 線形システム論

a. 数学的準備幾何学的方法においては、線形システムの性質・構造の解析や構成問題の可解性などに関して、さまざまな部分空間による特徴付けがなされる。このため、まず関連する数学的準備から以下にまとめておく。体上の線形空間の部分集合を考える。部分空間は和とスカラ倍に関して閉じている。したがって、部分空間自身も線形空間である。これは、和集合∪ではないことに注意する。また、部分空間の共通部分を∩とする。線形空間とその部分空間が与えられたとき、+=かつ∩=0となるの部分空間をの補空間という。このとき、=と表してはとの直和であるという。このように定義されるは部分空間となる。像は写像によって写された要素の全体からなる集合であり、零空間は写像によって零元に写される要素の全体からなる集合である。線形空間の線形変換を考える。また、を-不変部分空6―12制御システム間であるという。線形空間の線形変換を与え、それが不変部分空間⊂をもつとする。作用素の制限はの線形変換になり、の固有値を考えることができる。線形空間とその部分空間を考える。同値類の全体をと書いて商集合という。元の線形空間の和とスカラ倍を用いて商集合にも和とスカラ倍を与えることができる。このようにして作られた線形空間を商空間と呼び、記号が重複するがと書くことにする。さらに、dim=dim+dimが成り立つ。これはを含む最小の-不変部分空間である。これはKerに含まれる最大の-不変部分空間である。これを、制御不変(controlled invariant)ということもある。この,-不変部分空間に属する任意の初期状態に対して、ある制御入力が存在し、つねに状態トラジェクトリーをこの部分空間内にとどめておくことができる。これを、条件不変(conditioned invariant)ということもある。ここで、は新たな入力である。この可制御性部分空間に属する任意の初期状態に対して、ある制御入力が存在し、つねに状態トラジェクトリーをこの部分空間内にとどめながら、ある有限の時刻で原点へ移すことができる。この可安定性部分空間に属する任意の初期状態に対して、ある制御入力が存在し、つねに状態トラジェクトリーをこの部分空間内にとどめながら、閉ループ系を安定化できる。. DDPまず、すべての状態が利用可能であるとする。これを達成する制御問題を、外乱非干渉化(DDP: disturbance decoupling problem)、あるいは局在化(localization)という。この問題を、出力フィードバックによる外乱非干渉化(DDPM : disturbance decoupling problem by measurement feed-back)と呼ぶ。このように、幾何学的方法により外乱非干渉化問題の可解性などが線形システムに関連する部分空間により明解に特徴付けられる。制御システムを設計するときにまず確保しておかなければならな6―13いことは、システムの安定性であろう。1868年電磁気学で有名なマクスウェル(Maxwell)は論文『On Governers』で、ガバナによって回転速度が調整された蒸気機関の不安定現象(ハンチング)を解析し、安定に動作させるためには、微分方程式の特性多項式の根が複素左半面に存在するように設計しなければならないことを指摘するとともに、三次の多項式までについてその条件を導いた。これが安定性に関する厳密な数学解析の始まりだといわれている。一般の次多項式について、その根がすべて複素左半面に存在するかどうかを多項式の係数だけから判定する方法はラウス(Routh,1876)とフルビッツ(Hurwitz,1895)によって導かれ、現在も有用な方法として使用されている。線形システムの安定判別に関する大きな進展は、1920年代後半から1930年代にかけてのアメリカにおける長距離電話技術の発展に根ざしている。長距離有線通信を達成するには安定でひずみの少ない増幅器が不可欠であるが、天才的な回路技術者のブラック(Black)が負フィードバック増幅器を発明し、この安定性についてナイキスト(Nyquist)らのベル研究所の頭脳が、現在ナイキストの安定判別法として知られる安定判別法を世に出した。この方法は図式的に安定判別する方法であることから幾何学的な安定判別法と呼ばれている。それに対してラウス・フルビッツの方法は代数的安定判別法と呼ばれている。ラウスやフルビッツとほぼ同時代にロシアの数学者であったリアプノフ(Lyapunov,1892)は、非線形微分方程式を対象にしてその安定性を研究し、現在リアプノフの第一の方法、第二の方法と呼ばれる安定判別法を彼自身の博士論文の中で提案した。1960年代に非線形システムの安定性を微分方程式という束縛から離れて、入出力作用素という立場から新しく構築する研究が活況を呈し、現在、受動定理とスモールゲイン定理という形で知られている原理が発見された。以上述べた各種の安定判別法それぞれについて、少し詳しく述べることとする。
