# 1・3・1 ラウス・フルビッツの安定判別法(Routh-Hurwitz

> **章**: 第1章 システム解析
> **節**: 1・3 安定性解析

入力が零で初期状態による応答(零入力応答という)が時間とともに零に収束するとき、システムは漸近安定(asymptotically stable)であるという。このことは次のようにして確認できる。こうして、漸近安定である必要十分条件は、固有値、すなわち行列の特性多項式の根がすべて複素開左半面に存在することであることがわかる。6―14制御システム一方、伝達関数は零状態(初期状態が零)のときの入出力を関係付けるものである。伝達関数がプロパでその極(分母多項式の根)がすべて複素左半面Re<0にあるとき、システムは(または伝達関数は)有界入力有界出力の意味で安定[BIBO(boundedinput, bounded output)安定]であるという。BIBO安定であるとは、有界な入力に対して出力がつねに有界なときをいう(零状態応答で考えて)。プロパでない伝達関数では分子多項式を分母多項式で割るとわかるように、のべき乗を含んでいる。時間領域でいえば入力の高階微分をとることを意味している。したがって、入力として単位ステップ関数を加えると、出力にはデルタ関数の高階微分が含まれて有界でなくなるので、BIBO安定のためにはプロパであることが必要である。直達項はBIBO安定性の結果に影響しない。畳み込みの項がBIBO安定性を決定する。こうして、プロパ性と極が左半面に存在することがBIBO安定条件になることが理解されよう。根がすべて左半面にあるとき、フルビッツ多項式(Hurwitz polynomial)という。与えられた多項式がフルビッツ多項式かどうかは係数だけから判定できる。その方法としてラウスの方法とフルビッツの方法がある。a. ラウスの方法次の実係数多項式(1・103)が与えられたとき、それがフルビッツ多項式であるかどうかは、係数から次のラウス表を作って判定できる。ラウスは、この判定法をコーシー指数とスツルム定理によって導出した。例えば、文献(31)にラウスの用いた証明法が載っている。これ以外の証明法として、連分数展開に基づく方法などがある。ラウスの方法が連分数展開と関連付けられることは、物理的な考え方から解釈しやすく理解を深めるのに役立つであろう。そこで、結果だけを示しておくことにする。この係数はラウス表の値と次のように関連付けられている。この連分数展開と係数が正という条件は、質量とばね(コンデンサとコイル)からなるシステム(無損失な受動システム)の駆動点インピーダンスが満たすべき必要十分条件であり、無損失性とフルビッツ多項式が密接な関係にあることがわかる。複素係数をもつ多項式の場合にも上述の方法は拡張できる。例えば、文献(33)を参照されたい。そこでは、連分数展開の手法と複素係数をもつ回路素子のインピーダンスとの関連付けに基づいて、複素係数の場合も連分数展開で安定判別できることが示されている。b. フルビッツの方法フルビッツは、ラウスより少し遅れて独立に、現在フルビッツの安定判別法として知られている方法を提案した。その結果は純粋数学の分野で基本的には解かれていたエルミート(Hermite)の結果を基にしたもので、次のとおりである。式(1・103)で与えられた実数係数多項式が、フルビッツ多項式である必要十分条件は、次の一連の行列式がすべて正であること、つまり、>0,>0,...,>0となることである。フルビッツの方法の別証明として、三重対角行列化してリアプノフ方程式を適用するパークス(Parks)の方法がある[例えば文献(30)参照].フルビッツの方法とラウスの方法は次の関係で結ばれている。つまり、負の係数や零の係数があればフルビッツの方法を適用するまでもなく安定ではないと判定できる。そこで、正の係数からなる多項式だけを対象にすることにすると、このときには、フルビッツの安定判別でその条件数を半分に減らすことができる。安定化するを求めてみよう。c. シュール・コーンの方法離散時間の線形システムでも、BIBO安定性と漸近安定性は同様に定義され、その条件はそれぞれ伝達関数の極とシステム固有値が開単位円内<1に存在することとなる。シュール多項式かどうかは、双一次変換を用いてフルビッツ多項式かどうかの問題に変換できる。ラウス・フルビッツの方法のように、シュール多項式かどうかを係数から直接判定する方法として、シュール・コーン(Schur-Cohn)の方法が有名である。これら個の行列の行列式が正であることがシュール多項式となるための必要十分条件である。ここで、係数は複素係数を許していることに注意しよう。ラウスの方法に対応した方法もある。最近では、シュール・コーンの方法よりも、ジュリー(Jury)のインナを用いる方法もよく用いられている。
