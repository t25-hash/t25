# 6・5・4 海難事故の調査と審判 海難事故の場合には，司法判

> **章**: 第6章 マネジメントと法制度
> **節**: 6・5 事故調査と事故責任

断に向けての海上保安庁による調査と，「海難審判法」に基づき行政から必要に応じて，地方自治体等の公共機関，保険機関，所属船会社などの独自調査が行われることもある．
海難審判庁の組織と任務il組織 海難審判庁は国土交通省の外局であり，東京に高等海難審判庁および海難審判理事所が置かれ，地方の主要都市（函館，仙台，横浜，神戸，広島，門司，長崎）に地方海難審判庁および地方海難審判理事所が，沖縄に地方海難審判庁支所および地方海難審判理事所支所がそれぞれ置かれている．
審判官と理事官の陣容は，海難審判施行令に定められており，審判官55名，理事官41名，副理事官9名である．
任務 海難審判は海難が，船舶の乗組員等のヒューマンファクタが原因で発生したものか，また船体や機関の構造や性能などが原因で発生したものか，などを専門の知識と経験を有する審判官の合議体によって審理し，裁決をもって明らかにする．
海難が人の故意や過失により発生したときには，これらの者を懲戒（行政処分）し，さらに，海難の原因に関係した海運会社や造船会社などに対して，その業務や設備などの改善や改良を勧告する．
理事官は，海難の調査，審判の請求と立会いおよび裁決の執行を行う．
海難審判の手順i．
調査 理事官は，海上保安官，警察官および市町村長などから海難があったことの報告を受けた時や，ラジオ，テレビなどにより海難を知った時は，直ちに調査を開始する．
その実施事項は，①海難関係人を出頭させ，または質問する，②検査の実施と証拠品の領置，③事件（海難審判庁では一般に事故のことを事件という）に関する照会，鑑定，翻訳，である．
日本近海において発生した事故に対しては，各地方海難審判理事所が調査を担当し，それ以外の全世界にわたる水域に対しては，東経136度1分30秒と西経70度で二分して，太平洋側は横浜，インド洋裁決を行う海難審判庁による調査とが行われる．
また，事故の性格iii．
事故調査の課題 このように本委員会には，航空・鉄道事故調査委員会設置法により，関係者からの事情聴取，関係物件の検査，関係物件の留置，現場の保全などのかなり強い霧中権限が与えられている．
しかし，実際の調査にあたって，司法機関の調査（犯罪捜査）と競合し押収されるべき証拠類が同一となった場合は，どちらが優先されるべきであるかが問題となってくる．
この点について，同設置法には明確な規定がない．
ただし，警視庁と旧航空事故調査委員会の間で覚え書きが交わされていてゴ現場保存，事情聴取，関係物件の押収などにおいてお互いに協力することが確認されているといわれている．
しかし，事故発生時には，組織力と機動力のある警察の捜査が先行するのが現状であろう．
日航機相互のニアミス事故（2001年1月）の時には，真っ先に警察の捜査が行われ，その後に当時の航空事故調査委員会，運輸省，日本航空社内の調査が行われたように，種々の組織の調査が錯綜した．
自衛隊機の事故調査 航空・鉄道事故調査委員会では，民間機のみが関与する事故および民間機と自衛隊機が関与する事故までの調査を扱う．
自衛隊機のみが関与する事故については，自衛隊法第107条第7項により，航空・鉄道事故調査委員会設置法第3条の規定（航空事故の原因を究明するための調査等の所掌事務を定めた規定）は適用しないこととなっている．
このため，自衛隊機のみが関与した事故については，防衛庁内部に調査委員会が設置される．
本調査委員会は，現地調査と最終結論を出す委員会とに分かれている．
現地調査は専門家から構成される「航空安全管理隊」が担当し，事故発生後6時間以内に現場へ到着できるように現地派遣集団が常設されている．
なお，自衛隊機のみが関与した事故であっても，航空事故の防止のために有益であると認められる事故については，航空・鉄道事故調査委員会にその情報を提供するものとされている．
