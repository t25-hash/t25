# 2・1・1 フィードバック系の諸特性

> **章**: 第2章 線形制御
> **節**: 2・1 古典制御

実際の制御系においてはほとんどの場合フィードバック制御が用いられている。このフィードバック制御の本質的な利点は、制御対象の不確実さや未知外乱の影響を低減できることにある。本節ではまずこれについてフィードフォワード制御と対比させながら説明する。次いでフィードバック制御系の出力が定常状態において偏差なく目標値に追従するための条件について考察する。a. 感度特性(sensitivity characteristics)。 パラメータの変化に対する感度図2・1(a)のフィードフォワード制御系と、図2・1(b)のフィードバック制御系を考える。これは例えば回転体の速度制御系などの場合に当たる。しかし、現実の制御対象では時定数τやゲインが環境によって変化したり、もともと真値とモデル化したときの値が異なっている場合がほとんどである。フィードフォワード制御の場合はこのように制御対象のパラメータ変化が100%制御量の変動に反映されてしまう。このときフィードフォワード制御系で=1/=0.2とおいたときの応答を、図2・2(a)に示す。一方、フィードバック制御系で=20と選んだ場合の応答を図2・2(b)に示す。フィードバック制御系ではパラメータ変動の影響が劇的に減少していることがわかる。この例からも明らかなように、フィードバック系においては制御器のゲインを大きくすると、制御対象のパラメータ変動の影響を受けにくくなる、すなわちパラメータ変動に対する感度が低くなる。このことは一般的に次のように数式で表現できる。両者の比較により、フィードバックを用いれば、外乱の影響も感度関数の分だけ低減化可能であることがわかる。b. 定常特性(steady-state characteristics)制御のおもな目的の一つは制御量を目標値に追従させることである。このとき過渡状態において追従偏差をなくすことは困難であるが、定常状態においては偏差を零にすることがしばしば要求される。これは、例えば位置決め制御系などではその精度がこの定常偏差(steady-stateerror)で定まるためである。先に、制御器のゲインを大きくすることがフィードバック制御系の低感度化に効果があることを述べたが、実は、定常特性の改善にも役に立つ。b項ではこの点について説明する。. 目標値に対する定常偏差以下では、図2・1(b)の直結フィードバック系で=0の場合を考える。まず、ステップ目標値と制御量の間の定常偏差を具体的に計算し、偏差を零にするためのフィードバック制御系の条件を導く。これは一巡伝達関数あるいは開ループ伝達関数と呼ばれる。以下では基本的な目標値を加えたときの定常偏差を求める。以上の考察より偏差定数,が大きいほど定常偏差が小さくなることはわかるが、定常偏差を零にするためにはどうすればよいのだろうか。同様に、=2のときは定常速度偏差が零となる。直結フィードバック系における制御系の型と定常偏差の関係を表2・1にまとめておく。. 外乱に対する定常偏差目標値に追従することと同様に、外乱の影響を定常状態において零にすることも重要である。このため、実質的には制御器が積分器をもつことが重要となる。この場合、外乱の周波数成分(=0に対応)に対して制御器のゲインが無限大になること[0=∞]が本質的である。
