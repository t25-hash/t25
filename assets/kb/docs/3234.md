# 2・1・2 安定余裕(stability margin)

> **章**: 第2章 線形制御
> **節**: 2・1 古典制御

a. ナイキストの安定判別法(Nyquist’s stability criterion)図2・1(b)のフィードバック制御系に対するナイキストの安定判別法(安定条件)は、次のような手順にまとめられる。さらにこれを実軸に関して上下対称に描き、ナイキスト軌跡(Nyquist locus)Γを得る。ステップ2:ナイキスト軌跡Γが点-1,0の周りを時計方向に回る回数を調べ、これをとする(もし反時計方向に回転するならば、は負となる)。ステップ4:閉ループ系の不安定な極の数は=+Π となる。したがって、=0ならばフィードバック制御系は安定、≠0ならば系は不安定である。上のステップ4から、-=Π のとき=0となり系は安定である。したがって、ナイキスト軌跡が点-1,0の周りを反時計方向に回る回数が、開ループ伝達関数の不安定極の個数に等しいならば、制御系は安定であるといえる。このナイキストの安定判別法は、フィードバックループを閉じる前の開ループ伝達関数の周波数応答によって、閉ループ系の安定性を調べることができるという点で優れている。動特性が複雑で次数の高い系の場合でも、特性多項式の零点を計算する必要などなく、周波数応答という実測データに基づいて判定することができる。実際、むだ時間を含む系にも適用できることが知られている。また、図的な方法なので直感的に理解しやすく、以下で述べるように単に安定性の判別だけでなく安定性の程度を評価することも可能となる。破線はこれを実軸に関して上下対称に描いたもので、これよりナイキスト軌跡が図2・3のとおり得られる。次に、開ループ伝達関数が安定な(あるいは虚軸上の極を含む)場合について考えておく。実際の制御系では、制御対象が不安定であることは少ないので、このような場合について特別な考察を行っておくことは重要である。このときには、不安定な開ループ極はないのでΠ=0である。したがって、フィードバック系の安定条件は=0となる。これより簡単化されたナイキストの安定判別法は、次のように述べることができる。ステップ1:開ループ伝達関数の極の中に、その実部が正となるものがないことを確認する。ステップ3:ωを0から∞へ変化させたとき、この開ループ伝達関数のベクトル軌跡が点-1,0をつねに左に見るように動くならば、系は安定である。また、右に見れば系は不安定となる。この開ループ伝達関数の極の中には、明らかにその実部が正となるものはない。そこで、それぞれの場合に対してベクトル軌跡を描いてみると、図2・4のようになる。しかしながら、ゲインを上げて=6とすると、軌跡はちょうど点-1,0を通過するようになる。これが安定限界である。そしてさらにゲインを増大させて=12とすると、ついにその軌跡は点-1,0を右に見るようになり、不安定となってしまう。b. phasemargin)簡単化されたナイキストの安定判別法では、ベクトル軌跡が点-1,0を左に見るか否かが、制御系の安定性を決める鍵となっていた。図2・5を用い、これについてさらに詳しく考察してみよう。図2・5(b)で、ベクトル軌跡がちょうど点-1,0の上を通る安定限界のときを考える。この結果として振動が持続し、安定限界になると解釈できる。すなわち、位相の反転した信号がネガティブフィードバックにより再び加えられても、振幅は減衰していくので、その結果として系が安定になると理解できる。この角周波数ωは位相交差周波数(phase crossover frequency)と呼ばれ、ベクトル軌跡が負の実軸を横切る点Pに対応している。同様に、今度は開ループ伝達関数のゲインがちょうど1になっているときの位相の遅れに着目してみよう。いま開ループ伝達関数のベクトル軌跡の絶対値が1になるときの角周波数をωとし、これをゲイン交差周波数(gain crossover frequency)と呼ぶ。これは、ベクトル軌跡が原点を中心とする単位円と交わる点G に対応している。図2・5より明らかに、このときの位相が-180°より進んでいれば、ベクトル軌跡は点-1,0を左に見ることになり、系は安定である。以上の議論から、ベクトル軌跡が点-1,0を左に見るようにある程度離れているならば、フィードバック制御系は安定余裕があると考えることができる。この安定余裕を定量的に示す指標として実際によく用いられるものが、以下に述べるゲイン余裕と位相余裕である。図2・6において、ベクトル軌跡が実軸と交わる点をP、また原点を中心とする単位円と交わる点をG
