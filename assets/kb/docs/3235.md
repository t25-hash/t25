# 2・1・3 PID 制御(PID control)

> **章**: 第2章 線形制御
> **節**: 2・4 に示されているベクトル軌跡との対応に注意しよう.図2・8か

a. 制御系の設計手順制御系の目的は,入力を適切に操作することにより,出力のふるまいを望ましいものにすることである.レギュレータ(定値制御)問題とは,系に加わる外乱の影響を抑制し,出力をある一定の値に保持しようとする制御問題であり,一方サーボ(追従制御)問題とは,与えられた目標値の変化に対して,出力を良好に追従させるように制御することである.このような制御系を実現するための一般的な設計手順は,以下のようにまとめられる.ステップ1:制御対象の数学的モデルを求める.ステップ2:制御目的から,性能仕様を決める.ステップ3:性能仕様を満たすように,コントローラを設計する.ステップ4:シミュレーションにより設計された制御系を評価する.必要ならば,以上のステップを繰返し設計をやり直す.ステップ5:コントローラを実装しハードウェアでテストする.制御対象に対して,使用するセンサやアクチュエータの選択・配置を検討したり,対応する制御量・操作量を定めたり,系の動作環境や拘束条件を分析したりすることもたいせつである.本項と2・様を満たすコントローラの特性設計を古典制御の枠組で取扱う.まず本項でPID 制御について説明し,2・1・4項で位相進み-遅れ補償について述べる.b. 制御系の性能評価制御系設計においては,モデルの設定とともに性能仕様が重要となる.この制御系の性能を評価する際には,定常特性と過渡特性に着目しなければならない.定常特性については,すでに2・1・1・b項で議論しているので,以下では過渡特性(速応性および減衰特性)についてまとめておく.. 時間応答に基づく制御系の性能評価まず,制御系のステップ応答は一般に図2・9のようになるが,このような応答波形に図2・9過渡応答と諸特性値図2・10 閉ループゲイン特性対して以下の特性値が定義されている.욥:立上り時間(risetime,ステップ応答が定常値の10~90%になるまでに要する時間),욃:遅れ時間(time lag,遅延時間,ステップ応答が定常値の50%に達するまでの時間),욦:整定時間[settling time,応答が定常値の±2%の範囲内(±5%や±1%の場合もある)に収まるまでの時間],윯율융:オーバシュート[overshoot,最大行過ぎ量,応答のピーク値욣(定常値を超えて到達する最大値)と定常値との差.定常値に対する割合(%)として表示することが多い],욣:行過ぎ時間(overshoot time,最初の行過ぎに達するまでの時間),減衰比(最初の行過ぎ量と次の行過ぎ量の比).ここで,立上り時間,遅れ時間,整定時間,行過ぎ時間などは速応性の目安となり,オーバシュート,減衰比などは減衰特性の一つの目安として使われる.また,二次系=ω욈/+2ζω욈+ω욈の場合,その各係数はそれぞれ,減衰係数ζは振動減衰の特性を定める,自然角周波数ω욈は速応性を定める,といった意味をもつことが知られている.. 周波数応答に基づく制御系の性能評価次に,図2・10に示される一般的な閉ループゲイン特性ωに対し,以下のような特性値が知られている.まずバンド幅(band width,帯域幅)ω욁욌は,閉ループゲイン特性ωが定常ゲイン0の1/ 20.707倍(3dB 低下)になる角周波数であり,速応性の指標となっている.これに対して,ピークゲイン(共振ピーク)욥は,閉ループゲインの最大値を指し,減衰特性の指標となる.また,このときの角周波数を共振周波数ω욥という.閉ループ伝達関数は一般に低域通過特性を示すが,バンド幅は目標値に対して制御量がどれくらいの周波数成分までほぼ忠実に追従するかということを意味している.したがって,バンド幅ω욁욌が大きければ,それだけ高い周波数成分まで追従性能が保証されることになる.一方,減衰特性の指標であるピークゲイン욥については욥=1.1~1.5という値が経験的に知られている.욥=1.3あたりが適当とされることもある.
