# 6・2・4 ホログラフィーおよびスペックル干渉による変形計測

> **章**: 第6章 各種応用計測
> **節**: 6・2 光応用計測

ホログラフィー干渉法（holographic interferometry）やスペッ
クル干渉法（speckle interferometry）によると，粗面（通常の干
渉計は鏡面のみを対象）の変形分布や振幅分布を干渉計測でき
る ．従来はホログラフィー干渉がよく用いられたが，最近は
CCDカメラと画像メモリを用いて電子的に干渉計測のできるスペ
ックル干渉法［ESPI （electronic speckle pattern interferometry）
またはテレビホログラフィーと呼ばれる］が開発され，装置も市販
されている ．ここではESPIを用いて大きな変形を計測する方
法を紹介する ．大変形を求める手順は以下のとおりである．ま
ず，図6･28のESPI計測システムによって変形前から変形後まで
のスペックル画像（,…, ）を時系列で次々と画像メモリに蓄
える．次に，それらの適当な二つの差＝－をとってカ
ウント可能なしま画像を出現させ，それを解析して部分的変形
を求める．最後にそれらを加算することによって大変形を求
める．
実験では，光源はHe-Neレーザ（λ＝632.8nm）であり，計測
対象はマイクロメータヘッドで変形を与えた片持ばりとした．1フ
レーム当たり256×240×8ビットの画像を変形前 から変形後
まで10フレーム蓄積した．変形全体をとった場合の干渉じま
＝－は，図6･29のように片持ばりの先端付近の変形量
が大きすぎて干渉じまが解析できない．そこで図6･30のように
