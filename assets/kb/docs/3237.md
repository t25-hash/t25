# 2・1・4 位相進み-遅れ補償(phase lead-lag compensation)

> **章**: 第2章 線形制御
> **節**: 1・4 項では、制御対象の数学的モデルが与えられたときに、性能仕

PID 制御と並んで現在でも広く用いられている古典的な設計法として、位相進み-遅れ補償がある。この位相進み-遅れ補償は、これから説明するような位相遅れ要素や位相進み要素をループ中に直列に順次組込んでいく手法であり、簡単なPID 制御に比べて設計の自由度がある。このため、以下で述べるような開ループ伝達関数の周波数応答を整形すること(ループ整形)が可能となり、モデルの不確かさに対処できる設計法となっている。ただし、その設計手順は必ずしも系統的になるとは限らず、実際には繰返しを必要とすることが多い。しかし、実際の制御系では感度関数と相補感度関数を小さくしたい周波数帯域が異なっている。ループ整形に基づく設計で重要な点は、次のようになる。速応性:ゲイン交差周波数ωをできるだけ高くとる。減衰特性:位相余裕PM を十分に確保する。特にループ整形において難しいのは、交差周波数付近における取扱いである。もし交差周波数付近でのゲインの傾きが急峻(-40dB/dec以下)であるとすると、ゲインと位相の関係より、好ましくない位相遅れが引き起こされてしまう。よって、位相余裕を十分に確保するためには、ゲイン交差周波数付近で緩やかなゲインの傾き(-20dB/dec)が望ましい。また、位置偏差定数や速度偏差定数などの定常特性を考慮すると、低周波域における開ループゲインは十分に大きくなければならない。例えば、ステップ状の目標値を考慮するのであれば、低周波域でのゲインの傾きは-20dB/decであればよいし、もしランプ状の目標値を考慮するのであれば、傾斜は-40dB/dec必要である。一方、高周波域におけるゲインの傾きをしばしばロールオフと呼んでいる。雑音の影響を受けないためにも、開ループゲインは角周波数とともに急速に小さくなっていくことが望ましく、一般に高周波域では-40~-60dB/dec以下のロールオフ特性が望ましい。これから説明する位相遅れ補償や位相進み補償は、以上のような指針に基づいて設計される。b.
