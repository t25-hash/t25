# 2・1・4 位相進み-遅れ補償(phase lead-lag compensation)

> **章**: 第2章 線形制御
> **節**: 1・4 項では、制御対象の数学的モデルが与えられたときに、性能仕

PID 制御と並んで現在でも広く用いられている古典的な設計法として、位相進み-遅れ補償がある。この位相進み-遅れ補償は、これから説明するような位相遅れ要素や位相進み要素をループ中に直列に順次組込んでいく手法であり、簡単なPID 制御に比べて設計の自由度がある。このため、以下で述べるような開ループ伝達関数の周波数応答を整形すること(ループ整形)が可能となり、モデルの不確かさに対処できる設計法となっている。ただし、その設計手順は必ずしも系統的になるとは限らず、実際には繰返しを必要とすることが多い。a. ループ整形(loop shaping)パラメータ変動に対する低感6―33図2・18
