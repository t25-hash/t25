# 6・3・1 SEM応用計測（測長SEM，LSI検査）

> **章**: 第6章 各種応用計測
> **節**: 6・3 放射線応用計測

走査型電子顕微鏡（SEM:scanning electron microscope）の技
術を応用して，半導体（semiconductor ）プロセスの評価に供する
装置としては，(１)電子線測長装置（通称，測長SEMと呼ばれ
る），(２) SEM式欠陥検査装置，(３)欠陥レビューSEMがある．
いずれも細く集束した電子ビームを試料に照射し，そこから発生す
る二次電子や反射電子を検出して画像信号とし，評価・解析を行
う．
a.測長SEM 半導体プロセスの微細パターンの寸法を測定
する装置である．半導体プロセス材料は電気的導電材料と絶縁材料
の各薄膜の組合せで構成されており，絶縁体上では電子線照射によ
って帯電し，良好な像観察が難しくなる．そこでこの帯電を防止す
ること，また電子線による試料の損傷を防止するために加速電圧が
数百Vから1000Vくらい用いられる．
像分解能は2～3nmである．微細パターンを10万倍以上に
CRT上に拡大し，拡大像上でパターンのエッジを検出して寸法を
計測する（図6･32）．
計測時の繰返し再現精度は1～2nm（3σ)，あるいはパターン幅
の1%以内の精度で測定することができる．
装置のおもな構成要素としては，高輝度の電子源を得る電界放出
型電子銃，電子線を試料上に細く集束させる電子光学系，二次電子
検出系，ウエーハを載物して移動させる試料ステージ，ウエーハの
搬送・搬出を行うローダ系，真空排気系，およびこれらを制御する
電気系，制御ソフトウェアより成る（図6･33）．
ウエーハのロードから測長値の出力までの一連の動作は，すべて
自動で実行される（図6･34）．

出する．装置の特徴としては，(１)電位コントラストによるビア・
プラグ工程などの非導通およびショートの欠陥検出，(２)0.1m
以下の微細構造での欠陥を高感度に検査することができる．構成お
よび原理を図6･35に示す．チップごとの画像比較を行い，差画像
を用いて欠陥を抽出する．光学式検査に比べるとスループットが遅

c .欠陥レビューSEM 光学式欠陥検査装置，あるいは上述
のSEM式欠陥検査装置で検出された微小欠陥や異物に対して，よ
り詳しく観察したり，再評価するための装置である．
再レビューによって得られる情報は，欠陥の起因・原因を突き止
めることに役立ち，プロセスを改善するために利用される．
SEMによる欠陥レビューは，上記の欠陥検査装置によって検出
された欠陥のウエーハ上の位置情報（座標）を基に欠陥の検索を行
うので，その座標の位置決め精度が重要になる（図6･36）．
これらの位置決めは，ADR（auto defect review）で自動的に実
行され，欠陥画像が取得される．次に得られた欠陥画像に対して，
カテゴリー別に分類することになるが，これをADC（auto defect
classification）法を用いて弁別し，致命欠陥など欠陥の分類を行
い，プロセスへの迅速なフィードバックが実行できるような仕組み
になっている．また，異物などの元素分析をEDX（energy disper-
sive X-ray，エネルギー分散型X線分析器）法で実施できるよう
にしている．
