# 6・3・2 工業用X線CT

> **章**: 第6章 各種応用計測
> **節**: 6・3 放射線応用計測

a.概要 工業用X線CT［X-ray CT( computed tomogra-
phy)］は，物体内部の正確な三次元形状（three- dimensional
shape）を計算機内に取込み解析・表示する唯一の手段で，その利
用価値は極めて高い．しかし計測対象がX線を透過しにくい鉄な
どの高密度材料で，しかも高い空間分解能が必要なことなど技術的
に難しく，その普及は医療用のCTスキャナに比較して20年以上
も遅れている．近年，透過力の強い高エネルギーX線を発生する
電子線型加速器（electron linear accelerator ，LINAC:linear
accelerator ）をX線源とし，その強い透過力をもつX線を高感度
で検出する小型のシリコン半導体検出器（Si -SSD:silicon solid
state detector ）をX線検出器とする工業用X線CTが実用化し
たことで，その利用が広がり始めた．工業用X線CTは医療用
CTスキャナと原理は全く同じなので，本節ではその説明は省き，
工業用X線CT固有の構造と性能についてのみを記す．

c.X線源 例えば鉄に対するX線の透過力は約9MeVで最
大になる．したがって，厚い高密度材料が試料の場合はこのエネル
ギー領域のX線が有効である．このレベルの高エネルギーX線は
LINACにより加速した電子線の制動ふく射によって発生させる．
LINACによるX線はエネルギーが高いばかりでなく強度も強い
ので，短い撮影時間で高い分解能の断層像が得られる．ただし鉄で
数十mm，アルミニウムで200～300mmまでの試料であれば
LINACの代わりに数百kVのX線管でもCT撮影は可能である．
d.X線検出器 アレイ状に配列したX線検出器と検出器単体
を図6･38に示す ．焦点から出射して試料を透過したX線は短冊
形のシリコン半導体検出器の小口から長手方向に入射する．短冊形
の検出器はすべてがX線の焦点を向くように配列してある．この
構成により，約40mmのシリコンを透過する間に数MeVの高エ
ネルギーX線光子の20%ないし30 %が検出器と相互作用する．こ
の相互作用の確率はX線フィルムのおよそ3×10倍である．各検
出器は互いにタングステン板で遮へいして，検出器相互間のクロス
トーク（ある検出器に入射したX線が散乱して隣接する検出器で
検出される現象で，画像劣化の原因になる）を防止する．
e .工業用X線CTの性能 工業用X線CTの性能を決める重
要な因子は，(１)空間分解能，(２)撮影時間，(３)鉄換算厚さの三
つである．鉄換算厚さとは，どの程度まで厚く密度の高い試料の撮
影ができるかを鉄の厚さで代表して示すものである．これらの性能
因子には，試料透過後に検出されるX線光子数が深く関わる．そ
れはX線光子数の統計的ばらつき（揺らぎ）が画像上の雑音とし
て現れるためである．高密度で厚い試料ほど，高い分解能ほど，ま
た撮影時間が短いほど，画像上の雑音は増加する．つまり三つの性
能因子のいずれを向上させようとしても，画像上の雑音は増加する
方向にある．この統計的ばらつきを低減するには，透過後に検出さ
れるX線光子数を増す以外に手段はない．単位時間当たりの検出
光子数を増加させる手段には，X線源の強度を増すこととX線検
出器の検出効率を増すことの二つがある．X線CT用の線源とし
てのLINACは，12MeV程度を上限と考えるべきである．これ以
上にエネルギーを高くしても透過力は頭打ちになるし，中性子発生
による遮へいコストが増大するという問題も起きる．また検出器に
ついても，その感度はあと数倍で100 %に近づくことになり，それ
以上の改善はあり得ない．したがって12MeVのLINACを使う
場合，表6･4に掲げた性能の組合せでの撮影が高エネルギーX線
CTの技術的限界である．表6･4に「―」で示してある箇所は撮影
時間が非現実的に長くなり，実用上撮影が不可能なことを示す．工
業用X線CTの主たる利用目的が物体内部の三次元形状計測であ
れば，一試料に対して数百枚から数千枚の断層撮影が必要であり，
1枚当たりの撮影時間を短縮することは切実な問題である．鉄換算

f .工業用X線CTの応用 工業用X線CTは，物体の外表面
ばかりでなく内部の複雑な形状を計測・表示できる三次元計測器と
して，3D- CAD（3D- computer aided design）図面とそれに基づ
く製品・試作品との照合や，製品から3D- CAD図面の自動生成
（リバースエンジニアリング），さらにはラピッドプロトタイピング
（RP:rapid prototyping）の利用による現物からレプリカ作成な
ど，機械の設計・製造プロセスを変えるほどのポテンシャルをもっ
ている．図6･39は金属RPで作成した水車の模型を工業用X線
CTで撮影し，その外観形状・内部形状を表示した例である．これ
らの形状データと3D-CADによる設計図を重ね合わせて製作誤差
を算出することも可能である．この技術は正確な立体形状を保存・
表示する機能から古生物学，考古学，文化財保存学などへの利用も
進みつつある．今後は計測された精密な立体像をうまく利用するた
めのソフトウェアの開発が重要になる．
