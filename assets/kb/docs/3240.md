# 6・4・1 超音波の定義と超音波応用計測の特徴

> **章**: 第6章 各種応用計測
> **節**: 6・4 超音波応用計測

気体中，液体中，固体中を伝わる20kHz以上の弾性波（elastic
wave）を超音波（ultrasound，ultrasonic wave）と呼ぶことが多
い．しかし，実際は20kHz以下の可聴域の振動や音波も超音波と
して利用されており，聴くことを目的としない振動・音波を超音波
と定義することもある．光や電波などの電磁波の伝搬しにくい液体
中や固体中を伝搬すること，伝搬速度が電磁波の10万分の1程度
以下であることが計測へ応用する際の特徴となる．距離，速度，媒
質の流速計測のほか，温度測定や媒質の物性測定にもさまざまに応
用されている．また，粘度計，膜厚計，振動ジャイロなど超音波振
動の性質を利用したセンサデバイスも実用化されている．
