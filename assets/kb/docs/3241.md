# 2・2・1 最適レギュレータ(optimal regulator)

> **章**: 第2章 線形制御
> **節**: 2・1 項で述べる最適レギュレータのほかに、閉ループ極を指定され

a. 最適レギュレータ問題とその解一般にレギュレータとは、状態の平衡点からのずれを速やかに平衡点に戻すための制御系をいう。最適レギュレータは、それを二次形式評価関数を最小化する最適制御問題の解として合理的に求めるものである。まず、制御対象は次の状態空間モデルで記述される線形定数システムとする。いずれも原点を平衡点とし、そこからの偏差を表すものとする。式(2・36)の線形システムに対して式(2・38)の評価関数を最小にする制御入力を求める問題を、(無限時間)最適レギュレータ問題または線形二次最適制御(LQ)問題という。以下では、便宜上この形の評価関数に限定するが、そのように限定しても一般性を失わないことに注意する。以上の準備のもとに最適レギュレータ問題の解を与える。以上の事実を基に以下では、対,の検出可能性のもとで最適制御が式(2・41)で与えられることを示す。以上のことを具体例で説明する。次の可制御な一次システムを考える。前者が最小非負解であり、これに対応する式(2・40)の状態フィードバックは=0である。これが最適制御であることは評価関数の形から見て明らかである。次に≠0の場合を考える。この結果より、最適レギュレータは図2・28に示すように、状態の線形関数をフィードバックする閉ループ系で構成されることがわかる。ここで右辺右側の行列は、式(2・43)のベクトルφ,...,φの一次独立性から明らかに正則である。c. 最適レギュレータのロバスト性(robustness)定理1で述べたように最適レギュレータは状態の線形フィードバックで簡単に構成され、しかも閉ループ系は安定になる。最適レギュレータの工学的意義は、このような制御系の安定性や構成の簡単さにあるだけでなく、それが評価関数の重みの選び方にはよらない二つの優れた性質を有する点にある。その一つはシステムの特性変動に対する頑健な安定性であり、他の一つはパラメータ変動に対する低感度特性である。これらはいずれも最適レギュレータのロバスト性を示すものであり、特に前者はロバスト安定性と呼ばれ、実用上重要な意味をもつ。以下本項では、円条件と呼ばれる最適レギュレータの周波数特性に基づいて、そのロバスト性を1入力系の場合について示す。なお、本項では制御対象が1入力系であることを示すために、式(2・36)~(2・38)(2・40)(2・41)において次のような行列の置換えを行う。このため式(2・46)の不等式条件は円条件と呼ばれる。この条件に最適レギュレータの安定性に関するナイキストの判別条件を加味すれば、1入力最適レギュレータは、評価関数の重みの選び方と無関係に次のような大きな安定余裕をもつことがわかる(図2・29)。(1) 最適レギュレータのゲイン余裕は、増加側で∞、減少側で少なくとも2分の1ある。(2) 最適レギュレータの位相余裕は少なくとも±60°ある。これは、最適レギュレータの優れた安定性をゲイン余裕、位相余裕という古典的尺度で表現したものであり、これが最適レギュレータのロバスト安定性の根拠になっている。もう一つのロバスト性である低感度特性も円条件(2・46)から導かれる。この式は、閉ループ系の特性変動が全周波数帯で閉ループ系のそれよりも小さいことを意味し、最適フィードバック制御の低感度特性の根拠になっている。なお、最適レギュレータの逆問題の結果より、式(2・46)の円条件は、安定な状態フィードバック制御系の中で、最適レギュレータのクラスを特徴付ける条件になっており、その解釈を与えた上記の低感度特性は、最適レギュレータの工学的意義を明らかにするものとして重要である。以上1入力系に関して述べた最適レギュレータのロバスト性は多入力系にも拡張されており、その詳細については文献(9)または(10)を参照されたい。
