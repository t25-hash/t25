# 6・4・2 使用する周波数範囲

> **章**: 第6章 各種応用計測
> **節**: 6・4 超音波応用計測

図6･40のように，100Hz程度からGHz領域までの幅広い周波
数が用途に応じて使い分けられている．海洋の温度分布や流速分布
などを地球規模で測定する海洋音響トモグラフィーと呼ばれる技術
では，1000km程度伝搬させるために，減衰の少ない数十Hzか
ら200Hzの音波が使用される ．また，魚群探知 ，ソナー
（sonar ) などには，分解能と減衰量とのかねあいにより数kHz
から100kHzくらいが用いられている．一方，遺跡など土中の埋
設物探査のために，数百Hzの横波音響波の利用が試みられてお
常温付近で1℃当たり＋0.6m/ sの温度係数を有している．
