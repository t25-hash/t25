# 2・2・1 項では確定的なシステム

> **章**: 第2章 線形制御
> **節**: 2・1 項で述べる最適レギュレータのほかに、閉ループ極を指定され

に対して最適レギュレータ問題を考えたが、同様の問題は式(2・67)~(2・72)で表されるような白色ガウス雑音が加わる確率的なシステムに対しても考えられる。このような確率的最適制御問題は、対象のシステムが線形(linear)であり、評価関数が二次形式(quadratic form)であり、雑音がガウス性の白色雑音(white Gaussian noise)であることから、その頭文字をとってLQG(linear quadratic Gaussian)制御問題(LQG control problem)と呼ばれる。この問題の解は次の分離定理としてよく知られている。この結果は、上で述べた最適レギュレータとカルマンフィルタを独立に構成し、それらを図2・30のように統合させれば、LQG 制御系が構成できることを意味している。先に2・2・1項で最適レギュレータのロバスト性について述べたが、それが保証されるのは、最適レギュレータが完全状態フィードバックで理想的に実現された場合(図2・28)であり、最適レギュレータにカルマンフィルタを併合したLQG 制御系(LQG control system、図2・30)では、一般にこのようなロバスト性は保証されない。つまり、ロバスト性の劣化が起きることがある。本項の最後にそれを具体例で示しておく。図2・28と図2・30において次の1入出力システムを考える。いま、図2・30でシステムの入力側に正のスカラαの乗法的変動α(基準値はα=1)が生じたとする。なお、(2,2)ブロックは補償器のシステム行列であり、変動しないので注意する。このように、最適レギュレータのロバスト性がカルマンフィルタの併合によって劣化するのは、直感的には、モデル化誤差があるとLQG 補償器内で正確な状態推定が行われないためであるが、理論的には最適レギュレータのロバスト性を保証する円条件(2・46)が、LQG 制御系では成立しないことに起因している。その対策としてLQG/LTR と呼ばれる設計法がある。
