# 2・2・3 最適サーボ系(optimal servo system)

> **章**: 第2章 線形制御
> **節**: 2・1 項で述べる最適レギュレータのほかに、閉ループ極を指定され

制御対象の出力(制御出力)を目標の参照入力に追従させるサーボ系の設計問題は、多変数システム制御のうちで最も基本的な問題であり、また応用上最も重要な問題の一つである。サーボ系の設計では、まず追従偏差を漸近的に零にする(出力レギュレーション)ために、閉ループ系を安定化する必要がある。しかし、それ以外に制御対象のパラメータが閉ループ系の安定性を損わない範囲で任意に変動したり、外乱が加わってもつねに出力レギュレーションを達成(ロバスト追従、robust tracking)させることが実用上重要になる。このようなサーボ系を特にロバストサーボ系(robust servosystem)と呼び、それを実現するサーボ補償器は参照入力のモデルをダイナミクスとして含まなければならないことが、内部モデル原理(internal model principle)として知られている。本項では、すでに述べた最適レギュレータ理論を応用してこのようなサーボ系を設計する。その結果得られるサーボ系は通常最適サーボ系と呼ばれている。以下では簡単のために参照入力はステップ入力に限定するが、ランプ入力、正弦波入力など、一般に参照入力が線形定数システムの初期値応答として実現できる場合にも容易に拡張できる。なお、ステップ参照入力に対する内部モデルは積分器であり、したがって、ステップ参照入力に追従するロバストサーボ系は、内部モデル原理によりその補償器内部に積分器が含まれることから、特に積分型最適サーボ系と呼ばれている。その設計法を述べる前に、まずその基本となる積分補償を含まない最適サーボ系の設計について述べる。a. 積分補償を含まない最適サーボ系制御対象(2・36)(2・37)の出力をステップ参照入力に漸近的に追従させる。換言すれば追従偏差=-を定常状態で零にするサーボ系の設計問題を考える。ここに,は追従性能と入力エネルギーを調整する重みである。b. 積分型最適サーボ系(integral type optimal servo system)a項で得られた図2・31の最適サーボ系に積分補償を加え、制御対象のパラメータ変動や定値外乱に対してロバスト追従性をもたせた積分型最適サーボ系を、同じように最適レギュレータ理論を応用して導出する。その方法は、制御対象に積分補償を加えた拡大系と呼ばれるシステムに対する最適レギュレータの設計である。この拡大系には、積分器を制御対象の出力側に加えるもの(図2・32)と、入力側に加えるもの(図2・33)とがあり、どちらを用いても本質的な違いはない。まずわかりやすい前者の拡大系を用いた方法を説明する。この評価関数は、a項で用いた評価関数(2・91)に相当するが、ここでは追従偏差だけでなくその積分値(つまり積分補償器の状態)も評価の対象にしている点に違いがある。なお、準正定一意解の存在と閉ループ系の安定性は、次の性質により保証される(定理1参照)。これより求めるべき最適サーボ系は、図2・34のようにa項で求めた図2・31の最適サーボ系に積分補償を加えた形で構成される。なお式(2・109)(2・110)で逆行列の存在は、最適サーボ系(2・107)の安定性から保証される。以上述べた文献(16)の最適サーボ系設計法については、用いた評価関数(2・101)および積分器の定常値の決め方の二つの観点から、理論上の問題点が指摘されている。前者については、目標値追従というサーボ系本来の目的を反映した式(2・91)の評価関数を用いるべきとの立場から、式(2・91)の評価関数に関する最適性を保証しつつ、残された自由度を他の目的に活用した2自由度積分型サーボ系の設計法も提案されている。後者については、積分器の定常値の決定に自由度が残されている点に注目し、に関するさらなる最適性を追求することにより、図2・34のような構成をもつ最適サーボ系の設計法が提案されている。次に、積分器を入力側に加えた図2・33の拡大系に基づく最適サーボ系の設計法について説明する。なお準正定一意解の存在と閉ループ系の安定性は、システム(2・36)(2・37)の可安定性と可検出性に基づく拡大偏差系(2・113)の可安定性と可検出性により保証される。次に拡大系(2・111)の入力=に関する式(2・116)をシステム(2・36)の入力に関する式に変換する。また第3項めの初期値に関する項は、初期時刻を=0に選べば不要になり、その結果図2・35に示すような参照入力からのフィードフォワードがない積分型サーボ系の構成を得る。なお、=0と考えて、に関する最適性を追求すると、フィードフォワード制御が必要になり、図2・34と同一構成の積分型サーボ系が得られる。最後に、積分型最適サーボ系の実用的な設計法として、最適制御の逆問題を応用した設計法を紹介する。2・2・1項で述べた最適レギュレータ(LQ)設計法は、評価関数の重みが閉ループ応答などの工学的な仕様と直接結び付いていないのでその選定が本質的に難しく、それがこの設計法の実用上の難点となっている。しかし、その結果得られる最適レギュレータは、すでに述べたように、重みの選択によらず低感度特性やロバスト安定性など実用上好ましい特徴をもっている。これらの点を考慮すれば、LQ設計では従来のように重みを指定して最適制御を求めるよりも、むしろなにかの重みに関して最適であるような状態フィードバック制御を求め、最適レギュレータの特徴を残したまま設計を簡単化するほうが実用上得策である。そのような観点に立つ最適サーボ系設計法として、逆問題の結果を巧妙に応用したILQ最適サーボ系設計法(以下ILQ設計法(ILQ design method)と略記する)が提案されている。以下では、この設計法を簡単に解説する。すでに述べたように、図2・35に示す積分型最適サーボ系の状態フィードバックゲインと積分ゲインは、拡大偏差系(2・113)に対して式(2・114)、すなわち式(2・115)の二次形式評価関数を最小化する最適レギュレータ問題の解(2・116)を基に式(2・119)で得られた。最適制御の逆問題(inverse optimal control problem)とは、逆に状態フィードバック(2・116)を与えたときに、それがなんらかの二次形式評価を最小化するための条件を求めるものであり、2・2・1項で述べた円条件がそれに当たる。逆問題の解は、対象とする二次形式評価関数のクラスに依存するが、ここでは式(2・115)の二次形式評価において拡大偏差系(2・113)の状態の重みを正定対称行列に限定したものを対象にする。この表現を用いると、図2・35の最適サーボ系は容易に図2・36のように等価変換できる。これを積分型最適サーボ系の基本構造と呼ぶ。この基本構造を、図2・35の積分型最適サーボ系の設計に応用するために、図2・36の基本構造を特徴付ける三つのパラメータ行列,Σをその設計パラメータとして用いる。またその決定は、図の制御系に漸近することに着目して行う。すなわち、まず図2・37の制御系が望ましい特性をもつように基本ゲインを決定する。これはΣを十分大きくすると図2・35のサーボ系が所望の入出力特性(追従特性)を有することを意味し、したがって、本設計法では設計仕様として望ましい出力応答を与えることができる。その際、は設計の簡単化のために単位行列に選ぶ。また、Σについては、サーボ系の最適性を保証する下限値σを逆問題の結果を利用して求め、各対角要素σはすべてその値以上に選ぶ。明らかにσを大きくすると追従特性は設計仕様で与えた特性に近づくが、逆に制御入力は大きくなる。(ILQ設計法の設計可能条件)C1. 設計計算は、従来のリカッチ求解計算に代わり、非干渉化制御または極配置計算が主体となっているため簡単である。特に、基準最適ゲインは、制御対象の係数行列と設計仕様として与える所望の追従特性から解析的に求まるので、演算時間も大幅に短縮される。以上述べた設計理論は、制御対象と参照入力の双方の観点から大幅に拡張されており、詳細については文献(10)を参照されたい。1970年代後半からロバスト性を意識した制御系の解析・設計理論が注目を集めるようになってきたが、その一つに伝達関数の既約分解表現に基づく代数的制御理論(algebraic control theory)がある。これは、制御対象を安定化するすべての補償器のパラメータ表現を基礎として、フィードバック系の解析・設計を行うものである。非常に見通しがよく、個々の設計仕様を満たすすべての補償器を特徴付けやすいなどの利点をもっており、現在では基礎理論の一つとして定着した感がある。ここでは、代数的制御理論の基礎をなす安定有理関数上の既約分解(coprime
