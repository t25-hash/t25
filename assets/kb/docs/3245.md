# 2・2・3 最適サーボ系(optimal servo system)

> **章**: 第2章 線形制御
> **節**: 2・1 項で述べる最適レギュレータのほかに、閉ループ極を指定され

制御対象の出力(制御出力)を目標の参照入力に追従させるサーボ系の設計問題は、多変数システム制御のうちで最も基本的な問題であり、また応用上最も重要な問題の一つである。サーボ系の設計では、まず追従偏差を漸近的に零にする(出力レギュレーション)ために、閉ループ系を安定化する必要がある。しかし、それ以外に制御対象のパラメータが閉ループ系の安定性を損わない範囲で任意に変動したり、外乱が加わってもつねに出力レギュレーションを達成(ロバスト追従、robust tracking)させることが実用上重要になる。このようなサーボ系を特にロバストサーボ系(robust servosystem)と呼び、それを実現するサーボ補償器は参照入力のモデルをダイナミクスとして含まなければならないことが、内部モデル原理(internal model principle)として知られている。本項では、すでに述べた最適レギュレータ理論を応用してこのようなサーボ系を設計する。その結果得られるサーボ系は通常最適サーボ系と呼ばれている。以下では簡単のために参照入力はステップ入力に限定するが、ランプ入力、正弦波入力など、一般に参照入力が線形定数システムの初期値応答として実現できる場合にも容易に拡張できる。なお、ステップ参照入力に対する内部モデルは積分器であり、したがって、ステップ参照入力に追従するロバストサーボ系は、内部モデル原理によりその補償器内部に積分器が含まれることから、特に積分型最適サーボ系と呼ばれている。その設計法を述べる前に、まずその基本となる積分補償を含まない最適サーボ系の設計について述べる。a.
