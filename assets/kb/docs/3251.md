# 6・6・1 知識応用計測の範囲

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

知識応用計測（intelligence applied measurement ）をどのよう
に定義するかは，特に定まっていない．ここでは，センサから得ら
れる信号を，増幅したりフィルタリングしたり，非線形変換するな
どして所望の情報が得られるものに対して，そのようなデータに何
らかの知識に基づく計算が施されて初めて有意な情報が出現する計
測を，知識応用計測とする．
