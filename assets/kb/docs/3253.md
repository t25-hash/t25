# 6・6・3 反転法，マルチステップ法

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

高精度測定においては，測定の基準となるものの形状誤差等の大
きさが測定対象の評価値と同程度となり，計測値から両者を分離す
る方法が必要となる．その一つは反転法である．反転法（reversal
method）は，古くからある手法であるが，現在でも被計測物が達
成可能な参照基準や運動精度と同程度の形状偏差等で仕上がってい
る場合には，有効な手段として利用される．マルチステップ法
（multi step method）も同様である．反転法は，測定対象と基準と
の配置を反転（したがってセンサの検出方向も反転される）させ，
計測値に含まれる基準の形状誤差の寄与を逆向きにして，反転前後
の2回の測定値から両者を分離する方法である．直線形体の真直度
（straightness ) や円筒形体の真円度（circularity) などの評
価で用いられる．図6･68，6･69にその原理図を示す．それぞれ，
式(6･16)(6･18)のように観測方程式が立てられ，反転に伴う傾斜成
分や偏心成分であるα，βを取り除くことにより，それぞれ式(6･
17)(6･18)のように形状偏差と基準の誤差を分離することができる．
