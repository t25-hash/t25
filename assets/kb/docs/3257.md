# 6・6・7 GPS

> **章**: 第6章 各種応用計測
> **節**: 6・6 知識応用計測

GPS（global positioning system) は，ナビゲーションシステ
ムの中核として爆発的に普及し，携帯できるまでになった．その計
算原理は相関技術に負うところが大きいが，知識応用計測に属す
る．
対し，例えば脳計測のように，生体がある活動をしているとき，脳
のどの部分が最も活性であるか，あるいは，脳の指令がどのような
筋電位となって行動に結びつくかといった計測などにおいては，6･
6･5項に示した逆問題解法や，センサネットワークなどが関わって
くる．そこには，計測対象のモデル化とそのパラメータ推定，モデ
ルの再構成といったスパイラルな計測に関する真実探究への戦略が
存在し，計測研究の大きな一つのあり方を形成するように思われ
る．知識応用計測はその中核を占め，この分野ではあえて知識応用
などという必要がなくなる可能性がある．これは，計測論における
一頁を形作るものと考えられる．
