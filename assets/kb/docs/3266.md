# 2・6・2 分布定数系制御

> **章**: 第2章 線形制御
> **節**: 2・83 に示した集中定数部分の状態である。行列,,は適当

a. はじめに分布定数系(distributed parameter system)とは状態が時間変数と空間変数に依存するシステムのことであり、偏微分方程式と境界条件および初期条件によりシステムの挙動を表現することができる。ここで、式(2・380)の放物型偏微分方程式は熱伝導方程式、式(2・381)の双曲型偏微分方程式は波動方程式、式(2・382)のだ円型偏微分方程式はラプラス方程式を含んでいる。分布定数系が無限次元システムであることはよく知られており、その無限次元システムを有限次元制御器でいかに制御するかが問題である。分布定数系の制御のアプローチとしては大きく三つの流れがある。第一は、固有関数展開を用いた有限次元近似システムに基づいた制御系設計法である。第二は、分布定数モデルに基づいたリアプノフの方法による、有限次元近似を用いない制御系設計法である。第三は、発展方程式表現による無限次元系の抽象理論に基づいた制御系設計法である。b. 分布定数モデル機械系における分布定数システムの代表例として柔軟メカニカルシステムがある。まず、対象システムとして一端が制御用モータの回転軸に固定され、他端が自由で一様な1リンク柔軟アームを考える。また、重力は無視することとする。c. 有限次元近似(finite-dimensional approximation)システムに基づく制御分布定数モデルに基づいて制御系を構成する際に、分布定数系は無限次元システムであるのに対して制御器は実装の観点から有限次元でなければならない、という問題が生じる。このため、分布定数モデルを有限次元化することにより導出された有限次元近似モデルに基づいて、制御系設計を行う手法がよく用いられる。. 有限次元近似モデルまず、分布定数モデルの有限次元近似モデルの導出について考えてみよう。b項で導いた1リンク柔軟アームに対する分布定数システム式(2・385)(2・386)は減衰項をもたないので、開ループ系の極はすべて虚軸上に分布している。しかし、一般に減衰係数δは小さいので、振動モードのより速い減衰が望まれる。ここで、制御モードが可制御であれば、剰余モードの安定性をそのままに、制御モードの安定度を指定することができる。しかしながら、制御モードのうち直接計測できるのは剛体モードのみであるので、制御したい振動モードに関する状態を推定する必要がある。剰余モードによる影響であるに関する項を観測スピルオーバという。この観測スピルオーバにより、閉ループ系の安定性は保証されなくなる。したがって、剰余モードを考慮した制御系設計が必要となる。. 制御系設計二段階の有限次元観測器を設計することにより、無限次元システムとしての閉ループ系の安定性を保証する制御則が提案されている。しかし、この制御系は、システムのパラメータ不確実性に対してロバストではない。また、高域遮断特性をもった最適レギュレータや制御理論を用いたロバスト制御は無限次元システムとしての閉ループ系の安定性を理論的に保証するものではない。仮に、モデル化誤差が評価できたとしても、保守的になりすぎて、制御性能が著しく劣化する可能性も生じる。一般に、これらの制御器の次数は大きくなり、その構造は複雑になってしまう。また、分布定数系に対して、机上で無限次元制御器を構成し、その有限次元近似制御器を構築することもできる。しかし、この制御系は、次数を大きくしていけば、元の無限次元制御器に収束するように構成できるものの、有限次元近似制御器が本来の無限次元システムを安定化する保証はない。これらの手法を適用した場合に複雑な制御系しか構成できないのは、無限個のモードを切捨てて、有限次元化したことにより、本来システムがもっている性質が失われたからと考えることもでき6―75る。これらの問題点を解決するには、やはり、システム固有の性質をもれることなく反映させた、制御系設計が有効である。なお、c項では柔軟アームを対象とした振動方程式について説明を行ったが、熱伝導方程式などに対しても同様な手法が適用できる。d. 受動性(passivity)まず、システム式(2・385)~(2・387)のもつ固有の性質について考える。ここでは、制御により減衰を付加できることを明確にするために、柔軟なアームの内部減衰は導入しない(δ=0)。この結果は、アームが一様でなく(,ρが空間変数に依存)、線形近似を用いない場合にも拡張できる。したがって、対象システムの本質的な性質と考えられる。e. リアプノフの方法に基づく制御まず、d項で導出した受動性を生かした制御系設計について考える。ここでは、δ=0としているので、開ループ系の無限個の極はすべて虚軸上に分布していることに注意しておく。不変原理(発展方程式系における不変原理についてはf・iii項で述べる)を用いて、閉ループ系の漸近安定性が証明される。導出された制御則は、モータの角度()と角速度()とアーム根元のひずみ(strain)のセンサ出力の直接フィードバック(PDS 制御)であり、実装が容易である。また、制御系設計にシステムのモデルや物理パラメータが不要であり、パラメータの不確実性に関してロバストである。導出された制御系はPD 制御(=とした場合=0となる)も含めて、漸近安定性を保証していることに注意しておく。このほかに、ばね結合した2本の柔軟アームや2リンク柔軟アームなど、自由空間を運動するさまざまな柔軟メカニカルシステムに対して、リアプノフの方法を用いた制御系設計が有効であることがわかってきた。特に、2リンク柔軟アームのモデルは複雑な非線形分布定数系となり、分布定数モデルに基づいた制御系設計は困難であるように思われる。しかし、なんら有限次元近似を導入することなく、システムの全エネルギーを考慮したリアプノフ関数を構成することによって、閉ループ系の漸近安定性を保証するようなPDS 制御が導出できている。また、直列にばね結合した本の柔軟ビームが並進・回転運動するベースに取付けられた構造体に対して、並進・回転の(位置・角度)と(速度・角速度)6―76制御システムとモータに固定されたビームの根元のひずみとせん断力(share)のセンサ出力の直接フィードバック(PDSS 制御)を用いれば、閉ループ系の漸近安定性が保証されることもわかってきた。さらに、剛結合した2本の柔軟ビーム、1自由度柔軟アームによる力制御や2本の1自由度柔軟アームによる協調制御など、幾何学的拘束条件をもつ柔軟メカニカルシステムに、PDS 制御が拡張できることもわかってきている。e項で述べたPDS 制御、PDSS 制御などは柔軟メカニカルシステムを分布定数系として厳密にモデル化し、有限次元近似を導入することなく、システムの全エネルギーを考慮したリアプノフ関数を定義することにより導出できた。本来システムのもっている物理的本質をそのまま生かすことにより、実装が容易でロバストな制御系を構築できたと考えられる。これらの制御方策は、同様な手順で、さまざまな分布定数系の制御器を構成できる。しかし、その手法は対象に強く依存しており、統一的な制御系設計理論が確立されているわけではない。これに対して、さまざまな無限次元システムを統一的に表現し、制御系設計を抽象理論として展開する手法がある。これを次に紹介する。f.
