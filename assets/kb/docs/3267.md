> **章**: 第6章 情報システム
> **節**: 6・1 生産システムの情報化の体系

生産情報システム統合のための国際標準の作成は，近年盛んに進
められている．生産情報システムに最も関連が深いと考えられる国
際標準化機構ISO TC184：2003の構成を図6･2に示す（ISOは，
国際標準化機構：International Organization for Standardization
の略称であり，TCは，technical committeeの略である)．
図6･2国際標準化機構TC184の構成
TC184では，現在四つのSC（sub committee）が活動中であり，
各SCのもとでは，いくつかのWG（working group）に分かれて
活動が進められている．以下では，生産情報システムと密接な関係
をもつSCを紹介する．
活動は，ライフサイクル全般にわたる製品モデル（product
model ）の提供であり，通称STEP（standard for the exchange
of product model data）と呼ばれている ．一般に実世界にある
ものをコンピュータ上に表現するためにはモデルが必要となるが，
STEPでは，各種の用途で使用される製品モデルに関するデータ
モデル（data model ）を開発していることになる．

b.SC 1 （physical device control ：機械と装置の制御) この
SCで近年注目すべき規格は，WG7で開発が進められているCNC
データモデル（CNC data model ）である．このねらいは，従来か
ら広く使用されてきたNCプログラム言語（ISO6983：1982，通
称Gコード）に代わる新しいNC指令言語を開発することである．
この方法では，加工形状特徴（manufacturing feature，6･2節参
照）の情報およびその加工方法をNC制御装置に与えて，詳細な
工具軌跡の計算はすべてNC制御装置に行わせることになる．そ
のため，CNCデータモデルではおもに，加工形状特徴の加工方法
を記述するためのデータモデルが開発されており，このモデルも
STEP規格で開発されたEXPRESS言語で記述されている．
c.SC 5（architecture and communications ：アーキテクチャ，
通信およびフレームワーク) このSCでは，FAシステムに関
するフレームワーク（枠組）や通信規約などに関連する規格が開発
されている．その例として，図6･5にISO参照モデルを示す．こ
れは，製造制御システムを含めた企業内統合情報システムの設計な
どにおいて，システム機器の構成やソフトウェアの役割を明確にす
る面で有効と考えられる．
d.SC 2（robots for manufacturing environment ：工業用ロボ
ット) このSCでは，工業用ロボットに関する用語，ロボット
言語などが規格開発されており，近年では，ロボットの安全性に関
する規格も追加されつつある ．
ISO TC184以外に生産システムの情報化に関連する国際標準と
しては，ISO TC29/WG34：2004（通称small tools ）がある．こ
こでは，切削工具（cutting tool ）のもつ情報を表現するためのデ
ータモデルが開発されており ，今後もその動向を注目すべきと見
られる．
