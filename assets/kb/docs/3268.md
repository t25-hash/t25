# 2・7・2 LMI によるシステム解析

> **章**: 第2章 線形制御
> **節**: 2・7 LMI に基づくシステム解析と制御系設計

本項では、線形システムの重要な性質のうち、代表的なものに対するLMIによる定量的な解析を示す。LMIによるシステムの性能解析は文献(90)に詳しい。式(2・437)のシステムの伝達関数を式(2・438)とする。さらに、システムの応答特性と極の位置は深く関係しており、制御系設計においても極の存在領域を指定することは、重要かつしばしば用いられる設計仕様の一つである。. 代表的な領域図2・85に、極指定のための代表的な領域を例示する。実際にはこれらの共通領域がよく用いられる。図2・85(a)~(d)には、以下の各領域が示されている。これ6―79らの例の各領域に極が存在するためのLMI条件が以下のように与えられる。[定理2](極の存在領域の条件)図2・85の各領域(a)~(d)に行列のすべての固有値が存在するための必要十分条件が、それぞれ以下のLMIによって与えられる。しかも、連立により得られたLMI条件は、その共通部分への極の指定の必要十分条件であることが知られている。b.ノルム条件連続時間システム式(2・437)を考える。システムのノルムは、伝達関数周波数応答の仕様、入出力のゲインおよびRMS ゲイン、ロバスト安定性(obust stabil-rity)など、制御系設計における重要な仕様を表現することができる(2・4節参照)。ここでは、LMIによるノルム条件を示そう。[定理3](ノルム条件、norm condition)次の条件C1,C2は等価である。以下では、ノルムとゲインを合わせて性能と呼ぶ。そして、LMI条件C2をこの双対システムについて書き下し、ブロックの順番を入れ換える合同変換を施したものが条件C3になっている。前述の極の存在領域のLMI条件や、以降で述べるシステムの性質においても、上のLMI条件C2,C3に対応する双対なLMI条件が存在し、適当な合同変換により互いに変換することができる。以下では一方のタイプのLMIのみ示すことにする。c.ノルム条件ここでは、ノルムが有限となるための必要条件として=0を仮定する。[定理4](ノルム条件、norm condition)次の条件C1~C3は等価である。上に示したLMI条件C2,C3は、それぞれ可観測グラミアン、可制御グラミアンに基づくノルム評価を与えている。式(2・444)が成り立つとする。
