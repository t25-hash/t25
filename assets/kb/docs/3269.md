# 6・2・2 工程設計業務

> **章**: 第6章 情報システム
> **節**: 6・2 生産準備と情報システム

機械部品の工程設計業務は，図6･6に示す四つの作業から構成さ
れている．
この工程設計（process planning）を実施する目的としては，
･加工コストの算出
･製造部門への加工方法の指示
･生産管理部門への工程情報（使用設備，加工時間）の伝達
･ジグ，工具などのリソースの手配
などが挙げられる．
以下では，図6･6の作業内容を説明する．
a.加工部位および加工仕様の理解 これは，設計部門から受
取った製品仕様情報から，加工が必要とされる部位およびその加工
仕様を理解することである．従来からこの作業は，図面，または二
次元CADから出力される図面イメージの情報を対象として行われ
てきた．
部品形状が複雑な場合には，この作業は，工程設計者にとってか
なりの負担となることもある．
b.加工工程（machining process ）の設定 ここでは，
･対象部品の加工に必要となる工程数とその順序の設定
･各工程で使用する工作機械の選定
･各工程への加工部位の割付
･加工工程での部品の保持位置と保持方法の設定
などの作業を行う．この加工工程の設定業務を，狭義の工程設計と
呼ぶこともある．
この業務を行うには一般に，熟練した生産技術者が必要とされ
る．
c .加工作業（working step）の設定 ここでは上記b項で設
定された各工程に対して以下の作業を行う．
(１)加工部位ごとに，その加工に必要となる加工作業とその作
業順序を設定する．
(２)上記で設定された各加工作業に関して，使用する工具，工
具の動き方，切削条件，切込みしろ，仕上げしろなどの作業内容を
設定する．
(３)すべての加工部位について，上記の二つの作業を実施した
後で，対象とする工程全体での作業順序を設定する．
d.NCプログラムの作成 ここでは上記c項で設定された加
工作業情報および加工部位の幾何情報から工具軌跡を計算して，対
象工作機械ごとのNCプログラムを作成する（注：このNCプロ
グラム作成作業を工程設計業務に含めない考え方もある)．
