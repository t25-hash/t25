# 2・7・6 LMI によるゲインスケジューリング制御系の設計

> **章**: 第2章 線形制御
> **節**: 2・7 LMI に基づくシステム解析と制御系設計

従来より、非線形システムを制御するために、まずいくつかの動作点で近似線形化したシステムに対して制御則を求め、その他の点においてはその制御則の適当な補間により制御するという直観的な方法が用いられており、ゲインスケジューリングと呼ばれていた。一方、制御対象をあるパラメータに依存する線形システムととらえ、そのパラメータ値に依存した制御を行うことが近年提案されている。線形ロバスト制御理論を基礎におくことにより、システムを記述するモデルが成り立つ範囲で、安定性や制御性能を保証することが可能となった。ここではゲインスケジューリング制御(gainscheduling control)とは、近年発展した後者のアプローチを指すこととする。式(2・502)のように表現される制御対象を考える。[例5]このような状況の例の一つに、航空機における昇降舵の制御がある。昇降舵の角度とピッチレート、垂直方向機体速度との間には動的な関係があるが、それは高度、速度に依存して大きく変動する。高度、速度は観測できるから、それを用いて制御則を調整することで固定ゲインよりも高い性能が達成できると期待できる。(注意)式(2・502)では、以下での補償器の構成を容易にするために、パラメータは制御入力および観測出力に関わる係数にはスケジューリングパラメータが含まれないとした。そうでない場合は、制御対象の入出力端に適当なフィルタを導入することによって、式(2・502)の形に変換できる。フィルタの帯域を十分広くとっておけば、達成可能な制御性能を損なうことはほとんどない。式(2・502)を線形パラメータ変動システム(LPV システム:lin-ear parameter varying system)と呼ぶ。補償器として式(2・503)のLPV システムを考える。このような補償器をゲインスケジューリング補償器(gain scheduling control-ler)と呼ぶ。図2・94に、ゲインスケジューリング(図2・94ではGS と略記)制御系の構造を示す。係数行列は次式となる。ここでは、閉ループ系において二次安定性と性能を達成するゲインスケジューリング補償器を構成することを考える。具体的には、式(2・504)の閉ループ系について式(2・467)のLMIが成立するよう、式(2・503)の補償器の係数を式(2・506)の形で求める。条件C2が成り立つとき、式(2・508)を満たすゲインスケジューリング補償器の一つは式(2・511)で与えられる。他の制御性能についても、上の定理と同様にゲインスケジューリング制御系の性能を保証するLMI条件が次のように得られる。(1) 極の存在領域の指定パラメータが時不変の場合(もしくはゆっくり変化する場合)に有効である。図2・85の各領域(a)~(d)について、以下のLMI(ただし=1,...,を連立させる)が対応する。を補助変数とした式(2・512)のLMIが性能条件に対応する。また、複数のLMIを連立させることにより、多目的のゲインスケジューリング制御系設計も可能である。文献(1) 杉江俊治・藤田政之、フィードバック制御入門、(1999)、コロナ社。(2) 片山徹、フィードバック制御の基礎、(1987)、朝倉書店。(3) Franklin, G.F., ほか2名、 Feedback Control of Dynamic Sys-tems,4th ed.、(2002)、Addison-Wesley.(4) Dorf,R.C.and Bishop,R.H.,Modern Control Systems,9th ed.、(2001)、Prentice Hall.(5) 児玉慎三・須田信英、 システム制御のためのマトリクス理論、(1981)、386、計測自動制御学会。(6) 有本卓、計測自動制御学会論文集、2-1(1966)、1.(7) Potter,J.E.,SIAM J.
