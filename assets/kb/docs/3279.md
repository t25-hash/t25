# 6・5・2 制御ネットワークの位置付け

> **章**: 第6章 情報システム
> **節**: 6・5 製造制御システム

製造環境で用いられるネットワークは，場所あるいは用途によっ
て使い分けられており，用いられるネットワークも標準化されたネ
ットワーク，オープン化されたネットワークあるいはベンダ固有の
ネットワークなど多様に使い分けされているのが現状である．最近
ではFA機器のマルチベンダ化が進み，ハードウェアやソフトウェ
アの互換性がますます重要になってきている．
生産システムにおける情報ネットワークは大別すると管理情報系
と制御情報系に分けることができる．管理情報系では，国内・国外
とのネットワーク接続を行うWAN（wide area network）と工場
内通信におけるLAN（local area network）に分けることができ
る．いずれも通信を円滑に行うためのISO規格であるOSI （open
system interconnection）プロトコルやLANにおけるデファクト
スタンダードとなっているイーサネット（ethernet ）の採用などに
より，かなり標準化が進んでいる．一方，製造分野でのFAコンピ
ュータやコントローラなどによる制御情報系では使用目的により制
御レベルのネットワークと入出力レベルのネットワークが別れて存
在している．センサやアクチュエータなどの入出力（I / O）機器間
のネットワーク系は比較的標準化やオープン化が進展しているが，
制御機器間での制御情報の伝送あるいは上位の管理情報系との生産
情報の伝送に主として用いられる制御ネットワークでは，プライベ
ートネットワークの利用が多く，標準化あるいはオープン化された
ネットワークの採用が少ない．
工場ではさまざまなネットワークが稼働している．工場内のネッ
トワーク構成は三層の階層に分けることができる．図6･42に工場
内ネットワークの階層を示す．上位の工場管理レベルにおいては高
速光LAN，ISDN，インタネットにより工場外部との情報伝送を
行うとともに，工場内の基幹ネットワークとして生産管理情報や生
産技術情報のやりとりを行う制御レベルとの通信が主体である．制
御レベルのネットワークは生産システムの中核をなす数値制御装置
（NC），ロボット制御装置（RC）あるいはプログラマブルコントロ
ーラ（PC）などの制御装置とFAコンピュータ間を相互に接続し，
主として生産情報や制御情報の伝送に用いられる．さらに機器レベ
ルでは自動化機器の制御あるいは監視情報の通信を中心にそれぞれ
稼働している．わが国においては，これらの各レベルにおける管理
情報系としては，イーサネットが標準のネットワークとして稼働し
ている．制御・機器レベルにおいては各社のプライベートネットワ
ークと標準化あるいはオープン化されたFL-net （エフエルネッ
ト），OPCN-1（オーピーシーエヌ1），DeviceNet （デバイスネッ
ト）さらにPROFIBUS（プロフィバス）など種々のネットワーク
が稼働している．
製造制御ネットワークは，コントローラを主体とした制御系のネ
ットワークと，自動化機器のリモートI / Oやセンサ情報などの伝
送に用いられる機器レベルのネットワークとによって構成されてお
り，コントローラ間，I /O機器間あるいは上位のFAコンピュータ
間での制御情報や生産技術情報をおもに伝送する役割を担っている
ネットワークである．
