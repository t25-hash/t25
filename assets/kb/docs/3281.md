# 6・5・4 イーサネットをベースとした制御ネットワーク

> **章**: 第6章 情報システム
> **節**: 6・5 製造制御システム

イーサネットは主としてOA系の情報ネットワークとして標準
化されているが，近年この傾向がFA分野にも及んできている．現
在は10Mbps ～100Mbpsの伝送速度が主流である．さらに将来は
1Gbpsの高速LANへの規格化も検討が進んでいる．
イーサネットで使用されるTCP/ IP（transmission control pro-
tocol / internet protocol ）は情報伝送のプロトコルとして広く採用
されている．
そこで，イーサネットをFAコントローラ（FA controller ）間
の通信媒体として使用し，特に基本となるTCP/ IP，UDP/ IPプ
ロトコルの利用と，新たにFAコントローラ間通信を定義し，サイ
クリック伝送およびメッセージ伝送を実現するプロトコルを使用し
た制御ネットワークが実用化されている．
イーサネットをベースとした制御ネットワークでは，
･トータルシステム構築の容易性
･IEC1131-3への対応が容易
･パソコンとの高い接続性，互換性
･豊富なAPIの利用が可能
･オープン性が高い
･将来性が期待される
･ベンダが共通に開発可能
など多くの利点が考えられる．
