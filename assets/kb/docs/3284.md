# 6・6・2 生産システム用計算機とオペレーティングシステム

> **章**: 第6章 情報システム
> **節**: 6・6 生産システム情報化のためのソフトウェア基盤技

a.計算機ハードウェア 生産情報システム内で利用される計
算機は，大きく二つに分けられる．技術情報システムや生産管理情
報システム向けには，ビジネス向けPC（パーソナルコンピュー
タ）やサーバ機などはん用機が利用される．一方，生産プロセス制
御システムや設備に内蔵される計算機には，生産現場への設置に適
した専用機が利用される．すなわち，24時間連続稼働を前提とし
た耐久性・信頼性，耐環境性，処理のリアルタイム性，外部機器と
の接続性，自動復旧機能，保守部品等の長期安定供給などの特殊仕
生産現場で利用される計算機ハードウェアを，表6･3に分類す
る．FAコンピュータは，おもに工場のセルやフロア単位の制御に
利用されるが，はん用PCと類似のハードウェアを有するものも多
く，市販のパッケージソフト等も動作可能であることが多い．
PLC（programmable logic controller ）は，工作機械，ロボット，
自動組立機，搬送機などの設備や，センサ，アクチュエータ，操作
盤などに接続されシーケンス制御（sequence control ）を行う．ま
たマイクロコンピュータは設備の制御装置内に実装され，数値制御
（numerical control ）を行う．またセンサやアクチュエータごと
に，ネットワーク通信機能をもつ1チップ型小型プロセッサを接続
した分散制御システム（DCS:distributed control system）もプ
ロセス系生産システムで増加している．
b.オペレーティングシステム（operating system) 計算機
の性能向上と多機能化に伴い，FAコンピュータのみならず，制御
種類に大別される．
装置内のマイクロコンピュータにもオペレーティングシステムを搭
載した計算機が増えている．オペレーティングシステム（OS）は，
メモリ・ディスク・ネットワークなどのハードウェア資源を管理
し，アプリケーションがこれを利用するための基本ソフトウェアで
ある．はん用機では，Unix，Windows ，Linux等のOSが標準的
に利用される．一方，設備や機器の動作制御など応答時間に厳しい
高速性（数十 s ～数十ms以内）が求められる場合には，リアル
タイムOS（real time OS）が用いられる．リアルタイムOSには，
VxWork，iRMX，ITRON，OSEK/VDXなどがある ．また小
型機器組込用OSとして，Windows XP Embedded・Windows
CE.NET等も利用され始めている．
