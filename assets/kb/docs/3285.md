# 3・3・3 非線形

> **章**: 第3章 非線形制御
> **節**: 3・3 最適制御と非線形

制御まず、非線形システムのゲイン(gain)について、次いで、それをもとに状態フィードバックの非線形制御(non-linearcontrol)について述べる。これらはHJB 不等式とHJI不等式によってそれぞれ特徴付けられる。6―103a.ゲイン線形システムの制御では、ノルムが重要な役割を果たしている。この右辺をのゲインと呼ぶ。こうして、関係式(3・60)に基づいて、非線形制御問題ではノルムの代わりにゲインを用いて定式化する。線次の非形システムに対してゲインを定義する。[定義3](ゲイン)γは適当な正定数とする。線形システムのゲインと違って、一般に非線形システムのゲイン特性は大域的には(任意ので)成り立たない。ここで、時不変系なのでφは初期時刻には依存しない。また、0=0から原点近傍の任意の状態に有限時刻で達する入力が存在するとき、システムは局所可到達(locallyreachable)であると呼ぶ。このとき、線形システムの有界実補題の非線形版が次のように得られる。このとき、(1)の逆が成り立つ。これを式(3・69)に代入すると、式(3・67)を得る。□(2)の証明からわかるように有界実補題は最適制御問題と関連しており、式(3・67)はHJB 不等式と呼ばれる。有界実補題の名はこれに由来する。有界実性を一般化した概念として消散性(dissipativity)があり、これに関しても同様の議論ができる。さらにシステムの内部安定性を考慮した結果が、次のように得られる。このとき、(1)の逆が成り立つ。(1)の漸近安定性はリアプノフの安定定理から明らかである。この結果のほかに、システム式(3・62)の=0での線形近似系が原点で漸近安定で、かつγ未満のゲインをもつことと、元のシステム式(3・62)が原点で漸近安定で、かつγ未満のゲインをもつことが等価であることも知られている。このとき状態フィードバックによる非線形制御問題は次のように記述できる。[問題3](状態フィードバックによる非線形制御問題)正定数γが与えられている。有界実補題を用いると、この問題の解が得られる。□ここで、非線形制御問題と微分ゲームの関係について触れる。これを式(3・78)に再び代入すると、式(3・74)と似た式を得る。こうして、式(3・74)は、非線形制御問題におけるHJI不等式と呼ばれる。出力フィードバックによる非線形制御問題に対しても同様の結果が得られている。
