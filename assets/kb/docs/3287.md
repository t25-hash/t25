# 3・4・2 受動性に基づく制御

> **章**: 第3章 非線形制御
> **節**: 3・4 その他の設計法

受動性(passivity)とは非線形システムの入出力安定性の一つであり、この性質に基づいた制御手法を受動性に基づく制御(pas-sivity based control)と呼ぶ。特にロボットマニピュレータなどの機械システムがこの性質をもつことがよく知られており、人工ポテンシャル法などに利用されている。さらに通常の機械システムだけではなく、3・2節で扱った非ホロノミックな拘束をもつシステムやある種の電気回路等もこの性質をもつことが近年知られるようになり、幅広い物理システムに対して適用できる手法となっている。本節ではこの受動性に基づく制御の概要を述べる。a.
