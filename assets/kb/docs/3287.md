# 3・4・2 受動性に基づく制御

> **章**: 第3章 非線形制御
> **節**: 3・4 その他の設計法

受動性(passivity)とは非線形システムの入出力安定性の一つであり、この性質に基づいた制御手法を受動性に基づく制御(pas-sivity based control)と呼ぶ。特にロボットマニピュレータなどの機械システムがこの性質をもつことがよく知られており、人工ポテンシャル法などに利用されている。さらに通常の機械システムだけではなく、3・2節で扱った非ホロノミックな拘束をもつシステムやある種の電気回路等もこの性質をもつことが近年知られるようになり、幅広い物理システムに対して適用できる手法となっている。本節ではこの受動性に基づく制御の概要を述べる。a. 受動性まず、重要な役割を果たす受動性について述べる。受動性に関するより詳しい解説は1・3・4項を参照してほしい。ただしε>0.式(3・96)の左辺は系に蓄えられるエネルギー(蓄積関数(storage function)と呼ばれる)の増加率を、右辺は系に加えられる仕事率を表す。したがって式(3・96)[および式(3・97)]は、系に加えられるエネルギーよりも蓄えられるエネルギーのほうが少ないことを意味し、このことから受動的であるといわれる。b. 受動定理制御対象が受動性を有していれば、受動定理と呼ばれる安定性を利用して比較的簡単に安定化制御が行える。この定理を用いると、図3・5のような受動的な系と強受動的な系のフィードバック系において、新たな入出力が安定になること6―107が示せる(安定性に関しては3・3節参照)。いま制御対象が受動的だとすると、それに対して図3・5のように強受動的な系を構成してフィードバックすれば安定化補償器になることがわかる。つまり受動的な系は任意の正定行列を補償器として容易に安定化できることがわかる。この性質を利用した制御手法を一般に受動性に基づく制御と呼ぶ。c. 人工ポテンシャル(artificial potential)法上で見たように、通常のロボットマニピュレータは受動的であり、簡単に受動定理に基づいた制御を行える。いま式(3・98)の系を考えるが、簡単のためポテンシャル項=0を仮定しよう。受動定理に基づく安定化法では、制御対象の蓄積関数がリアプノフ関数の役割を果たす。したがって制御の観点からはこの関数が正定関数となることが望ましい。このように制御対象のパラメータを全く利用しないで補償器を設計できることから、受動性に基づく制御はロバストな手法として知られている。d. 軌道追従制御同様に受動性を用いた軌道追従制御問題を考えよう。正定対称行列Λを用いた次のような補償器を付加しよう。e. ハミルトン系(Hamiltonian system)受動性に基づく制御の応用範囲はロボットマニピュレータのような機械システムだけではない。非ホロノミック系や、受動性をもつ電気回路およびそれらと機械システムの組合せである電気機械システムにも同様の考え方が適用できる。そのような受動的な物理システムを表す枠組の一つが式(3・101)で表されるハミルトン系である。このことは次式のようにただちに確かめられる。ここでは電気機械システムの代表として、図3・8に示すような磁気浮上系を例にとってその手法を紹介する。この系はコイルの電圧を入力として、電磁気力により鋼球を浮上させる装置である。そのための最も簡単な方法は、式(3・101)のハミルトン系の形を保存することである。このようなハミルトン系の形を保つ変換を一般化正準変換と呼ぶ。図3・8の磁気浮上系に対しては、c項でロボットマニピュレータに行ったのと同じように蓄積関数となる新しいハミルトン関数を、次のような正定関数に指定してみよう。ここでαは平衡点での定常電流を表すための項である。この手順を図示すると図3・9のようになる。このようにハミルトン系の形式と一般化正準変換を用いれば幅広い物理システムの受動性に基づく制御が行える。また、座標変換を伴った一般化正準変換を用いることで、d項で扱ったような軌道追従も行うことができる。
