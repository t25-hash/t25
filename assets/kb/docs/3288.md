# 3・4・3 リシーディングホライゾン制御

> **章**: 第3章 非線形制御
> **節**: 3・4 その他の設計法

a. 問題設定リシーディングホライゾン(RH)制御(reced-ing horizon control)とは、各時刻において有限時間未来までの評価区間をもつ最適制御問題(optimal control problem)を解き、制御入力を決定する制御手法である。ただし、評価区間にわたる最適制御のうち、実際の制御入力としてはその最初の値のみを用いる。評価区間(horizon)が時間とともに後退(recede)することから、リシーディングホライゾン制御またはムービングホライゾン(MH)制御と呼ばれる。同様の手法はプロセス制御分野を中心に以前から用いられており、モデル予測制御(model predictive con-trol)とも呼ばれる。従来は主として線形系を制御対象とした研究が行われてきたが、近年非線形系への適用が盛んに研究されている。実際的な応用では離散時間で定式化されることが多いが、最適制御の観点からは連続時間で議論されることも多いので、ここでは連続時間で扱うこととする。以後、扱う関数は必要なだけ微分可能とする。制御対象は以下の状態方程式で表されるとする。許容入力関数の集合Ωは、区分的連続で、各時刻である閉集合⊂に属するような関数全体からなるとする。まず、次の評価関数を考える。そこで、ある固定された時刻までのみの制御性能を評価する代わりに、つねに一定時間未来までの制御性能を評価し続ける、という考え方もありうる。それがRH 制御である。非線形RH 制御の課題には、大きく分けると閉ループ系の解析と実時間アルゴリズムがある。閉ループ系の解析に関する話題としては、拘束条件や最適性と安定性との関係、そしてロバスト性とが挙げられる。これらの研究が進めば、安定性や性能を保証するためにどのような評価関数や拘束条件を選べばよいかが明らかになると期待される。一方、高速な実時間アルゴリズムが開発できれば、速いダイナミクスをもつ機械系などにも非線形RH 制御が実装可能となる。以下では、最適性の条件を整理するとともに、安定性とアルゴリズムについての考え方を概説する。次に、最適性の必要条件を考える。ハミルトン関数の最小値がつねにの内点で達成されるときは、最適軌道に沿って=0が成り立つ。RH 制御においては、現実の制御対象における時刻と最適制御問題における時刻とを混同しないよう注意する必要がある。また、一般には時変でもよいが、ここでは簡単のため一定とした。c. 安定性HJB 方程式を利用して閉ループ系の安定性を論じることができる。最適コストの単調性が意味するのは、評価区間の長さが6―110制御システム増えるとき評価関数の最小値が単調減少するということである。これは、終端ペナルティが一種の制御リアプノフ関数(CLF: control Lyapunovfunction)になっていることを意味する。また、終端ペナルティが無限評価区間最適制御問題の最適コストになっていれば、上式はHJB 方程式として等号で成り立ち、原点の漸近安定性もいえる。もちろん、一般に無限評価区間最適制御問題の最適コストを求めるのは困難なので、LQ問題の最適コストなどで近似することが考えられている。さらに、必ずしも最適性が成り立たない場合に安定性を示す研究もなされている。多くの場合、安定性を保証するには、制御対象の原点近傍での線形化が可制御であることが仮定される。d. 計算手法非線形RH 制御の計算には、基本的に通常の最適制御問題の数値解法が適用できる。また、非線形RH 制御に特化して計算効率の向上を図ったアルゴリズムも近年提案されつつある。しかし、主として最適化問題のヘシアンの構造を利用することを目指した研究であり、最適化手法としては通常の反復解法に基づいている。したがって、二次計画問題などを解いての探索方向決定と直線探索とを繰返さなくてはならない。最適制御問題の場合、評価関数の値を計算するためには状態方程式を評価区間にわたって積分(要するにシミュレーション)しなければならないから、直線探索一つをとっても計算量は無視できない。それに対し、RH 制御閉ループ系の微分方程式が実は二点境界値問題ではなく、初期値問題となっていることを利用することが可能である。このような考えに基づき反復解法によらないアルゴリズムが提案されている。これは未知量の変化を追跡するという意味で連続変形法(continuation method、またはホモトピー法、homotopy method)の一種である。適切な問題設定のもとでは、の次元が未知量の次元と等しい。上の方程式をニュートン法などの反復解法で解くのが一つの考え方であり、また、前述のような微分方程式に帰着させる方法もある。ここで、ζは解の追跡を数値的に安定化するための人工的な減衰係数である。近年の計算機の高速化とも相まって、10オーダ以上の次元の未知量を求める問題を、ミリ秒単位の計算時間で解けるようになってきている。e. 関連する話題非線形RH 制御(非線形モデル予測制御)の応用はプロセス制御を中心に広がりつつある。非線形状態フィードバック制御則をRH 制御として導く研究もなされている。論理変数を含むシステムのRH 制御も、システムの表現方法を含めて研究されている。計算方法に関しては、最適制御問題をテイラー展開で近似的に解く方法が提案されている。また、RH 制御が一定時間未来までの制御性能を最適化するのに対応して、一定時間過去までの推定誤差を最小化するMH 状態推定が研究されている。この場合も、RH 制御と同様の考え方により、未知量(状態推定値)の微分方程式を導くことができる。MH 状態推定とRH 制御を組合わせた出力フィードバック制御に関する理論的研究もある。
