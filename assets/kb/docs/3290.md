# 4・1・1 連続系のMRACS

> **章**: 第4章 適応・学習制御
> **節**: 4・1 モデル規範型適応制御系(MRACS)

MRACS(model reference adaptive control system)は、特性が未知あるいは特性が途中で変動するプラントを制御するための制御系の一つで、プラントパラメータに不確かさがあっても任意の参照入力に対して制御系の特性を良好に保持できる特徴を有する。MRACS の制御器は、制御系の出力(プラント出力、制御量)が規範モデルの出力(目標出力)に漸近的に一致するように適応的に構成される(ほとんどの基本構成は図4・1のようになる)。実際の適応は、制御器の可調整パラメータが適応機構でオンライン調整されることによって行われる。ここで規範モデルとは、参照入力から制御量に至る制御系の望ましい特性を記述したモデルで、通常、速応性や減衰特性などを考慮して決定される。漸近安定なMRACS を実現するためには、一般的に以下の仮定が必要になる。制御入力合成部が完全モデルマッチング(exactmodel matching)の制御器と同じ構造になっているので、プラントの次数や相対次数があらかじめ既知でなければならない、また、制御器はプラントの零点を相殺する働きをもつため、その安定性を保証する観点からプラントが非最小位相系であることが許されない。さらに、適応機構が正しく動作するためには、制御系にシステム外乱や観測雑音が存在してはならない。これらの仮定は非常に厳しいもので、すべてが満たされることは現実にはあり得ない。そこで現実には、次数や相対次数のミスマッチ、微小なむだ時間、外乱や雑音などの不確かさが存在しても安定に動作し、かつ、追従誤差が漸近的に許容範囲内におさまるようなMRACS の構成法が必要であり、これはロバスト適応制御法として知られている(4・1・3項参照)。MRACS の基本的な構成法は、制御器の可調整パラメータを適応則によって直接調整する直接法と、適応則が生成するプラントパラメータの推定値に基づいた設計計算を経て制御器の可調整パラメータが間接的に調整される間接法に大別される。さらにそれぞれの方式では、制御入力の合成方法と適応機構の設計法がいくつか存在し、目的に応じた組合せによってさまざまなタイプのMRACS が存在する。制御入力の合成方法はまず、(1)基本構成がモデルマッチング制御器と同じであるが制御器の可調整パラメータの最適値が未知であるため、その推定値を最適値とみなして制御入力を合成するCE原理(certainty equivalence principle)に基づく方法と、(2)同定モデルの出力がつねに目標信号に一致するように制御入力を合成するSMC(surrogate model control)法、また、その十分条件であるDyCE原理(dynamic certainty equivalence principle)に基づく方法に分けられる。さらに、それらの方法は入力合成時に割り算を必要とする方法と必要としない方法に細分化される。一方、適応機構の設計方法は、同定モデルがプラントの出力表現に基づく場合と入力表現に基づく場合に分けられる。さらにそれぞれの場合に、図4・1 MRACS の基本構成適応機構を駆動する誤差方程式が動的になるか静的になるかといった同定モデルに分類される。前者をベースにした適応則は誤差システムの強正実性とリアプノフの方法から導出されるのに対して、後者をベースにしたものは正規化手法を併用した最小二乗、最小二乗平均、指数重み付き最小二乗平均、ハイブリッド型などの各種こう配法から導出される。以上のような分類の組合せによってさまざまなMRACS
