# 4・1・2 離散系のMRACS

> **章**: 第4章 適応・学習制御
> **節**: 4・1 モデル規範型適応制御系(MRACS)

連続系のMRACS の設計法と離散系のMRACS の設計法の最も大きな違いは、後者において制御対象の特性を表す伝達関数の分母・分子の次数差の取扱いが容易であることと、パラメータ推定や制御系構成で用いる信号を特別に生成する必要のない点である。以下では、目標値に対する追従と、閉ループ系の特性の指定の二つの制御目的を達成する離散時間MRACS の基本的設計法を示す。ここでは、単一入出力の連続時間系で表される制御対象が離散化され、式(4・57・a)(4・57・b)によって記述されるものとする。設計に際して、制御対象は次の仮定を満足するものとする。このとき、上記の仮定(a)における制御対象の次数の上限は、連続時間伝達関数の次数に一致する。また、サンプリング周期が連続時間むだ時間の整数分の一に選ばれている場合は=-1、それ以外は=となる。仮定(c)は連続時間の場合の最小位相系の仮定に対応するが、連続時間系が最小位相系であっても、離散時間表現式(4・57)が逆不安定系となる場合が多く、実用的には大きな制約である。以下では、間接法による構成法を説明する。まず、式(4・61)の制御対象のモデルに対して次の同定モデルを考える。実用的な選択としては、以下のものが挙げられる。λ=1のとき、これは逐次最小二乗法(recursive least squaremethod)となる。この適応則は収束が速く、制御対象の特性が変化する場合にも有効であることが、応用例において実証されている。なお、ここで紹介した以外にも多くの適応則が提案、解析されており、いくつかの適応則は統一的な枠組で議論できることが示されている。なお、ここに示した間接法による設計は、直接法による設計と全く等価な適応制御系を与える(考え方、途中の式展開のみが異なる)ことが知られており、その構造はともに図4・2に示すものとなる。なお、離散時間MRACS における収束性の証明は続時間MRACS の収束性の証明に相前後してグッドウィン(Goodwin)らによって示されている。
