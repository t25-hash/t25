# 4・2・1 セルフチューニングコントローラの基本構成

> **章**: 第4章 適応・学習制御
> **節**: 4・2 セルフチューニングコントローラ

制御対象(システム)の動特性をあらかじめ把握することができない場合や,制御の途中でゆっくりと変動するような場合に対する制御手法として,セルフチューニングコントローラ(STC: self-tuning controller)がある.STCは適応制御の一手法として位置付けられており,その基本構成はカルマンによって提案され,その後1970年代になってアストローム(A°stro썦m)らによって離散時間確率系に対して理論的に体系化された.STCは図4・3からわかるように,三つの部分から構成されている.まず,パラメータ同定部において,制御対象の入出力データからシステムパラメータを推定する.次に,パラメータ調節部において,推定パラメータを真のシステムパラメータとみなして(CE原理),システムパラメータからコントロールパラメータに変換する.コントロールパラメータが計算されると,これにより制御入力がコントローラにおいて生成される.制御則としては,最小分散制御則(minimum variance control law),一般化最小分散制御則(generalized minimum variance control law),一般化予測制御則(generalized predictive control law)など評価規範の最小化に基づいて導出する方式か,極零配置制御則(pole-zero assign-ment control law),極配置制御則(pole assignment controllaw)など制御系の望ましい閉ループ特性を指定する方式がとられる.図4・3セルフチューニングコントローラの概念図6―119上述のようなSTCはExplicit型STCといわれ,制御対象の特性を把握したうえで制御系が設計されるため,プロセス系を中心に適用が試みられている.一方,制御対象の入出力データから直接コントローラパラメータを調整し,これを用いて制御入力を生成するImplicit型STCがある.
