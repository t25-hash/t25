# 4・2・2 最小分散制御

> **章**: 第4章 適応・学習制御
> **節**: 4・2 セルフチューニングコントローラ

対象とするシステムが式(4・97)で表されるものとする。式(4・97)と式(4・98)によるシステムは、以下の仮定を満足している。このことは、≠0を意味している。式(4・97)のシステムに対し、最小分散制御系を構成する。最小分散制御則は、式(4・99)の評価規範の最小化に基づいて導出される。まず、最適予測値に未来値が含まれないように、式(4・100)の恒等式ディオファンティン方程式(Diophantine equation)を導入する。したがって、最適予測値を式(4・103)として定義する。これまで、システムパラメータが既知として話を進めてきた。実際にはシステムパラメータが未知であるので、上述の最小分散制御法に基づいたSTCを設計する。4・2・1項において述べたように、STCはシステムパラメータを推定し、その推定結果に基づいてコントロールパラメータを調整するExplicit型と、直接コントロールパラメータを調整するImplicit型とに分類される。Explicit型STCの場合、システムパラメータを以下に示すパラメータ調整則[漸減ゲイン型逐次最小二乗法(least square method)]で推定する。これにより、最小分散制御に基づいたExplicit型STCを構成することができる。なお、推定パラメータが真値に収束するためには、仮定(1)~(5)に加えて仮定(6)として、次のような仮定(persistently spanning 条件)が必要となる。(6) 式(4・121)を満足する正定行列が存在する。
