# 4・2・3 一般化最小分散制御

> **章**: 第4章 適応・学習制御
> **節**: 4・2 セルフチューニングコントローラ

最小分散制御法では、が漸近安定多項式(すなわち、最小位相系)であるという仮定、あるいはむだ時間が既知という仮定が必要であった。しかし、零次ホールダを前置したうえで離散化(-変換)すると、連続時間系が最小位相系であっても離散時間系に不安定零点(unstable zeros)が出現し、非最小位相系(non-minimum-phase system)になることが報告されている。また、プロセス制御系などにおいては、むだ時間をあらかじめ把握することが困難な場合も多く存在する。したがって、これらの仮定はSTCの適用範囲を狭めていることになる。この問題を解決する方法として、さまざまな手法が考察されているが、クラーク(Clar-ke)らは、評価規範に制御入力項を含ませた一般化最小分散制御則(generalized minimum variance control law)を提案している。この手法は広く実システムに対して用いられており、その実用性も評価されている。そこで、4・2・3項ではこのGMVC法について述べる。対象とするシステムが式(4・122)で記述できるものとする。ここで、式(4・97)と式(4・122)の違いはむだ時間の取扱いである。また、全く把握できない場合には=0として考え、を式(4・123)を満足するように設定する。このことは、制御入力に制約を課したうえで、追従誤差の最小化を図るという意味に解釈できる。つまり、制御入力を評価規範に挿入したことにより、制御入力の発散を抑えることが可能となる。次に、式(4・127)のディオファンティン方程式(Diophantineequation)を導入する。また、≠0である必要もなく、むだ時間が未知である系に対しても適用可能であることがわかる(ただし、+≠0でなければならない)。したがって、最小分散制御法で必要であった仮定(2)と仮定(4)が不要となる。ここで、式(4・122)と式(4・135)により構成される一般化最小分散制御系のブロック線図を図4・5に示す。さらに、一般化最小分散制御系の入出力関係式は式(4・137)となる。この場合ステップ状の目標値に対し定常偏差が生じるため、式(4・139)におけるを式(4・140)のように調整する必要がある。この場合、ステップ状の目標値に対してはロバスト追従性が保証される。しかしながら、1-の影響で制御結果に残留振動が生じ、良好な制御結果を得るためのλの調整がかなり複雑になる。したがって、実用上は式(4・139)による一般化最小分散制御のほうが、簡単に制御系設計が行える場合が多い。なお、一般化最小分散制御法に基づくSTCの構成は、最小分散制御法と同様の手順で構成することができる。
