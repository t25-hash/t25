# 4・2・4 セルフチューニングPID 制御

> **章**: 第4章 適応・学習制御
> **節**: 4・2 セルフチューニングコントローラ

化学プロセスなどのプロセス系(process systems)においては、古くからPID 制御(PID control)法が用いられている。プロセス系は、運転条件の変更によりシステムの特性が変動したり、また、化学反応などによりシステムの特性が時々刻々と変動する場合も少なくない。このようなシステムに対しては、システム変動に対応して制御パラメータ(PID パラメータ)を適切に調整する必要がある。したがって、システム変動に対して制御パラメータを自己6―122制御システム調整するセルフチューニング制御法にとって、その有効性を発揮する意味において格好の制御対象であると考えられる。本項では、4・2・3項で紹介した一般化最小分散制御法とPID 制御法との関連に基づいて、PID パラメータを自己調整するセルフチューニングPID 制御法(self-tuning PID control method)について述べる。対象とするシステムの記述モデルとして式(4・142)を考える。また、むだ時間の取扱いは式(4・122)の場合と同様とする。次に、式(4・143)のようなディジタル型PID 制御則(digital PIDcontrol law)を考える。PID 制御系の設計においては、このPIDパラメータの調整が重要な問題となっている。なお、式(4・143)を式(4・144)として書き換えておく。すなわち、むだ時間が正確に把握できない場合は、を小さく見積もったり、を大きく設定することで上述の問題に対処することになる。しかしながら、の低次の係数や高次の係数の多くが零に近い値となり、その推定パラメータの相対誤差は極めて大きくなることが考えられる。すなわち、式(4・143)と式(4・152)の関係から、PID パラメータは式(4・153)として計算される。システムパラメータが既知である場合は、これまでに述べてきた方法によりPID パラメータを調整することができる。しかしながら、先にも述べたようにプロセス系の多くが非線形性を有していることから、平衡点の変更に伴って時々刻々とシステムの特性が変化する場合や、システムの動特性を前もって把握することが困難な場合には、システムパラメータに対応してPID パラメータを自己調整させる必要がある。そこで、セルフチューニングコントロールの考え方に基づいてシステムパラメータを逐次推定し、その推定結果を用いてPID パラメータを算出することで、セルフチューニングPID 制御法に拡張することができる。
