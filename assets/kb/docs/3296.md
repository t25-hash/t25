# 4・2・4 セルフチューニングPID 制御

> **章**: 第4章 適応・学習制御
> **節**: 4・2 セルフチューニングコントローラ

化学プロセスなどのプロセス系(process systems)においては,古くからPID 制御(PID control)法が用いられている.プロセス系は,運転条件の変更によりシステムの特性が変動したり,また,化学反応などによりシステムの特性が時々刻々と変動する場合も少なくない.このようなシステムに対しては,システム変動に対応して制御パラメータ(PID パラメータ)を適切に調整する必要がある.したがって,システム変動に対して制御パラメータを自己6―122制御システム調整するセルフチューニング制御法にとって,その有効性を発揮する意味において格好の制御対象であると考えられる.本項では,4・2・3項で紹介した一般化最小分散制御法とPID 制御法との関連に基づいて,PID パラメータを自己調整するセルフチューニングPID 制御法(self-tuning PID control method)について述べる.対象とするシステムの記述モデルとして式(4・142)を考える.=요-1+ξΔ(4・142)ただし,Δ=1-は差分演算子を示しており,およびは式(4・98)で与えられる多項式であるが,PID
