# 4・3・1 適応観測器の設計法

> **章**: 第4章 適応・学習制御
> **節**: 4・3 適応観測器と適応極配置

適応観測器(adaptive observer)は,制御対象のパラメータが未知な場合にもその状態を(同時にパラメータをも)推定可能な適応システムである.このとき,プラントパラメータが未知であることから,適応観測器は通常のパラメータが既知の場合の観測器構造に加えて,未知パラメータ同定機構が合まれることがその特徴である.始めに連続時間での設計法を示すが,この場合,パラメータが既知の場合の標準的手法であるルーエンバーガー(Luenberger)観測器の構造を基礎としない,プラントのパラメトリック表現に基づいた設計法のほうが適応観測器の構成を容易にすることが知られている.以下では,設計の容易なプラントのパラメトリック表現に基づく適応観測器の設計法を示す.a. 離散時間系の適応観測器これまでに紹介した適応観測器の構成法は,連続時間系を対象としたものであるが,制御対象が離散時間表現されている場合には,式(4・157)において,+1,,と対応させれば,離散形式でのプラント表現が得られる.適応観測器の構成も連続時間系の場合と形式的には全く同様となるが,離散時間系の場合,制御対象の過去の入出力のサンプル値を状態と等価に扱う表現が可能のため,離散時間適応観測器が適応制御系で実際に用いられる場面は比較的少ない.c. 多入出力系に対する適応観測器多出力系に対する適応観測器を構成するには,一般に二とおりの方法で1出力系の問題に帰着させることが有効である.一つはプラントのシステム行列がサイクリック(cyclic)な場合であり,また,プラントの可観測指数が既知ならば,多出力系に対する正準構造を利用して,プラントをいくつかのサブシステムに分割して1出力系の設計手法を適用することができる.なお,ここで述べた以外にも,適応観測器や適応同定器(adaptive identifier)の種々の設計法が知られている.
