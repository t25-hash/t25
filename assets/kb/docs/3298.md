# 4・3・2 適応極配置

> **章**: 第4章 適応・学習制御
> **節**: 4・3 適応観測器と適応極配置

system)を構成することができる。また、適応観測器の構成で導入した状態変数フィルタを多項式を用いて表現することで、状態変数を陽に用いない適応極配置制御系も構成可能である。さらに、離散時間系では制御対象の過去の入出力のサンプル値を状態と等価に扱うことで、状態フィードバックを用いない適応極配置制御系が構成される場合が多い。極配置制御系は、閉ループ系の極だけをあらかじめ指定した位置に配置することから、対象が非最小位相特性(逆不安定)の場合にも適用可能な制御手法である。これは特に逆不安定系の問題が生じやすい離散時間系で有用な性質である。a.
