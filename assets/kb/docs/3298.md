# 4・3・2 適応極配置

> **章**: 第4章 適応・学習制御
> **節**: 4・3 適応観測器と適応極配置

制御対象のパラメータが未知で、その状態を直接観測不可能な場合でも、4・3・1項に述べた適応観測器を用いることで、状態フィードバックによって閉ループ系の極を任意に指定できる適応極配置制御系(adaptive pole assignment control system)を構成することができる。また、適応観測器の構成で導入した状態変数フィルタを多項式を用いて表現することで、状態変数を陽に用いない適応極配置制御系も構成可能である。さらに、離散時間系では制御対象の過去の入出力のサンプル値を状態と等価に扱うことで、状態フィードバックを用いない適応極配置制御系が構成される場合が多い。極配置制御系は、閉ループ系の極だけをあらかじめ指定した位置に配置することから、対象が非最小位相特性(逆不安定)の場合にも適用可能な制御手法である。これは特に逆不安定系の問題が生じやすい離散時間系で有用な性質である。上記の構成法は、パラメータが既知の場合の状態フィードバックによる極配置制御系を適応化したものであるが、パラメータ調整機構をもつ適応制御独自の方法もある。b. 離散時間系の適応極配置制御対象が離散時間系の場合の適応極配置制御系は、制御対象の過去の入出力のサンプル値を状態と等価に扱うことができることから、状態フィードバックの制御構造をとることなく、また制御対象のむだ時間を知ることなく設計できる。このとき、制御対象を式(4・177・a)(4・177・b)で記述する。設計に際して、制御対象のモデルに対して以下の特性を仮定する。次に、設計の基礎となる以下の多項式方程式を導入する。パラメータ推定は一般に、式(4・177)を書き換えた式(4・182・a)に基づいて行う。なお、離散時間系では式(4・184・b)は逆行列の補題を用いて、次のように表現されることが多い。なお、ここで紹介した以外にも多くの適応則が提案、解析されており、いくつかの適応則は統一的な枠組で議論できることが示されている。上記アルゴリズムを用いて得られた制御対象の推定パラメータを式(4・178)に代入した式(4・187・a)の解を求めれば、適応制御時の入力発生に必要な推定値が得られる。ただし、パラメータ推定アルゴリズムによりパラメータθが正しく同定できるためには、適応観測器の状態やパラメータが真値に収束する条件と同様、信号ベクトルζがPS でなければならない。これは、適応制御系のロバスト性向上のためにも非常に重要な条件であるが、実際にそのような条件をつねに満たすことは困難である。なお、PS 性の仮定なしに適応系の漸近安定性を与えるアルゴリズムも提案されている。
