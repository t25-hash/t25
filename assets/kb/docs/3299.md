# 4・4・1 バックステッピング(back stepping)

> **章**: 第4章 適応・学習制御
> **節**: 4・4 非線形系の適応制御

ある種の非線形制御問題やそれに未知パラメータの推定が加わった非線形適応制御問題では,正実性や受動性(passivity)の概念が重要になる.しかし正実性や受動性の条件を満足するためには相対次数の障害が避けて通れず,特に相対次数が二次以上の対処方法が問題となる.バックステッピング法はもともとそのような非線形制御や非線形適応制御問題において,システムの受動性を相対次数が二次以上の場合に拡張する工夫として考案され,その後に線形系に対する拡張誤差を用いない適応制御系の構成法として提案されて脚光を浴びた.関連して非線形ダンピング(nonlineardamping),CE原理(certainty equivalence principle)に基づかない構成法,モジュラー設計(modular design),-項による補償(-term compensation),可観測正準形とstrict feedback form(ストリクトフィードバックフォーム)の類似性に着目した最小の調整パラメータ数による構成など,多くの概念や成果が得られた.さらにその後,ロバスト安定化問題やロバスト適応制御問題,分散系の制御問題,非ホロノミック系の制御問題などさまざまな非線形制御や適応制御の研究へとつながっていった.4・4・1項ではバックステッピングが最初に提唱され適用されたstrict feed-back
