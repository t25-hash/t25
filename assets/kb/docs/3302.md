# 6・5・1 ニューカーボンの種類

> **章**: 第６章 無機材料
> **節**: 6・5 ニューカーボン

定義が緩やかであるため，ニューカーボンの種類は非常に多岐に
わたっている．年代順にいうと，1960年代に開発された炭素繊維
は代表的なニューカーボンといえよう．炭素繊維の出現は，複合材
料に新しい局面を切り開いたといっても過言ではない．熱分解蒸着
炭素は気相熱分解という新しい製造方法によって得られ，炭素材料
の基本構造である六角網面の強い異方性をもつ材料として具体化し
たものである．ガラス状炭素材料は非晶質に近い構造でガス不透過
性，高い硬度など色が黒いことを除けば，性質はガラスに極めて近
いものでその開発はそれまでの炭素材料に対するイメージを一新し
た．高密度等方性炭素材料は，一般炭素材料の製造プロセスに静水
圧プレス法（CIP法：cold isostatic pressing）を採用することに
よって初めて得られた，均質で，異方性の少ない材料である．この
ほか，黒鉛層間化合物を出発原料とするシート状黒鉛は，脆性材料
である従来の炭素材料には全く期待できなかった可撓性を示してい
る．
このほか粉体形状のものでは，①表面修飾活性炭，②分子ふるい
炭，③メソカーボン，④フッ化炭素，⑤リチウムイオン二次電池負
極材などがあり，炭素繊維についても原料の違いによってアクリル
系，PAN系，レーヨン系，フェノール樹脂系などがあり，気相成
長炭素繊維もある．さらに気相成長技術を発展させた気相析出硬質
炭素，気相合成ダイヤモンドがある．用途面から見るとエネルギー
分野ではリン酸型あるいは固体高分子型燃料電池用部材，核融合炉
壁材，ガスタービンブレードへの展開などがあり，さらに炭素材料
は主体との親和性が良いことからインプラント材料として，またメ
ディカル用のX線検査装置，脱臭，吸脱着用材料など非常に多部
門にわたっている．最近の話題として，1996年度ノーベル化学賞
を与えられたフラーレンの発見があり，炭素原子60個がサッカー
ボール状に結び付いたC のほか，これに続くC，さらに大きな
高次フラーレン類，そして微細な中空体であり，高温における電子
デバイスへの応用が期待されるカーボンナノチューブもまたニュー
カーボンの仲間である．
このように，ニューカーボンについては枚挙にいとまがない．そ
こでここでは，ソリッドとしてほとんどそのまま機械用に利用でき
る材料について限定して述べる．
