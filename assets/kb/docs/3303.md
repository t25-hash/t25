# 4・5・1 繰返し制御(repetitive control)

> **章**: 第4章 適応・学習制御
> **節**: 4・5 繰返し・学習制御

一定の目標値に対しては、積分型のフィードバック制御により定常偏差を零にできることが知られている。繰返し制御は、これに似た原理を用いて、周期的な変動を目標信号あるいは外乱として受ける系の定常偏差を除去する制御であり、複雑な波形の変動に対しても追従偏差が除去できる。この特徴は、反復作業に従事するロボットの高精度動作の実現などに利用できる。a. 代表的設計手法繰返し制御系の設計においては、与えられた制御対象に対して安定条件(4・295)をいかに実現し、さらに応答の収束をいかに早くするか、外乱への応答をどう抑制するかが課題となる。ここでは、代表的な設計手法を紹介する。繰返し制御については、まず粒子加速器電源における実施結果とサーボ系に対する応用実験の結果が報告されたが、この時期に、古典制御論に基づく離散時間系としての設計が行われ、繰返し補償とともに安定化のための動的補償を行い、さらに必要に応じて繰返し制御器に帯域制限を施すという枠組が提示された。この枠組はその後連続時間上で理論化され、連続時間繰返し制御系における帯域制限の本質的な役割が明らかになるとともに、状態空間あるいは周波数領域における現代制御理論による新たな設計法として提示された。ただし、この設計法は、帯域制限に用いる要素への制約などから、実用上の価値はもたず、むしろ制御理論への寄与により評価されるべきものとなった。さらに、状態変数法に基づく離散時間系としての設計手法が提案された。離散時間系では、たとえむだ時間系でも有限次元となるので、従来の設計手法を利用できる。一方、有限次元とはいえ、むだ時間要素の内部状態も考慮するため制御演算が大規模になることと、特性面での考慮を欠くことが実用上の難点である。このほか、ディジタル繰返し制御系内外の変数を周期ごとに集めたベクトル量として扱うことにより、繰返し制御の1周期を1サンプリング時刻とする離散時間制御に対応づけるという見方もある。このような表現を使うと、繰返し制御が離散時間積分制御に単純化できることを利用して、複数周期間の演算によって非調和外乱特性を改善したり、時間的な順序にとらわれない制御演算が許されることを利用して特殊な機能を備えた繰返し制御を実現することができる。ただし、制御系の設計法は応用の目的に強く依存し、個別的な扱いを行うことになるので、ここでは触れない。これら各種設計法の基本については、成書『繰返し制御』にまとめられているので、その引用文献とともに参照されたい。なお、同書では、内山の反復制御、有本らの学習制御など、関連分野との理論的関係についても述べられているので、興味のある読者はごらんいただきたい。d.
