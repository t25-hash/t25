# 4・5・2 学習制御(iterative learning control)

> **章**: 第4章 適応・学習制御
> **節**: 4・5 繰返し・学習制御

学習制御という言葉は最近では広く学習機構をもつ制御系全般を指して使われることがあるが、もともとは試行の繰返しによって漸近的に目標の運動を達成する手法として研究された一連の制御則を指す言葉であった。ここでは、ロボットの運動の試行の繰返しからロボットに与えるフィードフォワード(feedforward)入力を更新し次第に目標とする運動を獲得していく、本来の意味での学習制御について取上げる。これらの制御則は、近年は英語ではiterativelearning control(ILC)と呼ばれ、広い意味で学習機構をもつ制御則を指すlearning control とは区別されるようになっている。a. 学習制御の前提条件学習制御では、与えられた一つの目標とする運動を達成するために、目標の運動の出力と実際の運動の出力の誤差を各試行ごとに測定し、その測定結果に基づいてロボットに実際に加えたフィードフォワード入力を修正して、次回の試行に用いる入力信号を生成する。学習制御が成立するための前提条件を示す。(1) 1回の運動は短い時間>0で終わる。(2) 学習制御の研究の流れ最初に学習制御と名付けられた制御則の研究は、1982年ごろから当時大阪大学にあった有本研究室で始められた。当初は対象を二足歩行ロボットとして、多変数の周波数領域で考えられたが、のちに時間領域の解析に移った。1984年有本らによって発表されたD 型学習制御は、ロボットの関節角加速度信号を関節トルク入力の修正に利用するものであった。この方式は実際のロボットに適用され、その有効性が実験によっても確認されている。その際には関節角速度信号はモータに取付けられたタコジェネレータから測定され、その速度信号を差分することで近似的に加速度を得ていた。のちに線形制御理論との関係として、D型学習制御では入力から出力への直達項が必要であり、誤差によって修正される入力システムの相対次数(relative degree)が零となるように微分操作が必要であることが杉江らによって強調された。1984年ごろD 型学習制御とともに、ロボットの関節角速度信号を関節トルク入力信号の修正に利用するP型学習制御と呼ばれる方法が、川村らによって検討された。P型学習制御の理論的な解析の結果、誤差収束には誤差システムの受動性が重要であることが明らかにされるとともに、その有効性が実験によって示された。1985年に川村らは指数関数で重みを付けた出力誤差のノルムを用いて、線形時変システムに関してはP型学習制御の収束性の証明を完全なものとした。さらにロボットが環境と接触して運動する場合に、接触によって生じる力の大きさが変位に比例する場合には学習制御が有効であることが理論と実験によって示された。また、おもにロボットダイナミクスに対して得られたこれらの結果は制御技術の興味を集め、美多らのフィルタを用いて高周波成分での収束を利用しない方式や、古田らの双対システムを用いて逆時間を利用する方式などが提案された。また、試行中の時間軸と試行回数を離散的な時間軸とみなした二次元システムとしての解析が宮里らによって示された。そのほかにもさまざまな学習の表現形式や説明が加えられるとともに、繰返し制御といわれる分野にも多くの成果が得られた。1987年ハウザー(Hauser)はロボットダイナミクスの可逆性を用いてD 型学習制御の制御対象範囲を広げている。すなわち従来は出力側に着目して誤差の収束を議論していたが、入力パターンと出力パターンが1対1に対応しているときには入力パターンが目標入力に収束すればよいことを指摘した。c. 対象となるダイナミクスの拡大最初期の学習制御の研究は二足歩行ロボットを対象としていたが、のちに電動モータによって駆動され、かつ手先が自由に運動する場合のロボットマニピュレータのダイナミクスを対象として、収束性の証明と実験による有効性の検証が行われた。その後、学習制御の対象となるロボットのダイナミクスのクラスが次第に拡大された。まず、マニピュレータ(manipulator)の手先が環境と接触して運動する場合に、接触によって生じる力の大きさが変位に比例する場合には学習制御が有効であることが、川村らによって理論解析と実験から示された。1990年代に入って、マニピュレータの手先が作業環境によって幾何学的に拘束される場合の位置と力のハイブリッド制御(hybrid control)の問題に学習制御を拡張する研究が進められた。D 型学習制御やP型学習制御をベースとする学習制御則に、手先に発生する力の誤差信号による更新則を付加することで、手先の位置軌道の収束性と手先力の収束性が証明された。また、単一のマニピュレータの手先が幾何学的拘束を受ける場合の拡張として、複数のマニピュレータやロボットハンドによって剛体の物体を協調して制御する問題に学習制御を適用する研究が行われている。ロボットが接触する対象物が剛体で変形しない場合はロボットの手先は幾何学的拘束を受けることになるが、接触する対象物が柔軟で接触によって変形する場合には力学的に異なる扱いをする必要がある。接触力が変位に比例する場合の収束性が証明されているほか、対象物が受動的である場合にD 型学習制御を基本とした制御則によって位置と力の収束性が証明され、シミュレーションによって有効性が検証された。また、柔軟接触環境にP型学習制御を適用する際の考察も行われている。その他のダイナミクスをもつロボットへの応用として、宮崎、阪口らが空気圧駆動のゴム人工筋肉やダイレクトドライブ(DD)ロ6―134制御システムボットに学習制御を適用して高速のジャグリングを成功させた。また、宮崎らは関節に弾性要素のある柔軟ロボットに適用可能な学習方式を提案した。ルカ(Luca)らもこの種の柔軟ロボットについて検討しており、さらに、冗長自由度をもつロボットにも学習方式を適用している。パラレルリンクマニピュレータについてもゲン(Geng)らによって学習制御が応用されていた。f. 学習結果の再利用学習制御では、与えられた一つの目標出力を実現するための制御入力を学習によって獲得する。そのため、目標の出力軌道が変われば、制御入力を獲得するために学習を一からやり直す必要があり、獲得した入力パターンを再利用することはできないと考えられてきた。川村らは、空間軌道が同一で時間軌道が異なる目標軌道が与えられる場合に、あらかじめ3種類の異なる時間軌道をもつ目標軌道に対する入力パターンをP型学習制御によって獲得しておけば、任意の時間軌道と同一の空間軌道をもつ目標軌道に対する入力パターンを学習済みの入力パターンから補完によって生成できることを示した。このとき、新しい目標軌道に対して学習の試行を繰返す必要はない。さらに、空間軌道が異なる目標軌道に対しても、参照軌道を実現するためにあらかじめ学習しておいた入力パターンから補完によって生成した入力によって良好な軌道追従制御が実現できることが、実験によって示されている。h.
