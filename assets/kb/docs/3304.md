# 6・5・3 ニューカーボン適用製品

> **章**: 第６章 無機材料
> **節**: 6・5 ニューカーボン

a.等方性高密度炭素材料 当初，機械的特性が優れているた
めホットプレス用型材として，また精密加工が可能でかつ電極消耗
に方向依存性がないことから放電加工用，電極用に開発され，使用
された．さらに全灰分を100ppm以下まで純化できることからSi
単結晶引上げ用ヒータ，るつぼあるいは封着ジグなど半導体用材料
としても多量に用いられている．このほか原子力関係では多目的高
温ガス炉用減速材，反射材などへ利用されている．
b.熱分解蒸着炭素材料 熱分解蒸着炭素材料は高純度で核的
性質に優れ，かつ不通気性で耐熱性が高いので核燃料の被覆材とし
て好適である．熱的異方性および比強度の大きい点ではロケットノ
ズル，スロート，ノーズコーンなどの宇宙航空機材，電気的異方性
を有する半金属的性質からエレクトロニクスおよびマイクロ波の分
野，不通気性と塑性を利用しての高温および腐食性流体に対するガ
スケット，熱膨張の異方性を利用した高温計，その他熱遮へい材，
ヒータ，ジグ用るつぼ，ボート，分析用電極などに応用されてい
る．
c.ガラス状炭素材料 ガラス状炭素材料は高純度であるうえ
に炭素粉の発生が少なく，吸着ガス量も少ない耐熱性の高強度材料
であることから，腐食性の強いGaAs ，Siなどの半導体用単結晶
引上げ用のるつぼ，サセプタ，ボートなどに使われる．また耐熱
性，耐リン酸性，耐酸化性，気体不透過性，小さい熱膨張係数，高
い熱および電気伝導度など非常に多くの要求項目のあるリン酸型燃
料電池用サセプタとしても採用されている．そして最近は平滑度を
高めたものはテープレコーダ，VTRやフロッピーディスクの垂直
記録磁気ヘッド用としても使われている．
さらに導電性と低反応性を利用した発光分光用電極，ポーラロ用
電極，電位差滴定用電極などがあり，金属に対する安定性から溶融
塩電解用るつぼへの利用，発じんが少ないことから歯根材，各種関
節等の生体用材料への展開もされている．
d.シート状炭素材料 グランドパッキンとして多量に使用さ
れるが極低温から高温まで広い温度範囲にわたって使用でき，その
間でほとんど性能の変化がない．また低い締付圧力によって十分に
シール効果を示し，増し締めの必要が少ないなどの利点があるため
各種バルブ，回転ポンプ，かくはん機，スートブロアほかのシャフ
ト部のシール材として使われる．
ガスケットとしては配管用，バルブ用，流量計用，ポンプ用，コ
ンプレッサ用，熱交換器用などのほかシリンダガスケット，マニホ
ルドガスケットなど自動車用としても多く使用されている．このほ
か高温炉用の断熱材，炉壁保護用内張材，クッション材などへの利
用もある．
