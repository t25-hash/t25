# 6・1・12 難削材の研削

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

難削材（diﬃcult -to-grind material ）のとらえ方は加工条件，
要求精度により大きく異なるが，一般に次に示す現象が現れて問題
となることが多い．
(１)研削抵抗が大きい．
(２)砥石の摩耗が大きい．
(３)研削焼けや研削割れが生じる．
(４)寸法精度や表面粗さが悪い．
(５)材料の脱粒やチッピングを起こす．
例えば，ダイヤモンド，セラミックス，純金属，耐熱合金，チタン
合金，オーステナイト系合金鋼などは代表的な難削材である．この
ような難削材を加工する場合，研削方式，研削盤の剛性や砥石駆動
モータの出力，砥石，研削油剤，研削条件などのさまざまな要因を
総合的に考慮する必要がある．しかし，基本的には難削性を示す原
因を解明し，それに対する対策を行うことにより，加工可能となる
ことが多い．難削性の原因としては以下の要因が考えられる．
(１)硬度が大きい．
(２)高硬度の粒子や繊維を含む．
(３)延性が大きい．
(４)熱伝導率が小さい．
(５)加工硬化しやすい．
(６)靱性値が小さく欠けやすい．
(７)活性度が高く砥粒や結合剤と反応，凝着しやすい．
実際には，これらの原因のいくつかが複合して難削性を示している
材料が多く，これらを加工する場合には，それぞれの原因に対する
対応策を試み，その中から最も効率のよい方法を選べばよい．
