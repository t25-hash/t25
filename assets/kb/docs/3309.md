# 5・1・4 進化的計算による探索機能の利用

> **章**: 第5章 知的制御
> **節**: 5・3 節参照)。重み係数は神経生理学で知られているシナプス伝達

a. 進化的計算進化的計算(evolutionary computing)とは、原理的にはダーウィン進化論の自然選択(淘汰)の考え方を取り入れたもので、システムとしての進化は変動する環境に適応する方向に生じるということが暗黙のうちに認められている。進化的計算法には遺伝的アルゴリズム、進化プログラミング、進化戦略、遺伝的プログラミングなど提案者によって異なった名前が付され、また内容も少し異なるが、ここでは代表的なものとして遺伝的アルゴリズム(以下genetic algorithmの略としてGA と記述する)を取上げる。一つの個体の遺伝子配列(生物ではDNA の並び)を遺伝子型(genotype)と呼び、GA ではこれを0と1の配列で表す。この遺伝子型はいわゆる個体の設計図に当たる。遺伝子情報により個体は発生と形態形成を経て一つの生き物となるが、これを表現型(phenotype)と呼ぶ。GA では進化論的考えを工学的問題に適用するわけであるから、逆にまず解こうとする問題の状態の集合(解の集合)を表現型の集合に対応させ、その集合の一つ一つの個体に対して、それぞれの遺伝子型を定義する。この関係をコード化といい、遺伝子型の集合は二進数の配列の集合である。通常の最適化問図5・5 GA による探索過程題では、なんらかの方法により、表現型である解集合の中を探索して最適点に至るが、GA では直接に解集合の中を探索するのではなく、遺伝子集合の中を探索して、最適解に至るところに特徴がある。図5・5にGA の過程を示す。個体の表現型には外部から与えられる基準による評価値が対応し、GA ではこれを適応度(fitness)と呼んでいる。おのおのの個体には外部環境による選択が作用する。すなわち、単純な方法では、適応度の低い個体の遺伝子型は集団から削除される。他方、遺伝子型集合では、二つの個体の交差(生物での交配)により新しい遺伝子型が生成される。この交差の方法についてはいくつかの提案があるが、単純な演算は、遺伝子型が二進数の配列であるので、二つの配列を適当な位置で切って部分ビット列を入れ換えるという方法である。また、自然界の現象としても認められていることであるが、遺伝子型は突然変異を受ける。これは演算としては、二進数表現の適当なビットを反転することになる。なお、この交差と突然変異の生じる頻度とビット列の位置の選択については、確率を設定する。アルゴリズムとしては以上の過程を繰返す。そして、集団として外部より与えられる要求に適した個体の多い方向へ進化していく。以上は単純GA と呼ばれている。この方法を適当な状態空間での最適探索法として見るとき、次の点に特徴が見られる。(1) 通常の極値探索で用いられる微分とか変分といったこう配に関する情報は利用しない。(2) 探索は集団を考えることによりなされるが、これは計算時間の増大を意味する。進化していく集団の中には適応度が高い個体も低い個体も存在する。この適応度を工学問題における最適化すべき評価関数と対応させるとき、適応度が高い個体が評価値が良いことはいうまでもない。b. 学習機能の応用進化的計算は原理的には最適点の探索を行うものであり、これを直接にPID のパラメータ調整などに利用することも可能である。また、これをルール表現された知識やニューラルネットワークの機能向上に利用できる。まず、ファジィ技術に関しては次のような内容が提案されている。(1) ファジィルールの前件部、後件部およびルール数の決定。ファジィ集合を定義するメンバシップ関数を、実験値などに合うように学習的に決めるなどである。(2) ルールの進化的獲得。ホランド(Holland)により、与えられたデータを分類するルールを学習的に獲得する分類システム(classifier system)が提案されている。ルールはIf-then型であるが、強化学習の枠組にておのおののルールの有用性を決定する過程と遺伝的アルゴリズムにより、新規なルールを生成する過程が用いられる。この手法はファジィクラシファイアシステムとして拡張されている。(3) 逆に、GA での個体数、交差確率、突然変異確率などの設定にファジィルールを生かそうという研究もある。またニューラルネットワークへのGA 応用は次のようである。(1) 階層型ネットワークの学習に、逆伝ぱ法でなく、GA を利用することができる。すなわち、ネットワークの集団を考え、適応度が増加するようにこの集団を進化させる。(2) 階層型ニューラルネットワークでは中間層の数、中間層のニューロン数を与えられたデータに合い、かつ構造が複雑とならない程度にこれらのパラメータを学習的に決定しようとするものである。
