# 6・1・5 砥石摩耗と砥石寿命

> **章**: 第6章 砥粒加工
> **節**: 6・1 研削加工の基礎

周速度に一致させて砥石を直線運動させることで，平面である砥石
の端面によって正確なインボリュート歯面を創成する．総形研削に
よるねじ・ボールねじの研削では砥石の干渉が起きることと，砥石
を溝形状に整形することに手間がかかることから，普通形状砥石あ
るいは不整形な砥石による創成法が試みられている ．
以上の幾何学的目的形状を得るために砥石切込み深さを設定して
加工を行った際に，工作機械や工具・工作物の変形等によって切り
取られる工作物の深さは設定値より小さくなる．図6･6はプランジ
研削におけるこの過程を示した例であり ，設定切込み（砥石台
移動量）に対して工作物半径は遅れて減少していることがわかる．
また，工作物の熱変形による切込み深さの変化も存在する ．
