# 5・2・3 状態評価ファジィ制御

> **章**: 第5章 知的制御
> **節**: 5・2 ファジィ制御(fuzzy control)

状態評価ファジィ制御は、人間による運転を制御対象システム自体のモデルを作成することなく、過去の操作経験に基づく状況の総合判断としてアルゴリズム化する方式である。例として、二つの観測データx とyから制御指令uを決定するような対象を考えると、「もし(If)x がA(高い)であり、yがB(高い)であるならば、(then)uをC(減らす)する」といった制御則により制御指令を決定する。ここで、「x がA」「yがB」「uをCする」といった命題は、その意味がファジィ集合により定義されたファジィ命題である。この状態評価ファジィ制御の制御則を一般的に示すと、If x is Aand y is Bthen u(i=1~n)なるIf-thenルールで記述できる。このルールを用いて対象システムから観測したデータに基づきファジィ推論することにより、制御指令を決めていく(図5・13)。a. 設計手順温度と圧力の観測値から状態評価ファジィ制御の設計手順はステップ1:対象特性の理解と操作経験則の記述熟練者が、そのような状況を見て制御指令を決定しているのかを観察し、運転の経験則を記述する。例えば、ルール1:もし、温度が高く圧力が高ければ、燃料を減らす。ルール2:もし、温度が低く圧力が低ければ、燃料を増やす。のような経験則である。ステップ2:言葉の意味の定量化次に、上記経験則に記述されている状況を表す、「温度が高い」「圧力が低い」などの言葉を、図5・7で示したようなファジィ集合で定義する。システム状況を等分割する場合も多いが、各状況を意味する適切なラベル付けを行う。ステップ3:経験則のファジィ制御則化ステップ2で定義したファジィ集合を用いて、ルール1:If‘温度が高い’and‘圧力が高い’,then‘燃料を減らす’.ルール2:If‘温度が低い’and‘圧力が低い’,then‘燃料を増やす’.の制御則をIf-thenルールの形式で記述する。この制御則は計算機でその意味を用いて処理できる知識である。ステップ4:制御則と言葉の意味の調整記述した制御則とファジィ集合を実際の観測データに当てはめ、思いどおりの制御指令が計算されているかチェックする。この計算は、ファジィ推論のアルゴリズムとそこで使用する知識に分割されているため、不具合部分の確認を容易に行える。細かな調整はそこに影響を及ぼしているファジィ集合を調整することで解決できる。新たな意味付けが必要な場合には、それに対応した制御則を知識として追加する。また、あまり効果のない制御則は、統合化して知識6―142制御システムの洗練化を図る。b.
