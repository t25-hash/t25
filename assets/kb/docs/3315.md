# 5・2・6 ファジィ制御器の設計

> **章**: 第5章 知的制御
> **節**: 5・16 )。

ファジィモデルに基づく制御の手順は手順1:制御対象をファジィモデルで表現する手順2:ファジィモデルの一つの規則に対して、ファジィ制御器の一つの規則を構成する(並列分散的補償)手順3:線形行列不等式条件を解くことにより、ファジィ制御器のフィードバックゲインを求めるの三つからなる。手順1については5・2・6・a項、手順2については5・2・6・b項、手順3については5・2・6・c項で述べる。a. 高木・菅野ファジィモデルファジィモデルに基づく制御では、制御対象のダイナミクスを高木・菅野ファジィモデル(Ta-kagi-Sugeno fuzzy model)で表現する。高木・菅野ファジィモデルのIf-thenルールは式(5・8)のように表現できる。このモデルの特徴は、後件部に状態方程式を用いている点である。後件部に状態方程式ではなく一般化プラント表現を用いるファジィモデルも提案され、このモデルに対する制御も行われている。式(5・8)のプラントルールを用いたファジィ推論は次のように行われる。また、式(5・9)のファジィモデルがuniversalであることが示されており、式(5・9)を用いることで任意のスムースな非線形システムを任意の精度で近似できる。ロボット、機械システム、電気システム、カオスシステムなど広いクラスの非線形システムが、厳密にあるいは高い近似精度で式(5・9)のファジィモデルに置換え可能である。複雑な非線形モデルをファジィモデルに置換える場合、プラントルールの数が膨大となる。見通しの良い設計のため、ファジィモデルの規則数を減らすルールリダクションやルールリダクション誤差を考慮した設計法なども論じられている。制御対象の非線形モデルが得られない場合は、入出力データを用いてファジィモデルの構造とパラメータを同定する。同定手法としてさまざまな方法が提案されている。b. 並列分散的補償並列分散的補償(parallel distributedcompensation)は、式(5・8)のファジィモデルに対してファジィ制御器(fuzzy controller)を設計する一つの方法である。この方法では式(5・10)のようなファジィ制御規則を構成する。式(5・8)のファジィモデルがr個の規則を有していることから、式(5・10)のファジィ制御器も個の規則から構成される。式(5・10)の後件部には状態フィードバック(state feedback)を用いているが、必要に応じて出力フィードバック(output feed-back)、動的出力フィードバック(dynamic output feedback)などを用いることもできる。動的出力フィードバックについては5・式(5・11)を式(5・9)に代入してまとめると、ファジィ制御系全体は次のように記述できる。しかし、この設計では制御系(5・12)の安定性は一般には保証されない。したがって、制御系全体のグローバルな安定性や制御性能を考慮して、を求める必要が生じる。5・2・6・c項では線形行列不等式に基づいて制御系全体のグローバルな安定性や制御性能を考慮した設計法を述べる。c. 線形行列不等式に基づく設計ファジィ制御系の安定条件だけでなく、decay rate、入出力制限(constraints on controlinputs and outputs)、外乱除去(disturbance rejection)、制御(control)、二次形式評価関数の上限値最小化制御、ロバスト安定条件(robust stability)、動的出力フィードバック、ファジィオブザーバ(fuzzy observer)設計などさまざまな制御性能が、線形行列不等式で記述できることを示す。線形行列不等式の特徴は、これらを組合わせることで組合わせたすべての条件を考慮する制御系が設計できる点である。一例として、ロバスト安定性と二次形式評価関数の上限値最小化の両者を考慮した制御について述べる。
