# 5・2・7 ファジィ適応制御

> **章**: 第5章 知的制御
> **節**: 2・6 ・c項で簡単に触れる.

ファジィ適応制御(fuzzy adaptive control)やファジィ学習制御(fuzzy learning control)は,なんらかの適応手法や学習手法を使ってファジィ制御規則の獲得やパラメータ調整を試みるものであり,古くはルールベースに基づく自己組織化制御の研究がある.これらのアプローチでは,制御規則がない状態,あるいは,少しだけ与えられた状態から制御性能を改善するように制御規則を作成していくものであった.その後,ニューラルネットワーク(neuralnetwork)の学習能力を利用して,ファジィ制御規則のパラメータを自己調整する手法が数多く提案された.現在ではニューラルネットワークだけではなく,遺伝的アルゴリズムなどの他のソフトコンピューティング(soft computing)手法を用いた学習,あるいは適応制御に関する手法も提案されている.ファジィ適応制御の理論的研究としてはLi-Xin Wangのアプローチが有名である.制御対象の多様化から,近年では,制御対象が線形,非線形,定常(時不変),非定常(時変)のいずれの場合でも動作する新しい制御系が望まれている.ニューラルネットワークの基本は,非線形な入出力関係を表現可能なニューロンモデルである.これらを人工的に多数結合し,並列分散的に情報処理を行うものがニューラルネットワークである.ニューラルネットワークは学習や自己組織化能力をもち,さらには汎化能力などの性質を有する.このため,ニューラルネットワークは非線形系の動特性を表現でき,学習によって動特性を獲得することが可能となる.この特質を生かし,非線形で時変といった特性をもつ制御対象のための新しい制御手法として,ニューロ制御に大きな期待がかかっている.また,制御対象が複雑化すると,そのモデルを数学的に表現することが困難となる.さらに,制御システムのアクチュエータやセンサの数が増え,対象システムが大規模化すると,制御系の調整に非常に手間がかかる.この点,ニューラルネットワークは学習機能を有しており,エンジニアのインタフェースとして使い勝手がよい.ニューラルネットワークを制御系に組込むことの理由は以下のように要約できる.(1) 任意の非線形写像関数を表現できる.(2) 学習則が比較的単純である.(3) 多種多様なデータを並列的に活用できる.ニューロ制御は制御法則として見ると,数学的に未完な部分が多い.現在は制御システムの知能化という新たな方向性への変革を行い,固定された枠組から脱皮して発展を遂げるうえでの一過程とも考えられる.ニューロ制御は,これまで,ロボットマニピュレータの位置,軌道制御や,振動制御,力制御,インピーダンス制御など,さまざまな場面に応用され,その研究報告が多数ある.例えば,振動制御にニューロ制御を応用したものとして,最適制御系を参考とし評価関数を最小とする方法や,フィードバック誤差学習によりフィードフォワード制御入力を獲得するものなどが提案されている.また,力制御,インピーダンス制御に関してはPID 制御ゲインをNN の学習によって獲得するもの,逆ダイナミクスをNN により表現させ目標インピーダンスを入力するものなどが提案されている.
