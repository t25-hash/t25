# 6・2・2 内面研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

穴の内面を研削する作業を内面研削（internal grinding）とい
う．内面研削盤には，工作物が回転する方式と工作物は回転せず砥
石が遊星運動するもの（プラネタリ型）の2種類があり，前者が一
般的であるが，精度を重視するジグ研削ではおもに後者が用いられ
る．図6･22は，工作物を(ａ)ロールや(ｂ)シューで保持して研削
する，心なし方式を示す．これらはおもに，ベアリング内，外輪の
ように，同心度が重視される場合に適用される．内面研削にもトラ
バース研削方式とプランジ研削方式があり，トラバース研削では深
穴やテーパ穴の加工ができる．プランジ研削では種々の軸断面形状
の穴を加工できる．
内面研削では，砥石軸が細長くなる場合が多いため，軸のたわみ
により円筒度が低下しやすい．このため，低切込み，高速回転で研
削を行い，研削抵抗を低下させるのが望ましく，毎分数十万回転に
及ぶものがある．この場合，砥石軸が振動を起こしやすいことか
ら，共振点を避ける配慮が必要である．内面研削では砥石と工作物
の接触弧長さが長くなり，切れ刃当たりの切込みが小さくなること
