# 5・3・3 ニューロ制御における学習アルゴリズム

> **章**: 第5章 知的制御
> **節**: 5・3 ニューロ制御

ここではニューラルネットワークにより逆システムを表現するための代表的な学習方法について述べる。図5・33に学習のための構成図を示す。学習は基本的にオフラインで行う。一般化学習でも1ステップ先の目標値を入力し、安定な出力フィードバックを付加した場合はオンライン学習が可能である。図5・34に学習と制御のブロック線図を示す。ここで、点線が学習するときのニューラルネットワークへの入力で、実線が制御入力を出力するときの入力である。学習データを得るためには出力フィードバックが必要であることが理解できる。b. 特殊化学習(specialized learning)制御対象の出力と目標値の差が少なくなるように、直接ニューラルネットワークを調節する。図5・35に学習のための構成図を示す。学習は一般にオンラインとなり、制御対象の出力の入力に対する偏微分(ヤコビアン)が必要となる。特殊化学習ではヤコビアンが重要な意味をもつため、以下に代表的な例として多層ニューラルネットワークに対して誤差逆伝ぱ法(error back propagation method)を用いて逆システムを学習するときの学習アルゴリズムを導く。いま、ニューラルネットワークを図5・36に示すような多層ニューラルネットワークとし、ネットワークの結合係数を最急降下学習則に基づいて学習する場合を考える。ここで、式(5・61)をさらに分解するために、制御対象が結合係数を固定した追加的な層であるとし、制御対象の出力と目標値の誤差が制御対象を通して逆向きに伝ぱされるとみなす。ここで、ニューロンの関数が式(5・44)で表されるようなシグモイド関数の場合には、式(5・65)のように微分計算が必要なくなる。以上は誤差逆伝ぱアルゴリズムと呼ばれている。c. ヤコビアンの計算方法式(5・64)(5・70)のように、特殊化学習の場合には制御対象のヤコビアン(Jacobian)が必要である。未知の非線形制御対象の場合にはヤコビアンを解析的に求めることはできないため、なんらかの対策が必要となる。ただし、並列構造のニューロンモデルの場合には、制御対象の数学モデルのヤコビアンを求めればよいため、厳密なヤコビアンを求めることができる。制御対象のヤコビアンを近似的に求める手法として次のようなものが提案されている。(1) 制御対象の順モデルを同定し、ヤコビアンを計算する。(2) 適応制御と組合わせてヤコビアンを獲得する。(3) 学習制御で得られる入力の修正量をヤコビアンに置換える。(1)の手法では、ニューラルネットワークによって順モデルが得られれば、ネットワーク内の結合重みを利用して、ネットワークを前向き方向に順次計算することによりヤコビアンを計算することが可能である。ただし、同定用のニューラルネットワークが順モデルを正確に表現できているという前提条件が必要であり、また、制御対象の動的な変動に追従するには同定用のニューラルネットワークの収束速度が高速でなければならない。(2)の手法ではニューラルネットワークによって制御対象を線形化し、この未知線形化システムにロバストな適応制御系を構成する。このとき、適応制御における調整パラメータを用いると、ヤコビアンが計算できる。(3)の手法ではまず、学習制御法として有本らが提案しているベターメントプロセス(betterment process)により入力の修正量を求める。次に、この修正量が誤差関数を減少させる方向に働いていると考え、誤差の最急降下方向であるヤコビアンと置換えて、ニューラルネットワークの学習を行う。d. フィードバック誤差学習(feedback error learning)制御対象の出力フィードバック信号を用いてニューラルネットワークを学習する手法であり、ニューラルネットワークの結合係数は逆ダイナミクスを補償するように更新される。図5・37に学習のための構成図を示す。フィードバック制御系は安定であるという前提条件のもとで、オンラインで学習を行う。特に中間層と出力層間の結合荷重はフィードバック信号を誤差信号とする異シナプス可塑性に従って変化させる。結合係数の変化量を以下に示す。中間層と入力層間の結合荷重は誤差信号をバックプロパゲートして変化させる。フィードバック誤差学習では、学習が進むにつれてニューラルネットワークがフィードフォワード出力を出して逆ダイナミクスを補償し、徐々に線形コントローラの出力を補いながら逆システムを構成する。フィードバック誤差学習はフィードバック制御系が安定であるという前提条件が必要である。このため、制御対象が全く未知な場合には安定なフィードバック制御系を構築すること自体が困難となる。しかし、制御系の安定性が確保できれば学習機能により逆ダイナミクスを獲得し、制御性能を改善することが可能である。
