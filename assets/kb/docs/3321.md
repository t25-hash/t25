# 6・2・3 心なし研削

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

心なし研削（centerless grinding）は，図6･23のように研削砥
石と調整砥石（regulating wheel ）の間に工作物を導入し，受け板
（work rest ）で支持しつつ，その外周を研削する方法である．工作
物が砥石全幅にわたって支持されるため，細長い工作物の加工が可
能である．また，工作物の脱着が容易で，自動化しやすく，加工能
率も高い．調整砥石の軸心は，図6･24のように研削砥石の軸心に
対し傾いており，工作物を一方向に押し付けたり送ったりするため
の力を発生する．工作物外周と調整砥石外周を工作物全長にわたっ
て接触させるため，調整砥石は鼓形に成形しておく必要がある．
心なし研削には，図6･25に示す(ａ)通し送り法，(ｂ)送り込み
法，(ｃ)端送り法の3方式がある．図6･25(ａ)では，砥石幅以上
の長さの工作物が連続的に加工できるので，長尺棒材などの外面研
削に適用される．図6･25(ｂ)では，段付き工作物の総形研削やテ
ーパを有する工作物の加工が可能であるが，工作物の直径の異なる
部分で調整砥石との間に滑りを生じる．図6･25(ｃ)では，図に示
すようなつばのある工作物の加工が可能である．
(c-1)横軸テーブル往復型
(c-2)横軸両頭型
図6･26 平面研削作業の方式
心なし研削では研削抵抗に起因する工作物のたわみを生じにくい
ので比較的高精度な加工が可能であるが，円筒面の一部が欠けてい
るなど，母線の長さが変化するような工作物の場合には真円度が低
下しやすい．
