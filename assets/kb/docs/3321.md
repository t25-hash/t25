# 5・3・4 ニューラルネットワークに基づくファジィ制御器設計

> **章**: 第5章 知的制御
> **節**: 5・3 ニューロ制御

(design of fuzzy controller)ファジィ制御(fuzzy control)の特徴が,ヒューマン・マシンインタフェースの良さにあることは多くの人の認めるところである.経験的な制御ルールをファジィ関係として記述し,ファジィ推論により操作量を求めるファジィ制御の手法は,一般にわかりやすく,制御ルールの設計,変更ならびにオペレータの介入などを容易にしている.最近ではファジィ制御系の安定解析および設計法の研究が進められている.一方,ニューラルネットワークは学習機能をもち,任意の入出力関係を同定できるという特徴がある.ベテランが経験を通して獲得した制御ルールをファジィ形式で記述した場合,最も大きな問題はファジィルールの学習である.この学習にニューラルネットワークを組合わせることが有効である.ニューラルネットワークとファジィ推論の制御への適用は,ファジィ制御ルール獲得(rule acquisition)の労力の低減,ファジィルールの同定とメンバシップ関数の調整の自動化などメリットは多い.ニューラルネットワークは,制御ルールの中で人間が記述することのどうしても難しい知識の獲得に用いられたり,ネットワークの結合構造をファジィ推論ルール獲得に適した形にするなどの工夫がなされている.a. ファジィ推論とニューラルネットワークの融合(fusion offuzzy inference and NN)ファジィ推論とニューラルネットワークの融合には種々の方法がある.図5・38はニューラルネットワークとファジィの融合手法の分類の一部を抜粋して示す.ニューロ/ファジィはニューラルネットワークとファジィ推論を並列に接続し,ニューロ-ファジィ,ファジィ-ニューロはそれらを直列に接続する手法である.ニューロ的ファジィは,ファジィ推論ルールのパラメータであるメンバシップ関数のチューニング・後件部定数の同定などにデルタルール学習則を適用する手法である.ファジィ的ニューロはニューラルネットワークの構造を改良してファジィルールを表現するようにした融合法である.(a) ファジィ制御への適用言語ルールが人にわかりやすいということと,経験的な制御ルールを記述することが容易であることとは別である.人の認知挙動に関するモデルとしてラスムッセンの階層モデル(Rasmussen’s hierarchical model)がよく知られている.このモデルによれば,ベテランのもつ経験的な制御ルールは,無意識的になされるスキルベースレベルから明確な状況認識とタスク処理を行うルールベースレベルまで,本人の自覚を伴うことなく分布している.ベテランは自らの操作がどのレベルのルールに基づこうともなんら不都合は感じないが,それらルールの明示化を要求された際には困惑する.ベテランはスキルベースレベルの制御ルールには対応する言語をもっていないことが多い.ファジィ制御はスキルベースレベルの制御ルールの記述に有効とされるが,それはこれらのルールが明示化されることが前提である.ルールの明示化はファジィ制御の重要課題の一つである.なお,このモデルの最上位の知識ベースレベルでは,不慣れな状況などにおいてより高度な判断・立案などがなされる.. ニューロ/ファジィ(neuro/fuzzy)ニューロ/ファジィの特徴は,ベテランから聞き出しうるルールはファジィルールで記述することにより人間の介入を極力容易にし,どうしても記述の難しい微妙な勘所をニューラルネットワークによって獲得させることで,知識獲得の労力の軽減を実現していることにある.すべてのルールをニューラルネットワークにより獲得することが可能であっても,入出力関係をブラックボックスとしてしまっては,人間が入出力関係を把握することが困難となってしまい保守性に欠ける.例えば,一部のルールの更新・調整などに,毎回,ニューラルネットワークの学習をやり直していたのでは手間がかかりすぎてしまう.また,ニューラルネットワークが獲得したノウハウをわれわれが全く知ることができなければ,ルールの改善策などを考えることも難しい.本手法は,ニューラルネットワークの使用を記述の難しい部分に限定している[図5・38(a)].. ニューロ的ファジィ(neuro-like fuzzy)このタイプは,ベテランから制御ノウハウを聞き出すことができた場合に,それをIf-then形式のファジィルールで記述し,ファジィルールのパラメータをデルタルール学習則により調整するものである.入力変数選択,ルールの追加・削除と組合わせることで,ベテランから完全なルールを獲得できない場合にも有効な制御器設計を可能とする[図5・38(b)].. ファジィ的ニューロ(fuzzy-like neuro)階層型ニューラルネットワークの結合を工夫して,ファジィ推論ルールとの対応関係をもたせることができれば,ニューラルネットワークの学習結果から入出力関係に関する知識をファジィルールとして抽出できる.ニューロ的ファジィと同様に,入力変数選択,入力空間分割とネットワークの学習を組合わせてモデリングを行うことで,制御知識獲得,非線形制御対象のモデリングなどを行うことができる[図5・38(c)].. ニューロ化ファジィ(neural fuzzy)複数のニューラルネットワークによるファジィ推論の実現方法には,大きく分けて次の二つの方法がある.(1)前件部と後件部を個別にニューラルネットワークで同定し,両者の出力を合成してファジィ推論を実現する方法と,(2)前件部と後件部を個別にニューラルネットワークにより同定し,両者の論理的な関係をもう一つのニューラルネットワークにより学習・記憶させる方法である[図5・38(d)].c. ファジィモデリング(fuzzy modeling)ニューラルネットワークによるファジィ制御器設計の統一的な方法論はまだ確立されていない.上述のi~iv項のいずれにおいても,基礎となる技術は入力変数選択,入出力空間分割,およびメンバシップ関数調整を行うファジィモデリングである.非線形対象のモデルを同定するため,探索アルゴリズムと最適化アルゴリズムが組合わせて適用される.進化アルゴリズムとニューラルネットワークの組合せはよく用いられている.ファジィモデリングにおける重要な指標にモデルの精度と明示性がある.明示性の基準は(i)少ないルール数,(ii)少ない入力変数,(iii)少ないメンバシップ関数の種類数である.モデルの精度を極力保ちながら明示性を高めるファジィモデリングが有効である.
