# 5・4・3 遺伝的アルゴリズムによる間接的最適化による制御

> **章**: 第5章 知的制御
> **節**: 5・4 遺伝的アルゴリズムによる制御

GA の特徴は、最適化の過程で最小二乗法などのように、最適化したい関数のこう配情報を必要としないことである。この特徴を生かして、個体のコーディングと最適化したい問題とを直接的に対応させずに最適化することができ、これを間接的最適化と呼ぶ。GAは、最適化に計算時間がかかるため、オンラインのパラメータチューニングには向いていない。また数値でのコーディングは探索空間が膨大になるため、大まかにしか最適化できない場合が多い。そこで、オンラインの適応的な制御では、コントローラ自身に適応側を与えることが必要である。このような場合に、ニューラルネットワークやファジィを用いて数値表現される問題を取扱うようにする。コントローラの内部では、ニューラルネットワークやファジィによって適応的に制御を行う。このとき、GA は、制御結果を適応度にして、コントローラに使用されているニューラルネットワークやファジィの構造とパラメータを最適化する。このように、最適化したい評価関数と直接的に関連する必要がないため、直接的な教師信号がなくてもニューラルネットワークやファジィルールを学習させることができる。この特徴を生かして、GA を用いて強化学習をさせることが可能である。6―154制御システム文献(1) Fu,K.S.,IEEE Trans.,AC 16-2(1971)、70.(2) 竹垣盛一・石岡卓也、知的制御システム、(1990)、海文堂。(3) Antsaklis,P.J.and Passino,K.M.,An Introduction to Intelli-gent and Autonomous Control, (1992)、 Kluwer AcademicPublishers.(4) 小林重信、計測と制御、24-2(1985)、155.(5) 小林重信、計測と制御、24-3(1985)、242.(6) 北村新三・片井修編著、画像と制御、(1991)、1-22、朝倉書店。(7) 安信誠二、ファジイ工学、(1991)、昭晃堂。(8) 西川一・北村新三、 ニューラルネットと計測制御、 (1995)、 94-116、朝倉書店。(9) Funahashi,K.,Neural Networks,2-3(1989)、183.(10) 計測自動制御学会編、 ニューロ・ファジイ・AIハンドブック、(1994)、619、オーム社。(11) 林勲・馬野元秀、日本ファジィ学会誌、5-2(1993)、179.(12) 三宮信夫・ほか3名、 遺伝アルゴリズムと最適化、 (1998)、 13-19、朝倉書店。(13) 高木英行、日本ファジィ学会誌、10-4(1998)、602.(14) 石渕久生、日本ファジィ学会誌、10-4(1998)、613.(15) Zadeh,L.A.,Inf.
