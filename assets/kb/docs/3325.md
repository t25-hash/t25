# 6・1・1 概要

> **章**: 第6章 自律分散・創発システム
> **節**: 6・1 自律分散システム

20世紀は生物を模倣した機械がはんらんした時代であった.特に1980年代以降は,ファジィ,ニューロ,遺伝子アルゴリズム,免疫系と生物的機械がたくさん提案され,産業応用も盛んに行われた.しかしながら表面的な模倣であり,生命そのものの模倣とは言い難いものであった.21世紀は生命の時代である.機械の分野でいえば,機械生命(mechanical life)である.機械による生命の模倣は,新たな生命を生み出すという連鎖も実現する.そのような質的な転換が起こり始めている.この変換の始まりが自律分散システムであり,機械に生命を与えるという研究開発の始まりである.以下,分散,自律というキーワードから始めて,機械が生命をもつ意味と意義を考えていく.
