# 6・2・7 工具研削（tool grinding）

> **章**: 第6章 砥粒加工
> **節**: 6・2 各種研削加工法

総形加工が可能である．
最近のcNC横軸角テーブル型平面研削盤の場合，研削砥石のド
レッシング，研削手順，定寸などのサイクルが自動化されたものが
多い．また，高速反転が可能な研削盤も開発されており，その性能
を生かすために研削砥石の回転数も高くなる傾向がある．この場
合，高速回転に適した，超砥粒ホイールを用い，研削砥石のバラン
スを高精度に調整するとともに，研削点を十分冷却することが肝要
である．さらに最近では，非軸対象非球面ミラーなど加工面形状の
複雑化，高精度化に伴い，超精密CNC三次元研削盤を用いた自由
曲面加工，加工面形状の機上計測とこれに基づく補正加工などが試
みられている．
ように，総形成形した砥石により研削する方式と，皿形砥石（dish
wheel ）によって成形研削する方式がある．前者には，図6･27(ａ)
のように成形した砥石によって歯溝を一つずつ研削する方法と，図
6･27(ｂ)のようにウォーム状に成形した砥石と工作物を高精度に同
期回転させながら連続研削するホブ盤に類似した方式がある．いず
れの場合も研削砥石の成形に困難を伴うが，前者は機械構成が簡単
で，非インボリュート歯形をもつ歯車の研削に使用されることが多
い．また，内歯車の研削にも使用される．後者は，非常に高能率
で，特に小型歯車では素材から直接仕上げることが可能である．図
6･27(ｃ)に示す方法は，2枚の皿形砥石の研削面を仮想のラック面
として，これに工作物を転動させながら接触させることで，歯車を
創成するもので，高精度な歯形の創成が可能である．この方法で，
はすば歯車を創成することも可能であるが，生産性において図6･
27(ｂ)の方式に劣る．
素材の高品位加工が必要となる場合がある．このようなときには，
あらかじめ高精度に成形した電着cBNホイールを用いるのが有利
図6･28万能工具研削盤
である．これは，砥石の摩耗が少ないことと，cBNの熱伝導率が
一般砥粒砥石よりも大きいことから工作物への熱流入が少なく，か
つ研削面に圧縮残留応力が生じやすいことによる．
ねじは切削によって創成されるものがほとんどであるが，高精度
の送りねじやねじ状工具などは研削によって仕上げられる．ねじ研
削（thread grinding）には，一山ねじ研削砥石または多山ねじ研
削砥石が使用される．前者では，工作物のねじ山形状に一致させた
砥石をねじのリード角と同じ角度傾け，回転するねじの進みに応じ
ねじの精度には，研削盤の運動精度，砥石の成形精度と加工中に
おける摩耗などが影響する．このため，cBNホイールの使用が効
果的である．リード角が5°程度より大きいねじの場合は，砥石の
干渉によりねじ山の先端と基部にアンダカットを起こすので，砥石
の輪郭を修正する必要がある．一方，多山ねじ研削では，ダイヤモ
ンドロールなどで成形した多山形状の砥石を用い，高能率にねじ山
を創成することが可能である．しかし，成形された砥石のねじ山形
状の誤差が，そのまま工作物に転写されるため，ロールには高い形
バイト，ドリル，エンドミル，ホブなどの工具にはいずれも高硬
度材料が用いられることから，その創成は研削に頼らざるを得な
い．さらに，工具の形状は複雑かつ多様であることから，使用され
る工具研削盤も多様で，専用機も多い．例えば，複雑な形状をした
エンドミルなどの研削加工のために，図6･28に示すような多軸の
万能工具研削盤（universal tool grinder ）が使用されている．
CNC装置を有する工具研削盤では，工具形状の創成運動をシミュ
レーションすることが可能で，創成後の工具断面形状を簡単にチェ
ックできる．しかし，形状の創成運動と砥石形状の組合せの最適化
については，今後のシミュレーション技術の進展を待たなければな
らない．
