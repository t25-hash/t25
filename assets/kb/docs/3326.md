# 6・1・2 はじめに

> **章**: 第6章 自律分散・創発システム
> **節**: 6・1 自律分散システム

機械の起源を明確化することは難しい.しかしながら,機械が自前の動力をもったとき,道具から生命に一歩近づいたことは間違いない.18世紀から19世紀にかけて,蒸気機関や電池が発明された.その意味で,この時代は機械の独り立ちの始まりと位置付けられる.そして,17世紀のパスカルに始まる機械式の自動計算機は,20世紀には可塑性が高い電子式の計算機,つまり電子計算機となった.そして,電子計算機を取込んだ機械は徐々に知能を持ち始めた.その知能を後押ししたものがシリコンを象徴とする半導体技術であり,マイコンという言葉に代表されている.そして,21世紀はネットワークの時代である.知能と動力を内包するという形で自律し始めた機械が,ネットワークを通じて連係する時代である.これを自律分散システムの時代と呼ぶことにする.このシステムの概念を解説するために,機械における分散,自律,新陳代謝を考えていきたい.
