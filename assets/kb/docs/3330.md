# 6・3・3 バフ加工

> **章**: 第6章 砥粒加工
> **節**: 6・3 遊離砥粒加工

バフ加工（buﬃng）は，綿布，フェルト，皮革などの柔軟性材
料で構成された回転体（バフ車，buﬃng wheel ）の外周面に研磨
材と油脂またはにかわなどを混合したバフ研磨剤を付着させ，それ
を高速回転させながら工作物を押し付けて装飾的な光沢仕上げを行
う加工法である ．バフ加工は，砥粒による切削作用，高速回転
に伴う加工温度の上昇による工作物表面層の塑性流動などの機械的
作用および油脂類による工作物との化学的除去作用が相乗して行わ
れる．バフ車の構成材料としては，研磨工程に応じて表6･7に示す
ような材質が使用される．バフ研磨剤には，油脂性，非油脂性およ
びエマルジョン系液状のものがあり，エアスプレー法，プランジャ
ポンプや，スプレーガンによる増圧法などの供給方法を用いてバフ
車の外周面に塗布される．
