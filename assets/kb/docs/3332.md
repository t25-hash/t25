# 6・2・1 創発の定義

> **章**: 第6章 自律分散・創発システム
> **節**: 6・2 創発システム

相互に関連する要素の集合では,多様な大域的秩序が形成される.この現象を自己組織化(self-organization)と呼び,自己組織化を形成するシステムを複雑系と呼ぶ.複雑系に形成される大域的秩序は,個々の要素の部分過程とシステムの全体過程の絡み合いの中から形成される.すなわち,要素の局所的な運動が大域的な秩序を形成する部分過程と,形成された大域的な秩序が個々の要素の運動を拘束する全体過程の双方向の動的過程を通して,多様で複雑な大域的秩序が形成される.特に,大域的秩序の拘束のもとで,構成要素がその運動とともに相互作用の変化を通してその性質をも変化させていくとき,系全体には大域的秩序が形成されるとともに崩壊し,また新たに形成されていく持続的な運動(進化)が進行する.その進行に伴ってシステムの内部には潜在的な構造が形成されていく.この部分と全体の同時形成過程を創発(emergence)と呼ぶ.複雑系を構成する要素は,自発的な運動をする要素であると同時に,その自発的運動に制限を加える存在であるシステムの構成要素でもある.このような構成要素に対してケストラーは,ホロン,すなわち部分との関係からすれば全体であり,全体との関係からすれば部分である存在と述べた.また,ヴィゴツキーは単位,すなわち要素とは異なり全体に固有な基本的特性のすべてを備え,それ以上にはこの統一体の生きた部分として分解できないものと定義した.物理系には,流体系における熱対流(ベナール対流),レーザにおける発振など自己組織化が広く存在する.ハーケンは物理系における自己組織化の研究を基に,シナジェティクス(synergetics)と呼ばれるシステム理論を提案した.シナジェティクスでは,大域的な秩序を支配者と呼び,大域的秩序が個々の要素の運動を拘束する全体過程を隷属原理(slaving principle)と呼ぶ.すなわち,支配者は個々の要素を支配する一方で,支配者自身が個々の要素から構成されるのだから,支配者とは自分自身に働きかけて操縦しようとする操り人形のようなものである.このような隷属関係(隷属原理)が,シナジェティクスの中心的役割を果たす.ポランニーは大域的秩序が個々の要素の運動を拘束する全体過程を,境界制御の原理(principle of marginal control)と呼んだ.ポランニーはある目的(作動原理)のもとで物理的,あるいは化学的属性をもつ諸部品から構成されたシステムを機械と呼び,機械を例に境界制御の原理を次のように述べている.すなわち,機械の構成要素である諸部品の動作はその属性である物理学と化学の法則に従うが,同時に物理学と化学の及ばない境界条件も存在し,機械の作動原理がその境界条件を制御する.自己組織化,特に大域的秩序の拘束のもとで,構成要素がその運動とともに相互作用の変化を通してその性質をも変化させていく創発は,形態形成,中枢系における認知過程あるいは生態系など生物系に広く見られる.フォン・ベルタランフィーは生物の本質は組織化された複雑さ(有機体)であるととらえ,有機体論的生物学そして,その延長としての一般システム論を提案した.また,ラングトンは,生命現象一般に関する共通の原理を人工的に再現することによって明らかにする理論として人工生命(artificial life)を提唱した.これらの理論においては創発が中心的な原理として採用されている.複雑系における大域的秩序形成は,工学においても重要な役割をもっている.動物は多数の筋骨格系の運動を巧みに制御して,さまざまな環境に自在に適応している.このような巧みな運動制御機能(スキル)を実現するためには,多数の構成要素をもつ非線形なシステムの運動制御を実現しなければならない.このような制御を実現するため,自律分散制御(autonomous distributed control)と呼ばれる制御理論が提案されている.自律分散制御は多数の要素を個々に制御するのではなく,上位レベルによる境界制御の原理をその制御原理として採用している.一方複雑系,特に部分と全体の同時形成の進む系では,系全体に潜在的な構造が形成され,状況に対して多義的な応答が可能になる.プリゴジンはこのようないくつかの異なる仕方で環境に適応するシステムでは,選択という過程を通して歴史性を獲得するとともに,情報が生成されていくと述べた.記憶・学習機能を実現する人工神経回路網モデル,探索・最適化機能を実現する進化型計算法などは,複雑系における大域的秩序形成を用いて高度な情報処理機能を実現している.最後に,複雑系に対してはその物質的基盤を明らかにする要素論的考察とともに,大域的秩序形成の機構を明らかにする過程論的考察が必要であるが,後者に対して要素論的考察に基づいて適切な単位を採用し,それを基にモデルを構成し考察を行う構成論的研究方法が提案されている.
