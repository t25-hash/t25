# 6・3・2 非線形ダイナミクス(およびCPG)

> **章**: 第6章 自律分散・創発システム
> **節**: 6・3 新しい設計法

a. 分散系での相互作用の局所性大規模複雑化に伴う集中管理型の限界から提案された自律分散的なシステム構造は、処理・通信の高速化、負荷の分散、拡縮性、耐故障性等に優れ、環境変化に対する適応性・柔軟性に富むと期待されている。その重要な側面の一つは、分散化してできた部分系間において、情報交換すなわち相互作用が局所的に行われることである。ここでは、相互作用の局所性という観点から分散システムの一設計法について述べる。b. 設計のアプローチ流体でのベナール対流、B-Z 反応の媒質模様、神経細胞の発火リズムのように、物理、化学および生物の多様な現象はいくつかのパターンをもって観察される。多様なパターンは、不動点近傍の解析で系全体の大域的なふるまいが決定する線形ダイナミクスでは表現しきれず、非線形なダイナミクスで記述することが必要となる。ダイナミクスに従い状態が遷移していくとき、多くの状態が最終的にどこに到達するのかを表すのがアトラクタ(attractor)である。非線形ダイナミクスによるシステム記述では、定常状態として現れるふるまいをアトラクタに対応づける。アトラクタが相空間に複数存在する場合もあり、それらを機能と結び付くように設計すれば、システムのふるまいの多様性が実現できることになる。また、このアトラクタが環境変動に対してロバストであるには、アトラクタが現れる相空間のベクトル場が構造安定である必要がある。構造安定な場合、近傍の同相性より摂動を受けても元のベクトル場とその性質は本質的には変化せず、したがって、摂動を受けない場合と同様のふるまいが期待できる。一方、多様なふるまいを記述させるには、分岐現象(bifurcation)が利用できる。分岐が起きる場合、あるパラメータの変化に伴い、ベクトル場の様相が大きく変わり、不動点の数やその安定性が変化する。環境変化に基づく自己評価などに従い、ある内部パラメータを変化させて分岐現象を起こさせると、いままでの定常状態が不安定化し、新しいアトラクタへと遷移を起こすことが可能となる。6―162制御システム分散系では部分系の数だけ状態量が増え、系全体のふるまいを示すダイナミクスは必然的に高次元な微分方程式となる。解析が困難となる高次元で非線形なダイナミクスを扱い、いかに環境に応じたふるまいを示す系を設計するか、その1ストーリーを以下に述べる。. 秩序変数の導入多数の状態変数間に関係や拘束があると、系のダイナミクスは低次元化する。関係や拘束が機能と結び付けば低次元化したダイナミクスで機能の形成を論じることができる。その関係や拘束を秩序変数(order variable)として定義する。低次元化はダイナミクスの時間スケールの違いを利用するものが多く、シナジェティクスの断熱近似や中心多様体定理はその一つといえる。. 環境条件の組込み環境に適した機能構造を生み出すには、系全体のダイナミクスに環境の影響を組入れる必要がある。つまり環境に対して系が開いていなければならない。そのため、環境条件はしばしば時空間的挙動を表現するダイナミクスの境界条件として扱われる。. 環境との相互作用による拘束条件の生成環境によって境界条件が与えられたダイナミクスは、秩序変数で表現されるある定常状態に到達する。これにより環境に適した系全体としての機能が決定する。いったんそれが決定すると、今度はその機能が逆に拘束条件となって部分系のふるまいを規定し、個々の部分系間の協調を採る。部分系から全体への機能形成と機能による部分系の拘束からなるループ構造がシステムの創発性に必要となる。c. こう配系による一設計法。 問題設定多くの部分系が局所的に相互作用を及ぼし、結果的に大域的な秩序が生まれる系として結合振動子系がある。結合振動子系は、化学反応現象や神経細胞の認知モデルから交通システムモデルまで、さまざまな分野のパターン形成の共通的なモデルを与える。各振動子は相空間でリミットサイクルを形成して独自の周期で振動するが、うまく結合すると振動子が引込みにより同期し、振動子の状態間に位相差が生まれる。大域的な機能は位相間の差によって表され、そのふるまいは状態差を秩序変数とする位相ダイナミクスで記述できる。位相差が機能と結び付く運動系の例として多脚動物の歩行パターンがある。歩行におけるリズムパターンはCPG(central pattern generator)と呼ばれるせき髄内の神経振動子群により自己組織的に形成され、そのリズムと結び付いた脚運動の位相差が状況に応じた多様な歩行パターンとして現れる。ここでは結合振動子系を例にとり、状態差すなわち位相差を局所相互作用のみで制御するようなダイナミクスの設計法(目標値の安定性)、および目標位相差が環境に合わない場合のその修正法(適応性)を述べる。. こう配系とポテンシャル関数非線形ダイナミクスを利用して設計を行う場合、見通しの立てやすい方法として、ダイナミクスをこう配系(gradient system)として設計する方法がある。こう配系ではその時間発展がポテンシャル関数の最急こう配の方向として与えられ、定常点つまりアトラクタがポテンシャル関数の極小点として設計できる。そこで状態差のダイナミクスがこう配系として構成できるような各振動子間の相互作用の構成法を示す。は部分系(振動子)間の結合の数である。ここで重要なのが、は部分系とは相互作用しない部分系の状態変数は含んでいないことである。つまり、相互作用を行う部分系の状態変数しか含まず、相互作用の局所性が満たされる。. 適応と相互作用部分系間の相互作用は、二つの部分系間の秩序変数がその目標値に到達させるように働く。もし目標値が達成された場合、相互作用は小さくなるのが自然である。しかし、現実的にはそうでない場合が多く、このとき環境との相互作用を通じてよいものを選択する必要がある。この選択のプロセスが拘束条件の生成に対応する。目標状態差が達成された場合、結合部分系間の相互作用は零となる。しかし、環境変化などで目標位相差の設定に矛盾が生じるような場合、目標状態差が達成されず、二つの部分系間に相互作用が残ってしまう。その原因は目標位相差が環境に合っていないためと考える。そこで、目標位相差を部分系間の相互作用が小さくなるように修正を行う。環境に合わせた目標状態差の修正プロセスを適応と考える。部分系のダイナミクスより遅くなるよう、τは十分小さくとる必要がある。このダイナミクスでは、こう配系のポテンシャル関数の極小点が変化するだけで、その形状変化は起きない。したがって、アトラクタ数が変化するような分岐現象は現れない。さらなる展開はこの分岐現象を組込むことによって期待できる。文献(1)
