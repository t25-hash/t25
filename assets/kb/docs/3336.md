# 6・2・5 油圧ダンパ

> **章**: 第6章 緩衝・制振要素
> **節**: 6・2 緩衝器およびダンパ

a.構造 流体動圧ダンパ，オイルダンパ，ショックアブソー
バとも呼ばれており，その原理は図Ⅰ-6･9のダシュポットと同じ
であるが，抵抗力を運動速度の関数として定めるため，オリフィス
の部分には表Ⅰ-6･4に示されるような調圧弁を設けている．オリ
フィスの面積はばねで押さえられた調圧弁のリフトによって定ま
り，そのリフトはシリンダ内圧に左右されるから，式(Ⅰ-6･12)に
より弁のリフトの釣合い位置がピストン速度によって定まり，抵抗
力は速度の関数として決定される．抵抗力が一方向のみに働く片効
きのものと往復方向に働く両効きのものとがある．
b.特徴 吸収すべきエネルギー量に対して小形に設計でき，
しかも抵抗力が広い振動数にわたって安定しているため，自動車，
鉄道車両の懸架装置をはじめ，広い分野に使用されており，その種
類と特徴を挙げると表Ⅰ-6･5のようになる．図Ⅰ-6･14，Ⅰ-6･15
およびⅠ-6･16には軸揺動形と直動形の例を示す．なお通常の標準
設計品では，振幅が5mm以下では抵抗力が低下し，温度が20℃
以下では増加する．
