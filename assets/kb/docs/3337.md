# 6・2・6 粘性ダンパ

> **章**: 第6章 緩衝・制振要素
> **節**: 6・2 緩衝器およびダンパ

a.構造 油圧ダンパが乱流による圧力降下を利用するのに対示されるような構造をもつ．
使用油としては，粘性が高くて，温度る．
空気を使うものは空気ダンパともいう．
衰係数と呼ばれる．
その計算式は表Ⅰ-6･6に示される．
運転方向の転換が容易である（揺動形共通）．
コンパクトな設計ができる．
内部油もれを生じやすい．
油もれが少なく耐久性が良好である．
エアレーション，キャビテーションを生じない．
軽量で比較的生産性がよい．
油もれに対する耐久性がある．
鋳鍛部品を利用するので重量が大きくなる．
減衰力の調整が困難である．
構造が簡単で軽量化でき生産性がよい．
自動車のサスペンションに適用したとき接地性がよい．
ガスもれ防止に技術を要する．
単筒形に比しトラブルの要因が少ない．
取付姿勢に方向性がある．
井上隆美・樋口順一,ばね論文集,25で，計測器によく使われ，乗り物にも特殊用途として使われる．
