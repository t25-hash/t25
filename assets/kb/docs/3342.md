# 7・2・1 項では、スーパーバイザがすべ

> **章**: 第7章 離散事象システム
> **節**: 7・2 スーパーバイザ制御

ての事象の生起を観測できる完全観測を仮定した。しかし、センサの制限などにより、一般にシステム外部からその生起が観測できない事象が存在する。このような場合を部分観測(partial observa-tion)と呼ぶ。部分観測のもとでは、事象の集合∑はその生起が観測できる事象の集合と、観測できない事象の集合とに分割される。前者を可観測事象(observable event)の集合、後者を非可観測事象(unobservable event)の集合といい、可観測事象の集合を∑とおく。すなわち、と′の生起後のσの制御動作が定まらなくなる。したがって、可観測性(observability)は各可制御事象の制御動作が部分観測のもとでも必ず定まるための条件である。[定理2]はブロッキングがない離散事象システムとする。制御仕様は空でない正規言語であり、は有限オートマトンであるとする。c. 可観測性とノーマル性制御仕様が可制御かつ可観測とはならない言語によって与えられたときは、の可制御可観測部分言語を求め、それに対してスーパーバイザを構成することになる。しかし、最大可制御部分言語とは異なり、一般に最大可制御可観測部分言語は存在しない。これは、可観測性が集合の和において保存されないためである。そこで、言語のノーマル性(normality)という概念を導入する。しかし、一般に命題2の逆は成立しない。一般に、最大可制御部分言語の計算において、ノーマル性は保存されない。また、最大ノーマル部分言語の計算においても、可制御性は保存されない。したがって、を計算するためには、不動点に収束するまで、最大可制御部分言語と最大ノーマル部分言語を繰返し計算する必要がある。一般に、可観測な言語がノーマルであるとは限らないが、可制御な事象がすべて可観測ならば次の性質が成り立つ。
