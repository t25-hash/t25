# 6・1・5 新陳代謝する機械

> **章**: 第6章 自律分散・創発システム
> **節**: 6・1 自律分散システム

機器が切り離されてもシステムの基本性能に影響を与えないシス
テムが自律分散システムであれば，これは恒常性を維持した新陳代
謝機能をもつことと等価である．故障した機器を取り外し，新しい
機器を取付ける．この一連の操作を無停止で実行できるものが自律
分散システムである．そのためには，活線脱着，プラグアンドプレ
イ（plug and play）などの機能が必要となる．
このようなシステムへの新しい機器の参加を自動化するために
は，加わった機器の機能情報をシステム内のすべての機器に通知す
る必要がある．通知の方法としては，掲示板システムと放送システ
ムの二つがある．前者は，参加した機器情報を特定サーバに用意さ
れた掲示板に書き込むものである．そして，ほかの機器は掲示板を
参照して新たに接続された機器の機能を利用するものである．もう
一つは，新たに接続された機器が放送により，ネットワーク上のほ
かの機器に機器情報を通知するものである．
掲示板方式は，サーバが故障すると利用できないし，サーバを先
に特定しなければならないなどの分散システムとしての問題点を抱
えている．一方，放送方式では放送された機器情報の管理は個別の
機器で行うため，各機器に高度の能力が必要となる．もちろん，掲
示板を複数化することや，掲示板方式と放送方式を組合わせるなど
の両方式の折衷方式もありうる．実際は，分散機器の能力向上に応
じて，掲示板方式から折衷方式，放送方式と結合を疎にする方向に
自律分散システムも発達していくと考えるべきである．
掲示板上の情報や放送された情報はシステム内の機器が共有した
情報であり，自律分散システム論においては，図6･1に示す場とい
う概念に対応する．システム内の機器が，場に提供された情報を基
に，自らの行動を選択することが自律性であり，自律性の確保に情
報共有の仕組み，すなわち場が不可欠である．以上のことから，す
べての情報を集中化し，分散機器に指令を出す集中方式とすべての
情報を場という形で共有し，動作決定を機器レベルで行う自律分散
システムの違いを認識することができる．
