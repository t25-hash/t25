# 7・3・1 シーケンス制御系の構成

> **章**: 第7章 離散事象システム
> **節**: 7・3 シーケンス制御

a. シーケンス制御(sequential control)対象となる制御系の処理方式に着目した場合、フィードバック制御と対比して、シーケンス制御は、あらかじめ定められた順序あるいは特定の条件に従って、制御を逐次的に行うものであると定義できる。つまり、シーケンス制御は、その制御内容から、あらかじめ定められた順序による順序制御と、あらかじめ定められた特定の条件による非順序制御に分けることができる。一般に、順序制御は制御の対象の動きと緊密に関連しているものであり、その制御の仕方は明確で、制御対象依存である。これに対して、非順序制御は安全保証、異常処理あるいは順序制御の動作を保証するために設けられたインタロックなどであり、ノウハウによる部分が多く、経験依存である。b. control circuit)シーケンス制御を実現するものとして制御回路が用いられる。制御回路を実現する方法としては、おもにリレーシーケンスとPLC(pro-grammable logic controller)がある。PLCは一般にシーケンサともいう。制御の順序を実現するためにはなんらかの仕掛けが必要である。電磁リレーが発明されるまでは、織機のようにおもに機械的仕掛けによる手法が中心であった。電磁リレーが出現してからは、電気信号の伝達が瞬時で実現できたこと、電磁気の利用で小さい磁場で大きな電流を流すことが可能となったことなどにより、制御はおもに電気によるものに一変した。さらにコンピュータの出現によって、より複雑な制御論理を実現するためにコンピュータが活用されるようになり、制御用に作られたのがPLCである。PLCはコンピュータのハードウェア技術だけでなく、ソフトウェア技術も応用できる。プログラムを可変にすることによって、制御回路の変更は設計から運転まであらゆる段階で可能になり、生産の内容に合わせてその制御回路を柔軟に作り直すこともできるようになった。d. 基本制御回路(basic control circuit)基本制御回路は、論理回路、記憶、計数、限時に分けられる。論理回路(logiccircuit)としては、NOT 回路(NOT circuit)、AND 回路(AND circuit)、OR 回路(OR circuit)などがある。論理回路を使用する場合は、制御系の入出力を論理回路の入出力に対応させることができる。出力をもう一度入力として使用する順序回路を構成するためには、出力結果をいったん保持する必要がある。結果の保持に使われるのが記憶(memory)であり、電気回路ではフリップフロップによって実現される。制御には電気的にON-OFFのような二値信号だけで不十分な場合が多く存在する。例えば、製品の個数を数えたり、機械の起動回数を集計したりすることがある。計数用の基本回路としてカウンタ(counter)があり、カウンタには、0から1ずつ数え上げていくアップカウンタ(up-counter)、規定値から0まで1ずつ減らしていくダウンカウンタ(down-counter)、0と規定値の間で1ずつ加えたり減らしたりすることのできるアップダウンカウンタ(up-down
