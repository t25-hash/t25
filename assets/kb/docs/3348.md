# 7・4・1 マルコフ連鎖

> **章**: 第7章 離散事象システム
> **節**: 7・4 パフォーマンス解析

マルコフ連鎖(Markov chain)は、マルコフ性(Markov prop-erty)と呼ばれる性質をもつ離散状態をとる確率過程(stochasticprocess)であり、離散時間マルコフ連鎖と連続時間マルコフ連鎖に大別される。以下ではマルコフ連鎖がとりうる状態の集合をとする。マルコフ性は連続時間確率過程に対しても同様に定義できる。b. 離散時間マルコフ連鎖(discrete-time Markov chain)。このような性質をもつ行列を確率行列(stochastic matrix)という。さらに状態とが互いに到達可能であるとき、状態とは連結(communicate)しているという。特に、状態はそれ自身と連結である。マルコフ連鎖の全状態の集合の中から一つの状態を選び、それと連結しているすべての状態を集めて集合の部分集合を作り(これを連結クラスという)、さらにに含まれない任意の状態を一つ選び、それと連結しているすべての状態を集めて連結クラスを作る、という手続きを繰返すと、状態集合は=∪∪...のように互いに素な複数個の連結クラスの和集合で書くことができる。各連結クラス内の状態はすべて正再帰的か、すべて零再帰的か、あるいはすべて過渡的かのいずれかである。同じ再帰的な連結クラスに属するすべての状態は同じ周期をもつ。=1のとき、連結クラスは非周期的(aperiodic)であると呼ばれ、≧2のとき周期的(periodic)であると呼ばれる。すべての状態∈が一つの連結クラスに含まれるような離散時間マルコフ連鎖は既約(irreducible)と呼ばれる。もし、定常状態確率が存在するならば、それは唯一であり、特に、状態数が有限である既約な離散時間マルコフ連鎖には必ず定常状態確率が存在する。c. 連続時間マルコフ連鎖(continuous-time Markov chain)。以下では、各状態にとどまる時間が確率1で正であり、かつ、遷移確率がに依存しないと仮定する。よっての各行の和は零である。. 状態の分類と定常状態確率連続時間マルコフ連鎖に対しても、離散マルコフ連鎖と同様に、到達可能性や再帰性を考えることができる。特に、状態はそれ自身と連結である。離散時間マルコフ連鎖と同様に、連続時間マルコフ連鎖においても、全状態の集合を=∪∪...のように互6―179いに素な複数個の連結クラスの和集合で書くことができる。各連結クラス内の状態はすべて正再帰的か、すべて零再帰的か、あるいはすべて過渡的かのいずれかである。すべての状態∈が一つの連結クラスに含まれるような連続時間マルコフ連鎖は既約と呼ばれる。もし定常状態確率が存在するならば、それは唯一であり、特に、状態数が有限である既約な連続時間マルコフ連鎖には必ず定常状態確率が存在する。
