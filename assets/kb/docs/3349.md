# 7・4・2 待ち行列理論

> **章**: 第7章 離散事象システム
> **節**: 7・4 パフォーマンス解析

待ち行列理論(queueing theory)とは,有限の資源が多くの利用者によって共有されている状況を数学的に記述した確率モデル(待ち行列モデル)に関する理論である.多くの利用者が有限の資源を共有している状況では,利用したい資源が他の利用者によって占有されている場合,他の利用者は資源が解放されるまで待つか,あるいは,その資源の利用をあきらめなければならない.このような待つという状況,あるいは競合により資源利用を拒否されるという状況を抽象的に表すため,待ち行列理論では,通常,資源利用要求を客,共有資源をサーバ,客が共有資源を占有する時間をサービス時間と呼ぶ.待ち行列理論において興味の対象となるおもな指標には,システム内客数分布や客の待ち時間分布などがある.a. 待ち行列モデルの記法待ち行列モデルは,共有の資源への利用要求発生に関する統計的情報(到着過程),個々の要求が資源を占有する時間に関する統計的情報(サービス時間過程),共有資源の数(サーバ数),システム内にとどまることが許される客の最大数(システム容量)などによって記述される.このような待ち行列モデルを記述する方法としてケンドールの記法(Kendall’snotation)がある.これは,一般に///の形をしており,それぞれ,到着時間間隔分布/サービス時間分布/サーバ数/システム容量を示している.,に関しては(指数分布),(一定分布),(一般分布:特定の分布を仮定しない),(独立同一分布:特定の分布は仮定しないが,到着時間間隔あるいはサービス時間は互いに独立)などが用いられる.システム容量が無限大の場合は単に//と書かれる.サービスの順序に関しては通常,先着順サービスが仮定されている.b. リトルの公式定常な待ち行列モデルにおける平均システム内客数と平均システム内滞在時間の間には,リトルの公式(Little’s law)と呼ばれる非常に単純な関係が成立する.λを単位時間当たりの平均到着客数(これを平均到着率という)とする.すなわち,0,を時間間隔0,における到着客数としたとき0,λ=limで与えられる.一方,を時刻≧0におけるシステム内客数,욈を番めに到着した客の待ち時間とすると,それらの平均およびは,それぞれd=lim욈∑욗=limで与えられる.もしとのいずれかが有限であれば,非常に緩い仮定のもとで,リトルの公式=λが成立する.6―180制御システムc. 待ち行列モデルとマルコフ連鎖任意の待ち行列モデルにおいて,を時刻≧0におけるシステム内客数,0,を時間間隔0,における到着客数,0,を時間間隔0,におけるシステムからの離脱客数とすると=0+0,-0,, ≧0となる.客の到着時間間隔が独立同一なパラメータλの指数分布に従い(これを率λのポアソン到着という),サービス時間が独立同一なパラメータμの指数分布に従うとき,個のサーバをもつ待ち行列モデル//のシステム内客数は遷移率=0,1,..., =+1λ,=1,...,-1, =-1μ,욡욢==,+1,..., =-1μ,その他0,をもつ連続時間マルコフ連鎖となる.ρ=λ/μとしたとき,ρ<ならば定常状態確率π욡が存在し,욂욪ρ요!+ρ욑-1!-ρπ=∑요ρ!π,=1,...,-1π욡=ρ!욑π,=,+1,...で与えられる.また,定常な先着順サービス//における客の待ち時間の分布関数はPr≦=1-ρπ욂욪-ρ+ρπ욂욪-ρ1-욑で与えられる.率λのポアソン到着によって客が到着し,サービス時間が平均μ,密度関数をもつ分布に従う単一サーバ待ち行列//1におけるシステム内客数はマルコフ連鎖ではない.しかし,客の離脱直後の客数욈は遷移確率=0, =0,1,...욢,=1,2,..., =-1,,...욡욢=욢용욪욡,その他0,をもつ離散時間マルコフ連鎖となる.ただし,욅は一つのサービス中に人の客が到着する確率であり,욅=λ요!
