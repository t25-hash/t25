# 7・4・3 摂動解析法

> **章**: 第7章 離散事象システム
> **節**: 7・4 パフォーマンス解析

待ち行列理論などによりシステムの性能を解析的・数値的に評価できるモデルのクラスは,残念ながらあまり広いとはいえない.そのため,システムの性能評価のための一般的な手段として,実際に稼働しているシステムを観測したり,あるいはシミュレーションによって,平均待ち時間や待ち行列長などの性能評価尺度を推定するという方法に頼らざるを得なくなる.ところで,現実のシステムには設計や運用に関するさまざまなパラメータがあり,性能評価の際には,与えられたパラメータの値に対する性能評価尺度の値だけでなく,パラメータの値が変わると性能評価尺度がどのように変わるのかといった問題も重要になる.こうしたパラメータ感度を推定する問題に対しては,従来,実際にパラメータの値を変えて性能評価尺度を推定し,その推定値を比較するという方法が一般的であった.この方法は,確かにほとんどすべてのモデルに対して適用可能であるが,得られる感度推定量の分散が大きくなることや,パラメータの値を変更するたびに推定を繰返さなければならないなどの欠点がある.摂動解析法(PA : perturbation analysis)は,パラメータが連続量である場合に,実際に稼働中のシステムを観測しながら,あるいはシミュレーションによって,(パラメータの値を変更することなく)性能評価尺度のこう配を効率良く推定する方法として知られている.以下,a,b項では,システムのふるまいを支配する確率分布がパラメータに依存しない場合に対する摂動解析法を概説し,c項では逆に,確率分布のみがパラメータに依存する場合について述べる.これらの手法は異なる対象を問題にしているわけではなく,システムのふるまいを支配する確率分布をどのようにとらえるかによって,どちらの手法も適用できる場合が少なくない.a. 無限小摂動解析法(IPA)パラメータを実数θとして,システムの性能評価尺度をθ=ψθと表す.ここにψθはシステムを観測することによって得られる性能評価尺度の標本値であり,はその期待値を表す.無限小摂動解析法(IPA :infinitesimal PA)はPA の中で最も基礎となる手法であり,単純に標本の微分係数dψθ/dθを性能評価尺度の微分係数dθ/dθの推定量とするものである.実際,多くの例で一度の観測から容易にIPA による推定値dψθ/dθを計算できることが報告されている.ところで,実際に推定したい量はdθ/dθであるから,IPAによる推定量が不偏(unbiased)であるためには,微分と期待値の交換dψθdθ=dψθdθ(7・9)が成り立たなければならない.等式(7・9)が成り立つための十分条件の一つとして,有限の期待値をもつ確率変数が存在して,任意のθ,θ+θ∈Θに対して,確率1でψθ+θ-ψθ≦θが成立すればよいことが知られている.ここに,Θはパラメータのとりうる値の集合である.この条件のもとで,ルベーグの有界収束定理を用いることにより,微分と期待値の順序の交換が可能となる.ただし,上の条件はψθがパラメータθに関して連続であることを要求しており,実際問題への応用を考えたとき,厳しい制約になるかもしれない.そこで,ψθがθに関して不連続な場合に対応したIPA の発展形が次に述べる平滑化摂動解析法である.b. 平滑化摂動解析法(SPA)IPA が不偏な推定量を与えることができないとき,式(7・10)のように条件付き期待値を用いることによって,微分と期待値の交換を可能にする方法が平滑化摂動解析法(SPA : smoothed PA)と呼ばれる手法である.dψθdθ=dψθdθ(7・10)ここで,は観測により得られる全情報の部分情報であり,式(7・10)の微分と期待値の交換が成り立つように(例えば,ψθ+θ-ψθ/θに有限の期待値をもつ上界が存在するように)選ばれなければならない.このはSPA における特徴付け(characterization)または平滑化σ-集合体と呼ばれる.SPA では,式(7・10)が成立するための適当なを見つけることや,不偏推定量dψθ/dθ自体の計算が困難になることがあり,場合によっては補助的な観測が必要になることもあるので注意を要する.また,として全情報が選ばれたとき,SPA とIPA は一致する.このことから,IPA はSPA の特別な場合であるということができる.一般に,複雑なシステムに対するこう配推定量は,IPA とSPA の考え方を組合わせて導かれる場合が多い.c. スコア関数法(SF)ここでは,確率分布がパラメータθに依存し,標本値ψはθの影響を受けない場合を考える.また,ψは実数値確率変数ベクトルの関数であり,このの分布がパラメータθをもつものとする.の確率分布が密度関数をもつものとすると,性能評価尺度は,θ=ψ=ψd(7・11)と表される.ここに,期待値は確率分布がパラメータθに依存することを表している.密度関数がθに関して微分可能であるとすると,式(7・11)を微分することにより,もし微分と積分の順序の交換が可能であれば,dψdθ=ψ′d=ψ′(7・12)となり,結局dθ/dθの不偏推定量としてψ′/を得る.ここに,′はのθに関する微分係数である.この′/はスコア関数(SF: score function)と呼ばれ,しばしばdln/dθとも表される.SFを用いると,シミュレーションの際に標本値ψと同時に′/を計算しておくだけで,微分係数の推定値を得ることができる.式(7・12)の微分と積分の順序の交換ができるための十分条件としては,任意のとθ,θ+θ∈Θに対してψ용-≦θを満足し,かつd<∞であるような関数が存在すればよいことが知られている.この条件は,本質的には密度関数に関する条件であるため,標本を調べることを要求するIPA と異なり,比較的容易に確認することができる.また,この条件さえ満たせば,IPA が不偏な推定量を与えられない場合でも,SFにより不偏推定量を得ることができる.しかし,IPA とSFの両方が利用できる場合には,IPA による推定量のほうが分散が小さくなることが知られている.d. 強一致性について一般にシステム性能評価においては,定常状態での評価が重要視されており,θを定常状態での性能評価尺度とすると,時間が十分たったときにその微分係数に十分近づくような推定量,すなわちlimΨ=dθ/dθが確率1で成立するような推定量Ψを求めることが問題となる.これに対しては,不偏性の条件Ψ욈=dθ/dθを満足し,かつエルゴード性を満たすような適当な確率変数列Ψ욈욈を求めることができれば,大数の強法則から,Ψ욈dθdθ∞∑욗が確率1で成り立つので,この左辺が強一致性を満たす推定量Ψとなる.この確率変数列Ψ욈욈を求めるために,再生型過程(regenerative process)の理論や定常点過程(stationarypoint process)の理論などが用いられる.
