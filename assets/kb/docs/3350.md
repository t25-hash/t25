# 7・4・3 摂動解析法

> **章**: 第7章 離散事象システム
> **節**: 7・4 パフォーマンス解析

待ち行列理論などによりシステムの性能を解析的・数値的に評価できるモデルのクラスは、残念ながらあまり広いとはいえない。そのため、システムの性能評価のための一般的な手段として、実際に稼働しているシステムを観測したり、あるいはシミュレーションによって、平均待ち時間や待ち行列長などの性能評価尺度を推定するという方法に頼らざるを得なくなる。ところで、現実のシステムには設計や運用に関するさまざまなパラメータがあり、性能評価の際には、与えられたパラメータの値に対する性能評価尺度の値だけでなく、パラメータの値が変わると性能評価尺度がどのように変わるのかといった問題も重要になる。こうしたパラメータ感度を推定する問題に対しては、従来、実際にパラメータの値を変えて性能評価尺度を推定し、その推定値を比較するという方法が一般的であった。この方法は、確かにほとんどすべてのモデルに対して適用可能であるが、得られる感度推定量の分散が大きくなることや、パラメータの値を変更するたびに推定を繰返さなければならないなどの欠点がある。摂動解析法(PA : perturbation analysis)は、パラメータが連続量である場合に、実際に稼働中のシステムを観測しながら、あるいはシミュレーションによって、(パラメータの値を変更することなく)性能評価尺度のこう配を効率良く推定する方法として知られている。以下、a,b項では、システムのふるまいを支配する確率分布がパラメータに依存しない場合に対する摂動解析法を概説し、c項では逆に、確率分布のみがパラメータに依存する場合について述べる。これらの手法は異なる対象を問題にしているわけではなく、システムのふるまいを支配する確率分布をどのようにとらえるかによって、どちらの手法も適用できる場合が少なくない。ここにψθはシステムを観測することによって得られる性能評価尺度の標本値であり、はその期待値を表す。実際、多くの例で一度の観測から容易にIPA による推定値dψθ/dθを計算できることが報告されている。ここに、Θはパラメータのとりうる値の集合である。この条件のもとで、ルベーグの有界収束定理を用いることにより、微分と期待値の順序の交換が可能となる。ただし、上の条件はψθがパラメータθに関して連続であることを要求しており、実際問題への応用を考えたとき、厳しい制約になるかもしれない。そこで、ψθがθに関して不連続な場合に対応したIPA の発展形が次に述べる平滑化摂動解析法である。b. 平滑化摂動解析法(SPA)IPA が不偏な推定量を与えることができないとき、式(7・10)のように条件付き期待値を用いることによって、微分と期待値の交換を可能にする方法が平滑化摂動解析法(SPA : smoothed PA)と呼ばれる手法である。このはSPA における特徴付け(characterization)または平滑化σ-集合体と呼ばれる。また、として全情報が選ばれたとき、SPA とIPA は一致する。このことから、IPA はSPA の特別な場合であるということができる。一般に、複雑なシステムに対するこう配推定量は、IPA とSPA の考え方を組合わせて導かれる場合が多い。c. スコア関数法(SF)ここでは、確率分布がパラメータθに依存し、標本値ψはθの影響を受けない場合を考える。また、ψは実数値確率変数ベクトルの関数であり、このの分布がパラメータθをもつものとする。ここに、期待値は確率分布がパラメータθに依存することを表している。この条件は、本質的には密度関数に関する条件であるため、標本を調べることを要求するIPA と異なり、比較的容易に確認することができる。また、この条件さえ満たせば、IPA が不偏な推定量を与えられない場合でも、SFにより不偏推定量を得ることができる。しかし、IPA とSFの両方が利用できる場合には、IPA による推定量のほうが分散が小さくなることが知られている。この確率変数列Ψを求めるために、再生型過程(regenerative process)の理論や定常点過程(stationarypoint process)の理論などが用いられる。
