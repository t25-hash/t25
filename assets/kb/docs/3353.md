# 7・5・1 ハイブリッドシステムの特徴

> **章**: 第7章 離散事象システム
> **節**: 7・5 ハイブリッドシステム

連続値をとる状態(連続状態)と離散値(シンボル)をとる状態(離散状態)が混在するダイナミカルシステムをハイブリッドダイナミカルシステム、略してハイブリッドシステム(hybrid system)という。連続状態としては物理的システムの物理量などがあり、離散状態の例としては、制御則の切換え、計算機のコマンドなどがある。離散状態の遷移については時間駆動型の場合もあるが、多くが事象駆動型であり、非同期的に事象が遷移することで離散状態が変化する。ハイブリッドシステムのモデルには、非線形微分方程式を基本とするモデルと、オートマトンなどの離散事象システムモデルを基本とするモデルに大別される。前者においては、離散状態の挙動が差分方程式で記述され、連続状態の挙動は不連続でインパルスがあるような微分方程式で記述されるので、連続状態の挙動が複雑な場合に適したモデルである。ただし、モデルの解の一意性が崩れる場合もある。後者においては、離散状態の挙動を離散事象システムモデルで記述し、各離散状態ごとに連続変数の挙動を記述する。そのため、離散状態の挙動が複雑な場合に適したモデルである。非線形微分方程式モデルを用いた場合には、リアプノフの定理などの非線形システム理論を拡張することによりシステムの解析、設計が行われる。離散事象システムモデルの場合には、ハイブリッド6―183システムのセマンティックスが非加算無限の状態をもつトランジションシステムで定義されるために、解くべき問題に関して等価な有限状態トランジションシステムを求めることでアルゴリズミックに問題を解くことができる。
