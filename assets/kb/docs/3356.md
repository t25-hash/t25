# 7・5・4 リアプノフ関数による安定解析

> **章**: 第7章 離散事象システム
> **節**: 7・5 ハイブリッドシステム

a. ハイブリッドシステムと解軌道リアプノフ(Lyapu-nov)の手法は、ハイブリッドシステムの安定性を示すための最も一般的で、かつ有力な解析手法であるといえる。ここでは、7・5・3項までとは少し表現を変え、次のように記述されるハイブリッドシステムを考える。:は連続状態変数に関し、リプシッツ(Lips-chitz)連続なベクトル場である。Init⊆は初期状態の集合であるとする。なお、ハイブリッドシステムでは連続状態のジャンプも許されるが、ここではそのような連続状態のジャンプはないとする。しかしながら、離散状態の変化による微分方程式の右辺の不連続性に注意しなくてはならない。このような離散的な遷移が生じる時刻(事象生起時刻)の列を(=0,1,2,...)で表すとする。一般に離散状態が変化する際、解軌道がの近傍に微小な時間とどまり続けることを許すなら、スライディングモードが発生する。いわゆる微分包含を用いたフィリポフ(Filippov)の意味での解はこのスライディングモードの極限として定義される。一方、実用上の観点から、スライディングモードは望ましくないという考え方もあり、その場合の解の定義は次に挙げるカラテオドリの解(Caratheodory デッドロックある事象生起時刻で切換わるべき離散状態が存在しなくなり、それ以降解が延長できない。より一般にはハイブリッド状態空間の正不変集合に対しても定義できる。正不変集合⊂とは、に属する任意の状態を初期値とする解軌道がそれ以降必ずに属するような集合である。平衡点は、この正不変集合が単一の要素のみからなる場合として定義されるが、同じ連続状態の値をとり続け、離散状態が動くことを許すような正不変集合をもってハイブリッドシステムの平衡点と定義することができる。また一般性を失うことなく、平衡点の連続変数を0∈とできる。特にこの平衡点の安定性は、連続状態に着目した通常のリアプノフの意味での安定性を適用することができる。c.
