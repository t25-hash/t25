# 7・5・5 スイッチング制御

> **章**: 第7章 離散事象システム
> **節**: 7・5 ハイブリッドシステム

a. スイッチング制御とはスイッチング制御系は、制御対象(図7・28の不確かなシステム)と複数のコントローラ、さらにこれらを切換えるスーパーバイザから構成される。ここでそれぞれのコントローラは、さまざまな制御理論の成果をふまえて設計することもできる。したがって一般的には、各コントローラは制御対象のモデルという、いわば事前のデータに基づき設計される。この点は、ロバスト制御理論などと共通である。一方スーパーバイザは、入出力データに基づきコントローラの切換えを行う。入出力データという、いわば事後のデータが活用される制御系の構成は、いわゆる適応制御理論と共通である。制御対象のモデルという事前データ、さらに入出力データという事後データの両方を活用する制御系設計の枠組ととらえると、例えばゲインスケジューリング制御もその一つとして挙げられる。ここで取上げるスイッチング制御は、理論的にも実用的にも発展途上にある制御系設計の枠組である。また現時点で、ゲインスケジューリング制御ほどの系統的な成果が得られているわけではない。しかしながら、制御系設計における工学的に重要な問題として(1) モデルの不確かさ(2) 拘束条件(3) 故障などを取上げ、これらを含む不確かなシステムに対処できる高機能な制御系設計の枠組を考えたとき、スイッチング制御系の重要性は増してくるものと考えられる。以下では、コントローラのスイッチングが有効に活用される例として、拘束システムに対するスイッチング制御則を紹介する。なおスイッチング制御は、さまざまなアプローチにより大きく発展している研究分野である。より詳しくは文献(79)(80)とその参考文献などを、近年の研究の動向については文献(81)~(84)などを参考にされたい。b. 拘束システムのスイッチング制御アクチュエータの性能限界や制御対象の保護を目的とする物理的な制約は、制御系が達成する性能に大きな影響を与える。ここでは、このような信号の大きさに関する拘束条件を満足する一方で、制御性能の劣化を防ぐようなスイッチング制御系の構成法を紹介する。. 最大出力許容集合制御系が拘束条件を破らず安全に動作するという条件が、最大出力許容集合と呼ばれる集合を利用することで表現される。まずはじめに、制御対象∑욣、コントローラ図7・28 スイッチング制御系6―189∑욂からなる図7・29の閉ループ系を考える。この閉ループ系は、次のように線形離散時間システムとして表現されているとする。+1=+(7・27・a)=+(7・27・b)式(7・27・b)の出力は、拘束条件を表現するために設けられる出力で、制御入力や状態変数の一部など、大きさを制限したい信号を集めてきたものである。そこでここでは、制御系が∈の拘束条件を達成しながら動作するよう求められているとする。拘束条件を規定する集合は=∈욡≦1, =1,...,とする。욡は、ベクトルの第要素を表している。また加わる外乱についても、∈のように大きさが制限されていて=∈욖욢≦1, =1,...,としておく。制御系の初期状態がで、外乱が加わったときの出力を;,で表すことにする。このとき制御系が拘束条件を満たすとは;,∈, ∀∈, ∀が成立することなので、逆にこれが成立する初期状態をすべて集めた集合を次のように考える。=∈욗;
