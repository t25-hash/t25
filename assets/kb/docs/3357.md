# 7・3・1 強化材と強化機構

> **章**: 第７章 セラミックス
> **節**: 7・3 複合セラミックス

セラミックス部材は高温環境下での使用が期待されるため，強化
図7･23 連続繊維強化複合セラミックスにおける破壊強度の（曲げスパ
粒内・外に1～100nmサイズの粒子（ナノ粒子）が分散した微細
構造を特徴とするナノコンポジット（nanocomposite，図7･17)
では1GPaを超える極めて高い曲げ強度が発現する．例えば，高
強度ナノコンポジットにおけるナノ粒子/母相セラミックスの組合
せでは，SiC/ AlO ，ZrO ，SiN ，MgO等が報告されている．
この高強度化の理由として，(１)組織微細化による潜在欠陥寸法の
減少，(２)ナノ粒子と母相の熱膨張率差に起因する残留熱応力によ
（stress -induced phase transformation），(２)き裂前縁のわん曲
（クラックボウイング，crack bowing），(３)き裂面の偏向・屈曲
（クラックディフレクション，crack deflection），(４)マイクロク
ラッキング（microcracking），(５)き裂面架橋（ブリッジング，
bridging）等が提唱されている．以下にそれぞれの概略を説明す
る．
るき裂進展抵抗の増加が挙げられる．さらに前者の原因として，粒
界上のナノ粒子による母相の粒成長抑制や母相粒内のナノ粒子周囲
に発達する転位構造（サブグレイン）等が考えられている．
セラミックスを機械部品に用いる場合に問題となるのは破壊強度
よりむしろ破壊靱性値や破壊エネルギーの低さであるので，高靱性
化を目的としたZrO粒子，ウィスカおよび連続・不連続繊維によ
る複合セラミックスが活発に研究・開発されている．一般にセラミ
ックスは高温を伴うプロセスを経て製造されるため，チラノ繊維や
ハイニカロン等，高耐熱性セラミック繊維の開発により複合セラミ
ックスの可能性が大きく拡がった．複合セラミックスでの有効な高
靱性化（toughening）機構として，(１) ZrOの応力誘起相変態
ZrOにMgO，CaO，YOおよびCeO等を少量固溶させると
高温安定相である正方晶は室温まで安定に存在できるが，き裂先端
近傍での高い集中応力場においてマルテンサイト変態により室温安
定相である斜方晶に変化し，体積が約4 %膨張する．この機構が応
力誘起相変態と呼ばれる．ZrO粒子の応力誘起相変態による体積
膨張がき裂面閉口応力を発生させ，き裂先端の応力拡大係数が減少
して高靱性化が達成される （図7･18）．
強化材によりき裂進展が局所的にトラップされるとき裂前縁のわ
ん曲（ボウイング）により高靱性化が実現する（図7･19）．これは
き裂前縁長さの増加による破壊エネルギーの増加に起因する ．
強化材によってき裂面が偏向（ツイスト）や屈曲（チルト）した
場合（図7･20），き裂先端では局所応力場が変化して混合モードの
き裂面開口状態となり，き裂は進展しにくくなる ．この高靱性
化機構がクラックディフレクションと呼ばれる．
き裂先端の高い応力集中によって主き裂とは合体しない多数の微
視き裂（マイクロクラック）が発生すると，その領域は微視き裂開
口変位の総和に相当する体積膨張が生じ，ジルコニア粒子の応力誘
起相変態と同じ理由により高靱性化が実現する ．マイクロクラ
ッキングを誘発する要因として，強化材-母相セラミックス間での
熱膨張率の相違（図7･21）や結晶粒内の熱膨張係数異方性等に起
因する残留熱応力がある．
アスペクト比の大きな強化材（ウィスカや不連続繊維等）または
連続繊維で強化された複合セラミックスにおいて強化材-母相セラ
ミックス界面での破壊が先行して主き裂がフィラーを貫通しない場
合，それらがき裂面を架橋（ブリッジ）してき裂先端後方でき裂を
開口させる応力を負担するため，き裂先端の応力拡大係数が減少し
高靱性化が達成される（図7･22）．き裂面架橋によって高靱性化さ
れた複合セラミックスではき裂進展に伴うき裂進展抵抗（臨界応力
拡大係数）の顕著な向上，いわゆる上昇型 曲線挙動が特徴的に
現れる．このき裂面架橋は（強化材の破壊強度/界面はく離強度）
比や（強化材の半径/界面はく離長さ）比が大きい場合に実現しや
すい．さらに，架橋した強化材がき裂面から引抜かれる際の摩擦抵
抗がそれらを破壊しない程度で大きいとき，破壊エネルギーの増加
も実現する．つまり，き裂面架橋による高靱性化をより促進させる
ためには，強化材-母相セラミック界面の特性制御が最も重要であ
るといえる．
