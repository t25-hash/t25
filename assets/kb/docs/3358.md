# 7・3・2 強度評価法

> **章**: 第７章 セラミックス
> **節**: 7・3 複合セラミックス

セラミックスの破壊強度は引張試験ではなく曲げ試験で評価され
ることが多い．その理由として
(１) セラミックスの破断ひずみが他の材料に比べて極端に小さ
いため，正確な引張強度の測定にはわずかな引張軸のずれも許容で
きず，非常に高い試験精度が要求される．
(２) ダンベル型やドッグボーン型等の典型的な引張試験片の加
工が容易ではない．
(３) 引張試験片のゲージ部ではなくゲージ部端や引張支持部等
の応力集中箇所での破壊が生じやすい．
などが挙げられる．同じ理由により，複合セラミックスの破壊強度
も一般的に曲げ試験で評価される．粒子またはウィスカ分散型複合
セラミックスの場合は単相セラミックスと同じ条件の曲げ試験
が採用されてもほぼ問題とならない．しかし，連続繊維で強化され
た複合セラミックスではせん断強度が低いことが多いため，（曲げ
スパン/試験片高さ）比の大きな曲げ試験でないと中立面近傍での
せん断破壊が引張破壊に先行する事態となりやすく（図7･23），異
種複合セラミックス間での破壊強度比較の際に混乱が生じてしま
う．さらに，連続繊維強化複合セラミックスの曲げ試験では荷重点
および支点での集中応力による圧壊が生じることも多いため，ナイ
フエッジの曲率半径や材質の選択に留意する必要がある．
