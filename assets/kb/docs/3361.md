# 1・2・4 人の役割

> **章**: 第1章 概論
> **節**: 1・2 生産システムの基本概念

これまでの生産システムでは機械を中心とした自動化が追求され、作業者(worker)には機械に合わせた単調な作業が強いられてきた。しかし、作業者の能力や働きがいを引出すような人間を中心とした新しい生産システムが模索されている。作業者の高齢化という背景もあり、作業者意識(達成感、働きがい)、生産性(pro-ductivity、作業効率、作業品質)、作業負荷(精神的苦痛、肉体的苦痛)、安全性、製造コスト(production cost、設備投資、製造リードタイム)などを評価基準に、適用の可能性が検討されるようになってきた。1990年代になって生産品種の多様化や生産量の変動に柔軟に対応できるセル生産方式が注目されるようになったが、これは作業者の能力や働きがいを引出して成功した事例といえる。また、トヨタ生産方式(TPS : Toyota production system)の一つに挙げられるカイゼン(kaizen)は、作業者による生産システムの継続的な改善活動であり、作業者の能力を生かした日本発の手法として英単語にもなっている。さらに、作業者の技能や熟練に依存してきたといわれる日本の製図1・2 製品の種類と生産量に応じた生産形態図1・3 次世代生産システムの基本機能造業において、技能(skill)の伝承や作業の訓練を支援する情報技術の必要性が指摘されるようになってきた。ものづくりと人の関わりの中で、これまでの製造技術(MT : manufacturing technol-ogy)と技能の伝承や作業の訓練を支援する情報技術(IT : infor-mation technology)を融合した新しい生産システムを構築することが期待されている。近年、技術的、経済的、ならびに社会的要因の相互影響により、生産システムを取り巻く環境は急速に変化している。特に、新世紀に入って高度情報化社会、少子・高齢化社会、地球環境との調和を目指した社会の実現が求められ、人間、環境、ならびに生産システムの相互関係を重視した新たな生産環境の構築が望まれている。その一つの方向として、人間の思考モデルに基づくグローバルネットワーク型フレキシブル生産システム(flexible manufacturing sys-tem)の構築が挙げられる。その実現のためには、(a)熟練設計者や技能者の経験および勘のディジタル化、(b)国際協調・分業体制を実現するための自律分散化、(c)人間と生産システムの協調作業化、(d)環境調和型生産体制の実現、に関わる要素技術の確立が基本的に重要になる。今後の生産環境や関連する要素技術の発展動向を勘案すると次世代生産システムは、図1・3に示すような基本機能を具備することが必要不可欠である。素材を入力とし、加工、組立、洗浄、検査などの各工程を経て部品や製品を出力とする離散系の生産システムを構築あるいは運用するためには、さまざまな要素技術が必要となる。図1・4には、離散系の生産システムを支える代表的な要素技術を示す。それらの要素技術は、生産プロセス機能、システム制御機能、生産情報処理機能、システム保守・保全機能、生産マネジメント機能に大きく分けられる。以下では、離散系の生産システムの中でも自動化が進展しているとされる機械加工を対象とした生産システムを例にとって、その基本機能とそれらを支える技術の中でも特に重要な要素技術について概観する。なお、このような機械加工用の生産システムは、広範な要素技術を必要とすることから、他の加工や組立、洗浄などの工程を含む生産システムにも適用可能な要素技術を数多く含んでいる。
