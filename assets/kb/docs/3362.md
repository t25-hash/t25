c .保証試験（proof test ) セラミックスは強度にばらつきの
ある脆性固体である．したがって，注意深く設計された部品におい
ても強度の小さい部品が紛れ込んでいる可能性がある．このような
部品を除去するため，あらかじめ荷重をかけ，弱い部品は壊してし
まい，生き残った部品に対し強度を保証しようとするのが保証試験
の考え方である．保証試験を行う際に気をつけなくてはならないい
くつかの注意点がある．まず第一は，セラミックスの強度が潜在欠
陥の強度に支配されているものとすれば，有害な欠陥を含む部品を
確実に取り除くような保証試験荷重を選定する必要があることであ
る．そのためには保証試験荷重による応力分布は実機荷重による応
力分布と等しいかそれを安全側に包含するようなものでなくてはな
らない．次に，SCGの影響を考えなくてはならない．除荷中に生
じるSCGによる強度劣化は保証試験の効果を減じてしまう．ま
た，保証試験負荷過程におけるSCGによる強度劣化は，保証試験
で棄却される部品数の増加をもたらし，歩留りを悪くする．したが
って，保証試験はSCGの影響を最小とするように手早く行わなく
てはならない．また，保証試験荷重は安全率を見込んで実機荷重よ
り大きく設定するのが普通であるが，あまり大きく設定すると歩留
りが悪くなるので，適正な保証試験荷重レベルを設定する必要があ
る．
を用いて基本的には評価できるが，適正な保証試験荷重レベルの設
定，長時間強度保証，温度荷重に対する保証試験法，環境効果の影
響，b項で述べた強度確認試験との違いなどまだ解決されていない
点も多い．今後実機の設計・開発を通じて理論および技術が進歩し
ていくものと思われる．
