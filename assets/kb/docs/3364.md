# 1・3・3 生産設計・計画技術

> **章**: 第1章 概論
> **節**: 1・3 生産システムを支える技術

製品開発環境においては,情報通信ネットワークを介した三次元ソリッドモデルの利用による設計部門と製造部門における情報の共有化や複数業務の同時並行化が推進され,コンカレントな生産設計図1・6超精密切削中の加工点近傍における現象図1・7 機能集積型インプロセスセンサの一例プロセスが実現されている.CAD/CAM/CAE技術や仮想環境によるシミュレーション技術は,広範な製造産業において,もはや特別な技術ではなくなり,最近では自動車産業や航空機産業における実用例が中心であった仮想生産(virtual manufacturing)やディジタルモックアップ技術が,工作機械製造産業においても広く適用されている.一方,CAD/CAM 一貫処理において,CAD とCAM のインタフェースであり,CAM の最上流に位置する工程設計(process plan-ning)は,CAD とCAM を統合化するうえでキーテクノロジーとして従来から生産設計技術分野における一つの研究分野を構成して活発に研究が進められてきたが,現状でもなお,図1・9に示す工程設計の上流部分の自動化は,下流に位置する作業設計に比べて遅れている.現在,開発されている自動工程設計システムは,対象とする部品形状と生産環境を限定してさえもなお,ツールパスの自動生成などの作業設計レベルを除いては,現実の問題に十分に対処できる段階には至っていない.今後,自動工程設計(computeraided process planning)技術を確立するためには,熟練工程設計者による工程設計プロセスを詳細に分析し,そこで用いられる知識の根拠や背景を明らかにしたうえで,総合的な観点からシステム構築を行うことが必要である.三次元ソリッドモデルによるCAD モデルを用いた生産環境のディジタル化により,仮想環境における生産計画シミュレーションや生産システムの稼働シミュレーションが活発に行われている.複雑かつ大規模化する生産システムにおける設計工程や製造工程の効率図1・8 インプロセスマイクロセンサの一例図1・9工程設計の基本構成的な情報管理と運用のために,システム要素の個別の高度化だけではなく,製品の設計・生産に関わるすべての情報を工程管理も含めて一元化・統合化し,生産性を向上することが望まれる.そのような観点から製品のリードタイムやサイクルタイムの短縮化をねらったPDM(product data management)の重要性も高まっている.
