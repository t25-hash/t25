# 1・3・4 自己診断・修復技術

> **章**: 第1章 概論
> **節**: 1・3 生産システムを支える技術

生産システムの自動化や高度化が進展する一方で，現場の直接作
業員や自動化されたシステムを陰で支えてきた熟練技能者が減少し
ている．その結果，いったんシステムが故障するとそのシステム構
成や構造が複雑であることから，システムの故障そのものも複雑化
し，原因究明に長時間を有したり，場合によっては原因特定が不可
能になる場面も少なくない．したがって，生産システムの中核
処理機能には，運転中に予期しないトラブルが発生した場合に，そ
の原因を特定するとともに，最適な処理を施すことのできる高度な
自己診断・修復機能を具備することが求められている．すなわち，
自動修復を基本とした保全システムを合理的に構築することが必要
となる．
近い将来，複数のメンテナンス用自走ロボットが工場フロア内を
自由に走行し，適宜，自己診断用センサからの信号情報に基づいて
セルの自己診断作業を行い，異常検知した場合には，迅速にセル内
の故障したモジュールを発見して新たなモジュールに交換し，自動
修復する光景を目にすることもあるだろう．その場合に，自己修復
容易性を考慮した生産システムの構造構成や構築方法についての新
たな検討が必要となる．なお，このような自己診断・自動修復ロボ
ットの実現は，移動ロボット技術，自律分散システム技術，セル構
造のモジュラデザイン，センサ技術などの要素技術の総合結果とし
て得られる．
