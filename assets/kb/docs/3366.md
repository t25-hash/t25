# 1・3・5 検査・品質管理技術

> **章**: 第1章 概論
> **節**: 1・3 生産システムを支える技術

品質管理(quality control,quality management)は,JISZ 8101:1999によれば「買い手の要求に合った品質又は,サービスを経済的に作り出すための手段の体系」と定義されている.最近では,品質について従来のように機能,構造,性能等の所要の仕様を満足するだけではなく,操作性,安全性,感性,環境適合性,倫理性などを含めて顧客満足を図ることが要求されるようになってきた.高品質の製品を生産するためには,総合的品質管理(TQC:total quality control,TQM : total quality management)のもとで全社的な品質管理体制の構築が必要不可欠であり,それを支える品質機能展開(QFD : quality function deployment)や実験計画法など品質工学に包含される一連の方法論が確立されている.生産システムの検査工程では,三次元画像データを三次元CADデータと参照することにより,最終的にデータの整合性を確保し,所要の品質を同定することが行われている.さらに,検査工程で得られた品質に関する不具合データを,上流の設計や生産にフィードバックすることによって再発や類似の不具合の防止に役立てられるような仕組みを合理的に実現することも必要である.ところで,生産システムの最終工程である検査・品質管理機能の高度化が進められているが,現状でもガラス製品,射出成形部品,半導体製品の一部の最終検査工程では,人間による官能検査が重要な役割を果たしている.特に,視覚は,人間の有する感覚器官の中でも最も高度な認識機能を有していることが知られているだけに,目視検査に依存した検査・品質管理システムは少なくない.次世代生産環境では,人間の感覚機能を代替可能な高度な機能集積型検査システムの実現が望まれる.
