# 7・2・4 噴射加工

> **章**: 第7章 特殊加工
> **節**:  

噴射加工は媒体に圧力を加え，ノズルより噴射，または遠心力に
より加速して工作物に衝突させ，表面を加工する方法であり，使用
する媒体の種類によって次のように分類される．
a.ショットブラスト 投射材にスチールショット，スチール
グリッド，ワイヤカット，砥粒，ガラスビーズ，有機物など種々の
ものが用いられ，投射方式には遠心投射式，エア加速式，ベルト投
射式，水噴射式などの方法がとられる．用途としては鋳物の砂落と
し，鉄板のスケールとり，ショットピーニング（疲労強度の向上が
目的），金型の清掃，バリ取り，プレス加工品の面取りなど多岐に
わたる．
b.乾式ブラスト（サンドブラスト，sand blast ) 砥粒を空気
によって噴射する方式であり，鋳物の砂落とし，各種表面研磨およ
び清掃，バリ取りなどに使用される．
c.液体ジェット加工 高圧ポンプで発生させた液体を直径
0.1～1mm程度のノズルから噴射し，その衝撃エネルギーにより
行う加工である ．一般には液体として水が使用され，ウォータ
ジェット（water jet ）加工と呼ばれる．圧力（3～400MPa）の選
択により，切断，バリ取り，研磨等に幅広く適用される．また最
近，水中噴射で発生するキャビテーション（cavitation）効果によ
るピーニング（peening，表面硬化加工）にも活用される．噴射圧
力（ ，水の密度（ρ）とすると，噴流の速度＝2ρであ
り，噴流が物体によりせき止められると動圧ρ2が発生する．
これは主としてノズル直後の収束噴流に適用できる．一方，噴流下
流の液滴が硬い物体に衝突するときに発生する衝突圧力は，水撃圧
力＝ρ（ ：水中の音速， ：液滴の衝突速度）となる．液体
ジェット加工はエネルギー密度が極めて高く，加工物に作用する力
が局所的かつ衝撃的であるため，変形しやすい構造のものや軟質材
料の加工に適している．加工点の温度が低いので，発火や爆発の危
険性がある作業にも適用可能で，加工部の材料組織に熱変性やガス
化を生じさせない．水によるわずかなぬれと洗浄作用によって，加
工中に発生した粉じんが飛散することがない．この特徴をもった液
体ジェット加工に研磨材を添加したアブレシブウォータジェット
（abrasive water jet ）加工は，水だけの噴流では加工しにくい金属
や複合材料等あらゆる材料の加工を超音速切削により可能とする．
加工機としてロボットとの組合せにより平面，曲面を問わず任意の
形状に自由かつ精密な加工が可能である．表7･3に高圧水のみの切
断例と研磨材を混入した切断例を示す．図7･14にアブレシブウォ
ータジェット用ノズルの構造図を表す．
