# 1・4・2 多量生産システム

> **章**: 第1章 概論
> **節**: 1・4 生産システムの歴史

a. フォード生産システム(Ford production system)アメリカの自動車会社のフォード(Ford)社が,1910年代にT 型フォードの生産に流れ作業を導入して,多量生産時代を築いた.1903年に自動車会社を設立したフォードは,低価格で,作りやすく,容易に運転でき,修理可能な大衆車を実現するために,標準化(standardization),単純化(simplification),専門化(specializa-tion)の3S を目標とした生産システムを考案した.3S とは,部品の完全互換性を保障することで調整作業をなくし,設計を単純化することで取付作業を簡単化し,作業を細分化,専門化することで習熟を早めることである.このような考え方を具体化した生産システムが,流れ生産方式である.流れ生産方式とは,細分化された作業を受持つ多くの作業者のところへ生産対象となる自動車をコンベヤによって搬送させ,受持ちの作業を繰返し行うことで自動車を完成させるものである.1920年に流れ生産が定着すると,T 型フォードは当初の価格の3分の1になり,自動車の大衆化を実現した.この生産方式は,自動車のみならず,家庭電化製品などさまざまな分野で用いられ,品種数や自動化の程度は異なるが,現在でも工場の基本となっている.流れ生産では,細分化した作業を繰返す時間(サイクルタイム)が短ければ短いほど,作業は単純化され,未熟練工でも作業が可能になる.サイクルタイムが短いと生産量も増加するので,極端に短いサイクルタイムで作業を設定するケースが生じたり,コンベヤという機械に作業をせかされたりするので,チャップリンの「モダンタイムス」に現れるような批判も起きた.b. オートメーション(automation)オートメーションとは,連続的な自動作業を行う機械と機械の間を自動的なハンドリング装置(ワークの搬入,挿入,取出し,搬送)によって結び付けて生産するシステムである.すなわち,一つの工程を受持つ機械を完全に自動化してもオートメーションではなく,機械化と呼ばれる.オートメーションは,いくつかの機械化された生産機械の要素をハンドリング装置で有機的に結合させたものを意味し,システムの概念としてとらえることができる.オートメーションは,初め,フォード社がプレス工場の自動挿入,取外しに用い,その後,1950年にプレス工場全体のオートメーション化を実現した.これは,組立ラインの流れ生産の利用をモデルとしたものであるとされている.しかし,同じような考え方では,トランスファマシンがある.連続する工程をハンドリング機械で結合するもので,これは,1924年にイギリスのモーリス自動車会社が製作し,使用した.自動化された生産システムで,多量に,安価に物を生産するという考え方は,欧米において第二次世界大戦で完成され,大戦後の多量な需要を支えてきた.c. 品質管理法多量な生産を維持するうえで重要な役割を果たすものの一つに,品質を維持することがある.効率良く生産を行うだけでなく,一定の品質の物を供給していく必要がある.流れ生産が導入されたときとほぼ同時の1920年代に品質管理に統計的手法の応用が試みられている.統計的手法は,数学の確立統計論に基づく方法で,簡易に現場で用いることができるように工夫がなされている.現在も用いられている管理図,抜取り検査法,実験計画法などがそれである.これは,基本的に不良品の市場への流出防止が目的であった.その後,不良品を作らない品質管理に変化を遂げ,1966年に日本におけるQCサークルによる不良原因の追究活動が紹介され,品質管理の組織化が行われる.1990年代になると,ISO9000品質保証システムが国際標準として制定され,不良品を作らない工程を管理する品質管理へと変化していく.
