# 1・4・3 多品種少量生産システム

> **章**: 第1章 概論
> **節**: 1・4 生産システムの歴史

a. 多品種混合ライン多量生産によって需要が満たされるようになると,市場の要求に対応する生産が求められてくる.同じ製品を多量に作り続けるライン生産から,同じ設備で,複数の製品を作るライン生産へと変わり始める.多品種混合ラインには,一定の期間同じ製品を作り続け,段取り換えをしたあとに,違う製品を一定期間作り続けるバッチ式混合ラインと,製品の生産比率に従って製品の混合比を決め,その順序に異なる製品を生産していく混流式ラインの二つの方式がある.このような生産は,1970年代に生産管理にコンピュータが導入されると同時に,自動車や航空機,家電,オーディオ機器などの組立ラインに用いられるようになってきた.b. FMS システム組立ラインが多品種組立ラインに変化すると同時に,機械加工においても同じ設備群で異なる製品,部品を生産する必要性が生じてくる.1970年代では,類似の形状をもつ部品を集めて必要な工作機械群で加工を行うことで,機械加工の多品種化に対応しようとするGT (group technology)が考え出され,ドイツ,日本の工作機械メーカを中心に用いられた.一方,NC工作機械をコンピュータで結合して集中的にコントロールするDNC(direct numerical control)システムの提案が行われ,複数のNC工作機械を有機的に結び付けて運用することが可能になった.1980年ごろから,多品種の機械加工部品を柔軟に生産する目的で,複数のマシニングセンタと搬送装置をシステムとして統合して,有機的に運用するFMS が実用化され始めた.1977年には,レーザ応用複合生産システムの開発が,通産省の大型プロジェクトとして,無人化工場の実現を目指して行われた.このプロジェクトでは,24時間無人運転を実施しながら多品種の生産を実現することが目的とされた.c. POP システム多品種生産が一般化した1980年代においては,仕掛品の識別番号を認識する必要が生じ,多品種自動化システムにおいてさまざまな取組みが行われた.製品に識別コードを付けるために,製品に凹凸を付けて二進法で識別する方法から始まり,バーコードによる識別,ICカードによる識別,二次元コードによる識別などが実現された.8ビット程度から1メガビットまで,リードオンリーからリードライト可能なものまで多くの取組みがなされ,これらの情報と設備情報を組合わせて,さまざまな情報を取扱えるようにした.このようなシステムを,POP(point ofproduction)あるいは情物一体システムと呼ぶ.d. CIM システム1980年代においては,多品種化の要求とコンピュータ化によって,生産情報の統合が意識されCIM(com-puter integrated manufacturing)が提唱された.CIM は,製造現場の情報と生産計画の情報を大規模なデータベースとLAN(localarea network)によって結合して,受注活動,生産計画,生産管理,生産指示を一元化するシステムである.現場情報の収集にPOPを,現場情報のLAN にMAP(manufacturing automationprotocol)を,生産計画にMRP(material resource plan-ning)を用いるなどの取組みがなされた.
