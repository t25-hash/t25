# 7・1・5 環境管理システム（environmental management sys

> **章**: 第7章 環境と生産システム
> **節**: 7・1 生産システムと地球環境問題

環境規制値が厳しくなり，その対策コストや賦課額が急増する一方で，許容された環境負荷量，例えば炭酸ガス排出許容量を地域間，企業間，部門間で売買取引きすることが認められるようになってきた．
生産システムでの生態系負荷の最小化が経済的メリットに直結するため，生態系負荷を可能な限り小さくする生産設計技術および生産管理技術をシステムに導入する必要がある．
地球環境問題の解決を目的としているため，ISO14000シリーズ環境管理の結果は環境報告書として定期的に情報開示するのが一般的である．
ISO14000シリーズで求められている環境活動の四段check），レビュー・見直し（A:action）］の内容，資源エネルギーの入出量，発生した環境負荷量，環境会計（環境関連投資，保全効果，経済効果）などが主たる内容として記載される．
こうした情報開示機能をコンピュータネットワークシステム化することも有効である．
