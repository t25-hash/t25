# 7・2・1 生産設備の地球環境問題への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・2 生産システムの省エネルギー

生産プロセスでの生態系負荷の最小化を実現するためには，徹底有害物質使用中止を実現する必要がある．
生産活動に伴って消費されるエネルギーは，地球温暖化問題，化石燃料枯渇，大気汚染などの原因となる．
とりわけエネルギーの生産・消費段階で発生する二酸化炭素（CO）は大気中の温室効果ガス濃度を増加させることにより，地表および大気の温度を地球規模で上昇させて生態系のバランスを崩すことから，省エネルギーによる温室効果ガスの削減は国際的な最重要課題である．
地球温暖化問題への取組みを定めた「気候変動に関する国際連合枠組条約」が1994年に発効し，先進国の温室効果ガスの削減を法的拘束力をもつものとして約束する京都議定書（Kyoto proto-col）が1997年に採択された．
排出の抑制および削減の対象となる温室効果ガスは，二酸化炭素（CO），メタン（CH），一酸化二窒素（NO），ハイドロフルオロカーボン（HFC），パーフルオロカーボン（PFC），六フッ化硫黄（SF）である．
これらは燃料，洗浄剤，冷媒，電気材料，廃棄物等から排出され，その削減には生産システムが大きな貢献を果たしうる．
日本国内では，温室効果ガスの総排出量を2008年から2012年の間に1990年（一部は1995年）レベルから6%削減することを内容とする京都議定書での国際的な約束のもとで，「地球温暖化対策の推進に関する法律」「エネルギーの使用の合理化に関する法律（省エネルギー法）」などの制定や改正が行われ，法的規制が強化された．
日本の場合，温室効果ガス排出量の90%は石油，石炭，天然ガス等のエネルギーを起源とした二酸化炭素であることから，省エネルギーが最も温室効果ガス排出削減に寄与する要素である．
日本のエネルギー消費を部門別で見ると，産業部門でほぼ50%，民生部門と運輸部門でそれぞれほぼ25 %となっており，産業部門での省エネルギーは非常に重要な役割を期待されている．
しかし，産業部門での生産額当たりのエネルギー消費原単位，二酸化炭素排出原単位は，欧米諸国に比較してすでに低い水準にあるため，産業活動の成長力を失わない中でエネルギー消費の総量増加を熱利用，バイオマスエネルギー，燃料電池などの新エネルギーを工備のエネルギー効率を高める努力が必要である．
新エネルギー源の理システムを高度化し，エネルギー発生特性に適応できる生産シス工場として法規制の対象となっている．
法律では，工場全体だけでとともに，省エネルギー管理上重要な指標について数値が設定され導入の検討を求めることも法的に要請されている．
検義務が規定されている．
工場内において，エネルギー消費原単ルギー目標を設定してエネルギー管理を実施するためには，生産シらない．
エネルギー管理士の配置も要請されている．
しかし一方では，先端技術の目標値が高くなるほどに環境は生産高（売上高）原単位で表示されることが多いため，高付加価た生産量が減ると逆に原単位は増大する．
例えば，大幅な工程短縮やエネルギー効率向上が可能な省エネルギー型生産技術の開発が目標となる．
高精度加工や微細加工に不可欠な大規模空調設備および高レベルクリーンルームは，大電力消費設備でもある．
空調設備の電力消費が全工場電力消費の50～60 %に及ぶこともある．
実加工の有無にかかわらず，連続運転が必要な生産設備や長時間のアイドリング運転が必要な生産設備も高環境負荷を伴っている．
工作機械も全電動化により総消費電力は増大しており，高効率化と摩擦低下などが課題である．
OA化が事務所の電源容量を増加させる例が多く見られるが，工場における生産自動化（FA化）も消費電力増大を招かないようにシステム設計段階でアセスメントすべきである．
この場合，直接的に生産プロセスに関係したエネルギー消費だけに注目するのではなく，工場建屋全体のエネルギー管理を統合化すること，エネルギー使用の平準化によってエネルギーの余剰や不足が発生しないように総エネルギー消費最小化を目指すことが重要である．
例えば，工場受電容量を抑制して需要平準化を実現するために，エネルギー需要が低下する夜間の自動化運転などが効果的である．
省エネルギー技術の研究では，高精度加工環境を維持するため費やされている膨大な空調エネルギーを削減するために，熱不感高精度工作機械の開発，また，工作機械熱変形を知能化機能で補正する仮想空調マシンシステム技術の開発が有効である．
アイドル運転の短縮と省エネルギー化，低摩擦機構の開発による熱発生抑制，高効率駆動モータの開発，待機電力削減，廃熱エネルギー回収技術の併用なども有効である．
工場建築構造の断熱性改善，建屋屋上の緑化，蓄熱装置やコージ抑えるという難しい課題がある．
場単位，部門単位，装置単位で積極的に利用するとともに，生産設多くは小規模単位での分散型利用が可能であるため，エネルギー管テムを開発することも有効である．
なく設備群単位や作業工程単位でのエネルギー管理の徹底を求めるている．
熱処理炉などに対しては，エネルギー効率の高い設備等のギー効率などについて，管理，計測・記録義務および設備保守・点位，設備別のエネルギー消費量とエネルギー効率を把握し，省エネステムに統合化されたエネルギー管理システムを構築しなければな防止活動の一環として評価される．
負荷も大きくなった例が少なくない．
生産系の省エネルギー効果等値製品の生産に移行した場合はエネルギー消費原単位が増大し，まる省エネルギー効果の増大にも留意する必要がある．
