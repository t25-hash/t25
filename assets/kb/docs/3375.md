# 2・1・3 生産工程設計と生産設備設計

> **章**: 第2章 設計・評価技術
> **節**: 2・1 生産システム設計・評価技術の体系

a. 生産工程設計生産工程設計では,加工対象製品の設計データに基づいて,その加工プロセスの設計,すなわち加工部位とその加工方法,加工方法のシーケンスを設計する.また,すべての製品の加工作業を分割あるいはグループ化して,生産システム内の加工設備に割当てる.さらに,作業設計を行い,加工作業の実行に必要となる詳細な加工データとジグおよび工具のデータを生成する.この生産工程設計プロセスをまとめると,図2・3のようになる.ただし,この場合,生産システムで加工を行う予定のすべての製品について生産工程設計を行う必要がある.これは,製品の設計時における生産設計とは異なる点である.第一段階では,加工対象製品すべての特徴を認識し,分析する.製品の特徴は,以下の情報を含む.(1) 製品全体の概略形状および寸法概略形状および寸法は,使用する加工設備の容量および加工設備に取付ける際のジグの選択に重要である.また,薄肉形状,板状形状のように剛性が低い形状の場合には,加工法,ジグおよび加工条件などについて詳細な検討が必要になる.(2) 製品に含まれる加工部位の形状および寸法素形材を製品に変換するための加工部位(加工すべき局所的な形状および寸法)は加工に必要な工具および加工設備の運動軸構成を設計するうえで必要になる.(3) 製品全体および加工部位の要求精度製品全体および加工部位の要求精度,すなわち仕上面粗さ,幾何公差,寸法公差,はめあいなどは,必要な加工設備,ジグ,工具の精度,加工方法,加工を実行する順序などを決定するうえで必要になる.第二段階では,それぞれの加工部位の加工法を選択し,加工する順序を設定する.また,加工部位に対する加工作業を実行することができる加工設備の候補を選択する.この部分を,狭義には工程設計と呼ぶ.このとき,すでに生産システムに含まれる加工設備の計画が作成されている場合,すなわち図2・2におけるシステム仕様および設備仕様の設計が終了している場合には,その加工設備の中か7―10図2・3 生産工程設計プロセスら選択する.そうでない場合には,類似の製品の生産を目的として過去に設計したシステムの加工設備あるいは加工設備のカタログなどから適当なものを仮に選択する.さらに,製品全体および加工部位ごとの加工時間,および加工コストを見積もる.第三段階の設備割当では,すべての製品の加工作業を分割またはグループ化し,生産システム内の適当な加工設備に割当てる.このとき,加工法だけではなく,要求される加工精度,製品の概略の寸法および形状なども考慮して割当てることが必要になる.また,TLあるいはFTLのように,ある期間内におけるライン当たりの生産量からサイクルタイムが定まっている場合,そのサイクルタイムに各加工設備における加工作業時間が合致するように加工工程を割当てることが必要である.これを,ラインバランシング(linebalancing)と呼ぶ.また,FTL,FMS,FMCなどの場合には,各加工設備で使用できる工具の本数に制約があるため,その制約の範囲内ですべての品種の製品を加工できるように割当てることが必要になる.第四段階では,各加工設備に割当てられた加工部位の加工を実行するための詳細な加工データを設計する.この部分を,狭義には作業設計と呼ぶ.ここでは,治工具の設計または選択を行い,加工条件などの設計を行う.この場合も,図2・2におけるシステムおよび設備仕様の設計が終了している場合は,その設備を使用することを想定して,加工データの作成を行う.そうでない場合には,過去に使用した加工設備またはカタログに基づいて加工データの作成を行う.b. 生産設備の選定・設計生産設備の選定または設計は,生産対象製品の生産工程データに基づいて行う.すなわち,各加工設備に割当てられた製品の加工作業を実現することができる加工設備の選定あるいは設計を行う.このとき,選定または設計すべき主要な項目を以下にまとめる.(1) 加工設備について・加工可能な製品の寸法,形状,重量,基準面の位置・必要な制御軸数(同時制御軸数,全体の制御軸数)・必要な加工精度・必要な加工能力(加工動力,剛性など)・必要な工具本数および工具寸法・搬送設備とのインタフェース部(2) 搬送設備について・搬送可能な物体の寸法,形状,重量・搬送経路とその自由度(ライン,経路固定,経路変更など)・搬送速度,加速度・位置決め精度・加工設備とのインタフェース部(3) バッファなどについて・バッファの位置,形状・バッファの容量,寸法・搬送設備とのインタフェース部(4) 治工具について・治工具の形状,寸法,必要個数・加工設備とのインタフェース部(5) すべての設備について・作業者とのインタフェース部・設備の占有面積,占有形状・モジュラ性,機能の拡張性,信頼性,寿命などなお,上記の項目は,設備のハードウェアとして加工機能などからおもに検討している.これ以外に,生産システムの制御および管理を行うソフトウェア面からの検討も必要になる.
