# 2・1・4 生産システムの評価項目

> **章**: 第2章 設計・評価技術
> **節**: 2・1 生産システム設計・評価技術の体系

生産システムの評価は,各種の観点から行うことが必要になる.その中で,代表的な評価項目を挙げると以下のようになる.a. 投資効果生産システムの計画,設計,開発,運用にかかる投資コストとそれから得られる利益との比較において評価するものである.投資効果を評価する手法としては多数の算定方法がある.例えば,最も単純な算定方法は,以下のようになる.総投資利益額=総投資収入金額-総投資額総投資収入金額=総キャッシュフロー+残存価格総投資額=設備および流動資産への投資額(2・1)b. 生産コスト製品の生産に要するコストであり,式(2・1)の設備および流動資産への投資額に含まれる.生産に関わるコストは多岐にわたっており,すべての項目を検討することは困難であるが,一例を挙げると以下のようになる.すなわち,調達する材料コスト,生産システム内外の物流コスト,生産システム内外での在庫コスト,加工および組立などの生産プロセスの実行に関わるコスト,生産設備の保守,修復などに関わるコストなどである.c. 生産管理コスト生産システムにおけるプロセスの管理を行うために必要なコストであり,式(2・1)の設備および流動資産への投資額に含まれる.具体的には,生産システムにおける生産計画,資材調達計画,在庫管理計画などの策定に必要な設備費および人件費である.d. 信頼性生産システムを構成する生産設備の信頼性および品質である.信頼性は,生産設備の保守および修復などのコストに直接影響する.生産システムの設計段階において設備の故障や異常などを事前に解析する手法としては,FMEA(failure mode andeffect analysis),FTA(fault tree analysis),ETA(eventtree analysis)などの手法が挙げられる.また,生産プロセスの運用段階においては,設備の平均修理時間(MTTR : mean timeto repair),平均故障時間間隔(MTBF: mean time between fail-ure)などを挙げることができる.e. フレキシビリティ生産システム全体および生産設備が,生産対象製品の品種および生産量の変更に対してどの程度柔軟に対処できるかを示す.すなわち,当初に設定した生産品種および生産量の変動に対して,生産ラインおよび生産設備のセットアップを変更するコストおよび時間で評価する.また,最近は,当初設定した生産品種以外の製品の生産に適用する場合のフレキシビリティも重要視されている.すなわち,新たな生産品種の導入に対応して,生産システムの全体構成および生産設備の生産機能および生産能力を拡張あるいは再構築するためのコストおよび時間も重要視されるようになってきている.このため,ハードウェアおよびソフトウェアのモジュール性,相互接続インタフェースの標準化およびオープン化が求められている.f. ヒューマンインタフェース完全な無人化生産システムであっても,必要最小限の保守要員が必要であり,生産ラインおよび生産設備と作業者とのインタフェースの評価が重要となる.このインタフェースには,物理的インタフェースと情報インタフェースを考える必要がある.すなわち,作業者が生産設備に対して作業を行う,あるいは生産設備と協調して作業を行う場合の物理的なアクセスおよび接触における安全性および操作性と,作業者が生産設備にデータを入力するあるいはデータを受取る場合のインタフェースである.g. 機械加工システムの評価項目上記の評価項目の具体的な内容は,対象とする生産プロセス,生産設備,生産システムの形態により異なる.そのため,ここでは,機械加工システムに関する代表的な評価項目を表2・2に例示する.ただし,表2・2中では,生産設備,生産ライン(TL/FTL/FMS/FMC),工場全体など,それぞれの生産システムレベルにおける生産能力および生産機能に関する評価項目と,生産システムの生産管理の観点から見た評価項目を示す.
