# 7・3・1 資源循環型システム構築への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・3 生産システムと資源リサイクル

資源の有効利用による地球環境問題解決への取組みに向けて，政
治，経済，社会，産業などのあらゆる領域において資源循環型シス
テムの構築を目指している．例えば，日本の法制度は，「環境基本
法」のもとで「循環型社会形成推進基本法（基本的枠組法）「廃棄
工場での大規模エネルギー利用については，エネルギー管理指定
物の処理及び清掃に関する法律（廃棄物処理法）」「資源の有効な利
用の促進に関する法律（資源有効利用促進法）」「（個別製品）リサ
イクル法」「国等による環境物品等の調達の推進等に関する法律
（グリーン購入法）」などによって，生産-消費-(廃製品化)-再生の
各段階で資源循環型システムの構築を促進させる体系となってい
る．
また，設備稼働に伴う過剰エネルギー，不要エネルギー，エネル
資源循環型の具体的な意味は，資源消費の減量化，廃製品資源の
リユースとリサイクル（recycle ）にある．生産システムにおいて
も，生産対象である製品だけでなく，生産設備，副産物などのすべ
てに資源循環を促進する配慮が求められる．生産工場への資源のイ
ンプットとアウトプットを詳細かつ厳密に管理するシステムを構築
することによって，減量化，リユース，リサイクルを促進させると
ともに，その内容の情報開示も要請される．製品資源の場合には，
なお，直接的な省エネルギーに加えて，企業の植林活動も温暖化
廃製品化段階でのテークバック義務を含めて，ライフサイクル管理
が要請される．
