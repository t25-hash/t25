# 2・1・5 生産システムの評価方法

> **章**: 第2章 設計・評価技術
> **節**: 2・1 生産システム設計・評価技術の体系

a. 評価手法生産システムの評価項目は、表2・2の機械加工システムの例から明らかなように、非常に多岐にわたるとともに、表2・2機械加工用生産システムの評価項目例生産設備生産品質(形状精度、寸法精度、仕上面粗さ)品種・工程別生産時間必要治工具の種類と数平均滞留時間/品種別滞留時間バッファ容量必要パレットの種類と数TL/FTL/FMS/FMC工場品種別リードタイム在庫容量生産可能な品種・工程品種・工程別生産能力品種・工程の拡張性品種・工程別不良率段取り時間(品種・工程変更時の段取り時間)作業性(ヒューマンインタフェース)作業者の総数、必要能力搬送システムの距離、台数セル・機器・治工具の総数、標準化率保守性(MTBF)生産コスト(初期コスト、運用コスト)生産能力・生産機能各レベル共通生産コスト・運用コストリードタイム納期遵守率仕掛り量・在庫量設備・作業者稼働率生産平準化生産品種・生産量の変動に対する追従性生産管理7―11定量的な評価が困難な項目が多い。そのため、設計者の過去の経験に基づいた定性的評価、簡単な数式あるいは過去の経験式などに基づく評価、試作生産設備あるいは生産ラインを用いた実験的評価、およびコンピュータモデルに基づくシミュレーション(simula-tion)による評価などが行われている。特に、近年のように、生産設備および生産ラインの高効率化、複雑化に伴い、コンピュータシミュレーションによる評価が不可欠となってきている。そのために、設計対象である生産システムのレベルに応じた各種のシミュレーションシステムが利用されている。例えば、生産設備レベルにおいては、加工プロセスおよび組立プロセスの実行状態のシミュレーションを行い、衝突チェックあるいは精度チェックを行うシステムが利用されている。また、生産ラインレベルあるいは工場レベルでは、生産対象製品のラインおよび工場内での製品の流れに関するシミュレーションを行い、生産プロセスの進捗状況、仕掛り状況、在庫状況、生産量、生産完了時期などの評価に用いられている。b. シミュレーションによる評価生産システムのシミュレーションは、基本的には生産プロセスの進展に伴う素材、部品、サブアセンブリ、アセンブリなどの流れと生産ラインおよび生産設備の状態の時間的変化を求め、その結果を数値あるいは図表にまとめて出力するものである。このシミュレーションを実行するためには、2種類のコンピュータモデルが必要になる。一つは生産システムのハードウェアモデルであり、これは生産システムを構成する生産設備の構成、配置、生産機能および生産能力、作業者の数および能力、生産対象製品のロットサイズ、生産工程および要求品質などが含まれる。他の一つは生産管理システムのモデルであり、製品の受注予測あるいは出荷予測に基づいて生産計画を立てるとともに、その生産計画を実行するために生産設備に製品の生産工程を割当てるための生産管理プロセスを表現するものである。このシミュレーションモデルと実際の生産システムの差異がシミュレーション結果にそのまま影響するため、基本計画段階の概略モデルを用いたシミュレーションから、詳細設計段階における詳細モデルを用いたシミュレーションに至るまで、シミュレーションモデルの精度を上げながら、各設計の段階に応じた評価を繰返し行うことが必要である。シミュレーションの実行結果は、生産システムにおける生産設備の稼働状態および製品の生産工程の進捗状態に関する時間的な変化、およびこれらの状態変化をある期間集計した統計情報であり、これらの情報から評価に必要な項目を求めるために、出力情報を変換あるいは再構築することが必要になる。また、シミュレーション結果として得られる情報は、シミュレーションモデルの詳細度あるいは入力されるパラメータに依存する。そのため、評価の目標が達成することができるシミュレーションモデルおよび入力パラメータを設計することが必要になる。図2・4および2・5は、機械加工を対象とするFMS に関して、経済性およびスケジュールの評価を行う場合についてシミュレーションの出力データと評価項目の関係の例を示す。また、この評価項目を求めるためのシミュレーションシステムへの入力条件およびシミュレーションシステムでの行うべき処理の内容を示す。シミュレーション結果に基づいて生産コストを評価する手法の一つとして、ABC分析(activity based costing analysis)に基づく手法がある。この手法は、製品の生産プロセスにかかる直接経費および間接経費を必要な活動(アクティビティ)に基づいて分析、積算するものであり、シミュレーション結果から得ることができる統計データからそのコスト評価を行うことができる。シミュレーションによる生産システムの評価手順をまとめると以下のようになる。(1) 評価項目の抽出シミュレーションにより評価したい項目を抽出する。7―12生産システム工学図2・4 経済性評価における評価項目とシミュレーション結果図2・5 スケジュール評価における評価項目とシミュレーション結果(2) シミュレーションモデルの構築設計プロセスにおいて決定している生産システムの構成とその生産管理システムの構成、および(1)の評価項目に基づいて、生産システムのシミュレーションモデルを構築する。このとき、モデル内のパラメータのうち、固定のものと可変のものを区別しておく必要がある。(3) 入力データの作成シミュレーションに入力するパラメータを収集し、シミュレーションモデルに入力する。この入力パラメータの精度がシミュレーション結果の精度に大きく影響する。(4) シミュレーションの実行一定期間について生産システムのシミュレーションを行い、結果を求める。(5) 評価シミュレーション結果に基づいて、生産システムの評価を行う。また、評価結果に基づいて、シミュレーションのモデルおよび入力パラメータを修正し、適切な評価結果が得られる設計解を求める。
