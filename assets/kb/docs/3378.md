# 2・2・1 生産ライン設計の概要

> **章**: 第2章 設計・評価技術
> **節**: 2・2 生産ライン設計

生産ライン設計(production line design)とは図2・6に示す、開発された製品を生産ラインによって生産活動に具現化するための、設計、製作する業務である。生産ライン設計では、開発された製品を製品図面に従って、まず、材料から部品を加工し、それらを組合わせて実際の商品に変換するための加工工法、組立方法を選定し、その手順を決めて加工工程(machining process)および組立工程(assembly process)を設計する。次いで、生産量、生産期間などの生産情報に基づき、具体的な生産ラインの構成、設備投資計画、それを構成する設備、治工具、管理ソフトウェアなどの諸元を設計するための設備仕様、設備投資計画を生産ライン仕様として出力する。そのとき、生産ライン仕様は経済性だけではなく、品質(Q: quality)、納期(D :delivery)、コスト(C: cost)、働きやすさ(H : human)、環境(E: environment)も考慮して設計されなければならない。生産ライン設計は図2・7に示すように、製品情報および生産情報図2・6 生産ライン設計業務の位置づけ図2・7 生産ライン設計プロセス図2・8 生産ライン設計手順7―13を入力条件とし、生産ライン設計方針、生産ライン設置制約条件、生産ライン設計基準などの設計与条件に基づき、ライン設計に必要となる各種資源情報を用いて、処理方法、処理順序、ライン構成、それらを構成する各種要素のスペックなど生産ラインの諸元を、経済性、面積生産性、稼働率、QDCHEなどの多様な評価尺度をバランス良く適正化した結果として出力するプロセスである。
