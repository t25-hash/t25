# 7・3・3 副産物資源循環型生産工程への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・3 生産システムと資源リサイクル

ファクトリー技術体系の基本条件は，製品ライフサイクルにかかわ
る資源循環の水平化を実現することであり，図7･10のオプション
⑤による高品位マテリアルリサイクルである．ただし，オプション
⑤の経路は，最初の出発地であった生産工場に単純回帰するものだ
けに限定されるということではなく，産業生態系 において連鎖
的に循環する経路として構成される場合もありうる．
いずれにしろ最下流段階においてオプション⑤を実現可能とする
ためには，ライフサイクルに包含される数多くのトレードオフ要素
を解決することが前提となる．すなわちライフサイクル視点による
上流側要素技術を含めたシステム化された技術連携を実現するグロ
ーバルコンカレントエンジニアリング概念である．上流側のライフ
サイクル的配慮が大であるほど，下流側の分担は簡易になることは
当然である．
つつある．
現在はオプション④のカスケードリサイクルを含めた全リサイク
ル率の向上が課題目標となっている．しかし，新資源の投入量を減
少させるためには，新製品における再生材料の使用率向上が要請さ
れていることから明らかなように，オプション⑤のホリゾンタルリ
サイクル（水平循環）を前提とする高品位資源循環技術が不可欠で
ある．
優位性が高まっている．
