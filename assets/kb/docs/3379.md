# 2・2・2 生産ライン設計手順

> **章**: 第2章 設計・評価技術
> **節**: 2・2 生産ライン設計

生産ラインの設計の手順は、設計納期、設計する組織などの設計環境の違いによって弾力的に行われることが多く、体系化された手順は存在しないが、その要素となる個々の設計業務に大差はない。本項では図2・8に示す生産ライン設計手順の事例に沿って各種業務の役割およびそこで用いられる設計手法について概説する。なお、実際の組立ラインの設計事例を図2・9に示すが、生産ライン設計は構想検討から始まって、機能検討、仕様検討と、検討の精度を高めながら繰返し行われるのが一般的である。a. 図面・設計情報の整理・確認まず、与えられた製品図面とそれに付随する設計情報を整理する。この段階では製品図面の作りやすさを考慮した検討が重要であり、VE(value engineering)/VA(value analysis)手法や、組立・加工性評価法(DFMA : design for manufacture and assem-bly)により製品図面の作りやすさを定量化する。これを基に、いっそうのコストダウンを製造サイドから追求した製品構成への変更案を導出する手法が用いられ、製品開発段階へフィードバックされる。さらに製品の多種化への対応も重要であり、多種化の視点から製造性を考慮した標準化手法が用いられる。グループテクノロジー(GT : group technology)などを用いて製造基準の統一化、加工方法や組立順序を考慮した部品のグループ化が用いられ、製品設計段階へフィードバックされる。さらに、生産性の検討だけでなく、品質に対する図面情報の整理も品質機能展開やFMEA/FTA などによって行われ、製品機能と製造機能との品質面からの関係が確認され、必要に応じて設計変更を要求する。これらの段階の作業は実際的には製品開発段階と同時並行的に行われることが多い。また、最近では製造部門および市場での環境対応を考慮し、製品図面の検討が行われて、必要があれば再検討を依頼する場合もある。b. 製品の種類と生産量に着目した生産ライン方式まず生産ラインの規模、寿命の目標設定が重要であり、対象製品の需要を予測するための手法や、投資規模を算定するための経済性評価手法が用いられ、生産ラインの規模、自動化方針、稼働条件、投資額などの目標値が設定される。図2・10に製品の種類と量による生産ラインの方式の選択方法例を示す。少種多量生産には全自動で高速に生産する全自動トランスファライン(transfer 加工・組立方法・手順の設計製品図面を部品図に分解し、それぞれの部品を加工していく方法と手順を設計する、さらに各部品を逐次組立てて製品を作り上げていく方法と手順を設計する工程設計業務を行う。一部品の加工であっても通常は複数の加工方法の選定、および複数の加工手順の設計が可能であり、コスト、品質、工程内在庫量などを評価して最適な工法と手順が設計される。図2・9組立ラインの設計手順例加工方法は、各種加工法の選定基準、それぞれの加工コスト・加工時間を設計するための加工条件データベースなどを用いて設計が行われる。その際に、荒加工は前工程で、高精度加工は後工程で行う、加工基準面の加工は前工程で行うなどの配慮が必要である。特に後工程で最終仕上面の表面形状・精度等に影響を与えないような品質面での配慮は重要である。既存設備の活用を前提に加工方法が選定される場合もある。また、製品図面に記載された項目だけでなく、組立工程での容易性、コストダウンを意図してより高い精度の加工が求められる場合もあり、組立工程との連携も重要である。組立方法は、理論的には多様な組立順序計画が部品間の結合関係によって求められるが、基準部品の選定、組付方向の一方向性、部品供給の容易性、多種組立容易性などを考慮して組立順序が決められる。組立ラインの在庫低減、段取り性向上、品質向上などの視点から、一つの基準部品を中心に全部品を組立てていく方法より、複数のサブ組立品をモジュール化してサブラインとして組立品質保証を行い、それらを基準部品に組立てていく方法が主流になりつつある。これらの設計に際しては、既存の類似ラインの工程分析、稼働分析(連続観測、ワークサンプリングなど)、動作分析(サークリップ分析、PTS 法)、品質分析(類似工程工程能力・不良内容)、設備信頼性分析(保全実績データ、設備稼働データ)などを行い、その分析データを参考にしてより迅速かつ信頼性の高い設計が行われる。加工および組立手順は一義的に決められる場合もあるが、多くの場合、入れ替えが可能な組合せが存在する。組合せによって在庫量、滞留・搬送時間、段取り費用などが変化するため、上記加工・組立方法についてもコスト、在庫量などを最適化する組合せ順を選択することが必要となる。設計者の経験で妥当な組合せ順を選択することもあるが、最適化問題として加工・組立方法をモデル化し、線形計画法、非線形計画法、動的計画法、分岐限界法などの各種最適化手法を用いることも可能である。また、加工や組立作業によって部品のもつ情報量が変化することに着目し、加工・組立手順を情報量の和によって表し、情報量の最小となる手順を最適なラインとして判断する情報積算法もある。d. 検査方法の設計c項で設計された加工・組立各方法の品質保証による安定的な供給の確保と、不良品の流出防止のために、検査方法の設計が重要である。各工程では、人、設備、条件により発生するばらつきを求め、統計的に品質保証を可能とすることが求められ、工程能力管理手法によって各工程での検査方法が設計される。検査条件、全数検査/抜取り検査の選定などが行われる。生産ライン全体ではシステム的に品質保証をとらえ、それによって各工程での検査条件を求めることも重要である。部品単品では工程能力を評価し、自工程での自己完結的に品質を保証する検査方法の検討を行い、製品全体としてはFMEA/FTA手法・QAネットワーク手法によってライン全体を俯瞰した不良の流出防止のための検査工程を確立する、などの対策によって生産ライン全体での製造品質レベルを向上させ、不良品の流出防止を図る検査方法の設計が行われる。この場合でも対象製品の試作段階での工程能力や類似製品、類似工程での工程能力、品質検知能力等の実績を把握して、これらを設計に活用することが迅速かつ高信頼の設計のために必要である。図2・11でも組立工程での品質チェック頻度・時間が安定した品質を保証するための条件として決められる。e. などによる間欠・連続搬送などの工程間の搬送方法と、搬送により具現化されるバッファ機能の諸元の設計、特定アルゴリズムまたは、後補充生産方式などによる流動方式および流動ロットサイズ、段取り方法の設計、流動状態を監視するためのワーク直接識別方式、パレット情報付与方式などの情報付与方法などを設計する。レイアウトは図2・10に示した生産ラインの形態によって変化する。少種多量生産では生産工程の順に設備が配置されるラインレイアウトが用いられ、設備間には通常一方向の搬送方式が採用され、干渉を防ぐために設備間には適正規模のバッファが設置される。図を示すが、設備間のバッファ量だけでなく、適切な作業速度の異なる自動化ラインと手作業ラインの間におけるバッファ量の設計が重要であることがわかる。さらに手作業ラインを2本並列に並べるレイアウト面での設計、2ラインへの振り分けを考慮した流動制御方法などが必要であることがわかる。この設計は概要設計的にはOR的な手法など各種最適化手法が利用可能であるが、詳細な設計要素の考慮が難しく、生産ラインシミュレーションを用いることが増えている。多種少量生産では、設備群を機種ごとにセルとして集約して配置する機種別レイアウトが用いられる。この場合はセルのグループ分けが重要であり、セル内では工程間在庫が非常に少ない状態での作業が行われ、むしろ作業性を高めるレイアウト、搬送方法、流動方法の設計が重要となる。図2・13に組立ラインにおけるセルの構成を示す。セル型方式は人のもつ知能的な側面、フレキシビリティ、7―15図2・12 生産ラインの評価上記設計活動を通じて設計された生産ラインを評価し、その設計品質を明らかにする。まず、生産ラインの評価はシステムとしての性能が評価される。量的能力が確保されていること、十分なライン稼働率が得られていることが重要である。加えて、単一工程の正味稼働時間、ライン内7―16生産システム工学在庫量、ラインの入口から出口までのリードタイムについても評価することが重要である。製品寿命の短い製品では工程内在庫量を最小にする評価も重要となる。次いで投資の妥当性について経済性の面から評価されなければならない。図2・15に製品コスト最小化の考え方を示すが、生産量、製品種類だけでなく、製品種類、製品自体の構造に起因する作りやすさ/作りにくさ、製品寿命など多くの評価項目があり、設計条件によって判断の優先順位を決めてヒューイスティックに目標コストを達成する設計値を決めなければならない。決定された設計諸元での経済性は、生産ラインの操業度に対する余裕度を損益分岐点分析により評価したり、経営的な視点から設備投資の妥当性について設備経済計算によって評価される。生産ラインの評価の視点は多様であり、総合的な評価が必要である。生産ラインで確保される品質水準については、d項で述べたQA ネットワークなどにより不良品が顧客に流出する確率が最小になっているかどうかの評価が必要となる。最近は生産ラインの環境評価も求められ始めており、生産ラインの環境負荷評価(LCA :life cycle assessment)が行われる場合も出始めている。生産ラインの作業性に関する評価も安全基準等の遵守だけでなく、エルゴノミクスによる評価が必要である。以上のように多様な評価基準の多様な視点からなる評価項目・指標について優先順位を決めながらヒューイスティックに結果を求め、生産ラインの諸元が決定される。多項目の評価指標を一元的に評価し、最適化する取組みもいくつか試みられているが、ベンチマーク評価としての利用の範囲にとどまっているのが実態と思われる。
