# 7・4・3 リサイクル設計（design for recycling)

> **章**: 第7章 環境と生産システム
> **節**: 7・4 環境に配慮した生産システム設計

科学技術によって大量生産を行い，それによって多くの資源を消
費した前世紀には，廃棄物も多量に排出した．廃棄物は焼却，埋立
てを主とし，資源回収は一部であった．その反省から，現在では，
循環型社会の形成と推進が世界的な環境課題と認識されている．
使用後の廃棄・再資源化は，ライフサイクルアセスメント上では
使用段階，原料・製造段階の負荷に比較すれば，大きな数値になら
ない場合が多い．しかし，資源循環型製品を世に送り出すガイドを
もつ，あるいはリサイクル実務を義務化された製造事業者も徐々に
増えている．環境負荷指数の小さいことを理由に取組みを避けるこ
とはできない．むしろ，リサイクル設計に積極的な取組みをしなけ
れば，購入者の理解は得られない ．
ここでは，わが国の先行となった大型の家電品を事例に，リサイ
クル設計とリサイクル技術を述べ，他の製品の参考としたい．
a.背景 消費後の大型家電品の廃棄に際し，一般廃棄物の処
理責任のある自治体では，大きい，重たい，複雑な構造と成分を有
し処分しにくいとの声を1970年代から上げていた．1998年に公布
された特定家庭用機器再商品化法（通称：家電リサイクル法）は，
製造業者等の競争原理のもとで再資源化の実務を行うようにしたも
のである．政令指定の使用済み家電品から部品および材料を分離
し，製品の原材料または部品として利用（有償または無償で売却）
することを製造業者等に，使用済みの家電品を引取る義務を小売業
者に，適正な引渡しと費用の支払いを排出者に，それぞれ義務付け
た．直接的には販売済みの大型家電品の資源リサイクル，間接的に
はこれからのリサイクル設計を推進する意図があり，2001年4月
から本格施行となった．
廃棄物削減と再資源化，排出者の費用負担による排出抑制という
原則は，早晩，わが国の多くの量産品には共通するようになると考
えたい．また，ヨーロッパでもリサイクル設計と環境影響物質の忌
避への要請は強く，環境配慮は生産システム設計の重要な項目とな
る．
b.リサイクル設計 生産を行う前に，この製品が生産・流
通・使用・廃棄・再資源化の各段階における安全や資源・環境への
影響を調査，予測，評価し，必要に応じて製品設計や生産方法の変
更を行って影響の軽減を図るという，製品アセスメントが一般化し
てきた．設計の意味するところは広く，予備設計，基本設計，詳細
設計という区分，あるいは構造設計，機能設計という区分もある．
部品に材質表示があり，見やすいか．破砕した際に，材質別に分か
これらの中に，共通してリサイクル設計を取込む．生産段階は7･
4･2項で扱ったので，ここでは流通段階，使用段階，廃棄・再資源
化段階に注目したい ．
流通段階は，包装の簡易化，再資源化しやすい包装材料の選定，
包装の再資源化や処分に関する表示が挙げられる．さらに細かくい
えば，再生資源としての利用の容易性，材質別の分離の容易性，材
質の表示，再生材料の利用度，収集・運搬の容易性，焼却や埋立て
の環境保全性，包装資材の削減などについて評価する．量産試作品
ができ上がってからの包装設計は，適切なものとならない．製品設
計時に，包装までも併せて検討することが双方の改善につながり，
寸法・質量とも過剰とならない．
使用段階は，消費電力や待機電力を減らす省エネルギー，消耗部
材や流し捨てる水を減らす省資源，修理しやすい構造，長期使用に
つながる部品の共通化，適切な使い方を勧める取扱説明書や本体表
示などの評価を行う．環境負荷が特に大きいのは使用中であり，消
費電力や待機電力を減らす設計がたいせつである．製品を省エネル
ギーに設計し，さらに使用者が適切な使い方をするよう，取扱説明
書で訴える．省資源についても，使い方によるところが大きい．修
理については，ブロック化した部品の一つが故障し，ブロック全体
を廃棄してしまうのは資源のむだとなる．故障部品の取出し・交換
のしやすさだけでなく，標準化による補修部品の共通化と供給，確
かな修理技術と即応体制など，本体だけに限らない課題がある．
廃棄・再資源化段階では，使用者には適正排出（不法投棄しな
い）を取扱説明書で訴え，本体そのものに関して設計者は次のよう
な項目を考慮しておかなければならない．
.分離・分解の容易化 高価な部品，破砕できない剛体，
リサイクルの後工程に支障のある部品や環境影響物質などの特定の
部品は，手解体で取出しやすいか．部品点数は減らしたか．異種材
料の接合はないか．ねじの数，寸法，種類は多くないか．
.材質別の選別の容易化 手解体する際に，プラスチック
れやすい構造にしてあるか．材質の統一をしたか．
.再利用（中古）部品，再生材料としての利用性 中古の
部品や材料として利用できるか，あるいはそのようなものを使って
製造できるか．
.破砕・選別の容易化 せん断破砕あるいは衝撃破砕する
とき，設備に損傷や障害を与えないか．所定の寸法に破砕されて，
選別工程で品位良く選別されるか．材質の種類を減らしたか．
.焼却・埋立ての安全性 ダストや再資源化できない混合
物は，有害な物質を含んでいないか．焼却や埋立てをして，有害な
物質を作らないか．
.収集・運搬の容易化 使用済みになってからリサイクル
工程に搬入するまでに，運びやすくなっているか．大きい機器には
取っ手，車輪などがあるか．荷重は偏っていないか．内部から油や
ガスが漏出しないか．輸送の衝撃で爆発や爆縮はしないか．
.体積の減容化，質量の減量化 資源の節約となる小型・
軽量化は可能か．
c .リサイクル技術 使用済み機器から資源回収する現状技術
を知ることは，適切なリサイクル設計に役立つ．自動車，大型家電
品などは，元の質量に対する回収資源の率あるいは売却できた資源
の率といった基準がある．基準の上乗せ見直し，使用済みの他の機
う．
器へのリサイクル基準の設定は今後に予想される．5年，10年後に
適用されると仮定すれば，現在の設計・生産品が優れたリサイクル
性をもっていなければならないということである．
大きい対象機器から重たい金属のみを回収するだけであれば，大
きな破砕機と磁力選別すればよい．ところが，多くの種類の金属や
サ
プラスチック，潤滑油，冷媒ガス，剛体，ガラス，磁性材料，爆発
性の粉末やガス，有害な水銀や鉛などを含んだ材料や構造の複雑な
機器が多い．例えば，家電品は構成材料がかなり複雑で，かつ小さ
い部品の組合せである．粒度にかまわず磁力選別してしまえば，異
なる材質の破砕片が多く，質の良い資源として回収はできない．混
在物の少ない資源を回収し，高く売却する（収入），埋立てや焼却
するダストを減量（支出）することで，リサイクルコストの改善に
つなげる ．
図7･17家電品のリサイクル事例（破砕・選別工程)
.処理基準 家電品の場合は，再商品化という言葉には次
の定義が使われる．①機械器具が廃棄物となったものから部品およ
び材料を分離し，自らこれを製品の部品または原材料として利用す
る行為，②機械器具が廃棄物となったものから部品および材料を分
離し，これを製品の部品または原材料として利用する者に有償また
は無償で譲渡しうる状態にする行為である．製品の部品または原材
料として分離して，これを燃料として利用することは，当初は再商
品化に含まない．また，製造業者等は引取った機器について，少な
くとも以下の基準以上の再商品化等を実施することとなっている．

冷蔵庫，冷凍庫：50%，洗濯機：50 %

の，製品は現在のプロセスでこの基準を満たす素材構成にしなけれ
ばならない．他の機器にこのような法的基準はないが，回収率を課
す仕組みがあるものとした取組みが望まれる．

義務付けている．これらの回収は，破砕前の手分解段階となるだろ
.リサイクルプロセス（recycle process ) 手分解と破砕
後の分別について，家電・OA機器を事例に記す．
手分解の対象は，次のものである．
(１)破砕機にかけられない剛体部品，例えばモータやコンプレッ
(２)そのまま資源として売れる部品，例えばブラウン管，モータ
(３)機械設備に有害なもの，例えば磁石，油，塩水，ガラス
(４)影響物質として事前取出しするもの，例えば冷媒フロン，
油，大きい基板
OA機器ではメーカに返す部品も手解体の対象であるが，家電品
では部品の再使用の市場性はなく，将来のプロセスといえるだろ
う．
(１) 手解体部分の具体的な説明
電源コードの切断．専用の回収機で冷媒と冷凍機油を回収．カッ
タでコンプレッサを切り離す．モータの取外し．ブラウン管を外
し，真空の解除．偏向ヨーク，プリント基板，配線の束等の取出
し．磁性のある部品を外す．ガラス製部品等の取外し．排水ホース
の切断．液体バランサから塩水の抜取り．所要時間の多少がリサイ
クルコストに大きく影響する．
(２) 破砕・分別工程
手分解後の破砕・分別工程を図7･17で説明する．衝撃式破砕機
へ投入して破砕片としたあと，磁力選別や比重選別などの組合せで
品位の良い資源とする．まず，機器の中・外壁に残ったじんあいや
破砕で発生する粉じんの類は，風力で集じん機に集める．破砕の際
にさまざまなサイズの破片ができ，磁力選別機で大きい鉄片を回
収，残りの破砕片は分級装置（回転ふるい）で四つの大きさに分類
する．サイズ別に振り落とした小さいサイズの破砕片どうしを比重
選別機にかけて銅を分離，さらに渦電流選別機でアルミニウムを分
離する．中サイズ，大サイズの破砕片もこのようなサイズ別の分離
工程を重ねると，物性の違いから比較的品位良く材質別に分けられ
る．
さまざまな種類が混在し，かつ，細い銅線等がからまったプラス
チック破砕片は，さらに微細化して，塩化ビニルを取り除き，金属
を減らし，高炉還元剤として資源化できる．
(３) 設計へのリサイクル情報
従来は事業者が異なり，時間的な差があった設計業務とリサイク
ル実務は，現在ではかなり近くなったし，同一の事業者が行うこと
もある．設計した結果（試作品または量産品）をリサイクルプラン
トに持ち込めば，解体しやすいか，資源がどのくらい回収できる
か，費用がどのくらいかかるかといったデータも得られる．安易な
設計は将来へのつけとなる．今後，社内では設計審査が以前に増し
て厳しくなり，リサイクルの実態を知った設計者でなければならな
い時代となる ．
