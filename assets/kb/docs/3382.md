# 2・2・4 生産ライン評価手法

> **章**: 第2章 設計・評価技術
> **節**: 2・19 に示す手順のように、相互関係、スペース、調整という三つ

設計された生産ラインを評価する手法について概説する。a. 設備投資(investment for equipment)の経済性計算生産ラインの投資が合理的かどうかを判定する手法はいくつかある。投資規模の大小等によって用いる計算方法を決められる。. 投資利益率法(ROI: return on investment)将来のキャッシュインフローすべての現在価値をキャッシュアウトフローの現在価値と等しくする。すなわち正味現在価値を零にする割引率(利子率)を求め、これを投資利益率として評価する。. 原価比較法いずれかの案を採用すれば他の案はすべて棄却されるような、相互に排反的な諸案の年平均総費用を比較して、年平均総費用最小値の設備投資案を選択する方法。この方法は図2・20に示すように操業費用の年々の増加を考慮できるため設備更新や増産投資判断に用いられる。. 正味現在価値法(NEV : net present value)承認しうる希望最低投資利益率(資本コスト)によって、将来のキャッシュインフローすべての現在価値とキャッシュアウトフローの現在価値との差額である正味現在価値を求める方法。これはキャッシュフローの現在価値と初期投資額との差額を求めることと同じである。. 回収期間法(CRP: capital recovery period)初期投資額と、対象ラインの年々のキャッシュインフローの現在価値の累計値とが等しくなる年数で求められる、資本回収期間で評価する。b. 損益分岐点(break-even point)分析図2・21に示すように、操業度を横軸に取り、縦軸に売上高、総原価、利益を示したもので、生産ラインでの原価-操業度-利益関係を示し、これを分析していく評価手法。生産ラインで複数製品が流動するプロダクトミックスによって、操業度を高めて収益性を高めるなどの分析を行える。c. ライフサイクルコスト(LCC: life cycle cost)製品、生産ライン、設備の開発・設計・導入・操業・保全・支援・廃却の一生涯の中で発生する総コストをライフサイクルコストという。LCCの内容は、設備(システム)や資材の設計、製作、導入を要するコスト、すなわちイニシャルコスト(AC、取得コスト)と、運転、保全、廃棄に要するコスト、すなわちランニングコスト(OC、維持コスト)、および解体、運搬、保管等の廃棄に関するコスト、すなわちディスポーザルコスト(DC、廃棄コスト)の総和で把握される。LCC=AC+OC+DC設備(システム)や資材の経済性を高めるためにLCCを最小化することが行われる。そのためには、設備(システム)の開発や取得の段階でLCCを評価し、ACとOCおよびDCのトレードオフの中で最適な設備(システム)を追求することが必要になる。LCCを最小にする設備の設計案を立案していくうえで鍵を握るのは、設備(システム)のユーザ(製造部門)の現場における経験を設備の開発・設計にいかに生かすことができるか、という点である。設備の信頼性、安全性、保全性、良品製造条件などに関する設備設計に織り込むべき内容の情報(MP情報)を、設備(システム)のユーザ(製造部門)から組織の壁を乗り越えて、メーカあるいは設備の開発・設計部門へ上手にフィードバックすることが要になる。設備(システム)が一生涯の中で生み出す利益をライフサイクルプロフィット(LCP: life cycle profit)と呼ぶ。LCPを最大にする設備(システム)を目指す考え方もこれからは必要になってくる。LCPを追求するということは、設備が作り出す製品需要の量変動を考慮して、売り逃がしや過剰能力を避けることができる、量変動適応型の設備・システムを追求することである。量変動適応型の設備・システムの基本的な考え方は、固定費型と変動費型の手段をミックスして対応することに求められる。さらに、客の真のニーズを掘り起こしてそれに的確に応じるために、製品と生産システム(設備)を一体化して開発し、製品とその図2・20 年平均費用と最適使用年数図2・21 損益分岐点評価7―19供給のパフォーマンスを商品とする考え方も登場してくることになる。
