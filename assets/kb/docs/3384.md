.リユーザブル技術（parts reuse technology) 回収した
製品を資源として，これらをリユース可能とする技術全般がリユー
ザブル技術である．
リユーザブル技術には，次のようなものがある．
(１) 洗浄技術 機械フレームや板金部品などの汚れを取り去
る（図7･28は，大物板金類用のブラスト洗浄装置である)．
(２) 選別技術 余寿命が十分にあり，ワンモアの使用が可能
な部品とリユースできないスペックアウト部品をより分ける．
(３) 修理技術 操作パネル部や電源ユニットなどを修理し，
ワンモアの使用を保証する．
修理や選別などの技術は新造部品による製造には必要ないが，循
環型生産システムでは必要不可欠な技術である．
.部品リユース設計（parts reuse design) 選別や修理な
どのリユーザブル技術を不要とし，かつ，部品をリユース可能とす
るのが部品リユース設計である．リユース不可原因を層別すると，
もともと寿命が短い，ダメージ・汚れや傷等による機能劣化，非共
通という三つに大きく分類される．これらリユース不可原因に対
し，部品の再使用を可能とする設計法を開発していくことが非常に
重要である．部品リユース設計には，長寿命化設計（long-life
design)，分離設計（separable design)，強度設計，分解設計など
がある（図7･29)．
寿命が長いと何回も使えるので，長寿命化設計が部品リユース設
計の基本となる．長寿命化設計は，リユースを前提にした設計仕様
（動作条件や負荷条件など）を定めて，その仕様を満足するように
部品を設計する．長寿命化が困難な部品は，摩耗部品や短寿命部品
だけを簡単に交換できるように分離設計を行う（図7･30は，分離
設計事例のキャスタである)．
部品リユースが可能か否かは，ほとんど設計段階で決まってしま
う．設計者がいかに知恵を出し，どうすればリユースが可能となる
かを考えていかねばならない．
.再資源化設計 リユースできない部品や寿命まで使い切
ったリユース部品も貴重な資源である．これらの部品は，再資源化
して材料に戻さなくてはならない．材料リサイクルとしての活用を
考える前に，グレードを落とさない材料リユースとしての活用（外
装カバーを外装カバーに再生するなど）を考えることも必要であ
る．
再資源化設計で取組むべきことは，
(１) 解体・分別容易化 解体しやすく，素材別に分別しやす
くする．
(２) 有害化学物質の削減 環境に悪影響を及ぼす有害な化学
物質（鉛，ハロゲン系難燃剤など）を使用しない．
(３) 再資源の容易化 再資源化に適した材料を使用する．
などである．
c .循環型生産システムによる新しいものづくり 複写機が市
場から回収される時点では，後継機が生産・販売されており，世代
を超えたリユース部品の活用が重要である．同一機種だけの部品リ
ユースから，2世代，3世代めの機種へのリユース部品投入を実現
してこそ，循環型生産システムのメリットが生じる．製品の企画段
階において，後継機が市場や顧客要求でグレードアップしなければ
ならないもの（性能や機能を向上させるユニット/部品）と，変更
の必要がないものとに二分化して，製品の開発を進めることが特に
重要である．すでに，3世代くらいの製品を共通とするプラットフ
ォーム戦略により，部品リユースを前提とした製品企画（product
planning for parts reuse）が行われている．また，回収した製品
から取り外し，リユーザブル技術によってリユース可能となった部
品，あらかじめ部品リユース設計された部品を，異なるタイプの新
製品に投入することが始まっている．そして，再資源化設計を行う
ことで，廃棄ゼロを達成できる．さらに，部品リユースや再資源化
が重要だと認識されるにつれ共通化も進む．共通化が進めば，調達
コストも下がる可能性がある．そして，長寿命設計や徹底的に寿命
を管理することにより，製品の信頼性は向上する．分離設計や分解
設計をすることで保全性が改善される．このように，循環型生産シ
ステムは，トータルコストダウンに貢献するのみならず，世代を超
えた新しいもの作りを可能とし，大量消費・使い捨て社会を変革す
る．
