# 2・3・2 工場内物流の機能

> **章**: 第2章 設計・評価技術
> **節**: 2・3 物流システム計画手法

a. 保管機能供給の変動に需要の変動が同期しない場合に,保管という機能が存在する.部品や素材を生産ラインが要求するときに即時提供できれば,また市場に対して生産品の供給が即応できれば保管機能は減少するが,季節性などによる市場の変動,調達タイミングとコスト変動のずれ,故障などによる生産機械ライン間の同期のずれ,当然管理の不手際による保管なども含め最小限の在庫の確保は必要となる.こうした状況でも保管,在庫に対する機能の変化は図2・23のように起こっており,また保管用の機器も単に保管するだけではなく自動倉庫の活用に見られるように,仕分け,搬送機能も併せてもたせるシステムが増えている.このため保管設備の計画に当たっては,ユニットロード,ボリューム算定などだけでなく保管機械の特徴を生かした計画を行うことが重要になっている.b. 搬送機能搬送装置は,連続搬送を行うコンベヤ系のもの図2・22 製造業の物流領域図2・23 保管機能の変化7―20生産システム工学と間欠搬送の台車系のものに区別される.特に最近の生産ラインの頻繁な変更に対応できる無人搬送車(AGV)の導入が多くなっている.加工や組立といった直接生産活動に対して素材や部品を供給したり,あるいは工場へ搬入されてきたものの搬送,製品の搬送等,工場内に多数存在する工程間には必ず搬送行為が存在する.この搬送行為はかつては有人で行っていたが,省力化,重労働からの解放,あるいは搬送スピード,搬送環境の確保といった観点から自動化手段をとることが多くなっている.搬送は必然的に生じる工程であるがしかし,省略,縮小可能なものはできるだけ排除すべきである.このため搬送方法を決定するとき以下の観点から検討する必要がある.・搬送距離が最小になること工場内のレイアウト,工程設計に依存・搬送回数を最小にすること搬送単位,工程設計に依存・搬送速度の最適化搬送機器能力,搬送方法,搬送経路の状況に依存・ライン変更への容易な対応搬送路のフレキシビリティc. 仕分けその他工場内では,物流機能として保管,搬送以外に分岐,合流,仕分け,移載,梱包,検索等さまざまな機能が要求される.特に最近ではこれらの機能は,保管機能よりも重要な要素となることもある.また,これらの機能が保管や搬送機能と連携しながら機能を果たすことも多くなっている.生産ラインに対する部品や材料の供給は,タイムリーに必要な単位(量,種類とも)での供給が求められている.このため生産の前段階として,ラインに必要な部品をそろえ,キッティングを行い,配膳する必要がある.これらの行為も加工や組立の活動とは別に物流の機能として行われる.最近ではこれらの機能を工場外に置いてラインに必要な状況で工場へ入庫させることも行われている.また,こうした仕分け機能は伝票や指示書を元に人手に頼っていたが,最近の情報技術を活用したID タグ等により,より細かく精度の高いかつ高速な仕分けが可能となっている.生産計画に基づいた入庫や配膳が生産設備からの供給要求とリア・物流品目の確認・物流量の確認・物流方式の確認・工程間の物量算定・物流機器の台数・能力選定・搬送手段の検討・運搬強度の検討・搬送先の決定・作業スペース・保管方式(対象物と保管庫)・入出庫方式・情報システムアーキテクチャ表2・8 レイアウトの基礎要素P:製品,材料Q:量R:経路S:補助サービスT:時間図2・24 計画のための確認事項ルタイムに結びつき,必要なものを必要なとき供給することができるようになってきた.
