# 2・4・1 工場に求められる要件

> **章**: 第2章 設計・評価技術
> **節**: 2・4 工場計画手法

生産活動を考える場合、機械と人がいれば生産活動ができた時代とは異なり、図2・37に示すようにさまざまな機能が融合して成り立っている。その生産活動に重要な要素の一つとなっているのが工場施設である。特に工場施設は、人や生産機械に最適な環境を提供するだけではなく地域社会に対する環境影響といった面でも重要となっている。かつての雨露をしのげばよかった時代から、最近では生産活動に必要な高度な環境条件を提供するようになり、生産活動の成否を大きく左右するようになっている。また、工場施設への投資は生産活動全体投資額のかなりの部分を占めるようになり、製造業の事業計画に多大な影響を与える。従来は工場施設と生産設備は別個に計画し、与えられた空間に生産機械等の生産機能を埋めていくといったやり方が多かった。しかし、施設を含めさまざまな機能の融合が求められるとともに、競争が激化してビジネススピードが求められる時代、いかに早く生産活動を立上げることができるかが最重要課題の一つとなっている。現在では生産活動の計画において施設計画をコンカレントに行うことは、全体の投資対効果、生産の立上げまでのリードタイムの短縮に大きく影響する。図2・38に、生産活動に求められる機能と、それに関連して工場施設に求められる機能をまとめた。計画の各ステップにおいて企画書や設計書としてまとめ、経営者や社内関連部署、場合によっては社外関連企業のレビュー、確認を受ける必要がある。これらの計画は段階が進めば進むほど詳細の検討が必要になるが、これらの検討結果が建設の主旨に合っているか絶えずチェックを行いながら進める。また、各ステップでの確認が不十分だと計画が進んだ段階で大きな手直しが起こり、計画の工程に大きな遅延を生じることがあるため、絶えず初期のコンセプトから逸脱しないようにチェックするとともに、他のシステムと同期を取りながらスパイラル状に計画を進めていく必要がある。実際には、各機能の各ス図2・37 機能関連図図2・38 工場施設に求められる機能7―24生産システム工学テップが同時に進行したり、フィードバックを繰返しながら計画が進行することになる。こうした各計画実行のための機能をエンジニアリングプロセス(engineering process)と呼び、図2・39に示すような計画のプロセス(エンジニアリングプロセス)の進行が行われる。
