# 2・4・2 計画の手順

> **章**: 第2章 設計・評価技術
> **節**: 2・4 工場計画手法

計画段階の重要性については、2・4・1項で述べたが、こうした計画段階もその内容によりいくつかのステップに分けることができ図2・39エンジニアリングプロセス図2・40 計画のステップ図2・41 計画手順のフローる。概念的な計画からより詳細な計画へ、ただし計画のスタートに当たっては全体構想や事業計画、基本方針から与条件が設定され、それに基づいて計画が進められることになる。実際の計画時には各機能の各ステップが同時に進行したり、フィードバックを繰返しながら進行することになる。こうした工場計画の計画ステップを図各ステップは、シーケンシャルに進められるわけではなく、特に工場立上げまでの期間が短い場合はある部分並行して作業が進められる。しかし大事なことは、各ステップの区切りでのレビューと評価を関係者の目で行い合意を取付けておくことである。各計画段階での作業項目やアウトプットを、施設計画と生産システム計画の二つの視点からまとめた計画手順の全体フローを図2・41に示す。この二つ以外にも、物流、情報システム、運営組織等があり、これらの機能が計画の中で相互に関連する。a. 企画設計段階事業構想や工場建設事業の方針決定によって事業計画を策定し、この基本方針から建設規模、施設の概要・構成、概略予算を決め、配置図、平面、断面、設備概要等の施設概要ができる。しかし、この段階ではすべての与条件が確定しているわけではなく、かなり変動要因を含んだ設計となる。b. 基本設計段階企画段階のアウトプットを元に生産ラインや機械レイアウト、生産動線や物流機能を考慮して建設計画をさらに詳細に詰めていく。建築材料や強度設備のフロー等についても検討を行って検討項目一つ一つの条件を見直し、より精度の高い仕様にシェイプアップしていく。設備はシステムの容量、規格、供給方法等についても建築計画と合わせて進めていく。また、工場立地法などさまざまな法規制も与条件として考慮し、計画に織り込んでいく。当然こうした各段階で見積り予算との整合を取っていく。c. 実施段階基本設計書に対し、具体的な詳細スペックを決定する。詳細な設計図書が完成し、見積り、金額の決定が行われる。当然、細部の見直しを機能面、予算面で行うことにもなる。同時に確認申請や次ステップの施工のための計画も行われる。
