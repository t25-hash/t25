# 7・1・3 計測機器の校正とその周期

> **章**: 第7章 計測における管理と教育
> **節**: 7・1 計測機器の管理

品質に影響を与える測定，検査および試験のための装置は，設備
する際，およびその後定期的に，あるいは使用する前に校正すべき
である．使用される標準器は校正の不確かさがわかっている適正な
校正方法で行い，その校正の上位の標準器を順次たどったとき，源
が国の標準器に結びつくことが求められる．これを全うしているこ
とを計測機器のトレーサビリティ（traceability）があるという．
測定値の正確さは，測定機器または標準器のトレーサビリティ体系
に従って得られている．測定機器，標準器の種類によってトレーサ
ビリティ体系が完備されていない測定量もあり，その場合，その基
準についての十分な記録が望まれる．
わが国では，計量法にJCSS（Japan Calibration Service Sys-
tem）制度として物理量の標準器の校正体系に従って標準の値，ま
た化学量を与える標準物質の供給がなされて，測定量，試験量の正
