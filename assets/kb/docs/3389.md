# 2・4・3 工場の立地条件

> **章**: 第2章 設計・評価技術
> **節**: 2・40 に示す.

工場の敷地の選定は,工場建設と必ずしも連続性をもたない.敷地の取得からかなりの年月を経過して工場建設に着手する場合も多々ある.また既存の敷地に建設あるいは増設,改築する場合もある.しかし,工場の計画の最初のステップとして立地条件の再確認,再評価を行うことは重要となる.ここで工場立地(plant loca-tion)の選定に影響を与える要因を表2・10に挙げる.こうした選定要因は,社会経済動向,市場,企業のおかれた立場等の背景によって項目も変化するし重要度も変わってくる.いくつ表2・10 工場立地の選定要因・自然条件気象:積雪,降雨量,風向風速,雷,気象災害(台風,洪水等)環境:水質,水量,塩害,空気清浄度,その他自然有害物質・敷地条件地形:面積,高低差,傾斜,方位,形状,開発行為の有無地盤:地質,地耐力,支持地盤深度,地震,微震動,地下水位,汚染地価:・地域条件調達:原材料,部品,ユーティリティ,協力業者製品需要先:消費地,納入先人材・労働力:賃金,労働供給能力,労働の質,教育機関輸送:道路,鉄道,空港情報:通信インフラ文化レベル:医療文化施設,住宅,生活費,教育施設,イメージ・ユーティリティ条件電力:量,質(電圧変動,予備回線,停電他),負担金燃料:ガス,石油等(コスト,供給安定性,供給量)水:水質,水量,料金,負担金,地下水規制排水:負担金,水利権,下水料金,排水規制・公的条件税制:融資優遇制度:用途規制:誘致政策:公害規制:放流放出規制(放出量,放出物質,濃度)近隣対策(騒音,震動,電波障害,日影)住民対策:物流動線○△○△×△フレキシビリティ◎○◎△△×ユーティリティ動線○△○○○○事務厚生ゾーン環境◎○◎○◎◎駐車場○△○△△○増築の容易性○○○△○△眺望・景観△△△△◎○地盤◎○×△◎△来客○○○○×○人的動線○◎○○△◎総合評価7―25かの候補地がある場合,選定要因の重要度に応じて評価項目を設定して重み付けを行い,候補地の評価をすることもある.状況により,選定要因をさらに詳細まで展開して評価する必要も出てくる.こうした工場立地の候補として比較的多いのは,工業団地など地域自治体が積極的に工場誘致を推進している場所である.これらは国外の立地を含め,国や地方自治体等でさまざまな優遇措置が講じられており,投資面で優位なだけでなく,許認可の迅速な推進など,計画を推進するうえで重要な要素となる.最近のビジネススピードの向上は,特に事業の立上げの迅速性を要求されるようになっており,生産をいかに早く立上げるかが推進者の重要な課題である中で,自治体のさまざまなバックアップは不可欠となっている.また,最近海外進出が増加する中で,日本においては比較的これらの選定要因の状況をつかむことはたやすいが,国外においては仲介者の情報をうのみにし,工場建設を断念する例も聞いている.自ら十分に調査を進めて対応する必要がある.
