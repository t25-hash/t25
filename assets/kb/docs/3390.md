# 2・4・4 配置計画(zoning,ゾーニング)

> **章**: 第2章 設計・評価技術

a. 工場敷地の配置計画計画に当たっては,ゾーン区分と表案を立案し,評価比較したうえで最終案を決定する.表2・11 配置計画のチェックポイント・敷地有効利用・近隣(騒音,電波障害ほか)・物の動線・日照条件,方位・人の動線・周囲道路との出入り口・エネルギー動線・防犯上の配慮・将来への増設改修対応・防災上の配慮・気象条件・自然こう配の配慮(排水ほか)季節風,積雪ほか・地盤・外からの景観・家相検討項目重み配置計画A配置計画B配置計画C配置計画D配置計画E配置計画F図2・42 配置計画例と評価表7―26生産システム工学工場の建設地が決定すると,その与えられた敷地条件に合わせて配置計画が行われる.敷地内のレイアウトは物と人の動線,ひいては生産の効率に大きく影響するとともに,増設等将来を見据えた配置計画を行わないと余分な土地の買い増しや別の敷地の手当を余儀なくされることがある.図2・42に配置計画の六つの案とこれらを評価項目により評価表として作成した例を示す.これら評価項目やその項目の重み付けは工場の用途や場所,業種等によって大きく変化する.これらの重要度と評価も適正に設定することが計画時の重要な課題となる.こうしたゾーニング計画は,工場に関連する社内各部署,時によっては社外組織も含め,部署の利己的な思惑が働く陣取り合戦となりかねない.また,複数事業部が関連するときは事業部間の力関係がそのまま出ることも多い.計画担当者はこうしたことを避けるためにも全体のバランスを機能面,投資対効果面,将来性等を踏まえ,客観的に判断し示すことが求められる.b. 工場内の配置計画敷地のゾーニングが決定すると,次は工場内の配置計画に入る.工場内の配置計画で最優先されるのは,当然,生産の各工程に関するものとなる.部品や素材の入荷から保管,加工,組立,検査ないし搬送といった生産活動の流れ,あるいは各生産工程において必要な空間,強度等により,それぞれの生産工程の相互の関連を考慮しつつゾーニングを行う.当然これらのゾーニングに当たっては動線がシンプルかつ最短になるような考慮が必要となる.このゾーニングの例を図2・43に,またゾーニングと図2・43 工場施設内ゾーニング例図2・44 工場施設内ゾーニングと動線例動線の関連を図2・44に示す.これら各工程や機能ゾーンの必要スペースを考える場合基本となるのは生産品目の生産量,大きさ,重要と生産機械ラインの必要面積,高さ,荷重により決定される.また他の機能ゾーンは利用人員や必要機能により算出される.ただし,算出に当たっては運転保守スペースや通路などの考慮も必要であるし,将来の増設や増員に対しても考慮されなければならない.これらは当初に作成される事業計画から算定すればよい.表2・12に工場内の各ゾーンと機能を示す.階段や設備シャフトは増設時の移設が難しいため前もって考慮する必要がある.また,生産設備の配置が決まってくるとこれらに供給するエネルギー等のユーティリティ(utility)の供給ルート方法も検討項目に入る.特に電気,熱源は生産機械に依存するため,設備計画との連携も必要となる.こうした生産設備との整合のために,工場施設の計画段階で,生産設備から見た空間条件や設備条件(電気容量,ユーティリティ量ほか)を生産設備計画と並行して検討し,双方とも仕様の変更に対して相互に吸収しながら計画を進めることが必要である.図2・45に,生産システムのシミュレーション(simulation)の機能の一つとして,従来生産能力算出が主体であったシミュレーション機能に計画の条件として機械スペースやユーティリティ量も与条件として設定し,評価した例を示す.図2・45で生産機械の大きさや必要なユーティリティの諸元,量を入力し,図2・46のシミュレーション結果で空間やユーティリティの評価も行う.
