# 7・2・2 教育・訓練のマニュアルの必要性とその基本

> **章**: 第7章 計測における管理と教育
> **節**: 7・2 計測技術者の教育・訓練

測定技術の向上を意図的に行うには，一定の測定教育（mea-
surement education）と測定訓練（measurement training）とが
必要である．計測に携わる人間の能力は人間の諸特性によって多岐
にわたり，千差万別であるといえる．その中にあって，測定者を意
図するレベルに到達させるためには，まず必要としている意識をも
たせることが必要である．その後，マニュアルに従って教育・訓練
を行う．マニュアル（manual ）は，企業の目的とそれに基づく品
質の概念によって，品質管理，計測管理の一つとして認められた，
その企業における規則に該当するものであることが望ましい ．
測定は製品の良否を判別する手段であるばかりでなく，工程の分
析，不良原因の探求や改善を行うために不可欠な手段である．正し
い測定装置の正しい操作による正しい判定が行われなければ，品質
の保証はできない．測定機器は適切な計測管理のもとに正しく機能
するもので，7･1節に述べたように明文化し，教育を行い，円滑に
運営しなければならない．マニュアルは現場作業標準との関係を考
慮して作られなければならないし，全社的教育の一環として位置付
ける必要がある．測定者教育・訓練のためのマニュアル作成におけ
る基本的方針としては，以下のことを考慮しなくてはならない．
(１)企業の計測技術の向上，測定技術者の技術の向上を意義ある
ものとして認識する
(２)企業の教育システムの一環として，適切な位置付けを行う
(３)企業における測定者の役割の認識，ならびに社内における確
認をする
(４)測定者の人としての役割と従来のシステムの改善との関係に
ついて配慮する
(５)教育・訓練の内容が，その人にふさわしい測定能力の要求水
準に基づき，どの程度の品質判定のばらつきに抑えるかを決
め，その目標を達成せしめる
計測の教育・訓練のマニュアルの一例としてそのおもな項目を挙
げてみると，ほぼ次のようなことになる．
(１)マニュアルの適用範囲
(２)用語とその定義
(３)測定の重要性
(４)教育・訓練の目標
(５)教育・訓練の条件
(ａ) 取引・契約書等の条件・問題等
(ｂ) 測定操作員の条件・能力
(ｃ) 関連規格
(ｄ) 環境条件
(ｅ) 測定法，試験法
(ｆ) 測定値の処理法
(ｇ) 測定の信頼性
(ｈ) 誤差，不確かさの概念と算出法
(６)測定検査と処理
(ａ) 自社製品の品質特性の概要
(ｂ) 品質保証のステップ
(ｃ) 品質特性の測定と工程能力
(ｄ) 検査の実施，検査手順，検査基準
(ｅ)統計的方法
(ｆ)事例
(７)教育の訓練の方法
(８)教育・訓練の評価
