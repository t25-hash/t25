# 2・4・5 建築設備計画

> **章**: 第2章 設計・評価技術
> **節**: 2・11 に示す配置計画のチェックポイントを基に,いくつかの計画

工場に求められる設備は,一般の事務所と違ってさまざまなもの表2・12 工場内のゾーニングと機能・生産ゾーン:加工,組立,塗装,検査・物流ゾーン:入出荷,梱包,倉庫,マテハン通路・事務厚生ゾーン:事務室,食堂,休憩室,会議室・サービスゾーン:電気室,機械室,コンピュータルーム・その他:通路,階段,エレベータ,設備シャフト図2・45 条件入力シート図2・46 空間評価シミュレーション例が要求される.また,その量も多く,要求基準も高度なものが多くなっている.これは生産環境の要求事項が高度になり生産機械の独自性から,オフィスビルでは温湿度の環境ぐらいであるが,クリーン度や振動,電磁環境,音などその生産品目や生産機械の特殊性により要求される質が異なる.一般のビルと同様に建物の環境維持のために必要なものとして,・一般電気設備・一般空調設備・一般給排水衛生設備が挙げられるが,最近の省エネルギーのニーズからさまざまな省エネルギー機器が活用されるようになっている.その他生産設備に供給するものとして,・生産設備用電源・圧縮空気・真空設備・純水設備・生産用冷却水設備・ガス設備(酸素,窒素,特殊ガス)・特殊排気設備・クリーンエアの他にさまざまなものがあるが,これら特殊なもの,特に危険物等は配置計画上自由にはできず,大きな制約があるので注意が必要となる.
