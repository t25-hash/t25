# 2・4・6 関連法規

> **章**: 第2章 設計・評価技術
> **節**: 2・11 に示す配置計画のチェックポイントを基に、いくつかの計画

建築物にはさまざまな法規制が加えられている。特に工場に対しては廃棄物、騒音、危険物の使用等一般の建物とは違い、環境に影響を与える要因が多種多様にわたるためさまざまな法規が関連する。これら関連法規を図2・47に示す(環境関連を除く)。これら関連法規を計画段階で十分に考慮するとともに早い段階での官公庁、近隣との折衝を行い、工場稼働予定どおりに進むようにする必要がある。これら関連法規には立地地域により県条例や地域独特のもの(瀬戸内海環境保全特別措置法など)が存在するため、立地選定時や計画段階で十分に考慮し選定基準の評価にする一方、工場誘致による緩和規定や助成制度等についてもよく調べておく必要がある。こうした関連法規の中で、特に工場建設に関連する三つの法規について述べる。とりわけ環境問題は地域性が高く、国の法規制に地方公共団体の条例による規制が加わるため注意が必要である。a. 工場立地法(factory location law)工場立地法の対象工場は敷地面積9000m以上、または建築総面積3000m以上が対象となる。当然敷地の増加や増築により、上記条件を超えた場合も届け出が必要となる。新設や変更の実行は、この届け出受理後90日をみておく必要がある。また、地区により緑地面積はまちまちとなる。b. 建築基準法(building standard law)工場施設に対しての建築規制の主たるもので、建築主は工事着工前にその建築計画が法令に適合しているかの確認を行うため、地方公共団体の建築主事に提出し承認を受ける必要がある。このとき提出するものとして、・開発行為または建築についての許可に関する証明書・建物の設計図書(図面および書類)が必要で、当然ながらこのとき消防法にも適合するものでなくてはならない。申請から許可まで約1.5~2箇月程度みておく必要がある。当然この許可が下りるまでは着工はできない。c. 環境基本法(basic environment law)1970年代ごろから当時の環境保全活動の高まりとともに、各種の環境基準や公害に対する法の整備がなされてきた。これらの関連法規を図2・48に示す。特に工場はさまざまな環境破壊物質を排出してきた歴史的経緯があり、地域住民や社会の厳しい目にさらされているといってよい。このため、これらの法規を遵守することは当然のことであるが、生産活動という事業を行うためには、効率の良い投資でこれらへの対策を行う必要がある。対策方法は、生産システムだけではなく工場計7―27・建築基準法・道路関係法・駐車場法・消防法・自然環境保全法・屋外広告物条例・危険物の規制に対する法律・中高層建築物指導要領・ビル管理法・電波法・省エネ法・耐震改修促進法環境・環境影響評価条例(アセスメント)開発・都市計画法・宅地造成等規制法建築全般敷地立地・農地法・工場立地法・高圧ガス取締法・水道法・電気事業法・ガス事業法・薬事法・食品衛生法・火薬類取締法・クレーン等安全規則・ゴンドラ安全規則・労働安全衛生法・冷凍保安規則・容器保安規則・液化石油ガス保安規則・一般高圧ガス保安規則・事業付属寄宿舎規定・ボイラおよび圧力容器安全規則・砂防法・砂防法・森林法・海岸法・鳥獣保護法・航空法・臨海地図における建築物等の規制・特定空港周辺航空機騒音対策特別措置法・石油コンビナート等災害防止法・地滑り等防止法・文化財保護法事業関係地域図2・47 工場建設に関わる関連法規図2・48 環境関連法規画時の立地や配置計画、設備計画により左右される。また、環境関連は県などの地方自治体が細かく条例や要項指導で規定しているために、立地の状況にあった対応が必要となる。
