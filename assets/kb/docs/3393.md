# 2・5・1 バーチャルファクトリー(virtual factory)

> **章**: 第2章 設計・評価技術
> **節**: 2・5 生産システムのモデル化とシミュレーション

近年の製造業においては、急激に変化する市場動向に追随して、厳しい競争に生き残るために市場が要求する製品をタイミングよく製造し、供給する体制を整備することが不可欠となってきている。このため、製品の開発・設計・製造の効率化はもちろんとして、部品の調達から生産、さらに製品の各種営業拠点・顧客への物流などの一連の活動をサプライチェーンとしてとらえ、全体を最適化するマネジメントの必要性が高まっている。このような背景のもとに、近年の高度な情報技術を活用することによって製品に関する情報を共有し、製品の開発から製造までの諸活動を多面的にかつ並行的に遂行することにより、高い生産性を実現することが考えられている。各活動においては図2・49に示すようなさまざまなシミュレーション技術が利用されており、それらを7―28生産システム工学図2・49バーチャルファクトリーとさまざまなシミュレーション有効活用する環境としてバーチャルファクトリーあるいはディジタルファクトリー(digital factory)と呼ばれるディジタル空間における製品設計・製造環境の実現が図られている。製品の開発・設計段階ではCAD/CAM/CAEとして有限要素法などに基づく数理モデルを用いた大規模な数値計算シミュレーションが行われる。また、製造プロセスの計画や生産準備段階においては、NCプログラム、ロボット動作、作業者動作など加工性や組立性、人間工学に基づく作業性の検証などを行うジオメトリックなシミュレーションとともに、効率の良い生産ラインの開発・設計を目指して離散型シミュレーション(discrete event simulation)が広く適用されている。また、離散型シミュレーションは個別製品の生産ラインを対象とするばかりでなく、多くの製品を生産する製造工場の計画・改善やその運用においても利用され、その有用性は広く認識されている。なお、図には示していないが、経営や営業活動においてはビジネスゲームやシステムダイナミクスなどのシミュレーションも活用されている。術の中で、生産ラインや工場の新規開発設計や改善計画の評価のために広く用いられている離散型シミュレーションを中心に解説する。
