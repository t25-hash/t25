# 2・5・2 シミュレーション(simulation)とは

> **章**: 第2章 設計・評価技術
> **節**: 2・5 節においては、上記のようなさまざまなシミュレーション技

自然界や人間社会におけるさまざまな現象やシステムの振舞いを解明するためには、現実の世界における振舞いを直接観察するばかりでなく、図2・50に示すように実世界を抽象化、簡略化した仮想世界を構築し、さまざまな解析を可能とすることが考えられている。仮想世界は解析対象に焦点を絞って実世界と同等な振舞いを再現できるようにしたものであり、モデル(model)と呼ばれる。理論的な数式モデル、風洞モデルなどのような実体モデル、あるいはコンピュータシミュレーションモデルとして実現される。シミュレーションは、対象とする現象やシステムの空間的、時間的な変化を逐次的に観察できる仕組みを仮想世界、すなわちシミュレーションモデルとして構築し、変化のありさまを追跡し、対象の特性を解析できるようにするものである。したがって、シミュレーションモデルの構成やパラメータをさまざまに変更することにより、モデルの特性がどのように変化するかを明らかにすることができる。さまざまな解析を容易に行えるようにするためには、できるだけモデルを簡略化することが望ましいが、対象とする現象の特性が的確に把握できる程度に詳細にわたるものでなければならない。システムが大規模、複雑化し、さらにその詳細な特性の解明が必要になるにつれて、その必要度に応じた適切なモデルを比較的容易に構築することができるコンピュータシミュレーションの有用性が広く認められてきている。
