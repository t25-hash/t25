# 7・3・2 計量単位 改正法における計量単位に関する規制の概

> **章**: 第7章 計量に関する法制度
> **節**:  

要は，次のとおりである．
① 計量単位は，「取引又は証明」において使用する計量単位につい
て，法律で定められた計量単位以外のものは使用してはならない．
②この「法律で定める計量単位（法定計量単位）」については，国
際的に合意された「国際単位系（SI）」による．
法定計量単位には，国際単位系（SI）に係る計量単位（表7・1），
SI単位のない量の非SI単位（表7・2）及びSI単位のある量の非
SI単位（表7・3）がある．
③ 改正前の法では，SI単位とともに非SI単位の使用が認められ
ていたが，一定（3年，5年，7年）の猶予期間を置いて，順次
