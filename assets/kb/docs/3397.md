# 2・5・5 製造業の情報化と分散型シミュレーション

> **章**: 第2章 設計・評価技術
> **節**: 2・5 節においては、上記のようなさまざまなシミュレーション技

図2･49に示したようなバーチャルファクトリーの実現は製品開
発環境の整備を推進するものであり，離散型シミュレーションは製
造ラインの開発に活用されている．また，製造部門においても，各
生産ショップなどにおいてはシステムの改善や効率的スケジューリ
ングのためにシミュレーションが用いられていることはすでに述べ
た．企業内外における情報ネットワークの整備が進み，高速・大量
の情報流通を活用した意思決定の重要性が高まっていることから，
従来からのシミュレーションの応用に加えて，例えば，以下のよう
な意思決定支援にシミュレーションを効率的に適用できる環境を整
備することが望まれている．
(１) サプライチェーンにおいて，要求された納期に間に合う製
品を要求された価格で提供できるか．
(２) 新生産ラインを導入する場合，工場内の他の設備との能力
的バランスや操業面から見て運用上の問題はないか．
(３) あるエリアで作成したスケジュール，あるいは故障等によ
り生じたリスケジュールは，前後の加工エリアにどのような影響を
及ぼすか．
(４) 新製品に対する生産ライン群を複数の企業に発注した．開
発された個別システムは，全体システムとして整合が取れている
か．
