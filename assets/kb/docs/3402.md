# 7・6・3 計量器の販売 法は，一定の特定計量器について，そ

> **章**: 第7章 計量に関する法制度
> **節**: 7・6 計量器の修理

の販売の事業を行おうとする者に対して届出を義務付けている．
販売の事業については，届出の対象は政令で定める特定計量器に
限られており，販売事業者が遵守すべき事項が定められ，その不履
行に対して都道府県知事の勧告・公表・命令がなされることとされ
ている．そして，販売の事業は，一定の確認，説明義務等さえ果た
せば，広く一般人であっても販売事業を行うことができるのである。
なお，販売の事業の届出の対象となる特定計量器は，施行令第13
条で次のものが定められている．
①非自動はかり（ヘルスメータ，ベビースケール・キッチンスケ
ールを除く），分銅・重もり
②ガラス製体温計
③抵抗体温計（電子体温計）
④アネロイド型血圧計
なお，輸出のための販売についての届出は含まれない．
