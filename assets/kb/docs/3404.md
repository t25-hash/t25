# 8・1・2 技術基準の定義 工業規格が品質，経済効果，安全，

> **章**: 第7章 計量に関する法制度
> **節**: 8・1 規格・技術基準

環境配慮など一般的な規定を行うのに対して，安全性，環境などに
重点を置いて，守らなければならない技術レベルを定めたものが技
術基準である．そのため，人が介在する装置，製品，建造物，施設
などについて，関係法規が技術基準を遵守することを義務付け，許
認可時の拠り所となる場合が多い．
なお，1SO／IEC Guide 2では，実施基準の定義が次のように規
定されており，技術基準に近い．
「設備，構造物又1ま製品の，設計，据付け，保全又は使用のための
方法論は手順を勧告する文書．」
備考 実施基準は，規格又は規格の一部門あることもあるし，ま
た，規格と無関係の独自のものであることもある．
