# 3・1・4 管理システムにおける諸管理活動の役割とそれらの関

> **章**: 第3章 では生産システムを所与とし,それを運用する管理システ
> **節**: 3・1 生産システム管理の体系

連性生産管理システムを効果的に運用するためには,さまざまな管理活動の実施が必要である.ここでは,管理活動として・生産計画の立案・作業管理・工程管理・購買・外注・在庫管理・品質管理・設備管理・原価管理を取上げ,生産管理システムにおけるそれぞれの役割を述べるとともに,それらがいかなる部門の活動として,また複数の部門がどのように連携して実施されるかを説明する.連携の仕方は企業によって,言い換えればその企業が用いる管理システムによって異なる.以下においては,今後製造業において一般的になると考えられる生産スケジューリング中心の管理システムの利用を前提として述べることにする.a. 生産計画の立案基本的には顧客の注文に基づいて生産計画が生産管理部門によって立案される.顧客は海外の販売代理店のような大口の発注を行うものから特殊な仕様の製品を少量発注する小口のユーザまでさまざまなものがある.注文の内容は,製品の仕様,数量,価格,納期がおもなものである.仕様によっては設計が必要であり,この場合は販売部門と設計部門が顧客と折衝しながら,仕様の概要と納期が決定される.生産計画の主要な内容は設備や作業員の稼働計画つまり生産スケジュールであり,これに基づいて顧客の要求する納期の実現性が検討され,必要に応じて販売部門を通して納期や生産量の調整が顧客との間で行われる.生産スケジュールの細部については製造部門の要求を考慮して決定する.資材の調達に時間を要する場合には,生産スケジュールの作成と並行して資材調達の可能性が検討され,生産スケジュールが修正されることがある.つまり,その際には購買部門が生産スケジュールの作成に係わる.b. 作業管理生産計画,特に生産スケジュールの作成に当たって,加工時間,段取時間のデータが必要であり,機械加工の場合は,マシニングタイム,組立作業の場合は標準時間が用いられる.これらのデータの作成,メンテナンスは,通常,生産技術部門によって行われる.BOM やM-BOM を新たに開発する場合は,設計部門と生産技術部門が連携してそれに当たる必要が生じる.作業の差立てや組立ラインの編成は製造現場つまり製造部門によって行われ,残業の計画やシフト数増減は製造部門,生産管理部門,工場の総務部門がその決定に関与する.c. 工程管理生産ラインの設計を含むことがあるが,本稿では生産システムは所与としているため,工程管理の主要な活動としては,生産スケジュールが予定どおり行われるように監督し,遅延が生じた場合に必要な対策を講じる進捗管理が挙げられる.また,ジグや工具が複数の工程で共通して利用される場合に,生産スケジュールが支障なく実行されるように必要なジグ・工具をタイミング良く各工程に提供することも工程管理の重要な活動である.通常,工程管理は製造部門によって行われる.d. 購買・外注管理生産スケジューリング中心の生産管理ではスケジュールの作成は,生産管理部門が中心になって行われるが,購買部門が加わって購買あるいは外注先に発注する資材の必要量・時期が決められる.したがって,購買・外注管理の主要な仕事は必要な種類の資材を,必要な量だけ,必要な時期に調達できるように,発注先の進捗状況を監督し,必要であれば督促を行うというものになる.このような監督・督促の業務は,通常,購買部門によって行われる.外注管理の中に外注先に対する生産技術や品質に関する指導が含まれることがあるが,この種の業務は生産技術部門や品質管理部門によって行われる.e. 在庫管理製品が完成後ただちに出荷される場合,工場においては製品の在庫管理は行われない.ただし,ここでいう在庫管理は在庫水準に基づいて発注量や発注時期を決定する狭義の在庫管理を指している.製品の在庫管理はむしろ流通経路中のものが中心になり,製造業が管理する場合には生産管理部門が係わる必要がある.近年,サードパーティロジスティクスが普及し,流通経路中に7―39ある製品の現物管理は外部の業者に委託することが多くなっている.したがって,製造業が行う前述したような狭義の意味での在庫管理は,原材料の一部や常備品がその主要対象となっており,他方,多くの資材は生産管理システムが保有するメカニズム(例えば,かんばん方式)によって発注量や発注時期が決定される.通常,狭義の在庫管理は購買部門の活動の一部として行われる.f. 品質管理生産スケジューリング中心の生産管理システムでは,特定の顧客オーダに対して要求量どおりの生産が行われるため,不良品の発生を極限まで抑えねばならない.資材の受入検査を受入工場側で実施する場合は,通常,品質管理部門が行う.工程間検査や製品検査は作業員が行うことが一般的であり,工程に異常が認められた場合,品質管理部門が中心になって,生産技術部門,製造部門が協力して原因を明らかにし,その対策が講じられる.g. 設備管理生産スケジューリング中心の生産管理システムでは,生産設備の使用日程はその稼働可能性を前提としたものであるため,設備故障による生産の遅延は許されない.設備管理は,通常,設備の更新と保全の活動を意味しており,近年,生産管理においては日常的な設備保全の重要性が増大している.特に,予防保全が事前に定められた日程に従って計画的に行われ,保全日程が生産日程の妨げにならないように,生産管理部門と設備管理担当部門との間で調整が行われる必要がある.h. 原価管理現代の生産管理では全体最適の観点に立った収益性の極大化が望まれる.そのためには,製造原価が的確かつスピーディに把握でき,製造工程の収益に対する貢献の程度を示した日々の付加価値の算定ができる必要がある.これは,原価管理の方式の問題にとどまらず,生産管理のあり方に係わった問題であるといえる.つまり,生産管理の仕組み自体が,購入資材の総額=消費資材の総額,出荷高=販売高の関係を指向するものであることが必要であり,それをより良く実現しうる生産管理システムのもとにすべての管理活動が統合され,原価管理活動はすべての活動の成果を適切かつ不偏的な財務的尺度によって評価する役割を担うものでなければならない.さらに,これらの評価値の変動が生産管理にフィードバックされることが重要であり,そのためには生産管理部門,原価管理部門を中心とした各部門の密接な連携と協調が望まれる.
