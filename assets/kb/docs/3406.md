# 3・2・1 生産計画(production planning)の目的

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

生産活動の最も重要な目的は,高品質の製品を市場に供給すること,顧客に対して約束の納期を守ること,同時に最小のコストでこれらのことを実行すること,である.この目的の達成のために,生産管理者(オペレーションズマネージャ)は,以下のような生産計画活動に携わることが必須になる.(1) 長期の計画期間(6~18箇月)に対して,総合的な生産能力計画を作成し,製品やサービスに対する顧客需要を満足する生産資源を確保すること.(2) 顧客納期を満足するスケジュールを作成し,生産コストが最小になるように,生産設備にジョブ(仕事)の割当を行うこと.(3) 部品や原材料の調達,保管,出庫の計画を作成し,必要な部材を生産スケジュールの適切な時期に供給すること.(4) 組織が顧客納期の遵守,目標在庫水準の維持,最小生産コストの確保に向かって機能できるような優れた生産管理システムを構築し,これを運営すること.
