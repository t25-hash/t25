# 7・1・3 配管

> **章**: 第7章 配管要素
> **節**: 7・1 管と配管

配管（piping）は主として流体の輸送を目的として，家庭用の上これらを配管の計画図，設計図，系統図に示すための基本的な図示系の流体とその状況を明確にするため，流体の種類に応じた色分けと取扱い作業誤りのないようにしている．
ガスケットには，小さな力で大きな変形が得られ，接合面に伴う取替えがあるので，接合面からのはがれやすさも要求されれる（3･5･2項参照）．
配管の支持（pipe support）に当たっては，配管の質量支持，熱流体振動，地震，風などによる配管の移動や振動の制限などの使用a.固定式管支え（固定管支え，固定管継手) 固定管支えはが多用される．
図Ⅰ-7･4は固定式の管継手の例で，このほかに固示す．
響を無視できる水平配管ではUボルトつりでよいが，温度の影響管つりか，ころ・V溝などで案内する管支え法か，支えの弾性変Ⅰ-7･6に管つり法と，可動管支えの例を示すとともに，図Ⅰ-7･7に適する）を示す．
下水，ガス，電気配線から産業設備まで多用される．
構成要素にジ（pipe flanges ；ガスケットも含む），配管支持部材などがある．
記号はJIS Z8205で規定している．
また，施設，装置などの配管による識別（JIS Z9102）や，名札，バンドを用いて安全性確保配管の接合面には流体の漏れ止めを目的とするガスケットが使用になじみやすく，外力が除かれれば原形にもどりやすく，かつ保守る．
膨張による配管の自由な動きを拘束または制限，さらに機械振動，目的に応じて管支えがなされ，固定式または可動式が選ばれる．
管を壁，柱などの構造物に固定する方法で，管支えを兼ねた管継手定式のバルブもある．
図Ⅰ-7･5に，これらを用いた配管法の例をb.可動式管支え（管つり，可動支え，引張り棒) 温度の影を無視できない場合には，配管の伸縮を拘束しないように可動式の形を利用する可動管支えや引張り棒式管支え法が適用される．
