# 3・2・4 生産能力/設備計画(capacity planning)

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

この計画は生産能力の大きさと種類を決定するためのものである。これには当然、部品やコンポーネントや最終製品を作る機械が中心であるが、これらの機械を支援するために必要な装置類、例えば、電力設備、空気/水/薬品供給装置、スペア在庫、マテリアルハンドリングシステムなども計画される。生産能力/設備計画の作成に当たって考慮すべき要素に次のものがある。a. 製品のライフタイム設置すべき設備のタイプや大きさは、生産する製品のライフタイムに依存する。近年、製品のライフタイムは年々短くなるばかりで、設備のライフタイムが製品のライフタイムよりも長い場合が多く、その場合、製品の生産中止によって残存設備能力がむだになることがしばしばである。したがって、設備は将来的に他製品への転用が可能なフレキシブル性を有することが望ましい。b. 部品の内作/購入設備計画は部品を内作するか、購入(外注も含む)するかによって大きく変わる。このことからも、部品の内作/購入の決定は重要である。そのため、以下のような考慮が必要になる。(1) 内作か購入かは単にコストだけでは決められない。アウトソーシングによって、これまで培われてきた生産技術や技能が失われるだけでなく、使っていた設備がむだになってしまう。当然、このような損失も考慮されなければならない。(2) また、アウトソーシングを考える場合は、長期的展望に立った判断がなされなければならない。生産を放棄して流通/サービスに移行することは必ずしも悪いことではないが、アウトソーシング市場が将来どのように変化するかを長期的展望のもとに把握しておくことが重要である。(3) 内作か購入かは、その製品全体を対象とする決定であれば明解であるが、その製品を構成する一部の部品だけをアウトソーシングする場合は面倒になる。それは、内作か購入かの決定の際に、工場のワークロード(負荷)を考慮しなければならないからである。どの製品のどの部品をアウトソーシングするかによって工場負荷が変わり、工場負荷の平準化に変化が生じることになる。c. 価格一般に、製品価格は生産能力/設備計画においてはあまり問題とされない。それは工場計画に対する影響が直接的でないからである。しかし、生産能力を決定する場合には価格予測ないしは経済分析が欠かせない。なぜならば、ある特定の設備構成が経済的に妥当であるか否かは、その製品の売上高から求められる利益で評価されるからである。d. 資金の時間的価値設備投資や設備改善は、通常、資金の時間的変化を考慮してなされる。したがって、利率や減価償却期間は設備選択に大きな影響を与える。e. アベイラビリティいうまでもなく、生産能力は次式で与えられるアベイラビリティによって決まる。また、アベイラビリティは設備のある一定の信頼性を確保するためにも重要な尺度となる。f. ボトルネックOPT(後述)の考え方にもあるように、ボトルネックになっている生産設備を増やせばスループット(生産量)は増すが、非ボトルネックの生産設備を増やしてもスループットは増えない。この理論からいえば、高速高性能の機械をボトルネック工程に投入することは非常に魅力的であるが、現実にはボトルネック工程がそれほど明確に識別できるわけでもないし、また、生産品種や需要状況によってもボトルネック工程は変化するので、問題はそれほど単純ではない。g. 変動(ばらつき)この影響はボトルネックほど注目されていないが、機械の変動、すなわちばらつき、例えば、機械故障による生産停止は、スループットにかなりの影響を与える。特に、ボトルネック工程の変動は、そのスループットに与える影響が大である。
