# 3・2・5 労働力計画

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

労働力計画は生産を直接実行する労働力の大きさを決定する計画である.労働者を対象とした計画であるから,ある程度長期にわたって安定的に労働力を確保するにはどうすればよいかを計画するのが,労働力計画の主たる目的である.しかし,実際には,企業の労働契約(例えば,雇用/解雇に関する協定や就業規則など)に基づいて,どのようなタスク(課業)に,どれだけの労働力を割当てればよいかを決定するのが主題であり,そのために,通常,必要労働者数よりもむしろ必要労働時間を計画の尺度とする場合が多い.必要労働時間の算定のベースとなるのが標準労働時間である.例えば,家庭用空調機の組立に1台当たりの標準時間が30分(1/2時間)かかるものとし,1週間の組立工1人当たりの労働時間を35時間とすると,1人の組立工が1週間に組立できる空調機の台数は35×2=70台である.生産計画が350台の空調機の組立を要求しているならば,必要とする組立工の数は5人である.このように,標準労働時間が労働力計画作成の基礎となるが,このとき,次のような問題が介在することを忘れてはならない.a. 労働力利用可能性標準労働時間を推定するときには,労働者の病欠,休暇,訓練などによって利用可能な実労働時間が失われることを考慮しなければならない.b. 労働力安定性生産調整によって生産量のアップ/ダウンはしばしばあるが,労働力の調整は雇用/解雇につながるため,頻繁かつ安易に行うことは慎まなければならない.技能レベルの高い労働力を安定的に確保することが,その企業の実力に関係する.c. 従業員訓練労働者を雇用すれば必ず訓練が必要である.訓練にはコストと時間がかかる.労働力と設備との違いは,労働力は教育・訓練によって技能が向上し,生産性向上につながるが,設備力は設置後,摩耗や劣化等によって性能が漸次低下し,それに伴って生産性も低下することである.したがって,労働者に対する教育・訓練を恒常的に実施することが重要となる.d. 労働力柔軟性労働力は設備力よりも柔軟性がある.短期の急な需要変動やプロダクトミックス(製品混合)の変化に瞬時に対応できる(ただし,当然,適用範囲や能力には限界がある).一方,長期的に見ても,現代のように,製品や工程が絶えず変化する生産環境にあっては柔軟かつ適応的に行動するシステム,すなわちアジャイルマニュファクチャリング(agile manufacturing)が要求されるが,そのシステムの中心が人間としての労働者である.e. 品質改善品質改善には種々の要因が関わるが,その中心的役割をなすのは労働者であるから,労働者に対する品質教育や多技能教育が品質改善を作り出す原動力となる.しかし,現実問題として,このような品質改善を労働力計画の中で考慮することは非常に難しい.なぜならば,品質改善教育によってたちまち効果が現れるものではないからである.しかし,品質改善による影響が顕著に認識される場合には,できるだけそれを考慮する必要がある.7―41
