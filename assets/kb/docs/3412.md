# 3・2・7 マスタプロダクションスケジュール(MPS : master

> **章**: 第3章 管理システム
> **節**: 3・2 生産計画

production schedule)a. マスタプロダクションスケジュールの目的マスタプロダクションスケジュール(以下MPS と呼ぶ)は短期計画の基準をなす重要な計画で,短期計画期間(例えば,3箇月)の各週ごとに,各エンドアイテム(end item)に対して計画される.エンドアイテムには最終製品として顧客に出荷されるものや,在庫品として保管されるものがある.MPS の作成担当者は需要予測,顧客オーダ,在庫水準,設備負荷状況などの情報を基にマスタプロダクションスケジュールを作成する.そして,これを基準にして,エンドアイテムを構成する部品や原材料に対するスケジュールを作成する.b. マスタプロダクションスケジュールのタイムフェンスMPS では計画期間が四つのセクションに分けられる.各セクショ図3・4 工場負荷表図3・5 MPS 作成手順ンを分ける境界をタイムフェンスと呼ぶ.第1セクションをフローズンと呼び,最初の2~3週間がこれに該当し,この期間のスケジュールは凍結される.第2セクションをファームと呼び,この期間のスケジュールは固定化の対象となる.第3セクションをフルと呼び,この期間のジョブは前詰め方式で割当てられる.最後の第4セクションはオープンと呼び,この期間のジョブには決まったスケジュールがない.新規オーダは通常この期間に割当てられる.図3・4にこれらの関係を示す.c. マスタプロダクションスケジュールの作成手順MPS の作成手順を図3・5に示す.顧客オーダ,予測需要量,在庫状況レポートおよび生産能力情報に基づき,まず,緊急オーダをオープン期間に割当てる.そして,各週ごとに各ワークセンタの生産能力に対して納期遅れや設備遊休が起こらないよう配慮しながら予測需要量を割当てる.この割当問題に対しては,オペレーションズリサーチの分野において種々の手法が開発されている.d. 需要管理生産スケジュールを実効あるものにするためには,方法論自体が優れていることはもちろんのこと,生産システムを取巻く環境をいかに管理するかが重要である.例えば,ジョブショップシステムを製品のグループ化によってセルという形にもっていけば,流れ生産システムの構築が可能となり,一般ジョブショップシステムよりも管理が容易になる.このように,生産環境を管理するうえで最も効果が上がるのが需要管理(demand management)である.これは積極的に顧客需要を組合・調整するもので,これによって生産管理が容易になる.例えば,JIT(just in time)においては,生産の平準化を図るために,製品混合,オーダエントリー,製品のグループ化などが積極的に行われる.
