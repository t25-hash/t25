# 7・2・4 フランジ式管継手

> **章**: 第7章 配管要素
> **節**: 7・2 管継手

管フランジ（pipe flange）は図Ⅰ-7･9に示すように，ガスケッ
トをフランジにはさみ，ボルト・ナットで締付けて接続する管継手
である．フランジの管への取付け方によって図Ⅰ-7･10のように分
類される．図Ⅰ-7･10で(ａ)は弁や機器と一体に作られるもの，図
Ⅰ-7･10(ｃ)には，ろう付けによるものも含まれる．管フランジの
規格化されているものは図Ⅰ-7･10(ｂ)と図Ⅰ-7･10(ｃ)だけである
が，互換性に影響する寸法は呼び径・呼び圧力ごとに基準寸法とし
て規格化されている．また管フランジは用途によって，一般用
（JIS B2210，2220，2238，2239，2240），水道用（JIS G5527)，
油圧用（JIS B2291)，冷凍用（JIS B8602)，真空用（JIS B
2290)，舶用（JIS F7804，7805）に分類することもできる．
JISの一般用管フランジの呼び圧力に対する流体の状態と最高使
用圧力との関係（pressure -temperature rating）に関して，鉄鋼
製管フランジについてはJIS B2238，2239，銅合金製については
JIS B2240に規定されている．
