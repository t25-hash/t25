# 3・3・1 作業管理の成り立ち

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

メイナード(Maynard)は,作業管理の内容を次の4項目にまとめている.1作業改善2作業方法,設備,治工具,環境の標準化3標準時間の設定4標準の維持すなわち,生産システムの諸要素である人・物・設備・治工具・土地・建物および情報などを有効に活用すること,換言すれば,有形・無形の製品・サービスをアウトプットする生産システムを構成する作業者や機械・設備の稼働状況,材料・仕掛品・製品の流れを把握して合目的的であるかの判断を行い,いわゆるムリ・ムラ・ムダのないシステムを構築するためのエンジニアリングアプローチをいう.これは,後述のIE(industrial engineering)の基礎ともなっている.作業管理の成り立ちを示すと図3・11のようになる.作業測定(work measurement)とは,標準の概念にかなう作業者が,標準的な作業を行う場合の標準時間と標準作業量を決めるための測定・分析(measurement)を意味しており,すなわち,標準時間(standard time)の設定にかかわる手法,具体的には時間研究がこれにあたる.方法研究(method engineering)は標準的な方法の設計と,設定にかかわる手法とその実践である.このように測定分析対象が方法(method)にまで広がったことより,作業研究(work study)といわれるようになった.方法設計(method design)とは,作業測定,方法研究・作業研究を踏まえて,システム全体が経済的にそして効率良くその目的を果たすように,個々の投入資源および複数資源の有機的つながりを設計することである.以上の四つの考え方を融合し,生産システムの設計から評価に至るプロセスを合理的に管理しようとするものが作業管理であり,それがインダストリアルエンジニアリングの根幹をなす.
