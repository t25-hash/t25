# 3・3・2 作業管理のアプローチ

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

すでに対象となるシステムが稼働状態にある場合は問題発見,改善活動といういわゆる分析的アプローチをとり,一方,新規にシス図3・11 作業研究の体系図3・12 分析的/デザイン的アプローチ7―45テムを設計する場合には,デザイン的アプローチをとることが一般的である.図3・12に両アプローチの違いを示した.前者は現状の制約条件のもとで問題を発見しボトムアップの形で改善を行うのに対し,後者は制約を考えない理想状態からFS(feasibilitystudy)を行うことが特色である.なお,制約条件の把握の観点,また,代替案の評価項目として次のものがよく用いられる.P : productivity(生産性)Q : quality(品質)C : cost(コスト)D : delivery(納期)S : safety(安全性)M : morale(モラル)
