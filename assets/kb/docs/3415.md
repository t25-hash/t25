# 7・1・1 離散事象システムの特徴

> **章**: 第7章 離散事象システム
> **節**: 7・1 システム表現

なお，論理モデルで扱われる時間は一般には決定性の時間である．
ステムの性能（スループット，反応時間，稼働率など）を定量的に評価するために用いるモデル．
事象が生起するまでの遅れ時間を確率変数として与えるなど，事象生起のタイミングは確率的にモデル化される．
以下の特徴をもつシステムのことである．
離散的状態 システムのとりうる状態の集合が離散集合として与えられる．
次のような場合がある．
各状態変数のとりうる値が離散値であり，それらの値の組合せによりシステムの状態が定義される場合（例：スイッチが入っている/切られている，機械が稼働中/待機中，バッファ内の部品の個数)．
