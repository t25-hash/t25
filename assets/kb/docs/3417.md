# 3・3・4 IE の発展

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

広義のIE(industrial engineering)はワットの蒸気機関の発明,ホイットニーによる互換性部品の思想に始まり,図3・13に示すような発展をたどっている.1927~1932年にかけてウエスタンエレクトリック社のイリノイ工場で行われたメイヨー(Mayo)らによるホーソン工場における実験は,最大の生産性を確保するための最適な物理環境の探索として行われたものであったが,作業者の心理的・社会学的要因という人的誘因が生産性に大きく寄与することが判明.また,バーナード,マズロー(Maslow)らの研究は作業者の作業に対する態度と労働意欲に関する理解を深め,マグレガー(McGregor)は人間の労働に対する基本的な考え方であるX 理論,Y 理論を提唱,その後の目標管理(MOB : management by objectives)の理論的な根拠となり,人間関係研究がIEの中に位置付けられた.第二次世界大戦の折,安定した品質を有する製品の供給を目的としたQC(quality control),混乱した状況のもとで人員,軍需品,食料などの資材を迅速かつ必要な量,必要なときに配達する必要に迫られて発達したOR(operations research),人間工学などの思想がIEに加わって新たな展開を見せている.その発展系譜と周辺技術について,図3・13に示した.IEに関する多くの定義の中で,最もよく用いられているものが1955年に制定されたAIIE(American Institute of IndustrialEngineers)による次の定義である.IEとは,人・物・設備を総合したシステムの設計・改善・確立に関する活動であり,そのシステムから得られる結果を明示し,予測し,評価するために,工学的分析・設計の原理・方法とともに,数学,物理学および社会科学の専門知識と経験を利用する.う.図3・13 IEの発展系譜ここでのキーワードは工学(engineering)であり,これは工学/技術審議会(ABET : Accreditation Board for Engineering andTechnology)によって,人類の利益のために自然の物質と諸力を経済的に利用する方法を開発するために,研究,経験および実践から得られた数理化学と自然科学の知識を評価し適用する専門領域と定義されている.経済的,数理科学と自然科学,評価して適用,この三つがポイントである.IEをこれらの言葉を用いて整理すると次のようになる.IEは能率的な作業を容易ならしめるために人・物・設備・情報からなるシステムを設計,改善・再設計するための工学的知識をはじめとするあらゆる知識を評価し適用すること.すなわち,作業管理そのものといえよう.
