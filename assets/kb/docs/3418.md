# 3・3・5 作業管理の評価指標

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理の評価指標にP,Q,C,D,S,M があることは前述した。また近年これらの評価項目に加えて、CS : customer satisfaction(顧客満足度)ES : employee satisfaction(従業員満足度)SS : society satisfaction(社会的満足度)ES : environment satisfaction (環境満足度)という観点も重要となっている。作業管理の目的は生産性、利益の追求であるが、決して、人間性、安全性そして経済性という観点を忘れてはならない。ここでは、主たる評価指標である生産性概念について示す。生産性=アウトプット/インプット×100%=生産数量/投入資源材料設備・方法人エネルギー土地・建物時間それぞれの投入資源に対して次のような生産性指標がある。作業管理を行ううえで、対象職場・システムの特色を十分に把握し、どの指標を用いるかの判断が重要となる。材料:歩留り、良品率設備・方法:設備生産性人:労働生産性エネルギー:原単位土地・建物:m当たり生産性時間:時間当たり生産性
