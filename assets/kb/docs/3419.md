# 3・3・6 IE 手法

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理に活用することのできるIE手法の代表として、材料から製品への時系列変化をとらえる工程分析、レイアウト図面上に人・物の動線を描き運搬およびレイアウト改善の手掛かりを得る流れ線図、D・I(distance×intensity)分析、人・機械の稼働状況を把握するための連続稼働分析、ワークサンプリング、複式活動分析、標準時間設定のための時間研究、人の動作に着目するサーブリック、PTS(predetermined time standard)法等がある。いずれも、稼働:付加価値を生じる作業準稼働:稼働をサポートするが本来の作業ではない非稼働:付加価値を生じない作業という観点より分析を行い、非稼働は排除、準稼働はシステムの変更で改善するという方策をとる。
