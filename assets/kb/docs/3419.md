# 3・3・6 IE 手法

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理に活用することのできるIE手法の代表として，材料か
ら製品への時系列変化をとらえる工程分析，レイアウト図面上に
人・物の動線を描き運搬およびレイアウト改善の手掛かりを得る流
れ線図，D・I （distance×intensity）分析，人・機械の稼働状況
を把握するための連続稼働分析，ワークサンプリング，複式活動分
析，標準時間設定のための時間研究，人の動作に着目するサーブリ
ック，PTS（predetermined time standard）法等がある．いずれ
も，
稼働：付加価値を生じる作業
準稼働：稼働をサポートするが本来の作業ではない
非稼働：付加価値を生じない作業
という観点より分析を行い，非稼働は排除，準稼働はシステムの変
更で改善するという方策をとる．
3・3・7作業管理の業務範囲の拡大
作業管理の対象である作業は，当初は有形な物の生産活動に直接
かかわる人の作業がおもな範囲であったが，生産工程の自動化，ロ
ボットの導入などによりマンマシンのインタフェースの管理，また
情報との一元化など，その範囲は広がっている．
生産の場は，顧客ニーズに対応した多品種少量・変種変量生産へ
の移行を余儀なくされており，それに伴う生産管理手法，在庫管理
手法も変化してきている．これらも作業管理のための要素となろ
う．
