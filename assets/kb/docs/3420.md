# 3・3・7 作業管理の業務範囲の拡大

> **章**: 第3章 管理システム
> **節**: 3・3 作業管理

作業管理の対象である作業は、当初は有形な物の生産活動に直接かかわる人の作業がおもな範囲であったが、生産工程の自動化、ロボットの導入などによりマンマシンのインタフェースの管理、また情報との一元化など、その範囲は広がっている。生産の場は、顧客ニーズに対応した多品種少量・変種変量生産への移行を余儀なくされており、それに伴う生産管理手法、在庫管理手法も変化してきている。これらも作業管理のための要素となろ図3・14 生産準備段階の作業管理対象また、グローバル生産の進展に伴い、開発・設計過程のみを国内に残し生産を海外で行うファブレス化も進展している。これに伴い図3・14に示した生産活動以前、すなわち生産準備の諸活動の作業管理も重要となってきている。納期の管理にPERT(program evaluation and review tech-nique)、CPM(critical path method)、品質の管理に各種管理図、統計手法、FMEA などが、また、製品設計にはVE(value engi-neering)が用いられている。これらも作業管理の有効なツールとなっている。
