# 3・4・1 生産統制とは

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

生産を実施する準備段階において，いかに生産計画を詳細に立て
ても，時間の経過とともに計画と実際に食違いが生じる．これは生
産計画が将来の生産活動に対する計画であり，多くの要因が不確定
要素をもっているからである．すなわち，欠勤，不良発生，機械故
障などの事故発生，むだな受注，標準時間の不正確さ，現有の生産
能力の把握不十分などの計画の不正確さ，顧客からの設計変更や納
期変更，飛込みなどが発生する．そこで生産計画で決められた線に
沿って日々の作業を実施させるためには，作業の実施状況をタイミ
ングよくチェックし，計画と実績とに差があるときには，その原因
を突き止め，それを取り除くためのアクションをとったり，計画を
修正したりする必要がある．生産統制（production control）は
日々の管理ともいわれ，日常の作業において作業状況を把握しなが
らできるだけ計画に近づけるため調整することである．生産統制は
生産の時間管理であり，納期の遵守がその最大の目的となるが，シ
ョップフロア（生産現場）での仕掛在庫の低減，フロータイム（シ
ョップフロアにオーダをリリースしてから加工完了するまでの時
間）の短縮，および機械設備の稼働率の向上も重要な生産統制の役
目となる．一般に，計画と実際との差異は進度の遅れあるいは生産
数量の不足という現象になって表面化するから，進度状況の把握と
日程の維持が生産統制の中心になる．
