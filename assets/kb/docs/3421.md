# 3・4・1 生産統制とは

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

生産を実施する準備段階において、いかに生産計画を詳細に立てても、時間の経過とともに計画と実際に食違いが生じる。これは生産計画が将来の生産活動に対する計画であり、多くの要因が不確定要素をもっているからである。すなわち、欠勤、不良発生、機械故障などの事故発生、むだな受注、標準時間の不正確さ、現有の生産能力の把握不十分などの計画の不正確さ、顧客からの設計変更や納期変更、飛込みなどが発生する。そこで生産計画で決められた線に沿って日々の作業を実施させるためには、作業の実施状況をタイミングよくチェックし、計画と実績とに差があるときには、その原因を突き止め、それを取り除くためのアクションをとったり、計画を修正したりする必要がある。生産統制(production control)は日々の管理ともいわれ、日常の作業において作業状況を把握しながらできるだけ計画に近づけるため調整することである。生産統制は生産の時間管理であり、納期の遵守がその最大の目的となるが、ショップフロア(生産現場)での仕掛在庫の低減、フロータイム(ショップフロアにオーダをリリースしてから加工完了するまでの時間)の短縮、および機械設備の稼働率の向上も重要な生産統制の役目となる。一般に、計画と実際との差異は進度の遅れあるいは生産数量の不足という現象になって表面化するから、進度状況の把握と日程の維持が生産統制の中心になる。
