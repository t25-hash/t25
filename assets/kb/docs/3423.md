# 3・4・3 差異の検出と処置方法

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

a. 差異の検出生産計画と実績の差異は,注文生産においては進度の遅れ,見込生産においては生産数量の不足という現象で表面化するから,それぞれの進度あるいは生産数量を把握することによって差異を検出するのが普通である.差異を検出するに当たって,どの位置で,どのくらいの間隔で,いかなる手段を用いて差異を検出するかを設計しなければならない.検出する位置を多くし,検出間隔を短くすると差異が生じてから早い段階で処置を執ることができるが,検出のための費用が増大する.逆に検出位置が少なく,検出間隔も長いと差異の検出に遅れが生じてしまう.検出位置・間隔・手段は,注文生産と見込生産との違い,工程の性質(工程の数,合流型・分岐型工程),製造所要日数などによって異なり,一概には決められないが,生産指示する工程,生産指示間隔および生産指示段階に対応した形で差異の検出を行うのが普通である.例えば,作業伝票を用いて生産指示する場合には,指示を受けた工程あるいは職場は,作業伝票で指示された内容と生産実施状況とを対比して差異の検出を行うことになる.また,作業伝票の代わりに生産現場にコンピュータの端末器を配置し,端末器を通して生産指示をする場合には,作業完了後,作業指示番号・作業者名・工程名・完成数量などの情報を端末器に入力してセンタで迅速に差異を検出することになる.見込生産において,各工程が同期化している工程系列においては,物の流れが速く仕掛期間も短いから,工程系列の最初の工程に作業指示を行い,最後の工程で生産量を把握するだけでよい.そのため,同期化している工程系列ごとに1箇所検出位置を設け,検出間隔も必要に応じて1日複数回,生産状況を把握することも可能であり,差異の処置も迅速にできる.最近では,加工する物にバーコードを付けて,検出する工程を通過するごとにバーレコーダで即座に数量を数え上げ,予定した計画数量と比較して進捗を常時表示するシステムを用いている工場もある.他方,注文生産においては,各工程が同期化しておらず,物の流れも錯綜しているから迅速に差異を検出しようとすれば各工程ごとに検出位置を設けなければならないが,経済性の面から検出する箇所は制約される.一般には,合流工程,分岐工程,能力面での隘路工程などを選んで差異を検出する場合が多い.b. 差異の処置方法生産計画と生産実績との差異を検出したときの処置は,差異の大きさや採算工程の性質にもよるが,次のいずれかの方法で処置することになる.1次の計画で差異を修正する.2差異を検出した時点で当該機の作業を再分配する.1の方法は差異が僅少であるか生産能力の変更,あるいは材料や部品の補給が困難な場合に行われ,一般に見込注文の場合に多く用いられる.2の方法は生産能力変更に柔軟性がある注文生産の場合に多く用いられる.具体的には次のような方法がとられる.(1) 負荷が能力より大きい場合・残業,休日出勤,外注利用・余裕のある職場からの応援・作業順序の入替え・ロットサイズの変更(一般に小ロットに分割して,一部を先行,一部を後回しにする場合が多い)(2) 負荷が能力を下回る場合・予定の繰上げ・余裕のない職場への応援・ロットサイズの変更(予定を繰上げてロットサイズを大きくする場合が多い)
