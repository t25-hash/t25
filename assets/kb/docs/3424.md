# 3・4・4 トヨタ生産方式(Toyota production system)

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

トヨタ生産方式は、自動車の最終組立ライン(後工程)が必要な部品を必要なときに、必要な量だけ前工程から受取り、前工程は、後工程に引取られた分の部品を作る、という基本的な考え方を実施しようとするものである。したがって、トヨタ生産方式のねらいはこのようなジャストインタイム(JIT)生産によって、職場にひそむあらゆるむだを排除して生産性を向上することにある。JIT を確実に実施するには、工程の流れ化・同期化が必要であり、このための手段としてプルシステム、かんばんおよび生産の平準化を用いる。a. プルシステム(pull system)製造工程がいくつかの工程からなっているとき、各工程に基準在庫水準を設け、後工程が前工程(にある基準工程)から部品を引取る(引張る)と前工程は後工程で引取られた量だけさらに一つ前の工程から部品や材料を引取っ7―49て生産し、もとの基準在庫量に戻す。このようにして、各製造工程はさらにその前工程から必要な量だけ引取る形で順次前へ前へとさかのぼっていく。このような生産指示方式をプルシステムという。このような方式のもとでは、生産計画を全工程に同時に提示する必要はなく、最終組立ラインにだけ伝わればよい。b. かんばん(Kanban)かんばんは通常1枚のカードで長方形のビニール袋に入れてあり、大別すると引取りかんばんと生産指示かんばんの二つに分けることができる。引取りかんばんは後工程が前工程へ部品を引取りに行くときの請求書であり、品名・品番、コンテナの収容数、前工程の置き場所、後工程の受入れ場所などが明示されている。生産指示かんばんは品名・品番、工程名、生産指示量、完成品の置き場所が明示されている。かんばんはむだを排除するためのいろいろな機能をもっている。第一はかんばんと物とがそろっているときは生産を行わない。かんばんだけのときだけ生産を行うから余分なものを作らないことになる。第二はかんばんの枚数やかんばんに明示されている収容数を少なくすることによって仕掛在庫量を下げることになる。第三はかんばんなしの生産・運搬はないし、かんばんのない在庫はないから、かんばんは目で見る管理の手段になりうる。c. に入っているかんばんが一定枚数になると、かんばんはポストA から集められて差立板に運ばれ差立板に並べられる(加工着手前部品置き場には加工着手前の部品と引取りかんばんがコンテナに入れられている)。4加工着手前部品置き場で、差立板に並べられている順番に従って引取りかんばんと生産指示かんばんを差替えて、コンテナから部品と生産指示かんばんといっしょに後工程に流す(生産指示かんばんは後工程で加工完了後、加工済部品保管場所1に戻る)。5差替えた引取りかんばんはポストB に入れられる。6一定の時間が経過するかポストB に入っているかんばん枚数が一定枚数になると、かんばんはポストB から集められ、1枚ずつ空のコンテナに入れて、台車またはフォークリフトで前工程の加工済材料保管場所に運びコンテナに取付けられている生産指示かんばんと今運んできた引取りかんばんとを差替える(生産指示かんばんはポストCに入れられる)。d. 生産の平準化(smoothing production)生産の平準化とは多種多様な需要変動に適応するために各製品を毎日平均して生産することである。さらに、1日の生産においても各種の製品を連続して生産するのではなく、各製品の生産量が時間的に一定になるよ図3・18 かんばんの流れ7―50生産システム工学うに投入することである。いま、3種類の製品A,B,Cを1箇月間(稼働日数20日間、1日の稼働時間420時間)でそれぞれ6000台、4000台、2000台生産する場合、1日の生産量および平均サイクルタイムは表3・2のようになる。組立ラインで1台当たりの組立時間がA は0.72分、B は0.65分、Cは0.74分必要とする場合、AAAAA...BBBB...CCCCC...と同一製品を連続して流すと製品A は0.7分のサイクル時間では続行できない。ABABAC ABABAC...のような混合流れの順序で流すと、製品A や製品Cの作業遅れは製品B の段階で回復し、0.7分のサイクルタイムで生産することが可能となる。したがって、平準化のもとでは特定の製品を大きいロットサイズで流すことはしないから、日々需要変動に適応でき、過剰の在庫が発生しなくなる。また、混合流れに従って製品を流すことによって作業の負荷が平準化できることになる。トヨタ生産方式を実現するためには、次のような条件を整えておく必要がある。. 生産の平準化後工程は前工程へ、必要なものは、必要なとき、必要な量だけ引取りに行く。もし後工程が引取りに行く量や時期にばらつきがあると、前工程へ進むほど変動が大きくなり、大きな在庫と設備や労働力をもたせなければならなくなる。. 段取り替え時間の短縮品種の切換えに段取り替えが必要な工程においては、従来のロットサイズを大きくして段取り替え時間(コスト)を最小に抑えることが常識であった。しかし、トヨタ生産方式では最終組立ラインが混合流れによって生産を平準化し、かつ工程間の基準在庫量を低く抑えるから、段取り替えを頻繁にかつスピーディーに行わなければならない。
