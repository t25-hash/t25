# 3・4・5 オーダリリース方式の選択

> **章**: 第3章 管理システム
> **節**: 3・4 生産統制

受注生産においては、生産設備の高い稼働率を維持しつつ、リードタイムの短縮と納期の遵守が生産管理上の重要な課題として挙げられる。ところが、これらの製造パフォーマンスは互いにトレードオフの関係にあり、オーダのリリースによって大きく左右されている。さらに、生産システムは予期できないさまざまな変動要因の影響を受けており、それらが管理をますます難しくしている。リリースとは、あらかじめ計画オーダとして認識されている素材を生産するための権限を与えることと定義されており、そのメカニズムは、システムがオーダを認識してからショップフロアで加工完表3・2 生産の平均サイクルタイム製品月間生産計画量1日の生産台数サイクルタイムA6000台300台B4000台200台420分/600台=0.7分C2000台100台了されるまでに起こるあらゆる活動から成り立っている。受注生産においては、図3・19に示すように引合いの段階で適切な能力設定を行った後、生産着手(オーダリリース、orderrelease)を行う。生産システムの主要なパフォーマンスに影響を与えるショップフロアの仕掛在庫レベルは、このオーダリリースによって左右されるため、ショップフロアコントロールを効率良く運営する必要がある。また、製造のパフォーマンスを高めるためには、このオーダリリースのメカニズムを解明する必要がある。オーダリリースにおいて決定される内容は二つある。一つは、いつリリースを行うかというリリースタイミングの決定であり、もう一つは、どのオーダをリリースするかというリリースオーダの選択である。典型的かつ対称的なリリースタイミング決定方式としてプルシステム(JIT 方式)とプッシュシステム(I/O方式)がある。プルシステムは、ショップ内のオーダの数、つまり仕掛在庫をつねに一定の値に維持するように、リリースを行う。プッシュシステムは、オーダが加工完了して、ショップを出る(アウトプット)間隔にあわせてリリース(インプット)を行う方式である。リリースするオーダの決定は、3・4・2項で述べたいくつかの差立ルールから選択されることになる。
