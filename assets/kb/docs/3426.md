# 3・5・1 購買と外注

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

製造企業(manufacturer)が,市場のニーズに応えて製品を供給するに当たって,製品の生産に必要なものをすべて自社内で調達できる企業はまれで,原材料,部品,資材,構成品(compo-nent),機器,工具,消耗品など,製品の生産に必要なものを,他の企業から買うかまたは他の企業に委託して作ってもらうことが多い.対価を払って経営上必要なものを外部から求めることを購買(purchasing)という.特に,他の企業に委託して自己の定める規格のものを作ってもらうことを外注という.通常,購買活動には外注を含むので,外注を除いた狭い意味での購買を狭義の購買と呼んで外注と区別することがある.狭義の購買と外注の基本的な違いは,前者は所有権の譲渡を内容とする売買契約(bargain)であるのに対して,後者は業務の完成を内容とする請負契約(contract)である点にある.いずれも,対価を払って経営上必要なものを購入することに変わりはない.一般に,狭義の購買は市販されている規格品や標準品の購入を対象とし,外注の対象は自社の仕様・規格により他の企業に製造を委託する規格外部品や特殊品であることが多い.狭義の購買の対象を購買品,外注の対象を外注品と呼ぶ.購買活動は経営活動の一部であり,販売活動,生産活動,財務活動など他の経営の基本活動と密接な関係がある.製品の製造原価を低減するためには合理的な購買活動を進める必要がある.
