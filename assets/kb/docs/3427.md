# 3・5・2 購買管理の役割

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

製造企業における購買活動は、生産活動に必要な原材料、部品、資材、構成品、機器、工具、消耗品などを調達する活動である。製造企業が、より良い製品をより安く生産するためには、経営上必要なものを、信頼できる取引先から、適正な価格で、必要なときに、必要な量だけ確実に購入できる仕組みを整え、維持する必要がある。経営上必要なものを合理的に調達することを計画し実施・統制する行動体系を購買管理(purchasing control)という。特に、自動車製造業のように多数の構成品・部品を組立てて製品を製造する加工組立型企業においては、使用する構成品・部品の多くを他の企業から買うかまたは他の企業に委託して作ってもらうので、製造原価に占める原材料費、買入部品費、外注加工品費など、購買・外注に伴う費用のウエイトが大きく、購買・外注管理の巧拙が製造原価に大きな影響を及ぼす。また、購入された原材料、部品、資材、構成品の在庫管理(inventory control)の巧拙が在庫投資に大きな影響を及ぼす。このように、製造企業においては原材料、部品、資材、構成品など生産に必要な資材の購買が特に重要であるから、以下では、これら生産用資材を購買管理の対象として考える。購買管図3・19生産計画と生産統制理の主要な役割には、以下のことが挙げられる。a. 用途に最も適したものを購入すること製品のもつべき機能から、その製品に必要な構成品・部品がもつべき機能を明らかにし、最小のコストで必要な機能を得る努力をすることである。これは、購買品、外注品の価値分析(VA : value analysis)や市場調査(marketing research)、品質管理(QC)により実現される。b. 信頼できる取引先から購入すること購買活動は自社の製品に利用するために他の企業が製造した構成品や部品を購入することであり、継続的に生産活動を行い、自社の製品の市場における競争力を向上させるためには、継続的に取引可能な信頼できる優れた取引先を選定しなければならない。そのためには、広い範囲から取引可能な供給源の情報を集め、潜在的な供給源をリストアップしておくことが必要である。c. 適正な価格で購入すること製品の製造原価を引下げ、市場での競争力を高めるためには購買品、外注品の適正な購入価格を設定するとともに、継続的に購入価格を引下げる努力をする必要がある。ただし、購入価格を引下げることと必要以上に安く買う(買いたたく)こととは全く異なる概念であることに注意しなければならない。前者は価値分析をとおして最小のコストで必要な機能を得る努力をすることを意味し、後者は場当たり的に不当に安い価格で購入することで、信頼できる継続的な取引は不可能となり、結果として高い買い物になるおそれがある。d. 納期に間に合うように購入すること購買品、外注品の納期(due date)を確保することは製品の製造原価を引下げるために特に重要である。納期の確保とは、必要なものを必要なときに必要なだけ入手することで、必要なときとは、生産計画であらかじめ定められた時期をいう。この時期に遅延することはもとより、早すぎてもよくない。e. 適正な在庫を維持すること購買したものを、使用する前に一時的に保管する現象を在庫(inventory)という。需要量の変動に柔軟に対応しながら生産活動を円滑に遂行するためには、購買品、外注品をある程度在庫しておく必要がある。しかし、過剰な在庫は製造原価の増大を招くので、購買品、外注品の在庫を適正量に維持することが製造原価を低減するうえで重要である。f. むだのない輸送方法で購入すること必要なものを必要なときに必要なだけ購入することは、多頻度小口輸送をすることとなり、輸送回数が増加し、結果として輸送コストが増大する。特に、自動車製造業のように多数の構成品・部品を購買・外注する加工組立型企業の場合、その影響が大きい。したがって、効率の良い輸送方法を利用する必要がある。以上のように、購買管理の基本的な役割は、生産活動を円滑に遂行し、製造原価の低減を図るために、必要な資材を、必要なときに、必要な量、適正な価格で獲得し、必要としている生産工程に効率良く供給することにある。したがって、購買に関する計画(購買計画、purchasing planning)は、販売計画、生産計画(produc-tion planning)に依存する。
