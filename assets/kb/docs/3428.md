# 3・5・3 外注管理の役割

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

外注は購買活動の一部である。一般に、製品が多様化して複雑になり生産量が膨大になると、その製品に必要な構成品・部品をすべて自社内で製造するには膨大な設備と資本を要するので、自社内では主要な構成品・部品を生産し、その他の構成品・部品の生産を関連産業や他の企業に委託することが多い。注文を受ける企業を外注先企業(supplier)、下請企業(subcontract)、協力企業、関係企業あるいは子会社といい、生産を委託する企業を発注企業あるいは親会社という。狭義の購買と違って、親会社は、図面、諸工具や材料を有償または無償で支給し、外注先企業に対して技術指導、経営指導や融資援助などを行うのが普通である。外注の意義は、1垂直的分業によって中小企業の生産技術や生産設備を利用できること、2社内製作に必要とされる生産設備や要員の固定費を削減できること、3景気変動に対する安全弁として利用できること、4賃金の二7―51重構造を利用して加工費の節減を利用できること、5外注先企業が地方に散在している場合良質の労働力を利用できること、などが挙げられる。自己の定める規格の構成品・部品などを、他の企業に委託して作ってもらうことを計画し実施・統制する行動体系を外注管理(subcontract control)という。外注管理の主要な役割は、基本的には3・5・2項の購買管理の役割と同様であるが、自社の仕様や設計で他の企業に生産を委託することから、外注先企業の善し悪しはただちに自社の業績に反映するので、3・5・2・a~f項のほかに、次のことが挙げられる。a. 外注先企業の工程管理能力を充実させること下請企業、協力企業の中には経営規模が小さく、経営管理能力や技術水準の低い企業もあるので、長期的な視野に立って、その強化・充実を図り、外注品の品質、コスト、納期、数量を確保する必要がある。また、専門技術をもつ外注先企業を、その分野の専門業者に育成し、協力関係を強めることが重要である。
