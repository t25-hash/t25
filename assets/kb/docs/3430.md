# 3・5・5 購買の理念と原則

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

購買管理とは、生産計画を達成しうるように、生産に必要とする資材を、適正な取引先から、適正な品質を確保しつつ、適正な時期に、適正な数量だけ、適正な価格で入手するための管理活動である。購買活動は、企業間の取引であるから、交渉に当たる担当者は、購買品、外注品に関する専門的な知識、経験、才能、手腕を必要とする。また、外部との金銭の取引があるので、倫理的な問題を生じないよう、明確な購買倫理をもたなければならない。企業として、次のような購買の理念をもつ必要がある。(1) 取引の対象となりうるすべての企業に対して、公正、公平、厳正、誠実に対応すること。(2) 取引先の業務上の秘密を守ること。(3) つねに市場の動向を把握し、不利な購買をしないように努めること。(4) より有利な取引先を求めるよう努めること。(5) 取引先からの謝礼、接待、贈り物等は辞退すること。以上の理念のもとで、購買に関する一般的な原則として、以下のことを考慮する必要がある。a. 取引先に関する原則1取引先は、購買を要するものごとに、原則として2業者以上とし、それによって価格、サービスの競争をさせるとともに、取引先におけるなんらかの事故の発生に対し、危険の分散を図る、2新しく取引先を求めるには、その企業の信用状態、管理状態、自社の購買要件に対する適否などにつき調査すること、3取引先ごとに、その納入成績、取引における協力度などを評価して対処すること。b. 価格に関する原則1価格はあらかじめ目標を定め、これに対して公正な取引により決定すること、2数量に応じて取り決めた単価は、その数量に限り適用すること、3継続して使用する単価は、1箇年を越えない一定期間ごとに、必要があると認めるときはそのつど検討すること、4業者の見積りを要しないもの、業者の見積りを要するもの、見積り合わせするもの、入札にするものの区分の基準を定めること。経団連は、1990年4月、公平かつ透明で開かれた「購買取引行動指針」を示し、各企業に、この指針に基づく自社の購買取引規定を作成するよう指導した。表3・3にその抜粋を示す。
