# 3・5・7 購買管理の重点

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

購買管理は、必要な品質の資材を、必要量、必要な時期にできる限り低い価格で獲得することにより、製造原価の低減に寄与することを目的として、3・5・2項で述べたような役割を有する。これらの役割を果たすための方法を概説する。a. 用途に最も適した物の購入用途に最も適した物を購入するには、製品のもつべき機能から、その製品に必要な構成品・部品がもつべき機能を明らかにし、最小のコストで必要な機能を得る努力をすることである。これは、購買品の価値分析により実現される。価値分析とは、必要な機能を最低のコストで得るために、その機能とコストの釣合いを研究し、設計や材料の仕様の変更、製造方法や供給源の変更などを社内外の知識を総合して組織的に行う行動であり、資材の品質と購入価格との関係を合理化するものである。価値分析を行うためには、コストの情報を集め、その機能は何かを念頭に置いて、より安くこの機能を満足する方法を、材料は変更できないか、製造方法に改善の余地はないか、標準品・共通品で代用できないかなどのあらゆる面から検討し、価値的に劣る点を発見し改善することである。価値分析は、製品を構成する部品ないし資材が、その果たすべき機能、すなわち顧客に対する外観的魅力や使用性能などにおいてむだな機能をもっていないかどうかを分析し、そのむだな機能の排除によってコスト切下げを図ることを目的としている。価値分析を新製品の開発段階において実施することにより生産設備や工場配置、資材の購入先などを新しく決定することにも結び付けられる。b. 信頼できる取引先の選定基準満足のいく購買をするためには、信頼できる取引先を選定し、確保する必要がある。信頼できる取引先とは、将来成長が見込める、人的資源の質と量が豊富な企業で、かつ、財務内容が健全な企業が望ましい。具体的な選定基準として、1コストダウンに積極的である、2後継者が育成されている、3受注品には最後まで責任をもつ、4納期を守る、5生産設備の改善に意欲的である、6優れた経営者がいる、ことが挙げられる。適正な取引先を選定するためには、あらかじめ供給源に関するいろいろな情報を収集し、分析・評価しておく必要がある。c. 適正な価格の算定製品の製造原価を引下げるためには、購買品・外注品の購入価格は安いほうがよい。しかし、価格は購買のための条件の一つであり、品質、数量、納期、価格を総合的に評価して決めなければならない。適正な価格を判断するためには日頃から価格についての情報収集と体系的な整理をしておく必要がある。d. 納期の管理生産活動を円滑に遂行するためには、購買品、外注品の納期を確保しなければならない。納期の遅延(delayof due date)は、生産活動に、1生産工程を停止させるため、生産能率を低下させる、2生産停止に伴う遅れを回復するため、残業や休日出勤することにより労務費が増加する、3製品の納期が遅延し、顧客の信用を失う、4生産現場の意欲を減退させる、5生産計画を変更する必要を生じる、というような悪影響を及ぼす。他方、早期納入も、1当面必要でないものが入荷するので、在庫費用が増大する、2一時的に保管するスペースが必要となる、3一時的に保管するので余分な運搬作業を必要とする、4在庫期間中の品質を保持する必要がある、というような理由で製造原価の増大を招く。このように、納期の確保は購買管理の重要な課題の一つである。納期を確保するためには、適切な納期を設定するとともに、ガントチャート(Gantt chart)などを利用して進捗状況を計画的に確認しなければならない。納期遅延のおもな原因は、生産計画の変更、緊急飛込み品、図面変更など発注企業側に起因することが多いが、受注企業に起因することや、発注企業と受注企業の双方に起因することもある。e. 適正な在庫の維持生産活動を円滑にかつ経済的に遂行するためには、購買品、外注品の在庫量を適正な水準に維持しておく必要がある。適正な在庫を維持することは、発注方式(orderingsystem)と関係する。代表的な発注方式には、定期発注方式(periodic ordering system)と定量発注方式(fixed size orderingsystem)がある。多段工程の定期発注方式には、プッシュシステム(push system)とプルシステム(pull system)がある。トヨタ生産方式(Toyota production system)におけるかんばん方式(Kanban managing system)はプルシステムであり、JIT 生産システム(just in time production system)と呼ばれている。外注品に用いられるかんばんを外注かんばんと呼んでいる。. 定量発注方式(発注点法)による購買あらかじめ発注点(order point)と呼ばれる在庫水準を定めておき、現品の入出庫のたびに在庫記録を更新して手持ち在庫量が発注点に達したかどうかをチェックし、発注点に達したときに一定量を補充注文する。発注量は一定で、発注間隔は変動する。発注量を決める方法の一つに経済的発注量(EOQ: economic ordering quantity)の公式がある。定量発注方式での発注点は、基本的には、発注点=納入リードタイム中の平均需要量+安全在庫量で与えられる。. 定期発注方式(定期点検法)による購買定期的にまとめて発注する方式で、あらかじめ発注間隔を定めておき、次式により発注量を計算する。発注量=(調達期間中の総需要量-発注済みで未入庫量)-(前期末在庫量-安全在庫量)この方式では、発注間隔は一定で、発注量は変動する。定量発注方式は、1価格が低いもの、2一定量をまとめて購入するもの、3調達期間が短いもの、4在庫負担が少ないもの、5需要予測が難しいもの、6年間消費量が比較的安定しているもの、7死蔵品になりにくいもの、などに適用される。これに対して、定期発注方式は、1価格が高いもの、2調達期間がかかるもの、3在庫負担が大きいもの、4需要予測ができるもの、5設計変更の危険があるもの、などに適用される。f. むだのない輸送方法必要なものを必要なときに必要なだけ購入することは、小口多回搬入することとなり、輸送回数が増加して輸送コストの増大につながる。特に、自動車製造企業のように多数の構成品・部品を購買・外注する加工組立型企業の場合、購買品・外注品の大きさや納入量、購入先・外注先からの輸送距離等を考慮して、効率の良い輸送方法を利用する必要がある。外注先から多品目を小口多回搬入する方法として巡回混載方式、デポ方式がある。. 巡回混載方式巡回混載方式は、いくつかの外注先を大型トラックで巡回しながら部品を収集し、まとめて組立ラインへ納7―53入する方式である。この方式は、トラックの積載率が向上し物流コストの削減につながる。輸送リードタイムが比較的短く小ロットで納入される部品が対象となる。. デポ方式(depot system)デポ方式とは、親会社の組立工場の周辺にデポ(倉庫)を設置し、外注先の部品工場からデポまでは、大型トラックでまとめ輸送し、デポで積替えて、デポから組立工場へは混載して適時・適量を納入し、全体の物流コストの削減を図る方式である。この方式では、デポでの在庫の増加や小分け作業・積替作業が発生し、荷役工数は増加するが、大型トラックで部品をまとめて輸送するので輸送費が削減できる。輸送リードタイムが長く、個別に小ロットで納入すると物流コストが割高になる場合に用いられる。
