# 3・5・8 外注利用の目的

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

製造企業が外注を利用する目的は,他社の資本,生産設備,生産技術,労働力を有効に活用することによって,よりいっそう良質廉価な製品を生産することにあり,一般に次のような場合に利用される.a. 技術能力が不足した場合自社にその構成品・部品を生産する技術と設備がない場合,これを補うために外注を利用する.b. 外部に技術的に優れた企業がある場合自社でも生産可能であるが,外注したほうが低原価で構成品・部品を手に入れることができる.c. 生産能力が不足した場合需要は変動するので,好況時に合わせて人員・設備を用意すると不況時に遊休を生じ経済性を損なう.このような危険を避けるために,安全な範囲の生産設備と要員を保有し,この能力以上になった分を外注を利用して補う.これは,一時的な能力不足を補う場合にも当てはまる.d. 資金が不足した場合自社にその構成品・部品を生産するための技術や設備を導入する資金が不足している場合,あるいは,仮に資金を調達できても,投資に見合う需要が見込めないような場合,自社で生産する代わりに外注する.また,技術進歩が早い場合,膨大な設備投資を繰返すことは危険を伴うので外注を利用する.以上のケースに当てはまらない場合,すなわち,1その構成品・部品が外部からは手に入らない場合,2サプライヤの信頼性が低い場合,3コストや品質面で外部よりも自社のほうが優れている場合,4自社の生産力が余っている場合,には内作する.また,5自社の生産工程の秘密を守りたい場合にも内作する.
