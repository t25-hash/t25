# 3・5・9 外注業務(subcontract operation)

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

外注業務は、購買業務とほぼ同じであるが、外注品が納品後ただちに生産工程に入るという点と、自社が定める仕様や規格の特殊部品(規格外部品)が対象であるという点で、特に納期管理と品質管理が重要になる。このため、外注先に立入った管理・指導が求められる点が購買の場合と異なる。また、外注先に提供する図面(指定図)を作ったり、外注先が提供する図面(承認図)を承認したりする技術を自社がもっているというのが前提である。特殊部品を生産する外注先には、その専門的な生産技術や品質管理技術が要求される。外注品がただちに自社の製品の品質に反映するため、外注先の決定には十分な慎重さが要求される。外注業務の要点は、外注先の選定と外注品の円滑な生産進行である。外注先の選定とは、外注品の品質、コスト、数量、納期を満足する外注先企業を選定し、契約発注することであり、この適切な選定が外注管理の最も重要な点である。一般に、その選定基準として、1技術、管理能力、2負荷状況、3外注単価のレベル、4過去の品質、納期などの確保程度、5総合評価、が挙げられる。外注先企業との折衝、打合せにより契約と発注がなされる。外注品の円滑な生産進行を図るために外注先企業と密接に連携して情報をつかむことや、たえず現場に出かけて進捗状況を把握することが必要である。外注品は、その形態から次のように分類される。a. 製品(完成品)外注外注先で製品として完成され販売できる形態になっているような外注。OEM 製品(相手先商標製品)7―54生産システム工学がこれに当たる。b. 構成品(ユニット)外注ユニットとして製品に組込まれる構成品の外注。例えば、自動車のシート、ワイヤハーネスがこれに当たる。c. 部品外注加工部品の外注。例えば、自動車のアルミホイール、クランクシャフト、ピストン、シリンダなどがこれに当たる。d. 粗型材外注鋳物や鍛造品のような素材の外注。e. 加工外注加工作業のみの外注。例えば、塗装、めっき、溶接などがこれに当たる。
