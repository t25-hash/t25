# 3・5・10 外注管理の重点

> **章**: 第3章 管理システム
> **節**: 3・5 購買・外注管理

外注管理は,他企業のもつ専門技術を有効に活用して,品質の良いものを必要な時期に適正な価格で調達するための管理活動である.その主要な管理活動については,すでに3・5・7項の購買管理の重点で述べたが,特に,以下のことが重要である.a. 外注品の品質に関する活動自社工場での品質管理体制と同じレベルでの品質管理活動の展開を指導し,納品時点での検査を省略しうるまでの品質保証能力の確立への協力が必要である.b. 外注品の適正単価の管理外注契約時における適正単価の算定にとどまらず,外注単価の引下げの可能性の追求を価値分析への指導,協力によって進めるべきである.c. 外注先企業の工程管理能力の充実への指導外注品の品質,価格,納期,数量を確保するためには,親会社は外注先企業の工程管理能力を向上させ,生産性を高めるべく指導・育成する必要がある.これを外注指導という.外注指導の内容は,経営・管理に関する指導と技術・技能に関する指導に大別できる.また,外注指導の実施方法には,直接的な方法と間接的な方法がある.直接的な方法は,親会社から外注先企業へ適切な人が出かけて,一定期間指導を行う方法であり,間接的な方法は,教育・訓練や研究会等を通して指導を行う方法である.これらの指導を通して外注先企業の生産技術と工程管理能力を高め,特徴をもった専門企業へと発展させるとともに,加工外注,粗型材外注から部品外注,構成品外注へ,さらには,製品の完成をも委託しうる完成品外注へと,外注先企業の能力の開発および自立化への可能性を追求させるべきである.
