# 3・6・1 在庫管理の意義

> **章**: 第3章 管理システム
> **節**: 3・6 在庫管理

a. 在庫とは在庫(inventory)とは,一般的には,原材料,半製品,部品,製品といった保有物のことである.在庫の不足は,諸活動に大きな影響を及ぼす.材料や部品の不足は,工程ストップや生産計画変更等,工程に混乱を生じさせ,また製品の不足は顧客の要求に応えられないことから機会損失や信用失墜を大きくする.逆に,過剰な在庫は保管費用等のコスト増大のみならず,資金を固定化させ,さらには在庫品の品質劣化や陳腐化が生じ,棚卸しの減耗につながる.特に,市場における製品寿命が短くなっている現在,陳腐化は重要な問題である.これらが,原価を押上げ,利益を低下させることになる.したがって,在庫の過不足から生じる結果のトレードオフ関係に配慮して適正な水準の在庫を保有することが必要である.b. 在庫の機能在庫保有の仕方によって生産や流通のシステムが大きく変わる.例えば,企業活動を生産活動と販売活動に二分して考えるならば,製品在庫を保有することによって,販売活動上の不確実性による影響を生産活動に直接及ぼすことを避けることができる.したがって,製品在庫は,販売活動と生産活動間のバッファとしての機能をもつ.さらにこの在庫保有量の大きさによって,販売活動と生産活動間の相互影響の仕方が変わってくることも明白である.このことは,種々の活動間においても言える.生産活動をさらに細分化すれば,各工程間の在庫保有の仕方,販売活動にあっては,工場在庫,中間在庫,末端在庫の保有の仕方によって各工程の関連の仕方が変わり,管理方法も変わる.このように,在庫には,その前後の活動を1切り離す機能と2結合させる機能がある.最近話題になっているサプライチェーンのあり方の議論はこの在庫の保有の仕方と大きくかかわっている.c. 在庫保有の動機在庫を保有する動機として,アローは次の三つを挙げている.1取引動機,2予防動機,3投機動機取引動機による在庫とは,需要と供給の時期や量のアンバランスを吸収するために保有する在庫である.また,経済的効果を得るためにあえて需要と供給の時期や量をずらすことによって生じる在庫のことである.例えば,・段取りや購入の効率を考え,まとめて生産または購入・需要が一時的に発生する商品(例えば季節商品)に対して,生産設備を需要のピーク時の必要量より低く抑え,需要時期に先行して生産し,在庫により対応(この場合,在庫は生産設備を保有することと同じ意味をもつ)・供給が一時的に集中してなされる場合(例えば農作物,海外からの大量の原料輸入など)に貯蔵して使用などの場合に保有するものであり,ロットサイズ在庫や見越在庫などがこれに当たる.予防動機による在庫とは,需要や調達期間の変動に対処するために保有する在庫のことであり,安全在庫と呼ばれる.例えば,・需要量や調達期間がばらつく・需要予測の誤差が大きい・供給量や時期が不確実などの場合に種々のばらつきや誤差等を考慮し,統計的に決められる.投機動機による在庫とは,価格変動を見込んで低価格時に購入し,保有するために生じる在庫のことである.これは,価格予測のあり方や企業のポリシーによって異なる.d. 在庫管理とは在庫管理体系(inventory managementsystem)とは,「在庫品の量を計画し,統制するための体系」であり,「生産体系,販売流通体系と特に密接な関係をもつ」(JISZ 8121:2004).また,資材計画・購買管理(外注管理を含む)・在庫管理・倉庫管理(運搬管理を含む)からなる資材管理の一部として位置づけられ,相互に密接な関係をもつ.在庫管理(inventorycontrol,inventory management)とは,「必要な資材(原材料・部品・仕掛品・製品)を,必要なときに,必要な量を,必要な場所に供給できるように,各種品目の在庫を好ましい水準に維持するための諸活動」(JIS Z 8141:2004)のことである.さらに,在庫管理は,生産活動のみならず流通と密接な関係をもち,調達-生産-流通-販売-回収という物の供給過程全体を通して最適であるよう設計されリアルタイムに反応できるシステムであることが要請される.e. 在庫管理の機能と効果在庫管理の機能は次のとおりである.(1) 在庫品の数量的・金額的・時間的な計画と統制各品目の入出庫特性や在庫特性を検討し,在庫計画を立てること.(2) 在庫品の入庫・保管・出庫記録活動発注・入庫・保管・出庫を管理し,その活動を記録し,在庫水準を適正に維持すること.(3) 在庫品の保管作業活動効率的な保管活動により,在庫関連費用を節減して原価を低減すること.(4) 品切れの防止製造活動,販売活動等を円滑にし,顧客の信用を高めること.(5) 過剰在庫の防止在庫投資額の節減を図り,資金を潤沢化すること.(6) 操業度の安定化製造と販売との間のバッファとなること.(7) 生産-流通システムのあり方を左右すること.以上のことから,適切な在庫管理によって,企業の長期的利益に貢献することができる.f.
