# 3・6・5 在庫品の管理

> **章**: 第3章 管理システム
> **節**: 3・6 在庫管理

購入資材の受入れ,貯蔵,出庫をスムースに行うためには,資材の保管場所や数量が明確に把握されていなければならない.ここでは,入出庫の確実な把握,在庫のロケーション管理が重要なポイントである.また,倉庫保管中の減耗や紛失,品質劣化等に注意を要する.a. 保管場所の管理在庫品の保管場所を決める方法は大きく次の二つがある.. 固定ロケーション(fixed location)方式品目ごとにそれぞれ固定した保管場所を決めておく方式のこと.管理が容易で有効な方法であるが,品目数が多く,在庫量の変動が大きい場合にはスペース効率が悪くなる.. フリーロケーション(free location)方式,(ランダムロケーション方式)入庫のつど適切な空きスペースに保管する方式のこと.扱う品目が多い場合,特に製品ライフサイクルが短い品目の場合に保管スペースを有効活用できる.保管場所の決め方や保管場所と数量の記憶が必要になり,コンピュータの支援が必要になることが多い.b. 在庫の数量把握:棚卸(inventory stocktaking)ある時点における在庫保有高を品目ごとに調査し,台帳の残高を確定する手続きを在庫棚卸という.在庫品は,保管中に目減りや破損,陳腐化・劣化等の減耗が生じるため,棚卸によって現在の手持ち数量を把握し,これらを金額的に評価して正しく記録しなければならない.棚卸の実施方法には,現品調査を行わずに帳簿上の在庫数量を検討する帳簿棚卸と,実際に現品の数量・価格・品質等を調査する実地棚卸があり,また,実地棚卸の実施方法には次のものがある.. 一斉棚卸期末など,特定の日に全品目の在庫をいっせいに調査し,在高を確定する方法であり,この方法が最も望ましいが,調査に多くの手間と時間がかかる.. 分散棚卸品目をいくつかに分け,対象期間中に分散して棚卸する方法で,棚卸作業を平準化できる.循環棚卸法はその代表的なものである.c. 在庫の評価在庫は棚卸資産として評価される.同じ材料・部品であっても取得価格に相違があり,保管や運搬中の目減りやき損,陳腐化・劣化等による減耗をも考慮して価値評価する必要がある.おもな評価方法には次のものがある.. 先入れ先出し法(FIFO: first-in first-out method)先に受入れしたものから順次払い出したものと仮定してその価値を評価する方法.この方法によると,在庫在高が現時点の時価に近い価値として評価されることになる.. 後入れ先出し法(LIFO: last-in first-out method)先入れ先出し法とは逆に,最も新しく受入れたものから順に払出したものと仮定して評価する方法.この方法によると,消費した材料や部品の価値を現時点の時価に近いもので評価することになるが,在庫在高は古い受入れ価格で評価することになる.インフレ時や価格変動が大きいものの単価計算に向いている.. 移動平均法単価の異なる物を受入れるごとに,受入れ額とその時点の在庫額の和を受入れ数量と在庫数量との和で除して,平均単価を求める方法.この方法は,物を受入れるごとに平均単価を計算する必要があり手間がかかる.また,割切れない場合には,若干の誤差が生じることがある.. 総平均法単価の異なる物を受入れた場合に,月初在庫額と月間受入額の和を月初在庫数量と月間受入数量との和で除して総平均を求め,これを単価とするもの.月単位の移動平均法ともいえる.この方法は,受入れ,払出しを月でまとめて計算するので事務の簡素化につながる.この方法も,割切れない場合には,若干の誤差が生じることがある.. 予定価格法過去の受入れ価格に現在の受入れ価格や将来の予想価格を考慮して,予定の単価を決めて用いる方法.予定価格は,通常1期間は据置くが,時価との差異が大きくなると改訂することが必要である.この方法によると,予定価格法で求めた単価から得られた期末在庫高と,実際の購入単価で求めた期末在庫高との間に差異が生じるため,その処理が必要となる問題はあるが,払出単価の据置や1箇月分の受入れ,払出しを一括計算することにより事務の簡素化につながるメリットがある.. その他の方法ほかにも,個別法(実価法),単純平均法(算術平均法),最終仕入原価法等がある.d. 在庫関連費用在庫を管理するためにかかる在庫関連費用として重要なものは次のものである.. 発注費用(ordering cost),段取費用(set-up cost)購買,受入検査,倉庫,経理,輸送部門等における業務に関する人件費,通信費,消耗品費,荷造包装費,運賃等.ただし,発注ロットを決める際には,これらのうち発注回数に直接関係する費用項目が対象とされ,生産部門では段取費用がこれに当たる.. 保管費用(inventory carrying cost,holding cost)在庫を保管するための費用で,次のものがある.(1) 貯蔵保管に要する設備費用(倉敷料,倉庫賃貸費,保管関係設備の減価償却費,光熱費,保管設備費,運搬機器費用等)(2) 貯蔵保管に要する業務管理費用(倉庫管理費,現品管理費,積卸費,棚卸費,入出庫費等)(3) 税,保険料(4) 運搬費用(入出庫時,倉庫内での移動や倉移しのための運搬費用)(5) 減耗費(品質劣化,破損,陳腐化,目減りによる損失,評価法による損失等)(6) 資本費用(在庫資金を調達するための費用,借入金利等)(7) 機会費用(保管費にまわさなければ得られたであろう利益)これらの費用は,単位期間の在庫品当たりまたは面積当たりの費用として把握されることが多い.単位期間の保管費用を在庫金額で除した保管費率という形でも表示されるが,その年間保管費率は20~30%にもなるのが一般的といわれている.. 品切れ(stock out,shortage)損失品切れ時に発生する損失のことで,金銭の支出を直接伴うものとそうでないものがある.・品切れ処理費用(つなぎ止め費用,緊急調達のための諸経費や原価の上昇,生産計画変更費等の品切れを処理するための費用)・機会費用(品切れがなかったら得られたであろう利益)・信用失墜費用(顧客に対する信用をなくすことによる今後への影響)機会費用や信用失墜費用は政策的に決められることが多い.e. 在庫の総合評価在庫管理の運営状況を総合的に評価するものとして,在庫量の側面やサービスの側面からの評価尺度がある.. 在庫回転率これは在庫がある一定期間内に何回回転するか,すなわち何回入れ替わるかを表すものであり,この値が大きいほどよいとされる.在庫回転率=期中払出高期中平均在庫高または使用実績量標準在庫量これの逆数に当たる在庫月数や在庫日数というとらえ方もある.在庫月数=平均在庫量月間使用量. サービス面の尺度納期遅延や品切れの評価尺度として用いられる.品切れ率=納期遅延件数受注件数または品切れ回数発注回数または品切れ日数営業日数この逆の評価尺度として,サービス率がある.さらに,品切れ時における状況を評価するものとして,遅延日数,品切れ発生による機械停止時間,特急購買の回数,督促件数,7―59督促費用なども間接的に在庫管理の善し悪しを表すものである.f.
