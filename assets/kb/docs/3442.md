# 8・2・4 生産システムでの考慮要件

> **章**: 第8章 社会と生産システム
> **節**: 8・2 民族性・地域性

8･2節では図8･3の概念に基づいて，調査や解析による民族性・
地域性に関係するマクロな挙動と効果の予測，そして予測結果に基
づいた生産システムの設計・制御について概説した．主要なトピッ
クスについてまとめると，次のようになる．
(１)産業社会全体の効用や活性化，利益の観点から，民族性や
地域性を生かした分業・特化による生産システムの設計・制御法が
ありうる．
(２)評価フェーズで環境変化を同定し，変化の度合いが異質
か，似通っているかに応じて，生産システムを特化するか，あるい
は総合化するか，などを設計・制御することはミクロな利益とマク
ロな効果の両点で有効である．
(３)技術移転をする場合，一方向教育を選択するか，双方向教
育を選択するかは，生産文化に適合する形で決定することが効率的
である．
(４)技術革新を生産システム全体に伝ぱさせるためには遷移率
を一定値以上に高めることが必要である．すなわち，科学技術に高
い価値を付ける，新しい考え方が外からすぐに入ってくる，計画
的・合理的に仕事をするなどといった生産文化を生産システムにも
たせることが重要と考えられる．
(５)生産システムならびに製品を特化する場合，民族性，地域
性に適合した文化的設計属性を求める調査と，その属性値を定量化
してそれを形とする技術が重要である．
最後に，8･2節での考慮例では時間項を無視したが，実際には評
価や調査のサイクルがあり，セル型生産システムの例で述べた，選
択権をもたせるような時間項を考慮する設計・制御法のあることを
注記しておく．
