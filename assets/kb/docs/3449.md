# 3・8・3 保全計画技術

> **章**: 第3章 管理システム
> **節**: 3・8 設備管理

a. 保全方式とその決定要因保全方式(maintenance pol-icy)は,設備の故障への対処法として整理されてきた.図3・31は,JIS Z 8115:2000による保全方式の分類である.ここでは,保全方式を大きく予防保全と事後保全に分類し,さらに,前者を時間計画保全と状態監視保全に分類している.これらは,時間基準保全(TBM : time based maintenance),状態基準保全(CBM : condi-tion based maintenance)とも呼ばれる.保全作業は,図3・30にも示すように,検査・診断(監視,点検なども含める)と,交換・補修・調整などの処置に大別されるが,JIS の保全方式の分類は,このうち,処置の実施基準に基づく分類となっている.すなわち,時間計画保全では,一定の時間(経過時間もしくは動作時間)経過したことをもって処置を実施するのに対して,状態監視保全では,設備の状態が基準を下回ったことをもって処置を実施する.検査・診断は,したがって,状態監視保全を前提とした保全作業になる7―63図3・30設備保全管理のフレームワーク図3・32 RCM による保全方式の決定手順が,それをどのようなタイミングと方法で行うかを決めることも保全方式決定の重要な課題である.効果的な設備保全管理を行うためには,設備に適した保全方式を選択する必要がある.保全方式の選択は,図3・30の基本保全計画のボックスに示すように,技術的評価と管理的評価に基づいて行われる.技術的評価は,技術的に意味のある保全方式を選択するために行われ,劣化とそれによって引き起こされる故障の特性,および,診断・処置などの保全技術の有効性が考慮される.劣化・故障の特性については,例えば,故障発生の時期が予測できない場合は,時間計画保全の適用が不可能であり,また,劣化進展速度が速く突発的に発生するものに対しては,状態監視保全の適用ができないといった関係がある.また,保全技術を考慮するのは,例えば,突発的に発生するように見える故障も,診断技術によっては,漸進的に進行する劣化過程をとらえることができ,状態基準保全が適用できるようになるからである.一方,管理的評価は,限られた予算や人員を,全体として最も効果が上がるように配分するために,保全作業の優先度付けを行う目的で行われる.管理的評価では,劣化・故障の影響度と設備管理特性が考慮される.劣化・故障の影響度とは劣化とその結果引き起こされる機能低下や故障によってもたらされる対象システムの外部に対するマイナス効果の程度を意味する.具体的には,労災,生産損失,納期遅れなどがこれに当たる.また,設備管理上の特性としては,予備品の有無や納期,代替設備の有無,設備の運用に関する習熟度合いなどの項目が考慮される.b. RCM(reliability centered maintenance)基本保全計画のための手法としては,RCM がよく知られている.RCM は,航空機の保全計画手法として開発されたものであるが,近年,他の分野でも広く適用されるようになってきた.図3・32にRCM における保全方式の決定手順を示す.RCM では,設備の構成要素に発7―64生産システム工学生する可能性のある機能故障を列挙し,影響度によってそれらを五つのカテゴリーに分類する.そして,カテゴリーに応じて定められた手順に基づいて,効果的な保全方式を選択する.基本的により簡便な保全方式からその有効性を検討することで,費用対効果の高い保全計画が立案できるようになっている.c. RBI/RBM(risk based inspection/risk based mainte-nance)RCM は,特定の対象やそこで発生する劣化・故障の種類を前提としない一般性のある手法であるが,基本的には定性的な判断に基づいて行われる.しかし,複雑で大規模な対象では,考慮すべき劣化・故障モードが多数に及ぶ.これらに対して,保全資源を適切に配分して,全体として効率的な保全を行うためには,想定される劣化・故障の重要度とそれに対する保全の効果を定量的に評価することが望まれる.RBI/RBM(当初はRBIと呼ばれていたが最近はRBM と称されることも多い)は,このような要求に応じた手法といえる.すなわち,RBI/RBM では,劣化・故障の重要度を,リスク=発生確率×影響の大きさという一元的な尺度で評価する.また,保全の期待効果についても,それによるリスク低減量として定量的に評価する.これらにより,保全で達成すべき目標の設定と,保全項目の優先度付けを定量的に行うことができるので,むだのない合理的なメンテナンス計画の立案が可能になると考えられる.ただし,リスクの評価のためには,発生確率と影響度を見積もる必要がある.しかし,各設備構成要素についてこれらの値を求めることは一般には容易ではない.実際にこの手法を適用するうえでは,対象設備や故障モードをある程度限定したうえで,これらの値を評価するための手順と,その中で必要なモデルやデータをそろえる必要がある.このような意味で,現在提案されている最も具体的な手法として,アメリカ石油協会(API)のRBI手法がある.これは,化学プラントにおける耐圧部からの漏えいという故障モードに対して開発されたものである.図3・33にその手順を示す.リスク評価に関しては,定性的,定量的,およびそれらの中間的な準定量的リスク評価手法が提案されており,目的に応じて使い分けられるようになっている.定性RBIでは,発生可能性(likelihood)と影響度(consequence)のそれぞれに関して,評価項目についての評点をつけ,それらを総合して五段階のランク付けを行い,その結果を図3・34に示すリスクマトリックスに当てはめることで,リスクを四段階に評価する.定量RBIでは,漏えい量,速度,形態などを予測し,それに伴う影響を,火災,毒性,環境,経済性の四つの観点から評価する.また,漏えいを引き起こす耐圧部の破損確率は,業界における一般的な損傷確率に,機器と管理体制の特性による修正係数を乗じて求図3・33 API-RBIの手順図3・34 リスクマトリックス図3・35 設備診断の手順める.
