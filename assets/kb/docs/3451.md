# 8・5・2 研究開発における協調i

> **章**: 第8章 社会と生産システム
> **節**: 8・5 国際分業と協業

研究開発における国際共同プログラムとして，IMS（intelligent
manufacturing system）がある．これは，1989年に日本が世界の
先進国に呼びかけて検討を始めた製造技術に関する研究開発の国際
的な協調プログラムであり，1995年に正式な合意を得て，2003年
時点で約300の企業（日本は約50社）と200の大学・研究機関が
参加し，約20のプロジェクトが進行している．参加地域は，アメ
リカ，カナダ，日本，オーストラリア，EU，スイス，韓国の7地
域である．
IMS構想では，地球生産性を向上させるための主要因としての
競争を効果的に促進するためには，協調が行われなければならない
と主張している．現在の企業ごとの競争力向上への努力は次のよう
な課題を含んでいると考えられる．
(１)多重研究開発投資
同種の製品において競争力で優位に立つためには，当然研究開発
を通じて技術的優位性を獲得しなければならない．そのための投資
は，製品の複雑化，技術的高度化によって高額化する傾向がある．
競争的状況では，この投資は同一目的のものが多数の企業によって
並列的に行われることになる．
(２)競争力の表層化
市場競争が激化し，しかも多数の企業が参入すると，勝者はほと
んど技術的に同じ力をもつ企業群の中から選ばれることになり，製
品のもつ微妙な差異の効果が過度に大きくなる．すると，この微妙
な差を創出するための努力が過大化して，地球生産性と無関係な努
力に資源を投入することになってしまう．
(３)製品サイクルの過小化
競争に勝つためには，消費者の要求をなるべく早期に実現した製
品を提供することが重要な要件である．これ自体は健全なことであ
るが，過当競争状態では消費者の要求の進展速度を新製品提供の速
度が上回ることになり，過度になると製品は売れるが消費者の本当
に欲していることにならず，結局地球生産性は低下する．
(４)技術における過度の急進性
競争に勝ち続けるために，製品技術，製造技術ともに最先端にい
続けなければならない．このことは，関連技術の高度化を極限的に
推し進める状況を生む．この結果，数々の社会的なコストを増大さ
せる問題を生じ，望ましい技術進展の姿をゆがめる．
これらの不整合の解決の一つに，研究開発の協調がある，という
のがIMS構想の立場である．協調は，製造業にとって必要な基礎
的技術の開発を目標として行われる．この協調は競争と矛盾するも
のではなく，従来の競争のもつ問題を解決しつつ，競争を正常化す
るというものである．自由主義経済において企業が自由な競争のた
めに秘密にする知識とは一部のものにすぎず，多くの知識は共有さ
れている．共有される知識のうちには，競争を行うための場を作る
知識と，共有するほうがよいと判断されて公共化された技術知識と
がある．この共有される両知識を生み出すことが協調的に行われる
べきことである．
