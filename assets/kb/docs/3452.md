# 3・9・1 原価管理の考え方

> **章**: 第3章 管理システム
> **節**: 3・9 原価管理

a.原価管理とは 企業は需要の3要素である品質，原価，納
期をコントロールし，顧客の満足を得なければならない．そのうち
の一つである原価に関するコントロールは，顧客の購入価格に影響
を与えて販売量に影響を及ぼすばかりでなく，原価が企業のあらゆ
る経営活動の経済価値を測定しており，すべての経営活動は原価に
集約されているので，企業の存続に不可欠な利益を決定付ける．
原価管理を行う目標は，原価効率性（cost eﬃciency，能率）を
高めることである．原価効率性は
アウトプット（品質・納期，生産量など）
インプット（原価）
という生産性指標であるから，なるべく少ないインプットで大きな
アウトプットを得るような方策が原価管理において採られることに
なる．インプットは原価で測定するが，アウトプットは生産額など
の金額面だけでなく，品質や納期達成などの質的な側面で把握する
こともある．
原価管理活動の基本は，他の現場管理手法と同じように，適切な
計画を設定し，それに基づいて活動を行って統制することである．
