# 3・9・1 原価管理の考え方

> **章**: 第3章 管理システム
> **節**: 3・9 原価管理

a. 原価管理とは企業は需要の3要素である品質、原価、納期をコントロールし、顧客の満足を得なければならない。そのうちの一つである原価に関するコントロールは、顧客の購入価格に影響を与えて販売量に影響を及ぼすばかりでなく、原価が企業のあらゆる経営活動の経済価値を測定しており、すべての経営活動は原価に集約されているので、企業の存続に不可欠な利益を決定付ける。原価管理を行う目標は、原価効率性(cost efficiency、能率)を高めることである。原価効率性はアウトプット(品質・納期、生産量など)インプット(原価)という生産性指標であるから、なるべく少ないインプットで大きなアウトプットを得るような方策が原価管理において採られることになる。インプットは原価で測定するが、アウトプットは生産額などの金額面だけでなく、品質や納期達成などの質的な側面で把握することもある。原価管理活動の基本は、他の現場管理手法と同じように、適切な計画を設定し、それに基づいて活動を行って統制することである。7―67したがって、原価管理のための計画値として原価の目標を設定し、これを達成することで計画した原価効率を達成する。あるいは計画段階で原価の目標値を引下げ、これを達成することで原価効率を向上させる。すなわち利益を確保できて、かつ実行可能な原価の目標を設定し、この原価目標を達成するように業務遂行の工夫を行い、その結果を目標値と比較検討して、差異原因を把握して作業改善を行って原価効率の向上を図るというプロセスを踏む。こうしたプロセスを実効あるものとするためには、原価集計の組織単位となるコストセンタを適切に設定することが重要である。原価目標を設定し、これを遂行するという原価責任はコストセンタごとに設定される。したがって、コストセンタごとにどの範囲の原価が誰の責任権限に基づいて集計され、マネジメントされるのかを明確化することがたいせつである。b. 原価管理の体系近年の原価管理は三つのアプローチから成り立っているといわれている。すなわち、1原価維持2原価改善3原価企画である。1原価維持(cost keeping)は、原価管理を最も狭い範囲でとらえたときのアプローチ方法で、原価統制とも呼ばれるが、目標として設定された原価である標準原価を達成して原価効率性を高める。したがって、生産条件が与えられ、かつ原価標準も与えられたもとで、これを達成するために日常的な方策を講じるのが原価維持の役割ということになる。すなわち原価維持においては、製品の仕様や品質あるいは設備、人員など、生産に必要な基本条件が定まっている。そうした制約条件のもとで、まず、標準となる基本的な生産方法が決定される。これに基づいて投入される生産要素ごとの消費量とその単価の標準が設定されて、目標の原価である標準原価が定められる。次に設定された標準原価を達成しているかどうかを確認するために、実際に消費された原価、すなわち実際原価を算定する。そして実績の原価である実際原価と目標の原価である標準原価の差異を計算し、差異が発生した原因の分析を行う。ここで、標準原価は一度設定されると少なくともある期間にわたっては、変更されるべきものではないとの立場にあるので、日常業務の問題点を改善して、次期においては標準原価が達成できるような措置が講じられる。2原価改善(cost improvement)は、原価計画とか原価低減ともいわれるアプローチ方法で、生産条件のうち基本的な経営条件、例えば大規模な設備変更などを除いた業務方法などを変更することで、目標となる原価設定を変更しつつ原価の引下げを図ろうとする方法である。すなわち、原価改善は原価の改善目標を掲げ、日常的に原価の目標である標準原価を引下げて原価低減を実現するものである。これは今日のように企業の生き残りを賭けた価格競争が激しい場合には、欠かせないアプローチである。原価の目標を引下げるために生産現場の改善を行っていくに当たっては、製品の基本的な仕様や品質、あるいは重要な設備等、経営の根幹部分は所与のものとして考えていく。3原価企画(target costing)は、生産条件がまだ定まらない製品の企画・開発・設計の段階で目標原価を設定し、実際に発生するであろう原価をシミュレーションによって見積もりながら、原価の引下げを図るアプローチであり、経営の基本構造の変革も同時に行うので、原価を引下げる効果が大きい。すなわち、今日のように製品が市場に現れてから消え去るまでの製品ライフサイクルが短くなると、生産を開始してから原価目標を設定し、原価維持、原価改善を行おうとすると、その製品はもはや市場から退場してしまっているということにもなりかねない。また、生産段階におけるコスト決定とコスト発生の関係では、コストが実際に発生するのは、生産の三要素(材料、マンパワー、設備)を消費する製造段階であるものの、その製造コストがいくらになる7―68生産システム工学のかを決定付ける製品の仕様や製造設備を決めるのは、開発段階や設計段階である。特に、現在のように多品種少量生産の時代には、従来よりも開発や設計が数多く行われなければならない。このように製品ライフサイクルが短くなり、多品種少量生産を行う状況では、製造を始めてから原価を管理するというアプローチでは、もはや原価管理が後手に回ってしまう危険性が高い。そこで、開発・設計段階で原価の目標を設定したうえで、実際に製造するとどれだけのコストが発生するかをシミュレーションで計算し、価値工学(VE: value engineering)などの手法を利用して原価を低減し、製造段階に至るまでに標準原価を設定してしまうという原価企画が、原価管理の新たなアプローチとして重要視されるようになってきている。
