# 3・9・2 原価と原価計算

> **章**: 第3章 管理システム
> **節**: 3・9 原価管理

a. 原価概念原価管理では以下の原価概念を用いて行う.. 原価の要件原価計算に関する基本的なルールである原価計算基準によれば,原価として扱うことができるのは以下の四つの条件を満たしているものである.(1) 経済価値の消費原価は生産活動において消費された価値であって,貨幣額で測定されなければならない.(2) 一定給付に転嫁される価値経済価値が単に消費されるだけでなく,それが製品・サービスへと形を変え新たな価値として生み出されなければならない.(3) 経営目的に関連原価が企業の本業に関わる活動の結果でなければならない.(4) 正常な状態のもとでの価値の減少日常の定常的な経営活動において発生したものでなければならない.こうした四つの条件を満たさないものは非原価項目として扱われる.. 実際原価と標準原価原価は過去の事実として発生した原価を実際原価(actual cost)といい,将来の計画値としての原価を標準原価(standard cost)という.原価は基本的に,原価=消費量×価格で示されるので,実際原価は実際原価=実際消費量×実際価格で算定される.ここで価格については原価計算を迅速化すること,あるいは価格に偶発的な変動が含まれることを避けるために,予定価格(将来の一定期間の取得価格等を予想して得た価格)を用いることも可能である.標準原価は将来の目標となる原価であるので,消費量の設定は科学的技術的な裏付けや過去の統計資料に基づいて行われ,標準消費量として扱われる.また,価格は予定価格あるいは過去の統計資料と将来のすう勢に基づいて算定される正常価格が用いられる.したがって,標準原価は努力すれば必ず達成できる可能性をもつ目標原価である.標準原価=標準消費量×予定価格(または正常価格)標準原価を計画値,実際原価を実績値として測定することで,原価差異を算出し,どの原価要素がどこの関連部署の責任においてどれだけの大きさの原価上の問題を引き起こしているかを把握することができる.標準原価と実際原価は原価管理における原価維持にとって欠かせない原価概念である.. 製品原価と期間原価原価をどこに集計するかという集計対象によって,製品原価と期間原価に分ける.製品原価はある一定の製品・サービスに集計された原価である.工場で発生する製造原価は,基本的に製品原価として製品ごとに集計される.一方,期間原価はある一定期間に集計された原価であり,営業費(販売費および一般管理費)は期間原価として期間ごとに集計される.製品原価は製品が顧客に販売されるまでは製品在庫として貸借対照表の棚卸資産に計上され,販売時に費用となって損益計算書の売上原価となるが,期間原価は発生したその時点の損益計算書に費用となって利益計算に組込まれる.. 全部原価と部分原価原価要素を製品やサービスなどに対してどの範囲まで集計するかという観点から全部原価と部分原価とに分ける.製品やサービスに対して発生するすべての原価を集計したものを全部原価といい,一部に限定したものを部分原価という.後述するように,原価を操業度との関連によって分類すると,変動費と固定費に分かれるが,製品の採算性を検討して原価低減を行う際に,固定費まで含めて集計すると誤った判断を下してしまうおそれがあるので,全部原価ではなく変動費のみの部分原価を検討する.b. 原価要素の分類原価管理を行うには,原価が経営のどの部分にどのように関わっているかという性質によって,以下のように原価をいくつかの視点によって分類することが必要である.それはどのような側面から原価の責任を明らかにしたらよいのか,また,どこに原価低減に寄与する方策があるのかを示す切り口になるからである.. 形態別分類企業が製造活動をしていく際に,消費したものは何かという視点から原価分類しようとするのが,形態別分類である.そこで生産の三要素である材料,労働力,機械(動力等を含む)を分類の基軸として原価を把握する.(1) 材料費(material cost)製品の一部分を構成するような素材や買入部品などに関わる原価.(2) 労務費(labor cost)賃金や従業員賞与など製造のために消費された労働力に関する原価.(3) 経費(expense)設備関係を中心とした材料費,労務費以外の原価.. 製品との関連による分類原価管理を行うには,製品やサービスごとの原価を算出する必要がある.原価を製品に集計するためには,原価の発生と製品との関係が明らかでなければならない.ところが,原価によっては,製品との関連性が直接的に把握できるものとそうでないものとがあるので,これを以下のように分類する.(1) 直接費原価の発生がどの製品に対するものかが明確なためにそのまま集計できる(直課という)原価.(2) 間接費どの製品のために発生した原価であるかがただちには特定できないので,適切な配賦基準を設定して,各製品に配分される(これを配賦という)原価.. 操業度による分類この分類は生産設備能力の利用度合いである操業度や生産数量の変化に対して,原価の発生額が変化するか否かによって原価を分類する方法である.原価額の変化しないものを固定費,変化するものを変動費という(図3・40).例えば,直接材料費は生産量が変化すれば,これに伴って変化するので,変動費であるが,設備などの減価償却費は操業度や生産量が変化しても,一定期間ごとに一定金額が発生するので固定費に当たる.原価を変動費(variable cost)と固定費(fixed cost)とに分類するのは,製品単位当たり原価を算出する際に,操業度の変化に対して変動費は影響を受けないのに対して,固定費はその額が変化してしまうからである.そこで,固定費を製品原価に集計せずに,期間原価として扱い,製品原価には変動費のみを集計して部分原価を算出する直接原価計算という方法があり,これが製品の採算性検討図3・40 固定費と変動費や原価低減に用いられる.c. 原価計算の手順原価計算は,費用別計算,部門別計算,製品別計算の順序で行われていく.. 費目別計算原価の構成要素を形態別に分類した材料費,労務費,経費を洗い出し,さらに,これを直接費・間接費に区分することで,直接材料費,直接労務費,直接経費および製造間接費ごとに集計する.費目別計算において重要な点は消費量と単価を測定することである.. 部門別計算この段階の計算は原価をその発生した場所ごとに集計する作業である.この計算は原価管理において各部門の原価責任を明らかにするのに役立つ資料となる.部門別に集計する際に,直接材料費,直接労務費,直接経費という製造直接費はその発生場所が性質上明らかであり,製品との関わりも明確であるから,そのまま各部門に直課すればよいが,製造間接費は,製品との関連が間接的であるので,製造直接費のようにはいかない.そこで,製造間接費では,各部門で個別に発生した部門個別費と,例えば工場建屋の減価償却費のように部門にまたがって発生した部門共通費とを区別し,部門個別費は直課するが,部門共通費については合理的な基準により各部門に配賦する.こうして,原価が各部門ごとに集計されるが,部門についても製品に直接関わる部門とそうでない部門とに分かれる.すなわち組立部門のように,製品の生産に直接関わる製造部門と,工場事務部門や修繕部門のように間接的に関わる補助部門とがあるので,補助部門に集計された原価は合理的な基準を設け,製造部門に配賦するという手順を踏む.. 製品別計算部門別計算までの結果をもとに,製品種類ごとに原価を集計する作業であり,製造原価や製品採算性などの情報を提供する機能をもっている.製品別原価の計算では生産形態によってその方法が異なっている.受注生産では個別原価計算が,また見込み生産では総合原価計算が行われる.
