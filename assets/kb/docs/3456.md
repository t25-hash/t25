# 8・2・3 拡散めっき

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

金属素材の表面層に他の金属を拡散浸透させて被覆層を作る方法
を拡散めっき（diﬀusion coating）という．通常行われているのは
粉末パック法（pack cementation）で，これはめっきをしようと
思う金属の粉末，例えばクロムの拡散めっきのときには金属クロム
粉末，活性剤としてのアンモニウム，焼結防止剤としてのアルミナ
粉を混合し，その中に素材を埋め込んで，不活性ガスまたは還元性
ガス中で加熱する方法である．このとき金属クロム粉末は塩化クロ
(２)電気式溶射
アーク溶射（arc spraying）は，2本の溶射材料ワイヤの先端間
に直流ジェットで微粒子として素材面へ吹付けて皮膜とする方法で
ある．本法は，溶射材料が金属に限られるが，出力を大きくして溶
射能率を著しく高くでき，しかも運転経費が安いなどの長所を有す
る．しかし，溶射材料を加熱し，そのため過度の酸化を生じやす
く，合金元素の損失が大きいことなどの短所をもっている．
プラズマ溶射（plasma spraying）は陰極と陽極ノズルの間に発
生させた直流アークで作動ガス（アルゴン，窒素あるいはアルゴン
と窒素，水素またはヘリウムの混合ガス）を超高温に熱し，プラズ
マジェットとしてノズルから噴出させ，この中へ溶射材料の粉末を
送り込み，加熱，加速して素材表面に吹付けて皮膜とするものであ
る．本法は，セラミックスなどの高融点材料から各種の金属，合金
あるいはプラスチックまで簡単に溶射でき，皮膜の素材への密着性
が良好で，高密度であること，不活性あるいは減圧雰囲気での溶射
が行えて，皮膜の性質を改善できることなどの長所をもつが，作業
中に騒音と強い光を発すること，設備費ならびに運転経費が高いこ
となどの短所をもっている．
溶射法のおもな特徴は次のようである．長所として，(１)素材と
して金属材料をはじめ，セラミックス，有機材料など，ほとんどの
材料を対象とできる，(２)溶射材料の種類が広範囲で，しかも，異
種材料を混合した皮膜を簡単に形成できる，(３)表面積の広い素材
の限定された部分に皮膜を形成できる，(４)溶射による素材の変形
が小さいなどが挙げられる．一方，短所として，(１)一般的に，皮
膜と素材との密着強さが比較的に低い，(２)素材表面の前処理に用
いられるプラスト作業ならびに溶射作業そのものの雰囲気が人体に
有害なので，適切な対策が必要であるなどが挙げられる．
