# 8・2・4 溶射

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

溶射（thermal spraying）は，溶融状態に加熱した溶射材料粉末
あるいは粒子を，素材表面に高速度で吹付けて層を作り，皮膜とす
る表面処理法である．溶射材料の加熱源の種類によって，ガス式と
電気式とに大別され，それらには下記の種類がある．
(１)ガス式溶射
フレーム溶射（flame spraying）は燃料ガス（アセチレン，プロ
パンなど）と酸素の燃焼の反応熱を溶射材料の加熱に用いるもの
で，溶線式，溶棒式および粉末式がある．前二者は，ワイヤ状ある
いは棒状の溶射材料を燃焼炎によって溶融し，溶融部を空気ジェッ
トで微粒子として素材表面に吹付けて皮膜とする方法である．粉末
式では溶射材料粉末をトーチ燃焼炎部へ送給する．本法では，溶射
材料は燃焼炎の温度の関係から融点が約2700℃以下のものとなる．
本法の特色は操作が簡易で，設備費，運転経費が低廉なことであ
る．
