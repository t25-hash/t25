# 4・1・2 生産システムの歴史

> **章**: 第4章 自動化システム
> **節**: 4・1 生産システム自動化の体系

生産システムの構築を理解するためには,その歴史を眺めることが必須である.ここでは組立の歴史を中心に生産システムの変化を概観してみる.中世までの生産システムは製品一つずつを製造する個別生産システムであったので,組立も部品ごとに現合(相手に合わせて加工すること,現場合わせ)であった.部品の互換性(compatibility)という概念が始まったのは18世紀に入ってからであるとされている.アメリカの独立戦争当時,1万丁の小銃の生産要求に対して入札したホイットニー(Whitney)は部品を機械加工し,寸法精度を管理することで部品互換性を確立した.1801年,山積みの部品の中から小銃のロック部分の部品を取出して組立てるデモをしたと記録にある.いまでは当然のように思われる部品互換性だが,このように寸法精度を保証して初めて成り立つのである.この寸法の保証図4・3 組立の工程のために限界ゲージが多数準備された.このように,別々に製作した部品を組立てれば,機械全体ができあがるという概念は,まさに要素還元主義の証明とさえいえる.この方法論はその後の大量生産の礎となった.組立作業の効率化追求は19世紀の末に行われた.テイラーが提唱した作業解析は複雑な作業を分解して単純作業の並びとすることで,非熟練工にも熟練工と同様の作業実現を可能とし,同時に効率化を図るものであった.一方,部品搬送を自動化する試みとして,重力を用いてシュート(chute)を滑り落とす方法が導入された(すでに17世紀には使われていた).これはいまでもプレス打抜き部品の排出など多くの現場で使われている.20世紀の初頭にはフォードがベルトコンベヤを導入し,T 型フォードの生産へ適用した.その基本的な理念は,ベルトコンベヤによる作業の連続化(流れ作業)と同期化(固定サイクルタイム)であった.ベルトコンベヤシステム(belt conveyor system)は大量生産システムとして発展し,20世紀を代表する生産システムとなった.大量生産システムは,「低コスト・高品質」の製品を生産するには製品1個当たりの固定費を圧縮すればよいとの理念に立つ.松下幸之助の「水道哲学」(水道の水のごとく,物質を安価無尽蔵たらしめ,楽土を建設することであるという松下幸之助の生産通念)が理想であった物不足の時代には,大量生産システムからの製品は市場においても歓迎されたが,製品が市場にあふれる時代になると,消費者の嗜好の多様化で同一品種が大量に販売されることはなくなっていった.これにより多品種中少量生産,あるいは変種変量生産と呼ばれる生産システムが求められるようになった.
