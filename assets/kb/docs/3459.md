# 8・2・6 物理蒸着（PVD:physical vapor deposition）

> **章**: 第8章 表面処理
> **節**: 8・2 金属皮膜処理

(１)スパッタリング法
真空容器内に導入した不活性ガスをイオン化し，原料ターゲット
表面に衝突させ，表面近傍の原子をはじき出し，これらを堆積させ
て薄膜とする方法である．プラズマを直接用いる方法とプラズマを
イオン源としてイオンビームを用いる方式に大別される．工業的に
は，前者のプラズマを直接用いる方法が多く使われている．直流を
されている．
図8･ 2 アークイオンプレーティング法の原理図
PVD法は，上記の煩わしいプロセスを省くことが可能である．
PVD法では，処理温度が500℃以下のものが多く，コーティング
後の熱処理は不要である．したがって，ファインブランキングの金
型をはじめ，精密・小径のものに使用されている．ただし，内面へ
の付きまわりはCVDに比べ悪く，深絞りのダイ等には適していな
い．
