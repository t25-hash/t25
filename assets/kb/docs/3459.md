# 4・1・3 生産方式

> **章**: 第4章 自動化システム
> **節**: 4・1 生産システム自動化の体系

20世紀を代表する生産システムはフォード生産システム(Fordproduction system)とも呼ばれるベルトコンベヤによる流れ作業システムである.流れ作業は,製造ラインにて実行される複雑な作業を単純な要素作業の並びに分解し,その作業を連続的に実現する方式であり,フォード生産システムは上述のようにこの連続作業をベルトコンベヤで実現する.流れ作業は組立ラインのみならず加工システムにおいても一般化した.自動車の部品製造におけるプレス機械を複数台並べたり,多重ヘッドの穴あけ加工機械を連結したりすることで,トランスファマシンと呼ばれる専用加工システムが作られた.組立工程においても,インデックスマシンと呼ばれるカム駆動の割出機構で複数の組立装置が連動する専用組立機が使用された.トランスファマシンは製品ごとに設計される専用システムである.しかし,組立作業の多くは機械化ができていなかったので,手作業のままに残された.例えば一見単純に見えるねじ締め作業も,(1)ねじ位置が散らばっていること,(2)その位置にばらつきがあること,(3)ねじ締め作業の成否の判断が難しいこと,などの理由から自動化は困難であった(現在でも,自動車のぎ装ラインでのねじ締め作業の自動化率は必ずしも高くない).フォード生産方式により製品価格は低下し,その結果,いっそう大量販売が可能となり,大量生産・大量消費が可能となった.しかし,生産ラインの設置コストは高く,かつ,生産ラインが長くなって仕掛品が多数ライン内に存在するなどの問題点が明らかになっていった.この問題に対応したのがトヨタ生産方式(Toyota productionsystem,徹底したむだの排除で注文によって生産し,切れ目のない流れ生産方式を用いた多品種少量生産方式)である.1980年代に大野耐一によって広められたこの生産方式は生産改善活動という側面も強いが,同時に多くの生産方法の概念を提示した.この方式では,生産に取り掛かってからできあがるまでのプロセスに着目し,むだを徹底的に排除することを目標とした.例えば,仕掛品を少なくすることで,生産ライン内に発生したトラブルはすぐ下流へと影響を与える.そのため,ライン全体の生産性を向上させるには,トラブル発生箇所の担当者のみならず,他の作業員も問題解決に協力しなければならない.このように,安定的な生産システムの7―73構築にはライン担当者全員で取組むことが必要となるとの発想をもっている.このシステムのもう一つの特徴は,プルシステム(pullsystem,下流側の要求によって,上流側が生産物を供給する方式)にある.これによって,いわゆる見込み生産を廃し,作りすぎのむだを排除する.このプルシステムでの物の流れを単純な方法で制御しているのがかんばんシステムである.かんばんとは1回に納入すべき部品の種類と量を表示した指示伝票であり,納入先(受取り手)のところに部品がなくなると,注文伝票として部品メーカへそのかんばんが渡され,かんばんの情報に従って,次の部品が納入される.このかんばんシステムは部品の流れと情報の流れを上手に同期させるために作られたシステムである.その結果,各工程は部品が必要となったときにのみ部品が供給されることとなり,ライン内の部品在庫が減少し,製品の仕掛品も減少する.この考え方はジャストインタイム(just in time)と呼ばれ,必要なときに必要な部品供給がされることを目標としたもので,かんばんシステムと組んで実現される.なお,トヨタ生産方式も生産ラインの構造としては流れ作業を基本としており,フォード生産システムとの共通性は高い.1990年代に入ると,インタネットの一般化により情報の流通においても実時間性と大量データ操作とが容易になった.CAD データやCAM データをインタネットで送ることで,設計地から遠く離れた海外での生産が可能となった.円高から多くの製造業が東南アジア・中国などへ移り,日本や工業化国では製造業の空洞化が進んだ.そのため,国内ではいままで以上に低コストでの生産システムの構築が必要となっている.2004年現在,日本における生産システム構築の考え方は二つに分化している.一つは高い人件費を圧縮することを目的として自動化を積極的に推進する方法論であり,いま一つは人を活用することで設備コストならびに設備設計コストを徹底的に圧縮する方法論である.後者の典型はセル生産システム(cell production system)と呼ばれ,少数(できれば単独の)多能工により製品の組立と品質保証のすべてを担当させる方式である.これはベルトコンベヤシステムの欠点である生産機器据付設置にかかる時間や手間をなくし,市場変化に迅速に対応できるという特徴をもつ.ただし,このセル生産方式は基本的に部品搬送が人手ででき,かつ,人手作業が容易に適用できる生産対象で強みをもつ.
