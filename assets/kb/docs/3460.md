# 8・2・7 化学蒸着（CVD:chemical vapor deposition）

> **章**: 第8章 表面処理
> **節**: 8・3 化成処理，陽極酸化処理

コーティングしたい材料の構成元素を含む単体または化合物の1反応により膜を作製する方法の総称である．
CVDでは広範囲にわり，全く新しい構造や組成の膜を作製することもできる．
化学反応CVD，レーザCVDに大きく分類できる．
熱CVD法では，一般にでは室温付近または比較的低温で良質の膜が作製可能である．
薄膜とや大量生産に適していることから圧倒的に多い．
機械部品の高精度加工に伴い，材料の変形や変質防止のために，CVD技術に代わり，プラズマやレーザを用いた低温CVD技術開 高平滑膜作製ダイヤモンド状炭素（DLC:diamond-like carbon）が大きく脚光強度のDLC膜の開発が注目されている．
種類以上の原料ガスを基板上に供給し，気相または基板表面で化学たる材料のコーティングが可能であり，また原料ガスの組合せによを生じさせるエネルギーの与え方により，熱CVD，プラズマ1000℃近くの高温を必要とするが，プラズマCVDやレーザCVD全体の売上げ高の点で，PVDと比較し，装置がシンプルであるこ 低温CVD技術より低温での成膜技術が望まれている．
したがって，従来の熱発が注目されている．
基板を平滑化する目的や精密機械のしゅう動特性への要求から，を浴びている．
