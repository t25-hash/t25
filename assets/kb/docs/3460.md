# 8・2・7 化学蒸着（CVD:chemical vapor deposition）

> **章**: 第8章 表面処理
> **節**: 8・3 化成処理，陽極酸化処理

コーティングしたい材料の構成元素を含む単体または化合物の1
種類以上の原料ガスを基板上に供給し，気相または基板表面で化学
反応により膜を作製する方法の総称である．CVDでは広範囲にわ
たる材料のコーティングが可能であり，また原料ガスの組合せによ
り，全く新しい構造や組成の膜を作製することもできる．化学反応
を生じさせるエネルギーの与え方により，熱CVD，プラズマ
CVD，レーザCVDに大きく分類できる．熱CVD法では，一般に
1000℃近くの高温を必要とするが，プラズマCVDやレーザCVD
では室温付近または比較的低温で良質の膜が作製可能である．薄膜
全体の売上げ高の点で，PVDと比較し，装置がシンプルであるこ
とや大量生産に適していることから圧倒的に多い．
(１) 低温CVD技術
機械部品の高精度加工に伴い，材料の変形や変質防止のために，
より低温での成膜技術が望まれている．したがって，従来の熱
CVD技術に代わり，プラズマやレーザを用いた低温CVD技術開
発が注目されている．
(２) 高平滑膜作製
基板を平滑化する目的や精密機械のしゅう動特性への要求から，
ダイヤモンド状炭素（DLC:diamond-like carbon）が大きく脚光
を浴びている．密着性の問題もかなり改善され，高品質かつ高密着
強度のDLC膜の開発が注目されている．
