# 8・2・8 各成膜法の長所と短所

> **章**: 第8章 表面処理
> **節**: 8・3 化成処理，陽極酸化処理

CVD法とPVD法を選択するうえで重要なのは，処理温度の違
いである．CVD法では，約1000℃前後での処理となるため，超硬
合金材を除いては，再度焼入れ，焼戻しの処理が必要となり，金型
等の精密機器の精度を上げるのが困難となる．一般的には3/100
mm以上の許容公差のない金型にCVD処理を行うのは精度の面で
リスクが大きい．また密着性が良いこと，内面への付きまわりが良
いなどの利点があり，自動車向けの金型など大型のものに多く適用
.六価クロム系化成皮膜 一般的にはクロメート処理と称
されている．クロム酸または重クロム酸塩を主成分とする溶液中に
被処理物を浸漬してその表面に防食皮膜を生成させる方法で，耐食
性の向上や塗装の下地処理を目的として行われている．処理できる
金属は亜鉛，アルミニウム，マグネシウムおよびその合金などであ
るが，銅，銀などに処理すると変色防止の効果がある．
皮膜はクロム酸クロムを主成分とした非晶質で，生成直後は水を
含む膠状であるが，これを乾燥し，脱水させることにより強固な皮
膜が形成される．皮膜種類には光沢，黄色，黒色，緑色などがあ
り，電気亜鉛めっきをした自動車や家電の部品，建築用材などには
この処理を行うことが必須条件となっている．中性塩水噴霧試験に
よる耐食性試験を行った場合，亜鉛めっきのみでは短時間で白色腐
食生成物（塩基性炭酸亜鉛など）が発生するが，例えば黄色皮膜を
施したときには腐食生成物が発生してはならない最低時間は72時
間と規定されている（JIS H8625：1993）．処理方法には浸漬方法
以外にスプレーや電解方法もある．
.三価クロム系化成皮膜 上記の六価クロム系化成皮膜の
代替として開発されたもので，安全とされている三価クロムが主成
分となっている．耐食性など六価クロム系と比較しても遜色がない
とされ試験，評価も原則的には同じ方法で行われる．作業性やコス
トなどに問題はあるが処理装置も大きな変更を加えることなく転用
可能なので，環境対応に厳しい自動車や家電業界を中心として採用
部品は増えつつある．
b.リン酸塩処理（phosphating，phosphate treatment ) リ
ン酸塩を含む水溶液を用いて化学的に皮膜を形成させる方法で浸漬
あるいはスプレー，はけ塗り方法などにより処理される．おもに鉄
鋼材や亜鉛めっき鋼板を対象とした塗装の下地処理方法で素地金属
と塗膜との密着性向上を目的として行われる．処理液にはリン酸-
亜鉛，リン酸-亜鉛-カルシウム，リン酸-鉄系などの多種類が市販
されており，素材や塗料の種類，塗装方法などの違いによって選択
される．
一般的に皮膜の付着量（g/m ）が大になるほど耐食性は良好と
なるが，反面，ある領域を超えると塗料との密着性が劣ってくる傾
向にある．近年，塗料品質の向上に伴って皮膜付着量の少ない薄膜
型が多く採用されている．
塗装下地を目的とする以外にもリン酸塩皮膜の特性を生かした使
用法もある．例えばプレス時における延展性向上やきず防止，ある
いはリン酸-マンガン系皮膜をしゅう動面などに処理しておくと初
期なじみ性効果が期待できるとされる．また，リン酸-マンガン系
皮膜を処理した後に防錆油を塗布すると防錆効果が長期に保てる．
