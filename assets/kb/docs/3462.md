# 4・1・6 自動化生産システム構成と評価

> **章**: 第4章 自動化システム
> **節**: 4・1 生産システム自動化の体系

現在行われている自動化生産システムの構造について自動組立システムを例に考察しよう.生産ラインの構造はワークの流し方によって表4・2のように分類される.フォード生産システムは固定間隔の流れ生産方式(fixed flow)である.これが流れの制御としては最も堅い.ワークを停留させることを許せば,間隔が自由となり,ステーションごとの作業時間に違いをもたせられる.これが自由間隔の流れ生産方式(free flow)である.バッチ生産方式はあるまとまった生産量ごとに作業を行う方式で,めっき作業は30個の製品を同時に,穴あけ作業は4個ずつといった処理量の違いがあってもまとめて処理する.セル生産方式では通常,製品一つ一つを順番に作っていく.どの方式を選ぶかは生産量(月産量や日産量),製品寿命,生産変動の予測,などから選択される.ワークをベルトコンベヤで流す方式を採用したとすると,通常図4・4(a)に示すように一直線に組付場所(ステーション)が並び,1ステーションにおいて一つの部品をワークに組込む.多数の部品を組込む場合には,その個数分のステーションを設置する.しかし,複数の製品を同じラインに流す混流生産が必要になると,同じステーションで複数種類の部品を組立てることが求められ,図4・4(b)のように部品供給装置がステーションの周囲に集中する.この部品供給装置や部品箱,および保守作業のための面積(チョコ停が起こると作業員がその解除にしばしば部品供給装置に近づくことが必要)が大きくなって,一つのステーションでは多くの部品組付ができなくなる.そこで図4・4(c)のように製品ごとに素通りするステーションを置く方法も採用される.部品供給を重視して図4・4(d)のように部品供給用のラインを別途設置するシステムも登場している.このように生産装置には製品やその部品の供給はもちろん,工具・刃具の供給,エアや電気の供給が必須である.マシニングセン表4・2 部品流れの確保製品と部品との流れを確保する.チョコ停(choko-tei,自動化ラインで部品供給のつまりなど,簡単に復旧できる理由でラインが短時間停止すること.それ自体は短いが頻発するとラインの稼働率を低下させる)対応,部品供給などの保守作業スペースを確保する.(2) 作業方法の確立組立情報を指令し,複数部品を組立てる.(3) 検査と管理部品の組上がり状態を検査し,かつ,検査結果を生産管理システムへ伝達する.(4) 治工具,エネルギーの供給エアや電力などの供給.(5) ステーション間の連結が求められる.これらの組合せからなる生産システムには当然良し悪しがでる.生産システムの評価に確定したものはないが,経済的視点からは純利益/投資額なる投資効果が最も基本的である.分母の投資額には設備の固定費のみならず変動費を算入する.また,期間はその生産設備の生涯を考えるべきであるが,実際には製品寿命によって支配される.では,自動化した生産システムの評価はどう考えるのか.これも生産システムであるから,同じ投資効果で考える.しかし,自動化システムは非自動化システムに比較して,(1) 投資額が大きい(2) 生産量変化への対応性が低い(3) 製品対応性(flexibility)が低いと,投資リスクが大きくなる.つまり自動化生産システムはハイリスクハイリターン型の生産財投資といえる.それゆえ,自動化生産システムの活用には(1)~(3)のそれぞれを緩和することが求められる.7―75図4・5 自由に動き回るモジュラユニット(1)への対応としては,設備投資額を極小にするセル型生産システムがその極端な姿である.設備自体の価格とその立上りコストを下げ,製品の短寿命化への対応を可能とする.一方,最終的には多ステーションの設備が要求されるが,製品の販売初期段階では販売量が見えないような場合には,段階的な投資が可能な生産システムが工夫されている(デンソーのシステム).まず1台のロボットが働く自動化生産設備を開発し,そこでの作業方法を完成させる.1ステーションなので生産時間がかかる.多ステーション化するには,1台のロボットが実行してきたプログラムを分割して,設備台数を増やす.ちょうど一人の作業者がすべての組立作業を単独で行うセルから,複数作業者による流れ作業への展開である.具体的には,知能ロボットに作業をティーチングし,それを複数台へ分割している.(2)は機械化システムのもつ大きな問題である.一般に組立ラインに生産量の設計値があると,それに対して60~120%の範囲でしかペイしない.ラインの運転速度を120%にすることはできても,200%に増加させることは製品の品質・機械の保守の点からできないのである.逆に生産量を減らそうとしても,固定費からせいぜい半分までしか落とせない.この生産量の可変幅をダイナミックレンジと呼び,これを増大させる方法として,(2-1) 混流生産システム(mixed manufacturing system)(2-2) 再構成可能型生産システム(reconfigurable manufac-turing system)などが試みられている.(2-1)は同じ生産ラインに別種の製品を流すことで,一つの製品についての生産量変動を許そうとの考えである.自動車の組立ラインでは,同一機種でもドア形状,ハンドル位置などが異なる種類を流し,1台ごとに異なる組付部品・位置を指示に従い実行する.この考えは,相互に製造時期が重なる製品が多数ある場合には,多用されている.一方,(2-2)は生産ラインの配置自体を変化させることを目指したもので,例えば日産100台の製品製造ラインを500台に変更するには,他の余っている製造機械を持ち込んで連結すればよいという考えに立つ.そのために組立ラインの設置(再構成)が容易に短時間でできる必要がある.このシステムはまだ研究段階ではあるが,日立製作所が図4・5のようなモジュラ型組立システムをデモして見せた.その後,自律分散型生産システム(autonomous dis-tributed manufacturing system)の研究が行われており,例えばホロニックロボットシステム(holonic robot system)では,ロボットが相互に交渉しながら作業を自動割付し,マニピュレータ付移動ロボットが組立セルへの新たに参加・退出することを可能としている.この製造機器をラインに追加・削除する機能をプラグアンドプロデュース(plug and produce,新たな製造機器を接続するとすぐに製造するという考え方.パソコン分野でのplug and playと類似する)と呼んで提案し,実現している.(3)に対しては,多品種中少量生産への対応が継続して進んできた.特にロボットの導入は,作業点や作業方法の変化への対応を容7―76生産システム工学易にしたが,それに加えて,センサフィードバックによる作業実現が作業を安定化した.画像処理装置の導入による位置姿勢合わせ,作業後の検査,力制御による複雑な組立作業などがその例である.この例は多数あるので省略する.
