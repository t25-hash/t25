# 4・1・7 新しい自動化システム

> **章**: 第4章 自動化システム
> **節**: 4・1 生産システム自動化の体系

製品の多様化が進み、一つの製品についての月産量のみならず、総生産量が少なくなってきている現在、機械化の方法、自動化の方法ともに変革せざるを得ない。それぞれの分野での手法を述べる。a. 切削加工自動旋盤のような専用機はなくなり、MCにATCやAWCを付けたシステムが一般化してきた。その結果、AWCにワークをセットする作業が人的経済的負担となった。そこで、例えばロボットにより加工ジグへのワークの自動取付けを実現したシステムが使われている。ファナック知能ロボットセルがその例である。b. 塑性加工プレスあるいはインジェクションモールドなど型を使う生産加工は金型のコストがかかることから、大量生産にのみ向くとされているが、それを少量生産に適合させるために型交換の時間を極端に短くする方法が採用されている。プレス型の場合、外段取り(加工機械自身を止めることなく外部で交換の準備をすること)でシングル段取り(1けたの分となる時間)を切ることが一般化したが、それをもっと短くする試みがされている。プラスチック射出成形では型ならびに射出材料のプレヒート技術によって極少量生産への試みが始まっている。c. 組立少量生産への対応は人手による組立への回帰で実現してきた。設備投資を徹底的に抑制することで、人件費の安い国での生産に対応してきた。しかしながら、超小型電子部品の組立のように人手だけではできない組立では機械化自動化の流れは変わらない。その場合も、コンピュータによる高度な制御技術によって高速位置決めによる高精度の実現を図るなどで高効率化を図っている。一方、組立システム自身をよりフレキシブルにする試みも多数なされており、その典型として作業者の代わりに移動マニピュレータが組立ラインで働くシステムが実現されている。これはライン生産方式に知能ロボットを自律分散的に導入した好例である。d. 生産システム全体の監視ネットワーク技術の低価格と高機能化によって、生産機器をネットワークで接続することが可能となった。生産機器の接続規格(例えばFL-net)が議論されており、商業システムも出ている。現状ではまだ監視業務が中心で、遠隔の生産機器制御は安全性の視点から採用は少ない。
