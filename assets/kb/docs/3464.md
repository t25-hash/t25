# 4・1・8 終わりに

> **章**: 第4章 自動化システム
> **節**: 4・1 生産システム自動化の体系

4･1節では生産システムの自動化を取上げ，その体系を論じた．
生産の機械化は生産の高効率化，製品の高信頼性化などを求めて実
現されるのに対して，機械化された生産システムの自動化は基本的
に少数の人手で生産量を増すこと，すなわち，高効率化が目標であ
る．それゆえ，生産システムの中で改善効果の高い部分が選ばれて
改善される．現状はなお情報ハンドリングと情報獲得部分が重要視
されているが，その部分の改善が進めば，循環的に物流部分の自動
化が進むこととなるだろう．
