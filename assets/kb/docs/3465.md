# 4・2・1 生産形態の変遷と離散系生産システム

> **章**: 第4章 自動化システム
> **節**: 4・2 離散系生産システム

工業製品の製造方法は,製品に対する社会的な要求と技術の進歩に伴い,時代とともに変遷している.産業革命以降のもの作りは,1910年代に実用化された流れ(型)生産システムにより大きく変化し,大量生産(mass production)による工業化が加速的に進んだ.これは,自動車の組立工程に適用されたもので,組立作業を細かく分解し,単純化し,組立途中の製品が搬送されるベルトコンベヤの移動に合わせ,個々の作業を順番に直列的(ライン状)に並べる方法である.それぞれの作業時間(タクトタイム)を均等に短くすることにより,一つの製品をタクトタイムで効率良く組立てることができ,同じものを大量に安く早く作る方法として普及した.図4・6 生産形態と生産システムの変遷同じものを多く作れば作るほど生産コストが低下することは量産効果として機械加工工程でも古くから知られている.流れ型生産システムは,当初組立工程に適用されていたが,自動化技術およびNC(数値制御)工作機械の発達に従い,機械加工工程にも波及した.組立工程と同様に,加工対象物(工作物)をコンベヤなどで自動的に搬送し,単能機化した複数の工作機械をライン状に並べて加工を同時に行う生産設備が開発され,実用に供されている.このように工作物の搬送と複数の加工機械が一体化された生産設備は,トランスファラインあるいはトランスファマシンと呼ばれ,自動車やトラクタのミッションなどの大量生産に大きく貢献した.大量生産による工業製品の普及は,社会の経済状態に影響を及ぼし,顧客ニーズの多様化を誘発し,工業製品の寿命(ライフサイクル)の短縮にも結び付くことになる.そして,生産技術および情報処理技術の飛躍的な発達とともに,生産形態も変遷している.図で,一般的に製品寿命はa~eの順に短くなる.a. 少品種多量生産製品の種類に対して生産量が極端に多く,その変動が少なく,製品の寿命が長い場合であり,流れ型生産システムが適用されることが多い.b. 中品種中量生産製品の種類がある程度多く,各種類に対して中程度の生産量がある場合で,種類の変化に対応できる融通性(フレキシビリティ)が生産システムに望まれる.c. 多品種少量生産生産量に比べ,製品の種類が多く,製品寿命も短い場合で,量産効果が挙げにくく,流れ型生産システムなどが適用しにくい.d. 変種変量生産製品の種類も生産量も一定せず,短期間で変動を繰返すため製品寿命も予測しにくい場合であり,種々の変動に対処できる生産システムが要望されている.e. 一品・単品生産製品を個別に1個単位で作るため,量産効果が得にくく,製造コストが高くなる場合で,短期間に低価格で生産できる生産システムの構築が課題である.生産システムの主要構成要素である工作機械はNC化,高機能化が進み,搬送装置などの周辺機器と有機的に結合されシステムとして統合的に制御される方向で発展している.複数の生産機器を分散配置し,制御系で結合した設備は,離散(分散)系生産システムと称される.その代表例としてFMS が挙げられ,生産性に優れているが,融通性に劣る流れ型生産システムと対比されることが多い.離散系生産システムの運用,制御は,構成要素が多く複雑であるため,スケジューリングあるいはマネジメントが必要であり,シミュレーションによるシステムの解析,評価が重要となっている.
