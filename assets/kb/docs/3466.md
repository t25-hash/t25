# 4・2・2 離散系生産システムの構成要素

> **章**: 第4章 自動化システム
> **節**: 4・6 は,生産形態を製品の生産量と種類で概略的に比較したもの

図4・7は2台のマシニングセンタを無人搬送台車(AGV)で結合したFMS の例で,システムの構成要素は図中に示すように5種類に大きく分類できる.a. 工作機械(生産機械)生産システムの中核をなし,工作物を加工するために工具を有し,創成運動を行う機械である.工具図4・7 MCとAGV によるFMS(カーネイアンドトレッカ社)交換装置(ATC)と一体化し,CNC制御されたマシニングセンタなどが利用される.b. 加工エリア(作業領域)マシンエリア,ワーキングエリアあるいはワーキングステーションと称され,工作物を加工する場所を意味する.従来は工作機械のテーブル部分が対応していたが,自動パレット交換装置(APC: automatic pallet changer)との組合せ,インデックステーブルの利用による多面加工,などのように,工作機械のコラム側と加工エリアを分離した形態が増えており,a項の工作機械とb項の加工エリアを分けて扱う必要性が生じている.c. 着脱エリア(着脱ステーション)ロード/アンロードステーションで,物の取付け取り外しを行う機能と場所を示す.工作物の着脱,工具マガジンへの工具の装てんなど,物流と結び付いた働きをするところである.図4・7では工作機械の着脱エリアはパレット交換部に相当するので,b項の加工エリアと区別が難しい例もある.d. 搬送装置(搬送システム)工作物や工具など,物を移動する装置,システムを意味する.搬送方法と搬送経路が物流の特性を決めることになり,工作機械の配置方法とも密接に関連する.e. 保存エリア(ストックエリア)物を蓄える場所である.パレットプールや自動倉庫などと直結したFMS が開発されていることから,重要性が増している.図4・7では工作機械の前に設置されたパレットをプールする場所が保存エリアに対応する.また,AGV への着脱を行う場所は,パレット上の工作物やマガジン内の工具を一時保存する場所ともみなせることから,c項の着脱エリアとe項の保存エリアを兼ねる場合もある.生産工程は加工以外にも組立工程,検査工程があるので,組立機械,測定機などがシステム内に共存する場合があり,工作機械と同様に重視される.個々の生産機器はNC化され,集中管理方式ではなく,自律分散で制御される傾向にあり,相互に協調してシステム全体を制御する必要性が生じている.離散系生産システムでは稼働率,運用効率を種々の状況でシミュレーションしながら最適化するためのシステムコントローラが重要な役割を果たしている.
