# 8・4・3 レーザ焼入れ（laser quenching）および電子ビーム焼

> **章**: 第8章 表面処理
> **節**: 8・4 表面硬化法

入れ（electron beam quenching）
レーザおよび電子ビームを用いた表面硬化処理法としては，焼入
れ硬化，合金化，グレージングなどがある．局部的な加熱であるた
め，処理品はひずまないほか，電源を切るとただちに周辺部に熱が
吸収されるため焼入れた状態（self quenching）となり，冷却剤が
不要である．省エネルギー性にも優れ，クリーン熱処理として有望
である．また加熱源であるレーザや電子ビームの制御は数値コント
ロールが容易であり，加工ラインとの一体化を図ることもできる．
ただし，欠点として加熱場所が狭小で焼入れ幅が狭く，硬化層の深
さも他の方法と比べると浅い．広範囲な表面を焼入れるためにはス
キャニングをしながら加熱することが必要であり，一般には狭い領
域の焼入れや特殊部品に限定されて利用されている．処理設備とし
ても現在では標準機は少ない．
