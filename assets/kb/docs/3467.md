# 4・2・3 工作機械の配置と物流経路

> **章**: 第4章 自動化システム
> **節**: 4・6 は、生産形態を製品の生産量と種類で概略的に比較したもの

工作物の搬送経路は工作機械の配置方法に依存し、加工順序と関連して生産性に大きく影響することが知られている。工作機械の代表的な配置方法は次の3種類に大別でき、工作物の種類やロットサイズなど生産形態により物流が異なる。a. 機種別配置(functional layout)工作機械の種類ごとにまとめて配置する方法で、工作物の移動距離が長く、生産性、品質管理の点では劣るが、代替できる工作機械が隣接しているため、工作機械の故障に対処しやすく、保守管理が容易であることから、多品種少量生産に適している。7―77図4・8 工作機械、生産システム、配置方法の生産性、融通性b. グループ配置(group layout)異なる複数の工作機械をグループ(セル)としてまとめて配置する方法で、工作物の搬送距離が短く品質管理が行いやすい利点をもつ。グループ内での変更が可能で、全体を把握できることから、作業者に適した方法といわれている。類似部品をファミリーとして集めて加工を行うことで量産効果を得る方法であるGT(group technology)と対応しており、セル生産方式、GT セル生産とも称される。c. 直列(直線)配置(line layout)工作物の加工順序に合わせて工作機械を直列(直線)に配置する方法で、流れ型生産システムと対応し、搬送経路を短くすることができ、高い生産性が得られる。工作物の種類ごとにラインを構成する必要があり、1台の工作機械が故障するとライン全体が停止してしまうことから、融通性には乏しいとされている。図4・8は工作機械の配置方法が有する特徴を生産性、融通性で表した例で、図中には工作機械や生産システムについても概略的な位置関係を示している。前出の図4・6と比較するとわかるように、生産性と融通性は相反関係にあり、工作機械の配置方法は生産形態とも密接に関連している。図4・8では単体の工作機械を□で、複数の機械で構成されたシステムを○で示している。FMS はグループ配置のものが多く、生産性と融通性の向上を目標に種々のシステムが開発されている。トランスファマシン(TM : transfer machine)は搬送ラインに加工エリアを直結させた直列配置に対応しており高い生産性を上げている。これに対してフレキシブルトランスファライン(FTL: flexible transfer line)は搬送ラインと工作機械を多機能化し、TM より融通性を向上させたものである。一方、個別生産やバッチ処理を対象とした従来のジョブショップでは、機種別配置が一般的で搬送経路が複雑で自動化が難しい。
