# 4・2・4 FMS の分類と特徴

> **章**: 第4章 自動化システム
> **節**: 4・6 は、生産形態を製品の生産量と種類で概略的に比較したもの

フレキシブル生産システム(FMS : flexible manufacturing sys-tem)は1960年代後半から1980年代にかけて盛んに研究、開発が行われた。製品の変更や生産量の変動など、種々の変化に柔軟(フレキシブル)に対応し、おもに中品種中量生産の生産性向上を図ることを目標に、機械加工を中心に特徴的な生産システムが国内外で数多く実用化された。ところが、FMS に関する定義は明確にはされておらず、システムの規模や対象範囲なども異なるため、FMS を統一的に整理することは難しい。例えば、FMC(flexible manufacturingcell)がFMS と同じ意味で使用される場合もあるが、FMS の部分を構成するサブシステムとして位置付ける場合もある。ここでは、規模の小さいものをFMCとしてFMS と区別し、生産システムの構成要素として重要な、工作機械と搬送装置に着目した分類とその特徴を以下に示す。a. 中核の工作機械によるFMS の分類箱物あるいは角物形状の工作物に対しては、マシニングセンタ(MC: machining cen-ter)が主役として用いられ、図4・7に示したように複数のMCを7―78生産システム工学搬送装置で結合した形態が多い。このように通常の工作機械でシステム構成をする例もあるが、FMS に適合する工作機械も新たに開発されている。例えば、MCと加工エリアを分離することで、工作物への接近性および搬送装置への結合性が改良できることから、コラム移動型のMCが、FMS やFTLで活用された。軸物形状、丸物形状の工作物に関しては、NC旋盤のほかに回転工具も装着できるターニングセンタ(TC: turning center)が利用される。このほか、砥石の自動交換が行えるグラインディングセンタ(grinding center)、種々の歯車を加工できるギヤセンタ(gear center)、組立作業を1台で行うアセンブリーセンタ(assembly center)など、高度に機能を集約した新しい生産機器が開発された。図4・9に示す例は、複数の回転工具が組込まれたヘッドを動かすことにより、一度に多数の穴あけ加工ができる、ヘッド交換型工作機械(headchanger,head changing machine)を中核とするFMS である。ヘッドチェンジャとその他の工作機械をコンベヤで結合したシステムで、穴あけ加工が多い場合に適し、加工時間を短縮して生産性を向上できるが、ヘッドの設計変更が容易でなく製作費が高いという課題がある。図4・10の例は、日本の大型プロジェクト「超高性能レーザ応用複合生産システム」で開発試作された、三つのコラムで構成された複合加工機である。数種類の主軸ユニットなどが自動交換できる特徴をもち、高出力のレーザ加工とも一体化することにより、旋削、フライス、研削、歯切り、焼入れなどを可能にしている。このように、高度に自動化した生産システムの構築を目標として、中核となる工作機械の構造、機能を設計する場合が増加している。既製の工作機械をシステムに適した形態に改良・改善した事例、あるいは従来とは異なる新しい発想で設計される事例が、実用化されたFMS に多く見られることから、これらはシステム適合型工作機械とも称される。b. 工作物の搬送方法によるFMS の分類FMS 内の工作物や工具などの搬送経路に主点をおくと、直線型、ループ型、ランダムアクセス型の3種類に分けられる。図4・9のFMS では、コンベヤによる工作物の搬送部分に直線型、ヘッドチェンジャのヘッドを保管する部分にループ型が適用されている。直線型、ループ型では一図4・9ヘッド交換型工作機械で構成されたFMS(インガソルミリングマシン社)図4・10ユニット交換型複合加工機(大型プロジェクト:超高性能レーザ応用複合生産システム)般的に搬送の順番が規制され、追越しなどは行いにくいが、経路内に一時待避場所を設置したり、搬送経路を往復できる機能を付与することで順番の変更を可能にし、融通性の向上を図っている。図型より経路の冗長自由度が高く、工作機械に供給する順番を変更できる点が特徴である。図4・11は搬送装置としてロボットを利用したシステムで、4台の工作機械への工作物の搬入出を1台のロボットが担っており、搬送の順番を自由に選択できるランダムアクセス型である。搬送経路と工作機械の配置方法を比較すると、直線型、ループ型、ランダムアクセス型はそれぞれ直列配置、グループ配置、機種別配置と対比することができ、生産性や融通性など類似の特性を有している。搬送方法としては、無人搬送車(AGV、図4・7)、ベルトコンベヤ(図4・9)、ロボット(図4・11)、スタッカクレーンなどが挙げられる。図4・12は立体倉庫と一体化したFMS の例で、素材、半加工品、工具などを立体倉庫に保管し、必要なものを必要なときにスタッカクレーンで搬送し、工作機械に供給するシステムである。倉庫の搬送システムを利用することにより、加工エリアとストックエリアを立体的に融合し、省スペースを実現している。
