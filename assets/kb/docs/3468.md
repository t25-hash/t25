# 4・2・4 FMS の分類と特徴

> **章**: 第4章 自動化システム
> **節**: 4・6 は、生産形態を製品の生産量と種類で概略的に比較したもの

フレキシブル生産システム（FMS:flexible manufacturing sys-
tem）は1960年代後半から1980年代にかけて盛んに研究，開発が
行われた．製品の変更や生産量の変動など，種々の変化に柔
軟（フレキシブル）に対応し，おもに中品種中量生産の生産性向上
を図ることを目標に，機械加工を中心に特徴的な生産システムが国
内外で数多く実用化された．
ところが，FMSに関する定義は明確にはされておらず，システ
ムの規模や対象範囲なども異なるため，FMSを統一的に整理する
ことは難しい．例えば，FMC（flexible manufacturing
cell）がFMSと同じ意味で使用される場合もあるが，FMSの部
分を構成するサブシステムとして位置付ける場合もある．ここ
では，規模の小さいものをFMCとしてFMSと区別し，生産シス
テムの構成要素として重要な，工作機械と搬送装置に着目した分類
とその特徴を以下に示す．
