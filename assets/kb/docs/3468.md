# 8・4・5 その他

> **章**: 第8章 表面処理
> **節**: 8・4 表面硬化法

表面層の化学組成を変えずに，その部分の組織のみを変化させる
方法として，火炎焼入れ（flame hardening）法がある．アセチレ
ンガスの燃焼炎を金属材料の表面にのみ当て，変態点以上に加熱
後，すばやく焼入れる方法であるが，最近では特殊な用途に使用さ
れるのみとなっている．
表面層の化学組成を変化させる方法としては，炭素や窒素のよう
な比較的小さい原子を母材中に浸入させる方法とは異なり，比較的
大きい原子と母材原子とを置換させたり，反応させる方法がある．
その一つに硫黄を浸入させる浸硫（sulfurizing）法がある．一般
に，金属材料の表面硬化法では耐摩耗性と潤滑性の向上は両立させ
られない．しかし，浸硫法では表面層に可塑性のある硫化鉄を形成
させることによりこの矛盾を解決し，熱間用金型などの表面硬化法
として応用されている．その他，Si ，Al ，Cr ，Vなどを単独ある
いは複合で拡散させる拡散浸透法があり，これらは耐摩耗性，耐熱
性や耐食性に優れた表面層を形成する表面処理法として過酷な条件
下で使用される部品に応用されている．
