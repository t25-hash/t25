# 4・2・5 モジュラ構成と生産システムの再構築性

> **章**: 第4章 自動化システム
> **節**: 4・7 では,AGV の搬送経路を多重ループ化することで,単ループ

生産設備は導入経費が高いことから,レトロフィット,リユースされ,一般的にその耐用年数は長い.ところが,工業製品のライフサイクルが短くなるに伴い,生産システムとしてのライフサイクルも短くなる傾向にある.短命化した製品のライフサイクルに合わせて,柔軟に対応できる生産システムがFMS,FTLといえるが,シ図4・11 ロボット組込み型FMS(ノルウェー工科大学)図4・12 立体倉庫と一体化したFMS[西部電機工業(株)]図4・13 モジュラ構成と生産システムの再構築性ステム内の部分的な変更,組替えを可能にすることで,対象製品,生産形態の大幅な変更にも即応できる再構築可能な生産システム(reconfigurable manufacturing system)の実用化が検討されている.図4・13はモジュラ構成と生産システムの再構成,再構築性を概念的に示したものである.工作機械を部分的な要素(モジュール,ユニット)に分解し,これらを組合わせて工作機械を製造する考え方は,積木式構成法(building block system)やモジュラ構成法(modular construction)として古くから提案されている.標準化,規格化した主軸モジュール,コラムモジュール,テーブルモジュールなどを数種類ずつ準備しておき,それらを組合わせて,工作機械を短期間で設計・生産する方法である.同一機種だけでなく,旋盤系とフライス系など異機種へのモジュラ構成の適用が試みられている.図4・13に示すように,モジュラ構成は単体工作機械の構築を目的としているが,モジュールやユニットに搬送装置やロボットなどを含めることにより,セルレベルのFMCの構成,さらにシステムレベルのFTLやFMS の設計・生産にも適用できる.既存のシステムに対しても,余分なモジュールを取り除き,必要なモジュールを追加することにより,簡単に新たなFMS やFTLに組替え可能となる.異種モジュール間の結合性,互換性を実現するとともにシステム制御,運用ソフトウェアと総合することで,システムレベルでの変更を行ってもすぐに全体の立上げ稼働が可能となり,離散系生産システムとしての再構成・再構築性(reconfigurability)の利点が生かされる.生産のグローバル化や要求仕様の多様化に従い,大量生産時代を支えた大規模な生産システムは変ぼうし,最近では身軽(leanmanufacturing)で機敏(agile manufacturing)に対応できるシステムへの要望が高まり,再構築可能な生産システムの実現が今後の重要な課題となっている.
