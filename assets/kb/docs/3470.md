# 4・3・1 人間と生産活動

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

ものづくりの能力は人間が太古から備えた本能であり、生産活動は何千年もの間継続されてきた人間特有の知的活動であるといえる。生産の自動化が進展した現在においても、自動化技術を支えるのはやはり人間のもつ創造力やノウハウであり、また、最終的な意思決定を行うのもやはり人間である。a. 製品開発・企画製品開発の初期段階において、何を、どのように作るか、どのような構造、機能にするか、社会のニーズにマッチするかなどといった思索は、開発者の経験や知識に基づき発見的(heuristic)な方法で展開される。生産活動の第一に行われるこういった創造的活動は、自動化が困難であり、この段階における人間の役割は大きい。7―79b. 設計・評価設計・評価においては、CAD/CAEシステムなどの支援システムはあるが、人間が必ず関与して主体的に進めなければならない。標準品の仕様変更や改良設計を行う際には、人間の役割として創造的活動の割合は減るが、仕様に関して暗黙の了解の部分が多く、設計者がそれまでにもっている経験や知識が特に必要とされる。これまでにどれだけの設計に携わり、どれだけの事例を知っているかが、よい改良設計を行うための鍵となる。c. 生産・製造生産(production)は製品を生み出す全工程を意味し、製品を加工するという意味では製造(manufacturing)ということばが使われる。最近では、製造工程の多くが自動化されている。しかし、その設備の完成度を高めていくには、現場で働く人々の経験と知識で改善できる余地を残しておく必要がある。自動化工程はコンピュータの役目であって人間とは無関係であるといった考えは誤りである。自動化工程ではむしろ人間のものづくりに関するノウハウや創造力が不可欠なのである。他方、組立工程においてはほとんどが人の手に依存しているのが現状である。これは組立作業が、複雑で多岐にわたるため完全自動化が困難であることが考えられる。この工程は労働集約型工程と称され、重筋作業を長時間余儀なくされることもある。これに対して、作業分析が行われ、機械化による軽労化、温熱環境の改善、一定時間ごとに作業交代を行うジョブローテーションなどの対策が講じられている。
