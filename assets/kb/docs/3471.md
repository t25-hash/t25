# 4・3・2 人間中心生産システム

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

a. 無人化工場の矛盾上述のように,人間が生産活動に密接に関わっていることは明らかであるが,人間は疲労しやすく,不注意なミス(ポカミス)を犯しやすいといった弱点をもっており,産業革命以来,利益追求型の生産工場にとって好まれる存在ではなかった.そのため,大規模なFA,FMS,ロボットなどによる自動化・無人化を促進し,人間をできるだけ排除しようという流れができてしまった.しかし近年,自動化・無人化において以下のような矛盾が指摘されるようになった.(1) システムが複雑なため設備投資が増大し,損益分岐点が高くなる.(2) システムが硬直し,需要の変動や作業者の高齢化といった環境変化に柔軟に対応できない.(3) 技術の伝承が行われないため,次世代の生産システムの構築が困難となる.(4) ものづくりの喜びが感じられず,作業者の就業意欲を低下させる.(5) 付加価値の高いものができない.b. 人間中心生産システムの誕生このような背景のもとに,人間中心生産システム(human-centered manufacturing sys-tem)が誕生した.極端な例では,コンベヤラインを導入せず,極めて少人数で製品の1台を組立てる方式が採用されている.このような生産方式をいち早く取り入れたのが,ボルボ社のカルマー工場であることから,カルマー方式と呼ばれたこともある.日本では1990年代後半に,パソコンや携帯電話などの組立に1人1品生産方式を採用する企業が現れた.これはセル生産方式あるいは屋台方式と呼ばれる.このような方式が結果的に製品完成までの生産期間であるリードタイム(lead time)を短縮できた例もある.また,このような生産方式では巨大な設備投資が不要であること,熟練者を養成できること,需要の変動や作業者の高齢化といった環境変化にも柔軟に対応できることなどの利点がある.
