# 4・3・3 人間工学(ergonomics)の役割

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

先に述べたように人間と生産活動が密接に関わっていることから、人間の一挙一動が生産の善し悪しを左右することは容易に推察できる。ここで生産システムにおいて人間を排除しようとするのではなく、人間の健康と働きがいを保証しながら、人間を有効に活用し、生産性の向上に役立てようとするのが人間工学である。7―80生産システム工学a. 人間工学の定義ヨーロッパ諸国では人間工学はエルゴノミクスまたはアーゴノミクス(ergonomics)と呼ばれている。これはギリシア語のergo(work)とnomos(law)の合成語であり、人間と仕事の間の生理・心理法則を意味する。一方アメリカでは、ヒューマンファクターズエンジニアリング(human factorsengineering)と呼ばれ、人間と機械のマッチングを工学的観点から検討を行うことにウエイトがおかれている。人間工学の定義は、一意に確定したものはないが、人間が使う道具・機械・システム、または人間が行う作業・活動を人間の諸特性に適合するように、これらを使いやすく、または作業しやすくするための基礎的および応用的な技術・方法論であると定義される。b. 研究課題生産システムのほとんどが人手に頼っていた時代、あるいは上述の人間中心生産システムにおいては、身長、可動範囲、姿勢、動作など人間の物理的諸機能を基礎として設計・評価を行うことが人間工学のおもな内容である。昨今の機械化・オートメーション化社会では、人間と機械のマッチング、人間と機械の共存・共栄、ヒューマンエラー分析などが人間工学の対象となる。さらにIT 革命の名のもとに発展したコンピュータ情報化社会では、人間性疎外、メンタルワークロード、テクノストレスなどの問題が人間工学の対象となる。
