# 4・3・4 人間工学的技法

> **章**: 第4章 自動化システム
> **節**: 4・3 人間工学と生産システム

人間工学的技法とは、機械あるいは作業と人間の適合性を検討するための手法であるが、それは人間工学独自の手法ではなく、医学、生理学、心理学、生体工学などから関連のある手法を抽出し、複合化したものであるといえる。以下は生産システムにおける作業特性を検討するうえでおもに用いられる人間工学的技法である。a. 生体計測生体計測には、身体寸法計測、姿勢・動作計測がある。これらに基づいた計測値は、作業者にとって作業しやすい作業場の設計に役立てられる。例えば、作業台の高さやコントロールパネルの位置などを決定する際に用いられる。身体寸法計測(anthropometry)は、機械的な測定器による従来のマルチン式に代わって、最近はレーザ式計測が主流となってきた。これによれば、多項目の測定値が高速にかつ正確に集められる。例えば、3D 注視点(fixation point)計測目は口ほどにものをいう」とことわざにあるように、眼の動きや注視点は注意力の欠如や関心事など、人間の内面を推し量る手掛かりとなる。計測にはアイカメラが用いられる。これは角膜突出部に発光ダイオード(LED)の光線を当て、その反射光をとらえて注視点を記録する。同時に視野に入った光景を超小型ビデオカメラでとらえ、その光景上に反射光を重ね合わせてVTR に収録する。c. フリッカ検査(flicker test)人間の中枢性疲労、意識水準や覚醒水準などを評価する検査である。検査の値は眼や身体の疲労の代表値として用いられる。検査にはフリッカ値測定器を用いる。LED による点滅光を点滅周波数の高い状態(60Hz)から徐々に下げていくと、ある周期のところで点滅を知覚するようになる。この周期を下降法によるフリッカ値(CFF: critical flicker frequency)といい、単位はHz で表す。通常30~40Hz の値をとるが、大脳皮質の活動水準や年齢などによって変化する。d. 筋電図筋電図(EMG : electromyogram)は作業姿勢や動作における筋負担を評価する一つの指標である。筋肉は収縮すると微弱な筋放電が起こる。これを皮膚表面から電極を通して測定し増幅して、正負に振動する電圧波形が得られる。筋に大きな負荷がかかると、筋収縮が増大し、筋電図の振幅は増大する。したがって、筋電図の振幅から筋負担の大小が推定できる。筋電図を時間積分して単位時間当たりの総放電量を求めたり、一定時間ごとの総放電量をプロットする積分筋電図(I-EMG)を求めたりして評価する。また、筋疲労に伴って筋電図の高周波成分が除波化される傾向にあるため、周波数成分を求めたり、平均周波数を求めたりして筋疲労を評価する試みも見られる。最近の筋電図測定器には、皮膚の表面電極からの信号を無線で受信できるものがあり、それを用いることによって被測定者は配線を気にしなくてよくなり、行動範囲も広がった。e. NASA-TLX(NASA task load index)アメリカのNASA で開発されたメンタルワークロードの主観的作業負荷評価尺度であり、広く使用されている。被験者は作業終了時に六つの尺度について評価し、目盛の付いた評価スケールに印をつける。六つの尺度名は、(1)mental demand(知的・知覚的負荷)、(2)physical demand(身体的負荷)、(3)temporal demand(時間的切迫感)、(4)performance(作業成績)、(5)effort(努力)、(6)frustration level(不満度、ストレス、いらいらの程度)である。これらの評価を0~100点に換算して素点とする。次いで六つの尺度を二つずつ組合わせ、どちらの項目がワークロードへの寄与が高いかを被験者に選択させる。15対の組合せに対して選択が終わると、六つの尺度のそれぞれが選択された数の合計を求め、それを各尺度の重み係数とする。最後に、先の素点の重み付け平均値(WWL: mean weighted workload score)を総合評価点とする。
