# 8・2・4 セラミック繊維（ceramic fiber )

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

繊維で強化したセラミック基複合材料では，強化繊維にセラミッ
ク繊維が用いられることが多い．これらの複合材料の成形は高温で
なされることがほとんどであるため，強化に用いられる繊維には高
温における長時間安定性が重要な因子となる．これらの点から，現
状で用いられている繊維としては，アルミナ（AlO ）系の繊維，
炭化ケイ素（SiC）系の繊維，炭素系の繊維，窒化ケイ素
（SiN ）系の繊維などがある．これらの繊維の直径は，太いもの
でも数十 mである．そして，これら繊維は，物理的な長さの観点
から長繊維系とウィスカ系に分類される（表8･7)．
セラミック系繊維の大きな特徴は，強度および弾性係数が高温域
まで低下しないこと，および，熱膨張係数が金属系の繊維に比べて
小さいことにある．一方，破壊の観点から見たとき，延性が乏しい
ことや靱性が低いという欠点も有する．代表的なセラミック繊維単
体の物理的，機械的性質を表8･7に示した ．
