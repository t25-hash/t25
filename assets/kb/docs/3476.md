# 8・3・2 力学的特性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

構造材料として用いられる複合材料は，連続繊維を一方向に配向
した薄い層（単層板あるいはラミナ）がいろいろな角度に積層され
た多方向積層板として用いられることが多い．実際にはプリプレグ
（prepregs ）と呼ばれる0.1～0.2mmの厚さの中間基材を重ね合
わせ，オートクレーブやホットプレスで成形する．まず，複合材料
の力学的特性を知るうえで重要である，一方向単層板（ラミナ，
lamina）の力学的特性を考える．表8･8に各種プラスチック基複
合材料の力学的特性を示す ．
繊維方向弾性率は高強度炭素繊維を強化材とする場合には，
130～170GPaと鉄鋼より若干小さいが，高弾性率炭素繊維を用い
たときには，それを超える値になる．一方，繊維直角方向弾性率は
6～9GPaと低く，大きな異方性を有する．この異方性の度合いは，
ガラス繊維複合材料よりカーボン繊維複合材料のほうが大きい．ま
た，繊維方向の強度は従来の工業材料と比較して非常に大きい値に
なっている．異方性の度合いは弾性率よりさらに大きい．繊維方向
の破断ひずみは，繊維の破断ひずみでほぼ決まるので金属材料に比
較して低い．繊維方向の熱膨張係数が小さいので，寸法安定性の要
求される用途に使用される．
構造物を設計する際には，強度，弾性率そのものではなく，強度
を比重量（単位体積にかかる重力）で除した比強度，弾性率（剛
性）を比重量で除した比弾性率または比剛性が重要になる．表8･9
に各種材料の値を示す．従来の材料では，比強度を4×10m，比
剛性で4×10 mを超えることが困難であった．しかし，一方向強
化複合材料ではこれらを大きく上回っている．ただし，その優位性
は，擬似等方性積層板（quasi - isotropic laminates ）や織物材では
低下する．
