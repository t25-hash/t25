# 4・3・7 人間工学(ergonomics)関連ISO

> **章**: 第4章 自動化システム
> **節**: 4・14 )を使用する。調査用紙中のI群は眠気とだるさに関する訴

企業における生産システムを考えるとき、ISO9000(品質)、ISO14000(環境)などの規格を組入れることが要求される。近年、企業における人間工学的配慮が重要視されつつあり、人間工学関連のISOも続々と制定されつつある。ISO6385:2004「作業設計のための人間工学の原則」、ISO10075:1991「精神的作業負荷に関する人間工学の原則」、ISO13407:1999「インタラクティブシステムの人間中心設計過程」などがその例である。また、現在規格作りが進められているTC(technical committee)やSC(sub-committee)には次のようなものがあり、こういった動向も視野に入れておく必要がある。(1) TC94「個人用安全・保護衣及び保護具」(2) TC108「機械振動と衝撃」(3) TCSC146「大気の質」(4) TC159「人間工学」SC1「人間工学の指導原理」SC3「人体測定と生体力学」SC4「人間とシステムのインタラクション」SC5「物理的環境の人間工学」
