# 8・3・3 破壊靱性

> **章**: 第8章 複合材料
> **節**: 8・3 プラスチック基複合材料

衝撃後圧縮強度試験（compression after impact test ）の結果が層
間破壊靱性（interlaminar fracture toughness ）と良い相関関係に
あることから，材料開発においては簡便な層間破壊靱性試験を利用
するようになってきている．モードⅠの層間破壊靱性はDCB
（double cantilever beam）試験で評価される．試験方法はJISや
ISOに規格化されている．ISO規格では，まず初期負荷によりき
裂を発生させ，いったん除荷した後再び試験を行う．初期負荷時，
再負荷時双方について，非線形点，目視によるクラック発生点，最
大荷重点のデータから靱性が算出される．再負荷については，さら
に5%オフセット点，クラック伝ぱ時の靱性も求める．モードⅡの
試験方法は，日本ではENF（end notched flexure）試験がJIS規
格に取り入れられている．この試験の場合，通常の荷重点変位を単
調増加させる方法では，クラックが伝ぱし始めた後は不安定な破壊
になる．一方，ヨーロッパではELS（end loaded split ）試験が
ESIS（European structural integrity society）の規格案になって
いる．この場合も，荷重線変位を単調増加させる方法では不安定な
破壊になる．ISOの検討グループでは，4ENF（4point ENF）試
験を採用する方向である．この方法では，クラック進展に際し，破
壊靱性の急激な低下がない限り，安定な試験が可能となる．モード
ⅠとモードⅡが混合した混合モード試験は，ELS試験を応用する
もの，ENF試験を変形したMMB（mixed mode bending）試験
等が考案されている．モードⅢの試験法はまだ確立されていない
が，ECT（edge crack torsion）試験という方法が提案されてい
る．
