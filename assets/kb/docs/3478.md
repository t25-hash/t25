# 4・3・9 生産システムにおける人間工学の応用事例

> **章**: 第4章 自動化システム
> **節**: 4・14 )を使用する。調査用紙中のI群は眠気とだるさに関する訴

実際の工場における作業姿勢(work posture)の改善を目的とし、長町らは作業姿勢から10タイプの基本姿勢を抽出し、それらについての力学モデル、筋電図、HR(heart rate、心拍数)、RMR(relative metabolic rate、エネルギー代謝率)、心理的つらさの5種の相関研究から、表4・5に示すような作業姿勢ごとにつらさ指数を求めている。次に中型クーラの組立ラインに生じる作業姿勢をVTR で撮影し、表4・5で分類された作業姿勢ごとに発生頻度を集計した。その結果は図4・16に示すようになり、図4・16で無理な姿勢に相当するNo.4以上の姿勢を軽減するため、ホイストによる重量物の移送や高低自在のプラットフォームなどの導入による対策がとられた。結果的に無理な姿勢の頻度が減少し、生産性が増大した。他の企業ではこの考え方をさらに発展させ、作業姿勢のつらさを得点化するとともに、1日に運搬する量の総重量点や継続時間などのパラメータから、作業強度基準を設定し、作業設計や作業改善に利用している。図4・17は自動車メーカで設定された中高年齢者(middle aged and elderly people)適作業条件基準の例である。図中の評価基準値は60歳まで作業ができる基準として設定された。実際の作業工程を調査し、14項目すべてについて適合している作業を中高年齢者に適した作業であると判定された。
