# 4・4・1 生産システムの新しい概念

> **章**: 第4章 自動化システム
> **節**: 4・4 新しい生産システム

生産システムの現状には、問題点として、(1)製品の多様化への対応の柔軟性に欠ける、(2)生産設備更新やシステムの複雑化への融通性に劣る、(3)設備故障やシステムの異常発生への対処が困難である、(4)人間の役割や自然環境への配慮が十分ではない、などがある。これらの問題点の背景として、ライフスタイルの個性化、文化の多様化、生産活動のグローバル化、自然環境負荷の増大などが挙げられるが、それらは端的にいえば、生産システムの内外の質的量的変動にいかに迅速に対処できるかということに集約できる。このような認識のもとに、新しい生産システムの概念の模索が1990年代以降、特に、知的生産システム(IMS : intelligent man-ufacturing system)国際共同研究プログラムを中心に盛んに行われてきた。例えば、アジャイル生産システム(AMS : agile manufacturingsystem)、自律分散型生産システム(ADMS : autonomous dis-tributed manufacturing system)、フラクタル企業(FC: fractalcompany)、ホロニック生産システム(HMS : holonic manufac-turing system)、生物指向型生産システム(BMS : biological7―83図4・18 自律分散システムの位置付けmanufacturing system)などの新しい概念が提唱されている。このうち、アメリカで提案されたアジャイル生産システムは、消費者の要求に起因する急速かつ予測不可能な変化に迅速に対応するために、企業活動において業務、作業、人的資源に評価尺度を提供することを求めている。例えば、製品の複雑さおよび市場の不確実性の二つを指標にして、両者がともに小さい大量生産型製品から、ともに高いアジャイル生産へと進化することを示し、その戦略を考察している。また、ドイツで提案されたフラクタル企業は、フラクタルと呼ばれる自己相似型の自律的ユニットで構成される組織構造を有し、(1)企業文化レベル、(2)戦略レベル、(3)社会心理レベル、(4)財務レベル、(5)情報レベル、(6)技術レベル、の6階層からなるモデルにより、生産環境の変動に対する適応性を高めようとする概念である。このうち、アジャイル生産システムは、環境変動に迅速に適応する生産を目指すという、いわば目標概念の提示であり、その達成方法を示しているわけではない。また、フラクタル概念は自律的要素から構成されるシステムによる企業活動の目標達成を目的としているが、その方法論は必ずしも明確ではない。これらに対して、自律分散型生産システムや、ホロニック生産システム、生物指向型生産システムは、具現化への原理とアプローチの手法を明示している。それらに共通しているのは、伝統的なコンピュータ統合生産(CIM)に依拠した集中管理ではなく分散管理を原理としていることである。以下では、自律分散型システムの基本的特徴を述べ、次いで、それに関連の深いホロニック生産システム、さらに、自己認識、自己組織性、進化性、適応性などの生物的特徴を取り入れた生物指向型生産システムについて述べる。これらは、分散型人工知能、人工生命、進化システム、学習システム、創発システムなど、1990年代に急速に発展した研究と連動しながら、研究・開発されてきたものである。
