# 4・4・1 生産システムの新しい概念

> **章**: 第4章 自動化システム
> **節**: 4・4 新しい生産システム

生産システムの現状には，問題点として，(１)製品の多様化への
対応の柔軟性に欠ける，(２)生産設備更新やシステムの複雑化への
融通性に劣る，(３)設備故障やシステムの異常発生への対処が困難
である，(４)人間の役割や自然環境への配慮が十分ではない，など
がある．これらの問題点の背景として，ライフスタイルの個性化，
文化の多様化，生産活動のグローバル化，自然環境負荷の増大など
が挙げられるが，それらは端的にいえば，生産システムの内外の質
的量的変動にいかに迅速に対処できるかということに集約できる．
このような認識のもとに，新しい生産システムの概念の模索が
1990年代以降，特に，知的生産システム（IMS:intelligent man-
ufacturing system）国際共同研究プログラムを中心に盛んに行わ
れてきた．
例えば，アジャイル生産システム（AMS:agile manufacturing
system)，自律分散型生産システム（ADMS:autonomous dis-
tributed manufacturing system)，フラクタル企業（FC:fractal
company)，ホロニック生産システム（HMS:holonic manufac-
turing system)，生物指向型生産システム（BMS:biological
図4･18自律分散システムの位置付け
manufacturing system）などの新しい概念が提唱されている．こ
のうち，アメリカで提案されたアジャイル生産システムは，消費者
の要求に起因する急速かつ予測不可能な変化に迅速に対応するため
に，企業活動において業務，作業，人的資源に評価尺度を提供する
ことを求めている．例えば，製品の複雑さおよび市場の不確実性の
二つを指標にして，両者がともに小さい大量生産型製品から，とも
に高いアジャイル生産へと進化することを示し，その戦略を考察し
ている．また，ドイツで提案されたフラクタル企業は，フラクタル
と呼ばれる自己相似型の自律的ユニットで構成される組織構造を有
し，(１)企業文化レベル，(２)戦略レベル，(３)社会心理レベル，
(４)財務レベル，(５)情報レベル，(６)技術レベル，の6階層から
なるモデルにより，生産環境の変動に対する適応性を高めようとす
る概念である．
このうち，アジャイル生産システムは，環境変動に迅速に適応す
る生産を目指すという，いわば目標概念の提示であり，その達成方
法を示しているわけではない．また，フラクタル概念は自律的要素
から構成されるシステムによる企業活動の目標達成を目的としてい
るが，その方法論は必ずしも明確ではない．これらに対して，自律
分散型生産システムや，ホロニック生産システム，生物指向型生産
システムは，具現化への原理とアプローチの手法を明示している．
それらに共通しているのは，伝統的なコンピュータ統合生産
（CIM）に依拠した集中管理ではなく分散管理を原理としているこ
とである．以下では，自律分散型システムの基本的特徴を述べ，次
いで，それに関連の深いホロニック生産システム，さらに，自己認
識，自己組織性，進化性，適応性などの生物的特徴を取り入れた生
物指向型生産システムについて述べる．これらは，分散型人工知
能，人工生命，進化システム，学習システム，創発システムなど，
1990年代に急速に発展した研究と連動しながら，研究・開発され
てきたものである．
4・4・2自律分散型生産システム（autonomous distributed
manufacturing system）
自律分散システムは森ら により提唱された概念であり，自律
性を有する構成要素が複数集まり，相互に協調することによって全
体として機能を維持するシステムである．図4･18 は，自律可制
御性と自律可協調性を軸に，従来システムとの比較で自律分散シス
テムの位置付けを示したものである．自律可制御性が高くなると耐
故障性が大きくなり，自律可協調性が高くなると機能効用が高くな
る．そのため，自律分散システムはあるサブシステムの故障や拡張
に対して残りのサブシステムが自律的に制御・協調することによ
