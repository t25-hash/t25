# 8・4・1 金属基複合材料の構成と組織

> **章**: 第8章 複合材料
> **節**: 8・4 金属基複合材料

金属基複合材料（metal matrix composite）は主としてセラミ
ック質の繊維や粒子を金属の母材（マトリックス，matrix）に埋
込んで複合させ，優れた機械的性質を発揮させるように人工的に作
った材料である．マトリックス金属に比べて，高比強度，高比弾性
であることを特徴とする．1959年にアメリカ空軍研究所
（AFML）でタングステンの心線にボロンを化学蒸着させて作っ
た，いわゆるボロン繊維が開発されて以来，1970年代にかけて炭
素繊維やシリコンカーバイド繊維などのセラミック繊維（ceramic
fiber ）が次々と開発され ，工業的規模で生産されるようになっ
た．金属基複合材料はセラミック繊維の開発とともに発達した．今
日，表8･14に示すような繊維や粒子が金属マトリックスの強化用
として使用される．表中の金属繊維も一部用いられることがある．
強化材（reinforcement ）の形態としては，円形断面形状をした
細長い連続繊維，長さ数mmの短繊維，連続繊維をチョップした
短繊維，ウィスカ（whisker ）に代表される針状結晶および粒子が
ある．これらがマトリックス金属に埋込まれて，光学顕微鏡的には
図8･3のような組織を呈する．図8･3(ａ)は連続繊維で一方向に強
化したもので，最も大きな強度が得られるが，性質に大きな異方性
がある．図8･3(ｂ)は短繊維またはウィスカが三次元ランダム方向
に配向された強化材である．繊維強化効果は低いが，ほぼ等方均質
的な性質を示す．これを圧縮成形するに従い，繊維は二次元ランダ
ム方向に配向し，ある面内で等方性を示す．また強い引抜きあるい
は押出し加工をすると繊維が一方向に配向し，一方向短繊維強化材
となる．図8･3(ｃ)はいわゆる粒子強化材であり，機械的，物理的
性質は均質等方的であり，強度はマトリックスの性質を強く反映す
る．複合材中の強化材の存在量は，通常体積含有率で表す．繊維体
積率と配向性は所望の性質を発揮させるためにあらかじめ材料設計
されるが，体積率は製造上，実用上の制約によって，通常5～50 %
である．
金属基複合材料のマトリックス金属としては，軽金属のアルミニ
ウム，マグネシウム，チタニウムとその合金が多用される．300℃
までの用途には，鋳造用アルミニウム合金AC1C，AC4C，
AC8A，AC8B，展伸用アルミニウム合金6061，2024，5056，
2024，7075などがある．600℃までの用途にはチタン合金Ti -6 %
V-4%Al ，Ti -15%Mo-5%Zr -3 %Al ，Ti -5%Al -2.5%Snなどが
ある．このほか，さらに高温特性を改善するため，金属間化合物
（intermetallic compounds ）TiAl ，NiAl ，MoSiなどや超合金
Fe- Cr -Alなどをマトリックスとする超高温用金属基複合材料が研
究途上である．
