# 4・4・3 ホロニック生産システム(holonic manufacturing sys

> **章**: 第4章 自動化システム
> **節**: 4・4 新しい生産システム

tem)-ホロニック生産システムの基本概念は、IMS プログラムのHMS プロジェクトで検討され、提案されたものである。その中心概念となるのはホロン(holon)である。ホロン(全体子)とは、生物の階層システムで見られるように、それ自身が全体性と部分性を有するサブシステムのことである。上位に対しては部分として振舞い、下位に対しては全体として振舞うサブシステムである。ホロニック生産システムでは、図4・19に示すように、生産システムの構成要素、例えば生産設備、搬送設備、工作物などをホロンと呼び、これらのホロンが自律的に意思決定を行うとともに協調しながら生産プロセスを実行する。ホロンは、以下の性質をもつ。a. 自律性外部から与えられた仕事を達成するための行動計画を自ら立て、かつ自ら状態を監視しつつ作業を遂行し、異常事態が生じた場合は自ら必要な処置をとる。ホロンは、図4・19に示すように、情報処理を行う部分を必ずもち、必要ならば生産プロセスを実行する部分をもつ。b. 協調性外部から与えられた仕事を達成するための行動計画を複数のホロンで立て、かつ複数のホロンで状態を監視しつつ作業を遂行し、異常状態が生じた場合には必要な処置をとる。さらに、ホロンの間には、ホラーキーと呼ぶ階層構造がある。ホラーキーは、システムを構成する要素間の包含関係を表すものである。すなわち、複数のホロンが集まったものが、一つの集合体を構成し、この集合体が一つのホロンとして振舞う。ホロンの概念を企業レベルで考えると、図4・20に示すよう図4・19生産プロセスと情報処理部のホロン構成図4・20 ホロニック生産システムの概念に、工場の中には、設計・資材管理・製造の各部門があり、さらに製造部門の中には、工程設計課・日程計画課・加工機械セル・組立セルなどが存在する。こうしたサブシステムはホロンとしての性質をもち、上下構造として構成されていると同時に内部では相互に調整しながら生産活動を実行する。さまざまな要因を背景に市場の複雑で急速な変動に対応すべく、レスポンスの良いダイナミックな生産形態が望まれている。しかしながら、起こりうるすべての事態を事前に想定して生産計画やスケジュールを組むことは困難であり、中央集中型の伝統的CIM のような方式には限界があるのは繰返すまでもない。換言すれば、発生以前の事象が対象問題フレームに含まれていることを前提とするモデルベース型のアプローチには限界がある。それゆえ、システム内外の未知環境を認知対象として扱い、直接的な大域制御が困難であることを前提としたうえで展開されるべきである。このような問題への有望なアプローチの一つが、生物の適応戦略に着想を得て提案された生物指向型生産システムであり、4・4・4項で述べる。
