# 4・4・4 生物指向型生産システム(biological manufacturing

> **章**: 第4章 自動化システム
> **節**: 4・4 新しい生産システム

system)生物指向型生産システムは生物に学び生物と共生する生産システムである.自己認識,自己成長,自己組織化,進化,学習などの生物的機能を人工物に導入し,人間と人工物の相互作用を考え,さらにはマクロ生態系の中で人工システムをとらえる概念であり,製品の計画から設計,製造,運用,修復,再利用,廃棄までのライフサイクルを包括する.生物では内在する遺伝子型(DNA)情報と内在化した知識型(BN)情報が個別的な実体に融合しており,それにより自律性(保存性)と多様性(発展性)という相反する機能を発現している.DNA は世代を超える集団進化を司り,BN は個体の生涯における学習を司る.これらの進化と学習は生物システムの適応戦略の基本となる.このような特性を人工物にもたせて,従来の生産システムの限界を超えようとするのがBMS である.BMS の理論構築には,進化的計算,強化学習,人工生命などの創発的方法論(emergent methodology)が有効である.創発とは,要素間の局所的な相互作用により大域的挙動が現れ,その大域的挙動が要素の振舞いを拘束するという双方向の動的過程を通して,新しい機能や形質,行動を発現する秩序が形成されることととらえる.創発は,明示的局所性から非明示的大域性の生成という特徴をもつことも示す.創発システムは,先に述べた自律分散システムに共通したところもあるが,自律分散システムが全体システムからサブシステムへのトップダウン的分散を基本とするのに対し,創発システムでは部分の記述から始め,その相互作用から全体システムの特性が現れるとするボトムアップ(bottom-up)に特徴がある.前者が変化への機能維持を主眼にするのに対し,後者は機能発現を主眼とするという違いがある.図4・21は,生物指向型生産システムの工場フロアレベルのモ図4・21 生物指向型生産システムの概念デルを示す.システムの構成要素は自律した生物指向エージェントである.工作物や部品,製品などの生産対象エージェントはDNA型情報をもち,工作機械や検査機械,ロボットなど生産設備エージェントはBN 型情報をもつ.生産対象エージェントは自身が何をしてほしいかを知っているだけであり,生産設備エージェントは自身に何ができるかを知っているだけである.このようなシステムは事前に互いの情報を知る必要がない.両者の相互作用の結果としてシステムの挙動が自己組織化され,設備の空間的秩序としてのレイアウトや,プロセスの時間的秩序としてのスケジュールが定まる.したがって,生産要求変動や設備故障・刷新という変化へのロバスト性が高いということになる.このような考えで構築した,自動車溶接工場のシミュレーションの例を図4・22に示す.この例では,処理能力の異なる2種類の溶接ロボットが合計65台,AGV が25台,製品は20車種,製品投入口は3箇所,製品回収口は1箇所である.製品の溶接箇所から放たれたポテンシャル場により溶接ロボットが引き寄せられ,製品と設備の配置が動的に自己組織化(self-organization)され,プロセスが進行している様子がわかる.溶接ロボットの故障や製品要求の変動があっても,外部から直接制御することなく,自己組織的に適応することができる.このようなシステムは,伝統的なライン型生産と異なりラインレス生産システム(lineless manufacturingsystem)と呼ぶことができる.
