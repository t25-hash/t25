# 8・4・3 機械的性質

> **章**: 第8章 複合材料
> **節**: 8・4 金属基複合材料

金属基複合材料は開発が始まってからまだ日が浅いために，材料
としての規格や標準試験法は確立していない．金属基複合材料の特
徴は，(１)軽量で高強度である，(２)耐摩耗性がある，(３)耐特殊
環境性（原子状酸素，放射線，高真空など）がある，(４)電気や熱
の良伝導体である，などがある．これらの特徴を生かして，機械や
装置の軽量化ならびに輸送機械の省エネルギー化に新しい解決法を
提供する．

b.一方向連続繊維強化材（unidirectional continuous fiber
reinforced composites ) 一方向連続繊維強化材の特徴は繊維方
向の強度と弾性率が大きいことである．表8･16にボロン繊維，化
学蒸着によって作られたシリコンカーバイド繊維（SiCと記す）
および炭素繊維で強化したアルミニウム合金ならびにチタニウム合
金の機械的性質を示す．ホットプレス法で作られたものは概して複
合則で予測される値に近い強度（1～2GPa）と弾性率を発揮する．
破断伸びは強化繊維の破断伸びにほぼ等しい値（約2%）をとる．繊
維と直行する方向の強度はマトリックス金属の強度と同程度である．
一方向連続繊維強化材の引張試験をすると最大荷重点からほぼ非
累積的破壊（non-cumulative fracture）が観察される．最大荷重
点に達する前には弱い繊維の先行破壊とマトリックスの塑性変形を
伴うことが金属基複合材料の特徴である．この先行破壊によって繊
維の負担していた荷重がマトリックスに伝ぱし，伝わった応力の集
中は金属マトリックスの塑性変形を伴い，応力が緩和される機構が
働き，複合材の強度の信頼性（strength reliability）は繊維強度の
信頼性より改善される ．
ボロン繊維/6061アルミニウム合金製の管材は，軽量高強度を生
かして，いち早くスペースシャトルのトラス構造材に使用された．
図8･6のようにスペースシャトルの胴体部に243本取付けられ，
145kgの重量軽減を達成した ．
焼成SiC繊維/アルミニウム合金複合材は直径約0.5mmのプリ
フォームワイヤとして供給されるが，これをより合わせ，さらにア
ルミニウムと複合させて高圧送電線としての用途が開発されてい
る ．
連続炭素繊維/マグネシウム合金複合材は複合則から予測すると
最も高い比強度と比弾性を実現する系であるが，製造法と耐食性に
問題を残している．
