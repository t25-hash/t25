# 4・4・5 その他の新しい概念

> **章**: 第4章 自動化システム
> **節**: 4・4 新しい生産システム

これまで述べてきたもののほかに,下に示すように,新しい生産システムの概念がいくつか提案されている.それらの多くは研究途7―85図4・22 自己組織化を導入したラインレス生産のシミュレーションの例図4・23生物指向人工物ネットワークの概念上にあるが,将来の生産システムのあり方として大変興味深いものである.(1) バーチャルエンタプライズ(2) インタラクティブ生産(3) マルチエージェント生産システム(4) トータルライフサイクル生産システム(5) サプライチェーンネットワーク(6) 人工物ネットワーク図4・23は,4・4・4項で述べた生物指向型生産システムを社会システムまで拡張したものであり,DNA 型情報とBN 型情報を埋込んだ人工物で構成されるネットワークの概念図である.このような人工物ネットワークでは,人工物と人間が局所的に相互作用しながら,人工物の設計・生産・運用・リサイクル・廃棄の過程に,いつでもどこでも,誰でも参入・関与するシステムである.これは,物質とエネルギーの循環システムに情報の循環という性質を加えることを意味することになる.将来においては,人工物の生産活動を人間社会や経済システム,さらには自然循環システムと切り離した孤立系として考えることは,意味をなさなくなり,図4・23のようなとらえ方がより重要になるであろう.ユビキタスコンピューティングの技術開発と相まって実現されると期待される.
