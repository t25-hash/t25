# 8・4・4 製造法（fabrication method)

> **章**: 第8章 複合材料
> **節**: 8・4 金属基複合材料

強化材とマトリックス金属から複合材を製造する方法を大別する
と，マトリックス金属を液相にして強化材のまわりに流動充てんし
複合化する方法（液相法，liquid process ）とマトリックス金属を
溶かさずに高温で塑性流動させて強化材のまわりに充てんする方法
（固相法，solid process ）がある．液相法では金型に繊維成形体
（preform）を入れて鋳造するプロセスが一般的であり，溶湯に加
える圧力によって，スクイーズキャスティング（高圧凝固鋳造）法
（squeeze casting process ），ガス圧溶浸法，真空溶浸法，無加圧溶
浸法などがある．固相法では連続繊維と金属はくの積層材を金型
（metal die）または黒鉛型（graphite die）に入れて一軸加圧のホ
ットプレス，あるいはカプセルに封入して静水圧加圧のホットプレ
ス（HIP）をするプロセスが一般的である．このほか，固液共存状
態の金属のスラリーにウィスカや粒子を入れてかくはんし複合化す
る方法にコンポキャスティング法，マトリックスのアトマイズ中に
ウィスカや粒子を吹付け複合化するアトマイズ共堆積法など固液共
存相を利用する方法がある．
金属基複合材料の製造法は強化材の形態によって制約があるが，
製品の最終形状および寸法，要求性能，生産性などによって，最も
適切な製造法が選択される．表8･20は各種製造法における，強化
材形状，複合プロセス，生産性，製品形状などをまとめて示す．
