# 4・5・1 自動車の生産システム

> **章**: 第4章 自動化システム
> **節**: 4・5 事例(自動車、家電、半導体等)

自動車の生産は図4・24に示すように販売会社から注文を受けた後、生産計画が立てられ、その計画は車体組立ラインの先頭に生産7―86生産システム工学図4・24 自動車生産の流れ指示されて、組立が開始される。完成した車体は塗装され、車両組立ラインでエンジン、トランスミッション、アクスル等のユニット、購買取引先から内外装品、電装品、シート、ガラス、タイヤ等の各部品が組付けられ、検査工程を経て完成、出荷される。車両は約20000点の部品で構成され、1ボデー当たりの車両種類はエンジン、トランスミッション、外板色などの基本仕様に加え、サンルーフ、ドアミラー、パワーウインドウ、ドアロック、アルミホイール等のさまざまなオプションがあり、これらの組合せの数は合計約700仕様にも及ぶ。したがって、生産計画は非常に複雑で、部品の所要量計算、負荷能力検討が行われ、資材調達計画、車両組立順序計画が作られる。投入順序の決め方は、組立ラインの運営によって決められるが、基本は種々の車種が同一ラインで生産される混流生産であり、そのため、車種間、仕様差間での作業量を平準化した平準生産が行われる。また、エンジン等のユニット組立、車両組立ラインに直結したモジュールユニットのサブ組立、購入部品の内シート、バンパ等の大物部品は組立順序に基づいた同期生産が行われる。自動車の生産には素材加工から車両の組立まで非常に幅の広い範囲の生産技術を必要とするとともに、コンピュータやロボットなどのハイテクを利用し、高度に自動化やシステム化が進んでいる生産ラインが多い。プレス、車体溶接組立ライン、エンジン、トランスミッションの機械加工ラインは90~95%の自動化率である。組立ラインにおいてはエンジン、トランスミッション組立で40~70%、車両組立ラインの場合は5~20%の自動化率であり、特に、車両組立ラインは車両の種類が非常に多く、数千点の部品が組付けられること、繁雑なモデルチェンジへの対応等の要因で労働集約的な作業形態となっている。a. プレス工程プレス工程は専用設備である金型によるロット生産である。車両の組立順序計画と同期を取り、中間在庫を最小にするためのプレス生産計画が立てられるとともに、金型の型交換を最小にするための金型交換の外段取り化、自動化が行われている。車体組立ラインへの部品供給は車体生産ラインへの生産指示に図4・25 車体組立システム基づいて必要部品が保管庫からNCクレーンで選別されAGV 等で搬送される。b. 車体溶接組立工程プレス工程で作られた車体を構成する内外作プレス部品はスポット溶接を主体とした溶接により接合・組立が行われる。組立順序計画の生産指示はこの工程の先頭に入り、組立順序に合わせた生産が行われる。ボデータイプにより組立ジグはそれぞれ専用となるが、図4・25に示すような三面切替装置やパレットジグの交換とロボットの作動プログラム変更によって、複数のボデータイプの生産が生産指示情報により自動的に切替えられる。c. 塗装工程塗装は電着塗装、中塗り、上塗りおよびそれぞれの工程で乾燥焼付工程が必要であり、ボデー色による塗料の色替えが必要となる。塗装工程では同一色のロット生産が行われ、前工程の車体組立工程と後工程の車両組立の編成替えと色替えによる塗料ロスの最適化のためのロット制御が必要となる。通常、バンパ、シートなどの車両組立ラインへの搬入指示は塗装オンの情報で行われる。d. 車両組立工程規模のそれほど大きくない自動車メーカでは、複数車種の混流生産が要求される。アセンブリーに要する工数にばらつきがないように、むだのない生産を行うためには、どの車種のどの仕様でも、その組合せで工数偏差が生じないようにすることが必要である。この平準生産のための一つの方法がモジュール生産である。部品群をモジュール化・サブ組立化することにより、メインラインの短縮化、生産リードタイムの短縮が図られるとともに、フレキシブルな自動化、簡易化、作業姿勢の改善が可能となる。図4・26にエンジンシャシモジュール組付システムを、図4・27にフロントエンドモジュール組付システムの事例を示す。工場の管理システムは図4・28に示すように、生産順序計画に基づいた車種情報により、種々の制御、製造管理、作業指示および大物購入部品の搬入指示が行われる。車種情報はメインラインとサブ組立ラインに送られ、メインライン上でサブ組立モジュールと車両がドッキングされる。自動組立装置の制御は車種情報からPLCあるいはワークステーションを介して行われる。また、モジュール組立用パレットに装備されたワークID により、車種情報との照合が行われる。車両組立ラインは労働集約的工程であり、マンマシンインタフェースが重要となり、車種図4・26 エンジンシャシモジュール組付システム図4・27 フロントエンドモジュール組付システム図4・28 工場管理システム7―87情報からCRT を用いた作業指示、部品供給指示、検査項目指示等が行われる。また、車種情報と車両に付けられたワークID の照合・確認により、生産実績管理が行われ、中央制御室にこれらの情報が送られる。
