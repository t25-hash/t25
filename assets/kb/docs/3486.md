# 4・5・2 自動車部品の生産システム

> **章**: 第4章 自動化システム
> **節**: 4・5 事例(自動車、家電、半導体等)

a. 自動車部品生産システムの動向自動車部品の生産システムは、20世紀においては、需要が右肩上がりで安定的な伸びが確保され、積極的な自動化により生産性向上に努めてきた。しかし、バブルの崩壊とともに右肩上がりの時代は終焉を迎え、一定の需要の中でシェアを競い合う厳しい時代となった。さらに、地球環境問題、少子化など生産環境を取巻く環境も多様化し、生産性に加えてさまざまな価値観にも対応した生産システムが求められている。とりわけ、需要が不確実、すなわち生産量や生産期間が予測しにくい生産環境において、高生産性を維持しつつ、生産品目の変更や生産量の増減に迅速かつ柔軟に対応できる、新たな生産システムの構築が求められている。このニーズに応える生産システムとして、セル型、一人生産方式などと呼ばれる手作業を主体とした生産システムが現れている。これは生産量の変化に応じて、人の増減あるいはセルの増減によって迅速な対応を可能とし、人のもつ作業の柔軟性によって新製品への迅速な対応を可能とするもので、パソコンなど需要が不透明で、生産期間も数箇月と短い製品で効果を発揮している。自動車部品分野では生産量や生産期間の予測が難しいとはいうものの、車両のモデルチェンジまでの生産期間は保証されるため、2年間ないし4年間での生産システムの最適性が求められる。この場合、自動化のもつ高い生産性に着目することは必要であり、自動化であっても生産量の変化に迅速に対応可能な生産システムが求められる。b. 自律分散(autonomous distributed)指向の自動組立(auto-matic assembly)ラインのコンセプト生産システムが解決すべき課題を、生産システムの経済性と生産量の変化への対応性に着目して図4・29に示す。トランスファラインは特定の生産量の範囲で生産システムの構造を最適化するために高生産性を実現できるが、生産量の変化に対してコスト変化の感度が高く、求められる需要のふれに対してはコストが大きく変化する。一方、手作業ラインは生産量の変化に応じて7―88生産システム工学図4・29生産システムの生産量と経済性の関係作業者を増減させることで需要の変化に追従させる順応性に優れた自律的な機能を有する形態であり、その結果、需要のふれに対してコストは大きく変化しないが、特定の生産量の範囲での経済性ではトランスファラインには及ばない。この経済性の高いトランスファラインに手作業ラインの順応性を統合させることが、課題を解決する方策であり、手作業ラインの順応性を分析し、この結果をトランスファラインに積極的に反映させる要件として技術開発を行い、目指す特性を具備する生産システムを開発した。開発システムは、生産量の変化に応じてシステム内に分散設置された自律指向の自動機械の台数を増減させることによってシステムの構造の最適な経済性を維持し続ける、新たな概念の自動化生産システム、アダプティブ生産システム(APS : adaptive productionsystem)である。APS は、自律・協調・継承という、いわば人のもつ社会的機能をトランスファラインへ組込むことによって生産量変化に対する順応性を実現したもので、以下の代表的な特徴を有する。(1) 生産システムは、移動機能を有する自動機械が分散的に配置された構造となっており、需要の変動に応じて、機械台数を機械の負荷が最大となるように増減することが容易にできる。(2) 自動機械は、自律的な機能を有し、作業範囲の決定、該当する工程への移動、および多様な工程での作業を、自律的に行うことができる。(3) 自動機械は、機械間で連携し合う協調的機能を有し、工程を共有することによって、それぞれの自動機械の稼働時間を増やし、生産システムの高生産性を実現できる。(4) 機械は、作業の継承機能を有し、生産システム構成の変化時に機械どうしで迅速に作業技能やノウハウの伝授が行える。c. 適用事例本コンセプトを自動車用スタータの組立ラインに適用した。図4・30に開発したスタータAPS
