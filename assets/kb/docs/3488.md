# 8・5・3 多層シート板（multi -layered sheet )

> **章**: 第8章 複合材料
> **節**: 8・5 サンドイッチ板・多層シート板

プラスチックの発泡倍率を厚さ方向に変化させることで表面材とり，その結果，表面材は硬い層を構成している．
ABS3層シート(ｄ)T型突合せ（優れている）(ａ) 二重重ね合わせ継手(ｂ)突合せ重ね合わせ（良い)(ｃ)突合せ二重重ね合わせ（優れている)に使用されている．
ABS3層シートの曲げ剛性は上下表面材の厚，νとおき，サンドイッチ板の場合と違って心材の剛性を考慮の質量の関係を示したが，同じABS単板に比べ3層シートは軽量3層シートはアルミ合金板，FRP板，鋼板などに比べ優れた軽量のものが成形されている．
心材を同時に，かつ連続的に変えて成形したものが多層シートであがすでに市販されている．
図8･23 は各種板材の曲げ剛性と単位面積当たりでかつ優れた曲げ剛性を有することが示されている．
