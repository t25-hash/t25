# 4・5・4 家電の生産システム

> **章**: 第4章 自動化システム
> **節**: 4・5 事例(自動車、家電、半導体等)

a. 生産システムの変遷戦後の1960年代から現在に至るまで、家庭電化製品の生産システムを以下のように大別することができる。(1) 創生期(1960~1970年代)人々の生活を豊かにする商品を大量に生産するために、1人1作業での流れ作業によるベルトコンベヤ方式が主流となる。(2) 成長期(1980年代)生産性の追求から、人手に替わって、ロボットが組立・検査を行う自動化生産システムが急速に普及する。末期ごろより、多品種生産への対応が要求され始める。(3) 成熟期(1990年代)人々の生活が豊かになり、価値の多様化と商品の短命化が進む中、本格的な多品種少量生産の時代に移行する。複数のモジュールの組合せで効率的なもの作りを実現する、フレキシブル生産システムが普及する。(4) 個別最適期(2000年代)中国・韓国・台湾メーカの台頭やマーケットイン志向の商品展開など、競争が激化する中、グローバルに市場が要求する高機能商品を、高品質・低価格でタイムリーに供給できる生産システムが必要となっている。b. ものづくりの最適化。 セット(完成品)のものづくり家庭電化製品は、大型のテレビ・冷蔵庫、中型のDVD レコーダ・ステレオコンポ、小型のムービー・ディジタルカメラなど多岐にわたっている。それぞれの、1生産数量、2商品サイズ・重量、3構成部品点数、4機種展開数、5作業者能力などにより、最適な生産システムの追求がなされている。図4・35~4・37にその一例を示す。図4・35はセル生産方式の例(ワンマンセル生産方式-屋台方式)で、1人もしくは複数の作業者がすべての部品や作業工具を身近に置いて商品を組立てる。複雑な形状の部品やゴム系の柔軟な部品の組立作業・調整作業、電気配線処理などの自動化が困難な作業にも対応できること、機種展開数が増えてもフレキシブルに対応できること、ジグ化などの創意工夫による作業効率の追求に対応できることが特徴であり、生産性向上が見込める。この方式は中・小型サイズで機種展開数が多い商品には非常に効果的ではあるが、大量生産を実現するためには能力の高い作業者を数多く養成する難しさがある。図4・36は自動化生産システムの例(マンマシン生産方式)で、工程内容に合わせて自動化設備や直接作業者を配置した構成となっており、大型サイズや生産数量が多い商品には効果的である。機種切替え時の非生産時間をいかに短縮するかが重要になる。図4・37はベルトコンベヤ方式の例である。ものづくりをグローバルに展開する場合、労働力の質・量などの背景から、安定生産にはものづくりの原点である1人1作業による流れ作業が効果的な場図4・35 ワンマンセル生産方式図4・36 マンマシン生産方式図4・37 ベルトコンベヤ方式7―9 1合もある。いずれの生産システムにおいても、生産性追求・品質管理・在庫管理は共通の重要項目であり、IT 化は不可欠な要素である。. デバイスのものづくりセット(完成品)の性能を決定付けるデバイス(セットの基幹部品)は、いちだんと高性能化・小型化が進み、これらの高密度化・微細化に対応した生産システムが必要となっている。この生産システムには、精密微細加工技術(precise micromachining technology)を駆使することと併せて、半導体デバイスのものづくりに代表されるようなクリーン化技術を適用する必要性が増してきている。図4・38にDVD-ROM ディスクの生産システムの例を示す。工程内のクリーン度を高いレベルで維持するために、設備内にHEPA フィルタを搭載し、温度・湿度なども含めて制御している。さらに、デバイス性能を高次元で品質維持するためのプロセス条件(例えばディスク成形では金型温度、樹脂温度、射出圧力等)を高精度で制御できる機能を有している。文献(1) 牧野洋・新井民夫、精密工学会誌、63-11(1997)、1503.(2) 杉戸克彦・ほか4名、精密工学会誌、70-6(2004)、737.(3) Tamaki, K., ほか5名、 Proc. 24th Int’nl Sym.
