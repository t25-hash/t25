# 5・1・1 加工設備の要件

> **章**: 第5章 生産設備
> **節**: 5・1 加工設備

加工設備は素材に物理的な作用を加えて製品または部品に変換するもので、生産システムの中核をなしている。生産システムでは、成形加工(鋳造、射出成形、鍛造、プレス、粉末加工など)、除去加工(切削、研削、研磨、放電加工、レーザ切断など)、付加加工(溶接、溶射、めっきなど)、改質加工(熱処理など)のための各種の加工機械が使用されている。生産システム中の加工設備は、加工機械単独としての基本的な性能のほかに、組立設備・搬送設備などの関連施設と有機的に連携して自動加工を行うための機能を備えている必要がある。いま、切削加工を主対象として、加工設備が備えるべきおもな機能をまとめると次のようになる。・工作物および工具の相対運動の制御機能・工作物の自動交換機能・工具の自動交換機能・ジグ・取付具の自動交換・設定機能・自動洗浄機能・自動ばり取り機能などの補助機能・設備・加工状態のモニタリング機能・オンマシン計測機能・加工品種変更への柔軟な対応これらの機能は、一般にNC工作機械あるいはNC加工セルによって実現されている。加工機械を高度にシステム化するうえでは、搭載NC制御装置のオープン化などネットワークへの対応の高度化の必要性も高まっている。また、保守・保全機能のために、機械設備の状態や製品を検査するモニタリング用センサおよび得られた信号の処理系をもっていることも有用である。
