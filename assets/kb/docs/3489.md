# 5・1・1 加工設備の要件

> **章**: 第5章 生産設備
> **節**: 5・1 加工設備

加工設備は素材に物理的な作用を加えて製品または部品に変換す
るもので，生産システムの中核をなしている．生産システムでは，
成形加工（鋳造，射出成形，鍛造，プレス，粉末加工など)，除去
加工（切削，研削，研磨，放電加工，レーザ切断など)，付加加工
（溶接，溶射，めっきなど），改質加工（熱処理など）のための各種
の加工機械が使用されている．
生産システム中の加工設備は，加工機械単独としての基本的な性
能のほかに，組立設備・搬送設備などの関連施設と有機的に連携し
て自動加工を行うための機能を備えている必要がある．いま，切削
加工を主対象として，加工設備が備えるべきおもな機能をまとめる
と次のようになる．
･工作物および工具の相対運動の制御機能
･工作物の自動交換機能
･工具の自動交換機能
･ジグ・取付具の自動交換・設定機能
･自動洗浄機能・自動ばり取り機能などの補助機能
･設備・加工状態のモニタリング機能
･オンマシン計測機能
･加工品種変更への柔軟な対応
これらの機能は，一般にNC工作機械あるいはNC加工セルに
よって実現されている．加工機械を高度にシステム化するうえで
は，搭載NC制御装置のオープン化などネットワークへの対応の
高度化の必要性も高まっている．また，保守・保全機能のために，
機械設備の状態や製品を検査するモニタリング用センサおよび得ら
れた信号の処理系をもっていることも有用である．

5・1・3加工のオンマシンモニタリング
加工設備を適正に運転するには，加工状態の計測と工作物の検査
が重要である．そのため，加工状態をオンマシンで検知し，その情
報を制御や管理システムにフィードバックすることが求められる．
オンマシンモニタリング（monitoring）は，おもに次のような目
的のために行われる．
a.高精度 所望の加工精度を維持するために，加工精度に変
動を与えるような要因，例えば工作機械の場合では，工具損耗や工
具変形，工作物の静・動特性，工作機械の運動精度や構造的振動，
熱変形などの状態量を検知する．
b.高能率 安定して加工が行える最大限の加工能率を維持す
るために，負荷動力などの加工プロセスの状態量を検知する．ま
た，加工機械の停止やアイドリング時間を最小限にすることを目的
に稼働に関する状態量を検知する．
c .高信頼性 品質の維持と安定，安全確保，環境保全，機械
の機能や性能の維持などを目的に，作業環境に関する状態量，機械
の劣化や異常に関する状態量などを検知する．
こうした目的のためにさまざまなセンサが用いられている．表

実用的な加工モニタリングの例としては，主軸駆動モータの温度
および駆動電流によって加工機械の負荷状態を監視することが行わ
れ，多くのNC制御装置がこれらの計測機能を内蔵している．ま
た，製品加工精度のインプロセス計測用にタッチセンサも多用さ
れ，寸法精度の測定，加工基準位置の確認に用いられている．図

図5･9 は研削中の工作物寸法を計測しているところであり，加工
機械にフィードバック補正信号を出力することにより安定した加工
を続けられる．温度センサを利用して加工機械の熱変形を予測し，
自動補正する加工機械も造られている．
