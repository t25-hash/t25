# 5・1・3 加工のオンマシンモニタリング

> **章**: 第5章 生産設備
> **節**: 5・1 加工設備

加工設備を適正に運転するには,加工状態の計測と工作物の検査が重要である.そのため,加工状態をオンマシンで検知し,その情報を制御や管理システムにフィードバックすることが求められる.オンマシンモニタリング(monitoring)は,おもに次のような目図5・5マシニングセンタを中心とした小規模FMS の構成例図5・6 FMC の概念:産業用ロボットがNC 旋盤に組込まれた例図5・7 ターニングセンタベースの複合加工機械的のために行われる.a. 高精度所望の加工精度を維持するために,加工精度に変動を与えるような要因,例えば工作機械の場合では,工具損耗や工具変形,工作物の静・動特性,工作機械の運動精度や構造的振動,熱変形などの状態量を検知する.b. 高能率安定して加工が行える最大限の加工能率を維持するために,負荷動力などの加工プロセスの状態量を検知する.また,加工機械の停止やアイドリング時間を最小限にすることを目的に稼働に関する状態量を検知する.c. 高信頼性品質の維持と安定,安全確保,環境保全,機械の機能や性能の維持などを目的に,作業環境に関する状態量,機械の劣化や異常に関する状態量などを検知する.こうした目的のためにさまざまなセンサが用いられている.表に整理したものである.実用的な加工モニタリングの例としては,主軸駆動モータの温度および駆動電流によって加工機械の負荷状態を監視することが行われ,多くのNC制御装置がこれらの計測機能を内蔵している.また,製品加工精度のインプロセス計測用にタッチセンサも多用され,寸法精度の測定,加工基準位置の確認に用いられている.図図5・9は研削中の工作物寸法を計測しているところであり,加工機械にフィードバック補正信号を出力することにより安定した加工を続けられる.温度センサを利用して加工機械の熱変形を予測し,自動補正する加工機械も造られている.
