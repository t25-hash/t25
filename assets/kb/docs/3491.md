# 8・6・3 複合材料接着継手の破壊

> **章**: 第8章 複合材料
> **節**: 8・6 複合材料の接着・接合

(単純)重ね合わせ継手の代表的な破壊様式は以下の5種類に分け
られる（図8･30)．
(１)接着破壊
a.界面破壊（接着剤層と被着体の接着界面での破壊）
b.凝集破壊（接着剤層内破壊）
c.混合破壊（aとbの破壊が混じった破壊）
(２)被着体破壊
a.被着体の引張破壊
b.被着体の局部破壊
界面接着強度より接着剤の強度が低い場合，接着剤の凝集破壊
［(１)-a］が起こる．一方，界面接着強度が接着剤の強度より低い
場合，接着界面が露出する界面破壊［(１)-b］が起こる．被着体

表面の処理は金属被着体の場合，サンドブラストあるいはサンドペ
ーパなどで研磨した後，溶剤で脱脂するのが一般的である．高い信
頼性の接着強度を得るためには，適当な薬液に浸すなどの表面処理
が行われる．一方，FRPなどの高分子系複合材料では，サンドペ
ーパ（あるいはサンドブラスト）により表面の樹脂リッチ層を取り
除いて表面を荒らすとともに，溶剤で脱脂するのが主たる表面処理
法である．この場合，強化繊維を露出するほどに研磨することも多
い（しかし，露出した繊維の表面からかつて塗布された表面処理剤
が取り除かれるため，樹脂との接着性は劣り，必ずしも接着強度を
高めるとはいえない)．この処理が不十分であったり，汚れていた
場合，界面破壊はいっそう起こりやすい．したがって，凝集破壊が
発生すれば，被着体の接着面の表面処理は適切であったとみなされ
る．しかし，接着剤の強度が低い場合にはたとえ凝集破壊が起こっ
ても接着継手の強度が高いことは意味しない．逆に，界面破壊が起
こったとしても接着継手の強度が低いわけではない．被着体の種類
と厚さ，接着剤と表面処理によっては混合破壊［(１)-c］が起こ
(２)の被着体破壊は被着体の厚さが厚い，あるいは強いと起こら
ない．これは紙の接着を考えれば理解できる．薄い紙では，壁など
に強度の低い接着剤で接着しても引きちぎれることがある．また，
厚いボール紙などでは，接着剤とともに紙が壁に残り，取り外すの
に苦労する．鋼やアルミ合金ではよほど薄くないと(１)- aの破壊
は起こらない．鋼やアルミ合金の接着を対象として，(１)-bの破
壊を起こすような接着剤は現在のところ見当たらない．一方，
PMC重ね合わせ継手では，荷重の多くは強化繊維により支えられ
ており，厚さ方向におおむね均等に分布しており，接着面へは各層
の層間を通して伝えられることになる．そのためPMC被着体のラ
ップ付近では，接着面に近い層間で層に沿うせん断応力とともに大
きなはく離応力が生じる．特に，単純重ね合わせ継手では二つの被
着体間の軸心が一致しないため，層間はく離応力の集中は顕著であ
る ．PMCの層間はく離およびせん断強度はあまり高くないため
(２)-bの破壊は起こりうる．被着体の1層め，あるいは2層めか
ら破壊が生じることも少なくない．
