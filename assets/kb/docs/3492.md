# 8・6・4 さまざまな複合材料接着継手

> **章**: 第8章 複合材料
> **節**: 8・6 複合材料の接着・接合

金属を被着体とする接着継手では，界面破壊であれ，凝集破壊で接着継手では，被着体破壊を防止することも重要である．
これらを夫が行われている．
に示されるように，引張せん断荷重を重ね合わせ継手ではラップ端とき，この応力集中は接着端でいわば部材厚さが急変するからであ少すると考えられる．
また，PMC被着体の層間応力の集中も減じよい．
あれ基本的に接着部分で破壊が生じる．
一方，上述のようにPMC背景に，PMC接着継手の強度（継手効率）を高めるさまざまな工でせん断応力，はく離応力とも集中する．
継手を一体材料と考えたる．
そこで，厚さの急激な変化をなくせば接着剤層の応力集中は減る．
(ａ)被着体を曲げて接着するある．
