# 8・6・4 さまざまな複合材料接着継手

> **章**: 第8章 複合材料
> **節**: 8・6 複合材料の接着・接合

金属を被着体とする接着継手では，界面破壊であれ，凝集破壊で
あれ基本的に接着部分で破壊が生じる．一方，上述のようにPMC
接着継手では，被着体破壊を防止することも重要である．これらを
背景に，PMC接着継手の強度（継手効率）を高めるさまざまな工
夫が行われている．
a.テーパ付き重ね合わせ継手（tapered lap joints ) 図8･28
に示されるように，引張せん断荷重を重ね合わせ継手ではラップ端
でせん断応力，はく離応力とも集中する．継手を一体材料と考えた
とき，この応力集中は接着端でいわば部材厚さが急変するからであ
る．そこで，厚さの急激な変化をなくせば接着剤層の応力集中は減
少すると考えられる．また，PMC被着体の層間応力の集中も減じ
る．そこで，図8･31のように被着体の一端にテーパを付与すれば
よい．また，テーパ状に加工したPMC面の保護とテーパ端でのは
く離を抑えるため，新たにマットあるいは平織布状の強化材を1層
