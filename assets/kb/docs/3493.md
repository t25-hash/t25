# 8・7・1 熱硬化性プラスチック系FRPの成形方法

> **章**: 第8章 複合材料
> **節**: 8・7 複合材料の成形法

熱硬化性プラスチック（thermo setting plastics ）は，硬化剤を
混入した液状の樹脂にさらに充てん材や強化材を混合し，型に入れ
形状を決めた後，加熱・硬化できることから広く実用化されてい
る．この方法の多くが長繊維を使用できることから高強度，高弾性
率が得られ，機械・電気機器の構成材料として，工業製品，スポー
ツ用品，建築用品に使用されている．代表的な成形方法を以下に示
す．

合では，両者の中心が中立面を一致させる必要が多い．そのままで
図8･24(ａ)のように突合せ継手となり，高い継手強度は望めない．
そこで，一方の被着体の接着部分を変形させた継手（一方の被着体
をあらかじめこのような形に成形してある）が考えられる［図8･

a.フィラメントワインディング法（FW:filament winding）
樹脂を含浸した強化繊維を回転するマンドルに巻付けた後，加熱硬
化する方法である．マンドルは樹脂の硬化・冷却後に除去する．こ
の方法は，細い管や太い円筒状物体の成形に適している．この成形
法による製品の直径は，数cmのものから5～6mに至る．繊維の
巻き角度は，マンドルの回転速度と軸方向への繊維の移動速度を調
整することにより自由に変えられる．巻き角度によって製品の強度
を任意に調整することが可能である．繊維の巻き角度が管の軸方向
に近いものほど軸方向引張強度が強く，また曲げ変形に強い素材と
なる．逆に，繊維の巻き角度が管の軸方向に対して直角に近くなる
に従い軸方向の引張強度が低下するものの，内圧に強い構造体とな
る．この構造体は，高圧容器に向いている．径の小さい管状のもの
は，ゴルフのシャフトやテニスのラケットに，大口径パイプではロ
ケット，魚雷の胴体に使用されている．現在プロパンガスなどの高
圧容器へのFW製FRPの応用が試みられている．
FW法と同じロール状に巻く成形様式の，テープレイイング
（tape laying）法が開発されている．この手法は，フィラメントの
代わりに一方向性プリプレグ強化シート（テープ状）をマンドルに
巻付け，その後加熱硬化させて製品を作製する．この成形法はFW
成形法より装置が簡便で済む．
b.ハンドレイアップ法（hand- lay- up) 代表的接触圧成形法
の一種で，手積み法ともいわれる．この方法は，強化材のシートや
マットを型の上に置きこれに樹脂を含浸させ，手積みにより積層
し，これを加熱して硬化する方法である．船体等大型の構造物の作
製に使用されている．この単層をラミナ（lamina）と呼ぶ．一般
には，単層では異方性が出すぎるため，一般には多層を積層して硬
化したものを用いる．これを積層材（laminate）と呼ぶ．この方
法では，層間に気泡が残留しやすいため，物性を安定させることが
難しく，訓練を必要とする．
c .圧縮成形法（compression molding) コールドプレス成形
法（cold-press ）とホットプレス成形（hot - press ）があるがホッ
トプレス法が多く使用されている．これはプリフォームマット材を
型に入れ，高温のもとで加圧成形する方法であり大型の浴槽などの
大量成形に使用されている．代表的な材料は，SMC（シートモー
ルディングコンパウンド）で，あらかじめ繊維と樹脂に硬化剤を混
ぜ，強化材，炭酸カルシウム等充てん材を混入したものをフィルム
に挟み，シート状にして巻取っておく．このSMCのロールは，低
温下では数週間保存が可能であり，熱硬化樹脂であるにかかわらず
生産の時間的調整が容易となる利点を有している．成形はロールよ
り適量を切取り，これをホットプレス機に設置した金型に入れ，加
圧・加熱して硬化する方法である．この方法は，大量生産化が容易
な方法である．繊維長は成形時に切断されることはないが，強化材
の場所的むら，充てん物の混合むら，繊維配向のむら，ストランド
の崩壊状態のむらなどが原因で，引張強度や疲労寿命に比較的幅広
いばらつきが出やすいことが欠点である．
d.オートクレーブ成形法（autoclave molding) 成形用の型
にプリプレグを充てんした後減圧による脱泡行程を行いその後，加
圧による緻密化ができる装置を用いる成形方法である．大型でかつ
高精度の寸法安定性を有するものの成形に用いられている．製品に
は，直径数mで長さ10mを超す送水用圧力管や大型容器，航空
機の尾翼，部品等の高強度で品質管理が必要な構造物の作製に使用
されている．大型の装置が必要となることが弱点となっている．
e.R- RIM成形法（reinforced reaction injection molding）
熱硬化性プラスチック系樹脂にミルド繊維や充てん物を混合したも
のを，射出成形機を用いて金型中に注入し成形する方法．金型中で
化学反応が進行し液状材料から固体樹脂へと変わる．強化繊維を混
入したウレタン系樹脂を使用した自動車部品が数多く作製されてい
る．バルクモールディングコンパウンド（BMC）材料は，強化材
と充てん材を不飽和ポリエステル樹脂に混合した材料に硬化剤を入
れ射出成形した材料である．精密機械部品の作製に使用されてい
る．
f.RTM成形法（resin transfer molding) 金型中に繊維を配
置した中に，2液混合型液状樹脂を射出成形した後，加熱・硬化し
て成形する方法である．射出成形の利点をもっている．射出成形法
では，繊維を樹脂と同時に射出するために高圧を必要とするため，
繊維は切断されてしまう．これに対して，この方法では長繊維によ
る強化が可能となり，さらに繊維の方向性を調整することが可能で
あり，金型も簡便なものを使用することが可能である．
g.引抜き成形法（pultrusion molding) 強化繊維に2液混合
型液状樹脂を含浸した後，必要とする断面形状をもつ金型（ダイ）
を通過させ，加熱硬化させながら引出す方法である．一方向性の長
尺物FRPを作製可能であり，この成形方法は，高強度・高弾性率
の棒や板の構造部材を作製するのに向いている．
