# 8・7・2 熱可塑性プラスチック系FRP

> **章**: 第8章 複合材料
> **節**: 8・7 複合材料の成形法

熱可塑性プラスチック系FRPはFRTP（fiber reinforced
thermoplastics ）と呼ばれており，この成形には熱可塑性プラスチ
ックの成形が使用されている．代表的な方法は，射出成形方法と，
ホットプレス法である．
a.射出成形法（injection molding) 溶融した樹脂を金型中
に射出した後，冷却して固化する方法である．材料には，強化繊維
を含む熱可塑性樹脂ペレットを使用する．射出成形は，各種ペレッ
トを調合して希望する量の繊維を含有するペレットを用意し，これ
を射出機内で溶解し，スクリューにより金型に射出して成形する．
この方法により作られているFRPは，全FRPの生産量の90%以
上に達しており，現状では大量生産が最も可能な方法である．はん
用の機械部品，特に吸振性，耐熱性と強度を向上する必要のあるプ
ラスチック部品，例えば電子機器内部に使用されている小型の変速
ギヤはこの方法で作製されている．航空機や自動車の小型構造用部
品，スポーツ用品，日常用電気機器のほとんどがこの方法で成形さ
れた製品が使用されている．この方法は，大量生産に向いている
が，強化材の充てん量に比較して，強度・弾性率は上がらない．こ
れは，ペレットの段階では，繊維のアスペクト比（繊維長/繊維直
径）は100～200程度あったものが，射出成形後には切断され10～
50程度に低下し繊維の充てん効果が著しく下がるためである．さ
らに，射出成形時に樹脂の流れにより繊維の配向方向の調整が困難
なため，荷重の伝達効果すなわち充てん効果が期待できない直交配
向の繊維が増加するためである．
b.ホットプレス成形法（hot -press ) 熱硬化性プラスチック
の場合と同じ方法であるが，強化繊維の一方向に配列し熱可塑性樹
脂で固めたプリプレグシートまたは二次元配列の布に熱可塑性樹脂
を含浸させたラミナを使用する．このプレグシートを積層して，ホ
ットプレス機に入れ，加熱して溶融した後，加圧して接合する．こ
の成形法の製品は，連続繊維を使用できることと熱可塑性樹脂の機
械的・化学的特徴をもつことから，広く使用され始めている．比強
度は，熱硬化性系FRPに匹敵する．母材として使用される熱可塑
性プラスチックは，ナイロン樹脂，ポリカーボネート樹脂などが使
用されている．
