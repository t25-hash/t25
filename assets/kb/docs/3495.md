# 8・1・4 デファクトスタンダード 経済のグローバル化に伴っ

> **章**: 第8章 規格・技術基準と法制度
> **節**:  

て，世界標準の必要性が高まる．その発生形態は，市場が強力に介
在するものと協議合意によるものとがある．前者はデファクトスタ
ンダード，後者はデジューレスタンダードといわれるものである．
デファクトスタンダードは，国家規格や団体規格のようなオーソ
ライズされた干満化団体ではなく，企業などが仕様や製品について，
その優位性を図る目的で規格を公表して，寡占状態で企業活動を行
う．そのため，公開状態にないデ7アクトスタンダードは，標準の
機能を有していないという欠点の中で運用される，という重大な欠
点がある．
身近な例としては，マイクロソフト社のWindowsとマッキント
ッシュ社のOSとが対比される．規格化されているわけではないが，
Windowsが圧倒的な優位を占めている．
デファクトスタンダードによって十分に利益を得た後にその更な
る適用を行うために，国際規格や国家規格として規定化を図るよう
に働きかけ，その時点で別の製品のためのデファクトスタンダード
を開発している例も多い．昨今は，企業戦略的な標準化が熾烈化し
ている．
デジューレスタンダードは，外的要因との交渉に重点が置かれて
成立する規格であるといえる．特に，国家間の外交努力によって成
立する場合が理想的であり，国家間の経済に多大の影響を与えるも
のである．
政治的審議や行政手続きを経て制定される標準の場合には，公的
標準又は法定標準と呼ばれ，強制力を有する標準となると基準ある
いは規制となる．
ISO 9000シリーズやISO 14000シリーズは，国際標準化機構が
定めた規格であるが，取引，流通，サービスなどが国を通過する際
の複雑さを回避する目的から国間で統一して適用を行うように働き
かけ，デジューレスタンダードとしての機能を有するようになった．
なお，ISO 9000シリーズは，文書化した規定事項が業務で履行
されているかどうかが重要視される．これに対してアメリカのビッ
グスリー自動車メーカが実行しているQs goooシリーズは品質の
平なる向上を図るために，測定の不確かさを算定して製品の合否判
定を行わなければ，取引を停止するという強い要求のもとで適用し
ている．
ISO／TC 213（製品の幾何特性仕様及び検証）の場では，部品の
製作から検証において，数学的に定義できて，あいまいさを排除し
て，すなわち，どう定義し，それをどのように図面指示し，どのよ
うな測定機器を使用し，それをどう校正し，不確かさを算定して合
否判定をするかという標準化を行っており，Qs goooシリーズに対
して修正すべき事項を盛り込みつつある．
