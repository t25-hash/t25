# 5・3・1 三次元座標測定機(三次元測定機)

> **章**: 第5章 生産設備
> **節**: 5・3 検査設備

a. 三次元測定機の構造NC工作機械に代表される加工技術の進歩により、複雑な形状の工作物を高精度に加工することができるようになり、機械部品の三次元形状、寸法、位置などを測定するために、三次元測定機が広く普及し重要な位置を占めるようになってきている。三次元測定機の一般的な構成としては、直交する3軸に対して空気静圧案内面とリニアスケールが取付けられ、三次元的な空間座標を測定、表示する。三次元測定機の構造を図5・14に示す。構造形式を大別すると、直交座標型と関節型に分けられる。また、直交座標型の軸を回転可能としたり、テーブル上に回転テーブルを搭載して4軸の測定装置としたものも見られる。一般に工作物を加工する際には、図面を基にすることになるが、実物からその形状を測定により求める必要の生じる場合がある。意匠設計などがその代表で、デザイナーが製作したモデル表面上の各座標値を三次元測定機で測定し、加工データを創成するなどの利用方法も、三次元測定機の重要な機能である。図5・15は多関節型の三次元測定機を示す。自動車車室内の部品配置など、通常の三次元座標測定機では測定不可能な部分の測定等に使用可能である。b. プローブ三次元測定機用あるいはオンライン計測用セン図5・14 三次元測定機の構造図5・15多関節型三次元測定機[(株)小坂研究所]7―9 9サとして、測定対象物との接触あるいは接近を検出するプローブは、三次元形状・寸法測定における最も重要な要素の一つである。図5・16に示すようにプローブには大きく分けて、タッチプローブ、倣いプローブおよび非接触プローブがある。タッチプローブで最も一般的なものが、図5・17に示す三点支持型である。3方向に配置されたボール上に接点が置かれ、スタイラスが測定対象物に接触すると、この接点がオープンとなり信号が出力される。三点支持型の難点である復帰不良、接触圧を低くできないなどの点を考慮し、ひずみゲージや圧電素子を用いた高精度プローブが開発されている。倣いプローブは、直線案内機構と変位センサを内蔵して三次元座標測定機との相対的変位を検出するもので、複雑な自由曲面等の形状誤差を高精度で評価するなどの用途に使用される。6方向(±,±,±)の変位量を差動トランスやスケールを用いて検出する。高精度測長ユニットを搭載したものでは、測定ストローク±1mmを0.1 mの繰返し精度で測定可能である。図5・18に倣いプローブの構造を示す。非接触プローブには、静電容量、原子間力、空気圧などを利用したものがあるが、光応用のプローブが一般的である。測定対象が布、粘土など柔らかく、接触式プローブでは、表面に傷を付けたり、接触による変形を招きやすい材質の測定では、非接触測定が必要である。図5・19は、光学式非接触プローブの例を示したものである。測定原理は三角測量方式であり、段差などの部分で光が検出器に戻ってこないという難点をリング状受光部を配置することにより低減する。c. 非接触三次元測定機三次元的な複雑形状を高速に測定する用途には、光学式の測定装置が適している。図5・20はこのような測定装置の例である。三次元モデル形状のCAD データの取得、金型あるいは製品形状の検査およびCAD データとの比較・評価、測定データよりカッタパスを作成し、NC工作機械による加工等の用途に使用される。測定原理は、モアレじまを応用したもので、モアレ3D カメラは図5・16 プローブの種類図5・17 タッチセンサの構造(レニショー社)7―100生産システム工学図5・18 倣いプローブの例図5・19光学式プローブの例図5・20 非接触三次元測定機(オプトン社)フリンジプロジェクタとCCD カメラ等により構成される。フリンジプロジェクタから格子パターンを投影し、測定対象物上の変形格子像をCCD カメラにより撮影し、モアレじまを解析して三次元形状を求める。
