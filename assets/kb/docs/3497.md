# 5・3・3 インプロセス測定機器

> **章**: 第5章 生産設備
> **節**: 5・3 検査設備

自動生産のためのオンライン計測機器、センサのもつべき機能にはさまざまなものが要求されるが、部品の寸法精度の計測、切削力、振動、工具損傷などの検出が重要な項目となる。インプロセスで使用する測定機器は種類が非常に多い。導入に当たっては、センサ、コントローラ、計測ソフトウェア等をシステム化し、ハードウェアの選定から、ソフトウェアの構成まで、測定作業ニーズに併せてメーカが関与する場合が多い。a. 自動定寸装置部品加工において内径、外径などを測定する必要性は、工作機械の稼働率を向上するために必要であり、自動定寸装置の開発が行われてきた。各種測定装置により加工機上にて寸法測定を行うことは、インプロセス、ポストプロセス測定を問わず、工作物を工作機械より取り外すことなく測定ができる点で、精度上、能率上のメリットが大きい。自動測定が求められる理由としては、(1) 加工後に工作機械を停止し、通常の測定器具により測定することは非常に時間のかかる作業であり、工作機械の稼働率を低下させる。(2) 熱変形の影響や工具の摩耗により、工具位置の補正を行わないと指定の寸法に仕上げることができない場合。などがある。図5・22に示す定寸装置は、インプロセスで工作物の寸法を測定し、砥石の切込み速度を変化させたり、加工機を制御する機能を有するものである。検出器を砥石と対向させたテーブル上に取付け、接触子を工作物に2点で接触させ、工作物の外径変化を差動トランスの出力として求めるものである。定寸装置を用いた研削サイクル制御例を図5・23に示す。工作物の直径の変化を研削中に連続的に測定し、所定の寸法になったところで砥石の切込み速度切換え指令、切込み停止指令、砥石後退指令を出力する。また、定寸装置の測定部位への接近の方法、測定箇所などの例を表5・2に示す。一方、旋盤用などのインプロセスゲージは切りくずの排出が障害となり、加工中におけるインプロセス測定が困難であり加工直後に測定して、工具補正信号を出す検出器が使用されている。検出器は基本的には研削盤用のものと同様ではあるが、測定寸法に合わせて専用に設計されたガイドを基準として、内径寸法測定を電気マイクロメータを用いて行うガイド付内径測定ボアゲージ等がある。b. 工具計測器工具に関する測定項目としては、長さ、径、回転時の振れ、損傷、熱変形が考えられる。工具に関する計測は、折損検知だけでは十分ではなくなってきている。工具プリセッタなどの機外計測機器が進歩しており、チップに書き込まれたデータを加工機械に転送することが可能である。しかし、機械ごとに差があるために、工具を機外で測定するだけでは十分ではない。工具が回転しているときには、熱変形によるスピンドルの伸び、工具の振れ、ATCでの工具の引込み量が変化するなど、静止状態では現れない現象がある。したがって、切削時と同じ回転状態での工具の測定が最も望ましい。また、工具の摩耗、欠損の検出も行う必要がある。図5・24にレーザ方式の工具径・長さ測定装置の例を示す。工具の刃先管理、スピンドルの熱変位、回転による位置変化、工具摩耗やチッピングを検知し、予備工具に変える管理などに適用できる。c.
