# 5・4・1 搬送機器

> **章**: 第5章 生産設備
> **節**: 5・4 搬送保管設備

搬送機器(transfer equipment)はその対象が半導体デバイスで使われるグラム単位の部品から、自動車・金型などのトン単位の製品まで、種類、形状などで多様であり、その特徴に対応するために搬送形態および自動化レベルによって表5・3に示すさまざまな機種が開発されている。その中の代表的な機種について以下に述べる。a. 無人搬送車無人搬送車(AGV : automated guided vehi-cle)は、本体に人手または自動で荷物を積込み、指示された場所まで自動走行し、人手または自動で荷卸しをする無軌道車両と定義される。無人搬送車(図5・27)は動力源を内蔵の蓄電池とし、走行路面上を無人誘導で専用の軌道をもたず自由に走行できるため、機動性、はん用性、融通性が高く広範囲に利用される。走行速度は最大60m/min、積載荷重は30kg から数千kg、停止精度は±10mm(センタリング方式で4mm)の仕様が多い。無人搬送車の経路を誘導する方式として、床面に誘導線を埋設して車上に取付けたピックアップコイルにより電磁力を検出する電磁誘導方式、床面に磁気テープを貼り車上の磁気センサにより磁界の強さを検出する磁気誘導方式、床面にステンレステープなどの反射テープを貼り車上から光を発して反射光を検出する光学誘導方式、工場内の位置をレーザナビゲーションシステムによって自律的に認識するレーザ誘導方式がある。荷積み・荷卸しの移載方式として、手作業による方式、荷台上の表5・3 工場間搬送機器生産のグローバル化、サプライチェーン化が進展するにつれて工場間をまたいで荷の搬送・保管を大規模に行う生産システムの重要性が増えつつある。工場間を搬送するための大量輸送手段としてトラック、鉄道、船、飛行機などが搬送機器となる。大量の荷を仕分け、保管を行うために工場の荷を集約したり、顧客へ分配する配送センター、港湾、空港でトラックから船舶、飛行機へ移載する貨物ターミナルなどの大型施設がある。
