# 5・4・2 保管機器

> **章**: 第5章 生産設備
> **節**: 5・28 ).

保管機器は物を物理的に保管(storage)する機器であり,一般には棚,倉庫などと呼ばれている.物流に滞留がなければ倉庫は必要ないが,前後工程の工程数差,能力差,ロットサイズ差,生産順序差などが生産システムの全体最適により発生し,そのために一時的に必要となる在庫量を保管するために表5・3に示したさまざまな機種の立体自動倉庫,棚設備が必要となる.以下,その代表的な機種について述べる.a. 立体自動倉庫(automated storage/retrieval system)立体自動倉庫は荷を格納する棚を立体高層化し空間格納効率を高めるもので,荷の入出庫方法によってスタッカクレーン方式,回転ラック方式などに分類される.スタッカクレーン方式は,荷を棚に搬入・排出する二次元移動を行うスタッカクレーンと入出庫のための設備およびこれらの制御装置と在庫管理装置で構成される.図5・34に一例を示すが,搬入された荷は在庫管理装置により入庫する棚が決められ,入出庫装置により入庫される棚通路の入口まで搬送され,スタッカクレーンにより棚に格納される.出庫は出庫する荷の情報を在庫管理装置に入力して管理情報と照合し,該当する棚が探し出されてスタッカクレーンへ出庫指令が与えられる.扱う荷の重量はバケット使用で50kg前後,パレット使用で1000kg 前後が多く,前者は水平走行速度最大300m/min,昇降速度100m/min,後者は水平走行速度最大図5・33パレットトラック図5・34 立体自動倉庫(スタッカクレーン方式)200m/min,昇降速度50m/min程度となっており,入出庫頻度の高能率化のための高速化が進んでいる.回転棚は水平型と垂直型とに大別される.回転ラックは棚が自動的に作業者の手元まで回ってくるため保管やピッキング用の通路が不要という特徴から多品種少量,小物部品の保管効率向上およびピッキング効率向上の目的で使用される.多段式の回転棚では複数の処理を同時並行的に処理できるため,入出庫平均サイクルタイム5秒程度の高速化のための小型自動倉庫として用いられる.b. 棚設備棚(rack)には固定ラック(パレットラック,ドライブインラック,流動ラックなど),移動ラックがあり,用途,保管物の形状により使い分けられる.固定ラックの中でパレットラック[図5・35(a)]はパレットに積載された荷の保管に用いられるラックであり,先入れ先出しができるので保管物の性質,品種の多少にかかわらず使用可能なはん用性の高いラックである.ドライブインラック[図5・35(b)]はパレタイズされた荷をフォークリフトにより入出庫する棚であり,フォークリフトの通路がそのまま保管通路になるため保管効率の極めて高いラックである.流動ラック[図5・35(c)]は保管物が棚部分に取付けられたローラコンベヤ,ホイールコンベヤなどによって重力で出口に向かって移動するラックであり,格納効率が良く,作業性も高い.移動ラックはラックが一列ごとに独立して動くため,棚が何列あっても作業通路は一列だけあればよい.そのため,同一設置面積では固定ラックに比べてはるかに多くの棚を設置できる(図5・36).
