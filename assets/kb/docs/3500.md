# 5・4・3 その他の機器

> **章**: 第5章 生産設備

搬送・保管作業を生産システムとして実現させるための補完的な機器として,仕分けコンベヤ,荷そろえ機器,包装・梱包機器,搬送物を識別・制御するための自動認識媒体および認識機器などが必要である.a. 仕分けコンベヤ生産システムにおける物流は材料から製品まで物理的に一つの流れで実現されることはまずない.したがって,物の流れに分岐・合流を行う仕分け作業が発生する.小型部品の仕分けや低頻度物流の仕分けでは,人手を主体とした流動ラックや回転ラックによる小規模な仕分け作業が行われるが,大物部品,高頻度の仕分け作業では比較的大規模な仕分け作業が自動化機器を用いられて行われることが多い.仕分け作業では,行き先情報に応じて所定の方向に荷を分岐させる仕分けコンベヤが用いられる.機械的な仕分け方法によって押出し式,送り出し式(図5・37),斜行式,傾倒式などがある.b. 荷そろえ機器段ボールケース,プラスチックケース,袋(c) 流動ラック(b) ドライブインラック(a) パレットラック図5・35 固定ラック図5・36 移動ラック図5・37 送り出し式仕分けコンベヤ物などの搬送物を定められた荷姿にパレット上に積付けることをパレタイズといい,その装置をパレタイザと呼ぶ.逆にパレット上に積付けられた搬送物を荷卸しすることをデパレタイズといい,その装置をデパレタイザと呼ぶ.JIS の分類では機械的パレタイザとロボット式パレタイザ(図5・38)とがある.c. 包装・梱包機器製品の搬送・保管に当たり,品質保持のために適切な材料,容器等で保護を行うことを目的とする機器で,包装機と梱包機がある.d. データキャリヤ(data carrier)搬送・保管作業においては荷の流れを管理・制御するために個々の荷に固有の情報を付与し,それを生産システムの随所で読み取ることによって荷の保管状況を管理し,物流制御に用いることは不可欠となっている.そのため,情報を認識するための媒体およびその読み書きを行う設備が必要となる.バーコードは最もはん用的に用いられており,一次元に配置されたバーの組合せによって情報を付与し,それを読み取ることによって物の流れを管理するものであるが,取扱える情報量に制約があるため,最近では二次元上にバーを配置した二次元コードが増え始めている.ISOなどでは複数のコード体系が規定されており,これらのコードはその特徴に応じて使い分けられている.図7―105図5・38 ロボット式パレタイザ図5・39認識媒体例工などによって情報が物に同期して搬送される認識媒体に書き込まれるか物に直接印字され,光学的な認識装置によって読み取られる.さらに認識媒体の情報を適宜書き換え,双方向で情報伝達を行えるRFID(radio frequency identification)なども用いられる.磁界,電波,光などによって情報を認識媒体に伝達し,その情報を認識媒体内で書換え可能かつ大容量の情報として保管することができる.荷の管理を自立的に行うことが可能となるので非常に柔軟な生産システムを構築できるが,認識媒体自体のコストが現時点ではまだ高価である.
