# 8・4・1 日本工業標準化法 工業標準化法は，1949年6月1日

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・4 日本の工業標準化

に制定公布され，同年7月1日から施行された．この法律は，戦後
の日本製品の品質の向上と経済成長に大きな貢献をし，その後のガ
ット・スタンダードコードの締結に及んで海外の製造業者に対して
もJISマーク表示制度を利用することができるようになり，国際化
の推進と国内の経済変革の道しるべとしてその存在意義は大である．
国内外の変革に対応すべく，1997年3月の改正までに9回の改正
が行われているが，その法理念は制定当時から引き継がれている，
といえる．
工業標準化の理念の基本は，適正で合理的な工業標準の体系的整
備とその妥当性の保持であって，基本原則は，次のとおりである㈹．
①工業標準の統一性の保持の原則
②工業標準制定の公正な協定制定の原則
③ 工業標準化の客観的妥当性及び合理性保持の原則
④工業標準の公衆性保持の原則
作成された規格は，その適用によって具体化される．すなわち，
企業における工業標準の実施の確保，技術的生産条件の改善と科学
的な品質管理の促進である．これらの促進は，表示制度の対象とな
る品目又は加工技術の指定，表示認定，工場等審査制度，品質管理
努力などを通じて行われる．そして，最終的には，表示制度のもと
での取引の公正化を求めるものである．
表示制度そのものは，JISマーク表示による生産者の自己保証で
ある．消費者がJISマークによって品質の良さを確認することがで
きるシステムを法制化している．
