# 6・1・1 生産のための情報

> **章**: 第6章 情報システム
> **節**: 6・1 生産システムの情報化の体系

a. 生産に関連する機能と情報の流れ企業における生産に関連する機能全体と、その間の主要な情報の流れ(一部に物の流れを含む)を図6・1に示す。図中でグレーになっている機能が、生産情報システムと密接な関係を有する機能であり、以下ではそれらの機能と関連する情報について解説する。b. 生産管理生産システムの管理の中枢を司る部分であり、品質・コスト・納期の管理を行う。販売からの受注情報を受けて、調達への購入指示や製造への生産指示を出すとともに、製造からの生産状況を受取り、出荷への出荷指示や販売への製品品質情報の伝達なども行う。この生産管理を行うための基本情報は、製品設計からの部品構成表と、生産準備からの部品工程情報である。c. 製造生産準備からの製造技術情報および生産管理からの生産指示により加工や組立を行う。その際には、調達からの原材料や購入部品、リソース管理からの工具・ジグなどのリソースが入力となる。d. 生産準備製品設計からの製品仕様情報を入力として、製造へ製造技術情報、生産管理へ部品工程情報(加工工程と加工時間など)、調達へ購入仕様を出力する。e. 調達生産管理からの購入指示を受けて調達を行い、工具・ジグなどのリソースに関する情報をリソース管理に渡す。f. リソース管理調達から受取ったリソースを保管し、製造からの生産状況を受けて、タイムリーに製造にリソースを渡す。ここでいうリソースとは、設備、工具、ジグなど生産に必要となる資源を示す。g. 設備管理製造からの保全要求により、設備保全を行う。
