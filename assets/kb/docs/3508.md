# 6・1・3 生産情報システムの変遷

> **章**: 第6章 で解説する生産情報システムの構成要素と、生産に関連す
> **節**: 6・1 生産システムの情報化の体系

生産情報システムは、企業における生産方式の変化と、コンピュータ技術の進歩と関連をもちながら変遷してきている。a. 大量生産時代の生産情報システム企業における生産方式の変遷を見ると、1980年ごろまでの大量生産時代と、その後現在までのフレキシブル生産時代に大別される。前者の大量生産時代は、高品質な同一または類似の製品を大量に生産するための製造技術を重点的に開発した時代であり、わが国の企業における製造技術の基本が確立された時期といってもよい。この時代の生産情報システムは、以下の二つがその中核を占めていたと考えられる。. 生産管理を中核とした企業内生産情報システム大型コンピュータ(メインフレーム)を使用した生産管理システムや購買システムであり、それまでは手書きで行われていた作業指示・購買に関する書類作成、および各種の集計作業の自動化がおもなねらいであった。. 生産準備系の生産情報システム数値制御工作機械(以下、NC工作機械と呼ぶ)は、わが国の企業においても1970年代の半ばから大量に導入されるようになった。このNC工作機械のためのNCプログラム作成を支援するツール、自動プログラミングシステム(通称、自動プロ)は、この時代から使用が始まってい7―113る。そのおもな用途は、曲面加工など手計算が不可能なNCプログラムに対してであった。b. フレキシブル生産時代の生産情報システム顧客の需要構造の変化に伴い、製造企業に対して従来からの生産方式の変更を余儀なくされた結果がフレキシブル生産システム(FMS)の出現といえる。このフレキシブル生産方式を支援する面で、生産情報システムの役割は非常に大きいと考えられる。以下では、その変遷について記述する。. 個別システムによる対応この時代の初期においては、次に示すような個別のシステムでフレキシブル生産方式の支援を行った。(1) 最適な生産計画の作成企業内統合情報システムの一つの機能であり、変化する顧客需要に対しても適正在庫量の中で、納期・コストを満足させる最適な生産計画を立案する。(2) フレキシブル生産システムの最適な制御NC工作機械を中心に構成されたFMS,FTL,FMC(flexiblemanufacturing cell)などをコンピュータを使用して最適な制御を行う機能であり、製造制御システムそのものである。(3) NCプログラムの作成支援従来からの曲面加工に加えて、一般のマシニングセンタやNC旋盤などでの加工プログラム作成支援システムが普及し、近年では、NC制御装置内にこの機能が内蔵されているものも増加している。. 全社統合システムの実現これは、i項の個別システムの機能をさらに充実させたうえで、それらを全社統合システムとして実現させたものであり、1990年代の前半に実用化された、CIM(computer integrated manufacturing)とも呼ばれるシステムである。CIM は本来、企業における販売・開発・生産の各機能をコンピュータネットワークにより統合して、生産リードタイムの短縮や在庫量の提言などを図ろうとしたものであったが、実用化された多くのシステムは、販売システム・生産管理システム・FA(fac-tory automation)システムを統合したものであった。この中のFA システムは、生産工場内の加工システム(FMS,FMCなどを含む)・組立システム・検査システム・保管用自動倉庫などを内蔵する大規模システムである。このCIM に代表される全社統合システムは、初期投資金額が非常に大きいことなどの理由から、バブル経済の崩壊後は新規に構築された例はほとんど見られない。. ネットワーク技術の有効活用バブル経済の崩壊後においては、生産設備そのものへの投資はかなり抑制されているものの、企業の情報(IT : information technology)分野への投資は継続されており、生産情報システムにおいても、ネットワーク技術を活用したグローバリゼーションへの対応などで着実に前進を続けていると見られる。その例を以下に紹介する。(1) 企業間統合情報システムの充実インタネットなどのネットワーク技術を活用して、この分野ではこの数年間で急速にシステムが充実してきている。その代表例としては、サプライチェーンマネジメント(SCM)システムが挙げられる。(2) 三次元CAD 情報の有効活用近年急速に普及しつつある三次元CAD で作成された製品情報を、生産情報システムの中で有効に活用する面ではまだ十分とはいえず、今後、特に生産準備系生産情報システムの充実が望まれる。
