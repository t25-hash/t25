# 6・2・1 生産準備業務の概要

> **章**: 第6章 情報システム
> **節**: 6・2 生産準備と情報システム

生産準備部門は,設計部門から製品情報を受取り,業務を行い,製造部門に加工技術情報,生産管理部門に部品工程情報,調達部門に購入仕様を渡すなどの役割をもつ.その代表的な業務としては,工程設計,加工設備・ジグ・工具の設計,加工ラインの設計,設備の購入計画などが挙げられる.対象とする工法も,機械加工,組立,板金加工,熱処理,溶接,鋳造,鍛造など多岐にわたっている.この生産準備業務をコンピュータ支援で行うことを,一般にCAM(computer aided manufacturing)と呼ぶことが多いが,6・2節では上記の生産準備業務の中から機械部品の工程設計業務を取上げて,生産準備と情報システムとの関係を解説することにする.
