# 6・5・1 製造制御システムの役割

> **章**: 第6章 情報システム
> **節**: 6・5 製造制御システム

生産活動は,物を生産するための要求から始まり具体的な形にするための設計,どのように作るか生産の手順や計画を決める工程設計や作業設計などがなされ,その後に実際の製造が行われる.完成した部品や製品は搬送され出荷される.このように製品や部品がわれわれの手に届くまでには多くの行為や過程がある.これらの生産活動の流れは,情報の流れと物の流れに大別できる.情報の流れはさらに技術情報と管理情報に分けることができる.技術情報は,部品や製品をどのように生産するかの技術や手法に関する情報で,主として設計,工程設計,作業設計などの工程で扱われる.技術情報は,実際の加工,組立および検査を行うための基本的な情報となる.一方,管理情報は,いつまでにどのくらいの数量を生産するのかといった生産計画に関する情報と,実際の生産が要求どおり実施されているかどうかの生産管理に関する情報である.生産活動を効率良く実施するために,生産計画では,工場の設備,要員,素材の手配,費用および納期に関する情報などを基礎に生産スケジュールを立案する.実際の加工が始まると,生産スケジュールに沿って作業が進捗しているか,作られた部品や製品が要求どおりに作られているかどうかの品質管理を行い,さらに機械設備の管理のために保守・保全などの情報も扱う.物の流れは,素材の納入に始まり加工,組立,検査さらに出荷に至る流れで,言い換えれば実際の製造工程である.物の流れにおいては上述の技術情報や管理情報に基づき生産がなされるが,ここでは実際のFA 機器や加工機械,ロボット,自動倉庫,自動搬送台車(AGV)といった種々の機械の制御情報が重要な役割を果たす.以前の生産システムでは,情報の流れと物の流れが独立していたために生産を効率良く実施することができなかった.しかし最近では,コンピュータとネットワーク技術の進展により情報の流れと物の流れを同時に制御しながら生産することが可能となってきた.FMS やコンピュータによる生産システムではこのような仕組みを取り入れることにより,生産効率が良くなり,生産リードタイムも短くすることが可能となった.おもに物の流れを制御するための製造制御システムは,生産技術情報を基に実施される.製造制御システムにおける要素技術の中核はFA 機器を制御するための制御ネットワーク(control net-work)技術である.
