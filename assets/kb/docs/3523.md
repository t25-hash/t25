# 6・5・4 イーサネットをベースとした制御ネットワーク

> **章**: 第6章 情報システム
> **節**: 6・5 製造制御システム

イーサネットは主としてOA 系の情報ネットワークとして標準化されているが、近年この傾向がFA 分野にも及んできている。現在は10Mbps~100Mbpsの伝送速度が主流である。さらに将来は1Gbpsの高速LAN への規格化も検討が進んでいる。イーサネットで使用されるTCP/IP(transmission control pro-tocol/internet protocol)は情報伝送のプロトコルとして広く採用されている。そこで、イーサネットをFA コントローラ(FA controller)間の通信媒体として使用し、特に基本となるTCP/IP,UDP/IPプロトコルの利用と、新たにFA コントローラ間通信を定義し、サイクリック伝送およびメッセージ伝送を実現するプロトコルを使用した制御ネットワークが実用化されている。イーサネットをベースとした制御ネットワークでは、・トータルシステム構築の容易性・IEC1131-3への対応が容易・パソコンとの高い接続性、互換性・豊富なAPIの利用が可能・オープン性が高い・将来性が期待される・ベンダが共通に開発可能など多くの利点が考えられる。表6・2 FL-netのおもな仕様サイクリック伝送ノード数伝送容量~8kbit~8kword(1word=2byte)スループット50ms~(片側スループット)メッセージ伝送通信形態1:1,1:N伝送容量1kbyte/フレームスループット~500ms(サイクリック伝送条件下でのスループット維持)コマンドデータ読出しレスポンスデータ書込み機器制御スタート・ストップリセットアラーム情報の通知その他ファイル転送プログラムデータ構造体ネットワーク情報ネットワークパラメータノードアドレスネットワーク管理情報メンテナンス情報その他7―132生産システム工学図6・43 FL-netのプロトコルスタックわが国において開発されたオープンな制御ネットワークとしてはFL-netがある。FL-netの特徴としては、・イーサネットをFA コントローラ間の通信媒体として採用・サイクリック伝送には、UDP/IPのブロードキャスト通信機能を利用・上記の基本的通信機能を使用しつつネットワーク内の各ノード間のアクセスを制御し、一定時間内の伝送を保証するなどが挙げられる。表6・2にFL-netのおもな基本仕様を示す。これを実現するためにFA リンクプロトコルが新たに開発され、UDP/IPの上位層として実現されている。FL-ネットを構成するプロトルスタックとしては図6・43に示すような構成となっている。FA リンクプロトコルの特徴としては、マスタレストークン方式により送出管理を行い衝突を回避する、トークンを一定時間で周回させることによりリフレッシュサイクル時間を規定できる、定められたトークンをサイクリックデータとともに送信できる、一定時間トークンが送出されない場合は次ノードがトークンを送出する、マスタレストークン方式により一部のノードが故障してもネットワークが停止しない、さらに運転の状況や監視などの管理情報テーブルを用意することで他のノードの動作状態を参照できるなど、製造制御に適した多くの特徴を備えている。制御ネットワーク用機器の認証制度はユーザの強く望むところであるが、過去の経験からその実施には多大な労力と費用がかかることが予測される。認証のための試験項目としては、適合性試験、相互運用性試験、通信性能試験、耐ノイズ試験などがあるが、適合性試験に関してはベンダ等による自主試験の方向やリモート試験など、信頼に足る新たな認証システムの構築が必須であろう。図6・44はFL-netを製造・組立ラインに適用した例であり、従来のプライベートネットワークからオープンなFL-netの採用により余計なインタフェースを介することなく通信が可能となりコストが大幅に削減された。
