# 9・2・1 旋盤・ターニングセンタ

> **章**: 第9章 加工機械
> **節**: 9・2 工作機械の種類

旋盤（lathe）は工作物を回転させて切削運動とし，おもにバイ穴あけ，中ぐり，ねじ切りなどの加工を行う機械である．
最も基本的なものは，普通旋盤（engine lathe，図9･2) であり，主軸を回復台，工作物の一端を支持する心押し台とこれらを載せるベッドでれる工具保持台をもつタレット旋盤，大形の工作物の旋削を可能とがある．
工工程をプログラム化した指令によって実行できる機械である．
