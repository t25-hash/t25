# 6・6・1 生産システム用ソフトウェア技術構成

> **章**: 第6章 情報システム
> **節**: 6・6 生産システム情報化のためのソフトウェア基盤技

生産システム(manufacturing system,production system)の一般的機能構成を図6・47に示す。生産システムでは、複雑な情報の流れが、物の流れを制御している。このシステムを支援する生産情報システムは、大きく技術情報システム、生産管理情報システム、生産プロセス制御システムからなる。技術情報システムは製品設計・工程設計情報の生成、生産管理情報システムは生産活動の計画・統制情報の処理、また生産プロセス制御システムは生産設備の動作制御と管理による生産活動の実行を目的とする。生産情報システムの構築には、異なるベンダのソフトウェア・ハードウェア間での相互運用性(interoperability)の確保が必須となる。この相互運用性を保つソフトウェア基盤技術として、1計算機OS(operating system)、2計算機ネットワーク技術、3分散する情報の交換を容易にする分散オブジェクト技術、4各種の生産図6・47生産システムと生産情報システムの構成表6・3 生産現場で利用される計算機ハードウェアの分類計算機ハードウェア制御・管理の対象特徴・使用目的FA コンピュータエリア・セルプログラマブルロジックコントローラ(PLC)セル・ステーションマイクロコンピュータ装置7―133情報を保存・管理するためのデータベース技術、5生産情報の正確な交換に必要な情報モデリング技術などが活用されている。また生産情報システムの開発コスト・開発期間・拡張性などの観点から、可能な限りオープンで標準化された技術規格の利用が図られている。
