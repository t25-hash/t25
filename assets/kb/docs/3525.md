# 6・6・1 生産システム用ソフトウェア技術構成

> **章**: 第6章 情報システム
> **節**: 6・6 生産システム情報化のためのソフトウェア基盤技

生産システム（manufacturing system，production system）の
一般的機能構成を図6･47に示す．生産システムでは，複雑な情
報の流れが，物の流れを制御している．このシステムを支援する生
産情報システムは，大きく技術情報システム，生産管理情報システ
ム，生産プロセス制御システムからなる．技術情報システムは製品
設計・工程設計情報の生成，生産管理情報システムは生産活動の計
画・統制情報の処理，また生産プロセス制御システムは生産設備の
動作制御と管理による生産活動の実行を目的とする．
生産情報システムの構築には，異なるベンダのソフトウェア・ハ
ードウェア間での相互運用性（interoperability）の確保が必須と
なる．この相互運用性を保つソフトウェア基盤技術として，①計算
機OS（operating system），②計算機ネットワーク技術，③分散
する情報の交換を容易にする分散オブジェクト技術，④各種の生産

6・6・2生産システム用計算機とオペレーティングシステム
a.計算機ハードウェア 生産情報システム内で利用される計
算機は，大きく二つに分けられる．技術情報システムや生産管理情
報システム向けには，ビジネス向けPC（パーソナルコンピュー
タ）やサーバ機などはん用機が利用される．一方，生産プロセス制
御システムや設備に内蔵される計算機には，生産現場への設置に適
した専用機が利用される．すなわち，24時間連続稼働を前提とし
た耐久性・信頼性，耐環境性，処理のリアルタイム性，外部機器と
の接続性，自動復旧機能，保守部品等の長期安定供給などの特殊仕
