# 6・6・2 生産システム用計算機とオペレーティングシステム

> **章**: 第6章 情報システム
> **節**: 6・6 生産システム情報化のためのソフトウェア基盤技

a. 計算機ハードウェア生産情報システム内で利用される計算機は、大きく二つに分けられる。技術情報システムや生産管理情報システム向けには、ビジネス向けPC(パーソナルコンピュータ)やサーバ機などはん用機が利用される。一方、生産プロセス制御システムや設備に内蔵される計算機には、生産現場への設置に適した専用機が利用される。すなわち、24時間連続稼働を前提とした耐久性・信頼性、耐環境性、処理のリアルタイム性、外部機器との接続性、自動復旧機能、保守部品等の長期安定供給などの特殊仕様を満たす計算機である。生産現場で利用される計算機ハードウェアを、表6・3に分類する。FA コンピュータは、おもに工場のセルやフロア単位の制御に・耐環境性に優れた筐体に収納される。・はん用PC と同じMPU が用いられることが多い。・ディジタル入出力をもつ。・ネットワーク通信インタフェースをもつ。・CRT や液晶パネル等と一体化されたものもある。・小型筐体に収納され、専用MPU をもつ。・拡張容易なユニット形態をとる。・ディジタル・アナログ入出力・位置決め装置入出力等をもつ。・数値制御装置・センサ・アクチュエータ・操作盤と接続される。・ネットワーク(コントロールバス)通信インタフェースをもつ。・セルまたはステーション内装置群のシーケンス制御を行う。・機器に収納可能な1ボード形状である。・ディジタル・アナログ入出力をもつ。・制御プログラムダウンロード、起動・停止制御、状態報告などのネットワーク通信機能をもつ。・工作機械・ロボット等の数値制御装置、自動機の制御装置などとして使用される。7―134生産システム工学利用されるが、はん用PCと類似のハードウェアを有するものも多く、市販のパッケージソフト等も動作可能であることが多い。PLC(programmable logic controller)は、工作機械、ロボット、自動組立機、搬送機などの設備や、センサ、アクチュエータ、操作盤などに接続されシーケンス制御(sequence control)を行う。またマイクロコンピュータは設備の制御装置内に実装され、数値制御(numerical control)を行う。またセンサやアクチュエータごとに、ネットワーク通信機能をもつ1チップ型小型プロセッサを接続した分散制御システム(DCS : distributed control system)もプロセス系生産システムで増加している。b. オペレーティングシステム(operating system)計算機の性能向上と多機能化に伴い、FA コンピュータのみならず、制御装置内のマイクロコンピュータにもオペレーティングシステムを搭載した計算機が増えている。オペレーティングシステム(OS)は、メモリ・ディスク・ネットワークなどのハードウェア資源を管理し、アプリケーションがこれを利用するための基本ソフトウェアである。はん用機では、Unix,Windows,Linux 等のOS が標準的に利用される。一方、設備や機器の動作制御など応答時間に厳しい高速性(数十s~数十ms以内)が求められる場合には、リアルタイムOS(real time OS)が用いられる。リアルタイムOS には、VxWork,iRMX,ITRON,OSEK/VDX などがある。また小型機器組込用OS として、Windows XP Embedded・WindowsCE.NET 等も利用され始めている。
