図6･53ウェブサービスの基本機能
ビスの提供者は，提供するサービス定義情報を，あらかじめブロー
カのもつサービス検索・照会システムであるUDDI （universal
description,discovery and integration）内にWSDL（web ser-
vices description language）という言語で記述し登録しておく．
利用者は，UDDIに対して必要なウェブサービスの検索を行い，得
られたサービス提供者との間でサービスの実行が行われる．ウェブ
サービスの利用により，社外システムと社内システムとをシームレ
スに連携させた情報処理サービスを柔軟に構築できるため，在庫管
理やSCM用の情報システムなどに普及しつつある．
テムは数多くのアプリケーションソフトウェアからなるが，これら
はお互いに製品構成，製造プロセス，生産計画，生産実績，在庫，
保守サービスなどの情報をデータベース（data base）として管
理・保有している．
