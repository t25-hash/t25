# 9・3・1 工作機械の形状創成運動

> **章**: 第9章 加工機械
> **節**: 9・3 工作機械の選択

a.形状創成運動の構成 工作機械による加工は，工具と工作
物を構造によって堅固に保持し，かつ両者に相対運動を与え，工具
と工作物の干渉部分を切りくずとして除去することによって行われ
る．その結果，工作物には工具形状と工具・工作物間の相対運動が
転写され，所要の形状が得られる．したがって，高精度で高能率の
加工を実現するためには，工作機械の幾何学的精度だけではなく，
高い静剛性，動剛性，熱変形特性，ならびに運動精度を具備するこ
とが要求される．
工作機械による代表的な加工方法を図9･22に示す．旋削，中ぐ
り，穴あけ，フライス削り，研削加工などの機械加工により，円筒
面や平面などのさまざまな幾何学的部品形状が，所要の工具および
工作物の形状創成運動により得られる．表9･1には，工具と工作物
の相対運動に着目し，工具と工作物に異なる形態の運動を与えるこ
とにより実現される加工方法 を示す．
最近，工作機械の一構造形態（structural configuration）とし
て空間6自由度を有するパラレルメカニズム（parallel mecha-
nism) を応用した工作機械の開発が国内外で進められている．
