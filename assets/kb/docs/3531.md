# 6・6・7 情報モデリング技術

> **章**: 第6章 情報システム
> **節**: 6・6 生産システム情報化のためのソフトウェア基盤技

生産情報処理の分散化・グローバル化が進むにつれ、システム間で情報を正確に交換するため、情報の仕様を人間と計算機が理解可能な形式であいまいさなく記述した情報モデル(informationmodel)の重要性が増している。情報モデルの仕様記述には、できるだけ実装から独立した形式言語(formal language)が用いられる。形式言語は、テキスト形式言語(textural language)や図的言語(graphical language)で記述される。現在、情報モデルを明示的に記述することが一般的な領域として、ソフトウェア設計、データベース設計、製品ライフサイクルデータ交換、インタネット上のドキュメント交換が挙げられるが、今後さまざまな情報交換の局面において情報モデリングを利用する機会が増えると思われる。a.
