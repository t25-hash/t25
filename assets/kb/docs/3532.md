# 9・3・3 加工要求と工作機械の選択

> **章**: 第9章 加工機械
> **節**: 9・3 工作機械の選択

加工要求に適合した工作機械を合理的に選択するためには，まず
工作機械の設計原理である母性原則（copying principle）の具備
の達成が必要不可欠であり，そのためには，以下の条件を満足しな
ければならない．
(１)所要の形状創成運動の正確な実現
(２)変位基準の設計による構造の実現
(３)全体構造による所要の加工空間（machining space）の実現
これらを考慮した工作機械の選択プロセスを模式化した結果を図
9･25に示す．加工要求に基づいて所要の加工空間内で形状創成運
動を実現することのできる構造形態を具備した工作機械を，一連の
プロセスを経て選択することが可能となる．その際に，加工精度や
加工能率は，最適な工作機械を選択するために必要な選択評価基準
となる．
工部品群から最適な工作機械の構造形態を決定しようとする試
み が行われているが，対象とする問題を限定，あるいは単純化
する以外に系統的な処理が困難である．工作機械の構造設計におい
ては，構造形態の創成 と創成された候補案からの最適な構造形
態の選択を定量的な評価を経て段階的に行う ことが現実的であ
る．
