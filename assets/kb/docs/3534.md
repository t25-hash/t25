# 9・4・2 位置決め精度・空間精度・円運動精度

> **章**: 第9章 加工機械
> **節**: 9・4 工作機械の性能評価

a.位置決め精度 位置決め精度（positioning accuracy）の
評価は，目標とする位置に正または負の方向から繰返して位置決め
したときの目標位置と実際の位置との差の標準不確かさを考慮し
て，移動の全長にわたって起こりうる狂いを評価し，その最大値を
測定値として求める．ISO230-2：2006に規定する評価方法は，
直進軸の移動範囲2000mm以下の場合に適用され，それよりも範
囲の大きな大形機の場合には，よく使用すると想定される2000
mmの範囲だけに適用される．
位置決めをする目標位置は，測定長さ1000mmについて少なく
とも5位置，1000mm未満の場合には全移動範囲で少なくとも5
位置を定めて往復5回の測定を行う．この測定によって得られた目
標位置と実際の停止位置との差を位置決め偏差として定義し，各目
標位置において正の向きに近づけた場合と負の向きに近づけた場合
との位置決め偏差の平均値および標準不確かさを求める．その後に
表9･4に示す項目について評価する．

b.空間精度 加工空間のひずみの程度を調べる一つの方法と
して，空間精度（volumetric accuracy）試験が規格化された．
ISO規格では，対角位置決め試験（diagonal displacement test ）
とも呼んでいる．加工空間を直方体であると考え，その直方体の一
つの長方形の対角線に沿って位置決め精度の測定を行う平面対角位
置決め（plane diagonal positioning），直方体の対角線に沿って行
う立体対角位置決め（body diagonal positioning）とがある（図
9･26) ．測定は，位置決め精度試験と同じ方法で行う．この試験
方法は，直交2軸または3軸をもつ工作機械の評価に適している．
c.円運動精度 円運動試験（circular motion test ）は，二つ
の直進軸を制御して円弧補間運動させたときの軌跡と理想円との差
を比較することによって，NC工作機械のさまざまな狂いを総合的
に評価することのできる試験方法である．この試験を実施すること
によって，運動の真直度や直角度などの幾何学的な精度だけでな
く，バックラッシ量，半径縮小，位置ループゲイン調整の適否など
を評価でき，特に輪郭制御をすることの多いマシニングセンタの検
査には欠かせない測定方法 になっている．
この測定のためにISO230-1：1996には3種類の測定器を規定
しているが，二つの球とその間に変位センサを内蔵した伸縮バーを
備えたボールバー（ball bar ）が最も広く使用されている（図9･
27) ．これ以外の同等以上の性能をもつ測定器として，交差格子
線を用いたグリッドエンコーダと呼ばれる二次元形状の測定に適し
た測定器 が開発されている．これを使用すれば，任意の二次元
形状の軌跡を切削することなく評価できる．
