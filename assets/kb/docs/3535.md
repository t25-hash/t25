# 9・4・3 熱変形評価

> **章**: 第9章 加工機械
> **節**: 9・4 工作機械の性能評価

さまざまな熱源が原因で発生する工作機械構造の変形や，テーブ
ルなどの位置決め系に発生する熱影響（thermal eﬀect ）が工作物
の形状と寸法に関係することから，熱変形の評価は工具と工作物と
の相対距離変化を測定することによって行う．そのために，まず
(１)工作機械を設置した環境の温度変化の影響を調べ，次に(２)主
軸を回転させたときの影響および(３)テーブルを繰返して移動させ
たときの影響をそれぞれ評価する．
一例として測定器の設置状態を図9･28 に示す．規格で定めて
いる測定条件は，機械が室温の状態から4時間運転し，運転を停止
した後少なくとも1時間は測定することが条件となっている．熱変
形評価の特徴は，測定に長い時間がかかることである．それは，機
械が十分に冷えた状態から測定を開始することを条件としているた
めに，一つの測定を終わった後にすぐに続けて次の測定ができない
からである．
