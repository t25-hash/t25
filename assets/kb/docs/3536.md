# 9・4・4 振動・騒音評価

> **章**: 第9章 加工機械
> **節**: 9・4 工作機械の性能評価

a.振動の評価 工作機械の振動の中でも最も重要な振動であ
るびびり振動の発生しにくさは，工作機械の性能を決定する一つの
指標と考えられている．しかし，工作機械の性能評価は切削しない
で評価し，切削は確認のために行うことを原則としていることか
ら，切削を伴うびびり振動の試験方法の標準化には取り組んでいな
い．ISOでは，(１)工作機械を据え付けたときの工場床からの振
動，(２)主軸の不釣合いによる振動および(３)急加減速したときの
テーブルの振動の3種類の振動の大きさを評価することを目的とし
て規格化を進めている．
工場床に据え付けた機械に伝搬する外来振動は，工作機械が稼働
すると予想される時間の間，連続して測定する．これはバックグラ
ウンド振動であり，(２)と(３)の振動測定で検出された振動が外来
振動であるかどうかを識別するためのデータになる．主軸の不釣合
いによる振動は，主軸端近くの軸受ハウジング近傍に取付けた加速
度計による測定を基本とし，速度をさまざまに変えて測定する．ま
た，送り軸の急加減速時の振動については，テーブル側に定置した
直定規と主軸側に取付けた変位計とによって測定する．
b.騒音の評価 ISO230-5：2000では，労働安全衛生と関係
して，作業者や立ち寄った人（bystander ）が機械のそばのある特
定位置で騒音にさらされることを想定して，騒音放射の音圧レベル
と音響パワーレベルの評価方法を規定している．騒音放射は，作業
者位置のほかに機械のまわりに多数のマイクロホンを設置して，所
定の運転条件で同時に測定する．評価項目は，表9･5に示すとおり
である．
