# 9・4・5 工作精度および加工能力の評価

> **章**: 第9章 加工機械
> **節**: 9・4 工作機械の性能評価

a.工作精度の評価 工作精度試験（machining test ）は，
ISO規格やJISの機種別規格に詳細に規定されている．その一例
として図9･29 に示すマシニングセンタの工作精度試験方法があ
る．この試験では真円度，真直度，直角度，同軸度などを評価す
る．この規格のほかに5軸制御マシニングセンタでは，古くからア
メリカ航空宇宙規格（National Aerospace Standard）NAS
979：1969 に準拠して同時5軸制御による円すい台の仕上削りが
行われてきた．これらの工作精度試験は，静的精度や補間精度
（interpolation accuracy）が工作物に転写されることを確認する
ために行うことを主眼としている．
b.加工能力の評価 びびりの発生限界を調べるためのさまざ
まな方法が提案されてきたが，工具，工作物およびその取付具の影
響が含まれてしまうことはよく知られている．そのため，例えば，
どのような旋盤にでも適用できるはん用性のある方法は確立されて
いない．しかし，多くのメーカでは独自の試験方法を採用し，評価
している．その評価結果は，自社の他の機械と比べてどの程度能力
が向上したかを知るために使用されていて，他社の機械と比較して
より多く削れるというような結果を出すためには使用されていな
い．
