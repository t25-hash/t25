# 9・5・3 主軸構造およびその駆動機構

> **章**: 第9章 加工機械
> **節**: 9・5 工作機械の基本構造

主軸（spindle）およびその駆動機構からなる主軸系の一例を図
9･41 に示す．このような主軸駆動系は，基本的には主軸，主軸
変速機構，モータ，検出器で構成されている．主軸は，切削抵抗に
抗して工具あるいは工作物をしっかりと支持するとともに，心振れ
などがないように正確に回転し，その回転速度も安定している必要
がある．最近この高精度化とともに，高速・高加減速化が強く要求
されている．
a.主軸構造 主軸には，高剛性で軽量コンパクトであること
が要求される．しかしながら，主軸には多くの機能を内蔵させる必
要があり，そのコンパクト化は容易ではない．例えば，ツールホル
ダを取付けるための主軸端部，そのクランプならびにアンクランプ
機構とそのための駆動機構，クーラントなどの流体供給機構（ロー
タリジョイント）とそれらの供給通路，主軸・主軸受の冷却機構，
オートバランサなどが挙げられる．主軸端部の形状は高速化に対応
するために，最近は図9･42(ａ)に示すような二面拘束形（double
face contact type）のものが多くなってきている．図9･42(ｂ)の
ような従来形の主軸では，高速回転時の遠心力によりテーパ穴が膨
張することから，その分ツールホルダのシャンク部がテーパ穴に引
き込まれ，結果的に刃具の刃先位置が変化してしまうとともに，そ
のまま停止するとシャンク部がテーパ穴にロックしてしまい，ホル
ダの交換が不可能になる現象があった．このような問題を解決し，
さらに高精度・高剛性化が図れることから，二面拘束形のツールシ
ャンクが注目され，高速用を中心として普及している．クランプ機
構も二面拘束形に対応するために増力機構内臓形となっており，そ
の駆動機構は皿ばねからコイルばね，らせん皿ばねなどに変わりつ
つある．また，さらなる高速化，コンパクト化に向けて，油圧を用
いたり，圧縮気体のばね作用を用いたりするものも開発されてい
る ．
b.主軸受（main bearing) 主軸系で最も重要視されている
のは主軸受である．それは，その回転精度などが直接的に加工精度
に影響を及ぼすからである．工作機械では表9･9に示すようなもの
が，主として採用されている．これらの中でも転がり軸受が，性
能，コスト，メンテナンス性などの面から最もよく使われている．
.転がり軸受（rolling bearing) 転がり軸受の性能を大
きく左右するものとして，軸受の潤滑法と予圧法が挙げられる．潤
滑法としては，最近ではグリース潤滑，オイルエア潤滑がよく用い

受にあらかじめ軸方向の荷重を付加する．これを予圧と呼ぶが，こ

.静圧軸受（hydrostatic bearing) 静圧軸受の基本原理
は静圧案内と同じで，軸受部に圧油または圧縮空気を供給して軸を
浮上させるもので，高い回転精度が必要とされる研削盤や超精密工
作機械に主として用いられてきたが，最近では高速高精度マシニン
グセンタにも採用されるようになってきた．
.動圧油軸受（hydrodynamic bearing) 動圧油軸受の原
理図を図9･43(ａ) に示す．軸の回転により軸受内に満たされた
油がくさび状の軸受すきまに引き込まれ，圧力を発生し，軸を浮上
させる．一般的には，図9･43(ｂ)に示すように，このくさび状す
きまを3箇所以上設けて軸の求心性能を高めている．このくさび効
果を発生するための各種機構が提案されている．動圧油軸受は回転
速度があまり変動しない円筒研削盤や平面研削盤を中心に用いられ
てきたが，最近では静圧油軸受と同様，高速マシニングセンタにも
採用されるようになってきた．
.磁気軸受（magnetic bearing) 図9･44 は，磁気軸
受を用いた主軸構造例である．主軸の半径方向の変位と傾き変位を
制御する4組8個の電磁石と，軸方向の変位を制御する1組2個の
電磁石により構成されている．軸の支持は，対抗する磁石の吸引力
を制御することにより行われている．この静剛性，動剛性は制御系
の特性に影響されるので，高度な制御技術が必要である．
