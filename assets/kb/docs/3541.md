# 9・5・4 周辺機器

> **章**: 第9章 加工機械
> **節**: 9・5 工作機械の基本構造

図9･46 は，マシニングセンタ本体とそれに付属している周辺
機器の一例を示している．前述のように工作機械は，図9･46に示
すような各種周辺機器なしにはその機能を発揮することはできな
い．周辺機器のうち，数値制御（NC）装置については9･6節で述
べるので，ここではそれ以外の周辺装置の概要を述べる．
マシニングセンタでは自動工具交換装置（ATC:automatic
tool changer ）は必須のもので，周辺装置というよりは機械本体構
成要素の一つといってもよい．切りくず処理装置（chip disposal
system）もほとんどの機械に付属している装置といえ，ATCと並
んで工作機械を中核とする生産システムの無人化には必須の装置と
なっている．さらなる無人化を進めるには自動工作物交換装置が必
要となり，長時間無人化を実現するためには，10台以上のパレッ
トで構成される自動パレット交換装置（APC:automatic pallet
changer ）も使われ，その形式にも各種のものが存在している．
