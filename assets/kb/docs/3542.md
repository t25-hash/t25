# 7・2・1 生産設備の地球環境問題への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・2 生産システムの省エネルギー

生産プロセスでの生態系負荷の最小化を実現するためには,徹底した省エネルギー(energy saving),省資源,廃棄物削減,生態系7―156生産システム工学有害物質使用中止を実現する必要がある.生産活動に伴って消費されるエネルギーは,地球温暖化問題,化石燃料枯渇,大気汚染などの原因となる.とりわけエネルギーの生産・消費段階で発生する二酸化炭素(CO)は大気中の温室効果ガス濃度を増加させることにより,地表および大気の温度を地球規模で上昇させて生態系のバランスを崩すことから,省エネルギーによる温室効果ガスの削減は国際的な最重要課題である.地球温暖化問題への取組みを定めた「気候変動に関する国際連合枠組条約」が1994年に発効し,先進国の温室効果ガスの削減を法的拘束力をもつものとして約束する京都議定書(Kyoto proto-col)が1997年に採択された.排出の抑制および削減の対象となる温室効果ガスは,二酸化炭素(CO),メタン(CH),一酸化二窒素(NO),ハイドロフルオロカーボン(HFC),パーフルオロカーボン(PFC),六フッ化硫黄(SF)である.これらは燃料,洗浄剤,冷媒,電気材料,廃棄物等から排出され,その削減には生産システムが大きな貢献を果たしうる.日本国内では,温室効果ガスの総排出量を2008年から2012年の間に1990年(一部は1995年)レベルから6%削減することを内容とする京都議定書での国際的な約束のもとで,「地球温暖化対策の推進に関する法律」「エネルギーの使用の合理化に関する法律(省エネルギー法)」などの制定や改正が行われ,法的規制が強化された.日本の場合,温室効果ガス排出量の90%は石油,石炭,天然ガス等のエネルギーを起源とした二酸化炭素であることから,省エネルギーが最も温室効果ガス排出削減に寄与する要素である.日本のエネルギー消費を部門別で見ると,産業部門でほぼ50%,民生部門と運輸部門でそれぞれほぼ25%となっており,産業部門での省エネルギーは非常に重要な役割を期待されている.しかし,産業部門での生産額当たりのエネルギー消費原単位,二酸化炭素排出原単位は,欧米諸国に比較してすでに低い水準にあるため,産業活動の成長力を失わない中でエネルギー消費の総量増加を抑えるという難しい課題がある.温室効果ガス排出削減に貢献できる風力発電,太陽光発電,太陽熱利用,バイオマスエネルギー,燃料電池などの新エネルギーを工場単位,部門単位,装置単位で積極的に利用するとともに,生産設備のエネルギー効率を高める努力が必要である.新エネルギー源の多くは小規模単位での分散型利用が可能であるため,エネルギー管理システムを高度化し,エネルギー発生特性に適応できる生産システムを開発することも有効である.工場での大規模エネルギー利用については,エネルギー管理指定工場として法規制の対象となっている.法律では,工場全体だけでなく設備群単位や作業工程単位でのエネルギー管理の徹底を求めるとともに,省エネルギー管理上重要な指標について数値が設定されている.熱処理炉などに対しては,エネルギー効率の高い設備等の導入の検討を求めることも法的に要請されている.また,設備稼働に伴う過剰エネルギー,不要エネルギー,エネルギー効率などについて,管理,計測・記録義務および設備保守・点検義務が規定されている.工場内において,エネルギー消費原単位,設備別のエネルギー消費量とエネルギー効率を把握し,省エネルギー目標を設定してエネルギー管理を実施するためには,生産システムに統合化されたエネルギー管理システムを構築しなければならない.エネルギー管理士の配置も要請されている.なお,直接的な省エネルギーに加えて,企業の植林活動も温暖化防止活動の一環として評価される.
