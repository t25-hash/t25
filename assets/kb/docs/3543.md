# 7・2・2 省エネルギーシステムへの取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・2 生産システムの省エネルギー

生産技術は、高精度化や高能率化を追求することによって進歩してきた。しかし一方では、先端技術の目標値が高くなるほどに環境負荷も大きくなった例が少なくない。生産系の省エネルギー効果等は生産高(売上高)原単位で表示されることが多いため、高付加価値製品の生産に移行した場合はエネルギー消費原単位が増大し、また生産量が減ると逆に原単位は増大する。しかし、地球温暖化防止のために総消費エネルギーの削減が今後の課題となっているため、従来の技術体系による高精度化、高能率化、自動化は環境負荷軽減の視点で根本的に見直す必要があるだろう。例えば、大幅な工程短縮やエネルギー効率向上が可能な省エネルギー型生産技術の開発が目標となる。高精度加工や微細加工に不可欠な大規模空調設備および高レベルクリーンルームは、大電力消費設備でもある。空調設備の電力消費が全工場電力消費の50~60%に及ぶこともある。実加工の有無にかかわらず、連続運転が必要な生産設備や長時間のアイドリング運転が必要な生産設備も高環境負荷を伴っている。工作機械も全電動化により総消費電力は増大しており、高効率化と摩擦低下などが課題である。OA 化が事務所の電源容量を増加させる例が多く見られるが、工場における生産自動化(FA 化)も消費電力増大を招かないようにシステム設計段階でアセスメントすべきである。この場合、直接的に生産プロセスに関係したエネルギー消費だけに注目するのではなく、工場建屋全体のエネルギー管理を統合化すること、エネルギー使用の平準化によってエネルギーの余剰や不足が発生しないように総エネルギー消費最小化を目指すことが重要である。例えば、工場受電容量を抑制して需要平準化を実現するために、エネルギー需要が低下する夜間の自動化運転などが効果的である。省エネルギー技術の研究では、高精度加工環境を維持するため費やされている膨大な空調エネルギーを削減するために、熱不感高精度工作機械の開発、また、工作機械熱変形を知能化機能で補正する仮想空調マシンシステム技術の開発が有効である(図7・4参照)。アイドル運転の短縮と省エネルギー化、低摩擦機構の開発による熱発生抑制、高効率駆動モータの開発、待機電力削減、廃熱エネルギー回収技術の併用なども有効である。工場建築構造の断熱性改善、建屋屋上の緑化、蓄熱装置やコージェネレーション設備の導入、照明の工夫、冷暖房温度設定などによる省エネルギー効果の増大にも留意する必要がある。
