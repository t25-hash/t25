# 7・3・1 資源循環型システム構築への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・3 生産システムと資源リサイクル

資源の有効利用による地球環境問題解決への取組みに向けて,政治,経済,社会,産業などのあらゆる領域において資源循環型システムの構築を目指している.例えば,日本の法制度は,「環境基本法」のもとで「循環型社会形成推進基本法(基本的枠組法)「廃棄物の処理及び清掃に関する法律(廃棄物処理法)」「資源の有効な利用の促進に関する法律(資源有効利用促進法)」「(個別製品)リサイクル法」「国等による環境物品等の調達の推進等に関する法律(グリーン購入法)」などによって,生産-消費-(廃製品化)-再生の各段階で資源循環型システムの構築を促進させる体系となっている.資源循環型の具体的な意味は,資源消費の減量化,廃製品資源のリユースとリサイクル(recycle)にある.生産システムにおいても,生産対象である製品だけでなく,生産設備,副産物などのすべてに資源循環を促進する配慮が求められる.生産工場への資源のインプットとアウトプットを詳細かつ厳密に管理するシステムを構築することによって,減量化,リユース,リサイクルを促進させるとともに,その内容の情報開示も要請される.製品資源の場合には,廃製品化段階でのテークバック義務を含めて,ライフサイクル管理が要請される.なお,有害危険物,とりわけ法的指定のある化学物質については,より厳密な管理義務を負うことになっている.およそ435種の化学物質は2001年施行の「特定化学物質の環境への排出量の把握及び管理の改善の促進に関する法律(環境汚染物質・排出移動登録,PRTR : pollutant release and transfer register)」により厳しい規制対象となっている.物質安全データシート(MSDS :material safety data sheet)の記録が要請され,大気圏,水圏,地圏(土壌)のいずれへの排出,漏出も規制される.揮発性有機化合物(VOA : volatile organic compound)は塗装工程などで多用される規制化合物の代表例である.加工組立製品に注目した場合の資源ライフサイクルの技術体系は図7・9のように表現できる.重要なことは,生産系-市場-還元系の流れを構成する学際的,業際的な要素技術分野をシステマティックに連携させるライフサイクル視点によって,資源循環技術体系を内在化して認識することである.究極の目標は,工場の廃棄物ゼロを実現するゼロエミッションである.従来までの技術体系は,製品ライフサイクルの個々のステージが独立しており,上流(設計・生産)から下流(廃棄処理)へ向けての一方向性概念で構成されてきた.しかし工業製品ライフサイクルを終了させるための廃棄処理過程を資源再生過程に置換させ,さらにはこの再生過程のアウトプットを同一の工業製品を生産する過程へのインプットとするためには(図7・2参照),トレードオフ要素図7・10 マテリアルライフサイクルオプション7―157を含むライフサイクル視点での問題認識と学際的技術による協調が不可欠である.換言すれば,従来の技術が生産過程すなわち図7・9の4領域のうち設計-加工・組立のみでの付加価値を重視してきたのに対して,廃製品化-資源再生の還元・再生過程における高付加価値化をも徹底追求することになる.最終的には,生産付加価値と還元付加価値を合算した場合の大きさが問われることになる.地球上の資源が素材生産プロセスに投入された後にたどるライフサイクルパターンは一つではない.とりわけライフサイクルを構成する最終ステージは複数のオプションとなる.資源の特性と利用形態およびユーザの意思決定とによっていくつかのオプションから選択される.図7・10は,こうした最終段階でのマテリアル(材料資源)ライフサイクルオプションを加工組立製品について示したものである.図7・10のオプション1の埋立てを選択することは次第に許されない状況になりつつある.オプション2のエネルギー(サーマル)図7・9資源循環のライフサイクルエンジニアリング(life cycle engineering)7―158生産システム工学リサイクルは,限定された可燃材料にのみ選択される.オプション3のマテリアルリサイクルは,オプション4のカスケードリサイクルと,オプション5のホリゾンタルリサイクルに分けられる.新資源の投入量を減らして再生材利用率を高めていくためには,資源再生の目標としてオプション5,すなわち質の低下を伴わないホリゾンタルリサイクル(水平循環)を選択する必要がある.多様な資源を加工組立した製品を想定対象とする場合であっても,エコファクトリー技術体系の基本条件は,製品ライフサイクルにかかわる資源循環の水平化を実現することであり,図7・10のオプション5による高品位マテリアルリサイクルである.ただし,オプション5の経路は,最初の出発地であった生産工場に単純回帰するものだけに限定されるということではなく,産業生態系において連鎖的に循環する経路として構成される場合もありうる.いずれにしろ最下流段階においてオプション5を実現可能とするためには,ライフサイクルに包含される数多くのトレードオフ要素を解決することが前提となる.すなわちライフサイクル視点による上流側要素技術を含めたシステム化された技術連携を実現するグローバルコンカレントエンジニアリング概念である.上流側のライフサイクル的配慮が大であるほど,下流側の分担は簡易になることは当然である.現在はオプション4のカスケードリサイクルを含めた全リサイクル率の向上が課題目標となっている.しかし,新資源の投入量を減少させるためには,新製品における再生材料の使用率向上が要請されていることから明らかなように,オプション5のホリゾンタルリサイクル(水平循環)を前提とする高品位資源循環技術が不可欠である.
