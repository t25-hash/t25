# 7・3・2 資源有効利用への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・3 生産システムと資源リサイクル

地球環境に負荷となる物質は,オゾン層破壊物質,温暖化ガス,環境汚染物質(大気汚染,水汚染,土壌汚染),有害危険物質,希少資源枯渇,廃棄物(重さ,かさ)などに分類できる.生産性や経済性を重視した生産技術から環境調和型生産技術へ転換することにより,これらの環境負荷物質の消費減量化,再生可能化(リサイクル,リユース),環境閉鎖系での使用などを実現させなければならない.例えば,オゾン層を破壊するフロンの使用禁止が契機となって,生産工程内で使用される溶剤や洗浄剤等を生態系にやさしい物質に転換するための技術革新が実現した.単に生態系にやさしい溶剤や洗浄剤を開発するだけでなく,溶剤や洗浄剤を使用しない加工処理法の開発も進展した.後工程での脱脂処理の省略/軽減化が実現できる加工技術の開発などである.冷凍機械や空調機械の冷媒からフロンを追放するために,圧縮機の潤滑系が新しく設計・開発された.無鉛はんだの採用は電子基板の有害物削減に寄与できるが,接合の信頼性確保も要請される.生分解性プラスチックの開発は化石資源消費および廃棄物の減量化に貢献し,形状記憶プラスチックの開発は分解性に優れた締結機構を実現可能にする.資源の再生循環を促進するためには,材料の標準化と認識性向上が有効である.例えばプラスチックや合金の多品種化は,廃製品化段階での経済的で高品位なリサイクルの妨げとなる.その解決には,含有成分を変化させずに材料機能を選択的に高度化できる表面改質技術が有効となる.また,使用している材種を製品ICタグに記入したり,文字,記号,色などで部品に表記することで廃製品解体後の材種分別が容易となり,高品位リサイクルが実現できる.生産段階で加工部品の材種や構造の外部認識性を高めるための付加加工を行えばよい.廃棄された加工組立製品の高品位リサイクルを実現するためには,生産段階であらかじめ解体性に優れた構造設計を採用しておくべきである.これによって例えば,摩擦締結機構,弾性締結機構,自動分離性の高いボルト締結機構などの採用例が増加することになるだろう.こうした締結機構の自動加工・組立技術,および締結部の安全性保証技術などを確立する必要もある.リユース部品を再加工処理するニーズの拡大も考えられる.従来の生産工場は,リサイクルやリユースを分担する再生還元工場との連携を強めて,やがては一体化,系列化していくことになるだろう.これによって,廃製品処理技術の生産性,経済性,品位の向上が容易となる.
