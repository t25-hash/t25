# 9・6・4 工作機械の適応制御

> **章**: 第9章 加工機械
> **節**: 9・6 工作機械の数値制御および適応制御

機械加工の目的は，十分な信頼性のもとに工作物を能率よく高精
度に仕上げることである．そのためには，高い剛性と精度をもつ工
作機械を使い，工作物に適した工具を選定し，ふさわしい切削条件
で加工をすればよい．しかし，加工形状とともに切削条件は変動す
るうえに，工具や被削材，工作機械ごとに特性も変化するので，稼
働中に適切な切削条件を維持することは難しい．
工作物の形状変化，さらには予測できない変動分に対しても効率
のよい加工ができるようにする手法が適応制御（adaptive con-
trol ）である．適応制御では，変動する作業条件に運転中の工作機
械の状況を適合させるように，切削条件や機械に関する設定データ
やパラメータを変化させながら適切化を図る．適切化の項目には，
機械負荷を最大に発揮しながら加工時間を最短化したり，加工コス
トの最小化，仕上げや寸法精度を考慮した加工などがある．
適応加工システムは，図9･51に示されるような基本構成をとり，
次のような種類がある ．
(１)ACC（adaptive control constraint ，拘束適応制御）
(２)ACO（adaptive control optimization，最適化適応制御）
(３)ACG（adaptive control geometry，形状適応制御）
ACCは，切削プロセスがある拘束条件内または限界内で行われ
るように切削パラメータを調整する．NCデータを用意する際に，
適切な設定値の選定が困難なときや，過負荷によって生じる工作機
械・工具・工作物の損傷を防ぐのに利点がある．多くのシステムで
は，切削抵抗やトルクに着目して機械と工具の両方に過負荷がかか
らない範囲で，可能な限り能率のよい加工を行うように送り速度を
変えることが多い．実用化されているものの大半はACCである．
ACOは，加工時間や加工コストを最適化するもので，おもに工
具の摩耗を最小にするように切削速度や送り速度のパラメータが調
整される．工具摩耗のオンライン計測は一般には難しく，それに基
づいて実時間で最適化を図ることは理論上可能でも，実用化までに
は至っていない．仕上状態や寸法許容値の要求から，ACCと組合
わせてびびりを最小にすることも行われている．
ACGは，要求された寸法や形状に工作物を仕上げることを目的
とした適切化システムである．ACCやACOは粗加工，中仕上加
工までに限られるが，ACGは仕上加工や精密加工に用いられる．
例えば，旋削加工で工具半径方向の変位を計測し，変位が最小設定
単位を超えて増減するたびにNC装置から送り機構に後退，ある
いは前進させて，リアルタイムに補正をすることが挙げられる．
このように，適応制御は加工制御技術として興味深いが，実用化
にはオンラインでプロセスを計測評価するための高信頼性で安価な
センサの開発が不可欠である．
