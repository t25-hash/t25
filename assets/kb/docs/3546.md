# 7・3・3 副産物資源循環型生産工程への取組み

> **章**: 第7章 環境と生産システム
> **節**: 7・3 生産システムと資源リサイクル

生産プロセスでは、切りくずや鋳物砂のような副産物が発生する。油、水、溶剤などの液体も大量に使用されるが、その使用過程で不純物が混入したり、変質する。これら生産工程での副産物は、その発生量を減らすとともに、発生した場合には回収、再使用することで環境負荷を軽減しなければならない。揮発性有機化合物VOCなどのように漏出防止に細心の注意を要する物質もある。PRTR 制度の規制を受ける物質の管理は入出量管理も厳密となる。精密機械産業にとって大量の洗浄水、冷却水の使用が不可欠である。地球は水の惑星などと表現され、水は無尽蔵に存在すると考えられているが、人類が使用できる淡水は地球上の水のわずか3%程度であり、しかも生態系を循環する淡水は0.8%でしかない。加えて生態系の破壊や汚染により、水の循環機能は質、量ともに低下しつつある。このため、水汚染への規制強化、水消費量の抑制、汚染水処理費用の上昇などが顕在化し、生産プロセスにおける水のリサイクルや水寿命の延長などへの要請が強まっている。ヨーロッパでは多数の国が同一河川の上流から下流までに位置しているため、水資源の汚染防止と消費削減に対しては特に敏感であり、乾式生産プロセスの優位性が高まっている。日本の場合も、人口当たりで見た水資源は決して豊富ではない。洗浄プロセスや空調設備などでは、循環型給水施設が導入されている。雨水の有効利用も考慮されるべきである。スラッジの少ない研削加工技術や、乾式高品位加工技術の開発も、生態系負荷軽減にとって重要である。加工に伴うスラッジ発生量の削減、加工油剤の長寿命化およびリサイクル、有害危険物質の使用禁止、超硬合金工具等のリサイクルなどは、加工技術分野での代表的課題例である。図7・11は、それ自身が工業製品でもある切削油剤(cuttingfluid)について、設計・製造(油剤メーカ)-消費(生産プロセスでの利用・管理)-再生・廃棄のライフサイクルでどのような環境配慮項目があるかを整理したものである。すなわち、油剤成分と要求性能、ライフサイクルモデル(設計、製造、使用、管理、変質、劣化、寿命)、管理、廃棄性、リサイクル性などが検討対象となる。これらの検討項目には、切削油剤の使用プロセスを構成する工作物、工作機械、工場、工具、加工方式、人間などの諸要素の要求条件および拘束条件の違いからトレードオフ的関係も発生する。企業の視点からは、コスト評価も加えた最適解の導出が要請される。図7・11 切削油剤管理のライフサイクル概念図7・12環境問題に配慮した切削油剤管理切削油剤関連の環境負荷の軽減化を実現するための取組みを図削工具寿命や加工品位への影響が実験的に調べられている。切削油剤が果たしてきた潤滑効果および冷却効果を乾式切削条件でどのように実現するかが主要研究課題である。切削油剤の長寿命化を達成するために、汚染、変質、腐敗の防止対策としてろ過設備や生物化学的処理装置の重要性を再認識すべきである。切れ味に優れる超砥粒砥石の採用は研削熱を低下させるだけでなく、研削比が高いために研削加工で問題となるスラッジ発生量減少にも効果的である。産業廃棄物を徹底的に削減するために、加工作業に伴う排出物すべての再生利用も目標となる。切りくず区分回収装置の付属、工程内廃棄物を削減する加工技術、またスローアウェイ超硬工具等のリユース/リサイクル技術などが今後ますます重要となる。切削工具の場合、工具材種、コーティング、工具形状などが多種多様であるが、切れ味、寿命、再研磨性、リサイクル性などから見たトレードオフ的関係の解消が難しさとなる。再生またはリユースした部品の機能、耐久性、信頼性を検査するための計測制御システム技術も重要である。製品は、その機能と価格が良いというだけではなく、環境への配慮が必要となっている。環境における保護の対象は、人間の体、生態系、そして資源である。人体や生態系に有害な物質を出さず、有限の資源を浪費しないようにする。言い換えれば、構成する材料・部品の調達においても環境負荷を削減し、生産工程のインプット(資源投入)と生産に伴うアウトプット(排出物)は把握されて、保護対象への影響の大きいものを減らす継続的改善がなされ、製造後の使用および廃棄段階の環境影響(environmental impact)にも対策を講じるなどである。生産システム設計は、環境対策を取り入れたアセスメント(assessment、事前評価)が望まれる。ここでは、考え方の基本となるライフサイクルアセスメント(life cycle assessment、製品の揺りかごから墓場までの環境負荷の定量評価)を始めに述べ、次に製造段階の環境負荷を扱い、最後にリサイクル設計に言及する。
