# 7・4・1 ライフサイクルアセスメント

> **章**: 第7章 環境と生産システム
> **節**: 7・4 環境に配慮した生産システム設計

ライフサイクルアセスメントはシステム分析の一つの手法で、ISO14040シリーズとして製品およびサービスに適用される。製品が環境や資源へ与える影響について、原材料の採掘から製品の廃棄段階における再利用までを含めたライフサイクル全体にわたり、定量評価を行う。これによって、製品であれば、生産、輸送、技術などの複雑な環境への影響を知り、環境負荷の低減を検討できる。唯一の方法というのではなく、提唱者ごとの特徴もあって、それらの7―159図7・13 製品ライフサイクルの一般的なプロセスバリエーションには触れずに規格の要点を記す。詳細は規格そのものを参照のこと。a. 目的と範囲の設定調査の用途、実施する理由、結果の伝達先、調査範囲を明確にする。b. システム境界図7・13はシステム全体を示すもので、システム境界とは、評価対象とするライフサイクルを考える範囲をいう。どこからどこまでを対象にしたか、例えば、資源採取を含めないと宣言すれば、必ずしも全体を含める必要はない。c. インベントリー(inventory)分析製品ライフサイクルの各段階における天然資源やエネルギーのインプットと、排出物のアウトプットを定量化するためのデータ収集や計算手順を含んでいる。実際にはデータ収集に制約があって、報告書の中に考慮した内容が記述される。例えば、収集データに地域的な相違がある、技術的に旧式な設備のデータである、精密なデータと文献値の利用などを考察しながら採否を決める。ISO14041:1998には、分析の構成要素、調査の目的および範囲、分析手順、分析の限界および調査報告がある。d. ライフサイクル影響評価インベントリー分析の結果を使って、潜在的な環境影響の重要性を評価する。特定の環境連鎖をモデル化し、その中の特定の環境影響を定量的に表す。ISO14042:2000には、インベントリーデータの分類化、特性化、重み付けの手順がある。e. ライフサイクル解釈設定された目的および調査範囲と整合性をもち、インベントリー分析や影響評価から得られた結果を統合するものである。ISO14043:2000には、得られた結果および推奨事項を解釈する要件とその一般的枠組を規定している。f. 調査報告意図した報告対象者に、公正、完全、正確に報告する。データ、方法、前提条件は明確にされていること、さらに第三者への報告が必要な場合は、必須の項目、考慮すべき項目がある。完全性とは、対象システムの中でライフサイクルのすべての段階が含まれ、それぞれのプロセスに対する環境影響も挙がっていることである。7―160生産システム工学環境負荷項目標準化係数()重み係数()(×) 事例電気冷蔵庫のシステムフローを図7・14に示す。鉄・銅・アルミニウム・プラスチック・ガラスなどの材料と、電力・燃料・水などの資源とを投入し、製品を生むとともに、二酸化炭素、二酸化硫黄、二酸化窒素や排水、廃棄物なども排出している。システムフローを明確にできると、インプット量とアウトプット量とがバランスする。通常の環境負荷項目は、地球温暖化、オゾン層破壊、酸性雨、湖沼の富栄養化、重金属、発がん性物質、スモッグ、農薬などである。これらをライフサイクルアセスメントで扱う際は、工程において使用・排出する量を標準化係数や重み係数を用いて統合化し、図る。環境負荷項目は図7・15に示したように多数あるが、電気冷蔵庫の評価では二酸化炭素、窒素酸化物、硫黄酸化物、スモッグの四つにして環境負荷項目を統合し、図7・16に素材・製造、輸送、使用、廃棄の四段階における環境負荷指数を示す。図7・16からは、使用段階の環境負荷が高く、地球温暖化と酸性雨の占める割合に注目すれば、使用時の消費電力を下げることが必要とわかる。素材・製造段階の環境負荷は、全体の1割に満たない。しかし、図7・16から図7・14電気冷蔵庫のシステムフロー図7・16 素材・製造、輸送、使用、廃棄段階の環境負荷指数製造段階に関して環境への配慮がいらないというのではなく、1割に満たない中にも工夫の余地がないか、また、製造段階の工夫が他の環境負荷を下げることにならないかと検討することがたいせつである。半導体を多く使用する家電製品では、使用時の環境負荷が半分程度のものもある。それは、素材・製造段階の環境負荷が大きいことを意味している。いくつかの業界でライフサイクルアセスメントの事例が発表されており、製品ごとの特徴を知ることもできるし、参考になる。材料や部品のインベントリーを使い、自社あるいは外注先の生産ラインの環境負荷データをもとにして、環境配慮された製品を訴求すること、また、そのような部品や製品を購入するグリーン購買(green purchasing)が普及しつつある。このような社会的要求から、ライフサイクルアセスメントは対外的報告に必須ともいえる。
