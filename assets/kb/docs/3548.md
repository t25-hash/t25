# 7・4・2 製造段階の環境負荷

> **章**: 第7章 環境と生産システム
> **節**: 7・15 のように単一の指標とする過程があるため,やや専門的とな

原材料の変更,工程の改善,製法の変更などによって,製造段階における排出物(副産物,廃棄物)を減らすこと,それら排出物を再利用できないか検討することが重要である.生産に必要な投入資源と排出物の物質収支を把握するのであるが,人体への安全性,生態系への影響の軽減,資源の保全性という視点が必要である.図灯油,LPG,都市ガスがある.また,排出物として,二酸化炭素,窒素酸化物,硫黄酸化物,生物化学的酸素要求量,化学的酸素要求量,浮遊物質量,塗料かす,廃油,廃液,汚泥,金属くず,部品くず,その他雑ごみなどが挙がっている.環境負荷を分析評価し改善するには,これらの項目ごとにデータを得られるように人員の配備,計測器の設置をする.従来のように,料金算出の根拠を得ようと工場全体値で把握しても,工程のどこが,どのように問題となっているかを解析できないことが多い.例えば,工程別に電力や水の使用量および1日の使用パターンを知れば,むだな投入電力や水を発見し,改善ができる.工場,工程の各所で電流,温度,湿度,水の流量,ガスの使用量など,計測対象ごとにセンシングし,パーソナルコンピュータへ信号を送り,1日のパターンを把握する装置が市販されている.これらの装置では,1週間,1箇月の変化,あるいは昨年との比較ができるものがある.a. 項目別の把握と改善排出する物質については,大気汚染項目別,水質汚濁項目別,廃棄物の種類別などにそれぞれ計測する.このように,工程の環境負荷を項目別に把握して改善する,あるいはもう少しまとめてエネルギー消費,固体廃棄物,水質への排出物質,土壌への排出物質という扱いで評価もできる.企業で盛んになった環境報告書では,主要項目別の全社的なデータが記載されることが多い.さらに,わが国で認証登録数の伸びが著しいISO14001:2004「環境管理システム(environmentalmanagement system)」では,著しい環境側面として抽出した項目ごとに,また,定めた期間ごとに増減を把握し,改善を行っている.この方法の利点は,着手しやすい,問題点が容易に検出できる,外部にわかりやすいことであろう.b. 統合化と改善環境負荷物質を項目別に分けて標準化係数と重み係数を掛け,環境負荷指数を算出する.地球温暖化,オゾン層破壊,酸性雨,湖沼の富栄養化,スモッグ,重金属排出,発がん性物質,農薬という分類であるが,一つの物質が一つの項目に分類されるわけではない.例えば,特定フロン(five types of CFCs,CFC: chlorofluorocarbon)はオゾン層(ozone layer)破壊と地球温暖化に,硫黄酸化物は酸性雨とスモッグにおいても計算される.環境負荷指数の大きさと構成項目の大小関係に注目し,この状態を維持するか,環境負荷指数を下げるかを決める.後者の場合は,環境負荷指数をどのくらい下げるか,どの環境負荷項目を対象にするかといった方向付けをする.市販のソフトがあるので,ねらいとする環境負荷項目に仮定をおき,試算で実行の容易性,効果の目安を得ることができる.例えば,消費電力量を5%削減,あるいは金属系の廃棄物を50%リサイクルする,という仮定で試算すると,環境負荷指数の変化を事前評価できる.消費電力量を5%削減する際に,設備ごとの消費電力量および稼働状態を把握できていれば,対象設備を絞り込みやすい.金属系の廃棄物を50%リサイクルするには,どこの設備から材質別にどのような金属廃棄物がいくら出るかを把握できていれば,分別,加工方法,材質の統合や変更,対象部署などを絞り込むことができる.環境負荷指数を下げる対象項目と目安が得られたら,実施方法を具体化して実行の経営判断をする.実行に移したら,適時に運用データを入手し,効果を判断できるシステムとする.
