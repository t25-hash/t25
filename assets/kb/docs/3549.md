# 7・4・3 リサイクル設計(design for recycling)

> **章**: 第7章 環境と生産システム
> **節**: 7・14 に示した生産プロセスでは、投入資源として電力、水、重油、

科学技術によって大量生産を行い、それによって多くの資源を消費した前世紀には、廃棄物も多量に排出した。廃棄物は焼却、埋立てを主とし、資源回収は一部であった。その反省から、現在では、循環型社会の形成と推進が世界的な環境課題と認識されている。使用後の廃棄・再資源化は、ライフサイクルアセスメント上では使用段階、原料・製造段階の負荷に比較すれば、大きな数値にならない場合が多い。しかし、資源循環型製品を世に送り出すガイドを7―161もつ、あるいはリサイクル実務を義務化された製造事業者も徐々に増えている。環境負荷指数の小さいことを理由に取組みを避けることはできない。むしろ、リサイクル設計に積極的な取組みをしなければ、購入者の理解は得られない。ここでは、わが国の先行となった大型の家電品を事例に、リサイクル設計とリサイクル技術を述べ、他の製品の参考としたい。a. 背景消費後の大型家電品の廃棄に際し、一般廃棄物の処理責任のある自治体では、大きい、重たい、複雑な構造と成分を有し処分しにくいとの声を1970年代から上げていた。1998年に公布された特定家庭用機器再商品化法(通称:家電リサイクル法)は、製造業者等の競争原理のもとで再資源化の実務を行うようにしたものである。政令指定の使用済み家電品から部品および材料を分離し、製品の原材料または部品として利用(有償または無償で売却)することを製造業者等に、使用済みの家電品を引取る義務を小売業者に、適正な引渡しと費用の支払いを排出者に、それぞれ義務付けた。直接的には販売済みの大型家電品の資源リサイクル、間接的にはこれからのリサイクル設計を推進する意図があり、2001年4月から本格施行となった。廃棄物削減と再資源化、排出者の費用負担による排出抑制という原則は、早晩、わが国の多くの量産品には共通するようになると考えたい。また、ヨーロッパでもリサイクル設計と環境影響物質の忌避への要請は強く、環境配慮は生産システム設計の重要な項目となる。b. リサイクル設計生産を行う前に、この製品が生産・流通・使用・廃棄・再資源化の各段階における安全や資源・環境への影響を調査、予測、評価し、必要に応じて製品設計や生産方法の変更を行って影響の軽減を図るという、製品アセスメントが一般化してきた。設計の意味するところは広く、予備設計、基本設計、詳細設計という区分、あるいは構造設計、機能設計という区分もある。これらの中に、共通してリサイクル設計を取込む。生産段階は7・化段階に注目したい。流通段階は、包装の簡易化、再資源化しやすい包装材料の選定、包装の再資源化や処分に関する表示が挙げられる。さらに細かくいえば、再生資源としての利用の容易性、材質別の分離の容易性、材質の表示、再生材料の利用度、収集・運搬の容易性、焼却や埋立ての環境保全性、包装資材の削減などについて評価する。量産試作品ができ上がってからの包装設計は、適切なものとならない。製品設計時に、包装までも併せて検討することが双方の改善につながり、寸法・質量とも過剰とならない。使用段階は、消費電力や待機電力を減らす省エネルギー、消耗部材や流し捨てる水を減らす省資源、修理しやすい構造、長期使用につながる部品の共通化、適切な使い方を勧める取扱説明書や本体表示などの評価を行う。環境負荷が特に大きいのは使用中であり、消費電力や待機電力を減らす設計がたいせつである。製品を省エネルギーに設計し、さらに使用者が適切な使い方をするよう、取扱説明書で訴える。省資源についても、使い方によるところが大きい。修理については、ブロック化した部品の一つが故障し、ブロック全体を廃棄してしまうのは資源のむだとなる。故障部品の取出し・交換のしやすさだけでなく、標準化による補修部品の共通化と供給、確かな修理技術と即応体制など、本体だけに限らない課題がある。廃棄・再資源化段階では、使用者には適正排出(不法投棄しない)を取扱説明書で訴え、本体そのものに関して設計者は次のような項目を考慮しておかなければならない。. 分離・分解の容易化高価な部品、破砕できない剛体、リサイクルの後工程に支障のある部品や環境影響物質などの特定の部品は、手解体で取出しやすいか。部品点数は減らしたか。異種材料の接合はないか。ねじの数、寸法、種類は多くないか。. 材質別の選別の容易化手解体する際に、プラスチック部品に材質表示があり、見やすいか。破砕した際に、材質別に分か7―162生産システム工学れやすい構造にしてあるか。材質の統一をしたか。. 再利用(中古)部品、再生材料としての利用性中古の部品や材料として利用できるか、あるいはそのようなものを使って製造できるか。. 破砕・選別の容易化せん断破砕あるいは衝撃破砕するとき、設備に損傷や障害を与えないか。所定の寸法に破砕されて、選別工程で品位良く選別されるか。材質の種類を減らしたか。. 焼却・埋立ての安全性ダストや再資源化できない混合物は、有害な物質を含んでいないか。焼却や埋立てをして、有害な物質を作らないか。. 収集・運搬の容易化使用済みになってからリサイクル工程に搬入するまでに、運びやすくなっているか。大きい機器には取っ手、車輪などがあるか。荷重は偏っていないか。内部から油やガスが漏出しないか。輸送の衝撃で爆発や爆縮はしないか。. 体積の減容化、質量の減量化資源の節約となる小型・軽量化は可能か。c. リサイクル技術使用済み機器から資源回収する現状技術を知ることは、適切なリサイクル設計に役立つ。自動車、大型家電品などは、元の質量に対する回収資源の率あるいは売却できた資源の率といった基準がある。基準の上乗せ見直し、使用済みの他の機器へのリサイクル基準の設定は今後に予想される。5年、10年後に適用されると仮定すれば、現在の設計・生産品が優れたリサイクル性をもっていなければならないということである。大きい対象機器から重たい金属のみを回収するだけであれば、大きな破砕機と磁力選別すればよい。ところが、多くの種類の金属やプラスチック、潤滑油、冷媒ガス、剛体、ガラス、磁性材料、爆発性の粉末やガス、有害な水銀や鉛などを含んだ材料や構造の複雑な機器が多い。例えば、家電品は構成材料がかなり複雑で、かつ小さい部品の組合せである。粒度にかまわず磁力選別してしまえば、異なる材質の破砕片が多く、質の良い資源として回収はできない。混在物の少ない資源を回収し、高く売却する(収入)、埋立てや焼却するダストを減量(支出)することで、リサイクルコストの改善につなげる。. 処理基準家電品の場合は、再商品化という言葉には次の定義が使われる。1機械器具が廃棄物となったものから部品および材料を分離し、自らこれを製品の部品または原材料として利用する行為、2機械器具が廃棄物となったものから部品および材料を分離し、これを製品の部品または原材料として利用する者に有償または無償で譲渡しうる状態にする行為である。製品の部品または原材料として分離して、これを燃料として利用することは、当初は再商品化に含まない。また、製造業者等は引取った機器について、少なくとも以下の基準以上の再商品化等を実施することとなっている。エアコン:60%、テレビ:55%冷蔵庫、冷凍庫:50%、洗濯機:50%したがって、プラントはこの基準をクリアできる能力を有するもの、製品は現在のプロセスでこの基準を満たす素材構成にしなければならない。他の機器にこのような法的基準はないが、回収率を課す仕組みがあるものとした取組みが望まれる。さらに、冷蔵庫、冷凍庫とエアコンに含まれる冷媒用フロン・代替フロンを回収して再利用または破壊すること、また、テレビの大型プリント基板については、取外し、金属類を再資源化することを義務付けている。これらの回収は、破砕前の手分解段階となるだろう。.
