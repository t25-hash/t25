# 7・5・1 事例1:環境にやさしい工作機械(省エネルギー工作

> **章**: 第7章 環境と生産システム
> **節**: 7・5 環境にやさしい生産システム(事例:省エネル

機械)a. 概要近年、オゾン層の破壊、地球温暖化などから、地球規模での環境保護・省エネルギーが叫ばれている。一方、生産現場では、ユーザの短納期化・コスト低減の強い要求から、高速で加工能率優先の生産が求められている。加工能率を高めるためには、電動機、アクチュエータの大馬力化が必要である。また、発生する熱を除去するために、高出力の冷却装置が必要となる。さらに、切りくず処理には大量のクーラントが用いられ、エネルギーの消費が高い。このような状況で、生産現場には、省エネルギー(energysaving)と生産性(productivity)アップを両立した機能をもち、環境影響も考慮に入れた、環境にやさしい省エネルギー工作機械が求められている。省エネルギー工作機械は、三つのコンセプトからなる。(1) 省エネルギーで稼働できる。(2) 廃棄性が良くなる。(3) 排出物の量と環境影響を抑制できる。省エネルギーで稼働するためには、電力消費の減少、重力の活用や油剤供給を不要にすることが必要である。廃棄性を良くするためには、切りくず(chip)の圧縮処理、油水分離、水・切りくずの分離、塩素の排除が必要である。また、排出物の量と環境影響を抑制するためには、不良品、電磁波、廃液、振動・騒音への対策が必要である。省エネルギー工作機械の例を図7・18に示す。図7・18は省エネルギー型立型マシニングセンタ(machining center)で、潤滑油の無補給化と油圧ポンプの廃止を可能としている。7―163b. 省エネルギー工作機械の要素省エネルギー工作機械に使われているおもな要素を以下に示す。. NCチャックNC旋盤における工作物の把持には、油圧チャックが一般的に用いられているが、チャックシリンダの高速回転での高発熱による加工精度への影響や、油圧油の早期劣化を招きやすいなどの問題がある。また、油圧ポンプの駆動に常時エネルギーを消費する。NCチャック(図7・19)は、チャックの爪の開閉動作(工作物把持機能)を、NC制御による主軸モータ駆動とスクロール機構により実現している。また、チャッキング時のチャック(ボデー)の回り止め機構や、モータの伝達経路の入切に用いるクラッチ機構には空圧源を使い、いっさい油圧を使用していない。その結果、油圧ユニットを廃止し、省エネルギー、騒音抑制効果が実現されている。また、サーボ制御による把握力、爪位置、および爪速度を任意に設定できるため、高品位な精度対応が可能となっている。油圧チャックは、電源が切れると、油圧の圧力低下で工作物の把持が困難である。NCチャックは、工作物を把持した後、内蔵されたクラッチ機構を作動させて機械的にクランプを保持する。したがって、クランプ完了後のエネルギーの連続的供給は不要である。また、電源が切れても工作物が外れることはないため、安全性が高い。. 潤滑油補給機能付潤滑機構潤滑油の無補給を目的として、案内面およびボールねじの潤滑に、潤滑油補給機能付潤滑方式ガイドやボールねじがある(図7・20)。これは、潤滑油を蓄えた図7・18 立型マシニングセンタ[日立精機(株)]図7・19 NC チャック(a) 補給機能付潤滑方式ガイド(b) 補給機能付潤滑方式ボールねじ図7・20 潤滑油補給機能付潤滑機構7―164生産システム工学タンクが、スライダが動くとガイド面に潤滑油を供給するものである。このため、潤滑ポンプが不要となり、省電力・省資源を実現できる。従来の転がり案内面およびボールねじのオイル潤滑方式では、廃油処理が問題となっていた。また、グリース潤滑では、グリースの微少流失、乾燥、熱や異物による酸化、劣化が問題となっていた。潤滑油補給機能付潤滑機構では、自己潤滑性を有し、しゅう動部および周辺に潤滑を供給できる含油樹脂製のシールを使用し、グリース封入に用いることにより、長期間のメンテナンスフリー、使用環境を油で汚さない、ランニングコストの削減などが実現している。. 油圧ユニットレス自動工具交換装置自動工具交換装置(ATC)は、マシニングセンタの主軸工具を自動で交換する装置である。主軸に挿入された工具を取り外すためには大きな力が必要であり、一般的には油圧を利用している。油圧ユニットレス自動工具交換装置は、特殊カム機構に流体伝達装置を内蔵したATC装置を、一つの電動モータで駆動することにより目的を達成している。油圧ユニットレス自動工具交換装置の基本構成を図7・21(a)に示す。ATCの動作全体を司るカムユニット、交換動作時に工具を把持するツインアーム、主軸に工具を固定する主軸内工具クランプ装置、同期連動機構[図7・21(b)]から構成される。次の順序で、カムユニットがATCの一連の動作を制御する。1カムユニット起動、2同期連動機構経由で主軸内工具クランプ装置にカム運動を伝え、工具をアンクランプ、3ツインアーム工具交換動作、4同期連動機構経由で主軸内工具クランプ装置にカム運動を伝え、工具をクランプ、5完了。同期連動機構は、伝達手段に流体(油)を使用することで、カムユニットと工具クランプ装置の配置に自由度をもたせることが可能となっている。カムの運動を流体に変換するため、カムでピストンを動作させ、ここで発生した圧油をホースにて工具クランプ装置アクチュエータに伝達する(流体密閉回路)。これにより、マシニングセンタに必要不可欠であった、油圧装置が不要となる。この結果、油圧装置を使う場合に比べ、20~30%の省電力となっている。
