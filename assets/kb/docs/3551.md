# 9・7・6 回転加工機

> **章**: 第9章 加工機械
> **節**: 9・7 塑性加工機械

棒材，板材，管材などの素材または工具を回転させて塑性加工を
行う機械を総称して回転加工機（rotary forming machine）と呼
び，その代表的なものを以下に挙げる．
a.転造機（form rolling machine) 転造機は，棒材または管
材の外周に素材と平行な軸を有する一対の工具を押し付けて，所定
の形状に回転成形する機械であり，一般に円形工具と平行工具の両

方式がある．
.ねじ転造盤（thread rolling machine) ねじ転造盤に
は2個の平ダイスによる平ダイス転造盤，2個または3個の丸ダイ
スによる丸ダイス転造盤，セグメントダイスと丸ダイスを用いるプ
ラネタリ転造盤がある．平ダイス転造盤はM24以下，プラネタリ
転造盤はM12以下の小径ねじの大量生産に適しており，一方，2
個の丸ダイスによる丸ダイス転造盤は広範囲のねじ寸法と材質に適
用できるので，精密ねじの多種生産に適している．このほかに，旋
盤に転造ヘッド（転造アタッチメント）を取付けてねじ転造を行う
こともある．
.歯車転造盤（gear rolling machine) 歯車の創成転造
法としてはラックダイス方式（Roto- Flo法)，ローラダイス方式，
内歯ダイス方式（WPM法）などがあり，成形転造法としては
Grob法がある．ラックダイス方式では比較的小径，小モジュール
のインボリュートスプライン軸，インボリュートセレーション軸，
インボリュートはすば歯車などが冷間転造されている．ローラダイ
ス方式ではインボリュート平歯車，はすば歯車の熱間転造が行わ
れ，またシェービングに代わる歯車の冷間仕上転造も行われてい
る．WPM法では内歯をもつ一対のセグメントダイスが平行を保っ
たまま同期して円運動することによって，インボリュートスプライ
ン軸などの冷間転造が行われている．Grob法では，同期して遊星
回転する一対または二対の総形ローラによって歯形を成形転造し，
スプライン軸の冷間転造法として長い実用実績があるが，薄肉のタ
イミングベルトプーリなどの加工にも利用されている．
.クロスローリング機（cross rolling machine) クロス
ローリング機は，対向する一対のくさび状の突起を有する平ダイス
またはロールによって段付き軸状部品を熱間転造する加工機である
が，小物精密部品の冷間クロスローリングも行われている．また，
スルーフィード転造を行うクロスヘリカルローリング機では大形ね
じ，フィンチューブ，球，ころ状部品などの連続加工が可能であ
る．
.回転鍛造機（rotary forging machine) 回転鍛造機に
は上型揺動回転方式と下型回転駆動方式がある．二重偏心スリーブ
で上型に種々の運動軌跡をさせる場合には，上型揺動回転方式が必
須であるが，連続した据込みを行う場合には，下型回転駆動方式の
ほうが装置の駆動系が比較的簡単になる．
b.スピニング機（spinning machine) スピニング機は，板
材，管材，鍛造材，鋳造材などの素材をマンドレル（成形型）に取
付けて回転させ，1個または複数個のローラを押し付けてマンドレ
ルと同じ形状に成形する加工機であり，薄肉製品用の横形加工機の
場合は旋盤と類似の主軸台，心押し台，ロール台および各種補助工
具などで構成される．しごきスピニングを除けば多サイクルパスの
成形となり，以前はテンプレートを利用した電気-油圧サーボなら
い制御でロールを油圧駆動していたが，現在ではNC制御が主流
となっている．厚板成形用の加工機では主軸台と心押し台の推力が
数百kNの高剛性で，板状素材から曲げ，絞り-しごきスピニング
を行って1チャックで内歯付きの部品などが加工されている．ま
た，2個のタレットヘッドに6個ずつのローラを取付けて複合加工
を行う加工機もあるが，これらの高剛性の加工機ではローラの軸方
向位置は固定で，数百kNの推力でクランプされた素材が，主軸
台ならびに心押し台とともに回転しながら軸方向に往復することに
よって成形するものが多い．1台のはん用加工機で，板状素材から
多サイクルパスの複合加工を1チャックで行うもの，複数の専用機
械をライン化してマルチステーションで複合加工を行うものなどさ
まざまである．
c .スエージング機（swaging machine) スエージング機は，
相対向する複数組のダイスに半径方向の往復運動を与えて，棒材や
管材を外周から圧縮して軸方向に絞り出す加工機であり，ダイスを
素材軸のまわりに回転する方式と，ダイスは固定で素材を回転する
方式のものがある．前者はロータリスエージング機（rotary swag-
ing machine）と呼ばれ，外径120mm以下の細物に適し，各種段
付き軸やテーパ部品の加工に用いられる．後者はダイスと中空軸固
定でアウタレースを回転させるもの，あるいは偏心軸によりダイス
を往復運動させるものなどがあり，大形製品に適している．
d.その他 リング状素材を圧延成形するリングローリング機
があり，ベアリングレースや車輪のタイヤ，そのほか異形断面のリ
ング状製品を熱間で高能率に加工できる．またディスクローリング
機ではサイドロール，ウェブロール，リムロールの位置制御によっ
て，種々の形状のディスクやディスクリングが加工できる．
