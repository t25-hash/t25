# 7・5・3 事例3:複写機作りにおける循環型生産システム

> **章**: 第7章 環境と生産システム
> **節**: 7・24 に新設ライン、新製品切換え等に伴う省エネルギー対策の基

a. 循環型生産システム(integrated recycling system)世紀の循環型社会を実現するために、複写機業界においても、より少ない資源とエネルギーの使用で、かつ、可能な限り廃棄を少なくする生産システム、すなわち循環型生産システムへの変換が強く求められている。循環型生産システムを構築していくうえで、次の四つの活動がポイントとなる。(1) 製品の回収顧客が使用した製品を回収する。(2) 部品リユース(parts reuse)回収した製品から、一連のリサイクル工程(洗浄/選別/修理/検査など)を経て、品質基準に適合した部品だけをリユースする。(3) 製品の品質保証リユース部品、新造部品のどちらを使用しても、全く同じ品質(外観/性能/信頼性/寿命)の製品として保証する。(4) 再資源化リユースできない部品を再資源化する。したがって、従来の生産システムにはなかった新しい技術や設計法が、循環型生産システムには必要となってくる。これらの全活動を図柄で表すと、閉じた輪が形成される。この閉じた輪をクローズドループシステムと称している(図7・26)。循環型生産システムを構築することにより、環境負荷の少ない部品リユースを第一優先として、資源の再活用を推進する。リユースできない部品は再資源化して、廃棄ゼロを達成する。このように、部品リユースと再資源化を両輪とする生産システムが循環型生産システムである。b. 循環型生産システムを支える技術・設計複写機は、数千点に及ぶ部品で構成されており、フレームやモータのように寿命の長い部品と消耗品カートリッジのように寿命の短い部品が混在している。通常、市場で平均3~5年間くらい使用された後に回収される。循環型生産システムでは、部品リユースから再資源化まで、ライフサイクル全体を視野に入れた技術・設計の取組みが必要である。循環型生産システムにおける技術・設計の概念を図7・27に示す。この中で、最も重要な技術は、部品寿命の判定技術とデータベー7―166生産システム工学図7・27 循環型生産システムにおける技術・設計図7・29部品リユース/再資源化設計指針ス化である。いままではワンライフもてばよく、それを保証するだけで終わっていたのを、どこまでもつか徹底的に追求していくのが部品寿命の判定技術である。カスタマエンジニアなどから送られてくる製品の品質情報をベースに、寿命を残している部品であるか、修理すればリユースが可能な部品であるかを見極める。また、製品寿命以上の期間を稼働した機械の分解調査を行って、各部品の摩耗状態や汚れ、劣化特性などを解析し、ワンモアライフ(one morelife)の使用が可能か否かを判定する。このように、個々の部品寿命がどのくらいあるかを解析することによってリユース不可原因を層別し、個々の部品に対するアプローチを選択することから循環型生産システムの技術・設計がスタートする。. リユーザブル技術(parts reuse technology)回収した製品を資源として、これらをリユース可能とする技術全般がリユーザブル技術である。リユーザブル技術には、次のようなものがある。(1) 洗浄技術機械フレームや板金部品などの汚れを取り去る(図7・28は、大物板金類用のブラスト洗浄装置である)。(2) 選別技術余寿命が十分にあり、ワンモアの使用が可能な部品とリユースできないスペックアウト部品をより分ける。(3) 修理技術操作パネル部や電源ユニットなどを修理し、ワンモアの使用を保証する。修理や選別などの技術は新造部品による製造には必要ないが、循環型生産システムでは必要不可欠な技術である。. 部品リユース設計(parts reuse design)選別や修理などのリユーザブル技術を不要とし、かつ、部品をリユース可能とするのが部品リユース設計である。リユース不可原因を層別すると、もともと寿命が短い、ダメージ・汚れや傷等による機能劣化、非共通という三つに大きく分類される。これらリユース不可原因に対図7・28 洗浄技術(ブラスト洗浄)図7・30 分離設計事例(キャスタ)し、部品の再使用を可能とする設計法を開発していくことが非常に重要である。部品リユース設計には、長寿命化設計(long-lifedesign)、分離設計(separable design)、強度設計、分解設計などがある(図7・29)。寿命が長いと何回も使えるので、長寿命化設計が部品リユース設計の基本となる。長寿命化設計は、リユースを前提にした設計仕様(動作条件や負荷条件など)を定めて、その仕様を満足するように部品を設計する。長寿命化が困難な部品は、摩耗部品や短寿命部品だけを簡単に交換できるように分離設計を行う(図7・30は、分離設計事例のキャスタである)。部品リユースが可能か否かは、ほとんど設計段階で決まってしまう。設計者がいかに知恵を出し、どうすればリユースが可能となるかを考えていかねばならない。. 再資源化設計リユースできない部品や寿命まで使い切ったリユース部品も貴重な資源である。これらの部品は、再資源化して材料に戻さなくてはならない。材料リサイクルとしての活用を考える前に、グレードを落とさない材料リユースとしての活用(外装カバーを外装カバーに再生するなど)を考えることも必要である。再資源化設計で取組むべきことは、(1) 解体・分別容易化解体しやすく、素材別に分別しやすくする。(2) 有害化学物質の削減環境に悪影響を及ぼす有害な化学物質(鉛、ハロゲン系難燃剤など)を使用しない。(3) 再資源の容易化再資源化に適した材料を使用する。などである。c. 循環型生産システムによる新しいものづくり複写機が市場から回収される時点では、後継機が生産・販売されており、世代を超えたリユース部品の活用が重要である。同一機種だけの部品リユースから、2世代、3世代めの機種へのリユース部品投入を実現してこそ、循環型生産システムのメリットが生じる。製品の企画段階において、後継機が市場や顧客要求でグレードアップしなければならないもの(性能や機能を向上させるユニット/部品)と、変更の必要がないものとに二分化して、製品の開発を進めることが特に重要である。すでに、3世代くらいの製品を共通とするプラットフォーム戦略により、部品リユースを前提とした製品企画(productplanning for parts reuse)が行われている。また、回収した製品から取り外し、リユーザブル技術によってリユース可能となった部品、あらかじめ部品リユース設計された部品を、異なるタイプの新製品に投入することが始まっている。そして、再資源化設計を行うことで、廃棄ゼロを達成できる。さらに、部品リユースや再資源化が重要だと認識されるにつれ共通化も進む。共通化が進めば、調達コストも下がる可能性がある。そして、長寿命設計や徹底的に寿命を管理することにより、製品の信頼性は向上する。分離設計や分解設計をすることで保全性が改善される。このように、循環型生産システムは、トータルコストダウンに貢献するのみならず、世代を超えた新しいもの作りを可能とし、大量消費・使い捨て社会を変革する。
