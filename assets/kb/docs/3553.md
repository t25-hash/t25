# 7・5・4 事例4:レンズ付フィルムの循環生産工場

> **章**: 第7章 環境と生産システム
> **節**: 7・24 に新設ライン,新製品切換え等に伴う省エネルギー対策の基

富士写真フイルムでは環境に配慮し,1992年より「写ルンです」のリサイクルシステムを開発,実用化してきたが,1998年にはこれまでの知見,技術を結集して世界初の循環生産システムを完成した.7・5・4項では,このシステムの考え方から実際の構成,必要技術,運用ポイントに至る全容を紹介する.a. 循環生産(inverse manufacturing)とはこれまでの大量生産,大量消費により,資源は枯渇の危機に直面している.一方で,消費後の廃棄によるごみ処理飽和問題が深刻化している.これらを改善し,地球環境負荷低減のために消費型経済システムから循環型経済システムへの転換が必要といわれている.資源を入手して製品を市場に出す現在の製造業は持続性が保てないという意味で不完全であり,むしろ積極的に使用後の製品を有効に利用し,再び新しい製品に再生する循環的な製品ライフサイクルを構築する必要がある.쓕写ルンです」循環生産工場は以下の3点をコンセプトとしてシステム開発を進めた.・쓕写ルンです」は「写ルンです」へクローズドループリサイクル・リサイクル適性のある製品設計重視・リサイクル自動化による品質保証と効率生産b. 쓕写ルンです」の循環쓕写ルンです」は使用後,いずれかの現像所に戻る商品であるため,このルートで使用済みのボデーを回収する.これを新品として再生する手段は「写ルンです」を機能ごとのユニットに分解して,再生時の環境負荷の少ない再使用(リユース)を中心に実施し,故障しているユニットは修理・検査のうえでリユースする.カバー部材は傷・汚れがあるため新しい「写ルンです」の材料として再生使用する(図7・31).c. 循環生産工程上記を実現するための工程および技術ポイ7―167図7・31 レンズ付フィルムの構成および再生分類図7・32 循環生産工程フローントは以下である.工程フローを図7・32に示す.. 仕分け他社品も含めて数十種類となるため,まず品種ごとに仕分ける.「写ルンです」は整列,識別,振分け,包材除去の作業を一つの工程として自動化している.寸法・形状の異なる製品の混在処理に工夫が必要である.. 分解自動分解装置によりカバー部材を外し,再生ユニット単位まで分解,集積する.分解しやすい製品設計と分解設備のコンカレント設計が重要である.「写ルンです」の設計では特に,部品間のフック(爪)留め構造が特徴である.分解装置はこれら10箇所程度あるフックを同時に外す工夫をしている.. クリーニング回収品ゆえに汚れが避けられないため,品質面から入念なクリーニングが必要である.レンズの汚れ,機構部のしゅう動悪化につながるじんあい等のクリーニングを自動で実施している.レンズのワイピング洗浄,樹脂の温水洗浄等,特に環境に配慮した方式を実用化している.. 検査回収品は履歴がつかめない.全数検査により新品同等の品質を保証するため,各再使用ユニットの機能,性能,外観,寸法を検査しており,自動化により一定条件で高精度の検査を可能としている.また,リユース品にはリユース回数のマーキングを実施しており,新たな回数の回収品を生産に使用する際には,そのつど品質評価を行うことで,リユース品の品質保証に活用している.. 故障品修理上記検査における検査結果を総合判定して,故障部位別に振り分けるシステムを構築し,故障品と判定されたものはそれぞれ自動または手作業で修理して再使用している.複合故障も含め,約20項目の修理を実施している.. 材料再生カバー部材は使用する樹脂をポリスチレンに統一し,分解後に樹脂再生工程を経て材料として再生使用している.徹底した洗浄,異物除去システムにより,写真材料としての品7―168生産システム工学質を確保し,再ペレット化せずに射出成形機に再投入できるペレタイズレスシステムを実用化している.ペレット化に比べ熱履歴を低減することで,品質劣化防止,環境負荷低減のメリットがある.. 組立工程組立工程ではリサイクル工程間等を効率良くつなぐ自動ラック,AGV(automated guided vehicle,無人搬送車)のマテハン(マテリアルハンドリング)によりリサイクル部品を供給し,新品部品との混合使用を行っている.組立,リサイクル設備は部品トレー,AGV 等のマテハン,情報ハンドリングシステム等の共通化を徹底しており,部品のスムーズな循環を可能としている.また,設備のベースマシン等の様式も標準化しており,生産品種,数量構成の変化に応じて工程間の設備転用改造すなわち設備のリユース循環も行っている.d. 製品設計ポイントクローズドループシステムを実現するためには企画・開発段階からライフサイクル全体をとらえた商品設計を行うことが最も重要となる.その内容は省資源,ユニット化,共通化,材料の統一,自動化設備適性の観点がある.循環生産を可能にしている自動化効率はこの設計段階で決定されてしまうといっても過言ではない.e. 結び쓕写ルンです」の循環生産はものの循環だけでなく,技術の循環という観点を大事にしている.商品設計から生産技術,リサイクル技術へ製品と技術が流れ,そして回収品からの情報も含め,日々,リサイクルから商品設計への技術のフィードバックが行われる.これらのリサイクルを継続させることで,高品質リサイクルが完成する.文献(1) Gradel,T.E.and Allenby,B.R.,Industrial
