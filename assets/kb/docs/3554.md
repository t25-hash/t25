# 9・7・9 特殊加工機

> **章**: 第9章 加工機械
> **節**: 9・7 塑性加工機械

a.高エネルギー速度加工機（high-energy- rate working
machine)
.放電成形機 大電流の放電によって発生する衝撃水圧を
利用する放電成形機では，電気回路の特性が重要であり，放電電極
とコンデンサ群とが低インピーダンスの導線で接続され，スイッチ
はエネルギー損失の少ないイグナイトロンが使われる．加工エネル
ギーは充電電圧によって制御され，水室と型との締付け力1MN，
最大容量150kJまでの機械が開発されている．
.電滋成形機 急激な磁界変化を生じるコイルと，これに
近接する導電体との間の衝撃的反発力を利用する電磁成形機では，
コイルの耐久力を増すため，コイルの補強を十分にし，冷却水を通
して過熱を防止する．また，磁束集中器により，同一のコイルで異
種寸法製品の加工ができるようにするとともに，コイルの負荷を低
減させる．電気回路は放電成形機の電極の部分にコイルが接続され
たもので，両者はほとんど同じである．
.爆発成形機 火薬の爆発による衝撃的流体圧力を利用す
る爆発成形は，水中爆発の場合，少量の加工であれば特別な機械は
不要であり，屋外のプールなどで加工できる．多量生産であれば，
その内壁を気泡のカーテンそして底部を空気の入ったホースによっ
て，衝撃圧から保護した鋼製タンクが利用できるほか，室内で成形
できる装置も開発されている．
.高速ハンマ 高速鍛造および高速せん断には，10～30
m/sのラム（ハンマ頭）速度が出せる高速ハンマが利用される．
これらの機械としては，高圧の窒素ガスや圧縮空気の膨張エネルギ

b.プラスチック射出成形機（injection molding machine)
一般的な射出成形機の構造を図9･70に示す．ホッパから供給され
た成形材料はスクリューによって混練されつつ先端部へ送られ，そ
の過程で加熱シリンダからの熱と材料内部の摩擦熱によって均一な
可塑化状態となる．スクリュー先端部に一定量の材料が蓄積すると
スクリューが停止し，型締め装置によって密閉されている金型内
へ，溶融した材料が射出シリンダによって高圧で射出される．材料
は型締めされたまま凝固して製品となり，型をあけて取出される．
通常，熱可塑性樹脂では，射出時の温度175～320℃，金型温度
30～120℃，熱硬化性樹脂では，射出時の温度100～120℃，金型温
度160～190℃の範囲であり，成形材料に適した温度に制御される．
c .レーザ/プラズマ加工機等 レーザやプラズマといった高密
度エネルギーを利用した加工機，特にレーザ加工機（laser beam
machine）が急速に普及している．被加工材を極めてエネルギー密
度の高い媒体（レーザ/プラズマ）により溶融・蒸散させ，あるい
は燃焼させて切断を行う．切断精度はレーザ加工機のほうが高い
が，高い加工精度が要求されない製品の加工には価格が低いプラズ
マ加工機（plasma processing machine）が用いられる．また，レ
ーザや電子ビームにより溶接を行う機械も普及し始めている．さら
にレーザ照射による熱/残留応力を利用したレーザ加工も，インク
リメンタル成形の一つとして注目されている．加工にはCOレー
ザ（気体レーザ，波長10.6m）やYAGレーザ（固体レーザ，波
長1.06m）が用いられている．YAGレーザ光は波長が短いので
光ファイバでの伝送が可能であるが，COレーザは不可能である．
これらのレーザ光は不可視光（赤外領域）であり，被加工物からの
反射光が人体を損傷しないように，使用に当たっては細心の注意が
必要である．
光硬化形樹脂とCAD/CAMシステムを組合わせ，三次元形状を
創成し，金型の原型を作るラピッドプロトタイピング機（rapid
prototyping machine）が産業界の注目を集め，普及し始めてい
る．塑性加工機械とはいえないが，塑性加工に必要な金型製造技術
として重要性は高い．
