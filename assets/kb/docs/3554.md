# 8・1・1 オープン化の必要性

> **章**: 第8章 社会と生産システム
> **節**: 8・1 生産システムのオープン化

生産システム技術は,1990年ごろから,自動化から情報化へと変化し始めた.以前は,作業者の代替として自動機が積極的に導入され,無人化を頂点とするオートメーションが行われてきた.しかしながら,高度な自動化を実施するにつれて,メカニカル制御から電子的制御へ,さらに,コンピュータによる情報的な制御へと変化してきた.特に,1990年代においては,情報化の進展によりネットワークとコンピュータを利用した高度な自動化を期待するようになってきた.また,同時に,製品の市場での飽和感が生まれ,多品種少量生産,多種多変量生産へと移行する時期と重なり,柔軟な生産システムを期待する流れが生じた.その後,アジャイル生産など製品のライフサイクルスピードの高速化に合わせるスピード経営が一般化し,それに適した生産システムの構築が望まれるようになってきている.しかしながら,工場の生産システムにおいては,製品のライフサイクルが短くなったとしても,設備投資の回収などの関係から設備のライフサイクルを極端に短くすることができないというジレンマが生じてくる.また,長期の生産量の動向や製品の技術動向も予測できないほど混とんとしてきて,設備の長期的設計が不可能になってきた.そこで,設備のモジュール化や複合化を行い,基本システムを長期間維持し,部分的な変更を行いながらその時々の製品に対応するシステムを作ろうとしてきた.このようなシステムを作るため,マルチベンダ環境で生産システムが作れること,設備やソフトウェア間の相互のデータ互換が可能なことなどの要求が出てきた.これが,生産システムのオープン化の基本であった.次に,生産の管理機能と営業機能とでデータ共有をすることによって,顧客ニーズに対応しようとするようになり,生産機能と他の機能とを結びつけるようなオープン化が考えられてきている.その後,生産そのものが海外で行われたり,外部の企業に生産部分を委託する必要性が生じたりして,製品データや生産管理データの交換が必要になり,生産情報のオープン化が積極的に議論されるようになった.生産システムは,それぞれの企業内の独自のシステムであり,外部と互換性のないユニークなシステムを作る時代から,さまざまな企業と連携して,高効率的な生産システムを作る時代へと変わってきた.これらを支える技術がオープン生産システム技術として注目されている.ここでは,通信ネットワークのオープン化と,生産システムの機能と生産情報のオープン化の視点に分けて,オープン生産システムを見ることにする.
