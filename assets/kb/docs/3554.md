# 8・1・1 オープン化の必要性

> **章**: 第8章 社会と生産システム
> **節**: 8・1 生産システムのオープン化

生産システム技術は，1990年ごろから，自動化から情報化へと
変化し始めた．以前は，作業者の代替として自動機が積極的に導入
され，無人化を頂点とするオートメーションが行われてきた．しか
しながら，高度な自動化を実施するにつれて，メカニカル制御から
電子的制御へ，さらに，コンピュータによる情報的な制御へと変化
してきた．特に，1990年代においては，情報化の進展によりネッ
トワークとコンピュータを利用した高度な自動化を期待するように
なってきた．また，同時に，製品の市場での飽和感が生まれ，多品
種少量生産，多種多変量生産へと移行する時期と重なり，柔軟な生
産システムを期待する流れが生じた．その後，アジャイル生産など
製品のライフサイクルスピードの高速化に合わせるスピード経営が
一般化し，それに適した生産システムの構築が望まれるようになっ
てきている．
しかしながら，工場の生産システムにおいては，製品のライフサ
イクルが短くなったとしても，設備投資の回収などの関係から設備
のライフサイクルを極端に短くすることができないというジレンマ
が生じてくる．また，長期の生産量の動向や製品の技術動向も予測
できないほど混とんとしてきて，設備の長期的設計が不可能になっ
てきた．そこで，設備のモジュール化や複合化を行い，基本システ
ムを長期間維持し，部分的な変更を行いながらその時々の製品に対
応するシステムを作ろうとしてきた．このようなシステムを作るた
め，マルチベンダ環境で生産システムが作れること，設備やソフト
ウェア間の相互のデータ互換が可能なことなどの要求が出てきた．
これが，生産システムのオープン化の基本であった．
次に，生産の管理機能と営業機能とでデータ共有をすることによ
って，顧客ニーズに対応しようとするようになり，生産機能と他の
機能とを結びつけるようなオープン化が考えられてきている．
その後，生産そのものが海外で行われたり，外部の企業に生産部
分を委託する必要性が生じたりして，製品データや生産管理データ
の交換が必要になり，生産情報のオープン化が積極的に議論される
ようになった．
生産システムは，それぞれの企業内の独自のシステムであり，外
部と互換性のないユニークなシステムを作る時代から，さまざまな
企業と連携して，高効率的な生産システムを作る時代へと変わって
きた．これらを支える技術がオープン生産システム技術として注目
されている．
ここでは，通信ネットワークのオープン化と，生産システムの機
能と生産情報のオープン化の視点に分けて，オープン生産システム
を見ることにする．
