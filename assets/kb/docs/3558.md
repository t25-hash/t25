# 8・2・1 大域的効果の予測に基づいた生産システムの設計・制

> **章**: 第8章 社会と生産システム
> **節**: 8・2 民族性・地域性

御近年、生産システムは地球規模で構築され、国際的な枠組の中で開発・設計・製造・調達・運用・販売などの関係が築かれるようになった。こうした時間と空間を超えた情報交換や生産取引のために、産業社会はより苛烈な競争社会へと導かれている。それゆえ、生産システムは、よりいっそうの遂行能力(performance)の向上を目指すことが求められ、国際分業、技術移転、企業改革、技術革新(innovation)、特化型製品などの開発や実施に力が注がれるようになってきた。しかしながら同時に、産業社会は資源の浪費・自然環境劣化など負の資産を残してきたために、その解決も求められている。すなわち、生産システムの遂行能力向上を地球環境の破壊回避などの産業社会全体に関わる大きなスケールの問題として解決することが望まれている。この問題は、産業社会のミクロな要素である一企業レベルの集団や個人に、要素や要素間の局所的な動作問題としての解決をゆだねれば済むという種類のものではない。例えば、地球温暖化に関して考えれば、人間は地球を暖めようと思って自動車に乗り火力発電をするわけではなく、仕事に便利だから自動車に乗り、危険が少ないから火力発電をしているのである。ところが、自動車や火力発電を利用することにより二酸化炭素が発生し、やがてこれが累積して、地球が温暖化を被ったわけである。すなわち、ミクロな要素の動作や要素間の局所的な相互作用により、新たな大域的な挙動や効果が出現したということであり、こうした新機能は企業や個人が意図した結果ではなかったのである。したがって、産業社会全体に関わる問題として生産システムの遂行能力向上を解決するためには、要素の局所的な動作やその制御の集積によって、新たな大域的な振舞い(global behavior)が現れるという可能性、意図せざる結果に注意するために、つねに調査や解析によってマクロな挙動や効果を予測し、そうした予測に基づいて生産システムを設計・制御することが重要である。図8・3にこの関係を示す。図8・3の評価フェーズでは、生産システムを取り巻く環境や状況を同定し、生産システムの現状評価を行う。解析フェーズでは、ミクロな要素の動作や要素間の局所的な相互作用と大域的な挙動や効果との関連をモデル化し解析する。調査フェーズでは、詳細な計測データを求め、定量的な関連を調査する。設計・制御フェーズでは、結果に基づいて具体的に生産システムを設計・制御したりする。実際の生産活動はこのフェーズに含まれる。この後、再度評価フェーズに入り、常時、環境や状況の同定とシステム評価を行う。以下、8・2節では調査、解析によるマクロな挙動や効果の予測と、予測結果に基づく生産システムの設計・制御概念との関連について述べる。なお、大域的効果として、環境対策は7章で扱ったので、ここでは民族性・地域性に関係する産業社会全体の効用(満足の度合い)、活性化、利益を対象とする。図8・3大域的効果予測に基づいた生産システム設計図8・4 行為の意味づけ7―171
