# 9・3・2 倫理綱領の機能 ところで，倫理綱領がもつ機能と

> **章**: 第9章 技術者倫理及び資格に関する制度
> **節**: 9・3 専門職能（集団）（profession）と倫理綱領

しては，（1）一般社会と専門職能集団との「契約」に関する明確な
意思表示，（2）専門職能集団のメンバーが目指すべき理想の表明，
（3）倫理的な行動に関する実践的なガイドラインの提示，（4）将来
のメンバーを教育するためのツール，などが考えられる．そして，
おそらく最も重要なものとして，（5）これまでの諸点並びに専門職
能集団の在り方そのものを議論する機会を提供するための場，を挙
げることができる．
まず，第一一の機能については，すでに述べた「社会契約説」の立
β9－200 β9編 法 工 学
場に立つと，プロクェッションとしての行動規範を明文化し，プロ
クェッションの内外に明示することは，「契約」に不可欠の要素であ
る．つまり，社会からの信託を受ける以上，プロクェッションの構
成員は明確に公示された規範に則り行動し，職務を全うしますとい
う約束事項の表明である．
第二の機能は，プロクェッションとしての向上目標の標榜である．
すなわち，プロクェッションのメンバー一人ひとりが自らの能力を
継続的に開発させていくべき方向性の提示と，プロクェッション全
体が社会の中でその地位をさらに向上するために理想とする姿の明
確化が図られる．
第三号機能は，実際に倫理的なジレンマに直面しているメンバー
が倫理的な判断を下すにあたり自律的に意思決定をするためのガイ
ドラインを提供することである．ここで注意しなければならないこ
とは，倫理綱領はあくまでもガイドラインであり，法律のように厳
密に適用条件が定義されものではないという点である．プロフェッ
ショナルに求められている「道徳的自律性（moral autonomy）」と
は，厳密に明文化された規定条項に盲目的に従うことではなく，自
ら情報を収集・分析し，関連する事柄を熟考した上で，自分が属す
る組織や他者からの影響からは独立に，道徳的意思決定を行うこと
である．また，そのような意思決定に従って行動し，その行動の結
果の道徳的責任を負うことが道徳的自律性である．つまり倫理綱領
は，道徳的自律性をもつプロフェッショナルがジレンマに陥ったと
きに行動の指針を与えるという機能をもつものである．
第四の機能は，例えば，大学生などのように，将来エンジニアリ
ングのプロフェッショナルになろうとする人たちに対して，専門的
な知識や技能の他に，どのような人格的資質や倫理的徳目が求めら
れているのかを明示することである．専門教育を受ける間に，倫理
綱領に触れることによって，自らが伸ばしていくべき資質が明確に
なる．このように達成目標あるいは向上目標が明文化されることに
よって，大きな教育的な効果が期待される．また，実際に技術者倫
理教育を行う上で，現存するプロクェッションの倫理規範を提示す
ることは不可欠である．第五の機能は，これまで挙げた諸点をすべ
て総合し，専門職能集団の在り方そのものを継続的に議論する場を
提供するという最も重要な機能である．専門職能集団が，その集団
しかなしえない専門的な業務を遂行する以上，そのメンバーが専門
分野に関する興味・関心を共有するのは当然であるが，加えて，自ら
のアイデンティティについて自省すべきである．しかしながら，現
状では，専門職能集団がそのような考察の機会をもつことは少ない．
しかし，継続的に倫理綱領を検討することにより，社会との契約事
項や自らの向上目標，またメ1ンバーが直面している具体的な倫理的
ジレンマ，又は後継者育成の在り方などを慎重に検討することがで
きる．さらに，自らの存在意義とは何かという本質的な問題につい
ても熟考する機会が与えられる．このような自省の場を提供するこ
とこそが，倫理綱領の最も重要な機能である．
