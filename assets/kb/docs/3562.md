# 8・2・4 生産システムでの考慮要件

> **章**: 第8章 社会と生産システム
> **節**: 8・1 からわかるように賃金上昇があり,同説は必ずしも当てはまら

地域性に関係するマクロな挙動と効果の予測,そして予測結果に基づいた生産システムの設計・制御について概説した.主要なトピックスについてまとめると,次のようになる.(1) 産業社会全体の効用や活性化,利益の観点から,民族性や地域性を生かした分業・特化による生産システムの設計・制御法がありうる.(2) 評価フェーズで環境変化を同定し,変化の度合いが異質か,似通っているかに応じて,生産システムを特化するか,あるいは総合化するか,などを設計・制御することはミクロな利益とマクロな効果の両点で有効である.(3) 技術移転をする場合,一方向教育を選択するか,双方向教育を選択するかは,生産文化に適合する形で決定することが効率的である.(4) 技術革新を生産システム全体に伝ぱさせるためには遷移率を一定値以上に高めることが必要である.すなわち,科学技術に高い価値を付ける,新しい考え方が外からすぐに入ってくる,計画的・合理的に仕事をするなどといった生産文化を生産システムにもたせることが重要と考えられる.(5) 生産システムならびに製品を特化する場合,民族性,地域性に適合した文化的設計属性を求める調査と,その属性値を定量化してそれを形とする技術が重要である.最後に,8・2節での考慮例では時間項を無視したが,実際には評価や調査のサイクルがあり,セル型生産システムの例で述べた,選択権をもたせるような時間項を考慮する設計・制御法のあることを注記しておく.
