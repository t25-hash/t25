# 8・3・3 リスクアセスメント

> **章**: 第8章 社会と生産システム
> **節**: 8・3 安全の国際的整合化

a. 機械使用の限定リスクアセスメント(risk assessment)とは,「機械類に付随する危険源の審査を系統的に実施するための論理的手順(ISO14121:1999)」であり,機械の使用の制限,危険源の特定,リスクの見積り,および,リスクの評価からなる査定作業である.機械の安全化のみならず,労働安全衛生マネジメントシステムの普及に合わせて職場への導入が促進されている.しかし,リスクアセスメントという概念は,機械の使用によって異なるので,この手順を標準化するには限界がある.多くの提案がなされており,今後,リスクベースドセーフティ(risk-based safety)の体系化がなされて有効なリスク評価法が完成するであろうが,それまでは,最適な方法は使用者が作成することが前提になっている.ISO12100で述べられているリスクアセスメントは,対象となる機械の使用に対し,以下の限定を明確にすることから開始される.(1) 合理的に予見可能な誤使用を考慮した,機械の使用上の限定(意図される使用の決定).(2) メンテナンス等の非定常作業も考慮して,人が機械可動部に接触するおそれのある危険な領域,または人の機械へのアクセス方法の限定(空間的制限).(3) 機械の寿命の限定(時間的制限).機械を設計する際に,これらの限定を明確にすることで,機械の機能上必要最小限の危険源を明確にできる.ただし,合理的に予見可能な誤使用には人の反射行動や省略行動,不注意といった人特有の挙動に加えて,防護方策の意図的な無効化も含まれる.これは,ISO12100が設計者の立場からの製造物責任予防(PLP: productliability prevention)の目的をもつためであり,仮に意図的な無効化によって災害が生じても,設計者(製造者側)がその責任を問われる場合がある.b. 危険源の同定危険源とは,物理的障害または健康障害を引き起こす潜在的根源を意味し,一般に,確定論的危険源と確率論的危険源に大別される.確定論的危険源は,機械の使用中につねに生じるものであり,危険な機械可動部の動き,溶接作業中のアーク,不自然な姿勢,高温などである.一方,確率論的危険源は,典型的には部品故障,構造体の腐食や摩耗によって確率的に発生するものであり,管破裂による高圧流体の噴出や機械の予期せぬ起動に起因する災害を指す.両者は対策の方法が異なり,特にISO12100で述べられているリスク低減方策はおもに確定論的な危険源に関するものである.危険源の同定は,過去のデータ,経験,資料等を参考にして該当するものを探し出すことである.危険源同定(hazard identifi-cation)を一つでも見逃すと,それに対するリスク評価が行われないため,設計時に防止できる安全対策の機会を失うことになる.危険源同定はリスクアセスメントにおいて最も重要な作業だといえる.この作業を補助する目的でISO12100では,以下の11の分類に基づいて40項目以上の基本的な危険源が準備されている.(1)機械的危険源,(2)電気的危険源,(3)熱的危険源,(4)騒音による危険源,(5)振動による危険源,(6)放射線による危険源,(7)材料/物質による危険源,(8)設計において人間工学的原則を無視することによる危険源(ヒューマンエラーを含む),(9)滑り/つまずき/転倒の危険源,(10)危険源の組合せ,(11)作業環境に関連する危険源.c. リスクの見積りと評価国際安全規格では,機械の危険性の度合いをリスクで表現する.危険源へのばく露(人のアクセス)の頻度が高く,また,被る傷害が大きいほど危険性が高いことは容易に理解でき,これをリスク要素で示すと図8・11の関係となる.実際のリスク解析では,さらに付加的要素を考慮する場合が多い.例えば,危害のひどさでは対象の人数,ばく露の頻度では機械への接近の種類(直接か間接),人数,回避の可能性では危険事象の発現速度,リスクの認識度,機械操作者の熟練度や反射作用などである.これらの要素からリスクの大きさを見積もる手法がいくつか提案されている.例として,IEC61508:1998にあるリスクマトリックス(インデックス)を表8・5に示す.傷害の起こりやすさと傷害の重篤さの組合せから,リスクの大きさを四段階に分類している.リスクがI~IIIに分類されればなんらかのリスク低減方策が要求される.一方,ISO13849:1999では,図8・12の方法を示している.これによると,図8・11のリスク要素の組合せを()と(,,)に再編成し,前者をリスクの見積りに,後者を防護方策図8・10
