# 8・4・1 生産システムにおける技能

> **章**: 第8章 社会と生産システム
> **節**: 8・4 熟練と人

a. 人の役割旋盤やフライス盤,研削盤のような従来のはん用工作機械を用いた生産においては,作業者が段取りを行い,それに基づいて作業者自身が工作機械を直接操作することによって目的の加工を行う.この場合,段取りよりも機械操作にかける時間のほうが長いことが一般的である.つまり,はん用工作機械における人の作業は,人のもつ意図を人が動作することによって加工作業に直接反映させることが中心である.ここで人は,加工状態を検知・診断し,それに基づいて所望の加工状態を維持するために工作機械の制御も行うセンサやアクチュエータとしての役割をおもに果たす.一方,NC工作機械やFMS に代表されるコンピュータ制御機器を中心にしたFA(factory automation)システムにおいては,加工意図はあらかじめプログラムされ制御情報として制御装置に蓄積されるので,作業は制御機器のタッチオペレーションが中心になる.このため,機械操作よりも段取りにかける時間のほうが長くなる.つまり,FA システムにおける人の作業は,段取りをはじめとしてシステムが目的を達成するように判断・処置を行う,知的管理機能としての役割がおもになる.また,CIRP(International Institution for Production Engi-neering Research)では生産における人の役割を,人-組織-技術要素のより密接な相互関係を実現するために必要なものとしてとらえ,その中で人は次のようなフレキシビリティを生産プロセスにおいて保証する役割を負わなくてはならないとしている.(1) 人のもつ想像力を企業の最も重要なリソースとして育てること.(2) すべての製品は,機能や信頼性,品質,環境調和性が関係する限り人間中心型であること.これに留意するほど,社会から技術の受容が大きくなる.(3) 工場内において,装置や機器間の情報交換を重要視するだけでなく,人どうしのコミュニケーションもさらに重要視すること.(4) ライフサイクルエンジニアリングも同様に人間中心型であること.生産に携わる者は,製品の品質や機能に注意を払うだけでなく,生産プロセスや使用期間の環境への負荷などにも留意することが重要である.こうした生産活動に関わる人のさまざまな役割を近年では技能としてとらえ,ものづくりにおける重要性を見直すことによって産業競争力の向上を図ろうとする活動が活発化してきている.b. 技能の諸定義技能の定義は図8・16に示すように言葉のうえでは循環的なものになり,必ずしも明確ではない.そのため生産活動における技能も多様に解釈され,さまざまな立場から多くの研究者によって定義されている.そのおもなものとしては以下のようなものが挙げられる.(1) 物事を行う腕まえ.技量.(2) ものづくりにおける知識,作り方,目的,イメージなどの明示的思念を技術,明示的な思念に従って個別性を極力抑えるようにものを具体化する非明示的な個人のわざ(能力)を技能と呼ぶ.(3) 技能は技の能(わざののう)すなわち,人間の生産行為の能力である.(4) 技能とは,生産に必要な腕まえであり,人に結びついたものである.(5) ものづくりに関わる人間側の要因.例えば能力や行為の手順・ノウハウなど.生産現場で人が発揮するわざであり,手技的(テクニック的,操作的)なもののみならず,知識・経験など知的なもの,仕事に対する態度など精神的なもの,コミュニケーションやマネジメントなど社会的なものなども含まれる.7―180生産システム工学(6) 技能とは,経験の蓄積によって特定の人に身についている,ものを作る能力や知識.技術とは,客観化・形式知化されたものを作る方法や手順.ノウハウとは,技術の中で一般的に公開されていないもの.c. 技能の構成要素生産における技能を構成する要素に関しても,技能の定義と同様にさまざまな考え方が示されているが,大きく二つの要素に分けて議論されることが多い.(1) 手さばきや器用さ,感覚に関する技能行動動作を思うとおりに,手早く,いつも同じように行える技能.あるいは道具・機械・装置などの限界を極限にまで突き詰めたかたちの加工・組立技能などである.(2) 新しい問題や類似問題に対応できる技能条件に即応した行動動作が行えるという発想,判断,気づきといった技能,あるいは製品やそれらを生み出す生産プロセス自体の不具合の発生原因を,迅速かつ的確に探り当て解決できる技能などである.いわゆる,勘やこつなどはこちらに一般的に分類される.(1)を意味する言葉としては,身体制御型技能,クラフト型熟練,感覚運動系技能,技的技能などが使われている.(2)に関しては,問題解決型技能(熟練),知的管理系技能,知的技能などで表現される.また,(1)と(2)の技能をさらに細分化して,(1)については加工・組立の技と感知力,(2)については問題把握・解決力と知識として議論されたり,自動車製造工程においては図8・17に示すように,手さばき,検査・感知,設備操作,知識・経験,現場対応,改善・工夫の6階層の要素で表現されることもある.近年,情報技術やネットワーク技術の急速な発達,また製品とその要素技術の複合化・高度化・微細化の進展,産業競争力強化の必要性などの理由により,新しい問題や類似問題に対応できる技能(2)の重要性が増大してきている.図8・16 技能者の分類技能者は,対象とする業務内容で決められる難易度や専門性などのレベルに応じて区別される.一般化,共通化されていないレベルをもつ技能者を高度熟練技能者あるいは高度技能者,また単に熟練技能者などと呼んでその他のほぼ共通化している技能レベルをもつ者と区別することが多い.技能の重要性が認識され始めた近年,積極的に高度熟練技能者の育成を目指して厚生労働省では1998年に高度熟練技能基盤強化支援事業を開始した.ここにおいて高度熟練技能者を,次のように定義している.高度熟練技能者:機械では代替できない高度な技能を駆使して,高精度・高品質な製品などを作り出すことができる技能者,または機械が作り出す製品と同等以上の高精度・高品質の製品を作り出すことができる技能者であり,例えば,機械や加工材料の微妙な状態を,五感を駆使して振動,音,火花の出方などから判断し,1/1000ミリメートル(1ミクロン)単位の加工を行うことができるというような技能者である.一方,技能者が生産システムにおいて果たす役割からその内容に応じて,スーパー技能者,ハイテク技能者,マルチ技能者,ノーマル技能者とした四つの類型で分類することも提案されている.e. 技能のモデル化生産における技能はその非言語性や属人性,状況依存性が強い.そのために,技能は認識論や社会論から分析した暗黙知として説明されることが多い.暗黙知の概念は,ポラニーが「われわれは語ることができるより,多くを知ることができる」という命題より導き出した知の体系的モデル化である.ポラニーは機能的側面,現象的側面,意味的側面,存在的側面から暗黙知を検討して技能の非言語性や属人性を説明しており,工学的アプローチだけでは記述できない技能分析に貴重な視点を提供している.近年,情報技術が発達するにつれ,技能者の思考過程を情報処理モデルとして取扱うアプローチも見られる.図8・18はきさげ作業者の技能を情報処理モデルとして分析した例である.この情報処理モデルでは,適切なパターン処理,高速なプランニング,質の高い解の導出,運動システムの制御機能という部分的問題と,それを階層的に統合した問題として記述されている.また,人工知能研究から派生したエキスパートシステムやニューラルネットワーク,ファジィ推論は,技能者のノウハウや知識の人工的構成を目的とした分析的なモデル化法と考えることができる.ロボティクス研究においては,作業の力学的モデル化のアプローチがある.技能のモデル化にはさまざまな概念やアプローチが提案されてい図8・17自動車製造工程における技能の分類図8・18
