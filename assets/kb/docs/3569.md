# 8・4・2 熟練技能の機械化

> **章**: 第8章 社会と生産システム
> **節**: 8・4 熟練と人

熟練技能の機械化の流れは、8・4・1項で示した技能の二つの構成要素に対応して、感覚・運動技能の機械化と問題解決技能の機械化とに分けることができる。a. 感覚・運動機能の機械化手さばきや器用さ、感覚に関する技能をセンサやアクチュエータ、制御装置で代替して、人の介在するところを極力なくし、作業者に任されていた一連のプロセスの無人化・全自動化を指向したものである。特に、ロボット分野の成果利用が積極的に行われている。人の作業を運動学的・力学的に解明し、人間の器用な動きの人工的な実現を目指すものを出発点としている。生産システムで使われる多くの産業用ロボットの動作は、基本的には指定された点を順次位置決めし、所定の動作経路を通過していくPTP(point-to-point)方式に基づくものが多い。そこで、こうした機能を利用して素材や製品の供給、搬送、取出し、また工具交換などに利用され、人間に比べてはるかに高速、高精度な作業を可能にしている。こうしたロボットの動作は必ずしも人間の動きをそのまま再現してはいないが、高自由度なロボットハンドを利用すれば熟練動作の記憶と再現、また人間動作の計測や分析も行える。感覚の機械化には一般的に、多くはセンサ信号を情報処理してなんらかの認識機能や制御機能を併せもったシステムとして実現される。視覚としてはおもにCCD センサが用いられ、その信号を画像処理することによって形状などの抽出と認識、異常や欠陥の識別などの機能を高速に行うことが多い。また、触覚としては力センサがおもに用いられ、その情報を用いた力制御と一体になってバリ取りや磨き、溶接ビード余盛り除去などの作業の自動化に適用されている。感覚や運動機能の機械化は、人間に近い動作や判断をもった、いわゆるインテリジェント生産システムの実現を指向している。こうしたシステムにおいては、感覚からの情報の質がシステムの最終性能を左右することが多い。b. 問題解決技能の機械化熟練技能者のもつ判断や発見、予測、意思決定などの機械化である。こうした問題解決技能の機械化に対しては、製造プロセスの科学的解明によるアプローチと、人間のもつ意思決定プロセスのシステム工学的解明によるアプローチに分けられる。製造プロセスの科学的解明によるアプローチでは、プロセスメカニズムが不明確なため人間の経験と勘に頼らなくてはならなかった加工などの製造プロセスを科学的に解析し、それに基づいた工学的モデルを開発して情報システムで代替しようとするものである。最も典型的な例は、CAE(computer aided engineering)である。一般的にCAEはCAD と併用され、経験と勘に頼ってきた製品の不具合や性能予測を、製品設計段階におけるコンピュータシミュレーションによって可能としている。意思決定プロセスのシステム工学的解明によるアプローチにおいては、人工知能の分野で開発されてきたエキスパートシステムやニューラルネットワーク、ファジィ推論などに代表される知識処理的手法を適用し、システムの制御や診断に関わる熟練技能者の思考プロセスのシステム化を目指すことが多い。これらは、複雑な制御則が必要な大規模システムに用いられている。グローバル化(globalization)の進展に伴い、大企業といえども製品の製造ビジネスプロセスすべてを自社で実行することは困難になってきている。顧客の要求も増大しており、タイムリーでローカルなニーズに合ったサービスの提供が必要不可欠になっているのだ。また、情報通信技術の進歩により、時間や空間を超えた企業間7―181協調作業が可能になり、コアコンピタンスに注力して最適な企業連携(分業や協業)により、差別化した製品やサービスを提供する動きが加速している。日本企業はさまざまな分業の体制を1970年代以降、東アジアの国々と営々と構築してきた。この東アジアとの国際分業体制は、もちろん産業によってパターンがかなり異なり、大きく分けると三つのタイプがある。第1は、日本と東アジアの国々が複雑な製品や工程間の分業を進めて、網の目状という意味でのネットワークにかなり近づいているもの。この分業の結果、東アジアで生産される製品の市場は、東アジアの域内市場がおもな市場であるが、日本をはじめ欧米にも輸出されている。産業としては、エレクトロニクス、精密機械などがこのタイプの分業をしている。最も進んだ分業体制といっていいだろう。第2は、日本が財の流れの源泉となっていて、日本からの輸出を促進させるような状況になっている分業体制である。東アジアの国々に最終工程の工場が作られ、そこで生産活動の最終段階が行われるものの、その工場への部品や資材の供給の多くが日本の源泉から行われている。東アジアで生産される製品の販売先は、圧倒的に域内の市場向けである。このタイプの分業では、現地国での生産や販売の規模が大きくなると日本からの対東アジア向け輸出は増えていく。産業としては、自動車、鉄鋼、化学、一般機械など、多くの産業財がこのパターンになっている。第3のタイプは、日本が財の流れの終点となっているもので、日本市場向けの供給を目的とした分業体制である。このタイプの分業では、その規模が拡大すると日本の東アジアからの輸入が増えていく。産業としては、繊維産業の縫製部門が典型的な例である。また今日、中国は世界の工場としてもてはやされているが、中国の著しい工業発展は、中国政府や企業の努力もさることながら、外国企業の進出とそれに伴う国内競争の激化に負うところが大である。中国の産業における外資系企業の比重は大きく、かつ年々増加している。しかし、外国企業からの技術導入と外資の中国市場への参入に伴う競争激化によって中国企業も着実に力をつけてきたことは事実である。最近では、成熟企業を中心に競争力をつけた中国企業が外資系企業とイコール・パートナーのアライアンスを結ぶ動きや、海外進出に踏み出す企業が目立って増え、中国の産業発展は新たな段階を迎えようとしている。ここでは、エンジニアリング産業を対象として国際分業・協業の動向を考察する。
