# 1・1・1 生命

> **章**: 第1章 生体の構造と機能
> **節**: 1・1 生命システム

生命とは何かという問題にはいろいろな考え方がありうるが、大方の意見が一致するところは、外界から区切られた構造をもち、外界と物質および情報の交換を行う能力があり、自己あるいは自己に類似する構造体を再生産する能力をもつ存在というものであろう。第一の項目が細胞(cell)と膜(membrane)の存在を、第二が代謝(metabolism)と刺激(stimulus)への反応を、そして、第三が生殖(reproduction)を意味することはいうまでもない。もちろん、これらの定義に基づけば、ある種の機械-人工物も生命の範ちゅうに入ってしまうという問題が生じるが、本章ではこの問題には深入りしないことにする。われわれが目にすることができる地球上の生命体にも相当風変わりなものがあり、固定観念をもって限定的に生物をとらえることは危険である。まして地球上の生命体が、われわれの知る宇宙における唯一の生命体であるかどうかは現状において不明であり、現在のわれわれの想像を絶する生命体が存在する可能性は否定できない。地球上においてわれわれが現在見ることができるような種類の生命体の歴史は、数十億年にさかのぼると考えられているが、その原初の存在が何であったかについても多数の異論がある。しかし、極端な創造説をとらない限り、何か極めて単純な存在からスタートして順次複雑化し、現在見られる多様な生物群が存在するに至ったというのが大方の一致するところであろう。こうして現在われわれが見る生物の多様性には驚くべきものがあり、この多様性(polymorphism)を説明するためにこそ進化(evolution)の概念を必要とすると考えられている。進化についてもまた多種多様な考え方が存在するが、少なくとも、その時間スケールが個体の生存の時間スケールよりはるかに大きいことを考えると、その現象が個体の再生産-生殖の過程と密接に結びついているであろうことは想像にかたくない。すなわち、生命体の変容は、長大な時間スケールの再生産の膨大な繰返しによって実現する。栽培される植物や家畜などの動物の品種改良は、たかだか人間が把握できる程度の時間で実現されるが、これで改変されるのは品種であって、種を超える変化が生み出されることは観察されていない。生命に関するものに限らず、どのような議論も、出発する定義にこだわると結局自縄自縛のトートロジーに陥りがちであるから、上述の生命の定義は、いわば作業仮説の域を出ないものと考えておく必要がある。細胞と膜の存在を生命の要件とみなす立場もそのようなものである。ある種の病原体、ウイルスやプリオンなどは、複製する機能をもった分子であって、自己の体制を維持するための機構を全くもっていない。これらの存在を生物と呼べるか否かも、神学論争にならざるを得ない。いずれにせよ、われわれが生命体と呼ぶすべてのものは、ウイルスなどを含めたとしても、複製する人工物とは大きく異なる特徴をもっている。その顕著なものの一つは、外界と生命体の情報交換は、物質的な方法によるということであろう。より限定的にいえば、細胞とその外界(他の細胞を含む)との情報・物質交換は、細胞膜を貫通するタンパク質(protein)による。特定のタンパク質(レセプタと呼び替えてもよい)によらない物理的な拡散などの物質輸送も存在するが、それが固有の情報伝達を伴うことはないと考えられている。生殖が生命の現状のありようにもたらす影響ははかり知れないものがある。進化とこれを支える遺伝の見方も多様であるが、俗説的なネオダーウィニズムから木村の中立説を含むすべての進化に関する学説において、変異しあるいは獲得・選択された形質が蓄積・実現するのが、はるかな世代交代を経た後であることにおいては一致している。甚だしい生気論(生命を生命たらしめるのは、いわくいいがたい生気であるとする論)はとるべきでないが、われわ8―1れ機械技術者は生命体を、短時間スケールで外界と反応する機械になぞらえて理解しようとする傾向があることは自戒しておかなければならない。代謝は、独立したシステムとしての生命体においてエネルギーの供給を図ることに第一の意義があり、さらに、ダイナミックに入れ替わっている生命体の構造要素を獲得し変容させることに第二の意義がある。第一の意義については、光合成によってエネルギーを獲得できるか否かという点で、植物と動物を区別することができる。第二の意義、すなわち体の構成要素の基本である炭素源を他の生命体に依存しない独立の栄養体と、他の生物に依存する従属性の栄養体を区分することができ、前者には植物のほとんどが含まれ、後者には動物のほとんどが含まれる。生物の分類において、動物と植物という二つの界に分ける見方ではなく、種々の微生物を別個の生物群とする見方が確立しているが、当面、われわれが機械工学・バイオエンジニアリングとの関係で見る限り、そこまで立ち入ることにはあまり意義がない。エンジニアリングとの関連でいえば、これらは、エネルギーと物質の輸送におけるインピーダンスマッチングの問題と考えることができ、代謝は1・3・6項で後述する消化の機能を含めて、生命体システム外において多様な様式で存在するエネルギーのインピーダンスを変換する過程であるといえる。
