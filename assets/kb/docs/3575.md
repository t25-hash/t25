# 1・1・2 ゲノム

> **章**: 第1章 生体の構造と機能
> **節**: 1・1 生命システム

分子生物学のセントラルドグマに従えば、生物をその生物たらしめるための情報はデオキシリボ核酸(DNA : deoxyribonucleicacid)の塩基配列(base sequence)として存在し、これが、リボ核酸(RNA)に転写され、さらに、タンパク質に翻訳されることにより伝達される。このような過程で伝達される遺伝情報(genetic information)の全体をゲノム(genome)と呼び、その実体は、遺伝子中のDNA の二重鎖分子における塩基の配列であることが明らかになっている。ここから当然の帰結として、塩基配列をすべて読み取ることができれば遺伝情報のすべてが得られ、ひいては、その生物全体を再現できるはずであるという考えが生まれ、これを全解読するというプロジェクトが20世紀の終わりに開始された。ヒトDNA についてのヒトゲノム計画(human genomeproject)は、すでに計画をはるかに前倒しして完成が宣言されており、広範な動植物、例えば、マウスやショウジョウバエ、ゼブラフィッシュ、酵母、線虫、また数多くの微生物や寄生虫などのモデル生物、有用な農産物であるイネなどについての解読が完成したことが報告されている。すでにDNA の配列の解読は全くのルーチンワークとなり、今後、全ゲノムが解読される生物の種類は増加する一方であろう。ここで注意しなければならないのは、解読された情報はあくまでも塩基配列にとどまり、そこから適当なタンパク質の合成を指示している遺伝子(gene)を認識し、その機能を理解することは別の事柄であるということである。さらに、遺伝子が塩基配列として確定したとしても、その情報が転写(transcription)・翻訳(trans-lation)を受ける過程で、環境に応じて編集、改変が起こることが知られており、それを担う細胞機能なしでは、遺伝と発生のメカニズムを理解するには至らない。つまり、遺伝子と細胞は、さながら卵と鶏の関係にある。ゲノム解析から生体全体の理解までの間には、相当の距離がある。特に、バイオエンジニアリングとの関連では、テーラーメイド医療が技術開発のターゲットとして重要であるが、断片的遺伝情報とシステムとしての生体との間を埋めるべき技術の登場までは、現在の技術進歩の加速をもってしても、まだ相当の時間が必要であろう。ゲノム解析の関係で研究手段としてポピュラーになっているのは、いわゆるDNA チップ(DNA chip)である。これは、断片化されたDNA(数十塩基)を高密度にガラスやシリコン性の小基板上に配置したもので、一度に数万種類の遺伝子の発現を調べること8―2ができる。当然、得られるデータは数万元のマトリックスとなるから、数学的にいえば適切な行列表現が不可欠であり、力学の用語でいえばテンソル解析の対象となる。後述するバイオインフォマティクス(bioinformatics)の一部では、このような解析が実際に行われている。ゲノム情報が明らかになっても、それが、転写される際の制御の複雑性(いわゆるトランスクリプトーム解析の対象となる)、さらに、タンパク質に翻訳される際の複雑性(プロテオーム)、そのタンパク質を酵素や材料として実現される代謝の複雑性(メタボローム)、そして、これらが組合わされた生理的システムの複雑性(フィジオームないしシステオーム)など、生命体の総合的理解への道はいまだ遠い。しかし、これらのポストゲノム研究のすべてにおいて、いわゆるバイオインフォマティクスが必須の研究手段となっていることでわかるように、これからは、培養細胞を含む生物の実験的研究(wet lab研究という)と、コンピュータを用いた研究(dry lab研究あるいはin-sillico研究)の併用が生物学研究の常識となるのであり、この方面におけるバイオエンジニアリングの貢献が期待されている。
