# 1・1・3 環境と相互作用するシステム

> **章**: 第1章 生体の構造と機能
> **節**: 1・1 生命システム

生命体は、その誕生以後、地球上における環境と相互作用し、それをドラスティックに変えてきた。周知の例は、光合成する植物が出現して、競争する生物を殺しつくすために地球上にまき散らかされた毒ガスである酸素が、さらに、酸素呼吸を行う有核細胞をもつ生物を地上と海中表層に出現させたという歴史である。植物のもつ環境改変能力には、その超長期的な戦略(このような擬人的表現が適当か否かはおくとして)も相まって驚くべきものがある。例えば、現在人類が発掘する数万年前程度の最近の遺跡を埋めたものは、基本的に植物の死骸(枯れ葉など)である。この意味で、ヒトがたかだかこの数千年において改変してきた地球環境への影響など、長い地球の歴史から見れば一瞬のエピソードにすぎず、遠くない未来に予測されるヒトの絶滅の後には問題となる痕跡など残さないであろう。上述のスケール、すなわち、進化論的スケールにおける環境との相互作用の基礎には、短期の物理化学的スケールにおける相互作用がある。個体としての生物は、陸棲のものは直接の重力環境において、海棲のものは水圧を通じて地球の重力環境に適応している。このために、ある程度以上のサイズの陸棲の生物は、なんらかの骨格(内外のいずれにせよ)をもつことによって、自己の体制を維持している。これらの体制維持のための構造が、重力を含む力学的環境に適応する機能と構築をもつことはよく知られた事実である。また、重力に抗して各種臓器を体内に配置し、機能させるための各種の機構、特にいわゆる結合組織(connective tissue)の力学は手つかずの分野が多いが、興味深い問題を含んでいる。その一つの例は、血液(blood)とそれを入れる容器であり、かつ、その循環を実現する血管系(vascular system)である。近年の研究の進展により、血管は血流を検知し自己適応を実現する能力をもつことが知られるようになった。この際、検知されているのは、血管内壁における摩擦であるが、この摩擦損失によって失われるエネルギーの源は心臓の拍動圧力であり、その圧力の相当部分は、姿勢変化などによる重力環境への適応のための余分の制御ゲインとして生み出されている。われわれが生命体について学べば学ぶほど、その巧妙な適応戦略に感嘆させられることが多いが、これらの根底に、生命体が与えられた物理的(力学的)環境において適応するために発達・確立されたシステムがあるという視点は重要であると考えられる。生物は細胞からなり、細胞は細胞膜で囲まれ、細胞分裂によって増殖する。細胞は、特殊な例を除けば、細胞質、DNA、タンパク質、および膜からなる。多細胞生物は、種々の機能を分担する多種類の細胞からなり、ヒトの体は、220種類の細胞から構成されているといわれる。図1・1に典型的な動物の体細胞の模式を示す。細胞質(cytoplasm)は種々の細胞小器官(organella)を含む。われわれ動植物を含む多細胞生物の細胞は真核細胞(eukaryoticcell)と呼ばれ、遺伝物質であるDNA :endoplasmicreticulum)、ゴルジ体(Golgi body)、エンドソーム、リソソーム(lysosome)、ミトコンドリア(mitochondria)、葉緑体(chloro-plast、植物の場合)などが含まれる。ミトコンドリアと葉緑体はそれぞれエネルギー産生に関わる。双方とも独自のDNA をもち、細胞分裂時に自己を複製することから、それぞれ、真核細胞に好気的バクテリアおよび光合成バクテリアが共生したものと考えられている。ミトコンドリアは、内部に電子伝達系(チトクローム系)をもち、酸素を使用して、酸化的リン酸化を営み、これがない場合に比べて10倍近いエネルギーを糖などから得ることができる一方、強力な細胞毒性をもつ酸素を解毒する。葉緑体は光合成により、光エネルギーを吸収して水を分解し、酸素とエネルギーを獲得することができる。小胞体は、細胞膜の構造が折り畳まれて細胞内に存在する装置で、この意味で、小胞体の膜の内側は細胞外と接続している。表面にリボソームを付着させているために粗に見えることから名付けられた粗面小胞体と、それがない滑面小胞体があり、粗面小胞体は、特に、細胞外へ分泌されるタンパク合成の場である。ゴルジ体も似た構造のもので、分泌タンパク質などの糖鎖修飾を行う。ゴルジ体が作る糖タンパク(glycoprotein)は、細胞膜表面における情報伝達を担う分子であり、細胞の機能と性質を決める重要な構造である。リボソームは、DNA 細胞膜の構造まれた小胞で、エンドソームは細胞内の物質輸送に、リソソームはその分解吸収に関係している。細胞膜、小胞体、ゴルジ体、エンドソーム、リソソームなどの膜系細胞小器官は、非常に速い速度で、移動、相互の融合、転換、分離を繰返しており、極めてダイナミックな存在である。図1・2には細胞膜の模式図を示す。細胞膜は、極性脂質であるリン脂質(phospholipid)が二層となった脂質二重層(lipidbilayer)を基本として、外側に親水基、内側に疎水基が向かい合う構造となっている。動物細胞はそのほかに相当の割合でコレステロール(cholesterol)を含む。この脂質膜の中に、埋め込まれたり結合したりした状態の膜タンパク質がある。この脂質や膜タンパク質には、ゴルジ体で作られた糖鎖が結合しており、これらの糖鎖は細胞内外の物質交換、情報伝達の要である。膜は、静的な構造体ではなく、イオンなどのチャネル、細胞外からのシグナルを受取る受容体(receptor)、さらに細胞膜の一部が小胞を形成して細胞内に物質を輸送する機能など、極めて多彩な活動を担っている。細胞膜も他の細胞構造同様固定したものではなく、これを構成する脂質が回転、移動、交換することにより、流動している。また、細胞の形を一定に保つために細胞骨格構造があり、ところどころで膜と結合していて、浸透圧(osmotic pressure)により内外圧力差が内側に正となっているため、その結果生じる膜張力と骨格の張力で細胞構造を支えている。細胞膜には内外のイオン濃度を維持するためのイオンポンプ機構(ion pump system)があり、細胞内部の電位は周囲に比べてつねに負に保たれている。神経、筋などの興奮性細胞では、この電位差が逆転する現象(活動電位)が生じ、情報伝達において主要な役割を果たす。バイオエンジニアリングにおける対象となる生物は、主としてヒトである。いわゆる医工連携において、機械工学が果たすべき役割は非常に大きく、医学がヒトを対象とする応用生物学であることを考えれば、『機械工学便覧』において生物学一般ではなく、ヒトの生物としての構造と機能が初めに取上げられることは当然であろう。本章に引き続く各章において、その対象となる生体構造、システム、機能、現象を特定して、そのバイオエンジニアリング的な視点からの解析と研究成果の紹介が行われるので、本章では、エンジニアリングの基礎として理解するべきヒトの構造と機能の概略を説明する。もとより、これらは伝統的な医学においては解剖学、組織学、および生理学として基礎医学教育において極めて大きな部分を占めるものであり、与えられた紙幅で叙述することなど到底不可能であるから、本当にこれらの知識を必要とする読者は、本章を手がかりに必要な解剖学、生理学の参考図書へ進むことを期待したい。
