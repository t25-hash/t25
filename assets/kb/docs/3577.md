# 1・3・1 筋骨格系(musculoskeletal system)

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

ヒトはいうまでもなく脊椎動物であるから内骨格で、かつ、直立歩行を行うために、特に四肢の構造に他の動物とは異なる特徴をも8―3図1・3 ヒトの骨格と外形つ。図1・3に成人の外形と骨格の概略を示すが、全身骨格は約200個の骨(bone)からできており、骨と骨は真関節(joint、可動の関節)か偽関節(不動の結合)で結合されている。筋は、表情筋などの例外を除き、骨と骨に関節を越えて結合する。体の構造を考える際には、頭頸部(head and neck)、体幹(trunk)、四肢(extremities)を区別する。頭頸部は、脳を入れる骨性の容器である頭蓋と、これと体幹を連結し、支える頸部からなる。頸部は体幹の上にあり、体幹、特に骨性のかご状の構造である胸郭(thorax)と接続する。体幹は頭頸部を除く全身から四肢を除いた部分であり、特に柔らかく虚脱しやすい肺(lung)と心臓(heart)を入れる胸部、およびそれ以下の腹部からなる。内部における胸部と腹部の境は横隔膜(diaphragm)であり、これは強力な呼吸筋でもある。体幹と頸部を構造として支えているのは文字どおり背骨-脊椎骨(vertebra,spine)であり、頸椎[cervical vertebrae(spine)]7個、胸椎[thoracic vertebrae(spine)]12個、腰椎[lumber ver-tebrae(spine)]5個、以下融合して1個となって骨盤腔の後壁を構成する仙椎5~7個からなる。ヒトでは尾が退化しているので、その下部にある尾骨は極めて小さく痕跡的である。胸椎には肋骨(rib)が付着し、胸郭を構成していることは上述したとおりである。胸腔の前壁は胸骨(sternum)であり、12対の肋骨のうちおよそ9対ないし10対は、背部において胸椎と関節で結ばれ、前胸壁においては胸骨と関節を作る。腹部では、体を重力に逆らって支える骨性の構造物は脊椎(胸椎と腰椎)のみである。腹壁前壁には強大な筋肉である腹直筋(rectus abdominis muscle)があり、側壁は外側から順次、外、内、最内と名付けられた腹斜筋群によって構成されている。背部は脊椎の周囲の多種多様な姿勢維持のための強力な筋(背筋群)で構成され、直立する体を支えている。四肢のうち上肢(upper extremity、四足歩行動物では前肢)に、最大限に自由な運動を許すために、これを体幹に結合する構造である上肢帯[pectoral girdle、鎖骨+肩甲骨(scapula)]は力学的にはそれほど重要でない胸骨と鎖骨の関節を除き、主として筋と腱によって体幹と非常に緩く結合されている。これに比べて、下肢帯(pelvic girdle)で肩甲骨に相同な腸骨は、脊椎(仙骨)と極めて強固に結合して骨盤(pelvis)を形成している。この差は、上肢と下肢(lower extremity)の運動の自由8―4生体工学度と、そこにかかる荷重(下肢帯には頭頸部および全体幹に働く重力が作用する)に関連しており、上肢の運動は下肢のそれに比べて極めて自由度が高い。上肢と下肢は本来、魚類の胸びれと腹びれに由来するもので、骨および筋の構成は、全く相同である。したがって、上腕と大腿、前腕と下腿、手と足は基本的に全く同じ構成をもっているが、ヒトは不釣合いに大きな頭をもって直立したので、もっぱら下肢が姿勢の維持と身体の推進に特化し、上肢が自由な作業のために特化せざるを得ず、運動範囲などには大きな差が生じている。頭頸部は頸と頭からなり、頭は頭蓋骨(cranium)を基礎とする構造物であるが、脳と神経を入れる脳頭蓋と顔面の諸臓器を入れる顔面頭蓋を区別する。脳は、硬膜(dura mater)、クモ膜(arach-noidea)、軟膜(pia mater)と呼ばれる三層の脳髄膜で包まれており、最も丈夫な組織である硬膜は、頭蓋の骨の骨膜と癒着して固定されている。軟膜は脳の表面に直接密着しており、クモ膜と軟膜の間はクモ膜下腔(subarachnoid space)と呼ばれる髄液[(cere-bro)spinal fluid]を満たす空間で、これが脳に対する衝撃吸収の機能をもつとされている。これらの膜構造は、脊髄管内の脊髄まで連続であり、脳の病変を診断するために腰椎を穿刺(せんし)して髄液を採取するのはこのためである。脳脊髄液は脳内にある脳室と呼ばれる空洞において分泌産生され、クモ膜下腔を循環している。その速度は、速いところではcm/sのオーダに達するが、大部分の脳脊髄腔内でははるかに緩やかである。頭蓋は多数の骨からなり、成人ではすべての骨は縫合と呼ばれる強固な結合で連結されているが、新生児では産道を通過する際に頭の変形を許す必要があるため、泉門と呼ばれる幅広い結合組織性の裂け目がある。顔面の頭蓋には上顎(upper jaw)、下顎(lowerjaw)があり、眼(eye)、鼻(nose)、口(mouth)などの構造を作る。眼は顔面正面に存在するが、構造的には脳の一部であって、視神経(optic nerve)などの神経ばかりでなく、これを灌流する血管も脳に由来する。このため、顔面の他の部分に比して、温度がやや高いことが観察されている。頭蓋底には脳神経および血管を通すための多数の孔があいている。最も大きいのは、脊髄(spinalcord)を通すための大後頭孔である。顔面には皮膚を動かすことができる表情筋が分布し、これは筋膜で囲まれていないため、感染症においては病変が拡大しやすい。頸部には1・3・4,1・3・6項で後述する呼吸器および消化器の上部の構造と、脳を灌流するための大流量の血管系を収容するほか、甲状腺(thyroid gland)および多数のリンパ節、さらに頭の位置を維持するための強い筋群が存在する。
