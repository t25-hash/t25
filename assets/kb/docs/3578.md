# 1・3・2 脳神経系

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

脳神経系は外界の情報を処理して,思考や感情などの高次の脳機能を営み,器官系の相互作用,あるいは運動器,呼吸器,循環器,消化器および泌尿生殖器と内分泌など,内臓の作用を調節する.脳神経系には中枢神経系(central nervous system),すなわち脳(brain)と脊髄,およびこれ以外の末梢神経系(peripheral nerv-ous system)を区別する.外界との情報および作用のやりとりをする神経系を体性神経系(somatic nervous system)と呼び,体内の臓器の制御に当たるものを自律神経系(autonomic nervoussystem)と呼ぶ.中枢神経は脳と脊髄からなり,いずれも骨でできた容器(頭蓋と脊髄腔)に液体に浮いた形で収納されている.脳内には脳室(ven-tricle)と呼ばれる空間を含み,脳脊髄液で満たされている.成人の脳の重量・寸法の個体差は大きく,重量でいえば1100gから1600g まで分布し,平均は男で1300g 程度,女で1250g 程度とされている(人種などによっても異なる).ヒトは動物の中で例外的に大きな脳をもっているが,絶対的な脳重量は脳機能を考えるうえで重要ではない(シロナガスクジラでは7kg).脳神経系の発生は,胎生のごく初期(18~20日)に外胚葉(ectoderm,皮膚と神経の原基)の一部が管状の構造物(神経管)となることによって構成される中枢神経系と,その周囲の細胞に由来する末梢神経に分化することで開始される.神経管から胎生4週に入り,前,中,菱の3種の脳胞が形成され,前脳胞からは終脳と間脳を経由して大脳(cerebrum)が生じ,中脳胞からは中脳(midbrain),そして菱脳胞からは一括して脳幹(brain stem)と呼ばれ,生命の基本機能を調節する橋(pons),延髄(medullaoblongata),そして運動の統合を調節する小脳(cerebellum)が形成される.脳の構造の概略を図1・4に示す.大脳は大脳葉(いわゆる大脳)とその深部にある基底核からなり,大脳葉は統合中枢として,思考と随意運動の調節などを統括する.大脳葉は前頭葉,頭頂葉,側頭葉,後頭葉に区分され,皮質(灰白質,grey matter),髄質(白質,white matter)で構成される.皮質は神経細胞(neuron)の細胞体を含み,白質は有髄の神経線維を含み,神経線維間の電気的絶縁のための脂肪が白く見える.大脳皮質の総表面積は,脳のひだ状の構造のため約2200cmあり,その1mm当たり1万~3万個の神経細胞を含むとされる.皮質には,このほかにグリア細胞(glia cell)と呼ばれる細胞群があり,ヒトでは神経細胞の10倍の個数を数える(これは他の動物に比べて圧倒的に多い).この細胞は従来,脳の構造の維持,血液脳関門など,副次的な脳機能を担っていると考えられてきたが,最近,高次の脳機能,情報処理に関係していることが明らかになりつつある.大脳ではこのほか,基底核(basal ganglia)と辺縁系(limbic system)と呼ばれる構造があり,基底核は運動の統合性の調節に当たり,辺縁系は情動,情緒に関係している.大脳皮質は機能が局在しており,例えば,運動野,知覚野,感覚野,言語野などが区別される.この結果,脳出血などで大脳の特定の部位が破壊されると,種々の類型の失語症など固有の機能障害をきたす.図1・5に,体性知覚に関する大脳皮質の領域と身体各部の関係,図1・6に運動に関するその関係を示す.間脳は視床(thalamus)と視床下部(hypothalamus)に分けられる.視床は知覚の中枢で,視床下部は生命維持に関する自律神経活動の中枢であり,その一部は脳下垂体(hypophysis)と直結し,内分泌の中枢を構成する.中脳は,下等な脊椎動物では感覚の中枢であるが,ヒトなどではこれらの機能は大脳に移動している.そのほかに,眼運動の調節などの反射に関係する脳神経の起点がある.後脳は橋と小脳からなる.小脳は,重量130~140g で左右の半球からなり,主として平衡感覚,随意運動の協調作用などを調節する.小脳はさらに,中脳,橋,延髄へと続く構造の一部で,中脳以下延髄までを脳幹と呼ぶ.脳幹(延髄)には,呼吸,循環など,生命の維持に必須の機能の中枢があり,この部位の機能停止を脳幹死と呼ぶ.また,脳幹には脳神経(cerebral nerve)12対の大部分の図1・4 脳の正中断面に見える各部分の構造図1・5体性知覚に関する大脳皮質の領域と身体各部の関係図1・6大脳皮質の運動野と身体各部の関係核がある.脊髄は脊柱管の中にあり,延髄に引き続き頭蓋底から始まり,40~50cmの長さで,第1腰椎の付近で終わる.このレベル以下の脊柱管の内部には,馬尾と呼ばれる脊髄神経が収納されている.脊髄からは脊髄神経(spinal nerve)が出る.頸髄からは頸椎7個に対し8対,胸髄から12対,胸椎から5対,仙骨から5対,尾椎から1~2対が出る.頸部および胸部の脊髄神経は,頸神経叢(そう)および腕神経叢という複雑な神経のネットワークを作って,上肢帯および上肢の筋に分布する.腰部と仙骨部の神経叢も,基本的にこれらと相同である.脳神経12対のうち,第I(嗅神経)と第II(視神経)は中枢神経の一部であるが,以下10対は末梢神経で,頭頸部から内臓(第X,迷走神経,vagal nerve)まで広く分布する.自律神経系は内臓の大部分を支配する.図1・7に概略を示すように,交感神経系(sympathetic nervous system)と副交感神経系(parasympathetic nervous system)の2系統があり,交感神経系は,心身がストレス状態になったときに優位になる.これは,野生の動物では捕食者を認めたときに相当し,逃走するか闘うかの態勢を準備するため,心拍数増加,気管支拡張,筋血流の増加,皮膚血流の減少などが起こる.交感神経系は,第1胸髄から第4腰髄までの脊髄神経から出て,脊椎の両側に神経幹を作って全身に分布する.交感神経の末端からはノルアドレナリン(noradrenaline)が分泌される.副交感神経は交感神経と逆の作用をもち,脳神経の8―5図1・7
