# 9・3・2 表面ロール加工（rolling)

> **章**: 第９章 表面処理
> **節**: 9・3 疲労強度向上表面処理

適当な荷重でローラなどを部材に押付け，部材表面層に圧縮塑性
変形を生じさせると，その際の加工硬化とひずみ時効，および表面
層に形成される圧縮残留応力によって疲労強度が向上する．
部材全体に圧延，プレス，引張りなどで加工を与えた場合にも加
工硬化とひずみ時効によって疲労強度の向上が期待できるが，形成
される残留応力が一般に小さく，表面層にのみ加工を与える場合に
比べて疲労強度向上効果は小さい．
表面ロール加工は車軸等の圧入軸（press -fitted shaft ）のフレッ
チング疲労（fretting fatigue）強度向上に古くから適用されてお
り ，近年ではクランクのフィレット部の疲労強度向上にも利用
されている ．ごく最近では12Cr鋼製の発電ロータに低合金鋼を
肉盛溶接した軸受部に圧縮残留応力を付与する目的でロール加工が
適用されている ．
表面ロール加工を施すと平滑材では5～30 %程度の疲労強度上昇
が見込めるが，図9･15に示すように段付棒のフィレット部やV溝
