# 1・3・4 呼吸器系(respiratory system)

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

われわれ動物は、酸素を含む外気を吸入して体内で合成される二酸化炭素を排出するために呼吸を営む。呼吸は、外気を吸入して、肺において血液との間でガス交換(gas exchange)を行う外呼吸(external respiration)のプロセスと、このようにして酸素化された血液が、体内の組織において組織を構成する細胞との間でガス交換する内呼吸(internal respiration)のプロセスに分けて考えることができる。なお、これらすべての過程を外呼吸と呼び、これに対して、この酸素を用いて細胞内で炭水化物を酸化し、二酸化炭素と水を排出する代謝過程を細胞呼吸(cellular respiration)と呼ぶこともある。ここでいう外呼吸のプロセスは、肺へ空気を輸送する換気(ven-tilation)と、肺の末梢に存在する肺胞においてガスが血液と交換される過程からなる。換気は、鼻(nose)、喉頭(larynx)、気管(trachea)、気管支(bronchus)、細気管支(bronchiole)と枝分かれする気道システムにおける往復する気流の作用である。鳥類では、気道の構造が哺乳類などと異なり、一方向流れになっている。鼻は、ヒトでは極めて単純な構造になっていて、鼻甲介(con-cha)と呼ばれるひだが上中下の3枚あるだけだが、寒冷地に生息する海棲哺乳類(アザラシなど)および乾燥に耐えるラクダなどでは、極めて複雑な迷路になっており、強力な対向流熱交換器として、それぞれ熱および水のトラップとして機能している。鼻腔の上部には嗅覚の受容器が存在しているが、これはほとんど、脳そのものから出る短い突起である嗅神経に直結している。気道は、喉頭の上部で食道(oesophagus)と交差してその前方に出て喉頭を通り、気管を通じて胸腔の中央に至り、左右の主気管支に分岐する。喉頭は、食物が咽頭(pharynx)から食道へと輸送される際に、気道の入口を閉鎖する喉頭蓋に始まる軟骨性の構造物で、特に、発図1・8 脳下垂体と視床下部の関係声のための構造をもつ。発声は喉頭に対で存在する声帯と周辺の空気の連成振動にその源があり、咽頭・喉頭腔、口腔、鼻腔、副鼻腔などの頸部から頭部の腔所における共鳴で修飾されて、われわれが聞く声となる。心臓が胸腔中央やや左側に位置しているため、肺は右葉が左葉に比べやや大きく、上、中、下の三葉からなり、左葉では上下の二葉からなる。主気管支もこれに応じて、右では3分岐、左では2分岐し、各肺葉に分布する。また、このため右の主気管支の気管に対する分岐角度は小さく、気管に誤嚥(ごえん)された異物は右に落ちることが多い。気管支は分岐を繰返し、有名なワイベル(Weibel)のモデルによれば、気管を0世代、これから分岐する左右の主気管支を第1世代として、平均23世代の分岐で肺胞(alveolus)に至るとされている。0~16世代は導管部と呼ばれ、第16世代の分岐が終末細気管支である。その先第23世代までは気管壁が肺胞壁と接するので、呼吸細気管支または肺胞管と呼ばれる。第23世代で肺胞という囊状の構造で終わる。気管支が分岐を繰返すたびに総断面積が増すので、第17世代(直径0.5mm)で気流のレイノルズ数が1になり、それ以後の細気管支内のガスは対流ではなく拡散で肺胞に移行すると長く信じられていたが、最近では、そのような見方は単純にすぎ、肺胞においても対流によるガス交換が行われる可能性があるとされている。肺胞壁には肺の毛細血管が分布しているが、これもファン(Fung)一派の研究により、管状の毛細管の網が肺胞を取り囲むという形ではなく、肺胞壁が毛細血管壁そのものであり、図1・9に示すように、肺胞の毛細管は、平板間にスペーサが立っている平行平板流路のようなものであることが明らかにされている。なお、ガスのうちでも、酸素は主として肺胞壁を通じて、二酸化炭素は細気管支の壁を通じても拡散・交換されている。ヒトの呼吸は、胸郭の容積を拡大、縮小することで行われている。胸郭の内壁と肺の外壁の間は、胸膜腔と呼ばれる少量の漿液を満たす空間で、胸郭の拡大によって生じる陰圧が肺を吸引して拡大させる。したがって、この胸膜腔と肺内、あるいは外気が交通するようになると、肺は拡張することができなくなり収縮したままになる。これを気胸(pneumothorax)という。この吸気の過程で最も重要な役割を果たす構造は横隔膜であって、その収縮によって吸気の大部分が行われる。呼気は、肺の弾性と肋間筋の収縮による胸郭の縮小によるもので、横隔膜による吸気よりはるかに発生する力が弱い。したがって、呼吸に関する障害は吸気時より呼気時に起こりやすい。その典型例は、喘息(asthma)における喘鳴(stridor)である。呼吸運動は随意運動であるが、延髄に中枢があり、不随意にも呼吸運動が保たれる。
