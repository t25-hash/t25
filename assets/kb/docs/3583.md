# 1・3・7 腎・泌尿器系

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

動物は,体内環境(milieu interieur)と体外環境を皮膚その他の障壁で区分し,体内環境を一定の状態に維持する機能をもっており,これをホメオスタシス(homeostasis)という.特に陸上に棲息する動物では,体内に保持する水分(体重の60%)の量を保ち,体液の電解質の組成などを維持することは生存に欠かせない.こうした調節の結果として排出される電解質,なかでもタンパク質代謝図1・13 ヒトの消化器の概観の過程で発生する窒素代謝産物(尿素,urea)などの不要な物質を血液から排出するための器官系を泌尿器系(urinary system)という.泌尿器系の一部は生殖器(genital organ)と重複しており,下等な哺乳類以下の脊椎動物では,泌尿器,生殖器,さらに直腸までが共通の腔に開口する総排泄腔をもつ.泌尿器には,血液から不要物をろ過して濃縮し尿を生成する腎臓(kidney),尿を外部に排泄する導管と貯蔵部である尿管(ureter),膀胱(bladder),尿道(urethra)を区別する.腎臓は,左右の後腹膜腔に1個ずつ存在する臓器で,右は肝臓のためにやや低めについている.ヒトではそれぞれ150g 程度の臓器であるが,極めて血流に富み,安静時の心拍出量の約25%が腎臓を灌流する.腎臓は,図1・14に示すように腎単位(nephron,ネフロン)と呼ばれる構造単位の集合体である.ネフロンは腎小体(glomerulus,renal corpuscle)と尿細管(renal tubule)からなり,腎小体では,血液から血漿中の電解質液がボーマン囊にろ過され,これが尿細管において再吸収される.残った成分が腎盂(じんう,renal pelvis)で集められ,尿管で膀胱へ排出される.ボーマン囊にろ過される原尿は1日170L作られるが,尿として体外に排出されるのは1.5Lのみである.腎臓は,自己の血流を制御対象として血液電解質の組成をコントロールし,血圧を調節する内分泌器官でもある.この系をレニン-アンギオテンシン系と呼び,高血圧症(hypertension)に関連して重大な意義をもつ.このほか,腎臓は造血機能(hematopoiesis)を促進するエリスロポエチン(erythropoietin)も分泌し,赤血球の量を調整することで血液の組成をコントロールしている.尿管を通って膀胱に送られた尿がある程度の量になると,尿意をもよおし,排尿が起こる.膀胱壁とその出口の括約筋は平滑筋であり,骨盤腔内に分布する自律神経の支配を受けている.ここは,腹腔内の手術などの影響で機能が障害されやすく,排尿障害(micturition disturbance)が起こる.ネフロン(nephron)は1個の腎臓に約100万個存在するが,損傷しても再生しないため,慢性化する腎臓の病気は遅かれ早かれ腎不全(renal failure)を起こし,人工腎臓(artificial kidney)による人工透析(artificial dialysis)が必要となる.人工腎臓は,特にカリウムなどの電解質の調節,尿素などの老廃物の除去には有効な手段であるが,当然のことながら上述した内分泌機能をもたないので,長期の人工透析ではこのことが問題となる.
