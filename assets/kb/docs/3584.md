b.振動環境（vibration environment ) 現場における測定の，
今ひとつの外乱要因が振動である．最初から測定用の部屋が用意で
きる場合はまれで，ほとんどの場合，すでにある部屋を用いるケー
スとなる．特に機械加工工場では，周辺に多くの振動発生源となる
工作機械が存在する．そしてこれらの機械類のレイアウトは，時と
して変更されるのが常である．このような場合，部屋全体の防振が
できず，必然的に個々の測定機器の設置方法で対策を行うこととな
る．そのための防振台，除振台が多く開発，市販されている．最も
効果の高い形式が空気式のアクティブサーボ型（図10･12）である
が，ほかに磁気式，ばね式，粘弾性ゴム，液体式などがある．基本
原理は低いばね定数の緩衝材の上に測定機器を置き，緩衝材の固有
振動数より高い外部振動の周波数成分を減衰させるものであるが，
アクティブ型では積極的に相対振動を抑えるようサーボ減衰させる
ため，広い周波数範囲で有効となる機能をもっている ．
あらかじめ設置環境の振動成分の分析結果があれば，少ない費用
で高い効果を得ることも可能である．また測定装置と，それを用い
る知識があれば自らの工夫により防振が達成できる．例えば，振動
を嫌う表面粗さ計を設置する際，鉄製のテーブル上に設置するより
木製のテーブルに置いたほうが結果が良い，ということも起こる．

れ，油などの影響も大きい．研削加工において用いられるインプロ

動作不良を起こすことがある．インプロセスゲージでなくとも工作
機械の横で使用される現場測定器類（例えばマイクロメータ，ダイ
ヤルゲージなど）は切りくず，じんあいなどの影響を受けやすいの
で，保管，管理に十分注意を払う必要がある．また，最近普及した
電子式マイクロメータ類では，水分が電池室に入り込むことによる
トラブルもある．静電式のスケールでは，近くに帯電したプラスチ
ック類をもってくると指示値が変化することもある．非接触変位セ
ンサを用いる場合，特に光学式センサの場合には測定表面の清浄性
が大きく測定結果に影響する．そのために接触式センサによる測定
結果のほうが信頼性が高い場合も多い．他方，軟らかい素材の測定
では測定力の影響による変形が誤差の大きな要因となる．プラスチ
ックなどはその代表である．
d.その他 現場における測定技術には，検査のためと解析の
ためとがある．両者で測定行為はほぼ同じであるが，結果の処理方
法は根本的に異なる．検査のための測定では，あらかじめ定められ
た許容値（tolerance）の範囲内に測定結果が入っているかどうか
が問われる．場合によっては時系列的な測定値の変化が要求される
こともある．他方，解析のための計測では，より多面的な測定結果
が要求される．単なる1点の測定結果のみならず，その周辺の傾向
も測定値として要求される．例えばシャフトの径の測定を考えてみ
る．検査では設定された数点の測定値より合否が決定される．解析
ではテーパの度合い，母線真直度，時にはいくつかの断面における
真円度も要求され，これらの結果を総合して加工工程，素材，加工
条件などの判定を行う．したがって，絶対値がうんぬんされるより
傾向を問題とするケースが多い．規格に定められた測定方法は検査
が目的であり，解析のためのものではないことを留意する必要があ
る．
