# 1・3・8 生殖器

> **章**: 第1章 生体の構造と機能
> **節**: 1・3 人体の解剖と機能

男女の生殖器は、発生上同一の原基が性ホルモンの作用によりそれぞれの方向へ分化するもので、特に胎生初期にスイッチを入れる副腎由来の男性ホルモンがなければ、個体は原則として女性になることが知られている。生殖器は、半数分裂(meiosis)により染色図1・14 ネフロン(腎単位)の模式図8―11体を体細胞の半分だけもつ卵子(egg)と精子(sperma)の生殖細胞を作り出し、これを融合させてできる胚(germ)を成熟させる臓器で、このために必要なホルモンを分泌する内分泌臓器でもある。生殖器は内生殖器と外生殖器とに区別される。男女それぞれで、生殖細胞を作る精巣(testis)と卵巣(ovarium)、輸精管(defer-ent duct)と輸卵管(ovarian duct)、前立腺(prostate)と子宮(uterus)、精管(deferent duct)と膣(vagina)までを内生殖器と呼び、陰茎(penis)と陰唇(labium)周辺の外陰部を外生殖器と呼ぶ。精巣は陰囊に包まれて体外にあり、他の臓器に比して低温に保たれる。精巣内部には精細管(seminiferous tubule)と呼ばれる細管組織があり、その上皮が精子を形成する。精巣は男性ホルモンであるテストステロン(testosterone)を分泌するが、これは、視床下部-下垂体系の調節を受け、精子形成と外生殖器の成長を促す。また、全身の物質代謝、特に筋の発達に影響するため、合成のテストステロン剤はドーピングに利用される。精子は長い鞭毛(50~60m)をもち運動する細胞で、膣内に射精された精子は、約1~3時間で卵管膨大部(ampulla)に自力で到達し、卵子と遭遇すれば受精する。これに比較すると女性の生殖器はやや複雑である。卵巣は腹腔内に1対存在し、卵子を作り出すとともに性ホルモンを分泌し、性周期(sexual cycle)を調節する。精子に比較すると卵子は極めて個数が少なく、生殖可能年齢を通じて月経周期ごとに平均1個のみ成熟するので、一生で成熟する卵の個数はたかだか数百個である。成熟した卵が卵巣から排卵されると、子宮から伸びる卵管の先端である卵管采(らんかんさい、fimbriae)がこれを察知して移動して取込み、卵管の蠕動により子宮に近い卵管膨大部に運ばれて受精(fertilization)が起こる。排卵後の卵巣には黄体(corpusluteum)という組織が形成され、黄体ホルモンを分泌する。受精した胚は黄体を刺激しプロゲストロンを分泌させ、妊娠(preg-nancy)の初期において妊娠を継続するための刺激を子宮(uterus)に与える。一方、卵が受精しないと、黄体は約2週間後に崩壊し血中プロゲストロン濃度が低下するために、それまで肥厚してきた子宮内膜が崩壊し、出血とともに排出される。これを月経(menstruation)という。卵管膨大部において受精した卵は、分裂を繰返しながら卵管内を輸送され、細胞数が16個となるころ(3~4日後)に子宮腔に到達する。この輸送には、卵管内の液体の流れが関与する。胚は分裂を続けて、次に桑実胚となり、2日後に胚盤胞(balstocyst)が形成され、子宮内膜に埋没し着床する。胚盤胞の外側の細胞は胎盤(placenta)となり、内部の細胞が胚(embryo)となる。胎盤は、妊娠中を通じて、母胎と胎児の間のガス交換と栄養補給を行う。胎盤はまた各種のホルモンを分泌し、母体における妊娠を維持させる。胎盤と胎児は臍帯(さいたい、umbilical cord)で結ばれ、臍帯には不対の静脈(胎盤から酸素化された血液を運ぶ)と1対の動脈(胎児から胎盤へ血液を運ぶ)を収容し、妊娠末期には、太さ1.5cm、長さ1mに達する。ヒトでは、最終月経第1日から平均40週(280日)±2週で胎児は成熟し、出産が起こる。これらのすべての過程で、子宮内に形成されるすべての組織・構造は胎児に由来し、妊娠から分娩までのタイミングもすべて胎児側によって制御されている。出産時には子宮が収縮し(陣痛)、胎児は産道を通って娩出される。胎児(fetus)は、娩出されて臍帯が収縮ないし結紮(けっさつ)されるために、血中の二酸化炭素濃度が高まると呼吸中枢が刺激されて産声を伴う呼吸を開始する。このとき、初めて肺には空気が入り、また、胎盤-臍帯静脈-大静脈-心臓という胎生循環(fetal circulation)が、新生児の肺循環と体循環に瞬時にスイッチされる。出産後の子宮は急速に収縮し、胎盤を娩出して出産の過程を終わる。これに伴い妊娠中に拡大していた子宮の動脈、静脈も急速に縮小する。8―12生体工学
