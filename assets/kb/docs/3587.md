# 10・2・3 幾何偏差（geometrical deviation）の測定

> **章**: 第10章 加工測定
> **節**: 10・2 加工部品の測定

a.一般事項 加工物は，図面上の理想的な幾何形状である図
示形状をもとに加工されるが，加工中の工作機械の変動などにより
加工物の実形体には形の偏差が生じる．加工物は，長さ測定機や座
標測定機によって寸法が測定され，座標測定機や形状測定機などに
より形状が測定される．形状測定では，幾何偏差および表面性状が
測定される．表面性状の測定については10･2･4項に示す．
b.幾何偏差の定義 加工物の実形体には形の偏差が生じる．
このような偏差を幾何偏差（JIS B0621：1984「幾何偏差の定義
及び表示」）と呼ぶ．幾何偏差は，部品として組立てられるとき，
機能，互換性の点で問題となる．幾何偏差は，表10･3のように形
状偏差，姿勢偏差，位置偏差，振れに分類される．機械部品を形成
する線，面等，単独形体の偏差は形状偏差で表す．形状偏差には真
直度，平面度，真円度，円筒度，線の輪郭度，面の輪郭度がある．
このうち，線および面の輪郭度は単独形体のほか関連形体として，
基準とするデータム（JIS B0022：1984「幾何公差のためのデー
タム」）に関連して規制される場合もある．データム（datum）に
関連して規制される関連形体の姿勢偏差には平行度，直角度，傾斜
度がある．関連形体に適用する位置偏差には位置度，同軸度および
同心度，対称度などがある．回転したときの軸に対する変位を表す
振れには，円周振れと全振れがある．

c .幾何偏差の測定 幾何偏差のうち，代表的なものである真
円度（deviation from roundness ，deviation from circularity）お
よび円筒度，真直度および平面度の測定方法について示す．
真円度および円筒度の測定では，図10･28に示すように被測定物
を回転テーブルによって回転し，または測微器を回転軸によって回
転し，被測定物の半径方向の出入りを測微器でたどる．そのデータ
はディジタル化され，取付けの偏心の影響を除いた真円度チャート
を表示し，最小領域法（minimum zone method）により真円度を
計算することができる．被測定物の使途によっては，最小二乗法，
最小外接円とそれと同心の内接円との間の半径差である内接円中心
法も同様に計算される（図10･29）．
円筒度は真円度の測定を軸方向に正しく送って数断面で測定して
得た真円度重畳の記録図形から，最小領域法によって求める．この
方法は測定前に被測定物の軸心と測定器の垂直軸との平行出しが必
要である．それらを測定できる範囲内に調整した後，得られた測定
値から計算で軸線を求め，軸線の傾きを補正した後，円筒度を求め
ることも行われる．真円度，母線の真直度，相対する母線の平行度
を円筒度に代わって用いることも多い．直径法や三点法によって円
筒度を測定することもある．
真直度および平面度の測定では，基準として用いられる直線また
は平面に対して，測定すべき形体の偏差を変位測定して表す方法
と，角度測定によって得た値を変位に換算して表す方法とがある．
直線を具体化した基準形状に，直定規，テストバー，強く張った鋼
線，光線などを用い，基準の平面には定盤，オプティカルフラット
などを用いる．
d.形状測定器（contour measuring instrument ) 幾何偏差
を測定する形状測定器としては，10･2･4項に示すような触針式輪
郭測定機がよく使われている．これは，表面粗さ測定機と同等の構
造をもっている．また，真円度および円筒度に対しては，専用の真円
度測定機が作られている．図10･30は真円度測定機の一例である．
