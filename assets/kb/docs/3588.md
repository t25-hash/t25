c .表面性状の測定 表面性状の測定には種々の方法が利用さ
れているが，表面性状の定義に従って加工面の断面曲線を測定し，
それから各パラメータを求める方法が多く使われている．このよう
な方法では，おもに触針法により触針式表面粗さ測定機が使用され
ている．このほか光触針や光切断を用いた光学的方法，AFM（原
子間力顕微鏡）による方法なども使用されている．また，パラメー
タと関係のある現象（反射率，光散乱など）を利用して直接パラメ
ータを求める方法もある．

ments ) JIS B0651：2001「製品の幾何特性仕様（GPS)―表

粗さおよびうねりを測定するための触針式表面粗さ測定機の一般的
な構造について規定されている．図10･38は，触針式表面粗さ測定
機の構成例である．測定機の測定ループの中に入っている各要素
（触針，プローブ，送り装置）が測定結果に影響を及ぼす．触針は

m，5mまたは10mでテーパ角度が60°または90°である．触

準線）によって正確な軌跡に沿って案内される．プローブは，触針
