# 10・3・2 切削加工における計測技術

> **章**: 第10章 加工測定
> **節**: 10・3 インプロセス・オンライン計測

機械加工の出力である工作物が最終的に工具を介して制御され
る．そのときの工具-工作物間の相対的位置関係が加工精度として
直接工作物に転写されることから，工具と工作物についての計測技
術の開発が積極的に行われてきた ．
a.工具系 加工の進行とともに工具の刃先形状は摩耗によっ
て変化し，工作物の寸法や表面粗さ等に影響を及ぼす．また工具の
破損や欠損は多くの場合突発的に生じ，その検出は可能であっても
予測は極めて難しい．損耗の計測法を分類すると図10･48のように
まとめられる．これらには，信号処理の工夫によって破損や欠損の
検出が可能な方式もいくつかある．
直接的検出法としては，工具の摩耗面に光を照射し，摩耗量と反
射率や反射パターンを関係づけて検出する光学式が従来から研究さ
れてきた ．しかし，光を用いる方法が環境に弱いことから，必
ずしも実用化レベルでは用いられていない．
これに代わり摩耗に伴って変化する切削抵抗，切削動力，切削温
度など切削プロセスの変化と摩耗量を関係づけた間接的検出法が従
来から提案されている ．なかでも複数のセンサを用いるセンサフ
ュージョン法が，切削条件の変更や被削材種に対し，ロバスト性の
ある検出法として興味深い ．その例を図10･49に示す ．AE
（acoustic emission）センサを工具ホルダの後部に取付け，さらに

偏差/平均値）の処理を行い，決定過程としてのニューラルネット

くず絡みつき発生の検出も含む5ユニットとし，隠れ層を10とし

表10･6に寸法および形状計測法を示す．工作機械の幾何学的誤差

d.工作機械系 軸受部，回転部などの過負荷や潤滑不良によ
る異常検出，駆動部分の異常動作，欠陥検出，締結部の異常状態の
検出，マシン全体の異常温度や振動の検出など，工作機械の異常状
