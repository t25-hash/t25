# 2・2・2 筋骨格系(musculoskeletal system)

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・29 に模式的に示したように、AFM の探針の先端を細胞表面に押

人体の構造維持・運動機能の主体となる筋骨格系は、多大な負荷を受けることが多く、その機能解明のためには、分子、細胞、組織、器官レベルに応じたバイオメカニクスの適用が必要とされる。また、人体質量のうち約40%を筋が、約7%を骨が占めており、筋骨格系として合わせると約半分を占めることになり、身体の形態や全身の諸機能に強い影響をもつものと考えられる。本項では、骨、関節軟骨と潤滑、関節靱帯と腱、筋肉に関する主要事項を記述する。なお、筋-骨格系の関節運動については、2・4節において各構造を考慮した運動学・動力学の視点から詳細な説明がなされているので参照願いたい。a. 骨(bone)骨格系は、新生児では約350個から構成されているが、成長に伴い融合し、形態や寸法を変化させ、成人(女性では15~16歳以降、男性では18歳ごろ以降)になると206余(尾骨の数や、膝・足周囲の骨に個人差がある)の骨から構成される骨格系として完成される。それらの形態やサイズ(1・3節参照)は多様であり、大腿骨のような大きい骨から、数mmサイズの耳小骨まで種々のサイズがあり、形態からは、肩甲骨・頭蓋骨・肋骨・骨盤などの扁平骨と、脊椎の椎体・手根骨・踵骨(しょうこつ)など厚みのある立方骨、大腿骨・脛骨・上腕骨のような管状の長管骨に大別され、それらの役割・機能に対応した構造を有している。骨は、主要な荷重支持組織としての構造物的役割を果たすとともに、骨髄は造血機能を有し、骨組織はカルシウムの貯蔵庫(人体のカルシウムの99%を含有)として、全身的鉱質代謝に関しても重要な役割を演じている。なお、骨自体は自らは動けないために、骨間は軟骨を介して靱帯などで連結され、腱を介して骨に付着した筋肉により荷重支持と運動を可能にしており、筋骨格系として総称される。このように、各骨は身体の支持や各種の運動、軟組織や他の器官の保護などの機能に応じて多様な形態と構造を有しており、それらは、力学的環境に適応した形態や強度・剛性を獲得しうる生体組織であり、最適構造や機能的適応という視点から骨の構造特性が探求されている。力学的刺激との関連では、長期臥床による骨萎縮は周知の現象である。また、長期の宇宙滞在において微小重力場にさらされると骨萎縮やカルシウムの流出が観測されたが、運動負荷を加えることにより骨濃度の低減がかなり抑制されることも判明した。また、骨は力学的環境に適応してその形態を変えうる。このような負荷条件の変化に応じて骨が再構築(remodeling、リモデリング)する現象は、ウォルフ(Wolff)の法則と称されており、多様な骨形態は応力やひずみの均等化に起因することなどが検証されたが、リモデリングの詳細な機構については不明な点が多い。しかるに、最近では細胞レベルの応答機構や流動電位・圧電効果の役割などの視点から、徐々に解明されつつある。また、骨折後の治癒として周知のように、骨組織は優れた自己修復機能を有している。すなわち、骨折後に適切な固定がなされると出血の結果として血腫が形成され、肉芽組織を経て仮骨が形成され、骨新生と骨吸収を繰返しながら自然治癒が達成される。なお、仮骨内の細胞は圧縮応力下では骨細胞化するが、引張応力下では線維組織化する傾向があるので、健全な修復を図るためには力学的条件の正確な把握が必要とされる。後述するように、骨組織におけるリモデリング過程では、力学的・生化学的環境の変化に応じて、骨系細胞が反応を生じ骨吸収や骨形成を制御しながら、骨形態・寸法の変化や骨密度の変化をもたらしている。したがって、リモデリング機構を解明するためには、細胞、骨組織、骨格系を対象にしたバイオメカニクスの集約化が必要とされる。骨は、長管骨骨幹円筒部などを構成する骨密度の高い皮質骨[cortical bone、あるいは緻密骨(compact bone)]と、骨端部などのはり構造をなす海綿骨[cancellous bone、あるいは柱状骨(trabecular bone)]に大別される(図2・37,2・38)。海綿骨は、長管骨骨端部のほか、脊椎椎体、踵骨などに含まれる海綿状の骨組織であり、骨梁構造による軽量化と構造強度の両立(最小材料で最大強度)を実現するとともに衝撃吸収的機能も有している。例えば、大腿骨頭部(図2・39)では、その直交する骨梁群は、骨頭関節面に作用する圧縮荷重に対応する主応力線と曲げ荷重に対応する引張りの主応力線に相当しており、軽量化のための合理的な構造を有している。また、健常な脊椎椎体[図2・40(a)]では、応力条件に対応した骨梁構造により体重の10倍程度の圧縮荷重に耐えうる。椎体の主要部である海綿骨は、皮質骨よりも代謝速度が5~8倍速く、1年間に全体の約20%が新生骨に置換される。した8―49図2・37
