# 2・2・3 脳神経系

> **章**: 第2章 バイオメカニクスの基礎

神経系の基本構成要素をニューロン(neuron)と呼ぶ.ニューロンは1個の細胞ではあるが,その構成要素が特殊に分化していて,入力部位としての樹状突起,細胞体と,出力部位としての軸索というように細胞が極性をもつ特徴がある.軸索はさらに次のニューロンの樹状突起などと神経接合(synapse)を形成する.神経系にはニューロン以外に,神経膠細胞(glia cell)や血管系の細胞が存在している.これらはニューロンがその機能を果たすための支持組織として働いている.中枢神経系の神経膠細胞には星状膠細胞,稀突起膠細胞,小膠細胞の3種類がある.神経系は脳と脊髄からなる中枢神経系と,そこに入出力する末梢神経系に分類される.さらに中枢神経系は白質と灰白質に分けられ,灰白質は神経細胞体がまとまって存在する部位で,情報の受渡しと処理が行われる場所であり,白質は軸索が集まっている部位で,神経情報の伝導路である.脳と脊髄は,内側より軟膜,クモ膜,硬膜(総称して髄膜と呼ばれる)の3枚の線維結合組織性膜構造物で取り囲まれ,外界から保護されている.a. 神経細胞情報科学の立場から,神経系の情報処理モデルについての研究が進められている.神経細胞は細胞体を中心にしてみると多入力-1出力の素子であり,出力の軸索がさらに多数に分岐して次の神経細胞とシナプス結合する.神経細胞での情報処理の本質をどのようにモデル化するかというとき,どの性質がいちばん重要かと考えることでいろいろなモデルが提案されている.マックロッホ(MaCulloch)とピッツ(Pitts)は,ニューロンのしきい値作用と空間加重がその本質であると考えて,ニューロンでの時刻における状態욡は욡=∑욡욢욢-욡(2・129)によって決まるとした.ここで,욡욢は神経とのシナプス間の結合係数であり,興奮性シナプスで욡욢>0,抑制性シナプスで욡욢<0である.すなわち,このモデルでのニューロンはしきい値論理素子である.マックロッホとピッツは,どのような論理回路もこの式の動作特性をもつ素子(人工ニューロン)によって構成できることを示した.b. 末梢神経(peripheral nerve)末梢神経は,周囲に厚い結合織性被膜(perineurium)である神経外膜(epineurium)があり,この中に最小1個から数個,あるいは多数の大小さまざまな神経束を含有している.これら各神経束(funiculus)の膜は厚い結合織性被膜からなり,束内の神経線維を被覆している.各神経線維間には縦走する膠原線維があり,神経線維間を結合している(神経内鞘,endoneurium).神経線維の軸索はランビエ絞輪を除いて髄鞘で被覆されている.一般的に,末梢神経はけん(牽)引に対して弾性度が極めて高い.図2・77 軟膜と脊髄スティフネス8―60生体工学しかし,末梢神経は完全な弾性体ではない.末梢神経のけん引試験ではクリープ現象が観察され,材料力学的には一種の粘弾性体であると考えられる.一般に,末梢神経けん引試験は大別して生体内と生体外の実験に分けられる.前者は組織学的に生体反応を検討しうる利点があるが,神経のみに加わる張力および神経の伸びを計測することは困難である.一方,後者は神経自体の張力および神経の伸びの測定は正確となる利点を有するが,生体反応を検索し得ない.イエウサギ脛骨神経を用いたin vivo 実験では,二峰性の応力-ひずみ曲線が得られ,in vitro 実験で得られた曲線とは異なる.第1峰は神経外組織,特に神経周囲結合組織の神経固定力によるものであり,また,第2峰はin vitro 実験のものと一致し,神経外膜および神経周膜の断裂によるものと考えられる.サンダーランドは,引張試験器を用いて,死後12時間以内に採取したヒトの主要末梢神経における弾性限界値,および機械的破断値を計測した(表2・9).組織学的変化は,ヒトの神経を75mm/minの速度でけん引したとき,ひずみが20%もしくはそれ以下で発生する.イエウサギ脛骨神経では,0.5mm/minの速度でけん引した場合,その変化はひずみが69.3%で生じる.つまり,ゆっくりけん引すると末梢神経に対する損傷が低くなる.また,末梢神経の残留ひずみは,ヒトの坐骨神経で3~4%である.このことは,末梢神経はけん引に対し,最大張力に達する直前までほぼ弾性体とみなされ,神経の機能的保護には適していると考えられる.
