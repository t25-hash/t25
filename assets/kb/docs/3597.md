# 2・3・1 水棲動物の遊泳

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・3 動物の遊泳と飛翔

水棲動物の推進方法は，生物の大きさや形態とともに棲息環境と
生活様式とにより多様である．また密度が水に近い生物は，空中を
飛ぶ生物と異なり自重を支える必要はないので，前進するための推
力を発生させるさまざまな方法が使われている．流体に関わる動物
の運動は，体の大きさ と推進する速度 で特徴付けられるレイ
ノルズ数（Reynolds number） （＝ /ν，νは水の動粘度）
で分類できる．バクテリアなどの原核生物では10，精子などの
真核生物で10，ゾウリムシで10，マスで10，マグロで10，
クジラで10と広範なレイノルズ数領域にわたっている．
低レイノルズ数領域で運動する微生物は，粘性力が大きく働くた
め，粘性による抵抗力を利用した推進を行うが，その他の水棲動物
は，ほとんど慣性力を利用した推進を行っている．それは，魚は自
らの体の運動によってまわりの流体に運動量を与えると，その反作
用で，魚自身が流体から力を受け推進するというものである．
慣性力を利用する推進方法の例として，イカやタコやホタテガイ
など体内に蓄えた水を噴出するジェット推進，アシカやアザラシや
ビーバーなどの水かきや，エビの遊泳脚などで水を後方に押したと
きの反力で推進するパドリング，多くの魚やイルカやクジラなど体

水棲動物が体をくねらせて推進する直進運動を取上げる．
