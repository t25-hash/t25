# 2・3・1 水棲動物の遊泳

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・3 動物の遊泳と飛翔

水棲動物の推進方法は、生物の大きさや形態とともに棲息環境と生活様式とにより多様である。また密度が水に近い生物は、空中を飛ぶ生物と異なり自重を支える必要はないので、前進するための推力を発生させるさまざまな方法が使われている。流体に関わる動物の運動は、体の大きさと推進する速度で特徴付けられるレイノルズ数(Reynolds number)(=/ν,νは水の動粘度)で分類できる。バクテリアなどの原核生物では10、精子などの真核生物で10、ゾウリムシで10、マスで10、マグロで10、クジラで10と広範なレイノルズ数領域にわたっている。低レイノルズ数領域で運動する微生物は、粘性力が大きく働くため、粘性による抵抗力を利用した推進を行うが、その他の水棲動物は、ほとんど慣性力を利用した推進を行っている。それは、魚は自らの体の運動によってまわりの流体に運動量を与えると、その反作用で、魚自身が流体から力を受け推進するというものである。慣性力を利用する推進方法の例として、イカやタコやホタテガイなど体内に蓄えた水を噴出するジェット推進、アシカやアザラシやビーバーなどの水かきや、エビの遊泳脚などで水を後方に押したときの反力で推進するパドリング、多くの魚やイルカやクジラなど体8―75を波打たせ、その波動の振幅を後方に向かって増大させて送るくねり推進、および尾びれを大きく振らせて推進する振動翼推進、そのほか、カメやペンギンなど両翼を羽ばたかせての推進や滑空が挙げられる。水棲動物は、直進、旋回、停止などの異なる運動を絶えず行っている。これらの運動は体自体を用いるほか、胸びれや尾びれなどの各種ひれを用いてなされる。また単細胞生物から脊椎動物まで多くの水棲動物は、推進力を得るための運動形態として、体をくねらせ後方に波を送るくねり運動を行っている。本項では、最も基本的な水棲動物が体をくねらせて推進する直進運動を取上げる。ところで、水棲動物の推進メカニズムを考える際には、水棲動物は自分の体を動かして変形させながら推進するので、船舶のように、抵抗となる船体部分と推力を発生する推進器を分離して考えられないことに注意する必要がある。魚が一定速度で泳いでいるときには、魚の発生する推力と魚に働く抵抗力とは釣合っているが、そのおのおのの値を求めることは困難である。以下に、細長い体のくねり運動による推進、鞭毛の粘性による抵抗力推進、そして魚の慣性力を利用した推進法の概略を述べる。a. 細長い体のくねり運動による推進まず、細長い物体がくねり運動をするときに発生する推進力のメカニズムについて、図その波形を時間の間に位相(位相速度)が後方へ욣移動し、体は進行方向に前進したとすると、体軸方向の要素dを表す点Pは点Qに移動することになる。この体の動きに対する単位長さ当たりの抵抗力は、体軸の接線方向および法線方向にそれぞれ,として働き、それらの合力の進行方向成分が、その体の要素が受ける単位長さ当たりの推進力となる。軸に対する体軸のこう配が点Pと逆になる点P′においても、同様に考えられる。両者の点において軸方向の抵抗力成分は同じ向きとなり、体全体の推進力に寄与する。b. 魚の推進魚の泳法としては、体の長さと変形の程度によってウナギ型(anguilliform)泳法、アジ型(carangiform)泳法、そしてハコフグ型(ostraciiform)泳法とに大別される。(b) 頭部固定(a) 自由遊泳ウナギ型泳法は、ウナギなどの細長い体の魚が頭から尾まで体全体を波打たせ、その波を尾端に向かって横方向変位の振幅を増大させながら送る泳法である。アジ型泳法は、体の後半あるいは後ろ3分の1をくねらせ、尾びれの直前で振幅を急激に増大させて波を送る泳法である。ハコフグ型泳法は、体はくねらせず尾びれだけを動かして泳ぐ泳法で、尾びれは主として方向転換用に使われ、リボン状の背びれや臀びれを波を送るように動かし、岩間での微細な位置決めをするのに適したものである。魚類や水棲哺乳類の泳ぎを見ると、レイノルズ数は10~10と高いので慣性力を利用して推進しており、粘性力は体表面上の薄い境界層のみに働くと考えられる。境界層の外側の流れは非粘性非回転流れとして考えられ、流れによる圧力場は水棲動物の体に作用し、体に働く力は体表面全体にわたって圧力を積分することにより求められる。一般に、体に働く圧力の進行方向成分は推力を与える。このことから高レイノルズ数の場合、推進力と抵抗力を別々に取扱うことができる。次に、魚の体のまわりの流れが渦なし流れの場合について述べる。.
