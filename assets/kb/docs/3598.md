# 10・10・2 契約に規定する事項 各契約に共通の事項として，

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・10 契約

特許共有（共同出願）契約 特許や実用新案を二者以上の者が共同して出願したり，その後の管理について取り決めるための契約である．
出願拠出願人が単独でも，その後権利の一部を譲渡することにより共有とする手続が可能である．
この契約では費用負担，出願手続き等の具体的作業について規定が必要である．
特許権（出願権）譲渡契約 契約の一方の当事者が所有する特許権，実用新案権，意匠権及び特許を受ける権利，実用新案登録を受ける権利，意匠登録を受ける権利を他方の当事者に譲り渡す契約である．
f特許実施許諾契約 当事者の一方が所有する特許権について他方の当事者に実施権を許諾する契約である．
実施権は，特許権者でも実施ができずまた別の第三者に実施許諾できない専用実施権と，一般に特許権者の実施が可能でありまた別の第三者に実施許諾できる通常実施権に大別される．
通常実施権でも，特約として別の第三者に実施許諾しない条件，更には特許権者の実施もできない条件を付すこともある．
専用実施権者及び通常実施権者は許諾者の同意が得られれば，実施権者自身が第三者に通常実施権を許諾できる．
許諾の範囲は，自身が許諾を受けている実施権の範囲に限られる．
実施権は設定登録が第三者対抗要件なので，手続についての協力義務を明記しておくのがよい．
企業としての財産の許諾と自己実施の制約から，長期的に利害得失を十分に検討し，許諾の範囲とその対価に反映させる必要があるg．
ノウハウ実施許諾契約 当事者の一方が秘密に管理している技術情報，営業情報，コンピュータプログラム等のソフトウェアに係わる著作権等の情報（ノウハウ）を他方の当事者に許諾する契約である．
独占的に又は非独占的に実施する権利を与えることを規定できるが，公開を前提とされる特許と異なり秘密性が高いほど価値が高いため，再実施権の許諾を規定することは少ない．
同じノウハウでも，許諾する相手の技術水準やそれに伴う許諾範囲の差により対価が変わることがあり，また当事者の責に帰することのできない第三者の開示による公知化等により対価の再評価が必要になることもある．
以下のような事項がある．
前文’契約当事者名，契約対象に加えて，契約に至る背景，経過，目的などの説明事項を記載する場合もある．
秘密保持 当事者双方が秘密として管理すべき情報につい・て秘密保持義務を記載する．
不可抗力 契約当事者の責に帰することのできない事由が生じたときの扱いを記載する．
紛争の予防と処理 契約後の解釈の食い違いを当事者間で解決できないときの協議とその不成立時の処理を記載する．
損害賠償 契約当事者の一方が自己の責任において債務不履行に至った場合の損害賠償について記載する．
f権利移転 契約当事者の一方が第三者に権利を移転する譲渡と，相続や合併による承継について記載する．
契約内容の変更 契約履行中における事情の変化に従い，合理的な内容に変更する際の要件について記載する．
一部無効 契約の無効原因により，全部無効と一部無効となるが，一部無効の際の解釈，扱いについて記載する．
t合意 契約書の作成途中の予備的な合意よりも，締結された契約書の合意内容が優先することを記載する．
通知 当事者間の意思表示や報告などの手続に関する必要事項を記載する．
解約・解除 当事者の一方からの意思表示により契約終了する際の債務不履行及び事情変更などの理由を記載する．
解約は遡及しないが，解除は契約が初めから無かったとされる．
契約有効期間 契約の権利義務の発生と終了の始期と終期を記載する．
契約書の始期は締結日と違うことがある．
契約終了時の措置 期間満了や解下等による契約終了後にも権利内容が継続するような事項の取り扱いについて記載する．
特に秘密保持，開発成果の取り扱いは重要である．
協議 契約書に定められていない事項について，適用可能な法規に基づき処理するか，その処理に疑義があるときの当事者間の処理を記載する．
後文 契約書締結と所在を明確にし，日付，当事者を記載する．
捺印又はサインがなされる．
