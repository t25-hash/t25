# 10・11・1 知的財産保護強化 近年「プロパテント」というこ

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・11 知的財産の動向

とばがしばしば使われるが，その定義は必ずしも統一されてはいな
い．しかし，近年はいわゆる「プロパテント」の時代と言われ，一
般的に知的財産の保護をより強めようという傾向にあることは事実
である．知的財産保護強化の傾向は，特にアメリカで顕著に見られ
期である．
1930年代から1980年頃の第二期は反トラスト政策を重要視する動
きの中で，独占行為や独占に結びつく行為は抑制された．特許が独
占行為の手段の一つに使われることもあり，特許制度に対してもや
や厳しい見方がされた時代である．
1970年代にはアメリカ経済の相対的地位が低下しており，産業界
は海外からの競争に対抗するために，特許の重要性を再認識するよ
うになった．特に日本が技術革新の面でアメリカに勝っていると脅
威論が出ており，経済競争力を急速に回復させる必要に迫られてい
た．そうした中で，1978年から79年にかけて，カーター大統領の産
業技術革新に係わる国内政策レビューがなされ，その提言が連邦巡
回控訴裁判所（CAFC）の設立や国家支援研究成果としての特許権
の民間移転に係わる法律制定につながった．特に，CAFCの設立は，
全米の特許係争を一元的に取り扱う控訴裁として，それまでの各控
訴裁での各判事の独自解釈による判決がでることを解消し，判決に
対する統一的・予測性を高めた．それにより特許価値の高まりから
企業の研究開発投資がなされ技術革新が促進化された．
1980年頃から現在までの第三期は，反トラスト政策の弱まりと共
に，知的財産保護強化の時期に入ったといえる．
1983年から85年にかけて，レーガン大統領のもとで産業競争力委
員会がいわゆるヤングレポートをまとめたが，その中の国内政策と
しての先端技術の保護強化，対外政策としての知的財産保護強化を
狙った二国間交渉，多国間交渉が提言され，通商法改正によるスペ
シャル301条の制定やGATT TRIPSの合意につながっていく．自
由貿易主義を進めている中で，保護貿易的な措置を取ることは困難
であったが，知的所有権の保護の形で対外に対する強硬路線を取る
ことができた．
それ以降もアメリカ企業が更に技術革新を推進することを支援す
るために，特許制度の見直しは継続されている．アメリカ経済にと
って，バイオ等の先進技術製品やビジネス方法発明等のソフトウェ
ア産業では知的財産の保護はその重要度を増している．
アメリカは，このような知的財産保護強化を海外にも強く働きか
け，いわゆる「プロパテント」ということばが使われるようになっ
た．そして技術立国を目指す各国においても知的財産保護強化の政
策が採用される傾向が見られる．一方，過度の知的財産保護に対す
る懸念もされており，今後の動向が注目されている．
