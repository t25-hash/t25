# 2・4・2 代表的筋骨格系と運動機能

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・4 ヒトの筋骨格運動のバイオメカニクス

以下では体幹系を主体とした、筋骨格系の運動発生と筋力の関係について述べる。その他については、2・4・5~2・4・7項の応用例や参考資料を参照されたい。a. 頭部-頸椎系の筋骨格系頭部-頸部の運動は主として、図2・137(a)(b)に示す屈曲、伸展のほか側屈、回旋などの四つの運動があり、図2・137(c)(d)は、これらの運動を発生する主要構成筋を示している。これらを組合わせることで、複雑な運動を可能にしている。頸椎の屈筋として椎体前部筋群(頸長筋など)、胸鎖乳突筋がある。椎体前部筋群のおもな役割は、その拮抗筋となる椎体後部筋群(後頭直筋など)とともに頸椎をまっすぐにし、しっかり固定することである。この作用なしでは安定した頭部-頸部運動を行うことができない。椎体前部筋群や背筋系の収縮により頸椎がまっすぐに保持されている場合に、胸鎖乳突筋が対称性を保ちながら収縮すると、頭部-頸椎系の屈曲をもたらす。椎体前部筋群の影響図2・135 脊柱系の筋骨格系(図2・138)頸椎、胸椎、腰椎系の各椎間板は並進と回転の運動が可能なため、椎体要素は6自由度の[図2・138(d)]運動が可能であるが、脊柱全体では三軸回転の運動発生が主体となる。胸椎の運動は、胸郭等により可動性が制限される。椎間関節は椎骨間の関節であり、椎間円板[図2・138(b)]はコラーゲン線維と線維軟骨が同心円状に配列され、中心の髄核の圧下で関節間の張力を保持している。また外周は関節包靱帯がリング状(包帯状)にこれを取り巻く。椎間円板の端面は椎骨と軟骨性に結合するとともに、関節包靱帯と縦靱帯に結合して、機能単位を構成する。縦靱帯のうち、前縦靱帯は椎体と、また後縦靱帯は椎間円板と強固に結合している。側屈に対しては左右横突起が隣接椎体間で椎間関節を、また後屈に対しては、棘突起が隣接関節間でそれぞれ相対運動を拘束する。c.
