# 2・4・4 多体リンク系の運動学と動力学

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・144 のような表情筋束群の交差部では、1系の筋束と2系の筋束

a. 多体リンク系(multi-body system)の基本運動学剛体系の運動解析では力を考えずに運動を扱い、動力学解析とは区別して並進や回転後の位置、速度、加速度などの運動量を扱う。変位、速度、加速度を表す独立した座標系として、地球(絶対)座標系のほかに剛体ごとのリンク座標を用いて、系の形状や運動の特性を記述する。運動の自由度の数や、その時間導関数が既知なら、他の座標系との関係やその時間導関数を用いて絶対座標系に変換し、系全体の運動量やその時間導関数が統一した座標系で表せる。始めに、静止座標系内にある剛体リンク運動の定式化の方法を述べる(隣接リンク間の相互作用問題も含める)。次に、得られた関係を多体リンク系の問題に一般化して、運動量や力学量の定式化を行ったうえで、運動方程式や関節等での動力学平衡方程式を求める。隣接リンク間の関節における筋収縮などの相互運動拘束がある系のモデル化や、連成解法も含める。図2・145に示す静止座標系内に一つの剛体リンクがある場合を考える。静止座標系内に一つの剛体リンク(骨は剛体リンクと定義する)を定義する。静止座標系のデカルト座標系直交三軸内で、運動前のリンクについてリンク座標系を定義する。運動前のリンクと静止座標系との座標変換関係を、各座標軸の単位ベクトル間の関係として記述する。次にこのリンクが、静止座標系内で直交三軸回りの各回転運動を行うとき運動前後の座標系の変換関係則を、上記とは別に記述する。これにはオイラー角と一般化座標を用いた座標変換則を用いる。並進運動と回転運動の変数を分け、静止座標系の原点からリンクの座標系原点への位置ベクトルを、並進ベクトル(三軸系並進変位)で記述する。リンクのデカルト系直交三軸について、各軸回りの回転運動とその順序を定義し、一つの軸回りの回転運動ごとに、回転前後の座標の関係を、この運動角を変数として記述する。同様に第2軸の角運動後の第3軸について、同様の回転運動前後の座標系間の関係の定式化を行う。このようにして、第1軸、第2軸、第3軸が行う三軸回転前後の座標系間の関係則を併せて定式化する。この三軸角運動前後の座標変換関係則は、三つの角座標運動を変数として記述されているので、一般化座標による変換則と呼ぶ。このような変換則に基づいて、剛体リンク上の任意点の位置での運動前後の関係を定める。b.
