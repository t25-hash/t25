# 2・4・5 骨格筋のモデルを中心とした上腕骨-肩複合体系の座

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・144 のような表情筋束群の交差部では、1系の筋束と2系の筋束

位-立位補助運動解析(実系への応用:その1)本項ではまず、骨格筋による多関節系のリンク運動発生や、外力と均衡する関節内力トルク発生などの筋駆動トルク特性を、筋の能動・受動構成則に基づく連続体力学理論に基づいて定式化した。次に、系の応答特性の非線形性や神経刺激による筋の能動収縮の非線形性を容易に扱うために、アップデーテッドラグランジュ法に基づいて、上述の筋駆動力特性や多関節体の運動を扱うため、一般的な多体動力学解析法を組合わせて上述の問題を取扱う解析法を考える。すなわち筋収縮により角運動を発生させる成分と、収縮引戻し筋力により外力による関節トルクと拮抗する成分を、独立変数として支配方程式に含めて問題を取扱う。この手法を用いれば、リフティングなどの動力学平衡に要する筋駆動力と、リンク角運動を独立変数として取扱って、支配筋別の筋収縮量や筋ごとの駆動力トルクが定量的に推定できるようになる。ここでは、上述の方法により構築したコンピュータシステムを用いて、上腕骨肩複合体系の座位立位運動の起立補助動作の解析を行って、その有用性を明らかにする。a.
