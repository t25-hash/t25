# 2・4・7 骸骨-アニメーションシステムを用いた介護過程の運動

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・144 のような表情筋束群の交差部では、1系の筋束と2系の筋束

解析システム(実系への応用:その3)a. はじめに高齢化社会の到来に伴い、介護・看護過程で、図2・161 関節拮抗筋系の作用内力と関節のトルクバランス患者のベッドまわりで頻繁に生じる各種姿勢でのリフティング(重量物の持上げや搬送)作業で腰痛や肩痛などが頻発し、大きな社会問題になっている。これらは上肢系で行う無理な姿勢での重量物のリフティング作業で、特にウィークポイントとなる、上肢-体幹結合部、下肢-体幹結合部、および胸郭等の補強構造をもたない腰椎系で集中的に発生し、特に腰椎系ではLBP(low back pain)と呼ばれる損傷が多く発生している。一方、体幹では数多くの体幹筋が表層・深層に分かれて張り付き、数多くの四肢-体幹系の運動を発生させるが、これら体幹筋は運動発生と併せて、関節ごとに動力学平衡系を構成するから、これを表層の筋電位計測のみから推定するのは容易ではない。そこで本研究ではこの種の研究の一貫としてまず、体幹系の胸郭、骨盤、腰椎等からなる骸骨ユニットと、これらの系の運動発生や筋トルク発生を司る体幹筋群を組合わせた系が、各種リフティング作業を行う場合のコンピュータアニメーションシステムを構築した。まず、両眼立体視ビデオカメラで計測した体幹代表点ごとの三次(a) 母指の3リンク運動(b) 筋の能動特性図2・162 各筋の能動特性の時刻歴図2・164 筋骨格系骸骨要素と組立系元座標を与えて、(1)運動発生に必要な等張性筋活動レベルと、(2)関節別の動力学平衡に必要な部分等尺性張力を、同時並行的に多体動力学システムにより解析できる方法を考えた。これを用いて介護過程の横振りリフティング運動における姿勢角と負荷条件を変化させ、一連の実験および解析を行い、両結果を比較して本手法の有効性を明らかにした。b. 研究内容頭部、背柱(7頸椎、12胸椎、5腰椎)、骨盤、下肢それぞれの各部位別骸骨模型(ひな型)データベースシステムを構築した(図2・164)。次にこの各部位別のひな型骸骨模型を組合わせて、人体全体系の骸骨模型を構築した。介護者や高齢者のリフティング問題で大きな問題をもつ、体幹系の胸郭-腰椎-骨盤系(体幹関節)の屈曲、伸展、側屈および体幹ひねりを組合わせた各種運動生成用アニメーションシステム(ani-(a)(b-2) 母指(b-1) 示指(b)図2・163 母指、示指系のつまみ動作と関節トルク特性mation system)を構築した(図2・165)。この骸骨アニメーションシステムの各部に、上述の体幹運動を発生させる運動面別駆動筋の拮抗筋系を装着して、それぞれ屈曲、伸展、側屈、体幹ねじりに対応する運動発生用の骨格筋の活動レベルを求められるようにした。実人間の各種モードの体幹運動を、2台のカメラを用いた両眼立体視装置で計測する(図2・166)。この結果得られる体幹各部の三次元運動計測データを骸骨アニメーションソフトに与えて、実人間と同等の体幹運動を発生させる(図2・167)。この運動過程で、先に装着した各種骨格筋の活動レベルが求められるシステムを構築した。図2・168は、屈曲・伸展運動の実験を行って得られる各筋の筋電位計測の結果に基づく計測ひずみを、筋装着アニメーション(図結果(図中太実線)と比較したものである。両結果は比較的よく一致していることがわかる。以上の手法を用いると、体幹系の面状筋以外の任意の筋の運動発生と筋力発生の逆同定が、任意の運動面の任意の運動時刻について容易に求められるようになる。(c) 側方回旋(b) 側屈(a) 前屈図2・165 胸-腰関節系の代表的な運動パターン図2・166 両眼立体視による三次元座標の発生図2・168 横振り運動時の筋活動レベル図2・167 筋骨格系骸骨アニメーション8―101
