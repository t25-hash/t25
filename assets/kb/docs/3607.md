# 10・12・3 日本弁護士連合会 日本弁護士連合会は弁護士及び

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・12 知的財産関係機関，諸団体とその機能

弁護士法人の事務の改善・進歩を図るために，その指導・連絡及び監は，日本弁護士連合会の承認を受けて会則を定めている．
弁護士は，始めとする法律事務を行う．
し，弁護士，弁理士，学識経験者が調停・仲裁等によって知的財産権は2000年から社団法人日本ネットワークインフォメーションセンタもなっている．
督の事務を行う．
地方裁判所の管轄区域ごとに設立される弁護士会弁護士法に基づく国家資格者として，知的財産権を含む訴訟事件を1998年に日本弁理士会と日本知的財産仲裁センターを共同で設立に関するさまざまな紛争の解決に当たっている．
