# 2・5・1 生体の熱収支と体温調節

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・5 生体熱工学

a. 生体と温度生体は一種の化学的エネルギー変換装置である。ヒトは、食物と水と空気を外界から取り入れる。食物と水とは消化器系で分解・吸収され、吸気中の酸素は肺で排出される二酸化炭素と交代に血液中に溶け込んでいく。吸収された食物と酸素とは、やがて化学的に結合して生命維持に必要なエネルギーを生み出す。生体が外界から物質およびエネルギーを体内に取り入れ、吸収・貯蔵し、物質およびエネルギー変換を行い、不要物質と余剰エネルギーを排出する一連の過程を代謝(metabolism)という。このような代謝の中間過程は複雑な化学反応からなっているが、そこでは熱の発生・吸収があり、総和として熱の発生が過剰の状態になっている。しかし、この熱は体温維持には役立つものの、他の形のエネルギーに再変換されることはなく、最後には体外に放出される。周知のように化学反応は、温度に極めて敏感な過程である。多くの化学反応の速度は、温度が10°C上昇するごとに約2倍に増大する。生物学あるいは生理学の分野では、これをもっと定量的に値という変数で表すことがある。値(ten value)は、ある化学反応の速度定数の温度(°C)における値を욧、温度が10°C上昇して(+10)(°C)になったときの値を욧용とするとき、次のように定義される。=욧용/욧(2・335)値はヒトの代謝においては、約2.3~2.5であるといわれている。このことは、化学反応によって生命を維持している生物にとって、温度が極めて重要な役割をもつことを意味している。何らかの原因で体温が大幅に低下するような事態が生じれば、生物の代謝活動の速度は大きく減退することになる。自然界には、外界の温度が変化しても自分自身の体温をある程度一定範囲に維持できるような生理的機能を備えた恒温動物(homeotherm)と、外界の温度に応じて体温が変化する変温動物(poikilotherm)とが存在する。個体の生存にとって有利なのは前者であり、その意味では恒温動物のほうが進化の度合いが進んでいるといわれる。もっとも恒温動物の体温維持機能にも限界があり、極度の低温に抗して体温の恒常性を保つことは困難である。北国に住む動物の中のある種のもの、例えば、クマ、リス、コウモリなどは冬眠(hybernation)によって、また他の動物は渡り(migra-tion)によって気温の低下に対応する。また、個体は死滅しても、子孫を残すことによって種の保存を図る生物もある。いずれにせよ、環境の温度変化にどのように対応して、個体あるいは種族を維持していくかは生物にとって大問題であり、進化の過程を経ての一つの解答ともいえるものが、b項以下に述べる恒温動物の体温維持機能である。b. 中核温(core temperature)と外殻温(shell temperature)恒温動物は、小さな鳥から巨大なゾウやクジラに至るまで、その中核部の温度をかなりよい精度で一定に保っている。中核部の温度は種によって異なり、哺乳類よりも鳥類のほうが高いのが一般的傾向である。外気温が低いとき、われわれの手足や顔の表面などの末梢部あるいは外核部(shell)は冷たくなるが、脳や内臓を包含する身体の中核部(core)の温度はほとんど変わらない。図2・169は、冷環境および温環境における人体内の温度分布の様子を示したものである。環境温度が高いと、図2・169の左半分に示すように、身体のほとんどの部分が中核温に近い温度(斜線部分)になる。これに対して、環境温度が低下すると末梢部の温度は下がり、中核温を保つ領域はずっと狭くなる。末梢部全体の熱容量は、大ざっぱに見積もって人体の全熱容量の40%を占めるといわれている。このように、冷環境においては、外核温が中核温よりもずっと低くなることによって身体から外界への放熱が抑制されるが、このこ8―105とは、冷環境でのエネルギー消費の増大を抑え生命活動を長続きさせる点では、合目的的な仕組みであるといえる。しかし、極端な低温のもとでは、末梢温の低下のために凍傷が起こり、時には組織の壊死にまで至ることがある。なお、冷環境下での外核温の低下が、末梢組織への血液循環量の減少によるものであることについては、e項で述べる。ヒトが恒温動物であるという言い方をするとき、一定に保たれるのは中核温である。われわれが病気にかかると、その状態を知る一つの目安として体温を測る。体温を測るには、通常棒状温度計を使って腋の下(腋下)あるいは口腔内の温度を測定する。しかし、上に述べたことから、腋の下や口腔内の温度がただちに身体の中核温を示すものでないことは明らかである。腋あるいは口をしっかり閉じて、その部分の温度が熱伝導によって中核部と等しい温度まで上昇するのを待つという条件が満たされて初めて、中核温に近い温度が検出できることになる。したがって、サーミスタ温度センサのように熱的時定数の短い体温計を用いたとしても、短時間で中核温を精度よく知ることは原理的に困難である。なお、十分時間をかけて測定しても、通常腋の下で測った体温は約0.8~0.9°C、口腔内温(舌下温)は約0.4~0.6°C、それぞれ次に述べる直腸内温よりも低いといわれている。以前は、生理学の分野でヒトの身体の中核温を知るのに、直腸内の温度を測定していた。図2・169から見れば、直腸の深部の温度は外界の温度に影響されにくいから、十分中核温を代表するものと考えられる。しかしその後、ヒトの体温調節の中枢が大脳の視床下部(hypothalamus)にあり、この部分の温度変化が体温調節のための諸機能と最も密接に結びついていることが認められたため、視床下部温が中核温を代表するものと考えられるようになった。もっともヒトの場合、視床下部の温度を直接測定することはできない。それに代わるものとして、耳内の鼓膜の温度を熱電対によって測定する方法が提案されたことがある。また、最近放射温度計測によって耳内温度を検出する体温計が市販され、普及しつつある。このような中核温に対し、外核温のほうは環境温度によって大きく変化し、部位による差異も大きい。しかし、後述するように、身体の最外表面である皮膚には、各部位の温冷の度合いを検知する受容器(神経の終末)が分布しており、それらからの信号と体温中枢の温度自体とを大脳が総合的に判断することによって、ヒトの寒暖の感覚、さらには快感・不快感が生じるのであるから、皮膚の演じる役割も極めて重要である。皮膚温は部位による差異が大きいので、部位ごとの温度に重みを付けた平均皮膚温を代表値として用いることが多い。平均皮膚温(average skin temperature)율윳の算出法にはいろいろな方式が提案されている。例えば、人体を頭部、腕、手、背中、胸、大腿部、脚、足の八つの部位に分け、それぞれの表面積の割合で重みを図2・169人体内の温度分布(斜線部分が中核温に保たれる)付けると次式のようになる。율윳=0.07읈윬율윫+0.14율읍윯읎+0.05읈율윰윫읎+0.17윩율윪윭+0.18윪읈윬읎읏+0.19읏읈읉읇읈읏+0.13윮윬읇읎+0.07읆윬윬읏(2・336)式(2・336)のような平均体温は、体表から外界への伝熱量を概算するのに役立つ。すなわち、体表から周囲の空気への対流熱伝達率が場所によらず同じ値をとるなら、外界への単位時間当たりの伝熱量は=율윳-욀(2・337)で表されるからである。ただし、は全体表面積、욀は気温である。しかし、体表と外界間の伝熱過程には、発汗、熱放射、気流、湿度、衣服などの影響が加わるので、式(2・337)の伝熱学的な意味については若干疑問の余地がある。c. 生体における熱収支ヒトの体内での発熱は、身体の一部あるいは全体の温度上昇、および身体から外部環境への熱移動に費やされる。単位時間当たりの代謝による発熱量を、人体の温度上昇に費やされる熱の総量を、身体表面から外界への放熱量の総和をとすれば=+(2・338)と書くことができる。ここで、およびは正・負いずれの符号もとることができ、については身体全体として熱が蓄えられつつあるときに正、逆の場合は負、またについては、身体表面からの放熱量の総和が正か負かで正負が決まる。しかし、が負になることはないから、ととが同時に負になることはない。身体における発熱源となる代謝の過程では、外界から摂取された物質のもっている全エネルギーが有効に利用されるわけではない。骨格筋の収縮、体内諸機能の維持などに使われるエネルギーは、アデノシン三リン酸(ATP: adenosine triphosphate)のような化合物の形で蓄えられ伝達されるが、その合成過程で約50%のエネルギーが熱に変化する。また、蓄えられたエネルギー源物質を仕事(筋収縮など)に変換する際の効率はたかだか25%どまりで、残りは熱に変わる。これらの熱の総和が代謝による発熱となる。代謝発熱量は、人間の性別・年齢・体重・活動状態によって大きく変化する。安静時の標準的代謝発熱量を基礎代謝(basal metab-olism)という。日本人の成人男子の基礎代謝は、単位体表面積当たり40~47W/mで、成人女子ではこれより約10%低い。なお、体表面積は、成人男子の場合約1.4~1.6mである。運動時の最高代謝発熱量は、基礎代謝の値の20倍に達することもある。d. 体温調節(body temperature regulation)動物が自分の体温を保つのに必要な代謝発熱が最小であるような環境温度を、中性温域と呼ぶ。ヒトの場合は28~32°Cである。この温度域での体表面から外界への放出量の調節は、主として皮膚表面付近にある毛細血管への血流の増減によって行われる。すなわち、毛細血管の拡張・収縮によって血液流量が変化すると、血液による対流伝熱量が増減することによって皮膚表面の見掛けの熱伝導率が変化し、表面に向かう熱流束(熱伝導率と温度こう配の積)が調節される。ヒトの皮膚の場合、血液流量の変化は100倍にも達し、それにより見掛けの熱伝導率は約4倍変化するといわれている。ヒトの体温調節は、おおよそ次のようにして行われる。環境温度が低下し始めると、人体は二つの仕方でこれに対応する。一つは代謝発熱の増大であり、もう一つは末梢部への血液流量の低減である。寒さを感じたときに起こるふるえは、骨格筋の運動によって代謝発熱を増やそうという生理反応の一つである。また、末梢部への血液流量が減少すれば、その部位の皮膚温度が低下するから、外部への伝熱量も抑制されることになる。先に示した図2・169の右半分は、このような状態を表している。このとき、血液流量の極端な減少によって末梢部への酸素や栄養物の供給、あるいは末梢部からの炭酸ガスや老廃物の排除に支障をきたさないように、多くの動物では、末梢組織へ向かう循環系の動脈・静脈間で対向流熱交換が行われている。クジラがそのひれの中での血液流間の熱交換によって、低温の海水中でも体温と運動能力との両方をうまく保持していることは有名である。ヒトの四肢の血管系においても同様の構造が見られるが、それがどの程度有効に働いているかはよくわかっていない。このほか、鳥類などは、低温環境では羽毛を逆立てて断熱性をよくすることができる。ヒトの場合の鳥肌(立毛筋反応)はその名残りである。一方、高温環境においては、代謝発熱は抑制され、末梢循環量の増大に伴う皮膚表面温度の上昇によって放熱が促進される。それでもなお皮膚温度が上昇する場合には、発汗とその蒸発による放熱が行われるが、その度合いは周囲温度の上昇とともに急増する。なお、蒸発は皮膚表面の汗のみでなく呼吸気でも行われ、発汗機能の劣る動物では重要な放熱機構となっている。ヒトの体温調節において、外界と身体との間の熱移動は重要な役割を演じるが、それには次の四つの伝熱形態が関与する。(1)熱伝導、(2)対流伝熱、(3)放射伝熱、(4)蒸発。簡単に説明を加えると、(1)は身体中心から皮膚表面までの熱移動、皮膚表面とこれに接する空気層、衣服、床材・家具などへの伝熱に関与する。(2)は体内を流れる血液と諸組織、皮膚表面や衣服表面と空気や水との間の伝熱、(3)は皮膚表面あるいは衣服表面と外界、逆に太陽や外部の高温物体から人体への伝熱を支配する。(4)は環境温度が高い場合の放熱機構として有効である。これら四つの伝熱形態のうち(1)の熱伝導は、多くの場合人体内部の温度分布および皮膚表面の温度を決定するもので、体表面と外界との熱交換への直接的寄与は少ない。残り三つがそれぞれどの程度の寄与をするかは、外界の温度・湿度・気流・熱放射の環境(特に日射)、身体側の条件(活動状況、姿勢、着衣)などによって大きく変化する。ヒトなどの体温維持機能の現象論的な側面は上述のとおりである。しかし、ヒトの中心体温を約37°Cというほぼ一定の値に長時間にわたって保持するための制御中枢の働き、特に体温調節における設定温度はどのようにして実現されるのかについては、まだ不明な点が多い。この問題については、次のような説明が有力と思われる。大脳内(より正確には視束前野・前視下部領域)には、温度の変化に鋭敏に反応して放電頻度が増減する神経単位(ニューロン)が存在する。これには温ニューロンと冷ニューロンの2種類があり、例えば温ニューロンでは温度が上がると放電頻度の増大が観察される(図2・170中の実線W)。正常状態では、これら両ニューロンの放電頻度はいわゆる正常体温(すなわち37°C)で等しくなり(図2・170中のS 点)、この点が体温調節の基準点の役目をする。風邪をひくと、体内に生成した化学物質のため温・冷ニューロンの放電特性はそれぞれW′,C′への移行し、その交点がS′となるため基準温度も変わる(図2・170では40°C)。このとき、基準温度が変わっても体温調節の機構は正常時と同じように作動する。風邪の初期症状として、体温は上がっているのに寒気を感じるのは、調節の基準温度が上がるためである。このような機構が、外部からの病原菌の侵入に対する生体の防御機能としてどのような役割を演じるかは別図2・170 体温調節の基準温度として、上述のような考え方によって、多くのことが説明できるようになったことは確かである。しかし、不明の部分もまだ多く残っており、今後の研究に待つところが大きい。e. 温度感覚(temperature sensation)これまで述べてきたように、ヒトの身体全体の熱平衡は、大脳視床下部にある体温調節中枢の働きによって維持されている。すなわち、中枢にある温・冷ニューロンが中核温の変化を検知し、得られた情報に応じて発熱と放熱のバランスをとっていくのがその機構である。しかし、ヒトは大脳内でそのような判断を下すばかりでなく、皮膚や口腔内の粘膜でも温度を感じることができる。このような末梢組織で感じとられた温度情報も体温調節中枢に伝えられるが、この情報の役目は、身体が置かれている環境の温度を早く知らせることによって、発熱や放熱(すなわち、ふるえや発汗など)の開始温度を変えることにあると考えられている。この意味で、皮膚などの温度感覚の役割は重要である。直径1~2mmの銅の棒を20°Cぐらいに冷やし、その先端を皮膚の表面に触れると、ある特定の場所では冷たさを鋭敏に感じる。また、同じ棒を45°Cぐらいに温めて皮膚表面を探ると、温かさを鋭敏に感じとれる場所が見つかる。前者を冷点、後者を温点と呼ぶ。このような温点・冷点はヒトの全身に分布しているが、その密度は部位によって異なる。表2・16によれば、冷点のほうが温点に比べてはるかに数が多く、例えば手の甲では前者が約8個/cmであるのに対し、後者は約0.5個/cmである。古い生理学の教科書によれば、冷点に対応してクラウゼ棍状小体、温点に対応してルフィニ小体と呼ばれる感覚受容器が皮下に存在していることになっていたが、現在ではこの説は誤りとされている。しかし、冷点・温点に対応する何らかの形の温度受容器があることは否定できず、いろいろな実験的事実から、冷受容器のほうが皮膚近く(深さ0.2mm程度)に分布しており、温受容器はやや深い位置(0.4mm程度)にあるものと考えられている。このような温度受容器がどのような刺激によって作動するかについても、これまでいろいろな説が出されている。生理学の教科書においては、温度感覚の発現条件として、初期皮膚温度・刺激温度・刺激温度の変化速度・刺激面積・刺激時間・末梢血液循環状態などが関与すると述べられているが、具体的な説明はほとんどなされていない。
