# 2・5・2 組織内の熱移動

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・5 生体熱工学

生体組織における熱移動は,熱媒体としての血流が主要な役割を演じているので,生体内の熱移動量を求めるためには,血流による影響を明確にする必要がある.ただし,生体組織構造や血管構築の複雑さを定量的に把握し,生体内の熱移動現象を直接測定することは現在のところ非常に難しいので,生体組織内熱移動モデルによる評価を行うことが現実的である.さらに,血流の影響を含めた組織の有効熱伝導率を求め,有効熱伝導率から血流の影響を把握することもある.そこで,本項では,生体組織内の熱移動量を求めるため表2・16 組織内血管網における熱移動血液循環系における熱移動は,太い血管系よりも細い血管系において顕著であるが,微小血管系の幾何学的構築は複雑であり取扱いが難しい.そこでまず,簡略化された血管床モデルにおいて熱伝達を見積もる試みがある.組織内にまっすぐな血管が,隣どうしの血管と一定の距離を隔てて分布しているとして,チェン・ホームズ(Chen-Holmes)は,血液の温度変化が周囲の組織との熱伝達のみによって決まるという仮定のもとに,循環系の各部位に対して簡単な計算を行った.チェン・ホームズは,血液の温度が周囲の温度に十分近づくときの血管の長さを平衡長さ(equilibrium length)욄と定義し,実際の血管の長さとの比/욄の大小によって各部位の熱交換の度合いを評価した(表2・17).平衡長さ욄は,内径が1mm以上の太い動脈ではに比べて非常に長いため,熱交換量はわずかで,一方内径が10 m以下の細動脈や毛細血管では/욄≫1となり,血液の温度はほとんど周囲組織の温度に等しい.したがって,おもな熱交換は,内径が0.2~0.5mm程度の細い血管床で生じるという.これは,熱交換は最も細い毛細血管レベルで行われると予想されていた従来の見解と異なり,興味深い結果である.c.
