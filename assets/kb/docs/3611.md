# 2・5・3 高温における生体

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・5 生体熱工学

生体の温度は、通常37°C前後の生理的温度範囲内にあり、それからあまり大きく離れない環境温度の範囲で生体は生きている。ここでは、生理的温度範囲より高い温度を高温、低い温度を低温とするが、生体は高温や低温によりもたらされる傷害(損傷)を受けたり、逆に、医療では高温・低温効果が利用される。高温環境での傷害として熱傷(火傷)を、高温の医療応用としてハイパーサーミア(癌温熱療法)を取上げる。a. 熱傷(thermal burn)。 熱傷の症状と深度生体は通常、外部からの高温・高熱流束の加熱を受けると組織温度が上昇する。これに基づく発赤、水疱、壊死などの組織変化を熱傷と呼び、熱傷はおもに皮膚に起こる。皮膚は、組織学的には体表面から順に、表皮、真皮、皮下組織の三層構造をなし、機能的には内部組織の保護、分泌・排泄作用、体温調節作用、知覚作用などを担う。したがって皮膚熱傷は、組織学的な形態変化、生化学的変化、微小循環系の変化などを伴う。熱傷の損傷の程度は、温度が高くその作用時間が長いほど大きいが、以下のように分類され、皮膚損傷の深さ(深度)と対応する。(1) 1度熱傷(紅斑性熱傷):表皮の発赤・紅斑。(2) 2度熱傷(水疱性熱傷):2度浅層性熱傷:真皮浅層まで及ぶ。発赤・水疱・びらん。2度深層性熱傷:真皮深層まで及ぶ。蒼白・水疱・びらん。表皮源となる細胞(毛囊、皮脂腺、汗腺)や知覚神経末端の多数破壊(植皮もある)。(3) 3度熱傷(壊死性熱傷):完全蒼白・黒こげ(炭化)。皮膚全層が壊死(要植皮)。ひどい場合には深部の筋肉層まで及ぶ。熱傷の重症度は、受傷部位での深度に加え受傷面積も含めて判定される。そのほか特殊な熱傷として、気道熱傷、電撃傷、化学損傷がある。. 損傷の生体工学的記述と評価火傷の発生と損傷の程度を生体熱工学で定量的に記述・評価するためには、火傷を組織の構成成分(タンパク質など)の変性という化学反応であると考える。組織温度(°C)の時間変化に対して、損傷の程度を表す損傷関数(damage function)Ωを導入し、損傷の進行速度(化学反応速度)を表す時間微分係数dΩ/dにアレニウス型を仮定するとΩ=exp -+273dここで、頻度因子=3.10×10s、活性化エネルギー=6.28×10J/mol、一般気体定数=8.31J/K である。との値に関しては種々改良がなされている。火傷の組織学的検討との比較から、Ωの大きさと損傷の程度が関係付けられる。すなわち、以下のようになる。(1) Ω<0.53:表皮の不可逆的損傷は発生しない。(2) 0.53<Ω<1:表皮の損傷は不可逆であるが不完全(1度熱傷)。(3) 1<Ω<10:表皮の完全な壊死状態(2度熱傷)。(4) 10<Ω:皮膚全層が壊死(3度熱傷)。b. 医療における高温効果の利用。 ハイパーサーミア(1) 概要と特徴癌などの腫瘍組織を約42~45°Cに加温することにより壊死させる手術療法を、ハイパーサーミア(癌温熱療法、hyperthermia)という。図2・172に例を示すように、一定温度に加熱された細胞の生存率は、温度が高く経過時間が長いほど低下し、特に42.5°C以上では急激に低下する。ハイパーサーミアは、基本的にこの温熱の致死効果を利用する。温熱の致死機構は、細胞レベル、組織レベルで解明されるとともに、分子生物学的にも解明されてきている。癌細胞は一般に、正常細胞に比べて温熱感受性が高い(温熱刺激に対して低い温度、短い時間で壊死しやすい)。さらに、癌組織は正常組織と比べ、低pH、低酸素、低栄養、低血流量状態にあり、これらの因子は温熱感受性を高める。温熱単独でも細胞の致死効果を示すが、温熱と放射線の併用、または温熱と抗癌剤の併用(温熱化学療法)は、致死効果を増大させるため多用される。標的とする腫瘍を完全に壊死させ、周囲正常組織への損傷を最小限にとどめる必要がある。(2) 加温法全身温熱療法と局所温熱療法に大別される。細胞が繰返し加温に対して示す耐性(熱耐性)を回避するために、日数をおいて加温する。(a) 全身温熱療法(systemic hyperthermia)遠隔転移進行性癌を対象に全身加温する。41.5~42°Cで2~10時間、1~2週間おきに2~5回加温。加温法としては、体外循環による血液加温、図2・172 チャイニーズハムスターの培養細胞を種々の温度で加温した場合の生存曲線(生存率と経過時間の関係)8―109温水浴、温熱パラフィン浴、マイクロ波や赤外線、遠赤外線の全身への照射などがある。(b) 局所温熱療法(local hyperthermia)局在性腫瘍を対象に、腫瘍が存在する局所または局所を含む領域を選択的に加温する。42~45°Cで40~60分間、週1~2回で5~10回加温。加温法としては、マイクロ波加温[マイクロ波(約300MHz以上の電磁波)を用いた誘電損失による発熱],RF(radio frequency)波誘電加温[RF波(約300MHz以下の電磁波)を用いた高周波電流のジュール発熱],RF波誘導加温(RF波を用いた高周波磁場の電磁誘導による渦電流のジュール発熱)、超音波加温(超音波エネルギーの吸収による発熱)などがある。生体に対して無侵襲で外部から加温する外部加温、または、侵襲的に体内を加温する内部加温(腔内加温、組織内加温)が行われる。加温のためのエネルギーの集束効果の利用もある。癌の種類、加温部位や範囲に応じて適当な方法が用いられる。(3) 術中モニタとシミュレーションによる手術支援病変部を超音波、X 線CT,MRI(磁気共鳴画像法)、内視鏡などでモニタするとともに、温度分布や壊死領域のモニタが必要である。超音波、マイクロ波、MRI,X 線、インピーダンスなどによる温度計測法は現在研究中であるが、熱電対、サーミスタ、光ファイバなどの温度センサを直接刺入する侵襲的方法が用いられている。さらに、施術の最適設計(治療計画)のために、体内発熱密度分布、血流による熱移動、熱伝導などを考慮した生体伝熱工学の数値シミュレーションによる手術支援が有用である。
