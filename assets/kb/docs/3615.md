# 2・5・7 生体内物質移動

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・5 生体熱工学

a. 生体内における物質移動の形態物質の移動が起きるためには駆動力が必要であるが,この駆動力の形態によって,生体内の物質移動は受動輸送と能動輸送に分けられる.さらに,細胞膜そのものを変形させて物質の輸送を行う膜動輸送がある.. 受動輸送(passive transport)生体内ではほぼ定温,定圧という環境であるため,生体内のエネルギー状態を表すにはギッブス(Gibbs)の自由エネルギーが適当である.物質の移動を許している開いた系において,自由エネルギーの物質移動により変化する割合옿/옿욡욣욈を電気化学ポテンシャル(electricalchemical potential)μ욡と呼ぶ.(1) 自由拡散(free diffusion)二つの溶液の間に電気化学ポテンシャルの差が生じると,そのこう配が駆動力となって,こう配を下る方向に物質が移動する.自由拡散による物質流束욦はテオレル(Teorell)によって次のように表される.流束=移動度×濃度×駆動力욦=욦욦-dμ욦/d(2・346)理想溶液の場合,よく知られるフィックの拡散の式となる.(2) 促進拡散(facilitated diffusion)物質を運搬する輸送体(carrier)が膜内に存在する場合,輸送体自身がその化学ポテンシャルのこう配に従って移動するため,あたかも物質を運ぶ渡し船のような役割を果たしている.このように受動的に特定の物質を選択的に透過させる物質輸送形態は,促進拡散(facilitated diffusion)と呼ばれ,生体内での特徴的な物質輸送形態の一つである.. 能動輸送(active transport)能動輸送とは,溶質分子が電気化学ポテンシャルの低いところから高いところに輸送されることをいう.ポテンシャルの登り坂を登るためには,ATPで代表される生体エネルギーを必要とする.. 膜動輸送(cytosis)通常は,大きな分子は細胞内に入らないが,膜自体が変形して,菌体,抗原,タンパク質などの大きな物質を取込んだり,細胞内で産出された酵素タンパク質,多糖類を分泌することがある.このように細胞膜が外液の一部を包み込み,細胞内に取込んで小胞を形成するような,膜の変形を伴う輸送を膜動輸送(cytosis)と呼ぶ(図2・180).最もよく現れる膜動輸送は,飲作用(pinocytosis)と呼ばれるものであり,毛細血管壁や上皮に小さな小胞を形成して,溶液のみを包む.次に,細菌や固体粒子のようなものを取込む場合を食作用(phagocytosis)と呼び,通常光学顕微鏡で認められるほどの大きさである.第三の取込み方として,レセプタが関与するエンドサイトーシスがある.細胞の表面に存在しているレセプタが溶質と結合して取込まれ,内部に小胞が形成される.リポタンパクが膜動輸送によって細胞内に取込まれることが確かめられている.b. 血液循環系の物質移動. 血液中の酸素および炭酸ガスの輸送(1) 血液中の酸素輸送血液中に含まれる酸素の一部は物理的に溶存した形で,大部分は赤血球内のヘモグロビンと結合した形で存在する.血液中のヘモグロビンが完全に酸素加されたときの結合酸素量を酸素容量というが,血液中に含まれている結合酸素量と酸素容量との比を酸素飽和度といい,通常百分率で表す.平衡状態における酸素飽和度と,酸素分圧との関係を表す曲線を酸素解離曲線といい,特異なS 字状の曲線で,この非線形性が血液における酸素輸送の特徴を決める.図2・180 細胞膜を変形させて物質を取込む膜動輸送の形態輸送抵抗赤血球内部赤血球膜赤血球近傍の血漿層血漿毛細管壁近傍の血漿層毛細血管壁組織拡散対流拡散解離쟭拡散拡散輸送メカニズム赤血球膜と毛細管壁との間げきの血漿層における拡散毛細血管内から組織への酸素輸送の過程を,図2・181に示す.赤血球内から吐き出された酸素は,血液中に溶存し対流によって毛細血管壁に到達し,組織へと運搬される.(2) 血液中の炭酸ガス輸送血液中に含まれる炭酸ガスの一部は物理的に溶存した形で,大部分は重炭酸イオンおよびヘモグロビンと結合した,カルバミノ化合物として存在する.血液中の総炭酸ガス含有量と,炭酸ガス分圧との関係を表す曲線を,炭酸ガス解離曲線という.赤血球内に存在する炭酸脱水酵素が,溶存炭酸ガスから重炭酸イオンへの解離に寄与している.(3) 生体組織におけるガス輸送筋肉や脳組織などの生体組織におけるガス輸送は,古くから生理学の重要課題であり,クロー(Krogh)による酸素分圧分布の古典的解析に始まり,多くの実験的理論的研究が行われている.特に微小血管網による酸素供給のメカニズムが,観測方法の進歩によって明らかにされ,毛細血管のみならず,細動脈からも酸素が多く供給されていることが示されている.例えば,図2・182に示されるように,細動脈,毛細血管および細静脈のネットワークの中で,対流と拡散により組織へ酸素を供給している.
