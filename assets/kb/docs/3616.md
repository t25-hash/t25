# 2・5・8 生体内物質移動の計測法

> **章**: 第2章 バイオメカニクスの基礎
> **節**: 2・5 生体熱工学

a. 細胞層の物質透過係数(permeability)の計測法細胞層の物質透過係数をin vitro で計測する最も一般的な方法は、細胞を微孔性膜(ポリカーボネートなど)上に播種して培養し、コンフルエントになった状態で、微孔性膜と細胞層を透過する物質の量を計測する方法である。物質透過が重要な機能である細胞の一つが血管内皮細胞で、血管内皮細胞の物質透過係数の計測が、微孔性膜を用いて広く行われている。特に動脈硬化症などの血管病変と直接関わっている、高分子物質の内皮細胞層の透過係数が重要であるため、透過する物質としては、アルブミンや低比重リポタンパクに関心が集まっている。図2・182 細動脈、毛細血管および細静脈のネットワークにおけるガス輸送の形態膜への溶解細胞内の拡散(細胞内経路)膜への溶解拡散酸化拡散、対流(細胞外経路)図2・181 毛細血管から組織への酸素輸送のプロセス図2・183 微孔性膜(フィルタ)上の細胞層を上室と下室が挟み込み、濃度差を与えて細胞層の透過率を計測する図2・185 生体組織標本にガスの濃度差を与え、拡散係数を計測する装置図2・183に示すような透過チャンバでは、上室と下室の間に微孔性膜上に培養された細胞層を挟む。その際、浸透圧差や圧力差が生じないようにすれば、上室と下室との濃度差で物質が透過するような状態になる。そこで、着目している高分子物質を蛍光物質あるいは放射性同位元素で標識し、透過して下室に抜けてくる物質の量を蛍光強度や放射線種によって決定する。蛍光物質で標識する場合が多いが、その場合は、下室からサンプルを採取して、蛍光分光光度計によってサンプル内の蛍光色素の濃度を測定し、以下の関係によって透過量と透過係数を決めることができる。このようにして得られた透過係数は、細胞層を通過する物質の平均的な値であるが、実際には細胞内を透過する場合や、細胞間を通り抜ける場合があり、共焦点レーザ顕微鏡や電子顕微鏡などの画像から細胞内や細胞間での物質透過の様子を把握することによって、細胞層を透過する状況を明確にする必要がある。図2・184に、内皮細胞の物質透過の形態を示す。大きく分けると、細胞内を透過する場合と細胞間を通過する場合に分けられる。細胞内を透過する場合は、レセプタを介して透過するものと、小胞内を拡散によって移動するものがある。このように細胞層の物質透過の形態が、透過する物質の種類に応じて変わるので、どのような透過経路をたどるかに大きな関心が集まっている。特に内皮細胞は、細胞に接する血流の力学的刺激を受容することが知られているので、力学的刺激が物質透過にどのような影響を与えるかも、重要な研究課題になっている。8―115図2・184 内皮細胞層の物質透過形態b. 組織内のガス拡散係数(gas diffusion coefficient)の計測法生体組織内の酸素や炭酸ガスの輸送を定量的に明らかにするためには、酸素や炭酸ガスの拡散係数などの物性値が必要となる。理想的にはin vivo で計測できることであるが、in vivo 測定は困難であり。摘出された生体組織の薄い層にガスの濃度差を与え、拡散係数を決めるというin vitro 測定が行われている。図2・185に示されるように、1室と2室の間に生体組織の薄いサンプルをOリングなどで張り、1室には低濃度の混合ガスを、2室には高濃度の混合ガスを、それぞれ一定の同じ流量で流し、生体組織に濃度こう配を設定する。定常状態に達した後、1室出入口のバルブを閉めて密閉室とする。1室が密閉された状態では、1室から2室に濃度こう配に沿ってガスが拡散し、1室の濃度が徐々に増加する。1室の濃度はガス電極で経時的に測定できるので、一定時間内の濃度が求まり、それから生体組織層を通過するガス量が求まり、最終的に以下の式で拡散係数が求まる。文献(1) Fung, Y.C.
