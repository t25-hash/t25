# 10・2・4 実用新案 実用新案は，特許と同様に自然法則を利

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・2 特許・実用新案

医薬，化学物質等の材料自体に係る考案は登録されない．
実用新案法では，出願から早期に実施が開始される考案やライフ
サイクルの短い考案の適切な保護を図るべく，平成6年1月1日移

なる（実第37条等）．
一方，権利行使には特許庁が発行する実用新案技術評価書（空曇
12条）を提示した警告が義務づけられており（実第29条の2），また，
安易に警告や権利行使すると，その後に実用新案登録が無効にされ
た場合に（実第37条等），損害賠償責任を負う場合があるため（実第
29条の3），権利行使には特に注意が必要である．
a．特許との主な相違点
i、保護対象において異なる点
（1） 自然法則を利用した技術的思想の創作において，高度である
ことは要求されない（実第2条，特第2条）．
（2） 物品の形状，構造又は組み合わせに係る考案に限定されてい
る（帰塁3条，6条の2）．
ii．進歩性において異なる点 実用新案においては，発明の水
準をある程度高く維持しつつ，創作意欲の減退を防ぐため，きわめ
て容易に考案できた場合でなければ進歩性は否定されない．ただし，
国際的には両者の差異を明確にすることが困難なので，実体的には
特許一実用新案間の進歩性のレベルは同程度である．

iii．無審査登録制度が採用されている 実用新案においては，
ライフサイクルの短い考案の早期保護等のため，無審査登録制度（三
豊14条）が採用されており，図10・3に示したように方式的要件（実
第2条の2）と基礎的要件（実装6条の2）を具備していれば，新
規性，進歩性などの実体的要件の審査なしに設定登録される．すな
わち，実体的要件を具備していない場合でも権利が発生するが，当
該実体的要件を具備していない場合は無効理由を有することとなる
（実意37条）．
なお，基礎的要件の審査においては，保護対象，公序良俗，出願
の単一性，明細書等の記載不備を具備するか否かが判断される．（実
第6条の2）．
iv．補正 登録前の補正は，出願後2ヶ月に限り可能であるが，
特許の場合と同様に新規事項を追加する補正は許されない（実第2
条の2，37条）．特許制度に比して，補正できる期間が極めて短いた
め，完成度の高い明細書を作成しておく必要がある．
v．訂正が請求項の削除のみに限定されている点 無審査登録
制度のもと（実事14条），みだりに権利範囲を変更すると第三者に不
都合であるため，訂正は請求項の削除のみに限定されている（雪曇
14条の2）．なお，訂正の要件を判断する必要がないため，訂正審判
（特第126条）は設けられていない．
vi．存続期間が短い 実用新案の存続期間は出願の日から6年
である（実事15条）．なお，平成5年12月31日までに出願された実用
新案の存続期間は，従来通り （出願公告の日から10年．ただし，出
願の日から15年を越えることができない）なので，注意を要する．
vii．権利行使において実用新案技術評価書を提示した警告が義
務づけられている．
（1） 実用新案技術評価書 実用新案技術評価書とは，特許庁か
ら発行される実用新案登録を受けることができるか否かの評価であ
って，出願後であれば，何人でも請求可能である（実第12条）。
実用新案技術評価書においては，調査の負担を考慮し，一定の登
録要件のみが判断される．当該一定の登録要件とは，頒布刊行物に
基づく新規性，頒布刊行物に基づく進歩性，拡大された先願の地位，
先後願である（幽趣3条，実第3条の2，実第7条）．
（2） 権利行使の際の警告の義務 実用新案技術評価書を提示
して警告をした後でなければ権利行使することはできない（実第29
条の2）．
（3） 損害賠償責任 権利行使，または警告の後に実用新案登録
が無効になった場合，権利者が高度の注意義務を果たしていないと
判断されると，損害賠償責任が生じる場合がある（実第29条の3）．
高度の注意義務を果たしていると判断されるためには，登録を受
け得るものであるという肯定的な実用新案技術評価書を提示したの
みでは足りず，実用新案技術評価書が調査対象としていない他の登
録要件についても，自ら先行技術調査を行ったり，弁理士の鑑定を
依頼する必要がある．
さらに，登録後の訂正により，削除された請求項に係る考案につ
いて権利行使，または警告を行った場合も同様に損害賠償責任が生
じるので注意が必要である（実第29条の3）．
viii．訴訟中止申立権 侵害訴訟を提起された被告が無効審判を
請求したときは，申立てにより訴訟手続を原則として中止する（実
葛40条の2）．
特許法においては，審判と訴訟手続の関係について，審判が継続
しているときは必要に応じて訴訟手続を中止できることを規定して
いる（特第168条）．一方，実用新案法においては，無審査登録制度
を採用するため（実第14条），権利の有効性を前提としたまま訴訟手
続が進行することは被告にとって著しく不利である場合も生じ得る
ため，原則として訴訟手続を中止することとしている．
ix．特許法にあり，実用新案法にはない制度 上述のように実
用新案法においては無審査登録制度（実第14条）を採用しているた
め，以下のように特許法にはあるが実用新案法にはない制度が存在
する．
