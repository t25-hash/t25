# 3・2・1 傷害バイオメカニクスモデル

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・2 傷害のバイオメカニクス(injury biomechanics)の

事故時の人体の力学的応答や,傷害メカニズムおよび人体の耐性を研究するために,ボランティア,死体,動物,衝突ダミー,数学モデルなどの人体モデルが用いられる.以下ではそれらの特徴について述べる.a. ボランティア(human volunteer)ボランティア試験は,傷害や痛みを生じない程度の低い負荷のもとで実施され,無傷害時の人体の応答を明らかにできる.ボランティア試験の結果は,筋の緊張が動的応答に及ぼす影響を解明するうえで役に立つ.またその結果は,衝突ダミーや数学モデルを開発する際の参考となる.ボランティアには,スポーツ選手等,若くて訓練された者がなることが多く,子供,高齢者,女性等の弱者の特性を代表しないことに注意する必要がある.ボランティア試験の実施には,ヘルシンキ宣言(ヒトを対象とする医学研究の倫理的原則.世界医師会の合意による)の趣旨に沿って設置された倫理審査委員会(機関により名称が異なるので,以下では単に倫理委員会と呼ぶ)への実験計画書の提出や委員会の承認等が必要とされ,被験者に対するインフォームドコンセントの実施等,厳格な倫理的配慮がなされる.b. 死体(human cadaver)死体は幾何学的形状が生体と同じであることから,事故時の人体応答を解明するための試験によく用いられる.例えば車両衝突時の乗員の挙動や,車両と衝突する歩行者の挙動を解明する試験などに用いられる.ただし,生体組織の力学特性は死体の保存技術と死後の経過時間に依存することから,試験の実施に当たっては,死後間もない死体を用いるなどの注意が払われている.死体実験の結果は,衝突ダミーや数学モデルの開発の基準となるなど,重要な役割を果たす.この方法の問題点は,死体には筋の緊張と生理的応答がないため,その影響を知ることができないことである.また献体の死亡時の年齢は高く,かつ人体耐性は一般に年とともに低下することから,死体実験の結果は必ずしも実際の傷害を代表しない.死体実験の実施には,倫理委員会の承認等が必要であり,本人の事前の意思表示と家族の同意等の厳格な倫理的配慮がなされる.c. 動物(animal)動物実験は,脳や脊髄,内臓などの人体の重要な部位に,傷害が生じる程度の荷重が作用した場合の生体反応を知る目的で実施される.動物実験により,死体と生体の応答の違いが明らかとなり,死体実験の解釈に役立つ.動物とヒトとでは形状や大きさが異なるので,動物実験の結果をヒトに適用するには困難が伴う.なお,動物実験の実施に当たっても,倫理委員会の承認が必要であり,苦痛のレベル,屠殺の方法等,厳格な倫理的配慮がなされる.d. 衝突ダミー(crash test dummy)衝突ダミーは,金属またはプラスチックの骨格と,それを覆う軟組織を模擬したプラスチックまたは発泡材でできている.これらは形状,寸法や質量分布がヒトと同じであるように,かつ衝突時の運動がヒトと同様になるように,すなわち生体忠実性に注意を払って作られている.衝突ダミーには,傷害の判断に必要な加速度,力,たわみ等に関する計測機8―124生体工学器が備え付けられている.衝突ダミーを用いて車両の認証試験が行われるため,同一の試験においてダミーが同一の応答を示す反復性の要件を満足する必要がある.e. 数学モデル傷害バイオメカニクスを研究するために,集中質量モデル,マルチボデーモデル,有限要素モデル等の数学モデルが用いられる.衝突ダミーや人体がモデル化されており,事故再現や安全な車両構造,安全装置の設計などのために用いられる.実験では得られない力学パラメータに関する情報を得ることができるのが特徴であり,また車両や安全装置の設計支援ツールとしても有用である.数学モデルの精度は,モデル化の仮定に強く依存するので,検証が重要な意義をもつ.
