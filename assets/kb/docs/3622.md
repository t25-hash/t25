# 3・2・2 荷重による傷害過程のモデル化

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・2 傷害のバイオメカニクス(injury biomechanics)の

事故により人体は力学的負荷を受け,それによって人体部位に生体力学的応答が生じる.生体力学的応答は,人体部位の位置や形状が時間的に変化する力学的変化と,それに伴って生じる生理学的変化とからなる.例えば頭部衝撃時の脳の運動と変形は力学的変化の例であり,反射の変化,めまい,頭痛は生理学的変化の例である.傷害とは,生体に回復不可能な変形が生じて解剖学的組織・構造が損傷を受け,正常な機能を果たせなくなることをいう.傷害が発生する仕組みを傷害メカニズム(injury mechanism)という.傷害メカニズムの解明は傷害バイオメカニクスの中心的課題の一つである.傷害は鋭的傷害と鈍的傷害に分けられる.鋭的傷害は,ナイフのような鋭利な物体や,弾丸のような高速発射物の貫通によって引き起こされる.傷害を誘起するエネルギーが局所に集中するのが特徴である.鈍的傷害は,生体が鈍器と接触し,広範な領域で負荷を受けて生じる.傷害メカニズムは複雑であり,人体各部の慣性,弾性,粘弾性的性質が荷重分布に影響を及ぼす.この傷害メカニズムには次の三つが考えられている.(1) 圧迫により組織の弾性耐性を超えることによる傷害.変形速度が遅い変形(つぶれ)でも高速の変形でも生じる.(2) 体内に衝撃波が生じ,粘性耐性を超えることによって内臓等に生じる傷害.大きな外的変形がなくても生じる.(3) 慣性力によって内部組織が断裂する傷害.脳傷害の場合に重要なメカニズムである.傷害の程度は傷害の重症度(傷害程度という用語もよく使われる)で表現される.重症度と相関のある物理量あるいはそれらの関数を,傷害基準あるいは傷害値と呼ぶ.人体部位の並進加速度や合力,モーメント,たわみ等,衝突ダミーなどの生体代用物を用いる試験において容易に測定できる物理量が用いられることが多い.傷害基準がしきい値を超えると傷害が生じる.そのしきい値を傷害耐性という.傷害耐性は個人間で大きなばらつきがある.したがって,集団の傷害耐性を定めるためには統計学的な考え方が不可欠である.衝突ダミーや数学モデルに対しても生体力学応答が求められ,傷害基準が評価される.この傷害基準レベルを生体の傷害に換算することは傷害バイオメカニクスの重要な課題である.その際,衝突ダミーや数学モデルが生体と同じように応答すること,すなわち生体忠実性が問題となる.この問題は,モデルの応答をボランティア試験や死体実験の結果と比較することで判定できる.
