# 3・2・3 傷害スケール(injury scaling)

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・2 傷害のバイオメカニクス(injury biomechanics)の

傷害の種類と重症度を数値的に表現するために、傷害スケールの概念が用いられる。傷害スケールは、解剖学的スケール、生理学的スケール、機能障害・能力障害・社会的損失スケールの三つのグループに大別される。解剖学的スケールは解剖学的部位、傷害内容、重症度によって、傷害の結果よりも傷害自体を評価するものであり、機能障害、後遺症については言及しない。生理学的スケールは受傷後の生理学的機能変化を評価する。解剖学的スケールではある傷害に一つのスコアを割り当てるのに対し、生理学的スケールでは治療期間中の生理学的状態の変化に応じてスコアが変わっていくのが特徴であり、臨床の場においてよく用いられる。機能障害・能力障害・社会的損失スケールは傷害自体や機能変化ではなく、長期予後や生活の質の観点から、傷害に経済的価値を当てはめる。傷害優先度評価IPR (injury priority rating)、傷害(harm)の概念、傷害コストスケールICS (injury cost scale)がその例である。傷害バイオメカニクスにおいて最もよく用いられるのは解剖学的スケールの一つであるAIS(abbreviated injury scale)である。1960年代中ごろに、人体傷害発生メカニズムを検討できるデータの提供を目的として、アメリカの交通事故調査チームが作成に取りかかったのが始まりであり、最初のAIS は1971年に発行された。最新版はAIS 90 update98である。AIS コードは傷害1箇所につき一つ割り当てられ、整数部6けたと、傷害重症度を表す小数点以下1けたの7けたの数値で表される。最初の1けたは傷害部位の大分類であり、頭部:1、顔面:2、頸部:3、胸部:4、腹部および骨盤内臓器:5、脊椎:6、上肢:7、下肢:8、その他:9、のように各部位に数値が付与される。2けためは傷害部位の解剖学上の構造であり、部位によって若干異なるが、全域:1、血管:2、神経:3、内臓:4、骨格:5、意識喪失:6、を基本に分類されている。3~4けためは傷害の内容、5~6けためは詳細な傷害部位および傷害重症度の記述を表している。小数点以下の数値はどの部位にも共通の傷害重症度を表し、AIS スコア1~6に分類している。AIS 1は軽傷、2は中等症、3は重症、4は重篤、5は瀕死、6は即死である。このほか、外傷は起きているが特定の器官や部位に関する詳細な情報がない場合に対して、コード9:不明を割り当てている。AIS スコアは生命に対する危険度を順序付けした定性的な測度であって、数値そのものには定量的な意味はない。AIS は1箇所の傷害に対するコードであり、一個人が複数箇所受傷する多発外傷を評価できない。多発外傷を評価する方法として最も受け入れられているのはISS(injury severity score)である。ISS では、人体を次の6部位に分ける。1:頭部あるいは頸部、2:顔面、3:胸部、4:腹部あるいは骨盤内臓器、5:四肢あるいは骨盤、6:皮膚。そして各部位において最も高いAIS スコアを求め、スコアの高い3部位についてその二乗和を求めればそれがISS となる。ただし1箇所でもAIS が6の場合には、ISS は最高値75(AIS-5が3箇所の場合)をとると規定している。ISS と死亡率の相関性は非常に高いと報告されている。なお、AIS とISS とでは部位の分け方が異なるので、注意が必要である。
