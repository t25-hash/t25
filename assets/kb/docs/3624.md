# 10・4・2 商標の類似 ある商標ともう一方の商標が類似する・

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・4 商 標

かどうかという判断は，商標法において非常に重要である．商標に
ついての類似とは，商品を購入する消費者がその商標が付された商
品の出所などを混同するおそれがある程に紛らわしいか否かを問題
とする．
例えば，商標権の侵害の有無が問題となる場面では，自分の使っ
ている商標が他人の登録商標と同一の場合はもちろん，類似する場
合にも，その他人の商標権を侵害していると判断され，また，出願
段階においても自分の出願した商標が他人が既に所有している商標
と類似する場合には，その商標については登録を受けることはでき
ない．
商標が類似する場合とは，商標自体（商標を構成する文字・図形
など）が類似し，かつ，その商標が使用される商品や役務が類似す
る場合をいう．したがって，例えば商標自体が同一又は類似のもの
であっても，商品が非類似であれば，両商標は抵触しないことにな
る．
商標自体の類否は，具体的には，それら文字・図形・立体形状な
どが有する外観（視覚），称呼（聴覚），、観念（意味）の3要素のう
ち；いずれかが同一又は類似であれば類似と判断される．
