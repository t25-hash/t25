# 3・3・1 頭部の解剖学

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・3 頭部傷害のバイオメカニクス

頭部は大別して頭皮、頭蓋骨、脳によって構成される。位置としては、前頭部、側頭部、後頭部、頭頂部として区分される。図3・7には頭部の断面構造とその層組織を示す。頭蓋骨は三層に分かれ、外板と内板は皮質骨、板間層は造血組織を含む海面骨である。頭蓋骨と脳の間にある膜組織は、表層部から脳に向けて硬膜、クモ膜、軟膜と呼ばれる三層で構成されている。硬膜は脳を包む最も厚い膜組織であり、クモ膜は硬膜の内面にある膜で、軟膜は脳に密着した最内層の薄い膜である。クモ膜と軟膜の間はクモ膜下腔と呼ばれ、脳脊髄液で満たされている。脳実質は神経細胞が主体となっている灰白質と、神経線維の集合体である白質で成り立っている。
