# 3・3・3 傷害発生のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・3 頭部傷害のバイオメカニクス

頭部傷害の発生メカニズムの解析では,サルやラットのような動物やヒト死体を用いた医工学実験,また模型実験や有限要素解析のような工学的解析法が適用されている.脳挫傷は古くからその発生メカニズムが注目されている頭部傷害であり,さまざまな研究が行われている.衝撃直下に生じる同側損傷については,衝撃位置の頭蓋骨がたわみ,さらに陥没骨折を起こすことによって脳表面が直接損傷を受けるためであるとされている(図3・10).一方,衝撃と反対側に脳挫傷が発生する対側損傷の発生メカニズムについては諸説があり,高度の陰圧が対側に発生するとのキャビテーション説(図3・11),脳実質の回転により対側脳組織にせん断力が発生するとの回転運動説,頭蓋骨の凹凸が影響するとの頭蓋構造説などがよく知られている.びまん性軸索損傷の発生メカニズムに関しては,オマヤ(Ommaya)とジェネラリ(Gennarelli)が神経路の損傷が大脳半球表層部から脳深部へと進展するとした求心性の意識障害説を提唱している.その後ジェネラリらがサルを供試体とした回転衝撃を負荷する動物実験(図3・12)により,びまん性軸索損傷による病理変化として大脳白質全域の神経線維の断裂と,重度の場合には脳深部へ出血が起こることを示した.そのため,自動車安全や脳神経外科学の分野では,びまん性軸索損傷は頭部へ回転衝撃が負荷されることが原因であるとの学説が広く流布されている.しかし,脳の有限要素モデルにより応力集中を解析し(図3・13),直衝撃図3・10 同側損傷のメカニズム図3・11 対側損傷のメカニズム(キャビテーション説)でも回転衝撃でも小出血が脳深部に生じることを示して,びまん性軸索損傷で生じる脳深部の小出血が大脳鎌のような頭蓋内構造によるものであるとした見解などもあり,さらに詳細にびまん性軸索損傷のメカニズムを研究する必要がある.
