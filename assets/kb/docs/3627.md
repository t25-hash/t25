# 3・3・3 傷害発生のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・3 頭部傷害のバイオメカニクス

頭部傷害の発生メカニズムの解析では，サルやラットのような動
物やヒト死体を用いた医工学実験，また模型実験や有限要素解析の
ような工学的解析法が適用されている．脳挫傷は古くからその発生
メカニズムが注目されている頭部傷害であり，さまざまな研究が行
われている．衝撃直下に生じる同側損傷については，衝撃位置の頭
蓋骨がたわみ，さらに陥没骨折を起こすことによって脳表面が直接
損傷を受けるためであるとされている（図3･10)．一方，衝撃と
反対側に脳挫傷が発生する対側損傷の発生メカニズムについては諸
説があり，高度の陰圧が対側に発生するとのキャビテーション説
（図3･11)，脳実質の回転により対側脳組織にせん断力が発生す
るとの回転運動説，頭蓋骨の凹凸が影響するとの頭蓋構造説
などがよく知られている．
