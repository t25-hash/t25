# 3・3・4 傷害基準と衝撃耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・3 頭部傷害のバイオメカニクス

傷害の基準を策定しようとする場合、最も軽度の傷害を目安にして基準を定めれば、より安全側に限界を設定できることから、頭部傷害では軽度の傷害として脳震盪が着目された。脳震盪はびまん性脳損傷に分類される一過性の意識障害である。脳震盪を人間に惹起させる実験は不可能なので、線状頭蓋骨骨折の80%は脳震盪を伴うとの理由から、リスナー(Lissner)やガージアン(Gurd-jian)がヒト死体の落下実験により頭蓋骨骨折発生の臨界点を求め、さらにその実験をin vivo のイヌ衝撃実験と比較し、WSTC(wayne state tolerance curve)と呼ばれる頭蓋骨骨折の耐性曲線が求められた。これを図3・14に示す。この耐性曲線は有効加速度とその持続時間との関係で整理されており、頭蓋骨骨折すなわち脳震盪の発生限界を表しているとされている。耐性限界を超図3・12 びまん性軸索損傷を発生させた動物実験(a) 回転衝撃(b) 直衝撃図3・13 脳の有限要素モデルによる衝撃応力分布図3・14 頭部衝撃の耐性曲線えた領域では、何らかの頭部傷害が発生するとみなしている。このWSTCに基づいて、加速度とその持続時間の関係を直線近似することが考案され、式(3・1)に示す頭部の傷害基準(HIC: headinjury criterion)が自動車製造における安全性能評価で用いられるようになっている。HIC=Max-욧d-<1000(3・1)ここでは頭部の合成加速度(G)、,は衝突中の任意の時間であり、-≦36msで評価され、傷害発生限界をHIC<1000と定めている。HIC<1000を頭部傷害の判定基準として、衝突実験用ダミーの頭部重心加速度により計算されるHICが1000以下であるならば頭部傷害が発生しないと評価している。しかしながら、最近では頭蓋骨骨折と脳震盪が15ms以上では発生しないとの実験や、被験者を用いた自動車エアバッグの拘束実験においてHICでは傷害を過大に評価しているとの報告に基づいて時間区間を-≦15msとし、HIC<700を評価発生限界とする考え方が妥当とみなされつつあり、傷害評価方法が改良される可能性がある。耐性曲線(WSTC)に基づく傷害基準(HIC)は自動車の試験法に採用された最も代表的な頭部傷害の評価方法である。頭部に加わる有効加速度とその持続時間の関係を表したWSTCは、頭蓋骨骨折の大半が脳震盪を伴うことから脳震盪の限界を示しているとみなされている。一過性の意識障害である脳震盪は、頭部傷害の中では最も軽い傷害であると考えることができるので、脳震盪を目安に基準を作ることは最も安全側に限界をおいて自動車安全を評価していることになる。しかし、頭部傷害を主原因とした死傷者は依然として多く、生命を司る最も重要な器官の一つである頭部の傷害を防御あるいは軽減する研究は今後さらに重要となっていくものと予測される。
