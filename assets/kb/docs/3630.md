# 3・4・2 傷害のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・4 頸部傷害(neck injury)のバイオメカニクス

頸部は、3・4・1項に示したように構造的に複雑な形状をしており、強度の異なる骨格、靱帯、血管・筋・神経などの軟部組織から構成されている。また、その形状や固さは意識の状態、つまり筋力によってもかなり異なる。衝撃は直接椎体に伝わるか、あるいは軟部組織を経由して分散されて下方の体幹に伝わる。頭部衝撃による頸部への影響は、衝撃の方向と頸部の向きによって異なる。頸部の傷害メカニズムは、(1)頭部への衝撃が直接頸部へ伝わるか、(2)頸部を支点とする頭部の慣性運動に関連して頸部に傷害が発生するか、のどちらかに分類される。前者の場合、衝突時の頸部の向きにより、その影響は千差万別となる。頸椎には、曲げ(屈曲-伸展)、圧迫(圧縮)、引張り、回転(ねじり)、せん断のいずれかの力が加わる(図3・31)。通常、曲げ(屈曲-伸展)による損傷部位は下位頸椎である。しかし、せん断力が作用し、頸部の向きによっては、上位頸椎のハングマン(Hangman)型骨折(せん断力による軸椎歯突起骨折など)やジェファーソン(Jefferson)型骨折(C1椎骨の環状骨折)などの傷害が生じる。屈曲によって椎間板や椎体前縁に損傷が発生するが、伸展の場合よりも曲げモーメントは大きくなる。ただ、屈曲の場合では顎部が胸部に接触(chin-chest図3・31 頸椎への負荷形態圧縮による傷害引張りによる傷害ねじりによる傷害ジェファーソン型(C1環状)骨折第1頸椎の複雑骨折椎体圧迫骨折粉砕骨折圧縮と屈曲による傷害引張りと屈曲による傷害せん断による傷害椎体楔の圧迫骨折過屈曲による捻挫一側性の脱臼両側性の脱臼teardrop型骨折圧縮と伸展による傷害引張りと伸展による傷害曲げによる傷害後方骨格系の骨折むち打ち椎間関節の裂離椎間板のはく離椎体の骨折ハングマン型(滑り)骨折teardrop型(過屈曲)骨折その他clay shovelers型(棘突起)骨折8―129impact)し、椎体に作用する負荷が変化する。後者の場合、頭部には直接衝撃を受けない、いわゆるむち打ち(whiplash)態様による損傷がある。典型的な衝突として追突がある。追突を受けると、まず、衝突の初期には、乗員はシートバックの反力を受け、円背化している脊椎が直線化を起こすと同時に体幹がシートバック面に沿って上昇する。頭部は慣性で初期の位置にとどまっているため、脊椎の直線化(spine straightening)と体幹の上昇(torsoramping-up)により下位頸椎に軸圧縮力が作用する。さらに、その後、シートバックと体幹上部との衝突により頸部下部にせん断力が作用し、次いで、下部頸椎を支点とした頭部の後屈回転運動が発生する。最近ではヘッドレストが装備されているため、衝突初期から脊椎・体幹の動き、さらに頭部の回転運動までの間での衝突現象は複雑となっている。一般に、頸部傷害の発生する衝撃時間域は数十msから数百msのオーダであり、傷害度は作用時間にも依存する。むち打ち傷害の発生する作用時間も200msに及ぶ場合もあるが、そのメカニズムはいまだ解明されているとはいいがたい。
