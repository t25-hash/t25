# 10・5・1 著作権法 著作権法は，著作物並びに実演，レコー

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・5 その他の法律

ド，放送及び有線放送に関し著作者の権利及びこれに隣接する権利
を定め，これらの文化的所産の公正な利用に留意しつつ，著作権者
等の権利の保護を図り，もって文化の発展に寄与することを目的と
している（第1条）．著作物は思想又は感情を創作的に表現したもの
であって，文芸，学術，美術又は音楽の範囲に属するものをいう（第
2条の1）と定義されている．したがって，電子計算機を機能させ
て一の結果を得ることができるようにこれに対する指令を組み合わ
せたもめとして表現したプログラム（第2条第1項第10号の2）や
論文，数値，図形その他の情報の集合物であって，それらの情報を
電子計算機を用いて検索することができるように体系的に構成した
データベース（第2条第1項第10号の3）は著作物に該当する（第
10条第1項第9号，第12条の2）．
著作権の存続期間は著作物の創作のときに自動的に始まり （第51
条），原則著作者の死後50年存続する．このような長期間の保護は，
模倣により創作者が十分越費用回収ができなくなることで創作活動
が衰退しないように定められている．
品質の劣化もなしにコンピュータプログラムは比較的コピーしゃ
すいので法的保護が必要である．
著作権では第三者が独自に開発したプログラムは権利が及ばない
ので，ソフトウェア全体の知的財産保護の観点では，プログラムは
著作権で保護するが，アイデアとコンピュータ制御手順を定めるア
ルゴリズムは特許権で保護するようにするとよい．著作権は自動的
に権利が発生するので一般に登録を要しないが，プログラムは改訂
ごとの履歴を残すこと及びその創作日の立証が難しいことから，後
日の争いをなくすためにソフトウェア情報センター（SOFTIC）に
登録しておくのがよい．
