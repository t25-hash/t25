# 10・5・4 民事訴訟法 民事訴訟とは，原告の申立に基づき，

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・7 第三者の知的財産への対応

民事の法律上の争訟につき，裁判所が対立当事者の主体的な関与の
下に事実を認定し，法を適用して権利・法律関係の存否を判断し，
その判断内容に裁判所及び当事者を拘束し，給付を命じ，または新
たな権利関係を形成する判決をするための手続である．
この手続に関し，平成11年の法律改正により，争点整理手続の整
備，証拠収集手続の拡充，小額訴訟手続の新設，最高裁判所に対す
る上告制度の整備がなされた．その中で特に証拠収集に関し，文書
提出命令が重要である．
公害・環境・医事事故・薬害・製造物責任といった訴訟では，一
般に被告である一方の当事者に証拠が偏在しており，原告である他
方の当事者による事実解明の障害になっていた．そこで，企業・私
人が所有する文書については原則として提出義務がある（第220条4
号）と定めて，それらの証拠を提出させようとしている．もっぱら
所持者が利用する文書（同工ハ）や技術もしくは職業の秘密に関す
る事項を記載した文書（同号ロ）等の例外を除き提出が必要となっ
た．文書提出を拒否する理由の妥当性を裁判官が判断するインカメ
ラ審理手続（第223条三項）や文書提出命令に違反したときは相手方
の主張を真実とみなす条件を拡大（第224条第2，3項）した．
