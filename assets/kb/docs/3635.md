# 3・5・3 傷害基準と傷害耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・5 胸部傷害(thoracic injury)のバイオメカニクス

胸部傷害耐性は多くの研究者により,防腐処理死体(embalmedcadaver),未防腐処理死体(unembalmed cadaver),動物およびボランティアを用いて研究されてきた.おもな試験方法は,インパクタ試験,落下試験,スレッド試験および実車試験が用いられてきた.衝撃耐性パラメータとしては加速度,力,胸たわみ(圧縮変形),粘性基準(胸たわみと胸たわみ速度の積)が提案されてきた.8―131a. 加速度耐性(acceleration tolerance). 前面衝撃メルツとガッド(Gadd)は,加速度計を装着したスタントマンが,17.4mの高さから背中を下にしてフォームマットレス上に落下し,46G の胸部加速度に耐えられたと発表した.ヴィアノ(Viano)らは,熟練者ボランティアを10.5mの高さからプールの水面に腹を下にして胸部加速度を測定した.この結果から68G に耐えられることを報告している.これらの結果をもとに,アメリカ自動車安全基準FMVSS 208項では胸部の許容加速度パルスを60G,加速度持続時間3ms以下と規定している.. 側面衝撃1980年代に自動車の側面衝突の乗員胸部傷害耐性を求める研究が進んだ.アメリカのNHTSA(NationalHighway Traffic Safety Administration)がスポンサーになり,ハイデルベルク大学で広範なスレッド試験が行われた.未防腐処理死体27体をスレッドに固定した低摩擦面シートに座らせ,パッド付きまたはパッドなしの側壁に衝突させた.肋骨,胸骨および胸椎で合計12個の加速度を計測した.エッピンジャー(Eppinger)はこれらのデータを解析し,胸部傷害指標(TTI: thoracictrauma 粘性基準(viscous criteria). 前面衝撃ラウ(Lau)とヴィアノは,胸部軟組織傷害は,圧縮変形量と圧縮変形速度の積の最大値윯율융と相関があると考え,粘性基準윯율융を提案した.その粘性基準の有効範囲を図3・36に示す.すなわち胸部傷害のうち特に軟組織傷害は変形速度の影響を考慮する必要があり,変形速度が3m/s以下では変形速度の影響は小さく,圧縮変形量に基づく圧縮耐性が最良の指標であり,傷害は圧迫傷害(crushing injury)となる.変形速度が30m/sを超えると変形速度が支配的になり,圧縮変形量の影響は二次的になるが,火薬の爆発による爆風傷害(blast injury)はその領域の傷害である.自動車衝突事故による軟組織傷害はそれらの中間の領域であり,圧縮変形量と圧縮変形速度の両方を考慮する必要があり,この領域における傷害を粘性傷害(viscous injury)と呼んでいる.ラウとヴィアノはクローエルおよび他の研究者が行った39体の未防腐処理死体の胸骨衝撃試験結果をプロビット解析し,윯율융=1.0m/sの場合AIS>3の傷害が25%生じ,윯율융=1.3m/sの場合AIS>3の傷害が50%生じると報告した.この耐性は,現在,ヨーロッパの安全基準に採用されている.. 側面衝撃キャバナー(Cavanaugh)らは,死体を図3・36
