# 3・5・3 傷害基準と傷害耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・5 胸部傷害(thoracic injury)のバイオメカニクス

胸部傷害耐性は多くの研究者により，防腐処理死体（embalmed
cadaver），未防腐処理死体（unembalmed cadaver），動物および
ボランティアを用いて研究されてきた．おもな試験方法は，インパ
クタ試験，落下試験，スレッド試験および実車試験が用いられてき
た．衝撃耐性パラメータとしては加速度，力，胸たわみ（圧縮変
形），粘性基準（胸たわみと胸たわみ速度の積）が提案されてきた．
a.加速度耐性（acceleration tolerance）

容加速度パルスを60G，加速度持続時間3ms以下と規定している．

c .粘性基準（viscous criteria）
.前面衝撃 ラウ（Lau）とヴィアノ は，胸部軟組織
傷害は，圧縮変形量 と圧縮変形速度 の積 の最大値
と相関があると考え，粘性基準 を提案した．その粘
性基準の有効範囲を図3･36 に示す．すなわち胸部傷害のうち特
に軟組織傷害は変形速度の影響を考慮する必要があり，変形速度が
3m/s以下では変形速度の影響は小さく，圧縮変形量に基づく圧縮
耐性が最良の指標であり，傷害は圧迫傷害（crushing injury）とな
用いた側面衝突試験をもとに，AIS≦2の胸部傷害が≦1m/
sで発生し，AIS4～5の傷害が≧1m/ sで発生するといって
いる．さらに，圧縮耐性と粘性基準は加速度耐性，荷重耐性
（force tolerance）に比べて傷害との相関がよいことを指摘してい
る．またヴィアノ はAIS4以上の傷害が＝1.47m/sを超
えると25 %の確率で発生することを報告した．
