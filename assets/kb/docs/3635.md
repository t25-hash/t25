# 10・6・2 第三者への実施許諾 権利者は自己実施いかんに係

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・5 その他の法律

らず，第三者に実施権を許諾して，ライセンス収入を上げる．
権利者が積極的にライセンス先を求めて交渉を申し入れるケース
や，第三者から条件合意を前提に許諾を申し入れるケースがある．
また，特許流通の仲介をするウェブサイトや市場に掲示することも
なされてきている．
ライセンスの条件を定める場合に，一括ライセンス，不争義務，
研究開発活動の制限，改良発明の譲渡，特許権消滅後の技術の使用
制限等，ライセンシーの事業活動に関しさまざまな制限を加えよう
としたとき，独占禁止法にもとづき特許法等による権利行使とみな
されない行為と判断されると不公正な取引きと判断されてしまう．
