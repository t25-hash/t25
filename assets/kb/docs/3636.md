# 3・6・1 腹部の解剖学

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・6 腹部傷害(abdominal injury)のバイオメカニクス

腹部は横隔膜と骨盤の間の部分である.横隔膜は,ドーム形状をしており,第4肋軟骨が胸骨と交わる位置まで上がっている.腹部の前面は数層の筋肉から成り立っており,背面は五つの腰椎(L1からL5)と数対の強い筋肉から成り立っている.腹腔の内壁および腹部臓器の外表面は,薄い半透明の腹膜で包まれている.腹部内臓は機能別に,消化器系,泌尿器系,内分泌系および神経系に分けられるが,ここでは構造的な見地から腹部臓器を中実臓器と中空臓器の二つのタイプに分けて記述する.中実臓器は,主として血液循環の機能をもち,肝臓,脾臓,腎臓,副腎および卵巣を含む.中空臓器は,消化機能を有する器官であり,胃,小腸,大腸,膀胱および子宮を含む.上腹部は肝臓,胃,脾臓,膵臓,十二指腸,腎臓の一部を含む.上腹部にある臓器のうち,肝臓は右側肋骨郭下部で囲まれた空間の大部分を占め,脾臓と胃は左側肋骨郭下部の空間を占めている.膵臓は腹部後壁上中央部にあり左右に横断している.中腹部には,幽門横断面から腸骨稜の隆起横断面の間にある臓器(上行結腸,横行結腸,小腸の一部,腎臓の一部,妊娠子宮)を含む.下腹部は,腸骨稜下のすべての腹部臓器(盲腸,S 状結腸,膀胱,小腸の残りの部分,子宮)を含む.下腹部の境界は,前方が筋組織,横方向は腸骨,後方は仙骨と尾骨である.腹部の大血管は腹部大動脈と下大静脈であり,それらは腹腔の背側にあり,やや右側を走っている.
