# 10・7・1 侵害の予防

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・6 知的財産の活用

a．新製品・新技術研究開発時の侵害可能性調査 新製品や新技
術を開発するときに同じテーマの特許情報を収集して，既存の権利
状況や出願状況を知り，あらかじめ権利侵害を回避するために調査
する．’N害可能性のある特許が抽出された場合は，出願経過書類を
取り寄せて権利範囲を確認し，登録原簿から権利が有効であること
も確認する．
開発計画の見直しやライセンス取得等の方針を開発初期に明確に
することで無駄な開発が避けられる．また侵害が無いと分かれば，
自社で特許網を構築する対応も図れる．
b．製品の販売・輸出時の侵害可能性調査 新製品の発売にあた
りその製品・製法が他社の権利を侵害する可能性について調査する．
特許権は属地主義を採っているために，製品を輸出する場合は，輸
出国における権利の侵害の可能性について，輸出相手国における調
査も必要である．
開発初期の調査から発売時までの期間が長い場合は，未公開であ
った出願が公開されている場合があり，再確認をするのがよい．早
期審査制度と共に，審査期間が短縮されていることから，公開段階
であっても十分な注意が必要である．
