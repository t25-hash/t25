# 3・6・2 傷害のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・6 腹部傷害(abdominal injury)のバイオメカニクス

アメリカの自動車事故統計データベース(NASS/CDC:National Accident Sampling System/Crashworthiness DataSystem,1994~1996年)によると,自動車事故における腹部傷害は,ステアリングホイール,シートベルト,ドアなどに激しく衝突した場合に生じる.傷害を受ける臓器は多い順に,肝臓,脾臓,消化器,腎臓,血管である.腹部臓器の傷害発生メカニズムはあまり明確にはされていないが,現在わかっている範囲で述べる.腹部臓器の傷害メカニズムに影響を与える特徴としては以下が挙げられる.中実臓器の密度は中空臓器に比べて大きい.腹腔の中の臓器は薄い腹膜で包まれている.この膜は漿液で潤滑されているため,臓器は腹腔内で剛に固定されておらず可動性が高い.このため,体の姿勢や向きによって各臓器が相対的に位置を変えられる.臓器の位置もまた傷害メカニズムに影響を与える.脊柱の前にある臓器は側面衝突時に比べ前面衝突時に圧迫を受けやすい.右の腎臓は,肝臓が右側にあるため,左の腎臓よりも若干低い位置にあり,傷害を受けやすい.さらに,下部肋骨郭の内側にある腹部臓器は肋骨により保護されるので多少余裕がある.a. 肝臓肝臓は腹部臓器の中で最大の中実臓器であり,鈍的衝撃による傷害を受けると死亡率が最も高い.軽い傷害では,被膜下血腫または浅在性裂傷があり,これらは傷害が実質まで広がっていない.重傷の場合は,実質が圧潰または破裂する.これら鈍的外傷の発生メカニズムは,(1)低速衝撃中に肝臓が脊柱や後腹壁に圧縮される,(2)高速衝撃中に腹腔内部の圧力が高くなり,過大な引張ひずみやせん断ひずみが生じる,(3)他の臓器との相対運動により,肝臓が支持されている靱帯,血管との結合点で引張強度を超える応力が発生する,および,(4)骨折した肋骨が肝臓を突き刺して裂傷を生じる,などが考えられる.また,心臓など肝臓から遠くにある臓器に衝撃が作用したときに肝静脈に大きな圧力が作用し,肝臓鬱血が生じるという報告もある.b. 脾臓脾臓は血管が多い器官で,腹部の左上部後方にあり,横隔膜の下で,胃の左側に位置している.血管が多く集まっているため,外傷を受けた場合,死亡率が高い.傷害は被膜下血腫として観察されるが,実質が裂傷を受けたり,切断される場合もある.この場合には大量の出血やショックを引き起こす.傷害発生メカニズムは,(1)左上腹部への直接的な打撃により実質に深い裂傷や圧潰が生じる,(2)急激な減速時に脾臓が大きく変位し,脾臓が取付いている椎弓根(pedicle)位置で引き裂かれる,の二つが報告されている.(1)のメカニズムは低速衝撃における圧迫メカニズムと高速衝撃時の粘性メカニズムにより発生する.c. 腎臓腎臓は第12胸椎と第2腰椎の間にあり,腹膜の後ろにある.腎実質は強い,粘着性の線維被膜で覆われており,それ自身は脂肪層の中にある.腎挫傷と浅い皮質性裂傷は,被膜が無傷であれば軽傷と思われる.ほかには,腎椎弓根傷害,深い実質裂傷がある.これらの傷害は急激な減速時の大きな相対運動によって生じると考えられている.8―133d. 膵臓と十二指腸膵臓と十二指腸の傷害は少ない.膵臓傷害は浅層挫傷から完全切断および離解まで広範囲にわたる.これらは膵臓が脊柱と衝撃物の間で圧迫されるときに生じる.十二指腸の傷害は出血や穿(せん)孔である.その傷害発生メカニズムとしては,(1)脊柱に衝突して圧縮される挫傷,裂傷,(2)腹腔内固定部でのせん断による引裂き,(3)衝撃中に両端が挟まれたことに起因する,内圧増加による破裂,が考えられる.e. 腸自動車衝突事故における腸傷害は多い.結腸傷害のほとんどは穿孔傷害である.鈍的腸傷害は,腸壁の挫傷,穿孔と切断,腸間膜裂傷に分類できる.(1)前部の腹壁と脊柱・肋骨・骨盤に挟まれることによる腸の圧潰,(2)斜めに作用する力により引き裂かれる,(3)空気,液体が詰まった管腔内圧力上昇による破裂,(4)急な減速を受けたときの大きな相対運動によるせん断と引裂き,(5)骨盤骨折,仙腸骨骨折時の穿孔,などのメカニズムにより生じる.f. 膀胱膀胱は骨盤によりよく保護されており,傷害はそれほど多くないが,合併症を伴う場合は死亡率が高くなる.傷害部位はおもに円蓋部と側壁部である.傷害メカニズムは,(1)骨盤骨折片が膀胱壁を突き刺す,(2)膀胱が膨張しているときに下腹部に衝撃が加わると,内圧が高くなり,円蓋部,側壁部の破壊強度を超えたときに破裂する.g. 横隔膜おもな外傷は横隔膜の破裂である.その発生メカニズムとしては,(1)膜伸張時のせん断および周辺固定部における裂離,(2)腹部臓器を通して作用する伝ぱ圧力による破裂,が考えられる.h. 子宮妊娠していない子宮は骨盤により保護されており,めったに傷害を受けないが,妊娠子宮は大きな形状をもち,液体などで満たされているため傷害を受ける可能性が高い.おもな傷害としては,子宮破裂,胎盤はく離,胎盤裂傷がある.これらの傷害の発生メカニズムは,衝突時に,車両内部の構造,シートベルトの不適正な使用などにより圧迫され,子宮内圧力が高くなることである.i. 大血管腹部大血管は脊柱の近傍にあり,よく保護されている.傷害は極めてまれであるが,腹部大動脈の外傷は致命傷となる.外傷としては挫傷,切断,はく離がある.傷害発生メカニズムとしては,(1)横力が作用した場合の血管のせん断,(2)脊柱骨折による裂傷,(3)血管の内圧増加,(4)急減速時に内臓が大きな運動をしたときの血管の裂離,などが挙げられる.
