# 3・6・3 傷害基準と傷害耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・6 腹部傷害(abdominal injury)のバイオメカニクス

腹部が衝撃荷重を受けたときの耐性を知るために、サル、ブタ、ウサギ、イヌなどの動物および人間の死体を用いて多くの実験がなされてきたが、腹部は人体各部位の中で、最も耐性が理解されていない部位である。頭部、胸部と同様に加速度耐性と傷害との相関が調査されたが、多くの研究者は、脊柱で計測した加速度と傷害の相関はよくないことを認識している。1970年代から腹部傷害耐性の研究が進み、荷重、変形、圧力、粘性などに基づく種々の耐性が提案されてきた。提案されている腹部衝突傷害耐性の詳細は文献(54)に記述されているので、ここでは代表的なものについて記述する。a. 荷重耐性スタルネーカーらは、4種類、96匹の動物(アカゲザル、リスザル、ヒヒ、ブタ)および人間の死体について、形状、大きさの異なるインパクタを用い、腹部の衝撃部位、衝撃方向を変えた広範な実験を行った。衝撃部位は上腹部、中腹部、下腹部である。膨大なデータを統計的に処理し、衝撃力、持続時間、体重、インパクタ接触面積と傷害との相関を調査した。動物の大きさと体重を次元解析により人間に合わせ、傷害尺度(ESI:estimated severity of injury,AIS 傷害スケールとほぼ等価)との関係を次式で近似できることを報告した。上腹部で傷害が最も発生しやすく、下腹部で発生しにくいことを示している。b. ベルト荷重耐性(belt loading tolerance)ウォルフィッシュ(Walfisch)らは、死体14体について、ラップベルトが骨盤の上部にかかるようにして一連のスレッド試験を行った。ベルト荷重と腹部侵入量を測定して傷害との対応を調査した。傷害は腰椎破壊4件、腸間膜引裂き2件、肝臓傷害2件、骨盤破壊1件、結腸穿孔1件であった。この結果から、腹部の控えめな耐性としてベルト荷重2kN、ベルト侵入量35mmを提案した。ミラー(Miller)は、24頭のブタの下腹部にラップベルトを取付けたインパクタによりラップベルト衝撃(速度2~7m/s、圧縮量6~70%の組合せ)を加えた。傷害発生頻度としてED(effective dose、有効量)を用いて結果を述べた。ここで、EDは全試験数のうち25%で傷害が発生したことを示す。AIS>3の腹部傷害がピーク荷重2.93kN(ED)、3.96kN(ED)で発生し、AIS >4の傷害がピーク荷重3.76kN(ED)、4.72kN(ED)で発生したと報告している。c. 圧縮耐性。 前面衝撃ラウとヴィアノは、後述d項のウサギの肝臓傷害実験結果を圧縮量で整理し、AIS>3の肝臓傷害が圧縮量16%(衝突速度12m/s時)で発生することを示した。ミラーは、前述b項の24頭のブタのラップベルト衝撃試験結果から、下腹部のAIS>3の傷害は圧縮37.8%(ED)、48.4%(ED)で発生し、AIS>4の傷害が圧縮48.3%(ED)、54.2%(ED)で発生すると報告している。. 側面衝撃スタルネーカーらは、前述a項の人体死体の左右腹部に、アームレストをストライカに取付け衝撃を加えた実験から、上腹部の傷害(ESI≧3)と圧縮量との関係を求めた。その結果ESI≧3の腹部傷害耐性として左腹部は圧縮変形量60%、右腹部は圧縮変形量54%と、右側の腹部の耐性が低いことを報告している。d. 粘性基準1981年に、ラウとヴィアノはウサギを用いた実験において、腹部圧縮変形量を一定に保ち、衝撃速度を増加させると肝臓傷害が著しく増加することを見つけた。その後、軟組織傷害の耐性研究において、衝撃速度または変形速度を考慮した傷害図3・38 ブタのステアリングホイール衝撃試験図3・39インパクタ試験クライテリアの研究が進展した。後者は粘性基準(viscous criteria)と呼ばれている。ヴィアノは、粘性基準は高速衝撃における内臓などの軟組織傷害に対し最も相関の高い傷害基準であると述べている。e. 腹部傷害耐性のまとめ腹部耐性実験は人体の腹部と構造、大きさが似ているブタを用いた動物実験に基づいて粘性基準が提案されている。腹部傷害に対する安全基準は、まだデータが少ないことや測定装置が開発されていないことから法制化されていない。腰部を構成する部位は、腰椎、骨盤および下腹部軟組織である。下腹部軟組織については3・6節で述べた。ここでは骨盤のバイオメカニクスについて述べる。
