# 3・7・1 骨盤の解剖学

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・7 腰部傷害(lumbar injury)のバイオメカニクス

骨盤は脊柱と下肢の間にある輪状の骨である.骨盤は頭部,上肢,胴体の重量を支える構造である.骨盤は頭蓋腔や胸腔の壁よりも丈夫で厚く作られ,四つの骨からできている.2個の寛骨は骨盤の側面および前面を形作り,仙骨と尾骨は後部の壁を作る.男性の骨盤と女性の骨盤とは多くの相違点があるが,最も重要で機能的な相違点は,骨によって囲まれている内部空洞の形状である.開口部形状は女性の場合ほぼ円形であるが,男性の場合,左右方向に広い.a. 寛骨寛骨は図3・40に示すように腸骨,坐骨および恥骨の三つの骨が癒合してできている.腸骨翼の上部前方を上前腸骨棘と呼ぶ.ラップベルトをかけるときには,この上前腸骨棘でしっかりと支えることが必要である.腸骨は耳状面と呼ばれる内側の面で仙骨に結合している.坐骨は寛骨の下側の後方部位を形成し,坐骨体と坐骨枝に分けられる.坐骨体は寛骨臼の後方の3分の1を構成している.坐骨枝は薄い平坦な坐骨を恥骨枝につなぐ部分である.恥骨は恥骨体および二つの恥骨枝(恥骨上枝と恥骨下枝)からなる不規則な形状をもつ骨である.恥骨上枝は恥骨体から正中矢状面へ延びて,反対側にある対応する恥骨上枝と関節をなしている.この関節を恥骨結合と呼ぶ.軟骨板を含むため若干動くことのできる関節である.b. 仙骨および尾骨仙骨は5個の仙椎が癒合したもので,その形は三角形である.その耳状面は骨盤と固く結合し,ここを通って上部胴体重量が坐骨結節へ伝達される.仙骨は前面が凹面で,上端で腰椎を支えている.尾骨は3~5個の椎骨が癒合した退化した尾である.仙骨と尾骨の関節は,通常の椎間板よりも硬い軟骨の薄い椎間板をもっており,若干動くことができる.
