# 3・7・2 傷害のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・7 腰部傷害(lumbar injury)のバイオメカニクス

自動車事故の場合,骨盤傷害(pelvic injury)は下肢傷害として分類され,乗員がフロア,ダッシュボード,ペダルおよびドアなどに衝突した場合に発生し,膝,大腿,骨盤を含めた構造として議論される場合が多い.骨盤傷害は以下の五つに分類される.a. 筋肉付着点の裂離過度の筋肉運動により筋肉の一部が筋肉付着点で骨盤から裂離または分離する.例えば,上前腸骨棘から縫工筋が裂離したり,あるいは膝屈曲筋が坐骨結節から裂離することがある.これらの傷害は運動競技中の若い成人に発生し,衝撃型の事故によるものではない.b. 骨盤輪の分離骨折骨盤輪まわりの骨折が1箇所であれば重大な傷害ではない.例えば恥骨上枝または恥骨下枝の単一骨折,あるいは両方の恥骨枝の骨折の場合である.その他の分離骨折としては,恥骨結合の若干の分離を伴う腸骨の単一骨折,恥骨結合の分離および仙骨と腸骨間の亜脱臼などを含んでいる.c. 多発骨折または骨折・脱臼多発骨折が生じると骨盤輪は不安定になる.この傷害の二つのおもな形態は,恥骨の多発骨折お(b)(a)(c)図3・40 右寛骨(側面視)8―135よび腸骨骨折を伴う恥骨の骨折である.恥骨の多発傷害は2箇所以上の恥骨枝骨折であり,恥骨結合の脱臼を伴う.これらの傷害は歩行者が自動車に骨盤の側面を衝撃されたときに発生する.一般に,恥骨枝の骨折は衝撃の反対側で生じる.もしも両側の恥骨枝に多発骨折があるときには,衝突側をはっきりと立証することはできない.d. 仙骨と尾骨の骨折この傷害は,通常,応力集中部位となる仙骨神経が通る孔を横切って発生し,神経障害を伴うこともある.垂直加速度が身長を縮め,仙骨に圧縮荷重が加わる場合に発生する.坐骨傷害は着座姿勢で落下する場合に生じる.これは重傷ではないが痛みが激しい.e. 併発性軟組織傷害骨盤骨折に併発する傷害のうち最も重いものは出血である.出血は骨折面そのものからと骨盤壁の大血管からくる.血液流出量は多量である.膀胱および尿道ならびに腹部内臓への傷害もまたよく発生する.
