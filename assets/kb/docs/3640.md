# 3・7・2 傷害のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・7 腰部傷害(lumbar injury)のバイオメカニクス

自動車事故の場合，骨盤傷害（pelvic injury）は下肢傷害として
分類され，乗員がフロア，ダッシュボード，ペダルおよびドアなど
に衝突した場合に発生し，膝，大腿，骨盤を含めた構造として議論
される場合が多い．骨盤傷害は以下の五つに分類される．
a.筋肉付着点の裂離 過度の筋肉運動により筋肉の一部が筋
肉付着点で骨盤から裂離または分離する．例えば，上前腸骨棘から
縫工筋が裂離したり，あるいは膝屈曲筋が坐骨結節から裂離するこ
とがある．これらの傷害は運動競技中の若い成人に発生し，衝撃型
の事故によるものではない．
b.骨盤輪の分離骨折 骨盤輪まわりの骨折が1箇所であれば
重大な傷害ではない．例えば恥骨上枝または恥骨下枝の単一骨折，
あるいは両方の恥骨枝の骨折の場合である．その他の分離骨折とし
ては，恥骨結合の若干の分離を伴う腸骨の単一骨折，恥骨結合の分
離および仙骨と腸骨間の亜脱臼などを含んでいる．
