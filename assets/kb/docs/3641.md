# 10・8・1 知的財産情報の性格，価値 特許情報は技術情報と

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・8 技術情報活用

パトリス 内外国特許データをオンラインで有料で提供する．
技術用語テーブルを独自に編集し，技術検索を容易にしている．
商標検索もできる．
ユーザ指定により，必要な項目でデータ編集し，必要な公報だけをCD－ROMに記録させて販売をしている．
特許調査会社 パトリスをはじめとするオンラインデータベースの代行検索を行い，CD－ROM公報や紙公報を所有しコピーサービスを行い，特許に係わる全般的な調査を行う会社が多数ある．
調査の目的・費用・納期等や調査会社の特性に合わせて選択すると権利情報の二面性を持つ．
企業においては，日々新技術の開発がなされているが，その開発を誘導し加速することが求められている．
このために，分類や発行日が明確な特許情報を技術情報として利用することが有用である．
特許情報では特許請求の範囲に記載された技術以外の，詳細な説明の欄及び図面に記載された技術は原則自由に実施でき活用の幅が広い．
