# 3・7・3 傷害基準と傷害耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・7 腰部傷害(lumbar injury)のバイオメカニクス

a. 垂直衝撃エヴァンス(Evans)とリスナーは死体22体について、両側坐骨結節が鋼鉄板に同時に衝突するようにして垂直落下試験を行った。筋肉と臓器を除去した骨盤に応力塗料を塗布し、骨盤のひずみと骨折を調べた。その結果、79歳の男性が落下エネルギー27.1N・mで坐骨恥骨枝骨折を受けたと報告している。ファソーラ(Fasola)らは、小さなくい打機を用いて未防腐処理成人の骨盤の耐性を計測した。筋肉と臓器が除去された場合は、衝撃力3.7kN で仙腸関節両側に分離が発生した。筋肉、内臓、靱帯がある場合の準静的試験においては、最大荷重3.4kN で分離が発生したと報告している。b. 前面衝撃パトリックらは、車両衝突シミュレータを用い、膝-大腿-腰の衝突傷害を調査した。無傷の未防腐処理死体10体を用いて膝を衝撃し、膝蓋骨、大腿骨、骨盤の骨折荷重をまとめた。骨盤骨折耐性(pelvic fracture tolerance)を表3・11に示す。7.12kN で左側恥骨枝骨折が発生している。また、6.9~17.1kNの範囲で恥骨の多発性骨折が発生している。パトリックらは、膝衝撃の場合、膝蓋骨、大腿骨、骨盤のうちいずれが最初に骨折するかは確実には予測できないが、骨盤および膝蓋骨の骨折荷重耐性は大腿骨に比べ高いレベルにあると述べている。さらにパトリックらは数名のボランティア試験を行い、衝撃時の痛みを我慢できる荷重が3.6~4.5kN であったと報告している。メルビンとヌショルツ(Nuscholts)も死体のスレッド試験により膝衝撃時の骨盤傷害を調査し、衝撃荷重ピーク値が8.9~25.6kN で骨盤骨折が発生したと報告した。ブルンカッサンらは、死体10体を用いて、より実際の事故に近い形で膝衝撃時の傷害を調査した。スレッド上にルノー社製の車体を取付け、死体はシートベルトを装着させてシートに座らせ、衝突速度は49.5~67.1km/hとした。膝荷重8.8kN で膝蓋骨と骨盤の腸骨稜で骨折が発生した。表3・11 側面衝撃自動車が側面衝突されたときや歩行者が側面から自動車に衝突されたときには、骨盤に側面から荷重が作用し傷害が発生する。骨盤の側面荷重に対する耐性の研究は1977年から報告されている。フェイヨン(Fayon)らおよびタリエールらは、死体18体を左側面を下にした状態でつるし、1~3mの高さから自由落下させ骨盤の側面衝撃時の加速度耐性を調べた。結果を図で計測した加速度をもとに80~90G(3ms)を提案した。セザーリ(Cesari)らは、剛シートに座った死体22体の大転子位置に半球面インパクタで側面衝撃荷重を加える試験を実施した。傷害は坐骨恥骨枝および腸骨恥骨枝の骨折が最も多く、市場事故とよく対応していた。傷害発生は衝撃加速度よりも衝撃荷重との相関が高く、骨の大きさを体重から換算した修正後の体重との相関が最もよかった。結果を図3・42に示す。図3・42中の直線は傷害が発生する衝撃荷重の推定式で、荷重(N)=193.85×体重-4710.6である。これらの結果から50%tile男性の骨盤骨折耐性として10kN(3ms)を、5%tile女性の骨盤骨折耐性として4kN(3ms)を提案した。マーカス(Marcus)らおよびキャバナーらは死体に側面衝撃荷重を加えた。この装置はロードセル付きの側壁とシートが固図3・41 傷害耐性のまとめ自動車の衝突傷害においては、エアバッグなどにより上半身の傷害が減ってきたため、下肢の傷害耐性の研究が最近多くなってきている。骨盤・大腿骨骨折は膝への入力荷重により発生する。骨盤は血管が集まる場所であるため、より重傷になる。アメリカでの法規においては死体実験結果をもとに、大腿骨入力荷重10kN と規定されている。なお、上記以外の腰部耐性の研究結果については文献(60)を参照されたい。
