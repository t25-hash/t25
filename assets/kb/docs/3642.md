# 10・8・2 技術情報としての調査

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・8 技術情報活用

外国特許や文献も取り扱うところが多い．
大阪府立特許情報センター 内外国の紙公報とCD－ROM公報を所蔵し，閲覧とコピーができる．
IPD：L専用端末やインターネット端末により，各種のデータベースの検索ができる．
特許以外の情報の所在と入手方法i．
技術文献 国立国会図書館の科学技術資料室には内外国の科学技術に関する参考図書・文献類が所蔵されており閲覧ができる．
またCD－ROMやオンラインによるデータベースの検索もできる．
国内外の技術文献については科学技術振興事業団のJOISのデータベースサービスがあり，必要な文献の取り寄せができる．
意匠，商標 オンラインで特許電子図書館（IPDL），パトリス，各国特許庁データベース，BRANDY等から検索ができる．
特許裁判の判例 特許的判断の妥当性を検証するために判例を参考にすることが多い．
オンラインでは最高裁判所や発明協会のホームページから検索することができる．
特許の流通 各企業や公的機関が所有する特許権には，実施されずに権利としてただ維持されているいわゆる休眠特許や，実施許諾をしても自社の技術競争力に影響の少ない特許が含まれている．
ヤ このような特許を活用するために，特許の流通をはかる動きは従来から見られていた．
最近では，自社の中核技術もライセンスすることで投資回収を早め知的資産価値を向上させようとする動きも出てきている．
特許市場活性化のための環境整備として特許庁が開催する特許流通フェアは，特許の提供又は導入を希望する企業・大学・研究機関，仲介業者などが直接交流する場となっている．
インターネット上にも，ベンチャー企業向けなど，特許流通の仲介をするウェブサイトが多く見られ，その中には特許の価値評価のサービスを提供するところも出てきている．
a，研究開発のための調査 新製品や新技術の研究開発に関連する特許情報を入手し，その分野における技術水準や技術開発の動向を分析したり，開発のヒントを得るために行う調査である．
既に出願されており特許の見込みのない技術や，既に権利化された技術の重複研究を避ける点でも役立っ．
新規参入時は必ず焦点となる技術に限らず，その成立に関連する技術についても，広範囲に調査し，自社の位置づけを明確にすることが大事である．
輸出を前提とする研究開発の場合は，日本及び主要国の調査で抽出された関係する特許について，他の輸出国に特許が存在するか否かを優先権主張のデータにもとつくパテントファミリー調査で代替する手法もある．
他社動向調査 研究開発する技術領域に参入している他社の出願を調査することで，その企業の技術レベルや研究開発の動向を分析するために行う調査である．
彼我比較により，開発の方向性や資源の配分等を決定する際の指標として用いられる．
単に出願・登録件数の推移を比較するだけではなく，個々の発明の評価，発明者，その数，同分野における実施例から想定される開発内容の推移なども検討し，他社の開発の方向性を導き出すようにする．
法制度や法規制の変化等の社会動向も加味するとよい．
新規に発行される特許公報を継続して収集・解析することにより，他社の動向を逐次把握することもなされている．
出願前先行技術調査 特許を出願する前に，この発明の新規性及び進歩性を確認し特許の可能性を判断するために行う調査である．
出願時には先行技術との対比説明が要求されていることからも，・できるだけ出願内容に近い先行技術を調べることが，拒絶理由を排除し広い権利を確保する上で重要である．
