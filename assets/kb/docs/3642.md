# 3・8・1 四肢の解剖学

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・8 四肢傷害のバイオメカニクス

四肢(extremities)とは左右一対の上肢(upper extremity)と下肢(lower extremity)の総称であり,道具の操作や起立歩行など人間のさまざまな活動に関わる重要な器官である.四肢傷害としては骨折(bony fracture)および靱帯傷害(ligament failure)が重要であることから,ここでは骨格と関節の構造について解説する.a. 上肢の構造図3・44は上肢骨格を示したものである.上肢には左右合計で64個の骨と40対の関節が含まれる.肩関節(shoulder joint)は肩甲骨(scapula)側端部に位置する球状関節(spherical joint)であり,関節上面まで鎖骨(clavicle)が回り込み肩鎖関節(acromioclavicular joint)を形成している.肩関節は関節面の深度が浅く可動範囲が非常に大きいことが特徴で,過度な回転や移動を制限し脱臼(dislocation)を防ぐため多数の靱帯(ligament)や筋肉(muscle)に包まれている.腕(arm)や手(hand)の多くの機能が肩関節の固定もしくは動作により生み出されるが,上肢全体の挙動においては肩鎖関節も重要な役割を担う.肩関節から肘関節(elbow joint)に至る部分は上腕骨(humerus)と呼ばれる長尺の骨である.肩関節に近い上半分は略円形断面を有するが,中央付近から断面形状が変化し,肘関節に近い下方3分の1の範囲は略三角形断面となる.上腕筋が付着するのはこの範囲となる.上腕骨の末端は内外両側に顆部(condyles)を有し,両者の間に肘関節面が形成されている.肘は腕尺・腕橈・橈尺間の3関節の複合体であり,複雑な回転挙動を可能にしている.肘関節から先は前腕骨と呼ばれ,橈骨(radius)と尺骨(ulna)により構成されている.親指(thumb)側に位置するのが橈骨であり,その上端部は肘関節において上腕骨端部の球体を受ける皿部を形成し,上端やや下方には前腕二頭筋(biceps)が付着している.橈骨の下端は手関節(carpus)に面しており,外側端部には茎状突起が形成されている.尺骨は肘関節側に肘頭と呼ばれる肘関節の背面に回り込んだ突起を有する関係で,その全長は橈骨より著しく長い.また,手関節側の端部は肘関節側に比べて小さい.手関節には八つの手根骨が含まれ,四つが2列に並んでいる.手首の動きには15の筋肉が関係しており,伸筋支帯(retaining band)により束ねられている.手掌内の骨は中手骨(metacarpal)と呼ばれ,手指の骨(phalange)は基節骨・中節骨・末指骨に分かれている.b. 下肢の構造図3・45は下肢骨格を示したものである.下肢には左右合計で62個の骨と36対の関節が含まれる.股関節(hip joint)は骨盤(pelvis)の左右両側に形成された球状関節であり,骨盤側の凹面と大腿骨(thigh)側の凸面の曲率はほぼ一致する.股関節は屈曲(flexion)・伸展(extension)・内転(inver-sion)・外転(eversion)・回旋(torsion)することが可能だが,その可動域は肩関節ほど大きくはない.股関節から膝関節(kneejoint)に至る部分が大腿骨(thigh)で,人体骨格の中で最も長い骨である.股関節側には球状の骨頭部と骨盤側からの筋肉が付着する突起を有する.膝側の末端には内外両側に顆部があり,膝関節面を形成している.膝関節は大腿骨側の凸面の曲率に対して脛骨(tibia)側の凹面のわん曲が浅い不一致関節であり,靱帯による支持がなければ力学的に不安定な構造である.膝関節の屈伸時には大腿骨の内外顆部が半月板(meniscus)上を転位もしくは滑動する.内側(medial)および外側(lateral)の半月板はそれぞれ膝関節の中心から外側へほぼC字形状を有し,くさび形断面の軟骨組織が大腿骨側と脛骨側の曲率差を埋めることにより,膝関節へ加わる負荷を関節面の広域に分散する役割を果たしている.膝関節の安定性は内側と外側の側副靱帯(MCL・LCL)および2本の十字靱帯(ACL・PCL)によっても保持されている.また,図3・45では省略されているが,膝関節前面の膝蓋骨は屈伸動作に大きく寄与している.膝関節から足関節までが下腿骨であり,足の親指側に位置する脛骨と小指側の腓骨(fibula)とで構成されている.脛骨は下腿の前方で脛の表皮の直下に位置し,その下端部は内果(medialmalleolus)を形成している.腓骨は細長い形状を有し,下端部は外果(lateral malleolus)を形成している.内外の果部に挟まれるように足関節(foot joint)があり,足側の滑面は距骨(talus)上面に形成されている.この距骨上面は主として伸展(plantar-8―137flexion)と背屈(dorsiflexion)に関係するが,内転や外転および内旋(internal rotation)や外旋(external rotation)では距骨側面や下面が関わることになる.足関節もまた骨格構造的には不安定であり,前記のさまざまな方向への可動は靱帯類により制限されている.足部には26個の骨があり,7個の足根骨(tarsal bone),5個の中足骨(metatarsal),14個の足指骨(phalange)で構成される.足根骨に含まれる踵骨(calcaneus)の背面にはアキレス腱(Achilles tendon)が付着し,連続するふくらはぎ筋肉の収縮により足関節が伸展される.
