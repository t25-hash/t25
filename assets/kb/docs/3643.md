# 10・8・3 知的財産情報の所在・入手方法

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・9 技術開発，製品開発，製造，販売における知的財産

特許情報の所在と入手方法と全国7都市の地方閲覧室でCD－ROM公報の閲覧が可能である。
産業省別館の第二公報閲覧室では日本公開公報や外国の紙媒体の特（IPDL）の専用端末から種々の検索が可能であり，その場で紙にii．
発明協会 各都道府県に支部をもち， CD－ROM公報，合ている．
技術情報及び権利情報に関する調査も行っている．
特許庁（工業所有権総合情報館）特許庁の第一公報閲覧室第一公報閲覧室では外国特許のCD－ROM公報も閲覧できる．
経済許明細書や公報が閲覧できる．
各閲覧室では特許電子図書館出力もできる．
