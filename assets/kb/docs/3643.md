# 3・8・2 傷害のメカニズム

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・8 四肢傷害のバイオメカニクス

車両衝突における四肢傷害の発生メカニズムを解明することは容易ではない。着座位置や身長などによりある程度範囲が特定される胴体(torso)や頭部(head)とは異なり、身体の末端(distal)ほど衝突時の状態(位置や姿勢)を特定することが困難だからである。車両の運転操作では手足を胴体の前方へ伸ばして行う場合が多いことから、四肢傷害の多くが前面衝突(frontal crash)で発生している。前面衝突における前席乗員(front-seat occupant)の受傷部位を拘束装置(restraint system)別に調査した研究では、ベルト非着用の乗員が頭部や胸部(thorax)を高い頻度で受傷するのに対し、ベルトを着用した乗員では四肢の受傷率が高いことが報告されている。シートベルト(seat belt)により乗員の上半身が保護された結果、四肢の受傷率が相対的に増加したものと考えられる。一方、衝突時にエアバッグ(air bag restraint system)が展開した事例では上肢の受傷率が高まることも指摘されており、上肢がエアバッグもしくは車室内装と干渉した可能性を示唆している。四肢傷害は生命に関わるほど重大な結果に至る確率は高くはないが、道具の操作や起立歩行など日常生活に及ぼす影響が大きいという理由で問題視されている。a. 上肢傷害のメカニズム車体変形により上肢が受傷するほどの損壊程度では頭部や胸部などへの致命傷(fatal injury)が問題となることが多いため、上肢傷害はエアバッグとの干渉(inter-action)という観点から研究が進められてきた。運転者側(driver side)の前面衝突用エアバッグはステアリングハブ(steer-ing hub)内に格納されている。通常、運転者はステアリングホイール(steering wheel)を握っているため、衝突時にエアバッグが展開(deploy)する際に運転者の上肢と干渉する可能性がある。考えられる傷害メカニズムとしては、エアバッグから上肢への負荷が直接的な原因となり傷害に至るケースと、エアバッグにより跳ね上げられた上肢が車室天井(roof)等の内装と二次衝突して受傷するケースである。後者の問題については重度の傷害に至る頻度が低いことなどから、もっぱらエアバッグと上肢の干渉が直接的な要因となりうる場合に焦点を絞った研究が行われている。ステアリングを握った状態でのエアバッグとの干渉では主として前腕骨の骨折が問題となる。前腕への衝撃入力は右折時や左折時などにステアリングハブ上方を横切る位置で最大となり(図3・46)、ステアリングハブからの距離も影響する。また、傷害の発生は衝撃入力の大きさのみならず、前腕の質量や前腕骨の骨密度にも依存すると考えられている。図3・45 下肢傷害のメカニズム下肢傷害の発生メカニズムは膝上と膝下に分けて考えられる。膝上の傷害としては、大腿骨骨折や骨盤骨折など比較的重度に分類される傷害モードがある。これらの傷害は、前面衝突時に乗員が慣性力により前方移動することで、膝部が計器パネル(instrument panel)等と干渉して発生する。干渉形態によっては異なる傷害モードが現れることが知られており、例えば、計器パネル側の剛性が高い場合は膝蓋骨骨折や顆部損傷など膝側の傷害が発生しやすく、接触面が柔らかい場合には大腿骨骨折や股関節脱臼など基部側の傷害頻度が高くなる。また、膝頭に負荷が集中すると荷重は大腿骨へ伝達されるが、負荷点が脛骨上端にずれると靱帯損傷の要因となりうる。開脚姿勢が股関節側の傷害モードに影響することも知られている。膝部と計器パネル類との干渉を防ぐためには乗員の前方移動量を抑制することが必要であり、シートベルトプリテンショナ(seat-belt pretensioner)等の安全装置が開発・利用されている。また、ベルト非着用の乗員など膝部と計器パネル類との干渉が避けられない場合を想定して、ニーボルスタ(knee bolster)等の保護装置も一部で採用されている。膝下の傷害は生命に関わるほど重度に至ることはまれであるが、前面衝突における下肢部位ごとの受傷割合を調べた研究によれば、膝上に比べて膝部や脛(tibia)および足首(ankle)の割合が高いことが報告されている。中足骨など足先の傷害は比較的治癒しやすいが、脛骨や足根骨の骨折は治療期間や医療コストおよび後遺症の観点から重要視されている。膝下の傷害メカニズムについては車体変形やペダル(pedal)との干渉が主たる要因であると考えられている。例えば、衝突時にペダルを踏み外してフロアに打ち付けるだけでも足首を捻転する可能性がある。しかしながら、下肢の慣性力(inertia)だけでは大きな負荷が発生しないことから、脛や足首など身体を支持する強固な骨の傷害については、図3・47に示すように膝(knee)が計器パネル下面に保持された状態で足下の車体パネル(toe board)が後退(intrusion)するなどして過大な負荷が入力されることが原因と考えられている。一方、中程度ではあるが頻度の高い傷害として靱帯損傷や踝骨折などがある。これらの傷害は運転者の足がペダル類と干渉して足首が背屈(dorsiflexion)・内転・外転方向に過度に回転させられることで生じると考えられている。このように衝突時の車体変形が膝下傷害の要因の一つであることは事実だが、膝上傷害の場合と同様に乗員の挙動も影響することを忘れてはならない。例えば、前述した膝部と計器パネルとの干渉は腰部の前方移動量に依存するし、水平面に対する下腿の角度は衝突後の負荷入力に大きく影響する。同様に足首の初期角度や位置により傷害結果が変わることも知られてい図3・47 前面衝突における膝下重度傷害の発生メカニズムる。最近では衝突を事前に認知した乗員の下肢筋肉の硬直が傷害モードに影響を及ぼすことが指摘されており、これを再現した実験により実際の衝突事故における傷害モード分布が説明できることも報告されている。膝下傷害への対策としては、入力側となる車体やペダルの変形を抑えること、負荷を受ける乗員下肢の姿勢や挙動をコントロールすること、両者の干渉による衝撃を緩和することなどが考えられる。
