# 3・8・3 傷害基準と傷害耐性

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・8 四肢傷害のバイオメカニクス

四肢傷害では骨折および靱帯損傷が問題となることから、骨に負荷される荷重やモーメント、関節部の変位や回転角などが指標とされている。以下に、法規や安全アセスメント試験で参照されているものや、提案中ながらも有力な指標を含めて解説する(表3・12)。a. 上肢傷害基準と耐性上肢傷害に関してはエアバッグとの干渉という観点からおもに曲げモーメントに対する耐性が論じられてきた。成人男性(mid-size male)を想定した上腕骨の骨折耐性としては217~230N・mという値が、前腕骨については橈骨と尺骨を合わせて91N・mという値が提唱されている。ただし、これらの耐性値は現在のところ車両の衝突安全性評価試験における評価項目に含まれていない。b. 下肢傷害基準と耐性大腿部(femur)の耐性を求めるため、死体の膝頭をインパクタで強打して骨折時の荷重を測定する実験が行われてきた。大腿部の骨折荷重は時間依存型であり、衝撃力の継続時間が20ms以上の場合は8.9kN だが、短くなるにつれ23.14kN に漸近すると報告されている。3・8・2・b項で述べたように、大腿部の傷害は負荷入力形態により受傷部位が膝関節・大腿骨・股関節に分かれる。過去に実施されたさまざまな実験データを統計的に分析した研究から、成人男性における大腿部の傷害耐性としては10kN という値が広く参照されている。膝関節の靱帯傷害では大腿骨顆部と脛骨上端の相対変位が問題となる。死体下肢に対する衝撃実験結果から、膝変位(knee sliding)14.4mmで局所的な靱帯損傷が始まり、22.6mmで全損傷に至ると報告されている。これらの結果を踏まえ、成人男性における膝関節の傷害耐性としては15mmという値が広く参照されている。下腿においては荷重負担の大きい脛骨の耐性が研究されており、曲げモーメント240N・mおよび軸力12kN という耐性値が報告されている。車両衝突ではモーメントと軸力が同時に作用して傷害に至るという観点から、傷害評価試験では両者を組合わせたティビアインデックス(tibia index)と呼ばれる指標がしばしば用いられている。脛骨については曲げモーメントと軸力による中間部の骨折だけでなく、膝および足関節に面する上下端部における骨折も問題となる。前述したティビアインデックスと並行して、軸力8kN と表3・12 四肢傷害の耐性一覧部位耐性値上腕骨217~230N・m前腕骨91N・m大腿骨10kN(ヨーロッパ:7.58)膝変位脛骨240N・m12kNティビアインデックス1.3(M/225+F35.9)脛軸力8kN足首背屈:60N・m内転・外転:40N・m*:2002年2月現在、各国法規・安全アセスメント試験で参照されている指標いう値が広く参照されている。足関節については、靱帯損傷に至る足首回転角として背屈(dorsiflexion)方向に45°および内転(inversion)・外転(eversion)方向に60°がしきい値と考えられてきたが、近年では足関節におけるモーメント値(背屈:60N・m、内転・外転:40N・m)を指標とすることも検討されている。また、足関節および足根骨の耐性は脛骨より低く、6.8kN という荷重値が提唱されている。車両の衝突安全性能評価試験で用いられる下肢傷害基準は、前述した耐性値をもとに、性別や体型ごとに傷害リスクカーブを求めて定められている。現在のところ、各国の法規および消費者向け情報公開試験(NCAP等)において評価されている部位は大腿部・膝部・脛部である。大腿部荷重(femur force)については、北アメリカ・オーストラリア・カナダ・日本では10kN、ヨーロッパでは7.58kN という規制値が採用されている。また、膝変位については15mm、ティビアインデックスは1.3、脛軸としては力8kNという値が用いられている。前述したようなさまざまな傷害耐性値が提唱されているにもかかわらず衝突試験における傷害評価基準として用いられていない理由としては、実験データ数の不足などにより傷害指標の妥当性が十分に検証されていなかったことに加え、評価実験において用いるダミー(dummy)の問題もあった。現在、各国で広く用いられている衝突実験用(Hybrid-III)ダミーの下肢部分は、生体忠実度(biofidelity)や計測項目などの面で課題が多いとされている。近年重要視されている膝下傷害を厳密に評価するため、人体構造をより忠実に模擬・再現した新しい衝突実験用下肢ダミーが研究開発されている。
