# 3・9・1 ダミーの目的と要件

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・9 人体の衝突ダミー

a. ダミーの目的近年、自動車の安全性が大きくクローズアップされてきており、世論の関心も高まっている。そのような状況の中、自動車メーカをはじめとして各国研究機関、政府によって衝突試験用ダミー(以下、ダミー)を用いた自動車の衝突試験が盛んに行われている。その衝突試験方法、形態も前面フルラップ衝突からオフセット衝突(以下、前突)、側面衝突(以下、側突)など多岐にわたり、使用するダミーも衝突方向、評価する傷害値の違いなどにより多様化してきている。また自動車の安全装備もシートベルトに加え、運転席、助手席エアバッグ、側突エアバッグ、カーテンシールドエアバッグなど高度化してきており、安全な車を開発するためには、各種ダミーを用いてそれぞれに適した条件で車両や安全装備を正確に評価していく必要がある。最近では女性、子供に対するエアバッグの副作用問題が大きくクローズアップされており、その評価のためのダミーも非常に重要となっている。また、幼児用シ図3・48 Hybrid-IIIファミリーと側突ダミー8―139ートも着用が義務付けられ、衝突時の幼児の保護も社会要請となっていることから、幼児用シートの安全性を正しく評価するダミーもますます重要になってきている。b. ダミーに要求される要件自動車の衝突ダミー(crashtest dummy)には四つの大きな要素が不可欠である。一つは人体忠実度が高いことであり、もう一つは計測値と人体傷害の関係が明確になっていること、耐久性があること、最後にどこの機関でやっても似たような結果が出ること、すなわち再現性があることである。人体忠実度が高いこと、および計測値と人体傷害の関係が明確であることは、すべてのダミーの基本である。衝突保護性能の評価と可能な技術的解決策の有効性は、衝突試験で使用するダミーの人体忠実度および関連する傷害値の測定性能に大きく依存するのである。さらに、耐久性と再現性は計測器としての側面から重要である。人体忠実度が高いといっても、1回の試験で壊れているようでは計測器として問題がある。近年ではアメリカ、ヨーロッパ、日本、オーストラリアでNCAP(New Car Assessment Program)が行われるようになっているが、ダミー個体差で再現性が大きく異なることは消費者をミスリードすることにもつながりかねない。
