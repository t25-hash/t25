# 3・10・1 マルチボデーモデル

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・10 人体の数値モデル

インパクトバイオメカニクスにおける人体数値モデルには,衝突ダミーをモデル化する衝突ダミーモデル(crash dummy model)と,人体を直接モデル化する実人体モデル(real human bodymodel)がある.モデル化の手法としてマルチボデー解析と有限要素解析が一般に用いられる.マルチボデー(multibody)のこの分野への適用は1963年マクヘンリー(McHenry)による人体,拘束装置,車両を表す二次元7自由度モデルが最初とされている.衝撃時の人体の挙動では関節が自由度をもち,他の部位は剛体と近似できることが多いため,マルチボデー解析が適用可能なことが多い.マルチボデーではボデー(剛体またはフレキシブルボデー)が図3・51のような多様なジョイントにより結合され,衝撃時の各ボデーの過渡応答が求められる.ボテーに働く外力は,加速度場,ばね-ダンパ要素,拘束装置,接触等の力の相互作用がある.衝撃時の人体の全身挙動は大きくなるため,これを効率よく計算し,保護装置の最適化を行うためには有限要素よりもマルチボデー解析が適していることが多い.衝突解析のマルチボデーシミュレーションプログラムでは,TNOによって開発されたMADYMOが用いられることが多い.特に,MADYMOには衝突解析用にシートベルトモデル,傷害値の出力などが準備されており,さらに多様な衝突ダミーやインパクタのデータライブラリをもっている.マルチボデーと有限要素法との結合も可能となっており,例えば有限要素でモデル化されたエアバッグとマルチボデーでモデル化された衝突ダミーの連成解析が可能である.この場合,変形は有限要素のみとなり,マルチボデーは変形しないものとして計算される.図3・52に示す衝突ダミー(Hybrid-III)による乗員シミュレーションは,拘束装置の最適化に用いられる.これまでのマルチボデーモデルは人体そのものよりも,衝突ダミーに対して開発されたものが多い.この理由としては,衝突ダミーではデータが求めやすく検証が容易であること,および認証試験でダミーが用いられるため,これに対する設計の効率化が挙げられる.衝突ダミーのマルチボデーによるモデル化では,運動学を考慮してダミーがセグメントに分割される.相対運動がないダミー部品が一つのセグメントとして選択され,セグメントどうしは互いの自由度を考慮したジョイントによって結合される.セグメントの幾何学的パラメータ,慣性特性,ジョイント位置・方向は,図面や計測によって決定される.ダミーの外形形状はだ円体のほかに,任意形状を表すファセットが用いられる.ダミーの荷重変形特性は衝突時に予測される変形を考慮して静的および動的測定が行われ,ジョイントについても静的,動的な特性が計測される.衝突ダミーモデルは部位ごと,あるいは全身についてインパクト試験,スレッド試験による検証が行われる.図3・53にEuroSID-1側突ダミーモデルとして,だ円体,ファセット面によるマルチボデーモデルと有限要素モデルを示す.衝突ダミーは代表的な傷害リスクを各部の加速度,荷重,変形量により評価するが,さまざまな部位の骨折や軟部組織損傷の危険性を評価することができない.これに対して実人体モデルは衝突ダミ図3・52 MADYMOによる乗員挙動シミュレーション(有限要素によるエアバッグとマルチボデーによるHybrid-III乗員モデル)8―141ーモデルに比較して,潜在的に生体忠実度が高く,体格,姿勢,筋活動,骨折後の応答などの研究が可能である.モデルは,死体実験やボランティア試験の人体各部軌跡から作られたコリドーに対して,応答がこの範囲内にあることを確認することによって検証される.実人体のマルチボデーモデルでは,任意の体格をモデル化するためにGEBODが用いられることが多い.GEBOD では身長,体重,性別を与えることによって,人体各部のだ円体で近似された形状と慣性特性,関節の位置が作成される.特に歩行者では衝突時の挙動が大きいためマルチボデーが用いられることが多く,GEBODによるさまざまな体格の歩行者モデルが発表されている.各ジョイント特性には死体・ボランティア実験による値が与えられる.これらのモデルは全身の応答が死体実験によって検証され,車体形状や剛性と各部の傷害の関係が検討されている(図3・54).さらに詳細に人体形状を表すため,ハッペ(Happe)らは外表面としてRAMSIS
