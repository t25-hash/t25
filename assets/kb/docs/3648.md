# 3・10・2 FEM モデル

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・10 人体の数値モデル

近年、有限要素法(finite element method、以下、FEM)を用いた数値シミュレーションが、自動車衝突時の人体(humanbody)の挙動や傷害の解析に適用されるようになってきた。3・10・1項で述べているマルチボデーモデルと異なり、FEM では解析対象物の形状や構造を忠実に有限要素で再現することができる。また、各要素にそれぞれの力学的な特性(ヤング率、ポアソン比、降伏応力等)を与えることで、外力を受けたときの対象物を構成する各要素の応力やひずみを計算により求めることができるという利点がある。その反面、モデルの作成に多くの時間を要することや、要素の分割数が増加すれば計算時間も増加するという欠点がある。このFEM を3・9節で述べているダミーや人体そのものに適用すれば、複雑な形状や構造が再現でき、リアリスティックな解析が可能になってくる。自動車の衝突現象等を解析するためには、FEM の中でも動的陽解法有限要素法が用いられる。図3・57はHybrid-IIIダミーのFEM モデルを示している。計算機能力の飛躍的な向上により、形状を忠実に再現した車両モデルとダミーモデルを組合わせた衝突解析が行われ、衝突時のダミーの応答を計算機上で試行実験することが可能になってきている。しかし、機械要素で構成された擬似人体ダミーモデルでは、生身の人間に発生する複雑な骨折や関節部の靱帯破断・内臓臓器の破裂といった軟組織の傷害等の発生メカニズムに対する、よりきめ細かでかつ直接的な解析を行うには限界がある。そこで、近年、人体そのものを解剖学的な構造ならびに力学特性に忠実に計算機上でモデル化し、傷害を再現・予測しようとする試みがなされている。この人体FEM モデルは人体の解剖学的構造を有限要素でモデル化し、組織を構成する各要素に実験で得られたその組織の材料特性(ヤング率、ポアソン比、降伏応力等)を与えることにより作られている。モデル作成のために必要となる解剖学的構造や形状の三次元座標データには、医療診断等に使用されるCTやMRI画像から得られる輪郭形状や、市販の人体形状に関するポリゴンデータが用いられている。材料特性としては、人体や動物の組織を用いた荷重試験データなどが用いられる。図3・58は衝突安全(crash safety)分野で下肢傷害解析のため図3・58 下肢傷害解析用FEM モデルの例[日産自動車(株)提供]図3・59胸部への側面衝撃計算における変形状況図3・60 胸部側面衝撃の荷重-変位曲線の比較に開発されたFEM モデルの一例を示している。形状を忠実に再現した骨・靱帯・腱・筋肉を含む軟組織により構成され、材料特性には文献から得られる実験データが使用されている。衝突安全分野では、このような人体各部位のモデルが多く開発されており、その妥当性(validity)は献体実験データと比較することにより評価されている。献体実験は主として人体耐性値の取得、ダミーの妥当性検証のため、欧米で実施されているが、実験により得られるデータは人体FEM モデルの妥当性を検証するためにも有用である。献体実験で計測された結果を、献体実験と同じ条件を与え計算した結果と比較することで、モデルの妥当性を検証することができる。図5.8m/sという献体実験と同じ条件で、側面衝撃計算を行ったときの胸部の変形状態を示している。ブーケ(Bouquet)らが報告した献体実験の応答範囲との比較を図3・60に示すが、モデルの応答は実験データとよく一致しており、モデルの妥当性を示している。この人体FEM モデルを用いて傷害評価を行うためには、FEM で計算される応力やひずみを用いることができる。例えば要素の応力やひずみが、あらかじめ定められた応力やひずみの限界値を超えた場合、人体のその部分に破壊すなわち傷害が発生したと判定する手法である。このような手法を用いて傷害発生を予測し、献体実験における傷害部位や程度と比較することで、モデルの傷害予測(injury prediction)精度を検証することが可能である。最近になって、計算機能力の向上に伴い、人体各部位を詳細にモデル化した全身のFEM モデルが登場してきた。その代表例を図の挙動や傷害を再現・予測するために利用されている。図3・63に全身モデルTHUMS を用いて、後面衝撃時の乗員の全身挙動を解析した例を示す。計算により得られた全身挙動を献体実験やボランティア試験により計測された全身挙動と比較し、モデルの妥当性の評価が行われている。このような全身モデルを用いれば、衝突事故時に起こりうる多重傷害を再現・予測できると期待され、注目されている。しかし、現状のモデルでは、あらゆる方向や速度に対して、モデルの応答特性と傷害を再現するのはいまだ困難である。今後は、生体組織の特徴であるひずみ速度依存性や異方性を考慮したモデル化も必要になってくる。また、現状の人体モデルは献体の応答特性に合うようにモデル化されているが、今後は生体の応答特性の再現へと進化していくと考えられる。そのためには、アクティブな筋肉特性や脳機能・神経伝達系のモデルも付加していく必要があ図3・61 ヨーロッパで共同開発中のHUMOS図3・62 人体FEM モデルTHUMS図3・63 後面衝撃時の挙動の解析例8―143る。また、血流の影響を表現することができる流体-固体連成問題解析等も必要と考えられる。
