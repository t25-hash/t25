# 3・11・2 ばね質量モデル(spring-mass model)

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・11 衝突時の車両と乗員の力学

車両衝突時に乗員に発生する傷害の程度は,ミクロ的には車体構造や拘束装置からの外力により人体(乗員)の受傷部位に生じる荷重(応力),変形速度,変形量などによって決まる.一方,乗員を一つの質点としてとらえた場合,車両と等しい初速を有する乗員の質量は車体構造や拘束装置からの外力の総和によって減速されるため,マクロ的には乗員の減速度を傷害の規模の指標とすることができる.したがって乗員の傷害を低減するには,これらの乗員応答の指標(傷害値)を低減することが重要であり,そのためには乗員応答のメカニズムを明らかにすることが必要である.乗員の応答は衝突時の車体の減速に起因するものであるが,車体と乗員とは剛体結合されていないことに着目することが必要である.拘束装置により乗員は車体との相対変位に従って拘束力を受ける.すなわち拘束装置はばねとして作用する.このため一般に車体と乗員の減速度は一致せず,乗員の減速度は車体の減速度発生から遅れて発生する.並進運動に着目してこれらの関係を単純化すると,乗員の減速度は,車体と乗員の等価質量(equivalent mass)と,車体,乗員,拘束系のばね特性(剛性)とで構成される一次元ばね質量モデルの応答と考えることができる.これを図3・67に示す.図3・67においては乗員の等価質量を示し,は車体が変形し室内侵入する部分(トーボードなど)の等価質量,は車体の変形を伴わない部分の等価質量を示す.それぞれの質量の初速は同じである.また읍윬읎읏읍율읉윰읏は拘束系の,읈윲윯율윰は人体(乗員)の,읉윰읏윬읍읉윱읍は車体が乗員と接触する部分の剛性をそれぞれ示す.衝突開始時において읉윰읏윬읍읉윱읍と乗員との間には間げきが存在する.읆읍윱윰읏は車室外(エンジンルームなど)の車体剛性を,윪율윩읉윰は車室内の車体剛性を示す.図3・67のモデルに示したそれぞれのばね特性(剛性)は,厳密には速度依存性を伴うが,条件を限定することにより一定のばねとみなして単純化できることが多い.胸部応答を例にとるとは胸部の等価質量,읈윲윯율윰は胸郭部分の剛性に相当し,胸部減速度はの減速度,胸部粘性基準(thor-acic viscous criterion)は읈윲윯율윰の変形速度と変形量の積,胸たわみは읈윲윯율윰の変形量として説明できる.正面衝突において,一般には車体変形や乗員の前方移動により乗읆읍윱윰읏:車体前部(エンジンルーム剛性)윪율윩읉윰:キャビン剛性읉윰읏윬읍읉윱읍:室内部材剛性읍윬읎읏읍율읉윰읏:拘束系剛性읈윲윯율윰:人体剛性図3・67 乗員-車体系のばね質量モデル員はエアバッグや内装部品からも大きな負荷を受けるため,읉윰읏윬읍읉윱읍は傷害値に対して大きな影響を及ぼすが,乗員の客室内部材との衝突(二次衝突)の影響が十分小さいとみなせる場合には,읍윬읎읏읍율읉윰읏(おもにシートベルトの剛性)が支配的となるため,とおよび읆읍윱윰읏と윪율윩읉윰を一体として取扱うことによりモデルをさらに単純化して解析することができる.側面衝突の場合ドアの等価質量がに,ドア内装の剛性が읉윰읏윬읍읉윱읍に相当するが,はとは異なる応答を示し,またサイドエアバッグやドア内装から大きな力が乗員に加わるので읉윰읏윬읍읉윱읍の影響はつねに無視できない.その反面シートベルトの寄与は小さく,읍윬읎읏읍율읉윰읏は支配的ではない.
