# 10・1・1 金属材料成分の分析法および鑑別法

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・1 金属材料の成分と組織検査

a.化学分析法 湿式法とも呼ばれ，化学反応により材料
の化学成分を定量する標準的方法である．各元素ごとに定量法は異
なり，JISでも数多くの分析法が規定されている．
b.機器分析法 発光分光分析法は試料をアークなどで発
光させ，その波長を測定して各元素の定量を行う．蛍光X線分析
法は一次X線を照射したときに発生する蛍光X線から同様に分析
を行う．工場の品質管理の分析法としての主流である．大がかりな
設備を必要とするが，大量処理，迅速化，自動化が可能であり，必
要試料も少量で済む．また，原子吸光分析やICP分光分析法など
化学分析の機器化も進んでいる．
c.簡易鑑別法 定性的または半定性的に化学成分を簡便
に判定する方法である．
.グラインダ火花試験（spark test ) 試料にグラインダ
を当てると火花が発生する．含有する化学成分の種類，量によって
火花の形態，すなわち流線の色，形，長さおよび破裂（花）の形
状，数，色彩などが異なる．既知の材料との比較からかなりの鑑別
が可能であるが経験を要する．鋼についての観測法は，JIS G
0566：1980に規定されている．
.粉末火花試験 試料の粉末をブンゼンバーナの酸化炎で
燃焼させ，発生する火花で判別する方法で，銅合金，軽合金にも利
用できる．
