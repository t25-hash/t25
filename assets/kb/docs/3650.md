# 3・11・2 ばね質量モデル(spring-mass model)

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・11 衝突時の車両と乗員の力学

車両衝突時に乗員に発生する傷害の程度は、ミクロ的には車体構造や拘束装置からの外力により人体(乗員)の受傷部位に生じる荷重(応力)、変形速度、変形量などによって決まる。一方、乗員を一つの質点としてとらえた場合、車両と等しい初速を有する乗員の質量は車体構造や拘束装置からの外力の総和によって減速されるため、マクロ的には乗員の減速度を傷害の規模の指標とすることができる。したがって乗員の傷害を低減するには、これらの乗員応答の指標(傷害値)を低減することが重要であり、そのためには乗員応答のメカニズムを明らかにすることが必要である。乗員の応答は衝突時の車体の減速に起因するものであるが、車体と乗員とは剛体結合されていないことに着目することが必要である。拘束装置により乗員は車体との相対変位に従って拘束力を受ける。すなわち拘束装置はばねとして作用する。このため一般に車体と乗員の減速度は一致せず、乗員の減速度は車体の減速度発生から遅れて発生する。並進運動に着目してこれらの関係を単純化すると、乗員の減速度は、車体と乗員の等価質量(equivalent mass)と、車体、乗員、拘束系のばね特性(剛性)とで構成される一次元ばね質量モデルの応答と考えることができる。これを図3・67に示す。図3・67においては乗員の等価質量を示し、は車体が変形し室内侵入する部分(トーボードなど)の等価質量、は車体の変形を伴わない部分の等価質量を示す。それぞれの質量の初速は同じである。図3・67のモデルに示したそれぞれのばね特性(剛性)は、厳密には速度依存性を伴うが、条件を限定することにより一定のばねとみなして単純化できることが多い。
