# 3・11・3 乗員拘束装置の機能

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・11 衝突時の車両と乗員の力学

乗員拘束装置の基本的な機能は,衝突前に車体と同一の速度をもっていた乗員を減速させ,二次衝突を防ぐことである.このため乗員拘束装置には,限られた客室空間の範囲内で乗員のもつ運動エネルギーを効率よく吸収することが求められる.ここでは代表的な乗員拘束装置の力学的機能を,3・11・2項で示したばね質量モデルにおけるばね特性の適正化としてとらえ説明する.a. シートベルトプリテンショナ(seat belt pretensioner)シートベルトのウェビングは線形ばねに近い特性を示すが,実際のシートベルト装置においては巻取り装置(リトラクタ)からの繰出しやたるみ(スラック)によりベルトの作用時間にわずかな遅れが生じ,乗員の車体に対する移動量が増加する.プリテンショナはこの点を改善する装置であり,衝突後ただちにリトラクタの巻取りシャフトを回転させる,またはバックルを引き込むなどして拘束系のばね特性を線形に近づける作用がある(図3・68).b. ロードリミッタ(load limiter)一定の乗員運動エネルギーをシートベルトで吸収する場合,シートベルトのばね特性を線形に近づけただけでは乗員への入力荷重の最大値を低減できない.そこでシートベルトの一部に塑性変形部を設け,乗員への入力が設定値以上に上昇しないようにするのがロードリミッタ機構である(図車体に対する移動量が増加するため,実際には室内部材との衝突の生じない範囲で設定される.c. エアバッグ(air bag restraint system)シートベルトが図3・68 シートベルトプリテンショナの例と機能図3・69ロードリミッタの例と機能図3・70 エアバッグの例と機能図3・71 胸部減速度応答のばね質量モデル乗員を拘束し客室内への衝突を防ぐものであり,その特性は3・11・2項のばね質量モデルでは읍윬읎읏읍율읉윰읏に対応するものであるのに対し,エアバッグは客室側から瞬時にバッグを展開させ乗員を受け止めるものであり,읉윰읏윬읍읉윱읍を構成する要素である.エアバッグの拘束特性は主として内圧と接触面積で決まるが,乗員のバッグへの押込みにより内圧が急激に上昇するため,排気機構(ベントホール)を設けて内圧上昇を抑える.エアバッグを設けない場合,乗員は客室内との間げきを空走して内装部材と衝突し,内装部材の高い剛性によって大きな衝撃入力を受けることになる.これに対しエアバッグを設けた場合には,衝突初期の間げきにバッグを展開して早い段階から入力を与えることになる.それにより,効率的に乗員の運動エネルギーを吸収しようというものである(図3・70).
