# 10・1・3 結晶粒度試験（grain size test )

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・1 金属材料の成分と組織検査

a.鋼のオーステナイト結晶粒度試験 通常，鋼（オーステナ
イト系の鋼を除く）は，高温でオーステナイト組織であり，その粒
度は，その後の冷却時の焼入れ性，鋼材の材質に大きな影響をも
つ．試験法は，JIS G0551：1998に規定されており浸炭粒度試験
方法と熱処理粒度試験方法がある．前者は925℃で浸炭を行い，後
者は実際作業に近い温度に加熱後，研磨，腐食し結晶粒の大きさを
測定し粒度番号で表す．粒度番号は付図の標準図と比較して決定す
る．
b.鋼のフェライト結晶粒度試験 使用状態の組織であるフェ
ライト組織の結晶粒度を求める．炭素含有量0.2 %以下の鋼に適用
される．JIS G0552：1998に規定されている．
