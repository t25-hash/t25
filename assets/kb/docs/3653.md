# 10・1・4 非金属介在物検査

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・1 金属材料の成分と組織検査

a.鋼の清浄度（inclusion content ）判定法 非金属介在物の
定量法としてJIS G0555：2003に規定されている．材料の被検面
を介在物が脱落しないように研磨し，400倍の倍率で縦，横20本
の格子線の交点に存在する介在物の数を測定し面積百分率で表す．
圧延などで長く粘性変形したものをA系，不連続に並んだものを
B系，不規則に分散したものをC系と分類する．
b.分離分析法 試料中の非金属介在物，または同定しよう
とする析出物以外を電解分離法などで完全に溶解し，残渣を補集し
て，化学分析またはX線分析を行う．
