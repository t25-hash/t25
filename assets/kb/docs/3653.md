# 3・12・1 歩行者事故の特徴

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・12 歩行者保護

自動車が発達したヨーロッパ各国および日本では、1970年代において、歩行者事故に大きな変化が見られ、歩行者死者数は約2分の1に減少した。この最も大きな理由は、歩行者と自動車の分離、教育といった第一次安全対策によるものであるといわれている。しかしながら、歩行者死者数の減少傾向はその後次第に鈍り始め、日本および欧米では依然として交通事故死者の約15~30%が歩行者である。ここでは、歩行者事故の特徴について概説する。図3・75は、日本の交通事故における歩行者の死者数と負傷者数の推移を年齢層別に示すとともに、その発生数と人口10万人当たりの発生数について1970年を100として表したものである。日本の歩行者死者数は、1970年代に半減したが、その後は減少傾向が鈍り、1980年代および1990年代は約3000人前後で推移している。交通事故死者数に占める歩行者の割合は約30%であり、歩行8―146生体工学(b) 日本における年齢別歩行者の死者数と負傷者数の推移者死者数の過半数は65歳以上である。65歳以上の年齢層では、最近、歩行者の死者数および負傷者数はいずれも著しく増加しているが、人口10万人当たりの死者数と負傷者数にはほとんど変化が見られない。したがって、65歳以上の年齢層における死者数と負傷者数の最近の著しい増加傾向は、この年齢層の人口増加に起因していることがわかる。日本の高齢者人口は諸外国に例を見ない勢いで増加しており、高齢歩行者の保護対策は重要な課題である。一方、15歳以下では、死者数と負傷者数はいずれも減少しており、人口10万人当たりの発生人数で見ても減少している。図3・76は、歩行者の死者数と負傷者数について、1歳ごとにその発生数と人口10万人当たりの発生数を示したものである。死者数を見ると、5歳前後と80歳前後において二つのピークが見られ、約10歳から約75歳までの範囲では年齢とともに死者数が増加し、65歳以上の高齢者の死者数が多い。負傷者数を見ると、5歳前後が最も多く、この年齢層の負傷者数は他の年齢層と比べて数倍の発生数となっている。人口10万人当たりの死者数および負傷者数を見ると、上記の特徴がより顕著になっており、特に65歳以上の高齢者になると死者数が急激に増大している。また、図3・76より、過去20年間において5歳前後の幼児歩行者の死者数と負傷者数が大幅に減少していることがわかる。図3・77は、歩行者事故における致死率(fatality
