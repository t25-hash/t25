# 3・12・3 歩行者保護試験法

> **章**: 第3章 傷害と修復のバイオメカニクス
> **節**: 3・12 歩行者保護

歩行者保護試験法は、ヨーロッパのEEVC(European Enhan-ced Vehicle-safety Committee, WG 10& WG 17)においては1980年代から、またISO(International Standard Organization,TC22/SC10/WG 2)においては1990年代初めから検討が開始された。1996年からは、日本、アメリカ、ヨーロッパおよびオーストラリアを含めた国際研究調和活動(IHRA : international har-monized research activities)がスタートし、その活動項目の一つとして歩行者保護試験法が取上げられた。2000年代になるとヨーロッパおよび日本において歩行者保護試験法の法規化が検討され、2002年からは国連(WP29/GRSP)傘下のインフォーマル会議において歩行者保護試験法の国際的な技術基準(GTR : global technical regulation)について検討を開始した。日本では、歩行者保護試験法の法規は2002年度中に公布され、新型車への適用は2005年9月となった。ヨーロッパにおいても、8―151図3・92 EEVC 歩行者保護試験法2002年現在、歩行者保護試験法の規制案(NA : negotiate agree-ment)が最終段階にある。また、ヨーロッパおよびオーストラリアでは、消費者への自動車安全情報提供(new car assessment program、自動車アセスメント)として、EEVC提案の歩行者保護試験法に基づき歩行者保護性能の評価試験(ヨーロッパ:Euro-NCAP、オーストラリア:Aus-NCAP)が実施されており、評価結果が公表されている。図3・92は、EEVCが提案している歩行者保護試験法を概説したものである。同試験法は、歩行者のおもな衝突部位である頭部、脚部および腰部/大腿部を模擬したインパクタ(衝撃子)を車両に衝突させ、それぞれのインパクタで測定した各種物理量(加速度、荷重、変位ほか)をもとに、供試車両の歩行者保護性能を評価する手法であり、一般的にはサブシステムテスト(subsystemtest)と呼ばれている。図3・93は、EEVC歩行者保護試験法において使用されるインパクタである。子供頭部インパクタ(child headform)は直径130mm、質量2.5kg、大人頭部インパクタ(adult headform)は直径165mm、質量4.8kg である。それぞれ半球体の周囲をラバーで包む構造となっており、重心位置に加速度計が取付けられている。衝撃速度は40km/hであり、衝撃角度は子供頭部インパクタが50°、大人頭部インパクタが65°である。衝撃エリアは、WAD(wrap arounddistance、図3・94)により、子供エリアと大人エリアに区分される。WAD 1000~1500mmの範囲が子供エリアであり、WAD1500~2100mmの範囲が大人エリアとなる。脚部インパクタ(legform)は、人体の大腿部と下腿部を模擬した上下二つの金属製の円柱を膝中心で一対の金属板(人体膝部の曲げ特性を再現したものであり、膝部材と称す)を用いて結合し、その外側を発泡材(foam flesh)とゴム製表皮(skin)で取り巻く構造となっている。質量は約13.4kg であり、下腿部に取付けた加速度計(膝中心から66mm下方)から脛骨上端部に相当する位置での加速度を計測するとともに、大腿部および下腿部の内部に取付けた二つの角度計から膝中心におけるせん断変位と曲げ角度を計測する。脚部インパクタは、車両との衝突時にインパクタ下端が地表面と同一の高さになるように、衝撃速度40km/hで水平に射出される。大腿部インパクタ(upper legform)は、衝撃面が半円柱状になっており、発泡材とゴム製表皮で周囲が覆われている。フロントメンバの上下支持部に取付けた二つの荷重計によって衝撃時の荷重を計測するとともに、フロントメンバの3箇所に張り付けたひずみゲージにより曲げモーメントを計測する。大腿部インパクタは、他のインパクタとは異なり、質量、衝撃速度および衝撃角度を車両の前部形状によって変化させる。サブシステムテストでは、インパクタの力学的特性が人体各部の諸特性と等価であること、すなわち、インパクタは人体各部の生体忠実度(biofidelity)を満足することが重要となる。また、インパ8―152生体工学(a)
