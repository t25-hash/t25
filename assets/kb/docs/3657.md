# 4・2・1 運動学(kinematics)的解析

> **章**: 第4章 スポーツバイオメカニクス
> **節**: 4・2 スポーツバイオメカニクスにおける解析方法

高い技術レベルを有する選手の動作を解析することは、優れた選手の技術的要因を抽出することに役立つ。このためスポーツバイオメカニクスでは、一流選手を対象として、その身体運動を測定・分析することが多い。動作を測定する方法には、角度センサや加速度センサなどを用いて、体節に相対的な変量を計測する直接計測法、および特定の空間内の位置を変量として計測する位置計測法がある。a. 直接計測法。 角度計(電気式ゴニオメータ、goniometer)隣接する体節の表面に装着し、運動時の関節角度変化を電気信号に変換する器具をゴニオメータと呼ぶ。可変抵抗器に二つの体節への固定用アームを取付けた同心型の一軸のものは、構造が簡単であり、また直線性やヒステリシス特性の優れたものを容易に作製できる。しかしながら被験者の自然な動作を妨げる場合があるため、柔軟性に富み、動作への影響が小さいフレキシブル型が現在多く用いられている。このフレキシブル型ゴニオメータには、導電性ゴムあるいはひずみゲージを利用したものがあり、20g 以下と小型・軽量であり、また関節とゴニオメータの回転中心を合わせる必要がないなどの特徴がある。. 加速度計(accelerometer)加速度計を体節や運動用具に張り付けて、その部位の加速度を測定することがよくある。加速度計は、センサ部の重量が数g から20g 程度と小型・軽量であること、比較的安価であること、また大きな計測スペースを要さずに、実時間計測できることなどの理由から、ピッチング、水泳、歩行などさまざまな動作を対象として利用されている。ただし、出力信号に、重力の加速度、体節の回転運動成分や並進運動成分が混在することに注意を要する。また身体重心等の加速度信号を2回積分すると、その位置情報が得られるが、長時間の測定では混入した誤差の影響が大きくなるため、補正が必要となる場合が多い。b. パンニングDLT 法固定DLT 法では、計測範囲が大きな場合にその分解能が低下し、同時に分析対象が小さくなってしまうため、計測精度も低下することになる。これらの問題点を解決するための一つの手法が、分析対象の移動に合わせてカメラをパンニング(水平回転)させながら対象を撮影するパンニングDLT 法であり、いくつかの方法が提案されている。ユらの方法では、まず分析対象範囲をいくつかの小領域に分け、複数のコントロールポイントからなるコントロールオブジェクトを各小領域に設置し、これらをカメラをパンニングさせながら撮影する。次に個々のコントロールオブジェクトを利用して小領域ごとのDLT パラメータを算出する。そしてこれらのパラメータを、地上に置いたリボンマークから得たカメラ角度を変数として多項式近似し、任意のカメラ角度時のパラメータを内挿により求め、分析点の座標算出に用いる。高松らの方法では、まずコントロールポイントのほかに、分析対象範囲内に複数の基準点を設け、画像面座標における基準点から各コントロールポイントまでの相対座標をあらかじめ求めておく。そして分析時には、画像面上の基準点と前述の相対座標を用いて、画像面上に仮想のコントロールポイントを設け、これらの座標を用いてDLT パラメータを逐次算出し、分析点の三次元座標算出に用いる。この方法は仮想校正点法と呼ばれる。. マグネティックデバイス(magnetic device)トランスミッタにより発生された磁界を、身体各部に貼付した受信コイルによって検出し、磁界の強度と向きから受信コイルの位置と姿勢を算出する。この手法の利点は複数のコイルの同定が時分割方式で自動化できること、およびマーカの隠れの問題がないことである。ただし測定空間の内部または近傍に磁性体が存在すると、磁界の分布が変わり、測定精度に影響を与える。c. モーションキャプチャ(motion capture)モーションキャプチャは動作を自動的に計測するシステムであり、光学式、機械式、磁気式がある。モーションキャプチャでは、大量の情報を連続的に処理できるため、多量の動作を扱うことが可能となる。8―157光学式は、複数台のカメラで同一のマーカを自動的に検出し、DLT 法を利用してその三次元座標を算出するものである。検出方法としてビデオカメラを用いるもの、半導体検出器を用いるものがある。. マーカの自動検出による位置計測装置ビデオカメラによって、体節に取付けた反射性能の高いマーカを撮影し、周囲との輝度の差からマーカの位置情報を取出す方式である。CCD 画像上でのマーカのコントラストを高めるために、赤外光や高輝度LEDを照明として用いることが多い。なお、視野の中には複数のマーカが入ってくるため、各マーカを同定する必要が生じる。また、マーカの同定を簡便にする方法として、可視光のもとで異なる色のマーカを取付け、色の違いによって識別する方法がある。この方法では、着衣や背景にマーカと近い色が混じらないように注意を要する。. 半導体方式位置計測装置カメラレンズの結像面すなわち受光面に二次元半導体を配置し、赤外領域のLED を用いた能動マーカの結像面上の座標を、二次元半導体の4辺からの距離に応じて流れる電流から算出する方式である。複数マーカの同定問題を避けるために、個々のマーカを時分割して発光させ、各マーカを認識する。なお、LED にはかなりの電流を必要とするため有線方式となっている。また結像面座標の検出の直線性を高めるために、二次元半導体の代わりにフォトセルアレイを用いるものがある。機械式は、各測定部位の位置情報ではなく、ゴニオメータを多用して全身の各関節角度情報を得るものである。この方式ではカメラや機器設置空間等の大規模な設備を必要としないが、その装備が擾乱となるため、スポーツ動作の測定における適用範囲は限られる。なお、磁気式には前述のマグネティックデバイスが該当する。一般に、モーションキャプチャによる動作解析は、身体各部へのマーク付けが必要であり、また背景や照明による制約のため、競技会といった公式試合ではなく実験室での適用がほとんどである。
