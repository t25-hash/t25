# 10・2・3 鋳造品の欠陥

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・2 金属材料の欠陥

a.ひけ巣（shrinkage cavity) 最終凝固部にできる収縮孔で
内びけと外びけがある．押湯不足，鋳込温度の不適が原因である．
b.空孔 球状の丸い孔で直径2～3mm以上をブローホール
（blow hole)，それ以下のものをピンホール（pin hole）という．
溶湯および鋳型から発生するガスによる．にえ（boiling）は同じ
原因で鋳肌があばた状になったものである．
c .ミクロキャビティ（micro cavity) 最終凝固部に生じる微
小な樹枝状の小孔の集まり．俗にザクともいう．
d.高温割れ（hot tear ) 最終凝固部に発生する．凝固冷却時
の体積収縮による引張りの応力が原因である．多くはその破面が酸
化している．
e.低温割れ・冷間割れ 常温近くになり冷却の不均一による
内部応力が原因となって発生するもろい破壊．破面は，へき開面を
示すことが多いので高温割れと容易に区別できる．防止には内部応
力が生じないように冷却する．常温に冷却する前に再加熱を行うと
効果がある．
f .砂かみ（sand inclusion) 鋳物の表面，内部に砂が介在し
たもの．鋳込み中に洗われ，すくわれなどにより型砂を巻込む場合
が多い．
g.洗われ（cut and wash) 鋳込み中，鋳型表面の砂が洗い
流され鋳肌に不規則な出っ張りを生じたもの．焼付き，砂かみを伴
うことが多い．
h.すくわれ（scab) 鋳型表面の一部がはく離し溶湯が浸入
したもの．鋳型の熱間強さを高くする必要がある．
i .焼付き 鋳物砂が表面に付着して，砂落しが困難になる現
象をいう．浸透型と非浸透型がある．浸透型は，溶鋼が砂粒間に浸
透し，鋼と砂が混ざり合った状態で鋳肌に固着したもので差込みと
もいう．非浸透型には，化学反応によるものと砂の焼結溶融による
ものがある．
j .ベイニング（veining) 鋳物の表面に生じた板状に盛り上
がった出っ張りで鋳型の角すみ部に生じやすい．
k.湯境，湯回り不良 いずれも湯流れが不十分で起こる欠
陥．湯境（cold shut ）は，溶鋼が合流した部分に酸化膜が介在し
湯じわが生じる．湯回り不良（misrun）は，溶鋼が鋳型内を完全
に満たす前に凝固したものである．
