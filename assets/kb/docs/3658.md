# 4・2・2 運動力学(kinetics)的解析

> **章**: 第4章 スポーツバイオメカニクス
> **節**: 4・2 スポーツバイオメカニクスにおける解析方法

身体を構成している部分は分離できないので、質量や重心を正確には測れないし、関節間の内力や筋力を直接測ることも困難である。したがって、運動の原因となる力の働きから結果としての運動をたどるという順方向の問題設定だけではなく、結果から原因を推定するという逆問題解析の発想が重要になってくる。a. 外力の測定法を基にした運動の解析。 地面反力と画像のセンサフュージョンによる身体重心速度の推定地面反力は運動者の重心の加速度と体質量の積を表している。一般に重心の変位、速度は画像分析法と他者の身体部位係数を組合わせて変位を推定し、それを微分演算して速度を算出するので、本人の実際の値ではなくさらに微分誤差も増大する。. 運動の周期境界条件の仮定による地面反力からの重心速度、変位の推定運動の周期性を仮定できる場合は、画像分析を用いないで地面反力から重心の速度波形、変位波形が精度よく短時間で求められる。走行で、着地の瞬間から空中動作を経て再び着地の瞬間までの運動の周期性を仮定すると、まず縦軸の地面反力目盛の静止体重が加速度目盛0になり地面反力0が空中動作中の加速度-9.8m/sになるように目盛を変換する。運動の周期性により、着地瞬間の速度の初期値を零と仮定して加速度波形の積分を進め、その値が初期値と同じ零になったときが空中動作が終わる1周期となり、仮の初期値の速度波形が求まる。次に変位の初期値を零として、仮の速度波形を1周期積分した値が零になるような速度の初期値を探索して求めれば、それが初期速度となる。. 張力計と加速度計のセンサフュージョンによるハンマ投げの曲率半径の計測ハンマ投げではハンマ頭部の回転運動の曲率8―158半径は、投てき技術の面から重要視されているが、身体の移動や変動する回転軌道面の傾斜等による姿勢の激しい変化などにより、画像による三次元解析でも精度を高めるのは容易ではなかった。ここではハンマの把手から球に向かうピアノ線の中間に直列に軽量の張力計を接続して求心力を測定し、同じ場所に把手に近い側から2個の加速度計A,Aを、検出軸がピアノ線の方向になるように間隔で装着し、加速度,を測定した。b. 身体部位慣性特性係数と測定法身体を関節部で分割し、それぞれ剛体と仮定した剛体リンクモデルにおける質量と重心の位置の推定値である身体部位係数に、慣性モーメントを加えたものを身体部位慣性特性係数と呼ぶ。スポーツ運動では画像分析した身体部位に、この係数を適用して運動力学的解析を行う場合が多い。. 生体標本を用いた各種測定法の例(1)水置換法:身体部位を水に浸して求めた体積と、仮定した密度の積から質量と重心を推定する方法、(2)物理振り子法:関節を軸とする振動周期から慣性モーメントを推定する方法、(3)CT 法:放射線の吸収率が組織密度に比例する性質から質量、重心、慣性モーメントを推定する方法などがある。. 数学モデル法画像計測により求めた身体部分の形状を円柱、円すい等の剛体モデルに置換えて質量、重心、慣性モーメントを推定する方法であるが、身体の形状との近似精度を高めるために、身長軸に直交して積層状に裁断した薄板をだ円板に近似(積層だ円板近似)し、各だ円板の体積と身体密度(推定値)から、各種の慣性特性係数が求められている。c. 逆問題(inverse problem)解析科学および工学においては、入力を与えてそれに対する応答を求める順方向の問題が多い。しかし特に何らかの新しい現象の内容を把握しようとするとき、材料特性や支配方程式がどう表されるかなど全くわからず、与えられるのは現象の観察結果のみである。このように観察を基に現象を理解する、すなわち出力から入力を求める過程が逆問題的アプローチである。これらの逆問題の解析を行う操作に対する呼び方としては、同定(identification)、推定(estimation)、逆計算(backcalculation)など問題により種々である。ここでは走運動に例をとりながら考えてみたい。. ランニングスピードと下肢の筋電図走速度が増大したときの下肢の働きと筋活動を、選手の走能力による比較と、個人が走速度を次第に増加させた場合の両面から考察した結果、膝関節と股関節にまたがる大腿直筋は、優秀な選手では低速ランニング中、一般人では速度にかかわらず、着地期の活動が滞空期のそれよりも大であった。走速度が増大するに伴い、滞空期の活動が位相を前に移動しながら著明に増大し、全力疾走中では、滞空期の活動が着地期の活動を大きく上回った。これは生理学的な面からの逆問題解析である。. ランニングの剛体リンクモデル(rigid body link model)競技中の運動計測には映像記録による以外にない場合が多い。画像から読み取った各関節の座標値と、関節により区分された部位の質量、重心、慣性モーメントの推定値を基に、各関節の発揮しているトルクを推定しようとする場合に用いられるものに剛体リンクモデルがある。この研究は、走運動を大転子を軸とする大腿の運動と膝関節を軸とする下肢の運動がリンクされたモデルと考え、接地期と滞空期に分け、接地期には足関節座標が固定されているという境界条件をモデルに入力した。モデルは、両脚の下肢、大腿の鉛直線に対する傾角θ~θおよび大転子の,座標を一般化座標に選び、各関節トルク~を一般化力とするラグランジュの運動方程式で表し、映像から読んだ一般化座標から関節トルクを推定する逆解析を行った。各被験者とも走速度の増加に伴い、大腿の振動の中心が前方に移り、その振幅が大きくなる。この大腿が上がってくるその傾向は、熟練者において顕著である。そして股関節トルク、膝関節トルクの振幅は、走速度増加に伴いともに増加していくが、股関節トルクのほうが膝関節トルクの増加率より大きい。この結果はi項で見た生理学的変化の状態を力学的変化の状態で表現していると考えられる。. ランニングの着地衝撃の2自由度可変質量粘弾性モデル(viscoelastic model)ランニングの着地衝撃では、初めに足の部分が床と衝突するときに尖った短時間の衝撃(受動的負荷)が生じ、続いて全身の推進動作による時間幅の広い床反力(能動的負荷)が生じるので、二つのピークが複合した波形が生じる。両質量間の明確な境界が特定しにくいことと、さらに走り方や筋の収縮の仕方によっても質量配分が変わるために、質量配分をパラメータとして扱った。各質量の入射速度は、パラメータとして扱い、合成速度が測定された重心の入射速度に等しくなるように設定した。このようにして、図4・2の運動方程式は、圧縮過程と復元過程で異なる非線形粘弾性要素のパラメータを含めて12のパラメータとなった。パラメータは非線形計画法の座標回転法[ローゼンブロック(Rosenbrock)の方法]を用いて探索を行った。各パラメータが決まれば、微分方程式の初期値問題として解き、実際の着地衝撃波形との比較を行う。解析の結果の一部を示すと、パラメータ探索によって求められたは多少のばらつきはあるが、走速度が増加するにつれ増加する傾向がある。このことは、走速度が小さいときは、は大転子と離れた下腿部の動きに関係が大きいためには小さいと考えられる。走速度が大きくなるとは大転子に近い大腿部の動きにも大きく関係し、このためには増大してくると考えられる。動作により、身体各部位間の協調的な結びつきの範囲や強さが変わり、身体組織の粘弾性も変わった結果がモデルのパラメータに反映され、それが重心の運動として統合されて床反力波形に現れてくることが考察された。.
