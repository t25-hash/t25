# 10・2・5 溶接欠陥

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・2 金属材料の欠陥

a.形状不良 未溶着部を残した溶込み不良（poor penetra-
tion)，溶接金属が母材に重なったオーバラップ（overlap)，逆に，
埋まり切らないで溝になったアンダカット（undercut ）がある．
b.溶接割れ（weld cracking) 溶接金属割れ（weld metal
cracking）は多くは凝固時に発生する高温割れである．ビードに
平行な割れを縦割れ，直角な割れを横割れという．終端に発生した
ものをクレータ割れという．母材の溶接熱影響部（HAZ:heat
aﬀected zone）割れのうち，低温割れは溶接により侵入した水素，
拘束応力，HAZの硬化組織が主因で，発生する場所により，ルー
ト割れ（root crack)，トウ割れ（toe crack)，ビード下割れ
（under bead crack）と呼ばれる．
c .スラグ巻込み（slag inclusion) 溶接金属中に溶接スラグ
が残存したもの．ブローホール（blow hole）は気泡が凝固時に離
脱できず溶接金属内に残留したものである．
d.ラメラテア（lamellar tearing) 溶接により介在物が開口
し段階状に連なったもの．板厚方向に大きな応力が発生するような
部材には耐ラメラテア鋼の使用が必要である．
e.SR割れ（stress relief cracking) 応力除去（SR）焼なま
しを行う際，HAZの粗粒域に発生する粒界割れ．低合金高張力
鋼，Cr - Mo鋼に見られる．
