# 4・2・4 シミュレーションによる解析

> **章**: 第4章 スポーツバイオメカニクス
> **節**: 4・2 スポーツバイオメカニクスにおける解析方法

身体運動の動力学的解析は,実際に行われた身体運動から関節トルク,モーメント,筋力などを推定する逆動力学的解析(inversedynamics)が主流である.しかしコンピュータの急速な発展とともに,関節トルクや筋力から仮想的な身体運動を発生させる順動力学的解析(forward dynamics),すなわちコンピュータシミュレーション(simulation)もスポーツバイオメカニクスの研究手法の大きな柱になりつつある.身体運動のシミュレーションを行う目的はさまざまであるが,以下のようなものが考えられる.(1) 生体内力の推定:実際に行われた身体運動を基に,コンピュータ上で関節角度などを変化させ,生体内に作用する力や筋力がどのように変化するかを明らかにする.(2) 運動の力学的原則の理解:体操競技,トランポリン競技や水泳の飛び込み競技などで見られるひねりを加えた宙返りがどのようにして行われるかなど,身体運動の力学的原則を明らかにする.(3) 動作基準の推定:身体は関節に冗長性をもっているため,ある目的の運動をするためには,さまざまな運動パターンをとることができる.例えば身体の前方にある物体に手を伸ばす場合,矢状面内だけで上肢を動かすことも可能である.また肘を肩の高さまで上げておき,水平面内だけで上肢を動かすことも可能である.しかし,実際には何らかの基準に従って運動が制御され,自然な動作を行っている.そこで,シミュレーションによりさまざまな仮想的な身体運動を作り,実際に行われる動作の基準を明らかにする.(4) 最適動作(optimal motion)の究明およびスポーツ技術の開発:消費エネルギーを最小にするような,また跳躍距離を最大にするような最適動作を推定したり,身体特性(形態や筋力)と最適運動との関連を明らかにする.さらにさまざまな仮想的な動作をコンピュータ上で作ることにより,新しいスポーツ技術の開発や競技成績を向上させるための動作の改善点を明らかにする.(5) 人と用具のマッチング:スポーツ工学におけるスポーツ用具の解析と組合わせることにより,選手に最適な用具の設計を行うとともに,用具に最適な動作の推定を行う.上述した目的のためにさまざまな身体運動のシミュレーションが行われてきているが,以下に代表的な研究を紹介する.ハッツェ(Hatze)は,大腿,下腿(+足部)の2セグメントと5組の筋群からなる二次元筋骨格モデル(musculoskeletalmodel)を構築し,シミュレーションと最適化手法を用いて,最短時間でターゲットをキックする動作(車の急ブレーキを踏むような動作)を推定した.そして,被験者にシミュレーションで推定した最適動作を提示せずに練習させると動作時間は短縮していき,約120回のトライアルで動作時間はほぼ一定になった.しかし,シミュレーションで推定した最短動作時間よりは長かった.そこで最適動作(optimal motion)と実際の動作との違いをチェックしながら練習を繰返すと,動作時間はさらに短縮してシミュレーションで推定した動作時間とほぼ一致した.またハッツェは,17個の剛体リンクと46組の筋群からなる三次元筋骨格モデルを構築し,走り幅跳びの踏切動作のシミュレーションを行った.シミュレーション結果を利用して3週間の練習を行ったところ,32回の跳躍の平均記録6.58m,最高記録6.96mであった選手が,平均記録で7.12mに向上したと報告している.これらは身体運動のシミュレーションが最適動作の推定に利用できることを示した例としてよく取上げられるものである.多賀は,下肢の剛体リンクモデルにリズム発生回路と神経振動子を組込むことで,コンピュータ上で歩行動作を発生させることが可能であることを報告し,歩行動作の制御理論について言及している.また,定常歩行中に外乱が加えられても安定した歩行を継続できるようなモデルの構築も行われている(長谷ら).そのほかにも,筋収縮シミュレーションや血流シミュレーションなど,身体運動に関連したさまざまなシミュレーションが行われている.このように,スポーツのみならずさまざまな身体運動のコンピュータシミュレーションが行われているが,その利点および欠点をまとめると以下のようになる(阿江・藤井).利点としては,次のことが挙げられる.(1) 危険な試技や実験を選手や被験者に強いる必要がなく安全.例えば,ある体操選手が3回宙返りを行うことができるかを,コンピュータシミュレーションで確認することができる.(2) さまざまな条件でのシミュレーションを瞬時に実行可能.コンピュータに入力する各種パラメータを変更することで,さまざまな条件での仮想的な試技を再現できる.(3) 最適動作の推定が可能.競技スポーツにおいては,競技成績を向上させることも可能である.(4) 物理モデルを作成することに比べて安価.ダミーロボットや実験装置を作製する必要がない.他方,欠点としては,(1)妥当性の検証が困難,(2)シミュレーション結果を実践に生かすのが困難,などが挙げられる.ここで,欠点として挙げた妥当性の検証が困難であることを示す例を紹介する.垂直跳びにおける二関節筋(腓腹筋や大腿二頭筋など)の役割を検討するために,シミュレーションを行った複数の研究がある.ボバート・シュナウ(Bobbert-Schenau)やフジイ・モリワキは,腓腹筋などの二関節筋を単関節筋に置換えた筋骨格モデルを用いて垂直跳びのシミュレーションを行うと跳躍高が減少するので,二関節筋は高く跳ぶために貢献していると報告した.一方パンディ・ゼイシャック(Pandy-Zajac)は,同様に二関節筋を単関節筋に置換えたモデルを用いてシミュレーションを行い,跳躍高が増大するので貢献していないと報告した.しかしこれらのシミュレーションでは,使用したモデル(筋の最大収縮速度を表すパラメータなど)が異なっており,それが跳躍高の増減に影響を与えたものと考えられる[ファンソエスト(van Soest)ら].このように身体運動のシミュレーション結果は,身体モデルおよびそのパラメータに大きく依存しているが,上述の二関節筋の機能評価の例では,実際の身体の二関節筋を単関節筋に置換えることは不可能であり,いずれのシミュレーション結果が妥当であるかを判断することは困難である.適切なモデルを構築し,さらに適切なパラメータを代入するためには,バイオメカニクスのみならず運動生理学,解剖学,制御工学などさまざまな知識を統合する必要8―161があろう.また,CFD(computational fluid dynamics)解析を用いてスキージャンプにおける空中姿勢の変化が抗力や揚力に与える影響を解析したものや,有限要素解析を用いて身体運動を解析した研究も報告されている.従来の剛体リンクモデルや筋骨格モデルを用いたシミュレーションのみならず,工学分野の解析手法と組合わせること,さらに,身体運動のシミュレーションとスポーツ用具のシミュレーションを組合わせることで,より利用価値の高いシミュレーションを行うことができるであろう.
