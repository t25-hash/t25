# 4・3・2 運動学習

> **章**: 第4章 スポーツバイオメカニクス
> **節**: 4・3 スポーツバイオメカニクスの適用例

a. テニスサーブ(tennis serve)における巧みさの指導テニスのストローク、野球のピッチング、サッカーのキックなど、スポーツにおける各種の技術は誰もが初めは全くの素人であり、練習によって習熟していくものである。その動きを見れば、熟練者か未熟練者かはすぐに見極めができることからもわかるように、運動学習によって動きのパターンが変わり、巧みさを獲得し、より目的にかなったフォームになっていく。そこでここでは、テニスを行ったことのない初級者を対象にサーブを1年間にわたって指導し、それに伴って何が変わっていくか追跡した研究を紹介する。分析の際には、表面的な動きの変化だけでなく力の発揮や筋の作用機序の変化などに注目し、運動学習による変化をバイオメカニクスの面から客観化するとともに、これらの変化と指導内容との関係を明らかにしている。その研究では、サーブにおける巧みな動作を達成するための技術図4・6スキージャンパーの重ね合わせ(ダートフィッシュ社のサイトより)図4・7 テニスサーブにおけるインパクト時の身体とラケットの角度8―163として、(1)体重の移動を大きくすること、(2)動作を大きくすること、(3)トスの位置を安定させることの3項目に指導の重点をおき、それらの習得度を判定するためラケットの移動距離、関節・セグメントが示す角度、右足・左足の床反力、上肢筋群の筋放電パターン等を測定した。上級者2名についても同様なデータ収集を1回実施し、指導効果を判定するための材料とした。いずれの測定項目について見ても、初級者の動作が上級者の示す巧みな動作に近づきつつあることが客観的に確認され、ここでの指導法が的を射たものであったということが示されているが、このうち関節・各セグメントが示す角度と上肢筋群の筋放電パターンについて、運動学習前後の様相を具対的に紹介しておこう。b. インパクトの瞬間の各関節角度および各セグメントの傾斜角度まず、各角度について望ましい値の範囲を検討しておく(図と身体全体を上方に一直線に伸ばして打球を行う必要がある。しかし、手関節θについては上級者は177°,155°と、ややラケットを後傾させた上体でインパクトしており、必ずしも最高点でインパクトしているわけではない。これは、インパクト中はラケットが上方へ動いているためであり、180°になるのはインパクト直後にラケットが最も高い位置に達した時点であると考えられる。θの値としては、170°前後の値を示すのが理想的であろう。一般に上級者の場合は、ボールに大きな運動量を与えるために、体重の移動を伴いながら身体全体をやや前傾させた状態で打球を行うことが多い。したがって、θ~θに関しては、2名の上級者が示すように、0°~30°前後の値を示すのが理想的であろう。図4・7は、角度θ~θの運動学習前後の値を比較したものである。学習前の値を上級者の値と比較すると、θとθについては小さめかほぼ同じ、θとθについては大きめ、というパターンを示す者が8名中4名いたが、全体的には個人差が大きく一定の傾向を見いだせなかった。しかし練習によって、上級者より小さい値を示していた者は大きい値へ、逆に上級者より大きい値を示していた者は小さい値へと変化していくという傾向が、θ~θすべての角度について認められた。この結果は、初級者それぞれが高い位置でイ8―164生体工学ンパクトし、身体全体で円を描く熟練者の打法に近づきつつある、ということを示している。c. 上腕三頭筋、橈側手根屈筋の筋放電パターンサーブのフォワードスイングの際の主働筋の一部と考えられる上腕三頭筋と橈側手根屈筋の筋放電開始時期(それぞれT,Fとする)を調べた。運動学習後の代表的なものが図4・8に示してある。身体の中心部から末端へ順序よく力を発揮していくのを理想的な動作であるとすれば、T→Fの順になることが望ましく、実際上級者2名はこの順序を示していた。一方初級者では、運動学習前における順序はF→T を示すものが8名中6名いた。橈側手根屈筋はグリップを強く握る際に動員される筋であり、ほとんどの者がスイングの早いうちからグリップを強く握りしめていたとみなされる。しかし指導を受けることによって、F→T の順を示した6名のうち4名がT→Fの順に逆転し、F→T のままであった2名もFとT の間隔が大幅に短くなっている。このことは、初級者それぞれがインパクト前にむだな力を入れないスイングを習得しつつあるということを示している。
