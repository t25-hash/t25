# 4・3・3 障害予防

> **章**: 第4章 スポーツバイオメカニクス
> **節**: 4・7 )。インパクトをできるだけ高い位置で行うためには、ラケット

a. 障害と打法、用具との関係運動学習を進めていく際には、障害の発生はできるだけ避けたいものである。これまでにも打具を用いるスポーツに対しては、発生頻度の高い障害に対する統計的な調査が行われており、打法や用具はどうあるべきか検討されてきた。しかしながら、打法のどのような点が障害を引き起こしやすいのか、あるいは打具のどのような性質が痛みの原因になっているのかといった、学習者や指導者が求めている具体的な情報となると、統計的な調査だけでは得られにくい。例えばテニスの場合、実際に指導現場で行われている「手首を使って打たないほうがよい」、あるいは「軟らかいラケットに変更するように」などのアドバイス内容にしても、その有効性が客観的に確かめられているものは少ない。そこでここでは、障害と打法およびラケットとの関係を、バイオメカニクスの立場から検討し、障害の予防法および障害が発生した際の保存的治療法(conservative treatment)を提示することを目的とした研究を紹介する。b. テニス肘装具の効果その研究では、実際にストロークを行ったときの肘関節部の衝撃振動(shock vibration)を導出し、種々のテニス肘装具がテニス肘(tennis elbow)の予防・治療に有効かどうか明確にしている。被検者は男子上級者1名で、スロート下部および上腕骨外側上顆部に加速度計を装着し、テニスマシンから発射された一定球速のボールを、フォアハンドのグラウンドストロークで返球した。高速度ビデオを用いて打球部位を確認した後、2チャネルFFTを用いて衝撃力の分析を行った。インパクトによる衝撃の大きさがテニス肘発症の重要な要因になっていると考えられるので、衝撃振図4・8筋電図の運動学習前後の変化(テニスサーブの場合)動の諸要素のうちから振幅の最大値に着目し、振幅の最大値をインパクト直後に得られる加速度波形(衝撃振動)の最大値と最小値の差(以下衝撃力と呼ぶ)で評価した(図4・9)。テニス肘装具は、ストリングス部分に固定するスポンジ状のもの(D 群)、一般に市販されているエルボーバンド(E群)、リストバンド(W 群)であり、それらの効果を判定するために、何も装着しないとき(U 群)との比較を行った。衝撃力は、センタインパクトとオフセンタインパクトの二つの場合について検討した。オフセンタでの打球部位は、ラケット面の下部でスロート近辺に限定した。得られた結果を図4・10に示す。まずセンタインパクトの場合の衝撃力の大きさを検討すると、ラケットの461~506G に対し、肘関節は15.1~17.0G 種々のテニス肘装具を装着した場合のインパクト直後の衝撃力の違い到達するということがわかる。次に打球部位で衝撃力を比較すると、これまでの報告と同様にセンタよりもオフセンタでの衝撃が大きく、ラケットでは1.2~1.3倍、肘関節では1.6~1.9倍を示していた。いわゆるスイートスポットで打球をすると衝撃が小さくなることが、今回の実打実験においても再確認されたわけである。さて、現場でもしばしば話題になるテニス肘装具の効果であるが、結果は図4・10に集約されている。図4・10を見ると、ダンパ(damping material)はラケットの衝撃を緩衝するものの、肘関節部には影響を及ぼしていない。むしろ肘関節部に対しては、リストバンドやエルボーバンドのほうが高い緩衝機能を保持していることが読み取れる。リストバンド、エルボーバンドはストローク中の過度の筋活動を抑制すると報告されていることを考慮すると、バンド状の装具はテニス肘の保存的治療に際してはダンパ以上の効果が期待できる。c. 分析結果の現場への活用法以上の結果より、リストバンドやエルボーバンドは、オフセンタで打球することが多い初・中級者に対しては威力を発揮する、と断言してもよさそうである。しかし一方、ダンパを装着することの是非について、この結果をもって肘には効果なしと否定的な見解を導くことは避けるべきであろう。この研究においても、ストリングスおよびフレームの衝撃振動を抑制する作用が確認されており、上肢に対しても何らかの影響を及ぼしていると考えられるからである。今後は、振幅の最大値だけでなく、減衰率や振動数といった衝撃振動の他の要素についても検討を加えることで、ダンパの上肢に及ぼす影響を詳細に解明していく必要がある。
