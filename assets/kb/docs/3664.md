# 4・3・4 スポーツ用具・施設の開発

> **章**: 第4章 スポーツバイオメカニクス
> **節**: 4・7 )。インパクトをできるだけ高い位置で行うためには、ラケット

スポーツ用具・施設には，競技中の大きなストレスから身体を保
護する機能，身体の出力（筋収縮力）を目的とする運動に効率的に
変換し，高いパフォーマンスを発揮させる機能が要求される．モー
タスポーツなどを除けば多くのスポーツの唯一の動力源は骨格筋の
収縮であり，全身の個々の筋の収縮のタイミングと規模を調整する
ことにより，目的とする身体運動は発現される．スポーツ用具・施
設と一般的な工業製品との最も異なる点は，それらが直接的にせよ
間接的にせよ身体と機械的に結合された状態で使用されることにあ
る．スポーツ用具の開発にとって最も重要なことは，動的な状態で
その機能が発揮されるかという点にあり，目的とする身体運動を促
しながら身体への負荷を低減させるものでなくてはならない．これ
らの目的を実現するために近年スポーツ用具開発の現場では，バイ
オメカニクス的な情報の重要性が高まりつつある．同時に情報処理
装置の高速化やシミュレーション技術の発達が，スポーツ用具の設
計段階でバイオメカニクス的な情報を利用することを可能にした．
スポーツの種類は数多く，それらすべてにおいてバイオメカニクス
的情報がどのように活用されているかを記述することは困難である
ので，ここではいくつかの代表的な種目における活用事例について
記述する．
