# 5・1・1 材料の生体適合性(biocompatibility)

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

生体内で人工物を安全に使用するには,人工物と生体との適合性について考慮を払う必要がある.人工物の適合性は多くの要因に支配されるが,考慮すべきおもな事項には次のようなものがある.a. 材料に対する生体の反応材料が生体内に埋植された場合には,まず材料表面にタンパク質などが速やかに吸着するとともに,組織からの出血により材料が凝血で覆われる.材料周囲には好中球やマクロファージなど白血球系の細胞が現れ,傷害された組織や細菌の分解,貪食を開始する.次いでリンパ球系の細胞も現れ,免疫応答の初期段階が開始される.材料の周囲では,活性化したマクロファージの刺激により,間葉系細胞が増殖しコラーゲンなどの細胞外マトリックスの産生が開始されるとともに,周囲組織より新生血管が侵入し,肉芽組織が形成される.その後マクロファージなど炎症性の細胞は姿を消し,間葉系細胞は筋線維芽細胞に分化して肉芽組織を収縮させるとともに細胞数が減少して,肉芽組織は線維組織として再構築されていく.最終的には,材料は線維組織のカプセルにより包まれることになる.ただし材料からのイオン溶出が多い場合や材料が抗原性をもつ場合などには,材料周囲での炎症反応が継続するため,材料はカプセル化されずに異物として排除される.またアパタイトなどの材料が骨組織内に埋植された場合には,カプセル化せずに,アパタイト材料と骨の細胞外マトリックス成分が線維組織の介在なしで連続する場合もある.b. 材料の生物学的安全性(biological safety)無機,金属,有機材料などでは,材料から低分子量の物質が溶出して周囲の組織に拡散していき細胞に為害作用を及ぼす.無機,金属材料では金属イオンの溶出が問題であり,特にNi,Cr,Cu,Hg,Coなど重金属のイオンの毒性はよく知られている.これは金属イオンが細胞内外のタンパク質に結合し,酵素の不活性化や細胞膜の変化を招くことによるものと考えられている.有機材料ではモノマや不純物の溶出が問題になる.有機モノマ類は程度に差があってもすべて毒性をもつと考えるべきである.例えばセメント使用型の人工股関節の手術時に急激な血圧低下を生じて死亡する例が発生し問題となっているが,これは骨セメントが重合する前のメチルメタクリレートモノマの毒性によると考えられている.しかし,手術中に髄腔内の内容物が循環系に流出し肺塞栓を起こすことや,骨セメント重合時の発熱などを原因とする意見もあり最終的な結論は出ていない.またモノマについては内分泌かく乱化学物質としての作用が報告されているものもあり,極微量でも問題となる可能性があるので注意が必要である.生体由来材料では材料の抗原性が問題になり,例えばI型コラーゲンであればコラーゲン分子の三重らせん構造の両端にある抗原性の強い部分を酵素処理で除いたアテロコラーゲンとして利用される.生体分解性材料では材料の分解産物が問題になっているが,例えばポリ乳酸では材料の分解により生じる乳酸により周囲組織が酸性となり,炎症を生じる可能性が指摘されている.わが国においては,医療用具の製造(輸入)承認申請に際し,その生物学的な安全性を確認するために必要な生物学的試験について,厚生労働省によるガイドラインが定められている.その内容には,細胞毒性試験,感作性試験,刺激性試験,急性全身毒性反応試験,亜急性毒性試験,遺伝毒性試験,埋植試験,血液適合性試験,および慢性毒性試験,発がん性試験,生殖/発生毒性試験,生分解性試験などがあり,培養細胞や実験動物を使用する試験方法が規定され,定量的な評価を行うことができるようになっている.c. 感染性(infectiveness)生体に適用する人工物が感染源となり疾患を生じさせることは,あってはならない.これまでにも乾燥ヒト脳硬膜によるクロイツフェルト・ヤコブ病の感染が発生し大きな問題となっている.このような生物由来材料では,未知の病原体による感染を防止するために最大限の注意が必要となる.生物由来材料については後述するEOG 滅菌やγ線滅菌などが常用されるが,このような方法では滅菌ができない病原体が存在することは事実であり,材料供給源であるドナーの選択が重要となる.このためヒトまたは動物由来,特にウシ由来の物質の取扱いについては厚生労働省の通知が出されている.金属などの人工材料では材料自体の感染性が問題になることはないが,表面に付着する病原体を死滅させるための滅菌処理は必須である.現在利用されている滅菌法には,高圧水蒸気を用いたオートクレーブ滅菌,高温を利用した乾熱滅菌,エチレンオキサイドガスを利用したEOG ガス滅菌,放射線を利用したγ線滅菌および電子線滅菌などがあり,材料特性に合わせて選択利用されている.これらの滅菌操作により材料は必ず劣化するので,滅菌の影響はつねに留意する必要がある.またEOG 滅菌では材料に残留するEOG が問題になることがある.生体分解性材料では,表面のみならず材料内部までの無菌性が要求される.医療用具の無菌性を保証するための検証方法については,厚生労働省の基準が設けられている.d. 術後感染(postoperative infection)健全な組織に細菌単独で感染を生じさせるには極めて大量の細菌が必要であり,皮下組織であれば組織1g 当たり10万個以上を要するといわれている.しかし人工材料があると,その1/500~1/5000の数の細菌で感染が生じると考えられている.生体内に埋植された後でも,血行を介した細菌感染が発生する可能性が排除できず,生体組織内に人工材料が存在すること自体が感染に対する危険性を高めている.さらに生体に挿入されたカテーテルやペースメーカなどの周辺で感染が成立した場合には,いくら抗菌剤を投与しても挿入物を抜去しない限り感染は治癒しないといわれている.人工関節手術などにおいてはクリーンルーム内で手術を行うなど,感染の防止に最大限の配慮がなされているが,それでも1%ほどの術後感染が生じ,その治療には極めて難渋する.生体分解性材料の場合には,最終的に材料がなくなった時点では材料に起因する感染の危険性はなくなる.しかし分解に長期間を要する場合では,分解が生じるまでは非分解性材料と同様の危険性をもつ.材料の周囲で感染が生じやすい原因としては,手術による侵襲の影響や材料周囲で血行が障害され感染しやすくなっている可能性も考えられるが,材料自身を原因とすることが大きい.材料が感染を招く一つの機構は,材料表面に凹凸があり死腔ができる場合である.これは組織からの浸出液で満たされ,血管はもちろん周囲組織が入り込まない空間であるが,このような死腔は細菌の増殖に好適となり,しかも細菌が生体の防御機構から逃れる環境となってしまう.多孔性の材料では材料の内部にも死腔を生じる可能性があるので,特に注意を払う必要がある.材料の存在が感染を招きやすくするもう一つの機構は,バイオフィルム(biofilm)である.材料表面に細菌が付着すると,細菌は細胞外に多糖やタンパク質を分泌しながら増殖していく.このため細菌のマイクロコロニーが材料表面を覆う分泌物の層の中に埋め込まれた状態で形成され,バイオフィルムと呼ばれている.バイオフィルム中の細菌は,好中球やマクロファージなどの食細胞から守られ,さらに抗菌剤に対しても抵抗性を増すことから,いったん感染が成立すると難治性となる.カテーテル,カニューレ,ドレナージチューブ,心臓ペースメーカ,人工関節などで細菌によるバイオフィルム形成が生じると報告されている.e. 材料の形状効果材料の化学的な性質のみが生体組織に影響を与えるわけではない.材料の形状(塊,多孔体,膜,メッシュ,繊維)や大きさにより材料の生体為害性に差異が生じる場合がある.まず材料の大きさが細胞に貪食されるほど小さいかどうかが問題になる.非分解性の材料で本来はカプセル化され生体内で安定的に存在できる材料であっても,大きさが小さくなると貪食され,しかも細胞内で分解されないので炎症反応が継続してしまう.よく知られた例にアスベストがある.これはケイ酸マグネシウム塩を主成分とする長さ10 m,直径がその1000分の1程度の繊維状結晶である.肺に吸入されるとアスベストはマクロファージに貪食されるが分解はされず,アスベスト繊維を包み込んだ線維化巣を形成して肺線維症を引き起こす.そして慢性的な炎症を生じて活性酸素を産生させ,DNA を傷害して発癌に至るとされている.動物実験ではアスベスト繊維の長さにより組織に及ぼす影響が異なるので,形状により体内での動態や刺激性が異なると考えられている.また人工関節では,しゅう動部に使用されるポリエチレンが摩耗して発生する粉が骨吸収を引き起こすことが知られているが,膝関節と股関節では摩耗粉の大きさと形状が異なり,生体への影響も異なるといわれている.材料が大きい場合には,その大きさ自体が問題になる可能性がある.例えばラットなどの小型の実験動物に比較的大きな材料を埋植することにより,発癌する場合が知られている.これは移植物が周囲組織に対して機械的な刺激を与えたり,あるいは循環障害をもたらすなどの機構により生じるものと考えられている.ただしヒトにおいては,このような機構による発癌は確認されていない.f. 材料の物理的生体適合性埋植した材料と生体組織の力学特性が適合しないことにより,問題を生じることがある.例えば骨折治療用の金属プレートを使用した場合,骨の癒合が生じた後もプレートを除去しないでおくと,プレート近傍の骨量が減少することがある.これは骨への荷重がより剛性の大きいプレートを通じて伝達されることから,プレート近傍の骨は免荷され,荷重の減少に適応して骨量が減少することで生じるものと考えられている.このような現象はストレスシールディングと呼ばれており,人工股関節大腿側コンポーネントの金属ステムや膝前十字靱帯の人工靱帯による再建などでも観察されている.g. 生体内での材料の破綻生体内に埋植された人工物は,移植後しばらくは意図した機能を発揮することができるが,人工物の劣化は進行していく.生体内で材料がカプセル化された後は,ほとんど生体外での液中浸漬状態と同じはずであるが,金属材料の場合は生体内での酸素分圧が低いため不動態膜の形成が悪いことから,腐食疲労が発生することがある.特に人工物どうしが接触する箇所においては,フレッチング腐食疲労が問題になる.また異種の金属が近接して置かれる場合には電池を形成し,電気化学的な腐食を生じることもある.無機材料では脆性が問題となる.強度的に十分な余裕をもたせたつもりでも,生体内における荷重状況は十分に明らかではなく,予期しない破壊を経験することがある.有機材料では,荷重負荷による疲労はもちろん,低分子物質の溶出や脂質の吸着による脆化,吸水による加水分解,結晶界面での酸化などさまざまな原因で劣化する.材料の劣化が進むと,イオンの溶出や分解産物の発生などにより,材料のカプセル化という生体内で材料が安定して存在するための前提条件が崩れる場合がある.いったんこのような状態に至ると,マクロファージの集合など生体側の反応が開始するが,そうなるとマクロファージによる活性酸素の放出などにより材料の劣化もますます加速される.こうなると,時間をおいても改善する可能性はほとんどなく,材料を除去せざるを得ない.このように生体内で材料は,力学的因子,化学的因子,生物学的因子の複合的な影響を受け,耐久性の評価は極めて難しい.現在のところ,ISO規格やASTM 規格において,一部のインプラント製品の疲労強度試験などが規定されているが,試験方法を定めたものが多く,製品が満足すべき耐久性についての標準化はまだ十分でない.8―171
