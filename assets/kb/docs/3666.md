# 5・1・2 金属材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

金属材料は適度な強度と弾性があり,加工性に優れていることから,古くから歯科治療や骨折の固定用に用いられてきた.口腔内で使用される歯科材料と,生体内に直接埋植されるインプラント用材料とでは,使用方法や安全性に違いがあるため,ここでは分けて取上げることにする.a. 歯科材料(dental materials)歯科に用いられる金属材料は,金合金(gold alloys),銀合金(silver alloys),コバルトクロム合金(cobalt-chromium alloys)に大別することができる.さらに1980年代からチタン合金や超弾性チタンニッケル合金(super-elastic titanium-nickel alloys)が使用されるようになってきた.歯科においては,歯列矯正の場合はワイヤとブラケットと呼ばれる金具など加工材を用いるが,その他は大部分精密鋳造により作製されるため,鋳造用合金が多い.金合金は歯科用金属としては最も歴史があり,現在でも重要な役割を果たしている.金それ自体は軟らかいため,Ag,Cu,Pt,Pdなどが合金元素として添加されている.金合金に関しては,JIST 6116:2000(ISO 1562:2004)に歯科鋳造用金合金の規格があり,力学的性質により使用目的別に四つのタイプに分けられている.合金組成については規定がないが,表5・1には市販の歯科用貴金属合金の代表的な組成と硬さについて示した.金合金は伝統的にカラットで呼ばれることが多いが,金100%が24カラットに相当する.白金加金というのは,白金を少量加えると優れた弾性を発揮することから,強度と弾性が必要な部分用として白金加金と名付けて市販されている.また陶材焼付用合金は,陶材を焼き付けるため十分な強度が必要である.銀合金は,JIS T 6108に規格があり,金を含まない銀含有量60mass%以上のものと規定されている.1種と2種に分類されており,1種はインジウム(In)含有量5mass%未満のもの,2種はIn含有量5mass%以上で白金族元素10mass%以下ということになっている.銀合金は金合金と比較すると低価格で,融点が低いため手軽に使用できるという特徴があるが,耐食性があまりよくないうえに,硫化により黒化しやすいために変色をいかに防ぐかが重要である.金銀パラジウム合金は,金12mass%,Pd20mass%,Ag 40mass%以上と,銀を主成分とする合金であるが,習慣的にこのように呼ばれ,国内では広範囲に用いられている.コバルトクロム合金は,弾性率が210GPaと高いことからおもに金属床に用いられている.歯科鋳造用コバルトクロム合金については,JIS T 6115:1998において,Cr25mass%以上,Mo 4mass%以上,(Co,Cr,Ni)の合計が85mass%以上でNiを0.1mass%以上含むものはNiの溶出量1.0mg/cmを超えてはならない,となっている.引張強さは685MPa以上,耐力は500MPa以上である.表5・1 歯科用貴金属合金の代表的な組成(mass%)種類AuAgCuPtPdその他硬さ(HV)金合金20カラット83.55~11 medicaluses)体内インプラント用材料は,おもに骨折の固定や人工関節に用いられている.大別すると,ステンレス鋼,コバルトクロム合金,チタンおよびチタン合金の3種類である.最近では生体適合性に優れているチタン合金の使用割合が増加している.そのほかにもタンタルや白金などもペースメーカや神経系の電極などとして少量用いられている.体内インプラント用材料の規格については,ASTM
