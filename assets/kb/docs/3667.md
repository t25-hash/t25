# 5・1・3 無機材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

a. 概説生体材料に分類されている数種類の無機材料(バイオセラミックス、bioceramics)は比類のない生体親和性を有し、インプラント材の開発に不可欠である。これらは2群に大別される。一つはハイドロキシアパタイトを中心とするリン酸カルシウム類である。リン酸カルシウム類は脊椎動物の骨や歯などの硬組織に見られる無機物質とほぼ同一の組成や構造を有し、顕著に生体組織と反応する生体活性(バイオアクティブ)材料群である。他の一つはリン酸カルシウム類以外の生体親和材料群である。アルミナや部分安定化ジルコニア、あるいはシリカ(SiO)やリン酸分を主成分とするガラスなどがこれに分類される。これらは従来は生体不活性(バイオイナート)材料として分類されてきたが、長期間の動物体内での埋入実験により生体組織との干渉作用があることが指摘されてきているので、生体不活性という表現は必ずしも適切ではない。特にバイオガラスと呼ばれるNaやCaを含むリンケイ酸ガラスは骨と結合することが報告されている。無機材料が医用に供されるのは人工歯根や人工骨のような生体硬組織の代替材料、あるいは骨補てん材のように骨修復材料が主である。リン酸カルシウム類の水和硬化反応を利用して歯科用セメントが実用化されている。また、セラミックスの機械的短所を補う目的で金属やポリマと複合化したバイオマテリアルやハイドロキシアパタイトをコーティングしたバイオマテリアルも開発されている。歯科用には長石や石英、陶土(AlO・2SiO・2HO)からなる陶材が歯冠(ポーセレンジャケットクラウン)として供される。陶材の特徴は優れた審美性で、天然歯に酷似した色調や光沢を有する。陶材の着色にはZnO,TiO,FeOおよびFeO,CoO,InOなどの酸化物が使用される。b. リン酸カルシウム類。 ハイドロキシアパタイト(hydroxyapatite)ハイドロキシアパタイトの組成はCa(PO)(OH)(HAp)で表される。結晶構造は一般的には空間群6/mをもつ六方晶(=0.9432nm,=0.6881nm)に分類される。六方晶アパタイト構造はOHの格子位置にフッ化物イオンが配位した場合に最も安定化される。HApは生体親和性以外にも触媒能やタンパク質の吸着性、センサ応答性など重要な性質をもつが、これら諸性質の発現に格子水酸化物イオンが重要な役割を果たす。また、特にバイオマテリアルとしては焼結プロセスを経て作製されるセラミックスが多く用いられるため、格子水酸化物イオンの熱的安定性を十分に理解しておく必要がある。水和物や水酸化物は通常は100~300°C程度で脱水が起こり、分解するが、HApは上述のように高温で焼結できるので他の水和物や水酸化物と比較すると非常に熱的に安定で、構造中に水酸化物イオンを強固に保持している。従来は大気中の水蒸気分圧が零でも1300°C以下ではHAp構造は分解しないと考えられていた。しかしながら複素インピーダンス法による最近の高精度測定解析により、HApは500~700°C程度の温度域で徐々に部分脱水を起こすことが明らかにされている。脱水が進行し、分解が起こる場合は以下のように表せる。HAp→3Ca(PO)+CaO+HO(↑)あるいはHAp→2Ca(PO)+Ca(PO)O+HO(↑)したがってHApセラミックスの焼成雰囲気に水蒸気を導入することは製品の性質に影響を及ぼすことを念頭においておく必要がある。. その他のリン酸カルシウム(calcium phosphate)CaO-PO系にはHAp以外にも多種類のリン酸カルシウム化合物が生体材料として用いられる。リン酸水素カルシウムは、無水塩(モネタイト、DCPA)と二水塩(ブルッシャイト、DCPD)として存在する。また、両者の中間的な水和物も存在することが知られている。両者とも歯磨剤中の研磨剤として使用される。リン酸三カルシウム(TCP: tricalcium phosphate)は現在までにα,β,γの3種類の多形の結晶構造が決定されている。α型は高温相で1125~1430°Cで安定であるが、室温でも準安定で容易に保持できる。β型は低温相で1125°C以下で安定であり、天然に存在する鉱物、ウィットロカイト[Ca(Mg,Fe)H(PO)]と類似した結晶構造をもつ。γ型は高温高圧相で、約10GPa,1000°Cで得られBa(PO)型構造をもつ。このうち生体材料として用いられているのは、α相(α-TCP)とβ相(β-TCP)である。α-TCPは水に対する溶解度が高く、また加水分解反応によりHApになり、高濃度であれば硬化体となることから、生体用セメントとして用いられている。β-TCPも水に対する溶解度が比較的高く、生体親和性に優れていることから、生体吸収性のインプラント材として、骨充てん材などとして用いられている。リン酸四カルシウム(TeCP)はスラグ中で発見された物質で、ヒルゲンシュトッカイトとも呼ばれる。TeCPはHApより塩基性で触媒活性が高い。また水中でも室温では安定であるが、80°C以上では加水分解する。バイオマテリアルとしては、酸性のTeCPと8―174生体工学組合わせて生体用セメントとして使用される。TeCPはモネタイトと炭酸カルシウムの固相反応で合成される。固相反応は低水蒸気分圧下で行い、急冷しなければならない。徐冷すると、HApと酸化カルシウムに分解する。リン酸八カルシウム(OCP)は、Ca(PO)・0.5HO組成のアパタイト構造に類似した層とCaHPO・2HO組成のブルッシャイトに類似した水和層の積層構造よりなる。層間の結合はイオン性であり層状粘土鉱物と比べるとはるかに強固である。OCPは板状の大型の結晶が比較的簡単に水溶液系から得られることから、生体用セメントのフィラーとして期待されている。. 生体用セメントリン酸カルシウム類の多くは水溶液中では安定なHApへ加水分解し、石こうやセメントのような硬化を伴いHAp多孔体となる。生体中での硬化体は生体骨に近い結晶性を有する。これを利用して生体用セメントが開発された。これらのセメントは生体中のカルシウムイオンやリン酸イオン種を組み入れ再石灰化するため、歯骨に類似の生体セラミックスとして歯科用あるいは医用のセメントや骨補てん材、再石灰化促進剤等へ応用される。また、自己硬化性を利用し骨形成タンパクの賦形材としても応用される。α-TCPは単独で転化反応を起こし硬化する。α-TCP粉末と硬化液として水、生理食塩水、ポリアクリル酸、および生体内に存在する有機酸の水溶液を用いたセメントが開発されている。TeCPとDCPA またはDCPD を用いる自己硬化型セメントは、これらの等モル混合物を溶解度が高い酸性領域で水または希薄リン酸液と練和して用いる。この反応式は次のように表される。2CaO(PO)+2CaHPO・2HO→Ca(PO)(OH)+4HO(2TeCP+2DCPD →HAp+4HO)セメントの性質に対しては各種有機酸水溶液も影響を及ぼす。歯科用裏装剤としても応用されるα-TCPでは、液剤としてポリアクリル酸水溶液を用いるもの、ポリアクリル酸とクエン酸の水溶液を用いるもの、クエン酸とタンニン酸の水溶液を用いるものなどがある。c. バイオガラス(bioglass)と結晶化ガラスSiO-PO-CaO-NaO系を基本組成とした一連のガラスを指し、これらは生体骨と結合するといわれている。ガラス中に含まれるNaイオンが溶出し、ガラス表面にSiOに富むゲル層が形成され、これを介してコラーゲンやHAp結晶が表面に形成されて骨と結合していくものと考えられている。SiO-PO-CaO-MgO-NaO-KO系結晶化ガラス(Cer-avital)とSiO-PO-CaO-MgO系の組成を用いたA-W 結晶化ガラスはガラス作製のプロセスの利点を生かしながらガラスの強度を向上させる目的で開発された。CeravitalはSiO46.2wt.%,PO11.7wt.%,CaO 34.0wt.%,M gO 2.9wt.%,NaO 4.8wt.%,KO0.4wt.%の組成をもち、ガラス中に酸素アパタイト[Ca(PO)O]を結晶化させている。またA-W 結晶化ガラスはSiO34.2wt.%,PO16.3wt.%,CaO 44.9wt.%,MgO 4.6wt.%の組成で表され、機械的性質に勝るウォラストナイト(CaSiO,W)マトリックス中に生体親和性を付与するためHAp(A)を同時に結晶化させた一種の複合材料である。A-W 結晶化ガラスの機械的性質に関しては曲げ強度が200MPa,は2.0MPamが報告されている。d. ジルコニア(zirconia)、アルミナ(alumina)安定化あるいは部分安定化ジルコニア(SZ およびPSZ)は耐火物や構造用セラミックス、あるいは固体電解質、センサ材料として知られているが、バイオセラミックスとしても重要な材料である。ジルコニアを生体内に埋入した場合、異物反応は見られず、また骨とのインタフェースにおいて骨と直接接する所見も得られており、アルミナと同様バイオイナートであることが明らかになっている。また耐摩耗性もアルミナをしのぐことから多結晶体として特に人工骨頭に用いられている。なお、水環境下では結晶転位による強度の低下の可能性がある。アルミナ(AlO)バイオセラミックスは、(1)強酸、強アルカリに対して強い耐腐食性を有するため、生体内に長時間埋入しても溶解、変質することがない。したがって、(2)バイオイナートである。主として単結晶アルミナは人工歯根や骨折ピンなどに用いられ、多結晶体として人工関節や人工骨に用いられている。また、多孔体状でも固定化酵素担体として用いられており、それぞれの形態によりさまざまな幅広い用途がある。生体用アルミナとは通常α-AlOのことを指す。アルミナの生体親和性は水蒸気が吸着して得られる極めて親水性に富んだ表面に起因する。アルミナ結晶表面の酸素が水分子をとらえ分極し、アルミナ表面は水酸基に覆われてこの水酸基に水が結合するものと考えられている。この水の膜が介在し生体組織とアルミナは接する。この水にぬれやすい性質から生体適合性がよいと推定されている。そのほかには高強度を有する窒化ケイ素(SiN)と光触媒で知られるチタニア(TiO)を用いたバイオセラミックスの応用研究も行われている。ケイ素はSiOイオンの形でバイオガラスのネットワーク構造を形成している事実、またチタンは金属そのものがバイオイナートマテリアルとしてインプラント材に用いられているという事実から、これらの二つのセラミックスは生体親和性を有するものと考えられている。
