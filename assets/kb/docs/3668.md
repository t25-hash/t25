# 10・4・1 ひずみゲージ法

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・4 残留応力の測定（residual stress measurement ）法

機械的な方法などによって残留応力を開放し，開放時に生じる変
形をひずみゲージで測定して，残留応力を推定する方法．対象品の
形状や残留応力の分布によって，孔あけ法と溝切込み法，表層逐次
除去法，切断法などが考案されている ．
a.孔あけ法（hole-drilling method）と溝切込み法（ring core
材質変化の境界，断面寸法の急変部は欠陥がなくても磁粉模様が
method) 図10･4に示すように，試料表面の測定位置に三軸ひ
ずみゲージを張り，その中心ないしは周りに孔あるいは溝を加工し
て残留応力を開放する．
図10･5に孔あけ法用のロゼットゲージの例 を示す．また，図
10･6に孔あけ装置の例 を示す．
