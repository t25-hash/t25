# 5・1・4 有機材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

a. ポリエチレン(polyethylene)ポリエチレン(PE、図5・1)は、メチレン基を繰返した分子鎖からなっており、ビニル系ポリマの中で最も簡単な構造を有する。PEの分子中には結晶相とアモルファス相が存在し、この結晶は球晶で光を反射するので、半透明なプラスチックとなる。PEは化学的には非常に安定であり、アルカリや弱酸には耐えるが、酸化性酸では劣化する。低分子のものを除くと60°C以下では有機溶媒に溶けないが、高温下の芳香族炭化水素やハロゲン化炭化水素には溶解する。PEの物性は、結晶化度、平均分子量、分子量分布などに影響を受ける。つまり、分岐度が小さく高密度になるほど、結晶化度、剛性、引張強さ、硬さ、化学的安定性や不透明性が増大し、液体やガスの透過性あるいは衝撃強さは低下する。分子量が大きくなると、耐衝撃性や化学的安定性は大きくなるが、溶融粘度が大きくなるので成形加工性は低下する。PEは安価で低毒性のプラスチックであるので、ディスポーザブル製品に適している。輸液バッグや診断カテーテル、バルーンカテーテル、経管栄養カテーテルなどの用途がある。疎水性のPE膜は一部親水性化させて、腹水ろ過器や血漿分離器に用いられている。分子量が数百万に達する超高分子量ポリエチレン(UHMWPE:ultra high molecular weight polyethylene)は耐摩耗性、自己潤滑性、耐衝撃性に優れていることから、人工股関節の臼蓋カップの素材、人工膝関節のインサート、人工肘関節の尺骨部品など人工関節のしゅう動面に使用されている。しかし最近、ソケットのPEの摩耗による骨吸収の報告もあり、摩耗粉の解決が一つの課題となっている。エチレンの共重合体(copolymer)であるエチレン-酢酸ビニル共重合体(EVA)は、その組成を変えると、プラスチックからゴム状の性質まで変化する。その特徴は、柔軟性があり、引張強度が高く、可塑剤を含まないことであり、ドラッグデリバリーシステム(DDS)の透過膜に使用されている。b. ポリエチレンメチル基が同一の方向に並んだアイソタクチックポリプロピレンである。分子内に三級炭素を有するので酸化されやすく、酸化防止剤が少量添加されている。PPは低比重で透明性に優れており、高融点(160~170°C)で耐熱性や成形加工性も良好で、高圧蒸気滅菌が可能である。PPの耐薬品性は結晶化度の増大とともに向上する。酸やアルカリに対しては、高温においても抵抗性が高く、エタノールなどによる応力き裂に対しても耐性が高い。80°C以上になると、ベンゼン、トルエン、四塩化炭素などに溶解する。PPは透明硬質で、生化学的に不活性であり、ディスポーザブル医療用具としてのニーズが多い。注射筒の外筒や押し子、ディスポーザブル針の針元、血液バッグ、スリーピース型眼内レンズの固定用ハプティックスの素材、単繊維の非吸収性縫合糸、人工靱帯などへの需要がある。微孔性PP膜の中空糸は膜型人工肺の多孔質膜として用いられている。疎水性のPP膜は短時間のガス交換能は優れているが、膜表面が血漿タンパク質の吸着などにより親水性化すると、血漿が外部に漏れる問題がある。血漿リークを防ぐためにシリコーンを薄くコーティングする試みも検討されている。PP膜は血漿分離膜にも使用されており、高い耐溶血性と溶出物の少ない安全性を特徴としている。c. ポリエチレンテレフタレート[poly(ethylene terephth-alate)]ポリエチレンテレフタレート(PET、図5・3)はポリエステルの代表的なポリマであり、テレフタル酸とエチレングリコールから重縮合反応により合成される。はん用製品ではPET ボトルとして広く知られているが、溶融紡糸して熱延伸すると木綿に近い性質のポリエステル繊維が得られる。医用材料としてよく用いられるPET の繊維にテリレン좲(ICI社)、ダクロン좲(デュポン社)やテトロン좲(東レ、帝人)がある。ポリエステル繊維は強度が大きく、耐久性や耐摩耗性はナイロンに次ぐ大きさであり、吸湿性や吸水性も小さく、水中浸漬による機械的強度の低下がない。ポリエステル繊維は生体に対する反応性が少なく、生体内での劣化も小さいことから編組人工血管に用いられる。この人工血管は多孔質であって、使用前に患者の血液に浸して血栓を形成させて孔をふさぐプレクロッチング(preclotting)操作が行われる。その後時間の経過とともに人工血管の内面に新生内膜あるいは偽内膜が形成されて抗血栓性が発現する。繊維の織り方には、多孔度や強度が異なる平織り、メリヤス編みやベロア型編みがある。ダクロン製人工血管は内径6~30mmの大動脈などの血行再建術で使用されている。ダクロン糸は非吸収性縫合糸に用いられ、シリコーンやテフロンによる表面加工を施したものもある。PET は織布が人工靱帯に用いられるほか、メッシュ状などの形で、補強材としても使用される。d. には、一般的な塊状重合であらかじめ成形されたものと、臨床の場において、術者自身が成形硬化させる粉液タイプレジン(powder-liquid resin)があり、後者は唯一メタクリル樹脂にのみ適用できる重合硬化方法である。PMMA の特徴として、(1)光線透過率が高く、耐光性を有する、(2)疎水性の高分子であり、水やアルコールには溶けないが、アセトン、塩化メチレン、クロロホルムやMMA initiator)からなっている。粉成分は懸濁重合で合成した微粒状のPMMA であり、液成分にはMMA 医用高分子材料の表面改質医用高分子材料は物理化学的および機械的性質以外に、生体組織との適合性が要求される。生体組織と高分子材料とが接する界面での相互作用が材料の組織適合性を左右する最大の因子となる。強度に優れた素材の表面を組織適合性化することが、医用材料設計の基本の一つであり、既存材料の表面を改質して機能を高めることも材料開発の有力な手段である。高分子材料の表面改質法は多岐にわたるが、医用高分子材料に適用されている手法は限定的であり、直接表面に施す、貼付、コート、配合、反応や一部の物理的処理、そして内部での混合、共重合などがある。. 化学反応による修飾活性物質をイオン結合や共有結合などの化学結合(chemical 基のジエチルアミノエチル基置換、生体適合性を向上させるビタミンEの結合、ヘパリン吸着排除のためのOH 基のベンジル基置換などもある。また、血液適合性改善のために、ポリスルホン膜表面にγ線照射でポリビニルピロリドンを含水ゲル化させる方法もある。. 物理的処理さまざまな物質のプラズマを表面に照射するプラズマ処理(plasma ポリマアロイブレンド複数の機能を発現させるためにポリマを混合する方法であり、表面の改質にも有効である。特異な性質をもつシリコーンが一成分に選ばれることが多く、抗血栓性のセグメント化ポリウレタンにも用いられている。PPベースにスチレン系エラストマをブレンドした血液バッグ(オレフィンバッグ)、コポリエステルとポリカーボネートのポリマアロイやポリエステルとポリエーテルスルホンのブレンド膜などもある。. 共重合体化複数のポリマ成分を共重合させて、各成分の特性を生かす方法であり、ブロックコポリマ、ランダムコポリマ、グラフトコポリマがある。ブロックコポリマでは、セグメント化ポリウレタン、PHEMA とポリスチレン、ポリアミドとシリコーン、ポリエチレンオキシドとポリスチレンなどで抗血栓性が改善される。各種ポリマ表面にポリエチレンオキシドをグラフト重合(graft polymerization)させると抗血栓性が向上し、親水性ビニルモノマをグラフト重合させると血液適合性や生体適合性が向上する。また、PPとPEのランダムコポリマでは、透明性、耐衝撃性、耐放射線性が改善する。
