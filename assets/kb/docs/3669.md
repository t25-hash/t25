# 5・1・5 生体由来材料(biological materials)

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

これまでにも多くの優れた人工の生体材料が開発されてきているが、生体組織は人工材料で実現できない特性をもっているために、生体由来材料の使用を余儀なくされている場合も多い。しかし例えばコラーゲンでは25種の型が報告されており、生体内では型に応じて異なった機能を発現しているはずであるが、コラーゲンを生体材料として使用する場合に、このような型別の使用法が開発されているわけではない。コラーゲン以外にもヒトやほ乳類はもちろん、節足動物、植物、微生物由来のさまざまな細胞外マトリックスが医療用に用いられている。生体内に用いられる生体由来材料の多くは分解性であるが、アルデヒドなどで固定し非吸収性として使用されるものもある。また分解といっても、細胞外における酵素的な分解以外に、マクロフ8―178生体工学ァージによる貪食を介した分解もあって、生体内での分解過程は単純なものではない。a. 生体由来組織生体由来の組織を、ほぼそのままの形で利用する場合がある。例えば、心臓弁の置換手術では、人工材料のみで作られている機械弁は耐久性に優れるものの血栓を生じやすい。そこでブタ大動脈弁などから作製される生体弁が適用されることがある。異種由来のため、抗原性や感染性の問題から、アセトアルデヒドなどで固定して用いられる。また、長期の使用で石灰化し再置換を余儀なくされるので、最近では石灰化抑制処理も行われるようになっている。機械弁と比較すると、長期耐久性で劣る可能性がある以外には、血栓を生じにくく血行動態的にも優れているという利点があることから、例えば患者の年齢が40~65歳であれば機械弁、60歳以上であれば生体弁というような使い分けがなされている。脳外科手術においては、硬膜欠損部を覆い脳脊髄液の漏出を止めるために乾燥ヒト硬膜が用いられることがあった。この場合は、縫合するだけで液漏れ止めできるという、人工膜では実現できない漏れ止め機能を有しているというのが、生体由来組織を使用する大きな理由であった。市販されたものはヒト由来の脳硬膜を紫外線などで処理しただけのものであったために、クロイツフェルト・ヤコブ病の感染を引き起こし、使用が禁止された。現在では、人工の膜材料に後述するフィブリンのりを組合わせて液漏れ止めを行うようになっている。現在臨床応用されているヒト由来組織には、角膜、皮膚、心臓弁、骨などがある。これらの組織の移植では臓器移植と異なり免疫抑制剤は通常は使用されない。移植時において、角膜や皮膚では移植組織の細胞は生存しているが、骨の場合には加熱処理や放射線処理を行って細胞を殺して用いられる。心臓弁の場合には内皮細胞は死滅しているものの、線維芽細胞はある程度は生存していると考えられている。b. コラーゲン(collagen)、ゼラチンコラーゲンは脊椎動物の結合組織を構成するタンパク質であり、2004年までに30種以上の型が報告されている。最も大量に存在するのはI型のコラーゲンであり、ヒトの皮膚、靱帯、腱、骨などの組織に存在し、その引張強度を確保する役目を担っている。これまでに生体材料として応用されているものは、このI型コラーゲンのみである。コラーゲンの原料としては、ウシ皮、ブタ皮、ブタ腱などが利用されているが、牛海綿状脳症(狂牛病)の発生により生体由来材料の安全性が問題となり、動物の原産国はもちろん、年齢、使用部位、誕生から屠畜までの飼育状態の管理が実施されるようになっている。I型コラーゲンは、分子量約10万のポリペプチド鎖が3本より合わさった三重らせん構造の部分に両端のテロペプタイド部分が付いた棒状の分子である。生体内では、このコラーゲン分子が規則的に配列して線維を形成しており不溶性である。テロペプタイド部分は抗原性が強いので、この部分を酵素処理で除くとともに可溶性にしたアテロコラーゲンとして利用されることが多い。アテロコラーゲンのゲルや成形体は強度が弱く生体内で速やかに分解されてしまうので、強度などを要求される場合には架橋を入れて使用される。架橋方法にはアルデヒドなどを利用した化学架橋、紫外線を利用した紫外線架橋、脱水して高温処理する熱架橋がある。酸性のコラーゲン溶液を中和して37°C程度に保つとゲル化する。このことを利用して体表近くの陥没部に中性コラーゲン溶液を直接注入しゲル化させることにより、補正修復を行うことがある。ただしその効果はコラーゲンゲルが吸収されるまでの、一時的なものである。コラーゲンは細胞接着性に優れているので、コラーゲンゲル上で細胞を培養することに用いられる。またコラーゲン溶液中に細胞を分散させてゲル化させると、コラーゲンゲル内で三次元的に細胞を培養することができる。このような培養基質としての利用により、コラーゲンの培養人工皮膚などへの応用がなされている。材料の表面をコラーゲンコーティングし、細胞接着性を高めることも行われている。またコラーゲンを湿式紡糸して綿状にしたものが止血剤として利用されている。これはコラーゲンに血小板が付着凝集して凝血することを利用したものである。膜状にコラーゲンを成形したものは、骨組織誘導膜として利用されている。コラーゲンの滅菌にはγ線や電子線滅菌が適用できるが、多少の変性は不可避である。ゼラチンはコラーゲンの熱変性物質であり、コラーゲンを加熱処理してコラーゲン分子の三本鎖をほどき、一本鎖の分子としたものである。ゼラチンの原料はウシ骨、ウシ皮、ブタ皮などであり、これらを酸処理もしくはアルカリ処理し、さらに加熱し水溶化させて抽出する。これらは、それぞれ酸処理ゼラチンおよびアルカリ処理ゼラチンと呼ばれ、アルカリ処理ゼラチンでは等イオン点がおよそpH 5であるのに対し、酸処理ゼラチンでは等イオン点がpH 8~9である。等イオン点において分子の荷電状態が変化するため、水溶液の粘度などはpH に依存することになり注意が必要である。ゼラチンを水とともに加熱すると、ランダムなコイル状の分子構造をとって可溶となるが、水溶液を冷却するとゼラチン分子の一部がコラーゲン様の三重らせん構造を作り、ゼラチン分子間のネットワークが形成されてゲル化する。ゼラチンは強力な被膜形成能をもつことから、カプセルやコーティング材として利用されている。またゼラチンゲルの表面には細胞がよく接着できるが、ゲル中には細胞は侵入できない。そのためゼラチンゲルの癒着防止材への応用が試みられている。さらにゼラチンゲルの生体分解性を利用して薬剤徐放材料としての応用も検討されている。ゼラチンの特異な使用形態としてゼラチンのりがある。これはゼラチンとレゾルシンの混合物をホルムアルデヒドとグルタルアルデヒドを用いて生体内で架橋させるものであり、解離性動脈瘤などの手術時に血管の接着に用いられる場合がある。ゼラチンはコラーゲンと同様な滅菌のほか、熱による滅菌も可能である。c. フィブリン(fibrin)血液凝固は複雑なカスケード反応の結果として生じるが、その最終段階では、トロンビンの酵素活性によりフィブリノーゲンからフィブリンモノマが作られ、このモノマは自動的に重合してフィブリンポリマとなる。そして血液凝固第XIIIa因子によりさらに架橋が導入されて、ゲル状の安定化したフィブリンクロットが形成される。このようにしてできたフィブリンクロットは、血のりや血餅の形状を保つ主たる成分であり、血管からの出血の止血や欠損した組織の救急的な補てんとして機能する。周囲組織から遊走してくる細胞は血餅中に侵入して増殖し、コラーゲンなどの細胞外マトリックスを産生する。その後時間とともにフィブリンクロットはプラスミンにより分解されていく。過去において、ヒト血液由来の非加熱フィブリノーゲン製剤が、フィブリノーゲンが欠乏している患者らにおける産科での異常出血などに対応するための止血剤として発売されていた。これは紫外線処理されたものであったが、C型肝炎ウイルスに汚染されたものが出回ったことから肝炎の感染源となって大きな問題となった。フィブリノーゲン欠乏患者は全国で数十人程度と少ないはずであり、止血剤としてのフィブリノーゲン製剤単独使用の妥当性には不明な点がある。しかしフィブリンは組織接着剤として臨床応用可能なほとんど唯一の材料であるので、現在ではヒト血液由来の加熱処理されたものが使用されている。これはヒト血液由来加熱フィブリノーゲン製剤を、血液凝固第XIII因子、抗プラスミン剤、塩化カルシウム、トロンビン溶液などとともに、フィブリンのりあるいはフィブリン接着剤として用いるものである。脳硬膜縫合部から脳脊髄液が漏出する場合、あるいは肺から空気が漏れ出す場合など、各種組織の接着の目的で使用されている。自家血由来のフィブリノーゲンも用いられることがある。フィブリンゲル中で細胞の増殖は活発であるので、コラーゲンと同様に細胞の担体として利用できる可能性があり研究されている。d. ヒアルロン酸、ヘパリン(heparin)、コンドロイチン硫酸ヒトの細胞外マトリックスを構成しているのは、タンパク質や糖鎖がタンパク質と結合している糖タンパク質だけではなく、糖鎖を含む高分子のプロテオグリカンがある。これはコアタンパク質と呼ばれるタンパク質にグリコサミノグリカンという二糖の繰返し構造をもつ一本鎖の高分子が結合したものである。プロテオグリカンの特性のかなりの部分は特有のグリコサミノグリカンに由来している。グリコサミノグリカンには、全身に広く分布するとともに創傷治癒過程でも重要な働きをするヒアルロン酸、軟骨などに存在するコンドロイチン硫酸、皮膚などに存在するデルマタン硫酸、基底膜などに存在するヘパラン硫酸、肺や肝臓などに存在するヘパリン、角膜などに存在するケラタン硫酸がある。これらのうち、医療に応用されているものに、ヒアルロン酸、ヘパリン、コンドロイチン硫酸がある。これらの材料は異種生体由来材料ではあるが、多糖としてはヒト由来のものと本質的には同じであると考えられており、抗原性は低い。ヒアルロン酸は、グルクロン酸と-アセチルグルコサミンの繰返し構造をもち、ニワトリのトサカや菌類などから作られている。変形性膝関節症などの治療において、関節の潤滑剤として関節内に直接注入して用いられる。滑膜の炎症を治めるとともに、軟骨の変性を抑制する作用があるといわれている。またヒアルロン酸の粘弾性特性を利用して、眼科の手術時に組織を物理的侵襲から保護するためにも利用されている。またコラーゲンと同様に、体表近くの陥没部に直接注入し補正修復を行うことに用いられている。ヘパリンはグルクロン酸もしくはイズロン酸と-アセチルグルコサミンの繰返し構造をもち、ブタ腸粘膜などから作られる。ヘパリンは血液中のアンチトロンビンと結合し、アンチトロンビンのもつ血液凝固因子阻害活性を著しく増強させることにより、血液凝固を抑制する機能をもつ。ヘパリンはさらにフィブロネクチンやラミニンあるいは種々のサイトカインとの結合能も有する。薬剤として用いられるほか、種々の材料に抗血栓性をもたせるためのコーティングに用いられる。コンドロイチン硫酸はグルクロン酸と-アセチルガラクトサミンの繰返し構造をもつ。ウシ、ブタ、サメの軟骨などから作られる。粘調剤として利用されているほか、軟骨組織の誘導などを目的とした利用方法が研究されている。e. キチン(chitin)、キトサンキチンはカニやエビなどの甲殻類やカブトムシのような昆虫類の外骨格や腱を構成する成分である。またキノコ類や細菌の細胞膜等にも含まれることがある。医療用のキチンはおもにカニ類を原料としている。カニの殻はキチン質、タンパク質、炭酸カルシウム、およびその他の微量成分からなっており、このような原料に脱灰および脱タンパク操作を加えて不溶性のキチンが得られる。キチンはセルロースと類似した化学構造をもち、グルコースが脱水縮合した直鎖状の多糖であるセルロースにおいて、グルコース残基の2位の水酸基がアミノアセチル基で置換されたものがキチン、すなわちポリ-アセチルグルコサミンである。キチンをアルカリ処理して、アミノアセチル基を脱アセチル化しアミノ基とすると、キトサンすなわちポリグルコサミンになる。ただしキチンのアミノアセチル基がすべて脱アセチル化されるわけではない。キチン、キトサンはタンパク質を含まないので、異種生体由来材料であるにもかかわらず抗原性が低い。ヒトはキチン合成を行う能力がないが、糖鎖は細菌の細胞壁を構成するものと共通なので溶菌酵素としてリゾチームをもち、これは多糖において糖を結びつけている化学結合を切断することができる。そのため生体内でキチンは加水分解され、アセチルグルコサミンに分解された後、さらに炭酸ガスなどにまで分解され、排出されるか生体内で再利用される。キトサンは酵素的には分解されないようであるが、生体内では劣化して最終的には分解していく。キチンを紡糸したものが吸収性縫合糸として研究されており、また微細繊維化し不織布として成形したものが創傷被覆保護材として用いられている。止血効果をもつほか、鎮痛、創傷治癒促進などの8―179効果があるといわれている。さらに人工靱帯や人工血管などへの用途も研究されている。キチンはアミド溶媒などにしか溶解しないが、キトサンは希有機酸に容易に溶け、アミノ基や水酸基にさまざまな化学修飾を加えることもできるようになる。またキトサンはアミノ基を有するため酸性溶液中では正に帯電するポリカチオンである。そこでカルボキシル基などをもち負に帯電する物質とともに用いれば、イオンコンプレックスを形成させることができる。このような特性を利用してマイクロカプセルなどを作製し、細胞の担体などに応用しようとする研究が行われている。f. アガロース、アルギン酸(alginate)アガロースは紅藻類から作られる寒天の主成分をなす多糖であり、ガラクトースとアンヒドロガラクトースの基本単位が交互に繰返される構造をもっている。加熱すると水溶性のゾルとなり、冷却するとゲル化する。細胞接着しないことからゲル中には周囲から細胞が侵入しないので、移植細胞を免疫隔離するためのマイクロカプセルなどの応用が研究されている。アルギン酸は褐藻類から採取される多糖であり、マンヌロン酸とグルロン酸が種々の割合で結合した高分子酸である。カルボキシル基をもちNaやK などの一価の金属との塩は水に可溶であり粘調なゾルになるが、Caなど二価の金属の場合には金属イオンを介してアルギン酸中のグルロン酸ブロックどうしが結合してエッグボックス構造を作り、ゲルを形成する。アルギン酸のカリウム塩やナトリウム塩は歯科用印象材として多用されている。アルギン酸のカルシウム塩を繊維状にして不織布にしたものは創傷保護被覆材として用いられている。アルギン酸は自重の15~20倍の水分を吸収するが、浸出液などのナトリウムイオンを含む水分を吸収するとゲル化し、このゲルが創傷面の湿潤環境を保つ。ゲル化する際にカルシウムイオンを放出し止血効果を発揮するので、アルギン酸塩被覆材は止血能をもっている。またアガロースと同様に免疫隔離用マイクロカプセルなどへの応用が研究されている。
