# 10・5・1 引張試験（tension test ）

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・5 材料試験

金属材料を静的に引張破断して，強度，延性，加工性を調べる．
降伏点または耐力は，材料が永久変形を生じる限界で，設計上最も
重要である．降伏比（降伏点と引張強さの比）は変形能，プレス成
形性の尺度として，伸びは曲げ加工性の目安に用いられる．塑性加
工性の尺度に加工硬化係数 値， （ランクフォード）値がある．
JISでは試験方法（JIS Z2241：1998）のほか，試験片の形状，採
取方法（JIS Z2201：1998）を定めている．
