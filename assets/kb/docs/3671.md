# 5・1・7 薬剤徐放材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

薬剤を、必要なときに、必要時間、必要な場所へ、必要量を供給するという、ドラッグデリバリーシステム(DDS : drug deliverysystem、薬物送達システム)技術があり、薬剤の徐放もその一環をなすものである。ただし、徐放とはいっても、薬剤が単にゆっくり放出されればよいというわけではない。薬剤を作用させる必要のある部位において、適切な薬剤濃度とその時間的変化が得られるように制御する、放出制御型DDS(コントロールドリリース型DDS)である。薬剤の生体への移行制御には、(1)生体に対する薬剤の制御された供給だけでなく、(2)薬剤が、生体表面にある吸収障壁を通過して、必要な部位に取込まれるようにする吸収改善技術、(3)薬剤を作用部位に選択的に到達させるように標的部位指向性を付与するターゲティング(targeting)技術が必要であり、これらがDDS 技術の全体像である。放出制御型DDS は、大きく分類すると表5・7のようになる。それはさらに、全身レベルおよび投与局所での作用発現を目的としたものに大別できる。前者では、経口投与、経皮投与、皮下に注入や埋込みを行うシステムなどがあるが、いずれも血中濃度の維持が目標となる。後者は、眼内治療システム、子宮内投与避妊システムなどに代表される局所放出制御型DDS であり、長期間にわたって局所で徐々に放出される薬剤はその近傍において吸収されて治療効果を発揮し、全身循環にはほとんど関与しない。放出制御するための原理・方法とそれに対応して用いられるおもな材料を表5・8に示す。a. 経口放出制御型錠剤型、カプセル剤型、複合顆粒型がある。錠剤型は歴史的に最も古く、半透膜を通って侵入する消化管の水分によって内圧を高め、放出口より薬剤を長時間にわたって一定速度で放出するもので、食欲抑制薬などの投与に応用されている。カプセル中に薬剤を含むサブユニット(顆粒や小型の錠剤)をつめたものもある。この場合には、サブユニットが放出制御機能を有し8―182生体工学ており、その放出速度の異なるものを適当な比率で組合わせることにより、望ましい放出パターンが得られる。イオン性の薬剤では、それをイオン交換樹脂に結合させ、エチルセルロースで被覆したマイクロカプセル製剤もある。これを懸濁したシロップ剤では、徐々に薬剤がイオン交換樹脂から解離、放出される。こうした従来の技術に加えて、消化管内での製剤の移動を制御することで薬剤の吸収パターンを制御することが進められている。比重を1より小さくして胃内溶液中に長時間浮遊するようにする、形状記憶ポリマを利用して胃内に滞留しやすくする、粘膜付着性のポリマを利用して消化管粘膜に付着させる、などである。b. 外用型このタイプのうちで、全身レベルでの作用発現を目的としたものの代表に経皮治療システムがある。従来、皮膚への薬剤投与は局所作用を目的として行われてきたが、皮膚透過性のよい薬剤については全身的な投与経路としても考えられるようになり、経皮治療システム(TTS : transdermal therapeutic system)として外用放出制御型製剤が実用化されている。代表的なものは狭心症発作予防のニトログリセリン放出制御製剤である。皮膚に粘着する製剤を貼付すると、透過速度を制御しているエチレン-酢酸ビニル共重合体の多孔性膜を通して、24時間にわたってニトログリセリンが一定速度で放出され、血中濃度が一定範囲に保たれて狭心症発作が予防される。TTS には、粘着剤と薬剤の間に放出制御膜を利用するタイプとポリエステル系フィルムを支持膜にして粘着剤と薬剤をいっしょにしたものがある。乗物酔い予防のスコポラミン製剤、禁煙目的のニコチン製剤などもある。外用型の放出制御膜としては、眼内治療システム、子宮内投与避妊システムなどにもエチレン-酢酸ビニル共重合体(ethylene-vinylacetate copolymer)が広く利用されている。この共重合体は、エチレンと酢酸ビニルの共重合割合を変えることにより、比較的硬いものから軟らかいものまで、物性の違うものが種々入手でき、目的によって使い分けができる。熱可塑性で、可塑剤を加えなくとも適度の軟らかさがあり、また安全性のうえでも問題が少ない。分解により酢酸ができる可能性もあるが、微量の酢酸は問題とはならない。アフタ性口内炎治療に使われる口腔粘膜適用製剤、鼻過敏症の治療に使われる鼻腔内徐放システムでは、ヒドロキシプロピルセルロースのような粘膜付着性のある基材が利用されている。粘膜に長時間付着して、薬剤を放出しながら少しずつ溶解していく。c. 埋込み・注入型埋込み型では、シリコーンゴムに性ホルモンなどを包埋し、年単位での作用持続を目的とした避妊用インプラントが代表的である。シリコーンゴムは安全性の高い、軟質基材である。ハイドロゲルでは、水を吸収して膨潤するとともに、拡散の原理に従って薬剤を放出する。乳酸-グリコール酸共重合体(lacticacid-glycolicacidcopolymer)に黄体形成ホルモン放出ホルモン誘導体を組込んだものが前立腺癌治療に使われている。直径1.2mm、長さ10mmほどの棒状物あるいは直径20 mのマイクロカプセル懸濁液があり、前者では注射針から押し出し、後者では注射する。これにより、有効血中濃度が約4週間維持される。このように、乳酸-グリコール酸共重合体やポリ乳酸(5・1・6項参照)は、埋込み・注入型システムにおける、おもな生分解性基材として、針状、シート状、注射用マイクロカプセル懸濁液の形で利用されている。乳酸-グリコール酸共重合体は分解が速く、1箇月ぐらいまでの薬剤の放出に適している。ポリ乳酸は分解が遅く、長期の薬剤放出に向いているが、場合によっては大部分の薬剤が放出された後も基材だけが長期間残ってしまう可能性もある。これらの生分解性ポリマを用いると、それに組合わせる薬剤の性質によっては、ポリマの分解挙動とともに薬剤の放出パターンがかなり変化することがある。生分解性基材を用いた場合には、マトリックス内の単なる拡散による放出と高分子の分解・溶解に伴う放出とがある。乳酸-グリコール酸共重合体と並んでよく利用されるのがコラーゲン(collagen)である。インターフェロンを組込んだアテロコラーゲンのミニペレットが開発されている。コラーゲンの誘導体であるゼラチンに抗菌剤のクロルヘキシジンを組込んだものは歯周病治療に使われている。脂質は生体を構成する主要成分の一つであり、安全な薬剤の担体用素材として利用されている。リン脂質を基本とする脂質二重層からなるリポソームのアンホテリシンB 製剤や水中油型エマルジョンであるリピッドマイクロスフィアのプロスタグランジンE製剤がある。
