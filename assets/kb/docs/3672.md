# 10・5・3 衝撃試験（impact test ）

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・5 材料試験

衝撃荷重に対する強さを求める試験で多くの試験法が定められて
いる．なかでも2mmVノッチシャルピー試験（JIS Z2242：
1998）が最も一般的である．鉄鋼材料など（面心立方格子構造の金
属材料を除く）は，切欠のある状態で衝撃荷重を加えると低温脆性
を示す．衝撃値（吸収エネルギー/断面積）と試験片の破面様相が
急激に変化する温度を遷移温度と呼び，低温脆性を示す材料特性値
として重視される．
