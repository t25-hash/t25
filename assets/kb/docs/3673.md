# 5・1・9 膜・中空糸材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

a.血液浄化膜の分類と特徴 血液浄化法（blood purifi-
cation）は，生体が対象といえども化学工学における単位操作を利
用したもので，その基本原理は拡散（透析），ろ過および吸着であ
る．現在，最も一般的に行われている血液浄化法は，半透膜を介す
る拡散，ろ過を利用して血液中の代謝産物と過剰の水分を除去する
血液透析（hemodialysis）である．2005年現在，日本では約25万
人の慢性腎不全患者がこれによって生命を維持し，10年以上にわ
たる延命も容易になっている．種々の新しい膜素材の開発に伴う血
液透析器の進歩は目覚ましく，クリアランスの向上による透析時間
の短縮や，幅広い分子量を有する有害物質に対応する高性能透析器
が登場している．また血液透析以外の血液浄化法として，膜を用い
る血液ろ過（hemofiltration），接続的血液透析ろ過（hemo-
diafiltration），血漿分離（plasmapheresis）および吸着剤を用い
る直接血液灌流（direct hemoperfusion）などが開発されてきた．
これらの血液浄化法は腎不全患者の生命維持にとどまらず，急性薬
物中毒，肝不全のほか，自己免疫疾患，精神分裂症などの治療に対
しても応用されるようになり，新しい治療法として注目されてい
る．
血液透析用の膜は，溶質透過性，水透過性および力学的強度に優
れていることが必要である．この膜素材としてはセルロース（cel-
lulose）系が中心であったが，血液透析で生命を長期間維持してい
る患者が増えるにしたがって，次第にセルロース膜を透過できない
微量の代謝産物が患者の体内に蓄積され，患者にいろいろな病態が
現れることが問題視されるようになってきた．これの解決策とし
て，合成ポリマ製の多孔質膜が利用されるようになった．代表的な
膜材料を図5･9に示す．
b.セルロース系膜 血液透析に用いられているセルロース系
の膜素材は2種類に大別される．キュプロファンは，銅アンモニア
法でセルロースを溶解して成膜した後，銅アンモニウムを除去して
セルロースを再生する．一方，アセテート膜といわれるものは，酢
酸セルロース（cellulose acetate）を成膜した後もアルカリ処理は
せず，酢酸エステル構造をとる．
銅アンモニア法によるセルロースの再生は，解重合が少ないため
分子量を大きく保持することができ，力学的強度に優れた膜を作製
することができる．一方でこの銅アンモニア法再生セルロース膜に
ついては，臨床面で種々の欠点が指摘されており，透析膜素材とし
ては，(１)中分子量溶質の透過性を2～3倍にする，(２)不均衡症
候群の症状が起こらないように，低分子量溶質の透過性は2倍を超
えてはならない，(３)吸水状態での破裂強度を高めると同時に，吸
水による膜の変形を小さくする，(４)熱あるいはγ線による滅菌
を可能にし，透析器が乾燥状態で供給されるようにするといった改
良が求められている．
酢酸セルロース膜では酢酸セルロースに可塑剤（スルホラン）を
混ぜて溶融紡糸後，アルカリで鹼化すると同時に可塑剤を溶出させ
て，膜に溶質透過性を付与する．また酢酸セルロースと相溶性の小
さい可塑剤（ポリエーテル）を加えて中空糸に溶融紡糸すると，膜
中の非晶質部分に可塑剤が局在化し，アルカリ加水分解では主とし
て非晶質部分が加水分解されるため可塑剤は除去される．その結
果，溶質透過性がよく，しかも力学的強度の大きな中空糸が得られ
る．中空糸型透析器の製造プロセスでは，吸水，乾燥工程を繰返す
が，この中空糸は乾湿における寸法変化が少ないため作業性がよ
く，膜面が滑らかで，しかも膜の滅菌にエチレンオキシドやホルマ
リンを使用せず，熱による滅菌が可能であるという特徴を有する．
力学的強度，コストなどの点で加水分解工程のある酢酸セルロース
法が有利なため，酢酸セルロースの多孔質中空糸が増加する傾向に
ある．β2-ミクログロブリンを除去しやすい多孔質膜が求められる
傾向が強まったことにより，酢酸セルロース製血液透析器が使われ
る頻度が増している．
c .合成ポリマ膜 透析膜が備えなければならない条件の中で，
疎水性の大きな合成ポリマは，溶質透過性の点で，比較的不利な素
材である．現在実用化されている合成ポリマ膜では，いずれも何ら
かの方法で親水化処理が必要である．単に多孔性とするだけでは，
透過膜に必要な溶質透過性と水透過性のバランスが十分でないた
め，セルロースの非晶質部分と同様の機能を加味する試みがなされ
ている．膜厚はセルロース系膜の7mに比べて20～50mと大き
い．
ポリメチルメタクリレート（PMMA）を単純に成膜すると，溶
質透過性と力学的強度を同時に満足させることは困難である．しか
し，イソタクチック（iso-）PMMAとシンジオタクック（syn-）
PMMAの溶液を混合して湿式で凝固させると，PMMA鎖が錯体
を作り，溶質透過性と湿潤時の力学的強度の大きい血液透析用の膜
が得られる．これは，錯体を形成したPMMA鎖がセルロース膜に
おける結晶部分のような機能をしているためと考えられる．補体系
