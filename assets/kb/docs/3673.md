# 5・1・9 膜・中空糸材料

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・1 生体材料

a. 血液浄化膜の分類と特徴血液浄化法(blood purifi-cation)は、生体が対象といえども化学工学における単位操作を利用したもので、その基本原理は拡散(透析)、ろ過および吸着である。現在、最も一般的に行われている血液浄化法は、半透膜を介する拡散、ろ過を利用して血液中の代謝産物と過剰の水分を除去する血液透析(hemodialysis)である。2005年現在、日本では約25万人の慢性腎不全患者がこれによって生命を維持し、10年以上にわたる延命も容易になっている。種々の新しい膜素材の開発に伴う血液透析器の進歩は目覚ましく、クリアランスの向上による透析時間の短縮や、幅広い分子量を有する有害物質に対応する高性能透析器が登場している。また血液透析以外の血液浄化法として、膜を用いる血液ろ過(hemofiltration)、接続的血液透析ろ過(hemo-diafiltration)、血漿分離(plasmapheresis)および吸着剤を用いる直接血液灌流(direct hemoperfusion)などが開発されてきた。これらの血液浄化法は腎不全患者の生命維持にとどまらず、急性薬物中毒、肝不全のほか、自己免疫疾患、精神分裂症などの治療に対しても応用されるようになり、新しい治療法として注目されている。血液透析用の膜は、溶質透過性、水透過性および力学的強度に優れていることが必要である。この膜素材としてはセルロース(cel-lulose)系が中心であったが、血液透析で生命を長期間維持している患者が増えるにしたがって、次第にセルロース膜を透過できない微量の代謝産物が患者の体内に蓄積され、患者にいろいろな病態が現れることが問題視されるようになってきた。これの解決策として、合成ポリマ製の多孔質膜が利用されるようになった。代表的な膜材料を図5・9に示す。b. セルロース系膜血液透析に用いられているセルロース系の膜素材は2種類に大別される。キュプロファンは、銅アンモニア法でセルロースを溶解して成膜した後、銅アンモニウムを除去してセルロースを再生する。一方、アセテート膜といわれるものは、酢酸セルロース(cellulose acetate)を成膜した後もアルカリ処理はせず、酢酸エステル構造をとる。銅アンモニア法によるセルロースの再生は、解重合が少ないため分子量を大きく保持することができ、力学的強度に優れた膜を作製することができる。一方でこの銅アンモニア法再生セルロース膜については、臨床面で種々の欠点が指摘されており、透析膜素材としては、(1)中分子量溶質の透過性を2~3倍にする、(2)不均衡症候群の症状が起こらないように、低分子量溶質の透過性は2倍を超(a) セルロース系膜(b) 合成ポリマ膜図5・9代表的な血液浄化膜材料えてはならない、(3)吸水状態での破裂強度を高めると同時に、吸水による膜の変形を小さくする、(4)熱あるいはγ線による滅菌を可能にし、透析器が乾燥状態で供給されるようにするといった改良が求められている。酢酸セルロース膜では酢酸セルロースに可塑剤(スルホラン)を混ぜて溶融紡糸後、アルカリで鹼化すると同時に可塑剤を溶出させて、膜に溶質透過性を付与する。また酢酸セルロースと相溶性の小さい可塑剤(ポリエーテル)を加えて中空糸に溶融紡糸すると、膜中の非晶質部分に可塑剤が局在化し、アルカリ加水分解では主として非晶質部分が加水分解されるため可塑剤は除去される。その結果、溶質透過性がよく、しかも力学的強度の大きな中空糸が得られる。中空糸型透析器の製造プロセスでは、吸水、乾燥工程を繰返すが、この中空糸は乾湿における寸法変化が少ないため作業性がよく、膜面が滑らかで、しかも膜の滅菌にエチレンオキシドやホルマリンを使用せず、熱による滅菌が可能であるという特徴を有する。力学的強度、コストなどの点で加水分解工程のある酢酸セルロース法が有利なため、酢酸セルロースの多孔質中空糸が増加する傾向にある。β2-ミクログロブリンを除去しやすい多孔質膜が求められる傾向が強まったことにより、酢酸セルロース製血液透析器が使われる頻度が増している。c. 合成ポリマ膜透析膜が備えなければならない条件の中で、疎水性の大きな合成ポリマは、溶質透過性の点で、比較的不利な素材である。現在実用化されている合成ポリマ膜では、いずれも何らかの方法で親水化処理が必要である。単に多孔性とするだけでは、透過膜に必要な溶質透過性と水透過性のバランスが十分でないため、セルロースの非晶質部分と同様の機能を加味する試みがなされている。膜厚はセルロース系膜の7 mに比べて20~50 mと大きい。ポリメチルメタクリレート(PMMA)を単純に成膜すると、溶質透過性と力学的強度を同時に満足させることは困難である。しかし、イソタクチック(iso-)PMMA とシンジオタクック(syn-)PMMA の溶液を混合して湿式で凝固させると、PMMA 鎖が錯体を作り、溶質透過性と湿潤時の力学的強度の大きい血液透析用の膜が得られる。これは、錯体を形成したPMMA 鎖がセルロース膜における結晶部分のような機能をしているためと考えられる。補体系の活性化を惹起しない点とβ2-ミクログロブリンを吸着できるため好まれている材料である。耐熱性の高いエンジニアリングプラスチックであるポリスルホン(polysulfone)を原料とした膜は水処理用に使われていたが、長期血液透析患者の血液透析膜としても近年利用されるようになってきた。特にセルロース膜が補体の活性化を惹起すること、β2-ミクログロブリンのようなポリマ量代謝老廃物を除去しにくいセルロース膜の欠点を克服するために、ポリスルホン製多孔質中空糸膜を使った血液透析器が、次第にセルロース膜透析器にとって代わるようになってきた。ポリスルホンにポリビニルピロリドン(PVPy)を混ぜて湿式紡糸する。PVPyは凝固時に溶出しポリスルホン膜を多孔化するとともに親水化剤として一部残留する。ポリスルホンは生体適合性(biocompatibility)が比較的よいこと、耐熱性がよいために加熱蒸気滅菌が可能である点が利点とされている。一方で、残留したPVPyの血液中への溶出も今後の課題として残されている。d. 血液ろ過膜および血漿分離膜限外ろ過性能の高い膜素材の開発により、ろ過法による血液浄化が発展してきた。透析法の原理とは異なり、用いる膜の分画分子量以下の溶質はほぼ均等に除去されるので、中分子量毒性物質の除去が可能となり、これが原因と考えられる症状の改善に威力を発揮する。利用されている膜は、いずれも多孔質の合成ポリマ膜で、アクリロニトリル-塩化ビニル共重合体、ポリスルホン、ポリアミド、酢酸セルロース、PMMA、ポリオレフィンなどである。分画分子量も膜により異なり、アルブミンを通さないものや、低分子量タンパク質は通すものがある。透析や限外ろ過はおもに分子量1万以下の溶質を対象とし、これを除去する方法であるが、最近膜を用いた血漿分離法が注目されている。これは患者の血液から血漿成分を分離除去し正常血漿と交換することにより、血液中の病因タンパク質、抗原、抗体、免疫複合体などを取り除く療法で、特に免疫異常による疾患、例えば全身性エリテマトーデス、慢性関節リウマチ、糸球体腎炎、重症筋無力症などの種々の難病に効果が認められている。血漿分離膜の性能としては孔径が大きく、孔径分布が狭く、さらに開孔率が高いことが要求されるが、合成ポリマはこのような膜を作成するのに好都合である。湿式紡糸では、膜素材、孔あけ剤、溶剤、凝固浴の組合せなどの成膜条件により多孔度に変化を与えることができる。また溶融紡糸では、中空糸の延伸工程で結晶性を制御して、膜面に適度の力を加えて孔を作る方法も用いられる。e. 生体適合性血液透析器の性能が向上し、慢性腎不全患者の長期延命が可能になると、透析膜素材の生体適合性あるいは血液適合性が問題となる。生体にとって異物である材料に血液が接触すると、材料表面において血液成分が粘着、凝集、変性、活性化などを起こし、それが血液凝固、溶血、アレルギー反応、免疫低下などさまざまな生体反応を誘起する。膜素材の凝固系に与える影響については、透析が抗凝固剤の使用下に行われるため、ほとんど検討されていない。膜が血液に接触すると、必ず血液成分が膜に付着する。透析終了後の膜表面を観察すると、白血球・血小板を中心とした血球の粘着が認められる。赤血球はわずかしか粘着しない。血漿中には多種類のタンパク質が存在し、しかもそれぞれ複雑な機能をもっている。それらが競争吸着や脱着を繰返すために吸着挙動の正確な評価が極めて難しい。透析後の膜には再生セルロース膜で70nm以上の厚いタンパク質層が堆積している。血液透析など体外循環では通常、抗血液凝固剤としてヘパリンが用いられているが、ヘパリンの大量または長期間使用によるさまざまな副作用が指摘されており、できるだけ小量のヘパリン量で治療が行える抗血栓性の優れた材料による膜の開発が望まれている。血液凝固以外に透析中に見られる現象としては、膜面への白血球の付着および補体系の活性化などがある。これらの現象は一過性であり、透析終了時には元の状態に戻っているが、その繰返しが長期的に見て生体にどのような影響を及ぼすか、合併症の原因になっていないかなどが検討されている。それぞれの現象は膜材料の種類により差があるが、すべてに満足する膜は現在のところ存在しない。8―185材料と血液の界面現象の研究とともに、生体との適合性のよいポリマ材料による膜の開発が今後の重要な課題である。細胞膜構造を模して合成された2-メタクリロイルオキシエチルホスホリルコリン(MPC)ポリマの血液透析膜、中空糸への応用に関して精力的に研究が行われている。ヘパリンを使わずに血液処理ができること、膜面に吸着されるタンパク質の量を減らせて、膜性能を長く維持できる可能性が指摘されている。f. 人工肺(artificial lung)用膜血液の酸素化は肺胞膜を介して血液と空気の間のガス交換で常時行われている。組織、特に脳に障害を与えないために血液は全身に酸素を送り続ける。心臓を手術するときには血流を遮断して行われるために肺で血液の酸素加はできない。そのために体外に血液を導いて酸素加をする必要がある。このときには空気と血液を直接触れさせる人工肺が使われたが、長時間血液を直接空気に触れさせることは血液にダメージを与えるため、気泡型人工肺は使われなくなった。今日では大部分膜を介してガス交換を行う人工肺になっている。間欠的血液浄化に比べ、人工肺に使う膜は使用中血液の酸素加を継続的に行う必要があるだけに要求は厳しい。そのために膜型人工肺は開胸手術中に使うのが大原則であるが、長時間血液の酸素加に使える膜型人工肺に対する要望は強い。人工肺用膜には均質膜、多孔質膜、複合膜が使われている。均質膜としては酸素、炭酸ガス透過係数の大きいポリジメチルシロキサン(PDMS : polydimethylsiloxane)の平膜、中空糸膜が使われているが、強度が低いことが難点といわれている。しかし、多孔質膜に比べ血液漏れがないため、長時間の使用が可能であり、ECMO(extracoporeal membrane oxygenation)として急性肺不全患者の呼吸補助装置に一部で用いられている。多孔質膜はガス透過係数や膜の強度は大きいため中空糸型人工肺に用いられている。ポリエチレンやポリプロピレン製の多孔質膜は疎水性であり、短時間では微細孔からの血液漏れはないが、経時的に血液により膜が親水化され、血液が漏れ始めることが欠点であり、使用時間は8時間が限界とされ、ECMOへの応用は難しい。ポリオレフィンとPDMS 膜がもつ欠点を改良し、長時間使用が可能な人工肺(IVOX)の開発を目指し、多孔質膜に酸素透過性のよいPDMS、ポリ(4-メチルペンテン-1)あるいはポリアルキルスルホンなどの均質薄膜を複合化する動きもある。さらにフッ素系のポリイミド膜も研究段階であるが埋込型人工肺への可能性を示唆している。
