# 5・2・1 ティッシュエンジニアリングによる医療のパラダイム

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・2 ティッシュエンジニアリング

転換人体の機能低下または機能喪失を補うためにこれまで金属、セラミックス、ポリマなどの人工物や生体物質を加工した代用品でデバイスを作製し、生体機能代替の医療用具として用いてきた。急激な高齢化によりこれら医療用具が体内で使用される期間もますます延長されることになり、今後20年以上の期間にわたって性能が維持されることが必要条件となる。一方、代替物は体内の厳しい環境下でその機能を維持することは至難のわざであり、動物、他者または患者自身の細胞組織の有効利用は必然の医療技術となる。すなわち、多種類の細胞を三次元的に組織化して培養し、臓器・生体組織様のモジュール構造を形成する技術を開発することである。この技術をもとにして、臓器、骨、皮膚などの代替となりうる組織を人工的に創出し、これまでの人工物による代替技術に革命的な転換を図る必要がある。医療費の爆発的高騰のため、今後わが国では医療福祉レベルを維持することが極めて困難な状況にある。バイオテクノロジー分野の技術の開発を核として、コストを下げつつ、医療福祉レベルを向上させることが強く求められている。例えば、臓器移植は医療技術として広がりつつあるにもかかわらず、臓器の供給には限りがあり、社会的な問題となっている。アメ8―186生体工学リカなど諸外国への一方的な依存もこの先できないものと考えられる。この問題を根本的に解決するためには臓器再生技術(organregeneration)の開発が必須である。アメリカにおいては、代替臓器技術は世界規模での新産業領域であると認識され、きたるべき世紀における重要産業技術としてとらえ、戦略的取組みがなされている。わが国においてもバイオテクノロジーおよび医療福祉産業の国際的な競争に対応するための方策が緊急に必要である。これまで血管、皮膚などの単純な組織についてはプラスチックなどを用いた代替材料が開発されてきたが、性能および耐久性に問題がある。さらに肝臓など高度な機能の臓器の代替は人工材料だけでは不可能である。したがって臓器疾患の究極的な治療法として、ヒト細胞を用いた人工的な代替臓器の開発が必要である。多種類の細胞を三次元的に組織化して培養し、モジュールを構成する技術を開発し、代替となりうる組織を創出するティッシュエンジニアリング(tissue engineering)の確立が求められている。臓器移植は供給量の不足が懸念され、動物の臓器あるいはその臓器細胞の利用が当面の技術として考えられている。しかしながら、未知の病原体の問題など未解決要素があり、根本的な治療としては自分の細胞を用いた臓器の再生以外は考えられず、その基盤的技術の整備が急がれる。代替臓器の研究開発には近年大きな進歩が見られる。例えば大口径血管(5mm以上)であれば人工材料で形成した血管を長期に利用できるようになった。骨、歯なども人工材料を長期安定に利用できる。また、培養細胞の利用についても皮膚の上皮細胞は口腔粘膜の組織培養により代替が可能である。しかしながら、上記の成果はごく一部の成功でしかない。人工血管も細いものは血栓が生じて使えず、心臓冠状動脈のように重要な部位の治療に使えない。また、重度の熱傷などでは表皮だけでなく真皮層まで取り替えなければならない。また、皮膚機能としても完全ではない。軟骨細胞の移植はアメリカで企業化されているが、軟骨細胞の体外培養によって損傷部位をカセット的に置換することはまだできていない。軟骨細胞の培養自体には成功しても形態を再現する組織形成ができていないからである。これらの問題点が存在するにもかかわらず、細胞の組織培養の進歩は劇的な進歩を遂げており、上記の問題解決の可能性は高い。例えば、体外での培養・再生不可能の代表と思われていた肝細胞についても急速な進歩があった。肝細胞の増殖因子が発見され、肝細胞の生体外培養が可能になった。さらに肝細胞の組織培養の研究が急速に進み、単に浮遊培養だけでなく実質細胞が100個程度集まったスフェロイド状態での培養が行われている。肝細胞だけでなく多種の細胞から増殖因子が見つかっており、動物細胞の生体外培養は基礎研究の進展によって可能となりつつある。また、遺伝子導入、組織培養技術も進みつつあり、ティッシュエンジニアリングの確立に向けて着実な進歩を遂げつつある。
