# 5・2・2 ティッシュエンジニアリングの基盤技術

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・2 ティッシュエンジニアリング

a. 細胞ソースティッシュエンジニアリングはこれまでの人工材料を用いて機能を代替しようとする試みと異なり、細胞を用いることにより生体の組織や機能を再生しようとするものであり、細胞が基本となる。ティッシュエンジニアリングで用いられる細胞のソースとしては、自己の細胞である自家細胞(autologous cell)、他人の細胞である同種または他家細胞(allogeneic cell)、そしてブタなどの動物の細胞である異種細胞(heterologous cell)が考えられる。自家細胞は、拒絶反応がない、ウイルスなどの問題がないという利点をもつが、採取部位が限られている、細胞の活性が低い場合が多い、などの欠点をもつ。一方、同種細胞は、細胞のアクティビティが高い場合が多く、また感染性のチェックも可能であるが、拒絶反応を引き起こす、ケースによっては倫理的問題を引き起こすなどのデメリットもある。また異種細胞の場合は、ヒト型のブタのように遺伝子改変を施すことで移植用臓器または細胞としての可能性が追求されているが、未知のウイルスの危険性を排除できないという欠点がある。このように細胞ソースは利点と欠点を併せ持っており、ケースによって適切なソースを選び取っていく判断が求められる。細胞をどこから採取するという観点では、組織ごとに存在し分化しきった成熟細胞を用いるのが一般的であると考えられる。しかしながら一方で、幹細胞をティッシュエンジニアリングへ応用しようとする方向性が強く打ち出されている。幹細胞とは、自己複製能をもち、かつ多分化能をもつ細胞として定義される。幹細胞の中でも体細胞分化の頂点にあり、いわゆる全能性をもつとされる胚性幹細胞(ES 細胞)が注目されている。胚性幹細胞は、分化を適切にコントロールすることができればいかなる細胞へも分化できる潜在能力をもつが、クローン胚が出現しない限り基本的に他家細胞であり、また受精卵に関わる倫理的問題を切り離すことはできない。一方、ES 細胞のような全能性はないが、比較的多くの細胞に分化することのできる能力をもつ骨髄性幹細胞(mesenchymal stemcell)が注目されている。骨髄性幹細胞は骨髄に含まれており、自家細胞としての道も開けている。骨髄の中には、血球に分化することのできる血液幹細胞のほか、中胚葉由来の細胞(例えば骨芽細胞、軟骨細胞、筋芽細胞、腱・靱帯細胞、脂肪細胞など)に分化することの可能な幹細胞が含まれていると考えられている。さらに、組織中にあり、組織特異的な幹細胞(例えば神経幹細胞、表皮幹細胞、肝臓幹細胞、小腸上皮幹細胞、毛根幹細胞など)の存在も示唆されており、一部はすでに見いだされている。これらの幹細胞はES 細胞を除けば自家細胞として用いることは可能であり、また成熟細胞とは異なり組織形成能が高いと考えられており、ティッシュエンジニアリングにおいて有望な細胞ソースである。生体組織を生体外で再構築する研究において最も進んでいる分野の一つが再生皮膚(正確には再生表皮)である。皮膚はそもそも再生能が高く、日々新しい組織に置換わっていることや、再生皮膚を必要とする場面ではかなり緊急性が高いことなどの理由から、通常は他家細胞移植を前提にして研究が進められている。この場合、他家細胞に対する免疫的拒絶と自己表皮再生とが並行して起こり、移植した再生皮膚は治癒とともに脱落することが期待されている。一方、軟骨組織は再生能に乏しく、また体の比較的深部にあることや、治療に緊急性を要しない場合が多いことなどから、自家細胞を用いる研究が主流である。培養自家軟骨細胞移植法では、膝の関節鏡検査の際にバイオプシーにより体重の非荷重部から採取された軟骨組織に含まれる軟骨細胞をソースとして、生体外で増殖させた後に再移植する。この場合、本来体内で最も長い細胞周期をもつ細胞の一つである軟骨細胞にその培養環境を変化させることにより増殖を誘起させているため、増殖後の細胞の軟骨細胞としてのphenotypeの査定については十分に慎重に行われるべきであると考える。このように軟骨細胞そのものをソースとして用いる研究のほかに、上述したように未分化な細胞をそのソースとして用いようとする試みがある。骨髄由来の間質細胞には軟骨幹細胞が含まれていることから、骨髄由来の未分化間葉系細胞から軟骨細胞へ分化させる試みが行われている。しかしながら、筋肉細胞への分化を決定付けるMyoD に相当する強力な転写因子が軟骨細胞分化の場合は見つかっていないため、未分化な細胞から軟骨細胞を誘導することは確実には達成されていないが、各種分化因子を用いて軟骨細胞へ分化させることは可能であると考えられる。b. 三次元培養担体(three dimensional scaffold)ティッシュエンジニアリングは、ダメージを受けた生体組織を生体外で再生させた組織で置換または生体内で再生させる医学・工学技術である。現在すでに培養皮膚は欧米では商業ベースに乗って供給体制が整備されつつある。それと並行して他の組織、例えば軟骨、骨、靱帯、血管、心臓弁などが臨床応用を目指して試行が続けられている。その中で軟骨は比較的組織構造が単純で、一方で潜在的な需要が他の組織に比べて圧倒的に大きいという事実があることから、世界的に研究が積極的に展開されている。したがって、ティッシュエンジニアリングのための三次元培養担体用材料として再生軟骨のための材料を例にとり概説する。. 生分解性高分子材料(biodegradable polymer)生分解性高分子は医療用の縫合糸などで用いられており、加水分解などにより一定期間後に低分子に分解する高分子である。医療用としては乳酸-グリコール酸共重合体(PLGA)がおもに使われている。このほか、生分解性高分子としてはポリ乳酸(PLA)などが医療用として使用されている。ポリグリコール酸(PGA)は親水性が高く加水分解を受けやすいため、生体内に移植すると2~4週間で分解されてしまう性質をもつ。一方、ポリ乳酸はポリグリコール酸に比べ疎水性が高いため、長期に体内でその構造を維持することが可能である。乳酸-グリコール酸共重合体は、これらポリ乳酸とポリグリコール酸の性質を併せ持っており、細胞を三次元培養するための一時的な担体として適している。(1) ポリグリコール酸(polyglycolic acid)ポリグリコール酸は脂肪族ポリエステルの中で最も単純な構造単位をもっている。融点は224~226°C、ガラス転移点は36°C、密度1.5~1.64g/mL、対数粘度0.5~1.6、分子量は20000~145000であり、ヘキサフルオロイソプロパンノールやトリクロロ酢酸以外の多くの有機溶媒に不溶である。ポリグリコール酸は、グリコール酸やハロ酢酸、一酸化炭素などのC1化合物から合成される。生体内では比較的短期間で加水分解されるため、手術用の縫合糸の素材として臨床応用されている。(2) ポリ乳酸(polylactic acid)乳酸は光学活性体であるため、ポリ乳酸にはポリL-乳酸、ポリD-乳酸、ポリDL-乳酸があり、前者二つは結晶性であるがポリDL-乳酸は非結晶である。ポリL-乳酸の融点は170~175°Cである。ガラス転移点は分子量に大きく依存する。ポリ乳酸も加水分解で分解するが、その速度は分子量、形態、相構造、純度に大きく左右され、ポリグリコール酸に比べ遅い。したがって、比較的長期にわたって強度を保持しなければならない部位への補強材としての応用が試みられている。例えば、繊維にしたポリ乳酸は縫合糸としては分解が遅いために適用されないが、リボン状にして靱帯への応用を図ったり、メッシュ状にして顎骨再建への応用を図ったりする事例が存在する。またDDS へ応用する試みも見られる。(3) 乳酸-グリコール酸共重合体(polylactic-acid-poly-glycolic-acid-copolymer)ポリグリコール酸は結晶性が高く、融点も高いため加工しにくいという欠点がある。また、分解が速いため、より遅い分解が期待される部位には利用できない欠点もある。これらの欠点を補う目的でさまざまな共重合体の作製が試みられているが、その中で医療用を目指したものとして乳酸-グリコール酸共重合体がある。これは機械的強度があるが分解が遅いポリ乳酸と、機械的強度に劣るが分解が速いポリグリコール酸の中庸としての特徴を併せ持つポリマである。(4) 生分解性高分子の軟骨再生への応用ハーバード大学医学部のバカンティ(Vacanti)らは軟骨細胞をPLGA テンプレートとともに10日間ほど培養した後、ヌードマウスの皮下に移植し3~9箇月後に軟骨組織が再生していることを報告した。またMIT(マサチューセッツ工科大学)のフリード(Freed)らはウサギの関節から採取した軟骨細胞を同じく生体内分解性高分子であるPGA とともに3~4週間培養した後、別のウサギの関節軟骨欠損部に移植したところ、PGA のみの移植に比べ軟骨再生が良好であると報告している。. コラーゲンスポンジ(collagen sponge)コラーゲンはヒトの体を構成するタンパク質の約3割を占める主要なタンパク質であり、14種類の型が確認されている。その中で、I型コラーゲンは最も豊富に存在し、バイオマテリアルとしても多く利用されている。コラーゲンは3本のペプチドシークエンスがらせん状に絡まったtriple-helix 構造をしている。それぞれのシークエンスはプロリ8―187ンやハイドロキシプロリンに富んでおり、種を超えてよく保存されている。したがって、免疫原性も比較的弱い。医療用としては、さらに免疫原性を示すテロペプチドを酵素的に切断したアテロコラーゲンを用いる。コラーゲンはそのシークエンス中にRGD 配列をもつ。RGD 配列は細胞膜に存在する糖タンパクであるインテグリンと相互作用することが知られている。相互作用したインテグリンはその信号を細胞質に伝達し、細胞骨格の再配列や遺伝子発現をコントロールする。血球以外の細胞は細胞外マトリックスに接着して生育している。このとき、上述の信号がインテグリンより細胞質へ伝達されているが、細胞外マトリックスへの接着を阻害しインテグリンの信号を遮断した場合には細胞死(アポトーシス)を起こすメカニズムが働くことが知られている。このように細胞の細胞外マトリックスへの接着は、細胞を培養するうえで必須の条件である。その意味でコラーゲンを培養担体として用いることは合理的である。コラーゲンを三次元担体として成形するためにはファイバどうしを架橋(クロスリンク)することが必要である。クロスリンクは一般的には、熱架橋、UV による架橋、グルータールアルデヒドなどによる架橋が用いられている。. コラーゲン・生分解性高分子複合体コラーゲンは細胞の接着サイトを有しており、抗原性も低いことから、生体組織の生体外での再構築といったティッシュエンジニアリングの分野で今後も多用されていくであろうと考えられている。しかしながら、コラーゲンスポンジはその機械的強度が低いことが欠点とされている。一方、生分解性高分子は成形が容易で機械的強度があり、細胞による細胞外マトリックスの産生と並行して分解され生体組織に置換わることができる、という特徴をもつが、コラーゲンのような細胞接着サイトを有していないため、細胞を接着させ培養するという細胞適合性において劣っている。事故や先天的欠損症、癌などの疾患により失った組織や臓器を再構築する際に、その組織の形態を維持するマトリックスは非常に重要な役割を果たしている。このマトリックスには、生体に影響を及ぼさない性質としての生体親和性や、新しい生体組織が形成するとともに分解・吸収される生分解性および高度な機械強度などが要求されている。しかしながら、これらの条件を同時に満たすマトリックスはいまだ実現されておらず、生体親和性、機械強度ともに優れている高機能化した生分解性生体材料の創製が急務となっている。そこで、機械強度があり、成形することが容易で、分解速度が制御できる乳酸とグリコール酸との共重合体(PLGA)と、生体親和性が優れているコラーゲンを複合化することにより、生体親和性、機械強度ともに優れているPLGA-コラーゲン複合材料の作製が試みられた。(1) PLGA-コラーゲン複合スポンジの創製乳酸とグリコール酸の共重合体PLGA をクロロホルム溶媒に溶かし、塩化ナトリウム粒子を入れ、クロロホルムを飛ばした後、水で塩化ナトリウム粒子を溶出させることにより、PLGA スポンジが得られる。PLGA スポンジは塩化ナトリウム粒子と同じサイズのポアをもち、ポアとポアの間はつながっていることになる。得られたPLGA スポンジをI型アテロコラーゲン酸性水溶液に浸漬し、-80°Cで凍結した後、真空減圧下(0.2Torr)で24時間凍結乾燥、25%グルタルアルデヒド水溶液によって飽和したグルタルアルデヒドガスを用いて37°Cで4時間架橋処理し、洗浄して凍結乾燥することにより、PLGA-コラーゲン複合スポンジが得られる。この操作により、PLGA スポンジのポアの中にコラーゲンマイクロスポンジが形成される。PLGA スポンジは疎水性で、細胞の播種は困難であるが、コラーゲンとの複合化により、水との接触角が低下し、水ぬれ性は改善され、細胞の播種が容易となる。(2) PLGA-コラーゲン複合スポンジを用いた細胞培養ウシ肩関節の軟骨から分離した軟骨細胞をPLGA-コラーゲン複合スポンジに播種し、10%ウシ胎児血清を含むDMEM とHAM’SF12培地で37°C,5%CO雰囲気下で培養すると、軟骨細胞は複合8―188生体工学スポンジのコラーゲンスポンジの上に良好に接着し、スポンジの中の一部の空間が軟骨細胞の産生する細胞外マトリックスによって占められ、軟骨細胞と細胞の周囲にはII型コラーゲンが検出される。II型コラーゲンは硝子軟骨に存在するコラーゲンのおもなタイプである。関節軟骨は硝子軟骨であり、コラーゲンがI型である線維軟骨と異なりII型コラーゲンを産生することにより硬い組織を形成している。したがって、関節軟骨の機能を代替する再生軟骨組織の形成のためにはII型コラーゲンの産生は必須であり、三次元培養担体の軟骨細胞にそのような機能を発現させるものでなくてはならない。このような生体適合性(組織適合性、細胞適合性)と機械的適合性の両者を兼ね備えた三次元培養担体はティッシュエンジニアリングには必須であり、今後とも組織別に最適の担体がこのような観点から設計され臨床応用されていくであろう。c. 物理的因子生体内ではさまざまな物理的刺激が生理的な条件下で組織・細胞に負荷されている。例えば大腿骨には歩行などにより圧縮・引張応力が負荷されており、骨組織の微小変形や骨組織内の微小な体液移動によるstreaming potentialが生じている。また血管系には血流によるずり応力(shear stress)が血管内皮細胞に、また拍動によるストレッチが血管平滑筋細胞、血管内皮細胞に負荷されている。このようないわゆるメカニカルストレス(mechanical stress)が組織の形成や機能維持にとって重要な役割を果たしていることが知られてきている。したがって、このような生理的な刺激をシミュレーションして、生体組織を生体外で三次元再構築しようとする研究が始まっており、例えば培養液の灌流を制御することにより再生血管にずり応力やストレッチを与えながら再構築する研究がある。一方で軟骨組織の再構築のために軟骨組織に生理的に負荷されている静水圧力を負荷する研究がある。軟骨組織(ここでは股関節や膝関節における軟骨組織)は大腿骨と同様、歩行などにより負荷を受けている。軟骨組織は高いスティフネスを示す硬い組織であることのほか、プロテオグリカンなど荷電性高分子により水分子が保持されており、高い含水率を保っている。そのため加重は主として静水圧力というメカニカルストレスにより軟骨細胞に負荷されると考えられている。関節軟骨は互いに面接触ではなく線接触しているため、生理的に負荷されている静水圧力は予想以上に大きく、実測値によれば歩行時には3~4MPaも負荷されている。したがって、軟骨細胞の場合にはメカニカルストレスとして静水圧力が軟骨細胞の機能維持に積極的に関わっているのではないか、という考えのもと研究が進められている。そこで静水圧力というインプットに対して細胞外マトリックスの産生がどのように影響を受けるかについて、一定圧や間欠的な圧力負荷、圧力値も0.01MPaのオーダから10MPaのオーダまでさまざまな条件で研究が進められてきたが、おおむね細胞外マトリックスの産生に促進的な効果をもち、関連する遺伝子発現が変動することが確認されつつある。細胞外マトリックスの産生に変動があるということは、遺伝子発現に何らかの影響が与えられていることを意味している。したがって、シグナル伝達経路の上流にあるタンパク質のリン酸化、cAMPやカルシウムイオンなどのセカンドメッセンジャにも影響を及ぼしていると考えられる。軟骨細胞が静水圧力を感知するメカニズムとしては、ライト(Wright)らの報告にあるようにイオンチャネルが静水圧力のセンサとして働いているという可能性がある。また静水圧力は、形質膜(plasma mem-brane)の活動電位に影響を与えるという報告や、タンパク質の高次構造に影響を与えるという報告もある。軟骨細胞に周期的な圧縮力を加えるとcAMPのレベルが上昇したという実験結果も含めると、これら細胞膜上の変化が何らかのメカニズムでcAMPや他のセカンドメッセンジャの変化へと変換された可能性が考えられる。このように、軟骨細胞が何らかのメカニズムで静水圧力というメカニカルストレスを感知し、軟骨細胞の機能発現を維持し、場合によってはその分化程度を制御しているとするならば、生体外で軟骨様組織を再構築するための基盤技術として静水圧力負荷を一つのファクタとして組み込んでいくことは有効な手段となりうる。
