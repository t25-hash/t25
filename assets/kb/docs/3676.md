# 10・6・1 腐食試験（corrosion test )

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・6 その他の試験

a.電気化学的測定法 金属の水溶液中の腐食防食に関する基
本特性を得るもので，電極電位測定，電位-pH線図作成，分極曲
線測定（反応速度を明らかにする）などの試験法がある．
b.大気腐食試験 ばく露試験では，材料を実際と同じ環境下
で長期間放置し腐食の程度を判定する．大気のばく露試験に関する
規定にJIS Z2381：2001，H0521：1996がある．塩水噴霧試験
（JIS Z2371：2000）は，海洋近くの腐食を実験室内で再現させ
る．その他，腐食を人工的に加速させる促進試験が団体規格などで
提案されている．
c .海洋ばく露試験 おもに鋼構造物について行われるもので，
海洋環境を，大気帯，飛まつ帯，干満帯，海水中に分けて腐食状況
を調査する．
d.各種水溶液中での腐食試験 対象となる金属と腐食環境の
組合せの数は膨大だが，比較的耐食性の良い金属材料と，腐食性の
強い環境のうち重要なものについて多くの規定がある．金属の酸・
アルカリ溶液に対する腐食試験（おもに質量減測定）のほか，ステ
ンレス鋼などに見られる孔食，すきま腐食，粒界腐食の各試験法
（JIS G0571：2003など）がある．
e .応力腐食割れ試験（stress corrosion cracking test ，SCC
test ) 応力腐食割れが問題となる腐食環境は，金属により異な
り独特のものである．試験法を大別すると定ひずみ法と定荷重法が
ある．き裂進展試験では，き裂進展速度と応力拡大係数（ ）の関
係や下限界条件 を得る．促進試験として低ひずみ速度法
（SSRTあるいはSERT法と呼ばれる）が利用されている．
f .腐食疲労試験（corrosion fatigue test ) 腐食環境下の疲労
試験で，大気中に比べ短寿命となり疲労限度も低下する．
g.高温ガス腐食試験 各種高温ガスによる腐食を測定するも
ので質（重）量法が多く用いられる．腐食の種類には，高温酸化，
水蒸気酸化，バナジウムアタック，硫酸露点腐食，高温高圧水素浸
食，硫化腐食などがある．
