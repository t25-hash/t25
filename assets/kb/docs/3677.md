# 10・6・2 摩耗試験（wear test )

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・6 その他の試験

摩耗を接触の仕方で分けると，2金属間の滑り摩耗，転がり摩耗
（軸受など)，土砂，岩石との間の土砂摩耗があるが，実際にはこれ
らが混在することが多い．耐摩耗性は，一般的には硬くて靱性に富
んだ材料が優れているが，摩耗の機構によって変化する．したがっ
て，摩耗試験は実際の使用状態をできるだけ再現したほうがよい．
摩耗試験機の構造には，①回転する円板の平面部に試験片を押付け
る方法（ピンオンディスク)，②回転する円板外周面を平板に押付
ける方法，③円筒外周面と小試験片を摩擦する方法，④往復運動す
る平面に試験片を押付ける方法，⑤回転する円筒どうしを押付ける
方法，⑥土砂などの中で試験片を回転させ摩耗量を見る方法などが
ある．
