# 5・3・1 生体から発生する信号の計測

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・3 生体計測

生体の大部分は、膜で仕切られた電解質水溶液で構成されている。これらの電解質の移動が生命活動を担っている。電解質の移動は電流に相当するので、電位差、磁界などの形で、生体から発生する信号を計測できる。電気信号は電極によって検出される。電極には、皮膚を貫通させる形式や、導電性ゲルを介して皮膚に密着させる形式のものがある。信号が微小の場合には、さまざまな方法で、目的信号をノイズから識別する。両方の電極に同時に現れるノイズに対しては、両信号の差をとることで、ノイズを低減することができる。生体の発生する信号は、完全な周期関数ではないが、短い期間内では繰返し同種の信号が発生していると考えられる。この期間内で、時間をいくつかに区切って複数回の測定とみなし、波形の重ね合わせなどによって平均値を算出する。a. 脳波(EEG : electroencephalogram)細胞膜は、細胞の内と外とを仕切っている。この膜は、選択的に物質を通したり、せき止めたりしている。膜における物質の通り道をチャネルと呼ぶ。チャネルは状況に応じて開閉する。チャネルを通してイオンが選択的に移動する駆動力は、細胞の内外間で0.1V 程度の電位差に相当する(図5・12)。通常の状態での電位を静止電位(restingpotential)と呼ぶ。刺激に反応してチャネルが開いたときに脱分極し、その後チャネルが閉じて元の状態(再分極)に戻る。このチャネルの開閉に伴う電位の時間変化を活動電位(action poten-tial)と呼ぶ(図5・13)。その変化のタイミングは、場所によって、また細胞によってずれる。このために、細胞集団における異なる2地点間には、電位差が生じる。脳において計測される電圧の時間変化を脳波と呼ぶ。脳波には、神経細胞ネットワークにおける活動電位の伝ぱを反映した信号が含まれている。ヒトの脳波信号は、頭皮に張り付けた電極によって検出される。電極と頭皮との接触部分は、電気的にコンデンサと等価の要素をもち、周波数特性を有する。一般に、電極と生体との接触面では、電極材料と生体部位との組合せにより、電気的に一方がプラス、他方がマイナスとなり電気二重層が形成される(図5・14)。したがって、電極と頭皮との間での機械的なずれ(振動)は、静電容量の変動という形で、ノイズの発生の原因となる。鼻根と後頭結節(突起)を結ぶ線、および左右の耳介を結ぶ線を図5・12静止電位図5・13活動電位8―19 1基準として、頭皮における電極の左右対称の設定位置を規格化している。電圧の測定においては、例えば耳垂に基準電極を配置し、この基準電位との差を採用する方法と、頭皮上の2地点に配置された電極間の電位差を採用する方法とがある。頭皮において検出される電圧信号は、数十V の振幅で、0.5~70Hzの周波数である。脳波は、周波数帯によって区分され、おのおのδ波(0.5~3.5Hz)、θ波(4~7Hz)、α波(8~13Hz)、β波(14~25Hz)、γ波(26Hz~)と呼ばれる。δ波は深い睡眠時、θ波は睡眠導入時、α波は安静時、β波は覚醒時に特徴的に見られる波形である。頭皮上に多くの電極を配置することによって、波形の局所的な異常を識別する。また、波形の類似性を分布図として表示して、脳における電気現象を推定する。眼球運動による信号や筋電図信号の混入を区別して計測するためには、被験者にリラックスしてもらう、まばたきを我慢してもらうなどの工夫が必要である。b. 脳磁図(MEG : magnetoencephalogram)電流は磁界を発生する。局所における磁界を計測できれば、その磁界は局所の電流に対応する。電流方向およびその強度によって発生する磁界が異なる。この磁界の脳における分布を示したものを脳磁図と呼ぶ。頭部全体の204箇所にセンサを設置して磁界分布を記録できる装置が開発されている。発生する信号は微小で、ノイズからの識別は容易ではない。超伝導の利用により、測定系の抵抗を減らすことによってノイズを低減させ、微小な磁界の計測を可能にしている。脳において発生する磁束密度は10~10T であり、地磁気のレベル(10T)よりもずっと小さい。磁気シールドで囲まれた空間内において、周囲の磁気を遮断し、液体ヘリウムを利用して低温下で超伝導状態を実現することによって熱雑音を低減させ、微小な磁気信号の計測を可能にしている。脳波測定では、複数の活動電位が加算された後の合成信号が測定される。その合成信号から各要素の信号を取出す試みが行われてきた。それに対して、脳磁図では、局所の磁気信号を個別に検出しようとしている。各磁界を発生している局所電流の位置や、局所における伝ぱの様子が、三次元的にとらえられる可能性がある。c. 程度の振幅である。1~20Hz程度までの高調波で波形を分類でき、診断に役立てられる。長時間にわたって連続的に心電図を計測する方法の例としては、ホルタ心電計がある。これは不整脈の診断などに対して有効である。d. ベクトル心電図(VCG :vectorcardiogram)心臓において発生する起電力の時間変化を表したものが心電図であるのに対して、その方向と大きさを空間内での軌跡として表現したものがベクトル心電図である。通常、立位において、左右方向を軸、上下方向を軸、前後方向を軸として座標軸を設定し、胸周囲上および足・手首に電極を配置して計測する。これらから、疾患の有無を見分ける。多数の電極から得られた信号を再構成して、各時刻における電位の分布図を三次元空間内で表示することも可能である。信号伝達異常部位の診断には有利であるが、不整脈のような信号発生のタイミングの異常の分析には向いていない。e.
