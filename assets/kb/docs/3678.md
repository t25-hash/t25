# 5・3・2 生体物性の計測

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・3 生体計測

生体高分子は、温度や周囲の流体のイオン濃度などの条件によって立体構造が変化する。このことが生体組織全体の物性を変化させるので、計測する際の温度条件を考慮することが必要である。生体の一部を取出して試験する場合には、乾燥を防ぎ、pH などの制御された等張の水溶液中で計測すれば、生体内での状態に近い形での計測となる。a. 力学的性質生体をさまざまな性質をもつ要素の複合体とみなすことができる。したがって、生体の力学的な性質は異方性を有した複合材料と同様に考察される。また、一般に粘性と弾性の複合した粘弾性を示すため、計測の際に変形速度を考慮する必要がある。. 化学的性質生体は多種類のイオンや分子から構成されている。その組成は一様ではなく、局所ごとに異なる。採取した試料を細分化した後に、化学反応を利用して沈殿物や着色物に誘導し、定性的に物質を特定する。蛍光物質に誘導して検出感度を高める工夫もある。各成分の含有量の測定方法としては、沈殿法による沈殿物の質量の測定、反応性の基の滴定、気体発生反応による気体体積の測定、呈色反応により生じた物質の吸光度測定などがある。溶液状態の試料においては、細管中を流しながら、定性分析と定量分析を流れ作業的に自動化した測定機械が利用される。また、試料の区分け流路、センサ、信号処理回路などをチップ板に集積させて、微量な試料で簡便に測定する工夫が開発されている。溶液中の溶質の分子量は、溶液の粘性係数によっても間接的に測定できるが、生体高分子など、非ニュートン性を示すものの測定については、a項のiiで述べた円すい・平板型粘度計が有効である。温度や溶質の濃度による変化を測定することによって、溶質分子の性質を推定する。溶液中の生体高分子は、pH(水素イオン濃度)によっても立体構造などが変化するので、計測の条件として考慮する必要がある。血液のin
