# 10・6・3 被削性試験（machinability test )

> **章**: 第10章 金属材料の検査と試験
> **節**: 10・6 その他の試験

被削性試験は，実用されている工作機械（ボール盤，フライス
盤，普通旋盤，ホブ盤など）で工具，切削条件を定めて行われる．
特に多く使用されるのは，ボール盤，普通旋盤である．通常，以下
に示す工業的特性を調査する．
(１)工具寿命：工具，送り，切削速度を定め，工具の摩耗が一
定の値に達する時間を求める．
(２)仕上面粗さ：一定時期の仕上面を調査する方法と，仕上面
が正常である切削条件の範囲を調査する方法がある．仕上面粗さの
測定の手順はJIS B0601：2001に示されている．
(３)切りくず処理性：作業性の面から問題となる性能で，一般
的には細断されやすいほうがよいとされる．しかし，細かすぎると
排出性が悪くなり判断基準が不明確である．目的によって種々の分
類法が試みられている．
その他，切削抵抗，切削温度を測定して，切削機構の解析も行わ
れる．
