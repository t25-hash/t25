# 5・3・3 生体の状態量の計測

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・3 生体計測

a. 流量生理液体にはイオンが含まれているので、その流れを電流に対応させることができる。イオン濃度を一定とみなせば、その電流は流量に比例する。電流に垂直な方向に磁界をかけると両者に垂直な方向に起電力が発生する(図5・32)。この起電力の大きさが流量に比例することを利用するのが電磁流量計(electromag-netic flowmeter)である。起電力を検出するための電極および磁界を発生させるコイルの配置を工夫する。例えば、血流を測定するには、血管の外側に接触するように配置するか、血流の内部に挿入するカテーテルと呼ばれる細い管に付属させるなどの方法がある。ただし、カテーテルを挿入する方式では、血管径を別に測定しないと流量を算出できない。人工心臓のように人工的に流路を形成する場合には、流路の途中に電極とコイルを組込むことが可能である。いずれの場合でも、電極と血流との電気的な接触状態を長時間にわたって安定させるのは困難である。気体が介在したり、血栓や生体組織が界面を覆ってしまうためである。流量零の点を出力信号のうえで保証するのに拍動流を利用できる。大動脈基部では、心臓からの血液の駆出は間欠的で、流量が零と最大値との間で変動を繰返す。この流量零のときの信号を利用して、計測系の零点がずれていかないように監視する。電気的に中性の流体に対しては、電磁流量計を適用できない。熱希釈の原理を利用して、温度変化から流量を測定できる。測定しようとする流体と温度の異なる一定体積の流体を瞬時に注入し、その後の流体の温度の時間変化を測定する。流量が大きいほど、元の温度に回復するまでの時間が短い。0°Cの濃度0.9%塩化ナトリウム水溶液1cmを心臓付近に注入すれば、その後の血液温度の変化から心臓の拍出流量を測定できる。また、流れの途中の位置で血液を加温すれば、その位置よりも上流と下流とで温度差が生じるが、流量が大きいときには温度差が小さくなる。熱電対を用いれば、2地点間の温度差を直接測定できる。弾動心拍出量計では、心臓が動脈系に血液を駆出する際、および駆出された血流が大動脈弓で進行方向が変化する際の反動を全身における振動から検出する。動脈では、心臓の拍動に伴って流量が周期的に変動する。その時8―19 圧力生体の状態量としての圧力の中では、血液循環における圧力が重要である。血管内圧と周囲環境圧力との差を血圧と呼ぶ。血圧は、心臓のポンプ機能、血液循環の抵抗、血管壁を通じての液体の出入り、血管壁の変形性などと関連しており、その計測値がさまざまな形で利用される。脈圧計では、既知の圧力で血管を外部から圧迫して、動脈血管の内圧を測定する。例えば、一般の血圧測定法に用いられるようにカフで上腕を周囲から圧迫し、周囲圧力を高いほうから徐々に低下させていき、血流が途絶えていた状態から再開されるときの変化を検出する。上腕動脈においては、約9~16kPaの間で心臓の拍動とともに周期的に変動する圧力が測定される。直接的に血圧を測定するには、カテーテルと呼ばれる中空の細管に濃度0.9%の塩化ナトリウム水溶液を充てんしたものを血管内に挿入して、センサに接続して測定する。センサとしては、ダイアフラム型が一般的であるが、これはダイアフラムの変形をひずみセンサなどによって電気信号に変換するものである(図5・34)。動脈圧を測定する場合には、圧力が時間とともに変動するので、カテーテルを含む測定系における信号の時間変動成分の伝達特性に注意する必要がある。液体の圧力は重力場の影響を受ける。立位では、頭部と足部では1m以上の落差が生じる。血液循環においては、心臓の位置を基準にする。右心房の内圧を中心静脈圧と呼ぶ。血圧以外の例として、眼圧計では、圧縮気体を吹き付けて、既知の圧力によって角膜がどの程度変形するかによって内圧を測定する。0.05N 温度生体内では、物質の拡散や化学変化が絶えず行われている。その速度は、温度によって大きく変化する。したがって、生体各部の温度の情報が重要な意味をもっている。体温はさまざまな状況に応じて変動するが、生体には体温を生命活動に適した温度に維持する働きがある。体温の変動には、周囲環境の温度およびそ図5・35ヘマトクリットれにさらされる時間による変動、日内・性などの生理周期的変動、筋などの活動状況による変動などがある。皮膚などの体表における温度の計測と、身体の深部の温度の計測とに分類することができる。基本的には、温度検出部としてのセンサが被測定温度に等しくなる必要がある。したがって、センサの熱容量が小さいほど被測定部から受け取るエネルギーが少なくて済む。被測定部の温度を乱さず、迅速に測定できる。または、演算によって最終平衡温度を推定し、測定時間の短縮を図ることも考案されている。水銀温度計やアルコール温度計では、液体の熱膨張を利用している。白金抵抗体温度計・サーミスタ温度計では、温度による電気抵抗の変化を利用している。ブリッジ回路の利用などにより、電気抵抗を測定するのに流す電流をできるだけ小さくする必要がある。抵抗における発熱(自己加熱)で被測定物の温度を変えないようにするためである。他方、a項で述べたように、抵抗周辺の流れを温度変化によって測定することができる。熱電対を用いて温度を測定するには、基準の温度が必要であるが、2地点間の温度差については、そのまま測定できる。深部の温度を測定するには、カテーテルなどに付随させてセンサを体内に挿入する。赤外線の放射を受取れば非接触状態で温度を計測できる。サーモグラフィー(thermography)では、この方法で体表の温度分布を測定する。耳の鼓膜からの赤外線を検出する方法は、脳を灌流する内頸動脈血流温度との対応関係があるとされており、深部体温の指標として利用される。e. 密度。 ヘマトクリット(hematocrit)赤血球の血液全体に占める体積百分率をヘマトクリットと呼ぶ。重力の加速度の2×10倍の遠心力を10分間加えることによって赤血球を沈降させて、上澄み液体との体積比率によって測定する。内径1mm、長さ75mmのガラス管に、毛細管現象を利用して抗凝固処理済みの血液を吸い込み、管の一端を粘土状のふたでふさぎ、専用の遠心装置で遠心する(図5・35)。用いるガラス管の内壁に抗凝固剤のヘパリンをコーティングしたものもある。. 赤血球数血液1mm当たりの赤血球の個数を赤血球数と呼ぶ。血液中の赤血球密度とも呼べる量である。文字通り、一定体積中の赤血球の数を計測する。所定の間隔に設定された平行平板間のすきまに毛細管現象を利用して血液を吸引し、一定区画内の赤血球の数を光学顕微鏡下に目視で計数する。トーマ・ツァイス計数板は、1mm×1mm×0.1mmの空間内をいくつかの区画に分割してあり、血球数を数えやすいようになっている。自動血球計数機では、所定体積の血液が微細な直径の管を通過する間に、血球の個数を計数する(図5・36)。管の途中において、光線の透過強度を測定し続けると、赤血球の通過に伴って、透過光強度がパルス状に変化する。このパルスをカウントすればよい。光以外に、電気抵抗や静電容量も用いられる。計数を容易にするためには、あらかじめ、血液を規定の倍率で希釈しておくことである。赤血球以外の粒子を区別するためには、計数する前の試料から他の粒子を取り除いておくか、計数中のパルスの形によって識別する。. 濃度生体の大部分を占める水溶液において、その濃度が適切な値に維持されていることは、生命活動に不可欠である。特に全身を循環する血液においては、タンパク質や各種イオンの濃度の計測がさまざまな指標に利用される。通常、溶液の光透過強度によって定量する。. 屈折率測定器濃度によって溶液の屈折率が変化することを利用する。既知の屈折率をもつ90°プリズムを用いて、被測定溶液との間での全反射の臨界角によって屈折率を求め、溶液の濃度に換算する。血漿中タンパク質濃度などの測定に用いられる。. 比濁法沈殿反応を利用する。すなわち、濃度を求めようとする目的物質と沈殿反応を起こさせ、生じた微粒子による散乱光強度を測定する。光線を微細なすきまを通して入射させれば、微粒子の濃度に比例して光線が散乱する。このことを利用して目的物質の濃度を求める。. 分光光度計特定の波長の光の吸収の程度から、溶液中の物質を同定したり、定量したりする。試料自体を発光させるか、または、タングステンランプ、水素放電管などの光源から、目的とする波長の光を選択して用いる。波長の選別には、プリズムや回折格子が用いられる。一定の波長の光を入射させたときの吸光度は、その光が透過する液相の厚さ、および溶液の濃度に反比例する。透過光の定量的検出には、光電管や熱電対を用いる。.
