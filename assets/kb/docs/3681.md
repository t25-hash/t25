# 5・3・5 生体の機能計測

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・3 生体計測

a. 力学的機能筋の収縮機能の計測においては、条件の単純化に関して二つの方法がある。等尺性収縮(isometric contrac-図5・40内視鏡8―19 9tion)と、等張力性収縮(isotonic contraction)である(図5・41)。前者では、筋を一定の長さに保ったまま収縮させて張力を測定する。後者では、筋に一定の荷重を加えた状態で収縮させて長さの変化を測定する。等尺性収縮では、筋の両端を固定して、収縮を起こさせ、ひずみゲージなどで収縮力を測定する。単一の刺激による収縮を単収縮、適度な頻度による複数の刺激により、大きな収縮力を発生する場合を強縮と呼ぶ。収縮力は筋の長さに依存し、静止長で収縮力が最大となる。それよりも長くなっても短くなっても、収縮力が減少する。等張力性収縮では、筋に一定の荷重をかけ、筋に収縮を起こさせて、長さの変化を測定する。測定系の慣性を無視できれば、収縮中に筋にかかる張力は一定とみなせる。荷重の増加とともに収縮速度が減少する。運動中の生体信号の計測では、運動に支障のないように、無線送信を利用して計測する。筋を支配している運動神経を皮膚上から電気刺激して得られる筋電図を誘発筋電図と呼ぶ。負荷心電図では、階段昇降、トレッドミルなどの運動を負荷したり、酸素が欠乏する条件を設定したりして、心電図を計測し、安静時と比較する。体動の計測では、あおむけに寝た状態で、自発呼吸による胸壁の運動を長時間にわたって計測する。例えば、睡眠中に突発的に呼吸停止が起こった場合にただちに蘇生術をとれるように監視するのが目的である。生体は、外界との接触面において、接触面積や圧力を調節している。手でものをつかんで保持したり、足底で地面をとらえて姿勢を保持したり、反力を得て運動する。また、睡眠中でも、寝返りを打って、応力の集中を防いでいる。これらの接触面における圧力の変動を計測するのは容易ではない。外界側の面を透明にできるときには、光学的方法で接触状況を観察できる。センサを挿入する場合には、元の接触状態を変化させないように注意しなければならない。薄いシート状のセンサを用いて、面内の圧力分布の時間変化を計測する。ひずみゲージセンサ、静電容量型センサ、感圧紙などを応用したものが利用される。立位における足底圧の時間変化から平衡機能を測定することができる。運動中の足底圧から運動機能・反射機能を測定することも可能である。寝具と体表面との間の接触面での計測は、褥瘡(じょくそう)を防止する方法の考案へとつながる。b. 代謝機能ヒトの赤血球は、骨髄で産生されてから、脾臓などで捕捉されるまで、平均120日間循環し続ける。この期間を赤血球寿命と呼ぶ。放射性同位体を用いて赤血球を標識すれば、循環している赤血球が脾臓に捕捉されるまでの時間を計測できる。皮膚において、周囲に比べて電気抵抗の低い場所がある。東洋医学の経穴との関連性において計測が行われる。発汗によっても、電気抵抗が変動する。手のひら表面の2点間に電圧を加えたときの電流値は、刺激に応じて数十%変動する。皮膚電気反射では、汗腺における電気抵抗の変化を測定する。血管壁が破れると、その破断部分は凝血塊によってふさがれ、壁が修復されていく。凝血塊は血液中の成分の変化、および血管壁からの物質の分泌の両者が共同して形成される。凝血塊形成能の計測においては、血液凝固時間をはじめとするいくつかの方法がある。採取した血液が試験管内で凝固するまでの時間を測定するのが凝固時間測定法である。例えば、温度を37°Cに保ちながら、採取した血液を入れた試験管をゆっくり回転させ、血液が凝血塊を作って液体成分の光の透過度が変化するのを検出して、採血してからの時間を表示する。トロンボエラストグラフィー(thromboelastography)では、血液凝固時間のみならず、形成された凝血塊の流動抵抗や退縮、線溶系による凝血塊の減少を計測する(図5・42)。二重円筒のすきまに試料血液を注入し、外筒容器を回転対称軸回りに一定の回転角振幅で回転振動させ、血液にクエット型の流動を加える。内筒はねじれ変形できるような線でつるされている。凝血塊が形成されて二図5・41等尺性収縮と等張力性収縮8―200生体工学図5・42トロンボエラストグラフィー図5・43血小板凝集観察装置重円筒間の流動抵抗が増加すると、内筒が外筒の振動に合わせて振動し始める。凝血塊の成長とともに内筒の振動の振幅は増加する。その後凝血塊が退縮し始めると、内筒の振幅も減少する。内筒の振幅の時間変化を記録すれば、凝固開始時間、凝血塊形成時点での血液の流動抵抗、退縮に伴う流動抵抗の回復など、凝血塊形成を動的に計測できる。円すい平板や凹凸円すいなどのクエット流試験機において、回転トルクを検出することによって、血液の流動抵抗の時間変化を測定すれば、流動下凝血塊形成能を測定できる。目的とするせん断速度の流動を加えた条件で、凝血塊形成までの時間、凝血塊形成の程度を測定できる。円すいの回転速度を時間とともに変動させれば、拍動流下のように周期的にせん断速度を変動させたもとでの凝血塊形成能の計測も可能である。血小板凝集能の測定法の例として、血小板凝集観察装置がある(図5・43)。血小板はADPによって凝集する。血液の流量とADPの流量を独立に制御する。両者の流れをADPを透過する半透膜で隔てて接触させる。ADPの濃度が高くなるほど血小板の凝集が促進される。また、流れが速くなってせん断速度が増加すると、血小板凝集塊の成長が抑制される。半透膜上での凝集塊の成長を光の透過度などで測定すれば、ADPおよび流れに対する血小板の反応をおのおの独立に評価できる。人工臓器と生体組織との界面での接着性についての評価試験も細胞培養技術等の導入により種々工夫されている。糖負荷試験では、口から消化器を通じて、または静脈に直接、ブドウ糖を注入し、その後の血液中の糖の濃度の時間変化を測定する。生体機能の測定法が進歩すれば、死の判定方法の問題点が改善される可能性がある。現在の脳死判定は、以下のように記述されている。(1)深昏睡、(2)自発呼吸の消失、(3)左右瞳孔の直径が4mm以上、(4)脳幹反射(対光反射、角膜反射など)の消失、(5)平坦脳波、(6)6時間経過後に(1)~(5)の状態が再度確認される。なお、以上の基準は、6歳未満の小児、急性薬物中毒、低体温、代謝・内分泌障害の場合には適用されない。判定の困難さは、図5・44バイオセンサ生体機能が十分には解明されていないこと、および計測法の開発が不十分であることと関連している。
