# 5・3・6 生物を利用した計測

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・3 生体計測

生体は、それ自体を計測システムとみなすことができる。マクロなレベルでは、個体が周囲環境に応じて行動したり、個体集団が環境変化に対応して行動したりする。ミクロなレベルでは、生体高分子が周囲の環境条件に応じて立体構造を変化させたり、細胞が周囲の特定の物質や環境に反応して変形・移動したり、物質を分泌したりする。前者に関しては、地震の予知にナマズや鳥類の行動を役立たせようとする研究がなされる。後者に関しては、研究成果がバイオセンサ開発などの応用分野へと発展している。バイオセンサは、生体を計測システムに組込むものの一つである(図5・44)。生体には、外界から情報を識別して選択的に受取り、その情報を変換し、内部に蓄積し、外界へ情報を発信するなどの働きがある。例えば、細胞のレベルで考えると、生体膜には、タンパク質や糖質の高分子が配列され、接触する物質・周囲の流れ・変形などの入力に対するセンサの役割を担っている。出力としては、電気信号・化学物質の分泌・形態変化・増殖・細胞どうしの凝集などがある。さまざまな信号は、電気信号に変換されれば、電気回路の技術によって計測システムを楽に実現できる。電子回路基盤上に、生体高分子、酵素、微生物などを固定化したセンサが作られるのはそのためである。固定化においては、いかに生体高分子などの立体構造を有効に作用できるような状態に保つかが課題となる。抗原抗体反応に代表される特異的な反応のためには、その鍵となる構造が生かされた形で固定される必要がある。適切な高分子を間に挟んで固定化することによって、電気化学的平衡を利用して、反応基の位置や向きを制御するなどの工夫が考えられる。温度や水溶液中の水素イオン濃度をはじめとする周囲の物質の状態などの環境条件もセンサの反応に影響を与えるので、周囲に余計な物質が近づかないように、多孔質膜などを配置する。これは、多孔質膜が選択的に分子などをふるい分ける効果をねらったものである。バイオセンサ(biosensor)は、生体反応を利用するので、糖、アミノ酸などの生体由来物質の検出に向いている。微量物質の検出にも有効であるが、反応時間が長いことも多く、計測時間の短縮が課題となる。赤血球をマイクロカプセルと見れば、流れの中でのせん断応力センサとしての応用が考えられる。また、血管内皮細胞は、血管壁付近の流れに応じて物質を分泌している。これを利用すれば、せん断流センサとしての血管内皮細胞の応用が考えられる。感覚器細胞を利用した計測システムも開発される可能性がある。胎児幹細胞や体性幹細胞からの分化制御、細胞増殖の研究の進展によって細胞の応用範囲が広がると思われる。
