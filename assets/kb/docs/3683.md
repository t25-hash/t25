# 11・1・4 RPの効果および影響

> **章**: 第11章 三次元造形技術
> **節**: 11・1 概説

RPの適用は生産工程へ大きな影響をもたらした．
その一つはデータに対する認識の変化である．三次元データであ
っても健全性が保たれなければ容易に使用できないこと，逆に完全
であればデータの使いまわしができることが認識され，CAD/
CAM（computer aided manufacturing）/CAE/ CAT（computer
aided testing）化が進むとともにデータの中身が充実していった．
それとともに不具合の前倒し検討や同時設計が可能となり，工程は
短縮され，コラボレーションによる前倒し加工などソリューション
革命を進めた（図11･9）．
最近では既存の加工方式も，積層の概念が普及することも相まっ
て高速化が進んでいるため，RPの概念も広がってきている．
