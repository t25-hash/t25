# 11・2・5 その他の造形法

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

a.インクリメンタル成形（incremental forming) 薄板成形
品は，シート成形法で作った紙積層（含浸）品や鉄板積層模型を用
いて，通常のプレス成形あるいは対抗液圧成形することも行われて
いるが，一枚の板から直接NC加工で打出す三次元造形法がある．
いわゆる，ハンマによる打出し成形を自動化したものである．通常
は最終形状を眺めて，全体を順次に変形させていくが，積層造形の
加工法，つまり等高線加工でパンチを制御して型なしで板金部品を
作る方法である．複雑形状で極端に個数の少ない製品，特に大物成
形には向いている．
b.高速切削加工 やはり等高線加工を切削に適用して実現し
た方法である．最近，高速回転スピンドルと高テーブル送りが実現
したため，可能になった．当然，曲面加工ではスピードの加減速が
あるため，直線部分の多い大物加工に向いている．
c .粉末レーザ溶融法 基本的には粉末積層法だが，焼結法で
はなく敷き詰めた金属粉末を完全に溶融しながら積層する方法であ
る．焼結とは異なり溶融時の収縮による材料移動を伴うため，変形
を防止する造形パターンを使用する（図11･22) ．
d.レーザ溶射法（laser cladding，laser generating) 敷き
詰めた粉末をレーザ溶融する方法では自由度が低いので，エネルギ
ー源としてレーザを用い，金属粉末をガスで吹込む造形法である．
