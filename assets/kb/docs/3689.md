# 11・2・6 応用，新展開

> **章**: 第11章 三次元造形技術
> **節**: 11・2 三次元造形法

三次元造形に関する期待は，ものづくりの状況が大きく変ぼうし
ようとしている21世紀のキーテクノロジーとなっている．おもな
ポイントは次の三つである．
(１)適用範囲の拡大
三次元造形に関しては，日本は世界と大きくその志向が異なって
いる．日本には下請制度に根差したこだわりが大きく，寸法や仕上
げ精度に関する要求が大きすぎる傾向がある．そのため三次元処理
への移行の遅れが大きい．その一方で，多品種少量生産への要求に
応えられる三次元造形の適用は着実に拡大すると思われる．
(２)バーチャル造形
ものづくりのベースとなる対象，オブジェクトに対する意味付け
が大きく変化してきているので，今後は形状寸法だけでなく構造特
性まで含めた定義と実使用環境をシミュレートできるComputer
Asisted Manufacuturingを実現するようになる．二次元加工に関
する過去の遺産を引きずっている日本とそれ以外の世界での競争は
激しくなり，過去のデータの使いまわしができる範ちゅうでは試作
は減少するが，全く新しいものづくりにおいては実感の占める割合
はさらに増加し，試作そのものは増加すると考えられる．
