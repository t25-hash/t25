# 5・4・7 神経系臓器:人工神経(artificial neuron)

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・59 )。膜に細孔のあいていない均質膜としてはシリコーン(ポ

人工臓器としての観点からの人工神経へのアプローチは、対象となる神経系より、末梢神経系と中枢神経系に分けて考える必要がある。また生体機能に依存した神経再生方法と生体以外の材料を利用した神経機能補助装置とに分類することができる。a. 末梢神経系における損傷神経系の再建末梢神経系では、損傷した神経系を回復させるために、神経鞘の形成を促す人工材料を用いたガイドチャネルによる末梢神経再生研究が行われている。このような材料として最近注目されているものに、神経再生をガイダンスするコラーゲンチューブやキトサンチューブがあり、動物実験によりその有効性が確認されている。また神経の再生を促す因子として動物レクチンの探索が精力的に行われており、とりわけガレクチン-1が神経の再生を促すということが知られている。b. 神経幹細胞(neuronal stem cell,neural stem cell)による中枢神経系の再生神経細胞は誕生直後から細胞死により取り除かれ減る一方であると従来考えられてきた。しかし最近になって、成体の脳内でも神経細胞が再生していることが広く認められてきている。例えばほ乳類の海馬歯状回、および側脳室に面した上衣細胞層あるいは脳室下帯には神経幹細胞があり、成体においても神経細胞は新生している。このような神経幹細胞の自己複製と多分化能を利用して、神経系の破壊に伴う重篤な機能障害を修復しようという試みもなされてきている。組織工学からの神経幹細胞の利用方法としては、ヒドロゲルに神経幹細胞を懸濁し、それを生分解性ポリマの足場とともにげっ歯類の脳内に戻す研究が行われている。c. 生体以外の材料を用いた感覚神経系へのアプローチ神経と同様な機能を有する人工的な神経素子を実現する方法として、有機色素膜や導電性高分子膜を利用して神経細胞のシナプス可塑性を模倣するような分子組織体による神経素子、半導体などのエレクトロニクスによる神経素子、神経細胞と電子回路を接続するためのインタフェースなどが考案されてきている。これらの神経素子では学習と記憶、多重情報入力と単一情報出力、情報の並列処理、自己組織化などの機能が期待されている。ここでは特に人工物による感覚神経代替器官としての人工神経について述べる。. 人工聴覚器官蝸牛(cochlea)はその名のとおりかたつむりの殻のような器官であり、内部は長軸に沿って二つの小室に分けられ、その中には液が満たされている。音による鼓膜の振動は蝸牛内の液体に伝えられ、それが基底膜に沿って並んでいる有毛細胞と呼ばれる感覚細胞を振動させることにより、音は中枢に興奮電位として伝えられる。感覚神経性の難聴においては、蝸牛の感覚神経細胞が破壊されている。そこで内耳内に人工聴覚器官を移植することにより、聴覚神経を直接電気刺激して重度の難聴を治癒することが行われてきている。最近では正常な耳で行われている音情報の符号化メカニズムをまねるような装置が開発されてきており、発声の音を六つの周波数帯域に分け、そのパワーに応じて複数の電極により聴覚神経を刺激することが行われている。. 網膜チップ感覚神経細胞とそのネットワーク構成をまねることにより、人工的な感覚受容器を作ろうという研究が1990年代初頭より行われてきた。とりわけ網膜における視覚演算に関しては、人の感覚入力のかなりの部分が視覚入力によるものであることと、その階層的な神経系構成が脳と似ていることから広く調べられてきている。特に最近ではその超並列性と特徴抽出メカニズムが注目されてきている。網膜(retina)には視細胞、水平細胞、双極細胞、アマクリン細胞、神経節細胞が層状に配列している。網膜上に外界の像が結ばれると、視細胞にはその明暗に応じた電流が発生し、水平細胞を介して網膜内に広がり、入力情報を空間的にならす。一方、双極細胞はこの空間的にならされた入力と個々の視細胞の応答の差分を作り、これにより網膜に結ばれた像のエッジに関する情報を形成する。またアマクリン細胞は動きの検出に関係しているものと考えられている。このように視覚情報の入口においてでさえも、画像は並列に複数の演算を施されることになる。網膜が通常のテレビカメラと比較して優れている点は、このような演算が入力に適応するように変更されることにより、ダイナミックレンジが広く、また雑音を抑えることに成功している点である。これらの網膜神経系をCMOS 集積回路で模倣する試みが行われている。この試みでは、視細胞はフォトトランジスタ、水平細胞はMOS トランジスタのチャネル抵抗、双極細胞は差動増幅回路で表現されている。このシリコーン網膜では、心理学で知られている錯視現象と同じ現象が見られる。また最近では、アマクリン細胞の機能について電気生理学的に調べたところ、アマクリン細胞が光強度変化にとりわけ敏感であることがわかった。このことは従来知られていたON 応答とOFF応答以外にも、全波整流という非線形演算をこの細胞が行っていることを示しており、従来の網膜モデルにこのアマクリン細胞の機能を付加したシリコーン網膜が開発されている。d. 中枢神経系での人工神経研究中枢神経系に対する人工神経からのアプローチとしては、人工視覚(artificial vision)と聴覚の研究が行われており、これは脳に刺激電極を埋め込むことにより、人工的な視覚や聴覚を実現させるというものである。視覚では、テレビカメラから得られた画像に適当な画像処理を施して、一次視覚野への電気的な刺激入力として加えることにより視力の回復を図るという研究が最近になって行われてきている。外界からの視覚情報は眼により受容されるが、像を形として認知するのは脳である。そのため事故などで盲目になってしまった人にビデオカメラを携帯させ、得られた画像にエッジ抽出などの処理を施し、脳の一次視覚野に対応する部位に埋め込んだ電極アレイにより電気刺激を加えると、その盲目の人はある程度身のまわりの様子を認知できるものと考えられる。このアイデアは実際に盲目のボランティアに対してこの装置を用いることにより確かめられ、外界にあるものの形のあらましを認知できるということが実証されてきている。また同様なアプローチが聴覚に対しても試みられていて、先に述べた人工聴覚器官の移植が難しいような難聴者に対し、音を知覚させることが行われている。蝸牛神経核では音の周波数に応じて神経核内の興奮部位が決まっていることから、電極アレイを蝸牛神経核表面に配置し、音を周波数成分に分解して、そのパワーに対応した電気刺激を加えることにより音を聴くことが可能になった。またこのような脳特定部位の刺激のために、二次元アレイ電極やシリコーンの微細加工技術を利用した剣山型の三次元刺激用の電極アレイが開発されている。
