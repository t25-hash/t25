# 5・5・1 リハビリテーション工学の意義と必要性

リハビリテーション工学(rehabilitation engineering)とは、広い意味でリハビリテーション(rehabilitation)を支援する工学といえる。その支援対象であるリハビリテーションを、通常、社会復帰や日常生活を営むうえでの自立を目指した機能回復訓練と考える人が現在の日本では多いが、外国ではやや異なる状況が見られる。よく知られている語源的な説明によれば、re-:쓕再び」と、ラテン語を語源とする形容詞habilis:쓕多くの人と異なる状態になったとき、人として望ましい状態に戻すこと」と、-ation:쓕語源の異なる語幹に付して名詞を作る」という用語の組合せあるいは合成した語とされている。このことから、いったん失った権利・資格・名誉等を回復するという意味でも使われ、日本でのリハビリテーションとはかなり異なるとらえ方がなされていたようである。他方、国、地域、時には時代によってもリハビリテーションの使い方や意味は異なり、必ずしも非健常者の回復・復帰を表すだけの言葉として使われたり、臨床を想定した目的のみで使われていたのではない。リハビリテーションを、担当内容に基づいて分類すれば、表多くの場合、(1)が最初に実施され、他の(2)~(4)はやや遅れて開始されることが多い。また、(4)の場合には、実施期間は長期にわたることが多く、リハビリテーション工学を含むいろいろな分表5・15リハビリテーションの分類と概要(1)医療に関わるリハビリテーション。........通常、最初に行われる。(2)教育に関わるリハビリテーション。........特殊性や障害を考慮して行われることが多い。(3)職業に関わるリハビリテーション。........快復度等、身体の状況に依存する。(4)社会生活に関わるリハビリテーション。..かなり長期にわたって実施されることが多い。...........................8―215野の専門家が協働で取組むことによってより効果的になると思われる。しかし、現実には(1)のリハビリテーションだけで社会に復帰し、正常な生活に戻るケースが決して少なくなく、すべての障害者が(3)および(4)のリハビリテーションやサービスを必要とするとは限らない。また、複雑な事態や状況にある障害者の場合には、当然、(1)のリハビリテーションだけで好転することは困難であり、総合的な観点からの取組みや施策を必要とする。現代社会でのリハビリテーションの主要な対象は障害(impair-ment,disability,handicap,illness)であり、表5・16に示す四つの側面を有することを理解しておく必要がある。例えば、障害者を対象とした場合のリハビリテーション工学では、単に工学的支援による手足の機能回復等、部分的な考え方に限定するのではなく、自立を目指し、日常生活行為(activities リハビリテーション工学の目的リハビリテーション工学の支援対象であるリハビリテーションの目的は、病気や障害等で低下した機能・能力を回復することだけにあるのではない。図5・74はリハビリテーションの目的と概念図であり、失った機能・能力に対して、別の能力あるいは潜在的な能力を引出して訓練することにより代替する新しい機能・能力を獲得することも、リハビリテーションの重要な目的であることを示している。自立した日常生活を送るうえでも、リハビリテーションを繰返すことによって障害等のためにできなくなった作業・運動を別の方法で行えるようにしたり、異なる部分を使って代替する機能を身につけることもある。リハビリテーションの過程でそれぞれ自ら工夫し、時には機器や道具を利用して、自らも気づかなかったような作業や運動が可能になることがしばしば見られる。単独で工夫することも、もちろん、大切であるが、他からのアドバイスや技術的支援も時には利用して合理的な考え方をすることも必要である。この意味からも、臨床現場をある程度知っておくことが重要であり、できれば主要な対象である障害者の特性について理解することが必要である。リハビリテーションの対象が機能回復のための訓練だけでないことは上述のとおりであり、障害者に対して、障害をもつことは、人間らしく生きることを心身の両面から極めて困難にすると考える人が少なくないが、これはあくまでも健常者の考えに沿ったものであり、必ずしも障害者自身の本質を述べたものではない。障害を克服し、再び健常な人と同様に生きることも、もちろん、重要であるが、障害が残ったとしても本人自身が生き甲斐をもって暮らすことは十分可能であり、時間を要するかもしれないが、心身の両面から苦痛を伴うことなく暮らすことは決して難しいことではない。リハビリテーションを必要とする人たちの病態、障害等の状況がいずれも異なるため、実際のリハビリテーション工学が取組む内容は、臨床リハビリテーションを強く意識したものとなっており、全人間的復権を目標としている。そのように質と量の両面で高い要求がなされる現状では、リハビリテーション工学だけで対処できる問題は限られ、電子工学やコンピュータ情報科学等との技術的な面での連携が不可欠となっている。その一方で、リハビリテーション医学(rehabilitation medi-cine)・歯学等に対しては、例えば、咀嚼(そしゃく)訓練や言語療法実施時などで互いに分野を異にする専門家の連携が種々の面で有利性を発揮している。従来であれば、医用工学(medical engineering)との差異を記述して、あえてリハビリテーション工学の独自性を示す考え方がなされていたが、今日では両者の境界が漠然としているため、厳密な違いを示すのは次第に困難となっている。強いていえば、検査・診断・治療用機器、人工臓器、人工関節等のデバイス、各種インストゥルメンツなど、疾患の検査、診断や治療への工学応用が医用工学の目的であるのに対し、リハビリテーション工学の目的は、病気など各種疾患によって生じる、機能・形態および能力の障害、社会的な不利を克服するための支援にある。このことを示す一例として、人口の1割ともいわれている脊椎不安定症(spinal instability)患者を例にして述べる。このような患者では、重度の場合には脊椎固図5・74リハビリテーションの目的と概念図定術などが適用され、固定用インストゥルメンツが利用される。医用工学では、これらインストゥルメンツの機能性や効果を評価対象とするが、リハビリテーション工学では、術後の背筋など筋力強化を含めた運動機能の回復や通常生活への復帰を前提とした訓練が主となる。なお、近年では、医用工学よりもむしろバイオメカニクスのほうがこの分野全般での主役を演じるようになっている。さらなる対象として、褥瘡の予防・防止、起立・歩行機能を代行する車椅子や歩行訓練機器などの利用、就労支援や交通機関利用による移動支援等を含むあらゆる訓練を挙げることができる。社会のインフラストラクチャが整備された都会に限らず、どこの市町村であっても、図5・75に示すように、病気や障害による生活上の不利益を工学技術の応用によって解決するのも、もちろん、リハビリテーション工学の目指すところといえる。b. リハビリテーション工学の新しい傾向と知的化身体に障害を有していても、スポーツを楽しみ、積極的に社会参加を実践する人が少なくない。そのための支援は極めて重要な意味を有し、機器、器具類の高性能化・高機能化や自立生活維持・訓練における高効率化や知的化への対応等に関わる部分では、リハビリテーション工学に求められる役割は極めて重い。その一方で、機能回復においては、高機能化、高性能化、知的化した訓練機器を利用することばかりが必ずしも効果的でない場合も見受けられ、むしろ原始的といわれる道具や用具を使った知的訓練が見直されることが少なくない。例えば、右片麻痺(hemiplegia)で右上肢の機能回復が不十分な患者に、アートセラピーで利用される描画手法を用いて感覚と書字能力の回復を目指す試みがなされ、患者の興味を誘導するとともに無意識のうちに運動機能の回復訓練に結びつけようとの試みもその一つである。機器を用いた自立(independence)か、本来の身体機能を生かした自立かはそれぞれ状況によって異なるが、とにかくできるだけ自らの意志でADLの充実に効率よく結びつくことが目標となる。手関節・手指の障害により手術等の治療を受けた患者に対するリハビリテーションでは、利き手交換練習が考慮され、通常、字を書く訓練(writing
