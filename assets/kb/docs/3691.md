# 5・5・2 リハビリテーションのシステム工学

> **章**: 第5章 バイオメディカルエンジニアリング
> **節**: 5・15 のようになる。

リハビリテーションにおけるシステム工学は、個人の機能障害や機能低下に対応して活動の制限や社会参加についての制約を解消するための方法を提供する。この目的のために、機械工学、電子工学、情報・通信工学、建築・土木工学など個別工学のすべてを活用する。本項では、制御システム、IT(情報技術)、教育、そしてリハビリテーションプログラムという側面からリハビリテーションにおけるシステム工学の特徴を述べる。a. リハビリテーションにおける制御システム。 制御システム(control system)に要求されること運動機能のリハビリテーションでは、身体と力学的な相互作用を行う機器が用いられる。重篤な運動機能障害からの機能回復では、病院内で人手による回復訓練が行われるが、一般外来患者や要介護高齢者などにマシントレーニングを実施して機能回復、介護予防(転倒、骨折の予防など)、自立支援そして介護軽減などを図るという考えが広がってきている。さらには、自立を支援する中で自然な機能回復をねらうという考えが採用されるようになってきている。このようなリハビリテーションで使用される機器は次に挙げるような点に注意して設計される必要がある。(1) 安全性:患者の肉体的、心理的不安に配慮した十分な安全性、信頼性が確保される必要がある。(2) 個人への適合:機器が支援したり、トレーニングのために負荷を与えたりする相手(患者)には個人差があり、障害の程度も異なるために、個人へ適合する機能を有している必要がある。(3) 時間経過への対応:人の機能が働く場合、その時間経過に対8―218生体工学応して機器が動作する必要がある。機器の動作タイミングが、人の動作に対してミスマッチを起こすとリハビリテーションの意味をなくすだけでなく、逆効果となり、二次障害を起こす可能性がある。(4) 回復過程への適応:リハビリテーションにおける回復過程は、機器にとっては支援の内容や訓練内容の変更を意味しており、回復過程に合わせた機器側の適応が求められる。これら四つの条件を満足させるためには、次に述べるような制御システム技術が有効である。(1) 対象者の運動状態を計測するセンシング技術(2) 運動の指示、支援、負荷を力学的に提示するアクチュエータ技術(3) 訓練計画および対象とする個人の回復過程や身体機能のデータを記憶し、必要に応じて取出せる技術(4) 対象者の訓練や機器の支援の効果を評価する技術(5) (1)~(4)までの技術を統合して身体動作タイミングや訓練過程、回復過程に対応して機器を制御する技術これらは高度な適応制御機器や自律的なロボットの制御技術と共通であることがわかる。次にいくつかの例を挙げる。. 制御システムの具体例身体動作のタイミングに合わせて支援する例として、図5・76に示す電動歩行器がある。歩行器のハンドル部に使用者までの距離を測る光学式のセンサが取付けられており、この距離がつねに一定になるように歩行器の電動モータが制御されている。人の動作タイミングに合わせて移動し、坂道でハンドルから手を離しても勝手に動いてしまわないので、ブレーキ操作に気を遣う必要がないなどの利点がある。回復過程に合わせて、訓練負荷を調整できる例として、自転車や車椅子で実用化されているパワーアシストを応用したものがある。身体運動の評価は身体の複雑さから難しい問題であるが、移動機械のエネルギー消費を計測評価することは可能である。制御技術によってパワーアシスト移動機械の仕事率を設定できるようにし、さらにその機械を駆動している人から供給されたエネルギーを計測評価することによって、リハビリテーションプログラムに沿ったエネルギー負荷を対象者に与えることのできる装置がある。本格的にロボット技術を適用した例として図5・77に示す下肢機能回復支援システムがある。この装置では、トレッドミル上に4本のロボットアームが配置されており、関節角、関節トルク、床半力、筋電位の測定が可能になっている。運動状態の評価のほか、歩行訓練パターンの提示機能や運動指示と訓練結果を提示して、バイオフィードバックができるように設計されている。b. リハビリテーションとIT利用。 IT技術(information technology)の必要性人の生活の中で周辺の人や社会とのコミュニケーションは大切であり、そのためには感覚機能と発話や手による機器の操作のための身体運動が正常に機能する必要がある。IT 技術は、現代社会のインフラストラクチャの一つであるとともに、コミュニケーションについての障害や機能低下を補償する有力な技術である。しかしながら、障害者や高齢者にとっては、それを利用するための教育を受ける機会が制限されていることが多い。そこでリハビリテーションにおけるIT 技術に求められることとして、次の条件が挙げられる。(1) IT 技術の利用に当たり、できるだけ学習や訓練を前提としないこと。(2) 自然に機能回復が進むように、利用できる対象者のコミュニケーション機能を活用するものであること。(3) 家族との交流や社会参加を容易にする点に配慮されていること。次に健康管理や生活支援に用いられているIT 技術の例を示す。. リハビリテーションと教育リハビリテーション工学は単に機械工学技術の応用としてとらえるのではなく、リハビリテーション支援という目的に応じて、電子情報工学など他分野の技術も積極的に導入する必要がある学際的な工学分野である。そのため、単なる機械工学や工学的な知識だけでななく、その支援の対象となるヒトそのものに対する生物機能的知識や医学的知識、患者の心理的側面や人権問題、ならびに医療保険などの社会制度的な側面の理解など、文理を超えた幅広い知識を必要とする。そのため、その教育プログラムは従来の工学技術教育にない新しい体系を必要としている。近年、工学系の大学学部でも福祉工学科などのように医療福祉関係の学問体系を教育研究する機関が多く見られるようになった。これらのカリキュラムを見ると母体となった学科の影響を強く受けている場合が多く、大学によって機械系が強いところや生化学系が強いところなどさまざまで、必ずしも標準と呼べるような教育プログラムは確立されてはいない。ここでは、工学部福祉工学系の学科の専門科目として取上げられている科目名の一つの例を表5・17に示す。リハビリテーション工学に必要な教育プログラムの体系化については関連学会や大学教育関係者などが協力し合い、その確立を進める時期に来ている。前述のように、リハビリテーション工学は目的指向であるので、機械工学系技術者においても従来の機械工学にとらわれない幅広い知識の習得が必要になってくる。さらに、そのような知識の習得だけではなく、医療施設の見学など現場のニーズをとらえる努力や、医療関係者との協力関係を築くことも重要である。医療関係者とエンジニアとはものの見方や考え方が異なる場合が多く、また、互いの専門用語が異なる場合があるため、意思の疎通が難しいことが少なくない。このような留意点の体得は工学系の学生を対象とした座学による講義だけでできるものではない。医療関係者と何らかの連携を行う機会を作ることが工学系の教育のうえでも重要である。また、身体の生理学、医学的特性に関する情報は医療生物学関係の専門書などを参考にすることになるが、これら情報は定性的な記述が多く、数式や数値のように定量的な情報を提供していることは8―219表5・17福祉工学系の学科専門科目の例生物・医学・心理学系基礎生物学、医学概論、リハビリテーション医学、心理学生体医工学系医用生体工学生体計測系生体計測装置学生体材料・力学系バイオマテリアル、バイオメカニクス、医用流体力学生体情報処理系認知・感性工学、生体信号処理、バイオサイバネティクス生活支援工学系ユニバーサルデザイン、リハビリテーション工学、人間工学医療・福祉機器系補装具工学、生体機能代行装置学表5・18大腿骨頸部骨折の理学療法プログラム(手術症の例)術前評価、筋力増強訓練、関節可動域訓練術後1日ギャッチ座位30°開始、徐々にアップ1~2週端座位、車椅子移動2週~平行棒内立位、歩行訓練3週~部分荷重開始、歩行器歩行開始5~6週T リハビリテーションプログラムリハビリテーションプログラムは患者の身体的特性や疾患の程度などさまざまな要因によって決定される。また、医師や理学療法士、作業療法士などの経験的な知識に基づいて決定される場合が多く、標準的プログラムを定めるのは難しい。しかしながら、医療・リハビリテーションの質と効率を向上させるため、クリニカルパス(clinical path way)と呼ばれる管理技法が近年注目されつつある。このクリニカルパスとは製造業におけるプロジェクト管理の理論を医療に応用したものであり、特定の疾患や手術・検査ごとに治療のルーチンワークをチャート様式にまとめ、医師、看護師、コメディカル、患者が治療経過の情報を共有し、必要なケアを適時に患者に提供する管理法である。実際のリハビリテーションプログラムの例として、表5・18に大腿骨頸部骨折の場合の理学療法プログラムを示す。骨折患者に対しては以下のような留意点に基づいて理学療法を進めている。(1) 筋力増強訓練(strength training):手術前は患側下肢の安静が保てる範囲で膝伸展筋、足底背屈筋などの筋力訓練を行う。器具としてはゴムチューブやおもりなどを用いる。手術後早期の段階では患者の状態に応じて、仰臥位や側臥位などの姿勢をとり、同様の筋力増強訓練を行う。(2) 関節可動域訓練(range of motion training):手術前には短縮しやすいアキレス腱や健側のハムストリングスについて可及的早期からストレッチを行う。手術後の関節可動域訓練は痛みのない範囲でactive-assistiveに開始する。手術後の関節可動域訓練は離床プログラムに沿って端座位許可までに股関節の屈曲角度90°を目標に行う。(3) 早期離床:高齢者においては臥床による弊害が容易に出現するため、可及的早期に離床プログラムを進める。車椅子への移乗が許可される時期にはベッドの高さなどの環境の整備を行い、転倒予防に努める。また、股関節の脱臼肢位になりやすい靴や靴下の脱ぎ履き、トイレの後始末などの動作に注意を払う。(4) 歩行訓練(gait training):車椅子への移乗が許可されると同時期に患部側を免荷したうえで立位・歩行訓練を進める。免荷の程度や患者のバランス能力に応じて、平行棒、歩行器、杖などの8―220生体工学歩行補助具を用いる。免荷歩行では肩関節痛や健側膝関節痛などを併発する場合があるので、これらの兆候に注意する。このような実際のリハビリテーションプログラムは医師の指導のもと、理学療法士だけでなく、看護師、作業療法士などさまざまなコメディカルスタッフが互いに協力して進められる。そこで、医療専門家の指示をより的確に患者に伝えたり、リハビリテーション過程を支援したりするために、以下のようなさまざまな形態の工学技術が有効になってくる。(1) リハビリテーション効果の定量的評価技術リハビリテーションにおける療法の効果は急激な変化として現れるものではなく、極めて微小な変化であり、リハビリテーション訓練を継続的に行う必要がある。そのため、患者の多くは自分にリハビリテーションの効果が現れているのか不安になり、リハビリテーションに対する意欲を失い、結果として十分なリハビリテーションが行われない場合も多々ある。そのため、リハビリテーションの過程において、その進ちょく状況を定量的にかつ簡便に評価できるシステムを提供することが重要である。次の5・5・4項で示す計測方法や計測機器、運動分析手法はこれらの例である。さらには身体構造系の数学モデルを用いた生体内力推定手法も提案されている。ただし、実際のリハビリテーションの現場では高度な計測機器や計測技術は必ずしも多くは利用されてはおらず、目視検査や徒手検査のような主観的な評価、あるいは分度器のようなごく簡単な計測器を用いた評価が行われているのが現状である。それは高度な計測機器は一般にコストが高く、また患者や医療関係者にとって必ずしも簡便なシステムになっていない点が一因と考えられる。リハビリテーション支援技術の場合、単なる計測精度などの工学的要因だけでなく、操作者や患者へのユーザインタフェースもより重要になってくる。(2) 訓練支援技術理学療法士などが行うリハビリテーション処方そのものを支援、あるいは代行する工学技術も重要である。例えば、身体の不自由な患者のリハビリテーションを行うためには、医療補助者は患者の身体を支持し、その動きを補助するため、その身体的負担は大きいものがある。これらを低減するためにはパワーアシスト的なロボット工学技術が有用であろう。これまでにも前述のような歩行の運動訓練装置などの開発が行われており、一部、実用化の段階になっている。(3)
