情報技術とユニバーサルデザイン近年,パソコンやインタネットをはじめとする情報技術が生活のあらゆる場面に浸透し,インタネットを使って多くの企業や自治体が情報を発信している.しかし,現在,物理的制約や身体的制約などからすべての人が利用できる状況にはなっていない.ここ数年,パソコンやインタネットのユーザインタフェースとしてGUI(graphical user interface)が一般的になってきた.その結果,情報技術はパソコンに慣れていない人にも親しみやすくなった.しかしその一方で,GUIは視覚障害者にとって,使いやすいインタフェースとは決していえない.高齢者には,やはりパソコンは難しく使いにくいと考えられている.情報のユニバーサルデザインとは,身障者・高齢者を含むできるだけ多くの人にとっても,パソコンやインタネットをより使いやすくすることである.インタネットでの情報発信において,できるだけ多くの人にとってインターネットの情報にアクセスできるデザインが必要となる.ウェブのアクセシビリティに配慮することが重要である.このようなウェブサイトを,アクセシブルウェブ(accessible web)と呼んでいる.ウェブのアクセシビリティについては,インタネットの国際的な標準団体であるWWW コンソーシアム(W3C)の中の組織であるWAI (Web Accessibility Initia-tive)が検討している.これをもとに企業や公共機関ではウェブのアクセシビリティに関するツールや機器を開発,提供している.現在わが国のインタネット利用者は20~30代が中心だが,確実に利用者の年齢は上昇している.特に,高齢者予備軍の40代後半のユーザの利用率が上がっており,2010年の超高齢社会では高齢者ユーザが一般的となる.郵政省(現総務省郵政事業庁)が2000年に発表した「21世紀の情報通信ビジョン―IT JAPAN for ALL―」には,緊急に解決すべき課題の一つとして情報格差(digi-tal divide)が指摘されている.そこでは情報バリアフリー社会の実現手段として,ウェブのアクセシビリティや情報のユニバーサルデザイン化の推進の必要性が挙げられている.また同じく郵政省から,「高齢者・障害者による情報通信の利用に対する人的支援及びウェブアクセシビリティの確保に向けた課題と方策」が発表された.この中では,WAIのガイドラインに基づいたユニバーサルデザインの視点から,高齢者・障害者のアクセスを支援する情報通信システムを制作し,具体的な問題点を抽出・改善する仕組みを作ることが必要とされている.こうした中で一部の地方公共団体などでは,ウェブのアクセシビリティの重要性に気づき,独自の取組みを始めている.情報バリアフリー(information barrier free)環境の整備の在り方に関する研究会からの報告書には,課題や指針などが提起されている.ウェブのアクセシビリティを保証するうえでは,障害者や高齢者などに対する理解と,インタネットの特性を合わせた配慮が必要である.インタネット利用上での身障者への配慮には,次の点がある.8―227(1) 視覚障害:GUIなどの視覚的情報への代替情報の提供(2)
