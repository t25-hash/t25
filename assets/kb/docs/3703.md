e.複雑形状加工 ダイヤモンド工具を用いた特徴の一つに，
ここでは，最も一般的な単結晶ダイヤモンド工具を用いた，軟質
金属の超精密切削について概説する．
a.単結晶ダイヤモンド工具（diamond tool ) よく知られて
いるように，単結晶のダイヤモンドは異方性に富んだ高硬度の物質
である．このため，工具寿命の長い耐摩耗性に優れた面方位を選択
することが望ましい が，最適面方位を選択したとき，刃付け研
磨することが極めて困難で，現状では上述のような経済性を理由
に，妥協した面方位を採っていることが多い．しかし，広い面の切
削が必要なオーディオ・ビジュアル（AV）機器の金型などでは，
長寿命工具の方位選択が必須である．

(１) バリの抑制できる切削条件の選択：バリの発生は，被削材
の伸びや硬さとの相関が強いことから，伸びが小さく硬い被削材を

用できず，同じ切削により除去している例が多い．
なお，このような急峻な傾き形状の高精度測定も，加工精度向上
の障害になっている．
