# 12・2・3 超精密研磨加工（ultraprecision polishing）

> **章**: 第12章 超精密・超微細加工技術
> **節**: 12・2 超精密・超微細加工技術各論

研磨加工は，光学部品の製作を支え，19世紀末には波動光学を
学問としてほぼ完成させるうえで大きく貢献した．20世紀後半に
なると加工技術として高度化がいっそう進み，レーザの発振や量子
光学の発展に寄与し，赤外光や可視光はもとより，紫外光，遠紫外
光，X線と短波長寄りの光学部品の要求にも応えてきた．また，
適用範囲は光学部品だけでなく，半導体デバイスや水晶発振子など
新機能をもつ部品製作にも広がった．最先端の加工技術に超精密加
工という名称が付いたのは1960年代からであり，研磨加工は，シ
リコンウェーハのメカノケミカルポリシングの加工原理の解明が発
端になり，加工技術改善や新加工原理導入に支えられて超精密化が
進んだ．最近では，高度の部品を製作するといったニーズに応える
べく，スーパースムーズ鏡面仕上げが可能な，実用的な加工技術・
生産技術になってきている．
a.超精密研磨で仕上げたスーパースムーズ鏡面の定義 スー
パースムーズ鏡面は，さまざまなレベルの加工面の中で最も高品質
であり，超平滑な面である．加工物が単結晶の場合，その全面がシ
リコンのへき開面のSTM（scanning tunneling micro-scope）像
のように，球形の原子が規則正しく配列した微小凹凸波形のプロフ
ァイルをもつ表面になる．多数個の原子配列でとらえるならば，階
段状に並んだテラスの表面状態も想像できる．触針式の表面粗さ測
図12･ 9超精密研削した光学ガラスBK7の表面粗さ
定機によれば，表面の微小凹凸は原子の大きさあるいはその数分の
1といった微小高さ，表面粗さで0.1nm 台になる．多結晶材
料や非晶質材料を研磨した場合もこれに準じる表面の微小凹凸と考
える ．
実際の研磨加工では，原子・分子の大きさから塊状のものまで幅
広い規模の材料除去が営まれる．砥粒や砥粒を埋め込んだポリシャ
の凸部の運動軌跡に沿ってこの材料除去が進めば，表面構造は田畑
のような連続した畝や回折格子のような方向性をもつ凹凸，全方向
からの運動軌跡の交差を考えに入れるとピラミッドや円すいが並ん
だような凹凸になる．表面粗さは切りくず生成の痕跡であり，その
面には，材料除去の際の原子配列の乱れやひずみといったいわゆる
加工変質層が残され，鏡面に仕上がっていてもそれは表面の数原子
層から数百原子層にまで及ぶものがある．スーパースムーズ鏡面
は，このような加工変質層の存在も皆無である．
b.従来の研磨加工の高度化・超精密化 研磨加工の超精密化
では，表面粗さや表面の原子配列の乱れに影響する機械的作用を小
さくしなければならない．一方，そうなると切りくずの生成規模が
小さく，加工量が大きくとれない．シリコンウェーハの鏡面研磨で
は，加工条件を3～4段階に分け，最終段階で超精密研磨条件を適
用している．光学レンズの製作でも，ラップした粗面の鏡面化と鏡
面の高度化や形状精度修正の2段階に分けた工程がとられてきた．
前加工であらかじめ表面粗さや加工変質層深さの微小化，表面汚染
の除去などを済ませ，超精密研磨によってスーパースムーズ鏡面を
得るのが普通である．
機械的作用の微小化では，まず，微細砥粒やきめの細かなポリシ
ャを用いる．研磨剤中の大きな粒子や凝集砥粒の混入，ポリシャ面
のじんあいなどの付着は，加工面にさまざまな種類のへこみを作
り，品質劣化を招く．研磨装置の設置環境，研磨装置の機械振動，
ポリシャ面の振れは，機械的作用を強調するように働き，加工欠陥
の発生を高める原因になる．
砥粒が研磨加工の際にポリシャ側と加工面側の双方に押込まれる
と仮定すれば，表面粗さは，砥粒の引っかきの交差で形成され，
＝4で表わせる． は砥粒の粒径， と は
それぞれポリシャと加工物の硬さに相当する．また，砥粒が非常に
微細になり，砥粒の大きさよりもポリシャの凹凸のほうが無視でき
なくなると，加工面がポリシャの凹凸に押付けられて発生する微小
圧力偏差のもとで擦られ，＝ζで表わせる． はポリシ
ャ面の凹凸の高さ，ζはポリシャの弾性変形定数に相当する．微細
な砥粒を選択した超精密研磨になると，凹凸のないポリシャあるい
は仮に凹凸があってもそれによって圧力偏差が生じないような微小
領域で軟質な性質をもつポリシャが必要になる ．
大粒の粒子が研磨剤に混入すると研磨加工が周期運動のもとで進
むので，無数の引っかき痕で形成される．ポリシングにおいて化学
作用の存在が無視できない場合でも，砥粒などの機械的作用が表面
粗さ形成の主要因であればこのような説明でも矛盾がない．
粗面から鏡面を仕上げていく過程では，軟質なポリシャ面に当初
から凹凸がある．しかし，順次平滑に変化していくものであれば良
好な鏡面に仕上がり，その代表的なものにピッチポリシャがある．
ピッチポリシャは加工物の凹凸の状態に従って変化しており，ニュ
ートンの時代から現代まで長年ポリシャの王座にあった理由でもあ
る．
c .ポリシングのメカニズムの分析と超精密研磨加工 ポリシ
ングにおける鏡面化のメカニズムに関しては，(１)機械的作用と，
(２)(電気）化学的作用によって進み，機械的作用には，(ａ)引っ
かいて削り取る除去作用と，(ｂ)原子の配列を乱す摩擦作用があ
り，（電気）化学的作用には，(ｃ)溶去作用と，(ｄ)被膜形成作用
がある ．
シリコンウェーハの湿式メカノケミカルポリシング（wet -type
mechanochemical polishing）では，0.01m台のSiO砥粒を弱
アルカリ性の溶液にコロイド状に分散させた研磨剤と，軟質発泡ポ
リウレタンのシート状ポリシャを用いる．研磨運動でウェーハとポ
リシャの間にハイドロプレーン現象によって研磨剤層を形成し，一
方ではその研磨剤層を薄くするように研磨圧力を付加しつつ擦り合
わせる．その際，化学作用による軟質な水和膜の形成とポリシャや
研磨剤による水和膜の機械的な除去の双方の作用が存在する ．
このような被膜形成とその除去が組合わさった鏡面研磨のメカニズ
ムは，GaAsなどの化合物半導体，多成分ガラス ，金属材料
などの鏡面研磨でも相通じるところがある．凹凸をもつ加工物の凸
部では，被膜除去と被膜形成が積極的に繰返されて平滑化が進み，
被膜は水和膜に限ることなく，さまざまな被膜でも，表皮材料の変
質さえも有効と考えられる．
d.超精密研磨加工例 超精密研磨のためのピッチポリシャの
準備は，じんあいを取込みやすい網目を押付ける従来法に代え，下
地の金属プレートにあらかじめ4mmピッチで碁盤目状に幅1
mm，深さ1mmの溝加工を行い，その面を平面研磨し，加熱して
溶融状態のピッチを均一に塗り付けてポリシャにする．微細砥粒に
は，7nm-SiO砥粒を用いて超精密研磨加工を行うと，BK7ガラ
スをはじめ結晶の蛍石も0.5nm 以下に仕上がり，従来の酸化
セリウム砥粒による光学ポリシングの5nm に比べ大幅に向上
する．
酸化セリウム砥粒など市販の砥粒の中の微細砥粒だけを用いる液
中研磨加工も超精密研磨の一つである．加工物とポリシャを容器の
中で研磨剤に浸した状態で擦り合わせる方法を採り，細かい浮遊し
た砥粒で研磨する条件を用いる．溶液も十分存在するので冷却効果
や緩衝効果もあり，高品質の表面が得られる ．
軟質なポリシャ材料で極端なものは液体や気体が該当するという
見方がある．EEM（elastic emission machining) ，フロートポ
リシング（float polishing) などは液体をポリシャのような工具
と見立てた研磨法といえる．研磨剤の流れの中で，比重の違う微細
去を行い，化学的溶去と同様の原子・分子のオーダの加工単位と考

ルポリシングは，両者の間に閉じこめられた研磨剤層があり，かつ
