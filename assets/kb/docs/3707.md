# 13・3・1 遠隔診断・保守

> **章**: 第13章 遠隔加工システム
> **節**: 13・3 遠隔加工システム事例

遠隔診断システムと遠隔保守システムとを実装した例について紹
介する ．遠隔診断システムでは遠隔に存在する対象機械の
CNC，PLC（programmable logic controller ）情報を電話回線を
使用して指示者側に伝送する．対象機械側に中継用コンピュータを
用意し，このコンピュータと指示者側のコンピュータとを電話回線
を用いて接続する．インタネット（internet ）に接続された対象機
械のCNC画面が，これらのコンピュータを介してイーサネット経
由で指示者側の手元のコンピュータに伝送される．指示者側は伝送
されるCNCの画面を見ながら，対象機械の故障診断および故障原
因の特定を行う．必要であれば手元のコンピュータにおいて，
CNCに対する操作を対象機械の目の前にいる場合と同様に行うと
ともに，電話回線を用いて遠隔地の作業者に作業指導を行うことに
より，故障の原因を取り除き，正常状態への復帰を可能とする作業
支援システムである．
遠隔保守システムでは，遠隔地の作業者に指示を出すための映
像・音声を送受信する．作業者は頭部に小型ビデオカメラおよびヘ
ッドセットを装着し，一連の保守作業を行う．作業者頭部に取付け
られたビデオカメラによって獲得した映像，およびマイクによって
獲得した作業者の音声・機械側の環境音を，まず遠隔地にある中継
局に無線機を介して送信する．中継局に送信された映像・音声は
ISDN回線を用いて遠隔地の指示者側に送信される．指示者側に送
信された映像・音声をもとに指示者が作業指示を与えたり，設備の
状況確認を行ったりするが，作業が必要な場合には指示者がマイク
により作業指示を与え，同様に回線を介して作業者のヘッドホンに
送信される．
これらのシステムにより，専門家がすぐに遠隔地の設備モニタリ
ングをすることができ，現場の作業者の習熟度によらず故障原因の
特定のための手順を踏むことが可能となる．また，映像と音声との
両方により遠隔の指示者と現場の作業者との共通認識の確立が可能
となる．ただし，これらのシステムであっても，作業者の頭部に装
着したカメラの動きによる指示者の疲れの問題やカメラ視野の問題
など解決すべき課題はある．
遠隔モニタリングの機能を用い，365日24時間体制で遠隔メン
テナンスサービスを行っている例もある．ここでは，顧客情報，納
入機械情報，機械修理履歴などをデータベースとして蓄積し，問題
解決まで最短ルートで対応可能となっている．
