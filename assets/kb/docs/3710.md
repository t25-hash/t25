# 1・1・2 人類初期のエネルギー変換

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・1 機械工学におけるエネルギー変換とは

人類は古代より長く奴隷や牛馬などの力学エネルギーを利用して（BC100年ころ）やスペイン風車（700年ころ）の利用が始まっ（BC201年）はまさにソーラエネルギー利用の始めといえよう．
ま作ったのが今までの熱から力学へのエネルギー変換の主力である熱それ以後の中世にはあまり著名な機械工学的変換の発明はないに広がり，リニア熱機関の原形としての大砲やその製作用金属工作が伝来したが，数年のうちに堺，国友等で急速に大量生産化されしかし17世紀から18世紀にかけての自然科学や，起電機等を含る産業革命の下地を作った．
いたが，文明の進歩とともに帆船（BC2000年ころ）や簡単な水車た．
またアルキメデスが多数の反射鏡で敵船を焼いた故事た2世紀初めころにギリシャのヘロンが簡単な反動蒸気タービンを機関の創始といえよう．
が，11世紀ころに中国で発明された黒色火薬は，まずヨーロッパ技術の急速な発展に貢献した．
わが国には1543年に種子島に鉄砲て，その生産性と技術は当時の世界に例を見ない．
