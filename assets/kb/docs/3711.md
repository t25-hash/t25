# 1・1・3 熱機関の繁栄

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・1 機械工学におけるエネルギー変換とは

機械工学的エネルギー変換の発展と栄光は18世紀から20世紀後
期までの熱機関の目覚ましい発明と進歩にあるといえよう．
まず，パパンが蒸気で動くピストン機関（piston engine）を
1706年に考案したのをはじめとして，次いでニューコメンが1742
年に揚水機関を実用化した．しかしこれは極めて効率が低かった．
実用的な蒸気機関は1789年のJ.ワットによる開発と完成で，この
使用は鉱山ほか諸産業に広がり産業革命をもたらした．またトレビ
シックに始まり（1804)，G.スティーブンソンによって実用化
（1825）された蒸気機関車が産業革命をいっそう拡張した．
この前後にモンゴルフィエ兄弟が1783年に熱気球を上げ，熱エ
ネルギーをかなり高い位置の力学エネルギーに変換することに成功
し，現在の宇宙ロケットも同じエネルギー変換線上にある．
19世紀には主として石炭を熱源とする蒸気機関の全盛期であっ
たが，その初頭にはボイラの爆発を避け，かつ任意の熱源で動く熱
空気スターリングエンジンが発明されている（1816）のも興味深
い．また19世紀後半では電気機器の進歩により電気エネルギーへ
の変換も重視されだした．
また19世紀末期の最も重要なエネルギー変換装置はルノアール
（1859)，オットー（1867)，ダイムラー（1889）らが実用に成功し
た内燃機関であり，自動車への採用が交通革命をもたらした．
また一方で1884年パーソンズらによって蒸気タービン（steam
turbine）が発明され工業的にも成功して急速な発展をなし，蒸気
タービンは発電・船舶に広く使用され始めた．また1897年にはデ
ィーゼル機関（diesel engine）が発明された．ディーゼル機関が加
わったことで内燃機関は多数の機械技術者の力により20世紀のエ
ンジンとなった．またガスタービンも多く試みられ，その基礎は
19世紀末にはほぼ完成された．日本の第1号ガスタービンは戦後
の「運研1号」(1949）である．
