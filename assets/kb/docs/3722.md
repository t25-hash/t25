# 1・2・5 エネルギー変換（energy conversion）の目的別分類

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

人間社会の立場からエネルギーを分類すると，自然に存在する一
次エネルギー（primary energy）と，人間がエネルギー変換をし
て初めて入手できる二次エネルギー（secondary energy）とがあ
る．前者には化石燃料や太陽・水力・風力エネルギーなど，後者と
しては電気エネルギーや水素などがある．
エネルギー変換は，目的別に分類すると4種類ある ．まず，化
石燃料のようなエネルギー資源中に含まれている化学エネルギーを
燃焼させて熱エネルギーを発生する場合，資源側変換という．次
に，エネルギー源の獲得とエネルギーの消費とは，空間的にも時間
的にも異なる点で行われることが多いので，輸送しやすさを目的と
する輸送変換や貯蔵しやすさを目的とする貯蔵変換が必要になる．
輸送変換の代表例は，一次エネルギーから電気エネルギーおよびガ
スへの変換であり，この結果，それぞれ電線・ガス配管網により容
易に輸送されることはいうまでもない．また，深夜に供給過剰とな
る電力で氷を作り，その冷熱を昼間の冷房に用いるのは，貯蔵変換
である．一方，火力発電所で化石燃料の燃焼熱を用いて，ガスター
ビンや蒸気タービンを回し，さらに発電機を回すことは，熱エネル
ギーを使いやすい電気エネルギーとして利用するための利用変換で
ある．
