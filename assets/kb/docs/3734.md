# 1・1・1 調木，チップ（chip）ハンドリング機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

a.バーカ（barker ) 樹皮はパルプ分が少なく，パルプ化工
程での歩留りの低下につながり，またパルプ中のちりの原因となる
ので除去する必要がある．バーカの選択は対象とする材のサイズ
（径）によって決まる．
.ドラムバーカ（drum barker ) やや傾斜して回転する
ドラムの高いほうから丸太が投入され，ドラムの回転による丸太と
ドラムあるいは丸太どうしの摩擦と衝撃により皮はぎが行われ，他
方から排出される．タンブリング型とパラレル型があり，パラレル
型は長尺の丸太を対象とする．
.リングバーカ（ring barker ) 大きなリングの内側にス
クレーパを中心に向けて取付け，それを回転させ，中心に送り込ま
れる丸太表面に直接スクレーパを押付け皮をはぎ取る．丸太を1本
ごとに処理するため，大径木ほど効率が高い．
b.チッパ（chipper ) チップ化の目的は，パルプ化工程での
均一化であり，チップの厚さは4～10mm，幅は25mm前後，繊
維方向の長さは15～35mmを標準としている．400～4450mmφ
のディスクをもったディスク型チッパが多く使用されている．
2～16枚のナイフが取付けられた回転ディスクに30～60度の角度
で材が供給される．ナイフの数が多いほど処理量が増加し，チップ
サイズも均一化する．チップはディスク型チップスクリーン等で処
理され，粗大チップはチップスライサやチップクラッシャにかけ再
びチップスクリーンで選別する．
