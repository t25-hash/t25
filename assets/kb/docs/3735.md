# 1・1・2 機械パルプ（mechanical pulp）製造装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

a.SGP（stone groundwood pulp) SGPは高速で回転する
砥石に丸太を押付け，シャワー水をかけながらすりつぶしパルプ化
する，いわゆる砕木パルプである．グラインダ（grinder ）には，
丸太の供給方式，加圧方式の違いにより，図1･1 に示すようなポ
ケット型，マガジン型，チェーン（キャタピラ）型，リング型等
種々の型式のものがある．パルプストーンとしてアランダム，カー
ボランダム製のセラミックストーンが使用されている．摩砕条件に
よって製造されるパルプの品質は異なるが，一般的に強度は低いが
不透明度が高く，吸油性に優れ，新聞用紙，中級印刷用紙，下級印
刷用紙などに使用される．
b.PGW（pressurized groundwood) PGWはSGPを改良
したパルプ化法で，高温のシャワー水を使用し加圧下で丸太を摩砕
するパルプ化法である．密閉構造になっているグラインダチャンバ
の加圧は圧縮空気で行われ，摩砕条件は圧力2～3気圧，温度
110～135℃である．図1･2 にPGWの製造工程を示す．加圧状態
でかつ高温における処理はリグニンの軟化を促し，SGPより長繊
維のパルプが得られ，SGPより強度が高くなる．不透明度はSGP
とほぼ同等であり，優れた印刷適性を示す．PGWはグラインダを
加圧容器にする必要があり，SGPに比べ設備費が高くなる．原木
として丸太材を必要とするため，丸太材の豊富な欧米諸国にプラン
トの設置例が多く見られる．
c.RGP（refiner groundwood pulp) RGPはチップやソー
ダストをリファイニングによりパルプ化する方法である．リファイ
ニングにはディスクリファイナ（refiner ）が使用され，一次リファ
イナで粗砕を行い，二次リファイナで精砕を行う．RGPは不透明
度がSGPに及ばず，結束繊維が多く強度もそこそこのところか
ら，次に述べるTMPが開発されて以来，新設はほとんどなくなっ
ている．
d.TMP（thermomechanical pulp) TMPはRGPの一次リ
ファイニングを蒸気加熱リファイニングに代替したものである．チ
ップウォッシャで土砂金属類を除去したチップを110～130℃に蒸
気で加熱し軟化させた後，加圧型リファイナで一次リファイニング
し，次いで常圧型リファイナで二次リファイニングを行いパルプ化
する方法である．図1･3 にTMP工程の基本的な系統図を示す．

比較を示す．

リグニンのガラス転移点を超えない温度で加熱した後リファイニン
グするので，SGPやGPと異なり，繊維の破壊，裂けなしに繊維
