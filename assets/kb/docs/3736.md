# 1・1・3 化学パルプ（chemical pulp）製造装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

化学パルプの製造法は，クラフトパルプ化法がわが国ではもちろ
ん，世界的に見てもパルプ製造法の主流になっている．その理由は
次のとおりである．
(１) 廃液からの薬品および熱回収法が完成している．
(２) ほとんどすべての樹種に適用でき，また建築用材の廃材チ
ップでも使用できる．
(３)蒸解時間が短く，操作も簡単で，連続・大量生産に適した
装置の発展があった．
(４)強度の大きなパルプが得られ，製造コストが安い．
a.蒸解装置（digester ) 省エネルギー，省力対策および生産
規模の拡大から，バッチ式蒸解釜に代わり，連続式蒸解釜が主流に
なっている．最近，エネルギー効率が改善されたバッチ式蒸解釜も
設置されている．図1･4 に連続蒸解釜（continuous digester ）の
一例を示す．チップは低圧蒸気で加熱されてから釜の頂部へ蒸解液
とともに送られ釜に入る．釜内では，薬液の浸透，加熱（間接加熱
式熱交換器による)，蒸解，洗浄が行われ，底部から連続的に排出
される．近年，脱リグニンの促進，収率の向上を図るため，蒸解液
を釜の数箇所に分割添加できる方式の蒸解釜が開発されている（図
1･5)．
b.エバポレータ（evaporator ) 蒸解釜から排出される廃液
［黒液（black liquor ）］はフラッシュした後，回収ボイラで燃焼し
うる濃度まで濃縮される．多管式強制循環型に変わってプレート式
の薄膜降下型エバポレータが主流になっている．濃縮濃度は
65～75%程度である（図1･6)．
c.薬品回収設備 濃縮された黒液は，回収ボイラで燃焼され，
スメルトの形になり，弱液を用いてディゾルバで溶解され緑液
（green liquor ）となる．緑液中には，ドレッグと称する不溶性の
不純物があるので，緑液クラリファイヤで沈降分離する．緑液は，
スレーカ，苛性化槽を経て，白液（white liquor ，蒸解液）変換さ
れる．苛性化槽を出た白乳は，白液と石灰スラッジ（炭酸カルシウ
ム）に分離される．クラリファイヤ，プレコートタイプのベルトフ
ィルタ，加圧でディスクフィルタなどで行われる．分離回収された
石灰スラッジは，ウラッジウォッシャ，フラッシュドライヤで処理
された後，キルンで焼成され，生石灰になり，再びスレーカへ供給
される．連続苛性化（recausticizing）工程の一例を図1･7 に示
す．
