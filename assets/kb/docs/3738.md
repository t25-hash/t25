# 1・1・5 精選（screening） ・除じん（cleaning）装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

古紙には，種々の異物が混入している．異物の性状は，大きさ・
形状・比重・表面性状など多様であり，この異物をできる限り細片
化させず除去することが，古紙パルプの品質向上につながる．
a.スクリーン パルパで離解後，できるだけ大きな形状のう
ちに異物を除去する粗選工程と細片化された異物を除去する精選工
程がある．粗選工程のスクリーンに使用されるプレートは，大部分
が丸孔（3～4mmφ程度）である．最近は除去効果を高めるため，
スリットプレートを併用するダブルセパレータが開発されている．
精選工程では，スリットタイプのプレートを用いた密閉型のスクリ
ーンがおもに使用されている．パルプの処理濃度は，使用するプレ
ートの目孔，スリット幅と処理する古紙原料によって決められる
が，粗選工程で3～4%，精選工程で0.8～2 %である．
