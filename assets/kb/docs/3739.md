# 1・1・6 漂白（bleaching）装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

漂白の目的は，パルプに含有される着色成分（主としてリグニ
ン，古紙においては染料も含む）を脱色あるいは除去することによ
って，用途に応じた白色度（brightness ）レベルのパルプを製造す
ることである．機械パルプ，高収率パルプ，古紙脱墨パルプでは，
着色成分を残して脱色することに主眼がおかれるため，過酸化水
素，ハイドロサルファイトによる漂白が一般的である．化学パルプ
の主流であるクラフトパルプの漂白では，着色成分を除去すること
に重点がおかれており，多段漂白法が採用されている．酸素処理
(Ｏ）を最初に行った後，塩素処理(Ｃ)，アルカリ処理(Ｅ)，次亜
塩素酸塩処理(Ｈ)，二酸化塩素処理(Ｄ）を組合わせて行っていた
が，近年の環境問題の高まりにより，塩素処理(Ｃ）は減少の傾向
にある．最近ではオゾン処理(Ｚ）や酵素処理(Ez）といった処理
段も組込まれるようになってきた．各段の処理後，パルプはドラム
フィルタあるいはディフューザによって洗浄される．図1･13 に
最近の化学パルプ製造工程に組込まれた漂白設備フローを示す．
