# 1・2・1 紙料調成装置（stock preparation）

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・2 製紙機械

リファイナの形式はディスク型が一般的であり，図1･18のような刃をもつディスクを対向させ，一方を回転し，その間に原料を流転ディスクの両面に刃を設け二対としたダブルディスクリファイナな紙品質を得るための叩解条件は，原料の種類に応じ，刃幅，溝よび原料濃度，処理量など多数の因子により決定される．
ル型，開放振動型に分けられる．
圧力セントリフューガルスクリー広く使用されているスクリーンである．
処理濃度は普通0.5～2 %濃度型もある．
構造は円筒型スクリーンプレートと回転するフォイセントリフューガル型，外側で回転するものをセントリピータル型のため，処理濃度が上昇するにつれ精選効率は低下する．
調成工程は抄紙機に原料を送る直前の工程で，離解，叩（こう）し，叩解を行う．
固定ディスクと回転ディスクが一対のものと，回がある．
図1･19にダブルディスクリファイナの一例を示す．
最適幅，刃の交差角などのディスク形状，ディスクの周速，押付け圧お一方，小規模の叩解を行う設備としてビータ（beater，図1･20）回転させ，その下方の固定受刃との間に原料を通し叩解を行う．
ンはパルプ，抄紙精選工程および古紙処理精選工程にで，ポンプで強制的に圧入させるが，4～5%程度まで処理可能な高ルを有し，スクリーンプレートの内側でフォイルが回転するものをという．
また，両者を組合わせフォイルの内側と外側にスクリーンプレートを配し，処理能力を増大させたものもある．
