# 1・2・3 仕上機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・2 製紙機械

a.スーパカレンダ（super calender ) 非常に高い光沢と平滑
を出す目的で，図1･52に示すスーパカレンダが用いられる．この
装置では，硬質金属ロールと軟質ロール（コットンロールなど）が
交互に配置され，紙は上方から下方に各ニップを通過する．ニップ
下では軟質ロールが変形して金属ロールと相対運動を起こし，紙と
の摩擦が起こるために表面の平滑が非常に高くなる．
b.カッタ（cutter ) カッタでは，リールで巻取った製品を連
続して縦横寸法どおりに平判状に裁断する．オンマシンとオフマシ
ンの2種類があり，オンマシン用はおもに紙厚の厚い板紙に用いら
れる．図1･53にカッタの一例を示す．巻き解しスタンドを出た紙
はスリッタで幅方向に切れ目が入れられ，回転ナイフによって流れ
方向に裁断される．平判の長さは回転ナイフの速度によって決ま
る．最近では，巻き解し後に欠陥検出器を設置して，欠陥のシート
を自動的に排出するシートソータ機能を付加したカッタも採用され
ている．
c.包装機 平判の包装は図1･54に示すような流れで処理され
る．カッタで裁断され積上げられたシートはコンベヤ自動包装機に
送られて包装される．
