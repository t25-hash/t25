1･2･5項に示す塗工装置を用いて，原紙上に塗工液を塗布する．最

d.乾燥部（dryer ) 前述の塗工部で塗工された紙は，
赤外線式ドライヤ（infrared dryer )，熱風ドライヤ（air dryer )，
あるいは蒸気加熱されたシリンダドライヤ（cylinder dryer ）によ
って乾燥される．これらのうち赤外線式ドライヤは，高速塗工時の
ドライヤ機長短縮と，乾燥工程におけるバインダマイグレーション
（binder migration）を抑制する目的で，乾燥部入口に設置される．
また図1･59に示す熱風ドライヤは，熱風によって紙を浮上させな
がら乾燥できるため，塗工層の水分が比較的多くシリンダドライヤ
に接触させられない領域で使用される．なお欧米地域では，赤外線
と熱風ドライヤのみで乾燥させる場合もある．
e.リール（reel ) 塗工紙を巻取る装置．装置構成は
1･2･2･g項に示した抄紙機のリールとおおむね同じである．しかし
ながら塗工紙の場合，紙の通気性が新聞紙，上質紙に比べ著しく低
いため，高速で塗工紙を巻取る場合には巻取られる紙層間に空気層
が残留しやすく，巻取り中に層間滑りが生じやすくなる傾向があ
る．このため，ドラム表面の溝形状，巻取りとドラムとのニップ
圧，および巻取りに作用させるトルク等の機械および運転条件を，
他の紙種より適正に選定する必要がある．
