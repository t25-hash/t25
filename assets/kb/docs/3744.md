# 1・2・5 塗工装置の種別

a.ロールコータ（roll coater ) ロールコータは，コンパクト
な構成で紙の表裏両面に同時に塗工できること，塗工部での断紙が
生じにくいことなどから，サイズプレスおよびオンマシンコータと
して用いられる．
紙塗工で用いられるロールコータは，2本の圧接したロール間に
形成した塗工液溜り（pond）に原紙を通して塗工する2本ロール
方式（pond press ）と，あらかじめアプリケータロール表面に形成

press ）に大別されるが，近年では塗工量制御性と高速操業安定性
等に優れる下記のフィルムコータが多く採用されている．

b.ブレードコータ（blade coater ) ブレードコータは塗工液
の供給部（applicator )，仕上げブレード部（coater head）とバッ
キングロール（backing roll ）から構成され，アプリケータで最終
的な塗工膜厚より厚めの液膜を原紙表面に転写した後，仕上げブレ
ード（finishing blade）で所定の塗工膜厚に計量するとともに，塗
工面を平滑に仕上げる塗工方式である．塗工膜厚の制御は，バッキ
ングロール上の紙に対する仕上げブレード押付け圧力によって制御
される．仕上げブレードには，おもにSK材が用いられているが，
最近では摩耗寿命の長いセラミック溶射またはクロムメッキ処理を
施したものも使用されている．
このコータは塗工品質に優れることから，近年多く採用されてお
り，アプリケータの形式によって下記のように分類される．

c .エアナイフコータ（air knife coater ) 図1･66に示すエア
ナイフコータは，アプリケータで原紙上に転写した余剰塗工液膜
を，原紙走行方向に対して対向する向きに噴出する二次元高圧空気
噴流によって計量，かき落とす方式のコータである．このコータ
は，アート紙などの高級重塗工紙の生産に多く用いられていたが，
最近ではほとんどブレードコータに置換されている．
d.カーテンコータ（curtain coater ) 図1･67に示すカ
ーテンコータは，アプリケータノズルのスリットから塗工液を膜状
に流下させ，原紙上に直接所定の塗工液膜を形成する方式のコータ
である．このコータは，他のコータに比べ原紙内への塗工液浸透が
少なく高い表面被覆性が得られる，またアプリケータロールもしく
は仕上げブレードのような摩耗部品がないため，部品コストと交換
メンテナンスコストが削減できる，といった特徴がある．
塗工量は，原紙走行速度に応じた流量の塗工液をアプリケータノ
ズルに供給することで制御できることから，前述の各コータに比べ
操業も容易となる．なお最近では，図1･67に示したように，非接
触で原紙同伴空気流の影響を抑制し，安定した高速塗工が可能とな
る装置も開発されている．
