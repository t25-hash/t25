# 1・3・5 編組機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・3 繊維機械

編組物には1本または多数本の糸でループを作り，このループをは浮き編み（フロート），(ｃ)はパイル編み，(ｄ)は挿入編み（イ添え糸編みである．
編組織のうち最も多く製造されるのは編物で1本の糸で編成する横編みと，多数の糸を製織の場合のように整経して斜交編地を編成する縦編みとに大別される．
横編にしても縦編にしても，ループを作るとともにこれを互いに掛かり合わせて布を作るのに使用される編機の編成部は，編針，シンカ，プレッサである．
編針にはひげ針と舌針（べら針）および複合針の3種類がある．
存在するが，代表的機種によって分類すれば表1･5に示すように大接，衣類が形成される機械も開発されている．
ンレイ），(ｅ)はペレリン編み，(ｆ)は添え糸編み，(ｇ)はからみ編組織の構成は機種別によりそれぞれ異なるが駆動部をはじめ，前述の編針をつねに時間的関連をもたせながら規則正しい運動をさせるカム装置を含む編成部や編地の巻取りまたは搬送装置および糸切断時等の機械自動停止装置なども備えている．
なお，近年，ホールガーメントと称する編み機から直横編機，丸編機，靴下編機の構造を図1･95～1･97に示す．
