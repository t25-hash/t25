# 1・3・6 染色仕上機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・3 繊維機械

染色仕上げは繊維品に色柄等審美性や各種の機能性を付加し，製
品の性能を改善向上する工程である．
染色工程は精練，漂白，浸染，なせん（捺染）および仕上加工な
どに分類される（図1･98）．
a.精練・漂白機（scouring and bleaching machine) 精練・
漂白装置は毛焼機，のり抜き機，水洗機，精練機，漂白機，乾燥機
より構成されるのが普通である（図1･99）．
b.染色機（dyeing and printing machine) 染色機は浸染装
置となせん装置に大別される（表1･6）．
c .液流染色機 浸染装置のうち液染色機は省エネルギー型の
低浴比機（浴比1：8以下）が開発され，省エネルギーと高生産性
の点から合繊用新規設備の主流となっている（図1･100～1･102，
表1･7）．
d.水洗機 布に付着している薬剤，不純物などを水を用いて
洗い落す機械である（図1･103，表1･8）．
