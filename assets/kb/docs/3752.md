# 1・3・7 その他の繊維機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・3 繊維機械

a.不織布機械（non-woven fabric machine) 繊維のウェブ
（短繊維からなるシート状のもの）を紡績機械，織機，編機等の機
械によらないで，物理的，化学的または機械的に結合させ布を作る
機械である．
b.不織布機械の種類 ウェブを結合する方法により表1･10
のように分類される．
c .不織布の製造法 乾式不織布製造法の代表的な製造工程は，
繊維の準備→ウェブの形成→ウェブの結合→乾燥およびキュアリン
グ→仕上加工となり，接着式（浸漬法）製造法の例を図1･106およ
び1･107に示す．また，ウェブをニードルにより結合させるタイプ
の一例を図1･108に示す．
d.用途 乾式不織布の用途は広範囲に及んでおり，衣料用と
しては洋服等の芯地，中入綿，工業資材用としては各種フィルタ，
電気絶縁材，研磨材，このほか土木，農業，建設用にも使用されて
いる．
