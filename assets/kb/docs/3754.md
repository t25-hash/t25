# 1・1・2 交通需要の予測

> **章**: 第1章 全体概要
> **節**: 1・1 交通システムの計画と政策

a.交通需要予測（transport demand forecast ）の基本的な考え
方 交通政策や計画案の工学的妥当性，経済的有効性，事業の採
算性等をチェックまたは比較検討するうえで，施策実施に伴う交通
量の変化の予測は欠かせない作業である．交通需要予測に当たって
は，対象とする交通需要の特性を表現できる計量モデルが用いられ
る．かなりの数のモデルが提案されているが，政策や計画案の目
的，対象地域の特性，目標年次，入手可能なデータの精度等を勘案
して，適切なモデルを選定することが重要である．
交通需要予測の標準的な方法は次のとおりである．
(１)予測対象の選定 検討すべき政策や計画に関して，事前
に想定される影響の大きさや範囲，種類等に応じて，対象とする地
域，期間，旅行者（あるいは貨物），交通機関等を決定する．
(２)モデルタイプの選定 モデルの論理性，操作性を考えな
がら必要最小限で交通需要動向を表現できるものを選ぶ．モデルに
取込む変数は，政策・計画の代替案間で異なる重要な要素（政策変
数と呼ばれる）を中心に，必要最小限な要因に絞る．
(３)モデルの推定 公表される各種統計データや独自の調査
を行って得られるデータから，モデルのパラメータを統計的に求め
る．モデルで仮定される交通行動のロジックに応じて，適切な推定
方法を用いる．
(４)モデルの検証 モデルの現況再現性，他地域や他時点へ
の移転可能性，政策変数の感度等の点から，統計的検定も含めて多
角的にモデルの妥当性を検証する．
(５)モデルを用いた予測 モデルに他の地域や時点のデータ
を入力して，予測値を求める．
b.交通需要予測モデルの概要 交通行動は，人間の種々の営
みとは切り離して考えられない事象だが，そもそも人間のアクティ
ビティは無限に多様である．また交通行動だけを取出しても，移動
者の個人属性，空間特性，時間特性，手段属性等，かなり高次元の
特性をもつ現象である．したがって，これらのすべてを取扱うのは
不可能であるし，また政策や計画の検討を行ううえで無意味なこと
も多い．そこで交通需要予測において，交通行動は，特定の範囲に
限定されるとともに，論理性が保持される範囲内で適宜分割され，
段階的に取扱われる．
その最も典型的な方法が，交通行動を四つの意思決定ステップと
して取扱う4段階推定法である．ここでは，一連の交通行動が，発
生・集中（出発するか否か），分布（どこへ行くか），分担（どの交
通機関を使うか），配分（どの経路を使うか）という四つの段階か
らとらえられる．4段階推定法では，発生・集中以外の3段階の行
動は，いずれも，上位段階の行動結果を制約として次を選択するも
のとみなされる．したがって，予測作業においても，交通需要を上
から順に予測し，上位段階から得られる結果を所与として次段階の
需要を求めるというステップを踏む．
なお，4段階推定法はあくまでも一つの考え方であって，例えば
これらの段階を統合した予測モデルも提案されている．また，必ず
しもすべての段階をモデル化する必要はなく，分析目的や対象に応
じて，特定の段階のみに着目し，他の段階の需要に何らかの仮定を
おいて予測することが多い．
c.4段階推定法の各モデルの概要
.発生・集中交通量予測モデル（trip generation/ attrac-
tion model ) 各ゾーンから出発する，あるいは各ゾーンに到着
する交通量を予測するモデルである．各ゾーンの人口指標や経済指
標が説明変数として用いられる．多くの場合，計量経済モデル等を
援用した，ゾーンを単位とする集計モデル（例：重回帰モデル）が
用いられる．
.分布交通量モデル（trip distribution model ) 各ゾーン
から発生した交通が，どこに行くかを予測するモデルである．ここ
で，発生側のゾーンのことを発ゾーン（origin），集中側を着ゾー
ン（destination），これらの組合せをODペアと呼ぶ．分布交通量
の予測には，ゾーンを単位とする集計モデルが用いられることが多
い．この代表的なものとしては，重力モデルが挙げられる．これ
は，物理現象において重力が二つの物質の重量と両者の距離の関数
によって表現されることを模したものであり，＝

なお，実際には，政策や計画の実行によっても交通量がほぼ変化
しないと考えられるODペアについては，分析時点（現状）での分
布パターンと同一という仮定を設ける方法（現在パターン法）がよ

ンで交通量に大きな変化を生じさせる事態が想定されるODペアの
みに適用される．
.分担交通量モデル（modal split model ) ODペア間の
交通について，どの交通機関を用いるかを予測するモデルである．
ゾーンを単位とする集計モデルと個人を単位とする非集計モデルの
両方があるが，近年は非集計モデルが多く用いられる．
集計モデルとしては，分担率曲線モデルや犠牲量モデルが挙げら
れる．分担率曲線モデルは，各交通手段の利用率をODペア間の距
離や時間の関数として表現する回帰式である．犠牲量モデルは，利
用者の時間評価値（特定の時間に対する金銭的価値．時間価値とも
いう）に分布を仮定し，異なる時間評価値をもつ利用者集団が，各
時間評価値に応じた犠牲量（交通による損失を表す言葉．一般化費
用と同義）の最小となる交通手段をそれぞれ選択すると考えるモデ
ルである．
一方，近年では，個人の意思決定過程を明示的に考慮できる非集
計モデルがよく用いられている．これは，経済学で仮定される効用
最大化原理と整合的な非集計モデルが提案されて費用便益分析等へ
の理論的拡張が可能となったこと，計算機性能が向上してパラメー
タ推定が容易になったことなどの理由による．非集計モデルでは，
まず個人の効用（utility）を基礎におく．ここで効用は，交通サー
ビス特性（ODペア間の時間や運賃など）や旅行者の個人属性の関
数として表される．次に，効用に対してランダム効用（効用は，確
定的効用と特定の確率分布に従う誤差との線形和によって表される
という考え方）を仮定する．そのうえで，人間は，対象とする交通
サービスについて，選択可能集合の中から最も効用が大きくなる選
択肢を選ぶ（効用最大化原理）と考える．
非集計モデルとして多くのタイプのモデルが提案されているが，
最も一般的なのは非集計ロジットモデルである．非集計ロジットモ
デルでは，効用の誤差に対し，選択肢間で無相関かつ同一のガンベ
ルの二重指数分布が適用される．このとき，個人 の選択肢 の
選択確率は，＝exp∑expのように求められる．た

で，効用関数中の未知パラメータは，アンケート調査等によって得
られる個人の行動データと交通サービス特性データを基に，最尤推
定法によって推定される．推定されたモデルは，同一特性をもつと
考えられる利用者集団に拡大して適用され，ゾーン単位の交通量に
集計化されることにより，ゾーン単位の予測交通量が求められる．
.配分交通量モデル（traﬃc assignment model ) 複数の
経路が存在するとき，ある交通機関利用者がどの経路を選択するか
を予測するモデルである．集計モデルと非集計モデルの両方が用い
られる．
集計モデルはさらに，経路選択の均衡状態を仮定するか否か（均
衡型，非均衡型），混雑が経路選択に与える影響を考慮するか否か
（フロー依存型，フロー非依存型），利用者の確率的な分布を考慮す
るか否か（確定型と確率型）等で，いくつかに類型化できる．代表
的な配分方法は，次のとおりである．
･最短経路一括配分法 想定される経路のうち，所与の旅行コス
トが最小となる経路にゾーン間交通量をすべて配分する方法（非
均衡型×フロー非依存型×確定型）．
･分割配分法 ゾーン間交通量を複数に分割し，各経路の交通量
と混雑による旅行コストの増大を考慮して，順に交通量を配分す
ることにより旅行コストを更新していく方法（非均衡型×フロー
非依存型×確定型）．
･確定的利用者均衡配分法 すべての利用者が，混雑による旅行
コストの増大を考慮しつつ，旅行コストが最小となる経路を選択
することによって，それぞれの利用者の選択可能なすべての経路
において旅行コストが同一になる均衡状態となるように，交通量
を配分する方法（均衡型×フロー依存型×確定型）．
非集計モデルも用いられるが，経路は選択可能集合が膨大である
にもかかわらず経路集合を特定しなければならないという問題があ
り，経験的に選択肢集合の特定が比較的容易な鉄道経路選択等に，
実際の適用は限定されている．
なお，近年では，計算機の能力の向上に伴って，地理情報システ
ムやシミュレーションを用いた動的な予測モデルも活用されるよう
になっている．
