# 1・2・3 エネルギー問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

人類は生活を豊かにするために多くの種類のエネルギーを使い続
けてきた．産業革命以前は，エネルギー源の主役は木材であり，直
接燃料としたり，木炭に加工するなどの方法で利用してきた．18
世紀から19世紀初頭にかけてのアメリカからイギリスへの最大の
輸出品は燃料用の木材であった．
産業革命は石炭の有効利用法の発明がきっかけだった．一つはこ
れを蒸気機関用の燃料として使い，もう一つはこれからコークスを
作って製鉄用の燃料とすることであった．そのおかげで工場におけ
る生産が活発化し，高層の建築物が造られ，鉄道が各地に敷設され
ていった．
20世紀に入ると，エネルギー源の主役は石油に取って代わられ
た．燃料としての扱いやすさ，採掘の容易さに加えて，これを有効
に利用するための内燃機関の発明があった．20世紀の半ば以後，
1970年までは石油の利用を増やすことは生活を豊かにすることと
同一のように考えられており，事実，それまではGDPの伸びと石
油の消費量の伸びとは1対1の関係にあった．油田の発見は絶え間
なく続いてきた．人々は，油田の発見がある限り，生活水準は向上
し，幸せな日々を送れるだろうと信じていた．
6―
このような石油エネルギーの利用の将来に対して警鐘を鳴らした
のは，世界中の財界人が地球規模の問題を話し合うために作ったロ
ーマクラブ（Club of Rome）であった．同クラブでは当時マサチ
ューセッツ工科大学の助教授だったD.メドウスらにコンピュータ
を用いた将来の社会の予測を依頼した．その結果明らかになったの
は，このままの割合で石油消費を続ければ，そう遠くない時期に石
油は枯渇し，逆に環境問題は大きくなり，生活の質は急激に低下す
るであろうという予測である．すなわち，エネルギーや資源の有限
性ということが初めて定量的な形で示された．これによって世界は
大きな衝撃を受けた．この結果を裏付けるように，1973年には世
界中が第1回めのオイルショックに見舞われた．それ以来，世界中
の人々はあらゆる資源やエネルギーは無限にあるわけではなく，い
つかは枯渇してしまうということを実感として感じるようになった．
現在，化石燃料として用いられているのは石炭，石油，天然ガス
の3種類である．それぞれの確認埋蔵量と，年間の消費量，そして
埋蔵量を消費量で割った予測寿命は年々変化するため，将来の予測
は困難である．しかし，いずれにしても，そう遠くない時期に枯渇
の時期を迎えることは間違いない．
