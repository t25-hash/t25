# 1・2・6 交通機械とエネルギー問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

エネルギーと交通機械は極めて密接な関係がある．産業革命の時
代に鉄道が広く普及するに至ったのは，イギリスをはじめとするヨ
ーロッパで大量の石炭を産出することができたからであり，20世
紀に自動車が世界中に普及したのも，初めにアメリカで，次いで中
東を中心に大量の油田が見つかったことによる．しかし，これらの
資源の有限性が，その将来に及ぶ利用を制限する可能性がある．
これまでの交通機械とエネルギー問題について特徴的なことは，
そこで使われる発動機のそれぞれが固有の燃料を必要とするという
ことである．
オットー型内燃機関エンジンはオクタン価の高い揮発油，小型の
ディーゼルエンジンはセタン価の高い軽油，航空機燃料はガソリン
と軽油の中間の灯油に近い性質のものが使われる．このため，全エ
ネルギーに占める交通機械の寄与率は10 %，全化石燃料に占める
割合は25%であるが，全石油消費量に占める交通機関が消費する
割合は50%にのぼる．
二酸化炭素による温暖化とエネルギー問題はいずれも化石燃料の
大量消費がその原因であるから，これらの問題に対する対応策も共
通の手法が用いられる．それらの方策は，よりエネルギー消費の少
ない技術に置換える，他の手段に変える，ルールや運用法の変更に
より輸送量を減らすということに大きくまとめられる．新しい技術
に置換えることの例として，蒸気機関車がディーゼル車に替わり，
さらに電車に替わったことが挙げられる．他の手段への置換えとし
ては自動車輸送を鉄道や船に置換えるモーダルシフトが代表例であ
る．輸送量を減らす手段としては共同輸配送のシステムなどがその
例に当たる．
