# 1・2・1 本編の構成 本編の構成は「安全工学」の分析をヒン

> **章**: 第1章 法と技術
> **節**: 1・2 キーワードとしての「危険」

トとし、「危険」をキーワードとしたものと理解できる。北川徹三(有)によれば、安全工学の特徴は、1新規の分野をもつ工学であること、2学際的な総合工学であること、3ヒューマニティーが基底をなしていること、4消極的ではあるが、産業能率の向上に欠くべからざる基礎を与えるものであることなどである。そして、安全工学のめざす社会的目標は、第1に、産業に従事する人々の生命と健康を守ること、第2に、産業の立地する地域住民の平安と健康を守ること、第3に、災害による生産設備や原材料製品の損失を防ぐこと、にまとめられるという。また、本来、安全工学がその研究対象として取り扱うものは、人為災害または人災であり、地震や台風などの天災は対象外であるとされる。というのは、人為災害は、その原因を取り除くことによって予防することが可能なものであり、これらの発生原因と経過を明らかにし、具体的な防止対策を立てることが安全工学の主たる目的であるからである。人災は、発生する場所によって「日常生活で発生するもの」と「職場で発生するもの」に区分され、また、発生後の被害によって「物的な財産上の被害が生じるもの」と「人的な生命・健康に影響が現われるもの」に区分される。そして、職業または産業に関連して発β9-10生する人災は、1労働災害または職業上の人身災害、2工場中毒および職業性疾病、3火災または爆発による災害、4破壊または崩壊による災害、5環境汚染または産業公害、の五つに分けられている。なお、労働災害には「機械的な災害による外傷」のほかに「化学物質の有害作用による工業中毒」が含まれ、後者は「急性」と「慢性」に区分される。「急性の工業中毒」は、「災害性中毒」と呼ばれ、「災害性外傷」とともに「労働災害」に含まれる。「慢性の工業中毒」は「化学的作用による職業性疾病」または「職業病」と呼ばれる。「職業病」には、異常な温度、湿度、気圧、音響、振動、放射線などの物理的作用によるもの、細菌、寄生虫などの生物的作用によるもの、作業姿勢、作業過重、運動不足などの作業方法によるものもある。結局、「労働災害」と「職業病」は、不時の「事故」によるものか、「定常作用」に起因するものかによって区別されるようである。以上の説明から推測すると、安全工学は、まず第1に「労働災害」の防止を念頭においた学問である。そして、その応用として、第2の領域である「産業公害」の防止が検討される。さらに、職業または産業の成果としての「製品」が消費されることによって「日常生活で発生する」「製品事故」も視野に入ってくる。最後の点は、上にみた整理においては明示されていないが、「家庭における安全」に関する記述のなかで、洗濯機による感電、湯わかし器による一酸化炭素中毒、家庭用プロパンガスの爆発などの問題が指摘されている。そこで、エンジニアが「工学的思考」の持ち主であり、安全工学の発想になじみやすいと仮定した場合、本編第2章以下を読むに当たっては、まず、第3章から始めることが薦められることになろう。主要部分は「労働災害または職業上の人身災害」と「工場中毒および職業性疾病」(上記12)の防止に関するものである。これらについては、それが「突発的」に生じるか、「晩発性」のものかという区別が有用である。しかし、その前半部分は、「事故の防止に関する法律」という表題からうかがえるように、工場・施設内部の事故が、周辺にも影響を及ぼす場合を想定している。「火災または爆発による災害」と「破壊または崩壊による災害」(上記34)は、これに相当する。この場合、実際には「工場・施設内部」のうける被害の方が甚大である場合もあることも看過してはならない。ただ、影響が周辺に及ぶものは、そうでないものと比較して、より複雑な法的問題を生じさせるので、後半とともに「環境汚染または産業公害」(上記5)に関するものである。い。しかし、放射性物質、有害化学物質については、特別の法制度が存在することから、独立の章が設けられている。以上を読み進んだ後であれば、第2章前半の「消費者保護」という問題を理解することは容易であろう。「環境保護」と「消費者保護」は、ある程度共通の社会的・技術的背景を有し、かつ類似の思想的基盤を有するからである。なお、後に触れるように、法的思考のひとつの特徴として、「弱者保護」という理念があり、この理念は「労働者保護」「環境保護」「消費者保護」に共通するものともいえる。る。メントシステム」「環境アセスメント」「リスクマネージメント」「リスクアセスメント」「リスクと保険」を扱っている、これらは、従来型の法制度の限界を意識して導入された仕組みである。基準」を扱っている。ここでは、国家レベルの法制度の限界が顕著にあらわれる。これも、従来型の法制度の限界という側面を有する
