# 1・2・2 法令の体系

> **章**: 第10章 は「知的財産に関する法制度」を扱っている.
> **節**: 1・2 キーワードとしての「危険」

a.法的ルールのピラミッド 現在の日本社会は,数限りのない法的なルールによって埋め尽くされている.第3章を読み始めた読者は,まず,その細かさにへきえきすると思われる.このような法的ルールの密林で迷わないためには,その構造を理解することが必要である.ポイントをいくつかあげよう.第1のポイントは,「法的ルールはピラミッド型になっている」ということである.日本という国家の中での法的なルールは,「日本国憲法」を頂点とするピラミッド型となっている.「日本国憲法」の条文数は103にとどまる.しかし,ここで定められているルールが,その他すべての法的ルールの内容を支配するのである.しかし,「日本国憲法」の内容は,あまりにも抽象的であるので,実際上は,もう1ランク下の法的ルールが最も重要である.これが,国会が定めるルールであり,「法律」と呼ばれる.ある研究(代)によると,2001年6月現在で生きている「法律」は1876本であり,その条文の総数は概算で,59900条である.たとえば,第3章第1節,第2節の記述にあらわれる「法律」は,「労働基準法」「労働安全衛生法」「放射性同位元素等による放射線障害の防止に関する法律」「核原料物質,核燃料物質及び原子炉の規制に関する法律」「医療法」「薬事法」「建築物における衛生的環境の確保に関する法律」「鉱山保安法」「鉱業法」「火薬類取締法」「消防法」「建築基準法」「作業環境測定法」「じん肺法」「毒物及び劇物取締法」などである.以上のような「法律」は,具体的な名称としては,「○○法」という場合と,「○○に関する法律」という場合がある.短い名称の場合には「○○法」ということが多い.条文の束として法的ルールを新たに定めることを「制定」と呼び,既存のものに変更を加えることを「改正」と呼ぶ.法律の改正は,「○○法の一部を改正する法律」という名称の法律でなされることが多く,この場合,既存の法律が修正された形で生き延び,当該法律は目的を達成して消滅する.注意が必要なのは,「○○法」という言葉が,そのような名称をもつ国会が定めたルール(条文の束)を意味する場合のほかに,特定の事柄についての法的ルールの全体を意味する場合があることである.たとえば,「労働法」という場合は,「労働法」という名称の「法律」は存在しない.そこで,「労働法」といえば,「労働」という特定の事柄についての法的ルールの全体だけを意味する.これに対し,「民法」という場合は,「民法」という名称の法律が存在し,かつ,「市民相互の関係」に関する法的ルールの全体を意味することもあるから,説明を読む場合には,どちらのことかを考えなければならない.なお,「労働安全衛生法」という名称の法律が定めるルールは,内容的にみて「労働者の保護」にかかわることから,法学部の教育では「労働法」の一分野として位置づけられているが,内容的には「行政法」の一分野でもある(1・2・3項のf.g参照).また,「法」と「法律」を厳密に区別しない用語法においては,「法律」という言葉で「法的ルールの全体」を表現することがあるので,「法律学」は「法学」と,「法律学者」は「法学者」,「法律学科」は「法学科」,「法律問題」は「法的問題」と互換的である.ただ,「法学部」が「法律学部」と,「法律家」が「法家」と呼ばれることはなく,「法律用語」が「法用語」と呼ばれることはない.b。行政機関が定めるルール 第2のポイントは,「行政機関が細かなルールを定める」という点である.法的ルールは「日本国憲法」を頂点としたピラミッド型になっており,「日本国憲法」の103条のルールが約1900本の「法律」の60000弱の条文によって具体化されているのであるが,多くの場合,「法律」の定めるルールはなお,抽象的である.たとえば,「労働安全衛生法」は,20条で〈事業者は,次の危険を防止するため必要な措置をとらなければならない.〉とし,1機械,器具その他の設備による危険,2爆発性の物,発火性の物,引火性の二等による危険,3電気,熱その他のエネルギーによる危険を列挙している.「危険を防止するため必要な措置」が具体的にはどのような措置なのかは,この条文の字面だけみても全くわからない.そこで,まず,日本国憲法73条によって「法律を誠実に執行する」ことを命じられている「内閣」が,より具体的な法的ルールを自ら定めることになる.内閣が定める法的ルールを「政令」と呼ぶ.「労働安全衛生法施行令」はこれにあたる.政令の名称は「○○法施行令」というものや,「○○に関する政令」というものが多い.労働安全衛生法は,37条で〈ボイラーその他の特に危険な作業を必要とする機械等で,政令で定めるもの(以下「特定機械等」という)を製造しようとする者は,厚生労働省令で定めるところにより,あらかじめ,都道府県労働局長の許可を受けなければならない〉と定め,40条で,検査をパスした「特定機械等」でなければ使用してはならない,と定めている。事業者は,一般的に,「危険を防止するため必要な措置」をとる義務を負うのであるが,「特に危険な作業を必要とする機械」については,さらに,一定の安全性を備えた機械のみを使用するという,より具体的な義務を負う.事業者が何をすべきは,やや明確になっている.しかし,具体的に何が「特に危険な作業を必要とする機械」に該当するのかは「法律」を見ただけではわからない.国会はそれを具体的に定めることを避けたのである.「労働安全衛生法施行令」は12条で,「特定機械等」のカタログを定めている.「ボイラー」「第一種圧力容器」「つり上げ荷重が3t以上のクレーン」「つり上げ荷重が3t以上の移動クレーン」「積載荷重が1t以上のエレベーター」「ガイドレール」「ゴンドラ」である.このカタログからの追加削除や荷重等の数値の変更は,政令の改正で行われる.国の行政機関は,「内閣」の下に,内閣府や省などの組織を設けて仕事を分担している.労働災害の防止は,従来は労働省の,2001年1月6日に発足した中央省庁の新体制のもとでは,厚生労働省の仕事である.各省の長である「各省大臣」は,それぞれ「省令」という法的ルールを定めることができる.「労働安全衛生規則」は労働省の長である労働大臣が昭和47年に定めたものである.「省令」は,「○○法施行規則」「○○規則」「○○規程」「○○に関する省令」などさまざまな名称をもつ.法律及び「○○法施行令」とセットになっている一般的なものは,「○○法施行規則」という名称をもつものが比較的多い。(4・1・1項参照)「労働安全衛生規則」は,労働安全衛生法を施行するための一般的な細則であり,そのほかに,特定の機械,物,作業ごとの細則を定める省令が存在する.「ボイラー及び圧力容器安全規則」「クレーン等安全規則」「ゴンドラ安全規則」「有機溶剤中毒予防規則」「鉛中毒予防規則∫「特定化学物質等障害予防規則」「高気圧作業安全衛生規則」「電離放射線障害防止規則」「粉じん障害防止規則」などがこれにあたる.以上のように,「政令」と「省令」が,行政機関が定める法的ルールの主要なものである.両者をあわせて「政省令」と呼ぶ.また,行政機関が定める法的ルールをまとめて「命令」と呼ぶことがある.ここでいう「命令」は,ドイツ語のVbrordnungの訳であり,一般的なルールである.これに対し,事業者等に個別的な義務を課す「改善命令」という場合の「命令」は,ドイツ語のBe艶hlの訳であり,注意して区別しなければならない.「法令」というのは,「法律」と「命令」をあわせて略称したものである.「法律用語」と「法令用語」は互換的に用いられる.各省が執行すべきものとされている「法律」のもとでは,多数のβ9-11政省令をはじめとする「命令」が制定され,また頻繁に改正がなされている.ある研究q7)によると,「命令」の本数は,法律の本数の約2.8倍であり,法律の制定・改正は一年間で約100本にとどまるのに対し,「命令」の制定・改正は平均してその10倍を超えるという.c.書かれていないルール 第3のポイントは,「法的ルールは条文だけではない」ということである.先にみたように,各省をはじめとする行政機関が執行すべき「法律」については,それを具体化する「政省令」がかなりの程度細かい法的ルールを定めているので,「危害防止のために何をなすべきか」がある程度具体的に示されることになる.しかし,このような構造が見られない法的ルールも存在する.たとえば,「民法」という「市民相互の関係」に関する法的ルールについては,「民法」という名称の法律が存在するが,「民法施行令」「民法施行規則」というような政省令は存在しない.これは,「市民相互の関係」についての法的ルールは,「法律」で一般的抽象的な形で定めるだけにすべきであるという政策によるものである.そこで,具体的な場面ごとに何が正義にかない公平であるかは,紛争が起きるたびごとに,その紛争の解決にあたる裁判所が判定することになる.これが「訴訟」あるいは「裁判」と呼ばれる仕組みであり,裁判所の判定を「判決」と呼ぶ.この場合,裁判所は,実際には「法律」で定められた一般的抽象的なルールをそのまま用いるのではなく,もう少し具体的な法的ルールを用いて紛争の解決をしている.たとえば,民法717条にいう「土地の工作物」には,高圧線や鉄道の軌道施設が含まれる.また,そこにいう「設置又は管理のi暇疵」とは,それらの施設等が〈通常有すべき安全性を欠く〉状態にあることをいうとされている.しかし,このようなルールは,類似の「紛争」の解決が集積するなかで生まれてきたものであり,はっきり条文の形で書かれているわけではない.「判例法」と呼ばれるこのようなルールをどのような性質のものとみるかは,法学者のなかでも見解が分かれる.しかし,事実として,このようなルールが重要な機能を果たしていることを否定することはできない.書かれていないルールは,別の根拠からも存在する.それは,憲法や法律の定めるルール自体が,抽象的な原理・原則の具体化として理解されるからである。法の理念は「正義」「公平」「安定」など極めて抽象的なものであり,そこから導き出される〈権力は濫用してはならない〉〈個人の自由を尊重しなければならない〉〈契約は守られなければならない〉〈他人の生命・身体を傷つけてはならない〉などの原則的な法的ルールは,たとえ条文の形で書かれていない場合であっても,理性的な人間の集合である国家の内部では,良識として存在するルールであると考えられる.以上のようにみてくると,〈他人にむやみに危険を及ぼしてはならない〉〈潜在的な危険を有する行為をした者は,事故が発生した場合には責任をとるべきである〉という抽象的な法的ルールが様々な分野で存在し,あるいはそのようなルールを採用すべきであると主張されることも不思議ではない.ただ,ひとくちに「危険」といい,「責任」といっても,実に様々なものがある.これをきめ細かく分析し,合理的な制度設計をするのが,「安全工学」をヒントにした「法工学」の出発点となろう。以下では,法には様々な分野があることを紹介しつつ,それぞれの分野で「危険」がどのような形で取り扱われているかをみることにしよう.
