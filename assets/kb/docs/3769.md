# 1・2・3 法の三大分野

> **章**: 第1章 法 と 技 術
> **節**: 1・2 キーワードとしての「危険」

a.民事法 現在の日本社会は、「日本国憲法」を頂点とし、1900本弱の法律、そしてその下に存在する無数の政省令その他によって構成されるピラミッド型の法的ルールによってコントロールされている。これらの法的ルールは、その性質によって、「民事法、刑事法、行政法」’の3大分野に分けることができるq8)。ある人のミスによって事故が発生し、他の人が死傷した場合、ミスをした人が「加害者」、死傷した人が「被害者」ということになる。β9-12このような事故が起きてしまったときに、事後的に、なぜそのような事態に至ったのか、誰が「悪者」なのかを明らかにし、「悪者」に「償い」をさせるのが「正義」にかなうと考えるのが、多くの人々の自然な感情であろう。このような常識を基礎にしているのが「民事法」「刑事法」という分野である。民事法とは、市民相互の関係及び会社ないし商取引についての法的ルールである。誰が誰に対してどういう場合にどんなことを要求できるか、という「実体」に関するルールと、そのような要求をどのようなプロセスで行うか、という「手続」に関するルールを含む。前者は、「民法」と「商法」という分野であり、後者は「民事訴訟法」という分野である。そして、前者の基本的な部分は、「民法」「商法」という名称の法律で、後者の基本的な部分は「民事訴訟法」という名称の法律で定められている。b.不法行為法 民事法における事故の後始末は、加害者が被害者の生じた「損害」を「賠償」するという形で「責任」をとることによってなされる。誰がどういう場合に加害者とされるのかについての「実体的ルール」は、「不法行為法」と呼ばれる。民事法において「不法行為」とは、ルール違反をして他人に損害を発生させることであり、どういう場合に「不法行為」をしたことになるのか、不法行為の後始末はどのように行うかについてのルールが「不法行為法」である。「不法行為法」として最も重要なのは、「民法」という名称の法律の709条から724条である。全体で1000条余りの「民法」のうち、1条から724条までの部分を「財産法」、725条以下を「家族法」と呼ぶ。財産法の部分は、明治29(1896)年に制定されたルールであり、その後修正を受けつつも、基本的な構造は維持されている。明治31(1898)年に制定された家族法の部分は、第二次大戦後、根本的な修正を受けている。民法の「財産法」の部分は、個々人が自由な経済活動ないし社会的活動を行うことを理想として、個人の有する生命・身体、自由、名誉、財産を相互に尊重し、取引は自由な交渉によってなすことを基本として様々なルールを定めている。そして、その一部である不法行為に関するルールの基本は、「過失責任の原則」及び「金銭賠償の原則」である。民法709条〈故意又は過失に因りて他人の権利を侵害したる者は之に因りて生じたる損害を賠償する責めに任ず〉民法722条1項〈第417条の規定は不法行為に因る損害の賠償に之を準用す〉民法417条〈損害賠償は別段の意思表示なきときは金銭を以って其額を定む〉以上のルールによれば、他人の生命・身体や財産に被害を与えた者は、それがわざと行ったものである場合だけでなく、うっかりして行ったものである場合にも、金銭を支払うという形で後始末をしなければならない。民法709条のルールは極めて抽象的であり、あらゆる事故や災害に適用可能である。事故の原因を分析し、誰の「不注意」が原因だったのかといゲ「悪者探し」がなされるという構造は、後で出てくる刑法211条の場合と同様である。なお、1994年に制定された「製造物責任法」という名称の法律は、民法709条の「過失責任の原則」を修正するものである(第2章参照)。c.刑事法 刑事法とは、国家と犯罪者の関係についての法的ルールである。誰がどのような場合に「犯罪者」とされるのか、どのような犯罪にどのような刑罰が科されるのか、という「実体的ルール」と、犯罪者をどのように見つけ出し、どのようなプロセスで刑罰を科すのかという「手続的ルール」の双方を含む。前者は「刑法」という分野であり、後者は「刑事訴訟法」という分野である。前者の基本的部分は「刑法」という名称の法律で、後者の基本的部分は「刑事訴訟法」という名称の法律で定められている。「刑法」という名称の法律は、明治40(1907)年に制定されたものである。「刑法」の条文の数は全体で300弱であり、第1条から第72条までに、「犯罪」と「刑罰」一般についてのルールが、第77条から第264条までに、個別の「犯罪」についてのルールが置かれている。第73条から第76条までが欠けているのは、皇室に対する罪が昭和22年に削除されたからである。「刑罰」の種類は、「死刑」「懲役」「禁門」「罰金」「拘留」「科料」「没収」である(刑法9条)。「懲役」と「禁銅」は、刑務所に拘置される点では同じであるが、「懲役」は、所定の作業を行う義務があるのに対し、「禁忌」の場合はそうではない。両者は、期間によって「無期懲役」と「有期懲役」、「無期禁鋼」と「有期禁鋼」に分けられる。「有期懲役」「有期二二」は、1ヶ月以上15年以下が原則である。「拘留」は、拘留場に留置するもので、期間は1日以上30日未満である。「罰金」及び「科料」はいずれも金銭を国に納付するものである。.「罰金」は1万円以上、「科料」は千円以上1万円未満である。何が「犯罪」であり、どの程度の「刑罰」を科すかを、あらかじめ「法律」で定めておかなければならないという考え方を「罪刑法定主義」という。わが国では、明治維新後の近代化の過程でこのような考え方が採用され、今日に至っている。当時の先進国でわが国の法制度の模範とされたのは、最初はフランスであり、次いでドイツであったq9)。ちなみに、フランス及びドイツの法制度は「大陸法系」と呼ばれ、イギリス及びアメリカの法制度は「英米法系」と呼ばれ、様々な点で相違がある。「刑法」の定める犯罪は、それによって何が侵害されるかによって「国家に関する法益に関する罪」「社会に関する法益に関する罪」「個人的な法益に関する罪」の三つに区分される(2。)。「国家に関する法益に関する罪」は、内乱や外患に関する罪(77条から89条)のほか、公務執行妨害(95条)や贈収賄(197条から198条)など、国家の存立を危うくしたり、国や公共団体の組織の正常な機能を阻害するような罪である。「社会に関する法益に関する罪」とは、多くの国民の利益に関するものであり、被害が広く及ぶ点に特徴がある・また、個々人に実害が発生したことを待たずに、かなり抽象的な危険性が発生した段階で「犯罪」の成立を認め、処罰する点に最大の特色がある20.「社会に関する法益に関する罪」は、さらに、放火罪(108条から110条)に代表される「公衆の安全に対する罪」、通貨偽造罪(148条)に代表される「信用に対する罪」、わいせつ物頒布罪(175条)に代表される「風俗秩序に対する罪」の三つに分類される。「個人的法益に関する罪」とは、個々人の生命・身体、自由、名誉・信用、財産に関するものである。殺人罪(199条)、傷害罪(204条)などは「生命・身体に対する罪」、監禁罪(220条)、未成年者誘拐罪(224条)、強姦罪(177条)などは「自由に対する罪」、名誉殿損罪(230条)、信用殿損罪(233条前段)などは「名誉・信用に対する罪」、窃盗罪(235条)、詐欺罪(246条)などは「財産に対する罪」に分類される。d.例外としての「過失犯」 以上のように、刑法は、社会の構成員であれば当然従うべきルールを列挙したもので、基本的には、わざと行ったルール違反、すなわち「故意犯」を「犯罪」として制裁を加えるものとしている。刑法第38条第1項〈罪を犯す意思がない行為は、罰しない。ただし、法律に特別の規定がある場合は、この限りではない〉.したがって、うっかりして行ったルール違反、すなわち「過失犯」が「犯罪」とされるのは例外である(株)。刑法の定める「過失犯」は、失火(116条:50万円以下の罰金)、過失激発物破裂(117条2項:50万円以下の罰金)、業務上失火(117条の2:3年以下の禁固、50万円以下の罰金)、出水(122条:20万円以下の罰金)、往来危険(129条1項:30万円以下の罰金)、業務上往来危険(129条2項:3年以下の禁門、50万円以下の罰金)、過失傷害(209条:30万円以下又は科料)、過失致死(210条:50万円以下の罰金)、‘業i務上過失致死傷及び重過失致死傷(211条:5年以下の懲役又は禁銅、50万円以下の罰金)に限られる。刑法は、先にみたように人の「生命・身体」「自由」「名誉・信用」「財産」を保護するために、様々な条文をおいている。その中で、人の生命・身体の保護は特に重要であるので、「故意犯」のほかに「過失犯」も処罰することにしたのである。しかし、これらの多くは、罰金を支払えば足りるものとされている。「犯罪者」の汚名を着せられて「刑務所」に入るということになり得るのは、「業務上」という形容詞のつくルール違反に限られる。最もポピュラーなのが「業務上過失致死傷罪」である。『刑法211条〈業務上必要な注意を怠り、よって人を死傷させた者は、5年以下の懲役若しくは禁素心は50万円以下の罰金に処する。重大な過失により人を死傷させた者も、同様とする〉。条文を書く際の文法では、andについて「及び」と「並びに」を、orについて「又は」と「若しくは」を厳密に使い分ける。連結する語句に段階があるときには注意が必要である。たとえば、AandBは〈A及びB>であるが、(AandB)andCは〈A及びB並びにC>と表現する。単なる並列には「及び」を使い、「並びに」は使わない。これを格言風にしたのがく及びなければ、並びになし〉である。内閣法制局という役所がある。その重要な仕事として「法令審査」がある。内閣が国会に提案する「法律」の条文や、内閣が定める「政令」の条文は、内閣法制局によって徹底的にチェックされる。このような「法令審査」のプロである内閣法制局の職員の結婚披露宴で、ある人がスピーチで「新郎並びに新婦は。..」と始めた途端、「新郎及び新婦!」というヤジが飛んだという、まことしやかなエピソードもある(株)。他方、AorBは〈A又はB>、(AorB)orCは〈A若しくはB又はC>と表現する。「又はなければ、若しくはなし」という点は同じであるが、単なる選択に使わない「若しくは」を、かっこ内の選択に使う点が異なる。刑法211条の「業務上過失致死傷罪」及び「重過失致死傷」と刑法209条の「過失傷害罪」及び刑法210条の「過失致死罪」との相違は、事故の防止のために注意深く行動するようにという要請を、刑法211条では「懲役」又は「禁鋼」という強い「サンクション」によって確保しようとしている点にある。「5年以下の懲役又は丸鋼」という処罰の上限は、1968年に引き上げられたもので、それ以前は「3年以下の禁銅」であった。さらに、常習的な飲酒運転など悪質な交通犯罪をより重く処罰するために、2001年には刑法の一部が改正され、第208条の2として「危険運転致死傷」が新設された。なお、「重過失致死傷」のルールは、1947年に付け加えられたものである。刑法211条によって高度の注意義務を課されるのは一般人ではなく、「業務者」に限られる。それでは、「業務者」とは具体的にどんな人なのだろうか。e.「業務」の意味に関する判例 刑法211条にいう「業務」とは、通常の日本語の意味とは異なり、〈各人が社会生活上の地位に基づき反復継続して行う事務で、かつ他人の生命・身体に危害を加えるおそれのあるもの〉とされている。職業である必要はなく、マイカーの運転であっても「業務」とされる。なぜそうなるのか。これは、最高裁判所がそのような扱いをしているからである(%)・(25)。刑法211条という条文は「法律」という形で国会が定めたルールであるが、具体的な事案について加害者とされたAが211条違反で有罪になるかどうかは、裁判所の判断によって決まる。現在の裁判所は、最高裁判所を頂点とするピラミッド型の組織である。個々の事件は、原則として、まず、地方裁判所(全国で50)で扱われ、必要に応じて、高等裁判所(全国で8)、最高裁判所と順次三段階の判断を受けることが可能な仕組みである。個々の「刑事事件」を最初に扱う地方裁判所においても、地方裁判所の判断を見直す高等裁判所においても、上記の最高裁判所の「判例」を前提に事件が処理される。したがって、刑法211条は、事実上次のように書きi換えられているのである。〈社会生活上の地位に基づき反復継続して、他人の生命・身体にβ9-13危害を加えるおそれのある行為をなす者が、必要な注意を怠り、よって人を死傷させた場合には、5年以下の懲役若しくは禁固又は50万円以下の罰金に処せられる。〉なお、以上のような扱いを「解釈」という。「解釈」とは、元来は言葉の中に含まれている意味を「発見」し、明らかにすることであるが、「法の解釈」は必ずしもそうではない。裁判所は法律を解釈する権限、すなわち、「国会」が定めたルールの意味をより具体的なものとする権限を与えられており、結果的には、新たな意味を持ち込んでいることもある。一般人とは区別して、「業務者」だけに「重い注意義務」を科すという考え方からすれば、タクシー運転手などのプロのドライバーはともかく、免許をとったばかりの初心運転者や久しくペーパードライバーだった者も「業務者」に当たるという解釈は違和感がある。しかし、〈日常的な行為と比較して危険度の高い行為をする者には重い注意義務を科すことが正義にかなう〉という政策判断が優先されているのである。以上のような解釈が定着していることによって、刑法211条違反として処罰される者のほとんどは、交通事故を起こした者となっている16。f行政法 「行政法」とは、行政の組織と活動についての法的ルールをいう。全部の法律1900本弱のうち、少なくとも1300本程度は「行政法」の分野に属する。そして、すでに触れたように、個。別の「法律」の下、多数の「政省令」をはじめとして、行政機関が定めた細かいルールが存在するのがこの分野の特徴である。だが、大学の法学部における行政法教育は、このような細かいルールがどうなっているかを逐一明らかにすることに主眼をおいてはいない。〈行政活動は法に従って、国民の権利利益を尊重しつつなされなければならない〉〈行政活動は、国民の批判に応えつつ、公正かつ透明性をもってなされなければならない〉などの基本的な理念、それを具体化している「行政手続法」「行政機関の保有する情報の公開に関する法律」などの「実体的ルール」及び「手続的ルール」、行政活動に不満をもった国民がどのように救済されるかに関する「実体的ルール」及び「手続的ルール」の部分が重視されている20.なお、後者の「行政救済法」といわれる分野についての基本的事項は、「国家賠償法」「行政不服審査法」「行政事件訴訟法」という名称の法律に定められている。g。「弱者保護」のための法 民事法、刑事法、行政法という3区分に収まりきらないものもある。「弱者保護のための法」とも性格づけられる「労働法」「社会保障法」「経済法」「環境法」「消費者法」などである。これらは、それぞれ重点の違いはあっても、民事法、刑事法、行政法のすべての要素を含んでいる。たとえば、「労働法」は、使用者との比較で「社会的弱者」である「労働者」を保護することを目的として様々なルールを定めている。このうち、3・2・1項で解説されている「労働基準法」の基本的な部分は、「民法」の基本原則である「契約の自由」「解雇の自由」に修正を加えるものである。したがって、この部分は、性質上は民事法に属することになる。他方、「労働安全衛生法」は、1947年に「労働基準法」が制定された当時は、それに含まれていた「労働者の安全と衛生の確保」についてのルールを、1972年に独立させたものである。労働安全衛生法は、「特定機械等」の許可制度、検定制度など、行政機関の活動に関するルールを多く含んでおり、理論上は「行政法」にも含まれる。h。「知的財産」 「知的財産」は、「無体財産」とも呼ばれる。発明やデザイン、文学作品などをいい、財産的価値をもつ「無形」のものというニュアンスである。これらを作り出した人を法的に保護し、他人が勝手に利用することを制限するためのルールが「知的財産法」である。これらのルールによって保護される人はb「民法」によって保護される人が「有体物」(土地建物、機械等々)について「物権」とりわけその中で最も強力な「所有権」を有する場合と類似した地位に立つので、「知的財産権」ないし「知的所有権」を有すると表現することがある。知的財産法は「活動促進機能」(1・3・2β9-14項参照)が特に重要な意味をもつ法分野である(28)。「知的財産法」は、「創作」にかかわるルールである「創作法」と、「標識」にかかわるルールである「標識法」に大別され、「創作法」がさらに主として産業目的のルールである「工業所有権法」と文化目的のルールである「著作権法」に区分されるのが通例である。「工業所有権法」に属するルールを定めるものとして、「特許法」「実用新案法」「意匠法」「半導体集積回路の回路配置に関する法律」「種苗法」という名称の法律が、「著作権法」に属するルールを定めるものとして、「著作権法」という名称の法律が、「標識法」に属するルールを定めるものとして「商標法」という名称の法律があげられる。(詳細は第10章参照)以上で、読者が第2章以下を読むに際してあらかじめ頭にいれておくとよいと思われる予備知識の提供を終わり、以下、節を改めて、「法の機能」という切り口で、それぞれの章で取り上げられている問題を考えるに当たって有益であると思われる点について、コメントを加えることにしたい。
