# 1・3・1 社会統制機能

> **章**: 第1章 法 と 技 術
> **節**: 1・3 法の機能

a．刑事法の社会統制機能 法が社会において果たしている役
割には様々なものがあるが，法哲学者である田中成明（29）は，「社会
統制機能」「活動促進機能」「紛争解決機能」「資源配分機能」の四つ
に整理している。
法のイメージとして最もわかりやすいのが，第1の「社会統制機
能」であり，そのような機能を有する典型的なものは「罪」と「罰」
について定める「刑法」である．
法は人々が一牢の行動様式をとることを要請し，社会に一定の秩
序をつくりだそうとする．そのために，法は，要請された一定の行
動様式をとらなかった者になんらかの不利益を与えるという形で定
められることが多い．このような不利益を「サンクション」（制裁）
という．そして，「サンクション」を避けようとして多くの人々が行
動する場合には，一定の行動様式をとるべきであるというルールに
よる一定の秩序が社会に存在することになる．
以上のように田中が指摘する「社会統制機能」は法の古くからの
基本的機能である．歴史をさかのぼってみると，日本においても，
『日本書紀』によれば645年に「大化の改新」と呼ばれる政治改革が
なされた．その目標は天皇を中心とした集権国家の樹立であった．
618年の唐による中国の統一を意識し，中国の律令制を模範とした改
革である．日本の律令制は701年の「大宝律令」によって完成した．
そこでの「律」は現在の意味における「刑法」に相当し；「令」は「行
政法」に相当する㈲．
「刑法」において用いられる「サンクション」は「刑罰」と呼ば
れる．「刑法」は，社会の構成員に「○○をした者は，△△という処
罰を受ける」あるいは「◇◇をしなかった者は，△△という処罰を
受ける」という形でルールを定める．このルールは「犯罪」とそれ
に対応する「刑罰」に分解できる。
多くの犯罪は「○○をしてはならない」という「禁止」に反した
場合に成立する．「人を殺してはならない」というルールは，わざわ
ざ別個のルールとして定められてはいないが，刑法199条の〈人を殺
した者は，死刑又は無期若しくは3年以上の懲役に処する〉という
ルールに論理的に含まれている．人々の行動を統制するこのような
ルールがまず成立し，そのようなルールに反した者が「犯罪者」と
いう汚名を着せられ，「刑罰」という不利益を受けるのである．
先に指摘した「罪刑法定主義」という考え方からすると，現在の
日本でどのような行為が「犯罪」とされ，どのような「刑罰」を科
せられるかを厳密に見るためには，国会の定めたルールである「法
律」を逐一検討しなければならない．個別の法律の罰則規定を「特
別刑法」と呼び，「刑法」という名前の法律を「一般刑法」と呼ぶこ
とがある．「特別刑法」の大部分は，行政規制に係わるものであり，
このような行政規制に係わる罰則は「行政刑法」とも呼ばれる．
行政機関が執行すべきものとされる「法律」にはほとんど例外な
く，末尾付近に「罰則」という部分がある．刑法は「犯罪」のカタ
ログを定める最も重要な法律であるが．交通事故関係を除外した統
計でみる限り，「刑法犯」，すなわち，刑法○○条違反によって有罪
とされる者は6割弱にとどまり，残りの4割強は，個別の法律違反
を理由としている（31）．
現在，日本で生きている法律の総数は1900本弱であるが，そのう
ち少なくとも900本程度の法律には，その末尾付近に「罰則」という
部分が設けられている．
