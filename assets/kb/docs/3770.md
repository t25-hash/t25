# 1・3・1 社会統制機能

> **章**: 第1章 法 と 技 術
> **節**: 1・3 法の機能

a.刑事法の社会統制機能 法が社会において果たしている役割には様々なものがあるが、法哲学者である田中成明(29)は、「社会統制機能」「活動促進機能」「紛争解決機能」「資源配分機能」の四つに整理している。法のイメージとして最もわかりやすいのが、第1の「社会統制機能」であり、そのような機能を有する典型的なものは「罪」と「罰」について定める「刑法」である。法は人々が一牢の行動様式をとることを要請し、社会に一定の秩序をつくりだそうとする。そのために、法は、要請された一定の行動様式をとらなかった者になんらかの不利益を与えるという形で定められることが多い。このような不利益を「サンクション」(制裁)という。そして、「サンクション」を避けようとして多くの人々が行動する場合には、一定の行動様式をとるべきであるというルールによる一定の秩序が社会に存在することになる。以上のように田中が指摘する「社会統制機能」は法の古くからの基本的機能である。歴史をさかのぼってみると、日本においても、『日本書紀』によれば645年に「大化の改新」と呼ばれる政治改革がなされた。その目標は天皇を中心とした集権国家の樹立であった。618年の唐による中国の統一を意識し、中国の律令制を模範とした改革である。日本の律令制は701年の「大宝律令」によって完成した。そこでの「律」は現在の意味における「刑法」に相当し;「令」は「行政法」に相当する(有)。「刑法」において用いられる「サンクション」は「刑罰」と呼ばれる。「刑法」は、社会の構成員に「○○をした者は、△△という処罰を受ける」あるいは「◇◇をしなかった者は、△△という処罰を受ける」という形でルールを定める。このルールは「犯罪」とそれに対応する「刑罰」に分解できる。多くの犯罪は「○○をしてはならない」という「禁止」に反した場合に成立する。「人を殺してはならない」というルールは、わざわざ別個のルールとして定められてはいないが、刑法199条の〈人を殺した者は、死刑又は無期若しくは3年以上の懲役に処する〉というルールに論理的に含まれている。人々の行動を統制するこのようなルールがまず成立し、そのようなルールに反した者が「犯罪者」という汚名を着せられ、「刑罰」という不利益を受けるのである。先に指摘した「罪刑法定主義」という考え方からすると、現在の日本でどのような行為が「犯罪」とされ、どのような「刑罰」を科せられるかを厳密に見るためには、国会の定めたルールである「法律」を逐一検討しなければならない。個別の法律の罰則規定を「特別刑法」と呼び、「刑法」という名前の法律を「一般刑法」と呼ぶことがある。「特別刑法」の大部分は、行政規制に係わるものであり、このような行政規制に係わる罰則は「行政刑法」とも呼ばれる。行政機関が執行すべきものとされる「法律」にはほとんど例外なく、末尾付近に「罰則」という部分がある。刑法は「犯罪」のカタログを定める最も重要な法律であるが。交通事故関係を除外した統計でみる限り、「刑法犯」、すなわち、刑法○○条違反によって有罪とされる者は6割弱にとどまり、残りの4割強は、個別の法律違反を理由としている(31)。現在、日本で生きている法律の総数は1900本弱であるが、そのうち少なくとも900本程度の法律には、その末尾付近に「罰則」という部分が設けられている。物及び劇物取締法」「労働基準法」「じん肺法」「薬事法」「食品衛生法」「高圧ガス保安法」「消防法」「石油コンビナート等災害防止法」「振動規制法」「騒音規制法」「水質汚濁防止法」「大気汚染防止法」「廃棄物の処理及び清掃に関する法律」「鉱山保安法」「肥料取締法」「農薬取締法」「農用地の土壌の汚染防止等に関する法律」などがある。次に触れる雪印乳業の事件で、大阪府警は、大樹工場と大阪工場の前工場長ら製造部門の7人については、刑法211条違反とならんで、食品衛生法違反の疑いで、大阪地検に書類送検している(32)。b.刑事法の問題点 刑事法の社会統制機能に関しては、二つの問題が指摘できる。一つは、刑法211条の「業務上過失致死傷罪」のように「伝家の宝刀」的なものではあるが、その運用が難しいものがあることである。いまひとつは、各省などの行政機関が執行すべきものとされる個別の法律に定められている罰則の多くについて、適用に慎重になりすぎ、義務の不履行が放置されることになっていたり、低額の罰金刑であるため適用されても抑止力が不十分であったりすることである(株)。第1点については、2000年夏、1万3420人が発症し、うち1人が死亡した雪印乳業の乳製品による食中毒事件が想起される。統計的にみる限り、刑法211条が適用される事件の大半は交通事故である。しかし、数の上では少ないが、医療ミスで患者が死亡したり、工場事業場での爆発で死傷者がでたり、あるいは、食品によって人の健康被害が発生したりした場合にも「後始末としての悪者探し」がなされることはあり得る。雪印事件では、大阪府警は、前社長、前専務、大樹工場と大阪工場の前工場長など製造部門の7人を業務上過失致死傷罪の疑いそ、大阪地検に書類送検した。刑法211条の条文は抽象的であるだけに、あらゆる事故について適用される可能性がある。その場合、大きな組織の一員として行為をした多くの者の誰に「犯罪者」の汚名を着せるべきかが難しい問題となる。誰を処罰すべきかが最終的に確定されるまでには様々なプロセスがある。まず、警察が捜査の対象とするかどうかという判断をし、次に、検察官が捜査の結果を踏まえて起訴すべきかどうかという判断をし、更に「刑事訴訟」において裁判官が、検察官と「被告人」の双方の言い分を聞いた上で、最終的に判断することになる。雪印乳業の事件についても結末は予断を許さない。「犯罪者」という汚名を着るという形で誰かが責任をとるべきなのか。被害者側は、誰も処罰されないのは「正義」に反すると考えるかもしれない。しかし、誰かが処罰されるべきであるとした場合でも、責任をとるべき者が、組織のトップである「社長」なのか。あるいは、技術者のトップである「工場長」ないし、現場の誰かなのか。更には複数の人間なのかは実に難しい問題である。いずれに決めても割り切れない面が残るように思われる。c.民事責任と刑事責任の分化 田中成明は、「社会統制機能」を有する法には、「刑法」と並んで「不法行為法」があるとしている。確かに、刑法と不法行為法は、いずれもルール違反をした者に対して「サンクション」を与える点では共通している。刑罰をおそれて人々がルール違反をしないようにするのと同様に、損害賠償責任を負わされることを避けようと人々が行動するのであれば、〈注意深く行動せよ〉とかく危険な物を管理支配するものは安全性を高める措置につとめよ〉というルールが「社会統制機能」を果たしていることになる。しかし、歴史的にみれば、一般には刑事責任と民事責任(不法行為責任)は未分化の状態にあったものが、次第に制度的に分化し、それぞれ異なった役割をもたされるようになっている(34)。すなわち、刑事責任は、違反行為者の社会に対する責任を問うものであり、その責任は国家によって追及され、反社会的行為の行為者に対する社会全体からの非難・制裁であるのに対し、民事責任(不法行為責任)は、違反行為者の被害者個人に対する責任を問うものであり、そこでの責任は被害者自身によって追及され、被害者に生じた損害を填補させることを目的とする。このような「刑事責任と民事責任の分化」は、「大陸法系」の法制度で顕著であり、日本でも近代化の過程でこのような考え方が採用され、今日に至っている。したがって、刑事責任を追及するかどうかは、原則として、検察官が、警察の捜査を踏まえて、加害者の反社会的な悪性の度合いや他の類似の事件とのバランスを考慮して決定する。被害者が加害者の処罰を望むかどうかとは直接の関係はない。ただ、近年は、被害者への配慮も必要であると意識されるようになっている。東名高速道路での酒酔い運転によって追突事故を起こし、子供2人を死なせ、道路交通法違反と、業務上過失致死罪という二つの犯罪についての責任が問われたが、検察官の求刑は懲役5年、地方裁判所及び高等裁判所の判決で懲役4年にとどまった例が新聞で報じられた(株)・16のは記憶に新しいところであろう。これに対し、民事責任を追及するかどうかは、被害者の自由である。民法は、不法行為に基づく損害賠償請求とそれに応じる支払についても、当事者間の自由な交渉と合意によってなされることを原則として想定している。加害者が支払いを拒否し、あるいは金額について折り合いがっかなかったときに、被害者がなお自己の意思を貫こうとする場合は、裁判所の判断を求めることになる。これが「民事訴訟」である。交渉が原則、訴訟は例外であり、刑事責任の追及の場合は、裁判所の判断を求めるプロセス(これが「刑事訴訟」である)が必要であるのと対照的である。なお、現在、多くの自動車事故の後始末としての損害賠償に関する交渉は保険会社によって代行され、「民事訴訟」も保険会社が被害者や加害者に代わって原告・被告となるケースが多い。以上で、田中成萌の指摘する第1の「社会統制機能」の説明をひとまず終え、第2の「活動促進機能」に移る。・
