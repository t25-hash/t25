# 1・3・4 行政機能及び行政法の機能

> **章**: 第1章 法 と 技 術
> **節**: 1・3 法の機能

a．行政活動の存在意義 行政法は，行政の組織及び活動に関
するルールである．行政活動の第一の存在意義は，民間の活動の監
督である．すなわち，民間の活動のあるものは，事前に行政機関の
許可や認可を受けなければ行うことができないものとされる．ある
いは，行政機関は，民間の活動の続行を禁止したり，あるいは特定
の設備を備えるよう命令したりする．また，特定の機械は，検査・
(C)日本機械学会
検定に合格しなければ使用してはならないものとされる．
これらの「公的規制」については，近年「規制緩和」がどの程度
なされるべきかという文脈で論じられることが多かった．行政法学
者である曽和俊文によれば，次のような整理が可能である（46）．
1990年代の日本は「改革の時代」であった．とりわけ，「行政改革」
及び「規制緩和」が重要な課題であるとされ，現在進行中の「司法改
革」もこれらの諸改革の仕上げとして位置づけられている．
司法改革の理念を表すスローガンのひとつにく行政による事前規
制から司法による事後規制へ〉というものがある．行政機関が「行
政指導」に代表されるような不透明・非公式な手法で利害調整を行
い，事前に紛争を防止するようなシステムは破綻しつづあるとされ
る．また，過剰な規制が産業の活力を殺いでいる．そこで，今後は，
自己責任と競争原理に基づく自由で公正な社会が整備されるべきで
あるとする．従来の社会が事前規制型社会であるという現状認識が
正しいかどうかには異論もある．行政の介入は事前にも事後にも見
られるし，事前規制中心といいながらも，対応が後手後手に回るこ
とも少なくないからである．しかしいずれにしても，今回の司法改
革が，21世紀の「この国のかたち」についての一定のイメージに基
づいて構想されているということは確かである．
2001年6月12日に公表された司法制度改革審議会の意見書は，政
治改革や行政改革：，そして今回の司法制度改革の根底にある考え方
を次のように説明している．
〈これら諸々の改革の根底に共通して流れているのは，国民の一
人ひとりが，統治客体意識から脱却し，自律的でかっ社会的責任を
負った統治主体として，互いに協力しながら自由で公正な社会の構
築に参画し，この国に豊かな創造性とエネルギーを取り戻そうとす
る志であろう〉．
また，司法の役割については，〈法の支配の理念に基づき，すべて
の当事者を対等の地位に置き，公平な第三者が適正かつ透明な手続
により公正な法的ルール・原理に基づいて判断する司法部門が，政
治部門と並んで，『公共性の空間』を支える柱とならなければならな
い〉と述べている．
以上のような将来の社会において，行政法の役割にはどのような
位置づけがなされるのであろうか．〈行政による事前規制から司法に
よる事後規制へ〉というスローガンを表面的に理解するならば，こ
れからは行政法よりは民法や刑法が重視されるようにも受け取られ
かねない．
確かに，規制緩和や規制の廃止により，従来の経済規制は縮小す
る傾向にある．国や公共団体が担ってきた公的機能や公共的事業の
民営化も進むことになろう．民営化の拡大は，民法や商法などの私
法秩序による統制領域を拡大することになる．
また，紛争は，最終的には裁判所によって解決されることを前提
として，司法制度改革においては法曹人口の大幅な増大とそのため
の制度的手当てが議論されている．法科大学院でも基幹科目として
民法・商法や刑法が重視されている．
このような動きは，民刑事法の役割を今まで以上に大きなものと
することは間違いない．しかし，これは行政法の役割が縮小するこ
とを意味するものではない．アメリカにおける規制緩和に関する議
論で明らかになったように，「規制緩和」は，同時に「新しい社会的
規制」を必要とする面があるからである．
〈自立した個人が，自己責任と社会的責務に基づき，自由に社会活
動を展開し，国民相互の紛争があれば，独立性と公平性の保障され
た司法部門が公正な法的なルール・原理に基づき紛争を解決する社
会〉が実現するためにはいくつかの前提が必要である．
第1に，現実の社会においては，未成年者や要介護老人やハンデ
ィキャップをもつ人や病人や絶対的貧困者など，自己の努力だけで
は「自立」できない一群の人々が存在する．これらの人々が「自立
した個人」として社会に参加できる条件を整備するのは国家あるい
は社会の責務である．この意味で「社会国家」「福祉国家」の理念は
維持されなければならない．
