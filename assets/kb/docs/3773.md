# 1・2・2 酸性雨の実態

> **章**: 第1章 地球規模の環境問題
> **節**: 1・2 酸性雨問題

酸性雨問題（acid rain problem）は最初に産業革命がなされた
ヨーロッパで始まり，次いで北アメリカ北東部，さらには日本，中
国等を含む東アジアで問題となりつつある．図1･3 は環境省の酸
性雨対策調査による日本の酸性雨の状況を示している．1･2･1項で
述べたように環境の酸性化問題を考えるうえで硫酸塩，硝酸塩の大
気中濃度，乾性沈着量も併せて見る必要があるが，ここでは紙幅の
関係で雨の酸性度のみを示している．日本においてはヨーロッパ東
部，北アメリカ東部に比べてpHはやや大きいものの，西日本から
中部日本にかけてpH4.7程度となっている．
図1･4 にヨーロッパ，北アメリカ，中国の酸性雨の状況を示し
ている．ドイツから東ヨーロッパ・北ヨーロッパにかけてpH4.5
以下の地域，アメリカの北東部とカナダ南東部との国境付近に
pH4～5の地域が大きく広がり，越境汚染として問題になってい
る．中国内陸部の工業地帯にはpH5以下の地域があり，今後産業
の発展に伴い硫黄分の多い中国産石炭の使用量が増大すると，中国
内の酸性雨問題と併せて風下側の日本や韓国への影響が懸念され
る．これを未然に防ぐためには，発生源での脱硫等の対策が不可欠
である．
