# 1・3・5 資源配分機能

> **章**: 第1章 法 と 技 術
> **節**: 1・3 法の機能

a.マクロの視点 法の第4の機能である「資源配分機能」は,すでにみた「社会統制機能」「活動促進機能」「紛争解決機能」とはやや次元の異なるものである.田中成明は,現代国家の特徴と関連づけて説明している.すなわち,現代国家が社会経済社会に広範かつ積極的に配慮・介入するにつれて,法が,経済活動の規制,生活環境の整備,教育・公衆衛生などに関する公的サービスの提供,社会保障,各種の保険や租税による財の再配分などの手段として広く用いられるようになってくる点に着目し,これらの領域における法の機能として「資源配分機能」を語っている.このような捉え方によれば,「資源配分機能」を有する法が多くみられるのは,行政法や租税法のほか,労働法,社会保障法,経済法,さらには公害・環境法や消費者保護法などの領域ということになる.β9-22他方,アメリカで有力な考え方となっている「法の経済分析」によれば,あらゆる法制度は,資源の最適配分という経済学的側面から分析可能であるとされるようである(代).たとえば,カラブレイジという学者が,不法行為法を「資源配分機能」から分析し,過失責任主義よりも無過失責任主義が政策的にすぐれていると主張したことは日本でも広く知られている(58)~(60).いずれにしても,法の「資源配分機能」に着目する場合は,あるルールが社会全体にとってどのような影響を与えるかというマクロの視点が表にでることは確かである.個別の過去の紛争をどのように解決するのが「正義にかない公平か」というミクロの視点は,法的思考のイメージの原型であるが,現代社会における法的思考はそれよりもはるかに広いパースペクティブをもっているのである。b.技術基準 社会的規制のあり方を考える際に重要な意味をもつのが「技術基準」である.「どの程度安全であれば十分安全であるか」というレベル設定が,一般的抽象的な形でなされる場合,それは社会全体に大きな影響を与える.また,わが国においては,技術基準の大部分が行政機関によって定められてきたが,近年の国際化のなかで揺らいできている.先に触れたように,法令の体系において「行政機関の定めるルール」は大きな比重を占めているが,現代国家においては「行政立法」の氾濫が不可避であるとされている.「行政立法」の概念は多義的であるが,広い意味では,行政機関によって定められる一般的なルールをさし,狭い意味では,そのうちで「法律」と同様に,国民に対して法的拘束力を有するものをいう.行政法学は,行政を法的に統制するための理論体系であるが,そのような視点から,ドイツの理論に従って,広い意味の「行政立法」を「法規命令(Rechtsverordnung)」(狭い意味の「行政立法」)と,「行政規則」(Vbrwaltungsvorschrifte n)」に区別するのが一般的である.現行法制において,どのような行政機関にどのような「行政立法」(広義)の「権限」が与えられているかを概観しておこう.この問題は「法規命令」については特に重要である.日本国憲法は41条で〈国会は国権の最高機関であり,国の唯一の立法機関である〉と定めている.したがって,国の行政機関は,国会によって「権限」を与えられない限り,憲法にいう「立法」をすることはできない.憲法にいう「立法」とは,一般には「法規」(Rec:htssatz)という特定の内容の法規範,すなわち,国民の権利を直接制限し,又は義務を課すルールを定立する作用であると解されている.そこで,「法規命令」の定立は,以下のように,法律によって「権限」を与えられた行政機関がその「授権」の範囲内でのみなし得ることになる.第1に,「政令」について,憲法73条6号は〈この憲法及び法律の規定を実施するために政令を制定すること〉を内閣の行う事務のひとつとして定めている.また,ただし書で〈政令には,特にその法律の委任がある場合を除いては,罰則を設けることができない〉とされているが,内閣法11条はこれを受けて〈政令には,法律の委任がなければ,義務を課し,又は権利を制限する規定を設けることができない〉と定めている.以上によって,内閣は,法律を執行するための「執行命令」について,一般的に「授権」されている他,罰則を設け,義務を課し,又は権利を制限する規定を内容とする「委任命令」についても,個別の法律によって「授権」されることが予定されていると解される.第2に,「総理府令」「省令」について,国家行政組織法12条1項は〈各大臣は,主任の行政事務について,法律若しくは政令を施行するため,又は法律若しくは政令の特別の委任に基いて,それぞれその機関の命令(総理府令又は省令)を発することができる〉と定めていた.2001年1月6日半ら発足した新たな中央省庁体制においては,「総理府令」に代わって「内閣府令」が登場した.すなわち,内閣府設置法7条3項はく内閣総理大臣は,内閣府に係る主任の行政事務について,法律若しくは政令を施行するため,又は法律若しくは政令の特別の委任に基づいて,内閣府の命令として内閣府令を発することができる〉と定め,平成11年法律90号によって改正された国家行政組織法12条1項は〈各省大臣は,主任の行政事務について,法律若しくは政令を施行するため,又は法律若しくは政令の特別の委任に基づいて,それぞれその機関の命令として省令を発することができる〉と定めている.また,国家行政組織法12条3項,内閣府設置法7条4項は,内閣法11条と同様の定めをおいている.以上によって,内閣総理大臣及び各省大臣は,法律又は政令を執行するための「執行命令」について,一般的に「授権」されている他,罰則を設け,義務を課し,又は権利を制限する規定を内容とする「委任命令」についても,個別の法律又は政令によって「授権」されることが予定されていると解される.第3に,「外局規則」について,国家行政組織法13条1項は〈各委員会及び各庁の長官は,別に法律の定めるところにより,政令及び前条第1項に規定する命令以外の規則その他の特別の命令を自ら発することができる〉と定めている.内閣府設置法58条4項は〈各委員会及び各庁の長官は,法律の定めるところにより,政令及び内閣府令以外の規則その他の特別の命令を自ら発することができる.〉と定めている.また,国家行政組織法13条2項は同法12条3項を,内閣府設置法58条5項は同法7条4項を準用して、いる.「行政規則」は「法規命令」とは異なり,憲法にいう「立法」を内容とするものではない.行政組織の内部では立派なルールで,公務員はこれに従わなければならないが,国民の権利義務を直接左右するルールではない.したがって,行政機関は,法律による「授権」がなくても,それを定める「権限」を有する.「行政規則」を制定する「権限」の根拠は,一般には行政組織法論上の「指揮監督権」又は憲法65条に求められる.国家行政組織法14条2項は,〈各省大臣,各委員会及び各庁の長官は,その機関の所掌事務について,命令又は示達するため,所管の諸機関及び職員に対し,訓令又は通達を発することができる〉と定めているが,これは確認規定に過ぎない.大臣,委員会,長官だけではなく,局長,課長なども一般的な条文の形式をもつ通達を発する権限を有する.また,「行政規則」の形式は,「法規命令」の場合のように,法律で定められたものに限定されない.したがって,「通知」「回答」「規程」「要綱」など様々な名称で呼ばれる.現代行政国家において行政は多種多様な行政立法を制定している.国レベルでは,政令を頂点として,総理府令(内閣府令)や省令,委員会や外局が制定する規則などがある.また,行政規則まで含めると,通達・訓令さらには行政処分に関係する審査基準(行政手事法5条)や処分基準(同法12条)などもある.これら各種の行政立法の総数は,国会が制定する法律の数をはるかに上回っている.一般にどの行政法規のもとでも多数の行政立法が制定され,日々制定されるこれら多数の行政立法が行政の様々な決定や行動のよりどころとなっている.現代国家では行政立法を見ずして行政活動の実態はわからないとさえいえるのである.憲法41条には,国会によって「立法」がなされるのが原則で,行政機関によって「立法」がなされるのは例外であるとの考え方が示されている.しかし,原則と例外が逆転しているのが現代国家の実態である.法律は大枠を定めるにとどまり,行政機関がその枠内で,自ら国民の権利義務に関する一般的な規範を定め,個別ケースにそれを適用するということが多くなっているのである.古典的な三権分立のモデルによれば,行政活動は「法律の機械的執行」であるべきとされる.しかし,現代国家においては,このような考え方を復活させて議会の権限を拡大することには限界がある.科学技術的な事項を含む複雑な利害調整を行うためには膨大な量の一般的なルールが必要であり,議会はその機構・能力の限界により,そのごく一部を取り扱うことしかできないからである.そこで,高度産業社会における国家機能の増大は,一般には行政活動の比重を高めることにつながり,「行政国家現象」が語られることになる.複雑な社会を管理するためには,行政機関の活動自体が複雑なプロセスをとらざるを得ないことは明らかである.以上によって,「行政立法論」は,議会の機能に限界があり,かっ行政立法が不可避であることを前提に,行政機関が「立法」を行うことによって生じる「法治主義」ないし「法律による行政の原理」からみた問題点にどう対処するかという形をとることになる(6D.c.リスクのレベル設定 社会的規制のあり方を考える場合には,〈リスクのレベル設定を行う権限は誰に与えられるべきか〉〈リスクのレベル設定はどのような手続で行われるべきか〉〈リスクのレベル設定によって誰にどのような法的義務が課されるのか〉〈リスクが顕在化した場合に,誰がどのような責任を負うのが公平か〉,そして,〈設定されたリスクのレベルが社会において尊重され,多くの当事者の行動に影響を与えることがどのように正当化されるのか〉〈そのような制度の運用,ルールの実現は誰の責任において行われるべきか〉が主たる関心となる(代).「社会的規制」の緩和を検討する際の第一のポイントは,リスクと規制レベルの均衡である.すなわち,リスクに比べて規制が過剰であると,リスク低減の便益よりも規制コストが大きくなってしまう.過剰な規制は望ましくないので,規制緩和がなされるべきであるという一般論を否定する者はいない.しかし,それぞれの規制を具体的に検討して過剰か否かを論じると見解が分かれることの方が多い.リスクをどう評価するか,そして,リスクを低減するためにどの程度の規制コストを負担すべきかが,いずれも検討の余地の多い問題であるからである。まず,リスク評価は,科学的に判断される問題であると一応考えられるが,「環境ホルモン」問題にみるように情報が不完全であるため,不確実性が大きい.そこで,たとえば,〈技術リスクの横断的管理目標は設定可能か〉を問題にする場合には,その前提として,多種多様なリスクについての相互比較が可能な程度にそれぞれの評価手法が確立しているのかが問われることになろう.次に,どの程度までリスクを低減すべきか,というレベル設定は政策判断の問題である.したがって,「社会的」に決定されるべき事柄であるが,どのような形で決定するかという決定態様の問題と,それを誰がどのような手続で決定するかという決定主体の神叩が絡み合ってくると考えられる.第1の「決定態様」の問題は,個人あるいは集団の発がんリスクを〈Aかける10のマイナスx乗〉というような形で直接的ないし定量的に定めるか,〈技術的に実現可能な限りで〉というような形で間接的ないし定性的に定めるかなどの選択である.また,この選択は,設定されたレベルを,達成されることが望ましい目標として扱うのか,直接に事業者等の活動の許容性に影響する「規制基準」として扱うのかという選択と絡み合っている.たとえば,発がんリスクを定量的に定め,それを「規制基準」として扱う場合は,表面上は,それを達成するためのコストを度外視してレベル設定を行い,それを達成できない事業活動は停止するしかないという政策判断を同時に行っていることを意味する.しかし,リスクのレベル設定が社会的に影響の大きい事項である以上,決定主体が技術的実現可能性等の要素を無視して判断し得るかは疑問である.なお,環境保護の領域では,環境基準と排出基準のどちらに重点を置くかという問題があり,行政機関による実効性をもった執行という観点からはシンプルな排出基準によるほうが結果的には環境改善のパフォーマンスに優れるという見解もある.したがって,「政府の失敗」を考慮にいれて規制のあり方を検討する場合,そもそもリスクのレベル設定という形をとるべきかという論点もあるように思われる.更に,多種多様なリスクについて横断的に定めるか,リスクの種類によって異なるレベル卜定をするのかという選択がある.リスクの原因たる活動,リスクが及ぶ範囲,リスクに関する理解などによβ9-23って,個々人の「受容」の度合いが異なることから,横断的設定には困難がつきまとう。他方,個別的なレベル設定だけでは,規制を受ける者が,自分は他の者と比較して過剰な規制を受けており「社会的公正」が阻害されていると不満を持ち,「受容」を拒むことが多くなるであろう。したがって,より多くの社会構成員の「受容」をめざすという観点から,一定の類型化をめざすのが穏当ではないかと思われる.第2の「決定主体・手続」の問題は,事前にレベル設定を行うにしても,それを行政機関が「規制基準」として設定して,許認可等の「強い」規制を行うのか,それとも,民間の「自主基準」に委ねて,後は事後的な損害賠償訴訟,保険などでカバーする「ルール型規制」を行うのかという選択である.ほかに,環境規制に関して提唱されている「排出課徴金」や「排出権取引」のように,基準に事業活動を直接コントロールする機能を与えるのではなく,間接的に影響を与える機能を与えるにとどめるという選択もある.規制緩和のためには,代替的な仕組みの整備が重要である.行政による事前のコントロールから,ルールを設定してその違反に対して制裁を課す事後的コントロールへ比重を移すべきであるという提言が一般的に受け入れられつつあるが,「自己責任の原則」が妥当するためには,供給サイドで競争が機能していること,また,消費者や労働者が取引をする際に十分な情報が与えられていることが前提であり,また,ルール違反をとがめるためのルール(損害賠償,刑罰,行政的制裁などのあり方についてのルール)の整備や,司法機能の充実が不可欠である.なお,行政機関が「規制基準」を設定する場合に,デザイン、(設備・仕様)基準とするか,パフォーマンス(性能)基準とするかという選択肢がある.この選択は,規制手段の費用効果性という規制緩和を検討する際の第2のポイント,すなわち,同じ規制目的を達成するためには,できるだけ規制コストの小さな手段を用いるべきであるという原則に係わるが,行政法的思考からは,パフォーマンス基準の許容性については一定の条件が備わっている必要がある.法的思考においては「正当化」というタームが頻繁に用いられる.「社会的規制」のレベル設定については,憲法的思考ないし行政法的思考によれば,議会制民主主義という仕組みが出発点となるので,社会的な影響の大きい事柄については,まず,国会の「権限」を重視すべきことになろう.次いで,行政機関が国会から「委任」を受けて,レベル設定を行う「権限」を有することになる場合に,それを「正当化」するためには,単に「行政の専門技術性」を説くだけでは不十分であると考えられている.そこで,「手続による正当化」が必要となり,審議会やパブリックコメントなどを通じた「情報収集」ないし「参加」という視点が強調されることになる.しかし,「技術基準」の領域では,「組織化された工学的知識」という要素が付加されることによって,かろうじて,その中に含まれる政策的判断が社会において尊重され,多くの当事者の行動に影響を与えることが「正当化」されるように思われる.たとえば,規制の態様として「パフォーマンス基準」の考え方を推し進めると,行政法規の規定はかなり抽象的なものとならざるを得ないことが指摘されている.そこで,政省令レベルでも〈○○は十分に安全でなければならない〉といったようなものにとどめられ,具体的にどのレベルが「十分に安全」なのかは関係業界の技術者の相場観によって定まるという状況を想定すると,行政機関の「権限」が「私人」に白紙委任されているのではないかという疑問が生じるのである.この場合,民間の「自主基準」などが,正確な情報を基礎にして,かつ,多様な意見を集約するような眠主的」かつ「公正」な「手続」を経て策定されたものである場合に限って,上記の疑問に答えることができるように思われる鰯.【参考文献等】(1)高木
