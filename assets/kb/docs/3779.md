# 2・2・1 環境法とは

> **章**: 第2章 製品に係わる法制度
> **節**: 2・2 環境法

a.環境法の定義i.環境 「環境」とは、広義では、一般に、人間又は生物を取り巻く周囲の状態や世界をいうが、環境法の対象となる「環境」とは、広義の環境のうち、汚染や悪化により、紛争の原因となるおそれがあり、国家が何らかの対策をとるべきものをいう8.ii.環境問題(1) 日本 一公害問題から環境問題ヘー 日本では、明治政府による富国強兵・殖産興業政策の結果、工業が成長し、公害が発生したといわれている。すなわち、1879年頃から、足尾銅山からの煙により山林被害、農作物被害が発生し、山林枯死による大洪水や鉱毒水による健康被害までひきおこされ、田中正造代議士らが対策活動を行った(足尾銅山鉱毒事件(9り。19世紀終盤頃からは、東京や大阪などで大気汚染・水質汚濁などが発生した。大阪では瓦工場や紡績工場からのばい煙が深刻化して大阪府令煤煙防止規則の制定につながり、1891年には工場の排出する亜硫酸ガスなどによる稲作被害について訴訟が起こされたqo)。第二次世界大戦により日本産業は壊滅的打撃を受けたが、いわゆる朝鮮特需などもあり急速に回復し、産業優先の政策下、全国各地で深刻な健康被害が発生した。その典型が四大公害病、すなわち熊本水俣病、新潟水俣病、富山イタイイタイ病、四日市ぜん息である。四大公害病をはじめ各地の公害については訴訟が提起され、患者側勝訴の判決が続いたqD.しかし、訴訟による解決はあくまでも事後的救済に過ぎず、健康被害の不可逆性を思えば、公害問題を根本的に解決するためには事前予防が必要であることは明白であった。そこで、1970年のいわゆる「公害国会」において、計14の関係法律の制定・改正が行われ、1971年には環境庁が設置された。公害の関係法律の整備と第一次オイルショックのため、1970年代半ば以降、公害問題は潜在化し、四大公害病ほど世間の耳目を集める状況ではなくなった。かわって、良好な自然環境を乱開発から守置り、快適な住環境の整備を推進することに人々の関心が向けられるようになり、「公害問題」よりも広い概念として「環境問題」が意識されるようになった。環境問題の特徴としては、その目標が多岐にわたること、原因を生む者が多様化していること、紛争形態が多様化していることなどを挙げることができる。すなわち、目標としては、有害な化学物質の乱用防止や公害防止以外にも、良好な自然環境を保っている地域の保護、希少な野生生物等の保護、歴史的風土の保護、快適な住環境の整備、廃棄物処理やリサイクルの推進のように、必ずしも人体に対する直接的被害を想定しないものも多い。また、原因についても、工場からの排出物に限らず、公共事業の実施や家庭からの排出物も環境問題の原因になっている。さらに、紛争形態も、四大公害病のように事後的に損害賠償を求めるだけではなく、差止訴訟を起こして事前に環境問題の発生を予防するケースが増加している。(2) 以上に述べたように、環境問題は多様化しており、環境問題に関するルールである環境法にもさまざまなものが存在するから、その定義にもさまざまなものが考えられる。ここでは、環境法とは、1)既に良好な状態にある環境の保護を推進すること、又は■)必ずしも良好な状態にあるとはいえない環境を人工的に良好な状態へと誘導することを目的とする法の総称をいう、と考えることにするq3)。なお、民法分野や刑法分野において、「民法」「刑法」のように、そのものずばり同じ名前の法律が存在するのと異なり、環境法の分野においては、「環境法」という名前の法律は存在しない。b.環境法の分類 既に述べたように、環境の定義にもいろいろあり、環境法に含めて考えることのできる法令等もさまざまなものがある。そして、その分類も、いろいろな角度から行うことができるが、ここでは、まず第1に、何を保護するか、という内容面での多様性に注目して分類し、第2に、どのようなスタイルのルールがあるか、という形式面での多様性に注目して分類する。i.内容面での多様性に注目した分類の例 活動単位の組合せによる行政システム 行政機関は、現実には以上に述べたような典型的な活動単位を組み合わせて実施することにより活動しており、そのような行政活動は、いわば“行政システム”と呼ぶこともできよう。例えば都道府県知事が、水質汚濁防止法第13条に基づいて、排水基準に適合しない排出水を排出するおそれがある者に対して、改善命令を下すことを検討したが、その前に行政指導をして自発的な改善を促すなどが考えられる。それはいわば“水質汚濁防止のための行政システム”といえる。“行政システム”のほかの例として、行政機関がいわば事業者となって、環境障害防止事業を行うというものもある。環境障害防止事業は、個別の法律の根拠がなくても行うこともできるが、実際には個別の法律に基づく行政計画にしたがって実施される事業が多い。例えば、自然環境保全法による保全計画に基づく保全事業(自然環境保全法第16条等)、下水道法による事業計画に基づく公共下水道整備事業(下水道四阿4条参照)、環境基本法による公害防止計画に基づく公害防止事業(環境基本法第17条参照)、農地土壌汚染防止法による土壌汚染対策計画に基づく客土事業(農地土壌汚染防止法第5条)などがある。環境障害防止事業の費用負担の考え方については、おもに二つの原則がある。原因者負担の原則(環壌基本法第37条・自然環境保全法第37条、公害防止事業費事業者負担法第2条の2等)および受益者負担の原則(環境基本法第38条、自然環境保全法第38条等)である。以上このci項において述べたところを図示すると、例えば図2・9のようになる。ii.紛争化した後における行政機関の活動(1) 行政機関の活動単位 紛争が発生した後においても、行政機関が、例えば命令行為に該当する行政処分を行ったり、その前に規制的行政指導を行ったりすることがある。また、行政上の契約として公害防止協定や環境保全協定などの協定を締結したり、「協定」の名のもとに私人同士の間の協定を行政機関が認可したり、紛争当事者双方に対して調整的行政指導をするなど、さまざまな活動を行うことは珍しくない。(2) 活動単位の組合せによる行政システム 行政機関が典型的な活動単位を組み合わせて実施していることは、i(3)項でも記したが、このことは紛争化した後であっても同様である。そのような“行政システム”の例として、1義務者たる私人が行政上の義務の履行をしないときに、行政機関が自らの手で義務の履行の実現を図る制度、2行政罰の制度、3違法又は不当な行政処分の取消しその他の是正を、私人が行政機関に対して求め、行政機関が判断を下す制度、4「公害紛争処理法」に基づいて行政機関が行う公害紛争処理の制度、5「公害健康被害の補償等に関する法律」に基づいて行政機関が行う療養費支給等の制度などが挙げられる。これらのうち、4,5は公害紛争に特有のものであるが、1,2,3は公害紛争に限らない(株)。1の“義務者たる私人が行政上の義務の履行をしないときに、行政機関が自らの手で義務の履行の実現を図る制度”には、大別して、(イ)代執行、(ロ)直接強制、(ハ)執行罰、(二)行政上の強制徴収、の四種類がある。(イ)の代執行とは、私人の側の義務が履行されないときに、行政機関が自ら義務者のなすべき行為をするか、又は第三者にこれをなさしめ、かかった費用を義務者に支払わせる制度をいう。一般法として行政代執行法が存在するが、建築基準法にも関係規定がある(建築基準法第9条第12項)。(ロ)の直接強制とは、義務者の財産等に直接力を行使して、義務の履行があった状態を実現するものである。(ハ)の執行罰とは、義務の不履行に対して一定の過料を課すことを通告して間接的に義務の履行を促し、それでも義務を履行しないときにこれを強制的に徴収する制度である。罰という言葉が用いられているが次に述べる2の行政罰ではない。(二)の行政上の強制徴収とは、租税債権などについての行政機関独自の徴収制度をいう。次に、2の“行政罰の制度”とは、行政上の義務を怠った者に対して制裁をすることをいう。行政罰には(イ)行政刑罰と(ロ)行政上の秩序罰がある。(イ)の“行政刑罰”とは、例えば水質汚濁防止法によると、排出基準に適合しない排出水を排出した場合、6ヶ月以下の懲役又は50万円以下の罰金に処されるおそれがある(第31条)、というように、個別の行政法規において懲役・罰金等の制裁が定められているものをいう。(ロ)の“行政上の秩序罰”とは、例えば水質汚濁防止法によって、一定の届出義務違反に対して10万円以下の過料に処されるおそれがある(第35条)、というように、個別の行政法規において過料という制裁が定められているものをいう。3の“違法又は不当な行政処分の取消しその他の是正を、私人が行政機関に対して求あ、行政機関が判断を下す制度”として、「行政不服審査法」に基づく行政不服申立て制度がある。これは、環境に関するものに限らず、違法又は不当な行政処分の取消しその他の是正を、行政機関に対して求める制度である。行政機関の活動の是正を行政機関自身に求めることから、公正さの点では裁判に及ばないが、違法ではないが不当だという行政処分も争うことができ、比較[紛争化前]住民等等定認可等行政機関協定締結等事業者等行政行為、協定締結、行政指導等の組合せ(環境障害防止事業等)図2・9 環境法に関するおもな行政活動および訴訟(紛争化後)題とせず、集団現象全体の因果関係を統計的に推計するもので(24)、イタイイタイ病事件や四日市ぜんそく事件などの判決で採用された考え方である(25)。なお、国際環境問題においては、例えばA国の国内で企業aが流した鉱毒により、B国の農産物を枯死させた場合、どこの国の法律を適用するか、ということが問題となる(株)。日本の通説では、B国法を適用するべきであると考えられている(27)。2 第2に、被害者が国又は公共団体を訴えて金銭での解決を図る類型がある。すなわち、国家賠償法を根拠とする国家賠償請求訴訟である。この訴訟は、行政機関が規制権限を適正に行使せず、又は各種の活動により国民の権利を侵害した場合などに起こされ、その意味で、おもに環境障害が現実化し、かつ紛争化もしているときの紛争解決手段である。3第3に、被害者が事業者を訴えて問題行為をやめさせる類型がある。すなわち、所有権など「物権」と呼ばれる強い権利又は人格権を根拠とする差止訴訟である。環境障害の発生を防止する局面でも、発生後の環境障害を除去する局面でも用いられる。なお、2・止請求を認めていない(株)。4 第4に、被害者が行政機関を訴えて問題行為をやめさせる類型がある。この類型はさらに四つに分けることができる。(イ)行政処分の取消訴訟、(ロ〉行政処分の無効等確認訴訟、(ハ)行政機関の不作為(何らかの処分をするべきなのにしないこと)の違法確認訴訟、(二)普通地方公共団体の長等が違法または不当な公金の支出や財産の管理、処分を行ったときに、住民監査請求を経て提起する住民訴訟、の四つである。(イ)、(ロ)及び(ハ)は「行政事件訴訟法」(同法に定めがない事項については「民事訴訟法」)の規定に基づくもので、(二)は「地方自治法」の規定に基づくものである。いずれも、実際にはおもに環境障害が現実化し、かつ紛争化もしているときに利用されることが多い。しかし、例えば林地開発許可に基づくゴルフ場の開発工事の場合、工事が完了してしまうと、開発許可処分自体を争う取消訴訟は、訴えの利益がないとして却下されてしまう(29)ので、環境障害が現実化する前に取消訴訟が利用されることも増えると予想される。また、行政処分の無効等確認訴訟は、環境障害が現実化する前に利用された例もある。以上に述べたところ以外に、刑事裁判も存在する。公害に関する刑事裁判は大きく二つの類型に分けられる。行政罰に関するものと公害犯罪処罰法に基づくものである。行政罰としては、既にii(2)2項で述べたとおり、例えば水質汚濁β9-46防止法などの個別の法律に存在する罰則規定(行政刑罰)と行政上の秩序罰とがある。行政刑罰は刑事訴訟法に基づいて処理され、行政上の秩序罰は非訟事件手続法に基づいて処理される。法律違反に対する秩序罰は住所地の地方裁判所において科せられ、地方公共団体の条例・規則違反に対してはその長がこれを科すとされる。いずれも、原則として具体的な危険が発生しなくても処罰される(“抽象的危険犯”といわれる)という特徴がある。これに対し、公害犯罪処罰法に基づく刑罰は、公衆の生命又は身体が危険にさらされた場合に処罰される(“具体的危険犯”といわれる)のが特徴である。いずれにせよ、環境障害に関する行政罰や公害犯罪処罰法が実際に適用された例は多くない。(2)裁判以外の手続 裁判所においてなされるが「裁判」ではない手続として、「民事調停法」に基づいてなされる民事公害調停があることは既にii項で述べた。これは、申立により調停委員会で紛争の調整をし、当事者が合意した場合に調停が成立し、裁判上の和解と同じ効力がある、というものである。以上cii項およびciii項において述べたところを図示すると例えば図2・10のようになる。
