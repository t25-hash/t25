# 1・1・1 人工衛星理論の先駆

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・1 人工衛星開発の歴史と現状

ニュートンは1687年に刊行した『プリンキピア』の中で，「もし
山頂から大砲で鉛玉が水平に発射され，空気の抵抗がないとすれ
ば，発射の速度をどんどん増大した場合に，ついには地球に落下す
ることなく地球を一周し，宇宙空間で永遠に運動を続ける」と記述
している ．人工の物体を地球周回軌道に乗せる考えは，その後，
ジュール・ベルヌやツィオルコフスキーにも引き継がれたが，もと
もと，人間を無重力環境や地球外に送り出すことを想定したもので
あった．観測や通信にこのような物体を用いる発想は，エレクトロ
ニクスの発展を待たなければならなかった．1945年には，無線技
術者であったアーサー・クラークが静止衛星の予言を行い，わずか
3個の静止衛星で全世界を無線通信で結ぶ可能性を示した ．これ
らの先駆的考えは，具体的な実現手段がない段階での思考実験の結
果生まれたものである．その後まもなく実証されたロケット技術
は，人工衛星実現の具体的手段を提供することとなった．
