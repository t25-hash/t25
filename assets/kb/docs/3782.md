# 1・1・2 人工衛星概観

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・1 人工衛星開発の歴史と現状

1957年のスプートニク1号（Sputonik1）以来，ロケットやス
ペースシャトルは2001年末までに合計4185回打ち上げられ，こ
れにより約5400個の人工衛星や人工惑星が軌道に投入された．こ
れらのうち約半数は，地上への回収，あるいは大気層への落下消滅
によって，すでに軌道上には存在しない．2001年末では，地球を
周回する人工衛星の個数は約2800個，地球の影響圏を脱出し，太
陽を周回する惑星間空間，あるいは太陽系自体を脱出する軌道にあ
る探査機は約90個となっている．ただし，これらの多くはすでに
寿命が尽きており，機能しているものは600個ほどであると見られ
ている．地球を周回する人工物体には，人工衛星のほかに，使用済
みのロケット機体，ロケットや人工衛星の破片が存在し，人工衛星
のうちすでに寿命を終えたものを含めてスペースデブリと称されて
いる．地球を周回する人工物体のうち，地球上の観測によって軌道
が確定しているものは，2001年末には9226個とされている ．人
工衛星のトレンドを知るため，軌道別，目的別に年間の打上げ個数
を1992年以降の10年間について図1･1と図1･2に示す．
