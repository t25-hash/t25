# 3・1・1 労働安全衛生法の概要

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・1 事故・労働災害等を防止するための法律

a.労働安全衛生法の沿革および目的 労働者の安全と衛生の確保については,昭和22年に制定された労働基準法に定められていひたが,昭和30年代半ばからの経済の急激な成長とともに労働災害が著しく増加し,死亡者数も毎年6000人を超えるようになり,また,一時に多数の死傷者を出す重大な災害も多発した.このため,労働災害防止のための法整備についての社会的ニーズが高まった.労働安全衛生法はこのような背景の中で昭和47年に制定され,その後幾多の改正を経て現在に至っている.労働安全衛生法の目的は,その第1条に「労働災害の防止のための危害防止基準の確立,責任体制の明確化及び自主的活動の促進の措置を講ずる等その防止に関する総合的計画的な対策を推進することにより職場における労働者の安全と健康を確保するとともに,快適な職場環境の形成を促進する」と規定されている.労働安全衛生法はその目的を達成するため,主に事業者(事業を行う者で,労働者を使用する者)に対し必要な措置などを義務づけているが,このほか,機械器具を設計し,製造し,輸入する者や建築物の設計や建設する者に対しても,これらの物の使用や建設などに際して,労働災害の発生の防止に資するよう務めさせている.労働安全衛生法(以下「安衛法」という)は,災害防止についての基本的事項を定めているもので,その適用範囲や対象設備などは労働安全衛生法施行令(以下「令」という)に,また具体的な措置基準などは省令(各種規則など)で定められている.安衛法は全13章,160条余からなり,これに基づく具体的な措置や対策などについて,労働安全衛生規則では包括的な事項について定めているほか,ボイラー及び圧力容器安全規則,クレーン等安全規則などの特別規則,各種構造規格,各種指針などからなる膨大な体系の法令である.主な省令については3・1・2項で述べることとし,ここでは特に技術者にとっても関連の深いと思われる第3章から第7章,第9章中の労働安全(衛生)コンサルタントおよび第10章についてその概要を述べることとする.b.「安全衛生管理体制」(第3章) 労働災害の防止には,まず事業場における安全管理体制を整備することが重要である.このため,事業場の規模や業種や作業の種類に応じて総括安全衛生管理者,安全管理者,衛生管理者,産業医などの選任と安全衛生に係る事項を審議する安全委員会(又は安全衛生委員会)の設置が義務づけられている.その概要については3・1・2項aで述べることとする。c.「労働者の危険又は健康障害を防止するための措置」(第4章)労働災害を防止するため,事業者は次の危険源などに対する措置を講じなければならないとされており,具体的措置については労働安全衛生規則その他の各種規則などに定められている。1機械などによる危険,爆発性の物,発火性の物,引火性の物などによる危険および電気,熱その他のエネルギーによる危険を防止するための必要な措置(安衛法第20条)2掘削,採石,荷役,伐木などの業務における作業方法から生じる危険を防止するための必要な措置および墜落または土砂崩壊する恐れのある場所などに係る危険を防止するための必要な措置(安衛法第21条)3有害なガス,蒸気,粉じんおよび放射線騒音その他の有害環境などによる健康障害を防止するための必要な措置(安衛法第22条)4 労働者が就業する作業場について,通路,床面,階段などの保全ならびに換気,照明,保温等,休養,避難および清潔に必要な措置(安衛法第23条)5労働者の作業行動から生ずる労働災害を防止するための措置(安衛法第24条)また,上記の措置の適切かつ有効実施を図るため厚生労働大臣は技術上の指針を公表するとしている.現在までに「工作機械の構造の安全基準」,「ボイラーの低水事故防止」など14本の指針が大臣告示として出されている。d.「機械及び有害物に関する規制」(第5章)i.機平等に関する規制 定置式の特定機械等を設置した者,主要部分に変更を加えた者は労働基準監督署長の検査(落成検査,変更検査)を受けること.落成検査に合格した特定機械等については検査証が交付される.(安衛法第38条第3項,第39条第2項)4 検査証を受けていない特定機械等は使用し,譲渡し,または貸与してはならないこと。(安衛法第40条)5検査証の有効期間は機械等の種類に応じ1年または2年と定められており,その更新を受ける場合は,労働基準監督署長または厚生労働大臣の指定する者(性能検査代行機関)が行う検査(性能検査)を受けること.(安衛法第41条)(2) 規格を具備すべき機械等 その他の機械等 上記分類に属さない機械等は,検査,検定の実施または規格の具備などの規定はないが,動力により運転される物で,回転体などの作動部分に突起物がある物や,歯車やプーリーなどの動力伝達部分が防護されていない物は,譲渡し,貸与し,または譲渡もしくは貸与の目的で展示してはならない.(安衛法第43条)(4) 定期自主検査(安衛法第45条)1 ボイラーなど危険な機械等のうち令第15条第1項で定めるものについては,定期に(機械等の種類,部分などに応じて1月ないし2年以内毎)自主検査を行いその結果を記録しておかなければならない。2定期自主検査のうち令第15条第2項で定めるプレス機,建設機械などについては,一定の資格を有する者または登録を受けた検査業者に実施させなければならない.(特定自主検査)ii.有害物に関する規制 有害物による健康障害を防止するため,化学物質の人体への影響の程度などに応じて,以下のような規制がある.(1) 製造等の禁止 重度の健康障害を生じる物で令第16条で定めるもの(黄りんマッチ,ベンジジンなど9物質およびその含有物)は,製造し,輸入し,譲渡し,提供し,または使用してはならない.ただし,試験研究用とする場合は,あらかじめ都道府県労働局長の許可を受ける必要がある.(安衛法第55条)(2)製造の許可 重度の健康障害を生じるおそれのある物で令第17条で定めるもの(ジクロルベンジジンなど7物質およびその含有物)を製造しようとする者は,あらかじめ,厚生労働大臣の許可を受けなければならない.(安衛法第56条)(3) 名称等の表示 ベンゼンなど労働者に健康障害を生ずるおそれのある物で令第18条で定めるものまたは製造許可物質を譲渡し,提供する者は次の事項を容器または包装に表示しなければならない.(安衛法第57条)(a)名称,(b)成分及びその含有量,(c)人体に及ぼす作用,(d)貯蔵又は取り扱い上の注意,(e)表示をする者の氏名(4) 文書の交付 製造許可物質の他,令別表第9に掲げる631物質について譲渡し,または提供する者は,上記の表示事項の他当該物質の物理的および化学的性質,事故が発生した場合の応急措置などを記載した文書,いわゆるMSDS(Material Sa艶ty DataSheet;製品安全データーシート)を相手側に交付しなければならない.(安衛法57条の2)(5) 化学物質の有害性の調査 過去にわが国において製造され,又は輸入された化学物質は2001年3月27日現在54794あり,既存化学物質としてその名称が公表されている.この既存化学物質以外の化学物質(新規化学物質)を製造し,又は輸入しようとする事業者は,あらかじめ,有害性の調査を行い,その結果並びに化学物質の名称などを厚生労働大臣に届け出なければならない.(安衛法第57条の3)ただし,年間の製造,もしくは輸入量が100kg以下である確認を受けた場合には届け出は免除される.(令第18条の4)e.「労働者の就業に当たっての措置」(第6章) 労働災害を防止するには機械,設備を安全なものとするとともに,労働者も安全に作業が行うことのできる知識,技能をもつことが必要である.このため,本章では労働者に対する雇い入れ時の安全教育の実施のほか,特に危険・有害な業務に就かせる場合には特別の教育の実施や資格の取得が義務づけられている.i。安全衛生教育 労働者に対して行わなければならない安全衛生教育として,次のものが定められている.1雇い入れたとき,および作業内容を変更したときの教育(安衛岬町59条第1項,第2項〉2高圧室内作業,特定粉じん作業など厚生労働省令で定められた危険,有害業務に就かせるときの教育(安衛法第59条第3項)3 新たに職務に就くこととなった職長,その他の作業中の労働者を直接指導又は監督する者に対する教育(安衛法第60条)ii.就業制限(安衛法第61条) ボイラーの運転,クレーンの運転など宇陀20条で定める16の業務(就業制限業務)については,必要な資格を有する者でなければ当該業務に就かせてはならないとされている.資格は業務に応じて,都道府県労働局長が交付する免許を要するものと,都道府県労働局長の指定する者が行う所定の講習修了証を要するものとがある.f「健康の保持増進のための措置」(第7章安衛法第65条~第71条) 労働者の健康を保持増進するための措置として,作業環境の測定およびその結果の評価に基づく措置,健康診断の実施およびその結果に応じた措置などが規定されている.細部については3・1・2項の省令の解説を参照されたい.g.コンサルタント(第9章安衛法第81条~第87条) コンサルタントには,労働安全コンサルタントと労働衛生コンサルタントの2種類があり,コンサルタント試験に合格し,厚生労働省に備えるコンサルタント名簿に,氏名,事務所の所在地その他必要事項の登録を受けた者がコンサルタントになることができる.コンサルタントは,他人の求めに応じ報酬を得て,労働者の安全又は衛生の水準の向上を図るため,事業場の安全又は衛生についての診断及びこれに基づく指導を行うことを業とする.コンサルタント試験等については,3・1・2項nを参照されたい.h.「監督等」(第10章) 本章では労働基準監督官の権限,産業安全専門官の権限などの他,建築物,機械等を設置するときの計画の届け出について規定されている.・計画の届け出 事業場を新たに設置したり,機械設備を設置する場合,またはこれらを変更する場合は,その工事を開始する前に所轄労働基準監督署長などに計画書を届け出,審査を受けることが規定されている.1業種および規模が令第24条で定めるもの(製造業〔食料品製造業平一部の製造業を除く〕,電気業,ガス業,自動車整備業および機械修理業で,電気使用設備の定格容量の合計が300kW以上のもの)に該当する事業場においては,建築物もしくは機械などを設置し,もしくは移転し,又はこれらの主要構造物を変更しようとするときは,その計画を工事開始30日前までに所轄労働基準監督署長に届け出なければならない(法第88条第1項).2上記1以外の事業場においても,危険な作業等を必要とする機械等で省令で定めるもの(3・1・2項の各省令を参照)を設置,移転もしくは変更しようとするときは,その計画を工事開始30日前までに所轄労働基準監督署長に届けなければならない(安衛法第88条第2項).3重大な災害を起こすおそれのある建設工事における特に大規模な仕事を行おうとするときは,その計画書を厚生労働大臣に仕事の開始30日前までに届け出なければならない(安衛法第88条第3項).4 建設業および土石採取業における仕事のうち特定のもの(上記3を除く)を行おうとするときは,その計画を仕事の開始14日前までに所轄労働基準監督署長に届け出なければならない(安衛法第88条第4項).5届け出を受けた行政官庁はこれを審査し,法令の基準に適合しない場合は工事開始の差し止め,計画の変更を命じることができる.(安衛法第88条第7項)二罰則(第12章) 安衛法に違反した事業者には,その規定の重要性に応じた罰則が適用される.例えば,製造の禁止の違反はもっとも重大で,3年以下の懲役又は300万円以下の罰金,危害の防止措置基準等の違反については,6月以下の懲役又は50万円以下の罰金に処せられるとされている.なお,罰則については,従業員が違反行為をしたときは,行為者のほか法人にも罰金刑が科せられるという両罰規定が適用される.
