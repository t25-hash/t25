# 3・1・2 労働安全衛生法関係規則

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・1 事故・労働災害等を防止するための法律

a．労働安全衛生規則 労働安全衛生規則（以下「安衛則」と
いう）では，労働安全衛生法（以下「安衛法」という）を施行する
ための一般的な細則が定められていて，第1編通則，第2編安全基
準，第3編衛生基準，第4編特別規制から構成されている．この安
衛則に対し，3・1・2項b以下の規則は特定の機械，物，作業等に
よる危険を防止するために定められた規則で，通称前者を一般規則，
後者を特別規則と呼んでいる．
ここでは安衛則について，第1編通則のうち事業場の安全衛生管
理に係る部分，第2編安全基準，第3編衛生基準，第4編特別規制
について概要を述べる．
i．安全管理体制 事業場の安全管理を推進するためのスタッ
フとして，総括安全衛生管理者，安全管理者，衛生管理者，産業医，
安全衛生推進者等の選任が，また，調査，審議機関として，安全（衛
生）委員会の設置が業種，事業場の規模に応じて義務づけられてい
る（表3・2）。

者をもって充てなければならないとされている．

ii．安全衛生教育
（1）雇入れ時の教育（安衛則第35条） 事業者は労働者を雇い
入れたとき，または労働者の作業内容を変更したときは，当該作業
に関して（機械・原材料などの危険性，有害性および作業方法など
の）必要な安全衛生教育を実施する事が定められている．
（2） 特別教育（安衛同等36条） 安衛法第59条第3項において
危険有害業務で，労働省令で定めるものに労働者を就かせるときは，
特別の教育を行わなければならないと定められており，安衛則で対
象業務として，研削といしの取り替えまたは取り替え時の試運転の
業務，アーク溶接機による金属の溶断・溶接の業務等，48種類の業
務が定められている．
最近では，放射性物質の取扱いの業務，ダイオキシン類が存在す
る焼却灰などの取扱い業務が対象業務として追加された．
ili．健康診断 安衛法第66条において，事業者は労働者に対し
て健康診断を実施することが定められている．その健康診断の種類
などについても安衛則では次のものが定められている．なお，この
他に有機溶剤や鉛など特定の有害物質を取り扱う業務に従事する者
に対しては，特殊健康診断が義務付けられているが，これについて
は3・1・2項目～iを参照されたい．
（1） 雇入れ時の健康診断（安衛則第43条） 事業者は常時使用
する労働者を雇いれるときは医師による健康診断を行うことが定め
られている．パートタイム労働者についても常時使用する場合は適
用される．
（2） 定期健康診断（安衛則第44条，第45条）
① 事業者は常時使用する労働者に対し，1年以内ごとに1回定
期に医師による健康診断を行わなければならないが，著しく暑
熱もしくは寒冷な場所における業務深夜業を含む業務など特
定の有害環境における業務に従事する者に対しては6月以内ご
とに1回の医師による健康診断を行うこどが定められている．
② 常時50人以上の労働者を使用する事業場にあっては，健康診
断結果報告書を所轄労働基準監督署長に提出することが定めら
れている．
（3） 海外派遣労働者の健康診断（安衛二二45条の2） 事業者
は労働者を海外に6月以上派遣しようとするとき，および6月以上
派遣した労働者を本邦で業務につかせようとするときは，あらかじ
め医師による健康診断を行うことが定められている。
（4） 健康診断結果の記録の作成（安衛則第51条） 上記健康診
断の結果については，事業者は健康診断個人票を作成し，これを5
年間保存することが定められている．
