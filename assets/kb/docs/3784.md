# 1・1・4 人工衛星の今後の発展方向

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・1 人工衛星開発の歴史と現状

人工衛星の今後の発展方向を見極めることは極めて難しい．科学
観測の分野では，ハッブル宇宙望遠鏡やX線，赤外線などの非可
視領域を使った地球周回軌道上の大型の観測装置が実現し，宇宙科
学の知識を急速に蓄積しており，今後もこの観測装置の高度化が望
まれるであろう．また，月や惑星の探査は，将来の人類生存域拡大
の可能性も含めて大きく推進されることが望まれる．近い将来には
火星，金星などの固体型惑星に限定せず，木星以遠のガス型惑星と
その衛星への探査も本格化することが期待される．実用分野では，
今後も引き続き通信や地球観測が主要なミッションとして継続され
るであろう．通信・放送衛星は商用運用の時代に入っている．地球
観測衛星の一種である高解像度衛星の画像を販売するビジネスが始
まったが，大きな流れになるかどうかは今後の動きにかかってい
る．日本で打ち上げた実用衛星は，政府機関，地方自治体，研究
所，企業が求めるサービスを提供しているが，さらにサービス対象
を一般に拡大し，できれば商用運用に移行させることが望まれてい
る．
