# 3・1・3 鉱山保安法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・1 事故・労働災害等を防止するための法律

a.はじめに 鉱山保安法は鉱山における保安を確保するために,事業者,従業員そしてこれを監督する国に対して行動を規定している.鉱山とは鉱業を行う事業場であり,鉱業は地表,地下の鉱物資源(法定鉱物として金鉱,銀鉱,銅鉱,鉛鉱,すず鉱,水銀鉱,亜鉛鉱,鉄鉱,硫化鉄鉱,クローム鉱,マンガン鉱ひ鉱,ウラン鉱,石炭,亜炭,石油,アスファルト,可燃性天然ガス,硫黄,石こう,石灰石,ドロマイト,けい石,長石,ろう石,滑石,耐火粘土などに限定されている)を対象とした事業であり,作業に従事する者を重大な危険にさらすだけでなく,鉱山周辺の人々に損害を与える恐れがあり,更にその実施方法によっては他の有用鉱物資源の開発を困難にするなど地下資源の合理的な開発を阻害する恐れがある.特に坑内作業は地下閉空間での鉱物の採掘作業であり,通気・衛生にも特段の措置が必要である.また,メタンガスや炭塵による爆発,落盤,岩盤崩壊,出水などの不慮の災害が一時に多数の従業員に死傷を与えることから,作業や火薬類,機械・器具などの取扱いにあたっては常時細心の対応をしなければ災害・鉱害を防止し得ない特質を有している.また,事業の実施方法によっては事業を休止または廃止した後においても有害坑内水の流出,地表陥没,採掘跡斜面および堆積斜面・堤体の崩壊などを発生させ,鉱山周辺の人々や財産に重大な影響を与える危険性を有していることは他の産業にはない特質である.鉱山保安法は,これら鉱山における人に対する危害防止,鉱物資源の保護,鉱山施設の保全および鉱害の防止に対処するため,事前予防の観点から立法され,事前許認可制,事前検査制を柱とし,さらに事業廃止後5年間の措置命令を特徴に,司法警察権による保安監督を含め,すべての事業態様に対応するべく政省令をもって細かく規定している.わが国の鉱山技術・施設は他産業のそれらの原点ともなっており,鉱山保安法の規定も独自性の強いものである.鉱業に従事する技術者は,保安上重要な役割を担?ているので,法上の資格義務付けなどはもちろんのこと,作業者に対しても鉱山内で遵守すべき細かな事項が定められている.鉱山内で使用する火薬類,機械・器具,施設・設備などについては,その管理方法や性能等が規定され,鉱山から排出されるガス体を含む物質すべてについて取扱いおよびその排出基準などが定められている.さらに関係機関による坑内用品検定を義務付けている.国に対しては許認可,検査などの監督・指導を義務付け,その監督機関(経済産業省原子力安全・保安院および鉱山保安監督部)に刑事訴訟法の規定による司法警察員として職務を行う鉱務監督官を置くことを定めている. ノb.制定経緯並びに制定の背景 わが国の鉱山に対する規制は,古くは徳川時代に幕府,藩主によりそれぞれ行われていたが,鉱山業を独占・維持するための色彩が強いものであった.明治政府は鉱山業を独占から解放し,無秩序な開発から積極的な開発の奨励を促し,1872年(明治5年)に「鉱山心得」,1873年(明治6年)に「日本坑法」を制定・公布した.七かし,急激な鉱山開発は,金属鉱山では深刻な鉱毒・煙害問題を発生させ,石炭鉱山ではガス・炭塵爆発などの災害を頻発させた.その結果多くの人命が失なわれ社会の注目を浴びるようになり,抜本的な改正として1890年(明治23年)「鉱業条例」が制定され,更にこれに基づく「鉱業警察規則」が制定された.これにより鉱山保安規制が一段と強化された.その後民法の施行に伴い1905年(明治38年)鉱業条例が廃止され,旧「鉱業法」が制定されたが鉱業警察の範囲としては同じであった.これらの施策によっても石炭鉱山における爆発予防に対する技術的知見が乏しかったこともあり数百名の死傷者を出す大爆発が回を重ねる事態が続き,1915年目大正4年)石炭坑爆発取締規則が制定された.特に石炭鉱山技術に対する国による試験研究が行われ,その成果を踏まえきめ細かな法規制がなされている.1949年(昭和24年),戦争による鉱山の荒廃建て直しと産業復興を背景に,連合軍最高司令部の要請を受け,抜本的な改正を行い現在の「鉱山保安法」が制定された.鉱山に対する監督・指導機関は,明治以来現在まで一貫して国(経済産業省)である.近年は国内稼動鉱山の衰退により,休止・廃止鉱山の鉱害問題が大きな保安上の課題になってきている.c.鉱山保安法即下の要旨i。目的 鉱山保安法は,鉱山労働者に対する危害を防止するとともに鉱害を防止し,鉱物資源の合理的開発を図ることを目的として1949年半公布された(昭和24年法律第70号).ii.法の適用対象 鉱業を行う事業場(鉱山),すなわち鉱物の採掘およびこれに付属する選鉱・製錬その他の事業を行う事業場である.なお,鉱山の付属施設として非鉄金属の11製錬場などが特定され,法の一部のみが適用されている.鉱山は,採掘する鉱物の種類により,金属・非金属鉱山,石炭・亜炭鉱山,そして石油・天然ガス鉱山の三つに大別される.また,本法は事業活動を一時休止していても,廃止しても一定要件を満たせば適用になることから操業形態からは稼動鉱山,休止・廃止鉱山の二つに大別される.鉱山保安の実施義務を負うのは事業を営む鉱業権者および鉱山労働者であり,監督義務を負うのは国(経済産業省原子力・保安院および鉱山保安監督部)である.iii.鉱山保安確保のための主な仕組み 鉱山保安法は保安確保のため「人」,「もの」,「行動」についてそれぞれおおむね次のように規定している.(適用は一種や規模などによりそれぞれ異なる)1鉱業権者の義務 危害・鉱害の防止,通気の確保,鉱物資源の確保,機械・器具・建設物および工作物の保全など2 鉱山労働者の義務 保安のため必要な事項の遵守3 保安教育の義務 鉱山労働者に対する保安教育4 機械,器具などに関する制限 ガス検定器性能,防爆構造,救命器性能などについて検定合格したものの使用など5 施設計画の認可・届出 事前認可,届出など6 性能検査等 機械,器具,建設物,工作物その他施設について設置または変更が完了したときなどの検査(落成検査,性能検査).β9-797集積場等に関する規制 措置しなければならない捨石,鉱さいの集積場などについての規定8 保安規程 鉱山事情に合致した独自の規程の策定9 資格者選任 原子力安全・保安院および鉱山保安監督部に鉱務監督官をおくことおよびその権限について,鉱山保安監督部長などに対する鉱山労働者の申告,その他について規定14 厚生労働大臣は鉱山における危害の防止に関し,経済産業大臣に勧告することができる.15罰則 両罰規定一方,法令で一律的に規定できない事項(例えば,鉱山における特殊な地下構造,鉱山の立地条件などに由来する個別具体的な危害,鉱害防止措置)も多く,それらについては,法の示す範囲内において二割監督官の命令により規制できることになっており,法目的を担保している.iv.各省令(規定)の概要 鉱山保安法は鉱業法の姉妹法として鉱山保安の基本的な事項を定めているが,広範多岐にわたる具体的な規定は省令に委任している.技術に関する主な省令は以下のとおりである。(鉱種や規模等により各省令の適用が異なる)(1) 鉱山保安法第2条第2項ただし書きの付属施設の範囲を定める省令 鉱物の採掘に係る事業を主たる事業としない付属施設,鉱物の採掘場から遠隔の地にある付属施設などを定めている.ここに定められている付属施設には廃水,鉱さい及び鉱煙の処理に伴う鉱害の防止などについてのみ鉱山保安法が限定適用されている.(2) 鉱山保安規則 gO7条で構成されている(長らく金属・非金属,石炭・亜炭,石油・天然ガスの3零種ごとの省令で運用されてきたが,1995年(平成7年)に統合され本規則となった)。石炭・亜炭鉱山,金属・非金属鉱山(核燃料物質鉱山を含む),石油・天然ガス鉱山に係るすべての保安に関する規定を網羅した鉱山保安法の中心となる省令である.鉱業権者および鉱山労働者・鉱山保安技術職員の遵守事項,施設計画の認可・届出の事項,性能検査等の手続き,保安規程に定めるべき事項,機械・器具・施設・設備の設置および管理事項などの制限,坑内通気規制,鉱山保安技術職員の資格,ガス・炭塵爆発防止,石油天然ガスの暴噴防止,露天掘採場の崩壊防止,人の運搬に関する規定,就業箇所に関する規定,火薬類・発破等に関する規定,鉱害防止に関する規定,放射線障害に関する規定,フレキシブルディスクを使用する諸手続き方法について,などを規定している.(3) 鉱山坑内用品検定規則 鉱山の坑内で使用する機械・器具などについて合格すべき種類別検定,型式検定,抜取検定,個別検定,特別型式検定の5種類を規定している.1 種類別検定は,火薬,爆薬,爆薬安全被筒,一酸化炭素検定器,検知管その他に対する検定.2型式検定は,内燃機関を動力とする機械および車両,発破用電気点火器,救命器,一酸化炭素用自己救命器その他に対する検定.3 抜取検定は,種類別検定に合格した坑内用品に対する検定.4個別検定は,型式検定に合格した坑内用品に対する検定.5 特別型式検定は,照明器具,電線,電気機械,電気器具その他に対する検定.これらの検定についてのそれぞれの検定基準を公布したときや種類別検定又は型式検定の合格証を公布したときなどは,検定公報により,その旨が公示されることになっている.またフレキシブルデβ9-80イスクによる検定手続き方法についても規定している.(4) 鉱山施設性能検査等手数料規則 定められた施設の設置完了後の落成検査,変更完了後の検査および性能検査に要する施設一基ごとの手数料(収入印紙)を定めている.(5) 保安技術職員国家試験規則 国家試験は上級保安技術職員試験と普通保安技術職員試験とに区分されている.上級保安技術職員は甲(気流中の可燃性ガス含有率が一定以上の石炭坑),乙(甲種石炭坑以外の石炭坑等),丙(石油・天然ガス鉱山),丁種(核原料物質鉱山)の4種に,普通保安技術職員にはその職員が従事する鉱山の種類・規模ごと並びに分掌する事項ごとに合格すべき16種類の国家試験が定められている.甲種および洋種の各坑外保安係員(坑外の保安にかかる係員),鉱場保安係員(石油・天然ガス鉱山の保安業務),甲種乙種および丁種の各坑内保安係員(坑内の保安にかかる係員),機械保安係員,甲種および乙種の各電気保安係員,甲種および乙種の各鉱害防止係員,汽缶係員(ボイラを取り扱う係員),火薬係員(火薬類の存置,受渡し,運搬などを行う係員),甲種および乙種の各発破係員(火薬類の携帯,装てん,結線,点火などを行う係員),溶接係員(ガス溶接を行う係員)の16に細分され,おのおの受験資格,筆記試験科目,試験科目の免除,各試験:合格者相互の関係などについて規定している.(6)鉱山における鉱害の防止のための規制基準を定める省令鉱煙(金属鉱山等),ばい煙(石炭鉱山,および石油鉱山),坑廃水,ダイオキシン類,騒音,振動,石綿粉じんなどについての規制基準を定めている.(7) 鉱業廃棄物の処理等に関する基準を定める省令 鉱業の実施により生じた不要物すなわち捨石,鉱さい,沈殿物,もえがら,ばいじん,廃油,廃部,廃アルカリ,廃プラスチック類,PCB汚染物などについて保管,運搬,埋立処分,埋立場,海洋投入処分などについての基準を規定している.ゆロ(8) 深海底鉱山保安規則 公海の深海底(5000m~6000m)に賦存するマンガン・ノジュールを商業的に採取する深海底鉱山に対する保安規則である.海底,海中,海上における作業に対する保安について規定している.本規則の適用鉱山は未だ存在しないが,調査・研究が継続されている.深海底鉱業暫定措置法の公布・施行に伴い制定された.d.他法令との関係 鉱山保安法が他の規制法令に比べて制定が早かったことおよび独自の技術基準,規制が多かった経緯から,他の規制法令は鉱山保安法が適用される鉱山への適用を除外しているものが多いが,鉱山保安法に準用規定等をおき同等の規制内容となっている。しかし,特に石炭鉱山の坑内に関する技術(例えば,通気対策,可燃性ガス対策,ガス・炭塵爆発防止対策,火薬及び火薬類,人を運搬する巻上装置,防爆構造,坑内用品検定など)や石油・天然ガス鉱山の技術などについては独自の規定となっているものが多い.
