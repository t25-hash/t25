# 3・1・4 火薬類取締法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・1 事故・労働災害等を防止するための法律

火薬類は，爆発の用途に供せられるもので，その危険性が故に取
扱を誤まると悲惨な災害事故を引き起こし，不正に使用されれば重
大な社会不安を招くことになる．そこで火薬類取締法（以下「火取
法」という）は，火薬類の生産から消費に至る間の全ての取扱を規
制することにより，火薬類による災害を防止し，公共の安全を確保
することを目的として昭和25年5月4日（法律第149号）に制定され
た．火取法のルーツは，明治5年太政官達第28号「銃砲取締規則」に
あり，以来連綿として今日の火取法に受け継がれている．
a．火取法の規制を受ける火薬類は法第2条によって，火薬，爆
薬及び火工品に大別し，各区分ごとに具体名を品名例示することで
定義されている．火薬類の規制システムは，許可なくして火薬類を
取り扱うことはできない仕組みなっている．火薬類の製造営業，販
売営業，火薬庫の設置については，それぞれ人的あるいは物的許可
基準を設けて行政庁の許可を前提条件とするとともに，技術的な許
可基準については，完成検査を行うとともにこれの維持遵守を法的
に強制し，かつ，火薬類の譲渡，譲受，輸入，運搬，消費及び廃棄
などについては，取締の必要上行政庁の許可あるいはこれに対する
届出の義務を課し，さらに行商及び屋外販売を禁止し，所持者の範
囲を明確化し，取扱者の制限を規定して災害の防止と公共の安全の
確保を期している．一方，保安の観点から火薬類の製造，販売，消
費その他についての一般的保安事項に関する事項ならびに監督的行
政的措置として，危害予防規程，保安教育，製造保安責任者及び取
扱保安責任者，保安検査，安定度試験，不良火薬類の措置，火薬類
の混包などの禁止，危険時の措置及び届出，喫煙などの制限，立入
検査，緊急措置などの規定がある．その他として，事故届等，現状
変更の禁止などの規定の他，法違反に対する罰則も定められている．
b．火薬類の製造（製造所）及び火薬庫に係る規制
i．事業者規制
（1） 製造の場合
① 製造営業許可（法第3条）
② 製造施設及び製造方法の変更許可（法第10条）
③ 製造施設及び製造方法の技術上の基準維持義務（法要9条）
④製造施設の完成検査受検義務ただし自ら完成検査を行うこ
とができる認定事業者は除く（法第15条）
⑤ 危害予防規程認可・遵守義務（法第28条）
⑥ 保安教育実施義務・同計画認可（法第29条）
⑦ 火薬類製造保安責任者等選任義務・職務を行わせる義務（法
第30条）
⑧ 定期自主検査実施義務・同計画届出（法第35条の2）
⑨保安検査受検義務ただし自ら保安検査を行うことができる
認定事業者は除く（法第35条）
（2） 火薬庫の場合
①火薬庫設置・移転許可（法第12条）
②構造・設備変更許可（法第12条2項）
③完成検査受検義務ただし自ら完成検査を行うことができる
認定事業者は除く（法第15条）
④ 火薬庫の技術上の基準維持義務（法第14条）
⑤ 貯蔵の技術上の基準維持義務（当芋11条）
⑥火薬類取扱保安責任者等選任義務・職務を行わせる義務（法
第30条）
⑦定期自主検査実施義務・同計画届出（法第35条の2）
⑧保安検査受検義務ただし自ら保安検査を行うことができる
認定事業者は除く（法第35条）
ii．従業者規制
（1） 製造の場合 製造の許可を受けた事業者によって選任さ
れた火薬類製造保安責任者は省令で定める職務を遂行しなければな
らない旨定めているが（法第32条），この規制は火薬類による災害を
防止するためには，その施設，方法などの物的側面について基準維
持がなされると同時に，それらを取り扱う者がその基準を完全に維
持するという人的側面についての基準維持がなされることが重要で
ある．特に後者については，客観的な規制基準を設けることは判断
の基準がないことと相まって困難である．このため火薬類の製造，
貯蔵，消費の保安の中心となる者について一定の規制を課し，もっ
て後者について補足しようとするものである。
（i） 火薬類製造保安責任者の職務内容（規則第70条の2）
①製造施設の構造，位置もしくは設備または製造する火薬類の
種類もしくは製造方法が，遠藤10条第1項の許可を受けないで
変更されることのないよう監督すること．
②製造施設の構造，位置および設備または製造方法が，法益7
条第1号または第2号の技術上の基準に適合し，または適合し
維持されるよう監督することならびに危害予防規程が遵守され
るよう監督すること．この場合において，法第7条第1号およ
び第2号の技術上の基準のうち，盗難防止に関する事項および
火薬類一時置場における無煙火薬の存置に関する事項について
は，特に注意しなければならない．
(C)日本機械学会
③保安教育の実施状況を監督すること
④定期自主検査を指揮し，および監督することなど
（丑） 火薬類製造保安責任者の義務及び権限（法第32条）
火薬類製造保安責任者は，上記職務の誠実な遂行義務と火薬
類製造作業に係る「火薬類を取り扱う者」に対し，保安上必要
と認めるときに指示する上記職務の執行に関する指示権を有す
る．また，一般従業者は，危害予防規程の遵守義務と製造保安
責任者が保安上必要と認めてする指示に従わなければならない
義務がある．
（2） 火薬庫の場合 火薬庫を所有または占有する者によって
選任された火薬類取扱保安責任者は，省令で定める職務を遂行しな
ければならない旨定めている．
（i） 火薬類取扱保安責任者の貯蔵に係る職務内容（規則第70条
の4）
①火薬庫の構造，位置または設備が法第12条第1項の許可を受
けないで変更されることがないよう監督すること．
②火薬類の貯蔵上の取扱い又は火薬庫の構造，位置及び設備が
法第11条第2項又は第12条第3項の技術上の基準に適合し，又．
は適合して維持されるよう監督すること．この場合において，
法第11条第2項及び第12条第3項の技術上の基準のうち盗難防
止に関する事項については，特に注意しなければならない．
③定期自主検査を指揮し，および監督することなど
億） 火薬類取扱保安責任者の義務および権限（四四32条）
火薬類取扱保安責任者は，上記職務の誠実な遂行義務と火薬
類の貯蔵に係る「火薬類を取り扱う者」に対し保安上必要と認
めるとき指示する上記職務の執行に関する指示権を有する．
また一般従業者は，火薬類の貯蔵に関する規定遵守義務と火
薬類取扱保安責任者が保安上必要と認めてする指示に従わなけ
ればならない義務がある
c．火薬類の製造及び火薬庫等に係る技術上の基準
（1） 火薬類の製造の技術上の基準
①製造設備に係る基準として，施設の構造，施設の位置（保安
距離の確保 例えば，ニトログリセリン1000kg製造する工室
の場合は製造所外の学校に対しては，320m以上の保安距離の
確保が必要となる），施設問の位置（停滞量，保安間隔 例えば，
ニトログリセリン1000kgを製造する工室は同じ製造所内の他
の設備に対して3．4×緬m以上の間隔をあける必要があ
る），消火設備，電気（含む静電気）設備（例えば粉じんに対す
る防護装置などの義務付け），温湿度調整設備，設備の材質・構
造（例えば鉄と鉄との摩擦のない構造など），暖房装置（熱源の
制限 例えば蒸気は使用可），高熱源装置（安全装置の設置義務），
圧力装置（安全装置の義務付けなど），乾燥装置（温度測定装置
の義務付け，加温装置の設置場所ρ制限など）について災害防
止の観点から規制．
②製造方法に係る基準として，製造する火薬類の成分配合比，
構造・組成，最大製造数量／日，定員，停滞量（火薬類およびそ
の原料），施設の目的外使用の禁止，施設・設備修理制限，設備
点検義務，火薬類等運搬規制，容器・包装等について災害防止の
観点から規制
（2） 火薬庫および火薬類の貯蔵の技術上の基準
①火薬庫等に係る基準として，火薬庫の位置（保安距離）およ
び火薬庫の種類（8種類）ごとの場所，構造（例えば，屋根壁
体扉の構造，材質および寸法など）および設備（例えば，照明
設置の防爆式，暖房装置の資源制限など）ならびに避雷装置，
土堤（高さ・幅など）について規制．
②火薬類の貯蔵に係る基準として，火薬庫の目的外使用の禁止，
搬出入装置，三内作業の制限，容器・包装の収納制限，保管火薬
類の管理方法等の規制．貯蔵火薬類の区分に対する貯蔵火薬庫
の種類および火薬庫の種類別の最大貯蔵量などの規制
d．危害予防規程 火薬類製造業者に規程の制定を義務づけ，
