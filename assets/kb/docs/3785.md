# 3・1・4 火薬類取締法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・1 事故・労働災害等を防止するための法律

火薬類は、爆発の用途に供せられるもので、その危険性が故に取扱を誤まると悲惨な災害事故を引き起こし、不正に使用されれば重大な社会不安を招くことになる。そこで火薬類取締法(以下「火取法」という)は、火薬類の生産から消費に至る間の全ての取扱を規制することにより、火薬類による災害を防止し、公共の安全を確保することを目的として昭和25年5月4日(法律第149号)に制定された。火取法のルーツは、明治5年太政官達第28号「銃砲取締規則」にあり、以来連綿として今日の火取法に受け継がれている。a.火取法の規制を受ける火薬類は法第2条によって、火薬、爆薬及び火工品に大別し、各区分ごとに具体名を品名例示することで定義されている。火薬類の規制システムは、許可なくして火薬類を取り扱うことはできない仕組みなっている。火薬類の製造営業、販売営業、火薬庫の設置については、それぞれ人的あるいは物的許可基準を設けて行政庁の許可を前提条件とするとともに、技術的な許可基準については、完成検査を行うとともにこれの維持遵守を法的に強制し、かつ、火薬類の譲渡、譲受、輸入、運搬、消費及び廃棄などについては、取締の必要上行政庁の許可あるいはこれに対する届出の義務を課し、さらに行商及び屋外販売を禁止し、所持者の範囲を明確化し、取扱者の制限を規定して災害の防止と公共の安全の確保を期している。一方、保安の観点から火薬類の製造、販売、消費その他についての一般的保安事項に関する事項ならびに監督的行政的措置として、危害予防規程、保安教育、製造保安責任者及び取扱保安責任者、保安検査、安定度試験、不良火薬類の措置、火薬類の混包などの禁止、危険時の措置及び届出、喫煙などの制限、立入検査、緊急措置などの規定がある。その他として、事故届等、現状変更の禁止などの規定の他、法違反に対する罰則も定められている。b.火薬類の製造(製造所)及び火薬庫に係る規制i.事業者規制(1) 火薬類取締法に特に関係の深い法令としては、労働基準法令、消防法令、鉱山保安法令1警察関係法令、建築基準法令、武器等製造法令、自衛隊法令などがある。労働基準法令のうち、労働者の安全および衛生に関しては労働安全衛生法の定めによるとされている。この法令の観点が、人的なものに対し、火薬類取締法は、火薬類について、広く社会および事業所内の物的財産的保護をも含んだ総合保安である。消防法令との関連においては、火薬類のうち危険物に指定されているもの、例えば硝酸エステル類があり、これらの製造所、貯蔵所等などについては、火薬類取締法と消防法の二重規制になるところもある。建築基準法では、火薬類製造所及び火薬庫について政令に定めるものについては、主要構造部を耐火構造としなければならない旨の規定があるが、火薬類取締法は耐火の点をも考慮のうえ、しかも爆発による被害の減少をも考慮して特別の規定を設けているので、実際上は法の技術基準を満足するものは建築基準法令の対応規定を満足する。
