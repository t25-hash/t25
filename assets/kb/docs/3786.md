# 1・2・2 スプートニク以後の展開

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・1 人工衛星開発の歴史と現状

第二次世界大戦直後，ロケットの運命が風前のともしびとなった
時期がある．A-4ロケットは，システムとしては画期的であるも
のの，兵器としての評価ははなはだ低く，アメリカでも旧ソ連（ソ
ビエト社会主義共和国連邦）でも軍が興味を示さなかったからであ
る．
一方，人工衛星の技術発展の方向は比較的明確であるといえよ
う．まず，科学・実用のどちらに対しても，軌道上の人工衛星がよ
り長期に安定した性能を発揮することが求められる．このため，信
頼性を確保する設計・製造手法の強化が求められるとともに，特に
大きな施設では軌道上における保守・点検を容易にする技術面の進
展が期待される．宇宙基地における有人活動と連動して，ロボット
などの活用が大きな役割を果たすであろう．惑星空間では，地球と
の通信に長時間がかかることから，装置の知能化・自動化が不可欠
である．次に，人工衛星のコストを削減することは大変大きな課題
となっている．人工衛星の各部の標準化が有効との論議があるが，
この方向はいまだ本格的に進んでいない．むしろ，地上機器の部品
や技術を積極的に人工衛星に取込むことが検討されている．衛星の
大型化は実用衛星では一般的にコスト的に有利である反面，科学観
測や一部の実用ミッションでも単機能化による小型化がコスト低減
とシステム信頼性の向上に資することも期待される．いずれの場合
でも，部品・機器のマイクロ化，ナノ化は人工衛星の機能向上とコ
スト低減に大きな影響をもたらす．
