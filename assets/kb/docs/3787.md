# 3・2・2 作業環境測定法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・2 労働者の福祉の向上を図るための法律

a.作業環境の中の有害因子の測定と管理 産業現場には種々の有害因子がある。それらは高温、高湿度、騒音、有害光線、電離放射線などの物理的因子と空気中に分散する粒子状物質及びガス・蒸気の化学物質である。これらの有害因子の作業環境基準は、大多数の健康な成人である作業者が、週当たり40時間のばく露を受けても障害が現れない程度のレベルに設定されている。作業環境測定は作業場にある有害因子を作業環境基準(作業環境測定関係の法令では管理濃度として定義している)以下に管理するための指標を得る手段として実施されている。労働安全衛生法は、労働者の健康確保を目的として事業者に種々の責任を負わせているが、その一つとして第65条において作業環境測定を規定し、作業環境中にある有害因子を定期的に測定して記録し、その結果を作業環境評価基準に照らして評価し、更にその評価に基づいて作業環境改善をすることを事業主の責務と定めている。この第65条に基づいて関連法規が制定されている。作業環境測定は労働衛生行政の3大要素である作業管理、作業環境管理及び健康管理の中の作業管理と作業環境管理のために、作業場の実態を把握する手段として実施されている。β9-84b.作業環境測定法の制定およびその社会的背景 、労働安全衛生法は、作業場内にある有害因子へのばく露に起因する職業病から作業者を保護するために、その第65条及び第65条の2において作業場における有害因子の作業環境測定とその結果に基づく作業環境の改善を事業主に義務付けている。この目的を具体化するために、作業環境測定の対象とする有害因子ゐ種類を労働安全衛生法施行令(政令)に、また該当する産業分野を労働安全衛生規則(省令)に示している。昭和50年には作業環境測定法が成立し、作業環境測定を実施する作業環境測定士の資格と登録制度、作業環境測定機関に関する規程、試験制度などを定めている。作業環境測定の技術的指針として作業環境測定基準が昭和51年に、また作業環境測定結果から作業環境の管理状態を評価する方式を示した作業環境評価基準が昭和63年に公布された。この二つの基準が示す作業環境測定は「場の測定」と呼ばれる考え方に基づいている。これはアメリカ及びEUで主に実施されている「個人ばく露の測定」と対比して論じられることが多い。個人ばく露の測定では、作業者がばく露する有害因子のレベルを測定するために、作業者個人にサンプラー又は計測器を携帯させて作業者の呼吸域で空気を捕集するか、又は影響を受ける感覚受容器の近くで有害因子を測定することにより、作業中に作業者がばく止する有害因子の量をできるだけ正確に測定しようとするものである。一方、「場の測定」は、作業者の移動や動作に関係なく、単位作業場所内に無作為に設定した格子状の区分の交点を測定点として一定時間空気を捕解し、捕糊したサンプルの定量値がサンプル間でバラツキを示すことを利用して、作業場の有害因子のレベルの幾何平均値と幾何標準偏差を算出し、有害因子の量の時間及び位置による分布を把握するもので、スクリーニングサンプリングとも呼ばれている。作業環境測定法では単位作業場所内のスクリーニングサンプリングの位置を6m以内の間隔で5点以上設定すること(A測定)、及び、ばく露が最も高くなる可能性のある発生源の近くにも作業環境測定を行うべき作業場測 作業環境測定の対象とする物理的有害因子および化学物質の種類は労働安全衛生法施行令(政令)第21条で、また具体的な作業場の種類を労働安全衛生規則(省令)第587条(熱暑、寒冷、多湿)、第588条(騒音)、第589条(坑内)に示している。化学物質については、鉛中毒予防規則第52条、特定化学物質等障害予防規則第36条、有機溶剤中毒予防規則第28条、酸素欠乏症等防止規則第3条、電離放射線障害防止規則第53条、事務所衛生基準規則第7条、粉じん障害防止規則第25条において、その実施が定められている。作業環境測定法は、作業環境測定士の資格要件・試験制度・登録制度・講習機関の指定、’作業環境測定機関の業務規定などを定めてしくる。労働安全衛生法上65条および政令によって指定された指定事業場の作業環境測定は必ず作業環境測定士によって実施させること、作業環境測定士は事業場の職員または外部の作業環境測定機関の職員であることができること、が定められている。この作業環境測定法に関連する政令として作業環境測定法施行令が、省令として作業環境測定法施行規則が、また告示として測定士の養成機関、・測定士のための試験:制度、測定士国家試験の受験、測定士の資格要件としての必須科目、測定士の登録機関、測定機関に求められる設備基準等が示されている。表3・17 労働安全衛生施行令第21条で指定された有害因子を発生する作業場では、それぞれ規定された回数の定期的な作業環境測定を実施し、記録を保存することが定められている。作業環境測定を行うべき作業場と測定の種類を表3・17に示す。作業環境測定士は測定対象とする有害因子の種類によって5分野(鉱物性粉じん、特定化学物質、有機溶剤、金属及び電離放射線)に区分されて登録されている。これらの分野の対象となっていない事業場における温度、湿度、騒音、酸素濃度、二酸化炭素濃度などは作業環境測定を実施する人材を限定していないが、事業場ごとに任命されている衛生管理者などの責任において測定・管理されるべき事項となっている。作業環境測定は事業主に課された義務であるから、この義務を実施しない場合には記せられる。しかし、作業環境測定士による正規の測定が実施され、その記録が保存されていれば、測定結果の評価に関わらず罰則の対象とはならない。測定結果の評価が第3管理区分となった場合には事業主は作業場の改善を実施することを自主的な義務として負わされているが、さらに所轄の労働局長から改善命令を受ける場合がある。厚生労働省は評価の低い事業場を指定して改善支援する政策を実施し、作業環境改善計画の立案に労働衛生工学コンサルタントの指導を導入して改善を推進している。2001年3月現在、(社)日本作業環境測定協会に登録されている作業環境測定士の総数は19998名、会員となっている作業環境測定機関は435機関である。事業場における作業環境測定の実情を調査する目的で、(社)作業環境測定協会は会員である事業場及び測定機関を対象として、測定対象となっている化学物質ごとの測定結果を集計している。また作業環境測定士の技術水準の均等化を図るために、標準サンプルき作業環境測定士に供給してその分析結果を回収、集計する精度管理事業を実施している。e.作業環境測定の最近の動き 作業環境測定の対象となっている有害因子は、実際に作業場にある有害因子の中の一部であるに過ぎない。作業場で使用される化学物質やエネルギーの種類や形態が変化すると、新しい有害因子が問題となり、作業環境測定の対象として追加される。最近では廃棄物焼却施設の運転・点検作業に従事する作業者のダイオキシン類へのばく露防止対策として、作業場の空気中ダイオキシン濃度を6月以内ごとに1回測定することが義務付けられた。また焼却施設の解体作業を計画する段階で、解体の対象となる施設の内部に付着する物のダイオキシン類含有率を測定することとなった。これらの措置は労働安全衛生規則の一部改正と通達「廃棄物焼却施設内作業におけるダイオキシン類ばく露防止対策要綱」に示されている。また病院などで滅菌剤として使用されるエチレンオキサイドに対して、労働安全衛生法施行令と労働安全衛生規則の一部改正により、平成13年5月1日から作業環境測定の対象とした。作用環境測定は多くの場合、規制対象外の因子が共存する状態で規制対象因子の測定と評価を行っている。規制対象外の有害因子に対しても事業主は自主的に管理する義務を負っているが、その手段や評価法は法的には示されていない。参考資料1)
