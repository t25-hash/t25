# 3・2・3 じん肺法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・2 労働者の福祉の向上を図るための法律

a.じん肺法制定の目的(第1条) じん肺法は、じん肺に関し適正な予防および健康管理などの必要な措置を講じ、労働者の健康保持その他福祉の増進に寄与することを目的として、昭和35年3月31日に制定された。じん肺発生防止のためには、作業環境対策に加えて、定期的健康診断によりその病状を早期に的確に把握し、適切β9-85な管理区分を設定し、管理区分に応じて、粉じんばく露の減少またはばく露機会をなくすことを基本とする的確な鯉康管理上の措置を行い、じん肺の発生とその進展を防止することが不可欠である。このため、昭和52年7月1日、じん肺の定義、粉じん作業、じん肺管理区分、じん肺健康診断、健康管理のための措置などを整備充実し、じん肺と密接な関連ある合併症の療養を含む健康管理について改正が行われている。b.じん肺法の構成 じん肺法は、第1章 総則(第1条一第6条)、第2章 健康管理、第1節じん肺健康診断の実施(第7条一第11条)、第2節じん肺管理区分の決定等(第12条一第20条)、第3節健康管理のための措置(第20条の2一第23条)、第3章 じん肺審議会(第24条一三31条)、第4章 政府の援助等(第32条一第35条)、第46条)、附則から構成されている。なお、省令としてじん肺法施行規則(以下「じん肺則」という)が規定されている。c.じん肺の定義(第2条第1項第1号)パじん肺は、粉じん(空気中に含まれる三生物体の固体粒子で、ヒュームも含む)を吸入し肺に生じた組織の線維増殖性変化(胸部X線写真に粒状影などを示す)を主体とする、気道の慢性炎症性変化、気腫性変化を伴うもので、線維増殖性の変化は治療により修復されない不可逆性の疾病とされている。d.合併症の定義、範囲(第2条第1項第2号、第2項) じん肺病変の素地に外因が加わり高頻度に発症するじん肺と密接な関連をもつ疾病で、適切な治療により症状を改善し得る可逆性のものである。合併症としては、肺結核、結核性胸膜炎、続発性気管支炎、続発性気管支拡張症、続発性気胸がじん肺則に挙げられている。e.粉じん作業の定義、範囲(第2条第1項第3号、第2項) 労働者がじん肺にかかるおそれがある作業であり、粉じん作業の範囲は、じん肺則別表の第1号より第24号までの作業のいずれかに該当するものである。ただし、粉じん障害防止規則(以下「粉じん則」という)により非粉じん作業の認定を受けた作業は除かれる。f じん肺健康診断の方法(第3条第1項) じん肺健康診断(以下「じん肺健診」という)はゴ粉じん作業従事労働者について、粉じん作業歴の調査及びエックス線写真(直接撮影による胸部全域の背暦世写真、以下「X線写真」という)による検査を実施する。その結果、じん肺所見があるか又はその疑いがある者については、胸部に関する臨床検査(既往歴の調査、胸部の自覚症状及び他覚所見の有無の検査)及び肺機能検査(スパイロメトリー及びフローボリューム曲線による検査、動脈血ガスを分析する検査)を実施する。g.X線写真の像(第4条第1項) 事業者および粉じん作業従事労働者は、じん肺の予防に関し、労働安全衛生法(以下「安衛法」という)及び鉱山保安法(以下「鉱安法」という)の規定によるほか、粉じんの発散の防止および抑制、保護具の使用その他について適切な措置を講ずるよう努めることが定められている。j.教育(第6条) 事業者は、安衛法および鉱安法の規定によるほか、常時粉じん作業従事労働者に対しじん肺に関する予防および健康管理のために必要な教育を行うことが定められている。その内容、時期、回数などは、医学および衛生工学の進歩、技術の変革、更には事業場の実情に即して最も効果的な方法により行う。k.じん肺健診の実施 じん肺健診については、以下のような種種の健診を実施することが定められている。i.就業時健康診断(第7条) 粉じん作業に新たに従事する労働者について、・あらかじめ、じん肺り患の有無およびその程度を確認し、適切な健康管理を行うため、その就業の際、じん肺健診を実施する。ii.定期健康診断(第8条) 常時粉じん作業従事労働者で管理1であるものは3年以内ごとに、管理2又は管理3であるものは1年以内ごとに1回、定期的にじん肺健診を実施する。なお、粉じん作業従事労働者であったもので管理2のものは3年以内ごとに、管理3のものは1年以内ごとに1回、じん肺健診を実施する。iii.定期外健康診断(第9条) 常時粉じん作業従事労働者で管理1であるものが、安衛法の健康診断でじん肺の所見があるか、又はその疑いがある者については、遅滞なく、じん肺健診を実施する。なお、合併症で1年を超え休業療養した労働者が、医師により休業療養を要しないと診断されたときも、定期外健康診断を行う。iv。離職時健康診断(第9条の2)事業者は、じん肺健診の対象者である労働者が、離職の際にじん肺健診の実施を求めたときには、離職の日までに、管理1のものは1年6月を超え、管理2または管理3のものは6月を超えて使用していたものにはじん肺健診を行う。v.受診義務(第11条) じん肺健診の対象者である労働者は、正当な理由がある場合(疾病、忌引など社会通念上労働者に受診を強要することができない場合)を除き、事業者が行うじん肺健診を受ける。ただし、事業者が行うじん肺健診を希望しない場合は、他の医師が行うじん肺健診を受け、X線写真およびじん肺健診結果証明書(以下「健診結果証明書」という)を使用者に提出する。1.じん肺管理区分の決定i.x線写真等の提出(第12条)事業者は、じん肺健診を行ったときは、遅滞なく、じん肺所見があると診断された労働者について、X線写真及び健診結果証明書を都道府県労働局長に提出することが定められている。ii.じん肺管理区分の決定(第13条) じん肺健診の結果、じん肺の所見がないと診断された者は、管理1である。また都道府県労働局長は、X線写真および健診結果証明書が提出された者について、これらを基礎として、地方じん肺診査医の診断又は審査により、じん肺管理区分(以下「管理区分」という)の決定を行う。iii.通知(第14条)事業者は、都道府県労働局長から当該労働者の管理区分の決定通知を受けたときは、その者について決定された管理区分および留意すべき事項を通知しなければならない。iv.随時申請(第15条、第16条)常時粉じん作業従事労働者または常時粉じん作業従事労働者であった者は、いつでも、じん肺健診を受けて、都道府県労働局長に管理区分の決定申請をすることができる。v.X線写真等の提出命令(第16条の2)都道府県労働局長は、常時粉じん作業従事労働者または常時粉じん作業従事労働者であった者について、適正な管理区分を決定するため必要があると認めるときは、事業者に対して、X線写真および健診結果証明書などの提出を命ずることができる。vi.記録の作成、保存(第17条)事業者は、じん肺健診を行ったとき、じん肺健診に関する記録を作成し、その記録およびX線写真を7年間保存することが定められている。m.健康管理のための措置 事業者は、労働者の健康管理のために以下の措置をすることが定められる。i.事業者の責務(第20条の2)事業者は、じん肺健診の結果、労働者の健康保持のため必要と認めるときは、その労働者の実情を考慮して、就業上適切な措置を行うとともに、適切な保健指導が受けられるよう努める。li.粉じんばく露の低減措置の実施(第20条の3)事業者は、管理2または管理3イの労働者に対し、粉じんばく露を低減するため、就業場所の変更、作業時間の短縮その他の適切な措置を講ずるよう努める。iii.作業転換の実施(第21条)都道府県労働局長は、管理3イの労働者が常時粉じん作業に従事しているときは、粉じん作業以外の作業に従事させるよう事業者に対し勧告できる。事業者は、この勧告を受けたとき、または管理3ロの労働者が常時粉じん作業に従事しているときは、粉じん作業以外の作業に従事させるよう努めなければならない。常時粉じん作業に従事している管理3ロの者について、地方じん肺診査医の意見により、健康保持のため必要があると認められるときは、都道府県労働局長は、事業者に対して、その者を粉じん作業以外の作業に常時従事させるよう指示することができる。iv.作業転換のための教育訓練(第22条の2)事業者は、管理3の労働者を粉じん作業以外の作業に常時従事させるため必要なときは、その労働者に対して、作業転換のための教育訓練を行うよう努める。v.転換手当(第22条)事業者は、都道府県労働局長より作業転換の勧告された管理3イの労働者または管理3ロの労働者が常時粉じん作業に従事しなくなったときは、その者の平均賃金の30日分の転換手当を支払う。都道府県労働局長より作業転換の指示があった管理3ロの労働者が作業転換をしたときは、平均賃金の30日分の転換手当を支払う。vi.療養(禦3条)管理4の者および合併症にかかっている者は療養を要する。n.政府の援助等i.技術的援助等の実施(第32条)政府は、事業者に対して、粉じんの測定、粉じん発散の防止及び抑制、じん肺健診その断じん肺の予防及び健康管理に関し、必要な技術的援助を行うよう努める。ii.粉じん対策指導委員(第33条)都道府県労働局及び鉱山保安監督部に、事業者が行うじん肺の予防に関する措置について必要な技術的援助を行わせるため、粉じん対策指導委員がおかれている。o.法令の周知(第35条の2) 事業者は、じん肺に関する法令の要旨を粉じん作業場の見やすい場所に常時掲示し、または備え付けるなどの方法により労働者に周知させることが定められている。p.じん肺健診に関する秘密保持(第35条の3) じん肺健診の実施の事務に従事した者は、その実施に関して知り得た労働者の心身の欠陥その他の秘密を漏らしてはならない。q.じん肺診査医(第39条) じん肺法の規定によるじん肺の診断または審査およびこれらに関する事務を行うために、厚生労働省に中央じん肺診査医が、都道府県労働局に地方じん肺診査医がおかれている。r.罰則(第45条) 教育の実施(第6条)、就業時健康診断の実施(第7条)、定期健康診断の実施(第8条第1項)、定期外健康診断の実施(第9条第1項)、X線写真等の提出(第12条、第13条第4項)、じん肺管理区分の通知(第14条第3項)、X線写真等の提出命令(第16条の2)、記録の作成及び保存(第17条)、転換手当の支払(第22条)、法令の周知(第35条の2)、じん肺健診に関する秘密保持(第35条の3)の規定に違反した者は、30万円以下の罰金に処せられる。
