# 3・3・1 食品衛生法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・2 労働者の福祉の向上を図るための法律

a.食品衛生法の制定と経緯 明治33年に初めて食品衛生に関する全国的包括的法律として「飲食物ソノ他ノ物品取締二関スル法律」が制定された。その後、昭和22年忌、食品衛生行政が内務省から厚生省の管轄下に置かれることとなるとともに、新たに食品衛生に関する包括的法律として食品衛生法が制定公布された。数度の改正を経て現在の食品衛生法がある。b.食品衛生法の目的 第1条により、「飲食に起因する衛生上の危害の発生を防止し、公衆衛生の向上及び増進に寄与することを目的とする」ことが定められている。 ・c.規制対象(第2条) 食品、添加物、天然香料、器具、容器包装の製造、加工、調理、販売、輸入などが規制対象となる。「食品」などの用語は、以下のように定義されている。1 「食品」:すべての飲食物。ただし、薬事法で規定されている医薬品と医薬部外品は含まれない。2 「添加物」:食品の製造過程において、または食品の加工もしくは保存の目的で、食品に添加、混和、浸潤その他の方法によって使用するもの。3 「天然香料」:動植物から得られたもの、または その混和物で、食品の着香の目的で使われる添加物。4「器具」:飲食器、割ぼう具、その他食品または添加物の摂取、製造、加工、調理、貯蔵、運搬、陳列、授受または採取に供さられ、かっ、食品または添加物に直接接触する機械、器具その他の物。ただし、農業と水産業において食品の採取に用いられる機械、器具その他の物は含まれない。5 「容器包装」:食品または添加物を入れ、または包んでいる物で、食品または添加物を授受する場合そのままで引き渡すもの。d.営業許可の基準 全粉乳などの乳製品、食肉製品、添加物など政令で定められた営業を営む者は、施設ごとに専任の食品衛生管理者をおかなければならない(食衛法論19条の17)。ただし、総合衛生管理製造過程の承認を受けた施設(後述)については、この限りではないことが定められている。飲食店その他公衆衛生に与える影響が著しい営業で、政令で指定する施設(表3・21)については、都道府県条例により、業種別に公衆衛生上必要な基準を定めることとしている(食牛法第20条)。この基準は、厚生省公衆衛生局長通知により示されている営業施設基準の準則(昭和32年9月9日衛環発第43号別添)に則って定められている。それらの営業を営もうとする場合には、都道府県知事の許可を受けることが義務付けられており、この営業許可を得るには、営業施設が上記都道府県の基準に合致していることが必要条件となっている(食衛法第21条)。さらに、都道府県知事は、営業者がその基準に違反した場合には、施設の整備改善、営業許可の取り消し、営業の禁止や停止を求めることができる(食衛法24条)。営業施設基準の準則には、施設の設置場所、施設周囲の地面、区画、広さ、更衣室、天井・内壁・床・扉の構造、排気装置、明るさ、手洗い設備、機械器具の位置・構造、温度計、用水、廃棄物容器、便所などについて、業種別に基準が示されている。いずれの基準も、食品衛生上の危害防止の観点から、施設設備の構造、材質、設計の各側面からの必要事項を包含している。e.有毒物質の混入防止等の措置基準 営業施設基準が施設設備の構造等、ハード面の基準を示しているのに対し、施設設備の管理に関するソフト面の基準は、有毒物質の混入防止などの措置基準(食衛法第19条の18)に定められている。すなわち、都道府県は、営業施設の内外の清潔保持、ねずみ、昆虫などの駆除、その他公衆衛生上講ずべき措置について、条例で基準を定めることができ、そのための管理運営基準準則は厚生省局長通知(昭和47年11月6日環食第516号)により示されている。同準則には、施設の管理、食品取扱い設備の管理保全など、給水および汚物処理、食品などの取扱い、β9-89従事者に係る衛生管理などの衛生措置の基準が示されている。f衛生規範 施設設備の構造と管理については、営業施設基準準則および管理運営基準準則よりさらに詳細な基準が、弁当およびそうざい’(昭和54年6、月29日環食第161号)、漬け物(昭和56年9月24日環食第214号)、洋生菓子(昭和58年3月31日環食間54号)、セントラルキッチン/カミサリー・システム(昭和62年1月20日環食堂6号の2)、生めん類(平成3年4月25日環食第61号)の各衛生規範によって示されている。(表3・22参照)g.HACCPによる衛生管理 H典CCP(Hazard AnalysisCritical Control Point、危害分析重要管理点)システムに基づく衛生管理は、平成7年に食品衛生法第7条3総合衛生管理製造過程の承認制度の中に取り入れられ、これにより、わが国においてHACC:Pシステムが法的に位置付けられるようになった。総合衛生管理製造過程とは、HACCPによる衛生管理と施設設備を含む一般的な衛生管理の二本柱によって、食品衛生上の危害発生を防止する措置が総合的に講じられた過程である。この承認制度により、食品衛生法の規定によって製造または加工の方法の基準が定められた食品であって政令(施行令第1条)で定めるもの(表3・23)を対象として、営業者は、厚生省が定めた製造方法の基準(施行規則第4条)に適合した方法による製造または加工であることの承認を厚生大臣から受けることができる。HACCPシステムの12手順(FAO/WHOコーデックス委員会により示された)を経て衛生管理計画を作成することに加え、施設設備の衛生管理や従事者の衛生教育などの一般的な衛生管理計画を適正に作成し、それら計画に基づいて製造加工を実行できる方法が、この製造方法の基準に適合した方法である。承認申請にあたっては、施設の図面や施設設備の管理方法を含む申請書類が必要となる。なお、厚生労働省は、総合衛生管理製造過程の承認制度により示されたHACCPシステムによる食品の衛生管理の方法が、この制度の対象食品以外ゐ食品等の製造と加工に対しても有効な方法であること、したがってこの方法をそれら食品等の製造・加工施設においても取り入れることを推奨している。総合衛生管理製造過程を実施しやすくするための施設の整備に対しては、「食品の製造過程の管理の高度化に関する臨時措置法」(平成10年5月8日)により、金融や税制面での支援が得られる。この認定を受けるには、建物について、清浄皇別区画、原材料や製品の交差が起こらない広さ、清浄区域の清浄化設備などを示すこと、さらに総合衛生管理製造過程を実施できる機械設備とその配置を示すことなどが必要である。h.その他個別に示された施設設備の基準 かきの生食による食中毒防止を図る一環として、厚生省局長通知「食品、添加物の規格基準の一部改正について(施行通知)一生食用かきの成分規格・加工基準および保存基準」(昭和42年9月7日環乳第7070号)により、かきのむき劇場の建物の構造、取り扱い設備、給水と汚物処理等の基準を含む施設指導基準が示された。平成8年に、学校給食等の大量調理施設において腸管出血性大腸菌による大規模集団食中毒事例が相次いで発生したことから、「大規模食中毒対策等について」(平成9年3月24日衛食第85号)において大量調理施設の施設設備の構造のあり方が示され、ドライシステム化の導入が推奨されている。表3・21
