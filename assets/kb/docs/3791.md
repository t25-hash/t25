# 3・3・2 薬事法

> **章**: 第3章 工場・施設内部に係わる法制度
> **節**: 3・2 労働者の福祉の向上を図るための法律

a.薬事法の制定と目的 わが国における薬事法制は,明治22年に制定公布された「薬品営業拉薬品取扱規則」をもって,日本薬制の始まりとすることができる.昭和18年,初めて薬事法が制定公布された.現行の薬事法は昭和35年に制定公布され,その後多くの改定が加えられ現在に至っている.平成14年に大改定される予定である.薬事法第1条(目的)に,その目的は「医薬品,医薬部外品,化粧品及び医療用具の品質,有効性及び安全性の確保のために必要な規制を行うとともに,医療上必要性が高い医薬品及び医療用具の研究開発の促進のために必要な措置を講ずることにより,保健衛生の向上を図る」と述べている.以下,医薬品,医薬部外品,化粧品及び医療用具を纏めて,「医薬品等」と呼ぶ.b.薬事法の規制対象 医薬品,医薬部外品,化粧品及び医療用具を製造するときには,「薬事法」の規制を受けることとなるので,規制内容を十分理解し対応することが必要である.・「医薬品」とは,人又は動物の疾病の診断,治療又は予防に使用されるもの,並びに「日本薬局方」に収められているものと定義されている.医薬品には製剤化された医薬品だけでなく,生理活性を有する原薬や体外診断薬も含まれる.「医薬部外品」は,人体に対する作用が緩和なもので,薬事法第2条第2項で定義されている.「医療用具」とは,人もしくは動物の疾病の診断,治療もレくは予防に使用される器具機械であり,薬事法施行令第1表に掲げられている.健康食品や食品添加物は,薬事法の規制対象外である.また,臨床治験の段階にある開発中の薬物などについては,薬事法第80条の2により,「治験薬等」として規制対象になるので注意が必要である.なお,動物用医薬品は農林水産省の管轄下にあって,その細部の運用などは「動物用医薬品等取締規則」(昭和36年置において定められている.医薬品は「薬事法」以外にも種々規制されている。「麻薬及び向精神薬取締法」「覚せい剤取締法」などである.c.医薬品等の製造業の許可および許可の基準(法第12,13条)医薬品等を製造しようとするときは,厚生労働大臣または都道府県知事の許可を,製造所ごとに受けることが定められている.また,医薬品製造業および医療用具製造業では5年ごとに,許可の更新を受けなければその効力を失う.また製造業の許可は,製造しようとする品目ごとに受けなければならない(品目追加許可).製造業の許可を受けるためには,1(ハード面)その製造所の構造設備が,省令で定める基準に適合していること.また薬事法施行令第1条の2の2で免除されたもの以外は,2(ソフト面)その製造所における製造管理および品質管理の方法が省令で定める基準に適合していなければならない.医薬品等の製造でGMPに適合しなければならないとよく言われるが,薬事法で上に揚げた1及び2をGMP(Good Manufacturing Practice)と一般に呼んでいる.d.製造所の構造設備基準(GMPのハード面)
