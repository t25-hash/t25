# 4・1・2 石油コンビナート等災害防止法 石油コンビナート等

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・1 事故の防止に関する法律

災害防止法はコンビナート地区にあって石油等(引火性の危険物,高圧ガス等)の貯蔵,取扱が一定以上となる特定事業所について,新設等の届出,防災組織,各施設地区(製造,貯蔵,用役,入出荷,事務管理,その他施設地区)の配置・面積,特定通路等の配置・幅員,特定施設等(流出油等防止堤,屋外給水施設,非常通報設備)の設置,必要とされる防災資機材(化学消防車,オイルフェンス等)が規定されている.特定事業所は次の定義により第1種又は第2種事業所と定義される.〈第1種事業所〉{(石油の貯蔵・取扱量)/1万kL}+{(高圧ガスの処理量)/200万m3}≧1となる事業所.ここで,第1種事業所のうち,石油及び高圧ガスを共に貯蔵・取扱う事業所は,各施設地区のレイアウトに関する規制を受ける.〈第2種事業所〉{(石油の貯蔵・取扱量)/1000kL}+{(高圧ガスの処理量)/20万m3}+{(第1,2,3,5,6類の危険物の貯蔵・取扱量)/2000t}+{(石油以外の第4類危険物の貯蔵・取扱量)/2000KL}+{(準危険物の貯蔵・取扱量)/1万t}+{(高圧ガス以外の可燃性ガスの貯蔵・取扱・処理量)/20万m3}+{(毒物の貯蔵・取扱・処理量)/20t}+{(劇物の貯蔵・取扱・処理量)/200t}≧1となる事業所で当該事業所及び第1種事業所の災害が相互に影響を及ぼすと認められるものとして,都道府県知事が指定するもの.a.新設等あ届出 石油及び高圧ガスを共に扱う第1種事業所(レイアウト規制事業所)は石油の貯蔵・取扱量又は高圧ガスの処理量を増加する工事,新たに石油の取扱い又は高圧ガスの処理をする工事を行う場合は工事開始の45日前に以下の事項を含む新設に関する計画を主務大臣(総務,経済産業両大臣)に届出る必要がある.又,(1)事業所敷地内を用途に応じ製造施設地区,貯蔵施設地区,溶液施設地区,事務管理施設地区,その他施設地区に区分し,各施設地区の面積と配置.(2)特別防災区域内の事業所間連絡導管及び連絡通路の配置.(3)敷地面積の一部変更を行う場合も同様に45日前に変更に関する計画を主務大臣に届出る必要がある.いずれの場合も不指示決定通知を受けてからでないと工事着工は出来ない.又,工事完了日から10日以内に新設等完了届出を主務大臣に提出し,確認検査を受けた後でないと設備を稼働させることはできない(試運転も含む).〈新設等の届出に必要な添付書類〉β9-961事業所の位置図2隣接事業所位置図3
