# 4・1・4 高圧ガス保安法

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・1 事故の防止に関する法律

a．法の目的 高圧ガス保安法（以下4・1・4項において単に「法」
という）は，高圧ガスによる災害を防止し，公共の安全を確保する
ことを目的とするものであり，高圧ガスによる災害を防止する手段
として，「規制」と「自主保安」とがある．
