# 4・1・5 液化石油ガスの保安の確保及び取引の適正化に関する法

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・1 事故の防止に関する法律

律 「液化石油ガスの保安の確保及び取引の適正化に関する法律」は、液化石油ガスの一般消費者に係わる保安の確保と取引の適正化β9-101を図ることを目的として昭和42年に制定され、:LPガスの販売、:LPガス設備工事、:LPガス器具等の製造・販売等に対して規制がされている。この法律では、:LPガスの災害から消費者を守るためおおむね次のことが定められている。(a)一般消費者等に対する:LPガスの販売事業を「登録制」とし、販売に必要な一定量以上の:LPガスの貯蔵については許可制とされている。また、:LPガスを供給するための設備に:LPガスを充てんしょうとする者は、充てん設備ごとにその所在地を管轄する都道府県知事の許可を受けることとされている。(b)LPガス販売事業者の規模によって、経済産業大臣、経済産業局長又は都道府県知事の登録を受けることになっている。この登録を受けた事業者は、「液化石油ガス販売事業者証」を販売所の見やすいところに掲げることになっている。(c)LPガスを供給するための設備の点検及び:LPガスを消費するための設備の調査、災害の発生防止のための周知及びガス漏れや災害時の対応を一般消費者等に対する保安業務を行う者(以下「保安機関」という)を認定制とし、一定以上の能力、体制が備わっていることが義務づけられている。(d) 液化石油ガス販売事業者は、:LPガスの販売契約を結んだときは、一般消費者等に対してLPガスの供給方法及び設備の管理方法等を記載した書面を渡すことになっている。また、液化石油ガス販売事業者又は保安機関は、:LPガスを消費する一般消費者等に対し、:LPガスによる災害の発生を防止するために必要な事項をパンフレット等により2年に1回以上周知することとされている。液化石油ガス販売事業者又は保安機関は、一般消費者等の内外の:LPガス設備が経済産業省令で定めた基準に適合しているかどうか4年に1回以上調査・点検することになっている。(e) L、Pガスの配管工事等重要な作業については、「液化石油ガス設備士」でなければその作業に従事できないことになっている。さらに、これらの作業を伴う工事を行う事業者は、都道府県知事に事業開始の届出をすることになっている。また、以上のような特定の工事に限らず液化石油ガス設備の工事は、経済産業省令で定めた基準に適合するように行わなければならない。(f)液化石油ガス器具等の製造、輸入又は販売の事業を行う者は、経済産業省令で定める表示が付されているものでなければ液化石油ガス器具等を販売し、又は販売の目的で陳列してはならない。「液化石油ガス器具等」には、次の15品目が指定されており、このうち、(1)から(3)の液化石油ガス器具等が特に液化石油ガスの災害の発生のおそれが多いと認められる特定液化石油ガス器具等にも指定され認定検査機関又は承認検査機関が行う適合性検査を受け、その証明書の交付を受たものでなければならない。(1)液化石油ガスこんろ(2) 液化石油ガス用瞬間湯沸器(半密閉式)(3)液化石油ガス用バーナー付ふろがま(半密閉式)(4)ふろがま(5)液化石油ガス用ふろバーナー(6)液化石油ガス用ストーブ(半密閉式)(7)液化石油ガス用ガス栓(8)液化石油ガス用ガス漏れ警報器(9) 液化石油ガス用継手金具付低圧ホース(10)液化石油ガス用耐震自動ガス遮断器(11)調整器(12)液化石油ガス用瞬間湯沸器(開放式、密閉燃焼式、屋外式)(13)液化石油ガス用継手金具付高圧ホース(14)液化石油ガス用バーナー付ふろがま(密閉燃焼式、屋外式)(15)液化石油ガス用ストーブ(開放式、密閉燃焼式、屋外式)
