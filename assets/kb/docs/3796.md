# 4・1・5 液化石油ガスの保安の確保及び取引の適正化に関する法

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・1 事故の防止に関する法律

律 「液化石油ガスの保安の確保及び取引の適正化に関する法律」
は，液化石油ガスの一般消費者に係わる保安の確保と取引の適正化
（f）液化石油ガス器具等の製造，輸入又は販売の事業を行う者
は，経済産業省令で定める表示が付されているものでなければ液化
石油ガス器具等を販売し，又は販売の目的で陳列してはならない．
「液化石油ガス器具等」には，次の15品目が指定されており，この
うち，（1）から（3）の液化石油ガス器具等が特に液化石油ガスの災害
の発生のおそれが多いと認められる特定液化石油ガス器具等にも指
定され認定検査機関又は承認検査機関が行う適合性検査を受け，そ
の証明書の交付を受たものでなければならない．
（1）液化石油ガスこんろ
（2） 液化石油ガス用瞬間湯沸器（半密閉式）
（3）液化石油ガス用バーナー付ふろがま（半密閉式）
（4）ふろがま
（5）液化石油ガス用ふろバーナー
（6）液化石油ガス用ストーブ（半密閉式）
（7）液化石油ガス用ガス栓
（8）液化石油ガス用ガス漏れ警報器
（9） 液化石油ガス用継手金具付低圧ホース
（10）液化石油ガス用耐震自動ガス遮断器
（11）調整器
（12）液化石油ガス用瞬間湯沸器（開放式，密閉燃焼式，屋外式）
（13）液化石油ガス用継手金具付高圧ホース
（14）液化石油ガス用バーナー付ふろがま（密閉燃焼式，屋外式）
（15）液化石油ガス用ストーブ（開放式，密閉燃焼式，屋外式）
