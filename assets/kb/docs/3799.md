# 4・2・1 環境保全に関する法規制の概要

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・2 環境保全に関する法律

a．環境問題とその対応の歴史
i．戦後から高度成長期 戦後の急速な経済復興に伴い，公害
問題が大工業地帯を中心として各地に顕在化していった．これに対
して大都市や工業地帯を持つ地方公共団体は工場・事業所に対する
規制を内容とする条例を制定した．東京都工場公害防止条例（1949
年），大阪府事業場公害防止条例（1950年夏，神奈川県事業場公害防
止条例（1951年）などがこれである．これらの条例は特定の工場や
事業場から生ずる騒音，ガス，粉じん，ばい煙，振動などを規制す
るために，事業主に届出義務，認可申請義務，防除措置義務などを
課すとともに，違反是正のための下命，勧告，立入検査権を定める
ものであった．以上のように戦後の1950年代前半までの環境問題に
対する法的対応は，もっぱら地方公共団体によって行われ，国に関
してはほとんど述べるものはなかった。その後1950年代後半からは
いくつかの公害規制に関する法律の整備がなされた．それは水質二
法（公共用水域の水質保全に関する法律，工場排水等の規制に関す
る法律）（1958年）とばい三等規制法（1962年）の制定である．前者
は国が特に指定した公共用水域について水質基準を定め，工場に対
し，これを遵守させるために必要な規制を加えることを内容とする
ものであった．後者のばい煙等規制法は国が指定地域を定め，そこ
では「すすその他粉じん」及び「亜硫酸ガス又は無水硫酸」の排出
が規制されたが，規制レベルは相対的な濃度方式のみで届出制をと
っていた．この間，水質汚染問題では1958年本州製紙江戸川工場で
工場の排水が原因で魚貝類に大きな被害が出るという事件が発生し
た．被害を受けた漁民が再三の交渉にも誠実に対応しない会社に抗
議するため工場に乱入するという行動を起こした．この事態を収拾
するため警察が導入され，多くのけが人が出る流血の乱闘騒ぎとい
うショッキングな事件に発展した．この事件がマスコミで大きく扱
われたため，工場排水対策の必要性を国に強く印象づけることとな
った。
ii．公害対策基本法の成立 1960年代前半の公害対策は，個々
の発生源を規制する個別規制であった．そのため，これらの措置の
実施によっても公害問題は解決されず，汚染状況は悪化していった．
そこで，Φ統一理念による総合的取組み ②応急的・臨時的対応
から予防的・計画的取組み ③公害の対象範囲，責任の明確化を望
む声が高まってきた．一方，公害行政を調査するため，総理府に関
係省庁事務次官等で構成する公害対策推進連絡会議が設けられた
（1964年3月目．この連絡会議は当面は騒音及び自動車排ガス対策を
中心に検討を行うこととなったが，やがて公害対策基本法律案の検
討の場となった．又，1965年に厚生省は厚生大臣の諮問機関として
公害審議会を設置し，公害に関する基本施策について諮問した．1967
年5月「公害対策基本法案」が閣議決定され・第55回国会に提出し
た．同法案は衆議院で「経済の健全な発展との調和」の規定が法案
の目的から分離され，法案の目的規定の第2項に留意事項的に規定
されるという修正を受けた後，1967年7月に可決成立し，同年8月
に公布・施行された．これを受けて1968年6月にこれまでのばい煙
等規制法に変わり，排出基準違反への直罰規定を導入した大気汚染
防止法が制定された．1970年のいわゆる公害国会では公害対策基本
法の一部改正，水質汚濁防止法の制定など14の公害関連法案が一挙
に成立し，1971年には環境庁が設置されるなど次々に国の法的，行
政的体制が整備された．一方，この間産業活動は一層活発化し，し
かも石油コンビナート等に事業場が集結していた事もあって，特定
の地域で大気汚染，水質汚濁等による深刻な健康被害を住民にもた
らし，いわゆる四大公害訴訟が発生した．（表4・13参照）

れを受けて，同年絶滅の恐れのある野生動植物の種の保存に関する
法律が制定された．このような国際的潮流を踏まえて1993年に国の
環境政策の基本方針を示す環境基本法が制定された。環境基本法は
近年の都市・生活型公害による環境の負荷の増大，地球規模の環境
問題や身近な自然の減少への対応を視野に入れたものであり，新た
な環境基本計画の策定，経済的措置や地球環境保全に関する国際協
力等の規定を盛り込むなど，従来の公害対策基本法には見られない
対策手法を導入している．
iv 地球環境問題等への対応 1987年に提唱された「持続的発
展」の概念は，1970年代の開発か環境保全かの対立概念を超えて，
世代間倫理を踏まえた環境と開発を統合する理念として，国際的に
定着するようになった．1992年6月に開催された国連環境開発会議
において採択されたアジェンダ21は，持続的発展の理念の具体化を
模索したものであるといえる．将来世代の環境に配慮した持続的発
展の実現のためには，南北間における抜本的改革の必要性に迫られ
ていることが指摘される．その一つは，先進国における現在の経済・
社会システムを地球にやさしい資源循環型社会システムにシフトさ
せること．二つは，開発途上国において生存にとっての基本的ニー
ズの充足を図り，貧困の克服や人口の増加を抑制しつつ，環境配慮
を組み込んだ経済開発政策を推進することであろう．さらには，現
在の貿易，投資，金融等の国際経済の構造の改善による南北格差の
縮小を図るべく，各国の自助開発努力を支援する国際環境協力政策
