# 4・2・1 環境保全に関する法規制の概要

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・2 環境保全に関する法律

a.環境問題とその対応の歴史i.戦後から高度成長期 戦後の急速な経済復興に伴い,公害問題が大工業地帯を中心として各地に顕在化していった.これに対して大都市や工業地帯を持つ地方公共団体は工場・事業所に対する規制を内容とする条例を制定した.東京都工場公害防止条例(1949年),大阪府事業場公害防止条例(1950年夏,神奈川県事業場公害防止条例(1951年)などがこれである.これらの条例は特定の工場や事業場から生ずる騒音,ガス,粉じん,ばい煙,振動などを規制するために,事業主に届出義務,認可申請義務,防除措置義務などを課すとともに,違反是正のための下命,勧告,立入検査権を定めるものであった.以上のように戦後の1950年代前半までの環境問題に対する法的対応は,もっぱら地方公共団体によって行われ,国に関してはほとんど述べるものはなかった。その後1950年代後半からはいくつかの公害規制に関する法律の整備がなされた.それは水質二法(公共用水域の水質保全に関する法律,工場排水等の規制に関する法律)(1958年)とばい三等規制法(1962年)の制定である.前者は国が特に指定した公共用水域について水質基準を定め,工場に対し,これを遵守させるために必要な規制を加えることを内容とするものであった.後者のばい煙等規制法は国が指定地域を定め,そこでは「すすその他粉じん」及び「亜硫酸ガス又は無水硫酸」の排出が規制されたが,規制レベルは相対的な濃度方式のみで届出制をとっていた.この間,水質汚染問題では1958年本州製紙江戸川工場で工場の排水が原因で魚貝類に大きな被害が出るという事件が発生した.被害を受けた漁民が再三の交渉にも誠実に対応しない会社に抗議するため工場に乱入するという行動を起こした.この事態を収拾するため警察が導入され,多くのけが人が出る流血の乱闘騒ぎというショッキングな事件に発展した.この事件がマスコミで大きく扱われたため,工場排水対策の必要性を国に強く印象づけることとなった。ii.公害対策基本法の成立 地球環境問題等への対応 1987年に提唱された「持続的発展」の概念は,1970年代の開発か環境保全かの対立概念を超えて,世代間倫理を踏まえた環境と開発を統合する理念として,国際的に定着するようになった.1992年6月に開催された国連環境開発会議において採択されたアジェンダ21は,持続的発展の理念の具体化を模索したものであるといえる.将来世代の環境に配慮した持続的発展の実現のためには,南北間における抜本的改革の必要性に迫られていることが指摘される.その一つは,先進国における現在の経済・社会システムを地球にやさしい資源循環型社会システムにシフトさせること.二つは,開発途上国において生存にとっての基本的ニーズの充足を図り,貧困の克服や人口の増加を抑制しつつ,環境配慮を組み込んだ経済開発政策を推進することであろう.さらには,現在の貿易,投資,金融等の国際経済の構造の改善による南北格差の縮小を図るべく,各国の自助開発努力を支援する国際環境協力政策β9-107が必要不可欠となる.これを受けて,環境基本法は国際的協調による地球環境保全の積極的推進(5条)という基本理念とその推進にあたっての国の責務(6条)を明確にするとともに,地球環境保全及び開発途上地域の環境保全に関する国際協力の努力義務(32条)を課し,国内外に日本の環境政策上の立場を明確にしている・廃棄物・リサイクル行政に対する社会的要請に応えるために,2000年6月に循環型社会形成推進基本法と個別法が制定された。循環型社会形成推進基本法は,社会における物質の循環を形成し,製品などの使用・廃棄に伴う天然資源の消費抑制と環境負荷の低減を目的としており,循環型社会の形成に向けた基本原則,施策の基本事項などの対策の枠組みが示されている.b.現状の法体系i.体系 環境法はその法的共通性から大別すると,環境基本法の元に, 1環境規制法2地球環境保全法3環境保全法4環境整備法5費用負担・財政措置法6公害の被害救済・紛争処理法7環境影響評価法,及び,環境型社会形成推進基本法となる.ii.環境基本法 環境基本法は環境行政の目標,施策の体系等の基本的方向性を定める法であり,全体的・一般的な施策の方針を規定し,その具体化は各個別法に委ねるものである.(項目4・2・2および表4・14参照)環境基本法は総則,基本的施策及び環境審議会等の3つの章から成り立っている.総則では,基本理念として,1環境の恵沢の享受と継承等(第3条)2環境への負荷の少ない持続的発展が可能な社会の構築等(第4条) 3国際的協調による地球環境保全の積極的推進(第5条)を明らかにしている.次に責務として,1国(第6条)2地方公共団体(第7条)3事業者(第8条)4国民(第9条)などのそれぞれの責務を規定している.基本的施策は,1環境基本計画(第15条)2環境基準(第16条)3公害防止計画(第17条)4環境影響評価の推進(第20条)5規制的手法(第21条)6経済的手法(第22条)7地球環境保全等に関する国際協力等(第32~35条)などを規定している.iii.環境規制法 環境規制法は環境汚染の原因となる事:業活動,人の活動等を規制したり,制限したり,禁止したりする法律である.具体的には大気汚染,水質汚濁等の公害発生の原因となる一定の施設等について排出基準を定め,基準遵守のための規制手段を定める法律である.具体的には大気汚染防止法,水質汚濁防止法,騒音防止法等がある.(1) 環境規制 各種の個別規制法が排出基準を設定し,それを一律基準もしくは上乗せ基準として個別の工場等の発生源に適用する.ことにより,環境規制を行っている.公害発生施設の設置者は環境規制法により排出基準の遵守義務を課せられている.規制行政庁は発生源の監視のため,1設置の届出 2立入調査 3改善命令の報告徴収等の権限をもっており,排出基準違反がある場合には行政罰の適用がある.特に大気汚染防止法,水質汚濁防止法の排出基準違反や廃棄物の処理及び清掃に関する法律に基づく不法投棄事犯には直罰主義が採用されている.(2) 環境基準と排出基準 公害防止協定 地方公共団体の主要な環境規制手続どして「公害防止条例」「公害防止協定」「行政指導」がある.地方公共団体は法律又は条例に基づいて公害発生源の取り締まりに加えて,公害発生の恐れのある事業者と個々に折衝し,各種の公害防止措置や発生時の対応策を事業者に約束させ,これを協定の形式で取りまとめ,独自の環境規制を実施することも多い.このように地方公共団体と事業者が,公害防止のために事業者がとるべき措置について取り決める相互の合意を一般に「公害防止協定」という.この方式については,1964年に横浜市と根岸臨海工業地域に進出を予定していた電源開発株式会社及び東京電力株式会社で締結された協定が本格的な公害防止協定の始まりと考えられている.協定の内容は事業者に公害防止措置を講じることを約束させるものとなっているが,環境目標レベル,対策内容,達成時期等は企業と自治体の合意のもとで設定されており,企業の自主的な対応を盛り込んだものとなっている.公害防止協定は法的拘束力を認めるか否かで紳士協定とする立場と契約と解釈する立場に大きく分かれているが,ある程度の拘束力を有する契約であるとする見方が一般的である。iv。地球環境保全法 地球環境保全法は,例えば気候変動枠組み条約における温暖化防止の対策のようにグローバルな地球的規模における環境問題を国際間の条約や法制度により環境保全型に規制,誘導,助成する法である.具体的には地球温暖化対策の推進に関する法律,オゾン層保護法,海洋汚染防止法等がある.地球環境管理に関する法・政策の対応は国際法の領域で実験的な試みが行われ,先進的にいくつかの国で国内政策として導入され,その後これらが国際的な取り決めとなって各国の国内政策として普遍化する傾向がある.地球温暖化問題に代表される地球環境問題には従来型の規制的手法に加えて,自主的取り組みを促進させるための制度的枠組み,排出量取引,環境税に代表される経済的手法等が注目を集めている.v.環境保全法 環境保全法はすぐれた自然環境や景観,歴史的環境等の保全にかかる法である.現在ある程度の広がりをもって良好な環境を形成している地域を指定して,その地域を保全するため一定の開発行為への規制手段として地域地区制(ゾーニング)を用いたり,土地所有者の権利を制限する等の手段を採用している.具体的には自然環境保全法や自然公園法などがある.vi.環境整備法 環境整備法は下水道,廃棄物処理施設等の社会的インフラストラクチャーの整備を主体とする地方公共団体等が行う,いわば公共サービスにかかる法である。vii. 費用負担・財政措置法 費用負担・財政措置法は公害防止事業の事業者負担や地方公共団体への財政措置,事業者に対する助成措置を定める法.viii.公害の被害救済,紛争処理法 公害健康被害補償法等がある.ix.環境影響評価法 環境影響評価法は平成9年6月に公布され,順次施行され,平成11年6月に本格的に施行される事となった.その骨子とするところは,国は事業者が事業に係る環境への影響について自ら適正に調査,予測又は評価を行い,その結果に基づき,、その事業に係る環境の保全について適正に配慮することを推進するため,必要な措置を講ずるものとしている.x.循環型社会形成推進基本法 循環型社会形成推進基本法は平成12年6月に公布・施行され,環境基本法の基本理念にのっとり,循環型社会の形成について基本原則を定め,並びに国,地方公共団体,事業者及び国民の責務を明らかにするとともに循環型社会形成推進基本計画の策定,その他循環型社会の形成に関する施策の基本となる事項を定めることにより,循環聖社会の形成に関する施策を総合的かつ計画的に推進する事を目的としている.循環型社会形成推進基本法のもとで,資源有効利用促進法の拡充整備と廃棄物処理法の拡充強化が行われ,既に,容器包装リサイクル法,家電リサイクル二等の公布・施行が行われている.
