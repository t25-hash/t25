# 4・2・2 環境保全一般

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・2 環境保全に関する法律

a.環境基本法 環境基本法は,公害対策基本法(昭和42年制定,昭和45年改訂)及び自然環境保全法(昭和47年制定)にかわって,平成5年11月に公布・施行された.i。成立の経緯 環境問題は地球環境という空間的な広がりと将来の世代にわたる影響という時間的な広がりを持つ問題として認識され,又,環境問題は我が国にとって重要な政策課題であるばかりでなく,人類の生存基盤としての有限な環境を守り,次の世代へと引き継いでいくという人類共通の課題として認識されるに至った.我が国では,かつて経済の高度成長期において,環境汚染や自然破壊が大きな社会問題となり,これに対処するため,昭和42年の公害対策基本法の制定とこれに引き続く昭和45年の公害関係14法の制定又は改正(公害国会),昭和47年の自然環境保全法の制定等により,鋭意対策を推進した.その結果,国民や企業の努力とあいまって激甚な公害の克服や優れた自然環境の保全に相当の成果を収めた.しかし,その後の経済的発展の中で,物質的にはより豊かになったものの,大量生産,大量消費,大量廃棄型の社会経済活動が定着するとともに人口や社会経済活動の都市への集中が一層進展した.その中で大都市における大気汚染や生活排水による水質汚濁等の都市・生活型公害等の改善は依然として進まず,又,廃棄物の増大等による環境負荷の増加が一層激しさを増している.地球温暖化やオゾン層の破壊,海洋汚染,野生生物の種の減少など,地球的規模で対応すべき地球環境問題が生じ,人類の生存の基盤であるかけがえのない地球環境が損なわれる恐れが顕在化してきた。以上のような環境問題に適切に対処し,環境の恵沢を現在及び将来の国民が享受していくためには,公害対策基本法や自然環境保全法のような問題対処型の法的枠組みでは不十分であるという認識の下,社会経済活動や国民の生活様式の在り方を含め,社会全体を環境への負荷の少ない持続的発展が可能なもりに変えていくととが必要である.そのために環境保全に関する各般の施策を総合的,計画的に進めていく法的枠組みとして環境基本法が成立した.ii.位置付け 環境基本法は環境行政の目標,施策の体系等の基本的方向性を定める法であり,全体的・一般的な施策の方針を規定し,その具体化は各個別法に委ねるものである.すなわち,環境基本法は憲法と個別法との間をつなぐものであり,具体的施策は個別の法律,予算上の措置,行政上の実施要綱等により行われる.表4・15 環境基本法第3条から第5条に定める環境の保全についての基本理念は環境をなぜ,また,どのように守っていくのかという行動原理を明らかにするものである.具体的には,環境の恵沢の享受と継承等(第3条),環境への負荷の少ない持続的発展が可能な社会の構築等(第4条),国際的協調による地球環境保全の積極的推進(第5条)からなる.2責務 基本理念の実現に向けて,国や地方公共団体が環墳の保全に関する施策を講じていくことはもちろんであるが・事業者や国民も,事業活動や日常生活において環境への負荷を減らすように努めるなど,進んで環境保全のために行動することが必要である.環境基本法では環境の保全の上でそれぞれの主体が果たすべき役割を責務として規定している.具体的には,国の責務:環境の保全に関する基本的・総合的な施策の策定・実施(第6条),地方公共.団体の責務;国の施策に準じた施策や地域の自然的社会的条件に応じたその後の施策の策定・実施(第7条),事業者の責務:事業活動に伴って生ずる公害の防止,自然環境の適正な保全のために必要な措置を講ずること他(第8条),国民の責務:日常生活に伴う環境への負荷の低減に努めること他(第9条)である.(2) 基本的施策1 環境基本計画(第15条) 広範多岐にわたる環境保全に関する施策を有機的連携を保ちろつ,全ての主体の公平な役割分担の下,長期的な観点から総合的・計画的に推進するため,政府全体の環境の保全に関する施策の基本的な方向を示す環境基本計画を定めることを規定している.2 環境基準(第16条) 政府は大気の汚染,水質の汚濁,土壌の汚染及び騒音に係る環境上の条件について,それぞれ人の健康を保護し,及び生活環境を保全する上で維持されることが望ましい基準(環境基準)を定めるものとする.3公害防止計画(第17条) 内閣総理大臣は,関係都道府県知事に対し,その地域において実施されるべき公害の防止に関する施策に係る基本方針を示してその施策に係る計画(公害防止計画)の策定を指示するものとしている.β9-1094 環境影響評価の推進(第20条) 国は土地の形状の変更,工作物の新設その他これらに類する事業を行う事業者が,その事業に係る環境への影響について自ら適正に調査,予測又は評価を行い,その結果に基づき,そあ事業に係る環境の保全について適正に配慮することを推進するため,必要な措置を講ずるものとすると規定している.環境影響評価法は平成9年6月に公布され,順次施行され,平成11年6月に本格的に施行された.5 規制的手法(第21条) 国は環境の保全上の支障を防止するため,規制の措置を講じなければならないとレている.6 経済的手法(第22条) 経済的手法とは環境への負荷の低減のための行動を誘導するための経済的なインセンティブ(動機)と定義し,国は環境の保全上の支障を防止する効果,我が国の経済への影響等について適切に調査・研究し,これらの調査・研究等を踏まえ,個別の措置を講ずる必要がある場合には経済的な負担を課.す施策を活用して環境の保全上の支障を防止することについて国民の理解と協力を得るように努めることとしている.7 地球環境保全等に関する国際協力等(第32~35条) 基本理念の一つとして「国際的協調による地球環境保全の積極的推進」(第5条)を揚げるとともに,第32~35条において地球環境保全等に積極的に取り組んでいくことを内外にアピールした.(3) 環境審議会等 中央環境審議会等を設置する事を規定b.環境基本計画i.位置付け 環境基本法第15条において,政府は環境の保全に関する施策の総合的かつ計画的な推選を図るため,環境の保全に関する基本的な計画(環境基本計画)を定めなければならないとしている.又,環境基本計画は 1環境の保全に関する総合的かつ長期的な施策の大綱 2環境の保全に関する施策を総合的かつ計画的に推進するために必要な事項を織り込む事とされている.環境基本計画は平成6年12月に策定され平成12年12月に第1回目の見直しが行われている.ii.内容 平成12年12月に策定された新環境基本計画について概説する.(1) 新環境基本計画の構成 新環境基本計画は第1部~第4部で成り立つ.環境基本法の環境政策の理念を実現し,持続可能な社会で構築するために「循環」「共生」「参加」「国際的取り組み」という四つの長期的目標を揚げている.新環境基本計画の構成を図4・6に示す.・第一に「循環」とは環境への負荷をできる限り少なくし,循環を基調とする社会経済システムの実現を目指すことである.このため,社会経済活動の全段階を通じて資源やエネルギーの利用の面でより一層の効率化を図り,再生可能な資源の利用の推進等,物質循環をできる限り確保するという考え方である.・第二に「共生」とは環境の特性に配慮しながら健全な生態系を維持,回復し,自然と人間との共生の確保を目指すことである.また,社会経済活動を自然環境に調和したものとしながら,自然とのふれあいを図り,自然と人との間に豊かな交流を保って行く必要がある.・第三に「参加」とはあらゆる主体が環境への負荷の低減や環境の特性に応じた賢明な利用などに自主的積極的に取り組み,環境保全に関する行動へ主体的に参加する事である.・第四に「国際的取り組み」とは,国際社会の舞台における我が国からの積極的な取り組みにより地球環境の保全への着実な寄与を目指すものである.(2)
