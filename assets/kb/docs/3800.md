# 4・2・2 環境保全一般

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 4・2 環境保全に関する法律

a．環境基本法 環境基本法は，公害対策基本法（昭和42年制
定，昭和45年改訂）及び自然環境保全法（昭和47年制定）にかわっ
て，平成5年11月に公布・施行された．
i。成立の経緯 環境問題は地球環境という空間的な広がりと
将来の世代にわたる影響という時間的な広がりを持つ問題として認
識され，又，環境問題は我が国にとって重要な政策課題であるばか
りでなく，人類の生存基盤としての有限な環境を守り，次の世代へ
と引き継いでいくという人類共通の課題として認識されるに至った．
我が国では，かつて経済の高度成長期において，環境汚染や自然
破壊が大きな社会問題となり，これに対処するため，昭和42年の公
害対策基本法の制定とこれに引き続く昭和45年の公害関係14法の制
定又は改正（公害国会），昭和47年の自然環境保全法の制定等により，
鋭意対策を推進した．その結果，国民や企業の努力とあいまって激
甚な公害の克服や優れた自然環境の保全に相当の成果を収めた．
しかし，その後の経済的発展の中で，物質的にはより豊かになっ
たものの，大量生産，大量消費，大量廃棄型の社会経済活動が定着
するとともに人口や社会経済活動の都市への集中が一層進展した．
その中で大都市における大気汚染や生活排水による水質汚濁等の都
市・生活型公害等の改善は依然として進まず，又，廃棄物の増大等
(C)日本機械学会
による環境負荷の増加が一層激しさを増している．地球温暖化やオ
ゾン層の破壊，海洋汚染，野生生物の種の減少など，地球的規模で
対応すべき地球環境問題が生じ，人類の生存の基盤であるかけがえ
のない地球環境が損なわれる恐れが顕在化してきた。
以上のような環境問題に適切に対処し，環境の恵沢を現在及び将
来の国民が享受していくためには，公害対策基本法や自然環境保全
法のような問題対処型の法的枠組みでは不十分であるという認識の
下，社会経済活動や国民の生活様式の在り方を含め，社会全体を環
境への負荷の少ない持続的発展が可能なもりに変えていくととが必
要である．そのために環境保全に関する各般の施策を総合的，計画
的に進めていく法的枠組みとして環境基本法が成立した．
ii．位置付け 環境基本法は環境行政の目標，施策の体系等の
基本的方向性を定める法であり，全体的・一般的な施策の方針を規
定し，その具体化は各個別法に委ねるものである．すなわち，環境
基本法は憲法と個別法との間をつなぐものであり，具体的施策は個
別の法律，予算上の措置，行政上の実施要綱等により行われる．

iii．内容 環境基本法は総則，基本的施策及び環境審議会等の
三つの章から成り立っている。
（1） 総則
① 基本理念 環境基本法第3条から第5条に定める環境の
保全についての基本理念は環境をなぜ，また，どのように守ってい
くのかという行動原理を明らかにするものである．具体的には，環
境の恵沢の享受と継承等（第3条），環境への負荷の少ない持続的発
展が可能な社会の構築等（第4条），国際的協調による地球環境保全
の積極的推進（第5条）からなる．
②責務 基本理念の実現に向けて，国や地方公共団体が環
墳の保全に関する施策を講じていくことはもちろんであるが・事業
者や国民も，事業活動や日常生活において環境への負荷を減らすよ
うに努めるなど，進んで環境保全のために行動することが必要であ
る．環境基本法では環境の保全の上でそれぞれの主体が果たすべき
役割を責務として規定している．具体的には，国の責務：環境の保
全に関する基本的・総合的な施策の策定・実施（第6条），地方公共．
団体の責務；国の施策に準じた施策や地域の自然的社会的条件に応
じたその後の施策の策定・実施（第7条），事業者の責務：事業活動
に伴って生ずる公害の防止，自然環境の適正な保全のために必要な
措置を講ずること他（第8条），国民の責務：日常生活に伴う環境へ
の負荷の低減に努めること他（第9条）である．
（2） 基本的施策
① 環境基本計画（第15条） 広範多岐にわたる環境保全に関
する施策を有機的連携を保ちろつ，全ての主体の公平な役割分担の
下，長期的な観点から総合的・計画的に推進するため，政府全体の
環境の保全に関する施策の基本的な方向を示す環境基本計画を定め
ることを規定している．
② 環境基準（第16条） 政府は大気の汚染，水質の汚濁，土
壌の汚染及び騒音に係る環境上の条件について，それぞれ人の健康
を保護し，及び生活環境を保全する上で維持されることが望ましい
基準（環境基準）を定めるものとする．
③公害防止計画（第17条） 内閣総理大臣は，関係都道府県
知事に対し，その地域において実施されるべき公害の防止に関する
施策に係る基本方針を示してその施策に係る計画（公害防止計画）
の策定を指示するものとしている．
