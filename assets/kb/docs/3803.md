# 1・3・1 インタネットとは

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・3 インタネット

インタネット（internet ）とは，インタネットプロトコル（IP:
internet protocol ）を用いて多種多様なコンピュータを相互接続す
ることによりさまざまなコンピュータ資源やサービスを共用するネ
ットワーク基盤である．インタネットは世界に広く分布するコンピ
ュータ資源の利用を可能とし，現在では単なる物理的な通信ネット
ワークという枠を超えて，そのうえでさまざまな知識，情報ベース
が形成され，また物理的位置の制約を克服する協調作業環境として
の世界最大の情報基盤となっている．
インタネットはアメリカ国防総省の実験用コンピュータネットワ
ーク（ARPANET）をその起源としている．これは，戦時下にお
ける攻撃破壊にあっても確実な通信の実現を目的としたものであ
り，この結果，コネクションを必要とせず，柔軟な経路制御が可能
な通信プロトコルであるインタネットプロトコル（IP）が開発さ
れた．さらに，IPの上位レイヤにTCP（transmission control
protocol ）を設けることによって，信頼性の高い通信が容易に実現
できるようになった．
1980年代に入ると，IP技術は，NSFNet等のアメリカ政府が所
有する他のコンピュータネットワークだけではなく，それらとは無
関係な商用コンピュータにも実装されるようになった．特に，低価
格化の進みつつあったワークステーション上のバークレー版
UNIXにTCP/ IP（transmission control protocol /internet pro-
tocol ）通信プロトコルとそれをアプリケーションから容易に利用
できるインタフェースが標準搭載されたことにより，大学，研究機
関を中心に爆発的に普及した．この結果，研究用ネットワーク間の
相互接続が急激に進んで，現在のインタネットの母体が誕生したの
である．
その後，TCP/ IPはスーパーコンピュータやマイクロコンピュー
タにも搭載されるようになり，マルチベンダコンピュータネットワ
ークの事実上の標準として利用されることになった．
現在では，既存のIP（IPv4）の次世代版であり，IPv4よりも
はるかに広大なアドレス空間をもつIPv6の導入も盛んになりつつ
あり，IP技術の普及をさらに促進するものとなっている．
