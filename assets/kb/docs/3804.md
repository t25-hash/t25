# 4・2・6 水質汚濁

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 10・15 M

a.水質汚濁問題の背景 河川や地下水など流れる水には,汚染物質を希釈,拡散,分解する自然の働きがあるため,環境中に汚染物質が侵入しても,その濃度は時間的・空間的に減衰する。しかし,これらの自然浄化力を上回る汚染物質の負荷が生じれば,水質の汚濁が発生し,ついには工業用,飲料用,農業用などの水資源や,生物を育み人に安らぎを与える環境資源としての価値を損ねるだけでなく,人の健康に重大な影響を与えるおそれもある.近代の水質汚濁の歴史は足尾鉱山問題に遡るが,昭和30~40年代にかけての高度経済成長期に水俣病やイタイイタイ病をはじめとした深刻な水質汚濁問題が発生したことは記憶に新しいことである.これらの水質汚濁問題解決のため,1970年4月に水質の汚濁に係る環境基準が定められ,そして同じ年に開催された,第64臨時国会(いわゆる公害国会)において,旧水質保全法と旧工場排水規制法を抜本的に改正し統合した水質汚濁防止法が制定され,我が国における水質汚濁防止のための法体系が整備された.水質汚濁防止法は公害対策基本法β9-118表4・37 水質汚濁による環境への影響を防止するための行政上の目標として,環境基本法第十六条に基づき水質汚濁に係る環境基準が定められている.水質の汚濁による環境への影響は,1カドミウム等の有害物質を含む環境水を飲用に供することや有害物質が蓄積した魚介類を食することによる人の健康への影響,2自然環境や快適な生活,水利用への影響に区分されるが,それぞれに対応した「人の健康の保護に関する環境基準」,「生活環境の保全に関する環境基準」が定められている。前者には全国一律の基準値が設定されているが,後者の基準値は公共用水域の種類(河川,湖沼,海域)及び利水域類型ごとに設定されている.また,地下水の水質汚濁に係る環境基準も定められているが,これは人の健康の保護に関する環境基準と同じ基準値となっている.c.水質汚濁防止法i.法の対象 この法は,工場及び事業場から公共用水域に排出される水の排出及び地下に浸透する水の浸透を規制するとともに,生活排水対策の実施を推進すること等によって,公共用水域及び地下水の水質汚濁の防止を図り,これにより国民の健康を保護するとともに生活環境を保全すること等を目的としている.ここで,公共・用水域には,河川,水路,湖沼,港湾,沿岸海域,及び終末処理場を持たない下水道が含まれる.また,水質汚濁防止法の適用となる事業場は,特定施設を設置する工場又は事業場(特定事業場)および,油を貯蔵する施設や油を含む水を処理する施設を設置する工場又は事業場(貯油施設等)である.ii.特定施設 特定施設とは,人の健康に係る被害を生じる恐れや生活環境に係る被害を生じる恐れがある物質を政令で定められた以上に含む汚水又は廃液を排出する施設であり,政令によって指定されたものである.特定施設は第一次産業から第三次産業まで広く指定されている.特定施設の設置あるいは構造等の変更を行う場合には,あらかじめ都道府県知事への届出が義務付けられている.また,届出られた特定施設の排出水が排水基準に適合しないか,あるいは地下浸透水に有害物質が含まれると認められるときには,都道府県知事は計画あ変更文は廃止を命じることができることとなつている。iii,排水基準 排水基準は,排出水の汚濁状態について総理府令で定めるとされている.これは,全ての公共用水域に最低限適用される全国一律の基準である(一律排水基準 表4・38)。一方,一律排水基準に示された項目について,都道府県条例によって一律排水基準よりもきびしい排水基準(上乗せ排水基準)を定めることができるとされている.さらに,地方公共団体は,一律排水基準が定められた項目以外の項目あるいは特定事業場以外の事業場から排出される水に対する規制を条例で定める(横だし基準)ことができる.排水基準は排出水の濃度を規制することを目的としているが,特定施設の増加による水域への負荷量の増大に対応するものではない.そこで,昭和53年の法改正により,広域的な閉鎖性水域の汚濁負荷量を総合的に削減するため,水質総量規制制度が導入されている.水質総量規制の対象項目は化学的酸素要求量等の生活環境の保全に関する項目の中から指定され(指定項目),指定項目ごとに特定の広域的な閉鎖性水域の指定(指定水域),及び指定水域の水質の汚濁に関係のある地域(指定地域)が定められる.現在,東京湾と伊勢湾が指定水域となっている.これらの指定水域に係る指定地域内の特定事業場で日平均排水量50m3以上のものには,都道府県知事が定めた総量規制基準が適用されることとなる.なお,瀬戸内海沿岸地域の特定事業場で,日最大排水量が50m3以上のものには,瀬戸内海環境保全特別措置法に基づく許可又は届出が必要となる.特定事業場から公共用水域に水を排出するものは,排水口において排水基準に適合しない水を排出してはならない.これに違反した場合には,直ちに罰則が科せられる(直罰).また,都道府県知事は,排出水を排出する者が排水基準に適合しない排出水を排出するおそれがある場合には,特定施設の構造や汚水の処理方法の改善,排出の一時停止などの改善措置を命令することができる.iv.特定地下浸透水の浸透の制限 特定施設から公共用水域に排水する者,地下に汚水を浸透させる者は,排出水,浸透水の状態を測定し,その結果を記録しておかなければならない.特定施設あるいは貯油施設の破損等の事故により,有害物質又は油を含む水が公共用水域に排出され,あるいは地下に浸透し,人の健康又は生活環境に係る被害を生じるおそれがあるときには,引き続く排出等を防止するための応急の措置を講ずるとともに,速やかに都道府県知事に届出なければならない。vi.地下水の水質の浄化に係る措置命令等 特定事業場から有害物質が地下に浸透したことにより,人の健康に係る被害が生じ,又は生じるおそれがあるときは,都道府県知事はその被害を防止するために必要な限度において,当該特定事業場の設置者(相続又は合併により地位を継承した者を含む)に対し,汚染地下水の揚水処理や土壌の浄化など,地下水の水質浄化のための措置を命じることができる.d.水質汚濁防止に係る関連法 水質汚濁防止法に加えて以下の各種関連法においても水質汚濁防止のための措置が規定されており,水質汚濁防止怯とあわせて水環境保全のための法体系を形成している.i.下水道法 下水道法は,都市の健全な発達および公衆衛生の向上に寄与することとともに,公共用水域の水質の保全に役立てることを目的としている.水質汚濁防止法の特定施設の設置者が公共下水道を使用して下水を継続して排除する場合には,下水道法が適用されることとなる.この場合の水質の保全に関する取扱いは水質汚濁防止法と同等であり,公共下水道管理者への使用開始の届出義務,特定施設の設置・変更等の届出義務,政令で定める水質項目についての基準(下水道受け入れ基準)に適合しない下水排除の禁止が定められている.ii.鉱山保安法 鉱石の採掘等鉱業活動を行う事業には鉱山保安法が適用される.この法律では,鉱業権者は廃水の処理に伴う危害又は公害の防止のための必要な措置を講じなければならないとされている.具体的には,鉱山保安規則により水質汚濁防止法の排水基準に基づいた坑廃水排出基準が定められており,この基準に適合しない坑廃水の公共用水域への排出の禁止,および有害物質を含む坑廃水の地下浸透の禁止が定められている.iii.電気事業法 発電や送電等,電気の供給を行う事業には電気事業法が適用される.同法では,事業用電気工作物を設置する者は,これが人体に危害を及ぼさないよう技術基準に適合するように維持しなければならないとされている.具体的には,電気設備に関する技術基準を定める省令によって,発電所又は変電所,開閉所若しくはこれらに準ずる場所から排出される排出水は排水基準に適合させなければならないこと,貯油施設等の破損による事故時の措置などが定められている.iv.湖沼水質保全特別措置法 水質の保全に関する施策を総合的に講ずる必要がある湖沼は指定湖沼とされ,その水質汚濁に関係のある地域が指定地域として定められている.指定地域内の特定事業場で一定の規模以上のものから公共用水域に排出される水の汚濁負荷量は指定湖沼の水質を保全するための規制基準を遵守しなければならない.v.海洋汚染および海上災害の防止に関する法律 海洋汚染と海上災害を防止することを目的とする法律で,船舶等から海洋に油,廃棄物等を排出すること等を規制したものであり,油,有害液体物質,廃棄物等が排出された場合の防除措置などが規定されている.vi.農薬取締法 農薬の登録制度や使用等の規制等により,農薬の安全な使用の確保を図り国民の生活環境の保全に寄与することβ9-120等を目的とする.この法律では,人畜に被害を生ずるおそれがある公共用水域の水質汚濁の原因となる等の要件を備えた農薬を水質汚濁性農薬として指定し,その規制が行われている.vii.化学物質の審査及び製造等の規制に関する法律 難分解性の性状を有し,かつ,人の健康を損なうおそれがある化学物質による環境の汚染を防止するため,新規の化学物質の製造又は輸入に際し事前にその化学物質が難分解性等の性状を有するかどうかを審査する制度を設けるとともに,その有する性状等に応じ,化学物質の製造,輸入,使用等について必要な規制を行うことを目的とする。e.ダイオキシン類対策特別措置法 ダイオキシン類が人の生命および健康に重大な影響を与えるおそれがある物質であることから,ダイオキシン類による環境の汚染の防止とその除去等をするための必要な措置を定め,国民の健康の保護を図ることを目的に制定された(平成11年7月).この法律では,ダイオキシン類による大気の汚染,水質の汚濁,および土壌の汚染に係る環境上の条件として,人の健康を保護する上で維持されることが望ましい基準(環境基準)が定められている(水質の基準は1P9-TEQ/リットル以下).また,工場又は事業場に設置される施設のうち,ダイオキシン類を大気,水域へ排出する施設を「特定施設」とし,特定施設から排出される排出ガス又は排出水の基準が定められている(水質排出基準:水質の基準は10P9-TEQ/リットル).水質汚濁防止法と同様に,ダイオキシン類対策特別措置法でも,特定施設の設置,変更,廃止等の際の届出義務,都道府県知事による計画変更命令や改善命令,事故時の措置等が定められている.排出基準等に違反した場合,直ちに罰則が科せられることも同様である.
