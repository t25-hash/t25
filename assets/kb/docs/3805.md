# 4・2・7 土壌汚染・地盤沈下

> **章**: 第4章 工場・施設周辺に係わる法制度
> **節**: 10・15 M

a.土壌汚染 土壌は,大気,水とともに環境の主要な構成要素であり,人をはじめ植物,小動物,微生物等の生育の場として,また雨水を貯留し地下水を酒養する場としての重要な役割を担っている.しかし,土壌は大気や水と比べてその組成が複雑で有害物質に対する反応も多様であり,また,いったん汚染されるとその影響が長期にわたり持続する蓄積性の汚染となる等の特徴も有している.土壌汚染の歴史は,足尾鉱山から流れ出た鉱毒水により広い範囲の農用地が汚染されたことに遡ることができる.この土壌汚染は農作物の生育阻害を生じさせるものであったが,人の健康への影響のおそれがある土壌汚染問題の顕在化は,神通川流域のイタイイタイ病の原因となったカドミウムによる環境汚染が周辺の農用地の土壌にも及んでいることが判明してからである。これを契機にカドミウムによる農用地の汚染ボ全国各地で顕在化したため,土壌汚染防止対策が緊急な課題としてとりあげられ,昭和45年末に開催された第64臨時国会(いわゆる公害国会)における公害対策基本法の一部改正により,典型公害の一種として新たに土壌汚染が追加された.また,土壌の汚染に係る環境基準を設定することとされるとともに,その実施法として,農用地の土壌の汚染防止等に関する法律が制定された.この法律に基づき農用地の土壌汚染の防止および除去等の対策が推進されており,対策の必要がある地域の8割以上ですでに・対策が完了している.ただし,土壌中の有害物質の量と農作物の生育および農作物中の有害物質の量との因果関係がきわめて複雑である等から土壌の汚染に係る環境基準は,平成3年まで制定されなかった.一方,昭和50年代になって,東京都における六町クロム野津問題や国立研究所の移転に伴う土壌汚染の発覚など,工業活動や廃棄物の埋め立て等が原因となる土壌汚染問題による社会問題が発生した.また,土壌汚染が原因となった地下水汚染が全国的に拡がっていることが判明したことなどから,これら農用地以外の土壌汚染,いわゆる市街地における土壌汚染問題への取り組みが必要であることが認識されるようになった.そこで,平成3年8月に公害対策基本法に基づく 「土壌の汚染に係る環境基準」が制定され,また,平成11年1月には土壌汚染調査対策の進め方を示した「土壌・地下水汚染に係る調査・対策指針および運用基準」が策定された.そして,平成14年5月,我が国における土壌汚染対策の基本的なルールを示した,「土壌汚染対策法」が制定された.i.土壌の汚染に係る環境基準 土壌汚染は,有害物質を含む表流水,灌慨水,排水,工業材料,.廃棄物その他工業活動に伴う生成物などが土壌中に浸透することにより発生する.地中に浸透した汚染物質は土壌中に蓄積され,その移動や希釈はほとんど期待されない。このため,水質汚濁や大気汚染が現在の事業活動等が原因となるのに対し,土壌汚染の発生は過去の事業活動等による有害物質の蓄積が原因となっていることが多い.したがって,土壌汚染対策では,土壌中に有害物質が侵入し土壌汚染を発生させることを未然に防止する対策に加え,汚染された土壌の回復対策が重要となる。iii.土壌汚染未然防止対策 土壌を汚染するおそれがある有害物質の土壌への排出を規制した法律には以下のようなものがある.(1) 大気汚染防止法 煤煙の排出規制により煤煙中に含まれた有害物質が地表に降下し,土壌汚染を発生させることを防止する.(2) 水質汚濁防止法 有害物質を含む排水等の地下浸透禁止により,工場施設における直接的な土壌汚染発生を防止する。また,排水規制により,公共用水域を通じて拡散した汚染物質が土壌中に浸透し土壌汚染を発生することを防止する。(3)廃棄物の処理及び清掃に関する法律 土壌を汚染する恐れがある物質を含む廃棄物の埋立処分基準,最終処分場の構造・維持管理基準墨,廃棄物の適正処理確保のための諸規制を定めることにより,廃棄物の拡散,埋設された廃棄物からの有害物質の流出等が原因となる土壌汚染を防止する.(4) 鉱山保安法 鉱業廃水の処理に伴う危害又は公害の防止のための必要な措置を講じることにより,排水の拡散や地下への浸透による土壌汚染の発生を防止する.(5) 肥料取締法 汚泥を原料とする肥料等,有害成分を含有するおそれが高い肥料の登録,及び含有を許される有害物質の最大量の基準を定めることにより,農地における土壌汚染を防止する.(6) 農薬取締法 農薬の使用に係る農地等の土壌の汚染が生じ,かつ,汚染された農作物等の利用が原因となって人畜に被害を生ずるおそれがある種類の農薬を,土壌残留性農薬として指定し,その使用を規制する.(7) 条例等 千葉県等の自治体では,有害物質を含む建設発生土(土砂)による埋め立て等により,新たな土壌汚染が発生することを防止するため,汚染土砂の埋め立ての禁止と土砂埋め立て事業の許可を条例によって定めている.,iv.土壌汚染対策 土壌汚染対策法 この法は,土壌汚染の状況の把握,土壌汚染による人の健康被害の防止に関する措置等の土壌汚染対策を実施することにより,国民の健康の保護を図ることを目的とする.鉛,砒素,トリクロロエチレン等の特定有害物質の製造,使用または処理をする水質汚濁防止法の特定施設に係る工場・事業場の敷地であった土地が住宅地等に転用される場合,あるいは土壌汚染による人の健康への影響が生ずるおそれがあると都道府県知事が認めた場合に土壌汚染状況調査が行われる.この結果,土壌の汚染状態が基準に適合しない土地は指定区域に指定される.また,指定区域内の土壌汚染により人の健康被害が生ずるおそれがある場合には,汚染の除去等の措置が命令される.指定区域に指定された土地は,土地の形質の変更の制限等による管理を継続することが必要となるが,汚染の除去による措置により基準に適合する状態に回復した場合には,指定区域の指定が解除される.(3) 水質汚濁防止法 水質汚濁防止法の特定施設あるいは貯油施設の破損等の事故により,有害物質又は油を含む水が地下に浸透し,人の健康又は生活環境に係る被害を生じるおそれがあるときβ9-121には,引き続く排出等を防止するための応急の措置を講ずる必要がある.また,特定施設から有害物質を含む排水が地下浸透し地下水汚染が発生することにより,人の健康に係る被害が生じるか,そのおそれがあると認められる場合には,都道府県知事は当該特定施設の設置者等に対し,地下水の水質浄化のための措置をとることを命じることができるとされている.これらの措置には,土壌中の汚染物質の除去,地表の被覆や周囲の遮水等による土壌中の汚染物質の移動抑制等,土壌汚染対策も含まれる.(4) 廃棄物の処理及び清掃に関する法律 不法投棄等,廃棄物の処理基準に適合しない処分等により生活環境の保全上の支障が生じる場合,都道府県知下等はその支障の除去(原状回復)のための必要な措置を命じることができる.措置命令の対象者には不法投棄を行った者のほか,規定に違反する委託を行った者等も含まれる.(5) 条例等 東京都は,「都民の健康と安全を確保する環境に関する条例」により,1土壌汚染により人の健康に係る影響が生じるかそのおそれがあるとき,2地下水の汚染が認められるとき,3有害物質を取扱った落場等を廃止・除却するとき,及び43000m2以上の土地の改変時に,有害物質取扱い事業者及び土地改変者に対する汚染土壌の調査及び対策を義務付けている.また,同様に新潟県,神奈川県等でも,、条例により工場跡地の用途転換および再開発等の際に土壌汚染調査及び必要な対策等を実施するよう求めている.環境庁の実態調査(平成12年度)によると,都道府県及び水質汚濁防止法に定める水質汚濁防止法政令市,合計140地方公共団体のうち,46団体で土壌汚染対策に関連する条例等が制定されている.v.ダイオキシン類対策特別措置法 ダイオキシン類が人の生命および健康に重大な影響を与えるおそれがある物質であることから,ダイオキシン類による環境の汚染の防止とその除去等をするための必要な措置を定め,国民の健康の保護を図ることを目的に制定された(平成11年7月).この法律では,ダイオキシン類による大気の汚染,水質の汚濁,および土壌の汚染に係る環境上の条件として,人の健康を保護する上で維持されることが望ましい基準(環境基準)が定められている(土壌の汚染に係る環境基準は1000P9-TEQ19以下).また,工場又は事業場に設置される施設のうち,ダイオキシン類を大気,水域へ排出する施設を「特定施設」とし,特定施設か’ら排出される排出ガス又は排出水の基準を定めることにより,これらを経由した土壌汚染の未然防止が図られている.さらに,ダイオキシン類により汚染された土壌に係る措置として,土壌のダイオキシン類による汚染の除去等をする必要がある地域の指定(対策地域),土壌の除去,あるいは土地の利用等により人の健康に係る被害が生じることの防止措置等を含めた対策計画の策定が定められている.b.地盤沈下 環境基本法第二条では,公害とは,環境の保全上の支障のうち,事業活動その他の人の活動に伴って生じる相当範囲にわたる大気汚染などにより,人の健康又は生活環境に係る影響が生じること.とされている.鉱物の掘採のための土地の掘削によるものを除き,地盤の沈下もこれらの公害のひとつとしてあげられている.地盤沈下が発生する原因には,1重量構造物や埋め立て過重等による地層の変形,2地震等による地殻の変動,3石炭等の地下採掘跡地の崩落,4地下水位の低下による粘土層の圧密などがある.これらのうち,上記に示した公害の定義に該当する原因は,地下水位の低下による粘土層の圧密である.地下水は身近な水資源であり,一般に良好な水質を有しているため,水道用,工業用,農業用等として広く用いられてきた.しかし,過剰な地下水利用は地下水位を大きく低下させ,地盤沈下の原因となる.昭和30年代の高度経済成長期には,工業用を中心とした地下水利用が急増したため,これに伴い東京や大阪では年間20cmもの著しい沈下が発生した.また,東京都や大阪市における戦前からの累積沈下量は,最大で2m近くにまで達した.地盤沈下による被害には,建造物の破損や用水路の機能低下といった直接的なものと,高潮や出水時における浸水被害の拡大や地震時の建物への被害といβ9-122つた間接的なものがある.その代表的なものとして,1947年のカスリン台風1961年の第2室戸台風における水害被害があげられる.これらの地盤沈下に起因する被害の防止のためには地下水採取を規制する必要があることが認識されたため,1956年に工業用水法が制定され,1962年掛は建築物用地下水の採取の規制に関する法律(ビル用水法)が制定された.工業用水法とビル用水法による地下水採取の規制の枠組みはほとんど同じであり,指定地域内における一定規模以上の揚水設備を用いて地下水を採取する場合には,当該揚水設備のストレーナーの位置及び揚水機の吐出口の断面積が経済産業省令,環境省令で定める技術上の基準に適合していなければ許可されない.現在,工業用水法,ビル用水法は,それぞれ10都府県,4都府県の一部が指定地域となっている.一方,東京都や千葉県をはじめとした各自治体では,上記の法律の適用とならない地域や用途に対しても地下水採取の規制(努力規定,届出,許可,禁止等)が及ぶよう,独自の条例を制定している.これらの取り組みの結果,現在では地下水の採取が原因となる地盤沈下は,一部の地域を除きほぼ沈静化へと向かっている.
