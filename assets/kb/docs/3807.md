a.セルラシステムの基礎 図1･26を用いて小ゾーン（セル
ラ）システムの基本を，大ゾーンとの比較で説明する．二つの方式
を比べてわかる違いは，小さな多数のセルでサービスエリアを覆う
か，大きな一つのセルで覆うかの点にある．どちらの大きさのセル
も，その中心部に無線基地局を設置し，基地局とセル内の端末（携
帯電話）とが無線で通信をする．以下に，セルラシステムの特徴
（大ゾーン方式と比較）をまとめる．
① 小さなセルを用いることで，無線機の送信電力を小さくできる
（無線機の小形化，低コスト化．さらに低消費電力化による通
話時間の拡大）
② 干渉が生じない程度に距離を離すことで，異なるセルで同一の
周波数を利用できる（周波数の面的再利用による周波数利用率
の向上，第二世代移動通信では4～7セルごとの繰返し利用が
り，それぞれの特徴を表1･13にまとめる．GSMはヨーロッパや
可能）
③ セル境界をまたぎながらの通信も途切れることなく実現するた
めに，制御・交換局で瞬断を発生させないセル切換制御が必要
（良好な通話品質を確保するために，制御が複雑になる）
以上，端末の小形化・低コスト化，通話時間の拡大，周波数の有
効利用，通話品質の向上を実現するために，セルラ構成を採るとと
もに高機能なセル切換制御を適用していることを説明した．移動通
信の世代の進展に応じて，またユーザ数の増加に応じて，ますます
小セル化の傾向にある．なお，周波数は公共の資源であり，無線通
信において周波数の有効利用（与えられた周波数帯域の中で多くの
ユーザを収容）は必須の条件である．
b.セルラシステムの現状
.第二世代移動通信（second generation mobile radio sys-
tem) 第二世代移動通信にはGSM（global system for mobile
communications ），PDC（personal digital cellular ），NADC
（North American digital cellular ），およびcdmaOne方式があ
アジアなど世界で最も普及している方式である．PDCは日本でサ
ービスされており，周波数利用効率やi -modeサービスなど技術的
に先進性をもつ．cdmaOneは移動通信で初めてCDMA（code
division multiple access ）方式を実用化した点に特徴をもつ．
NADCはアメリカ中心にサービスされているが，それほど普及が
進んでいない．これら第二世代のユーザは2004年末で，日本では
5000万以上，世界では20億人に迫る増加である．今後，日本では
第二世代のユーザ数は（第三世代への移行により）減少するが，世
界ではさらに増加すると予想されている．なお，第二世代方式の反
省として，世界統一方式ではない点が挙げられる．
