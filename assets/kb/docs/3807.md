# 5・1・1 原子力、放射線利用の現状

> **章**: 第5章 放射性物質、有害物質に関する法制度
> **節**: 5・1 放射性物質に関わる特別の法制度

a.放射線の発見 放射線の存在が初めて発見されたのは、今から100年以上前の1895年である。ドイツの物理学者レントゲンは、陰極線(電子線)を発生させるための放電管から目に見えない何かが発せられ、それが物体を透過して写真乾板を感光させることを発見し、これを「X線」と名づけた。この発見の翌年、フランスのベクレルがウラン鉱石からある種の放射線が出ていることを発見した。1898年にはキュリー夫妻(マリー・キュリーとピエール・キュリー)が放射性核種のラジウムとポロニウムという新しい元素を発見し、この放射線を出す性質を放射能と呼んだ。同年にはイギリスのラザフォードがアルファ線とベータ線を、1900年にヴィラールがガンマ線を発見した。1938年には、ドイツのハーンとシュトラスマンが初めてウランの核分裂を確認し、この発見が原子爆弾の開発に利用された。核分裂に伴って放出されるエネルギーを熱に変換することにより、1951年にアメリカで初めて原子力発電が行われた。また、原子炉で作られた放射性同位元素が民間へ供給され、医療や生物学、化学の進歩を促進した。b.原子力利用 原子力の利用は原子力発電の導入という形で1960年代後半から先進各国で始まり、1999年末には425基、設備容量にして359百万kWに達し、世界の電力の約16%を供給し、一次エネルギー供給の約7.6%を担っている。これらの原子力発電所は主としてアメリカ、ヨーロッパ、日本に存在しており、これらの国々に世界の設備容量の約8割が設置されている。我が国では、1966年に最初の商業用原子力発電所が運転を開始して以来、1999年度には51基、総設備容量にして4492万kWの原子力発電所が稼働しており、国内総発電電力量9176億kWhのうち34.5%にあたる3165億kWhを供給している。これは一次エネルギーに換算すると原油換算約8200万ldに相当し、我が国一次エネルギー供給の約16%を担っている。原子力利用では発電、原子燃料サイクル、廃棄物処分等の各プロセスにおいて、作業に従事するものに対する放射線防謬が必要になるとともに、それぞれの施設から環境に放出される放射性物質による周辺公衆の防護が必要になり、「核原料物質、核燃料物質及び原子炉の規制に関する法律」を中心とした法体系の整備が行われている。c.放射線利用i.医学利用 放射線は、がんをはじめ心臓病、脳血管疾患、痴ほう等、さまざまな病気の診断やがん等の治療に使われている。私たちの生活の中で最も身近な放射線の利用例は、定期健康診断等でおなじみのX線撮影である。これは、物質中を放射線が透過する特性をいかして、普通では見ることができない体の内部を透視するものである。また、放射性同位元素を用いた核医学検査やPET(ポジトロン断層撮影)では、組織や臓器の機能を観察でき、全身のがんの転移巣を検出するなどの診断が可能である。放射線を用いた診断は、X線撮影を含め広く普及しており、放射線診断抜きには現在の臨床医学は成り立たないのが現状である。全国に約16万施設にある病院のX線撮影装置など医療用放射線照射装置については医療法関係により放射線障害の防止に関する規制を受けることになる。ii.農業利用 植物に放射線を照射することにより遺伝子に突然変異を起こさせて、その植物の品種改良をおこなうことができる。この方法は在来の品種にはない形質(遺伝的性質)についても突然変異で誘発することが可能である。放射線による品種改良により、すでに世界中で1800近い品種が育成された。我が国でも、農業生物資源研究所放射線育種場が中心となってガンマ線を用いた品種改良をおこなっており、我が国で新しく育成された品種はこれまでに110品種に達するなど、我が国の農業に大きく貢献している。たとえば、ナシの「ゴールド二十世紀」は、黒斑病に弱い「二十世紀」を耐病性品種にしたもので、この品種改良により使用する農薬の量を約半分に減らすことができた。このほかにも放射線の農業への利用として、食品の腐敗防止や保存期間延長を目的とした食品への放射線の照射、農薬を使わずに野菜や果物の害虫を駆除する技術としての不妊虫放飼三等がある。iii.工学利用 放射線は工学的な分野でもいろいろな形で利用されており、生活に役立ち、社会を豊かにしている。放射線が物を透過する能力を利用して、対象物を壊さずにそのものの内部を調べることができる。これはガンマ線ラジオグラフィ(放射線撮影)や中性子ラジオグラフィと呼ばれる非破壊検査として利用され、構造物の内部欠陥の発見や空港での荷物検査等に利用されている。日本において放射線利用に関わる事業所数は2000年度において医療、教育、研究機関及び民間企業等あわせて約5000事業所となっており、膨大な量の放射性同位元素(密封、非密封線源)や1000以上におよぶサイクロトロンや直線加速装置などの放射線発生装置を使用している。したがって、これらの事業所において放射性同位元素を取り扱う作業に従事するものや、事業所周辺公衆に対する放射線防護が適切におこなわれるよう「放射性同位元素等による放射線障害の防止に関する法律」を中心とした法体系の整備が行われている。
