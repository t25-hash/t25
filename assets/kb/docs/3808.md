# 5・1・2 放射線と人体への影響

> **章**: 第5章 放射性物質,有害物質に関する法制度
> **節**: 5・1 放射性物質に関わる特別の法制度

a.放射線の基礎i.放射線と放射能 「放射線」と「放射能」という一般には区別の付きにくい言葉があるが,この二つの言葉は別々の意味を持っている。これをひかり輝く白熱電球に例えてみると,電球から出される光が「放射線」,電球の持つ光を出す能力あるいは性質が「放射能」に相当する.また,放射能を持った核種を「放射性核種」といい「放射性同位体」,「放射性同位元素」,「ラジオアイソトープ(RadioIsotope)」,「RI」などとも呼ばれる。放射性核種はエネルギーを放出して別の種類の核種に変わり,その際に放射線を放出するが,その原子数が最初の数の半分になるまでの時間を「半減期」と呼んでいる.半減期の長さは,放射性核種の種類によって異なる.ii.放射線の種類 放射線は原子から放出されるエネルギーで,何がそのエネルギーの本質か,によって放射線の種類を分けている.広い意味では電波や紫外線なども含むが,通常は物質と反応して「電離を起こす」ものを放射線と呼ぶ.この電離を起こす放射線には,アルファ線ベータ線ガンマ線などがある.アルファ線(α線)は陽子2個,中性子2個からなる粒子で,ヘリウムの原子核と同じである.α線はβ線等の他の放射線と比べると物質を透過する能力は弱く,薄い紙1枚で止めることができる.一方,この短い距離の間にすべてのエネルギーを物質中に与えることになるので,放射線としての影響は大きくなる.ベータ線(β線)は電子の流れである.物質を透過する能力はαβ9-126線よりも大きく,空気中では数十cmから数mの距離を飛ぶが,薄い金属板1枚で止めることができる.また,α線と比べて物質に及ぼす影響は小さくなる.ガンマ線(γ線),X線は電磁波であり,α線やβ線のような粒子ではなく,光や通信で使用されている電波なども同じ電磁波である.γ線やX線は電波と同じように空間を伝わっていくとともに,物質中も比較的よく透過する.電磁波は波長の長さ(エネルギー)で分類し,波長の短いもの(エネルギーの大きいもの)がγ線やX線である.γ線とX線はその発生機構が異なるだけで,放出された電磁波としての性質は一緒である.γ線は原子核内のエネルギー変化により放出される電磁波で,X線は原子核の外での電場や軌道電子が関与した結果生じるものである。中性子線は原子核を構成している粒子のひとつである中性子が原子核から放出されるもので,核分裂が起こる際に中性子線が放出される.放射線にはこのほかに陽子線,重粒子線などがある.iii。単位放射線,放射能の量を表現するために「ベクレル」,「シーベルト」という特別な単位が使用されている.放射能を表す単位がベクレルで,記号は「Bq」が用いられている.ベクレルというのは放射能を発見したフランスの物理学者であるベクレル博士の名にちなんで付けられた単位で,物質が放射線を出す能力を1秒間に壊変する原子核の数で表した値である.また,放射線の人体への影響の程度を表す単位がシーベルトで,記号は「Sv」が用いられている.シーベルトというのはスウェーデンの放射線物理学者であるシーベルト教授の名にちなんで付けられた単位で,単位重量あたりの吸収エネルギー量に放射線の種類による生物への影響の違いを考慮した係数を乗じた値である.このシーベルトという単位はこの放射線の種類により異なる影響を比較することができるように作られた単位であるので,シーベルトで表した数字が同じならばその影響も同じといえる.iv.自然に存在する放射線 一般的に放射線や放射能という言葉は,原爆や原子力関連施設の事故など異常な事態に発生するというイメージで受け取られているが,実はわれわれの身の回りにも日常的に放射線がたくさんある.我々の住む宇宙は約150億年前にピックバン(大爆発)により生まれたとされているが,星ができる過程などで起きる核融合反応や星が一生を終える際の超新星爆発などにより,宇宙空間には軽い元素から重い元素までいろいろな元素がガスやチリの形で漂い,そのなかには放射性元素も多く含まれる.したがって46億年前に地球が誕生したときから,地球には放射性元素と放射線が存在し,その中で生物は放射線を受けながら生きてきた.そして現在でも地球上の生物はすべて自然界からの放射線を受けながら生活している.自然放射線はその起源により大きく四つに分類できる.ひとつは「大地からの放射線」である.これは地面の中に地球創生以来の放射性物質が含まれていて,その放射線を人間は受けている.この放射線の量の平均は1年間に約0.5mSvだが,その土地の岩盤を:構成する岩石の種類や地形により変わる.このため地域にタる差があり,たとえば関東より関西の方が多くの放射線を受けている.ノまた,外国にはこの10倍程度の放射線を受けている地域もある.2番目に「宇宙線」がある.これは宇宙の彼方で起こった超新星爆発等により放出された放射線が地球に到達するものである.この放射線の量の平均は1年間に約0.4mSvだが,高度の影響を受け,たとえば富士山頂では平地の約5倍にもなる.3番目に,人間が食べているいろいろな「食べ物」にも放射性物質(カリウム40等)が含まれている.それを食べることにより,体の中から放射線を受け,この放射線の量は平均で1年間に約0.3mSvである.4番目は,人間が吸っている空気,その空気の塵にも放射性物質が含まれている.通常の呼吸にともない,その放射性物質を吸い込むことにより体の中から放射線を受けており,この放射線の量は平均で1年間に約1.3mSvである.この空気中に存在する放射線を出す主な物質はラドンで,ラドン温泉として知られているものと同じである.これら大地,宇宙,食物,ラドン等の四つを合わせると,人間は平均して1年間に2.4mSvの放射線を受けていることになる.b.放射線の人体影響 放射線の人体影響に関しては他の有害物質と比べてその科学的な情報は豊富にあると言える.放射線の発見から100年以上が経過したが,人類はこの間に多くの経験を積んできた。初期の頃から放射線の利用に医師が参加していたこともあり,放射線の影響を医学的・生物学的な知見として集約することができた.放射線を計測する分野の技術的な進展もあり,受けた線量とその影響を比較することも可能だった.また,国際的研究活動が比較的早期に発足し,これらの情報交換がおこなわれて,国際的に知識を集約することができた.i.放射線障害の歴史 x線が発見された頃は大量の放射線が人体に障害を及ぼすということがわからなかったので,研究者や作業員は放射線に対して特に防護や制限をすることもなく放射線を利’回していた.このため,X線管球を製作している作業者の手に皮膚炎が発生した例や,アメリカの夜光塗料工場においてダイヤルペインターが大量のラジウムを体内に取り込む例など,いろいろな障害が発生した.これらの事件を契機として放射線の生物への影響の研究が開始された.放射線による人体影響を評価する際の基礎となっている疫学的な調査研究には,1脊椎疾患のラジウム治療,胸腺肥大のX線治療等の放射線治療,2血管造影(トロトラスト),妊婦のX線診断等の放射線診断,3ウラン鉱山,文字盤工,初期の放射線技師らの職業被ばく,4広島・長崎,マーシャル群島の原爆被爆者,等があり,また多くの動物実験や試験管レベルの細胞実験もある.ii.放射線障害の特徴(メカニズム)放射線を受けた場合に発生する障害は他の有害物質の場合と比較していくつかの特徴がある。1放射線を受けた時点ではそのことを自覚することがなく,痛くもかゆくもない.2症状は決して放射線障害に特有なものではなく,他の原因によっても現れる.3放射線を受けてから数十年の潜伏期を経て症状が現れる障害もある.4障害は臨床的に複雑な経過をたどり,治りにくい.人体が放射線を受けた場合の影響について,以下にそのメカニズムを説明する.放射線には物質を電離する能力があるが,放射線が細胞にあたると細胞中のタンパク質や核酸などに電離作用が働く.細胞の働きを助けている酵素が電離作用により化学変化を起こして機能が低下すると,細胞の機能も低下する.大量の放射線を受けることにより細胞の機能低下が広範囲におよんだ場合には,組織,臓器として機能しなくなり,最悪の場合は個体の死に至る.iii.放射線障害の分類 放射線を受けた場合の人体への影響は,放射線を受けた本人の身体にでる「身体的影響」と放射線を受けた本人の子孫に現れる「遺伝的影響」の二つに分けることができる.さらに,・身体的影響は放射線を受けてから数週間以内に症状が現れる「急性障害」と数ヶ月から数年後になって症状が現れる「晩発障害」に分けられる.これらの影響は受けた放射線の種類や量,放射線を受けた部位により現れ方が異なる.急性障害では,全身に一度に1000mSv以上の放射線を受けるとおう吐や全身の倦怠感が現れる.一度に4000mSv以上の放射線を全身に受けると,何の治療もおこなわれない場合には受けた人の半数が死亡する.晩発障害では広島・長崎の原爆被爆者の場合,眼に5000mSv以上の放射線を受けた人では数ヶ月から数年後に白内障になる人が現れた.放射線の人体への影響については,「確定的影響」と「確率的影響」という分け方もある.確定的影響というのは,低い線量では影響が現れないが,ある程度以上の線量の放射線を受けると影響が現れる影響である.多くの放射線を受けることによりたくさんの細胞がダメージを受けた場合に組織器官として必要な機能が果たせなくなり,毛が抜ける,皮膚が紅くはれる,白内障になる等のといった症状が現れる.確定的影響では受けた線量が多くなるにしたがって症状が重くなる.確率的影響というのは,放射線を受けたからといって必ず影響が現れるというのではなく,受けた線量が多くなるほど症状のでる確率が高くなるような影響で,発がんと遺伝的な影響がこれにあたるとされている。放射線によるがんの発生は確率的影響のひとつであり,その発生には潜伏期が存在し,放射線を受けてから数年から数十年を経過して発病する.広島,長崎の原爆被爆者の疫学調査によれば,100mSv以上の放射線を受けた場合は,線量が増えるにしたがって,自然に発生するがんの発生率を上回ってがんになる人が増えることがわかっている.またがんが放射線を受けた本人に現れる影響であるのに対して,遺伝的影響は本人ではなくその子孫に現れる症状で,ショウジョウバエやマウスを用いた実験では親が受けた放射線量が大きくなると,それに比例して遺伝的障害を持つ子供が生まれる確率が増加することが知られている.しかレ人間については広島,長崎の原爆で大量の放射線を受けた場合でも,遺伝的な影響は認められていない.以上のように,大量の放射線は人体に悪影響をもたらすことは明らかであるが,1980年代以後,少量の放射線は人体にとってよい影響をもたらすかもしれない(放射線ホルミシス)とル、うことが注目されている.これは放射線に対して細胞がいわゆる「適応応答」し,がん抑制遺伝子を活性化したり,酸化抑制酵素などさまざまなストレス応答蛋白を生成して,放射線の刺激から人体を保護しようとする現象をいう.原子力,放射線利用に従事する人の受ける放射線量は,少量の放射線であり,この領域における人体影響を明らかにするため,この分野の研究が生命科学の進展と合わせて大いに発展することが期待される。
