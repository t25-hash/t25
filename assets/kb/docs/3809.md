# 5・1・3 放射線防護の基本的考え方

> **章**: 第5章 放射性物質、有害物質に関する法制度
> **節**: 5・1 放射性物質に関わる特別の法制度

a.ICRPとは 国際放射線防護委員会(lnternational Com・mission on Radiological Protection:通称ICRP)は、1928年に医療従事者を中心とした放射線取扱者の放射線安全に関する国際的な基準および手法を確立するために国際医学放射線会議が設けた委員会で、当初は「国際X線・ラジウム防護委員会」として発足した。その後、放射線利用が多くの分野に拡大されるのに伴い、1950年に大幅な組織替えを行い名称も「国際放射線防護委員会」と変更され現在に至っている。ICRPは主委員会と4つの専門委員会(1放射線効果、2放射線被ばくの線量、3医学における防護、4委員会勧告の適用)からなる国際的な委員会で、それぞれの委員会は国の代表としてではなく専門分野(たとえば、放射線計測学・物理学・医学・遺伝子学・放射線防護学等)を考慮して選ばれた委員によって構成されている。委員会は政治・行政から独立した団体で、委員もそれぞれ個人の資格で参加している。ICRPは、放射線影響・放射線防護に関する最新の知識・技術に基づいて放射線防護の基本的な考え方・基準などについて検討し、その結果を勧告あるいは報告として公表している。ICRPは、 ICRU(国際放射線単位・測定委員会)、WHO(国際保健機関)、IAEA(国際原子力機関)、UNSCEAR(国連原子放射線の影響に関する科学委員会)、ILO(国際労働機関)、 OECD・NEA(経済協力開発機構原子力機関)およびIRPA(国際放射線防護学会)等と密接な協力関係を保ち活動している。各国の放射線防護に関する法律・行政もICRPの勧告を尊重しており、我が国の放射線防護に関する法令も、これらの勧告に沿って定められている。ICRPの最新の勧告は1990年のもので、現在の我が国の法令もこの勧告に概ねしたがっている。b.ICRPの放射線防護体系 ICRPは「線量を有害な確定的影響のそれぞれに対するしきい値より低く保つことによりその発生を防止し、また確率的影響の誘発を容認できるレベルにまで制限すること」を放射線防護の目的としている。そのために次の3項目からβ9-127なる放射線防護体系を設けて、被ばく線量を制限するように勧告している。i.行為の正当化(ju8t:血cation)放射線被ばくを伴うどんな行為も、その行為によって、被ばくする個人または社会に対して、それが引き起こす放射線障害を相殺するのに十分な便益を生むのでなければ、採用すべきでないこと。ii。防護の最適化(opt陸墨ation)ある行為内のどんな特定の線源に関しても、個人の線量、被ばくする人の数を経済的および社会的要因を考慮したうえ、合理的に達成できるかぎり低く保つべきであること。これをA:LARA(As Lρw As Reasonably Achievable)の精神という。iii.線量の制限(do8e 1血itation)関連する行為すべての複合の結果生ずる個人の被ばくは線量限度に従うべきであること。iとiiの条件を満たしていても、線量は限度を超えるべきではない。確定的影響に関連するそれぞれの線量限度は影響のしきい線量以下に制限することにより確率的影響の発生を防止できるように決定されている。また、確率的影響に関連した線量限度は「低線量の影響は不明な点があるので、どんなに微量でもその線量に比例した影響がある」という保守的な仮定に基づき、その影響の発生を容認できるレベル以下に制限するよう決定されている。現在、ICRPが勧告している線量限度は以下の通りである。◎職業被ばくに対する線量限度実効線量限度’5年間につき 100mSvかつ1年間につき 50mSv等価線量限度150mSv/年500mSv/年、500mSv/年2mSv/妊娠中眼の水晶体’皮膚手・足妊娠中の女子の腹部◎公衆被ばくに対する線量限度1mSv/年実効線量限度等価線量限度15mSv/年50mSv/年眼の水晶体皮膚
