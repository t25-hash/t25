# 5・1・4 法体系

> **章**: 第5章 放射性物質、有害物質に関する法制度
> **節**: 5・1 放射性物質に関わる特別の法制度

a．ICRP勧告と日本における法体系
日本においては，放射線
障害の防止に関わる基本的な法律として，「放射線障害防止の技術的
基準に関する法律」がある．この法律には放射線障害の防止に関す
る技術的基準を策定するにあたっては，放射線を発生するものを取
り扱う従事者及び一般国民の受ける放射線の線量をこれらの者に障
害を及ぼすおそれのない線量以下にすることを持ってその基本方針
とすることと，放射線関係の学識経験者からなる放射線審議会を設
置し，関係行政機関の長が法令などにより放射線障害の防止に関す
る技術的基準を定めようとする時には放射線審議会に諮問しなけれ
ばならないことが定められている．
通常，ICRPが新しい勧告を出せば，放射線審議会が日本の実態
を考慮した上で，これを法令に取り込むかどうかを検討して意見具
申をおこない，これを受けて関係行政機関の長が法令改正要綱を作
成し放射線審議会に諮問し，放射線審議会の答申後，法令化される
という手順となる．
b。概要 原子力，放射線利用に関わる規制に関する国内法令
には，①原子力基本法，②核原料物質，核燃料物質及び原子炉の
規制に関する法律（以下，「原子炉等規制法」という），③放射性同
位元素等による放射線障害の防止に関する法律（以下，「放射線障害
防止法」という），④労働安全衛生法とその下部規則である電離放
射線障害防止規則（以下，「電離則」という），⑤原子力損害の賠償
に関する法律，⑥原子力災害対策特別措置法，等があり，放射線障
害の防止に関する措置が各法令の中で具体的に規定される仕組みと
なっている．
(C)日本機械学会
β9－128 β9編 法 工 学
原子力基本法は我が国の原子力開発利用に関する政策の指針とな
る法律で，原子力の研究・開発および利用についての基本的な方針
を明示している．法的規制を包括的に示し，その規制の骨子を述べ
ているが，具体的な規制は個々の法令にゆだねている．原子炉等規
制法は精錬，加工及び再処理の事業並びに原子炉の設置及び運転に
関して必要な規制をおこなうほか，国際規制物質の使用に関する必
要な規制をおこなうことを定めている．放射線障害防止法は放射性
同位元素の使用・販売・廃棄その他の取扱い，放射線発生装置の使
用について定めている．労働安全衛生法では放射線下で作業をおこ
なう労働者を保護するために事業者が守るべきことを定めている．
原子力損害の賠償に関する法律は原子炉の運転等により原子力損害
が生じた場合の賠償に関する基本的な制度を定めている．原子力災
害対策特別措置法は原子力災害の特殊性に鑑みて，関係機関の連携
強化，緊急時対応体制の強化，防災対策上の責務の明確化などを定
めている．
