# 5・1・4 法体系

> **章**: 第5章 放射性物質、有害物質に関する法制度
> **節**: 5・1 放射性物質に関わる特別の法制度

日本においては、放射線a.ICRP勧告と日本における法体系障害の防止に関わる基本的な法律として、「放射線障害防止の技術的基準に関する法律」がある。この法律には放射線障害の防止に関する技術的基準を策定するにあたっては、放射線を発生するものを取り扱う従事者及び一般国民の受ける放射線の線量をこれらの者に障害を及ぼすおそれのない線量以下にすることを持ってその基本方針とすることと、放射線関係の学識経験者からなる放射線審議会を設置し、関係行政機関の長が法令などにより放射線障害の防止に関する技術的基準を定めようとする時には放射線審議会に諮問しなければならないことが定められている。通常、ICRPが新しい勧告を出せば、放射線審議会が日本の実態を考慮した上で、これを法令に取り込むかどうかを検討して意見具申をおこない、これを受けて関係行政機関の長が法令改正要綱を作成し放射線審議会に諮問し、放射線審議会の答申後、法令化されるという手順となる。b。概要 原子力、放射線利用に関わる規制に関する国内法令には、1原子力基本法、2核原料物質、核燃料物質及び原子炉の規制に関する法律(以下、「原子炉等規制法」という)、3放射性同位元素等による放射線障害の防止に関する法律(以下、「放射線障害防止法」という)、4労働安全衛生法とその下部規則である電離放射線障害防止規則(以下、「電離則」という)、5原子力損害の賠償に関する法律、6原子力災害対策特別措置法、等があり、放射線障害の防止に関する措置が各法令の中で具体的に規定される仕組みとなっている。β9-128原子力基本法は我が国の原子力開発利用に関する政策の指針となる法律で、原子力の研究・開発および利用についての基本的な方針を明示している。法的規制を包括的に示し、その規制の骨子を述べているが、具体的な規制は個々の法令にゆだねている。原子炉等規制法は精錬、加工及び再処理の事業並びに原子炉の設置及び運転に関して必要な規制をおこなうほか、国際規制物質の使用に関する必要な規制をおこなうことを定めている。放射線障害防止法は放射性同位元素の使用・販売・廃棄その他の取扱い、放射線発生装置の使用について定めている。労働安全衛生法では放射線下で作業をおこなう労働者を保護するために事業者が守るべきことを定めている。原子力損害の賠償に関する法律は原子炉の運転等により原子力損害が生じた場合の賠償に関する基本的な制度を定めている。原子力災害対策特別措置法は原子力災害の特殊性に鑑みて、関係機関の連携強化、緊急時対応体制の強化、防災対策上の責務の明確化などを定めている。表5・1 日本における放射線防護関係の法体系原子力基本法(昭和30年法律第186号)原子力委員会及び原子力安全委員会設置法(昭和30年法律第188号)・原子力委員会および原子力安全委員会の組織および権限を定めるもの放射線障害防止の技術的基準に関する法律(昭和33年法律第162号)・放射線審議会の組織及び権限を定めるもの核原料物質、核燃料物質及び原子炉の規制に関する法律(昭和30年法律第166号)・核原料物質、核燃料物質及び原子炉の利用について、平和利用三原則に基づき精錬、加工、再処理、使用及び原子炉の設置、運輯について必要な規制をおこなうほか、条約等の国際的な約束を実施するため国際規制物質の使用に関しての規制を定めるもの放射性同位元素等による放射線障害の防止に関する法律(昭和32年法律第167号)・放射性同位元素(放射線発生装置)の使用、販売、廃棄等の規制を定めるもの原子力損害の賠償に関する法律(昭和36年法律第147号)・原子力損害が生じた場合の賠償の制度を定めるもの労働安全衛生法(昭和47年法律第57号)L電離放射線障害防止規則(放射線業務従事者の放射線被ばくの防止に関する部分)医療法(診療X線、放射線医薬品の管理に関する部分)原子力災害対策特別措置法(平成11年法律第156号)・原子力災害時の関係機関の連携、緊急時対応体制、防災対策上の責務を定めるもの
