# 5・1・5 各法令の概要

> **章**: 第5章 放射性物質,有害物質に関する法制度
> **節**: 5・1 放射性物質に関わる特別の法制度

a.原子炉等規制法(昭和32年法律第166号) こめ法律は原子力基本法の中の核原料物質,核燃料物質および原子炉の管理の規定を受けて定められており,次の二つのことを目的としている.1核原料物質,核燃料物質および原子炉の利用が平和の目的に限り,その利用が計画的に行われることを確保し,その利用による災害を防止して公共の安全を図るために必要な規制をおこなう.2原子力の研究,開発及び利用に関する条約そのほかの国際的な約束を実施するために,国際規制物質の使用に関して必要な規制をおこなう.法律は,精錬の事業を行う者,加工の事業を行う者,原子炉を設置し運転する者,再処理の事業を行う者,廃棄の事業を行う者,その他の形態で使用する者,国際規制物質を使用する者ごとに規制する形を取っている。以下に原子炉等規制法について,原子炉の設置,運転等に関する規則をもとに概要を述べる.i.原子炉の設置,運転等に関する規制 原子炉を設置しようとする者は原子炉の区分ごとに経済産業大臣,国土交通大臣,文部科学大臣(以下,「主務大臣」という)の許可を受けなければならないことが定められている.1実用発電用原子炉(以下の2~4を除く)は経済産業大臣,2船舶に設置する原子炉(以下の45を除く)は国土交通大臣,3試験研究炉(245を除く)は文部科学大臣,4研究開発段階にある発電用原子炉は経済産業大臣,5研究開発段階にある発電用原子炉以外の原子炉1ま文部科学大臣.この申請を許可する基準は,1原子炉が平和の目的以外に利用されるおそれがないこと,2その許可をすることによって原子力の開発及び利用の計画的な遂行に支障を及ぼすおそれがないこと,3申請者に原子炉を設置するために必要な技術的能力及び経理的基礎があり,かつ,原子炉の運転を的確に遂行するに足りる技術的能力があること,4原子炉施設の位置,構造及び施設が核燃料物質,核燃料物質によって汚染された物又は原子炉による災害の防止上支障がないものであること,の四つである。主務大臣は許可をする基準の適用について,あらかじめ原子力委員会,原子力安全委員会の意見を聞き,これを尊重することとしている.原子炉施設を設置しようとする者はその工事に着手する前に原子炉施設に関する設計及び工事の方法について主務大臣の認可を受けなければならない.原子炉施設の建設段階ごとに原子炉施設の工事及び性能について主務大臣の検査を受けなければならない.この検査に合格した後でなければ,原子炉施設を使用してはならない.原子炉設置者は原子炉施設の性能が主務省令で定める技術士の基準に適合しているかどうかについて,主務大臣が毎年1回定期に行う検査を受けなければならない.原子炉施設の性能が技術上の基準に適合していない場合は,主務大臣は原子炉設置者に対して原子炉施設の使用の停止,改造,修理又は移転,原子炉の運転方法の指定,その他保安のために必要な措置を命ずることができるとしている.原子炉設置者は主務省令で定めるところにより,1原子炉施設の保全,2原子炉の運転,3核燃料物質又は核燃料物質によって汚染された物の運搬,貯蔵又は廃棄,の三つについて保安のために必要な措置を講じなければならない.原子炉設置者は主務省令で定めるところにより,保安規定を定め,原子炉の運転開虚心に主務大臣の認可を受けなければならない.原子炉設置者は原子炉を解体しようとするときは,主務省令で定めるところによりあらかじめ主務大臣に届けなければならない.主務大臣は届出を受けた場合には,原子炉設置者に対し1原子炉の餌体方法の指定,2核燃料物質による汚染の除去,3その他核燃料物質,核燃料物質によって汚染された玉造は原子炉による災害を防止するために必要な措置を命ずることができる。原子炉設置者は,原子炉の運転に関して保安の監督を行わせるため,原子炉主任技術者免状を有する者のうちから,原子炉主任技術者を選任する.原子炉の運転に従事する者は,原子炉主任技術者がその保安のためにする指示に従わなければならない.以上,原子炉の設置,運転等について計画,設計,施工,運転,解体の各段階における総括的な保安のために講ずべき措置について概要を述べた:このほか精錬,加工,再処理,廃棄,使用等それぞれの事業についてもほぼ同様のことが規定されている。以下に放射線業務従事者及び公衆の放射線障害防止に関する措置に焦点をあわせて本法令に規定されていることを解説する。ii。従事者に対する放射線防護の措置 放射線業務従事者の放射線障害を防止するための防護措置を規定.(1) 管理区域 放射性物質の拡大防止と不要に放射線を受けることがないように管理を行う区域を「管理区域」とする。法令では,外部放射線に係わる線量(実効線量が3ヶ月について1.3mSv),空気中の放射性物質濃度(3ヶ月についての平均濃度が1.3mSvに相当する濃度),物の表面の放射性物質の密度(α線を放出しない放射性物質について4Bq/cm2,α線を放出する放射性物質では0.4Bq/cm2)について具体的な値が示されており,これらの基準を超える区域は管理区域とする必要がある.原子力施設では建屋・機器の配置,入退二等の管理上の利便を考慮して,法令の定める基準未満の範囲も含めて区域の設定を行うのが実際的である.管理区域への出入口は原則1ヶ所とし,そこにチェックポイントを設けて,あらかじめ入域を許可された者(放射線業務従事者)以外の立ち入りを禁止し,立ち入る者については放射線測定器の着用.を確認する.また,汚染管理区域から退出する際は身体,物品の汚染を検査し,汚染がある場合には洗身,手洗い等の除染を行う.(2) 原子力施設で発生する放射性廃棄物には,気体状,液体状,固体状の3種類の形態がある.気体状の廃棄物はろ過,減β9-129衰ののち排気口を通じて大気中へ希釈放出する.液体状の廃棄物はろ過,脱塩,濃縮等の処理の後,施設用水として再利用もしくは排水口を通じて希釈・放出している.固体状の廃棄物は減容を目的として焼却,圧縮,固形化等の処理ののちドラム缶に封入し,廃棄物貯蔵庫へ保管廃棄している.排気,排水中の放射性物質濃度は,監視設備により監視するとともに,周辺監視区域外の濃度については一般公衆がそれを呼吸もしくは飲むことにより年間1mSvを超えないような濃度限度が定められている。(4) 線量目標値 法令ではないが,原子力安全委員会により「発電用軽水型原子炉施設周辺の線量目標値に関する指針」が定められている.ここでは原子炉施設から放出される放射性物質(放射性希ガス,放射性ヨウ素など)により公衆が受ける線量の目標値を年間0.05mSvとしている.これは「線量を容易に達成できる限り低く保つ」の考え方に基づきICRP勧告やこれにもとつく法令の年限度1mSvを,更に1/20のレベルに設定された.この線量目標値を評価に使用するモデルとパラメータ等が指針として示されている.この指針の中で,気体廃棄物は排気筒の高さ,気象要素(風向,風速,大気安定度)から大気中の拡散を考え,施設境界外で最大線量を与える地点における線量を評価し,線量目標値を守れる値として放出基準値を定め,これを守ることしている.また,液体廃棄物は放射性廃液の放出量,希釈水量だけでなく,海洋生物による濃縮,人間の海産物摂取量を考慮して評価し,線量目標値を守れる値として放出基準値を定め,これを守ることしている.(5) 環境モニタリング 原子力施設から放出される排気,排水中の放射性物質を確認することにより一般公衆の安全は確認されているが,更にこれを補完する意味で敷地周辺の放射線及び放射性物質を直接測定し,一般公衆の安全を確認している.環境モニタリングは放射性物質の放出形態,環境条件などを考慮して一般公衆への線量当量の寄与が大きいと予想される経路に着目しておこなう.軽水炉においては,たとえば気体廃棄下中の放射性希ガスは直接線が主な経路となるので,空間γ線線量を測定している.また,放射性ヨウ素は葉菜への沈着や牛乳への移行が主な経路となるので,葉菜,牛乳中の放射性ヨウ素の測定を,液体廃棄物中のコバルト等は海産生物中に濃縮される性質があるので,海底土や海産食品中の放射性物質を測定している.原子力施設周辺では施設設置:者と国,地方自治体が「安全協定」にもとづき共同で環境モニタリングを計画,実施しており,放射性物質の環境における移行状況,長期的な蓄積状況を把握することにより,一般公衆の安全の確保を図っている.’b.放射線障害防止法(昭和32年法律第167号) この法律は放射性同位元素等の使用等を規制することにより,放射線障害を防止し公共の安全を確保することを目的としている.実際には放射性同位元素等の使用は中小の施設が多いので,使用者に対して適切な施設において,専門家の監督の下,適切な取扱・管理をおこなうことを義務付け,これを文部科学大臣が点検・監督するという規制を原則としている.このため法令には施設基準,使用者の義務,主任技術者の役割等がより具体的に記載されている.i.法令の構成 以下に放射線障害防止法の概略を述べる.D(1) 規制の対象 規制の対象となるのは,1放射性同位元素の使用,販売,廃棄,その他の扱い(たとえば保管,運搬,譲渡,所持等),2放射線発生装置の使用,3放射性同位元素により汚染された物の廃棄,である.放射性同位元素等の用語は学術的な概念とは同一ではない,法律上の定義である.たとえば,放射線を放出する同位元素及びその化合物,含有物で文部科学大臣が定める数量及び濃度を超えるものを法令上の「放射性同位元素」としている.(2)使用等の許可 この法律では放射性同位元素等の取扱いを一般的に禁止とし,所定の法律的な手続きにより事前に国の審査,点検を受け,放射線管理が適格に実施できると判断された者のみにβ9-130使用が許可される.放射性同位元素等を使用,販売等しようとする者は,事業所ごとにそれぞれ文部科学大臣に申請して許可を受けた後に,その業務を開始する.ただし,表示付放射性同位元素装備機器(たとえば,ニッケル63を装備したガスクロマトグラフ等)または1事業所当たり総量含.7GBq以下の密封された放射性同位元素を使用とする者は,事前に文部科学大臣に届出ることで使用できる.法律上,これらの許可を得た者を使用者,販売業者,廃棄業者(以下,「使用者等」という)と呼ぶ.使用等の許可基準は,使用施設,貯蔵施設等の必要な放射線施設が文部科学大臣の定めた技術的基準に適合する施設であり,放射性同位元素等の使用,詰替え,廃棄等をおこなうにあたって放射線障害のおそれがないことである.放射性同位元素装備機器のうち,機器自体によって放射線のしゃへい等が十分おこなわれ放射線障害を防止する機構になっているものについては,機器の設計承認及び機器の確認(機構確認がされた機器を「表示付放射性同位元素装備機器」という)することにより,届出により使用できる.(3)使用者等の義務 使用者等(その貯蔵能力が密封された放射性同位元素では37TBq,密封されていない放射性同位元素では740MBqを超える使用者等に限る)は使用施設,貯蔵施設,廃棄施設について‡部科学大臣の検査を受け,この検査に合格した後でなければ当該使用施設等を使用してはならない.また,使用者等(その貯蔵能力が密封された放射性同位元素では111TBq,密封されていない放射性同位元素では740MBqを超える使用者等に限る)は使用施設等が技術上の基準に適合していることについて文部科学大臣の検査を定期的に受ける.使用者等が実際に放射性同位元素等を取扱う場合には,各施設の位置,構造,設備を施設基準に適合するように維持しなければならない.適切な取扱,管理のために具体的な取扱基準が定められている.この取扱基準には使用,運搬及び廃棄等の技術上の基準と測定,健康診断,教育訓練,記録の基準が含まれている.そのために使用者等は当該事業所の放射線の取扱に関する責任管理体制を確立し,放射線障害予防規定を作成し,これを従事者に周知することにより,法令に定められた使用者等の義務を守ることとしている.事業所外において一定以上の放射性同位元素等を運搬する場合は,文部科学大臣または国土交通大臣の承認を得なければならない.(4) 放射線取扱主任者 使用者等がその義務を遵守するためには放射線管理についての専門的な知識のある補佐が必要である.この補佐役は放射線防護について使用者に適切な助言ができなければならない.法令では放射線障害の発生の防止についての監督を行わせるため,放射線取扱主任者免状を有する者のうちから,事業所ごとに放射線取扱主任者を選任することを使用者等に義務付けている.ii.従事者に対する放射線防護の措置 原子力防災対策については災害対策基本法の枠組みの中で対策を図ることとしていたが,1999年9月に発生した燃料加工工場における臨界事故の経験から,次のような問題点が明らかになった.1迅速な初期動作と国,地方自治体の連携強化.2原子力災害の特殊性に応じた国の緊急時対応体制の強化.3原子力防災における事業者の役割の明確化.このような点を踏まえて,従来の災害対策基本法と相きって機能する「原子力災害対策特別措置法」が成立した.i.防災対策の概要 原子力事業者に対して異常事態の通報を義務付け,通報を受けた主務大臣が緊急事態が発生したと認められる場合は,直ちに内閣総理大臣に報告する.内閣総理大臣は「緊急事態宣言」をおこない,「原子力災害対策本部」を設置し,自ら本部長どして迅速な初期動作をおこなう.国は現地にも「原子力災害現地対策本部」を設置し,あらかじめ指定された「緊急事態応急対策拠点施設(オフサイトセンター)」に自治体の現地災害対策本部と共に原子力災害合同対策協議会を設置して,情報交換や対策の相互協力をおこなう.内閣総理大臣は原子力災害対策本部として,緊急時対応応急対策の実施に関し関係機関や原子力事業者に対して必要な指示をすることができる.原子力安全委員会は内閣総理大臣の求めに応じて技術的な事項について助言をおこなう.また,緊急事態の際に現地で情報収集等をおこなう「原子力防災専門官」を各原子力事業所の所在地域に配置した.これらにより国の緊急時対応体制を強化した.原子力事業者に対しては放射線測定設備等の必要な資機材の設置のほか,原子力防災に備えた防災業務計画の策定,原子力防災組織の設置及び原子力防災管理者の選任を義務付けて,原子力事業者の責任,役割を明確化した.ii.緊急事態への対応 原子力緊急事態の判断基準は,1原子β9-131力事業所の境界付近において500μSv/時の空間線量率が測定された場合,2通常の排気口からの放出により500μSv/時の被ばくが予想される場合,3原子炉冷却機能喪失,4炉心溶融,等である.放射性物質又は放射線の異常な放出が発生した場合,周辺住民等の被ばく線量をできるだけ低減するために必要な防護対策を講じる.防護対策は,何らかの対策を講じないときに個人が受けると予想される線量(予測線量)に基づき実施の有無を判断する.(1)
