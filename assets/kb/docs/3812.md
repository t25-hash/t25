ピン安定衛星はバイアスモーメンタム方式である．衛星全体をスピ
ンさせるので，軸対称の形状をしており，CS-2，CS-3のような円
筒形状の衛星が典型的な例である．通信衛星の場合はアンテナをつ
ねに地球の方向に向けることが必要であるため，アンテナ部分だ

って逆回転（de - spin）させる．一方，三軸安定では，衛星の姿勢
をジャイロ等の情報から把握し，ガスジェット，イオンエンジン等
のスラスタで制御するゼロモーメンタム方式と，リアクションホイ
ールの回転のみで姿勢を安定させるバイアスモーメンタム方式，そ
してリアクションホイール，ジャイロ，スラスタという両方式の機
器を備えるハイブリッド方式とがある．
d.構体および熱制御（spacecraft structures and thermal con-
trol ) 衛星構体の役割はバス機器，ミッション機器が打上げ環
境に耐え，軌道上環境において動作するよう支持固定することにあ
る．また搭載したアンテナ，センサ類を要求された位置精度で支持
する役割がある．打上げ環境に対しては荷重経路（load pass ）を
構体のどの部分に設けるかで，中央円筒方式，パネル方式の二つの
方式に分けることができる．スピン安定型の衛星では中央円筒方式
が採られ，三軸安定型の衛星では中央円筒方式あるいはパネル方式
が採られる．このほかに中央円筒方式ではバッテリやテレメトリコ
マンド機器，姿勢制御機器等を搭載するミッションパネルが設けら
れる．また推進薬のタンクやアポジキックモータのような重量物は
中央円筒内に収納する．パネル方式ではバス機器とミッション機器
とを別モジュールとし，組立の簡素化と衛星バスの標準化とを図
る．

.中継器（transponders ) 中継器は受信した電波を送信
用周波数に変換し，増幅する機能をもつ．N-STAR中継器の構成
を図1･37に示す ．N-STARは，四つの周波数（Ka帯，Ku
帯，C帯，S帯)，五つのミッション（Ka帯マルチビーム，Ka帯
成形ビーム，Ku帯成形ビーム，C帯成形ビーム，S帯マルチビー
ム）の通信機器を搭載している．Ka帯マルチビーム通信系では，
受信ビームと送信ビーム間をサテライトスイッチにより時分割接続
する．Ka帯，Ku帯，C帯の成形ビーム通信系では，それぞれ最
大2チャネルずつを異なる送受信周波数帯間で接続可能なクロスス
トラップ機能を有している．S帯マルチビーム通信系では，電力増
幅としてマルチポート増幅器を使用している．マルチポート増幅器
の構成要素である単位増幅器では，各ビームからの通信信号をすべ
て共通に増幅しており，これによりビーム間にトラヒック変動があ
ってもトータルの容量が下がることがなく，衛星電力を有効に利用
できる．
