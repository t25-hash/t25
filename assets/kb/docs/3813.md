# 5・2・2 危険性と有害性から見た法体系 化学物質の危険有害

> **章**: 第5章 放射性物質,有害物質に関する法制度
> **節**: 5・2 化学物質の総合管理

性は大きく次の二つに分けることができる.一つは,火災・爆発のβ9-132危険性(以下「危険性」という)であり,他は人の健康や環境に対する有害性(以下「有害性」という)である.危険性に対応する法には,高圧ガス保安法,消防法,労働安全衛生法,コンビナート等災害防止法のいわゆる「保安四聖」がある.有害性に対応する法には,労働安全衛生法,化学物質の審査及び製造等の規制に関する法律(以下「化審法」という),毒物及び劇物取締法(以下「旧劇法」という)などがある.危険性に対応する法と,有害性に対応する法には次のいくつかの違いがある.(1) 規制の方法 危険性に対応する法では,例えば,引火点が21°C以上70°C未満の化学物質は「消防法危険物第4類・第2石油類」となるように,化学物質の性質によって自動的に法規制を受けることになる.一方,有害性に対応する法では,例えば,毒劇法で「毒物」と指定されている物質だけが「毒物」であり,毒物に相当する強い毒性があっても毒物に指定されていなければ「毒物」には該当しない.すなわち,危険性は性質指定による規制,有害性は物質指定による規制である.(2) 取扱設備に関する規制 危険性に対応する法には,取扱設備に関して詳細な規定がある.一方,有害性に対応する法では,取扱方法について詳細な規定のない場合が多い.その他の危険性と有害性の違いを参考として以下に示す.1 影響の現れる時期 危険性の影響(火災・事故)は,取り扱いを間違うと直ちに起こるが,有害性の影響は直ちに表れる場合(急性毒性)もあるが,慢性毒性や発がん性のように何年も後に現れる場合もある.2影響を及ぼす範囲 影響の範囲は,危険性では局地的だが,有害性は広い範囲にわたっていて,PCBのように地球規模で影響を及ぼす化学物質もある.3 危険有害性の試験 危険性を調べる試験に比較して,有り害性を調べる試験は難しく,費用がかかるので,すべての化学物質について十分に有害性を把握することは難しい.上記に述べたように,人の健康や環境への有害性への対応に関しては,その有害性を十分把握できているとは限らないので,法規制を遵守するだけでは不十分なことが多い.このため,化学物質の総合管理という場合は,有害性への対応に重点をおいている場合が多い.
