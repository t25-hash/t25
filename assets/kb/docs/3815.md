(３)地上ディジタル放送
地上ディジタル放送は2003年に関東，近畿，中京の3広域圏で
放送が始まり，2006年には全国に展開される．映像フォーマット
はCSディジタル放送やBSディジタル放送と共通化が図られてい
る．BSディジタル放送と同様に高品質放送とデータ放送を行うほ
か，自動車等の移動体や携帯端末でも安定した受信が可能である．
地上ディジタル放送は現行アナログ放送が使っているのと同じ
UHFを使って放送される．チャネルの確保が難しいことから，ア
ナログ放送を行っている現在の中継放送局のチャネルを一部変更し
ながらディジタルチャネルに移行する．
(４)ディジタルCATV
CATVは多チャネル化，大容量化，双方向伝送が可能なシステ
ムへ発展してきた．1993年にそれまでの地元事業要件が廃止され，
MSO（multiple system operator ）による広域的な事業展開が進ん
でいる．ディジタルCATVは1996年のCSディジタル放送開始と
ともに，CATVに再送信する方式が規格化された．この方式は一
つの搬送波で一つのTSを伝送するものであった．しかし，CSデ
ィジタル放送のサービスはSDTVのみであったため，アナログに
変換して再送信する方法がおもに用いられた．BSディジタル放送
開始とともに，一つの搬送波で複数のTSを伝送できる新たな伝送
方式が開発・標準化され，ディジタル放送のサービスをすべて提供
できるようになった．
(５) ディジタル音声放送
放送電波によるディジタル音声放送は，CS，BSディジタル放送
で行うもの，2.6GHz帯衛星ディジタル音声放送，地上ディジタ
ル音声放送がある．
2.6GHz帯衛星ディジタル音声放送はAM放送程度からCD並
みのステレオ音声放送サービスのほか，簡易画像，字幕，スーパ
ー，図形表示も可能である．2004年10月にモバイル放送(株)が放
送を開始した．
地上ディジタル音声放送は高品質な音声サービスに加えて多彩な
データサービスを行うもので，2003年に東京，大阪で試験放送が
始まった．地上ディジタルテレビ放送と方式の共通化が図られ，テ
レビ番組の音声のみを受信することも可能である．
