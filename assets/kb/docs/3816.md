# 5・2・5 化学物質の輸送に関する法

> **章**: 第5章 放射性物質,有害物質に関する法制度
> **節**: 5・2 のようになる.現在の法体系では,法規制が最も充実している

a.船舶及び航空機による輸送 近年は貿易の自由化などにより,国境を越えて化学物質の輸送が行われることが多い.その場合,国によって法規制が異なっていては円滑な輸送が行えないので,早くから国連の主導の下に国際的な規制の整合が図られ,1956年に陸海空に共通する輸送要件の基礎となる「危険物輸送に関する勧告」が策定され,以後この国連勧告に基づく規制が行われている.国連勧告は危険物輸送の基本的要件を定めたものであり,実際の輸送にあたっては,それぞれの輸送手段に応じた輸送基準が策定されている.船舶で輸送する場合は船舶安全法,航空機で輸送する場合は航空法の規制を受けるが,これらの法は国連勧告に基づいて定められている.国連勧告はその後,各輸送基準にそのまま採り入れられるように改訂され,1997年発刊の第10改訂版からは「モデル規則」が示された.その中で,火災・爆発の危険性から人の健康や環境に対する有害性まで各種の危険有害性を有する物質に対して,輸送する物質の危険有害性によって,クラス1からクラス9まで分類され,それに属する物質の品名ごとの容器・包装,積載方法,隔離規程などの詳細規程が定められている.クラス1(爆発性物質),クラス2(ガス),クラス3(引火性液体),クラス4(可燃性物質類),クラス5(酸化性物質類),クラス6(毒物類),クラズ7(放射性物質),クラス8(腐食性物質),クラス9(有害性物質)b.日本国内での陸上輸送 危険性のある化学物質を道路を使って輸送する場合は,国連勧告に基づく規制が採用されておらず,日本独自の法律の規制を受ける.すなわち,火薬類取締法,高圧ガス保安法,消防法,毒劇法,道路交通法などによって危険性のある化学物質の積載方法,運搬方法,容器,運搬する車両の構造などが規制されている.火薬類取締法だけは,火薬類の定義や容器について国連勧告を取り入れている.c.イエローカード 化学物質の輸送過程における安全性確保と事故時の適切な対応を目的とした「イエローカード」の制度が定められている.「イエローカード」は,車両の乗務員がこれを携帯し,事故が発生した場合に,公設消防機関,警察官など関係者が到達するまでに乗務員が行う措置(応急措置,緊急通報,緊急連絡)及び乗務員,公設消防機関などが行う災害拡大防止措置を必要最小限に,かっ,簡潔に記載した黄色のカードである.
