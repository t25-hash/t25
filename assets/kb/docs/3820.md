# 6・1・1 現行の労働安全衛生管理

> **章**: 第6章 マネジメントと法制度
> **節**: 6・1 労働安全衛生マネジメント

a.労働安全衛生に係わる法令 現在我が国では、表6・1に示すように、実に多くの労働安全衛生に係わる法規類が制定されており、それによる規制が行われている。表6・1 労働安全衛生に係わる法規類の例・建築基準法・労働安全衛生法・建設業法・労働基準法・貨物自動車運送事業法・作業環境測定法・船舶安全法・じん肺法・港湾法・毒物及び劇物取締り法・消防法・食品衛生法・廃棄物の処理及び清掃に関する・電波法・道路交通法法律・化学物質の審査及び製造等の規制に関する法律・災害対策基本法・交通安全対策基本法・原子力基本法・放射性同位元素等による放射線障害防止に関する法律・核燃料物質及び原子炉の規制に関する法律・鉱山保安法・採石法・火薬類取締法・高圧ガス保安法・電気事業法・騒音規制法・電気工事士法・振動規制法・ガス事業法・悪臭防止法・工場立地法・石油コンビナート等災害防止法・液化石油ガスの保安の確保及び取引の適正化に関する法律さらに、これらの法規類に対しては、制定後も重大な労働災害の発生するたびに、法改正または詳細な規制基準などの追加・改正がなされてきている。本来、制定された法令は、自主的に遵守されることが法の理想とするところであるが、現実には遵守されない場合がある。そのときは、法的な制度として刑罰などの制裁が課されるか、行政機関によって営業停止命令、使用禁止命令、改善命令などの行政処分が発せられる。すなわち、労働安全衛生に係わる現在の法システムは、伝統的な命令・監督型の規制手法を中心としているといえる。b.労働安全衛生法に基づく労働安全衛生管理 我が国における現行の労働安全衛生管理においては、a項に記したように法システムを遵守することが基本となっている。ここでは、労働安全衛生法(昭和47年6月8日 法律第7号、以下「安衛法」という)に基づき、労働安全衛生管理がどのように行われているかを示す。i.安衛法の目的 安衛法の目的は、その第1条に次のように規定されている。(1)職場における労働者の安全と健康を確保すること(2) 快適な作業環境の形成を促進することそして、この目的を達成するために、国、労働災害防止団体および事業者等には、その責務として、労働災害防止に関する総合的かつ計画的な対策を推進することが求められている。具体的な実施事項は次の3点である。(i) 労働災害防止のための危害防止基準を確立すること伍)労働災害防止のための責任体制を明確化すること(皿) 労働災害防止のための自主的活動を促進する措置を講ずることこのうち(血)に記されているように、現在の安衛法では、命令・監督型の規制だけではなく、事業者自体のリソースを使いながら、自主的活動の促進も考慮した総合的かつ計画的な対策を推進することとしている。このため、例えば、日本独特の日常的な活動として、KY(危険予知)活動、ヒヤリ・ハット・気がかり提案活動、4S(整理・整頓・清潔・清掃)活動、始業前ミーティングなどが実施されてきている。しかし、基本的には安衛法は、命令・監督型の規制に重点を置いていることに変わりはない。ii.安衛法の内容 安衛法の主な内容について以下に記す(詳細は3・1節参照)。(1)事業者等および労働者の責務の明確化 安衛法第3条に、事業者(事業を行う者で労働者を使用する者)に対してのみならず、機械等の設計者・製造者または輸入者、原材料の製造者または輸入者、建設物の建設者または設計者、さらに建設工事の注文者等仕事を他人に請け負わせる者に対しても、労働災害防止の責務を広く規定している。また、労働者自身にも、必要な事項を守ることや事業者等が実施する労働災害防止措置に協力することが求められている。(2) 労働安全衛生管理体制の確立 一定規模以上の事業場には統括安全衛生管理者を、また、所定の要件を満たす事業場には安全管理者、衛生管理者、安全衛生推進者、産業医、作業主任者等を、それぞれ選任する必要がある。建築業や造船業のように請負契約に基づき複数の事業者が一つの場所で仕事を行う場合には、所定の条件に対応して、統括安全衛生責任者、元方安全衛生管理者、店社安全衛生管理者、安全衛生責任者等を選任しなければならない。さらに、事業場の規模と業種に応じて、安全委員会、衛生委員会、または安全衛生委員会を設け、所定の事項を調査・審議させ、事業者に対して意見を述べさせなければならない。そして、事業者は、上述した各選任者および労働災害防止のための業務に従事する者に対して、従事する業務に関する能力の向上を図るための教育や講習などを行うか、または受講機会を与えるよう努力する義務がある。(3) 労働者の危険または健康障害を防止する措置 事業者は、1機械、器具、その他の設備による危険、2爆発性、発火性、引火性の並等による危険、3電気、熱、その他のエネルギーによる危険、などを防止するために、必要な措置を講じなければならない。この規定の具体的な内容は、厚生労働省令である労働安全衛生規則(以下「安衛則」という)のなかに、安全基準や衛生基準として詳細に定められている。(4)機械等および有害物に関する規制 ボイラーやクレーン等の危険な作業を伴う機械室については、製造の許可、製造垂直の検査、性能検査、個別検定、型式検定、定期自主検査の実施などの規制が行われている。(5) 労働者の就業にあたっての措置 安全衛生教育として、雇用時の教育、作業内容変更時の教育、危険または有害物作業従事者の教育、職長等の教育が定められている。ボイラーやクレーン等の16業務は就業制限業務とされ、資格者が免許を携帯して作業しなければならない。また、中高年齢者等で配慮の必要な者は、心身の条件に応じて適正配置するよう規定されている。(6) 健康の保持増進のための措置 作業環境の測定と結果の評価、作業の管理、健康診断、病者の就業禁止、健康教育などの実施について規定されている。(7)快適な職場環境の形成のための措置 快適な作業環境の形成のみならず、より安全で、健康的で、疲労やストレスを感じることの少ない快適な職場環境の形成の促進について規定されている。(8) 機械等の設置届、報告・記録等の義務、書類保存の義務ほか省令で定められた建築物や機械等の設置、移転、変更等は、工事開始の30日前迄に計画を労働基準監督署に届け出る義務がある(一β9-136般に「第88条の届出」という)。さらに、事故や死傷病等の報告義務、また安全衛生委員会記録、定期検査結果、作業環境測定結果、健康診断結果、安全衛生教育記録等の書類の保存義務についても規定されている。c.労働災害と事業者責任 労働災害に関連して事業者には次のような責任が発生する。i.刑事責任 労働災害が発生すると、司法警察官としての労働基準監督官が場内に立ち入り、発生状況や災害原因調査をし、安衛法違反の有無について捜査をする。労働災害防止の責任は事業者にあるとの考えから、法人の代表である社長が責任の対象となるが、現実には安全衛生に対して権限と責任を与えられている工場長や管理・監督者なども違反行為者として追求される。いわゆる両罰規定が適用される。また、同時に警察機関による、業務上過失致死傷違反の有無についての捜査もなされる。安衛法違反については、災害が発生しなくても災害発生のおそれのある異常状態が放置されている場合には、事前送検という形で罰則が科せられることがある。ii.民事責任 安衛法違反の有無にかかわらず災害が発生した場合には、民法上次の三つの責任が問われることがある。(1) 民法第415条による債務不履行責任 使用者が労働契約を締結して雇った以上、労働災害にあわせない約束をしたことになる。それにもかかわらず労働災害にあわせたということは契約違反であり、通称「安全配慮義務違反」となる。(2) 民法第709条と715条による不法行為責任 危険予知や防護措置が可能であるにもかかわらず、防止措置をとらないで災害が発生した場合は「注意義務違反」となる。(3) 民法第717条による工作物i出丸責任 点検整備の未実施や不備、また安全装置の無効などの、物や機械等が本来備えているべき構造、性能、性質等に欠陥があったり、正常状態に維持されていなかったりしたことに対する責任である。これら三つの責任に対して、いずれも損害賠償の義務が生じる。もちろん、被災者に過失があれば損害賠償請求に対して過失相殺される。iii.行政責任 労働基準監督官の捜査により、労働安全衛生法令(以下「安衛法令」という)に基づく設備の使用停止命令や作業停止命令などの行政処分が行われたり、安衛法令違反の是正を勧告されることがある。これらの命令や勧告に故意に従わない場合、書類送検され刑事責任を問われることもある。官庁の許可事業における許可の取消し、営業停止、指名停止などの処分を受ける場合もある。iv.社会的責任 これは法的なものではないが、近隣住民へも影響を及ぼした場合や重大な災害を発生させた場合には、マスコミなどからの追求を受け、社会的合意に基づく責任をとる必要もある。
