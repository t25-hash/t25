# 1・2・1 軸系の危険速度

> **章**: 第1章 気体機械一般
> **節**: 1・2 ターボ圧縮機の軸系

回転軸で実際に現れる危険速度（critical speed）は，軸を軸受
点で単純支持として計算した数値から，軸受油膜および軸受支持部
の，弾性および減衰効果によって大幅に変わってくる．このような
軸系の応答計算は，不釣合い応答計算や複素固有値計算によって詳
細に求めることができる．軸受部のばね定数と危険速度の関係を示
す危険速度線図を作ると，回転軸の振動特性の検討や支持部の最適
ばね定数の選択，軸系の設計変更の検討などに便利である（図Ⅲ-
1･7)．
危険速度に対する裕度として，通常，一次危険速度より低い回転
速度で使用するリジッドロータ（rigid rotor ）の場合には最大連続
回転速度から20 %以上，危険速度に挟まれた中間で使用するフレ
キシブルロータ（flexible rotor ）の場合には最小連続回転速度か
ら15%以上，最大連続回転速度から20 %以上離れたところに危険
速度がくるように設計される（図Ⅲ-1･8)．もし，このような裕
度がとれない場合や危険速度が運転速度範囲に入る場合でも，振動
に対する感度が低ければ実用上支障がない．図Ⅲ-1･9に，判断基
準の一例を示す ．また軸系のねじり振動の計算が必要とされるこ
とが多い．
