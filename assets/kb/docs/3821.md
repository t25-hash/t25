# 6・1・2 自主対応型アプローチによる労働安全衛生管理

> **章**: 第6章 マネジメントと法制度
> **節**: 6・1 労働安全衛生マネジメント

a.法規準拠型アプローチの問題点 労働安全衛生管理とは,「事業の運営に伴う労働災害の根絶を目的として,経営者が行う合理的,組織的な一連の施策」をいう.これらの施策を具体化するにあたり,従来からの労働安全衛生管理は,我が国はもとより欧米においても,労働安全衛生に係わる法令を遵守するという,いわゆる法規準拠型のアプローチを中心に進められてきた.しかし,この方法には限界があり,問題点が指摘されている.我が国の安衛法が制定されたちょうど同じ1972年に英国において公表された「労働における安全と健康」,いわゆるローペンス報告のなかで,法規準拠型のアプローチの問題点として,次の4点があげられているω.(1)法律があまりにも多く詳細すぎるため,全体の理解が困難となり,現場において改善に取り組む上で逆効果である.また,法はすべてをカバーできず,本質的に不満足なものとなっている。(2) 限られた数の監督官で,すべての事業場に法規を徹底させることは物理的に不可能である.(3) 規制的要素が強すぎ,外部からの働きかけで改善されるものだという受動的な態度が職場に広がり,自主性,自発的努力が薄らぎ,現場に則した具体的な改善が進みにくい.(4) 行政管轄が細分化され過ぎ,多様な法規定に支配される事業場がある一一方,全く範囲外となる職場が存在する。また,既存の法規定と施行制度の多くのものが時代遅れになっている.また最近では,法規準拠型のアプローチは,「不確実性を伴う先端的な分野の規制においては,規制内容の不確実性等からの基準の欠陥と,規制基準を執行する段階での人的リソースの不足等からの執行の欠陥,という二大欠陥を伴いうる」ωとの指摘もなされている.b.自主対応型アプローチへの移行 ローペンス報告では,a項に記載したような問題点を解決するための施策を提言として取りまとめている.それは基本的に次の3点に要約される(3)。(1)細分化され二二している安全衛生行政を一元化し,統一的な執行体制を確立すること.(2)複雑化している安全衛生関連法規を体系化し,明確化すること.(3) 時代の進歩に柔軟に対応できるように,規制や監督中心から,自主基準を活用した自主対応型の安全衛生活動へ変換すること.イギリスではこの提言を全面的に取り入れ,真974年には自主的対応を重んじる法制度が成立している.最近の世界的な動向としても,企業の自己責任と労使の協力による自主対応努力およびそれによる現場改善が重視されてきている.そして行政側としては,従来は,個々の設備・作業に対する技術基準の追加設定などの,より細分化された法規制の整備を主として推進してきたが,最近では,職場自主改善システムづくりの支援や,そのために必要なサービスの推進を行うことに重点を移してきている.より技術的な基準は,対策指針集という形などで整理し情報提供され,企業における自主対応努力を支援するものと位置付けられている.すなわち,従来進めてきた法規準拠型のアプローチを転換し,今後は,企業の自己責任と労使の協力による自主対応型のアプローチで進めていく必要があるとの認識が世界的に広まってきている.表表6・2 労働安全衛生マネジメントシステム(OHSMS)は, b項で述べた自主対応型のアプローチによる労働安全衛生管理を進めるにあたって,その中心となるシステムとして作り出されたものである.ここでは,OHSMSに関する指針や規格化の動向について概説する。なお,以下に記載の・OHSAS18001のみはISO9001やISO 14001のような審査登録用の規格であるが,それ以外はこれに該当しない指針またはガイドラインとなっている.i.海外の動向1991年:イギリス安全衛生庁(HSE)が「成功する安全衛生管理の指針;HS(G)65」を公表.1996年5月:イギリス規格協会(BSI)が「労働安全衛生マネジメントシステムの指針;BS 8800」を公表.1997年1月:国際標準化機構(ISO)の技術管理評議会(TMB)は「OHSMSの国際規格化は時期尚早につき,これ以上作業は行わない」とした.1999年4,月:BSI他がOHSAS 18001(OHSAS;OccupationalHealth and Safbty Assessment Series)規格を発行.1999年12月:BSIが「OHSMS規格制定に向けての技術委員会(TC)の設置」をISOに提案。2000年6月:ISOのTMBは「OHSMS規格制定のTCは設置しない.国際労働機構(ILO)との共同開発もしない」ことを決定.2001年6月:ILOはOSHMS(Occupational Sa艶ty and HealthManagement System)のガイドラインを理事会で承認ii.我が国の動向 1999年4月に労働省(現厚生労働省)から「労働安全衛生マネジメントシステムに関する指針」(以下「厚生労働省OHSMS指針」という)が出された.この厚生労働省OHSMS指針と前後して,各種団体からOHSMS指針またはガイドラインが出されている.例えば,1996年には中央労働災害防止協会,1997年には自動車産業経営者連盟,1998年には日本化学工業協会と日本鉄鋼連盟,1999年には建築業労働災害防止協会,2000年には日本造船工業会などが相次いで出している.なお,日本では規格化の動きは特にみられない.d.OHSMSの法的な位置付け c項ではOHSMS規格や指針・ガイドラインをいくつか紹介したが,これらはすべてあくまでも自主的活動を促進するためのものであり,法的に規制や強制されるものではない.ただし,EUでは, OHSMSの中核となるリスクアセスメントについては,その実施を法的に義務付けている.すなわち,1989年に採択された自主対応型の安全衛生活動を目指した「労働安全衛生の改善を促進するための施策の導入に関する理事会指令89/391/EEC」,別名「枠組み指令」のなかで,「事業場においてはリスクアセスメントを行うとともに,そのリスクの低減策を講じること」を雇用者の一般的義務として規定している.加盟国は1996年末までにすべて法令を改正し,枠組み指令の内容を取り入れている.例えば,イギリスでは1992年に労働安全衛生管理規則のなかで,全雇用者にリスクアセスメントの実施を義務付け,従業員が5人以上の規模の雇用者はリスクアセスメントの重大な事項について記録をとらなければならないとしている.我が国では,厚生労働省OHSMS指針を公表することを前提に安衛則を改正し,その第2章第8節の次に「第8節の2 自主的活動の促進のための指針」を加え,第24条の2として「厚生労働大臣は,事業場における安全衛生水準の向上を図ることを目的として事業者が一連の過程を定めて行う自主的活動を促進するための指針を公表することができる」を規定した.この安衛則第24条の2を根拠として,厚生労働省OHSMS指針は告示として公表されたものである.リスクアセスメントについては,この厚生労働省OHSMS指針のなかで,「危険又は有害要因の特定手法として」,また厚生労働省基発第501号「機械の包括的な安全に関する指針」のなかでは,「その実施について」,それぞれ触れられている程度で法的な実施義務はない。しかし,指針として公表された以上は,社会通念に照らし,労働安全衛生に関する相当な措置と考えられることから,労働災害が発生した場合,指針に則した措置をとっていないと,民事責任である「労働者の生命および身体等を危険から保護するよう配慮すべき義務」,すなわち「安全配慮義務」の責任を負うこととなる.
