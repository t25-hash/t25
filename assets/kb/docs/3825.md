# 6・2・3 環境監査およびサイトアセスメント

> **章**: 第6章 マネジメントと法制度
> **節**: 6・2 環境マネジメント

a.監査関連規格の概要 ISO 14000シリーズ規格において,環境監査に関する規格は,図6・3に示したように,ISO 14010,14011,14012の3件である.これらの規格は,‘14001規格に要求されているマネジメントシステム監査を第三者機関が行う場合にも,また組織が内部的に行う場合にも,適用可能なように書かれている.14010は「環境監査の一般原則」を示したもので,環境監査に共通の原則として,監査の目的,客観性,監査基準,監査所見および監査報告などを規定している.14011には「環境マネジメントシステム監査の手順」が,14012には「環境監査員の資格基準」が,それぞれ規定されている.乏ころで,環境マネジメントシステムにおけるこれら三つの規格の位置付けであるが,ISO 14001規格の要求事項には環境マネジメントシステム監査がありながら,これら三つの規格は14001には引用されていない.この意味は,ISO 14001で要求されているマネジメントシステム監査には,これら三つの規格を使っても使わなくてもよいということである.なお,品質マネジメントシステム監査のためにも同様にして3件の監査指針が出されている.同じ監査の規格としてこのように6件の規格があるのはいかにも繁雑であるので,現在統合作業が進んでおり,ISO 19011として2002年には発行される予定である.b.監査の手順 a項で述べたように,ISO 14001規格で要求されているマネジメントシステム監査では,ISO 14011のガイドラインを使わなくてもよいが,実際にはこの手法が取り入られている場合が多い.このガイドラインには,監査の目的,監査に関わる人の立場と役割,監査のプロセスにしたがった手順などが規定されている.i.監査の目的 監査の目的は,環境マネジメントシステムの適合性の評価,適切な実施・維持に対する評価および改善の可能性のある部分の特定,であると記されている.最後の項目である改善の可能性のある部分の特定は,第三者機関が行う監査の場合には実施されないことになっている.ii.監査に関わる人の立場と役割 監査に関わる人の立場としては,監査の依頼者,監査者および被監査者があり,ガイドラインのなかではそれぞれの立場の人の役割が,6項目から14項目あげられている.依頼者は,監査の必要性,目的および範囲などを決め,監査者を選択する役割を持つ.監査者にはリーダ格と一般監査員の立場があり,それぞれで役割が異なる.リーダには監査基準の確定,計画・報告の作成,チーム員に対する指示,、スムースな監査の実施・遂行などの役割が課せられている.被監査者の役割は,情報や施設等の提供など監査に協力することである.iii.監査のプロセスにしたがった手順 監査の手順は,準備,実施および報告の3プロセスに分けて規定されている.準備の段階では,被監査者からの情報入手・分析,必要文書の作成,チーム員への作業割当およびチーム内のコミュニケーション形成などが実施される.監査の実施段階では,初回会議に始まって,証拠収集(要求事項が満たされていることを示すもの),適合に関する分析,所見のまとめおよび所見を相手に伝える最終会議などが行われる.引き続き報告書の作成作業が行われるが,報告書作成で最も重要な点は正確な証拠に基づいた客観性のある記述がなされることである.実際の監査では,この後に指摘に基づく是正(これは被監査者の役割である)および是正結果のフォローアップ(監査者の役割)が行われるが,これらの部分はISO 14011規格には含まれていない.c.監査員の資格基準 ISO 14012のガイドラインには監査員の資格基準が規定されている.この規格もISO 14001に引用されていないので,ISO 14001規格に則って環境マネジメントシステム監査を実施する場合の制約とはならない.あくまでも参拍了すればよい.ガイドラインには,資格要件,資質要件,さらに監査能力の維持について,それぞれ規定されている.資格要件については,教育として中等教育(日本の高等学校程度)β9-146が必要であり,業務経験としては環境科学・技術に関連する業務,環境施設の運用や技術に関する業務,あるいはマネジメントシステム監査の運用に関する業務などの経験を4~2年持つことが求めら・れている.さらに監査員は,マネジメントシステム関連事項を含む正規の訓練を経た上,認定試験などに基づく資格を持つことおよびその後に続く20日間以上の監査実績を持つことが必要である.監査員の資質については,ある意味では資格よりも重要と考えられるが,ガイドラインでは表現能力,対人能力,独立性,組織化能力および判断能力などが示されているに過ぎず,その評価には触れられていない.また,能力の維持については,業務経験事項に関連する知識を新しくすることだけが示されている.d,サイトアセスメント ISO/TC 207では設立後間もない1993年11月に,監査関連の検討項目10件をあげて着手の優先順位を設定した.その一項目として,サイトアセスメントが取り上げられた.その後規格のタイトルは何回か変更されたが,1997年頃から標準化作業が本格的に進み;ISO 14015として2002年中に発行予定である.現在の規格のタイトルは「サイト及び組織の環境アセスメント」とされている.以下にその内容を述べる.まず,規格運用の原則として,下記の4項目が合意されている.・アセスメント指向(監査ではない)・土壌に限定せず,大気・水質も対象とする(サイト及び組織のアセスメント)・商取引に限定しない・マネジメント文書として運用する次に,規格の構成に沿って,その内容を示す.L適用範囲 この規格は,環境側面および環境事項を特定し,適切であればその結果のビジネスへの影響の決定を下す体系的なプロセスを通じて,サイトや組織のアセスメントを行うためのガイダンスであるとしている.ii.用語及び定義 用語は15語あるが, Iso 14001で規定されているものが5語,アセスメントに係わる人に関するものが3語,アセスメントに関するものが7語となっている.iii.役割・責任 アセスメントに係わる人の立場として,依頼者,アセサー(評価者),被評価代表者を取り上げ,それぞれの役割を規定している.注目される点は,監査に比べると依頼者の立場が非常に強いことである.’アセスメントの必要性,目的,範囲,アセスメント基準,評価者等の決定をはじめとして,評価者への指示,計画承認および報告書の受領など,すべてが依頼者の手に委ねられている.iv。アセスメントプロセス サイト・組織の環境アセスメントプロセス計画の段階では,依頼者によって目的,範囲およびアセスメント基準が決められ,計画書が作成される.アセスメント基準としては次のようなものでがある.1法的要求事項(現在適用される法律および予測される法律)2 依頼者が特定した要求事項(環境方針・手順・特殊な環境事情,マネジメントおよびパフォーマンス上の要求事項,業界基準など)3第三者的利害関係者(金融保険機関)の要求事項および潜在的苦情4技術的配慮事項以上のようにして計画が完成すると,次のステップは情報収集およびその検証である.監査と同様に情報収集と検証は,まず文書・記録類から始まる.原材料から製品,廃棄物・排出物などすべてがその対象となり,’ここでは労働安全・衛生関連事項も含まれる.次は,サイト内で行われている活動や物理的な条件が検証の対象となる.物理的条件のなかには,冷暖房・換気からサイトの景観や動植物への影響事項までが含まれる.さらに,サイト各階層へのインタビューが行われる.このようにして収集された証拠が評価の対象となる.アセスメントが事業上にどのような影響を与えるかの決定は,依頼者の意図に基づいてこの評価プロセスのなかで行われる.
