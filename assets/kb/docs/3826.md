# 6・2・4 環境パフォーマンス評価

> **章**: 第6章 マネジメントと法制度
> **節**: 6・2 環境マネジメント

a．概要 ISO 14000シリーズ規格において，環境パフォーマン
ス評価に関する規格は，図6・3に示したように，ISO 14031と14032
の2件である．治014031規格は環境パフォーマンスを評価するため
のガイドラインであり，ISO 14032規格はパフォーマンス評価の事例
集である．
ISO 14001規格は監視・測定のなかで，パフォーマンス評価の実施
を求めているが，前述の監査と同様，上記二つの規格も14001には引
用されていないので，パフォーマンス評価をするにあたってはこれ
らの規格はあくまでも参考にすればよいことになる．
なお，ISO 14001規格ではパフォーマンス評価は求めているが，パ
フォーマンろレベルの設定は組織にまかされている．
b．パフ才一マンス評価 ISO 14031規格のなかで，環境パフォ
ーマンス評価は，「組織の環境パフォーマンスに関して，経営判断を
しゃすくするプロセス」とされている．そしてさらに具体的に，「環
境指標を選定すること，データを収集分析すること，これをパフォ
ーマンス基準と対比・評価すること，報告・コミュニケーションを
すること，およびそのプロセスの定期的なレビュー・改善をするこ
と」と記されている．
すなわち，組織の環境側面に関する情報を的確に表す指標を定め，
これに関連するデータを収集して，これらの数値を指標に置き換え，
それが目標値に対してどのレベルにあるかを評価ずる．そして，評
価結果に関して内（外）とのコミュニケーションを図ることである
（外部へのコミュニケーションは，情報開示であるからその実施に
ついてはISO 14001は組織に委ねている）．さらに，この環境パフォ
ーマンス評価そのものを見直し，改善を加えて，次年度の計画・実
施へとつなげていく．これが，ISO 14031規格が定めている環境パフ
ォーマンス評価のマネジメントモデルである．
c．評価の事例 ISO 14032には，従業員11人の小企業から10万
人を超す大企業までを対象に，環境パフォーマンス評価の実施例あ
るいは計画事例が紹介されている．国別では日本，ドイツ，アメリ
カなど10ヶ国，また業種としては各種製造業（食品，家具，プラス
チック，家電品，化学薬品等），サービス業（鉄道，病院），さらに
地方自治体，NGOなど，における実施例が含まれている．パフォ
ーマンスレベルが大幅に上昇している例が示されるなど，継続的改
善を目指す組織にとって参考になる．
