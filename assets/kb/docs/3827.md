# 6・2・5 環境ラベル

> **章**: 第6章 マネジメントと法制度
> **節**: 6・2 環境マネジメント

a.概要 ISO 14000シリーーズ規格において、環境ラベルに関する規格は、図6・3に示したように、ISO 14020,14021,14024,14025の4件である。i.環境ラベルの狙い 自由競争の市場社会において、各種の製品は工夫をこらしてその特徴を発揮しようとする。従来の製品のコンセプトには、機能、コストなどは含まれていたが、環境への視点が盛り込まれたものは少なかった。製品の環境に与える負荷は、原料・材料の生産、製品製造、販売・流通、使用および廃棄のそれぞれの段階で発生し、製品によってどの段階での負荷が高いかは異なる。環境ラベルの考え方は、これら製品のライフサイクルを通して環境影響に配慮されたものを評価し、差別化しようとするものである。環境ラベルは、14001規格とは直i接的には関係ないが、製品の環境側面を改善していく上では有効なツールである。ii.環境ラベルの三つのタイプ 環境ラベルは世界の各国でさまざまな形で実用化されているが、ISO/TC 207ではこれらを三つのタイプに分類して、それぞれのガイドラインをまとめ上げた。その一つ目は、マネジメントシステムの認証と同じように、第三者機関がある基準に基づいて評価した上で与えるもので、タイプ1と呼ばれている。二つ目は、その製品を生み出す組織自身が評価を行った上で自発的な宣言の形で表現するもので、このなかには省エネルギー型、再生可能など12のクライテリアがある。これをタイプHと呼ぶ。三つ目は、製品の製造に要する資源、エネルギー、あるいは化学物質等の使用量などの環境情報をラベル化して製品に表示するもので、タイプ皿と呼ばれている。iii.環境ラベルの一・般原則 ラベル表示が適切でなかったり、評価が不当である場合には、自由競争の原則を破壊しかねないし、特に貿易障壁の原因となる懸念がある。こうした点に配慮して、ISO14020の「環境ラベルの一般原則」のなかでは、すべてのタイプのラベルに共通する下記9原則を重視すべきであると記されている。1 ラベル・宣言は正確、検証可能、適切で誤解を生じないこと。2ラベルの表示手続要求事項は貿易障壁の誘因となってはならない。3ラベルによる主張の裏付けは正確かつ科学的でなければならない。4 ラベルの証認判定の根拠となる情報は利害関係者が入手可能なこと。5ラベル・宣言の評価基準には製品のライフサイクルにおける環境側面すべてが配慮されていること。6 ラベル・宣言は環境パフォニマンスの維持・改善および技術革新を抑制してはならない。7 ラベル運用上の要求事項は適用判断基準および評価基準の確立に必要なものに限定される。8 ラベル・宣言の評価基準開発プロセスには情報の一般公開および一般のコンセンサスを得る努力が強く望まれる。9 ラベル・宣言の対象は製品の持つ環境側面に関する情報の入手が購入者にとって可能でなければならない。b.各環境ラベルの特徴i.第三者機関認証型ラベル。..タイプ1 このラベルは、製品・サービスを提供する企業や購買者から独立した第三者機関が、決められた製品カテゴリと判定基準によって審査を行い、合格した製品にラベルの使用を認めるものである。ドイツ、北欧を中心にして世界中に25以上の実例があり、日本国内では日本環境協会によるエコマークが68品目の製品を対象に実施されている。すでに実施されているこれらのラベルの運用を統一するρは難しいので、ISO 14024規格は個々の製品分野や判定基準を規定することはやめて、新しいラベルの開発や運用に関するガイドラインとなっている。ISO 14024規格には、自主参加の原則に始まって、法規遵守、ライフサイクル的思考、差別化、環境基準、製品機能、検証期限、透明性、科学性などを含めて18の原則が列記されている。この規格は、環境面で優れた製品に対して、他の製品との差別化を行った上、有利な条件を与えようとするものであるから、途上国への差別、非β9-147関税貿易障壁の発生など種々の問題を産みかねない。これを避ける目的で、環境基準選定のマトリックスが表6・3のように示されている。内容は空欄のままであるが、ここには資源の利用あるいは汚染に関する指標が記述されることになる。表6・3 このラベルは、製品を提供する企業が自己宣言によって行う環境主張である。したがって必ずしも製品に貼付するとは限らず、種々の媒体を通して主張することが可能である。ISO 14021規格には、主張項目、主張を適用するにあたっての要求事項、評価・検証のための要求事項および主張項目ごとの特別要求事項などが規定されている。自己宣言による主張であるが、製品の製造者がこの規格に則って自己宣言を実施するためには、ここに規定された要求事項は守らなければならない。自己宣言によって主張できるのは次の12項目である。1コンポスト化可能2分解可能3解体可能設計4長寿命製品5回収エネルギー6リサイクル可能7リサイクル材含有率 このラベルは、製品単位当りの環境関連情報として、資源・エネルギーの使用量、汚染物質の排出量、廃棄物発生量、化学物質あるいは有害物質の使用量などを数値データとして、一覧表もしくは棒グラフなどの形で製品に表示するものである。通常、薬品、食品には成分表が付けられているが、これに加えてエネルギーや廃棄物のデータを付け加えた形を想像すれば良い。タイプ皿ラベルでは生のデータがそえられるが、単に表示するのみでなく、第三者の評価が可能なように表示する必要がある。タイプ皿ラベルの実用例としては現在、アメリカのSCS(Scien・ti丘。
