# 1・3・4 寸法効果（scale eﬀect ）

> **章**: 第1章 気体機械一般
> **節**: 1・3 性能試験と性能換算

模型で試験を行う場合，寸法の違いはレイノルズ数の中に代表寸
法として含まれているので，この効果を修正する必要はないように
思われるが，幾何的相似条件を細部にわたるまで満たすことは実際
上困難であり，表面粗さ，機械要素間のすきま，角の丸みなど，相
似にできない要因の影響を受け，模型の寸法比が小さくなるほど性
能は低下する．これらの効果をまとめて寸法効果またはスケール効
果という．寸法効果の現れ方は機種によって異なるが，公表された
資料は少ない ．
