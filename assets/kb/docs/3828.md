# 6・2・6 ライフサイクルアセスメントおよび環境配慮設計

> **章**: 第6章 マネジメントと法制度
> **節**: 6・2 環境マネジメント

a.製品を指向した環境マネジメント ISO 14001は、組織に対してその組織の持つ環境側面のマネジメントを求めたものである。環境側面とは組織の活動、製品またはサービスの要素をいう。これらの側面がもたらす影響を評価してマネジメントを行うが、使用者の手に渡った後の製品やサービスの産み出す環境影響には、使用者の取り扱いが関ってくる。また、製品製造のさらに上流側では材料の製造プロセスがあり、さらにその上流には例えば鉱物資源の産出過程があって、そのそれぞれで環境影響を生じている。したがって、製品の環境影響を厳密に評価しようとすれば、いわゆるライフサイクルアセスメント(LCA)が必要となる。ISO 14001はすべての環境側面を配慮することは求めてはいるが、製品の側面に関して上述のようなLCA的な取り扱いは求めていない。しかし、環境マネジメントシステムが普及し、システムが成熟してくると、利害関係者の目は次第にシステムそのものよりも、適用組織の環境パフォーマンス、製品の環境影響(LCA的な取り扱い)β9-148および情報の開示に向けられるようになる。このような背景のもとに、製品の環境影響をより広い範囲でマネジメントシステムに取り込むことが考えられるようになっており、「製品を指向した環;境マネジメントPOEM(Products OrientedEnvirommental Management)」という言葉が聞かれるようになっている。ただし、ISO 14001に基づくマネジメントシステムのなかで、この部分がどのように扱われるか(適合性の評価対象として)は、今のところ定かでない。b.ライフサイクルアセスメント規格 ISO 14000シリーズ規格において、ライフサイクルアセスメント(:LCA)に関する規格は、図6・3に示したように、ISO 14040,14041,14042,14043,14047,14048,14049の7件である。ISO 14040では、 LCAの原則及び枠組みが示されており、 LCAのフェーズを図6・9のようにとらえ、各フェーズごとに注目事項が記述されている。特に製品の比較をする場合には公平を期するために、利害関係者によるクリティカルレビューの実施が規定されている。ライフサイクルアセスメントの枠組み(14040)目的及び調査ヘ囲の設定iISO 14041)直接の用途解@ 釈E製品の開発及ム改善E戦略立案E政策立案インペントリ分析i1SO 14043)iISO 14041>EマーケティングEその他影響評価iISO 14042)注)数宇は規格番号図6・9 ライフサイクルアセスメントISO 14041では、 LCAの目的及び調査範囲の設定並びにインベントリ分析が規定されている。インベントリ分析とは、製品に関るすべての環境側面のリストアップおよび関連付けで、LCAの基本作業である。ISO 14049はインベントリ分析の事例集で、分析手順のうち、特に重要な部分に対して集中的にまとめられている。ISO 14042は、影響評価を規定した規格で、 ISO 14047の影響評価事例集およびISO 14048のデータフォーマットは、14042を支援するツールとなっている。ISO 14043では、:LCAの最終段階である解釈の部分が規定されている。解釈とは、実施された:LCAの結果に対して、その不確実性や感度に対する評価を与えるもので、附属書には解釈事例が集められている。c。環境配慮設計 1999年度からISO/TC 207のなかでは、新たに環境配慮設計DFE(Design fbr Environment)の議論が始まっている。LCAは体系的な技術分野の豊富なデータベースを必要とするものであるが、DFEには実務に添ったガイドラインを作成し設計への環境配慮を拡げようという意図がある。すなわち、このガイドラインは、製品の環境影響を指向した環境マネジメントをより前進させることと製品連鎖のなかでのトラブル回避が目的で、特に中小企業あるいは途上国のニーズが多い。内容は予めところ、製品の企画・開発者あるいは設計者を対象とした:LCAの応用編を目指したものである。この規格はISO/TR 14062(TR:技術レポート)として発行の一型であり、10数下国の代表が参加して作成が進められている・規格の主要な構成は以下のとおりである。・期待される効果・組織の方針・戦略に沿ったDFEの検討』・組織のマネジメントに沿ったDFEの検討・製品における検討事項・製品設計、開発プロセスにおける検討事項
