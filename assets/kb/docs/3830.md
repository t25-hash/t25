# 6・3・1 環境影響評価法の成立経緯 我が国では1950年代頃

> **章**: 第6章 マネジメントと法制度
> **節**: 6・3 環境アセスメント

から,戦後の急速な高度経済成長の結果,京浜工業地帯や四日市石油コンビナート地区において,深刻な大気汚染などの公害問題を引き起こし,川崎喘息や四日市喘息(ぜんそく)などの健康被害が生じた.その対策はまず法制度の整備から始められ,公害対策基本法(1967年)や自然環境保全法(1967年号が成立した.環境影響評価については,1972年の閣議了解事項「各種公共事業に係る環境保全対策」(一般に閣議アセスと呼ばれている)によって行われてきたが,実際に環境影響評価制度が法制化されるには紆余曲折があった.まず,制度化が行われたのは地方自治体であり,東京都環境影響評価条例および神奈川県環境影響評価条例が1980年10月に制定され,首都圏で環境影響評価制度の採択が先行した.国では,1981年4月に環境影響評価法案が国会に提出されたが,衆議院が解散され廃案になった.ただし,1984年8月には,実質的に環境影響評価を行うために,「環境影響評価の実施について」が閣議決定されている.その後,1993年に環境基本法が制定され,そのなかで,環境影響評価制度を推進するため必要な処置を国は講ずること,が明記された.このため1994年から96年にかけて,環境庁では内外の環境影響評価制度についての調査を行い,その結果に基づき,1996年6月に内閣総理大臣から中央環境審議会に,「今後の環境影響評価制度の在り方について」の諮問が行われた.同審議会では約半年の審議を経て,1997年2月に法制化に向けた答申が取りまとめられた.そして,同年3月に環境影響評価法が国会に提出され,6月9日に可決成立した.環境影響評価の法制度導入の背景として,環境影響評価制度を各国が採用し始めたこと,また輸出障害とならないように,我が国においても環境影響評価制度を公開された法制度として定着させることが必要であったこと,などがあげられる.
