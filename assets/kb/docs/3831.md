# 6・3・2 環境影響評価の手順 ここでは、環境影響評価法に

> **章**: 第6章 マネジメントと法制度
> **節**: 6・3 環境アセスメント

基づく環境影響評価の手順について、図6・10のフロー図にしたがって述べる。(1) 事業の概要 事業者は、事業の概要を取りまとめて、当該事業が第一種事業か第二種事業かの判定を行う。第一種、第二種の事業規模は事業の種類ごとに表6・4に示すように決められている。一定規模以上の事業(第一種事業)を行うには環境影響評価手続を必ず実施し、これに準ずる規模の事業(第二種事業)については主務大臣が知事意見を聞いて環境影響評価を行うかどうかを決めると定められている。以上の手順はスクリーニングと呼ばれている。(2) 方法書の作成 事業者は、環境影響評価方法書(環境影響評価計画書と呼ぶ場合もある)を作成し、主務官庁または自治体へ提出する。方法書は、事業区域およびその周辺における自然的状況と社会的状況を踏まえて、選定した環境影響評価項目ならびに調査、β9-149事業の概要スクリーニングスコーピング知事意見←判断(主務大臣)対象外方法書の作成公示・縦覧意見(住民等)意見(知事等)必要に応じ技術的助言(主務大臣)調査の実施準備書の作成公示・縦覧意見(住民等)意見(知事等)評価書の作成意見(環境大臣)↓意見(主務大臣)公示、縦覧「・葡・。..●・・。●。。 ・’・。”●、免許等審査(主務大臣)=:::=:正二事業開始図6・10環境影響評価のフロー重予測および評価の方法を記載したものである(詳細は6・3・5項a参照)。方法書は公示・縦覧され、住民や知事が意見を述べることができる。以上の手順はスコーピングと呼ばれている。(3) 調査の実施 事業者は、環境影響評価方法書に沿って、必要な調査を実施し、その結果に基づき予測および評価を行う(詳細は6・3・4項参照)。(4) 準備書の作成 事業者は、調査結果に基づき、環境影響評価書の原案にあたる環境影響評価準備書を作成し、主務官庁または自治体へ提出する。これは公示・縦覧され、住民や知事が意見を述べることができる。(5) 評価書の作成 事業者は、これらの意見を踏まえて準備書を訂正し、環境影響評価書を作成し、主務官庁または自治体へ提出する。評価書は、各環境影響評価項目に対し、現況調査結果、予測結果および評価結果を詳細に取りまとめ、事業の環境に及ぼす影響の評価の結論を提示したものである(詳細は6・3・5項b参照)。この評価書に対し、環境大臣は意見を述べることができ、主務大臣は意見を考慮して評価書の修正や補強を事業者に申し出ることができる。事業者は最終的な修正を:行う。(6) 公示、縦覧 事業者は、環境影響評価書を公示、縦覧しなければならない。公示縦覧後、主務大臣は事業者に事業の許可(免許など)を与え、事業が開始される。以上環境影響評価の手順の概要を述べたが、そのほかに手続きのβ9-150表6・4 第一種、第二種の事業規模1.道 路・高速自動車国道E首都高速道路等E一ハ国道・ダムE堰(せき)E湖沼水位調節施設E放水路2.河 川・新幹線鉄道E普通鉄道E軌道すべてP0km以上P0km以上7.5km以上10km未満V.5km以上10km未満4.飛行場滑走路2500m以上1875m以上2000m未満3.鉄 道2.25万kW以上3万kW未満P1.25万kW以上15万kW未満V500kW以上1万kW未満6.廃棄物最終処分場30ha以上25ha以上30ha未満7.公有水面埋め立て又は干拓50ha超40ha以上50ha以下8.土地区画整理事業100ha以上75ha以上100ha未満9.新住宅市街地開発事業100ha以上75ha以上100ha未満10.工業団地造成事業100ha以上75ha以上100ha未満11.新都市基盤整備事業100ha、以上75ha以上100ha未満12.流通業務団地造成事業100ha以上75ha以上100ha未満・水力E火力E地熱5.発電所75ha以上100ha未満V5ha以上100ha未満V5ha以上100ha未満14.港湾計画埋立、堀込300ha以上13.宅地造成事業・環境事業団E住宅・都市整備公団E地域振興整備事業団経過日数などがそれぞれ定められており、それに従わなければならない。それらは事業の種類ごとにそれぞれマニュアルに取りまとめられており、詳細は参考文献(1)~(7)などを参照されたい。なお、環境影響評価においては、その予測には必ず誤差が伴ケし、また手法には固有の不確定性がある。誤差や不確定性が大きいと推定されるときには、「事後調査」によってこれらを確認しておくことが望ましい。過去に、地方自治体の環境影響評価条例に基づいて、事業者が「事後調査」を求められた例もあるが、環境影響評価法では、「事後調査」を義務付けてはいない。
