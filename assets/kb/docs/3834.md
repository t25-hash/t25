# 1・2・7 シール

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

水力機械の主軸がケーシングを貫通する部分で，内液が外部へ漏
れたり，外の空気が内部へ侵入するのを防ぐためにシール機構を設
ける．
a.グランドパッキン グランドパッキン（gland packing）
は，最も一般的で多少の漏れを許す場合に用いる．木綿，金属はく
（アルミはくなど）などに黒鉛，テフロンなどを潤滑剤として含浸
させ，四角断面のリング状に成形したパッキンをパッキン箱につ
め，パッキン押さえで締付ける．パッキンの種類は液質，圧力，温
度，しゅう動速度などから選定するが，ふつう圧力0.5MPa（金
属はくは1MPa）以下に用いられる．図Ⅱ-1･27は中間に封水リン
グを設けた構造で，ここに軸封液を注入して空気の侵入を防いだ
り，ポンプ揚液中の砂などの異物がパッキン部に入るのを防ぐ．
b.メカニカルシール メカニカルシール（mechanical seal ）
は，回転軸に取付けられた従動リングと，ケーシングに取付けられ
た固定のシートリングを軸に直角な面で向かい合わせ，固定面に回
転面をばねで押付けて回転させる構造のものである．漏れはほとん
どないので，原子力用ポンプをはじめ，漏れをきらう場合に用い
る．一般にシートリングは焼結カーボン，従動リングはステンレス
鋼母材にステライト，セラミックスなどをコーティングしたものを
用いる．なお，しゅう動部分を冷却し，しゅう動面がドライ接触に
なるのを防ぎ，ポンプ揚液中の異物の侵入を防ぐため常温洗浄液を
注入してフラッシングを行う．
図Ⅱ-1･28はバランス形メカニカルシールの例で，圧力3MPa
以下に用いられる．液質，圧力，シール面のしゅう動速度などによ
り種々の構造のものが製作されている．なお，液圧で押される従動
リングの受圧面積がしゅう動面の面積より小さいものがバランス形
で，そうでないものをアンバランス形という．
