# 1・3・2 性能試験

> **章**: 第1章 水力機械一般
> **節**: 1・3 性能試験と寸法効果

かわる研究のために，水車・ポンプ水車ではおもに模型を使用した
模型試験（model test ）で，また必要に応じて現地の実物を用いた
実物試験（prototype test ）で実施される．ポンプでは，大型のも
の以外は工場で実物を用いた試験が実施される．性能は無拘束速度
特性を含む効率特性とキャビテーション特性がおもなもので，前者
はキャビテーションが発生しない条件で試験する決まりとなってお
り，後者は1･4節で述べるので，ここでは前者に限定する．
も性能試験に含まれ，必要に応じて同じ模型を用いて模型試験が実
施される．さらに，1970年代以降ポンプ水車の高落差化が顕著に
なるに伴って，機器の信頼性を高めるための特殊試験，すなわち実
揚程・実落差模型試験，および過渡現象模型試験が特別な試験設備
で行われようになっているが ，これらについては省略する．
