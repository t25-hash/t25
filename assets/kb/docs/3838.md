# 1・3・4 実物試験

> **章**: 第1章 水力機械一般
> **節**: 1・3 性能試験と寸法効果

現地での実物試験（prototype test ）は水車・ポンプ水車に関す
るものがほとんどであるので，以下これに限定して述べる．
水車・ポンプ水車の実物試験は，通水しない停止状態で各機器の
動作，特性などをチェックする無水試験から，通水して主機を無負
荷で回転させながら軸受温度，軸振れ，振動および漏水などの異常
をチェックするメタルならし試験，次いで自動始動・停止試験，負
荷をとっての出力開度試験などの有水試験が行われ，正常な運転が
確認された後，性能試験が実施される．回転速度一定で運転される
水車またはポンプ水車の実物試験で，落差または揚程は正確に測れ
るが，動力や流量は測定量が大きいので，模型試験に比べて測定精
度を確保することが難しい．水車出力は測定された発電機出力とそ
の効率から，ポンプ入力は駆動電動機入力の測定値からその損失を
差引くことによって求められるが，これら効率または損失は現地測
定が困難なため，工場試験結果によることが多い．流量測定は，模
型試験とは異なる現地測定に適した流速計法，ピトー管法，圧力時
間法，超音波法，トレーサ法，インデックス法などがおもに使われ
る．なお，高落差水車・ポンプ水車の現地効率試験には熱力学法も
使用されている．これら実物試験は国内および国際規格 にのっ
とって実施され，使用測定機器および測定方法が規定されている．
