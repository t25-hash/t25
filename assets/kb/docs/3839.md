# 6・5・1 一般事故の調査 我が国では,航空・鉄道事故,原子

> **章**: 第6章 マネジメントと法制度
> **節**: 6・5 事故調査と事故責任

力事故,海難事故に対しては,それぞれ常設の専門機関が設けられ,調査等にあたっている.これらの事故調査体制については6・5・2,6・5・3,6・5・4項で述べる.ここでは,それ以外の事故として,身近に発生する道路交通事故,一般家庭でも起こる火災事故および産業事故である化学プラント事故を取り上げて,それぞれの調査体制について述べる.a.道路交通事故 道路交通事故は年間約85万件発生しており,「人間」,「道路環境」,「車両」の要因が複雑に関与している.また,非職業人である一一般人が加害者である場合が多い点で,他の事故と状況が異なっている.現実にどのような交通事故が発生し,各種の安全対策がどのように役立っているかなどを含め,安全性を向上させるための事故調査が望まれる.i.警察による調査 警察の責務は,警察法第2条によって,犯罪の捜査などの「司法警察」と公共の安全と秩序の維持にあたる「行政警察」である,と定められている.この責務は,警察法第36条第2項により,基本的には都道府県警察によって遂行される.警察法第2条では,交通の取り締りを公共の安全と秩序の維持の例:示として掲げており,道路交通法に基づく警察の活動は「交通警察」と呼ばれている.また,道路交通法第72条により,交通事故が発生した時は直ちに警察官・警察署に連絡しなければならない,とされている.これらから道路交通事故の調査は警察の担当となっている.しかし,警察の調査はあくまでも責任所在の明確化(捜査)が主目的となっている.ii.第三者機関による調査 交通事故は,警察以外の第三者機関が安全性向上を目的に調査し,原因を客観的に分析して事故防止に資するべきであるとの世論もあり,(財)交通事故総合分析センターが道路交通心土六章の三に基づいて設立されている.本分析センターは,1993年から茨城県つくば地区で年間約300件程β9-166度,加えて1997年からは全国を対象として年間約100件程度の交通事故例の調査を行っている.実際の交通事故例について,運転者・同乗者・歩行者等の道路利用者,道路交通環境,車両,救急・救助,医療(傷害状況)などの総合的な観点から,詳細かつ科学的な調査を実施し,事故防止,被害軽減などの交通事故防止対策に資するための基礎資料を収集している.調査対象となった事故については,警察の行う事故調査と並行して調査される.本分析センターで実施された個別事故の調査結果(ミクロデータ)については,情報提供者との外部提供に関する了解事項やプライバシー保護あるいは非訴訟目的利用などの観点から,外部提供は行われていないが,その分析結果をまとめた「交通事故例調査・分析報告書」が毎年公表されている.このミクロデータは調査件数に限りがあり,十分な統計データを得るにはいたつていないため,これとは別に,関係官庁から全国の自動車事故データ,運転免許データ,自動車登録データ,道路センサスデータの提供を受けて,交通事故統合データベースを構築するなど,「交通事故の防止および被害の軽減」を目的とした活動も行っている.b.火災事故 火災事故の場合には,事故を教訓とするために多くの調査と事後評価がなされてきた.火災の場合は,燃焼により原因究明のための物的証拠が失われる場合が多く,火災特有の困難さがある.i.火災調査権と犯罪捜査権 消防法第31条では,消防長または消防署長に消火活動とともに火災原因,損害の調査を義務付けている.一方,刑法上九章には放火および失火の罪が規定されており,刑事裁判を受けさせ刑罰を科すことを目的として検察官や警察官が捜査を行う.このため,火災現場において,いわゆる「火災調査権」と「犯罪捜査権」が競合することがある.火災調査は火災の原因または失火の原因を究明するために行われ,捜査は犯罪の事実を立証するために行われる.これらは同じ対象を別々の観点から見ているといえる.ii.火災調査の特殊性 ホテルニュージャパンの火災(1982年2月)の時,学会関係者が消防関係者と偽り真っ先に現場に入り,早い時点で報道関係者に事故内容を伝えたことがある.そのため,兵庫県長崎屋の火災(1990年3,月)の時は,学会関係者が現場に入るのを警察がストップし,建物の構造,煙の広がり方,管理責任などの一切が不明のまま終わってしまった.事故の再発を防止し国民の命を守る立場からの調査が望まれるが,警察は犯罪捜査が主目的となりがちであり,結果として調査内容のすべてが公表されるとは限らず,原因究明や予防の観点からの情報が得られ難くなる.火災原因の究明は現場の燃焼痕を主として取り扱うが,これは火災の最終状態を示すもめで拡大過程を示してはいない.火災は本質的に発達拡大していき,その過程で人的被害を及ぼす.動的な火災のダイナミックスを把握するためには,複数の在筆者の行動や供述が貴重な情報源となる.c.化学プラント事故i.安全性確保のための法規制 化学プラントや危険性化学品を製造・貯蔵している施設・設備では,火災・爆発や有毒物質の漏洩などの事故により大きな被害をもたらすことがある.この場合,被害が周辺住民へ及んだり,環境汚染を引き起こし重大な社会問題化することもある.そのため,これらの施設・設備に対しては,安全性を確保するための法的規制が行われており,事故に対する公的な調査はこの法規則に基づいて行われる.例えば,危険物を貯蔵あるいは滞留する施設・設備は消防法(所轄:市町村),高圧ガスの製造や貯蔵等は高圧ガス保安法(経済産業省および都道府県),火薬類を製造する施設・設備は火薬類取締り法(経済産業省)により規制されている(詳細は4・1節参照).ii.各種公的機関による事故調査 これらの施設・設備で事故が発生した場合の調査は,事業者自らが行う場合と公的機関によって行われる場合がある.公的機関が実施する事故調査は,法規に基づいた行政処置として行われ,事故発生直後から施設・設備には使用停止命令が出され,調査後十分な再発防止対策が講じられて初めて使用が許可される.社会的に大きな問題となるような被害が出た場合は,専門家で構成される事故調査委員会が市町村,都道府県,経済産業省などのもとに臨時に設置され,調査が行われる.また,人的被害や重大な損害がある場合は,警察が捜査上必要な事故調査を行う.さらに,厚生労働省は,労働安全衛生法に基づいて,労働災害の再発防止の見地から事故調査を実施する.被害が小さい場合は1当該事業所に事故調査を命じ,その調査結果の報告と再発防止の処置を確認して,しかるべき行政処置をとる.このように,種々の公的機関が事故調査に関与しているので,相互の連携と協力がスムースに行われることが,有効な再発防止策を講じ安全性を高めるために必要である.
