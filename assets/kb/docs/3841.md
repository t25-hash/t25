# 6・5・3 原子力事故の調査と災害対策

> **章**: 第6章 マネジメントと法制度
> **節**: 6・5 事故調査と事故責任

a.事故調査 原子力施設の事故調査は,第一義的には所轄:官庁が実施している.すなわち,発電に供する施設の事故時には経済産業省原子力安全・保安院が,それ以外の施設の事故時には文部科学省が,それぞれ調査にあたる.それとともに,原子力安全委員会の「原子力事故・故障調査専門部会」が事故の把握に務め,重大事故時には調査委員会を設置し,所轄官庁とは別途に調査にあたる.ここで,原子力事故・故障調査専門部会の設立経緯について若干β9-167触れておく.原子力基本法と原子力委員会及び原子力安全委員会設置法では,原子力安全委員会の任務として,「原子力の研究,開発及び利用に関する事項のうち,安全の確保に関する事項について企画し,審議し,及び決定する」と述べられているだけで,事故調査については特段の規定はなかった.しかし,1999年9月に発生した燃料加工工場における臨界事故の時には,原子力安全委員会内に臨時の「ウラン加工工場臨界事故調査委員会」が設置され,調査を実施した.この事故が契機となり,同安全委員会の機能と独立性を強化する必要性が認識され,2000年4月からは事務局が92名に拡充され,科学技術庁から総理府へ移管された.2000年9月には同安全委員会のもとに,事故時の調査に対応する常設の組織として「原子力事故・故障調査専門部会」が設置された.2001年1月には省庁再編に伴い,同安全委員会は内閣府の所属となり,事務局も100名へと増員されている.b.災害対策 燃料加工工場における臨界事故の場合は,災害対策基本法に基づく防災基本計画にしたがい,政府事故対策本部(本部長:科学技術庁長官)が設置され,防災の面からの対応が行われた.さらに1999年12月には原子力災害対策特別措置法が制定され,原子力災害の態様の特殊性や事業者の責務などの原子力災害固有の問題をカバーする体制が整えられた(詳細は5・1・5項e参照).
