# 6・5・4 海難事故の調査と審判 海難事故の場合には、司法判

> **章**: 第6章 マネジメントと法制度
> **節**: 6・5 事故調査と事故責任

断に向けての海上保安庁による調査と、「海難審判法」に基づき行政裁決を行う海難審判庁による調査とが行われる。また、事故の性格から必要に応じて、地方自治体等の公共機関、保険機関、所属船会社などの独自調査が行われることもある。a.海難審判庁の組織と任務il組織 海難審判庁は国土交通省の外局であり、東京に高等海難審判庁および海難審判理事所が置かれ、地方の主要都市(函館、仙台、横浜、神戸、広島、門司、長崎)に地方海難審判庁および地方海難審判理事所が、沖縄に地方海難審判庁支所および地方海難審判理事所支所がそれぞれ置かれている。審判官と理事官の陣容は、海難審判施行令に定められており、審判官55名、理事官41名、副理事官9名である。ii.任務 海難審判は海難が、船舶の乗組員等のヒューマンファクタが原因で発生したものか、また船体や機関の構造や性能などが原因で発生したものか、などを専門の知識と経験を有する審判官の合議体によって審理し、裁決をもって明らかにする。海難が人の故意や過失により発生したときには、これらの者を懲戒(行政処分)し、さらに、海難の原因に関係した海運会社や造船会社などに対して、その業務や設備などの改善や改良を勧告する。理事官は、海難の調査、審判の請求と立会いおよび裁決の執行を行う。ノb.海難審判の手順i.調査 理事官は、海上保安官、警察官および市町村長などから海難があったことの報告を受けた時や、ラジオ、テレビなどにより海難を知った時は、直ちに調査を開始する。その実施事項は、1海難関係人を出頭させ、または質問する、2検査の実施と証拠品の領置、3事件(海難審判庁では一般に事故のことを事件という)に関する照会、鑑定、翻訳、である。日本近海において発生した事故に対しては、各地方海難審判理事所が調査を担当し、それ以外の全世界にわたる水域に対しては、東経136度1分30秒と西経70度で二分して、太平洋側は横浜、インド洋と大西洋側は神戸が担当している。外国船舶に対しては、日本の領海内で発生した海難事故のみを扱う。また、軍艦や公用船などで治外法権を有する船舶は除かれる。なお、審判のための証拠としては、海上保安庁などが公権力を行使して取得した調書は使用されない。また、事故の調査、分析、解析の一部を大学や民間へ依頼することもある。ii.審判 理事官は調査の結果、海難の再発防止のために審判による原因究明が必要と認めたときは、地方海難審判庁に審判開始β9ヨ68の申立を行う。このとき、海難の原因に関係ある者を受審人または指定海難関係人として指定する。海難審判は次のように二審制をとっている。(1) 第一審(地方海難審判庁):公開の審判廷で、審判官3名および書記が列席し、理事官立会いのもとに、受審人、指定海難関係人およびこれらの者を弁護する補佐人が出廷し、口頭弁論により審理する。簡易な事件では1人の審判官で行うこともできる。 。(2) 第二審(高等海難審判庁):審判官5名によって、第一審と同様の手続によって新たに審理する。第一審と第二審とも、複雑な事件や重大事件のときは、学識経験者2人を審判官と同等の職務権限を有する参審員として加えることができる。受験人には弁護をする補佐人がつくことがある。補佐人選任に必要な資力のない人には、(財)海難審判協会が必要な費用の全部または一部を負担する。審理が終結すると、海難の原因および事実を明らかにした裁決が言い渡される。その際、受超人への懲戒や指定海難関係人への勧告が言い渡されることがある。懲戒裁決とは、受審人に対する免許の取消し、当該部門についての1ヶ月以上3年以下の業務停止、または戒告である。指定海難関係人(例えば造船会社など)に対しては勧告がなされる。それ以外の懲戒は行われない。なお海難審判裁決録は、現在ではデータベースとして日本財団のホームページ上で公表されている。高等海難審判庁の裁決で懲戒処分を受け、その処分に不服のある場合は、裁決言渡の日から30日以内に東京高等裁判所(専属)に裁決取消しの訴えをすることができる。海難審判は二審制をとっているため、地方海難審判庁の裁決に対して直接裁判所に訴えを提起することはできない。さらに、東京高等裁判所の判決に不服のある場合は、最高裁判所に上告できる。犯罪が関与したと思われるときは海上保安庁が捜査する。強制力があり、拘束や現場保全が可能である。裁判は地方裁判所で行われ・る。これに対して、一事不再理の法原則に反するのではとの意見もある。海難審判で単なる機関室の火災事故と裁決された後、保険金詐欺目的の火災であったという裁判結果が出た例もある。c.海難審判庁の広報活動 海難審判庁のその他の活動として重要なものに広報活動がある。海難審判庁が行っている業務を広く国民に紹介し、理解を深めてもらうために、毎年「海難審判の現況」を刊行している。2001年版は、21世紀を迎えた特集として、20世紀における海難審判制度および重大海難事件などの歴史を振り返るとともに、今世紀初頭の海難審判行政の目指す方向について記述されている。さらに2000年における、審判により究明された海難原因、海難発生の状況、調査および審判の状況などについて、統計的な分析を加え紹介している。また、特定の事項についての分析結果を「海難分析集」として随時刊行している。最近の分析集としては、「漁船海難iの実態」(2001年皐月)、「底びき網漁船の操業中における転覆・沈没海難の分析結果について」(2001年4月)、「遊漁船・瀬渡船海難の実態」(2001年8月)などがある。
