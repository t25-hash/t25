# 6・5・7 事故調査における免責制度

> **章**: 第6章 マネジメントと法制度
> **節**: 6・5 事故調査と事故責任

a．免責制度の必要性 事故の真の原因を探るためには関係者
の証言が欠かせないが，証言者自らに刑事責任が及ぶおそれがある
ときには有効な証言は得られにくい．特に我が国では過失責任が刑
事罰においても問われる場合があるので，ヒューマンエラーに起因
する事故について真の原因がわかりにくいとの指摘がなされている．
事故にいたったメカニズムの解明と再発防止を目標とする事故調
査において，事故にいたった人間行動を引き起こした背景や諸要因
を虚心坦懐に分析し，有効な教訓と対策を引き出すことが安全性向
上につながる道といえる．このため，人間が関与した事故の調査に
おいては，事故にいたるまでの人間行動の解明を行うことが不可欠
であり，当事者の責任追及を念頭におく司法捜査とは一一線を画す必
要がある。すなわち，このような事故に対しては，事実究明を目的
とし処罰を目的としない，という免責制度の考え方を導入すれば，
真相究明が容易となり類似事故の再発防止，安全性向上にとって貴
重な事実が明らかとなる．
ただし，定められた規則や法律を守らない，常識に照らし当然す
べき注意を怠った，などの意図した逸脱や故意の違反により事故を
引き起こした場合は，当然責任は問われなければならない．しかし
ながら，明らかに責任を問われる行動があった場合でも，特異な事
故や未解明の事故が当事者の証言により解明され，将来起こるかも
知れない大事故を未然に防ぐ効果がある場合には，免責制度の導入
は有用である．
現在の我が国の法体系においては，免責制度は存在しないが，事
故調査においては積極的に免責制度の導入を検討する価値があると
思われる．
b．免責制度の課題 免責制度の導入にあたっては，免責の基
準を明確にして広く合意を形成し，公正なる運用をすることが特に
重要である．
免責だけでなく，偽証に対するペナルティも考慮すべきではない
かという意見もある．また，事故調査にあたった調査員には，裁判
所あるいは国会などからの証人召喚を拒否できる身分的保証を与え
ることまで考える必要もある．さらに，民事訴訟では故意と過失を
区別せず，過失の場合も責任が追求される場合があるので，刑事免
責により知り得た事実が民事訴訟において使用される可能性がある
場合は，いくら刑事免責制度があっても当事者が真実を述べないこ
とが想定され，その対策も必要となる．

一方において，交通事故などの事故に見舞われた被害者は，いわ
ゆる加害者の処罰を望むのが一般的な自然な感情である．事故調査
の結果が公開され加害者の存在が明らかとなったとき，免責により
何の処罰もなされないとなると，被害者感情を著しく軽んじること
になる．また，公衆災害などの大規模事故においては，一般社会も
事故の起因者をはじめとする関係者の処罰を望むものである．これ
は社会一般のもつ自然な感情である．このような感情に対する社会
的な合意も必要となる．
以上述べた免責制度の課題については，事故調査の趣旨を踏まえ
て，国民的な議論をさらに深める必要がある．
