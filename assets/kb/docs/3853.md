# 7・6・2 計量器の修理、 法では,特定計量器の修理の事業を行

> **章**: 第7章 計量に関する法制度
> **節**: 7・6 計量器の修理

おうとする者に対して製造及び修理に届出を義務付けている.この届出は,電気計器にあっては経済産業大臣に,その他の特定計量器にあっては都道府県知事に対して行う.なお,届出製造事業者が修理の事業を行う場合には,改めて届出を行う必要はない.ここで修理とは,計量器がその性能,構造の一部を失った場合に,その失われた性能・構造を回復することをいうが,法では次の4種のものがある.1 軽微な修理(法第46条,施行規則第10条)誰でも行うことができる修理であり,修理事業の届出や検定証印等の除去の必要がないものである.2 簡易修理(法第49条第1項のただし書,施行規則第11条)届出製造事業者・届出修理事業者が行う修理又は適正計量管理事業所の指定を受けた者がその指定に係る事業所において使用する特定計量器について行う修理のうち,検定証印等の除去の必要のないものである.β9-176軽微な修理よりは修理の程度は大きい.3修理(法第46条第1項)法で「修理」というときは,前述の軽微な修理を除き,簡易修理を含んだ概念である.これは,修理事業の盧出の対象となる.4 型式承認の表示を除去しない修理(法第49条第1項のただし書,施行規則第12条)特定計量器の改造又は修理をした者は,検定認印等のほか「型式承認の表示」も除去しなければならないが,届出製造事業者,届出修理事業者が行う修理,また,適正計量管理事業所の指定を受けた者がその指定に係る事業所において使用する特定計量器について行う修理のうち,(検定証印等の除去は必要だが)型式承認の表示の除去は必要がないものがある.この修理を受けた特定計量器は,簡易修理についてはそのまま使用することができ,同一の型式の範囲内のものについては型式承認の表示が付されてあるものとして一部の性能及び器差の検定を受けて取引又は証明に使用することができる.検定証印等の有効期間のある特定計量器のうち,一定期間の経過後修理が必要なもの(施行令第12条)については,一定の基準(経済産業省令で定められる.施行規則第14条)に従って修理をしたときは,これに表示を付すことができることとされている(法第50条).この対象となる特定計量器は,型式承認の表示があっても,その付された日から一定の期間(経済産業省令で定める.検定検査規則第18条で「表示を付した日から次の年の末日まで」とされている.)を経過したものについては,この修理の表示が付され,かつ,その表示が付されてから上記の期間を経過していないものでなければ検定における構造の技術上の基準に適合するものとみなされないこととされている(二丁71条第2項括弧書き).これは,特定計量器の中には,長期間使用しても一部の部品以外に基本的な部分は劣化せず,検定証印の有効期間経過後も再検定を受けて使用することが可能であるものの,一定期間経過後は一部の部品等の劣化により,性能が落ちる可能性が高く,当面の検定には合格しても,その有効期間の間に器差が大きくなるおそれの高いものがあるため,(再)検定の前に修理を義務付けたものである.なお,この対象となる特定計量器は施行令第12条で,水道メータ,ガスメータ,温水メータ,積算熱量計,電気計器が指定されている.
