# 8・1・1 規格の定義 産業活動が活発になり、複雑化する中で

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・1 規格・技術基準

標準化が重要な役割を担ってきた。この標準化は、対象とする物事を統一化し、共通化し、単純化することを目的としているが、標準化した内容を関係者に伝達し、定着させなければならないので、文書化を行う。これが規格である。文書化したものが紙であれ、電子媒体であれ、それは問わない。ISO/IEC Guide 2(標準化及び関連活動一一般用語)では、規格を次のように定義しているω.「規格とは、与えられた状況において、適切な程度の秩序を達成することを目的に、諸活動又はその結果に関する規則、指針又は特性を、共通に、かつ、繰り返し使用するために定める文書であって、合意によって確立され、かつ、公認機関によって承認されたもの。」工業標準化法では、工業標準を、工業標準化のための基準と定義しているω.「工学の分野でいう規格とは、標準と同義であり、特に、工業製品の形状、寸法、公差、等級、=構造、品質、試験方法、検査方法などを関係者が協議して、一定の様式にまとめた文書情報をいう。」一方、規程は、ある目的のために定められた一連の条項の総体であるから、規格とは少し異なった概念として区別されている。例えば、企業内部の組織、人事・事務手続きの要領などであり、経営的な条項を定めた規則が規程である。特に、事務手続きや管理業務を定めたものは、規定という場合が多く、規程と区別している。工学の分野や産業界で適用するものは、工業標準又は工業規格である。国が制定権者の場合には、これを国家規格という。我が国の国家規格は、日本工業規格(JIS:Japanese Industrial Standard)である。
