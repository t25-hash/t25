a.燃料噴射装置（fuel injection system) ガソリン機関に使
われる燃料供給方式は気化器方式 と燃料噴射方式 に分類され
る．自動車用としては，排出ガス規制の強化に対応するため，空燃
比制御性に優れ，高出力化が可能な燃料噴射方式が主流となり，気
化器方式は，現在では二輪車やはん用機関のみに使われている．
燃料噴射方式では，各シリンダの吸気ポート近傍に取付けられた
燃料噴射弁［インジェクタ，injector ，図1･45(ａ)］から燃料が供
給されるが，冷間始動時には燃料は気化しにくく，吸気ポートや燃
焼室の壁面に付着する．このため，機関の燃焼が不安定となり一部
は未燃焼のまま機関から排出される ．したがって，燃料噴射弁
に要求される噴霧としては，吸気ポートやシリンダ壁面に付着しに
くく，機関が吸入する空気と混合しやすい特性をもつ必要がある．
このため，図1･45(ｂ)に示すように，燃料噴射弁の噴口の数を増
やしたり，エアアシスト方式のように噴口内で空気と燃料を混合
し，燃料の微粒化を促進するなどの工夫がなされている ．
b.吸入空気の流れ制御 図1･46(ａ)に示すような弁が機関の
吸気ポートの近くに取付けられ，吸気通路を絞ることにより吸入行
程で空気流速を高める方法であり，その機能によりスワール制御弁
（swirl control valve）またはタンブル制御弁（tumble control
valve）と呼ばれている ．この強められた空気の流れにより，燃
料噴射弁から噴射された燃料と空気の混合が促進され，シリンダ内
にも強力な流れと乱れが生成されるため，安定した燃焼が得られ，
燃焼限界空燃比を希薄側に拡大することができる．特に，吸入空気
量の少ない運転領域や触媒暖機を促進するために点火時期を遅らせ
たとき，大量にEGRを導入し燃焼が不安定になったときなどに有
効である．図1･46(ｂ)は，機関冷却水の温度が313Kのときの希
薄燃焼限界空燃比の拡大効果（トルク変動の改善）を示した例であ
る．機関が冷えている状態でも，安定な希薄燃焼運転が可能とな
り，機関から排出するHCやCOを低減することができる．
c .吸気弁片弁停止 吸入空気量の少ない運転領域でシリン
ダ内の吸入空気流速を高める方法として，二つある吸気弁の片方を
閉じ，他方の弁からのみ空気を吸入することにより強力なスワール
を生成させ，燃焼を安定させる方法も実用化されている．この方式
もスワール制御弁を使う方式も，機関の暖機後や高負荷時には通常
の作動に戻すため，出力性能への影響は小さい．

e.燃焼室形状など 燃焼室の壁面近くやピストンのトップ
ランドとシリンダ壁との狭い空間（クレビス）には消炎作用のため
火炎が伝ぱせず，燃料は未燃焼のまま機関から排出される．したが
って，燃焼室は凹凸が少なく表面積の小さいコンパクトな形状に，
クレビスはピストンリングの位置を最適化することで容積が最小と
なるように設計される．その他，機関からの排出ガス低減のために
表1･9に示すような技術が，必要に応じて組合わされ使われてい
る．
