# 8・2・1 標準化の意義 「標準化とは、実在の問題又は起こる

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・2 標準化の意義

可能性がある澗題に関して、与えられた状況において、最適な程度の秩序を得ることを目的として、共通に、かつ、繰り返して使用するための規定を確立する活動」とISO/IEC Guide 2で規定している。標準化を工業標準化に限定するならば、工業が発達段階にあるときは、規格の制定とその遵守に重きを置いた。しかし、現在のよう、に成熟した工業社会にあっては、部品の高精度化、品質の適正化、製品の低コスト化、企業のグローバル化、環境などを最大限に配慮した標準化戦略が必要となっている。時代は進んでも、標準化の意義は対象とする事象の統一化、単純化及び共通化である。これらは、標準化の三原則であり、低コスト化や環境への配慮も可能となる。1統一化とは、各種製品に使用される材料や部品、生産方式などを関係者と協議して、標準化する行為である。標準化の基本的、かつ、常套的手段である。統一というのは、ある一定の観点に立って、科学的に検討した結果、これがもっともよいと思われるところ(範囲)にすべてのものをもっていくことであるω、としている。2 単純化とは、設計方法、試験・検査方法、作業方法などにっいて、要求精度、設備、工場の特性、構成人員などを考慮したうえで標準化する行為である。もちろん、製品や部品の形状の単純化なども含まれる。「単純化は、市場に流通している商品について、形状、寸法、品質などによる種類のうち、需要の少ないものは製造、販売又は使用をやめて、需要の多いものを選り抜き、これに生産を集中することである。」と平成9年度版工業標準化法解説で例示している。3 共通化とは、部品の種類、材料、サイズ、マニュアル、梱包方法などを標準化する行為である。特に、この共通化は、統一化と混同して用いられることがある。共通仕様、共通材料などの用語とともに使用される場合が多い。統一仕様、統一材料などのほうが少し概念が大きい。
