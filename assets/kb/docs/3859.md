低減するための補助的手段として用いられる．
a.HC吸着システム（HC adsorber system) 触媒が
活性化する前に排出するHCを吸着材に一時的に吸着し，活性開
始後に吸着したHCを浄化する．図1･53に三元触媒とHC吸着材
を一体化したHC吸着システムの構成を示す．HC吸着材には，排
出ガス中のHC成分に適合した細孔をもつゼオライトが使われて
いる．吸着期間中は弁を閉じておき，吸着材側に排出ガスを流す．
吸着終了後には弁を開き触媒側に排出ガスを流して触媒を活性化さ
せる．この間にHC吸着材の温度も徐々に上昇し，HCは離脱を始
め，触媒に流入し浄化される．弁の切換えは，機関水温や吸入空気
量を基に吸気管負圧をアクチュエータに導くことで行われる．

c.電気加熱式触媒（EHC:electrically heated catalyst )
一般に，図1･55のように三元触媒の前に小容量の電気加熱式の触
媒を配置し，着火用触媒として使われる．機関の始動前あるいは始
動直後に蓄電池から電力を供給して加熱するため，数秒後には触媒
反応が開始する．後段の三元触媒が反応を開始した後に通電を停止
する．電気加熱式触媒は耐熱ステンレスはくなどの材料で作られ，
反応が早められるように触媒が担持されている．加熱部分の容量は
加熱時間の短縮と消費電力低減のため，100cm程度と比較的小さ
い．構成は空気ポンプと組合わせ，未燃成分の多い排出ガス中に空
気を供給し，酸化反応熱により触媒の温度上昇を早め，消費電力を
低減する着火触媒的な使われ方が一般的である．
d.二次空気供給システム（secondary air supply system)
従来はHC，COを酸化させるために使われていたが ，機関の制
御や触媒技術の進歩とともにほとんど使われなくなった．しかし，
1990年代後半から排出ガス規制が厳しくなると，触媒が活性化す
るまでの間に排出されるHCのいっそうの低減が必要となり，触
媒を早期に活性化させる手段として再び注目されるようになってき
た ．図1･56のように，二次空気供給系は空気ポンプと制御弁
で構成され，各シリンダの排気弁近傍に空気を供給する．機関には
理論空燃比よりも濃い混合気を供給し，排出されてきた高温の未燃
焼ガスと反応させる．反応によりガス温度が上昇するとともに，触
媒に流入するガスは酸化雰囲気となるため，触媒の活性が早まる．
二次空気供給系はおもに冷間運転時のみ使われるため，空気ポンプ
は必要なときに駆動できる電気駆動方式のポンプが用いられてい
る．

c .触媒と排出ガスセンサの配置 図1･59に示すような触媒と
排出ガスセンサの配置が，排出ガス規制値の厳しさや1･4･10項で
述べる触媒劣化検出等の要求により使い分けられている．

を推定しながら，高い浄化率が得られるように供給空燃比を制御す
る．
