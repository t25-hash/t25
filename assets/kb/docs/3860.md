機関の制御技術はマイクロコンピュータの進歩により，運転状態
に応じて空燃比，点火時期，アイドリング回転速度，EGR，吸排
気弁の開閉時期などが適切に制御できるようになり，高い排出ガス
浄化性能を得ながら運転性と燃料経済性を両立させることができる
ようになってきた．
1･4･5項で述べたように，現在のガソリン自動車の排気浄化装置
は三元触媒を使い，排出ガス中の三成分（HC，CO，NO ）を同
時に浄化する方式が主流であり，触媒の機能を十分に発揮させるた
めに電子制御技術が駆使されている．図1･57に示すように，機関
に吸入される空気量は空気流量計で計量され，それに基づいて機関
に供給される燃料量が計算され，燃料噴射弁から供給される．
しかし図1･50に示すように，排気三成分を同時に浄化できる空
燃比の範囲は非常に狭いため，この制御のみでは高い浄化率を得る
ことは難しい．このため，排気マニホルドには排出ガス中の酸素濃
度や空燃比を検出するセンサが取付けられ，最大の浄化性能が得ら
れるように燃料量がフィードバック制御される．
a.吸入空気量の計量 機関が吸入する空気量の計量方法
には，吸気管圧力と機関回転速度から計算する（speed density）
方式と空気流量計（air flow meter ）で直接計量するマスフロー
（mass flow）方式とがある．
マスフロー方式には，熱式，ベーン式，カルマン渦式があるが，
現在では熱式が質量流量に対応した出力を直接得ることができ，空
気密度による補正が不要で，可動部分がなく，圧力損失が小さいこ
とからおもに使われている．
b.酸素（O）センサ（oxygen sensor ），空燃比（ ）センサ
（air - fuel ratio sensor ) 図1･58に酸素センサの構造を示
す ．ジルコニア素子（ZrO ）には，高温状態で素子の内外面
に酸素濃度差があると起電力が発生する性質がある．図1･58(ｂ)
図1･59(ｂ)の触媒を挟んでセンサを置く方式では，触媒後の酸
素センサ信号により触媒の状態を検出できるため，図1･59(ａ)の
方式に比べて正確な空燃比制御が可能になる．
最近では，機関の吸入空気量や触媒の酸素吸蔵量をモデル化する
ことで，さらに制御精度を高める技術が開発されている ．
