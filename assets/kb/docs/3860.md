# 8・2・2 標準化の便益 工業標準化は,利便性や経済効果を求

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・2 標準化の意義

めるために適用実施される.特に,大量生産の場で標準化の効果は大きい.これは,種類削減効果や工程管理などあらゆる生産活動の中での標準化効果をねらいとしている.!しかし,生産活動を行うに当たっては,投資コスト,開発コスト,実施コスト,運用コスト,消費者コストなどを総合評価して,標準化の利益を算出し,企業活動の間接費用として経常利益を上げるように行われているのが実情である.一方,国際標準化や国家標準化は,大局的な観点から,工業国,発展途上国の双方の標準化利益が享受できることを考慮しつつも,公共の利益を優先させた標準化が鋭意行われている.標準化の過程においては,工業国での適用実施例が有益な情報を提供している.ある規格から得る便益(bene翫s)の量の大きさは,その規格が影響を及ぼす活動量,言い換えれば,規格の影響を何らかの形で受けるところのすべての費用,行動,努力などを貨幣に換算した価値に直接に左右される.便益価値のこの活動量に対する比は,残りの動因の組合せである(有)。
