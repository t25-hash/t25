かれ，吸気管負圧を使ってキャニスタ内に空気を流すことでHC
を脱離し，吸着能力を回復させる．

ると同時に機関にも吸入されるが，キャニスタには，続く駐車時や
給油時に発生する燃料蒸気をすべて吸着できる能力が残されている
必要があり，走行中に十分な脱離量が得られるように設計される．

b.燃料蒸発ガス規制（evaporative emission regulation）と評
価方法 燃料蒸発ガスの規制と評価方法（evaporative emission
test procedure）について，アメリカを例に解説する ．蒸発ガス
の規制には，駐車中（DBL:diurnal breathing loss ）と停車中
（hot soak loss ）の規制，走行中（running loss ）の規制，給油時
（ORVR）の規制がある．いずれも密閉された部屋（SHED:
sealed housing for evaporative determination）で評価される．
これらの評価は決められた手順に従って連続して行われるため，キ
ャニスタが燃料蒸気を吸着する能力と，機関がその蒸気を吸入でき
る能力の両方が評価される．走行中と給油時の規制はアメリカでの
み実施されている．
.駐車中の燃料蒸発ガス評価 図1･66に示すように，夏
季の屋外に3日間駐車されたことを想定した試験法である．密閉さ
れた部屋に温度制御装置を設け，1日の気温の変化を模擬し，3日
間の評価を行う．規制に適合しているか否かは，車両から放出され
るHCの1日当たりの最大量で判定される．日本やヨーロッパも
ほぼ同じ評価方法を実施しているが，1日の駐車を想定している．
.走行中の燃料蒸発ガス評価 夏季の日中を想定した走行
を模擬するために，密閉された部屋にシャシダイナモメータを設置
し（図1･67），評価モードを走行する．車両の走行とともに燃料タ
ンク内の温度は上昇するため，蒸発ガスの量は増加するが，評価期
間内に規定量以上のHCが大気中に放出してはならない．この試
験では，走行中に機関が吸入できるHC量の処理能力が評価され
る．
.給油時の燃料蒸発ガス評価 ガソリンタンドと同じ設備
を備えた給油専用の部屋（ORVR SHED）で給油を行い，給油口
とキャニスタから漏れ出るHCの量と給油性能が評価される．
