# 1・5・1 内燃機関の出力特性と自動車の要求特性

> **章**: 第1章 火花点火機関
> **節**: 1・5 動力伝達系

動力源である内燃機関の出力特性はそのままでは自動車に不適切
であり，自動車の要求特性に適合させるためには以下に示す三つの
機構を有する変速機（transmission）が必要である．
a.動力遮断機構 内燃機関は，アイドリング回転速度以下で
は回転を維持することができないので，自動車の停止時には内燃機
関と駆動輪の間で動力を切り離す遮断機構が必要となる．またこの
遮断機構は発進時に徐々に締結することにより，内燃機関の回転速
度を維持しながら駆動輪に動力を伝える発進機能も有する．
b.変速機構 自動車は坂道登坂や加速時には大きな駆動力を
必要とし，高速で走行するときは高い駆動輪の回転速度が必要であ
る．さまざまな状況に応じて運転を行うためには，一般的に図1･
69に示すようにいくつかの変速比（gear ratio）を選択できる変速
機構が必要となる．
