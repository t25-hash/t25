手動変速機は上記3機構を備え，運転者の操作により作動する．
a.動力遮断機構 手動変速機の遮断機構は，内燃機関と変速
クラッチ（clutch）は，フライホイール，クラッチカバー，クラ
b.変速機構 手動変速機の変速は，車両走行状態に応じた適
ている．乗用車の例をとると，前進4～6段の歯車列と後進1段の
歯車列で構成されるものが主流である．さらに，より大きな駆動力
が必要な車両においては，極低速段や副変速機構が追加された例が
ある．
前進段の場合，それらの変速段を形成する歯車どうしは常時かみ
あっており，運転者がその歯車段を選択したとき，その歯車段に設
置されている同期装置の作用により回転速度が同期させられること
で，円滑な変速操作が可能となる．通常，黄銅製の同期リングと鉄
製の同期コーンをもった同期装置が主流で，それぞれの被同期歯車
に1個設置されているが，操作性および耐久性の向上を目的とし
て，その摩擦面を複数有するマルチコーンタイプの同期装置や，摩
擦面をカーボン材などにして高容量化と耐久性を両立させたものも
ある．図1･72に同期装置の構造例を示す．
c .前後進切換機構 手動変速機の前後進切換機構は，常時歯
車がかみあっている前進歯車とは別に専用の歯車列をもち，後進時
に後進専用の歯車どうしがかみあう選択しゅう動方式と，常時後進
用歯車どうしがかみあって，かつ同期装置を備えた常時かみあい方
式がある．図1･73に常時かみあい式後進歯車の一例を示す．
