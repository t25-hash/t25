# 1・5・5 燃費性能と動力伝達系

> **章**: 第1章 火花点火機関
> **節**: 1・5 動力伝達系

自動車の燃費（燃料消費率）は，内燃機関と動力伝達系の適合性
改善とおのおのの単体効率の改善により向上してきた．
a.エンジンの単体効率 内燃機関の単体効率の一例として，
図1･86の正味燃料消費率［BSFC:brake specific fuel consump-
tion，g/(kW･ h）］を示す．自動車用の内燃機関では，中回転速度
の高負荷領域でBSFCが少なく効率が良い．
b.動力伝達系の効率 内燃機関で発生した動力を少ない損失
で駆動輪に伝達するには，動力伝達系の効率の良いことが要求され
る．
変速機，動力分配装置，終減速機等の動力伝達装置の損失は，軸
受のしゅう動部摩擦，油のかくはん，油ポンプの駆動消費，トルク
コンバータのスリップ，歯車のかみあいやベルト駆動などが挙げら
れる．図1･87のATのトルクコンバータを除いた内部損失の分析
例では，油ポンプの駆動損失の占める割合が大きいことがわかる．
変速機の伝達効率（eﬃciency of power transmission）は，自動
車の定地走行車速に応じた部分負荷効率と，変速機単体で入力トル
クを変数に出力トルクとの割合を示す全負荷効率の二つがおもに用
いられる．図1･88に変速機の方式による，部分負荷での伝達効率
の違いを示す．

c .自動車の燃費 内燃機関の動力の一部は，内燃機関の内部
摩擦損失，動力伝達系の損失，車両の転がり抵抗・空気抵抗・加速
抵抗などに消費される．
自動車の燃費（燃料消費率）は，定地燃費や運行燃費，排出ガス
規制のために定められた走行モードなどから求められ，いずれも単
位燃料消費量当たりの走行距離数で表される．自動車の燃費性能を
表す指標として10・15モード走行燃費が一般的に用いられている．
燃費向上技術としては，変速機の内部損失低減や機関本体の正味
燃料消費率の低い領域での運転を実現するための変速比，自動変速
線図の最適化がある．
無段階な変速比をもつCVTは，MTやATに比べ伝達効率は
劣っているものの，燃費性能は同等以上を実現している．図1･86
に示すように，定速走行時にはCVTのほうがMTに対して低回
転高トルク側で内燃機関の高効率な領域を使っている．また，図
1･89に示す15モード走行時の各種比較でわかるように，CVTで
は変速比の連続可変特性を使って内燃機関の回転速度を下げずに走
行が維持できるので，減速時燃料を噴射しない（燃料カット，fuel
cut ）領域が拡大されている．
燃費向上技術として，自動変速機では，図1･90に示すように直
結クラッチによる締結領域の拡大が図られ，トルクコンバータのス
リップ損失の影響が低減されてきている．最近では，アイドル時の
燃料消費量を低減するアイドルストップ機構や，Dレンジでのニュ
ートラル制御技術およびハイブリッド車などの開発が進んでいる．
また，駆動構成要素が増加すればその伝達効率は低下し，燃費が
悪化する．このため駆動構成要素が少なく効率の良いFF駆動方式
が一般大衆車から中型車まで幅広く採用されている．また，4WD
車に関しても駆動効率損失を極力抑えるために，通常は2WDで
必要なときに車輪の回転数差などを感知し，自動的に4WDとな
る可変配分式が増加している．さらに最近では，この駆動力配分を
電子制御によりさらにきめ細かく可変制御させ，燃費向上を達成す
る機構が採用され始めている．図1･91に一般的な駆動方式と，そ
の駆動力配分量における効率の概念を示す．
