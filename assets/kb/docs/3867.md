# 8・4・3 日本工業規格(JIS)の標準化の概要 工業標準化事

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・4 日本の工業標準化

業は、経済産業省産業技術環境局標準課が計画立案し、工業標準化法に定められた主務大臣が制定権者である。我が国の工業標準化事業に関する組織図を図8・8に示す(17)。標準化作業自体は、国からの委託を受けて、関係団体(公益法人、産業団体、学協会等)が行う場合、工業標準化法第12条に基づいて、利害関係人が原案を添えて主務大臣に申し出る場合、まれに日本工業標準調査会事務局が作成する場合とがある。利害関係人が原案を提案するのは、新しく導入されたもので、自主提案ともいわれている。JIS原案の審議は、日本工業標準調査会が行っており(工業標準化法第11条)、一般的には標準化項目ごとに原案作成委員会、専門委員会の審議を経て日本工業標準調査会が主務大臣に答申する。』JIS規格ではないが、 JIS技術報告書(TR)が1998年から発行できるようになった。ISO/TRと同様に、規格の前段階のものとして、あるいは規格化の審議が未完了のものなどが発行の対象となる。3年を経過する時点で継続するか、廃止するかの審議が当該専門技術委員会で行われる。機械工学の分野で有用なTRがいくつか発行されている。例えば、次の2規格である。TRBOOO3:1998;製図一幾何公差表示方式一形状、姿勢、位置及び振れの公差方式一検証の原理と方法の指針TR B OOO7:1998;製品の幾何特性仕様(GPS)一マスタープランJISの制定・改正・確認・廃止の権限は、主務大臣にある。主務大臣が適正であると認めた答申案を決裁して、それを官報公示する(工業標準化法第16条)。
