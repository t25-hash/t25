.高圧縮比化
(１)高圧縮比化の効果
過去15年間の国内市販車の圧縮比（compression ratio）変遷を
図1･97に示す．おおむね10年で0.5上昇する傾向がある．高圧縮
比化は熱効率を向上するための方策であり，言い換えれば後述する
ノック発生に対する抑制策が少しずつ向上していることを意味す
る．
圧縮比9.0における熱効率を基準とした正味熱効率の向上率は，
行程容積が小さいほど，負荷が低いほど高圧縮比化の効果が小さ
い．この差は燃焼室の / 値（上死点時の燃焼室表面積/容積）
の違いによる冷却損失と燃焼効率の差によるものと考えられ
る ．一般に圧縮比が高く行程容積が小さいほど / 値が大き
くなり，冷却損失は増大する．また，燃焼室壁面付近の消炎層内に
ある燃料が未燃となるため，燃焼効率は悪化する．
運転条件の影響としては，負荷が小さいと供給燃料のもつ熱量に
占める冷却損失の割合が大きいため，高圧縮比化による熱効率の向
上効果が小さく，回転速度が低いほど冷却損失の割合が大きいた
め，高圧縮比化の効果は小さくなる．
高圧縮比化できるかは，ノックを抑制できるかどうかにかかって
いる．
次に，ノック現象とその発生機構について説明する．
(２)ノック（knocking）
ノックは，火炎伝ぱ（flame propagation）途中における末端部
混合気（エンドガス，end gas ）の自発着火（auto-ignition）に起
因し，このとき生じる圧力波が燃焼室壁を加振して，ノック音が発
生する．
点火プラグから広がる火炎面は，既燃部の膨張により未燃部を圧
縮しつつ進行するため，未燃部の温度，圧力は上昇し，これらの温
度，圧力の時間履歴が燃料によって決まるある限界（過酸化物濃度
の限界）を超えると自発着火に至る．したがって，ノックの強さは
自発着火する未燃部のガス量による．言い換えれば，燃焼最終領域
に該当する容積は小さいほうがよい．
(３)高圧縮比化の方法（ノックの抑制法）
圧縮比を高めるためには燃料のオクタン価（octane number ）を
高めるか，燃焼期間の短縮，燃焼室の冷却改善などにより機関の耐
ノック性を向上し，要求オクタン価（全負荷時の最適点火時期にお
いて，ノックの発生しない運転を可能にする燃料のオクタン価）を
低減させる必要がある．
いろいろな種類の燃焼室形状について圧縮比9.0前後から高圧縮
比化する場合を見てみると，要求オクタン価は圧縮比1.0当たり
5～7オクタン価（リサーチオクタン価）増大する ．このため，
燃料のオクタン価以外の燃料特性をあまり変えずにオクタン価を向
上できれば，機関側の変更は比較的少なく熱効率向上が可能とな

性を向上させるためには，多点点火や吸気ポート内の吸気通路を絞

各気筒の点火プラグに配電する機械式に対して，電子配電システム
は気筒ごとに点火信号を制御ユニットで制御し，各気筒の点火コイ
ル（パワートランジスタ内蔵）に配電する（図1･137）．点火コイ
ルは点火プラグに直接取付けられており，また電子的に配電される
ので，高い二次電圧を確保し，高回転速度域まで安定して点火エネ
ルギーを供給することができる．
(２) 点火時期制御部
点火時期制御部は，一次コイルに通電を開始するクランク角と一
次コイルの電流を遮断するクランク角とを制御して，通電時間と点
火時期を制御する．通電時間は，点火に必要なエネルギーが蓄えら
れる最小の時間に設定される．点火時期制御には，基本点火時期制
御とノック制御がある．
(ａ)基本点火時期制御 ガソリン機関においては，出力，燃
費が最良となる点火時期（MBT:minimum advance for the best
torque）が存在する．吸入空気量の増大とともに燃焼速度は速く
なるため，MBTは遅角する．また，回転速度の上昇によりクラン
ク角度に対する時間が短くなるので，MBTは進角する．これらの
要求を満たすため，基本パルス幅 と機関回転速度 を軸とす
るデータマップ（図1･138）に基本点火時期を設定し，マップを検
索して得られた値に冷却水温などに対応する補正を加えて点火時期
としている．
(ｂ)ノック制御 点火時期はMBTに設定されるべきであ
るが，高負荷域ではノックが発生してMBTまで進角できない場
合がある．ノックの発生は湿度や燃料性状（オクタン価）などによ
り大きく変わるため，点火時期は余裕をもって遅角側に設定するこ
とになる．したがって，点火時期をMBTに向けてなるべく進角
しておき，ノックの発生を検出したら遅角させてノックを回避する
ノック制御は，機関のもつ能力を最大に引出すため，また，過給機
関や高圧縮比自然吸気機関のようにノックが発生しやすい機関の損
傷を防止するために有効である．
ノックを検出するノックセンサ（図1･139）は，エンジンブロッ
クにボルト締めで取付けられ，燃焼ガスの圧力振動をブロック表面
の振動として検出する．ノックによる圧力振動は，燃焼ガスの音速
とシリンダ内径で決まる特定周波数の燃焼室内の共鳴振動として発
生するので，特定周波数成分の強度を検出することにより，ノック
が検出できる．吸気弁や排気弁の着座振動などのノイズを除去する
ために，信号処理方法が工夫されている．
