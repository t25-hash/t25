# 8・5・3 日本船舶標準協会規格1日本船舶標準協会は,船舶に

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・5 日本の団体規格・技術基準と法制度

関するJIS化, ISO, IECの国内対策を行っており,標準化の対象は,船舶一般,船体,機関,電気並びに航海用機器・機器などである.船舶関係JISは,国内外の船舶安全法関係法令に規定する技術規準を補完する形で適用される.また,省庁の調達関係における技術規準にJISが引用されると,強制力が伴う.しかし,標準化の根底に流れる思想は,品質の確保,互換性の確保,コストの低減,取引の合理化などである.グローバル化する造船企業活動においては,ISO及びIECの標準化は国内に大きな影響を与えるので,これら国際規格の国内対策は重要度が年々増加している.日本船舶標準協会では,ISO/TC 8(造船及び海洋構造物),ISO/TC 188(小型の船)及びIEC 18(船用電気設備)の国内対策を鋭意行っている.中でも近年の比重は,ISO/TC 8の幹事国業務に注がれている.備考 IEC/TC 80(電子式航行装置)は,電子情報技術産業協会が対応している.EU経済統合, WTO/TBT協定の成立など,国際的な標準化の環境変化は大きく,船舶設計,製作,流通など,国際化を迫られている.グローバル化する造船業界にあっては,FRP船リユース等環境保護,資源リサイクル分野,旅客船におけるバリアフリー化等ハンディーキャップ関連分野,新たな分野における標準化の課題20に対応している.
