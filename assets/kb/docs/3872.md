# 1・7・1 はん用機関

> **章**: 第1章 火花点火機関
> **節**: 1・7 はん用機関等

火花点火機関の中でも自動車や二輪車のように道路を走る輸送用
機械ではなく，定置または移動しながら使われる多種多様な機械の
動力源として利用されているものははん用機関（engine for gen-
eral use ）と呼ばれており，表1･15に示すような農業・林業，土
木・建設，電気・産業やレジャー・家庭用などの機械に広く搭載さ
れている．はん用機関の中で最も多く生産されているのが，出力
19kW以下の小型はん用ガソリン機関であり，わが国では2001年
に2ストローク，4ストローク合わせて約600万台が生産され，
80%以上が直接あるいは機械に搭載され世界中に輸出されている．
小型はん用ガソリン機関は各種作業機械への搭載性と廉価なことか
ら，軽量コンパクトな空冷単気筒およびV形二気筒が主流となっ
ており，用途に応じて2ストローク機関と4ストローク機関が選択
されているが，近年では環境問題への対応のため，これまではほと
んど2ストローク機関が搭載されていた手持ち式の機械においても
4ストローク機関が採用されるようになってきた．小型はん用ガソ
リン機関の使用条件と使用環境は用途や地域によってさまざまであ
り，家庭用として年間で数時間しか使われないものから，業務用と
してほとんど毎日，長時間連続して使われるもの，負荷条件におい
ても軽負荷から高負荷まで一定または大きく負荷が変動するものが
ある．使用環境の面では258K以下の酷寒から313K以上の酷暑
まで，さらにほこりや泥の多い場所や標高1500m以上の高地な
ど，まさにはん用機関は世界中のあらゆる所でさまざまな機械の動
力源として使用されている．

c.構造上の特徴 小型はん用4ストロークガソリン機関の構
造上の特徴を以下に説明する．
.主機部分 クランクケースとシリンダは，アルミダイカ
ストの一体成形で特殊鋳鉄製のシリンダライナを鋳込み，ピストン
しゅう動部の耐摩耗性向上を図る構造が一般的に採用されている．
シリンダの外周には冷却フィンを付けて冷却を行うが，OHV式や
OHC式のシリンダ部はSV式に比べて吸排気のバルブやポートが
ないため構造が簡単で冷却性に優れ，シリンダの熱変形が小さいこ
とから，燃焼室への潤滑油上がりを大幅に低減できる．したがっ
て，潤滑油消費量の低減と出力や排出ガス性能の劣化が少ないとい
った長所がある．クランクケースは燃焼による爆発力に加え，搭載
機械から加わる外力にも十分に耐えられるだけの剛性をもたせる必
要があるが，一方では軽量化の追求も必要であり，リブを効果的に
配置してその両立を図っている．ピストンとコネクティングロッド
はともにアルミニウム製で軽量化を図り，ピストンは低圧鋳造品に
熱処理を施してアルミニウムの機械的性質を向上させ，コネクティ
ングロッドは高圧のアルミダイカストで油スクレーパを一体成形す
るものが大多数を占めているが，一部には鍛造品も採用されてい
る．一般的なアルミダイカスト製法では鋳造時に巻き込まれた空気
が鋳巣となって製品内部に残り，熱処理を施すことができないとい
う問題があったが，ダイカスト技術の進歩により，鋳巣がなく熱処
理が可能な製法が開発されたため，大排気量の機関においても鍛造
からダイカスト化が進んでいる．コネクティングロッドの大端部は
本体とキャップに二分割され合わせ加工により仕上げられている．
その際，クランクピンへの組立時にズレが生じないように本体とキ
ャップの合わせ面にインロー加工を施すことが多い．クランク軸は
炭素鋼の鍛造またはダクタイル鋳鉄を使った鋳造の一体構造で，一
端を搭載機械への動力取出しに利用しているため，搭載機械の要求
に応じて何種類もの出力軸形状が準備されている．主軸受にはボー
ルベアリングを採用して爆発荷重のみならず，搭載機械を駆動する
際に発生する曲げ荷重やスラスト荷重に耐えられる強度・剛性をも
たせている．クランクピン部はアルミニウム製コネクティングロッ
ドの直軸受方式を採用するため，高周波焼入れが施されている．単
気筒機関ではピストンなどの往復質量が発生する慣性力を釣合わせ
ることはできないが，クランクピンと反対側に釣合いおもりを付け
て上下方向の慣性力の一部を左右方向へ置換え，機関の振動を低減
している．比較的排気量の大きい機関ではダイナミックバランサと
称して不釣合いおもりをもったバランサ軸をクランク軸と逆回転さ
せることにより，一次の慣性力を釣合わせ，機関の振動低減を図っ
ている．カム軸は鋳鉄製の鋳物でカム歯車を一体化し，カムローブ
部には耐摩耗性を向上させるために高周波焼入れを施したものが一
般的に採用されているが，騒音低減や軽量化などを目的に樹脂製の
カム軸も一部の小排気量機関には採用されている．軸受はクランク
室のアルミニウム部分をそのまま利用する直軸受方式である．シリ
ンダヘッドはアルミダイカスト製でOHV式やOHC式では吸排気
弁とポートおよび点火プラグを配置するため，SV式に比べて構造
が複雑である．燃焼室形状はバスタブ型，多球型やペントルーフ型
が採用されているが，設計に当たっては弁配置や冷却性のほかに，
燃料タンク，マフラ，空気清浄器などの補機部品レイアウト，点火
プラグの保守性なども考慮する．シリンダヘッドの設計は，出力・
排出ガス性能に加えて耐久性にも重大な影響を及ぼすため，十分に
検討することが必要である．
.始動装置（starting system) 始動にはロープ始動方式
が一般的に用いられている．始動ロープを引くことにより始動装置
の内部にあるラチェット式の爪が飛び出し，はずみ車に取付けた起
動プーリを介してクランク軸を回転させる．機関の初爆に必要な点
火エネルギーを発生させるためには，クランク軸の回転速度をおよ
そ300～350rpm以上にする必要があり，始動時の圧縮乗り越しを
軽くしてクランキング速度を高めるため，自動圧縮抜き装置（オー
トデコンプ）をカム軸に備えることが多い．始動ロープを放せばぜ
んまいばねの戻し力により，引出したロープは始動装置内部のリー
ルに巻き戻される．排気量が大きい機関や用途によってはセルモー
タによる始動方式も採用されている．
.点火装置 点火装置はフライホイールマグネト
（flywheel magneto）方式で，ほとんどの機関が無接点のトランジ
スタ点火方式を採用し，保守点検を不要としている．はずみ車に張
り付けた永久磁石により点火コイルの一次側を発電するこの方式
は，ロープによる始動方式と組合わせることにより電池を不要と
し，小型はん用ガソリン機関を軽量コンパクトなものにしている．
点火コイルははずみ車外側のシリンダ部に直接取付ける方式が一般
的に採用されているが，点火コイルには一次側と高電圧を発生する
二次側のコイルに加えトランジスタが内蔵されているため，353K
以上の高温にならないように配慮する必要がある．点火時期は出力
性能やノックなどの運転調子，排出ガス性能などを考慮して決定す
るが，上死点前0.1～0.14πradの範囲で固定した点火時期を採用
する機関が多い．また，始動回転域での点火時期を上死点付近まで
遅角させて始動時のキックバックを防止するものや，低中速域での
ノッキングと高速域での出力性能を両立させるために，機関回転数
におおむね比例する電子式進角方式を採用することもある．小型は
ん用ガソリン機関は用途や使用者によって運転状態がまちまちであ
り，点火プラグの熱価の選定には注意が必要である．特に，アイド
リングや軽負荷での運転を多用する場合には，焼け型の点火プラグ
に交換して中心電極温度を自己清浄温度以上に保ち，電極の汚損を
防止しなければならない．
.燃料供給装置 小型はん用ガソリン機関は図1･182～1･
184に示すように，機関の最上部に燃料タンクを配置し，気化器
（carburetor ）への燃料供給は重力による自然供給方式を採用して
いる．燃料タンクを機関に直接取付けるため，タンク容量は定格出
力負荷で約2時間の無補給運転を目安に設定することが一般的であ
るが，長時間の無補給運転を必要とする発電機では，大型の燃料タ
ンクを発電機フレームに搭載したり，車両などに搭載される場合に
は，別置きの燃料タンクから燃料ポンプを使って燃料を供給する方
式も採用される．燃料タンクを直接機関に取付けることから，機関
からの伝熱やふく射熱によりタンク内のガソリンが温まらないよう
にすることや，機関振動に耐えうる強度とすることが必要である．
また，搭載機械によっては傾斜状態で運転されることもあり，注油
口の位置を高くしてふたから燃料が漏れるのを防止したり，Y字
配管と称して燃料タンクの両側から燃料を取出すものもある．気化
器はフロート式で固定ベンチュリをもつサイドドラフトタイプが一
般的に採用されており，チョーク弁による始動系統以外は，主系統
と低速系統のみをもつシンプルな構造ですべての運転条件に適応し
ている．低燃費化や排ガスの浄化のためには空燃費（ /）を大
きくする必要があるが，始動・加速やハンチングなどの運転性能や
シリンダヘッドの冷却を考慮して適合を行う必要がある．気化器は
機関からの伝熱を防止するために熱きり用のインシュレータを介し
て取付けられている．燃料系にまつわる問題として燃料供給通路内
で発生するベーパロック，フロートチャンバ内のガソリンが沸騰す
るパーコレーションおよび絞り弁などが結氷するアイシングがあ
り，機関の搭載状態や運転状況により対策が必要となる．また，田
植機のような農業機械では使用する期間が短く長期間保存されるた
め，気化器のフロートチャンバにガソリン抜き用のドレンを追加
し，腐食ガソリンによるジェットの詰まりを防止するほか，内部に
特別な防錆処理を施すこともある．
.調速装置（governor ) 小型はん用ガソリン機関では，
負荷変動に対して機関の回転速度を一定に保つ調速装置が付いてい
るのが一般的である．クランク軸と一定の回転比で回転するガバナ
歯車やカム歯車などにガバナウエイトと称するおもりを付け，おも
りの発生する遠心力と操作リンク系に取付けたガバナスプリングと
の力の釣合いにより，気化器の絞り弁開度を自動的に調整する方式
が機械式調速装置である．ガバナ特性は，定格出力で運転している
状態から急激に負荷を抜いた場合の瞬間的な回転上昇の割合を示す
瞬時変動率と，安定時の回転速度との割合を示す整定変動率，回転
が安定するまでの時間を示す整定時間で表すことができる．整定変
動率は無負荷高速回転速度と定格負荷運転時の回転速度の変化量で
あり，一般的には5～7%程度としている．なお，調速装置は機関
の過回転防止にも役立っている．
.潤滑装置 潤滑はコネクティングロッドの大端部にある
油スクレーパによる引っかけ飛まつ方式が一般的であるが，一部の
単気筒やV形二気筒機関では油ポンプによる強制圧送式が採用さ
れている．油スクレーパによりはねかけられた潤滑油が霧状になっ
てクランク室内に充満することにより，シリンダ，ピストン，クラ
ンク軸やカム軸などのしゅう動部を潤滑する．OHV式やOHC式
機関ではロッカアームなどの動弁系の部品がシリンダヘッドのロッ
カ室にあるため，ブローバイガスを空気清浄器に排出するブリーザ
機構をロッカ室に設け，ブローバイガスとともに霧状の潤滑油をロ
ッカ室へ導いて潤滑する．潤滑油はしゅう動部の焼付きを防止する
ほか，機関を冷却する役割も果たしているが，飛まつ量が多すぎる
場合にはブリーザからの潤滑油の吐出しに注意する必要がある．特
に単気筒はクランクケース内圧力の変動が大きいため，ブリーザ機
構の遮油構造に工夫が必要である．また，クランク室内に潤滑油を
ためる構造のため十分な潤滑油量を確保することができず，使用者
の点検不足による焼付きを未然に防止するため，規定油量以下にな
ると機関を自動的に停止させる油面センサの装着率が増えている．
.冷却装置（cooling system) 冷却ははずみ車と一体成
形したシロッコファンによる強制空冷式が一般的に採用されてい
る．ファンカバーとシリンダバッフルにより，シリンダヘッドやシ
リンダの高温部へ冷却風を導き効率的に冷却する．冷却不足は機関
性能の低下のほか，ヘッドガスケットからのガス漏れやしゅう動部
の焼付きなど機関そのものの耐久性に影響するため避けねばなら
ず，馬力当たり1mの風量を目安に設計する．冷却風は始動装置
にあけられた窓から取り入れるが，搭載機械によっては機関が筐体
などで囲われることがあり，冷却風の取入れ口の確保や温められた
冷却排風が筐体内に滞留しないように工夫する必要がある．
