# 8・5・6 日本電機工業会規格・技術基準 電機産業は、1936

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・5 日本の団体規格・技術基準と法制度

年に関係者による定例会合を開催し、1940年に業界団体として日本電機i製造協会を設立した。1946年には、日本電気機械製造協会となり、1948年に至って日本電機工業会(J:EMA:Japanese Electricalmanufacturers’Association)となった。現在のように社団法人として認可されたのは、1954年のことである。JEMAの標準類は、次の3種類がある(23)。a.日本電機工業会規格(JEM) 電気機器に関わる設計、製造、試験及び使用に係わる事項についての規格b.日本電機工業会将来規格(迂EM-F) 電気機器に関わる設計、製造、試験:及び使用に係る事項のうち、将来に対し推奨する事項についての規格。この将来規格は、可及的速やかに審議のうえ、日本電機工業会規格に切り換えられる。c.日本電機工業会技術資料(JEM-TR) 規格(JEM又はJEM-F)に該当しない保守・点検指針、適用指針、比較対照表などの技術的推奨事項及び参考事項についてまとめた資料。 .
