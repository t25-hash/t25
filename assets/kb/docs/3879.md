# 8・6・6 ロシア規格(GOST) 1923年に中央標準局を創設し、

> **章**: 第8章 規格・技術基準と法制度
> **節**: 8・6 主要工業国規格

国内の標準化の実情調査及び中央標準化機構の設置を準備する業務を開始して、1925年に国防会議附属標準化全ソ委員会が設置された。そして、1954年には、ソ連邦閣僚会議に規格・計測器委員会が設置され、更には1970年にこの委員会を改組して、現在のGOSTとなった。GOST規格の最大の特色は、すべて強制規格である。そのため、規格違反は、法律によって追及され、国家規格の指定期日を守らず、不良製品を製造する責任は、ロシア又は連邦共和国の法律の適用を受けることになる(32)。ロシアの標準化の最高責任を負う機関は、ロシア閣僚会議の国家標準委員会である。 、この委員会の主な業務は、次のとおりである(株)。1)製品及び測定技術の標準化、部門間の統一を図るため、技術的・経済的な基礎とともに、科学的・方法的な基礎を開発・確立するための最も重要な方針を決定すること。2)標準化及び測定技術の体系を改善すること。国防産業を含め国民経済のすべての領域において、技術や品質の向上、製造・開発・使用におけるコストの低下を図るための有効手段として、標準化及び測定技術を適用する範囲を拡大すること。3)製品の品質指標、すなわち、製品の開発・製造・受入れ・試験に関する一般的要求事項を標準化すること。4)一般用の製品に対する規格における要求事項と、国防上の要求との間のつり合いを保つこと。5)国内における製品品質認証制度を運営すること。6)生産の専門化、製造工程の複合的機械化・自動化、製品の互換性及び使用・保全の効率の向上という重要な面から、製品の大規模な統一を図ること。7)測定方法の統一及び測定の精度の向上を図ること、国の測定サービス及び測定標準の体系を強化し発展させること、高い精密度を必要とする測定について、新しい方法・手段を開発するとともに、既存のものを改良すること。8)物理定数とともに、測定・管理・試験:の方法・手段を標準化すること、標準物質の検定を行うこと。9)測定手段の国家試験制度を運営すること。10)測定器の管理・使用の状況とともに、規格・仕様書の実施・遵守の状況を、国の立場から監督すること。ロシアにおける標準化関係機関及びそれらのつながりを図8・15に示す(34)。β9-194計伽汀家(会員口写委点画熟究
