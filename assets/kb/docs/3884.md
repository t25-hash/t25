# 9・1・1 高度技術社会における公衆と科学技術(者) 第三千年

> **章**: 第9章 技術者倫理及び資格に関する制度
> **節**: 9・1 なぜ今,技術/技術者の倫理か

紀を迎え,科学技術と人間社会の関係を歴史的な視点に立って傭日敢(ふかん)すると,現代ほど科学技術が深遠,かつ,広範な影響を人間社会に与える時代はないと明言できる.科学技術の高度化,細分化,ブラックボックス化に伴い,科学技術者の専門化が進み,科学技術の研究・開発の最前線において,技術的な判断のみならず,倫理的判断を含むさまざまな価値判断を少数の専門科学技術者に依存せざるを得ない状況が明らかになってきた.この事態を敏感に感じ取った一般大衆は,一方で科学技術の成果を享受しながらも,他方で科学技術やその専門家に対して「漠然とした不信感」を募らせているように思える。このような危機的状況の中で,科学技術のあり方が問い直されている.特に,人類が直面する問題の解決のため,科学技術を担い,意思決定の当事者である技術者が,その社会的責任を認識し,高い倫理観をもつことが求められている.このような背景の下,20世紀末,我が国では東海村臨界事故,山陽新幹線コンクリート崩落事故,雪印乳業食中毒事件,三菱自動車リコール隠し事件など,科学技術に関連する不祥事や事故といった技術者の倫理が問われる大事件が頻発した.確かに,原因の「端は当事者である技術者が,自分が属する組織(会社や教団)への責任や個人的な利害を一般社会に対する責任よりも優先させたことにある.さらに,上述の科学技術(者)に対する「漠然とした不信感」を反映してか,これらの事故や不祥事の原因の究明をおざなりにしたままで,短絡的に科学技術者の「モラルの低下」や「倫理観の欠如」という曖昧模糊とした標的を作り上げ,そこに原因を見いだそうとする報道や論評が数多くなされている.しかしながら,問題の根幹には,これまでの日本,特に戦後の高度成長期において,日本社会全体が,また技術者自身が「技術の目的とは何か」,「技術と社会の関係はどうあるべきか」,「技術者の使命とは何か」,などの技術者倫理の問題に真剣に取り組んでこなかったことが挙げられる.また,技術者の教育や研修において,技術者倫理の問題がほとんど無視されてきたか,あるいは「周辺領域」として軽視されてきたことにも起因する.さらに,これらの事故や事件は,単に科学技術に直接関係する技術的問題や現場の技術者の行動だけに起因するものではなく,経営陣の価値観や判断能力,組織構造の変化や体質,その他のヒューマン・ファクタなど,さまざまな要因が複雑に連関して引き起こされたものである.ある意味では,個別でわずかな判断ミスが次第に複合的に絡み合い,最終的には甚大な影響を社会に与えるという高度技術社会の脆弱さを如実に示す徴候とみなすこともできる.高度科学技術の偏在化に伴い,科学技術と社会との連関が複雑さを増すと同時に本質的に変化したことを,技術者自身を含めて,科学技術に携わるすべての人々が自覚すべきであり,さらに,科学技術と社会のあり方に関する新たな価値体系とそれに基づく倫理的規範を構築すべき時期を迎えている.
