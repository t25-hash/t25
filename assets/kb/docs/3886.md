# 9・1・3 我が国の技術系学協会の対応 このような状況の中

> **章**: 第9章 技術者倫理及び資格に関する制度
> **節**: 9・1 なぜ今,技術/技術者の倫理か

で,技術/技術者倫理を検討し,倫理規範の確立や倫理教育の振興に向けての活動が近年各方面で顕在化してきた.例えば,日本学術会議は,1997年5月の報告書で,大学の学部段階で技術者倫理教育を行うべきであると提言した.また,倫理規範に関しては,1996年5月に情報処理学会が,会員が遵守すべき「倫理綱領」を制定した。また,電気学会,電子情報通信学会,土木工学会,日本建築学会,日本機械学会,日本原子力学会などでも倫理綱領の策定や倫理教育についての組織的な取り組みが始まり,1998年以降,相次いで倫理綱領を制定した. - 一年12月に評議員会で承認され,2000年2月の学会誌でその制定に関する主旨が説明された.さらに,同年4月には,政策・財務部会の下に常置委員会として「技術倫理委員会」が新設され,倫理規定の継続的見直しを含む諸活動を開始した.技術者教育・資格の面でも,技術者倫理への取り組みが始まった.1999年11月目発足した日本技術者教育認定機構は,日本の大学における技術者教育プログラムが共通してもつべき教育目標のひとつとして,「技術の社会及び自然に及ぼす影響・効果に関する理解力や責任など,技術者として社会に対する責任を自覚する能力(技術者倫理)」を要求しており,この基準に基づく認定の試行を2000年秋から開始した.これらの活動を支援するために日本工学教育協会は,大学における技術者倫理教育の具体的な実施方法に関するワークショップを2000年11月及び2001年9.月に開催した.2000年4月には,技術者資格の国際的な相互認証問題を背景に,技術士法の一部が改正されたが,その中で「技術士等の公益確保の責務」という倫理規範が明文化された。また,技術士が自らの資質・能力の維持・向上に努めるべきことが要請され,それを支援する継続教育の必要性が認識された.日本技術士会が事務の中心機関となり,関係学協会などとの連携の下で,具体的な教育・研修プログラムの開発に取り組み始めたが,その基本方針には「技術者倫理の徹底」が明示されている.高度技術社会を担う技術者の倫理的判断力の向上に貢献しようとするこれらの活動は,高く評価できる.しかしながら,技術/技術者倫理の構築を推進すると同時に,今後真剣な検討を必要とする多くの課題がある.今後の取り組みの一助とするために,以下では,技術者の倫理に焦点を当て現状の整理をする.,「技術者倫理」とは何かを定義するのは簡単ではない.「技術」とその担い手としての「技術者」及び「倫理」,それぞれの定義は多様で,共通した理解があるというわけではない.そこで,まず,「倫理」及び「技術者」の一般的な定義を基に,本章の作業仮説として,以下に,’「技術者倫理」とは何かを定義する.
