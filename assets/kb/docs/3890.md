# 9・3・1 Prdb88ionとは 最近制定された我が国の学協会の

> **章**: 第9章 技術者倫理及び資格に関する制度
> **節**: 9・3 専門職能(集団)(profession)と倫理綱領

倫理綱領は、欧米、特にアメリカ流の技術者倫理規範に強い影響を受けているが、欧米の技術者倫理の背景には、「learned profbssion(知的専門職能集団)」という、日本では馴染みの薄い概念が深く関係している。伝統的に西欧では、聖職者、医師、法律家に代表されるような、高度の専門知識と技術をもつプロクェッション(専門職能集団)の存在を認め、育成してきた。これらプロクェッションの一員として認められるためには、長期にわたる高度な専門的教育と厳しい訓練を受け、客観的な方法(例えば、国家試験など)で自らの専門能力を証明しなければならない。しかし、ひとたび、加入が認められると、他の職業では得られないような高い報酬と特権(特に自治権)が与えられる。これは、職能集団と一般社会との間の一種の契約と考えることができる(6)。つまり、プロクェッションは、他の人にはできない、かつ社会や個人の健康や安全、福利の維持と向上にとつそ不可欠なサービス(奉仕)を、責任をもって行い、その見返りとして、社会は高い社会的地位と自治権をその集団に与えるのである。この互恵的な関係を維持するために、プロクェッションは厳格な倫理規範を構築する。ヒポクラテスの誓いに見られるような医師の規範や法律家の倫理綱領がその例である。倫理綱領は、専門家がプロクェッションの一員として自らを厳しく律する行動規範であり、基本的には依頼された仕事の達成のためには常に最善を尽くし、模範的なサービスを提供するという宣誓である。特に、専門家としての仕事が単に個人の利害の為ではなく、社会一般のために行われるというのだという自負と使命感を強調するω.専門的な知識と能力をもち、この行動規範を共有できる者だけが職能集団の一員として認められるのである。つまり、社会からの信託を受け、社会に本質的な影響を与える可能性のある意思決定を行う立場にある技術の専門家として、厳格な倫理規範に従って行動していることをエンジニアリング・プロクェッションの内外に公示するために、技術者の倫理綱領が必要なのである。そして、その綱領の根幹は、技術者として仕事をする上で、他のいかなる事柄よりも、「(環境への配慮を含めて)公衆の安全、健康、福利を最優先する」という基本姿勢である。最近制定された我が国の倫理綱領・規程の多くもこの基本的な憲章を何らかの形で含んでいる。
