# 9・3・2 倫理綱領の機能 ところで,倫理綱領がもつ機能と

> **章**: 第9章 技術者倫理及び資格に関する制度
> **節**: 9・3 専門職能(集団)(profession)と倫理綱領

しては,(1)一般社会と専門職能集団との「契約」に関する明確な意思表示,(2)専門職能集団のメンバーが目指すべき理想の表明,(3)倫理的な行動に関する実践的なガイドラインの提示,(4)将来のメンバーを教育するためのツール,などが考えられる.そして,おそらく最も重要なものとして,(5)これまでの諸点並びに専門職能集団の在り方そのものを議論する機会を提供するための場,を挙げることができる.まず,第一一の機能については,すでに述べた「社会契約説」の立β9-200場に立つと,プロクェッションとしての行動規範を明文化し,プロクェッションの内外に明示することは,「契約」に不可欠の要素である.つまり,社会からの信託を受ける以上,プロクェッションの構成員は明確に公示された規範に則り行動し,職務を全うしますという約束事項の表明である.第二の機能は,プロクェッションとしての向上目標の標榜である.すなわち,プロクェッションのメンバー一人ひとりが自らの能力を継続的に開発させていくべき方向性の提示と,プロクェッション全体が社会の中でその地位をさらに向上するために理想とする姿の明確化が図られる.第三号機能は,実際に倫理的なジレンマに直面しているメンバーが倫理的な判断を下すにあたり自律的に意思決定をするためのガイドラインを提供することである.ここで注意しなければならないことは,倫理綱領はあくまでもガイドラインであり,法律のように厳密に適用条件が定義されものではないという点である.プロフェッショナルに求められている「道徳的自律性(moral autonomy)」とは,厳密に明文化された規定条項に盲目的に従うことではなく,自ら情報を収集・分析し,関連する事柄を熟考した上で,自分が属する組織や他者からの影響からは独立に,道徳的意思決定を行うことである.また,そのような意思決定に従って行動し,その行動の結果の道徳的責任を負うことが道徳的自律性である.つまり倫理綱領は,道徳的自律性をもつプロフェッショナルがジレンマに陥ったときに行動の指針を与えるという機能をもつものである.第四の機能は,例えば,大学生などのように,将来エンジニアリングのプロフェッショナルになろうとする人たちに対して,専門的な知識や技能の他に,どのような人格的資質や倫理的徳目が求められているのかを明示することである.専門教育を受ける間に,倫理綱領に触れることによって,自らが伸ばしていくべき資質が明確になる.このように達成目標あるいは向上目標が明文化されることによって,大きな教育的な効果が期待される.また,実際に技術者倫理教育を行う上で,現存するプロクェッションの倫理規範を提示することは不可欠である.第五の機能は,これまで挙げた諸点をすべて総合し,専門職能集団の在り方そのものを継続的に議論する場を提供するという最も重要な機能である.専門職能集団が,その集団しかなしえない専門的な業務を遂行する以上,そのメンバーが専門分野に関する興味・関心を共有するのは当然であるが,加えて,自らのアイデンティティについて自省すべきである.しかしながら,現状では,専門職能集団がそのような考察の機会をもつことは少ない.しかし,継続的に倫理綱領を検討することにより,社会との契約事項や自らの向上目標,またメ1ンバーが直面している具体的な倫理的ジレンマ,又は後継者育成の在り方などを慎重に検討することができる.さらに,自らの存在意義とは何かという本質的な問題についても熟考する機会が与えられる.このような自省の場を提供することこそが,倫理綱領の最も重要な機能である.さて,上述のようにプロクェッションという概念を使い,社会契約説の立場に立てば,技術者がプロフェッショナルとして特別の責任をもつことは明らかである.また,プロクェッションの一員として認められるためには,専門的な知識や能力をもつだけでなく,プロクェッションがもつ倫理規範に則って行動するという誓約を,プロクェッションめ内外に公示する必要がある.これが,技術者の資格制度と技術者倫理が密接に結びついているゆえんである.
