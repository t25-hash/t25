# 1・4・1 熱交換器フィンのシミュレーション計算

> **章**: 第1章 熱交換器
> **節**: 1・4 数値シミュレーションによる最適設計の例

一手法として熱回路網法（thermal network method）を応用した例えば，加賀ら による計算においては，熱交換器の内部をのパス変更による熱交換能力の改善効果は113%であった．
また冷媒の相変化を計算するために有効比熱圧力損失とそれに伴う飽和温度の変化を考慮するため，一次元の管路網モデルを導入して熱回路モデルと連立させて解を求める．
熱交換器の能力予測手法がいくつか提案されている．
段列ごとのブロックに分割し，各ブロックの構成要素（空気，冷媒，フィン，パイプ）ごとに節点を設けて熱抵抗で連結し，熱回路（eﬀective specific heat）評価法 を導入し，冷媒の流れに沿ったとができる．
