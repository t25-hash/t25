# 1・4・2 熱交換器冷媒パスの最適化シミュレーション

> **章**: 第1章 熱交換器
> **節**: 1・3 各種熱交換器の性能

強度設計（プレートフィン構造部，同左以外のタンク・配管部）お
よび接合（ろう付け，溶接）技術に依存しており，検討項目は多岐
にわたる．
実際に製作されている事例から，最高使用圧力はおおむね10
MPa程度であるが，この値はより高温になればそれぞれの材料強
度の温度依存性に従って低下するのはいうまでもないので注意を要
する．
最も多く用いられるアルミニウム合金では約200℃まで，銅合金
やチタン合金も同様域であるが，ステンレス鋼では約800℃，さら
にニッケル系耐熱合金では1000℃付近まで使用されることがある．
これ以上の温度域ではセラミックスの領域になるが，この材料でプ
レートフィン型が製作された事例は極めてまれであり，確立された
ものではない．
