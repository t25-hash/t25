# 10・1・1 知的財産権 財産と呼ばれるものには,土地や家屋,

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・1 知的財産制度概要

貴金属類,自動車などの有形のものと,発明やデザイン,文学作品,絵画などの無形のものがある.知的財産権は,その内人間の知的活動によって創り出された無形の財産に係わる権利である.知的財産権は,知的所有権又は無体財産権とも称されるが,ここでは知的財産権を用いる.知的財産権は,その保護されるべき対象物により,各種の法律が制定されている.これらの知的財産権に係わる法律は,大きく産業及び文化的創作を保護する「創作法」と,営業上のマークや標識を保護する「標識法」に分けて考えられる.創作法はまた,産業目的に寄与する工業所有権と,文化目的に寄与する著作権とに分けて考えられてもいる.工業所有権は,特許法,実用新案法,意匠法,商標法の4法からなる.しかしながら,技術の発展と共に保護されるべき知的財産の範囲が拡大しており,著作権法にコンピュータソフトを含めたり,半導体集積回路保護法や種苗法の新設がなされ,知的財産の不正使用に対しては不正競争防止法を利用して対抗をしてきている.標識に係わる法律として,商標以外にも商号に関する商法や,商品の地理的表示に関する法律などがある.従来は工業所有権のみが重視されていたが,近年,コンピュータやネットワーク等の情報技術の急速な発展と共に経済のグローバル化が進み,工業所有権だけにとらわれずに著作権やノウハウ等その他を含めて知的財産権として包括的に権利の取得や保護活動等を行う必要が強まっている.とはいえ,工業所有権による保護対象も拡大してきており,依然として工業所有権は知的財産権の中核をなす権利であり実務上下も重要な権利である.知的財産権は独占的排他権の一面を持つために,産業の保護発展に寄与するが,過度に独占権を行使されると円滑な経済活動が阻害されることになりかねない.例えば,技術の共有化や流通が妨げられて技術改良や幅広い活用の障害となる.また,特許技術を利用した商品・サービスの供給量や提供地域が限られることになると社会生活にも好ましくない現象が発生する.したがって,知的財産制度は調和のとれた適正な運用がなされることが望ましい.
