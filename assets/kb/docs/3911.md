# 10・2・3 登録後

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・2 特許・実用新案

a.特許権i.発生 特許権は設定登録により発生する(細雨66条).ii.効力の内容 特許法第68条には「特許権者は,業として特許発明の実施をする権利を専有する」と規定されている.すなわち特許権の効力は,業として特許発明を独占的に実施し得ることである。「業として」というのは,「事業として」という意味である.したがって家庭的又は個人的に実施することは,業としての実施ではない。「実施」については,特許法では定義が示されており,常識的な解釈とは多少異なる点があるので次に説明する.特許法第2条では,発明の実施を三つに分け次のように定義している.(1)物の発明にあっては,その物を生産し,使用し,譲渡し,貸し渡し,譲渡若しくは貸渡のために展示し,又は輸入する行為(2)方法の発明にあっては,その方法を使用する行為(3)物を生産する方法の発明にあっては,前号に掲げるもののほか,その方法により生産した物を使用し,譲渡し,貸し渡し,譲渡若しくは貸渡のために展示し,又は輸入する行為「権利を専有する」というのは,他人を排して特許権者のみが,独占的に実施する権利を有するという意味である.したがって他人がβ9-214正当な理由がなく特許発明を実施するときは権利を侵害することになる.この場合は特許権者は,その実施を差止めたり,侵害によって受けた損害を賠償させることができる.また侵害者が粗悪品を造って特許権者の信用を害した場合は,この信用を回復するのに必要な措置を命ずることができる(特第106条)。ここで注意すべきことは,上記の規定から,特許侵害品であることを知らないで購入した物を業として使用することが,特許侵害になるということである.これは新式の工作機械や生産設備を購入する際に発生しがちである.したがって新式の工作機械や生産設備を購入する場合は,性能・価格等の確認と共に,まず特許調査を十分に行うようにすべきである.もちろん機械や設備のメーカは生産と販売という行為で特許を侵害しているが,機械や設備の購入者も業としての使用という行為で特許を侵害しているのであり,特に購入者が大量生産を行っている場合には,損害賠償額が多額になってしまう.・iii.効力の範囲 明細書中の特許請求の範囲の記載を基礎にして,発明の詳細な説明や図面を参酌して定めることになっている.したがって説明や図面のみに記載されている発明又は考案は,効力の範囲に属さない.iv.効力の制限 特許権には,公益上又はその他の理由による種々の制限がある.これらを内容的制限と時間的制限とに分けて説明する.(a) 内容的制限(特第69条,77条,78条等)(1)試験又は研究のための実施には,効力は及ばない.(2) 出願時にすでに日本国内にある物には,効力は及ばない.(3) 単に日本国内を通過するにすぎない船舶若しくは航空機,又はこれらに使用する機械等には,効力は及ばない.(4) 実施権を有する者の実施には,効力は及ばない.これらについて簡単に説明すると,(1)は他人の特許発明を試験又は研究して,更に改良することを法が奨励していることから,試験又は研究が自由にできるようにするためである.(2)は既存の状態を保護するためである.物とは形のある物という意味であり,しかもその物限りの意味であって,その物と同じ物を出願後製造することは許されない.また物の存在は日本国内に限られているから,外国において出願前に存在し,出願後に日本国内に持ち込んだ物には効力は及ぶことになる.(3)は国際交通の便宜の上から望ましいことであり,また権利者に損失を及ぼすこともまれであるためである.パリ条約第5条の3と同趣旨である.(4)は権利者以外の者も,特許発明を業として実施できるようにするための権利が実施権である.また産業政策上等の見地から権利者の意向と無関係に(当然に又は強制的に),このような権利の設定を認めることが必要な場合がある。実施権は専用実施権と通常実施権とに大別される.専用実施権は取り決めた範囲内では,専用実施権者が業として特許発明の実施をする権利を専有し,権利者自身も実施できないというもので,わが国特有のものである(特第77条).専用実施権の設定は特許庁に備える特許原簿に登録しなければ,その効力を生じない(特第98条).契約をしたその日から成立する通常実施権と,この点で大いに異なる.通常実施権は権利者自身も実施できるばかりでなく,重複する範囲でいくつも通常実施権を許諾できる(耳聞78条).通常実施権は権利者の許諾によるもののほか,行政庁の裁定によるもの,法律の規定によるものとがある(特第92条等).行政庁の裁定による通常実施権の主なものとしては,他人の先願の権利を利用した後顧の権利者が取得できる実施権がある(特第92条).法律の規定による通常実施権の主なものとしては,職務発明について従業者が特許を受けたときは使用者(会社)が有する実施権(受端35条)と,他人の出願前からその内容と同じ物を実施している者,又は実施の準備をしている者が有する実施権(特第79条)とがある.後者は先使用権といわれており,企業間において先使地権の有無について争われることがある.先使用権の事実は立証が困難な場合が多いうえに,モデルチェンジした物にまで権利が及ぶかどうか解釈が分かれることがあるので,先使用権の存在を主張するよりは出願をしておくことが望ましい.(b) 時間的制限 特許権に時間的制限をした理由は,特許権は発明考案を保護し利用することによって,発明考案を奨励し,もって産業の発達を図るために付与されるものであるから,一定期間独占を許すことによって,その目的を達することができるからである.存続期間に制限がないと,一つの発明を実施しようとする場合は,利用関係にある過去の無数の権利について実施許諾を得る必要が生じ,その煩雑さと経済的負担とにより,産業の発達を阻害することになる.期間に多少の差はあるが制限を持たせたことは各国とも同じである.日本においては次のように定められている.特許権の存続期間は,出願の日から20年をもって終了する(通論67条)。なお,医薬品等の特定分野においては,薬事法等により,許可を受けていなければ特許を実施することができない場合がある.このような他の法律による存続期間の浸食を担保するため,存続期間の延長制度により,最大5年の存続期間の延長が認められている(特第67条の2).b.異議申立 日本においては,早期権利化の要請に答えるとともに,第三者の不測の不利益を防止するため,特許付与の見直しの観点から,公衆の意見を勘案するべぐ,特許付与後に行う異議申立制度を採用している.すなわち,特許異議申立とは,特許後の一定期間内に広く第三者が特許の取消を求めるものである(特第113条).異議申立のできる期間は特許掲載公報発行の日から6ヶ月以内である.なお,異議申立の必要の有無は技術者でないとわからない場合も多いので,特許部署と技術部署とが緊密な協力をし,特許掲載公報について速やかな検討ができるような体制を日頃からつくっておく必要がある.異議申立には証拠が必要である.発行日や開示技術内容の証明のしゃすさの点から,証拠は内外の特許公報や文献などが望ましい.異議申立の手続きの経過は図10・2に示されているが,異議申立をするには異議申立書(正本及び副本)を特許庁へ提出する.特許庁では,特許権者に異議申立書の副本を送付する(特第115条).異議申立の審理は原則3名の審判官の合議体によって審理される(特第114条).審理において,取消理由がある場合は取消理由通知が特許権者に送られる(特第120条の4).特許権者は意見書によって反論することができるとともに,訂正請求により,明細書,図面について訂正することができる(特第120条の4).,最終的に審判官の合議体が取消理由ありと判断し,取消決定が確定した場合,特許権は初めから存在しなかったものとみなされる(特第114条).一方,取消理由なしと判断した場合は特許維持の決定がなされる(特第114条).なお,維持決定についは不服申立をすることはできない.維持決定について不服の場合,第三者は別途,無効審判を請求することができる(特第123条).また,取消決定に不服の場合は審決取消訴訟(特第178条)等による不服申立が可能である.C.無効審判,訂正審判 特許法における審判には,上述した拒絶査定不服審判の他(特第121条),無効審判(特第123条),訂正審判(特第126条)がある.i.無効審判 蝦疵ある特許権は特許権者に不当な利益を与え,第三者の自由な実施を不当に制限するという不都合を生じる.そこで,蝦疵ある特許権を遡及的に消滅させるべく無効審判が設けられている(特旨123条).実務上では,例えば侵害警告等を受けたときの対抗手段等として利用される.特許権の設定登録後であればいつでも,権利消滅後であっても請求することができる(特第123条).なお,無効審判において,特許権者は,訂正請求により,明細書,図面の訂正を行うことができる(特第134条).ii。訂正審判 設定登録後は明細書等は一種の権利書としての役割を果たすため,みだりに変更すべきではないが,特許請求の範囲が広すぎる場合には,有効な部分も含めて全体の無効が主張されたり,不明瞭な記載のために関係者間で無用の争いを生じる場合がある.そこで,第三者に不測の不利益を与えない範囲で明細書等の訂正の機会を与える訂正審判(特第126条)が認められている.なお,特許権の設定登録後であればいつでも,権利消滅後であっても請求することができるが,異議申立,無効審判に継続している場合は,訂正請求により訂正を行うことができるため,訂正審判を請求することができない(特第126条,特第120条の4,特第134条).
