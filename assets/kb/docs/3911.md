# 1・1・4 学問の名称

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

ある．
(ａ)生体・生物機能解明のためのロボティクス研究の構造
進化型ロボティクス（evolutional robotics )
にロボットが構築されるが，この場合のロボットも要素技術検証の
実験装置として位置付けられるもので，この種の研究においては効
果が検証された要素技術が成果であり，ロボットそのものが最終成
果物ではないことに注意する必要がある［図1･1(ｃ)］．
近年，ロボット研究の成果の解釈は混乱しており，科学的動機の
もとでの研究や，要素技術の開発において開発技術の検証に使われ
たロボットが興味深い動きを見せるとあたかもそのロボット自体が
開発対象であるかのごとく宣伝される傾向が見られるが，それはロ
ボット技術に誤った評価を与えると同時にロボット技術の健全な応
用展開にも障害をもたらすことになるので，注意する必要がある．
ロボットは実際に構築された人工物であるが，こうしたロボット
に関し，ロボットとは何かということからそれをいかに作るかな
ど，さまざまな考察・研究・開発を展開するのがロボットの学問で
一般に，対象とするロボットが扱う領域，応用分野などの相違に
よりさまざまな学問的名称が使われるが，その代表的な名称は本書
のタイトルの一部に使われているロボティクスである．ロボティク
スの英語はroboticsだが，-ticsはphysics （物理学），math-
ematics （数学）などに見られるように～学と訳される接尾語であ
る．したがって，roboticsは一般に日本語ではロボット学と訳さ
れ，工学的な問題だけではなく社会科学的課題も含む，ロボットに
かかわる学問全体を意味する言葉として使われる．しかし，工学と
してのロボット以外の学問がそれほど活発に行われていないことも
あり，ロボティクスを工学的なロボットのみを扱う学問と解釈する
場合も多い．本書もその例に従っている．
一方，日本語ではロボット工学という名称を特にロボティクスと
分けて工学的問題のみを扱う場合に使うことも多いが，外国ではこ
の用語は一般的ではなく，例えばrobot engineeringなどという言
葉はあまり使われていない．これは，欧米ではロボティクスが工学
（engineering）というよりも科学（science ）の一分野ととらえら
れていることによると思われる．欧米の大学では，一般に，コンピ
ュータサイエンス学科の中にロボティクスの講座がおかれているこ
とが多い．
(ｂ)役立つロボットを開発するロボティクス研究開発の構造
ロボティクスではさらに，扱う研究分野や応用分野によって，生
物型ロボティクス（bio robotics ），医用ロボティクス（medical
robotics ），宇宙ロボティクス（space robotics ），水中ロボティク
ス（underwater robotics ），産業用ロボティクス（industrial
robotics ），マイクロロボティクス（micro robotics ），人間共存型
ロボティクス（human friendly robotics ）等のようにさまざまな
名称が使われている（表1･1）．これらの名称は新応用分野の出現
とともに案出される傾向にあり，今後も，研究分野，応用分野の展
開に応じて増えていくことが予測される．
