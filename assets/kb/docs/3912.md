テムということから，その構成要素として以下のものが挙げられる．
(１) 運動要素（効果器と呼ぶ)：実世界で物理的に作用を及ぼ
す，人の手足に相当する何らかの要素．
(２) センサ・センシング要素：実世界で効果的に仕事をするた
めに実世界のようすを把握する必要性があるが，そのために用いる
見る・聞く・触るなどの各種感覚機能をつかさどる要素．
(３) 知能・制御要素：使用者である人の指令を聞いて理解し，
それに基づき，センサからの情報により理解した実世界の状況下で
適切な行動を生成して運動系に指令を発する要素．
(４) 環境構造要素：知能・制御要素およびロボットの機能を発
揮しやすくするように工夫された環境としての要素．
以上に述べた構成要素が統合化されたシステムがいわゆるロボッ
トであるが，このロボットがチャペックの戯曲のように社会におい
て人の役に立つ働きをするには，さらに忘れてはならない重要な要
素がある．それは，人に具体的にサービスを提供するソフトウェア
や応用技術である．一般的には，ロボットは上記4種の要素のうえ
にサービス機能が実装された工学システムあるいは機械システムと
して構築される（図1･2）．
サービスの実装については後述することにして，各要素をさらに
詳細に見ていく．まず運動要素としては，物をつかんだり操る人や
生物の手・腕に相当する機能をつかさどるマニピュレーション要素
と，脚の機能をつかさどる移動要素とに大別され，それを実現する
さまざまな機構や制御技術，例えば腕や脚の運動学，動力学，多関
節協調制御理論，安定歩行理論等々が開発されている．移動要素に
ついては，最近では路面移動だけでなく飛行機能も取り上げられる
傾向にある．
一方，センサ・センシング要素で検討が行われるのは，さまざま
な情報を検知する各種センサとその信号処理技術である．一般に人
や生物は，自身の体の状態を制御するための内界センサ（internal
sensor ）と，自身の周囲の状況を検知したり他とコミュニケーショ
ンするために他の個体の発する情報を検知する外界センサ（exter-
nal sensor ）の2種をもつが，ロボットにおいてもこれら2種のセ
ンサとその処理技術が検討されている．ここで，内界センサとして
重要なものは運動要素機構の可動部の位置や速度，加速度などを測
るセンサ，また外界センサとしては視覚，力覚，触覚，聴覚センサ
などがある．また，ロボットの応用分野によっては，人や生物には
ない情報を検知するセンサ（例えば原子力プラントメンテナンスロ
ボットなどの放射線検知センサなど）も検討されている．
知能・制御要素は，ロボットを設計者の要求どおりに知的に動か
すために必要な，ロボットがもつべき仕組みである．ロボットが一
般には人に使われる機械であることを前提とするとき，人にどのよ

るタイプの2種類が検討されており，前者は遠隔操作型ロボット，

化（された）環境と呼ぶことがある．
