# 10・3・4 意匠権侵害の未然防止 今までは、意匠登録を行い、

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・3 意匠

他人の実施を阻止するための方法を述べてきたが、競合他社も同様な対応を講じていると思われるため、相手の意匠権を侵害することがないような対策も検討しなければならない。意匠権に関する情報は、意匠登録後2~3ヶ月の間に特許庁から発行される意匠公報で確認できる。現在はCDで発行され、特許庁ほか各地の発明協会、主たる図書館で見ることができる。インターネットの特許庁図書館でも見られるが、ブラウズに時間がかかり、たくさんの資料を見る予備調査には向いていなと思われる。このように閲覧したら、開発関連製品に係わる意匠公報を入手し設計開発関連部門に渡し、設計者に競争他社の意匠の開発状況を認識させる。他社の意匠公報を入手したとして、どこまで似たらその意匠権に侵害となるのかであるが、意匠権の効力範囲は登録意匠とこれに類似する意匠である。したがって、非類似の意匠なら実施可ということになるが、類似・非類似は、意匠は物品の外観であり、その美感で判断するのが基本とされている。そして、意匠法が意匠権として保護するのは、登録意匠が新しくて、物品の創出に役立つ価値があると認められたからである。とすれば、どこが出願時点で新しかったのかが類似範囲を決定する基本的ファクタとなる。また、機能的な構成や、周知、ないし公知の構成はそれほど重要視すべきではない。さらに、需要の創出なのであるから、需要者の心理ないし視覚を前提としている。したがって、当該物品の需要者の立場に立って、新規部分を共通にしていると判断されるかどうかを判断すべきである。多くの意匠の類否に係る判決は、需要者が混同を生じやすいかどうかを判断基準としている。
