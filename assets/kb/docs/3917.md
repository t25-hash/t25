# 1・2・2 定義

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

メカトロニクスとは，機械と電子技術の融合にかかわる何かであ
る．これを機械と電子技術を融合させて開発された機器とする考え
があるが，これはメカトロニクスの一面しかとらえていない不適切
な解釈である．機械と電子技術が融合した機器が社会に多数普及し
社会に多大の便宜を与えていることから，こうした機器に注目がい
くのはもっともだが，mechatronicsの用語が-nicsという語尾を
有することから，それはむしろ学問として定義されるべきものであ
る．
従来われわれが新しい機械を設計する際に使う手法は機械工学で
あり，それが動力を含む機械である場合は電動機などの電気工学の
知識を統合したものであった．近年，電子技術，情報技術の目覚し
い進歩に伴い，ある種の機器の開発や工学的問題を解く場合に機械
工学，電気工学を活用するだけでなく，電子工学，情報工学の成果
をも活用して問題解決を図ることが一般的となってきた．これは，
電子工学，情報工学がそれまでの機械工学や電気工学だけでは解決
できない，あるいは達成できない解決策を提供することが可能だか
らである．
メカトロニクスとは，「ある工学的問題の解決が要求されたとき
に，機械工学，電気工学，電子工学，情報工学の知識を活用して，
既存の手法を超える解を追求する学問」と定義することができる．
こうした学問的努力を通して開発される機器類がメカトロニクス機
器である．
ここでメカトロニクスの学問の本質を考えるうえで，上記定義中
の既存の手法を超えるという点が重要である．単に異分野を融合し
て機械やシステムを構築するということではない．現状の手法に限
界が生じたときにそれを超える方法を考え，または現状では全く実
現できないシステムを機械・電気・電子・情報工学の手法を駆使し
て構築する努力がメカトロニクスの学問としての本質である．
この過程を図示すると図1･8のようになる．すなわち，ある技術
が極められ成熟するとその性能は限界に達する．そこで初めて，技
図1･8メカトロニクス技術開発の構造
術を極めた専門家は次の解決手段として別の分野の技術を探索す
る．それにより，その開発者の頭の中に技術の融合が生まれ，新し
い製品が生み出される．この技術の融合を生み出す学術的努力がま
さにメカトロニクスなのである．別の分野における解は，機械工学
的手法から化学工学的手法への転換など，対象とする製品によって
さまざまな分野への転換を引起すことが考えられるが，メカトロニ
クスでは，別の技術への転換は一般に機械工学から電子工学・情報
工学的手法への転換，あるいはそれらの融合という形で行われるわ
けである．
(ａ) 古典的機械システム (ｂ)最近の機械システム
こうしたメカトロニクスの学問的アプローチは産業界では日常的
になりつつあり，さまざまな事例を挙げることができる．その一つ
はディジタル時計の開発である ．
図1･ 9機械システムの構造の変遷
時計（腕時計）は，従来は純粋に機械部品で構築されたシステム
である．かつて毎年スイスで開催された機械時計の正確さを競うコ
ンテストでは，日本の時計は世界第1位の地位を占めていたが，そ
の背景には機械工学に深い知識をもつ優れた技術者の努力が存在し
た．しかし，この優れた技術者が技術を極めた結果，機械工学的手
法でそれ以上精度を高めることが困難であることを悟ったとき，当
時の半導体技術の進展を見て，電子技術すなわち水晶発信機と半導
体技術を活用した新しい仕組みの時計の開発に目を転じ，従来とは
異なる機械工学の技術者ではない電子工学の専門家からなる開発チ
ームを編成し，長年の努力の末，機械式時計を超える技術を確立し
たのである．
また，高層ビルの制振問題にもメカトロニクスの活用事例を見る
ことができる．従来は，制振問題は建築分野の構造工学の問題とし
て解決され，それでユーザニーズに対応できた．しかし，昨今の建
築物の高層化，大型化により単なる構造工学の問題としてその解を
見い出すことは困難になり，制振理論をベースとするアクティブ制
振の問題としてそれを扱う必要性が生じている．その結果，この種
の問題は，アクチュエータとコンピュータを使ったメカトロニクス
の問題として最近は取扱われている．この場合，問題が与えられた
当事者は，構造工学，アクティブ制振技術など学際的に分野を俯瞰
して解を見い出す努力が強いられる．
さらに，磁気ディスクのヘッドの精密高速位置決め問題に対して
も同様な視点が要求されている．磁気ディスクの高機能化が追求さ
れるにつれて，ヘッドの設計に際し，その位置決め時の振動をいか
に瞬時に止めるかがディスクの性能を左右する重要課題となってい
るが，これは従来の位置決めサーボ技術の問題だけでは解決でき
ず，位置決め制御，機械設計，機械力学などの学際的知識を総合し
て初めてその解が見い出される．
本書の6･2節で，メカトロニクスの活用により問題解決が図られ
た事例を解説している．
