# 10・3・5 意匠出願の戦略 意匠を十分な保護が受けられるよ

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・3 意匠

うにどのように出願するかの検討は,まず,部分意匠が導入されたことにより,従来のように完成品意匠とその部品意匠だけでなく,部分意匠の出願が検討されねばならない.しかも,第3条の2の規定があるので,部品や部分の意匠は,他人の先願意匠はもちろん,自分の出願であっても先願の完成品意匠の一部であったかどうか検討すべきである.既にそれらの意匠を含む他の意匠が先に出願されている場合はその先の出願を取り下げ,改めて同時に出願する必要がないか検討すべきである.逆に,部品意匠又は部分意匠を含む完成品の場合,第3条の2は働かないため,その完成品の意匠の後日出願は可能である.したがって,設計部門に対しては,完成品ができて出願を提案するのではなく,完成品ができていなくとも,部品ができた段階で出願の提案をするよう徹底させる必要がある.次に,関連意匠であるが,自己の出願意匠に類似する意匠であっても,同日に出願された場合に限り,主従関係を設けて登録するとされたため,重要な意匠については近似する周辺意匠を整理し,防衛意匠も含めて権利取得ができるように出願時期を検討すべきである.β9-220類似,非類似意匠は,審査の段階で決あられ,非類似だと思って数日後に出願しても,審査官によって類似と判断されると,それだけで後願は,登録を否定されるためである.意匠権は関連意匠でも独自の権利範囲を有し,侵害事件で,本意匠の意匠権の効力範囲の掛図材料とされるだけではないため,関連意匠権もそれなりに重要性を帯びてくることに留意すべきである.,第3に,第3条の2や関連意匠の関係で,複数の出願を同日に行わなければならないため,公開期日との関係で出願期限が限定され非常に忙しくなるという点である.出願要否を速やかに決定し,意匠図面の作図に余裕を持って行えるよう配慮すべきである。現在特許庁は出願書類をオンラインで受付けており,ディジタルイメージでの出願が容易になっているので,現品があれば,写真等のデータでとりあえず出願を済ませ,細部については後で補正するという方法もあるが,出願当初から必要部分のデータを提出しておかないと,後日の補正では補正却下となるケースがあることに留意すべきである.【参考文献】牛木理一,意匠法の研究,(1980),発明協会.加藤恒久,意匠法要説,(1981),ぎょうせい。斎藤瞭二,意匠法,(1985),発明協会.高田 忠,意匠,(1969),有斐閣.田村善之,知的財産法,(1999),有斐閣。工業所有権審議会,特許法等の改正に関する答申一意匠制度の見直しについて一,(1997)。特許庁編,意匠審査基準・意匠審査便覧,(1996),発明協会.特許庁編,平成10年改正意匠法 意匠審査の運用基準,(1998).
