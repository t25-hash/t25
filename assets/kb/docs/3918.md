# 10・4・1 商標法とは

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・4 商 標

a.商標法の目的 商標は,商品やサービスに付される標識のことであり,工業所有権の一つとして商標法による保護を受けることができる。商標が保護される理由は二つある.一つめの理由は,商品やサービスを提供する企業は,その製品開発力や営業努力,宣伝活動などを背景として,その商品に他社と区別するために付した商標を通じて消費者からの評価や信頼などを獲得する.したがって,これらの商標を他社が自由に使用してよいこととすると,せっかく優れた商品や品質の高い商品を提供し,評判(ブランドないしはグッドウィル)を得ている企業の努力を無に帰することとなb,その企業は不利益を被ることとなる.二つめの理由は,消費者は,商品選択の基準として商品に付された商標を目印としてよい商品を購入しようとすることが一般的である.しかしながら,その商標が付された商品が,当初購入を予定していた商品とは出所も中身も異なっていたとすると,やはり消費者にとっても多大な不利益を被ることとなる.このように本来商標のもつ,自他商品を識別する機能に着目し,法律をもって制度としてその商標を保護することにより商品を提供する者とその提供を受ける消費者の両者の利益を守り,それにより終極的には健全な流通秩序の維持を通じて「産業の発達」を図るという役割を果たす法律が商標法なのである.b.商標とは 保護の客体としての商標とは,文字,図形,記号,もしくは立体的形状もしくはこれらの結合又はこれらと色彩との結合であって,商品や役務について使用をするものであると定義されている.現実の商標には,「王様」のように文字のみからなるものや,ペンギンの形のように図形からなるもの,看板の人形のように立体形状からなるもの等,さまざまな態様のものがある.アメリカでは,更に,音響なども保護の対象としている.また,商標が使用される対象である「商品」とは例えば「自動車」のように有体動産で転々流通するものをいい,「役務」とは例えば「修理」のようなものをいう.
