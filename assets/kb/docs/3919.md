# 1・2・4 期待されるメカトロニクス技術者像

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

産業活動に伴う種々の問題を解決するうえでメカトロニクス技術
の重要性が高まるにつれて，その人材をいかに育成するかが重要に
なる．
メカトロニクス技術者にはどのような素養が必要であろうか．す
でに述べたように，メカトロニクスにおいて必要なのは，従来の方
法では解決困難なある問題に直面したときに，異なる専門分野の技
術を用いてその問題の解決を図れる能力である．異分野の技術を使
った方法が見い出され，その技術が定着すれば，その後はその技術
の専門家が問題に対処できる．例えば，腕時計は従来機械工学技術
者の独壇場であったが，それがディジタル時計の技術に置換わった
後は，その技術の維持・発展は電子技術者の仕事になる．メカトロ
ニクス技術者とは技術の転換部分に貢献する人である．
メカトロニクスは，機械工学，電気・電子工学，情報科学（工
学），計算機科学（工学）の融合した学問領域とマクロには定義で
きるから，メカトロニクス技術者にはこれらの個別工学の知識をも
つことがまずは求められよう．
しかし，個別工学の知識をもつだけでは不十分で，さらに重要な
ことは，ある分野に対する深い知識を少なくとも一つはもつという
ことである．というのは，1･2･2項でも述べたように，ある問題の
解決が従来の方法では困難と判定することは，その分野に対する深
い知識をもち技術を極めた人によって初めてできることだからであ
る．技術者が自身の専門ではこれ以上の進化進展は困難と判断した
ときに別の解法の探索が行われ，問題解決における異分野間の技術
の意義ある転換・融合は起きる．根拠もなしに異分野を融合してシ
ステムを構築しても，それは機・電融合システムを構築したという
ことだけで，ベストの問題解決法は得られない．
この点で，単に機械工学，電子工学，情報工学などのひととおり
の知識しかもたない技術者には真のメカトロニクス製品は開発でき
ない．
企業では，T型人間，Π型人間という言葉が使われている．T，
Πの縦線｜はそれぞれ専門を意味し，また横線―はその専門をも
とに周囲に知識を広げて学際的に問題が考えられるようになること
を意味している．Π型人間は基幹となる専門が二つあり，T型人間
はそれが一つであることを示しているが，いずれも複数の個別工学
の知識をもち，そのうちの一つないし二つはより深い知識を有する
専門分野をもっており，それをベースに他分野にも目を配れる人を
意味している．
深く技術を極めた専門分野を一つか二つもつ人が，その極めた自
己の知識では問題解決ができないときに別の分野の解を探索するこ
とで初めて意義ある成果が生まれる．こうした意味で，メカトロニ
クス技術者にはT型人間，Π型人間が不可欠である．
ここで，周辺領域の知識は必ずしも深いものである必要はない．
必要なのは，自身の問題を異分野の人にも理解できる言葉で説明す
ることだけである．
メカトロニクス技術者には，自己の専門では解決不可能な問題に
直面したときに自身がもつ周辺知識をにらみ，異分野の個別工学分
野の技術者に問題を提起し，異分野の技術者との討論を通して問題
解決の方法を見つけ，それを個別工学の問題に分解してそれぞれの
個別工学の技術者にミッションを明確にして振り分けることが期待
される．このとき，メカトロニクス技術者は個別工学の詳細まで必
ずしも立ち入る必要はない．問題が個別工学の問題に分解された後
は，個別工学の研究者・技術者の成果を統合する視点から支援し
て，結果を得るべく努力すればよいのである．
産業界では，こうした人材をいかに育成するかに多くの努力を費
やしている．一般に大学を卒業したての技術者は出身学科に応じた
個別工学の知識しかもたないため，メカトロニクス技術者を育てる
には各企業の独自努力による新たな社内教育を必要としているのが
実情である．
