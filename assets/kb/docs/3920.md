# 10・4・3 商標権

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・4 商 標

a.権利化手続 商標登録を受けるためには、商標登録出願の手続を行わなければならない。商標登録出願は、出願人氏名、商標、指定商品などを記載した願書を特許庁に提出することにより行う。商標登録出願が特許庁にされると、まずその出願が定められた書式や手続に従ってなされているか否かをみる方式要件の審査がなされる。次に、方式要件を充たしている出願については、その商標が登録を受ける適格性を有するか否かをみる、実体的登録要件の審査が特許庁の審査官により行われる。審査官は、商標登録出願について拒絶の理由を発見しない場合には、登録査定を行う。登録査定に対し出願人が登録料を納付すると、商標権の設定の登録が行われ、商標権が発生する。また、商標権の登録がされると、その内容が商標公報に掲載される。b.登録要件 審査官により審査される実体的な登録要件として、例えば、商標として自他商品を識別する程に識別力を有しているということや、他人が既に所有している商標と抵触しないこと等が必要とされる。識別力を有しない商標とは、具体的には以下のようなものをいう。1 商品や役務の普通名称:商品「時計」について「時計」、役務「航空機による輸送」について「空輸」など2 商品や役務の慣用商標:商品「清酒」について「メーカ名」、役務「宿泊施設の提供」について「観光ホテル」など3 商品の産地、販売地、品質、原材料、効能、又は役務の提供の場所、質、効能、用途、態様等を示す記述的商標:「一級」「一番」「スーパー」「よくきく」τ東京」などや、商品の形状をそのまま立体商標としたものなど4 ありふれた氏又は名称:「伊藤」「すずき」「渡辺株式会社」など5極めて簡単でありふれたもの1三角形の図形、ローマ文宇1文字、立方体など6 その催識別力がないもの:地模様、標語、現元号などc.存続期間 商標権は設定の登録が行われた日から発生する。商標権の存続期間は設定の登録が行われた日から10年であるが、権利者が10年ごとに更新登録の申請を特許庁に行い、更新登録料を納付することにより、半永久的に権利を存続させることができる。この点で、存続期間の上限を設けている特許法や意匠法などと相違する。商標は使用を継続することにより更に価値が増大するからである。d.無効審判・異議申立・取消審判 存続期間の満了以外で、商標権が消滅する場合として、無効審判制度、登録異議申立制度、取消審判制度によって無効又は取消となる場合がある。無効審判制度と登録異議申立制度は、審査官の過誤によって本来登録要件を充たさない商標が登録を受けた場合に、その商標権を事後的に消滅させるための制度である。無効審判制度と異議申立制度の違いは、無効審判が登録の有効性に関する当事者間の紛争解決手段として位置付けられるのに対し、異議申立制度は特許庁が自ら審査の適法性を見直すための制度として位置付けられるという存在意義の違いである。取消審判制度には、大きく分けて不使用取消審判と不正使用取消審判の2種類がある。不使用取消審判は、商標権者又は使用権者のいずれもが3年以上登録商標を使用していない場合に、その商標の登録を取り消す制度である。したがって、不使用取消審判を請求された商標は、請求に係るいずれかの指定商品について登録商標を使用している事実を権利者が証明しない限り、その商標は取り消されることになる。不正使用取消審判は、禁止権の範囲内における使用(第51条)、分離移転後の使用(第52条の2)、使用権者の使用(第53条)により、一般公衆や他の商標権者などの利益が害された場合に、制裁として●その商標登録全体を取り消す制度である。
