# 1・2・6 ロボティクスとメカトロニクス

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

ロボットとは1･1･2項で述べたように，何らかの人間あるいは生
物的な特徴を備えた，実世界に作用を及ぼす能力をもつ機械であ
り，こうした機械を目指して人・生物の行動，仕組みを記述する理
論を構築したり，それらの理論を背景に具体的に人にサービスを提
供する人工物を設計したりすることが，学問としてのロボティクス
である．一方，メカトロニクスは，ある工学的問題の解決が要求さ
れたときに，機械工学，電気工学，電子工学，情報工学の知識を活
用して，既存の手法を超える解を追求する学問と定義される．
開発された機器レベルでロボティクスとメカトロニクスを比較す
るとき，両者ともに機械，センサ，コンピュータといった共通の要
素をもつゆえに，両者を同一視したり，あるいはメカトロニクスが
生物模倣以上の広範囲な機器を扱うことから，ロボティクスをメカ
トロニクスの一分野と解釈する傾向も見られる．しかしそれは，ロ
ボティクスを単にロボットというハードウェアを構築する学問とし
