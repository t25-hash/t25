# 10・4・4 権利の活用

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・4 商 標

a.商標権の効力 商標権の効力には、自分の登録商標を他人に妨げられることなく使用できる使用権と、自分の登録商標又はそれに類似する商標を他人が使用することを禁止することができる禁止権とがある。登録商標と類似の範囲については、他人が登録商標に類似する商標を使用することによる出所混同を防止するために禁止権が認められるが、法律による積極的な使用権が認められているわけではない。したがって、登録商標と類似する商標については、他人の商標権などに抵触しない限り、事実上の使用ができるに過ぎない。他人の商標権を侵害した者に対しては、侵害行為を中止させる差止請求権、過去の侵害行為による損失を賠償させる損害賠償請求権などの民事上の救済手段を行使することができる。また、商標権を侵害した者は、刑事上の制裁として5年以下の懲役又は500万円以下の罰金に処せられる。侵害をした者が法人である場合には、その法人は1億5000万円以下の罰金に処せられることになる。なお、商標権の侵害は、告訴を待つことなく制裁を受ける非親告罪である。b.商標権の譲渡・使用許諾 商標権については、有償・無償を問わず、他人に権利を移転させることができる。移転は一部の指定商品についてのみを分割して行うことも可能である。また、商標権については、他人に対して使用権を設定することができる。使用権には、使用権の範囲内における他人の使用を認めるに過ぎない通常使用権と、使用権の範囲内でその他人に独占的な使用を認め、商標権者も使用をすることができなくなる専用使用権の2種類がある。これらの使用権の設定に際しては、商標権者は契約により使用権者から対価を得ることも可能である。なお、商標権は相続の対象ともなる。
