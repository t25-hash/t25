# 1・2・6 ロボティクスとメカトロニクス

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

ロボットとは1･1･2項で述べたように，何らかの人間あるいは生り，こうした機械を目指して人・生物の行動，仕組みを記述する理供する人工物を設計したりすることが，学問としてのロボティクスれたときに，機械工学，電気工学，電子工学，情報工学の知識を活開発された機器レベルでロボティクスとメカトロニクスを比較す素をもつゆえに，両者を同一視したり，あるいはメカトロニクスがトロニクスの一分野と解釈する傾向も見られる．
機械振興協会経済研究所,メカトロニクス,機械産業の施策に関す日本学術会議,人工物設計・生産研究連絡委員会メカトロニクス内橋克人,同時代への発言３>実の技術・虚の技術,第4章,か見ない誤った見方である．
物的な特徴を備えた，実世界に作用を及ぼす能力をもつ機械であ論を構築したり，それらの理論を背景に具体的に人にサービスを提である．
一方，メカトロニクスは，ある工学的問題の解決が要求さ用して，既存の手法を超える解を追求する学問と定義される．
