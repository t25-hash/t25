# 10・4・5 防護標章制度 商標について出所の混同が生じる範

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・4 商 標

囲とは、一般的にはその商標と類似する商標の範囲である。しかしながら、多大な信用を獲得した著名商標においては、類似範囲を超えて出所の混同を生ずる場合がある。また、他人が著名商標を使用することは、著名商標主の信用に容易にただ乗りすることを可能とさせるため、著名商標は通常の商標よりも模倣される危険性が大きいといえる。このような理由から、登録著名商標の商標権者は、自己の著名商β9-221標を強力に保護する手段として、防護標章の登録を受けることができる。防護標章は、自己が所有する登録著名商標と同一の商標の非類似の指定商品について登録を受けることができる。防護標章が登録されると、第三者は、防護標章と同一の標章について使用することや、商標の登録を受けることができなくなる。
