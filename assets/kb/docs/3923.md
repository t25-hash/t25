# 10・4・6 商標と外国法 権利は各国で独立して与えられる.

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・4 商 標

したがって,外国で商標権を取得したいときはおのおの手続と費用が必要となる.また,国によって既に先登録があったり,商標として成立しないとして拒絶される場合があるなど,各国の法制の違いには留意が必要である.例えば,日本の商標法は,登録主義(商標権は登録により発生するとする考え方)的な色彩が強いのに対して,アメリカの商標法は,使用主義(商標権は使用実績により発生するとする考え方)的な色彩が強く,現実の運用においてもギャップがある.また,国によって商標として保護される対象も相違し,例えばアメリカでは音響についても保護される.
