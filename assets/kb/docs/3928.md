b.トリウム資源（thorium resource ) 天然に存在するトリウ
ムのほとんどはThで，Thは中性子捕獲により核分裂性の同
位体であるUになる．したがって，トリウムをエネルギー資源
として利用するには，原子炉において中性子を照射してUを作
った後，再処理によりUを取出し核燃料とする，すなわちTh-
Uサイクルを確立する必要がある．トリウムの核燃料としての可
能性は古くから知られているが，その実用化は達成されていない．
現在，トリウム資源がウラン資源よりも多いインドだけが，核燃料
としてのトリウム利用の開発に積極的である．
トリウムの大陸性地殻中の平均含有量は，ウランの約4倍に当た
る7ppmであるが，ウランと異なり水に溶けにくいため，その濃
集過程は，マグマにおける鉱物および元素の分別作用と，表層での
漂砂鉱床形成の二つの過程に限られる．このためトリウム資源は，
火成岩（特にカーボナタイトや過アルカリ岩などの特殊な貫入岩）
に伴うものか，海浜ないし河口の砂に含まれる重鉱物に限定され
る．鉱物として量的に重要なのはモナザイトとバストネサイトで，
前者は希土類のリン酸塩鉱物，後者は希土類の炭酸フッ化鉱物であ
る．
エネルギー資源としての利用の見通しがないため調査が不十分
で，レッドブックでは1986年版 を最後にトリウム資源に関する
報告がなされていない．これによれば旧東側諸国を除く世界のトリ
ウム資源量（ただしコスト$80/kgTh未満のもの）として，確認資
源量約66万tTh，推定追加資源Iとして約90万tTh，推定追加資
源IIとして約86万tThを計上している．資源量の多い国は，ト
ルコ，ブラジル，アメリカ，ベネズエラ等，また，インドはコスト
および資源量確度が未分類の資源として，40万tThを報告してい
る．

.市場 世界におけるウランの組織的な探査・開発は
1940年代に始まった．当初，ウランは希少な資源と考えられてお
り，冷戦構造が深刻化する中，1950年代から1960年代初頭にかけ
てアメリカなど政府主導の探鉱開発が行われ，生産物は軍事用に供
された．結果的に，資源量が増大し，この時期にその後の原子力発
電のための資源ベースが蓄積されたともいえる．原子力発電の導入
が本格化した1960年代後期以降，ウランの市場はその時々の政
治・経済状況等を反映して激しく変化してきた（天然ウランのスポ
ット価格の変遷，図2･25)．天然ウランのスポット価格は，2002年
12月末時点で7＄という最安値から多少もち直した$10/l
bU308（1978年の最高値43.40$の4分の1）前後の水準にある．
価格低迷の原因は供給過剰にあるが，そのおもな要因は，ロシア等
旧東側諸国の天然ウラン在庫の西側市場への放出，ロシアの解体，
核の希釈による低濃縮ウランの販売，電力会社の余剰在庫の取崩し
等の，いわゆる二次的供給の増加である．レッドブックによると，
1999年の全世界の年間ウラン必要量は約61600tU，これに対して
鉱山からの生産量は約34200tUで，一次的供給（鉱山生産）量
は，年間ベースで必要量の6割に満たない．低迷する市場価格の影
響を受けて，1997年以降，鉱山の閉鎖・休止・減産・新規開発計
画の延期や見直し等が相次ぐ中，ウラン資源産業の企業買収・統合
による寡占化も進み，カナダのカメコ（Cameco）社とフランスの
コジェマ（Cogema）社の上位2社で全世界の生産量の5割弱を占
めている．2000年の時点で，世界的に見てもウラン探鉱を実施し
ているのは，少数の小企業を除くと，この2社だけとなっている．
