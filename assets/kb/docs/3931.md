# 10・6・3 第三者の実施の差止め、賠償請求 特許権にもとづ

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・6 知的財産の活用

き、権利期間中又は一時期市場を独占したい場合に第三者の実施を禁止する。実施を調査して、警告する場合と訴訟を提起する場合とがあるが、費用と判決又は仮処分までの期間から先ず警告してから当事者間で交渉することが一般になされている。過去の実施分を金銭で解決することはあまり問題とならないが、将来の実施を止めさせることは生産・販売体制を作り上げていることから難航することもある。将来分について金銭による解決を望まない場合は、訴訟に移行することも想定した交渉が必要である。裁判段階では、差止めの仮処分や、賠償請求を含む判決を求めていくことになる。 ’なお、第三者の実施を止めるので、逆に損害賠償請求を受けないように、侵害の事実と自己の特許権の有効性については事前に弁理士の鑑定を得るなど十分な確認が必要である。実用新案権の権利行使にあたっては、実用新案技術評価書の発行を特許庁に請求しなければならない。
