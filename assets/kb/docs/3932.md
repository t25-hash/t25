# 10・7・1 侵害の予防

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・7 第三者の知的財産への対応

a.新製品・新技術研究開発時の侵害可能性調査 新製品や新技術を開発するときに同じテーマの特許情報を収集して,既存の権利状況や出願状況を知り,あらかじめ権利侵害を回避するために調査する.’ N害可能性のある特許が抽出された場合は,出願経過書類を取り寄せて権利範囲を確認し,登録原簿から権利が有効であることも確認する.開発計画の見直しやライセンス取得等の方針を開発初期に明確にすることで無駄な開発が避けられる.また侵害が無いと分かれば,自社で特許網を構築する対応も図れる.b.製品の販売・輸出時の侵害可能性調査 新製品の発売にあたりその製品・製法が他社の権利を侵害する可能性について調査する.特許権は属地主義を採っているために,製品を輸出する場合は,輸出国における権利の侵害の可能性について,輸出相手国における調査も必要である.開発初期の調査から発売時までの期間が長い場合は,未公開であった出願が公開されている場合があり,再確認をするのがよい.早期審査制度と共に,審査期間が短縮されていることから,公開段階であっても十分な注意が必要である.
