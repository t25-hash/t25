# 10・8・2 技術情報としての調査

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・8 技術情報活用

a,研究開発のための調査 新製品や新技術の研究開発に関連する特許情報を入手し,その分野における技術水準や技術開発の動向を分析したり,開発のヒントを得るために行う調査である.既に出願されており特許の見込みのない技術や,既に権利化された技術の重複研究を避ける点でも役立っ.新規参入時は必ず焦点となる技術に限らず,その成立に関連する技術についても,広範囲に調査し,自社の位置づけを明確にすることが大事である.輸出を前提とする研究開発の場合は,日本及び主要国の調査で抽出された関係する特許について,他の輸出国に特許が存在するか否かを優先権主張のデータにもとつくパテントファミリー調査で代替する手法もある.b.他社動向調査 研究開発する技術領域に参入している他社の出願を調査することで,その企業の技術レベルや研究開発の動向を分析するために行う調査である.彼我比較により,開発の方向性や資源の配分等を決定する際の指標として用いられる.単に出願・登録件数の推移を比較するだけではなく,個々の発明の評価,発明者,その数,同分野における実施例から想定される開発内容の推移なども検討し,他社の開発の方向性を導き出すようにする.法制度や法規制の変化等の社会動向も加味するとよい.新規に発行される特許公報を継続して収集・解析することにより,他社の動向を逐次把握することもなされている.c.出願前先行技術調査 特許を出願する前に,この発明の新規性及び進歩性を確認し特許の可能性を判断するために行う調査である.出願時には先行技術との対比説明が要求されていることからも,・できるだけ出願内容に近い先行技術を調べることが,拒絶理由を排除し広い権利を確保する上で重要である.なお,審査請求をせずに出願した場合には,審査請求までに調査をし必要な補正をすれば,拒絶を回避できる可能性が高まる.関係する技術分野における技術文献・学術論文も調査するとよい.また,発明者(研究開発者)が知っている一般清報も確認すると効率がよい.
