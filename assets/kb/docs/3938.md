# 10・8・3 知的財産情報の所在・入手方法

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・8 技術情報活用

a.特許情報の所在と入手方法i.特許庁(工業所有権総合情報館)特許庁の第一公報閲覧室と全国7都市の地方閲覧室でCD-ROM公報の閲覧が可能である。第一公報閲覧室では外国特許のCD-ROM公報も閲覧できる。経済産業省別館の第二公報閲覧室では日本公開公報や外国の紙媒体の特許明細書や公報が閲覧できる。各閲覧室では特許電子図書館(IPDL)の専用端末から種々の検索が可能であり、その場で紙に出力もできる。ii.発明協会 各都道府県に支部をもち、 CD-ROM公報、合本公報、分冊公報目次公報等を販売し、定期購読サービスも提供している。技術情報及び権利情報に関する調査も行っている。β9-223特許出願をせずに技術を公開するための公開技報も発行している。なお、この開示内容は原則自由に実施できるが、これは特許出願と併用されることもあるので確認は必要である。iii.知的所有権センター 特許庁の認定を受けて設置されている各都道府県の機関であり、発明協会の各支部と同じ場所にあるところが多い。特許CD-ROM公報、特許公報類を所蔵し、 IPDLをはじめとする、各種オンラインデータベースの検索もできる。iv.パトリス 内外国特許データをオンラインで有料で提供する。技術用語テーブルを独自に編集し、技術検索を容易にしている。商標検索もできる。ユーザ指定により、必要な項目でデータ編集し、必要な公報だけをCD-ROMに記録させて販売をしている。v.特許調査会社 パトリスをはじめとするオンラインデータベースの代行検索を行い、CD-ROM公報や紙公報を所有しコピーサービスを行い、特許に係わる全般的な調査を行う会社が多数ある。調査の目的・費用・納期等や調査会社の特性に合わせて選択するとよい。外国特許や文献も取り扱うところが多い。vi.大阪府立特許情報センター 内外国の紙公報とCD-ROM公報を所蔵し、閲覧とコピーができる。IPD:L専用端末やインターネット端末により、各種のデータベースの検索ができる。b.特許以外の情報の所在と入手方法i.技術文献 国立国会図書館の科学技術資料室には内外国の科学技術に関する参考図書・文献類が所蔵されており閲覧ができる。またCD-ROMやオンラインによるデータベースの検索もできる。国内外の技術文献については科学技術振興事業団のJOISのデータベースサービスがあり、必要な文献の取り寄せができる。ii.意匠、商標 オンラインで特許電子図書館(IPDL)、パトリス、各国特許庁データベース、BRANDY等から検索ができる。iii.特許裁判の判例 特許的判断の妥当性を検証するために判例を参考にすることが多い。オンラインでは最高裁判所や発明協会のホームページから検索することができる。c.特許の流通 各企業や公的機関が所有する特許権には、実施されずに権利としてただ維持されているいわゆる休眠特許や、実施許諾をしても自社の技術競争力に影響の少ない特許が含まれている。ヤこのような特許を活用するために、特許の流通をはかる動きは従来から見られていた。最近では、自社の中核技術もライセンスすることで投資回収を早め知的資産価値を向上させようとする動きも出てきている。特許市場活性化のための環境整備として特許庁が開催する特許流通フェアは、特許の提供又は導入を希望する企業・大学・研究機関、仲介業者などが直接交流する場となっている。インターネット上にも、ベンチャー企業向けなど、特許流通の仲介をするウェブサイトが多く見られ、その中には特許の価値評価のサービスを提供するところも出てきている。管理
