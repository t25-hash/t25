# 10・9・1 研究時点 製品・製法の研究開発に着手する際に、

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・9 技術開発、製品開発、製造、販売における知的財産

その研究が現在の技術水準に対しどのような位置づけにあるかを正確に知ることが経営判断上重要である。技術水準を推し量る手立てとして、出願日及び公開日の明確な特許公報は有効である。権利化された一部の技術内容があるとはいえ、技術内容が明確であると共に一般が自由に使える技術内容もあり、新規な着想に結び付けることができるためである。権利者や技術分野を特定して定期的に動向分析をすることで、競合相手がどの方向に向かおうとしているのか、またどこが未着手の技術領域なのかを判断することができる。その中から、少ない出願件数で広範囲を強固に保護する効率的な出願のやり方を決めていく。一方、他社の権利の存在を明確にすることで、それを回避して開発するのか、そのライセンスを受ける等の手段を講じて利用しながら開発するのかを選択できる。また、どうしても回避もライセンスβ9-224申し入れもできなければ、その研究を中止して無駄な開発投資をなくすこともできる。なお、多くの人と時間、資金を投入して明らかに第三者より遅れた技術を開発するより、実施料を支払って特許権等を導入した方が効率的と考えてよいかは検討を要する。いずれにしても、第三者の権利を尊重することは、それを乗り越える動機ともなり結果的に技術蓄積や自らの研究開発力増強の元になるので、安易に無断流用しないことである。場合によっては、製品の製造販売中止や製造設備の停止、損害賠償の支払い、マスメディアへの謝罪広告文掲載など、企業活動にとって甚大な不利益を生ずることにもなる。
