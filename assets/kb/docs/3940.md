# 10・9・2 市場調査 これまで特許出願は保険といわれたこと

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・9 技術開発、製品開発、製造、販売における知的財産

もあるように、自社実施を保証したり、自社の技術水準を示すために、出願されてきた面があった。しかしながら、権利者意識が強まることにより、市場の独占、商品競争力の支援、及び実施料収入を上げるといった観点から権利行使をする動きが活発になってきている。権利行使のためには、自己の権利を侵害しているものをいかに市場で発見するかが重要である。一般には、自社の技術者や営業担当者が新技術紹介、技術発表会、展示会などで情報を入手し、知的財産部門に連絡するような社内体制が敷かれている。いずれにしても、関係部門に自社所有の特許権がいかなるものかの最新情報を継続して知らせることが大事であり、権利の概要を示し、併せて発見時の社内連絡行動を記載したマニュアルを配布しているところもる。侵害品を発見した人に対して報奨金を支払う会社もある。最近では、特定の自社特許権が審査で引用された第三者の特許を連鎖的に抽出し、その記載された技術が自社特許権を侵害する可能性が有れば、その会社の製品を集中的に調査するといったシステムを提供する調査会社もある。このシステムは、逆に自社特許が第三者にとって侵害品摘発の材料となることを意味するから、他社特許権に対する販売前侵害有無調査は十分野行う必要が有る。
