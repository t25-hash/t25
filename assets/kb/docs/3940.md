# 2・3・3 ローエンドコンピュータ

> **章**: 第2章 コンピュータ
> **節**: 2・3 実際のコンピュータ

a.概説 1971年にマイクロコンピュータ（microcomputer ）
が誕生して以来，その高性能化が進むとともにその小形化も急速に
進んできた．1980年代に卓上を独占していたコンピュータも，今
日ではポケットに入る携帯端末がその性能を上回るほどである．
このような背景には，半導体集積回路の急速な技術発展ととも
に，その応用製品を受け入れる市場からの要求とが相まって，これ
ら技術発展の進化を推し進めてきた．この進化は，マイクロコンピ
ュータにとどまらず，あらゆる情報機器に波及している．とりわ
け，携帯電話においては，1980年代には大きなセカンドバックほ
どの大きさであったものが，現在では，手のひらに納まるほどにな
った．
また，進化は筐体の大きさだけではなく，その性能向上から，各
パーソナル小形情報端末が提供するサービスの多様化，高度化を生
じている．性能の向上はパーソナルコンピュータ利用を一昔前のス
ーパーコンピュータの利用領域に広げ，そして，サービスの多様化
は，各パーソナル小形情報端末間の利用領域を超え始めている．ゲ
ームやインタネットを使ったメールやWeb閲覧はパーソナルコン
ピュータの独占であったものが，携帯電話でも行えるようになり，
また，音声通話は携帯電話だけのものではなくなり，パーソナルコ
ンピュータなどでも行えるようになってきた．
b.パーソナルコンピュータ（personal computer ) パーソナ
ルコンピュータは，1970年代後半から1980年代初頭に誕生した．
当時は現在と異なり，各社独自の仕様の基づいたパーソナルコンピ
ュータがいくつも存在していた．日本では，日本電気がPC-8001
を発表，海外に至っては，アップルコンピュータ社のAppleⅡ，
IBM社のIBM PCが市場の拡大を図っていた（図2･23）．
その後，仕様の完全オープンアーキテクチャが起因し，IBM社
以外の新興企業が低価格のパーソナルコンピュータを生産できるよ
うになった．さらにこれに大きく貢献したのが，マイクロソフト社
のWindowsである．各社の仕様の違いを隠ぺいし，各種アプリケ
ーションの普及，拡大を推し進めた．これらの要因が相互に相まっ
て，IBM PCが主流になり現在のAT互換機がパーソナルコンピ
ュータの大勢を占めるようになった．
このような経緯をたどり，現在のパーソナルコンピュータは，
CPUの処理性能やアプリケーションの高度化もあり，従来専門の
技術者が行っていた情報処理サービスを個人でも行えるようにし
た．ワードプロセッサ（word processor ）ソフトの普及により写
植作業は，一般の人でも行えるようになり，IEEE1394などの周辺
機器とパーソナルコンピュータとの高速接続インタフェースの普及
により，ディジタルビデオカメラで撮影された映像をビデオ編集の
プロのように編集し，個人的なDVD（digital versatile disk）の
作成を行えるまでに至っている．また，ネットワーク技術の発達に
より，クラスタ（cluster ）コンピュータ，グリッドコンピューテ
ィング（grid computing）などの新たなパーソナルコンピュータ
の利用が進んでいる．クラスタコンピュータは，並列計算機の一種
であり，その種類は，パーソナルコンピュータをネットワーク接続
したものや，パーソナルコンピュータの部品を流用した，ラックマ
ウント型，ブレード型（図2･24）などがある．
グリッドコンピューティングは，複数のコンピュータをネットワ
ークを介して接続し，仮想的に一つのコンピュータに見せる技術で
ある．例えば，パーソナルコンピュータは1台当たりの処理能力が
低く，大規模な処理をこなせないが，多くのパーソナルコンピュー
タをネットワークで接続することにより，全体としてスーパーコン
ピュータ並みの処理能力をもつことが可能となる．また，パーソナ
ルコンピュータは実際には使われていない時間も多く，その時間を
利用することにより，計算リソースの有効利用も図られる．狭義の
意味で，このような利用方法もグリッドコンピューティングと呼ぶ

c.PDA（personal digital assistance) マイクロコンピュー
タが誕生する少し前の1970年初頭，日本国内では電卓の低価格化，
小形化の競争が起こっていた．この競争はやがて，電卓の高機能化
という方向に向かい，プログラマブル（programable）電卓や住所
録機能が搭載された電卓が誕生，シャープのザウルス等，PDAの
片鱗を示した．一方海外では，1990年代の半ばころに，数社から
PDAが販売された．アップルコンピュータ社からはNewton Mes-
sagePad，USロボティクス社からはPilot ，そしてマイクロソフ
ト社のOS，WindowsCEを搭載したPDAを各社が販売した（図
2･25）．
このころのPDAは機能こそ現在のベースとなるものが搭載され
ていたが，処理が重くなかなか軽快な使い勝手とはいかなかった．
その中で，Newton MessagePadは1998年2月に開発中止が発表
され，しばらくPDAの話題は落ち着いた．その後，OSやCPU
などに改良が加えられ，また，半導体集積回路の急速な技術発展に
より，その使い勝手は向上していった．2000年になり，再びUS
ロボティクス社を中心にPDAの話題が上がってきた．ハードウェ
アの性能向上もさることながら，ソフトウェアの操作性も向上し，
その実用性を増してきた．CPUはインテル社のStrongARM SA-
1110206MHzが広く使われるようになった．アーキテクチャの改
良と動作周波数の向上は継続しており，StrongARMの後継である
XScaleプロセッサを搭載したPDAが現れた（図2･26）．
このような半導体集積回路の発展は，PDAの新たなアプローチ
を提供することになった．それは，従来，搭載機能を必要最小限に
絞り小形化を図るアプローチであったが，パーソナルコンピュータ
そのものを小形化するアプローチである．このようなコンピュータ
はマイクロPCやUPC（ウルトラPC）と呼ばれる．その一例が，
OQO社が開発したOQOウルトラパーソナルコンピュータ（図2･
27）である．OQO社は，アップルコンピュータ社のPowerBook
の開発に携わっていたジョリー・ベルが1999年に設立した会社で
ある．これは，サイズ（124×86×23mm，重さ397g）こそPDA
であるが，その機能，性能はパーソナルコンピュータそのものであ
る．おもな仕様は，OS:Windows XP，CPU:Transmeta-
Crusoe TM58001GHz，ディスプレイ:5インチワイドVGA
LCD，HDD:20GB，メモリ:256MB，I /F:USB1.1×1，
IEEE1394 ×1，オーディオ出力×1，有線・無線LANである．
このようにPDAの世界でもパーソナルコンピュータの利用領域
を凌駕する傾向が現れてきた．
d.携帯電話（cellular phone) 携帯して移動しながら電話が
かけられる電話機が出たのは1970年代後半である．このころは，
携帯といっても大きなショルダーバック位の大きさであり，重さも
2～3kgはあった．それが，やはり半導体集積回路の発達により，
ようやく片手でも持てるサイズになったのが，1980年代後半であ
る（図2･28）．
その後，携帯電話のサイズはますます小さく，それに伴い重さも
ずいぶん軽くなった．また，通信技術の発展も急速に進み，音声通
話中心であった初期の携帯電話に比べ，データ通信が行えるように
なった．さらに通信方式もPDC（personal digital cellular ）方式
からW-CDMA（wideband code division multiple access ）方式
へと，より高速なデータ通信が行える方式に移行が進んでいる（図
2･29）．
このようなハードウェアの発展は，携帯電話の利用形態も変えて
きた．音声のやり取りだけでなく，Webの閲覧，メールのやり取
り，双方向のTV電話さえ扱えるようになった．また，共通のOS
を搭載することにより，簡単なアプリケーションも携帯端末で操作
可能になっている．このような傾向は，今後，携帯電話の利用領域
がPDAの利用領域へと拡大していく予兆である．
