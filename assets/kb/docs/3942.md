# 10・9・4 販売 従来,知的財産の所有は技術発表や新製品発

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・9 技術開発,製品開発,製造,販売における知的財産

売時などに自社技術の高度さ,先進性を示したり,いかに多くの出願で保護されているかを知らせるために発表されてきた.しかしながら,その時だけであり,それだけでは世間に周知徹底されるまでにはなかなか至らなかった.模倣者には,権利を知らずにフルコピーするもの,権利を承知でフルコピーをするもの,権利を承知で多少の変更,場合により改悪,をして非侵害を主張するものなどがいる.他社に模倣をさせないためには,自社所有の特許権・意匠権・商標権等を必要のつど発表すると共に,これを侵害するものには断固たる処置をすることを言明しておくことが,進展国で見られるようになってきた.また,日本への輸入を水際で阻止する為に,新聞等に発表することも有る.外観から見て直ぐ分かる,商標権,意匠権の侵害は発見しやすいが,特許は外観で分かるのは極めて希であるため,分解検証をしないと侵害が特定できない.侵害監視には,関係者に自社商品がいかなるものかを教育することが必要であり,それに類似する他社商品を知財部門に通知する仕組みを構築するとよい。
