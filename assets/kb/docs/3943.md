# 10・9・5 技術提携 技術の高度化,広汎化,開発費用の高額

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・9 技術開発,製品開発,製造,販売における知的財産

子等から,一社だけですべての必要な開発をするには限度が出てきている.そのため,技術提携,・技術援助,.共同開発,委託開発などが行われている。企業は需要の先取りのために,市場動向の的確な把握をし,同業者に先行する開発をしなければならない.とりわけ,環境問題など地球規模で非常に大きな技術課題が提起されている昨今では,世界規模での標準技術(グローパルス:タンダード)やデファクトスタンダードを確立することが,技術戦略上優位に立つことができると共に安定した実施料収入にも繋がるようになる.その成果の取り扱いについては,技術提携などの契約締結前に十分に議論することが将来の問題発生を回避する上で重要である.
