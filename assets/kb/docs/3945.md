# 10・10・1 契約の概要 契約とは、一定の法律効果を発生させ

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・10 契約

るために複数当事者が対立する意思表示を合致させた合意により成立する法律行為をいうとされている。当事者は、契約をするしない、相手方の選択、契約内容及び方式について、法の制限に触れない限り自由に決定できる。知的財産権に係わる契約の作成にあたっては、特許法、実用新案法、意匠法、商標法をはじめとして、民法、民事訴訟法、独占禁止法、不正競争防止法、印紙税法等に代表される法令に十分な注意が必要である。契約内容の錯誤の排除、記録、保存のため、また後日の契約違反の予防、別契約との矛盾、第三者への提示等のために合意内容を契約書として取り交わしている。契約は契約当事者間の権利と義務を規定するので、契約当事者が特定しなければ遂行できない事項は必ず取り決める必要がある。契約に係わる書面として、契約書中の字:句の解釈について合意する文書としての覚書や、一方的に意思表示を相手方に示す念書、差入れ書などがあるが、法律的効果に差違はなく厳密な履行が要求される。
