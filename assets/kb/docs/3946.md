# 10・10・2 契約に規定する事項 各契約に共通の事項として,

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・10 契約

以下のような事項がある.a.前文’契約当事者名,契約対象に加えて,契約に至る背景,経過,目的などの説明事項を記載する場合もある.b.秘密保持 当事者双方が秘密として管理すべき情報につい・て秘密保持義務を記載する.c.不可抗力 契約当事者の責に帰することのできない事由が生じたときの扱いを記載する.d.紛争の予防と処理 契約後の解釈の食い違いを当事者間で解決できないときの協議とその不成立時の処理を記載する.e.損害賠償 契約当事者の一方が自己の責任において債務不履行に至った場合の損害賠償について記載する.f権利移転 契約当事者の一方が第三者に権利を移転する譲渡と,相続や合併による承継について記載する.g.契約内容の変更 契約履行中における事情の変化に従い,合理的な内容に変更する際の要件について記載する.h.一部無効 契約の無効原因により,全部無効と一部無効となるが,一部無効の際の解釈,扱いについて記載する.t合意 契約書の作成途中の予備的な合意よりも,締結された契約書の合意内容が優先することを記載する.j.通知 当事者間の意思表示や報告などの手続に関する必要事項を記載する.k.解約・解除 当事者の一方からの意思表示により契約終了する際の債務不履行及び事情変更などの理由を記載する.解約は遡及しないが,解除は契約が初めから無かったとされる.1.契約有効期間 契約の権利義務の発生と終了の始期と終期を記載する.契約書の始期は締結日と違うことがある.m.契約終了時の措置 期間満了や解下等による契約終了後にも権利内容が継続するような事項の取り扱いについて記載する.特に秘密保持,開発成果の取り扱いは重要である.n.協議 契約書に定められていない事項について,適用可能な法規に基づき処理するか,その処理に疑義があるときの当事者間の処理を記載する.o.後文 契約書締結と所在を明確にし,日付,当事者を記載する.捺印又はサインがなされる.
