# 10・10・3 知的財産に関する契約の概要

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・10 契約

a.秘密保持契約 経営、技術、営業等の秘密情報を一方の当事者が開示するに当たり、相手方に守秘義務を課すための契約である。開示する当事者は営業秘密については不正競争防止法に従う管理を実施していることはもちろん、開示を受けた者も同様に管理することが求められる。口頭による開示事項を含め開示された秘密情報は書面で残して、特定できるようにしておくことが望ましい。なお、将来の共同研究や共同開発の可能性を評価するために工場見学や新技術の説明を受ける場合には、将来の秘密保持義務違反を問われないように、見学や説明を受ける事項には秘密情報が含まれていない旨の書面を開示者から提出してもらうこともある。b.共同研究契約 異なる技術を所有する者が新技術を生み出すために共同して研究活動を行うための契約である。契約当事者として民間企業や大学等の公的機関がある。契約の対象として、新技術開発のための基礎研究、応用化研究、工業化研究、新用途開発研究、既存技術改良研究などがある。契約前には、相手方の技術力、研究成果の事業化、独占禁止法抵触の可能性等について十分な調査が必要である。c。研究委託契約 一方の当事者が委託者として研究テーマと費用を提供し、他方の当事者が受託者として研究活動をするための契約である。契約当事者が国や公的機関の場合には、そこが所有するβ9-225契約書に従うことが多い。契約では、費用提供を理由に委託者に成果が帰属する規定を設けることが一般であるが、受託者が不利益を受けないような注意が必要である。d.特許共有(共同出願)契約 特許や実用新案を二者以上の者が共同して出願したり、その後の管理について取り決めるための契約である。出願拠出願人が単独でも、その後権利の一部を譲渡することにより共有とする手続が可能である。この契約では費用負担、出願手続き等の具体的作業について規定が必要である。e.特許権(出願権)譲渡契約 契約の一方の当事者が所有する特許権、実用新案権、意匠権及び特許を受ける権利、実用新案登録を受ける権利、意匠登録を受ける権利を他方の当事者に譲り渡す契約である。,、f特許実施許諾契約 当事者の一方が所有する特許権について他方の当事者に実施権を許諾する契約である。実施権は、特許権者でも実施ができずまた別の第三者に実施許諾できない専用実施権と、一般に特許権者の実施が可能でありまた別の第三者に実施許諾できる通常実施権に大別される。通常実施権でも、特約として別の第三者に実施許諾しない条件、更には特許権者の実施もできない条件を付すこともある。専用実施権者及び通常実施権者は許諾者の同意が得られれば、実施権者自身が第三者に通常実施権を許諾できる。許諾の範囲は、自身が許諾を受けている実施権の範囲に限られる。実施権は設定登録が第三者対抗要件なので、手続についての協力義務を明記しておくのがよい。企業としての財産の許諾と自己実施の制約から、長期的に利害得失を十分に検討し、許諾の範囲とその対価に反映させる必要があるg.ノウハウ実施許諾契約 当事者の一方が秘密に管理している技術情報、営業情報、コンピュータプログラム等のソフトウェアに係わる著作権等の情報(ノウハウ)を他方の当事者に許諾する契約である。独占的に又は非独占的に実施する権利を与えることを規定できるが、公開を前提とされる特許と異なり秘密性が高いほど価値が高いため、再実施権の許諾を規定することは少ない。同じノウハウでも、許諾する相手の技術水準やそれに伴う許諾範囲の差により対価が変わることがあり、また当事者の責に帰することのできない第三者の開示による公知化等により対価の再評価が必要になることもある。ノウハウ実施許諾において、地域・使用分野・数量・時間的な実施の制限を規定する場合には独占禁止法に十分な配慮が必要である。特許権とノウハウを併せて一つの契約とすることもあるが、有効へ期間が法的に定められている特許権と異なり、契約後の市場における技術水準の向上に対し経時的な価値の低下があるとしても、ノウハウは秘密に管理されている限り法的な有効期間はないので、特許権が失効しても価値が残る場合があり、契約期間については十分検討が必要である。h.クロスライセンス契約 契約の両当事者が所有する特許・実用新案又はノウハウなどを、一つの契約書で相互に同時に実施許諾をする契約である。実施許諾の対象となる特許等が相互に複数ある場合を、パッケージクロスライセンス契約と呼び、一般的に特定技術領域に属する特許群を相互に許諾することが多い。当事者が二者以上で、互いに所有する特許権等を許諾し合うパテントプールもこの契約の一態様と見ることができる。パッケージクロスライセンスでは、対象の技術分野や事業分野で不当に競争を制限する恐れがあると独占禁止法に抵触する可能性があるので、注意が必要である。クロスライセンスでも、その特許権の価値を等価とみなして金員の授受がない場合と、価値に差があるとしてその差を金員で補填する場合とがある。i.オプション契約 共同研究契約、特許実施許諾契約、ノウハウ実施許諾契約等の技術契約の締結に先立ち、一方の当事者から相β9-226手方に対象技術を評価するために必要な情報を提供し、一定期間内に契約を締結するか否かの選択権(オプション)を与える契約である。将来の技術契約の基本条件を明確にしておくと共に、相手方の評価結果で契約締結が不成立になることもあるので、ノウハウの流出を防止する規定を設けるが、秘密保持契約にオプション条項を定めるやり方もある。」。請負契約 当事者の一方(請負会社)が請負業務の完成を約し、他方(発注会社)がそれに対して報酬を支払うことを約する契約である。知的財産関係では、製作物供給契約が重要である。この場合、請負会社は発注会社の指示通りに生産を行う機関と位置づけられ、研究開発活動は一般的にないものとしている。しかし製造工程の改善などの新規なまたは改良の発明が生れる可能性もあり、その取り扱いを規定しておくのがよい。一方、請負業務は発注会社の設計・仕様等によるものであるから、発注会社が所有する特許権等のライセンス契約は不要である。ただし、その成果(製品等)を第三者に提供する場合にはその締結が必要となる。k.購買契約 一方の当事者が発注者として、他方の当事者(受注者)の設計・仕様による市販製品または受注生産用の製品を購入する際に締結する契約である。これらの製品に係わる特許権等の侵害については、受注者が保証すべきであり、そのような規定を一般に設けている。しかしながら、特許此等を有する第三者は、部品メーカや製造装置メーカより最終製品の製造・販売会社を特許権等の権利行使先とする傾向にあり、特許保証条項の作成に当たっては十分な注意が必要である。権利の対象が一部品であっても製品となった時の価値を基準に、より高額のライセンス料を求められることがあるが、その発注者の係争・訴訟における費用を当該部品の利益では受注者は賄えない可能性があるためであり、保証額に上限を規定することもある。L商標権(出願権)譲渡契約 一方の当事者の所有する商標権や商標登録出願により生じた権利を他方の当事者に譲渡する契約である。商標は指定商品及び商品の区分又は指定役務及び役務の区分に基づいて権利が付与されるが、譲渡が必要とされる範囲におけるその権利の評価を十分に行い、適正な価格で譲渡が成立するように交渉する。商標権は使用を継続している限り、何度でも権利の更新が可能で’ありその価値も向上するが、定められた期間不使用なら取消審判により取り消されることになるためである。m.商標使用許諾契約 商標権者が所有する登録商標を使用する権利を、その使用を望む者に許諾するための契約であり、専用使用権又は通常使用権の許諾に大別される。商標調査で使用を予定している商標と同一又は類似の先登録商標を発見したり、出願中の商標について先登録商標に基づく拒絶理由を受けたり、商標登録異議申立てを受けたり、侵害警告または訴訟を受けたりした等の場合、その商標を使用したいときに締結する。n.商品化権契約 一般には漫画やテレビなどに登場する人物、動物(キャラクタ)の絵や工業製品の外観等をテレビゲームを含む玩具、文具、衣類等の商品に応用してなんらかの経済的価値を上げる権利(商品化権)を許諾する契約である。意匠権、商標権、著作権等の個別契約とせずに、一つの商品について、許諾者が所有する権利を一括して許諾するような効果を表す。
