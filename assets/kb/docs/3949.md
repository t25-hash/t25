# 10・11・2 ハーモナイゼーション

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・11 知的財産の動向

a．特許の国際調和 1623年イギリスの専売条例によって確立
された特許制度は，その後フランス，アメリカに採用され，19世紀
に入るやオーストリア，ロシア，等々と相ついで採用されていった．
しかし，これら各国の特許法は当然のことながらその内容は大なり
小なり異なっており，運用もまちまちであった．
時代の推移に伴い，国際間の交流が活発になるにつれ，各国の特
許制度を基本的に統一しようという声が高まった．しかしながら多
くの国の利害が対立したために統一特許法の実現はならなかった．
次善策として生まれたのが，1883年パリで締結された「工業所有権
保護同盟条約」，別名「パリ条約」と呼ばれるものである．日本は1899
年（明治32年）にこの条約に加入した．1956年には長い間孤立主義
をとってきた旧ソ連（現在のロシア）もこの条約に加盟し，1998年
現在における加盟国数は約150ヶ国に達している．
その他にも後述する種々の条約等が締結され，検討されてきてお
り，最終的には全世界統一特許法の実現に向け進んでいくと思われ
る。
