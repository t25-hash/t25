# 10・11・3 アメリカにおける知的財産権の価値評価の動き こ

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・11 知的財産の動向

れまで特許権の活用とは、権利行使かライセンス収入が主流であった。一方で知財活動の評価のために、投資対効果の観点から、さまざまな評価方法が提案されてきたが、これといったものはでてこなかった。近年企業の価値を技術力やノウハウの結集された知的資産で量ろうとする動きが現れてきた。アメリカでは無形の資産も将来どの程度のキャッシュフローを生み出すかで資産価値を決めようとしている。アメリカの会計基準を決めるアメリカ財務会計基準審議会(FASB)は無形資産の時価評価を促進すべく、企業がM&Aで外部から資産を購入したとき、従来の簿価評価を時価評価することに切り替えた。買収価格と資産価値の差額をのれん代として無形資産に計上する際には、知的資産を時価評価した価格を記載することになる。その時価評価は将来キャッシュフローから算出されることになるが、国際会計基準では企業が自ら創り出した無形資産も時価評価することを取り決めている。今後知財活動の評価がこのような時価評価でなされる可能性があるが、金融工学の手法を取り入れたやり方が提案されている中で、その時価評価をいかになすかについてはこれからの動きに注目したい。知的財産関係の国内の主要な機関の概要を以下に紹介する。
