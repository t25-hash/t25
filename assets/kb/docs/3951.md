# 10・12・1 特許庁(JPO) 特許庁は総務部・審査業務部,特許

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・12 知的財産関係機関,諸団体とその機能

審査第一部から第四部・審判部等の部門を有し,世界各国からの出願の受付と審査官による独占権付与の判断を行い,また審査結果に対する不服に対しては審判官による審理を行っている.その他の主たる業務として,工業所有権施策の企画立案,国際的に調和のとれた工業所有権の保護といった国際交渉,日米欧先進国間情報化協力や発展途上国への人材育成・審査協力,工業所有権の制度改正・運用改正,電子公報の発行や特許電子図書館(IPDL)サービス等工業所有権情報の普及を行っている.
