# 2・10・1 膜分離（membrane separation）の種類

> **章**: 第2章 化学機械
> **節**: 2・10 膜分離装置

しかし払い落とし効果が弱いのではく離性の良いダストに用いられる．
パルスジェット式：圧縮空気をろ布上部から間欠的に極短時間・噴出する払い落とし方式であり，払い落とし中も気流を停止する必要はない．
このため風量変動が少なく高濃度排ガス処理に適し，装置の小型化も可能である．
セラミックフィルタ（ceramic filter) ろ材としてセラミックを用いたものであり，バグフィルタと同様織布にしたものと，多孔質焼結体やセラミック繊維成形体などがある．
いずれも高温高圧集じん用に開発されたものである．
いずれも捕集性能に問題はないが，引張強度などの機械的特性および寿命などが加熱により膜分離プロセスは，省エネルギーであること，コンパクトな装置で運転が容易であることなどの特徴を有するために，産業界で徐々に実用化が進んでいる．
膜分離プロセスは，分離対象の相，駆動力の種類に応じて分類されており，便覧 や成書 に膜分離法の詳細がまとめられている．
圧力を分離駆動力とする液相系の膜分離プロセスを表2･8に示すが，細孔径に応じて，精密ろ過（細孔1nm）に分類されており，操作圧力も異なってくる．
分離したい物質に応じて，適切な膜分離プロセスをまず選択する必要がある．
