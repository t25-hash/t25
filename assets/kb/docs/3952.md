# 10・12・2 日本弁理士会 弁理士は1899年特許代理業者登録規

> **章**: 第10章 知的財産に関する法制度
> **節**: 10・12 知的財産関係機関、諸団体とその機能

則に基づく特許代理業者の登録に始まり、1909年特許弁理士と改称され、1921年に弁理士法が制定され、翌年弁理士会が設立された。弁理士は、国家資格者として国内における工業所有権の出願や拒絶・異議申立て・審判等の特許庁に対する手続きを行う。外国における工業所有権の取得・対応も行う。また、権利範囲に関する鑑定、審決取消訴訟における訴訟代理人や権利侵害の訴訟における補佐人も行う。工業所有権制度の専門家として種々の助言を求めることができる。
