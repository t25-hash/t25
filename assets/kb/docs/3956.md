# 2・11・2 流動層触媒反応装置（fluidized-bed reactor ）

> **章**: 第2章 化学機械
> **節**: 2・11 固体触媒反応装置（catalytic reactor ）

工業的に多くの重要なプロセスに反応物を気体とした気固流動層
が利用されている ．層内の温度をほぼ均一に制御できることや，
触媒粒子の連続的な供給や抜き出しが可能であることが長所とな
る．温度制御が比較的容易であるため，反応熱が非常に大きな反応
に対して利用されることもある．図2･42に示すように気泡として
層内を抜けていくガスが多いため，反応収率は固定層に比べると小

さくなる点が短所となる．また，良好に流動化させるための層高や
ガス速度に範囲があるために，操作範囲が制限される．
気固流動層に使用される触媒は大きさが60m程度と非常に小
さい．気泡とエマルション相（触媒粒子相）との間の物質の移動速
度が見掛けの反応速度に影響するため，小さな気泡をよく分散させ
た流動化状態を達成することが重要なポイントである．そのために
ガスを均一に分散させるような分散器の設計や，良好な流動化状態
を得るために適切な物理物性をもった触媒粒子の選択も必要であ
る．また，粒子の飛び出しも起こるため，反応装置内部もしくは外
部にサイクロンなどを設置して粒子を捕集しなければならない．ま
た，層内の気泡の分裂を促進するために，装置内部に各種の内挿物
（インターナル）を設置することもある．
触媒を連続的に供給・排出することが可能であるということも大
きな特徴であり，石油の接触分解プロセス（ガソリン留分の製造）
に生かされている．この反応では，触媒活性は非常に高いが，劣化
速度も速い．固定層反応器では反応操作を止めて再生操作を行う必
要があったが，流動層反応器を採用し，図2･44に示すように反応
塔と再生塔の間で触媒粒子を循環させることによって，連続的な反
応操作が可能となった．最近の装置においては，反応部はライザと
呼ばれる形式で，空塔ガス速度は10m/ s以上で操作されることも
あり，触媒粒子が気流によって運ばれる状態である．この部分での
反応は数秒で終了する．このように，劣化が大きいといった触媒の
短所を装置によって補うことができ，数万t /dといった大規模な装
置が精油所では稼働している．
