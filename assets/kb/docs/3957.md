# 2・11・3 気泡塔型反応装置とかくはん槽型反応装置

> **章**: 第2章 化学機械
> **節**: 2・11 固体触媒反応装置（catalytic reactor ）

液体中に気体を吹込み，気液の反応を行わせる装置は気泡塔型反
応装置と呼ばれるが，これに固体触媒を添加した形式もある．この
場合には触媒粒子を液体中に懸濁させて使用するため，懸濁気泡塔
と呼ばれる．また，液で固体触媒粒子を流動化したところに気体を
吹込む三相流動層という形式もある．
油脂の水素添加反応では粉末ニッケル触媒を油脂に分散させて水
素を吹込むことによって，硬化油（マーガリンなどの原料）を製造
する．未反応の水素を循環させて使用するような懸濁気泡塔の形式
で操作されていたこともあるが，反応する量だけ水素を供給し機械
的なかくはんで液相に吸収させる方式が現在は使用されてい
