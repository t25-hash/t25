# 2・11・4 反応装置の伝熱方式

> **章**: 第2章 化学機械
> **節**: 2・11 固体触媒反応装置（catalytic reactor ）

化学反応においては反応熱が発生する．吸熱反応であれば加熱
を，発熱反応であれば除熱を適切に行わなければ，反応温度を一定
値に制御することができず，反応を円滑に進行させることができな
い．そのため，伝熱方式の選択が反応装置形式の選択を決定するこ
とさえある．
通常は反応装置内部に伝熱管を挿入して，熱媒体を循環させるこ
却効率を低下させることがある．
とによって熱の供給や除去を行う．また，反応熱が大きくない場合
には反応器の周囲にジャケットを設け，熱媒体を流すことによって
温度を制御することもある．反応装置を周囲から断熱した状態で操
作することもある．重質油の水素化脱硫がそれにあたる（図2･43）．
図2･43の場合には途中で水素を吹込むことによって冷却を行って
いる．固定層反応装置では，内径数センチの小さな反応管を数千本
たばねて，その周囲に熱媒体を通す多管式反応装置として操作され
め，効率の良い冷却を行うことができる．
