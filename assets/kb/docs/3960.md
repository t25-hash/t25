a.かくはん槽型反応装置（stirred tank reactor ) かくはん
槽型反応装置は，均一液相反応をはじめとして，液-液反応，気-液
反応，気-液-固触媒反応などの不均一反応にも用いられており，適
用範囲が広い．液の粘度が低い場合の均一液相反応のかくはんには
特に問題はないが，重合反応のように粘度が高くなると，かくはん
翼の形状などに注意しないとかくはんの効果が上がらない．反応熱
の除去（または供給）には，図2･51に示すように，槽の外側にジ
ャケットを付けるか，あるいは内側にコイルを入れて，それらの中
にスチームあるいは熱媒体または冷媒を流す．
b.流動層型反応装置（fluidized- bed reactor ) 流動層は固体
粒子を槽の底部より送入した流体によって浮遊懸濁させながら流動
化させる装置である．流動化された固体粒子は，あたかも沸騰状態
にある液体のように挙動するので，固体の輸送，取出しが容易とな
り，伝熱能力も高い．この種の反応装置は石油の接触分解反応，炭
化水素の酸化反応，硫化鉱の燃焼，石油のガス化等に活用されてい
る．
c .移動層型反応装置（moving- bed reactor ) 移動層型反応
装置も気体または液体に固体（触媒）を接触させる反応装置で，固
体（触媒）粒子を装置上部に連続的に供給し，重力で垂直方向に緩
やかに下降させ，それに対して反応流体を向流，並流あるいは十字
流的に接触させて，触媒反応あるいは気（液）固反応を行う装置形
式である．
