# 1・1・1 調木、チップ(chip)ハンドリング機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

a. バーカ(barker)樹皮はパルプ分が少なく、パルプ化工程での歩留りの低下につながり、またパルプ中のちりの原因となるので除去する必要がある。バーカの選択は対象とする材のサイズ(径)によって決まる。. ドラムバーカ(drum barker)やや傾斜して回転するドラムの高いほうから丸太が投入され、ドラムの回転による丸太とドラムあるいは丸太どうしの摩擦と衝撃により皮はぎが行われ、他方から排出される。タンブリング型とパラレル型があり、パラレル型は長尺の丸太を対象とする。. リングバーカ(ring barker)大きなリングの内側にスクレーパを中心に向けて取付け、それを回転させ、中心に送り込まれる丸太表面に直接スクレーパを押付け皮をはぎ取る。丸太を1本ごとに処理するため、大径木ほど効率が高い。b. チッパ(chipper)チップ化の目的は、パルプ化工程での均一化であり、チップの厚さは4~10mm、幅は25mm前後、繊維方向の長さは15~35mmを標準としている。400~4450mmφのディスクをもったディスク型チッパが多く使用されている。2~16枚のナイフが取付けられた回転ディスクに30~60度の角度で材が供給される。ナイフの数が多いほど処理量が増加し、チップサイズも均一化する。チップはディスク型チップスクリーン等で処理され、粗大チップはチップスライサやチップクラッシャにかけ再びチップスクリーンで選別する。
