# 2・14・1 プロセス設計（process design)

> **章**: 第2章 化学機械
> **節**: 2・14 プラント，プロセス設計・制御

プロセスシステムは，高品質の製品を安全に効率良く製造すると
いう目的のために，反応器，蒸留塔，熱交換器など，種々の機能を
もった装置が結合された集合体である．設計時には，特定の条件下
での最適性だけではなく，市場の変化や性能の劣化などの不確定性
も考慮して，その柔軟性・拡張性を検討しながら，プロセスシステ
ム全体のありかたを判断し，各機器の設計を行うことになる．
実験室において，ビーカやフラスコといった装置で新たに発見，
開発された物質や反応経路が，実際に工業化されるのは，1～3%と
いわれている ．そのため，プロセス開発に関わる技術者が，工
