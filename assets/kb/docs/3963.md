# 1・1・2 機械パルプ(mechanical pulp)製造装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

a. SGP(stone groundwood pulp)SGPは高速で回転する砥石に丸太を押付け,シャワー水をかけながらすりつぶしパルプ化する,いわゆる砕木パルプである.グラインダ(grinder)には,丸太の供給方式,加圧方式の違いにより,図1・1に示すようなポケット型,マガジン型,チェーン(キャタピラ)型,リング型等種々の型式のものがある.パルプストーンとしてアランダム,カーボランダム製のセラミックストーンが使用されている.摩砕条件によって製造されるパルプの品質は異なるが,一般的に強度は低いが不透明度が高く,吸油性に優れ,新聞用紙,中級印刷用紙,下級印刷用紙などに使用される.b. PGW(pressurized groundwood)PGW はSGPを改良したパルプ化法で,高温のシャワー水を使用し加圧下で丸太を摩砕するパルプ化法である.密閉構造になっているグラインダチャンバの加圧は圧縮空気で行われ,摩砕条件は圧力2~3気圧,温度110~135°Cである.図1・2にPGW の製造工程を示す.加圧状態でかつ高温における処理はリグニンの軟化を促し,SGPより長繊維のパルプが得られ,SGPより強度が高くなる.不透明度はSGPとほぼ同等であり,優れた印刷適性を示す.PGW はグラインダを(a) ポケットグラインダ(b) マガジングラインダ(c) キャタピラグラインダ(d) リンググラインダ図1・1 グラインダのいろいろ1―1加圧容器にする必要があり,SGPに比べ設備費が高くなる.原木として丸太材を必要とするため,丸太材の豊富な欧米諸国にプラントの設置例が多く見られる.c. RGP(refiner groundwood pulp)RGPはチップやソーダストをリファイニングによりパルプ化する方法である.リファイニングにはディスクリファイナ(refiner)が使用され,一次リファイナで粗砕を行い,二次リファイナで精砕を行う.RGPは不透明度がSGPに及ばず,結束繊維が多く強度もそこそこのところから,次に述べるTMPが開発されて以来,新設はほとんどなくなっている.d. TMP(thermomechanical pulp)TMPはRGPの一次リファイニングを蒸気加熱リファイニングに代替したものである.チップウォッシャで土砂金属類を除去したチップを110~130°Cに蒸気で加熱し軟化させた後,加圧型リファイナで一次リファイニングし,次いで常圧型リファイナで二次リファイニングを行いパルプ化する方法である.図1・3にTMP工程の基本的な系統図を示す.リグニンのガラス転移点を超えない温度で加熱した後リファイニングするので,SGPやGPと異なり,繊維の破壊,裂けなしに繊維は一次壁と二次壁の間で開裂し,長繊維が生成され,リグニンが著しく着色することもない.繊維表面は親水化的な性質を示し,フィブリル化しやすいので,繊維間結合が強いパルプとなる.SGPやRGPの欠点である低強度性およびシャイブ含量の多いことを改善し,しかも機械パルプの長所である良好な印刷性をもつので,新聞用紙をはじめ,中級印刷用紙などに使用され,その優れた強度特性から化学パルプの配合減に寄与している.問題はエネルギー消費が大きいことである.表1・1にGPとTMPの品質,電力原単位の比較を示す.e. CTMP(chemi-thermomechanical pulp)CTMPはチップを薬品処理した後,加圧リファイニング次いで常圧リファイニングを行いパルプ化する方法である.すなわち,薬品で前処理したチップを使用するTMP法である.チップの加熱による軟化と薬品処理による軟化とがあいまって,パルプ品質の向上をもたらす.薬品としては,亜硫酸ソーダを使用するのが一般的であり,加圧リファイナの前に,薬品をチップに浸透させるためのインプレグネータを設けている.CTMPはTMPに比べ,長繊維含有量が多く,繊維図1・2 PGW の製造工程表1・1 TMPとSGW,PGW の比較SGWPGWTMPフリーネスmL電力消費量kWh/tAD繊維ふるい分け+28%15.129.030.0-200%32.033.528.0引張指数Nm/g32.241.735.0破裂指数kPam/g1.592.342.55引裂指数mNm/g3.505.856.80伸び%3.13.13.8白色度%66.062.961.3光散乱係数m/kg65.064.255.01―2の柔軟性が大きいため,強度が向上する.また,亜硫酸ソーダを使用しているので,未ざらしパルプの白色度が高く,過酸化水素さらしによって高い白色度のパルプが得られる.ただし,収率がやや低下し,不透明度もいくぶん低下する.CTMPは次に述べるさらしクラフトパルプに近づいた機械パルプといえる.シャイブが少なく,かさ高で柔軟で吸水性が高いことから,ティッシュ,タオル,フラッフへの利用,板紙への利用をはじめ,最近は印刷用紙,新聞用紙への利用も進んでいる.
