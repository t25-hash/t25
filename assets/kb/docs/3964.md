# 1・1・3 化学パルプ(chemical pulp)製造装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・1 パルプ製造機械

化学パルプの製造法は,クラフトパルプ化法がわが国ではもちろん,世界的に見てもパルプ製造法の主流になっている.その理由は次のとおりである.(1) 廃液からの薬品および熱回収法が完成している.(2) ほとんどすべての樹種に適用でき,また建築用材の廃材チップでも使用できる.図1・3 TMPの基本工程図図1・4クラフトパルプ化における連続蒸解装置(3) 蒸解時間が短く,操作も簡単で,連続・大量生産に適した装置の発展があった.(4) 強度の大きなパルプが得られ,製造コストが安い.a. 蒸解装置(digester)省エネルギー,省力対策および生産規模の拡大から,バッチ式蒸解釜に代わり,連続式蒸解釜が主流になっている.最近,エネルギー効率が改善されたバッチ式蒸解釜も設置されている.図1・4に連続蒸解釜(continuous digester)の一例を示す.チップは低圧蒸気で加熱されてから釜の頂部へ蒸解液とともに送られ釜に入る.釜内では,薬液の浸透,加熱(間接加熱式熱交換器による),蒸解,洗浄が行われ,底部から連続的に排出される.近年,脱リグニンの促進,収率の向上を図るため,蒸解液を釜の数箇所に分割添加できる方式の蒸解釜が開発されている(図b. エバポレータ(evaporator)蒸解釜から排出される廃液[黒液(black liquor)]はフラッシュした後,回収ボイラで燃焼しうる濃度まで濃縮される.多管式強制循環型に変わってプレート式の薄膜降下型エバポレータが主流になっている.濃縮濃度は65~75%程度である(図1・6).c. 薬品回収設備濃縮された黒液は,回収ボイラで燃焼され,スメルトの形になり,弱液を用いてディゾルバで溶解され緑液(green liquor)となる.緑液中には,ドレッグと称する不溶性の不純物があるので,緑液クラリファイヤで沈降分離する.緑液は,スレーカ,苛性化槽を経て,白液(white liquor,蒸解液)変換される.苛性化槽を出た白乳は,白液と石灰スラッジ(炭酸カルシウム)に分離される.クラリファイヤ,プレコートタイプのベルトフィルタ,加圧でディスクフィルタなどで行われる.分離回収された石灰スラッジは,ウラッジウォッシャ,フラッシュドライヤで処理された後,キルンで焼成され,生石灰になり,再びスレーカへ供給される.連続苛性化(recausticizing)工程の一例を図1・7に示す.
