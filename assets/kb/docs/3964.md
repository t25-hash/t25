# 2・14・2 プロセス制御（process control )

> **章**: 第2章 化学機械
> **節**: 2・14 プラント，プロセス設計・制御

市況による生産要求や原料の変化など，製造プロセスを取り巻く
環境はつねに変化しており，このような外乱や不確定要素の存在下
で，安全で効率的に，高品質な製品を製造することを可能にする技
術がプロセス制御である．
化学プラントは，多くの装置の結合で構成されており，多数（エ
チレンプラントでは4000以上）の変数が管理対象になっている．
そのため，多数の弁やポンプ，ヒータなどの操作量が用意され，望
ましい状態を維持するための制御系が構成される．例えば，製品流
量を2倍に変更しようとすると，ほとんどすべての操作端を変更し
なければならない．これらの膨大な数の操作量の適正値を設計する
のは，非常に困難な問題である．例えば，水道の蛇口を流量が2倍
になるように開くとすると，どれだけ蛇口をひねればよいかは簡単
には算出できない．水道の元圧変動も影響する．しかし，フィード
バックを基本にすると，一つの測定端の情報を見て，一つの操作端
をさわるというシングルループの組合せで，うまく連動する世界を
実現できる．蛇口の場合も，流量計がついていれば，その流量測定
値が目標値と一致するように蛇口を調整することは可能である．

積分制御を用いると，被制御量の変動は抑制され，圧力を安全な
値に維持することなどが実現される．そして，変動を打ち消すため
の操作が他の被制御量に対する外乱になり，その制御系が被制御量
を目標値に保つように操作をする．さらに，その操作がまた他の制
御系への外乱となることにより，被制御量が多数存在するプラント
においても，目標の状態を実現する操作が，シングルループの組合
せにより実現される．図2･55に，シングルループコントローラの
組の例を示した．この構成では，原料Bの流量の制御系に生産量
を2倍と設定するだけで，温度制御，圧力制御，組成制御などが連
動し，原料Aや製品Cなどすべての流量がその2倍の製造量を実
現するように定まる．原料Bの流量変化だけでなく，冷却水温度
など，さまざまな変動に対しても有効に働く．
基本的には，シングルループコントローラを組合わせ，うまく連
動するように構成するのがプロセス制御の基本であるが，もっと積
極的にうまく動くという状態を実現するために，より効率的な運転
条件をリアルタイムに求め，多変数制御を行うというモデル予測制
御（model predictive control ）も数多く導入されている．プロセ
ス制御は，ロボット制御等よりも，はるかに長い2秒や1分という
サンプル間隔で実現でき，計算機のハードウェアへの制約もあまり
ないので，1970年代からモデル予測制御が導入されている．エチ
レンプラントでは，被制御量600以上，操作量200以上という大規
模なプラントモデルを基にした最適化制御が導入されており，生産
量を最大にする条件を与えると，夏はコンプレッサ能力，冬は蒸留
塔の能力が制約となる最適運転条件で，安定な操業が実現されるよ
うになっている ．
プロセス制御では，ポリマ反応など，詳細なモデリングが困難な
現象が存在し，むだ時間＋一次遅れや，有限長のパルス応答モデル
という簡便な構造のモデルで近似して，モデル誤差の存在を前提と
した制御系設計を行うというアプローチが一般的に用いられる．
PID制御は，制御性能と調整パラメータの関係があまり敏感では
ないし，把握しやすいので，簡便なモデルにより初期設定を行い，
現場での再調整を期待することができる．また，モデル予測制御
は，ロバスト性が高いという特徴以外にもその速応性の調整パラメ
ータが用意されており，プロセスモデルがパルス応答モデルという
把握しやすい形式で与えられるので，やはり，現場での調整が期待
できる．そのため，PID制御とモデル予測制御が用いられること
が多い．
