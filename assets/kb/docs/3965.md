# 1・1・4 古紙パルプ(deinked pulp)製造装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・5 ).

わが国の古紙の利用率は年々伸びており,1999年には56%を超え,2005年には60%を目指している.世界に目を向けると,EU諸国ではすでに70%以上の古紙を回収している国があり,アメリカでも40%以上が回収されており,古紙は製紙原料の主要な位置を占めている.古紙のおもな用途を図1・8に示す.このように古紙が多量に使用されるようになったのは,環境・資源面,経済面からの社会要請のみならず,古紙処理設備,脱インキ薬品などの著しい技術革新によるところが大きい.古紙利用上の厄介な問題の一つは,異物・禁忌品の混入である.古紙中の異物の混入度合いが,古図1・5 MCC法のフロー図1・6プレート式薄膜流下型エバポレータ図1・7 連続苛性化工程1―3紙再生パルプの品質を大きく左右する.古紙処理の基本は,離解,脱墨,精選・除じんである.a. 離解(pulping)装置工程内に持ち込まれる異物は,精選・除じん工程での分離効率を考え,破砕・細裂化をできる限り抑制する必要があり,ドラムパルパ,高濃度パルパがおもに採用されている.ドラムパルパ(図1・9)は異物の細裂化が少なく,エネ図1・8 古紙の用途1―4産業機械・装置ルギー消費も小さいが,設置面積,設備費が大きいという特徴をもっており,連続的に離解できることから,大規模プラントに適している.一方,高濃度パルパ(図1・10)は設置面積,設備費が小さく,また,バッチ運転のため小容量の古紙離解にも効率的に使用できる.b. 脱墨装置. 水洗法本法は古くから行われてきた方法で,基本的には数m程度に分散させたインキをパルプスラリーから希釈,濃縮等を繰返して系外に排除する.洗浄には,1置換洗浄,2希釈・脱水洗浄,3プレス洗浄といった基本型があり,1では,バルブレスフィルタ(filter),サクションフィルタ,2ではエキストラクタ,デッカマシン,3では傾斜シックナ,スクリュープレスが使用される.水洗法は,用排水量が多いこと,微細繊維の留まりが悪いことなどから,わが国ではフローテーション法,折衷法が広く採用されている.. フローテーション(flotation)法鉱物の浮遊選鉱に用いられていた方法を脱墨に応用したものであり,親油性のインキを親油性の気泡に付着させ浮上分離する方法である.使用するフロテータは1空気を小さな気泡にしパルプスラリーと混合する,2浮上図1・10 高濃度パルパしたインキをフロスとして排出する役割をもつ.空気の取込み方式は,原料とエアを別々に供給し回転する散気管にて混合するロータリミキシング方式と原料の供給圧力を利用してエアを吸引し,ディフューザ部でエアを適当な大きさで分散させるエアインジェクタ+ディフューザ方式に大別される.フローテーション法の効率は,パルプ濃度,空気量,気泡径,起泡剤,滞留時間,pH と温度等の影響を受ける.高白色度の古紙パルプを製造する場合は漂白工程の前・後に脱墨工程を設置すると効果が大きくなる.図1・11に種々のフローテータを示す.c. 分散(dispersing)装置25~35%の高パルプ濃度の原料に,強い機械的せん断力を与え,難離解古紙の離解,ホットメルト,ワックス等の難除じん異物の分散用に用いられている.最近では,新聞のオフセットインキ,OA 古紙のトナー型インキのはく離に必須装置となっている.図1・12に示すようなニーダタイプとディスクタイプのものがある.
