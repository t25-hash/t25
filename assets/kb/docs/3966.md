# 2・2・2 段塔（tray column）

> **章**: 第2章 化学機械
> **節**: 2・2 吸収・蒸留装置

図2･6(ａ)に示すように，塔内がトレイ(段)で仕切られた構造の
塔を段塔という．液は上段からダウンカマー（downcomer ）を通
ってトレイに入り，トレイ上を流れて次のダウンカマーに入る．一
方，蒸気はトレイ上の孔あるいは蒸気上昇管からトレイ上に入り，
トレイ上の液中を気泡や気柱として上昇して上段に流入する．段塔
は，その設計法が確立している，スケールアップが容易である，高
圧下での操作が容易である，固体粒子を含む流体にも適用できる，
多段フィードが容易であるなどの特徴をもつ．その反面，充てん塔
に比べて，飛まつ同伴量が多い（効率の低下につながる)，圧損が
大きいなどの欠点もある．
