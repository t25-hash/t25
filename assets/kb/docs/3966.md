# 1・1・5 精選(screening)・除じん(cleaning)装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・5 )。

古紙には、種々の異物が混入している。異物の性状は、大きさ・形状・比重・表面性状など多様であり、この異物をできる限り細片化させず除去することが、古紙パルプの品質向上につながる。a. スクリーンパルパで離解後、できるだけ大きな形状のうちに異物を除去する粗選工程と細片化された異物を除去する精選工程がある。粗選工程のスクリーンに使用されるプレートは、大部分が丸孔(3~4mmφ程度)である。最近は除去効果を高めるため、スリットプレートを併用するダブルセパレータが開発されている。精選工程では、スリットタイプのプレートを用いた密閉型のスクリーンがおもに使用されている。パルプの処理濃度は、使用するプレートの目孔、スリット幅と処理する古紙原料によって決められるが、粗選工程で3~4%、精選工程で0.8~2%である。図1・9ファイバフローシステム(a) ディスク型(b) ニーダ型図1・12 分散機図1・13最新の漂白工程(蒸解工程を含む)(a) コマーサイバーセル(c) ホイト・アイエイチアイエコ-セル(e) 新浜ポンプ製作所ハイフローフローテータ(f) スエマック・ヘルベルグフローテータ(h) PDM(加圧型フローテーションセル)(i) 王子エンジニアリングOK 式フローテータ図1・11 各種フローテータ1―5(b) ブラック・クローソンIIM/BC フローテータ(d) ラモーマックフローテータ(g) バーチカルフローテータ1―6産業機械・装置b. クリーナ比重差を利用したハイドロサイクロンを使用し、無機物、プラスチック類を除去する。処理濃度は、低いほうが除去分離効率は向上するが、設備規模、設置面積を考慮すると0.2~1.2%程度である。重量異物を分離する重量クリーナとプラスチック類を分離する軽量クリーナとに大別される。
