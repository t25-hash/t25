# 1・1・6 漂白(bleaching)装置

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・5 ).

漂白の目的は,パルプに含有される着色成分(主としてリグニン,古紙においては染料も含む)を脱色あるいは除去することによって,用途に応じた白色度(brightness)レベルのパルプを製造することである.機械パルプ,高収率パルプ,古紙脱墨パルプでは,着色成分を残して脱色することに主眼がおかれるため,過酸化水素,ハイドロサルファイトによる漂白が一般的である.化学パルプの主流であるクラフトパルプの漂白では,着色成分を除去することに重点がおかれており,多段漂白法が採用されている.酸素処理(O)を最初に行った後,塩素処理(C),アルカリ処理(E),次亜塩素酸塩処理(H),二酸化塩素処理(D)を組合わせて行っていたが,近年の環境問題の高まりにより,塩素処理(C)は減少の傾向にある.最近ではオゾン処理(Z)や酵素処理(Ez)といった処理段も組込まれるようになってきた.各段の処理後,パルプはドラムフィルタあるいはディフューザによって洗浄される.図1・13に最近の化学パルプ製造工程に組込まれた漂白設備フローを示す.
