# 1・2・1 紙料調成装置(stock preparation)

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・2 製紙機械

調成工程は抄紙機に原料を送る直前の工程で、離解、叩(こう)解、除じんおよびパルプ、填(てん)料、薬品類の配合など、製品となる紙の基本的な品質を決定する重要なところである。図1・14 パルパ図1・15 濃度と離解度の関係a. パルパ(pulper)パルパはパルプ、古紙および損紙の離解に用いられるもので、構造は一般的に図1・14のごとく槽およびインペラにより構成され、槽内にパルプ、古紙などを投入し、水中で離解させる装置である。装置を大別するとインペラが側面に付いた横型と、底面に付いた縦型とに分かれる。処理方式としてバッチ式と連続式、さらに離解濃度により低濃度型(4~6%)と高濃度型(15%程度)とがある。低濃度型はパルプ、古紙などの処理に使用され、高濃度型はおもに新聞脱墨パルプ用に使用される。離解作用は図1・15に示すように10%以上の高濃度域ではおもに流体せん断力による繊維どうしの摩擦力で行われ、6%以下の低濃度域ではインペラとステータ(固定部材)間で生じる摩擦作用が主体となることから、それらに適したインペラ形状および槽形状が工夫されている。耐水紙などの難離解紙は機械的な引裂き・せん断作用と摩擦作用を利用した低濃度域での処理が有効であり、インペラのベーン数増加、ストレーナ形状の工夫のほか、図1・16のようにディスクを組合わせることで離解性能の向上を図ったものがある。b. リファイナ(refiner)リファイナはパルプの叩解を行う装置である。叩解により木材繊維は水に膨潤、柔軟になり、フィブリル化が促進され繊維間結合が行われやすくなる。したがって、叩解は紙の品質に直接影響を与える製紙工程上重要な要素である。叩解を進めると、図1・17のように引張強さは増加していくが、引裂き強さは減少する傾向にある。これは叩解が進むに従い、繊維が膨潤するとともに繊維の切断が進み、繊維長が短くなるためである。リファイナの形式はディスク型が一般的であり、図1・18のような刃をもつディスクを対向させ、一方を回転し、その間に原料を流し、叩解を行う。固定ディスクと回転ディスクが一対のものと、回転ディスクの両面に刃を設け二対としたダブルディスクリファイナがある。図1・19にダブルディスクリファイナの一例を示す。最適な紙品質を得るための叩解条件は、原料の種類に応じ、刃幅、溝幅、刃の交差角などのディスク形状、ディスクの周速、押付け圧および原料濃度、処理量など多数の因子により決定される。一方、小規模の叩解を行う設備としてビータ(beater、図1・20)がある。この装置では、回流型の水槽に円周に刃を植えたドラムを回転させ、その下方の固定受刃との間に原料を通し叩解を行う。c. クリーナ(cleaner)パルプ中にはスクリーンでは除去できない砂などの重量異物、フィルムなどの軽量異物が含まれており、これらを除去するのがクリーナで、スクリーンの形状分離に対し、遠心力と流体せん断力を利用した比重分離により、異物をパルプ中から除去する。重量異物除去用、軽量異物除去用、および両者を同時に除去するものがあり、また、低濃度型と高濃度型とがある。図1・25,1・26に一例を示す。サイクロンを応用した比重分離のため、処理濃度が上昇するにつれ精選効率は低下する。
