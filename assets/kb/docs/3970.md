# 1・2・3 仕上機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・50 にツードラムワインダを示す。2本のロール上に載せられた巻

a. スーパカレンダ(super calender)非常に高い光沢と平滑を出す目的で、図1・52に示すスーパカレンダが用いられる。この装置では、硬質金属ロールと軟質ロール(コットンロールなど)が交互に配置され、紙は上方から下方に各ニップを通過する。ニップ下では軟質ロールが変形して金属ロールと相対運動を起こし、紙との摩擦が起こるために表面の平滑が非常に高くなる。b. カッタ(cutter)カッタでは、リールで巻取った製品を連続して縦横寸法どおりに平判状に裁断する。オンマシンとオフマシンの2種類があり、オンマシン用はおもに紙厚の厚い板紙に用いられる。図1・53にカッタの一例を示す。巻き解しスタンドを出た紙はスリッタで幅方向に切れ目が入れられ、回転ナイフによって流れ方向に裁断される。平判の長さは回転ナイフの速度によって決まる。最近では、巻き解し後に欠陥検出器を設置して、欠陥のシートを自動的に排出するシートソータ機能を付加したカッタも採用されている。c. 包装機平判の包装は図1・54に示すような流れで処理される。カッタで裁断され積上げられたシートはコンベヤ自動包装機に送られて包装される。
