# 2・3・5 電気再生式脱イオン装置EDI （electrodeionization）

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

原理を図2･12に示すが，陽イオンおよび陰イオン交換膜で仕切
って交互に配列した希釈室と濃縮室に，外部電極により電位こう配
を形成させた電気透析装置において，希釈室に陽イオンおよび陰イ
オン交換樹脂を充てんしたものである．希釈室に供給された原水中
のイオンはイオン交換樹脂に捕捉されるが，これは電位こう配の方
向に樹脂中を移動，イオン交換膜を通過して濃縮室に移行する．こ
うしてイオン交換樹脂は薬剤をいっさい必要とせずに再生され，希
釈室からは連続的に脱イオン化された純水が得られる．
とが必要である．
