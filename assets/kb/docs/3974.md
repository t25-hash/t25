# 1・2・5 塗工装置の種別

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・50 にツードラムワインダを示す.2本のロール上に載せられた巻

a. ロールコータ(roll coater)ロールコータは,コンパクトな構成で紙の表裏両面に同時に塗工できること,塗工部での断紙が生じにくいことなどから,サイズプレスおよびオンマシンコータとして用いられる.紙塗工で用いられるロールコータは,2本の圧接したロール間に形成した塗工液溜り(pond)に原紙を通して塗工する2本ロール方式(pond press)と,あらかじめアプリケータロール表面に形成した塗工液膜を原紙に転写するフィルムコータ(film coater,filmpress)に大別されるが,近年では塗工量制御性と高速操業安定性等に優れる下記のフィルムコータが多く採用されている.. ゲートロールコータ(gate roll coater)図1・60に示すとおり,本体は表裏計6本のロールで構成される.塗工液は回転するファウンテンロール(fountain roll)とメタリングロール(metering roll)のニップ部に供給され,両ロール間のニップを通過した後,メタリングロール上の塗工液がアプリケータロール(applicator roll)を経て原紙表面に転写される仕組みとなってい図1・59熱風ドライヤ1―13る.なお塗工量の制御は,一般にアプリケータロール周速に対するファウンテンおよびメタリングロールの周速比と,塗工液またはサイズ液の濃度によって行われる.. ロッドメタリング式フィルムコータ(rod meteringcoater)図1・61に示すとおり,本体は2本のアプリケータロールと二つの給液ヘッド(coater head)で構成される.給液ヘッド内に設けられたスリットからアプリケータロール表面に供給された塗工液は,アプリケータロールに押付けられたメタリングロッド(metering rod)で所定の液膜厚さに計量された後,2本のアプリケータロール間のニップ部で塗工原紙に転写される.塗工液膜厚さを計量するメタリングロッドには,表面にスパイラルな溝を形成したグルーブドロッド(grooved rod)と表面が平滑なスムーズロッド(smooth rod,plane rod)がある.スムーズロッドの場合における塗工量の制御は,アプリケータロールに対するメタリングロッドの押付け圧力とアプリケータロールのゴム硬度および塗工液濃度もしくは粘度によって行われる.一方グルーブドロッドの場合には,溝のピッチ,深さ,形状,および塗工液濃度によって塗工量を制御する.b. ブレードコータ(blade coater)ブレードコータは塗工液の供給部(applicator),仕上げブレード部(coater head)とバッキングロール(backing roll)から構成され,アプリケータで最終的な塗工膜厚より厚めの液膜を原紙表面に転写した後,仕上げブレード(finishing blade)で所定の塗工膜厚に計量するとともに,塗工面を平滑に仕上げる塗工方式である.塗工膜厚の制御は,バッキングロール上の紙に対する仕上げブレード押付け圧力によって制御される.仕上げブレードには,おもにSK 材が用いられているが,最近では摩耗寿命の長いセラミック溶射またはクロムメッキ処理を施したものも使用されている.このコータは塗工品質に優れることから,近年多く採用されており,アプリケータの形式によって下記のように分類される.. ロールアプリケータ型(roll applicator)図1・62に示すように,アプリケータロールによってカラーパン(color pan)中にためられた塗工液を原紙に転写する方式のコータである.このコータは,塗工速度が比較的低速で,重塗工を行う場合に採用される.. ショートドゥエル型(short dwell applicator)図1・63に示すように,アプリケータ部と仕上げブレード部が合体したコン図1・60 ゲートロールコータ図1・61 ロッドメタリング式フィルムコータ1―14産業機械・装置パクトな装置構成になっており,前述のロッド式フィルムコータのアプリケータとしても用いられる.このコータは,塗工液が転写されてから仕上げブレードで計量されるまでの時間(dwell time)が短いことから,原紙内への塗工液浸透が比較的少なく,微塗工および軽量塗工に適している.また最近では,図1・64に示すような,塗工液を原紙表面に転写する液溜り(chamber)内での塗工液流動を整流する装置を備えたタイプも実用化されている.. フリージェット型(free jet applicator)図1・65に示すように,アプリケータノズルのスリットから吐出した塗工液を原紙表面に転写する方式であり,高速塗工性と塗工品質に優れることから,近年高級塗工紙を生産する高速コータとして多く採用されている.c. エアナイフコータ(air knife coater)図1・66に示すエアナイフコータは,アプリケータで原紙上に転写した余剰塗工液膜を,原紙走行方向に対して対向する向きに噴出する二次元高圧空気噴流によって計量,かき落とす方式のコータである.このコータは,アート紙などの高級重塗工紙の生産に多く用いられていたが,最近ではほとんどブレードコータに置換されている.d. カーテンコータ(curtain コルゲータに流下させ,原紙上に直接所定の塗工液膜を形成する方式のコータである.このコータは,他のコータに比べ原紙内への塗工液浸透が少なく高い表面被覆性が得られる,またアプリケータロールもしくは仕上げブレードのような摩耗部品がないため,部品コストと交換メンテナンスコストが削減できる,といった特徴がある.塗工量は,原紙走行速度に応じた流量の塗工液をアプリケータノズルに供給することで制御できることから,前述の各コータに比べ操業も容易となる.なお最近では,図1・67に示したように,非接触で原紙同伴空気流の影響を抑制し,安定した高速塗工が可能となる装置も開発されている.
