# 1・2・6 紙工機械(paper converting machine)

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・50 にツードラムワインダを示す。2本のロール上に載せられた巻

a. 段ボール(corrugated fiberboard)波型に成型した中芯の片面または両面にライナを貼ったもので、段の山数と高さによりA,B,Cフルートなどの規格があり、また構造により片面(ライナが片側しかない)、両面、複両面(中芯を二段貼り合わせたもの)などの種類がある。使用上からは、内装用段ボール、外装用段ボールに分類される。b. 段ボール箱(corrugated fiberboard container (box))段ボールで成型された包装容器で、1988年のJIS 規格の改正により、国際規格に従って4けたのコード番号で表示される。4けたの上位2けたは基本形式を、下位2けたは個別形式を表す。上位2けたの基本形式は、02:溝切り型、03:テレスコープ型、04:組立型、05:差込型などとなっている。c. コルゲータ(corrugator)中芯、ライナから段ボールを製造する装置で(図1・68)、高速化、広幅化がすすみ、現在では最大速度450m/min、最大紙幅3300mmとなっている。機械全長は、従来100m近くあったが、2階建て化によって従来の1/2程度に短縮した機械も登場している。. ミルロールスタンド直径や紙幅が異なる原紙巻取りを自動的に装着し、適当なテンションをかけながら原紙を巻出す。. オートスプライサコルゲータを連続して運転するため、新旧の原紙巻取り間で紙継ぎを自動的に行う装置。. シングルフェーサ(single facer)中芯原紙を加熱して、広幅の歯車のような上下段ロールの間を通して波型に成型し、成型された中芯の山頂部にのりを塗布し、加熱した裏ライナと貼り合わせ片面段ボールを作る。のりとしては、デンプンのりが一般的に用いられる。従来ライナと中芯との接合は、段ロールと圧力ロールと呼ばれるロールとの接触面で高圧を掛けて行われていたが、最近ではベルト式など接合時間を長く低圧力で接合することで高品質かつ低振動を実現する機械が開発されている(図1・69)。. グルーマシンダブルフェーサに入る前に設置され、片段シートの段頂にのり付けする装置。. ダブルフェーサ(double facer)シングルフェーサで作られた片面段ボールと表ライナとを貼り合わせて両面段ボールを図1・69シングルフェーサ図1・70 フレキソフォルダグルア1―15作るユニットであり、2種類または3種類の片面段ボールを同時に貼り合わせて、複両面段ボール、複複両面段ボールを作ることができる。. スリッタスコアラ段ボール箱の寸法に応じて幅方向に切断し、折り目(罫線)を入れる装置。箱サイズに応じて、複数の罫線ロールやスリッタナイフを幅方向に正確な位置に短時間でセットする。スリッタナイフは薄刃の1枚刃が主流で、切り口が鋭利で美しく製函工程での加工性向上に貢献している。. カットオフ箱サイズに応じて、流れ方向に段ボールを切断する装置。切断寸法が箱サイズによって変化するが、ロールに固定されたナイフを箱サイズに応じて速く回転させたり遅く回転させることによって、任意の箱寸法に切断する。. スタッカ箱サイズごとにでき上がった段ボールシートを積上げ、次工程に払出す装置。生産管理システム:最新のコルゲータには小ロット化の進む段ボールの生産工程で効率的な連続生産を可能にするため、生産する段ボール箱ごとに異なる原紙の紙種、紙幅に対応して、全ユニットを連動自動制御するシステムが搭載されている。d. 製函機(box making machine)製函機とは、コルゲータで生産された段ボールシートに印刷し、罫線を入れ、溝を切って、接合して段ボール箱を作る機械の総称である。この製造工程は単一ではなく、また箱の形式によってもいろいろな装置がある。. フレキソフォルダグルア(flexo folder gluer)フレキソフォルダグルア(図1・70)を例に主要な構成部分を説明する。(1) 給紙部シートを積上げ、1枚ずつ印刷シリンダの回転図1・71 フレキソ印刷部図1・74 乾式紡糸法図1・72 フレキソロータリダイカッタ1―16産業機械・装置に同期して送り出す装置。搬送機構としては、サクションを利用したロール搬送などが用いられる。またシートを連続的に供給し積上げるオートフィーダが付帯装置として用いられる。(2) 印刷部(図1・71)段ボール印刷では、シート表面に段の凹凸があり、また大きな印圧がかけられないため、柔らかいゴムや樹脂版を用いた凸版印刷が多く用いられ、水性フレキソインキを用いたフレキソ印刷がおもに行われる。フレキソインキは、インキ粘度が低いためロール本数が少なく簡単な機構で構成できる特徴がある。(3) 排紙部罫線入れ、溝切り、継ぎ代の加工を行う。(4) フォールディング部継ぎ代部にのりを塗布し、ベルトで搬送しながら、罫線に沿って折曲げ接着する。のりとしては、熱を必要としない酢酸ビニル系ののりが一般的に使用される。(5) カウンタエジェクタ部印刷、折曲げ、接着されたシートを所定枚数だけ計数して、束にして排出する。. フレキソダイカッタ(flexo diecutter)平板状の刃の上下運動で静止状態の段ボールシートを打抜くプラテンダイカッタと、シリンダの円周上に刃を取付けて段ボールシートを走行させながら打抜くロータリダイカッタ(図1・72)とがある。前者は寸法精度が高い反面速度が~100枚/分と遅く、後者は寸法精度は劣るが200枚/分以上と高速化できる特徴がある。. プリンタスロッタ(printer slotter)印刷、罫線入れ、溝切りを行う装置。. スティチャ継ぎ代部をのりの代わりにワイヤ数本を折込み接合する装置。重量物用段ボールにおもに使用される。上記装置名称のフレキソは印刷ユニット、フォルダは折りユニット、ダイカッタは打抜きユニット、スロッタは溝切りユニットを表し、これらの組合せで一つの機械が構成されている。
