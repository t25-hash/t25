# 2・4・3 特殊な抽出プロセスとその装置

> **章**: 第2章 化学機械
> **節**: 2・4 抽出装置（extractor ）

変化を利用するのが超臨界抽出である．
圧力，温度の調節で溶剤とり，熱に弱い香気成分などの抽出に有利である．
高圧操作に対応でb.水性ニ相抽出 ある種の高分子水溶液がある条件で二相度やイオン強度に依存する性質を利用する．
原料の分離が容易に行われる．
抽出速度も大きいという利点があきる装置が必要で充てん槽式のものが一般に使用される．
に分離し，タンパク質やアミノ酸などの生体物質の分配特性が塩濃が装置として使われる．
