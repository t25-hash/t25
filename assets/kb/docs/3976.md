# 1・3・1 化学繊維製造機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・3 繊維機械

天然繊維以外の化学繊維を製造する機械で一般には紡糸機以降を指す。a. 化学繊維の分類化学繊維製造機械によって製造される化図1・76テキスタイルガラスファイバの製造工程学繊維(chemical fiber)は再生繊維、半合成繊維、合成繊維、無機繊維に分類される。b. 化学繊維の紡糸化学繊維においては、原料となる高分子を一度流動化し、細孔(紡糸口金)を通して繊維の形に作り出す紡糸という工程を経なければならない。それらの高分子の性質により、紡糸の方法は(1)湿式紡糸法、(2)乾式紡糸法、(3)溶融紡糸法に分類される(図1・73~1・75)。c. 無機繊維(inorganic fiber)ガラス繊維、炭素繊維、金属繊維、岩石繊維などがあるがここではガラス繊維と炭素繊維を記載する。. ガラス繊維の分類ガラス繊維の種類と用途を表1・2に示す。. ガラス繊維の製造工程ガラス繊維のうち、長繊維(テキスタイルガラスファイバ)の製造プロセスを図1・76に示す。d. 炭素繊維(carbon fiber)。 炭素繊維の分類アルミより軽く、鉄よりも強い繊維と図1・73 湿式紡糸法図1・75 溶融紡糸法図1・74 乾式紡糸法いわれる炭素繊維は、レーヨン系、PAN 系(ポリアクリルニトリル)とピッチ系がある。レーヨン系とPAN 系は繊維を炭化、黒鉛化して製造されるが、ピッチ系はコールタール、あるいは石油ピッチを加熱してメソフェーズを生成させ、これを紡糸、炭化、黒鉛化して製造される(図。 用途炭素繊維トウから、糸、織物、組ひも等を作りそれらに樹脂を含浸してプリプレッサし、複合材料として脚光を浴びている。その用途は次のとおり。(1) スポーツ、レジャー用材料(釣ざお、ゴルフクラブ、テニスラケット等)(2) 航空、宇宙産業用構造材料(二次構造材、一次構造材)(3) 産業用材料(ロボット、アンテナ、X 線機器等)。 フィラメントワインダ紡出された糸は高速ワインダで巻取られる(図1・78)。ワインダは巻取りスピンドル、トラバース装置、自動切換装置、ローラベール装置と制御装置からなる。軽量高強度材料の代表選手である炭素繊維も、使いこなすためにはさまざまな工夫が必要とされる。普通は繊維を樹脂で固めて必要な形状を付与するが、パイプ状の部品を作る場合に有効なフィラメントワインディングがある。マンドレルと呼ぶ回転する心金に巻付けていく。軽くて強いパイプは宇宙基地等の大型構造物で大いに活躍することが期待される。フィラメントワインディングの成形機自体は二軸式のものから、多軸式のコンピュータ制御へと開発が進んでおり、マンドレルの工夫とあいまってかなり複雑な形状のものが成形できるようになってきている。このワインダはガラス繊維にも使用される。
