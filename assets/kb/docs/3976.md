# 2・4・3 特殊な抽出プロセスとその装置

> **章**: 第2章 化学機械
> **節**: 2・4 抽出装置（extractor ）

a.超臨界抽出 臨界条件以上の流体の特殊な溶解性，物性
図2･13ミキサセトラ
図2･16ポドビルニヤク抽出機（旧版図2･3･4を簡略化）
図2･ 14多孔板抽出塔 図2･15往復回転式抽出塔
変化を利用するのが超臨界抽出である．圧力，温度の調節で溶剤と
原料の分離が容易に行われる．抽出速度も大きいという利点があ
り，熱に弱い香気成分などの抽出に有利である．高圧操作に対応で
きる装置が必要で充てん槽式のものが一般に使用される．
b.水性ニ相抽出 ある種の高分子水溶液がある条件で二相
に分離し，タンパク質やアミノ酸などの生体物質の分配特性が塩濃
度やイオン強度に依存する性質を利用する．小規模なミキサセトラ
が装置として使われる．
c .その他の抽出操作 逆ミセル抽出 や液膜分離 など新
しい抽出操作が提案されており，開発が進められている．
