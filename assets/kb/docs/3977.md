a.調湿装置 増湿の場合は，空気中へ水を噴霧したり，充て
ん物により形成された水膜と空気を接触させたりする，空気と水の
直接接触法が一般的である．減湿の場合は，湿りガスを露点よりも
低い温度の液面（直接冷却）または固体面（間接冷却）と接触させ
て冷却し，蒸気の一部を凝縮除去する冷却減湿法が一般的である．
直接冷却減湿装置（dehumidifier ）では，直接接触増湿装置
（humidifier ）のほとんどをそのまま利用でき，多量の処理に適し
ている．増・減湿装置ともに噴霧室，噴霧塔や充てん塔がよく採用
され，一般に充てん塔の性能が高い．代表的な横形噴霧室式増湿装
置（horizontal spray type humidifier ）を図2･18に示す．

c .乾燥装置 被乾燥材料には，大きな成型品から粉粒体まで，
湿り固体が多いが，泥状，液状のものもあり，材料の性質や熱源の
種類等により，多種の乾燥器が考案されている．一般に少量処理に
は回分式，大量処理には連続式が適しており，加熱方式と乾燥器内
の材料状態により，表2･2のように分類される．以下では湿り粉粒
体を対象とした代表的な乾燥器（dryer ）について述べる．粉粒体
の乾燥では，粒子と熱風の接触を促進させる工夫が重要であること
から，材料を動かさない静置式やその原理を連続式に応用した移送
式の乾燥器が利用されることは少ない．
.流動層乾燥器（fluidized- bed dryer ) 多孔板などのガ
ス分散板上の湿り粉粒体材料層に下方より熱風を送って材料層を流
動化させつつ乾燥させる装置で，粒子と熱風との接触が良いため熱
効率も良く，限界含水率を低くでき，大量処理も可能である．回分
式，連続式ともに広く利用されており，連続式の場合でも材料の平
均滞留時間を任意に設定できる長所を有する．熱風の持ち込む分だ
けでは熱エネルギーが不足するとき，層内に加熱管を入れて伝導加
熱を併用する場合もある．伝導加熱併用形流動層乾燥器の例を図
2･20に示す．
.気流乾燥器（pneumatic conveying dryer ) 湿り粉粒
体を高速で流れる熱ガス中に分散して供給し，熱ガスと並流に送り
ながら乾燥する連続式熱風乾燥器である．可動部分のない簡単な構
造であり，熱効率も高い．乾燥器内での滞留時間が短いことからフ
ラッシュドライヤと通称されることもあるが，水分の保有力が強い
材料では十分な乾燥が行えないときもある．分散器付の例を図2･
21に示す．
