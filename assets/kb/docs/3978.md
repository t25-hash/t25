# 1・3・3 糸仕上・加工機械

> **章**: 第1章 パルプ紙・繊維機械

a. ワインダ(winder)紡績の仕上工程および、製織、編成の準備工程として使用される糸巻機がワインダである。ワインダは精紡機でボビンに巻取られた糸を、糸むらの欠点となる部分を除去して、後の工程に都合のよいようにコーン、またはチーズ形状に高速で大きく巻取るのが目的である。近年このワインダの開発はめざましく、現在では精紡管糸の供給や糸欠点除去したあとの糸継ぎ、および巻上ったパッケージの取外しのすべてを自動で行う自動ワインダが主流をなしている。図1・85にその代表的な自動ワインダの構造を示す。また、自動ワインダの性能を大きく左右する糸継ぎ装置もノッタ多錘型から単錘・単独ノッタ型へ、さらにエアスプライサへと発展してきた。このエアスプライサを用いると糸の欠点部分をカットして除去したあと、圧縮空気で瞬時に結び目なしに糸を継ぎ合わせることができるので、従来の結び目によるトラブルは解消した。図一方、機台は高速化されて高生産性が可能となり、品質面でも電1―19子式スラブキャッチャの採用によって、より高品質の糸が生産できるようになった。さらに、できるだけ効率良く運転させるために、コンピュータによる工程管理(モニタリング)も開発され、各ドラムごとの効率、生産高、糸継ぎ目数、太さむらの欠点の個数等の記録が可能になった。そして、多品種、小ロット、短サイクル化に対応して自動ワインダと精紡機をコンベヤで連結する方法が開発され管糸の自動運搬、空ボビンのリターン、ボビンの残糸除去等、一連の装置のシステム化が実用化されている。b. 加工糸(textured yarn)加工糸とは、合繊の熱可塑性を利用して、合繊フィラメントによりと巻縮を付与して各種の長繊維かさ高伸縮加工糸である。これによってストレートな糸にバルキー性を与え、織物や編物に使用した場合に紡績糸に似た風合いをもたせることができる。素材別には、ポリエステルとナイロンが圧倒的に多い。ポリエステル加工糸は、織、編物に主として使用されているが、ナイロン加工糸はヤング率が低いことから衣料として必要な腰の強さの点でポリエステルに劣るため、衣料用途より耐摩擦性が要求されるストッキング、カーペット用に向けられている。加工方法により、仮より、空気噴射、押込み、擦過などに分かれる。. 仮より加工法一般に加工糸用機械の主体をなす。図ねん、熱セット、解ねんを連続的に行う方法である。原理的には一種の緊張熱処理であって、巻縮の耐熱性、堅ろう性が他の方法に比較してよく、また仮より数、張力、温度などの加工条件を変更することにより、物性を大幅に変更できる。仮より装置はスピンドル式、フリクション式、ニップベルト式の三つに大別される。スピンドル式は、キャプステン型スピナを内蔵したものが一般的である。スピンドルをマグネットで圧接させて増速させるようになっており、最高80万rpmの高速回転が出せるものが市販されている。フリクション式は、複数円板群やシリンダの外周面、円筒の内面または側面などの高速回転面に糸を直接圧接して、その摩擦力で直接仮よりを与えるものであり、スピンドル式に比べてはるかに高速加工が可能である。ニップベルト式は2本のベルトを互いにある角度をもって対置させ、その間に糸を把持させて仮よりする。原理は、わらで縄を作るときに手でよっていくのと同じ簡単な方法である。糸は2本のベルトで完全にニップされた状態でよりがかけられるため、糸とベルトの間には、スリップがほとんどなく、糸のダメージが少なく、高速加工も可能である。従来は完全な延伸糸を供給して仮より加工が行われていたが最近では紡糸巻取機の高速化から半延伸糸(POY)の供給が可能となり、仮より加工は延伸仮より加工に移行しつつあ(a)エアスプライサの装置(b)つなぎ目図1・84 空気噴射、押込み、擦過加工法仮より加工がより、ひずみの固定による旋回性バルキー加工糸であるのに対して、空気噴射、押込み、擦過による加工糸は旋回性をもたない。空気噴射式は空気噴射によって繊維を乱し単繊維にランダム状の巻縮を形成させる方法で、オーバフィード量と気流の大小によって巻縮状態が異なる。押込み式は、繊維軸方向の圧縮力によって原糸に座屈を起こさせ熱セットするもので、できた加工糸はジグザグ状の巻縮をもっている。擦過式はエッジ部で繊維をこすり、エッジに接触する部分と反対の接触しない部分で受けるひずみの差を固定して巻縮を作る方法である。
