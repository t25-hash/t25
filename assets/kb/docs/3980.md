# 1・3・5 編組機械

> **章**: 第1章 パルプ紙・繊維機械
> **節**: 1・87 に示すように仮より装置の前に加熱器を置くことにより、加

編組物には1本または多数本の糸でループを作り、このループを互いに掛かり合わせて作る編物や、数本または数十本の糸を互いに斜め方向により合わせ、編み合わせ、からみ合わせ、組合わせて作るレース、網、組ひもなどがある。図1・94は編組織の変化を示すもので、(a)はタック編み、(b)は浮き編み(フロート)、(c)はパイル編み、(d)は挿入編み(インレイ)、(e)はペレリン編み、(f)は添え糸編み、(g)はからみ添え糸編みである。編組織のうち最も多く製造されるのは編物で1本の糸で編成する横編みと、多数の糸を製織の場合のように整経して斜交編地を編成する縦編みとに大別される。横編にしても縦編にしても、ループを作るとともにこれを互いに掛かり合わせて布を作るのに使用される編機の編成部は、編針、シンカ、プレッサである。編針にはひげ針と舌針(べら針)および複合針の3種類がある。編組機械は専用機的性質が強く、布組織、模様柄、布サイズの相異等を編組するために各種の装置が備えられ、またコンピュータによる柄出し、サイズ制御、スタイル制御等特殊装置の組合せによって千差万別のものが図1・97靴下編機の構造図1・98綿織物染色仕上工程図の一例図1・99連続式精練・漂白装置存在するが、代表的機種によって分類すれば表1・5に示すように大別される。なお、近年、ホールガーメントと称する編み機から直接、衣類が形成される機械も開発されている。横編機、丸編機、靴下編機の構造を図1・95~1・97に示す。
