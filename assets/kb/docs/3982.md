# 2・7・3 供給（feeding)

> **章**: 第2章 化学機械
> **節**: 2・7 粉粒体分級・貯蔵・供給

粉体供給機は粉粒体の物性やプロセスの配置・構成が多様である
ため，多種多様であり ，次の条件を満たすことが必要である．
1.粉粒体の物性に適合すること．
2.流量操作範囲が十分広いこと．
3.静特性が安定しており，供給流量の再現性が良いこと．
4.動特性が良いこと．
ゲートバルブ，ロータリフィーダ，スクリューフィーダ，振動フ
ィーダなどから，プロセスの条件に合わせて適切な機種を選定する
ことが必要である ．おもな供給機について，選定の目安を表2･5
に示す．
定量性を保つためには，流量検出器を用いてフィードバック制御
を行う．このとき，動特性の良い供給機を用いることが必要であ
る ．表2･6はおもな粉体供給機の動特性の概要を示す ．
