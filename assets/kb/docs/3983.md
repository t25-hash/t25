# 2・8・1 概説

> **章**: 第2章 化学機械
> **節**: 2・8 混合・かくはん装置

混合・かくはん操作は幅広い分野で日常行われている操作であ操作は混合操作と呼ばれ，混合終了の条件は，混合物からあるサイあり，化学反応を行う場合には分子サイズまで一様に混ぜ合わせるの混合操作は分散操作とも呼ばれ，この場合一方の流体を連続相流ば，液体に解け合わない流体が液体か気体かにより，それぞれ液滴固体粒子を分散させる操作も混合操作として取扱われる．
さらに広せる操作も混合操作に含めることがある．
これら装置の選択は，混合・かくはんの主目的が何であの量はどれほどであるか，混合操作にどれほどの時間を必要とするて決められる．
ここで，お互いに溶解し合う2種類以上の流体を混ぜ合わせるズのサンプルを採ったときどのサンプルも一様であることが条件でことが要求される．
混ぜ合わせる二流体がお互いに溶解しない場合体，他方の流体を分散相流体として混ぜ合わせることになる．
例えか気泡として液体中に分散させることが混合操作となる．
