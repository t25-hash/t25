# 2・1・1 熱交換器(heat exchanger)

> **章**: 第2章 化学機械
> **節**: 2・1 伝熱・蒸発装置

a. 熱交換器の分類熱交換器は高温流体-低温流体の間で熱の授受を行う装置であり,熱交換を行う流体の間に固体隔壁の有無により隔壁式熱交換器と直接接触式熱交換器があり,さらには蓄熱式ならびに二つの隔壁式熱交換器の間で熱媒体を循環させるヒートパイプ式のような方式がある.各種熱交換器の構造の特徴による分類を表2・1に示す.b. 総括伝熱係数(heat transfer coefficient)熱交換器の性能を表す総括伝熱係数(熱通過率)は,伝熱管壁を通しての熱交換に対して高温と低温流体の組合せによって求められた概略値が示されている.c. 汚れ係数(scale coefficient)特に流体が液体の場合は,使用過程で伝熱面の内外表面に汚れ(スケール)が付着し,伝熱特性の低下の原因となる.汚れの原因には,(1)塩類[CaCO,CaSO,Mg(OH),NaSO,ケイ酸塩,有機カルシウム塩など],(2)非水溶性微粒子,(3)腐食,(4)コーキング,(5)生物(スライム),(6)凍結物などが挙げられる.一般に,汚れの伝熱抵抗を汚れ係数(伝熱係数の逆数)といい,熱交換器の設計,運転において重要な因子となる.代表的な汚れ係数が実測されている.d. 熱媒体水蒸気が加熱媒体として最も広く使用されているが,473K を超える高温で使用する場合には高圧になるため食品関係か原子力以外ではあまり使用されていない.このような温度域では種々の熱媒体が用いられる.なお,熱媒体の具備すべき特徴としては(1)熱安定性,(2)熱伝導度(thermal conductivity),比熱,潜熱が大きい,(3)蒸気圧が低い,(4)腐食性,毒性がないなどが挙げられる.e. ピンチポイント与熱流体と受熱流体のエクセルギー変化を考慮して求めた両流体間の最小接近温度差を示す点をいう.最小接近温度差を小さくとれば省エネルギー効果が得られるが,逆に伝熱面積が増えることになるため,この両者のバランスを考慮して最適化を設計する.
