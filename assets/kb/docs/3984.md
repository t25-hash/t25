# 2・1・2 蒸発装置(evaporator)

> **章**: 第2章 化学機械
> **節**: 2・1 伝熱・蒸発装置

不揮発性溶質を含有する溶液から揮発性溶媒を気化分離させ、溶液を濃縮する単位操作を蒸発と呼び、砂糖、食塩、苛性ソーダなどの希薄水溶液の濃縮、吸収式ヒートポンプにおける再生過程での吸収溶液の加熱濃縮、さらには海水の淡水化などに用いられる。なお、溶液がいずれもすべて揮発性物質から構成される場合には蒸留操作を行い、蒸発とは区別して用いられる。工業的な蒸発操作では多くの場合、(1)蒸発潜熱が大きく、(2)温度制御が容易で、(3)表2・1 構造上の特徴による熱交換器の分類管形二重管熱交換器シェルアンドチューブ熱交換器スパイラルチューブ熱交換器プレート形熱交換器拡大伝熱面蓄熱式回転式熱交換器軸流半径流固定マトリックス熱交換器1―27入手しやすく、(4)凝縮温度が一定で凝縮伝熱係数が大きい水蒸気を加熱源とした沸騰蒸発が行われる。この場合、沸点上昇の大きい溶液に対しては真空または減圧操作を行い、溶液の沸点を低下させ伝熱に有効な温度差を増大させるのが普通である。このような減圧下での蒸発操作は熱経済性を高めるうえで必須であり、後述する多重効用蒸発において用いられる。図2・1に各種蒸発装置の主要な形式の概念図の一例を示す。これらの蒸発装置はそれぞれ、溶液の強制流動による総括伝熱係数の増大、溶液の流動やかくはんによるスケール析出の防止、熱に敏感な溶液の処理に適するような加熱時間の短縮化、清掃の簡単化、省エネルギー化などの点で工夫がこらされ、溶液の性質、製品の性質、使用経験などによって適宜使い分けられる。加熱方式としては、溶液を入れた容器を炎や高温ガスで直接加熱する直火式、(a)のようにバーナを液中に設置して液中で加熱する液中燃焼方式、主として水蒸気のような加熱媒体によって加熱する間接加熱方式[(b)~(g)]に分類される。(h)のフラッシュ蒸発方式では、蒸発室上部で予熱された原液を加熱器で加熱したのち減圧下の蒸発室にてフラッシュ蒸発させ、濃縮液および原液の予熱に使われた蒸気は凝縮して取出される。蒸発装置は加熱器、蒸発器以外に、供給液、蒸気中に含まれる空気や非凝縮性ガスの抽気排出を行うと同時に蒸発器内を減圧にするための抽気減圧装置、さらには発生した蒸気を効率良く除去するために凝縮器、蒸発によって平均径が100 m以下の飛まつ(ミスト)が蒸気の流れに乗って同伴することを防ぐための飛まつ捕集器が備えられる。
