# 2・8・3 かくはん翼を用いない混合装置

> **章**: 第2章 化学機械
> **節**: 2・8 混合・かくはん装置

a.ジェット流による混合装置 円筒槽または角槽内の流体を
外部に備えられた循環ポンプにより吸引し，再びジェットノズルか
ら槽内に吐出させることによって混合・かくはんする装置である．
この場合，ジェットノズル（jet nozzle）は槽底近くの槽側壁から
やや上方に向けて設置されることが多い．混合状態はジェットノズ
ルから吐出される流体の運動エネルギーと運動量のバランスによっ
て左右される．ジェットノズルにエジェクタを用いることにより気
体を吸引しながらかくはんすることも可能であり，気液反応槽とし
て用いられることもある．
b.揺動運動による混合装置 混合すべき流体を入れた円筒槽
Maxblend翼 Hi -F Mixer翼 Fullzone翼 MR-205翼

を揺り動かすことによって混合操作を行う装置が揺動混合装置
（shaking mixer ）である．槽全体を動かさなければならないので
大型槽には向かないが，実験室においては頻繁に用いられる混合装
置である．円筒槽には何の可動装置も必要としないので，汚れを嫌
うバイオリアクタとして用いられることが多い．最近は生物工学の
進展でこの装置の大型化の検討も進められている．
c .静止型混合装置 パイプ内に特殊なエレメントを装着し，
混合すべき二流体をこのエレメントに通すことによって混合操作を
行う装置を静止型混合装置（static mixer,motionless mixer ）と
呼ぶ．この場合，エレメント内に流体を1回だけ通過させる流通系
混合操作と，ジェットかくはん装置と同様に容器の外部にポンプを
設置して流体を循環するよう配管されたパイプ内にエレメントを装
着し，流体をエレメント内に繰返し通過させる回分式混合操作に分
けられる．混合すべき流体の粘度が高粘度か低粘度か，流体相互に
解け合うか解け合わないか，二流体の体積比がどれほどかなどによ
って，パイプに装着するエレメントの形状は多種多様である［エレ
メント形状については文献(52)参照］．
