# 2・1・3 蒸発缶の設計

> **章**: 第2章 化学機械
> **節**: 2・1 伝熱・蒸発装置

a. 沸点上昇(elevation of boiling point)溶液の沸点は溶質が溶けていることによる沸点上昇のために、純溶媒の沸点よりも高くなる。図2・2に各種水溶液の沸点上昇の例を示す。溶液の沸点上昇は溶質の濃度、解離度に依存するが、理想溶液の沸点上昇はラウールの法則(Raoult’s law)に従って次式で求めることができる。希薄溶液、高分子溶液などは普通理想溶液とみなせるが、蒸発操作の対象となる溶液の多くは電解質溶液および濃厚溶液などであり非理想溶液のため、ラウールの法則は適用できない。しかし、非理想溶液に関してはラウールの法則の変形である次式が用いられる。b. デューリング線図(Duhring’s chart)非理想溶液の沸点上昇を求める別の方法としてデューリング(Duhring)線図を利用プレート間にガスケットを挟んだ熱交換器(プレート形熱交換器)プレートをらせんに巻いた熱交換器(うず巻形熱交換器)プレートアンドフィン熱交換器(コンパクト熱交換器)フィンアンドチューブ熱交換器プレートフィンアンドチューブ円周フィン1―28(e) 強制循環式水平管型(g) かくはん膜型(f) コイル型(h) フラッシュ蒸発方式図2・1 各種蒸発装置(A:非凝縮性ガス、F:供給液、P:濃縮液、D:凝縮液、S:加熱用蒸気、V:蒸発蒸気)する方法がある。デューリング線図は、一定濃度の溶液と純溶媒とが同一の蒸気圧を示す温度は互いに直線関係にあることに基づいて描かれたものである。例として、図2・3にNaClのデューリング線図を示す。溶液の沸点は雰囲気圧と蒸気圧が平衡にあるときのものであり、蒸発缶のように伝熱面が溶液中にある場合には、伝熱面上での溶液の沸点は溶液表面での沸点に比べて溶液の静頭圧に相当する分だけ高くなる。一般に、真空、減圧下での蒸発操作ではが小さいため相対的に静頭圧の寄与が増大する。c. 蒸発潜熱(vaporization heat)水溶液の濃度が十分希薄であれば蒸発潜熱は水蒸気表から求めた純粋の値で代用できるが、濃縮が進むと溶質の影響を考慮しなければならない。近似的にはデューリング線図の直線のこう配から次式で求められる。d. 総括伝熱係数蒸発缶の総括伝熱係数は次式で与えられる。各種蒸発缶の総括伝熱係数の代表値および代表的な液側伝熱係数について実測データが与えられている。e. 蒸発計算図2・4に示す単一蒸発缶における熱と物質収支をとる。
