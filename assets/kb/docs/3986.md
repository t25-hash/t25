# 2・1・4 蒸発操作の省エネルギー化

> **章**: 第2章 化学機械
> **節**: 2・1 伝熱・蒸発装置

蒸発缶では溶液を濃縮すると同時に，多量の蒸気を発生させてい
る．そこでその蒸気を蒸発の熱源として再利用することによってエ
ネルギー効率の向上を図る方法が考えられている．
a.蒸気圧縮法 蒸発缶内で発生した蒸気をターボ圧縮機など
による(１)機械的圧縮，または蒸気エゼクタによる(２)熱圧縮によ
って蒸気の温度を上昇させ，溶液の加熱に再利用する．
b.多重効用法 蒸発によって発生した蒸気をそれより低圧，
低温度で操作している次缶の加熱蒸気として利用することによって
熱効率を上げる． 重効用では1kgの加熱蒸気でおおよそ (kg)
の水を溶液から分離できるとされているが，効用数は設備費と運転
経費ならびに熱損失，沸点上昇などとの関係によって決まる．多重
効用法は連続操作であり，液と蒸気の接触に関して4種の缶配列法
（順流，逆流，錯流，並流）がある．
c .多段フラッシュ法 図2･1(ｈ)のフラッシュ蒸発（flush
evaporation）を多段化したものである．蒸発缶上部の蒸気室に設
けた伝熱管にて溶液を予熱し，さらに外部の熱交換器で加熱したの
ち蒸発缶でフラッシュ蒸発させると蒸気は伝熱管の外表面で凝縮す
る．凝縮水は溶液と分離することによって濃縮液が得られる．目安
として3段で1効用缶に相当する効率をもつ．
2・2吸収・蒸留装置
混合気体中のある成分を液に溶解させてその成分を回収あるいは
除去する操作を吸収（absorption）といい，混合溶液中の揮発しや
すい成分と揮発しにくい成分をそれぞれ蒸気相と液相に濃縮させて
溶液を各成分に分離する操作を蒸留（distillation）という．いず
れも，蒸気または気体と液の接触界面を通して物質を移動させて目
的物質を分離する操作である．このような操作で用いられる装置を
総称して気液接触装置といい，その代表的な装置に充てん塔，段
塔，スプレー塔，気泡塔がある．このうち，充てん塔と段塔はいず
れも吸収装置（absorption column）や蒸留装置として広く用いら
れている．しかし，スプレー塔や気泡塔は吸収装置として用いら
れ，蒸留装置（distillation column）にはほとんど用いられない．

2・2・1充てん塔（packed column）
充てん塔は塔型容器に充てん物を詰めた装置［図2･5(ａ)］で，
液が充てん物表面上を，蒸気または気体が充てん物の間げきを流れ
る．後述の段塔と比べると，構造が単純，気体の圧力損失が小さ
い，腐食性あるいは発泡性液に適用できるなどの特徴をもつ．充て
ん塔はこれまで吸収装置として広く用いられてきたが，規則充てん
物が開発されてからは蒸留塔としても積極的に用いられるようにな
ってきた．
a.充てん物（packing) 充てん物は気液の接触を起させるた
めのもので，比表面積が大きく高い分離効率が得られること，空げ
き率が大きく圧損が小さいこと，その他に機械的強度が大きいこ
と，見掛け密度が小さいことなどが性能として求められる．充てん
物には，規則充てん物と不規則充てん物がある．
規則充てん物には，波板状の金属板や金網を重ね合わせた構造の
ものやグリッド構造のものがある．波板状規則充てん物［図2･5
(ｂ)］では，波板によって一定の形状をもつ空間が形成され，気液
流路が確保されている．このため，不規則充てん物に比べて圧損が
小さく，充てん塔内を気液がむらなく流れる（偏流が起こりにく
い）という特徴を有し，スケールアップが容易に行えるという利点
をもつ．なお，段塔と異なり高圧系には不向きである．
不規則充てん物には，円筒型やサドル型のものがある［図2･5
(ｂ)］．規則充てん物に比べると，充てんの仕方によって空げきに
むらができやすく，偏流を生じやすい．このため，スケールアップ
に難があるなどの欠点をもっている．
b.液分散器（liquid distributor) 塔頂部で液を均一に分散さ
せて，液が充てん塔内を均一流下するようにするための機器で，ラ
ダー型，オリフィス型などがある．前者はパイプに孔を開けたもの
で液量の変動が少ない場合に用いられ，後者は広い範囲の液量に対
応できるという特徴を有する．
c .液再分散器（liquid redistributor) 塔頂で液を均一に分散
させても液が流下するにつれ塔壁に集まる傾向がある．これを再度
充てん物内に液を戻すための機器が液再分散器で，通常，3～5m
おきに配置する．
