# 2・1・4 蒸発操作の省エネルギー化

> **章**: 第2章 化学機械
> **節**: 2・1 伝熱・蒸発装置

蒸発缶では溶液を濃縮すると同時に、多量の蒸気を発生させている。そこでその蒸気を蒸発の熱源として再利用することによってエネルギー効率の向上を図る方法が考えられている。a. 蒸気圧縮法蒸発缶内で発生した蒸気をターボ圧縮機などによる(1)機械的圧縮、または蒸気エゼクタによる(2)熱圧縮によって蒸気の温度を上昇させ、溶液の加熱に再利用する。b. 多重効用法蒸発によって発生した蒸気をそれより低圧、低温度で操作している次缶の加熱蒸気として利用することによって熱効率を上げる。重効用では1kg の加熱蒸気でおおよそ(kg)の水を溶液から分離できるとされているが、効用数は設備費と運転経費ならびに熱損失、沸点上昇などとの関係によって決まる。多重効用法は連続操作であり、液と蒸気の接触に関して4種の缶配列法(順流、逆流、錯流、並流)がある。c. 多段フラッシュ法図2・1(h)のフラッシュ蒸発(flushevaporation)を多段化したものである。蒸発缶上部の蒸気室に設けた伝熱管にて溶液を予熱し、さらに外部の熱交換器で加熱したのち蒸発缶でフラッシュ蒸発させると蒸気は伝熱管の外表面で凝縮する。凝縮水は溶液と分離することによって濃縮液が得られる。目安として3段で1効用缶に相当する効率をもつ。混合気体中のある成分を液に溶解させてその成分を回収あるいは除去する操作を吸収(absorption)といい、混合溶液中の揮発しやすい成分と揮発しにくい成分をそれぞれ蒸気相と液相に濃縮させて溶液を各成分に分離する操作を蒸留(distillation)という。いずれも、蒸気または気体と液の接触界面を通して物質を移動させて目的物質を分離する操作である。このような操作で用いられる装置を総称して気液接触装置といい、その代表的な装置に充てん塔、段塔、スプレー塔、気泡塔がある。このうち、充てん塔と段塔はいずれも吸収装置(absorption column)や蒸留装置として広く用いられている。しかし、スプレー塔や気泡塔は吸収装置として用いられ、蒸留装置(distillation column)にはほとんど用いられない。1―29(b)(a)図2・5 充てん塔一方、これらの装置を気液の分散状態で分類すれば、段塔と気泡塔は液相中に蒸気あるいは気体が分散する気体分散型装置であり、充てん塔とスプレー塔は液分散型装置である。一般には、液量に比べてガス量が少ない場合には段塔が、逆にガス量が多い場合には充てん塔が有利となる。
