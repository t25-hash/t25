# 2・8・4 粉体混合装置（powder mixer )

> **章**: 第2章 化学機械
> **節**: 2・8 混合・かくはん装置

製薬工業，食品工業等では粉粒体の混合・かくはん操作が必要な
場合が多い．通常は粉粒体を入れた容器を回転または揺動運動さ
せ，数種類の粉粒体を同時に混合する装置が開発されている．粒径
や粒子密度の異なる粉粒体を混合する場合には分級が生じることが
あるので注意が必要である．回転式粉体混合装置の一例を図2･29
に示す．
その他，輸送ポンプそれ自信も混合・かくはん機能をもってお
り，流体輸送と同時に混合・かくはん操作も行う目的で設置された
ポンプをラインミキサ（line mixer ）と呼ぶこともある．混合装置
の選択・設計に当たって，装置寸法や操作条件等の定量的な査定は
「化学工学便覧」（丸善）等の専門書を参照されたい．
