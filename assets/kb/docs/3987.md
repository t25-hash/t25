# 2・2・1 充てん塔(packed column)

> **章**: 第2章 化学機械
> **節**: 2・2 吸収・蒸留装置

充てん塔は塔型容器に充てん物を詰めた装置[図2・5(a)]で、液が充てん物表面上を、蒸気または気体が充てん物の間げきを流れる。後述の段塔と比べると、構造が単純、気体の圧力損失が小さい、腐食性あるいは発泡性液に適用できるなどの特徴をもつ。充てん塔はこれまで吸収装置として広く用いられてきたが、規則充てん物が開発されてからは蒸留塔としても積極的に用いられるようになってきた。a. 充てん物(packing)充てん物は気液の接触を起させるためのもので、比表面積が大きく高い分離効率が得られること、空げき率が大きく圧損が小さいこと、その他に機械的強度が大きいこと、見掛け密度が小さいことなどが性能として求められる。充てん物には、規則充てん物と不規則充てん物がある。規則充てん物には、波板状の金属板や金網を重ね合わせた構造のものやグリッド構造のものがある。波板状規則充てん物[図2・5(b)]では、波板によって一定の形状をもつ空間が形成され、気液流路が確保されている。このため、不規則充てん物に比べて圧損が小さく、充てん塔内を気液がむらなく流れる(偏流が起こりにくい)という特徴を有し、スケールアップが容易に行えるという利点をもつ。なお、段塔と異なり高圧系には不向きである。不規則充てん物には、円筒型やサドル型のものがある[図2・5(b)].規則充てん物に比べると、充てんの仕方によって空げきにむらができやすく、偏流を生じやすい。このため、スケールアップに難があるなどの欠点をもっている。b. 液分散器(liquid distributor)塔頂部で液を均一に分散させて、液が充てん塔内を均一流下するようにするための機器で、ラダー型、オリフィス型などがある。前者はパイプに孔を開けたもので液量の変動が少ない場合に用いられ、後者は広い範囲の液量に対応できるという特徴を有する。c. 液再分散器(liquid redistributor)塔頂で液を均一に分散させても液が流下するにつれ塔壁に集まる傾向がある。これを再度充てん物内に液を戻すための機器が液再分散器で、通常、3~5mおきに配置する。
