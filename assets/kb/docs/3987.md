# 2・9・1 固液機械的分離装置

> **章**: 第2章 化学機械
> **節**: 2・9 固液・固気機械的分離装置

液中に懸濁している固体粒子を機械的に分離する固液分離操作に
は，ろ過，圧搾，沈降，遠心分離などがあり，化学工業における分
離精製プロセスから廃液処理に至るまで広範に利用されており，そ
の目的に応じて最適な装置を用いる必要がある．
a.ケークろ過装置 スラリー中に懸濁している不溶解物質を
ろ布，ろ紙，金網，膜，粒子層等のろ材により捕捉粒子とろ液とに
分離する操作を一般にろ過（filtration）といい，古くから工業の
広範な分野で用いられている．捕捉粒子径やろ材の細孔径にもよる
が，通常1vol%以上の固体を含むスラリーのろ過では，ろ過開始
後まもなく捕捉粒子からなるろ過ケークがろ材面上に形成され，そ
のケークが以後のろ過に対してろ材として作用しつつろ過が進行す
る．この種のろ過をケークろ過（cake filtration）という．ケーク
ろ過器は，ろ液流動のための推進力によって，重力ろ過器，加圧ろ
過器および真空ろ過器に大別される．
加圧ろ過器はバッチ式が多く，ポンプ圧または圧縮空気圧をスラ
リーに作用させてろ過を行う．比較的高圧が利用できるので高いろ

程度，浸液率は40～65%程度である．

b.圧搾装置 圧搾（expression）は，スラリーや半固体状の
固液混合物を，液体は通過させるが固体は通過させない隔壁内に収
容して，これを機械的に圧縮脱水して圧縮ケークと搾出液とに分離
する操作である．ろ過操作で得られるろ過ケークより，さらに含水
率の低いケークが生成し，ろ過に比べてさらに高度な脱水を目的と
する場合に用いられる．乾燥，燃焼処理，埋立，造粒などの前処理
に利用されることも多い．各種食品工業，発酵工業，金属工業のほ
か，上下水汚泥，各種廃液や廃棄物の処理にも広く用いられてい
る．
圧搾装置はプレスともいい，バッチ式と連続式とがある．バッチ
式には，圧搾型フィルタプレス，チューブプレス，プレートプレ
ス，ケージプレスなどがある．一方連続式には，ベルトプレス，ス
クリュープレス，円板プレスなどがあり，近年その適用範囲が拡大
