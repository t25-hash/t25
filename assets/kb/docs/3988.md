# 2・2・2 段塔(tray column)

> **章**: 第2章 化学機械
> **節**: 2・2 吸収・蒸留装置

図2・6(a)に示すように,塔内がトレイ(段)で仕切られた構造の1―30産業機械・装置塔を段塔という.液は上段からダウンカマー(downcomer)を通ってトレイに入り,トレイ上を流れて次のダウンカマーに入る.一方,蒸気はトレイ上の孔あるいは蒸気上昇管からトレイ上に入り,トレイ上の液中を気泡や気柱として上昇して上段に流入する.段塔は,その設計法が確立している,スケールアップが容易である,高圧下での操作が容易である,固体粒子を含む流体にも適用できる,多段フィードが容易であるなどの特徴をもつ.その反面,充てん塔に比べて,飛まつ同伴量が多い(効率の低下につながる),圧損が大きいなどの欠点もある.a. トレイ(tray)段上の液と下段から入る蒸気を効果的に接触させるためのもので,泡鐘トレイ(bubble captray),バルブトレイ(valve tray),多孔板トレイ(sieve tray)などがある[図2・6(b)].泡鐘トレイは蒸気上昇管の上に釣鐘状のものをかぶせたもので,圧力損失が大きく,飛まつ同伴も起こりやすい.また,構造が複雑で重量があるため代替品があれば使われない傾向にある.バルブトレイは段上の孔の上部に可動式の板(バルブ)を載せたもので,下段からの蒸気流量に応じてバルブが上下して開孔面積を調整できるようになっている.他のトレイに比べて安定操作範囲が広く,操作条件の変動に対しても安定に操作できるという特徴をもつことから,トレイの主流になりつつある.多孔板トレイは段に多数の孔を等間隔にあけた構造のもので,他のトレイに比べて安価であるが,ウィーピング(孔からの液漏れ)を起しやすい,安定操作範囲は比較的狭いなどの欠点がある.b. ダウンカマー(downcomer)トレイ上で気液接触後,液だけを下段に流すための機器である.液量によるキャパシティはダウンカマーの断面積と高さにより決まる.また,ダウンカマー内で液が下段に落ちなくなる(これをフラッディングという)現象が起こるため,この構造は塔の操作性に重大な影響を与える.
