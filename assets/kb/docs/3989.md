# 2・2・3 スプレー塔(spray column)

> **章**: 第2章 化学機械
> **節**: 2・2 吸収・蒸留装置

スプレー塔は、ガス分散板を通して塔内にガスを導入し、その中にスプレーノズルを通して液を噴霧させるタイプの装置で、吸収プロセスで用いられる。構造が簡単でガスの圧損が低いことから、大容量のガスの処理に適している。また、吸収効率はあまり高くないため、吸収が容易に起こる系に適している。また、吸収と同時に粉塵を除去したい場合や吸収によって固体を析出する場合などにも有効である。しかし、液の飛まつがガスに同伴されやすい(これを飛まつ同伴という)ため、塔の上部には飛まつを取るためのデミスタを設けることが多い。
