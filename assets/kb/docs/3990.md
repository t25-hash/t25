# 2・2・4 気泡塔(bubble column)

> **章**: 第2章 化学機械
> **節**: 2・2 吸収・蒸留装置

気泡塔は,塔型容器に液を導入し,容器下部からガス分散器を通して気体を吹込むタイプの装置で,吸収プロセスで用いられる.気(b)(a)図2・6 段塔体の圧損が高いため,低圧,大容量気体の処理には不向きである.液の滞留時間が他の装置に比べて長いため,液側での吸収が全体の吸収速度を決定する(液側律速)系に適している.またスラリーを含む系や固体を析出する系にも適している.ガス分散板には,単孔ノズル(このときの気泡塔を標準気泡塔という),多孔板,多孔質板,二相流ノズルなどがある.気泡が塔内を上昇するにつれて気泡の合一が起こり,吸収効率が低下するため,塔内に多孔板を設置してガスの再分散を行うこともある.このような塔を多段気泡塔という.吸着とイオン交換はいずれも固体の分離剤を用いる点で装置上の共通点が多い.吸着操作では,吸着材と呼ばれる多孔性物質の細孔表面に特定成分を濃縮させることによって,気体または液体混合物の分離精製を行う.イオン交換操作では,高分子母体からなるイオン交換樹脂に固定した正または負のイオン種と水溶液中の不純物イオンとの間の可逆反応を利用する.吸着剤・イオン交換樹脂の形状としては,粉末,粒状(球,破砕物・押出成型品など)のほかに,繊維状,ハニカム状などがある.
