# 2・3・1 固定層吸着(回分法)

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

装置内に静置した吸着材に不純物を吸着させる間欠操作であり、最も広く使用されている装置である。吸着材が入口から順次吸着質で飽和していき、出口濃度が所定値より高くなる破過時間に達すると同一装置内または別の再生装置で吸着材を再生して繰返し使用する。吸着材をバラのまま充てんして、(a)上(下)向流、または(b)環状流に流すもののほか、カートリッジに充てんした箱を吸着塔に配置する(c)カートリッジ型がある。固定層を通る伝熱速度が遅いので断熱操作となり、気相吸着では発生した吸着熱で気体温度が上昇する。
