# 2・3・2 流動層吸着装置(fluidized bed adsorber)・移動層吸

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

着(連続法)吸着材を流動化して吸着部と脱着部を循環させる連続操作であり,有機溶剤回収装置としての応用例を図2・7に示す.吸着材として耐摩耗性,高強度の均一径真球状活性炭が使われている.装置上部の吸着部は6段の流動層からなるが,良好な伝熱特性のため,固定層で見られる局所的発熱が避けられる.装置下部の脱着部は移動層となっており,吸着材がチューブ内を下降する間にシェルから加熱を受けるとともに,塔底からの水蒸気と接触して再生され,気流図2・7 流動層吸着装置(溶剤回収)輸送で塔頂に戻される.
