# 2・3・3 固定層吸着の疑似連続化

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

装置個体としては固定層という回分式でありながら，原料供給や
製品取出し位置など流体側の流路を順次切替えることによって，相
対的には吸着材が移動しているとみなせる疑似連続化を図ったもの
に，圧力スイング吸着（pressure swing adsorption），温度スイン
グ吸着（temperature swing adsorption），疑似移動層（simulated
moving bed）などがある．
a.圧力スイング吸着（PSA:pressure swing adsorption)
PSAは高圧（0.1～1MPa）で吸着させ低圧（0.01～0.1MPa）で
脱着再生させるもので，吸着・脱着工程で流体は向流に流す．その
一例として図2･8に合成ゼオライトを用いた酸素製造PSAの流路
構成図を示す．4個の小さな平行四辺形は吸着塔を表しており，外
向きに時間軸を取って圧力の変化を定性的に示している．各塔は分
のオーダで順次工程が切換えられ，高圧時の吸着工程で難吸着成分
の酸素が製造される．各工程に要する時間が均等でないときには，
各工程を受けもつ吸着塔の数を複数にすることによって調整され
る．空気の除湿，空気分離，水素精製，溶剤回収，CO回収濃縮
など，気体分離の分野で利用される．
b.温度スイング吸着（TSA:temperature swing adsorption)
TSAでは，常温で吸着させそれと向流に流す高温（60～150℃）
流体で脱着再生させる．迅速な熱応答のために厚さ0.2mm程度
のシート状吸着材を1～4mmピッチのハニカム構造体に成型して
用いる．図2･9に示すように，二つのセクタに分けたケース内でロ
ータ状のハニカム吸着材を5分程度の周期で回転させて吸着と再生
を繰返すことにより，空気の除湿や溶剤回収が行われる．また周期
を5秒程度に速めることにより，顕熱・潜熱を同時に交換する全熱
交換機となる．ロータの幅は0.2～0.4mで，直径4m，処理流量
70000m /hまでのものが製造されている．
