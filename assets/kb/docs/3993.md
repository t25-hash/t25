# 2・3・3 固定層吸着の疑似連続化

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

装置個体としては固定層という回分式でありながら、原料供給や製品取出し位置など流体側の流路を順次切替えることによって、相対的には吸着材が移動しているとみなせる疑似連続化を図ったものに、圧力スイング吸着(pressure swing adsorption)、温度スイング吸着(temperature swing adsorption)、疑似移動層(simulatedmoving bed)などがある。a. 圧力スイング吸着(PSA : pressure swing adsorption)PSA は高圧(0.1~1MPa)で吸着させ低圧(0.01~0.1MPa)で脱着再生させるもので、吸着・脱着工程で流体は向流に流す。その一例として図2・8に合成ゼオライトを用いた酸素製造PSA の流路構成図を示す。4個の小さな平行四辺形は吸着塔を表しており、外向きに時間軸を取って圧力の変化を定性的に示している。各塔は分のオーダで順次工程が切換えられ、高圧時の吸着工程で難吸着成分の酸素が製造される。各工程に要する時間が均等でないときには、各工程を受けもつ吸着塔の数を複数にすることによって調整される。空気の除湿、空気分離、水素精製、溶剤回収、CO回収濃縮など、気体分離の分野で利用される。b. 温度スイング吸着(TSA : temperature swing adsorption)TSA では、常温で吸着させそれと向流に流す高温(60~150°C)流体で脱着再生させる。迅速な熱応答のために厚さ0.2mm程度のシート状吸着材を1~4mmピッチのハニカム構造体に成型して用いる。図2・9に示すように、二つのセクタに分けたケース内でロータ状のハニカム吸着材を5分程度の周期で回転させて吸着と再生を繰返すことにより、空気の除湿や溶剤回収が行われる。また周期を5秒程度に速めることにより、顕熱・潜熱を同時に交換する全熱交換機となる。ロータの幅は0.2~0.4mで、直径4m、処理流量70000m/hまでのものが製造されている。c. 擬似移動層(SMB : simulated moving bed)SMB では、キャリヤとなる脱着流体を並流に流して脱着再生させる。並流再生図2・8 圧力スイング吸着PSA(酸素製造)図2・9温度スイング吸着(TSA)1―31を行うとき、難吸着成分B の濃度ピークは易吸着成分A のそれより速く移動するので、両者の中間の速度で吸着材を脱着流体と逆方向に動かすことにより、B 成分は脱着流体D と同方向に、A 成分はそれと逆方向に移動して両者の分離が進行する。図2・10の流路構成図は、複数個の吸着塔を直列につなぎ、原料供給位置、脱着剤供給位置、両製品取出し位置をロータリ弁で右周りに順次切替えてこの挙動を実現したものである。炭化水素異性体分離、果糖/ブドウ糖分離等に応用されている。
