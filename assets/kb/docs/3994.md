# 2・3・4 イオン交換装置

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

水の脱イオン化すなわち純水製造として普及しており,通常図2・11に示すように,陽イオン交換塔と陰イオン交換塔の2個の下向き流固定層からなる.イオン交換が破過に達し出口の電気伝導度が所定値以上になると,図中の破線で示すように酸またはアルカリ水溶液を塔内に同方向に流してイオン交換樹脂を再生する.強型および弱型のイオン交換樹脂を複層に充てんしたもの,樹脂再生液図2・10 疑似移動層(SMB)図2・11 イオン交換装置図2・12 電気再生式脱イオン(EDI)の原理1―32産業機械・装置を向流に流すものなど多くの変種がある.
