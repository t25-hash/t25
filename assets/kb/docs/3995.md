# 2・3・5 電気再生式脱イオン装置EDI(electrodeionization)

> **章**: 第2章 化学機械
> **節**: 2・3 吸着・イオン交換装置

原理を図2・12に示すが、陽イオンおよび陰イオン交換膜で仕切って交互に配列した希釈室と濃縮室に、外部電極により電位こう配を形成させた電気透析装置において、希釈室に陽イオンおよび陰イオン交換樹脂を充てんしたものである。希釈室に供給された原水中のイオンはイオン交換樹脂に捕捉されるが、これは電位こう配の方向に樹脂中を移動、イオン交換膜を通過して濃縮室に移行する。こうしてイオン交換樹脂は薬剤をいっさい必要とせずに再生され、希釈室からは連続的に脱イオン化された純水が得られる。抽出(extraction)は、固体または液体状の原料を適当な溶剤(solvent)で処理して可溶成分を溶解、分離する操作であり、この目的で抽出装置が用いられる。装置には回分式と連続式がある。原料が液体の場合を液液抽出、原料が固体の場合を固液抽出と呼び、抽出して得られた液体を抽出液、抽出後の原料を抽残液または抽残物(原料が固体の場合)と呼ぶ。特殊な条件の操作として超臨界抽出、水性二相抽出がある。
