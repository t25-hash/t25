c .燃料高圧噴射管系 ジャーク式噴射装置のユニットイン
ジェクタ以外では，ポンプで発生させた高圧エネルギーを噴射ノズ
ルに伝達するのに燃料噴射管（injection pipe）を用いる．噴射管
内径が大きすぎると噴射圧力が上がらず，また調量できる最小噴射
量が大きくなり，逆に小さすぎると流体抵抗が増して圧力損失が大
きくなる．高圧化傾向において噴射管内の圧力変動により気泡が発
生し，崩壊することによるキャビテーションエロージョンの問題が
注目されている．噴射管内を正圧に保ち，気泡発生を抑えることが
対策として有効である．
d.給油系 定置機関などの場合は，燃料タンクを高所に配置
し重力を利用してポンプへの給油を行う場合もあるが，その他では
燃料供給用のポンプを燃料噴射ポンプに取付けたり，または内蔵す
るもの，あるいは別置きにしているものもある．
