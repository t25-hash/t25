# 2・3・2 噴射制御機構

> **章**: 第2章 圧縮点火機関
> **節**: 2・3 燃料噴射系統

プランジャによる燃料圧送工程が一定の場合でも，ポンプの回転
数が変化すると噴射特性が変るため，噴射する燃料の量は必ずしも
一定にならない．このため回転領域によっては，図2･23のように
エンジンの負荷トルクのこう配よりも機関の発生トルクのこう配が
大きくなることがある．このため図2･24のように機関トルクを制
御する調速機が必要である．調速機，噴射ポンプ，機関の制御系は
閉ループを構成する．したがって，機関の負荷特性，噴射ポンプの
噴射量特性などを考慮して，調速機の仕様を決める必要がある．
燃料ポンプが圧送を開始してから，圧力波がノズルに伝わり噴射
が開始するまでに噴射遅れ期間があり，一方，機関自体には着火遅
れ期間がある．これらの遅れ期間は機関の回転数にそれほど影響さ
れないため，回転数が高くなると着火時期（クランク角度）が遅く
なる．このため使用回転域の広い機関では，回転数に応じて最適な
着火時期が得られるような噴射時期制御装置が必要になる．また始
動性改善，騒音低減のために回転数だけでなく負荷に対しても噴射
時期を制御する装置が必要になってきている．
a.噴射量制御機構 調速機（speed governor ）はアイドリン
グ回転数から最高回転数まで，あらゆる回転数範囲で制御を行うオ
ールスピード調速機，アイドリング回転数制御と最高回転数制御を
行い中間域の回転数では制御を行わない高・低速調速機，さらに，
中間域の回転数制御を細かく行うハーフオールスピード調速機があ
る（図2･25）．
車両用としてはガソリン機関の運転特性に類似しているハーフオ
ールスピード調速機や高低速調速機が一般的であり，建設機械，発
電機などには負荷変動に対して回転変動の少ないオールスピード調
速機が使用される．
発電用などの定速エンジンでは，次式で定義される瞬時回転数変
動率（instantaneous speed change）と整定回転数変動率（perma-
nent speed change）δの規格を満足するように調速機の仕様を定
める必要がある．
δ＝－×100% (2･18)
ただし， ：瞬時最高（整定）回転数， ：定格回転数．
さらに外部条件，運転感覚などに合った細かい制御特性を得るた
め，電子技術を利用した調速機も実用化されており，図2･26に電
子制御調速機の原理図を示す．
b.燃料噴射時期制御機構（automatic timing device) 機構
的には，機械式，電気式などがある．
図2･27に機械式タイマの一例を示す．二つのフライウェイトは，
回転数によって増減する遠心力により変位する．遠心力とばね力と
が釣合ったところで二つの偏心カムを介して機関駆動軸と噴射ポン
プ駆動軸との位相差が定まり，噴射時期が決定される．
回転数のみでなく，負荷やその他の外部条件に応じた複雑で精度
の高い精密な制御特性を得るには電子技術を利用したタイマが用い
られている．図2･28に模式図を示す．
