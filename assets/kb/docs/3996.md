# 2・4・1 液液抽出装置(liquid liquid extractor)

> **章**: 第2章 化学機械
> **節**: 2・4 抽出装置(extractor)

工業的には連続式が一般的であり,装置の形式はミキサセトラ式,塔式および遠心式に大別される.a. ミキサセトラ式(mixer-settler extractor)図2・13のように,二液をかくはん,混合して抽出を進行させる槽(ミキサ)とその混合液を静置して密度が大の液(重液)と小の液(軽液)とに相分離させる槽(セトラ)の二つの組合せから構成されている.操作が容易で,操業範囲が広く,連続多段で運転される場合が多い.b. 塔式装置(column type extractor)多段ミキサセトラの機能を一つの塔に収めて設置面積と装置コストの低減を図っている.重力のみを向流接触の推進力とするスプレー塔,多孔板塔(図図2・13 ミキサセトラ図2・14 多孔板抽出塔図2・15 往復回転式抽出塔ぼ同じであり,一般的に広く使用されている.一方,機械的動力を利用して抽出効率の向上を図るものとして,回転かくはんを与えるシャイベル塔(Sheibel column),回転円板塔(R.D.C.: rotatingdisc contactor),周期的に回転方向を反転させる往復回転式抽出塔(図2・15),多孔板を上下運動させるカール塔,装置内の液を脈動させる脈動塔(pulsed column)などがある.抽出効率が高いが,機械的動力が大きすぎると微細なエマルションが形成され,分離が困難になり,飛まつ同伴を起こすので最適な運転条件を選ぶことが必要である.c. 遠心式抽出装置(centrifugal extractor)遠心力を利用して,二液の接触と分離の効率を上げ,装置の小型化を図っている.ポドビルニアク抽出機(Podbielniak extractor)(図2・16),十字流接触遠心抽出機などがある.比重差の小さい液体や,粘度の高い液体の処理に有利であり,また不純物の混入や原料の損失などを極力抑えたい場合にも使用される.医薬品の分離精製や放射性物質の分離に利用されている.
