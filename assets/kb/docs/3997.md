# 2・4・2 固液抽出装置(solid liquid extractor)

> **章**: 第2章 化学機械
> **節**: 2・14 )、充てん塔などの構造は蒸留やガス吸収で使われるものとほ

対象とする固体の大きさや性状に対応して各種装置が使用される。処理量が少ない場合は回分式抽出槽が使われ、多い場合は連続式抽出機が使われる。a. 抽出槽下部に多孔板または格子を備え、その上にろ布を敷いた槽で、その中に原料を装入し溶剤と接触させる。粉体用として、エアリフトによるかくはんを利用したパチューカタンク(pa-chuca tank)がある。b. 連続式抽出機原料の入った網底バスケットを移動させながら溶剤と原料を向流接触させるボールマン抽出機(Bollmanextractor)、回転移動する抽出槽に溶剤を噴霧、貫流させるロトセル抽出機(Rotocell extractor)(図2・17)などがある。スクリューや掻取り羽根、パドルなどで固体原料を移動させながら溶剤と向流接触させるタイプとして、ケネディ抽出機(Kennedy extracor)、ヒルデブランド抽出機(Hildebrandt extractor)などがある。
