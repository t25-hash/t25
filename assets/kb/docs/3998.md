# 2・4・3 特殊な抽出プロセスとその装置

> **章**: 第2章 化学機械
> **節**: 2・14 ),充てん塔などの構造は蒸留やガス吸収で使われるものとほ

a. 超臨界抽出臨界条件以上の流体の特殊な溶解性,物性図2・16 ポドビルニヤク抽出機(旧版図2・3・4を簡略化)図2・17 ロトセル抽出機変化を利用するのが超臨界抽出である.圧力,温度の調節で溶剤と原料の分離が容易に行われる.抽出速度も大きいという利点があり,熱に弱い香気成分などの抽出に有利である.高圧操作に対応できる装置が必要で充てん槽式のものが一般に使用される.b. 水性ニ相抽出ある種の高分子水溶液がある条件で二相に分離し,タンパク質やアミノ酸などの生体物質の分配特性が塩濃度やイオン強度に依存する性質を利用する.小規模なミキサセトラが装置として使われる.c. その他の抽出操作逆ミセル抽出や液膜分離など新しい抽出操作が提案されており,開発が進められている.ガスの湿度と温度を任意に調節することを調湿といい,ガスが空気の場合には空気調和(air conditioning)ともいう.材料に熱を与えて水分等の湿り成分を気化蒸発させ,乾き固形製品を得ることを乾燥という.
