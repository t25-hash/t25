# 2・5・1 各種調湿、冷水、乾燥装置

> **章**: 第2章 化学機械
> **節**: 2・5 調湿・冷水・乾燥装置

a. 調湿装置増湿の場合は、空気中へ水を噴霧したり、充てん物により形成された水膜と空気を接触させたりする、空気と水の直接接触法が一般的である。減湿の場合は、湿りガスを露点よりも低い温度の液面(直接冷却)または固体面(間接冷却)と接触させて冷却し、蒸気の一部を凝縮除去する冷却減湿法が一般的である。直接冷却減湿装置(dehumidifier)では、直接接触増湿装置(humidifier)のほとんどをそのまま利用でき、多量の処理に適している。増・減湿装置ともに噴霧室、噴霧塔や充てん塔がよく採用され、一般に充てん塔の性能が高い。代表的な横形噴霧室式増湿装置(horizontal 乾燥装置被乾燥材料には、大きな成型品から粉粒体まで、湿り固体が多いが、泥状、液状のものもあり、材料の性質や熱源の種類等により、多種の乾燥器が考案されている。一般に少量処理には回分式、大量処理には連続式が適しており、加熱方式と乾燥器内の材料状態により、表2・2のように分類される。以下では湿り粉粒体を対象とした代表的な乾燥器(dryer)について述べる。粉粒体の乾燥では、粒子と熱風の接触を促進させる工夫が重要であることから、材料を動かさない静置式やその原理を連続式に応用した移送式の乾燥器が利用されることは少ない。. 流動層乾燥器(fluidized-bed dryer)多孔板などのガス分散板上の湿り粉粒体材料層に下方より熱風を送って材料層を流動化させつつ乾燥させる装置で、粒子と熱風との接触が良いため熱効率も良く、限界含水率を低くでき、大量処理も可能である。回分式、連続式ともに広く利用されており、連続式の場合でも材料の平均滞留時間を任意に設定できる長所を有する。熱風の持ち込む分だけでは熱エネルギーが不足するとき、層内に加熱管を入れて伝導加熱を併用する場合もある。伝導加熱併用形流動層乾燥器の例を図。 気流乾燥器(pneumatic conveying dryer)湿り粉粒体を高速で流れる熱ガス中に分散して供給し、熱ガスと並流に送りながら乾燥する連続式熱風乾燥器である。可動部分のない簡単な構造であり、熱効率も高い。乾燥器内での滞留時間が短いことからフラッシュドライヤと通称されることもあるが、水分の保有力が強い材料では十分な乾燥が行えないときもある。分散器付の例を図2・21に示す。. その他の乾燥器赤外線や高周波乾燥器のように、基本原理の異なるものを除くと、その大部分は、熱風と粒子の接触を促進するという基本原理に基づいており、使用条件によって構造上変形したものである。粉砕(grinding, milling, crushing, comminution, pulveriza-tion)は、機械的な力を利用して固体を細かくし、粉を作る操作であり、凝集粒子の解砕、複数種類の粉体の混合・分散、粒子の表面改質、固体の活性化などに利用される。一方、造粒(granulation, agglomeration, pelletization)は粉末、溶液、泥しょう等から取扱いに便利な形態と材質の粒状物あるいは小成形物を作る操作であり、医薬品(顆粒や錠剤等)、食品(角砂糖やインスタントコーヒー等)、農薬や肥料(粒状品)、日用品(洗剤など)等の最終製品製造のほか、製鉄、窯業、粉末冶金などの工業における物質の流動、反応、成形、溶融、焼結などの操作性改良に利用される。
