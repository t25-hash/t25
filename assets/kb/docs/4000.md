# 2・6・1 粉砕装置

> **章**: 第2章 化学機械
> **節**: 2・6 粉砕・造粒装置

粉砕は,原料の大きさと生成する砕成品粒度に応じて,粗砕,中粉砕,微粉砕などと呼ばれるが,これらに明確な区別はなく,あくまでも便宜的である.より広い許容・到達粒度範囲を示す分級機内蔵型ミルや,メカノケミカル現象を利用する粉体合成や,メカニカ(a) メカノフュージョン装置(b) ハイブリダイザ(c) マイクロス図2・23 代表的粒子加工型ミルルアロイング,表面改質が可能な粒子加工型ミルの開発も盛んである.a. 代表的粉砕機(ミル)[typical grinders (mills, crushers,pulverisors)]粉砕域の目安となる粒径の幅と使用される代表的な粉砕機を表2・3に示す.これら粉砕装置の詳細については,粉体工学便覧(第2版),化学工学便覧(改訂6版)を参照されたい.b. 粒子加工型ミル(mills for particle design)粉砕という操作を利用して材料の化学的あるいは物理化学的特性を変化させ,あるいは機能を付与する装置として粒子加工型ミルが各種開発されている.最近注目されているのは,メカノケミカル効果による粒子の表面改質・複合化,メカニカルアロイングである.典型的なものを図2・23に示す.なお,表2・3に示した遊星ミル,媒体かくはんミルなどもその機能をもっている.
