# 2・6・2 ミル設計の最適化(optimization for mill design)

> **章**: 第2章 化学機械
> **節**: 2・6 粉砕・造粒装置

ミル内部形状・構造の望ましい設計については,従来,最適と思われる装置を製作し,労力と時間を要してデータを得て決定する,いわゆる経験的手法に依存してきた.最近,一部のミルに限定されるが,離散要素法[discrete element method(DEM)]に基づくコンピュータシミュレーション(CS)法(computer simulationmethod)を用いて,操作条件やミル内構造の最適化を図る研究があり,産業界から評価を得ている.このCS 法によるとミル内媒体や回転部品の運動が空間的,時間的に詳細に再現でき,装置を製作することなく操作条件やミル構造の最適化が可能になっている.これは,これまでの経験的手法からの脱却であり,大幅なコスト,エネルギー,労力の削減をもたらすものと期待される.操作条件に関する例として,図2・24には遊星ミルの自転と公転運動を順方向にした場合と逆方向の場合での媒体(ボール)の速度(ベクトル表示)の違いを示す.図2・24より,逆方向のほうがボールの速度が(a) 逆方向(b) 順方向図2・24 遊星ミルの自転と公転運動による媒体の速度ベクトル表2・4 典型的造粒装置の概略分類形式123回転パン回転ドラム/コーン回転水平円板成長様式の造粒転動造粒回転ナイフ(垂直)回転ナイフ(水平)回転バー圧密様式の造粒解砕造粒スプレー塔板上滴下液中滴下液滴様式の造粒液滴固化造粒1―35相対的に大となり,粉砕に好都合であることがわかる.また,セメント用大型転動ミル内におけるボール運動のライナ形状による違いについての検討では,摩耗前のライナによるとボール群はミル回転に伴って全体的に持ち上げられ,その落下運動も激しくなるが,摩耗が進行するとボール運動が著しく低下する状態になる.その他,CS 法と実験との融合によって粉砕速度,メカノケミカル反応速度が予測可能になっている.
