# 2・6・3 造粒装置(granulator,agglomerator,pelletizer)

> **章**: 第2章 化学機械
> **節**: 2・6 粉砕・造粒装置

造粒装置は、下記のように造粒様式により分類される。a. 成長様式の造粒転動、流動層、噴流層、かくはんなどの作用で運動している湿潤粉体、あるいはこれらの作用下にある乾燥状粉体に、スプレー液(バインダ)を供給し、凝集させて顆粒、コーティング造粒物、ペレットなどを作る方法。b. 圧縮様式の造粒乾燥状粉体、またはバインダを含む湿潤粉体を押出し、解砕、ロール間圧縮[ロール表面に多数のポケット(凹部)がある]、打錠などによってペレット、錠、ブリケットなどを作る方法。c. 液滴様式の造粒滴下またはスプレーで溶融液を冷却しながら気流中、あるいは液中に分散・固化させ、顆粒状の造粒物(ビーズ、プリルともいう)を作る方法。表2・4には典型的な造粒装置の概略をまとめて示す。なお、造粒装置類の詳細については、造粒ハンドブックを参照されたい。
