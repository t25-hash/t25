# 2・7・1 分級(classification)

> **章**: 第2章 化学機械
> **節**: 2・7 粉粒体分級・貯蔵・供給

粉粒体を粒子個々の特性の差異によって分ける操作を総称して分級というが、普通は粒子の大きさによって分ける、いわゆる粒度分級を指す。分級性能としては、分級点におけるカット特性が重要であり、分級効率を粒子径の関数として表した部分分級効率を装置の性能表示に用いることが必要である。その他、いろいろな分級効率が定義されているが、これらは原料粉体の粒子径分布に依存し、分級装置に固有のものとはならない。分級操作は湿式と乾式に大別される。微粉体はふつう凝集しており、湿式分級は微粒子の分散が比較的容易であるという特徴があるが、排水処理や製品の乾燥を要するという欠点がある。乾式では微粉体の分散に特に注意する必要があり、乾式分級機は内部に粉体の分散機構を備えた機種も多い。a. 湿式分級(wet classification)。 重力分級機(沈降分級機)液中での粒子の重力沈降を利用する分級機。角型あるいはコーン型の沈降槽に原液を整流して供給し、沈降した粒子を粗粒、溢流した粒子を微粒として回収する。部分分級効率は粒子の重力沈降速度を用いて推定する。. 水力分級機沈降分級機の底部から圧力水を供給して上昇流を作り、この流速と粒子の沈降速度の差を利用して分級する。. 機械式分級機重力沈降を利用するが、粗粒を掻き揚げて回収する可動機構をもつ。. 遠心分級機粒子の遠心沈降速度は、重力沈降速度の数百~数万倍にでき、100 m以下の粒子の湿式分級には主として遠心分級機が用いられる。液体サイクロンや、内部に回転体をもつ強制渦型の遠心分級機がある。b. 乾式分級(dry classification)。 重力分級機水平流型と上昇流型に大別される。水平流型は沈降速度差による粒子の落下位置の違いを利用する。上昇流型は水力分級機と同様、沈降速度と上昇流速度の差により粗粒と微粒に分ける。. 慣性分級機気流の方向を急変させると、粒子の軌道は慣性の差によってそれぞれ変化するが、この現象を利用して粒子を分級する。直線型、曲線型、ルーバ型、インパクタ型などがあり、インパクタ型は高精度分級ができる。. 遠心分級機自由渦型と強制渦型があり、前者ではサイ1―36産業機械・装置図2・25ターボクラシファイア(強制渦型遠心分級機)クロンが広く実用され、最近はサブミクロンの分級も可能となっている。また、後者の強制渦型は高速回転体を有するため構造は複雑であるが、分級精度が良く、分級点の変更が容易にできるので、30m程度以下の微粒子の分級に最も広く実用されている。図2・25にその一例を示す。c. ふるい分け機。 振動ふるい網面に対し垂直方向の振動成分をもつふるいで、ふつう粗大粒子のふるい分け機は大振幅・低振動数であり、微細粒子のそれは小振幅・高振動数である。微粉では超音波振動も用いられる。網面の運動形式は円、だ円、直線、円弧、スパイラルなど多様である。. 面内ふるい網面に対し垂直方向の振動成分がほとんどないふるい機。網面の運動形式は多様である。. その他のふるい回転ふるい、二重円筒ふるい、気流を併用するふるい、強制かくはん式ふるいなどがある。
