# 2・8・1 概説

> **章**: 第2章 化学機械
> **節**: 2・8 混合・かくはん装置

混合・かくはん操作は幅広い分野で日常行われている操作であ
る．ここで，お互いに溶解し合う2種類以上の流体を混ぜ合わせる
操作は混合操作と呼ばれ，混合終了の条件は，混合物からあるサイ
ズのサンプルを採ったときどのサンプルも一様であることが条件で
あり，化学反応を行う場合には分子サイズまで一様に混ぜ合わせる
ことが要求される．混ぜ合わせる二流体がお互いに溶解しない場合
の混合操作は分散操作とも呼ばれ，この場合一方の流体を連続相流
体，他方の流体を分散相流体として混ぜ合わせることになる．例え
ば，液体に解け合わない流体が液体か気体かにより，それぞれ液滴
か気泡として液体中に分散させることが混合操作となる．液体中に
固体粒子を分散させる操作も混合操作として取扱われる．さらに広
範な混合操作の定義では，2種類以上の粉体粒子を均一に混ぜ合わ
せる操作も混合操作に含めることがある．
混合・かくはん装置は，操作方法によって次に述べる3種類に大
別される．これら装置の選択は，混合・かくはんの主目的が何であ
るか，混合すべき流体の性質はどのようであるか，混合すべき流体
の量はどれほどであるか，混合操作にどれほどの時間を必要とする
か，装置の投資価格はどれほどであるかなど，多くの条件を加味し
て決められる．
