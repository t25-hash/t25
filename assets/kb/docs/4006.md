# 2・8・1 概説

> **章**: 第2章 化学機械
> **節**: 2・8 混合・かくはん装置

混合・かくはん操作は幅広い分野で日常行われている操作である。ここで、お互いに溶解し合う2種類以上の流体を混ぜ合わせる操作は混合操作と呼ばれ、混合終了の条件は、混合物からあるサイズのサンプルを採ったときどのサンプルも一様であることが条件であり、化学反応を行う場合には分子サイズまで一様に混ぜ合わせることが要求される。混ぜ合わせる二流体がお互いに溶解しない場合の混合操作は分散操作とも呼ばれ、この場合一方の流体を連続相流体、他方の流体を分散相流体として混ぜ合わせることになる。例えば、液体に解け合わない流体が液体か気体かにより、それぞれ液滴か気泡として液体中に分散させることが混合操作となる。液体中に固体粒子を分散させる操作も混合操作として取扱われる。さらに広範な混合操作の定義では、2種類以上の粉体粒子を均一に混ぜ合わせる操作も混合操作に含めることがある。混合・かくはん装置は、操作方法によって次に述べる3種類に大別される。これら装置の選択は、混合・かくはんの主目的が何であるか、混合すべき流体の性質はどのようであるか、混合すべき流体の量はどれほどであるか、混合操作にどれほどの時間を必要とするか、装置の投資価格はどれほどであるかなど、多くの条件を加味して決められる。
