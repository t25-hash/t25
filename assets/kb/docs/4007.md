# 2・8・2 かくはん翼を備えた混合装置

> **章**: 第2章 化学機械
> **節**: 2・8 混合・かくはん装置

かくはん翼(impeller)を備えた混合装置[かくはん装置(agitated vessel)]の基本的な構成は,図2・27に示すように,流体の入った円筒容器の上部からかくはん翼を挿入し,モータによってかくはん翼を回転することで,槽内の液体をかくはんするようになっている.かくはんする流体の性質によっては円筒槽の側壁または近傍にじゃま板(baffle plate)と呼ばれる平板を数枚垂直に取付けられることがある.液体中に気体を分散させる気液かくはん操作では,槽底部分に気体吹込み用のスパージャを取付けて操作する.かくはん翼は取扱う液体の性質によってその形状は多種多様である.a. 低粘度流体に適した混合装置低粘度流体(0.001~0.1Pasオーダ)をかくはんする場合,操作は乱流状態となるため,比較的小型の翼を用いる.低粘度流体のかくはんでよく用いられるかくはん翼としては,タービン翼(turbine impeller),パドル翼,傾斜パドル翼,プロペラ翼がある.発酵槽のような縦長の円筒槽に対してはかくはん翼を多段に取付けたかくはん装置が用いられる.b. 高粘度流体に適した混合装置高粘度流体(1~100Pasオーダ)をかくはんする場合,操作は層流状態となり,流体の動きが翼近傍に限られるため,かくはん翼は槽全体をかくはんできる大型の翼形状が採用される.また,槽側壁からの熱移動を促進するため翼先端と槽側壁の間げきはできるだけ小さくするよう工夫される.高粘度流体といっても流体の粘度範囲には幅があるため粘度に応じて(したがって,かくはんレイノルズ数に応じて)翼の形状も変えられる.汎用的な高粘度流体用のかくはん翼には,ヘリカルリボン翼(helical ribbon impeller),アンカー翼,スクリュー翼,ゲート翼がある[a,bでのかくはん翼形状については文献(51)参照].c. 広い粘度範囲に適した混合装置混合・かくはん操作の間に化学反応等により流体の粘度が低粘度から高粘度へと時間変化する場合がある.この場合,どんな粘度の流体に対しても良好な混1―37合・かくはん操作が行えるかくはん翼が要求される.この要求に対応して近年多くのかくはん翼が開発されてきた.その一例を図2・28に示す.図2・28に示したかくはん翼は汎用的な使用において差異はないが,詳細にはそれぞれ特徴があるので使用に際しては混合・かくはん目的を明確にして翼の選択をされることをお勧めする.d. かくはん翼の挿入方法の変形石油タンクのように円筒槽が大きくなると上部からのかくはん翼の挿入は不可能となる.この場合はかくはん翼を円筒槽の側壁から挿入する方法がとられる.通常円筒槽の周方向に3~4個取付けられる.また,化学反応等の操作上円筒槽の上部からかくはん翼を挿入できない場合もしばしばある.このような場合には槽の底面からかくはん翼が挿入される.この場合シャフトのシールに対する配慮が必要である.e. 翼を振動させる混合装置特殊な翼を振動モータによって液中で振動させ,そのとき発生する吐出流で槽内全体をかくはんする装置である.めっき槽,廃水処理槽等の幅広い分野でのかくはん操作に有効である.f. 横型二軸混合装置かくはん翼を用いる混合・かくはん操作が可能な流体の粘度の限界は100Pas程度と考えられている.プラスチック成形で用いられる押出式かくはん機(エクストルーダ,extruder)は超高粘度流体の混合装置として使用されることもある.
