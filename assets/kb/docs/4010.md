# 2・9・1 固液機械的分離装置

> **章**: 第2章 化学機械
> **節**: 2・9 固液・固気機械的分離装置

液中に懸濁している固体粒子を機械的に分離する固液分離操作には、ろ過、圧搾、沈降、遠心分離などがあり、化学工業における分離精製プロセスから廃液処理に至るまで広範に利用されており、その目的に応じて最適な装置を用いる必要がある。a. ケークろ過装置スラリー中に懸濁している不溶解物質をろ布、ろ紙、金網、膜、粒子層等のろ材により捕捉粒子とろ液とに分離する操作を一般にろ過(filtration)といい、古くから工業の広範な分野で用いられている。捕捉粒子径やろ材の細孔径にもよるが、通常1vol% 以上の固体を含むスラリーのろ過では、ろ過開始後まもなく捕捉粒子からなるろ過ケークがろ材面上に形成され、そのケークが以後のろ過に対してろ材として作用しつつろ過が進行する。この種のろ過をケークろ過(cake filtration)という。ケークろ過器は、ろ液流動のための推進力によって、重力ろ過器、加圧ろ過器および真空ろ過器に大別される。加圧ろ過器はバッチ式が多く、ポンプ圧または圧縮空気圧をスラリーに作用させてろ過を行う。比較的高圧が利用できるので高いろ図2・30 板枠型圧ろ器図2・31 press)では、側面にろ布を張ったろ枠とろ液流路となる溝をもつろ板を交互に並べて端板間で締付け、ポンプによりスラリーをろ枠の中へ圧入してろ過を行う。ろ枠内にろ過ケークが充満しろ過速度がある値まで低下するか、ろ室内部の圧力が許容値まで達したら、その時点でろ過を中止する。次いで、ケーク洗浄、装置分解、ケーク除去を行い、再び装置を組立てこの一連のサイクルを繰返す。真空ろ過器では、ろ材の背面に真空を作用させ上流側を大気圧としてろ過を行うため、連続ろ過が容易である。ろ過、ケーク洗浄、脱水、ケーク除去などの一連の操作が周期的に行われる。大量処理が可能で維持費が安価であるが、利用する圧力が真空圧のため低く、難ろ過性スラリーのろ過には不適である。多室円筒型真空ろ過器(オリバー型ろ過器)では、図2・31のように上部開放で液面が一定に保たれた原液槽にろ過円筒を浸しつつ回転させながら連続真空ろ過を行う。円筒の周囲は多数の小ろ過室に分割され、各ろ過室はそれぞれパイプで中央部の自動弁に連結されている。自動弁はろ過、ケーク洗浄、脱水区間では真空側、ケーク除去の区間では圧縮空気側に直結されている。通常、円筒の回転速度は1/3~3rpm程度、浸液率は40~65%程度である。生成するろ過ケークによって引き起こされるろ液の流動抵抗をある値以下に抑えて高い分離速度を維持しつつろ過を行うための、ろ過ケークの掃流機構を備えたろ過器が開発されており、これをケークレスろ過器(ダイナミックろ過器)といい、種々の形式がある。b. 圧搾装置圧搾(expression)は、スラリーや半固体状の固液混合物を、液体は通過させるが固体は通過させない隔壁内に収容して、これを機械的に圧縮脱水して圧縮ケークと搾出液とに分離する操作である。ろ過操作で得られるろ過ケークより、さらに含水率の低いケークが生成し、ろ過に比べてさらに高度な脱水を目的とする場合に用いられる。乾燥、燃焼処理、埋立、造粒などの前処理に利用されることも多い。各種食品工業、発酵工業、金属工業のほか、上下水汚泥、各種廃液や廃棄物の処理にも広く用いられている。圧搾装置はプレスともいい、バッチ式と連続式とがある。バッチ式には、圧搾型フィルタプレス、チューブプレス、プレートプレス、ケージプレスなどがある。一方連続式には、ベルトプレス、スクリュープレス、円板プレスなどがあり、近年その適用範囲が拡大している。バッチ式のフィルタプレスでは、図2・32のように0.3~0.5MPa程度の圧力で凹板のろ室内にろ過ケークを形成させた後、ろ布の裏側に設けたゴム製のダイヤフラム内に高圧の流体(空気圧の場合1.5MPa、水圧の場合2.0MPa)を導入してダイヤフラムを膨張させることによりケークを圧搾脱水する。連続式のベルトプレス(belt press)では、図2・33のように2枚のベルト間に試料を挟んで、多数のロールの間をベルトにより屈曲移送させて圧搾し、ケークを排出する。ろ布の張力によりロール曲面に作用する圧縮圧力と屈曲走行によって生じるせん断力を利用して圧搾脱水を行う。脱水過程は、凝集操作による前処理区間、遊離水を分離し供給スラリーを濃縮する重力脱水区間、および圧搾脱水区間よりな図2・32 圧搾型フィルタプレス図2・33 沈降および遠心分離装置液中の懸濁粒子を、重力を利用して沈殿濃縮させる装置をシックナ(thickener)といい、図2・34のように底部が緩傾斜の円すい状をなす浅い円筒型槽が普通である。スラリーは槽の中央部から供給され、清澄液は外周の溢流ぜきから流出する。固形分は、圧密沈降の過程で濃縮脱水され、濃縮汚泥として槽底中央部より排出される。レーキと呼ばれる排泥機構を備えていることに特徴があり、下底に沈積する大量の濃縮汚泥を円滑に排泥口に集めて排出を促進させる。クラリファイヤは、その構造はシックナのそれとよく似ているが、清澄液を得ることが目的である点が異なる。試料を高速回転場に置き、遠心力を作用させて行う遠心分離も有力な固液分離の手段の一つである。遠心分離機は他の分離装置に比べて装置の床面積を小さくできるという利点があるが、1台当たりの処理量が小さく、操作費や維持費が比較的高いという欠点がある。遠心分離機は遠心機ともいい、遠心沈降機と遠心ろ過・脱水機に大別される。
