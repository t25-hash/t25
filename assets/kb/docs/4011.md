# 2・9・2 固気機械的分離装置

> **章**: 第2章 化学機械
> **節**: 2・9 固液・固気機械的分離装置

気体中に浮遊しているダストやミストなどの粒子状物質を分離除去する操作を集じん(dust collection)といい,生産活動で得られる有価物の回収や大気汚染物質の除去などの目的で使用される.集じんの対象となる粒子の大きさはサブミクロンから数百ミクロン,濃度は数十g/m程度の高濃度から数個/m程度の極低濃度までと非常に広範囲に及ぶ.このため,対象に応じて用いられる粒子分離形態,装置様式はさまざまである.集じん装置は,粒子分離原理により,重力集じん,遠心力集じん,慣性集じん,電気集じん,ろ過集じん,洗浄集じんなどに分類できるが,気流と粒子の分離形態により分類すると,流通形式,障害物形式,隔壁形式の三つに分けられ,模式的には図2・35のように示される.また,図2・35には代表的な装置の例を示す.各形式の特徴は以下のとおりである.a. 流通形式重力,遠心力,静電気力などの外力を作用することにより粒子を流れの系外に移動させる形式であり,捕集性能は時間的に,ほとんど変化しない.圧力損失は処理ガスの流通抵抗のみなので,外力の大きさに応じ,処理速度を大きくできる.したがって,重力集じん装置,遠心集じん装置,電気集じん装置などがこの形式に属する.集じん装置の集じん率は,装置へ流入した粉じんのうち除去された粉じん量の割合で定義されるが,本形式の場合,粒子の捕集面への移動速度,全捕集面積,処理ガス流量および含じん気流の流動状態により図2・36のように定まる.すなわち,は流動状態によらず,とともに増加するが,同一のでは乱れ,すなわち気流混合度が大きいほど集じん率は悪くなる.この形式に属する個々の装置に固有の特徴は以下のとおりである.. 重力集じん装置(gravitational dust collector)装置に流入した粒子のうち装置出口までに重力沈降により床面に到達した粒子のみが分離される.重力沈降速度は粒径が大きいほど速いので,分離率も大粒子ほど高くおもに数十m以上の大粒子の予備集じんに用いられる.1―40産業機械・装置. 遠心集じん装置(centrifugal dust collector)気流に旋回運動を与え粒子の遠心力によって気体中から分離捕集する装置であり,サイクロンが最も代表的である.小型ほど高集じん率が得られるが処理風量が大きいときは多数を並列(マルチサイクロン)にして使用する.構造が簡単で可動部がなく,粒子の沈降速度を,重力沈降速度より大きくできるので,数m程度の微小粒子の分離が可能である.. 電気集じん装置(electrostatic precipitator)コロナ放電によって荷電された粒子を電界に導き,集じん電極に捕集する.性能は高く,バグフィルタに匹敵する.さらに,圧力損失が小さいので大風量用集じん装置として広く使用されている.電気集じんの集じん性能は粒子の見掛け電気抵抗率ρ욃により大きく影響される.一般にρ욃<10Ωmでは荷電,捕集後の電荷の放出ともに容易かつ速いので,粒子は効率良く捕集される反面,電荷を直に失い逆に静電誘導により極板と同極性に荷電され,粒子は電界中に引き戻され,再飛散するため集じん率は低下する.逆にρ욃>5×10Ωmでは,粒子は荷電されにくくしかも極板上での電荷の放出速度が遅いため堆積粉じん層内にイオンが蓄積し,層内で図2・35 気流及び粒子分離形態による集じん装置の分類電位が急激に降下する.このため最終的には電位降下が過大となって逆電離し,集じん率を著しく低下させる.これに対し10<ρ욃<5×10Ωmでは電荷は適当な速度で放出され粒子の再飛散や逆電難が起きないので集じん率は高い.粒子の見掛け電気抵抗率ρ욃は温度に対して最大値をもち,その値は水分率が低いほど高くなる.これは低温では表面電導,高温では内部電導が主となるためである.したがって,ピーク値が5×10Ωmを超えるような場合には,ガス温度を高温または低温にすることにより見掛け電気抵抗率を適正範囲に調整することがある.b. 障害物形式気流中に円柱,球などの障害物を挿入して,気体-粒子間に大きな相対運動を作り,粒子を障害物に捕集する集じん方式である.分離された粒子は障害物の集合体内に残るので,捕集性能は時間とともに向上するが,同時に圧力損失も増加する.固体粒子の場合,捕集された粒子を障害物表面から除去することは困難なので,使い捨てされることが多い.この形式の装置の集じん率は,障害物一つ当たりの粒子捕集効率ηがわかれば一般に次式で表現できる.=1-exp-η(2・13)ここで:全捕集表面積,:集じん装置体積,:集じん装置長,また,は障害物形状により定まり,=1πε,球では=14ε,εは障害物の空げき率である.ηは捕集体の形状,気流,粒子物性などにより大幅に変わるが,図2・37は,種々の形状の障害物による慣性捕集効率の計算値を示したものである.この形式に属する個々の装置に固有の特徴は以下のとおりである.. 慣性集じん装置(inertial separator)気流の急激な方向変換により,その中に含まれる粒子の慣性力を利用して分離捕集する装置であり,圧力損失を1kPa(100mm水柱)以下にすれば予備集じん用として使用できる.. 洗浄集じん装置水または液体を主たる捕集媒体として,粒子を慣性,拡散,さえぎり作用によって液中に捕集する装置で,スクラバとも呼ばれる.各種の型式があるが,ベンチュリスクラバは圧力損失が高いが細かい粒子もよく捕集できる.. エアフィルタ(air filter)ろ材内部で粒子を捕集する装置であり,大気じんのような低濃度の粉じん処理に使用される.HEPA あるいはULPA フィルタなどと呼ばれる高性能フィルタは非常に高い清浄度が要求される半導体製造工程,病院手術室などの局所空間の清浄に多く使用される.. 粒子充てん層フィルタ(packed bed filter)ろ材として砂などの粒状物質を捕集体として用いた集じん装置である.従来実用装置としてほとんど使用されなかったが,高温,摩耗性粉塵の捕集にも使用できるので近年注目されるようになってきた.この形式では,捕集粉じんがフィルタ内部に堆積するので,充てん粒子の払い落としをいかに連続運転するかが重要である.また,この形式では時間が経過すると捕集粒子の再飛散が避けられない.c. 隔壁形式気体は通すが粒子はほとんど通さない捕集体を,流路をさえぎるように配置して,粒子を捕集体表面に分離する形図2・36 流通形式集じん装置の集じん率図2・37 種々の形状の障害物の慣性捕集効率式.この場合も圧力損失は時間とともに増加するが,適当な間隔で堆積粉塵を除去し,捕集体を再生すれば,連続運転が可能であり,高濃度の粉じんが発生する産業の各分野で広く使用されている.特にバグフィルタは,集じん性能が非常に高いので,小規模装置として広く使用されている.また,セラミックフィルタは高温ガス処理,特に高温ガスの熱エネルギー回収やダイオキシン発生抑止装置として用いられる.この形式に属する装置固有の特徴は以下のとおりである.. バグフィルタ(bag filter)織布またはフェルトのバグを装置内に懸垂した構造をしており,払い落とし方式により振動,逆洗,パルスジェット式に分類できる.振動式:ろ布の上部または中央部付近を振動して堆積粉じんを払い落とす方式である.振幅と周期を変化することで払い落とし力を制御できるので付着性の強い粉じんにも使用できる.逆洗式:ろ過方向と逆向きに清浄空気を吹込むことにより払い落とす方式である.しかし払い落とし効果が弱いのではく離性の良いダストに用いられる.パルスジェット式:圧縮空気をろ布上部から間欠的に極短時間・噴出する払い落とし方式であり,払い落とし中も気流を停止する必要はない.このため風量変動が少なく高濃度排ガス処理に適し,装置の小型化も可能である.. セラミックフィルタ(ceramic filter)ろ材としてセラミックを用いたものであり,バグフィルタと同様織布にしたものと,多孔質焼結体やセラミック繊維成形体などがある.いずれも高温高圧集じん用に開発されたものである.いずれも捕集性能に問題はないが,引張強度などの機械的特性および寿命などが加熱により大きく低下するので,価格とともに耐久性が解決されるべき課題である.以上のように,集じん装置は,適用可能粒度範囲,集じん率,圧隔壁形式外部ろ過バグフィルタ>10.0110000セラミックフィルタ>10.0110000表2・8膜分離プロセスの分類と応用分野分離法操作圧力透過成分/非透過成分応用例項目定義,コメントなど流量供給流量욞は,適切な範囲が膜モジュールごとに決められている.膜間差圧入口圧욞,あるいは,供給側平均圧力(욞+욥)/2と透過側の圧力욣(通常は大気圧)との差.温度液温度により流体の粘度は変化し,温度上昇により透過性は一般に向上する.供給水質懸濁物やスケール生成物による膜性能劣化が起こりうるので,前処理が通常行われる.透過水質膜種類の選定,および,回収率の設定に必要となる.1―41力損失などが,集じん方式によりさまざまである.このため,処理対象ガスに応じて適切な集じん装置を選択しなければならないが,複数の装置を直列接続し,前置集じん装置(プレダスタ)では,ダスト濃度の低下,粗粒の除去,後続装置では微粒を分離し,全体として高集じん率とすることがある.集じん装置には,捕集性能が最高となる気流速度と,運転費を最も安価にできる気流速度が存在するので,両者を勘案して実用速度を選定する必要がある.さらに,装置の固定費,運転費のほか保守費も耐久性に関連して重要であり,捕集したダストの払い落としや排出,運搬,処理も集じん操作と同時に十分考慮しておかなければならない.表2・7に集じん装置の性能の概略をまとめて示す.
