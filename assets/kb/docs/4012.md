# 2・10・1 膜分離(membrane separation)の種類

> **章**: 第2章 化学機械
> **節**: 2・10 膜分離装置

膜分離プロセスは,省エネルギーであること,コンパクトな装置で運転が容易であることなどの特徴を有するために,産業界で徐々に実用化が進んでいる.膜分離プロセスは,分離対象の相,駆動力の種類に応じて分類されており,便覧や成書に膜分離法の詳細がまとめられている.圧力を分離駆動力とする液相系の膜分離プロセスを表2・8に示すが,細孔径に応じて,精密ろ過(細孔径>100nm),限外ろ過(ultra filtration,細孔径2~100nm),ナノろ過(細孔径1~2nm),逆浸透(reverse osmosis,細孔径<1nm)に分類されており,操作圧力も異なってくる.分離したい物質に応じて,適切な膜分離プロセスをまず選択する必要がある.
