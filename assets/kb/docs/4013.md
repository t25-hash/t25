# 2・10・2 膜分離装置の基本フローと性能評価

> **章**: 第2章 化学機械
> **節**: 2・10 膜分離装置

膜分離プロセスの概略は図2・38に示すフローで表され、膜モジュール(membrane module)への供給流は、透過流および非透過流(濃縮水とも呼ばれる)に分かれる。回収率=1では、供給液が全量膜透過することになるので、目標水質に基づき決定する。1―42産業機械・装置要な膜輸送式の詳細は省略するが、膜モジュール設計に当たって必要となる基本的な設計条件および評価項目を表2・9に示す。膜モジュールの膜間差圧を増大すると、膜透過流量が増加し、回収率は増加する。また、供給流量を増大すると、透過水量は膜間差圧で決定されるために、回収率は低下する。膜モジュールの本数を増やすことで、分離システム全体としての回収率を上げることができる。ただ、回収率を上げると、非透過成分が供給側で濃縮し、膜の阻止性によっては、シリカ、炭酸カルシウムなどのスケール形成物質が飽和濃度に達し析出する可能性もある。なお、膜モジュール性能の操作条件依存性(流量、圧力、温度)の詳細は膜メーカの技術資料を参照する必要がある。膜プロセス実用化の問題点は膜機能の劣化であり、膜表面あるいは細孔内部に懸濁物、高分子物質、無機物などが沈着あるいは吸着し、透過流束の低下あるいは膜分画性能を変化させるファウリングが問題となる。逆洗(膜透過側から溶液を逆流させることで沈着物を除去)などの物理洗浄、化学洗浄、操作条件の最適化などプロセス面からの対応がなされる。また、親疎水性の制御など分離膜自体の改良が行われ、種々のグレードの分離膜が上市されているが、膜プロセスの実用化に際しては、分離対象となるプロセス流体を用い図2・38 膜分離装置のフローの概略図るパイロットテストが依然として必要である。
