# 2・10・3 膜モジュール

> **章**: 第2章 化学機械
> **節**: 2・10 膜分離装置

実際の工業プロセスで分離膜を用いるためには,膜モジュールと呼ばれる耐圧容器に組込む必要がある.実用化されている膜モジュールの形態は,図2・39に示すような平面状膜をのり巻き状に巻上げたスパイラル型モジュール(spiral module),および,図2・40の管状の膜を充てんしたモジュールが一般的である.管状の膜の場合,膜の径によって,管状膜(内径1cm以上),中空糸膜(hol-low fiber membrane,おおむね外径0.5mm以下)を用いる中空糸型,および,中間のサイズのキャピラリー型モジュール(capil-lary module)がある.膜モジュールの使用条件(耐圧,温度,pH,供給流量範囲など)に十分注意する.いずれのモジュールも,供給流れの混合を促進する工夫がなされている.一般に,モジュール体積当たりの膜面積は,中空糸(16000~30000mm)>キャピラリー膜(6000~1200mm),スパイラル(300~1000mm)>管状膜(300mm以下),の順番に大きいが,一方,耐汚染性や洗浄性はこの順番に劣る.また,膜の素材もファウリングに影響を与えるため,供給流体の性状に応じて,適正な膜と膜モジュールを選択する必要がある.
