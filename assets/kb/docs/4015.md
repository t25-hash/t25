# 2・3・2 入出力インタフェースと通信インタフェース

> **章**: 第2章 基礎技術
> **節**: 2・3 制御用コンピュータの基礎

ロボティクス・メカトロニクス要素を制御するためには，多くの
インタフェースを利用する．ここでは，代表的な入出力インタフェ
ースと通信インタフェースについて示す．
a.入出力インタフェース
.ディジタル入出力（DIO:digital input outpnt ) DIO
はHigh/ Low（高/低）の信号で外部機器を制御する．この機能は
スイッチ的な用途で利用可能であり，LED（light emitting diode）
の点滅のように単純な機能もこれだけで実現される．さらには，ポ
ートのHigh/ Lowを周期的に実行することによる独自プロトコル
による通信にも利用可能であることから，広い用途で利用できる機
能であり，メカトロニクス機器の制御における基本的なインタフェ
ースの一つといえる．
.A-D変換器（AD converter ) 光センサや加速度セン
サ，超音波センサ，ポテンショメータなどから取得される電圧値を
量子化してコンピュータで取扱える情報にする処理である．使用す
るA- D変換器により，A- D変換器の分解能が異なる．センサなど
から取得できる電圧値を忠実にディジタル化するためにはA- D変
換の分解能が大きく影響するため，正確なデータを必要とする場合
は高分解能のA- D変換器が求められる．また，データの取得周期
（サンプリングタイム）もA-D変換器の選定において重要な要素
である．
これらのマイクロコントローラはプログラムによってその動作を
変更でき，その開発においては各メーカから開発環境が無償で提供
されているものもあれば有償のものもあるが，いずれにせよ第三者
によって作られている開発環境を利用している．また，このとき利
用する開発言語についてもアセンブラ言語，C言語，BASIC（ベ
ーシック）言語などが用意されているものが多い．また，開発した
ソフトウェアをマイクロコントローラで利用するために，ライタ
.D- A変換器（DA converter ) A- D変換とは逆の機能
であり，コンピュータからの出力をアナログの電圧値に変換する機
能である．モータドライバに速度などの指令を送るときに，PCな
どから出力するディジタル信号を電圧値に変換する場合などに利用
される．
.PWM（pulse width modulation) PWMは出力電圧
をHigh/ Lowの時間の比により平滑化することで，最終的な出力
電圧を制御する機能である．DCモータの制御やラジオコントロー
ラサーボモータの回転角の制御，LEDの輝度制御など多くで利用
される．制御対象により動作周波数は異なるため，通常はこの周波
数を1として，その間でHighとLowの比を決定する．制御対象
によりこの動作周波数は異なり，PWM出力機能を有していても実
現できない周波数の場合があるので，注意する必要がある．
b.通信インタフェース
.RS 232 C 古くからPCに装備されているシリアル通信
用のインタフェースの一つであり，モデムなどとの通信で利用され
てきている．ケーブルにはストレートケーブルとクロスケーブルの
2種類が存在し，コンピュータ間の接続にはクロスケーブルを，機
器との接続にはストレートケーブルを用いる例が多い．通信速度は
他の通信方式に比べて速くはないが，古くから利用されてきた技術
のため，シリアル通信の代表的な規格である．現在はこの規格のポ
ートを有するコンピュータが減少してきている．
.IC（inter integrated circuit ) フィリップス社によっ
て開発されたシリアル通信の規格である．低速な周辺機器のマザー
ボードへの接続や，組込みシステム，携帯電話などで利用されてい
る．マイクロコントローラにおいてこの通信規格に対応したものも
多く，さらに近年ではこの方式を採用したセンサも登場してきてい
る．一般的には100kbpsの高速モードと10kbpsの低速モードだ
が，最近では3.4Mbpsでの動作も可能となってきている．接続可
能なデバイス数は112個である．
ICのいちばんの利点は，マイクロコントローラとデバイスの接
続がわずか2本の線で可能なことである．
.CAN（controller area network) ボッシュ社が策定を
行ったネットワーク規格であり，当初は自動車におけるLAN
（local area network）として開発された．転送レートが最大1
Mbpsと高速であるだけでなく，エラー検出機能が複数存在するた
め，ICとは異なりマルチマスタによる通信が行える点が特徴であ
る．最近では，CANを搭載したマイクロコントローラも開発され
てきている．自動車で実装されていることによる信頼性などから，
注目を集めている組込み機器用のネットワークの一つである．
.USB（universal serial bus ) PCの周辺機器を接続す
るためのバス規格であり，現在はUSB2.0が主流である．電源を
供給することが可能であることだけでなく，接続元のデバイスが稼
働中でも接続できるホットプラグを可能としている．最大で127台
の機器の接続が可能である．USBポートに差すだけで簡単に利用
できることから，標準的なコンピュータのインタフェースとなって
いる．
.IEEE（Institute of Electrical and Electronics Engi-
neers ）1394 アップルコンピュータ社により開発されたFire-
Wireがもととなり規格化されたディジタル機器向けの通信規格で
ある．IEEE1394aとIEEE1394bの2種類が存在するが，
IEEE1394aは通信速度が400Mbpsであり，IEEE1394bは現在
では800Mbpsまでのものが見られる．USBと同様にホットプラ
グに対応しており，接続先の機器が稼働中でも接続可能である．接
続可能機器は63台である．USBと大きく異なる特徴として，機器
を介して別の機器を接続すること（デイジーチェーン）が可能であ
る．
.イーサネット（Ethernet ) IEEE802.3によって標準化
されているLAN規格の一つである．一般的にLANといった場
合，イーサネットを指すことが多い．接続形態には，回線を複数の
機器で共有するバス型と，ハブを介して各機器に接続するスター型
がある．現在，通信速度は10Mbpsから10Gbpsまでと高速化が
進んでいる．ネットワークに接続するための標準インタフェースで
ある．
.無線LAN（wireless LAN) IEEE802.11によって規
定されている無線通信を用いたLANの規格である．現在，通信速
度として100Mbps程度までが利用可能である．無線を用いた通信
であるため，スタンドアローンで動作するロボティクス・メカトロ
ニクス機器において広く利用されている．
