# 2・10・4 膜分離装置の実際:表層水の浄水プロセス

> **章**: 第2章 化学機械
> **節**: 2・10 膜分離装置

図2・41には,日本最大級(14万m/日)の浄水プラント(栃木県・今市)の概略を示す.原水は有機物も少なく水質も良好なためストレーナによる前処理のみで,中空糸型限外ろ過膜モジュールにポンプで加圧・供給される.なお,原水によっては,凝集剤を添加しファウリング成分を低減させる必要がある.膜処理部では120本のモジュールが6系列で並列配置され,各モジュールの回収率90%(残り10%は濃縮水と逆洗排水),透過流束1~1.7mmD,有効膜間差圧20kPa程度で設計・運転されている.膜ろ過水の濁度を連続的に高感度モニタすることで,膜破断による透過水の原水への混入対策を講じている.さらに,排出水の処理として,凝集剤(ポリ塩化アルミニウム)を添加し凝集沈殿させ,上澄み水図2・41 浄水プロセスの例(栃木県・今市市)図2・39スパイラルモジュールの構造図2・40キャピラリーモジュールの構造は放流し,濃縮汚泥はフィルタプレスで加圧脱水する廃水・汚泥処理設備を付設している.工業反応装置を構造形式で分類すると,固定層(fixed bed),移動層(moving bed),流動層(fluidized bed),気泡塔(bubblecolumn),かくはん槽(stirred tank)などに分類できるが,固体触媒を用いる反応装置としては,固定層と流動層などが利用されている.固定層触媒反応装置では格子,金網,多孔板などによって装置内に充てんされた固体触媒を支持する.反応流体は触媒粒子間の空げきを流れ,触媒反応が起こる.流動層触媒反応装置では,反応流体を装置の底部から吹込み,触媒粒子を浮遊懸濁させた状態で反応を行う.反応流体が気体の場合には,大部分の気体は気泡を形成して層内を上昇する.流動層の概念図を図2・42に示す.反応に関与する流体の形態で分類すると,気固触媒反応,液固触媒反応,気液固触媒反応に分けられる.この分類と上に述べた構造による分類を組合わせると反応装置には多くのバリエーションがある.反応装置は触媒を入れるための単なる容器ではない.触媒と反応流体をいかに効率良く接触させるか,さらに,発熱または吸熱反応の場合にはどのようにして熱の除去や供給を行うかなどさまざまな機能を付与させた合理的な設計が必要となる.
