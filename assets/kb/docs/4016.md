# 2・11・1 固定層触媒反応装置(fixed-bed reactor)

> **章**: 第2章 化学機械
> **節**: 2・11 固体触媒反応装置(catalytic reactor)

化学工業においては、反応物として気体を扱うことが多いが液体を反応させる場合やその双方が原料となる場合もある。充てんされる触媒は通常、数ミリから数センチの大きさであり、形状は球状や円筒状などをしている。また、触媒活性のある固体表面積を広くとるために、多孔質体とすることが多い。固定層の特徴として、軸方向の流体混合が少なくなることにより、反応収率が高くとれることや、反応流体と触媒との接触時間を広範囲に変化させることが容易で、広い範囲の反応速度に対して適用できることが挙げられる。その反面、伝熱速度が大きくとれず、反応熱が大きな反応では、温度分布をなくすために除熱や加熱にさまざまな工夫が必要である。反応物および生成物が液体である場合や、液に溶解している場合には、液固触媒反応となる。酵素をイオン交換樹脂や多孔質固体などに固定化して、酵素反応を行う場合などがこの形式に相当する。反応物として液体と気体を供給する気液固触媒反応としては重油の水素化脱硫反応(油中の硫黄分を除去する)が代表例である。図において脱硫反応を行う。
