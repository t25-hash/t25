# 2・11・2 流動層触媒反応装置(fluidized-bed reactor)

> **章**: 第2章 化学機械
> **節**: 2・43 に装置概略を示す。装置上部から油と水素を流し、触媒層

工業的に多くの重要なプロセスに反応物を気体とした気固流動層が利用されている。層内の温度をほぼ均一に制御できることや、触媒粒子の連続的な供給や抜き出しが可能であることが長所となる。温度制御が比較的容易であるため、反応熱が非常に大きな反応に対して利用されることもある。図2・42に示すように気泡として層内を抜けていくガスが多いため、反応収率は固定層に比べると小図2・42流動層型反応装置図2・43 水素化脱硫固定層反応装置1―43さくなる点が短所となる。また、良好に流動化させるための層高やガス速度に範囲があるために、操作範囲が制限される。気固流動層に使用される触媒は大きさが60 m程度と非常に小さい。気泡とエマルション相(触媒粒子相)との間の物質の移動速度が見掛けの反応速度に影響するため、小さな気泡をよく分散させた流動化状態を達成することが重要なポイントである。そのためにガスを均一に分散させるような分散器の設計や、良好な流動化状態を得るために適切な物理物性をもった触媒粒子の選択も必要である。また、粒子の飛び出しも起こるため、反応装置内部もしくは外部にサイクロンなどを設置して粒子を捕集しなければならない。また、層内の気泡の分裂を促進するために、装置内部に各種の内挿物(インターナル)を設置することもある。触媒を連続的に供給・排出することが可能であるということも大きな特徴であり、石油の接触分解プロセス(ガソリン留分の製造)に生かされている。この反応では、触媒活性は非常に高いが、劣化速度も速い。固定層反応器では反応操作を止めて再生操作を行う必要があったが、流動層反応器を採用し、図2・44に示すように反応塔と再生塔の間で触媒粒子を循環させることによって、連続的な反応操作が可能となった。最近の装置においては、反応部はライザと呼ばれる形式で、空塔ガス速度は10m/s以上で操作されることもあり、触媒粒子が気流によって運ばれる状態である。この部分での反応は数秒で終了する。このように、劣化が大きいといった触媒の短所を装置によって補うことができ、数万t/dといった大規模な装置が精油所では稼働している。
