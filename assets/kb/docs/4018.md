# 2・11・3 気泡塔型反応装置とかくはん槽型反応装置

> **章**: 第2章 化学機械
> **節**: 2・43 に装置概略を示す。装置上部から油と水素を流し、触媒層

液体中に気体を吹込み、気液の反応を行わせる装置は気泡塔型反応装置と呼ばれるが、これに固体触媒を添加した形式もある。この場合には触媒粒子を液体中に懸濁させて使用するため、懸濁気泡塔と呼ばれる。また、液で固体触媒粒子を流動化したところに気体を吹込む三相流動層という形式もある。油脂の水素添加反応では粉末ニッケル触媒を油脂に分散させて水素を吹込むことによって、硬化油(マーガリンなどの原料)を製造する。未反応の水素を循環させて使用するような懸濁気泡塔の形式で操作されていたこともあるが、反応する量だけ水素を供給し機械的なかくはんで液相に吸収させる方式が現在は使用されてい図2・44 石油接触分解装置の模式図(a) 水素ガス循環形式水素ブロワー(b) かくはん槽形式図2・45 油脂水添反応装置1―44産業機械・装置る。油脂水添反応の装置を図2・45に示す。
