b.メカトロニクス応用例 ネットワーク技術のメカトロニク

る．それらの応用に分散コンピューティング技術が利用されてい
る．

載するホームロボットやネットワーク家電などの対象機器と，ユー
ザ端末やアプリケーションプログラムなどのコントローラとの間
で，XML記述された機器や搭載機能の仕様や制御メッセージを，
ウェブサービスやUPnPなどの分散コンピューティング用のミド
ルウェアを用いて送受信することで制御を行う．このようなミドル
ウェアを利用することにより機器側のセンサやアクチュエータをオ
ブジェクト化し，その操作方法および仕様を定義する程度で，コン
トローラがネットワーク家電やホームロボットの搭載機能の仕様を
理解したうえで必要な制御を行うことが可能になる．
上記ミドルウェアのプロトコルは，HTTPやSOAP（simple
object access protocol ）などの標準プロトコルによって構成され
る．SOAPメッセージ自身もHTTPプロトコル上で動作し，最終
的にはTCP/IPのパケットとしてネットワーク上を転送される．
