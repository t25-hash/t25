# 2・12・2 液相反応装置(liquid phase reactor)

> **章**: 第2章 化学機械

液相の反応装置は形状に基づいて、槽型と管型に大別できる。反応に伴って固体が析出する場合は、2・12・1節で述べた晶析装置が該当する。操作法から回分式・連続式・半回分式に分類できる。槽型の反応装置には、一般にかくはん機が取付けられており、特に十分に混合されて、反応流体の濃度および温度が均一とみなすことができる反応装置を完全混合槽型反応装置(complete mixing tankreactor)という。槽型反応装置(tank reactor)を回分操作する場合を単に回分反応装置(BR : batch reactor)といい、連続操作する場合を連続槽型反応装置(CSTR : continuous stirred tankreactor)と呼ぶ。また、複数のCSTR を直列に連結した直列連続槽型反応装置もよく用いられる。これに対して管型反応装置(tube reactor)では、反応流体が反応装置に均一な濃度で供給され、軸方向には混合されずにあたかもピストンで押出されるように流れつつ、反応が進行する。したがって、軸方向に濃度分布が生じる。このような反応流体の流れを押出し流れ(piston flowあるいは栓流:plug flow)と呼び、管型反応装置を押出し流れ反応装置あるいはピストン流れ反応装置(PFR :piston flow reactor)と呼んでいる。a. かくはん槽型反応装置(stirred tank reactor)かくはん槽型反応装置は、均一液相反応をはじめとして、液-液反応、気-液反応、気-液-固触媒反応などの不均一反応にも用いられており、適用範囲が広い。液の粘度が低い場合の均一液相反応のかくはんには特に問題はないが、重合反応のように粘度が高くなると、かくはん翼の形状などに注意しないとかくはんの効果が上がらない。反応熱の除去(または供給)には、図2・51に示すように、槽の外側にジャケットを付けるか、あるいは内側にコイルを入れて、それらの中にスチームあるいは熱媒体または冷媒を流す。b. 流動層型反応装置(fluidized-bed reactor)流動層は固体粒子を槽の底部より送入した流体によって浮遊懸濁させながら流動化させる装置である。流動化された固体粒子は、あたかも沸騰状態にある液体のように挙動するので、固体の輸送、取出しが容易となり、伝熱能力も高い。この種の反応装置は石油の接触分解反応、炭化水素の酸化反応、硫化鉱の燃焼、石油のガス化等に活用されている。c. 移動層型反応装置(moving-bed reactor)移動層型反応装置も気体または液体に固体(触媒)を接触させる反応装置で、固体(触媒)粒子を装置上部に連続的に供給し、重力で垂直方向に緩やかに下降させ、それに対して反応流体を向流、並流あるいは十字流的に接触させて、触媒反応あるいは気(液)固反応を行う装置形式である。図2・50 精製晶析装置図2・51 かくはん槽型反応装置1―46産業機械・装置気相薄膜・微粒子合成装置とは、原料物質をガス相中に分子・イオン・ラジカル状ないしクラスタ状として分散し、最終的に固相として析出させる装置をいう。薄膜合成装置では固体(基板)上での析出が、一方微粒子合成装置ではガス相中での析出が行われるが、装置に関する共通点は多い。原料物質と同じ物質が析出する場合は物理的蒸着・析出(PVD : physical vapor deposition)、原料が化学反応による転換を経由して析出する場合は化学気相成長・析出(CVD : chemical vapor deposition)と、それぞれ呼ばれる。合成される物質は多種多様であり原料の種類もさまざまである。また、原料の分散や反応を行わせるためのエネルギー源の種類も多く、結果として装置の種類も多い。したがってここでは合成装置に用いられている手法を中心に述べる。
