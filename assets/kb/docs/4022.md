# 2・13・1 気相薄膜合成装置

> **章**: 第2章 化学機械
> **節**: 2・13 気相薄膜・微粒子合成装置

一般には、平坦で均一な膜厚の薄膜を高速に堆積させることが求められる。近年は、分子・原子レベルの精度での膜の微細構造の制御、異種膜の多層化なども重要である。基板によっては、大面積領域(メートル角サイズ)への成膜や、低温での成膜も必要とされる。a. PVD 法。 真空蒸着法10~10Paの減圧下で蒸気化させた原料を、基板上に堆積させる。電気ヒータ、高周波誘導、電子ビーム、レーザ光などが蒸発エネルギー源として用いられる。原理も装置も比較的簡単で、利用分野は広い。蒸発源と基板の間でプラズマにより蒸気をイオン化する方法(イオン化プレーティング法)では、イオンが基板へと加速され、膜の付着強度を大きくできる。また、超高真空(10~10Pa)下では、蒸気以外のガス分子の影響が減るため、高純度でかつ原子層レベルまで制御性の高い膜が得られる。. スパッタリング(sputtering)法加速されたアルゴンなどのイオンを原料表面に衝突させることで、原料原子(分子)を気相にはじき飛ばす方法である。蒸気圧の低い材料にも適し、またスパッタされた原子、分子の基板への衝突エネルギーが大きいため、ち密な膜を作ることができる。加速イオンは、10~1Paのガス雰囲気中で気体放電によって生成させる。高周波(RF: radiofrequercy)放電の場合(RFスパッタ)では、絶縁物の成膜もできる。マグネトロン放電(電場と磁場が直交する放電)では、低温で高速のプロセスが実現でき、プラスチック基板にも成膜できる。スパッタリング雰囲気中に混ぜた微量ガス(窒素、酸素、水素など)を放電場中で活性化し、これらを膜中に取込んで化合物薄膜を作る方法は、反応性スパッタリングと呼ばれている。b. CVD 法薄膜の構成元素自体の蒸気圧が低い場合には、構成元素を含みかつ蒸気圧の比較的高い化合物分子を原料物質とすると、化学反応を経由して目的の組成として析出させられる。CVD 法は原料、温度などによりいろいろな分類が可能であるが、ここでは反応を開始するためのエネルギー源の違いにより分類して述べる。. 熱CVD 法400°C~1500°Cの高温空間および基板表面でガスの熱分解、還元、酸化、置換反応などを生じさせる。原料ガスの適切な選択と混合により多種類の材料薄膜が得られる。有機金属を気化させて原料とする方法はMOCVD(metal organicCVD)と呼ばれ、化合物半導体などの製造に用いられている。原料蒸気は通常、キャリヤガスに同伴させて反応容器に送り込むが、反応性の高い原料の場合には液体状で混合した後容器の直前で気化させる(液体原料CVD)。反応容器内には1枚ないし多数枚の基板が置かれ、原料ガスが基板にできるだけ均等に送り込まれる構造となっている。容器壁面を抵抗加熱する方式をhot wall、基板のみを高周波誘導加熱や赤外線で直接加熱する方式をcold wallと呼ぶ。また雰囲気圧力により、常圧CVD と減圧CVD に分類されるが、後者では装置内の温度やガス濃度の制御性が良く、一般に膜厚や膜品質の均一性が向上する。さらに近年は、異なる原料ガスを交互に供給することで単原子(分子)層を一層ずつ成長させる単層析出(ALE: atomic layer deposition、ディジタルCVD などとも呼ばれる)により、原子・分子レベルの成膜制御も行われている。. プラズマCVD(plasma enhanced CVD)法プラズマの高いエネルギーで原料ガスを励起ないし分解すると、基板温度を低く保ったまま(通常400°C以下)での成膜が可能となる。プラズマの作成法には、直流、高周波(RF)、電子サイクロトロン共鳴(ECR)などがある。RFを用いた代表的な装置の模式図を図2・52に示す。太陽電池やディスプレイ基板など大面積の膜を製造する場合には10Pa前後で操作される容量結合型RF非平衡プラズマCVD 装置が多用され、基板をプラズマ領域に連続的に通過させて処理する場合もある。また、プラズマによる薄膜へのダメージを防ぐために、基板をプラズマ領域外に置くこともある。プラズマCVD では、ガス相反応の進行が速いため、薄膜の製造速度を上げようとすると、ガス相中での核生成により微粒子(ダスト)が発生し、ダストによる薄膜の汚染が生じやすくなるという課題がある。. 光CVD 法光照射によって、原料ガスの分子を励起して光化学反応を起こしたり、基板を局所的に励起・加熱したりする方法であり、光励起CVD 法とも呼ばれる。励起に用いる光(紫外光など)の波長に応じて化学反応を選択的に進めることが可能である。また、基板照射の場合は、照射領域により成膜領域を制御できる。
