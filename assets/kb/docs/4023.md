# 2・13・2 気相微粒子合成装置

> **章**: 第2章 化学機械
> **節**: 2・13 気相薄膜・微粒子合成装置

機能性材料の観点からサイズ分布,組成,形態,結晶性などが微細に制御された粒子が要求されている.気相の粒子合成では一般に,粒径が数nmから数百nmの粒子が得られ,固相法,液相法に比べて高純度の粒子を製造しやすい.しかしながら,凝集粒子の生成,装置壁への粒子付着,粒子の回収に関して解決すべき課題が多い.また,装置の操作条件と製造される粒子の性状との関係も複雑である.a. PVD 法. ガス中蒸発法10~10Paの不活性ガス雰囲気で金属を蒸発させ,凝縮により粒子を形成する.蒸発の方法としては,蒸発金属塊を電極としたアーク放電,金属塊へのレーザビーム照射(レーザアブレージョン法),プラズマビーム照射,るつぼ中での高周波誘導加熱などがある.超高温の熱プラズマ(~10K)中に原料を導入するプラズマジェット法では,高融点の金属粒子が製造で(a) 容量結合型装置(b) 誘導結合型装置図2・52 高周波プラズマ薄膜合成装置の例図2・53 電気炉加熱を用いた微粒子合成装置の模式図きる.. 電気炉加熱法塊状,粉末状,メッシュ形状にした金属や金属塩原料を不活性ガスを流した管内に置き,管壁を電気ヒータで加熱して,蒸発・凝縮により微粒子を発生させる(図2・53).大量合成には向かないが,温度や合成時間の操作条件の調整により,ナノメートルオーダでサイズのよく揃った粒子を得ることができる.b. CVD 法. 化学炎法有機物を直接燃焼させてすす粒子を合成する方法と,水素やメタンなどの燃焼炎中で金属塩や有機物の蒸気を熱分解反応させて金属酸化物粒子やカーボンブラックを得る方法がある.凝集粒子が生成することが多いが,大量合成法として古くから用いられている.後者では,炎の作成法や蒸気の混合方式の工夫によって粒子性状の制御が行われている.. プラズマ法上述のプラズマジェット法において,ガス状,液滴状,粉体状の原料を反応させると,高融点の化合物粒子や,合金粒子が多量に合成できる.プラズマの空間的・時間的不均一性により製造粒子の性状が不均一になりやすいことが課題である.. レーザ法炭酸ガスレーザなどからの高出力レーザビームで原料ガスを加熱し,熱分解反応などを促進して粒子を合成する方法である.効率的な加熱のためには,ビームの波長と用いるガスの吸収波長を検討する必要がある.化学炎内で粒子を合成する途中で,粒子をレーザ照射加熱して粒子の結晶相を制御する試みもある.. 電気炉加熱法CVD による電気炉加熱法では,図2・53のように管の上流側から原料ガスを流し,炉内での反応で粒子が合成される.上述の3法に比べて,反応温度や反応時間を調節しやすく,図2・53のように電気炉を多段とすればガスの温度分布も変化させられる.したがって粒子性状の制御の自由度が高い.反応ガスとともに粒子(シード粒子)を導入すると,条件によってはシード粒子の周りに固相が析出し,コーティング粒子を製造できる.
