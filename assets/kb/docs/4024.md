# 2・14・1 プロセス設計(process design)

> **章**: 第2章 化学機械
> **節**: 2・14 プラント、プロセス設計・制御

プロセスシステムは、高品質の製品を安全に効率良く製造するという目的のために、反応器、蒸留塔、熱交換器など、種々の機能をもった装置が結合された集合体である。設計時には、特定の条件下での最適性だけではなく、市場の変化や性能の劣化などの不確定性も考慮して、その柔軟性・拡張性を検討しながら、プロセスシステム全体のありかたを判断し、各機器の設計を行うことになる。実験室において、ビーカやフラスコといった装置で新たに発見、開発された物質や反応経路が、実際に工業化されるのは、1~3%といわれている。そのため、プロセス開発に関わる技術者が、工1―47業化に値するか否かをできるだけ早い段階で見極める必要がある。プロセスの合成問題には、使用する装置やその結合関係について、非常に多くの自由度が存在するため、原料や大まかなフローシートの候補を人間が想定し、選択することになるが、市販のCAD システム(図2・54に一例を示す)が充実されてきているため、早い段階からさまざまな検討ができ、コンカレントなプロセス開発が可能になってきている。現在のCAD は、主要な構成要素の設計・操作条件、エネルギー収支、大まかな建設コスト、運転コストの算出はもちろん、動的シミュレーションも可能で、プロセス設計だけでなく、プロセス制御系の設計・改良などにまで、同一のCAD が利用できるようになっている。CAD システムには、充実した物性データベースが組込まれており、熱の有効利用を図るヒートインテグレーションの設計支援などの機能も充実している。また、配管設計などの三次元CADも連動できる。設計、運転、保全、生産計画など、すべての業務で発生したデータを共有して、とことん使いこなすことをめざしたデータウェアハウスの構築を可能にするため、データ構造の国際標準STEP(the Standard for the Exchange of Product Model Data)[ISO 10303]が検討されている。原油精製やエチレンプラントなどの大規模プラントから、薬品のように少量ではあるが高付加価値の製品を作るプラントまで、プロセス設計の対象はさまざまである。大量生産には、連続プラント、多品種少量生産には、バッチプラントが一般的に採用されるが、マイクロプラントという微少流路を用いた連続プラントも検討されている。マイクロリアクタ(micro reactor)では、少量の試薬を用いて高精度に制御した反応場、界面で実験を行うことができるので、分析が容易であるし、従来危険で不可能だった反応も、量を微少量に限ることで安全性を確保できるなど、反応系の開発に有用な特性を有している。開発後スケールアップなしに商業化が可能になるように、装置群を多数、並列化する構造も検討されている。
