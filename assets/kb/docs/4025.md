# 2・14・2 プロセス制御(process control)

> **章**: 第2章 化学機械
> **節**: 2・14 プラント,プロセス設計・制御

市況による生産要求や原料の変化など,製造プロセスを取り巻く環境はつねに変化しており,このような外乱や不確定要素の存在下で,安全で効率的に,高品質な製品を製造することを可能にする技術がプロセス制御である.化学プラントは,多くの装置の結合で構成されており,多数(エチレンプラントでは4000以上)の変数が管理対象になっている.そのため,多数の弁やポンプ,ヒータなどの操作量が用意され,望ましい状態を維持するための制御系が構成される.例えば,製品流量を2倍に変更しようとすると,ほとんどすべての操作端を変更しなければならない.これらの膨大な数の操作量の適正値を設計するのは,非常に困難な問題である.例えば,水道の蛇口を流量が2倍になるように開くとすると,どれだけ蛇口をひねればよいかは簡単には算出できない.水道の元圧変動も影響する.しかし,フィードバックを基本にすると,一つの測定端の情報を見て,一つの操作端をさわるというシングルループの組合せで,うまく連動する世界を実現できる.蛇口の場合も,流量計がついていれば,その流量測定値が目標値と一致するように蛇口を調整することは可能である.このフィードバック制御の基本は積分制御である.積分制御は,操作量の現在値からの補正幅を誤差により決定し,誤差がある限り補正動作を続ける制御で,この補正動作を数式で表現すると,式(2・13)になる.=(2・13)この式から,誤差が零でなければ一定にはならず,落ち着いたときには誤差がないはずであるということが理解できる.式(2・13)は,式(2・14)と等価であるので,積分制御(Integral の流量の制御系に生産量を2倍と設定するだけで,温度制御,圧力制御,組成制御などが連動し,原料A や製品Cなどすべての流量がその2倍の製造量を実現するように定まる.原料B の流量変化だけでなく,冷却水温度など,さまざまな変動に対しても有効に働く.基本的には,シングルループコントローラを組合わせ,うまく連動するように構成するのがプロセス制御の基本であるが,もっと積極的にうまく動くという状態を実現するために,より効率的な運転条件をリアルタイムに求め,多変数制御を行うというモデル予測制御(model predictive control)も数多く導入されている.プロセス制御は,ロボット制御等よりも,はるかに長い2秒や1分というサンプル間隔で実現でき,計算機のハードウェアへの制約もあまりないので,1970年代からモデル予測制御が導入されている.エチレンプラントでは,被制御量600以上,操作量200以上という大規模なプラントモデルを基にした最適化制御が導入されており,生産量を最大にする条件を与えると,夏はコンプレッサ能力,冬は蒸留塔の能力が制約となる最適運転条件で,安定な操業が実現されるようになっている.プロセス制御では,ポリマ反応など,詳細なモデリングが困難な現象が存在し,むだ時間+一次遅れや,有限長のパルス応答モデルという簡便な構造のモデルで近似して,モデル誤差の存在を前提とした制御系設計を行うというアプローチが一般的に用いられる.PID 制御は,制御性能と調整パラメータの関係があまり敏感ではないし,把握しやすいので,簡便なモデルにより初期設定を行い,現場での再調整を期待することができる.また,モデル予測制御は,ロバスト性が高いという特徴以外にもその速応性の調整パラメータが用意されており,プロセスモデルがパルス応答モデルという把握しやすい形式で与えられるので,やはり,現場での調整が期待できる.そのため,PID 制御とモデル予測制御が用いられることが多い.多品種少量生産に適するバッチプロセスでは,シーケンスコントロール(sequential control)が用いられる.多品種生産では,その生産スケジュールが生産効率に強く影響し,製品の切替えを容易に安全に確実に行えることがコントローラに望まれる.シーケンスコントロールは,製品や装置に依存するが,製造レシピや装置コントローラの表現について,ANSI/ISA
