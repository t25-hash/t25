# 3・1・4 タンパク質源食品群の加工

> **章**: 第3章 食品機械
> **節**: 3・1 食品加工機械

a. 畜肉加工と(屠)畜は,対向した2台のベルトコンベアで畜体を挟むことで移送し位置決めし,頭部に0.3~3kV の電撃で失神させる.脱血後,脱毛機,皮はぎ機,背割機,内臓処理機で処理する.円盤カッタ,帯鋸,油圧駆動鋏(角骨せん断)なども使う.筋肉の動作物質であるグリコーゲンおよびATP(adenosine triphosphate)の閉サイクルは死後開き,グリコーゲンから生成される乳酸が蓄積し,生前7であったpH は5.4に下がり,肉の保水性も下がり,筋肉収縮信号であるCaが筋肉内に漏出して筋肉を硬直させる.硬直に至る時間は牛24,豚12,鶏3時間程度である.硬直中は加工不適なだけでなく食べて旨くない.数日経過すると保水性と柔軟性を回復し,これを解硬という.この時期,自己消化酵素によってATPが旨み成分であるイノシンに変化する.硬直後5°C程度に保存して旨みをもたらす操作を熟成という.獣鳥肉に限らず魚肉も同様であるが変化の速度は速い.硬直前に凍結すると,解凍時に硬直収縮し氷結晶が細胞を破壊するので肉汁を損失する.欧州での畜肉加工の歴史は古く,古来,ソーセージ(sausage)は生肉,塩蔵肉,血液,皮または内臓などを細切混合しケーシングと呼ぶ消化管または泌尿器管に詰めたもの,ハム(ham)は骨付きの豚腿肉を塩漬けしたもの,ベーコン(bacon)は豚バラ肉を指す.これに比べ,日本農林規格では原料獣も加工方法も多様である.食塩,砂糖,酸化防止剤,結着剤および発色剤などの混合液をピックル(pickle)またはブライン(brine)と称し,広く畜肉加工品に使用され,くん製されることもある.ハンバークなども含め,代表的な工程は,(1)骨,筋,腱,皮を除去し裁断する原料処理操作,(2)ピックルを注射し,冷暗室等熟成機で2~3日熟成する漬け込み操作,(3)香辛料と調味料を添加する配合操作,(4)ケーシングに詰め結さつする充てん操作,(5)くん製,湯煮,蒸煮などの加熱操作,(6)冷却操作,(7)包装の順である.伝統的手作業の一部が専用機械化されている.外皮や内臓膜の除去に使う皮はぎ機は,作業面を上向に固定した木工鉋様の工具上に肉をベルトで押付けて削り取る方式,および回転冷却ドラムに皮を氷結させドラム面に平行な刃で切り削く方式などがある.ピックル注入機は直径約2mm,長さ約150mmの注射針を,行および裂ピッチ双方とも約15mm一定で平板に鉛直に多数植え付け,肉塊に押圧してピックルを注射する.熟成機は直径約600mmの鋳鋼ドラムに肉塊を封入し回転させて揉む.またはロール軟化機(ten-derizer)で揉む.肉を大きく切断するのには円盤カッタまたは帯のこぎり,数mmの立方体には小口切断機またはフードカッタなどが使われる.副原料と混ねつ(捏)する操作には,サイレントカッタ,ミートチョッパ,挽肉機または擂潰機が使われる.サイレントカッタは刃渡り約150mmのナイフ4~6枚を翼車状に水平回転軸に取付け,翼車面内で水平にオフセットした垂直軸でうす皿を回転させ,うす皿上の肉を高速切断混合する.大気中で操作すると気泡を分散させたクリーム状になり,真空中ではち密に混ねつされる.擂潰機は回転うすとオフセットした回転軸に取付けた複数の杵で肉をすりつぶす.すりつぶすという操作は諸外国にはない.一定の条件ですりつぶした肉のペーストは加熱するとかまぼこ状の弾性体になる.ペースト状肉をケーシングに加圧注入する肉詰機(stuffer),その他,凍結肉挽捏機,整列機,くん煙発煙機,蒸し函など機械化されているが,ソーセージ用の腸管のスタッファへの装着,工程間の移動,被加工肉の位置決めなど手作業も多い.内部発熱する直接通電またはマイクロ波も利用されるようになった.b. 鶏卵加工鶏卵消費量は国民1人1日当たり1個弱である.鶏卵を洗浄後,割卵機で可食部を取り出した液卵は水分約75%で保存性がない.ネズミ糞尿等で汚染されていることもあるので,加工卵はサルモネラ菌を対象に61.1°C,3分殺菌する.一次加工品としては,割卵機で卵黄と卵白を分離する場合と全卵とがある.冷凍全卵,噴霧乾燥卵,濃縮卵に加工する.濃縮卵はショ糖を加え,殺菌を兼ね60~65°Cの減圧濃縮装置で水分25,ショ糖50,卵固形分25%に仕上げる.マヨネーズは食用油に約10%の卵白または卵黄と約10%の酢を加え,食塩,香辛料を添加して乳化する.
