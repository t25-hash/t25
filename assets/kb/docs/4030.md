# 3・1・5 無機質源食品群の加工

> **章**: 第3章 食品機械
> **節**: 3・1 食品加工機械

a. 製塩食塩は海水をろ過し,イオン交換膜法で塩分18~20%に濃縮後,四重効用蒸発缶で濃縮・晶析し,洗浄遠心分離して乾燥した粒度300~500 mの結晶粒である.陽および陰イオン交換膜を交互に並べ仕切って作られた多数の直列セルの奇数番に原海水を供給し電流を流すと,偶数番のセルに食塩が濃縮される.重金属および有害分子などは透過しないので安全な塩ができる.b. 清涼飲料法的飲料適の水を,塩素殺菌し水酸化カルシウムでpH 調整し水酸化第二鉄で凝集沈殿させ,砂,活性炭,紙でろ過精製する.酒および乳成分が規定値以下で,香り,甘味,酸味,着色料などを加えた飲料を清涼飲料という.炭酸飲料は充てん工程で発泡するので溶存空気を脱気してから,清涼飲料に二酸化炭素ガスを溶解させる.ガス圧力は,果汁飲料では20kPa以上,コーラは70kPa以上,炭酸水は300kPa以上である.脱気装置は絶対圧力10kPa以下に減圧したタンク内で液を薄膜流下または噴霧する.炭酸ガス圧入装置(carbonator)は低圧ガス飲料では15°C,高圧では5°Cの液を200~500kPaの二酸化炭素ガス中に噴霧する.c. 果実飲料原料の種類および製品規格とも多種ある.製品の約7割を占める温州ミカン果汁を例に工程および機械を以下に示す.収穫期間は約1箇月である.果皮が弱いので荷重で損傷しないように一時貯蔵はコンテナなどに小分けする.病害,未成熟,古果,障害などの果実を排除し,洗剤を散布し回転ブラシで洗浄する.製品によっては塩素殺菌も義務づけられる.柑橘類の搾汁方式は多彩である.外果皮の香味油,パルプ,果汁などが適度に含まれるように,しかも酸化しないように搾汁する.商品名インラインエキストラクタは,内径50~100mmの半球状カップ内に果実を置き,対向する上部カップを押下げると同時に,下部カップ中央部の孔から上方に突き上げる管状の刃で果実下部を穿孔して,果実内容物を管内に取り込む.管下部のろ過器で種および大型パルプを捕集し,後半の動作で排泄する.処理能力は毎分1―53800個にも達する.商品名アンダーソンエキストラクタおよび商品名シトラスエキストラクタは,円盤カッタで果実を2分割して,それぞれ切断面を外側にして回転するようにリテーナで囲まれた円盤に乗せ,内容物をかきだし,円すい状対向円盤に挟み込んで搾汁する.商品名ブラウンエキストラクタおよび商品名インデリカートエキストラクタならびに商品名ベルツーチシトロプレスは2分割した果実にリーマを旋削して搾汁する.果汁は適度にパルプをろ過あるいは遠心分離し,pH<4.0では65°C・10分,pH>4.0では85°C・30分殺菌する.四重効用缶または円すい回転型濃縮機で低温蒸発濃縮する.蒸発濃縮過程で芳香成分は捕集し精留塔で精製する.冷却し純水の氷結晶を分離する凍結濃縮装置も実用化された.濃縮果汁は-25°Cで凍蔵され,需要期に解凍希釈し,ビタミンCを酸化する溶存酸素を脱気した後,容器に熱間充てんする.加熱殺菌冷却後,炭化してから無菌充てんすることもある.
