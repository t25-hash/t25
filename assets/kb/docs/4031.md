# 3・1・6 醸造・醗酵・製パン

> **章**: 第3章 食品機械
> **節**: 3・1 食品加工機械

a. 醗酵用細菌・菌微生物個体の表面積/容積の比が大きいことから生体膜で起こる生化学反応の速度が大きく,反応も選択的なので太古より食品変換に利用された.生成物が次段階の変化過程を順次誘導するので回分操作が多い.遺伝情報をもつDNA を囲む明確な核膜をもたない微生物を細菌(bacteria)といい,乳酸,アルコール,酢酸を糖から生産する乳酸菌など有用な細菌がある.カビや酵母は細菌よりも大きく明確な核があり菌類(fungi)に属す.肉眼で菌糸コロニーを認める真菌類を俗に糸状菌またはカビといい,有用種は,接合菌145属765種,子嚢菌2700属28650種,不完全菌1680属17000種に含まれ,チーズを始め各地で酒や調味料など醗酵に利用される.蒸し米,炒り麦,炒り大豆などに培養された胞子を麹(コウジ)という.カツオブシに接種するアオカビは風味付与だけでなく物理限界を超える脱水効果を認める職人もいる.酵母は短径3~4 m,長径8~10 mの単細胞真菌類で約60属600種含まれおもに醸造に利用される.製パンでは小麦粉の2~3%に相当する酵母が使われ,日本では年間約4万tの圧搾酵母が廃糖蜜から製造される.始め試験管培地に接種し,フラスコ,タンク,流加装置の順に10t規模までスケールアップされる.流加装置では液体培地の酸素濃度,呼吸商およびアルコール生成量がコンピュータ制御される.遠心分離し水分65%程度に圧搾ろ過する.廃棄物等未利用糖質を原料にした飼料食用酵母も生産される.b. 酒類醸造酒(清酒,ビール,果実酒)は単式では糖を,複式ではデンプンを糖化し,酵母でアルコール発酵させろ過した酒で,アルコール度およびエキス分は中程度.蒸留酒(焼酎,ウイスキー,ブランデー,スピリッツ類)は醸造酒を蒸留し植物樽で熟成した酒でアルコール度は高くエキス分は少ない.混成酒(合成清酒,リキュール,みりん)は醸造酒や蒸留酒に花,葉,根または果実を浸漬し色・香味をつけた酒でアルコール度はやや高くエキス分も多い.ビールは主原料大麦を洗浄後12°Cで約40~50時間水に浸漬し,発芽缶に移して最初12°Cから毎日1°Cの割合で20°Cまで昇温し,発芽させ酵素増強させ麦芽とする.麦芽は低温乾燥後緩慢昇温して80~85°Cで3~5時間水分3%まで煤焦発色させる.これを粉砕して仕込み槽に移し水を加え間歇的に一部を煮沸しては槽に戻し,温度と濃度を上げていくことで糖化する.完了後はろ過し,ホップ(苦味を含む未受精雌花)を添加して煮沸した後ホップ粕を除去し醗酵槽に送る.新しい麦汁に醗酵槽に沈殿した泥状酵母0.5%を加え5~10°Cで8~10日醗酵させ若ビールになる.若ビールは酵母を分離し密閉タンクで0~2°C,30~90日追醗酵させ二酸化炭素が溶解した精澄ビールになる.回分操作が多いが,大規模工場では槽を並列設置し切り替えていくことで見かけ上連続操業される.煮沸には凝縮温度127°C,蒸発温度100°Cの蒸気再圧縮濃縮缶が採用されることもある.清酒は,タンパク質や脂質をできるだけ除去した精米歩合75%1―54産業機械・装置の米を洗米して25~30%吸水させ,20分蒸し,冷却後麹を接種し2日醗酵する.この麹と蒸し米,水,酵素液を混合したものを(モト,酒母)といい,7~8°Cで間歇撹拌糖化しながら20~25日かけて硝酸還元菌と乳酸菌も生育させる.このに蒸し米,水,麹を3回に分けて添加し増量しながら糖化とアルコール発酵を平行させる.20日でアルコール濃度は20%に達し(モロミ,諸味)となる.を圧搾し60°Cで火入れ酵素失活させ清酒となる.c. 醗酵調味料みそ・醤油では,食塩濃度7%前後で生育する通性嫌気性(酸素がなくても増殖する)好塩細菌が利用され,タンパク質を分解して旨み成分を生成する.醤油は,大豆を浸漬後1~2時間蒸煮し,別に小麦を炒煎後粉砕し,両者を混合して麹を接種し通風式ドラムなどの醗酵装置で25°C,46時間で出麹を作る.これに18~23%の食塩水と酵母を加え数箇月(菌相が次第に変化)熟成させ,圧搾ろ過し80°Cで火入れし製品とする.みそ原料出荷重量の8割が米で,麦,大豆も使う.醗酵方式は,酵素のみで短期間に熟成する分解型は甘口みそに,酵母も併用して長期熟成する醗酵型は辛口のみそに適用される.蒸煮原料を整形機で直径15~20mmのみそ玉にし麹を接種し25~40°C,3~4日で出麹にする.これに食塩と蒸煮大豆を混ねつ後,嫌気的に熟成する.d. 製パン伝統的風味ある欧州風硬めのパン(crustybread)は原料(食パンでは小麦粉100,砂糖5,食塩2,油脂4,脱脂粉乳2,酵母2,水64の割合)を1回に混合する直ねつ(strait)法が採用される.混ねつ(10~20分),醗酵(27°C,1~2時間),分割・丸め,ねかし(15~20分),整形・型詰,醗酵(38°C,40~60分),焼成(200~240°C,10~40分),型はずし,冷却,包装の順ですべて回分式操作である.混ねつ,丸めなど塑性加工に機械が用いられる.皮を薄く焼き中身を味わうアメリカ風柔かいパン(creamybread)は大量生産される.最初小麦粉の70%を醗酵させた後,残りの原料を追加しパン生地(dough)を作る中種(sponge)法が採用される.中種混ねつ(小麦粉70%,水,酵母,3~5分),第1醗酵(27°C,3~4時間),生地混ねつ(小麦粉30%追加,水,砂糖,食塩,油脂,脱脂粉乳,10~20分),ねかし(10~40分),分割・丸め,中間醗酵(27°C,10~20分),整形・型詰,最終醗酵(38°C,30~50分),焼成(200~240°C,10~40分),型はずし,冷却,包装の順で,混ねつ以外はすべて連続操作で,人員8名で毎時1万斤の規模である.
