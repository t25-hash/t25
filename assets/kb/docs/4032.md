# 2・1・4 ローリングピストン形

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

a.構造 代表的なロタスコ形圧縮機を図Ⅲ-2･6に示す．シリ
ンダ内に偏心して取付けられた回転ピストンが，平板状のベーンと
接触を保ちつつシリンダ内面を回転して圧縮作用を行い，吐出し弁
を経て圧縮ガスを排出する．ベーンは軸とクランク機構で同期して
動き，その下端は回転ピストン外周に対してつねに垂直に向き，波
形ばねでピストン外周に軽く圧着して機密を保持している．ピスト
ン外周には弾性のあるスリーブをはめ込んでいる．多量の潤滑油が
圧縮熱の除去とシール，潤滑の目的で供給されるので，1段で25
の高圧力比を得ることができる．

c .性能 ロタスコ形では吐出し終了後に，弁のすきま部分に
たまった圧縮ガスが再膨張して吸込口につながるので，絞り損失が
生じる．1MPaの吐出し圧力の場合，油冷式のため吐出しガス温
度も100℃程度で等温圧縮に近く，体積効率も良い．真空ポンプに
使用すると体積効率は99.9 %まで増加する．
