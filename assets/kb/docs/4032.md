# 3・1・7 食品加工の重要操作

> **章**: 第3章 食品機械
> **節**: 3・1 食品加工機械

a. 加熱殺菌加熱殺菌はHACCPの重要監視点(criticalcontrol point)である。原料の汚染度、製品形態および流通方式によって微生物制御の方法は異なる。生育細胞は熱に弱く60~63°C,20~30分で死滅するが、胞子など休眠細胞は120°C,4分でも死滅しないこともある。低温流通食品では製品中に生存する微生物の種類および密度が制限され、pH 5.5以上の常温流通食品では微生物の胞子までも死滅させる殺菌が義務づけられている。高温の食品を容器に充てん密封後一定時間高温を維持し容器ともども殺菌する熱間充てん法(hot charge method)、殺菌済みの食品を無菌室で容器に充てん密封する無菌充てん法(aseptic method)、容器に充てん密封後加熱殺菌する耐圧容器法(retorted method)がある。以下は高温加熱殺菌を例に殺菌価および調理価の概念を概説する。.
