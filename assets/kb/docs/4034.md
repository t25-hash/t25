# 2・1・6 スクロール形

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

a.構造 スクロール形圧縮機は，複数巻きの渦巻で構成され
る固定スクロール（fixed scroll ）と同一形状の渦巻で位相が180°
ずれている旋回スクロール（rotating scroll ）が，各中心を旋回半
径分ずらした状態で軸方向，径方向とも数十 mの間げきでかみあ
わされるように構成される．旋回スクロールが固定スクロール中心
のまわりを自転せずに旋回公転することにより，かみあわされた渦
巻と端板で形成される圧縮室の最外周に閉じこめられたガスが，
除々に中心部へ送られていく過程で圧縮される（図Ⅲ-2･11)．
その行程は，以下のとおりである．
(１)固定スクロール外側の吸込口からガスを吸入する．
(２)圧縮空間に封じ込められたガスは，旋回運動に伴う圧縮室の
縮小によって，渦の中心に向かって圧縮される．
(３)圧縮空間は中心部で最小となるため，ガスは最高に圧縮さ
れ，中心部にある吐出し口から外へ押出される．
このように，旋回スクロールの1回転中に吸込み→圧縮→吐出し
の行程が同時に連続的に繰返されるため，往復動圧縮機に比べ，一
回転中のトルク変動が少なく，低振動・低騒音を達成するのに有利
な構造である．
スクロールの形状は，円のインボリュート曲線が一般的に使用さ
れている．
.油冷式 図Ⅲ-2･12に油冷式スクロール形圧縮機の代表
的な本体構造を示す．油冷式においては，自転防止をオルダム継手
により行うのが一般的である．また，旋回スクロールには，半径方
向と軸方向のガス荷重により傾けようとするモーメントが作用する
ため，これに対抗して旋回スクロールを固定スクロールに押付ける
必要があり，ガス圧を背圧として作用させたり，ばね力を加えたり
する方法などが取られる．油冷式は，一対の渦巻の軸方向および，
径方向の間げきを冷却油がシールするため，高い体積効率
（volumetric eﬃciency）を得ることができる．一方，吐出しガス
のオイルセパレータや給油装置，給油配管が必要となるため，圧縮
機周辺機器は複雑になる．
.無給油式 図Ⅲ-2･13に無給油式スクロール形圧縮機の
代表的な本体構造を示す．無給油式の本体構造上の大きな特徴は，
補助クランク機構とチップシール（tip seal ）である．潤滑材とし
て油が使用できないため，自転防止とスラスト荷重を受けることを
目的として補助クランク機構を採用し，グリースを封入した軸受を
用いる方法が一般的である．また，軸方向の間げきからの漏れは圧
縮機としての性能を著しく低下させるため，渦巻の先端に溝を設
け，その部分にシール材を挿入する方法が一般的に用いられてい
る．このシール材すなわちチップシールは，渦巻が形成する各圧力
室間の差圧により浮上し，しゅう動面となる相手の渦巻底面に押付
けられ，シール材として機能する．
無給油式の場合は，圧縮等により発生した熱を放出するために空
冷方式を採用するのが一般的であり，周辺機器として冷却ファンを
備えている場合が多い．

c .性能 スクロール形圧縮機は，その構造より隣接する圧縮
室の差圧が小さく，さらに低圧部と高圧部の間に直接のガス漏洩通
路が形成されないため，ガス漏洩による損失が小さいことや，しゅ
う動速度が比較的小さく，機械損失が少ないことにより，往復式圧
縮機やローリングピストン式に比べて，広い圧力比範囲で優れた体
積効率および全断熱効率特性を示す．
