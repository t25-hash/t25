# 3・2・3 容器の製造

> **章**: 第3章 食品機械
> **節**: 3・2 食品の包装と充てん機械

金属缶は缶胴と天地2枚の蓋で構成される3ピース缶と、底部と缶胴とを一体で成形し、蓋からなる2ピース缶がある。a. スチール缶3ピース缶はTFS 薄板に、印刷・焼付け後、缶胴1個分へ裁断し、ハンダ付・接着・溶接のいずれかでサイドシームし缶胴を作る。上下の縁口の部分を外側に曲げフランジを作り、蓋とのつなぎシロを作る。上下の蓋は薄板を丸く打ち抜き、缶胴部分と巻き締め接着するための折り返しを作りこの周縁内部にシーリングコンパウンドと呼ばれるゴム系密封材を塗布する。缶胴部と下蓋とを二重巻締機で結合し、空缶ができる。真空能力を1缶ずつチェックする。2ピース缶は円盤状に切断した薄板をプレス、深絞りし、上縁部を切断する。缶胴部ができたところで外装、内装を施し、入口部分を缶胴直径よりも小さい径に絞り込む。蓋との接合のためにフランジを設け、リークテストを行い製品となる。b. アルミ缶アルミ2ピース缶の製造もスチールに準じている。円板状に打ち抜いた板をプレスと深絞りで所定の大きさにした後、底部のドームを作って成形は終了する。その後洗浄を行い胴の外面に印刷焼付けし、リークテストを行う。
