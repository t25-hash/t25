# 3・2・4 液体充てん機(liquid filler)

> **章**: 第3章 食品機械
> **節**: 3・2 食品の包装と充てん機械

容器の液体充てん方式は定水位式と定量式とに区分される.定水位式は液面がそろっており,商品の見栄えがよい.容器上面からの液面の高さをそろえるグラビティ方式と底面から一定の高さまで充てんするレベルセンサ方式がある.定量式には,ピストン式,計量マス式,ウエイト式,フローメータ式がある.容器精度が向上した現在は定量式が主流となっている.充てん機の構造はおおむね図3・3に示すような半球状の液だめ(フィラボウル)があり,その底面円周部に充てんノズルが等間隔で配置され,ノズル直下には容器を載せる台がある.液だめと容器台は同じ速度で回転しており,1周する間に充てんは終了する.図3・4にグラビティ式充てん状態を模式的に示した.充てん時のノズルの開閉は容器台がカム機構により上下し,容器の上縁部がノズル周辺に設けられたフランジを上げてノズル開度を調節し充てん状1―56産業機械・装置図3・3 飲料ボウル付充てん機外観図3・4グラビティ式充てん機の充てん過程模式図図3・5 バルブ昇降式充てん機態を決めており,メカニカルな方法である.容器の強度が低い場合には適さない.図3・5はバルブ昇降レバーの動きをカムで制御し充てんを行う様子を示した.図3・6はフローメータ式充てんシステムである.液だめ,あるいは圧送ラインからの液は,フローメータを経由し一定量充てんされ,バルブが閉じる.定重量法は容器台に荷重計があり,一定荷重に達するとバルブが閉じる.炭酸飲料は充てん時に炭酸ガスが発泡するので,充てん開始時の缶内圧力を液だめ上部のガス圧に等しくして充てんが開始される.終了時にも缶上部の脱気を行う.これらの操作はメカニカルあるいはメカトロニクス制御で行われている.充てん機は作業終了後各部を洗浄し,清潔に保たなければならない.CIP(Clean In Place)により,自動洗浄,図3・6 フローメータ式充てんシステム図3・7 充てん機―キャッパにおける容器の動き図3・8 無菌充てん包装システム化学殺菌,水蒸気殺菌ができるシステムがよい.しかし,自動殺菌システムには装置限界があるので,人の手でメンテナンスを行うことが必要である.図3・7に充てん機,キャッパの配置を模式的に示した.左から進入する空の容器はホイールにつかまれ周回する充てん機,キャッパへと移動する.
