# 3・2・5 無菌充てん機(aseptic packaging machine)

> **章**: 第3章 食品機械
> **節**: 3・2 食品の包装と充てん機械

充てんした食品を長期保存するためには、充てん後加熱殺菌するか、充てん時に容器を殺菌しながら、殺菌済みの食品を充てんする無菌充てん法がある。図3・8に無菌充てん包装システムを示した。包材を過酸化水素水で洗浄殺菌し、過酸化水素を高温で飛ばした図3・9粉体用充てん機の基本型図3・10 ケーサロボット後、原紙を袋状に成形しつつ、殺菌済みの内容物を充てんし、シールする。紙パック入り牛乳、ジュース類等、液状食品では複合用紙を使用し、インライン法で生産が行われる。無菌充てん機を収容する部分はクリンルーム化し、陽圧にする。
