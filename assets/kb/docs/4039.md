# 3・3・1 はじめに

> **章**: 第3章 食品機械
> **節**: 3・3 荷造り機械と荷さばきシステム

食品は,容器に詰めただけで流通経路に載るわけではない.多くはある最小のロットで流通している.個別包装された製品を一定量ケースに詰め,流通の最小単位を作り,場合によってはさらに大きな単位を作って取扱う場合もある.従来,食品工場において人手が最も多かったのは箱詰め作業であり,見落としが発生するのもこの段階である.しかし,近年はロボットの導入によりこの部分での人手は省かれつつある.流通過程では販売業者が必要とする商品を迅速に取りそろえ,指定された場所と時間に届けることが要求される.そのためには,品物を適切な量だけ保存し,迅速に取出し,配送することが可能となるシステムも必要である.
