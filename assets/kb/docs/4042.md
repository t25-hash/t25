# 3・3・4 パレタイジング(palletizing)

> **章**: 第3章 食品機械
> **節**: 3・3 荷造り機械と荷さばきシステム

段ボールに詰めた製品を輸送に便利なフォークリフト用のパレットの上にさらに大きな塊とし積み込むシステムである。図3・12は食品関連製品の個別包装、段ボール箱収納、パレタイジングを模式的に示したものである。最終段階では段ボールをパレットの上に一定の個数ずつ積み上げている。積み上げ方が不安定であれば輸送中の荷崩れの原因となるので積載には細心の注意を払う必要がある。一定の個数が積まれると、プラスチックシートで厳重に包装され、パレットへの固定と防水対策がなされる。パレットを無人搬送台車に載せ、所定の倉庫に収納する場合もある。1―58産業機械・装置
