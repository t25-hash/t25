# 3・3・5 商品管理システム

> **章**: 第3章 食品機械
> **節**: 3・3 荷造り機械と荷さばきシステム

商品の生産時点の特性を個々の製品と関連づけて記録しておくことは食品に万が一問題が発生した場合、その原因を追及する手がかりとすることができる。従来食品の管理はバーコードによって管理するだけであったが、近年、ICタグシステムが導入され、より多くの情報が記録されるようになってきている。その代表例はRFID(Radio Frequency Identify)システムである。マイクロ波によるICへのエネルギー供給と同時に情報の交換を非接触方式で行い商品の情報を親コンピュータとの間で交信し、商品の状況が把握できる。このシステムの導入により、さまざまな商品の管理が一元化でき、流通量を明らかにすることができる。
