# 3・3・6 立体倉庫と仕分けセンタ

> **章**: 第3章 食品機械
> **節**: 3・3 荷造り機械と荷さばきシステム

多種多様な商品を揃え，保管する施設としては立体倉庫が優れて
いる．コンピュータ管理と商品の取出しのために三次元的に動く高
速リフトの組合せにより倉庫の機能が発揮される．冷凍食品の保管
にも立体冷蔵庫が採用されるが，低温状態でも十分能力が発揮でき
る機材と設備が必要である．
さまざまな配送条件を満たすためには商品を保管する大規模な倉
庫とともに，品揃えをするための物流センタシステムを構築するこ
とになる．物流センタでは配送先が要求する商品を効率よく集める
ラック方式では収納部全体を冷却または加熱する．
ピッキングシステムの構築とともに，在庫品を管理し「最適在庫
3・4・3機内加工する自動販売機
化」を実現することにより無駄な在庫を避けることができる．
