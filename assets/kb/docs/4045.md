# 3・4・1 種類および機能

> **章**: 第3章 食品機械
> **節**: 3・4 自動販売機

自動販売機(automatic vending machine)は貨幣またはそれに代わるカードを挿入することで自動的に物品等を販売する装置と定義される。普及台数は1984年に500万台に達し、以後飽和しているものの自販金額は現在も上昇している。清涼飲料の半数は自販され、社会の公器ともいわれ、物品等自動販売、サービス情報販売、自動サービス機に大分される。それぞれ種類用途とも多彩である。物品等自動販売機は、(1)飲料(清涼飲料、コーヒー、アルコール、乳飲料、各種併売、その他)、(2)食品(菓子、調理食品、汎用食品、冷菓および氷、各種併売、その他)、(3)たばこ、(4)券類、(5)切手・はがき・印紙および証紙、(6)新聞および雑誌、(7)日用品および雑貨、(8)各種物品複合・併売、(9)その他の物品、に商品分類される。業界団体等では、その機能構成を、(1)金銭管理部、(2)商品保存部、(3)冷却・加温部、(4)制御・通信部、(5)接客部、(6)きょう体に区分している。ここでは食品機械として、飲料用および食品用自動販売機の商品保存部機能に着目し、常温流通包装食品用、機内調理加工するもの、冷凍食品専用に分けて取扱うのが、構造的にも法的にも都合よい。
