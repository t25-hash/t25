# 3・4・2 常温流通包装食品を扱う自動販売機

> **章**: 第3章 食品機械
> **節**: 3・4 自動販売機

機械が食品に直接接触しないか、または冷凍食品を解凍しなければ調理する行為に当たらない。ただし、常温流通といえども、保蔵目的ではなく嗜好のために、包装のまま冷却または加熱されることがある。狭い設置面積に多種商品を保存し、先に収容した商品から先に販売するため、垂直方向に縦長の収容空間の上部に商品を供給し下部から取出す。個装状態であるから衝撃で下の商品が破損または変形しないように、次の待機商品を支える。a. 直積みスタック方式たばこまたは紙パック容器詰め食品のように箱型で、自重が軽く、十分な強度があり、形状が単純で、面に折り返しなどがない平滑な商品は直積みとする。スタック(stack)は文字どおり上に積み重ねることであるが、最下部商品を、スライダ(slider)で滑り出させる方法、または大小二つのフラッパ(flapper)で交互に支持して落とす方法がある。b. サーペンタイン方式円筒状で強度の高い商品に適する。160~500mL缶、または100~300mL瓶の比較的小重量商品に適用される。商品通路兼保存のための垂直方向に長い空間をわずかに蛇行させ、壁面は円弧翼状の板金セクタで構成し、片方の壁面セクタは上端をヒンジで支持し、下端が商品を対向壁面に押え付けるようにする(図3・13)。蛇行(serpentine)壁面が商品自重を分散支持するので、収容量に関係なく下向きのほぼ一定の搬送力が生じる。商品形状は、当初缶または瓶であったが、樽状、円すい台、多角形にも、また材質はスチール、アルミ、ガラスに限らずプラスチックス、紙にも対応可能になった。細缶(直径53mm)または太缶(直径66mm)に調整できる。販売順の近づいた収納部下半分の商品のみ、冷却または加熱することができ省エネルギーおよび品質管理上有利である。c. ラック方式上記aおよびbの2方式は商品が互いに接触するので、大重量または複雑な形状の商品に適さない。商品は個々に支持する。チェーンラック(chain rack)方式は、0.5~2Lの大型円筒形に適し、上下の二つのスプロケットによって回転する無限チェーンに一定ピッチで取付けた突起と垂直壁とによって支持される。コンベアラック(conveyer rack)方式は、商品を積載する短い水平コンベアを、棚様に多層設置し、奥に設けたエレベータを介して商品を排出する。スパイラル(spiral)方式は、軸が垂直ならせん状の棚に商品を積載し、らせん棚を回転して商品を排出する。三角すい、四角すいまたは家型など複雑な形状に適用できる。ラック方式では収納部全体を冷却または加熱する。
