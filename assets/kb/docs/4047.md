# 3・4・3 機内加工する自動販売機

> **章**: 第3章 食品機械
> **節**: 3・4 自動販売機

食品を機内で調理し販売することは、飲食店または喫茶店営業などの行為とみなされる。弁当、ハンバーガ、機内給湯するカップ麺、生麺、機内加熱する冷凍食品、および機内油調するフライドポテトなどの自動販売もこれに該当する。食品別に操作温度、操作時間、洗浄、殺菌および乾燥方法など詳細な基準が制定されている。以下はカップに注いで飲料を提供するコップ式自動販売機の調理機構をおもに概説する。カップ供給装置(cup dispenser)、炭酸飲料用カーボネータ(内圧400~500kPa、注水圧1MPa)、貯湯槽、製氷装置および貯氷槽などの構造は共通している。a. シロップ飲料原料が液状の飲料を指す。飲料製造工場でコンテナに封入して配送するので衛生的である。プレミックス方式はすでに最終製品に加工されコンテナに封入された飲料を販売時点でカップに注ぐ。構造が簡単である。氷を入れることもある。ただし、pH が高く微生物増殖に適するので、85°C以上の温湯による洗浄消毒が必要となる。これに対しポストミックス方式は、5倍の濃縮液をコンテナに封入配送し、水道水によって機内で希釈するので輸送量も少なく済み、pH が低く耐菌性が優れ衛生管理上も有利である。いずれの方式も、送液にはガス圧またはポンプを使う。ガス圧式は機内の炭酸ガスボンベより400kPaにコンテナを加圧する。炭酸ガスを嫌う飲料に対しては、ポンプ式とする。耐圧性の低いバッグインボックス式のコンテナにも適用できる。b. 粉末飲料インスタントコーヒー、ココアなど粉末即席飲料の加水方式は二つある。カップ内かくはん方式は、家庭での用法と同様に、カップに粉末と湯を同時に投入しインペラでかくはんする。清潔的である一方たまに未溶解物が残ることもある。ミキシン図3・13 サーペンタイン方式(左)、コーヒー抽出機(右)グボール方式は、粉末と湯を上部から取込むろ斗型密閉容器内に、インペラがあり高速回転中は液が流出せず、かくはんする。止めると液はカップへ注がれる。湯のみ2回めに給湯し残液を完全販売し洗浄も兼ねる。冷飲料は少ない湯で溶解し、かくはん完了後カップに砕氷を入れる。c. レギュラーコーヒー煎じる直前に豆を粉砕する正統様式を自動化した販売機である。粉砕機と独特の茶煎装置(brewer)を備えている(図3・13)。抽出時間を短縮するため、円筒容器に粉砕豆と湯を注入し圧縮空気でかくはん後一定時間むらし、かくはん容器を上方に移動し栓を密封し空気圧で底部ろ紙でろ過する。容器底部を下げて、残渣はろ紙とともに横へ送り排出処理する。d. リーフティー茶煎装置は底部のストレーナが下方向に脱落できるろ斗に茶葉と湯を注入し、自然対流でかくはんし、重力流下する。後半の動作で底のストレーナを下げ、残渣はスクレーバで掻き落す。ストレートの場合と、ミルク、またはレモンなどを添加する場合とでは別の回路を通る。添加を希望するときはミキシングボールに導き混合してからカップに注入する。e. 複合型上記a~dを1きょう体に混成した自動販売機である。
