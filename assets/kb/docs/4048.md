以上のアルゴリズムに基づく解析例として，図2･75に示すよう
な中央を遮音カバーで仕切られた音場解析（sound field analysis ）
例を紹介する．音場空間は複合材からなる平板で音源室側と受音室
側の2部屋に仕切られており，カバー周辺部は完全拘束されている
ものとする．遮音カバーとして，図2･76に示すようにガラス短繊
維（short glass fiber ）とガラス長繊維（continuous glass fiber ）
により補強された樹脂の2種類を仮定する．音源室側の底面に音源
を設け，各節点に対して1.0Paの圧力を与えることにより音源室
内に約100dBの音を発生させる．そのときの音源室内と受音室内
の平均音圧レベルの差から透過損失を求め，カバーの遮音性を評価
する．比較のため，樹脂が弾性（φ＝0 ）の場合と位相差がφ＝
0 .05πの場合の二とおりについて解析した．音源周波数を
400～3000Hzに変化させたときの音源室内と受音室内の音圧レベ
ル差をとったものが図2･77(ａ)(ｂ)である．図2･77(ａ)はガラス
短繊維を混入させた結果，図2･77(ｂ)はガラス長繊維を混入させ
た結果である．両者とも音場室内の共鳴によって850，1700，
2500Hzにおいて遮音性が著しく低下している．弾性の場合，短
繊維で補強された樹脂カバーは500，1200，2200Hzで，長繊維
で補強されたほうは600，2050，2800Hzにおいて遮音性が低下
している．これは，遮音壁の固有振動数によるものである．一方，
粘性を考慮した解析では，固有振動数近傍において遮音性の顕著な
低下は見られない．粘性を考慮した樹脂カバーは1750Hz近傍で
弾性仮定の樹脂カバーよりもやや遮音性が低下している．これは，
位相遅れを考慮することにより固有振動数が低下し，音場の共鳴周
波数（1700Hz）の影響を強く受けたためである．
以上の解析例より，平板内部に生じるひずみの位相遅れによっ
て，樹脂カバーは固有振動数近傍における透過損失の低下が緩和さ
れることを確認した．また，ガラス繊維の形状や寸法によっても遮
音性能にかなりの相違が生じることを明らかにした．本手法を用い
れば，樹脂複合材に混入された補強材の形状や混入率に応じた音圧
レベル分布を調べることができるのと同時に，樹脂の粘性の影響を
考慮した，複合材の遮音性能を予測することが可能となる．
ここでは主として複素弾性率を用いた定式化を示した．より一般
的なマクスウェル型の粘弾性構成則による均質化法の定式化と遮音
解析については文献(83)を参照していただきたい．
