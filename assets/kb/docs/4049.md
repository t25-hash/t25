# 2・1・2 音質の定量化技術

> **章**: 第2章 快適環境技術
> **節**: 2・1 快適音場の創生

機械騒音の音質（sound quality）を改善する際に，人間の主観
的な感覚を基に改善対策を講じることになるが，その結果として改
善効果があったか否かを判断するためには，対策技術を講じるたび
に対策前と対策後の音を対象とした音質評価実験をする必要があ
る．正確な主観評価実験を行うためには，周囲環境に制約を受けな
い実験施設と音質判断に慣れた多くの被験者が必要となるが，両方
を満足することは難しく，これが大きな制約となる場合がある．
そこで音質の主観的な判断結果と対応の良い心理音響尺度（psy-
cho- acoustical scale) を相関分析結果に基づいて探しておくこと
や，主観的な音質判断に合う感覚尺度の定量化を行い，音質の差を
客観的に表すことができる尺度を用意しておくと，経験と労力の必
要な音質の主観評価実験に頼ることなく，尺度の測定値によって対
策効果の有無とその大きさが判断可能となるため具合が良い．
a.音質空間の次元 音質の定量化を行うには，機械音の音
質を表す意味空間 の次元が何次元であるかを知る必要がある．
自動車の走行時の車内音を例に表2･2に示す14対の両極形容詞尺
度（polar adjective scale）を用いてSD法 により音質評価を実
施し，その結果を因子分析すると図2･12，2･13に示すような結果
が得られる．
図2･12，2･13から機械騒音の音質を表す意味空間は三次元空間
であるといえる．表2･2に示された形容詞尺度群を用いて図2･12，
2･13に布置された形容詞尺度の意味を考えると，第1軸は迫力感
因子，第2軸は快・不快感因子，第3軸はこもり感因子（周波数因
子，金属性因子）を表す軸であると解釈することができる．機械音
のデータにより軸の順番の入れ代わりは起こり得るが，この結果に
示されたように，三つの因子軸が現れることは一般的なことといえ
る．したがって，音質の定量化を考えるには，この三つの感覚の客
観尺度化が必要となる．

c.快・不快感因子の定量化尺度 快・不快感と密接な関係の
ある音の物理的な属性は，音響エネルギー，高周波成分，音圧の時
間エンベロープ変動，および衝撃性である．このうち音響エネルギ
ーの大小については，これまで古典的な意味で騒音対策が講じられ
てきた際にその考えの基になっているものであり，音質とは一線を
画するものといえる．音質対策と関連しては，高周波成分の大き
さ，音の時間エンベロープ変動および衝撃性の有無と快・不快感の
対応について考えればよい．
.高周波成分の大きさと快・不快感の対応尺度 高周波数
成分の大きさと密接な関係のある心理音響尺度はsharpness （鋭
さ) である．
図2･15に機械音の快・不快感評価結果とsharpnessとの対応を
示してあるが，2kHz以上，特に3kHzを超える高周波数成分が
強いとsharpnessが高くなり，不快な音として評価されることがわ
かる．
.音圧の時間エンベロープ変動と快・不快感の対応尺度
音圧の時間エンベロープ変動の大きさと関係のある心理音響尺度に
roughness （粗さ) がある．図2･16にroughnessと音の快・不
快感評価結果との対応を示してある．Sharpnessとの対応ほどは相
関が高くないが，やはり音圧の時間エンベロープ変動が大きく
roughnessの大きな音は不快と評価されていることがわかる．
.衝撃性の強さと快・不快感の対応尺度 例えばディーゼ
ルエンジン音は，ガソリンエンジンと比べてエンジンのシリンダ内
の圧縮比が高いため燃焼が衝撃的であり，不快感が高いといわれて
いる．衝撃感と対応の良い心理音響尺度としてimpulse という
尺度がある．これは，衝撃音を200msごとに細分化し各区間の
音圧の実効値を超えるピーク値のエネルギーを取り出して区間ごと
の値をエネルギー加算平均した尺度である．図2･17に，音の衝撃
感とimpulseの関係を示したが高い相関があることがわかる．衝
すことができるとわかる．
