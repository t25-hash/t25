# 3・5・2 凍結処理能力(freezing capacity)

> **章**: 第3章 食品機械
> **節**: 3・5 食品凍結装置

凍結処理量を決定するために必要な項目を表3・3に示す。対象食品の凍結開始時と凍結完了時の品温(中心温度)、その間の冷却媒体温度および凍結時間h(この関係をグラフ化したものを凍結曲線という)を実測あるいは計算で求める。食品の厚さ、包装形態、冷却媒体の種類、温度および流速は凍結時間に大きな影響を及ぼす。この凍結処理量から凍結設備の方式が決められる。次に、凍結設備の処理量に適合した冷凍機の選定が必要である。冷凍負荷についてはγ3編3章に詳細説明。自動搬送式の凍結設備の場合は、食品が流れている場合(正味負荷+装置負荷)と、流れていない場合(装置負荷のみ)の負荷変動が大きいので容量制御可能な冷凍方式を選ぶ。一般に、食品の終温より10~20K 低めの冷却媒体温度とし、蒸発器の大きさはさらに冷却媒体温度より8~10K 低い温度になる蒸発温度を選ぶので、仮に食品終温を-20°Cとした場合、蒸発温度が-50~-40°Cになり、この蒸発温度で冷凍熱負荷以上の冷凍能力が得られる冷凍機を選定する。
