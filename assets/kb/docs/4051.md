# 2・1・4 高速鉄道車両の音質設計技術

> **章**: 第2章 快適環境技術
> **節**: 2・1 快適音場の創生

a.高速鉄道車両の音源および伝達経路 高速鉄道車両の車内
騒音における音源としては転動騒音，駆動系騒音，空力騒音，機器
騒音，空気調和（空調）装置騒音などがあると考えられており，こ
の中でエネルギー的に大きい成分は転動騒音，駆動系騒音，空力騒
音であると考えられている．以下に各音源の特徴について述べる．
.転動騒音（rolling noise) 転動騒音は，レールと車輪
の接触面の粗さが原因となって，レールおよび車輪が振動して騒音
を放射する現象である．騒音の放射が大きい周波数帯域は一般的に
800～2000Hzであることが知られている．高速鉄道車両ではブレ
ーキディスク付き車輪が一般的であるため，車輪の固有振動の影響
は顕著には現れない．その他のレール-車輪系に起因する音源とし
ては，台車ばね下質量とレールをマスとして，道床をばねとする
60～70Hz付近の共振や，レール締結装置によるレール支持剛性の
変動に起因する約130Hz付近（車両速度を300km/hとした場合）
の騒音成分などが知られている ．これらの成分は狭帯域である
ため，エネルギー的には寄与は小さいと考えられている．輪軸のア
ンバランスは車両速度を300km/ hとした場合には30Hz程度であ
り，周波数が低いために車内騒音として問題になることはほとんど
ない．
転動騒音の伝達経路としては，800Hz以上では，レールおよび
車輪が振動して放射された騒音が車体の床面を透過してくる音響伝
ぱ成分が支配的であり，100～300Hzでは，車輪の振動が台車の車
体支持装置を介して車体に伝ぱしてくる振動伝ぱ成分（固体伝ぱ
音，solid- borne sound）が支配的であると考えられている．
.駆動系騒音（driving noise) 狭帯域の音源として，主
変換装置のVVVF（variable voltage variable frequency）制御に
起因する高調波音，歯車装置のかみあい音，主電動機の回転子の風
切り音 などが知られている．発生する周波数帯域としては，主
変換装置のVVVF制御に起因する高調波音はGTO（gate turn oﬀ
semiconductor controlled rectifier ）素子を採用した場合では約
500Hz付近，歯車装置のかみあい音は車両速度を300km/ hとし
た場合には約2000～2600Hz程度，主電動機の回転子の風切り音
は車両速度を300km/ hとした場合には約4300Hz程度である．
これらの音源は狭帯域であるため，エネルギー的に大きく寄与する
ことはない．また，主変換装置のVVVF制御に起因する高調波音
については，IGBT（insulated gate bipolar transistor ）素子の採
用により高周波帯域にシフトしたため，近年は車内騒音としてはあ
まり問題にはならなくなった．歯車型継手に起因する騒音が特に惰
行走行時に発生し，車内騒音としてはエネルギー的にもかなり大き
くなることがある．周波数帯域としては200Hz付近を中心に広い
帯域で発生する．原因としては，惰行走行時に歯車部分の押付けト
ルクが緩和されて，歯打ちが発生するためと考えられている．主電
動機の冷却ファン騒音については，高速鉄道車両では強制通風式の
冷却ファンが一般的であるため，車内騒音としての寄与はほとんど
ないと考えられている．
伝達経路としては，100～300Hzでは台車の車体支持装置を介し
た固体伝ぱ音が支配的であると考えられ，また歯車装置のかみあい
音や主電動機の回転子の風切り音は音響伝ぱ成分が支配的であると
考えられる．
.空力騒音（aerodynamic noise) 走行中の高速鉄道車
両における空力騒音の主要な発生部位は，先頭部，車間部，台車
部，パンタグラフであることが知られている ．図2･32に走行
中の高速鉄道車両における車外音源の分布状況を測定した事例を示
した．また車内騒音において空力騒音が寄与する周波数帯域は500
Hz以下であろうと推定されている．伝達経路としては，車室に直
接面している台車部で発生した騒音が床を介して車室へ侵入してく
る割合が大きいと考えられており，また台車部で発生する空力騒音
の発生機構としては，キャビティ音であると考えられている ．
転動騒音や駆動系騒音などの機械的な騒音は車両速度の三乗に比例
するといわれているのに対して，空力騒音は車両速度の六乗に比例
するといわれており，車両速度が高速化されるに従って空力騒音の
寄与は増大していくといわれている．
.機器騒音 床下機器の中では主変圧器が特に問題となる
ことが多く，冷却ファンの騒音，電磁音などが音源である．停車中
では寄与が大きいが，走行中では転動騒音，駆動系騒音，空力騒音
に比較するとエネルギー的な寄与は小さいと考えられている．床か

.空調装置騒音 吹出し口からの気流音が車内に放射して
くる騒音であり，広い周波数帯域に分布した成分をもつ．停車中で
は寄与が大きいが，走行中では転動騒音，駆動系騒音，空力騒音に
比較するとエネルギー的な寄与は小さいと考えられている．
b.低騒音化設計技術
.音源の低騒音化設計技術 転動騒音については，レール
の削正が効果的であることが知られている．削正の頻度や粗さの管
理レベルの設定は環境騒音も含めて決められている ．図2･33
にレールの削正によりレール振動とレール近傍騒音が低減された結
果を示す．駆動系騒音については，主変換装置のVVVF制御に起
因する高調波音に対してはIGBT素子の採用が効果的である．空
力騒音については，キャビティ音を低減する方法として，エッジに
スポイラを設置することや，エッジを丸めたり傾斜面をつけたりす
ること などが知られている．主変圧器については，主変圧器本
体の防振支持が有効であると考えられる．
.台車の低騒音化設計技術 高速鉄道車両の車内騒音の周
波数的な特徴としては，A特性フィルタによる重み付けの影響に
より低い周波数帯域ではレベルは低下していき，また一方で高速鉄
道車両の車体のもつ気密の高い構造や出入り台と車室が仕切られて
いる構造に起因して高周波数帯域においても防音・防振効果は向上
し，高い周波数帯域でもレベルは低下していく．その結果，周波数
的な特徴として100～300Hzの帯域が卓越してくる．転動騒音お
よび駆動系騒音は，100～300Hzでは，おのおのの音源である車輪
の振動や駆動系の振動が台車枠や車体支持装置を介して車体に伝ぱ
してくる固体伝ぱ音が支配的であると考えられている．このため，
台車におけるこれらの振動伝達を低減することは，高速鉄道車両の
低騒音化設計を進めるうえで重要である．
100～300Hzの周波数帯域における台車の振動伝達の中で考慮す
べきこととして，一本リンクやヨーダンパなどの車体支持装置の共
振や台車枠の曲げ共振が，車輪や駆動系の音源から車体への振動伝
達を増幅していることが考えられる．これらについては，共振系に
ダンピングを与えて振動伝達の増幅を低減させることが有効である
と考えられる．リンクの中空部に粒状体（granular materials ）を
封入した粒状体封入式一本リンクによる低減の試みがなされてい
る ．図2･34 に粒状体封入式一本リンクによる車内騒音低減の
結果を示した．
.車体の低騒音化設計技術 近年，高速鉄道車両ではアル
ミニウムの中空押出し形材を使用した車体構造が普及してきてい
る．この構造は高い剛性が確保できることにより，剛性が効果を発
揮する周波数領域では有効であるが，他方でコインシデンス効果に
よる遮音性の落込みが低い周波数帯域にシフトしてくるため，注意
が必要である．a項で述べたように，走行中の騒音としてエネルギ
ー的に大きい成分は転動騒音，駆動系騒音，空力騒音であろうと考
えられている．そして，これらの音源はいずれも台車部に集中して
おり，車体としても防音ならびに防振構造を，台車部の床面に対し
て特に重点的に施すことが有効であると考えられる．そして，この
周波数帯域では固体伝ぱ音が支配的であると考えられている．
このように低い周波数帯域の固体伝ぱ音が支配的な音源に対して
は，外板の気密構体と床板，内装材の二重構造を利用した遮音性の
向上が有効であるといわれている．具体的には浮き床構造の採用，
内装材の柔支持などが考えられる．100～300Hzに対して，二重構
造を利用した防音・防振効果を発揮させるためには，この二重構造
の共振周波数を70Hz以下に設定することが必要である．また，
さらに床板や内装材の膜振動による共振が増幅していることが考え
られ，この膜振動の抑制も有効であると考えられる．
c .制御技術を適用した防音・防振技術 従来の受動的な防
音・防振技術では，構造上の工夫のみによって重量増加や大きなス
ペースを費やすことを避けるのは難しかったが，近年，軽量・省ス
ペースと両立する防音・防振構造としてセンサとアクチュエータで
構成される制御技術を適用した手法が注目されている．この場合の
アクチュエータとしてはピエゾ素子（piezoelectric element ）がよ
く試用されている．鉄道車両の車体構造に関連が深い内容として
は，ピエゾ素子を用いたスピーカにより二重構造の中空部分に音を
発生させて透過音を打ち消す方法 や，床板に板状のピエゾ素子
を張り付け，分岐回路を付加して床板の膜振動を低減させる方
法 などが試みられている．図2･35 にピエゾ素子と分岐回路
を組合わせて床板からの音響放射を低減させた結果を示した．
現状は，まだ実験室レベルの試みが多いが，今後，コストの面や
信頼性の面でさらに改良され，実用レベルに発展していくことが期
待される．
