# 3・6・1 ショーケースの概要

> **章**: 第3章 食品機械
> **節**: 3・6 食品貯蔵装置(ショーケース)

ショーケース(display case, display cabinet)は、生鮮食品など、その収納商品の品質を維持しながら、展示効果をもたせ、消費者に販売することが目的であり、消費者に直接商品を販売する重要な機器と位置づけられている。日本での歴史は、1960年頃にアイスクリームストッカが魔法瓶式のストッカの代わりとして登場したことに始まり、小型ショーケースは日本独自で発展を遂げてきたものである。一方、スーパマーケットに見られる大型のショーケースは、アメリカからの技術支援を受けたことに始まり、現在まで各時代ニーズにあわせた製品開発が遂げられ、多くの新技術が誕生している。近年は、冷蔵・冷凍につぐ新温度帯と呼ばれる温度帯にて食品を高鮮度に保存するショーケース、マイコン制御方式を活用した集中管理システム、環境保全・省エネルギー対応のショーケースなどが登場し、脚光を浴び始めている。
