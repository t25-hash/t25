3・7食品プロセスセンサ（food process sensor）
3・7・1はじめに
食品産業は食品のための素材生産から，加工食品・調理食品の生
産まで多岐にわたっている．製品品質が均一であり，美味しく，安
全であり，しかも栄養学的にも安定したものが要求される．このた
めには品質管理（quality control）が重要であり，3･1･7項に示し
たHACCPとGMP（good manufacturing practice）の視点から
品質を評価・管理しなければ消費者には受け入れられない．
センシングシステムは製造ラインとのかかわりの浅深度から，オ
フライン，アットライン，オンライン，インラインに分けられる．
a.管理すべき箇所 食品の品質は，①原料，副原料の品質，
②生産プロセスの運転状態・条件，③最終製品の保存・保管状態に
よって左右される．さらに，消費者への流通過程では，④異物混
入，⑤量目違い，⑥包装等の外見上の異変，などがクレーム事項と
して挙げられる．
また，生産現場からは原料，副原料の安定供給が重要な管理事項
となる．
b.管理すべき事項 上記の箇所で管理すべき事項はさらに複
雑である．①では，原料の成分，水分量，鮮度，不純物混入，②で
は，製造過程における物理的なパラメータが製品の品質を左右する
ことはよく知られており，加熱調理や缶詰生産における温度と被加
熱時間の管理は重要な管理事項である．生産過程における圧力，温
度，pH，粘度，糖度，塩分濃度，液体連続殺菌における温度と液
流量，かくはん速度，凍結乾燥時の真空度，噴霧乾燥における高温
空気温度と噴霧液滴サイズなど，生産プロセスによってさまざまで
あり，管理すべきパラメータを選ばなければならない．物理的パラ
メータ，化学的パラメータ，生物化学的なパラメータを測定するこ
とによって目的を達成することができる．③では食品が置かれてい
る温度，雰囲気中の水分や酸素ガス組成，紫外線量，④では金属，
人毛，昆虫，その他の異物，⑤では製品包装前の重量，液量，⑥で
は標準製品との画像での比較，によって，標準からのずれが判定で
きる．
c .測定結果に基づくアクション 測定結果が設定値から離れ
ている場合，ただちに設定値に復帰するための処置をする．時間遅
れがあっても結果的に製品に危害因子となり得ない程度の範囲での
復帰ができるように許容範囲を設定することが必要である．

b.クローズドタイプ 多くの小型ショーケースはこのタイプ
であり，多段タイプで扉を備えているものをリーチインショーケー
ス（reach- in display case）と呼ぶ．扉部にはガラスを用いている

ス，熱線反射フィルム，防露ヒータを使用するなど工夫がなされて

c .強制循環方式 庫内の空気をファンなどにより強制的に循
環させるもので，オープンショーケースの場合，底面または背面に
置かれた冷却器により熱交換を行い，冷却した空気によりエアカー
テン（図3･17）を形成し，外気の侵入を防止している．この方式
はリーチインショーケースでも採用されている．
d.自然対流方式 冷却コイルによりその周りの空気が冷やさ
れて下降し，他の冷却されていない部分の空気が上昇する．この循
環によりケース全体を冷やす方式である（図3･18)．強制循環方式
ほどの冷却能力は得られない．一般にクローズドタイプで用いられ
ている．特殊な例として，スポット型ショーケース（催事用）があ
る．この場合，チューブオンシート（tube-on-sheet evaporator）
やロールボンド（roll bond evaporator）（図3･19）と呼ばれる冷
却器が内面材に使われ，熱伝導を利用して商品を冷却している．
