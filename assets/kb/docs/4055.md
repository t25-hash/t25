# 3・7・2 オフラインセンサ(off-line sensor)

> **章**: 第3章 食品機械
> **節**: 3・7 食品プロセスセンサ(food process sensor)

食品生産工程に使用されるセンサシステムはオフライン技術を元に発展したといえる.ここでは,生産ラインで必要とされる非破壊食品成分分析法のセンサシステムである近赤外分光法について述べる.a. 近赤外分光法(near infrared spectroscopy)近赤外光は800~2500nmの電磁波であり,食品成分に含まれる水,タンパク質,デンプンの各分子のO-H,N-H,C-H の官能基によってこの帯域で光吸収が起きる.したがって,食品基本成分の基準吸収パターンと測定対象物の吸収パターンを比較することにより定性分析と定量分析が可能となる.図3・21に水,タンパク質,脂質の基本吸収曲線と大豆,米,デンプンの吸収曲線を示した.1935nmは水の,2100nmはデンプンによる吸収バンドである.大豆における脂質吸収バンド,米のデンプン吸収バンドは,標準のものとよく対応している.近赤外光源を上記の特定波長透過フィルタで分光し,食品による吸収変化を観察することにより連続的に成分分析が実現できる.近赤外法を利用することにより,穀物の水分,タンパク,デンプンの各量が測定されている.また,飲料中のアルコール量,ジュースの糖度,ビスケット類の脂質も測定可能である.さらに,図3・21 近赤外線基本吸収曲線図3・22 金属検出機の原理酪製品,肉類,加工食品,青果物の成分測定に利用できる.しかし,反射光の分析を行っているため,定量的な測定については表面状態の違いが結果に現れることに注意したい.
