# 3・7・3 オンラインセンサ(on-line sensor)とインラインセ

> **章**: 第3章 食品機械
> **節**: 3・7 食品プロセスセンサ(food process sensor)

ンサ(in-line sensor)食品生産工程管理におけるセンサシステムとして備えるべきことは測定精度もさることながらセンサそのものが,食品の汚染源と成らないことである.食品に非接触の測定法が最良であり,接触部がある場合でも構造が簡単であり,センサ部分が破損しても破片の混入がないことが必要である.また,洗浄等衛生管理しやすい,いわゆるサニタリー式装置でなければならない.光を利用したセンシング法はその点で優れている.上記,近赤外分光法は小麦粉の水分,タンパク質オンラインセンサとして用いられる.近赤外分光法はモモ,カキ,スイカ等の果実類の熟度を判定する方法に利用されており,蜜入りリンゴの等級分けにも用いられている.インラインセンサシステムは直接生産ラインにセンサを組込み,測定に供するものである.電磁流量計,コリオリ質量流量計は液体食品のパイプラインに組込み,液体食品の充てん量制御に利用される.液体食品ではパイプラインに直接センサを組込むことが可能であり,粘度計も組込まれている.圧力計はダイヤフラムを介して接液するものが多く,ダイヤフラムを清浄にすることが必要である.固形食品の重量選別機はライン中にロードセルを装備した秤量区画部を設ける.この部分はラインのベルトからは独立しており,食品は上流部よりこの区画に乗り移り秤量後再び下流のベルトに乗り移る.下流域では秤量結果を基に試料の分級を行う.ラインのベルト上に振り分け用腕木,あるいはスライド式振り分け装置があり,1―63図3・23 分光選別機(spectro-separator)の構成指定された重量区分へ自動的に振り分けられる.このシステムは量目の過不足検定にも用いられる.食品中に混入する磁性金属はいわゆる金属探知機(metal detec-tor)で確認できる.原理を図3・22に示した.食品が流れるベルトがコイルをくぐり抜けるようになっており,コイルは送信コイルと二つの受信コイルから構成されている.送信コイルと受信コイルとの間には一定の磁界が作られているが,食品に金属が含まれる場合にはこの磁界が乱される.磁界の乱れの状況をさまざまな異物について求めておくことにより異物の混入を監視することができる.非磁性体については近年,軟X 線装置が使用されている.しかし,毛髪,昆虫等微細なものについて識別は困難である.食品生産上の基本的物理量である温度計測は,熱電対,サーミスタ,白金抵抗の各温度計が温度領域に応じて用いられる.食品保存期間中の温度履歴は品質と密接に関係しており温度-時間積算計(time temperature integrator)が必要である.このためには温度ロガーによる管理が必要である.低温輸送期間中にも測定が可能な小型ロガーが開発されており,品質評価の重要な装置である.
