# 2・2・1 はじめに

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

室内環境保全技術というと，空気清浄機，室内換気装置，室内空
調装置，タバコの分煙脱臭装置等を思い浮かべると思われる．機能
的に分類しても，換気，空気浄化，脱臭，殺菌，タバコ臭の除去等
があり，これらの機能を組合わせた多機能をうたったものも見受け
られ，最近ではマイナス空気イオン，クラスタイオン等の用語も耳
にするようになっている．すなわち，室内環境保全に対する，その
社会的背景，行政的背景，技術的背景が多岐にわたっており，また
例えば3章「大気環境保全技術」に密接に関連する分野でもある．
本節では，室内空気清浄技術について概説する．
