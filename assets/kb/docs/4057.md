# 3・7・4 分光法による米粒の選別

> **章**: 第3章 食品機械
> **節**: 3・7 食品プロセスセンサ(food process sensor)

図3・23に米粒を連続的に選別するシステムを示した。樋状のシュートを流れる米粒が分光検出部における光源(ハロゲン光、蛍光灯)と光センサの組合せにより不良米粒を検知し、落下の時間遅れを見込んだうえで、選別部分においてノズルからの圧縮空気で不良品を選別するシステムである。このシステムではシュートを増やすことにより選別能力を高めることができる。100シュートで3~5tの選別が可能である。このシステムは米粒のみならず、アフラトキシン付着ピーナツの選別、コーヒー豆の選別等に用いられている。原料を1個1個選別するという究極の選別法といえる。1―64産業機械・装置文献(1) 澁谷工業、フイラーカタログ、(2000)、12.(2) 塚野健一、食品機械装置、35-8(1998)、84.(3) 八木直樹、 食品の無菌化包装システムハンドブック、 (1993)、 292、サイエンスフォーラム。(4) 日本機械工学会編、機械工学便覧、C1(1991)、58.(5) 澁谷工業、F-CAM ボトリングシステムカタログ、(1995)、4.(6) 梅田宜暉、食品設備・機器事典、(2002)、672、産業調査会。(7) イシダ、製品総合カタログ、(2003)、3.(8) 河野澄夫、食品設備・機器事典、(2002)、863、産業調査会。(9) イシダ、検査機器総合カタログ、(2003)、40.(10) 佐竹覚、米の分光選別機の開発と設計、(2001)、10、農業機械学会。→ Next Chapter
