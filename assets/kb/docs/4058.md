# 4・2・1 造粒(pelletizing)の目的

> **章**: 第4章 プラスチック加工機械
> **節**: 4・2 混練機

プラスチックは重合反応によって製造され,その性状はプラスチックの種類や重合方法により粉末状(ビーズ状,フレーク状を含む)と溶融状の2種類に分けられる.さらにプラスチックは熱可塑性樹脂と熱硬化性樹脂に分類される.熱硬化性樹脂は打錠機により圧縮後ペレット化されるものと,押出機により溶融,混練された後ペレット化されるものとがある.一方,熱可塑性樹脂は硬質ポリ塩化ビニル(RPVC)を除いてほとんど溶融,混練された後ペレット化される.そして二次加工業者に供給されている.ペレット化の目的は次のような理由による.1取扱いおよび貯蔵性の向上:溶融状原料は取扱い上ペレット状に造粒する必要がある.粉末原料も一般に引火爆発性を有し,また長期間粉末貯蔵すると酸化劣化を生じるため各種添加剤を適量混練する必要がある.またペレット化することによって,取扱い,輸送などの作業性が容易となる.2各種添加剤およびフィラーの混練:最終製品の使用目的,要求物性,加工条件に応じて種々の添加剤,フィラー類を適量混練する必要がある.3物性の改質:混練機による機械的エネルギーを加えてポリマを均一混練することにより最終製品の諸物性を改善する.例えば,低分子量物と高分子量物から構成された樹脂のフィルム強度の向上,高圧法ポリエチレン(LDPE)の光学的性質の改善などがある.4原料中の残留モノマ,水分の抽出:原料中に含まれる残留モノマ,揮発成分および水分などを押出機内で溶融,混練しながら真空下で脱気処理して取り除く必要がある.5原料中の灰分などの異物の除去:重合工程で原料中に混入した図4・1 非かみあい型異方向回転二軸スクリュー押出機灰分,触媒などの異物を造粒の際に取り除く必要がある.
