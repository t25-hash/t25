# 4・2・2 混練機の種類

> **章**: 第4章 プラスチック加工機械
> **節**: 4・2 混練機

混練機(mixer)はバッチ式と連続式に大別される。バッチ式はあらかじめ計量された樹脂、フィラー、添加剤を所望の割合で溶融、混練し、次工程の押出機やロール機に供給して造粒する方式である。今日、カーボンブラック、酸化チタンなどのマスタバッチの製造に利用されている。連続式は単軸スクリュー式、二軸スクリュー式および多軸スクリュー式に大別される。歴史的に単軸スクリュー押出機(single screw extruder)が古くから使用されている。最近では溶融、混練性能を向上させたミキシング型スクリューが使用されている。一方、二軸スクリュー押出機(twin screw extruder)は生産性、混練性能において単軸スクリュー押出機よりも数段優れていることから、今日プラスチックのコンパウンディング工程に主要装置として使用されている。この二軸スクリュー押出機はスクリューの回転方向、かみあい形式などにより分類される。これらを大別すると、かみあい型と非かみあい型、さらに同方向回転と異方向回転になる。工業的には、非かみあい型異方向回転二軸スクリュー押出機(non-intermeshed counter-rotation twin screwextruder)、かみあい型同方向回転二軸スクリュー押出機(inter-meshed co-rotation twin screw extruder)が広く使用されている。非かみあい型異方向回転二軸スクリュー押出機はスクリューの/(スクリュー長さと直径との比)が比較的小さく(/=5~10)、かつスクリュー軸は両端で支持されているので高速で回転可能な構造である。また単軸押出機や歯車ポンプなどと組合わせて用いられる。ポリエチレン(PE)、ポリプロピレン(PP)樹脂などの大容量造粒機として幅広く使用されている。図4・1,4・2に非かみあい型異方向回転二軸スクリュー押出機とそのスクリューを示す。かみあい型同方向回転二軸スクリュー押出機は一般にスクリュー/が大きい(/=40~60)。このためフレキシビリティに富んでおり、複合剤などを含むプラスチック材料の混練、ポリマブレンド、脱気、リアクティブプロセッシングなどに適している。図4・3,4・4にかみあい型同方向回転二軸スクリュー押出機とそのスクリューを示す。これらの二軸スクリュー押出機は、スクリューとシリンダとが多くの押出成形目的によって自在に組み替えられるように、セグメントに分割されているものが多い。
