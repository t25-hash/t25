# 2・2・11 終わりに

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

室内空気保全技術は，現状では，いわゆる決定打といえる絶対的
な技術が確立されているわけではない．しかしながら最近，多分野
の研究者が精力的に研究開発を進めているために，かなりの技術的
な知識が蓄積されつつある．
報告されつつあり，その効果については疫学的な検証も必要となっ
てきている．現実的にはすべての有害物質，不快物質をすべて除去
することは大変難しい．このため，各空気浄化技術を組合わせた浄
化装置の開発，省エネルギーに配慮した常時換気設備の開発等が望
まれる．
従来，公害対策の延長線上にあった空気浄化だが，安全性に加え
て快適性（アメニティ）を求める，いわば空気質を高めた室内空間
を志向する現代社会では，さらなる社会的要望をかなえるためにも
各種技術における今後の進展を期待したい．
