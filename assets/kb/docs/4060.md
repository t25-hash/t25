# 4・2・3 造粒装置(pelletizing equipment)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・2 混練機

プラスチックの造粒を効果的に行うには、原料の種類、性質、水分、揮発分、粒度および添加剤の種類、性質、処理量など目的にあった混練機と造粒装置を用いなければならない。造粒方法として、カッティング時のプラスチックが溶融状である場合をホットカット図4・2 非かみあい型異方向回転二軸スクリュー法といい、冷却固化している場合をコールドカット法という。さらにカッティング方式により分類すると、前者は水中カット法と、空中ホットカット法があり、後者にはストランドカット法とシートカット法がある。a. 水中カット法(underwater cutting method)図4・5に示すように、押出機先端に設置したメルトギアポンプによりスクリーンチェンジャを通して多孔ダイより溶融樹脂を温水中にひも状で押出す。これをダイ表面で回転するカッタ刃で切断してペレット化し、ただちに温水で輸送冷却する。次に脱水フィルタおよび遠心乾燥機で脱水乾燥を行い、さらに振動ふるいで不良ペレットを分離して造粒する。本カッティング法は温水中でカッティングするため、ペレットの融着がなく、したがってダイ孔の配列間隔を小さくできると同時に、押出速度を速くしてカッタ回転数を高速にできる利点図4・5 水中カット装置図4・6空中ホットカット法カッタ構造1―67がある。しかし、ダイ表面が温水中にさらされているためダイ表面温度は下がり、ダイ孔内で溶融樹脂が固化し目づまりを生じやすい。このためダイは表面近くまで十分加熱できる構造となっている。また、樹脂圧力による弾性変形を最小とする設計が必要である。また、ダイ表面にはカッタ刃との接触摩耗を防止するため超硬合金などを埋め込んである。本方式によってPE,PP、アクリロニトリルブタジェンスチレン(ABS)、ポリスチレン(PS)などほ図4・4 かみあい型同方向回転二軸スクリュー図4・3 かみあい型同方向回転二軸スクリュー押出機1―68産業機械・装置図4・7 ストランドカット装置とんどのプラスチックが処理可能である。処理能力も現在70000kg/h程度までの大容量が生産可能である。ペレット形状は球状である。b. 空中ホットカット法(hot cutting method)図4・6に示すように、押出機先端に取付けた多孔ダイから押出された溶融樹脂を空中でカッティングし、循環箱壁面に噴出している温水で切断ペレットを冷却した後、スルースを介して遠心乾燥機に送り込み脱水乾燥して造粒する方式である。ダイに対するカッタ回転駆動軸の位置により、図4・6のようにダイの中心にカッタ回転駆動軸を設置したセンタホットカット方式と、カッタ回転駆動軸が多孔ダイの外側にあるサイドホットカット方式がある。センタホットカット方式は小型化できることとペレットの融着を少なくできる利点がある。サイドカット方式はダイ表面全面に孔を設けることができ処理能力を大きくとれるが、カッタ装置は大きくなる。本方式は水中カット方式に比べ、ダイ表面が空気中にあるのでダイの目づまりがなく、そのためダイ圧力による変形も小さく、ダイの圧力も小さくできる。本方式で処理できるプラスチックはPE,PP,ABS、ポリ塩化ビニル(PVC)、PS、ポリメチルメタアクリレート(PMMA)などであり、生産能力は7000kg/h程度である。ペレット形状は錠剤状、凸レンズ状となる。c. ストランドカット方式(strand cutting method)押出機先端または歯車ポンプ出口に取付けられた多孔ダイから溶融樹脂をひも状で押出し、これを連続して水槽中で冷却し、エアワイパで水切りした後ストランドカッタで切断して造粒する。ストランドカッタ装置は、回転している2本のロールで冷却したストランドを挟み込んで引取りながら回転刃で切断する構造となっている。図4・7に押出機方式によるストランドカット装置を示す。本方式では溶融樹脂を一定速度で押出す必要がある。ほとんどのプラスチックを処理することができる。処理能力は7000kg/h程度であり、多品種少量生産に適している。ペレット形状は円筒状である。また、ポリアミド(PA)、ポリエチレンテレフタレート(PET)などの低粘度樹脂は多孔ダイから押出されたひも状の樹脂を傾斜した水槽内の上流から下流へ冷却水とともに搬送し、下流に設けられた回転刃でストランドを切断して造粒する水中ストランドカット法(under-water-strand cutting method)が用いられる。
