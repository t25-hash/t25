# 2・2・2 社会的背景

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

空気清浄に対する社会的関心・要請は，いわゆる公害問題から始
まったといえる．工業由来のばい煙，硫黄酸化物，窒素酸化物等の
大気汚染から始まり，自動車等の移動発生源からの窒素酸化物，硫
黄酸化物，粒子状物質の大気汚染へと社会的関心が移った．行政と
してはこのような社会的要請にこたえるべく，「大気汚染防止法
（Air Pollution Control Law）」を制定して，工業由来の固定発生
源，自動車等の移動発生源に対する法的整備を行い，大気汚染の低
減に努めている．
大気汚染に対する社会的関心が高まるにつれて，室内環境に対す
る社会的関心も高まってきたと思われる．最初，都市部における住
宅の密集に伴い，いわゆる悪臭に対する苦情が増加し，このような
社会的要請に応じるために，「悪臭防止法」が制定された．次に，
冷暖房における省エネルギー化を目指した住宅の高気密化による室
内空気質の低下について，関心が払われるようになった．一例とし
て，局所暖房機器による室内空気汚染が挙げられる．さらに，微量
の化学物質による人体への悪影響が懸念されるようになり，化学物
酸化触媒としての金の可能性を見いだしている．金触媒は室温程度
質過敏症候群（いわゆるシックハウス症候群）が広く知られるよう
における一酸化炭素の酸化除去に優れている．
になった．これらの対策として，厚生労働省，国土交通省，経済産
触媒燃焼を効率よく行うためには，除去対象となるガスの成分，
業省が協力して，住宅等の居住空間でのシックハウス原因物質の濃
濃度，処理空気量に合わせて，最適な触媒成分，触媒構造体の選定
度基準，建材等の規格制定，住宅建築時の24時間換気方法のガイ
を行うことが重要である．
ドライン制定等の法整備等を行っている．最近では，各種アレルゲ
また，各種家庭用電化製品に触媒燃焼機能を組込むことにより，
ン，雑菌，ウイルス等へも関心が広がっているようである．
高機能化，快適性向上を果たした生活関連機器が市販されている．
これら上述の気運をビジネスチャンスとしてとらえた各業界が，
例として，脱臭機能付き生ごみ処理器，脱臭機能付き洗浄トイレ便
建材，換気システム，空気清浄機等に関する研究開発を積極的に行
座，自己浄化機能付き調理器，脱臭機能付きこたつ等がある．
い，次々と市場に新製品を導入している．
