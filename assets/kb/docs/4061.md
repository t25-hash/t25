# 4・3・1 はじめに

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形,発泡成形を含む)

加工対象材料の種類,形態によって成形加工機に求められる特性が異なる.成形材料は,熱可塑性樹脂と,熱硬化性樹脂の二つに分けられ,その形態には,ペレット,グラニュール,パウダ,フレー図4・8 成形法の概要ク,粘土状,液状などがある.成形法には通常の射出成形,2材成形(サンドイッチ成形),インサート成形(アウトサート成形),射出圧縮成形,超高速射出充てん成形,ガスアシスト成形,発泡成形(マイクロセルラ成形),組立成形,反応成形などがある.図4・8にその概要を示す.成形材料の特性,形態と成形法から考えられる特性を整理し,それぞれの特性を満足することができる機能,性能などに関して考えてみる.
