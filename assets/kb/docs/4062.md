# 4・3・2 射出成形の概要

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

ここでは熱可塑性樹脂の射出成形(injection molding)に関して眺めてみる。材料の形態は、通常はペレットである。この材料が可塑化機構の中で、加熱シリンダからの熱により可塑化溶融する(溶かす)。この工程ではより一様に溶融し混練されているかが、次の射出充てんを良好に実行することができるかにかかっている(溶融・混練)。さらに、計量値の再現性に優れることが重要である(溶融・混練・計量)。図4・9射出成形のプロセス次に、しっかり閉じられている金型に、射出充てんする。一般に製品部に関しては、その形状精度、表面粗さなどは十分に配慮されたものとなっているが、スプル、ランナに関しては製品部ほどの配慮がなされない。本来適用する樹脂の流動特性を十分考慮し、用い1―69る成形機の性能仕様をもとにそれぞれの寸法形状を決定することが要件である。また、金型は熱交換器であり温度の安定化が大きな要点であるので、そのための配慮がなされていなければならない(固める)。このように配慮された金型への射出充てんが行われる(流1―70産業機械・装置す)。このときの制御要点は、スクリューあるいはプランジャの移動により押出される樹脂量が決定されることと、樹脂の圧縮性を補償するための圧力制御が容易に決定できることである。圧力の決定には用いる樹脂の特性と、製品形状とゲート位置により製品部の樹脂圧力がどのように分布するかの見極めが重要である(固める)。簡単にまとめると、「溶融・混練・計量」「射出流動」「圧縮・保圧」「離型・脱型」のプロセスを所定の条件により実施することが射出成形ということができる。これを実現するシステムを射出成形機と呼ぶ。より詳細に示すと、次のようになる。図4・9からわかることは、まず成形のアイテムが提案されたとき、どのような観点でどのようにプラスチックを取扱うかに関しての議論から、成形法を取りまとめなければその先へ行かないということである。その議論のとき役に立つのはプラスチック材料の加工物性の値である。また、類似製品や同一成形材料の成形データ、さらには流動解析によるシミュレーションのデータである。成形法が決定し、その方法でどのような条件になるかをさらに具体化し、この結果に基づき金型の各部の仕様、用いる成形機の仕様を具体的に検討する。これらの設備が調達できたらまず検討した条件での成形を行い、製品の出来具合を見て細かいチューニングを行う。目標とした製品品質ができたところで、量産に向けて管理のためのデータを確認し、材料受入れまで含めてあらゆるデータを取りそろえて生産準備作業を終わる。このフローの重要なポイントは、成形法と成形条件の検討である。この2項目が成形機や金型の機能・性能・仕様を決定している。したがってこの部分を不十分にして進めると生産段階になって大きな問題を起こすことになる。成形条件といういい方でもの作り条件を表現することがある。しかし、詳細を見ると、機器の運転条件があり、この条件によりシステムがどのように動いたかの計測データがあり、さらにどのような樹脂状態となったかの計測データがあり、これに成形品の品質特性データがあることが必要となる。一般的には機器の運転条件だけをとらえて成形条件とすることがあるが、それが後々問題を引き起こす。この段階で重要なことは、成形プロセス条件をより確実にそろえるときには、成形プロセスをより確実に計測することである。生産時はその計測データを基準にして設定値を合わせ込むことが必要である。成形条件まで踏み込んで考えなければならないかといえば、金型に対する圧力負荷、熱負荷が金型を具体化するときにぜひ必要なデータとなる。これらの負荷により溶融樹脂の充てん時に局部的なひずみを生じ、品質に大きな影響をもたらすことがあり、これに対し事前の対策を必要とする。また成形条件の幅を確保するためにもこの対策が必要である。
