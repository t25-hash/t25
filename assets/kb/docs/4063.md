# 4・3・3 それぞれの機能

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

溶融プロセスは、加熱シリンダからの熱による方法と、スクリュー回転により成形材料が移動していく時点での摩擦・せん断発熱による方法があるが、射出成形機においてはスクリュー回転が休止するため、摩擦・せん断発熱による方法では成形条件によっては同じ条件での結果が異なることになる。より詳細にいうと1ショット内での溶融状態の差異が大きくなる。したがって、同じ条件で同じ結果を得なければならないとすると加熱シリンダからの熱で溶融することが必要条件となる。これを満足するためのそれぞれのパーツ設計においては、スクリュー溝内の樹脂の状態分布をペレットから半溶融、完全溶融、設定温度状態がどのように分布されていなければならないかを想定し、さらにそのために必要条件が何かを考慮しなければならない。特にフィード部に関しては十分な検討が必要となる。スクリュー溝に樹脂が満杯となったとき、スクリュー谷部のペレットが容易に外周に出てくることはない。溶融がある程度進行しなければこの樹脂が移動することがないため材料温度に大きな差異が生じることになる。具体的には結晶性樹脂を可塑化したとき、加熱シリンダの直径方向の温度分布を計測すると、温度が細かく上下する挙動が見られた。これは成形材料の熱伝導率は小さく、成形サイクルタイムという短い時間の中で完全に一様とならないことによる。そのため、ペレットの加熱段階から温度差が極力小さい状態で温度を高めなければならない。これを実現するためにはスクリュー回転を停止しないこと、ペレットが自由に移動できることが必要条件となる。さらに、スクリューストロークは加熱シリンダの温度制御ゾーンとスクリューデザインの整合性を乱し、所望の樹脂状態分布を不安定な状態とするため、このようなことがない使用方法を取ることもたいせつである。不良品を作らない、不良を出さないというもの作りの実現のためには、再現性をさらに細分化して、プロセスの挙動の再現性、プロセスの状態の再現性、成形材料の再現性について改めて眺めなおし、管理しなければならない。従来は挙動の再現性のみを重視してきたが金型を含めた状態の再現性、成形材料の再現性に関しても十分な監視の目を向けなければならない。特に成形材料をチェックなしで使うほど不良に関して問題となることはない。このようにして、良好な可塑化溶融状態を得ることができたら、その樹脂の計量値をきちんと制御することが必要となる。射出成形における計量は、スクリューあるいはプランジャの移動量を確認するいわゆる容積法であり、樹脂密度が変化すると再現性は得られない。樹脂の密度はスクリュー背圧の制御と溶融樹脂温度で決定される。スクリューを含めた力関係を確認すると、スクリューの回転に伴う後退力が変化すると計量樹脂の樹脂圧力の変化となることがわかる。スクリューの回転に伴う後退力が変化するときは、スクリューの樹脂状態分布が変化したときであり、樹脂の温度分布の変化となっていることが考えられる。また、スクリューの後退力や前進力の計測方法に問題があることがある。油圧式の場合は、スクリュー径×(3.0~4.4)の射出シリンダ径に油圧油を供給し射出を行い、流出油を圧力制御して背圧としている。圧力制御能力により最小背圧が決定する。設定状態に対して、遅れ、応答性、安定性などの機械による特性があるので、成形条件は図4・10のように計測データを加えたもの図4・10 成形条件の概念とすることが必要になる。しかしながら電気式ではスクリューの負荷を荷重センサで確認しているため、背圧を5MPa程度とするとその荷重は用いている荷重センサの1.5~3.2%と極めて小さい。そのため計測エラーと正しい値との判別がつけにくく、正しい状態で制御されているか不明である。制御要件である射出圧力、背圧の計測値と制御に用いるときの値との差異など精度面での課題に答えるように配慮したうえでのシステムの組上げ方が必要である。
