# 4・3・4 成形材料の形態への対応

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

ホッパからの材料供給は通常はペレットを自重落下させてスクリ
ューへくい込ませる方式としている．そのため落下後部の形状と温
度に関しては落下を阻害しないような配慮が必要である．温度に関
しては成形材料の熱変形温度以下とする．これは成形材料のホッパ
重量が負荷されても押しつぶされてスクリューへのくい込み不良と
なることを避けるためである．標準的には成形材料の乾燥温度とす
ボスなどを成形するものである．

難である．容易にブリッジを構成し落下しなくなる．したがって，

る．
粘土状の材料は，スクリューへくい込ませるために一定に圧力を
加えることが必要となる．さらに成形材料の密度変化が少ないた
め，リードと溝深さを勘案することが必要となる．そして多くは熱
硬化性樹脂が多いため，加熱シリンダの設定温度と樹脂温度の乖離
を少なく保つことが必要となる．
液状材料は，多くの場合2液型とすることが多い．この2液を十
分に混練しなければ目的とした物性の確保が困難となる．そのため
にはスクリュー直前にダイナミックミキサを備え，混練した材料を
スクリューに移行する．2液を混練してからの容積は可能な限り小
さくすることが必要条件である．加熱シリンダとスクリューのすき
まから樹脂が漏れ出さないようにシールが必要となる．
以上のように，材料の形態によってホッパ周りの変更改善が必要
であり，スクリューの形状デザインを改める必要がある．
