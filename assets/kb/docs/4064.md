# 4・3・4 成形材料の形態への対応

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

ホッパからの材料供給は通常はペレットを自重落下させてスクリューへくい込ませる方式としている。そのため落下後部の形状と温度に関しては落下を阻害しないような配慮が必要である。温度に関しては成形材料の熱変形温度以下とする。これは成形材料のホッパ重量が負荷されても押しつぶされてスクリューへのくい込み不良となることを避けるためである。標準的には成形材料の乾燥温度とすることでよい。ペレットより小さな粒状のグラニュールの供給に関しては、ほぼペレットと同一であるがペレットより細粒であり、空間が少なくなっているため供給制限装置をホッパ部に取付け、スクリューに過充てんとならないような配慮が必要である。さらに理想的には、溝深さ、リードの調整が必要となる。パウダは、ペレットに比較してスクリュー回転時の力関係が異なり、これを調整することを含めてスクリューのリードを変えることが必要となる。あわせて、スクリュー溝の中で材料が容易には動かないため、外周側の材料と谷径側の材料の状態の差異が大きくなり、溝の深さも変える必要がある。ホッパ重量がパウダのかさ密度を変えるため、ホッパレベルの管理維持機構が必要である。フレーク状の材料は、自重による落下では安定したくい込みは困難である。容易にブリッジを構成し落下しなくなる。したがって、ホッパ部にはブリッジ防止の細工を施す。さらに安定した供給が実現できる供給装置を付帯する。スクリューデザインはリードと溝深さを勘案し、安定した可塑化計量特性を実現することが必要である。粘土状の材料は、スクリューへくい込ませるために一定に圧力を加えることが必要となる。さらに成形材料の密度変化が少ないため、リードと溝深さを勘案することが必要となる。そして多くは熱硬化性樹脂が多いため、加熱シリンダの設定温度と樹脂温度の乖離を少なく保つことが必要となる。液状材料は、多くの場合2液型とすることが多い。この2液を十分に混練しなければ目的とした物性の確保が困難となる。そのためにはスクリュー直前にダイナミックミキサを備え、混練した材料をスクリューに移行する。2液を混練してからの容積は可能な限り小さくすることが必要条件である。加熱シリンダとスクリューのすきまから樹脂が漏れ出さないようにシールが必要となる。以上のように、材料の形態によってホッパ周りの変更改善が必要であり、スクリューの形状デザインを改める必要がある。
