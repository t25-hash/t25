# 2・2・7 低温プラズマ分解法

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

放電を利用する方法ではプラズマが形成されるが，熱プラズマを
方法がある（6･2･2項参照)．熱プラズマはアーク放電等を用いる
が，電子温度，イオン温度，分子温度のいずれもが高い温度であ
る．これに対して低温プラズマ（非平衡プラズマともいう）の場合
には，発生する電子がもつエネルギーである電子温度のみが高くな
る．このため，低温プラズマでは処理ガスの温度が上昇しないた
め，熱プラズマと比較して室内環境浄化技術として適している．
この低温プラズマを発生させる方法として，パルスストリーマ放
電，無声放電，部分放電，沿面放電，コロナ放電等がある ．
低温プラズマにより発生した高速電子は，除去対象ガスの分解反
応に直接関与する場合もあるが，この高速電子が空気中の酸素，水
蒸気と反応して各種の活性酸素種やヒドロキシルラジカル種を生成
し，これらの反応活性種が除去対象ガスと反応することにより分解
除去されると考えられている．
低温プラズマ法の特徴としては，種々の除去対象ガスに適用でき
高い分解能力を示すが，生成物の選択性が低く，場合によっては副
生成物による二次汚染のおそれがある．また，プラズマ発生のため
の電力効率を向上させる必要がある．このため，酸化触媒との組合
せによる完全分解（二酸化炭素，水蒸気等に変換する）への生成物
選択性の向上，放電方法の改良が検討されている．
一例として，尾形 は，酸化触媒や吸着剤を低温プラズマ反
応器内に設置することにより，副生成物発生の抑制や電力効率が向
上することを報告している．香川らは，ストリーマ放電の発生装置
を改良 し，後段に酸化触媒を設置することで除去能力が向上す
ることを報告し，この浄化技術を取り入れた家庭用空気清浄機が市
販されている ．
