# 4・3・5 成形法への対応

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

a. 2色2材成形(two color molding、図4・11)模様柄が全く再現性のないマーブル成形、マクロ的に見て再現性がある混色成1―71形(サンドイッチ成形を含む)、ミクロ的に見ても再現性があり品質保証ができる2色2材成形の3種に分けて考えることができる。マーブル成形は、一つの射出装置、一つの金型を用い、2種以上の可塑化溶融特性に差異がある材料を投入することにより模様を得ることができる。混色成形は、二つの射出装置と一つの金型により、それぞれの射出装置のプロセスコントロールとシステムコントロールにより、ほぼ同一の品質を確保することができる。2色2材成形は、二つ以上の射出装置と二つ以上の金型を用い、順次射出充てんすることにより、一つの製品とすることができる。b. インサート成形(insert molding)製品の主たる部分は樹脂である。金属などの小さな部品を埋め込んで一つの製品とする。ある種の組立成形である。この方法に対応する表現として、アウトサート成形がある。これは比較的大きな金属部品に組付け部品用のボスなどを成形するものである。インサート成形では樹脂の収縮によりインサート部品を締付ける作用があるため、脱落防止は比較的容易である。これに対し、アウトサート成形では収縮が脱落を助長することになるため、多くの成形ポイントを結合して製品全体で支えることを考えなければならない。c. 射出圧縮成形(injection compression molding)可能な限り金型に忠実な製品を得ようとするときや、圧力分布を可能な限り均一にして良好な製品を得ようとするときなどに使われる。圧縮代の作り方と圧縮の方法により、ローリンクス法、マイクロモールド法、IC法などに分かれる。・ローリンクス法。.....低圧で金型を閉鎖し、射出充てん圧により金型がわずかに開く。充てん後強力に金型を閉じることにより圧縮を行う。・マイクロモールド法。.....圧縮用のコアが油圧シリンダにより駆動される金型を強固に閉じ、射出充てんの後、圧縮コアを駆動して圧縮するものである。・IC法。.....金型を圧縮代だけ開いておき、ここに射出充てんする。充てん後金型を大きな力で閉鎖することにより圧縮を行う。これらの方法と金型の加熱冷却サイクルを重ね合わせた方法により形状、大きさに関する自由度が増大する。充てん中は加熱状態とし、冷却しながら圧縮する。手法が複雑化されればされるほど金型の作りは複雑なものとなる。一つめはバリ対策であり、二つめは金型温度対策である。温度むらが存在すると成形品の品質に影響するため、均一な温度で成形されなければならない。特にレンズは両面を所定の状態にすることが必要であり、その状態と位置関係が確保されていなければならない。そのための金型の作り込み精度は厳しく管理されており、用いる成形機の動的な精度は十分管理されたものでなければならない。d. 超高速射出充てん成形(ultrahigh-speed injection molding)多くの成形材料では流動時のせん断速度が大きくなると溶融粘度が小さくなる。この溶融粘度の低下と充てん時間が短くなることによる冷却スキン層の薄層化によって、流動距離がより長くなり、より薄肉の成形品を製作することができる。高速射出を行う成形機の特性には、応答性に優れること、慣性エネルギーの処理ができること、速度を補償するパワーがあること、設定制御する領域が広いため非線形の処理ができるシステムであること、コントローラの処理速度が十分に高速であることなどが要求される。超高速充てん成形の金型は、ランナの検討から行わなければならない。ランナはキャビティ流動の助走部分であり、キャビティ流動時に想定しているせん断速度を容易に実現できる。しかも圧力損失があるレベル以下(成形機の性能仕様からキャビティ流動圧力損失を十分まかなえるレベルを差し引いた範囲より小さい値とする)になるサイズとすることが必要である。長いランナの場合、徐々にせん断速度を増加させるスプル側を太い形状とすることも考える。図4・11 2材成形(混色・サンドイッチ成形の射出装置)1―72産業機械・装置キャビティは基本的にはゲート側から順次充てんされることを考える。流動途中の薄肉部は、流動するときは周りと同一の厚さとし、薄肉部が流動完了したら薄肉部を構成するコアを駆動し所定の形状にする。e. ガスアシスト成形(gas-assisted molding)成形品の軽量化を目指すことが可能である。成形品の部材の曲げ強度は板の表面が効果的に作用し、中心部はほとんど機能しないため、肉厚中心部に空洞があってもよい。肉厚部の中央部に大きな空洞を構成することにより、軽量化と冷却時間の短縮を実現する。成形システムは、比較的充てん時間を短くすることができ、ガスを充てんするノズルがあり、ガス圧力のコントロールができることが必要条件である。また、成形品のデザインに関しては、ガスを必要とする部分に行き渡らせるためには、製品にガスが流れ得るガスランナを配置しなければならない。f. 発泡成形(expansion molding、微細気泡発泡成形)プラスチック中に気泡が混在しており、密度の低下が見られる成形体を得ることができるものである。気泡の作り方は、化学発泡剤を熱分解してガスを発生させ混入させるものと、不活性ガスを所定の圧力状態(超臨界状態を含む)で混入させるものがある。成形システムで特徴的なことは、加熱シリンダ内では気泡を小さくしておかなければならず、そのために樹脂圧力を高くしておかなければならない。ノズルが開放状態では圧力の維持ができないため、バルブノズルが必要となる。計量中はスクリュー背圧をガス圧力より高めに設定するが、計量完了で背圧を開放すると樹脂圧力でスクリューが後退してしまうので、これを防止するためにスクリュー位置をホールドする。射出充てん速度は通常の成形よりは短時間で完了することが望ましい。その他ガス、臨界ガスの充てん装置が必要になる場合は、加熱シリンダ、スクリューの長さを標準のものより長くし、溶融材料による樹脂シールを構成し、ガス充てん口よりホッパ側にガスが逃げないようにすることが必要である。気泡入りの材料を射出充てんすると、流動先端の樹脂圧力が低く気泡が拡大することと、流動がファウンテンフローとなり流動先端がつねに新しい気泡を含んだ状態となり、金型壁面に接するときに気泡を押しつぶすために、表面の平滑さに課題が残る。この対策として、充てん速度を可能な範囲で大きくすること、あるいは流動先端部にある圧力を与えるようにカウンタプレッシャを負荷することがある(ガス量をロスすることもなく、表面もきれいにできる)。g. 組立成形製品の機能の確保、組立工程の省略などを考慮して金型内での組立を行う。インサート成形、2材成形などでこの範ちゅうに入るものもある。成形システムは多くは金型のノウハウにより実現可能である。製品特性と用いる成形材料によりその構成方法が変わり、ゲート位置などの制約もある。h. 反応射出成形(reaction injection molding)熱硬化性樹脂の成形を指す。この成形では基本的にはスクリューヘッドに樹脂を滞留させない方法を採用している。充てん後のキャビティへの圧縮・保圧時の調圧はスクリューのリードに沿った樹脂の逆流時の移動抵抗にバランスした圧力となる。したがって、可塑化時の樹脂の状態分布の構成の仕方が要点となる。また、金型におけるバリの発生は材料の溶融粘度が小さいため、熱可塑性樹脂と同程度の金型のすきま管理では大きな課題が残る。射出成形機の特性からダイプレートのたわみはかなり大きく許容されているため、金型のたわみによりバリが発生する。極めてバリの少ない良好な成形を行うときは専用機を用いることが必要である。i. インモールドコーティング(in-mold coating、図4・12)成形後、離型前に金型をごくわずか開き、塗料を注入する。その後再び金型を閉鎖することにより、製品表面に塗料の膜を均一に製成する。図4・12 インモールドコートサイクルチャート(b) 射出成形(c) 成形品取出し(a) フィルム位置決め図4・13 インモールド転写j. インモールド転写(図4・13)印刷フィルムを金型の中に供給し、ここに樹脂を射出充てんする。充てんした樹脂と印刷のいちばん外の層(接着層)の接合によりフィルムに印刷した模様がフィルムから樹脂に付着する。k. 金型温度冷熱サイクル成形積極的な制御により、あるいは充てん時に射出された樹脂の熱により金型表面温度を高くし、金型表面の転写を良好にする成形法である。l. 加振成形金型、あるいは溶融樹脂を加振し、溶融樹脂の粘度を低減したり、金型と樹脂の界面のエアを排除し、金型の表面の転写を良好にする。
