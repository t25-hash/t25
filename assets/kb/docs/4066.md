# 4・3・6 成形不良対策

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形,発泡成形を含む)

成形時の品質に関して議論するとき,射出成形は極めて単純に再現性の追求により成り立っていることを認識し,この中でさらに厳密に考え,プロセス挙動,プロセス状態および成形材料特性の再現の三つに分けて考えてみる.プロセス挙動とは,成形機の動作とその制御量の構成を指す.プロセス状態とは,成形機の状態,具体的には加熱シリンダや金型の温度制御状態を指す.成形材料の特性の再現性とは,分子量分布や添加剤,ペレットサイズの変動などを指す.これら三つの要件がよりよく再現しているときに良好な品質が得られる.品質の限界と三つの再現性の限界幅を関連づけて品質管理を行うことがたいせつである.
