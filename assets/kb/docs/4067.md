# 4・3・7 射出成形機の概要

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

射出成形機は、射出機構、型締機構、駆動制御要素、電気制御要素、ベッド架台の五つの機構・要素から成り立っている。インラインスクリュー(in-line screw)式射出成形機とスクリュープリプラ(screw preplasticizer)式射出成形機の二つの方式が現在市販されている。射出機構は可塑化、混練、計量、射出の機能をもっている。金型との接合部を構成するノズルは温度が急激なこう配をもっていなければならず、そのための機構と断熱処理が必要となっている。インラインスクリュー方式は、可塑化、混練、計量、射出のすべての工程を1本のスクリューで行い、スクリュープリプラ方式では可塑化、混練、計量と射出を分離し、別な機構で行う。型締機構は、金型の開閉、金型保護、強力型締め、製品突出しの機能が必要である。型開閉時は無理なくこれを行わなければならず、必然的に動的な精度が必要となる。駆動制御要素は、所定の性能を発揮し、さらに運転効率が高くなければならない。最近では動作のオーバラップがサイクルタイムの低減に効果的であり、これを実現するための駆動方式がとられる。電気制御要素は、動作を組立てるための仕様が第一に必要であり、動作状態のモニタができ、これを統計的に処理できることが必要事項である。ベッド架台は、剛性があり、射出機構と型締機構を精度良く搭載し、駆動制御要素、電気制御要素機器を格納する。形状は中空の箱状であり、軽量・高剛性が条件となる。a. 射出機構インラインスクリュー式射出機構の場合は、基本的な構成要素は加熱シリンダ、シリンダヘッド、ノズル、スクリュー、スクリューヘッド、スクリュー回転駆動ユニット、スクリュー前進・後退駆動ユニット、ノズルシフトユニットからなる。インラインスクリュー式射出機構ではスクリューが可塑化計量と射出を行うため、スクリューヘッドにチェックリングを備えている。スクリュープリプラ式射出機構は、可塑化をスクリューで行い、射出をプランジャで行うもので、射出時のスクリュー室への逆流を防止するメカニズムを備えている。加熱シリンダ温度制御はノズルを含め4~8点のコントロールゾーンをもっている。射出速度は3~6段、保圧は3~4段の設定を可能としている。油圧駆動制御方式では速度は流量制御であり、電磁比例流量制御弁あるいはサーボ弁の制御、保圧は圧力制御であり、電磁比例圧力制御弁あるいはサーボ弁の制御である。サーボ弁を用いるクローズドループ制御ではスクリューの位置を時間微分した速度、圧力センサによる圧力がフィードバックされる。電気駆動制御方式では速度はモータの回転量の制御であり、圧力はモータのトルクの制御となる。それぞれの制御時のフィードバックデータはモータのエンコーダ、スクリュー位置センサの微分値が速度データであり、スクリュー後部に配置されているロードセルのデータが圧力データとなる。また速度領域の圧力リミッタは1~3段である。スクリュー回転トルク、スクリュー前進力は極めて大きいため、100%の負荷になったときはたわみも大きくなることが予測される1―73ため、負荷が大きくかかる部分の設計には十分な注意が必要となる。機械間の調整が十分になされないと同一の成形条件での成形品質に差異が生じることになる。射出速度、射出圧力、スクリュー回転数などを差異なく調整することは比較的容易であるが、成形材料の可塑化・溶融・混練状態の差異などをなくすことは極めて困難である。加熱シリンダ、スクリューなどの機械部品はある寸法公差をもっている。そのためスクリューサイズ、材料ペレットの大きさなどにより材料の挙動が微妙に変わり、その結果溶融状態の差異につながることがありうる。一般的に、加熱シリンダ設定温度と計量された樹脂温度は異なり、樹脂の種類、運転条件などによって樹脂温度が高い場合もあれば、低い場合もある。さらに成形時の樹脂圧力は樹脂温度を変えるため、樹脂温度のより正確な値を把握することは困難である。また、スクリュー回転数に応じた押出量が確保できないときは材料に過大な力が負荷され、異常な状態となることが予測される。具体的には摩擦発熱が極めて大きくなることによる樹脂温度、樹脂状態の異常である。それを防ぐには材料の種類に応じたスクリューデザインとヒータ容量が必要となる。また、結晶性樹脂のエンタルピー変化量が非結晶性樹脂のそれより大きいことが明らかであり、その対応が求められる。b. 型締機構大きな型締力を確保するために、力の拡大機構を用いている。拡大機構として、液圧式と機械式に大別される。液圧式では大口径のシリンダを用いて大きな力を確保している。機械式ではねじ機構とトグル機構がある。ねじ機構では回転力をねじの傾斜面とねじ径により拡大するものである。トグル機構はクロスヘッドの移動量に対してリンクの伸び量が極めて小さいことによる力の拡大を利用するものである。機械式の拡大機構は液圧式でも電気式でも採用が可能である。直圧方式はシリンダに所定の圧力の油圧作動油を供給すれば目的とする出力を得ることが可能である。このときタイバシャフトの伸びが計算値どおりとなっているかに関しては詳細には論じられない。しかし、トグル式においては出力の絶対値を知る方法が乏しく、タイバシャフトの伸びを手がかりとしてその概要を知ることができる。したがってトグル出力が効果的にタイバシャフトに負荷されるような機構設計が必要である。ダイプレートのたわみ、トグルサポート部のたわみはタイバシャフトの伸びを小さくするものであり、金型を締める力が小さくなることを示している。さらに、トグルリンクのたわみは直接型締め力の低減となるため注意が必要である。具体的な設計では、たわみ設計を実施することが必要となる。応力設計で破壊しないパーツを作ることができても目的を達成することは困難である。機構の評価要件は、型締力が設定値と実行値に差異がないことが基本であり、さらに環境によるドリフトがないこと、機構精度が十分高く、その寿命が長いことが必要条件となる。c. 駆動制御要素油圧式の場合は、性能仕様が算出される動力をまかなえるポンプモータとなっているかの確認でよい。最近の省エネ型のシステムでは必要動力容量と比較すると明らかに負荷率が大きな値となっていることがある。これは1サイクルの平均動力値がモータ定格値を下回っていればよいためである。電気式の場合は、サーボモータが用いられている。理論動力と負荷計算による動力との差異を十分調整したモータ容量、ドライバ容量でなければならない。d. 電気制御要素設定どおりの挙動・状態を実現するために、駆動制御要素を制御することが主体である。その機能は、まずシーケンスコントロールである。可塑化・計量、射出充てん保圧、型開、エゼクト、型閉の動作の組立が主要な仕事である。制御のディジタル化が進み、計測がより詳細に行われるようになると、微小な差異が明らかになってくる。この差異が認識できないくらいにコントローラのリフレッシュサイクルタイムが小さくないと製作された品質のばらつきは無視できないものとなるだろう。1―74産業機械・装置挙動・状態の再現性が確保されているかを知るために、計測・表示ができ、さらに統計的な手法でまとめることが必要である。この処理機能の実施によりコントローラのリフレッシュサイクルタイムに変動があったり、長くなることは許容できない。ハードウェアはより高速な処理が可能なものはより多く生産されており、ソフトウェアはリフレッシュサイクルタイムが安定して短くなるような方策を考えなければならない。加熱シリンダの設定温度により樹脂温度が制御されるためには、どんな条件でも樹脂温度が設定温度よりわずかに低いことが必要条件である。このことは、温度制御をどのように理想的にしても、スクリューデザインにより条件幅があり、運転条件の制約となることがある。e. ベッド架台機構の精度を長期間確保維持するためには、射出機構、型締機構を載せ、それぞれの機構が稼動したときにもその精度が変動しないことが必要条件となる。これに加えて駆動制御要素、電気制御要素を格納するため、全体の剛性を保持することは困難である。成形材料によっては金型温度が高く、ダイプレート温度が上昇し、ベッド温度に影響することになる。温度が上昇することにより良好な状態維持ができなくなることも生じる。したがって熱・温度に対する十分な配慮をした構造を採用している。
