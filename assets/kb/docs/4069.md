# 4・3・9 成形シミュレーション

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形、発泡成形を含む)

成形のCAEの方法として流動の方程式を組み、これを前提に詳細に解いていき、キャビティ内の流速、流動圧力を求める。前提条件として粘性流体であること、流動中は固化層ができないこと、圧縮性は無視することなどがあり、成形の実情をよりよく反映したものではない。さらに現状の多くは三次元流動ではなく、二次元の流動で圧力を算出するために、厚さを定義するものである。いずれにしても成形を十分に反映したものではなく、状況によっては実成形のデータで多少修正しているため、その成形から大きく離れたときは不具合を生じることになる。特に圧縮性の影響が出やすいもの、固化層の影響が出やすいものに関しては成形機の運転条件予測は不可能である。したがって現在の流動解析システムは実用的に重要なのではなく、考察、検討するために有益である。なぜなら、金型を設計製作し、成形機を準備する時間的なロスタイムを省いた状態で、成形評価しようとするものである。その結果を解釈するときはそのシステムの前提条件まで踏み込んだ中で考察しなければならない。固化層の影響は確実に流動層を薄くしているため、流動圧力損失が極めて大きくなる。成形機の能力にもかかわるが流動が停止することにもなる。したがって、射出成形の流動限界は理論的な製品肉厚より明らかに小さくなることがいえる。特に金型の温度が低いときは限界厚さは大きいといえる。また、流動時間がより短いほうが限界厚さは小さくなることがわかる。この領域をシミュレーションで算出することは困難である。実際に成形したほうがよりわかりやすいし、金型の必要条件がより明らかに出せる。実用可能なもの、適合領域が十分広いものとするためには限界条件を可能な限り取り除いたシステムでなければならない。これができるまでには多くの時間が必要とされることが予測される。むしろこの必要条件が満足されるシステムとするためにはどのようなアプローチが必要かをしっかり調査することがたいせつである。そのうえでより適合範囲が広いシステムを構築することがたいせつである。
