# 2・10・1 貯水池，調整池，逆調整池

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・10 水力発電設備

a.貯水池（reservoir ) 河川の渇水期や必要な時期に必要な
使用水量を補給する目的で，豊水期に貯水する湖や池を貯水池とい
う．
b.調整池（regulating reservoir ) 水力発電所を1日または
短期間内の負荷変化に合わせて運転する目的で，河川流量を調整す
るために設けた池を調整池という．調整池の容量は，渇水期におい
て数時間のせん頭出力を出すことのできる程度のものとすることが
多い．
c.逆調整池（reregulating reservoir ) 上流の調整池式発電
所の出力調整により放流量が変化する場合，この流量を平滑化する
ように発電所下流に設けられた池を逆調整池という．
