# 4・3・10 その他の成形法

> **章**: 第4章 プラスチック加工機械
> **節**: 4・3 射出成形機(反応成形,発泡成形を含む)

a. 振動を付与する・金型加振......流動樹脂が金型に入ってくるとき,振動している金型に接すると,金型の表面に樹脂の界面を作りにくくなり,あたかも固化層を構築しない状況が発生する.結果として転写が極めて良好な製品とすることができる.・高せん断付加成形......成形機のノズル近傍に振動メカニズムを配置し,流出材料にせん断を与え,溶融粘度を低下させ,成形性をさらに良くするものである.1―76産業機械・装置・プッシュプル成形......金型の中に充てんされた樹脂を二つのゲートから出し入れすることによりウェルド部の欠陥をなくす方法である.b. 射出膨張成形狭いキャビティに射出充てんし,その後キャビティを拡大することにより肉厚中央部に空間をもった製品を得ることができる.通常の発泡成形を行ったとき製品表面がなし地になるが,この方法では表面の状態は良好なものとなる.樹脂の中に膨張要素がない場合は表面にひけが観測されることがある.一般には発泡剤入りの樹脂で成形を行うことが多い.c. MIM,CIM,パウダ成形粉体を混入させた材料を成形する.MIM はmetal injection molding であり,CIM はceramicinjection molding の略語である.いずれの場合も金属とセラミックスを粉体とし,樹脂バインダに混入し均一な複合体を作り上げることがグリーン体の成形で,このグリーン体を脱脂・焼結することにより製品とするものである.この技術の要点は粉体の粒度分布である.固体状態のときに単位容積当たりの空間がどのくらいできるかにあり,空間が確実に算出できれば脱脂・焼結の収縮量が求められるため精度的な満足度を得やすい.粉体は完全球形が望ましく,フレーク状は好ましくない.成形機ははん用的な特性をもたせ,特徴的な仕様としてはスクリュー,加熱シリンダなどの樹脂が直接接する部品に耐久性をもたせることである.また,射出率は可能な範囲で大きなものが求められる.d. ボンデッド磁石(プラマグ成形)磁性粉体をPA 等に混入させ,パウダ成形時のグリーン体を成形するときと同様な扱いをする.このとき磁性パウダを積極的に配向させるために金型内に磁性体を設置することもある.金型の磁性は製品取出しのときじゃまなものとなる.そのため磁性を除去することも考慮し,製品を取出した後に改めて着磁を行う.成形機は型締装置は可動,固定のダイプレートに電磁石を非磁性体で囲むように配置する.射出側はスクリュー,加熱シリンダなどの樹脂に直接接する部品に耐久性をもたせることが必要である.さらに射出率は可能な範囲で大きくすることが求められている.e. トランスファ成形熱硬化性樹脂の成形に多く使用される.計量され,円筒状に成形された樹脂(タブレット)を,金型のポットに投入し,金型を閉鎖した後にプランジャを駆動して溶融樹脂をキャビティに押し流す.この方法で,ポットからキャビティまでのランナの容積は完全に材料ロスになり製品コストを押上げることになる.これを防ぐために,ポットをたくさん設けてランナを短くするマルチポットの方法もある.f. チキソ成形マグネシウム合金の成形時に固溶体にした材料を金型のキャビティに射出充てんする方法.2元素系の合金では組成によって固溶体となる温度範囲があり,これに基づく温度の固溶体を作る.ただし,加工物の特性,特に熱伝導率が樹脂より大きいことが安定した状態を確保しにくくしている.より詳細に見ると,合金の中に含まれている不純物が,成形品物性などの品質を阻害している.したがって技術の難しさと得られた品質の課題から大きく広がりにくいものとなっている.
