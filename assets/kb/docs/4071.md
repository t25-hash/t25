# 4・4・1 押出機(extruder)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

押出機とは,プラスチック材料を溶融させながら,混練していくとともに目的とする形状に連続的に賦形するために使用する機械である.対象とするプラスチック材料は,ポリプロピレン(PP),ポリエチレン(PE)などの熱可塑性樹脂が大部分であるが,一部熱硬化性樹脂も含まれる.押出機において,連続的に溶融,混練するために,スクリューが使用されており,このスクリューの本数と構造によって押出機が分類されている.a. 単軸スクリュー押出機(single screw extruder)単軸押出機の一般的構造を図4・18に示す.機械は,プラスチック材料を入れるホッパ,材料を輸送・混練・加圧するスクリュー,材料を外部から加熱・冷却するシリンダ,スクリューを回転駆動させるモータと減速機から構成されている.また,押出機の先端にはダイが取付けられ,成形目的によりその形状,構造を使い分けている.単軸押出機において,材料はホッパに投入される.投入された材料は,回転しているスクリューによって前方に輸送され,徐々にシリンダ側から加熱されて溶融し始める.溶融領域におけるプラスチック材料の溶融形態は,多くの研究者によって提案されており,その中のTadmorのモデルを図4・19に示す.溶融した材料は融体輸送領域において混練,加圧されて,押出機先端に取付けられたダイを通過し成形される.単軸押出機内の材料のこれらの挙動を図4・20に示す.スクリューの最も基本的な形状は,フライトがスクリューの始まりから終端にわたって一定のピッチでらせん状になっている,いわ図4・18 単軸押出機の構造図4・19溶融モデル図4・20 単軸押出機内の材料状態図4・21 各種スクリュー形状図4・22 溝付シリンダの構造図4・23ベント式押出機シリンダ内圧力分布図4・24 二軸押出機の形式ゆるフルフライト型であるが,近年押出量の増大,要求品質の高度化により混練性を向上させる必要性が増大している.このため,溶融領域において,固相と溶融物プールを機械的に分離させ溶融を促進させる,バリアフライト型やマドック型スクリューの採用が,また融体輸送領域においてダルメージ型,クロスソー型などの採用が増えている.図4・21にバリアフライト型やマドック型,ダルメージ型など各種スクリュー形状を示す.押出機の固体輸送能力を上げたり,高分子量のポリエチレンなどの安定押出しを行うため,スクリューの供給領域に溝付シリンダを用いることがある.図4・22に溝付シリンダを示す.これは,シリンダと供給材料との摩擦抵抗の増大を利用して固体輸送能力を向上させるものである.一方,材料に含まれるモノマや巻込み空気などを押出機の途中から除去しながら押出しを行う目的には,ベント式押出機を使用する.ベント式押出機では,溶融した材料を一時的に圧力低下状態にして,ガス状揮発分を真空ポンプにより強制排出させる構造となっている.図4・23にベント式押出機におけるシリンダ内圧分布を示す.b. 二軸スクリュー押出機(twin screw extruder)二軸押出機は,かみあい形成,回転方向形式により数多くの種類がある.b項では,フィルム,シート,異型成形などで使用される,低速かみあい型異方向回転タイプと高速かみあい型二軸押出機について記述1―77図4・25 キャストフィルム製造装置図4・26 エアナイフ成形概念図図4・27 ヒートボルト式自動T ダイする.低速かみあい型異方向回転タイプ二軸押出機は,完全かみあいで,が20前後,スクリュー回転数も40rpm以下である.この押出機は,熱安定性の悪いポリ塩化ビニルの押出しに用いられる.このタイプには,スクリュー軸が平行のものと斜軸(コニカル型押出機と呼ばれている)がある.高速かみあい型二軸押出機は,が30~50で,スクリュー回転数も300rpm程度である.複合剤を含むプラスチック材料の混練,脱気などに適しており,コンパウンド用途として多くの実績がある.近年,コンパウンドでペレットを作りそれを単軸押出機にかける従来型から,コンパウンドの工程を省略し,高速かみあい型二軸押出機の先端にダイを取付け,直接シート等の成形をするものが増えている.図4・24に二軸押出機の形式を示す.
