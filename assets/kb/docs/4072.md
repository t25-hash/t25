# 4・4・2 フィルム成形(film forming)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

一般に、厚み0.1mm以下のものをフィルムと呼んでいる。図4・25にキャストフィルム製造装置を示す。装置は、基本的に押出機、T ダイ、キャストロール装置、巻取機から構成される。押出機で輸送、溶融、混練された溶融樹脂が、T ダイから所定の厚さのフィルム状に賦形される。これが、キャストロールで冷却固化され、一定の速度で引取られながら、ロール状に巻取られる。特にオレフィン系フィルムでは高速化が進み、冷却固化のために溶融樹脂をキャストロールへ密着させる補助手段が必要となり、そのためにエアナイフ等が付属されている。エアナイフは、スリットからエアを吹出させ、その押付け作用により、溶融樹脂をロールへ密着させる機能をもつ(図4・26)。フィルムの厚みは、T ダイのスリットすきまを局部的に調整することにより調整できる。キャストロール後に設置する放射線式等の厚み計によってフィルム厚さを自動計測し、T ダイのスリットのすきまを自動調整することで、フィルム1―78産業機械・装置厚薄を向上させる装置も多く用いられている。図4・27にヒートボルト式自動T ダイを示す。上述するT ダイ法によるフィルム成形のほか、インフレーション法と呼ばれるフィルム成形方法もある。図4・28にインフレーションフィルム製造装置を示す。インフレーションフィルムは、押出機先端に取付けられたダイから溶融樹脂を円筒状に吐出し、その中に空気を吹込んで膨張させて薄いフィルムを賦形する。その後、冷却リングからの冷却空気で冷却、固化させ、巻取機で巻取る。図4・29に逐次二軸延伸装置を示す。二軸延伸フィルムは、T ダイより吐出された溶融樹脂を原反冷却装置でシート状に冷却固化させ、これを、多段ロールからなる縦延伸装置で再加熱後縦方向に延伸する。次に横延伸装置で横方向に延伸し、熱固定後巻取機によりロール状に巻取られる。通常延伸は、樹脂の二次転移点以上融点以下の温度でなされ、縦方向、横方向に分子配向したフィルムとなる。このフィルムは、強度、伸度等の物性が向上し、使用用途が広がる。代表的適用樹脂として、ポリエチレンテレフタレート図4・31 シート用T ダイ図4・28 インフレーションフィルム製造装置図4・29逐次二軸延伸装置図4・30 シート製造装置(PET)、PP、ポリアミド(PA)がある。
