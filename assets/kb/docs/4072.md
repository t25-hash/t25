# 2・10・3 導水設備

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・10 水力発電設備

a.ダム（dam) 発電用のダムは使用目的によって，取水ダ
ムと貯水ダムに大別することができる．取水ダムは河川から取水す
る目的で設けた低いダムをいい，貯水ダムは河川をせき止め貯水す
るために設けた比較的高いダムをいう．
ダムは構造により図Ⅱ-2･61のような種類がある．
(１)重力ダム コンクリートを材料とし，堤体重量により水
圧を支えるもので，昔より広く用いられてきた．
(２)アーチダム コンクリートダムの一種で，上から見たと
き上流側にはり出したアーチ形をなし，そのアーチ作用で水圧を支
えるものである．アーチ作用を利用するためコンクリート量は少な
く経済的であるが，堤体より伝えられる応力は大となるので，基礎
岩盤が良質なことが必要である．
(３)中空重力ダム 普通のコンクリート重力ダムを中空とし
たもので，上流側遮水壁面上の水重もダムの安定に寄与するもので
ある．コンクリート量が重力ダムに比べて少なくて済むが，施工が
複雑となる．
(４)フィルダム フィルダムは，土，岩石などの天然材料を
積み上げて作ったダムで，遮水機能を果たす部分の構造によって次
のような種類がある．
均一型ダム：堤体の大部分がほぼ均一な材料によって構成され
たもの．
ゾーン型ダム：遮水ゾーンおよび透水係数の異なるいくつかの
ゾーンで構成されたもの．
表面遮水壁型ダム：上流面にアスファルトコンクリートなどの
遮水壁をもつもの．
b.ゲート（gate) ダム越流部に設置するクレストゲート，
取水量調節のための制水ゲート，土砂を排出するための排砂ゲー
ト，放水用のゲートなどがある．ゲートの形式は大別して，ローラ
形式，ヒンジ形式，スライド形式の3種類がある．
c .取水口（intake) 取水口は，取水ダムや貯水池に設けら
れ，流木などの流入を防ぐスクリーン，除じん機や取水量を調節す
る制水ゲートなどが設けられる．
d.導水路（headrace) 導水路は発電用の水を取水口から水
槽（あるいはサージタンク）に導く水路をいい，通常，トンネル，
開きょ，暗きょ，などから成る．渓流，低地などを横断する場合に
は水路橋，サイホンを使用する．水路内の流速は通常2～3m/sで
あるが，圧力トンネルの場合は3m/sを超えるものもある．水路
こう配は，小水量の場合は600分の1程度，大水量では2000分の
図 - 2･63地下発電所
(ａ)単床バレル式
(ｂ)二床式
図 -2･62据付方式
j .放水路サージタンク（tailrace surge tank) 放水路が延長
の長い圧力トンネルの場合には，負荷の変動による水撃作用を吸収
するため，放水路の始端近くにサージタンクを設ける．
