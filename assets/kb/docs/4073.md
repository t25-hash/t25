# 4・4・3 シート成形(sheet forming)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

厚み0.15mm以上のものをシートと呼んでいる.図4・30にシート製造装置を示す.装置は,押出機,T ダイ,ポリシングロール装置,引取機,定尺切断装置,または巻取機から構成される.ポリシングロール装置は3本構成で,各ロールとも高精度に加工されている.中央ロールは固定され,上下のロールが上下に移動できるようになっており,それぞれ,中央ロールとのロールすきまが調節でき,中央ロールに対して加圧できる.T ダイから吐出された溶融樹脂をこれらのロールに挟み込み冷却しながら,一定速度で引取り,一定寸法に切断する(巻取ることもある).近年,シートの外観品質要求が高度化され,ポリシングロール装置の駆動は,各ロールとも高精度ウォーム減速機とサーボモータの組合せとなっている.図4・31にシート用T ダイの断面を示す.シートダイでは,リップ各部での幅方向の流量を調整するためのチョークバーが設けられている.これでリップ部での流量と流速を均一に保つ必要がある.
