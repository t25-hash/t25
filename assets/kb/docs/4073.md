# 2・10・4 発電所の形式

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・10 水力発電設備

発電所の形式は，設置する場所の地形，規模，水車の形式などに
よって異なるが，一般には経済的見地から決定される．
a.据付方式 水車，発電機の配置方法には，大別して立軸と
横軸がある．立軸は水車，発電機の主軸が垂直に一直線上に接続さ
れているもので，これに対して横軸は水車と発電機とが同一床面に
据付けられ，その主軸が横に一直線上に接続されたものである．横
軸は基礎の掘削深さが浅く，建屋の高さも低くて済む利点はある
が，床面積が大きくなるので，一般に小容量機に採用されている．
立軸の据付方式には単床バレル式と二床式がある（図Ⅱ-2･62)．
(１)単床バレル式 発電機の全重量を水車のステーリングの
上に設置したバレルで支持する方式で，大容量機に採用される．据
付工事期間が長くなる欠点はあるが，建屋高さが比較的低くて済
む．

e .沈砂池（settling basin) 水路内に土砂が流入するのを防
ぐために設ける池で，池内の流速は0.2～0.3m/s程度とし，土砂
が沈殿するのに十分な池の長さとする．
f .水槽（head tank) 導水路の終端に，負荷の変化に対して
水位をできるだけ一定に保つために，適当な容量の水槽を設置す
る．水槽には，水車が急に停止した場合に最大使用水量を流すこと
のできる余水路を設ける．なお，経済性の面から他の設備を設けて
余水路を省略することもある．
g.サージタンク（surge tank) 圧力水路の場合には，水車
への流入水量が急変したときの水撃作用による圧力トンネルの圧力
上昇を防ぐため，サージタンクを設ける．
h.水圧管路（penstock) 水槽から水車まで水を導く圧力管
を水圧管といい，水圧管を支持する固定台，支持台および水圧管を
布設する路面を総称して水圧管路という．水圧管は鋼板で作られる
ものが多いが，最近では強化プラスチック管を使用した例もある．
i .放水路・放水口（tailrace，outlet ) 水車からの放流を，
河川への放流場所である放水口まで導く水路を放水路という．放流
量の大小による放水位の変動を少なくするため，比較的幅の広い水
路とする．
(２)二床式 水車と発電機を支持する床を別に設ける方式
で，床の強度の関係から中小容量機に適用される．水車と発電機を
別々に分解・組立できる便利さがある．
b.地上式と地下式 従来は発電所の建屋は地上にあったもの
が多かったが，最近の大容量発電所，特に揚水式では自然，環境保
護のため，また吸出し高さなどの関係から地下に発電所を設置し，
水車から出た水を放水路トンネルを通して放流する地下発電所とし
て建設されるものが多い（図Ⅱ-2･63)．
