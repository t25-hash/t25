# 4・4・4 異形成形(profile extrusion)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

窓枠などに代表されるいろいろな断面形状をもつ建材,工業製品を押出すものである.装置は,押出機,ダイ,サイジング,冷却装置,引取機,切断機から構成される.使用原料として,硬質塩化ビニル(PVC)を筆頭に軟質PVC,アクリルニトリルブタジェンス図4・32 軟質塩化ビニル窓枠ダイ図4・34 フィルムシート多層化法チレン(ABS),ポリカーボネート(PC),メタアクリル(MMA),PP,PEがある.押出機は,一般的に小サイズの単軸押出機であるが,熱安定性の悪い硬質PVCについては,低速かみあい型異方向回転二軸押出機を使用することが多い.ダイは,形状,樹脂の種類,機械加工を考慮し作製される.図4・32にダイの例を示す.また,断面形状がポイントであり,ダイを出た後,真空で形状を形成するサイジング装置で寸法を規定し冷却する.引取機で一定速度で引取り,一定寸法に切断する.
