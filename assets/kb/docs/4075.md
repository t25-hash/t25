# 4・4・5 発泡押出成形(extrusion foaming,foam extrusion)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

発泡押出しには,物理発泡と化学発泡がある.物理発泡は,ブタン,プロパン,窒素,炭酸ガスなどのガスをスクリュー押出機内の溶融したポリマ中に圧入し,ダイ出口で減圧発泡させ,後冷却固化させて高発泡成形品を得る方法である.一方化学発泡では,原料に化学発泡剤を混ぜ,押出機に供給されてシリンダ内で加熱されると,その限界温度以上で発泡剤が分解し,窒素,炭酸ガスを放出して発泡の役目を果たす.図4・33にPS 発泡シート製造装置の概念図を示す.装置は,押出機,ガス供給装置,丸ダイ,マンドレル,引取機,巻取機から構成される.押出機は2段構成で,1段めで溶融,ガス混合をし,2段めでガスを混合した樹脂を冷却する.高発泡なので,ダイは円筒状で,冷却方法としてはマンドレルを使用して内側から冷却する.ダイには冷却リングが付いており,外側からも空気で冷却可能である.シートは,同筒状の両サイドをカッタで切り開き,2枚のシートとして別々に巻取る.
