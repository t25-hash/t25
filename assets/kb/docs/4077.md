# 4・4・7 多層押出成形(multilayer extrusion)

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

押出成形における高機能化の一つとして多層化(co-extrusion)1―79図4・33 PS 発泡シート製造装置概念図がある。これは、2台以上の押出機を用いて一つの共通ダイから単一製品として成形するが、その断面は多層構造をなしている。多層化することで、単層では得られない種々の物性を得ようというものである。フィルムシートの場合、多層化する方法には、フィードブロック法とマルチマニホルド法の2種類がある。フィードブロック法は、T ダイの前でフィードブロックにより多層化した後、T ダイで幅方向に広げる。マルチマニホルド法は、T ダイにおいて幅方向に広がった各層が、T ダイ内部で合流、多層化する。図4・34にフィルムシートの多層化法を示す。熱可塑性樹脂の管状溶融体[普通パリソン(parison)と呼ばれる]を左右一対の金型の間に導入し、金型を閉じた後、気体(普通は圧縮空気)を吹込み、キャビティ形状に成形するとともに冷却固化して各種の中空容器を製造する機械がブロー成形(blow mold-ing)機である。熱可塑性樹脂として、結晶性ではHDPE(高密度ポリエチレン)、LDPE(低密度ポリエチレン)、L-LDPE(直鎖状低密度ポリエチレン)、PP(ポリプロピレン)、PA(ポリアミド)、PET(ポリエチレンテレフタレート)、非晶性ではPVC(ポリ塩化ビニル)、PC(ポリカーボネート)、PMMA(メタクリル樹脂)等が多用されている。このパリソンを異種材料で積層して成形する多層成形においては、ガスバリア性樹脂、接着材、再生材等が製品内部の中間層として用いられている。ブロー成形法(blow mold-ing)には多くの種類があるが、図4・35のように大きく分けられる。これらの各方法に応じて成形機も当然構造、機能が異なり、その種類は非常に多い。シングルステージ法は、押出しあるいは射出によって成形したパリソン内部に残る予備熱を利用する成形法で、ホットパリソン(hot parison)法とも呼ばれる。ツーステージ法は押出成形または射出成形により一度チューブまたは有底パリソンを成形した後冷却し、別のラインにて再加熱後ブロー成形するもので、コールドパリソン(cold parison)法とも呼ばれる。また、ブロー成形法の分類として、融点以下における延伸工程の有無により延伸ブロー法と無延伸ブロー法とに分けられる。
