# 4・5・1 押出ブロー成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・5 ブロー成形機

図4・36に押出ブロー成形(direct blow molding, extrusionblow molding)の概念図を示す。可塑化された樹脂はクロスヘッ1―80産業機械・装置図4・35 ブロー成形法の種類ドに送られ、製品に見合った量で円管状に押出される。その後、パリソンは割ブロー型で挟まれ、内部へ高圧空気を導入し壁面に押付けられ、冷却固化して取出される。成形装置は、図4・37に示すように1押出機、2クロスヘッド、3型締機構、4吹込み装置、5制御装置から構成される。本方式の成形機は、割ブロー型のみで成形が行われるため、比較的安価な成形装置である。しかし、ボトルなどの成形においては、製品のねじ山形状などの寸法精度は得られにくく、後工程でバリ取り、ボトル口部の仕上加工が必要となる。a. 押出機(extruder)熱可塑性樹脂を輸送、過熱、溶融、冷却、混練、定量圧送するもので短軸スクリュータイプのものが多く使われる。スクリュー形状は使用原料により異なるが、は一般に22~25でフィルムやシート成形用などのスクリューと比べて3~5短い。良品質のパリソンを押出すために、樹脂材料に適したスクリュー選択が重要となる。ブロー成形機に用いられる押出機(extruder)の要求性能として、押出量が変動せず安定していること、混練が良いこと、気泡を含まないこと、溶融樹脂温度が低いことなどが挙げられる。ブロー成形の場合も他の成形法と同様に、成形品のバリや不良品を粉砕して再使用する必要があり、このような再生原料の混入は、押出量の不安定要因となっている。その対策として、粉砕物はできるだけ粒度を均一にして混入し、押出機にはホッパ近傍のシリンダに軸方向に溝を設ける、シリンダ温度制御の精度を高めるなどの方法を用いる。最近では二軸スクリューも使用されており、混練、低温押出しに優れた特徴が得られる。b. クロスヘッド押出機で可塑化された材料を管状のパリソンに成形する部分であり、使用樹脂原料のレオロジー特性に合致した流路形状(偏流や滞留がないことが必要)、滑らかな表面仕上げ、温度制御が必要である。形式としてサイドフィード(side feed)型とセンタフィード(center feed)型に分けられる(図4・38、場合には多層クロスヘッド(multilayer cross feed)が用いられる(図4・40)。c. 型締機構型締めは油圧による直圧方式やトグル機構を使った方式があるが、直圧方式のものが多い。最近では、高精度の位置決めと動作時間の一定化などの要求により、サーボモータ(servo motor)による電動式型締機構も採用されている。d. 吹込み装置パリソンを左右一対の金型で挟んだ後、圧縮空気を吹込みキャビティ形状にパリソンを膨らませる装置である。図4・37 押出ブロー成形機図4・36 押出ブロー概念図図4・39センタフィード型クロスヘッド図4・38 サイドフィード型クロスヘッド図4・40 パリソンコントローラを組込んだ多層クロスヘッドパリソンに針状吹込みノズルを突き刺し、パリソン内部に空気を導入する。上吹込み(down ward blow)、下吹込み(up wardblow)、横吹込み(side blow)に大別される。e. 制御装置成形機の動作は金型の開閉、移動、吹込みノズルの打込み、戻りなどがある。これらの動きは油圧により制御されるものが多いが、サーボモータによる駆動のものもある。また、ブロー成形品の肉厚を均一に制御するために、押出すパリソンの肉厚を微少量変化させる装置がパリソンコントローラ(parison con-troller)である。これはあらかじめ設定した値にダイとコアの間隔を変更でき、パリソンの流出時に自由にパリソン厚みを変えられる。装置には高い応答性が要求されるため、必要に応じてサーボ弁や比例電磁弁などが用いられる(図4・40)。
