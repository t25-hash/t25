近では，広い落差に対して良好な性能を得るために回転速度を変え
られるようにした可変速水車（adjustable speed turbine，vari-
able speed turbine）が採用されることがある．これは回転速度を
落差に応じ，最適な に合わせて運転するものである．
図Ⅱ-2･16において，線Aより左側ではランナ羽根入口に対す
る相対流れの入射角が大きくなり，入口負圧面にキャビテーション
や流れのはく離による騒音が発生する．また線Bより右側では負
の入射角が大きくなって，入口圧力面から流れがはく離して騒音が
発生する（1･4･2項，図Ⅱ-1･41参照)．線Cより下の部分負荷領
域ではランナ羽根間流路渦（channel vortex）が発生し（図Ⅱ-2･
17の羽根間渦），運転上制限が加えられることがある．線Dはラン
ナ出口流れに旋回成分のない状態を示す．線Eは部分負荷時にお
いて，ランナ出口の旋回流が発達して死水コア（図Ⅱ-2･17の中心
部ホワール）が生じ，偏心運動してサージングが最も強くなる状態
を示す．サージングが強い場合には，騒音，水圧脈動が大きく，場
合によっては電力系統との共振を生じ，電力動揺（power swing）
を生じることがある．このような場合の対策として，吸出し管上部
へ給気する方法が一般に採用されている．
また，無拘束速度は低比速度機で定格回転速度の約1.5倍，高比
速度機で約2.3倍である．ガイドベーン開度が大きくなると無拘束
速度は大きくなる（図Ⅱ-2･16）．
