# 4・6・3 複動式圧縮成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・6 圧縮成形機

単動式のように型締め方向だけでなく、型開き方向にも油圧が働く方式で、この場合には金型は強制開放できるので、取付金型方式が採用でき、自動成形ができる。複動式には複動ラム式、サイドシリンダ式、ブースタラム式がある。複動ラム式は、段付きラムを使用し、型を閉じるときにはラムの段付き部から油圧を押込んでラムを上昇させ、型を開くときには油圧バルブを切換えて段の上部から油圧を押込み、可動盤を下降(復帰)させる。サイドシリンダ式は、単動プレスの両側にサイドシリンダを設けて、型開き方向にも加圧できる。ブースタラム式は、複数ラムの内部にブースタラムを設置したもので、作動状況は複動ラム式と同じである。(a)(b)(c)図4・43 圧縮成形の要領図4・44 圧縮成形機(上押し形の例)トランスファ成形(transfer molding:移送成形、プランジャ成形)機は、トランスファチャンバと呼ばれる金型の一部に設けた加熱室内で成形材料を可塑化させ、これを狭い通路を通してキャビティ(雌雄金型にはさまれた空間部分)内に圧入して成形する装置(図4・45)で、熱硬化性樹脂の成形に用いられる。ただし、ポット式トランスファ成形は圧縮成形機を用いるので、狭義のトランスファ成形機は、型締用ラムのほかに材料注入用の補助ラムを備えた成形機を指す。
