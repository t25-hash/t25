# 4・7・2 補助ラム式トランスファ成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・7 トランスファ成形機

型締ラムのほかに、トランスファポットから成形材料をキャビティ内に圧入するための補助ラムをもったもので、上部補助ラム式、下部補助ラム式および二重ラム式がある。上部補助ラム式は、上押し型型締機構の固定盤の上部に下押し型補助ラムを有するもの(図4・46)、下部補助ラム式は、下押し型型締機構の固定盤の下部に上押し型補助ラムを有するもの、二重ラム式は、主ラム内に補助ラムが収められているものである。このほかに、型締ラム一つに対して補助ラムを二つ備えたダブル補助ラム式、下押し型型締機構に対して補助ラムが水平に置かれたアングル式、垂直型締機構の両側に補助ラムが水平に配置された対図4・45 トランスファ成形(補助ラム式)の要領図4・46 トランスファ成形機(上部補助ラム、上押し型の例)向式などがある。特殊なタイプのものとしてはスライド式トランスファ成形機がある。これは、下定盤(可動盤)が上定盤(固定盤)の2倍の面積をもち、下定盤が左右にスライドするように構成されており、上定盤に取付けた1個の上型に対し、下定盤には2個の下型を取付ける。一方の下型で成形している間に、もう一方の下型で製品取出し、金型内の清掃、インサート物の装てんなどを行い、下型が交互にスライドして上型の下部に移動して成形する。積層プレス成形機(laminating press)は、紙や布などの基材に液状の熱硬化性樹脂を含浸させ、適宜な枚数を重ね合わせて加熱圧縮して成形する装置で、通常は板状製品を作るのに用いられる。1枚だけを作る場合は通常の圧縮成形機でよいが、通常は多数の板を作ることが多く、その場合には多段式積層プレス成形機(多段プレス)を使用する。多段式積層プレス成形機は図4・47のように上部の固定盤と下部の可動盤の間に多数の加熱板を設け、可動盤下部のラムシリンダによって可動盤を上昇させ、多数の加熱板の間に挿入した樹脂含浸シートを加熱圧縮する。加熱板の製品に当たる面には鏡面研磨仕上げ板を用い、加熱板は通常は蒸気で加熱されるようになっており、フレキシブルパイプで相互に連結され、このパイプ中を熱媒体が通る(図4・48)。また、パイプは通水して冷却する場合にも使用される。積層プレス成形機は熱可塑性樹脂の加工にも使われる。押出機で作った薄いシートを何枚か重ねて肉の厚い板を作る場合、シート表面につやをつける場合、熱可塑性樹脂シートと鋼板などを重ね合わせた複合シートを作る場合などにも使用する。図4・47 多段式積層プレス成形機1―83粉末成形(powder molding)は、粉末状の樹脂を原料とし、金型内での加熱、加圧、冷却、固定などの工程を経て、所定形状の連続固体製品を得る成形加工技術の総称である。粉末成形の範ちゅうとその分類は必ずしも統一されていないが、その中で代表的な3種の成形法について概説する。
