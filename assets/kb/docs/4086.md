# 4・9・1 回転成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・9 粉末成形機

回転成形(rotational molding)は、粉末樹脂を原料として中空のプラスチック製品を成形する方法の一つである。その基本原理は、(i)中空の金型内に製品重量と同量の熱可塑性樹脂粉末を仕込み、(ii)金型に水平-垂直方向の同時二軸回転を与えながら加熱して、金型内壁面上に均一厚の樹脂融液層を形成させた後に、(iii)金型を冷却して樹脂層を固化させ、中空成形品を取出すプロセスである。回転成形の特徴として、小さな中空ボールから数万Lに及ぶ大型タンクに至るまで応用範囲が広く、厚肉壁構造、二重壁構造、多層構造、繊維強化構造などの構造設計が比較的容易である。さらに、成形工程でかかる圧力およびせん断が小さいため、成形品中の残留応力が小さく、設備、機械、金型などの製造コストが低い、などの利点を有する。その反面、成形サイクルタイムが長く、衝撃強度が小さい、などの欠点も有する。回転成形機械は、基本的には金型、金型を回転するための駆動装置、金型の加熱装置および冷却装置からなり、それぞれいくつかの方式がある。それらの組合せにより、数種の回転成形装置が開発され技術的にも発展してきた。その中の代表的なものとして、アーム型回転成形機の概略を図4・49に示した。中央のターレットと3本のアームを有し、各アームの先端に金型を取付けて水平および垂直方向の同時二軸回転を与える。個々のアームの金型においてそれぞれ製品取出し・原料供給、加熱、冷却の異なる工程を同時に行うことができる。所定時間間隔ごとにターレットが回転してそれぞれのアームを順次次の工程位置に移行して成形工程を進めていくことができるから成形効率上有利である。材料としては、ポリオレフィンを主流に、フッ素樹脂、ポリアミド、ポリエステル、ポリカーボネート、ポリアセタールなどの樹脂も可能であるが、高温での加熱時間が長いために熱安定性が特に要求される。
