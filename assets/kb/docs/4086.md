# 2・6・1 構造

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・6 斜流形およびプロペラ形ポンプ水車

斜流形ポンプ水車（diagonal flow type reversible pump- tur-
bine）は約30～150mの落差，揚程に適用される中比速度に適し
たポンプ水車であり，プロペラ形ポンプ水車（propeller type
reversible pump- turbine）は約20m以下の落差，揚程に適用され
る高比速度に適したポンプ水車である（2･1･1項参照)．現在実用
1：ランナ羽根，2：主軸，3：ガイドベーン，4：ガイドベーン開閉機構，
5：カバー，6：主軸受，7：ステーリング，8：ケーシング，9：吸出し管，
10：ランナ羽根サーボモータ，11：ランナ羽根開閉機構
図 -2･40立軸可動羽根斜流形ポンプ水車の構造
化されているものは可動羽根構造のものが多い．ポンプ運転時と水
車運転時とでは回転方向が逆となるが，これ以外に構造的には斜流
水車，プロペラ水車と大きな差はない．
斜流形ポンプ水車は同じ要項の斜流水車に比べて，ランナ入口径
が約1.2～1.3倍，羽根長さは約1.3倍となり，羽根枚数も若干少
なくなる．ランナ羽根に発生する応力も，遠心力，水圧力の影響で
過酷となるので，ランナ羽根付根の厚さ，応力集中緩和対策などに
配慮が必要である．
図Ⅱ-2･40は立軸斜流形ポンプ水車の構造例を示す．外観および
ランナ羽根開閉機構などは斜流水車とほとんど同じである．一方，
プロペラ形ポンプ水車では，渦巻ケーシングが用いられることはほ
とんどなく，円筒ケーシングが用いられる．
図Ⅱ-2･41は潮力発電所（tidal power station）用として設計さ
れたプロペラ形ポンプ水車であり，発電電動機はケーシング内にお
さめられ，チューブラ水車と同様な構造である．
斜流形，プロペラ形ポンプ水車では，ランナ羽根を全閉にするこ
とによりポンプ運転時の起動動力が小さくなり，水中で始動でき
る．大容量機ではフランシス形ポンプ水車と同様に，水面押し下げ
による空中起動方式とすることが多い．
