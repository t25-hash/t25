# 4・9・2 焼結成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・9 粉末成形機

焼結成形(sintering molding)は,従来から窯業や粉末冶金などで用いられてきた焼成技術を粉末樹脂に応用したものである.その基本原理は,(i)金型内で熱可塑性樹脂粉末を常温圧縮して予備成形し,(ii)この予備成形品を加熱炉内で加熱して,粉末粒子間での表面融着による多孔構造,さらには溶融粒子の流動による均質構造を形成させ,(iv)冷却して成形品を取出すプロセスである.粉末樹脂の溶融粘度が非常に高くて焼結過程での粒子の流動変形が困難な樹脂の場合は,粒子間に形成される空げきが成形品中に残図4・48 積層成形法図4・493本アーム型回転成形機1―84産業機械・装置存することから,多孔構造の通気性素材や各種フィルタ素材などへの応用も試みられている.焼結成形品の空げき率の制御については,原料樹脂の粉体物性(粒子形状,粒度分布,かさ密度,安息角,硬度など)や樹脂素材特性(分子量,化学組成,溶融流動特性など),予備圧縮条件,焼結条件などがおもな要因となる.成形装置として,焼成炉の設備規模は,成形品の材質,形状,寸法などにより違ってくるが,金属やセラミックスの場合のような高温・高精度の炉は必要がなく,一般のバッチ炉やトンネル炉であれば使用可能である.加熱方式は,熱風循環式,遠赤外線ふく射加熱式,高周波加熱,熱プレスなど,場合によって使い分けられている.この成形法はポリエチレンやポリプロピレンなどの通常の熱可塑性樹脂粉末にも用いられるが,フッ素樹脂や超高分子量ポリエチレンなどのように溶融粘度が極端に高いために通常の熱可塑成形が難しい樹脂がおもな対象となる.さらに熱可塑成形が困難とされていたポリイミドやポリベンツイミダゾールなどの超耐熱性樹脂粉末に対しても一部実用化されている.
