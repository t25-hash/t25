# 4・9・3 粉末加圧成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・9 粉末成形機

粉末加圧成形は,金型内で原料樹脂粉末を加圧圧縮して粒子の塑性変形や内部発熱などによる粒子合体を行い成形品を得る方法であり,(i)この加圧を常温で行う冷間加圧成形(cold press mold-ing),(ii)加熱を伴う熱間加圧成形(hot press molding),さらには(iii)常温で超音波振動を加えながら加圧する超音波成形法などの工業的応用の有用性についても検討されてきた.冷間粉末加圧成形法は,加熱せずに圧縮力による塑性変形で粒子を合体させて成形品を得ようとするもので,省エネルギー,省資源,生産性などの点から有効な技術として期待されるが,粒子間の不均質な変形挙動や内部発熱挙動,弾性回復による変形や寸法変化が問題となり,その対策が検討されている.加圧方法は機械的加圧のほかに,静水圧により加圧する方法も用いられる.熱間加圧成形法は,加圧時に加熱を併用することによって粒子の合体を促進し成形品の物性向上を図るものである.さらに,超音波成形法は,図4・50に例示するように,冷間加圧過程で超音波振動エネルギーを付与するものであり,粉末粒子間の摩擦熱を増大させて粒子どうしの合体と構造の均質化を促進させる効果が認められている.その結果,成形時間の短縮,成形精度や製品物性の向上などが期待される.二次成形は,一次成形した中間製品にさらになんらかの加工を加えて最終製品を製造する工程である.二次加工としてはシート加工,機械加工,接着,塗装などの多くの加工があるが,本稿ではその主たるものとして,(1)熱可塑性樹脂シートを原料とする熱成形,(2)熱可塑性樹脂とガラス繊維の複合シートを用いるスタンピング成形について概説する.
