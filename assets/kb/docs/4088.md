# 2・7・1 S形チューブラ水車

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・7 小型水車

図Ⅱ-2･45に示すS形チューブラ水車（S-type tubular turbine）
は，吸出し管をS形にわん曲させて水車軸を貫通させ，発電機を
外部に設置する形式のチューブラ水車であり，落差約20m以下の
低落差地点に採用される．ランナ羽根・ガイドベーンともに可動
型，ランナ羽根かガイドベーンのどちらかを固定とするなどの形式
がある．
ランナ出口での排棄速度エネルギーが有効落差の約30 %もある
軸流水車では，水車効率に及ぼす吸出し管の影響が大きい．このた
めに吸出し管に貫通軸および二重の曲がりをもつS形チューブラ
水車の効率は，流路が直線状のバルブ形チューブラ水車に比べて低
下する．
