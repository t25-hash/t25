# 4・10・1 熱成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・10 二次成形機

熱成形(thermoforming)は、熱可塑性樹脂のシートや薄板などを原料として、これを加熱軟化させ、真空力や圧空力などの外力の強度部品、あるいはコンクリート型枠など非自動車分野での展開図4・51 ストレート法真空成形金型図4・50 超音波粉末成形金型により変形を与えて賦形し、冷却・固化して成形品を得る方法[真空成形(vacuum forming)、圧空成形(pressure forming)]である。真空成形の基本原理は、図4・51に示すように、(i)原料シートを所定の凹形金型にクランプして加熱(シートのガラス転移点と融点の間でゴム弾性を帯びた軟化状態)した後、(ii)金型内を真空に引いて軟化シートを金型内壁面に張り付かせて所定の形を与え、(iii)冷却固化して成形品を取出すプロセスである(ストレート法)。この方法では、軟化シートを真空力で金型内壁に張り付かせる際に、場所によりシートの延伸量が異なるために成形品の厚さむらが生じ底面のコーナ部が薄くなるなどの問題があるが、この改良策として次のような方法が用いられている。(1) ドレープ法は、固定枠にクランプした軟化シートを凸型で突き上げて予備伸張した後に、真空に引いて凸型外壁に張り付かせて成形する方法である。この方式は深絞り製品に適するが、底部が厚く側面上部が薄くなる傾向がある。(2) プラグアシスト法は、固定枠にクランプした軟化シートを凸形のプラグで凹型の内部に押込んで予張した後に、真空に引いて凹型内壁に張り付かせる方法で、厚さむらが改良される。(3) エアブロー法は、軟化シートをクランプした凹型内に圧搾空気を送ってシート全体を外側に所定量だけ膨らませて予張した後に、凹型内を真空に引いて凹型内壁に張り付かせることにより、ストレート法における厚さむらを軽減する方法である。真空成形機は、一つの金型で1サイクルで1個の製品を成形する単発機と、広幅連続シートと多数個の金型を用いて、一度に多数の製品を連続的に成形するためのライン式連続成形機、ロータリ式成形機、ドラム式成形機などが開発されている。成形型は金属型が主体でアルミニウム合金、亜鉛合金などが多い。少量生産用の廉価な木型、サンプル型用の石膏型や樹脂型も用いられる。熱成形法は、原料樹脂がすでにシート形状を有しているために、低い成形圧で容易に薄肉形状の中空成形品が得られ、成形機や金型などの生産コストを低減でき、生産エネルギーの面でも有利である。その結果、低コスト大量生産用途として、食品包装用容器やブリスタ包装などに多く応用され、さらには冷蔵庫の内装部品や浴槽、自動車部品などの大型品にも用いられる。
