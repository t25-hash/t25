# 4・10・2 スタンピング成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・10 二次成形機

スタンピング成形(stamping molding)は、(i)熱可塑性プラスチックを基材として長繊維で強化した1~5mm厚のシート(スタンパブルシート)を作成し、(ii)このシートから製品の形状に合わせて切断した材料(ブランク)をフリーの状態で赤外線ヒータなどで融点以上に加熱した後、(iii)これをプレス金型の所定位置に置いて常温付近の温度でプレス加圧を行って、所定形状を付与すると同時に冷却・固定して成形品を取出すプロセスである。スタンパブルシートは使用するマトリックス樹脂と強化繊維の種類により種々の組合せが可能であるが、工業的な主流はポリプロピレンとガラス繊維の組合せである。現在おもに用いられているスタンパブルシートの製造法としては、PPシートの間にガラスマットを積層してラミネータで加熱、溶融、圧着含浸させて作成する。ハイサイクル化による生産性向上のために、各工程の自動化や多数個取り成形、金型冷却法の効率化、金型上下のすり合わせ部分の精度向上によるバリ発生の低減などの技術的改良が進められている。スタンピング成形の特徴は、ガラス長繊維マットで補強されたシートを金型に挟んで圧縮成形するので、他の方法の成形品より機械的特性、低温衝撃特性、クリープや疲労などの耐久性に優れている。この成形技術により、大型で複雑な形状の部品も短いサイクル時間で一体成形が可能であり、従来の鉄鋼材料の半分以下に軽量化できることなどから、自動車のバンパビーム、バッテリトレーなども行われている。プラスチックに繊維を強化材として複合化させたものを繊維強化プラスチック(FRP: fiber reinforced plastics)という。プラスチックには熱可塑性樹脂と熱硬化性樹脂があり、強化材にはガラス繊維や炭素繊維などの無機材料、アラミドやナイロンなどの有機材料が用いられる。中でも熱硬化性樹脂である不飽和ポリエステル樹脂やエポキシ樹脂とガラス繊維や炭素繊維の組合せが一般に多く、強化繊維の種類によりGFRP(glass fiber reinforced plastics)、CFRP(carbon fiber reinforced plastics)と表現する場合もある。使用する繊維の形態や製品の形状により次に挙げるような成形方法、成形機が開発されてきた。
