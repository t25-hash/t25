b.水車調速機の構成 図Ⅱ-2･50に調速機を含む水車制御系

.速度検出部 機械式調速機の場合は，水車発電機の回転
速度をベルトまたは電動機により速度検出部（スピーダ）に伝達
し，このスピーダが回転速度に応じて変位して速度検出する方式が
一般的である．アナログ電気式あるいはディジタル式調速機の場合
は，発電機に専用の信号発電機やパルス発信器を直結させ，回転速
度を正弦波やパルス列の形で入力し，これを共振回路やパルス計数
回路を介して回転速度に応じた信号に変換する．
.演算増幅部 機械式調速機の場合は，レバー機構を使っ
た比例演算のみであるが，アナログ電気式あるいはディジタル式調
速機の場合は，機能的にP（比例)，I （積分）要素に，D（微分）
要素を加味してより即応性に優れ，安定性の高いPID型調速機が
多く採用されている．PID型調速機の場合は下記ⅴ項の弾性復元
は不要である．アナログ電気式ではアナログ演算素子によって，デ
ィジタル式ではPLC（programmable logic controller ）のソフト
ウェア処理によってこれらの制御信号演算が行われ，これが変換部
に伝えられて機械式変位となり，後段の油圧増幅部に入力される．
.油圧増幅部 電気-油圧コンバータとこれによって操作
される補助サーボモータから構成され，後段のガイドベーン操作機
構に対して十分な力と変位の操作信号を伝えるためのものである．
.ガイドベーン操作機構 補助サーボモータからの操作信
号を受ける主配圧弁と，ガイドベーンサーボモータおよび復元機構
から成る最終段の油圧増幅部である．
.復元部 剛性復元と弾性復元がある．前者は補助サーボ
モータの開度に比例した信号を演算増幅部に復元し，回転速度と補
