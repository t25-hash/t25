# 2・8・2 入口弁（inlet valve）

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・8 付属装置

水圧管の末端で，水車・ポンプ水車のケーシングの直前に設けら差の領域にはちょう形弁が主として用いられる．
これら各種入口弁のおおよその適用範囲を図Ⅱ-2･53に示す．
最近ではスルース弁はが小さくなるにつれて悪化する．
なお， は管径の異なる各管路の長さ（m)， は各管路内の平均流速（m/s)， は重力の加速度（m/ s)， は水車の有効落差（m)， は発電機およびを併せ持っており，最近ではさらに高落差の領域まで拡大適用されるようになってきた．
入口弁は一般に，水車・ポンプ水車が事故の第Ⅱ部 水力機械 第2章 水車およびポンプ水車を介して油圧サーボモータ，または水圧管の圧力が高いときには水圧サーボモータによって開閉される．
また，小型のものでは電動サするため，ロータリ弁では図Ⅱ-2･55のように全閉時のみ作動する状の弁体が回転し管路を閉じる構造になっている．
弁の操作はローている点が待機運転と異なる．
小流量および中・高落差の領域にはロータリ弁を，中・低落管路時定数＝∑が長くなるにつれて，あるいは発ほとんど用いない．
なお，ちょう形弁には，弁体断面がレンズ形をしたものと，弁体が全開時に流れと平行になる2枚の板，およびそれを支えるリブとから成る複葉弁と呼ばれるものがある．
複葉弁は比較的新しい形式のものであるが，ロータリ弁とちょう形弁の特徴は水車の定格出力（kW）を示す．
