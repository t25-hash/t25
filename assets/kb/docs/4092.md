# 4・11・2 マッチドダイ成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・11 繊維強化プラスチックの成形機

雌雄一対のFRP型に成形材料を充てんし、プレスにより成形する装置を圧縮成形機という。圧縮成形法はFRP成形法の中でもよく利用される方法の一つであり、高温・高圧を要するホットプレス法と低温・低圧でも成形可能なコールドプレス法に分類される。精密で剛性が高く、温度調節の精度に優れた金型(マッチドダイ)を用いる成形装置はホットプレス成形機の一つである。マッチドダイ成形(matched die molding)の原理は、図4・52に示すように下型に成形材料を置いて型締めし、加熱・加圧して成形するというものである。成形材料には型の形状に予備成形した繊維補強材(プリフォーム)と樹脂を分けて用いる場合(pre-form matched diemolding)とガラスマットなどにあらかじめ樹脂や硬化剤、充てん材、低収縮剤、増粘剤などを混合させた樹脂ペーストを含浸させたシート材料(SMC: sheet molding compound)やパテ状材料(BMC: bulk molding compound)を用いる場合がある。図4・52 マッチドダイ原理図図4・53 オートクレーブ概念図1―85
