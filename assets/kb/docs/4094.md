# 4・11・4 FW 成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・11 繊維強化プラスチックの成形機

フィラメントワインディング(FW : filament winding)と呼ばれる成形を行う装置のことであり,ガラスや炭素からなる繊維束(ロービング)に樹脂を含浸させながら回転マンドレル(心材)に巻付けた後,加熱硬化させる(図4・54).成形品の性能よりも生産性を重要視した成形方法であり,大型タンクの胴体や地中埋設管といった大型成形品の生産にも利用されている.巻付け方式には図リカルワインダ,(c)ポーラワインダなどがある.強化繊維はガラス繊維や炭素繊維が主流であるが,用途に応じてアラミド繊維やビニロン繊維,ナイロン繊維を用いる場合もある.樹脂材料には,コストや性能の点において一般的には不飽和ポリエステル樹脂を用いる場合が多く,次いでエポキシ樹脂,フェノール樹脂などが用いられる.これらの樹脂は,加熱により硬化するが,時間とともに粘度が上昇する.繊維束をマンドレルに巻付ける作業は,成形品が大型になるにつれ時間を要するため,粘度変化の少ない樹脂の選択が必要となる.
