# 4・11・5 引抜成形(pultrusion)機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・55 に示すように(a)サーカムファレンシャルワインダ、(b)ヘ

ガラスなどのロービング状繊維を引きそろえたものに液状樹脂を含浸させた後、パイプ形状や円形状、異型断面形状のダイ(金型)を通して引出し、加熱硬化させて成形する方法を引抜成形法図4・54 フィラメントワインド概念図(b) ヘリカルワインダ(a) サーカムファレンシャルワインダ(c) ポーラワインダ図4・55 フィラメントワインド巻付け方式1―86産業機械・装置(pultrusion)という。図4・56に引抜成形の工程図を示す。引抜成形装置は、ロービングラック、含浸漕、プリフォーム部、金型、引取機、カッタから構成される。硬化方法としては、加熱金型を用いる場合や高周波加熱する場合がある。引抜成形品は、丸棒などの中実断面形材を農業用ビニルハウスの骨材として使用することから広く行われるようになってきた。圧縮成形法やオートクレーブ成形法と異なり、連続的に成形されるので所定の長さにカットする必要がある。
