# 4・11・6 レジントランスファ成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・55 に示すように(a)サーカムファレンシャルワインダ,(b)ヘ

レジントランスファ成形(RTM : resin transfer molding)とは一般的にアメリカで呼ばれる名称であり,ヨーロッパではレジン射出成形(RIM : resin injection molding)と呼ばれている成形方法である.図4・57にRTM の概念図を示す.型内にあらかじめプリフォームやマット,テキスタイルなどを置いて型締めし,注入孔より低圧で液状樹脂を注入し,硬化させる成形機である.また,樹脂の流動を促進させるために真空ポンプで型内を負圧にする(vac-uum assisting)こともある(図4・58).硬化方法は常温硬化や硬化炉内で硬化させる方法がある.これらの場合はFRP型などの比較的安価な型が使用される.また,生産性と寸法安定性向上のためにヒータを内蔵した金型を使用する場合もある.図4・56 引抜成形工程図図4・57 RTM 概念図
