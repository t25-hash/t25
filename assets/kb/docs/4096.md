# 2・9・4 振動・脈動

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・9 運転と制御

水車の運転中に生じる振動・脈動の原因として，次のようなもの
がある．
(１)軸受すきまの過大，主軸の曲がり，たわみ，回転体の不釣合
いなどによる軸振動．
(２)非設計点の運転におけるランナ周辺のはく離や二次流れによ
る静的な不釣合い力（1･2･5項参照)，および水圧脈動
（pressure fluctuation）に基づく軸振動．
(３)ガイドベーンの後流と回転するランナ羽根との干渉による水
圧脈動に基づくランナの振動，および静止側流路部の振動．
(４)部分負荷や過負荷時に，フランシス水車，プロペラ水車のラ
ンナ下部に発生する渦心に起因した水圧脈動，およびサージ
ング（2･3･3項参照）による騒音，振動．
(５)ランナ羽根出口端におけるカルマン渦とランナ羽根が共振す
ることによる鳴音，振動．
(６)ランナおよび吸出し管内のキャビテーションによる騒音，振
動．
(１)(２)については工作精度を維持することはもちろんである
が，必要に応じてコンピュータによる軸系の振動解析を行い，回転
体不釣合いやランナに加わる水力加振力による軸振動を，許容値内
に収めるのに必要な軸や軸受の剛性を求めて設計を行う．(３)につ
いてはランナとガイドベーンの間隔を大きくするか，羽根枚数を変
える．また，羽根入口を傾斜させ，ガイドベーンの後流をランナ羽
根入口が同時によぎらないよう配慮する．(４)(６)に対しては，吸
出し管内に空気を送入することによって軽減できる．(５)は，ラン
ナ羽根の出口端近傍の形状を，例えば表面と裏面とを非対称な形状
に変えればよい．
