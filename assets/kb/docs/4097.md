# 2・9・5 可変速ポンプ水車

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・9 運転と制御

ポンプ水車は通常一定回転速度で運転され，ポンプ運転時の軸入
力は，ある揚程に対して大きく変化させることはできない．ポンプ
水車を可変速で運転するとポンプの可変速制御と同様に流量，軸入
力を変化させることができ，ポンプ水車を負荷として系統周波数調
整を行うなど，よりきめ細かな電力調整が可能である．可変速ポン
プ水車（adjustable speed pump- turbine，variable speed pump-
turbine）の構成を図Ⅱ-2･60に示す．一定回転速度のポンプ水車
の発電電動機は回転子を直流で励磁する同期発電電動機であるが，
2･5･3項に示すように，水車とポンプの最高効率点の回転速度は異
なる．回転子を周波数変換装置を介して交流励磁する交流励磁方式
発電電動機は，可変速幅が数%程度に制限されるが経済性は高く，
ポンプ水車の可変速化に用いられ実用化されている．可変速ポンプ
水車の水車運転では，同期回転速度より低い回転速度で運転するこ
とによって，定速機より水車の最高効率点に近い特性で運転でき，
効率向上と運転範囲の拡大が可能である．また，回転体のフライホ
イールエネルギーを放出または吸収することで，過渡的に回転速度
が変化して高速の電力応答を行うことができる．
