# 4・12・1 二次加工の種類

> **章**: 第4章 プラスチック加工機械
> **節**: 4・12 二次加工装置

プラスチックの二次加工(fabrication, secondary processing)とは、射出成形や押出成形等の一次成形で得られた成形品になんらかの加工操作を加えることである。プラスチックの一次成形品がそのままで完成品として使用されることは意外に少なく、多くの場合、なんらかの二次加工を施すことにより初めて実用化されるものである。例えば押出成形により製造されるパイプは、曲げ加工やフィッティングとの接合等の二次加工により初めて実用価値が生まれる。また射出成形により製造される電子機器のハウジングなどは、電磁遮へい用の金属粉を塗装することにより初めて使用可能となるものも多い。このように二次加工は一次成形品に商品機能をもたせ、さらに商品価値を高めるという点で非常に大きい役割を果たしている。二次加工およびその装置は非常に種類が多く、カバーする範囲も広いが、ここではそのポイントのみを示すにとどめる。詳しくは別の成書を参考にされたい。図4・59は二次加工の種類をまとめたものである。二次加工は大別して図の2種に分類するのが理解に便利である。その(A)は一次成形品の商品化のための後成形、後加工工程であり、一般に一次成形品の性能はそのまま維持される。その(B)は新しい特殊機能を付与するための後処理工程である。(A)のうち、シート成形は二次成形機の項で述べられるので、ここでは(A)のうちの機械加工と接着および(B)を対象としている。図4・58 RTM 成形装置図4・59二次加工の種類
