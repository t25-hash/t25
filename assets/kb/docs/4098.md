# 2・1・1 作動油の種類

> **章**: 第2章 油圧システム
> **節**: 2・1 油圧作動油

作動油は一般的には油圧作動液（hydraulic fluid）を指すことが
多いが，広義には機能性流体（functional fluid）を意味し，ブレ
ーキ液や自動車の自動変速機油なども含まれる．ここでは，油圧作
動液の意味で用いる．作動油は圧力エネルギーの伝達媒体であり，
以下の性能が要求される．①適正な粘度を有すること，②温度変化
に対して粘度変化が小さいこと（粘度指数が高いこと），③潤滑性
が良いこと，④体積弾性係数が大きいこと，⑤寿命が長いこと，⑥
燃えにくいこと，⑦キャビテーションに対して抵抗が大きいこと，
⑧汚染物除去が容易なこと，⑨廃液処理が容易で，毒性のないこと
などである．作動油には図Ⅴ-2･1に示すように非常に多くの種類
があるが，上記の性能を満足し，用途または目的等に合った適正な
作動油を選択することが必要である．作動油は大きく分類すると，
鉱油系作動油（petroleum base hydraulic fluid），難燃性作動液
（fire resistant hydraulic fluid），生分解性作動油（biodegradable
hydraulic fluid）に分かれる．
