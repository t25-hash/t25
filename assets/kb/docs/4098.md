# 4・12・2 二次加工の際の表面の重要性

> **章**: 第4章 プラスチック加工機械
> **節**: 4・12 二次加工装置

二次加工の大部分、すなわち図4・59(A)の接着および(B)のすべてはプラスチックの表面または界面が関与している。したがってプラスチックの表面状態についての十分な知識をもっていないと満足できる加工技術および装置を駆使することは困難である。表4・1は各種材料の表面張力を示したものである。なおポリマについては固体状態での真の表面張力を測定することが困難なので、臨界表面張力γ욂で表現している。プラスチックとして用いられるポリマのγ욂は18.5~46mN/mの範囲にあり、無機物の表面張力が100~200mN/m程度、金属の表面張力が数百~1000mN/m程度であるのに比べて著しく小さい。すなわちその表面は典型的な低エネルギーであり、その表面を十分に濡らして接着、塗装や各種の表面処理を行うには特殊な操作や装置を活用することが必要であることが多い。このような表面処理では、原理的にはまず表面に活性層を形成させ、次にこの面に機能性薄膜を作ることが必要である。図4・60はこれらの間の関係を示したものである。現実には活性層の形成が即機能性膜の形成と直結する場合があるが、その各々の意義を分離して考えるのが妥当であろう。図4・59(B)の表面極性化処理には火炎処理、化学処理、コロナ放電処理、プラズマ処理等があり、それぞれ専用機が開発されているが、この表面極性化処理により通常30mN/m以下の表面張力は40~60mN/mにまで高エネルギー化される。このようにして形成された活性層の上に塗装、印刷、メ表4・1 各種材料の表面張力分類種類表面張力(N/m×10)温度(°C)ポリマポリ4フッ化エチレンポリプロピレンポリスチレンポリメタクリル酸メチルポリ塩化ビニルポリアミド6ポリカーボネートポリエチレンテレフタレート18.5無機物水シリカ酸化鉄無機ガラス炭酸カルシウム72.8金属水銀鉄銅471.6注:ポリマの値は室温での臨界表面張力。1―87タライジング等による機能性薄膜の形成がなされるのである。
