# 4・12・3 おもな種類の概説

> **章**: 第4章 プラスチック加工機械
> **節**: 4・12 二次加工装置

a. 接着接着(adhesion)には溶液やエマルジョン,ホットメルト接着剤等の液状接着による接着,熱溶接,高周波溶接,超音波溶接,エレクトロフュージョン等の種々の方法がある.ドライタイプの接着方法はウェットタイプのものに比べ,液体の蒸発,乾燥の工程を必要としないのが大きな特徴である.高周波溶接は,被着体が分極された分子構造を有しないと高周波により発熱しないので適用できない.この方法は軟質塩化ビニルの接着に最も多用されている.超音波溶接は超音波振動により接合面に発生する摩擦熱によって溶接する方法で,原理的にはすべてのポリマに適用できる.エレクトロフュージョン法は熱溶接の一種で,近年特にポリエチレンパイプの現場での接合法として開発されている方法である.それぞれの方法には独自の装置が開発されている.b. 塗装プラスチックの塗装(coating, painting)には図4・59に示したように下地の保護,美観の付与,特殊機能の付与という3種類の目的があるが,その内容をさらに詳しく示したのが表の大きな目的となりつつある.塗装には成形品の形状や量により刷毛塗り,エアスプレーコート,カーテンロールコート,ディップコート,静電塗装等の種々の方法が使用されており,それぞれ独自の装置が開発されている.c. 印刷(printing)プラスチックの成形品とフィルムで,印刷方法は相当異なっている.前者に対してはおもにシルクスクリーン印刷,ドライオフセット印刷,パッド印刷,インクジェット印刷等が使用され,それぞれ独自の装置が開発されている.またプラスチックフィルムの印刷には,おもにグラビア印刷,次いでフレクソ印刷が適用されているが,特に多色輪転機を用いるグラビア印刷には高速大量印刷を可能にする高性能の装置が開発されている.d. メタライジングメタライジング(metallizing)はプラスチックの表面を金属化する方法であり,化学めっき,蒸着,スパッタリング,イオンプレーティング等の方法がある.表面の金属化は美観,電導性,光反射性,耐候性,耐薬品性,ガスバリア性などの表4・2 プラスチック塗装の目的目的の種類内容下地の保護耐薬品性付与,きず防止,耐汚染性付与美観の付与着色,模様付け,光沢付与,表面平滑化,表面均質化,真珠光沢付与特殊機能の付与耐候性付与,帯電防止性付与,電磁波シールド,表面硬度アップ,耐摩耗性アップ,ガスバリア性アップ,防曇性付与図4・60 機能性表面処理技術1―88産業機械・装置機能を付与することができる.化学めっきは生成する大量の廃液の処理が大変であるが,膜厚が厚く密着性に優れた膜を得ることができる.他のドライ法では廃液問題はないが,蒸着は一般に蒸着金属の種類に制限があり,また膜の密着度が不十分となりがちである.PPやPET フィルムに対するアルミニウムの蒸着が代表的なものである.スパッタリングやイオンプレーティングは多種類の材料が使用可能で,強固で厚みの均一な金属薄膜を作ることができる.酸化インジウムベースの透明導電性膜,CD 面への金属薄膜など高度の製品が実用化されている.ただし設備費が相当高価になることは避けがたい.e. 表面処理(surface treatment)図4・59(B)に示したように各種の方法が実用化されている.特に表面硬化と帯電防止処理はプラスチックの二大欠点ともいうべき表面のきずのつきやすさ,およびほこりのつきやすさを改善するもので,多方面に活用されている.表面硬化処理は塗装による高機能化の一種であるが,その方法には大別して熱硬化法,紫外線硬化法(UV 硬化法),電子線硬化法(EB 硬化法)の3種類がある.特によく使用されるのは熱硬化法とUV 硬化法である.熱硬化法にはおもにオルガノシラン樹脂が用いられ,形成された三次元網状構造の硬化膜の硬度は無機ガラスに近いレベルにまで到達させることができるが,樹脂液の安定性や硬化時間(0.5~3時間)の点で欠点を有している.UV 硬化法に用いられる樹脂はおもに多官能アクリレートモノマおよびオリゴマが主成分であり,数分以下の短時間のUV 照射により硬化膜が形成される.しかしながらその硬度はなおオルガノシラン系硬化膜に劣り,耐候性も十分ではないことが多い.ただし,硬化時間が短いので設備の連続化が容易であり,大量生産には適している.塗布型の帯電防止処理も塗装による高性能化の一種である.一般にポリオキシエチレン誘導体を主体とする非イオン系およびアルキル4級アンモニウム塩を主体とするカチオン系の界面活性剤が帯電防止剤として使用されている.この処理により表面抵抗は10~10Ωにまで低下し,帯電防止機能が発現する.ただしこの方法の欠点は耐久性が低く,水洗や摩擦により比較的容易に表面層が消失し機能が低下するという点である.これに対し,練込み処理,永久帯電防止処理などの手法が開発されている.一般に表面処理による高機能化の方法は,材料自体の改質(ポリマ構造の変化やアロイ化,ブレンド化等)や多層化(共押出し,共射出成形,積層成形等)等の方法と競合することが多く,種々の状況を勘案して,より有利な方法が採用されている.文献(1)
