# 5・1・1 はじめに

> **章**: 第5章 製鉄機械
> **節**: 5・1 総論

世界の鉄鋼業は20世紀後半になり，生産技術，設備ともに飛躍
的に発展してきた．鉄鋼生産量（steel production）も急激な増加
を示し，1980年以降は年産7～8億t （粗鋼）の水準となっている．
日本では1973年に年間生産量1億2千万t （粗鋼）を記録したが，
1980年代に入って生産量はやや減少したものの，技術面では鉄鋼
製品の高品質化，製造コストの低減，省力化が積極的に進められ，
今日に至っている．図5･1に世界主要国の生産高推移，表5･1に
2000年の日本の生産高を鉄鋼製品別に示す．
鉄鋼製造プロセス（steel making process）は大別すると，鉄鉱
石から銑鉄を製造する製銑，溶銑または鉄くずから鋼を製造する製
鋼，鋼片から最終製品を製造する圧延，表面処理からなる．
鉄鋼製造プロセスは，1970年以降の製品の高品質化，製造コス
トの低減，生産能力の向上などの目的でプロセスも変革をとげ，特
に製鋼での連続鋳造設備の採用，圧延工程でのコンピュータ制御に
よる製造ライン制御，表面処理での表面加工技術の進歩は著しい．
図5･2に鉄鋼製造の代表的なプロセスを示す．
5・1・2鉄鋼製造プロセス
a.製銑（ironmaking) 製銑プロセスは銑鉄を製造する高炉，
高炉に装入する原料の焼結鉱を製造する焼結設備，コークスを製造
するコークス炉，および鉄鉱石，石炭などの原料を貯蔵，配合，粉
砕を行う原料処理設備からなる．焼結機は粉鉱石と石灰石などの副
原料を配合混合したものを焼結して焼結鉱を製造する．コークス炉

は強粘結炭と弱粘結炭を配合し，1100～1300℃の高温で乾留して
コークスを製造する．高炉は焼結鉱，コークス，石灰石などを装入
し，1200～1300℃の熱風炉で加熱した空気を吹込みコークスを還
元剤として銑鉄を製造する．
製造プロセス，設備での最近の動向は，高炉炉頂圧発電，熱風炉
廃熱回収，焼結機のクーラ廃熱回収，廃熱ボイラ，コークス炉の乾
式消火方式による赤熱コークス顕熱回収等の省エネルギーの徹底，
ダストリサイクル設備による高炉ダスト，転炉ダストの高炉原料と
しての再利用，廃プラスチックの高炉，コークス炉での有効利用な
どが挙げられる．高炉は操業技術の進歩によりコークス消費量が減
少し製造コストの低減に寄与している．さらに装入するコークスの
一部を石炭のまま高炉に直接吹込む微粉炭吹込み技術が広く採用さ
れ，必要コークス量の30～40 %を占めている．また高炉設備は内
張りれんがの張替え，炉体冷却装置の取替えなどのため稼働10年
前後で改修を行うが，稼働期間延長のための操業技術，設備の長寿
命化の改善が著しい．
しており，このうち高炉内容積が4000m以上の大型高炉が16基
あり，設備の大型化が進んでいる．
b.製鋼（steelmaking) 製鋼は高炉で製造された銑鉄（溶
銑）から鋼を精錬する転炉，鉄くずを熔解して鋼を製造する電気
炉，転炉または電気炉で製造された鋼（溶鋼）から直接スラブ，ブ
ルーム，ビレットを製造する連続鋳造設備が主要プロセスである．
最近は高級鋼製造のため転炉精錬の前工程で脱りん，脱硫を行う溶
銑予備処理，転炉精錬後脱ガス，スラグ精錬，かくはん，成分調整
などを行う二次精錬が普及し，この結果溶銑予備処理→転炉→二次
精錬→連続鋳造のプロセスが一般的になった．
転炉には上吹き炉，底吹き炉，上底吹き炉の3方式がある．当初
はすべて上吹き炉であったが，上吹き炉の酸化ロスが多い欠点，底
吹き炉の脱りん効果が悪い欠点を補う上底吹き複合精錬を行う上底
吹き炉が多くなっている．
連続鋳造は生産性，高歩留り，省エネルギー，材質の均一性など

c.圧延（rolling）および表面処理（surface treatment） 連
続鋳造または分塊圧延で製造された鋼片（スラブ，ブルーム，ビレ
ット）から最終製品を製造するプロセスは，図5･2に示すように製
品により多岐にわたっている．代表的なものとして条鋼，線材，厚
板，薄板製造の各圧延プロセスと鋼管製造プロセスがある．
条鋼，線材は製品と同一断面形状になるような孔型を2本の回転
するロール間に作って，加熱された鋼片をこの間に通して圧延す
る．鋼片から一足飛びに複雑な形状の製品にはならないため，この
種の圧延機を数台横に並べ多段圧延を行う．最近は生産性向上のた
め圧延機を縦列に並べ連続的に鋼片から製品を作る連続圧延方法が
主流になっている．
厚板は厚さ3mm以上の熱間圧延鋼板をいい，2本のロールの間
に加熱された鋼片を通し，次第にロールの間隔をせばめながら何回
も往復圧延して所定のサイズに仕上げる．
