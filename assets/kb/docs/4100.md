# 2・1・3 鉱油系作動油

> **章**: 第2章 油圧システム
> **節**: 2・1 油圧作動油

鉱油系作動油は耐摩耗性を有するか否かで，一般作動油（R&O
型）と耐摩耗性作動油に分かれる．油圧システムには必ず高圧を発
生する油圧ポンプがあるため，耐摩耗性作動油が一般に用いられる
が，圧力が低い場合，または油圧ポンプがピストンポンプなどで耐
摩耗性が必ずしも要求されない場合には，一般作動油が用いられ
る．そのほか，建設機械のように屋外で使用される場合，または冬
季に低温になる場合などは，起動時の運転を容易にするため高粘度
指数作動油を使用するほうが好ましい．
