# 5・1・2 鉄鋼製造プロセス

> **章**: 第5章 製鉄機械
> **節**: 5・1 総論

a. 製銑(ironmaking)製銑プロセスは銑鉄を製造する高炉、高炉に装入する原料の焼結鉱を製造する焼結設備、コークスを製造するコークス炉、および鉄鉱石、石炭などの原料を貯蔵、配合、粉砕を行う原料処理設備からなる。焼結機は粉鉱石と石灰石などの副原料を配合混合したものを焼結して焼結鉱を製造する。コークス炉図5・1 世界と主要国の粗鋼生産高の推移1―89は強粘結炭と弱粘結炭を配合し、1100~1300°Cの高温で乾留してコークスを製造する。高炉は焼結鉱、コークス、石灰石などを装入し、1200~1300°Cの熱風炉で加熱した空気を吹込みコークスを還元剤として銑鉄を製造する。製造プロセス、設備での最近の動向は、高炉炉頂圧発電、熱風炉廃熱回収、焼結機のクーラ廃熱回収、廃熱ボイラ、コークス炉の乾式消火方式による赤熱コークス顕熱回収等の省エネルギーの徹底、ダストリサイクル設備による高炉ダスト、転炉ダストの高炉原料としての再利用、廃プラスチックの高炉、コークス炉での有効利用などが挙げられる。高炉は操業技術の進歩によりコークス消費量が減少し製造コストの低減に寄与している。さらに装入するコークスの一部を石炭のまま高炉に直接吹込む微粉炭吹込み技術が広く採用され、必要コークス量の30~40%を占めている。また高炉設備は内張りれんがの張替え、炉体冷却装置の取替えなどのため稼働10年前後で改修を行うが、稼働期間延長のための操業技術、設備の長寿命化の改善が著しい。設備設置状況は、2001年12月末現在日本で31基の高炉が稼働しており、このうち高炉内容積が4000m以上の大型高炉が16基あり、設備の大型化が進んでいる。b. 製鋼(steelmaking)製鋼は高炉で製造された銑鉄(溶銑)から鋼を精錬する転炉、鉄くずを熔解して鋼を製造する電気炉、転炉または電気炉で製造された鋼(溶鋼)から直接スラブ、ブルーム、ビレットを製造する連続鋳造設備が主要プロセスである。最近は高級鋼製造のため転炉精錬の前工程で脱りん、脱硫を行う溶銑予備処理、転炉精錬後脱ガス、スラグ精錬、かくはん、成分調整などを行う二次精錬が普及し、この結果溶銑予備処理→転炉→二次精錬→連続鋳造のプロセスが一般的になった。転炉には上吹き炉、底吹き炉、上底吹き炉の3方式がある。当初はすべて上吹き炉であったが、上吹き炉の酸化ロスが多い欠点、底吹き炉の脱りん効果が悪い欠点を補う上底吹き複合精錬を行う上底吹き炉が多くなっている。連続鋳造は生産性、高歩留り、省エネルギー、材質の均一性など表5・1 bloom-ing)に代わって採用が進められた。1965年から小規模設備でテストランが開始され、1970年代に入って能力100~300t/hの本格的な大規模設備が設置された。2000年には平均99.7%(日本)の連鋳比率となっている。さらなる生産性向上のため連々鋳技術、高速連続鋳造技術、プロセス異常予知技術などの開発が進んでいる。c. 圧延(rolling)および表面処理(surface treatment)連続鋳造または分塊圧延で製造された鋼片(スラブ、ブルーム、ビレット)から最終製品を製造するプロセスは、図5・2に示すように製品により多岐にわたっている。代表的なものとして条鋼、線材、厚板、薄板製造の各圧延プロセスと鋼管製造プロセスがある。条鋼、線材は製品と同一断面形状になるような孔型を2本の回転するロール間に作って、加熱された鋼片をこの間に通して圧延する。鋼片から一足飛びに複雑な形状の製品にはならないため、この種の圧延機を数台横に並べ多段圧延を行う。最近は生産性向上のため圧延機を縦列に並べ連続的に鋼片から製品を作る連続圧延方法が主流になっている。厚板は厚さ3mm以上の熱間圧延鋼板をいい、2本のロールの間に加熱された鋼片を通し、次第にロールの間隔をせばめながら何回も往復圧延して所定のサイズに仕上げる。コイル状に巻かれた長尺の熱間圧延鋼帯は連続式熱間圧延機で製造され、厚みは1.0~25mm程度まで可能である。通常はスラブを連続式加熱炉で加熱し、表面のスケールを取り除いた後、粗圧延機により25~50mm程度の厚さまで圧延し、6~7台よりなる連続仕上げ圧延機で所定の厚みまで圧延された後、コイラで巻き取られて製造される。冷間圧延鋼帯は、熱間圧延鋼帯(熱延コイル)を連続酸洗装置で硫酸または塩酸で洗浄し表面スケールを除去した後、常温のまま5~6台の連続圧延機を通して製造する。圧延後電気清浄装置によって表面を洗浄し650~750°Cの焼なまし炉で焼なましをした後に調質圧延を行って材質を均一化する。薄板製造はこのような連続式圧延機が主流であり、最近では六重式圧延機、ペアクロスミル等、幅方向の厚さを均一化する技術が採用されている。このほか1台の可逆式圧延機で往復圧延するレバーシングミル、多数の小径ロールを配置したゼンジミアミルがあり、用途に応じて採用されている。鋼管製造プロセスの主要なものを図5・3に示す。製法別に分類すると、継ぎ目なし鋼管と溶鍛接鋼管に大別され、さらに溶鍛接鋼管は製法により電縫鋼管、スパイラル鋼管、UOプレス鋼管、ガス溶接鋼管、鍛接鋼管に分かれる。最近は溶接鋼管製造技術の進歩により鋼板や鋼帯を素材とする溶接鋼管の比率が高まり70%前後を占めている。自動車用、電機製品用、建材用などの鋼板は用途に応じて表面処理加工を行う。表面処理の目的は防錆、耐食性、表面光沢、表面色彩などがある。表面処理鋼板には、亜鉛めっき鋼板、合金化溶融亜鉛めっき鋼板、亜鉛系合金電気めっき鋼板、Al-Zn合金亜鉛めっき鋼板、アルミめっき鋼板、ターン合金めっき鋼板、塗装鋼板、樹脂ラミネート鋼板、ブリキ、ティンフリースティールなどがある。これらの表面処理鋼板を製造するプロセスは、大別すると、溶融めっき、電気めっき、塗装、ラミネートのプロセスとなる。圧延プロセスの最近の技術動向は省エネルギー、省力、歩留り向上のための製鋼から製品に至る各プロセスの連続化、直送化である。連続化、直送化の一つは連続鋳造と熱間圧延プロセスの直送技術で、連続鋳造で製造されたスラブの顕熱を利用して加熱炉を通さない直送圧延と、不足温度を加熱炉で補う熱片装入の二とおりが行われている。薄板の熱間圧延では、仕上圧延機の直前に溶接機を設け、粗圧延バーを熱間溶接して連続仕上げ圧延を行う技術も導入され、歩留り、品質、生産能率の向上に寄与している。冷延薄鋼板の1―9
