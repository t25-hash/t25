# 5・2・2 直接還元設備

> **章**: 第5章 製鉄機械
> **節**: 5・8 )とがあり,おもに外燃式熱風炉が高温送風に採用されている.

鉄鉱石を固体状態で還元し金属化率が通常90~95%の還元鉄を図5・9天然ガスを利用する還元設備図5・10 石炭を利用する還元設備製造する方法を直接還元製鉄法(direct reduction)と称し,製品はおもに電気炉のスクラップ代替として使われる.2000年までに建設された工場は全世界に146基があり,生産能力は約5700万t/yに達する.直接還元製鉄法は,還元炉外で天然ガス等を改質して製造した還元ガスを使用するものと石炭等を還元炉内で直接利用する方法に2分類される.還元炉の形式には,天然ガスを利用するものとして固定層炉,シャフト炉および流動層炉方式が,石炭を利用するものとしてシャフト炉,ロータリキルンおよび回転炉床炉(rotaryhearth furnace)がある.図5・9および図5・10に還元設備の代表例を示す.
