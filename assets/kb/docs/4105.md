# 5・3・2 アーク炉

> **章**: 第5章 製鉄機械
> **節**: 5・3 製鋼設備

アーク炉（electric arc furnace）は，炉内に装入した鉄くず（ま
熱により鉄くずを溶解・精練し鋼を生産する製鋼炉である．鋼の品
質や生産性の向上を目的として，特殊鋼分野のみならず普通鋼にも
炉外精練炉が導入され，アーク炉の役割は，鉄くずの溶解専用炉と
なる傾向にある．
製鋼用アーク炉は，三相交流アーク炉（three phase AC arc fur-
nace）と直流アーク炉（DC arc furnace）がある．当初，三相交
流アーク炉のみであったが，1988年国内初の直流アーク炉が稼動
して以来，そのメリットが評価され，その後，建設されたアーク炉
の多くは直流アーク炉となった．直流アーク炉は，電極が1本であ
るため電極が3本の三相交流アーク炉より電極の消耗量が少ないと
いうメリットがある．それよりも無効電力変動量に依存するフリッ
カ発生量が50 %以下になり電源障害が低減される．それにより電
源制約は緩和され，電源増強が可能になった．その結果，生産の向
上を目指した大型の直流アーク炉が数多く建設された．
