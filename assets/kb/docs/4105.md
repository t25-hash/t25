# 5・3・2 アーク炉

> **章**: 第5章 製鉄機械
> **節**: 5・3 製鋼設備

アーク炉(electric arc furnace)は,炉内に装入した鉄くず(または,還元鉄)と黒鉛電極との間にアークを発生させ,そのアーク熱により鉄くずを溶解・精練し鋼を生産する製鋼炉である.鋼の品質や生産性の向上を目的として,特殊鋼分野のみならず普通鋼にも炉外精練炉が導入され,アーク炉の役割は,鉄くずの溶解専用炉となる傾向にある.製鋼用アーク炉は,三相交流アーク炉(three phase AC arc fur-nace)と直流アーク炉(DC arc furnace)がある.当初,三相交流アーク炉のみであったが,1988年国内初の直流アーク炉が稼動して以来,そのメリットが評価され,その後,建設されたアーク炉の多くは直流アーク炉となった.直流アーク炉は,電極が1本であるため電極が3本の三相交流アーク炉より電極の消耗量が少ないと図5・15 OG 式排ガス処理設備いうメリットがある.それよりも無効電力変動量に依存するフリッカ発生量が50%以下になり電源障害が低減される.それにより電源制約は緩和され,電源増強が可能になった.その結果,生産の向上を目指した大型の直流アーク炉が数多く建設された.直流アーク炉および三相交流アーク炉の構造図を図5・16,図アーク炉は,鉄くずのかさ比重が0.6程度と小さいために1ヒート当たり2~3回に分けて炉上部から鉄くずを専用のバケットを使用して炉内に装入する.そのために炉ぶたを昇降・旋回できる構造になっている.炉体の炉内側には溶鋼と接触しないほとんどの範囲で水冷構造物が設置され,炉内高温雰囲気から炉体を保護している.それ以外の部分は耐火物構造となっている.また,炉ぶたもまた同様に水冷されている.溶鋼は,傾動装置により炉体を傾けながら炉側に取付けられた出鋼樋を通してレードルへ注がれる.炉外精練の普及とともに炉底に出鋼口を設けスラグフリー出鋼を目的とする偏心炉底出鋼(eccen-tric bottom tapping)方式が普及している.炉体側部には,出鋼口のほかに作業口が設けられている.スラグの排出や鉄くずのカッティング,脱炭のための酸素吹精,酸化鉄を還元するためのカーボンインジェクションに使用される.直流アーク炉の給電設備は,大電流を供給できる炉用変圧器,直流に変換するためのサイリスタ,直流リアクトル,二次側導体,陰図5・16 三相交流アーク炉図5・17 直流アーク炉(1)ビレットCCM80□~220□線材,コンクリートバー(2)ブルームCCM220□以上長方形もある鋼管,型鋼,棒鋼火造り材(3)薄厚スラブCCM厚み40~70幅790~1680熱延鋼板(4)中厚スラブCCM厚み70~150幅740~2591熱延鋼板(5)スラブCCM厚み150~300幅800~2700熱延,冷延鋼板ステンレス鋼板(6)ラウンドCCMφ150~450鋼管,輪(7)ビームブランクCCMH 型鋼(8)ストリップキャスタ2~5熱延,ステンレス鋼板1―9 5極側水冷ケーブル,陰極導体,電極クランプおよび黒鉛電極,炉底電極,陽極導体,陽極側水冷ケーブルから構成されている.溶解中の炉内環境により生じるアーク電力の変動に対処するために電極と一体の電極支腕を昇降させ,アーク電力を制御する電極昇降機構を含む電極自動制御設備がある.電極昇降機構は,モータと減速機からなる電動式と油圧シリンダで支柱を直接的に昇降する油圧式の二とおりがある.省エネルギーの観点からは,排ガスによって持ち出される入熱の20~30%に相当するエネルギーにより装入される鉄くずを600~1000°Cに予熱することで回収することを目的とするシャフト型アーク炉が稼動している.これは,電力の原単位に換算して100~200kWh/t低減をもたらすことになる.環境面においては工場内の作業環境や工場周辺の環境改善が進められている.さらに,都市ゴミ焼却炉と同様に製鋼用アーク炉にもダイオキシン排出規制が適用され,それに対応すべく設備改造が検討されている.連続鋳造設備(continuous casting machine)とは,鋼板,形鋼,棒鋼や線材および火造り品などの製品を作る素材を溶融金属から連続的に鋳造し凝固鋳片を作る設備をいい,通称CCマシン,または略して連鋳機という.連続鋳造法の発想は古く,1850年代にH.ベッセマー(H. Bes-semer)が2個の水冷ロール間に溶鋼を流し込んで,直接製品に近い薄鋼板を製造しようとしたのが始まりとされている.この試みは成功しなかったが,その後多くの開拓者の手によりAlやCu合金などの圧延板の素材を作る連続鋳造法が開発された.しかし,この方法が鋼の鋳造に適用され工業化の基礎を作ったのは,1933年にユンハンス(Junghans)が水冷銅鋳型を上下に往復運動を与え,潤滑剤の使用と併せて焼付き防止の方法を開発したのが始まりとされている.1960年代までの製鋼から圧延製品を作る工程は,鋼板を例にとると,溶解精錬→造塊→均熱→分塊圧延→加熱→熱間圧延であった.連鋳法によると,溶解精錬→連続鋳造→加熱または均熱→熱間圧延となり,設備費,材料歩留り,工程短縮,エネルギーの各面で大幅な合理化となった.
