# 5・4・3 設計基本条件とおもな設計諸元

> **章**: 第5章 製鉄機械
> **節**: 5・4 連続鋳造設備

a. 基本条件マシン設計の前提条件としては、転炉または電気炉からの1回の出鋼量()と供給ピッチ、鋳造鋼種、鋳片寸法および生産能力などがある。b. おもな設計諸元機器設計に先立って次の主要諸元を設計する。すなわち(1)マシン形状、(2)鋳込速度(m/min)、(3)ストランド(strand)数、(4)鋳片パスラインプロフィール、または湾曲半径、(5)ガイドロール径とピッチ、(6)凝固完了長さ(metallurgical length)、(7)機械全高と機長、などがある。マシン形式は鋳造鋼種、鋳片断面形状と寸法および生産性の面から選定する。鋳込速度(casting speed)には上限と下限値があり、鋳片サイズと鋼種によって決める。表5・3に鋳込速度の参考値を示す。マシンの能力は鋳込速度とストランド数によって決まるが、これらは連続操業をするために溶鋼の供給ピッチ(min)と鋳込処理時間表5・3 スラブ、ブルームの鋳込速度分類鋳片寸法mm最大鋳込速度m/minスラブ、薄板用250×1000~15001.5~2.5スラブ、厚板用250×1500~20000.7~1.0特殊鋼ブルーム200~3000.45~1.2スラブ(中厚CC)90×900~16503.0~5.0がマッチするように決める必要がある。マシンプロフィールは鋳片が通過するパスラインであり、機械形式との関係が大きい。湾曲型と垂直-曲げ型のマシンでは、直線と円弧の変曲点で鋳片断面に曲げひずみが発生する。高温状態の鋳片はあるひずみ速度を超える曲げ変形を与えると割れが発生するので、半径の値と曲げまたは矯正域でのパスライン曲線の設計は極めて重要である。多点矯正の場合の鋳片断面の矯正ひずみεは次式によって与えられる。割れ発生の危険な領域は固液界面近傍とされており、許容ε値は専門分野によることとして、実際の設計では0.1~0.25%が採用されている。マシン半径は一般に鋳片厚みの35~45倍にとられている。次に機長を決定する場合、鋳型内湯面から鋳片が完全凝固するまでの長さを算定することが基礎となる。この長さを凝固完了長さといい次式で求める。図5・19の引抜き矯正装置の終端がほぼ(m)の位置に相当する。c. ロールピッチロールレイアウトの設計はパスライン設計とともに最も重要な基本設計である。スラブ連鋳機の場合、未凝固部をもつ鋳片は静鉄圧(ferrostatic pressure)によってバルジング(bulging)現象を起こすので、パスラインに沿って(m)全域にわたり鋳片支持のガイドロールを配置する。バルジングによるシェルのひずみを小さくするためにロールピッチやロール径は小さいほど望ましい。反面ロールは静鉄圧、矯正反力および熱応力などの外力に耐える剛性も必要である。実機におけるロール径は150~350mmの間にある。ロール間にはスプレーノズルを配置するので最小30mmの間隔をとる。バルジングひずみの計算には多くの発表がなされており、設計値としては0.15~0.3%が採用されるが、矯正部ではそれによるひずみが加わるので注意を要する。
