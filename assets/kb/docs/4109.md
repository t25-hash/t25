motor ）は，建設機械，舶用機械などの各種アクチュエータとして
広く用いられている．構造的には，特に低速用に作られたアキシア
ルピストン形，偏心形（モノストローク形）とカム形（多行程形）
のラジアルピストン形モータなど，油圧力による回転トルクを直接
出力軸に伝えるものと，差動機構（diﬀerential mechanism）を利

a.アキシアルピストン形低速高トルクモータ 斜板式ピスト
ンモータを特に低速用に製作したもので，図Ⅴ-2･26に示すような
分配弁や球面弁板を採用し，比較的大きなピストン径を用いて高ト
ルクを得られるようにしている．斜板を対向に使用してピストン数
を2倍にしたもの，また，斜板傾角を二，三段階に切換えることが
できるものもある．
b.偏心形ラジアルピストンモータ（radial piston motor )
図Ⅴ-2･27に示すように星形に配列された数本のシリンダの中でピ
ストンが往復運動し，これを偏心カムにより回転運動に変えて低速
高トルクを得る構造のもので，構造が簡単で部品数が少なく取扱い
が容易である．なお，カムの偏心量を外部からの油圧で変えて二段
階，あるいは無段階に変速できるもの，また，星形の配列を軸方向
に2列配置したものも製品化されている．
c .多行程形ラジアルピストンモータ 多行程形ラジアルピス
トンモータは，図Ⅴ-2･28に示すようにシリンダブロックに対応す
るカムハウジングの内周に波状のカムを配し，これに数本のピスト
ンの油圧力が同時に作用して回転力を発生させる構造になってい
る．一般にこの形式は，原理的に次のような構造ならびに性能上の
特徴を有する．

(４)変速用の分配弁を使用することにより有効に作用する山数を

(５)ピストンをカム面から遊離させることにより，フリーホイー

なお，このモータでは複雑な曲率のカム面にピストン推力を伝え
なければならないので，ピストン先端には転がり機構が必要であ
り，多種のピストン機構が用いられている．
d.多行程形ベーンモータ（vane motor ) 多行程形ベーンモ
ータは，ロータ直径を大としてトルクの作用半径をできるだけ大き
くとり，ロータに対応したカムリングの内周に波状のカムを配し，
供給，排出口をそれぞれ3～4個もつ．1回転でベーンは3～4回ス
トロークするが，多行程形ラジアルピストンモータと同様に，負荷
に応じて圧力油を送る供給口の数を変えるなど，2速，3速の切換
えも可能である．
e.多行程形アキシアルピストンモータ（axial piston motor )
多行程形アキシアルピストンモータは，カムと接触，転動する球が
ピストン先端に自由に回転可能に取付けられたピストンを用い，こ
のカムのしゅう動面には駆動軸方向にほぼ正弦波状となる突起を
2～3個設けた多行程形であり，占有容積に比べて大きなトルクを
得ることができる．
f .差動減速式内接歯車モータ（gear motor ) 差動減速式内
接歯車モータは，図Ⅴ-2･29に示すように内接歯車をケーシングに
固定し，歯数が1枚少ない外接歯車をその中で油圧により転動さ
せ，この歯車と出力軸とを特殊な軸継手でつなぐことにより差動減
速機の作用を行わせ，低速で大トルクの出力を得る形式のものであ
る．

g.減速機付きモータ（reduction gear combined motor ) 斜
軸式あるいは斜板式の高速モータと，二段あるいは三段の遊星減速
機や差動歯車式減速機とを組合わせて低速高トルクを得る構造のも
ので，減速機部分を変えることによりモータ部分を変えることなく
種々の高トルクを得ることができる．また，斜板の傾角を二段階に
変化させ，容積を変えるものもある．図Ⅴ-2･30に示すように，モ
ータのケーシングと減速機のハウジングを共通にして，小型・軽量
化を図ったものもある．
