# 5・4・4 主要機器の構造

> **章**: 第5章 製鉄機械
> **節**: 5・4 連続鋳造設備

a. 取りべ支持の形式図5・19に示すようなタレット式と、ほかに走行台車式や固定スタンドによる支持方式がある。装置内には計重装置やなべ昇降装置が含まれることがある。b. タンディッシュおよび同台車タンディッシュ(tundish)は耐火物で内張りされた中間容器であり、取りべからの溶鋼を溜める。容量は一般的に1分間当たりの鋳込量の5~10倍にとられる。底部には耐火物ノズルがセットされてあり、ストッパやスライディングノズル装置を介して内部の溶鋼は鋳型内に注入される。c. 鋳型(mold)大型スラブ用鋳型は、四辺が別々の水冷銅板の組立構造であり、小型ビレット鋳型は一体のチューブラ型を使用する。鋳片厚が一定で幅のみ可変の鋳型はよく使用され一例を図表面はクロムやニッケルまたはコバルトめっきが施工される。鋳型高さは700~1000mmであり、溶鋼面は上面より約100mm下方に設定され、鋳込中はレベル計とノズル開度制御により設定位置が保たれる。銅板背面は水冷されており、その流速は6~9m/sである。d. 鋳型オシレーション(mold oscillation)鋳型は操業中、常時パスラインに沿った軌道を上下運動する。運動の波形は、通常サイン波形であって、ストローク4~15mm、サイクル数は、100~200回/分が多い。この両数値は鋳込速度の関数として決められる。構造は、偏心カム運動を特殊なリンク機構をもつレバーを介して鋳型に伝える方法と、鋳造中に波形、ストロークおよびサイクル数が変更可能な油圧シリンダを動力とする構造とがある。この装置による鋳型の振動は、伝達機構のばね系のなかで共振を起こさない剛性をもつように設計する注意が必要である。e. ガイドセグメントスラブ用連鋳機では鋳型の出口より完1―9 7図5・21 ガイドロールセグメント全凝固長さの末端まで、パスラインに沿ってガイドロールを配置する。図5・21に示すごとく一般には数対のロールをセグメント構造としてベースフレーム上に連接して固定する。ロール間には多数のスプレー冷却ノズルが組込まれており、流量制御の配管系に連結していて鋳片を冷却する。ロール材質は強度のほかに耐摩耗と耐熱き裂性に富む鍛造材が使用され、かつ、表面は高合金の肉盛りをして使う場合が多い。広幅スラブ用のロールはたわみを低くとるために、軸方向で2~4分割する。また、中央部の軸受は熱や水を考慮した設計が必要である。異なった厚みの鋳片を鋳造する場合は各セグメントの上ロールをフレームごと昇降させてロール間隔を調整する。ロール間に異常な外力が負荷された場合、皿ばねのたわみで上フレーム全体が逃げる構造もある。f. 矯正セグメント湾曲形などのマシンの矯正域では、駆動ロールを配置したセグメント構造を有し、引抜き矯正反力にも耐える十分な剛性をもった設計とする。すべてのガイドロールは内部水冷構造である。g. ダミーバー装置ダミーバーは、鋳造の始めに底なし鋳型の底部をプラグし、かつ、鋳片をクランプして下方へ引抜いてゆく案内バーの役目をもつ。したがって湾曲形のマシンではチェーンリンク構造となっており、鋳片と連結する頭部は特殊なフック形状か、鋳ぐるみ突起物を有している。挿入形式にはマシン底部からの形式と頭部より装入する場合とがあり、それによってマシン構造は大幅に異なる。h. その他の設備マシンの下流には鋳片切断装置、刻印機、切断バリ取り機、計重装置などが設置される。上流側の周辺機器に溶鋼シール装置、鋳型パウダ供給装置、鋳片未凝固相の電磁かくはん装置などがある。
