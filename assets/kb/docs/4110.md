# 2・2・7 油圧ポンプ・モータの騒音

> **章**: 第2章 油圧システム
> **節**: 2・2 油圧ポンプ・モータ

油圧ポンプ・モータは作動中に，空気伝ぱ騒音（air - borne
noise），流体伝ぱ振動（fluid-borne vibration），および構造伝ぱ
振動（structure-borne vibration）の三つの形態の振動エネルギー
を発生する．このうち油圧ポンプ・モータの騒音は，狭義にはポン
プから直接放射される空気伝ぱ騒音を意味する．ここでは，直接放
射騒音についてのみ記す．ただし，大型の油圧装置においては，油
圧ポンプの吐出し流量脈動と管路内作動油の脈動伝達特性との相互
作用によって発生する圧力脈動に起因する流体伝ぱ騒音が，全騒音
の大部分を占めるようになる．この油圧ポンプの吐出し流量脈動
（その測定法，シミュレーション法など）に関しては文献(15)～
(18)を参考されたい．
