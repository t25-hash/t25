# 5・5・2 分塊圧延の主要機器

> **章**: 第5章 製鉄機械
> **節**: 5・5 分塊圧延設備

分塊圧延設備には均熱炉、分塊圧延機、せん断機、スカーファが設置され、ビレットを生産する設備では鋼片圧延機が加わる。a. 均熱炉(soaking pit)製鋼工場でインゴットケースから型抜きされた鋼塊は、この均熱炉に送られ分塊圧延に適した温度まで均一に加熱される。均熱炉はすべてバッチ式で1基当たり20~85t/hの能力の炉が数基設置されている。燃料は高炉ガスとコークスガスの混合ガスを使用するものが多く重油燃焼のものもある。b. 分塊圧延機(slabbing mill,blooming mill)生産量が少ない設備ではスラブとブルーム圧延兼用のハイリフト二重逆転式圧延機が使用される。大型化、高能率化、自動化が要求されるスラブ圧延では水平ロール機に垂直ロール機を組合わせたユニバーサルミルが多く用いられる。生産能力は600~1000t/hに及び、大型化に伴い主電動機は上下ロールを別々に駆動するツインドライブ方式が多い。分塊圧延機の前後にはローラテーブルとマニプレータを設置して圧延材の搬送と転倒を行う。図5・22にスラビングミルの全体図を示す。c. せん断機(shear)スラブまたはブルームを所定の長さに切断するため、大容量のせん断機が設置される。せん断形式としてはスラブ切断ではダウンカット式が用いられ、ブルーム切断ではアップアンドダウンカット式が用いられる場合もある。刃物の駆動は電動式と油圧式とがあり、せん断力はスラブシャーにて25~30MN 程度となる。d. 鋼片圧延機(billet mill)小断面のビレットを作る場合、二重逆転式の分塊圧延機のみで圧延することは極めて非能率であるため、180~230mmのブルームで分塊圧延を終え、引続き鋼片圧延機を通して100mm角程度のビレットを生産する。この圧延機は水平垂直の各ロール単独駆動式のもので4~6基連続に設置され、材料を捻転なしに圧延する。さらにこの連続圧延機の下流側には走間せん断機が設けられ、切断されたビレットは冷却床に送られて冷やされる。生産された角または丸ビレットは次工程の棒鋼ミル、線材ミルに送られるほかに継目なし鋼管の素材にもなる。e. ホットスカーファ(hot scarfer)鋼片の表面傷を除去するための装置で、酸素およびプロパンガスにより鋼片が通過中に上図5・22 スラビングミル圧延機の種類ロール形式ロール回転ロール寸法ミルモータ鋼塊鋼片下と側面の4面を同時に薄く溶削する。歩留り低下の欠点があるが手作業に比し能率は高い。表5・4に分塊圧延機の代表的な例を示す。板厚6mm以上の熱間圧延鋼板を厚板(plate)と呼ぶ。厚板圧延機(plate mill)では、通常、板厚4.5~300mm、板幅は約1~5.4m、板長は最大55mまで圧延可能である。厚板製品は、船舶、橋りょう、圧力容器、大径鋼管をはじめ、一般構造物および海洋構造物など鋼構造物の重要部材に広く使用される。品質面でも時代の要求に応じて、低温用鋼板、溶接構造用調質高張力鋼板などが製造できる。わが国の厚板圧延設備は1970年代半ば以降、能率および品質向上を目的とし、設備の大形化、高速化、自動化が図られた(図5・23)。1990年代以降は高精度高効率のための設備改造が主となった。
