a.分類 油圧揺動形アクチュエータ（oscillating actuator ，
semi rotary actuator ）とは，流体（油圧）のエネルギーを用い
て，出力軸の回転往復運動を行わせる機械のことである．揺動形ア
クチュエータは，ベーン形とピストン形に分類される．ベーン形
は，油圧を受けて回転ベーンがある一定角度回転する構造である．
ピストン形は，油圧シリンダのピストンの動きを，内蔵したラック
ピニオンやヘリカルスプライン，クランク機構などを用いて回転軸
の揺動運動に変換する構造である．揺動形アクチュエータを使用す
る場合，駆動物の大きな慣性力をアクチュエータ自身で停止させる
と，その衝撃によってアクチュエータが破損することがあるので，
外部にストッパを設ける．また，著しく慣性力が大きい場合には，
外部にショックアブソーバを使用し，緩やかに停止させるのが望ま
しい．

.スクリュー形揺動アクチュエータ（screw type rotary
actuator ) スクリュー形揺動アクチュエータは，図Ⅴ-2･42に
示すように，回転軸に加工されたリードの長いスクリュー（ヘリカ
ルスプライン）に沿って，回転が規制されているピストンの直線運
動により軸が回転運動する構造である．スクリューおよび回り止め
ガイドの摩擦損失によって効率は低いが，外力によって逆回転しに
くく，回転角度は360°を超えることが許される．揺動角はスクリ
ューのリード角と長さにより決まる．
.ラックピニオン形揺動アクチュエータ（rack and pinion
type rotary actuator ) ラックピニオン形揺動アクチュエータ
は，図Ⅴ-2･43に示すように，ピストンの往復運動を，回転軸に直
結されたラックピニオン歯車機構により回転運動に変換する．歯車
の圧力角による反力を受ける構造が十分であれば，摩擦損失が少な
いので効率は高い．回転角度はシリンダのストロークとラック歯車
の長さにより決まる．

付けたスコッチヨーク機構により，揺動運動に変換する構造であ
る．スコッチヨーク形揺動アクチュエータの出力トルク （kN･
cm）は，ピストンの有効断面積 （cm ），回転軸心と駆動ピンま
での距離 （cm），スコッチヨークが回転軸と作る角度θ，供給圧
力 （MPa），背圧 （MPa）とすれば，
＝0. 1－cosθ
となり，スコッチヨークが駆動軸の運動方向に対して直角となった
とき，出力トルクは最小となる．このため，起動トルクや締付けト
ルクを必要とする弁の開閉などによく使用される．
.チェーン形揺動アクチュエータ（chain type rotary
actuator ) チェーン形揺動アクチュエータは，シリンダピスト
ンにチェーンを連結し，出力軸に設けたスプロケットを揺動させる
構造である．揺動角はピストンストロークとスプロケットのピッチ
円半径により決まる．出力トルクはチェーンの強度により制限され
る．
