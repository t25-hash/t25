# 5・6・1 加熱炉設備

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

加熱炉(furnace)は鋼片を圧延可能な温度まで加熱する設備であり、その温度は鋼種ごとに異なり通常900~1250°Cと広範囲にわたる。加熱炉には、省エネルギー(燃料消費量の最少化)、品質の高度化(加熱温度の均一性)、環境保全(NO等排出量の最少化)、生産性向上、省力化(保守・点検の容易さ)に関わる工夫・技術が多く採用されている。炉の能力は一基当たり100~300t/hで、厚板工場能力に合わせて数基設置される。炉前のテーブルまで搬送された鋼片はプッシャまたはチャージャで炉に装入され、油圧または電動式のウォーキングビームで炉内を搬送され、エキストラクタで抽出される。これら一連の搬送は、電算機を用いた自動搬送制御で行われる。炉内の鋼片は通常二列操業であり、炉幅は8~12m、炉長は30~50mである。鋼片の昇温率は鋼片厚さ1mm当たり0.7~0.9分程度である。燃料は重油、ガス(コークス炉ガス、混合ガス、液化石油ガスなど)が使用され、上下複数帯に区分されて配置された複数バーナおよび自動燃焼制御により鋼片の加熱を行う。また鋼片内部の温度分布や昇温パターンを数式モデルを用いて電算機で計算させ、省エネルギーや品質の高度化を図っている(図5・24)。
