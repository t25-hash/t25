# 5・6・2 圧延機

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

a. 概要厚板圧延はその特徴として胴長が大きく,圧延幅が広い.胴長に比べ幅の狭い数種類のスラブから,多種類の幅の鋼板図5・23 厚板工場レイアウト例図5・24 加熱炉を圧延しなければならないので幅出し圧延(broad side rolling)される場合が多い.品質面では圧延による鋼板の機械的性質の異方性を少なくするためにクロス圧延が行われたり,寒冷地向ラインパイプ用素材のように低温靱性の優れた鋼版を圧延のままで製造するために,コントロールドローリングが多用されるようになった.また長手方向の板厚精度を上げるための油圧AGCの採用,さらに平面および断面形状の矩型化を図るための制御が行われている.最近のおもな圧延技術を図5・25に示す.b. 圧延機の形式厚板圧延機には,二重可逆式圧延機,ラウト式三重圧延機,および四重可逆式圧延機があるが,生産量,板厚精度の面で優れている四重式が,現在では一般的に用いられている(図5・26).c. 付属設備デスケーリング設備,ロール冷却設備,スラブ転回設備,サイドガイド,ロール組替装置などがあって,さらに圧延中の材料形状の修正と一次スケールを破砕しデスケーリングの効果をより高める目的で,エッジャーを設置する場合がある.
