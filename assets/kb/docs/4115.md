# 5・6・3 矯正機

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

a. 概要厚板用矯正機は,その矯正機構からローラレベラ(図5・27)とプレス矯正機に分類される.ローラレベラは鋼板に繰返し曲げを付与することにより,反り,部分ひずみ(小波)を矯正し,さらには内部残留応力も低減する.一方,プレス矯正機はローラレベラでの矯正能力を超える範囲の厚板を対象とし,逆曲げを与えることにより鋼板の反りを矯正する.また,ローラレベラは設置場所により,ホットレベラ(hot leveler),熱処理レベラ(heattreatment leveler),コールドレベラ(cold leveler)に区分される.ホットレベラは仕上げ圧延機の下流に設置され,圧延直後また1―9 9図5・25 厚板圧延技術は加速冷却直後のひずみを熱間にて矯正する.熱処理レベラは熱処理設備の後面に設置され,熱処理により発生するひずみを温間にて矯正する.コールドレベラは精製ライン中に設置され,製品出荷前の最終矯正を冷間にて行う.b. ローラレベラの最近の動向近年は加熱冷却パターンの制御により機械的性質を制御した鋼板の生産が増加し,そのひずみの低減および内部残留応力の低減がローラレベラに要求されるようになっている.そのため大きな矯正反力をもつローラレベラが建設されており,コールドレベラでは50MN を上回るものもある.これらのローラレベラは,大矯正反力でも所定の加工率を得られるようにレベラ自身のたわみの自動補正制御を具備し,より精度の高い矯正効果を実現している.1―100産業機械・装置図5・26 四重圧延期の例図5・27 ローラレベラの例
