# 5・6・3 矯正機

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

a.概要 厚板用矯正機は，その矯正機構からローラレベラ
（図5･27）とプレス矯正機に分類される．ローラレベラは鋼板に繰
返し曲げを付与することにより，反り，部分ひずみ（小波）を矯正
し，さらには内部残留応力も低減する．一方，プレス矯正機はロー
ラレベラでの矯正能力を超える範囲の厚板を対象とし，逆曲げを与
えることにより鋼板の反りを矯正する．また，ローラレベラは設置
場所により，ホットレベラ（hot leveler），熱処理レベラ（heat
treatment leveler），コールドレベラ（cold leveler）に区分され
る．ホットレベラは仕上げ圧延機の下流に設置され，圧延直後また
は加速冷却直後のひずみを熱間にて矯正する．熱処理レベラは熱処
理設備の後面に設置され，熱処理により発生するひずみを温間にて
矯正する．コールドレベラは精製ライン中に設置され，製品出荷前
の最終矯正を冷間にて行う．
b.ローラレベラの最近の動向 近年は加熱冷却パターンの制
御により機械的性質を制御した鋼板の生産が増加し，そのひずみの
低減および内部残留応力の低減がローラレベラに要求されるように
なっている．そのため大きな矯正反力をもつローラレベラが建設さ
れており，コールドレベラでは50MNを上回るものもある．これ
らのローラレベラは，大矯正反力でも所定の加工率を得られるよう
にレベラ自身のたわみの自動補正制御を具備し，より精度の高い矯
正効果を実現している．
