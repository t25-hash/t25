# 5・6・4 精整ライン

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

a.概要 圧延され矯正された鋼板を冷却し所定の寸法に切断
し，表示，刻印などの識別表示を行い，各種検査を実施して製品に
仕上げるラインである．精整ラインは種々の工程があり，その組合
せも複雑である．したがって，レイアウトも各工場の敷地その他の
条件により異なっている．以下おもな設備について説明する．
b.冷却設備 冷却床（cooling bed）の機能は次の二つに大別
できる．
(１) 熱間矯正された平坦な鋼板をひずみが生じないように冷却
し，きずをつけることなく，せん断ラインへ搬送すること．
(２) 圧延ラインとせん断ラインとの接点に設置し，処理能力の
差を吸収するバッファの役割，冷却床の形式としては，最近ウォー
キングビーム方式とキャリヤチェン方式が一般的となっている．
c.せん断設備 せん断設備（shear）は一般に，長手方向分割
せん断機（ライン方向断面を図5･28に示す），耳切せん断機，幅分
割せん断機により構成され，刃の動作，機構により表5･5のように
分類される．なお，最近はカッティングサイクルが短く，被せん断
材の変形が少ないローリングカット形式が多い．
また，製品を直接溶接可能に幅を切削するエッジミラーを設置し
ているところもある．
せん断ラインの各シャーの配置例を表5･6に示す．
d.その他の設備 上記設備のほか，精整設備としては，採寸
および識別表示設備，検査設備，手入設備，塗装設備などがあり，

5・6・5熱処理設備
厚板の熱処理は圧延と冷却を組合わせた加工熱処理（TMCP:
thermo-mechanical control process）と精整ラインでの再加熱熱
処理がある．最近はTMCP処理材が増加している．
a.加工熱処理設備（加速冷却設備) TMCPのために圧延機
後面のローラテーブル上で熱鋼板に，スプレーノズル，ラミナノズ
ルで鋼板上下面に1m当たり1分間に0.5～5mの水を散水して
冷却する厚板冷却機である．鋼板の冷却停止温度は加速冷却で
500℃前後，直接焼入れで150℃以下である．冷却率は鋼板厚さ25
mmの場合5～30℃/sが得られる．設備長さは10～40mである．
