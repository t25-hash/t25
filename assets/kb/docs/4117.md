# 5・6・5 熱処理設備

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

厚板の熱処理は圧延と冷却を組合わせた加工熱処理(TMCP:thermo-mechanical control process)と精整ラインでの再加熱熱処理がある.最近はTMCP処理材が増加している.a. 加工熱処理設備(加速冷却設備)TMCPのために圧延機後面のローラテーブル上で熱鋼板に,スプレーノズル,ラミナノズルで鋼板上下面に1m当たり1分間に0.5~5mの水を散水して冷却する厚板冷却機である.鋼板の冷却停止温度は加速冷却で500°C前後,直接焼入れで150°C以下である.冷却率は鋼板厚さ25mmの場合5~30°C/sが得られる.設備長さは10~40mである.b. 再加熱熱処理設備熱処理炉と焼入れ機で構成される.熱処理炉は炉内ローラで鋼板を搬送しながら加熱するローラハース炉が主として使用される.加熱方式としてはバーナの燃焼ガスで鋼板を加熱する直火式炉とバーナで炉内に設けた放射管を加熱し,放射管により鋼板を加熱する間接加熱式がある.間接加熱式では炉内に雰囲気ガスを使用するため無酸化加熱が行え,900°C前後の焼入れ,焼ならしに使用される.直火式は主として750°C以下の焼戻しに使用される.加熱率は板厚1mm当たり1.2~2分程度で,炉長は30~100mである.焼入れ機はローラクエンチ式の厚板冷却機が使用される(図5・29,5・30).
