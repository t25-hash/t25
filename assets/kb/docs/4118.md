# 5・7・1 ホットストリップミル(hot strip mill)

> **章**: 第5章 製鉄機械
> **節**: 5・7 薄板圧延設備

furnace)が薄スラブ連鋳機出側に設置され(図5・32)、ス図5・32 コンパクトホットストリップミル(ミニミル)図5・33 加熱炉とその前後設備1―102産業機械・装置図5・34 四重圧延機(エッジャ付)ラブを炉内ローラで搬送しながら加熱および均熱を行い、圧延機に送り込む。b. 粗圧延設備加熱炉から抽出されたスラブは粗圧延機(roughing mill)で4~7回の厚さ方向の圧延(水平圧延)と幅方向の圧延(エッジング)を加えられて25~60mm程度の厚さの粗バーとなる。加熱中に生じたスケールは少量のエッジングを行って割れ目を入れた後高圧水(約16MPa)で吹き飛ばすか、直接高圧水で除去される。圧延中に発生するスケールは各圧延機の入口で高圧水を噴射することにより吹き飛ばされる。圧延機の配列には半連続式や全連続式または3/4連続式などがある。粗圧延ラインは通常1台の二重圧延機(2-hi mill)と2~3台の四重圧延機(4-hi mill)(図5・34)と2~4台のエッジャ(edger)から構成されている。四重圧延機は図5・34のように直接圧延材と接するワークロールをモータで駆動し圧延荷重はバックアップロールで受ける構造となっている。バックアップロール軸受には高荷重低回転油膜軸受が多く採用されている。四重圧延機の上下のワークロールのすきまは上ワークロールと上バックアップロールを圧下スクリューにより昇降することで設定できる。エッジャは2本の竪ロールでスラブを幅方向に圧延するもので図5・34に示すような四重圧延機と組合わさったアタッチドエッジャ等の形式がある。エッジャロールは作業側・駆動側にそれぞれ設けた圧下スクリューにより開閉される。ロールの移動量が大きいのでロールの駆動はユニバーサルジョイントを介して行われる。またストリップの幅変動を粗圧延段階で修正しておくため油圧シリンダによりロールの位置を圧延中に調節して自動板幅制御(AWC)できるエッジャもある。c. ディレイテーブルおよびコイルボックス(coilbox)粗バーは粗圧延機群と仕上圧延群をつなぐディレイテーブル(図5・31参照)上を通過して仕上圧延機へ進入する。また粗バーの保温性・均熱性向上やディレイテーブル長短縮のため、ディレイテーブル上に粗バーをいったん巻取りそれを逆転して仕上圧延機に送り出すコイルボックス(図5・35)を設置することも行われている。d. クロップシャー(crop shear)粗バーの先後端部は、一般に形状が悪く、また粗圧延中に温度が低下しているので仕上圧延の直前にクロップシャーで切断する。クロップシャーには一般にドラムタイプシャーが使われている(図5・36)。これは歯車で結合された上下二つのドラムをモータによりスタート・ストップして、ドラム上に取付けられたナイフにより切断するものである。e. 仕上圧延機(finishing ストリップ冷却装置仕上圧延を終えたストリップは必要な冶金学的性質を与えるためランアウトテーブル上で所定の冷却パターンで冷却される。ここでは、高能率に冷却するために図5・39に示すようなヘッダから一定の低圧で水を流して層流で冷却するように配慮されている。h. ダウンコイラ(down coiler)冷却されたストリップはダウンコイラによりコイル状に巻取られる(図5・40)。パスライン上に設けたピンチロールでストリップの進行方向を下方へ曲げ、ラッパロールがストリップの先端部をマンドレルに巻付ける。その後ストリップはピンチロールによりバックテンションを与えられながら巻取られる。コイルはマンドレルの外径を縮小させた後抜き出される。
