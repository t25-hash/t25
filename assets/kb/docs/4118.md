# 2・4・5 流量制御弁

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

流量制御弁とは，流量を制御するバルブ（弁）の総称である．絞
り弁，流量調整弁，分流弁などがある．
a.絞り弁（restrictor ，metering valve) 絞り作用によって
流量を規制する流量制御弁で，圧力補償機能のないもの．開口面積
が一定な固定絞り形と可変調整形とがあり，いずれも逆止め弁（チ
ェック弁）と組合わせたものが多く使用される．
b.流量調整弁（pressure compensated flow control valve)
圧力補償機能により入口圧力や出口圧力の変化にかかわりなく，流
量を所定の値に保持する流量制御弁である（図Ⅴ-2･53）．絞り弁と
圧力差の変化を補償する定差圧減圧弁とが組合わせてある．絞り弁
前後の差圧が定差圧減圧弁により一定に保たれるため，入口圧力や
出口圧力にかかわらず流量が一定となる．さらに，作動油の温度変
化に基づく粘度変化にも対応できる温度補償付き流量制御弁もあ
る．
c .流量調整弁のジャンピング現象 流量調整弁は圧力補償機
構の応答遅れがあるため，流量が急激に変化すると，初めに大流量
が短時間流れてアクチュエータの飛び出し現象が起こる．これをジ
ャンピング現象という．
d.分流弁，集流弁，分集流弁 分流弁（flow dividing valve）
は一つの流れを任意の二つの流れに分割するバルブ，集流弁は二つ
の流れから任意の一つの流れを得るバルブ，分集流弁は両方の機能
を兼ね備えたバルブである．
このほか，流量制御弁にはカム操作などによって流量を減少させ
るデセラレーション弁（deceleration valve），ブレスの油タンク
とアクチュエータを接続するプレフィル弁（prefill valve）などが
ある．
