# 5・7・2 酸洗設備(pickling line)

> **章**: 第5章 製鉄機械
> **節**: 5・38 に示すように台車等で引出した旧ロールと新ロールをモータ

ホットストリップミル(5・7・1項参照)で圧延された熱延鋼板(strip)は,その冷却過程において鋼板の表面に酸化皮膜(millscale)が形成されるが,これは次工程の冷間圧延(5・7・3項参照)を行う前に,完全に除去しておく必要がある.酸洗設備は二つに大別でき,後述に示す入側,出側ルーパを配置し入側,出側がコイル取替えのため停止した場合でも酸洗部は連続運転できるものを連続酸洗設備(continuous pickling line)と呼び,大容量生産,高品質製品の連続生産に適する.これに対し,入側,出側ルーパを省略し入出側が停止した場合は酸洗部も停止するプッシュプル酸洗設備(push pull pickling line)と呼ばれる設備がある.これは品質面の要求が比較的厳しくなく生産量が少ない場合に採用されており,設備費用を抑えることができる特徴がある.a. ダウンコイラ図5・41 連続酸洗設備配置図1―103および後行コイルを接続する溶接機(welder)等から構成される.. 中央(酸洗)部[process (pickling) section]本装置の中心部であり,入側停止時でも中央部で連続的に酸洗をするための入側ルーパ(looper),酸洗反応促進のため,あらかじめスケールにき裂を入れるスケールブレーカ(scale breaker),酸洗を行う酸洗槽(pickling tank),酸洗後の鋼板表面の酸を洗浄する水洗槽(rinse tank),洗浄後の鋼板を乾燥させるドライヤ(dryer),出側停止時でも中央部を連続運転するための出側ルーパ等から構成される.. 出側部(delivery section)中央部で酸洗された鋼板を一定の板幅に耳切りするサイドトリマ(side trimmer),防せい油を塗布するオイラ(oiler),コイルを一定の長さまたは重量に分割するためのシャーおよび巻取り機(tension reel)等から構成される.b. 前処理近年酸洗設備の大容量化と取扱(処理)鋼種の多様化および酸洗時間短縮,酸洗槽長短縮をはかるため機械的にスケールブレーキングを行う前処理が重要となっている.これに用いるスケールブレーカにはおもにテンションレベラ(tension leveler)とスキンパスミル(skin pass mill)がある.テンションレベラは元来鋼板の形状矯正に用いる機器であるが,この機器がもつ引張曲げ機能を利用して酸洗設備へ応用している.引張曲げによりスケールにき裂(crack)を発生させ酸液との接触面積を増加させることにより塩酸との反応が促進される.通常,1%程度の伸び率を与え,酸洗促進と形状矯正の両方の効果を狙っている.スキンパスミルも同様に鋼板に伸びを与えスケールに亀裂を発生させ酸洗を促進させるが,同時に調質圧延により鋼板の降伏点を除去する効果もある.c. 酸洗槽槽本体は通常20~30m長さのものが3~5槽で構成されており,材質は鋼板製の缶体にゴムライニングおよび耐酸れんが内張りである.最近では缶体をポリプロピレン(poly-propylene)で製作することによりゴムライニングおよび耐酸れんが内張りを省略したプラスチック製酸洗槽も普及しつつある.1―104産業機械・装置d. 酸洗液1960年代中頃までは取扱い面より硫酸が使用されてきた.今日では塩酸回収装置(acid regeneration plant)の進歩により,ほとんどの設備に塩酸が使用され,消費した酸の約95%以上(最新のものでは99%)は回収再生され再使用する技術が確立されている.さらに塩酸回収工程での副産物として生成される酸化鉄は磁気テープ等の磁性体原料として活用されている.e. 噴流酸洗近年の設備の大容量化による高速酸洗の必要性と難脱スケール材(深絞り用鋼板をホットストリップミルで製造する際の巻取り温度は700°C以上と普通鋼時の約600°Cより高く,鋼板表面のスケール厚みが約2倍となり酸洗が難しくなる)の増加により,従来の酸洗槽の槽深さを浅くし,噴流液を使用し酸洗効率を上げた噴流酸洗(jet pickling)が開発された.これは酸洗が熱と物質の同時伝達であることに着目し,走行中の鋼板表面に発生する境界層厚みを抑えることにより熱伝達率,物質伝達率を上げ,酸洗を促進させることを目指したものである.これにより酸洗に必要な時間を従来型の約半分にすることができるとともに,板張力が高いため高速運転時の通板性が安定する利点より,特に大容量設備や難脱スケール材専用設備に採用されている.f. 連続化以前は単独の連続酸洗設備がおもに建設されてきたが,近年では省力化,生産の効率化,品質の向上改善のため,次工程である冷間圧延設備と接続した連続酸洗冷間圧延設備(con-tinuous pickling and tandem cold mill)も多く建設されている.また,既設の連続酸洗設備と冷間圧延設備を接続する連続化の改造工事も行われている.
