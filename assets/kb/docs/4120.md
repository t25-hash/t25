# 5・7・3 冷間圧延機

> **章**: 第5章 製鉄機械
> **節**: 5・38 に示すように台車等で引出した旧ロールと新ロールをモータ

板材を常温(冶金学的には変態点以下の温度)で圧延する圧延機で、一般に熱間圧延された板材を圧延し、より薄く、寸法精度・形図5・42 冷間圧延工程(a) 可逆式冷間圧延機(b) タンデム式冷間圧延機状の良い表面の美麗な板材を得るために使用される。図5・42に冷間圧延工程を示す。板材を所定の厚さまで圧延する冷間圧延機(cold reduction mill)と冷間圧延後焼なましした板材に所定の機械的性質を与える調質圧延機(temper mill,skinpass mill)を総称し冷間圧延機という。a. 冷間圧延機の構成圧延加工を行う圧延機とその駆動装置、コイル状に巻いた板材の巻出機(pay offreel,uncoiler)と巻取り機(tension reel,coiler)、油圧装置、潤滑油給油装置、板材とロールを潤滑し冷却するロールクーラント装置、コイル搬送装置などより構成される。b. 冷間圧延機の種類1台の圧延機で正逆転し圧延材を数回往復させ所定の厚みを得る可逆式圧延機(reversing mill)、1台の圧延機で一方向にグループ化した圧延材を数回通過させ所定の厚みを得る一方向圧延機(one way mill)、数台の圧延機をくし形に並べ圧延材を一方向に1回(圧延材によっては数回)通過させて所定の厚みを得るタンデムミル(tandem mill)がある。可逆式圧延機は小規模生産に、一方向圧延機はアルミニウムの圧延、銅合金の中間圧延に、タンデムミルは大規模生産に用いられている。また、スズめっき原板のように板厚が薄く硬さの要求される調質圧延に2~3スタンドのタンデムミルが使用されている。現在では、圧延材を溶接し連続的に供給する全連続式タンデムミル、さらにその変型として酸洗設備と結合した酸洗-冷延全連続式、連続焼なまし設備と結合した設備などにより、生産能率向上、工程間在庫縮小、運転要員の削減等が図られている。図5・43(a)(b)に冷間圧延機の構成例を示す。c. 圧延機の種類圧延加工を行う上下2本の作業ロール(work roll-WR)のみよりなる圧延機を二段(または二重、2-hi図5・43 冷間圧延設備の種類と略す)圧延機と称す。この場合、2本のロールで圧延加工を行い、同時に圧延反力-圧延荷重(RSF: roll separating force)を支持する必要があり、圧延荷重の低い圧延機にしか用いられていない。これを解消するため作業ロールは圧延加工のみに使用し、圧延荷重を支える補強(または控え)ロール(BUR : back up roll)を備えた四段(または四重)圧延機(4-hi mill)が出現し、長年にわたり板用圧延機の代表形式として広い分野にわたって使用されてきた。硬い材料を薄く圧延するには小径の作業ロールが有利で、この場合、補強ロールの数を増した多段圧延機(multi roll mill)-12段圧延機、20段圧延機-が使用されている。特に20段圧延機(send-zimir mill)は広幅のステンレス鋼板の圧延に数多く使用されている。図5・44に各種圧延機を示す。d. 形状制御圧延機圧延材の長手方向の伸び量が、幅方向で一様でない場合、形状が悪くなる。端部が多く伸びると端伸び、中央部が多く伸びると中伸びとなる。この幅方向の伸びの不同は、圧延荷重や熱膨張による作業ロールの変形の不均一により生じる。これを補正するため、作業ロールを樽型に加工(crown加工)、ロール冷却の幅方向の制御、ロール軸端に油圧力を加えるロールベンディング(roll bending)が行われてきた。作業ロールの圧延荷重による変形の根本原因を除去すべく、軸方向に移動する中間ロールを備えた六段圧延機(HC millまたはUC mill)、作業ロールシフト式四段圧延機、補強ロール径を油圧により軸方向に変化させるVCロール、作業ロールを水平方向に曲げるFFCミル、ロールに幾何学的曲線を設け、シフトさせるCVCミルなどが開発され、六段圧延機が多く用いられている。e. 圧延機の構造図5・45に示す四段圧延機は、ロールを収容し圧延荷重を担うハウジング、作業ロール、補強ロール、圧下シリンダ、ロール駆動機、圧延材の案内装置、ロール交換装置などより構成される。ハウジングは鋳鋼製または鋼板溶接製である。作業ロールは鍛鋼製ショア硬さ90~95に表面焼入れされ、鋳鋼製軸受箱(chock)に収容された転がり軸受に支持される。補強ロールは鍛鋼製ショア硬さ60~65に表面焼入れされ、軸受箱に収容された軸受に支持される。補強ロール用軸受は圧延荷重を受けるので大容量のものが必要で、小型圧延機には転がり軸受が、大型圧延機には油膜軸受や転がり軸受が使用されている。油膜軸受はコンパクトで負荷容量大、転がり軸受は精度および起動性が良い。駆動機はロールに回転を伝えるスピンドルと、1台の電動機の回転を上下に分けるピニオンスタンド、減速機、増速機などの歯車装置より構成される。スピンドルは上下に動くロールに追従するため可傾性が必要で、ギヤタイプやユニバーサルジョイントタイプが使用されている。圧下装置(roll gap control system)は所定の板厚を得るため圧延荷重下でロールすきまを調整するもので、圧下ねじを減速機を介し電動機にて上下する電動圧下が用いられていたが、板厚精度向上のため応答性の良い油圧圧下装置が現在は多く採用されている。圧下装置にはロールの平行調整、レベリング調整と位置保持の機能が必要で、油圧でこれを実現するため、機械式フィードバック方式と図5・46に示すロール位置検出器と電気-油圧サーボバルブより1―105図5・44 ロール冷却装置板材を冷間圧延する際に発生する加工熱の除去と、圧延材と作業ロール間の潤滑のために用いられ、冷間圧延機に不可欠の装置である。圧延油と温水のエマルションをロール冷却と潤滑に循環使用する循環式と[図5・49(b)]、高濃度の圧延油を潤滑に、水を冷却用に別々に使用する直接式[図5・49(a)]があり、前者は運転コストが安く一般用に、後者はブリキ原板の圧延に用いられる。
