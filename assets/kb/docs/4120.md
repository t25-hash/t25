# 2・4・7 電気操作弁（電気-油圧制御弁)

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

電気信号を連続的な油圧出力に変換する電気操作弁として，比例
電磁弁とサーボ弁があり，オンオフ的に制御する弁として電磁弁
（2･4･6項参照）や高速電磁弁（high speed solenoid valve）があ
る．
a.比例制御弁（proportional control valve) 圧力制御弁・
流量制御弁・方向切換弁の手動操作部，あるいは電磁操作部に比例
電磁石（proportional solenoid）を用いたものが比例制御弁である
（図Ⅴ-2･54は比例電磁式流量制御弁）．サーボ弁と比較すると応答
性は劣るが，汚染感度は低く一般油圧機器と同等レベルでの使用が
可能である．
b.サーボ弁（servo valve) 多くのサーボ弁（図Ⅴ-2･55）で

プール弁が開き，流量が発生して油圧エネルギーが制御される（二
段式）．スプール変位はフィードバックされ，入力電流に比例した
スプール変位が得られるように調整される．作動油の汚染管理に注
意する必要があるが，応答性は高く，一般的に閉ループで使用され
