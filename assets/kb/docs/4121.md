# 5・7・4 焼なまし設備(annealing furnace)

> **章**: 第5章 製鉄機械
> **節**: 5・38 に示すように台車等で引出した旧ロールと新ロールをモータ

冷延鋼鈑の焼なましは、冷間圧延によって加工硬化した金属組織を再結晶温度以上に加熱して、結晶粒を適度に成長させ機械的性質を改善することを目的とする。焼なましに使用される炉は、操業方法によってバッチ式と連続式に分けられる。a. バッチ焼なましカバー型もしくはベル型と称される焼なましで、代表例としてタイトコイルを積み重ね、これにインナカバーをかぶせ、この外側をバーナで加熱する方法がある。インナカバーの内部には雰囲気ガスを通し光輝焼なまし(bright annealing)を行う。図5・50に焼なましサイクルの一例、図5・51に炉の断面を示す。b. 連続焼なましブリキ用連続焼なまし設備(continuousannealing line)では、処理材は化学清浄、電解清浄を経て焼なま形式構造開閉方法使用設備ピラミッド1を油圧シリンダにて軸方向に移動させることによって4セグメント2を開閉させる。・リダクション・センジマーミル・タンデム・スキンパス軸方向に傾斜の付いたウェッジ1を油圧シリンダにて軸方向に移動させることによって2セグメント2を開閉させる。・リダクション・スキンパス・タンデム軸方向に傾斜のついたウェッジ1を油圧シリンダにて軸方向に移動させ2のガイドバーを介して4セグメント3を開閉させる。・スキンパス・プロセッシングライン軸方向に傾斜の付いたウェッジ1を油圧シリンダにて軸方向に移動させることによって4セグメント2を開閉させる。・プロセッシングライン・リダクション・タンデム軸方向に傾斜の付いたウェッジ1を油圧シリンダにて軸方向に移動させることによって4セグメント2を開閉させる。・プロセッシングライン(a) 直接方式(b) 循環方式図5・49ロール冷却装置し炉に入り、加熱、均熱、徐冷、急冷の各帯で所定のヒートサイクルにて通板された後巻取られる。ペイオフおよびテンションリールは各々2基ある。近年では時効性を良くするため、炉において過時効処理を行ったり、表面粗度と硬度を調整するため炉出側においてテンパーミルにより圧下する場合もある。図5・52にブリキ用連続焼なまし炉、図5・53に焼なましサイクルの一例を示す。一方冷延鋼板の連続焼なましは、社会的ニーズである品質安定性と経済性の追求、さらには冶金学の進歩、設備技術の進歩が相まって急速に進歩、普及してきた。この方法の特徴は急速加熱、短時間均熱により再結晶、結晶粒成長をさせた後、急速冷却と過時効処理によって過飽和固溶C(カーボン)を析出させてバッチ焼なましと同等の材質を得ることができる。ライン構成は、入側部、焼なまし(加熱、均熱)、冷却、過時効炉、(酸洗)、出側部からなっているが、冷却方式によって多少異なっている。各部の機能、特徴は次のとおりである。. 入側部冷延コイルを巻きほぐし、次々と溶接し連続走行、表面を化学洗浄、電解洗浄した後、焼なまし炉へ送り込む。. 焼なまし加熱、均熱帯からなり、鋼板を所定の温度まで昇温し、約1分程度保持する。加熱方式はラジアントチューブによる間接加熱が大半だが、最近、熱効率と急速加熱の観点より直接加熱(ただし、無酸化雰囲気となる燃焼)が研究、実施されている。均熱帯には通常ラジアントチューブか電熱ヒータが用いられている。. 冷却図5・54に冷却方式によるサイクル例と特徴を示図5・50タイトコイル焼なましサイクルの一例図5・51シングルスタックタイトコイル焼なまし炉図5・52 ブリキ用連続焼なまし設備1―107す。. 過時効炉約400°Cで数分の過時効処理を行うため炉内に電熱ヒータが装備されている。過時効炉では、鋼板の繰返し曲げによる応力時効を防ぐため、ロール径を加熱、均熱帯のφ600~φ800mmに対しφ1000~φ1500mm程度に大きくしているのが普通である。. 出側部調質圧延機、サイドトリマ、検査、塗油設備、巻取り設備よりなる。調質圧延機には六重式ミルも利用され、形状制御能力が向上している。
