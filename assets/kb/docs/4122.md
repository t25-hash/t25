# 5・7・5 せん断設備

> **章**: 第5章 製鉄機械
> **節**: 5・38 に示すように台車等で引出した旧ロールと新ロールをモータ

せん断設備には，シートを作るシャーライン（shearing line）と
フープを作るスリッティングライン（slitting line）とがあり，そ
れらの性能を表5･8に示す．
a.シャーライン 高性能のフライングシャーラインと小容量
のストップシャーラインとがある．特にフライングシャーラインに
おいては，せん断時刃物がストリップの移動速度に極力同調して移
動しながらストリップをせん断し，良好なせん断面を保つ構造にな
っている．
フライングシャーには，揺動型とドラム型とがあり，ドラム型は
高速せん断ができるため，薄物高速用として用いられ，揺動型は良
好なせん断面が得られ，厚物ストリップ用に多く用いられる．ドラ
ム型はDDS（direct digital servo）制御により，せん断精度を向
上させたものもある．
フライングシャーラインは図5･55に示すように，ストリップコ
イルを巻出し，サイドトリマにて所定の製品幅にせん断後，レベラ
にて矯正し，必要に応じて，さらに仕上げレベラにて平坦なシート
とし，パイラに自動積載される．
b.スリッティングライン 広幅のコイルから，任意の板幅の
多条のフープ状コイルを生産する設備で，ループタイプとテンショ
ンタイプとがある．
スリッタは板幅のせん断条数に応じて，1本のシャフトに多数個
の丸刃カッタを組込み，これを上下一対にセットし，任意の板幅の
フープを生産する設備である．
