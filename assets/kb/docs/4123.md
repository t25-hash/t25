# 5・7・6 プロセスライン(processing line)

> **章**: 第5章 製鉄機械
> **節**: 5・38 に示すように台車等で引出した旧ロールと新ロールをモータ

冷間圧延後の帯鋼(ストリップ)は最終製品の用途に応じ、焼なまし(5・7・5項参照)、めっき、塗装を行い、コイル状またはせん断加工後シート状にして出荷される。このように圧延加工されたストリップに最終加工処理を行う設備を総称してプロセスラインと呼ぶ。これに熱間圧延と冷間圧延の間に位置する酸洗設備も加えてプロセスラインと称することもある。表5・9に各種プロセスラインとその機能について示す。各種プロセスラインは通常2文字または3文字の略称で呼ばれており、非連続ラインと連続ラインとに分別される。非連続ラインはせん断設備(5・7・5項参照)のようにコイル分割、幅方向のスリットおよびシート状への切断等の機械的処理や洗浄のみを施すものに適用され、一つのコイルの処理が終わるたびにラインを停止し、コイルの入替えを行う。連続ラインは目的の処理装置を配置する中央部(process sec-tion)および中央部にストリップを送り込む入側部(entry sec-tion)と中央からのストリップを巻取る出側部(delivery sec-tion)を総称したターミナルセクション(terminal section)から構成されている。入側部~中央部間および中央部~出側部間にはストリップを蓄えるルーパ(looper)を設け、入側部がコイル入替えのため、および出側部がコイル切替えのために減速・停止している間でも中央部は一定速で連続運転できることにより一定品質の製品の大量生産を可能としている。したがって連続的に処理することにより品質面が安定する酸洗、焼なまし、めっきなどの表面処理や機械的性質調整用のラインに適用される。複数のプロセスラインを接続して連続化し、同時に2種類以上の処理を行う複合ラインも建設されている。例として酸洗と溶融亜鉛めっきを連続化した連続酸洗溶融亜鉛めっきラインや酸洗、冷間圧延、焼なましを連続化したライン、電気めっきと塗装ラインを連続・GAS(N)と水との混合冷却・冷却速度は中間的(50~400°C/s)・再加熱の必要なし・過時効時間は水焼入れとガスジェット冷却との中間・内部を水冷されたロールに鋼板を接触させて冷却・冷却速度は中間的(50~100°C/s)・再加熱の必要なし・過時効時間は水焼入れとガスジェット冷却との中間図5・55 フライングシャーライン化したライン等がある。また、ストリップを処理後再度コイル状に巻取るのみならず、出側部にせん断設備を接続してシート状に切断して出荷する設備もある。このように用途に応じて複数の機能を柔軟に組合わせて設備構成できるのがプロセスラインの特徴であり、これらの複合設備は複数の処理を連続化して1ラインにすることにより生産の効率化、省力化、品質の向上を図ることができるが、制御、運転が複雑になるという問題点もある。図5・56に代表的プロセスラインである連続溶融亜鉛めっきライン(continuous position control)装置をライン内数箇所に設置する。ストリップのずれ検出には光学的センサが用いられてきたが、最近では電磁誘導式(inductive)や静電容量式(capacitance)センサも用いられている。高精度の芯ずれ制御が要求される個所ではCCD カメラによる画像信号処理を採用する。テンションリールではEPC(edge position control)装置にて巻取り中のコイル端面が一定位置になるよう制御している。ストリップの最終加工処理においては基礎となる処理プロセス技術が冶金学、制御技術、計測技術の進歩とともに変化している。プロセスラインとそれを構成する各機器はこれらの技術と密接に連携して常に最新技術を適用していくことで、品質向上、歩留り向上、省力化、能率向上の各面で大きな成果を上げている。
