# 5・8・1 形鋼製造設備のレイアウト

> **章**: 第5章 製鉄機械
> **節**: 5・8 大中型形鋼圧延設備

形鋼製品は品種および寸法が多種多様であり,工場によっても生産品種が種々ことなるので,他の鋼材の圧延設備のように工場のレイアウトがあまり定型化していない.その中でも代表例を挙げると図5・58に示すとおりである.図5・58 レイアウトの代表例図5・58(a)は並列式圧延機配列工場であり,H 形鋼以外の比較的小さな通常の形鋼(steel shapes,steel sections)を製造する工場である.図5・58(b)は多品種の大型形鋼を製造する直列式圧延機配列工場で,可逆式圧延機であるブレイクダウン圧延機(break-downmill)および中間の圧延機群(図中の記号RUM およびEM)で鋼材を何度も往復させて圧延するので,圧延機の台数が少ない.一方,冷却床以降の精整設備は,多品種の形鋼に対処するため比較的大きな面積を占有している.図5・58(c)は(b)の直列式圧延機の仕上圧延機を中間の圧延機群の直後に配置したものである.粗ユニバーサルミル(RUM)と仕上ユニバーサルミル(FUM)によるタンデムリバース圧延を行い生産性向上とレイアウトのコンパクト化を図っている.図5・58(d)はH 形鋼(H-steel,H-section)のほか一部の通常の形鋼の多量生産型連続式圧延機配列工場で,多くの圧延機台数を必要とするが,ブレイクダウン圧延機以外は可逆式圧延機がないので,形鋼を高能率で圧延できる特徴を有しているが,工場の長さが長くなる.
