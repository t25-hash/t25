# 5・8・2 形鋼圧延機の種類

> **章**: 第5章 製鉄機械
> **節**: 5・8 大中型形鋼圧延設備

a. ブレイクダウン圧延機ブレイクダウン圧延機(break-down mill)は圧延機の原点となる構造の二重圧延機で、上下の水平ロールで可逆圧延するものである。ロール表面には各製品、サイズに応じた複数の穴型(caliber)が配置されている。素材断面形状(スラブ、ブルーム、ビームブランク)から次工程のユニバーサル圧延に必要な中間断面形状まで圧延する。図5・59は素材と製品の寸法関係例を、図5・60はブレイクダウン圧延機の全体を示す。b. ユニバーサル圧延機ユニバーサル圧延機(universalmill)はH 形鋼を圧延するための特別な構造の圧延機で、図5・61に示すように同一平面内に上下の水平ロールと垂直ロールの軸心がある圧延機である。ユニバーサル圧延機では、材料は上下の水平ロールの間および上下水平ロールの側面と左右の垂直ロールとの間に形成される空間で材料を圧延するので、H 形鋼のフランジの先端部は圧延されず、図5・59素材と製品の寸法関係例図5・60 ブレイクダウン圧延機1―111先端部は丸くなり、またフランジの幅も規定寸法にするのが難しいので、図5・62に示すようにフランジ先端部を圧延するエッジング圧延機(edging mill)と組合わせて使用される。c. オープンヨーク式ユニバーサル圧延機ユニバーサル圧延機のもう一つの特色は、4個のロール軸が同一平面内にあるためロール交換がしづらいことである。このため種々のロール交換装置が考案されている。図5・63に示す例は圧延機本体の内垂直ロールの圧延荷重を支持する部分を開閉できるように設計された圧延機で、オープンヨーク式ユニバーサル圧延機(open yoke type universalmill)という。ロール交換の際には、図5・64に示すように4個のロールを同時に圧延機から出し入れすることができ、ロール交換が短時間でできる特長を有する。
