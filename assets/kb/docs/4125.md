# 2・5・4 オイルリザーバ

> **章**: 第2章 油圧システム
> **節**: 2・5 油圧付属機器

オイルリザーバ（oil reservoir ）は作動油を蓄えるとともに，主
戻り管路からの作動油の流速を低下し，吸込管路到達までに気泡
するために，タンク内油量は一般に，油圧ポンプの1分間の吐出し
量の3～7倍が推奨されてきた．これに対して，車載の油圧システ
ムでは油量は極端に少なく，油圧パワーステアリングシステムで
は，ポンプの1分間の吐出し量の10～20%程度の油量である．
近年，産業用油圧装置においてもフィルトレーション，温度管
理，気泡除去を考慮した設計により，ポンプの1分間の吐出し量の
1～2倍に油量を減らす例も増加している．これらの機能を強化す
るものとして，鉄系の摩耗粉を永久磁石により吸着するもの，ま
た，戻り管路に設置して旋回流による遠心力場を発生させ，旋回流
の中心に集まる気泡を除去する装置 が開発されている．
