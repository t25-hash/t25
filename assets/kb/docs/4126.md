# 5・8・3 スタンド交換方式形鋼圧延機

> **章**: 第5章 製鉄機械
> **節**: 5・8 大中型形鋼圧延設備

形鋼製品はその種類、寸法が多岐にわたり、かつ形状が複雑であるのが他の鋼材とちがう大きな特徴である。ブレイクダウン圧延機で圧延後、これらの種々の形鋼を圧延するために二重圧延機とユニバーサル圧延機を使い分けて使用する。H 形鋼圧延ではユニバーサル圧延機とエッジング圧延機を、その他の一般形鋼を圧延するときは二重圧延機で製品形状、寸法に対応したロールで穴型圧延を行う。形鋼の種類、サイズに応じて圧延ロールとガイドを交換しなければならない。また、その頻度も多いため、交換時間を短縮し、さらに各ロールの関係位置を精度よく設定できることが圧延機の稼働率を上げ、生産性を向上させるうえで大切なことである。このために考案され、多く実用化されている方法がロールを組込んだ圧延機ごと交換する方式で、スタンド交換式形鋼圧延機と呼ばれている。
