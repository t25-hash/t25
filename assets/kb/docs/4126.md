# 2・5・5 油圧用継手・配管

> **章**: 第2章 油圧システム
> **節**: 2・5 油圧付属機器

油圧装置を構成する要素機器は継手・配管で接続され，油圧エネ
ルギーを移動させて機能を発揮する．
配管方式はパイプの接続方式で，ねじ込み方式，くい込み方式，
フランジ方式，ユニオン方式，フレア方式に大別されるが，システ
ムの圧力，流量の平均値，および変動値に適合する方式が用いられ
る．
管内径は一般に，管内流速が吐出し側配管で3～4m/s ，吸込側
で0.6～1.5m/s以下となるように決定される．
配管には，金属管またはホースが使用されるが，省スペースのた
めにはマニホルド（manifold）が採用される．金属管としては油
圧配管用鋼管が最もよく使用され，各種JIS規格の配管用鋼管が
あるが，くい込み式管継手には専用の外径寸法精度の高い精密炭素
鋼管がJIS B2351：2000の附属書2で規格化されている．鋼管以
外には，中・低圧用として銅管も使用される．
ホースはたわみやすく，接続時の調整が容易で，接続箇所移動に
よる相対位置が変化する装置への対応が可能であり，また，内部減
衰が大きく，機器間の振動の絶縁に効果がある．ホースアセンブリ
ー（hose assembly）は，おもにNBR（ニトリルゴム）のゴムホ
ースの内面層，鋼線または，おもにポリエステル繊維による補強層
および外面層から成るホースと，その両端に取付けられた継手から
成る．
図Ⅴ-2･60は，レゾネータホースと呼ばれるゴムホースアセンブ
リーであり，ゴムホースの中に継手金具に接続された金属製円管を
設置し，配管内を伝ぱする圧力脈動を低減する．さらに油圧システ
ムの低騒音化に効果があるもので，おもに車載用油圧システムで広
く使用されている ．
