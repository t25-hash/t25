# 5・8・4 精整設備

> **章**: 第5章 製鉄機械
> **節**: 5・8 大中型形鋼圧延設備

圧延された形鋼は通常熱間鋸断機(hot saw)によって所定製品長さに切断され,その後冷却床(cooling bed)上で常温まで冷却される.図5・61 ユニバーサル圧延機のロール配置図5・62 ユニバーサル圧延機とエッジング圧延機1―112産業機械・装置冷却された形鋼は形鋼としての真直性と正しい断面形状を得るためにローラ矯正機(roller straightener)によって矯正される.図矯正された形鋼は,形状,寸法,外観などを検査した後,表示,仕分,組込,結束などの工程を経て出荷される.図5・66に仕分,組込装置の一例を示す.小型条鋼(long products),線材(wire rod)圧延設備は,製造工程に従って,加熱設備(reheating furnace),圧延設備(rollingmill),精整設備(product handling equipment)に大別される.
