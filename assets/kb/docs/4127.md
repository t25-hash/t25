# 2・6・1 油圧源回路

> **章**: 第2章 油圧システム
> **節**: 2・6 油圧回路

容量ポンプ②を用いて省エネルギーを図る回路である．
ポンプ吐出第Ⅴ部 フルードパワーシステム 第2章 油圧システム超えるとポンプ①がアンロードされ，ポンプ②がリリーフ弁④で設によって調整する．
ポンプはタンクからフィルタを介して油圧作動油を吸い込み，回路に供給する．
吐出し圧力はリリーフ弁で制限される．
の省エネルギー回路の例である．
図Ⅴ-2･62では，アキュムレータ達したときにポンプをアンロードし，代わりにアキュムレータに蓄図る．
c.HILO回路 図Ⅴ-2･63は，低圧大容量ポンプ①と高圧小し圧力がアンロード弁③の設定圧力以下の場合には，二つのポンプの吐出し流量の合計が回路に供給される．
吐出し圧力が設定圧力を定された回路圧力を維持するのに使用される．
