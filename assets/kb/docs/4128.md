# 2・6・2 圧力制御回路

> **章**: 第2章 油圧システム
> **節**: 2・6 油圧回路

路は可変容量形ポンプを用い，ポンプ容量を吐出し圧力で調整する省エネルギーを図る．
応回路（ロードセンシング回路）と呼ばれる回路で，負荷に応じた二次側の圧力を一定に保つことができる．
圧力や流量を供給することで省エネルギーを図る回路である．
可変a.圧力遠隔制御回路 回路の圧力を制限するためにリリーフができる．
サージ圧力が発生する．
図Ⅴ-2･69の回路は，リリーフ弁と逆止める．
ことによって，回路が必要とする最低限の流量だけを吐出すことで続することで，回路圧力を遠隔制御することができる．
容量形ポンプを使用し，ポンプ容量を圧力補償弁①と流量補償弁②弁が使用されるが，リリーフ弁の設定圧力を多段階に切換えることを高低二段の設定圧力をもつ2個のリリーフ弁と切換弁を介して接合，図Ⅴ-2･67に示すように減圧弁を回路の入口に設ければ，ポンウンタバランス回路という．
図Ⅴ-2･68では，直動形カウンタバラによる加速を抑える回路である．
