# 5・9・1 加熱設備

> **章**: 第5章 製鉄機械
> **節**: 5・9 小型条鋼線材圧延設備

加熱設備は,素材である鋼片(ビレット)の送り機構により,プッシャ式加熱炉とウォーキング式加熱炉に分類される.ウォーキング式加熱炉は,(1)均熱性が良い,(2)装入,抽出が自在である,(3)鋼片にすり傷が付かない,(4)鋼片単重や炉容量に制限が少ない,などの利点がある.近年,圧延設備の大容量化や歩留り向上のために大単重(大断面積)の鋼片が用いられるようになり,さらに品質向上のための均熱への要求が高まり,ウォーキング式の中でも鋼片の4面が加熱できるウォーキングビーム式加熱炉が一般的に採用されている.また,省エネルギーを目的としたリジェネレイティブバーナ(蓄熱式バーナ)を採用した加熱炉が増えている.図5・66 仕分・組込装置図5・63 オープンヨーク式ユニバーサル圧延機
