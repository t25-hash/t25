# 5・9・2 圧延設備

> **章**: 第5章 製鉄機械
> **節**: 5・9 小型条鋼線材圧延設備

圧延方式は圧延機の配置(mill layout,圧延機のレイアウト)によって決定されるが,小型条鋼,線材圧延機配置の代表例を以下に述べる.a. 基本レイアウト圧延機の基本レイアウトとしては,(1)単基式配置,(2)一軸式配置,(3)連続(タンデム)式配置に大別され,その代表レイアウトを図5・67に示す.圧延機は,通常,粗圧延機列,中間圧延機列,仕上圧延機列にグループ化され上記基本レイアウトの単独または組合せで配置される.b. 棒鋼,形鋼圧延機の配置(レイアウト)棒鋼(bars),形鋼(steel sections)圧延機のレイアウトは,圧延方式によって,(1)不連続式,(2)半連続式,(3)全連続式に分類され,各々の圧延方式の分類定義は次のとおりである.. 不連続式圧延材が同時に2台の圧延機で圧延されることがなく,かつ,圧延方向がパスごとに変わる.この配置としては,単基式配置,一軸式配置,多軸クロスカントリー式配置などがある.. 半連続式不連続と連続の組合せ配置である.通常,粗圧延機列は不連続式,中間・仕上圧延機は連続式の組合せで,粗圧延機列と中間圧延機列間の間で,圧延材がフリーとなる.この配置としては,粗圧延機列は単基式配置,一軸式配置,中間・仕上圧延機列は一軸および多軸ループ式配置,タンデム式配置などがある.. 全連続式圧延材の方向が一定で,圧延機間で圧延材がフリーにならない.この代表的な配置は,粗圧延機列から仕上圧延図5・64 オープンヨーク式ユニバーサル圧延機図5・65 片持式可変ピッチ形ローラ矯正機機までの全タンデム配置である.近年の大量生産を目指した圧延設備には全連続式配置が採用され,半連続式配置は生産量が比較的少ない設備や難加工材の圧延を目的とした特殊鋼圧延設備に採用されている.不連続式は,製品寸法が比較的大きく多品種少量生産の工場で採用される.全連続式配置のレイアウト例を図5・68に示す.最近の設備では,中間・仕上水冷装置を用いた制御圧延,制御冷却の実施,特殊鋼圧延での製品寸法精度向上,中間サイズ圧延のためのサイジングミルの設置,異形棒鋼圧延での生産性向上,電力原単位向上のためのスリット圧延などが行われている.c. 線材圧延機の配置(レイアウト)線材圧延機のレイアウトも基本的な分類は棒鋼圧延機と同じであり,棒鋼工場の中には巻線機を備え,太物線材を生産できる工場も多い.しかし,細物線材圧延では棒鋼に比べて製品寸法が小さいことから,パス回数(圧延機台数)が多くなり,また,生産量を増やすためには高速で圧延することが必要であることから,近年建設された細物線材の大量生産工場では仕上圧延機列に高速仕上ブロックミル(block mill)を採用している.このブロックミルの採用により従来圧延機では30~40m/sであった仕上圧延速度は80~100m/s以上と飛躍的に向上し,高能率生産が可能となったばかりではなく,コイル単重の大型化と品質向上も図られた.仕上圧延機列にブロックミルを採用した大量生産細物線材工場の代表例を図5・69に示す.最近の設備としては,棒鋼圧延と同様に,仕上ブロックミルの下流にサイジングブロックミルが採用された例がある.
