# 5・9・3 精整設備

> **章**: 第5章 製鉄機械
> **節**: 5・9 小型条鋼線材圧延設備

a. 棒鋼,形鋼精整設備精整設備の構成を製造工程に従って列記すると,分割切断機,冷却床(cooling bed),定尺切断機が圧延ラインに直結されて配置される.オフラインに設置される設備は,製品の品質,等級により大きく異なるが,製品手入れ,熱処理,矯正,検査,刻印,計数などに区分される.冷却床への圧延材搬入速度は,ローラテーブルでの搬入では最高18m/s程度であるが,トラフでの搬入では最高33m/sの高速圧延を実現している設備もある.従来のオフライン設備についても,異形棒鋼,山形鋼などを生産する量産工場,棒鋼専用の大量生産工場では,定尺切断後の計数,仕分け,集積,段積,結束などの工程が自動化され,ほとんどの工場においてオンライン化が採用されている.また,従来,形鋼は冷却床で冷却され定尺切断後に矯正されていたが,近年の特定品種の専用化,生産量の増大に伴って,山形鋼などを生産する量産工場では,冷却→矯正→定尺切断工程によるオンラインでの矯正(a) 単基式配置(b) 一軸式配置(c) タンデム式配置図5・67 代表的な圧延機のレイアウト例1―113切断方式が採用されている.b. 線材精整設備精整設備は,圧延ラインに直結する設備となっており,水冷装置,巻取り機,コイルコンベヤ,直接熱処理装置,コイル搬送コンベヤ,自動結束機,秤量機などで構成される.巻線機(reel)は,細物線材用にレイング式,太物線材用にポーリング式が一般的に使用される.ポーリング式巻線機(pouringreel)では,製品寸法φ55mm,単重3.5tのコイルを巻取れる大形巻線機もある.近年,硬鋼線材については圧延直後の線材が保有する熱を利用し,その冷却制御を行い微細なパーライト組織とするための直接熱処理(direct heat treatment)装置が実用化された.直接熱処理装置の代表的なものは,レイング式巻線機(layingreel)により線材を走行コンベヤ上に非同心円上に展開し,下方より強制空冷を行うものである.このほか,線材の鋼種,用途に応じて,圧延材の熱を利用した各種の制御冷却によるオンラインでの熱処理技術が実用化されている.
