# 5・10・1 継目なし鋼管製造設備

> **章**: 第5章 製鉄機械
> **節**: 5・10 鋼管製造設備

a. 継目なし鋼管製造概要継目なし鋼管(seamless tube)とは,丸または角断面の素材を用いて,継目なく製造された鋼管をいい,製造方法としては,大別して熱間によるものと,冷間によるものとがある.5・10節では,主として,熱間圧延設備について,以下に紹介する.b. 継目なし鋼管の製造法製造される鋼管の材質,寸法などにより種々の製造法が採用されており,表5・10にわが国で使用されている代表的な製造法を示し,その製造法の代表例を図5・70に示す.またさらに精整設備として,曲り矯正機,切断機,管端加工機,水圧試験機,管端ねじ切り機,非破壊検査機なども必要となる.c. 最近の継目なし鋼管製造設備の特徴. せん孔機(piercer)(1) ディスクシュー付マンネスマンせん孔機:従来の固定式ガイドシューに代わって,回転式ディスクシューが採用され,ツールの寿命延長,製品品質の向上に寄与している.(2) 交させん孔機:マンネスマンせん孔機のロール配置を,従来の傾斜角に加えて交さ角をもたせ,せん孔時のメタルフローの改善を行ったもので,パイプ内面傷の解消や高合金鋼のせん孔を可能にした.また,近年この交さ角を15度以上として,より高合金へのせん孔性や,拡管せん孔性能を拡大し,素材・製品材質の多様図5・68 全連続式棒鋼圧延設備の配置例図5・69仕上ブロックミルを設置した細物線材圧延設備の配置例1―114産業機械・装置化,製造スケジュールの柔軟性に対応した高交させん孔機が実用化されている.(3) プレスロールせん孔機(PPM):角ビレットを使用でき,ロールで外形を丸に変形させながらプラグでせん孔する.. マンドレルミル(mandrel mill)近年のマンドレルミルはマンドレルバーの速度変化に伴うストマック現象のような特有な過渡現象を改善し,さらに長尺圧延が可能なリテインドマンドレルミルが主流であるが,マンドレルバーの操作方法により,次の2方式がある.(1) リトラクト法:圧延完了と同時にマンドレルバーをミル入側に引き戻す方法をいう.(2) セミフロート法:圧延完了と同時に,シェル内にマンドレルバーが挿入された状態で,シェルとマンドレルバーが一体でミル出側に搬送される圧延方法をいう.. 計算機制御と計測技術最近の計算機制御技術の進歩は著しく,新設ミルのみならず,従来のミルにも計算機制御システムが導入され,管の長さ,外径,肉厚,温度等を自動計測する計測制・厚肉製造に適す・内外面肌良好・偏肉が少ないマンネスマン-アッセルミル法(mannesmann-assel mill process)φ50.8~165.2mm/厚み10~50mm炭素鋼低合金鋼傾斜圧延方式マンネスマン-プラグミル法(mannesmann-plug mill process)・大量生産可能・製造可能範囲が大きくとれるφ27.2~426.0mm/厚み3.2~50mm低合金鋼以下が主・大量生産に最適・内外面肌良好・偏肉もかなり少ないマンネスマン-マンドレルミル法(mannesmann-mandrel mill process)御技術の進歩と相まって,生産スケジュール,各種自動測定,品質モニタリング,操作の自動化により,品質向上,歩留り向上,省力化,能率向上の面で大きな成果が得られている.
