# 2・6・5 シーケンス回路

> **章**: 第2章 油圧システム
> **節**: 2・6 油圧回路

あらかじめ定められた順序に従ってアクチュエータを動作させる
回路をシーケンス回路と呼ぶ．アクチュエータの位置を検出して動
作する回路と，圧力を検出して動作する回路の2種類がある．図
Ⅴ-2･76は，圧力を検出して動作する回路である．アクチュエータ
①が前進してストロークエンドに達すると圧力が増大し，シーケン
ス弁③が動作してアクチュエータ②が前進を始める．アクチュエー
タの後退時は，シーケンス弁④の働きによってシリンダ②①の順で
動作する．
