# 2・1・2 流体の伝達エネルギー

> **章**: 第2章 流体機械によるエネルギー変換
> **節**: 2・1 流体のエネルギー

断面1から流入する流体は，それより後から来る流体によって押
されており，境界が1から1′ まで移動する間に押込み仕事
dを受ける．これは単位質量当たりにすれば，
dρd＝ρである．一方，断面2から流出する流体
はその前にある流体を押し，境界が2から2′まで移動する間に
dの押出し仕事をする．単位質量当たりでは，
dρd＝ρである．
すなわち，流体が通過するのに伴って，単位質量当たりρ
（J /kg）のエネルギーが各断面を通過する．これを，流体が断面を
図 -2･1固定検査空間におけるエネルギーバランス
通って伝達するエネルギーとみなすことができる．
