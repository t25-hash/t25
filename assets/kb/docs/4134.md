# 5・12・2 検出部

> **章**: 第5章 製鉄機械
> **節**: 5・12 制御装置

機械設備や材料などの制御対象がもっているプロセス情報を検知
する機器を検出器または，センサ（sensor）と呼ぶ．
製鉄設備では，おもに金属の素材を物理的に加工，搬送，仕上処
理を行って製造するので，検出量としては位置，変位，長さ，角度
などの幾何学量が多く，その他に温度，圧力などの物理量が挙げら
れる．検出量は電気に変換されてプロセスI /Oやネットワーク経
由で制御装置に入力される．
製鉄設備に用いられる検出器の種類は，光や電磁気現象を応用し
たもの，さらには材料の物性を利用したものと極めて多様である．
また，使用される数も多く大規模プラントでは，数千個を超える場
合もあり，小規模プラントでも数百個に達する．代表的な検出器を
用途別に分類し，表5･11に示す．
なお，近年ではいろいろな半導体の物性を利用した検出器の開発
が各方面で行われている．また，集積回路技術や微細加工技術を利
用して，小型で高性能の検出器が得られるようになった．
また，これら小型高性能の検出器とマイクロプロセッサやコンピ
ュータを組合わせた検出器も製品化されている．鋼帯の形状検出器
やきず検出器などがその例である．これらは，さらに検出したデー
タを自身で加工処理し，データを長期間保存できるデータベースを
備えたものも製品化されている．

5・12・4プロセス制御（process control）
製鉄設備は，製品の品質向上，歩留り向上および省力化を達成す
るため，設備の複合化，連続化またはコンパクト化が実施されてい
る．

熱間圧延設備においては，連続鋳造されたスラブをトンネル炉へ

冷間圧延設備においては，コールドストリップミルのコイルハン
ドリング時間の短縮や，コイル1本ごとに通板するときに生じるオ
フゲージおよびロールきずの発生を少なくするために，コイルを溶
接する前処理設備を設けて，圧延を停止させない連続ミルもある．
a.材料トラッキング制御 圧延または鋼板処理プロセスの中
で，例えば，品質を作り込むための設定制御や搬送動作のため，機
械を動作させるタイミングを正確にとらえることが必要である．こ
のため搬送材の進行を認識する検出器や，搬送速度を利用した予測
位置検出または，それらを組合わせて正確さを向上させたトラッキ
ング制御方式がある．搬送材の速度は，線材圧延機の仕上げで約
110m/ s以上に達するものもあり，また，棒鋼圧延設備では，加熱
炉から製品倉庫入側までの間に，製品が2000～3000本に達するも
のもある．このように高速で大量の処理が必要なトラッキング制御
は，設備の自動化を達成するための高度な基本技術である．
b.品質制御 高品質の製品を生産するために，素材，鋼種，
温度などと圧延との関係や，機械構造，強度と製品形状などについ
て圧延設備全般の基礎研究，応用研究などがなされている．
品質制御の一例として，ホットストリップミルの仕上スタンドに
おける形状制御（クラウン，平坦度制御）がある．近年，製品の品
質要求が厳しくなっており，これらの制御が注目されている．各機
械メーカにおいても，仕上スタンドのワークロールなどに対し，製
品の形状を向上させるためのさまざまな工夫がなされている．形状
制御のシステム概要例を図5･76に示す．
