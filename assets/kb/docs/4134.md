# 5・12・2 検出部

> **章**: 第5章 製鉄機械
> **節**: 5・12 制御装置

機械設備や材料などの制御対象がもっているプロセス情報を検知する機器を検出器または,センサ(sensor)と呼ぶ.製鉄設備では,おもに金属の素材を物理的に加工,搬送,仕上処理を行って製造するので,検出量としては位置,変位,長さ,角度などの幾何学量が多く,その他に温度,圧力などの物理量が挙げられる.検出量は電気に変換されてプロセスI/Oやネットワーク経由で制御装置に入力される.製鉄設備に用いられる検出器の種類は,光や電磁気現象を応用したもの,さらには材料の物性を利用したものと極めて多様である.また,使用される数も多く大規模プラントでは,数千個を超える場合もあり,小規模プラントでも数百個に達する.代表的な検出器を用途別に分類し,表5・11に示す.なお,近年ではいろいろな半導体の物性を利用した検出器の開発が各方面で行われている.また,集積回路技術や微細加工技術を利用して,小型で高性能の検出器が得られるようになった.また,これら小型高性能の検出器とマイクロプロセッサやコンピュータを組合わせた検出器も製品化されている.鋼帯の形状検出器やきず検出器などがその例である.これらは,さらに検出したデータを自身で加工処理し,データを長期間保存できるデータベースを備えたものも製品化されている.
