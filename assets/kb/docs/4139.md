# 6・2・1 油圧ショベル

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・2 掘削、積込み、運搬機械

a. 油圧ショベルの概要1954年に西ドイツで油圧を使ったショベルが現れ、1961年には日本でも機械質量7tの油圧ショベルが作られた。その後、時代の要求と油圧機器の信頼性向上により大型化が進み、世界最大機は900t級、国産では515tが開発されている。一方、日本において都市土木工事用として小型ショベル(機械質量6t未満)の開発が活発となり、機械質量300kg の小型機も出現した。車格は一般にはバケットの容量で表示するが、用途によりその大きさが変わるので特に大型機では全装備質量で表している。b. 構造上部旋回体は旋回主フレームとその上にのせたエンジン、油圧ポンプ、制御弁、旋回モータ、操縦装置、電気装置、キャブなどからなる。下部走行体は旋回輪、フレーム、走行モータ、走行駆動部などからなる。クローラ式は接地圧が低く軟弱地や不整地の多い土木作業に適している。一般に接地圧は質量の大きさに従って大きくなり、低接地圧用として広幅シューや三角シューを取付けた仕様などもある(図6・4)。小型ショベルでは走行体に排土作業と掘削時のバランスのためにブレードを備えているものが多い(図6・5)。図6・5 小型油圧ショベル(ミニショベル)図6・6 ホイール式油圧ショベル図6・7 ローディングショベル1―126産業機械・装置走行速度は一般に3~5km/hであり、高速化および2速化の方向にある。ホイール式は上部旋回体の動力により走行するもので駆動方式には油圧式と機械式とがあり、最高速度38.0km/hのものもある。掘削時の反力を受けるため、アウトリガ(outrigger)を付けることもある(図6・6)。バックホウでは現場条件に合わせブーム、アームの長さを変えたり、バケットの大きさ、形状を変えたりすることもある。ローディングショベル(図6・7)は主として機械の置かれた地面より高いところを掘削するもので、バケットにはバケット内蔵の油圧シリンダによる底開き式と反転ダンプ式とがある。バケットの水平押出しや水平持上げの操作を容易にするためシリンダを追加した方式やコンピュータにより制御する方式も開発されている。積込時の視界を良くするためハイキャブとしているものが多い。c. 動力系基本的な油圧システムは2ポンプ2バルブ方式である。油圧ポンプはほとんど可変容量型ピストン式(小型ショベルは一部ギヤ式を使用)が使われ、エンジン出力を有効に利用するためレギュレータにより圧力と流量を制御している(図6・8,6・9)。軽作業で負荷の小さいときは、この曲線を下げて(a′-b′-c′)省エネルギーを図る方式もある。またポンプのリリーフ時には流量を少なくしたり(R 点)レバー中立時には流量、圧力とも下げて(S 点)エンジンへの負荷を減らし、省エネルギーを図っているものもある。作業条件に合わせてエンジン出力とポンプ流量、圧力をコンピュータにより制御する方式が増加している。吐出圧はギヤ式で最大20MPa、ピストン式で20~30MPaが用いられ高圧化の傾向にある。ポンプへの油の吸入をよくするためにピストン式では加圧式作動油タンクを使用している。フロントアタッチメントと旋回、走行などの複合動作を円滑にするため旋回優先回路、走行合流回路などが開発されている。d. 操作系その他操作は制御弁を運転室から操作することによって行う。油圧パイロット式が主流となっており、操作レバーのストロークとパイロット圧力の特性およびその圧力を受ける制御弁の流量特性は操作感覚を良くするための重要な性能である(図6・10)。図6・8 油圧ショベルの油圧回路運転室には燃料計、アワーメータ(hour meter)などのほかにエンジンオイルレベル等の点検モニタやエアクリーナ(aircleaner, suction filter)の目づまり警報ランプ等の警報装置を取付けている。運転室内や周囲の騒音を下げるため特別に防音装置を施した低騒音型機もある。e. 作業装置ショベルが三次元の動きをすることおよび油圧配管により容易にエネルギー伝達ができることにより多くの作業装置が開発された。また、そのための予備バルブをもつものも増えた。作業機の動きをさらに拡大するために、ハイリフトフロント、旋回式アーム、テレスコピック式アーム(telescoping arm)、スライドアームなどが開発された。作業装置としてはバケットのほか鉛直につかむクラムシェル(clamshell)、軟岩などのリッピングに使う1本つめリッパ、岩石を破砕するブレーカ、建築物の破壊に使う圧砕機等さまざまなアタッチメントが開発されている(図6・11)。
