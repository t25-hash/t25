# 2・2・2 隔膜真空計（diaphragm gauge）

> **章**: 第2章 真空計測
> **節**: 2・1 真空度

現在われわれが作成，使用できる真空の圧力範囲は，10～10
隔膜真空計は図Ⅳ-2･2に示すような構造で，十分に張力を与え
Paのおよそ15けたに及ぶため，1種類の真空計で全圧力範囲を測
られた金属製ダイアフラムと，セラミック基層上に厚膜蒸着した固
定することは不可能である．そこで，表Ⅳ-2･1に示すような種々
定電極との間に生じる静電容量の変化を測定し，ダイアフラムの変
の原理に基づく真空計が考案・製造されている．なお，表Ⅳ-2･1
には気体の全圧を測定する真空計（全圧計）のみを挙げたが，2･3
節で述べるような気体の組成を知るための分圧計（質量分析計）も
ある．
ついて簡単に説明する．
