# 6・2・4 ダンプトラック(dump truck)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・2 掘削、積込み、運搬機械

土木建設・鉱山の諸作業の中で運搬作業の占める割合は大きい。おもな運搬機械としてはコンベヤ、スクレーパ等があるがダンプトラックは目標運搬量、積込対象機、走行ルート、走行条件(走路こ1―129図6・23 リッパ図6・24 リッパ能力う配、路面状況など)の変化に容易に対応できるというはん用性から道路建設、ダム建設、土地造成、鉱山、採石現場などに広く使われている。ダンプトラックは公道上を走行するダンプトラックと専用の工事・鉱山用道路を走行する重ダンプトラックの2種類にシャシの強度上から大別されるがここでは重ダンプトラックについて説明する。a. 概要公道外を走行することからオフロードダンプトラックともいい、大量の積載物を効率的に運搬し、未舗装路面での過酷な作業条件でも高い耐久性を発揮するよう設計されている。ダンプトラックには、トラクタ部(運転席含む)と荷台部が固定されているリジットダンプトラックと、中折れするアーティキュレートダンプトラックの2種類ある。大きさは積載重量で表し、リジットダンプトラックは22t級から大きいものは220t級までとなっており、アーティキュレート式は、22t級から大きいものは36t級までとなっている。b. 構造ダンプトラックは、トラクタ部と荷台部から構成されており、リジットダンプトラックは後軸駆動、アーティキュレートダンプトラックは全輪駆動が主流となっている(図6・25,6・26)。c. 動力系エンジンの動力は、ロックアップクラッチ付トルクコンバータを介してトランスミッション、ドライブシャフト、ディファレンシャル、終減速機へ伝達される。トルクコンバータはロックアップクラッチ付のため、各速度段においてダイレクトドライブ域があり、燃費効率の高い走行が可能となっている。1―130産業機械・装置図6・25 重ダンプトラックの構造図6・26 車体屈折式全輪駆動ダンプトラック(CAT 740)d. 操作系操作装置は、ステアリング、ベッセル、アクスルコントロール等があり、最近では電子制御式油圧を採用するなどして、操作力を軽減し、操作量を少なくすることによってオペレータの疲労を軽減している。e. ベッセル(荷台)ベッセルは岩石の積込みに耐えられるように材質は高張力鋼が使用されている。また、ベッセルからの荷離れを良くするために、ベッセルのリブの中に高温の排気ガスを通しベッセルの表面を高温にするベッセルヒーティングを採用している機械もある。
