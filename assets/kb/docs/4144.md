# 2・2・5 熱陰極電離真空計（hot cathode ionization gauge）

> **章**: 第2章 真空計測
> **節**: 2・2 真空計の構造および特徴

代表的な熱陰極電離真空計は，シュルツ真空計，三極管型真空
計，B-A真空計（図Ⅳ-2･5）である．熱陰極電離真空計は，熱電
子を放射するカソード（陰極），電子を集めるアノード（陽極)，イ
オンを集めるイオンコレクタの三つの電極で構成される．カソード
はふつう，高温に加熱されたタングステン線のフィラメントで，こ
のフィラメントから放射された電子はアノードへ向かう途中で加速
されて，高エネルギーを得る．電子は進路の途中にある気体分子と
