# 6・4・1 コンクリートプラントおよびミキサ

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・4 コンクリート機械

コンクリートを製造するプラントは、原材料の貯蔵、供給・貯蔵システム、計量・混練システム、操作・品質管理システム、環境・リサイクルシステムの各システムを機能的に組合わせた総合システムである。そのうちのメイン装置であるコンクリートプラントは原材料のセメント、水、砂、砂利および混和剤などを各々個別に、種類別に貯蔵し、これらを所定の配合で計量し、コンクリートミキサに所定のタイミングで投入し、十分に練り混ぜ、品質の一定なフレッシュコンクリートを効率良く、経済的に製造する設備である(図a. コンクリートプラント(concrete plant)コンクリートプラントは、ミキサの形式、容量、生産能力により分類されるが、プ図6・42コンクリートプラントのフロー図1―135ラントの計画には、生コンクリートの用途、生産規模、トラックミキサ車数、設置場所、地形、面積や原材料の入荷条件、工事期間等に適した形式を選択し、周囲環境の条件と原材料入荷および製品出荷の車の流れを考慮した配置に計画する必要がある。b. 計量部各材料の計量は質量計量方式であり、近年の傾向として、ロードセル+ディジタル表示盤方式に移行して、機械部分がなく、維持管理が簡便になるとともに、データをソフト処理して計量精度が向上してきている。操作盤の構成がコンピュータ化されてくるにつれ、コンクリート配合の計量機への設定はキーボード方式になり、その記憶データも数百から数万種類も可能になっている。計量は骨材、セメント、水、混和剤などを個別に計量するが、それぞれ品種ごとに個別と累積計量方式がある。計量機の計量精度は荷重の検出ロードセルの精度とアンプの精度に影響されるとともに、材料が貯蔵槽より円滑に定量流下させる機構とゲートの作動制御に影響される。すなわち、各ゲートおよび計量バルブの開度の適正さは計量時間に影響するが、微量計量の調整、落差補正値の調整等により計量精度を向上させている。さらに、最近は、当バッチの計量速度を瞬時に測定し、落差を推測することにより計量速度を落とさずに高い計量精度を維持できるシステムが開発され、プラントの瞬発力の向上に寄与している。c. ミキサ形式および容量コンクリートミキサ(concretemixer、図6・43)はバッチ式ミキサがおもで、生コン工場用、ダム用として重力式ミキサで始まったが、1960年代には短時間で練り上がるパン型強制練りミキサが生コン向けに普及し、その後1970年代後半からはメンテナンスの簡便なパグミル型の水平二軸強制練りミキサが導入され、この形式のミキサがダム用を含め主力になっている。その後、かくはん翼付き重力式ミキサや最近では、ダブルスパイラル羽根式の水平二軸強制練りミキサが稼働を始めている。コンクリートプラントのミキサは、JIS によりバッチ式ミキサと指定されて、その混練形態から重力式ミキサと強制練りミキサとに分けらる。その性能はJIS(A 8603-1994)に練混ぜ性能、再起動性能、排出性能の基準を示している。. 重力式ミキサ(gravity mixer)古くからコンクリートプラントに用いられているミキサで、混練する材料は回転するドラ1―136産業機械・装置ム内部の羽根によりすくい上げられ、自重(重力)でドラム内の傾斜方向に落下させ、材料全体を移動させながら練り混ぜられる。このミキサは一般に混練時間が長く必要で、プラント内に2台対向に設置し、能力を合わせているので、一式の計量機から2機のミキサに分配するシュートが必要であり、ミキサの機構は単純であるがこのシュート部分が複雑になっている。ドラムの周速が1.1~1.6m/sと遅く、摩耗が少なく、ミキサの保守管理は容易であるが、シュートの保守管理にも注意が必要である。. 強制練りミキサ(compulsory mixer)1パン型ミキサ(pan-type concrete mixer)浅いタライ型の混練槽に減速機を中心に、回転軸には放射状に緩衝用ダンパを介してパドルが取付けられている。このミキサは強制的に練るために、硬練りコンクリートを短時間に練り混ぜられ、トータルのイニシャルコストのメリットから広く普及した。反面、径が大きいため、外周部の周速が3.5m/s程度と速く、羽根、ライナの寿命が短く、メンテナンス費用がかさみ、現在では、特殊な条件下でのみ稼働している。2二軸ミキサ水平な二軸に、それぞれ逆方向に一定の遅れ角で取付けられた数多くのアームの先端に混練羽根が取付けられ、軸間部にもち寄るように回転させることにより、強制的に回転流動、螺旋流動、交錯流動、循環流動を材料に与え、短時間のうちに混練を完了する。二軸ミキサは羽根の回転半径が小さいので、練り回数は多いが、ブレード周速は1.4~2.0m/sと比較的遅く、さらにブレードと底部ライナとの接触面が少ないため、両者の摩耗が少ない。そのため部品交換の周期も長く、計画的な保守管理ができやすい。また底部に大きなゲートが設けられていてコンクリートの排出時間が短いために、出荷サイクルタイムが短くなり、能力の面でのメリットにもなっている。さらに練り容量に比しコンパクトであるために他種のミキサよりも大型機種の製造が可能であり、プラント内の設置にスペースの余裕ができるなどの利点がある。また構造的な弱点である軸シールもグリスの必要のないものも開発されており、この二軸ミキサが現状のコンクリートミキサの主流になっている。図6・44にその構造を示す。二軸ミキサのブレードの取付け遅れ角度は90°から45°~60°に、また油圧駆動やインバータによる可変速駆動方式、さらに、羽根をダブルに設けあるいは連続的に構成し、混練回数を大幅に増やした形が生まれ、現在も発展を続けている。一軸ミキサは二軸ミキサに近似するが、槽の深さと交錯流動と循環流動に相違が見られる。図6・45 トラックミキサ外観図6・43 コンクリート用ミキサの種類3連続ミキサ(continuous mixer)(コンティニュアスミキサ)連続ミキサは各材料を配合割合に応じた容量または質量計量している材料を連続的に投入し、各断面での材料の分布を細分化することで混練を完了し、生コンクリートとなって排出される。よって、この種のミキサは材料供給装置と一体となって機能発揮するもので、現在では、現場練り等の特殊事例に採用されている。. 練り混ぜ時間ミキサの形式によって練り混ぜに必要な時間はそれぞれ異なるが、ミキサの公称容量、配合、練り量、材料の投入順序とそのタイミング、さらにはブレードの摩耗状態などによっても変化するので、練り混ぜ性能試験(JIS A 1119)によって確認しておかなければならない。近年、モルタルの練り混ぜと砂利を加えた練り混ぜを別々の専門ミキサで分担させる階層式のミキサや二軸ミキサでは、インバータや油圧モータで軸回転の変化や、羽根をダブルに設けて混練回数を増す方法で、混練タイムサイクルを短く、プラントの能力の向上を図っている。. ミキサ容量各種ミキサの機械的な容量は練り混ぜるコンクリート材料の量に対して最適の容量が各メーカで設定されており、それが練り混ぜ時間にも影響し、材料が多すぎても、少なすぎても練り混ぜは悪くなる傾向にある。d. プラントの制御機構プラントの材料の受入制御や各材料の計量、放出、練り混ぜ、排出、積込み等の一連の操作はコンピュータにより自動制御で行われ、遠隔操作している場合が多くなっている。骨材、セメントの供給搬送の制御は本体の貯蔵槽のレベル計と供給元の供給ゲートの開閉、またはロータリバルブの回転を連動させ、材料切れを起こさせない確実で、高精度なシステムができている。プラントの制御はコンピュータ化し、計量精度が向上し、各材料の配合設定値、多数の配合選択の容易さ、さらに粒度補正、容量変換、一車積載量を自動分割するなどの操作性が向上してきた。
