# 6・4・2 トラックミキサ(truck mixer)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・42 ).

生コンクリートは練り混ぜ後,時間経過とともに硬化が進み流動性が悪くなったり骨材や水が分離したりするので,これを防ぐため適当なかくはんを回転ドラム内で行い,コンクリートの品質を維持しながら運搬を行う形式のトラックが一般的である(図6・45).そのほか材料を乾燥状態でドラムに受けて運搬し,目的地において水を加え混練する機能をもつものも一部で使用されている.また特殊な形式で現場に到着後,コンクリートの打込み場所までパイプ圧送を行うための小型コンクリートポンプをいっしょに搭載したものも図6・44 二軸ミキサの構造図6・46 油圧駆動方式の機器構成例海外では使用されている.ドラムの最大混合容量は道路運送車両法による車両総重量の制限値が20tから25tに拡大したことに対応し,従来の4.5mから5.2mあるいはセミトレーラ式の6.3mのものも生産されている.
