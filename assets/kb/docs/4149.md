生コンクリートをトラックミキサあるいはプラントのミキサから直接受けて、パイプを通して打込み場所まで輸送する機械である。生コンクリートの性状は対象構造物の種類により多岐にわたっており、この性状や現場施工条件に応じて各種のポンプが使用される。a. ポンプ形式と構造コンクリートポンプは、圧送方式、バルブ機構、架装方式により分類され、コンクリートの性状、打込み対象物の構造、施工条件などに応じて目的に最も適合する機種が選定される。ピストン式コンクリートポンプは、高い吐出圧力が得られるので、コンクリート配合の広い範囲に使用できる。軟練りコンクリート用としてスクイズ式(ゴムチューブによるしぼり出し式)も用いられている。トンネル工事用として、圧力容器内にコンクリートを入れ、圧縮空気により圧送を行うコンクリートプレーサもある。. ピストン式ポンプ(piston type concrete pump)コンクリートピストンの駆動方法、吸入吐出弁の開閉方法にいくつかの種類がある。駆動方式としては機械式、水圧式、油圧式があるが油圧駆動式が一般的である(図6・46)。本体は、コンクリートの供給を受けるホッパ(かくはん装置付)、吸入吐出弁、コンクリートシリンダ、コンクリートピストンにより構成される。コンクリートシリンダは複列式が主流である。大容量吐出量、高吐出圧力を得るのに適している。. 吸入吐出弁による分類ピストン式コンクリートポンプには吸入吐出弁が、最重要部品として組込まれている。吸入吐出弁はホッパのコンクリートをコンクリートシリンダに吸入し、またシリンダから押出されるコンクリートを輸送管へ送給するための切換弁である。吸入吐出弁にはいろいろな形式があり、それぞれ特徴をもっている。. 揺動弁(スイングバルブ)(swing valve type concretepump)揺動弁の形式には幾種類かあるが、いずれもホッパの中で吐出管を揺動させ、左右のコンクリートシリンダの出口に交互に接続する方式である。現在国内で稼動している標準的な揺動弁を図6・48に示す。1―138産業機械・装置このタイプの弁をもつコンクリートポンプの中には、比較的硬練りの土木用コンクリートの圧送に適してないものがあるので注意を要する。この揺動弁に低スランプコンクリートの対応性を改良したものに吸入ガイド付揺動弁がある(図6・49)。. 縦型往復動板弁(滑り弁)[gate valve for concretepump(slide valve)]滑り棒を駆動シリンダで上下運動させる。コンクリートの吸入、吐出は、この滑り棒で支持されている弁板でコントロールされる。図6・50は向かって左側の吸入口は弁板で閉じられ、吐出口は弁板が上がっているので開放され、コンクリートは、コンクリートピストンで押されて吐出される。このとき向かって右側の吐出口は弁板で閉じられており、ホッパからのコンクリートを吸入している状態を示している。. 横型往復動板弁(ゲートバルブ)作動棒を弁駆動シリンダで水平方向に左右の運動をさせる。コンクリートの吸入、吐出はこの作動棒で支持された弁板でコントロールされる。図6・51は向かって左側のシリンダがコンクリートを吸入し、吐出口は閉じられている。このとき右側の吸入口は閉じられ、吐出口が開放されているので、コンクリートは、コンクリートピストンで押されて吐出している状態を示している。. pump)定置式コンクリートポンプは、原動機とポンプを一体としてスキッドの上に架装したもので、一現場で長期間使用するような場合に適する。また、長距離圧送や、高所圧送の中継用としても使用される。最近では、バッチャプラント船に、デストリビュータブームと組合わせて搭載され、海洋土木工事などに使用されている。. トラック搭載式コンクリートポンプ車(mobile concretepump)1配管車コンクリートポンプの動力を、トラックエンジンから取出したもので、機動性が良く、現場間の移動が容易である。現場で輸送管を接続して使用するもので、長距離圧送、高所圧送用の高圧タイプもあり、吐出量も100m/hを超えるものまである。またトンネルなどの坑内での排気ガスを考慮してシャシのエンジンとは別に電動機を搭載し、いずれかに切替えて使用できるタイプもある。2ブーム付現在生産、使用されているコンクリートポンプの形式の95%はブーム付である。これは、地上配管作業の省力化、あるいは輸送管振動による型枠、配筋の乱れ防止、さらには、現場で打設場所を容易に変えられることなど、取扱い性および経済性の面で大きなメリットがあるからである。また、ブームを使用しない場合は配管車と全く同じ使い方もできる。外観図の一例を図6・53に示す。ブームは、油圧屈折式が主で、三段屈折式が多く、近年四段式が増加している。ブーム地上高は30m前後のものが多いが、1993年の道路運送車両の保安基準の改正により車両総質量の上限値が25tまで緩和されたことにより36mのものまで生産されている。ブームの折図6・51
