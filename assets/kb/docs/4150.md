# 6・5・1 舗装道路構築作業と施工機械の特徴

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・5 路盤関係(締固め,舗装など)

交通車輌を走行させるための舗装された道路構造は,一般の車輌ばかりではなく飛行場の滑走路駐機場また港湾の荷役ヤードなどにも採用されており,荷重を担った車輌を円滑に走行させる機能を有し,走行に伴って生じる繰返し荷重に耐える強度と,車輪の回転をスムースに保つ平滑度を備えた路面形状が要求される.走行車輌からの荷重を分散させる機能をもつ路盤の施工には,使用する材料を所定の厚さ(高さ)と幅に敷広げる作業と,その材料が所定の密度を得るための締固作業が必要となる.さらに直接走行車輪と接する舗装表面を構築する作業においては,一般土木作業とは異なり,使用するセメントコンクリートまたはアスファルト混合物の硬化が進行する状況下で,定められた順序に各作業を連続して行わなければならない時間的制約を受ける.さらにこれらの作業は図6・55に示すように,製造業とは異なり,不動の作業対象(work)面上を処理装置(tool)が移動しながら作業を行う形態であり,作業装置には現場間の移動を容易にするため,制約を受けた寸法重量内で必要な各種機能が搭載されている.さらに操作は,あらかじめ測量作業によって設置された基準杭(丁張り)を目標として行われるが,表層部の施工においてはミリメータオーダの精度が要求される.そのため,舗装用施工機械の開発時期と平行して,早くから各種センサを用いた位置制御機構が開発され採用されている.構築に使用される装置は表6・15のように機能別に分類される.以下作業機能別におもな装置(機械)の概要を記す.(a) 製造業:tool(座標軸)が固定されているため,必要な座標が得やすい(b) 建設業:tool(座標軸)が移動するため,相対位置座標の設定が困難図6・55製造業と建設業の作業形態の相違1―140産業機械・装置表6・15 構築に使用される装置の機能別分類作業機能作業対象物装置(機械)名舗装用混合物製造アスファルト混合物セメントコンクリートアスファルトプラントコンクリートプラント使用材料改質路盤材・路床土ロードスタビライザ(自走式)ソイルプラント(定置式)敷均し整形路盤材・路床土アスファルト混合物セメントコンクリートモータグレーダアスファルトフィニッシャコンクリートフィニッシャ振動ローラマカダムローラ,タイヤローラ振動プレート締固め路盤・路床アスファルト舗装
