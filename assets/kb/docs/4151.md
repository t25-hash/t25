# 6・5・2 舗装用混合物製造装置

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・5 路盤関係(締固め、舗装など)

a.アスファルトプラント（asphalt plant) 熱を加えると軟
化し，冷えると硬化するアスファルトの熱可塑性を利用して，使用
骨材のバインダとして舗装用アスファルト混合物を製造する装置で
ある．使用材料を定められた割合で計量し，一練ごとに混合するバ
ッチ式（batch type）と連続して生産する連続方式（continuous）
があるが，国内においては多品質・少量出荷に適したバッチ式が主
流となっている．また加熱リサイクル工法として，新規骨材を使用
するプラントにリサイクルユニットを併設し再生アスファルト混合
物を生産する設備が多くなってきている．
.おもな構成部分と機能 サイズ別にストックされた骨材
（砕石砂）を各使用割合で供給用のフィーダより，引出ベルトコン
ベヤなどでバーナを備えた回転ドラム形状のドライヤに連続投入さ
れる．ここで骨材は170℃程度に加熱乾燥される．加熱された骨材
はバケットエレベータで本体タワー最上部に位置する振動ふるい装
置に運搬され，各サイズ別に分級（4～5種）された後ホットビン
に一時ストックされる．次に1バッチ分の各骨材が使用割合で累積
計量される．同様に，別にポンプアップされた液状のアスファルト
と微粒材料として使用される石粉が，各々計量されて二軸パドルミ
キサに投入され混合された後，アスファルト混合物（asphalt mix-
ture）として排出される．この投入から排出されるまでの混練時間
は約1分を要し，ミキサバッチ容量と時間当たりの混練り回数によ
り時間当たりの能力を示す．1バッチ容量としては0.5～3tまであ
り，定置プラントとしては1～2tのものが多く設置されている．

6・5・3使用材料改質装置
a.スタビライザ（stabilizer) 路盤または路床の支持力を確
保するために，自走しながら路盤原材料または路床土にセメントや
石灰などの土質安定剤を混合する装置である．ゆえに，交通荷重に
応じて支持力を強化する場合や，良質な材料が容易に確保できない
場合にも現地盤改良工法として使用される．さらに従来のアスファ
ルト舗装盤と路盤材を破砕し，同様に混合することによって路盤の
支持力を強化させる再生路盤工法としても使用されている．
.おもな構造と特徴 一般的な構造としては，使用粒状や
土と添加剤を効率良く粉砕混合させるために，先端に耐摩耗硬質ビ
ットを備えた図6･57の形状のロータと，それを回転させ本体を移
動させるために，用途別に応じて150kWから400kWまでの高出
力エンジンを搭載している．車体形状はタイヤ式とクローラ式があ
り，タイヤ式はその移動性を生かして路上で使用されるが，クロー
ラ式は低接地圧による走行安定性を生かして軟弱地盤の安定処理作
業に使用される．
混合深さはその用途に応じて，路盤処理は60cm程度であるが，
軟弱地盤の改良には1m前後の深層混合が可能なものがあり，混
合装置の形状はロータ方式（rotor type）と深層混合用のトレンチ
ャ方式（trencher type）がある．
図6･58に車体形状と混合方式を示す．ロータ方式の場合には構
造が単純であり多く採用されている．またロータを左右にシフトで
きる機能を備えた機種も多い．
