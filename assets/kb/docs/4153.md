# 6・5・4 敷均し・整形装置

a. モータグレーダ(motor grader)モータグレーダは路面地表などを車体中央に取付けられたブレード(blade)により、平滑に切削したり砕石などを敷均し成形することを目的とした車輪式建設機械である。道路の構築(おもに路床路盤の整形)作業、砂利道などの維持補修、法面の切削、整形作業、さらにこの機構を利用して除雪作業にも使用される。そのためブレードは上下、回転、チルト左右シフトが可能な構造となっている。また移動を容易にするため、長い軸間を生かして車体幅より広いブレードを車体フレーム長手方向に回転し車体幅内に格納することができる。図6・59に示す。. finisher)アスファルトフィニッシャはアスファルト合材を所定の幅と高さに(厚さ)に敷均す装置であり、材料を均一な密度で下面の凹凸の影響を受けにくい機構により、平滑に一定の高さで敷均す機能を備えており、用途上つねに移動をするために、運搬上の制限寸法質量内で多くの要素、機能を搭載している。図6・62に示す。. おもな構造と特徴アスファルトフィニッシャは、敷均し装置とそれをけん引するトラクタ部分で構成されている。トラクタは移動性の良いホイール式とけん引力の強いクローラ式があり、用途に応じて使用される。トラクタの型式を図6・63に示す。敷均し装置の厚さ調整装置(スクリード装置)は独特のフローティング機構を採用しており、トラクタに左右2本の先端をピンジョイントされたビーム(レベリングアーム)によりけん引される整形用プレート(スクリードプレート)に角度をもたせることにより、路面に敷広げられたアスファルト合材を下面にくさび状に流入させ、垂直方向の分力と装置の質量とのバランスを取っている。ゆえにプレートの角度を変化させることにより、厚さのコントロールが可能となる。一方、舗装幅員は補助スクリードを油圧装置により伸縮して行う機種が多く生産されている。図6・64に舗装厚調整機構とスクリード装置の原理図を示す。また、この機構に距離計測センサを組合わせた自動厚さ調整機構が使用されている。c. コンクリートフィニッシャ(concrete finisher)コンクリートフィニッシャはセメントコンクリートを所定の幅と厚さに敷均す装置であり、道路用の硬練りコンクリートを使用する場合には、材料に振動を与えることにより流動性を高めて充てん度の高いコンクリート版を成形する。. おもな構造と機種コンクリートフィニッシャは粗均し装置、締固め装置、仕上げ装置で構成されており、締固め装置は起振装置を備えた振動板を50~70Hz で振動させ、コンクリートを締固める。また棒状のインナバイブレータで締固める機種もある。また施工方法として、路上にレール付の鋼製型枠を設置しフィニッシャをレール上で移動させながら打設するセットフォーム方式と、図6・64
