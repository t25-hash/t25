# 6・5・5 締固め機械

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・5 路盤関係(締固め,舗装など)

締固め機械は締固め方式から次のように分類される.1静的荷重方式:ロードローラ,タイヤローラ2動的荷重方式:振動ローラ,振動コンパクタ3衝撃的荷重方式(突固め):タンピングローラ,タンパまたこれらの締固め機構を組合わせた機種も使用されている.一方締固め機械を機械の形態から分けると,ロール状のものとプレート状のものに大別できる.さらに用途別に分類すれば,おもに盛土工などの土工工事用とアスファルト舗装の締固め作業に使用される舗装用とに分けられる.a. ロードローラ(road roller)ロードローラは平滑胴ロールを車輪とし,その配列形式によりマカダムローラ(三輪ローラ),タンデムローラなどの別がある.マカダムローラの自重は8~12tのものが最も多く使用されている.最近ではマカダムローラの形状は車体屈折式で前後輪とも駆動し,かつ同一直径のロールで質量配分も1:1としたマカダムローラが舗装現場では使用されている.タンデムローラは前後二つの平滑ロールを車輪とし,3~10tの自重のものがあり舗装材の締固め作業に使用されるが,この範囲の質量機種は最近では振動ローラに取って代わっている.締固め能力の目安となる線圧(車輪の接地質量をその車輪の接地幅で割った値:単位はN/cm)は10~12tの車体屈折式で430~550N/cm程度である.b. タイヤローラ(pneumatic tire roller)タイヤローラは空気タイヤの特性を生かして,ニーディング作用(こね返し)により敷均されアスファルト混合物の砕石のまわりにモルタルを詰める効果があるので,舗装表面のシール性を高めることができる.さらに砕石を破砕することがなく,舗装体中の砕石の方向を整えることにより,舗装体の安定度を増すこともできる.またバラストを増減させ質量を変えることも可能である.用途としては,路床路盤の転圧からアスファルト舗装の表層の転圧まで,広範囲に使用されている.車体の質量は3~25t位まで用途に合わせて各種あり,前後輪のタイヤの配列は踏み残しのないように奇数と偶数に配置されている.大型の機種は他の締固め機械と異なり,車体に水タンクを備えており,砕石路盤転圧時に散水できる機能をもっている.c. 振動ローラ(vibratory roller)振動ローラはおもにロールに内蔵された起振体により,鉄輪を振動させて自重の1~5倍の動荷重を付加することで締固め効果を上げる機能をもっている.振動数が23~40Hzと比較的低く振幅の大きい土工用と,振動数が40~60Hzと高く振幅の小さい舗装用に分別できる.また起振体の構造は一般的には一軸の偏心軸を回転することにより発生させるが,各種締固め材料によって二軸にして,垂直振動のみを発生させるものやロール円周上に水平に振動させる機構のものもある.図d. タンピングローラ(tamping roller)タンピングローラはロールの外周面に突起物を取付け,突起部先端の高い接地圧を利用して締固める転圧機で,自走式で振動するタイプが主流であり,大型の土工場として使用されている.e. 平板式タンピング機械(plate compactor, vibratory tam-per)平板式タンピング機械には平板に装着させた起振機により,平板を振動させ締固める振動プレートコンパクタと,機関の回転をクランクで往復運動に変えスプリングなどを介して打撃板を上下させ連続的な衝撃力を締固め対象物に与えて締固めを行う振動タンパとがある.ともに機械質量が軽い割に(60~400kg 位)比較的大きな締固め力を有し,ローラなどの転圧機が使用できない狭い場所での締固め作業に使用される.また振動プレートコンパクタの(a) 一般の振動ローラ(b) 水平振動ローラ(c) 垂直振動ローラ図6・68 シールド機の構成図1―143特徴として,偏心軸を回転することによって発生する振動は回転方向に斜めに作用分力するため,本体を締固め作業をしながら移動させる機能がある.さらに大型のプレートには,このメカニズムを利用して前後進を可能とした機種も開発されている.トンネル工法(construction of tunnels)はその施工条件でシールド工法(shield tunneling method)と山岳トンネル工法(moun-tain tunneling method)に大別される.
