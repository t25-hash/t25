# 2・2・4 ジェロンテクノロジー

> **章**: 第2章 福祉工学
> **節**: 2・2 高齢者とジェロンテクノロジー

a.ジェロンテクノロジーの役割 従来，高齢者の病気につい
ては医学が，身体障害や介護技術については福祉工学がもっぱら対
応してきたが，健康高齢者を含めた高齢者全体が抱える生活上の問
題を解決するための明確な技術的分野は存在しなかった．しかし，
高齢社会の急速な進行に伴い，高齢者と製品・情報・環境等の社会
システムとの不適合が顕在化してきたため，健康な高齢者を含むす
べての高齢者が社会的に適合することを促進する技術分野の創出が
必要となった．その結果生まれたのがジェロンテクノロジー（ger-
ontechnology）といえる．この研究分野における役割について，
国際ジェロンテクノロジー学会のホームページでは，「高齢者が良
好な健康状態を保ち，快適かつ安全に自立的な生活と社会参加を可
能にするための，技術及び環境の設計」と示されている．
b.歴史的経緯 ジェロンテクノロジーという言葉は，英語の
gerontologyとtechnologyの複合語として，1980年後半にオラン
ダのアイントフォーヘン工科大学で誕生した ．この技術分野の
目指すものは，二つの言葉が示すように老年学とテクノロジーの合
体であり，老年学等で得られた科学的な知見を，人間生活の向上に
役立つテクノロジーとして社会に発信しようとするものである．欧
米では，日本に比べ人口の高齢化が早くから進行しており，1956
年の「国際連合経済社会理事会報告書」 では，7%以上の高齢化
率をもつ社会を高齢化社会とするとの定義がなされるなど，すでに
高齢化社会への対策が重要なテーマとなっている．そのような社会
背景の中で，1960年前後から高齢者特性とテクノロジーを結合さ
せた論文が人間工学や心理学の分野で出現し始めた．1970年代か
ら1980年代にかけて，人口の高齢化問題が社会的に重大な問題と
して広く認識され始め，アメリカ人間工学会の学術誌における加齢
に関する特集号の企画や，国際人間工学会における加齢技術部会の
形成など，この分野の研究が盛んになった ．1991年にジェロン
テクノロジーに関する最初の国際会議が開催され，2回めの国際会
議の翌年1997年に国際ジェロンテクノロジー学会が誕生した．わ
が国においては，高齢化率が1970年に7 %になった後も急激に増
加を続け，2000年には17%を超える世界一の高齢化国家となった．
そのような状況の中で，1999年に日本人間工学会の部会として，
日本で初めてジェロンテクノロジーの名前を冠した組織が誕生し
た．2000年にはジェロンテクノロジー国際ワークショップ が筑
波で，さらに2005年に国際ジェロンテクノロジー大会が名古屋で
開催され，この分野に関する期待が急速に高まった．
c .成果 基盤的な成果としては以下のことが挙げられる．
(１) 高齢者特性の理解に関する多くの研究が実施された結果，
高齢者特性データベース（characteristics database of the elder-
ly）の整備が世界的規模で飛躍的に進展した．わが国でも，(財)人
間生活工学研究センターが3万人規模の人体寸法・形態データとと
もに，数百人から数千人規模の視聴覚や動作特性に関するデータベ
ースを構築中である．
(２) 高齢者特性データをディジタルヒューマンモデルに入力す
ることにより，人と環境，製品とのインタラクションの評価がコン
ピュータ上で可能になった．
(３) 高齢者対応製品の設計指針となる国標準作りが盛んになっ
た．2001年には，わが国主導で提案された，高齢者・障害者のニ
ーズに配慮した規格の策定に関する指針がISO/IECガイド
71（ISO/ IEC Guide71）として承認された．
また，応用的な成果としては以下のものがある．
(１) 住宅や公共的環境，公共交通機関等におけるバリアフリー
が進み，高齢者の生活安全や社会参加の促進が見られた．
(２) 製品におけるユニバーサルデザイン化が進み，高齢者にも
使いやすい製品が増加し，高齢者の行動の自立促進に役立った．
(３) 人間特性に配慮した福祉機器の設計が可能となった．
(４) 高齢労働者が定年後も就労可能な職場改善がなされた．
