# 2・3・1 福祉機器を取り巻く問題

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

福祉機器は，使用者の生活に密着に結びついている．したがっ
て，使用者の障害や特徴，介護者の状況，使用環境，社会制度など
多くの問題が複雑に絡み合い，福祉機器の開発や設計，使用に対し
て影響を及ぼす．福祉機器を考えるには，これらを総合的に考慮し
なければならない．
最も基礎的なことは，障害の医学的な理解である．使用者は何ら
かの身体的障害をもっているため，それを高進させないことが福祉
機器には要求される．また，障害の特徴を把握するとともに，障害
の個別性（individuality）を考慮する必要がある．例えば，頸髄損
傷の人が使う電動車いすを，そのまま脳性麻痺の人が使えるかとい
うと，そんなことはない．頸髄損傷の人には軽い力で動かすことの
できるジョイスティックが適しているのに対し，脳性麻痺の人には
不随意運動の影響を受けにくい硬めのジョイスティックが適してい
るかもしれないからである．
なわち，大量生産によるコストダウンが図れず，むしろ多品種少量
生産が求められる．究極の福祉機器は，頸髄損傷の人用，脳性麻痺
の人用というように，オーダメイドで作ることになってしまう．し
かし，これでは経済効率が上がらない．そこで，モジュール化やア
ジャスタブル化を図ることが行われている．これにより，共通の部
品としてある程度の量を確保しながら生産することができる．ま
た，CAD（computer aided design)/CAM（computer aided man-
ufacturing）システムを利用した個別生産システムを構築し，一人
ひとり違ったものを効率よく生産する試みもなされている．
