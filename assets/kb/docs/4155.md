# 6・6・1 シールド掘進機(shield machine)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・6 トンネル掘削

シールド工法は,わが国の都市の地下空間に建設される地下鉄,上・下水道,電力・通信設備の施工技術として利用されている(図用途拡大や効率化の目的で実用化されている.シールド掘進機は,前面の構造によって開放型および密閉型に大別される.開放型シールドは全面開放型と部分開放型に分けられ,さらに全面開放型は掘削方法によって手掘り式,半機械掘り式,機械掘り式シールドに,部分開放型ではブラインド式シールドに分類される.切羽の安定を図るため,手掘り式シールドでは,フード部に切羽を機械的に押さえる山留装置が装備されている.密閉型シールドは土圧式と泥水式シールドに分けられる.各機構について密閉型を例に取ると,シールド本体は,外部から作用する荷重に対して内部の機構・装置を保護する鋼殻部分であり,フード部,ガータ部,テール部で構成されている.密閉型ではフード,ガータ部に隔壁がある.フード部はカッタヘッドで切羽の地山を掘削して取込んだ土砂を排土装置へ送る排出路であり,さらに切羽の安定を図る機構と装置をもつ.ガータ部はカッタヘッド駆動装置,排土装置,シールドジャッキ等の機器と装置を格納する空間として利用され,シールド機にかかる荷重を受ける主骨格部となる.テール部内は,セグメントの組立を行うとともに,テールシールを後端に配置して止水機構をもたせている(図6・68).本工法では,シールド掘進機を地中に掘進させ,周辺図6・66 各種の振動機構図6・67 シールド機本体例(8.66mφ泥土圧式)(c) 泥水加圧式(b) 土圧式(a) 手握り式1―144産業機械・装置図6・69特殊断面シールド例図6・70 特殊断面複円形シールド(3連円形シールド)例の地山の崩壊を防ぎながら,その内部で,掘削からずり搬出,さらにセグメントに代表される覆工作業を安全な環境下で行いながらトンネルを構築する.密閉型でカッタヘッドを有する機種の場合,カッタヘッドで切羽を押さえ,かつそれを回転させて前進する.その際,装備トルク(tf・m)はシールド外形(m)より=αの式で表され,α(トルク係数)は,機械掘り式で0.8~1.4,土圧式で1.4~2.3,泥水式で0.9~1.5程度となっている.またシールド掘進は,建て込まれたセグメントを反力として推進する.その推進力は土圧,泥水式で単位断面積当たり1000~1300kPa程度である.テール部にはセグメント組立のためのエレクタが装備されている.シールド掘進機は,種々の新機構が開発されており,上述した大断面多様化(図6・69,6・70)のほかに,急曲線施工で施工精度や地盤沈下抑制のためにシールド本体の中折れ構造の多分割化や,れき層あるいは一般土層と岩盤層という複合地盤を掘削するシールド機も実用化されている.この複合地盤掘削機種には,岩盤層ではグリッパで地山から直接反力をとって推進(グリッパ推進)し,一般の土砂地山では通常のセグメントに反力をとるシールド推進を行うタイプもある.このようにシールド掘進機の用途はもとより,地山条件でも範囲を拡大する傾向にある.
