# 6・8・1 道路維持修繕用機械

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・8 道路・河川維持修繕機械および災害対策用機械・

道路の機能を常時確保する必要から効率的な機械と工法が求められるため道路巡視,道路点検調査,路面と各種施設の機能維持のための清掃作業などの用途別に,それぞれの作業に適した機械類が考案導入されている.(1) 道路巡視用機械は,道路パトロールカー,路面補修機械を搭載した巡回作業車などがあり,路面状況,道路付属物の点検や通行車両からの荷崩れなどによる落下物の処理作業に使用される.パトロールカーの仕様は,おおむね四輪駆動車が用いられ,黄色灯,赤色灯を備え通信機器を搭載しており,一部の車輪は標識装置探照灯をも装備している.(2) 調査点検機械(research maintenance equipment)には,路面性状測定,滑り抵抗測定,路面下空洞探査測定(車道用および歩道用,図6・95),トンネル内壁空洞探査,橋りょう点検に使用する専用車両がある.(3) 道路路面・道路付属施設清掃機械は,車道路面のじんかい(塵芥)を清掃して,路面の機能と環境を維持する.路面清掃車,道路側溝の集水桝や側溝そのものに堆積した土砂を除去する側溝清掃車(drainage cleaner),排水管内の堆積物を除去する排水管清掃車,トンネル内壁を清掃するトンネル清掃車,ガードレールを洗浄するガードレール清掃車(guardrail cleaner),道路案内標識を1―151図6・95路面下空洞探査車清掃する車両,さらには近年タイヤ走行騒音を吸収することで,施工が増加している排水性舗装の機能維持と回復を目的とした清掃車,路面の洗浄および散水に使用する散水車などだが,ガードレール等,道路付属物は車道側の清掃である.道路の原点である歩行者側からの清掃も一部の国道では実施されるようになり,ガードレール・ガードパイプを歩道側から清掃するバッテリ駆動の車体後部立位搭乗式の清掃車が開発導入されている.また歩道植栽などに投げ込まれたコーヒー,ジュース缶類やタバコなどを吸引してしまう人力けん引式ポイント清掃機なども登場している.(4) 路面清掃車(road sweeper)はブラシ式(brush type)と真空吸込式(vacuum type)の2種に分類されるが,いずれも道路路面清掃専用車として製作され,ブラシ式は専用シャシの前方側面に傘型のブラシを取付け,歩車道境界のガッタ部で回転させ,じんかいを車道部方向に送り出し,これを円筒型のメーンブラシにてベルコンに乗せじんかい収納ホッパに積込むものである.当初は前部二輪後部一輪(ダブルタイヤ)の後部ステアリングの三輪方式で,小回り性を確保していたが近年HST 駆動四輪操舵の採用により作業速度30km/h回送時前輪操向60km/hを確保した都市型清掃車に進化している.また真空吸込式はトラックシャシに清掃装置専用機関付排風機とじんかい収納ホッパを架装したもので,近年の主流機械であるがじんかいの初期移動を容易にするためにガッタブラシ,吸込ブラシはブラシ式と同様についているためブラシ周りで粉じんが舞い上がり,事前散水が必要であった.あわせて,ガッタ部に集まっていた細粒土が車道中央方向に掃出されることから排水性(低騒音)舗装の目づまりが懸念されていた.これを解消する目的でガッタブラシとサクションホース吸込部を接近させ,エアーフードを設けた一体構造とし,じんかいをガッタ部付近にて吸込む方式が考案され清掃作業の効率性と清潔性が確保され環境衛生上の問題克服も間近と思われる.(5) 道路付属物清掃車であるトンネル壁面清掃車,ガードレール清掃車,道路案内板清掃車,照明灯清掃車の洗浄用装置は,洗浄時に使用される水の飛散防止構造をもったブラシ式が主流である.トンネル壁面清掃については,固有トンネル専用の装置として設置されているものもあるが,大多数はトラックシャシに装置を架装した専用車タイプである.多目的作業車にトンネル内壁各部位ごとの清掃ブラシを取付けたものがあり,車輌左側前部にトンネル監査路下部壁面清掃用ブラシ,車両後部荷台上に側壁部およびトンネルアーチのコーナ部清掃用ブラシ3本を架装している.また透光性遮音壁清掃車として,上部の湾曲部および垂直の遮音壁を1本のブラシにて清掃する機械もある.さらに,海岸沿線道路における透光性遮音壁については潮風の関係で外側に塩分がへばり1―152産業機械・装置付き,内側は排ガス等により汚れて透光性が失われる.これらの汚れを内外同時清掃する機械装置も開発されている.なお,これら透光性遮音壁清掃用のブラシは毛足を細く長くすることにより透光板表面上のすり傷をおさえて,透明性を持続させている.(6) 排水管清掃車(suction cleaner)は,高圧水噴射により路面下横断管などの清掃をするもので,加圧した水をホースの先端に取付けた特殊型状ノズルより前方および後方噴射してその反力によりノズルが前進する.これを引戻し前進を繰返すことで管内を清掃する.いい換えれば前方噴射は,堆積した土砂を射掘し,後方噴射は射掘した土砂の後方送りと合わせて,ノズルとホースを前進させる推力となり,管内土砂を開口部までかき出して,側溝清掃車により吸上げて処理する.また,一部には使用する水を回収し,水と泥・砂に分離して浄化した水を再利用する機械もあり,この機械では,従来型の作業に必要な給水車,吸引車がなくとも作業が可能である.この清掃は高圧水を噴射しての作業のため圧力水が外部に噴射しないよう流末などを十分調整して作業することがたいせつである.(7) 側溝清掃車は集水桝や側溝に堆積した汚泥を吸汚管を介して,ホッパに吸込む方式で,ロータリブロア式,真空ポンプ式,ターボブロア式がある.トラックシャシに吸込み装置とホッパを架装したものが一般的であるが,吸引車とホッパ車を専用化したものもある.なお,ターボブロア式のものは吸込管経が200~250mmと粗大ゴミの吸込みが可能であるが,吸引距離は短い.ロータリブロア式,真空ポンプ式のものは遠距離の作業が可能であるが,吸込管経は75~125mmと小さくなるが,この方式は吸込車1台に対してホッパ車を数台組合わせることにより連続作業や大量処理を行うことができる.(8) 道路路面維持補修用機械には通行車両の大型化と交通量の増大に伴い轍掘れや,摩耗した舗装部分の切削りや全面切削りをする路面切削機,段差のすり付けや,ポットホールの補修,小規模なパッチングなどを行う路面補修車,締固め機械,アスファルト合材機を作るプラント,区画線を引くラインマーカなどがあり,道路輸送が物流の76%(運輸省貨物地域流動調査)を占める現状を考えれば常時道路機能を確保する必要があり,維持補修作業においては,作業により生じる交通渋滞を少なくする効率的な機械と工法が求められている.(9) 自走式路面切削機(road planer truck)は,アスファルト舗装面でしばしば発生する轍掘れに起因するコブなどを取り除き道路環境の改善を図る目的で開発された機械である.切削作業は1~5m/min,移動走行時90km/h全輪駆動(2WD と4WD 手動切替式),切削幅は左1000mm,右650mm,最大切削深85mm,切削用カッタはマシン外側へ左右各200mmスライド可能,作業時ステアリング操作は助手席の作業用操作盤(切削部監視カメラ付)と後部作業用リモコン操作盤による.本機は自走高速移動と作業時の微速切削およびカッタ部のスライド機構により作業性が良い(同種の路面切削機は,自走速度の関係から現場間をトレーラなどによる移動の必要があった).(10) その他の維持用機械としては盛土法面等の草刈機,植栽せん定車等がある.
