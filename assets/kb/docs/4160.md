# 2・3・6 福祉機器開発手法

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

福祉機器開発でよくいわれることは，しっかりとしたニーズ
（needs ）を把握して機器開発に臨むことである．これは非常に重
要であり，ニーズをつかみ損ねたために，全く見当違いの機器を開
発してしまう場合も多く見受けられる．しかし，ニーズばかりを考
えて機器開発を進めてしまうと，工学技術の進歩は減速し，技術の
限界を迎えてしまう．特に，重度障害者を対象とした福祉機器開発
では，技術シーズの進歩がなければ越えられない問題が多くある．
したがって，工学としての技術シーズをしっかり開発することも重
要なファクタとなる．
以上を踏まえたうえで福祉機器開発を次の三つの段階に分けて考
えることができる．
