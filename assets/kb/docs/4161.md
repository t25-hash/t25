# 6・8・3 災害対策用機械

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・8 道路・河川維持修繕機械および災害対策用機械・

地震大雨などによる自然災害が発生した現場での復旧活動の指揮を取る対策本部用として、移動式会議室兼宿泊設備である対策本部車、待機支援ユニット、待機支援車がある。また昼夜兼行で行う復旧作業には、土工用重機械のほか、夜間作業用に照明車、現場状況により応急組立橋、流出油回収車、水質監視車、土のう造成機と堤内地排水用として、排水ポンプ車などがある。また、二次災害の防止を含めた情報収集・伝達用として、ヘリコプタをはじめ衛星通信車、巡視点検用として、水陸両用車などがあり、これらを有効活用し被災地の救援や復旧作業を速やかに、かつ継続的に実施する体制を確立しておくことが肝要である。(1) 対策本部車4×2,4×4の5tトラックシャシをベースとして電動拡幅型アルミパネル構造で厨房設備、電気燃焼式トイレ、簡易宿泊設備、簡易打合せ室をもち、発動発電気エアコンを備え前線本部として機能すべき情報収集伝達設備(VHF,UHF,BS 放送受信設備、無線装置)目視用デッキをもっている。支援車は拡幅機構のない簡易宿泊設備であるが、4人程度の打合せ可能スペースと乾燥設備、燃焼式トイレ、電子レンジを備えている。支援ユニットは、本部車から情報収集伝達設備を除き、かつ自走機能を省いたものである。近年都市部の河川流域において、時間雨量が少し多いと浸水災害が頻発する傾向を受け、国土交通省は、水害の恐れのある都市河川を指定し、流域での新たな土地開発を制限し、国と自治体が共同で流域内に雨水をためる施設を整備する法案をまとめた。都市型水害の増加は流域内の上流部に残されていた田畑や山林が開発により消え雨水が浸透せずに舗装道路や下水道を一気に流れ河川や窪地に集まり周辺部を浸水被害地としてしまうとされる。このため指定河川の流域では開発に知事の許可が必要となる。排水ポンプ車(drainage pump truck)は、車両1台に発電設備、排水ポンプとホース類を一式搭載した自走式排水設備であり、特筆すべきことは、排水機場などの大規模な施設を作ることなく浸水災害現場の直近まで侵入できるなどの機動性と短時間内に排水を開始できることである。排水ポンプ車に搭載している排水ポンプには複数のタイプがあり、水中モータタイプポンプ、高圧水力駆動タイプポンプ、油圧駆動タイプポンプがあり、ポンプ駆動動力によりポンプ単体質量が異なる。ポンプ単体重量の軽い排水ポンプ車には、機付クレーン装置を備えているものが多く、ポンプ本体の積降作業と照明設備支柱として使用している。ただし大容量のポンプの場合は大型移動式クレーンが必要となる。アンダパス排水用の4.8m/min~150m/min級まであり、30~45m/min級がはん用タイプである。
