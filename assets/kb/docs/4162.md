# 6・8・4 除雪機械(snow remover)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・8 道路・河川維持修繕機械および災害対策用機械・

旧運輸省調査の貨物地域別流動調査によれば国内の貨物総量約77億tのうち、航空9%、海運8%、鉄道7%、残りの76%実に58億tを自動車輸送が背負っている。対馬暖流が北上する日本海を挟んでユーラシア大陸と対峙するわが島国日本は、その地理的条件から、世界有数の豪雪地帯、日本海に面した国土の60%が雪積寒冷地域であり全人口の23%(約2900万人)の人々が生活している。ちなみにこの地域の物流は28億3千万tに達している。また、この地域でのGDPは日本全国GDPの約28%に達している。日本海に面した平野部に降る湿気の多い重い雪は、道路除雪の草々期に導入した機械類では歯がたたず、1963年時の北陸豪雪では北陸3県の道路設備を数週間にわたって途絶させ多量の降雪は社会的災害と認識された。以後、官指導による開発改良を継続して、日本型ともいえる機械を開発配備し道路構造の改良と相まって、その後は1963年の豪雪を超える降雪にも耐え国道の交通途絶は発生していない。a. 除雪機械の種類除雪機械類は、その作業区分から4種類に大別される。. dozer)土工用のホイールローダの改良型でローダバケットを取り外してアングリングプラウ等を付けた車両である。小回りがきいて狭隘道路や多雪地域の拡幅除雪などに適している。バケットアームにプラウを付けたためにブレードを高所まで押上げることができる利点がある。回走時のピッチングを防止するダイナミックダンパ付車両のほか、70km/h回走可能の機種もある。. ロータリ除雪車(rotary snow 級~441kW 級まであり、大型機種には回送時70km/h走行の機種もある。. スノーローダ(snow loader)積込み排雪作業としてバケット式とロータリ式がある。バケット式ははん用のホイールローダを用いるのがほとんどで、使用するバケットを容量の大きいスノーバケットとし、サイドダンプ機能をもたせたものもある。ロータリ式は、シュートの先にスライダまたはベルトコンベアを取付けて車体後方に排出し、一車線内にてダンプトラックに雪を積込む方式であるが、ベルトコンベアの場合は、ダンプトラックの運転室上部を越えてベッセルに雪を積込むため、作業現場の管理が容易である。. 凍結防止剤散布車(truck mounted spreader)路面の凍結を防止、防滑を目的として路面上に均一に塩または塩化カルシウム等の凍結防止剤を散布する機械である。固形剤のみを散布する方式を乾式散布式といい、固形剤に塩化カルシウム溶液、または塩水を混合し同時に散布する方式を湿式、または湿潤式と呼んでいる。一般的に4t~10tトラックシャシに2.5m~8mの散布剤を積載するホッパを搭載し、大きさの表示はホッパの容量で表している。. 安全性・操作性・機能性の向上についてプラウ系、ロータリ系、散布車については外観上、一昔、二昔前の機械とほとんど変化が見られないために、20年前の機械と変化のない機械を作り続けているなどの声が多々聞かれるが、各機械の安全性、操作性、機能性の向上はめざましいものがある。例えていえば、プラウ系ではブレード荷重自動制御装置(線圧調整)ブレードの安全装置のレバーによる復元機構、油圧操作によるアングリングプラウ左右方向変更機構、ロータリ系ではシュート操作などは多数のリンクレバー機構をジョイスティック化、オーガ装置のシャーピンレス機構などがある。散布剤の車速同調による効率化散布やRTK-GPS 利用の自動散布等や時代の先端技術の実用化が図られている。環境保全・リサイクル機械とは、建設・土木工事に伴い発生する建設副産物(図6・101)、産業廃棄物、および家庭から発生する粗大ゴミ等の処理を行い、従来廃棄物として処分していた物を、再利用可能な製品にする機能を有するものをいう。日本では、1991年に制定された廃棄物処理法(図6・102)が起点となり、環境保全・リサイクル機械の需要が高まった。本分野は今後もさらに拡大する傾向にあり、従来工法から、現場内循環型工法(on site recycling method)への大きな転換期を迎えようとしている(図6・103)。本リサイクル工法を活用することにより、新材使用量を低減し、砕石、山砂利採取による環境破壊の進展を防止するとともに、運搬に係わるエネルギー消費も抑えることができ、地球温暖化や大気汚染の抑制といった副次的効果も期待できる。また、表6・16環境保全・リサイクル機械の分類アスファルト・コンクリート塊コンクリート塊コンクリート・アスコン破砕機械(自走式または、被けん引式破砕機)路盤材、埋戻し材、裏込材建設発生土土質改良機械(自走式土質改良機)改良土廃木材、伐根、枝葉木質系破砕機械(自走式または、被けん引式木材破砕機)堆肥、燃料、パルプ原料、マルチング材混合廃棄物・粗大ゴミ(産業廃棄物)せん断機械(自走式せん断機)減容化(最終処分場の延命化)土砂・岩塊混合物(不定形混合廃棄物)スクリーン(選別機械)(自走式または、被けん引式スクリーン)栗石、路盤材・埋戻し材・裏込材、骨材、砂最終処分場の残存容量(年数)が限られており、猶予年数が非常に逼迫しているという理由もある。環境保全・リサイクル機械は用途別に、5分類(表6・16)に大別図6・101
