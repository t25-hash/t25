# 6・9・1 コンクリート・アスコン破砕機械(図6・104)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・9 環境保全・リサイクル機械

建設廃棄物(waste from construction site)の種類別排出量において、全体の72%(1995年度)を占めるアスファルト・コンクリート塊、コンクリート塊、および自然石を破砕対象とする。現場にて小割された原料を破砕するため、クラッシャには一次クラッシャとしての機能が要求され、破砕後製品は、そのまま、あるいはふるいを通して使用される。鉄筋コンクリートも破砕可能であり、その際、鉄筋はコンベア下流に装着された磁選機により吸着され、製品とは別に排出される。従来は定置プラントにて処理が行われていたが、機動性、はん用性の面から、被けん引式さらには、自走方式へと工法が変化してきている。原料の種類と破砕後製品の用途により、クラッシャ形式も数種存在する。それぞれのクラッシャの特徴は次のとおりである。a. ジョークラッシャ(jaw crusher、図6・105)破砕機として最も一般的に古くから知られているタイプであり、構造が簡単で頑丈であり、機械の大きさの割には大塊処理が可能である。動歯の駆動方法の違いで、ダブルトグルとシングルトグル方式の2種類がある。材料はV 型断面をもつ破砕室上部から投入され、動歯の働きにより、圧縮破砕→自由落下を繰返し、破砕室下部より排出される。排出口のすきまをあらかじめ設定することにより、希望サイズの製品を生産することができる。はん用性に富み、硬い自然石まで対応可能である。b. インパクトクラッシャ(impact crusher、図6・106)衝撃図6・107 自走式土質改良機械1―155を利用した破砕機。打撃板を取付けたロータを高速で回転させ、上部より投入された材料に打撃・衝撃を与え、さらにこれを反発板に衝突させて破砕を行う。自生粉砕(材料どうしの破砕)も加わることで、破砕物の形状が球形に近くなり、整形効果も期待できる。c. ロールクラッシャ(roll crusher)二軸の平行ローラ間に材料をかみ込み、圧縮破砕を行う。細割機能をもたせるため、位相を変え二軸のローラを上下多段に組合わせたクラッシャもある。
