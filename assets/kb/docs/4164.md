# 6・9・2 土質改良機械(soil recycler)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・9 環境保全・リサイクル機械

建設工事現場からの建設発生土は,全国で約4億3700万m発生し(1995年度),年々増加する傾向にあるが,再利用率は3割程度に過ぎず,処分地不足や不法投棄問題を起こしている.この建設発生土を現場内処理する機械の代表が,自走式土質改良機(図6・107)である.機械の基本構成はコンクリート・アスコン破砕機械に近似するが,建設発生土(原料土)を改良する固化材用ホッパがフィーダ上部に設置されている点が大きな相違点である.この固化材添加量は土質に応じて,任意に調整可能であり,建設発生土だけでなく,地盤改良,砕石ケーキの改良等に応用できる.混合方式はロータリハンマ方式,パドルミキシング方式の2方式が一般的であるが,粘性のある土でも混合性が良いことから,ソイ図6・105 ジョークラッシャ(シングルトグル方式)図6・106 インパクトクラッシャ図6・104 自走式コンクリート・アスコン破砕機械1―156産業機械・装置ルカッタ+三軸ロータリハンマ方式も採用されている.
