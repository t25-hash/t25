# 6・9・3 木質系破砕機械(wood grinder)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・9 環境保全・リサイクル機械

建設廃棄物の種類別排出量において,全体の6%(1995年度)を占める建設発生木材,および伐根,廃木材は,リサイクルがあまり進んでおらず,再利用率は建設発生木材で4割程度である.しかし,野焼き禁止,焼却処理によるダイオキシン発生問題等,木質系廃棄物の処理問題解決は急務となっている.自走式木材破砕機(図6・108)は,これらの背景を受け急速に市場を拡大している.破砕機は,カッティングロータタイプ,ハンマミルタイプが一般的であり,いずれもロータを高速回転させ,ロータ外周部に取付けられたビット・プレートカッタ等にて原料を粉砕する.原料の供給方法は上方投入タイプと横入れタイプがある.図6・108 自走式木材破砕機械図6・109自走式せん断機械図6・111 自走式スクリーン
