# 6・9・4 せん断機械(shredder)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・9 環境保全・リサイクル機械

家庭からの粗大ゴミ(洗濯機,自転車等),産業廃棄物等の幅広い対象物をせん断・減容化し最終処分場の延命化を主目的として開発された機械である.これら廃棄物の処理は1箇所集中化,大規模化が進めば進むほど周辺住民の同意が得られにくいのが現状であり,廃棄物発生の源流のできるだけ小規模な段階で処理することにより,周辺環境への影響を抑え,社会的トータル処理コストの低減が可能となるため,機動性に富む自走式タイプ(図6・109)が脚光を浴びている.破砕機としては,二軸せん断機(図6・110)が一般的に広く使用されている.基本は,平行に配置された二軸が互いに逆方向の回転をし,原料をかみ込み・破砕する機構であるが,二軸の回転数に適当な差をもたせることによるせん断力と,隣り合った刃と刃の間の図6・110 二軸せん断機表6・17 製品サイズと用途製品サイズ用途100mm~捨石,栗石40~100mm栗石0~40mm裏込材,埋戻し材,路盤材5~25mm骨材0~5mm砂せん断力により破砕効率を向上させている.破砕機のカッタは外周部分のみを分割交換可能としたピースカッタ構造と,一体型のモノカッタ構造がある.
