# 6・9・5 スクリーン(選別機械、screen)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・9 環境保全・リサイクル機械

スクリーン自身は、土砂・岩塊、不定形混合廃棄物から、希望の製品・サイズを選別する機械であり、古くから使用されているが、近年では環境保全の観点から、スクリーンの利用頻度が増えてきている。第一の利用方法は、6・9・1項にて紹介したコンクリート・アスコン破砕機械から排出された製品を選別するもので、使用目的に合った高品質・高付加価値の製品(表6・17)を生産できる。第二の利用方法は、処分地の減少・遠隔化が要因で、砕石・砕砂・骨材の生産・処理費用が高騰する中で、従来は廃棄していた土砂・岩塊混合物、不定形混合廃棄物を選別し、無用だったものから新たな価値を生み出す使用法である。この場合も機動性の面から、移動式が主流となってきており、近年は被けん引式から自走式(図6・111)へと変化してきている。
