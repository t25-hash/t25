# 6・10・2 建設ロボットの発達史(表6・19)

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・10 建設ロボット(robot for construction work)

1965年代には、建設機械搭載の作業装置の自動制御や建設機械の遠隔操縦が始まった。自動制御の対象として、浮子式の傾斜角度計を用いたモータグレーダのブレード接地角度制御機構、レーザ光線やGPS(global positioning system)を利用した水平認識装置を使用したブルドーザのブレード接地高さ制御機構(図6・112)などがある。遠隔制御機構を採用した機械として、製鉄所の炉下や鉱滓処理を行うトラクタショベル(1967年)、浅海工事用の水中ブルドーザ(1968年)などがあり、危険・苦渋な現場での遠隔操縦機械の導入の先鞭をつけた。水中土工の遠隔施工(図6・113)が1990年代の無人化施工に発展した。1975年代に入りマイクロプロセッサの集積化(IC)による制御装置の小型化と信頼性向上と油圧サーボ機構の高圧化・小型化が進んだ。これを契機に建設機械の自動化・ロボット化の研究が活発化し、クレーン、ダム用コンクリートバケットの脱着装置の遠隔操作化(図6・114)、シールド掘進機械や計測の自動化などが行われた。1980年前後から、コンクリートプラント、地下連続壁掘削機、1―158産業機械・装置グラウト機械、NATM(New Austrian Tunneling Method)用コンクリート吹付け機など、コンピュータ制御を装備した建設機械の生産が始まった。転圧機械の無人運転などの研究が行われ、建設ロボットという言葉が用いられ始めた。1980年代の石油危機を契機に、施工能率の向上、省力化、省エネルギー化の要望がいちだんと高まった。ワンマンコントロールが可能な油圧ジャンボ、コンクリート吹付けロボット、小口径推進機械などが発表された。また、自動化の程度がさらに進んだシールド機械、サーフェスリサイクル機械、耐火被覆材吹付けロボット、大深度地下連続壁施工機械、転圧ローラの自動運転などの機械が相次いで発表された。メカトロニクス技術を利用したいわゆるメカトロ建機と呼ばれる機械が急増した時代であった。1985年代に入ると専用機の自動化が目立つようになり、自動玉掛け外し装置、重量鉄筋配筋機械、コンクリート床均しロボット(図6・115)、ビル外壁塗装ロボット、外壁タイルはく離診断ロボット等各種の機械が開発されている。複数の機械群の管理を前提とした振動ローラの自動化システムの開発研究も行われた。1990年代ダンプトラック群の無人走行システム、全自動ビルディング建設システムなど、作業全体のシステム化・自動化を目的としたシステムが開発された。ダンプトラック群の無人走行システムは、大型石灰石鉱山におい図6・113 水中ブルドーザて同一の運搬路を往復するダンプトラックの無人化(図6・116)によって省人化を図ると同時に、濃霧時などの安全走行を実現した。全自動ビルディング建設システムは部材のユニット化、搬送と組付けの自動化、無足場工事などが特徴である。表6・20に各社の全自動ビル建設システムの特徴を示した。1994年に建設省が雲仙普賢岳の復興工事において採用した遠隔操縦建機群による施工(無人化施工)の実現によって、災害救援や図6・114 コンクリートバケットの脱着装置図6・115 コンクリート床均しロボット図6・112 ブレード接地高さ制御装置復旧など二次災害発生の恐れのある危険な現場での遠隔施工導入が一般化した。以来、北海道豊浜地区の古平トンネル、有珠山など全国規模で無人化施工が導入された。機械群の遠隔施工に際しては、現場を眺望できない場所や裸眼では見えない遠距離からの機械操縦を可能とするため、作業機械や専用のカメラ車に搭載したカメラの映像情報が採用された。現場に人がいっさい立ち入らずに施工する無人化施工を実現するために、把持装置など作業機構の開発、施工手順の工夫、新工法の開発、施工材料の研究など新しい技術展開が行われた。1995年代には、GPS
