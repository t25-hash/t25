# 6・10・3 建設ロボットの現状

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・10 建設ロボット(robot for construction work)

生産性の向上,災害発生率の低減,労働環境の向上,高齢化対策竹中工務店清水建設大成建設前田建設大林組コア部を先行構築し,これを支持体として最上階躯体の鉄骨部を利用したハットはりを構築する.このハットはり部下部に取付けられた天井走行クレーンにより,各階の鉄骨やPCa,各種ユニットを搬送・組立てる.1階分の施工終了後コア部分に支持を取りながらガイド柱でクライミングする.最初に最上階部(CF)を構築し,CF 下部に取付けられたクレーンにより,鉄骨や外壁等の部材を搬送し,天候に影響されない環境下で組立てる.1階分の工程を終了後,CF を上昇させる.最上階の骨組を地上で構築し,これにクライミング装置,クレーン等を装着し,工場部分(SCF)を構築する.このクレーン等により,鉄骨や外壁等の部材を搬送・組立し,1階分が完成した後,SCF をクライミングする.フジタ大林組鹿島建設五洋建設戸田建設地上部分に構築した組立工場部分にて,躯体取付装置等により最上階部分を構築し,これを全天候屋根架構として,完成した躯体部分をプッシュアップする.新たにできた空間でその下の階を構築することを繰返す.地上部でリフトアップフレーム架構を構築し,これにリフトアップ装置,シャトルクレーンを装着する.このリフトアップフレーム内で,鉄骨や外壁,各種ユニット等を搬送・組立てる.リフトアップ装置はジャッキ装置部分が支柱フレーム内を移動しながらクライミングする構造になっている.初めに建物の最上階の全天候型屋根システムを架構し,この全天候型屋根架構下部に自動搬送クレーンや自動施工機械などを装着する.その後屋根架構全体をせり上げ装置により1~2階分せり上げ,1層分の躯体工事が完了するごとに順次せり上げを繰返し建物を施工する.1―160産業機械・装置などを目的として建設ロボットの研究・開発が推進されてきた.しかしながら,現在までのところ,その導入効果が100%達成されているとはいい難い.作業環境の改善,部分的な施工能率の向上,作業員の負担軽減等の効果については実証されているが,施工における経済効果が得られない,稼働条件が制限される(はん用性に欠ける),全体作業として見た場合に省力化になっていないなどの問題点が指摘されている.シーケンス制御,遠隔制御から始まった建設機械のロボット化図6・117 本州四国連絡橋における維持管理ロボット(磁石内蔵式吸着車輌)図6・118 無線LAN を用いた無人化施工(システムテクニカル株式会社)も,自律型システムへの展開が始まっている.シールド,トンネル,基礎工事などの地下施工システムや建築など一部自律機能をもった工事全体の自動化システムが実用水準に到達している.数多く試みられた車両系建設機械の自動化は,上述のロボット化しやすい機械や施工システムにおいては実用に供せられているが,一般土工の分野ではことごとく実験システムに止まった.この袋小路を打開したのが,雲仙復興工事での機械群の遠隔施工である.共有の操作室に集まったオペレータがお互いに連繫しつつ工事全体にわたる最適な施工を実現した.この方式では,機械を1台ずつ自動化するのではなく,一人で複数の機械を操作し,自動化された機械にトラブルが発生した場合にのみ,他の機械のオペレータが介入操作するなど,柔軟なシステムの構築を通じた現実的省人化を達成できる.特に,高度成長期には,メカトロニクス技術の目覚ましい発展と相まって,建設機械のロボット化や施工システムの高度化への研究が量・質とも飛躍的に拡大した.景気の停滞によって開発件数は減少したが,従来の先進技術導入型の研究から遠隔(無人化)施工システムなど,利用現場が特定され,現場で実際に使用できる効率のよい研究へと転換している.
