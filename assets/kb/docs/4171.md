# 6・10・4 建設ロボットの今後

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・10 建設ロボット(robot for construction work)

今後は,ロボット導入による危険・苦渋作業の回避や,経済効果を実証するためにロボット化すべき施工分野と開発目標を明確にして取組む必要がある.ロボット化が期待される施工現場として,災害救援・復旧,高所作業,長大法面,狭隘現場,高温・多湿現場,粉じん・悪臭・有害汚染物質が存在する現場などがある.特に,焼却場や原子力施設の再開発,汚染土壌の浄化工事などのロボット化は次世代に課せられた課題である.建設投資の抑制と循環型社会への転換で維持管理ロボットへの期待も大きい.人間の腕(ブーム&アーム),手(ハンド)の形をした作業機構をもつ油圧ショベルがはん用建設機械の過半を占めるようになったいま,その手を自由に動かしてさまざまな作業を行う建設ロボットの実現も期待される.超高速無線LAN を用いた機械の遠隔制御,現場映像,機械や施工データなどを一元的に伝送することによって工事全体を一つのシステムとして高度な情報化施工を実現する試みが始まっている.文献(1) 日本建設機械化協会編,日本建設機械要覧, (2004), 日本建設機械化協会.(2) 日本建設機械化協会編,建設機械図鑑,(1994).(3) 土木学会編,新体系土木工学/100建設機械,(1996),技報堂出版.(4) 国土交通省総合政策局建設施工企画課監修, 建設機械等損料算定表,(1998),日本建設機械化協会.(5) 国土交通省総合政策局建設施工企画課監修, 建設機械施工技術の基礎知識,(2004),建設物価調査会.(6) 厚生労働省安全衛生部安全課監修, 特定自主検査マニュアル車両1―161系建設機械,(2004),建設荷役車両安全技術協会.(7) 日本規格協会編,JIS ハンドブック26 土木,(1999).(8) 竹村征三編,土工事ポケットブック,(2000),103-142,山海堂.(9) 三浦養一・ほか2名,多目的建設機械の開発,(1999),47-53,建設機械.(10) 永田隆・久武経夫,周辺分野への建設機械の適用事例,(1999),30-33,建設機械.(11) 福川光男,舗装,36(2001),13,建設図書.(12) 日本建設機械化協会編,建設機械ハンドブック,整地・締固め舗装機械編,(2001),78,125,133,146,147,149,日本建設機械化協会.(13) 日本建設機械化協会編,日本建設機械要覧,(2001),891,892,901,日本建設機械化協会.(14) 文献(13)の1077-1102ページ.→ Next Chapter
