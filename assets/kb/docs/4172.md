# 7・1・1 農業生産の要素とシステム

> **章**: 第7章 農業・水産機械
> **節**: 7・1 農作業と機械

農業生産には植物生産と動物生産がある。さらに最近では堆肥やメタン発酵に代表される微生物生産もその範ちゅうに入る。植物生産には水稲生産、飼料生産、野菜生産、果樹生産などがあり、動物生産はおもに畜産で、わが国では、牛、豚、ニワトリなどの動物が対象であり、その生産物は乳、肉、卵などである。植物や動物を生育する過程に合わせて、適切な生育環境を整えるのが農作業である。生産の規模が大きくなり農作業の質と量が人間の力の限界を超えるようになって、農作業に機械力が導入されるようになった。作物栽培における作物の生育と環境、人間、機械の関係を図7・1に示す。また、図7・2に動物生産を合理的に行うための飼養管理の概念を示す。図7・1 作物栽培作業の流れ図7・2 飼養管理の概念
