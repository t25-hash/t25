# 2・3・5 熱源装置容量

> **章**: 第2章 空気調和と空調機器
> **節**: 2・3 空調負荷

熱源装置負荷 （W）は，空調機負荷と配管（冷温水および冷
却水）の熱損失・熱取得，熱源用ポンプの動力などによる負荷で，
次式で与えられる．
＝＋＋＋＋ (2･34)
ここに， ：空調機負荷， ：配管の熱損失・熱取得による負荷，
：ポンプ入力による負荷， ：装置の立上げに要する過渡的
な負荷， ：加湿蒸気用負荷である．空調機負荷 は，各空調機
の時刻別負荷の合計が最大となる時刻の値を用いる．
熱源装置の消費エネルギーの平準化のため，夜間電力や排熱，自
然エネルギーの有効利用などを目的として，しばしば蓄熱方式
（thermal storage system）が取入れられる．図2･8に示すように
蓄熱方式における熱源機は空調運転時間帯のほか，蓄熱用として夜
間などにも運転されるので，熱源機容量は非蓄熱方式に比べて小さ
くなる．蓄熱方式の設計は，1日の熱収支に基づいて計算されるの
が通例である．この場合，熱源機の容量は次式で与えられる．
＝ d (2･35)
＝1＋ (2･36)
ここに， ：熱源負荷の1日当たり積算値（W･ h)， ：熱源
の時刻別負荷（W)， ， ：それぞれ空調の運転開始時刻と終了時
刻（ｈ)， ：蓄熱方式の熱源負荷（W)， ：蓄熱槽の運転特性
で決まる割増し係数（0.05～0.15程度）， ：熱源機の1日当たり
運転時間（ｈ）である．
