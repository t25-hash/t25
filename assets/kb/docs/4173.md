# 7・1・2 農作業と機械

> **章**: 第7章 農業・水産機械
> **節**: 7・1 農作業と機械

農作業(farm work)は生物生産作業であり、生物の生育段階で順次異なった作業が行われる。一般に作物の生産はほ(圃)場で行われ、土造りから収穫までの作業がある。それらは移動しながら作業を行うことが主で、機械にはトラクタのように動力部をもち、作業機を取換えることによって多くの作業の動力源ともなるはん用機と田植機やコンバインのように独自の原動機を備えた自走式の専用機がある。一方、温室や施設の中で行う施設生産、畜舎やサイロ等の飼料貯蔵施設の中での作業に使用する定置式の機械がある。表農業は、おもに工場の中で行う工業と異なり、作業には以下のような制約がある。このため農業用の機械にはこれらに対応できることが要求される。a. 時間的制約作物の生育段階により、作業が決まった順序で一定期間内に行わなければならない。機械は作業適期性を満足する作業能率を要求される。b. 空間的制約作物は地上に固定され、作物の配列は不斉一、不均一である。また、ほ場表面も不整である。しかし、作業精度は求められる。c. 物性上の制約対象作物は形状、硬さ、大きさなど不均一で個体差がある。これらを損傷なく取扱わなければならない。7章では以下に、動く動力源であるトラクタ、播種機・移植機、水田作収穫機としてのコンバイン、農業用施設、農用ロボット、その他畜産用、果樹用機械、水産機械などについて紹介する。
