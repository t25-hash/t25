# 2・3・6 PAL（perimeter annual load factor ，年間熱負荷係

> **章**: 第2章 空気調和と空調機器
> **節**: 2・3 空調負荷

設計段階における建築物の省エネルギー性能の評価基準として，るPALおよびCECがある．
これは，建築主が省エネルギー性能る．
PALは建築外皮の取得熱と損失熱を，外周部（ペリメータ）どの断熱性能を示す建築的省エネルギー指標である．
一方，CECの逆数に相当する指標で，空調設備（CEC/ AC），機械換気設備いる．
PAL［MJmy ］とCEC（－）はそれぞれ次式で与えらPAL＝ ペリメータゾーンの年間熱負荷（MJy）CEC＝∑ (年間エネルギー消費量）（MJyただし，上式のエネルギー消費量は一次エネルギー換算値である．
「エネルギーの使用の合理化に関する法律」（通称，省エネ法）によを容易に判断できるように，設計者が行う計算手順として示され床面積当たりの年間冷暖房負荷として計算するもので，外壁，窓なは建物の負荷当たりのエネルギー消費量をいい，システム成績係数（CEC/ V），照明設備（CEC/ L）などに分けて基準値が定められてれる．
