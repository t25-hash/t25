# 7・2・1 トラクタの特徴とその発達

> **章**: 第7章 農業・水産機械
> **節**: 7・2 トラクタ

トラクタ(tractor)とは,多様な農作業を担うさまざまな農業機械の中で,中心となる機械であり,耕うん,整地,代かき,管理,運搬などの作業に用いられる.トラクタは自動車のように単に走るだけではなく,走りながら前記のような仕事をする.また,そ表7・1 ほ場作業と代表的な作業機械ほ場作業作業対象・方法代表的機械耕うん整地未耕土プラウ,すきロータリ耕うん機整地耕起土ハロー,代かきロータリ条播すじまき機(ドリル)播種点播点播機(プランタ)散播散播機(ブロードキャスタ)粒状肥料ブロードキャスタ紛状(石灰など) ライムソーワ施肥堆肥堆肥散布機液肥パイプライン施肥スラリースラリーインジェクタ移植稲田植機畑作物畑用移植機接ぎ木畑作物接ぎ木機中耕・除草稲中耕除草機畑作物カルチベータ,ウィーダ,ロータリホー間引畑作物間引機液剤噴霧器,ミスト機,スピードスプレーヤ防除乳剤煙霧機粉剤散紛機,散粒機穀物コンバイン,バインダ収穫野菜野菜用ハーベスタ牧草飼料収穫機果実果実収穫機文献(2)を参考に作成の速度も自動車に比較すると低い.すなわち,つねに大きな負荷を受けながら,水田や畑などの軟弱な地表面を低速で走行する.このような過酷な稼動条件下で高性能を発揮するために,エンジン,動力伝達機構,動力取出し軸,走行装置,作業機装着部などに独自の技術が開発されてきた.日本では1950年頃を境に,農作業の機械化が劇的に進行した.それまでは人力と牛馬によって行われていた農作業が,まず耕うん機(二輪歩行型)によって行われるようになった.同時に北海道などの規模の大きい水田地帯に欧米の大型トラクタが導入され始め,1960年には純国産の小型トラクタが出現した.1961年,農業従事者と第二次産業従事者の間の労働・生活水準の格差を是正するために農業基本法が,さらに財政措置として農業近代化資金助成法が制定され,長期・低金利の資金導入の道が開かれ,農業機械導入の促進剤となった.このような農業機械化のための基盤の整備が進むのに合わせて,耕うん機などの歩行型機械から,さらに労働を軽減化する乗用型のトラクタが開発され,普及していった.日本経済と政治の動向によってトラクタ生産・普及には浮沈があったが,1975年頃には技術は完成したものに達し,高性能化に加えて,易操作性,安全性,居住性などが向上した.最近はエレクトロニクスを応用して種々の自動制御機構が組込まれ,不慣れなオペレータでも容易に操作できるようになっている.このようなトラクタを始めとする各種農業機械の普及によって,日本農業の労働生産性は機械化以前の約5倍に上がっている.トラクタの普及台数は現在約200万台である.図7・3に代表的なトラクタを示す.
