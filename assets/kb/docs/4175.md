# 7・2・2 トラクタ用エンジン

> **章**: 第7章 農業・水産機械
> **節**: 7・2 トラクタ

トラクタのエンジンはほとんどがディーゼルエンジンである。その性能に要求されるものは、高出力、高トルク、低騒音、低燃費、コンパクト、易始動性、クリーン排気、低速回転時のねばりなどであり、近年は特に環境への負荷低減が重要となっている。高出力化には、ターボチャージャ、インタクーラを装着し、燃焼方式を直噴化する方法がとられる。ターボチャージャからの圧縮空図7・3 農用トラクタ(四輪駆動、29PS)図7・4 インタクーラ付ターボチャージャの仕組み図7・5 エンジン一部構造図1―163気を吸引して、排気量を変えることなく、経済的にパワーアップと低燃費を実現でき、インタクーラは吸気を冷却し、空気密度をより高くして、さらに出力を向上させる(図7・4)。最近では地球環境の保全への意識の高まりを受けて、排ガスのクリーン化が大きな課題となり、ディーゼルは直噴から再び渦流室式燃焼方式へと移り変わっている。圧縮時に副燃焼室(渦流室)内に三つの強い渦流と乱流を生じさせ、理想的な混合燃焼を実現し、ピストン頂部に特殊形状の凹部を設けて渦流室内の燃焼ガスの流出をスムーズにし、主燃焼室での燃焼を効率的に行う方法である。吸気弁2・排気弁1の3弁機構をもち、ビルトイン型デュアルモードダイナミックバランサを採用して上下左右方向の振動を少なくし、ウェストゲート付ターボエンジンで低回転時に強力な過給効果を実現、高回転域では過給圧が上がり過ぎないようウェストゲートで規制して低回転域での強力な粘りを生み出す。トルク調整機構付噴射ポンプを装備したエンジンが開発されている(図7・5)。高効率化と燃料消費量の低減を狙う電子ガバナ(electronic gov-ernor)を装備した電子制御式ディーゼルエンジンがある。電子ガバナは、図7・6のような構造を有し、次のような制御を行う。すなわち、(1)負荷が変化してもエンジン回転数が変化しないアイソクロナス制御、(2)機械式ガバナではエンジン負荷が大きくなるとエンジン回転数が下がる(ドループ制御)のに対して、逆にエンジン回転数を上げてトルクアップを図る逆ドループ制御、(3)エンジン冷却水温度に応じた始動時燃料噴射量制御、(4)エア(グロー)ヒータ制御、(5)例えばトラクタの二輪駆動・四輪駆動の切換えをエンジン負荷に応じて自動制御する外部リレー制御、(6)その他、ロードモニタ機能、オーバロード警報、緊急停止機能、自己診断機能、フェイルセイフ機能などである。
