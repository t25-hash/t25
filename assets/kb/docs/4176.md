# 7・2・3 動力伝達機構

> **章**: 第7章 農業・水産機械
> **節**: 7・2 トラクタ

エンジンから走行装置や作業機への動力伝達機構(drive train)は、自動車の場合と若干異なっている。トラクタによる農作業は多様であり、走行速度はトレンチャによる溝掘りの場合のようなクリープ速度(約0.2km/h)から路上運搬時の約30km/hまで変化する。また、種々の作業はどれも負荷変動が激しいが、それに対してほぼ一定の車速を保つ必要がある。トラクタに装着された作業機を駆動するために、動力取出し軸(PTO schaft: power take-offschaft)があるが、PTO軸の回転速度も変化させねばならない。一例を挙げると、34PS の中型トラクタで前進30段、後進20段、PTO軸4段、逆転PTOが備わっている。トラクタ作業中、オペレータは頻繁に速度変化や停止発進、枕地旋回などを繰返す。これらの操作を容易にするために、トランスミ1―164図7・6 電子ガバナ構造図図7・7 電子多段トランスミッション図7・8 ニューラグタイヤッションには主クラッチを切ることなく変速が可能なパワーシフト変速装置(power shift transmission)が導入され、刻々の負荷状況に応じた迅速なきめ細かい制御を16ビットマイコンなどを利用して電子油圧制御によって可能にしている。図7・7はマイコンと油圧で変速を制御し、クラッチ操作不要で変速ができる高速多段変速の一例であり、ほ場への移動がスピーディになり、変速ショックの少ないフィーリングの良い操作ができ、クラッチを使わないので足の疲れもないなどの高能率化と易操作性を向上させている。また、油圧の静圧を利用して変速を行うHST(hydrostatictransmission)がある。これは油圧ポンプと油圧モータを組合わせて、油圧変化によってモータ出力軸回転を変化させるもので、時速0km/hから無段階に作業速度を選ぶことができる。
