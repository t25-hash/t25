# 7・2・4 走行・操舵装置とその駆動方法

> **章**: 第7章 農業・水産機械
> **節**: 7・2 トラクタ

トラクタの走行装置の主流は空気タイヤである．トラクタは水田
のような軟弱でときには粘着性のある土から反力を得て走るので，
反力を十分確保でき，かつタイヤへの土の付着を少なくするハの字
型のラグパターンや，ラグを高くしたハイラグタイヤ（high lug
tire）が用いられている．さらに，ラグ断面形状を変化させてけん
引力を増加し，タイヤへの土の付着を抑える方法も考案されている
（図7･8）．
水田における走行性をさらに向上させるために，タイヤに替えて
フルクローラを装備することもある．また，操舵性を保持するため
に，図7･9 に示すような後輪だけを揺動型のクローラにした2ク
ローラトラクタが現れている．
低速で軟弱な地表面を走行しつつ操舵する場合のステアリングの
労力を軽減するため，油圧を利用したパワーステアリングが用いら
れている．オペレータがハンドルから手を離すと，乗用車と同様に
ハンドルがほぼ直進時の状態に戻るロードリアクション型のパワー
ステアリングもある．
