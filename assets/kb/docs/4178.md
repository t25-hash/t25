# 7・2・5 作業機装着・昇降および制御システム

> **章**: 第7章 農業・水産機械
> **節**: 7・2 トラクタ

ここでは作業機としておもに直装タイプのロータリを想定する。ロータリは三点リンクヒッチ(three-point link hitch)を介してトラクタに装着され、昇降される。日本の水田作での耕うんは大半がロータリ耕であり、しかもほ場が狭いため、代かきも含めてほ場への出入りや枕地旋回のときにロータリ部があぜや畝に当たらないように、すばやく高く上げねばならない。そのため、JIS やISOなどによる規格とは別の三点リンクヒッチが考案されている。図7・11はトラクタ側上部リンク取付点と上部リンクの間に回動アーム(リンクアーム)を取付けたものである。こうすることによってリフトアームを上げるとリンクアームを介して上部リンクは本機側に引き寄せられ、ロータリの地上高を大きく稼ぐことができる。三点リンクへの作業機の脱着・交換作業は熟練を要し、時間のかかる厄介なものであったが、これをトラクタに乗ったままワンタッチで行う自動ヒッチ部が開発され、作業機の脱着のみならずPTO軸からの回転軸の接続も自動的に行うことができるようになっている。回転軸のクラッチ部には爪クラッチが採用され、これによって装着時に自動的にクラッチが入り、作業機をつり上げるだけで作業を開始することができる。作業機の制御に関しては、ポジションコントロール(position1―165図7・11 新三点リンクヒッチ図7・12 ポジションコントロール機構図control)・ドラフトコントロール(draft control)・耕深制御(till-age depth control)・水平制御・傾斜制御・その他の制御に分けることができる。油圧を利用したポジションコントロールのリンク機構作用図(図(図の左へ)シフトすると、ポジションアームの先端部が右へ動き、その結果スプールアームを介して油圧制御弁のメインスプールが右1―166産業機械・装置へ移動、油圧回路に油が流れ、油圧シリンダのピストンが動いてリフトアームが時計周りに回転する。そうするとリフトクランクカムによってロッドアームのピンが左に押され、それによってドラフトアーム(ポジションアームと重なって一点鎖線で示されている)の先端が左へ動き、これが制御弁のスプールを左へ動かし、油の流れを止める。このようにしてポジションレバーの動きに比例した安定したリフトアーム回転変位を得ることができ、作業機の位置が定まる。ドラフトコントロールはプラウなどの直装作業機のけん引負荷が増加(あるいは減少)した場合にそれを減らす(あるいは増す)ように三点リンクを介して作業機をもち上げ(あるいは自由にし)、耕深を浅く(あるいは深く)するもので、けん引負荷制御のことである。ポジション・ドラフトコントロールとも原理は変わらないが、現在はコンピュータ制御システムとなっている。ロータリの耕深制御については、耕深変化に伴うロータリ後方の均平板の動きをセンサで検知してその信号(ワイヤの動き)に基づいて自動的に油圧レバーを上下させて耕深を一定に保つ方式が用いられている。ロータリの横方向の水平制御については、水平センサを利用し、トラクタ車体の傾斜と関係なくロータリが常に水平に制御されて凹凸の多い耕地や傾斜地・あぜ際での作業でも美しく均平な仕上がり図7・14 電子メータパネルを可能にし、また、スイッチ一つでロータリに任意の角度を保持させることができ、土寄せや傾斜地での作業に便利となっている。これらの作業機の種々の制御をマイコンを利用してシステム化している(図7・13)。
