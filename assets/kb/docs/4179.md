# 7・2・6 安全性・快適性確保のための人間工学的技術

> **章**: 第7章 農業・水産機械
> **節**: 7・12 )において、ポジションレバーをA の位置からB の位置へ

回転体・高熱部などを完全に覆うことによって安全性を確保し、閉鎖構造のボンネット内に車体後方からラジエータ・エンジン・ファンを配置した搭載方式を取り、遮音効果・ラジエータ通過風量の均一化・通過音の低減を実現している。クラッチを切らないとエンジンが始動できない機構や、クラッチを踏むとPTO軸回転が停止する機構もある。騒音・振動を抑えるには、ダイナミックバランサによるエンジンそのものへの対策とともに、エンジンのゴムマウントによる振動防止策がある。究極の騒音・振動対策はトラクタキャブ(tractorcab)の採用である。最近のトラクタキャブは、快適性・易操作性も加味してその質も向上しており、例えば、フロントに曲面ガラスを採用して視界性を向上し、エアコン・ラジカセ・リアワイパ・リアウォッシャ・ドアキーロック・チルトルーフを装備、右フェンダのリモコンスイッチにより三点リンクの昇降、リフトシリンダの伸縮が操作できる便利な機能を装備している。トラクタキャブも転倒事故時にオペレータを保護する機能を有しているが、小型トラクタなどには、2柱安全フレームや4柱ロプス(roll-over protective structure)などが装備されている。最近は可倒式2柱ロプスや折り畳み式安全フレームが装備され、ハウス作業も可能で低い納屋にも格納できるようにしている。運転操作の快適性をもたらすその他の工夫として、これまでオペレータの足元を圧迫していたミッションケースや変速レバーなどの障害物をステップから取り払い、開放感あふれるデッキとし、作業時も乗り降り時も快適性が飛躍的に向上、疲労度を大幅に減少している。また、スラントノーズボンネット(前方が低くなるように傾斜したボンネット)を採用して視界性を増したり、ハンギングペダル・チルトハンドル・スライドシートなどの快適さをもたらす装置を付けている。さらに、オペレータにトラクタ・作業機の種々の状態を情報として伝えることは安全に快適に作業するために重要なことであり、トラクタの状態を液晶ディスプレイに表示する電子メータパネルを採用している。作業手順・注意事項・故障診断・対処方法など約280項目をわかりやすい日本語メッセージで表示して、初心者・不慣れなオペレータでも安心して作業ができ、誤操作の修正や故障時の的確な対応をすることを可能にしている(図7・14)。環境問題に対処する技術の一例として、1992年、モントリオー図7・13 作業機制御システムブロック図ル締約国会議において、地球環境保護のためオゾン層破壊物質である特定フロンの1995年全廃が決定されたのを受けて、特定フロンR 12を冷媒とした現行のエアコンシステムを、R 134aを使用したエアコンシステムに変更している。また、トラクタ作業時における燃料消費量の低減、すなわち省エネルギーが環境問題への一つの解決策であるとの認識から、電子制御式ディーゼルエンジンを搭載し、土中に食込むロータリ爪の形状を緩やかな曲率でねじるとともに、幅広に変更することで本数を削減、爪の先端で周囲の土をすくうように掘り起こすことで省動力化を図っている(図7・15)。
