# 7・3・1 播種機

> **章**: 第7章 農業・水産機械
> **節**: 7・3 播種機・移植機

播種機(seeder)は、ほ場に作物種子を直接まく機械で、大別して、散播機、条播機、点播機に分類される。多くは、ムギ・マメなどの畑用とダイコン、ニンジン、ホウレンソウなど野菜用で使用されるが、水稲作における利用も近年増加しつつある。播種機は、基本的に種子繰出装置(metering device)、種子導管(seed tube)、作溝器(furrow opener)、覆土器(coveringdevice)で構成され、対象作物や栽培方法に応じて組合わされる。また、耕うん・砕土作業、フィルムマルチ敷設作業、施肥作業などと組合わせた、同時工程作業機として構成されることも多い。ホッパ内の種子を所定の量だけ計量して繰出す種子繰出装置は、重要な構成要素であり、容量一定で回転速度を変更して播種量を調整する方式、容量を変更して播種量を調整する方式、さらには種子図7・20 散播機散播機(broadcaster)は種子をほ場全面にばらまく機械で、専用機ではなく、肥料などを散布する施肥用機械が利用され、麦・牧草などを対象としたトラクタ搭載式のブロードキャスタが利用される。ホッパ容量が200~1000L、対応トラクタ10~50kW で、2~4m/sの高速で作業ができる。この機械は特別な種子繰出機構をもたず、ホッパ底部にある開口部面積をシャッタで調整する単純な方式となっている。このほか、水稲湛水直播に利用される乗用管理機用散粒機などがある。b. 条播機条播機(drill)はすじ状に播種する機械で、種子繰出装置、作溝器、覆土器、鎮圧輪で構成される(図7・21)。耕うん・砕土用ロータリと播種部を組合わせたロータリシーダは、はん用的な条播機で、ロータリの作業幅に合わせて複数個の播種部が取付けられる。種子繰出装置はスパイクの付いた鉄製車輪の回転動力によって駆動され(グランドドライブ)、繰出された種子は導管を通ってダブルディスク式作溝器で形成された溝内に落下する。次いで進行方向に角度をもって設置されたディスク式覆土器で溝が埋められ、最後に鎮圧輪で鎮圧される。ロータリ幅1.5~2.4mの15~30kW 程度のトラクタ用が一般的で、麦・水稲で4~8条、大豆で2~3条である。一方、耕うん作業を行わないで直接播種する不耕起播種機では、トラクタのPTO動力を用いて強制的に駆動させるディスク式や耕図7・21 条播機の構成図7・23
