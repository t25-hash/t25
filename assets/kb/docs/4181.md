# 2・5・4 蒸気配管

> **章**: 第2章 空気調和と空調機器
> **節**: 2・5 搬送システム

a.配管方式 蒸気とその凝縮水とを同一の管で運ぶものを単
管式，別々の管で運ぶものを複管式といい，ほとんど後者の方式が
用いられている．
凝縮水を重力によってボイラへ戻すものを重力還水式（gravity
return system）といい，ポンプを用いてボイラへ戻すものを機械
還水式という．一般に後者が用いられる．
凝縮水を確実に戻すために，還水管系の終端（ボイラへ最も近い
側）に真空ポンプを設けるものを真空還水式（vacuum return sys-
tem）といい，圧力200kPa以下程度の蒸気配管系の還水法として
広く用いられている．真空ポンプの真空度は－25～－35kPa程度
である．
b.許容圧力降下と制限速度 蒸気配管の設計に当たっては，
全圧力降下をボイラの初期蒸気圧の2分の1以下，普通は3分の1
程度とする．表2･28 に一般的に用いられている初期蒸気圧と許
容圧力降下の関係を示す．圧力300kPa程度以上の蒸気管では，
管内径を mm としたとき，最大流速を0 .2 m/ s程度を標
準とするが，圧力200kPa程度以下の蒸気管では，蒸気と凝縮水
とが逆流する逆こう配管や立上り管では表2･29 に示すような制
限速度以下に抑える必要がある．
c .蒸気配管の摩擦損失線図 圧力101.325kPa，温度100℃
の飽和蒸気を基準とし，種々の圧力における蒸気の流量，管径と管
長1m当たりの摩擦損失，および管内流速を求める線図（流量線
図ともいう）を図2･33 に示す．図2･33の使用法は図中に点線
で示してある．ただし，管材は配管用炭素鋼鋼管（JIS G3452：
1962）および配管用アーク溶接炭素鋼鋼管（JIS G3457：1962，管
呼び径350A以上，肉厚7.9mm）とし，内面の絶対粗さε＝
0 .045mmとしている．
