# 7・3・2 移植機

> **章**: 第7章 農業・水産機械
> **節**: 7・3 播種機・移植機

移植機は，苗床において集中的に管理された幼植物（苗）を，一
個体から数個体に分けて田畑に植替えるために使用される機械で，
使用する苗の形態によって根のまわりの土ごと植える土付き苗用
と，土を落して植える引抜き苗（根洗い苗）用に大別される．現在
わが国で最も多く使用されている移植機は水稲の苗を水田に植付け
る田植機で，畑用としてはキャベツ，レタス等の葉菜，ビート（て
ん菜），ネギなどの移植機がある．
a.田植機（rice transplanter) 現在市販されている田植機
はすべて土付き苗を使用するもので，9割以上を占めるマット苗用
とポット苗専用に分類され，それぞれに歩行型と乗用型とがある．
図7･24，7･25に歩行型田植機と乗用型田植機の例を示す．
.歩行型田植機（walking type rice transplanter) 歩行
型田植機の植付条数は2，4，6条植えで，機関には0.75～3kWの
空冷ガソリンエンジンが用いられる．変速は前進二段，後進一段の
ものが多く，後進時は植付部は駆動されない．車輪は外径60cm，
幅6cm程度のラグ付きソリッドゴム製が多い．一輪または二輪
で，全輪が駆動される．二輪の場合の操向はおもに爪クラッチで行
う．主クラッチにはベルトクラッチが多く使用されている．
車輪の後方，ハンドルの付け根部分に植付部がある．植付部の駆
動軸は走行部の駆動と同期しており，株間が一定に保たれる．株間

で調節できる．
車重は65～180kg程度で，車輪とフロートによって，それぞれ
耕盤および土表面で支持される．フロートはポリエチレンを成形し

.乗用型田植機（riding type rice transplanter) 乗用型
田植機には4，5，6，7，8条植えがあり，機関には3～7.5kW程
度のガソリン機関，一部にディーゼル機関が用いられている．車重
は大きさによって180～900kg程度であり，おもに車輪によって耕
盤で支持される．
車輪は前輪は外径60cm，幅6cm内外，後車輪は外径90cm，
幅15cm程度のゴム車輪が多く用いられる．通常四輪で，全輪が
駆動されるが，湿田では後車輪を二重にする場合もある．またポッ
ト苗用には三輪のものもある．機体は耕盤上を走行し，植付部はフ
ロートおよび機体からのリンクを介して支持される．フロートは6
条以下は3個が一般的である．歩行型と同様にフロートが土表面か
ら受ける反力を検出し，油圧シリンダによりリンクを動かして植付
部のみを昇降させ，土表面に追従させるようになっている．また，
左右の傾きを検出して植付部のローリングを防ぐ装置も多く用いら
れる．
変速は前進二段，後進一段に副変速を組合わせたものが多く，無
段変速機を備えたものもある．操向は主としてハンドル操作により
行い，回行時などは片ブレーキを併用する．後車輪の駆動系には，
差動歯車を備えたものと爪クラッチによるものがある．条間は30
cmで調節はできないが，株間は歩行型と同様に10～20cmの範囲
で調節できる．植付深さも歩行型と同様に植付爪とフロートとの相
対位置を変えて調節する．

的である．マット苗用田植機には，クランク式および回転式の植付
機構が用いられる．
クランク式の植付機構は，平面4節リンク機構により構成され，
クランク軸の回転によって植付爪の先端が特殊な軌跡を描き，その
軌跡の上部で分苗し，最下点近くで挿苗する．歩行型と乗用型では
苗載台の傾斜方向が反対であるため，リンクの構成が異なってい
る．図7･26(ａ)は歩行型，図7･26(ｂ)は乗用型のものである．

.性能 田植機の作業能率は作業速度，作業幅，回行時
間，苗補給時間により決定される．一般にクランク式の植付機構と
しては，ほ場条件が良ければ作業速度0.7m/ s程度までは，作業
精度に支障なく作業できる．回転式の植付機構の場合はさらに数倍
の速度で植付けが可能だが，市販機では1.2～1.4m/sが最高速度
作業能率は2条植歩行型で10a/ h，4条植乗用型で25a/ h，8条
田植機の作業精度は，欠株，1株本数，苗の損傷，植付深さ，植

場合，欠株率，1株本数の変動は苗の苗立ちの均一性に大きく影響

整っていれば，欠株率は1～2%以内で植付けができる．
