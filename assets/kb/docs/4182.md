# 7・3・2 移植機

> **章**: 第7章 農業・水産機械
> **節**: 7・3 播種機・移植機

移植機は、苗床において集中的に管理された幼植物(苗)を、一個体から数個体に分けて田畑に植替えるために使用される機械で、使用する苗の形態によって根のまわりの土ごと植える土付き苗用と、土を落して植える引抜き苗(根洗い苗)用に大別される。現在わが国で最も多く使用されている移植機は水稲の苗を水田に植付ける田植機で、畑用としてはキャベツ、レタス等の葉菜、ビート(てん菜)、ネギなどの移植機がある。a. 植付部の構造分苗および挿苗は、植付爪により1株の苗を取出し、そのまま保持しながら土中へ植付ける方法が一般的である。マット苗用田植機には、クランク式および回転式の植付機構が用いられる。クランク式の植付機構は、平面4節リンク機構により構成され、クランク軸の回転によって植付爪の先端が特殊な軌跡を描き、その軌跡の上部で分苗し、最下点近くで挿苗する。歩行型と乗用型では苗載台の傾斜方向が反対であるため、リンクの構成が異なっている。図7・26(a)は歩行型、図7・26(b)は乗用型のものである。回転式の植付機構では、植付爪は回転ケースに2個装着され、ケ1―170産業機械・装置ース内の不等速遊星歯車機構の働きにより適切な軌跡を描いて、交互に分苗・挿苗を行う。基本的な構造は図7・27のようなもので、高速運転に適しており、乗用型に用いられる。これらのマット苗用植付機構は、植付爪からの苗の分離と挿苗とを確実に行うため、植付爪に沿って上下動する植込みフォークを備えている。この動きにはばねによって急速に作動するものと、カムによって徐々に作動するものがある。ポット苗専用の田植機では、植付爪の軌跡は三角形状をしており、クランク式のものはマット苗用よりも複雑な機構になっている。また、回転式の植付機構は円板の周辺に2本の植付爪を取付け、これを回転させるとともに溝カムにより植付爪の軌跡を得るもので、乗用型、歩行型ともに用いられる。実際の植付けでは進行方向の変位が加わるため、植付爪は図り変化するが、10~20cm程度の範囲では植付けに支障はない。植付爪の形状は丸鋼を加工したもの、板をプレス加工したものなどがあり、いずれも特殊鋼を用い熱処理を施している。ポット苗用はマット苗と異なり、苗を側方から保持するため1本爪である。苗載台の大きさは苗マットに合わせ幅は28cm、長さは歩行型では70cm前後、乗用型では100~120cm程度であり、55°~60°傾斜している。ポット苗用では1箱で2条同時に植付けられるので、苗載台の横幅は小さい。マット苗用田植機における苗送りは、各条ごとに仕切られた苗載台を、かき取り口のついたしゅう動板に沿って横方向に往復させる横送りと、苗載台上の苗マットをしゅう動板に向けて滑り落とす縦送りによって行われる。横送りは図7・28に示すようなナピアのスクリューを用いて行われ、植付機構の駆動と同期して所定のかき取り回数で往復動する。縦送りは苗載台の裏側にあるローラ、あるいは突起付ベルトにより横送りの終端部で間欠的に行われる。ポット苗用田植機では、苗箱の両端に設けた送り穴を使用して、箱を1列ずつ送る。苗は押出しアームによって押し抜かれ、さらに横送りベルト上に放出され、植付爪による分苗位置まで送られる。植付けた苗の1株本数は、マット苗では苗立ち密度とかき取り面積によって決まる。横送り用のスクリューと植付爪の回転比を変え(b) 性能田植機の作業能率は作業速度、作業幅、回行時間、苗補給時間により決定される。一般にクランク式の植付機構としては、ほ場条件が良ければ作業速度0.7m/s程度までは、作業精度に支障なく作業できる。回転式の植付機構の場合はさらに数倍の速度で植付けが可能だが、市販機では1.2~1.4m/sが最高速度となっている。作業能率は2条植歩行型で10a/h,4条植乗用型で25a/h,8条植乗用型で40a/h程度である。田植機の作業精度は、欠株、1株本数、苗の損傷、植付深さ、植付姿勢および株間、条間などによって評価される。マット苗方式の場合、欠株率、1株本数の変動は苗の苗立ちの均一性に大きく影響を受ける。一般に1株本数が4~5本の場合、苗条件、ほ場条件が整っていれば、欠株率は1~2%以内で植付けができる。. 特殊な田植機田植えと同時に、苗の側方所定位置の土中に施肥を行う側条施肥機付田植機、乗用田植機の植付爪の直前に溝切り円板を装着した不耕起田植機、狭幅のロータリの後部に乗用田植機の植付部を装着した部分耕田植機、再生紙を田面に敷きながらその上に移植を行う紙マルチ田植機などがある。また、乗用田植機の植付部を切り離し、各種作業機を装着できるようにした水田用栽培管理ビークル(はん用田植機)は、薬剤散布や除草など、立毛中の水田管理にも使用される。b.
