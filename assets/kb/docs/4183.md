# 7・4・1 コンバインの形態と発達

> **章**: 第7章 農業・水産機械
> **節**: 7・4 コンバイン

コンバイン（combine）とはその名のとおり，収穫機（ハーベス
タ，harvester）と脱穀機（スレッシャ，thresher）をコンバイン
ドした作物収穫機で，イギリスではコンバインハーベスタ（com-
bine harvester）と称され，トラクタと並んで最も重要な農業機械
の一つである．世界各地で作付けされるさまざまな作物の特性に合
わせて異なったコンバインが発達してきた．
わが国は1970年代に高度経済成長期に入り，農村の所得向上と
労働力不足が生じ，農業機械化の条件が整った．当初，欧米のコン
バインの導入が試みられたが，このコンバインは，わが国で栽培さ
れる脱粒難で倒伏しやすいジャポニカイネを刈り取れない，未脱粒
米が多く脱穀損失（grain loss）が多い，湿田を走行できないなど
水稲の収穫には適さなかった．このため，図7･32に示すわが国独
自の自脱コンバインが開発された．自脱の由来は，茎の一方を回転
するチェーンで挟持し，穂先のみを自動的に脱穀部に供給するわが
国独自の脱穀機を，それまでの人力脱穀機（人脱），動力脱穀機
（動脱）と区別するため自動脱穀機（自脱）と称したことによる．
図7･32に示す自脱を搭載したコンバインを欧米のコンバインと
区別するため，自脱コンバイン（head feeding combine）と称し，
図7･33に示す欧米のコンバインを普通コンバイン（conventional
combine）と称してきた．わが国でも図7･34の普通コンバインが
開発されている．しかし，1990年代に入り東南アジアでも経済発
展が進み，農業機械化が進展してきた．東南アジアでは茎が太く脱
