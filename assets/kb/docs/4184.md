# 7・4・2 刈取・搬送部

> **章**: 第7章 農業・水産機械
> **節**: 7・35 に示すコンバインが開発された.このため,自脱コンバイン,

自脱コンバインの断面を図7・37に,刈取部詳細を図7・38に示す.引き起こし装置は対地角約70度に取付けられ,ピックアップタイン(pick-up tine)によりイネをかき揚げるようにもち上げ,スターホイールで茎の下部をまとめ,株元搬送チェーンで保持して図7・37自脱コンバイン構成要素(ヤンマー農機)(a) こぎ胴(増速ツイン,三菱農機)図7・40 自脱コンバイン脱穀部1―173往復動刃で切断する.通常,固定式の受刃と動刃の組合せが用いられるが,振動低減のため両方の刃が移動するものも用いられる.また,柔らかいイネの茎の切断に適するようにのこぎり刃が使用される.刈取後は,搬送チェーンで挟んで後部に送り,フィードチェーン(feed chain)に受け渡し,茎の端部を挟持したまま穂先だけをこぎ室内に供給する.自脱はこぎ胴と穂の相対位置が重要なため,フィードチェーンが最適の位置で茎を挟持できるよう搬送チェーンの位置を自動制御して,こぎ室内での穂の位置を調整するこぎ深さ制御が行われる.また,多くの機種に刈高さ制御装置や方向制御装置が装着されている.倒伏しやすいイネを刈り取り,茎の端部を挟持しながら搬送チェーン,フィードチェーンと受け渡していく機構は世界に例のないもので,開発初期には大変な苦労があった.これに対して,普通およびインディカイネコンバインには通常,リール(reel)が使用される.現在は偏心スパイダ機構によりリールが回転してもタインはつねに垂直に保たれるピックアップリールが使われる.刈り取られた作物は,プラットフォームオーガで中央に集められ,コンベアでもち上げられ,脱穀部に供給される.投げ込み式であるため構造は簡単である.
