# 7・4・4 動力伝達、走行部

> **章**: 第7章 農業・水産機械
> **節**: 7・35 に示すコンバインが開発された。このため、自脱コンバイン、

a. 動力伝達装置作業能率の向上のため、刈取幅の増加が構造的に難しい自脱コンバインでは、収穫作業の高速化と枕地での旋回性の向上が求められる。作業速度は、現在1.5m/sのものも出現しており、エンジン出力は増大傾向にある。収穫時のほ場状態に合わせた作業速度を得るため、小型では油圧パワーシフト、大型ではHST(hydrostatic transmission)が使用される。最近では、2条刈りでもHST を搭載したものが増加している。さらに、作業能率の向上に加えて、農家はクローラにより湿田が荒らされることを嫌うため、2ポンプ-2モータ式HST、あるいは独特の歯車機構で、左右のクローラを逆転させてその場で旋回するスピン旋回、あるいは左右のクローラに任意の速度差を与えて緩旋回を可能としたものが使用されてきた。近年、左右の出力軸に遊星歯車機構を取付け、HST で左右の出力軸の回転を自在に変えるHSS(hydro-static steering)を装着して、旋回性能を飛躍的に向上させた機種が、フルタイムドライブシステム(FDS)と名づけられて普及している。HSS は、各国の戦車に1970年代から使用されていたが、前進時と後進時でハンドル操作が逆になることから、戦車以外では使用されることがなかった。今回、円すいリンク機構と名づけられた図7・43のようなリンクを設け、丸ハンドルにて円すい底面を回転させて旋回用HST の斜板角を変え、また、変速レバーで円すい底面を傾けて、変速用HST の斜板角を操作する方式が開発された。このリンク機構は、HSS の操作上の欠陥を解消した画期的なもので、今後コンバインだけでなく他用途の車両に使用されることになると注目されている。b. 走行装置イネ用のコンバインにはクローラが装着される(図7・32,7・34,7・37)。自脱コンバインでは、当初金属製の履帯が使用されたが、耐久性に難があり、現在は、すべてゴムクローラを使用している。クローラは、駆動スプロケットとかみ合う心金とスティールワイヤが合成ゴムと一体化されている。あぜ越えが容易特許公報特許第30131771―175なように中央に可動転輪が設けてある(図7・34,7・37)。接地圧は10~15kPaで、人間の足の沈下が5cm程度の湿田は楽に走行できるよう設計されている。図7・34のコンバインでは、接地幅600mm、接地長2500mmのゴムクローラが使用されている。また、コンバインは選別部を常に水平に保っておく必要があるため、左右のクローラの高さを自由に調節できるようになっているもの、水平制御機構を有するものもある。インディカイネコンバインでは、部品の入手しやすい日本製の小型ブルドーザの履帯が流用されている。
