# 7・4・5 その他

> **章**: 第7章 農業・水産機械
> **節**: 7・35 に示すコンバインが開発された。このため、自脱コンバイン、

自脱コンバインの排わら処理法としては、フィードチェーン後部に処理装置を装着し、切断してほ場に散布するものと、再利用のため、結束して放出するものなどがある。脱穀部で単粒化された穀粒はグレーンタンクに貯められ、排出オーガ(図7・32)で運搬車のコンテナに移される。小型のものでは、ホッパに貯めた後、袋詰するものもあるが、処理時間の短縮と労働負荷の軽減のため、小型でもグレーンタンク付が増加している。タンク容量は2条刈で300~500L,4条刈で850~1000Lである。普通コンバインは構造上、わらの結束はできないため、切断してほ場に還元する。また、すべてグレーンタンク付である。インディカイネコンバインでは人手の雇用が容易なこともあり、袋詰式が多い。この場合、2名の補助者が付く。安全装置についても刈取部の落下防止装置、始動安全装置、あるいは誤操作防止のための表示等、種々の配慮がなされている。1990年代初めから、農業機械のほ場内での位置計測が、GPS(global positioning system)の利用で可能となった。これにより、土壌栄養分、生育量、収量を小区画ごとに情報化して、施肥量の削減はじめ精密な管理を行う精密農業(precision agriculture)が普及してきた。このため、アメリカを中心に収量モニタを装着したコンバインが増加している。図7・33のコンバインは、収量モニタ付でキャビンの上部にGPS が取付けられている。わが国でも収量モニタの開発が進んでおり、今後、4条刈り以上では収量モニタ付コンバインが普及すると考えられる。
