# 7・5・1 穀類乾燥調製施設

> **章**: 第7章 農業・水産機械
> **節**: 7・5 農業施設

農家が生産した穀物の乾燥、調製、貯蔵に共同して利用する施設を穀類共同乾燥調製施設(共乾施設)と呼ぶ。施設利用の対象穀物はコメ、ムギ、ダイズがおもなものである。この施設の中で乾燥、調製まで行う施設は一般にライスセンタ(RCと略す)と呼ばれる。また、乾燥、調製に加えて貯蔵まで行う大規模施設はカントリーエレベータ(CE: country elevator)と呼ばれる。図7・44にRCの工程と使用されている機械装置を示した。CEでは乾燥工程の後にもみの精選工程と貯蔵の工程が加わる。a. 粗選機(paddy cleaner)荷受した高水分のもみ(生もみ)に混入しているわらくず(藁屑)や穂切れ、ごみなどの挟雑物を除去する。図7・45はファンを装備した風選別部と金網が左右に振動する振動ふるい部からなる粗選機である。粗選機に入った生もみはまずファンにより風選され軽いものはここで除かれる。次いで振動ふるいで選別され金網を通過しない挟雑物が排出され除かれる。b. 再脱穀機粗選機から排出された穂切れ、枝梗付着粒、わらくずなどをこぎ(扱ぎ)歯により処理し穂切れ、枝梗付着粒は再脱穀され単粒とされる。c. 自主検定装置自主検定は荷受した米麦の未熟粒、整粒の歩合測定と品質の評価により生産者のもち分を決定するために行われる。オートサンプラで採られたサンプルはテストドライヤで乾燥された後、自動自主検定装置においてもみすり(籾摺)と未熟粒、整粒の選別を行い重量測定、データの印字記録を行う。最近では、図7・43 FDS 操作円すいリンク機構(ヤンマー農機)1―176産業機械・装置図7・44 ライスセンタの工程図7・45 粗選機品質評価に外観品質を判別する米質計と食味を判定する食味計が使われている。d. 乾燥機穀類乾燥機(grain dryer)の種類として穀物を乾燥機内で移動しながら乾燥するものと移動しない方式のものに分けることができる。共乾施設で使用されている乾燥機は穀物移動方式のもので、循環式と連続送り式のものがある。しかし、近年ほとんどのCEでは操作性の良さ、夜間作業の低減、荷受けのための容器が不要などの理由により循環式、累積かくはん式(ビン方式)が導入されている。移動方式の乾燥機は穀物流下槽およびこれに付随する給気室、排気室などを含む乾燥塔、送風機、火炉、風炉および穀物投入、排出のための搬送機よりなる。普通もみは水分15%まで乾燥される。もみ水分は乾燥中自動的に測定され、乾燥機の操作もコンピュータにより自動化されている。. 循環式(図7・46)バッチ式の乾燥機で乾燥機上部のテンパリング(乾燥を休止し穀物内部の水分分布を平均化させること)部に続いて通風乾燥部があり、穀粒が自動的に常時または間欠的に乾燥機内を循環し乾燥とテンパリングを繰返しながら最終水分まで乾燥される。乾燥機の穀物張込量は個人農家用で600~6000kg、共乾施設用では45~90t規模の乾燥機が使われている。もみ乾燥における乾燥速度(乾減率)0.8~1.0%/hである。. 連続送り式CEなどの大規模乾燥施設で使われている乾燥機で、穀物は乾燥機内を1回通過するたびにサイロまたはタンクに必要時間一時貯留されテンパリングを行う。穀類は乾燥、テンパリングを繰返し仕上げ水分まで乾燥される。CEでの標準的な乾燥操作は、乾燥機通過時間約30分、乾燥熱風温度45°C、風量比2.5m/s・t、乾燥機1回通過当たりの水分低下はもみ水分17%までは2.2%、それ以降の仕上げ水分までは1.5%となっている。. 静置式(ビン方式)この施設は穀類をビンの中で乾燥図7・46 循環式乾燥機図7・47 もみすりプラントの工程し、乾燥終了後そのままビンに貯蔵するもので乾燥機と貯蔵倉庫を兼用している。ビンには丸ビンと角ビンがある。CE,RCでは、容量50~400tのビンを複数設置している。ビン方式では生もみをすでに乾燥が進行しているもみと混合かくはんし乾燥する。角ビンでは他のビンのもみと混合し、もみを移動(ローテーション)しながら乾燥する方式もある。いずれのビンにおいても乾燥中は、もみは静止している。ビン間のローテーションを自動で行うシステムや、かくはん機能によりローテーションの頻度を低減する装置が導入されている。e. もみすり調製プラント(図7・47)もみからもみ殻を脱離させて玄米を得るとともに、玄米を良玄米、未熟粒、くず米に選別し、さらに混入している石などを除去する装置である。もみすり機、もみ選別機、粒選別機、石抜機から構成されている。. もみすり機(husker、図7・48)もみは回転速度の異なる二つのゴムロールの間げきを通過する際せん断力を受けてもみ殻図7・48 もみすり機が分離する。高速ロールの回転数は1000~1200rpm、低速ロールは700~900rpmである。もみすり機(husker)のサイズはロール幅で表す。共乾施設で使われているもみすり機のロールの幅は150~250mm(6~10in.)である。もみすり機の処理能力はほぼロール幅に比例すると考えてよい。ロール通過後もみは大部分もみ殻が除かれ玄米となるが一部はもみのまま通過する。もみ殻は風選され除かれ、玄米ともみはもみ選別機に送られる。. もみ選別機傾斜した凸状突起を有する選別版を穀類の流下方向とほぼ直角方向に揺動させて玄米ともみの表面の粗滑の差を利用して選別を行う。選別版を20枚程度重ねて選別面積を広くするとともに省スペース化を図っている。もみ選別機により玄米、玄米+もみ、もみの部分に区分され、玄米は粒選別機に送られる。もみはもみすり機に戻され、玄米ともみの混合部分は再度もみ選別機にかけられる。. 粒選別機回転している打抜鋼板の円筒または角筒のふるいの中へ玄米を供給し選別する。ふるいは玄米を薄層にして選別能力を向上される目的から複数多段に設置されている。f. もみ精選機CEではもみで取引されることからもみ精選機が使われている。精選機にはシリンダ内面に半球状の凹みをもった回転円筒(インデンテッドシリンダ)や揺動選別機が使われている。脱ぷ()粒、枝梗付着粒が除かれる。
