# 2・1・2 自動車の種類（kind of automobile）

> **章**: 第2章 自動車
> **節**: 2・1 自動車一般

自動車には，使用される場所や車両の構造，原動機の種類，動力
を発生するためのエネルギー源の種類などによっていろいろな分類
法がある．
a.原動機とエネルギー源による分類
.内燃機関自動車（internal combustion engine vehicle)
内燃機関は，単位出力当たりの重量，容積が小さく，高性能でかつ
耐久性，生産性，経済性，サービス性に優れているため，原動機の
ほとんどを占めている．エネルギー源によって，ガソリン自動車，
ディーゼル自動車，LPガス自動車，ガスタービン自動車，アルコ
ールエンジン自動車等がある．
.電気自動車（electric vehicle) 蓄電池（バッテリ）を
搭載し，電気エネルギーによって，電動機（モータ）を回転させて
車輪を駆動する．排出ガス零の自動車であるが，蓄電池の性能によ
り，走行距離が限定されるので，用途を限って使用されている．
.ハイブリッド自動車（hybrid vehicle) 2種類の原動機
をもった自動車で，走行領域に応じて原動機を使い分けることによ
り，低排出ガス，燃費向上の両立を目的としている．ガソリンエン
ジンと電動機の組合わせ，ディーゼルエンジンと原動機の組合せに
よるものなどがある．
.燃料電池自動車（fuel cell vehicle) 一般に，水素と酸
素を反応させて直接電気を取出し，電動機を回転させて車輪を駆動
する燃料電池自動車は，水だけを排出し，排出ガスやCOを出さ
ない究極の環境対応自動車として，将来内燃機関に代わるものとし
て期待されている．燃料は水素のほか，石炭，石油，天然ガスなど
数あるエネルギー資源の中から検討されているが，水素を燃料とす
る燃料電池自動車が，将来有望視されている．
b.構造による分類
.原動機（エンジン）の位置による分類 エンジンの搭載
位置により分類するもので，フロントエンジン自動車，リヤエンジ
ン自動車，ミッドシップエンジン自動車，アンダシートエンジン自
動車等がある．
.駆動方式による分類 駆動方式により分類するもので，
次のようなものがある．
･前輪駆動車：前車輪に動力を伝えて駆動する自動車で，エンジ
ンを車両前部に搭載している．広い室内空間を確保しやすい構
造となっている．
･後輪駆動車：後車輪に動力を伝えて駆動する自動車で，エンジ
ンを車両前部または車両後部に搭載している．高性能なエンジ
ン出力を発揮しやすい構造となっている．
･総輪駆動車：前後すべての車輪に動力を伝えて駆動することの
できる自動車で，四輪の場合，四輪駆動車，4WDと呼んでい
る．あらゆる路面状態でも走行可能な構造となっている．
.フレームの有無による分類 フレームの有無により分類
するもので，フレーム構造自動車，フレームレス（モノコック）自
動車がある．
c .車体の形態による分類
.乗用車の分類 乗用車は形状により，セダン，リムジ
ン，クーペ，コンバーチブル，ステーションワゴン等に分類され
る．
.バスの分類 バスは運転席の位置や乗車定員などによる
形状の違いにより区分され，ボンネットバス，キャブオーババス，
箱型バス，ライトバス，マイクロバス等に分類される．
.トラックの分類 トラックは運転台の位置，積載物を保
護する屋根の有無による形状の違いにより区分され，ボンネットト
ラック，キャブオーバトラック，パネルバン，ライトバン等に分類
される．
またトレーラトラックとしては，セミトレーラトラック，フルト
レーラトラック，ダブルトレーラトラック等がある．
