# 7・5・3 青果物予冷施設

> **章**: 第7章 農業・水産機械
> **節**: 7・5 農業施設

収穫後の青果物の品質を保持するためには,できるだけ速やかに青果物の温度を下げ,生理作用に伴う品質劣化を抑制する必要がある.青果物を出荷あるいは貯蔵する前に短時間に青果物を冷却することを予冷(precooling)と呼んでいる.予冷方式として強制通風式,差圧通風式,真空式が広く使われている.最近は立体自動冷蔵を併設した大型の予冷施設が建設されている.a. 強制通風方式貯蔵庫内で冷風を庫内に積上げられた青果物の周囲に循環して冷却する方式である.一般の青果物低温貯蔵庫に比べて循環送風量は大きい.b. 差圧通風方式専用のファンにより青果物の入っている段ボール箱の両側面に空気圧差を与え,冷気が段ボール側面の孔を通して箱内を通り抜けるようにして,冷却速度を高める方式である.この方式には中央吸込方式(図7・50)と壁面吸込方式がある.c. 真空式圧力約611Paのもとで水は0°Cで蒸発することを利用した冷却方式である.青果物内部の水分が蒸発しその蒸発潜熱で青果物が冷却される.真空冷却(vacuum cooling)では葉菜類のように体積に比べて表面積の大きな青果物の冷却に有効である.この方式は真空ポンプ,真空チャンバ,コールドトラップ,冷凍機などから構成されている.
