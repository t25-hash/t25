# 7・5・3 青果物予冷施設

> **章**: 第7章 農業・水産機械
> **節**: 7・5 農業施設

収穫後の青果物の品質を保持するためには，できるだけ速やかに
青果物の温度を下げ，生理作用に伴う品質劣化を抑制する必要があ
る．青果物を出荷あるいは貯蔵する前に短時間に青果物を冷却する
ことを予冷（precooling）と呼んでいる．予冷方式として強制通風
式，差圧通風式，真空式が広く使われている．最近は立体自動冷蔵
を併設した大型の予冷施設が建設されている．
a.強制通風方式 貯蔵庫内で冷風を庫内に積上げられた青果
物の周囲に循環して冷却する方式である．一般の青果物低温貯蔵庫
に比べて循環送風量は大きい．
b.差圧通風方式 専用のファンにより青果物の入っている段
ボール箱の両側面に空気圧差を与え，冷気が段ボール側面の孔を通
して箱内を通り抜けるようにして，冷却速度を高める方式である．
この方式には中央吸込方式（図7･50）と壁面吸込方式がある．
c .真空式 圧力約611Paのもとで水は0℃で蒸発することを
利用した冷却方式である．青果物内部の水分が蒸発しその蒸発潜熱
で青果物が冷却される．真空冷却（vacuum cooling）では葉菜類
のように体積に比べて表面積の大きな青果物の冷却に有効である．
この方式は真空ポンプ，真空チャンバ，コールドトラップ，冷凍機
などから構成されている．
7・5・4貯蔵施設
乾燥されたもみは乾燥施設のサイロや乾燥貯蔵ビンに貯蔵され
る．玄米は普通30kg詰紙袋で米保管倉庫に貯蔵（保管）される．
これらの倉庫では保管中温度が15℃を超えないように管理されて
いる．
青果物の貯蔵は凍結しないよう一般には安全を考慮して0℃より
少し高い温度領域で行われる．貯蔵施設には冷凍機による低温貯蔵
庫や天然の冷気（低温）を利用した施設などが利用されている．ま
た，CA貯蔵（controlled atmosphere storage）あるいはMA貯
蔵（modified atmosphere storage）と呼ばれる貯蔵法がリンゴの
貯蔵で広く行われている．この貯蔵では低温貯蔵庫内の酸素および
炭酸ガスの組成を制御して貯蔵期間の延長を図っている．リンゴの
貯蔵では酸素および炭酸ガス濃度はともに1.5～2.0 %の範囲で制
御される．CA貯蔵に使われる貯蔵庫は高い気密性が必要である．
そのため外気の気圧変動により貯蔵庫は外力を受ける．これを避け
るため貯蔵庫の天井裏などにブリーザバック（breather bag）と呼
ばれる貯蔵庫の1～2 %の容積に当たる大きな袋を備えている．酸
素濃度を下げるためPSA（pressure swing adsorption）方式によ
る窒素ガス発生装置が使われている．以前プロパンガスなどを燃焼
させ炭酸ガスを使って酸素のプルダウン（pull -down）に広く使わ
れたガス発生器は現在では使われなくなった．最近ではガス交換膜
の実用化が図られている．CA貯蔵庫では過剰な炭酸ガスを吸収す
るための炭酸ガススクラバ（COscrubber）が設置されている．
スクラバに消石灰や活性炭が使われる．
7・5・5育苗施設
共同で多量かつ省力的な作物の育苗を行うのが育苗施設（nurs-
ery house）である．育苗は種子の前処理，播種，出芽，緑化，硬
化などの作業工程からなっている．前処理では種子の選別，消毒，
浸種，催芽が行われる．
a.催芽機 種子の発芽をよくし，そろえるために30℃程度に
加温して鳩胸状に発芽させる．温水式と蒸気式がある．
b.出芽装置 育苗箱を台車などに積み重ねたものを保温カバ
ーで覆い，適切な温度，湿度に保って均一な発芽を促す．
c .播種プラント（図7･51) 育種箱への播種作業を行う．育
苗箱をコンベヤ上で移動させ床土詰め，播種，覆土，潅水を能率的
に行う．作業は自動化されている．
d.その他 床土乾燥機，砕土器，肥料混合機，脱ぼう（芒）
機，育苗箱消毒洗浄機などが使われている．
7・6自動化・ロボット化
7・6・1自動化の現状
a.自動化のニーズ 農業機械は，大自然のもとで変化に富ん
だ土壌や作物を対象に，広大なほ場で営まれるという農業の特質に
対応して，走破性に優れた移動機能，変化への柔軟な適応性，1人
でより広い面積を処理する作業能率などを併せもつように進化して
きた．反面作業者は，場所ごとに作物や土壌の条件に合わせて作業
機の調整を行いながら，なるべく早い速度で作物列などの目標経路
（target path）に合わせて機械を操縦することとなった．このため
作業者は，ともすれば超人的な負担を課せられることになり，その
解決のため，作物列に沿った運転操作や作業機の調整などが自動化
されるようになった．また，シーズン中は昼夜連続して運転される
穀物乾燥機のような屋内で使用される機械も，プラント設備などと
同様に作業者による監視や操作を必要としないように自動化が進め
られてきた．以下に，代表的な農業機械における自動化の例を示
す．
