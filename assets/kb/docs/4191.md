# 7・5・4 貯蔵施設

> **章**: 第7章 農業・水産機械
> **節**: 7・5 農業施設

乾燥されたもみは乾燥施設のサイロや乾燥貯蔵ビンに貯蔵される.玄米は普通30kg 詰紙袋で米保管倉庫に貯蔵(保管)される.これらの倉庫では保管中温度が15°Cを超えないように管理されている.青果物の貯蔵は凍結しないよう一般には安全を考慮して0°Cより少し高い温度領域で行われる.貯蔵施設には冷凍機による低温貯蔵庫や天然の冷気(低温)を利用した施設などが利用されている.また,CA 貯蔵(controlled atmosphere storage)あるいはMA 貯蔵(modified atmosphere storage)と呼ばれる貯蔵法がリンゴの貯蔵で広く行われている.この貯蔵では低温貯蔵庫内の酸素および炭酸ガスの組成を制御して貯蔵期間の延長を図っている.リンゴの貯蔵では酸素および炭酸ガス濃度はともに1.5~2.0%の範囲で制御される.CA 貯蔵に使われる貯蔵庫は高い気密性が必要である.そのため外気の気圧変動により貯蔵庫は外力を受ける.これを避けるため貯蔵庫の天井裏などにブリーザバック(breather bag)と呼ばれる貯蔵庫の1~2%の容積に当たる大きな袋を備えている.酸素濃度を下げるためPSA(pressure swing adsorption)方式によ図7・51 播種プラントる窒素ガス発生装置が使われている.以前プロパンガスなどを燃焼させ炭酸ガスを使って酸素のプルダウン(pull-down)に広く使われたガス発生器は現在では使われなくなった.最近ではガス交換膜の実用化が図られている.CA 貯蔵庫では過剰な炭酸ガスを吸収するための炭酸ガススクラバ(COscrubber)が設置されている.スクラバに消石灰や活性炭が使われる.
