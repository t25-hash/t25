# 2・1・6 自動車の情報，ITS，LAN

> **章**: 第2章 自動車
> **節**: 2・1 自動車一般

a.自動車の知能化 自動車交通は国民の生活に不可欠で，そ
の生活向上に大きく貢献しているが，2･1･5項までに述べた環境・
エネルギー，安全に加えて，大きな時間損失を発生させている渋滞
や，今後の高齢社会における安心で快適な道路交通の実現という大
きな課題がある．これらの課題に対して，自動車の機構・構造など
自動車本体の改良が図られてきたが，まだ十分な解決を見るに至っ
ていない．これらを解決していく方向として自動車交通の知能化が
期待されている．従来，自動車の運転に関わる知的な機能はほとん
どが運転者に依存していたため，交通事故など運転者のミスに起因
する問題が多く発生している．今後は自動車交通システムが運転者
の知的機能を支援・代行して高度化させることが必要である．自動
車の知能化は図2･12に示す三つの機能の支援である．
.運転者機能支援 運転者の機能には思考的機能と反射的
機能がある．思考的機能とは，大局的な運転戦略の立案である．道
路交通情報などを入手して分析し，行動目的に合わせた具体的走行
計画を立案する．これには出発・到着時刻，経路，休憩，給油など
の計画を含み，出発前に立案されるが，走行中にも状況変化に応じ
た計画変更が行われる．反射的機能とは，走行中に行われる運転操
作に関する機能である．認知，判断，操作機能であるが，より具体
的には環境情報の検出と認識，状況判断と運転操作計画の立案，運
転操作目標の設定とそれに基づく操作である．これらの運転者機能
を自動車や道路インフラストラクチャ（infrastructure）が支援・
代行して適切な運転が行われるようにする必要がある．
.同乗者機能支援 自動車の利用者は運転者だけでなく，
同乗者がいる．同乗者が必要とする機能は運転者のそれとは異な
り，業務や趣味などに関連する機能であり，職場や家庭における
IT社会での知的活動を移動中にも行えるようにしたいという期待
がある．
.社会システム機能支援 渋滞や環境エネルギー課題の解
決には，自動車個々の知能化だけでなく関係する車両群全体として
の知能化が必要である．自動車は運転者の自由意志によって運転さ
れているが，社会全体として見ると必ずしも適切な行動が行われて
いるわけではない．有限の交通環境を全体として最適に活用するた
めの社会システムとしての機能支援が必要である．
b.自動車の知能化に必要な情報 自動車交通の知能化の鍵は
情報である．必要な情報を入手して適切に処理することにより，知
的な行動をとることができる．
.運転者機能支援に必要な情報 思考的機能の支援には走
行計画立案に用いるさまざまな情報が必要である．出発地から目的
地までの間の地図や経路情報，道路交通環境情報，渋滞や事故など
の交通障害情報などが基本的なものである．行動目的に応じて関連
するイベント情報や関係者の行動予定情報など，多くの情報も必要
である．反射的機能の支援に必要な情報は，走行進路上の環境情報
と自車状態情報である．走路における自車位置（縦方向と横方向の
位置）情報，進路前方の道路形状や路面状態などの走路状態情報，
他車両や歩行者など道路を共同利用する他者の状態情報と相対位置
情報などが基本的情報である．反応時間や減速・停止などの運転操
作に必要な時間の範囲の情報が必要である．安全で安心な運転を行
うためには余裕をもった行動が可能な情報が必要で，取得する範囲
を拡大する必要がある．
.同乗者機能支援に必要な情報 同乗者が必要な情報は多
様である．従来は入手方法が限定されていたため，自動車は情報に
関する孤島状態であった．通信手段が整備されて入手できる情報が
増えると，より広範囲の情報が欲しくなる．理想は職場や家庭で得
られるすべての情報を移動中にも活用できるようにすることであ
る．IT社会の進展につれて必要な情報は飛躍的に増大すると考え
られる．

.社会システム機能支援に必要な情報 道路交通を社会と
して適切にするためには，社会目的に沿って自動車が統一された秩
序立った行動をとることが理想であるが，運転者が自らの意志で動
かす現在の自動車においては非現実的である．現実的には道路交通
に関する状況情報を提供して，運転者が適切な行動を判断・選択で
きるようにすることである．交通流の速度，密度，旅行時間などさ
まざまな情報が必要であるが，個々の車両の位置，速度，走行方向
（目的地），大きさなどが基本的情報であり，これらを情報処理する
ことによって必要な情報が得られる．
c.ITS（intelligent transportation systems ) 道路以外の交
通も対象とする交通の知能化のことであるが，日本ではITSを高
度道路交通システムと呼んでいる．
.ITSの特徴 ITSは道路交通の抱えるさまざまな課題
を解決するために，b項で述べたような自動車の知能化に必要な情
報の収集と情報処理，情報の提供，情報を用いた判断や操作の支援
などを行うシステムである．従来の自動車のシステムが自動車の内
部の情報を基に動作していたのに対して，ITSの大きな特徴は，
自動車の外の情報（環境情報）を用いて支援を行うことである．
.支援の内容 ITSは，a項で述べた運転者機能支援，同
乗者機能支援，社会システム機能支援を含む幅広いものである．運
転者機能支援の代表がナビゲーションシステム（navigation sys-
tem）である．自車位置の検出機能，地図などの走路情報機能など
知能化に必要な基本的機能を有するとともに，通信機能，情報処理
機能，運転者とのヒューマンマシンインタフェース機能などを備え
ており，ITS応用システムの核になるものである．従来のナビゲ
ーションシステムは経路誘導など主として運転者の思考的機能の支
援を行ってきた．今後のナビゲーションシステムは支援機能の発展
が期待されている．
縦方向挙動の支援機能である車間保持（アダプティブクルーズコ
ントロールシステム，図2･13）や，横方向挙動の支援機能である
車線保持などの反射的機能の支援も実現されてきている．今後は交
差点や合流部などにおける交差挙動の支援機能の実現が期待され
る．
