# 7・6・2 ロボット化

> **章**: 第7章 農業・水産機械
> **節**: 7・6 自動化・ロボット化

a. ロボット化のニーズ日本農業は、この50年ほどの間に一斉作業可能なコメ、ムギなどの作物を対象に機械化が進み、水稲10a当たりの平均労働時間は、1945年の約250時間から1997年には37時間にまで減少した。しかし、規模拡大や国際的価格競争力強化などのニーズに応えるためには、飛躍的な省力化と軽労化が求められている。一方、野菜・果樹等では、一斉作業よりも収穫作業など熟度や大きさ・形状などを判別しながら行う選択的な作業を要するケースが多く、旧来の機械技術では機械化できなかった作業が多い。このような背景から、メカトロニクス(mechatronics)技術を駆使した人の認識・判断にかかわる部分を含む機械化、すなわちロボット技術を導入した機械化が指向されている。ロボット化の目的としては、(i)車両の運転操作の自動化・無人化、(ii)人の動きに近い柔軟な動作による未機械化作業の機械化、(iii)労働環境改善と作業の安全性・快適性向上、などが挙げられる。b. ロボット化の要素と条件農業用ロボットの構成要素は、一般のロボットと同様であるが、産業用ロボット(industrialrobot)の成果をそのまま農業に移転することは、以下のように(表7・3)農業固有の条件により容易ではない。(1) 作物は、成長速度を人為的に制御することが困難であり、生産量を思いどおりに制御できない。このため、農産物の供給は需要に対し機敏に対応できず、価格に安定性を欠く要因となっており、農業への計画的な投資は行われにくい。(2) 作物および農産物は生命体であって、色、形状、大きさ等が不ぞろいで、工業部品のような規格化は考え難い。また作物は、一般に田畑に根ざしているので、ロボットには移動機構が欠かせない。(3) 農業は、作物の成長過程に応じて数日ずつ異なる作業が必要であり、作業ごとに使用される機器の稼働日数は少ないので、新たな投資を要するロボットには稼働率向上のためはん用性が要求される。(4) 作業環境は、屋内と比べ劣悪であり、かつ、商用電源や照明などの付帯設備はないのが普通である。路面は、ぬかるみや急傾斜地も多く、安定した路面はほとんど見られない。(5) 多くが個人経営であるため、個々の作業に特化した熟練者は容易に育成できず、ロボットは取扱容易なものでなければならない。c. 農業用ロボットの例前項のような条件を考慮したうえで、以下のようなロボットが開発されている。. ミニトマト収穫ロボット本ロボットは、おもにハウス内で栽培されるミニトマトの自動収穫を目指したものである。三次元視覚センサは、エンドエフェクタ(end effector)に備えられ(図7・58)、波長1(685nm)と2(830nm)のレーザビームを上下左右にスキャンし、その反射光をPSD(position sensitivedetector)素子で受光して三次元画像を得る。波長2によって測距を行い、波長1と2の反射光量の比から赤く熟した果実の識別を行う。また、エンドエフェクタの左右振り機構によって、正面からだけでなく側方からも果実の認識が可能で、房状に重なり合ったミニトマトを確実に認識する。エンドエフェクタには、果実を吸引するゴムパッドが備えられ、果実吸引後押さえ板で果実を固定後マニピュレータ(manipulator)の運動により果実をもぎ取るものである。マニピュレータは、3自由度の円筒座標型に上下旋回を加えた4自由度のものであり、サーボモータ(servo motor)によって駆動される。マニピュレータは、走行用レール上を移動する電動台車に搭載される(図7・59)。本ロボットにより収穫適期にある果実は、測距誤差5mm以下で茎葉に隠れた果実を含めおよそ90%が認識可能であり、一つの果実を収穫するのに要する時間は4~6秒であったという。このほか、野菜、果樹の収穫ロボットとしてオレンジ収穫ロボット(アメリカ、フロリダ大学)、柑橘収穫ロボット[(株)クボタ]、キャベツ収穫ロボット(農業研究センター)、ブドウ収穫ロボット(岡山大学)など多数開発されたが、いずれも慣行の人による作業能率を超えることが今後の課題になっている。.
