# 7・7・1 果樹用機械

> **章**: 第7章 農業・水産機械
> **節**: 7・7 その他の農業機械

a. 管理作業機械. せん定(pruning)・整枝(training)用機械樹形を整え,収穫を増すためにせん定作業が行われる.この作業には空気圧を利用したせん定ばさみが利用されている.せん定枝など残稈を処理するのに粉砕機,細断機,チョッパなどがある.. 土壌管理用機械管理作業として,除草,中耕,施肥などがある.除草用には刈払機(bush cutter)・草刈機(grassmower)が用いられる.中耕,施肥などの作業には果樹独特の機械がある.例えば,樹の周囲の草が刈れるように幹を自動的に避けて作業する幹周草刈機,圧縮空気を根元に瞬間的に吹込んで中耕を行う機械などである.. 防除用機械薬剤散布は作業回数が多いこと,作業が暑い時期に集中すること,作業者が危険な農薬を扱うことなどから機械化の要望の高い作業である.果樹園に散布する農薬はほとんど液剤であるので,動力噴霧機(power sprayer)が多く用いられる.これは加圧しながらノズルの先端より噴霧状態で散布するものである.散布方法によりスプリンクラやスピードスプレーヤがある.b. 収穫用機械果樹の収穫には,外国では振動収穫機が用いられている.トラクタに装着して油圧を利用して幹を加振するものと,枝を加振するものがある.振り落とした果実はキャンバス製のキャッチフレームで受けて,傷つくのをできる限り防いでいるが,もっぱら加工用の果実の収穫に使われる.生食用果実の収穫は人力によっており,補助的に高所作業車など移動式収穫作業台などが使われる.
