# 7・7・2 畜産用機械

> **章**: 第7章 農業・水産機械
> **節**: 7・7 その他の農業機械

a. 飼料生産用機械. 粗飼料収穫機械牧草収穫体系を図7・62に示す.作業を枠で囲み,下に利用する機械名を記している.草は刈取ってそのまま家畜に与える場合と,含水量の高いままでサイロに入れて発酵貯蔵する場合(サイレージ),および乾燥して貯蔵する場合(乾草)1―183がある.それぞれ作業体系と機械システムで異なる.サイレージは,通常モーアまたはフォレージハーベスタ(forage harvester)で刈取って,必要に応じて細断の後,サイロに積込む.乾草を作るにはモーアで刈りながらコンデイショナのロールの間を通して茎を砕き,葉だけが早く乾燥して落ちてしまわないようにする.さらに,テッダ(tedder)で草を時々かきまぜながら天日乾燥し,レーキで,一列に集める.これを通常ベーラのピックアップ装置で拾いあげながら乾草の塊り(ベール)を作る.ベールはピストンで圧縮して角型にするものとロールベーラのように,じゅうたんを巻くようにしてロール状にする場合とがある.最近はヘイベーラ(haybaler)で梱包し,屋外でビニールシートをかけて貯蔵する方法や,ベールラッパでプラスチックフィルムを気密に巻付け,サイレージにする方法が行われている.. 飼料調製加工機械飼料用の穀実,茎葉,根葉などを調製・加工し,飼料の嗜好性や消化率を向上したり,飼料の取扱い,貯蔵,配合,給じ(餌)などを容易にするための機械で,飼料細断機,飼料粉砕機,緑飼細断機(チョッパ),根菜細断機(ルートカッタ),飼料配合機(フードミキサ),飼料成形機などがある.b. 家畜管理用機械. 給じ(餌)機械(feeder)飼料を必要に応じて細断,粉砕し,ある比率で混合する機械である.大規模経営では,配合調製された飼料を機械で給じする.乳牛の場合には,個体を識別して,必要量をコンピュータで計算して給じする自動システムもある.. 搾乳機械(milker)乳牛の搾乳には通常パイプラインミルカが用いられる.ミルカは真空ポンプ,ティートカップ,ミルククロー,パルセータ,送乳パイプで構成されている.真空で乳を吸引するとともに,乳頭に接したゴムのライナを動かし,マッサージをする.搾乳が終わると,自動的にミルカが乳房から離れるようにできるが,装着は現在のところ人手によっている.搾乳した乳は送乳パイプで集乳缶または牛乳冷却機に直送される.一貫自動化のできる搾乳ロボットも市販されている.. ふん尿処理装置家畜のふん尿は集めて乾燥するか,発酵させて肥料にする.牛舎に取付けた収集用のバーンクリナ(barn cleaner)の,エンドレスチェンに一定間隔で取付けられたスタレーバでふん尿溝をかいて集める.ふん尿は固液分離して,固形分はたい肥化または焼却し,液分は発酵によって殺菌した後,尿図7・62 牧草収穫作業体系1―184産業機械・装置散布機,スラリインジェクタ,スラリガンなどを用いてほ場に還元される.
