# 2・2・5 制動性能

> **章**: 第2章 自動車
> **節**: 10・15 モードの走行サイクルを示す．

近年の日常生活での自動車の利用が一般化する中，自動車の安全
ブレーキ性能としては，効き，停止距離（stopping distance），
ブレーキの効きは入力に対する出力の関係で示されるが，運転者
作力は避けるべきである．この点を考慮したブレーキアシストシス
テム（BA:brake assist system）の採用は，アンチロックブレー
キシステム（ABS:anti -lock brake system）の普及と相まって，
通常使用時には軽すぎず，かつ緊急時には操作力の増加を抑えて必
要なブレーキ力を得られることを可能にした．
ブレーキ力は路面で吸収されるため，タイヤと路面の特性も効き
において重要な役割を占める．タイヤと路面の特性限界を超えてブ
レーキ力を発生することはできない．
効きの安定性も運転者にとって重要である．降坂時にブレーキの
使用頻度が高まるときや高速からの制動時に，ブレーキの摩擦部分
の温度が上がり摩擦係数が変化する．この現象をブレーキフェード
（brake fade）といい，熱による効きの変化を意味し，操作力に対
しブレーキ力発生が低下する効きの悪化が一般的である．また，浸
水地帯を走行し，摩擦部分が水にぬれた場合も摩擦係数が変化す
る．この現象をウォータフェードという．ウォータフェードの場合
は，水にぬれた直後に効きが低下することが一般的であるが，その
後の回復過程で効きが上がりすぎる場合があり注意が必要である．
これら効きの変化を小さくして安定性を確保するため，最近はブレ
ーキ力発生の占める割合の大きい前輪のブレーキにディスクブレー
キ（disk brake）が採用されている．
停止距離性能は，速度を制御することや意図した場所に停止する
など運転者の意図した制御のブレーキの使用に対し，機械装置の緊
急停止に匹敵する最短停止距離を意味する場合が多い．安全の面か
ら緊急時において最短距離で止まれることが重要であるが，ブレー
キとタイヤの特性を合わせた性能であり，タイヤ-路面間の摩擦係
数が最大となるタイヤが回転している状態を得られるよう，前後輪
のタイヤがロックせずにタイヤの特性を使いきるブレーキ力のバラ
ンスを設定する必要があるが，現実的には非常に難しい．一般の運
転者にとって，タイヤがロックせずに最短停止距離を得ようとブレ
ーキを制御することは非常に難しく，これを補うためABSの採用
が一般化してきた．
車両挙動安定性は，前輪ロック（front wheel lock up）または後
輪ロック（rear wheel lock up）を引き起こす前後のブレーキ力バ
ランス（brake force distribution）によるところが大きい．前輪
ロックの場合は操舵性が失われ，後者の場合はスピンを引き起こす
原因となる．ブレーキ力のバランスをつねに車両の諸元に合わせて
最適にすることは不可能に近い．そこで安全性の面から，後輪ロッ
クによる自動車のスピンを避けるためプロポーショニングバルブを
用いて前輪ロックを先に発生させるか，ABSを用いてロックを避
ける特性としている．
また，左右のブレーキ力バランスも重要である．左右のブレーキ
力が異なる場合，特に前輪の左右のバランスが大きく異なるとブレ
ーキ力の大きいほうにモーメントが発生してハンドルをとられるこ
とになり（片効き），自動車が思わぬ方向に向かうことになる．
自動車関連法規は，自動車を登録するうえで最小限満たしていな
ければならない要件として，各国でそれぞれ規定されている．ブレ
ーキ性能についても，多くの国で定められている各国の要件を満た
さなければならない．1990年代前半までは，アメリカ（FMVSS
105），ヨーロッパ諸国（ECE R13），日本（保安基準12条）がそ
れぞれ独自の内容のブレーキ法規制を採用し，他の国がそれぞれに
準拠した法規を採用していた．それぞれの国が独自の法規を採用す
ることは，自動車の輸出入が世界化する中，安全面，経済面の課題
となった．1980年に，乗用車の統一法規を検討する会議体がUN/
ECE（国際連合/欧州経済委員会）の中に設定され，日本，アメリ
カ，ヨーロッパ各国の参加により乗用車の統一法規が制定され，批
准が進みつつある ．

a.ブレーキ効き性能 ブレーキの効き性能は，液圧式ブレー
キシステム（hydraulic brake system）を例にとると，運転者の踏
力に対する発生減速度の比で，この間の関係は，まず運転者の踏力
がブレーキペダルへ伝えられ，ブレーキペダルのペダル比（pedal
ratio）とブレーキブースタの倍力比により増大され，ブレーキマ
スタシリンダ（M/ Cyl. ）を押し込む結果，力が圧力に変換される．
発生した圧力は，ブレーキ液を媒体にして伝達し，ホイールシリン
ダピストンを押し，ホイールシリンダ（W/Cyl. ）で力に変換され
て，ライニングを回転体に押付ける．
踏力→ペダル比×倍力比
押し力
M/ cyl.断面積 ×液圧×W/Cyl. 断面積→ライニング
ライニングと回転体の摩擦力によりブレーキ力が発生し，これが
路面に吸収されることで減速度が発生する．
ブレーキの効きを良好なものに確保するためには，踏力をいかに
倍力して，W/ cyl. 断面積を押す力を大きくするか，またはW/ cyl.
により回転部に押付けられるライニングの摩擦係数（μ）を大きく
とることが考えられる．
ただし，各要素の増幅度合いを高める場合，それぞれのトレード
オフについて考える必要がある．
ペダル比を大きくする場合は，ブレーキペダルストロークが大き
くなるデメリットがあり，これはM/cyl.断面積を小さくする場合
も同様である．
ペダルストロークも運転者にとっては重要な指標の一つで，長す
ぎると操作性の悪化を伴い，効きに対する感覚を低下させる．
倍力装置の倍力比を大きくする場合は，ペダルストロークが大き
くなるなどのデメリットはなく，効果的であるが，設定するスペー
スやコストが高くなる．W/cyl. 断面積を大きくすることでも増幅
効果は増すが，ディスクホイール内のスペースとの兼合いで限界が
ある．ライニングのμを高めることは機能的なデメリットはない
が，一般的にブレーキ鳴きが出やすくなるなど性能面でのデメリッ
トを考慮する必要がある．

c .車両挙動安定性 車両挙動安定性の確保は，急制動時に車
線を逸脱することなく停止することの可否につながるが，この特性
は前後ブレーキ力配分によるところが大きい．
制動時には減速度により荷重移動が起こり（前進時は，前輪軸重
が増加し，その分，後輪軸重が減少する），その動的荷重に促した
適正なブレーキ力を算出したのが理想ブレーキ力配分である．理想
ブレーキ力配分線の一例を図2･37に示す．
図2･37においては，実ブレーキ力配分が後輪ロック限界線を横
切る形になっており，後輪ロックの特性を示している．
また，実ブレーキ力配分が前輪ロック限界線を横切る場合は，前
輪ロックの特性となる．
前輪ロックでは，後輪が回転しているために発生する後輪のコー
ナリングフォースが車両姿勢を進行方向（車両の慣性力の方向）に
矯正する力として働き，直進状態で車両は停止するが，前輪のコー
ナリングフォースは零となり，操舵性は失われる．
後輪ロックでは，後輪のコーナリングフォースが零となり，わず
かの車両姿勢の変化に対して，前輪のコーナリングフォースが車両
をさらに巻き込む方向に働くため，車両のスピンを引き起こす．
四輪同時にロックした場合は，前後輪ともコーナリングフォース
が零となるため，車両にはヨーが発生せず，外乱がなければ進行方
向に平行な状態を維持して停止する．
以上のことから，急制動時に車両の挙動が安定する前輪が先にロ
ックするよう，前後のブレーキ力配分を設定することが一般的であ
る．
一般の運転者が，タイヤをロック直前の状態でブレーキ力を維持
することは非常に困難であり，ロックを避け操舵性を確保しつつ，
高い減速度を得られるABSの装着が一般化している．
