# 7・8・3 漁業計測機械

> **章**: 第7章 農業・水産機械
> **節**: 7・8 水産機械

航空機を電波により探知するレーダ(radar)に対して、水中の音響による距離測定は、ソナー(sonar: sound navigation andranging)と呼ばれている。一般に水産では、漁業計測機械とは音響機器を用いた漁業資源生物の計測機械を指す。ソナーを大別すると、i. 超音波パルスを船底から真下方向に発射する方式のもの(魚群探知機)ii. 超音波パルスを船底から水平ないし斜め下方向に発射する方式のもの(サーチライトソナー、スキャニングソナー)iii. 遠隔の魚群探知機より得られた情報を伝送し表示、観測するもの(ネットゾンデ)などがある。なお、超音波を垂直方向に発射するものを魚群探知機、水平ないしは斜めに発射するものをソナーと呼ぶ(図7・77)。a. 魚群探知機魚群探知機(fish finder)は、漁船の船底に装備した送受波器から海中に超音波パルスを発射し(ビーム)、これが魚群や海底に当たって反射して戻るまでの時間を測ることにより、魚群の存在とその距離、海底間での距離などを計測する機械である。魚種や漁業種別にそれぞれ適したものが開発されているが、周波数は15~200kHz を一般に用いている。そのうち、単一の周波数を用いるもの、15~50kHz のように低い周波数と200kHz のようなミリ波を併用してそれぞれの特色を発揮させて用いる方法も図7・74 自動イカ釣り機図7・73 自動カツオ釣り機図7・77 各種音響探知機図7・75 魚体大小選別機図7・76 漁網洗浄機ある。計測距離レンジも50m以下の浅海用のものから、1000m以上の深海用のものまで漁業種類に応じて使用されている。スルメイカなど超音波の反射損失の大きいものには、超音波の発射エネルギーと受信感度の大きいものが、底魚を海底から分離して発見するには分解能の高いミリ波が用いられている。なお、原理は魚群探知機と同じであるが、コンピュータによるエコー信号処理により魚の量を解析できる計量魚群探知機がある。b. サーチライトソナーサーチライトソナー(search lightsonar)は、送受波用振動子に適当なふ(俯)角をもたせて旋回させ、その指向方向における目標物の反射波を検知する方式のものである。超音波の指向方向は垂直面(ふ角0~90°)、水平面(全周360°)で変化させられるから、広範囲の探索が可能である。ただし、超音波を発射し、設定された距離範囲を往復するのに要する時間を待って、次の超音波が少し角度を変えて発射される。このため、探索を一巡するのに、ある程度時間がかかる欠点がある。c. スキャニングソナースキャニングソナー(scanningsonar)はサーチライトソナーの欠点である探索時間を低減させるため、多数のビーム発信用振動子群を備え、送信時は均一レベルの超音波を広範囲に瞬時に送波し、受信時は尖鋭な受信ビームを電子的に切替えて、高速度で旋回させて反射波をとらえ、CRT 画面に表示させるものである。レーダとは異なりスイープの方向も直線状ではなくスパイラルである。このため、その繰出されるスパイラル状のラスタは、送信されてから超音波伝ぱの速度と方位が同期しているので、距離と方位がレーダを見るように探知でき、反射波の強弱により輝度が変化して魚群の分布と密度がわかりやすく表示できるようになっている。現在、このスキャニングソナーは魚群を探索するのに必要不可欠な漁業計測機器になっており、漁船の大小にかかわらず、イワシ・サバ巻網漁船、カツオ巻網漁船、サンマ棒受網漁船に設置されている。d. ネットゾンデネットゾンデ(net zonde)は水中にある漁具の状態を連続測定し、その情報を船上に伝達する装置である。本器は発信器と受波器および受信器よりなり、発信器を網に取付け、発信器内に内蔵された音響測深器によって測定した値を超音波FM方式で伝達し、この信号を船外に垂下した受波器で受け、さらにこれを有線で船内の受信器に導いて紙上に記録するものである。信号伝達距離は2000mぐらいである。中層びき網の深度測定、底びき網の網口の左右および上下距離や入網魚群量の測定、巻網の網裾の沈降状況を観測するのに使用されている。また、水中を超音波FM方式で伝達しないで、空中を超短波で伝達するテレサウンダ(tele-sounder)方式のネットゾンデもある。これは、定置網に入網する魚群量を測定したり、巻網の船団操業において、付属船の情報を主船に送るのに使用されている。(a) 魚群探知機(c) スキャニングソナー(b) サーチライトソナー
