# 7・8・4 栽培・養殖機械

> **章**: 第7章 農業・水産機械
> **節**: 7・8 水産機械

養殖魚介類は,労働力の安価な発展途上国からの輸入だけでなく,ある程度賃金の高いノルウェーなどの先進国からも急増している.このことは,日本における海面養殖漁業の基盤の確立が重要な課題であることを示しており,栽培・養殖機械(aquaculturemachinery)の開発,導入による省力化および機械化が重要である.a. かくはん造粒機養殖漁業で供給される飼料は,生餌と配合飼料に大別できるが,近年の生餌であるマイワシ資源の減少,価格の上昇により,生餌離れが進み配合飼料への転換がいっそう加速しつつある.粉末の配合飼料(マッシュ)に生餌を混ぜて粉砕,かくはん,造粒してペレット状に成型したものをモイストペレット(MP)と呼び,養殖現場で多く用いられている.また,工場で生産されるドライペレット(DP)は,配合飼料を含む原料,各種添加物等をペレットミル装置で粉砕,かくはん,造粒,乾燥させたものである.さらに,ソフトドライペレットまたはエクストルーディッドペレット(EP)はDPに油脂を混入させ,造粒させたものである.養殖現場で使用されるかくはん造粒機(extruder)は,かくはん機と造粒機の上下一体型が多く見られ,上部のかくはん機では原料のクラッシャやチョッパでの粉砕,ミキサによるかくはん,混合を経て,下部のスクリュー式造粒機による圧縮,成型によりペレット状の飼料を製造する(図7・78).b. 自動給じ機自動給じ機(automatic feeder)は,ペレット状の配合飼料を生けすおよび養殖槽にばらつきなく散布させる装置である.散布方法には自然落下式や送風式がある.また近年,給じする各生けすに,指定した時間に,選択した配合飼料を,設定した量だけ,散布する自動給じシステムが開発された.一組の給じ機図7・78 かくはん造粒機図7・79自動給じ機(a) 外形(b) 内部計数部図7・80 稚魚計数カウンタ図7・81 活魚大小選別機1―187と分岐装置から成り立ち,分岐装置以降は直径80mm前後のゴムホースにより空気搬送が行われる(図7・79).c. 通気装置通気装置(aerator)は,ウナギやコイの止水池で見られるように,電動機によりベーンを直接駆動させる水車式と,圧縮空気を多孔質のエアストーンを通して,微細気泡を発生させるブロア式があり,両者とも水中の溶存酸素を増加させる装置である.d. 稚魚計数カウンタ稚魚計数カウンタ(fish counter)は,1尾が通過可能な直径を有する管と光センサとの組合わせた計数装置や,CCD カメラを用いた画像処理による計数装置などがある.前者は装置が安価であるが気泡,固形物および飼料などを誤計数する場合がある.後者は装置が高価であるが計数精度が高い.しかし,両者とも複数の魚体が重なれば,誤差が生じるため,なんらかの対策が必要である(図7・80).e. 活魚大小選別機活魚大小選別機(grader)は,スクリーン式と呼ばれるもので,断面形状がA 型を有する全長2mの型材を7本程度,適宜間隔を設けて,こう配をつけて配置する.間隔は上流側が狭く,下流側が広く,かつ固定された型材と上下方向に可動する型材が交互に配置されている.魚体幅が同間隔より狭くなった箇所で下に落下する.前述した魚体大小選別機は鮮魚に,本装置は活魚に使用されるが,同様な使われ方をする場合もある(図f. ろ過装置ろ過装置(filter)は固形物,懸濁物を除去する物理フィルタ,魚介類の排泄したふん,尿,魚の粘液等により生じるアンモニア態窒素を,ろ過材に付着させた海水浄化微生物により亜硝酸に分解し,さらに他の微生物により亜硝酸を窒素に分解する生物フィルタなどがある.
