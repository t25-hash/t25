# 2・3・2 ハイブリッドシステム（hybrid system）

> **章**: 第2章 自動車
> **節**: 2・3 自動車の構造

ハイブリッドシステムの歴史は古く，1900年代初頭の内燃機関
の揺籃期に自動車に適用された例がある．その後の内燃機関の発達
により，しばらく自動車用としては顧みられることがなかったが，
1970年代，大気汚染問題の顕在化，オイルショックなどを契機に
ハイブリッド技術が見直され，さまざまな研究が進められてきた．
1990年にはアメリカ，カリフォルニア州の大気浄化法の成立に
より，ZEV（zero emission vehicle），すなわち電気自動車（EV）
の実用化が推進されることとなった．しかしコスト，一充電当たり
の走行距離不足など，従来の自動車の代替としてのEVの成り立ち
が困難なことが明らかになると，ハイブリッド車などもZEVの代
替として認められるようになった ．
一方，地球温暖化の防止策としてCO削減の国際的な合意が形
成されつつある中，COの排出量に少なからず影響を与えている
自動車に対しても，日本国内における2010年の燃費規制，ヨーロ
ッパにおけるCO排出量自主規制などの形で低減要請が厳しくな
ってきている．
このような状況の中，ハイブリッドシステムは，排出ガス・燃費
低減効果の高いシステムとして注目を集めており，ハイブリッド車
を含む低公害車導入に対するインセンティブも各国で導入されるな
ど，これからの普及が期待されている．
a.ハイブリッドシステムの概要 ハイブリッドシステムは，
技術的には作動原理の異なる2種以上の原動機を有するかあるいは
それらを組合わせて作動させるシステムと定義される ．
自動車用ハイブリッドシステムにおける原動機の一つは，一般的
に用いられるエンジンであるが，もう一つの原動機としては車両エ
ネルギーを回生利用する目的から，電気モータ，油圧モータなどが
使われる．近年，EVの開発に伴いモータの小型化技術，システム
制御技術，バッテリ技術などが進展し，電気モータシステムの車両
適合性が向上したことにより，エンジンと電気モータを使ったハイ
ブリッドシステムの実用化が進んできている．
ハイブリッドシステムのねらいとするところは，複数の動力源そ
れぞれの長所を生かし，単一動力源でのシステムをはるかに上回る
諸性能を発揮することである．反面，電気システムの追加による質
量の増加，コストの上昇が避けられず，商品としての位置付けが問

燃費を目的としたハイブリッドシステムの基本機能は，エンジン
の運転点の最適化と運動エネルギーの回生の二つである．
ハイブリッドシステムにおけるエンジン運転点最適化は，熱効率
の悪い軽負荷域でのエンジン運転を避けることが基本である．その
ために，軽負荷時のエンジン停止やEV走行（バッテリ電力による
走行），高効率領域運転の多用化などができるよう，システムが構
成される．
一方，運動エネルギーの回生は，車両減速時にモータを発電機と
して作動させ，電気的に制動力を得ることにより行われ，回収した
電気エネルギーはバッテリなどに蓄えられる．蓄えられた電気エネ
ルギーは，EV走行や，エンジン駆動力のモータによるアシスト，
補機類の駆動などで消費され，エンジンの消費する動力を低減す
る．
ハイブリッドシステムを構成する要素としては，エンジン，変速
機構に加え，駆動・回生可能なモータおよびその制御装置，充放電
可能なバッテリ，およびこれら全体を制御するシステム制御装置が
必要になる．またエンジンを車両の運転中に停止するシステムの場
合は，これらに加え，エンジンの始動装置の高性能化や，エンジン
で駆動される補機類の機能をエンジン停止中にも維持するための装
置または代替手段が必要となり，従来システムの大規模な変更が必
要となる．
b.ハイブリッドシステムの分類 電気モータを併用する自動
車用ハイブリッドシステムを動力伝達の形態から分類すると，まず
次の2種に大別できる．
一つはエンジンからの動力をいったんすべて電力に変換し，電気
モータで車両を駆動するシリーズハイブリッドシステム（series
hybrid system）で，動力はエンジンから発電機，バッテリを経由
し，駆動軸につながったモータへと直列（シリーズ）に伝達され
る．このシステムは，バッテリ電力を主たる動力源とし，エンジン
駆動の発電機で効率良く電力を補いながら走行するもので，EVに
近いシステムといえる［図2･73(ａ)］．
もう一方はエンジンの動力and/ or電気モータの動力で車両を駆
動可能なパラレルハイブリッドシステム（parallel hybrid system）
であり，動力はエンジンからの経路と，バッテリからの経路の二つ
が並列（パラレル）して駆動軸に伝達されるものである．このシス
テムは主としてエンジンの動力により走行するもので，従来の車両
の延長上に位置するシステムといえる．
この方式にはいろいろな種類が考えられるが，エンジン動力主体
で運転することから，変速機構が必要となる．変速機構としては，

前者の変速機機構としては，無段変速機（CVT），多段式自動変

プリット式ハイブリッドと呼ばれる．

いたシステム，後者はハイブリッド性能を限定し電気系のパワー・
エネルギーを極力リーンにしたシステムとしてとらえられている．

リーンにしやすく，マイルドハイブリッド構成に適していると考え
られる．
c .シリーズハイブリッドシステム シリーズハイブリッドシ
ステムの構成例を図2･74に示す．このシステムではエンジンが増
速機を介して発電機と接続されており，発電機の電気出力は整流器
を介してバッテリに接続されている．また，バッテリはインバータ
を介して駆動用モータに接続されており，モータ出力は減速機を介
して出力軸に接続されている．
このように，シリーズハイブリッドシステムではエンジンが駆動
軸と機械的には結合しておらず，エンジンの動力を電気エネルギー
に変換して動力伝達するということや，バッテリを駆動エネルギー
のバッファとしても利用できるため，エンジンは負荷変動の少ない
運転ができるという特徴をもつ．またエンジンの出力を車両の平均
的な要求動力に設定することにより，エンジンをつねに高効率な負
荷状態で運転でき，燃費や排出ガス性能を向上させることが期待で
きる ．
車両の運転に必要なパワーは直接的にはバッテリから供給され
る．バッテリへの充電は，充電状態（SOC:state of charge）がつ
ねに所定の値となるように，エンジン動力で発電し充電制御する．
また，回生制動によりブレーキエネルギーを回収することも可能で
ある．

e .パラレルハイブリッドにおける燃費向上手段
.エンジン運転点の最適化 エンジンの熱効率は負荷によ
って変化し，ガソリンエンジンでは特に軽負荷時の熱効率低下が大
きい．そのため軽負荷運転状態では，エンジンで直接駆動するとき
のエネルギー効率よりも，エンジンの高負荷域で発電しバッテリへ
蓄えた電気エネルギーを再びモータで機械的駆動エネルギーに変換
するときのエネルギー効率のほうが優れる領域が存在することにな
る．
これまで発表されたハイブリッドシステム搭載車では，エンジン
の高効率化や無段変速機能によるエンジンの燃費最適運転，車両の
各種損失低減などを含めて，市街地走行モードでの燃料消費量を従
来車比最大で約半減する．しかし，平均車速の高い使用域では，
EV走行，回生の頻度の減少により燃費向上効果は減少する．
f .システム構成要素
.エンジン 燃費重視の観点から，高膨張比エンジン，リ
ーンバーンエンジンなど，高効率エンジンが選ばれる．バッテリパ
ワーの併用により，特にエンジン低回転時の動力性能を向上させた
り，高効率化によるエンジンパワーの低下を補助することが行われ
ている．
図2･78は模式的にその状況を示した例である ．
パラレルハイブリッドにおけるエンジン運転点の最適化は，この
ような考え方を基本にEV領域（エンジンを使用しない領域）を設
定することにある．EV走行の領域は，走行条件や回生によるエネ
ルギーの割合，エンジンで発電するときのエンジン熱効率などによ
って異なるが，小型乗用車では概略10kW程度の領域である．パ
ラレルハイブリッドのバッテリは一般的には外部充電機構をもた
ず，エンジン動力による発電と回生により，充電状態（SOC）が
維持される．したがって，パラレルハイブリッドにおけるEV走行
は一種のシリーズハイブリッド運転と考えることができる．
.回生（regeneration) 回生時，駆動用モータを発電機
.モータ 高効率でコンパクトな設計が可能なPMシン
クロナスモータが選ばれることが多い．モータに必要な性能はシス
テムのコンセプト，適用車両により異なるが，数kWから百kW
程度の出力が要求される．作動電圧はおおむね出力に比例し，
42～650V程度が選ばれている．自動車用として生産性の良い構造
が求められるため，ステータコアを分割式としたり，コイルの巻線
方式に集中巻きを採用する例がある ．
.バッテリ（電源系) 高電圧系，低電圧系（12V）の2
系統のバッテリをもつ．低電圧系は従来車同様，ECU，照明等の
電源であり，高電圧系は車両駆動用モータの電源である．発電機能
を効率の良い高電圧系にまとめ，低電圧系にはDC-DCコンバータ
で電力供給し充電するのが一般的である．高電圧バッテリとして
は，比較的低電圧（42V）やエネルギー容量の大きな適用例には
鉛電池，より高電圧な適用例にはニッケル水素電池，リチウム電池
が使われている．
.インバータ（inverter ) 直流電源をPWM制御により
交流に（あるいは逆の制御も）変換し，PMシンクロナスモータを
駆動する機能をもつ．電流を直接スイッチングするものがIGBT
等のパワートランジスタであり，近年のモータ技術の進展はこのパ
ワー素子によるところが大きい．
