# 7・8・6 加工機械

> **章**: 第7章 農業・水産機械
> **節**: 7・81 ).

魚介類のおもな加工品には,生食用フィレー(二,三枚下ろし),切身,撒塩品・塩蔵品,漬物,むき身,ボイル品,素干・煮干,塩干品・乾物,くん製,フライ・天ぷら,調理品(ハンバーグ等),ねり製品(カマボコ等),佃煮,缶詰,節類,すりみ,調味液等がある.a. 割さい機三枚下ろしは左右側体と中骨の3枚に切断,三枚開きは魚体を左右に開き中骨抜きの処理方法をいう.また二枚下ろしおよび開きは中骨をどちらか一方の魚側体に残す処理方法である.この処理を機械化した装置が割さい機(身開き機,filletingmachine)である.魚体の保持および送り機構は,背骨に沿った縦方向に切り開く場合は,体高(背腹の高さ)に等しい幅の,平行する2枚のベルトコンベヤに魚体を挟んで縦に送る.切断部では重ね合わされた2枚の回転刃を一組として,背びれおよび胸びれ方向から二組の回転刃により切断する.中骨抜きの処置では,重ね合わされた2枚の回転刃の間隔を調整することにより三枚下ろしの精度を向上させている(図7・86).b. ねり製品製造装置カマボコ等のねり製品の種類は極めて多く,さらにその中間製品を原料として新たな製品が開発されたので,工程形態も複雑に発展してきた.以下に,おもな工程を示すが,その多くが連続処理で行われるプラントである.ただし{}内の工程がない場合の製品もある.i. 原料魚-調理-魚洗-採肉-さらし-脱水-裏ごし-すりみ(中問製品)-ブレンド-充てん-凍結-冷凍すりみ(製品)ii. 冷凍すりみ機械タラ類は独特の凍結変性により直接凍結すると,カマボコ形成能力を失うので,鮮度のよいうちに採肉して,多量の清水でさらし,凍結変性前駆原因成分を溶出除去した後,変性阻止のための糖類を加え,すりつぶし,練り混ぜ,ぺ一スト状のまま袋詰めして成形凍結硬化する.これを冷凍すりみといい,カマボコやコピー食品の原料になる.水産加工ではミール製造に次ぐ規模の装置産業であり,接触凍結装置が用いられる.母船工場では造水装置,陸上工場では排水処理設備も必要になる.d. 採肉機採肉機(meat separator)は,開き状に処理された魚体を,穴径4mm程度のパンチングメタル製のドラム外面にゴムベルトで押付けて,ドラム内に肉質部をこし出す.開き魚体に噴水を当てて血あい部分でない普通肉のみ採肉する方式も実用化されている(図7・87).e. さらし装置さらし装置(meat washer)はかくはん機をそなえた水槽での回転処理方式,または網製の回転ドラム内の肉に散水する連続処理方式がある.脱水機は直径約400mm,長さ5000mm程度のスクリュープレス,または遠心装置が用いられる.裏ごしは網製のドラム内面をしゅう動回転するブレードで肉をこし出す.f. ミキサ冷凍すりみの製造では,脱水魚肉に砂糖や添加物を加えてかくはん混合し,さらに磨砕作用を加えてペースト状にする.これに用いられるのがミキサ(mixer)である.このミキサでは古くから擂潰(らいかい)機が用いられる.これは,回転うすと回転きねとの相対運動で魚肉をすりつぶして,混ぜ合わせる機械である.現在では,同様の目的で,S 字型回転刃物をうす中で40rpm程度の速度で回転させ,練り混ぜるミキサもある.g. 成型機加工成型後,時間の経過に伴って,すりみが弾性体に変性することを座り(すわり)といい,すりみを商品の形に加工するものを成型機(shaping up machine)という.カマボコ成型は,リテーナに対してすりみを押出し成型する.複合成型は,一度そうめん状に成型したものを圧縮して弱く再結着させ,カニ足風のテキスチュア(力学的な食感)を作る.竹節成型は,底面がピストン運動するくぼみを設けた回転ドラムにすりみを注入し,くぼみ底面がドラム外径面にせり出した位置で,回転する金属棒にすりみを絡める.ソーセージ類はビニリデンフィルム等に注入することで成型する.h. 肉詰め機容器に固形状食品を詰めるものを肉詰め機(meat filler)という.マグロおよびカツオは,あらかじめ加熱し硬化させた普通肉のみ精選して肉詰めする.この場合肉が硬いので向かい合う2条のキャタピラコンベヤで肉を缶直径まで圧縮しながら送り,缶高さに合わせて切断する.サバ,サケ等は,あらかじめ缶高さに合わせて魚体を輪切りにする.缶高さと等しい厚みの回転円板の円周を切欠いて形成した缶と同形のくぼみ(メジャーボックス)に,間欠回転する円板の外周から輪切り魚肉を押込み,円板円筒面に接する刃物で余剰肉を切断し,空缶のある位置に送り肉詰めする.イワシ等の小形魚はドレス状のものを数尾缶に整列肉詰めする.いずれの肉詰め機も毎分100~200缶処理できる.i. 缶詰製造装置缶,びん,気密箱,気密袋等に酸素を脱気して,あらかじめ加熱殺菌した食品を詰め密封し,または密封後加熱殺菌して貯蔵性の高い食品を製造することを封蔵といい,代表的なものに缶詰がある.これらの機械はシステム化された流れ作業により缶詰が製造されている.j. 缶詰巻締め機缶にふたを取付け,密封するために溝付きのロール(seaming roll)で塑性加工することを巻締め(seam-ing)といい,その機械を缶詰巻締め機(can seamer)という.巻締め前の缶の胴部上端は外側につばが形成されている.円板状の薄板製ふたの周囲もくぼみが加工され,くぼみ内面にシール剤が塗布されている.缶胴のつばが,ふたのくぼみにはまるように組立て,1―189缶底をリフタ(can lift pad)という円板で上方向にもち上げるように支持して,ふたはチャック(chuck)という円板で下方向に押付ける.チャックは加工の際のダイスでもあり,テーパがついている.加工の第一段階は,溝の深いロールをチャックの方向に押付けるようにして缶軸周りを公転させ,ふたの外縁を絞り込むと同時に,缶胴のつばを下方に絞りリフタの外縁先端が缶胴材にフックした多層構造にする.リフタの押上げ力の大きさによって缶胴のつばの変形位置が決まる.第二段階は溝の浅いロールで多層構造を密着させる.ロールは加工のダイスとして作用し,その溝をシーミンググルーブ(seaming groove)という.チャックを中心に2種類4個のロールを配置したシーミングヘッド(seaming head)が多用される.ヘッドは中空四軸同軸のスピンドルの内から3番めの軸の軸端に取付けて回転される.ロール軸は偏心スリーブでヘッドに軸受支持される.偏心スリーブに取付けたピニオンは,スピンドルの最外中空軸の軸端に取付けた歯車で駆動される.偏心スリーブのピニオンは遊星歯車として回転して,ロール軸心をサイクロイド運動する.スピンドルの内から2番めの軸は回転せずにチャックを支持し,中心軸は巻締め終了時に下方に運動して,缶をチャックからノックダウンする.ヘッドは真空チャンバ内に納め,缶は間欠運動するロータリバルブでヘッド中心直下に送り,脱気と巻締めを同時に行う.この方式の巻締め機(seamer)をサニタリーシーマといい,毎分80缶程度処理できる.多数のヘッドを備えたものでは毎分1500缶程度処理できる.このほか,ロールを固定して缶を回転する方式,大きなリングの内側に付けたシーミンググルーブに缶を押付け回転する方式,だ円缶,角缶,馬てい(蹄)形缶等を巻締めるもの,真空にしないでガスを吹付ける方式,初め緩く巻締め,次に缶を加熟して,発生した缶内の蒸気で脱気し巻締める方式等がある.図7・88および図7・89に,二重巻締め構造と,巻締め機を示す.k. press,エキスペラともいう)で脂肪分と水分を除去して,絞り粕を乾燥・粉砕し,貯蔵・輸送性を高めた高タンパク質飼料である.脂肪分は酸化しやすいので,プレスのとき可能な限り除去する.プレスで分離された液は,分離板形遠心分離機等で顕濁物を分離し,さらに円筒形遠心分離機で油脂を分離する.なお,油脂は製品となる.液分は,多重効用濃縮缶で濃縮し,ソリュブルという高タンパク製品を得る.ソリュブルは絞り粕にまぜることもある.煮じくはスクリュー式連続移送装置に蒸気を送り加熱すると,煮じく工程でも油脂が分離される.絞り粕の乾燥はロータリキルンが用いられる.原料処理量は日産数千tに及ぶ設備もある.
