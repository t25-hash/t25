# 7・8・8 加熱・乾燥・解凍装置

> **章**: 第7章 農業・水産機械
> **節**: 7・93 )。

加熱および乾燥の目的の一つに、殺菌がある。水産原料は、特異な菌で汚染されることが少なく、おもにポツリヌス菌胞子の殺菌が対象とされる。この菌は毒性が強く、嫌気性で真空中でも繁殖する特徴を有している。この種の細菌を一定の温度(°C)で加熱したときに、完全に死滅させるのに要する時間τ(min)は通常、log τ=--121.1(7・5)と表される。は121°Cでの死滅時間、は死滅時間が10分の1に減じるのに要する温度上昇量である。ポツリヌス胞子は=4図7・92 接触式凍結装置図7・94袋包装機図7・93 ブライン浸漬式凍結装置1―19 1min,=10°Cである。非定常で温度が時刻τの関数の場合は、缶内の温度上昇の最も遅い部分で、1τ=1(7・6)になったとき殺菌が完了したとみなす。ビタミン等有効成分も同様の傾向で破壊され、その値は細菌の約2倍であるので高温短時間殺菌が望まれる。温度上昇に要する時間は寸法の二乗に比例するので、薄い形状の平たい缶が用いられるようになってきた。加熱操作のタイムスケジュールを設定する試験生産の際に、缶中心に差し込んだ温度計測定値より式(7・6)を演算する専用装置がある。また、任意の位置に差し込んだ温度計の加熱初期における時系列測定値が、熱伝導方程式の厳密な解を満たすように、表面の熱伝達率、内部の温度伝導率、および温度計挿入位置を推算決定し、中心温度の変化を予測し、冷却開始適正時刻をリアルタイムで予報するシステムが開発されている。加熱媒体は、115~160°Cの飽和水蒸気、または加圧水が用いられる。したがって、加熱殺菌装置(retort)は圧力容器である。ふつう直径1500mm前後、長さ1200~10000mm程度の円筒容器である。バッチ式では缶をかごに入れて台車で搬入する。連続式では水封ロータリバルブ、または内圧と平衡する高さの水柱塔内コンベヤによって搬送する。伝熱性能を上げるため、熱媒をかくはんしたり、缶を回転することもある。殺菌完了後の冷却開始初期には、缶内の温度が高く、内圧も高いので、プログラム制御で装置に空気を送り缶に外圧加え宙の変形を防止している。
