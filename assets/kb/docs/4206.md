# 2・3・6 シャシ制御装置（chassis control system）

> **章**: 第2章 自動車
> **節**: 2・3 自動車の構造

自動車のシャシ制御装置は，運動性能の機能，走る，曲がる，止
まる，乗り心地の性能改善を目的としたシャシ構成要素を制御する
装置である．制御対象により，駆動系，ブレーキ系，サスペンショ
ン系，ステアリング系の制御に分類される．
a.駆動力制御システム
.トラクションコントロールシステム（traction control
system) 駆動車輪のスリップ率制御を主体として，車両の駆動
性能を制御するシステムである．電子制御スロットルとアンチロッ
クブレーキ制御装置を応用し，駆動輪がスリップすると，スロット
ルを閉じ，駆動輪にブレーキ力を加えて，駆動力を減らしスリップ
を抑える制御を行う．四輪駆動車では四輪の駆動力を制御し，発進
時等の滑りを防止する（図2･126) ．
.駆動力配分制御システム（traction distribution control
system) 四輪駆動車のセンタデファレンシャルの差動を油圧ク
ラッチで制御し，前後輪の駆動力配分を差動制限することにより，
加速減速で四輪の駆動力を最大にする装置である．また，左右輪間
に増速関係を作り出して駆動力を左右に配分するギヤと湿式クラッ
チをもつ駆動力制御装置で旋回中の旋回外側輪の駆動力を内側より
大きく配分制御し，旋回加速時のアンダステアを改善するシステム
があり，両者を組合わせたものもある（図2･127) ．
b.制御ブレーキ
.アンチロックブレーキシステム（ABS:anti - lock brake
system) 路面の摩擦係数が著しく低下したり，前後，左右輪で
異なったりしたときも車輪ロックを生じさせないために，あるスリ
ップ率になるように制動油圧（または空気圧）を制御するシステム
図2･128 スタビリティコントロールシステム
である．ロックが防止され，車としての方向安定性，操縦性が保た
れ，正しい姿勢で停止し，停止距離も多くの場合短くなる．制御は
車輪からの速度信号をコンピュータに入れ，車輪の減速レベルを調
整しながら行われる．
.ブレーキアシストシステム（brake assist system) ド
ライバが強いブレーキを必要とする場合にブレーキ力をアシストす
るシステムである．ドライバの急ブレーキを油圧の立上り速さなど
で検出し，ブレーキ油圧を加圧する制御を行う．
.スタビリティコントロールシステム（stability control
system) 急なハンドル操作や路面状況の急変など，不測の事態
におけるスピンやドリフトアウトを抑制するスタビリティコントロ
ールシステムのシステム構成の概略を図2･128 に示す．システ
ムのハード部品はABS，TRC，ARSなど既存の運動制御システ
ムですでに開発しているものをベースに，センサの精度，分解能向
上，ダイナミックレンジの拡大，ブレーキアクチュエータを四輪ア
クティブに加圧可能にするなどの変更が加えられている．車体横滑
り角βとその角速度dβの位相平面で後輪横滑りを判定し，四輪の
制動力で車体モーメントを制御している（図2･129) ．しかし，
車体スリップ角βは一般的なセンサでは直接検出することができ
ないため，操舵角センサ，ヨーレートセンサ，前後横加速度セン
サ，車輪速度センサなどを組合わせて推定している．
.電子制御ブレーキ（electronic control brake system)
ハイブリッド車では，回生ブレーキによりエネルギーを回収し，燃
費向上を図っているが，図2･130 に示すように，ドライバが要
求する制動力の中で，最大限の回生制動力を使用し，不足分を摩擦
ブレーキで調整する必要がある．そこで，ドライバに回生ブレーキ
との協調動作を感じさせずに，リニア弁で油圧制動力を制御するた
めに，ドライバ操作を電気的に検出する入力部と，ホイールシリン
ダ油圧の出力系を切り離し，電気的にホイールシリンダ油圧を制御
する，いわゆるブレーキバイワイヤシステムを電子制御ブレーキと
呼んでいる．また，予防安全の観点からは各輪の制動力を高応答で
精度良く独立に制御するために，ブレーキ油圧は各輪ごとに各2個
ずつ配置したリニア弁で制御することにより，通常ブレーキから
VSCなどの車両運動制御なども含めたすべての領域の制御を実現
している（図2･131) ．

c .電子制御サスペンション
.車高調整装置（height control system) 乗車人員，積
載荷重の変化による車体姿勢変化の防止，高速走行時に車高を下げ
ての走行安定性向上，悪路走行時に車高を上げて底付き防止などの
目的のために，車高調整装置が用いられている．車高調整装置は油
圧式と空圧式に大別され，油圧式はジャッキ式とハイドロニューマ
チック式に分類される．ハイドロニューマチック式と空圧式には金
属ばねを併用するものもある．制御は機械式と電子制御式がある．
ハイドロニューマチックの電子制御の例を図2･132 に示す．車
高センサで検出した各車輪のストロークをフィードバックして油圧
または空圧を制御する．減衰力制御と組合わせたものもある．
.減衰力制御 ショックアブソーバの減衰力バルブの油路
を切換えて，減衰力を可変にしたショックアブソーバ（図
2･133) を用いて，減衰力をそれぞれの走行条件に対応して切換
えるフィードフォワード方式と，車体加速度センサやホイールスト
ロークセンサを用いるフィードバック方式がある．フィードフォワ
ード方式は良路の通常走行で減衰力を低くして乗り心地を確保し，
急操舵，急発進，急制動時に減衰力を高く設定して車体姿勢の変動
を低減する．フィードバック方式は車体振動を抑制しながら路面入
力に対する乗り心地を向上させる方法として，スカイフック理論を
用いたカルノップ（Karnopp）の制御や最適制御理論の 制御な
どいろいろな方法が試みられている．
.アクティブサスペンション（active suspension) アク
ティブサスペンションはサスペンションの減衰力やばね力に相当す
る力を外部から付加して，車体振動を能動的に制御するシステムで
ある．低周波のみをアクティブ制御する空圧式（図2･134) と高
周波までアクティブ制御する油圧式がある．スカイフック制御や最
適制御理論の応用による路面入力に対する乗り心地制御と，車両の
加減速度や横加速度による車体姿勢の変動を抑える姿勢制御が行わ
れている．前輪と後輪のロール剛性比率を制御し操舵応答を改善す
ることも試みられている．力を外部から付加するシステムのため，
高価で消費パワーが大きく，これが普及の課題となっている．
.ロール剛性可変制御（roll stiﬀness control system)
良路での乗り心地向上のためロール剛性を低くし，悪路走行および
高速走行におけるロール角低減と接地性向上のため，ロール剛性を
高くし，状況に合わせロール剛性を制御するシステムである．機械
式と油圧式があり，どちらもスタビライザとロアアームとの取付け
部に装着し，結合部の剛性を可変させる機構である．

図2･142 バックボーン型フレーム
.レーンキープサポートシステム（lane keep support sys-
tem) 車線に追従するようにアクティブにステアリングの操舵
補助を行うシステムである．車両前方の車線を認識するCCDカメ
ラと，ステアリングに操舵力を付加するステアリングアクチュエー
タ，および制御演算を行うコントローラより構成される（図
2･139) ．システムの機能は，車線検出センサ（カメラ）から得
られた車両前方の車線情報により，車線のほぼ中央を走行するため
の操舵力を計算し，その操舵力の一部を補助的にステアリングに付
加するものである．これによりドライバのステアリング操作負荷を
軽減し，高速道路走行時の快適性を向上させる．また，装置作動
中，ドライバの運転状況からその後の車両位置を予測し，車線から
逸脱するおそれがあると判断した場合には，音と表示によりドライ
バに警報し注意を喚起する機能もある．
e .シャシ制御の将来 いろいろなシャシ制御システムが実用
化され，それらの制御の効果が明らかにされてきた．これらのシャ
シ制御技術は安全で快適な車社会のため，ドライバを支援する機能
としての進化とシステムの淘汰が行われていくと思われる．また，
レーダを利用した障害物の自動検出による警報や，自動回避なども
試みられている．これには相手の車や道路に何かの装置を要求する
インフラストラクチャ協調型に対し，何も求めない自律型の運転支
援システムのほうが普及しやすい．運転支援のためのシャシ制御ア
クチュエータ系はほぼ出そろったので，今後はドライバの目と頭の
機能である検出と判断の技術が確立されると，ドライバと協調した
ドライブバイワイヤシステムが完成するであろう．
