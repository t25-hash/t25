# 7・8・10 水産流通機械

> **章**: 第7章 農業・水産機械
> **節**: 7・93 )。

水産流通機械は，生産地から消費地までに用いられる流通および
a.製氷装置 製氷装置は，角氷を作る缶式と砕氷を作るプレ
他面に水を流下させ氷板を落下させる脱氷工程，必要な砕氷サイズ
に粉砕する砕氷工程がある．なお，脱氷時に使用された水は，次サ
イクル時の結氷用水として使用し，エネルギーの回収を図っている
（図7･96）．
b.活魚輸送装置 活魚輸送は，従来から酸素を供給するのみ
の簡単な装置により，トラックや船舶などで行われてきた．しか
し，近年，活魚の商品価値が向上するに伴い，従来のシステムに比
べて死亡率の低い装置が開発された．本装置は，輸送中だけでな
く，生産地市場および消費地市場における定置式活魚蓄養装置にも
使用されるようになった．活魚システムは，活魚槽の水を側流清浄
するもので，固形物，懸濁物を除去する物理フィルタ，アンモニア
態窒素を分解する微生物による生物フィルタ，微細な固形物の除去
と微生物に酸素を供給する曝気システム，水温調節装置からなり，
長期間の魚介類の蓄養を可能にしている．
