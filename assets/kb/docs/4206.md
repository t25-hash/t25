# 7・8・10 水産流通機械

> **章**: 第7章 農業・水産機械
> **節**: 7・93 )。

水産流通機械は、生産地から消費地までに用いられる流通および貯蔵に関する水産機械であり、製氷装置、活魚移送装置などがある。a. 製氷装置製氷装置は、角氷を作る缶式と砕氷を作るプレート式に大別される。角氷は従来から流通している135kg(300lb)型の1種類しかなく、水が注水される氷缶を塩化カルシウムのブラインで凍結する。プレート式は、冷媒が内部を通過する結氷板の片面を、循環水ポンプにより水を滴下させ氷板を作る結氷工程、図7・95 深絞り真空包装機他面に水を流下させ氷板を落下させる脱氷工程、必要な砕氷サイズに粉砕する砕氷工程がある。なお、脱氷時に使用された水は、次サイクル時の結氷用水として使用し、エネルギーの回収を図っている(図7・96)。b. 活魚輸送装置活魚輸送は、従来から酸素を供給するのみの簡単な装置により、トラックや船舶などで行われてきた。しかし、近年、活魚の商品価値が向上するに伴い、従来のシステムに比べて死亡率の低い装置が開発された。本装置は、輸送中だけでなく、生産地市場および消費地市場における定置式活魚蓄養装置にも使用されるようになった。活魚システムは、活魚槽の水を側流清浄するもので、固形物、懸濁物を除去する物理フィルタ、アンモニア態窒素を分解する微生物による生物フィルタ、微細な固形物の除去と微生物に酸素を供給する曝気システム、水温調節装置からなり、長期間の魚介類の蓄養を可能にしている。文献(1) 木谷収編著、生物生産機械学、(1993)、240-241、コロナ社。(3) 笈田昭・ほか3名、 関西支部から見た農業機械技術の発達、(2001)、 55-132, 農業機械学会関西支部創立50周年記念事業実行委員会。(4) 笈田昭・ほか7名、新版農業動力学、(1991)、195-241、文永堂出版。(5) 農業機械学会編、生物生産機械ハンドブック、(1996)、485-496、コロナ社。(6) 並河清、農業機械学会誌、37-3(1975)、295.(7) 小西達也、農業機械学会誌、51-6(1989)、89.(8) 津賀幸之介、農業機械学会誌、59-2(1997)、109.(9) 文献(5)の598ページ。(10) 日高茂實、農業機械学会誌、64-2、(2002)、111.(11) 文献(10)の117-121ページ。(12) (株)クボタ、 ワークショップマニュアル-トラクタKingwelKL 21ほか自動化装置編、(2000)、2-3.(13) ヤンマー農機(株)、 技術マニュアル-ヤンマー乗用型トラクタUS 535/US 540/US 545/US 550、(2001)、60.(14) (株)クボタ、 ワークショップマニュアル-コンバインAR 60・
