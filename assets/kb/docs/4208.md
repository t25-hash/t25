# 8・1・1 概要

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・1 基板材料製造装置

半導体シリコン結晶基板の製造工程の概要を述べる。通常、シリコン(けい素)は二酸化けい素SiO、けい石として存在しているので、高純度の半導体シリコン結晶を作る工程は、以下のようにけい石を還元することから始まる。(1)けい石を還元して金属けい素(約98%)を得る。(2)金属けい素に塩化水素を反応させてけい素塩化物を作り、これを蒸留・精製して高純度のトリクロロシランSiHClを得る。(3)高純度のトリクロロシランガスを水素ガスとともに電気炉内に導入して高温に加熱して分解し、高純度シリコン多結晶の塊を作る(シーメンス法)。(4)高純度シリコン多結晶を融点(約1400°C)以上の温度にて融解し、種結晶に従って固化させることにより単結晶塊(単結晶インゴット)を作る。その方法(8・1・2項)にはCZ 法(Czochralski method、引上げ法)とFZ 法(floating zone method、浮遊帯域法)がある。(5)単結晶塊の頭部と尾部を切断し、規定の直径に外周を削る。(6)単結晶塊を切断して円板(ウェーハ)を切り出す。(7)ダイアモンド砥石にてウェーハ外周を研削し、端面を円弧状に研削する。(8)回転する上下の円盤の間に挟んで両面を研磨して厚さをそろえる(ラッピング)。(9)機械加工による損傷が残っている層をエッチング液により除去する。(10)研磨布が貼られている定盤にウェーハを押付けて研磨材を供給しながら回転させ、鏡面になるまで磨く(ポリシング)。(11)化学薬品でウェーハを洗浄する。(12)必要に応じて、ウェーハの表面に気相エピタキシャル成長法によりシリコン結晶薄膜(エピタキシャル膜)を形成する。(13)平坦度、比抵抗、表面に付着している粒子の大きさと数などを検査する。(14)ウェーハをプラスチックの箱に収納して出荷する。以上の工程により現在生産されているシリコンウェーハの直径は、最大300mmである。
