# 8・1・2 単結晶育成装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・1 基板材料製造装置

多結晶から単結晶を育成する方法にはCZ 法とFZ 法がある.CZ法では図8・1に示すように,引上げ装置内に黒鉛ヒータ,黒鉛るつ図8・1 CZ(Czochralski)法による単結晶引上げ装置図8・2 縦型MCZ 法図8・3横型MCZ 法図8・4 カスプ型MCZ 法ぼ,石英るつぼを組上げ,るつぼの中に多結晶シリコンを入れて不活性雰囲気の減圧下で加熱して溶融し,融液に種結晶を付けて徐々に引上げることにより単結晶を育成する.CZ 法では,るつぼ(石英ガラス)から酸素が溶け出すことによってシリコン結晶中に微量の酸素が混入する.混入した酸素はさまざまな目的に活用されるものの欠点もあるため,混入量を適量に制御する必要がある.酸素の混入量はシリコン融液の流れの様子によって変わるので,磁場を利用してシリコン融液の流れを制御するMCZ(magnetic field applied Czochralski)法が開発された.MCZ 法には,電磁石を用いた磁場のかけ方(方向)により三つの方式(縦磁場,横磁場,カスプ磁場)がある.縦型MCZ 法は図8・2に示すようにシリコン融液表面と直交する方向に磁場をかける方法,横型MCZ 法は図8・3に示すようにシリコン融液表面と平行な方向に磁場をかける方法であり,カスプ型MCZ 法は図8・4に示すように上下2組の電磁石により互いに逆向きの磁場をかける方法である.FZ 法は図8・5に示すように誘導加熱で多結晶シリコン棒の一部を溶融して種結晶を付け,融解している領域を移動させる方法であるため,石英るつぼを使用しない.シリコン融液は多結晶シリコン棒と単結晶シリコン塊のすきまに表面張力で保持される.高温の領域が他の材質と接触しないため高純度であるが,結晶中の酸素濃度が小さいために機械的強度が弱いなどの欠点があり,用途が限定されている.
