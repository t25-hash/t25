# 8・1・3 単結晶の切断装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・1 基板材料製造装置

シリコン単結晶の切断には、外周刃、ワイヤソー、バンドソーなどが用いられてきたが、現在は内周刃(inner diameter saw)とワイヤソー(wire saw)が用いられている。内周刃とは、ドーナツ状の鋼板を基材として、円周の内側にダイアモンド粉を固着させた薄刃である。内周刃方式では、図8・6に示すように単結晶塊を内周刃の内側に置いて、刃を回しながら1枚ずつ切断する。刃は大きな張力で張られているので、切断時に力がかかっても刃のブレが小さい。内周刃方式は直径200mmまでのウェーハの製造でおもに用いられてきた方式であるが、切断する直径の4~5倍の外径のブレード材が必要となるため、300mm以上の結晶を切断することが困難になった。このため、内周刃に代わる方式として、ワイヤソーが用いられるようになってきている。ワイヤソー方式では、図8・7に示すように何本も平行して張った鋼線を刃として、研削砥粒の入った図8・5 FZ(floating zone)法図8・6 内周刃方式図8・7 ワイヤソー方式スラリーを供給しながら1本のブロックを一度に切断する。
