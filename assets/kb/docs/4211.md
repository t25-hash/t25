# 8・1・4 ウェーハ研磨装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・1 基板材料製造装置

ウェーハを研磨する工程にはラッピングとポリシング（鏡面研
磨）の2種類がある．ラッピングは，ウェーハの両面を機械研磨し
て厚さをそろえる工程であり，図8･8に示すように回転する上下の
ラップ盤の間にウェーハを挟み，そこに砥粒を供給してウェーハの
両面を削る．
ポリシングでは，図8･9に示すように研磨布を貼ってある定盤
に，ウェーハを接着したプレートを押付けて回転させ，そこに研磨
材を供給しながら化学的機械的研磨（CMP:chemical mechani-
cal polishingまたはmechanochemical polishing）により鏡面に
なるまで磨く．
