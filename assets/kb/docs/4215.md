# 8・3・3 不純物導入(doping)・拡散装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・3 半導体チップ製造装置

不純物の導入は半導体素子の電気的特性を決定する工程であり、素子の高性能化と微細化に伴って、その導入、活性化および拡散に高度な制御が要求されるようになってきた。最近では不純物の導入は、イオン注入装置によって不純物を注入し、その後の熱処理(アニール)工程によって不純物の拡散と電気的な活性化を行うのが主流である。1―200産業機械・装置図8・23 イオン注入装置a. イオン注入装置(ion implantation)イオン注入装置とは注入する元素をイオン化し、電場で加速して基板に注入する装置のことで、加速電圧により注入深さを精密に制御することができる。装置の構成は図8・23に示すように通常、イオンの発生部、引出部、分離部、加速部、レンズ、スキャン電極、イオン中性化部、基板処理室からなる。イオン発生室はプラズマによるイオン生成をガス原料から行い、出口外側にあるイオン引出電極によってイオンが取出され、予備加速される。次に磁場中で所望の元素のイオンのみが質量分離され、所定のエネルギーに加速され、磁気レンズでビーム制御された後、基板に照射される。スキャン電極によってビームを振る場合もある。またイオンにより基板が帯電し、素子にダメージを与えるため、処理室直前にイオンのニュートライザが設けられる場合もある。基板処理室には注入イオン量を計測するためにファラデカップを備えた装置が多い。注入するイオン種としてはp型(アクセプタ型)としてボロンが多用されるほか、ガリウムやインジウムなども用いられることがある。n型(ドナー型)としてはリンやヒ素等が多く用いられる。b. イオン注入の問題点と動向イオンにより、基板に原子を注入する(イオン打込み)と、打込んだ原子が無衝突で(電子との衝突はある)基板の結晶格子の空間を進むチャネリングといわれる現象が起こる。この原子は所望の打込み深さより深いところまで達するため、不純物の分布に問題が起こる。これを防ぐには基板をイオンの入射角度に対して若干(7°~10°)チルトさせることで対応する。しかしボロンのように小さい原子ではチャネリングはいまだに問題である。イオン注入深さはますます浅くなる方向にある。そのためイオンの打込みエネルギーを従来の20~200keV から2~10keV 程度に下げる必要がある。しかし、打込みエネルギーを低下させると、チャネリングに対する基板の限界チルト角度が大きくなる、あるいは基板損傷の低下によりチャネリングが起こりやすくなるなどの不利な点もある。その解決方法の一つとして、目的元素の注入前にあらかじめゲルマニウムやシリコンなどの元素を打込むことで基板をアモルファス化しておく方法がある。また、低エネルギーイオン打込みには、高速イオンを減速して低速イオンを作る方法があるが、電気的に中性化したイオンが減速されずに基板に到達するといわゆるエネルギー汚染が起こる。エネルギー汚染はこのほか導入したガス(例えばBFをBFとして注入する場合)が加速中に他の分子と衝突するなどして分解した場合にも起こりうるので、装置内の圧力はできるだけ、低圧に保つ必要がある。金属汚染も深刻な問題を引き起こすが、これは2種類ある。一つは質量分離部の問題で、例えばSiとFeは質量分離では区別がつかず、鉄が混入し金属汚染となる。もともとの原因は汚染物質イオンがビームに混入することであるが、ハロゲン化ソースの使用図8・24 その他の不純物導入装置不純物の導入には、拡散ソースを用いる方法がある。この方法は大がかりなイオン注入法に比べて、装置が単純で安価である利点があり、精度をそれほど必要としない不純物導入に使用される。代表的な装置は図8・24に示すようにヒータ、処理室、ガス供給系からなるバッチ式の炉で、多数枚の基板が一度に処理できる。原理は、不純物源として気相にBF,BHやPOClなどのガス状原料を供給し、次に基板の表面になんらかの方法で析出させ、それを基板中に固体拡散させるものである。表面への析出の方法としては、(1)基板表面とガスソースとの表面反応を利用するもの、(2)気相中でのガスソースの分解による表面析出を利用するもの、(3)プラズマによるガスソースの分解とイオン化による打込み効果を利用するもの、(4)CVD(chemical vapor deposition: 化学気相成膜法)等により不純物を高濃度に含む薄膜を堆積させるもの、などがある。基板をエキシマレーザ照射することにより、ガスソースの気相分解、表面析出、固相拡散を同時に行う方法もある。最近注目される応用例としては、不純物を高濃度に含むトランジスタゲートの酸化膜に高濃度B ドープし、そこからの固相拡散によって、浅い接合を作るものがある。d. 拡散装置導入された不純物は、イオン注入の場合にはそのままでは電気的な活性化率が低いため、アニール(anneal)して活性化する必要がある。アニールの過程で不純物は同時に固相中を拡散するために、拡散深さを浅く制御する必要のある素子では、いかに拡散の程度を制御するかが重要な問題である。一般的な拡散炉は基本的には図8・24に示すような不純物導入装置と同型式のもので、昇降温度を厳密に制御し、不活性ガス雰囲気中で行われる。最近ではさらに拡散時間を短くすることを目的として、高温で短時間のアニールを行う装置が多くなってきている。代表的なものは、図8・25に示すランプアニール(RTA
