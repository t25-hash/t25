# 2・5・2 変速機

> **章**: 第2章 自動車
> **節**: 2・5 二輪自動車

二輪車の変速機は，小型軽量化を図るためクラッチ室を含めてク
ランク室と一体構造となっている場合がほとんどで，クランク軸か
ら歯車を介して駆動される構造が一般的である．変速機は，左足で
変速レバーを踏み下げたり引き上げたりして行うことで，歯車列の
かみあいを一段ずつ順次に切換える構造となっている．四輪車のよ
うに中立位置を中心に任意の位置へ変速できる構造ではない．変速
段数は，商品特性から実用車では三段から五段，スポーツ車では五
段から六段変速の構造が多い．しかし，スクータの変速機は簡単な
操作が要求されるため，ベルト式の無段変速機を組込んだ構造が多
い（図2･183）．
クラッチは変速機軸に設けるのが一般的であるが，遠心式自動ク
ラッチを用いる場合と，車の進行方向と平行にクランク軸を配置し
た，いわゆる縦置きクランク軸方式では，クランク軸に直結の構造
も少なくない．クラッチ形式は，ノイズが少なく耐久性に優れた湿
式多板式が一般的である．しかし，競走用車両のクラッチ構造は，
軽量化などの要求から乾式多板式が使われている．一方，スクータ
等の遠心式自動クラッチにはシュー拡張型の乾式と，ディスク型の
湿式が用いられている．クラッチ操作は，ハンドルの左側に設けら
れたクラッチレバーを握って行う．
