# 2・5・3 フレーム

> **章**: 第2章 自動車
> **節**: 2・5 二輪自動車

フレームは，原動機，動力伝達装置をはじめ座席，荷物積載装置
などを保持するとともに，前後輪をステアリング，サスペンション
機構を介して連結する車両の基幹となる部品である．また，運転者
や荷物の荷重，走行中の路面からの衝撃力に耐えなければならない
強度部材であると同時に，操縦性・安定性の観点から，軽量で高剛
性であることが要求され，二輪車全体のスタイルを決める重要な要
素でもある．フレームの形態は，使用目的，原動機の形態・大きさ
によりさまざまな種類がある．
構造的には複数の部材で空間を構成するスペースフレーム構造
と，1本の主部材で構成されるはり構造のものがある．スペースフ
レーム構造にはクレードル型とダイヤモンド型がある．一方，はり
構造にはバックボーン型とステップスルー型がある．
a.クレードル型フレーム［cradle frame，図2･184(ａ)］ 原
動機全体を上下の部材で囲む揺りかご状の形状のフレームである．
原動機下部を抱き込む部材（ダウンチューブ）の形状によりシング
ルクレードル，セミダブルクレードル，ダブルクレードルに分類さ
れる．フレーム単体で比較的強度・剛性が高く，オフロード車や大
型車に用いられる．
b.ダイヤモンド型フレーム［diamond frame，図2･184(ｂ)］
ダウンチューブを省略しているのが特徴であり，原動機を強度・剛
性部材の一部として利用する形式のフレームである．フレーム下部
が省略できる分，軽量化が図れる利点があるが，反面，フレームに

c .バックボーン型フレーム（back bone frame) 原動機を抱
き込む部分がなく，ヘッドパイプから後部にかけ背骨状の部材で構
成するはり構造のフレームである．部材は主として鋼板プレス成形
にて製造されることが多い．
d.ステップスルー型フレーム［step through frame，図2･
184(ｄ)］ 主としてスクータやファミリー車に用いられる形式
で，乗降の容易さを目的としてヘッドパイプとシートの間に空間を
設けている．主部材は下部を通ることから，アンダボーン型とも呼
ばれる．
フレームを構成する部材は一般的にはスチールのパイプ材やプレ
ス材が用いられるが，最近では軽量化と高剛性の両立を目的として
アルミニウムの鋳造材と押出し材を組合わせた構造のフレームが多
く採用されている．主部材にバフ仕上げやアルマイト処理など施す
ことにより，高品位で美しい外観が可能で，デザイン要素としても
重要な位置付けになっている．
