# 8・3・4 薄膜形成装置(thin film deposition reactor)

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・3 半導体チップ製造装置

a. 概要半導体の素子構造はおもにシリコン基板上に立体的に形成された10層に及ぶ薄膜から構成されている。この素子構造を形成するために用いられるのが薄膜形成装置およびエッチング装置である。したがって、電子素子、特に、半導体製造工程においては薄膜形成が必須である。素子構造の中にトランジスタ、ダイオード、コンデンサ、抵抗などに相当する回路を作り込むため、用いられる薄膜も、半導体、金属、絶縁性などの各種の物質が用いられる。また、その形成方法も物理的方法、化学的方法など多岐にわたっている。表8・3に薄膜の種類と応用を示す。b. 目的薄膜形成装置の目的は半導体ウェーハ上に各種の材料の薄膜を形成させることにある。単結晶膜(エピタキシャル膜)成長、多結晶膜成長、酸化膜成長、金属膜成長、絶縁膜成長などが行われる。ウェーハの大口径化が進んでいるため、いずれの薄膜形成についても(1) 膜厚、膜組成の均一性(2) 大きなスループット(単位時間当たりの処理量)(3) 不純物、ダストなどの汚染が少ないことが要求される。薄膜の種類に応じて形成方法、装置が固有の形式で開発されている。c. 各種の方式薄膜の形成方法は大別してドライ方式(PVD :物理的気相堆積法、CVD : 化学的気相堆積法、など)とウェット方式(塗布法、電解めっき法など)がある。ここではドライ方式の代表例としてスパッタリング装置、CVD 装置、ウェット方式の例として電解めっき装置について説明する。. スパッタリング(sputtering)装置スパッタリング法は一般的な薄膜形成法として幅広い材料に用いられている。真空容器内でプラズマによりイオン化したArをターゲット電極表面に衝突させ、生成したスパッタ粒子を対向して配置したウェーハ上に堆積させる。スパッタ粒子はイオン化され、バイアスを印加したウェーハ側に引き込まれる。さらにウェーハ側に印加したバイアスにより形成膜は再スパッタされるため、その結果、微細孔の側壁と底部にも膜が形成される。近年はCu配線形成の前処理プロセスへの適用が図られている。配線にCuを用いることが進みその形成工程としてダマシンプロセス(絶縁膜に溝を掘りその溝内に電解めっきなどの方法でCuを堆積し配線を形成する方法)が用いられ始めた。スパッタリングはCuを堆積するに先立ってCu配線のシード形成、段差部へのTa,TaN 系バリア層の形成に用いられる。. CVD 装置(chemical vapor deposition reactor)目的元素を含有する原料(水素化物、塩化物、有機物など)の気体をウェーハに供給し気相または表面での化学反応を利用して目的物質を堆積させる。このとき用いる化学反応の励起手段として熱、プラズ1―201図8・26 バッチ式CVD 装置図8・27 枚葉式CVD 装置マなどが用いられ熱CVD、あるいはプラズマCVD と呼ばれる。適用材料としてバリアメタル、Cu,Al,W、層間絶縁膜などがある。金属膜形成には熱CVD 装置というように膜に応じて用いられる。装置形式としてはバッチ方式あるいは枚様方式が用いられる。装置図を図8・26、図8・27に示す。プラズマCVD では大面積に均一にプラズマを形成することが必要であるため枚葉式(singlewafer)がよく用いられる。原料ガスとして金属原料の有機物(有機金属)を用いる場合をMOCVD(metal organic chemical vapor deposition)と呼ぶ。青色レーザなどの材料であるGaN(チッ化ガリウム)の単結晶薄膜成長などで実用化されている。装置としては熱CVD 装置が多く用いられる。. 電解めっき(electroplating)装置電解めっき装置は上述のCuを堆積させる装置として多く用いられるようになった。ウェーハにはCu堆積する前に、あらかじめCuシード層が全面に形成されている。ウェーハは外周で接触するコンタクト電極からの給電によりカソード電極となっている。Cu析出の面内均一性を確保するため回転機構を有している。めっき液はめっき槽とタンクの間を循環し、噴流式にウェーハに接触する。アノード電極はウェーハに対向してめっき液中に設置されている。アノード電極はリンを0.04~0.08%含有した含リン銅アノードもしくは純銅アノードが用いられ、めっき液中に溶解して析出により消費されたCuを補給する。LSIに用いられるめっき装置はスループット確保のため複数のめっき槽とスピンリンス/ドライ槽を有している。また、滴定やCVS が組込まれた自動分析・薬液供給装置が接続されめっき液の液組成、添加剤濃度が自動で管理される。さらに、後工程でのクロスコンタミメーションやCuの自己拡散を防止するためウェーハの裏面洗浄、エッジ部のCu除去機能、めっき後のアニール処理までを同一装置内で行うことのできる装置も市販されている。
