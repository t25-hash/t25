# 8・4・1 グラインド装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

バックグラインドは、1テープ貼付け、2バックグラインド、3テープはがし、4純水洗浄/乾燥、5厚み測定の各工程からなる。近年用いられている先ダイシングの工程は、1ハーフカットダイシング(溝深さは最終チップ厚み+20 m程度)、2デバイス面保護テープマウントウェーハローディング、3オリフラ/ノッチ検出、4裏面研削(この最中にチップセパレーション)、5ウェーハ洗浄と乾燥、6転写装置へ搬送である。粗研削、仕上げ研削、ウェーハアンローディングが行えるようになっている。バックグラインドはウェーハの厚さの均一性が要求されるがほとんど装置およびテープの選定でその精度は決定される。テープ貼付け工程ではウェーハ表面を保護するために保護テープを張る。テープは基材100~200m、粘着剤1020 mからなる。バックグラインド装置は通常二軸の砥石からなる。粗研磨から仕上げ研磨を行い、ウェーハ裏面を鏡面仕上げする。一軸めに#360メッシュのダイヤモンド砥石を使い、約50ms程度の周速度で回転させながらウェーハに押付け仕上げ厚み+20 m程度のところまで削り込む。二軸めでは#2000メッシュの砥粒を含んだレジノイドボンド砥石で約58ms 程度の周速度で回転させながらウェーハに押付け最終仕上げ厚みまで削り込む。元厚さ700 m程度のウェーハを通常200 m程度に仕上げる。テレホンカード等ICカードに搭載されるLSIでは50 mまで薄くする技術が求められている。
