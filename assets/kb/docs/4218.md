# 2・5・5 ホイールおよびブレーキ

> **章**: 第2章 自動車
> **節**: 2・5 二輪自動車

ホイールは目的に応じて種々の形態のものが使われる．オンロー
ド車では，軽量・高剛性でかつデザイン自由度もあるアルミニウム
鋳造一体型のキャストホイール［図2･187(ａ)］が，オフロード車
では，しなやかな特性の得られるワイヤスポークホイール［図
図2･190ディスクブレーキ
図2･188前後連動ブレーキシステム
図2･189アンチロックブレーキシステム
2･187(ｂ)］がおもに使われる．また，スクータなどの小径のもの
は，廉価なプレス製のディスクホイールが使われる．
ブレーキは一般的には前後独立式で，右手で前ブレーキ，右足で
後ブレーキが操作される．しかし，最近ではより理想的な制動力を
得る目的で一部の大型車やスクータに前後連動ブレーキ（図
2･188）が採用されつつある．さらに，摩擦係数の低い路面でのホ
イールのロックを防止するために，アンチロックブレーキシステム
（図2･189）を採用する二輪車も増えている．ブレーキ形態として
は油圧式ディスクブレーキが主流で，高い制動力を要求されるスポ
ーツ車は前輪ブレーキに大径のディスクをホイールの両サイドに2
枚装着する場合が多い（図2･190）．ドラムブレーキは軽合金のホ
イールハブに鋳鉄製のスリーブを鋳込んだドラムに，機械式のブレ
ーキシューを組合わせたものが主流である．
