# 8・4・2 ダイシング装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

ダイシングはウェーハから個々のチップに分割する工程である.基本工程は,1ウェーハマウント,2ダイシング,3UV 照射,となっている.マウント,ダイシングの工程図を図8・30に示す.装置はウェーハの供給から収納までを全自動化しており,構成はアライメント部,ウェーハ切断部,洗浄・乾燥部からなる.供給されたウェーハはマウンタ工程によりキャリヤテープ上に貼付けられる.キャリヤテープは有機フィルム上に接着剤が施されている.キャリヤテープにはダイシング時には分割されたチップがテープからはく離しないよう強固な接着力が要求されるが次工程のダイボンド工程においては接着力が少ないことが要求される.過度の接着力が働くと,チップにマイクロクラックが発生したりチップをピックアップできなくなる恐れがあるためである.このためダイシング時は強い接着力を保持し,チップピックアップ時は接着力を小さくするようテープの接着剤に工夫が凝らされている.UV 照射後急速に粘着力の弱まるUV 硬化性樹脂を接着剤として用いるのが一般的である.ダイシング工程ではウェーハはダイサに装着されたブレードにより個々のチップに分割される.図8・31,図8・32にダイシング機構,ブレードの配置方法の概念図を示す.ブレードはダイヤモンド粒子が有機材料の円盤に埋込まれたものを用いる.厚さは2550 mが一般的である.このブレードが数万回転/分で回転し,250400 mのウェーハを100200mm/sのスピードでカットする.ダイシング工程では発生する残渣および熱を除去するためカット部分に超純水を高圧噴射する.本来この超純図8・30 ダイシング工程図図8・31 ダイシング機構図8・32 ブレード配置方法1―203水は比抵抗が非常に大きくウェーハ上のパッシベーション膜と高速で接する際,多少なりとも静電気を発生する.この発生する静電気量が大きいと静電気により形成された回路が破壊されてしまう場合がある.このような静電気による回路の損傷を防ぐため,超純水の比抵抗値の適正化が,高圧噴射の圧力の適正化とともに行われる.
