# 8・4・3 ダイボンド装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

ダイボンド工程ではカットされたチップがAg ペーストなどのダイボンド剤を用いてリードフレームのダイパッドに固定される.工程は図8・33に示すとおりである.ボンディングステージ上でAg ペーストをリードフレームのダイパッドに塗布する.ピックアップステージにてダイシング工程でキャリヤテープ上に個々に分割されたチップをキャリヤテープ下面からニードルで突き上げる.テープから分離されたチップは直上で待機しているバキューム穴をもつピックアップツール(コレット)に吸着される.ピックアップされたチップはボンディングステージに移動しAg ペーストの塗布されたダイパッド上にボンディングされる.Ag ペーストを硬化させチップをダイパッド上に固着させる.図8・34にダイパッド上でのボンディング工程の模式図を示す.最近三次元実装(ダイの上にダイを重ねる)が進むに伴いダイの薄型化がますます進められている.そのためダイボンディング装置としてはダイシングテープからのピックアップを中心に工夫を凝らされている.具体的には上からコレットで吸い付けるタイミングと下からニードルで突き上げるタイミングを完全同期化する(どちらか一方が過剰な場合ダイに荷重がかかり結果的に破損する),ボンディングヘッドを軽量化する,などである.
