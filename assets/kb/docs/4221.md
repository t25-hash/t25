# 8・4・4 ワイヤボンディング装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

パッケージングの中で最も高密度実装を実現する基幹技術がチップとパッケージとの接続技術である。ワイヤボンディング工程は接続技術の中でも最も安定した技術である一方、最も技術開発の激しい分野である。模式図を図8・35に示す。金ボールを形成した金線がキャピラリーに保持され、ボンディングパッド上に待機する[図3・35(a)].ワイヤボンディング工程はまず、待機した金線がファーストボンディングであるAlパッド上に適切な荷重をもって接触する。金ボールが接触した後キャピラリーが超音波振動し、このときの荷重や超音波によるエネルギーで金ボールとAlパッドとの間に接合を得る[図8・35(b)].次にキャ図8・33 ダイボンド工程図図8・34 ボンディング工程図1―204産業機械・装置(a)(b)(c)(d)(e)(f)(g)図8・35 ワイヤボンディング工程図図8・36 キャピラリーの形状ピラリーが適切な軌跡を描きながらセカンドボンディングポストに移動する[図8・35(c)].この図8・35(c)から図8・35(e)に至る軌跡によってパッケージに適切な金線ループが得られる。セカンドボンディングにおいても適切な荷重と超音波が加えられる[図8・35(e)].次に、キャピラリー、クランパが上昇する過程でクランプが金線を保持し、金線をセカンドボンディング点から切断する。続いて金線先端に電気トーチが移動し、金線先端と放電を行う[図8・35(f)].この放電によって金ボールが形成され[図8・35(g)]1サイクルが完了する。この1サイクルが0.10.2msで行われる。装置の基本構成はチップの供給搬送部、位置決めテーブル、パターン認識部、ボンディングヘッドである。回路の高密度化に対応して多ピン化(ボンディングパッドピッチのファインピッチ化)が進んでいる。ファインピッチ化に対応するため、ボールボンダのイニシャルボールを小さく安定して作ることや安定したボンドの形成に対応して、軸の制御性の向上、ボンディングヘッドの軽量化、耐久性の向上、画像認識ソフトの最適化などが重要である。装置スループットの目安としてボンディング速度が重要で0.07秒/ワイヤにまで高速化されている。位置決め精度については10.1m/ステップが多い。画像認識においてはグレースケール方式を採用し、画素補間機能を装備したものが採用されている。ボンディングヘッドキャピラリーの改良に関しては、図8・36に示すとおり、ボトルネックキャピラリーが採用されている。材質としてセラミック、ルビーがあるが、耐久性向上などの工夫が行われている。ボール接合性の向上に関しては超音波の高周波化(100kHz~138kHz)が図られている。金ボールとチップのAlパッドが接した時点から荷重がかけられる。荷重により金ボールはキャピラリー先端の形状に沿って変形され、この工程で金ボールとAlパッドの間のずれが発生しAlパッド上の薄い酸化膜が破壊されAl面が現れる。次に荷重と超音波のエネルギーによりAu-Al界面にAu-Al接合層を形成する。
