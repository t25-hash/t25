# 8・4・5 モールド装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

モールド(封入工程)は樹脂によりチップ,金線,フレームを封図8・37 モールド工程図(a) シングルプランジャ方式(b) マルチプランジャ方式図8・38 プランジャ形状止する工程である.図8・37に工程を示す.ワイヤボンドを完了したリードフレームが160~180°Cに加熱したモールド金型に装てんされる.ミニタブレット化されたモールド樹脂が樹脂ポットにセットされ金型が閉められる.プランジャにより金型のキャビティー内に樹脂を充てん,圧力をかけながら樹脂を硬化させる.樹脂硬化完了後金型の型締めを開放しモールド成型されたリードフレームを取出す.金型にすきまなく樹脂を封入することを目的とした装置であるため成型条件の適正化には金型形状の最適化とモールド樹脂の粘度特性適正化が重要である.この問題を解決するため,装置的には従来のシングルプランジャ金型方式に替わってマルチプランジャ金型が採用されている.図8・38に示す.マルチプランジャ方式はモールド樹脂ポットとキャビティー間の距離を最小限かつ等距離に保つよう設計できるため,低速で,各キャビティーとも均等に樹脂の充てんができる.成型条件の許容範囲が格段に向上し樹脂開発の制限も軽減した.樹脂開発に伴う環境問題への配慮も必要である.
