# 8・5・1 ウェーハ洗浄装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・5 洗浄装置

a. 目的と種類Siウェーハ製造ラインや半導体デバイス製造ラインにおけるウェーハ洗浄の目的は,ウェーハの表裏面を清浄に保つことである.デバイスパターン形成を阻害し,デバイス特性を劣化させる要因としては,粒子状物質,金属不純物や有機物質といったものが挙げられる.現在,Siウェーハ製造ラインや半導体デバイス製造ラインにおける洗浄プロセスは1970年にカーンらにより発表されたRCA 洗浄(RCA cleaning)をベースに洗浄シーケンスが組上げられている.RCA 洗浄の原理を図8・39に示す.その後,LSIの微細化,高集積化が急激に進み,プロセスは複雑化し,工程数も増え続けている.一方で,パーティクルや金属不純物だけでなく,表面吸着分子状物質などの微小かつ微量な汚染物質が,LSIの歩留りや信頼性に影響を及ぼすようになってきた.さらに,デバイス特性を向上させるために電極材料,シリサイド材料,配線材料等の材料選択が多岐にわたり,洗浄対象となる元素が広がってきている.過去には金属汚染の洗浄効率は,薬品の純度や装置を構成する部材の清浄度によって左右されていた.装置の清浄化が進み,高純度の薬品が得られるようになってから,製造ラインで加工されたウェーハ自体によってもち込まれる汚染量によって洗浄効率が支配されるようになり,洗浄プロセス/装置が多様化してきている.また,300mmφウェーハの採用により洗浄装置が大型化し,洗浄プロセスにおける薬品,純水使用量が増加の傾向にあり,省スペース,省エネルギー,省コスト,さらには地球環境への負担低減の観点から洗浄プロセス/装置の見直しが迫られている.ウェーハ洗浄装置としては,薬液を使用したウェット洗浄のみならず,ガスを用いたドライ洗浄もある.Arと窒素の混合ガスを冷却し,固体粒子としてウェーハ表面上の異物に衝突させ,物理的な作用により,異物をウェーハ表面上から除去する技術である.本プロセスはエアロゾルによる物理洗浄であり,不活性ガスを利用しているために,化学反応は全く伴わない.耐薬品性が問題となりやすい,材料に対して有効となる.しかしながら,一般的にウェット洗浄に比してドライ洗浄では,ウェーハ表面に付着した種々の汚染を一挙に除去することが困難であるために,製造レベルではウェット洗浄が今後も主流であろう.ウェット洗浄装置は25枚から100枚のウェーハを一括で処理するバッチ式洗浄装置とウェーハごとに処理する枚葉洗浄装置に大別される.さらにバッチ式洗浄装置としては,薬液槽,水洗槽および図8・39 RCA 洗浄の原理図8・40 バッチ式洗浄装置1―205乾燥機能を備えた多槽式洗浄装置,洗浄,水洗を同一槽で行う単槽式洗浄装置や薬液,純水をスプレーノズルから供給するスプレー式洗浄装置がある.b. バッチ式洗浄装置浸漬方式のバッチ式洗浄装置の例を図に各薬液処理に応じて水洗槽を備えており,薬液処理が増えるに従って,水洗槽も増加するのが一般的である.例えば,SPM(硫酸と過酸化水素水の混合液),DHF(希ふっ酸),APM(水酸化アンモニウムと過酸化水素水の混合液),HPM(塩酸と過酸化水素水の混合液)処理を行う場合には,最低4種類の薬液処理槽と水洗槽と乾燥装置から構成される.実際には,生産性を考慮して複数の同一薬液槽を備えたり,水洗槽を追加したりしている.薬液槽は一定期間(回数),一定濃度,一定温度で薬液を循環しており,必要な処理レシピに応じた専用装置となる.したがって,異なる洗浄レシピ(異なる薬液濃度や処理温度)が必要となる場合には,それぞれのレシピに応じた装置が必要となる.薬液は通常何度も循環して繰返し用いていることから,つねに同じ洗浄効果を得るために槽内洗浄液の濃度,温度のコントロールのみならず,洗浄液の清浄度の管理が重要となる.何槽もの処理槽からなる巨大な装置となり,スループットは大きいが,クリーンルーム面積の占有率の増大,純水/薬液使用量の増大,フレッシュエアーの取込み量の増大など,省エネルギー,省スペース,省コストの観点から課題も多い.このような課題に対して,プロセス面での対策としては洗浄シーケンスの簡略化が図られている.装置面では槽数,槽容量を減らすことである.槽容量削減に対する具体策として,処理時のウェーハ間隔の半減(ハーフピッチ)化,ウェーハの直接移載によるキャリアレス化が試みられている.これらの対策により装置を小型化でき,薬液/純水の消費量を削減することができるだけでなく,キャリアから処理槽への汚染物もち込み防止,次の処理槽への薬液もち出し量の削減など,多くの利点がある.洗浄プロセスではウェーハから汚染物を除去するだけでなく,再付着を防止することが重要である.浸漬方式の洗浄装置の欠点として,ウェーハ裏面に付着したパーティクルが汚染源となり,ウェーハ表面への再付着をもたらすことが知られている.このために処理槽内の流れの改善によるパーティクルの滞留防止,ウェーハの表面対抗配置による裏面パーティクルの影響低減策などを施している.バッチスプレー式洗浄装置は同一チャンバ内で薬液処理,純水リンス,乾燥を行うことが可能なため,多槽式洗浄装置に比して装置は非常にコンパクトであり,少ないフレッシュエアーの取込み量で,クリーンルーム環境への薬品ミストの暴露を防止できる.また,処理レシピも自由に組替えることができるため,多槽式に比してフレキシビリティが高い.しかしながら,同一チャンバ内で薬液処理・水洗・乾燥を行うために薬液の循環システムが複雑となり,薬液の回収,濃度コントロールに関しては,同一薬液を循環して用いる多槽式に比して不利である.多槽式洗浄装置,スプレー式洗浄装置に共通した問題として,パーティクルのコントロールが挙げられる.多槽式の場合にはウェーハが処理槽間を移動する場合に,ウェーハが気液界面を何度も横切ることによって,液界面に浮遊するパーティクルが付着しやすい.この現象はHF処理などによりウェーハ表面を疎水性にする場合に特に顕著に現れる.特にスプレー方式の場合には,薬液,純水の供給自体が多くの気液界面を形成するために,疎水性表面形成には注意を要する.そこで,ウェーハが大気に接触しないように薬液処理に引き続き,同一処理槽内で純水にてオーバフローすることにより,連続的に薬液を直接置換処理するワンバス(単槽)方式がある.ワンバス方式は1回ごとの洗浄液の使い捨てにより,パーティクルの制御性がよいことに加えて,装置面積を縮小できる利点を有する.リンス工程において処理槽に残留した薬液を置換するために多量の純水が必要であることが最も大きな問題であり,先に述べた処理槽容量を削減するなどの装置上の改良が必須である.また,装1―206産業機械・装置置構成上,洗浄液は毎回使い捨てになるために,コスト面で高濃度薬液の洗浄には向かず,希薄溶液での洗浄性能向上が課題として挙げられ,オゾン水や水素水といった機能水などの新洗浄薬液の採用が期待される.c. 枚葉式洗浄装置枚葉式スピン洗浄装置は1枚ごと,表面と裏面をそれぞれつねにフレッシュな洗浄液を供給し処理するために,裏面汚染物のウェーハ表面転写を受けずに済む方法として期待されている.また,洗浄槽をもたないために薬液の置換効率が良く,純水使用量の削減が達成される.応用例として最も多いのは,裏面洗浄やCMPの後処理としてのブラシスクラバであり,メガソニックや薬液洗浄モジュールとの組合せにより多種の形態が見られる.一方でバッチ処理である拡散工程などの前処理としての展開は少ない.この理由として,枚葉洗浄装置はバッチ式洗浄装置に比して低スループットであることに加えて,1枚めのウェーハを処理してから最終ウェーハを処理するまでに時間差があるために,その間に1枚めのウェーハをいかにして清浄に保つかが課題として挙げられる.最近では,配線工程がRIEからダマシンプロセスへ変化してきており,プロセスや使用する材料の変更に伴い,エッチング残渣処理や,ベベルエッチングといった新しい用途が開けてきている.また,今後の大口径化,少量多品種生産におけるQTAT 化の対応策としての適用が進むであろう.d. 乾燥装置ウェット洗浄工程の仕上げとして必ず乾燥工程が必要となる.洗浄な表面を維持するために乾燥工程は非常に重要である.一般的な乾燥方法として,スピン乾燥,IPA 蒸気乾燥,窒素ブロー,熱風乾燥,ランプ乾燥,真空乾燥,温水引き上げなどが知られている.バッチ式洗浄装置のウェーハ乾燥にはスピン乾燥が広く用いられてきたが,回転部分からの発塵,ウェーハのチッピング,高速回転によるウェーハの帯電およびパーティクル,ミストの吸着,ウォータマークの発生などの問題を抱えている.図8・41に乾燥時の汚染として最も問題となるウォータマークの発生メカニズムを示す.これらの種々の課題に対して,IPA を用いた蒸気乾燥方法が用いられてきたが,IPA の消費量が多く,IPA 自体の清浄度や安全管理に多くの配慮が必要である.また,ウォータマーク発生に関してはリンス終了,純水引き上げから乾燥までのウェーハ搬送時間の短縮化が問題となる.最近では,マランゴニ現象を利用した乾燥方法が主流となってきている.IPA を供給するための種々の方法が装置化されており,窒素希釈のIPA の供給により,蒸気乾燥方法に比して,IPA の使用量が少なく,純水から引き上げながら乾燥を終了することができる.しかしながら,同一処理槽を用いて酸化膜エッチング後にIPA 乾燥した場合に,ウェーハ表面にパーティクル状の異物が付着するなどの問題も報告されており,さらなる改良が必要である.その他の熱風やランプ加熱などを用いた乾燥方法では,清浄な環境での乾燥を行うことはできても,ウォータマークの発生に関して問題解決されていないのが現状である.製造ラインにおいて,ウォータマーク対策としてウェーハ乾燥はスピン乾燥からIPA 乾燥に主流は移っているが,ウェーハ表面の有機汚染や地球環境保全の観点から,IPA を用いない減圧,不活性雰囲気下でのスピン乾燥も報告されている.
