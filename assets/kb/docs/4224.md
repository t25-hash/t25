# 8・5・2 クリーンルーム(cleanroom)

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・40 に示す。多槽式洗浄装置では、数種類の薬液処理を行う場合

a. クリーンルームの定義クリーンルームとは、そこで行われる作業に対し有害な汚染物質量を制御した限られた空間であり、精密工業製品の生産工場、製薬・食品工場、病院、科学工学研究施設などで、汚染を排除すべき作業を行う場合に設けられる。汚染には、空気中に浮遊するもの、室内で使用する部材に含まれるものなどさまざまな種類がある。クリーンルーム内では、空気中浮遊微粒子などの汚染物質が特定の清浄度レベル以下に管理され、そこに供給される材料、ガス、水などについても要求される清浄度が保持されていなければならない。また、必要に応じて温度、湿度、圧力、振動などのレベルも管理されていることが要求される。b. クリーンルームの用途別分類クリーンルームの構造や要求性能は、その用途・目的により異なるため、大きく次のように分類されている。. インダストリアルクリーンルーム(industrial clean-room)半導体集積回路、フラットパネルディスプレイ等の電子部品や精密機器などの工業製品の製造工程として用いられるクリーンルームであり、おもに空気中浮遊微粒子濃度が既定値以下となるよう制御される。半導体集積回路等の製造工程として用いられるクリーンルームでは、空気中のガス状汚染物質(AMCs:airbornemolecular contaminants)に関しても厳しい管理が要求される。その他、対象とする製造工程の必要に応じて温度、湿度、静電気、微振動等の環境条件が制御されている。. バイオロジカルクリーンルーム(biological cleanroom)微生物による汚染が積極的に管理されたクリーンルームで、必要に応じて温湿度などの環境が制御される。手術室、特殊治療室、無菌動物飼育室、製薬工場、食品工場などに用いられている。. バイオハザード安全施設(biohazard safety room)危険な細菌を扱ったり、遺伝子組替え実験などを行うための、危険物質を外部に出さないことを主目的とした施設。扱う微生物などの危険度により、P1,P2,P3,P4の4レベルに分類され、特に最も危険度の高いP4施設には専用の給排気装置、完全な隔離域をもつ安全キャビネットの設置など高度な封じ込めが要求される。各レベルの規定は「内閣総理大臣決定組替えDNA 実験指針」に定められている。バイオハザード安全施設は、狭義の意味でクリーンルームではないが、高度な給排気処理が必要な特殊施設であるためクリーンルームの一種として分類される。c. クリーンルームの構成クリーンルームは室内に清浄環境を維持するための施設である。このため、クリーンルームの内装材・仕上げ材には、耐摩耗性が高く、微粒子やオフガスなどの汚染物質を発生しない材料を用いて、極力完全な密閉空間を構成する。浮遊微粒子は静電気によって表面沈着しやすくなるため、床材等には導電性を付与したものを用いる。また、室内には高性能エアフィルタでろ過した清浄空気を大型の送風機により循環させ、室内を陽圧として汚染の侵入を防止するとともに、室内で発生した汚染物質を速やかに除去させる(ただしバイオハザード安全施設では危険物質を外部に漏えいしないことを目的とするため、室内を負圧に維持する)。クリーンルーム内では、温度、湿度等の環境条件も一般室よりも厳密に制御されることが多く、空調設備も大型のものが用いられる。d. クリーンルームの方式別分類クリーンルームの方式は室内気流の送風方式により分類されることが多い。目的に応じて以下のようなさまざまなタイプがある。. 一方向流型クリーンルーム(unidirectional cleanroom)室内全域に一様な垂直下降流または水平流を形成する方式のクリーンルーム。室内によどみを形成しにくいため、高い清浄度を得ることができる。半導体集積回路やフラットパネルディスプレイの製造工程では、天井全面をエアフィルタ吹出し口、床全面をグレーチングとした垂直下降流方式が多く用いられている。垂直下降流方式の図8・41 基板乾燥時の汚染(water mark)一方向流型クリーンルームには、送風方式、レイアウトの違いにより以下のようなものがある。(1) ダクト方式図8・42に示すように、クリーンルーム上部にサプライチャンバ、床下にレターンチャンバを設け、集中型の空調機、送風機とダクトで連結して循環送風経路を形成する方式。サプライチャンバ内はクリーンルームに対し陽圧となるためフィルタとその取付け枠との気密性は特に重要で、液状シール材などを用いた特殊なシール方法のシステム天井が採用される。(2) FFU (fan filter unit)方式図8・43に示すように、吹出し口にFFU(小型の送風機とエアフィルタをユニット化したもの)を使用して送風を行う方式。FFU 方式では、サプライチャンバ内が負圧となるため、天井面からの汚染の漏えいが少なく、各FFU の風量制御により吹出し風速の均一性を調整できるなどの利点がある。送風動力の供給に集中型送風機を用いないため、循環送風経路は必ずしもダクトで構成する必要はない。(3) ベイ方式半導体製造工程などに多く用いられる、図に並べた平面レイアウトをもつ垂直下降流方式のクリーンルーム。高清浄度領域を作業域に限定することで運転コストの削減を図っている。また(2)のFFU 方式と組合わせ、レターンエアの循環経路としてユーティリティゾーンを利用すると、さらにスペース効率の向上を図ることができる。半導体製造装置の多くは、図示したようにクリーンルーム領域とユーティリティ領域との間の壁を貫く形で配置されることが多いため、スルー・ザ・ウォール方式とも呼ばれる。このほか、クリーンルーム領域の幅をさらに縮小したクラスタ・ベイ方式などがある。図8・42ダクト方式一方向流型クリーンルーム図8・43 非一方向流型クリーンルーム(non-unidirectionalcleanroom)図8・45のように吹出し口、吸込み口を分散的に配置した構造をもつクリーンルーム。高性能エアフィルタでろ過した空気を循環することで、室内で発生した汚染物質を換気効果により除去する方式である。一方向流型クリーンルームに比べて設備コスト、運転コストを低く抑えることができるが、後述する清浄度クラス5以上の高清浄度を必要とする用途には向いていない。バイオロジカルクリーンルームや比較的要求清浄度の低い生産工程などに用いられている。. 混在流型クリーンルームクリーンルーム内のゾーンごとの用途に応じて一方向流型と非一方向流型とを混在させた方式のクリーンルーム。高清浄度が必要な領域を一方向流型、通路などの補助的な領域を非一方向流型として、クリーンルーム全体の運転コスト削減を図ることを目的としている。高清浄度領域の構成方法によりアイランド型、クリーントンネル型、クリーンチューブ型などさまざまなバリエーションがある。. ミニエンバイロメント型クリーンルーム(mini-environ-ment ユーティリティ設備クリーンルームでは、空気環境のみならず作業に使用される水、ガス、薬液なども清浄度の高いものでなければならない。また周囲の環境に考慮して排水、排ガスを設ける必要がある。このため、クリーンルームに付随してさまざまなユーティリティ設備が設置される。以下に半導体工場を例にユーティリティ設備の概要を示す。. 超純水設備超純水は洗浄用として全工程を通じて使用されるため、その水質は製品歩留りに大きく影響する。半導体用超純水水質の推移を表8・10に示す。超純水製造設備は、凝集、沈殿、ろ過により原水中の懸濁物質やコロイドを除去する前処理システム、イオン交換装置と逆浸透膜装置によりアニオン、カチオンを除去する一次純水システム、紫外線酸化装置とカードポリシャでTOC(total oxide carbon)濃度を下げ、比抵抗18.2MΩ以上の超純水を製造するサブシステムなどから構成される。製造した超純水は、端末配管システムでユースポイントに供給される。端末配管材には、使用条件によりCPVC,PVDF,PEEK などの材質が使い分けられている。ユースポイントでは、配管内に供給された超純水が滞留して水質劣化することの図8・49エアシャワー表8・10 ガス供給設備半導体工場で用いられるガスは、パージガス、キャリアガス等の用途に用いられるN,O,Hなどのバルクガス、成膜、エッチングなどの反応ガスとして用いられるシラン、ホスフィン、アルシンなどの半導体材料ガス、装置可動部の制御用に用いられるドライエアに大別できる。バルクガス中の不純物濃度に対する要求性能の推移は表8・11に示すとおりである。パージガスは大量に使用されるため、半導体工場では窒素製造装置、水素発生装置などのガスヤードをもつ場合が多い。ガスヤードで製造されたパージガスは、それぞれのガス種別に一次フィルタでろ過され、機械室内の精製器と二次フィルタを介して端末配管に供給される。半導体材料ガスはボンベで供給されるので、ユースポイントに隣接したボンベボックスから装置へ配管接続される。半導体材料ガスの多くは強い毒性や爆発性をもつため、ボンベボックスの周辺には、散水設備、ガス検知機、温度センサ、排気ダクトなどの防災設備の設置が不可欠である。ガス供給配管材には、一般にステンレス管(SUS 316L)が用いられ、必要に応じて内表面粗度の小さいBA 管、EP管などが使用される。. 薬液供給設備有機溶剤や酸、アルカリなどの薬液は洗浄工程、エッチング工程、フォトリソ工程、CMP工程などのウェット工程で使用される。薬液に対する要求品質の推移を表8・12に示す。各種薬液はタンクから自動供給されるが、有機溶剤系はステンレスタンクからステンレス配管で、また無機系薬液はフッ素樹脂系タンクからフッ素樹脂配管の外側にPVC配管をもつ二重配管でユースポイントに供給される。.
