# 3・1・5 燃焼器（combustor ）

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・1 ガスタービン全般

a.要求性能 ガスタービン燃焼器に要求される性能として
は，高い燃焼効率（combustion eﬃciency），広い火炎安定範囲，
確実な点火，圧力変動や燃焼に起因する不具合のないこと，小さい
圧力損失，タービン翼の長寿命化に適した出口ガス温度分布，スモ
ーク・未燃焼成分・NO等の大気汚染物質の排出が少ないこと，
低コストで維持費の少ないこと，エンジンに適した大きさと形状，
耐久性に優れることが挙げられる．航空用ガスタービンでは加え
て，高空再着火性が必須である．また，常用・非常用兼用の産業用
ガスタービンでは，気体燃料を常用するが非常時には液体燃料に切
換えるなど，多種燃料に対する適応性の高いことが求められること
もある．
b.構造と形態 ガスタービンの代表であるジェットエンジン
の燃焼器の構造を図3･15に例示する．主要部品は，圧縮機出口と
タービン入口との間の空気通路を形成するディフューザ（diﬀuser ）
とケーシング（casing），環状の燃焼空間を形成する内外燃焼器ラ
イナ（liner ），燃料ノズル（fuel nozzle，燃料噴射弁ともいう)，
保炎器（flame holder ），点火栓（ignition plug）である．この形
態の燃焼器は環状燃焼器（annular combustor ）と呼ばれる．他の
形態には図3･16 に例示する，筒状燃焼器（tubular combustor ）
ライナを筒状ケーシング内に納めた筒状燃焼器，単一の環状ケーシ
ング内に複数個の筒状燃焼器ライナを周方向に配列したキャニュラ
燃焼器（cannular combustor ）がある．なお，筒状燃焼器ライナ
はフレームチューブ，筒状燃焼器は缶型燃焼器（can type combus-
tor ）とも呼ばれる．筒状燃焼器ライナは，環状燃焼器ライナに比
べて機械的強度が大きい，試験に必要な空気量が小さくて済むなど
開発がやりやすいという利点がある．短所としては，体表面積比が
小さいために冷却空気を多く要し，圧力損失が大きく，重くなるこ
とが挙げられる．多缶の場合には燃焼器ライナ間の火炎伝ぱ管やタ
ービンとの接続に尾筒，あるいはスクロールが必要となる．
図3･15，3･16に例示した燃焼器では，ケーシングと燃焼器ライ
ナとによって形成される環状通路の空気流と燃焼器ライナ内の燃焼
ガス流とが同方向である．このような燃焼器ライナの配置を順流型
といい，逆方向のものを逆流型という．
航空用ガスタービンにはエンジンの前面面積が小さい環状燃焼器
が使用される．燃焼器ライナの配置は，大型ファンエンジンは例外
なく順流型，ターボシャフト用ガスタービンはほとんどが逆流型で
ある．産業用ガスタービンでは，保守性の点から筒状燃焼器の逆流
型配置が採用される．大型では同一の円筒燃焼器を10本内外配置
した多缶燃焼器が用いられることが多いが，特殊なものとして大型
のサイロ燃焼器を1個備えたものがある．
c .ディフューザ 燃焼器での圧力損失を許容範囲に抑えるた
めには，圧縮機からの空気流を低損失で減速しておくことが必要で
ある．ディフューザは，この作用に加えライナに対してスムーズで
安定な空気流れを与え，空気量配分を決めるという役目ももつ．従
来からの動圧の回復を重視した空気力学的ディフューザに代わっ
て，出口に急拡大部を設けたダンプディフューザが用いられる傾向
せることが良好な微粒化に有効である．
にある．後者は，圧力損失はやや大きいが，圧縮機出口の速度プロ
ファイルの影響を受けにくく，エンジンの軽量化にもつながる点が
評価されている．この急拡大部において抽気を行い，渦により流れ
をいっそう安定化する方式が，作動温度の高温化によりタービンの
所要冷却空気量が増大する状況にあって広まっている．

置，タービンノズルにつながる流路形状などは適切な温度パターン

方，周方向には一様なことが，タービン翼に周期的な熱負荷を与え
ないことから望ましい．
e .燃料噴射 航空用ガスタービンには各種のジェット燃料が
使用される．産業用ガスタービンには天然ガス，灯油，A重油の
ほか，高炉ガスや汚泥分解処理などで発生する消化ガスなどの利用
も進んでいる．これらの気体燃料の単位体積当たりの発熱量は幅が
非常に広い．
わゆる多孔ノズルが用いられる．液体燃料の場合は，かつては燃焼
室内に延伸した管内に燃料を吐出させて壁面からの熱により蒸発さ
せ，過濃混合気として燃焼室内に噴出させる蒸発管方式が用いられ
たことがあるが，現在では下記の微粒化ノズル方式により噴霧とし
て供給する方式が用いられている．
17(ａ)］ 液体燃料の微粒化手段として最も広く使用されてい
る．燃料は渦室に流入することで旋回が与えられ，噴射圧がほぼ
0.4MPaを超えるとオリフィスから中空円すい状の液膜となって
流出する．この液膜はオリフィスから離れるに従って薄くなり，分
裂して液糸となり，最後に微細な液滴となる．これが噴霧である．
基本は渦室，オリフィスがそれぞれ1個のシンプレックスノズルで
ある．燃料流量のターンダウン比を大きくするために大小2個の渦
室とオリフィスを同軸に配置したのがデュプレックス（デュアルオ
リフィス）ノズルである．また，渦室の背面の中心に燃料を逃がす
通路を設けることで，この課題を解決したのがスピル-リターンノ
ズルである．逃がし量を増やせば，噴射流量が小さい場合でも十分
強い旋回が渦室に形成され，微粒化の悪化を防止できる．
ジェットエンジンの作動圧力の上昇が始まったころから離陸時のス

g.点火装置 混合気の点火には，高エネルギー型点火装置が
用いられる．この装置に使用される点火栓は表面放電型と呼ばれ，
中心電極と外周電極とはセラミックス製円筒で絶縁されており，そ
の円筒の先端には両電極をつなぐ半導体の薄膜が形成されている．
この半導体は電極間のイオン化を容易にする働きがあり，放電はこ
の面に沿って起きる．小型ガスタービン用には蓄積エネルギー2J ，
繰返し放電数毎秒4回，大型エンジン用には4～12Jで毎秒1回が
一般的である．点火栓位置は確実な点火と点火栓の寿命とを考慮し
て選定される．通常は，一次領域において燃料噴霧の外縁で，高温
のスパーク核が再循環により上流に運ばれる，600℃以上にならな
い場所が選ばれる．火花放電では点火の難しい低質油燃料の場合に
は，グロープラグあるいは軽質燃料やガスによる火炎でのトーチ点
火が採用される．
