# 9・2・1 文字製版

> **章**: 第9章 印刷機械
> **節**: 9・2 製版

印刷文字の製版(plate making)は,活字組版から写真植字方式に,そして,コンピュータを利用した組版に移行し,最近ではDTP(desk top publishing)と呼ばれ,電算写植システムなどのように大がかりなシステムでなく,机上でパーソナルコンピュータを使い,容易に組版・編集できる簡易システムが多用されている.a. 活字組版(type-setting)活字は鉛,アンチモン,スズからなる活字地金を溶かして母型に流し込んで作られる.活字ケースに並べられた活字を拾い集める手作業の文選と,自動的に機械で鋳造する方式があったが,現在では,用いられていない.b. 写真植字(photo type setting)レンズと光源により,文字盤から写真的に印画紙に印字するのが,写真植字である.レンズの交換により,文字のサイズが自由に選択できるほか,変形レンズを使うことにより,長体,平体,斜体などの変形文字の印字も容易にできる.この方式も次の電算写植に移行している.c. 電算写植(computer type setting)電算写植はコンピュータを使った写真植字システムで,漢字入力機,コンピュータ,全自動写植装置から構成される.漢字入力機のキー操作により入力し,字間,行間の計算,編集処理をコンピュータで行う.ディジタル化された字形を高解像度CRT 上に出力するとともに,同じように光学的に印画紙やフィルムに出力するCRT 方式が活用されている.さらに最近では,CRT 方式での出力にレーザ光が用いられ,高速化や高精細化が計られている.
