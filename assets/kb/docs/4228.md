# 9・2・2 写真製版と色分解

> **章**: 第9章 印刷機械
> **節**: 9・2 製版

印刷原稿には文字のほか,線画,白黒写真,カラー写真,絵画などがある.この文字以外の原稿部分の製版用原版フィルムを作成する工程を写真製版(photomechanical process)という.例えば,カラー写真や絵画のカラー印刷を行うためには,原稿から,印刷の3原色である藍(シアン,cyan),紅(マゼンタ,magenta),黄(イエロー,yellow)の三つの色の成分を取出し,別々の版にする必要があり,これを色分解(color separation)という.実際の印刷においては,上記の3原色だけでは黒の濃度や色味の完全な再現が困難であるため,墨色の版を加え,4色で色再現を行うのが基本となっている.現在,図9・4に構成を示すカラースキャナ(color scanner)によって色分解されている.カラースキャナでは,原稿走査部の原稿ドラムに巻付けられたカラー原稿を光電走査し,赤,緑,青の3色分解信号を得る.この信号をもとに電子回路部で,色修正,調子修正等の処理を施し,修正済の藍,紅,黄,墨の4色分解信号を出力する.露光記録部では,それらの信号によって,露光シリンダに巻かれたフィルムに走査露光を与え,修正済みの4色分解画像が得られる.最近のディジタル製版では,スキャナで読み込んだ画像はデータ化され,文字データといっしょに画面上ですばやく編集できる.図9・4 カラースキャナの構成また,このように編集したディジタル画像データからフィルムを介さずに直接,版に画像を生成する直接製版が一般的になっている.
