# 9・2・3 各種製版

> **章**: 第9章 印刷機械
> **節**: 9・2 製版

a. 凸版凸版印刷の刷版には金属凸版,ゴム版,感光性樹脂版などがある.金属凸版は,感光液を塗布した亜鉛板にネガフィルムを密着して露光し,水洗,バーニングの後,硝酸などで腐食して作られる.感光性樹脂版の処理方法は版材によって異なるが,感光性樹脂層は紫外線で硬化または不溶化する性質があり,画線部を光硬化させた後,非画線部である未露光部を水や溶剤で洗い出したり,エアで吹き飛ばすことにより,露光された部分を残し,凸版レリーフを形成する.最近では,フレキソ印刷の製版のようにネガフィルムを用いることなく,感光性樹脂版材上に直接レーザで出力される.b. 平板平版印刷の刷版としては,通常0.2~0.3mm厚さのアルミニウムの平板に感光剤をコーティングしたPS(presensit-ized)版が用いられる.光が当たった部分が画線部となるネガタイプと,光が当たらない部分が画線部となるポジタイプがある.フィルム原版とPS 版を焼付機で密着,露光後,現像処理することによって製版されていたが,最近では,フィルムを使用せず製版データから直接刷版にレーザで製版する装置が多く使用されている.c. グラビア製版方式には二つの形式があり,一つは,製版画像データを出力し,画線部を薬品で腐食させる方法である.この方法での凹部の製作には,次の三つの方法がある.1グラビアでの濃淡表現の種類に対応して,凹部の大きさを一定にし,深度を変化させるコンベンショナル法,2深さを一定とし凹部の大きさを変化させる網グラビア法,3大きさと深さの両方を変化させる方法であり,印刷物によって使い分けられている.もう一つは,シリンダを回転させながら,レーザ等で直接に版を作る方法である.凹部の面積と深さがともに変化して濃淡を表現する.d. スクリーン印刷(screen process printing)スクリーン印刷の製版方法は.手工的なカッティング法,描画法,写真製版法,フォトエッチング法,めっき法,電子製版法などいろいろな方法がある.現在おもに使われているのは写真製版法であり,直接法と,現像して得られた版膜をスクリーンに転写する間接法とがある.
