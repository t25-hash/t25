# 9・3・1 用紙(printing paper)

> **章**: 第9章 印刷機械
> **節**: 9・3 印刷資材

a. パルプの種類印刷用紙はアカマツ、ツガなどの針葉樹やブナ、カエデなどの広葉樹を化学的または機械的に処理して作られたパルプを主原料として製造される。パルプには、木材を機械的に破砕したメカニカルパルプ(MP)、木材チップに薬品を添加し蒸煮したケミカルパルプ(CP)、薬品処理と機械杓破砕を組合わせたセミケミカルパルプ(SCP)、ケミグラウンドパルプ(CGP)などの種類がある。最近では、環境保護の観点からリサイクルパルプ(故紙パルプ)の配合が増えている。b. 印刷用紙の種類。 新聞用紙メカニカルパルプを主原料とした紙で、両面とも印刷に適する。紙切れが少なくワニス吸収性が大きく、印刷後の不透明度を維持できることが要求される。. 印刷用紙A(上質紙)ケミカルパルプ100%に適当なサイズ、クレーなどの塡料を配して抄造したもので、主として書籍、教科書、商業印刷、一般印刷に使用される。最近では、古紙が配合されながら上質紙並みの印刷適性が得られる再生上質紙も現れている。. 印刷用紙B(中質紙)ケミカルパルプ70%以上、メカニカルパルプや古紙パルプ30%未満を原料としたもので、教科書、文庫本、雑誌の本文などに使用される。. 印刷用紙C(上更紙)ケミカルパルプ40%以上に塡料を配合して抄造したもので、雑誌本文、電話帳などに使用される。. 印刷用紙D(更紙)ケミカルパルプ40%未満のもので、主として雑誌本文に使用される。印刷は活版印刷が主体である。. グラビア用紙中質紙をスーパカレンダで光沢仕上加工したもので、主としてグラビア印刷に使用される。. 塗工紙表面に鉱物性白色塗被料を糊料とともに薄く塗被し、乾燥後スーパカレンダで仕上げ、平滑でち密な表面にした用紙。塗工量によりアート紙(20g/m/片面前後)とコート紙(10g/m/片面前後)に分けられる。美術書、雑誌の表紙、口絵などに使用され、印刷は主としてオフセットである。最近では、両面塗工量12g/m程度の微塗工紙も多く用いられている。
