# 9・3・2 印刷インキ(printing ink)

> **章**: 第9章 印刷機械
> **節**: 9・3 印刷資材

a. 印刷インキの組成印刷インキは色材である顔料とこれを分散し流展性を与え,固着させる媒質(vehicle,ベヒクル)が主成分である.これに乾燥性,流動性,皮膜特性などを調整する補助剤が加わっている.色材はフレキソグラフィで特に透明性を要求されるとき,あるいは布印刷など特殊な場合は染料が用いられるが,一般的には顔料の型で使用される.色材顔科染料主剤媒質バインダ(固着剤)ソルベント(溶剤)印刷インキ助剤乾燥促進剤印刷効果調整剤b. 印刷インキの分類印刷インキの分類方法にはさまざまなものがあり,(1)色別によるもの,(2)印刷方式によるもの,(3)被印刷物の用途によるもの,(4)媒質の相違によるものなどがあるが,おもには次の7種に分類されている.(1)平版インキ,(2)凸版インキ,(3)グラビアインキ,(4)フレキソインキ,(5)スクリーンインキ,(6)凹版インキ,(7)新聞インキ.c. 特殊なインキ印刷技術の進歩に伴い開発された特殊インキの例として,以下のようなものがある.. 紫外線硬化型インキ(UV-cure ink)感光性樹脂と光反応性希釈剤からなるベヒクルに,光増感剤を加えインキ化したものである.紫外線(UV)の照射により瞬時に硬化し,強い耐摩擦性が得られるため,紙器印刷,金属印刷,ビジネスフォーム印刷などに使われている.. 蛍光インキ蛍光顔料を使ったインキで,雑誌の表紙,ポスタ,広告宣伝物などにおいて人目を引くために使われる.. 磁気インキ(magnetic ink)磁性酸化鉄の粉末を使ったインキで,印刷された文字を磁気リーダで自動読取りするために使用される.小切手,通帳等の証券類の記号,番号の印刷に使われる.
