# 9・4・1 凸版印刷機(letterpress printing machine)

> **章**: 第9章 印刷機械
> **節**: 9・4 印刷機械

凸版印刷機は1440年代にグーテンベルグが発明して以来、長い歴史があり、多くの方式が生み出されてきた。しかし、現状では、巻取凸版輪転機やフレキソ印刷機が代表的なものとなっている。a. 巻取凸版輪転機(web rotary letterpress machine)巻取紙を使った高速の輪転機で、通常、凸版印刷機といえばこの型式をいう。おもな用途は新聞と書籍、雑誌本文などであったが、新聞輪転機の場合は、1970年頃から巻取オフセット輪転機に切換わっており、ほとんどがオフセット輪転機となっている。その他、書籍、雑誌用の輪転機もオフセット方式となっている。版としては感光性樹脂版が用いられる。b. フレキソ印刷機(flexographic press)フレキソ印刷は凸版印刷の一種であるが、ゴムや感光性樹脂版材で弾力のある凸版を貼付け、液状のインキを用いて印刷する。最近はフレキソ印刷の品質が格段に進歩し、パッケージ分野を中心に急速に普及する可能性がある。この印刷機には図9・5に示す三つの型式がある。. スタック型各色の印刷部を積み上げたものであり、最も基本的な型式であるが、伸縮しやすいフィルムやしわになりやすい金属はくには適さない。. センタインプレッション型共通圧胴の周囲に印刷部を配した型式で、被印刷物は圧胴に巻付いた状態で印刷されるので、伸縮しやすいフィルムやしわになりやすい金属はくの印刷に適する。各色の刷重ね位置精度が良好である。. インライン型印刷部を横に並置したもので、厚くて硬いクラフト、段ボールなどの印刷に適している。
