# 9・4・2 平版印刷機

> **章**: 第9章 印刷機械
> **節**: 9・4 印刷機械

平版印刷機は枚葉式、巻取式とも広く普及しており、ほとんどがオフセット方式であることから、通常、オフセット印刷機(offsetprinting machine)と称される。枚葉式と巻取式の特長は次のようである。枚葉式の特長1紙サイズが可変である。2損紙率が小さく、小ロット印刷が可能。3厚紙印刷が可能。4追刷りが可能。巻取式の特長1印刷速度が高い。2ドライヤがあり、乾燥待ち時間がない。3両面刷りをし、折った状態で出てくるので、ただちに製本が可能。図9・5 フレキソ印刷機の3型式1―216産業機械・装置a. 枚葉オフセット印刷機(sheet-fed offset press)枚葉オフセット印刷機は、版胴、ブランケット胴、圧胴から構成される印刷ユニットを印刷する色の数だけ並べ、前後に給紙装置と排紙装置が配置されている。それぞれのユニットの圧胴に紙を順次に受渡すためには、ユニット間には中間胴あるいはチェーンが必要であり、それらと圧胴には多数の爪が組込まれる。この爪は、爪軸に多数取付けられており、爪軸の一端に取付けられた腕が、フレームに固定されたカムで動かされ、爪の開閉が行われ、紙を受渡しする。版胴には、版を巻付け引張って固定する版クランプ装置がある。また版面に水を供給するための湿し装置と版面にインキを供給する多数のローラから成るインキ装置が接している。ユニット間に印刷胴の2倍径の中間胴を設けた4色印刷機の胴配列を図9・6に示す。最近では、枚葉紙の両面を同時に印刷できる両面枚葉オフセット印刷機が多用されつつある。b. 巻取オフセット輪転機(web offset press)巻取式平版輪転機は通常オフセット輪転機(rotary offset press)と呼ばれ、図9・7に示すように巻取紙を取付ける給紙部、紙を送り出すインフィード部、印刷部、インキ乾燥部、冷却部、折部からなる。印刷のみでなく、後加工のスリット、断裁、折りなどの作業が一貫して行えることが大きな特長である。. 給紙部各給紙部には巻取紙が装着されたリールスタンド(reel stand)があり、ここから紙が印刷部に送られる。この際に紙に緩みが生じる、逆に、紙が切れるなどのトラブルを防止すべく、張力制御されている。また、給紙部には、機械速度を落とすことなく、連続印刷が可能なように自動紙継装置(auto-paster)が組込まれている。現在主流のスピードマッチ式では、印刷中の巻取紙がなくなる寸前に次の巻取紙ロールの周速を現走行紙のスピードに同調させた後、現走行紙を次の巻取紙ロール表面に押付け、両者を貼合せるものである。. 印刷部印刷ユニットの構造は、大きく分けると共通圧胴型(common impression cylinder type)とBB 型(blanket toblanket type)に分類される。共通圧胴型は、図9・8に示すように、圧胴のまわりに、圧胴とブランケット胴を複数セット配置したものでサテライト型あるいはドラム型とも呼ばれる。BB 型のオフセット輪転機は、図9・9に示すように、二組のブランケット胴を相接するように配列したもので、一方のブランケット胴が、互いに他方のブランケット胴の圧胴としての働きをする。共通圧胴型は機械が大型化するが、圧胴に用紙が密着した状態で図9・6 枚葉オフセット印刷機図9・7 巻取オフセット輪転機重ね刷りが行われるため、見当精度が高く、一時は高級印刷用として盛んに導入された。しかし、BB タイプも用紙テンションの安定、資材の改良などにより品質が向上し、両者の品質上の差はなくなっている。そのため、現在では機横がコンパクトで、紙パスも短く繰作性の良いBB 型が主流となっている。. 乾燥部オフセット輪転機の乾燥装置には、直火式、熱風式、直火熱風併用式があるが、最近はほとんどが高速熱風方式となっている。熱風温度の制御方式も、乾燥装置を出た印刷紙面の温度を130°C前後に保つ紙面温度制御方式となっている。乾燥装置の排気ガスには特異の溶剤臭(芳香族系、パラフィン系)があり、かつ化学反応性が強いので光化学スモッグの原因物質にもなり、脱臭装置が必要である。脱臭装置としては、排気ガスをアフタバーナによって700~800°Cの高温で燃焼させる直燃式と触媒を使用して300~400°Cの比較的低温で燃焼させる触媒燃焼式がある。. 冷却部乾燥装置で加熱された用紙は、印刷品質の安定化と張力の安定化のために冷水を適した3~4本のクーリングローラにより冷却されインキの定着が行われる。. 折り部折り機については、ニーズの多様化を反映し、さまざまのものが設計されている。図9・10はオフセット輪転機の折り機の一例を示す。印刷後、乾燥、冷却された印刷紙は、まず折機上部の三角板で縦に二つ折りされて折胴に導かれる。折り機は通常3本~8本くらいの折胴から構成され、それぞれの折胴には、印刷紙を断裁するカッタ刃、断裁された折丁を受渡すための針、横に二つ折りするための突っ込み刃とくわえ装置などが組込まれている。折胴から排出された折丁はさらにチョッパ装置により縦折りされ羽根車の羽根の間に一部ずつ落とし込まれ整列してコンベヤ上を流れ、スタッカ(集積装置)へ導かれる。図9・8 system)熱練と時間を要するオフセット印刷のインキ供給量の設定を自動的に行うシステムであり、(1)刷版絵柄面積率計、(2)インキプリセット入力装置、(3)インキ供給量リモコン調整装置から構成される(図9・11)。刷版絵柄面積率計は、刷版の画線部の量を読取り、ディジタル信号化する。これをインキプリセット入力装置に送ることにより、インキ供給量を決定するインキキーの開度が自動的に最適状態に設定され、インキ供給量のプリセットが完了する。最近では、製版データから刷版の画線部データを演算し、印刷機のインキプリセット入力装置にネットワークを介して直接送られるシステムが開発され、刷版絵柄面積率計は不要になりつつある。. ブランケットウォッシャ(blanket washer)オフセット印刷では、一定印刷枚数ごとにブランケットを洗浄し、紙粉やインキカスを除去する必要がある。これを自動化するためにブランケット洗浄装置が開発された。ブランケット胴に溶剤を吹付け回転するブラシでこすり取る方式と、ペーパタオルで拭き取る方式がある。. カットオフコントロール装置(cut-offcontroller)印刷された紙を折り機で断裁(カットオフ)するとき、紙面の天地方向の断裁位置は絵柄に対し所定の位置となるように調整する必要がある。これを自動的に調整するもので、特殊マークをスキャニングヘッドで検知し、基準パルスと比較し、誤差をなくするように印刷部から折り部の断裁位置までの紙経路長さを変える。具体的には、図9・12のようにU 字型に紙を巻付けたコンペンセータローラをねじの回転により移動させて自動修正する。. エッジガイドコントロール装置(web edge controller)印刷中、巻取紙の巻きの不ぞろいなどで紙が左右にずれることがあり、これを自動的に検出・修正するものである。修正は2本の平行ローラを油圧シリンダまたはモータでひねることによって行われる。紙端位置の検出は、光電管方式とエア方式とがある。図9・11 インキ量プリセットシステム図9・12 カットオフコントロール装置1―217. 色調制御システム(automatic color control system)印刷物の色は、見本の色に一致する必要がある。これを自動的に修正するもので、印刷物の色を測色計により検知し、見本の色になるようインキ供給量を自動調整する(図9・13)。
