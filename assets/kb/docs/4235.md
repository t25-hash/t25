# 9・4・3 グラビア印刷機(gravure press)

> **章**: 第9章 印刷機械
> **節**: 9・4 印刷機械

a. 枚葉グラビア印刷機(sheet-fed gravure press)枚葉グラビア印刷機の版は図9・14のように版胴に巻付けて固定する.印刷された紙は紙取胴に移り,排紙爪で乾燥装置に運ばれた後,排紙部に積み上げられる.b. グラビア輪転機(rotary gravure press)巻取式のグラビア輪転機の場合は,版胴自体に製版される.印刷部は版胴,圧胴,インキ供給装置,ドクタ,乾燥装置から構成される.印圧を加える機構としては,単に1本の圧胴を用いる方式のほかに,バックアップロールを用いる方式もある(図9・15).インキを版に供給する方式には,ドブ漬け方式,着けローラ方式,カスケード方式などがある(図9・16).乾燥装置は,蒸気,電熱による加熱ドラム式,赤外線ランプ式もあるが,火災に対する安全性と乾燥効率の点から現在では,ほとんどが電気,ガス,蒸気などにより加熱された熱風を用いる方式となっている.グラビア輪転機の用途はさまざまであり,排紙の形態は用途によって異なる.書籍,雑誌カタログなどを印刷する出版用グラビア輪転機の場合は,折り機によって所定の頁の折丁として排出されるか,シータによって1枚ずつ断裁され積み上げられる.包装用紙,化粧板,壁紙などの印刷の場合は,リワインダによって巻取られ図9・13 色調制御システム図9・14 枚葉グラビア印刷機図9・15 バックアップロール図9・16 インキ供給方式1―218産業機械・装置図9・17 枚葉スクリーン平面印刷機る.また,紙器用グラビア輪転機には,インラインで打抜きまで行うものもある.
