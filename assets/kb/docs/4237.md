# 10・2・1 物流とロジスティックス

> **章**: 第10章 物流機械
> **節**: 10・2 ロジスティックスと物流機械

わが国の物流は,自動倉庫,無人搬送車の導入など機械化を積極的に推進し,高度化を図ってきた.さらに,それらはサービス,調達,廃棄,回収システムを取込み広領域化し,情報を活用しながら全体を通して最適となるシステムが構築されるようになり,ロジスティックスの登場へとつながっていった.物流とロジスティックスについては,識者によってさまざまな定義付けがなされているが,JISでは以下のように両者の違いを表現している.a. 物流(physical distribution)物資を供給者から需要者へ,時間的,空間的に移動する過程の活動.一般的には,包装,輸送,保管,荷役,流通加工およびそれらに関連する情報の諸機能を総合的に管理する活動.調達物流,生産物流,販売物流,回収物流など,対象領域を特定して呼ぶこともある.b. ロジスティックス(logistics)物資流通の活動目標を最終需要の必要条件や環境保全などの社会的課題への対応に求め,包装,輸送,保管,荷役,流通加工およびそれらに関連する情報の諸機能を高度化し,統合化を進めるとともに,調達,生産,販売,回収などの分野との一体化,一元化を図る経営活動.また,アメリカロジスティックスマネジメント協会では,ロジスティックスを,「顧客の要求に適合するように,産出点から消費点までの効率的かつ費用対効果の大きい物,サービスおよび関連する情報の流れと保管を計画し,実行し,管理するプロセス」と定義している.このように,定義はいろいろとあるが,物流は,生産した製品や仕入れた商品を販売し,顧客まで間違いなく届ける活動を指すのに対して,ロジスティックスとは,市場の需要動向に合わせて生産・仕入,販売,物流を一貫したシステムにするためのマネジメントであるといえる.
