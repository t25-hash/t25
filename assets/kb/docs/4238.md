# 10・2・2 ロジスティックスの進展と流通構造の変化

> **章**: 第10章 物流機械
> **節**: 10・2 ロジスティックスと物流機械

ロジスティックスの進展は、市場動向に従って商品の生産や原材料、部品などの調達が一貫して動き出すので、適正在庫が維持され、無駄なコストを抑えることが可能となる。つまり、ロジスティックスによって必要な在庫が必要量だけ配置されるため、徹底したローコスト化を実現することとなる。わが国の流通構造は、メーカ、問屋、小売という多段階で構成されており、それぞれの段階で物流システムが構築されているため、流通コストを押上げる要因になっていた。このような構造の中、ローコスト化は、これまで自社のコスト低減を視点に各種施策を実施してきたが、それも限界に近づいていた。そこで、各段階の企業は、ロジスティックスの進展に伴って、お互いに協力してコスト低減を求めるようになってきた。この考え方は、サプライチェーンマネジメント(supply chain management)と呼ばれており、情報化の発展により双方が同じデータを見ながら行動するので、あたかも一つの企業のように顧客に対応することが可能である。サプライチェーンマネジメントは、これまでの流通を構成する主体ごとに物流システムを構築する視点から、流通全体を捉えて物流システムを構築する視点へ切替えるものである。視点の切替えに当たっては、企業間の情報開示が必要となるなどデリケートな問題も含まれるが、小売などが主導となり、情報システムの高度化と相まって確実に進んでいる。また、サプライチェーンマネジメントの進展は、ロジスティックスに次のような変革をもたらした。1流通構造の簡素化メーカから小売店までの多段階な流通構造が、簡素化されるようになった。2アウトソーシングの促進企業は、これまで行っていた物流業務をロジスティックスシステムの構築や運営が得意な他の業者に任せることで、より効率的な運営を図るサードパーティロジスティックス(3rd party logistics)という仕組みを積極的に導入するようになった。3配送センタ機能の変化顧客への配送は、一括物流への取組みを重視するようになった。それは、従来の配送センタ機能に加え、小分け機能などより多くの物流機能を集約し、売り場の棚別商品をグループにまとめて同時到着させることで、店舗での棚補充作業の煩雑さを解決するためである。4物流センタ内作業の効率化の推進物流センタでは、保管効率と作業効率は相容れないものである。しかし、顧客のための物流センタという視点に立てば、在庫による作業効率の悪化を防ぐためにセンタ内のエリアを分割し、作業オペレーションの効率化を図ることが重要である。さらに効率を上げるためには、作業者の動きを少なくさせることが必要であり、コンピュータを利用したロケーション管理などを採用するのも一つの方法である。そして、こうした流れは、「海外で生産された商品を国内品と同じく、市場ニーズに合わせて、市場の求める量だけ、短時間で、ローコストで届ける仕組み」であるグローバルロジスティックス(global logistics)へと発展していく。
