# 10・2・3 物流機械の活用

> **章**: 第10章 物流機械
> **節**: 10・2 ロジスティックスと物流機械

商品の多くは,消費者の多様化するニーズや,技術革新の速さによって,商品価値としての寿命が短くなってきている.このため企業は,短期間のうちに物流機械を変更する可能性があると考えられる商品については,設備投資を慎重にせざるを得なくなってきている.また,物流システム市場は,景気後退による製造業からの需要の減少や,生産拠点の海外シフトの増加によって低迷を続けている.こうした状況では,サプライチェーンマネジメントの進展はますます重要となり,単品種生産型から多品種・小ロット生産型への移行が必要となってきている.企業の中には,大量生産ではなく,多品種,小ロットに適する屋台方式といわれる方式の採用も増えている.物流機械は,流通構造に対応する物流システムを構築するうえで,システムの中核となる物流センタを効率的に稼働させるための重要な要素であり,適切な活用は生産性の向上につながる.例えば,効率的な保管を実現するための立体自動倉庫,またはセンタ内作業のスピードと正確性の向上を実現するために,ディジタルピッキングシステム,自動仕分け機,さらにはバーコードを活用した検品システムのような高度情報システムなどがある.しかし,前述のとおり全般に商品のサイクルが短くなっていることから導入に当たっては,1今後の市場動向の把握(商品ごと)2人手を使用した場合のコストと機械化に伴うコストの比較3人手を使用した場合の作業精度の算定4機械化する際のレイアウト上の問題などを検討材料に判断する必要がある.こうした状況の中,物流機械メーカは,サプライチェーンマネジメントの進展に対応するため,機械の性能向上を目的とした技術開発を推進していくことが大切である.そして,さらに重要なことは,これまでのような単体機械の性能向上から,物流システム全体の仕組みの中で機械の性能を考えることである.例えば,単にコンベヤの速度を早めて処理能力を向上させようとしても,物流システム全体を通してみた場合,コンベヤの高速化が適切なことであるのか,サプライチェーンマネジメントを目指すに当たって有効な方法であるか否かを検討することが大切である.物流機械メーカは,流通構造の各段階にどのような性能の機械を導入することがサプライチェーンマネジメントの進展にとって最もよい方法であるかを提案するとともに,機械を導入した後であっても,現状に満足することなくつねに改善提案を行う姿勢が求められている.
