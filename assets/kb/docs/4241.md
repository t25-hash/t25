crane)一般工場、倉庫、製鉄所、発電所などで使用されるクレーンである。一般的に巻上げ、横行および走行装置を有し、これらの動きを組合わせて三次元的につり荷を運搬する。構造的にも単純で、安定性があり、運転操作が容易で、保守・点検が行いやすく、建設費・運転経費・維持管理費が比較的少ない。フックの上がりや寄りが良いこと、機体の高さや側方の出が少ないこと、機械部品や電気部品が点検・保守に便利なように配置されていることが望ましい。故障や損傷による事故を予防するために、クレーン等安全規則(safety ロープトロリ式アンローダ(rope trolley type unloader)ガーダ上の機械室内に巻上・開閉装置および横行装置が設けられ、また機械室内の一部を電気室として仕切り電気品もこの中に設置されている。トロリは、機械室内に設置された横行装置からワイヤロープによりけん引されるため、トロリ上には巻上・開閉用のシーブブロックのみとなりトロリの重量は、クラブトロリ式に比べ極めて軽くなる。トロリ重量が軽減できることからアンローダの機体重量も軽くなり、基礎費および設備費を低減することができる。また、ロープトロリ式の大きな特徴としては、トロリ横行時にグラブバケットを水平移動させるために補助トロリを設けている。ロープトロリ式の巻上・開閉および横行ワイヤロープの掛け方を図10・15および10・16に示す。保守性については、横行ワイヤロープの取替が必要となるが、耐曲げ疲労性に優れた異形線ワイヤロープなどの採用によりワイヤロープの長寿命化を図り、取替頻度を低減することも考えられている。図10・17シングルトロリ式コンテナクレーン1―227作業性については、クラブトロリ式同様運転室は、トロリとは独立して移動できるため、良好な視界を確保できる。.
