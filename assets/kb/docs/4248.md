# 10・4・2 棚設備

> **章**: 第10章 物流機械
> **節**: 10・4 保管設備

棚設備は固定ラックと自動ラックに大別することができる。固定ラックには重・中・軽量ラック、ドライブインラック、スライドラック、カンチレバーラック、ハンガラックなどがあり、自動ラックには回転ラック、流動ラック、移動ラックがある。図10・153 流動ラック図10・152 ドライブインラック図10・151 パレットラックa. 固定ラック。 重・中・軽量ラックパレットラックやケースラックに代表される固定ラックであり、棚に収納できる重量によって分類される。重量ラックはおもにパレットに積載された荷の保管に用いられることの多いラックで、パレットラックと呼ばれることも多い。これらのラックは、保管している荷の先入れ先出しができるので荷の性質、品種の多少にかかわらず使用できる非常に汎用性の高い設備である(図10・151)。ラックは支柱と棚から構成されており荷の高さに応じて棚を上下に移設することもできる。. ドライブインラックパレタイズされた保管物をフォークリフトにより入出庫する棚であり、フォークリフトの通路がそのまま保管スペースになるため、極めて保管効率の高い棚である(図となり、少品種多量の荷を保管するのに適している。また、上段の棚に荷を入出庫する場合、フォークリフトを必要高さまでリフトしたままラック内を走行するため、奥行の長いドライブインラックは、作業性を良くするためにフォークリフトの下部にガイドロール、ラック下部にガイドレールを取付けたカイドレールシステムを併用することもある。. カンチレバーラック支柱と支柱に接合された片持ばりで構成されているラックで、保管する荷はこの片持ばりで支持して保管することになる。通常はパレタイズされていないパイプ、鋼材、巻物等の長尺物がベニヤ板、鋼板等の保管に使用されることが多い。b. 自動ラック。 回転ラック回転ラックは水平形と垂直形に大別される。回転ラックは棚が自動的に手元まで回ってくるため、保管やピッキング用の通路が不要という特長から多品種少量、小物部品の保管効率向上およびピッキング効率向上という目的で利用される。ただし、荷をラックに補充する場合、出荷作業と同時に並行して行うことが困難なことが多いため、設備に際しては、作業スケジュールを十分に考慮する必要がある。回転ラックには次のような種類があるが、最近はコンピュータを使って情報管理機能を有する小物自動倉庫としての使用例がある。(1) 水平型複数の棚を有する一体構造のラックが水平に回転するもので、小物部品の大量保管・人手によるピッキングに適した回転ラックである。棚の取付けは自在であり、収納物に応じて変更できる。またラックを各段ごとに個別に、しかも任意方向に水平回転して呼出し口に自動的に停止させる型式のものもある。(2) 垂直型天井空間を最大限に有効利用でき、しかも手元まで自動的に回ってくるので、定位置作業ができるのが垂直型の最大の特長である。事務所の書類保管から工場の小物部品・電子部品等軽量小物、さらには最近は食器保管等にも利用されるなど、各分野に適用範囲が1―261図10・154 移動ラック広がっている。. 流動ラック保管物が棚部分に取付けられたロールコンベヤ、ホイールコンベヤなどによって出口へ向かって移動してくるラックである(図10・153)。重力で移動するタイプと駆動コンベヤによって移動するタイプがある。前者は設備費用が安価だが前面の荷に後方の荷の重力が加わり、衝撃を与えたり破損したりすることがある。後者はこの問題はないが前者に比べて高価である。このラックは先入れ先出しができ、格納効率が高く、しかも入出庫は棚の前後で行えるので作業性が良い。格納物の底面形状、荷重のばらつきにより流れ速度が左右されるため、途中で止まったり、流れすぎて暴走したりする欠点もある。この流動ラックにはパレット単位の重量用と、カートンケースやプラスチックコンテナによるピッキング作業を主体とした軽量用がある。. 移動ラック移動ラックはラックが一列ごとに独立して動くため、何列の棚であっても作業通路は一列だけあればよい。そのため設置面積が同じならば固定ラックに比べはるかに多くの棚が設置でき、収容効率は格段に増える(図10・154)。倉庫や事務所、図書館などで多く使われている事例を見ることができる。ラックの移動を電動式で行うものと、動力を用いず人力により棚を動かすものに分類することができる。移動ラックは床に角鋼または軌条レールを埋込み、台座に車輪(両端にはフランジ付車輪)を取付け、この車輪を動力あるいは人力によって駆動して、上部に載せたラックを移動させるものである。動力を使用した移動ラックの場合、人がラックに挟まれるのを防止するため非常用バンパ、進入検出装置、開通路インタロック、電源投入インタロック等二重、三重の安全対策が施されているのが一般的である。
