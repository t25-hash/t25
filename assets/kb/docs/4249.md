# 10・4・3 貯槽

> **章**: 第10章 物流機械

粉粒体の貯槽(storage vessel)は、原料・中間製品・製品出荷の一時的な保管、物質の物理・化学・生物的な操作および備蓄など1―262産業機械・装置を目的として使用される。その特徴は、次のとおりである。(1) 倉庫貯蔵に比べて数倍の貯蔵能力がある。(2) 袋づめに比べ、大量の取扱いが可能である。(3) 品質管理に必要な温度・湿度・水分・ガス濃度計測、くん蒸、入替えが容易なため、変質・虫害が防止できる。(4) 粉じん飛散が少なく、騒音も比較的小さい。貯槽は、大容量をサイロ(silo)またはバンカ(banker)、密閉構造をタンク(tank)またはビン(bin)、底部が円すい型または角すい型をし、下部に供給あるいは排出装置が付いている場合をホッパ(hopper)と呼ぶ。粉粒体の輸送方式は、方法によって分類すると表10・29に示すとおりである。また、貯槽を計画するに当たって注意しなければならない一般的な事項は次のとおりである。(1) 位置搬入、搬出、全製造工程が合理的に行えるよう計画する。(2) 形状地耐力、地震力、風圧に対する強さ、敷地の形、周囲の状況などから制約されることが多い。(3) 大きさ粉粒体が最も小さいかさ比重量の状態で、かつ表面に安息角をもった形で蓄えられても、所定の量を十分収容できる有効面積をもち、かつ上部に空間をもつように計画する。(4) 地耐力できるだけ地耐力の大きい場所に設置する。(5) 搬入・搬出装置粉粒体の性状と、貯槽の使用条件に応じて適切な方式と機器を選定する必要がある。(6) 偏析の防止製造工程中でより均一な操作をしたい場合は、偏析現象を最小限度に止めるような投入および排出の方法を計画する。(7) 粒子の破砕の低減破砕を少なくするには、原則として衝突、摩擦をできるだけ少なくして、遅い速度で投入する必要がある。(8) 強度と構造内部気体の圧力、粉体圧、積雪、地震力、風圧に対して十分な強度をもつように各部分を設計する。(9) 除湿、加熱、保温、冷却など粉粒体の品質を保持し、所定の処置を円滑に行うために必要な手段をとる。(10) 架橋現象の防止架橋現象は、貯槽の根本的な問題なの無端のチェーンまたはベルトの循環による方法⎧얧⎨얧⎩⎧|얧⎨||⎩機械的輸送方法樋型容器の振動による方法⎧⎨⎩空気コンベヤ⎧얧⎨얧⎩空気による方法⎧⎨||⎩⎧||얧⎨얧||⎩流体を利用する方法⎧||⎨⎩ハイドロホイスト方法水による方法容器による輸送方法⎧⎨⎩コンテナ、バグ、タンクローリー専用貨車、船舶単面型二方差しS 2片面使用型二方差しD 2両面使用型二方差しR 2単面型四方差しS 4片面使用型四方差しD 4両面使用型四方差しR 4単面けたくり抜き型SP4片面使用けたくり抜き型DP 4――で十分考慮する必要がある。(11) 付属機器設備必要に応じて、点検、安全管理のための付属機器を設備し、運行と安全と合理化を図る。(12) 基準、規格設計規準は、日本建築学会規準による。(13) 調査事項これまで述べたことを検討するに当たっては、以下の事項を調査し、明らかにする必要がある。(a) 貯蔵量(b) 要求される使用材料と表面仕上げの程度(c) 粉粒体の性状(d) 必要とする付帯設備(e) 荷重(積載荷重、積雪状態)(f) 地盤強度、地耐力(g) 前段後段の機器と運転条件(h) 内部にこもる圧力の程度(i) 搬入、搬出の度数、貯蔵日数(j) 保温、加温、冷却の要否(k) 万一の事故の種類と処置の方法貯槽における諸トラブルは、以下の項目が考えられる。(i) 架橋現象(ii) 付着(iii) 流出速度均一(iv) 分離・投入排出順序の不同(v) 粉じん爆発
