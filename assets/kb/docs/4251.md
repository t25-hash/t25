# 11・1・1 はじめに

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・1 安全規格,法制制定の歴史的展望

近年の化学プラントをはじめとする産業機械は多くの場合システム化され,無数の機構部品や電子部品,それらを制御するコンピュータにより構成されるようになった.システム要素には当然劣化故障や偶発的な異常が考えられるわけであるから,その結果産業機械周辺への人間への危害,あるいはシステム自体の動作不安定性が現実の問題となってきた.しかし従来,機械工学者の通念としては,そのような懸念があったとしても,それは一時的な人間の過誤によるもので設計改善や,品質管理で防止できる性質のものととらえてきた.ところでヨーロッパ先進国では20世紀後半に至り,チェルノブイル事故に代表される化学プラント等の重大事故を数多く経験することによって従来の常識を根本的に変革しなければならないという機運が生じた.
