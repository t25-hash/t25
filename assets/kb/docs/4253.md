# 3・4・4 実例

> **章**: 第3章 ターボポンプ
> **節**: 3・4 軸流ポンプ

軸流ポンプは必要NPSHが大きくなるため，ふつう4m以下の
揚程に使用されるが，立軸の場合には揚程5mのものも見られる．
灌漑，排水などの農業用や，都市の雨水排水および下水道用に用い
られる．最近は，吐出しエルボの損失を低減するため，斜軸形，円
筒形が採用される例も増している．可動羽根形や回転数制御の軸流
ポンプは正規吐出し量以外でも効率が良いので，省エネルギーの観
点から大型ポンプで採用される例が多い．
横軸軸流ポンプは図Ⅱ-3･47に示すように，案内羽根と一体のケ
ーシングおよび吸込エルボによりポンプケーシングが構成され，ま
た案内羽根ケーシングに内蔵された水中軸受（submerged bear-
ing）と吸込エルボの外軸受（スラスト軸受兼用）とにより，羽根
車軸が支承される形式がふつうである．立軸の場合には図Ⅱ-3･50
に見るように，さらに数個の中間ケーシングと中間水中軸受を設け
ることがある．
図 -3･512400mm立軸軸流ポンプ据付図
大型ポンプでは立軸が採用される例が多いが，その場合，吸込ベ
ル部分（または吸込エルボ）および吐出しエルボは，コンクリート
で作られることもある．
軸流ポンプの水中軸受の材料，潤滑方法にはさまざまな形式があ
る．軸受材質としてはバビットメタル，カーボン樹脂または青銅が
あり，潤滑方法としては，ポンプ運転中に自動または手動で外部か
らグリースを注入する方法が一般的である．立軸ポンプに使用され
る水潤滑のゴム軸受は，青銅またはステンレス鋼等の金属円筒に軟
質ゴムを裏張りしたもので，ゴムの内面には潤滑および冷却水の通
水用溝が設けてある．この軸受は耐油，耐熱性に乏しく，また許容
荷重が小さいことが欠点である．
図Ⅱ-3･51は，排水用2400mm立軸軸流ポンプの据付図の例で
ある．仕様は，吐出し量：13m / s ，全揚程：2.4m，回転速度：
110min ，軸動力：640kWである．水は左から流入し，除じん
機，クローズド形吸込水槽を経てポンプに流入し，ポンプ吐出しエ
ルボを経て右側へ排水される．ポンプ上部床面に設置されたディー
ゼル機関の動力は，かさ歯車減速機を介してポンプの垂直主軸に伝
達され，ポンプは駆動される．近年は原動機に，冷却水不要による
簡素化や始動性に優れているガスタービンが用いられる場合もあ
る．立軸ポンプは羽根車をつねに水中に置いているので，呼び水が
不要で始動が容易であると同時に，原動機を水面上高く置くことが
できる利点がある．軸流ポンプは一般に吐出し弁を全開して運転さ
れることが多いので，吐出し口の末端に簡単な逆流防止弁を設ける
のがふつうである．
