# 11・1・3 ISO/IECガイド51

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・1 安全規格、法制制定の歴史的展望

a. 適用範囲本ガイドは安全がかかわる規格を作成する場合、その規格の体系を提供するものであり、安全が関連するすべての項目に適用される。ここでよく既往の品質管理規格ISO 9000などと混同されることがあるが、品質と安全とは同義語ではなく、そ図11・1 リスク査定の流れれぞれの固有の役割を区別して理解しなければならない。しかし両者は全く無関係というわけでもない。b. 安全という用語の意味通常、安全(あるいは安全な)という用語は、全く危険がない状態を表す言葉として使用される。しかしその認識は往々にして限られた時間領域や経験の中での狭い認識であり、未来永劫の安全状態を保証するものではない。そこで時間や空間にかかわらず安全を保持するには安全は危険な状態にないことという発想を捨て、危険が最小な状態にあることと定義するほうが賢明である。そのためリスク概念とリスク低減技術が重要な意味をもってくる。以下にガイド51が推奨する主要な安全概念を挙げておく。・安全(safety):許容し得ないリスク(unacceptable risk)が存在しないこと・危険事象:損害を生じうる危険な状況・リスク(risk):損害の発生確率とその損害の重要度の組合せ・残留リスク:予防処置を講じた後に残留するリスク・防護処置:少なくとも許容しうるリスクに到達するために組込む複数のリスク低減策(低減プロセスの一例を図11・1に示す)。このISO/IECガイド51(ISO/IEC Guide 51)はセベソ指令に続く機械指令、圧力機器安全指令、機器安全に関するヨーロッパ規格、国際規格の基礎概念となっている。
