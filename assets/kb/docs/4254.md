# 11・1・4 機械指令

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・1 安全規格、法制制定の歴史的展望

安全に関するヨーロッパ指令の中で、次に重要なものは機械指令である。この指令は機械の製造業者に対してそのリスクを最小にするように安全装置の設置を義務づけたもので、その対象とする機械の範囲は簡単な電動器具からロボットに至るまで非常に広い範囲を網羅している。ここで安全装置として電気的制御装置を使う場合、設計基準やその装置の作動信頼性を定性的に定めた規格が包括的な機械安全規格である(ISO 13849)。ヨーロッパではその前身のEN 954といったほうが有名であり、電気的な安全装置は五つのカテゴリーに分類される。さらに化学プラント等の装置産業においてはボイラや圧力機器の安全装置が重要になる。重大危険が予想される化学プラントでは多重防護の思想で安全が確保されており、個々の防護層の安全機能をどの程度の信頼性で設計するかが問題になる。また最近の安全計装はPLC等のプログラム可能な電子装置が使われる傾向にあり、ソフトウェアを含めた安全規格が必要とされる時代に入っている。その要請に応えて作成された国際規格がIEC 61508(機能安全)であり、対象は化学プラントに限らず医療機械、原子力発電への適用も意識した非常に広い、基幹的な概念規格となっている。本規格は翻訳されてJIS C-0508として出版されている。
