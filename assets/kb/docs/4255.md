c .羽根取付角（翼角）制御（vane angle control ) 大型の軸

のように広い流量範囲にわたって，高い効率で使用できる．短所は
翼角操作機構が増し，駆動部の設計に制約が生じることである．

g.自動制御 ポンプの自動制御とは，省力化・省エネルギー
または最適化等を目的として，制御量である流量，水位，圧力，原

ある．制御ループの内容から，図Ⅱ-3･62に示すようなものに大別
できる．このうちフィードバック制御は，プロセス制御の主流をな
すものである．一例として，ポンプで揚水される水槽の水位を一定
に制御するような場合である．制御方法として上水道分野では，重
回帰モデル，カルマンフィルタによる配水量の的確な予測が行われ
ている．配水量は人間の活動によってランダムな変動を示すが，天
候，曜日などの要因によって，ある程度パターン化することが可能
である．そこで各種要因を取り込んで，重回帰モデルによる日間配
水量，時間パターンを予測する．誤差が大きい系に対しては，カル
マンフィルタを利用している．雨水排水ポンプ機場の運転は，遊水
池の貯留効果を利用して比較的緩慢な操作が行われてきたが，集中
豪雨の頻繁化，流域の市街化などでポンプの運用が次第に困難にな
ってきていることから，ファジィ制御が適用されている．これはオ
