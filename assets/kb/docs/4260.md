# 11・4・1 OSHA

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・4 プラントの安全規則(OSHA, EPA, Seveso II,

アメリカ労働安全衛生局(OSHA : Occupational Safety andHealth Administration)が1992年に施行したプロセス安全管理(PSM : process safety management)は、従業員あるいはその他の人々に重大な危険を及ぼすようなおそれのある、危険な化学物質の望ましくない流出を防ぐことを目的としている。OSHA-PSM は、施設が次の条件に該当する場合に適用される。(1) 特定の化学物質(137物質)をそれぞれの指定量以上取扱うプロセス。(2) 特定の可燃性液体あるいはガスを、1箇所で一万ポンド(4535.9kg)以上取扱うプロセス。ただし、消費や貯蔵に関わる特定の条件に該当する場合には、適用除外される。プロセス安全管理を効果的に行うために、各事業者は次のプロセス安全管理規定を確立し、運用することが求められる。従業員の参画、プロセス安全に関する情報の整備、プロセス危険解析、運転手順書、教育および訓練、コントラクタ(契約企業)、試運転前安全レビュー、機器の健全性維持、火気取扱い許可、変更管理、事故調査、緊急時対応計画、適合監査。
