# 11・4・4 OECD

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・4 プラントの安全規則(OSHA, EPA, Seveso II,

経済協力開発機構(OECD : Organisation for Economic Coop-eration and Development)が2003年に発行したガイダンスは、装置の安全な計画、建設、操作等を含む安全管理のためのフレームワークであり、危険物質の取扱いに関わる事故を防止すること、および、有効な土地利用計画(land use planning)、緊急事態への備え、および緊急時対応によって事故の影響を緩和することを目的とするものである。このガイダンスには、事故の際に潜在的に影響を受ける一般市民のような利害関係者のみならず、公共企業体、企業経営者、従業員、および非政府組織等の役割、責任が示されている。なお、このガイダンスで用いられている安全とは、危険物質を含む事故の予防、備え、あるいは対応に関わる限りにおいて、健康、環境、資産の保護を含むものとされている。全体は次のパート(括弧内は章)で構成される。パートA:化学物質に関わる事故の予防(一般原理、企業(マネジメントおよび従業員)、公共企業体、市民および他の関係者)パートB:緊急時への備えおよび緩和措置(緊急時への備えおよび計画、土地利用計画、市民とのコミュニケーション)パートC:緊急時対応(一般原理、企業、公共企業体、市民および他の関係者)パートD:事故およびニアミスの後処理(影響評価、事故後の医療的側面、事故報告書、事故調査)パートE:特記事項(二国間協力・多国間協力・国際協力、固定設備および輸送)
