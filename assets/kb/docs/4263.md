> **節**: 3・1 遠心・斜流送風機および圧縮機の設計

ディフューザは，羽根なしディフューザと羽根付きディフューザ
に分類される．
a.羽根なしディフューザ（vaneless diﬀuser ) 平行壁羽根な
しディフューザでは，流れが非圧縮で損失がないと仮定すると，連
続の式と角運動量保存則から，ディフューザ内で流れ角は一定つま
り流線は対数らせんとなる．しかしながら，実際には流れは圧縮性
があり損失もあるので，ディフューザ内で流れ角は一定ではない．
圧縮性が強い場合には，密度が下流に向かって大きくなるので流れ
角は小さくなる．圧縮性が弱い場合には損失により，角運動量が減
少するため流れ角は下流に向って大きくなる．
羽根なしディフューザは，構造が簡単であるためよく使用され
る．しかしながら，一般に流路長さが長くなるため，摩擦損失が大
きくなり圧縮機の効率は低くなる．また，流量の減少とともに流入
角が減少し流路長さが増加するため，損失係数は増加して静圧回復
率は減少する，つまり右上がりの流量-圧力（静圧回復率）特性を
示す．したがって，出口羽根角度が大きい羽根車と組合わせると設
計点付近で圧力係数が最大になりサージが発生しやすい．
低比速度段では一般に流路幅が狭く，ディフューザの流入角が小
さくて流路長さが大きくなるので，やはり圧縮機効率が低下するこ
とになる．さらに，羽根なしディフューザでは流入角が小さい場合
には壁面近くで逆流が発生し，旋回失速の原因となる ことが
あるので注意が必要である．
また，高比速度羽根車と組合わせると羽根車出口の幅方向のひず
みが大きいため，ディフューザの静圧回復率は低下する ．
斜流羽根車出口の流れを円すい面の羽根なしディフューザに導く
回復が可能である ．
と，速度の遅い流れは内側の円すい面に集積し，圧力回復率は低
