# 11・5・1 産業機械の設備診断の基礎

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・5 設備診断

a. 設備診断の流れ設備診断(machine diagnosis)とは,生産設備などの作動状況の確認,劣化進行の度合い,故障(fault)の検知を行うことである.必要に応じて,故障の種類や故障個所の特定も行われている.設備が大規模な場合や故障が事故に結び付く可能性のある場合などでは,特に重要な技術となる.設備診断が有効であるには,(1) 予防保全が目的であれば,故障の徴候が検知できること(2) 故障後の被害拡大の防止が目的であれば,故障の発生が検知可能で,その後,機械の停止などの処置を講じる時間的な余裕があることが必要である.一般に,設備診断は大きく3ステップに分けられる(図11・11).Iは,診断に必要な情報を計測するステップで,ここで必要な信号が得られなければ以降のステップが無意味になる.IIは,検知のために信号から特徴抽出を行い,次のIIIにおいて,抽出された特徴成分の大きさなどから判定を行う.II,IIIの方法としては,(1) 信号ベースの方法:計測された信号に含まれる特徴を抽出して判断する手法(2) 解析的冗長を用いる方法:対象システムをモデル化し,対象システムとモデルの挙動の差違を検出して判断する手法とに大別できる.簡単な例を図11・12に示す.軸受転動面のはく離発生を振動の増加から検知するものである.振動の実効値のみでも大きなはく離は検知可能である.このとき発生する振動の周波数やパルスの間隔は,軸受の幾何学的形状から求まるので,その周波数の振動を監視すれば,より小さなはく離も検知可能となる.また,時間推移(トレンド)を監視すれば,はく離の進行状況や大きなはく離発生までの時間を予測することもできよう.このように,目的に応じて,信号処理,判断方式を選ぶ.b. 設備診断の実施例産業機械の設備診断のおもな部位,故障の種類,発生する信号の特徴を表11・5にまとめる.c. 設備診断を行う際の注意点一般に,信号にはノイズが含まれているから,誤報と欠報が避けられない.つまり,高感度化すれば検出率は向上するが同時に誤報(false alarm)(異常がないと図11・11 設備診断の基本的な手順図11・12 軸受の診断の例表11・5 代表的な診断例と信号の特徴要素故障形態信号の特徴軸アンバランス回転数と同じ周波数成分の増加継手ミスアライメント回転数の2,3倍の周波数成分の増加歯車歯面摩耗振動波形が正弦波状でなくなり,かみあい周波数の高調波成分が発生転がり軸受転動面はく離数kHzの振動が幾何形状から決まる時間間隔で発生潤滑不良数kHz以上の振動(加速度)が増大(本表はごく一部の例示である)図11・13 知的設備診断の流れきに異常と判定する)が多くなる.逆に,低感度化すれば誤報は下がるが,欠報(検知漏れ)や検知遅れが発生する.このため,設備診断の設計に当たっては,検出率,誤報率のトレードオフの問題となる.そこで,11・5・2項のように誤報などをいっそう下げる研究が行われている.
