# 11・5・2 知的設備診断技術

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・5 設備診断

知的設備診断技術(intelligent technology for machinery diag-nosis)とは計算機を用いた設備診断技術のことで、計算機が人間の代わりに単独で自動診断を行う場合と、計算機が人間の診断活動を補助的に助けて診断の精度を高める場合が考えられる。知的診断の流れは図11・13に示す。a. 信号測定の手法設備の信号を測定するセンサの種類は設備の種類および現場状況によって異なる。能動機械設備(特に回転機械)の診断には音響信号、振動信号およびAE(acoustic emis-sion)信号が最も多く用いられる。音響センサは人間が接近不能な箇所や少数のセンサで多数の設備状態を広域監視するのに適する。知的点検・診断ロボット、エンジンの診断、パイプラインの診断およびコンプレッサバルブの診断などの例がある。振動センサを用いて回転機械を診断する場合、センサの最適な測定位置および異常信号を検出する最適な周波数帯域についての研究例がある。AEは金属などの材料内部のクラック等によって発生するため、AEセンサを用いて軸受や歯車等の機械部品の初期異常を鋭敏に感知することができる。b. 特徴抽出の手法異常信号を抽出した後、異常の種類や異常の程度を判別するために、対象の信号の特徴(設備状態)を示す有限個のパラメータを計算する必要がある。このように設備状態を鋭敏に反映するパラメータを診断用の特徴パラメータ(symptomparameter)という。特徴パラメータは、時間領域の特徴パラメータや周波数領域の特徴パラメータなどがある。どのような特徴パラメータを用いるかは診断の内容や異常の種類などによって異なる。人工的に試行錯誤で診断用特徴パラメータを選定する場合、多大な労力が必要なだけでなく、良好な特徴パラメータが見つかるとも限らない。計算機で自動的に有意な特徴パラメータを抽出するために、遺伝的プログラミング(GP)による特徴パラメータの自己再組織化が提案されている。例えば、ある回転機械のシャフト・ミスアライメント状態を識別するために、図11・14に示すように、従来定義された特徴パラメータ~の異常識別率が低いが、遺伝的プログラミング(GP)による特徴パラメータの自己再組織化で得られた特徴パラメータの識別率は約100%に達している。d. 自動診断の手法設備診断は、簡易診断と精密診断に分けられている。(1) 簡易診断では、診断用の良好な特徴パラメータを選出した後、設備状態を定量的に評価し、異常の有無を判別する。簡易診断の場合、判定基準の決定は重要である。統計検定理論や可能性理論などによる判定基準の決定法が提案されている。(2)
