# 3・2・2 ラジアルファン（radial fan)

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・2 遠心送風機および圧縮機各論

a.構造 通常，羽根出口角度が半径方向（90°またはそれに
近い角度）の羽根車の送風機をラジアルファンという．ラジアルフ
ァンの中で，特に羽根形状が曲線で前向き形状のものをラジアルチ
ップファン，直線状の羽根をハブから放射したスポークに取付けた
構造の羽根車をパドルファン（paddle fan）といっている．ラジア
ルファンは羽根形状が比較的単純，簡単であり，羽根の補修・修理
やライナの交換もしやすいため，気体中に多量の摩耗性ダストや付
着性ダストを含む場合，または粉体・固形物の空気輸送用などに使
用されている．またラジアルファンは多翼ファンに次いで圧力係数
（pressure coeﬃcient ）が高く，羽根の強度も強いことから，ガス
温度が800～900℃の高温用送風機にも使われている．激しい摩耗
性ダストや付着性ダストを含む気体を取扱う場合にはパドルファン
が適している．パドルファンは羽根枚数が6～8枚で，羽根の両側
に側板をもつクローズド形とオープン形（図Ⅲ-3･8）があるが，一
般にはオープン形が採用されている．
ダストの摩耗・付着対策としては，必要に応じて，羽根，主板，
側板に鋼板製または硬化肉盛り溶接をしたライナを取付けたり，セ
ラミックチップを取付けている．この場合ケーシングも同様な摩耗
対策をしている．
b.性能 ラジアルファンの圧力曲線は比較的フラットで，抵
抗の変化に対して風量の変化が大きく，動力曲線は風量の増加とと
もにほぼ直線的に増大する．一般的にサージ限界は最高効率点風量
の50～60%程度である．オープン形パドルファンの特性曲線例を
図Ⅲ-3･9に示す．効率は主板・側板をもつ構造の羽根車で普通
65～75%である．パドルファンは羽根車の構造上効率は低く，オー
プン形で50～60%，クローズド形で55～65 %程度である．
