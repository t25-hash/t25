# 3・2・3 後向き羽根ファン（backward curved fan)

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・2 遠心送風機および圧縮機各論

a.構造 圧力比1.3程度以下の，後向き羽根ファンを通常タ
ーボファン（turbo fan）と呼んでいる．羽根は回転方向に対して
後方にわん曲し，その出口角度は30° ～60°である．羽根入口部か
ら出口部に向けて曲線状または翼形状に成形された羽根は，主板・
側板に溶接/リベット締めされ，さらに主板はハブに溶接またはリ
ベット締めされている［図Ⅲ-3･10(ａ)(ｂ)］．後向き羽根ファンに
は，等厚の板羽根構造のものと羽根断面が飛行機の翼のようにふく
らみをもった翼形構造のものがあり，翼形羽根は板羽根に比べて効
率がよく，羽根の強度も高い．この形式のファンは同一の風量・圧
力に対して羽根車とケーシングの大きさは遠心ファンの中で最も大
きくなるが，効率が高く性能も安定しているなど優れた特質をもっ
ているので，一般産業用の押込み・誘引用，およびブースタファン
として広く用いられている．摩耗性ダストを取扱うファンであって
も，ダスト量が比較的少ない場合には，近年は省エネルギーの観点
からしかるべき摩耗対策（耐摩耗ライナ，セラミックチップの取付
けなど）をして，この構造のファンが使われている．また，耐腐食
性を要求される場合にはステンレス鋼などの耐食材料を使用する．
ケーシングは鋼板を渦巻状に成形し，摩耗性ダストを含むガスを取
扱う場合はライナを内面に取付けることもある．また，高温ガスを
取扱うファンでは，熱膨張を考慮してケーシングの据付けレベルを
回転軸心近くとしたセンタサポート構造とする場合がある．他の機
種と同様，風量・用途に応じて片吸込形，両吸込形が選定される．
.片吸込形ファン（single suction fan) 羽根車の吸込口
が片側だけにあるファンで，ケーシングの吸込口を軸方向に設ける
場合と，袖を付けて軸と直角方向に設ける場合とがある．片吸込形
速運転される構造（図Ⅲ-3･15）のものとがある．
ファンの軸受は，通常，ケーシング吸込口と反対側に設置される
（片持ち形，図Ⅲ-3･11）が，羽根車にアンバランスを生じやすい用
途や軸の強度上片持ち形に適さない場合は，軸受をケーシングの両
側に配置する構造（両持ち形）とする．小型のファンでは電動機の
軸端に羽根車を直接取付けた構造（直動形）とする場合もある．
またはファンの圧力による軸スラストが問題となる場合などに採用
される（図Ⅲ-3･12）．
サージ限界が遠心ファンの中で最も小さい（最高効率点風量の
30～40%）ので，安定運転領域が広い．また，圧力曲線の頂点より
右側の圧力こう配が大きいため，抵抗の変化に対して風量変動が小
さいので運転しやすい．動力曲線は風量が増大すると頭打ちになる
傾向（リミットロード性）があるので，他の遠心ファンに比べて駆
動機容量の余裕が少なくて済む（計画動力の10 %前後）．効率は板
羽根，翼形羽根とも70～85%であり，騒音も低い．
れている．羽根車の外周速度が増すに従って応力が増大するため，
それに耐えられるように主板・側板・羽根の形状，材料，制作方法
が考慮される．形状は高速になるに従って主板・側板の内周側の厚
図 -3･13後向き羽根ファンの特性曲線
みが増し，最も高速になると側板のない形状となる（図Ⅲ-3･14）．
a.単段形ブロワ 1個の羽根車で所定の要項を満足するもの
で，片吸込形と両吸込形とがある．片吸込形は片持ち構造とする場
合が多く，軸貫通部が1箇所となるため，複雑な軸封装置を必要と
する危険ガスを取扱う用途に有利である．両吸込形は片吸込形では
ガス流量が足りないときや片吸込形では圧力による軸スラストが大
きい場合などに採用される．高速で回転させるため，一般に歯車装
置で増速する駆動方式をとることが多い．この場合，ブロワの羽根
車を高速回転側の歯車軸に直接取付けたものと，ブロワ自体が軸と
軸受を単独にもち，カップリングを介して電動機直結または歯車増
b.多段形ブロワ ブロワの圧力が高くて羽根車の強度上単段
形では所定の圧力を出すことができない場合や，比較的圧力は低く
ても圧力に比べガス流量が少なく（いわゆる比速度が小さい），羽
根出口幅が狭くなり効率が悪くなったり製作限界を超えるときに
は，一軸上に数個の羽根車を取付けた多段形が使われる．片吸込形
.両吸込形ファン（double suction fan) 風量が多い場合
