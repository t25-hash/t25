# 2・1・3 全エンタルピー

> **章**: 第2章 流体機械によるエネルギー変換
> **節**: 2・1 流体のエネルギー

結局、任意の断面を流体とともに流れるエネルギー量は、流体自身が保有するエネルギーと流体が伝達するエネルギーの和となり、単位質量当たり+12++ρである。流れている流体を扱う場合、内部エネルギーとρはつねに結びついて現れるから、この両者の和を一つの量=+ρ (J/kg)(I-2・3)で表すのが便利である。これを比エンタルピー(specific enthal-py)という。比エンタルピーを用いて断面を流体とともに通過する全エネルギーを表せば、単位質量当たり≡2 ++(J/kg)(I-2・4)となり、これは全エンタルピーと呼ばれる。全エンタルピーは輸送エネルギーの総量を示す量として便利であり、流体機械分野ではよく使用される。図I-2・1の流れ系にエネルギーの保存則を適用すれば、(断面1-2間を通過する全エネルギーの増加)=(熱入力)+(機械入力)であるから、+ρ+2 +++=+ρ+2 +(J/kg)(I-2・5)となる。式(I-2・4)で定義される全エンタルピー(J/kg)を導入すれば、上式は、=--(I-2・6)と表される。いま、変化の過程の一微小部分に着目すれば、それに必要な比仕事dは、式(I-2・5)からd=d-d=d 2 +d+d ρ -d+d=d 2 +d+d 1ρ +dρ-d+d(I-2・7)となる。ここで、熱力学の第一法則の式d=d+d1ρにおいて、d+d1ρ=外部熱d+内部熱d욆であるから、d=d 2 +dρ+d+d욆(I-2・8)これを入口1から出口2まで積分すれば、=12 ++-+dρ+욆=+욆(I-2・9)ここで、≡12 ++-+dρ(I-2・10)が有効エネルギー(available energy)であり、욆は内部摩擦によって内部に発生する熱でエネルギー損失となる。このとき、>0ならば圧縮過程(圧縮機)、<0ならば膨張過程(タービン)となる。
