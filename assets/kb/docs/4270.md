# 3・2・6 遠心圧縮機（centrifugal compressor )

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・2 遠心送風機および圧縮機各論

a.構造 遠心圧縮機は，主軸，羽根車，軸受，ケーシング，
軸封装置，駆動機などから構成されている．吸込管から入ったガス
は羽根車，ディフューザ，リターンチャネルあるいはスクロールを
通って昇圧され，次段または吐出し管へとつながる．
遠心圧縮機を構造面から分類すると，羽根車の数により単段形と
多段形に，ケーシングの形状により水平割り形（図Ⅲ-3･18）と垂
直割り形（図Ⅲ-3･19）に，主軸の数により単軸形と多軸形（図Ⅲ-
3･20）に分けられる．そのほか中間冷却を行うものや，ケーシング
の中間で抽挿気するサイドフロー形や軸流段の後に遠心段を併用す
る複合形もある．
圧力比の高い多段遠心圧縮機ではガスが高温となるので圧縮過程
の途中で中間冷却を行い，等温圧縮に近づけて所要軸動力を減らす
形式をとる場合もある．中間冷却（intercooler ）は各段ごとまた
は数段ごとに設置され，ケーシングの内部に組込まれるもの（図
Ⅲ-3･21）や外部に設置され配管でケーシングと結ばれるものがあ
る．このほか，取扱うガスによっては高温になると危険性が増すも

一般産業用としては，高圧力のガスを得るために多段かつ高速回
転のものが使われ，一つのケーシング内に10段以上の羽根車を組
込んだ例もある．さらに高い圧力を必要とする場合には，第2，第
3のケーシングを設け，ガスを直列に流すことにより所要の圧力を
得る．駆動機としては蒸気タービン，ガスタービンまたは歯車増速
装置を介した電動機が使用される．最近では高周波電動機が使われ
ることもある．
単段形のものは，小型ガスタービンや過給機の遠心圧縮機として
用いられるほか，単段形を組合わせた多軸形圧縮機（図Ⅲ-3･20）
などに使用される．
羽根車は，使用目的に応じて形式，材質，製造方法が選定され
る．羽根車の形式は，側板をもつクローズド形と，側板をもたない
オープン形に分けられる．一軸多段形圧縮機では，羽根車と静止流
路との間のすきまが熱膨張などの影響を受けるためクローズド形が
用いられ，単段形では強度上有利なオープン形が用いられる．オー
プン羽根車（open impeller ）は外径基準の周速度が750m/ s程度
で運転される場合もあり，クローズド羽根車（closed impeller ）は
一般に周速度360m/ s程度までで使用される．羽根車の材質は，
一般に低合金鋼，高合金鋼，アルミニウム合金あるいはチタン合金
（Ti -6Al -4V）などが用いられている．
製造方法には，羽根車を一体で精密鋳造する方法や，一体の材料
から5軸NC（numerical control ）機械で削り出す方法，放電・
電解加工を行う方法がある．また，接合により一体の羽根車を製作
する方法には，羽根を主板や側板に羽根側から溶接する方法，逆に
外側から電子ビーム溶接やスロット溶接を行う方法，羽根と主板ま
たは側板の間に特殊ろう箔を挟み，加熱して接合するろう付け法な
どがある．精密鋳造や5軸NC機械加工はオープン羽根車に，放
電・電解加工，溶接やろう付け接合はクローズド羽根車に用いられ
ることが多い．このようにして製作された羽根車は，設計回転速度
の10～20%の過回転で強度の確認が行われる．
ケーシングは耐圧気密構造を必要とし，吐出し圧力が6MPa程
度までは水平割り形であるが，それ以上の圧力や吐出し圧力が1
MPa級でも分子量が小さく漏れやすいガスなどの場合には垂直割
り形が使用される．水平割り形は，下半ケーシングに吸込口と吐出
し口を設け，上半ケーシングを取り外すことにより，内部のロータ
や静止流路部の開放点検が行いやすくなっているものが多い．垂直
割り形は，円筒状のケーシングにロータおよび静止流路を含む内蔵
品を軸方向に挿入して組立てられるため，片側または両側に十分な
強度をもつ円盤状のふたを設けている．これらケーシングの材質
は，水平割り形では鋳鉄，鋳鋼，特殊鋼または炭素鋼溶接構造が多
い．垂直割り形では鋳鋼または鍛鋼で吸込口，吐出し口は溶接によ
って取付けられることが多い．
主軸に羽根車を組付ける方法としては焼きばめだけのもの，アキ
シアルキーやラジアルピンを焼きばめに併用したもの，羽根車をは
めあいにて組合わせ，ボルトにて軸方向に結合したものなどがあ
る．
圧縮機を運転すると羽根車の吸込側には入口圧力が，また主板の
背側には出口圧力が作用して軸スラストが発生するので，最終段羽
根車の後に釣合いピストンを設けてスラストを低減させる（図Ⅲ-
3･21）．圧力が高い場合には，この方法に加えて羽根車列の前段側
と後段側を背向させて配列したり，羽根車を同じ向きに配列する場
合には，ダイアフラムラビリンスの直径を羽根車入口部ラビリンス
の直径と同一にして軸スラストの低減を図ることもある（図Ⅲ-3･
18）．
主軸を支える軸受としては，真円軸受，多円弧軸受（2～4円
弧），ティルティングパッド軸受（3～5パッド）などの滑り軸受を
使用することが多い．特に，高速回転で比較的軸受荷重の小さい場
合には，オイルホイップやオイルホワールを防止するために，ティ
ルティングパッド軸受が使用される．
主軸がケーシングを貫通する部分には，内部のガスが大気に漏洩
するのを防止するため，各種の軸封装置が用途に応じて使い分けら
れている．
b.性能 遠心圧縮機の選定に当たっては，比速度（specific
speed）を効率の良い範囲に選ぶことが重要である．一般には
380（m,m / min,min ）程度の比速度において良好な性能を示す
（図Ⅲ-3･22）．一軸多段形圧縮機では，順次気体が圧縮されて体積
流量が減少するため後段側ほど低比速度となり効率が低下するの
で，初段の比速度をできるだけ大きくして後段側の比速度を相対的
に大きくすることにより効率の改善を図ることが必要である．一
方，多軸多段形圧縮機では，各段の比速度を最適に近い状態に選ぶ
ことができるので効率上有利になる．
遠心圧縮機において羽根車1段当たりの圧力上昇は，羽根車の特
性および周速度により決定される．大気を吸込む場合，羽根車の外
径基準の周速度と圧力比の関係を実例から示したものが図Ⅲ-3･23

遠心圧縮機の特性は，羽根出口角度によって異なり，径向き羽根
車は後向き羽根車に比べて同じ周速度で高い圧力比を得られるが，

い．図Ⅲ-3･24にその一例を示す．
また，冷凍機用遠心圧縮機などは高圧力比，高マッハ域で運転さ
れるにもかかわらず広い作動範囲を要求されるため，一般に入口案
内羽根あるいは可変羽根付きディフューザなどが用いられている．
多段遠心圧縮機では中間冷却により等温圧縮に近づけ，所要動力
を減少させている．圧力比の大きい場合，この効果が非常に大きい
ことが図Ⅲ-3･25からわかる．中間冷却を行う多段遠心圧縮機の等
温効率は65～75%程度である．
遠心圧縮機は従来容積形の領域であった高圧の分野にも適用さ
れ，吐出し圧力65MPaの超高圧圧縮機が実用化されている．この
圧縮機の後段では流量が非常に少なくなり，比速度は100（m,
m / min,min ）以下となるため，摩擦損失やラビリンスからの漏
れ損失などが大きく，これらを低減させることが重要である．
遠心圧縮機のガス流量範囲としては，一般に15～5000m / min
であるが，6500m / minを超えるものも製作されている．
c.用途 低圧のものでは，高炉送風用，空気分離装置原料空
気用，はん用空気用，各種ガスの圧送用，極低温（－150℃）の
LNG(liquefied natural gas ）ボイルオフガス用および一般化学プ
ロセス用などがある．空気分離装置原料空気圧縮機には，軸流段と
各段中間冷却をした遠心段の複合形で吸込ガス流量7500m /min
の大容量圧縮機も出現している．また，はん用空気圧縮機は小型高
速化され，パッケージ形で使用されている．
高圧のものでは，天然ガスパイプライン用（4～10MPa），メタ
ノール合成用（6～10MPa），重油脱硫用（10～20MPa），アンモ
ニア合成用（15～35MPa）などがあり，また油田やガス田で使用
されるガスリフト用やリインジェクション用圧縮機（10～65MPa）
もある．
さらに小型ジェットエンジン，小型ガスタービン，過給機，ター
ボ冷凍機などにも遠心圧縮機が使用されている．
