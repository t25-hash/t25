# 2・3・2 圧縮性流体の場合

> **章**: 第2章 流体機械によるエネルギー変換
> **節**: 2・3 有効仕事の算定

次に、圧縮性のある気体の場合を考えてみよう。式(I-2・10)の有効エネルギーを求めるには、図I-2・2において、入口1から出口2までの流体の状態変化を知らなければならない。しかし、それは不可能であり、真の有効仕事を算定することはできない。したがって、有効仕事は一定の仮定のもと、それに対応する有効仕事として求められる。有効仕事を算出するとき、図I-2・3のように、流速も含めてエネルギー変換を考えると便利である。a. 断熱比仕事(adiabatic specific work)図I-2・4のように、せき止め状態の変化が一つの等エントロピー変化(01→02)と、それに引き続く一つの等圧過程(02→02)で構成されると仮定する。断熱(α=0)の場合、圧力比(a) 圧縮機における圧縮過程(b) タービンにおける膨張過程図-2・4圧縮機とタービンのエンタルピー-エントロピー線図図-2・5 効率の細分化2―5が1に近づくにつれて、断熱効率とポリトロープ効率は一致する。軸動力と有効仕事の比として定義される全効率ηは、機械の各部で生じる損失の合計によって決まる。効率向上のためには、損失の発生原因を分析、分類しておくことが必要である。図I-2・5のような損失の分類・細分化に対応して、効率も細分化される。
