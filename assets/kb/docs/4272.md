# 2・4・1 機械効率

> **章**: 第2章 流体機械によるエネルギー変換
> **節**: 2・4 効率の細分化

エネルギー授受とは直接関係のない部分における損失1-η욇を機械損失、η욇を機械効率(mechanical efficiency)という。機械損失は、軸受や軸封装置などに働く機械摩擦による損失(機械摩擦損失という)욇=1-η욇욞、および羽根車シュラウド外側に働く流体摩擦による損失(円板摩擦損失という)1-η욃욞η욇욞に区別される。ここで、η욇욞:機械摩擦効率、η욃욞:円板摩擦効率である。η욇욞,η욃욞を用いて機械効率η욇を表せば、η욇=η욇욞η욃욞となる。
