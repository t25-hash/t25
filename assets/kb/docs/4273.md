# 3・4・1 特徴と適用範囲

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・4 軸流送風機および圧縮機の設計

軸流送風機および圧縮機は，二重円筒の環状流路にある羽根車に
よって圧力を与えられた気体がほぼ回転軸と同心の円筒面上を旋回
しながら流れるのが特徴であり，長所として次の点が挙げられる．
(１)流路断面積の割合に大流量の気体が圧送できる．
(２)他の形式の送風機や圧縮機に比べて効率が高い．
(３)動・静翼を可動翼にすることによって高効率の作動範囲が広
くなる．
(４)構造上容易に多段にでき，また送風管の途中に簡単に挿入で
きる．
一方，短所として次の点が挙げられる．
(１)1段当たりの圧力上昇が小さく，高圧の場合多段とし，高速
回転にしなければならない．
(２)失速点が最高効率点に近く，サージングまでの領域が狭い．
(３)遠心式に比べて翼が構造的に弱く，また翼の汚損による性能
低下が著しい．
(４)同一風量風圧の場合，遠心式や斜流式より騒音が大きい．
JIS規格では，圧力比または吐出し圧力の高低により，表Ⅲ-3･1
のように軸流ファン（axial fan)，軸流ブロワ（axial blower )，
軸流圧縮機（axial compressor ）に分類している．
しかし，一般に使用されている呼称はそれほど厳密なものではな
い．例えば，発電所ボイラ用の押込通風機は2台送風の場合，吐出
し圧力は5～9kPa程度であり，1台送風の場合15～17kPa程度
になるが，これを特にファンとブロワで区別することはない．ま
た，高炉用圧縮機は圧力比が2をはるかに超えるものが多いが，送
風が主目的なので高炉用ブロワと呼ばれることが多い．また特殊な
例として，航空機用のバイパスエンジンの前方に設けた羽根車は翼
先端の周速度マッハ数は1.6，圧力比は1.55程度であるが，この
部分はファンといい，エンジン全体はターボファンエンジンの呼称
で親しまれている．したがって，ここではJIS規格に従わず，吐
出し圧力が10kPaを超えても圧縮性の影響を補正係数によって考
