# 2・4・3 流体効率

> **章**: 第2章 流体機械によるエネルギー変換
> **節**: 2・4 効率の細分化

吐出し口から吐出される流体に伝達されるη욇ηの動力のうち、流体効率(hydraulic efficiency)η욠だけが有効動力として伝達される。したがって、全体としてのエネルギー変換効率はη=η욇ηη욠となる。円板摩擦損失動力、内部漏れ損失욆욡욈욧、流体損失읏윱읏は、作動流体中で直接発生する損失であり、内部損失と呼ばれる。この損失に対応する効率は内部効率(internalefficiency)と呼ばれ、2―6流体機械η욡=η욃욞η욡욈욧η욠で与えられる。これを用いると全効率は、η=η욇욞η욄욧η욡で表される。
