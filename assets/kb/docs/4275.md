# 3・4・3 旋回失速，サージングおよびフラッタに対する配慮

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・4 軸流送風機および圧縮機の設計

a.負荷限界（loading limit ) 動・静翼列を設計する際，段
当たりの圧力上昇（または比仕事）をあまり大きくとると，翼面ま
たはケーシング-ハブ面から流れがはく離して翼列は失速する．こ
の限界を負荷限界といい，その評価は設計の際に重要である．負荷
限界の算定式については文献(49)を参照するとよい．
b.旋回失速（rotating stall ) 軸流送風機および圧縮機の流
量を設計点から減じていくと，翼の迎え角が大きくなり失速する．
圧力比があまり高くない場合，失速は翼の先端側または根元側に生
じ，流量の減少とともに半径内向きまたは外向きに伝ぱする．失速
の初期の段階では一部の翼が失速し，周方向に非対称の状態にな
る．失速した翼の領域を失速セルという．失速セルは1ないし複数
個生じ，翼の背面側へ伝ぱするので翼の回転速度よりも遅い回転速
度で旋回する．これを旋回失速という．多段圧縮機を設計値より低
回転で運転（部分負荷運転）する場合，吐出し側の気体は十分圧縮
されないので，後段の翼列が抵抗となり前段の翼の迎え角が大きく
なって旋回失速を起こす（図Ⅲ-3･30）．旋回失速が生じるとその翼
列や後段の翼列に非定常力が作用し，翼に共振を引き起こす原因と
なるので，設計の際に，①旋回失速セルの数と速度の評価および翼
の固有振動数の検討，②圧縮機中間段からの抽気，③可変ピッチ静
翼の採用，④二軸圧縮機の採用，など十分対策を講じる必要があ
る．起動時に旋回失速領域通過の際，高圧の圧縮機では翼への加振
力が大きいため，極力低回転速度で脱出するよう配慮する．
c .サージング（surging) 圧力比が低い軸流送風機では，翼
列の失速領域は流量の減少とともに広がり，低流量側で圧力-流量
特性は負こう配となる．圧力比が高い場合には失速の伝ぱ速度が速
く，同時に圧縮機の環状流路全体に失速領域が広がって圧力は急降
下する．いずれの場合も圧力-流量特性に負こう配が生じると，管
路系の減衰性が大きくない限り圧力と流量がともに激しく変動す
る，いわゆるサージング現象を生じる．特に多段軸流圧縮機はサー
ジングを起こしやすい．サージングが生じると圧縮機や管路系を破
壊するなど重大な事故を誘発して危険なので，翼列が失速しない領
域で運転するように設計することがたいせつである（図Ⅲ-3･30）．
翼の失速は先端から失速することが多いので，サージング線をなる
べく低流量側に寄せるため，ケーシングに小さい溝をたくさん設け
て失速を抑制する方法がある．これをケーシングトリートメント
（casing treatment ）と呼ぶ．ほかにも，エアセパレータ ，サク
ションリングなどの諸施策も見られる．また，サージング線よりや
や大きい流量を設定流量とし，作動点がこれに近づくとバイパス弁
で放風するなど，サージング防止制御を施すことが重要である．
d.フラッタ（flutter ) 軸流圧縮機の翼が何らかの原因で振動
した場合，それによって翼に作用する流体の非定常力が振動を増幅
する方向に働き，それが翼材質固有の減衰力よりも大きくなると，
自励振動によって翼が破壊する場合がある．この問題は航空機用圧
縮機の高出力化および軽量化に伴い生じた問題で，この現象をフラ
ッタという．翼列では作動状態においてもフラッタは起こり，これ
を非失速フラッタという．これに対して失速状態で起こるフラッタ
を失速フラッタといい，大流量の流れがチョーキング現象を生じた
際に起きるフラッタをチョーキングフラッタという．翼列を通る流
れが音速を超える遷音速フラッタや超音速フラッタでは，衝撃波と
境界層との干渉などにより大きな励振力が生じる危険性もある．し
たがって，動翼の材料が軽く回転速度が大きい場合にはフラッタの
発生の予測と翼の固有振動数の検討が重要な課題となる．軸流圧縮
機の特性曲線上で起こる可能性がある各種フラッタの領域を図Ⅲ-
3･30に破線で示している．フラッタに関する理論的ならびに実験
的研究は進歩しつつあるが（α4編「流体工学」10･4節参照），い
まだ十分ではない．
