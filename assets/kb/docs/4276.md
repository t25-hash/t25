# 3・1・1 基本構造とエネルギーの授受

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・1 ターボ形流体機械

流体中を運動する物体には，流体との速度差の二乗に比例する力
が働き，その反作用として流体は加速あるいは減速を受ける．この
流動の慣性力を利用して，物体と流体との間にエネルギー授受を連
続的に行わせるには，回転運動を利用すればよい．すなわち，回転
する羽根（動翼）の間の流路を流体が通り抜ける間に，慣性力を利
用してエネルギー授受を連続的に行わせる機械を，ターボ形流体機
械あるいはターボ機械（turbomachine）という．動翼をロータの
周りに多数取付けたものを羽根車（impeller）といい，流れが半径
方向に流出する遠心式，軸方向の軸流式，両者の中間方向の斜流
式，そして回転軸に直角に流入・流出する横流式がある．横流式は
流れが非軸対称になり，他の三つの形式とは異なるので，ここでは
遠心，斜流，軸流の3形式に限定して，共通な関係を述べる．
a.基本構造 ターボ機械は，容積形流体機械に比べて流れが
連続的で羽根車全周を一様に流れるので，寸法の割に大量の流体を
処理することができ，しかも部品点数が少なくしゅう動部も少ない
ので高速化に適している．しかし，機械が停止したときには逆流が
生じ，また規定流量からはずれると，流れのはく離や二次流れの発
生などにより損失が増大する．
羽根車の上流および下流には，羽根車に流れを導き，また羽根車
から流出する流れを吐出し管に導くために，種々の固定流路（sta-
tionary channel）が設けられる．被動機（ポンプや送風機）では，
羽根車から流出する流れは大きな旋回速度をもつことが多く，その
ままでは周速度に基づく運動エネルギーは粘性で熱になってしまう
ので，これを減速して圧力として有効に回収するために，羽根車の
下流に広がり流路（ディフューザ，diﬀuser）や案内羽根（guide
vane）が設置される．ディフューザや案内羽根にも，羽根車の形
式に応じて遠心，斜流，軸流の3形式があり，遠心式および斜流式
では全周から一様に流出する流れを集めるために，ディフューザの
周りにさらに渦巻き形のケーシング（ボリュート，volute）が設置
される．羽根車の周りにボリュートだけを設けることも多い．羽根
車の吸込み側では，流れが軸に垂直方向から流入する場合には，吸
込みケーシングが用いられる．
軸流式では，動翼と静翼の組合せが段と呼ばれる一つの単位を構
成し，多段化することで高い圧力が得られる．遠心式では，羽根車
の周りに案内羽根が配置され，案内羽根出口の径が大きくなるの
で，多段化する場合は，次の段の吸込み口まで流体を導く内向き戻
り流路が設置される．
一方，原動機（水車やタービン）では，原理的に流れ方向および
回転方向を被動機の場合と逆にすればよく，動翼に大きな運動エネ
ルギーを与えるために，羽根車の上流側には狭まり流路（ノズル，
nozzle）や案内羽根（静翼）が設置される．一方，羽根車の下流側
には，羽根車から流出する流体をさらに減速して静圧回復するため
の，広がり流路が設置されることがある．
一般に，流れを増速する場合は安定で損失は少ないが，減速する
場合は境界層がはがれやすく損失も多くなるので，被動機では羽根
車および固定流路の速度変化は緩やかで流路が長くなる．これに対
して原動機では流れを増速させるので，流路は比較的短く速度変化
も大きい．
