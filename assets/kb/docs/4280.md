# 3・1・5 ディフューザの働き

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・1 ターボ形流体機械

羽根車から流出する流れは通常は旋回速度が大きいため,下流の渦巻ケーシングや次の段に送り込むには適していないことが多い.この流れを減速して,動圧の大部分を静圧に回復するために用いられる広がり流路がディフューザ(diffuser)である.遠心式のディフューザ出口では旋回が残るので,次段の羽根車に旋回のない流れあるいは所定の強さの旋回流れを送り込むためには,ディフューザの後に戻り流路(return channel)を設置して所定の出口流れを実現させる.a. 軸流機械の静翼軸流のターボ機械では流体はほぼ円筒面上を流れるので,静翼間の流れは羽根車の場合と同様に,その円筒面を展開して得られる二次元直線翼列内の流れと同じである.しかし,静翼の上流と下流の断面における旋回強さの半径方向分布によっては,翼列を通過する間に流れが半径方向にやや移動し,それに図-3・13 軸流速度が変化する場合伴って翼列前後の軸流速度が変化することがある.同一円筒面上の軸流速度が下流に向かって増加するような場合には,図I-3・13に示されるように,軸流速度が変わらない場合(一点鎖線)に比べて転向角は増大するが旋回速度の変化量はほぼ同程度であり,式(I-なる.一方,軸流速度が下流に向かって減少するような場合には,転向角は少なくなるが圧力上昇量は大きくなり,翼列は失速しやすくなる.羽根車の羽根が先端から根元まで流れに対して一様な仕事をする場合には,羽根車が流れに与える旋回量は半径に反比例するから,その旋回速度に基づく動圧を回収する静翼の負荷は,根元で最大となる.そのうえ,翼面に発達する境界層内では根元に向かう二次流れが生じ,ボス表面の境界層内には圧力面から負圧面に向かう二次流れが生じるから,ボス付近の翼負圧面には二次流れが集積し,境界層の厚い領域ができて流れがはく離しやすい.b. stall),振動の原因となることがある.ディフューザ内に逆流が起き始めるときの平均流入角は,ディフューザ入口の流路幅と半径との比により異なり,流入速度2―13第I部流体機械一般が一様な場合,=0.05では9°,0.10で13°,0.20で17°である.羽根なしディフューザ内の流れは三次元的であるが,圧力損失および旋回強さの減衰の様子は,一次元的らせん流れの壁面摩擦によるものとして,ほぼ正しく求めることができる.流量が減少してディフューザへの流入角が小さくなるほど入口から出口までの流れの経路は長くなるので,圧力損失は大きく圧力回復率は低くなる.羽根なしディフューザの流路幅を狭くすれば水力半径は減少するが,流れ角が増大するので壁面の摩擦損失はあまり変化しない.したがって,羽根車からの流れ角が小さい場合には,羽根なしディフューザ入口部の短い区間で,ディフューザ幅を狭めて平均流れ角を増しても摩擦損失が増加することはなく,小流量では局所的逆流に基づく不安定現象を防止できる.しかし,大流量では出口動圧の損失が大きい.. 羽根付きディフューザ(vaned diffuser)通常の遠心流路は減速流路で,しかも流路幅が狭いので境界層の発達が著しく,流路は境界層で埋まる.しかも翼のアスペクト比は極めて小さく,翼列の弦節比が大変大きいので,翼列としての取扱いが難しい.そこで,ディフューザ流路の中心線に沿う断面積の分布を,広がり角が7°~12°の広がり流路の断面積分布にならって決めることが多い.もし,直線翼列を等角写像した円形翼列を用いて羽根を設計すると,出入口の減速比は,対応する直線翼列の減速比に出入口半径の相違による減速比を乗じたものとなるから,直線翼列に比べて減速の割合が大きくなり,失速限界入射角などが異なってしまう.反りのある薄い翼からなる円形翼列を用いると,翼作用により羽根面に圧力差が生じ,側壁面境界層内には高圧面から低圧面に向かう二次流れが生じる.この二次流れは全圧の小さい流体を低圧面に集積し,ディフューザでの圧力回復を妨げやすい.そのために,ディフューザを反りのある翼列とせず,直線壁の広がり流路を多数円周上に配列することも多い.広がり角一定の直線流路では,一定の出入口面積比を得るに必要な流路長さは入口部の流路幅,すなわちスロート寸法に比例するから,羽根が多いほどディフューザの外径を小さくできる.しかし,羽根が多いほど羽根前縁の厚さのために流れが増速され,またストールからチョークまでの流量範囲が狭くなる.ディフューザの性能はスロート面積によってほぼ決まる.羽根車から流出する流れが,吐出し量の減少に伴って流出角は減少しても流出速度はあまり変化しないのに対して,スロートより下流の速度は流量にほぼ比例する.すなわち,吐出し量の減少に伴って,迎え角をもってディフューザに流入する速度はあまり変化しないのに,スロート下流では減少するため,入口の半開放区間における圧力上昇率は増大し,その値が限界を超えると流れははがれて,ディフューザ性能は急に低下する.つまり,ディフューザの失速は一次元的には,平均流入速度とスロート断面の平均速度との比が限界値を超えたときに起きる.一方,羽根車からの吐出し量が増大すれば,流量に比例してスロートの流速も増大し,ついには羽根車出口の速度よりも大きくなって大きな圧力損失を伴うから,ポンプや圧縮機が性能よく運転できる流量範囲は,スロート断面積によって左右される.したがって,食違い角の大きい翼列では,スロートができない程度に弦節比を小さくとれば,広い流量範囲で安定に運転できて,小流量で高い圧力回復が得られる.また,この翼列下流に羽根なしディフューザを設ければ,大流量でもかなりの圧力回復ができる.
