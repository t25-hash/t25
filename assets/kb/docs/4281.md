# 3・1・6 戻り流路と渦巻ケーシング

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・23 )で与えられる圧力上昇は少なくなるので翼列は失速しにくく

a. 戻り流路多段ポンプや圧縮機では、ディフューザを出た旋回流れを次段の羽根車に導くために、図I-3・14に示すように軸対称曲がり流路(U ターン部)によって内向きの旋回流に変え、案内羽根を設けた戻り流路(return channel)に導いて旋回を除去2―14流体機械図-3・14 戻り流路する。軸対称曲がり流路内にはふつう案内羽根を設けない。この部分の流れは、旋回成分による遠心力と、メリディアン断面内の曲がりによる遠心力を受けて外側に偏り、内側では流れが壁面からはがれやすい。曲がり流路(U ターン部)入口で流れが内壁側に偏っていても、半径の最も大きい曲がり中央の断面では外壁側に偏り、断面内で流れの入れ替わりが行われる。曲がり流路出口では流れの混合が進み、断面速度分布は一様に近付くから、下流に設けられた案内羽根は比較的良好な流入状態になる。なお、軸対称曲がり流路の圧力損失は主として壁面摩擦によるもので、あまり大きくない。戻り流路の上流が羽根なしディフューザの場合には、戻り流路への流入角が運転状態によって変化するので、小流量では案内羽根への入射角が過大となって、流れが羽根面からはがれやすい。このため、案内羽根の前半はゆるやかに曲げて翼負荷を軽くし、後半でやや強く曲げて流れを半径方向に導くと良い結果が得られる。b. 渦巻ケーシング羽根車外周から出た流れを、直接あるいはディフューザを経たのちに集めて吐出し管に送り出すために、渦巻ケーシング[ポンプではボリュート(volute)、送風機ではスクロール(scroll)と呼ばれる]が用いられる。したがって、渦巻ケーシングの形状を、設計流量における羽根出口流れの流れ角に合わせた対数らせんにすれば、設計点において軸対称な流れが実現でき、羽根車の外周に沿って圧力が一定になり、半径スラストは生じない。実際の流れでは壁面摩擦が働くが、境界層の影響はあまり大きくなく、周速度成分はほぼ自由渦分布に近く、流れ角の半径方向への変化も比較的小さい。渦巻ケーシング内の周方向の速度を断面内で一定と近似し、設計点において羽根車から流出する流量に比例して、渦巻ケーシングの各断面の面積を定める方法もある。羽根付きディフューザの下流に渦巻ケーシングが設けられる場合には、渦巻ケーシングに流入する流れはディフューザ羽根によって定まり、流量によってあまり変化しない。一方、羽根車から直接あるいは羽根なしディフューザを経て渦巻ケーシングに流入する場合には、羽根出口流れが流量とともに変化するので、渦巻ケーシング流れも流量とともに周方向に変化するようになる。もしも渦巻ケーシング流れの速度が、流量に比例して全周で一様に変化するとすれば、流量によらずつねに軸対称な流れが実現されるはずである。しかし、羽根出口流れの周速度成分は、速度三角形から、ケーシング流れとは逆に流量の増加とともに減少するので、渦巻ケーシング流れは設計流量以外では周方向に変化するようになり、非軸対称な流れになる。すなわち、渦巻ケーシングは設計流量で軸対称流れを実現しうる渦巻形状に設計されるが、過大図-3・15 渦巻ケーシングの圧力分布図-3・16 渦巻ケーシングの羽根出口下流のメリディアン速度流量では流れにノズル作用を及ぼし、舌部(tongue、巻始め、水切りともいう)から出口のスロートに向かって速度を増大させ、圧力は図I-3・15に示すように低下する。一方、部分流量ではディフューザ作用により下流に向かって速度を低下させ、圧力は増大する。いずれの場合もスロート近くの圧力は流量によりあまり変化せず、その圧力分布に基づいて羽根車には半径スラストを生じる。この場合、羽根出口流れの周速度と渦巻ケーシング流れの速度の差は、設計流量から外れるとともに大きくなり、混合損失を発生する。正規流量において、圧力が周方向に一定となる断面積の分布をもった渦巻ケーシングよりもやや広い渦巻ケーシングにすれば、羽根車から流出した流れが渦巻ケーシングに流入するときに混合損失を生じるが、渦巻ケーシング内の平均速度が遅いために摩擦損失が少なく、渦巻ケーシングはディフューザの役目もして、ポンプ効率はやや高くなる。
