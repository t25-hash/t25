# 3・1・7 翼の傾斜:スイープとダイヘドラル

ターボ機械の翼を流れに対して傾斜させ、航空機のようにスイープ(sweep、後退角、前進角)やダイヘドラル(dihedral、上反角、下反角)をつけることは、ターボ機械の性能向上、騒音低減、小型化などに対して有力な手段である。これらを組合わせることによって、翼列自体の流れを制御できるとともに、下流にある翼列の入口流れの制御も可能である。a. スイープとダイヘドラルの定義翼のスパン方向が流れに垂直でないとき、翼はスイープをもつという。また翼面がハブとケーシングの間の子午流面(両壁面を含む)に垂直でないとき、ダイヘドラルをもつという。図I-3・17は、軸流羽根におけるスイープ角λとダイヘドラル角νを表している。ここでは、翼の負圧図-3・17 ターボ機械のスイープとダイヘドラル図-3・18 ダイヘドラル翼が誘起する流れ面と壁面が鈍角のときνを正、またスイープについては、翼の端面がスパン中央部より前にあるときλを正(図I-3・17ではハブでλが負、チップで正)としている。翼の周方向変位を周方向リーン(tangential lean)あるいはスキュー(skew)、軸方向変位を軸方向リーン(axial lean)と呼び、スイープとダイヘドラルの組合せになる。スキュー角は図I-3・17においてδで与えられ、動翼では回転方向に端面が向いているときδを正とする。b. 流れ場の制御スイープやダイヘドラルを用いれば、流れ場の圧力、速度を局所的に変えることができる。また、これにより翼列の負荷を減らし、境界層の発達や二次流れを抑制できれば、流れの失速を遅らせ、圧力損失の低減も可能になる。. 騒音の制御軸流ファンの動翼にスイープを付けると、後退翼ではスパン方向で外側に向かう流れの成分、前進翼では内側に向かう成分をもち、これに遠心力が加わる。その結果、図I-3・24に示すように、後退翼では境界層の低運動量流体が後縁まで流れる経路が長くなり、境界層が厚くなる。一方、前進翼はこの経路が短く、境界層が薄い。乱流騒音の音響パワーが翼後縁の境界層厚さに比例すると考えると、前進翼の騒音は後退翼に比べて小さくなる。また、動翼と静翼との干渉を弱くするため、両者あるいはどちらか一方にスイープを付けて両者の間隔を大きくする方法や、静翼にスイープおよびダイヘドラルを付けて騒音を低減する方法などがある。d. キャビテーションの制御航空機のスイープによる臨界マッハ数の増加に似た現象があり、ポンプインデューサの前縁部にスイープを付けることによって、キャビテーションの初生が遅れる。e. 衝撃波の制御航空機では衝撃波の発生を遅らせるためスイープが使われるが、似たような考え方はターボ機械にもある。超音速圧縮機や衝撃波の強い遷音速圧縮機・ファンでは、動翼を傾けて斜め衝撃波にして衝撃波の強さを弱めるという考えであるが、内部流れであるため、翼がケーシングの壁面や隣り合う翼と干渉を起こし、衝撃波を含む流れが複雑になる。ケーシングの壁面の影響については、図I-3・25のようなモデルがある。これは壁付近で粘性を無視すると衝撃波は壁面に垂直でなければならないという条件から、衝撃波の位置を予測するものである。図I-3・25の例では、動翼チップの衝撃波は後退翼では前進翼より上流にでき、離脱しやすい。したがって、流れが不安定になりやすく、前進翼のほうが広作動域である可能性を示している。衝撃波は騒音と関係するので、航空機のファンでは騒音低減のため、衝撃波制御にスイープ翼(swept fan)が使われる場合がある。以上の制御のほかに、部分負荷時や流れに脈動がある場合、この流れにできるだけ翼形状をマッチさせるために翼の傾斜を使うこともある。f. 実機への適用ダイヘドラルは翼高さ方向の圧力を調整し、スイープは二次流れを抑制する効果があり、またスイープ・ダイヘドラルともに翼負荷を減らすことができるなどの特徴をもつので、最近多くのターボ機械に適用されつつある。例えば、高アスペクト比の翼をもつ低圧タービンではハブの反動度が小さくなる傾向にあるが、ダイヘドラルを付けることにより圧力を上げて、これを回避するのに使われている。また、圧縮機、ポンプなどでは作動域の拡大のため、翼負荷を減らし、失速を遅らせる目的でスイープ、ダイヘドラルが使われ、特に圧縮機静翼に多く適用されている。設計に際しては、翼傾斜を過度につけるとかえって性能低下を招くので、注意が必要である。また、翼傾斜の効果を左右する翼のアスペクト比、漏れ、入口境界層厚さ、翼形状、翼厚さならびに動静翼の流れのマッチングなどを考慮しなければならない。動翼への適用には、翼強度が厳しくなるため、強度解析、材料の選定、加工技術などが重要になる。
