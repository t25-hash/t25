ターボ機械は、羽根車の回転により流体にエネルギーを与える機械であり、そのエネルギーは静圧および動圧の形で与えられる。このような点が、ターボ機械が他の往復または回転式の容積形流体機械と異なる点である。a. 相似則(law of similarity)一般に、二つの相似形状の流路をもつ流体機械において、入口・出口の流れが相似で、かつレイノルズ数やマッハ数が等しければ、二つの内部流れは力学的に相似になり、流れに関する無次元量はすべて一致し、定常な性能曲線も互いに一致する。ターボ機械にはガスを取扱うものと液体を取扱うものがあり、その定常性能に影響を及ぼす因子としては、圧縮性の有無により多少異なる。この場合、相似な流路形状をもつ二つのターボ機械A,B において、回転速度をとすると以下の単純相似則が成り立つ。b. 性能表示(performance representation)と特性曲線(char-acteristic curve)ターボ機械が定常状態で運転されているとき、回転速度、流量、入口・出口の圧力差、駆動動力などの間の関係は一義的に定まる。ポンプを例にとると、回転速度、流量、全ヘッド、駆動動力のうち二つを定めれば、ほかの二つはこれに従属して定まり、これらの間の関係を示す線図を性能曲線という。ターボ機械の性能は、従来から機種や使用目的に応じて最も使いやすい表示方法がとられており、性能曲線を無次元表示したものは特性曲線と呼ばれる。ポンプの場合、試験結果を一定回転速度に換算し、流量を横軸にとって揚程と効率を表す。c. 模型試験(model test)大型の水車やポンプなどでは、実物性能を精度良く測定することが困難な場合が多い。このような場合には、幾何学的に相似な模型を作り、その性能から実機の性能を予測する。しかし、模型を実機と幾何学的に完全相似にしても、レイノルズ数を実機と同一にできない場合が多く、また流路の表面粗さまで相似にすることは困難である。したがって、模型試験の結果から実機性能を予測するには、レイノルズ数および相対粗さの違いが性能に及ぼす差を補正する必要があり、これを性能換算という。また、ハットン(Hutton)はカプラン水車に対して=0.7,α=15を与えている。もっと正確には、は比速度や運転条件によっても異なるので、効率以外の性能の換算式も含めて精度の良い性能換算式が提案され、日本機械学会基準となっている。詳細は第II部1・3・5項参照。また、圧縮機についてはα=とし、はマッハ数の影響を含む係数、は長さと環状部高さの影響を含む係数である。なお、翼の大部分が層流境界層で覆われているような場合には、層流はく離のために損失が大きいので、効率換算は難しい。そこで、羽根車の形状や特性を代表する特性数として次式のを用い、これを比速度と呼ぶ。SI単位ではの代わりに無次元数である形式数(type num-ber)を用いる。またはが定まると羽根車の形状がほぼ決まり、その関係はおおよそ図I-3・27のとおりである。これに伴って羽根車の特性曲線の形状もほぼ決まり、その模様をポンプの場合を例にとって図I-3・28に示す。が著しく小さいと、円板摩擦が増加して効率が低下し、容積形のほうが適している。の増大とともに適応する羽根車形式は遠心式から斜流式、軸流式へと変化する。容積形流体機械(positive displacement type fluid machinery)は、シリンダとピストン、あるいはケーシングとそれに内接する可動部材であるロータなどとの間に作られる密封区間の容積が周期的に増減することを利用し、容積が増加する間に一定量の流体を吸込み、容積が減少する間にこれを吐出し側に送り出す作用をする流体機械である。この動作を繰返して行い、流体を加圧したうえで吐出し、原動機の軸動力を流体動力に変換する。この流体動力は、液体の場合には液体動力、空気の場合にはエアパワー(air power)である。
