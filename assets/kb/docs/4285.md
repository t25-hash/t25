# 3・2・1 作動原理

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・2 容積形流体機械

流体の吸込み・吐出しを繰返す部分を押しのけ室(displace-ment chamber)と呼び、それを実現する方法によって、容積形は往復式と回転式に大別される。往復式(reciprocating type)は、図I-3・30(a)に示すように、シリンダ内を往復するピストンによって容積の増減を行う。ピストンの引抜き行程では、吸込口から流体が流れ込み(受入れ行程)、押込み行程では吐出し口から流れ出る(押出し行程)。必要なとき以外には吸込口や吐出し口を閉鎖して逆流を防ぐため、チェック弁またはそれと類似な機構が必要である。ピストン、シリンダ、吸込み・吐出しチェック弁が往復式の主要部品であり、その駆動系としてクランクやカム、斜板などの機構が用いられている。回転式(rotary type)は、図I-3・30(b)の例に示すように、ロータの回転につれて押しのけ室が吸込側から吐出し側に移動し、吸(a) 往復式(b) 回転式図-3・30 容積形流体機械の分類2―20流体機械込側で取込んだ流体を吐出し側に押し出すものである。吸込み・吐出しチェック弁が省略でき、二つ以上の歯車あるいはベーンを有するロータなどの主要部分が、直接回転駆動される基本構造である。図示する構造では、ケーシング、ロータおよびベーンで区切られた押しのけ室の容積は、ロータの回転につれて増減し、1回転の半分が吸込行程に、残りの半分が押出し行程に相当する。したがって、回転式と往復式は、押しのけ室を作り出す機構が異なるだけである。流体機械の基本構造は、油用のものでは、粘性が大きく流体抵抗が大きい反面、自己潤滑性が期待しうること、気体用のものにおいては、その密度、粘性が小さく、かつ圧縮性とその状態変化に伴う大幅な温度変化を起こし、また水用のものは非圧縮性で粘性が小さく、自己潤滑性に欠けることなど、機械の設計要素が流体の物性値の相違により変化し、その基本的な構造形式をそれぞれ異にしている。
