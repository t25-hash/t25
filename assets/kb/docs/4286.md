# 3・2・2 吐出し流量

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・2 容積形流体機械

容積形流体機械の理論流量は、非圧縮性流体の場合、幾何学的な設計構造により定まる押しのけ室の最大容積と最小容積との差として求められる。図I-3・31に示すように、理解が容易な往復式についてその吐出し流量を説明する。いま毎秒のピストン往復数をとすると、=は吸込状態の体積に換算した理想的な体積流量を表し、理論体積流量と呼ばれる。この効率は容積の利用率を示し、エネルギー損失とは直接の関係はない。図-3・31 往復式の押しのけ量実際の吐出し質量流量と理論質量流量の比は、体積効率(volumetric efficiency)と呼ばれる。したがって、実際の吐出し流量の推定は、結局体積効率の推定に帰せられる。容積形の特徴は、吐出し流量がほとんどとによって決まり、吐出し圧力による変化が極めて小さいことである。
