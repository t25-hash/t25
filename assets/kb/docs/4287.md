# 3・2・3 インジケータ線図

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・2 容積形流体機械

横軸にシリンダ容積,縦軸にシリンダ内圧力をとり,容積形要素が1サイクルする間にシリンダ内の流体の状態がどのように変化するかを示したものを,インジケータ線図(indicator dia-gram)という.流体の状態変化は,ピストンの駆動方法などの条件によって異なるが,特に流体の圧縮性に大きく影響される.非圧縮性流体を扱う容積形流体機器のインジケータ線図の例を,図I-3・32(a)に示す.損失も漏洩もない理想的な場合は,一点鎖線に示す経路をたどって1サイクルの変化を行う.軸動力を加えて昇圧するとき,1→2→3→4→1の変化を行う.実際の変化は,実線に示されている.液体のわずかな圧縮性,チェック弁の開閉時期の遅れ,チェック弁を通過するときの圧力損失,吸込管や吐出し管における流体の慣性などが,理想線図とのずれを生じる原因である.図I-3・32(b)では圧縮性流体の例として,空気圧縮機における空気の状態変化を示す.実際の容積変化の繰返し速度が速いので,空気の圧縮はわずかの時間で終了し,その状態変化は等エントロピー変化(断熱変化とも呼ばれる)に非常に近い.これを踏まえ,インジケータの理論線図は等エントロピー圧縮1→2,等圧押出し2→3,等エントロピー膨張3→4,等圧吸込み4→1の一点鎖線によって描くことができる.実際の線図は,シリンダのすきま容積や各部の流体抵抗,シリンダの冷却状態などの影響でずれが生じる.
