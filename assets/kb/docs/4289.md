# 3・2・5 吐出し流量の脈動

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・2 容積形流体機械

容積形でも回転式の場合は,ロータの1回転当たりに,押しのけ室の個数だけ流体が吸込側から吐出し側に送り込まれ,吐出し流量の脈動は比較的小さい.これに対して,図I-3・30(a)に示した往復式では,流体はピストンの押込み行程中にだけ吐出され,引抜き行程中の吐出し量は全く零となる.ピストンの往復運動がクランク機構により実現されるものとすると,図I-3・33(a)に示すように,ピストン位置とクランク角θ=ωの関係は近似的に=욂+2 +2cos θ(I-3・72)で表される.圧縮性がない液体の場合に限定して考えると,瞬間流量は,0°<θ<180°の間で=-dd=-dd=πsinθ(I-3・73)となり,180°<θ<360°の間で=0となる.この関係が図I-3・33(a)に示されている.往復式の基本である単動単ピストンの場合は,瞬間最大流量は平均流量のπ倍に達する.非圧縮性流体の場合は,単動単ピストンのように吸込流量や吐出し流量の変動が激しいと,吸込管路や吐出し管路における液体の慣性によって大きな圧力変動が起こり,管路内でキャビテーションや水撃現象が誘起されていろいろなトラブルの原因となる.液体管路におけるこのような圧力変動を吸収するため,アキュムレータが取付けられる.管路内の流量変動は,アキュムレータのクッション作用により吸収され,アキュムレータの下流の流量はほぼ一定となる.圧縮性流体の場合は通常,タンクが圧縮機の出口に取付けられ,アキュムレータと同様なクッション作用をし,吐出し流量の変動を抑える.図I-3・33には,単動単ピストンと複動単ピストン,単動120°位相3ピストンの3種類の機構と,その吐出し圧力の変動を示す.そのうち,図I-3・33(c)の構成は,シリンダ数の割に脈動が著しく小さいので,最も広く実用化されている.大気圧以下の低圧を実現するための流体機械を,真空ポンプ(vacuum pump)という.圧力レベルによって真空は,低真空(大気圧~10Pa),中真空(10~10Pa),高真空(10~10Pa),超高真空(<10Pa)などに便宜的に分類される.排気機構から,吸い込んだ気体を圧縮して連続的に排出するポンプである気体輸送式ポンプ,吸い込んだ気体を表面に吸着させるなどしてポンプ内にため込み,後でポンプ外に排出する気体ため込み式ポンプに分けられる.代表的なものとして,気体輸送式ポンプでは,容積形流体機械である油回転ポンプ,ルーツブロワと同様の構造をもつルーツポンプ,油や水銀の蒸気の超音速流れで気体分子に運動量を与えて排気する拡散ポンプ,回転翼によって気体分子を低圧側から高圧側にたたき出して排気するターボ分子ポンプなどがある.気体ため込み式ポンプでは,極低温表面に気体分子を凝縮させて排気するクライオポンプ,チタンなどの活性の強い金属表面に気体分子を吸着させて排気するゲッタポンプなどがある.ここでは,他の流体機械にはない特徴をもつターボ分子ポンプ,クライオポンプの作動原理を示す.
