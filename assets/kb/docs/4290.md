# 3・3・1 ターボ分子ポンプ

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・3 真空ポンプの作動原理

ターボ分子ポンプ(turbo molecular pump)は、(1)油分子を含まない清浄な超高真空が比較的容易に得られる、(2)ガスの種類によらず、広い圧力範囲にわたり排気速度がほぼ一定である、(3)気体輸送式ポンプであり、真空容器内に導入した大量のガスを連続して排気することができる、などの特徴を備えているため、高真空を用いる研究分野や半導体製造工業等、さまざまな分野で広く使用されている。ターボ分子ポンプは、分子の熱速度と同程度の速さで回転する動翼と、翼の角度が逆向きの静翼を、十数段にわたって組合わせた構造になっている。作動原理を説明するために、図I-3・34に示すような簡単な翼列からなる系を考えてみよう。翼列は下向きに動いているとすると、上流または下流側から翼列に入った分子の多くは動翼の下面に衝突する。固体表面に衝突した気体分子は拡散反射する(表面に衝突した分子は固体と十分なエネルギー交換をし、その表面温度に相当する速度で、等しい確率ですべての方向に飛び出していく)とすると、翼の傾きに応じて気体分子が下流側に移動する確率は大きくなる。上流側1から分子が入射する場合の通過確率をΣ、下流側2から入射する場合の通過確率をΣとする。この差Σ-Σが排気速度となる。これらの確率の詳細な解析は数値解析によらざるを得ない。速度比が正の領域はΣを、負の領域はΣを表す。ここでは排気速度効率と呼ばれ、実際に単位時間に翼列を通過していく分子の流れと、そのとき翼に入射する分子の量との比で、排気の効率を表す値図-3・34 ターボ分子ポンプの翼列モデル図-3・35 翼列に入射する分子の通過確率図-3・36 クライオポンプの構造図-3・37 低温における気体の蒸気圧である。上流、下流で温度は等しいとすれば、圧力を,として、次式が成り立つ。また、=、すなわち翼による圧力上昇がない場合には排気速度効率は最大値=Σ-Σをとる。圧縮比を大きくするためには、ΣとΣとの比を大にすればよく、排気速度を大きくするには=Σ-Σを大きくすればよいことがわかる。多段の翼列をもつポンプでは、上流側の数段は圧縮比を犠牲にしても大きい排気速度をもつようにし、下流側の段では排気速度よりも圧縮比を大きくするように設計すると、能率が良いことがわかっている。
