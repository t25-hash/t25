# 3・2・2 アクチュエータの分類とその特徴

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・2 アクチュエーション技術

アクチュエータの分類は動作原理によるものが一般的であり，従
来型と新原理アクチュエータに大別される．従来型アクチュエータ
は電気や流体のエネルギーを機械エネルギーに変換するものであ
り，それぞれについて小型・軽量化，高効率化，制御性能向上など
が進められている．新原理アクチュエータは新しい材料や現象を用
いたアクチュエータであり，圧電素子や形状記憶合金などを用いた
アクチュエータが実用化されている ．さらに，機能性流体や高
分子材料を用いたアクチュエータの研究開発が進められている．
a.電気アクチュエータ（electric actuator ) 電気アクチュエ
ータは電流と磁界の相互作用（フレミングの左手の法則）により生
じる電磁力により機械的運動を得るものであり，回転運動を行う電
気モータと直線運動を行う電気リニアモータに大別される．電気モ
ータは単なる動力用あるいは運動制御用として使用され，後者の用
途に用いるモータはサーボモータと呼ばれる．
電気サーボモータには，直流電源で動作する直流（DC:direct
current ）サーボモータ，交流電源で動作する交流（AC:alternat-
ing current ）サーボモータおよびステッピングモータがある．
ステッピングモータは一つの入力パルスに対して一定の角度だけ
回転するモータである．その特徴は開ループ制御ができることであ
り，位置検出器を必要としない．入力パルスに同期して回転し，停
止時にはモータ自身に高い保持能力があるため，簡便に高精度の速
度や位置の制御を行うことが可能である．これに対し，DCサーボ
モータやACサーボモータを用いた制御系は位置検出器を備えたフ
ィードバック制御方式となる．

b.油圧アクチュエータ（hydraulic actuator ) 油圧アクチュ
エータは圧油が有する流体エネルギーを直進，回転および揺動運動
の機械エネルギーに変換するものであり，それぞれシリンダ，モー
タおよび揺動型アクチュエータと呼ばれる．出力パワーが大きい，
パワー密度が大きく小型化が可能，力-質量比およびトルク-慣性比
が大きく高速な応答が可能，高精度の位置や速度の制御が可能など
の特徴を有する．
アクチュエータの動作速度はアクチュエータに供給する圧油の流
量により制御でき，油の流量は油圧制御弁によって調整される．油
圧制御弁に電気油圧サーボ弁を用いた制御系は電気油圧サーボ系と
呼ばれ，各種の産業用ロボット，工作機械，建設機械，航空機や船
舶，自動車など広範な分野で使用されている．作動油の温度や汚染
管理の必要性，油漏れによる環境汚染，油圧タンクなどの付帯設備
によるスペース効率の低下などの欠点があり，使用環境が制限され
ることがある．
c .空気圧アクチュエータ（pneumatic actuator ) 空気圧ア
クチュエータは圧縮空気のエネルギーを機械エネルギーに変換する
ものであり，直線運動をする空気圧シリンダが代表的である．ま
た，空気圧モータ，揺動型アクチュエータ，ベローズ，ダイアフラ
ムなども使用される．
空気圧アクチュエータは手軽に利用できるアクチュエータとして
生産システムの自動化・省力化，各種産業機械，輸送機械など多分
野で使用されている．空気圧アクチュエータは発生力の調整が容易
であり，空気漏れがなければエネルギー消費なしでクランプ状態を
維持できるため，ワークの搬送や端点位置決め，クランプ作業，ロ
ボットハンドなどに多用されている．
空気圧シリンダが代表的であり，基本構造は従来から大きな変化
がないが，小型・薄型化が進むとともに，ロッドレスシリンダや低
摩擦型，低速型などの特殊用途向けシリンダが開発されている．ま
た，方向切換弁やガイド機構，ロック機構などを組込んだ複合型シ
リンダも一般的になっている．ガイド付きシリンダはテーブルやガ
イドとシリンダが一体化され，設置時の平衡度などを気にすること
なく搬送装置や位置決め機構が構成できる．

特性を有し，圧力の調整によって発生力を容易に制御できる利点が
ある．また，シリンダなどの従来型アクチュエータに加えて空気圧
ゴム人工筋が福祉・介護分野などへの応用を目的とした次世代アク
チュエータの一つとして注目されている．
