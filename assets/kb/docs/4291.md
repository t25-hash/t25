# 3・3・2 クライオポンプ

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・3 真空ポンプの作動原理

クライオポンプ(cryopump)は,液体ヘリウム,液体窒素,あるいは小型冷凍機を用いて極低温の面(クライオパネル)を作り,その表面に気体を凝縮固化し,さらに水素,ヘリウム,ネオンを極低温の収着剤に収着させることにより,真空を作り出すポンプである.極低温面に,気体を凝縮させることによって行う排気作用をクライオポンピング(cryopumping)という.クライオポンプは気体の捕集面として滑らかな面(研磨された銅面など)を使ったポンプで,モレキュラーシーブなどの多孔質吸着剤の表面を利用するものはクライオソープションポンプ(cryosorption pump)と呼ばれている.図I-3・36に代表的なポンプの構造を示す.ポンプケースの中心に冷却された第1段シリンダと,これに続く第2段シリンダがあり,それぞれの先端部を第1段コールドヘッド,第2段コールドヘッドと呼ぶ.第1段コールドヘッドには80K シールドおよび80K バッフルが取付けられ,第2段コールドヘッドにはその内部に活性炭を付けた15K クライオパネルが設置されている.水のように蒸気圧の低い気体は80K バッフルで凝縮され,CO,O,N,Arなどの一般の気体は15K クライオパネルの外面で凝縮固化して排気される.H,He,Neは15K でも凝縮しないので,15K パネルに接着してある活性炭で収着排気する.低温における気体の蒸気圧は図I-3・37に示すようになってお2―23第I部流体機械一般り,20K に冷却された面が存在すると水素,ヘリウム,ネオンが残り,4K の面が存在すると水素,ヘリウムが残るのみで,他の気体は凝縮して極高真空ができることになる.また,冷却面に衝突する分子の凝縮係数が1であれば,面に到達した気体分子はすべて凝縮し,小さい冷却面で大きな排気速度が得られることになる.そのときの排気速度は,冷却面の面積をとすると,2πとなる.文献(1) Busemann,A.,ZAMM ,8(1928),372.(2) Wiesner,F.J.,Trans.
