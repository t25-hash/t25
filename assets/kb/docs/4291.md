# 3・3・2 クライオポンプ

> **章**: 第3章 流体機械の作動原理と性能
> **節**: 3・3 真空ポンプの作動原理

クライオポンプ（cryopump）は，液体ヘリウム，液体窒素，あ
るいは小型冷凍機を用いて極低温の面（クライオパネル）を作り，
その表面に気体を凝縮固化し，さらに水素，ヘリウム，ネオンを極
低温の収着剤に収着させることにより，真空を作り出すポンプであ
る．極低温面に，気体を凝縮させることによって行う排気作用をク
ライオポンピング（cryopumping）という．クライオポンプは気
体の捕集面として滑らかな面（研磨された銅面など）を使ったポン
プで，モレキュラーシーブなどの多孔質吸着剤の表面を利用するも
のはクライオソープションポンプ（cryosorption pump）と呼ばれ
ている．図Ⅰ-3･36に代表的なポンプの構造を示す．ポンプケー
スの中心に冷却された第1段シリンダと，これに続く第2段シリン
ダがあり，それぞれの先端部を第1段コールドヘッド，第2段コー
ルドヘッドと呼ぶ．第1段コールドヘッドには80Kシールドおよ
び80Kバッフルが取付けられ，第2段コールドヘッドにはその内
部に活性炭を付けた15Kクライオパネルが設置されている．水の
ように蒸気圧の低い気体は80Kバッフルで凝縮され，CO，O，
N，Arなどの一般の気体は15Kクライオパネルの外面で凝縮固
化して排気される．H，He，Neは15Kでも凝縮しないので，15
Kパネルに接着してある活性炭で収着排気する．
