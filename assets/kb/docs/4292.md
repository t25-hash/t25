# 4・3・1 作動点の静的安定性

> **章**: 第4章 流体機械の運転と不安定現象
> **節**: 4・3 不安定現象

図I-4・7に示すポンプ系の作動の静的安定性(static stability)を考える.ここでは管路中央のタンクがなく,ポンプ通過流量φ욣とバルブ通過流量φ욃が等しい場合を考える.まず,図I-4・7(a)で負荷曲線に実揚程のないψ욃φ욃の場合に,平衡点A から流量がわずかに増加すると,抵抗ψ욃φ욣がポンプ揚程ψ욣φ욣すなわち駆動力を上回り,通過流量を減少させる.このときψ욃φ욃に対して作動点A は静的に安定である.次に負荷曲線が実揚程をもつψ욃′φ욃の場合を考える.このとき平衡点A から流量がわずかに増加すると,ポンプ揚程ψ욣φ욣,すなわち駆動力が抵抗ψ욃′φ욣を上回り通過流量をさらに増加させ,作動点はH に移動する.すなわちψ욃′φ욃に対して作動点A は静的に不安定である.したがって,2―25옿ψ욃옿φ>옿ψ욣옿φのとき平衡点は静的に安定,そうでない場合は不安定となる.ψ욃φ욣に対するA 点,ψ욃′φ욃に対するG,H 点は静的に安定,ψ욃′φ욃に対するA 点は不安定となる.
