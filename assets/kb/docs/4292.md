.課題 数多くの特徴を有し，実現すればメカトロニクス
設計にブレークスルーをもたらす多次元アクチュエータであるが，
表3･4に示したさまざまな駆動原理のアクチュエータを含め，多次

以下，個別の要素技術について課題を示す．
(１)設計・解析（design/analysis ）技術
多次元アクチュエータの構造・自由度・駆動原理は多岐にわた
り，十分な調査が行われていない．今後の可能性も含め，これらの
多様なモータの特徴をまとめることが重要である．
三次元電磁場解析などで実用的な結果が得られるようになり，多
次元アクチュエータの構造を決定することができれば最適設計が行
えるまでになりつつある．超音波振動の三次元解析による多次元超
音波アクチュエータの最適設計も可能になった．
(２)支持・案内（support /guide）技術
多次元アクチュエータの実用化には，固定子に対して可動子を多
次元運動可能に支持・案内する機構が必須である．球面ベアリン
グ，空気浮上，磁気浮上など用途ごとにさまざまな機構が提案され
ているが，より実用的な支持・案内機構の開発が切望されている．
マイクロアクチュエータの分野においてはヒンジや静電力などさ
まざまな支持方法がある．マイクロ支持機構を集積して通常サイズ
の多次元アクチュエータを支持する方法もある．
(３)センサ（sensor ）技術
球面の位置や速度情報のセンシング方法が確立していない．ジン
バル機構の交差部に一軸のセンサを用いる，ホール素子を用いるな
どの方法が提案されているが，実用化にはまだ遠い．非接触三次元
空間内座標計測法を応用できる可能性がある．
(４)駆動・制御（drive/ control ）技術
多次元アクチュエータは動きそのものが最も重要な出力であり，
高精度の駆動・制御を適用する必要がある．しかし，センシングが
難しいこともあり，フィードバック制御などを組込むのは今後の課
題となっている．

.光アクチュエータ（optical actuator ) 光アクチュエー
タは，誘導ノイズを発生しない，耐電磁ノイズ性が高い，非接触接
続が容易などの利点を有し，航空機，ロボット，マイクロマシンな
どへの応用が期待されている．現在，光アクチュエータの確定した
定義は存在しないが，例えば光エネルギーの変換過程に注目した場
合には，次の3タイプに分類できる ．(１)トリガ型光アクチュ
エータは光信号を光電変換素子により電気パワーなどに変換して使
用し，光を駆動エネルギーとして用いない．(２)変換型光アクチュ
エータは種々の現象を用いて光をいったん別の種類のエネルギーに
変換したのち，最終的に力学的な駆動エネルギーを発生する．エネ
ルギー変換には熱膨張などの光熱現象，粘性などの化学的変化，光
電効果などの電気的変化が利用される．(３)直接型光アクチュエー
タは光が直接に力学的な駆動エネルギーに変換されるものであり，
光の放射圧を利用したレーザアクチュエーションなどがある．それ
ぞれに一長一短があるが，変換型光アクチュエータが光アクチュエ
ータとして応用範囲が広く，最も実用的である．
変換型光アクチュエータの一つとして，図3･113に示すような光
圧電素子［PLZTセラミックス，(Pb,La)(Zr,Ti )O ］を利用した
光アクチュエータが開発されている．PLZTはPZT［チタン酸ジ
ルコン酸鉛，Pb( Zr,Ti ) O ］にLaを微量添加したものであり，こ
れを薄板状に形成して光を照射すると分極方向に高い電圧が発生す
る．この電圧がPLZTの圧電性に作用してひずみ（光ひずみ効果）
が生じることにより，光エネルギーが力学的エネルギーに変換され
る．PLZT素子の変位はPZT素子と同程度であり，実用化のため
にはバイモルフ構造などによる変位の拡大が必要である．これによ
りかなり大きな変位が得られるが，応答速度は遅い．この遅れは光
起電圧効果の応答遅れに起因する．

柔軟でバイオミメティックなアクチュエータであり，人工筋肉
