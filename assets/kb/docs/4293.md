# 4・3・2 サージ

> **章**: 第4章 流体機械の運転と不安定現象
> **節**: 4・3 不安定現象

サージ(surge)の数学的取扱いはα4編「流体工学」10・1・5項で述べられているので、ここではその物理的説明を行う。図I-4・7のようにポンプと出口バルブの間にタンクがあり、バルブ抵抗がψφで与えられる場合(負荷曲線に実揚程がない場合)を考える。このとき、平衡点A は静的には安定であるが、動的には不安定となりうる。平衡点A で運転中にψだけタンクの圧力を増加させると、圧力差ψ-ψ>0によってポンプ流量が急減少し、ポンプ作動点はB 点に移る。一方、バルブ流量はφとなり、単位時間当たりφ-φだけタンク内の液量は減少する。C点に達し、タンク圧力がψよりわずかでも減少すると、ポンプ通過流量はψ-ψ>0に駆動されて急増し、作動点はD 点に移る。これをサージと呼ぶ。以上のように、平衡点A は静的には安定であるが、動的には不安定である。入口管の流体の慣性が大きい場合には、図I-4・7(b)のようにポンプ出口圧力(タンク圧力ψ)は、慣性分だけポンプ揚程ψと異なる。が大きく慣性の影響が小さい場合には、図I-4・7(a)のE→F,C→D のようにわずかの圧力差で、瞬時にポンプ流量が変化する。これに対してが小さく慣性の影響が無視できない場合には、ポンプ流量が変化している間にタンク液面も変化し、図I-4・7(b)の破線のようなリサージュを描く。が極めて小さい場合には、一点鎖線のように作動点は平衡点A に収束する。α4編10・1・5項に示されるように、サージが成長するための条件は次式で与えられる。はに比例するので、同じシステムでも回転速度を増加させるとサージが発生することがある。以上より、サージが発生するためにはdψdφ>0の右上がり特性(rising characteristic)をもつこと、流体を蓄え圧力が貯蔵量に従うタンクをもつことが必要であることがわかる。管路系がタンクをもたない場合にも、ポンプでは配管内に空気がたまるとその部分がタンクの役割を果たし、サージが発生することがある。キャビテーション(cavitation)が発生したときには、ポンプ性能に右上がり特性がなくタンクもない場合でも、サージと同様の現象が発生する。これは、ポンプ流量が増加すると翼に対する迎え角が減少し、これに伴ってキャビティ体積も減少し、この減少を埋めるべくポンプ上流の流量がさらに増加することがその発生原因である。通常のサージと発生原因が全く異なるので、これをキャビテーションサージ(cavitation surge)と呼ぶ。これが、タンク要素がなくてもキャビテーションサージが発生しうる原因となっている。一般に>0であるので圧力が上がればキャビティが縮み、キャビティは圧力変動を緩和する効果をもつ。
