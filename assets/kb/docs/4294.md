# 4・3・3 旋回失速

> **章**: 第4章 流体機械の運転と不安定現象
> **節**: 4・3 不安定現象

図I-4・8に示す全体特性をもつ羽根車が図中の負荷曲線のもとで作動し、静的に安定な作動点A で平均流量がφに保たれている場合を考える。いま、管路出口の抵抗と羽根車の間に一定圧力ψに保たれた空間があるとする。このとき作動点A は局部的に不安定で、羽根車の一部は失速状態にあるB 点、他の部分では非失速状態にあるC点で作動し、平均流量がφとなるようこれらの部分の割合が決まる。これが旋回失速(rotating stall)の発生原因である。羽根車周りの流れの状態を図I-4・9に模式的に示す。翼1が失速状態にあるとき、流れはこの部分を迂回するように流れ、翼0に対する迎え角は低下、翼2に対する迎え角は増加して、失速領域は翼1から2に移動する。これが旋回失速の伝ぱ機構である。図I-4・9から明らかなように、失速領域は通常羽根車と同方向に羽根車より遅い速度で回転する。旋回失速の発生条件はdψ욧욦dφ>0で与えられる。ただし、ψ욧욦は入口全圧、出口静圧に基づく揚程係数である。旋回失速は羽根車だけでなくインレットガイドベーン(inletguide vane)、羽根付きディフューザ(vaned diffuser)、羽根なしディフューザ(vaneless diffuser)などでも生じる可能性がある。また、これらの要素が近接して配置される場合には、これらが結合した状態で発生する場合がある。そのときには関係する要素全体の特性がその発生を支配する。旋回失速は右上がり特性dψ욧욦dφ>0によって生じる局所的不安定現象である。キャビテーションのもつ不安定機構によっても旋回失速と類似の現象、すなわち旋回キャビテーション(rotatingcavitation)が発生することがある。その発生条件は>21+σφで与えられ、キャビテーションサージとほぼ同様、設計点付近の非失速領域でも発生する。旋回失速と異なり、キャビティの発生領域が羽根車の1.1~1.4倍程度の速度で回転する。キャビティの長さが翼列ピッチの65~80%に達すると、キャビテーションサージ、旋回キャビテーションのほかにも多くの種類の不安定モードが発生しうる。そこで、キャビテーションを発生しながら運転されるインデューサでは、これらの不安定現象が完全に防止されていることを確認する必要がある。2―27第I部流体機械一般対象とする現象の波長が管長に比べて十分に大きい場合や、擾乱が管内を伝わる時間が状態変化の時間スケールに比べて十分に小さい場合には、4・3節で仮定したように、管内の流体は剛体的に振動するとして扱える。しかし、それ以外の場合には、管内を擾乱が有限な速度で伝わるとした動的な解析が必要である。管内流れの動的解析には、α4編10・1・2項、10・1・3項で述べられたような特性曲線法(method
