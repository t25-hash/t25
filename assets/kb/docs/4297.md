# 5・1・3 軸系の振動

> **章**: 第5章 流体機械の振動と騒音
> **節**: 5・1 振動

回転体、軸受、シール、ケーシング、架台等からなる系を軸系と呼んで説明する。流体機械にとって軸系の振動は、流体機械が所期の機能、性能を実現するために、必ず解決しなければならない重要課題である。表I-5・1に示すように種々の振動問題が現実に発生する。a. 軸系の危険速度流体機械にとって不釣合いや流体的な加振力は必ず存在するので、軸系に振動が発生する。振動を許容値内に収めるためには、これらの加振力を減少させるか、一定の加振力に対する振動応答、すなわち振動感度を低減させるかが必要である。加振力の低減は、釣合せや低流量運転の回避により達成すること表-5・3ディスクタイプの振動が励起されるベーン組合せランナベーン枚数直径節数ガイドベーン枚数×●××××●×●××××××××●●×××●×●:各枚数の組合せで生じる直径節パターン×:圧力のパターンが釣合って直径節が生じない組合せができるが、全運転領域で可能とは限らない。したがって振動感度の低減が望ましい。軸系の振動感度が高いのはほとんどの場合共振によるものなので、共振をいかに回避するか、また可変速機等の場合、共振倍率(Qファクタ、Q-factor)をいかに下げるかが重要となる。共振を避けるためには、危険速度(critical speed)の正確な推定が必要であり、これには軸のデータ、軸受のデータ、軸受支持のデータが正しく考慮されなければならない。流体機械では、流体の付加質量やシール等のばね特性も危険速度の特性に大きな影響を与える。しかし、基本的には軸と軸受の最適な設計が最も重要であり、これによりQファクタを抑え、かつ共振も適切に回避することができる。図I-5・4にQファクタ判定規準を示す。これは横軸が危険速度/使用回転数で、縦軸がその危険速度でのQファクタをプロットしたものである。これにより低感度設計が可能となる。図I-5・4のQファクタ判定規準度をベースに使用回転数上での応答規準を規定したのがISO
