# 3・3・4 配線・接続

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・3 実装技術

配線（wiring）は個々の部品間を接続し，信号，電力を伝達す
るために使用される．半導体チップ内のトランジスタ間，プリント
基板上の電子部品間，ハウジング内のプリント基板間，ハウジング
間，インタネットというさまざまなレベルの配線・接続があり，全
く異なる技術が使用される．ここではプリント基板間，ハウジング
間の配線を中心に説明する．
a.コネクタ（connector ) コネクタは部品，配線の脱着を容
易にするためのものであり，電圧，電流，周波数，信号数，かん合
強度等を考慮して選定する必要がある．また，メカトロシステムで
はコネクタの接点に振動，衝撃が加わらないように注意する必要が
ある．表3･11にコネクタの種類を示す．
b.電線（electric wire，electric cable) 電線は電気信号を
比較的長距離にわたり伝えることが目的であり，表3･12に示すよ
うに多種類が存在する．電圧，電流，周波数特性，特性インピーダ
ンス等を考慮して選定する必要がある．電線被覆の難燃性，燃焼時
の有毒ガス発生等についての規制が存在するので注意する必要があ
る．
c .プリント基板用ラック 複数のプリント基板を使用する場
合，図3･139 のようなラックを使用する．ラックの背面にバッ
クプレーンと呼ばれるコネクタが多数搭載されているプリント基板
を取付け，スロットと呼ばれる部分にプリント基板（ボード）を挿
入してコネクタでかん合させ，挿入したプリント基板間を電気的に
接続する．ラック，バックプレーンは規格化されており，その規格
に適合するプリント基板のみスロットへ挿入できる．表3･13に代
表的なプリント基板間接続のための規格を示す．
d.フィールドバス（field bus ) 複雑なシステムでは一箇所に
大規模なラックを設けるのではなく，図3･140のように小さい規模
の複数個のラック，もしくはプリント基板に分割し，その間をフィ
ールドバスと呼ばれる通信ラインで接続することが多い．このよう
にすることで配線数が削減でき，センサ等の劣化しやすい信号を直
近で処理可能となる．
フィールドバスで伝送される情報はディジタル化されており，ア
ナログ信号に比べ，距離が長くなっても劣化しにくい．また，まれ
に電磁ノイズ等によりディジタル値が変化してもその誤りを検出す
ることが可能で，少量の変化であれば訂正可能な仕組みが設けられ
ている．また，誤りが検出されても訂正できない場合に再度，伝送
する仕組みが設けられている．このような仕組みを通信プロトコル
と呼び，種々のものが規格化されている．表3･14に代表的な規格
を示す．
