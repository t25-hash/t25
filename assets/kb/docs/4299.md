# 3・3・5 放熱

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・3 実装技術

a.放熱（heat radiation）の必要性 電子部品は以下の理由
から放熱が必要である．
(１)温度により電子部品のパラメータが100ppm/℃のオーダで
変化するため，誤差が増加し，誤動作につながる．
(２)温度が10℃上昇すると電子部品の劣化速度，故障の確率が2
倍になる．
(３)電子部品の温度が125℃程度を超えると恒久的な破壊に至
る．
b.熱設計 個々の電子部品について式(3･39)の関係を満たす
必要がある．

(１) の高い電子部品を採用する．しかし，一般にこのよう

(２)ハウジング内機器の消費電力削減，機構を含めた効率改善

グ素子にオン抵抗が小さくスイッチング時間が短いものを採用する
こと，機構の摩擦を低減させることなどが考えられる．
(３)ハウジングの放熱効率を以下のように向上させる．
① ハウジング内空気の上下の対流を考慮して機器を配置する．
② 自然対流による換気が行われやすいようにハウジング下部に吸
気口，上部に排気口を設ける．
③ 換気ファンは自然対流，粉じんを考慮して送風の向きを決め
る．
④ 密閉式ハウジングの場合，ハウジング壁面からの放熱効率を向
上させるため，ハウジング内の空気が壁面とよく触れるよう
に，内部に空気循環ファンを設ける．
⑤ 密閉式ハウジングで高い放熱効率が要求される場合，ハウジン
グの壁面に放熱器（radiator ）に取付け，この放熱器の内側，
外側にファンを設けて熱交換効率を向上させる．
(４)電子部品に放熱器等を取付けることにより温度上昇を抑え
