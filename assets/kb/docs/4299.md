# 5・2・2 騒音の伝搬

> **章**: 第5章 流体機械の振動と騒音
> **節**: 5・2 騒音

a. 空気伝搬と固体伝搬音は流体中(大気中)ばかりではなく、固形物体(固体)内も伝搬する。騒音源から空気中に放射された音が、空気中を疎密波として伝わって聞こえる場合が空気伝搬音(air-borne noise)である。一方、騒音源に接続している固体内を弾性波として伝搬し、騒音源から離れた受音者の近傍の固体から空気中に放射されて伝搬する音が固体伝搬音(solid-borne noise)である。流体機械から発生する騒音が空気伝搬音だけであれば、その伝搬・減衰の様子はかなり精度良く予測できる。特に吐出し口噴流音や送風機の吸込口騒音のように、音源が伝搬する空気と直接つながっている場合は、音波のエネルギー評価的な伝搬理論に従うと、容易にその特性が理解できる。一方、流体機械の架台や配管部材などの構造体を伝わってくる固体伝搬音は、未解明な部分が多い。ケーシングや配管、さらには建物構造物を音波や振動が伝搬してくるので、空気伝搬音を低減対策した後で問題になることが多い。b.
