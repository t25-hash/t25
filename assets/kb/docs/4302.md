# 5・2・5 予測と低減対策

> **章**: 第5章 流体機械の振動と騒音
> **節**: 5・2 騒音

a. 騒音の予測方法流体機械の騒音を予測するのであるから、CFD(computational fluid dynamics、数値流体力学)を応用して、流体力学的発生音の音源に相当する非定常流体の諸量を予測することで、機械からの放射騒音が予測可能と期待される。しかし、2005年の時点ではこのような騒音予測方法は実用的段階としては未完成であり、経験的に確立された予測式が各種の流体機械に応じて作られ、用いられている。その詳細は各流体機械を説明する章に任せ、ここでは経験的予測手法の基本になる共通項目を記す。前出の5・2・1項に記されているように、流体力学的な原因による騒音の発生源の強さは、原因となる現象の代表流速の四乗から八乗に比例する。一方、予測騒音レベルを流体機械の出力を用いて表現する場合に、単位出力の機械の発生騒音レベルを[dB(A)]とすると、次式で予測することもできる。比騒音レベルや基準騒音レベルは、機種に応じて経験的に求められている。これらが小さい流体機械であるほど低騒音である。以上のような経験的な予測手法では騒音スペクトルを詳細に予測することは困難であるが、機種に応じて相対的1/1オクターブバンドスペクトルが統計的に求められている。また、音響パワーレベルを予測する場合には、式(I-5・11)のような方法を応用するとよい。b. 騒音源の同定騒音が大きい流体機械を静かにするための騒音対策を行うときには、事前に対象とする流体機械のどの部位からどのような流体の現象に、あるいは機械力学的な現象に原因して騒音が発生するのかを知る必要がある。騒音源の同定は、流体機械の運転条件(回転数または流量)を変えて、発生騒音のスペクトルの変化の傾向を解明することで可能になる。特徴的な周波数成分(周波数およびその音圧レベル)が運転条件の変化に伴って比例するのか、変動するのか、変化しないのかなどを把握して、各種の騒音発生現象と照合する。さらに、音響インテンシティプローブで対象機械周囲の音響インテンシティ(acoustic intensity)分布を計測し、インテンシティベクトルの変化から発生源の部位を同定する。または、対象の機械の表面各部の振動加速度信号と騒音信号とのコヒーレンス関数を求めて、影響度が大きい部位を同定する。これらの計測手法は騒音発生源だけではなく、騒音の伝搬経路の同定にも有効である。c. 伝搬系での騒音対策機械の騒音は、発生源で低減することが省資源・省エネルギーの観点から望ましいが、発生源対策が困難な場合には、騒音の伝搬経路上で低減対策を実施する。騒音伝搬経路は複数あることが通常であり、それらの影響度を同定して対策の順番を決める。空気伝搬音と固体伝搬音では対策方法が異なる。流体騒音も通常の空気伝搬音と同様に周囲へ伝わるので、その伝搬過程で減音を行う方法は、一般的に行われている防音対策と同様である。すなわち、遮音・吸音・消音という音波だけの物理的な性質を利用した対策手法であり、受動的な制御手法と能動的な制御手法がある。空気伝搬音は遮音すると低減効果が大きいが、流体騒音の伝搬系制御で無視できないことは、音が伝わる空気経路に流体を流すことが一般的であり、遮音対策が困難なことである。ダクトや配管に接続する騒音伝搬の低減装置が消音器(silencer)であり、受動的消音器としては特定の周波数成分を低減するために共鳴型を用い、幅広い周波数帯域の成分を低減するために吸音型を用いる。一方、最近のディジタル技術の進歩によって実用化されている能動的消音器(active silencer)の基本構成は、図I-5・14に示すようなものである。受動的消音と能動的消音は、それぞれが得意とする周波数範囲が異なるので、ハイブリッドの形で利用することが望ましい。なお、能動騒音制御・能動消音の詳細に関しては文献(13)(14)が参考になる。ポンプや圧縮機などのケーシングから放射される空気伝搬音は、騒音が伝搬しないように固体壁や固体膜で音源を囲んで遮音する。音響インピーダンス(acoustic impedance)が大きく異なる境界面では、透過できる音波が少なくなるので、固体壁や膜で覆うことで遮音ができる。空気伝搬音に対しては面密度が大きい固体壁ほど遮音量が大きく、周波数が高い音ほど透過量が少ない。防音カバー(noise insulation cover)も遮音装置の一つである。騒音源の機械を単純にカバーで囲うと、カバー内が拡散音場になって内部で音響エネルギーが蓄積増加するので、十分な遮音効果を実現できない。カバー内に吸音材を張り付けて、内部を自由音場に近づけることがキーポイントである。また、騒音源の機械が発熱するものであれば、自然換気ができるような消音性の流路を設けたカバーにすることも重要である(図I-5・15)。固体伝搬音(solid-borne noise)は未解明な課題が多いが、人間のごく近傍で騒音として放射されるので問題になっている。特に、流体機械は音波を伝えやすい金属配管と接続していることが一般的であり、低減対策が必要になる。加振原因である流体中の圧力変動が低周波数の場合は、ケーシングや構造体の曲げ振動として伝搬するので、防振材や防振構造の採用である程度の減衰が可能である。一方、中高周波数成分は固体内を縦波弾性波として伝搬し、金属接触している各種部材中を透過して、予測しないような部位から騒音を放射することが多い。この場合には、基礎架台と床の接合部、配管と支持金具の接合部、配管の壁・天井貫通部等に縦波弾性波が伝搬しにくい緩衝材を挟み込むことが有効である。固体伝搬音対策の詳細は未解明点が多いが、密度が大きく硬い構造体ほど音をよく伝えるので、伝搬経路にあまり採用しないことが対策の基本である。d. 流体力学的騒音の制御流体機械で発生する流体力学的騒音の真の音源は、固体境界面に圧力変動を誘起する流体の非定常な動き(渦)である。しかし、固体境界面に誘起された圧力変動を見掛け上の音源とすると、工学的に取扱いやすくなる。流体機械の周囲に放射される流体音の大きさや特性は、次式のように表すことができる。放射音=圧力変動×相関面積(長さ)×構造振動・周囲音場の伝達関数この式には時間的な変動特性や周波数分布なども含めているが、右辺のいずれかの項が少なくなるように流体を制御することができれば、流体機械からの放射音が低減する。その低減制御手法は機種によって詳細な点では異なるが、共通する項目の要点をまとめると、次のようになる。図-5・14 流れを変える流体騒音の低減手法の基本的事項である。対象にする流体音の発生機構が把握できたら、それぞれの機構(原因になる流れの状態)に応じて、以下のような方法を検討する。・速度を遅くする(圧力変動の抑制)・偏流させる(相関長さの制御)・渦の発生を抑える(圧力変動の抑制)・乱流成分を低減する(圧力変動の抑制)・流速分布の平滑化(圧力変動の抑制、相関長さの制御)・乱れの一様性(二次元性)をなくす(相関長さの制御)・微小乱れを利用する(相関長さの制御)・境界層流れの制御(圧力変動の抑制、相関長さの制御)これらの制御方法の中から最適な方法を選択するためには、理論だけでは対処困難であり、経験と実験シミュレーションが必要である。(2) 共鳴(acoustic resonance)・共振(resonance)を避ける共鳴現象や共振現象が騒音の主因であれば、次のような対策が有効である。・補強材の利用、固有振動モードの変更(伝達関数の制御)・干渉体・仕切り板・吸音材などによる共鳴周波数を変更(伝達関数の制御)・流速を変えて発生音の周波数を変更(相関長さの制御)・流れの自励振動(self-excited vibration)現象の構成条件を変更(相関長さの制御、伝達関数の制御)これらの対策を実施するためには、発生音・圧力変動・速度変動などの周波数成分を知る必要がある。特に流れと構造体や音場とが連成して共振や共鳴している場合は、非定常流れが強制的に共振現象や共鳴現象に引きずり込まれている(ロックイン現象)ので、実験による現象解明と共鳴条件、共振条件の制御が不可欠である。(3)
