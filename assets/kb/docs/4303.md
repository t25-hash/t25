# 1・2・1 用語の説明

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

a. 有効落差・全揚程流体のもつエネルギーについては、第I部2・1節で詳細な説明がなされている。エネルギーの基準量としては、単位重量当たりの力学的エネルギー、すなわちヘッド(m)が概念的にも理解しやすく、従来から広く用いられているので、本章ではヘッドによる表示を主として用いる。なお、比エネルギーへの換算には重力の加速度を乗じるだけでよい。水車の駆動に利用される水の全ヘッドを有効落差(effectivehead)、ポンプ運転により液体が得る全ヘッドを全揚程(totalhead)という。前者は水車入口と出口の全ヘッド差であり、後者はポンプ吐出し口と吸込口の全ヘッド差である。水車の場合、吸出し管も水車の一部とみなすが、ポンプでは吸込管は除外する。水力発電所における水車運転時の落差には、図II-1・1を参照して、以下のような用語が用いられる。ただし、添字,はポンプ吸込口、吐出し口の断面上の値を表す。b. 入力・出力および効率全ヘッド(m)を有する液体が流量(m/s)で流れているとき、液体とともに単位時間当たりに通過するエネルギー、すなわち仕事率(動力)は次式で与えられる。この水車出力は負荷(load)により吸収されることになるので、負荷を出力と同義語に使うことも多い。ポンプでは原動機入力が与えられ、これを液体の仕事率に変換する。大容量機が多い水車では効率ηは高い値をとり、形式によって相違するが、一般的にη=0.8(小型水車)~0.95となる。ポンプでは、効率ηは吐出し量や扱う液の種類などで著しく異なり、高粘度液用ポンプやごく小型のものではη=0.36~0.6程度、産業用のものはη=0.7~0.92、揚水発電所に用いられるポンプやポンプ水車はη=0.88~0.94程度である。
