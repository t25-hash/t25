# 1・2・1 用語の説明

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

a. 有効落差・全揚程流体のもつエネルギーについては,第I部2・1節で詳細な説明がなされている.エネルギーの基準量としては,単位重量当たりの力学的エネルギー,すなわちヘッド(m)が概念的にも理解しやすく,従来から広く用いられているので,本章ではヘッドによる表示を主として用いる.なお,比エネルギーへの換算には重力の加速度を乗じるだけでよい.水車の駆動に利用される水の全ヘッドを有効落差(effectivehead),ポンプ運転により液体が得る全ヘッドを全揚程(totalhead)という.前者は水車入口と出口の全ヘッド差であり,後者はポンプ吐出し口と吸込口の全ヘッド差である.水車の場合,吸出し管も水車の一部とみなすが,ポンプでは吸込管は除外する.水力発電所における水車運転時の落差には,図II-1・1を参照して,以下のような用語が用いられる.総落差=取水口水面高さ-放水口水面高さ静落差=水車停止時の上水槽水面高さ-水車停止時の吸出し管出口水面高さ有効落差=総落差-取水口から水車入口までの損失ヘッド-吸出し管出口水位と放水口水位の高低差-吸出し管出口の速度ヘッドポンプでは,全揚程(m)は吸込口と吐出し口の全ヘッド差であり,=욃-욦ρ+욃-욦2+욃-욦(II-1・2)で計算される.ただし,添字,はポンプ吸込口,吐出し口の断面上の値を表す.下部水槽から上部水槽へ揚水する場合には,図II-1・2に示す全揚程の内訳は以下のようになる,=-ρ+욀+욆욃+욆욦=욀+욆(II-1・3),はそれぞれ上部,下部水槽の液面圧力で,右辺第1項は液面が大気に接していないような場合の,上下液面の圧力の差に抗して揚水するためのヘッドを表す.第2項の욀は,上下水槽水面の高低差で実高さ(total static head)といい,욀≡-ρ+욀は実揚程(actual head)と呼ばれる.욆≡욆욃+욆욦は下部水槽からポンプ吸込口までの損失ヘッド욆욦と,ポンプ吐出し口から上部水槽までの損失ヘッド욆욃の総和で,管摩擦損失,曲がりなどによる損失,弁による損失などから成る.b. 入力・出力および効率全ヘッド(m)を有する液体が流量(m/s)で流れているとき,液体とともに単位時間当たりに通過するエネルギー,すなわち仕事率(動力)は次式で与えられる.図-1・1 水車の落差説明図第II部水力機械図-1・2 ポンプの揚程説明図욠=ρ×10(kW)(II-1・4)水車では,水のもつ仕事率욠を水車入力(turbine input)といい,これを機械的動力に変換する.式(II-1・4)のを有効落差(m)とし,ρ=10(kg/m)として水車効率(turbineefficiency)η욧および発電効率ηを乗じれば,発電所出力=9.8η욧η(kW)が得られる.また,水車入力욠から水車での諸損失動力を差引いたものが水車出力(turbine output)=η욧욠になる.この水車出力は負荷(load)により吸収されることになるので,負荷を出力と同義語に使うことも多い.ポンプでは原動機入力が与えられ,これを液体の仕事率욠に変換する.욠を水動力(water power)という.式(II-1・4)のを全揚程として,ポンプ効率(pump efficiency)η욣,電動装置効率η욧욥および原動機効率η욃で除すれば,原動機入力=ρ×10η욣η욧욥η욃(kW)が得られる.また,水動力욠にポンプの諸損失動力を加えたものがポンプ入力(pump input)=욠η욣になる.大容量機が多い水車では効率η욧は高い値をとり,形式によって相違するが,一般的にη욧=0.8(小型水車)~0.95となる.ポンプでは,効率η욣は吐出し量や扱う液の種類などで著しく異なり,高粘度液用ポンプやごく小型のものではη욣=0.36~0.6程度,産業用のものはη욣=0.7~0.92,揚水発電所に用いられるポンプやポンプ水車はη욣=0.88~0.94程度である.
