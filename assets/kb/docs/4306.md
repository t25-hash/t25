# 1・2・3 水力損失および漏れ

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

a. 水力損失ターボ形水力機械の羽根車、およびその上流・下流に設置される固定流路における摩擦およびはく離は、流れに損失ヘッド욆を引き起こして動力損失ρ욆を発生させるので、ポンプの入力増大や水車の出力低減を引き起こす。また、羽根車背面の摩擦や羽根入口に発生する再循環流れはトルク損失を引き起こし、ポンプインペラを通して流体に有効に供給されるはずの動力や、水車ランナを通して有効に得られるはずの動力をむだに消費させる。これらの動力損失を水力損失(hydraulic loss)といい、最終的には熱となって液温を上昇させる。両吸込渦巻ポンプの場合を例にとり、最高効率点における各部の損失割合を比速度に対して示すと、図II-1・13のようになる。図中の機械損失は、軸受や軸封部で無効に消費される損失動力であり、また漏れ損失は、シール部を通して吸込側に漏れる流量욆(b2―42流体機械項参照)により引き起こされる損失動力ρ욆η욠である。水力損失には上記のように、ターボ形水力機械を通過する流れに直接影響を及ぼして全ヘッドを減少させるものと、羽根車の背面すきまや入口逆流のように、機械内部に滞留してトルク損失を引き起こすものがある。このうち羽根車背面流れによるトルク損失は円板摩擦損失(disk friction loss)と呼ばれ、第I部3・1・8項で説明されている。そこでここでは、羽根車の流路を通過する流れに直接影響を及ぼして全ヘッドを低下させる損失について説明する。羽根車および固定流路の水力損失には、衝突損失(incidenceloss)、曲がり損失(turning loss)、はく離損失(separationloss)、二次流れ損失(secondary flow loss)など、さまざまな水力損失とその計算式が提案されているが、これらは基本的には三つの損失、すなわち摩擦損失、混合損失、減速損失から成ると考えてよい。このうち、減速損失は境界層の排除厚さ分を支えるための運動量欠損に基づく圧力損失、また混合損失は不均一流れが均一化するときに生じる損失であり、前2者はレイノルズ数依存性をもち、混合損失はレイノルズ数によらない。はく離や後流などに基づく損失は、均一流れに回復するまでに混合損失となり角運動量は保存されるが、速度分布が著しくひずんだ流れであるため、下流側の固定流路で壁面と干渉して大きな摩擦損失および減速損失を発生する。また、二次流れに基づく損失は、二次流れ自体が境界層内の力の不釣合いによって生じるから、大部分は摩擦損失および減速損失に費やされ、残りが混合損失となる。主流の圧力が流れ方向に減少していく増速流では、境界層のはく離が起こりにくいため水力損失は比較的小さいが、圧力が流れ方向に増大していく減速流では大きな水力損失が発生する。水車の場合、渦巻ケーシングからステーベーン、ガイドベーン(可動案内羽根)を通りランナへ流入する流れは増速流となるので、水力損失はポンプに比べてかなり少ないが、吸出し管内の流れは拡大流でしかも曲がりをもつことが多いので、損失が問題となる。非設計点ではガイドベーン開度が変化するため、ステーベーンとのマッチングが悪くなり、また吸出し管内の流れも旋回をもつようになるので損失が増大する。一方、基本的に減速流であるポンプのディフューザおよびインペラ内の流れでは、大きな水力損失を生じる。そこで以下に、ポンプインペラの水力損失についてもう少し詳しく説明する。. 摩擦損失(friction loss)羽根車内のメリディアン速度が、流量とともに断面内で一様に変化するならば、摩擦損失は流量の二乗に比例して変化する。しかし、実際の羽根車流れは、設計点付近でさえも二次流れのために一様にはならない。その結果、摩擦損失は羽根間流路を等価な円管で置換えた場合の摩擦損失の約3倍にもなる。流量が低下すると、二次流れや羽根負圧面のはく離のために主流が圧力面側に押付けられるが、その速度はあまり低下しないので、摩擦損失ヘッドもそれほど減少しない。羽根間流路の表面粗さが粗くなると、特に羽根入口付近の境界層が薄いため、摩擦損失がかなり増加する。羽根角度が小さい場合には、羽根長さが長くなるため摩擦損失も大きくなり、また羽根数が多いと入口部分の流路が狭くなり、メリディアン速度が大となるので摩擦損失も大きくなる。. 混合損失(mixing loss)遠心羽根車の出口では、羽根車内で発生した二次流れや局所はく離のためにひずんだ速度分布となり、流路幅>羽根幅の場合にはさらに羽根出口直後での三次元逆流のため、著しく大きな速度ひずみを生じる(図II-1・14)。しかし、これを積分して流量平均全圧を求めると、羽根車効率が異常に高くなることがある。これは、二次流れや局所はく離が羽根車出口ではまだ損失を発生しておらず、ひずんだ速度分布をしており、下流で均一化する際に混合損失として現れるためである。羽根出口の周方向の速度ひずみは可逆的に均一化するとして取扱えるので、羽根幅方向のひずみがおもに下流流路における混合損失を引き起こす。. 減速損失(deceleration loss)遠心ポンプのメリディアン流路は、減速損失を減らすべく相対速度ができるだけ一定になるように設計される。しかし、相対流れに直角に作用するコリオリ力、および流路曲がりによる遠心力のために、羽根の圧力面と負圧面では境界層の発達は大きく異なる。圧力面では、コリオリ力のために境界層の発達は少なく減速損失は小さい。一方、負圧面では二次流れにより低エネルギー流体が集積されて境界層は発達するが、通常の遠心ポンプでは羽根が長いため、羽根に沿った真の圧力上昇(圧力上昇から回転による遠心力分を差引いたもの)は必ずしも大きくないので、減速損失はあまり大きくない。しかし、羽根車入口に再循環流が発生したり、羽根間流路にはく離が起こるようになると、主流は羽根車内で急拡大して減速され、減速損失が大きくなる。.
