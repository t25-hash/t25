# 1・2・4 相似則と比速度

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

幾何学的に相似な二つの水力機械を,相似な運転状態(例えば最高効率点)で運転すると,第I部3・1・9項で示した相似則が成り立つ.しかし,実際には力学的無次元数(例えばレイノルズ数)をすべて一致させることは難しく,それらの差異は無視して同一ポンプで同じ液を扱う場合は,回転速度の変化が±20%程度以内なら吐出し量∝,全揚程∝,軸動力∝が成り立つとして取扱ってよい.ポンプの場合,,,の値により大きさや形状が大幅に変化するので,一つの特性数でポンプの形や特性が表示できれば非常に便利である.この特性数が比速度(specific speed)で,욦=(II-1・11)で与えられ[第I部3・1・8項の(4)参照],通常,最高効率点の値を用いて示す.なお,多段ポンプの場合は1段当たりの揚程,両吸込ポンプでは吐出し量の半分を用いて算出する.욦が定まると羽根車形状およびポンプ形式がほぼ定まり(図I-3・27参照),特性曲線の形状,効率,キャビテーション性能もほぼ定まる.特に욦の小さいポンプはが大きくが小さいので,羽根車径が大きくなり円板摩擦が増大し,漏れ損失も増加する傾向をもつ.逆に욦が大きいポンプはが小さいので,損失ヘッドが効率に大きい影響を与えるようになり,水力効率の良いポンプ形状とすることが重要である.水車の場合は出力が重要な意味をもつので,욦の式のの代わりに(kW)を用いて表した水車比速度(specific speed ofhydraulic turbine):욦=(II-1・12)が用いられる.この比速度の値によって水車の形式も分類される2―44流体機械
