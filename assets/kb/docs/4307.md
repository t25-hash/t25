# 1・2・4 相似則と比速度

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

幾何学的に相似な二つの水力機械を、相似な運転状態(例えば最高効率点)で運転すると、第I部3・1・9項で示した相似則が成り立つ。ポンプの場合、,の値により大きさや形状が大幅に変化するので、一つの特性数でポンプの形や特性が表示できれば非常に便利である。なお、多段ポンプの場合は1段当たりの揚程、両吸込ポンプでは吐出し量の半分を用いて算出する。が定まると羽根車形状およびポンプ形式がほぼ定まり(図I-3・27参照)、特性曲線の形状、効率、キャビテーション性能もほぼ定まる。特にの小さいポンプはが大きくが小さいので、羽根車径が大きくなり円板摩擦が増大し、漏れ損失も増加する傾向をもつ。逆にが大きいポンプはが小さいので、損失ヘッドが効率に大きい影響を与えるようになり、水力効率の良いポンプ形状とすることが重要である。この比速度の値によって水車の形式も分類される2―44流体機械
