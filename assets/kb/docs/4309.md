# 1・2・6 材料

> **章**: 第1章 水力機械一般
> **節**: 1・2 ターボ形水力機械の流れと性能

水力機械に用いられる部品は,接液部品とそうでない軸受などの一般部品から成る.前者は液に対する耐食性,圧力,温度などから,必要とされる強度を考慮して材料を選定する.最も一般的な常温清水用ポンプでは,ケーシング:鋳鉄,羽根車:青銅,鋳鉄,ライナリング:青銅,主軸:炭素鋼,スリーブ:青銅,13Crステンレス鋼が用いられる.なお,球状黒鉛鋳鉄は普通鋳鉄の2倍程度の許容応力がとれるので,圧力や周速度の条件がやや厳しい場合によく用いられる.ポンプ適用材料の一例として,高圧,高温あるいは低温用ポンプのケーシングなどに使用される材料の適用範囲を図II-1・26に示す.強度上,羽根車外周速度による目安として,鋳鉄35m/s以下,青銅45m/s以下,鋳鋼65m/s以下,ステンレス鋼70~130m/s以下,強度を特に必要とする主軸の場合には,高抗張力の合金鋼,特に13Crステンレス鋼が広く用いられている.また,腐食液に対しては18-8ステンレス鋼,モネルメタルが用いられ,特殊な場合は金属ケーシング表面にゴムや樹脂をライニングして酸性液などに用いる.水車は大型となることが多いので,ケーシングは鋼板の溶接構造とするのがふつうで,強度を大きくする場合は高張力鋼が用いられる.ランナについても鋳鋼製が多いが,大型のものは溶接構造とする.特に強度が必要な場合は,高Niを含有した13Crステンレス鋼を採用する.耐キャビテーション性水車材料の例が1・4・3項に示されている.
