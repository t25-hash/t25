# 1・3・1 性能・特性の表示

> **章**: 第1章 水力機械一般
> **節**: 1・3 性能試験と寸法効果

水車・ポンプ水車では、実物が大型であるので模型試験が多用されており、単位寸法・単位落差(または単位揚程)に換算された単位回転速度(unit speed)、単位流量(unit discharge)および単位水車出力(またはポンプ軸入力)を模型特性の表示に使用する。式(II-1・19)において、は羽根車に正味に作用する値を用いるべきであるが、水車では有効落差が、ポンプでは全揚程が慣用されている。相似則が成り立っており、かつ前述の寸法効果を無視すれば、これら三つの値は機械の大小に関係なく模型と実物で一致する。ポンプ水車の水車特性は図II-1・29のように表示されるが、ポンプ特性は図II-1・30(a)に示すように、ポンプ分野で慣用されている回転速度′一定条件での′-′特性の表示が用いられる。ポンプ水車は可動案内羽根であるガイドベーンをもつので、図中破線で示す各′曲線の包絡線(オンカム線という)上で運転される。ポンプでは、実物をそのまま試験することが多く、ガイドベーンももっていないので、図II-1・30(a)の表示で一つの実曲線のみの性能となる。ここで、:羽根車出口面積、:羽根車出口平均径での周速度、:重力の加速度である。同じ羽根車を正・逆回転させることにより、ポンプと水車機能を発揮する可逆式ポンプ水車では、正転・正流をポンプとすれば、逆転・逆流が水車となり、これら二つの運転領域とその間の過渡的運転領域も含めた特性表示例を、フランシス形ポンプ水車について図II-1・31に示すが、これを完全特性(complete characteristics)という。水車特性表示にならって横軸、縦軸で表され、第2―48流体機械1象限から第4象限にかけてそれぞれポンプ域、逆転ポンプ域、水車域およびポンプ逆流域(ブレーキ域ともいう)と呼ばれる。一方、ポンプでは最高効率点(添字0で示す)の値で無次元化した表示を用いることが多く、図II-1・32に両吸込遠心ポンプの完全特性の一例を示す。
