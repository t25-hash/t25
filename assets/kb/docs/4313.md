# 1・3・2 性能試験

> **章**: 第1章 水力機械一般
> **節**: 1・3 性能試験と寸法効果

性能試験(performance test)は水力機械の受け渡しや性能にかかわる研究のために、水車・ポンプ水車ではおもに模型を使用した模型試験(model test)で、また必要に応じて現地の実物を用いた実物試験(prototype test)で実施される。ポンプでは、大型のもの以外は工場で実物を用いた試験が実施される。性能は無拘束速度特性を含む効率特性とキャビテーション特性がおもなもので、前者はキャビテーションが発生しない条件で試験する決まりとなっており、後者は1・4節で述べるので、ここでは前者に限定する。前述の二つの特性以外にも完全特性、水圧脈動および水推力なども性能試験に含まれ、必要に応じて同じ模型を用いて模型試験が実施される。さらに、1970年代以降ポンプ水車の高落差化が顕著になるに伴って、機器の信頼性を高めるための特殊試験、すなわち実揚程・実落差模型試験、および過渡現象模型試験が特別な試験設備で行われようになっているが、これらについては省略する。
