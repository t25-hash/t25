# 1・3・3 模型試験

> **章**: 第1章 水力機械一般
> **節**: 1・3 性能試験と寸法効果

受注品の性能検証または研究のため、小型の模型を用いて模型試験が行われ、対象とする実物の運転状態に対応する模型の特性が調べられる。水車は実物が大型で、費用その他の理由により実物試験が難しく、模型試験の性能によって実物を受取る商取引(模型受取り試験)が慣用的に行われており、この場合は模型と実物の寸法相似性および模型試験精度、さらには模型から実物への性能換算法などが、国内および国際規格で厳しく管理されている。ポンプでは実物を製作工場で試験できるものが多く、大型のもの以外は実物での工場試験で済むことが多い。図II-1・33は模型試験装置の一例であり、流量計を較正する場合は計量タンクから貯水槽に戻す開ループで、その他は測定時間を短縮できる閉ループで実施される。また、キャビテーション試験においてサクションタンクの真空度を上げるため、貯水槽水面から約図-1・32 両吸込遠心ポンプの完全特性表示例10m上部に模型が設置される。ここで、試験水の温度からρ′を、試験場所からをあらかじめ求めておき、試験中は一定値とする。測定結果は式(II-1・19)によって整理され、図II-1・29のような速度特性または目玉曲線として表示される。実物水車はほぼ一定落差、一定回転速度で運転されることが多いので、式(II-1・19)の関係より模型特性上は一定の運転となり、流量のみ変化する。一例として図II-1・29の目玉曲線のにおける特性値,η′を各開度′について読み取ると、図II-1・34(a)のような流量特性が求まる。可動ランナ羽根をもつ水車(デリア水車、カプラン水車など)では、ランナ羽根取付角度(β′)ごとに図II-1・34(a)のような曲線が得られるので、図II-1・34(b)に一点鎖線で示すこれら曲線群での最高効率点を結んだ最良運転状態の特性曲線(オンカム特性という)上で、実物が運転される。すなわち、水車の負荷(トルク)を減じていくと回転速度は上昇し、水車が無負荷になる回転速度(図II-1・29の)を無拘束速度(runaway speed)という。水車・発電機の遠心力に対する構造上の安全性、および軸系振動を含む回転系の信頼性を確保するための設計に必要な特性量を得るため、無拘束速度試験が実施される。実物ポンプ水車は最良運転状態のオンカム特性上でポンプ運転されるので、図II-1・35(b)に示すように実物性能はオンカム性能のみ表示する。また、実物ポンプでは系統とつないだ電動機などによって所定の回転速度で運転し、実際の使用条件で工場試験を行うことが多いので、通常模型試験は実施しない。ポンプ性能は流量を減じていくと羽根車内の流れが急変する、いわゆる逆流によるサージ現象が発生して運転不能に陥ることや、起動・停止などの際、流量が零となる締切運転が行われるので、振動・騒音の測定などを含め、これらの現象を検証することも性能試験に含まれる。
