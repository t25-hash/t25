# 1・4・3 キャビテーション損傷

> **章**: 第1章 水力機械一般
> **節**: 1・30 )で욦の代わりに욦+を使ってσを求める.

機械的強度を十分有する羽根車であっても,キャビテーションまたは土砂摩耗および両方の相乗効果によって損傷される.キャビテーション損傷(cavitation damage)は,発生した気泡が下流の高圧部で圧壊される際に発生する圧力波,ないしはマイクロジェットが金属表面に衝撃を繰返し与え,材料の塑性変形と疲労破壊を引き起こすことによる.したがって,キャビテーション発生と損傷の間にはキャビテーションの発生部位と強さ,材料の耐キャビテーション性が関与し,いまだ十分解明されていない.図II-1・42はポンプのキャビテーション強さを,羽根車の目玉周速度[욄(m/s)]とキャビテーション長さ[욂욀욋(mm)]を用いて=log욄×욂욀욋で定義し,実物の損傷データに適用して욄×욂욀욋≦10300の損傷発生限界を求めた結果を示す.材料の耐壊食性については,水車材料の磁わい振動法による実験結果例を図II-1・43に示す.硬度が高い材料ほど壊食されにくい傾向があり,おおよそ硬さの2.5乗に比例するといわれている.高Ni-13Crステンレス鋼が水車・ポンプ水車ランナに多用されており,現地修理では補修が容易なオーステナイト系溶接材料が使われている.1960年代以降コンピュータの発達とともに,コンピュータを利用したソフトウェアの開発およびその適用研究が盛んになり,水力機械に各種解析技術が導入された.まず,機械と付設された管路系の過渡運転時の水撃現象などを予測するための過渡現象解析手法が開発され,1970年代には実用化されている.この年代は差分法,有限要素法など,コンピュータ解析に適した各種離散化手法を用いた流れの数値解析(numerical analysis)が行われ始めており,1980年代以降水力機械の内部流れ,特に性能にとって最も重要な羽根車流れの三次元解析が実用化された.水力機械の性能予測(performance prediction)は,実用的精度をもつ手法が開発されれば,多大の時間と費用がかかる模型試験を省略できる利点があり,1990年代以降,前述の流れの数値解析を利用して予測精度の向上を図る研究が盛んに行われている.これら数値流体力学,すなわちCFD(computational fluid dynamics)をベースとした研究・設計開発のツールについて,水力機械への適用例を述べる.
