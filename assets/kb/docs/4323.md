# 3・4・1 微粉炭火力発電

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・4 石炭火力発電

a.微粉炭火力発電の開発経緯 微粉炭火力発電は，微粉炭機
により粉砕された石炭を炉内に送り込んで燃焼させる微粉炭燃焼方
式による発電であり，燃焼効率が高く，運用性に優れており，現在
の石炭火力発電の主流をなすものである．
1961年ごろより，火力発電は石炭から石油へと燃料転換が行わ
れたが，一方，国内産炭地振興のための微粉炭火力も建設され，微
粉炭燃焼技術，環境対策技術等の蓄積が図られてきた．
1973年の第一次オイルショックを契機に，石油消費の節約が国
際的な共通課題となり，1979年の国際エネルギー機関（IEA）の
石油火力新設禁止決議において石油火力の新設が原則としてなくな
った．これにより，石油代替エネルギーの一つとして石炭が注目さ
れ，実績のある微粉炭火力発電が脚光を浴びるようになった．国内
炭での微粉炭火力発電において蓄積された技術を基に，海外の多銘
柄炭を使用する大容量微粉炭火力（単機容量500MW）が1981年
に登場し，単機容量は700，1000MWと大型化した．わが国最大
の微粉炭火力としては，単機容量1050MWのものが2000年7月
より稼働中である．
一方，発電効率の向上による燃料費の低減，環境負荷の低減
（SO ，NO ，ばいじん，COの低減）のため，蒸気条件の向上も
合わせて行われ，主蒸気圧力25MPa，主蒸気温度600℃/再熱蒸気
温度610℃が現在国内最高の蒸気条件である．
また，微粉炭火力は当初ベース電源の位置付けにあったが，近
年，原子力発電の増加，昼夜の電力需要差の拡大から，より高い運
用要求を満たすため，変圧運転，タービンバイパスシステムを採用
した中間負荷対応可能なものが登場している．
b.揚運炭設備 石炭は貯蔵のために多くの敷地面積を要し，
石炭ハンドリング時の飛散防止，自然発火の防止について考慮しな
ければならない．
石炭は通常，船舶（5000～150000t級）により輸送され，荷揚
げ，貯炭，払出しが行われる．
石炭の荷揚げを行う揚炭機（アンローダ，unloader ）は，水平
引込式，橋型，連続式があるが，荷役効率，粉じん防止の観点から
連続式アンローダが主流となってきている．
また，発電所側に揚炭機をもたず，石炭船に揚炭機能をもたせた
セルフアンローダ船を使用するケースもある．
貯炭方式は，従来シンプルで経済的な屋外貯炭方式（図3･29）
がほとんどであったが，環境，景観の面から建屋内貯炭方式（図
3･30），ドーム貯炭方式（図3･31），サイロ貯炭方式（図3･32）の
採用が多くなってきている．
ボイラ建屋までの石炭の輸送は，ベルトコンベヤによって行われ
るが，新しいタイプとして密閉パイプ構造でベルトを空気圧で浮上
させる空気浮上式も採用されている．空気浮上式ベルトコンベヤ
（図3･33）は，従来のコンベヤに比べ，低騒音化が図れ，設置スペ
ースも少ない特徴がある．

c .微粉炭燃焼装置
.微粉炭燃焼方式 微粉炭燃焼方式（pulverized coal fir-
ing system）には，石炭を微粉炭機で粉砕して直接バーナに供給す
る直接燃焼方式（direct firing system，図3･34）と石炭を粉砕し
た後，いったん貯蔵してからバーナに供給する貯蔵燃焼方式があ
る．
直接燃焼方式は，貯蔵燃焼方式に比べて設備構成がシンプルで微
粉炭貯蔵による発火防止のための保安対策も必要なく，コスト面で
有利であり，現在の微粉炭火力発電で標準的に採用されている．こ
こでは直接燃焼方式について説明することとする．
直接燃焼方式では，貯炭設備からバンカへ送られた石炭が，バン
カ下の給炭機によって微粉炭機へ供給される．微粉炭機で乾燥・粉
砕された微粉炭は，搬送用空気（一次空気）によって微粉炭管を通
り，微粉炭バーナから燃焼用空気（二次空気）とともに火炉へ吹込
まれ燃焼する．
.微粉炭燃焼の概要 微粉炭燃焼では，石炭を微粉炭機で
粉砕し，微粉炭として浮遊燃焼させる．これにより，着火時間およ
び燃焼時間が極端に短縮され，油の噴霧燃焼に近い形でバーナで燃
焼させることができる．
バーナから出た微粉炭の着火点は，バーナからの噴出速度と火炎
伝搬速度によって決まる．したがって，微粉炭を炉内に吹込む速度
が過大であると火炎は吹き飛ばされ，反対に遅いと逆火の可能性が
ある．一方，微粉をバーナ部へ搬送する空気（一次空気）は，給炭
量に応じた風量とすることから，一般に微粉炭の噴出速度は微粉炭
機の負荷によって変化する．また，火炎伝搬速度は石炭性状（おも
に燃料比）によっても左右されるため，着火位置の適正な調整が必
要となる．
石炭は微粉炭機で通常200メッシュ通過70 %以上程度の微粉炭
に粉砕される．
一次空気によってバーナから炉内に噴射された微粉炭は，炉内か
らの放射熱，二次空気からの対流熱および自己化学反応熱によって
急速に加熱され，揮発分とチャーに熱分解する．
揮発分は一次空気と混合し，ただちに燃焼する．一方，チャーは
揮発分燃焼領域の後流側で二次空気と混合し，その中を浮遊しつつ
燃焼する．燃焼温度は1300～1500℃程度である．チャーの燃焼速
度は揮発分の燃焼速度と比べて極めて遅く，石炭全体の燃え切り時
間の約80～90%は，チャーの燃え切り時間によって占められ
る ．したがって，良好な燃焼性を得るためにバーナ部で強い旋
回を与え，燃焼速度を速めたり，ガス，油焚きボイラに比べて火炉
寸法を大きくすることが必要となる．
.微粉炭機 微粉炭機（pulverizer ）は一般にミルとも呼
ばれる．微粉炭機は，石炭性状，発電プラントの運用条件を十分考
慮して最適な容量・台数を選定する必要がある．
微粉炭機は，立て型，横型（チューブミル），ハンマ型（ハンマ
ミル）に大別されるが，現在の国内の微粉炭火力においては，幅広
い炭種に適用でき，微粉粒度の調整，起動停止が容易で負荷追従性
に優れ，据付面積，消費動力が小さい立て型微粉炭機が多く採用さ
れている．
微粉炭機の粉砕能力に主として影響を及ぼす要因は，使用する石
炭のHGI （hardgrobe graindability index）と水分である．HGI
が小さい石炭ほど粉砕性が悪くなる．
また，石炭中の水分が多いと微粉炭機での乾燥が十分に行えず，
粉砕能力が低下する．
微粉炭機の粉砕能力の点から，石炭性状としてHGI40以下，全
水分20%以上の炭は注意が必要である．
.微粉炭バーナ 微粉炭バーナは，微粉炭とこれを搬送す
る一次空気がバーナのノズルを通り，空気予熱器によって暖められ
た燃焼用の二次空気が一次空気の外側から供給される．
現在の微粉炭火力におけるバーナは，低NO燃焼性を有するい
わゆる低NOバーナ（low NO burner ）が用いられている．
燃焼において発生するNOは，フューエルNO（fuel NO），
サーマルNOに大別される．ガス，油焚きボイラの場合は，サー
マルNO が多いが，石炭焚きの場合は，80～90%をフューエル
NO が占める ．したがって，石炭焚きの場合は，フューエル
NOを積極的に還元させることがNO低減において有効である．
図3･35に低NOバーナの概念の一例を示す．
.ボイラ 火炉寸法は，石炭性状，バーナ本数・容量，
水・蒸気の収熱バランス，機器配置，形状等を考慮して決定される
が，微粉炭火力の場合，主として石炭性状に対する配慮から火炉の
主要寸法が定められる．火炉寸法に影響する石炭の諸特性として，
石炭の燃焼性，スラッギング性，ファウリング性，灰による摩耗
性，NO発生特性が挙げられる（図3･36）．

炉容積をとることが必要である．

ゾーンが還元雰囲気となる．バーナ- OFA間のNO還元域での滞
留時間を十分確保するとともに，灰中未燃分低減のため，OFA-火
炉出口の完全燃焼域においても必要な滞留時間が得られるよう十分
な火炉高さが必要となる．
d.環境対策設備
.システム構成 微粉炭火力用の環境対策設備は，SO，
NO，ばいじんの規制強化に伴い，高効率化が図られてきた．最
近の微粉炭火力では，特にばいじん濃度規制10mg/mNという非
常に厳しい値をクリアするため，より高性能な方式が採用されるよ
うになっている．
高性能な方式の例として，ノンリークGGH，低低温EP，湿式
石灰石-石こう法（wet limestone-gypsum process ）の脱硫装置を
組合わせたシステム（図3･37）のほか，活性炭を用いた乾式脱硫
装置と前置EP，後置EPを組合わせたシステム（図3･38）もあり，
立地条件，環境規制によって用いられるシステムも異なっている．
.脱硝装置（DeNO) 脱硝装置反応器は，垂直流と水平
流があるが，微粉炭火力の場合，灰の堆積防止の観点から垂直流が
用いられる．また，触媒（catalyst ）は，灰による摩耗対策を考慮
したものが採用されている．
.脱硫装置（DeSO) 脱硫プロセスは，湿式石灰石-石
こう法がほとんどであり，副生品の石こうはセメント，石こうボー
ド用に有効利用される．
現在では，吸収塔は脱じん，脱硫，酸化を1塔で行うものが主流
であり，1000MW級の脱硫装置においても吸収塔は1系列，1塔
化されている．また，新技術として，活性炭を用いた乾式脱硫装置
が600MWの事業用微粉炭火力に採用されている．
.電気集じん装置（EP) 従来は，低温EPあるいは高温
EPであったが，ばいじん濃度10mg/mN対応として，ノンリー
クGGHとの組合せによる低低温EPを採用するケースが増えてい
る．低温EPの場合140℃，高温EPの場合380℃程度であったガ
ス温度を90℃程度以下とする低低温EPでは，ダストの電気抵抗
率の低下（約1×10 Ω･ cm以下）により，逆電離が発生しない領
域となるため，従来のEPと比べ，より安定した性能でEPが運用
できる ．
e.灰処理設備
.石炭灰の生成と有効利用 微粉炭燃焼により発生する石
炭灰は，フライアッシュ（fly ash），シンダアッシュ，クリンカア
ッシュに大別される．
フライアッシュは，電気集じん装置で捕集されたもので，球形の
微粒子であり，コンクリートに混和して使用すると，施工時の流動
性向上，発熱の抑制等の効果があり，土木・建築分野で利用されて
いる．また，フライアッシュに含まれるケイ酸を利用した肥料も作
られている．
シンダアッシュは，節炭器（ECO），空気予熱器（AH）を通過
する際ホッパに落下したもので，フライアッシュとシンダアッシュ
を総称してフライアッシュと呼ぶこともある．
クリンカアッシュは，燃焼中に石炭灰が相互に凝集し，多孔質な
塊となってボイラ底部のホッパに落下したもので，道路の路盤材等
に利用されている．
石炭灰の発生割合は，おおむねフライアッシュ（シンダアッシュ
を含む）が85～95%，クリンカアッシュ5～15 %である ．
.フライアッシュ処理装置 フライアッシュの処理は，空
気輸送方式が一般的である．これには，真空輸送方式（図3･39），
圧力輸送方式（図3･40），真空・圧力組合せ方式（図3･41）があ
り，輸送距離，容量によって使い分けられる．また，灰による埋立
てを行う場合，高濃度の石炭灰・水スラリーとして輸送するケース
もある．
.クリンカアッシュ処理装置 クリンカアッシュをジェッ
トポンプあるいはアッシュポンプによって輸送する水力輸送方式
（図3･42）が一般的であるが，ヨーロッパで多く採用されている水
封式チェーンコンベヤ方式（図3･43）も国内事業用として採用さ
れている．
