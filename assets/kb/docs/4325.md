# 3・4・3 石炭ガス化複合発電

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・4 石炭火力発電

石炭ガス化複合発電（IGCC:integrated gasification combined
cycle power plant ）は，高効率でクリーンな石炭利用新発電技術
として国内外で技術開発が進められており，次のような特徴を有す
る．

a.システム構成 図3･50の基本構成に示すように，IGCCは
大別して石炭ガス化炉（coal gasifier ）設備，ガス精製（gas
clean-up）設備，ガス/蒸気複合発電（gas / steam combined cycle
power plant ）設備により構成され，それぞれの設備について種々
の方式のものが開発されている．
まず，石炭をガス化剤（酸素/水蒸気または空気）により部分燃
焼させて，水素（H ）や一酸化炭素（CO）を主成分とする生成ガ
ス（raw syngas ）を発生させ，ばいじんや硫黄分を除去してクリ
ーンにした後，ガスタービン（gas turbine）の燃焼器（combus-
tor ）で燃焼させてガスタービンを駆動するとともに，その高温の
排ガスから熱回収して得られる蒸気により蒸気タービン（steam
turbine）を駆動して発電する．
b.石炭ガス化方式 石炭ガス化技術は，図3･51に示すよう
に酸素/水蒸気または空気を用いて石炭を乾留・還元反応により分
解して，HやCOをおもな可燃成分とするガスを製造する技術で
あり，化学工業界においては化学製品の原料製造用あるいは都市ガ
ス製造用として古くから用いられている．
石炭のガス化方式は，供給石炭の粒径ならびに炉内における粒子
の挙動と灰の状態やガス化炉構造，ガス流れ，石炭供給方式，ガス
化剤，温度・圧力レベルなどの違いにより特徴付けられ，ガスの製
造目的に応じて今日まで噴流床（entrained flow），流動床（fluid-
ized bed），固定床（fixed bed）等種々の石炭ガス化プロセスが開
発されている．それぞれのガス化プロセスの熱力学的，技術的な特
徴や石炭ガスの製造目的あるいは経済性などに応じて，反応性，粘
結性，灰の融点や含有量など固有の特性を有するさまざまな石炭の
中から適合する炭種が選定される．
発電用としては，大容量化が容易で，負荷追従性に優れた加圧噴
流床石炭ガス化炉（pressurized entrained flow 図3･50 IGCCの基本構成
coal gasifier ）が適
しており，その研究開発が進められている．表3･5に国内外で開発
中の主要な加圧噴流床石炭ガス炉の概要を示す．
噴流床石炭ガス化炉では，微粉炭火力発電と同様に平均粒径約
(１) 埋蔵量が豊富で世界中に広く賦存する石炭資源，特に従来
の微粉炭火力発電（pulverized coal fired thermal power plant ）
では利用が困難であった低灰融点炭が利用できるため，発電用とし
て利用できる炭種が拡大できる．
(２) 熱効率が高いため，発電電力量当たりの二酸化炭素，硫黄
酸化物，窒素酸化物，ばいじんの発生量を低減できるとともに，ガ
ス/蒸気複合発電のため，温排水量も低減でき環境保全性に優れて
いる．

.ガス化剤 ガス化剤（gasifying agent ）としては，一
般に酸素/水蒸気または空気が用いられる．
酸素/水蒸気をガス化剤とする場合には，発熱量が2500kcal /
mN程度の中カロリーガス（medium-Btu gas ）が生成される．
生成ガスの流量が少ないためガス化炉はコンパクトになるが，酸素
製造のために必要となる空気分離装置（air separation unit ）の所
要動力が大きいことから，送電端熱効率（net thermal eﬃciency）
は一般的に空気吹き（air blown）IGCCシステムより低い．しか
し，近年では酸素吹き（Oblown）IGCCシステムにおいても，ガ
スタービンの抽気を用いて高圧で酸素を製造し，さらに発生した高
圧窒素で燃料ガスを希釈することにより，ガスタービン燃焼器での
NO発生量を抑えるとともに，その動力をガスタービンで回収す
るインテグレーション（integration）技術の進展により空気分離
装置の酸素製造原単位が小さくなり，酸素吹きIGCCシステムの送
電端熱効率も向上しており，現在海外で運転されているプラントの
多くは酸素吹きを採用している．
空気をガス化剤とする場合には，発熱量が800～1200kcal /mN
程度の低カロリーガス（low- Btu gas ）が生成される．日本では，
技術的難度は高いものの，高効率なIGCCの開発を指向して空気吹
きIGCC実証機の開発が進められている．石炭処理量200t / dのパ
イロットプラント（pilot plant ）による運転試験研究の成果を受け
て，ガスタービン燃焼器における燃焼安定性の確保という観点か
ら，実証機ではプラントで発生する余剰酸素を活用して若干酸素を
富化することが想定されている．
.石炭の供給方式 ガス化炉への石炭の供給方式として
は，ドライフィード（dry feed）とスラリーフィード（slurry
feed）の2方式がある．
ドライフィード方式はロックホッパ（lock hopper ）やロータリ
フィーダ（rotary feeder ）等を介し，窒素や空気などを用いた気
流搬送により石炭をガス化炉へ投入するものである．この方式は，
熱効率的には優位であるが，石炭を炉内に供給するためのロックホ
ッパシステムおよび各バーナ（burner ）への均等分配システムが
必要となる．
スラリーフィード方式は，石炭を水と混合（CWM:coal water
mixture）してスラリー化し，それをスラリーポンプ（slurry
pump）によりガス化炉へ供給するものである．この方式は，ガス
化炉への加圧供給が容易であり，多くの実績もあって技術的な信頼
度も高い．しかし，搬送媒体であるスラリー水が炉内に投入される
ため，ガス化時に部分燃焼した熱の一部が水蒸気の潜熱として奪わ
れること，ならびに湿式ガス精製を採用した場合には冷却過程で系
外へ持ち出される水蒸気の熱損失などが大きいことなどのため，熱
効率はドライフィード方式に比して低くなる．
c .ガス精製設備 ガス化炉で生成した石炭ガスは未反応のチ
ャーや灰分，硫黄分（おもにHSとCOS）等の不純物を含んでお
り，ガスタービンで燃焼する前に，タービンの保護ならびに環境保
全対策のため集じん装置，脱硫装置からなるガス精製を行う必要が
ある．ガス精製技術には，表3･6に示すようにガス温度を下げて吸
収液等を用いる実績豊富な湿式ガス精製（wet type gas
purification）と高温のまま行う開発途上の乾式ガス精製（dry
type gas purification）の2方式がある．
一般に，酸素吹きガス化炉を採用した場合は，生成ガス量が空気
吹きガス化炉より少ないため，湿式ガス精製での効率低下も空気吹
きに比して少なくて済むことから，湿式ガス精製を適用しやすく，
空気吹きガス化炉を採用した場合には，乾式ガス精製の適用による
熱効率向上効果が大きい．
.湿式ガス精製方式 湿式ガス精製方式は，一般に水スク
ラバによる集じんと化学吸収法または物理吸着法あるいは直接酸化
法による湿式脱硫設備から構成され，実績が多い．この方式では，
アルカリ金属やアンモニアが除去されるため，ガスタービンの信頼
性，環境保全性の面からは有利であるが，ガスの顕熱損失および水
蒸気や水溶性ガスの損失により熱効率的には不利である．
近年では，チャーのリサイクルによる熱効率の向上ならびに排水
処理量の低減等を目的として，湿式ガス精製方式においても，セラ
ミックフィルタによる集じんが併用されている場合が多い．
.乾式ガス精製方式 乾式ガス精製方式は，400～500℃の
温度条件下で，セラミックフィルタまたは移動床方式等の乾式集じ
ん設備と流動床または固定床方式の脱硫設備から構成される．ま
た，集じんと脱硫を一体化した移動床同時脱硫集じん設備を採用す
る方式もある．これらの方式では，アンモニアが除去されないた
め，ガスタービン燃焼器においてフューエルNO（fuel NO）の
発生量が多くなり，排煙脱硝（De- NO）装置が必要となるが，ガ
スの顕熱損失が少ないなどのため熱効率的には有利である．

e.IGCCプロジェクトの概要 表3･7に内外の主要なIGCC
プロジェクトを示す．欧米ではすでにIGCC実証機・商用機が運転
されている．日本では，200t / d噴流床IGCCパイロットプラント
の試験研究を終了し，2007年開始予定の試験運転に向けて図3･52
に示す石炭処理量1700t / d級，発電端出力250MW級の実証機建
設が進められている．その達成目標は，信頼性に関しては年利用率
70%以上の見通しを得ること，熱効率については送電端（HHV）
で40.5%，環境性については16 %O換算でSO：8ppm，NO：
5ppm，ばいじん：4mg/mN，炭種適合性については灰溶融温度
1400℃以下の低灰融点炭を使用して安定運転できること，経済性
については発電原価が微粉炭火力発電と同等以下となる見通しを得
ることとされている．
将来的には，1500℃級ガスタービンや乾式ガス精製技術の適用
等の技術進展により，48%程度の送電端熱効率が期待できると試算
されている．
