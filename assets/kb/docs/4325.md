# 1・2・4 項の水車比速度(specific speed of hydraulic turbine)

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・1 形式と選定

で整理され、比速度(specific speed)はポンプ水車のポンプ性能のみに使われる。比速度は使用する諸量の単位によって値が変化するので、必ずそれらの単位を併記する。また、羽根車の特性量であるのでランナ1個当たり、ペルトン水車ではさらにノズル1個当たりの値とする。前述の仕様に加えて、回転速度が決まれば比速度が求まる。おもなものを表II-2・1に示す。表II-2・1中、ペルトン水車はランナが利用する水のエネルギーが速度エネルギーのみであるので衝動水車(impulse turbine)、他は圧力と速度エネルギーを利用するので反動水車(reaction tur-bine)と呼ばれ、ポンプ水車は反動形のみである。小水力発電所用として貫流水車(cross flow turbine)、S 形チューブラ水車(S-type tubular turbine)などもある。
