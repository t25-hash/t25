# 2・1・2 形式の選定

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・1 形式と選定

形式のおおよその選定は図II-2・1,II-2・2で行うことができるが、かなり幅があり、最適設計に基づく形式の選定は、回転速度の選定も含め比速度ベースで行われる。ある落差および出力のもとで回転速度を高くすることは比速度を高くすることを意味し、高比速度化は機械を小型化でき、性能・経済性から有利となる。しかし、高比速度化することは周速度の増加とともにランナ内部の流速を増加させることになるので、キャビテーション性能および振動・騒音などが問題となる。すなわち、水車や発電機の強度面から、経験的に落差に対する高比速度化の限界、すなわち比速度の適用範囲が存在する。1・4・1項に示すように、キャビテーションの発生は吸出し高さでも制御できるが、一般に推奨されている吸出し高さにおける従来の実績を形式別に図II-2・3,II-2・4に示す。両端○付きの直線は、最高と最低全揚程時の揚水量で定義したの範囲を示し、曲線の限界線は最低全揚程時のに対応している。これら図中の限界曲線より左側が安全領域であるが、すでにこれを超えて使われている実績もあり、耐キャビテーション材料およびキャビテーション性能の良いランナの開発、さらにはキャビテーション現象の予知と予防技術の進歩などによって、高比速度化へと促進される傾向にある。
