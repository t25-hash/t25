> **節**: 3・4 石炭火力発電

a.はじめに 熱機関のエネルギー変換効率はカルノー効率を
上回ることはできないため，サイクル効率を向上させるためには熱
機関の最高温度を高めることが不可欠である．この点でMHD発
電方式とりわけオープンサイクルMHD発電方式では3000K程度
の高温を最高温度とするためサイクル効率を飛躍的に向上させる可
能性をもっている．MHD発電［MHD(magnetohydrodynamics )
power generation］の原理は，強磁場中を導電性の流体を高速度
で流し，磁場および流れに垂直な方向に電気を取出すものである．
機械的回転部分が不必要で高温を利用でき，低温部分には従来の蒸
気サイクルを用いることにより，総合効率を上げることができる．
従来MHD発電方式はMHD発電機壁の耐久性の点で問題がある
と考えられていたが，最近の地道な努力によってようやく実用化の
展望が見えてきた．すなわち，わが国においては，ナショナルプロ
ジェクトとして旧電子技術総合研究所（ETL）が研究開発した
ETL MarkVII実験装置における400時間以上の耐久試験により，
石油燃焼MHD発電方式における電極の耐久性は4000～5000時間
と評価された．一方，石炭燃焼MHD発電方式においてはアメリ
カAVCO社のMarkVII装置を用いて1200時間以上の耐久試験
が実行され，電極の耐久性は8000時間程度と評価された．現在の
火力発電所の定期検査は1年ごとに約1箇月実施されているので，
実用化のためには11箇月連続運転すなわち8000時間連続運転が
要求される．したがって，オープンサイクルMHD発電方式開発
の現状は，実用化に向けての開発目標が達成されるめどのついたこ
とを示している．オープンサイクルMHD発電方式（open cycle
MHD power generation）においては，作動ガスの導電性を高め
るためカリウム化合物をシード物質として添加するので，燃料中の
硫黄成分が選択的にカリウムと結合して脱硫がほぼ完全に行われ，
余分な脱硫対策を必要としない．
火力発電所の排気ガスから二酸化炭素を回収する場合，空気の代
わりに一部循環した二酸化炭素を含む酸素で燃焼すると，除湿した
排気ガス中の二酸化炭素の割合が空気燃焼の場合の15 %程度から
90%以上になり，二酸化炭素の回収が容易になる．オープンサイク

で得られた高温燃焼ガスのエクセルギーを有効に利用でき，MHD
発電をトッピングサイクルとして結合することは，有力な選択とな
りうる．一方，シュタインバーグらは二酸化炭素回収が困難な分散
的エネルギー使用者から排出される二酸化炭素を削減するため，石
炭，天然ガス，バイオマスなどから効率的にメタノールなどの液体
燃料と炭素を生産する方法を提案しているが，素過程であるメタン
の解離は吸熱反応であり，MHD発電と結合すると，MHD発電下

c .理論数値解析
.はじめに オープンサイクルMHD発電方式の研究開
発における理論・数値解析的研究は1990年代に飛躍的に進展して
いる．ダイアゴナル型（diagonal configuration）MHD発電機チ
ャネルにおいて，MHD発電機動作の非線形性に起因する発電機動
作の分岐現象が京都大学の松尾らにより発見された．この分岐現象
は発電機とディフューザを同時に解析する場合にのみ現れ，MHD
発電機における非線形現象として非常に興味深い現象であるのみな
らず，工学的にも発電出力の突然の大変動の可能性を示唆してお
り，運用上重要な発見である．大規模MHD発電機の安定性に関
して，大型ダイアゴナル型チャネルが不安定になる可能性が指摘さ
れ，その安定化の努力の中で，発電機を低いマッハ数で運転するこ
とが提案され，ディフューザに要求される性能を大幅に低下できる
という有利点も京都大学の岩下らにより発見された．また，これま
でオープンサイクルMHD発電方式大型ディスク発電機は不安定
とされてきたが，筑波大学の石川らにより不安定性の検討がなさ
れ，安定化の方法が提案されたことも大きな進展の一つである．
数値解析による実験結果の検討も進展しており，中国電工研石炭
燃焼ファラデー型（Faraday configuration）MHD発電チャネル
（図3･55），ロシアパルスMHD発電チャネルなど，MHD発電機
チャネル内の物理現象の把握が進んできた．また計算機と解析手法
の進展により，時間依存三次元解析も開始されており，電極近傍に
おける電流集中現象と隣接電極間短絡現象に関して筑波大学の石川
らによりなされ，興味深い現象が解明されつつある．また，MHD
発電機の実用化のためにはどうしても必要なMHD発電機と交流
電力系統との接続の研究に関して，最近10年ほどの研究はほとん
どわが国の研究グループによってなされている．石炭を燃料とする
大規模ダイアゴナル型発電機と大規模ディスク型（disk configura-
tion）発電機を他励式インバータシステムを介して交流電力系統に
接続する場合の検討がなされ，いずれの場合にもインバータの制御
角の制御により短時間で事故から復旧できることが示され，MHD
発電機が系統の安定化に役立つ可能性を示唆している．
