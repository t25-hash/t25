# 2・2・1 水の作用(図II-2・5)

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・2 ペルトン水車(Pelton turbine)

一つのバケットにジェットが流入し始めてから流出し終わるまでのランナの回転角θは比速度が大きいほど大きくなる[図II-2・5(b)].多ノズル機の場合、隣り合うノズル(nozzle)のなす角度が小さすぎると、最初のノズルからの水が出終わらぬうちに次のノズルから水が流入してジェット干渉(jet interference)を起こし、水力効率の低下を引き起こす。このために、多ノズル機はジェット干渉が起こりやすく、比速度の限界があるが、大容量化のため多ノズル化が技術進歩の方向である。
