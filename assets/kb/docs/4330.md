# 2・3・1 水の作用

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・3 フランシス水車(Francis turbine)

図II-2・10に示すフランシス水車の子午断面流路の断面積を、流路幅方向に二等分した流線上での平均流れを仮定する。周方向に平面展開した図II-2・11において、渦巻ケーシング(spiral case)によってある程度の循環を与えられた水は固定翼のステーベーン(stay vane)を通り、可動案内羽根であるガイドベーン(guidevane)の開閉によって、流量(ms)や流入角α(rad)が調整されてランナ(runner)へ流入する。水車出力はに体積効率および機械効率を掛けて求める(1・3・5項参照)。
