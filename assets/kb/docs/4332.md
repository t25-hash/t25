# 2・3・3 特性

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・3 フランシス水車(Francis turbine)

固定羽根ランナをもつフランシス水車の効率は、負荷の大小により大きく変化する。この変化の傾向は比速度によって異なるが、特に욦の大きいランナでは軽負荷および過負荷の効率低下が著しい。また効率は回転速度の変化、落差の変化によっても変わる。図II-2・16はフランシス水車の特性を、横軸に、縦軸にをとって表したものである。αはガイドベーン開度(図II-2・11)、等高線は水車効率を示す。このような図を水車の目玉曲線(1・3・1項、図II-1・29参照)または等効率曲線という。実際の水車では回転速度は一定であるから、一定落差のもとでは図II-2・16の対応する一定のの特性で使用される。また、落差が変化した場合の水車性能の変化は、図II-2・16でが変わった場合に相当する。最図-2・16 フランシス水車の目玉曲線図-2・17 ランナ羽根間に発生する流路渦近では、広い落差に対して良好な性能を得るために回転速度を変えられるようにした可変速水車(adjustable speed turbine,vari-able speed turbine)が採用されることがある。これは回転速度を落差に応じ、最適なに合わせて運転するものである。図II-2・16において、線A より左側ではランナ羽根入口に対する相対流れの入射角が大きくなり、入口負圧面にキャビテーションや流れのはく離による騒音が発生する。また線B より右側では負の入射角が大きくなって、入口圧力面から流れがはく離して騒音が発生する(1・4・2項、図II-1・41参照)。線Cより下の部分負荷領域ではランナ羽根間流路渦(channel vortex)が発生し(図II-2・17の羽根間渦)、運転上制限が加えられることがある。線D はランナ出口流れに旋回成分のない状態を示す。線Eは部分負荷時において、ランナ出口の旋回流が発達して死水コア(図II-2・17の中心部ホワール)が生じ、偏心運動してサージングが最も強くなる状態を示す。サージングが強い場合には、騒音、水圧脈動が大きく、場合によっては電力系統との共振を生じ、電力動揺(power swing)を生じることがある。このような場合の対策として、吸出し管上部へ給気する方法が一般に採用されている。また、無拘束速度は低比速度機で定格回転速度の約1.5倍、高比速度機で約2.3倍である。ガイドベーン開度が大きくなると無拘束速度は大きくなる(図II-2・16)。
