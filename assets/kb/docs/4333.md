# 2・4・1 水の作用

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・4 斜流水車とプロペラ水車

斜流水車(diagonal flow turbine)、プロペラ水車(propellerturbine)においては、それぞれ図II-2・18,II-2・19に示されるように、ランナの子午断面流路の断面積を流路幅方向に二等分したほぼ円すい面および円筒面流線上での平均流れを仮定する。固定翼であるフランシス水車に対して可動翼を用いることが多く、ランナ羽根取付け角が変化するとの流入および流出角βとβも変化し、負荷が変化しても効率の低下が少ない特性をもつ。水車出力はに体積効率および機械効率を掛けて求める(1・3・5項参照)。プロペラ水車についてはオイラーの法則を適用しないで、翼素理論(blade element theory)から導出する方法もある。符号については図II-2・19参照。これを評価するために、最近三次元CFD または三次元粘性流動解析が適用されている(1・5・2項参照)。これによるプロペラ水車内の流動の例を図II-2・20に示す。
