# 2・4・3 特性

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・4 斜流水車とプロペラ水車

斜流水車,プロペラ水車の適用範囲および比速度に対する相対効率を図II-2・25に示す.図II-2・26に立軸可動羽根プロペラ水車(カプラン水車)の特性を示す.θはランナ羽根の取付角度である.図II-2・26のθが一定(例えばθ=22°)の場合の特性は,固定羽根プロペラ水車の特性に極めて近い.図II-2・26から固定羽根プロペラ水車では,運転条件が最高効率点からはずれると効率が急に悪くなるが,可動羽根プロペラ水車では図の鎖線のように広範囲の運転条件に対して効率が良いことがわかる.このことは流量(出力)の変化に対してだけでなく,落差の変化に対しても同様である.図II-2・27に立軸可動羽根斜流水車(デリア水車)の特性を示す.図II-2・27ではフランシス水車との特性比較を示すが,プロペラ水車と同様に,流量変化,落差変化に対し広範囲にわたり高い効率を保持できる.図II-2・28にデリア水車,カプラン水車の主要諸元を示す(図中の욀,욁は図II-2・19参照).これらの中・低落差の水車では吸出し管の損失が全損失に占める割合が大きいため,効率の高い水車の設計ではランナ出口の旋回と吸出し管の形状を注意深く設定する必要がある.
