# 2・5・1 水の作用

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・5 フランシス形ポンプ水車

ポンプ水車(pump-turbine)のランナは、水車とポンプの両特2―68流体機械図-2・29ポンプ水車ランナの入口・出口の速度三角形(a) タンデム式ポンプ水車の例(b) 可逆式ポンプ水車の例図-2・30 高落差揚水発電所機械方式の比較性を兼ね備えていなくてはならない。ポンプ水車のポンプ運転では、鉄管と放水路損失のため全揚程は水車運転の有効落差より大きく、図II-2・29に示すようにポンプ出口に向かって減速するポンプ流れとなる。減速流れにあるランナ内の水力損失を低減するには、図II-2・29に示すように、増速流れのフランシス水車(図II-2・11参照)と比較してランナ入口半径を大きくして絶対速度を小さく、相対速度もランナ内で大きく変化しないようにする必要がある。このため、ポンプ水車のは、フランシス水車に比べて1.3~1.6倍になっている。ポンプ運転では、効率の良い運転範囲で予旋回のない流れが流入するためほぼα=90°で、水車運転ではα=90°でηが最大となる。
