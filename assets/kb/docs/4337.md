# 2・5・2 構造

> **章**: 第2章 水車およびポンプ水車
> **節**: 3・1 項の式(II-2・4)と同じく、

構造が簡単で建屋スペースの経済性からわが国では可逆式ポンプ水車が、かつその中でも立軸のフランシス形ポンプ水車が最も多く用いられている。フランシス形ポンプ水車は単段のものと多段のものがあり、単段のものは800m程度以下の落差に適用される。600m以上の高落差に対して、図II-2・30(a)に示す立軸ペルトン水車と多段ポンプのタンデム式が採用された例があるが、図II-2・30(b)のような多段ポンプ水車(multistage pump-turbine)を採用するほうが機械の全長が短く、建屋寸法を小さくできるため経済的に有利となる。負荷調整をあまり必要としないヨーロッパ地域の発電所では、ガイドベーンを省略したステーベーンのみの多段ポンプ水車が実用化されている。多段ポンプ水車の構造例を図II-2・31に示す。b. 構造図II-2・32に構造の例を示す。渦巻ケーシング、ステーリング、ガイドベーンとその操作機構(図中の456)、ランナおよび吸出し管などの基本的構造は、外観的にフランシス水車と大差がない。しかしながら、ポンプ水車はフランシス水車と比べると起動停止頻度が高く、ポンプ締切などの特性上過酷な運転もあるため、各部品の剛性や信頼性に配慮した設計がなされる。また、日常の保守でもガイドベーン、ランナなどのすきま量の変化、ライナ、軸封装置などの摩耗、しゅう動部のがた、潤滑状況には特に注意が払われる。c. 部品とその特徴ランナは同じ設計要項のフランシス水車に比べて外径が大きく、羽根枚数が少なく(6~10枚程度)、羽根のポンプ出口角度が小さく長さも長くなる。材質は、強度、耐キャビテーション性などに優れた高Ni-13Crステンレス鋼がおもに用いられ(1・4・3項、図II-1・43参照)、一体鋳造または羽根とクラウンおよびバンドを別々に作り、それらを溶接で組立てて製作される。ランナの強度については、高い頻度の始動・停止に対する応力計算とともに、ガイドベーンとランナとの翼列干渉(blade rowinteraction)による振動に配慮する。特に500mを超える高落差ランナでは、ランナの固有振動数が翼列干渉による圧力脈動の励振周波数に近づくことがあり、ランナとガイドベーンの羽根枚数の組合せによる励振条件の回避、有限要素法を用いた振動強度計算、模型および実物での応力測定による応力レベルの検証など、解析・実験両面から構造・材質面の安全性が確認される。また、海水を用いたポンプ水車が開発されており、ランナに海水耐腐食性材料の採用、すきま腐食等を防止するための材質の組合せや腐食電位の高い犠牲電極を適正に配置するなど、材料面や構造面で種々の耐海水対策が施されている。ランナの大きさが輸送の限界を超える場合は、ランナを二分割して製作・輸送し、分割フランジをボルト締めして現地で一体に組立てられる。図II-2・33に二分割ランナ(two-divided runner)の構造例を示す。分割フランジは大きな水中かくはん損失を生むため、フランジ端面を覆って全周に鋼板のカバープレートが溶接またはボルトで取付けられる。ガイドベーンおよびステーベーンは、水車とポンプの正逆両方向の流れを考慮して、入口側と出口側が対象に近い形状にされている(図II-2・29)が、特にポンプ運転時の圧力回復を配慮し、ステーベーンは長く急激な流路面積の変化がないよう設計される。軸受も両方向の回転ができるように設計されており、円筒形またはセグメント形が使用される。ポンプ水車は吸出し高さが深く、主軸封水装置は高い値(面圧×周速度)に耐えるような構造になっている。また、ポンプ始動や調相運転(condenser operation)などの空転運転中の高圧空気を漏洩させない機能ももっており、その冷却に特別な配慮がなされている。ポンプ運転始動時および調相運転時には、圧縮空気によってランナ部の水を押し下げ空転させることよって、ランナの始動抵抗や入力を減少させる方法がとられ、圧縮空気の供給・排気装置が設けられる。空転中は、ランナシール部の過熱を防止するために自動的に冷却水が供給されるとともに、ガイドベーンからの漏水に対応するためのガイドベーン漏水防止装置と、ランナ外周部に滞留する漏水の排水装置が設けられる。は比速度ととも図-2・33 分割ランナの例2―69第II部水力機械に増加するが、0.12~0.15程度である。羽根形状のうち、水車入口角度βは遠心ポンプの出口角度と同じく13°~25°の範囲で選ばれ、出口角度βは比速度や使用される条件によって異なるが、フランシス水車の出口角度と同じように、最高効率点付近で各流線ごとにαがほぼ90°となるように決定される。ガイドベーン枚数はフランシス水車と同じく16~28枚程度で、落差、水車の大きさ、ランナ羽根枚数との組合せなどを考慮して決められる。ステーベーン枚数はガイドベーンと同枚数または半分の枚数が選定される。ケーシング入口の流速は、水車運転とポンプ運転のいずれを重視するかによっても異なるが、だいたい図II-2・35のように、落差が高くなるに従って速い値が選ばれる。吸出し管(draft tube)は、エルボ形曲がり管(図II-2・14参照)に長い放水路トンネルを接続する場合が多く、土木費の節約から出口の形は幅が狭く高さの大きい正方形あるいは長円形とすることがある。ポンプ水車では=3.0~4.0で、高落差機では吸出し管の押し下げ空気の放水路側への漏気を避けるため、吸出し管の高さが大きめにとられる。長放水路に接続して、出口形状を円形あるいは長円形とする場合には、それらの径・幅・高さなどは出口径の1.4~1.8倍程度にとられている。
