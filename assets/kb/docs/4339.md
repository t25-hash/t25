# 1・3・1 項の図II-1・31に流量に関する完全特性の一例を示す.単位

> **章**: 第2章 水車およびポンプ水車
> **節**: 3・1 項の式(II-2・4)と同じく,

回転速度を横軸にとり,縦軸に単位流量または単位トルクをそれぞれ流量特性,トルク特性として表す.図II-1・31の流量特性では,,ともポンプ方向を正に,水車方向を負の座標にとって,ガイドベーン開度ごとに描かれる.第1象限が通常のポンプ運転,第3象限が通常の水車運転領域に相当する.b. 水車とポンプの特性の関係ポンプ水車の水車効率が最高になる回転速度욧は,ポンプ効率が最高となる回転速度욣より小さく(図II-2・36),その比は水車比速度に関しておよそ図II-2・37に示す範囲にある.言い換えれば,一定回転速度のもとでは,ポンプ効率はつねに水車効率が最高となる落差より低い落差で最高になる.これは基本的に水車とポンプの性能によるもので,さらにポンプ流れでは流出角度が羽根角度より小さくなる,いわゆるすべり速度の存在などが関与している.図II-2・36は横軸に[この値は式(II-1・19)で定義されている]をとって,水車とポンプの両効率特性を比較した例を示す.ポンプ水車に直結される発電電動機の回転速度は一般に,発電・揚水運転が最も経済的になるような同一速度に選ばれ,上下貯水池のある利用水深幅で運転される落差領域も,鉄管と放水路の損失により,ポンプ運転の場合のほうが水車運転よりも高いので,軸におけるそれぞれの運転領域もずれることになる.したがって,ポンプ水車の発電電動機の設計の最適化を行うためには,ポンプ水車の振動,キャビテーション特性を配慮しながら落差の変動幅,運転条件ごとの頻度・重要度の配分,発電機出力と電動機入力の値等も考慮して,釣合いのとれた運転領域を選定する必要がある.基本的には,ポンプの効率特性の変化が急峻なので,水車運転領域は,最高効率点より高いの効率のやや低下した領域を使わざるを得ない.水車運転時の効率の高い領域を使用するため,発電電動機の回転速度を変えられるようした例(図II-2・36の2速度タイプ)もあったが,最近では2・9・5項に示す回転速度を±数%連続的に変えられる可変速ポンプ水車(adjustable speed pump-turbine,variablespeed pump-turbine)が実用化されている.ポンプ水車は上下貯水池が人工的に造られることが多く,同一容積に対して利用水深をできるだけ高くとって運用したほうが,土木工事費を含め経済的になる場合がある.利用水深を高くとるとポン図-2・38 吸出し高さ推奨値図-2・39水圧上昇値プ水車の落差変化幅が大きくなり,運転されるが広くなる.この運転範囲の広さを(ポンプ最高揚程)/(水車最低有効落差)で定義する変落差比(operating head ratio)として表し,ポンプ水車の基準落差に対しておよその許容変落差比がある.変落差比が大きい場合,水車では低落差運転領域での振動,ポンプ運転では高落差側の逆流や低落差側のキャビテーションなどが発生しやすくなる.c. 各種特性ポンプ水車の効率が最も良いのはポンプの設計点욦が45~50m,m/s(水車出力比速度욦で140m,kW 程度)のもので,これより比速度が大きくなっても小さくなっても効率は低下する.最高効率から離れた状態における水車特性は,フランシス水車と同様,部分負荷,過負荷において,吸出し管内に不安定な渦心が発生して振動や電力動揺の原因となることがある(図II-2・16参照).また,低落差においては,ランナ入口への流入角度が羽根角度に対して小さくなるため,流れのはく離等によって振動が増加し,高落差のポンプ水車ではさらに羽根圧力面にキャビテーションが発生し,壊食を受ける場合がある.ポンプ運転の高揚程では,遠心ポンプと同様,流量の減少に伴う逆流によって振動やサージングが発生することがあるが,この領域は通常運転では使用しないようにする.ポンプ水車のキャビテーションは,水車運転時よりポンプ運転時のほうが発生しやすいので,吸出し高さ(suction head)욦はポンプ特性から決められる.キャビテーション特性は,それぞれフランシス水車および遠心ポンプとほぼ同様であるが,揚水発電所ではポンプ締切運転など正規状態以外での運転が多いので,実際の욦は種々の運転状態のキャビテーション特性を勘案して決定される.通常推奨される吸出し高さ욦に対するσ値(1・4・2項図II-1・39参照)の選定例を図II-2・38に示す.流量変化(または出力変化)に対する水車効率の低下量は,フランシス水車同様,比速度が大きいほど大きくなる.揚程変化に対する揚水量および軸入力の変化量は,遠心ポンプ同様,比速度が大きいほど少なくなる.無拘束速度(runaway speed)は,フランシス水車に比べてランナ直径が大きいので,定格回転速度の1.35~1.6倍程度である.負荷を遮断した際に水圧鉄管に発生する水圧上昇値は,ポンプ水車のが1・3・1項の図II-1・31の完全特性例に示されるようにに対して大きく変化するため,ポンプ水車の完全特性を用いて,水路条件とガイドベーンの閉鎖特性を基に過渡現象解析による数値計算(1・5・1項参照)が広く使われている.圧力波が鉄管を往復する時間よりガイドベーンの閉鎖時間が長い通常の場合,水圧上昇は鉄管の長さ(m)と遮断直前の流速(m/s)の積にほぼ比例し,実績の一例を図II-2・39に示す.
