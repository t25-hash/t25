# 2・6・1 構造

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・6 斜流形およびプロペラ形ポンプ水車

斜流形ポンプ水車(diagonal flow type reversible pump-tur-bine)は約30~150mの落差、揚程に適用される中比速度に適したポンプ水車であり、プロペラ形ポンプ水車(propeller typereversible pump-turbine)は約20m以下の落差、揚程に適用される高比速度に適したポンプ水車である(2・1・1項参照)。現在実用1:ランナ羽根、2:主軸、3:ガイドベーン、4:ガイドベーン開閉機構、5:カバー、6:主軸受、7:ステーリング、8:ケーシング、9:吸出し管、10:ランナ羽根サーボモータ、11:ランナ羽根開閉機構図-2・40 立軸可動羽根斜流形ポンプ水車の構造2―71第II部水力機械化されているものは可動羽根構造のものが多い。ポンプ運転時と水車運転時とでは回転方向が逆となるが、これ以外に構造的には斜流水車、プロペラ水車と大きな差はない。斜流形ポンプ水車は同じ要項の斜流水車に比べて、ランナ入口径が約1.2~1.3倍、羽根長さは約1.3倍となり、羽根枚数も若干少なくなる。ランナ羽根に発生する応力も、遠心力、水圧力の影響で過酷となるので、ランナ羽根付根の厚さ、応力集中緩和対策などに配慮が必要である。図II-2・40は立軸斜流形ポンプ水車の構造例を示す。外観およびランナ羽根開閉機構などは斜流水車とほとんど同じである。一方、プロペラ形ポンプ水車では、渦巻ケーシングが用いられることはほとんどなく、円筒ケーシングが用いられる。図II-2・41は潮力発電所(tidal power station)用として設計されたプロペラ形ポンプ水車であり、発電電動機はケーシング内におさめられ、チューブラ水車と同様な構造である。斜流形、プロペラ形ポンプ水車では、ランナ羽根を全閉にすることによりポンプ運転時の起動動力が小さくなり、水中で始動できる。大容量機ではフランシス形ポンプ水車と同様に、水面押し下げによる空中起動方式とすることが多い。
