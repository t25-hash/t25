# 2・7・1 S 形チューブラ水車

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・7 小型水車

図II-2・45に示すS 形チューブラ水車(S-type tubular turbine)は、吸出し管をS 形にわん曲させて水車軸を貫通させ、発電機を外部に設置する形式のチューブラ水車であり、落差約20m以下の1:ランナ、2:ガイドベーン、3:ケーシング、4:水車軸、5:吸出し管、6:増速機、7:発電機図-2・45 S 形チューブラ水車1:ランナ、2:主軸、3:ガイドベーン、4:ケーシング、5:吸出し管、6:真空調整弁図-2・46 貫流水車図-2・47 マイクロ水車(横軸3ノズルペルトン水車)低落差地点に採用される。ランナ羽根・ガイドベーンともに可動型、ランナ羽根かガイドベーンのどちらかを固定とするなどの形式がある。ランナ出口での排棄速度エネルギーが有効落差の約30%もある軸流水車では、水車効率に及ぼす吸出し管の影響が大きい。このために吸出し管に貫通軸および二重の曲がりをもつS 形チューブラ水車の効率は、流路が直線状のバルブ形チューブラ水車に比べて低下する。
