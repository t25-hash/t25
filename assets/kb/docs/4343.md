# 2・7・2 貫流水車

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・7 小型水車

貫流水車(cross flow turbine)は,衝動形と反動形の中間的な性格をもつ水車であり,落差5~100mで比較的流量の少ない地点に採用される.ランナは円筒かご形で円弧状の羽根20~30枚をもつ.ガイドベーンは1枚で通常軸方向に二分割されており,負荷に応じて切換え運転を行うことにより広い負荷範囲で良好な効率が保てる.図II-2・46に示すように,吸出し管を設置してランナから放水位までの落差の回収が行われる場合もある.ランナからの出力軸図-2・48 マイクロ水車(立軸ペルトン水車)図-2・49マイクロ水車(横軸円筒形プロペラ水車)はケーシング側面を貫通しており,増速機を設置して発電機の小型・高速化によってコスト低減を図る組合せ例が多い.
