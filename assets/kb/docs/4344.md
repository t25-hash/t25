# 2・7・3 マイクロ水車

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・7 小型水車

出力100kW程度以下のマイクロ水力では，さらに小型化，構造
簡素化およびメンテナンスフリー化が追求されており，構造が簡単
なペルトン水車やプロペラ水車などの機種が採用されている．
ペルトン水車（2･2節参照）は，高落差地点で流量の少ない地点
に採用される．流量調整をノズル開度とノズル本数で調整できるた
め，低流量まで高い効率で運転することが可能である．通常の横軸
ペルトン水車は2ノズルが採用されてきたが，図Ⅱ-2･47に示した
例では3ノズルを採用して，出力に対する体格の小型化，高速化が
図られている．また，固定ノズルを採用するなどして構造が簡素化
されている．図Ⅱ-2･48は立軸ペルトン水車の例であるが，ハウジ
ング側面に発電機が配置されており，ベルトを介して水車軸と発電
機軸が結合されるなどコスト低減が図られている．
図Ⅱ-2･49は横軸円筒形プロペラ水車の例である．流量が大きい
場合は並列に，落差が高い場合は直列に同一の水車を複数台設置し
て，流量および落差を分担する．水車の流路の外に発電機が抱き合
わせにされてベルト機構などで駆動し，水車の入口・出口の配管を
すれば簡単に据付けられ，発電を始められるように工夫されてい
る．また，大型水車のようにランナを可動とすると構造が複雑とな
るので，固定または手動で，例えば季節的な流量変動に対応して動
かせるようにしている．
