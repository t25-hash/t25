# 2・8・1 調速機(governor)

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・8 付属装置

a. 概要調速機のおもな使用目的には、(1)水車の始動および水車に直結する発電機の並列までの揃速作用、(2)並列運転中の系統周波数の変化に対する速度調整や発電機出力のレベル調整、(3)事故時の負荷遮断または水車停止などが挙げられる。このほか、発電所の運用により、系統の周波数変動に応動しない負荷制限運転を行うこともあり、水位調整装置なども付加されることがある。また(3)に属するが、自動周波数制御、経済負荷配分制御などもある。調速機は発電所の自動運転方式に組込まれて、発電所全体2―73第II部水力機械図-2・50 水車制御系のブロック線図の集中制御をつかさどる別置の配電盤からの指令で自動的に始動、停止を行うほか、ランナ室の水面を押し下げランナを空転させる調相運転などの運転モード切換えも自動的に行う。調速機の調速作用は所定の回転速度に達してから生きるようになっているので、所定の回転速度に到達するまでのガイドベーン制御や、ポンプ運転時の揚程に応じたガイドベーン適正開度設定には、負荷制限装置が用いられる。さらに、保護リレーなどの動作によって急停止や非常停止指令が与えられれば、ガイドベーンを急閉して速やかに水車を停止する。また、小水力などでは速度検出要素をもたず、ガイドベーン開度調整のみを行うスピーダレスガバナも採用されている。調速機は主としてその速度検出部および増幅部の形式によって、機械式調速機、アナログ電気式調速機およびディジタル式調速機に大別されるが、最近の水車用調速機のほとんどは性能、保守性に優れた電気式が採用されている。水車のガイドベーンを操作する最終段の増幅には、油圧サーボ機構を使用するのが一般的であるが、近年においては発電所の油レス化指向に沿って、電動機で油圧シリンダを直接駆動する電動サーボ機構も広く採用されている。b. 水車調速機の構成図II-2・50に調速機を含む水車制御系のブロック線図の代表例を示す。. 速度検出部機械式調速機の場合は、水車発電機の回転速度をベルトまたは電動機により速度検出部(スピーダ)に伝達し、このスピーダが回転速度に応じて変位して速度検出する方式が一般的である。アナログ電気式あるいはディジタル式調速機の場合は、発電機に専用の信号発電機やパルス発信器を直結させ、回転速度を正弦波やパルス列の形で入力し、これを共振回路やパルス計数回路を介して回転速度に応じた信号に変換する。. 演算増幅部機械式調速機の場合は、レバー機構を使った比例演算のみであるが、アナログ電気式あるいはディジタル式調速機の場合は、機能的にP(比例)、I(積分)要素に、D(微分)要素を加味してより即応性に優れ、安定性の高いPID 型調速機が多く採用されている。PID 型調速機の場合は下記v項の弾性復元は不要である。アナログ電気式ではアナログ演算素子によって、ディジタル式ではPLC(programmable logic controller)のソフトウェア処理によってこれらの制御信号演算が行われ、これが変換部に伝えられて機械式変位となり、後段の油圧増幅部に入力される。. 油圧増幅部電気-油圧コンバータとこれによって操作される補助サーボモータから構成され、後段のガイドベーン操作機構に対して十分な力と変位の操作信号を伝えるためのものである。. ガイドベーン操作機構補助サーボモータからの操作信号を受ける主配圧弁と、ガイドベーンサーボモータおよび復元機構から成る最終段の油圧増幅部である。. 復元部剛性復元と弾性復元がある。前者は補助サーボモータの開度に比例した信号を演算増幅部に復元し、回転速度と補助サーボモータ開度の関係を一定に保つものである。後者は機械式ではダッシュポット、電気式では抵抗・コンデンサ回路による不完全微分演算部で、各種増幅部の動作遅れの補償と水撃による不安定化作用(d項参照)の抑制を図るものである。弾性復元を付けた調速機をダンピング型調速機と呼び、弾性復元が不要なPID 型調速2―74流体機械図-2・51 速度調定率、速度垂下率および周波数調整図-2・52 水車の速度制御系のブロック線図機と対比させる。. 速度調整装置発電機が電力系統に並列されるまでは基準速度設定機能を果たし、並列後は出力調整機能を果たす。. 負荷制限装置速度検出部からの制御信号に関係なく、ガイドベーンを操作するときに用いる。速度調節演算部出力と負荷制限設定器の設定値の低い信号を、演算部から出力する方式が一般的である。. 急停止装置補助サーボモータあるいは主サーボモータの閉側を圧油に、開側を排油に強制的に切換える電磁切換弁を含む機構で、水車の急停止や発電機の非常停止を要する場合に用いる。c. 調速機の性能代表的な項目を次に示す。. 不動帯主サーボモータが開動作から閉動作、あるいは閉動作から開動作に反転するのに要する最小回転速度変化の定格回転速度に対する割合で、一般的に0.02%程度に設計される。. 速度垂下率図II-2・51に示すように、回転速度(周波数)の変化と主サーボモータの変位の関係を示し、一般に0~6%の間で自由に変えられるように設計する。.
