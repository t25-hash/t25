# 2・8・2 入口弁(inlet valve)

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・8 付属装置

水圧管の末端で,水車・ポンプ水車のケーシングの直前に設けられる.小流量および中・高落差の領域にはロータリ弁を,中・低落差の領域にはちょう形弁が主として用いられる.これら各種入口弁のおおよその適用範囲を図II-2・53に示す.最近ではスルース弁はほとんど用いない.なお,ちょう形弁には,弁体断面がレンズ形をしたものと,弁体が全開時に流れと平行になる2枚の板,およびそれを支えるリブとから成る複葉弁と呼ばれるものがある.複葉弁は比較的新しい形式のものであるが,ロータリ弁とちょう形弁の特徴を併せ持っており,最近ではさらに高落差の領域まで拡大適用されるようになってきた.入口弁は一般に,水車・ポンプ水車が事故のときでも単独に閉じうるように設計製作されるのが一般的である.図II-2・54はロータリ弁の一例である.弁体は弁軸およびレバーを介して油圧サーボモータ,または水圧管の圧力が高いときには水圧サーボモータによって開閉される.また,小型のものでは電動サーボモータにより駆動されることもある.弁閉鎖時の漏水を少なくするため,ロータリ弁では図II-2・55のように全閉時のみ作動する可動シールを備えている.ちょう形弁の場合は,流水中にある円板状の弁体が回転し管路を閉じる構造になっている.弁の操作はロータリ弁と同じくサーボモータが使われるが,全閉時の漏水防止は弁体の外周に固定のゴムパッキンを入れ,弁胴に押付けて行う方式が多い.入口弁は一般に主弁とバイパス弁から成り,開くときにはまずバイパス弁を開き,上下流の水圧がバランスしてから主弁を開き,閉じるときには逆に主弁を閉じてからバイパス弁を閉じる.バイパス弁の口径は,水車のガイドベーンからの漏水量を十分に補うことができるように決める.
