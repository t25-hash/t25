# 2・8・3 制圧機(pressure regulator)

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・8 付属装置

制圧機は,水車のケーシングや水圧管の水圧上昇の軽減を目的として,ケーシングやその上流の水圧管を分岐して取付けられる自動排水装置である.ガイドベーンサーボモータに連結されて動き,負荷が急減して水車の回転速度が上昇したときに調速機はガイドベーンを急閉するが,他方で制圧機は開いてケーシングまたは水圧管より水が放出され,水圧管内流速の急変を防いで水撃現象を軽減する.図II-2・56に負荷遮断時のガイドベーンと制圧機の水口変化の例を示す.ガイドベーンの水口変化AC′と制圧機の水口変化B′CDEとを合成した水口変化ABCDEによる,水圧管内の水圧上昇値が許容値以内になるよう制圧機の口径が決められる.制圧機の取付け適否は,水圧管の水圧上昇値および発電機の慣性モーメントの増加による価格上昇と,制圧機取付けによる経済比較をして決める.中落差以上のフランシス水車やペルトン水車で用いられたが,高張力鋼の採用や回転軸系の過速度耐力の向上などによって現在ではあまり用いられない.
