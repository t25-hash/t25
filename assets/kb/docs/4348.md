# 2・9・1 運転モード

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・9 運転と制御

水車・ポンプ水車は主として発電または揚水のために運転される図-2・57 発電および揚水運転の起動および停止の制御シーケンス2―75第II部水力機械ほか,ポンプ水車では待機運転(standby mode operation)や調相運転といったモードで運転される場合がある.待機運転は,電力系統の発電量または揚水負荷を急増したいときに備えて,発電機を電力系統につないだままポンプ水車のランナまわりの水を圧縮空気で押し下げて空転している状態である.調相運転はポンプ水車にとっては待機運転と同じであるが,発電機または発電電動機が電力系統の力率調整のために,調相機として運転されている点が待機運転と異なる.
