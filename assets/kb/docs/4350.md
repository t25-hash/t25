# 2・9・3 過渡現象

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・9 運転と制御

水車、ポンプ水車の起動停止や負荷の急増急減などのように、運転状態を急変させると回転速度、流量、水圧などが変動する。これを過渡現象(hydraulic transient)という。発電所の水路系や発電機、水車などの主機の設計に当たっては、水車負荷遮断、ポンプ入力遮断など変動の大きい過渡現象を十分検討しておかなければならない。a. 水車負荷遮断水力発電所の水車負荷遮断の例を図II-2・58に示す。負荷遮断(load rejection)によって水車の入出力のバランスが崩れてランナが回転上昇すると、調速機がそれを検出してガイドベーンを急閉鎖し、回転上昇を抑える。それに伴い流量が急減し、管路の水撃作用により鉄管水圧が上昇して最大水圧に達する。その後、圧力波が鉄管内を往復して水圧変動を繰返しながら減衰していく。負荷遮断で生じる最大水圧、最高回転速度は、ガイドベーンの閉鎖速度、回転部のはずみ車効果によって変わり、完全特性と与えられた水路条件をもとに、過渡現象解析によって負荷遮断特性を数値的に計算することができる。水車および水圧鉄管は、これらの過渡現象により生じる最大水圧、最高回転速度を考慮して設計される(1・5・1項参照)。負荷遮断の場合の水撃による水圧変動は、水口開度(ペルトン水車のニードル開度、フランシス水車、プロペラ水車ではガイドベーン開度に相当)が近似的に直線閉鎖とするとみなせる場合は、水口開度の閉鎖時間より略算することができる。速度変動率(momentary speed variation)δとは、水車の負荷が急変したときの過渡状態における回転速度の変化量と定格回転速度との比をいい、次の式で表される。このδは、急変する負荷の大小、調速機、水車の特性ならびに発電機の回転部のはずみ車効果などによって決まる。負荷遮断時のδを所定の値にするために、水車の要求するの計算式としては次のものがある。δ,δは、水圧管、水車発電機を含めた発電所全体の経済性から設計値が決められる。通常、δは30~40%であるが、60%くらいまで選定される場合がある。δは水圧管路の強度とサージタンクの形状寸法などのよって制限され、普通20~50%の値をとる。c. ポンプ入力遮断揚水発電所のポンプ入力遮断の例を図II-転は急速に低下し、流量の急減に伴い鉄管水圧の低下が起こる。ここで、水圧管の最低水圧および回転速度の減速度は、水圧管路内の水の慣性との比、ポンプ水車の特性によってほとんど決まり、ガイドベーンの閉鎖速度はほとんど影響しない。ポンプの締切からポンプ制動領域に入るときは、ガイドベーン開度はすでに小開度となっているが、ガイドベーン開度によって流量が急変するため、水圧降下後に生じる水圧上昇は、逆にガイドベーンの閉鎖速度によって決まる。水圧降下を少なくするには、を増すか水圧管路内の水の慣性を小さくする。d. 吸出し管内の水柱分離水車の負荷遮断では吸出し管側の圧力が低下するが、放水路側のサージタンクがなく放水路が長いと、吸出し管内の圧力が水の飽和蒸気圧以下となって水柱分離(water column separation)を起こすことがある。水柱分離が発生すると、放水路出口で反射して戻ってきた圧力波によって分離した水柱が再結合した際、異常な圧力上昇を発生する。この異常な水圧上昇は機器に重大な障害を与えるため、過渡現象解析によって吸出し管内の最低圧力を予測する。ポンプ水車は地下発電所が多く、長大な放水路を有するので、水柱分離を起こさないための十分な配慮が必要である。
