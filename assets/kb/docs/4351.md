# 2・9・4 振動・脈動

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・59 に示す.入力遮断(input failure)が発生するとランナの回

水車の運転中に生じる振動・脈動の原因として,次のようなものがある.(1) 軸受すきまの過大,主軸の曲がり,たわみ,回転体の不釣合いなどによる軸振動.(2) 非設計点の運転におけるランナ周辺のはく離や二次流れによる静的な不釣合い力(1・2・5項参照),および水圧脈動(pressure fluctuation)に基づく軸振動.(3) ガイドベーンの後流と回転するランナ羽根との干渉による水圧脈動に基づくランナの振動,および静止側流路部の振動.(4) 部分負荷や過負荷時に,フランシス水車,プロペラ水車のランナ下部に発生する渦心に起因した水圧脈動,およびサージング(2・3・3項参照)による騒音,振動.(5) ランナ羽根出口端におけるカルマン渦とランナ羽根が共振することによる鳴音,振動.(6) ランナおよび吸出し管内のキャビテーションによる騒音,振動.(1)(2)については工作精度を維持することはもちろんであるが,必要に応じてコンピュータによる軸系の振動解析を行い,回転体不釣合いやランナに加わる水力加振力による軸振動を,許容値内に収めるのに必要な軸や軸受の剛性を求めて設計を行う.(3)についてはランナとガイドベーンの間隔を大きくするか,羽根枚数を変える.また,羽根入口を傾斜させ,ガイドベーンの後流をランナ羽根入口が同時によぎらないよう配慮する.(4)(6)に対しては,吸出し管内に空気を送入することによって軽減できる.(5)は,ランナ羽根の出口端近傍の形状を,例えば表面と裏面とを非対称な形状に変えればよい.
