# 2・5・3 項に示すように,水車とポンプの最高効率点の回転速度は異

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・59 に示す.入力遮断(input failure)が発生するとランナの回

なる.回転子を周波数変換装置を介して交流励磁する交流励磁方式発電電動機は,可変速幅が数%程度に制限されるが経済性は高く,ポンプ水車の可変速化に用いられ実用化されている.可変速ポンプ水車の水車運転では,同期回転速度より低い回転速度で運転するこ図-2・60 可変速ポンプ水車の構成とによって,定速機より水車の最高効率点に近い特性で運転でき,効率向上と運転範囲の拡大が可能である.また,回転体のフライホイールエネルギーを放出または吸収することで,過渡的に回転速度が変化して高速の電力応答を行うことができる.
