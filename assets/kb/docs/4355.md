# 2・10・2 発電方式

> **章**: 第2章 水車およびポンプ水車
> **節**: 2・10 水力発電設備

水力発電所の種類を、落差を得る方法と運用方法により分類すると、次のようになる。a. 落差を得る方法による分類(1) 水路式(conduit type)河川の水をせき上げ、取水口から比較的長い水路を経て発電所に導き、その間の河川との高低差により落差を得る方式。自流式(流れ込み式)発電所に多い。(2) ダム式(dam type)河川をせき止めて高いダムを造り、このダムに貯水して落差を得る方式。この方式では年間あるいは月(週)程度の発電調整を行うようにしたものが多い。(3) ダム水路式(dam and conduit type)ダムと水路の両方によって落差を得るようにした方式。b. 運用方法による分類(1) 自流式(流れ込み式、run-of-river type)河川流量を調整せず自然流量のまま発電する方式で、一般には出力調整を行わない。豊水期には水がむだに流れてしまい、渇水期には出力が減少する。(2) 調整池式(pondage type)ダム上流または水路中間などに調整池を設け、負荷変化に応じて1日または1週間程度の出力調整を行う方式である。(3) 貯水池式(reservoir type)河川の水を貯水池に蓄え、年間の季節的な負荷変動や河川水量の変化に応じて、出力調整を行いながら発電する方式である。2―77第II部水力機械図-2・61 ダムの種類(断面)(4) 揚水式(pumped storage type)深夜などの軽負荷時の火力、原子力の余剰電力を利用して下池から上池に揚水し、せん頭負荷時にその水を利用して価値の高い電力を得る方式である。上池の流域が小さく自然の流入量がほとんどないものを純揚水といい、自然流入の水も合わせて利用するものを混合揚水という。
