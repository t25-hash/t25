# 3・2・3 ハイブリッドロケット

> **章**: 第3章 推進工学
> **節**: 3・2 固体ロケットモータ

ハイブリッドロケット（hybrid rocket ）は推進薬に液体と固体
の組合せを用いるロケットで，液体推進薬を用いる液体ロケット，
固体推進薬を用いる固体ロケットに対して，両者の混成
（hybrid）であることからハイブリッド推進薬ロケットエンジン
（hybrid propellant rocket engine），略してハイブリッドロケット
といわれる．液体酸化剤と固体燃料，および液体燃料と固体酸化剤
の二とおりの組合せが考えられるが，通常使用されるのは前者の，
液体酸化剤と固体燃焼の組合せである．
a.特徴 ハイブリッドロケットは以下に示すような長所をも
つ．
(１)安全：燃料と酸化剤が完全に分離されているため，事故等に
よる爆発の危険性がない．
(２)推力制御・停止および再点火：液体酸化剤の流量を制御する
ことにより，推力制御やオンオフ制御を行うことが可能であ
る．
(３)低公害：燃焼ガスに塩酸や酸化アルミニウム等の汚染物質を
含まない推進薬の組合せを選択することが容易である．
(４)ロバスト性：固体燃料にクラックが生じても不感である．
(５)低温度感度：推進薬温度が燃焼速度にほとんど影響を及ぼさ
ないため，エンジン性能が打上げ環境温度の影響を受けな
い．
(６)安価：燃料，酸化剤ともに安価であり，供給系が1系統で済
むため，エンジンが安価である．また，エンジンの安全性に
より安全対策の簡略化や市販レベルの部品の選択が可能であ
り，システム全体でも低コストとなる．
一方，短所をまとめると以下のようになる．
(１)低推力：固体燃料の燃焼速度が低いため，大推力発生には不
向きである．
(２)低充てん密度：固体燃料内部に液体酸化剤および燃焼ガスの
流路を設ける必要があるため，固体燃料の充てん密度が低く
なる．
(３)低燃焼効率：燃料と酸化剤の混合不十分および燃え残りの発
生により，低燃焼効率になりやすい．
(４)混合比（O/ F）シフト：燃焼の進行により固体燃料の形状が
変化するため，O/Fが燃焼経過時間に従って変化し，比推
力の低下を招く．
b.原理 代表的なハイブリッドロケットの概念を図3･39に
示す．液体酸化剤は加圧され，制御弁を経て噴射器から燃焼室に噴
射される．燃焼室には固体燃料が充てんされ，点火器により燃焼が
開始する．酸化剤および燃焼ガスは固体燃料の軸方向に設けられた
ポートを流れ，ポート側壁が燃焼面となる．

d.燃焼速度 ハイブリッドロケットでは，霧化・ガス化した

傍で燃焼が起こる．基本的には酸化剤とガス化した燃料成分との拡
散燃焼である．燃焼面近傍の様子を図3･40に示す．固体燃料表面
