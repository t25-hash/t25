# 3・3・2 スクラムジェット系エンジン

> **章**: 第3章 推進工学
> **節**: 3・3 空気吸込エンジン

スクラムジェット（scramjet ）とは，燃焼器内の流れが超音速
であるラムジェットを指し，ラムジェットに超音速燃焼（super-
sonic combustion）の頭文字SとCを冠して，そのように呼ばれ
る．超音速燃焼の意義，必要性は，亜音速燃焼と対比させて次のよ
うにまとめることができる．
(１)静温が低く抑えられるため，燃料エネルギーが有効に利用で
きる．
(２)静圧が低く抑えられるため，エンジン構造への強度的，熱的
負担が緩和される．
(３)インレットでの減速に伴う総圧損失が緩和される．
これらの超音速燃焼の採用による利点は，飛行マッハ数がおよそ
6を超えれば有効であるといわれている．エンジンとして効率的に
作動するためには，それ以下の速度で亜音速燃焼を併用することが
理想的である．このような併用作動を行わせるエンジンをデュアル
モードラムジェットと呼ぶ．ラム・スクラムジェットは自らの飛行
によって取込む空気を酸化剤として利用するため，作動には一定以
上（通常マッハ1.5～2.0以上）の飛行速度までの初期加速用エン
ジンを必要とする．ラム・スクラムジェットと初期加速用エンジン
を単一のエンジンとして組合わせる形式を複合サイクルエンジンと
呼ぶ．
スクラムジェットエンジンの研究開発は1950年代にさかのぼる
が，それらについての記述は他に譲り ，ここでは最近の研究状
況，とりわけエンジンの地上試験および飛行試験について紹介す
る．
a.諸外国のスクラムジェット研究開発の状況 諸外国におけ
る今後のスクラムジェット研究開発計画については，3･3･3項で網
羅的に記述されるため，ここでは，これまでの主要な研究開発成果
についてのみ述べる．
近年の動きで特筆すべきは，実際の飛行環境でエンジン性能を確
認することを目的として，世界の研究開発担当者たちが永年追求し
続けたスクラムジェット飛行試験の実現への動きである．過去にも
アメリカのX-15に代表されるようにその企ては何度か見られた
が，最近まで実現には至っていなかった．しかし，ここ数年その動
きが再び活発化し，ロシアやオーストラリア，さらにごく最近はア
メリカにおいても実現にこぎつけている．

アメリカではNASAのラングレー研究所がスクラムジェットの
研究開発をリードし，過去30年，20基以上のスクラムジェットの
試作・地上試験を行っている．1983年には，同研究所の成果に基
づき，スクラムジェットを主推進エンジンとする単段式宇宙輸送機
の実現のための試験機X-30の開発を意図した，極めて野心的な
NASP（National Aero- Space Plane）計画を開始した．しかし，
当時の関連技術，特に推進系技術が開発を進める段階に至っていな
いと判断され，1994年には中止に至った．総額約20億ドルを注ぎ
込んだ同計画で得られた成果は極めて貴重な資産として受け継が
れ，その後の関連技術開発，研究に有効に活用されている．
アメリカの近年の動きとしては，飛行試験を目標とするHyper -

ジンデータを取得する．合計3回の試験を計画し，飛行条件は最初
の2回がマッハ7，3回めがマッハ10である．試験に要する期間は
6年間で，機体3機の製作を含む資金，総計は1億8500万ドルで
ある．エンジンの概略サイズは幅20インチ，長さ30インチで，マ

では超音速燃焼を意図している．2001年6月，マッハ7試験の初
号機を打ち上げたが，ブースタロケットで加速上昇中，安定翼が破
損し試験速度に達する前に墜落，失敗した．その原因調査と対策の
後，2004年3月には2号機を同じ条件で打ち上げ，機体組込形式
のスクラムジェットで世界初の飛行試験成功を収めた．さらに，
2004年11月にはマッハ10試験を成功させた．
上記とは別に1995年からは，NASP計画のフォローオンとし
