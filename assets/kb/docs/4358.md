# 3・1・1 分類

> **章**: 第3章 ターボポンプ
> **節**: 3・1 分類と選定

羽根車をケーシング内で回転させ、液体にエネルギーを与えるポンプをターボポンプ(turbo pump)という。ターボポンプの形式は表II-3・1に示すごとく遠心ポンプ、斜流ポンプ、軸流ポンプに分類される(図II-3・1~II-3・5)。遠心ポンプは、羽根車から吐出される流れが主として主軸に垂直な面内にあるポンプであり、羽根車の吐出し側に直接渦巻形ケーシングをもつ渦巻ポンプと、ディフューザをもつディフューザポンプとがある。斜流ポンプは、羽根車から吐出される流れが主軸の中心線を軸とする円すい面内にあるポンプで、一般には羽根車の吐出し側に案内羽根形のディフューザをもっているが、渦巻形ケーシングをもつ渦巻斜流ポンプもある。軸流ポンプは、羽根車から吐出される流れが主軸と同心の円筒面内にあるポンプで、通常、案内羽根形のディフューザをもっている。ターボポンプは形式により、さらに表II-3・2のごとく分類される。特定の形式のポンプを呼ぶときには、これらのうちから必要なものを選び適宜配列する(例えば、横軸水平割り形両吸込渦巻ポンプ、立軸輪切り形三段ディフューザポンプなど)。なお、これら以外にも据付けおよび支持方式、特殊構造、軸スラスト釣合い方式を示す分類があり、さらに用途別の呼び方によっても分類されている。
