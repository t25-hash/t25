# 3・1・4 キャビテーション(cavitation)

> **章**: 第3章 ターボポンプ
> **節**: 3・1 分類と選定

与えられた吸込全ヘッドにおいて,キャビテーションが発生しな2―82流体機械いよう仕様と形式の選定を行う.キャビテーションについては1・4・1,1・4・2項および1・4・3項に詳説されているので,ここでは選定の手順について述べる.まず,与えられている吸込全ヘッド욦より有効吸込ヘッド(またはNPSH)욦욋を求める.飽和蒸気圧力を욋(Pa)とすると,욦욋=욦-욋ρ(II-3・3)次に羽根車がキャビテーションを発生させないために必要とするヘッドNPSHを求め,욦욋>NPSHであることを確認する.これがキャビテーションを発生させない条件である.NPSHを正確に求めるには実験等の必要があるが,一般的にはトーマの臨界キャビテーション係数σ욧,または吸込比速度を使用して욦욋を求める[CFD(computational fluid dynamics)技術の進歩により現在では解析的に求めることも可能になってきている].キャビテーション係数および吸込比速度の詳細は1・4・2項を参照されたい.
