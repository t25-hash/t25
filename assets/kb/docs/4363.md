# 3・2・1 構成

> **章**: 第3章 ターボポンプ
> **節**: 3・2 遠心ポンプ

遠心ポンプ(centrifugal pump)を構成する部分または部品を回転部と固定部に分け、それぞれを機能別に分類すると、表II-3・3のようになる。ここでは、遠心ポンプの部分あるいは部品を、機能回転部羽根車スリーブ羽根車リング⎫⎬⎭水力性能部を兼ねることが多いバレルケーシングガスケットボリュートディフューザ案内羽根戻り流路吸込流路吐出し流路ケーシングカバーパッキン箱パッキンメカニカルシール固定部図-3・12 片吸込渦巻ポンプ上から以下の六つに分けている。(1) 水力性能部流体にヘッドを与えたり、減速により速度ヘッドを圧力ヘッドとして回復する部分である。(2) 圧力容器部取扱い液を外部と遮断する部分で、一般にはケーシングが水力性能部を兼ねながらこの部分を構成するが、高圧のバレル形ポンプのように、別の部品で構成することもある。(3) 軸封部圧力容器部と軸との間の漏れを防ぐために必要となる部分である。(4) 耐摩耗・耐食部回転部と固定部が細いすきまを形成する部分などで、摩耗しやすい部品を交換可能にするために用いられる。また、腐食性液を扱う場合、耐食性材料で接液部を被覆する部分などである。(5) 動力伝達部この中にはポンプ特有のものとして、軸スラスト釣合い装置がある。(6) その他付属装置ではあるが、ポンプ運転上必要になるものである。図II-3・12に、遠心ポンプとして最も代表的な片吸込渦巻ポンプ(single suction volute pump)の構造を示す。
