# 3・1・6 戻り流路と渦巻ケーシング

a.戻り流路 多段ポンプや圧縮機では，ディフューザを出た
旋回流れを次段の羽根車に導くために，図Ⅰ-3･14に示すように軸
対称曲がり流路（Uターン部）によって内向きの旋回流に変え，
案内羽根を設けた戻り流路（return channel ）に導いて旋回を除去
図 -3･14戻り流路
する．
軸対称曲がり流路内にはふつう案内羽根を設けない．この部分の
流れは，旋回成分による遠心力と，メリディアン断面内の曲がりに
よる遠心力を受けて外側に偏り，内側では流れが壁面からはがれや
すい．曲がり流路（Uターン部）入口で流れが内壁側に偏ってい
ても，半径の最も大きい曲がり中央の断面では外壁側に偏り，断面
内で流れの入れ替わりが行われる．曲がり流路出口では流れの混合
が進み，断面速度分布は一様に近付くから ，下流に設けられた
案内羽根は比較的良好な流入状態になる．なお，軸対称曲がり流路
の圧力損失は主として壁面摩擦によるもので，あまり大きくな
い ．
戻り流路の上流が羽根なしディフューザの場合には，戻り流路へ
の流入角が運転状態によって変化するので，小流量では案内羽根へ
の入射角が過大となって，流れが羽根面からはがれやすい．このた
め，案内羽根の前半はゆるやかに曲げて翼負荷を軽くし，後半でや
や強く曲げて流れを半径方向に導くと良い結果が得られる ．
b.渦巻ケーシング 羽根車外周から出た流れを，直接あるい
はディフューザを経たのちに集めて吐出し管に送り出すために，渦
巻ケーシング［ポンプではボリュート（volute），送風機ではスク
ロール（scroll ）と呼ばれる］が用いられる．羽根車から流出する
流れは，壁面摩擦を無視すると，式(Ⅰ-3･38)で与えられるように，
周速度成分およびメリディアン速度成分ともに半径に反比例するの
で，流れ角α≡tanは半径によらず一定となり，流線は対
数らせん＝expθtanαを描く（ は対数らせんの基礎円半
径）．したがって，渦巻ケーシングの形状を，設計流量における羽
根出口流れの流れ角に合わせた対数らせんにすれば，設計点におい
て軸対称な流れが実現でき，羽根車の外周に沿って圧力が一定にな
り，半径スラストは生じない．実際の流れでは壁面摩擦が働くが，
境界層の影響はあまり大きくなく，周速度成分はほぼ自由渦分布に
近く，流れ角の半径方向への変化も比較的小さい．
渦巻ケーシング内の周方向の速度を断面内で一定と近似し，設計
点において羽根車から流出する流量に比例して，渦巻ケーシングの
各断面の面積を定める方法もある ．
羽根付きディフューザの下流に渦巻ケーシングが設けられる場合
には，渦巻ケーシングに流入する流れはディフューザ羽根によって
定まり，流量によってあまり変化しない．
一方，羽根車から直接あるいは羽根なしディフューザを経て渦巻
ケーシングに流入する場合には，羽根出口流れが流量とともに変化
するので，渦巻ケーシング流れも流量とともに周方向に変化するよ
うになる．もしも渦巻ケーシング流れの速度が，流量に比例して全
周で一様に変化するとすれば，流量によらずつねに軸対称な流れが
実現されるはずである．しかし，羽根出口流れの周速度成分は，速
度三角形から，ケーシング流れとは逆に流量の増加とともに減少す
るので，渦巻ケーシング流れは設計流量以外では周方向に変化する
ようになり，非軸対称な流れになる．すなわち，渦巻ケーシングは
設計流量で軸対称流れを実現しうる渦巻形状に設計されるが，過大

流量では流れにノズル作用を及ぼし，舌部（tongue，巻始め，水
切りともいう）から出口のスロートに向かって速度を増大させ，圧
力は図Ⅰ-3･15に示すように低下する ．一方，部分流量ではディ
フューザ作用により下流に向かって速度を低下させ，圧力は増大す
る．その結果，設計流量点以外の流量では羽根車からの流出状態も
非軸対称になり，羽根出口下流の半径比＝1.10におけるメリ
ディアン速度成分 は，図Ⅰ-3･16に示すように，過大流量では
周方向に増大し，部分流量では減少する ．いずれの場合もスロ
ート近くの圧力は流量によりあまり変化せず，その圧力分布に基づ
いて羽根車には半径スラストを生じる ．この場合，羽根出口流
れの周速度と渦巻ケーシング流れの速度の差は，設計流量から外れ
るとともに大きくなり，混合損失を発生する．
正規流量において，圧力が周方向に一定となる断面積の分布をも
った渦巻ケーシングよりもやや広い渦巻ケーシングにすれば，羽根
車から流出した流れが渦巻ケーシングに流入するときに混合損失を
生じるが，渦巻ケーシング内の平均速度が遅いために摩擦損失が少
なく，渦巻ケーシングはディフューザの役目もして，ポンプ効率は
やや高くなる ．
