# 3・3・1 特徴

> **章**: 第3章 ターボポンプ
> **節**: 3・3 斜流ポンプ

斜流ポンプ(diagonal flow pump)の羽根車のメリディアン面上の流路は,軸心に対して傾斜しており,この傾斜は比速度が増大するにつれて小さくなり,軸流形に近づく.性能は遠心ポンプと軸流ポンプの中間の性能を有し,軸動力は全流量にわたりほぼ一定で締切運転が可能であり,遠心ポンプに比べて締切揚程比は大きく,軸流ポンプに比べて効率の良い流量範囲が広い.遠心ポンプとするより高速度で運転できるため小型化され,軸流ポンプよりもキャビテーション性能や全域運転可能の点が優れているため,遠心ポンプと軸流ポンプの中間の比速度の広い範囲で使用されている.横軸と立軸形式が多く,羽根車出口側に案内羽根を設ける場合と,渦巻ケーシングを設ける場合とがある.揚程は横軸形式では3~10m程度,立軸形式ではおよそ5~50mの範囲であり,高揚程では多段を用いる.大きさは,小口径から3000mmを超えるものまである.上下水道用,農業用,雨水排水用,発電所冷却水用,その他一般産業用ポンプとして広い用途に用いられている.
