# 3・3・2 設計

> **章**: 第3章 ターボポンプ
> **節**: 3・3 斜流ポンプ

斜流羽根車(diagonal flow impeller)の設計法としてまだ確立されたものはないが、部分羽根車に分けて、遠心式と同様に二次元的に取扱い、出口角の決定にすべり係数を用いる方法と、羽根車内の流れをメリディアン面内の流れと羽根間の流れに分けて準三次元的に取扱い、翼形の選定には直線翼列の資料に補正を加えたものを用いる準三次元的方法とがある。一般には、最初経験値を用いて二次元的な取扱いにより羽根車の形状を与えた後、羽根車内の流れを準三次元的に解析し、良好な流れの状態が得られるまで羽根形状を修正する。さらに水力損失を分析して性能予測を行い、与えられた仕様を満足しない場合は、再び羽根形状に修正を加えて計算を繰返す方法が用いられている。の適用範囲が広いので回転速度は有効NPSH,と性能曲線の変化の傾向、経済性などを考慮して定める。最近では、コストダウンなどの要求により、その対策として与えられた仕様を満足するポンプの高速化、すなわち高比速度化を行い、比速度が約1600(m, mmin, min)の高比速度斜流ポンプが開発されている。a. 羽根車(1) 羽根車メリディアン形状を定めるには、経験的な手法を用いる。遠心ポンプと同様、ステパノフの設計線図を用いるのが一般的であるが、最近では、出口角や羽根数を考慮した新しい設計線図も提案されている。ここでは、ステパノフの設計線図を用いた方法を説明する。また、の値は羽根入口の速度三角形を求めた時点で、キャビテーション(cavitation)性能の面からも検討を加える(3・2・4・b項参照)。これらの主要諸元に羽根チップ側流線の傾斜角δを加味して、羽根車のメリディアン面の形状を定める(図II-3・38)。この分割曲線を第一次のメリディアン流線とする。(3) 羽根入口角βおよび出口角βを個々の流線について次式より求める。このほかが比較的小さい場合の出口角の決定には、βの代わりに遠心羽根車のすべり係数(第I部4・3・1項参照)の値を用いる場合もある。(4) 誤差三角形による展開面上[図II-3・39(c)]、または流線の接線を母線とする円すい面上(半径差による羽根角度の補正が必要)で羽根形状を定めた後、正面図[図II-3・39(b)]に移し換える作図を各流線について行い、羽根幅方向に積み重ねて三次元にわん曲した羽根車とする。(5) 羽根車内の流れの数値解析を行い、より正確に性能を予測する方法が確立されている。その中で実用的な理論として広く使用されているのは、文献(21)(22)による準三次元的な方法(a) 側面図(b) 正面図(c) 展開図(m-m回転流面)図-3・39誤差三角形による斜流羽根の作図2―9 2流体機械で、図II-3・40はこの計算法による羽根圧力面負圧面の圧力分布の計算値と、実測値との比較を示す一例である。この羽根面の圧力差から、羽根に作用するトルクを求め、理論ヘッドを計算により求めることができる。また、縦断面形状までは従来どおり設計して羽根負荷分布を仮定し、逆解法により羽根形状を決定する計算手法も開発されている。b. 案内羽根(guide vane)羽根車出口側に案内羽根を設ける場合は、図II-3・38に示すように縦断面の最大ふくらみの位置を羽根車出口に近づけ、以後案内羽根出口端までの間は、曲がりが少ない直線的な流路となるように滑らかな曲線でつなぎ、流路断面積の変化は、最大ふくらみの位置までは羽根車出口面積と等しく、以後出口断面まで漸次拡大するように流路幅(m)および(m)を定める。λは斜流案内羽根流路の水力損失(hydraulic loss)の最小値が、入射角が零度より小さい負の領域-20°~0°付近の範囲にあることを考慮して決める。また、λはαがα94°~100°の範囲になるように定める。
