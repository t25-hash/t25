# 3・3・2 設計

> **章**: 第3章 ターボポンプ
> **節**: 3・3 斜流ポンプ

斜流羽根車(diagonal flow impeller)の設計法としてまだ確立されたものはないが、部分羽根車に分けて、遠心式と同様に二次元的に取扱い、出口角の決定にすべり係数を用いる方法と、羽根車内の流れをメリディアン面内の流れと羽根間の流れに分けて準三次元的に取扱い、翼形の選定には直線翼列の資料に補正を加えたものを用いる準三次元的方法とがある。一般には、最初経験値を用いて二次元的な取扱いにより羽根車の形状を与えた後、羽根車内の流れを準三次元的に解析し、良好な流れの状態が得られるまで羽根形状を修正する。さらに水力損失を分析して性能予測を行い、与えられた仕様を満足しない場合は、再び羽根形状に修正を加えて計算を繰返す方法が用いられている。比速度(specific speed)욦がおよそ600~1300(m, mmin,min)の範囲になるように、全揚程(m)と流量(mmin)から回転速度(min)を定める。욦の適用範囲が広いので回転速度は有効NPSH,욦と性能曲線の変化の傾向、経済性などを考慮して定める。最近では、コストダウンなどの要求により、その対策として与えられた仕様を満足するポンプの高速化、すなわち高比速度化を行い、比速度욦が約1600(m, mmin,
