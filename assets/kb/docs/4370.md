# 3・3・3 性能

> **章**: 第3章 ターボポンプ
> **節**: 3・3 斜流ポンプ

一定速度で運転した場合の斜流ポンプの性能曲線の一例を図II-(maximum efficiency point)流量の50~60%付近に変曲点があり,締切揚程は最高効率点の1.6~2.3倍程度になる.軸動力曲線は流量の広い範囲でほとんど変わらないが,締切軸動力(shutoffpower)は締切揚程の大小に応じて,最高効率点の0.8~1.4倍程度に変化する.斜流ポンプはこのような特性を有するため,最高効率点以外の広い流量範囲にわたって安定した運転を行うことができる.図-3・44 斜流ポンプのキャビテーション性能曲線図-3・45 立軸斜流ポンプ図-3・46 多段立軸斜流ポンプ図-3・47 軸流ポンプの構造図2―9 3第II部水力機械図II-3・43に可動羽根斜流ポンプ(movable vane type diagonalflow pump)の性能曲線の一例を示す.羽根取付角をさまざまに変えると羽根角と同時に出入口径も変化するため,可動羽根軸流ポンプでは羽根取付角を変えても最高効率点の全揚程はほとんど変わらない.それに比べて,軸動力が変化する割合は大きくなり,固定羽根斜流ポンプを弁開度制御で流量調節する場合に比べて,運転軸動力を大幅に節減させることができる.斜流ポンプのキャビテーション性能は,軸流ポンプと類似した傾向をもっている.吸込圧力を変えて運転した場合の性能曲線の一例を図II-3・44に示す.トーマのキャビテーション係数(Thoma’scavitation factor)σが小さくなると,揚程,効率は低下するが,低下の度合いは遠心ポンプにおけるほど急激ではない.揚程または効率低下開始点のσの値σ욂욥は,過大流量域では一般に急激に増加するが,比速度が大きくなるにつれて低流量域においても増大する傾向を示す.羽根先端の形状は,キャビテーションの初生点におけるσの値σ욡およびσ욂욥の値の変化に著しく影響を及ぼす.また,流量とσ욡の変化を示す曲線から求めた無衝撃流入角は羽根入口角とは一致せず,この差は1枚の羽根単位長さ当たりの比循環量により変化する.
