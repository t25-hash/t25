# 3・3・4 実例

> **章**: 第3章 ターボポンプ
> **節**: 3・42 に示す。揚程曲線は右下がりの安定した曲線で、最高効率点

横軸斜流ポンプの場合は、図II-3・3に示したように案内羽根と一体のケーシング、および吸込ケーシングでポンプケーシング(水平二つ割り)を形成し、案内羽根ハブ内部の水中軸受(バビットメタルまたは青銅製でグリース潤滑の場合が多い)と、吸込ケーシング外部の外軸受にて主軸を支持する構造となっている。立軸斜流ポンプ(図II-3・45は単段、図II-3・46は多段の例)の場合には、中間に数個の水中軸受(硬質ゴム軸受で水潤滑の場合が多い、またはセラミック軸受)と中間軸継手を設け、上部は軸継手で立軸電動機に直結される場合や、かさ歯車減速装置を介して原動機で駆動される場合がある。さらに、ポンプ機場の省スペースを目的として、かさ歯車減速装置を吐出し、エルボ外側に組込んだ歯車減速機搭載形立軸ポンプが開発されている。案内羽根の代わりに渦巻ケーシングを用いた渦巻斜流ポンプ(volute type diagonal flow pump)の例は図II-3・4に示してある。可動羽根操作機構(構造例はポンプ図集参照)には油圧式と機械式があり、大型になると油圧式が使用される。
