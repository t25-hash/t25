# 3・4・2 設計

> **章**: 第3章 ターボポンプ
> **節**: 3・4 軸流ポンプ

その際、必要NPSHが過大にならないように、吸込比速度(suction specific speed)が最高効率点で約1200(m, mmin, min)を超えないように回転速度を選ぶ(図II-1・39参照)。ポンプの口径は普通一般のように管内流速により決定され、羽根車外形とポンプ口径とは必ずしも一致しない。b. 羽根車の羽根形状羽根車を軸と同心のいくつかの半径位置の円筒断面で設計するとき、円筒を平面に展開すると二次元減速翼列(two-dimensional pump cascade)が得られる。各羽根には通常翼形(blade profile)が適用される。これは遠心や斜流羽根車で用いられる等厚の板羽根では損失が大きく、かつキャビテーション性能が悪いからである。この翼形の設計には、翼や翼列のデータや理論が用いられる。揚液に与えられるべきエネルギーと羽根断面が受ける揚力のなす仕事が等しいとおき、既知の翼形性能からこの揚力が得られる翼形および迎え角αを選定することにより、羽根断面形状を得ることができる。は羽根先端部からハブ部に向かうにつれて、25~50%大きくするのがふつうである。ハブ部でのが上記の値を超える場合は、ハブ比を大きくするか、あるいは羽根車外径を大きくしなければならない。なお、既存の翼性能データが得られない場合は、翼素理論の計算により所定の揚力が得られる迎え角αを求め、羽根角度θを設定する。CFD による三次元流れ解析を、羽根車単独やc項で後述する案内羽根と組合わせた全体について行い、設計したポンプの性能を予測し、設計仕様を満足するように形状を修正する設計が普及しつつある。設計では、半径方向の理論揚程の分布が一定となる自由渦形が多用されるが、羽根車をより小型化するには、周速度の大きな羽根先端側で揚程を大きく設定できる強制渦形なども用いられる。ここに、ηは水力効率である。c. 案内羽根案内羽根は、羽根車出口で生じた流れの周方向(旋回)成分を、減速させて圧力回復させる静止流路である。羽根車と同様に円筒状断面を展開して、二次元減速翼列として式(II-3・32)の関係を用いて設計する。翼形、翼枚数、迎え角αを決定し、羽根角θを設定する。
