# 3・4・3 性能

> **章**: 第3章 ターボポンプ
> **節**: 3・4 軸流ポンプ

a. 性能曲線(performance curve)軸流ポンプで,種々の羽根角度における性能曲線を図II-3・48に示す.羽根角が17.5°や20°の場合,吐出し量が最高効率点の70%付近には,揚程曲線に折れ曲がった右上がりの部分(不安定部)が存在している.このように通常の軸流ポンプでは低流量域に不安定部が生じるので,それ以上の流量域が運転域とされる.締切点では,全揚程は最高効率点の2~3倍,軸動力は1.6~2.5倍にもなるのがふつうであり,揚程曲線の安定性が大きいほど倍数は高くなる傾向にある.したがって,軸流ポンプは揚程変化の大きい場合に適しているが,締切軸動力が大きいことが運用上の欠点である.このため,吐出し弁を締切って始動するには,吐出し側にバイパスを設けて吸込側に還流させたり,回転速度を順次上昇させつつ吐出し弁を開いていく方法(回転数制御),あるいは羽根を可動羽根とする方法(可動翼羽根車)等が用いられる.b. キャビテーション性能図II-3・49は羽根車径350mmの軸流ポンプの回転速度,および吐出し量を一定に保ちながら吸込圧力を種々変えて運転したときの全揚程,軸動力,および効率の変化を示す.軸流ポンプではキャビテーションの発達により,渦巻ポンプのように全揚程曲線が急に低下するようなことはなく,最高効率(φ≒0.34)付近ではキャビテーションの発達とともに,全揚程および軸動力はいったん増加し,そののち徐々に低下する.これは,羽根背面のわずかなキャビテーションの発生が羽根の反りを増した効果を表すため,および軸流ポンプでは渦巻ポンプに比べて羽根ピッチ,および半径方向の広がりが大きく,羽根面にキャビテーションが起きても羽根流路全体がただちにその範囲に入ることはなく,影響を受けない流路部分が残っているためである.一方,効率は全揚程および軸動力がいったん上昇を示す点付近か図-3・48 軸流ポンプの性能(不安定性能あり)図-3・49キャビテーション性能[λ:式(II-3・10)参照]ら低下し始める.このため,揚程低下開始点または3%揚程低下点では,すでにかなり効率が低下することになる.吸込圧力の低下に伴い,キャビテーションが発達して騒音が生じるが,軸動力が最大値を示す点付近で最大となり,軸動力の低下とともに若干減少する.軸流ポンプでは,羽根枚数なども,必要NPSH(requiredNPSH),욦욋を決定するうえで大きな影響をもつ.次式は욦욋を決めるための一つの目安となる.욦욋=0.7+욇2(II-3・33)c. 可動羽根(adjustable vane)広い流量範囲で高効率の安定な運転が求められる場合は,可動羽根軸流ポンプが用いられる.可動羽根とは,ハブに植込まれた羽根が羽根ステム軸のまわりに回動可能な機構がハブ内に設置され,羽根角を任意に変えることが可能で,ポンプ回転速度を変化させた場合の性能と類似した性能を得ることができ,必要な流量で高い効率を得ることができる.通常,低流量域における軸動力低減の省エネルギー運転のために採用される.また,a項で前述したように軸流ポンプは,揚程曲線に低流量域にて右上がり部をもつ不安定部を有するものが多く,安定な運転が困難な場合も,可動羽根軸流ポンプでは安定運転が可能となる.
