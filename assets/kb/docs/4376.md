# 3・5・1 ポンプ運転点(pump operating point)

> **章**: 第3章 ターボポンプ
> **節**: 3・5 ポンプ運転とポンプ設備

ポンプ運転点は、吐出し量を横軸、ヘッドを縦軸にとった座標の上で、ポンプの揚程曲線と管路の抵抗曲線(pipeline resistancecurve)の交点として求められる。管路の抵抗曲線は、吸込水槽水面から吐出し槽水面までの高低差、すなわち実揚程と送水管系の損失ヘッドの和として求められる。図II-3・52はこの関係を示したものである。また、大小さまざまなポンプが複雑に管路で結ばれている場合も全く同様であり、この場合のポンプ群の運転点は、各ポンプの揚程曲線を合成した合成揚程曲線と、全体管路の合成抵抗曲線との交点として求められる。合成揚程曲線の求め方の例を表II-3・4に、また合成抵抗曲線の求め方の例を表II-3・5に示す。
