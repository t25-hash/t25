# 3・5・2 運転制御

> **章**: 第3章 ターボポンプ
> **節**: 3・5 ポンプ運転とポンプ設備

ポンプを運転制御するのは,次に示す目的のためである.(1)ポンプの運転点を適正な範囲に維持する.(2)必要な吐出し量に制御する.(3)必要な圧力を保つ.(4)吸込水槽または吐出し水槽の水位を,必要な範囲に保つ.この目的の達成のために,さまざまな方式で制御される.実際の適用では二つ以上の制御方式が採用されることが多い.a. 弁制御(valve control)小型,中型ポンプの一般的な吐出し量制御方法である.図II-3・53に示すように,弁で絞ったヘッド욋だけ管路抵抗が増える.運転効率は욋だけ悪くなるが,軸動力は吐出し量-軸動力曲線に沿って変化するだけで,弁抵抗によって増すことはない.図II-3・54は弁を含めた運転効率を,ポンプ効率から求める方法を示す.ポンプ特性욀より弁抵抗욋を引いた等価なポンプ욁を考え,運転点bから,ポンプの運転点cを求める.弁を含めたη′は′ηとなる.この弁制御の長所は操作が簡便で,設備費が安いということである.短所は動力費が不経済であり,弁の振動,騒音が大きくなることである.特に弁の下流側圧力が低いと,キャビテーションのおそれがある.なお,全揚程に比して管路損失が小さい場合には適している.b. ポンプの運転点回転速度制御の長所は動力費が節約できることや,部分流量でポンプに無理のない運転となることである.短所は設備費がやや高くなること,関係機器のスペースが少し増すことである.c. 羽根取付角(翼角)制御(vane angle control)大型の軸流ポンプや斜流ポンプでは,羽根の取付角(setting angle)を可変とし,ポンプの揚程曲線を変える方法がとられており,これを可動羽根ポンプと呼んでいる.図II-3・56は揚程の変化に応じて軸流ポンプの翼角度を変え,軸動力一定のポンプ運転が可能になる.固定翼の軸流ポンプは小流量での運転が困難であるが,翼角制御を採用することによって,小流量でも効率良く運転することができる.このように広い流量範囲にわたって,高い効率で使用できる.短所は翼角操作機構が増し,駆動部の設計に制約が生じることである.d. 台数制御(multiple pump control)図II-3・57に同一性能の5台のポンプを並列に接続した場合の,ポンプ群の合成揚程曲線を示す.運転点における総吐出し流量はとなる.吐出し流量を減らすことは1台ごとにしかできず,連続的に変えることはできない.それを補うため,この方法をポンプの回転速度制御や弁制御と組合わせて,広い吐出し流量範囲を連続的に制御できる方法がとられることが多い.e.
