# 3・1・1 生活工学のコンセプト

> **章**: 第3章 生活工学
> **節**: 3・1 生活工学のコンセプトと歴史

生活工学（life engineering）は，われわれが生活の中で遭遇す解決することにより，生活の安全，快適，自己実現などの質的な向る．
具体的な研究分野としては，12～14章に示されるような健康ェロンテクノロジー，製品，環境におけるバリアフリーとユニバー学，社会システム工学，安全工学，住居学，衣服に関わる繊維工も，広義の生活工学に含まれる．
これらのことからわかるように，この分野は医学，生理学，心理として，物理学，化学，生物学等の基礎科学と機械工学，電子工合して構成される学際的な研究領域である．
