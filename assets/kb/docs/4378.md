# 3・1・2 生活工学と福祉工学の相違点

> **章**: 第3章 生活工学
> **節**: 3・1 生活工学のコンセプトと歴史

生活工学は2章の福祉工学（welfare engineering）と重複する
ところが多い研究領域である．しかし，両者には研究対象の重点に
違いがある．福祉工学では，障害をもつ人や機能低下の生じた高齢
者を対象とし，生活に支障をもたらす心的あるいは身体的な問題点
を解決し，障害をもたない生活者の生活機能に近づけることを主要
な目的とすることが多い．他方，生活工学においては，障害の有無
にかかわらず，生活の快適性の維持と向上を目指す工学的な取組み
をおもな対象とする．別な表現をすれば，福祉工学は人間生活の基
盤の確保を目指すのに対し，生活工学では，生命の安全，安心のみ
ではなく，生活における快適性，個性化，感性などの生活を豊かに
する付加的価値にも関わるといえる．ただし，福祉工学において
も，障害者を対象としたスポーツ用具などの開発に見られるよう
に，生活における社会参加促進や生きがい，趣味への積極的な取組
みが盛んとなっており，福祉のコンセプトが全生活範囲に拡張しつ
つあるため，生活工学と福祉工学の研究領域の境界線は不明りょう
になりつつある．
