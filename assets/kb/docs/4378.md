# 3・5・3 吸込水槽(suction sump)

> **章**: 第3章 ターボポンプ
> **節**: 3・5 ポンプ運転とポンプ設備

a. 吸込水槽形状がポンプへ及ぼす影響吸込水槽の設計に当たり配慮すべきことは、水の流れをスムーズにして偏りやよどみをなくすことである。つまり、図II-3・63に示すように空気吸込渦(air-entraining vortex)や水中渦(submerged vortex)、旋回流が発生しないことである。これらを連続的に吸込むようになると、性能低下や振動、騒音の原因となる。旋回流は吸込水槽の偏流、渦、羽根車の回転により発生し、羽根車への均等な流入を乱し、ポンプの吐出し量、揚程、軸動力に直接影響する。ポンプと同一回転方向では、揚程、軸動力とも減少し、逆方向では増加する。b. 吸込水槽の代表寸法吸込水槽設計では、各種代表寸法で基準値を決めている。(1)
