# 3・5・4 低騒音化技術(pump noise reduction technology)

> **章**: 第3章 ターボポンプ
> **節**: 3・7 に目安を示す。底面間げきが大きい場合は水中渦はなくな

ポンプから発生する騒音の要因は、機械的原因と流体的原因がある。機械的原因としては回転部の不釣合いによる振動、軸受の振動がケーシングを振動させて騒音を発生する。流体的原因には羽根車による騒音、キャビテーションによる騒音、流路形状による流水音、渦、サージング、水撃による騒音がある。ポンプ設備の騒音は、ポンプ単体の騒音とポンプを含む送水配管からの騒音に分類される。これらの騒音のうち最も影響が大きいのは、圧力脈動である。a.
