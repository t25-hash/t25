# 3・5・5 ポンプの据付けと配管

> **章**: 第3章 ターボポンプ
> **節**: 3・7 に目安を示す.底面間げきが大きい場合は水中渦はなくな

a. 基礎の種類基礎には独立基礎とスラブ上の基礎がある.. 独立基礎独立基礎は直接基礎と,基礎地盤に十分な支持力がない場合に採用する杭基礎がある.いずれも機器のみを,地盤で直接支持するものである.基礎の重量の目安として,ポンプ・電動機の場合,機械総重量の3~5倍,ディーゼル機関の場合4~6倍が一般に推奨される.ディーゼル機関の基礎の大きさは,経験上おおむね次のように決定する.基礎の幅>3×機関台床の幅基礎の長さ>2×機関の全長基礎の厚さ>5×シリンダ径. スラブ上の基礎スラブ上の基礎は,地盤上に建造された土木建築構造物のスラブ上に設ける基礎で,ポンプ設備の基礎は図-3・71 基礎に掛かる荷重大部分がこの方式である.機器をスラブ上に設置する場合には,機器重量や水力スラストを土木・建築構造物で支持する必要がある.ディーゼル機関等,加振力が大きい機器の振動対策,スラブと機器との締結方法等に留意することが重要である.b. 考慮すべき機器重量機器の重量と運転時の運転荷重がある.. 自重機器の製品重量に潤滑油の重量等を加算した運転時の重量が鉛直下方に作用するが,ポンプにおいては,吸込水位よりも上の部分のポンプケーシング内の水の重量を加える必要がある.スラブ上の基礎の場合には,機器の基礎コンクリートの重量を加算する必要がある.. 伝達トルク電動機やディーゼル機関が動力を伝達する場合には,トルクが基礎に作用する.. ポンプの軸推力ポンプを運転すると,羽根車の吸込側と吐出し側の間に圧力差が発生する.ほとんどのポンプ形式ではポンプ自体で反力を支持しているが,二床式以上の立軸ポンプでは,スラスト軸受の位置いかんで軸推力を基礎荷重として考慮する必要がある.. 圧力圧力によって生じる力を基礎で支持する必要があるのは,次のような場合である.(1)ポンプの吐出し圧力の反力,(2)控えボルトがない伸縮継手や可撓管の両側,(3)圧力水槽または押込み圧がある水槽へ接続する配管が貫通する壁,(4)立軸斜流ポンプの据付け床よりも,吸込水位のほうが高い場合のポンプの基礎ボルト部分.. 水の慣性力流れ方向が変化する場合には,図II-3・71に示す力が作用する.. フラップ弁や逆止め弁閉鎖時の衝撃力この力はごく短時間作用するものである.水撃作用によって短時間作用する力は,水撃対策の方法により大きく変わるので注意が必要.. 風荷重・地震荷重ほか建築基準法にこれらの値が決められている.c. 基礎荷重例立軸斜流ポンプの場合の基礎荷重の実例を,図II-3・72に示す.二床式でも,ポンプの軸スラストを支持するスラスト軸受をポンプ自身に設ける場合には,原動機床にスラスト荷重は作用しない.逆止め弁をはじめとして,ほとんどの弁では重量以外の荷重を脚で受けることができないので,フランジで接合された配管系全体で流れ方向の水平力を支持する必要がある.d. 吸込管吸込管における偏流や圧力損失は,ポンプ性能や振動・異常音の原因となるので,設計・施工に当たっては,次の事項に十分留意する必要がある.(1) 配管の損失ヘッドを小さくする.管路長さを短く,配管口径を大きく(流速を小さく)する.(2) 空気だまりの生じない配管レイアウトとする.吸込配管に空気だまりが生じると流れを妨げ,揚程低下,振動の原因となる.対策:1吸込管路が長い場合には,ポンプに向かって1/50~1/200の上りこう配とする(図II-3・73).2管径の変化する部分は,偏心片落管を使用する(図II-3・74).(3)
