# 3・1・5 生活工学への社会的な期待

> **章**: 第3章 生活工学
> **節**: 3・1 生活工学のコンセプトと歴史

生活工学の研究領域が1990年代以降重視され始めた理由の一つ
として，それ以前の高度経済成長時代に対する反動を挙げなければ
ならない．高度成長期においては，産業界はひたすら新機能の開発
と低コストを目標として使い捨ての生活財の生産を進めることが多
どの環境インフラストラクチャの悪化に代表される地球環境問題が
ていた．時を同じくして，バリアフリーやノーマライゼーションな
どの福祉的な取組みが欧米の先進諸国を中心として広がりを見せつ
つ，人口の高齢化による切実な問題が顕在化し始めた．このような
生活工学の領域は，すでに述べたように人間生活が関わる膨大な
目標が定められている ．これらの中には12章で取り上げる健康
