# 4・1・1 形式、特徴、用途

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・1 往復ポンプ

往復ポンプ(reciprocating pump)は容積形ポンプに属し、ピストンまたはプランジャの往復動によって揚液の吸込・吐出し作用を行うもので、駆動方式により蒸気直動式、クランク式、ダイアフラム式に、また作動形式により単動、複動に分類される(図II-4・1)。図II-4・2は電動クランク式単動3連プランジャポンプで、最高圧力は約20~60MPa、歯車減速機を内蔵している。図II-4・3は蒸気直動式複動2連ピストンポンプで、が蒸気ピストン、がポンプピストンである。ダイアフラム式(図II-4・4)は、補助ピストンまたはプランジャによって膜に往復動を与える構造で、漏れや異物の混入の許されない薬液注入ポンプなどに使用される。往復ポンプは構造上、最も高圧作動に適するポンプで、数十MPaに達する高い吐出し圧を有するものも多いが、機構上、高速回転が困難で、このため同一容量のターボポンプに比してかなり大型となり、振動も大きいのでより強固な基礎が必要である。往復ポンプは一般送水用のほか、石油精製、石油化学工業、合成繊維工業、合成樹脂工業等の高圧を必要とする用途に多く用いられている。また、最近はアンロード時の軸動力が小さいことが着目され、鉄鋼業でのデスケーリングポンプとして再認識されている。なお、油圧用については第IV部3章を参照のこと。
