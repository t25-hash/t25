# 3・2・3 健康モニタリング技術

> **章**: 第3章 生活工学
> **節**: 3・2 健康工学

わが国では，2001年から健康寿命のさらなる延伸とそれによる
3･2･2項で，二つのモニタリングシステムに言及したが，今後，
健康工学の研究を発展させていくために必要不可欠な基盤技術が，
生体モニタリング技術と，時系列的に得られたデータから心身の不
具合をいち早く検出する解析技術である．健康維持を目的とする日
常生活のモニタリング技術においては，特に実現しなければならな
い計測のための重要な四大要素がある．(１)非侵襲，(２)非拘束，
(３)無意識，(４)継続性である．非侵襲は言うまでもなく，測定に
より痛み，出血等の身体的な侵害を被らないことである．非拘束と
は，測定のために身につけるセンサや記録・解析システムにより身
体的活動の負荷が増えないことである．無意識とは，被測定者が測
定されていることを意識することなく測定できることである．そし
て，継続性とは長期間にわたり，容易にモニタリングが続けられる
ことである．近ごろ，マイクロエレクトロニクス等の発展により，
ウェアラブルセンサとしての超小型軽量のシステムが増加し，生活
モニタリングが容易になりつつある（12･1･2項参照）．また，無意
識なモニタリング技術については，12･1･3項に詳細があるように，
排便，排尿，入浴，睡眠などの生活に欠くことができない行動を利
用して，日常生活行動の中で無理なく計測する技術が研究されてい
る．今後，さらにナノテクノロジー等の微小化技術の進化が予想さ
れ，錠剤サイズのナノカメラや，痛みや出血を伴わないナノ注射針
等の開発が進めば，健康に関わる臓器内の生理的データを含めて，
無意識かつ長期にわたり安定して抽出できるセンサの出現が期待で
きる．
最近の新たな生体モニタリング技術の動向としては，バイオテク
ノロジーの急速な進化に伴うバイオセンサの研究が盛んであること
が挙げられる（6･2･5項参照）．例えば，血糖値センサとしてグル
コースセンサの使用が検討されている．マイクロ化されたグルコー
スセンサは，従来の電流検知方式や吸光度方式の血糖値センサのよ
うに，測定のたびに採血針で皮膚を突き刺し，侵襲的に出血させる
ことなく，数箇月の長期にわたり非侵襲的な血糖値モニタリングが
可能とされている（12･1･2項参照）．
