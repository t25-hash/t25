# 4・2・1 ロータリ形ポンプ(rotary pump)

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・2 回転容積形ポンプ

ロータの回転に伴い、ポンプ室がその周辺を半径方向に移動するポンプである。ギヤがかみあい、一方のロータが他のロータに直接動力伝達を行うギヤポンプと、動力伝達用のタイミングギヤを別にもち、ロータが同期回転するロブロータリポンプ、および回転するロータの内部に可動部品をもち、ポンプ室を仕切るベーンポンプの基本的な三つの構造である。a. ギヤポンプ(gear pump、歯車ポンプ)構造が簡単であるが、ギヤの空間がポンプ室となるために、1回転当たりの吐出し流量が比較的小さい。外接ギヤポンプ、内接ギヤポンプがあり、後者には内外の歯数の差が1であるトロコイドポンプと、差が複数であり月の輪(クレセント)構造をもつものとがある。油圧発生用ポンプとしても用いられる(第V部2・2節「油圧ポンプ・モータ」参照)。b. ロブロータリポンプ(lobe rotary pump)外部に設けたタイミングギヤ駆動のために接液する二つのロータは動力伝達する必要がなく、大きなポンプ室空間をとれるため、食品などの高粘度液の移送に多く用いられる(図II-4・12)。c. ベーンポンプ(vane pump)ポンプ室中に偏心配置されたロータの一部にしゅう動する板(ベーン)をもち、ポンプ室内壁とのシールを行う。しゅう動部品が交換可能で、しかもベーンがスライドするためにシール性が良く、真空用途にも使われる。ロータとベーンが一体になり、弾性体でできているフレキシブルベーンも吸引用途に用いられる。ロータリ形ポンプは、ポンプ室が多段ポンプのように連構造をしているが、すべてのポンプ室が側板で共通にシールされるため、原則として大きな吐出し圧力をもちにくいが、移送液の粘度が高い場合には、差圧が大きくても使用できる。図-4・12 ロブロータリポンプ
