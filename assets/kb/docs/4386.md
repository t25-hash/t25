# 3・2・5 睡眠に関わる研究：睡眠工学

> **章**: 第3章 生活工学
> **節**: 3・2 健康工学

および寝具内の温熱環境，寝室の照明，音，におい環境に関する研防止のためのベッドや車いすの研究が盛んである（9･9･2項参照）．
響する生体のもつ日内周期を光刺激により積極的に統制して，睡眠るために重要な要素である．
通常，人は睡眠に1日の生活時間の約要性は高い．
従来は眠ってしまえば後は同じなどと，睡眠環境に十より，睡眠の質的な側面の重要性が明らかになってきた．
したがっ要なターゲットとなっている．
夜型社会，興奮やストレスの過多，日中における運動不足，時睡眠深度が浅く，目が覚めやすいことが問題となる．
このような睡究を睡眠工学（sleep engineering）と呼び，健康工学の重要な分野睡眠の促進のため，ベッド，枕等の睡眠用具の研究に加え，寝室究が行われている．
また，福祉工学においても褥瘡（じょくそう）そのほか，より直接的に生体に働きかける手法として，睡眠にも影に入りやすくする高照度光療法などがある．
