# 4・2・2 スクリュー形ポンプ(screw pump)

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・2 回転容積形ポンプ

ロータの回転に伴い,回転軸方向にポンプ室が移動するポンプである.ユニバーサルジョイントをもち,回転軸と並行偏心回転をする一軸のスクリューポンプと,タイミングギヤをもつ二軸スクリューポンプ,主軸と二つの縦動軸をもちタイミングギヤが不要な三軸スクリューポンプがある.二軸,三軸スクリューポンプは,ギヤポンプと同様に剛体のロータが直接互いに,または剛体のポンプ室内壁と接触しゅう動するために固形分を含まない高粘度流体の移送に適する(第V部2・2節「油圧ポンプ・モータ」参照).一軸スクリューポンプは,一般にポンプ室内壁(ステータ)が弾性体にて作られるため,水分の少ない脱水ケーキのような個液混相流も移送可能である.スクリューポンプは,ポンプ室が軸方向に独立して重なる多段構造であるため,部屋数が多いと1段当たりの差圧が小さくなるので,長くしてポンプ室を増やし,高圧を得ることができる.図II-4・13のように一軸スクリューポンプは,ロータ中心軸に対して円を偏心させてらせん状に配置(軸直交断面が円形のおねじ)し,偏心量の2倍の半径に内接させて回転させると,円の軌跡(ハイポサイクロイド曲線)が直線になることから,おのおのの断面でロータがステータの中を自転しながら往復運動をすること(a) 一軸スクリューポンプの作動原理(b) 一軸スクリューポンプの構造とシール線図-4・13 一軸スクリューポンプ図-4・14 アルキメデスポンプ図-4・16 再生ポンプ2―111第II部水力機械でポンプ作用をする.ポンプ室はねじれた空間となり,1/2リードごとに隣り合って重なる.スクリュー形ポンプの一種にアルキメデスポンプがあり,傾斜させた円筒の内壁に接する,らせん状の板により仕切られたポンプ室をもつ(図II-4・14).他の面に密閉されていないが,空気との自由境界面によりポンプ室を構成する.傾いてしか使えないが,構造が簡単であり大量の土砂を含む水の揚水が可能である.
