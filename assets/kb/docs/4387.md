# 4・3・1 渦流ポンプ(再生ポンプ,渦ポンプ)

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・3 その他のポンプ

遠心ポンプと同様,羽根車の回転に伴う遠心作用によって羽根から流出した液は,ケーシング内の液と運動量交換を行った後再び羽根に流入し,繰返しエネルギーを与えられる.これには,液の大部分が羽根車を通過する形式[図II-4・15(a)(b)]と,流れの一部のみ羽根車を通過する形式[図II-4・15(c)]とがあるが,いずれも種々の名称が付されている.前者は商品名のウエスコポンプ,カスケードポンプ,タービンポンプ,摩擦ポンプ,渦ポンプ,渦流ポンプ,再生ポンプ,円周流ポンプなど多くの名称があり,後者も渦ポンプ,ボルテックスポンプの名称が使われているが,ここでは混乱を避けるため,前者を再生ポンプ(regenerative pump),後者を渦ポンプ(vortex pump)と呼ぶ.a. 再生ポンプ比速度(m, m/min, min)が50~150の範囲に適したポンプで,最高効率点における揚程が同じ羽根車直径をもつ遠心ポンプの約3倍あり,小さい駆動動力で高揚程が要求される家庭用電動井戸ポンプに広く用いられているほか,小型ボイラの給水ポンプや灯油,A 重油,低粘度の化学薬品輸送用にも使用されている.このポンプはすきまの摩耗によって著しく性能が低下するので,固形物を含む流体の輸送には適さない.構造の一例を図II-4・16に示す.羽根車には,図II-4・15(a)に示すように周辺に多数の溝があり,羽根車の回転に伴って溝内の流体は角運動量を得て羽根外周から流出し,流路内から再び羽根根元に流入する.このような作用がそれぞれの溝内で繰返され,流体はらせん状の経路を経て吐出され,小型の割に高い圧力を発生する.吸込通路と吐出し通路との間には隔壁があって,高圧流体の逆流を防いでいる.吸込口に逆止め弁を備えた自吸式のものが多い.(a)(b)(c)図-4・15 渦流ポンプの分類2―112流体機械組立,漏れ防止,および保守・点検の便宜上,図II-4・16のように羽根車を片持構造としたものが多いが,二段ポンプなど高圧用では羽根車の両側の軸受で支持するものもある.再生ポンプは渦巻ポンプに比して,大きな半径方向推力が羽根車に働くので,軸がたわんで羽根車側面がケーシングに接触しないよう注意が必要である.羽根車は一般に砲金製で,高温水あるいは腐食性液体を扱うときは,羽根車や流路壁面などにステンレス鋼を用いる.ケーシングはいずれの場合も鋳鉄製が多い.羽根車の作用に関する理論的取扱いには,回転円板の溝による流れの乱れに基づくレイノルズ応力に根拠をおくものと,溝にある流体が羽根車の回転によって与えられる角運動量変化に基づく特性記述とがある.さらにこれらの理論的背景のもとに,具体的設計に役立つ実験資料のとりまとめや,それらの表式化が行われている.図II-4・17にポンプ性能の一例を示す.図中φ:流量係数(=욧ω),ψ:揚程係数(=욧ω),μ:軸動力係数[=ρ욧ω],욧:羽根車先端半径,:羽根を囲む流路の断面積,ω:羽根車角速度,ρ:流体の密度,:流量,:揚程,:軸動力,η:効率である.ポンプの諸元(図II-4・18)は=2욧=105mm,욡=85mm,=8mm,=117mm,ε=ε=0.04mm,ω=261.5rads(2500min)である.揚程は締切点で最大で,流量が増加するとほぼ直線的に低下する.軸動力も同様で,このため原動機に過負荷のおそれのあるときは安全弁を取付けておくか,圧力スイッチによる自動停止装置が必要である.ポンプ効率の変化は遠心ポンプに似ているが,最高効率はかなり低く,ふつう30~40%である.効率を高くする条件として,図II-4・18の流路形状のものに対して次の範囲が示されている.욇=10~30,溝数=24~72,=1~8,=0.30~0.45,=1.0~1.7,=0.15~0.07.ここに,욇:流路代表寸法[=2+].流路断面形状を図II-4・19のように変えて,羽根出口流れがスムーズに羽根入口に流入する構造とし,かつコアリングを設けると,最高効率は50%近くまで向上する.これに案内羽根を挿入して特性を変える試みもなされている.b.
