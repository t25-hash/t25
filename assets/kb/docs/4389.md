# 4・3・3 特殊ポンプ

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・3 その他のポンプ

4･2･2項までに取上げたポンプのほか，特殊な用途に使用される
a.粘性ポンプ 円筒または円板を箱内で回転させ，これに接
円筒形粘性ポンプは，円筒形の回転子に溝のある接片をばねによ
ねじ形粘性ポンプには，回転軸にねじ溝を切り，これにはめ合わ
らせん溝形粘性ポンプは，固定台にらせん形の溝を切り，その上
に密着して円板を回転する構造で，これを液中に浸せば溝を通って
外から中心に液が送り込まれるので，これを導いて軸受などへの給
油に用いる．対数らせん溝とアルキメデスらせん溝との特性の比較
も報告されている．
b.気泡ポンプ（air lift pump) 図Ⅱ-4･27に示すように揚水
管を水中に入れ，その中に挿入した空気管の下端より圧縮空気を吹
出すことによって，密度の小さい気泡水を作り，管外の水との比重
差を利用して揚水する装置である．空気圧縮機以外には運動する部
分がないため，摩耗や故障のおそれはなく，耐食性材料の使用が容
易であるが，井戸を深く掘る必要があり，また効率が低いことが欠
点である．石油井における自噴採油やガスリフト採油は気泡ポンプ
の原理を利用したものであり，最近は立坑内の土砂や石炭の水力輸
送にも利用され，その基礎研究もなされている．
ポンプ効率ηは揚水効率η （＝揚水の得た動力/気体の等温圧縮
動力）と圧縮機効率η （＝気体の等温圧縮動力/圧縮機の軸動力）

c .水撃ポンプ 低落差の流水中に水撃作用を起こして，その
一部を給水源よりも高所に押上げるポンプを水撃ポンプ（hydrau-
lic ram）という．図Ⅱ-4･28のように水撃を発生させるための導
水管1，ポンプ室2，水を排出して導水管内に流れを起こす排水弁
3，逆流防止吐出し弁4，空気室5，吐出し管6より成る．弁3を開
くと導水管内の水が排出され，長い導水管内を水が流下する．流速
が適当な値に達したとき，弁3は流れによって自動的に急速に閉じ

ポンプ室2の圧力は低下し，弁3が開いて再び水の流下が起こる．

e .ピストンなし押しのけポンプ（pistonless hydraulical drive
pump) 圧縮空気あるいは蒸気を直接液面に作用させて揚水す
るポンプで，各種のものがある．太陽熱駆動水ポンプもその一種
で，構造簡単，原動機不要の低廉な家庭用ポンプとして，また発展
途上国での使用を目的として研究されている．これは図Ⅱ-4･29の
ような構成で，日中の太陽熱による蒸気の発生で，水は上部の逆止
め弁より送り出され，夜間の低温による蒸気の凝縮で下部の逆止め
弁より水を吸い上げる．

d.電磁ポンプ（electromagnetic pump) 電気伝導率の高い
液体金属を電磁力によって輸送するポンプで，液体ナトリウム冷却
高速増殖炉の補助ポンプとして用いられる．これには導電形と誘導
形とがある．前者は長方形ダクト内の導電性流体（液体ナトリ
ウムなど）に磁界を掛け，これに垂直方向に電流を流し，フレミン
グの左手の法則に従う電磁力を発生して流体を送り出す．後者は長
方形流路の上下の積層絶縁鋼板内にコイルを設置して，各コイルに
多相交流を流すことによりダクト軸方向に垂直磁界が移動し，ダク
ト内の導電性流体に生じた誘導電流と磁界とによる電磁力によって
駆動する．円形ダクトの周囲にコイルを設置し，ダクト中心に磁性
材のコアを設けた構造のものもある．
電磁ポンプは流量が容易に連続的に変えられること，可動部分が
なく完全に密封できること，設置位置に制限がないことなどの利点
があるが，効率が低く（45 %以下），形状が大きくなる欠点がある．
