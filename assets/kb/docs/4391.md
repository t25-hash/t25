# 5・1・1 分類

> **章**: 第5章 流体伝動装置
> **節**: 5・1 流体による動力伝達の分類と特徴

流体伝動装置(hydraulic transmission)とは,入力された機械的動力を流体動力に変換し,これを再び機械的動力に戻して出力することによって動力伝達を行う装置のことである.流体伝動装置には,大きく分けてターボ形流体伝動装置(hydrodynamic trans-mission)と油圧伝動装置(hydrostatic transmission)の二つがある.前者はターボ形ポンプとタービンを,後者は容積形ポンプとアクチュエータを組合わせたものをいう.伝達される流体動力は圧力差と流量の積・で表されるので,一定の流体動力を伝達するにはを小さくしてを大きくする方法と,を大きくしてを小さくする方法とがある.伝動装置の構成要素の特性から考えると,前者はターボ形流体伝動装置に,後者は油圧伝動装置にそれぞれ対応している.ターボ形流体伝動装置は,入力軸トルクと出力軸トルクがつねに等しい流体継手(fluid coupling)と,入力軸トルクに対して出力軸トルクが変化する流体トルクコンバータ(hydraulic torque con-verter)とに大別される.両者の構造上の違いは,ステータの有無にある.流体トルクコンバータは単にトルクコンバータ(torqueconverter)と呼ばれることが多い.また,流体継手は,ドイツ(現・ポーランド)のフルカン造船所で初めて実用化されたことにちなんでフルカンと呼ばれることがある.油圧伝動装置に用いられるアクチュエータには油圧シリンダと容積形モータがあり,前者では往復運動,後者では回転運動の出力がそれぞれ得られる.容積形ポンプ・モータには1回転当たりの押しのけ容積が一定の定容積形と,変化する可変容量形とがあり,それぞれの組合せによって特性が異なる.ターボ形流体伝動装置,油圧伝動装置のいずれにおいても,動力変換時に発生する損失を減らすため,伝達動力を分割して一部の動力を機械的に伝達する方式がある.これを動力分割式流体伝動装置(split hydraulic transmissionまたはhydro-mechanical transmis-sion)という.油圧伝動装置の場合,特にこれを油圧-機械式伝動装置(hydro-mechanical transmissionまたはsplit hydrostatictransmission)と呼び,動力分割のないものを純油圧伝動装置と呼ぶこともある.これらの流体伝動装置とは別に,入・出力軸にそれぞれ結合された多数の円板(または円筒)をすきまが十分小さくなるように配置し,すきま内の流体の粘性に基づくせん断力によって動力を伝達する装置がある.これを粘性継手(viscous coupling)と呼ぶ.
