# 5・1・2 特徴

> **章**: 第5章 流体伝動装置
> **節**: 5・1 流体による動力伝達の分類と特徴

ターボ形流体伝動装置は以下の長所を有する。(1) 構造が比較的単純である。(2) 最高効率が比較的高く、また効率が速度比(=出力軸回転速度/入力軸回転速度)の関数で表され、伝達動力の大小によってあまり変化しない。(3) 始動時に原動機に負担をかけない。(4) 負荷トルクの変動に応じて自動的に滑らかに変速する。(5) 軸系のねじり振動や衝撃トルクを緩和する。(6) 伝達動力や回転速度に特定の制限がない。一方、ターボ形流体伝動装置の短所は以下の点である。(1) 出力軸を逆転させることができない。(2) 制動効果が小さい。(3) 速度比を一定値に固定することができない。これに対し、油圧伝動装置には以下の長所がある。(1) 無段階に出力軸回転速度の調節(制御)ができる。(2) 負荷の始動トルクを大きくできる。(3) 出力軸の正逆転の変換が可能で、その特性を等しくすることができる。(4) 管路を用いた遠隔伝動ができる。(5) 一つの原動機から複数の系統に動力を分配することや、逆に複数の原動機から一つの系統に動力を集中することができる。(6) 制動効果が大きい。油圧伝動装置の短所としては以下のものが挙げられる。(1) 構造が複雑で質量が大きくなる。(2) 最高効率があまり高くなく、しかも効率が回転速度や伝達トルクの大小によって変化する。(3) 伝達動力に関係する圧力や回転速度に特定の制限がある。これらの特徴から、ターボ形流体伝動装置は高速低トルク負荷の自動変速用に、油圧伝動装置は低速高トルク負荷の速度制御用に適している。したがって、粘性継手は、自動車用内燃機関の冷却ファンの伝動装置や四輪駆動車の差動装置などのように、高速回転時に滑りを大きくしたい用途に適している。
