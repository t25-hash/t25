# 5・2・1 構造および作用

> **章**: 第5章 流体伝動装置
> **節**: 5・2 流体継手

図II-5・1に示すように、入力軸と出力軸とを一直線上に配置し、入力軸にポンプ羽根車を、出力軸にタービン羽根車を結合して対向させ、内部に作動液体(一般には鉱物油)を満たす。一般にポンプ、タービン両羽根車の断面形状は等しく、数十枚の直線放射状の羽根がシェル(流路外周面)に組付けられているが、断面の中心にコアリング(流路内周面)のないものもある。羽根車は圧延鋼板にプレスとろう付けなどの加工を行って製作されることが多いが、鋳造によりコアリングと一体で作られる場合もある。入力軸が回ると、ポンプ羽根車によって作動液体が回されて出口から流出し、タービン羽根車を通過して出力軸を回す。定常状態では、ポンプの入口から出口までの作動液体の角運動量変化は、タービンの入口から出口までの作動液体の角運動量変化と絶対値が等しく、符号が逆である。角運動量法則から、ポンプが作動液体に与えるトルクを(N・m、一般には入力軸トルクに等しい)、作動液体がタービンに与えるトルクを(N・m、一般には出力軸トルクに等しい)とすると、=である。したがって、動力伝達の損失は、入力軸回転速度に対する出力軸回転速度の減少となって現れる。通常の運転状態では、つねに滑りが存在する。ここで、:羽根車流路外径(m)、:回路部分の形状、作動流体の種類と状態(密度に比例する)、および滑り(または速度比)によって変化する係数を表す。流体継手は一般に滑りが零に近いところで常用される。=1付近では、伝達動力に比べて軸受や軸シールの摩擦、風損などが相対的に大きくなり、速度比の増加とともに伝達効率は急激に減少していくため、伝達効率が最高となる速度比付近が常用運転点となる。したがって、設計点は=0.95~0.98程度に選ぶ。このような範囲では、流体継手の入力(kW)を次のように表すことができる。式の適用範囲は<10%で、は構造が相似であればだいたい一定値となる(図II-5・1の形式で鉱物油使用の場合、≒0.08~0.13)。一定の動力を伝達する場合、回転速度、直径ともに大きくするほど滑りが小さくなり、高効率となる。作動液体としては、潤滑性がよく、粘度が低く、密度が大きく、変質しにくく、凝固点が低く、沸点が高いものがよい。一般的には流体継手用の鉱物油が使用される。滑りによって発生する動力損失は熱に変わり、油温は上昇するが、100°C程度までは十分な実績があり、通常の使用状態では油を外部に取出して冷却する必要は特にない。
