# 5・2・3 補助機構

> **章**: 第5章 流体伝動装置
> **節**: 5・2 流体継手

速度比零(滑り100%)の状態を失速(ストール)状態といい、この状態におけるトルクをストールトルク(stall torque)という。ストールトルクは設計点におけるトルクの10倍以上に達することがある(図II-5・2)。過大なストールトルクは始動時や過負荷時に原動機に負担をかけるので、設計点トルクに対してストールトルクをできるだけ下げることが要求される。a. 一定充てん式流体継手(constant filling fluid coupling)一定量の液を充てんするもので、ストールトルクを下げるために以下のような方法がとられる。. じゃま板失速点(stall point)近くで循環流量が増大するとき、流れに対する大きな抵抗を発生させてストールトルクを下げるよう、回路内に円板状のじゃま板(baffle plate)を設ける(図II-5・8)。回路内に作動液体を充てんした場合にじゃま板の効果は大きい。. 貯油槽(oil container)タービン羽根車と一体の構造で、その背面に設けられた一種の油だめである(図II-5・8)。出力図-5・9特殊回路形状の流体継手図-5・10 可変充てん式流体継手軸の回転速度が低下すると、貯油槽外周部の圧力が回路内の圧力よりも低下し、回路内の作動液体が空気と入れ替わって貯油槽に流れ込み、回路内の作動液体量が減る。その結果、ストールトルクが減少する(図II-5・2)。. 環状弁流れを絞ることによってストールトルクを下げるもので、回路内に環状の弁を人為的に出し入れする。. 特殊回路形状回路断面形状を特殊なものにすることによってストールトルクを下げることがある(図II-5・9)。このような構造で発生する軸スラストの増加に対しては、2個の回路を直列配置(双子形という)することで対応できる。b. 可変充てん式流体継手(variable filling fluid coupling)回路内の作動液体量を自由に調節できる構造のもので、伝達トルク容量を任意に変えられる。この調節によって、入力軸の回転速度を一定に保ちつつ、出力軸の回転を停止状態から全速状態まで滑らかに自由に変えることができる。これを変速用流体継手とも呼ぶ。液量調節の機構はいろいろと考えられるが、図II-5・10はすくい管(scoop tube)使用の例である。すくい管の先端は作動液体の取入れ口となっており、継手とともに回転するドラム内面の油層に差入れて作動液体をすくい取らせる。すくい管の先端を適当な半径位置におけば、その位置で平衡を保つ液量に落ち着く。回路内の液量を調節して使用する場合には、コアリングを連続したものとせず、これを取り除くか、または羽根間で一部切除する。連続コアリングでは、ある液量の範囲で液量と滑りとの関係に段差を生じて速度調節が困難となる。
