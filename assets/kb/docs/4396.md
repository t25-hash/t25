# 5・3・1 構造および作用

> **章**: 第5章 流体伝動装置
> **節**: 5・3 トルクコンバータ

図II-5・11に示すように、流体継手と異なり、回路がポンプ羽根車とタービン羽根車だけでなく、ステータを含んだ3種の流体要素で構成される。一般にポンプ、タービン両羽根車の羽根は複雑な三次元曲面形状をしていることが多い。羽根車はシェル(流路外周面)、羽根とコアリング(流路内周面)から構成され、圧延鋼板にプレス、かしめ、溶接、ろう付けなどの加工を行って製作されることが多いが、アルミニウム鋳造などによりコアリングと一体で製作されることもある。一般的にコアリングは必要とされるが、コアリングをもたない特殊な例もある。ステータは一般に翼形断面をもち、アルミニウム鋳造で製作されることが多い。ポンプ羽根車から流出した作動液体は、タービン羽根車を通り、ステータを経てポンプ羽根車に戻る。ステータにトルクが発生するので、そのトルクの大きさだけ入力軸トルクと出力軸トルクとの間に差を生じる。すなわちトルクコンバータ(torque converter)は、トルクの変換を伴う変速装置である。各羽根車やステータをつねに有効に作動させるために、油は回路内を充満するよう、ある程度の圧力で押込まれている。これには回路内の低圧部でキャビテーションを発生しにくくする効果もある。比例係数,はトルク容量係数(torque capac-ity coefficient)で、回路部分の構造、作動液体の種類と状態(液体の密度に比例する)および速度比によって変化する。相似形に作られたもので作動液体が類似であれば、近似的に,は速度比のみの関数となるが、厳密には表面粗さ、流れのレイノルズ数など(a)(b)(c)⎭――――――――⎬―――――――――⎫3要素1段P:ポンプ、T:タービン、S:ステータ、I:入力軸、II:出力軸、F:一方向クラッチ2―119第II部水力機械によって多少は変化する。図II-5・12に、図II-5・11の構造をもつトルクコンバータの特性曲線の一例を示す。特性曲線の形状は、各羽根車やステータの配置、羽根形状などによって大きく変化する。式(II-5・4)からわかるように、トルクコンバータは流体継手同様、そのトルクはポンプ羽根車外径の五乗に比例し、五乗則と呼ばれる。トルクコンバータは、失速点を含めて広範囲の速度比にわたり入力トルク容量係数の変化が少なく、また出力トルク容量係数は出力一定に近い特性を示す。したがって、負荷トルクに変化があれば、それに順応して原動機に負担がかかることなく出力軸回転速度が自動的に変化する。トルクコンバータの最高効率は90%前後で流体継手の設計点効率に比べれば多少低いが、効率曲線が速度比に対して上方に膨らみをもっているので、流体継手が1に近い速度比で常用運転される用途に適するのに対し、トルクコンバータは広範囲の速度比にわたって運転される用途に適する。したがって、トルクコンバータの設計点というものは1点ではなく、また必ずしも明確でない場合がある。一般にトルクコンバータの特性には次のことが要求される。(1) 入力トルク容量係数と速度比との関係が原動機の負荷特性に適合すること。(2) 高効率で運転できる速度比の範囲が広いこと。(3) 出力軸停止の失速点におけるトルク比(ストールトルク比)が大きいこと。出力要素であるタービンの数を段(stage)で表し、回路を形成する羽根車およびステータの数を要素(element)数で表す。図II-5・13に各種の羽根車配置例を、図II-5・14には1段形と多段形の特性例を比較して示す。トルクコンバータの動力損失は熱に変わり作動液体の温度を上昇させるので、一般に100°C以上でも使用できるトルクコンバータ油(鉱物油)が用いられる。油の粘度が低いほど特性はよくなるが、機械部分の潤滑を行なわなければならないので、これを極端に下げることはできない。油の冷却は、一般に外部に取出して冷却器を通して行われる。
