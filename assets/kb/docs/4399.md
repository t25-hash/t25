# 5・4・1 自動変速機

> **章**: 第5章 流体伝動装置
> **節**: 5・4 歯車装置などと組合わせた流体伝動装置

流体継手を単独で使用しても入力軸トルク以上の出力軸トルクは得られないので,流体継手は歯車装置などと組合わされ,トルク増幅を行う変速装置として自動車などに用いられる.流体継手は滑らかな滑りを許すクラッチとして作動し,歯車装置の変速の自動化を容易にする.また,トルクコンバータもそれ自体ではストールトルク比,常用範囲の効率や速度比の範囲が不足するような場合,トルクコンバータに補助の歯車装置などを組合わせて変速装置とし,自動車などに用いられる.これらの歯車装置には3速から4速の遊星歯車(planetarygear)機構を用いることが多く,その変速段切換えは一般に自動化される.変速段切換えが自動化されたこれらの変速装置を自動変速機(automatic transmission)と呼ぶ.図II-5・32に直結装置付きトルクコンバータ継手と前進3速,後退1速の補助歯車機構を組合わせた自動変速機の構成図を示す.図II-5・32中,C,C,C,Cはそれぞれ変速用摩擦クラッチ(C,Cは摩擦ブレーキの機能CCCC1速offonoffon前進2速offononoff3速ononoffoff後退onoffoffon図-5・32 自動変速機の構成図の例図-5・33 入力分割形図-5・34 出力分割形をもつ),Cは直結用摩擦クラッチを表し,それらの締結(on)と解放(off)によって変速段の切換えと直結の有無の切換えが行われる.これらの締結と解放は一般に油圧によって自動的に行われるが,油圧をきめ細かく制御する,さらに原動機の発生トルクを一時的に低減させるなど変速段の切換え時における衝撃を緩和する手段が講じられている.
