# 1・1・1 分類

> **章**: 第1章 気体機械一般
> **節**: 1・1 気体機械の分類と選定

気体機械(gas machinery)は、気体を作動媒体とする流体機械の総称であり、気体と機械の間のエネルギーの伝達方向により、気体に機械的エネルギーを与えて圧送する被動機と、気体のエネルギーを機械的エネルギーとして取出す原動機に大別される(1・2節参照)。被動機は、圧送する気体の吐出し圧力または圧力上昇により、送風機と圧縮機に分類され、その圧力の範囲は概略表III-1・1のとおりである。送風機は用途により、通風機、循環機(サーキュレータ)、昇圧ファン(ブースタ)、換気扇、扇風機、冷却ファン、排風機など種々の呼称がある。このうち排風機は、吐出し圧力が大気で吸込圧力は大気圧以下である。吸込圧力がおよそ100Pa以下のものは真空ポンプと呼ばれ、その形式と分類については第IV部3章に記す。原動機は、タービンや風車のように動力を取出す動力機械と、空気圧アクチュエータのように機械的作業をする作業機械に分類される。また、気体のエネルギーを動力として吸収し気体を冷却するエキスパンダも原動機に属する。タービンは作動気体の種類により蒸気タービン、ガスタービン、空気タービンなどと呼ばれる。タービンの分類・形式については4章で述べる。空気圧アクチュエータは、作業の機能により、直線運動をする空気圧シリンダ、揺動する空気圧揺動形アクチュエータ、回転運動をする空気圧モータがあり、その分類については第V部で述べる。作動原理により気体機械を分類すると、第I部1章で述べたように、回転する翼(羽根)の作用を用いたターボ形、運動部分と静止部分の間にできる空間の容積変化を利用した容積形のほか、せん断応力を利用した渦流式や噴流式がある。a. ターボ形ターボ形には以下の形式がある(第I部1章参照)。. 遠心式(centrifugal type)気体が回転軸にほぼ垂直な面を旋回しながら通過する羽根車(遠心羽根車、centrifugalimpeller)を有するもので、被動機では、遠心送風機、遠心ファン、遠心ブロワ、遠心圧縮機がある。原動機としてのタービンの場合は内向きに流れるので遠心式とはいわず、ふく流式タービン、ラジアルタービンと呼ばれる。入口(出口)側にインデューサ(エクスデューサ)を付けて軸方向に流入(流出)させ、大流量での流動条件を良くした羽根車を特に混流羽根車(mixed flow impeller)といい、このような羽根車をもつものを混流式ということがある。. 軸流式(axial flow type)気体がほぼ回転軸と同心の円筒面上を旋回しながら通過する軸流羽根車(axial flow impel-ler)を有するターボ機械で、軸流送風機、軸流ファン、軸流ブロワ、軸流圧縮機、軸流タービン、軸流風車がある。. 斜流式(diagonal flow type)気体が回転軸に対して傾斜した円すい面を旋回しながら通過する斜流羽根車(diagonalflow impeller)を有するターボ機械で、斜流送風機、斜流圧縮機、斜流タービンがある。表-1・1 送風機および圧縮機の分類吐出し圧力圧力比ファン約10kPaゲージ未満約1.1未満送風機ブロワ約10kPa~0.1MPa約1.1~2.0圧縮機約0.1MPaゲージ以上約2.0以上2―129. 横流式(cross flow type)内外径比の大きい遠心羽根車と同様の形状で、外周上のある部分から流入し別の部分から流出する横流羽根車(cross flow impeller)をもつ。貫流式とも呼ばれる。横流ファンは非軸対象のケーシングをし、翼高さを大きくとることにより直径と無関係に流量を大きくできる特徴があるが、効率は低い。ダリウス風車やクロスフロー風車は、ケーシングのない横流式ターボ機械である。b. 容積形容積形は、回転式と往復式に大別される(第I部。 回転式(rotary type)ロータとケーシングで気体を閉じこめる空間を形成し、ロータの回転により容積を変化させて気体とのエネルギー授受を行う。圧縮機、真空ポンプ、エキスパンダ、空気圧モータ、空気圧揺動形アクチュエータとして使用される。ロータの形により表III-1・2のように分類される。. 往復式(reciprocating type)シリンダ内を往復運動するピストンの運動により、容積を変化させて気体とのエネルギー授受を行う。圧縮機、真空ポンプ、空気圧シリンダとして使用される。シリンダとピストンの配置により、単筒複動式(図III-1・1)、単筒タンデム式(図III-1・2)、多筒往復式(図III-1・3,III-1・4)などの形式がある。c. せん断応力形気体のせん断応力を媒介として機械と気体のエネルギーの授受を行うもので、乱流せん断応力(乱流混合)によるものと層流粘性によるものがある。. 渦流式外周に溝を設けたロータを回転させ、ケーシングとロータの間の気体の乱流混合によりエネルギーの授受を行うもので、渦流ブロワが使用されている。. 噴流式噴流の混合作用により気体を誘因するエジェクタが、真空ポンプとして使用されている。表-1・2
