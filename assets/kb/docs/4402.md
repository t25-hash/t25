c .スクロールポンプ（scroll vacuum pump) 図Ⅳ-3･10に
示すように，一方のスクロールを固定し，他方を自転させずに一定
の半径で旋回させると，二つのスクロールに囲まれた三日月状の体
積は，外周から回りながら縮小しつつ中心に移動し，中心の排気口
から排気される．連続的に吸入，圧縮，排出の各行程が進行するの
で，駆動トルクの変動が小さく，騒音，振動も小さくすることがで
きる．また隣接する三日月状体積の圧力差が小さく，スクロール側
面は接触式（チップ）シールにより，ほぼ完全にシールされている
ので漏れが小さく，高い圧縮比が得られる という構造上の利点
がある．チップシールのすべり速度は1m/ s以下と小さく，油等
による潤滑なしで長時間の使用にも耐える．また重量/排気速度が，
他の機種に比べ小さいことも利点である．対になっている二つのス
クロールとも回転させる回転式スクロールポンプもあるが，二つの
スクロールの相対的な運動は，一方のスクロールを固定した旋回式
と同じである．しかし，旋回式に必要な回り止めピン（吸入側にあ
り，グリースで潤滑されている）が不要のため，いっそう完全なド
ライ化の可能性がある．スクロール式は構造的に中小規模流量のポ
ンプに適しており，排気速度は20～50m / h程度，大気圧からの
到達圧力は10 Paである．
