# 1・1・2 選定

> **章**: 第1章 気体機械一般
> **節**: 1・4 節参照)。

気体機械の種類と形式は前述のようにさまざまであり、機種の選定に際してはそれぞれの特徴をよく把握し、使用目的に適合したものを選定する必要がある。機種の特徴と選定の詳細については、送風機・圧縮機(2章、3章)、タービン(4章)、風車(5章)、真空ポンプ(第IV部3章)、空気圧機器(第V部1章、4章)に記載されているので、ここでは一般的な記述にとどめる。a. 形式による適用範囲機種には、それぞれ適用範囲がある。送風機・圧縮機では、一般に大容量の場合にはターボ形、高圧力の場合には容積形が適している。ターボ形は、効率が高くメンテナンス性も良いので、高圧力比の用途で多段にするが、制作費が高価になる。図III-1・5に送風機および圧縮機の適用範囲の概略を示す。タービンは、大出力の場合には軸流式タービン、小出力の場合はラジアルタービンまたは斜流式タービンが使用される。高圧力比の多段軸流式では、体積流量の小さい前段を衝動タービン(impulseturbine)、体積流量の大きい後段を反動タービン(reaction tur-bine)とする。原動機として容積形は、空気圧アクチュエータや小容量のエキスパンダ(小動力回収・気体冷却)にしか使用されない。風車は大型の場合、翼のソリディティが小さいプロペラ風車かダリウス風車(Darrieus rotor)が適しており、小型の場合は制作費や騒音の観点から種々の形式が提案されている。真空ポンプおよび空気圧機器は、ほとんどが容積形であり、その詳細はそれぞれ第IV部3章および第V部を参照されたい。b. 使用目的および使用条件による選定大出力で常時運転を必要とする機械の選定には、効率とメンテナンス性が最も重要な要因であり、高性能のターボ機械が使用される。その際、非設計点の効率が落ちないように動・静翼可変ピッチ、回転数制御、抽気などの工夫が行われる。発電プラントで使用される圧縮機、ブロワなどは、すべてこのような工夫が行われている。作動特性も選定の重要な要素である。ブロワや圧縮機の場合、ターボ形は流量が回転数に比例し、圧力は回転数の二乗に比例する。図-1・5 送風機および圧縮機の適用範囲また、一定回転では一般に流量の減少に伴い圧力が増加するが、ある流量まで減少すると急激に圧力が下がり、脈動・振動が起こることがあるので、作動特性と流路特性とのマッチングが重要である。容積形は流量が回転数に比例し、圧力にはほぼ無関係であるので、吐出し圧力によらず必要な流量を圧送したい場合に適する。しかし、吐出し圧力が高すぎると過負荷になり原動機を損傷すること、一定回転の原動機では流量調整ができないことなどの欠点があり、バイパス回路などの工夫を要する。使用環境も機種選定の重要な要件である。例えば粉じんを含むガスや腐食性ガスの輸送には、耐摩耗性や耐腐食性を考慮する必要がある。一般に、軸流ブロワよりも遠心ブロワのほうが摩耗や腐食による性能低下が少ない。摩耗の激しい環境では羽根の取り替えを頻繁に行うため、オープン形の遠心羽根車(バトルファン)を使うこともある。流量が少ない場合には、渦流ブロワが耐摩耗性に優れている。騒音も選定の要件になることが多い。一般に軸流送風機よりも遠心送風機のほうが騒音が低い。特に、翼形の後向き羽根をもつ翼形ファンは低騒音である。また、騒音は回転数の5~6乗に比例するので、効率が低くても圧力係数の高い多翼ファン(multibladefan)は有効である。同様の理由で、人家近くの小型風車には高効率・高速回転のプロペラ風車よりも、低速回転のクロスフロー風車が好んで設置される。
