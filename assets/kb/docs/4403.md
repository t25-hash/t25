# 1・1・3 比速度と機械の形式

> **章**: 第1章 気体機械一般
> **節**: 1・4 節参照)。

すなわち、気体機械の効率は無次元比速度と無次元比直径の関数になる。図III-1・6に、単段の送風機・圧縮機についてこの概略の関係を図示している。図III-1・6からわかるように、高効率を示す気体機械の無次元比速度と無次元比直径の関係は概略決まっており、形式を選定する際にはどちらか一方のパラメータをとればよい。通常、比速度がとられるが、慣用的に式(III-1・1)のを省略し、回転数および体積流量の単位も慣用の単位を用いた有次元の比速度(specific speed)が、形式選定のパラメータとして使用される。それ以上の比速第III部気体機械度では斜流羽根車を使用する。遠心ブロワ・圧縮機の場合は、羽根車入口に、軸方向から流入するインデューサを取付けた混流羽根車を採用することにより、比速度を500~650まで上げることが可能である。多段遠心圧縮機では後段に進むほど体積流量が減少するのに伴い比速度が減少し、高圧の圧縮機では初段で比速度が400以上でも、最終段では比速度が100以下になる。しかし、単体で比速度が100以下の場合には、効率の観点から二葉式ブロワなど容積形を採用するほうがよい。軸流羽根車では比速度が小さい場合はハブ比を大きく取り、比速度が大きくなるに従ってハブ比を小さくする。タービンや風車についても、図III-1・6と同様の比速度-比直径線図を描くことができる。比速度と羽根形状の関係は、送風機・圧縮機の場合と同じ傾向を示す。しかし、原理的に送風機や圧縮機のような逆圧力こう配の流れを伴わないので、比速度による効率変化が少なく、比速度に従って形状を選定するよりも、製作の観点から出力の大小によって軸流タービンとラジアルタービンなどの選定をすることが多い。
