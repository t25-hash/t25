# 1・1・3 比速度と機械の形式

> **章**: 第1章 気体機械一般
> **節**: 1・4 節参照).

気体機械の性能に関係する物理量として,直径,回転数,気体の密度ρ,体積流量,断熱ヘッド욀욃,軸動力を選び,次元解析を行うことにより,次の無次元量が導出される.욦=욀욃욦=욀욃욦=ρ욀욃(III-1・1)ここに,:重力の加速度.욀욃=-1-1(III-1・2)なお,圧力比が1.03以下の場合には,断熱ヘッド욀욃の代わりに=ρを使用する(:圧力上昇).上式の욦と욦は,それぞれ無次元比速度,無次元比直径で,욦は原動機の場合は断熱効率,被動機の場合は断熱効率の逆数である.すなわち,気体機械の効率は無次元比速度と無次元比直径の関数になる.図III-1・6に,単段の送風機・圧縮機についてこの概略の関係を図示している.図III-1・6からわかるように,高効率を示す気体機械の無次元比速度と無次元比直径の関係は概略決まっており,形式を選定する際にはどちらか一方のパラメータをとればよい.通常,比速度がとられるが,慣用的に式(III-1・1)のを省略し,回転数および体積流量の単位も慣用の単位を用いた有次元の比速度(specific speed)욦が,形式選定のパラメータとして使用される.욦=욀욃(III-1・3)回転数,体積流量,断熱ヘッドの単位を,それぞれmin,m/min,mとするとき,式(III-1・1)と式(III-1・3)の関係は욦=2575욦であり,効率の良いターボ送風機と圧縮機の形式と比速度の関係を示すと,およそ次のとおりである.遠心ファン욦=300~1000遠心ブロワ욦=150~500遠心圧縮機욦=150~500軸流ファン욦=1000~3000軸流ブロワ욦=1000~2500軸流圧縮機욦=800~1500遠心羽根車は,比速度が大きいほど内外径比が大きく,ディスクとシュラウドの幅(羽根高さ)が広く設計される.それ以上の比速第III部気体機械度では斜流羽根車を使用する.遠心ブロワ・圧縮機の場合は,羽根車入口に,軸方向から流入するインデューサを取付けた混流羽根車を採用することにより,比速度を500~650まで上げることが可能である.多段遠心圧縮機では後段に進むほど体積流量が減少するのに伴い比速度が減少し,高圧の圧縮機では初段で比速度が400以上でも,最終段では比速度が100以下になる.しかし,単体で比速度が100以下の場合には,効率の観点から二葉式ブロワなど容積形を採用するほうがよい.軸流羽根車では比速度が小さい場合はハブ比を大きく取り,比速度が大きくなるに従ってハブ比を小さくする.タービンや風車についても,図III-1・6と同様の比速度-比直径線図を描くことができる.比速度と羽根形状の関係は,送風機・圧縮機の場合と同じ傾向を示す.しかし,原理的に送風機や圧縮機のような逆圧力こう配の流れを伴わないので,比速度による効率変化が少なく,比速度に従って形状を選定するよりも,製作の観点から出力の大小によって軸流タービンとラジアルタービンなどの選定をすることが多い.
