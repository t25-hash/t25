# 3・4・6 ゲッタポンプ

> **章**: 第3章 真空ポンプ
> **節**: 3・4 真空ポンプの構造と特徴

ゲッタポンプ（getter pump）には蒸発形と非蒸発形がある．蒸
発形は，チタンワイヤあるいは図Ⅳ-3･16のようなチタン球をタン
グステンヒータなどにより加熱，蒸発させて，周囲の壁にチタンの
すポンプである．
蒸着膜を形成し，チタン膜による気体分子の吸着および収着によっ
て排気を行うポンプである．水素，酸素，窒素等の活性気体に対し
ては大きな排気速度をもつが，メタンなどの飽和炭化水素に対して
は排気能力が低く，不活性気体に対しては排気能力がない．そこ
図 -3･18クライオポンプ
で，活性の弱い気体に対する排気能力を高めるためには，チタン付
着面を液体窒素温度まで冷却し，その吸着効果を上げている．
非蒸発形は，ひだを付けて面積を拡大した金属ストリップの両面
に，非常に活性なジルコニウム-アルミニウム合金を50m程度の
厚さでコーティングすることにより，ゲッタ膜として作用させ，排
気を行うものである．非蒸発形は，蒸発形で起こる蒸着膜による真
空容器内の汚染や，蒸着膜による絶縁性の低下などの問題は起こら
