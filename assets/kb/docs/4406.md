# 3・4・7 スパッタイオンポンプ（spatter ion pump)

> **章**: 第3章 真空ポンプ
> **節**: 3・4 真空ポンプの構造と特徴

このポンプの原理は，気体イオンによるチタンのスパッタリング
を利用したものである．気体のイオン化を超高真空領域の圧力でも
持続させるために，強い磁場の中での放電（ぺニング放電）を利用
する．ぺニング放電（Penning discharge）というのは，図Ⅳ-3･17
のように2枚のカソード（陰極）とその間にある円筒のアノード
（陽極）に対して，軸方向に磁場を作用させた中で起こる放電のこ
とで，磁場を作用させることにより電子の飛行距離が長くなり，よ
り多くの気体分子をイオン化できる．このぺニング放電でイオン化
された気体はカソードに衝突し，カソードからチタン原子をスパッ
タする．スパッタされたチタン原子は周囲の壁に堆積して膜を形成
し，気体分子を吸着する．水素，酸素，窒素などの活性ガスはこの
原理で排気される．ヘリウムやアルゴンのような化学的活性の弱い
気体は，イオンとしてカソード面に入射してそこに捕えられる．
クライオポンプ（cryopump）は液体ヘリウムや液体窒素を用い
るか，小型冷凍機を用いて15，80Kの極低温の面（クライオパネ
ル）を作り，その表面に気体を凝縮固化し，さらに極低温の収着剤
に水素，ヘリウム，ネオンを収着させることにより，真空を作り出
