# 1・2・3 推力釣合い

> **章**: 第1章 気体機械一般
> **節**: 1・7 )。

ターボ圧縮機では吸込みと吐出しの圧力差のために軸スラスト(axial thrust)が発生する。遠心機では羽根車の吸込側に吸込圧力が、また背面側に吐出し圧力が作用し、背面から吸込側に向かって軸スラストが発生する。軸流機についても同様に吐出し側から吸込側に向かって発生する。軸には各段の軸スラストの合計が作用する。高圧圧縮機になるほど軸スラストは大きく、運転状態の変動時や非定常運転時には定常運転時よりさらに大きくなり、スラスト軸受の負荷が過大になって軸受の焼損を起こす場合がある。このため普通は、最終段羽根車の後に釣合いピストンを設けて軸スラストを軽減する方法がとられるほか、羽根車や軸流機の翼を対向させて配列したり、遠心機のダイアフラム部ラビリンスの径を大きくして中間段の羽根車は各段ごとに釣合わせ、第1段と最終段の推力を釣合いピストンで釣合わせる構造などがとられる。軸流機の場合は、ロータ両側の段付部で吸込側の段部には吐出し圧力をかけ、吐出し側の段部は吸込管につないで圧力を下げ、軸スラストを軽減することがある。軸継手(shaft coupling)が歯車継手の場合には、歯の切線力2―134流体機械、歯面の滑り摩擦係数をμとすると、継手に力がかかってもμの力になるまでは歯面が滑らず、この力がスラスト軸受にかかる。ダイアフラム式継手の場合は、ダイアフラムのたわみによる反力が軸スラストとなる。固定式軸継手の場合には、駆動機や歯車から伝達される軸スラストを考慮しなければならない。このようにして、圧縮機自体で推力釣合いをとった後の残留軸スラストと、これら駆動機や歯車、継手などで生じる軸スラストを総合して軸系全体の軸スラストを決定し、スラスト軸受を設計する必要がある。
