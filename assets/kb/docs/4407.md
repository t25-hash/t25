# 1・2・4 軸受

> **章**: 第1章 気体機械一般
> **節**: 1・7 )。

ターボ圧縮機の軸受としては各種のものが使用されるが、代表的なものを示す。a. ジャーナル軸受(journal bearing)一般的には平軸受かティルティングパッド軸受が使用される。後者は一般に、若干許容荷重が小さくなりコスト高とはなるが、振動に対する安定性が良い。図III-1・16に、比較的大型圧縮機に使用される球面座をもつ軸受を示す。軸受は炭素鋼、合金鋼、鋳鋼などで作られ、遠心鋳造などでホワイトメタルが内張りされている。負荷を支える油膜形成面にのみ直接潤滑油を供給し、その後すぐに軸受外に排出する構造にして機械的損失を低減させる例もある。b. スラスト軸受(thrust bearing)一般的には、テーパランド軸受とティルティングパッド軸受が最も多用されている。荷重方向に後者を、反荷重方向に前者を組合わせることもあるし、両方とも後者または前者のタイプを使用することもある。図III-1・17にティルティングパッド軸受の例を示す。ジャーナル軸受と同様、負荷を支える油膜形成面にのみ直接潤滑油を供給し、その後すぐに軸受外に排出する構造にして機械的損失を低減させる例もある。c. 磁気軸受(magnetic bearing)磁気軸受は電磁石あるいは永久磁石を用いて、磁力により回転軸を宙に浮かせるもので、ターボ分子ポンプなど小型高速回転機械では一般化しているが、最近では石油プラントやパイプラインなどの大型回転機械でも使われる例がある。最大の長所は潤滑油が不要となることで、ガスの清浄度維持や保守管理などにメリットがあるが、磁力の限界から寸法的に大型化することや停電・電気回路故障時の対策などに注意を要する。
