# 1・3・1 相似条件(similarity condition)と相似パラメータ

> **章**: 第1章 気体機械一般
> **節**: 1・3 性能試験と性能換算

(similarity parameter)気体機械の性能試験を行う際、実際の使用条件と同じ条件で行えるケースはまれであり、試験の条件を相似に保って行う。三つの相似条件を以下に示す。a. 幾何的相似条件模型により試験を行うとき必要な条件で、対応するすべての形状が相似でなければならない。しかし、模型と実機のスケールが著しく異なる場合、機械のすきま、角の丸み、表面粗さなどを相似にすることが不可能になり、寸法効果(scale effect)が現れる(1・3・4項参照)。b. 運動学的相似条件実際の運転と試験運転において、対応するすべての点の速度比が等しくなければならない。この条件は、通常、a項とc項の条件を満たしていれば、流入境界、流出境界における速度分布が相似になるように配慮することにより得られる。性能試験に関する諸規格では、流量計などの計器がこの条件を阻害しないように、装置の形状・寸法が定められている。しかし、試験に使用するガスの比熱比が実際と異なる場合、または効率が著しく異なる場合には気体の圧縮・膨張の仕方が異なるので、圧力比の大きい気体機械では、流入・流出の相似条件を合わせても中間段で密度比が異なり、運動学的相似条件を満たさなくなる(1・3・3項参照)。c. 力学的相似条件次元解析により得られる空気力学的相似パラメータが等しくなければならない。圧力比が大きく圧縮性が無視できない場合、ガスを理想気体とすれば相似パラメータは以下のようになる。このほかに、b項の条件によりガスの比熱比も重要な相似パラメータになる。なお、すべての相似パラメータを等しくするのは困難であり、通常、機械レイノルズ数の相似条件を除いて試験を行い、その影響は別途考慮する。ファンの特性は、これらの無次元量に比例する流量係数、圧力係数、軸動力係数を用いて性能曲線を描けば、機械レイノルズ数が著しく違わない限り、回転数が異なっても一本の特性曲線で記述できる。風車では、圧力と流量は性能表示に直接関係せず、風速を用いて、次の無次元数で特性を表示する。
