# 1・3・2 性能換算(performance conversion)

> **章**: 第1章 気体機械一般
> **節**: 1・3 性能試験と性能換算

a. 圧縮性の影響がある場合比熱比が同じガスで相似条件を保って性能試験を行うとき、試験と規定状態における機械レイノルズ数が著しく違わなければ、相似パラメータに基づいて性能換算を行う。規定の状態に*を付して表すと、規定条件の圧力比、流量、回転数、軸動力は、次式により換算できる。ターボ形圧縮機の場合には、通常、横軸に換算流量、縦軸に圧力比、軸動力、効率、パラメータに換算流量をとって性能曲線を表す。容積形圧縮機では、回転数や圧力比を横軸にとって性能曲線を表すことも多い。また、タービンの場合には横軸に換算回転数をとり、ほかの値を縦軸で示す速度特性がよく用いられる。b. 圧縮性が無視できる場合(圧力比<1.03)次式により性能換算を行う。
