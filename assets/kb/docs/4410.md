# 1・3・3 比熱比が異なるガスによる性能試験

> **章**: 第1章 気体機械一般
> **節**: 1・3 性能試験と性能換算

圧縮機では規定のガスで性能試験を行うことが困難で、比熱比の異なるガス(例えば空気)で手軽に試験を行いたい場合がある。このとき1・3・1・b項の運動学的相似条件を満たすことはできないが、入口と出口の密度比が等しくなる等価回転数を用いて試験を行えば、ある程度の誤差で性能換算を行うことができる。図III-1・18は、ポリトロープ指数が1.6の場合に式(III-1・21)の等価回転数で運転したときの、入口と出口の中間の圧力比の位置における密度比のひずみを示している。ひずみが2%以下であれば、異なるガスを用いて等価回転数で試験を行ってもよい。
