# 1・3・4 寸法効果(scale effect)

> **章**: 第1章 気体機械一般
> **節**: 1・3 性能試験と性能換算

模型で試験を行う場合,寸法の違いはレイノルズ数の中に代表寸法として含まれているので,この効果を修正する必要はないように思われるが,幾何的相似条件を細部にわたるまで満たすことは実際上困難であり,表面粗さ,機械要素間のすきま,角の丸みなど,相似にできない要因の影響を受け,模型の寸法比が小さくなるほど性能は低下する.これらの効果をまとめて寸法効果またはスケール効果という.寸法効果の現れ方は機種によって異なるが,公表された資料は少ない.文献(1) 生井武文・井上雅弘,ターボ送風機と圧縮機,(1988),109-112,コロナ社.(2) Balje,O.E.,Turbomachines―A Guide to Design,Selection andTheory,(1989),John Wiley& Sons.(3) 白木万博・ほか3名,機械の研究,29-8(1977),881.(4) API Standard 617,6th ed.,(1995),18.(5) Shiraki,K.and Kanki, H., Proc. 8th Turbomachinery Symp.,(1979),59.(6) API Standard 617,6th ed.,(1995),11.(9) TSJ S 001:2004, 模型によるファン・ブロワのモデル試験方法,ターボ機械協会.→ Next Chapter
