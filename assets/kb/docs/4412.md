# 2・1・1 種類および使用範囲

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

回転容積形圧縮機(rotary type positive displacement com-pressor)は,ケーシング内壁とロータとで作られた押しのけ室の容積を,ロータの回転により吐出し側にそのまま,あるいはその容積を減少して送り出し圧縮するもので,以下のものに大別できる.a. 二葉形二葉形圧縮機(two lobe compressor)は機械的強度,温度上昇の制限により1段での圧力比が2程度,圧力上昇が約0.1MPa,ガス流量が2~600m/minの範囲で使用できる.空気その他のガスの圧送,真空用に使用される.ガス流量調整は主としてバイパス制御による.b. ベーン形ベーン形圧縮機(sliding vane compressor)はガス流量1~30m/min,吐出し圧力1MPa以下の範囲で,おもに空気圧縮機として使用される.ガス流量7m/min以下のものは油冷式1段形としており,1段で0.7MPaまで昇圧している.大型機では油冷式2段圧縮,吐出し圧力0.7MPaが一般的である.ガス量調整には回転速度制御(エンジン駆動の場合),吸気閉鎖制御を併用,または単独使用している.c. ローリングピストン形代表的な強制駆動弁形(ロタスコ圧縮機)が,押しのけ流量1~16m/minの範囲でR-12,R-22,NHの冷媒圧縮機として使用されている.3.75kW 以下の冷媒圧縮機用としてカム形(センコ形)圧縮機も使用される.揺動ピストン形,カム形圧縮機は真空ポンプとしても使用される.d. ねじ形ねじ形圧縮機(screw compressor)には,ほとんどすべてのガス圧縮機に用いられる無潤滑式(dry type)と,空気,冷媒,一部のプロセスガス圧縮機に用いられる油冷式(oilflooded type)とがある.無潤滑式は1段の圧力比で8以下,ガス流量5~650m/min,最高吐出し圧力は4段圧縮で3.5MPaの範囲で使用される.油冷式は,空気圧縮用には0.85MPa以下の吐出し圧力の場合1段形でガス流量25m/min,2段形で110m/min以下の範囲で使用される.冷媒,プロセスガス圧縮機用としては,吐出し圧力2.3MPa以下,ガス流量2~150m/minの範囲で使用される.圧力比は,1段形で冷媒圧縮用の場合25,プロセスガス圧縮用には20程度まで使用される.e. スクロール形スクロール形圧縮機(scroll compressor)はガス流量0.03~3m/min,圧力比2~10の範囲で,おもに冷媒圧縮機,空気圧縮機として使用される.油冷式と無給油式とがあり,空気圧縮機としては無給油式で吐出し圧力0.83MPa,0.75~11kW の範囲で使用される.
