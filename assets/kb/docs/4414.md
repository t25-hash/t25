# 2・1・3 ベーン形

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

a. 構造シリンダ内部に偏心したロータを設け、ロータに多条の溝を軸方向に設け、溝内に径方向に可動のベーンを装入する。ロータが回転するとベーンは遠心力によりシリンダ内面に押付けられてしゅう動する。図III-2・4にベーン形の圧縮原理を示す。ベーンには普通、積層フェノール樹脂を使用し、小型機では特殊金属を採用して耐摩耗性を向上させている。ガス流量が5m/minを超えるものは、ベーンの強度がガス圧に耐えられなくなるため2段形となるが、1段形と基本的な構造は同一である(図III-2・5)。吸込口の位置はベーン間の容積が最大になるように選定する。この位置はベーンの厚みを=0とした場合、ロータの中心から-β2の位置になる(図III-2・4)。吐出し開始の位置は、遅すぎると過圧縮を生じて動力が増加する。
