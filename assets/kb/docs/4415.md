# 2・1・4 ローリングピストン形

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

a. 構造代表的なロタスコ形圧縮機を図III-2・6に示す。シリンダ内に偏心して取付けられた回転ピストンが、平板状のベーンと接触を保ちつつシリンダ内面を回転して圧縮作用を行い、吐出し弁を経て圧縮ガスを排出する。ベーンは軸とクランク機構で同期して動き、その下端は回転ピストン外周に対してつねに垂直に向き、波形ばねでピストン外周に軽く圧着して機密を保持している。ピストン外周には弾性のあるスリーブをはめ込んでいる。多量の潤滑油が圧縮熱の除去とシール、潤滑の目的で供給されるので、1段で25の高圧力比を得ることができる。c. 性能ロタスコ形では吐出し終了後に、弁のすきま部分にたまった圧縮ガスが再膨張して吸込口につながるので、絞り損失が生じる。1MPaの吐出し圧力の場合、油冷式のため吐出しガス温度も100°C程度で等温圧縮に近く、体積効率も良い。真空ポンプに使用すると体積効率は99.9%まで増加する。
