# 2・1・5 ねじ形(二軸形)

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

a. 構造ケーシングの内部にねじれたおす,めすの一対のロータをかみあわせ,ねじれた歯溝内の容積をロータの回転により減少させ圧縮する(図III-2・7).おすロータ(male rotor)の巻き角は250°~360°である.. 無潤滑式おす,めすロータ(female rotor)の軸端に同期歯車(timing gear)を設け,ロータ間に一定のすきまを保ち,ロータとケーシング間にも微小なすきまを与え,接触がないようにロータを回転させて無潤滑圧縮する.各部のすきまは内部漏れによる体積効率の低下を防ぐよう微小とし,普通ロータ直径の1000分の1程度とする.ケーシングの周囲には冷却ジャケットが設けられている.ロータは十分に剛性のあることが必要で,材料には炭素鋼2―138あるいはステンレス鋼が使用される.また,空気圧縮機の場合,ロータ表面には防錆コーティングが施される.軸受はおもに転がり軸受が使用され,ロータの半径方向・軸方向の移動量を微少に保つ.軸方向のロータ移動を防いだり,圧力差によるスラスト荷重低減のために釣合いピストンを設けることもある.軸封にはフローティングリングやラビリンスシール等のすきまシールが用いられるが,可燃性ガスの場合にはオイルフィルムシール,あるいはメカニカルシールを使用する.. 油冷式おす,めすロータの駆動面の潤滑,圧縮熱の除去,油膜によるシールを目的として多量の油が注入される.ロータ間が潤滑されており,おす,めすロータを駆動するので同期歯車は不要である.軸受には一般に転がり軸受が使用されるが,吐出し圧力の高い冷媒圧縮用には滑り軸受が使用される.独特なロータプロフィルのためにスラスト力を発生するので,差圧が大きい場合にはスラスト力の分散を図る釣合いピストンを設ける.ガス吐出し温度は70~80°Cと低いためにケーシング,ロータは冷却の必要がなく,ロータすきま,ロータ,ケーシング間のすきまも無潤滑式に比較して小さくできる.冷凍機の場合,潤滑油系統はガスを密封サイクルにする必要があり,圧縮機および油ポンプの入力軸部分にメカニカルシールを用いるか,あるいは密閉形電動機駆動とする.冷媒圧縮用のものとほぼ同一の構造で,プロセスガス圧縮用に使用できる.この場合,潤滑油はガスに対する化学的安定性,油の蒸気圧,油に対するガスの溶解度,油中の添加物の影響を考慮して選択する必要がある.b. ロータ形状一般に使用されている非対称形ロータ形状の一例を図III-2・8に示す.これは中心線に対していくらか変形しており,ブローホール(blow hole,おす,めすロータとケーシングとで生じる三角状のすきま)を小さくし,歯溝間の吹抜けを少なくして効率を改善している.ロータの歯数は,おす4枚,めす6枚の組合せやおす5枚,めす6枚の組合せ等が使用されるが,低圧用では歯数を少なくして押しのけ容積を大きくしたもの,高圧用では剛性をもたせるため歯数を多くしたものが使用される.c. 回転速度一定の場合無潤滑式,油冷式ともに全断熱効率は設計点付近で最高で,圧力比の増減とともに低下するが,その変化は圧力比の相当大幅な変化に対してもわずかである(図III-2・9).油冷式の場合も圧力比の変化による体積効率の低下は小さく,圧力比が2から12まで変化してもその変化は10%以下である(図III-2・10).. 圧力比一定の場合押しのけ流量と吐出しガス流量との差である内部漏れ量は,圧力比が一定であれば回転速度の変化にかかわらずほぼ一定である.したがって,η욋は高速になればよくなる.全断熱効率η욧욪욀욃は最適速度から外れると動的損失の増大により低下するが,いずれもかなり広い範囲にわたりその変化はわずかである.e. ガス流量調整法. 無潤滑式空気圧縮機の場合,全量吐出しによる負荷運転と吸気閉鎖・吐出し開放による無負荷運転の組合せにより,空気供給圧力を一定範囲内に保持する2段階制御が一般的である.可変速度方式の場合は,100~30%程度の範囲で吐出し圧力をほぼ一定に保つように速度制御をする.30%以下の範囲では2段階制御などを併用する.ガス圧縮機の場合にはバイパス制御も使用される.多図-2・9回転速度および圧力比と全断熱効率およびガス流量の関係図-2・10 油冷式小型空気圧縮機の場合には電動機ON/OFF制御,吸気閉鎖制御を使用しているが,可変速度方式により100~20%程度の範囲を速度制御することで動力節減を図る場合もある.大型空気圧縮機と冷媒圧縮機では,100~10%の範囲の容量制御に滑り弁を使用している.この方式は,滑り弁の移動によって生じる滑り弁とケーシングとの空間より吸込側にバイパスさせ,必要なガス流量のみ圧縮するものである.所要動力はほぼガス流量に比例して低下する.
