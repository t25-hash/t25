# 2・1・6 スクロール形

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・1 回転送風機および圧縮機

a. 構造スクロール形圧縮機は、複数巻きの渦巻で構成される固定スクロール(fixed scroll)と同一形状の渦巻で位相が180°ずれている旋回スクロール(rotating scroll)が、各中心を旋回半径分ずらした状態で軸方向、径方向とも数十mの間げきでかみあわされるように構成される。旋回スクロールが固定スクロール中心のまわりを自転せずに旋回公転することにより、かみあわされた渦巻と端板で形成される圧縮室の最外周に閉じこめられたガスが、除々に中心部へ送られていく過程で圧縮される(図III-2・11)。その行程は、以下のとおりである。(1) 固定スクロール外側の吸込口からガスを吸入する。(2) 圧縮空間に封じ込められたガスは、旋回運動に伴う圧縮室の縮小によって、渦の中心に向かって圧縮される。(3) 圧縮空間は中心部で最小となるため、ガスは最高に圧縮され、中心部にある吐出し口から外へ押出される。このように、旋回スクロールの1回転中に吸込み→圧縮→吐出しの行程が同時に連続的に繰返されるため、往復動圧縮機に比べ、一回転中のトルク変動が少なく、低振動・低騒音を達成するのに有利な構造である。スクロールの形状は、円のインボリュート曲線が一般的に使用されている。. 油冷式図III-2・12に油冷式スクロール形圧縮機の代表的な本体構造を示す。油冷式においては、自転防止をオルダム継手により行うのが一般的である。また、旋回スクロールには、半径方向と軸方向のガス荷重により傾けようとするモーメントが作用するため、これに対抗して旋回スクロールを固定スクロールに押付ける必要があり、ガス圧を背圧として作用させたり、ばね力を加えたりする方法などが取られる。油冷式は、一対の渦巻の軸方向および、径方向の間げきを冷却油がシールするため、高い体積効率(volumetric efficiency)を得ることができる。一方、吐出しガスのオイルセパレータや給油装置、給油配管が必要となるため、圧縮機周辺機器は複雑になる。2―139第III部気体機械図-2・13 無給油式スクロール形圧縮機。 無給油式図III-2・13に無給油式スクロール形圧縮機の代表的な本体構造を示す。無給油式の本体構造上の大きな特徴は、補助クランク機構とチップシール(tip seal)である。潤滑材として油が使用できないため、自転防止とスラスト荷重を受けることを目的として補助クランク機構を採用し、グリースを封入した軸受を用いる方法が一般的である。また、軸方向の間げきからの漏れは圧縮機としての性能を著しく低下させるため、渦巻の先端に溝を設け、その部分にシール材を挿入する方法が一般的に用いられている。このシール材すなわちチップシールは、渦巻が形成する各圧力室間の差圧により浮上し、しゅう動面となる相手の渦巻底面に押付けられ、シール材として機能する。無給油式の場合は、圧縮等により発生した熱を放出するために空冷方式を採用するのが一般的であり、周辺機器として冷却ファンを備えている場合が多い。b. 押しのけ流量읏읈=2ε2 1+ε λ-2π+πmminここで、ε:旋回半径(m)、:スクロール高さ(m)、:回転数(min)、:スクロール歯厚(m)、λ:巻き終わり角度(°)。c. 性能スクロール形圧縮機は、その構造より隣接する圧縮室の差圧が小さく、さらに低圧部と高圧部の間に直接のガス漏洩通路が形成されないため、ガス漏洩による損失が小さいことや、しゅう動速度が比較的小さく、機械損失が少ないことにより、往復式圧縮機やローリングピストン式に比べて、広い圧力比範囲で優れた体積効率および全断熱効率特性を示す。
