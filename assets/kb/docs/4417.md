# 3・2・3 空気圧モータ

> **章**: 第3章 空気圧システム
> **節**: 3・2 空気圧アクチュエータ

空気圧モータ（pneumatic motor ）は，コンプレッサによって得
られた圧縮空気のもつ流体エネルギーを，機械的回転エネルギーに
変換するアクチュエータである．その出力は比較的小さく，20kW
程度までのものが一般産業用機械として用いられている．
a.ベーンモータ（vane motor ) ベーンモータは一般に小・
中容量で，高速回転を必要とする分野において多く使用されてお
り，空気消費量はピストン形空気圧モータに比べて多いが，小型軽
量で，構造が簡単である．

c.歯車モータ（gear motor ) 図Ⅴ-3･15に平歯車を使用した
歯車モータの構造図を示す．歯車モータは構造が簡単なことから，
小型軽量にできる．小型のものでも1万min程度の高速回転が
可能であり，大型のものでは約50kWのものも製作されている．
d.空気圧モータの特性 図Ⅴ-3･16に空気圧モータ性能曲線
を示す．一定供給圧力の状態において，速度が零のときに出力は
零，速度が速くなるにつれて馬力は増大し，無負荷速度の50%近
くでその値は最高値をとる．
トルクの減少は速度の増加によって平衡を保たれ，無負荷速度
（最高速度）においてトルクは零になる．この場合，空気圧モータ
の発生出力は，モータ内を通過する空気の抵抗によってすべて消費
される．
空気圧モータは，運転すれば空気量の増大とともに入口と出口間
の圧力損失のため，トルクが減少していく．さらにモータの運転の
ために必要な機器である切換弁，マフラ，絞り弁および配管抵抗が
さらにエネルギーを消費し，実際に使用する場合の空気圧モータの
特性は，トルク，出力とも図Ⅴ-3･16より小さくなる．
