# 3・2・4 空気圧揺動形アクチュエータ

> **章**: 第3章 空気圧システム
> **節**: 3・2 空気圧アクチュエータ

出力軸がある角度で回転往復揺動するアクチュエータが空気圧揺
動形アクチュエータであり，その構造はおもにベーン形とピストン
形がある．

体の軸を挿入したもので，シングルベーン形とダブルベーン形があ
る．シングルベーン形は回転角度が大きく，最大で約270° ，ダブ
ルベーン形は回転角度が小さく，最大で約100°の回転角度をもつ．
揺動アクチュエータの中では出力に比べて最小型，最軽量とな
る．しかし，コーナ部からの漏れを止める手段が難しく，かつパッ
キンの接触長さが他の方式に比べて長いことから，パッキンの抵抗
損失が大きい．出力効率は，空気漏れの少ないものでは摩擦損失の
ために80%を超えることは困難である．
b.スクリュー形揺動アクチュエータ（screw type rotary
actuator ) 図Ⅴ-3･18に示すように，回転軸に加工されたリー
ドの長いスクリューによって回転が規制されているピストンに沿っ
て，軸が回転運動をする．スクリューおよび回り止めガイドの摩擦
損失によって効率は低いが，外力によって逆回転はしにくく，回転
角度は360°を超えてもよい．揺動角度の大きい要求に向いている．
c.ラックピニオン形揺動アクチュエータ（rack and pinion
type rotary actuator ) 図Ⅴ-3･19にラックピニオン形の一例を
示す．ピストンに連結したラックによりピニオンを回転させる構造
で，ピストンは直線運動のみを行う．歯車の圧力角による反力を受
ける構造が十分であれば，摩擦損失が少ないので効率は高い．回転
角度はシリンダのストロークによって定まるもので，360°を超え
るものも製作は可能であるが，あまり大きいものはラックが長くな
り不適合である．
d.クランク形揺動アクチュエータ（crank type rotary
actuator ) 図Ⅴ-3･20にクランク形揺動アクチュエータの一例
を示す．ピストン出力をクランクにより揺動運動に変えるもので，
構造が簡単で信頼性が高い．クランクとピストン運動方向との角度
により出力に差が出るので，出力の割りにやや大きなピストンが必
要である．
e .使用上の注意 被駆動物の停止時における揺動アクチュエ
ータの出力軸への衝撃エネルギーはかなり大きく，そのうえ揺動ア
クチュエータの出力軸はねじれを受け，衝撃耐力が小さい．したが
って，被駆動物の停止ストッパを揺動アクチュエータの外に設ける
注意が必要である．
また，揺動アクチュエータの場合，低摩擦のものが最近市販され
てはいるが，一般に低速回転ではスティックスリップ現象に注意が
必要である．
