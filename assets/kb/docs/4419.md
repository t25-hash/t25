# 2・2・2 圧縮機の理論と実際

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・2 往復圧縮機

そして各段の圧縮比を等しくすれば、ガス圧縮のための理論動力が最小となり有利である。各段の圧縮比は、圧縮ガスの種類、吐出しガス温度、ピストン力およびガス圧力などを考慮して決められる。吐出しガス温度についてはシリンダ潤滑方式、潤滑油の種類、ピストンリングの材質、シリンダ冷却方式およびガスの物性によって許容値は異なるが、吐出しガス温度が低いほどシリンダ内の潤滑は安定する。また、付属配管および容器の熱膨張、熱応力の影響が少なくなり、シリンダへの反力も減る。なお、圧縮比の一般的な値を表III-2・4に示す。b. 吐出しガス温度吐出しガス温度は一般的に150°C以下としているが、低圧段では180°C程度まで許容している。しかし、潤滑油として合成油を使用したとき、無給油式のシリンダで耐熱性の高いリング材を使用した場合は180~200°Cを許容している。一方、ガスによっては爆発性、重合および腐食性より温度制限があり、100~120°C以下と低く抑える場合があるが、プロセス用圧縮機で分子量12以下の水素リッチガスでは135°C以下に抑える要求がある。また、酸素の場合は安全性を考慮して吐出しガス温度も一般に比べて10~20°C低く抑え、給油式空気圧縮機では潤滑油の酸化による燃焼を防止するため、特に多段シリンダの2段め以降を圧力により150~120°C以下程度にしている。また、無給油式の高圧シリンダではリング強度、摩耗を考慮して低くする。実際のシリンダ吐出しガス温度′′(°C)の求め方として、一般には2種類の方法がある。温度効率は0.5~2.0と幅が広く、効率設定には類似機の運転実績によるところが大きい。例えば100%負荷運転の条件で、低温ガス-空冷シリンダで0.6~0.9、常温ガス-空冷シリンダで0.7~1.0、常温ガス-水冷シリンダで0.8~1.2程度であるが、常温ガス-鋳鉄製水冷小型シリンダでは、冷却効果が大きいものは1.2~1.8程度である。もう一つの方法は、シリンダ内のガスがポリトロープ変化で圧縮されるとして、断熱指数の代わりにポリトロープ指数により求める。. efficiency)ηは次式による。断熱圧縮としたとき圧縮比3程度の常温ガスでは0.80~0.90、低温ガス、単動式、高圧無給油式、ラビリンス式ピストンでは0.75~0.85、分子量が軽いガスでは0.85~0.95程度である。吸入ガス圧力がある昇圧機では、圧縮機吸入行程での押込み効果により、0.03~0.08程度効率が上がる。εは一般に、ピストン径が大きいほど、ピストン行程が長いほど、そしてガス圧力が低いほど小さい。500kW 以上の中・大型機の低圧段で0.05~0.12、中圧段で0.10~0.20、高圧段で0.15~0.30,100~500kW の小・中型機の低圧段では0.08~0.15、中圧段で0.15~0.25、高圧段で0.20~0.30,100kW 以下の小型機で0.12~0.35程度である。体積効率をよくするためにεを小さくする手段として、吸入弁、吐出し弁およびシリンダのガス通路を狭くすると圧力損失が増加し、かえって全断熱効率を減じて得策でない。一般にηはガス容量が小さいほど、高速であるほど、圧縮比が高いほど、そしてガスの比熱比が小さいほど悪くなる。500kW 以上の中・大型機では0.65~0.85,100~500kW の小・中型機では0.60~0.80,100kW の小型機では0.50~0.70程度である。しかし、圧縮比が1.3以下の循環機では0.85~0.95になる。d. 軸動力と全断熱効率。多段圧縮の場合は、各段ごとの理論断熱動力を個別に計算して総和を求めとする。ηは500kW 以上の中・大型機で0.75~0.85,100~500kW の小・中型機で0.65~0.80,100kW 以下の小型機で0.50~0.75である。また、圧縮機クランク機構の軸受関係の機械損失動力は、軸動力の0.02~0.04程度である。は連続運転する最大動力を考慮し、また、圧縮機の安全弁噴出時の動力に対し、少なくとも1.05以上とする。回転数は圧縮機の形式による構造上の違いから、発生する回転および往復運動部分の不釣合い慣性力に差があるため、圧縮機および基礎の耐震設計上の問題より125~1800minと差が大きい。一般に、回転数が速いのは対向形、大型より小型であるが、1000min以上の高速機は対向形が適している。一方、平均ピストン速度は3~5m/s程度であるが、無給油ピストン式の場合リングの摩耗を考慮して、給油式より遅く2.5~4m/s程度である。また、小型の場合は1.5~2.5m/s程度である。表III-2・5に圧縮機の回転数を示す。なお、行程中のピストン速度(ms)は次の近似式で表され、ほぼ行程の中央部で最大となる。f. ピストン力と慣性力圧縮機運転中のクランク軸の回転運動、およびピストン等の往復運動により、運動部分にはピストンに働くガス圧力によるピストン力(piston force)(kN)とピストン慣性力(piston inertia force)(kN)が発生する。また、回転トルクにはピストン往復運動によるガス圧縮仕事およびクランク機構の運動によるすべての損失動力が含まれているので、回転トルクによる仕事量は圧縮機軸動力(kW)である。クランク軸トルク線図(例)を図III-2・21に示す。が大きい場合は、クランク軸のカップリング部に回転慣性力を付加したフライホイールを設けてを小さくする必要がある。は普通クランク数が少ないとき、圧縮機負荷が小さい容量調整時に大きくなる。なお、回転エネルギー変動率により影響される回転軸の速度変動率は、動力伝達装置および駆動機の種類に応じて、安定した運転をするための目安が決まっている(表III-2・6)。h. 圧縮機起動トルク圧縮機を起動して正規回転まで加速するのに必要なトルクで、駆動機の起動を容易にするため、一部の小型機を除き、圧縮機は2・2・3項に示す容量調整装置により無負荷状態にして起動される。通常、起動トルクは軸動力の25~40%である。また、電動機駆動の場合、起動後に定格回転数までになる起動時間は普通3~6秒程度である。
