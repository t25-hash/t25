# 3・3・1 分類

> **章**: 第3章 空気圧システム
> **節**: 3・3 空気圧制御弁

空気圧制御弁（pneumatic control valve）は，空気圧機器へ供れる．
機能の面から分類すると，空気流れの通過・閉鎖および方向一定の量に保つものと，操作入力により連続的に制御する比例制御ある．
一方，流量や圧力の制御は，可変絞りによって空気流路変えることにより行われる．
代表的な弁構造を図Ⅴ-3･23に示ノズルフラッパ弁（nozzle flapper valve）は上流側（背圧側）フラッパの間隔により操作される下流側可変絞りをもつ．
これらの低くなるので，間隔を連続的に調整すれば圧力が操作できる．
直接少ないが，空気圧制御弁の内部で，主弁を操作するパイロット圧をを閉鎖したり，下流側圧力や流量を操作するために使われる．
空気縮性が高いため，多くの場合絞り部で，流れはチョーク状態となが大きい．
絞りの特性と類似するが，流路は単純な絞りよりも複雑なので，チる．
の，電気入力操作によるもの，および空気圧パイロット操作による切換えたり偏向させるフルイディクス も制御弁の一種である．
給する空気の圧力を調整したり，機器の動きを制御するために使わ制御，流量の制御，圧力の制御の3種になる．
流れの通過・閉鎖および方向制御は，可動弁によりオンオフ式にの開口面積を変化させ，絞りの上流側あるいは下流側圧力や流量をす．
の圧力変化を利用するもので，上流側固定絞り，およびノズル絞り間の圧力（背圧）はノズルフラッパ間隔が狭いと高く，広いとノズルフラッパ弁を用いて，圧力の調整や機器の制御をすることは発生する用途に広く使われる．
圧制御弁の構造自体は油圧式と変わらないが，空気は油に比べて圧る．
弁に働く流体力は油圧式と同様に求められるが，摩擦力の影響弁の流量特性は，3･1･1項に記述されている基礎式で与えられるョーク流れが発生する臨界圧力比は，絞りの場合の0.528より一般操作方式で分類すると，人や機械が力を加えて直接操作するもものがある．
