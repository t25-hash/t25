# 3・4・1 フィルタ

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

空気圧用フィルタには回路中の微細な固形異物をろ過する以外
に，圧縮空気中の液化水分（ドレン）を除去するという二つの機能
が要求される．図Ⅴ-3･36 に代表的な空気圧フィルタの構造例を
示す．入口から入った圧縮空気はデフレクタにより旋回運動を与え
られ，そのサイクロン効果により，比較的大きな遊離水滴や異物を
ケースの内壁に押しやり，壁面に伝わって落下させ底部にためる．
このようにして大部分の異物が除かれた圧縮空気は，中央の合成樹
脂や焼結金属で作られた微細な穴が無数にあるフィルタエレメント
を通ることで，さらに小さな微粒子を除去し，出口側から流出す
る．分離された水分やごみなどは，ボウルの下部に付いているコッ
ク式，プッシュ式などの手動バルブ，あるいは自動ドレンバルブに
より外部へ排出される．
ろ材の目の細かさは粒度（ m）表示するが，用途に応じ概略次
のように使い分ける．
