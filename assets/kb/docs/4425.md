# 2・2・8 圧縮機付属機器

> **章**: 第2章 容積形送風機および圧縮機

a. 逆止め弁圧縮機のガス出口配管に圧縮機保護のために逆止め弁(non-return valve,check valve)が設置されるが,往復圧縮機の場合は出口配管内の送出ガスに脈動があるため,一般によく使用されるスイング式,バタフライ式は,ガス脈動の影響を受けるために適さない.さらに,圧縮機が容量調整により送出ガス量が少なくなってガス流速が遅くなると,スイング式では内部弁へのガス衝突荷重が不足して弁が一時的に閉じてしまい,結果として内部弁がガス衝突荷重とガス差圧荷重により開閉を繰返す,チャタリング現象を起こすことがある.したがって,往復圧縮機の場合は逆止め弁の形式として,一般的に圧縮機吐出し弁形を使用している.弁としては輪形弁が多く,弁板はステンレス鋼に加え,そのシール性より樹脂製が使用されている(図III-2・41).b. 巻管式巻管式は別名侵漬式(しんせきしき)ともいわれ,小型圧縮機,ガス圧力5MPa以上の中・高圧用途に使用されている.水槽は鉄板溶接構造,コンクリート製があるが,水中にある冷却管および管支持架設の防食対策が必要である.冷却管の材質としては,高圧用炭素鋼管,ステンレス鋼管,銅および銅合金などが使用されているが,冷却管の口径が大きいので管が肉厚となる.また,本冷却器は構造上管外の冷却水の流速が遅くなるので,熱貫流率は低い.. 二重管式ガス圧力5MPa以上の高圧で使用され,冷却管の内部をガスが流れ,冷却管の外側に別の鋼管により水室を作り,冷却水をガスの流れ方向と対向して流す構造であるため,熱貫流率は高い.また,水室は溶接固定式と組立式があり,冷却管の材質としては高圧用炭素鋼管,ステンレス鋼管が使用されている.. 灌水式二重管式の冷却管の組合せ方法と同じであり,冷却水室はなく,冷却器上部から冷却水を冷却管の両側より伝って流れ落して冷却する方式で,冷却水が積まれた冷却管にできるだけ均一に伝って落ちるよう,各冷却管の間は伝達板で接続して風の影響も防いでいる.構造は二重管式より簡単であるが,寒冷地には適さない.. 空冷式冷却管は外周にフィンをもったフィンチューブが普通で,チューブの両端はヘッダに拡管固定か溶接固定される.冷却用の空気ファンは電動機で,直結またはV ベルト減速などにより駆動される.冷却器の形式としては,冷却管にファンで送風する送風形(forced type)と,ファンの吸込みで冷却する吸引形(induced type)の2種類がある.冷却管の材質としては,アルミニウム合金管,炭素鋼管,ステンレス鋼管銅および銅合金管が一般に使用されていて,冷却フィンの材質は耐食性,熱伝導性,加工性を考慮して,埋込み式ではアルミニウム製が多い.. separator)高温になった圧縮ガスを冷却すると,ガス中に含まれる過飽和の水分,油分,凝縮性ガスが凝縮,液化するので,冷却器の出口側にドレン分離器を設置して,ガス中に含まれるこれらをドレンとして捕集分離している.ガス中の腐食成分が溶け込んだ凝縮水,液化ガスは,潤滑の阻害,腐食の促進,シリンダ弁の異常作動の発生など有害な作用をするので,劣化した潤滑油とともに分離する必要がある.しかし,凝縮成分を含まないガスを圧縮する無給油式では,ドレンが発生しないので,ドレン分離器を設置する必要はない.ドレン分離器としては,ガスとドレン,油分の比重差,粘度差を利用して気液を分離する原理は基本的に同じであるが,構造,構成部品によって,ろ過式,衝突式,膨張式,サイクロン式がある.分離器でのドレン分離捕集率は,形式にもよるが,80~95%である.ろ過式のエレメント材には特殊紙,セラミックス,焼結金属,グラスウールなどが使用され,衝突式では衝突部が積層金網のワイヤメッシュ式,円すい多孔板のエリミネータ式,仕切り板式などがある.d. ガスフィルタ. ガス吸入フィルタ吸入ガス中に含まれるさび,触媒粉,生成物,ミスト等の異物だけでなく,運転当初は下流側より建設時のさび,砂,ごみなども飛んでくるので,これらの異物を捕集するためフィルタを設けるが,吸入ガス中にはガス脈動があるので,構造的に考慮する必要がある.一般には円すい形(cone type)が使用されており,ろ過精度は40~200メッシュ程度である.. 吐出しガスフィルタ吐出しガス中に異物の混入,特に無給油圧縮機のリング,パッキンの摩耗粉を捕集する必要がある場合に設ける.エレメントはセラミックス製が一般的である.e. 脈動緩衝器(pulsation snubber 以上で設けている.脈動防止器には容積形(volume bottle type)とフィルタ形(pulsation filter type)の2種類があり,運転条件に合わせて選択して採用している(図III-2・43).脈動緩衝器の大きさは,形式,運転条件にもよるが,一般にピストン行程容積片側分の10~30倍程図-2・43 直接伝動方式軸継手としてリジットフランジ形の継手をリーマボルトで結合する直結方式が一般的であるが,駆動機がガスエンジンの場合,または高速機およびスキッドにまとめたパッケージ圧縮機の場合には,フレキシブル継手(flexible coupling)を介して圧縮機と駆動機を接続する方式が採用されている.また,中間軸を設ける場合もある.. 減速方式減速装置としては,V ベルト方式と歯車減速方式がある.V ベルト方式は一般に150kW 以下の駆動機に適用しているが,防爆環境では帯電防止ベルトを使用する必要がある.歯車減速装置の場合は,圧縮機クランク軸のたわみと速度変動の影響を避けるためフレキシブル継手を使用するが,十分な勢車効果をもつフライホイールを設け,速度変動を十分小さく抑えて,フレキシブル継手と減速歯車の寿命を安定させる必要がある.フレキシブル継手としては弾性ゴム継手が多く用いられている.. フレキシブル継手一般にギヤカップリング形,SFカップリング形,たわみディスク形,ダイアフラム形,弾性ゴム形などがあり,圧縮機,減速歯車,駆動機のそれぞれの結合に最適なものが使用されるが,回転軸系のねじり固有振動数への影響を配慮して,共振を避ける必要がある.g. ターニング装置圧縮機の開放点検作業時および長期停止中の,圧縮機のクランク部防錆のために行う低速でのクランク軸回転に必要なターニング装置(turning device)には,手動式,電動式,空気作動式があり,大型機は電動式が多く用いられている.
