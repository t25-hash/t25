# 3・4・3 ルブリケータ

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

一般的に，空気圧機器の中で空気圧アクチュエータやバルブの内
部にしゅう動部分が存在しており，これらを円滑に作動させるた
め，さらには耐久性を向上させるため，給油潤滑を行うことが必要
となり，この目的のために用いられる給油器がルブリケータ
（lubricator ）である．ただし，無給油の空気圧機器にはあらかじ
めグリースが封入されているので，必ずしも給油の必要はない．
ルブリケータはその構造と機能により，全量式と選択式とに大別
される．全量式ルブリケータは発生させた油粒の大小にかかわら
ず，その全量を空気中に混入するもの［図Ⅴ-3･39(ａ) ］で，油
粒の大きさは一定ではなく，到達距離も比較的短いなどの欠点はあ
るが，構造が簡単で使いやすいため，一般の空気圧機器の潤滑に多
用されている．また，低流量域にも使用できるように，弾性体など
による可変調整板を用いて，空気流量にほぼ比例した油量を供給で
きる構造のものもある（JIS B8378：2000に性能，構造，試験方
法等が規定されている)．
選択式ルブリケータも霧化の原理は全量式と類似しているが，霧
化した直後に衝突板にあてて油の粗大粒子を除去するため，全量式
のものに比べて送油量は少ないが，油霧の到達距離が極端に長いの
が特徴である［図Ⅴ-3･39(ｂ) ］．
空気圧機器への適正給油量は，一般に空気量10mにつき0.3～
1cmとされている．この油量を確実に給油点まで送るために，ル
ブリケータの設置位置はできるだけ給油点の近くにする必要があ
