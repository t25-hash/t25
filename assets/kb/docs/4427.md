# 2・3・1 据付け,運転および保守

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・3 据付け,運転,振動とその防止法

容積形回転送風機および圧縮機は,往復圧縮機に比べて一般に小型,軽量,高速であって構造も簡単なので,据付け,運転,保守は簡単である.往復圧縮機は,クランク機構によりピストンがシリンダ内で往復運動してガスを圧縮するので,回転部分,往復運動部分により不釣合い慣性力および慣性偶力を生じるうえに,他の形式の送風機,圧縮機に比べて低速,大型,構造が複雑で構成部品が多いため,据付け(installation)および保守管理(maintenance)は計画的に慎重に行う必要がある.ここに往復圧縮機として,特に注意しなければならないおもな事項を掲げる.a. 据付場所および据付け(1) 圧縮機室は清掃,点検,修理などに便利なように,特にピストンおよびピストンロッド,クランクまわり部品,駆動機,伝動装置,ガス冷却器の管束などの引出し場所,設置場所を考慮して圧縮機の周囲に十分な余地を確保する.また,往復圧縮機は振動,騒音を伴うから,周囲環境に対する影響も考慮しなければならない.(2) 圧縮機および付帯設備の据付け,点検作業のために必要なクレーンなどのつり上げ設備について,能力,形式などについて事前に計画をしなければならない.(3) 往復圧縮機は運転中に回転,往復運動に対応する繰返し荷重を受けるので,各部のボルトナットは規定トルク,要領に従って十分に締め,特に内部の運動部分に対しては必ず設計どおり,回り止めを施さなければならない.(4) 圧縮機の振動が圧縮機室建屋へ伝わり共振などの影響を及ぼさないように,圧縮機本体基礎と建屋,架構基礎とは分離させて,接続しないよう配慮したほうがよい.(5) ガス配管の固有振動数と吸込・吐出し回数が一致して共振を起こさないように管の長さ,配管サポートの位置を決め,併せて配管の熱膨張,熱収縮により配管に異常応力が発生しないよう配管の配置をしなければならない.必要に応じて熱応力解析,振動解析,脈動解析を行い,問題の発生を未然に防止することが望ましい.(6) 圧縮機の基礎ボルトはテンプレートなどの使用による先埋め方式か,箱抜きによる後埋め方式か,圧縮機の大きさ,形式,設置条件などを考慮して決めなければならない.b. 運転(1) 始動に先立ちターニング装置(turning device)で圧縮機のターニングを行い,しゅう動部をなじませると同時に,回転が滑らかに回り,異音などの異常がないことを確認する.(2) 起動前に,冷却水通水,潤滑油液面,給油圧力,給油温度,圧縮機無負荷状態,バイパス弁開またはブロー弁開,ガス吸入ストップ弁開,ガス吐出しストップ弁開,ターニング装置離脱などの,圧縮機の起動条件が成立していることを確認する.(3) 圧縮機起動後,規定回転になってから,圧縮機負荷を上げながらバイパス弁またはブロー弁を徐々に閉じて昇圧する.バイパス弁およびブロー弁を閉止状態で起動した場合は,圧縮機負荷を容量調整により徐々に上げるが,昇圧速度は出口系統の容積で成り行きとなり,一般に昇圧速度は速いので注意を要する.(4) 運転状態の診断は主として圧力,温度,音響,振動,動力,容量などの変化で行うので,正常の運転状態を確認しておく必要がある.通常,圧縮機の使用状態に合わせて圧縮機運転状態の異常を検知し,警報表示および自動停止をする保護装置が設けられているが,すべての異常を検知することはできないので,定期的な運転状態の確認が必要である.(5) シリンダ周囲の異常音の原因は,シリンダ弁の破損,リングおよびロッドパッキンの摩耗または破損,シリンダ内の水,ドレン,その他の異物の侵入などによる場合があり,フレーム周囲では軸受の摩耗,締付けボルト,ナットの緩みなどがあるが,運動部品の締付けナットの緩みなどは比較的静かなクランク室,クロスガイド筒部に反響して大きく聞こえる場合があるので,異常音の判定には注意が必要である.(6) 軸受関係の異常確認には,フレーム給油装置の油フィルタエレメントの開放点検が有効である.また,給油式圧縮機ではドレン分離器の採取ドレンの確認によっても,リングの異常摩耗の早期確認が有効である.c. 保守(1) 往復圧縮機はクランク機構の各種軸受,リングおよびパッキンなどしゅう動し,摩耗する部品が多く,吸入弁,吐出し弁のように繰返し衝撃荷重を受けるものもある.取扱いガスによっては腐食,摩耗も伴うので,あらかじめ必要な予備品を整備しておき,必要時に対応できる体制にして稼働率を高める必要がある.(2) 日常点検,毎月,半年,1年,2年ごとの定期点検の検査部品および点検項目を決め,各部品の摩耗量,交換寿命について整備基準を作成しておき,摩耗程度が規定値以上になったら,また寿命時間に達したら定期的に部品を交換する.また,定期点検時の寸法記録,非破壊検査記録,補修状況,次回交換予定部品リストなどを経歴書として残しておけば管理上有効である.
