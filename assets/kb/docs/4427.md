# 3・4・4 消音器（muﬄer ，silencer ）

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

空気圧システムにおいて作動流体に関連する騒音発生源として，
次のようなものが挙げられる．
(１)圧縮機の吸込口から大気を吸い込む際に生じる気流音や，吐
出し時に生じる空気圧力の脈動によって発生する騒音．
(２)切換弁や空気圧モータ駆動の工具などから，空気が排気され
るときに生じる気流音・衝撃音．
これらの騒音を減衰し，より良い作業環境を得るために消音器
（サイレンサ）が用いられる．消音器に求められる機能条件として
は，
(ａ)インサート部があるもの
(ｂ)インサート部がないもの
図 -3･42 締め込み式継手
(１)流量抵抗が小さく，切換弁などの切換性能やアクチュエータ
の性能に影響を与えない．
(２)消音効果が大きい．
(３)長時間の使用に対して目詰まりなどの抵抗変化が少ない．
(４)電磁弁などに取付けやすいコンパクトな形状である．
(５)排気の衝撃や振動などで変化を起こさない．
などを考慮しておくことが必要である．
構造の違いにより消音器は，リアクタンス型（膨張型，共鳴型，
挿入管型の総称）と吸音型との2種類に大きく分類される．リアク
タンス型消音器（reactive muﬄer ）は圧縮機の吸・排気騒音防止
に多く使用され，通気抵抗が小さいことが特徴であるが，その形
状，寸法により，特定の周波数の騒音に対して選択的な減衰効果を
もつ特質がある．一方，吸音型消音器は，焼結金属などの通気性多
孔物質およびグラスウール，不織布，メタルウールなどの吸音材を
空気通路に充てんして振動エネルギーを吸収するもので，高周波に
対する吸音率が高く小型化できるが，通気抵抗が比較的大きくなっ
てしまう．一般に空気圧機器用消音器として市販されているもの
は，排気管の騒音減衰効果を得るために，リアクタンス型と吸音型
の構造を併用したものが多い（図Ⅴ-3･40) ．
