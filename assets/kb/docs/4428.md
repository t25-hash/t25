# 2・3・2 振動とその防止法

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・3 据付け、運転、振動とその防止法

容積形回転送風機および圧縮機はターボ形に比べて回転速度が小さいので、回転部のアンバランスを十分に除去しておけば振動(vibration)は小さくなる。往復慣性力(reciprocating inertiaforce)による加振力が大きく、付属配管、機器の影響を大きく受ける往復圧縮機について、以下に述べる。a. 圧縮機の振動圧縮機本体が引き起こす振動の原因には次のようなものがある。. 往復質量による慣性力圧縮機加振力の主因となるもので、クランク軸を通じて主軸受からフレームを介して基礎あるいは床に作用する起振力で、フレーム本体の振動、基礎振動に大きく影響を与える。単シリンダ分について、クランク軸のコネクティングロッド連結部に発生するピストン運動方向の往復慣性力(kN)は、四次以降の成分を省略し、近似的に次式により求められる(図III-2・47)。. トルク変動往復慣性力およびガス圧によるピストン力が合成されて、クランク軸系に変動トルクとしてかかり、回転角速度の変動をもたらす。変動トルク周波数とクランク軸系固有振動数が一致してねじり振動の共振を起こさないよう配慮が必要である。圧縮機と駆動機が直結される場合はほとんど問題ないが、高速機の場合、歯車減速装置を使用する場合、回転数変動する場合などの使用条件では特に注意を要する。. ピストン力ピストンの往復運動によるガス圧縮によってフレーム、クロスガイド筒、中間筒およびシリンダは、反フレーム側軸方向にシリンダ内圧縮ガス圧力により回転ごとに伸びる。この伸びはシリンダの振動として、先端でピストン行程の1000分の1程度あるので、シリンダサポートを設ける場合は構造的に配慮が必要である。また、クロスヘッド部において回転運動が往復運動に変わるが、圧縮ガス圧により発生するピストン力(kN)のクロスヘッド運動と直角方向に分力のクロスヘッド荷重(kN)が、1回転当たり2回の周期で起振力として作用する。. 圧縮機の機械的異常クランク機構の軸受、ロッドパッキン、ピストンなどしゅう動部の締付け部緩みによるがたつきまたは異常摩耗、また、シリンダ弁、ピストンリング異常が原因のガス圧変動によるピストン力アンバランス増大、シリンダ内の滞留ドレンによる異常圧縮、さらに、レベル、心出し、締付けなどに据付不良がある場合の機械異常がある。. 振動の主原因を除去する圧縮機運転により生じる不釣合い慣性力、慣性偶力ができるだけ小さくなるよう設計する。特に往復慣性力、回転慣性力に対しては、クランク軸に釣合いおもり(balance weight)を付加したり、クランク軸のアームの形状を工夫して不釣合い質量を減らすようにする。また、クランク角およびシリンダ配置を考慮し、例えば対向形、半星形にするなどして慣性力を互いに打ち消すようにして起振力を小さくする。特に高速機の場合、クランク軸は回転バランスをとる設計をし、大きな不釣合い回転慣性力をなくすようにすべきである。. 圧縮機と基礎との共振を避ける圧縮機基礎の振動を抑えるため、必要に応じて土壌データから基礎の振動解析を行い、圧縮機と基礎の共振を避ける。そのために、基礎の固有振動数を圧縮機加振周波数の2倍以上にするのが望ましい。基礎は地盤に注意して強固な場所を選び、弱い所では地固め、必要に応じて杭の打設、地盤改良を行い、基礎コンクリートは鉄筋にて十分に補強する。なお、圧縮機本体基礎の振動の大きさは機械の形式、回転速度により異なるが、機械自身の耐振性だけでなく、不快な有感振動の面も配慮して中・大型横形圧縮機を除き、片振幅で0.01~0.03mm程度に抑えるべきである。. 圧縮機の振動を絶縁する圧縮機とその受け台(基礎あるいは床)との間に、防振材料として金属ばね、防振ゴム、コルク、フェルトなどの弾性体を挟む。この方法は6~10Hz 以上の高加振周波数の場合に有効である。. 圧縮機への外力を減らす圧縮機接続配管の低温、高温図-2・50 垂直面回転偶力計算結果例による収縮、伸びを十分吸収するように配管設計を行い、圧縮機への荷重を小さくする必要がある。また、シリンダのガス吸込口および吐出し口に接続する脈動緩衝タンクは、内部のガス脈動により生じる軸方向の振動荷重(shaking force)が極力小さくなるように設計すると同時に、水平シリンダの場合、上部に接続する吸入脈動緩衝タンクはシリンダの横振れを防ぐため、シリンダ軸と同じ軸方向に設置するのが有効である(図III-2・52)。
