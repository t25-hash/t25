# 2・3・3 圧縮機管路の共振とその防止法

> **章**: 第2章 容積形送風機および圧縮機
> **節**: 2・3 据付け、運転、振動とその防止法

a. 圧縮機管路の共振往復圧縮機はガスを間欠的に吸込み、あるいは吐出し、その圧縮機回転周期の整数倍に存在する周波数が形状、寸法によって決まる管路ガスの固有振動数に一致すると、管路ガスが共振状態となる。共振状態においては、配管に対する起振力が異常に増加し、配管系およびその付属機器に疲労破壊をもたらしたり、シリンダ弁室内の圧力変動が増えて吐出しガス容量、所要動力が増減し、性能が不安定化したり、あるいは吸入弁、吐出し弁の弁スプリング、弁板の作動を不安定化し、寿命を短くするなど種々の悪影響を及ぼす(図III-2・53)。圧縮機につながる配管系は、吸入弁、吐出し弁、バイパス弁、ストップ弁などの閉端および吐出しレシーバタンクなどの大容積部、大気吸込口の開端、または吸入ドラムなどの大容積部によって、例えば、ガス吸入管路、ガス吐出し管路、あるいはシリンダ吐出し弁室から次段吸入弁室までの中間段管路というように、いくつかのガス管路に区切られて、圧力脈動(pressure 配管の固有振動数との共振ガス管路の圧力脈動は、理論的に圧縮機の吸入み・吐出しの周波数욂の整数倍の成分にて構成されているが、一般に一次、二次が主成分である。しかし、ガス管路によっては、四次~八次の高次成分が大きく発生する場合もあり、特に音速の速い水素などはその傾向にある。それに対し、大型の貯蔵タンク、ガスタンク、ノックアウトドラムまわり、および配管架構まわりの配管サポートの設置が少ない配管の固有振動数は、5~10Hz と圧縮機の第一次波数욂と同じ領域である。また、圧縮機まわりの配管サポートが多く、個々の配管が比較的短い配管の固有振動数は15~45Hz 程度である。したがって、配管設計では配管剛性を上げる配管サポートシステムの検討とともに、ガス管路の圧力脈動と配管との共振を避けるように、また、ガス管路の圧力脈動を減衰させて、配管への起振力を問題ない程度まで小さくする配慮が必要である(表III-2・12)。d.
