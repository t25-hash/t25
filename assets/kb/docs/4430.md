# 3・1・1 特徴と適用範囲

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・1 遠心・斜流送風機および圧縮機の設計

ターボ形送風機(blower)および圧縮機(compressor)は,容積形に比べて小型・軽量で振動が少ないこと,摩耗部分が少なくてメンテナンスが容易であること,油の混入がないことなどの利点を生かし,図III-3・1に示されるように低圧力のものから高圧力のものまで,また広い流量範囲にわたって利用されている.ターボ形の中でも遠心式(centrifugal type)および斜流式(diagonal flow type)は,軸流式に比べて1段当たりの圧力比を高くとることが容易で(第I部4・1節参照),少ない段数で所定の圧力比を達成することができる.ガスタービンやターボ冷凍機用では単段の圧力比が4を超える遠心圧縮機も使用されている.また,遠心式は軸流式に比べて羽根車の剛性が高いので高圧分野(最高圧力60MPa)にも適用される.流量に関しては,軸流式は羽根車入口径と出口径がほぼ等しいのに比べ,遠心式は羽根車外径に対して入口径が小さくなるので大きさが同じ場合には軸流式に比べて小流量のものに適している.したがって,軸流式と同じ流量を処理する遠心式流体機械は機械の外径が大きくなる.安定作動(流量)範囲は,遠心式が軸流式に比べて広く,流量圧力特性曲線のこう配(流量変化に対する圧力変化)が小さいため容量制御が比較的容易である.特に圧縮機においては,軸流式ではサージ限界が設計流量に近いが,遠心式では軸流式に比べて作動範囲の広いものが得られる.最高効率に関しては,遠心式は流路形状が複雑で長いため,軸流式に比べて数%~10%程度低い.しかしながら,軸流式では設計点から離れると効率が急激に低下するのに対して,遠心式では比較的広い範囲で高い効率を維持する.図-3・1 送風機および圧縮機の適用範囲2―163遠心式の各機種で最高効率を与える比速度(specific speed)욦(m, m/min, min)はファンでは後向き羽根,径向き羽根,前向き羽根の場合,それぞれ400,300,500(無次元比速度:0.98,0.73,1.22)程度である(図III-3・2).圧縮機では圧力比にもよるが,300~400で最高効率が得られる.斜流式は遠心式に近い形状から軸流式に近い形状まであり,製作実績は遠心式ほど多くないが,径向きファンで800(1.95)程度,ブロワや圧縮機で500程度で高効率のものが得られており,遠心式では高効率が期待できない高比速度領域で使用するのに好都合である.
