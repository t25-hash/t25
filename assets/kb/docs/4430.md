# 3・1・1 特徴と適用範囲

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・1 遠心・斜流送風機および圧縮機の設計

ターボ形送風機（blower）および圧縮機（compressor）は，容
積形に比べて小型・軽量で振動が少ないこと，摩耗部分が少なくて
メンテナンスが容易であること，油の混入がないことなどの利点を
生かし，図Ⅲ-3･1に示されるように低圧力のものから高圧力のも
のまで，また広い流量範囲にわたって利用されている．
ターボ形の中でも遠心式（centrifugal type）および斜流式
（diagonal flow type）は，軸流式に比べて1段当たりの圧力比を
高くとることが容易で（第Ⅰ部4･1節参照），少ない段数で所定の
圧力比を達成することができる．ガスタービンやターボ冷凍機用で
は単段の圧力比が4を超える遠心圧縮機も使用されている．また，
遠心式は軸流式に比べて羽根車の剛性が高いので高圧分野（最高圧
力60MPa）にも適用される．
流量に関しては，軸流式は羽根車入口径と出口径がほぼ等しいの
に比べ，遠心式は羽根車外径に対して入口径が小さくなるので大き
さが同じ場合には軸流式に比べて小流量のものに適している．した
がって，軸流式と同じ流量を処理する遠心式流体機械は機械の外径
が大きくなる．
安定作動（流量）範囲は，遠心式が軸流式に比べて広く，流量圧
力特性曲線のこう配（流量変化に対する圧力変化）が小さいため容
量制御が比較的容易である．特に圧縮機においては，軸流式ではサ
ージ限界が設計流量に近いが，遠心式では軸流式に比べて作動範囲
の広いものが得られる．
最高効率に関しては，遠心式は流路形状が複雑で長いため，軸流
式に比べて数%～10 %程度低い．しかしながら，軸流式では設計点
から離れると効率が急激に低下するのに対して，遠心式では比較的
広い範囲で高い効率を維持する．

3・1・2全体設計
a.機種の選定 プラントや装置の必要な気体の流量，圧力
比，管路系の圧力損失などから機械の仕様が決定される．機種は設
計仕様から図Ⅲ-3･3などを参考に決められる．機種の選定に際し
ては，流量圧力特性，入口圧力，経済性（製造コストおよび運転コ
スト），大きさの制限，保守性等を考慮して決めるが，各項目の重
要度は用途により異なる．適用範囲内に複数の機種がある場合（例
えば遠心両吸込みと斜流，高速単段と低速多段など）には上記項目
を考慮して最も適した形式を選定する．

c .原動機の選定 設計仕様から算定した理論ガス動力と推定
した効率から羽根車の駆動動力が計算される．さらに，変速機や軸
受等の機械損失を考慮して軸動力を求める．遠心および斜流機械の
軸動力は一般に流量が変わると変化するので，定格点だけではなく
ある流量範囲にわたって使用される場合には，その流量範囲の最大
軸動力に対して対して1.05～1.10の余裕をとって原動機を選定す
る．
d.流量制御（flow control) 機械のガス流量が大幅に変化す
る場合には，台数制御が有利である．機械の故障の際の危険分散に
もつながる．ガス流量の変化があまり大きくない場合には，安定作
動範囲が広いという遠心機械の利点を生かして，入口絞り，出口絞
り，入口案内羽根，羽根付きディフューザ等により流量を制御でき
る．特に，入口案内羽根制御は羽根車入口部で流れに予旋回を与え
るので理論仕事が減少することから，絞り制御に比べて軸動力が減
少して省エネルギーになる．入口案内羽根と羽根付きディフューザ
を同時に動かす制御法はさらに省エネルギーになる．
このほかに，可変速原動機，トルクコンバータを用いた回転速度
変更による流量制御も行われる．回転速度制御を行う場合には，軸
系のたわみおよびねじりの固有振動数との共振を避けるよう注意し
なければならない．
