# 3・1・2 全体設計

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・1 遠心・斜流送風機および圧縮機の設計

a. 機種の選定プラントや装置の必要な気体の流量、圧力比、管路系の圧力損失などから機械の仕様が決定される。機種は設計仕様から図III-3・3などを参考に決められる。機種の選定に際しては、流量圧力特性、入口圧力、経済性(製造コストおよび運転コスト)、大きさの制限、保守性等を考慮して決めるが、各項目の重要度は用途により異なる。適用範囲内に複数の機種がある場合(例えば遠心両吸込みと斜流、高速単段と低速多段など)には上記項目を考慮して最も適した形式を選定する。b. 使用材料の選定周速度の高い圧縮機では強度的に余裕がとれないので、機械の振動、材料の疲労等も正確に評価して羽根車図-3・2 遠心ファンの形式と全断熱効率の関係送風機名称圧力上昇圧縮機ファンブロワ10kPa未満10kPa以上100kPa未満100kPa以上種別軸流式軸流多翼ターボ形遠心式ラジアル遠心二葉回転式ベーン容積形ねじ往復式往復図-3・3 送風機、圧縮機の分類[入口圧力が大気圧(100kPa)の場合]2―164の使用材料を選択しなければならない。腐食性のガスや摩耗性のある物質を含んだガスを取扱う場合には使用材料を十分に吟味しなければならない。腐食、摩耗の激しい部分には保護、交換などの対策を講じることも考える。c. 原動機の選定設計仕様から算定した理論ガス動力と推定した効率から羽根車の駆動動力が計算される。さらに、変速機や軸受等の機械損失を考慮して軸動力を求める。遠心および斜流機械の軸動力は一般に流量が変わると変化するので、定格点だけではなくある流量範囲にわたって使用される場合には、その流量範囲の最大軸動力に対して対して1.05~1.10の余裕をとって原動機を選定する。d. 流量制御(flow control)機械のガス流量が大幅に変化する場合には、台数制御が有利である。機械の故障の際の危険分散にもつながる。ガス流量の変化があまり大きくない場合には、安定作動範囲が広いという遠心機械の利点を生かして、入口絞り、出口絞り、入口案内羽根、羽根付きディフューザ等により流量を制御できる。特に、入口案内羽根制御は羽根車入口部で流れに予旋回を与えるので理論仕事が減少することから、絞り制御に比べて軸動力が減少して省エネルギーになる。入口案内羽根と羽根付きディフューザを同時に動かす制御法はさらに省エネルギーになる。このほかに、可変速原動機、トルクコンバータを用いた回転速度変更による流量制御も行われる。回転速度制御を行う場合には、軸系のたわみおよびねじりの固有振動数との共振を避けるよう注意しなければならない。
