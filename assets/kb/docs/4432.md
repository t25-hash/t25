# 3・1・3 羽根車(impeller)の設計

単段遠心流体機械の設計過程べるとまだ完成されているとはいえない.すなわち,軸流の翼列資料のような設計基準となるものがないために,経験を参考とすることが多い.遠心羽根車には,後向き羽根(β욁<90°,β욁:接線方向から測った出口羽根角度),径向き羽根(β욁=90°)および前向き羽根(β욁>90°)がある.これらの特徴については,第I部4・3節および3・2・1~3・2・3項を参照されたい.また,羽根車としては,二次元形状の羽根を有する二次元羽根車と,三次元形状の羽根を有する三次元羽根車があり,三次元羽根車は子午面流路形状により遠心式と斜流式に分けられる.a. 段数および回転速度の選定全断熱ヘッドを単段の達成可能な断熱ヘッド(周速度により決まる)で割ることにより求める.回転速度は,原動機の回転速度から定める.あるいは,回転速度が任意に選べ,効率が最優先される場合には効率が最高になる比速度から決定する.小型化が必要な場合には,高回転速度が選定される.b. 羽根枚数の選定羽根枚数は,β욁とともにすべり係数μ(第I部4・3・1項参照)に影響を及ぼす重要なファクタである.羽根枚数が少ないとすべりが大きくなり,圧力係数が低下することになる.羽根枚数が大きくなると流路のぬれ縁面積が増加して摩擦損失が増大し,効率が低下することになる.圧縮機の最適羽根枚数に関しては文献(2)が参考になる.(4) 入口形状の決定羽根車の入口諸元は,制約条件の中で入口相対速度ができるだけ小さくなるように選定することが好ましい.羽根車入口部の相対速度を下げることにより,羽根車内の相対速度レベルが低下して羽根車の圧力損失が減少することになる.三次元羽根車の場合には,設計仕様(回転速度,質量流量,入口温度,入口圧力)と羽根枚数,心板側の入口径が与えられると,側板側相対速度욦が最小になる側板側入口径욦が存在する.入口羽根角度β욁욦は,通常は入射角が0°になる(入口相対流れと一致する)ように決められる.なお,욦が最小となるときのβ욁욦は,30°~33°になる.(5) 出口流路幅羽根車性能に重要な影響を及ぼすパラメータとして相対速度の減速比(diffusion ratio)がある.減速を大きくすると大きな圧力上昇が得られるが,失速が生じやすくなりサージマージンが小さくなる傾向がある.逆に,減速を小さくするとサージマージンは大きくなるが,圧力上昇の低下や摩擦損失の増加により効率低下を招くことになる.三次元羽根車では,減速比としては減速が最大となる側板側の平均相対速度比욦が選ばれる.욦=1.5~1.6がインデューサ付き羽根車の限界減速比といわれている.出口流路幅は욦が所定の値になるよう決定する.なお,出口速度三角形を決める際に羽根車効率η욡욇욣が必要であるが,η욡욇욣は経験値を与える場合と,性能予測法により求めた値を使用する方法がある.このほかに設計で考慮すべきパラメータとして,周速マッハ数(Mach number)(:羽根車入口よどみ状態の音速)と2―165効率の関係,インデューサ先端相対マッハ数욥욦(=욦,:羽根車入口音速)の許容限界値等がある.羽根車諸元が決まると,性能予測法により機械の性能(ヘッド,効率等)を予測し,所定のヘッドを満足しない場合には,諸元を変更し仕様を満足するまで計算を繰返す.性能予測法に関しては文献(10)が参考になる.c.
