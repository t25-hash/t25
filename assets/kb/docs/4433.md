# 3・1・4 ディフューザの設計

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・1 遠心・斜流送風機および圧縮機の設計

ディフューザは,羽根なしディフューザと羽根付きディフューザに分類される.a. 羽根なしディフューザ(vaneless diffuser)平行壁羽根なしディフューザでは,流れが非圧縮で損失がないと仮定すると,連続の式と角運動量保存則から,ディフューザ内で流れ角は一定つまり流線は対数らせんとなる.しかしながら,実際には流れは圧縮性があり損失もあるので,ディフューザ内で流れ角は一定ではない.圧縮性が強い場合には,密度が下流に向かって大きくなるので流れ角は小さくなる.圧縮性が弱い場合には損失により,角運動量が減少するため流れ角は下流に向って大きくなる.羽根なしディフューザは,構造が簡単であるためよく使用される.しかしながら,一般に流路長さが長くなるため,摩擦損失が大きくなり圧縮機の効率は低くなる.また,流量の減少とともに流入角が減少し流路長さが増加するため,損失係数は増加して静圧回復率は減少する,つまり右上がりの流量-圧力(静圧回復率)特性を示す.したがって,出口羽根角度が大きい羽根車と組合わせると設計点付近で圧力係数が最大になりサージが発生しやすい.低比速度段では一般に流路幅が狭く,ディフューザの流入角が小さくて流路長さが大きくなるので,やはり圧縮機効率が低下することになる.さらに,羽根なしディフューザでは流入角が小さい場合には壁面近くで逆流が発生し,旋回失速の原因となることがあるので注意が必要である.また,高比速度羽根車と組合わせると羽根車出口の幅方向のひずみが大きいため,ディフューザの静圧回復率は低下する.2―166流体機械斜流羽根車出口の流れを円すい面の羽根なしディフューザに導くと,速度の遅い流れは内側の円すい面に集積し,圧力回復率は低い.したがって,短い区間で流路を半径方向に曲げる半径流移行形ディフューザを使用するほうが性能は高くなる.b. 羽根付きディフューザ(vaned diffuser)半径の羽根車の羽根通路間のジェットウェーク(jet-wake)の混合は半径が1.2~1.3を超えなければ完了しないから,案内羽根を羽根車に接近して設ければ激しい騒音が発生する.しかし,摩擦損失を少なくするために案内羽根先端の位置を1.05~1.1とすることが多い.またディフューザ入口の通路幅は羽根車出口幅の1.0~1.1倍に選ばれる.ディフューザ設計上重要なのは,隣接する羽根間に生じるスロートの寸法である.案内羽根の先端からスロートまでの区間は半ば開放されており,羽根車からの流出角に応じてこの区間での圧力上昇率が変化する.また,境界層によるスロートの閉そく率は,案内羽根前縁からスロートまでの圧力上昇率が入口動圧の0,0.2,0.3の場合にそれぞれ8%,13%,20%程度でありディフューザ失速限界での圧力上昇率は0.4である.スロートでの閉そく率が大きいほど下流の広がり通路の圧力回復率が低くなるので,ディフューザ全体として圧力回復率が最大となるスロート面積がある.また,ディフューザ失速限界流量やチョーク流量または最大流量もスロート面積によって左右される.案内羽根の取付け角,負圧面形状もスロート面積や閉そく率を介して圧力回復率に影響する.スロートの下流は広がり通路である.直線広がり通路に関してはアスペクト比やマッハ数の影響を含む多くの資料があり,広がり角としては普通7°~10°が用いられる.ディフューザの通路数が個の場合,隣接する通路相互の角は360°/である.したがって,直線広がり通路で個数が少ない場合には通路間に,くさび(楔)状の島(図II-3・21参照)を設けて各通路の広がり角が7°~10°になるように調節する.スロート下流の流路をわん曲させれば薄い羽根でも通路断面積の広がり率を調節できるが,側壁に沿う二次流れが生じて凸面の境界層が厚くなるので,そのはく離防止の配慮が必要である.ディフューザによって一定の出入口面積比を得るには,通路数が多いほど全体形状を小さくできる.一方,通路数が少なければ羽根先端の厚みに基づく増速量が少ないばかりでなく,失速限界流量から最大流量までの間の作動流量範囲がやや広くなる傾向がある.これは,羽根数が少ないほど羽根先端のくさび角が大きいので失速流入角を定める羽根負圧面が周方向となす角が小さく,一方,圧力面が周方向となす角が大きいために最大流量が大きくなるためである.ディフューザの通路断面を円形にしたパイプディフューザでは,円すい形あるいは円筒形の隣接する2本の通路の間に削り残されてできた案内羽根の先端は,壁面近くでは羽根車に近くて円周方向となす角が小さい.また通路中央では,羽根車から離れているので周方向に対してかなり立っていて,羽根車から吐出されるひずんだ流れの方向の分布に似ているので,高い圧力回復率が得られる.減速翼列を羽根なしディフューザ内に設ければ,出入口半径の変化に基づく減速効果が加わるのでかなりの圧力回復が得られる.弦節比を1より小さくしてスロートをもたないようにすると,流れは翼によってあまり拘束されず,亜音速域,遷音速域いずれでも広い流量範囲で効率の良い運転が可能である.また,円形翼列を2列あるいは3列重ねたディフューザの運転可能な流量範囲は通路形ディフューザよりもやや広いものが多い.なお,円形翼列はアスペクト比が甚だ小さいにもかかわらず,その性能は二次元翼列性能からほぼ正しく推定できる.遷音速の翼列ディフューザには,薄い二重円弧翼形が用いられる.超音速域に通路形ディフューザを設けると,その前縁付近に衝撃波が生じ,ディフューザ性能はマッハ数の増加とともに連続的に低下するが,マッハ数が1.3でも入口動圧の0.6倍程度の圧力回復が可能である.
