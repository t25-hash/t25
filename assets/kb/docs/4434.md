# 3・1・5 旋回失速およびサージング

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・1 遠心・斜流送風機および圧縮機の設計

a. 旋回失速(rotating stall)羽根車もディフューザも,羽根に対する入射角が増大して流れが羽根面からはがれる限界に達すると,円周上に数個の失速域ができる.失速域は羽根通路間を隣接する通路の負圧面の方向に伝ぱしていく.羽根車の失速域は相対的に回転と逆方向に伝ぱするので,静止側からは遅い速度で回転するように観測される.ディフューザの失速域は羽根車の回転と同方向に遅い速度で伝ぱする.いずれの場合も圧縮機特性の変化はわずかであるので旋回失速に気付かないことが多い.しかし,オープン羽根車では失速域の通過に共振してインデューサが折損するおそれがある.また,高密度の気体を取扱う場合には旋回失速に基づく圧力変動が大きく,軸振動の原因となる.弦節比を1より小さくして,スロートをもたないようにした円形翼列ディフューザの場合,サージ限界流量でディフューザ翼列には旋回失速の発生が見られなかったとの報告がある.このように弦節比を1より小さくする設計でディフューザの旋回失速を抑制できる場合がある.なおa項で述べられているように,旋回失速は羽根なしディフューザでも発生するので,羽根を除くのみでは旋回失速を抑制するのは困難である.b. サージング(surging)サージングは激しい圧力変動と騒音を生じ,流れの脈動を伴う圧縮機系全体の不安定現象である.系の振動には管路系の長さや容積も関係するから,圧縮機をある管路系で試験したときに安定に運転できた流量範囲でも,その圧縮機を別の管路系に取付けた場合に安定に運転できるとは限らない.しかし,サージングには圧縮機系の圧力比-流量特性曲線のこう配が正であることが必要であるから,こう配が正の最大流量値がサージ限界流量とみなされる.圧縮機の理論ヘッドの流量に対する負のこう配は羽根車出口における羽根後傾角に比例するから,後傾角の大きいものほど運転可能な範囲は小流量側にある.一方,流量が設計流量よりも減少すると羽根車もディフューザも圧力損失が増大して特性曲線のこう配を正にする傾向があるが,特にある限界値以下の流量で流れが羽根からはがれると圧力上昇量が不連続に低下して,サージングの原因になることが多い.高圧力比に設計された圧縮機は,回転速度によって羽根車出入口間の密度比が変化するので,低速回転では羽根車のインデューサが失速状態でも安定な特性をもっていることがある.また,高速回転ではディフューザが失速状態でありながら安定な特性をもっていることもあり,インデューサとディフューザがともに失速して,初めてサージングに入るといわれる.これは,高圧力比用の羽根車は低速回転ではインデューサが失速しても性能の低下がほとんどないことと関連している.一方,羽根車の失速限界を定める羽根車出入口相対速度比は羽根車の通路形状や長さによって異なり高圧力比のものほど小さい.高圧力比に設計された圧縮機を低速回転から高速回転まで連続して使用する場合は,チョークと失速がともに発生しにくいディフューザを用いると有利になる.したがって,スロートをもたないように弦節比を小さくした円形翼列ディフューザを用いると,スロートをもつ羽根付きディフューザを用いた場合に比較して広い作動範囲と高い効率を実現できる.
