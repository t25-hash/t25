# 3・5・6 シーケンス回路（sequence circuit )

> **章**: 第3章 空気圧システム
> **節**: 3・5 空気圧回路

a.機械-空気圧方式 シーケンス回路はシリンダをあらかじ回路を実現するには，機械-空気圧方式と電気-空気圧方式がある．
弁の駆動も空気圧で行う．
弁，シャトル弁を組合わせればAND，FF回路等も構成できる．
気的なものに置換わったもので，制御にはリレーやプログラマブル化機械では多数のシリンダを駆動する必要があるが，シリアル通信も利用されている．
現在，その利便性から電気-空気圧方式のシーケンス回路が大部分を占めており，機械-空気圧方式のものは防爆め定められた順序，または条件により駆動させる回路である．
機械-空気圧方式ではリミットスイッチの信号伝達，制御，切換に電気を使用せずとも切換えられるタイプのもので，これらの切換(ａ)非可動形ピストン空油変換器の構造(ｂ) 速度制御回路b.電気-空気圧方式 電気-空気圧方式は信号と制御部分が電コントローラが，切換弁の駆動にはソレノイドが用いられる．
自動により1本の配線で駆動する，図Ⅴ-3･60 に示す省配線システムができる．
