# 3・2・4 遠心ブロワ(centrifugal blower)

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・2 遠心送風機および圧縮機各論

ファンに比べ、ブロワの羽根車は高速回転に適するように設計されている。羽根車の外周速度が増すに従って応力が増大するため、それに耐えられるように主板・側板・羽根の形状、材料、制作方法が考慮される。形状は高速になるに従って主板・側板の内周側の厚図-3・11 片吸込形ファン断面図図-3・12 両吸込形ファン断面図みが増し、最も高速になると側板のない形状となる(図III-3・14)。a. 単段形ブロワ1個の羽根車で所定の要項を満足するもので、片吸込形と両吸込形とがある。片吸込形は片持ち構造とする場合が多く、軸貫通部が1箇所となるため、複雑な軸封装置を必要とする危険ガスを取扱う用途に有利である。両吸込形は片吸込形ではガス流量が足りないときや片吸込形では圧力による軸スラストが大きい場合などに採用される。高速で回転させるため、一般に歯車装置で増速する駆動方式をとることが多い。この場合、ブロワの羽根車を高速回転側の歯車軸に直接取付けたものと、ブロワ自体が軸と軸受を単独にもち、カップリングを介して電動機直結または歯車増速運転される構造(図III-3・15)のものとがある。b. 多段形ブロワブロワの圧力が高くて羽根車の強度上単段形では所定の圧力を出すことができない場合や、比較的圧力は低くても圧力に比べガス流量が少なく(いわゆる比速度が小さい)、羽根出口幅が狭くなり効率が悪くなったり製作限界を超えるときには、一軸上に数個の羽根車を取付けた多段形が使われる。片吸込形は図III-3・16のように、片吸込形羽根車を低圧側から高圧側に向かって順次一軸上に取付けたもので、圧力による軸スラストを低減す図-3・13 後向き羽根ファンの特性曲線図-3・14 ブロワ羽根車の形状図-3・15 単段形ブロワ断面図図-3・16 多段形ブロワ断面図図-3・17 産業用斜流送風機るために釣合いピストンを吐出し側に設ける。両吸込形は左右両側に吸込口をもち、片吸込形羽根車を両端から中央の両吸込形羽根車に向かって対称に取付け、中央で両側のガスをまとめて吐出すもので、正常運転の場合、軸スラストは発生しない。ケーシングは水平割り形のものと輪切形(各段を同形の軸垂直割り形のケーシングに分割し、これらを組立ボルトでつなぐ構造)のものとがある。空気以外のガス、特に危険ガスを取扱うものでは、軸貫通部の気密を保つため特殊な軸封装置が設けられる。
