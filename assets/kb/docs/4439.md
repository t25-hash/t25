b.ディジタル制御弁 オンオフ電磁弁をパルス変調駆動
（pulse modulation drive）することにより，アナログ制御弁と等
価な機能が得られる．これまでに，パルス幅変調（PWM:pulse
width modulation）方式と，パルス符号変調（PCM:pulse code
modulation）方式を用いたディジタル制御弁が提案されている．
図Ⅴ-3･66はこれらの動作原理を示す．PWM方式は，オンオフ
弁を一定の搬送周期 で駆動し，パルス幅 を制御信号に応じ
て変化させる方式である．図Ⅴ-3･66(ａ)に示すように，弁の平均
的な開口面積 は駆動信号のデューティ比 （＝ ）に比例
する．オンオフ弁の動作遅れが大きい場合には，デューティ比が0
および1の付近で，比例関係が損なわれることがある．また，搬送
周期が長い場合には，サーボシステムの出力に搬送波成分が現れる
ので注意を要する．
PCM方式は，制御信号を ビットの2進信号 に符号化して，
これにより並列結合された 個のオンオフ弁を駆動するものであ
る．図Ⅴ-3･66(ｂ)に示すように，各弁に弁自体も含めた有効断面
積の比が， : :…:＝2:2:…:2になるような絞りを接続
しておけば， により2段階の総合開口面積 を設定できる．複
数のオンオフ弁が必要であるが，PWM方式のような常時の高周波
切換えは必要なく，弁寿命や騒音の点で優れている．
図Ⅴ-3･66(ａ)(ｂ)はそれぞれ一つの2ポート流量制御弁として
働く．3ポートオンオフ弁を用いれば，3ポートのアナログ制御弁
と等価な機能が得られる．これらの弁では，弁制御信号はディジタ
ル量であり，D-A変換器を必要とせず，ディジタルコンピュータ
の出力信号で直接制御できる利点がある．
