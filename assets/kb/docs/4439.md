# 3・2・5 斜流送風機および圧縮機

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・2 遠心送風機および圧縮機各論

a. 構造斜流送風機(diagonal flow blower)および圧縮機は遠心式と軸流式の中間に位置するもので,軸方向から吸込まれたガスを軸と傾斜した方向に吐出す斜流羽根車(diagonal flowimpeller)をもつことを特徴とする.斜流羽根車は軸と傾斜した出口形状をもつため軸方向寸法が長くなり,構造上多段形として利用されることは少ない.これを片持ち形にした場合には,オーバハング部が長くなるため軸の危険速度については十分な検討が必要となる.羽根車には遠心式同様オープン形,クローズド形のものがあり,材料は一般にアルミニウム合金やステンレス鋼などが用いられ,鋳造や削り出しなどの方法で製作される.特殊な用途においてはチタンなどを用いることもある.羽根車を除く他の構成要素は遠心式の場合とほぼ同様であり,インレットベーン,ディフューザ,スクロールなどをもつ.図III-3・17に斜流送風機の構造例を示すが,駆動機としてガスタービンや高速モータなどを用いることにより増速機をもたないものもある.b. 性能斜流羽根車では,軸方向から吸込まれたガスが軸のまわりを円すい状に流れながら昇圧される.斜流式は遠心式と軸流式の中間の性能をもつため,同程度の昇圧能力をもつ遠心式に比べて大流量を処理することができ,性能面では多段形の初段部分に適している.斜流送風機および圧縮機は300~1000(m, m/min,min)の比速度(specific speed)において特性上の優位性をもつ2―169第III部気体機械図-3・18 水平割り形遠心圧縮機図-3・19垂直割り形遠心圧縮機とされるが,一般に比速度300~500(m,m/min,min),ガス流量50~1500m/min,圧力比1.5~3程度の単段形で使用されることが多い.斜流羽根車は側板(シュラウド),主板(ハブ)の曲率が遠心式のものに比べて小さくできるため,流れが急激に曲げられることがなく,羽根車内の負荷分布が緩やかになる.したがって,二次流れなどの損失が少なく,効率が高い.しかし,遠心式および軸流式羽根車に比べて内部流の三次元性が強く,設計が難しくなる.ディフューザには羽根なしまたは羽根付きが使用されるが,羽根車出口が傾斜しているためにその組合せが難しく,設計には注意を要する.軸方向長さを短くするために,ディフューザ入口部に曲率をもたせた径方向ディフューザを使用することもあり,この入口部の曲率の選定が性能に大きな影響を及ぼす.
