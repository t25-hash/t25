# 3・2・6 遠心圧縮機(centrifugal compressor)

a. impeller)は外径基準の周速度が750m/s程度で運転される場合もあり、クローズド羽根車(closed impeller)は一般に周速度360m/s程度までで使用される。羽根車の材質は、一般に低合金鋼、高合金鋼、アルミニウム合金あるいはチタン合金(Ti-6Al-4V)などが用いられている。製造方法には、羽根車を一体で精密鋳造する方法や、一体の材料から5軸NC(numerical control)機械で削り出す方法、放電・電解加工を行う方法がある。また、接合により一体の羽根車を製作する方法には、羽根を主板や側板に羽根側から溶接する方法、逆に外側から電子ビーム溶接やスロット溶接を行う方法、羽根と主板または側板の間に特殊ろう箔を挟み、加熱して接合するろう付け法などがある。精密鋳造や5軸NC機械加工はオープン羽根車に、放電・電解加工、溶接やろう付け接合はクローズド羽根車に用いられることが多い。このようにして製作された羽根車は、設計回転速度の10~20%の過回転で強度の確認が行われる。ケーシングは耐圧気密構造を必要とし、吐出し圧力が6MPa程度までは水平割り形であるが、それ以上の圧力や吐出し圧力が1MPa級でも分子量が小さく漏れやすいガスなどの場合には垂直割り形が使用される。水平割り形は、下半ケーシングに吸込口と吐出し口を設け、上半ケーシングを取り外すことにより、内部のロータや静止流路部の開放点検が行いやすくなっているものが多い。垂直割り形は、円筒状のケーシングにロータおよび静止流路を含む内蔵品を軸方向に挿入して組立てられるため、片側または両側に十分な強度をもつ円盤状のふたを設けている。これらケーシングの材質は、水平割り形では鋳鉄、鋳鋼、特殊鋼または炭素鋼溶接構造が多い。垂直割り形では鋳鋼または鍛鋼で吸込口、吐出し口は溶接によって取付けられることが多い。主軸に羽根車を組付ける方法としては焼きばめだけのもの、アキシアルキーやラジアルピンを焼きばめに併用したもの、羽根車をはめあいにて組合わせ、ボルトにて軸方向に結合したものなどがある。圧縮機を運転すると羽根車の吸込側には入口圧力が、また主板の背側には出口圧力が作用して軸スラストが発生するので、最終段羽根車の後に釣合いピストンを設けてスラストを低減させる(図III-と後段側を背向させて配列したり、羽根車を同じ向きに配列する場合には、ダイアフラムラビリンスの直径を羽根車入口部ラビリンスの直径と同一にして軸スラストの低減を図ることもある(図III-3・18)。主軸を支える軸受としては、真円軸受、多円弧軸受(2~4円弧)、ティルティングパッド軸受(3~5パッド)などの滑り軸受を使用することが多い。特に、高速回転で比較的軸受荷重の小さい場合には、オイルホイップやオイルホワールを防止するために、ティルティングパッド軸受が使用される。主軸がケーシングを貫通する部分には、内部のガスが大気に漏洩するのを防止するため、各種の軸封装置が用途に応じて使い分けられている。b. 用途低圧のものでは、高炉送風用、空気分離装置原料空気用、はん用空気用、各種ガスの圧送用、極低温(-150°C)のLNG (liquefied natural gas)ボイルオフガス用および一般化学プロセス用などがある。空気分離装置原料空気圧縮機には、軸流段と各段中間冷却をした遠心段の複合形で吸込ガス流量7500m/minの大容量圧縮機も出現している。また、はん用空気圧縮機は小型高速化され、パッケージ形で使用されている。高圧のものでは、天然ガスパイプライン用(4~10MPa)、メタノール合成用(6~10MPa)、重油脱硫用(10~20MPa)、アンモニア合成用(15~35MPa)などがあり、また油田やガス田で使用されるガスリフト用やリインジェクション用圧縮機(10~65MPa)もある。さらに小型ジェットエンジン、小型ガスタービン、過給機、ターボ冷凍機などにも遠心圧縮機が使用されている。遠心圧縮機の運転中に生じる応力としては、主として遠心力による応力と流体力などにより羽根車が加振される振動応力がある。クローズド羽根車は、側板の拘束により羽根の固有振動数を上昇させうるが、側板が付加質量となって、羽根、主板に過大な遠心応力を生じさせることがあり、一般には外径基準周速度で360m/sまで使われる。一方、オープン羽根車は750m/sを超える周速度でも使われるが、出口温度が高い場合には熱応力を加味した計算が必要となる。遠心応力や熱応力は機械の起動停止の繰返し数なども考慮し、疲労強度面でも許容値を超えないようにする。羽根車各部の固有振動数は、運転時に予測される主要な励振周波数と共振しないようにする。
