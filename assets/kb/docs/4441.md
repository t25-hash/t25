# 3・1・1 情報処理システムにおける記憶装置（storage unit ）

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・1 記憶装置概論

の役割
人類が今日の高度な文明を築き得たのは，言語を発明し，過去か
ら現在までに多くの人々が得た情報や知恵を記録し，先人の知識遺
産を現在のすべての人々のものとすることによって，より高度な技
術を開発し社会の生産力を高めてきたからである．大量の情報を高
密度・高速に蓄積する記憶装置は，人間のこのような知識生産力を
高めるための機械である．人類の知識生産技術の歴史の中で画期的
な出来事は，紙の発明，印刷機械の発明，通信技術の発明，そして
20世紀半ば以降に生まれた電子計算機と大量の情報を高密度・高
速に蓄積できる記憶装置の発明であった．
情報処理システムは，図3･1にパーソナルコンピュータ（パソコ
ン）の構成例を示すように，機能の異なる五つの装置から構成され
る．すなわち，数字，文字，図形，音声，画像などの自然情報を機
械的に処理できる2値符号の機械情報に変換する，マウス，キーボ
ード，スキャナ，ディジタルカメラ，ディジタルビデオカメラなど
の入力装置，データを処理・加工する中央処理装置（CPU:cen-
tral processing unit ），処理中のデータを一時的に記憶する内部記
憶装置［internal storage unit ，主記憶装置（main memory
unit ）］，データおよび処理プログラムを蓄積する外部記憶装置
［external storage unit ，補助記憶装置（auxiliary storage unit ）］
および処理したデータを人間の理解できる自然情報に変換するプリ
ンタなどの出力装置である．情報処理システムには，パソコン，ワ
ークステーション，サーバ，電算機センタ，携帯情報機器などがあ
るが，ここでは現在生産額の大半を占めているパソコンにおける記
憶装置に注目する．
内部記憶装置には電気的に高速ランダムアクセスが可能な半導体
メモリが使用される．その主体は構造が簡単で集積度が高いが記憶
保持動作が必要なDRAM（dynamic random access memoryの
略，ディーラムと呼ぶ）である．CPUとの直接の演算処理には，
集積度は低いが最も高速アクセスが可能で記憶保持動作が不要な
SRAM（static random access memoryの略，エスラムと呼ぶ）
が用いられる．半導体メモリは通称メモリとも呼ばれる．
一方，外部記憶装置には通称ストレージと呼ばれる機械的装置が
使用される．まずランダムアクセス可能な大容量記憶装置として磁
気ディスク装置（通称，ハードディスク装置ないしHDDと呼ばれ
る）がある．また小容量データの保存・配布・交換媒体としてフレ
キシブルディスク装置（1.4MBの3.5インチフロッピーディスク
装置をFDD，ディスク媒体をFDと呼ぶ）が，大容量プログラム
データ，画像データ等の保存・配布・交換媒体としてCD，MO，
DVDなどの光ディスク装置が用いられる．さらにワークステーシ
ョン，サーバ，データウェアハウスなどの大規模データのバックア
ップに磁気テープ装置（MTD）が用いられる．一方，ディジタル
機器一般の基本ソフトの格納，携帯情報端末，携帯電話，ディジタ
ルカメラの外部記憶装置およびパソコン用情報交換媒体として半導
体メモリの一種であるフラッシュメモリが用いられている．
一般に記憶装置に要求される性能は，ランダムアクセス時間が短
く，データ転送速度が高く，かつビットコストが低いことである．
しかし，これらすべての面で優れた記憶装置は存在せず，例えば
DRAMに対してHDDはアクセス時間が5けた以上遅いがビット
コストは2けた以上低い．そこで図3･2に示すように，CPUに対
して各種記憶装置を階層的に構成し，演算処理のためCPUと常時
データをやり取りするメモリとしては，ビットコストは高いがアク
セス時間が最も小さいSRAMを用い（キャッシュメモリという），
そこにデータがなければその約3けた大容量のアクセス時間はやや
遅いDRAMからデータをもってくる．また，そこに処理すべきデ
ータがなければ，アクセス時間はmsオーダであるがビットコスト
がさらに小さく2けた以上大容量のハードディスク装置からデータ
をDRAMに移動させるようにしている．さらにコンピュータシス
テムへのデータ，プログラムの入出力・保存・交換等には，媒体価
格の安いフラッシュメモリ，フレキシブルディスク（ドライブや媒
体の絶対価格が小さいことも重要）を小容量用に，容量の増大に応
積蓄積密度が高く，データ転送速度が比較的高い磁気テープを用い
成という．また携帯情報機器では，ビットコストは比較的高いがア
領域を作る必要があり，これを記憶細胞（ビットセル）という．記
憶細胞には番地がふられ，任意の番地を選択し，これに書込み読出
細胞は3個のトランジスタによるフリップフロップ回路からなり，
記憶に用いられる．しかし，電荷が時間とともに失われるので，一
じてCD，MO，DVDを用い，大規模データのバックアップには体
ている．用途に応じて各種記録媒体を使い分け，全体的にシステム
の性能・価格比を高める，このような使用形態を記憶装置の階層構
クセス時間が短いフラッシュメモリが外部記憶として用いられてい
る．以下に記録原理が異なる半導体メモリと機械的なストレージに
ついて技術の特徴と動向を述べる．
