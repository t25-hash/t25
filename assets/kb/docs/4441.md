# 3・3・1 羽根車の応力

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・3 遠心圧縮機羽根車の強度

羽根車各部の遠心応力は,有限要素法(FEM : finite elementmethod)を用いた羽根車全体としての応力解析で求められている.計算に際しては羽根車の周期性を利用し,羽根間1ピッチ分を取出し,周期条件を適用して全体解析とする.計算要素の分割には自動分割法を利用し,また,応力の高い部分には解析精度を向上させるため,この部分を拡大した解析を行うなどの方法がとられる.後向き羽根車では,高周速度の場合,主板と側板に周方向および半径方向の相対変位が生じ,羽根根元部には遠心応力と曲げ応力が重畳して大きな応力を生じる.
