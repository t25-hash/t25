# 3・3・2 羽根車の固有振動数

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・3 遠心圧縮機羽根車の強度

羽根車に入る流れに偏流やウェークがある場合、さらには後流側からの影響を受ける場合には、羽根車は加振され、それらが羽根車の固有振動数と一致すると振動応力による材料疲労によりき裂を発生することがある。クローズド羽根車では主板、側板が直径を節とする固有振動モードから羽根を含む複雑な高次振動まで多数のモー編流体機械γ2(a) 一次モード(b) 二次モード図-3・26 オープン羽根の固有振動モード図-3・27 ホログラフィー干渉法による羽根車の固有振動モードドをもつ。これらすべての振動モードとの共振を避けることは難しいが、高次振動モードの共振は一般に振動応答が小さいので、低次モードにおける共振を避けるようにするのが一般的である。オープン羽根車では通常、主板との接合部を固定条件に羽根単独にて振動モードを計算し(図III-3・26)、主要な励振周波数との共振を避けるようにする。羽根高さがあり、主板の剛性が小さい羽根車については主板の剛性を考慮し、羽根と主板の連成振動を計算する必要がある。この場合にも有限要素法を用いれば、振動数、モードのほか最大振動応力点も詳細に求めることができる。また、ホログラフィー干渉法により固有振動数やモードを計測し、信頼性を確認することもある(図III-3・27)。
