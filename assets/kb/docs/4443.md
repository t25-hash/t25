# 3・4・1 特徴と適用範囲

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・4 軸流送風機および圧縮機の設計

軸流送風機および圧縮機は、二重円筒の環状流路にある羽根車によって圧力を与えられた気体がほぼ回転軸と同心の円筒面上を旋回しながら流れるのが特徴であり、長所として次の点が挙げられる。(1) 流路断面積の割合に大流量の気体が圧送できる。(2) 他の形式の送風機や圧縮機に比べて効率が高い。(3) 動・静翼を可動翼にすることによって高効率の作動範囲が広くなる。(4) 構造上容易に多段にでき、また送風管の途中に簡単に挿入できる。一方、短所として次の点が挙げられる。(1) 1段当たりの圧力上昇が小さく、高圧の場合多段とし、高速回転にしなければならない。(2) 失速点が最高効率点に近く、サージングまでの領域が狭い。(3) 遠心式に比べて翼が構造的に弱く、また翼の汚損による性能低下が著しい。(4) 同一風量風圧の場合、遠心式や斜流式より騒音が大きい。JIS 軸流送風機吐出し圧力は、扇風機や換気扇のようにほとんど無圧のものから18kPa程度のものが多い。流量は0.1~500m/sの範囲のものが多いが、航空機の風洞用として6000m/sに及ぶ大型送風機が作られた例もある。動力は小型のもので数分の一kW から大型のもので数千kW にも及ぶ。設計の基準となる比速度の範囲は、回転数(s)、流量(m/s)、気体の比仕事[J/kg:ヘッド(m)×重力の加速度9.806(m/s)]で表した無次元比速度=で0.4~1.2程度である。用途は鉱山の主通風や局所排風、トンネルの送・排風、ボイラへの押込通風機や誘引通風機、セメント製造用キルンへの送風、工場や建物の換気、集じん、脱硫設備への送風、作業場における粉じんの局所排気、塗装設備、乾燥設備、冷却塔の送風機、風洞の送風など極めて広い。軸流送風機は形式によりさまざまな呼称がある。扇風機のようにケーシングがなく羽根車が自由空間に開放され、案内羽根をもたないものをフリーファン(free fan)またはプロペラファン(propel-ler fan)、換気扇のように吸込口側と吐出し口側の空間が壁で仕切られ、羽根車先端のまわりにディスク状のシュラウドがあるものをディスクファン(disk fan)、ダクト状のケーシング内で羽根車が回転するものをダクトファン(ducted fan)という。ダクトファンの中で案内羽根のないものをチューブ軸流ファン(tube axialfan)、案内羽根のあるものをベーン軸流ファン(vane axial fan)と呼ぶこともある。案内羽根が羽根車の上流側にあるものを前置静翼形送風機(axial fan with inlet guide vane)、下流側にあるものを後置静翼形送風機(axial fan with outlet guide vane)といい、両側に案内羽根を付けたものもある。左右対称翼をもち、正逆両回転できるダクトファンでトンネルの天井に設置して換気するものをジェットファン(jet fan)と呼ぶ。また、2段の送風機でおのおのの羽根車を逆方向へ回転させることにより、中間静翼を省いたものを反転軸流送風機(contra-rotating axial fan)またはコントラファンといい、静翼の損失分を減少させて失速点も小流量側へ寄せることができるが、2段軸流ファンほどの圧力ではない。広い作動領域で高効率が要求される軸流送風機においては動翼可変ピッチまたは静翼可変ピッチにする。この典型的な例が発電所ボイラ用の押込送風機(FDF: forced draft fan)および誘引通風機(IDF: in-duced draft fan)であり、以前は遠心式が使用されたが、発電所の大型化と高効率化の要請および可変ピッチ技術の定着によって軸流式のものが多くなっている。b. 軸流圧縮機ジェットエンジン、ガスタービン用として1950年ごろから急速に進歩し、一般産業用としても各種空気源、高速風洞用、製鉄業における高炉送風用、化学工業における石油精製、酸素分離などの化学プロセス用など比較的大流量で中圧力の用途に使用されている。遠心式に比べて効率が高く、小型軽量化できる利点があるが、圧力-流量特性が急こう配で最高効率点からサージング点までの裕度が狭いので、広範囲の作動領域が要求される場合には可変ピッチ方式を採用する必要がある。産業用の軸流圧縮機は、圧力比1.2~2.0のブロワの範ちゅうに入る低圧のものから圧力比7までのものが多いが、ガスタービン用では圧力比30に及ぶものがある。1段当たりの圧力比は1.1~1.2であり、圧力比4を超える圧縮機は通常2段群として中間に冷却器を入れる。これは段数が増えると軸間距離が長くなり危険速度に対する安全が保てないこと、後段の流路断面積が狭くなり起動の際の対策が面倒であること、および高温で圧縮すると同じ圧力比に対して消費動力が大きいことの理由による。ただし、高温空気を必要とする高炉用圧縮機などでは圧力比が4を超えても中間冷却を行わない。流量は大きいもので200m/s、軸動力は7万kW にもなる。航空機ジェットエンジン用の軸流圧縮機は圧力比40に達するものがあり、圧力比5~7を1段群としている。1段当たりの圧力比は1.2~1.4であり高圧力比のものは翼に対する相対流入速度が先端で音速を超える。このような圧縮機を遷音速圧縮機(transoniccompressor)という。流量は大型広胴機用のバイパスエンジンで、圧縮機を通過する流量が100m/sにもなる。設計の基準になる無次元比速度は、(s)、(m/s)、(J/kg)を用いて、==0.4~0.55が普通である。しかし、航空機用圧縮機の最終段のようにハブ比が0.9を超えるとは0.3以下になることもある。
