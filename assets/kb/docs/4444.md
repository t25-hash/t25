# 3・1・4 ストレージの今後の動向

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・1 記憶装置概論

HDDの高密度化がさらに進むと，パソコン，サーバ等では2.5
インチ形の高性能機が3.5インチ形を代替していくであろう．大容
量・低価格化により，HDDとDVDは情報家電分野への本格的な
普及期を迎えた．ディジタルハイビジョン放送が普及し，また映像
をパソコンで自由に処理できるようになれば，HDDとDVDのさ
らなる大容量・高速化機種の普及が期待できる．また家庭用ビデオ
カメラ，ナビゲーションを含む車内ディジタル機器などの新分野
に，耐衝撃性に優れた小形大容量HDDが使用されていくと予想さ
れる．他方ではブロードバンド通信が普及し，携帯電話等の携帯情
報機器のストレージに大容量のフラッシュメモリや1インチ以下の
HDDが搭載され，いつでもどこでも各種の知識情報，音楽，静止
画，動画等を自由に享受できるユビキタス時代の到来も近いと思わ
れる．
