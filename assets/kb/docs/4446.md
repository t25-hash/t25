# 3・5・1 軸流送風機

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・5 軸流送風機および圧縮機の構造と特性

a. 構造数枚ないし数十枚の動翼をもつ羽根車、動翼の前方または後方に置かれる静翼、ボス部を支える支持翼、ケーシングなどから成り立っている。電動機の軸端に羽根車を取付け流路内に内蔵したものは、最も簡単な構造である。大容量のものや高温ガスを扱う場合には、ダクトの曲がり部を利用して電動機をケーシングの外に置き、軸継手を介して駆動される。図III-3・31はトンネル換気用や風洞用に採用される形式である。通常は横形で使用されるが、用途によっては立て形に配置されるものもある。また、換気用として羽根車を逆転して送風機を排風機として使用できるようにしたものもある。特殊なものとして、二つの羽根車を直列に配列し、互いに反対方向に回転させる反転軸流ファンがある。これは静翼が不要で効率が良いという特徴をもっている(図III-3・32)。流量調節の方法として、動翼あるいは静翼の取付け角を停止中または運転中に変える機構をもったものがある。特に動翼可変形(adjustable rotor blade type)は部分負荷時の効率低下が少ないため、ボイラ通風用などに広く用いられている(図III-3・33)。材料としては動翼は鋼板、ステンレス鋼、アルミニウム合金、耐図-3・31 軸流送風機の例図-3・32 反転形2段軸流ファンの例図-3・33 動翼可変形軸流送風機図-3・34 等容量の送風機を2台並列運転する例2―175第III部気体機械熱合金、合成樹脂などが使われ、ケーシングは一般的には鋼板が使われる。b. 特性圧力係数の小さいファンの特性は、図III-3・34に示すように一般に右上がり特性の部分をもつ。このため、2台並列運転の場合にはA で示すS 形の総合特性曲線ができ、抵抗曲線がIIIより大きい場合には特性曲線との交点が3点できることになり、わずかの擾乱で作動点が1~3を移行し不安定状態となる。このため、バイパスなどを設けて抵抗曲線をIIIより小さくする必要がある。図III-3・35には動翼可変形の特性例を示す。この場合のように圧力係数の高い場合には流量を減らしていくと、右上がり特性に入る前にサージング領域に入り運転が不可能となる。図III-3・36に形式、制御方法の違いによる部分負荷特性の比較を示す。
