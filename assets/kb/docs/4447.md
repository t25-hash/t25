# 3・5・2 軸流圧縮機

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・5 軸流送風機および圧縮機の構造と特性

a. 構造多段構造となり、動翼を植えたロータ、静翼、ケーシング、軸受などから構成される。軸流圧縮機ロータの出口側に遠図-3・35 動翼可変形軸流送風機の特性曲線の例図-3・36 部分負荷時の軸入力比較図-3・37 静翼固定形軸流圧縮機2―176流体機械図-3・38 静翼可変形軸流圧縮機図-3・39静翼可変形軸流圧縮機の特性曲線の例心羽根車を取付けて一体化した構造もあり、複合圧縮機と呼ばれる。多段軸流圧縮機の場合には、図III-3・37およびIII-3・38に示すように静翼固定形と静翼可変形(adjustable stator bladetype)がある。特殊用途を除いて横形で、水平分割ケーシングをもつものがほとんどである。大型の場合には軸受箱はケーシングと独立に、基礎の上に置かれる。b. 特性軸流圧縮機の特性曲線は遠心圧縮機のそれに比べて垂直に近く、定流量運転に適した特性をもっている。特に静翼可変形は静翼固定形の速度制御などに比べて作動範囲が広く、部分負荷時の効率が良いので広く用いられている(図III-3・39)。圧力比の高い圧縮機の場合には高圧力側のサージング、低流量側の旋回失速、大流量側のマッハ数による制限、低圧力側の後段チョーキングなどによって運転範囲が制約される。
