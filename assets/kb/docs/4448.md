# 3・6・1 動翼および静翼の構造と強度

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・6 軸流圧縮機の強度

a. 動翼(rotor blade)材料は一般的には12クロム系ステンレスが多いが、耐食性あるいは強度上の理由で17-4pH、チタン合金などが使われることもある。動翼はT ルートまたはダブルテイルルートによりロータに固定される(図III-3・40)。このルート部分には翼全体の遠心力がかかるので、あごの部分の応力と接触面圧を調べておく必要がある。翼部には遠心応力と空気力による曲げ応力のほかに、空気力による励振応力がかかる。一般に遠心力による引張応力は翼の根元で最大となり次式で表される。空気力による最大曲げ応力および励振応力は翼の根元付近に生じ図-3・40 動翼図-3・41 グッドマン線図の例るが、これら翼に生じる応力を定常応力と振動応力に分け、一般にグッドマン線図と呼ばれる疲労限度線図に示して強度を確かめる(図III-3・41)。動翼の固有振動数は、近似的には片持ばりの計算式を使って計算することができる(α2編「機械力学」12・2節参照)。遠心力を考慮した場合、サウスウェル(Southwell)によれば遠心力が働いたときの固有振動数は=+と表される。実用上は、はりとして計算した値を類似の翼の実測値を用いて補正したり、有限要素法による計算を行って推定をすることができる。この固有振動数が動翼にかかる励振力と一致しないように、回転速度の整数倍成分(通常1から4~6まで)や、前方および後方の静翼枚数×回転速度、ステー枚数×回転速度などの周波数と一致しないように設計する。使用回転速度範囲で共振しないよう一般にキャンベル線図と呼ばれる線図に示して確かめる(図III-3・42)。b. 静翼(stator blade)材料は動翼とほぼ同じものが使われる。曲げ応力、固有振動数の計算は動翼と同じようにする。動翼に比して遠心力がかからないため引張応力がなく、取付け部のダンピング効果が大きいため強度的には動翼より軽減される(図III-3・43)。静翼可変機構は、静翼に取付けたアームを周方向に出し、アームの先に取付けたローラを案内リングの溝にはめ、案内リングをケーシングの左右に設けた2個の油圧シリンダにより軸方向に動かして静翼角度を制御する方法が一般的であるが、アームを軸方向に出し、案内リングを圧縮機の回転方向にまわして制御する方法もある(図III-3・44)。静翼軸の軸受としては、二硫化モリブデンコーティングやカーボンなどのドライベアリングが用いられる。
