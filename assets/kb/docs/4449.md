# 3・6・2 ロータおよびケーシングの構造と強度

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・6 軸流圧縮機の強度

a. ロータ材料は一般に合金鋼や鍛鋼が用いられる.ソリッ図-3・42 動翼のキャンベル線図の例図-3・43 静翼(固定式)図-3・44 静翼可変機構ド形が最も一般的に採用され,数万kW の大型ロータまで製作されている.中空形も使用されるが,内外周の同心度と,軸端部との結合部の強度に注意を要する.このほか,ディスクはめ込み形,ディスク溶接形も使用される.強度的には,中心孔表面,あるいは動翼取付け部に最大応力が発生するので,十分な検討と探傷検査が必要である(図III-3・45).b. ケーシング材料としては,鋳鉄,鋳鋼,鋼板が一般的に用いられる.大型多段圧縮機では吸込部,中間部,吐出し部に3分割され,ボルトで締め付けて組立てられる.ケーシングの全長を短縮し剛性を上げるために,吸込・吐出しケーシングの中にも翼列を配置した多段圧縮機の例もある.2―177第III部気体機械(a) ソリッド形(b) 中空形図-3・45 ロータ構造図-3・46 ケーシング吐出し圧力500kPa以上の高圧空気圧縮機の場合,吐出し温度が250~300°Cになるために,ケーシングの均等な熱伸びができるようにスライド面,リンク,たわみ板などを用いるとともに,上下方向の移動を防ぐために中心線付近で支持し,支持脚を冷却するなどの工夫をする.軸流圧縮機のケーシングは,大きさの割に薄肉で変形しやすいので,配管反力によって変形しないよう伸縮継手などで配管の伸びを吸収する必要がある(図III-3・46).
