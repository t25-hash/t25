# 3・2・6 SAN（storage area network）と仮想化

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・2 記憶システム

並行して行うには高度なソフトウェアが必要とされた．
この流れは，ディスク装置をより高付加価値な装置へと変ぼうさ
せるきっかけとなっていくと考えられる．インタネットを介したビ
ジネスが広く浸透するにつれ，バックアップによるシステム停止が
無視できなくなってきたことを背景に，ディスクシステム内での二
次データの作成（スナップショット）取得機構と非同期バックアッ
プという形態が提案されている．さらに，システム障害への対処と
して，ディスクシステム自体にリモートコピー機構を導入し，ホス
ト介在のない高信頼なファイルシステムを実現する手法が導入され
た．最近では，メインフレームとオープン系システムとの処理コー
ドの違いをディスクシスム上で自動変換することにより，ファイル
共有を可能にするなど，極めて高度な機能が実現されつつある（図
3･7）．
