# 3・7・1 横流送風機

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・7 特殊送風機

横流送風機(cross flow fan)は、図III-3・47に示すように流体が前向き羽根をもつ羽根車を貫流して流れるためクロスフローファン、貫流ファンとも呼ばれ、各種の空気調和機器用や電子機器の冷却用送風機として用いられる。羽根車形状は、内外径比0.70~0.85、周方向から測った外周羽根角20°~30°、内周羽根角80°~100°で節弦比は0.8~1.0程度である。流れが二次元的で軸方向長さを変えることで風量を制御でき、また、羽根車翼列を二度通過するために圧力係数が高いなどの特徴をもつ。しかし、羽根車内部に偏心した強い循環渦が形成されること、および羽根と舌部との干渉に基づく騒音の関係で大きい周速度を取ることができないため、圧力上昇は数十Pa程度である。流れ場が軸対称でないうえに性能に及ぼす設計パラメータが多く、いまだ一般的な設計法が確立されていない。空気調和機器用では静音化が重要で、回転速度()と羽根枚数()の積に基づく音を低減させるため千鳥配列、スキュー、ランダムピッチ羽根をもつ羽根車が使用される。性能曲線の例を図III-3・48に示す。流量係数φ=、圧力係数ψ욧=욧ρ2(:羽根車外径、:羽根車周速度、:羽根車軸方向長さ)が大きく、同一風量を得るのに他のファンより小型、薄型にできるが、効率ηは低い。圧力係数はケーシング形状にもよるが一般に流量に対して前向き羽根特有の右上がり特性を示す。また、動力係数λ=ρ2は流量とともに増大する。臨界羽根車レイノルズ数(1.0~1.4×10)以上の送風機でも性能、騒音特性に一般的な相似則は成り立たず、その特性に羽根2―178流体機械図-3・47 横流送風機の流れ図-3・48 横流送風機の性能図-3・49渦流送風機車寸法の影響が現れる。
