# 3・7・2 渦流送風機

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・7 特殊送風機

渦流送風機(vortex blower,regenerative blower)は、通常、流量15m/min以下、吐出し圧力30kPa以下で用いられ、その形状(図III-3・49)は渦流ポンプと類似しているが、羽根高さ、羽根幅と流路面積がポンプに比べて大きい。羽根車の回転に伴って、羽根外周から角運動量を得て流出した流体が流路断面を旋回する間に圧力に変換され、再び羽根根元に流入する。流路をらせん状に流れ何度も羽根を通過するため、小型の割に圧力は一般のターボ形ブロワに比べて高く、吐出し、吸込両用として事務用機器、食品機械図-3・50 渦流送風機の性能等に幅広く用いられる。性能曲線の例を図III-3・50に示す。小型で高風圧である、圧力係数は右下がりで小流量域でも安定な運転ができる、構造がシンプルで信頼性が高いなどの特徴をもつが、効率は40~50%である。二次元羽根に対して三次元前向き羽根を用いることによって高圧力で高効率が得られる。渦流送風機の羽根車の作用や基本性能解析は、乱流摩擦理論や角運動量理論で取扱われる。
