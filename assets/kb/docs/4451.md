# 3・2・7 ストレージ，ファイル，データベース

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・2 記憶システム

ストレージ（storage）にはデータが格納され，それを管理する
最も基本的なソフトウェアがファイルシステムであることから，ス
トレージにファイルシステムというソフトウェアを組込むことが容
易に考えられる．ファイルサーバとも呼ばれてきたNAS（net-
work attached storage）と名付けられている製品がこれに相当す
る．2004年時点ではファイバチャネルの性能のほうがイーサネッ
トに比べて高いため，入出力負荷の高い本格的なアプリケーション
はSAN上に実装される場合が多い．ファイルよりもさらに厳密な
データ管理を司るソフトウェアがデータベース管理システムであ
り，今後ストレージのアプライアンス化が進むものと予想される．
NASとSANは排他的な技術ではなく，NASを実現するために
ストレージとしてSANを利用することが考えられる．また，メタ
データをNASで管理し，実データの転送をSAN経由とする手法
も提案されている．記憶装置とそれに格納されるデータを管理する
ソフトウェアの一体化は今後の大きな方向と考えられる．
