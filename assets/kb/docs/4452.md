# 3・8・1 据付け,運転および保守

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・8 据付け,運転,振動および保守

a. 設置場所場所の選定と周辺設備の計画に当たっては次の点に注意を必要とする.(1)周囲温度の極端な変化(季節,昼夜),(2)降雪,強風,地震,砂じん,高潮,日射など,(3)地盤の地耐力,(4)騒音,振動,防火などの規制,(5)運搬道路,搬入口,分解点検に必要なスペース,(6)クレーン容量,(7)周辺機器,配管の配置計画,(8)将来計画.b. 基礎基礎(foundation)が地上にコンクリート,鉄骨などで作られる場合,機械がモジュールなど鉄骨架構上に設置されて運搬される場合,地上に直接設置される場合などがあるが,基礎は次のような条件を備えていることが必要である.(1)機械の静荷重,動荷重に耐えるだけの強度と剛性をもつこと,(2)基礎の固有振動数が機械の励振力と共振する範囲にないこと,(3)不等沈下のないよう十分な地耐力のある地盤上に置かれること,(4)振動が建屋に伝わらないように絶縁されていること.c. 据付け(1)基礎とベッドの間には基礎ボルトの両側に心出し用の調整ライナまたは調整ボルトを設け,基礎ボルトを締め付けたときにベッドが水平になるようにする.基礎とベッドの間にモルタルあるいはエポキシグラウトを流し込み,硬化後基礎ボルトを締め付ける.(2)大型機械の場合にはロータのたわみを考慮した心出しを行う.(3)被動機と駆動機の心出しは,軸継手部でダイヤルゲージを用いて相互の振れを計測しながら行い,許容値におさめる.(4)高温ガスを扱うものや駆動機が蒸気タービン,ガスタービンなどの場合は,熱膨張を考慮した心出しを行う.(5)軸方向についても,ロータの熱伸び,軸継手部の移動距離を考慮して位置決めを行う.d. 配管(1)配管と取付け時には,正しくフランジ面を合わせ,配管の重量が過大に機械にかからないように適当な支柱やつり具を設ける.(2)配管の熱伸びによって機械に過大な力が働かないよう伸縮継手を挿入したり,配管ループを設ける.(3)送風機,圧縮機には吸込側に直管長さを要求されることがあるので注意する.(4)現地で配管工事を行う場合には,現物合わせができるようにルーズフランジとする箇所を設けておく.(5)大型の弁や消音器など,配管と接続される機器についても同様に配管の力の働き方を十図-3・51 振動状態の判定基準分考慮して据付ける.(6)騒音対策が必要な場合には配管にも対策が要求されることがある.e. 運転(1)運転前には機械,配管内が十分清掃され異物が残っていないことを確認する.潤滑油系統はオイルフラッシングにより十分に清浄となっていなくてはならない.ドライガスシールのシールガス配管についても十分に清浄にしておく.(2)運転開始前に必要な安全装置,制御装置,補機などが完全に作動することを確認しておく.(3)起動中は特に振動,異常音,軸受温度,給油圧力などに十分注意する.(4)運転中には一般性能のほかに,ケーシング,軸受,ベッドおよび配管などの振動状況ならびに異常音,給油圧力および軸受温度,送風機よりのガス漏れ,中間冷却機の温度などをよく監視する.f. 保守(1)運転中の監視事項のおもなことは運転記録として残しておく.特に運転データの変化の履歴は機械の異常診断のために極めて重要である.(2)潤滑油の管理は特にたいせつで,油の汚れや劣化によって故障の原因となる.(3)取扱いガスが腐食性であったりダストやタールなどを含むものについては,これらの付着や腐食によって重大事故に至る危険があるので運転状況を十分監視する必要がある.(4)長時間の連続運転に耐えるよう設計されるのが普通であるが,年1回または2年ごとの定期点検を行うことが好ましい.その際にはケーシング,ロータ,冷却器,増減速歯車装置および潤滑装置などの清掃点検,しゅう動部の間げき計測と摩耗した部品の交換,潤滑油の劣化度の調整,機械の腐食状況調査などを行う.g. 故障対策おもな故障と原因について列記する.(1) 性能低下:羽根,ケーシング,中間冷却器などの腐食,摩耗,ダストの堆積,ラビリンスなどのすきま増加,吸込フィルタの詰まりなど.(2) 腐食,摩耗による破損:材料選定不良,ガス性状の変化,運転不良による水分等の混入,フィルタ性能低下など.(3) 軸受温度の上昇:給油圧力低下,油量減少,ごみの混入,潤滑油不良,油冷却器性能低下など.
