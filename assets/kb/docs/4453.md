# 3・8・2 振動、騒音

> **章**: 第3章 ターボ形送風機および圧縮機
> **節**: 3・8 据付け、運転、振動および保守

a. 振動(第I部5・1節参照)発生する振動源はさまざまであるが、振動の原因となるものを挙げてみる。流体的振動としては、(1)サージング、(2)渦流または偏流、(3)配管との共振、(4)吐出しガスの脈動がある。機械的振動としては、(1)回転体のアンバランス、すなわち、バランス不良、回転体の変形、破損、腐食摩耗、異物付着、軸の曲が2―179第III部気体機械りなど、(2)心出し不良、(3)軸受の損耗あるいは間げき増大、(4)軸継手不良、すなわち、加工精度不良や歯当たり不良など、(5)危険速度に近接、(6)オイルホイップやオイルホワール、(7)基礎不良または据付け不良、(8)共振その他、すなわち、駆動機との結合状態での共振、配管支持不良などによる振動伝ぱなどがある。運転中の振動の許容値については、JIS B 8330:1981「送風機試験および検査方法」、JIS B 8340:1976「ターボ形ブロワ・圧縮機試験方法および検査方法」に、図III-3・51のような軸受部の振動状態の判定基準が参考として示されている。APIあるいはISO等にも規定値が示されている。b. 騒音(発生騒音については第I部5・2節参照)騒音対策を立てるに当たって、まず(1)工場周辺の境界線における規制値、(2)工場内の環境としての規制値を明確にする。次に騒音対策をしない場合の目標地点での騒音レベルを計算すると、その差が必要減音量となる。騒音対策としては、機械に防音カバーを設けること、吸込口、吐出し管内、放風口などに消音器を設けること、配管に吸音材を取付けること、低騒音弁を用いることなどが行われる。消音器には各種の形式があるが、吸音材形、スプリッタ形、セル形、共鳴形などがあり、圧力損失消音性能、コスト、スペースなどを考慮して決定される。文献(2) Rodgers,C.,ASME Paper,2000-GT-455(1972)。(3) 大橋秀雄・ほか9名編、 流体機械ハンドブック、 (1998)、 321-323、朝倉書店。(4) Ferguson,T.B.,The centrifugal compressor stages、(1963)、37-38,Butterworths.(7) Dallenbach,F.,SAE Tech.
