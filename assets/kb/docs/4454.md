# 4・1・1 形式

> **章**: 第4章 軸流タービンおよびラジアルタービン
> **節**: 4・1 形式と選定

タービン(turbine)とは、作動流体のもつエネルギーを機械エネルギーに変換する回転形(ターボ形)機械のうち、水車や風車を除いて、空気、燃焼ガス、蒸気などおもに気体を作動流体とし、流体の有するエネルギーを角運動量の変化を利用して回転仕事として外部に取出す機械のことである。タービンには、作動流体の流れの方向が動翼の回転軸にほぼ平行なもの(軸流タービン、axial flowturbine)、垂直なもの(ラジアルまたは半径流タービン、radialflow turbine)がある。両者の中間のものを、特に斜流(混流式)タービン(diagonal flow turbine,mixed flow turbine)と呼んで区別する場合もある。一般にタービンは上流側に設置されるノズル翼[nozzle vane、静翼(stator blade)]と下流側に設置される動翼(回転翼、rotor blade)で一つの段を構成する。図III-4・1にさらに詳細な分類と、それらの形状の模式図、内部での圧力変化、絶対速度変化を示す。a. 軸流タービン一段の圧力降下(膨張)をすべて静翼で行う衝動形タービン(impulse turbine)と、静翼と動翼で分担する反動形タービン(reaction turbine)とがある。さらに衝動形は図III-4・1(イ)、(ロ)、(ハ)に示すような圧力比の比較的小さい単段形、大きな速度複式形(カーチスタービン)や多段形に分けることもできる。また反動形にも単段形、多段形があり、ここでは多段形のみ図III-4・1(ニ)に示した。反動形ではノズル翼、動翼の両翼列内の流れ(翼に相対的な流れ)は増速流であるため、大きな損失は起きにくく効率が高い。軸流形はラジアル形に比べて段当たりの膨張の度合いは小さいが、構造上、段を軸方向に数多く重ねるのが容易で、全体としての圧力比を高く取れ、効率も高い。b. ラジアルタービン(radial turbine)流れが半径方向に外向きのタイプ(outflow type)と内向きのタイプ(inflow type)がある。外向き半径流形では動翼の入口と出口の径の差が大きくなる単段(イ)速度複式(カーチスタービン)(ロ)多段(ハ)反動形(ニ)(a-2)衝動形(a-1)軸流タービン(a)ターボ形機械外向き半径流(ユングストロームタービン)(ホ)(b-1)内向き半径流(ラジアルタービン)(ヘ)(b-2)半径流タービン(b)(ハ) 衝動形多段(ニ) 反動形多段図-4・1 タービンの分類2―181と、出口での周速度が入口でのそれよりかなり大きくなるので、入口と出口間の角運動量変化(取出す仕事量)を大きくすることが困難である。そこで入口と出口の径の差を少なくした、薄い翼列を複数用いた図III-4・1(ホ)のようなユングストロームタービンが考案される。これは図のようにノズル翼がなく、互いに反対方向に回転する動翼のみで構成され、各動翼列で仕事を取出すものであり、したがって出力軸としては2軸になる。圧力降下が動翼のみで起こる意味では100%の反動形ともいえる。一方、内向き半径流形は作動流体が外径側から内径側に流れるタイプで、図III-4・1(ヘ)のように外周よりまず渦巻室に流入し、次にノズル部を通って高速旋回流となって動翼に流入する。ノズル部に翼があるものを羽根ありノズル、ないものを羽根なしノズルと呼ぶ。動翼内で流れが外径側より内径側に移るにつれ、流れの旋回速度が減る。動翼の出口に案内翼を付け、さらに旋回速度を下げてほぼ軸方向に流出させることが多い。遠心力による半径方向の圧力こう配(外径側が高圧、内径側が低圧)にそのまま沿って流れるので比較的損失が少なく、反動度(degree of reaction)も50%近傍に取ることが多いので小型のものでも効率が高い。
