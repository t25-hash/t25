# 4・1・2 選定

> **章**: 第4章 軸流タービンおよびラジアルタービン
> **節**: 4・1 形式と選定

a. 衝動形軸流タービン(impulse type axial flow turbine)(1) 動翼の周速度の割に段当たり圧力比(仕事量)を大きく取れる。(2) ノズル翼のみで圧力降下させるため流量を調節しやすい。(3) 反動形に比べて効率はあまり高くない。(4) 構造が比較的簡単で軽量化できる。(5) 軽量、高圧力比の点でロケットエンジン用タービンなどに適用されている。(6) 段当たり大きな圧力比を吸収できるので蒸気タービンの初段などに使われている。b. 反動形軸流タービン(reaction type axial flow turbine)(1) 反動度を適当に選ぶことにより、高いタービン効率を得るこ(ロ) 衝動形速度複式(カーチスタービン)(イ) 衝動形単段(ホ) 外向き半径流(ユングストロームタービン)(ヘ) 内向き半径流(ラジアルタービン)2―182とができる。(2) 流速を上げ遷音速タービンにすることによって、1段当たりの圧力比を大きく取ることができる。(3) ラジアル形に比べて容易に多段化することができ、全体として高い圧力比を取ることができる。(4) 各翼列、各段での圧力降下の配分、翼形状の改善、二次流れの制御、翼端すきまの制御などを反動度で制御できるので、さらに性能改善の余地が大きい。(5) 翼の材料耐熱温度以上にタービン入口ガス温度を高めることができる冷却タービン(cooled turbine)では、設計に自由度の多い反動形が適しており、高効率なガスタービンのタービンに多く採用されている。c. 外向き半径流タービン欠点が多いので現在ほとんど作られていない。d. 内向き半径流タービン(1) 反動度を50%付近で選定することが多く、小型でも効率が高い。(2) 構造上頑強であるから高い周速度にすることができ、1段当たりの圧力比を高くできる。(3) 構造が簡単なため軽量、安価である。(4) 軸方向長さを短くできるが、多段にすると流路形成が複雑になり効率が低下しやすい。(5) エンジンの効率向上の目的からタービン入口ガス温度を高くする要求があり、軸流タービンと同様、冷却タービンを採用する研究も行われている。また、自動車用など小型のエンジン用として、翼やケーシング材を耐熱セラミックスで作るセラミックタービン(ceramic turbine)の研究開発も行われている。
