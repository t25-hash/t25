軸流タービンの段負荷と段効率の関係が多いが、この高温化によって同一マッハ数でも実際の流速を大きく取ることができ、高圧化により流量を増すことができる。タービン内の流れは高亜音速となり、この高亜音速形タービンが最も広く用いられている。さらにタービンの性能向上を図る目的で、タービン部材をさらに高級な耐熱材料にしたり、翼面に耐熱用コーティングを施したり、高性能な冷却翼技術を採用したりすることで、タービン入口ガス温度を高める努力もつねに行われている。冷却タービンは製作費が高いので、1段当たりの仕事量(負荷)を大きくとり段数を減らすこと、温度降下を大きくして下流段での冷却の必要性を減らすことも重要である。また、航空用など多くのエンジンでは、冷却用としての空気はエンジンの構成要素である圧縮機から抽気するので、その分燃焼器やタービン初段に入る作動流体の量が減りエンジンの性能を下げる要因ともなっている。航空用エンジンを例にとると、現在、軸流タービンの断熱効率は負荷の高い高圧段でも90%に達し、低圧段で93%を超すまでになっている。この意味では今後の性能向上は極めて難しい状態にあるが、さらなるエンジンの軽量化、段数を減らしての製作コストの削減などを目指してタービンの1段当たりの負荷を増やす場合、断熱効率が低下する傾向があるので、高負荷化によって生じる性能劣化をいかに食い止めるかが今後の課題である。軸流タービンの最適設計においては空力的設計技術のみならず、冷却技術などの熱的な設計技術、用いる材料の選択や開発、翼の製作や加工技術、性能などの試験計測技術など、極めて広い分野の技術的知識が必要であり、これらを統合した計算機による対話型自動設計加工(CAD : com-puter aided design/CAM : computer aided manufacturing)手法が企業などでは一般的になっている。b. タービン段の設計手法従来から多くの設計は、作動流体が静翼および動翼を通過するときに生じる遠心力と、作動流体にかかる半径方向の圧力との釣合いから導かれる半径方向平衡条件(radial equilibrium)式を用いて行われてきた。この平衡条件を、ある仮定のもとでさらに単純化して導かれる設計法として自由渦形設計(free vortex design)法や流出角一定形設計(constantangle design)法などがあり、簡単なため現在でもよく用いられている。これらの方法では半径方向の反動度が内径側で低くなりすぎるので、反動度を制御できる方法として、コントロールボルテックス形設計(control vortex design)法がよく用いられる。これによると動翼内径側での反動度を上げ、全体としてタービン段の断熱効2―183率を高めることができる利点がある。反動度は静止翼列と動翼列での熱降下の配分割合を意味し、段全体の熱降下に対する動翼列での熱降下の比として、図III-4・3(b)を参照して次式で定義される。=--=(III-4・1)衝動タービンは反動度零、それ以外は反動タービンであるが一般に反動タービンと呼ばれるものの多くは0.5付近の値をとる。反動度を適切にとれば静止翼列と動翼列の両者で、後縁直前の翼背側(blade suction side)に発生する高速域での圧力上昇(減速域)を抑えることができ、流れのはく離などによる圧力損失を低くすることができる。航空用や発電用ガスタービンでは効率の良い反動タービンを一般に用いている。液酸液水ロケットの燃料ポンプ駆動用タービンのような特殊なものは、断熱効率をある程度犠牲にしても1段当たり出力の大きい超音速衝動タービンが使われることも多い。ところで、半径方向平衡条件をもとにした上記設計法は単純化した流体方程式を用いるが、詳細設計にはもっと複雑な形で流体方程式を計算機用にプログラムしたものが用いられる。例として流線曲率法(streamline curvature method)やマトリックス法(matrixmethod)などがある。これらはいずれも流体の粘性を無視した流体方程式(オイラー方程式、Euler’s equation)に基づいているので、粘性によって引き起こされる圧力損失などの算定はできない。そのためタービンの性能を求めるには、実験から求めた圧力損失のデータを導入することが多い。また、流量の算定をより正確にするため、境界層を翼面や翼端面の固定壁上で計算することも行われている。最近では、超高速の計算機が出現して、より完全な流体方程式(ナビエ・ストークス方程式、Navier-Stokes equations)を解く方法も実際の段設計や翼列設計に利用されている。しかし、粘性の影響を考慮するために取り入れている乱流モデルの不完全さのためか、現段階ではまだ、十分な精度で性能(損失)を予測できるまでには至っていないものが多い。この計算には膨大な計算時間と計算コストがかかるので、主として設計の最終段階で得られた翼の流れが妥当かどうかを細かくチェックする目的で利用することが多い。近い将来、計算機の計算速度がさらに高速になると多くのケーススタディが可能になり、実験データの蓄積による従来からの設計方法とは異なって損失に関する実験データなどは不要になる可能性がある。そして従来の翼形にとらわれない、自由で新しい翼列を有するタービンの設計開発が容易になる。c.
