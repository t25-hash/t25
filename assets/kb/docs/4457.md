# 4・2・2 構造

> **章**: 第4章 軸流タービンおよびラジアルタービン
> **節**: 4・2 軸流タービン

図III-4・8に軸流タービンの構造の例を示す。この例は産業用ガスタービンに採用されている多段タービン(multistage tur-bine)であるが、構造は蒸気タービンのそれと似ている。航空用と産業用ガスタービンのタービンは使用条件が異なるため、多少異なった構造をもつ。航空用では軽量化のため、高級材を用いて翼やその他の部品に対して複雑な加工が行われ、ケーシングも薄肉軽量構造にするのに対し、産業用は据置き型のため重量制限がなく厚肉で大型構造のものが多い。最近では、航空用のガスタービンエンジンが高効率である点を考慮し、たとえ高価なエンジンであっても開発費を節約できるので、産業用として既存の航空用ガスタービンを転用する例も多い。1段めの静翼は、図III-4・8に示すように外径側ケーシングと内径側ケーシングで支持されるが、2段め以降の静翼は外径側ケーシングから片持ち支持されている。動翼は回転軸に組付けたタービンデ図-4・9クリスマスツリー形動翼取付けィスクの外周端に、後述する種々の取付け方法で取付けられる。回転軸は軸受、ストラットを介してケーシングに支持され、圧縮機や発電機等を駆動するのに用いられる。まず静翼と動翼について述べる。最近では燃焼器に近い上流の高圧段の翼には、高温化に対応して中空冷却翼が採用されるものが多く、ロストワックス法による中空精密鋳造などで製造される。翼の支持構造は、いくらかの熱膨張が起こっても変形が許容できるように考慮される。静翼では翼を1個1個組込んで変形に対処するが、翼内外端壁(シュラウド)の接合部にすきまが生じ主流のガスが漏れたり、冷却翼では冷却空気が漏れたりするので、シュラウド部に漏れを阻止する薄板を挟んだりする。翼を数個まとめて溶接して接合部を少なくし、漏れを減少させるとともに、取扱い部品数を減少させることもよく行われる。一方、動翼では翼をディスクに取付ける方法として、ダブテール形、フォーク形やクリスマスツリー形(図III-4・9)などの翼植込み法があり、いずれも翼にかかる力を分散させる構造である。動翼のそれぞれの翼は静止時には容易に動くが、回転時には遠心力の引張り力で強固に固定される。また、動翼とディスクが一体になった形式のものもある。動翼先端と外部ケーシング間のすきまからの漏れ流れを防ぐため、翼自体の先端にシュラウドを設けることもある。この場合、翼の重量増加により翼の根元部やタービンディスクに大きな遠心力が作用する。初段動翼のような高温のものでは強度上に問題があるので、シュラウドを設けないことが多い。翼先端部の翼厚みを薄くして翼先端がケーシングと接触したときに、容易に翼先端部が摩滅するようにして動翼先端すきまを最少にすることも行われている。また、動翼の外周ケーシングの内壁に研削容易な材料を被覆して、接触時にケーシング内面を動翼端で削り取ることで接触時の動翼先端すきまを少なくすることも行われている。また、タービン動翼には遠心力と空力曲げ力が作用するため、動翼の重心線をわずかに半径方向からずらして遠心力の周方向成分と周方向に働く空力曲げ力とをバランスさせるよう考慮している。タービンディスクは動翼をディスクの外周に植込んで、動翼にかかる力を回転軸に伝えるものであるが、ディスクに作用する大きな遠心応力と、起動時などに生じるディスク内外径での温度差による熱応力とに十分耐える必要がある。特にガスタービンのように起動回数が多い場合、それに見合うサイクル(低サイクル)の疲労破壊に耐えることが要求されるため、十分な耐久確認試験が必要である。作動ガスが動翼の植込み部のすきまや静止部と回転部のすきまなどを通って軸受の潤滑油空間などに漏れないように、ラビリンスシール(labyrinth seal)やオイルシール(oil seal)などのシールリングを組込むなどの配慮も行う。図III-4・10に軸流タービンに多く用いられているラビリンスシールの例を示す。ラビリンスシールは一般にケーシング側に植込まれるが、回転するロータ側に植込むものもある。特に、高温高圧のタービン作動ガスが軸受部の潤滑油に漏れ込むと排油が困難となるので、高温に耐えるカーボンシールを用いることも多い。また、動翼にかかる力や漏れたガスの圧力などによりディスクにかかる軸方向の推力が、それを受け持つスラスト軸受の許容負荷値を越えないようにバランスピストン(balancing pis-ton)を設け、高圧空気により反対推力を発生させバランスをとる方策もとられる。2―185第III部気体機械図-4・10 航空用エンジンの高圧タービンの構造とラビリンスシール図-4・11 カービックカップリング多段タービンの場合、複数のディスクどうしの結合方法はボルト締めが一般的である。また、図III-4・11に示すようなカービックカップリングとの組合せもある。この場合、大きなトルク伝達と自動調心が可能であるが高価である。ディスクと回転軸との結合は焼きばめ、あるいはそれらの一体形が多い。内部、外部ケーシングは高温高圧に耐える必要があるが、そのうち外部ケーシングは特に圧力を、内部ケーシングは温度に耐えるように配慮して設計されることが多い。タービンケーシングおよび回転軸にはともに温度変化による軸方向の熱膨張が生じるが、互いの熱膨張差による相対変位は軸受の支持構造や位置で定まるので、構造設計の際にこのことを考慮する必要がある。また、両ケーシングの直径も熱膨張により変化し、特に外部ケーシングでは動翼先端とのすきまが増し、漏れによる性能劣化も増す可能性がある。このすきまを極力抑えるために外部ケーシングを冷却するものもある。外部ケーシングは分離組立が容易にできるように水平二分割の構造にすることが多い。航空用エンジンでは軽量化のためにケーシングを薄肉構造にするので、二分割するとさらに変形しやすい。これを防ぐため、高応力に耐える高級材料を用いてケーシングと分割用フランジ等の肉厚を極力薄くし、組付け用ボルトも細く、かつ本数を多くしている。ラジアルタービン(radial turbine)は図III-4・12に示すように、動翼(rotor blade)出入口流れの方向、ノズル(nozzle)羽根の有無、ガス入口孔数によって分類される。純半径流タービンは、混流式タービン(mixed flow turbine)に比べて効率が低く、圧力比も小さいため、現在では大部分の用途において混流式ラジアルタービンが用いられている。近年ではタービン小型化、大容量化の要求のため、混流式(または斜流式)タービンも用いられるようになってきている。
