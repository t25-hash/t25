# 4・3・1 性能

> **章**: 第4章 軸流タービンおよびラジアルタービン
> **節**: 4・3 ラジアルタービン

軸流タービンと異なり、半径方向の流れを主体として、膨張過程(図III-4・13)においてはタービン動翼入口から出口までの周速度差による熱落差を有することがラジアルタービンの最大の特徴である。膨張過程における周速度差による熱落差は全熱落差の30%以上にもなるので、1段で吸収されるエネルギーは軸流タービンより2―186流体機械大きくとることができる。このため、1段の圧力比を10以上にとっても高い効率が得られている。翼端すきま損失(tip blade)を設けることもある。
