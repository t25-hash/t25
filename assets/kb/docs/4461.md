# 5・1・2 風の変化

> **章**: 第5章 風車
> **節**: 5・1 風の特徴

風は大気中の空気塊のマクロな運動であり、太陽からのエネルギーの吸収度の差、すなわち温度差や圧力差、地球の自転による遠心力、コリオリ力(Corioris force)によって支配される。風力エネルギーを議論する場合が、風車羽根の強度、騒音などを議論する場合′が重要になる。フィールドでの時間平均速度は、大きさと方向が変化するが、それには(1)時間による変化、(2)地形の起伏、(3)垂直方向への変化が考えられる。図III-5・2は風速変動のスペクトルの実測例である。スペクトルの大きい値はほぼ、=0.02h(1~2min)と100h(4日)にあり、前者は大気の変動渦スケール、後者は気象の変化による風速の変化に対応している。また、=10min~2hの場合、スペクトルは小さい値を示し、風速の時間平均値を求める場合、この時間間隔の測定が妥当であることを示している。また、大気の運動ではガストと呼ばれる突風が生じる。風車の直径が年々増大し、その直径が100mにもなると、最上部と最下部では風速が異なり、翼の回転に伴ってその特性が周期的に変化する。垂直方向に風速が変化する境界層厚さは平地でおよそ2000mである。図III-5・3は代表的な垂直方向の分布の一例である。最下層部では地表による摩擦力で減速させられ、地表から離れた高度では、地球の自転によるコリオリ力の影響を受ける。エクマン(Ekman)層と呼ばれるこの地表に近い層では、地表から垂直方向に風の速度、方向が変化し、最上層では等圧線に沿った流れとなる。風速が大きい領域では乱流混合が促進されるため、地表近くの速度分布に及ぼす温度分布の影響は少なく、大気の流れは風洞で得られる乱流境界層とスケールは異なるが定性的には類似である。わが国で風車設置は、洋上、山岳地域、孤島など極めて複雑な地形の場所に限られ、洋上を除くと複雑地形での最適設置位置の決定、年間の総発電量予測が必要になり、このための高精度な風況モデルの開発が重要である。一般に二次元または三次元丘陵を過ぎる流れでは、地面から等しい高さにおける速度は排除効果のために頂上付近で大きくなるが、丘陵のこう配が急であると流れは壁面からはく離するため、起伏による速度変化の予測は困難である。二次元丘陵で図-5・4 測定値と対数速度分布との比較図-5・5 二次元丘陵での無次元増速値は斜面のこう配が1:3以下の場合には一般にはく離は発生せず、地面からある高さ以上での増分は境界層の影響が消えてポテンシャル流れから予測できる値とよく一致する。
