# 3・6・3 再生方法

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・6 次世代記憶装置

図3･54に再生方法を整理して示す．これらは図3･51に示す記録
方式とは必ずしも一対一の対応はしていない．まず，追記方式また
は再生専用（ROM）方式として，原子オーダの凹凸や原子クラス
タ（塊）については導電性探針を用いたトンネル電流検出方式が，
また絶縁材料を含めた凹凸に関してはAFM探針による表面形状の
図3･52高密度記録の基本方式
追従方式（片持ばりのたわみ検出）が用いられる．また，光の波長
を下回る超微小開口（aperture）近傍に定在する光の場（近接場，
near -field）を用いた再生 も可能である．
書換え方式のうち，導電率変化方式に関してはトンネル電流検出
と，導電性探針を接触しゅう動させての接触コンダクタンス検
出 の二つの原理が用いられている．誘電分極や電化蓄積方式で
は，導電性プローブ（探針）と対向電極となるp型半導体基板と
の間の空乏層の大小に起因する静電容量の変化を検出する ．光
磁気方式では，微小開口近在の近接場を点光源とする散乱伝搬光の
偏光回転 を，垂直磁気方式では磁気力こう配検出 などの原理
が用いられている．また，相変化媒体に透過率（反射率）変化とし
て記録されたピットは，微小開口による近接場の散乱・透過光強度
変化により検出できる．
