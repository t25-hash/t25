# 5・2・3 構造および風力発電関連技術

> **章**: 第5章 風車
> **節**: 5・11 に,オランダのデルフト工科大学風力エネルギー研究所で

図III-5・9には水平軸風車の全体図を示す.図は,ロータ(rotor),ナセル(nacelle),タワー(tower),基礎,電力系統連携装置(grid connection system)から成り立つ.図III-5・14に大型水平軸風車(large scale HAWT)の代表的ナセル構造2例を示す.図III-5・14(a)は通常の増速ギヤ(increased gear)をもつ大型水平軸風車で,図中にナセル構成要素を示す.図III-5・14(b)には,ギヤレス多極同期発電機使用大型水平軸風車の一例を示す.風力発電は一見単純に見えるが,総合エンジニアリングの世界である.基礎の土木工学から,タワーの建設工学,発電機および発電システムの電気工学,ナセルを構成する主軸(main shaft),増速機(increased transmission),ブレードのピッチ角制御(bladepitch control),風車のヨー制御(yaw control of wind turbine)等の機械工学的要素,ブレードの空気力学,回転翼としての空気力学が必要事項である.さらに,自然の風を利用するわけであるから,地表風の工学的評価が重要な課題になってくる.
