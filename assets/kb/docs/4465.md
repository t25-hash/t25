# 3・1・1 社会的背景

> **章**: 第3章 鉄道
> **節**: 3・1 鉄道一般

営業走行速度は，在来線の場合，最高速度から停止までのブレーキ距離（braking distance）が600m以下という決まりが以前あったため，それに対応して地上設備が作られており，このことが制約になって130km/hが最高速度である．
しかし，踏切のない区間では例外で160km/hで営業している区間がある．
新幹線では，現在300km/ hまで営業運転しているが，さらに他交通機関との競争力をつけるため，速度向上の技術開発を推進している．
この場合のいちばん高いハードルは環境対策である．
世界的に見ても鉄道の営業最高速度は，現在300km/hである．
鉄道とは一般に，左右2本のレール上を，鉄の車輪を用いて走行する車両により，人または物を輸送するシステムをいう．
索道，モノレール，無軌条電車（トロリバス）なども，特殊な鉄道として分類される．
イギリスで1825年に世界で初めて鉄道が開業して以来，線路と車輪を用いる構造にあまり変化はないが，システムとして鉄道を見た場合，それぞれの時代の最新技術を取り入れ，最も効率的な輸送機関として今日まで発展してきた．
その後，鉄道はヨーロッパ，アメリカで発展し，日本にはイギリスから輸入された．
