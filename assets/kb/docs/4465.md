# 5・2・4 諸外国の実例

> **章**: 第5章 風車
> **節**: 5・11 に、オランダのデルフト工科大学風力エネルギー研究所で

2000年に入って洋上風力発電がヨーロッパにおいて本格化し始
めた．洋上風力発電そのものは1990年代半ばになると建設が試み
られている．漁港の防波堤の内側に建設されたものや沖合2～3
km，水深5～10m程度に建設されたものとして，デンマークのビ
ンディビ，オランダのソルデムブリンクがある．この当時の機種は
単機出力600kW程度であるが，2000年に入るとデンマーク，コ
ペンハーゲン沖合ミッデルグルンデンに2000kW×20台（図Ⅲ-
5･15）という本格的な洋上風力発電所（oﬀ-shore wind park）が
建設された．
ヨーロッパ連合の計画によると，全ヨーロッパ連合の風力発電所
の40%を洋上に建設しようということである．洋上風力発電をき
っかけに，風車の大型化が加速されている．2500～4000kW程度
（直径80～100m程度）が実証機として建設され始めている．
