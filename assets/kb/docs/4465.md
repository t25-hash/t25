# 5・2・4 諸外国の実例

> **章**: 第5章 風車
> **節**: 5・11 に,オランダのデルフト工科大学風力エネルギー研究所で

2000年に入って洋上風力発電がヨーロッパにおいて本格化し始めた.洋上風力発電そのものは1990年代半ばになると建設が試み2―19 3第III部気体機械図-5・15 デンマークの洋上風力発電所(ミッデルグルンデン,2000kW×20台=4万kW,2000年)図-5・16 北海道苫前町の集合型風力発電所(5万2800kW,2000年)られている.漁港の防波堤の内側に建設されたものや沖合2~3km,水深5~10m程度に建設されたものとして,デンマークのビンディビ,オランダのソルデムブリンクがある.この当時の機種は単機出力600kW 程度であるが,2000年に入るとデンマーク,コペンハーゲン沖合ミッデルグルンデンに2000kW×20台(図III-建設された.ヨーロッパ連合の計画によると,全ヨーロッパ連合の風力発電所の40%を洋上に建設しようということである.洋上風力発電をきっかけに,風車の大型化が加速されている.2500~4000kW 程度(直径80~100m程度)が実証機として建設され始めている.
