周波数）に適合したシステムをもつことが必要である．日本の鉄道

類になっている．交流電化は電車線電圧を高くして，地上変電設備
を少なくすることが可能である反面，車両には変圧器や整流器など
を設置する必要がある．

g.客車（passenger car ) 客車は動力装置をもたない旅客車
で，走行するためには機関車の連結が必要で，性能的には電車と比
較して劣る面が多い．反面，機関車を取換えれば，電化区間・非電
化区間など地上設備の異なる線区へ直通運転をすることが可能であ
る．また，動力装置をもたないため，客室内の静粛性にとっては有
利である．このような点から，客車は寝台車などに用いられてい
る．
h.貨車（freight car ) 貨車は，客車と同様に動力装置をも
たず，機関車を取換えることで多線区に乗入れが可能である．屋根
を有する有がい車，屋根をもたない無がい車，液状貨物用のタンク
をもつタンク車，コンテナを積載するコンテナ車などがある．
