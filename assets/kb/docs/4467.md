# 3・1・3 基本性能

> **章**: 第3章 鉄道
> **節**: 3・1 鉄道一般

列車が走行する際の加速性能やブレーキ性能は，列車としての能
力である速度特性・引張力特性，ブレーキ力と，車両構造や地上設
備の条件から決まる列車抵抗とによって決定される．列車抵抗につ
いては，走行エネルギーを減らすために小さいほうが好ましい．特
に新幹線のような高速列車では消費エネルギー低減に大きな影響を
及ぼすため，先頭形状，車体の平滑化，重量軽減など新型車両検討
の際，重要な事項となる．
列車抵抗を理論式により求めることも可能であるが，実際の値と
なかなか一致しない．したがって，実際の車両を使用した試験によ
り走行抵抗を測定し，結果から実験式を作り，列車抵抗として使用
しているのが実態である．
a.列車抵抗（train resistance) 列車抵抗は，走行抵抗，こ
う配抵抗，曲線抵抗，出発抵抗，ずい道抵抗をいう．
.走行抵抗 列車の走行抵抗は，車輪とレール間の転がり
摩擦抵抗，車輪・駆動装置の軸受や歯車の摩擦抵抗，車体の空気抵
抗などがおもなものであり，速度を変数とする二次式で表される．
走行抵抗に関する理論式もあるが，一般的に使用されている式は次
のとおりである．
