# 5・3・2 変動荷重

> **章**: 第5章 風車
> **節**: 5・3 性能と出力の算定

大気中で作動する風車は常に非定常流れに直面しているが、特に短時間にその出力が変化する場合としては、風向が急に変化する場合(yawed inflow)と突風などで風速が急に変化する場合(dynamic inflow)、およびその両方が同時発生する場合が考えられる。図III-5・19は3枚羽根の水平軸風車において風向きが=0.0758sおよび3.0s間にφ=30°変化した場合の出力の過渡状態の変化を示す。個々の羽根は回転とともに迎え角の変化を受け、揚力が変化する。風車の出力は3枚の羽根が生じるトルクの和であるので、出力の変動周期は1/3回転である。角度φが変化した場合の静的な出力の低下は(cosφ)に比例するので、一定時間後はその値にまで減少する。一方、風速が変化する場合における動的変化を調べるには、一般に風速を一定にして羽根のピッチ角を変化させる方法が用いられている。図III-5・20は羽根のピッチ角がθからθに急に変化した場合の羽根の揚力、迎え角度の変化を示す。ここでは半径位置、Ωは風車の回転角速度である。図III-5・21には風速=8.7m/sでピッチ角が=0.0758sの間にθ=0.070°からθ=3.716°に変化した場合のトルク変動の、いくつかのシミュレーション結果を示す。これらの計算から変化直後には大きなオーバシュートが見られる。
