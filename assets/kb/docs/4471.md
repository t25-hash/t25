# 5・5・1 トルク制御(torque control)

> **章**: 第5章 風車
> **節**: 5・5 風車の制御

風車の出力制御には、風車ブレード全体のピッチ角を変えるフルスパンピッチ制御と風車ブレードの先端部分のみを可動構造とした翼端失速制御がある(図III-5・27)。もちろん、水平軸風車ブレードは基本的に固定ピッチであっても差し支えないが、風速変動に風車トルクを効率よくなじませていくためには、風速変動の平均値に合わせてブレード全体のピッチ角を変化させる方法が最適である。また、フルスパンピッチ制御は、ロータの回転数制御に威力を発揮する。翼端失速制御では、翼は翼全体のトルクのコントロールには役立たず、緊急時の翼車回転数増加を防止するブレーキの役割を担っている。固定ピッチのみであると発電機にかかる負荷変動量が大きくなり、あまり好ましくない。
