# 3・2・2 電気車の制御装置

> **章**: 第3章 鉄道
> **節**: 3・2 動力発生装置と制御装置

鉄道車両は速度範囲が広く，かつ，けん引重量の変化，線路条件
による所要けん引力の変化があるので，これに対応するために駆動
電動機の回転数と回転力を制御しなければならない．制御の方法を
定める大きな要素としては，駆動電動機に直流電動機を使うか交流
電動機を使うか，架線から受電する電力が直流であるか交流である
かの2点である．最近では新製される電気車の制御は三相誘導電動
機をインバータで制御する方法であるが，簡単に直流電動機を用い
た制御方法も述べる．
a.直流電動機の制御（DC motor control ) 直流（直巻，分
巻，複巻）駆動電動機は，原理的に電機子電流とトルク，誘起電圧
と速度が対応するので，一般に電流制御，電圧制御，界磁制御の三
つの基本制御方式の組合せによって制御される．電流制御は，起動
時や発電ブレーキ時など，所要トルク（したがって，所要加速度・
減速度）を制御するのに用いる．また電圧制御は，所要の速度が得
られるように電動機電圧を制御する．起動時は電流を制御しつつ速
度の増加とともに電圧を高めていき，発電ブレーキ時は同じく電圧
を低くしていく．界磁制御は，界磁のアンペアターン（界磁電流に
界磁巻線を乗じたもの）を変えることによりトルクや出力を制御す
る．

直流複巻電動機の分巻界磁を界磁チョッパにより制御する方法
や，直巻電動機のままで界磁添加励磁制御を行うことによる強め界
磁によって力行性能向上や回生ブレーキを行う例もある．
b.交流電動機の制御（AC motor control ) 半導体，特に大
容量GTO，IGBTの発達により可変電圧可変周波数（VVVF:
variable voltage variable frequency）インバータによる誘導電動
機が実用化されている．インバータ制御車の主回路を図3･16に示
す．この方式の特徴は，保守の大幅低減，駆動電動機の高出力化・
小型軽量化，粘着性能向上，省エネルギー，信頼性向上などであ
る．
(１) 保守性 直流電動機のようなブラシがなく，取換えが不
要である．整流子がないため，整流子の保守が不要．また，整流不
良によるフラッシュオーバが発生しない．制御装置の可動部分の保
守が大幅に削減できる．
(２) 小型・軽量化 整流機構がないため，回転数を高くでき
る小型化が可能．台車内に大容量モータを搭載できる．特に新幹線
のように大出力が必要な車両に有利となる．
(３) 粘着性能向上 空転，滑走制御が容易となり，期待粘着
性能を高くとれる．電動車比率を下げることができる．また，ブレ
ーキ時の電気（回生）ブレーキ負担を多くすることが可能となり，
省エネルギー率向上とともに，付随車の制輪子摩耗の低減が可能で
ある．
わが国の電気車で一般的な誘導電動機のインバータ制御について
説明する．誘導電動機は交流電源の周波数によってほぼ一定の回転
速度で回転しようとする．これの回転数，回転力を制御するために
は，可変電圧可変周波数（VVVF）の駆動電源が必要である．
