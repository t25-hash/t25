# 1・1・1 希薄気体流

> **章**: 第1章 真空機器一般
> **節**: 1・1 気体分子の運動

標準状態(0°C,1atm)の気体1cm中に含まれる分子の数(ロシュミット数、)は、=2.68699×10cm(IV-1・1)である。大気圧下では非常に多くの分子が存在し、これらの分子は頻繁に衝突を繰返している。したがって、分子は直線的に進むことができず、衝突のたびにその方向が変化する。分子が直線的に進むことのできる距離、すなわちある分子と衝突後、次に他の分子と衝突するまでに進むことのできる平均距離を平均自由行程(meanfree path,λ)と呼んでおり、次式で定義される。λ=2π(IV-1・2)ここで、は分子の直径、は数密度である。数密度は圧力、絶対温度と状態方程式=(IV-1・3)で関係づけられる。ここで、はボルツマン定数(Boltzmannconstant)である。これより数密度は圧力に比例し、絶対温度に逆比例する。したがって、数密度に逆比例する平均自由行程λは圧力に逆比例し、低圧になるほど大きい値となる。いま、空気を単一の気体と考え、ロシュミット数と空気分子の平均直径=3.7×10mから大気圧における平均自由行程を計算すると、6.1×10m(61nm)となる。希薄度を表す重要な無次元パラメータとして、次式のクヌッセン数(Knudsen number,)がある。=λ(IV-1・4)ここで、は流れ場の代表長さである。クヌッセン数は、例えば円管内の流れを考えたとき、流れ場の構造が気体分子の分子間衝突λ、または円管内壁との衝突(円管の直径)のどちらに支配されるかを表すパラメータとも考えられる。大気圧下のように分子間衝突が多い場合には、λが小さいためにクヌッセン数は小さくなり、真空機器内部のように分子間衝突が少ない場合は、λが大きくなるためにクヌッセン数は大きくなる。流れ場を気体分子の運動から解析する希薄気体流として取扱う必要のあるのは、後者のクヌッセン数が大きい場合である。数値的にはクヌッセン数が、>0.01(IV-1・5)になると流れを希薄気体流として扱わなければならず、気体を連続体として扱えなくなる。また、マイクロメカニクス内部のすきま流れのように、たとえ大気圧下でも流れ場の代表長さが小さければクヌッセン数が大きくなるため、希薄気体として扱わなければならない場合もある。
