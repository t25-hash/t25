# 1・1・2 速度分布関数

> **章**: 第1章 真空機器一般
> **節**: 1・1 気体分子の運動

速度分布関数は、次式のボルツマン方程式に従う。このボルツマン方程式は微積分方程式であるために、平衡状態を除いて解析的に解を求めることは不可能であり、一般に、直接シミュレーションモンテカルロ法(DSMC法、direct simulationMonte Carlo method)のような統計的手法が用いられている。希薄気体流では、気体分子の固体表面との衝突回数が気体分子どうしのそれに比べて無視できず、気体分子の固体表面における挙動が極めて重要となる。壁面におけるすべり速度・温度飛躍、半導体製膜過程における気体分子の付着確率などでは特に重要である。
