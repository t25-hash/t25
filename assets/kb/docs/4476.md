# 3・2・6 集電装置（current collector ）

> **章**: 第3章 鉄道
> **節**: 3・2 動力発生装置と制御装置

a.形状 車両の屋根上に設置され，架線としゅう動して集電
する装置はパンタグラフであり，床下に設置され，レールと平行に
設置された第三軌条としゅう動して集電する装置は集電靴である．
また，低速の路面電車では，屋根上にビューゲル，トロリポールな
どが用いられていた．パンタグラフの一例を図3･38に示す．
薄肉の鋼または軽合金のパイプで組まれた骨組の上部に集電舟体
およびすり板を取付けている．主ばねは，パンタグラフの機構全体
を押上げ，集電舟体を55N程度の力で架線に押付ける機能を有
し，集電舟体を支持するなびきばねは舟体の上下動，ピッチング動
作の緩衝を行い，架線への追随性を向上させる機能をもつ．パンタ
グラフ下げ動作は空気シリンダにより主ばねを縮めて行う．
パンタグラフの上昇動作は，電車に用いられる主ばねによるばね
上昇式と，電気機関車に用いられる空気シリンダによる空気上昇式
がある．
近年，装置の簡素化と部品点数の削減および新幹線電車の空力騒
音低減を目的として，シングルアームパンタグラフが数多く使用さ
れている．在来線に使用されているタイプを図3･39に示す．
b.性能 車両が高速で走行すると，架線振動，架線の高さ変
化，硬点や波状摩耗の存在，車両動揺などにより，架線と集電舟体
の間の接触力が変動し，離線が発生しやすくなる．離線はすり板の
摩耗を増加させ，電波雑音や騒音レベルを増大させるので避けなけ
ればならない．離線率の目安は，概略，交流集電で5 %，直流集電
で1%以下が望ましいとされている．パンタグラフの追随性は，も
っぱら集電舟体を支持する復元ばね系の特性で決まる．復元ばねが
二重系のものは，主ばねと合わせて三元系と称し，一重系のものは
主ばねと合わせて二元系と称する．三元系のほうが追随性に優れ離
線率も低い．なびきばねと舟体は架線に追随する振動系を構成し，
舟体の質量が小さいほど，追随性は向上する．
パンタグラフは架線のセクション等で衝撃的な力や，走行に伴う
空気抵抗や揚力が作用する．このような力に対して疲労破壊が生じ
ないような強度が必要である．また，トンネル内等で揚力が発生し
ても離線が増大しないような空力特性が必要である．
パンタグラフの消耗品の中で大きな比率を占めるのがすり板であ
る．大別して，カーボンすり板と鉄系の焼結合金すり板がある．カ
ーボンすり板は架線の潤滑も行うので，架線系のメンテナンスの観
点からは有益である．焼結合金すり板は摩耗量の少ない特徴があ
る．
パンタグラフは主回路の一部であり，大きな電流が流れるので電
気回路としても十分な通電容量を確保しなければならない．近年，
インバータ電車では，主電動機の出力が大きくなり，編成中の電動
車の数が少なくなっており，それに伴いパンタグラフの数が減り1
パンタグラフ当たりの集電電流が増加している．
