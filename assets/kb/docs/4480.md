# 1・3・3 オリフィスの自由分子流コンダクタンス

> **章**: 第1章 真空機器一般
> **節**: 1・3 コンダクタンス

図IV-1・4のように、二つの真空容器の間に小さなオリフィス[面積(m)]が設けられているとすると、オリフィスの単位面積に左右から単位時間に飛来する分子の体積は、分子の平均熱運動速度を(ms)、一般ガス定数を(JKmol)、絶対温度を(K)、ガスの分子量を(kgmol)とすれば14=2πとなるから、容器内の圧力をそれぞれ,(>)とし、オリフィスの左右から単位時間に飛来する分子の量を、圧力× 体積/時間で表してそれぞれ,とすれば、=2π,=2πとなる。したがって、オリフィスを通って左から右へ流れる流量は=-=2π-=-(IV-1・31)となり、コンダクタンスは=2π(IV-1・32)である。20°Cの空気についてのコンダクタンス율읉읍は율읉읍=116.4ms(IV-1・33)となる。
