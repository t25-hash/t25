a.粘着力と車輪滑走 粘着ブレーキでは車輪-レール間の摩
（図3･49）．最近の新幹線車両で測定された粘着係数の分布の例を，
先頭車両および中間車両に分けて図3･49に示す．
b.滑走再粘着制御（wheel slide readhesive control ) 車輪フ
ラット防止等のため，車輪滑走時に当該車輪のブレーキ力を一時的
に低め，車輪を再粘着させるため滑走再粘着装置（アンチスキッド
ブレーキ）が導入されている．日本における滑走再粘着制御は，新
幹線では1964年開業時の車両に，在来線では1987年651系特急電
車に装備されてから本格的な使用が開始された．
空気ブレーキ用滑走制御装置では，車軸端部や主電動機駆動装置
に取付けた速度検出器からの速度パルスをディジタル処理して滑走
を検知し，オンオフ制御による電磁給排気弁を2個組合わせた滑走
防止弁により，ブレーキシリンダ内の高圧空気の排気，保持，復帰
の動作を行う．電気ブレーキに起因する車輪滑走に対しては，主電
動機の端子電圧や電流値を比較する方式により，滑走が継続する
間，電気ブレーキ力を低下させる．
滑走制御（slide control ）の目標は，車輪-レール間の粘着力を
最大限に有効利用することであり，必要以上にブレーキ力を低下さ
せず，必要以上に長時間ブレーキを緩めないことにより，ブレーキ
距離を延伸させないことが必要である．このような方法の例とし
て，車輪の滑り率を5 %以下に制御して，粘着係数が高い状態で滑
走を維持させる，滑り率滑走制御方法 が，新幹線車両，在来線
車両に使用されている．また，ファジィ推論を適用した滑走制御方
法が実用化されている．
