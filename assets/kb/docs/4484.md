# 2・2・1 マクラウド真空計(McLeod gauge)

> **章**: 第2章 真空計測
> **節**: 2・2 真空計の構造および特徴

マクラウド真空計はボイルの法則に基づくもので、一定体積の気体を圧縮したときの圧力が、その圧縮比に比例することを利用している。図IV-2・1に一般に使用されているマクラウド真空計の構造を示す。測定しようとする圧力(Pa)の装置に真空計を接続し、あらかじめ真空に引いてあった水銀だめ容器にガスを導入して水銀を押上げると、Cの部分で一定体積の気体が分離される。さらに水銀を押上げ、B の毛細管に押し込めることにより、圧力、体積の気体が圧縮されたことになる。圧縮されたこの毛細管B 内の気体の圧力増加は、A とB との水銀柱の高さの差となって現れる。毛細管A およびB の断面積は等しく、(m)に製作されている。ただし、ρは水銀の密度(kg/m)、は重力の加速度(m/s)である。
