# 2・2・2 隔膜真空計(diaphragm gauge)

> **章**: 第2章 真空計測
> **節**: 2・2 真空計の構造および特徴

隔膜真空計は図IV-2・2に示すような構造で、十分に張力を与えられた金属製ダイアフラムと、セラミック基層上に厚膜蒸着した固定電極との間に生じる静電容量の変化を測定し、ダイアフラムの変位から圧力を測定するようになっている。ダイアフラムはインコネルで、インコネルあるいはステンレス鋼のケースに溶接されており、その両側に基準側と測定側の室をもち、基準側を永久封止し、ゲッタを利用して高真空(10Pa以下)を保持するようになっている。最大測定圧力が大気圧から100Paまでの各種があり、1台の測定圧力範囲は4けたないし5けたである。測定側はインコネルとステンレスの全溶接構造をもつため腐食に強く、半導体工業等の薄膜応用工業に多く用いられている。
