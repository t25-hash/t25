# 2・2・3 ピラニ真空計(Pirani gauge)

> **章**: 第2章 真空計測
> **節**: 2・2 真空計の構造および特徴

圧力変化による気体の熱伝導率の変化を利用した真空計で、図IV-2・3のようにフィラメントをホイートストンブリッジの一部に図-2・1 マクラウド真空計図-2・2 隔膜真空計図-2・3 ピラニ真空計第IV部真空機器(a) シュルツ型(b) 三極管型(c) B-A 型図-2・5 熱陰極電離真空計図-2・4 スピニングロータ真空計組込み、フィラメントの温度変化による抵抗変化を検出する。発熱体となるフィラメントは化学的に安定で、抵抗温度係数が大きくかつ温度によらず一定であることが望ましいので、白金線、あるいはモリブデン線に白金を被覆したものが使われる。ピラニ真空計には定電流型、定電圧型、定温度型があり、ブリッジの平衡をとる作業を手動に頼っていたころは、定電流型および定電圧型が主流であったが、ブリッジの平衡を自動的にとることができるようになった現在、次の利点のために、定温度型が普及している。すなわちフィラメント温度が一定であれば、フィラメント電流の二乗を圧力に直接読み換えることができ、さらにフィラメントの表面状態が一定に保たれるため適応係数の変動が小さく、精度の良い測定を可能にする。
