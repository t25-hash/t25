# 2・2・4 スピニングロータ真空計(spinning rotor gauge)

> **章**: 第2章 真空計測
> **節**: 2・2 真空計の構造および特徴

スピニングロータ真空計のセンサは,図IV-2・4に示すような非磁性体の短管(シンブル)の中に,フェライト系ステンレスのボール(ロータ)を入れたものである.このロータを管の外部から磁気により浮上させると同時に,回転磁場を加えて回転させる.一定回転数が得られたところで,回転磁場を取り去りロータを自由回転させると,回転数が気体の粘性力により低下する.ロータの回転減衰比は,球の磁気モーメントの水平成分を検出することにより求められる.減衰比は密度(圧力)に比例し,また気体の質量の平方根に比例する.したがってこの真空計は,気体の種類がわかると理論的に感度を求めることができる.さらに計算値が測定値とよく一致し,かつ経時変化が少ないなどの優れた特性をもつために,真空標準のトランスファ真空計(transfer gauge,真空標準の精度を国際間等で確かめ合うために移動させて比較を行う真空計)としても使われる.
