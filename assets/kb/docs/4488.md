# 2・2・5 熱陰極電離真空計(hot cathode ionization gauge)

> **章**: 第2章 真空計測
> **節**: 2・2 真空計の構造および特徴

代表的な熱陰極電離真空計は、シュルツ真空計、三極管型真空計、B-A 真空計(図IV-2・5)である。熱陰極電離真空計は、熱電子を放射するカソード(陰極)、電子を集めるアノード(陽極)、イオンを集めるイオンコレクタの三つの電極で構成される。カソードはふつう、高温に加熱されたタングステン線のフィラメントで、このフィラメントから放射された電子はアノードへ向かう途中で加速されて、高エネルギーを得る。電子は進路の途中にある気体分子と2―203衝突し、気体を電離させ、イオンにする。こうして生じたイオンは、負の電位にあるイオンコレクタに集められる。イオンコレクタを流れる電流の大きさは衝突分子数に比例するから、気体分子の数密度が測定できることになる。三極管型真空計が最も一般的に使われているが、その測定可能圧力の下限は、熱電子がグリッドに衝突する際に発生する軟X 線の影響で決まり、上限は気体の電離の際に発生する電子数が、熱電子数に比べて無視できなくなることで決まる。三極管型真空計における軟X 線の影響を減少させるためにイオンコレクタを細くし、管球の中心部に配置したのがB-A 真空計であり、陰極と陽極間の距離を短くして感度係数を下げ、電離電子の影響を減らしたのがシュルツ真空計である。質量分析器は、2・2節で述べたような全圧真空計では測定できない気体の成分や組成を計測するための分圧計である。ここでは、一般的に利用されている四重極型質量分析器について、その検出原理を概略する。四重極型質量分析器は、おもにイオン源部、質量分離部、イオン検出部で構成されている。イオン源部では、フィラメントから放出された熱電子を衝突させることで、質量分析器に導かれた気体分子がイオン化される。このイオンは、電圧を印加した2~3枚の金属プレートで構成される電界レンズによって加速・集束され、質量分離部へ入射する。質量分離部は、4本の平行電極(四重極)で構成され、対向する2本の電極に直流電圧と高周波電圧が印加され、他の対向する2本の電極にはそれらと逆極性の電圧が印加される。四重極の内部では、直流電圧と高周波電圧を調整することで、特定の質量を有するイオンのみが四重極の中心線周りをらせん運動しながら進み、それと異なる質量のイオンは四重極の外へ放出される。四重極によって選択されたイオンはイオン検出部へ入射し、二次電子増倍管によって分子数に比例したイオン電流として検出される。直接見ることのできない分子流れの全貌を可視化したり、流れ場の局所的な物理量を計測できれば、流れ場の構造や流れ場中で生起する諸現象の物理的な理解が容易になる。しかし分子流れは密度が低いため、密度変化に基づく屈折率変化を利用したシュリーレン法や、マッハツェンダー干渉法などによる流れ場の可視化は困難であり、分子それ自体からの発光や散乱を利用する方法が有効である。20世紀初頭にNaやCdの共鳴放射や空気、Ar,Heのアフタグローやグロー放電を利用した分子流れ(原子ビーム)の可視化が行われ、光学的な分子流れの可視化や計測が始まった。1960年代になると、分子流れの局所的な密度、温度、速度、分子の内部状態の定量的な計測の必要性から、エネルギーの高い準位へ励起された原子・分子の発する蛍光を利用する方法が提案され、励起源としてまず電子線が用いられた(電子線蛍光法)。レーザが比較的容易に入手できるようになると、励起源として電子線に代わってレーザが利2―204用されるようになり、レーザ誘起蛍光法やレーザ散乱法などの、空間・時間分解能の高い光学的可視化および計測法が提案された。特にレーザ誘起蛍光法を利用した分子流れの可視化および計測法は、レーザシートにより流れ場を二次元的にとらえることができ、しかも場に存在する特定の分子だけを選択的に励起・発光させて、その分子の内部構造に関連した非常に多くの情報が得られることから、近年急速に発展した。本節では、分子流れの代表的な可視化および光学的計測法として、アフタグローおよびグロー放電法、電子線蛍光法、レーザ誘起蛍光法の原理、実験装置、可視化・計測例、問題点等について概説する。
