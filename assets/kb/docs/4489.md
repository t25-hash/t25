# 2・4・1 アフタグローおよびグロー放電法

> **章**: 第2章 真空計測
> **節**: 2・4 分子流れの可視化

アフタグロー(afterglow)は高周波電場中での気体の化学反応に伴う発光であり、分子流れの可視化に古くから利用されてきた。空気流または窒素と数%の酸素の混合気体流に高周波電場を印加すると、NとOが解離してNOを形成し、さらにこのNOがOと反応して励起状態のNOを生じる。励起状態のNOは基底状態に戻る際に蛍光()を発し、これを利用して分子流れの可視化ができる。O+NO NONO+(IV-2・3)OとNOとの衝突頻度が増すほど蛍光は強くなるから、高密度の場所では蛍光が強く、低密度の場所では弱くなり、密度場の可視化ができる。ArやHeの流れに高周波電場を印加しても気体が発光する。これはグロー放電(glow discharge)と呼ばれ、比較的発光強度が強いことから分子流れの可視化に利用されている。
