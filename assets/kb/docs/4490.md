# 3・5・5 操舵台車

> **章**: 第3章 鉄道
> **節**: 3・5 走行性能と走り装置

従来の台車では，蛇行動安定性と曲線通過性能との両立を図るも
のの，急曲線での転向性能は十分でなく，著大な横圧による軌道狂
い進み，車輪フランジやレールの摩耗，きしり音等を生じていた．
そこで操舵性能改善を目的として，従来と異なる構造の台車が開発
され，実用化されている．この操舵台車には，主として輪軸の自己
操舵機能を最大限に引出す自己操舵台車（self -steering bogie）と
強制的に輪軸の舵をとる強制操舵台車（forced steering bogie ）の
二つのタイプがある．前者の代表例として，2輪軸をクロスアンカ
リンクで結合し軸箱を柔支持とした貨車用シェッフェル台車や，軸
箱前後支持剛性を前軸柔支持・後軸剛支持とした前後非対称方式台
車があり，後者の例としてボギー角連動強制操舵台車がある．図
3･67にこれらの構造を示す．軸箱柔支持のボギー角連動操舵台車
では，車輪踏面こう配を大きくしたほうが，むしろ蛇行動安定性が
