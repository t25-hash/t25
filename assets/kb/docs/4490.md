# 2・4・2 電子線蛍光法

> **章**: 第2章 真空計測
> **節**: 2・4 分子流れの可視化

電子線蛍光(EBF: electron beam fluorescence)法は,1960年ごろから分子流れの非接触な光学的計測法として利用されてきた.数十keV の電子線を低密度流中に入射すると,電子が気体分子に衝突して気体分子が励起され,気体分子が蛍光を発する.この蛍光を利用した分子流れの可視化や,蛍光の解析による流れ場の密度や温度の計測例が報告されている.電子線蛍光法による密度および温度計測のための実験装置の基本構成を,図IV-2・6に示す.電子線を発生させるためには電子銃と電子線コレクタ(ファラデー箱)が必要で,電子銃内は10Pa以下の真空に保たれなければならない.蛍光はレンズを通して,分光器または光電子増倍管によって検出される.Nの場合は,基底準位(NX∑g)のN分子に電子線が照射されると,N分子は励起および電離されてNイオンの励起準位(NB∑u)に遷移し,この励起準位のNイオンが,その基底準位(NX∑g)に遷移する際に蛍光を放射する(firstnegative system).
