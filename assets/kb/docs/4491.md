# 2・4・3 レーザ誘起蛍光法

> **章**: 第2章 真空計測
> **節**: 2・4 分子流れの可視化

図IV-2・7に示すように,分子にその共鳴周波数に一致するレーザ光を照射すると,分子は基底状態から励起状態へと遷移し,励起状態から基底状態へ遷移するときに蛍光としてエネルギーを放出する.この蛍光をレーザ誘起蛍光(LIF: laser induced fluorescen-ce)と呼んでいる.一般に,2準位モデルを用いた場合(ただし,レーザ光の線幅が広く,飽和していない場合)の蛍光強度は次式で与えられる.図-2・6 電子線蛍光法の実験装置概略=욡욢욢욡욢욡+(IV-2・4)ここで,:定数,:レーザ光強度,:観測体積,:蛍光を発する分子の数密度,욡욢:励起状態への遷移確率,욢욡:自然放射の遷移確率,:衝突(無放射)失活の確率である.また,:基底状態における振動・回転準位への分布関数である.式(IV-2・4)から蛍光強度は数密度に直接比例するように思われるが,が圧力と温度,が温度の関数であるため,は圧力と温度の複雑な関数となる.また,は場に存在する分子種にも依存するため,容易に見積もることができない.したがって,から直接に密度,圧力,温度を計測することは困難である.分子流れの可視化にレーザ誘起蛍光法を利用する場合,蛍光発光させる気体分子を励起するのに必要なレーザ光源が存在し,その気体分子が強い蛍光を発することが必要なため,一般にトレーサが作動気体に混入される.Naやハロゲン系のI,Br,Clのような2原子分子は可視光領域に吸収線があり,比較的強い蛍光を発する.なかでもIはその使用の容易さから,トレーサとして分子流れの可視化に多く用いられている.これは,Iが適当な蒸気圧[20±1°Cで26.6Pa(0.2Torr)]を有すること,可視域に多くの吸収線があり,アルゴンイオンレーザ(514.5nm)により容易に強い蛍光を発すること,比較的広い圧力範囲[0.1Pa~0.15MPa(1.0×10~1.1×10Torr)]にわたって可視化に利用できること,超音速流れにおいても比較的よく作動気体に追随すること(ただし,Heのように軽い分子を作動気体として用いる場合は,質量分離を起こすことがある),Iに関する分光学的性質が詳細に調査されていることに基づいている.ただし,Iには強い腐食性があるため,装置設計には注意が必要である.図-2・7 レーザ誘起蛍光法の原理図-2・8 Iのレーザ誘起蛍光を用いた超音速自由噴流の可視化実験装置図-2・9 Iのレーザ誘起蛍光による超音速自由噴流の可視化像および構造模式図作動気体中に混入したトレーサ分子のレーザ誘起蛍光を利用した分子流れの可視化は,上述したようなトレーサ分子の腐食性および作動気体への追随性の問題に加えて,非平衡になりやすい分子流れにおいては,トレーサ分子が作動気体に緩和しているかどうか注意する必要がある.作動気体を直接蛍光発光させることができれば,この問題は解決される.最近,真空紫外の波長可変レーザが入手できるようになり,Oの蛍光が観察できるようになった.Oの蛍光を利用した可視化や計測は比較的高温(1000K 以上)に限られて2―205第IV部真空機器いたが,冷却型CCD カメラやイメージインテンシファイアの開発により,室温程度でも可視化が行えるようになった.Oはその励起状態において強く前期解離させられ,これに律速されて非常に短い時間(psオーダ)の間に基底状態へ戻る.したがって,衝突失活を無視することができ,高精度な計測が期待される.このような性質を有する蛍光をレーザ誘起前期解離蛍光(LIPF: laser in-duced predissociation fluorescence)と呼んでいる.Iのレーザ誘起蛍光を用いた超音速自由噴流の可視化実験装置の一例を,図IV-2・8に示す.この例では作動気体としてArを用いている.作動気体は可変リークバルブを通して貯気室(6.7×10~4.0×10Pa)へ導かれ,ここで気化したIと混合する.この混合気体は,膨張室内(40~400Pa)のオリフィス(直径約0.5mm)を通して噴出し,超音速自由噴流を形成する.Iの励起には,エキシマレーザ励起の波長可変な色素レーザ(色素としてCoumar-in500を使用)を利用している.なお,レーザ光を遮断し,蛍光のみをカメラに入射させるために,シャープカットフィルタを使用している.図IV-2・9にIのレーザ誘起蛍光によって可視化された超音速自由噴流の全体像,およびその構造模式図を示す.超音速自由噴流は,膨張室圧力と平衡した噴流境界内に,オリフィス出口端から発生した膨張波が噴流境界で反射した圧縮波と重なり合って形成された,たる形衝撃波(barrel shock)を有する構造となる.また,たる形衝撃波はマッハ反射を起こして,垂直衝撃波(Machdisk,マッハディスク)と反射衝撃波を形成する.このように,衝撃波構造が明りょうに可視化されている.イメージインテンシファイア付きのCCD カメラを使用すれば,時間的に変動する流れ場を観察することも可能である.また,レーザ誘起蛍光を用いた温度,密度,圧力,速度の計測法も数多く提案されている.文献→
