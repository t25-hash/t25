# 3・5・6 特殊な台車

> **章**: 第3章 鉄道
> **節**: 3・5 走行性能と走り装置

一体輪軸ではなく，左右車輪が独立に回転する独立車輪（in-
dependently rotating wheel ）を用いた台車も実用化されている．
独立車輪は，前後クリープ力による自己操舵機能がないため，蛇行
動の動的な安定性に優れているが，曲線通過性能の確保に操舵機構
を付与する必要がある．左右車輪の接触角差による重力復元力を利
用し静的安定化を図った踏面形状も提案されている．図3･68にス
ペインのタルゴ客車を示す．この台車は，一軸独立車輪の連接台車
（articulated bogie）・ボギー角連動操舵台車で，車体傾斜，軌間可
変の機構を有する．LRV（light rail vehicle）では，低床化のため
に独立車輪が広く採用されている．
