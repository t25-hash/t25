# 3・6・2 車体振動および車内騒音

> **章**: 第3章 鉄道
> **節**: 3・6 車体

車両の高速化に対する構体の軽量化とLCAの追求のためにアル
ミニウム合金製構体を多用するようになった．その結果，乗客の快
適性，特に静音性，制振性を低下させることになり，設計段階でそ
の予測とともに対応策を盛り込むことが重要になってきた．
a.車内騒音の伝ぱ経路 図3･81に車内騒音の伝ぱ経路
（propagation path of noise）の模式図を示す．車両内の乗客の耳
元に届く騒音は，加振源で発生した振動が空気中を波動として伝わ
る空気伝ぱと，壁体の中を振動として伝わり最終的に壁体表面から
空気中に波動として放射される固体伝ぱとに分類される．実際の騒
音の伝ぱはこの空気伝ぱと固体伝ぱを複雑に繰返しながら伝わり，
最終的には車体内装の表面からと空調の吹出し口・吸込口とから波
動として放射される空気伝ぱに集約される．加振源としては台車・
床下機器の振動と振動音およびファン流体音，空調・換気装置のダ
クト流体音，レール振動音，軌道反射音，床下の凹凸による流体
音，および構体表面の乱流境界層による流体加振と，トンネル等の
地上設備からの反射音による音響加振とがある．
b.音源寄与度解析の事例と対応策 図3･82 に車内騒音に
対する音源の寄与度解析（analysis of individual noise source
contribution to over all noise）の事例を示す．この寄与度解析は
車内騒音のO.A. 値に対し，車内のどの放射面からの空気伝ぱが支
配的かを割出し，その支配的な壁体に対し有効な遮音対策を施し
て，少ない質量投資により最大の遮音効果を得ようとするものであ
る．
