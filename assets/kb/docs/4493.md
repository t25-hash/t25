# 3・6・2 車体振動および車内騒音

> **章**: 第3章 鉄道
> **節**: 3・6 車体

車両の高速化に対する構体の軽量化とLCAの追求のためにアル適性，特に静音性，制振性を低下させることになり，設計段階でそa.車内騒音の伝ぱ経路 図3･81に車内騒音の伝ぱ経路元に届く騒音は，加振源で発生した振動が空気中を波動として伝わ空気中に波動として放射される固体伝ぱとに分類される．
実際の騒最終的には車体内装の表面からと空調の吹出し口・吸込口とから波床下機器の振動と振動音およびファン流体音，空調・換気装置のダ音，および構体表面の乱流境界層による流体加振と，トンネル等のb.音源寄与度解析の事例と対応策 図3･82 に車内騒音に配的かを割出し，その支配的な壁体に対し有効な遮音対策を施しる．
