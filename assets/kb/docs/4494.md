# 3・1・3 排気速度

> **章**: 第3章 真空ポンプ
> **節**: 3・1 真空ポンプの性能の表し方

真空ポンプの吸気口を単位時間に流れる気体の体積を、排気速度(pumping speed)という。一般に真空ポンプの排気速度は気体の種類によって異なるので、図IV-3・2のように気体の種類を指定して表示する。図-3・1 流量と吸入圧力図-3・2 排気速度(ターボ分子ポンプの場合)真空容器として以前は、小型のガラス製容器が多く使われたが、近年装置が大型化するにつれて、ステンレス鋼やアルミニウム製の金属のものが増えてきた。図IV-3・3に示すような、漏れや壁面からのガス放出のない完全な容器を、排気速度(m/s)のポンプでコンダクタンス(con-ductance)(m/s)の配管を通して排気する場合を考える。この点からを実効排気速度(effective pumping speed)という。図-3・3 完全な真空容器の排気図-3・4 実効排気速度第IV部真空機器図-3・5真空ポンプの種類と作動圧力範囲大気圧から高真空までを1台で作成可能な真空ポンプも開発されてはいるが、それでも一般的には、高真空(high vacuum)、超高真空(ultrahigh vacuum)および極高真空(extreme high vac-uum)を得るためには、いくつかの真空ポンプを直列に組合わせて使用している。これらのポンプは排気機構、作動圧力、構造により分類することができる。排気機構によって分類すると、大きく二つに分けられる。第一は気体を大気側へ輸送する気体輸送式ポンプ(exhaust pump)であり、第二は気体をポンプ自体の中に物理的に収着、あるいは化学的に吸着させて溜め込む気体溜め込み式ポンプ(capture pump)である。気体輸送式ポンプは、さらにある容積に閉じ込められた気体を機械的に大気側に輸送する容積輸送式ポンプ(positive displace-ment pump)と、気体分子に運動方向を与え、流れを作って大気側に輸送する運動量輸送式ポンプ(kinetic vacuum pump)に分けられる。また、作動圧力範囲から分けると低真空ポンプ、中真空ポンプ、高真空ポンプ、超高・極高真空ポンプとなる。それぞれのポンプの作動圧力範囲は図IV-3・5に示すとおりであるが、真空ポンプの性能を表す重要なもう一つの尺度として真空の質、すなわち作成された真空空間の残留ガスの種類がある。油蒸気の分圧が低いほど質の高い真空とされ、真空中にさらされた物体の表面が、油蒸気分子に覆われずきれいなまま長時間保たれるので、このような真空を清浄真空(clean vacuum)と呼ぶ。清浄真空を作成するためのポンプとして、ターボ分子ポンプ、スパッタイオンポンプ、クライオポンプ、ゲッタポンプがあり、各種ドライポンプもその目的のために開発されたものである。
