# 3・4・1 油回転ポンプ

> **章**: 第3章 真空ポンプ
> **節**: 3・4 真空ポンプの構造と特徴

油回転ポンプ(oil sealed rotary vacuum pump)は、ある容積中に飛来した気体分子を閉じ込め、機械的に大気側に輸送することによって排気するもので、基本的な補助ポンプ(粗引きポンプともいう)として広く用いられてきた。図IV-3・6に示すようにカム形、回転翼形、揺動ピストン形がある。カム形は、円形ケーシング内壁に沿って回るロータによりケーシング内壁、ロータおよび上下に運動する仕切板で囲まれた室の体積が変わり、吸込み・吐出しを行う。回転翼形は、ケーシング内で偏心した位置を中心としてロータが回ると、ロータの溝に取付けられた2枚の羽根板がケーシング内壁に沿ってしゅう動し、吸込み・吐出しを行う。揺動ピストン形2―207(b) 回転翼形(a) カム形(c) 揺動ピストン形図-3・6 油回転ポンプ図-3・7 水封式ポンプは、ケーシング中心を軸として回転する偏心ロータがあり、ロータ周囲を滑動する円筒には弁が固定されている。ロータの回転により円筒は、ケーシング内壁とわずかなすきまを保って周方向に移動しながら、弁を上下運動により開閉させ、吸込み・吐出しを行う。油回転ポンプの到達圧力は油の蒸気圧で決まるので、蒸気圧の低い油を使用する必要がある。
