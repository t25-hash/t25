# 3・4・3 ドライポンプ

> **章**: 第3章 真空ポンプ
> **節**: 3・4 真空ポンプの構造と特徴

現在市場に提供されているドライポンプ(dry pump)は,軸受等に潤滑油を使用しているので,厳密な意味でドライとはいえないが,油蒸気やミストがガス通路に入らないように種々の工夫がなされている.ドライポンプには,容積移送式のルーツ形,スクリュー形,スク2―208ロール形,往復ピストン形,ダイアフラム形と,運動量輸送式のターボ形がある.a. 多段ルーツポンプルーツポンプ(roots vacuum pump)は,送風機として使用されているルーツブロワと全く同じ構造・原理で,古くから真空ポンプとして使用されてきた.かつては単段ないし二段で,油回転ポンプのブースタとして使われることが多く,メカニカルブースタ(mechanical booster pump)とも呼ばれていた.もちろん現在も,このような目的でも使用されている.新しい多段ルーツポンプは,一対の軸上に3~5組のルーツロータが配置されている.図IV-3・8に三段ルーツポンプの構造を示す.各ロータは壁で仕切られており,前段の排気口が次段の吸気口に直列に接続されている.ロータの直径は同じで,ロータ幅が高圧側ほど狭くなっている.また圧縮比を増すために三葉のロータがしばしば用いられるが,二葉の場合に比べ,排気速度は若干低下する.この形の真空ポンプのオイルフリー度は,b項で述べるスクリュー形と同様,軸受とタイミングギヤ部の潤滑油の逆流をいかに防ぐかによる.大部分の多段ルーツでは,ポンプ室とギヤボックスの間はラビリンスシールでシールされている.よりいっそうオイルフリー度を高めるために,ギヤボックスを別の真空ポンプで排気する機種もある.三~六段で大気圧からの到達圧力1Pa,排気速度60~500m/h程度が一般的である.また最近は,ガス温度の上昇を抑えるために冷却されたガスの一部を作動室の中間に逆流させたり,逆にガス温度を上昇させるために,対になるロータの一方の形状を変えて,吸入口から吐出し口に向かい体積が縮小するように設計されたものなど,用途拡大に対する対応が見られる.b. スクリューポンプスクリューポンプ(screw vacuumpump)には,従来圧縮機として使用されてきたリショルム形(Lysholm type)を真空ポンプに作り替えたものと,最近開発された矩形溝スクリュー形(クインビー形,Quimby type)とがある.. リショルム形リショルム形は,図IV-3・9(a)のような特殊なねじ溝形状を採用することにより,ねじリード角を大きくし,大きな閉込め体積を実現している.そのため矩形溝スクリュー形より排気速度は大きいが,ねじ巻き数が矩形溝スクリュー形に比較して少ないため,到達圧力は高くなっている.単段で大気圧から図-3・8 三段ルーツポンプ(b) 矩形溝スクリュー形(a) リショルム形1Paオーダの到達圧力が得られる.排気速度60~120m/h程度のものが一般的である.. 矩形溝スクリュー形図IV-3・9(b)のように矩形断面の右ねじ・左ねじ一対のロータを,タイミングギヤを介して非接触で高速回転させ,気体をロータ上端から吸込み,ケースとロータで仕切られたねじ溝内に閉じ込め,二つのロータのかみあいにより次第に排気側へ移送する.ロータは,一条ねじならばロータ1回転につき1回排気を行う.ロータとケース内面は一種のねじシール構造をなしているので,漏れを小さくすることができ,高圧縮比が得られる.圧縮比および排気速度はねじのリード角,ロータとケースの間のすきま,ねじのかみあい率,溝のアスペクト比,ねじの巻き数,ロータ直径および回転速度の関数となっている.リショルム形よりねじ巻き数が多いため,高圧縮比が得られるが,排気速度はリショルム形より小さい.大気圧からの到達圧力10Pa台,排気速度40~80m/h程度のものが市販されている.スクリューポンプは流路が他の機種に比べて簡単で,粒子の堆積がしにくく,粒子生成を伴う半導体製造に適しているため,最近開発が急速に進んでいる.c. スクロールポンプ(scroll vacuum pump)図IV-3・10に示すように,一方のスクロールを固定し,他方を自転させずに一定の半径で旋回させると,二つのスクロールに囲まれた三日月状の体積は,外周から回りながら縮小しつつ中心に移動し,中心の排気口から排気される.連続的に吸入,圧縮,排出の各行程が進行するので,駆動トルクの変動が小さく,騒音,振動も小さくすることができる.また隣接する三日月状体積の圧力差が小さく,スクロール側面は接触式(チップ)シールにより,ほぼ完全にシールされているので漏れが小さく,高い圧縮比が得られるという構造上の利点がある.チップシールのすべり速度は1m/s以下と小さく,油等による潤滑なしで長時間の使用にも耐える.また重量/排気速度が,他の機種に比べ小さいことも利点である.対になっている二つのスクロールとも回転させる回転式スクロールポンプもあるが,二つのスクロールの相対的な運動は,一方のスクロールを固定した旋回式と同じである.しかし,旋回式に必要な回り止めピン(吸入側にあり,グリースで潤滑されている)が不要のため,いっそう完全なドライ化の可能性がある.スクロール式は構造的に中小規模流量のポンプに適しており,排気速度は20~50m/h程度,大気圧からの到達圧力は10Paである.d. 往復ピストンポンプ(reciprocating piston vacuum pump)回転形機構の最大の利点は高速で運転でき,その結果高排気速度が得られることであるが,ドライポンプではロータの冷却が困難である.その点,往復ピストン形はピストンとシリンダの外部表面が大気に接しているため,容易に大気によって冷却でき,熱発生の原因となる駆動機構も大気中にあるので有利である.四段往復ピストンポンプのピストンの配置を図IV-3・11に示す.このポンプは5×10~5×10の圧縮比をもち,大気圧から1Pa台の到達圧力を得る.図-3・10 スクロールポンプ図-3・9スクリューポンプ排気速度は24m/hである.ピストンは段付きの設計で,全長にわたって強化PTFE(polytetrafluoroethylene)をベースにした複合材料でライニングされている.シリンダ表面は硬化処理が施されていて,真空に面した表面は潤滑されていない.段付きピストンによりリップシールの周りの漏れを減らし,次の段によるピストン背部の排気を容易にしている.排気速度を増すために吸入側の二つのピストン(ピストン1とピストン2)は並列に接続され,4番めのピストンは両側作動になっている.このポンプの最大の特徴は,他に類を見ないオイルフリー度であるが,その反面,粒子の侵入に弱いという欠点をもっている.e. ダイアフラムポンプ(diaphragm vacuum pump)図IV-往復動させ,ダイアフラムの動きと連動して作動する弁により,吸・排気を行う真空ポンプである.弁が無潤滑で作動すれば,完全なオイルフリー真空が得られる.構造上から振動を低くすることは難しいが,低騒音化が進められている.到達圧力は四段で40Pa程度,排気速度は数m/hである.f. ターボポンプ(turbine vacuum pump)a~e項のドライポンプはすべて容積輸送式で,到達圧力が10Pa以上にとどまっているのに対し,この形は運動量輸送式で,大気圧から10~10Paまで排気可能である.図IV-3・13のように,高真空側に円板形(ジークバーン)分子ドラグポンプ(molecular drag pump)を二段配し,渦流要素を八段下流側に直結したもの,および円筒形(ホルヴェック)分子ドラグポンプと渦流要素を組合わせたものがある.渦流要素はかつて浅井戸用などのポンプ要素として使用されており,この要素をもつポンプは再生ポンプとも呼ばれていた.図IV-3・13に示したものには,外形寸法が同じで回転速度が異なる90,360,540m/hの3機種があり,到達圧力はいずれも1.2×10Paとなっている.しかし,大部分の容積輸送式の排気速度が大気圧まで低下しないのに対し,ターボポンプは吸気口圧力が数Pa以上で排気速度が極度に低下する.これは高圧において円板形要素の圧縮能力が急激に低下し,ポンプ全体の排気速度が渦流要素図-3・11
