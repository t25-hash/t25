# 3・7・2 自動列車制御装置（ATC:automatic train control ）

> **章**: 第3章 鉄道
> **節**: 3・7 運転保安システム

地上からの信号・速度情報により，自動的に制限速度以下にブレ
ーキをかける列車制御装置である．しかしながら，この装置は，あ
る区間内に制限速度以下になるようブレーキをかけたり，緩めたり
するものであり，加速する機能はもっていない．加速機能を加え，
規定の速度で運転する装置は，3･7･3項で説明する自動運転装置と
なる．ATCは，JRの新幹線や大都市の通勤線区および民間鉄道
等で採用されている．信号機を地上に設置するものと車上に設置す
るものがある．地上信号方式は，民間鉄道で多く採用している方式
であり，通常運転士は，現示速度以下で走行するので，ATCで自
動的にブレーキはかからず，速度超過した場合に，ATSと同様に
ブレーキ操作のバックアップをする．一方，車上信号方式は，新幹
線等で採用されており，信号現示が車内（運転台）に表示され，
ATCが自動的にその速度以下までブレーキをかけるもので，機械
優先のブレーキ方式となる．
速度制御の方式としては，速度制限をいくつかの段階に分けて，
つねにその区間の制限速度以下で運転するべく制限速度を超過する
と減速制御する多段ブレーキ方式と，速度制限パターンにより減速
制御する一段ブレーキ制御方式がある．多段ブレーキ制御方式は停
止までの閉そく区間において，閉そく区間ごとに順次制限速度を設
定する従来型の制御方式であり，広く用いられている．しかしなが
ら，先行列車と後続列車の間隔を広く確保するため，運転時隔（先
行列車と後続列車の運転可能な時間間隔）の短縮が図りにくい．例
えば山陽新幹線では300，275，230，170，0の基本速度段をもって
おり，最高速度から停止まで減速するためには，4段階のブレーキ
区間が必要であり，そのブレーキ距離も相当なものとなる（図3･
91)．一方，一段ブレーキ制御方式は，中間の速度段を省略して先
行列車が在線する閉そくまでに停車する減速制御方式であるため，
多段ブレーキ方式に比べ，ブレーキ距離の短縮が図られ，運転時隔
の短縮，高密度運転が可能となる．最近，ディジタル伝送の一段ブ
レーキ方式のものが開発されている．この方式では，従来アナログ
信号を地上から伝送し，周波数の違いを検知して車両で信号速度段
を分別する方式に変わり，地上より停止目標までの位置をディジタ
ル信号により伝送し，車両側の速度制御パターンと比較してブレー
キ制御を行うものである．
