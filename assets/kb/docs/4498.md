# 3・4・4 蒸気噴射ポンプ(vapor pump)

> **章**: 第3章 真空ポンプ
> **節**: 3・12 に示すように,クランク機構により弾性体のダイアフラムを

a. 油拡散ポンプ油拡散ポンプ(oil diffusion pump)の構造を図IV-3・14に示す.油拡散ポンプは,蒸気を噴出するノズル,および噴射された蒸気を凝縮させる低温壁面から成っている.細いノズルから超音速で噴出した蒸気分子は,噴流の方向に極めて大きな速度成分をもっている.その噴流の中に飛び込んだ気体分子は,重い油蒸気分子と衝突すると,蒸気分子のほうが運動量は1けた程度大きいので,蒸気の噴流にはね飛ばされる.その結果,気体分子は吸気口から排気口に流されるため,ポンプ作用が行われる.1980年以前は,高真空排気システムは油拡散ポンプと油回転ポンプで構成するのが一般的であったが,薄膜応用工業等においては,油蒸気の逆流および油の短時間における劣化が問題となり,ターボ分子ポンプとドライポンプの組合せに置換えられて,使用台数は減少している.しかし,安価なため,油蒸気が問題とならない用途に根強い需要がある.b. エジェクタポンプエジェクタポンプ(ejector vacuumpump)は油拡散ポンプと同様,ノズルから噴出した蒸気噴流の運動量により気体を吸引・排気するが,これら二つの間の違いは,油拡散ポンプが自由分子流領域で有効に作動するのに対し,エジェクタポンプは粘性流域で有効に作動することである.エジェクタポンプの蒸気噴流の密度は油拡散ポンプよりはるかに高いため,気体分子は噴流の中心まで達することができず,噴流表面に近い層によっ2―210流体機械て吸引・排気される.
