# 3・4・5 ターボ分子ポンプ

> **章**: 第3章 真空ポンプ
> **節**: 3・12 に示すように、クランク機構により弾性体のダイアフラムを

従来のターボ分子ポンプ(turbo molecular pump)は、タービンブレードに類似の円板状動翼を十数枚もった回転子と、各動翼の間に動翼と逆向きの傾斜角をもつ静翼が組合わされた構造(図IV-ポンプ(分子ドラグポンプともいう)を付加した、図IV-3・15のような新世代のターボ分子ポンプが1980年に開発された。このポンプは従来形に比べて、高圧域(1~10Pa)における性能が改善されており、広域形分子ポンプあるいは複合分子ポンプ(compoundmolecular pump)と呼ばれる。以前はボールベアリングをもち、軸が回転することにより自動的に下端の油槽から油を吸い上げて潤滑する機種が多かったが、後にここに示したような、磁気軸受を用いた完全にドライな機種が開発され、普及している。ターボ分子ポンプは、次のような多くの特徴を備えているため、核物理学実験や核融合実験をはじめとする研究分野、および半導体製造に代表される薄膜応用工業等に広く使用されている。(1) 高分子ガスに対する圧縮比が高く、清浄真空が得られる。(2) 排気速度がガスの種類によらずほぼ一定である。(3) 広い圧力範囲にわたって排気速度が一定である。(4) 容易に超高真空が得られる。(5) 高圧にさらされても自力回復ができる。(6) 多量のガスを連続的に処理できる。(7) 保守が容易で、運転費および維持費が少ない。
