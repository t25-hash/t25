# 3・8・1 検査体系と検査手法

> **章**: 第3章 鉄道
> **節**: 3・8 車両の保守

鉄道車両の保守（maintenance）は，旅客および貨物を安全，正れ，定期検査による予防保全を基本としている．
し，それに対応した検査体系が整えられている．
車両の種類に応ており，大別すると次の三つである．
や運行状況に応じ，動力発生装置，走行装置，ブレーキ装置，車体る．
び使用状況に応じ，機能確認のため，動力発生装置，走行装置，ブ.全般検査 長期間使用した後，車両の種類および使用状の検査である．
合わせて実施しているが，その組合せ方は，期間の短い軽度な検査事後保全 故障の処理に，次の上位の検査を行い，さらにそれを繰返した後，最終的に主おりである．
等の状態，作用および機能等について在姿の状態で行う検査であ.重要部検査 比較的長期間使用した後，車両の種類およレーキ装置その他重要な装置の主要部分について行う検査である．
を繰返し行って，一定期間（または一定走行距離）後の検査時期要部分全般について行う全般検査を行う方式で，図3･92に示すとおよび使用状況に応じ，構造，重要度，経済性を加味して定めていたは修繕した車両を初めて使用する場合，脱線その他の運転事故が車両間に伝送線を引き通し，各機器の動作状態を遠隔監視するたら導入されてきた．
当初は列車乗務員に対する支援装置としての位り込めるようになり，列車全体の情報を管理できるようになってき定期検査時に，モニタリング装置を通じて各機器の機能を試運転時に，モニタリング装置によって自動的にデータを障の兆候を予知して検査・修繕する（状態監視保全とも呼ばれる）力手段としては，以前は車内にある中央局に読出し用の端末を接続これを地上側の装置で読み出す方法が一般的になった．
