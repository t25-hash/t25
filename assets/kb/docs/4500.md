# 3・4・6 ゲッタポンプ

> **章**: 第3章 真空ポンプ
> **節**: 3・15 の上半分)をもつが、従来のターボ分子ポンプにねじ溝真空

ゲッタポンプ(getter pump)には蒸発形と非蒸発形がある。蒸発形は、チタンワイヤあるいは図IV-3・16のようなチタン球をタングステンヒータなどにより加熱、蒸発させて、周囲の壁にチタンの蒸着膜を形成し、チタン膜による気体分子の吸着および収着によって排気を行うポンプである。水素、酸素、窒素等の活性気体に対しては大きな排気速度をもつが、メタンなどの飽和炭化水素に対しては排気能力が低く、不活性気体に対しては排気能力がない。そこ図-3・15 ターボ分子ポンプ図-3・16 蒸発形ゲッタポンプで、活性の弱い気体に対する排気能力を高めるためには、チタン付着面を液体窒素温度まで冷却し、その吸着効果を上げている。非蒸発形は、ひだを付けて面積を拡大した金属ストリップの両面に、非常に活性なジルコニウム-アルミニウム合金を50 m程度の厚さでコーティングすることにより、ゲッタ膜として作用させ、排気を行うものである。非蒸発形は、蒸発形で起こる蒸着膜による真空容器内の汚染や、蒸着膜による絶縁性の低下などの問題は起こらない。
