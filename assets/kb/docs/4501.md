# 3・4・7 スパッタイオンポンプ(spatter ion pump)

> **章**: 第3章 真空ポンプ
> **節**: 3・15 の上半分)をもつが,従来のターボ分子ポンプにねじ溝真空

このポンプの原理は,気体イオンによるチタンのスパッタリングを利用したものである.気体のイオン化を超高真空領域の圧力でも持続させるために,強い磁場の中での放電(ぺニング放電)を利用する.ぺニング放電(Penning discharge)というのは,図IV-3・17のように2枚のカソード(陰極)とその間にある円筒のアノード(陽極)に対して,軸方向に磁場を作用させた中で起こる放電のことで,磁場を作用させることにより電子の飛行距離が長くなり,より多くの気体分子をイオン化できる.このぺニング放電でイオン化された気体はカソードに衝突し,カソードからチタン原子をスパッタする.スパッタされたチタン原子は周囲の壁に堆積して膜を形成し,気体分子を吸着する.水素,酸素,窒素などの活性ガスはこの原理で排気される.ヘリウムやアルゴンのような化学的活性の弱い気体は,イオンとしてカソード面に入射してそこに捕えられる.
