# 3・4・8 クライオポンプ

> **章**: 第3章 真空ポンプ
> **節**: 3・15 の上半分)をもつが、従来のターボ分子ポンプにねじ溝真空

クライオポンプ(cryopump)は液体ヘリウムや液体窒素を用いるか、小型冷凍機を用いて15,80K の極低温の面(クライオパネル)を作り、その表面に気体を凝縮固化し、さらに極低温の収着剤に水素、ヘリウム、ネオンを収着させることにより、真空を作り出すポンプである。図IV-3・18に代表的なポンプの構造を示す。ポンプケースの中心に、130K 以下に冷却された第一段シリンダとこれに続く第二段シ図-3・17 スパッタイオンポンプ図-3・18 クライオポンプリンダがあり、それぞれに先端部を第一段コールドヘッド、第二段コールドヘッドと呼ぶ。この先端部が最も冷えるので、第一段コールドヘッドには、80K シールドおよび80K バッフルが取付けられ、第二段コールドヘッドには、その内部に活性炭を付けた15Kクライオパネルが設置されている。水のように蒸気圧の低い気体は80K バッフルで凝縮され、二酸化炭素、酸素、アルゴンなどの一般の気体は、15K クライオパネルの外面で凝縮固化して排気される。水素、ヘリウム、ネオンは15K でも凝縮しないので、15K パネルに接着してある活性炭で収着排気する。
