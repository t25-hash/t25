# 4・1・1 設計項目

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

真空装置とは、真空雰囲気を作成して種々の用途に使用する装置である。魔法瓶、蛍光灯、ブラウン管などのように密閉容器から成る真空装置と、真空容器と真空ポンプで構成される真空装置の両者がある。前者は真空雰囲気を長期間にわたって維持するため、放出ガスを十分に低減しなければならない。そのため、容器を封止する前の脱ガス処理が欠かせない。また、チタンなどのガス吸着材料(ゲッタ材)を内部に設置した装置もある。後者は一般的な真空装置である。装置構造に加えて、真空材料、真空排気系、真空計測系(真空計、質量分析計など)、ガス導入系、試料導入と搬送系、その他の装置固有の系などが設計対象である。真空装置の設計において考慮すべき項目は、(1)用途、(2)装置の寸法、(3)圧力範囲、(4)真空の質(HO分圧)、(5)使用条件などである。真空容器にはつねに、大気圧1atm(10.13N/cm)の外力が作用している。また、フランジ部の変形は漏れ(リーク)に結びつく場合が多いので、大気圧による変形と強度を考慮した、圧力容器としての設計が前提となる。使用条件として決めておくべき点は、(a)到達圧力、(b)残留ガス組成、(c)使用環境(特に温度)、(d)化学反応の有無(特に腐食性ガスの使用の有無)、(e)プラズマや高エネルギー粒子の存在、(f)特殊条件として、磁場の有無および電磁波の許容レベルや許容振動レベルなどが挙げられる。
