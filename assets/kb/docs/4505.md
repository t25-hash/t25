# 3・9・3 軌間可変電車

> **章**: 第3章 鉄道
> **節**: 3・9 新技術

軌間（ゲージ，gauge）の異なる路線を直通運転する車両とし
て，広軌のスペインと標準軌のフランスからスイス，ドイツ等へと
ヨーロッパ各国間を走る「タルゴ列車」が知られている．日本でも
新幹線の標準軌と在来線の狭軌路線を直通運転できるシステムを開
発しており，軌間可変電車（GCT:gauge change train）と呼ば
れる3両編成の試験電車が1998（平成10）年に製造された．
a.開発の経緯 軌間可変試験電車は，1994（平成6）年に運
輸省（現：国土交通省）の運輸技術審議会の答申（21世紀に向け
ての鉄道技術開発のあり方について：SUCCESS21計画）で「標
準軌の路線と狭軌の路線の乗り換えをなくし，鉄道ネットワークの
利便性の向上と所要時間の短縮を図るためには，相互に乗り入れ可
能な軌間可変台車の開発が重要である」と提言され，日本鉄道建設
公団（現：(独)鉄道建設・運輸施設整備支援機構）からの委託を受
けた(財)鉄道総合技術研究所が「新幹線/在来線直通運転システム
の開発」として，この軌間可変システムを実証試験するために製作
した車両であり，軌間可変する電車としては世界初である．
試験電車は，1999（平成11）年から約2年間にわたり，山陰本
線で確認走行の後，アメリカ，コロラド州プエブロ実験線で200
km/ h以上の速度で約60万km走行の高速耐久試験を行った．
2001（平成13）年に交直流電車化・新幹線対応に車両を改造し，
九州の日豊本線および四国の予讃線で在来線での速度130km/h走
行と本則＋25km/ hの曲線通過性能試験，新下関試験基地で軌間
変換試験，交流25kV/直流1500Vの交直セクション直通試験を
行った．2004（平成16）年には，山陽新幹線新山口～新下関間で
速度210km/hまでの走行性能試験を行った．
b.軌間可変電車の特徴と概要 スペインのタルゴ車両が動力
をもたない客車であるのに対して，軌間可変電車は電動機付きの軌
間可変台車を各車両に装備した全電動電車である．タルゴ車両が広
軌（1668mm）と標準軌（1435m）間233mmの変換に対し，軌
間可変電車は標準軌（1435mm）と狭軌（1067mm）間368mm
の変換である．
軌間可変電車の車体寸法は，新幹線と在来線を直通運転するた
め，秋田新幹線「こまち」と同じ在来線の車両限界を適用した．設
計最高速度は，新幹線で300km/ h，在来線で130km/ hである．
内に設置されている．

図3･94に電車の外観，図3･95に軌間変換装置を示す．
