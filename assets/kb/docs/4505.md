# 4・1・2 真空排気(vacuum pumping)過程

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

真空を作成するには、大気から真空容器内に気体が漏れないようにするとともに、真空容器壁から放出されるガス分子を排気する必要がある。ガス分子の放出とは、材料表面に吸着しているガス分子の脱離、材料内部に含まれているガス分子の拡散放出、真空容器壁をガス分子が透過、真空容器の漏れ(リーク)などである。真空排気過程は、どのようなガス放出現象が支配的かに大きく影響される。排気速度が一定の、理想的な真空ポンプで排気した場合の排気特性を図IV-4・1に示す。大気圧に近い領域では、もともと真空容器内にあった気体が排気される。真空容器内にあった気体は短時間で排気され、真空容器内壁面から放出されるガス分子の排気となる。圧力はおおむね、排気時間の逆数に比例して減少する。さらに排気を継続すると、真空容器材料の内部に含まれていた分子が、拡散により表面まで達して放出される現象が続き、圧力は排気時間の1/2乗の逆数に比例して減少する。最後は、真空容器を透過するガス分子の排気となる。以上の排気過程を排気の式で表すと、次式となる。低真空(10~10Pa)では、真空容器内の気体の排気が主であり、中真空(10~10Pa)および高真空領域[10~10(あるいは10)Pa]では、真空容器内壁面に吸着しているガス分子の放出が支配的である。特に、HO分子の吸着脱離挙動を考慮した排気が必要になる。超高真空(10~10Pa)から極高真空(10Pa以下)では、拡散や透過によるガス放出も考慮しなければならない。以下、真空容器の漏れはないものとし、それぞれの排気過程について記す。a. 真空容器内の気体の排気式(IV-4・1)の右辺第1項が、真空容器内にもともとあった気体の排気の式である。圧力まで排気するのに要する時間は次式で表される。b. 吸着(adsorption)ガス分子の排気低真空から高真空および超高真空領域では、材料表面に吸着しているガス分子の脱離現象が支配的である。式(IV-4・1)において、右辺第2項が吸着ガス分子の排気を示す。この領域では、圧力が排気時間の逆数に比例して減少する。吸着脱離がほぼ平衡状態にあり、吸着ガス分子の脱離の活性化エネルギーが単一である場合の排気の式は、次式で表される。単一の脱離の活性化エネルギーであるという仮定は、式(IV-4・3)においてガス分子の平均滞在時間τが一定であることに対応する。式(IV-4・3)の排気曲線は、両対数グラフ上で圧力が排気時間の逆数に比例して低下せず、実測結果と一致しない。排気過程のモデルは種々提案されている。デイトン(Dayton)は、表面のポーラス状酸化膜層に吸着したHO分子が脱離する過程として排気モデルを提案した。ガス放出現象を、種々の微細な孔からHO分子が放出される過程としてとらえると、排気曲線の傾きはほぼ-1となり、実測値と一致する。ベネマ(Venema)は、ガス分子の吸着がラングミュア型に従うとして、各エネルギーに対する排気曲線を求めた。各エネルギーの排気曲線を加え合わせたものが、実際の排気曲線であるとすると、圧力は排気時間のほぼ逆数に比例して低下する。脱離の活性化エネルギーが十分小さい場合は、吸着ガスが容易に脱離するために排気時間は短い。一方、脱離の活性化エネルギーが十分大きい場合は、吸着ガスが脱離しないので圧力への寄与が無視できる。すなわち、適度に脱離しやすい吸着ガスの挙動が、真空容器の圧力を決めていることになる。室温におけるHO分子の脱離の活性化エネルギーは92~100kJ/molであり、適度に脱離しやすいエネルギー範囲と対応する。c. 拡散(diffusion)によるガス分子の排気吸着ガス分子が排気されてさらに圧力が下がり、10Pa以下の領域になると、材料内部に含まれているガス分子の放出が圧力に影響する。ステンレス鋼には水素や炭素、酸素、窒素などが含まれており、材料表面に拡散して気相中に放出される。拡散により表面まで達した後に、そのまま脱離する場合、表面で拡散(表面拡散)した後に脱離する場合、他の原子と結合して脱離する場合など、拡散によるガス第IV部真空機器放出現象は複雑な経路をたどる。排気時間を支配するのは、ガス分子(あるいは原子)が材料内部から表面まで拡散してくる時間である。金属材料においては水素が拡散しやすく、ガラスではヘリウムの拡散速度が大きい。d. 透過(permeation)によるガス分子の排気大気中のガス分子が真空容器壁を透過する現象は、大気側の壁表面へのガス分子の吸着(解離する場合もある)、材料中の拡散、真空側の壁表面での脱離(表面拡散、再結合なども含まれる)という経路をたどる。真空容器の外側および内側の圧力を,、容器の板厚をとすると、単位時間、単位面積当たりにガス分子が透過して放出されるガス放出量は、次式で表される。10Pa以下の圧力領域では、気相中のガス成分は水素が主である。水素は大気中に0.5ppm(分圧:0.051Pa)含まれているが、水素はステンレス鋼などの金属材料を透過するので、吸着ガスや材料内部のガス分子がすべて排気されたとしても、避けることのできないガス源となる。ただし、室温では透過量は極めて小さい。
