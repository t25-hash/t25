# 4・1・3 ガス放出量(outgassing rate)

> **章**: 第4章 真空システム
> **節**: 4・2 )の排気時間より長くなることが知られており、経験的な目

ガス放出量は、材料の種類、加工条件、洗浄条件、保管条件、排気の履歴などの影響を強く受ける。真空溶解したステンレス鋼SUS 316Lのガス放出特性を、図IV-4・2に示す。最初の排気1の後に、250°C×24hのベーキング処理(baking)を実施した2のガス放出量は、1に比較しておよそ3けた減少している。次に大気開放して再び真空排気を開始すると、1と比較し3は約1けた近くガス放出量が低下した。さらに150°C×15hのベーキング処理を実施した後のガス放出量4は、2よりも低い。この後、大気開放とベ図-4・2 真空溶解したSUS 316Lのガス放出特性図-4・4 アルミニウム合金のガス放出特性2―213ーキング処理を繰返しても、5および6に示されるように、ガス放出量の変化は認められない。この現象は、ステンレス鋼の表面に、CrやFeの酸化物が形成される挙動と考えられている。ガス放出量が表面状態や排気過程の影響を受けることを活用し、事前に真空ベーキングや表面析出などの特殊な処理を実施することにより、ガス放出量が低減できる。図IV-4・3,IV-4・4はベーキング処理の最高温度と、単位面積当たりのガス放出量の関係を示したものであるが、ステンレス鋼およびアルミニウム合金のいずれにおいても、特殊な前処理の有無でガス放出量が1けた近く異なっている。なお、単位面積当たりのガス放出量をガス放出速度という。表IV-4・1に各種金属材料のガス放出量を示す[文献(31)(34)(36)~(42)].表IV-4・2に非金属材料のガス放出量を示す[文献(5)(21)(43)~(45)].概して非金属材料は、金属材料に比較してガス放出量の大きいものが多いので、事前にガス放出量を把握しておく必要がある。ガス放出量は、材料固有の物性値ではなく種々の影響を受けるので、ある程度の誤差を許容したうえで真空装置を設計する。洗浄条件によってもガス放出特性が影響を受ける。
