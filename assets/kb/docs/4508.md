# 4・1・5 真空計測

> **章**: 第4章 真空システム
> **節**: 4・2 )の排気時間より長くなることが知られており、経験的な目

大気圧から超高真空までの圧力領域を、単一の真空計(vacuumgauge)でカバーすることはできない。そこで、(1)大気~10Pa程度、(2)10~10Pa、(3)10Pa以下(10Pa程度まで)の3領域に対応した真空計を選定する。用途別では、(1)高精度圧力測定、(2)通常の圧力測定、(3)高精度圧力モニタ(圧力制御用)、(4)圧力モニタ(大気か真空の判定)に分類できる。要求仕様別では、(1)高精度(圧力の測定精度が高い)、(2)長期安定性(測定値の変動量が小さい)、(3)耐久性(故障しにくい)、(4)耐食性に優れている、(5)ガス放出量が少ない、(6)操作性に優れる、などに区分される。圧力計の測定精度が必要な場合は、真空計を校正しながら使用しなければならない。スピニングロータ真空計は長期の安定性に優れていることから、低真空領域の基準真空計として広く使用されている。高真空用の基準真空計は、標準マクラウド真空計にて校正された三極管型熱陰極電離真空計VS 1-Aが、日本真空協会から供給されている。超高真空から極高真空領域では電離真空計が使用されるが、軟X 線効果の影響で、測定可能な真空計が限定される。市販の真空計ではエクストラクタゲージが最も測定下限圧力が低く、2×10Paである。
