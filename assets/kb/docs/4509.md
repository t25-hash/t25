# 3・1・3 脂質源食品群の加工機械

> **章**: 第3章 食品機械
> **節**: 3・1 食品加工機械

a.製油 風力および磁気選別によって挟雑物を除いた大豆
を，はく（ ）皮容易にするため50～80℃に加熱し水分8～10%に
乾燥する．粗砕機で2分割し，脱皮機で表皮を風選除去する．この
胚乳を水分7 %まで加熱し，ロール圧延の後，圧搾抽出機（screw
expeller ）で残油16～18 %まで搾油する．圧搾固形残渣は粗砕後
ロール圧延して溶媒抽出機（solvent extractor ）に送る．抽出液
はセラミックスろ材でろ過し，精留塔（rectifier ）で溶媒（ノルマ
ルヘキサン）を蒸発させ油を分留する．分留油と搾出油を混合し遠
心分離の後，真空乾燥機で脱水し原油として精油工程に送る．抽出
固形残渣は脱溶媒装置で溶媒を蒸発させた後，冷却，粉砕，ふるい
分けして飼料原料とする．
精留塔および脱溶媒装置で蒸発した溶媒蒸気は凝縮器で液化し，
吸着装置で純化し，精留塔で脱水し，再び溶媒抽出機に送る．
精油工程では，原油と温水をかくはんしてガムを除去し，さらに
苛性ソーダを加えアルカリ油滓を除去し，活性白土を分散させ脱色
する．白土を遠心沈降装置で除去し，脱 ，脱臭して製品になる．
常温で液体の植物性油または魚油はニッケル触媒下で120℃，
0.5～3MPaで水素を加え，不飽和脂肪酸の二重炭素結を一重にし
水素を結合し，常温固体の飽和脂肪酸にする．精油工程に水素添加
硬化操作を含めて，マーガリンやショートニングを生産する場合も
ある．
b.乳製品 タンクローリで集荷された原乳は，成分検査後
5℃に急速冷却し生乳として用途別サイロに一時貯蔵する．
.市乳 生乳を50～70℃に予熱し，挟雑物を遠心分離除
去する．脂肪コロイドを1m以下に微細化分散（homogenize）
させるために，共振子に8～25MPaの圧力で乳を噴射し超音波を
発生する一種の笛である均質化装置に圧送する．130℃，2秒
～150℃，0.5秒の高温短時間加熱殺菌後5℃以下に急冷，無菌充て
んする．微生物の生活細胞のみ殺菌する殺菌乳（pasteurized
milk）は低温流通し，胞子まで無菌化した滅菌乳（sterilized
milk）は常温流通する．
.練乳粉 生乳の挟雑物または乳脂の一部を遠心分離除去
し，低温多重効用缶（80～40℃）で固形分40～50%に濃縮する．
塔高約5mのタンク塔頂より噴霧し150～200℃の空気で乾燥し粉
体化する．流動層で急冷しふるい分けの後包装する．
.クリームバター 生乳を40～50℃に昇温し脂肪分
30～40%のクリームを遠心分離する．均質化後，70℃，15秒
～90℃，10秒の殺菌を経て4℃に急冷しクリームとする．塩と水を
加えてかくはん（churning）するとクリームの脂肪球膜（タンパ
ク質）が破壊し脂肪が凝集し直径約7mmのバター粒子になる．
煉圧機で均質化し包装する．
.アイスクリーム 生乳，液糖，バター，水，乳化剤，増
粘安定剤などを混合し挟雑物を除去し均質化する．80～90℃で加熱
殺菌後，香料着色料を添加混合し急冷する．容積の70～80%の空
気で発泡させ－2～－8℃の液状で容器に充てん－18℃以下で硬化さ
せる．
.チーズ 生乳を75℃，15秒加熱殺菌し30℃に冷却す
る．乳糖を分解する乳酸菌を接種し酸度0.20～0.22になるまで醗
酵させる．凝乳酵素および塩化カルシウムを添加して1時間静置す
るとカゼインタンパク質の凝固物（curd）と乳清（whey）に分離
する．凝固物を切断し脱水して熟成する．チーズ製造ではスタータ
と称する乳酸菌が10℃の醗酵で数週間後死滅するが，生成した酵
素（生体触媒）が残存してタンパク質から多くの有用成分を生成す
る．
.ヨーグルト 脱脂乳，粉乳，ショ糖，増粘剤を混合し，
80℃・2分～140℃・2秒加熱殺菌し，スタータと香料を添加して容
器に充てんする．40～50℃で4～6時間醗酵させ5℃以下に冷蔵す
る．
