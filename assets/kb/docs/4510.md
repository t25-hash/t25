# 4・1・7 駆動機構および真空部品

> **章**: 第4章 真空システム
> **節**: 4・2 )の排気時間より長くなることが知られており、経験的な目

真空中の試料駆動方法には、真空中に設置した駆動源を用いる場合と、大気中に駆動源を設置し、運動導入機構により真空中の試料を駆動する方法がある。真空中にモータを設置して使用する場合、ガス放出と放熱対策が必要である。真空中では気体による熱伝導が期待できないため、モータの放熱対策が難しい。その結果、モータ温度が上昇してガス放出量も増加する。発熱の比較的少ないステッピングモータが市販されているが、低出力のものが多い。大気中から真空中への動力伝達は、大気中の駆動源の動きを、軸シールを用いた運動導入機構を介して真空中に導入する方法が一般的である。軸シール部の漏れを防止するため二重にシール部を設け、その中間部を差動排気する場合もある。磁力を利用した動力伝達は磁気カップリングとも呼ばれる。磁力を利用した運動伝達機構であるため、大気側と真空側の運動が必ずしも一致しないが、回転や直線導入を同じ機構で導入することができる。2―215第IV部真空機器運動機構の設計では、真空中の摩擦抵抗が大気中に比較して数倍大きいことに注意しなければならない。しゅう動部を避けて転がり接触にするような構造上の工夫や、接触部の材料は異種材料を選定するなどの配慮を要する。潤滑材は、蒸気圧が低ければ使用可能で、フッ素系のグリースや真空ポンプ用の潤滑油などが使用できる。フッ素系樹脂やポリイミド系樹脂などの高分子材料もしゅう動部に使用される。また、しゅう動部の母材には、Au,Ag,Inなどの金属薄膜や、MoS,WSをコーティングした固体潤滑も使用されている。真空用ベアリングには、Ag コーティングやMoSコーティングを処理したものもある。電流導入端子には、電流導入の金属線をガラスやセラミックスで封止したものや、Oリングなどのエラストマシール方式のものが市販されている。耐電圧や許容電流により種々のタイプがある。光の導入としては、硬質ガラス、石英、サファイアなどのフランジ付き窓が市販されている。透過させたい光の波長に応じて窓材料を選定する。そのほか、X 線用のベリリウム窓やマイクロ波用のセラミック窓などもある。真空中で爆発性のガスを使用する場合は、真空容器に破壊しやすい部分を設けて大事故を防止する。金属薄膜を用いたラプチャディスクが市販されている。ラプチャディスクの取付位置は、周囲の状況を十分考慮して決めなければならない。真空容器の大きさ(容積および内表面積)、到達圧力および処理圧力、導入ガスの流量、排気時間などを考慮し、排気システムを設計する。目標到達圧力から真空ポンプの種類を決め、真空容器の大きさならびに導入ガス量および排気時間を考慮して、真空ポンプの容量を決める。主ポンプの種類と容量が決まれば、それに見合った補助ポンプが選定される。補助ポンプを主ポンプと離れた場所に設置する場合は、真空配管のコンダクタンスを考慮して設計する。そのほか、緊急遮断や大気開放などを想定して、真空バルブの配置や真空計の種類と配置を設計する。複数の真空容器が接続された真空システムの圧力分布は、真空系を電気回路に置換えて求めることができる。圧力を電圧、ガス放出量(ガス流量)を電流、コンダクタンスと排気速度はその逆数を抵抗に対応させ、キルヒホッフの法則を用いる。
