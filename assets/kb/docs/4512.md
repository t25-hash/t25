# 4・2・2 真空ポンプ(vacuum pump)の選定と排気システム

> **章**: 第4章 真空システム
> **節**: 4・2 排気システムの設計

高真空(high vacuum)領域の真空排気システム目標到達圧力が10Paである真空排気システムを図IV-4・6に示す。真空装置の用途は半導体製造プロセス処理とする。真空処理室には処理ガスが供給され、処理圧力は数Paから数十Paとする。処理ガスが供給されることと到達圧力が10Pa台であることを考慮すると、主ポンプは溜め込み式ではなく、連続排気式のターボ分子ポンプが望ましい。油回転ポンプは処理ガスによるポンプ油の劣化や、油分子による真空処理室の汚染が懸念される。真空処理室への油分子逆流を防止するには、補助ポンプにドライ真空ポンプを選定する。次に、処理圧力が数十Paであることを考慮し、ターボ分子ポンプの上流側に排気速度調整弁を設ける。真空処理室を大気に開放するときも真空ポンプを運転しておき、短時間で再排気でき図-4・7 超高真空(ultrahigh vacuum)領域の真空排気システム図IV-4・6は連続排気式の真空排気システムであったが、図IV-4・7は溜め込み式の真空排気系の例である。図IV-4・7の排気システムの目標到達圧力は10Pa以下である。高真空と超高真空の違いは、残留ガス成分中にHO分子が多いか少ないかに関連しているので、HO分子の排気速度が他の真空ポンプに比較して圧倒的に大きいクライオポンプを選定した。補助ポンプはドライ真空ポンプである。大気圧から真空排気を開始する場合は、初めにクライオポンプと真空処理室の両者を補助ポンプで排気する。クライオポンプが動作可能な圧力(10~10Pa)に達した時点で、クライオポンプの運転を開始する。次に補助ポンプの排気系を遮断し、クライオポンプ単独の排気とする。再排気の場合は、クライオポンプと真空処理室および補助ポンプを切り離しておき、補助ポンプ単独で真空処理室を排気する。所定の圧力に達した時点で、クライオポンプの排気に切換える。クライオポンプは、10~15K に冷却したクライオパネルと、50~80K のシールドやバッフルから構成され、冷却部にガス分子を吸着することで排気を行う。蒸気圧は、HOが130K で10Paであり、N,COなども20K で10Pa以下である。したがって、クライオパネルで凝縮させることで、超高真空まで排気することができる。しかし、He,Hは10K では高い蒸気圧を示すため、冷却面への凝縮作用を利用して排気することができない。そのため、活性炭による吸着作用を利用した排気が行われている。d. 圧力-流量線図(pressure gas flow
