# 4・2・3 真空配管の設計

> **章**: 第4章 真空システム
> **節**: 4・2 排気システムの設計

図IV-4・9のように、真空ポンプが配管を介して真空容器に取付けられている場合は、排気特性の設計において、真空ポンプの排気速度の代わりに配管のコンダクタンス(conductance)を考慮した実効排気速度(effective pumping speed)を用いる。コンダクタンスを求める前に、対象とする圧力範囲において、配管内が粘性流領域であるか分子流領域であるかを判断する。円形配管のコンダクタンスは、粘性流領域および分子流領域に対し、それぞれ次のように表される。中間流領域も含めて広い圧力範囲で使用できるコンダクタンスの式が、クヌッセンにより求められている。配管内は粘性流領域と仮定する。配管内を流れるガス流量と排気の方程式は、連続性を考慮すると次のように表される。容積1mの真空容器を、排気速度0.1m/sの真空ポンプを用いて大気圧から1Paまで排気する場合を想定する。配管の長さが10mになれば、排気時間は240sとなる。なお、<1×10になると、真空ポンプの排気速度を大きくしても排気時間は短縮されない。したがって、実効排気速度が配管のコンダクタンスに依存して決まってしまうことがわかる。式(IV-4・14)を圧力で整理すれば、排気時間と圧力の関係が得られるが解析的に解くことが困難なので、式(IV-4・13)を数値解析して求めた排気特性を図IV-4・11に示す。排気開始直後の圧力が高い状態では、コンダクタンスが大きいので実効排気速度も大きく、圧力は速やかに低下する。排気とともに圧力が減少してコンダクタンスが小さくなると、実効排気速度が減少し、圧力減少も緩やかになる。大気圧から所望の圧力までの排気時間を短縮するためには、補助ポンプの排気速度と配管寸法を適切に設計しなければならない。次に真空処理室にガスを導入する状態を考える。圧力は定常状態とする。排気速度が仕様を満たさない場合は、真空ポンプの機種を変更し、上記の検討を繰返すことになる。配管のコンダクタンスを式(IV-4・8)から求め、所定の圧力が得られるように配管を設計する。分子流領域でガスを導入しながら所定の圧力を得たい場合は、主ポンプの上流側に排気速度調整弁(コンダクタンス調整弁)を設ける。極高・超高真空システムでは、ガス放出量が十分少ない材料を選定したうえで、種々のガス分子を排気できるように、複数の主ポンプを選定するなどの工夫が必要になる。また、真空容器内壁面での吸着ガスの挙動を把握するとともに、脱ガス処理が必須となる。
