# 3・2・1 はじめに

> **章**: 第3章 食品機械
> **節**: 3・2 食品の包装と充てん機械

食品は水分影響の除去，化学変化防止，毒物・微生物・昆虫等の
危害物質の混入を避けるため袋・容器（以後容器と記す）に収納さ
れ流通している．したがって容器の材質は食品の品質劣化を防止す
るに適したものであることが要求される．その要件は，強度，ガス
バリヤ特性，内容物の収着特性であり，食品によって容器素材が決
まる．
食品の状態は液，スラリー，ペースト，固体，粉体，粒体とさま
ざまであり，形状も食品によって異なっており，個々の食品の特性
と特徴を考慮したうえで容器の材質と構造が決まる．
食品の容器（food package）には上記の機能に加え，ファッシ
ョン性も見逃すことはできない．容器デザインの良さは品物の売り
上げや，食べるときの感性に影響を与えるものである．
