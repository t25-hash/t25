# 4・3・1 吸着ガスの挙動と脱ガス処理

> **章**: 第4章 真空システム
> **節**: 4・3 極高・超高真空システム

分子流領域での真空排気は、ガス分子が真空ポンプ排気口に飛び込んでくる頻度に依存する。あるガス分子に着目すると、壁面への吸着と再脱離を繰返しながら、真空ポンプ排気口に飛び込んで排気される。脱離して再吸着するまでの時間を、真空容器の大きさとガス分子の平均速度から推定すると、数十msのオーダである。吸着・脱離を何回繰返して排気されるかは、真空装置の大きさや構造に依存するが、気相中に存在している時間は非常に短い。したがって、排気時間は脱離の活性化エネルギーに大きく依存する。高真空と超高真空の違いは、真空容器内壁面に吸着しているHO分子の有無(多少)とみなせる。4・1・2・b項で述べたように、壁面に吸着しているHO分子は、脱離の活性化エネルギーが92~100kJ/molである。脱離の活性化エネルギー욃と、平均滞在時間(meanresidence time、表面に吸着したガスが再脱離するまでの平均時間)τの間には、次式が成立する。τ=τexp 욃(IV-4・16)ここで、τは10s程度の値で、吸着分子の垂直方向の振動周期に対応し、=8.31J/K・molはガス定数である。욃とτの関係を図IV-4・12に示す。室温(300K)におけるHO分子の吸着時間は욃=92~100kJ/molに対し17min~7hであり、排気されるまでの時間は非常に長いことがわかる。平均滞在時間τは温度に大きく依存する。ベーキング処理とは、壁面に吸着しているガス分子を強制的に脱離させる処理であるが、吸着から再脱離するまでの滞在時間を短縮することにある。ベーキング温度を200°CとするとHO分子の平均滞在時間は、17min~7hであったものが1.4~11msまで短縮される。ベーキング前後の排気特性を図IV-4・13に示す。図IV-4・13のHO分圧の減少は、ベーキング処理により吸着の図-4・12 脱離の活性化エネルギーと平均滞在時間図-4・13 脱脂処理したSUS 304のベーキング処理と排気特性滞在時間が短縮されたことに対応している。脱離の活性化エネルギーを욃=100kJ/molとし、20°Cから150°Cに昇温されることで、HO分子の平均滞在時間がどれだけ短縮されるかを図IV-4・12から見積もると、約5けた短縮される。図IV-4・13では、150°Cのベーキング処理によりHO分子のガス放出量は約5けた減少している。HO分子が排気された後のおもな残留ガス成分は、HとCOである。Hは真空容器材料内部に含まれている主要なガス成分であり、あらかじめ高温脱ガス処理を施すか、H拡散防止の被膜を表面に形成するなどの方法で、低減することができる。真空容器の加熱が困難な場合の脱ガス処理として、放電洗浄が使用される。放電洗浄とは、真空容器内にArなどのガスを導入してプラズマを作成し、真空容器壁面をイオンで照射することにより、吸着ガス分子を強制的に脱離させる手法である。
