# 3・2・2 容器材質

> **章**: 第3章 食品機械
> **節**: 3・2 食品の包装と充てん機械

紙は天然素材であり古くから使用されているが，現在，食品容器
素材としての利用は，高分子フィルムやアルミはくを層状に重ね合
わせ，軽量，高耐久性，バリヤ性に富んだ複合化素材である．ブロ
ック状容器はインラインでロール原紙から成形され，充てん，シー
ルを一貫して行うことができる．
ガラスは化学的に安定な物質であるが，強度と重量に問題があ
る．近年，ペットボトルに移行している．リターナブルビンは環境
負荷が少ないので，軽量化と強度の向上をはかり，利用率を高める
必要がある．
高分子の容器・包装材料としてはガスバリヤ性と水蒸気バリヤ性
に富んだものが要求される．ガスバリヤ性は高分子の一次構造であ
る主鎖にアミド基，エステル基，フェニレン基，エーテル基を含
み，側鎖基に，水酸基，シアノ基，クロロ基を含むものがよい．バ
リヤ性は高分子の凝集エネルギー密度と分子の自由体積分率とに関
係している．防湿性の良好なものはポリエチレン，ポリプロピレ
ン，ポリ塩化ビニリデンでありガス透過性との相関は少ない．食品
中の親油性物質の収着性についても注意を払う必要がある．
金属素材ではスチールとアルミニウムが用いられる．スチールに
は不純物の少ない連続鋳造材を使用し，圧延，焼なまし，調質し，
めっき原板とし，電気ブリキ，TFS（Tin Free Steel ）処理後1m
角の板に裁断する．缶材のアルミニウム合金は非熱処理型合金であ
り，純アルミのものは少ない．Mn，Mgを含んだ合金が使われる．
EOE（Easy Open End）用の材料としてはスコア（切込み）加工
に耐える強度をもつ材料が選ばれる．
1997年に施行された「容器包装に係る分別収集及び再商品化の
促進等に関する法律，略称：容器包装リサイクル法」では，包装材
料の明記とそのリサイクルが促されるようになっている．
