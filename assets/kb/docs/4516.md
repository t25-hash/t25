# 2・1・1 作動油の種類

> **章**: 第2章 油圧システム
> **節**: 2・1 油圧作動油

作動油は一般的には油圧作動液(hydraulic fluid)を指すことが多いが,広義には機能性流体(functional fluid)を意味し,ブレーキ液や自動車の自動変速機油なども含まれる.ここでは,油圧作動液の意味で用いる.作動油は圧力エネルギーの伝達媒体であり,以下の性能が要求される.1適正な粘度を有すること,2温度変化に対して粘度変化が小さいこと(粘度指数が高いこと),3潤滑性が良いこと,4体積弾性係数が大きいこと,5寿命が長いこと,6燃えにくいこと,7キャビテーションに対して抵抗が大きいこと,8汚染物除去が容易なこと,9廃液処理が容易で,毒性のないことなどである.作動油には図V-2・1に示すように非常に多くの種類があるが,上記の性能を満足し,用途または目的等に合った適正な作動油を選択することが必要である.作動油は大きく分類すると,鉱油系作動油(petroleum base hydraulic fluid),難燃性作動液(fire resistant hydraulic fluid),生分解性作動油(biodegradablehydraulic fluid)に分かれる.生分解性作動油は,環境に配慮するため最近になって開発された油種であり,土中あるいは水中に漏れた場合に微生物等によって分解されやすい成分で構成されている.
