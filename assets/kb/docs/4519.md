# 3・2・6 粉体充てん機（powder filler ）

> **章**: 第3章 食品機械
> **節**: 3・2 食品の包装と充てん機械

粉体の流動特性は粉の種類，水分量によって著しく異なる．粉体
出口にはブリッジができ，粉体が停留する．これらの難点を克服す
るためには強制的に流れを作り出し充てんする方法を用いる．
図3･9は粉体用充てん機の基本形である．充てんホッパ，供給孔に
スクリュー状のオーガを設け，かくはん羽根でホッパ斜面に滞留す
る粉体を掻き落とす．充てんは定荷重方式であり，充てん後容器に
振動を加え，充てん時の体積を揃える，タッピング操作を行う．袋
の口のシールのためには糊面への粉体の付着を避ける必要がある．
また，完全密閉すると重ねたときに荷重圧によって破袋する恐れが
あるので息抜き用の細孔が必要である．しかし，酸化しやすい粉体
はこの限りではない．砂糖袋底部に極めて小さな穴があり，空気抜
きになっている．
