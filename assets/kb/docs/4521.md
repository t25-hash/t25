# 2・1・6 汚染管理(contamination control)

> **章**: 第2章 油圧システム
> **節**: 2・1 油圧作動油

作動油は固体粒子、異種液体、空気(気体)により汚染される。異種液体とは水(水成系の場合を除く)、洗浄用液体、他潤滑油などを指し、空気は油中に気泡として混入している場合である。異種液体や空気は作動油の寿命および油圧装置の特性に悪影響を及ぼすが、最も問題になるのは、金属粉、じんあいなどの固体粒子である。固体粒子はその硬さや大きさにもよるが、機器しゅう動部に入り込むことによってしゅう動面をきずつけ、機器性能を低下させ、さらには作動を不能にする。固体粒子による汚染状態は等級で示され、表V-2・3,V-2・4に示すような粒子の数と大きさによるカウント法、あるいは質量に基づく質量法で規定される。電気-油圧サーボ弁を用いる装置にはカウント法でNAS 6~9級以下、質量法でNAS 101~108級以下が推奨されている。
