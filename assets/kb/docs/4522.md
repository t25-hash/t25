# 3・3・3 ケーサ（caser ）

> **章**: 第3章 食品機械
> **節**: 3・3 荷造り機械と荷さばきシステム

図3･10はロボットがビンを一度に4本つかみ，もち上げ，水平
移動し，段ボールケースに収納するシステムである．単純な機能の
ロボットを利用するためには，ケーサ部分に達した商品をロボット
が扱いやすいように商品の流れを調整しておく必要がある．
図3･11はパラレル運動型ロボット2台が，生産ラインからラン
ダムに送り込まれてきた製品を平行に流れるベルト上のケースに詰
込むシステムである．インフィードコンベヤ上の製品の向きはさま
ざまではあるが，入ってくる製品の情報はベルト上部のカメラが捉
え，画像処理により，形状，方向のデータとしてロボットに伝達さ
れる．ロボットはアームを動かし，製品を取り，アタッチコンベヤ
上の収納容器に商品を確実に収納するため，アクチュエータの向き
を変え，ベルトに置かれた容器に所定の方向で製品を収納する．処
理能力は極めて高く毎分200個以上を処理する能力をもっている．
