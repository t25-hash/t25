# 2・2・4 ピストンポンプ・モータ

> **章**: 第2章 油圧システム
> **節**: 2・4 は内接形の歯車ポンプ・モータの原理的構造を示す.

a. ピストンポンプ. 分類と構造油圧用として用いられるピストンポンプ(piston pump,プランジャポンプともいう)は,流れの脈動を低く抑えるため通常5本以上のピストンを有し,その配列形式からアキシアル形,ラジアル形,レシプロ形とに分類される.また,シリンダブロックがピストンとともに回転して,両者が相対的に往復運動を行う回転シリンダ形と,静止したシリンダ内をピストンが往復する固定シリンダ形とがある.アキシアルピストンポンプ(axialpiston pump)はピストンが軸に平行に配列され,斜軸式,斜板式(以上回転シリンダ形),および回転斜板式(固定シリンダ形)がある(図V-2・12).ラジアルピストンポンプ(radial piston pump)は,ピストンが放射状に配列される(図V-2・13).レシプロピストンポンプは,シリンダが軸を含む平面内に軸と直角に配列され,クランクまたは偏心カム機構によってピストンの往復運動を行わせる(図V-2・14).以上のうち回転シリンダ形は同一構造で,油圧モータとしての作動が可能である.また,運転中に押しのけ容積を変化しうる可変容量形としやすい大きな特徴があるので,最も多く用いられている.図V-2・15は斜軸式ピストンポンプの一例で,駆動軸とこれに傾斜したシリンダブロックから成る.シリンダブロックは,コンロッドまたは自在継手などにより駆動軸とともに回転し,2―227第V部フルードパワーシステム(a)
