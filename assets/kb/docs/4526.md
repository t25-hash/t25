# 2・2・5 ねじポンプ・モータ

> **章**: 第2章 油圧システム
> **節**: 2・4 は内接形の歯車ポンプ・モータの原理的構造を示す。

ねじ軸の数により一軸、二軸、三軸形があるが、油圧用は、三軸ねじポンプ・モータ(triple-screw pump/motor)が多く用いられている。三軸ねじポンプ・モータの構造およびねじ形状は、ポンプ用、モータ用とも同じで、その構造を図V-2・24,V-2・25に示す。3本のねじは、円滑にかみあいながら回転し、吸込側でねじ1リードごとに作動油を満たした密閉室を次々に形成し、作動油を連続的に吐出し側へ送り出す。この圧送過程には急激な圧力や流量の変化がなく、運転音は極めて低い。圧力の脈動は、内部漏れ量および作動油の圧縮性に影響されるが、全振幅は吐出し圧力の2~5%で、周波数は回転速度の2倍である。図-2・22 斜軸式ピストンモータの作動説明図図-2・23 回転シリンダ形ラジアルピストンモータ(偏心形)の作動説明図2―230流体機械3本のねじ軸は完全に動釣合いがとられ、慣性力も小さいので高速回転に適している。ポンプは25~60s、モータは20~100sの回転速度で使用される。定格圧力は、主として有効ねじ長さ(形成される密閉室の数)に依存する。一般作動油のほかに、高含水系作動油、水溶性切削油にも適用されている。
