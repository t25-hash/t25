# 2・2・6 低速高トルクモータ

> **章**: 第2章 油圧システム
> **節**: 2・4 は内接形の歯車ポンプ・モータの原理的構造を示す。

起動トルクが高く、数回転以下から定格回転まで無段階に円滑な図-2・24 ねじの軸直角断面図図-2・25 三軸ねじポンプ・モータ断面図図-2・26 アキシアルピストン形低速高トルクモータ図-2・27 偏心形ラジアルピストンモータ速度制御ができる低速高トルクモータ(low speed high torquemotor)は、建設機械、舶用機械などの各種アクチュエータとして広く用いられている。構造的には、特に低速用に作られたアキシアルピストン形、偏心形(モノストローク形)とカム形(多行程形)のラジアルピストン形モータなど、油圧力による回転トルクを直接出力軸に伝えるものと、差動機構(differential mechanism)を利用したり、高速モータと遊星歯車などの減速機を一体に組合わせて高トルクを得られるものに大別される。a. アキシアルピストン形低速高トルクモータ斜板式ピストンモータを特に低速用に製作したもので、図V-2・26に示すような分配弁や球面弁板を採用し、比較的大きなピストン径を用いて高トルクを得られるようにしている。斜板を対向に使用してピストン数を2倍にしたもの、また、斜板傾角を二、三段階に切換えることができるものもある。b. 偏心形ラジアルピストンモータ(radial piston motor)図V-2・27に示すように星形に配列された数本のシリンダの中でピストンが往復運動し、これを偏心カムにより回転運動に変えて低速高トルクを得る構造のもので、構造が簡単で部品数が少なく取扱いが容易である。なお、カムの偏心量を外部からの油圧で変えて二段階、あるいは無段階に変速できるもの、また、星形の配列を軸方向に2列配置したものも製品化されている。c. 多行程形ラジアルピストンモータ多行程形ラジアルピストンモータは、図V-2・28に示すようにシリンダブロックに対応するカムハウジングの内周に波状のカムを配し、これに数本のピストンの油圧力が同時に作用して回転力を発生させる構造になっている。一般にこの形式は、原理的に次のような構造ならびに性能上の特徴を有する。(1) ケーシング回転形、あるいはケーシング固定でシリンダブロ図-2・28 多行程形ラジアルピストンモータ図-2・29差動減速式内接歯車モータック回転形の両方式を自由に選ぶことができる。(2) 起動特性が良く、また、1回転中の理論押しのけ容積の変化を限りなく零に近づけることができるので、1~2min程度まで非常に小さな回転変動となり、良好な性能が得られる。(3) ピストン推力の合力を小さくできるので、軸受が小さく寿命が長い。(4) 変速用の分配弁を使用することにより有効に作用する山数を変えて、2速あるいは3速の切換えが得られる。(5) ピストンをカム面から遊離させることにより、フリーホイーリングができる。なお、このモータでは複雑な曲率のカム面にピストン推力を伝えなければならないので、ピストン先端には転がり機構が必要であり、多種のピストン機構が用いられている。d. 多行程形ベーンモータ(vane motor)多行程形ベーンモータは、ロータ直径を大としてトルクの作用半径をできるだけ大きくとり、ロータに対応したカムリングの内周に波状のカムを配し、供給、排出口をそれぞれ3~4個もつ。1回転でベーンは3~4回ストロークするが、多行程形ラジアルピストンモータと同様に、負荷に応じて圧力油を送る供給口の数を変えるなど、2速、3速の切換えも可能である。e. 多行程形アキシアルピストンモータ(axial piston motor)多行程形アキシアルピストンモータは、カムと接触、転動する球がピストン先端に自由に回転可能に取付けられたピストンを用い、このカムのしゅう動面には駆動軸方向にほぼ正弦波状となる突起を2~3個設けた多行程形であり、占有容積に比べて大きなトルクを得ることができる。f. 差動減速式内接歯車モータ(gear motor)差動減速式内接歯車モータは、図V-2・29に示すように内接歯車をケーシングに固定し、歯数が1枚少ない外接歯車をその中で油圧により転動させ、この歯車と出力軸とを特殊な軸継手でつなぐことにより差動減速機の作用を行わせ、低速で大トルクの出力を得る形式のものである。g. 減速機付きモータ(reduction gear combined motor)斜軸式あるいは斜板式の高速モータと、二段あるいは三段の遊星減速機や差動歯車式減速機とを組合わせて低速高トルクを得る構造のもので、減速機部分を変えることによりモータ部分を変えることなく種々の高トルクを得ることができる。また、斜板の傾角を二段階に変化させ、容積を変えるものもある。図V-2・30に示すように、モータのケーシングと減速機のハウジングを共通にして、小型・軽量化を図ったものもある。
