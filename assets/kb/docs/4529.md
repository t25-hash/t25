# 2・2・8 油圧ポンプのキャビテーション

> **章**: 第2章 油圧システム
> **節**: 2・4 は内接形の歯車ポンプ・モータの原理的構造を示す.

油圧ポンプにおいて,キャビテーション(cavitation)が発生すると吐出し流量が減少し,振動や騒音が増大する.また,キャビテーション気泡の崩壊時に発生する高い衝撃圧によって,金属表面が壊食を受ける.さらに油中への気泡の混入によって,油圧システムの剛性が低下して応答性が悪くなる.ポンプ入口の圧力を욦,入口からロータ吸入部までの圧力損失を,ロータに吸入される際の圧力損失をとすると,ロータ内の圧力윯읉윰は次式となる.윯읉윰=욦--(V-2・18)ロータ吸入部の周速度をとすると,=ξρ(V-2・19)であり,ξはポンプの種類,構造,油の粘度などに依存するが,大略1~2となることが多い.歯車ポンプのξは小さいが,ピストンポンプではξが大きく,自吸が困難な場合には予圧用のポンプを別に設ける必要がある.斜板式アキシアルピストンポンプのξについては次式が与えられている.ξ=tanα0.765+1.4tanα(V-2・20)上式において,:ピストン断面積,:シリンダポート断面積,α:斜板の傾斜角である.윯읉윰が油の空気分離圧または飽和蒸気圧以下になると,キャビテーションが発生する.図V-2・33は斜板式アキシアルピストンポンプについて,αとを種々変えた場合について,式(V-2・18)~(V-2・20)による윯읉윰と容積効率η욋の関係を示したものである.ピストンポンプの場合,ピストンの下死点において,シリンダポートが弁板の吸込ポートから吐出しポートに切換わる際に,吐出し側の高圧油が噴流となってシリンダ内に流入し,シリンダ内壁に衝突してキャビテーション壊食を生じることがあり,そのための対策も種々考案されている.
