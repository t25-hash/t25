# 2・2・8 油圧ポンプのキャビテーション

> **章**: 第2章 油圧システム
> **節**: 2・4 は内接形の歯車ポンプ・モータの原理的構造を示す。

油圧ポンプにおいて、キャビテーション(cavitation)が発生すると吐出し流量が減少し、振動や騒音が増大する。また、キャビテーション気泡の崩壊時に発生する高い衝撃圧によって、金属表面が壊食を受ける。さらに油中への気泡の混入によって、油圧システムの剛性が低下して応答性が悪くなる。歯車ポンプのξは小さいが、ピストンポンプではξが大きく、自吸が困難な場合には予圧用のポンプを別に設ける必要がある。斜板式アキシアルピストンポンプのξについては次式が与えられている。ピストンポンプの場合、ピストンの下死点において、シリンダポートが弁板の吸込ポートから吐出しポートに切換わる際に、吐出し側の高圧油が噴流となってシリンダ内に流入し、シリンダ内壁に衝突してキャビテーション壊食を生じることがあり、そのための対策も種々考案されている。
