# 2・3・2 揺動形アクチュエータ

> **章**: 第2章 油圧システム

a. 分類油圧揺動形アクチュエータ(oscillating actuator,semi rotary actuator)とは,流体(油圧)のエネルギーを用いて,出力軸の回転往復運動を行わせる機械のことである.揺動形アクチュエータは,ベーン形とピストン形に分類される.ベーン形は,油圧を受けて回転ベーンがある一定角度回転する構造である.ピストン形は,油圧シリンダのピストンの動きを,内蔵したラックピニオンやヘリカルスプライン,クランク機構などを用いて回転軸の揺動運動に変換する構造である.揺動形アクチュエータを使用する場合,駆動物の大きな慣性力をアクチュエータ自身で停止させると,その衝撃によってアクチュエータが破損することがあるので,外部にストッパを設ける.また,著しく慣性力が大きい場合には,外部にショックアブソーバを使用し,緩やかに停止させるのが望ましい.b.
