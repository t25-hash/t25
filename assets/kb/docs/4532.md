# 2・4・1 分類と機能

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

油圧制御弁(hydraulic control valve,表V-2・8)は,機能面から圧力制御弁,流量制御弁,および方向制御弁と大別する分類法が最も一般的である.また構造の面からは,ほとんどがスプール弁(spool valve,図V-2・46)あるいはポペット弁(poppet valve,図V-2・47)と呼ばれる弁が基本となっている.さらに操作方式か表-2・8油圧制御弁一覧表(説明は,JIS B 0142:1994「油圧及び空気圧用語」による)分類バルブ名称圧力制御弁リリーフ弁,減圧弁,シーケンス弁,アンロード弁,カウンタバランス弁,アンロードリリーフ弁,圧力スイッチ(一定の圧力に達すると,電気的信号を出す弁)流量制御弁絞り弁,圧力補償付き流量調整弁,分流弁,集流弁,分集流弁方向制御弁切換弁,電磁弁,逆止め弁(チェック弁),パイロット操作チェック弁,シャトル弁,プレフィル弁(大型プレスの高速行程でタンクからシリンダへ油を補う弁)電気操作弁比例制御弁,サーボ弁,直動形サーボ弁(ボイスコイルなどの電磁力で流量や圧力を制御するスプールを直接駆動する弁),高速電磁弁モジュラスタック弁,ロジック弁,カートリッジ弁,複合弁[サンドイッチ形マルチプル弁(積み重ね可能な数個の同種弁を,タイボルトなどで一体に組立てた切換弁を主とした弁ユニット),モノブロック形マルチプル弁(数個の同種弁を共通の本体に組込んで一体にした切換弁を主とした弁ユニット),舶用弁(圧力補償機能を組込んだ方向制御弁に,カウンタバランス弁,ブレーキ弁を組込んだ弁ユニットで,舶用に頑丈にしたもの)]その他の弁(a) スプールの形状(b) 弁室に入る噴流(c) 弁室から出る噴流図-2・46 スプールに作用する軸力ら見ると,何らかの機械的入力(人力操作,機械操作,パイロット操作など)による機械操作弁と,電気的入力(電磁操作,比例電磁操作,トルクモータやフォースモータなど)による電気操作弁に分類される.
