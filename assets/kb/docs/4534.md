.弁前後の容積室による不安定 弁体が弾性支持された弁

.管路内油柱に基づく不安定 弁の上流側あるいは下流側

2・4・4圧力制御弁
油圧回路の圧力を制御する弁を圧力制御弁（pressure control
valve）といい，リリーフ弁，減圧弁，シーケンス弁，アンロード
弁，カウンタバランス弁などがある．
a.リリーフ弁（relief valve) 回路の圧力を設定値以内に保
持するために，流体の一部または全部を逃がす圧力制御弁であり，
直動形とバランスピストン形とがある．
.構造 バランスピストン形リリーフ弁は，図Ⅴ-2･50に
示すようにパイロット弁として働く直動形リリーフ弁Ⓐと主弁Ⓑか
ら成る．一次圧力室⑨の圧力は，小穴⑧を通って二次圧力室⑦に通
じ，主弁の弁体の両側の圧力は平衡している．一次圧力が上昇し
設定圧力に近づくと，まずパイロット弁であるポペット⑤が開き，
一次圧力室から少量の作動油が小穴を経由して流出する．そのため
小穴両側に圧力差が生じ，主弁の弁体の上側の圧力が下側より低く
なって主弁が開き，多量の一次圧力室の作動油が出口へ流出して
一次圧は設定値以下に保たれる．
.特性 リリーフ弁は，弁体に掛かる圧力が上昇して設定
圧力に近づくと，弁が開き始める．このときの圧力をクラッキング
圧力（cracking pressure）と呼ぶ．さらに圧力が上昇して設定圧

リリーフ弁は，設定圧力を順次上昇させたときと降下させたとき

から負荷状態になるとき，リリーフ弁により回路圧力が設定圧力に
達するまでの時間，そのとき発生するサージ圧力の大きさ，圧力変

どがある．このほか圧力安定性，流速音などがある．

なく，出口側圧力を入口側圧力より低い設定圧力に調整する圧力制
御弁である．リリーフ弁と同様，バランスピストン形と直動形とが
あり，後者は前者に比べて応答速度は速いがドレン量が多いので，
ポンプ流量の決定時には注意をする必要がある．

ドレンとすれば，バランスピストン形のシーケンス弁となる．圧
力-流量特性は，直動形リリーフ弁と同じである．

ータとポンプから成る油圧源回路に用いられる弁である．アキュム
レータの圧力が所定の値に達すると，アンロード弁として作用し，

2・4・5流量制御弁
流量制御弁とは，流量を制御するバルブ（弁）の総称である．絞
り弁，流量調整弁，分流弁などがある．
a.絞り弁（restrictor，metering valve) 絞り作用によって
流量を規制する流量制御弁で，圧力補償機能のないもの．開口面積
が一定な固定絞り形と可変調整形とがあり，いずれも逆止め弁（チ
ェック弁）と組合わせたものが多く使用される．
b.流量調整弁（pressure compensated flow control valve)
圧力補償機能により入口圧力や出口圧力の変化にかかわりなく，流
量を所定の値に保持する流量制御弁である（図Ⅴ-2･53）．絞り弁と
圧力差の変化を補償する定差圧減圧弁とが組合わせてある．絞り弁
前後の差圧が定差圧減圧弁により一定に保たれるため，入口圧力や
出口圧力にかかわらず流量が一定となる．さらに，作動油の温度変
化に基づく粘度変化にも対応できる温度補償付き流量制御弁もあ
る．
c .流量調整弁のジャンピング現象 流量調整弁は圧力補償機
構の応答遅れがあるため，流量が急激に変化すると，初めに大流量
が短時間流れてアクチュエータの飛び出し現象が起こる．これをジ
ャンピング現象という．
d.分流弁，集流弁，分集流弁 分流弁（flow dividing valve）
は一つの流れを任意の二つの流れに分割するバルブ，集流弁は二つ
の流れから任意の一つの流れを得るバルブ，分集流弁は両方の機能
を兼ね備えたバルブである．
