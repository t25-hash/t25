# 2・4・4 圧力制御弁

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

油圧回路の圧力を制御する弁を圧力制御弁(pressure controlvalve)といい、リリーフ弁、減圧弁、シーケンス弁、アンロード弁、カウンタバランス弁などがある。a. リリーフ弁(relief valve)回路の圧力を設定値以内に保持するために、流体の一部または全部を逃がす圧力制御弁であり、直動形とバランスピストン形とがある。. 構造バランスピストン形リリーフ弁は、図V-2・50に示すようにパイロット弁として働く直動形リリーフ弁Aと主弁Bから成る。一次圧力室9の圧力は、小穴8を通って二次圧力室7に通じ、主弁の弁体の両側の圧力は平衡している。一次圧力が上昇し設定圧力に近づくと、まずパイロット弁であるポペット5が開き、一次圧力室から少量の作動油が小穴を経由して流出する。そのため小穴両側に圧力差が生じ、主弁の弁体の上側の圧力が下側より低くなって主弁が開き、多量の一次圧力室の作動油が出口へ流出して一次圧は設定値以下に保たれる。. 特性リリーフ弁は、弁体に掛かる圧力が上昇して設定圧力に近づくと、弁が開き始める。このときの圧力をクラッキング圧力(cracking pressure)と呼ぶ。さらに圧力が上昇して設定圧力に達すると、余剰流量はすべてリリーフ弁から流出する。クラッ図-2・50 バランスピストン形リリーフ弁図-2・51 リリーフ弁の圧力オーバライド特性キング圧力と設定圧力との圧力差をオーバライド圧力(over ridepressure)と呼び、その設定圧力に対する比率、その間の圧力変動の様子を圧力オーバライド特性という。オーバライド圧力は、作動油のもつエネルギーを有効に利用するために、設定圧力との比がなるべく小さいほうが良い。図V-2・51に示すように、バランスピストン形リリーフ弁は直動形リリーフ弁より圧力オーバライド特性が良いため、通常は前者がリリーフ弁として用いられ、後者は安全弁として用いられる。リリーフ弁は、設定圧力を順次上昇させたときと降下させたときとで、同一圧力に対するリリーフ流量に若干の差が生じる。チャタリング(chattering)は、リリーフ弁が開閉するとき動作が不安定になり、往復運動して弁体が弁座をたたき騒音を発する現象で、一種の自励振動であり、回路圧力がクラッキング圧力に近いとき、流量の少ないとき、または共振する他の機器があるとき起こりやすい。リリーフ弁の応答性として重要な特性には、回路が無負荷状態から負荷状態になるとき、リリーフ弁により回路圧力が設定圧力に達するまでの時間、そのとき発生するサージ圧力の大きさ、圧力変動が収束して設定圧力に安定するまでの時間(整定時間という)などがある。このほか圧力安定性、流速音などがある。b. 減圧弁(pressure reducing valve)入口側圧力にかかわりなく、出口側圧力を入口側圧力より低い設定圧力に調整する圧力制御弁である。リリーフ弁と同様、バランスピストン形と直動形とがあり、後者は前者に比べて応答速度は速いがドレン量が多いので、ポンプ流量の決定時には注意をする必要がある。c. シーケンス弁(sequence valve)回路が二つ以上の分岐回路をもっている場合、その動作順序を回路の圧力によって制御する弁をシーケンス弁という。リリーフ弁と異なり、図V-2・52のように外部ドレンとなっている。図V-2・50の弁の詰栓2を外して外部ドレンとすれば、バランスピストン形のシーケンス弁となる。圧力-流量特性は、直動形リリーフ弁と同じである。2―239第V部フルードパワーシステム図-2・52 valve)負の負荷力が作用した場合に負荷の自由落下を防止するため、背圧を保持する圧力制御弁である。逆にシリンダの負荷を油圧で上昇する際に自由流れが得られるように、必ず逆止め弁を内蔵している。ここに示したシーケンス弁、アンロード弁、カウンタバランス弁は、基本的に同じ弁体構造を用いることによって実現可能である。図V-2・52において、ドレン用のふたとパイロット用のふたを180°回転させると、ドレンおよびパイロットを内部および外部に切換えることができ、どの弁としても使える。なお、カウンタバランス弁として使うには、中央部に逆止め弁を内蔵するものを使う必要がある。f. アンロードリリーフ弁(unload relief valve)アキュムレータとポンプから成る油圧源回路に用いられる弁である。アキュムレータの圧力が所定の値に達すると、アンロード弁として作用し、圧力が所定の値まで低下するとオンロードし、リリーフ弁として作用する。一般にアンロード圧力(カットアウト圧力)は、オンロード圧力(カットイン圧力)より20%低い。
