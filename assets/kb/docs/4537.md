# 2・4・6 方向制御弁

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

方向制御弁とは、流れの方向を制御する弁の総称である。切換弁、逆止め弁(チェック弁)、パイロット操作チェック弁、シャトル弁などがある。a. 方向制御弁の構造。 弁体の構造方向制御弁の弁体には、スプール形、ボール形、ディスク形、ポペット形がある。圧力の平衡がとりやすく操作が容易で、高圧・大流量にも対応の容易なスプール形が広く使用される。. 操作方法による分類方向制御弁は外部から操作力を加えるか、または操作用の弁を本体に取付けて操作する。操作力および操作方法により下記のように分類できる。(1) 人力操作弁:手・足でレバー・ペダルなどを操作して切換えを行う弁。(2) 機械操作弁:カム・リンク機械などによる機械力によって切換えを行う弁。(3) 油圧パイロット操作弁:弁体の端面に直接またはピストンを介して油圧を作用させ、弁体を切換える弁。(4) 空気圧パイロット操作弁:弁体に端面にピストンを介して空気圧を作用させ、弁体を切換える弁。(5) 電磁切換弁(電磁弁):弁体の軸線上に電磁石(solenoid)を取付け、その吸引力で直接弁体を動かす弁。電磁石は、鉄心が作動油に浸されている油浸形と浸されていない乾燥形とがある。また、交流電磁石と直流電磁石とがあり、前者は動作速度が速い(0.01~0.02s)が焼損しやすく、後者は動作速度が遅い(0.05~0.1s)が焼損しにくい。(6) 電磁パイロット切換弁:電磁切換弁をパイロット弁として、主弁と一体に組立てた弁。. ポートと切換位置切換弁はそれに接続するポートの数によって、2ポート弁、4ポート弁などと呼ぶ。またその切換位置の数によって、2位置弁、3位置弁などと呼ぶ。各位置でのポートの接続状態、操作方法などがJIS により記号化されている(表V-2・9,V-2・10)。表-2・94ポート切換弁の各動作位置における流路の接続状態を表す図記号と表示文字表示文字ABCDEFGH流路の記号表示文字KMNPQRS流路の記号表-2・10 図記号による切換弁の機能表示例記号備考2ポート2位置切換弁切換は人力方式4ポート3位置切換弁両ソレノイドスプリングセンタ4ポート絞り切換弁切換えが連続的に行われるもの(トレース弁)3ポート2位置切換弁片ソレノイドスプリングオフセット過渡位置は全ポート閉止b. 方向制御弁の種類。 切換弁(directional control valve)切換弁は、二つ以上の流路の接続状態をもち、2個以上のポートをもつ方向制御弁である。. 逆止め弁(check valve、チェック弁)逆止め弁は、一方向にだけ流体の流れを許し、反対方向には流れを阻止する弁である。. パイロット操作チェック弁(pilot operated check valve)パイロット圧力によって逆止め弁としての機能をオンオフできる弁である。逆止め弁としての機能をオフにした場合は両方向に流れる。内部ドレン形と外部ドレン形があり、また2個を一体にしたものを多く使用する。. シャトル弁(shuttle valve)二つの入口と一つの共通の出口をもち、出口は入口圧力の作用によって入口のいずれか一方に自動的に接続される弁。高圧側の入口が出口に接続されるものと、その反対のものとがある。
