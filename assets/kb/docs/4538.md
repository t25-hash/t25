# 2・4・7 電気操作弁(電気-油圧制御弁)

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

電気信号を連続的な油圧出力に変換する電気操作弁として,比例電磁弁とサーボ弁があり,オンオフ的に制御する弁として電磁弁(2・4・6項参照)や高速電磁弁(high speed solenoid valve)がある.a. 比例制御弁(proportional control valve)圧力制御弁・流量制御弁・方向切換弁の手動操作部,あるいは電磁操作部に比例電磁石(proportional solenoid)を用いたものが比例制御弁である(図V-2・54は比例電磁式流量制御弁).サーボ弁と比較すると応答性は劣るが,汚染感度は低く一般油圧機器と同等レベルでの使用が可能である.b. サーボ弁(servo valve)多くのサーボ弁(図V-2・55)では,まずトルクモータを使用して入力電流から得た機械的な力を,機械-油圧変換器であるノズルフラッパによって油圧力に変換する(前段増幅部).これをスプール弁の両側に作用させることによりスプール弁が開き,流量が発生して油圧エネルギーが制御される(二段式).スプール変位はフィードバックされ,入力電流に比例したスプール変位が得られるように調整される.作動油の汚染管理に注意する必要があるが,応答性は高く,一般的に閉ループで使用され図-2・54 比例電磁式流量制御弁図-2・55 サーボ弁る.二段式で,さらに大容量の主弁を駆動する三段式のものもある.c. サーボ弁の性能評価サーボ弁の性能は,周波数応答,ステップ応答,入力電流-出力流量特性などで評価される.. 周波数応答特性入力電流を一定振幅で正弦波状に非常にゆっくり変化させた場合,出力(流量・圧力)もある振幅でゆっくり変化する.入力電流の変動周波数を次第に大きくすると,出力の振幅は次第に小さくなる.ある周波数における出力の振幅を욈とするとき,振幅比욈をゲインと呼び,ふつうはデシベル値20log욈で表す.一般にゲインが-3dB に達したときの周波数を,その弁の応答周波数という.また,入力電流の周波数が大きくなると出力の位相が次第に遅れるが,この遅れ角度が90°に達したときの周波数で弁の位相遅れを示す.. ステップ応答入力電流をステップ状に変化させたときの出力をステップ応答といい,その際の出力が最終値の10~90%の間を動くのに要する時間である立上り時間,出力が最終値の±5%に収まるまでの時間である整定時間などで評価される.. 入力電流-出力流量特性入力電流を0から100%まで,また100%から0までゆっくり変化させたときの出力を計測し,バルブの入力電流-出力流量特性を把握する.行きと帰りの出力の差を表すヒステリシス,ある区間の出力の直線性を表す直線性などで評価する.d. 高速電磁弁PWM 制御によるオンオフ動作によって流量や圧力を制御する弁であり,コンピュータによる制御が簡単である.2方向形と3方向形があり(図V-2・56),ソレノイドの構造およびその駆動回路の工夫により,弁の切換時間は通常の電磁弁よりかなり短く,1~5ms程度である.
