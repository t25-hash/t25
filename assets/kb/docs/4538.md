# 3・7・4 分光法による米粒の選別

> **章**: 第3章 食品機械
> **節**: 3・7 食品プロセスセンサ（food process sensor ）

図3･23に米粒を連続的に選別するシステムを示した ．樋状の
シュートを流れる米粒が分光検出部における光源（ハロゲン光，蛍
光灯）と光センサの組合せにより不良米粒を検知し，落下の時間遅
れを見込んだうえで，選別部分においてノズルからの圧縮空気で不
良品を選別するシステムである．このシステムではシュートを増や
すことにより選別能力を高めることができる．100シュートで3～5
tの選別が可能である．このシステムは米粒のみならず，アフラト
キシン付着ピーナツの選別，コーヒー豆の選別等に用いられてい
る．原料を1個1個選別するという究極の選別法といえる．
