# 4・10・1 熱成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・10 二次成形機

熱成形（thermoforming）は，熱可塑性樹脂のシートや薄板な
どを原料として，これを加熱軟化させ，真空力や圧空力などの外力
により変形を与えて賦形し，冷却・固化して成形品を得る方法［真
空成形（vacuum forming），圧空成形（pressure forming）］であ
る．
真空成形の基本原理は，図4･51に示すように，(ⅰ)原料シート
を所定の凹形金型にクランプして加熱（シートのガラス転移点と融
点の間でゴム弾性を帯びた軟化状態）した後，(ⅱ)金型内を真空に
引いて軟化シートを金型内壁面に張り付かせて所定の形を与え，
(ⅲ)冷却固化して成形品を取出すプロセスである（ストレート法)．
この方法では，軟化シートを真空力で金型内壁に張り付かせる際
に，場所によりシートの延伸量が異なるために成形品の厚さむらが
生じ底面のコーナ部が薄くなるなどの問題があるが，この改良策と
して次のような方法が用いられている．
(１)ドレープ法は，固定枠にクランプした軟化シートを凸型で
突き上げて予備伸張した後に，真空に引いて凸型外壁に張り付かせ
て成形する方法である．この方式は深絞り製品に適するが，底部が
厚く側面上部が薄くなる傾向がある．
(２)プラグアシスト法は，固定枠にクランプした軟化シートを
凸形のプラグで凹型の内部に押込んで予張した後に，真空に引いて
凹型内壁に張り付かせる方法で，厚さむらが改良される．
(３)エアブロー法は，軟化シートをクランプした凹型内に圧搾
空気を送ってシート全体を外側に所定量だけ膨らませて予張した後
に，凹型内を真空に引いて凹型内壁に張り付かせることにより，ス
トレート法における厚さむらを軽減する方法である．
真空成形機は，一つの金型で1サイクルで1個の製品を成形する
単発機と，広幅連続シートと多数個の金型を用いて，一度に多数の
製品を連続的に成形するためのライン式連続成形機，ロータリ式成
形機，ドラム式成形機などが開発されている．
成形型は金属型が主体でアルミニウム合金，亜鉛合金などが多
い．少量生産用の廉価な木型，サンプル型用の石膏型や樹脂型も用
いられる．
熱成形法は，原料樹脂がすでにシート形状を有しているために，
低い成形圧で容易に薄肉形状の中空成形品が得られ，成形機や金型
などの生産コストを低減でき，生産エネルギーの面でも有利であ
る．その結果，低コスト大量生産用途として，食品包装用容器やブ
リスタ包装などに多く応用され，さらには冷蔵庫の内装部品や浴
槽，自動車部品などの大型品にも用いられる．
