# 2・4・8 その他のバルブ

> **章**: 第2章 油圧システム
> **節**: 2・4 油圧制御弁

a. モジュラスタック弁(modular stack valve)電磁弁と取付ブロックとの間に挿入して使用するのが、モジュラスタック弁である。リリーフ弁、減圧弁、シーケンス弁、絞り弁、チェック弁などがあり、挿入するバルブを変更することによりさまざまな回路に対して簡単に対応できるので、工作機械用などへ広く応用されている。b. ロジック弁(logic valve)ロジック弁は、ロジックエレメントと呼ばれるポペット形状のバルブのパイロットを制御し、各種の機能をもたせたものである。通常、四つのエレメントを使用して4ポート多機能弁としてまとめられる。ロジック弁は、エレメントがポペット形でオーバラップがないために切換時の応答が早く、圧力降下も小さく、内部漏れも少ない。c. カートリッジ弁(cartridge valve)カートリッジ弁は、圧力制御・流量制御・方向切換機能のほとんどのものを、本体を設けずに内蔵部品のみをカートリッジ形としたものである。マニホル(b) 3方向形(a) 2方向形2―241第V部フルードパワーシステムドブロックの設計・加工は多少面倒になるが、コンパクトになり、建設機械に使用されている。d. 複合弁複合弁は、さまざまな機能を組合わせた弁である。代表的なものとして、トラッククレーンなどに使用されるサンドイッチ形マルチプルコントロール弁、油圧ショベルなどに使用されるモノブロック形マルチプルコントロール弁、船舶機械などに使用される舶用弁などがある。
