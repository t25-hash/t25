# 4・10・2 スタンピング成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・10 二次成形機

スタンピング成形（stamping molding）は，(ⅰ)熱可塑性プラ
スチックを基材として長繊維で強化した1～5mm厚のシート（ス
タンパブルシート）を作成し，(ⅱ)このシートから製品の形状に合
わせて切断した材料（ブランク）をフリーの状態で赤外線ヒータな
どで融点以上に加熱した後，(ⅲ)これをプレス金型の所定位置に置
いて常温付近の温度でプレス加圧を行って，所定形状を付与すると
同時に冷却・固定して成形品を取出すプロセスである．
スタンパブルシートは使用するマトリックス樹脂と強化繊維の種
類により種々の組合せが可能であるが，工業的な主流はポリプロピ
レンとガラス繊維の組合せである．現在おもに用いられているスタ
ンパブルシートの製造法としては，PPシートの間にガラスマット
を積層してラミネータで加熱，溶融，圧着含浸させて作成する．
ハイサイクル化による生産性向上のために，各工程の自動化や多
数個取り成形，金型冷却法の効率化，金型上下のすり合わせ部分の
精度向上によるバリ発生の低減などの技術的改良が進められてい
る．
図4･51ストレート法真空成形金型
スタンピング成形の特徴は，ガラス長繊維マットで補強されたシ
ートを金型に挟んで圧縮成形するので，他の方法の成形品より機械
的特性，低温衝撃特性，クリープや疲労などの耐久性に優れてい
る．この成形技術により，大型で複雑な形状の部品も短いサイクル
時間で一体成形が可能であり，従来の鉄鋼材料の半分以下に軽量化
できることなどから，自動車のバンパビーム，バッテリトレーなど
の強度部品，あるいはコンクリート型枠など非自動車分野での展開
も行われている．
