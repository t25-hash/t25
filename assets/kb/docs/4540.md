# 2・5・1 油圧用フィルタ

> **章**: 第2章 油圧システム
> **節**: 2・5 油圧付属機器

油圧作動油の汚染管理(contamination control)は,油圧システムの信頼性,寿命に大きく影響し,油圧システムの高圧化,高速化,小型・軽量化により正常な機能を長期にわたり発揮するために,適切なフィルタ(filter)の選択,適用がますます重要になっている.フィルタは,オイルリザーバ内に設置されるタンクフィルタ,配管部に取付けられる管路用フィルタに大別される.フィルタのエレメントは,ノッチワイヤ,巻き線,積層金属板,金網などにおける表面ろ過,フェルト,樹脂,焼結金属における内部ろ過,フェノール樹脂を含浸処理し,プリーツ成形したろ紙などの中間ろ過に大別される.一般に,表面ろ過では作動油は外側から内側に流れて,汚染粒子はエレメントの表面のみで捕捉される.その捕捉量は少なく,また,30 m以下のろ過粒度のものを製作することが難しいが,捕捉できる粒子の最大径が決まり,洗浄により反復使用可能でろ材の脱落がなく圧力損失も少ない.内部ろ過では,汚染粒子は厚みのあるエレメントの内部で捕捉されるため,捕捉量が大きく寿命が長いが,ろ過粒度の規定ができず,ろ材の脱落の懸念,圧力損失が比較的大きいなどの欠点が挙げられる.中間ろ過ではろ過面積が大きく,捕捉できる粒子をある程度限定できるため小型・軽量である.フィルタの圧力降下,安全弁の作動,ろ過粒度などに関する性能と試験方法に関してJIS B 8356:2000にて規定されている.なお,フィルタエレメントの交換時期を明らかにするために,通過流量によって生じるエレメントの前後に発生する差圧の表示器を取付けたものもある.図V-2・57に代表的な管路用フィルタを示す.
