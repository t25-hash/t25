# 4・11・1 スプレーアップ装置

> **章**: 第4章 プラスチック加工機械
> **節**: 4・11 繊維強化プラスチックの成形機

FRP成形法で最も簡単なものは，繊維に液状樹脂を含浸させな
がら手作業で積層し，所定の形状と厚さにして硬化させる方法であ
り，これをハンドレイアップ（hand- lay-up）法という．スプレー
アップ（spray- up）はハンドレイアップの作業をやや機械化した
もので，型の上に繊維と樹脂を同時に吹付けて積層する．スプレー
アップ装置は，樹脂と硬化剤を混合させる部分とスプレーガンから
なり，スプレーガンノズルの上部に取付けられたロービングカッタ
から所定の長さにカットされた繊維を噴射樹脂の上に落下させ，樹
脂と繊維を同時に型上に吹付ける．
