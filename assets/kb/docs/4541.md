# 2・5・2 アキュムレータ

> **章**: 第2章 油圧システム
> **節**: 2・5 油圧付属機器

アキュムレータ(accumulator)は,高圧容器の中に高圧の窒素ガスを封入し,油圧回路の圧力変化に伴う窒素ガスの体積変化により,作動油を容器内に吸入,または排出して,主として油圧エネルギーの蓄積,脈動,衝撃圧力の吸収の機能を得る.窒素ガスの封入圧力は,エネルギー蓄積用としてはシステム最低使用圧力の80~90%,脈動吸収用としてはシステム平均作動圧力の60%,さ図-2・57 管路用フィルタ図-2・58 ブラダ形アキュムレータ図-2・56 高速電磁弁2―242流体機械らに弁閉止時のサージ圧力の吸収用としては,弁閉止前の弁上流側圧力の60~70%が推奨される.アキュムレータは,窒素ガスと作動油の分離法により,ブラダ形,ピストン形,ダイアフラム形に大別され,近年,金属ベローズ形も使用されている.また,圧力容器については,一般的に鋳鋼を含む鉄系鋼材が用いられるが,軽量化が要求される油圧システムではチタン,アルミニウム合金,さらにはカーボンファイバ製のものも用いられる.図V-2・58は,最も一般的なブラダ形である.
