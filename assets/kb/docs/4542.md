# 2・5・3 熱交換器

> **章**: 第2章 油圧システム
> **節**: 2・5 油圧付属機器

油圧装置では,その損失動力の多くが最終的に熱エネルギーに変換され,作動油の温度を上昇させるが,油温の過度の上昇は作動油の劣化を早め,また作動油の潤滑性を低下させ,装置の作動不良の原因となる.一方,作動油温度が著しく低下すると,サーマルショックを起こしたり,作動油が高粘度状態になり圧力損失の増大を招き,装置の作動不良の原因となる.油圧装置の油温としては,通常作業時油温として30~70°C,始動時,高温側作業限界を含めても16~80°C程度の範囲であり,油温を適切に保つために戻り管路にオイルクーラ(oil cooler),またはオイルリザーバ内にヒータが設置される.オイルクーラには水冷式と空冷式があり,前者はさらに多管式と平板式が多く使用される.図V-2・59は多管式熱交換器の一例であり,管内を冷却水が流れ,作動油が管の周りを流れて熱交換する.空冷式は冷却水が用意できない場合にも用いられ,冷却水用の配管が不要であるが,交換熱量が水冷式に比べて小さく,フィンプレートで表面積を大きくし,冷却用ファンで風速を上げて熱交換量を増大している.なお,ヒータには電熱式,蒸気式があり,作動油の劣化を防ぐために,局部的に温度が上がりすぎないように作動油をかくはんしたり,温度センサを用いて油温を制御する必要がある.
