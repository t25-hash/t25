# 2・5・4 オイルリザーバ

> **章**: 第2章 油圧システム
> **節**: 2・5 油圧付属機器

オイルリザーバ(oil reservoir)は作動油を蓄えるとともに、主として、1壁面からの放熱による作動油の冷却、2作動油中の気泡の浮上、ポンプ吸込口近くへの金網の設置等による気泡の吸込防止・低減、3作動油中の粉じんの沈殿などの機能を有する。戻り管路からの作動油の流速を低下し、吸込管路到達までに気泡の浮上、汚染物の沈殿を実現するとともに、急激な油温上昇を防止図-2・59多管式熱交換器するために、タンク内油量は一般に、油圧ポンプの1分間の吐出し量の3~7倍が推奨されてきた。これに対して、車載の油圧システムでは油量は極端に少なく、油圧パワーステアリングシステムでは、ポンプの1分間の吐出し量の10~20%程度の油量である。近年、産業用油圧装置においてもフィルトレーション、温度管理、気泡除去を考慮した設計により、ポンプの1分間の吐出し量の1~2倍に油量を減らす例も増加している。これらの機能を強化するものとして、鉄系の摩耗粉を永久磁石により吸着するもの、また、戻り管路に設置して旋回流による遠心力場を発生させ、旋回流の中心に集まる気泡を除去する装置が開発されている。
