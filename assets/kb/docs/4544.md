# 4・11・4 FW成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・11 繊維強化プラスチックの成形機

れる成形を行う装置のことであり，ガラスや炭素からなる繊維束性を重要視した成形方法であり，大型タンクの胴体や地中埋設管と4･55に示すように(ａ)サーカムファレンシャルワインダ，(ｂ)ヘニロン繊維，ナイロン繊維を用いる場合もある．
樹脂材料には，コる場合が多く，次いでエポキシ樹脂，フェノール樹脂などが用いらが上昇する．
繊維束をマンドレルに巻付ける作業は，成形品が大型要となる．
を通して引出し，加熱硬化させて成形する方法を引抜成形法オートクレーブ（autoclave）とは圧力・温度の制御が可能な釜フィラメントワインディング（FW:filament winding）と呼ばリカルワインダ，(ｃ)ポーラワインダなどがある．
強化繊維はガラ巻付けた後，加熱硬化させる．
成形品の性能よりも生産いった大型成形品の生産にも利用されている．
巻付け方式には図ス繊維や炭素繊維が主流であるが，用途に応じてアラミド繊維やビストや性能の点において一般的には不飽和ポリエステル樹脂を用いれる．
