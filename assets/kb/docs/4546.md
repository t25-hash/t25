# 2・6・2 圧力制御回路

> **章**: 第2章 油圧システム
> **節**: 2・6 油圧回路

a. 圧力遠隔制御回路回路の圧力を制限するためにリリーフ弁が使用されるが,リリーフ弁の設定圧力を多段階に切換えることができる.図V-2・66に示すように,リリーフ弁のパイロット圧力を高低二段の設定圧力をもつ2個のリリーフ弁と切換弁を介して接続することで,回路圧力を遠隔制御することができる.b. 減圧回路一つの油圧源で複数の回路に油圧を供給する場合,図V-2・67に示すように減圧弁を回路の入口に設ければ,ポンプ側(一次側)の供給圧力の変動によらず,リリーフ弁を設置した二次側の圧力を一定に保つことができる.c. カウンタバランス回路(counter balance circuit)負荷を重力方向につり下げる場合,下降時の負荷の加速を抑える回路をカウンタバランス回路という.図V-2・68では,直動形カウンタバランス弁を用いて負荷の下降時にロッド側に背圧を与え,負荷の落下による加速を抑える回路である.d. ブレーキ回路油圧モータで慣性の大きな負荷を駆動する図-2・66 リリーフ弁による圧力遠隔制御回路図-2・67 減圧回路2―244流体機械回路で,切換弁によってモータを急停止させようとすると,大きなサージ圧力が発生する.図V-2・69の回路は,リリーフ弁と逆止め弁によるブレーキ回路を用いて,サージ圧力の抑制を図る回路である.
