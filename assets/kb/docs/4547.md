# 2・6・3 速度制御回路

> **章**: 第2章 油圧システム
> **節**: 2・6 油圧回路

速度制御の基本回路として,メータイン回路,メータアウト回路,ブリードオフ回路がある.可変容量形ポンプを用いて速度制御を行うこともできる.インバータ駆動による速度制御回路は省エネルギー回路として利用される.a. メータイン回路(meter-in circuit)シリンダや油圧モータ等のアクチュエータの入口側に流量調整弁を設ける回路である(図V-2・70).アクチュエータの速度は,この流量調整弁の設定により制御される.アクチュエータの負荷がつねに正の場合に用いる.背圧を得ることができないが,アクチュエータの速度はポンプの吐出し流量の変動には影響されない.b. メータアウト回路(meter-out circuit)アクチュエータの出口側に流量調整弁を設ける回路である(図V-2・71).アクチュエータの速度はこの流量調整弁によって制御される.アクチュエータに背圧を与えながら速度制御を行う場合に用いられる.回路効率は低いが,負の負荷を与えることが可能である.c. ブリードオフ回路(breed-offcircuit)ポンプとアクチュエータの間で回路を分岐させて流量調整弁を設置する回路である(図V-2・72).ポンプからアクチュエータに供給される流量の一部を,この流量調整弁を介してタンクに戻すことにより,アクチュエータの速度を制御する.流量調整弁はポンプの全流量を通す必要がないので小型のものでよく,回路効率が高い.アクチュエータの速度はポンプの流量変動の影響を受ける.d. 可変容量形ポンプによる速度制御回路可変容量形ポンプを用い,ポンプの容量を調整してアクチュエータの速度を制御する回路である(図V-2・73).ポンプ容量を調整する方法として,手動式,機械式,油圧式,および電気油圧式などがある.図-2・69ブレーキ回路図-2・75 同期制御回路図-2・71 メータアウト回路図-2・70 メータイン回路e. インバータ制御による速度制御回路電気モータの回転数をインバータによって調整することでポンプ吐出し流量を制御し,アクチュエータの速度制御を行う回路である(図V-2・74).定容量形ポンプを使用することができる.アクチュエータが油圧を必要とするときだけポンプを駆動すればよく,省エネルギー化に有効である.
