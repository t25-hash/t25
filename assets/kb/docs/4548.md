# 4・12・2 二次加工の際の表面の重要性

> **章**: 第4章 プラスチック加工機械
> **節**: 4・12 二次加工装置

二次加工の大部分，すなわち図4･59(Ａ)の接着および(Ｂ)のすプラスチックの表面状態についての十分な知識をもっていないと満表4･1は各種材料の表面張力を示したものである．
なおポリマに臨界表面張力γで表現している．
プラスチックとして用いら張力が100～200mN/m程度，金属の表面張力が数百～1000mN/的な低エネルギーであり，その表面を十分に濡らして接着，塗装やであることが多い．
せ，次にこの面に機能性薄膜を作ることが必要である．
べてはプラスチックの表面または界面が関与している．
したがって足できる加工技術および装置を駆使することは困難である．
ついては固体状態での真の表面張力を測定することが困難なので，れるポリマのγは18.5～46mN/mの範囲にあり，無機物の表面m程度であるのに比べて著しく小さい．
すなわちその表面は典型各種の表面処理を行うには特殊な操作や装置を活用することが必要このような表面処理では，原理的にはまず表面に活性層を形成さこれらの間の関係を示したものである．
現実には活性層の形成離して考えるのが妥当であろう．
図4･59(Ｂ)の表面極性化処理にそれぞれ専用機が開発されているが，この表面極性化処理により通化される．
注：ポリマの値は室温での臨界表面張力．
タライジング等による機能性薄膜の形成がなされるのである．
