# 4・12・2 二次加工の際の表面の重要性

> **章**: 第4章 プラスチック加工機械
> **節**: 4・12 二次加工装置

二次加工の大部分，すなわち図4･59(Ａ)の接着および(Ｂ)のす
べてはプラスチックの表面または界面が関与している．したがって
プラスチックの表面状態についての十分な知識をもっていないと満
足できる加工技術および装置を駆使することは困難である．
表4･1は各種材料の表面張力を示したものである．なおポリマに
ついては固体状態での真の表面張力を測定することが困難なので，
臨界表面張力γで表現している ．プラスチックとして用いら
れるポリマのγは18.5～46mN/mの範囲にあり，無機物の表面
張力が100～200mN/m程度，金属の表面張力が数百～1000mN/
m程度であるのに比べて著しく小さい．すなわちその表面は典型
的な低エネルギーであり，その表面を十分に濡らして接着，塗装や
各種の表面処理を行うには特殊な操作や装置を活用することが必要
であることが多い．
このような表面処理では，原理的にはまず表面に活性層を形成さ
せ，次にこの面に機能性薄膜を作ることが必要である．図4･60は
これらの間の関係を示したものである ．現実には活性層の形成
が即機能性膜の形成と直結する場合があるが，その各々の意義を分
離して考えるのが妥当であろう．図4･59(Ｂ)の表面極性化処理に
