# 3・1・1 絞り要素を通過する流量

> **章**: 第3章 空気圧システム
> **節**: 3・1 空気圧システムの基本特性

絞り(restriction)において、その前後の圧力差と流量との関係は、ベルヌーイの定理を用いて次式の関係で表される。また一定の差圧に対して、機器を通過する水の流量を定義とした値[gal/(min・psi)、インチ系]や値[(m/(h・bar)、メートル系]も、空気圧機器の通過能力の指標として用いられることがある。空気圧システムで使用される速度制御弁、切換弁などの各種弁の流量特性も式(V-3・1)の特性に近いものもあるが、それらの大部分では流れは複雑であり、多少異なったものとなっている。特に、式(V-3・1)では臨界圧力比は0.528となり固定された値をとるが、実際の空気圧機器の流量特性を調べると、臨界圧力比は機器により値がまちまちである。そこでJIS B 8390:2000「空気圧―圧縮性流体用機器―流量特性の試験方法」では、圧縮性流体用機器の圧力流量特性(pressure-flow characteristics)を次式により表示することを取り決めている。音速コンダクタンスが上記の単位であるとき、当然のことながら式(V-3・2)中において計算時の圧力の単位はbarとなる。上流圧が一定のもとで下流圧を変えた場合の圧力流量特性を、をパラメータとして式(V-3・2)により算出し、図示したのが図V-3・1である。式(V-3・2)で=0.528としたとき、式(V-3・1)とは音速域では完全に一致する。式(V-3・2)における亜音速領域の式は、式(V-3・1)のそれをだ円近似したものであり、亜音速域での差異は流量で最大3%程度である。そのため、図V-3・1には式(V-3・1)の計算結果も示してあるが、重なって見える。JIS には,の測定法も記されており、それによると臨界圧力比は、厳密に流量が一定となる圧力比を意味するのではなく、あくまで亜音速域の流量特性が、上手く近似するように求めたものである。式(V-3・2)による表示と実際の機器との流量の差異は、10%以内であることが確かめられている。
