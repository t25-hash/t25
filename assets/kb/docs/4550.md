# 4・2・1 造粒（pelletizing）の目的

> **章**: 第4章 プラスチック加工機械
> **節**: 4・2 混練機

プラスチックは重合反応によって製造され，その性状はプラスチ
ックの種類や重合方法により粉末状（ビーズ状，フレーク状を含
む）と溶融状の2種類に分けられる．さらにプラスチックは熱可塑
性樹脂と熱硬化性樹脂に分類される．熱硬化性樹脂は打錠機により
圧縮後ペレット化されるものと，押出機により溶融，混練された後
ペレット化されるものとがある．一方，熱可塑性樹脂は硬質ポリ塩
化ビニル（RPVC）を除いてほとんど溶融，混練された後ペレット
化される．そして二次加工業者に供給されている．ペレット化の目
的は次のような理由による．
に造粒する必要がある．粉末原料も一般に引火爆発性を有し，また
長期間粉末貯蔵すると酸化劣化を生じるため各種添加剤を適量混練
する必要がある．またペレット化することによって，取扱い，輸送
などの作業性が容易となる．
物性，加工条件に応じて種々の添加剤，フィラー類を適量混練する
必要がある．
均一混練することにより最終製品の諸物性を改善する．例えば，低
分子量物と高分子量物から構成された樹脂のフィルム強度の向上，
高圧法ポリエチレン（LDPE）の光学的性質の改善などがある．
マ，揮発成分および水分などを押出機内で溶融，混練しながら真空
下で脱気処理して取り除く必要がある．

灰分，触媒などの異物を造粒の際に取り除く必要がある．
混練機（mixer ）はバッチ式と連続式に大別される．バッチ式は
あらかじめ計量された樹脂，フィラー，添加剤を所望の割合で溶
融，混練し，次工程の押出機やロール機に供給して造粒する方式で
ある．今日，カーボンブラック，酸化チタンなどのマスタバッチの
製造に利用されている．連続式は単軸スクリュー式，二軸スクリュ
ー式および多軸スクリュー式に大別される．歴史的に単軸スクリュ
ー押出機（single screw extruder ）が古くから使用されている．最
近では溶融，混練性能を向上させたミキシング型スクリューが使用
されている．一方，二軸スクリュー押出機（twin screw extruder ）
は生産性，混練性能において単軸スクリュー押出機よりも数段優れ
ていることから，今日プラスチックのコンパウンディング工程に主
要装置として使用されている．この二軸スクリュー押出機はスクリ
ューの回転方向，かみあい形式などにより分類される．これらを大
別すると，かみあい型と非かみあい型，さらに同方向回転と異方向
回転になる．工業的には，非かみあい型異方向回転二軸スクリュー
押出機（non-intermeshed counter-rotation twin screw
extruder ），かみあい型同方向回転二軸スクリュー押出機（inter-
meshed co- rotation twin screw extruder ）が広く使用されてい
る．非かみあい型異方向回転二軸スクリュー押出機はスクリューの
/（スクリュー長さと直径との比）が比較的小さく（ /＝
5～10），かつスクリュー軸は両端で支持されているので高速で回転
可能な構造である．また単軸押出機や歯車ポンプなどと組合わせて
用いられる．ポリエチレン（PE），ポリプロピレン（PP）樹脂な
どの大容量造粒機として幅広く使用されている．図4･1，4･2に非
かみあい型異方向回転二軸スクリュー押出機とそのスクリューを示
す．かみあい型同方向回転二軸スクリュー押出機は一般にスクリュ
ー / が大きい（ / ＝40～60）．このためフレキシビリティに
富んでおり，複合剤などを含むプラスチック材料の混練，ポリマブ
レンド，脱気，リアクティブプロセッシングなどに適している．図
