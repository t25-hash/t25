# 3・1・2 空気圧容器への充てんと放出

> **章**: 第3章 空気圧システム
> **節**: 3・1 空気圧システムの基本特性

a. 解析のための基礎式図V-3・2のように、一定体積の容器と絞りから成る抵抗容量系の圧力応答を考える。この系に成立する方程式は、流量に関する式(V-3・2)に加え、式(V-3・3)の気体の状態方程式と、熱力学の第一法則より導かれる式(V-3・4)のエネルギー方程式である。等温変化、ポリトロープ変化などを仮定すれば、熱力学的な関係から温度変化の項が消去でき、エネルギー方程式を用いなくてもよい。エネルギー方程式の右辺は1項目から流入する内部エネルギー、流動仕事、流入する熱量を表し、これらの和が左辺の内部エネルギー変化に等しくなる。示したエネルギー方程式は放出時であり、充てん時は、絞りで空気温度が低下しても、その直後にはかなり温度回復を示すことから、第1項目だけ異なり、流入する内部エネルギーの温度を室温にとる。熱伝達率の値は実験的にしか求まらないが、これを一定としても等温変化等より現実的な応答に近づく。熱伝達は、一次遅れ系で近似される抵抗容量系に位相進み的な効果を与え、このことにより圧力の過渡応答は、初期では断熱変化の応答に近く、その後、等温変化の応答へと移る。b. 圧力応答等温変化を仮定して、真空から供給圧力(=600kPa)まで充てんするときの応答は、図V-3・3のようになる。ただし=0.528としている。絞りが音速状態にあるとき、圧力は直線的に上昇する。亜音速域の圧力応答を求めると正弦曲線になるので、充てんが完了する時間はより長くなる。放出時の圧力応答は、図V-3・4のように音速域では指数関数的になり、亜音速域では正弦関数となって、さらに応答が遅くなる。結果として放出時間は充てん時間より2倍ほど長く、空気圧システムの非線形性が強いといわれる一因ともなっている。の違いによる放出時間の変化は、充てんのそれと比べて大きい。温度変化を考慮した場合、が小さいほど断熱変化に近づくので、圧力応答は速くなる。による応答速度の差は、断熱変化(=0)と等温変化(=∞)の間で最大となり、1.4倍程度である。以上のことを考慮すれば、圧力応答を求めなくてものみを計算することで応答を予測できる。また、容器の体積が既知であるとき、圧力応答から音速コンダクタンスと臨界圧力比を簡易的に測定することが可能で、規格化されている。
