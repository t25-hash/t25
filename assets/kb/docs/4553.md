# 3・1・4 空気圧シリンダシステムの設計法

> **章**: 第3章 空気圧システム
> **節**: 3・1 空気圧システムの基本特性

シリンダを用いて物体を搬送しようとするとき、一般にその質量と移動距離、移動時間が駆動条件として与えられる。これらの条件を満たすように、使用するシリンダ、速度制御弁の大きさを決定することがシリンダシステムの設計(選定、sizing)に当たる。この手順は大まかに図V-3・7のような流れになる。シリンダの運動は主として慣性係数と負荷率で決まる。そこで選定方法は、これらの値に基づいて行われる。a. 負荷率の決定とシリンダ径の計算≦0.8の範囲で負荷率を決定する。この条件は排気側の絞りが閉そく状態となることから導かれ、これを満たすと、基準速度が負荷に依存しないというメータアウト特有の性質が得られる。負荷率に余裕がないとシリンダを駆動できないという事態を生じ、逆にありすぎると効率が悪化する。一般に速度が比較的遅く、負荷の摩擦や供給圧力の状態がよい場合は0.5~0.7程度、逆の場合は0.3~0.5が推奨されている。使用可能な供給圧力は空気圧源より決まるので、負荷率から式(V-3・12)を用いて必要なシリンダ径が計算できる。当然、ストロークは移動距離となる。b. 慣性係数の計算と排気コンダクタンスの決定排気コンダクタンスを仮定すると、慣性係数が計算可能となる。仮定した慣性係数の値とa項で決定した負荷率から、図V-3・8を使って時間係数を求め、移動時間を次式により算出する。욦=욈(V-3・13)時間係数は図表で与えられることが多いが、あくまで図V-3・8はその一例であり、給気側の絞りや配管容積などもパラメータとした図表など多様なものが存在する。通常、シリンダは≧10で使用されることが圧倒的に多い。この範囲では運動中の平均速度はとみなしてよく、運動時間は욈で一定である。このことを念頭におくと選定が容易となる。図表等がない場合は簡易的に次のように計算できる。. 負荷の慣性力が小さい場合(≧10)無負荷の場合はが移動時間となる。小さくとも慣性がある場合は、運動時間욈に運動を始めるまでの時間遅れが加わり、移動時間は概略、次式となる。욦=욈1+ln1-욈1+2(V-3・14)式(V-3・14)が適用できる範囲は供給圧力が300~600kPaで、負荷率が≦0.4のときである。速度の減衰が悪い場合を式(V-3・14)は考慮しておらず、そのときには、若干速度の変動分だけ移動時間は式から外れる。. 負荷の慣性力が大きい場合(<10)慣性が相対的に大きく排気にかかる時間が無視でき、運動中も内圧は給気側が供給圧力、排気側は大気圧に保たれると仮定できるならば、負荷の摩擦抵抗を無視するとき移動に要する時間は次式となる。욦=2욂욃욦-욀=욈1-욀(V-3・15)図-3・7 シリンダシステム設計のフローチャート図-3・8 時間係数を見出す図表一般に仮定は成り立たないが、式(V-3・15)は<1であれば図V-的に욈に近づく。が小さくなるにつれ反比例して移動時間が増大する傾向は、負荷率≠0も同様である。このことから、0の場合は式(V-3・15)に負荷率による排気遅れの時間だけ加えればよい。c. クッション容量と圧縮機吐出し量の確認シリンダ径が32mmを超えるものには、クッション機構を備えたものがあり、シリンダ終端では、これにより制動される。このクッション機構にピストンが突入する際に許容される運動エネルギーが決まっており、シリンダ速度がその上限以内であることを確認する必要がある。突入速度は<10のときは計算を行う必要があるが、≧10であればである。上限の範囲外であれば負荷率、慣性係数などを再度検討する。また、クッションでは吸収しきれない場合や、ゴムクッションでは衝突により支障をきたすときには、外部に緩衝器を設ける方法もとられる。最後にシリンダの空気消費量を計算し、圧縮機の必要吐出し量を決定する。圧縮機が既設の場合は圧縮機増設の要否を検討する。現在、シミュレーション技術の発達により、シリンダ応答を基礎式から直接計算することが容易になり、各空気圧機器メーカから、計算機を使ったシリンダシステムの設計ソフトウェアが提供されている。これにより機器の選択を行うと、移動時間や応答曲線が計算されたり、駆動条件を入力するだけで機器が選定されたりするなど、設計に必要な計算の大部分は自動的に行われる状況となっている。
