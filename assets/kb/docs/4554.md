# 3・1・5 合成有効断面積(composite effective area)

> **章**: 第3章 空気圧システム
> **節**: 3・8 の負荷率=0のときの曲線に近く、>1では式を外れて漸近

シリンダの速度調節は速度制御弁により行うが、3・1・4項の計算において、これに接続される管路、電磁弁の音速コンダクタンスが速度制御弁に比して無視できないとき、この影響を考えることが必要である。このとき空気圧システムでは合成有効断面積という考え方がよく用いられる。このとき管路の有効断面積も、管摩擦による定常的な圧力損失から近似的に置換える方法がとられる。有効断面積と音速コンダクタンスは等価であるため、音速コンダクタンスについても、有効断面積を音速コンダクタンスに置換えるだけで同様な計算が行える。これらの式は、非圧縮性流体が絞り要素を通過する際の圧力流量特性を基にしており、あくまで近似であることから、使用時には誤差に注意する必要がある。
