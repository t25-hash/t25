# 3・1・6 アクチュエータの理論動力、空気消費量および効率

> **章**: 第3章 空気圧システム
> **節**: 3・8 の負荷率=0のときの曲線に近く、>1では式を外れて漸近

ここに、:単位時間当たりのストローク数、:締切率、ε:すきま容積率である。また、図V-3・9のbc間を断熱膨張としている。c. 効率(efficiency)効率を計測する際には、シリンダの動力と実気体消費量だけを測定することが多い。流入した圧縮空気の有効エネルギー(エクセルギー、exergy)を考えると、大気圧まで等温膨張させたときに最大仕事が得られ、このことから空気圧アクチュエータ効率ηは、をこの空気の有効エネルギーで除した次式の値となる。また、無膨張アクチュエータ効率や、等温ではなく断熱膨張させたときに得られた仕事により算出する、完全断熱膨張アクチュエータ効率も慣習として用いられている。空気圧アクチュエータ(pneumatic actuator)の基本的な構造と作動原理は、対応する油圧アクチュエータと同じである。ただし、作動流体である空気が大きな圧縮性を有している点、および粘性の小さい点が油圧システムと比べたときの大きな相違点である。この違いに伴う構造上および特性上の差が存在する。空気圧アクチュエータの特徴は、システムの構成の容易さ、簡便さ、低コスト性や安全性などにある。圧縮空気で作動することから、引火爆発性ガスの発生する場所で使用しても危険がなく、防爆システムを容易に組むことができる。重量、容積に対する出力が比較的大きいので、装置を小型、軽量にできる。また操作性も良いことから、自動組立機械をはじめとする各種自動化、および省力化装置に利用されている。
