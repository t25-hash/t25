# 3・2・2 空気圧シリンダ

> **章**: 第3章 空気圧システム
> **節**: 3・2 空気圧アクチュエータ

空気圧シリンダ(pneumatic cylinder)の中ではん用品として一般によく用いられるのは,片ロッドの単動形および複動形である.図V-3・10は最も一般的な複動形シリンダの例であり,円筒状のシリンダチューブの両端をヘッドカバー,ロッドカバーで覆い,このカバーを4本のタイロッドで締結し,生じた円筒状室の内部にピストンを設け,ピストンロッドを装置したものである.ピストン前後の圧縮空気を給排してピストンの前あるいは後ろに圧力を加え,ピストンの出力をピストンロッドに伝えて外部に仕事を行う.このように,ピストンロッドがシリンダチューブの両端を覆う両カバーの片側にだけ貫通しているシリンダを片ロッド形と呼んでいる.これに対し,ピストンの前後両側にピストンロッドが装着され,両カバーを貫通している構造を両ロッド形という.シリンダの内部には,空気漏れを防ぐためのパッキンやシールが適所にそれぞれ機能的に設けられ,さらにシリンダの機能に付加してクッション機構が設けられている.空気圧シリンダの中には,ピストンロッドの代わりに柔軟なワイヤやロープ,チェーンなどを用いた構造のもの,シリンダチューブの一部が可撓性シートベルトになっていて,ピストンとテーブルは直接連結されているが,ピストンの作動に沿ってシールしていく構造のもの,ピストンと外部のテーブルを磁石などで間接的に連結させたものがある.これらのロッドレスシリンダは,従来のシリンダに対してロッドが出ていくことがないので,シリンダの設置スペースが約半分にできる.図V-3・11にロッドレスシリンダの一例を示す.表V-3・1にロッドレスシリンダの方式と構造図をまとめて示す.図-3・10 空気圧シリンダの内部構造また,シリンダのストロークを,途中でロッドにブレーキをかけることにより中間で停止させたり,ピストン位置をその場で保持させるブレーキ付きシリンダがある.さらにピストンの位置をセンサで検知する磁石をピストンに埋め込み,シリンダチューブ外側のリミットスイッチなどによって検知するリミットスイッチ付きシリンダなどがある.なお,空気圧シリンダのサイズ等についてはJIS B 8368:2002「空気圧シリンダ取付寸法」を参照されたい.
