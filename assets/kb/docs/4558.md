# 3・2・3 空気圧モータ

> **章**: 第3章 空気圧システム
> **節**: 3・2 空気圧アクチュエータ

空気圧モータ(pneumatic motor)は,コンプレッサによって得られた圧縮空気のもつ流体エネルギーを,機械的回転エネルギーに変換するアクチュエータである.その出力は比較的小さく,20kW程度までのものが一般産業用機械として用いられている.a. ベーンモータ(vane motor)ベーンモータは一般に小・中容量で,高速回転を必要とする分野において多く使用されており,空気消費量はピストン形空気圧モータに比べて多いが,小型軽量で,構造が簡単である.図-3・11 磁石方式ロッドレスシリンダ表-3・1 ロッドレスシリンダ一覧表方式構造図メガジョイント方式磁石方式ケーブル方式チェーン方式チューブ方式その出力は0.1~20kW 程度であり,回転速度は500~25000min程度である.ベーンモータの構造を図V-3・12に示す.ベーンの枚数は一般的に4,5枚である.b. ピストンモータ(piston motor)ピストンモータは中・大容量で,低速回転を必要とする用途に多く使用されており,起動トルクも優れている.空気消費量は他の空気圧モータに比べて少ないが,構成部品が多く高価である.その出力は0.2~20kW 程度であり,回転速度は200~4500min程度である.ピストンモータにはラジアルピストン形とアキシアルピストン形がある.図V-3・13にラジアルピストンモータ(radial piston motor)の構造図を示す.このモータは3~6本のシリンダを星形に配置し,クランク軸に連結されたロータリバルブによって,圧縮空気をシリンダ室内に順次送り込み,各ピストンを駆動させる.図V-3・14にアキシアルピストンモータ(axial piston motor)の構造を示す.このモータは,多数のピストンの出力を斜板により回転運動に変換する形式であり,ラジアル形よりも小型軽量であ(a)(b)図-3・12 ベーン形モータ図-3・13 ラジアルピストン形モータ2―251第V部フルードパワーシステムる.ベーン形に比べて構造が複雑になることから,高価である.しかし,回転が円滑であり,低速で最大出力を発生する.容量は小さいものが多く,大きいものでも3kW 程度である.c. 歯車モータ(gear motor)図V-3・15に平歯車を使用した歯車モータの構造図を示す.歯車モータは構造が簡単なことから,小型軽量にできる.小型のものでも1万min程度の高速回転が可能であり,大型のものでは約50kW のものも製作されている.d. 空気圧モータの特性図V-3・16に空気圧モータ性能曲線を示す.一定供給圧力の状態において,速度が零のときに出力は零,速度が速くなるにつれて馬力は増大し,無負荷速度の50%近くでその値は最高値をとる.トルクの減少は速度の増加によって平衡を保たれ,無負荷速度(最高速度)においてトルクは零になる.この場合,空気圧モータの発生出力は,モータ内を通過する空気の抵抗によってすべて消費される.空気圧モータは,運転すれば空気量の増大とともに入口と出口間の圧力損失のため,トルクが減少していく.さらにモータの運転のために必要な機器である切換弁,マフラ,絞り弁および配管抵抗がさらにエネルギーを消費し,実際に使用する場合の空気圧モータの特性は,トルク,出力とも図V-3・16より小さくなる.
