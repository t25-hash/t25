# 3・2・4 空気圧揺動形アクチュエータ

> **章**: 第3章 空気圧システム
> **節**: 3・2 空気圧アクチュエータ

出力軸がある角度で回転往復揺動するアクチュエータが空気圧揺動形アクチュエータであり、その構造はおもにベーン形とピストン形がある。a. ベーン形揺動アクチュエータ(vane type rotary actuator)図V-3・17に示すように、円筒形のシリンダ内に回転可能なベーン図-3・14 アキシアルピストン形モータ図-3・15 歯車モータ図-3・16 空気圧モータの性能曲線2―252流体機械図-3・17 ベーン形揺動アクチュエータ図-3・18 スクリュー形揺動アクチュエータ(a) シングルタイプ(b) ダブルタイプ図-3・19ラックピニオン形揺動アクチュエータ体の軸を挿入したもので、シングルベーン形とダブルベーン形がある。シングルベーン形は回転角度が大きく、最大で約270°、ダブルベーン形は回転角度が小さく、最大で約100°の回転角度をもつ。揺動アクチュエータの中では出力に比べて最小型、最軽量となる。しかし、コーナ部からの漏れを止める手段が難しく、かつパッキンの接触長さが他の方式に比べて長いことから、パッキンの抵抗損失が大きい。出力効率は、空気漏れの少ないものでは摩擦損失のために80%を超えることは困難である。b. スクリュー形揺動アクチュエータ(screw type rotaryactuator)図V-3・18に示すように、回転軸に加工されたリードの長いスクリューによって回転が規制されているピストンに沿って、軸が回転運動をする。スクリューおよび回り止めガイドの摩擦損失によって効率は低いが、外力によって逆回転はしにくく、回転角度は360°を超えてもよい。揺動角度の大きい要求に向いている。c. ラックピニオン形揺動アクチュエータ(rack and piniontype rotary actuator)図V-3・19にラックピニオン形の一例を(a) クランクタイプ(b) アームタイプ図-3・20 クランク形揺動アクチュエータ図-3・21 平行開閉形エアチャック図-3・22 支点開閉形エアチャック示す。ピストンに連結したラックによりピニオンを回転させる構造で、ピストンは直線運動のみを行う。歯車の圧力角による反力を受ける構造が十分であれば、摩擦損失が少ないので効率は高い。回転角度はシリンダのストロークによって定まるもので、360°を超えるものも製作は可能であるが、あまり大きいものはラックが長くなり不適合である。d. クランク形揺動アクチュエータ(crank type rotaryactuator)図V-3・20にクランク形揺動アクチュエータの一例を示す。ピストン出力をクランクにより揺動運動に変えるもので、構造が簡単で信頼性が高い。クランクとピストン運動方向との角度により出力に差が出るので、出力の割りにやや大きなピストンが必要である。e. 使用上の注意被駆動物の停止時における揺動アクチュエータの出力軸への衝撃エネルギーはかなり大きく、そのうえ揺動アクチュエータの出力軸はねじれを受け、衝撃耐力が小さい。したがって、被駆動物の停止ストッパを揺動アクチュエータの外に設ける注意が必要である。また、揺動アクチュエータの場合、低摩擦のものが最近市販されてはいるが、一般に低速回転ではスティックスリップ現象に注意が必要である。
