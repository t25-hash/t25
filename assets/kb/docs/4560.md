# 3・2・5 エアチャック

> **章**: 第3章 空気圧システム
> **節**: 3・2 空気圧アクチュエータ

エアチャック(air chuck)は指先でものを挟む役割をするアクチュエータであり,フィンガをシリンダで動作させるものと,ロータリアクチュエータで動作させるものがある.エアチャックにはフィンガの動作形式によって,平行開閉形と支点開閉形がある.平行開閉形はフィンガが平行に動作するもので,クロスローラ平行移動のもの,ロータリ駆動形とスライド方式がある.また,フィンガの数によって2爪タイプ,3爪タイプ,さらには4爪タイプがある.支点開閉形はフィンガの開き角度が30°程度までのトグル形,180°の開閉が可能なカム方式やギヤ方式がある.いずれのエアチャックにおいても,把持力はサイズが大きくなるにつれて当然ながら大きくなる.図V-3・21に平行開閉形エアチャックの一例を示す.また,図V-3・22に支点開閉形エアチャックの一例を示す.エアチャックの選定は,ワークの大きさ,形状や質量,および使用目的が組立,取出しあるいは搬送なのかなどによって行う.高い繰返し精度と長寿命を実現したものが市販されている.
