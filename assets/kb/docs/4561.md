b.インサート成形（insert molding）機 インサート成形で
はインサートするワークが安定して所定の位置にいること，樹脂の
充てん時にはワークが樹脂の挙動に影響されないように固定され，
配置されていること，自動化・無人化しやすいことが要件である．
一般的には，縦型の成形機とすることが多い．それは，インサート
ワークが姿勢を維持しやすく，取出し，インサート，射出充てんの
複合動作が可能なシステムを構成しやすいからである．
c .射出圧縮成形（injection compression molding，図4･15）機
圧縮モードを成立するための構造と制御が必要である．ローリンク
ス法では型締力の制御は当然のこととして，動的に微小距離だけ移
動があるが，この移動時には確実にダイプレートの平行度を確保維
持することが必要条件となる．IC法では金型が微小開き状態で射
出充てんする．やはりダイプレートの平行度が必要となる．成形目
的からは平行度もミクロンオーダが必要となる．また圧縮力は設定
どおりの力でなければならず，基本的には油圧駆動方式とし，その
圧力制御を精度良く行うことが必要である．樹脂の比容積変化から
算出される圧縮量は比容積変化量を厚さの変化で吸収するとすれ
ば，製品厚さの約7%（PMMA：ポリメタクリル酸メチル）を開
き，これを圧縮する．したがって製品の肉厚が大きいほど圧縮量は
大きくなる．圧縮量が大きいことは溶融樹脂の移動量が大きく所定
の成形が困難であることを示している．ディスク（厚さ1.2mm）
などの薄肉製品では圧縮量も84mとなり，より精密な制御が必
要となる．
d.超高速射出充てん成形（ultrahigh- speed injection molding，
図4･16）機 射出速度が極めて高速であり，そのときの立上り
時間が短いものである．射出速度が1000mm/ s以下のときは立上
り時間がほとんどの場合0.02sであればほぼ問題ない状況である．
射出速度がこれ以上となる場合は，立上り時間を短くしなければな
らない．最近の射出速度が2000mm/ sを超える性能仕様をもった
成形機のときは，立上り時間は0.005～0.01s程度としなければな
らない．この応答性を満足させるためには，制御システムはサーボ
弁を用いたクローズドループ制御となる．この制御システムを効果
的にまとめ上げるためにはサーボ弁とアクチュエータの距離を極力
短くする，等距離のレイアウトが実現できることが必要条件とな
る．スクリュー位置スケールはより詳細な速度データを得るために
分解能が極力小さいものが必要となる．あわせて可動部分のすきま
は正しい速度を得ることが困難となるため機構部品もより高精度の
ものを使うことが求められる．
型締機構では，高速充てんによる金型に対しての負荷が大きなも
のとなり，大きな負荷による金型のたわみが大きくなる．この金型
のたわみを抑えるために，剛性の高い，ロケートリング穴部の小さ
なダイプレートが求められる．またダイプレートの平行度に関して
型締力の有無によらずより精密さが求められる．たわみ量は小さい
範囲に抑えなければならない．
