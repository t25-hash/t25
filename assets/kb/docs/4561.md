# 3・3・1 分類

> **章**: 第3章 空気圧システム
> **節**: 3・3 空気圧制御弁

空気圧制御弁(pneumatic control valve)は、空気圧機器へ供給する空気の圧力を調整したり、機器の動きを制御するために使われる。機能の面から分類すると、空気流れの通過・閉鎖および方向制御、流量の制御、圧力の制御の3種になる。流量や圧力の制御は一定の量に保つものと、操作入力により連続的に制御する比例制御式があり、後者ではさらに流れの方向制御をする機能をもつものがある。流れの通過・閉鎖および方向制御は、可動弁によりオンオフ式に行われる。一方、流量や圧力の制御は、可変絞りによって空気流路の開口面積を変化させ、絞りの上流側あるいは下流側圧力や流量を変えることにより行われる。代表的な弁構造を図V-3・23に示す。ノズルフラッパ弁(nozzle flapper valve)は上流側(背圧側)の圧力変化を利用するもので、上流側固定絞り、およびノズルフラッパの間隔により操作される下流側可変絞りをもつ。これらの絞り間の圧力(背圧)はノズルフラッパ間隔が狭いと高く、広いと低くなるので、間隔を連続的に調整すれば圧力が操作できる。直接ノズルフラッパ弁を用いて、圧力の調整や機器の制御をすることは少ないが、空気圧制御弁の内部で、主弁を操作するパイロット圧を発生する用途に広く使われる。ニードル弁(needle valve)、ポペット弁(poppet valve)およびスプール弁(spool valve)、スライド弁(slide valve)は、回路を閉鎖したり、下流側圧力や流量を操作するために使われる。空気圧制御弁の構造自体は油圧式と変わらないが、空気は油に比べて圧縮性が高いため、多くの場合絞り部で、流れはチョーク状態となる。弁に働く流体力は油圧式と同様に求められるが、摩擦力の影響が大きい。弁の流量特性は、3・1・1項に記述されている基礎式で与えられる絞りの特性と類似するが、流路は単純な絞りよりも複雑なので、チョーク流れが発生する臨界圧力比は、絞りの場合の0.528より一般図-3・23 弁の構造2―253第V部フルードパワーシステムに小さくなる。弁の流量特性の計測、表示方法はISO 6358:1989に準じたJIS B 8390:2000で規定されている。ここでは図V-たときの質量流量をLあるいはL′で計測し、次式により音速コンダクタンスを求めるようになっている。図下側は下流を大気として利用する場合の回路である。JIS B 8390にはまた、従来行われていた放出法により弁の有効断面を測定する方法も、付属書として残されている。一般に大きな流路をもつ弁の流量測定は、放出法でも困難であるため、容器へ空気を充てんしたときの圧力変化から、音速コンダクタンスを求める方式も準用される。操作方式で分類すると、人や機械が力を加えて直接操作するもの、電気入力操作によるもの、および空気圧パイロット操作によるものがある。内部に可動部をもたず、制御流体流れにより主流れを切換えたり偏向させるフルイディクスも制御弁の一種である。
