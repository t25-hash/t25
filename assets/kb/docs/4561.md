# 3・3・1 分類

> **章**: 第3章 空気圧システム
> **節**: 3・3 空気圧制御弁

空気圧制御弁（pneumatic control valve）は，空気圧機器へ供
給する空気の圧力を調整したり，機器の動きを制御するために使わ
れる．機能の面から分類すると，空気流れの通過・閉鎖および方向
制御，流量の制御，圧力の制御の3種になる．流量や圧力の制御は
一定の量に保つものと，操作入力により連続的に制御する比例制御
式があり，後者ではさらに流れの方向制御をする機能をもつものが
ある．
流れの通過・閉鎖および方向制御は，可動弁によりオンオフ式に
行われる．一方，流量や圧力の制御は，可変絞りによって空気流路
の開口面積を変化させ，絞りの上流側あるいは下流側圧力や流量を
変えることにより行われる．代表的な弁構造を図Ⅴ-3･23に示
す．
ノズルフラッパ弁（nozzle flapper valve）は上流側（背圧側）
の圧力変化を利用するもので，上流側固定絞り，およびノズル
フラッパの間隔により操作される下流側可変絞りをもつ．これらの
絞り間の圧力（背圧）はノズルフラッパ間隔が狭いと高く，広いと
低くなるので，間隔を連続的に調整すれば圧力が操作できる．直接
ノズルフラッパ弁を用いて，圧力の調整や機器の制御をすることは
少ないが，空気圧制御弁の内部で，主弁を操作するパイロット圧を
発生する用途に広く使われる．
ニードル弁（needle valve），ポペット弁（poppet valve）およ
びスプール弁（spool valve），スライド弁（slide valve）は，回路
を閉鎖したり，下流側圧力や流量を操作するために使われる．空気
圧制御弁の構造自体は油圧式と変わらないが，空気は油に比べて圧
縮性が高いため，多くの場合絞り部で，流れはチョーク状態とな
る．弁に働く流体力は油圧式と同様に求められるが，摩擦力の影響
が大きい．
弁の流量特性は，3･1･1項に記述されている基礎式で与えられる
絞りの特性と類似するが，流路は単純な絞りよりも複雑なので，チ
ョーク流れが発生する臨界圧力比は，絞りの場合の0.528より一般
る．
操作方式で分類すると，人や機械が力を加えて直接操作するも
の，電気入力操作によるもの，および空気圧パイロット操作による
ものがある．内部に可動部をもたず，制御流体流れにより主流れを
切換えたり偏向させるフルイディクス も制御弁の一種である．
