# 3・3・2 方向制御弁

> **章**: 第3章 空気圧システム
> **節**: 3・24 の試験回路を用い、G の試験弁を通過する流れがチョークし

a. 切換弁切換弁(directional control valve)は流れの通過、閉鎖あるいは流れる方向の切換えを行う。用途に応じて2ポート弁、3ポート弁、4ポート弁および5ポート弁がある。2ポート弁は流れの通過、閉鎖のみ可能である。他は流れの方向の切換えができるので、空気圧シリンダなどの動作方向制御に使われる。単動形シリンダは3ポート弁、複動形では4ポート弁と5ポート弁が使われる。図V-3・25は3ポート電磁弁の例である。入力が加わらないとき、主弁はスプリングの作用により図の上側に押され、A とR のポートがつながる。入力電流が与えられたとき、ソレノイドで発生する電磁力がプランジャを下側に引き寄せ主弁を押すので、A とPをつなぐポペット弁が開き、A とR は閉じる。この主弁は手動ボタンを人が押して動かすこともできる。この弁を図記号で表すと図V-3・26になる。切換弁の図記号は操作方式、ポート数、弁位置、復帰方式を図示したもので、整理参照文字要素の名称参照文字要素の名称A空気(圧縮ガス)源およびフィルタG供試機器H下流圧力測定管B圧力制御機器I差圧計または差圧変換器C遮断弁J上流の圧力計または圧力変換器D温度測定管E温度計K流量制御弁F上流圧力測定管L流量計図-3・24 流量特性の試験装置2―254流体機械すると図V-3・27になる。これらは代表的な表現で、操作方式や機能をより具体的に表現するため細部を変えることもある。なお、ポート記号は従来P,A,B,R の文字記号が使われていたが、ISO5599:2001に整合させるため1,2,3,4,5の番号が使われるようになった。弁位置は弁が止まる位置を示し、4ポート弁、5ポート弁では二つの切換位置のほか、その中間に中立位置をもつ3位置のものも存在する。図V-3・26の例は電磁入力操作方式、すなわち電磁弁であるが、人力による操作も可能であり、3ポート、2位置、ばね復帰であることがわかる。b. 逆止め弁、シャトル弁逆止め弁(check valve、チェック弁)は一方向の流れを許し、逆方向を阻止してちょうど半導体素子のダイオードと同様の働きをする。逆止め弁を組合わせた構造をもつシャトル弁(shuttle valve)は、出力ポートを一つだけもつ3図-3・25 3ポート電磁弁の構造例図-3・26 3ポート電磁弁の図記号入力操作方式ポートの数弁の位置復帰方式人力2ポート2位置機械3ポート電磁4ポートパイロット圧5ポート図-3・27 図記号による切換弁の分類例ポート弁で、出力ポートは他2ポートのうち圧力が高い側に接続される。逆止め弁の図記号を図V-3・28に、シャトル弁を図V-3・29に示す。c. 急速排気弁空気圧アクチュエータに往復動作をさせるとき、排気側に残圧があると動作速度が遅くなる。急速排気弁(quick exhaust valve)は排気側を大気に十分な開口面積で開放し、残圧を急速に下げるために使われる。図V-3・30にその図記号を示す。
