# 3・3・3 流量制御弁

> **章**: 第3章 空気圧システム
> **節**: 3・24 の試験回路を用い、G の試験弁を通過する流れがチョークし

a. 絞り弁、速度制御弁絞り弁(restrictor,meteringvalve)は、固定絞りと手動により弁の開度が変わる可変絞りがある。可変絞りは通常ニードル弁が使われ、ねじによりニードル位置を前後に動かし、弁座とのすきまに生じる空気流路の断面積を変えて流量を調整する。空気圧回路の項(3・5・3項)で示されるように、空気圧アクチュエータでは絞りにより流量を制限すれば容易に等速度制御が可能となる。このような目的に絞り弁が用いられるとき、流量を制限する流れ方向と制限しない方向が必要であるため、逆止め弁と並列に組合わされる。これを一体化した弁は速度制御弁(speed controller)と呼ばれる。絞り弁(左側)と速度制御弁(右側)の図記号を図V-3・31に示す。図-3・28 逆止め弁の図記号図-3・29シャトル弁の図記号図-3・30 急速排気弁の図記号ばね3位置(4ポート)ダブル電磁弁ダブルパイロット圧図-3・31 絞り弁と速度制御弁の図記号図-3・32 減圧弁の構造例b. 比例流量制御弁比例流量制御弁(proportional flow con-trol valve)は、操作入力の大きさにより連続的に流量を操作する制御弁である。操作入力は通常電圧(電流)であり、コイルで発生する電磁力を用いてスプール弁の開度が操作されることが多い。内部にスプールのリターンばねを設け、ばね力と電磁力とが釣合う位置までスプール弁が移動し、弁の開度が変わる。入力により操作されるのは弁の開度であり、流量が直接操作されるのではない。なお、内部に位置センサを設置し、スプール位置のフィードバック操作を行い、応答性や精度を向上させたサーボ弁があり、3・6・1項で述べられる。比例流量制御弁には切換弁としての機能をもつものがあり、これらには3ポート弁、4ポート弁、5ポート弁がある。
