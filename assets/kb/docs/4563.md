# 4・4・1 押出機（extruder ）

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

押出機とは，プラスチック材料を溶融させながら，混練していく
とともに目的とする形状に連続的に賦形するために使用する機械で
ある．対象とするプラスチック材料は，ポリプロピレン（PP），ポ
リエチレン（PE）などの熱可塑性樹脂が大部分であるが，一部熱
硬化性樹脂も含まれる．
押出機において，連続的に溶融，混練するために，スクリューが
使用されており，このスクリューの本数と構造によって押出機が分
類されている．

ゆるフルフライト型であるが，近年押出量の増大，要求品質の高度
化により混練性を向上させる必要性が増大している．このため，溶
融領域において，固相と溶融物プールを機械的に分離させ溶融を促
進させる，バリアフライト型やマドック型スクリューの採用が，ま
た融体輸送領域においてダルメージ型，クロスソー型などの採用が
増えている．図4･21にバリアフライト型やマドック型，ダルメー
ジ型など各種スクリュー形状を示す．
押出機の固体輸送能力を上げたり，高分子量のポリエチレンなど
の安定押出しを行うため，スクリューの供給領域に溝付シリンダを
用いることがある．図4･22に溝付シリンダを示す．これは，シリ
ンダと供給材料との摩擦抵抗の増大を利用して固体輸送能力を向上
させるものである．
一方，材料に含まれるモノマや巻込み空気などを押出機の途中か
ら除去しながら押出しを行う目的には，ベント式押出機を使用す
る．ベント式押出機では，溶融した材料を一時的に圧力低下状態に
して，ガス状揮発分を真空ポンプにより強制排出させる構造となっ
ている．図4･23にベント式押出機におけるシリンダ内圧分布を示
す．
b.二軸スクリュー押出機（twin screw extruder ) 二軸押出
機は，かみあい形成，回転方向形式により数多くの種類がある．b
項では，フィルム，シート，異型成形などで使用される，低速かみ
あい型異方向回転タイプと高速かみあい型二軸押出機について記述
する．
低速かみあい型異方向回転タイプ二軸押出機は，完全かみあい
で， が20前後，スクリュー回転数も40rpm以下である．こ
の押出機は，熱安定性の悪いポリ塩化ビニルの押出しに用いられ
る．このタイプには，スクリュー軸が平行のものと斜軸（コニカル
型押出機と呼ばれている）がある．
高速かみあい型二軸押出機は， が30～50で，スクリュー回
転数も300rpm程度である．複合剤を含むプラスチック材料の混
練，脱気などに適しており，コンパウンド用途として多くの実績が
ある．近年，コンパウンドでペレットを作りそれを単軸押出機にか
ける従来型から，コンパウンドの工程を省略し，高速かみあい型二
軸押出機の先端にダイを取付け，直接シート等の成形をするものが
増えている．図4･24に二軸押出機の形式を示す．
