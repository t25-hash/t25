# 3・3・4 圧力制御弁

> **章**: 第3章 空気圧システム
> **節**: 3・24 の試験回路を用い、G の試験弁を通過する流れがチョークし

a. 減圧弁減圧弁(pressure reducing valve)は高圧(一次側)圧力を低圧(二次側)に変換する。このとき一次側に圧力変動があっても二次側では除去され、ほぼ一定の圧力が保持される。一般にこのような変換はエネルギー損失を生じるが、空気圧回路では低圧になるにつれて、空気は状態変化により体積が増すため、弁でのエネルギー損失は油圧回路に比べ少ない。減圧弁の構造例を図V-3・32に示す。二次側圧力の設定は、図下側の軸まわりに回転する部分を手で操作して行う。本体内部のばね支持部との間には、ねじが設けられており、回転によりねじに押されたばねが、さらに主弁を押す。主弁が開くと一次側から高圧空気が二次側に流れこみ、圧力が上昇する。二次側圧力による力がダイアフラムを押し、主弁を閉じるよう作用するフィードバック機構が働くため、ねじ位置により設定されたばね力に釣合って、二次側圧力が整定する。何らかの原因で二次側圧力が設定圧力を超えたとき、設定圧が保たれるようリリーフ弁(b項)の機構を内蔵したものが広く使われる。図記号を図V-3・33に示す。b. リリーフ弁リリーフ弁(relief valve)は回路圧力が設定圧力を超えたとき、空気を排気して設定圧力が保たれるよう動作する弁で、設定圧力を境に急峻に開放・閉鎖動作をする。図記号を図V-3・34に示す。c. 比例圧力制御弁比例圧力制御弁(proportional pressurecontrol valve)は、入力に比例する圧力を発生する制御弁である。減圧弁を電気入力により操作できるようにした電空比例制御弁(電2―255第V部フルードパワーシステム図-3・33 減圧弁の図記号図-3・34 リリーフ弁の図記号図-3・35 電空比例制御弁の図記号空レギュレータ)も、この一種である。図記号を図V-3・35に示す。電磁入力操作を表す記号の部分に矢印があるが、操作入力の大きさが連続的に変わることを意味する。スプール弁を用いて、リターンばねの代わりに二次側圧力をリターン側操作圧とすると、スプールは電磁力と二次側圧力が釣合った開度位置で整定し、二次側圧力は入力電流に比例する。このような構造をもつ比例圧力制御弁があり、多ポート弁も存在する。また、サーボ弁として設計された比例圧力制御弁もある。
