# 4・4・3 シート成形（sheet forming）

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

厚み0.15mm以上のものをシートと呼んでいる．図4･30にシ
ート製造装置を示す．装置は，押出機，Tダイ，ポリシングロー
ル装置，引取機，定尺切断装置，または巻取機から構成される．ポ
リシングロール装置は3本構成で，各ロールとも高精度に加工され
ている．中央ロールは固定され，上下のロールが上下に移動できる
ようになっており，それぞれ，中央ロールとのロールすきまが調節
でき，中央ロールに対して加圧できる．Tダイから吐出された溶
融樹脂をこれらのロールに挟み込み冷却しながら，一定速度で引取
り，一定寸法に切断する（巻取ることもある）．近年，シートの外
観品質要求が高度化され，ポリシングロール装置の駆動は，各ロー
ルとも高精度ウォーム減速機とサーボモータの組合せとなってい
る．図4･31にシート用Tダイの断面を示す．シートダイでは，リ
ップ各部での幅方向の流量を調整するためのチョークバーが設けら
れている．これでリップ部での流量と流速を均一に保つ必要があ
る．
