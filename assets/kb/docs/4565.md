# 3・4・1 フィルタ

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

空気圧用フィルタには回路中の微細な固形異物をろ過する以外に、圧縮空気中の液化水分(ドレン)を除去するという二つの機能が要求される。図V-3・36に代表的な空気圧フィルタの構造例を示す。入口から入った圧縮空気はデフレクタにより旋回運動を与えられ、そのサイクロン効果により、比較的大きな遊離水滴や異物をケースの内壁に押しやり、壁面に伝わって落下させ底部にためる。このようにして大部分の異物が除かれた圧縮空気は、中央の合成樹脂や焼結金属で作られた微細な穴が無数にあるフィルタエレメントを通ることで、さらに小さな微粒子を除去し、出口側から流出する。分離された水分やごみなどは、ボウルの下部に付いているコック式、プッシュ式などの手動バルブ、あるいは自動ドレンバルブにより外部へ排出される。ろ材の目の細かさは粒度(m)表示するが、用途に応じ概略次のように使い分ける。(1) Oリング、各種成形パッキンを使用した機器および一般空気2―256流体機械図-3・36 空気圧フィルタ図-3・37 冷凍式エアドライヤ圧回路要素→50~70 m(2) 空気圧モータ、空気タービンなどの高速回転部をもつ機器、メタルシールの切換弁および機械要素潤滑用ルブリケータなど→25 m以下(3) 空気圧マイクロメータのような精密計測用→5~10 m(4) 純流体素子回路→5 m以下
