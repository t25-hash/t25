# 3・4・4 消音器(muffler,silencer)

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

空気圧システムにおいて作動流体に関連する騒音発生源として、次のようなものが挙げられる。(1) 圧縮機の吸込口から大気を吸い込む際に生じる気流音や、吐出し時に生じる空気圧力の脈動によって発生する騒音。(2) 切換弁や空気圧モータ駆動の工具などから、空気が排気されるときに生じる気流音・衝撃音。これらの騒音を減衰し、より良い作業環境を得るために消音器(サイレンサ)が用いられる。消音器に求められる機能条件としては、(1) 流量抵抗が小さく、切換弁などの切換性能やアクチュエータの性能に影響を与えない。(2) 消音効果が大きい。(3) 長時間の使用に対して目詰まりなどの抵抗変化が少ない。(4) 電磁弁などに取付けやすいコンパクトな形状である。(5) 排気の衝撃や振動などで変化を起こさない。などを考慮しておくことが必要である。構造の違いにより消音器は、リアクタンス型(膨張型、共鳴型、挿入管型の総称)と吸音型との2種類に大きく分類される。リアクタンス型消音器(reactive muffler)は圧縮機の吸・排気騒音防止に多く使用され、通気抵抗が小さいことが特徴であるが、その形状、寸法により、特定の周波数の騒音に対して選択的な減衰効果をもつ特質がある。一方、吸音型消音器は、焼結金属などの通気性多孔物質およびグラスウール、不織布、メタルウールなどの吸音材を空気通路に充てんして振動エネルギーを吸収するもので、高周波に対する吸音率が高く小型化できるが、通気抵抗が比較的大きくなってしまう。一般に空気圧機器用消音器として市販されているものは、排気管の騒音減衰効果を得るために、リアクタンス型と吸音型の構造を併用したものが多い(図V-3・40)。
