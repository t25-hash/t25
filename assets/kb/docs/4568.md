# 4・4・6 モノフィラメント成形（monofilament extrusion）

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

モノフィラメントの成形は，基本的には，押出機を用いた溶融紡
糸法であり，押出機，冷却槽，延伸装置，熱処理装置，巻取装置か
ら構成される．モノフィラメントとは，通常50～10000d（デニー
ル）程度の繊維を成形する（デニールとは，繊維の太さを表現する
もので，9000m当たりの重量が1gのとき，この繊維の太さを
1dという）．適用樹脂としては，高密度ポリエチレン（HDPE），
PP，PA，PETなどがある．押出機から吐出された溶融樹脂は，
多数の細孔のあるダイから吐出され，冷却槽で冷却後，再加熱し延
伸・熱処理を行う．逐次二軸延伸フィルムと同様，延伸温度は材料
の二次転移点以上で行われ，強度，伸度，剛性を向上させる．
