# 3・4・5 配管および継手

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

システムを構成する各種空気圧機器を物理的に接続し，有効かつ
効率的に空気の供給・排気を行うために必要となる機械要素が，配
管および継手である．
おもな配管材料は表Ⅴ-3･2 に示すように分類される．一般に
空気圧供給源からの主配管部などには金属管が多用されるのに対
し，配管末端部に近づくほど樹脂チューブ（ナイロン，ポリウレタ
ンなど）が多用されている．樹脂チューブについては材料等の進歩
により経年変化の影響が少なく耐腐食性に強いものに年々改良され
ており，可撓性が良くシステムを小型・コンパクトに設計できると
いう点を含めて考えると，将来的には従来金属配管で行われていた
部分についても樹脂チューブに置換わっていく傾向にあるものと推
測される．
継手は各種配管材料に対応して選定される．金属管に対しては，
ねじ込み式継手，フレア継手などが用いられる．ねじ込み式継手は
鋼管の端部に管用テーパねじを切って直接ねじ込む継手であり，フ
レア継手は薄肉のステンレス管，銅管などの端部を円すい状（フレ
ア）に広げて，この部分をナットで締め込み固定・シールするもの
である（図Ⅴ-3･41)．一方，樹脂チューブに対しては，締め込
み式継手，インスタント（ワンタッチ）継手などが用いられる．締
め込み式継手は，樹脂チューブをスリーブを介してナットを締付け
