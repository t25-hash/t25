# 4・4・7 多層押出成形（multilayer extrusion）

> **章**: 第4章 プラスチック加工機械
> **節**: 4・4 押出成形機

押出成形における高機能化の一つとして多層化（co-extrusion）
がある．これは，2台以上の押出機を用いて一つの共通ダイから単
一製品として成形するが，その断面は多層構造をなしている．多層
化することで，単層では得られない種々の物性を得ようというもの
である．フィルムシートの場合，多層化する方法には，フィードブ
ロック法とマルチマニホルド法の2種類がある．フィードブロック
法は，Tダイの前でフィードブロックにより多層化した後，Tダ
イで幅方向に広げる．マルチマニホルド法は，Tダイにおいて幅
方向に広がった各層が，Tダイ内部で合流，多層化する．図4･34
にフィルムシートの多層化法を示す．
