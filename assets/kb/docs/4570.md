# 3・4・6 シールパッキン

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

空気圧システムの機器、配管には、圧縮空気の漏れや外部からの異物の侵入を防止するための密封装置として、シール(seal)が用いられる。これら空気圧用シールは、機能別に概略図V-3・45のように分類することができる。一般に固定用シールはガスケット(gasket)と呼ばれ、部材接合部、配管ねじ部、フランジ固定部などに使用されるのに対し、運動用シールはパッキン(packing)と呼ばれ、往復・回転・揺動・らせん運動などを伴う機器のしゅう動部分に使用される。特にパッキンについては、運動する二つの部材間に封入されて用いられるため、摩擦力が増大しないようにルブリ図-3・43 インスタント継手図-3・44 急速継手ガスケット(固定用シール)シートガスケットゴムシート、石綿ジョイントシートなどゴム成形ガスケットOリング、異形Oリングなどその他シールテープ、液状ガスケットなどパッキン(運動用シール)セルフシーリングパッキンスクイーズタイプ。..O,X、角リング、スリッパシールなどリップタイプ。..U,Y,L,Jパッキンなどダスト侵入防止シールワイパ、スクレーパ、ブーツなどその他ダイアフラム、ベローズ、弁座シール、スプールシール、クッションパッキンなどシール図-3・45 空気圧用シールの分類図-3・46 一段エジェクタの概念図図-3・47 圧力源回路ケータあるいはグリースにより、適当な潤滑を行うことが重要となる。図V-3・45でパッキンに分類されたシールについて、簡単に解説を加えておく。セルフシーリングパッキンは、流体圧力に応じてシールの接面圧力が増す自己密封作用をもつもので、Oリングで代表されるスクイーズタイプと、シールリップの張力を利用して密封効果を得るリップタイプに分けられる。ダスト侵入防止シールは、シリンダロッドのように往復動する軸のすべり面をダストなどのかみ込みから守る目的で用いられるワイパ・スクレーパ・ブーツなどの総称である。また、これらパッキン材料としては、合成ゴムの一つであるNBR(ニトリルゴム)が多用されている。
