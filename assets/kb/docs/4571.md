# 4・5・2 射出ブロー成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・5 ブロー成形機

押出ブロー成形では，管状のパリソンから中空体を成形するのに
対し，射出ブロー成形（injection blow molding）は，まずプリフ
ォーム（preform）と呼ばれる試験管状の有底パリソンを射出成形
した後，成形されたプリフォームを射出金型の雄型につけたまま雌
型より取出して，ブロー型に移行させ，雄型から圧縮空気を吹込ん
でブロー成形するもので，ホットパリソン法の一つである．その工
程を図4･41に示す．
本成形法では，口部は最初に射出成形により成形されるため，形
状，寸法の精度が良い．また，ピンチオフなどの融着部がないこと
などが特徴である．ただし射出金型とブロー金型を合わせ持ち，最
適にプリフォームを温調する装置等が付帯されるため，設備費は押
出ブロー成形機に比べて高く，成形品形状や容量もある程度制約を
受ける．
