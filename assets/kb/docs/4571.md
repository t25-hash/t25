# 4・5・2 射出ブロー成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・5 ブロー成形機

押出ブロー成形では，管状のパリソンから中空体を成形するのにォーム（preform）と呼ばれる試験管状の有底パリソンを射出成形型より取出して，ブロー型に移行させ，雄型から圧縮空気を吹込ん程を図4･41に示す．
状，寸法の精度が良い．
また，ピンチオフなどの融着部がないこと適にプリフォームを温調する装置等が付帯されるため，設備費は押受ける．
対し，射出ブロー成形（injection blow molding）は，まずプリフした後，成形されたプリフォームを射出金型の雄型につけたまま雌でブロー成形するもので，ホットパリソン法の一つである．
その工本成形法では，口部は最初に射出成形により成形されるため，形などが特徴である．
