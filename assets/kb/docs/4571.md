# 3・4・7 エジェクタ

> **章**: 第3章 空気圧システム
> **節**: 3・4 付属機器・その他

空気圧システムの構成要素の中には、吸着パットや吸着板など真空圧力(負圧)を利用するような機器が存在する。これら機器を含んだシステムを構成する場合、真空発生装置が必要となる。空気圧機器の中で代表的な真空発生装置として、エジェクタを挙げることができる。図V-3・46に示したように、エジェクタは空気圧を動力源として利用した一種の流体圧ポンプである。ノズルとディフューザとで対に構成されており、ノズルからディフューザに高速流れを発生させるとノズル-ディフューザ間の流体を真空口から吸引し、真空を発生させている。作動空気はディフューザ、消音器を通って大気あるいは排気ダクトに排出される。多段のエジェクタを用いればより高い真空圧力を得ることが実現でき、また複数のエジェクタを並列に並べて用いれば、大容量化を図ることができる。システムに真空ポンプを用いる場合と比較して、エジェクタを採用することで装置を安価でコンパクトに制作することができるという利点があるが、駆動に十分な空気消費量を確保する必要があることや、ランニングコスト面で不利になる場合があることなどを、実用面に照らして注意しなければならない。
