# 4・5・3 二軸延伸ブロー成形機

> **章**: 第4章 プラスチック加工機械
> **節**: 4・5 ブロー成形機

熱可塑性樹脂の中でも特に結晶性の樹脂は，その融点以下の温度
で延伸することにより引張強度，落下衝撃力，ガスバリア性，透明
度などが向上する性質をもつ．この性質をブロー成形に適用したも
のが二軸延伸ブロー成形（bi -axially stretch blow molding）であ
る．対象樹脂はおもにPET，PPなどである．成形機も押出し法
と射出法，さらにシングルステージタイプとツーステージタイプと
の組合せにより多くの種類がある．射出ブロー成形と異なる点は，
延伸棒（stretch rod）により縦延伸を行うことである．延伸棒の
利用により，成形時の延伸倍率を高く設定でき，より成形品の性能
を向上できる．
成形工程は射出法によるシングルステージタイプ（ホットパリソ
ン法）の場合，まず射出成形によりプリフォームを成形し，プリフ
ォームの内外面をヒータにより延伸温度まで均一に加熱する．次い
でブロー金型に移送し，延伸棒にて軸方向に延伸するとともにブロ
ー成形し二軸延伸を行った後，成形品を取出す．この成形法は，生
産速度は遅いが，型数が少なく，型替え性に優れているので多品種
小ロットの生産に適する．
また，射出法によるツーステージタイプ（コールドパリソン法）
の場合，これはPETボトルの成形によく用いられる方法である
が，成形工程を図4･42に示す．まず，射出成形でプリフォームを
成形した後，いったん室温まで冷却させる．そして，プリフォーム
を近赤外線ヒータで再度最適成形条件まで加熱し，その後ブロー金
型内部に移動させ，高圧空気を金型内部に吹込みながら，延伸ロッ
ドによりプリフォームを延伸し，ブロー成形を行う．この方法はブ
ロー成形のプリフォーム加熱条件を均一にでき，また，射出成形の
成形間隔に依存しないため，生産速度を速くでき，少品種大ロット
の生産に適する．しかし，型数が多くなるため，設備費は高くな
る．
最近では，共射出成形（co- injection molding）による多層プリ
フォーム（multilayer perform）のブロー成形が行われている．ブ
ロー成形の段階では，多層プリフォームを単層プリフォームと同じ
工程で二軸延伸ブロー成形している．共射出成形法は，射出金型の
中に異なる樹脂を交互に射出して，中間層としてガスバリア性樹脂
などを封入した形態の多層プリフォームを成形する方法である．
