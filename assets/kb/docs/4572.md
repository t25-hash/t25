# 3・5・1 圧力源回路

> **章**: 第3章 空気圧システム
> **節**: 3・5 空気圧回路

油圧源が個々の機械装置ごとに独立して配置されることが多いのに対して,空気圧の圧力源は各工場の棟ごと,あるいは事業所単位ごとに大容量の圧縮機群が配置され,空気圧アクチュエータを駆動する以外の用途の供給源も兼ねることが多い.典型的な空気圧源の回路構成を図V-3・47に示す.空気圧縮機からの吐出し空気は,アフタクーラによって温度を下げられた後,空気タンクへ入る.圧縮機の吐出し圧力は,圧力スイッチによる制御やアンロード弁の使用などによって,一定範囲内に保たれている.また省エネルギーを目的として,圧縮機の回転数を圧力センサとインバータにより緻密に制御したものもある.圧縮機を出た空気は,メインラインフィルタにより空気中の固形異物や圧縮機潤滑油の一部が除去され,さらにドライヤによって除湿した後,使用先に送られる.なおアフタクーラ,タンクは必要に応じて設置し,タンク,ドライヤ,フィルタの設置箇所は,状況に合わせて適宜決めてよい.図V-3・47の回路では,圧縮機の脈動,オンオフ等で圧力は一定とならないので,空気圧ではこれを減圧して一定の圧力に保つ.このとき用いられるのが減圧弁で,減圧は使用目的に応じた圧力まで,使用箇所付近で行われる場合が多い.減圧弁は図V-3・48のように,フィルタ,ルブリケータのように組合わせた減圧回路(pressure reducing circuit)として使用されることが多く,これらの回路を一体化したFRL(filter, regulator, lubricator)調質ユニットもある.また食品機械や静圧軸受など細かい異物を嫌う装置には,ミストセパレータなどさらに細かいフィルタを追加し,シリンダが無給油タイプであるなど使用機器が潤滑を必要としない場合には,ルブリケータを省略する.
