# 3・5・2 方向制御回路

> **章**: 第3章 空気圧システム
> **節**: 3・5 空気圧回路

a. 方向制御回路(direction control circuit)シリンダの移動方向を制御する回路であり、単動シリンダでは3ポート切換弁が、複動シリンダでは4あるいは5ポートの切換弁が、図V-3・49のように取付けられる。切換弁が図の位置にあるとき、シリンダはともに引き戻り方向(左方向)に移動する。b. 中間停止回路(intermediate stop circuit)シリンダの中間停止を行うときは、3位置切換弁を図V-3・50のように接続すればよく、切換弁がオフの状態となればシリンダは停止する。図V-切換弁の場合もある。切換弁の種類によって中間停止時の機能が異なり、用途によって使い分けられる。また、空気の圧縮性により停止精度、位置剛性が問題となる場合は、ブレーキ付きシリンダやハイドロチェッカ、空油変換回路が利用される。
