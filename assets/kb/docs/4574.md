# 3・5・3 速度制御回路(speed control circuit)

a. メータアウト回路とメータイン回路速度制御回路は速度制御弁を用いて行われ、排気する空気量を加減してシリンダの速度を調節するメータアウト回路[meter-out circuit、図V-3・図-3・48 減圧回路(a) 単動シリンダの場合(b) 複動シリンダの場合図-3・49方向制御回路2―259第V部フルードパワーシステム51(a)]と、給気側のそれを調整するメータイン回路[meter-incircuit、図V-3・51(b)]とに分類される。空気圧の場合、取付負荷、摩擦等に依存せず、速度制御弁の開度のみでシリンダ速度が決まる性質がメータアウト回路にあり、安定な動作を得られるメータアウト回路が特別な場合を除いて用いられる。速度制御弁の位置は、シリンダ近くに設置されるもののほか、速度制御弁のチェック弁機能がないメタリングバルブ(metering valve)を図V-3・50のように、切換弁のポートに直接取付けたメータアウト回路もある。また、片ロッドシリンダで低速度制御を行う場合や、シリンダを垂直に取付け高負荷を駆動する場合において、メータアウト回路では、シリンダの初期速度が一時的に大きくなる飛び出し現象が問題となることがある。これを防止するときには、メータアウト回路にメータイン回路を組合わせた回路が用いられる。なお、単動シリンダを往復動で速度調整するときには、図V-3・52のメータイン・アウト回路で行われる。b. 急速排気弁を用いた急速後退回路(quick retraction speedcontrol circuit)一方向のみ速度制御が必要なときには、シリンダの戻りはできるだけ高速であることが望ましい。通常の回路では、管路や電磁弁が抵抗となり、どうしても速度が遅くなりがちである。このようなときに使用される回路が、急速排気弁を用いた急速後退回路である。図V-3・53(a)のように一般的な急速排気弁にはダイアフラムが利用され、給気時にはダイアフラムが下に押付けられるので、空気は給気ポート(P側)からシリンダ(A 側)に供給される。弁の操作により給気ポート側の圧力が低下すると、今図-3・50 センタクローズ形切換弁による中間停止回路(a) メータアウト回路(b) メータイン回路図-3・51 複動シリンダの速度制御回路図-3・52 単動シリンダの速度制御回路2―260流体機械度はダイアフラムが下側に押付けられ、排気ポート(R 側)から一気に排気されることになる。したがって、この弁を図V-3・53(b)のように取付けることにより、シリンダ速度を最大限とすることができる。急速排気弁を用いない場合に比べて数倍の速度が得られ、シリンダのサイクルタイムを著しく縮めることができる。c. 二段速度制御回路(2-step speed control circuit)シリンダの起動時に高速始動を要する場合や、シリンダ端でクッションの容量不足、停止時の衝撃が問題となる場合に、図V-3・54の二段速度制御回路が用いられる。その回路は2ポート常時開形の切換弁を並列に接続し、高速移動させるときにリミットスイッチなどの信号によってその弁を作動させ、排気をバイパスして高速側に切換えるものである。
