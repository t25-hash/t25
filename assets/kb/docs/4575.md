# 3・5・4 同期制御回路(synchronizing control circuit)

> **章**: 第3章 空気圧システム
> **節**: 3・50 にあるセンタクローズ形切換弁は、ABR 接続、ABP接続の

複数のシリンダを同期させるには、シリンダが同期するように個々の速度制御弁を調節する必要がある。これを回避する最も簡単な方法は、一つの切換弁でまとめて駆動する回路である。しかしながら、配管抵抗の差異や空気の圧縮性などにより、厳密な同期制御を行うことは極めて困難である。したがって、シリンダを同期させるには、ラックアンドピニオンやリンクなど機械的な機構を組合わ(a) 急速排気弁の原理(b) 急速後退回路図-3・53 急速排気弁を用いた急速後退回路図-3・54 二段速度制御回路せて同期させる方法が最も確実である。このほかでは、ハイドロチェッカを利用した回路や、図V-3・55のように同期させるシリンダの片側圧力室に油を封入し、それらを結合する回路がある。その回路では空気に比べて油の圧縮性が無視できるので、油漏れがなければシリンダ1で押出された油はシリンダ2に流入し、シリンダ間に掛かる負荷に違いがあっても両シリンダ速度は同一となる。
