# 3・5・7 油圧を併用した回路

> **章**: 第3章 空気圧システム
> **節**: 3・50 にあるセンタクローズ形切換弁は,ABR 接続,ABP接続の

a. 空油変換器を利用した回路図V-3・61(a)のような空油変換器(air-hydro converter)を使用して空気の圧力を油圧に変換し,図V-3・61(b)のような回路でシリンダを駆動する回路である.速度を調整するスロットル弁はメータインで接続する.空油変換器にはほかにも,空気と油の間をピストンで仕切った形もあり,ピストンの断面積を変えて増圧するタイプのものもある.油圧に変換する目的は,負荷変動による速度のばらつきや,低速送り時のスティックスリップ現象など,圧縮性による空気圧の欠点を補うことと,増圧することにより,シリンダ径を大きくせずとも容易に大出力が得られることにある.この回路は中間停止や同期制御にも用いられ,これに必要なスロットル弁,ストップ弁をユニット化した機器もある.(a) ハイドロチェッカの構造(b) 一方向の速度制御回路図-3・62 ハイドロチェッカを利用した回路図-3・63 メータアウト方式による空気圧モータの駆動回路図-3・64 空気圧サーボシステムの基本構成b. ハイドロチェッカを利用した回路シリンダの駆動は空気圧で,速度調節はシリンダに取付けたハイドロチェッカ(hydrochecker)の油圧回路で行うというものである.図V-3・62にその構造とハイドロチェッカを利用した回路例を示す.空油変換器を利用した回路と同様に,空気の圧縮性による速度制御性の悪さを回避することができ,中間停止,二段速度制御なども容易に行うことができる.
