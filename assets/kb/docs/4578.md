# 4・9・2 焼結成形

> **章**: 第4章 プラスチック加工機械
> **節**: 4・8 積層プレス成形機

焼結成形（sintering molding）は，従来から窯業や粉末冶金な
どで用いられてきた焼成技術を粉末樹脂に応用したものである．そ
の基本原理は，(ⅰ)金型内で熱可塑性樹脂粉末を常温圧縮して予備
成形し，(ⅱ)この予備成形品を加熱炉内で加熱して，粉末粒子間で
の表面融着による多孔構造，さらには溶融粒子の流動による均質構
造を形成させ，(ⅳ)冷却して成形品を取出すプロセスである．
粉末樹脂の溶融粘度が非常に高くて焼結過程での粒子の流動変形
が困難な樹脂の場合は，粒子間に形成される空げきが成形品中に残
存することから，多孔構造の通気性素材や各種フィルタ素材などへ
の応用も試みられている．
特性（分子量，化学組成，溶融流動特性など），予備圧縮条件，焼
結条件などがおもな要因となる．
法などにより違ってくるが，金属やセラミックスの場合のような高
温・高精度の炉は必要がなく，一般のバッチ炉やトンネル炉であれ
式，高周波加熱，熱プレスなど，場合によって使い分けられてい
る．
性樹脂粉末にも用いられるが，フッ素樹脂や超高分子量ポリエチレ
対しても一部実用化されている．
