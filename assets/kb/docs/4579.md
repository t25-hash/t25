# 3・5・8 シリンダ以外の回路

> **章**: 第3章 空気圧システム
> **節**: 3・50 にあるセンタクローズ形切換弁は、ABR 接続、ABP接続の

揺動形アクチュエータは構造的にシリンダに近く、基本回路はシリンダと同じである。空気圧モータの場合は図V-3・63のように2ポート弁で回転と停止を制御する。正逆転タイプのものには3位置5ポート弁を使用する。回転数の調節は流量制御弁により行い、シリンダと同様、消費流量が少ないメータイン回路と負荷変動に強いメータアウト回路がある。空気圧駆動は、各種自動化装置における簡便な位置決めや力制御の手段として、ワークの搬送やクランプなどに多用されている。これらの制御方式は、アクチュエータ動作端点や外部ストッパを用いた当止め方式が主流である。一方、多品種少量生産に対応するためのフレキシビリティの確保や、新たな応用分野を確立するためには、中間停止による位置決めや連続的な位置、力の制御が求められる。これらの連続的制御を実現するためには、空気圧サーボ技術の導入が不可欠である。空気圧サーボ(pneumatic servo)は、各種試験機や鉄道車両、振動制御装置などへ応用されている。また、空気の柔軟性を生かした各種福祉機器や、人間親和メカニズムへの応用研究が進められている。空気圧サーボシステムの基本的な構成を図V-3・64に示す。アクチュエータには空気圧シリンダや空気圧モータが使用され、外力に対抗して慣性負荷を駆動する。電気-空気圧変換部は電気-空気圧比例制御弁やサーボ弁により構成され、アクチュエータに供給する空気の流量や圧力を調節する。センサには、おもに位置センサと圧力センサが使用されるが、必要に応じて速度センサや力センサも用いられる。また、通常、コントローラにはディジタルコンピュータが使用され、種々の制御則はプログラムの形で記述される。
