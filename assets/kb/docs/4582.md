# 4・1・1 概説

> **章**: 第4章 水圧システム
> **節**: 4・1 水圧システムの特徴

水圧システム(water hydraulic system)は古くから使用され、産業革命以後、石炭による蒸気の力でポンプを駆動し、発生させた水圧エネルギーを供給する水力供給会社がヨーロッパ各地に設立され、広く用いられていた。その代表的なものは、ロンドンのタワーブリッジの開閉装置、エッフェル塔のエレベータである。当時の水圧駆動システムは水漏れ、潤滑の悪さ、さびの発生などの問題があったことから、20世紀初めからは作動流体として油を使用する油圧システムが主流となった。しかしながら、近年環境に関する意識の高まりにつれて、またセラミックスなどの新材料の開発、表面処理技術の進歩、コンピュータによる高度な解析技術などの応用技術により上記の問題点が解決され、再び水圧システムが見直され、水道水を作動流体とするシステムの研究開発が進められている。しかし、再び1990年代前後から、添加剤を加えない清水(tap water)を媒体とした水圧機器の開発およびその応用が活発化している。初期の水圧技術と比較して今日の水圧駆動技術は、新しい材料の開発、加工技術や解析技術、制御技術の進歩に支えられた、初期の水圧とは全く異なる新しい水圧技術であり、アクアドライブシステム(ADS : aqua drive system)とも呼ばれている。水圧駆動システムにおける作動流体については、大きく分けて二つの考え方がある。日本、北ヨーロッパは水道水を作動流体としているが、アメリカ、イギリス、ドイツでは高含水作動液を含むとしている。地球環境保護という21世紀の科学技術の最大課題に対する貢献という観点から、ここでは水道水を作動流体とする水圧システムについて述べる。水道水は液体として手に入れやすく、安価である。廃棄処理に関しても環境に与える影響や問題も少なく、不燃性であり、その物性も明らかで、液圧システムの長所である高い出力-重量比を保ちつつ、作動流体の漏洩による環境汚染や火災の危険性などの問題点を解決でき、特別な管理設備などがいらないなど、作動流体としては便利な点が多い。現在、材料・加工・潤滑技術上の進歩や比較的クリーンな環境が要求される分野(例えば遊戯施設、医療、医薬、食品加工、半導体製造、原子力機器など)への応用を期待して、水を作動流体とする可能性の検討が進んでいる。水を作動流体として用いるには、それらの水が引き起こす問題点をどのように抑えて、システムの安全性・信頼性を確保するかを考える必要がある。水の体積弾性係数は油に比べて大きく、圧力波の伝わり速度が大きく、応答が速くなる。また、圧力変化に伴う体積減少が少ないことから、位置サーボシステムにおける負荷剛性が、油の場合に比べて高くなるという利点もある。一方、体積弾性係数が大きく、圧力波の伝わり速度が速いことから、弁の急閉鎖に伴うサージ圧が大きくなるという問題が生じる。水圧システム中で液体が加速され、液体の静圧が下がるとき、飽和蒸気圧より低下すると気泡が生じる(キャビテーション)。水の気化によるキャビテーション現象は、気泡の崩壊時の衝撃圧力による流体機器の破壊作用や性能の低下、振動や騒音の発生をもたらすため、極めて重要な問題となっている。飽和蒸気圧は流体の種類と温度に依存している。水の飽和蒸気圧は鉱物油よりも高く、絞り部に生じる壊食量は鉱物油の5,6倍ともなる。また、温度が高くなればなるほど飽和蒸気圧も上がっていくので、それに伴い水蒸気によるキャビテーションの危険性が増す。水の場合では温度を50°C以下に保たなくてはならない。そのほか、水は沸点が鉱油に比べて低く、凝固点は高く氷結時に膨張するなどを考慮すると、使用可能温度は非常に狭いといえる。粘度は液圧作動流体の最も重要な特性の一つであり、液圧システムの性能に大きな影響を及ぼす。水は油に比べて粘度が小さく、流体摩擦によるパワーロス(もしくはこれによる圧力降下)が小さくて済むという利点がある。一方、粘度が小さいことにより、内部漏れが増加してパワーロスが発生する。温度変化に対する粘度の変化が少ないことから、温度変化による動特性の変化が少ないという利点がある。粘度は潤滑性にも関わってくるもので、ある程度粘度がないと適度な潤滑性は得られない。潤滑性がないとしゅう動抵抗や摩擦が増加してしまい、性能の低下にもつながるおそれがある。水による構成要素の壊食は、水の質を維持し(例えば水のpH 値を6.5~8.5、塩素イオン含有率25mg/Lまでに抑えるなど)、さらに適した材質(ステンレス鋼、チタン合金、銅合金など)を選択することで制御することができる。水には電気伝導性があるため、洗剤などの不純物や添加物が存在した場合には、電解液として作用することによる壊食が生じる場合がある。今後は上述の問題が生じない適切な表面処理方法を確立して、製品コストを下げることが重要な課題である。防腐性の面では、細菌類の発生・成長がしばしば問題になる。概して、バクテリアや細菌、かびや他の微生物の発生は、フィルタの詰まりや悪臭を引き起こす。そのような微生物が繁殖し、生き延びるような状況は二つある。一つは作動流体中に塩分などの養分が存在していること、もう一つは温度が5~60°Cの範囲の環境にあることである。衛生的なレベルを維持するためには、3 m以上の細かさのフィルタをリザーバタンクのエアフィルタに付ける、連結部やタンクのふたを固く密閉してクリーンな状態に保つ、ピストンロッドやモータシャフトにパッキンなどを取付ける、水温の管理やフラッシング方法などの水質管理などが考えられる。現状では、14MPa定格の水圧機器(ポンプ、モータ、制御弁など)については市場で入手可能であるため、これらの機器を用いた水圧駆動システムの応用例が多く報告されている。上述の水圧駆動の利点を生かした自然環境保護という観点からは、臨海・河川に係る土木作業機械や水門などの稼働構造物の機械装置への応用があり、軟弱地盤改良用土木システムへの応用、汚泥回収システムへの応用などが報告されている。どちらも、水圧を利用した理由は、使用環境が水場であり電気駆動の場合の漏電による労災問題の発生、油圧の場合の漏れやホース破損が原因の油による土壌汚染を防ぐことにある。このほか、人に対する安全が重視されるアミューズメント、スポーツ関連装置への応用、クリーン性、清浄性を生かし、すべてのアクチュエータに水圧駆動を採用した食品加工機械(鳥の胸肉自動脱骨ロボット)などが文献で紹介されている。これらのシステムを構成する機器は市販されているが、値段は高く、油圧機器の数倍あるいは十倍程度であり、コスト面が水圧駆動システムの市場拡大の大きな障害となっている。一方、手近な水圧源である水道圧力(約0.35MPa)を利用したシステムも開発が進んでおり、環境と人に優しい機器として、水道圧力により作動するプール施設向けの介護用水圧リフタが開発され、下半身が不自由な障害者や高齢者が一般のスイミングプールを利用、あるいはリハビリテーションに活用する際に、プールサイドから水の中に前者を移動させるために利用できる。このほかにも水道圧力を利用したシステムとして、ドアの開閉装置などの開発例が文献で紹介されている。また、水道圧よりは高い7MPaまでの低圧水圧駆動システムについて、各地で研究が行われている。低価格で信頼性のある低圧水圧用機器を開発することを重要課題とし、既存の各種機器(ポンプ、モータ、バルブ、シリンダ、パイプ、ホース、継手)の実用可能性が広く調査され、実用化が期待できる機器に対して性能試験と耐久試験が数多く実施されている。多段の遠心形ポンプあるいは数万minで高速運転される単段遠心形ポンプは、低圧水圧用の水圧源として有望であること、多少の改良により空気圧シリンダを低圧水圧用アクチュエータとして使用できること、バルブ(弁)や配第V部フルードパワーシステム図-4・1 水圧駆動システムの応用分野とそれを支える基盤技術管材など空気圧用や他の産業用に従来から使われている機器で低圧水圧用に転用できるものがあることなどが示されている。水圧シリンダの位置制御を、低価格な低圧水圧システムにより実現することを目標に、低価格なオンオフ弁を用いた制御システムの研究が行われている。また、構造が簡単、ノンリークで水の清浄度に大きく左右されないシート形オンオフ弁が、低圧水圧用に有望であることなどが報告されている。このように7MPaまでの低圧領域では、他の産業分野で用いられている既存の機器を転用することで水圧駆動システムを構成できることから、コスト面でも問題はなく、近い将来広く用いられるようになることが期待されている。14MPaまでの中圧力領域用の水圧用に開発された機器であっても、その性能は油圧用機器に比べると劣っており(内部漏れ、大きな摩擦トルク)、水圧システムの精密制御は容易ではない。これに対処するために新しい制御理論を応用する試みがなされている。その例として半導体関連精密成形プレス機がある。本プレス成形機は半導体ウェーハの表面に樹脂を被膜するための成形機で、低速であっても十分滑らかな動作を実現しており、仕様を十分満たしている。このほか、油圧モータに比べて損失トルクが大きく、大きな非線形特性をもつ水圧モータの制御に新しい制御理論を応用し、性能改善を図った例など、油圧機器と同等の水圧機器開発がされるまでの間、制御理論で機器の性能を補う方向の研究もなされている。以上をまとめて、水圧駆動システムが広まっていくために必要な基盤技術と、その応用分野を図V-4・1に示す。潤滑性とシール性の低下、キャビテーション、腐食など水圧システムにおける基本的な問題点が土壌の部分に示されており、幹の部分に示した新材料、表面加工、トライボロジーなどの技術を使って機器を開発する必要がある。この基盤技術を応用した機器の開発と水圧駆動システムに対する需要の増大により、近い将来コストの問題は解決されるであろう。水圧システムの応用分野としては、図V-4・1の枝および果実の部分に示した自然環境保全に適した作業分野、クリーン性が求められる製造技術分野、防災・安全が求められる対人分野、特殊作業分野などである。これらの応用分野は、従来の油圧の応用分2―267野と重なるものばかりでなく、フルードパワーの新しい応用分野の開拓でもある。以上のように、水圧駆動システムは電子技術、制御技術の発達とも相まって、今後新しい水圧技術として展開していくものと期待されている。
