# 4・1・2 水圧機器

> **章**: 第4章 水圧システム
> **節**: 4・1 水圧システムの特徴

水圧システムは、添加剤をいっさい用いない清水あるいは水道水を作動流体とした駆動システムである。水圧システムに使用される水圧機器(water hydraulic component)は、主として従来の油圧機器に相当する機能を実現する機器として開発されている。しかし、水を作動流体とすることから、水の物理的あるいは化学的な特性によって、油圧機器とは異なる観点からの設計が必要とされる。例えば、水の粘性は油圧作動油より低いため、しゅう動部や漏れ流量の条件がより厳しい。また、水に対する耐食性のある材料を選択しなければならない。最新の材料技術やコンピュータによる解析・制御技術などを駆使して、さまざまな水圧機器が開発されている。a. ポンプ遠心ポンプは低圧大流量に適したポンプとして、およそ2MPa程度の圧力範囲で広く使用されている。効率は、おおむね40~50%とそれほど高くない。また、ベーンポンプ(vanepump)も低圧用に用いられており、効率は50~60%である。従来から用いられてきたこれらのポンプは、おもに低圧領域における水圧システム用のポンプとしての発展が期待されている。より高圧の領域における水圧システム用には、ピストンポンプ(piston pump)が適している。アキシアルピストンポンプ、ラジアルピストンポンプ、レシプロケーティングポンプに大きく分類されるが、アキシアルピストンポンプとレシプロケーティングポンプが大半を占めている。レシプロケーティングポンプは100MPa程度の超高圧を発生させることができ、ウォータジェットなどに従来から利用されている。しかし、クランク室とピストン室の間を遮断するためのシールの寿命や吐出し圧力の脈動に注意が必要である。中高圧領域の水圧システム用のポンプとしては、アキシアルピストンポンプが代表的である。定格圧力は14MPa前後のものが典型的である。ピストンなどのしゅう動部には、プラスチックとステンレスの組合せが使用されている。また、しゅう動部の面圧を低下することを目的として、機構的な工夫を施したポンプも開発されている。効率面では、油圧ピストンポンプと同程度か若干下回る。b. 制御弁(control valve)絞り弁やボール弁は、従来から他の分野で使用されていたものを転用できる可能性がある。チェック弁(逆止め弁)も同様である。方向切換弁は、比較的低い圧力範囲であればさまざまな種類のものが入手できる。中高圧の範囲では、小流量のものに限られるようである。高圧力では、ボール弁を電動モータや空気圧によって開閉する弁がある。高速に大流量を切換える弁は水圧駆動回路を構成するうえで重要な制御弁であるが、ポペット弁やボール弁のタイプのものが開発されている。リリーフ弁は、オーバライド特性に優れたバランスピストンタイプのものが、水圧システム用に開発されている。キャビテーションやエロージョンへの対策として、材料の選択、流路形状の工夫などがある。比例制御弁やサーボ弁では、先進技術を駆使した開発が行われている。スプールを静圧軸受で支持し、しゅう動特性を改善したサーボ弁がある。また、ボールねじ駆動形やムービングコイル形などのサーボ弁が開発されている。スプールには、ステンレスやセラミックスが利用可能である。スプールの変位制御系をマイナループとして内蔵し、油圧と同程度の流量特性と周波数特性が得られ、高速高精度な制御を実現する回路に用いられる。c. アクチュエータ(actuator)シリンダは、ステンレスを材質とするさまざまな仕様のものが利用できる。ロッド部やピストン部でのシールやウェアリングの選択は重要で、耐久性や摩擦力、負荷能力を考慮して選択する。また、ストロークエンドでクッション効果を得るために特別な設計をすることもある。水圧システム用のモータとして利用できるものは限られているが、ピストンモータやベーンモータが開発されている。これらの水圧システム用モータでは起動トルクが大きく、最低回転数が比較的高いものが多く、位置や速度などを高精度に制御する場合には注意が必要である。また、低速度で駆動するために水潤滑ギヤボックスを取付ける場合がある。d. 補器類アキュムレータは、ステンレス仕様のものが従来から利用可能である。フィルタやクーラも同様である。タンクは、ステンレス製のものやプラスチック製などが利用できる。タンクの材質は遮光性に優れたものが水管理上有利である。配管は、高圧の部分にはステンレス鋼管や高圧用の水用ホースが利用できる。低圧部には、ふつうのビニルホースなどを利用することもある。管継手は、高圧用のステンレス製継手がある。低圧では空気圧用の継手類が転用できる。水圧システムに用いる要素機器は、油圧システムの要素機器とおおむね同じ構造・機能を有しているが、油と水の物性が著しく異なる点を考慮しながら、機器の選定および回路の構成を検討する必要がある。水圧技術はいまだ発展途上にあり、今後の水圧機器の開発に伴って回路構成にもさまざまな工夫がなされると予想されるが、現時点での基本回路の構成は油圧回路(2・6節)を参照するものとする。以下では作動流体の相違に起因する水圧回路構成上の留意点を記す。
