# 4・2・1 水圧回路構成上の留意点

> **章**: 第4章 水圧システム
> **節**: 4・2 水圧回路(water hydraulic circuit)

a.粘性（動粘度，kinematic viscosity）の影響 液圧システ
ムでシステムの特性に最も顕著な影響を及ぼすのが，作動流体の粘
性（動粘度）である．水の粘度は作動油のおよそ20分の1から50
分の1であるため，水圧システムでは回路内の流れはおおむね乱流
となるが，同一寸法，同一流量で回路内の圧力損失を見積もると，
乱流，層流のいずれの場合でも水圧システムの圧力損失が小さ
い．したがって，配管内径や機器の公称口径が小さくなった
り，配管が長くなっても，同じ圧力損失となる場合がある．また，
圧力損失低減のため，ポンプ軸動力が若干低減する場合もある．油
圧回路からの置換えの際には，配管や機器の口径など選定の見直し
が必要となる場合がある．
また，水の粘度は絶対値だけでなく，温度や圧力による変化率も
作動油に比べて小さく，サーボシステムの周波数特性の温度
依存性が油圧に比べて小さくなる ことなどが挙げられる．した
がって，システム全体の特性を適正に維持するための温度調節器
は，油圧回路より簡便なものでもよい．
一方，粘性が小さいために同じすきまからの漏洩量は水のほうが
多くなり，スプール弁の漏れ量は油圧のおよそ3倍との実験結果が
ある．ポペット弁の組合せで，スプール弁の機能を実現する
方法も提案されており，この場合には，保圧のためのチェッ
ク弁が不要になる利点もある．スプール弁の利用の際には，安易に
油圧回路を置換えるのではなく，その利点を有効利用する構成にす
る必要がある．
b.蒸気圧（vapor pressure）の影響 水は作動油に比べて蒸
気圧が高いため，キャビテーションが発生しやすい．キャビ
テーションについては数多くの研究報告があり，素
材や形状，試験条件などによって発生限界は異なる．しかし，作動
油と比較して壊食量が大きい点では一致しており，キャビテーショ
ン発生時には機器やシステムが大きな損傷を受けると考えてよい．
絞り部でのキャビテーションの発生を防止するためには，一つの絞
り部での圧力降下を小さくし，絞りを二段に構成する，背圧を作用
させるなどの方法が有効である．また，温度上昇により蒸気圧
が急激に上昇すること，50℃近辺で壊食量が極大になること
から，作動水の適切な冷却も有効である．ポンプ吸込み部も
キャビテーションが発生しやすい箇所である．吸込み圧力と蒸気圧
との関係でキャビテーションは発生するが，ポンプの吸込み能力は
構造により異なるため，ポンプ選定の際は吸込み部でのNPSH
（net positive suction head）を確認し，適切な回路構成とする必要
がある．またアクチュエータの動作時に生じるキャビテーショ
ンも，原理的にはポンプ吸込み部と同じであるが，連続的には生じ
ず過渡的な現象である．油圧と比較して明りょうな相違は確認され
ておらず，むしろ水のほうが発生しにくい との報告もある．こ
のように，キャビテーションに対しては油圧回路よりも配慮が必要
である．
c.体積弾性係数（bulk modulus）の影響 水の体積弾性係数
はおよそ2.2GPaであり，油圧作動油の約1.5倍である．実シス
テムでの気泡混入や管路弾性を考慮に入れても，ほぼ同程度の違い
があることが知られている．このため，高応答化や高剛性化
が期待できる 反面，急激な弁の開閉やアクチュエータの急停止
に伴うサージ圧も大きくなる．サージ圧の増大を抑えるため
には，弁の開閉時間を長くしたり，弁の開閉動作を調整することが
有効である．
d.比熱（specific heat）の影響 水は作動油に比べ，大きな
比熱と熱伝導率をもつ．このため，圧力降下による温度上昇は作動
油のおよそ半分 であり，配管内の流体の移動に伴う熱伝達効
率は，作動油のおよそ20倍 となる．また，冷却器の熱交換
率もおよそ2.5倍となる．これらの特性は，流れが存在する箇
所での熱の局所的な滞留が起こりにくく，比較的小型の冷却器でも
十分な冷却特性を得ることができる ことを示している．
e .水質（water quality）の変化 液圧システムでは，システ
ムの特性を維持するために，作動流体の清浄度や粘度の経時変化を
適切に管理することが重要である．水圧システム用の清浄度，粘度
に関する規定はまだ明確化されていない．清浄度に関しては，現時
点では油圧のNAS（National Aerospace Standard，アメリカ航
空宇宙規格）等級に準じて適切なフィルタを設けて管理することが
望ましい．この際，ポンプの吸込み側にはポンプ吸込み性能との関
係から，フィルタを設置できないこともあるため，注意が必要であ

4・2・2応用回路
4･2･1項では，作動流体の相違から生じうる回路構成上の留意点
を示した．しかし実システムでは，これらすべてを考慮する必要は
ない．適切な機器を用いれば，仕様を満足するための簡単な再計算
や検討で，十分実用的な水圧回路を構成することができる．以下で
は，水圧回路の例として二つの実用システムの回路を紹介する．

b.棒状振動機駆動システム（drive system for vibrofloat)
図Ⅴ-4･3は軟弱地盤改質土木工法のバイブロフローテーション工
法に用いる，大型棒状振動機（バイブロフロット）を駆動する水圧
回路である．棒状振動機に内蔵した偏心荷重を，水圧モータで回転
させて振動を発生し，先端より噴出する水と振動を利用して，地盤
の所定の深さ（約18m）まで貫入する．噴出する水で土粒子の移
動を容易にしながら，地盤を振動させて地盤を締め固め，機体付近
に生じる間げきに砂れきなどの骨材を投入する．締め固めによる土
質密度の増加と，砂れき杭によるドレーン効果で土質が改善され
本回路は，ポンプの吐出し水を水圧モータと水圧サーボ弁に分岐
し，サーボ弁の開度を調節することで，モータに供給する流量（す
なわちモータの回転数）を調節するブリードオフ回路である．戻り
流量を流量計で計測してモータの回転数を間接的に計測し，サーボ
弁開度を水圧ユニット内部のコントローラによって制御している．
本システムでは，バイブロフロットに動力を供給する水圧配管が
極めて長く（50～100m）なるにもかかわらず，細いホースを用い
ても圧力損失を少なくできることが水圧化の利点の一つである．従
来の油圧駆動式システムでは，ブリードオフ部分の流量制御に圧力
補償形流量調整弁を用いているが，本例ではサーボ弁を用いること
で，遠隔操作や土質や深度に対応した広範囲の振動周波数設定を可
能としている．
