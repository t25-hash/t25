# 4・2・1 水圧回路構成上の留意点

> **章**: 第4章 水圧システム
> **節**: 4・2 水圧回路(water hydraulic circuit)

a. 粘性(動粘度,kinematic viscosity)の影響液圧システムでシステムの特性に最も顕著な影響を及ぼすのが,作動流体の粘性(動粘度)である.水の粘度は作動油のおよそ20分の1から50分の1であるため,水圧システムでは回路内の流れはおおむね乱流となるが,同一寸法,同一流量で回路内の圧力損失を見積もると,乱流,層流のいずれの場合でも水圧システムの圧力損失が小さい.したがって,配管内径や機器の公称口径が小さくなったり,配管が長くなっても,同じ圧力損失となる場合がある.また,圧力損失低減のため,ポンプ軸動力が若干低減する場合もある.油圧回路からの置換えの際には,配管や機器の口径など選定の見直しが必要となる場合がある.また,水の粘度は絶対値だけでなく,温度や圧力による変化率も作動油に比べて小さく,サーボシステムの周波数特性の温度依存性が油圧に比べて小さくなることなどが挙げられる.したがって,システム全体の特性を適正に維持するための温度調節器は,油圧回路より簡便なものでもよい.一方,粘性が小さいために同じすきまからの漏洩量は水のほうが多くなり,スプール弁の漏れ量は油圧のおよそ3倍との実験結果がある.ポペット弁の組合せで,スプール弁の機能を実現する方法も提案されており,この場合には,保圧のためのチェック弁が不要になる利点もある.スプール弁の利用の際には,安易に油圧回路を置換えるのではなく,その利点を有効利用する構成にする必要がある.b. 蒸気圧(vapor pressure)の影響水は作動油に比べて蒸気圧が高いため,キャビテーションが発生しやすい.キャビテーションについては数多くの研究報告があり,素材や形状,試験条件などによって発生限界は異なる.しかし,作動油と比較して壊食量が大きい点では一致しており,キャビテーション発生時には機器やシステムが大きな損傷を受けると考えてよい.絞り部でのキャビテーションの発生を防止するためには,一つの絞り部での圧力降下を小さくし,絞りを二段に構成する,背圧を作用させるなどの方法が有効である.また,温度上昇により蒸気圧が急激に上昇すること,50°C近辺で壊食量が極大になることから,作動水の適切な冷却も有効である.ポンプ吸込み部もキャビテーションが発生しやすい箇所である.吸込み圧力と蒸気圧との関係でキャビテーションは発生するが,ポンプの吸込み能力は構造により異なるため,ポンプ選定の際は吸込み部でのNPSH(net positive suction head)を確認し,適切な回路構成とする必要がある.またアクチュエータの動作時に生じるキャビテーションも,原理的にはポンプ吸込み部と同じであるが,連続的には生じず過渡的な現象である.油圧と比較して明りょうな相違は確認されておらず,むしろ水のほうが発生しにくいとの報告もある.このように,キャビテーションに対しては油圧回路よりも配慮が必要である.c. 体積弾性係数(bulk modulus)の影響水の体積弾性係数はおよそ2.2GPaであり,油圧作動油の約1.5倍である.実システムでの気泡混入や管路弾性を考慮に入れても,ほぼ同程度の違いがあることが知られている.このため,高応答化や高剛性化が期待できる反面,急激な弁の開閉やアクチュエータの急停止に伴うサージ圧も大きくなる.サージ圧の増大を抑えるためには,弁の開閉時間を長くしたり,弁の開閉動作を調整することが有効である.d. 比熱(specific heat)の影響水は作動油に比べ,大きな比熱と熱伝導率をもつ.このため,圧力降下による温度上昇は作動油のおよそ半分であり,配管内の流体の移動に伴う熱伝達効率は,作動油のおよそ20倍となる.また,冷却器の熱交換率もおよそ2.5倍となる.これらの特性は,流れが存在する箇所での熱の局所的な滞留が起こりにくく,比較的小型の冷却器でも十分な冷却特性を得ることができることを示している.e. 水質(water quality)の変化液圧システムでは,システムの特性を維持するために,作動流体の清浄度や粘度の経時変化を適切に管理することが重要である.水圧システム用の清浄度,粘度に関する規定はまだ明確化されていない.清浄度に関しては,現時点では油圧のNAS(National Aerospace Standard,アメリカ航空宇宙規格)等級に準じて適切なフィルタを設けて管理することが望ましい.この際,ポンプの吸込み側にはポンプ吸込み性能との関係から,フィルタを設置できないこともあるため,注意が必要である.一方,粘度については,水道法に規定された水を利用するとの観点から,油圧のISO VG(国際標準化機構ISO 3448粘度グレード)に対応する規定は不要と考えられる.また,作動水のその他の水質の変化に関しては,一般細菌の一時的な増殖がある程度で,変化は大きくないことが報告されている.作動水中の細菌の繁殖が望ましくない場合には,回路内に適宜殺菌装置の設置などを考慮する必要がある.
