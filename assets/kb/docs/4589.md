# 4・2・6 推力方向制御装置（TVC:thrust vector control )

> **章**: 第4章 ロケット機関
> **節**: 4・2 固体ロケット

固体モータの推力方向制御装置には，固定ノズル内に二次液体を
噴入して発生する斜め衝撃波によって生じた内壁面の圧力不平衡を
利用する液体噴射LI （liquid injection）TVCとノズル全体を機械
的に偏向させる可動ノズルMN（movable nozzle）TVCがある．
最近では，耐圧性を備えたゴムエラストマと金属シムの積層構造を
もつフレキシブルジョイント（可撓継手）を介してケースに結合さ
れたノズル全体を駆動・偏向させるMNTVCが一般的である．こ
の方式では，つねに主推力 とノズルの偏向角δによって決まる
横推力sinδを得ることができ，大きな制御力が必要な場合，二
次噴射量に対して制御力が飽和する傾向をもつLITVCより有利で
ある．下段モータでは，偏向角に依存するフレキシブルジョイント
のばね反力が大きいことと大型のノズル構造全体を偏向させる必要
から，大駆動力の油圧駆動アクチュエータを用いる．小型の上段モ
ータでは，空気力が無視できるため所要偏向角が小さく，またノズ

クチュエータを用いて高精度な制御を可能にしている．
フレキシブルジョイントはノズルの偏向方向だけでなく機軸方向
にも自由度をもち，燃焼内圧の上昇に伴いエラストマが圧縮されて
ノズル全体が後方に押し出される．これをPcバイアス（Pc-
bias ）と呼ぶ．ノズルには，通常，2基のアクチュエータが90°位
相で装着されるから，それらのストロークが固定されていると，
Pcバイアスによってノズルは，相対的にアクチュエータ装着側に
引き寄せられて不整偏向し，制御上のミスアライメントとなる．こ
の制御偏差を極小化するため，図4･22に示したように，ノズル軸
に関してアクチュエータと対称位置にその部位での変位を計測する
ポテンショメータを装着する．その出力をアクチュエータの駆動指
令信号に加算フィードバックしてアクチュエータの伸展量を修正
し，両者のストロークが等しくなるように制御する．剛体の場合，
この原理は両者のストロークの平均値をフィードバックすることと
