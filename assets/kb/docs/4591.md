# 4・3・2 液体推進薬

> **章**: 第4章 ロケット機関
> **節**: 4・3 液体ロケット

液体推進薬には燃料と酸化剤を燃焼させる二液推進薬（bi-
propellant ）と，一液のみで分解ガスを発生させる一液推進薬
（monopropellant ）がある．さらに二液に固体または液体の第3成
分を加えたり，2種類の燃料を切り換えて使用する3成分（三元）
推進薬（tripropellant ）も研究が進められている．貯蔵性の観点か
らは，常温で貯蔵できるものを貯蔵性推進薬（storable propel-
lant ），極低温での貯蔵が必要なものを極低温推進薬（cryogenic
propellant ）と呼ぶ．なお，極低温推進薬のうち，沸点の比較的高
いもの（例えば123K以上）は，宇宙貯蔵性（space storable）推
進薬と呼ぶ場合がある．貯蔵性推進薬のうち，ヒドラジン，モノメ
チルヒドラジンや四酸化二窒素は長期間の保管でも劣化が少なく，
システムを簡単にできることからロケットの姿勢制御，宇宙機/人
工衛星の推進系を中心に広く使用されてきた．しかし，これらの燃
料は毒性が強く，射点や着陸場，宇宙ステーションなどに対する安
全性確保のために，適切な機体のハードウェア設計，防護設備の準
備が必要となり，コスト増大/運用制約要因となっている．このよ
うな状況から，近年，四酸化二窒素/ヒドラジン系に代わる無毒性
（non- toxic）推進薬（グリーンプロペラント）の研究が盛んにな
っている．
推進薬を選定するには，理論的に発生しうる性能のほかに，推力
室の冷却特性，材料との適合性，応答性，密度，毒性，取扱いの難
易，貯蔵性，入手性，価格等さまざまな面からの検討が必要とな
る．
