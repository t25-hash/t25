# 1・1・1 分類と用途

> **章**: 第1章 は、専門外の人が、熱交換器(heat exchanger、以下H.
> **節**: 1・1 節では、さまざまな熱交換器を、いくつかの分類方法で分け

a.隔壁の有無による分類と用途 熱交換器は，隔壁の有無に
より，直接接触式熱交換器（direct contact type H.E.）と隔壁式
熱交換器（separate wall type H.E.）とに分類される．
直接接触式熱交換器は，熱交換する流体どうしが分離しやすい場
合に限って用いられる．冷却塔（クーリングタワー）は，代表的な
直接接触式熱交換器で，細かい水滴になった水と空気が直接接触し
て熱交換している．隔壁が不要で，接触面積を大きくとれるという
長所がある．
隔壁式熱交換器では，熱交換する流体どうしが固体壁で隔てられ
ている．大部分の熱交換器は隔壁式である．流体は壁を介して熱交
換するので，流体と壁との熱伝達率が重要であり，さらに隔壁の熱
抵抗も問題となる．
全熱交換器（total heat exchanger）は，隔壁式熱交換器の中
で，ガスは通さないが水分は通すような隔壁（例えば和紙）を用い
たもので，顕熱だけでなく潜熱も交換することができるので，この
ように呼ばれる．空調用に用いられることが多い．
