# 1・1・2 熱交換器設計に際し考慮すべき点

> **章**: 第1章 熱交換器
> **節**: 1・2 に示す。熱交換器としては古くから用いられている最も一般的

a. ポンプ動力、送風動力一般的に熱交換器に流体を送り込むポンプまたは送風機の動力と、得られる熱交換効率は相反する関係にある。熱交換器内の流体の速度を速めると熱伝達率は向上するが、そのために圧力損失が増加するので、高い流速を得るために必要なポンプまたは送風機の動力は増加する。同じ圧力損失で高い熱伝達率を実現できる熱交換器または伝熱面が良い設計であり、そのために新しい伝熱面の開発が行われているが、伝熱面または熱交換器の構造が既定のものであれば、運転による動力消費と熱交換器の効率向上によるシステム全体の省エネルギー効果を比較して最適な条件を選定することが必要である。b. 流体の選定システムの中に熱交換器を設置する場合に、一般には加熱したい、または冷却したい流体があり、それを加熱または冷却する手段を検討するという場合が多い。まれには加熱したい流体と冷却したい流体が同時にあって、それらをお互いに熱交換させることによって両方の目的を同時に達成することもあるが、通常は加熱または冷却の手段を別に準備することになる。加熱または冷却の媒体は水である場合が最も多い。水を用いる場合には開放系か密閉系か、水質は耐食性を考慮して適切か、必要十図1・23 コンパクト性による分類3―8熱機器分な量が確保できるか、などを考慮して決定する。水以外の熱媒、冷媒を用いる場合には熱交換器を構成する材料との耐食性、粘性、熱伝導率などを考慮して選定することが必要である。c. 温度範囲熱交換器の使用温度範囲を検討する場合に考慮すべきことは、流体側では流体の密度変化、粘性の変化による伝熱特性の変化、流体自身の温度変化による熱分解、低温の場合には凝固、結晶析出などである。熱交換器を構成する材料のほうでは一般的には流体の温度よりも高い温度に耐える材料が使用されるが、プラスチック材料を用いる場合などは高温による変質、変形に注意が必要である。金属材料は低温にさらされるともろくなる低温脆性に注意が必要なものがある。d. 構成材料熱交換器の設計に当たり構成材料の選定は重要である。熱交換器の材料として一般的に用いられている金属は鋼または合金鋼、ステンレス、銅および銅合金、アルミニウムおよびアルミニウム合金、チタンなどであるが、使用する流体に対する耐食性を第一に考慮して選定することが必要である。チタンやステンレスは耐食性能の高い材料であるが、チタンは銅などに比較して高価で加工性が悪いこと、ステンレスは塩素イオンを含む流体を流すと応力腐食割れを生じることがあるので注意を要する。材料自体の熱伝導率が高い材料を用いることは効率の高い熱交換器の設計の必須要件である。銅および銅合金は熱伝導率が非常に高く加工性も良く、価格的にも使用しやすいので、一般的に広く用いられているが、アンモニアを含む流体には容易に腐食されるので注意が必要である。耐食性を向上させるためにプラスチックの伝熱管や伝熱管の表面にプラスチックのコーティングを施す場合もあるが、プラスチックは一般的に熱伝導率が低く、かつ強度も金属に比較して弱いので、圧力の高い用途や流体によって表面が侵食されるような用途には向かない。e. 信頼性(reliability)、耐久性熱交換器の信頼性、耐久性を検討する際に考慮すべき点は、流体強度、熱応力、腐食である。使用流体自身の圧力が高く運転停止の繰返し頻度の高い場合には繰返し応力による疲労破壊に注意する必要がある。円形シェルに円形断面の伝熱管を使用するシェルアンドチューブ型熱交換器(shelland tube type H.E.)の場合には比較的問題はないが、プレート型など薄板構造の場合、角形シェルの場合など局部的な応力が発生し、これが繰返されることによって破壊に至ることがあるので計算と応力実測による確認が必要である。熱応力はあらゆる熱交換器で必ず発生する。シェルアンドチューブ型熱交換器などのようにシェル側と管側とで温度差が生じる場合には、管をU 字型にして熱膨張による変形を吸収させるような工夫がなされる。熱交換器全体が温度膨張するような場合には、接続する配管側で伸縮継手を使うなど、熱膨張による変形を吸収するような配慮が必要である。腐食についてはd項で述べたとおり材質の選定が最も重要であるが、使用している間に流体の中に含まれている不純物がスラッジとなって伝熱面に付着して、これが堆積することによって発生するデポジット腐食や、流体の流速が早すぎるためにエロージョンが発生する場合があるので、設計流速に注意が必要である。円管内に水を流す場合、エロージョンを考慮した場合の最大流速は、用途や使用温度によっても異なるが、一般には1~3m/s程度の値が採られている。f. 保守、点検熱交換器の保守、点検を考えるときに最初に留意すべきことは、流体が密閉系で使用されているか開放系で使用されているかという点である。空調用冷凍機の冷媒側のように、使用されている流体が不純物をほとんど含んでおらず、かつ密閉系で外部から運転に伴って不純物が混入する可能性のない場合には、熱交換器の冷媒側の保守、点検はかなり簡単になる。これに対して、冷却塔循環水のように系が開放系であったり、海水や井水などの一過性のものである場合には、それらの流体の中に含まれている不純物や腐食性因子をもつイオンなどが伝熱面に付着してスケールを作ったり、伝熱面表面を腐食したりするので、定期的な保守、点検が非常に重要になる。冷却水による伝熱面腐食は、水の管理方法によって防ぐこともできるし促進もされる。熱交換器を設計する立場では、内部の点検ができるようにするなどの保守を考慮した構造設計が必要になる。一般的なシェルアンドチューブ型熱交換器の場合には両端の水室カバーを外すことによって管内側をブラシなどによって洗浄することができる。シェル側については弱酸などの薬液を含んだ洗浄水を循環させる、いわゆる化学洗浄によってスケールやスライムの除去を行う。プレート熱交換器やその他の複雑な形状の熱交換器の場合にもブラシ洗浄の代わりにしばしば化学洗浄が行われる。飲料水を加熱冷却する熱交換器などのように、万一伝熱面が腐食などによって穴あきを生じたときに、相手の流体によって飲料水が汚染されるような場合には、ダブルウォール型の伝熱管や伝熱板を使用することがある。これは管や板の肉厚方向のほぼ中央部に外部とつながっている空げきを設け、なんらかの理由で管が板厚の半分まで腐食されると、空げきを通じて汚染流体が外部に漏れ出すので、これを検知することによって飲料水が汚染されるのを事前に防止することができるものである。g. 環境に対する配慮熱交換器における環境保護では二つの側面を検討することが必要である。一つは設計、製作、使用する熱交換器を最終的に廃却するまでの環境負荷および資源使用量をできるだけ低減すること、もう一つは熱交換器自身の伝熱効率をできるだけ高めて、それ自体を省エネルギーの手段として利用することである。前者に関しては使用する熱交換器のLCA(life cycle assess-ment)評価を行い、単位量の材料を製造するために必要とされるエネルギーの少ない材質の選定、製品の軽量化による資源使用量の低減、過大な圧力損失の発生防止によるポンプ運転エネルギーの削減、および廃却時のエネルギー消費の少ない熱交換器設計などが重要であり、この中でも特に運転エネルギーの削減効果が全体の環境負荷低減に大きく寄与することがわかっている。一方後者に関しては、温度効率の高い熱交換器の設計、流体保有量の少ないコンパクトな熱交換器の設計、熱交換器を含む全体のシステムとしての効率の最適化を目指したシステム設計などが環境改善に有効である。これらのような地球温暖化防止という環境保護のほかに、使用する流体が熱交換器から外部に漏れ出すことによる環境汚染防止も重要なテーマであり、このために熱交換器単独の腐食防止のほかに、万一漏れ出しが生じたときにそれを域内で食い止めるための防護堤の設置などの注意も環境保護のための検討必要事項であろう。が壁を隔てて互いに交じり合わずに流れる形式)を扱う。
