# 1・2・1 熱交換量

> **章**: 第1章 熱交換器
> **節**: 1・2 節では,隔壁式熱交換器(高温側および低温側の二つの流体

熱交換器内において,外部に熱の逃げがなければ,単位時間当たりに高温流体が失う熱量は低温流体が得る熱量に等しく,それを熱交換量と呼ぶ.すなわち,熱交換器内を流れる流体の質量流量,比熱および温度をそれぞれ,욣およびとし,高温側および低温側の諸量にそれぞれ添字およびを,また,いずれの流体についても入口および出口状態をそれぞれ添字1および2で表すこととすると,エネルギー保存則より熱交換量は次式で与えられる.=욠욣욠욠-욠=욂욣욂욂-욂(1・1)熱容量流量(heat capacity rate)=욣を用いると,式(1・1)は次のように書くことができる.=욠욠-욠=욂욂-욂(1・2)
