# 1・2・2 対向流、並流および直交流

> **章**: 第1章 熱交換器
> **節**: 1・2 節では、隔壁式熱交換器(高温側および低温側の二つの流体

高温側および低温側流体の相対的な流れ方向により、熱交換器(b) 並流(a) 対向流(d) 直交流(いずれの流体も非混合)(c) 直交流(一方の流体が混合)図1・24 熱交換器の基本的流路形式図1・25 二流体間の熱交換と温度変化(並流の場合)の基本的な流路形式は、図1・24に示されるように、対向流(counter flow)、並流(parallel flow)および直交流(cross flow)などに分類される。直交流において、一方の流体が、他方の流体の流れ方向に混合しない場合を非混合と呼ぶ。例えば、図1・24(c)において、管軸に直角方向に流れる流体は混合するが、管内を流れる流体は混合しない。また、図1・24(d)の場合には、フィン間に流入した流体および管内を流れる流体のいずれも混合しない。対向流、並流および直交流のように比較的簡単な流路形式については、設計計算手法が確立されている。
