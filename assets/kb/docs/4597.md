# 1・2・3 熱通過率

> **章**: 第1章 熱交換器
> **節**: 1・2 節では、隔壁式熱交換器(高温側および低温側の二つの流体

一般に、高温流体から低温流体への熱移動を熱通過と呼ぶ。式(1・1)あるいは(1・2)により表される熱交換量は、熱通過によってもたらされる。熱交換器全体の熱交換量は、式(1・3)を全伝熱面積にわたって積分することにより、次式で与えられる。実際には、伝熱面積は高温側と低温側で同じではないことが多い。このように、熱通過率の定義には、基準となる伝熱面積の指定が必要である。熱交換器の伝熱面の汚れおよび隔壁内の熱伝導による抵抗を無視できる場合には、式(1・9)より次式を得る。
