# 1・2・5 フィン効率と伝熱面効率

> **章**: 第1章 熱交換器
> **節**: 1・2 節では、隔壁式熱交換器(高温側および低温側の二つの流体

図1・27に示されるように温度の基板壁面にフィン(拡大伝熱面)を設けると、フィンの温度はフィンの先端に向かって降下するために、フィンと周囲流体との温度差-は先端に近づくにつれて減少し、フィンからの放熱量も低下していく。いま、このような温度降下がなくフィンの温度がどこでもであるような理想的なフィン(例えばフィン材の熱伝導率が無限大のフィン)を考えると、このフィンによる伝熱面拡大効果は100%である。ここで、αはフィンおよび基板面と周囲流体との間の熱伝達率で、その値は場所によって変化しないものと仮定している。式(1・23)で与えられるηをフィン効率(fin efficiency)という。例えば、図1・27に示されるような一次元フィンにおいて先端が断熱されている場合のフィン効率は次のようになる。このほか、さまざまな形状のフィンについてフィン効率は計算されている。図1・28に断面形状が長方形で、断面積が(a)一定の場合と、(b)直線的に減少する場合とについてフィン効率を示す。図1・28より、他の条件を同じにしたままフィンの長さを増やすとフィン効率は急激に低下することがわかる。ηはフィン効率ηの大きさおよび全伝熱面積のうちフィン面積が占める割合によって変わる。もしもフィン効率が100%であればηも100%となる。
