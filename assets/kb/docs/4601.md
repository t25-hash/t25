# 1・2・7 熱交換器の伝熱設計計算

> **章**: 第1章 熱交換器
> **節**: 1・2 節では、隔壁式熱交換器(高温側および低温側の二つの流体

熱交換器の基本的な伝熱設計計算は、1・2・6項までに示した熱交換器の諸パラメータ(욠,욂,욠,욂,욠,욂,,,等)の間の関係式を用いてなされる。その際に、基本式として式(1・21)を用いる場合を対数平均温度差法(LMTD method,log-mean temperature difference method)、また、式(1・38)の関係を3―11図1・31 入口・出口圧力損失係数と流路断面開孔率の関係用いる場合をε-法(ε-method)と呼ぶ。例えば、次の二つの場合:(1) 욠,욂,욠,욂,욠,욂,が与えられたときにを求める(所要伝熱面積を求める問題)。(2) 욠,욂,욠,욂,,が与えられたときに、욠,욂を求める(出口流体温度を求める問題)。を考えると、(1)については、対数平均温度差法あるいはε-法のいずれを用いても同程度の容易さで結果を得ることができるが、(2)に対して対数平均温度差法を適用すると、욠,욂が未知のため、繰返し計算を行わねばならない。この場合には、ε-法が便利である。
