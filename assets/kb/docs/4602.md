# 4・5・2 姿勢制御用ロケットシステムとコンポーネント

> **章**: 第4章 ロケット機関
> **節**: 4・5 制御用ロケット

ここでは一般的な制御用ロケットシステムである一液式システム
と二液式システムに関し，そのシステムとコンポーネントについて
a.一液式システム（monopropellant system) 一液式システ
ムは1種類の推薬を，触媒反応させることによって推力を得るもの
である．比推力などの性能面では二液式システムより劣り，また触
媒劣化による性能低下等も考慮する必要があるが，構造が簡素化で
きるため，特に大推力を必要としない姿勢制御系に広く使われてい
る．
推薬としてはヒドラジン（NH ）がおもに用いられ，触媒はイ
リジウムを用い，多孔質のアルミナ等に含浸させ粒状で使用され
る．
代表的な一液式システムの系統図を図4･67に示す．図4･67に示
されるように，システムの主要構成コンポーネントは，推薬タン
ク，遮断弁，スラスタ等である．この種のシステムでは推薬タンク
にあらかじめ高圧ガスを封入しておき，その圧力によって推薬を圧
送するブローダウンシステムが主流であり，通常推薬タンクを加圧
するために高圧ガスを封入する気蓄器や，推薬を昇圧するためのポ
ンプ等の昇圧源は有さない．
主要コンポーネントのうち，一液式スラスタの典型的な例を図
4･68に示す．スラスタの主要構成品は，推薬弁，触媒層，フィー
ドチューブ，サーマルバリア，ノズル等である．
b.二液式システム（bipropellant system) 二液式システム
は2種類の推薬（燃料・酸化剤）を混合反応させることにより推力
を得るものである．一液式システムに比べて高比推力を得ることが
できる．ただし，使用する推薬の組合せによってはパルス作動での
運用性に劣る面があり，国内の人工衛星ではアポジキック用エンジ
ン等，比較的要求インパルス量が大きくパルス噴射のない推進系に
使用されている．構造簡素化のため，推薬としては点火源を要しな
い自己着火式推薬が用いられることが多い．代表的な推薬は，ヒド
ラジン（NH )・モノメチルヒドラジン（MMH）/四酸化二窒素
（NO ）の組合せなどである．
この種の推進系は，要求インパルスが大きく燃焼秒時が長いもの
が多いため，推薬タンク加圧源として気蓄器・調圧弁などを有する
ことが多い．
二液式システム制御ロケットの例として，「かけはし（通信放送
技術衛星：COMETS）」のアポジエンジンを図4･69に示す．
