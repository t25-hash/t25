# 4・5・4 打上げ前整備

> **章**: 第4章 ロケット機関
> **節**: 4・5 制御用ロケット

制御用ロケットの発射前整備作業としては，推薬充てん作業，推
薬タンク加圧作業および各種機能試験になる．
一液式，二液式システムでおもに使用される常温貯蔵性推薬は毒
性を有しているものが多いため，充てん作業は自閉式耐環境作業服
（self - contained atmospheric protective ensemble suits ）を使用
し，専用の充てん室内において打上げまたは衛星搭載直前に実施さ
れる．
打上げ前整備作業として，「かけはし」の例を図4･71に示す．
