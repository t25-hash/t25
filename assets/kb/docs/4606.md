# 1・3・3 マルチフロー型凝縮器の性能

> **章**: 第1章 熱交換器
> **節**: 1・47 を見るといっそう明らかである.図1・47のグラフは,タイプ

マルチフロー型熱交換器(multi-flow type H.E.)はカーエアコン用凝縮機に使われている.凝縮器は単相流部となる加熱部,過冷却部と二相流凝縮部に分けられる.加熱部と過冷却部は単層流の管内熱伝達率,空気側熱伝達率と空気との平均温度差を基に計算すればよい.凝縮部の熱伝達率は流路の形状や寸法の影響を受けるため実験によって求められる.凝縮器全体の性能を簡易的に求める場合には,全体熱負荷の大部分を占める凝縮部のみを計算する.a. 偏平チューブ内の凝縮熱伝達率チューブ内部の凝縮熱伝達率はチューブの形状によって変化し,ベイパークオリティとの関係で実験より求められる.図1・48,1・49は代表的な4種のチューブ形状と冷媒として用いられるHFC-134aの管内凝縮熱伝達率の測定結果を示す.b. 空気側熱伝達率空気側には軽量性とコンパクト性を考えてルーバフィンが多く用いられる.ルーバフィンの熱伝達率の予測は実験により求めることが多い.代表的なルーバフィンの特性は伝熱工学資料により下式で示されている(図1・50).図1・50 ルーバフィン形状図1・51 マルチフロー型凝縮器の放熱性能例=0.25욣300<<4000=5.47욣욠욣70<<900:フィン間風速とフィン流体直径を基準とするレイノルズ数욣:ルーバピッチを代表長さとしたレイノルズ数욠:ルーバ曲げ高さ(mm) =sin욀×욣2で表せる욄:ルーバ切れ長さ(mm):フィン長さ(mm)伝熱特性はファクタ,抵抗特性はで示される.上式は次元的には成立しないが,これらをmmで表現された定数とみなせば実用上は問題ない.また,奥行寸法が40mmの場合であり,異なる場合はに対しておおよそ,熱伝達率は∝の関係がある.c. マルチフロー型凝縮器の性能例図1・51は代表的な自動車用凝縮器の前面面積を基準とした交換熱量を示す.交換熱量は風速との関係で示され凝縮温度や空気の温度を一定値として比較される.冷媒偏平チューブとして押出しチューブ[幅20×2(c),幅16×3(b)]およびロール圧延方式によるNRT チューブ(マルチフロー型凝縮器,1・1・1・b・vi項参照,幅16×高さ1.7mm)を示す.
