# 1・4・1 熱交換器フィンのシミュレーション計算

> **章**: 第1章 熱交換器
> **節**: 1・4 数値シミュレーションによる最適設計の例

熱交換器の性能を向上するには、伝熱面積(フィン)を増やして熱抵抗を低減する方法が簡単であるが、伝熱面積を増やしすぎると熱交換器のコンパクト性を損なうばかりか、材料がかさんでコストが増加する欠点がある。そこで、図1・59に示すような種々のフィンパターンを設けて伝熱促進を行い、フィン1枚当たりの熱交換量を増やす工夫がなされている。最近はコンピュータを用いたフィン周りの流れ解析をもとに、フィン配列の設計を行う例が多い。ここでは、フィンパターンの検討に用いられるシミュレーション計算について概説する。初期の研究では、パタンカー(Patankar)の開発した差分手法を用いた二次元の流れ解析手法を基に、スリットやルーバの二次元フィン配列の検討が行われた。図1・60に解析結果の代表例を示す。解析モデルはおよそ1万個の直交格子を用い、フィンを等温と仮定した定常計算を行っている。この時代の研究はフィン配列の比較がおもな目的のため、スリットの分割数やピッチ、ルーバの長さや角度を変化させた解析事例が多い。図1・60より、フィン間の流れは蛇行しているものの層流であり、フィンの性能は速度と温度場の助走域を含んだ層流伝熱に支配されていることがわかる。(a) プレートフィン(b) ワッフルフィン(c) ルーバフィン(d) スリットフィン図1・59フィンパターンの例最近、計算機の急速な普及と大幅な計算能力の向上を背景として、フィンと管を組合わせた三次元の熱交換器モデルによる解析が行われている。従来の直交格子では円管周りの格子生成が難しいため、モデルの形状に合わせて複数の座標系を使い分ける複合格子(compound grid system)や、BFC(boundary fitted coordina-tion、境界適合格子)が用いられている。図1・61に複合格子座標系を用いて作成した解析モデルと結果の一例を示す。管の後流を制御するねらいで、フィン面にフェンスを立てたモデルの解析を行っているが、フェンス周囲で流れに乱れができる様子が視覚的にとらえられている。このように、流れ解析はフィン周りのミクロな熱流動現象を調べるときに有効なツールである。しかしながら、複雑な形状の構造物周りの流れ解析では、格子生成に膨大な手間がかかるため、最近ははん用の流体解析コードを利用することが多い。図1・62ははん用コードで作成した解析格子と解析結果のフィン温度分布を表しているが、このように格子数が百万を超える複雑な形状でも、比較的容易に格子生成が可能である。ただし、はん用コードを利用する際には、格子の形状や間隔、計算アルゴリズムの選択が妥当であるかを、従来研究との比較によって確認することが望ましい。
