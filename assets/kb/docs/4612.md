# 1・4・2 熱交換器冷媒パスの最適化シミュレーション

> **章**: 第1章 熱交換器
> **節**: 1・4 数値シミュレーションによる最適設計の例

熱交換器を空調機用に使用する場合、管内側には相変化を利用できる冷媒(refrigerant)を使用することが多い。相変化を伴う流れでは圧力損失が単相流に比べて大きくなるため、通常、圧力損失低減のため熱交換器内を流れる冷媒パスを複数とする。その場合、各パスの冷媒流量がパスの長さや重力ヘッドの影響により変化し、熱交換能力に大きく影響を及ぼす。さらに、実際の空調機設計においては、空気側の風速分布やフィン(拡大伝熱面)の内部熱伝導の影響などが無視できず、これらを考慮した熱交換能力の予測計算により冷媒パス設計を行うことが必要となる。このような予測計算の図1・60 二次元流れ解析結果(上:速度場、下:温度場)図1・61 複合格子座標系の解析モデルと三次元流れ解析結果一手法として熱回路網法(thermal network method)を応用した熱交換器の能力予測手法がいくつか提案されている。例えば、加賀らによる計算においては、熱交換器の内部を段列ごとのブロックに分割し、各ブロックの構成要素(空気、冷媒、フィン、パイプ)ごとに節点を設けて熱抵抗で連結し、熱回路を構成する。また冷媒の相変化を計算するために有効比熱(effective specific heat)評価法を導入し、冷媒の流れに沿った圧力損失とそれに伴う飽和温度の変化を考慮するため、一次元の管路網モデルを導入して熱回路モデルと連立させて解を求める。このとき飽和温度の変化を等価吸熱項として熱回路網モデルに組込み、流れに沿った温度低下を表現している。図1・63は凝縮器に適用した場合の計算結果例であり、経路の異なる二つの冷媒流路を有する熱交換器を動作させた場合の冷媒の温度分布を冷媒の流れ方向に沿ってプロットしたものである。経路によりサブクールの始まる位置が異なる。これは、液冷媒の重力ヘッドにより冷媒パスごとの冷媒流量にアンバランスが生じる(このケースでは第一のパスの流量が増加する)ためであるが、冷媒の圧力変化を重力ヘッドも含めて考慮した計算結果はこの傾向をよく表している。次に、本計算手法を用いて空調室内機の熱交換器を蒸発器として用いた場合のパスパターン最適化を行った例を図1・64に示す。この検討では室内機の吸込み空気風速の影響を考慮しており、逆V の字に構成された熱交換器の背面側の風速が前面側に比べて不足する傾向にある。図1・64によれば、改善前[図1・64(a)]において背面側の冷媒パスの過熱ガス領域が大きくなり、熱交換器の面積を有効に利用できないことが示唆されている。これを回避するた(a) 解析格子(フィン部分のみ)(b) フィン温度分布図1・62 はん用コードによるフィン解析格子とフィン温度分布図1・63 計算結果の例(凝縮器)3―23めに背面側パスの流路長さを短くした改善後[図1・64(b)]においては、冷媒流量の分配が最適化され過熱ガス領域がほとんどなくなることが計算により示され、実験結果ともよく一致している。このパス変更による熱交換能力の改善効果は113%であった。以上のように、熱回路網法などを応用して熱交換器内部の冷媒の局所的な相変化状態を予測する数値シミュレーションが可能であり、これを用いることで机上において最適な冷媒パスを設計することができる。
