# 1・4・3 自動車用ラジエータの最適設計

> **章**: 第1章 熱交換器
> **節**: 1・4 数値シミュレーションによる最適設計の例

現在主流のコルゲートフィン型ラジエータ(corrugated fin typeradiator、図1・65)は、1950年代後半にアメリカで開発され、放熱性能に優れ、かつ大量生産が容易で安価に作れることより、その後急速に世界中に広まっていった。日本においても1963年ごろ、従来生産されていたプレートフィン型ラジエータ(図1・66)に代わり、コルゲートフィン型が登場した。1957年には自動車部品工業会が中心となって工業技術院機械試験所に風洞試験装置を設置し、ラジエータに関する基礎データが得られるようになった。ここでラジエータの放熱性能のおよそ80%を占めるルーバフィン(louver fin)の解析例について述べる。熱伝達に大きく影響を及ぼすルーバ間の流れは、図1・67に示すように境界層の発達により排除厚さ(displacement thickness)δ′を考慮する必要がある。流体力学の専門書から、次式が誘導されている。この判定基準として次式が与えられる。このように、ルーバ間の流れがシミュレーションで求められるので、,に関する熱伝達特性も計算にて容易に推定でき、最適解のパラメータ検討が可能になった。図1・68は、局所ヌセルト数の計算結果の一例である。この結果と過去の実験データとの比較を図1・69に示す。このように、ラジエータ単体での最適値をシミュレーションにて求められるようになり、効率的かつ精度の良い仕様検討が行われるようになった。現在主流のSR ラジエータ(シングルロータイプラジエータ)は、こうした解析技術を駆使して最適仕様が決定され製作されている。このラジエータの特徴は、フィンピッチが密、コア厚みが従来の半分、チューブは2列から1列となり、軽量化30%かつ大幅なコスト低減に効果があった。さらに近年では、自動車のシステム全体のシミュレーションが可能となり、前述のラジエータ単体での最適設計に加え、実車での使用環境までも考慮した条件での最適設計が行われている。熱交換器をルームエアコンに実装する場合の実装技術の実例を、室内機、室外機に分けて解説する。
