# 1・5・1 ルームエアコン(room air conditioner)室内機への

> **章**: 第1章 熱交換器
> **節**: 1・5 熱交換器実装技術の例

熱交換器(heat exchanger)実装技術熱交換器での熱交換量を増大するためには、伝熱面積拡大と熱伝達率向上の二つの方法がある。しかしながら、前者の場合には、熱交換器のサイズアップにより機器の大型化をまねき、製品としての実用性を損なう場合がある。また、後者の場合には、伝熱促進手段により作動流体の圧力損失が増大し、熱伝達率向上が機器全体での効率向上に結びつかない場合もある。ルームエアコンは、内部に封入された冷媒と空気との熱交換により、空気を冷却あるいは加熱する機器である。したがって、熱交換器はルームエアコンの機能を実現し、その効率を左右する極めて重要な部品である。通常、ルームエアコンの室内機と室外機のそれぞれが熱交換器を有している。一方、ルームエアコンは家庭内に設置されるために、コンパクトなサイズが好まれ、居住空間に設置される室内機に対しては、特にこの傾向が強い。図1・70にルームエアコン室内機とその内部を示す。室内機には、熱交換器のほかファンやファンモータといった主要部品が収納されている。ファンは熱交換器にて熱交換する空気を流す働きをもつ。したがって、熱交換器とファンとは、ファンによる空気流れが効果的に熱交換器を通過するような位置関係にあり、かつ両者の占有する容積を最小化できるような配置が望まれる。その結果、図1・70に示すような相対位置関係が採用される。ファンによって流される空気は、吸込みグリルより室内機内に入り、熱交換器を通過した後、吹出し口から居住空間に戻される。図1・71はルームエアコン室内機の断面図であり、室内機内での熱交換器とファンの相対関係と熱交換器の断面形状を示す。熱交換器は銅管にアルミフィンを密着させたフィンアンドチューブ型(fin and tube type H.E.)であり、フィン厚さは通常0.1mm程度、フィンピッチは1.1~1.3mmである。a. 室内熱交換器の伝熱面積拡大図1・71に示すように、熱交換器はファンの外周に沿うように配置されており、熱交換面積を最大化するために、熱交換器の形状(アルミフィンの形状)をファンと同心円に近づける工夫がなされている。図1・71の例では、フィンを複数回折曲げて円弧に近づけている。図1・72には、その他の室内熱交換器形状の例を示す。図1・72(a)の熱交換器は、1箇所の折曲げからなる比較的単純な形状であり、図1・71の形状に比べると熱交換面積は小さい。この形式は初期のころのエアコンにしばしば用いられ、現在でも比較的廉価な機種に採用されている。図1・72(b)の熱交換器では、熱交換器のフィンが円弧状の形をしており、その結果熱交換器がファンの外周図1・70 ルームエアコン室内機図1・71 ルームエアコン室内機断面と熱交換器(a)(b)図1・72 ルームエアコンの室内熱交換器の例図1・73 アルミフィンへのスリット加工3―25に沿うように配置でき、熱交換面積が増加している。また、サブ熱交換器と呼ばれる別ピースの熱交換器を追加していっそうの伝熱面積増加を図っている。図1・72の二つの熱交換器を比較すると、図1・72(b)の熱交換器の伝熱面積は図1・72(a)に比べ約40%増加している。熱交換器はアルミフィンで構成されているが、アルミフィンのピッチを狭くすればフィン枚数が増加し、伝熱面積も増加する。一方、狭いフィンピッチでは空気の通風抵抗が増大するので、フィンピッチを狭くすれば、同一風量を送風するためのファン入力が増大する。特に冷房運転時においては、熱交換器内にて空気中の湿分が凝縮するので、狭いフィンピッチではその水滴の離脱が困難となり、通風抵抗の急激な増大を伴う場合がある。したがって、ルームエアコンの設計においては、伝熱面積の増加と通風抵抗の増大のバランスにより最適なフィンピッチを見いだしている。b. 室内熱交換器の伝熱促進熱交換器の性能向上手段としては伝熱促進があり、ルームエアコンの室内熱交換器においても、いくつかの伝熱促進技術が採用されている。最も代表的な手段が、図加工である。アルミフィン表面では境界層が発達するため、後流部分では熱伝達率が低下する。そこで、図1・73のようなスリットをアルミフィンに加工し、境界層を乱すことにより熱伝達率の向上を図っている。しかしながら、これらのスリット加工も通風抵抗を増加させることになるため、伝熱促進と通風抵抗をバランスさせた設計が必要となる。熱交換器の銅管の中には冷媒が流れ、この冷媒の伝熱促進手段として、銅管の内壁面にさまざまな溝形状を加工した内面溝付き管(grooved tube)が考案され実用化されている。内面溝付き管には、伝熱面積の増大のほか、蒸発時の液膜の保持や気液のかくはんといった複数の効果をねらったものが多い。
