# 2・1・1 概要

> **章**: 第2章 空気調和と空調機器
> **節**: 2・1 空気調和の概要

空気調和(air conditioning)とは,室内の温度,湿度,静浄度および気流を,その室の目的に適合するように調整することである.事務所や住宅など,在室者を対象とする場合を保健空調,生産プロセスなどを対象とする場合を産業プロセス空調という.保健空調では,在室者を快適で健康な状態に保つことが必要であり,産業プロセス空調では,生産プロセスに必要な環境を提供することによって合理的な生産を支援することが必要である.保健空調では,人体に快適な温熱環境を提供することが基本であり,人の活動量と着衣量に対応して適切な空調条件が設定される.産業プロセス空調では,生産プロセスによって必要な環境条件が異なり,保健空調とは著しく異なる空調条件が設定される場合もある.恒温恒湿室のように時間変動幅が制限された空間や,クリーンルームのような超清浄空間もある.
