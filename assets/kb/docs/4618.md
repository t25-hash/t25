# 2・1・3 産業プロセス空調(industrial process air condition

> **章**: 第2章 空気調和と空調機器
> **節**: 2・1 空気調和の概要

ing)-産業プロセス空調は製品の製造プロセスに必要な環境を提供することにより、合理的な製品の製造を支援することを目的としている。生産プロセスに必要な環境は製品によって異なり、生産プロセスに対応したさまざまな環境条件が設定される。製品の加工精度や品質に大きな影響を与える室内空気の環境要素は、温度、湿度、空気の清浄度であり、気流も重要な要素になる場合がある。温度と湿度は、繊維、紙、食品、医薬品などの吸湿性物質の生産プロセス、日本酒などの発酵プロセス、金属製品の生産プロセスなどで重要になる。湿度が低い場合に発生する静電気は半導体やフィルムなどの生産プロセスで障害となる。精密機械の加工や組立、半導体などの微細加工を行うプロセスでは、空間における均一性や時間変動が加工精度や品質に影響を与える。空気の清浄度は、電子デバイスや精密機器の製造など、多くの産業分野で空気中の浮遊微小粒子が生産プロセスの障害となる。浮遊微小粒子が管理された清浄空間は、合理的な製造を支援する重要な環境要素の一つである。医薬品や食品の製造などでは、空気中の浮遊微生物が製品の品質低下の原因となる。医薬品や食品は、人の健康や生命維持に密接な関わりをもつものであり、空調設備の果たす役割は極めて重要である。空気中の浮遊微小粒子や浮遊微生物の制御を必要とする空間にはクリーンルーム(clean room)が用いられる。おもに空気中の浮遊微小粒子が管理された空間は工業用(インダストリアル)クリーンルーム、おもに空気中の浮遊微生物が管理された空間はバイオロジカルクリーンルームと呼ばれる。汚染物質が外部から室内に侵入するのを防止したり、室内で発生した汚染物質が外部に漏えいするのを防止するため、室内の圧力を制御することも行われる。例えば、クリーンルームでは室内圧力を外部よりも高く設定して汚染物質の侵入を防止している。また、原子力施設や封じ込め施設などでは、室内圧力を外部よりも低く設定して汚染物質の漏えいを防止している。
