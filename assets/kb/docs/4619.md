# 2・1・4 換気および排煙

> **章**: 第2章 空気調和と空調機器
> **節**: 2・1 空気調和の概要

換気(ventilation)は,自然または機械的な手段によって室内の空気を外気と入れ替えることである.換気のおもな目的は室内空気の清浄化および汚染物質の除去であるが,燃焼装置への給気を目的とする場合もある.排煙(smoke exhaust)は,火災によって発生した煙を迅速に屋外に排出し,避難路に侵入させないようにすることである.換気の方法には自然換気と機械換気がある.自然換気は自然風による圧力差や浮力を利用して,機械換気は送風機や排風機を用いて換気を行うものである.機械換気には,給気と排気を機械で行う第一種換気,機械給気と自然排気口を用いる第二種換気,自然給気口と機械排気を用いる第三種換気がある.自然換気と機械換気の方式は目的によって使い分けられる.換気には,室内全体を換気する全体換気と,汚染物質が発生する場所を局部的に換気する局所換気がある.全体換気は,室内で発生した汚染物質を外気で希釈しながら排出することにより,室内環境を基準値以下に保とうとするものである.汚染物質の種類や発生量によっては大量の換気量が必要になることがある.局所換気は,発生した汚染物質が室内に拡散する前に捕捉して室外に排出しようとするものであり,汚染物質の発生源が固定されている場合に有効である.局所換気では,発生した汚染物質を確実に捕捉することが重要であり,有害物質を取扱う労働環境に設置する局所排気装置では法令で制御風速が定められている.換気設備の性能は人の健康に影響を及ぼすため,法令で設置基準や管理基準が定められているものがある.建築基準法には換気設備の設置を義務付ける規定があり,構造や能力などの基準は施行令や告示に定められている.また,ビル衛生管理法には特定建築物の室内環境の管理基準が規定されている.労働環境に関しては,特定業務を行う作業場について,作業環境の測定が労働安全法に規定され,作業環境の管理濃度が告示に定められている.換気設備は,これらの基準や管理濃度に適合するばかりでなく,対象とする室の使用目的にも合致するように計画される.排煙設備は人の生命に関わるため,設置と構造が建築基準法および施行令と告示,ならびに消防法に規定されている.建築基準法に従った排煙設備を設置した場合は,原則として消防法の適用を受けない.ただし,建築基準法では設置義務がない部分や免除された部分でも消防法で設置義務がある部分は消防法の適用を受ける.
