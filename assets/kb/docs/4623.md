# 2・3・1 概要

> **章**: 第2章 空気調和と空調機器
> **節**: 2・3 空調負荷

空調負荷(air conditioning load)は、空調する目的の空間を一定の温熱環境に維持するために空調設備で処理する熱量のことで、単に熱負荷と呼ぶこともある。空調負荷は、室内および室外の設計条件に基づいて、室を出入りする熱や室内で発生する熱を計算して求めた室内負荷と、その室内負荷に取入れ外気の負荷、搬送機器の動力、ダクトや配管における侵入熱、損失熱などを加えた空調機負荷や熱源装置負荷に分類される。このような空調負荷計算で得られる数値は、空調システムの設計の基本となるので重要である。空調負荷は、冷房負荷と暖房負荷に分けてそれぞれの最大負荷を計算するのが一般的であるが、年間シミュレーションのように時々刻々変動する条件に対して計算する場合もある。最大負荷計算は、気象データや建物使用条件などにより年間の最大負荷が発生する日について、負荷変動が同一のパターンで繰返されるものとして計算する周期定常計算と、最大負荷の条件が1日中続くとして計算する定常計算がある。冷房負荷は周期定常計算、暖房負荷は定常計算によって行うのが通例である。これに対して年間シミュレーションは非定常計算であり、年間エネルギー消費量や負荷変動特性を知るための計算などに適用される。室内負荷を室の熱収支で考えると、室内に流入する熱と室内で発生する熱、すなわち取得熱を除去するために必要な熱が冷房負荷である。一方、室内から流出する熱、すなわち損失熱を補うために供給される熱が暖房負荷に相当する。空調負荷のおもな構成要素を表気象や建物などの条件により異なるため、その要素を考慮するか、無視するかは設計者の判断による。空調負荷計算の手順を図2・6に表2・1 空調負荷の構成要素負荷構成要素顕熱潜熱冷房暖房ガラス窓透過日射熱負荷○○△壁体○○○ガラス窓○○○通過熱負荷屋根○○○室内負荷土間床・地下壁○×○すきま風熱負荷○○○○照明○○△内部発熱負荷人体○○○△器具○○○△室内負荷○○○○送風機による負荷○○×空調機負荷ダクト熱負荷○○○外気負荷○○○○空調機負荷○○○○熱源装置負荷ポンプによる負荷○○×配管熱負荷○○○装置立上り負荷○○×△注:○考慮する、×無視する、△通常は無視するが、影響大の場合は考慮する。
