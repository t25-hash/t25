# 2・3・2 設計条件

> **章**: 第2章 空気調和と空調機器
> **節**: 2・1 に示す.各構成要素が冷房負荷や暖房負荷に占める割合は,

a. 室内条件保健空調における室内温湿度の設計条件として,夏期は乾球温度26°C(25~27°C)で相対湿度50%(50~60%)程度,冬期は22°C(20~22°C)で50%(40~50%)程度の値が一般的に使用される(2・1・2項参照).( )内の値はおおよその選定幅であるが,設計条件は建物および室の用途,在室者の活動や服装の状態,省エネルギー性などを考慮して決定しなければならない.産業プロセス空調においては,生産される製品の種類や生産プロセスに応じて最適な条件とすることが求められ,室内あるいは特定箇所の温湿度の許容変動範囲を指定される場合もある.b. 外界条件(outdoor condition)主要都市の冷暖房設計用の乾球温度と露点温度を表2・2に示す.また,計算精度を上げるために用いられる時刻別温湿度の例を表2・3に示す.表2・2,2・3は,過去の気象データを統計処理したもので,超過確率を2.5%としている.この統計処理をTAC法(Technical Advisory Com-mittee method,TACはASHRAEの技術諮問委員会)といい,この場合TAC2.5%と称する.
