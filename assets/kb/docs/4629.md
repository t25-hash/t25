# 2・4・1 空調計画

> **章**: 第2章 空気調和と空調機器
> **節**: 2・4 空調方式

建物の空調について計画するときには、当該建物の性格に適合したものとすることが肝要である。特に、負荷側設備としての空調方式の選定には十分な配慮が必要である。空調方式を分類する場合に、その取上げ方によって種々の方法があり、例えば以下のようになる。(1) 対象による分類中央(セントラル)式、各階式、個別式、パーソナル(個人)式(2) 熱媒による分類全空気方式、水-空気方式、全水方式、冷媒方式、蒸気方式(3) 人体への作用による分類対流方式、放射方式(4) 負荷側使用機器による分類エアハンドリングユニット(AHU)方式、ファンコイルユニット(FCU)方式、誘引(インダクション)ユニット方式、放射パネル方式、フロアヒーティング(床暖房)方式、パッケージ空調機(エアコンディショナ)方式(空冷式、水冷式、ヒートポンプ方式)など(5) 熱媒の制御方式による分類定風量(CAV)方式、変風量(VAV)方式、単一ダクト方式、二重ダクト方式、ターミナルリヒート(再熱)方式、ゾーンリヒート方式、マルチゾーン方式(6) 配管方式による分類冷水・温水4管式、冷水・蒸気4管式、3管式(冷水往管+温水往管+還り管)、冷温水切換え2管式上記の分類による方式を組合わせて空調方式を選定するのが一般的である。よく採用される方式を表2・14に示す。
