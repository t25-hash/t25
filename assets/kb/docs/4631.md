型
機械式マニピュレータでは，マスタアームとスレーブアームの構
造は同一である必要があった．これはマスタアームとスレーブアー
ムで対応する個々の関節が機械的に結合されていたため，同じ構造
でないと操縦者が指令するマスタの手先の動きがスレーブ側で再現
できなくなってしまうからである．サーボマニピュレータにおいて
も，初期のころは対応する個々の関節ごとにサーボ系を組んでいた
ので，基本的にマスタとスレーブは同構造である必要があり，大き
さが異なる場合でも相似形が許されるだけであった．このようにマ
スタアームとスレーブアームの構造が同一（もしくは相似形）であ
るものを同構造型マスタスレーブシステム（master -slave system
with same configurations ）と呼ぶ．同構造型は，マスタとスレー
ブの構造が同一でなければならないという制約があるが，基本的に
は対応する関節ごとに1自由度の制御系を組めばよく，システムと
しては簡単になる．また同構造のため操縦者は自然にスレーブアー
ムの特異状態を避けることができ，万が一特異状態になっても関節
レベルの制御のため制御上問題になることはない．
