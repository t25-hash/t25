# 2・5・1 吹出し口と室内気流

> **章**: 第2章 空気調和と空調機器
> **節**: 2・5 搬送システム

a. 吹出し口の種類吹出し口(air diffuser,air outlet)は、種類と形状により誘引特性、広がり、到達距離など吹出し気流の特性、性状が大きく相違する。表2・23は現在使用されている吹出し口をその形状と気流特性により分類したものである。吹出し口は大別して、ふく流吹出し口、軸流吹出し口、線状吹出し口、面状吹出し口に分けられる。b. 吹出し気流の諸性質周壁の影響のないような十分に広い空間に吹出される自由吹出し気流は、周囲空気を誘引・混合しつつ広がり、次第に速度が減少する。円形またはあまりアスペクト比の大きくない長方形開口からの軸流自由吹出し気流は、その大部分が流域に対して、展開角18°~20°程度で円すい状に展開する。空気調和でおもに利用される気流域は、第3域である。c. 非等温吹出し気流吹出し口から気流風速が0.25m/sになる位置までの距離を到達距離(throw)という。空気調和では一般に室温と異なる温度の空気を吹出すので、吹出し空気と室内空気との混合空気は、その温度によって降下(冷風吹出しの場合)または上昇(温風吹出しの場合)し、到達距離に達するまでに生じた気流の降下または上昇距離を気流の降下度または気流の上昇度という。円形またはアスペクト比のあまり大きくない長方形の吹出し口から水平に吹出された非等温気流の自由空間における中心速度、到達距離、断面最大温度差は次式で与えられる。代表的な吹出し方式について、冷房時と暖房時との室内気流の特徴を図2・28に示す。
