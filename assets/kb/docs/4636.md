# 2・5・4 蒸気配管

> **章**: 第2章 空気調和と空調機器
> **節**: 2・5 搬送システム

a. 配管方式蒸気とその凝縮水とを同一の管で運ぶものを単管式,別々の管で運ぶものを複管式といい,ほとんど後者の方式が用いられている.凝縮水を重力によってボイラへ戻すものを重力還水式(gravityreturn system)といい,ポンプを用いてボイラへ戻すものを機械還水式という.一般に後者が用いられる.凝縮水を確実に戻すために,還水管系の終端(ボイラへ最も近い側)に真空ポンプを設けるものを真空還水式(vacuum return sys-tem)といい,圧力200kPa以下程度の蒸気配管系の還水法として広く用いられている.真空ポンプの真空度は-25~-35kPa程度である.b. 許容圧力降下と制限速度蒸気配管の設計に当たっては,全圧力降下をボイラの初期蒸気圧の2分の1以下,普通は3分の1程度とする.表2・28に一般的に用いられている初期蒸気圧と許容圧力降下の関係を示す.圧力300kPa程度以上の蒸気管では,表2・28 蒸気管の許容圧力降下初期蒸気圧(絶対圧)全圧力降下単位摩擦損失kPakPaPa/m真空環水式7~1430~600.530~70100~175500~1200175~200500~2300表2・29逆こう配および立上り蒸気管内の許容最大蒸気流速(m/s)(a) 逆こう配管こう配管径()0.02 0.04 0.08 0.12 0.17 0.25 0.35 0.422.43.44.04.34.95.26.76.72.73.74.65.25.86.77.37.63.44.35.26.16.77.67.99.43.74.95.86.77.37.98.510.14.65.57.38.29.19.89.810.1*:0.01=1/100(b) 立上り管管径()最大速度9.110.3 12.2 13.5 16.0 18.3 19.2 21.9管内径をmmとしたとき,最大流速を0.2m/s程度を標準とするが,圧力200kPa程度以下の蒸気管では,蒸気と凝縮水とが逆流する逆こう配管や立上り管では表2・29に示すような制限速度以下に抑える必要がある.c. 蒸気配管の摩擦損失線図圧力101.325kPa,温度100°Cの飽和蒸気を基準とし,種々の圧力における蒸気の流量,管径と管長1m当たりの摩擦損失,および管内流速を求める線図(流量線図ともいう)を図2・33に示す.図2・33の使用法は図中に点線で示してある.ただし,管材は配管用炭素鋼鋼管(JIS G 3452:1962)および配管用アーク溶接炭素鋼鋼管(JIS G 3457:1962,管呼び径350A 以上,肉厚7.9mm)とし,内面の絶対粗さε=0.045mmとしている.
