# 2・5・4 蒸気配管

> **章**: 第2章 空気調和と空調機器
> **節**: 2・5 搬送システム

a. 配管方式蒸気とその凝縮水とを同一の管で運ぶものを単管式、別々の管で運ぶものを複管式といい、ほとんど後者の方式が用いられている。凝縮水を重力によってボイラへ戻すものを重力還水式(gravityreturn system)といい、ポンプを用いてボイラへ戻すものを機械還水式という。一般に後者が用いられる。凝縮水を確実に戻すために、還水管系の終端(ボイラへ最も近い側)に真空ポンプを設けるものを真空還水式(vacuum return sys-tem)といい、圧力200kPa以下程度の蒸気配管系の還水法として広く用いられている。真空ポンプの真空度は-25~-35kPa程度である。b. 許容圧力降下と制限速度蒸気配管の設計に当たっては、全圧力降下をボイラの初期蒸気圧の2分の1以下、普通は3分の1程度とする。表2・28に一般的に用いられている初期蒸気圧と許容圧力降下の関係を示す。c. 蒸気配管の摩擦損失線図圧力101.325kPa、温度100°Cの飽和蒸気を基準とし、種々の圧力における蒸気の流量、管径と管長1m当たりの摩擦損失、および管内流速を求める線図(流量線図ともいう)を図2・33に示す。図2・33の使用法は図中に点線で示してある。ただし、管材は配管用炭素鋼鋼管(JIS G 3452:1962)および配管用アーク溶接炭素鋼鋼管(JIS G 3457:1962、管呼び径350A 以上、肉厚7.9mm)とし、内面の絶対粗さε=0.045mmとしている。
