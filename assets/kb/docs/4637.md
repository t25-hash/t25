# 4・6・3 感覚機能

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a.視覚（vision) 人とかかわるロボットにおいては，人と同
様の感覚機能，特に視聴覚情報が重要となる．ATR知能ロボティ
クス研究所で開発されたロボットRobovieでは，人を発見するた
めの全方位カメラと，人の顔を発見するための通常のカメラを組合
わせている．
b.聴覚（audition) 人とかかわるロボットの聴覚処理におい
て重要なのは，言語情報を抽出する音声認識のみではない．ロボッ
トは実空間で活動するためにセンサ情報処理上の問題を多く抱えて
おり，音に関する信号レベルでの処理がむしろ重要になる．例え
ば，ロボットから少し離れた位置から話者が発話するだけでも従来
以上に背景雑音の影響が大きくなるため，既存の音声認識ソフトウ
ェアをそのまま動作させても正しく動かない場合が多い．ロボット
聴覚には2本以上のマイクロホンが必要とされるが，複数のマイク
ロホンを利用して音源方向や雑音処理を行うことも，実空間ロボッ
トに必要となる聴覚処理である．また，聴覚処理から抽出されるこ
とが期待される結果も，音声認識から得られるような言語情報のみ
でなく，音源方向のような空間性をもった信号レベルの情報や，音
の高低などの非言語情報も重要となる．
c .触覚（tactile sensing) 工場など限られた環境で活動する
産業用ロボットには，ロボットの活動のための安全領域が確保され
ており，周囲の人間とは安全のため直接触れ合わない距離がとられ
る．しかし，人間とのコミュニケーションに主眼をおくロボットの
場合は，人間と物理的に接触する可能性が非常に高くなる．そのた
め，ロボット表面の柔らかさと敏感さは，コミュニケーションの相
手となる人間への安全性という面において重要であり，産業用ロボ
ットではあまり考えられていない人間との接し方に対する考慮が必
要となる．またそれだけではなく，スキンシップという言葉どお
り，体の表面は最初に触れるインタフェースであることから，コミ
ュニケーションの相手がどのようにどの場所を触れているかを知る
ことは相手の状態を知ることにつながる．
ATR知能ロボティクス研究所で開発した人とかかわるロボット
RobovieⅡS［図4･135(ｂ)］は柔らかく敏感な全身触覚をもって
いる．RobovieⅡSでは，この全身皮膚感覚と多感覚とのマッピン
グとして，三次元動作計測システムによるコミュニケーション相手
の位置・姿勢とのマッピングを行っている．マッピングの方法は，
被験者にRobovieⅡSとコミュニケーションをとってもらい，そ
のときの皮膚センサ出力と，三次元動作計測システムによって計測
されたロボットの動作と被験者の動作を同期させて記録する．その
後，記録した皮膚センサ出力をISO- DATA法等によってクラスタ
リングし，各クラスタに含まれている皮膚センサ出力に対応するコ
ミュニケーション相手のロボット座標系での位置・姿勢の分布を解
析する．この分布がある範囲にまとまっているときは，そのクラス
タを代表する皮膚センサ出力に近い値が得られた場合に，相手の位
置・姿勢の推定を高い確率で行うことが可能となる．
