# 2・6・1 冷凍機・ヒートポンプ

> **章**: 第2章 空気調和と空調機器
> **節**: 2・6 空調機器

熱を低温側から高温側に運ぶのが冷凍サイクルであるが,このうち低温側の熱を利用して冷却する場合を冷凍機(refrigeratingmachine,refrigerator),高温側の熱を利用して暖房や給湯用などの加熱を行う場合をヒートポンプ(heat pump)というのが一般的である.広義には,低温側を利用する冷凍機も,ヒートポンプといっている.空調分野で使われている冷凍機は,住宅エアコン用の冷凍能力2kW 程度の小容量機から,大は地域冷暖房用の30MW くらいまで非常に幅が広い.表2・30に,冷凍サイクルや形式で分類したおもな空調用冷凍機の種類を示す.現在わが国の一般的な建物には,電動機駆動の蒸気圧縮冷凍機と都市ガスまたは灯油,A 重油を用いる吸収冷凍機が広く採用されている.特殊な空調用冷凍機として航空機用の空気サイクル冷凍機などもある.a. 蒸気圧縮冷凍機(vapor compression refrigeratingmachine)低圧部で冷媒が蒸発するときの冷却作用を冷凍機として利用,蒸発した冷媒ガスを圧縮して高圧部で常温で凝縮液化させ,これを再び低圧部で蒸発させる.冷媒ガスを機械的に圧縮して冷凍作用を行うものを蒸気圧縮冷凍機と呼んでいる.圧縮の方法によって容積型と遠心型(ターボ型)に分類される.容積型は,冷媒ガスを圧縮空間内に閉じ込め,動力によりその容空調方式駆動源冷凍方式圧縮方式冷却方式熱搬送冷凍容量機器冷媒ルームエアコン(往復動/ロータリ)ウインドウ型セパレート型個別式電気容積型水冷式・空冷式蒸気圧縮レシプロ冷凍機中スクリュー冷凍機中央式間接電気(蒸気・ガス)遠心型ターボ冷凍機(モータ駆動,タービン駆動)積を小さくして圧縮するもので,圧縮の作動機構から往復動式と回転式に分けられ,回転式はさらにロータリ型,スクロール型,スクリュー型などに分けられる.往復動式は古くから使用されている代表的な冷凍機で,業務用空調機などに広く普及している.回転式は往復動式のピストンシリンダに相当する機構を回転運動で圧縮作用が得られるようにしたもので,高速回転に適し圧縮機本体の小型軽量化,低振動・低騒音化ができ,用途によっては往復動式に取って代わりつつある.遠心型は羽根車の回転によって生じる遠心力で冷媒ガスを圧縮するもので,高速回転が可能なことから大容量になっても重量,据付面積とも小さくて済み,大規模なビル空調,地域冷房に採用されている.遠心型は容積型に比べ容量制御が容易で制御範囲も広く,負荷変動に対して追従性が良い.容積型,遠心型ともに,ヒートポンプに利用されている.これらは,通常の冷凍機より圧縮比を高くして,温熱が使えるようにした3―49図2・33蒸気流量線図(ε=0.045mm)表2・30一般空調用冷凍機・ヒートポンプ冷媒直接小パッケージエアコン(往復動/ロータリ/スクロール)セパレート型(リモートコンデンサ型/コンデンシングユニット型)R machine)低圧部で冷媒が蒸発するときの冷却作用を冷凍機として利用し,蒸発した冷媒ガスを吸収溶液に吸収させる.この吸収溶液を高圧部で燃焼ガスなどの熱で加熱し,先ほど吸収した冷媒を蒸気として追い出し,溶液を再生するとともに,冷媒蒸気は常温で凝縮液化させ,これを再び低圧部に送って蒸発させる.この吸収溶液を用いた冷凍機を吸収冷凍機と呼んでいる.吸収溶液を熱で再生するわけであるが,再生方式により単効用,二重効用などのサイクルがあり,さらに,用いる熱の種類によっても種類が分かれ,表2・30のように多様化している.吸収冷凍機はビル全体を対象とするような中央式(セントラル)空調に多く用いられ,さらに地域冷房用にも採用されていて,どちらかというと中容量から大容量向きである.吸収冷凍機は,スクリュー冷凍機あるいは遠心冷凍機など中・大容量向きの蒸気圧縮冷凍機と比べて電力使用量が少なく,振動騒音が小さく,容3―50熱機器量制御範囲が広いなどの利点がある.吸収冷凍機に,燃焼ガスの熱で温水を暖める機能をもたせたものは吸収冷温水機(absorptionwater chiller-heater)と称し,ビルの暖房にボイラを不要としたことで,ビル空調に広く普及している.ただし,この暖房はいわゆるヒートポンプではなく,ボイラとしての機能を利用したものである.吸収冷凍機は熱を駆動源とすることが特徴であり,排温水,排ガスなどの排熱を利用する冷暖房装置として,コージェネレーションシステムに組込まれて利用されることが多くなっている.
