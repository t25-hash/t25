# 2・6・2 冷却塔(cooling tower)

> **章**: 第2章 空気調和と空調機器
> **節**: 2・6 空調機器

温水と周囲に多量にある空気とを直接接触させて、温水を冷却する装置を冷却塔または冷水塔という。温水が空気中で1%蒸発したとき、温水温度は約6°C低くなるという冷却効果がある。自然通風に頼らず送風機によって空気を強制的に流す冷却方法を用いている図2・34の強制通風冷却塔が、冷却効果が大きく性能も安定しており、安価でかつ小型軽量化が可能なため空調用に広く使用されている。冷却塔内の水と空気の流れ方向から、熱効率の良い向流型(図流型(図2・35)、また熱交換効率が悪くあまり使われない並流型や混合流型に分けることができる。これらのおもな構成は、電動機、送風機、エリミネータ、散水装置、充てん物、塔体および下部水槽からなっている。冷却塔に入った水は散水装置によって散布され、熱交換効率を良くするために設図2・34 向流型冷却塔図2・35 直交流型冷却塔表2・31 空調用ボイラ種類容量kg/h蒸気圧MPa温水温度°C効率%おもな用途鋳鉄製ボイラ100~5000~0.2~12085~90ビル用炉筒煙管ボイラ600~18000~1.7~16085~96ビル用、産業用貫流ボイラ100~60000.3~2.1―85~96ビル用、産業用けた充てん物の表面を水膜や水滴状態で流下する。充てん物を通過し冷却された水は下部水槽を介して外部に取出されて冷凍機の冷却水として供給されたのち、散水装置に戻り、循環再使用される。通常の冷却塔は、前述のように循環水と空気を直接接触させて水を冷却するもので、開放式冷却塔と呼ばれる。これらは、空気中のガスや汚れが溶け込んで冷凍機の性能低下や伝熱管の腐食を招くことがあり、水質管理や熱源機器の保守管理などに若干課題が残る。その対策として図2・36に示すように密閉コイル(熱交換器)の外面に水を散布し、その蒸発潜熱を利用してコイル内の水を冷却する密閉式冷却塔がある。充てん物と密閉コイルが置換わっただけで、他の機器は開放式冷却塔とほぼ同じである。密閉式は間接式の熱交換となり、開放式に比較して冷却塔のサイズや送風動力が大きくなってコストも増加するが、冷却水が汚れないことで保守管理の費用は軽減される。高い信頼性が要求されるクリーンルーム用の冷却塔などに採用されている。
