# 4・6・4 要素機能

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a.ロボットによる表現（representation by a robot ) 人とか
かわるロボットと従来のインタフェースとの大きな違いは，それが
実際に実空間に存在するかどうかにある．ロボットに実際に触れる
ことができ，また立体的にその存在を認識できることが，ロボット
の表現系の最大の特徴である．ロボットはスクリーンでは表現され
ない情報を提示するという特徴をもち，そのためにインタフェース
の操作とは違ったコミュニケーションが可能になる．
多くの人とかかわるロボットは，必要十分な情報を入力できるよ
うな指示やそのための特別なデバイス（例えばマウスやキーボー
ド)，文字や地図などの正確な情報を提示するためのディスプレイ
といった，従来のインタフェースが必ず備えていた通信手段を一部
あるいは全くもたない．むしろ擬人的あるいは動物に似せた外見や
表現手段を通じて，人間が他の人間や動物に対して行うような自然
なやりとりを促進する．すなわち人間が日常的なコミュニケーショ
ンで用いるのと同じように，音声や視覚，触覚，身振り，手振りを
用いて情報のやりとりを行うのである．もちろん，このようなロボ
ットの定義は，ロボットがディスプレイのような従来のGUI
（graphical user interface）同様の装置をもってはならないという
ことを意味するわけではない．これまでに用いられてきたさまざま
な情報表現手段に加えて，ロボットがロボットならではの表現能力
をもつことは，ロボットの可能性を大きく広げるものである．
b.状況依存性（situatedness ) 状況依存性はロボットの行動
設計の最も基礎となる概念であり，特定の状況を仮定することで効
率的で信頼性の高いセンシングが可能になる．特に人とかかわるロ
ボットにおいては，ロボットが適切な行動を能動的に行うことによ
って，ロボット自らが状況を限定することができる．例えば，ロボ
ットが人間と握手をするという行動を例にとると，ロボットが人に
握手を求めるという状況は，人がロボットの前方の一定距離にいる
状況であると考えられる．さらに，ロボットが人と握手を求め，人
のほうに手を差し伸べている（図4･137）という状況下において
は，ロボットの手先に接触があったことは，高い確率で握手が行わ
れたとみなせる．また，ロボットがひとたび握手をし終わった状況
を考えると，人はまだロボットの前にいる可能性が高いと考えられ
る．このように，状況を限定することで効率的なセンシングが実現
し，次に生じる状況が仮定され，そして次の行動への状況が限定さ
れるという状況依存性の連鎖が生じるのである．
c .ユーザ主導とロボット主導（user -initiative and robot initia-
tive) 状況依存性を利用する処理は，おもにロボット主導型の
対話である．ロボットが提供した話題や身体動作といった表現に対
して，人が何らかの反応をすることを期待し，人の反応に応じてロ
ボットが次の表現を行う．これに対して，ロボットにはユーザ主導
型の対話への対応も求められる．例えば，ロボットが先に行った
「何かお手伝いしましょうか」といった発話に対して，人が「新聞
をとってきて」と答えた場合には，これはロボット主導の対話であ
るが，もし人がお手伝いロボットに「新聞をとってきて」と任意の
状況で発話するのであれば，ユーザ主導型の対話である．日常的に
は人間は互いに主導権をとり合い，譲り合いながら対話を続けてい
る．
ただし，ロボットのセンサ処理能力は人間の任意の行動に対応で
きるほどには性能がよくない．現時点では，ロボット主導を前提と
したうえでロボットと人間の対話を実現することが現実的である．
d.視線（gazing) 人間同様ロボットの顔は，非言語情報の
かなりの部分を表現する．特に，意図伝達に関しては視線の果たす
役割が大きい．人間は他者の視線から意図を感じ，また視線によっ
て他者に意図を伝達する．また，目が合うことは会話のきっかけと
なる．逆に，コミュニケーションの意図がない場合には，あえて相
手と目を合わさないようにすることさえある．コミュニケーション
が始まった後も，他者がふと視線をそらして周囲のものを見ると，
つられて同じものを見たり，それを話題にしたりする．視線には，
このように注意の対象を共有する機能もあり，このような機能は共
同注意機構（joint attention）として発達心理学分野で研究が進め
られている．
e .顔の表情（facial expression) 視線以外にも，顔からは喜
怒哀楽といった表情に関する情報が伝達される．コンピュータグラ
フィックスでは徐々に自然な表情を作れるようになってきており，
ロボットに関しても，機械的な顔，人間似の顔の双方について，喜
怒哀楽の基本感情を自然に表出するために感情表現の研究が行われ
ている．
顔の表情に関しては，エクマン（Ekman）の定義したアクショ
ンユニット（action unit ）を用いた合成方法がよく知られている．
アクションユニットとは，人間の表情を観察する際に，どのような
観察者でも同じ表情に一意に分類できるように分類方法を定式化し
たFACS（facial action coding system）において用いられる44
個の動き（眉毛を上げる，唇を動かすなど）のことである．例え
ば，喜びはAU6（頰を上げる）＋AU12（唇の端を引き上げる）
＋AU26（あごを下げる）といった動きの組合せに対応するといわ
れる．近年，コンピュータグラフィックス分野において，逆にこの
分析方法の要素であるアクションユニットを用いて表情を合成する
ことが一つの一般的な方法となってきている．

f .ジェスチャ（gesture) 人間がコミュニケーションの際に
身体を動かす行為を広い意味でジェスチャと呼ぶ．なかでも身振り
手振りによる明示的な情報伝達行動は，典型的なジェスチャであ
る．例えば，（方向を示すために）指を指す，指でオーケーサイン
を作る，うなずく，握手のために手を差し出すといったものであ
る．喜多の分類 によれば，ジェスチャには以下のような特徴が
ある．
.自己指向性，他者指向性 ジェスチャは相手が見ている
ことを意図し，相手に情報が伝わることを意図して行われるものが
多い．一方で，独り言を言う際にも身体を動かし，誰も見ていない
ところでもガッツポーズを作るといった事例に見られるように，人
間の内部状態が自然と外側に表出することによって生じるジェスチ
ャもある．
.内容の表現，会話の流れの調節 ジェスチャは大きく，
二つの役割に分類が可能である．内容の表現は指差し，オーケーサ
インのように発話の内容を補足的に，あるいは発話の代わりに用い
られるものである．また，会話の流れの調整とは，発話が終わった
際に相手のほうをちらっと見て相手の発話を促すといったターンテ
ーキング（発話交替）に用いられるようなジェスチャである．
g.引込み（entrainment ) 人間どうしのコミュニケーション
において身体動作の同調現象が報告されているが，このように無意
識に生じる身体動作の引込み現象は引込み，またはエントレインメ
ント（entrainment ）とも呼ばれる．引込みは，ダイナミックシス
テムのアトラクタと類似の概念であるが，コミュニケーションにお
いては，互いの発話の韻律や身体動作が同期・同調的になる現象の
ことである．
岡山県立大学の渡辺らは，このような人間どうし，あるいは人間
とロボットとの身体動作の引込み現象を研究する中で，上半身人型
のヒューマノイドロボットInterRobot （インタロボット）を開発
した．InterRobotは，話者の発話音声に対して適切なタイミング
でうなずき，まばたき，身振り動作を同期的に行うメカニズムをも
ち，これによって身体動作のエントレインメントを引き起こし，遠
隔地間の人間どうしのコミュニケーションを支援する．
函館みらい大学の小野らは道案内場面という空間的情報伝達場面
での腕の動作の引込み現象を，ヒューマノイドを用いた実験によっ
て発見した．道案内をするロボットに適切なジェスチャを行わせる
と，聞き手が無意識に引き込まれて同調的な身体動作を行い，さら
に言語情報の理解も促進される．
h.模倣（imitation) ロボットの行動設計における設計方針
の一つは，既存の相互作用対象を模倣することである．例えば，イ
ヌのようにふるまう，ネコのように身勝手な，といった表現がされ
るように，ペット型ロボットはイヌやネコのように明確な模倣の対
象をもつことが多い．このアプローチの長所は，既存のモデルが存
在することで，トップダウンな設計によりロボットの開発ができる
ことである．
i .感情（emotion) 感情とは，喜怒哀楽といった人間の気持
ち，心的な内部状態のことを指すと考えられている．しかし，心理
学などの社会科学の分野においてさえも，感情の定義については議
論の分かれるところである．社会科学においては有機論と相互作用
論の二つの代表的な感情モデルに関する考え方が古くから議論され
てきた．生物学者のダーウィンなどが提唱した有機論においては，
感情を人間や動物が共通にもつ本能的な身振り手振りの一種であり
進化の過程で残ったものであると考えている．また，このような考
え方は，人種や文化にかかわらず普遍的に理解可能な感情が存在す
るという考え方につながる．有機論の中でも，感情表現（特に基本
的な顔表情）は生得的であるとする考え方（エクマンなど）と，言
語のように文化によって変化するという考え方がある．現在では，
基本的な表情は生得的であるが付随的な表情には文化差があるとい
う考え方が優勢である．一方，相互作用論では，人間の感情は本能
のみに基づくのではなく，コンテクストのような社会的要因の影響
をより受けやすいとの考えからモデル化が行われている．例えば，
社会学者のゴフマンは，感情は時に意識的に制御され，管理され，
抑制されると考えた．また，コミュニケーションの中で人は感情を
意図的に利用する場合もあり，近年では感情をコントロールするこ
ともサービス業の労働の一部となってきている．
顔表情は感情の表出したものであり，おもな顔表情は感情と1対
1対応の関係にあると考えられている．感情は顔表情以外にも発話
の韻律，ニュアンスの中にも表出する．顔表情に関してはエクマン
の6基本表情（幸せ，怒り，驚き，悲しみ，恐れ，嫌悪）がよく知
られている．
コミュニケーションロボットに関しても，すでに複数のロボット
が感情表現や内部に感情モデルをもつようになってきている．
MITの顔ロボットKismetは，基本9表情の表出（幸せ，怒り，
悲しみ，驚き，興味，退屈，恐れ，不機嫌，穏やか）を実現した．
また，早稲田大学のWE-3は基本6表情＋2表情（幸せ，怒り，驚
き，悲しみ，恐れ，嫌悪，酔っ払い，恥ずかしい）の表出が可能で
あり，喜び-活発-覚醒の三次元感情モデルをベースに，内部に用意
された情動モデルに従って，アルコールを近づけると喜び覚醒が下
がる，たたかれると怒り覚醒するといった感情モデル上での変化を
主体とした相互作用を行うように設計されている．
