# 2・6・3 ボイラ(boiler)

> **章**: 第2章 空気調和と空調機器
> **節**: 2・34 ),納まりが良く何台も並べて建物屋上などに設置できる直交

ボイラには,蒸発量3000t/hを超える火力発電所用から,蒸発量20kg/h程度の家庭用小型ボイラまで,容量幅が広い.構造的に見ると,空調用に用いられているボイラには表2・31のように多くの種類があり,その中でも図2・37の鋳鉄製ボイラ,図2・38の炉筒煙管ボイラおよび図2・39の貫流ボイラが広く使われている.ボイラは取出す熱媒により温水ボイラと蒸気ボイラの二つに大別されるが,空調用熱媒には温水と低圧蒸気が利用されている.ボイラは,燃焼室と対流伝熱面からなるボイラ本体と,燃焼装置,給水装置,水面計,圧力計などの付属品,安全弁や逃がし弁,図2・36 密閉式向流型冷却塔図2・37 鋳鉄製ボイラ水管ボイラ1000~750000.5~4.1~22085~96工場用,地域冷暖房用低水位や不着火などに対する安全装置および自動制御装置から構成される.燃焼室は,ガスや油を燃焼させる空間であるとともに,火炎の放射伝熱で熱を吸収する部分でもあり,その内壁はボイラ水で冷却される水冷壁構造をしたものが多い.対流伝熱面は,燃焼室を出た燃焼ガスとの接触によって熱を吸収するので,煙管(管内に燃焼ガスが流れ,管外面にボイラ水が接触),水管(管内にボイラ水が流れ,管外面に燃焼ガスが接触)または鋳物セクションで構成され,さらに空気予熱器や給水予熱器などの付属伝熱面を備え,排ガスから熱を回収するものもある.
