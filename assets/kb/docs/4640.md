# 4・6・6 人とかかわるロボットの諸問題

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a.認知科学研究（cognitive science) 人間の認知機構を計算
論的・構成論的に調べることを特徴とする認知科学では，提案する
モデルの検証はおもに計算機上でのシミュレーションによって行わ
れてきた．しかし，人間の認知機構を解明するためには，環境との
相互作用や身体との関連を明らかにすることが必要である．これら
について，従来の認知科学は実証的な研究手段をもたなかった．こ
のような認知科学において，ロボットは実世界において提案したモ
デルを検証するための有効な方法論を提供する．これら認知研究で
得られた成果はロボット研究へフィードバックされることになる．
b.不気味の谷（uncanny valley) ロボットの擬人化は，親
しみやすいインタフェースやロボットを作る際の最も単純で効果的
な開発方針である．非常に機械的な産業用ロボットよりは人間に少
し似たおもちゃやロボットに，人は親しみを感じるものである．し
かし，ロボットが人間に似れば似るほど人はその違いに敏感にな
り，ちょっとした違いを不気味に感じるようになる．
例えば，生きている人間に対しわれわれは不気味さは感じない
が，死体がそのまま起きて歩き出すと非常に不気味な感覚を覚える
であろうし，文楽人形も人間に似ているがゆえに，時におもちゃの
人形以上に不気味さをもつ．また，このような傾向は対象が動くこ
とによって際立つ．これは不気味の谷と呼ばれる．1970年に森に
よって提唱された不気味の谷 は，発表当時はそれほど注目され
なかったものの，近年コンピュータグラフィックスによる人間顔エ
ージェントやヒューマノイド，アンドロイドといった擬人的なイン
タフェースが研究されるにつれて注目を集めている．
c .メディアイクエーション（media equation) 擬人化のよ
うに人工物を人間的に取扱う現象は，表面的な外見についてのみ生
じるわけではない．スタンフォード大学のリーブス（Reeves ）と
ナス（Nass ）は，人間がコンピュータを扱う際にも，意図せずと
も対人的に，そして社会的にふるまってしまうことを多数の心理学
実験を通じて明らかにし，著書 の中に示している．例えば，人
はたとえコンピュータ相手であってもおせじを言われるとうれしく
思い，おせじを言うコンピュータに好意をもってしまう．また，自
画自賛するコンピュータよりは，他のコンピュータが賞賛するコン
ピュータのほうが，人は有能であるように感じる．このような知見
はそもそも従来のメディアを介したコミュニケーションに当てはま
ってきたものである．人は，テレビに映るニュースキャスターがま
るで自分の目前で話しているかのように，現実に生じた出来事のよ
うに感じる．そして，メディアイクエーションで紹介されるコンピ
ュータに関するこのような体系的な積上げは，人間がコンピュータ
を，これまでのテレビやラジオといったメディアと同様に取扱うこ
とを示してきた．
人とかかわるロボットについても，人間はロボットという新しい
メディアを，見掛けの人間との類似性（例えば，丸に二つの点があ
るだけで顔のように見える）だけでなく，内面やふるまいについて
も，無意識のうちに擬人的に扱う傾向があると考えられる．
d.身体性（embodiment ) 従来の人工知能に関する研究で
は，実世界と切り離されたコンピュータ上の知能を実現すれば，実
世界で動作する知能システムが実現すると考えられていた．これに
対して身体性とは，ロボットの研究においては，体をもっているが
ゆえに可能になることや，必要となる性質を広く指す．体をもつこ
とで，環境とのかかわり合いの中から知的な行動が生じるといった
考え方を意味する場合もある．
人とかかわるロボットにおいても，人間に似た身体をもっていな
くとも，移動したり体の向きを変えたりといった身体を利用した対
話を行うことが可能である．英語には身体性という語に直接対応す
る言葉はなく，embodiment ，body propertyといった用語が用い
られたりする．コンピュータ上の対話エージェントでも，手足や顔
といった身体をもつ場合にはembodied agentと呼ばれることがあ
る．
