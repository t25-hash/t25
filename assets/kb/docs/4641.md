# 2・6・5 自動制御と中央管制

> **章**: 第2章 空気調和と空調機器
> **節**: 2・34 ),納まりが良く何台も並べて建物屋上などに設置できる直交

建物の快適環境を目的とした保健空調や製品の品質向上を目的とした産業プロセス空調など,高効率で適正かつ安全に運転するため,空調設備の自動制御は不可欠である.空調設備は,空気調和機,ファンコイルユニット,変流量(VAV)ユニット,冷凍機,ボイラ,ポンプ,ファンなど多数の機器で構成されている.これら各機器は,最近では,マイクロコンピュータを搭載してインテリジェント化を図り,機器内部の多くの温度センサ,圧力センサなど,さまざまなセンサやスイッチ類からの入力を演算処理し,膨張弁,圧力調節弁などの調節弁,調節端を操作して運転制御,安全制御を図2・44 水冷パッケージ空調機図2・45 空気熱源ヒートポンプパッケージ空調機図2・46 ルームエアコン(スプリット型)行い,また機器の状態表示などを行っている.空調機器における自動制御の目的は下記のようになる.(1) 省力化人の技能,経験に頼ることなく,均質で高質な自動操作をし,省力化を図る.(2) 安全性の確保重大な支障をきたす前に異常を検出し,安全に機器を停止する.(3) 最適な運転状態の確保居住者(空調対象)にとって,快適(最適)な空間を提供し,また各機器を最適な状態で運転する.(4) 省エネルギー空調に要するエネルギーの低減を図る運転を行う.(5) 機器品質の維持機器の状態を把握あるいは記録し,メンテナンスを容易にする.これらの機器の自動制御は,基本的には自律的に行っているが,制御目標値を中央あるいは上位の制御装置から受取り,また状態値を返すなど,情報交換のできるものも多くなっている.各機器を中央管制と有機的に結合し,制御することによって,自動制御の効果を高めている.空気調和設備の中央管制(BA : building automa-tion)は,機器の運転状態監視,機器の故障監視,スケジュール運転,遠方からの運転/停止操作,室温の設定変更,機器間の連動運転,台数制御,メンテナンス情報の記録などの機能を備えていることが多い.また,さらに建物の空調設備のほかに,電気設備,給排水設備などを含めて一元的な運用管理を中央監視室あるいは中央監視盤で行う中央管制も,多くのビルで採用されだしている.ビルの中央管制を含むビル管理システム(BMS : building management system)により,室内空間の快適性を保持しつつ,省エネルギーを図り,またビル管理者業務を支援して省力,省人化を図り,設備運用管理の質の向上を図っている.電話回線やインタネットなどを使用し,中央管制の情報を機器メーカも共有したり,あるいは,中央管制経由で機器の情報を直接取出すことでメンテナンスの省力化を図るなどの取組みも始まっている.文献(1)
