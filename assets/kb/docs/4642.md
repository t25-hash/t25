# 3・1・1 冷凍の原理と冷凍サイクル

> **章**: 第3章 冷凍機器
> **節**: 3・1 冷凍の基礎

a. 冷凍の原理物質を環境温度以下に冷却する操作を冷凍といい、この操作を行うサイクルを冷凍サイクル(refrigeratingcycle)という。環境温度以上の物質を冷却するには環境にある空気や水を用いて冷却すればよいが、環境温度以下の低温を発生させるには、低温から熱を奪い高温の環境に熱を捨てなければならないので自然にはできない。そこで、外部より仕事を与えることによって低温部から高温部へと熱を移動させて冷凍する。高温の物質を低温にするには断熱膨張あるいは絞り膨張を用いるのが普通である。図3・1は高さ方向に温度をとった熱エネルギーの山にヒートポンプを模式的に表現した図である。エネルギーレベルの高い山頂にふもとから熱を移動させるためには、外部から仕事を与えなければできない。水を低いところから高い場所へ送るには電気で動くポンプを用いなければならないのと同様である。このCOPは1を超える場合が多いので、効率とはいわない。図3・2は環境温度を27°Cとするときの最高冷房COPと最高暖房COPの値を示している。環境温度とは、冷房、暖房ともに室外温度のことである。各種冷凍サイクルの構成を表3・1に示す。冷凍能力の単位はkW のほかに、0°Cの水1000kg を24時間で0°Cの氷にする能力=日本冷凍トンJRt(=3.86kW)とアメリカ冷凍トンUSRt(=3.52kW)がある。b. 蒸気圧縮冷凍サイクル(vapor compression refrigeratingcycle)作動流体[冷媒(refrigerant)という]の相変化を利用すると、等圧変化は等温変化に等しくなり、逆カルノーサイクルに近いサイクルを実現することができ、サイクル性能が高くなるので、液化機など極低温機器を除いて冷蔵庫やエアコンディショナ(エアコン)など広く一般的に利用されている。電気モータまたはエンジンにより圧縮機が駆動され、凝縮器、膨張弁、蒸発器の順に作動流体が循環する。飽和蒸気または過熱蒸気(状態1′または1)が圧縮機(compressor)に吸入され、可逆断熱圧縮されて高圧の過熱蒸気となる(状態2′または2)。次に、凝縮器(condenser)において等圧的に冷却されて凝縮し飽和液または過冷却液となる(状態3′または3)。膨張弁(expansion valve)を通過するときに絞り膨張をして低圧で低温の湿り蒸気となる(状態4′または4)。最後に、蒸発器(evapora-tor)において等圧的に加熱されて蒸発し蒸気に戻る。蒸発器では低温熱源から熱を吸収する。適当な過冷却度を付けることによってCOPを向上させることができる。実際の冷凍サイクルでは、圧縮機内の変化が可逆断熱変化でないこと、熱交換器の伝熱抵抗のために凝縮器や蒸発器で外部流体と冷媒の間に温度差が生じること、弁や熱交換器内の圧力損失などにより図3・5のようなサイクルになる。例として、二段圧縮一段膨張冷凍サイクルを図3・6,3・7に示す。. refrigeration)熱機関のブレイトンサイクルと同じ過程からなり、逆方向に変化させることによって冷凍効果を得るサイクルを空気冷凍サイクルという。作動物質は空気で、図3・10にサイクルの構成を、図3・11に温度-エントロピー線図を示す。低圧空気は圧縮機によって可逆断熱的に圧縮され(状態1→2)、高温高圧になった空気は等圧的に環境に熱を放出することにより冷却される(状態2→3)。次にタービンで可逆断熱膨張して低温低圧の空気となり(状態3→4)、この空気は低温熱源に放出され使用される。このように空気冷凍では低温空気を直接使用する開放システムが用いられる。空気冷凍機は、高圧空気がジェットエンジンから得られる航空旅客機のエアコンに用いられている(図3・12)。d. 吸収冷凍サイクル(absorption refrigerating cycle)蒸気圧縮冷凍サイクルでは電気で駆動される圧縮機によって低圧蒸気を高圧蒸気に変換しているが、この圧縮を熱エネルギーを用いて行うのが吸収冷凍サイクルである。図3・13は吸収冷凍サイクルの原理を蒸気圧縮式に似せて示している。蒸気圧縮式冷凍サイクルと同様に蒸発器や凝縮器を有しており、それらを通って冷却効果を得る物質を冷媒という。蒸気圧縮式では蒸気を圧縮するために圧縮機を用いているが、吸収冷凍サイクルでは冷媒蒸気を液に吸収させて液ポンプで昇圧している。この液ループを循環する物質を吸収剤(absorbent)という。吸収剤は吸収器において蒸発器にある冷媒蒸気を吸収する。吸収剤と冷媒の混合溶液は沸点上昇が起こり、吸収器(absorber)の温度は蒸発器の温度(3~4°C)より高くする図3・5
