# 3・2・1 種類と用途

> **章**: 第3章 冷凍機器
> **節**: 3・2 蒸気圧縮冷凍機

蒸気圧縮冷凍機（vapor compression refrigerating machine）
は冷凍サイクルにおいて，蒸発した冷媒蒸気を再び凝縮させるため
圧力を高める圧縮作用を圧縮機（compressor）により実現する冷
凍機である．現在広く使用されている圧縮機は圧縮方式の違いによ
り容積圧縮機（positive displacement compressor）と遠心圧縮機
（centrifugal compressor）に大分類され，容積圧縮機は，さらに
いくつかのタイプに分類される（表3･4)．
容積圧縮機は，圧縮空間内に閉じ込められた冷媒ガスを動力によ
り，その容積を小さくすることによって圧縮し，圧力を高めて圧送
する圧縮機である．この容積圧縮機は往復圧縮機と回転圧縮機に大
別される．
往復圧縮機（reciprocating compressor）は古くから使用されて
いる代表的な圧縮機であり，現在でも製氷・冷凍・冷蔵はもとより
各種産業用プロセス冷却から，業務用空調，電気冷蔵庫などの民生
用に至るまで用途は広い．
回転圧縮機（rotary compressor）はロータリ式，スクロール
式，スクリュー式に一般的に分けられる．ロータリ式は，圧縮機構
の羽根（ベーン）の取付方法により，回転ピストン式（ローリング
ピストン式）とロータリベーン式（スライディングベーン式）に分
類される．またスクリュー式はツインロータ式とシングルロータ式
に分類される．
回転圧縮機は往復圧縮機に比べ低騒音・低振動で，高効率の運転
ができること，高速回転に適していること，圧縮機本体が小型化で
きることなどの理由から，普及が拡大している．回転式のロータリ
圧縮機は，所要動力4.5kWまでの小型圧縮機としておもに家庭用
空調機（ルームエアコンディショナ)，パッケージエアコンディシ
ョナ，電気冷蔵庫や車載用空調機（カーエアコンディショナ）など
に用いられている．スクロール圧縮機は所要動力12kWまでの小
型中型圧縮機としておもに車載用空調機，パッケージエアコンディ
ショナや冷凍冷蔵装置などに使用される．スクリュー圧縮機は回転
圧縮機の中では大容量に属し，最大2000kWクラスまである．農
