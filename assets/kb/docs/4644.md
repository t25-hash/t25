# 3・2・1 種類と用途

> **章**: 第3章 冷凍機器
> **節**: 3・2 蒸気圧縮冷凍機

蒸気圧縮冷凍機(vapor compression refrigerating machine)は冷凍サイクルにおいて,蒸発した冷媒蒸気を再び凝縮させるため圧力を高める圧縮作用を圧縮機(compressor)により実現する冷凍機である.現在広く使用されている圧縮機は圧縮方式の違いにより容積圧縮機(positive displacement compressor)と遠心圧縮機(centrifugal compressor)に大分類され,容積圧縮機は,さらにいくつかのタイプに分類される(表3・4).容積圧縮機は,圧縮空間内に閉じ込められた冷媒ガスを動力により,その容積を小さくすることによって圧縮し,圧力を高めて圧送する圧縮機である.この容積圧縮機は往復圧縮機と回転圧縮機に大表3・3 compressor)はロータリ式,スクロール式,スクリュー式に一般的に分けられる.ロータリ式は,圧縮機構の羽根(ベーン)の取付方法により,回転ピストン式(ローリングピストン式)とロータリベーン式(スライディングベーン式)に分類される.またスクリュー式はツインロータ式とシングルロータ式に分類される.回転圧縮機は往復圧縮機に比べ低騒音・低振動で,高効率の運転ができること,高速回転に適していること,圧縮機本体が小型化できることなどの理由から,普及が拡大している.回転式のロータリ圧縮機は,所要動力4.5kW までの小型圧縮機としておもに家庭用空調機(ルームエアコンディショナ),パッケージエアコンディショナ,電気冷蔵庫や車載用空調機(カーエアコンディショナ)などに用いられている.スクロール圧縮機は所要動力12kW までの小型中型圧縮機としておもに車載用空調機,パッケージエアコンディショナや冷凍冷蔵装置などに使用される.スクリュー圧縮機は回転圧縮機の中では大容量に属し,最大2000kW compressor)は,インペラの回転によって生じる遠心力で冷媒ガスを圧縮するもので,おもな用途としては,中大規模建築物の空調用をはじめ地域冷暖房用・産業用大規模プロセス冷却用など幅広い.また圧縮機は,ハウジングの密閉方法により全密閉型・半密閉型・開放型に分類される.全密閉型圧縮機は,電動機と圧縮機構部が密閉ケース(分解不可能)内に収められており,漏れもなく,小型で低騒音であるため,冷蔵庫,ショーケース,ルームエアコンディショナ,パッケージエアコンディショナ,列車用空調機などで大量に使用されている.また,開放型圧縮機は別置きの駆動源により駆動されるため,回転軸からの漏れを防止する軸封装置が必要である.この開放型は自動車エンジンを駆動源とするカーエアコンディショナに大量に使われているほか,ガスエンジン駆動や蒸気タービン駆動のヒートポンプや冷凍装置・冷却装置に使用される.半密閉3―59開放0.4~120冷凍・空調カーエアコンディショナピストン式半密閉0.75~120冷凍・空調全密閉0.06~15電気冷蔵庫・冷凍・空調パッケージ型空調機全密閉0.06~4.5電気冷蔵庫・冷凍・空調家庭用空調機開放2~4カーエアコンディショナスクロール式全密閉0.5~12冷凍・空調パッケージ型空調機・家庭用空調機半密閉/開放2~2000冷凍・空調カーエアコンディショナ(開放)産業用・地域冷暖房(開放)半密閉/開放12~1100冷凍・空調半密閉/開放70~7000中大規模建物の空調(半密閉)産業用・地域冷暖房(開放)型圧縮機は,部分的に分解可能なハウジング形状をしており,全密閉型にするには量産効果が少ない大型の空調ユニット,ウォータチリングユニット,コンデンシングユニットを用いた低温冷凍装置に広く用いられている.
