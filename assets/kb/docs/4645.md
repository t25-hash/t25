# 3・2・2 圧縮機

> **章**: 第3章 冷凍機器
> **節**: 3・2 蒸気圧縮冷凍機

a. 往復圧縮機(reciprocating compressor)往復圧縮機では、シリンダ内をピストンが往復動して、ピストンが下がると冷媒ガスがシリンダ内に吸入され、ピストンが上がると冷媒ガスが圧縮されて送り出される(図3・18)。冷凍用圧縮機の原形であり、冷凍100年余の歴史とともに歩み、今日なお広く利用されている。構造上より、立て型、横型、多気筒型に分類されるが、横型は現在ほとんど用いられていない。現在、ハウジングの密封構造の異なる開放型、半密閉型(図3・19)と密閉型(図3・20)があり、幅広く使用されている。用途として開放型は冷凍、冷蔵、船舶用に、半密閉型は冷凍、パッケージエアコンディショナのようなユニット製品に、全密閉型は電気冷蔵庫、冷凍、パッケージエアコンディショナなどの製品に用3―60熱機器いられている。開放型圧縮機の小型の分野では、通常2シリンダの直列配列のものがフロン冷媒用として広く活用されている。多気筒型は所用動力7.5~120kW の範囲で、いわば中型の主流をなし、シリンダをV,W,VV 型に配列し、それぞれ4,6,8シリンダとなる。特別なものとして、W 型をくし型にした12シリンダといった例もある。半密閉型圧縮機(semi-hermetic compressor)は、その構造は開放型と共通しているが、圧縮機が電動機と一体化されて、小型、軽量、低コスト化が図られている。密閉型圧縮機(hermetically sealed compressor)では、電動機の周囲には冷媒蒸気が存在するため、冷媒は電気的絶縁性を有し、電気絶縁材料を劣化させないものが選定されている。また、密閉型往復圧縮機では、回転軸は垂直に配置されることが一般的である。b. ロータリ圧縮機(rotary compressor)回転式の代表的な容積圧縮機であり、シリンダ内に偏心して配置された回転ピストン(ローラ)と高低圧間を仕切る羽根(ベーン)によって圧縮作用を得る方式である(図3・21)。機構上寸法精度を必要とするが、高速化にも適した圧縮機構といえる。ロータリ圧縮機の特徴は回転部分の形状が単純で、軸系の不平衡力が小さく高速回転が容易であることから小型化ができ、振動も少ない。また、構造上吸入弁を必要としないうえ、往復圧縮機に避けることができないトップクリアランスが少ないことから体積効率の向上が図られ、これによる小型化の効果もある。構造が簡単であるため部品点数は少なく、量産効果が図3・18 全密閉型圧縮機の構造発揮できる。高低圧を仕切る羽根の取付方法により、回転ピストン型(またはローリングピストン型)と可動羽根型(またはスライディングベーン型)とに分類(表3・4)される。回転ピストン型には、1気筒(1シリンダタイプ)と2気筒(2シリンダタイプ)のものがある。2シリンダロータリ圧縮機は二つのシリンダ部を180°位相をずらして配置したもので、低振動・低騒音の特徴を有するため、インバータ駆動能力可変圧縮機として広く使用されている。用途としてはエアコンディショナ、電気冷蔵庫、パッケージエアコンディショナや冷凍機器など小型の分野で全密閉型のものが広く活用され、現在では所要動力4.5kW までの範囲に拡大している。容量制御は代表的なものとしてインバータ制御による圧縮機回転数制御が広く普及しており、2シリンダロータリ圧縮機は最小10Hz から最大180Hz までの広い回転数範囲での運転が可能である。ロータリ圧縮機は冷媒を直接シリンダに吸入するため、液バック運転を避ける必要があり、アキュムレータの取付けと冷媒封入量の厳守が、信頼性上特に重要である。c. スクロール圧縮機(scroll compressor)スクロール圧縮機は回転式の一種であり、渦巻状の形状をもつ固定スクロールと可動旋回スクロールを組合わせ、圧縮作用を得る方式である(図運転が可能なこと、高効率で可変速に適することが特徴である。このスクロール圧縮機は、1905年に発明されているが、実用化されたのは1980年初めになってからである。精密加工技術の進歩を背景に、1970年代に実用化に向けた研究開発が行われ、1980年初めにカーエアコンディショナ用スクロール圧縮機、パッケージエアコンディショナ用スクロール圧縮機の製品化が始まっている。スクロール圧縮機の用途としては、店舗や事務所用のパッケージエアコンディショナ、ルームエアコンディショナ、カーエアコンディショナ、列車クーラなどの空調機およびショーケース、低温用冷凍機、冷凍・冷蔵ユニットなどに使用されている。また最近では、ヘリウムガスを用いた冷凍機にも一部実用化されている。スクロール圧縮機の構造としては、カーエアコンディショナなどを除き、一般的に全密閉構造のものが用いられる。シェルの内部に吸込み圧力が作用する低圧式とシェルの内部に吐出し圧力が作用する高圧式の二つのタイプがあり、パッケージエアコンディショナ用としては低圧式、ルームエアコンディショナ用としては高圧式が一般的に使用されている。d. スクロール圧縮機の圧縮機構形をもつロータのかみあいによってガスを圧縮する(図3・23)。本来オイルレス圧縮機として開発され、これにロータ部への噴射注油機構を組込んでガスシール性を高め、冷凍用圧縮機への転用が1958年ごろ図られた。高速回転で大容量のガスが処理でき、摩耗部分も少なく、ロータの油シールで一段の圧力比が高くとれ、しかも液戻りの影響も受けにくいという利点をもつ。現在、開放型・半密閉型・密閉型、そして単段型・単機二段型などの形式があり、容量的には最大2000kW までのものが製造・販売されている。近年中型大型の分野で普及しており、比較的、凝縮圧力が高いか、または蒸発圧力が低くて、結果として高圧力比の運転が要求される場合に適している。さらに保守性と耐久性に優れていることなどの特徴があるため、ビルの空調、水産物・農産物・畜産物の凍結、冷蔵および化学薬品工業のプロセス分野まで幅広く使用されている。シングルロータ式スクリュー圧縮機(図3・24)は、一つの円筒型スクリューロータと二つの平板型ゲートロータを組合わせたもので、1960年にフランスのジンメルにより発明されている。この圧縮機構はツインロータ型に比べ、高圧部から低圧部への吹抜け部分がなく、かつシール面積が小さいため、漏れ損失が減少できる。特に上下二つの圧縮室を個別に制御することで部分負荷時の効率が向上できる。スクリュー圧縮機の容量制御はスライド弁方式をとり、100~10%の範囲で可能である。さらにスライド弁制御に回転数制御を組合わせることにより、圧縮機の省エネルギーと小型・軽量化を図ることが可能である。スクリュー圧縮機は上記のようにロータのかみあい部のシールが重要で、これを潤滑油で行っているために吐出しガス中の油分の除去が重要である。また、この油分離回収装置で分離された油の清浄管理が保守のうえでたいせつとなる。e. 遠心圧縮機(centrifugal compressor)遠心圧縮機は、インペラの回転によって生じる遠心力で冷媒ガスを圧縮するもので、高速回転が可能なことにより、大容量になっても重量・据付面積とも小さくて済む。一般空調用は、ほとんど密閉式単段型が採用されている。大容量のものは、インペラを二つにした二段型が使用される。開放式は産業プロセス冷却用に中容量のものが使われているが、大半は大容量のもので、地域冷暖房・大規模プラント用に採用されている。往復式に比べて容量制御が容易で、制御範囲も広く、負荷変動に対して追従性が良い。遠心冷凍機は遠心圧縮機を用いた冷凍機である。冷凍サイクルは容積式のものと変わらない。5~10°C程度の冷水から-75°Cまでのブラインを製造するために用いられ、0.35~35MW の容量(冷凍能力)のものが製作されているが、冷水用が多い。10MW 以上は地域冷暖房用で蒸気タービン駆動が多い。図3・25は冷水製造用に図3・23ツインロータ式スクリュー圧縮機の圧縮機構図3・24 シングルロータ式スクリュー圧縮機の圧縮機構3―61広く用いられている密閉単段遠心冷凍機の構造説明図である。ユニシェルとその上に設置されている増速歯車箱の両端に、それぞれ遠心圧縮機と電動機とを取付けたモータコンプレッサから構成されている。これらのほかに電動機冷却装置、潤滑装置、抽気冷媒回収装置、容量制御装置および安全装置などが付属している。駆動機は誘導電動機が用いられ、その冷却方法は容積式と異なり、冷媒液噴霧方式がとられている。容量制御には、圧縮機入口の可動サクションベーンを冷水出口温度によって開閉する方法が用いられている。年間運転をする工業用、地域冷暖房用のように、凝縮温度の最高と最低間の差が大きいときは、速度制御とサクションベーン制御とを組合わせて用いる。低温用には、遠心圧縮機の羽根車段数を増す。また凝縮温度と蒸発温度の差が大きくなると、動作係数が低下するので、段間にエコノマイザを設け、凝縮器から蒸発器に戻る中間で液冷媒を蒸発させ、残りの液冷媒の温度を低下させ冷凍効果を増大させている。蒸発した冷媒は圧縮機の中間段に吸引される。
