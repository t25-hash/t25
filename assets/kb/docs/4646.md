# 3・2・3 空調用ヒートポンプ

> **章**: 第3章 冷凍機器
> **節**: 3・22 ).回転中のトルク変動が少なく,スムーズな圧縮作用で静粛

a. 概要空調用ヒートポンプでは蒸気圧縮ヒートポンプが最も多用されている.蒸気圧縮ヒートポンプでは,蒸発器での吸熱に,圧縮機の圧縮動力を加えた熱量を凝縮器で放熱し,これを利用するので,ヒートポンプサイクルでの成績係数(COP: coefficientof performance)は,一般に高い成績係数が得られる蒸気圧縮冷凍サイクルでの冷房COPに比較して,理想的にはさらに1だけ大きくなり,燃焼式や電熱式の加熱・暖房機器に比較して省エネルギー型の加熱・暖房機器として発達した.空調では一般に冷房と暖房の両方の機能が要求される場合が多いので,空調用ヒートポンプは,暖房運転専用のヒートポンプよりは,冷房運転と暖房運転を切換えて運転する冷暖房兼用型のものが多い.空調用ヒートポンプは当初,冷房専用機をベースに開発されたが,1台の機器で冷暖房が可能なため,直膨式空調機(エアコン)では大部分が冷暖房兼用型のヒートポンプで,冷房専用は少ない.図3・26に直膨式空調機に占める冷暖房兼用ヒートポンプの割合の推移を示す.表3・5に蒸気圧縮式の空調用ヒートポンプの分類を示す.最も数の多い空調用ヒートポンプは,直膨式では大気を熱源とする空冷ヒ図3・25 直膨式の特徴蒸気圧縮冷凍機に多用されているフルオロカーボン系冷媒は作動状態での冷媒の圧力が高く,冷媒密度が比較的大きいので,蒸発器と圧縮機あるいは圧縮機と凝縮器の間の冷媒配管での圧力損失による能力や成績係数の低下が,水などの低圧・低密度の冷媒と比べて格段に小さい.したがって,これらの要素間の連絡配管を比較的長くできるので,圧縮機と蒸発器や凝縮器を分離して設置する,いわゆるセパレート型のヒートポンプの構成が可能になる.室内に設置した熱交換器に供給した冷媒の相変化により,冷温水を介さずに空調を行うため,直膨式(direct expan-sion system)と呼ばれる.室外機(熱源部分)と室内機(熱の利用部分)に分離することで,設置や機器形態の自由度が高くなることや,水の顕熱の温度差を熱搬送に用いる冷温水システムに比較し,冷媒の潜熱を利用するために同一循環量当たりの熱搬送能力が大きく,搬送動力の低減が可能なことが特徴である.さらに,複数台のヒートポンプを個別分散設置すれば,個別の運転停止や部分負荷運転が容易で,利便性や省エネルギー性に優れている.これらの理由から,住宅用や中小業務用の空調用途では,大部分がこの直膨個別分散式のヒートポンプエアコンである.冷房専用や一体型のものも含め,住宅用のものはルームエアコン,業務用のものはパッケージエアコンと呼ばれて多用されている.. ヒートポンプエアコンの構成図3・27に構成例を示す.パッケージエアコンは一体型から実用化され,圧縮機を室内機内にもち,室外機に熱交換器とファンのみをもつリモートコンデンサ型,圧縮機を室外機内にもつリモートコンデンシングユニット型の二つのタイプのセパレート型が順次開発されたが,現在の大部分のヒートポンプエアコンは室外機に圧縮機をもつリモートコンデンシングユニット型である.リモートコンデンシングユニット型では,圧縮機が室外機にあることで,室内機は空気熱交換器とファンのみになることから,室内機が小型軽量になり,室内機形態の自由度が高まるので,建物への設置性に優れている.住宅用では室内機を壁に設置する壁掛け型が大部分であり,業務用では建物や用途の多様性に合わせて,床置き型,壁掛け型,天井つり型,天井埋込み型など種類が多いが,天井面に半埋込み型で設置する天井カセット型の室内機が,天井面設置による省スペース,分散配置による快適性,メンテナンスの容易性などの利点が多く,最も多用されている.多数の室内機を設置する必要のある業務用建物では,室外機を室内機と同数設置するよりも,室外機を集合し,台数を少なくすることで室外機の設置スペースが削減できるマルチ型のヒートポンプエアコンが主流で,ビル用マルチと呼ばれ,1冷媒系統に数十台の室内機が接続可能である.. 応用例ビル用マルチのバリエーションとして,冷暖房を切換えて運転する方式のほかに,1冷媒系統に接続した複数の室内機で,冷房と暖房を同時に行う冷暖房同時運転型のビル用マルチや,夜間製氷を行い昼間の冷房運転に用いる氷蓄熱ビル用マルチなどがある.また,冷暖房機能だけでなく,外気取入れや加湿などの空調機能を実現するために,室内機と同じ冷媒系統に接続できる加湿機能付きの外気処理ユニットや全熱熱交換器との連動運転制御などが実用化されている.直膨式ヒートポンプとしては空気熱源のヒートポンプが多用されているが,図3・28に示す水熱源の熱回収型ヒートポンプもある.これは複数台の水熱源のヒートポンプを建物内に個別分散配置して水配管で冷却塔およびボイラなどの補助熱源と結合したシステムであり,おのおののヒートポンプは個別負荷に応じて冷房,暖房運転を行う.冷房と暖房負荷が同時に発生する場合には,冷房排熱を水回路を介して暖房運転のヒートポンプの吸熱源とする.冷房排熱が余った場合は冷却塔から大気に放熱し,暖房の熱源として不足の場合には補助熱源で加熱し,熱を補う.また,空調用以外での応用例では給湯ヒートポンプがある.特に,二酸化炭素を作動媒体としたヒートポンプは高圧側が臨界点を超えて運転するので,いわゆる凝縮過程がなく,高圧高温となった作動媒体は,等圧冷却過程で連続的に温度低下し,このときに被加熱流体に大きな温度変化が得られるので,低温の水を高温まで加熱するのに都合がよい.c. 空冷ヒートポンプチラー空調用の中小型の熱源機としては空冷ヒートポンプチラーが多く用いられており,圧縮機は低温外気から吸熱する高圧縮比のヒートポンプ運転でも高い成績係数を維持することができるスクロールまたはスクリュー圧縮機が主流である.. 水熱源ヒートポンプチラー河川水や海水などの熱源水が利用できる場合には水熱源ヒートポンプチラーが用いられる.ヒートポンプ運転時にも吸熱源の温度が外気ほどには下がらないために圧縮比は空気熱源ヒートポンプほど大きくはならず,大容量ではターボ式も多い..
