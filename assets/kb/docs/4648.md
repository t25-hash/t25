# 3・3・1 一重効用吸収冷凍機(single effect absorption chiller)

> **章**: 第3章 冷凍機器
> **節**: 3・3 吸収冷凍機

吸収冷凍機(absorption refrigerating machine)は蒸気や温水を熱駆動源として冷水を製造する機械である.最近の吸収冷凍機のほとんどが冷媒に水,吸収液に臭化リチウムを使用した水-臭化リチウム系のもので,この系の一重効用吸収冷凍機は基本的に図3・35に示すように吸収器,蒸発器,再生器,凝縮器の四つの熱交換器で構成され,一重効用(単効用)サイクル(single effectcycle)によって作動している.この冷凍機が一重効用サイクルを行うと,冷媒は蒸発器の管束内の冷水から蒸発潜熱を奪って蒸発するため,温度の下がった冷水が製造されることになる.この冷水は空調機などに接続されて冷房用の冷風を作り,温度が上がった冷水は再度蒸発器で冷やされて循環利用される.一方,蒸発器で蒸発した冷媒蒸気は,蒸発器と連なる吸収器内の濃度の濃い吸収液に吸収される.水蒸気を吸収し薄くなった吸収液(希溶液)は温度が上がり,吸収能力も低下するので,冷却水で冷却する必要がある.一方,希溶液は再生器に送られて,高温の蒸気や温水などによって加熱され沸騰状態になる.それによって,希溶液はいったん吸収した冷媒を分離蒸発して吸収能力が戻った高温の濃溶液となる.この高温濃溶液は再び吸収器に戻されるが,途中の溶液熱交換器で吸収器から送られてくる希溶液と熱交換し,温度が降下して吸収器に入る.他方,分離蒸発した冷媒蒸気は再生器と連なる凝縮器に入り,吸収器を経た冷却水を流している凝縮器内の管束表面で凝縮して冷媒液となり再び蒸発器に戻される.実用の一重効用吸収冷凍機の冷凍容量は32~7000kW と広い範囲にわたっている.一般的な一重効用サイクルの蒸発器と吸収器の圧力は約0.9kPaで,冷媒の蒸発温度は約4°Cとなり約7°Cの冷水を製造する.再生器と凝縮器圧力は約9kPaで,希溶液の蒸発分離させる動作温度は約90°Cとなる.したがって,この温度より高い加熱熱源が必要となる.実際には,0.3MPa以下の蒸気,100~150°Cの高温水や80~95°Cの熱源が利用されている.このように,すべての容器は低真空で作動していることが特徴である.四つの熱交換器においてそれぞれ性能改善技術が施されている.特に吸収器では管束表面での吸収液のぬれ性を確保するための散布方式がとられている.また,吸収器の最下部には不凝縮性ガスが集積するため,それを抽気するための開口部が設けられている.吸収器と蒸発器の間には液滴分離装置が設けられ,冷媒蒸気に水滴が同伴しないよう設置されている.同様に,再生器の上部には液滴分離装置を設けて,溶液の液滴が冷媒の循環系に混入しないようにしている.
