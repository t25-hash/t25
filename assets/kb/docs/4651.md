# 3・3・4 高効率化技術(improvement in efficiency)

> **章**: 第3章 冷凍機器
> **節**: 3・3 吸収冷凍機

現在商品化されている最も効率の高い二重効用吸収冷温水機への高効率化技術の適用例を図3・38に示す。上記の例も含め、吸収冷凍機の高効率化の手法を整理すると以下のように分類できる。a. 高効率サイクルの採用サイクルの高効率化としておもなものは、多重効用化、多元化が挙げられる。多重効用化とは、吸収サイクルで発生する凝縮熱(または吸収熱)を低温部再生器の加熱源として利用するなどの方法で熱をサイクル内でカスケード利用するものである。現在では、三重効用サイクルまでが試みられている。多元化は、多重効用化と同様の効果により、高効率化を図る方3―66熱機器図3・38 吸収冷凍機の高効率化技術法であるが、多重効用とは異なり、サイクルの数が増加する。(1) 多重効用化、多元化(2) 内部熱回収型高効率サイクル(GAX サイクル、AHEサイクル)b. 内部熱回収効率の向上サイクル内部の濃溶液および冷媒の保有熱を希溶液で回収することにより、効率を向上させることが可能となる。具体的手段を下記に示す。(1) 溶液熱交換器の温度効率向上(2) 冷媒ドレン熱交換器の追加(3) 溶液冷却吸収器の追加c. 吸収溶液循環量の減少による顕熱ロスの低減(吸収溶液濃度幅の拡大)吸収溶液循環量を減少させて、溶液が保有する顕熱のロスを低減することで効率が向上する。ここで、吸収溶液循環量の減少とは、希溶液と濃溶液の濃度幅を拡大させることを意味する。具体的手段を下記に示す。(1) 二段吸収・蒸発サイクルの採用(2) 吸収器・蒸発器の高性能化(3) 冷水大温度差化d. 排ガスの熱回収効率の向上および排熱の有効利用直火式吸収冷温水機に関しては、高温再生器の加熱に用いられた燃焼排ガスの保有熱を回収することにより効率を向上させる。また、コージェネレーション等からの排熱を有効に利用する要素を付加することにより、燃料の消費を削減することが可能となる。(1) 空気予熱器または排ガス溶液熱交換器の追加(2) 排熱回収用熱交換器または排熱回収用再生器の追加
