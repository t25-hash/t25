# 4・1・1 極低温冷媒

> **章**: 第4章 極低温機器
> **節**: 4・1 極低温技術

ヘリウムの定圧比熱を示すがλ線の温度で鋭いピークがある．比
極低温で最も一般的に使用される冷媒は液体ヘリウムと液体窒素
熱がピークをもつことは，λ線以下でエントロピーが急激に減少す
である．液体水素や液体酸素はロケットの推進剤として使われてい
る．表4･1に極低温で取扱う代表的な冷媒の物性値を示す．
大気圧において，酸素［（O），oxygen］は90.19Kで薄い青色
の液体に，窒素［（N），nitrogen］は77.31Kで無色透明な液体
となる．アルゴン（Ar)，ネオン（Ne）などの希ガスは液体空気
から窒素，酸素とともに分離される．窒素は63.17Kで固化し，
54.37Kで酸素も青みがかった固体になる．
水素［（H），hydrogen］は室温では75 %のオルト水素［（o-
H），ortho-hydrogen］と25%のパラ水素［（p-H），para-
hydrogen］の混合物でノーマル水素（n-H）と呼ばれる．o- H
は二つの核スピンが同じ方向であるのに対し，p-Hは二つの核ス
ピンが互いに逆向きである．液体水素温度を長期間保持すれば，オ
ルト・パラ変換（ortho-para conversion）により大部分の水素は
p-Hに変換される．平衡状態になったとき，p-Hの濃度は
99.79%で，沸点は20.28Kである．n- Hから平衡状態の水素に

なる際，669kJ / kgの熱が放出される．
ヘリウム［（He），helium］は4.222Kで液化する．ヘリウムは
冷媒として使用されるほかに，冷凍機の作動流体としても使用され
る．図4･1にヘリウムの状態図を示す．他の冷媒と異なり，ヘリ
ウムには固体，液体，気体の三つの相が共存する三重点が存在しな
い．また，固体ヘリウムは加圧しない限り存在しない．これは液体
ヘリウムでは量子力学の零点振動のために，固体のときのように原
子をある一定の位置の近傍にとどめるには外部からの圧力を加えな
くてはならないためである．
