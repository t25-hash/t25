# 4・1・1 極低温冷媒

> **章**: 第4章 極低温機器
> **節**: 4・1 極低温技術

極低温で最も一般的に使用される冷媒は液体ヘリウムと液体窒素である.液体水素や液体酸素はロケットの推進剤として使われている.表4・1に極低温で取扱う代表的な冷媒の物性値を示す.大気圧において,酸素[(O),oxygen]は90.19K で薄い青色の液体に,窒素[(N),nitrogen]は77.31K で無色透明な液体となる.アルゴン(Ar),ネオン(Ne)などの希ガスは液体空気から窒素,酸素とともに分離される.窒素は63.17K で固化し,54.37K で酸素も青みがかった固体になる.水素[(H),hydrogen]は室温では75%のオルト水素[(o-H),ortho-hydrogen]と25%のパラ水素[(p-H),para-hydrogen]の混合物でノーマル水素(n-H)と呼ばれる.o-Hは二つの核スピンが同じ方向であるのに対し,p-Hは二つの核スピンが互いに逆向きである.液体水素温度を長期間保持すれば,オルト・パラ変換(ortho-para conversion)により大部分の水素はp-Hに変換される.平衡状態になったとき,p-Hの濃度は99.79%で,沸点は20.28K である.n-Hから平衡状態の水素になる際,669kJ/kg の熱が放出される.ヘリウム[(He),helium]は4.222K で液化する.ヘリウムは冷媒として使用されるほかに,冷凍機の作動流体としても使用される.図4・1にヘリウムの状態図を示す.他の冷媒と異なり,ヘリウムには固体,液体,気体の三つの相が共存する三重点が存在しない.また,固体ヘリウムは加圧しない限り存在しない.これは液体ヘリウムでは量子力学の零点振動のために,固体のときのように原子をある一定の位置の近傍にとどめるには外部からの圧力を加えなくてはならないためである.図4・1の飽和蒸気圧曲線上の2.1768K の点はラムダ(λ)点(λpoint)と呼ばれ,ここから融解圧曲線上の1.7673K を結ぶλ線(λline)の右側をHeI,左側をHeIIと呼んでいる.同一物質で表4・1 代表的極低温冷媒の物性値*1:0.101325MPa,*2:300K から沸点(ガス)までのエンタルピー差2種類の液相が存在する.ここでの相転移は二次の相転移であり,潜熱は零であるが比熱,熱膨張率に不連続な飛躍がある.図4・2にヘリウムの定圧比熱を示すがλ線の温度で鋭いピークがある.比熱がピークをもつことは,λ線以下でエントロピーが急激に減少することに対応する.図4・1 ヘリウムの状態図図4・2
