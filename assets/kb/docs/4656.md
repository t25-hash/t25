# 4・3・1 ビデオカメラ（video camera）

> **章**: 第4章 入出力装置
> **節**: 4・3 映像入力機器

動き画や，映画などのように網膜残像を応用して動画を得るとい
う考え方は，電子的に動画を得る今日のテレビジョンシステムにも
反映されている．テレビジョンシステムの基本的な構成は，撮像と
受像であり，それらに関する研究は19世紀後半に始まった．撮像
に関する初期の研究例として，ニポーの円盤と呼ばれる機械式走査
のものが1884年に開発された．その約半世紀後の1931年に初めて
の全電子式の撮像デバイスとして解像管が出現した．解像管は，蓄
積動作が行われないために感度が悪いという欠点があったが，1933
年のアイコノスコープによる蓄積型の撮像管がこの問題点を解決し
た．蓄積型の撮像デバイスは一つ一つの画素に容量を設け，それぞ
れの容量に蓄積された信号電荷を走査ごとに一気に読み出せるよう
な構成になっている．固体撮像デバイスも含め最近の撮像デバイス
の主流は，この蓄積型である．1946年に，初期の白黒テレビ放送
用として，イメージオルシコンがRCA社により開発された．同じ
くRCA社により1948年に，三硫化アンチモンを光導電膜として
用いたビジコンが開発された．光導電膜は，光量に応じて電気抵抗
が変化する光導電効果を有し，現在の撮像管の多くは，この光導電
膜を用いた光導電型撮像管である ．
その後，光導電膜として種々の物質が検討された．1962年に，
光導電膜としてPbOを用いたプランビコンが，フィリップス社に
より開発された．このプランビコンは，放送局用の3管式カメラに
最も多く採用されている．日本においても，光導電膜にCdSeを用
いたカルニコンが1971年に東芝から，Se，As ，Teを用いたサチ
コンが1972年にNHK，日立製作所から，ZnSe，Zn，Cd，Teを
用いたニュービコンが1974年に松下電器産業からそれぞれ発表さ
れた ．
他の電子部品と同じように真空管の一種である撮像管の機能を固
体化しようとする試みは1960年ごろから活発になった．撮像デバ
イスのおもな機能として光電変換と走査がある．入射光を光電変換
する光電変換素子を多数配置し，各素子に発生した信号電荷をパネ
ル信号による走査で取出すXYアドレス型のホトスキャナ が
1963年にハネウェル社から発表された．1964年に，同様なXYア
ドレス型の撮像デバイスとして，IBM社によりスキャニスタ
が，RCA社によりTFT（thin-film field eﬀect transistor ) が開
発された．ホトスキャナとスキャニスタはシリコン単結晶を用いた
モノリシック型で，TFTは絶縁基板に真空蒸着を施した薄膜型で
あった．これらは，第一世代の撮像デバイスであるといえる ．
その後の半導体技術の進歩に伴い，1967年にウエックラーが
MOS（metal oxide semiconductor ）トランジスタのソース結合を
フォトダイオードとして利用するMOS型撮像デバイスの概念を発
表して以来，BBD（bucket brigade device) ，CCD（charge
coupled device) などの新しい機能デバイスの発表があった．
BBDは1969年にフィリップス社により発表され，CCDは1970年
にベル研究所により発表された．特に，光の入射により発生した電
荷をアナログ量としてSi - SiO界面に蓄積し転送できるCCDを撮
像デバイスとして利用する研究開発が活発になった．このころから
固体撮像デバイスおよびそれに関連した機能デバイスの開発は，第
二世代を迎えることになり，1971年にフィリップス社からCID
（charge injection device) の原理が紹介され，1973年にGE社
により実用化が発表された ．
第二世代の固体撮像デバイスの開発に関しては，日本の果たした
役割が大きい．VCR（video cassette recorder ）の急速な普及に伴
う撮像装置の小型化などの要求から，固体撮像デバイスを実用化す
ることが急務となったためである．1981年に固体撮像デバイスと
して初めて2/3インチ384画素（水平)×485画素（垂直）のMOS
型デバイスが実用化された．このころから固体撮像デバイスは，本
格的な実用化の段階を迎えることになる．1980年代前半は，メー
カ間の激しい競争のため急速に進歩したRAMの製造プロセス技
術をそのまま利用でき，量産性に優れたMOS型デバイスの実用化
が先行していた．しかし，MOS型は，信号線に多数のMOSトラ
ンジスタが接続されているため，寄生容量が大きく熱雑音のレベル
が高いことや，水平走査回路のばらつきによる固定パターン雑音の
発生が多いことなどから，1980年代後半から，高感度，高S/Nの
信号検出が可能なCCD型が固体撮像デバイスの主流となった ．
その他の固体撮像デバイスとして，MOS型とCCD型の特徴を
備えたCPD（charge priming device) が，1979年に松下電器産
業から，加速転送という手段を用いて水平転送CCDに効率よく信
号電荷を転送する加速転送MOS型デバイス が，1983年に日立
低コスト化の面で単板撮像装置より不利である．単板撮像装置は，
製作所から，高い開口率を得られることが特徴でおもに赤外用撮像
素子として利用されているCSD（charge sweep device) が，
1984年に三菱電機から発表された．その他にも種々のデバイスに
関する提案があったが，現在は上述のようにCCD型が主流であ
る．今後は，SIT（static induction transistor ) など増幅機
能を有するデバイスが，微細加工技術の進歩とともに有力となる可
能性がある ．
いままでに述べた固体撮像デバイスは，いずれも入射光の強弱に
応じて出力信号振幅が変化するだけで色に関する情報を直接得るこ
とができない．CCDなどを用いてカラー撮像装置を実現する際に
重要となるのがカラー撮像方式である．3板式，単板式などのカラ
ー撮像方式の開発によりカラー画像を得ることが可能になった．
以上のような歴史を経て，撮像技術はわれわれにとって身近なも
のとなり広く普及するに至った．
