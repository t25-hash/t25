# 4・1・2 極低温の断熱技術

> **章**: 第4章 極低温機器
> **節**: 4・1 極低温技術

液体ヘリウムや窒素などの極低温冷媒を貯蔵するデュワー(4・3・1項参照)および超伝導コイルなどを収納するクライオスタット(4・3・2項参照)に利用される断熱技術について述べる。始めに、高温面と低温面の間に存在する気体を介しての伝熱について考える。ここでは簡単にするため対流は考慮しない。図4・3に2面間の距離を変えずに気体の圧力のみを変化させた場合の、高温面から低温面への単位面積当たりの伝熱量(熱流束)を模式的に示す。気体の圧力が常圧あるいはそれ以上で、2面間の距離()が気体分子の平均自由行程(λ)に対して十分大きい場合、気体の熱伝導によって高温面から低温面に熱が移動する。気体の熱伝導率は圧力に依存せずほぼ一定で、熱流束は一定の値を示す。次に、気体の圧力を下げて2面間の距離が平均自由行程と同程度あるいはそれ以下になると、空間の状態は粘性流領域から分子流領域へと遷移する。平均自由行程が2面間距離に対して十分大きい状態(λ>10)まで圧力が下がると、分子熱伝導と呼ばれる伝熱形態に変わり、圧力の低下とともに、伝熱量は減少していく。この現象を利用して、極低温では、真空断熱(vacuum insulation)が使用されている。実際には、分子流領域で圧力を下げてもある圧力以下で再び伝熱量はほぼ一定の値を示すようになる。図4・4に、高温側300K、低温側77K で2面間の距離が12.7mmの場合の、真空度と平均熱伝導率の関係を示す。図4・4(a)は、単純に真空断熱のみを行った場合の結果である。圧力が0.1Torr程度以下では、粘性流領域から分子流領域に遷移し、圧力に比例して平均熱伝導率も減少しているが、10Torr程度より圧力が下がると再び、平均熱伝導率はほぼ一定の値を示すようになる。図4・4(b)は、分子熱伝導のみを考慮して気体の圧力に比例して平均熱伝導率を減少させた計算値である。(a)と(b)を比較すると、低圧力の領域では実際の伝熱量(a)は、分子熱伝導のみを考慮した場合(b)よりも大きくなっている。これは、低圧力領域では、2面間の伝熱で支配的な要因が、分子熱伝導による伝熱からふく射による伝熱に変わっているためであ図4・3 insulator)が使用される。多層断熱材は、おもにポリエステルフィルムにアルミニウムを蒸着して放射率を低減させたものを重ねて使用している。さらに、各アルミニウム蒸着フィルムの間にナイロン製スペーサを挿入したり、フィルムにしわやディンプルを加工、あるいは蒸着面を片面のみにするなどして、フィルム間の断熱を図る工夫が施されている。また、フィルムに直径2mm程度の穴をあけて真空排気特性を向上させたものもある。図4・4(c)は、2面間に50層の多層断熱材を挿入した場合の平均熱伝導率を示している。平均熱伝導率は真空度に対して図4・4(a)と同様の曲線を描いているが、真空度が高い(圧力が低い)場合、平均熱伝導率が10分の1程度低くなっている。多層断熱材によって熱侵入低減の効果が得られたことがわかる。多層断熱材によるふく射伝熱低減の効果は、理想的には層の積層に対して伝熱量は1/+1となるが、実際には、多層断熱材の挿入条件や施工方法によって著しく性能を落とし、期待された設計値が得られない場合がしばしば見受けられる。特に、層方向に圧縮力がかかった場合、部分的に層間の熱伝導が増して断熱性能を低下させてしまう。次に高温面と低温面を直接つなぐ固体による熱伝導について述べる。デュワーやクライオスタット(cryostat)では、シールド板や液体ヘリウム容器、超伝導磁石(superconducting magnet)などを支える荷重支持材、室温に置かれた電源から超伝導コイルに電流を供給するための電流リード、室温部とヘリウム容器をつなぐヘリウムの供給および回収配管などによって、高温部と低温部が直接つながれている。極低温で一般的に使用される材料の熱伝導率を図4・5に示す。材料によって傾向は異なるが、どの材料も温度によって熱伝導率が大きく変化することがわかる。したがって、熱伝導による熱侵入量を計算する際には、つねに使用される材料とその温度範囲に注意することが重要である。表4・2に材料の熱伝導積分値を示す。熱伝導による熱侵入の低減には、1熱伝導率の小さな材料を用い3―76表4・2
