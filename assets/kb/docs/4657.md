# 4・2・1 極低温冷凍機の概要

> **章**: 第4章 極低温機器
> **節**: 4・2 極低温冷凍機

極低温冷凍機(cryocooler)あるいは液化機は、一般に約120K以下の冷凍を行う。大型のものは、空気液化プラントや天然ガス、酸素、窒素、水素、ヘリウムなどの液化機として使用され、小型のものは超伝導磁石やクライオポンプに使用されている。極低温冷凍機には、その冷凍サイクルや構成機器によって、さまざまな種類のものがあるが、それぞれ使用される温度範囲やその温度での冷凍能力が異なっている。図4・6に各極低温冷凍機を冷凍温度と冷凍能力によって分類した図を示す。図4・6により、使用したい冷凍温度とその温度における熱負荷が決まると、適用可能な極低温冷凍機を知ることができる。極低温冷凍機の効率は一般の冷凍・空調機器と同様にCOP(coefficient of performance、成績係数)を用いて表される。COPは冷凍能力と入力の比で表され、室温付近の冷蔵庫、空調機などはほとんど1を超える値を示すが、極低温冷凍機の場合、冷凍温度が低いために非常に小さい値を示す。このように極低温冷凍機の場合、冷凍温度がわずかに異なるだけでその冷凍温度における理想的なCOPも大きく変化する。FOM の値を百倍して%表示にし、%カルノーとも呼ばれる。図4・8に実際の極低温冷凍機の効率を単位冷凍能力当たりの冷凍機入力(入力/冷凍能力=1/COP)を用いて示した。効率が高い冷凍機ほど単位冷凍能力当たりの冷凍機入力は小さい。また、各温度において効率に幅をもっているが、一般に冷凍能力の小さい冷凍機ほど低い効率を示す傾向となっている。
