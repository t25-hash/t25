# 4・2・2 小型極低温冷凍機

> **章**: 第4章 極低温機器
> **節**: 4・2 極低温冷凍機

a.小型極低温冷凍機の種類および基本的構造 小型極低温冷
凍機にはGM（Giﬀord-McMahon）冷凍機（GM refrigerator），
スターリング冷凍機（Stirling refrigerator），パルス管冷凍機
（pulse tube refrigerator）などがある．いずれも蓄冷器（regener-
ator）を使用することに特徴があり，これらは総称して蓄冷型冷凍
機（regenerative refrigerator）と呼ばれる．液体ヘリウム温度で
用いられる場合，上記冷凍機を予冷機とし，JT膨張機が用いられ
る．しかし，近年，GM冷凍機やパルス管冷凍機単体でヘリウム液
化が可能になったため，単体で用いられることが多くなった．それ
ゆえここでは蓄冷型冷凍機の種類と基本的構造について説明する．

と膨張機を一体構造にできるため小型化が可能になる．なお，20
K以上ではGM冷凍機より効率が高い．
