# 4・2・2 小型極低温冷凍機

> **章**: 第4章 極低温機器
> **節**: 4・2 極低温冷凍機

a. 小型極低温冷凍機の種類および基本的構造小型極低温冷凍機にはGM(Gifford-McMahon)冷凍機(GM refrigerator)、スターリング冷凍機(Stirling refrigerator)、パルス管冷凍機(pulse tube refrigerator)などがある。いずれも蓄冷器(regener-ator)を使用することに特徴があり、これらは総称して蓄冷型冷凍機(regenerative refrigerator)と呼ばれる。液体ヘリウム温度で用いられる場合、上記冷凍機を予冷機とし、JT 膨張機が用いられる。しかし、近年、GM 冷凍機やパルス管冷凍機単体でヘリウム液化が可能になったため、単体で用いられることが多くなった。それゆえここでは蓄冷型冷凍機の種類と基本的構造について説明する。図4・9はGM 冷凍機の構成を示した図である。圧縮機とそれにつながる吸気バルブと排気バルブの切換えにより、圧力変動を作り出す。一方、ディスプレーサは往復動し、膨張空間を体積変動させる。この体積変動により膨張空間でPV regenerator material)が冷凍能力に及ぼす影響蓄冷器の全エネルギー流束を小さくするには蓄冷器の熱交換の効率を向上する必要がある。蓄冷器の熱交換の効率は蓄冷材の比熱、伝熱面積に大きく依存する。特に液体ヘリウム温度付近になると、冷凍能力は蓄冷材の比熱に顕著に依存する。そこで、蓄冷材の比熱が冷凍能力に及ぼす影響について液体ヘリウム温度付近で動作するGM 冷凍機(4K-GM 冷凍機と呼ばれる)の計算結果を基に説明する。計算諸元および動作条件を表4・4に示す。図4・19は蓄冷材の比熱のみを仮想的に変化させて、到達温度に及ぼす影響を示したものである。比熱は温度に依存せず一定としたものである。比熱が0.063J/(K・cm)で到達温度は4.2K になり、大気圧のヘリウムを液化させるにはこれ以上の比熱が必要となる。ヘリウムのλ線温度(例えば1MPaでは2.068K)以下では膨張時に温度上昇が生じ、圧縮時に温度が低下する。圧力変化に対する温度変化が通常と逆になるので、現サイクルではλ線温度近傍が原理的に到達可能な最低温度になる。図4・20は比熱が冷凍能力に及ぼす影響を計算した結果で仮想的な比熱[0.05~1.5J/(K・cm)]をもつ蓄冷材と実際に使用されている蓄冷材(鉛、HoErRu)を用いた場合の計算結果を示している。比熱が大きいほど温度に対する冷凍能力のこう配が急になることがわかる。比熱が0.05~0.8J/(K・cm)までは比熱の増加とともに大幅に冷凍能力が向上するが、0.8J/(K・cm)以上では増加率は鈍る。低温で比熱を大きくするには限度があることを考え合わせると、比熱は0.8J/(K・cm)あれば十分であるといえる。
