# 4・2・2 小型極低温冷凍機

> **章**: 第4章 極低温機器
> **節**: 4・2 極低温冷凍機

a. 小型極低温冷凍機の種類および基本的構造小型極低温冷凍機にはGM(Gifford-McMahon)冷凍機(GM refrigerator)、スターリング冷凍機(Stirling refrigerator)、パルス管冷凍機(pulse tube refrigerator)などがある。いずれも蓄冷器(regener-ator)を使用することに特徴があり、これらは総称して蓄冷型冷凍機(regenerative refrigerator)と呼ばれる。液体ヘリウム温度で用いられる場合、上記冷凍機を予冷機とし、JT 膨張機が用いられる。しかし、近年、GM 冷凍機やパルス管冷凍機単体でヘリウム液化が可能になったため、単体で用いられることが多くなった。それゆえここでは蓄冷型冷凍機の種類と基本的構造について説明する。図4・9はGM 冷凍機の構成を示した図である。圧縮機とそれにつながる吸気バルブと排気バルブの切換えにより、圧力変動を作り出す。一方、ディスプレーサは往復動し、膨張空間を体積変動させる。この体積変動により膨張空間でPV 仕事が発生するように流速変動をコントロールする。PV
