# 4・3・1 デュワー

> **章**: 第4章 極低温機器
> **節**: 4・3 極低温応用機器

デュワー(dewar)は液体ヘリウムや液体窒素などの極低温液体をためておく真空断熱が施された容器をいう。その大きさは用途に応じて数Lから数千Lまでさまざまなものが使用されている。a. 断熱技術デュワーの断熱は上述のとおり真空断熱を基本とするが、そのほかにもいくつもの工夫がなされている。. 対流防止極低温液体を貯液する容器は広い表面積をもつことから、その空間を真空断熱することは固体断熱材を用いるよりも効果的である。製作時に真空断熱槽は十分な時間をかけて真空引きを行い、容器表面や多層断熱材に吸着されている残留ガス成分を脱ガスさせる必要がある。製作段階で真空処理を行っても少量の脱ガスが生じるので、真空槽内の低温部には吸着器が取付けられ、極低温の液体が充てんされた際に脱ガス成分を有効に吸着させるよう工夫されている例もある。. 伝導防止低温となる貯液部分は室温部のトップフランジから真空槽内につり下げられている。そこで室温部分からの熱侵入を減少させるために以下の対策を行っている。(1) 断面積を小さくする熱侵入量は断面積に比例するため、使用圧力等を考慮したうえで可能な限り薄肉化するとともに開口部を小さくする。(2) 伝導距離を長くする熱侵入量は伝導距離に反比例することから容器のつり下げ距離を可能な限り長くする。(3) 熱伝導率の低い材質を選定するステンレス、ガラス、高分子樹脂等の熱伝導率の低い材質の中から使用目的に沿って選定する。. ふく射防止室温部にある外筒から極低温部へのふく射による熱侵入量は構造材の放射率に関係し、温度の4乗の差に比例する(4・1・2項参照)。そのため真空槽内の表面は放射率を高めるため研磨されており、その結果、ステンレスを例にとると完全黒体に比べて侵入熱量を40分の1程度に減少できる。加えて多層断熱材を数十層巻くことでさらに熱侵入量の減少を図っている。また、液体ヘリウム用デュワーでは、常温の外側容器と極低温容器の間に液体窒素貯槽や蒸発ヘリウムガスによる低温シールド層を設けることでふく射熱の減少を図っている。b. 構造図4・28に2000L級の液体ヘリウムデュワーの断面図を示す。容器は縦型構造で主要部はステンレス製である。液体ヘリウムをためる極低温容器(1)は上部が熱侵入防止のためくびれたネック部(2)を構成し、その上部は真空槽(3)を貫通して大気に露3―84熱機器図4・28 2000L級の液体ヘリウムデュワー出している。トップフランジ(4)には配管接続口や安全弁が設けてある。容器の肩の部分には吸着剤(5)を配置し、内部の極低温ヘリウムで冷却されることで真空槽内の希薄な残留ガス分を吸着している。ネック部下部は後述するシールドガスにより外側から冷却しているほか、ネック全体はネック上部から蒸発ヘリウムガス(ボイルオフガス)を抽気(6)し、トップフランジで温められた高温のヘリウムガスが極低温部に流れることを防いでいる。ボイルオフガスは調節弁を経て回収系に排出される。さらにトップフランジからアルミ製のバッフル板(7)を数層配置することでふく射熱の極低温部への侵入とガスの対流を防止している。室温の外側容器(8)と極低温容器の間は真空断熱層を形成している。極低温容器の外側には銅製のシールド板(9)が配置される。その表面には同じく銅製のコイル(10)が巻かれ、極低温容器上部から導かれた低温の気化ガスが還流している。ガスはシールド板を冷却した後、ネック部上部の接続口を経て外部配管に接続されている。シールドガスは調節弁を経て回収系に排出される。極低温容器とシールド板にはふく射防止のため多層断熱材(11)が巻かれている。c. 用法デュワーは十分な真空引きを行ってから出荷されているが、それでも構造材からの微量な放出ガスは完全にはなくならない。したがって、極低温容器が常温にあるときは可能な限り真空引きを行うことが望ましい。その際、容器を外側から温めれば脱ガスが促進されてより効果的である。運転前には極低温容器内をヘリウムガスで十分に置換することが必要である。液体窒素シールドをもつデュワーでは液体ヘリウム注入前にシールド内に液体窒素を入れ、一昼夜ほど放置し、十分な予冷を行う。液体窒素シールドがないデュワーでは液体ヘリウム貯液後1箇月程度はシールド板が完全に低温とはならないため、シールドガス量およびボイルオフガス量の調節が必要である。気化したヘリウムのシールドガス:ボイルオフガスの分配割合は1:10程度であるが、液体ヘリウム注入直後は内部圧力に注意しながらシールドガスおよびボイルオフガスの量を調節する必要がある。
