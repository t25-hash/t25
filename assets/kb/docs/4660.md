# 4・4・2 CRT（cathode ray tube）

> **章**: 第4章 入出力装置
> **節**: 4・4 映像表示機器

CRTは1897年にブラウン（Braun,C.F. ）により発明されたと
されている ．テレビ受像機用のディスプレイとして1900年代後
半に特にカラーCRTとしての開発・改良・普及が進んだが，1970
年ごろからは文字信号を電子的に発生させるキャラクタジェネレー
タの開発とLSI化により，コンピュータ用のディスプレイとして
も多く用いられるようになった ．LCD，PDP，EL等の他方式が
開発されるまではほかに代替のない技術であったが，次第に特徴を
生かしての他方式とのすみ分けがなされるようになりつつある．
図4･13に一般的なカラーCRT（シャドウマスク型）の断面概念
図を示す．電子銃から発射された電子は軌跡を偏向ヨーク部で制御
され，蛍光面に到達して蛍光体を発光させる．赤（R），緑（G），
青（B）信号に対応する3本の電子ビームが発射され，蛍光体塗布
面に配列塗布されたR，G，B，3色の蛍光体に対してシャドウマ
スクにより選別されて到達することによって，カラー表示が行われ
る（RGB加法混色：色を混ぜるほど明度が上がって白に近づく）．
