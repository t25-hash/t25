# 5・2・1 マイクロ波加熱の原理

> **章**: 第5章 では電磁気を利用した直接的な物体の加熱・冷却機器のう
> **節**: 5・2 マイクロ波加熱

周波数(frequency)が数百MHz から数十GHzの電磁波を用いて食品などの誘電体を加熱することをマイクロ波加熱と呼んでいる.マイクロ波加熱はアメリカのマイクロ波装置メーカにより1945年に導波管の開口部に食品を置いて行ったのが始まりといわ3―85れている.その後,安全性や加熱効率向上のために,開放された空間で加熱するのではなく,家庭用電子レンジに見られるように密閉された金属容器の中へマイクロ波を導入して加熱する方式が考案され,今日に至っている.a. ISM 周波数帯マイクロ波加熱に使用される電磁波の周波数は,国際的に定められたISM(Industrial, Scientific and Medi-cal)周波数帯に割当てられており,915MHz±15MHz2450MHz±50MHz5.8GHz±75MHz22.125GHz±125MHzである.このうち915MHz帯はアメリカのみで,わが国ではISMとして割当てられていないため,この周波数帯を用いてのマイクロ波加熱は電波法により厳重に規制されている.そこで,わが国では一般にマイクロ波加熱で使用される周波数は2450MHz である.b. マグネトロンマイクロ波加熱に用いられる発振管としては,マグネトロン(magnetron)とクライストロンがあるが,比較的発振効率が高く負荷の変動に対して安定しているマグネトロンが用いられている.マグネトロンは二極真空管で電子を放出する円筒形の陰極とこれを取巻くように配置された陽極で構成されている.陽極は多数の陽極片(ベイン)で共振器を構成し,磁場の作用により陰極から放出された電子が陽極との間で円形軌道を描くような電子流を発生させて速度変調を行い,電子流の粗密波から発振エネルギーを得るようにしたものである.図5・1はマイクロ波加熱用に用いられているマグネトロンで,表5・1にその特性,仕様を示している.電子レンジなどの比較的低出力の民生機器ではマイクロ波出力が500~1000W,工業用マイクロ波加熱装置では1.5~7.5kW マイクロ波加熱装置マイクロ波加熱装置の一般的な構成を図5・3に示す.アプリケータと発振器であるマグネトロン間はマイクロ波電力を伝送するための矩形型の導波管で結合され,この間には,アイソレータ,パワーモニタ,チューナ(整合器)が設けられている.家庭用電子レンジではマイクロ波出力電力が1kW 以下と比較的小さいので,図5・3に示すアイソレータ,パワーモニタなどは省略されている.アイソレータはアプリケータからの反射波を吸収してマグネトロンを保護するためのものであり,パワーモニタは方向性結合器(directional coupler)によりアプリケータに伝送されるマイクロ波電力を入射電力と反射電力に分離して検出・表示するものである.また,チューナはマグネトロンとアプリケータ内の負荷との整合をとるためのもので,EH(イーエッチ)チューナや3(スリー)スタブチューナなどがおもに用いられる.また,アプリケータの形状は,電子レンジなどに見られるようなバッチ式が一般的である.アプリケータ内には均一加熱を目的に金属製のかくはん羽(スターラ)や回転皿(ターンテーブル)を設け,供給されたマイクロ波をかくはんしたり,被加熱物の位置替えを行うことにより,むらなく加熱できるよう工夫されている.また,被加熱物を連続して加熱したい場合は連続式(コンベヤ式)等もある.図5・4にバッチ式およびコンベヤ式のマイクロ波加熱装置の一例を示した.両方式とも被加熱物を出し入れする扉部分には,チョーク構造などの電波遮へい機構を設け,マイクロ波がアプリケータの外部に漏えいしないような構造となっている.図5・3 11である.日本国内では電波法および電気用品取締法で規制されている.電波法や電気用品取締法では,占有周波数帯幅に含まれる周波数規格値,不要放射による電界強度の周波数別での規格値,また,漏えい電力量の電力密度の規格値などが細かく決められている.したがって,マイクロ波加熱装置では,通信機器や人体への影響などを考慮して定められた電波法や電気用品取締法で規制された値を満足する必要がある.g. マイクロ波加熱の特徴マイクロ波加熱は従来の電気ヒータやガスなどの外部加熱と異なり誘電体損失による内部加熱であるためいくつかの特徴をもっている.(1) 短時間,高効率加熱従来の外部加熱は,一般に熱伝導性の悪い被加熱物(誘電体)は周りから加えた熱が内部に伝導するのに時間を要するので加熱時間が長くなるが,マイクロ波加熱では,被加熱物自体が発熱するので電力密度を大きくとることにより,急速で高効率な加熱ができる.(2) 選択加熱被加熱物はそれぞれに異なる損失係数をもっているので,この差によりマイクロ波を吸収しやすいものを選択的に速く加熱することができる.(3) 温度制御が簡単電源部の簡単な制御により,容易に加熱時間を決められるので温度調整ができる.(4) 作業環境周囲の雰囲気温度を上げることがないので,加熱時の環境が悪化しない.
