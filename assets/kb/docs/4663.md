# 5・2・2 マイクロ波加熱を利用した機器

> **章**: 第5章 加熱・冷却機器
> **節**: 5・2 マイクロ波加熱

マイクロ波加熱を応用した機器はその利点を生かし、家庭用から工業用と幅広くある。身近なマイクロ波加熱装置としては家庭用電子レンジがある。電子レンジはそのマイクロ波出力が1kW くらいであるが、工業用マイクロ波加熱装置では数十kW、なかには数百kW と大電力のものが多い。また、医療分野では比較的低電力のマイクロ波電力(100~200W)が用いられ、治療や手術用マイクロ波加熱装置が開発されている。マイクロ波加熱の利用分野は食品、化学、医療、印刷、窯業・セラミックス、木材・紙工業、繊維工業分野と多分野にわたり、それぞれに専用のマイクロ波加熱装置が開発され実用化されている。ここでは食品、化学、医療分野でのマイクロ波加熱装置の例を示す。a. 食品分野食品の調理では電子レンジ(microwave oven)がマイクロ波加熱装置の代表である。電子レンジは家庭で主として食品の加熱調理に利用されているが、電気ヒータを併用したもの(オーブンレンジ)が日本国内では主流となっている。図5・5に家庭用電子レンジ(オーブンレンジ)を示す。工業用としては、食品の熟成、殺菌、防かび、スナック食品などの膨化乾燥、茶葉の乾燥用マイクロ波加熱装置がある。殺菌や防かびではアプリケータ内の圧力を上げることにより沸点を上昇させて、高温菌を死滅させる加圧式マイクロ波加熱装置がある。また、茶葉の乾燥用マイクロ波加熱装置では、従来の外部加熱方式と比較して、茶葉の葉緑素の安定化が図られ長期保存による変色がなく甘味を引出すことができる。さらにはビタミンCの残存率量が高いので広く実用化されている。茶葉の乾燥用マイクロ波加熱装置は、一般的にはコンベヤ式が多く利用され、連続的に茶葉を乾燥するようになっている。b. 化学分野化学分野では、顔料・染料の乾燥用、樹脂の発泡用マイクロ波加熱装置がある。特殊なものとしては、図5・6に示すようなマイクロ波化学合成反応装置がある。この装置は複数の化学薬品を混合し、これにマイクロ波を照射して化学反応を促進す3―87図5・5 家庭用電子レンジ図5・6 マイクロ波化学合成反応装置図5・7 マイクロ波メス図5・8 電極部るための装置で、従来の方式に比べて短時間処理が可能である。さらに生成された物質も新しい性質や機能をもったものが得られるなどの点で、近年、注目されているマイクロ波加熱装置である。c. 医療分野癌細胞は約43°Cに加温すると死滅することが実験的に確かめられている。そこで低電力のマイクロ波をホーンアンテナなどを利用して癌細胞に局部的に照射して加温し、癌細胞を死滅させるマイクロ波加熱装置(ハイパサーミア装置)が開発され、実用化されている。また、人体内の病変組織に電極を刺入れてマイクロ波を供給し、病変組織の止血凝固を図り、切除するマイクロ波メスもある。図5・7,5・8にマイクロ波メス本体装置と電極部を示した。マイクロ波出力電力は200W 以下のため装置本体と電極部までは同軸ケーブルが用いられている。
