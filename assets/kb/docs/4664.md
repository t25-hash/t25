# 5・3・1 インダクション加熱の原理

> **章**: 第5章 加熱・冷却機器
> **節**: 5・3 インダクション加熱

インダクション加熱(induction heating)は1831年に発表されたファラデーの電磁誘導によるエネルギー変換を基本としている。磁界中にある金属製の被加熱物において、その磁界を変化させることにより金属内部に誘導電流を発生させ、その誘導電流によって発生する金属内部でのジュール熱により加熱を行う加熱方式である。図5・9にインダクション加熱の加熱原理のモデルを示す。ここでは一般家庭用に使用されているIH(induction heater)調理器をモデルにその原理を説明する。銅線を渦巻き状に巻回した磁力発生コイルに交流電流を供給して交流磁界の磁力線を発生させる。この発生した磁力線は、トッププレートを介して磁力発生コイルと磁気結合した被加熱物である鍋底に吸収されることになる。磁力は交流であるため変動し、この変動分をキャンセルする磁界を誘導する誘導電流が鍋底に誘起される(電磁誘導)。これは鍋底を渦を巻くように流れるため渦電流と呼ばれる。この渦電流は金属導体中を流れるため、その経路での抵抗分による発熱がジュール熱として発生する。熱発生のパラメータは以下の式で表現される。発熱量は、=π・욦・・d(5・5)욦:表皮抵抗(Ω):径方向の電流(A):半径(m)ここで욦は金属表面の表皮抵抗のことで、3―88熱機器욦=1σδ(5・6)σ:導電率(Ω・m)δ:表皮深さ(m)により求められる。また、δ(表皮深さ)は導体表面に電流が集中する度合いを示すもので、δ=ωμσ(5・7)δ=πμσ(5・8)ω:角周波数(s)(2π,:周波数)μ:透磁率(N・A)により求められ、周波数が高くなるほど、小さくなる。式(5・8)からわかるように、誘導加熱は発生磁束の周波数が高いほど、表皮深さが浅くなるため表皮抵抗욦は大きくなる。したがって、より小さい電流で効率良く加熱することが可能になる。現在、インダクション加熱のシステムにおいて周波数は、周波数変換装置も含めたシステムとしての容量・効率などから数十kHzから100kHz 程度が一般的である。
