# 5・3・2 インダクション加熱を利用した機器

> **章**: 第5章 加熱・冷却機器
> **節**: 5・3 インダクション加熱

インダクション加熱は古くから,金属の溶解・焼入れ等において工業用として実用化されてきた.しかしながら,一般加熱用の機器として広く使用されるようになるのは1970年代以降のこととなる.これは旧来工業用として実用化されたインダクション加熱のシステムの多くが商用周波数(50Hz/60Hz)を用いるものが主流であったため,インダクション加熱するには非常に大きな電流容量の磁力発生コイルが必要で,システムとして大規模なものが必要であったためである.その後1960年以降に,サイリスタ・GTO(gateturn-offthyristor)など半導体スイッチング素子の進化とそれを応用した周波数変換装置(インバータ)に関する技術が急速に進化し,高周波化が容易に得られるようになるとともに,インダクション加熱のシステムも大幅に小型・低コスト化が進むことになる.現在一般家庭用で広く普及しているインダクション加熱調理器の原型は1972年アメリカで,そして1974年日本にて登場することになる.インダクション加熱を応用した機器としては一般家庭用として,図5・9加熱原理のモデル図5・10 ビルトインIH クッキングヒータ台所に組込むタイプの図5・10に示すようなビルトインIH クッキングヒータと,卓上での調理を対象にした一口タイプのIH 調理器がある.IH クッキングヒータはキッチンに組込まれるタイプで,ガスコンロと同等以上の高火力が求められるため,入力電圧200V,定格電力2~3kW のものが一般的である.また,卓上型ではテーブル上での団らん調理などを対象としているため,小型・薄型化が図られている.また,入力は手軽に使用するため,100V 入力で定格電力が1.2~1.4kW となっている.これら家庭用調理器のインダクション加熱としてのスペックは以下のとおりである.[定格電力]1.2~3.0kW[加熱効率]83~90%[動作周波数]20~40kHzインダクション加熱は鍋そのものを加熱するという加熱のしくみから,1直火のない安全性2電力から熱への高い変換効率と高火力3清掃性の良いトッププレートという優れた特徴があり,また,マイコンやセンサの進化に伴う機器の進化とともに今後ますます伸張するものと考えられる.
