# 5・4・2 ペルチェ素子の構造と機能

> **章**: 第5章 加熱・冷却機器
> **節**: 5・4 ペルチェ効果を利用した冷却機器

a. ペルチェ素子の構造図5・11(d)に示すように、p型およびn型熱電半導体を銅などの電極を介して型にはんだなどで接合した接合対を熱電変換の基本的な機能をもつペルチェ素子(Peltier element)または熱電素子(thermoelectric element)という。ペルチェ素子は、通常、熱負荷や直流電源電圧および電流容量などの条件に応じて、型素子対を数個から数十個、多い場合には100個以上を並べ、電気的に直列に接続して一体化した状態で使われる。b. ペルチェ素子の種類ペルチェ素子のおもな種類を図5・12に示す。図5・12(a)はセラミックタイプと呼ばれ、平滑なセラミック基板が電極面に接合され一体化されている。セラミック基板は補強と電極の電気絶縁を兼ねているため、取扱いに便利である。セラミック基板にはアルミナ(AlO)や窒化アルミニウム(AlN)が用いられる。図5・12(b)はスケルトンタイプと呼ばれ、冷却や加熱による素子接合部の熱応力を緩和するために、片面もしくは両面のセラミック基板をなくしている。冷却対象の金属面との接触は、薄い電気絶縁層を介する必要がある。図5・12(c)は多段またはカスケードタイプと呼ばれ、一段モジュールでは希望する低温が表5・3熱電半導体の熱電能と物性値の一例(温度:300K)物質熱電能αV/K比抵抗ρΩ・m熱伝導率W/(m・K)BiTe(p型)1.0×101.7BiTe(n型)-2101.0×101.45SiGe(p型)1.14×104.90SiGe(n型)-1080.85×104.59(a) セラミックタイプ(b) スケルトンタイプ(c) 多段(カスケード)タイプ図5・12 ペルチェ素子の種類3―89得られない用途に対し、モジュールをピラミッド状に数段積み重ねたものである。それぞれの段で、下段は上段の発熱量を吸収し、さらに低温を維持するだけの十分な冷却能力をもつ必要がある。c. 冷却性能ペルチェ素子の冷却量Φ(W)、放熱量Φ(W)は、それぞれ以下の式で表される。電気抵抗と熱コンダクタンスは、次のように表される。前述の式(5・11)と式(5・12)において、右辺カッコ中の第1項は電流によって接合部で発生するペルチェ熱で吸熱・発熱ともに可逆的に生じる。第2項は電気抵抗によるジュール発熱であり、冷却面側と放熱面側に均等に分配される。第3項は熱伝導による放熱面から冷却面への熱移動量である。熱電冷却に消費されるエネルギー(電力)は、放熱量から冷却量を差引いた値となる。冷却の成績係数(coefficient ofperformance)は次式で定義される。一般に、素子対の形状(断面積、高さ)と、ペルチェ素子の電気抵抗、最大電流、最大冷却量、サイズ、耐久性などの特性には、表5・4の関係がある。d. 性能指数(figure of merit)熱電材料の性能を表すパラメータとして、性能指数(1/K)が次式で定義される。これらの関係から、ペルチェ素子の性能向上には、性能指数の大きい、すなわち熱電能αが大きく、電気抵抗と熱コンダクタンスが小さい熱電材料が必要である。近年量産されている熱電材料の性能指数は、常温で2.0~3.0×10K程度であり、さらに性能指数の高い熱電材料の誕生が待たれる。
