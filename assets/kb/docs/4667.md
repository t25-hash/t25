# 4・5・4 感熱・熱転写プリンタ

> **章**: 第4章 入出力装置
> **節**: 4・5 プリンタ（printer ）

a.感熱プリンタ（thermal printer ) 直接発色型の感熱プリ
ンタは，熱によって発色する感熱記録層を紙やフィルムの上に塗布
し，多数の抵抗発熱体ドットからなるサーマル印字ヘッドを紙に押
付けて感熱記録層を加熱することにより，画素を印画するものであ
る．通常は単色のみの印刷であるが，一部の用途で赤・黒2色発色
型も使われている．ヘッドには，薄膜技術によるものと圧膜印刷技
術を使ったものの2種類がある．またプリンタの機構には，シリア
ル型とライン型がある．数mm程度の小さいヘッドを用い，往復
操作させながら，1行ごとに記録するのがシリアル型である．低コ
d.画像処理技術 画像データを基に各色インクのドット配置
ストでできることから，パーソナル用途に多い．一方，印字幅（数
十mmから数百mm）分のヘッドを固定し，用紙の走査によって
記録するのがライン型である．1回の走査で1ページの印刷を行う
ため速度が早い．印刷サイズが大きいとヘッド製造が困難となって

ックス，バーコードプリンタ，券売機，計測器，CADプロッタ，
医療画像プリンタの出力などに用いられている．
b.熱転写プリンタ（thermal transfer printer ) 熱転写プリ
ンタは，インクリボン上のインクを発熱体を有するサーマルヘッド
の発熱を利用して記録紙に転写するものである．溶融熱転写でのレ
ジンインクの転写プロセスは大きく分けて図4･34に示すような4
工程からなる．サーマルヘッドにより加圧，加熱されたインクがヘ
ッドのエッジ位置でリボンをはく離することによって紙へ転写され
る．原色を重ねることによってカラー印刷が可能である．転写剤に
顔料を使う溶融型と昇華染料を使う昇華型がある．溶融型は，普通
紙から平滑なコート紙，合成紙，フィルムまで広く転写用受容紙と
して使用できるが，昇華型は記録紙の表面に必ず昇華染料を受容で
きる熱可塑性樹脂をコーティングした専用紙が必要となる．
