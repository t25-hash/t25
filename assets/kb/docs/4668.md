# 5・4・3 ペルチェ素子を応用した機器

> **章**: 第5章 加熱・冷却機器
> **節**: 5・4 ペルチェ効果を利用した冷却機器

a. 特徴と利点ペルチェ冷却・加熱は、固体素子中の電子の挙動によって生じるものであるから、可動部がなく、軽量・小型で、電流方向の切換えで冷却も加熱もでき、温度応答性が速いなどの特徴をもつ。そのため局部的な冷却や精密な温度制御が可能であり、メンテナンスも不要で、高い信頼性が期待できる。しかし、ペルチェ素子を冷却能力300W 以上の単なる冷却に用いることは、コンプレッサ式冷凍機に比べて成績係数の上から必ずしも得策ではない。ペルチェ素子の応用に際しては、軽量・小型による局所冷却や、精密な温度制御などの基本的な利点を積極的に生かせる用途を選ぶ必要がある。b.
