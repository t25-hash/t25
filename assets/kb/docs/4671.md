# 6・2・3 平面型ヒートパイプ

> **章**: 第6章 熱輸送デバイス
> **節**: 6・2 ヒートパイプの種類

ヒートパイプはその名が示しているように,一般的な形状としては管状(パイプ状)のものが広く用いられてきた.しかし,電子機器の冷却,特にCPU やトランジスタ等の半導体素子の冷却に用いる場合,限られた実装条件下で効率的な放熱を行おうとすると,従来の管状のものだけでは十分対応しきれない場合も増えてきた.そのため近年,応用が進んできたものに,いわゆる平面型ヒートパイプ(flat type heat pipe)がある.この平面型ヒートパイプは形態的な特徴と機能的な特徴の二つの特徴を有している.まず形態的な面からの特徴としては,実装上限られた空間内に設置するために,省スペースを目的として薄型化して平面をもつようにしたということが挙げられる.この場合,熱は一端から他端へ移動させるという機能が主となる.一方もう一つの特徴は,より積極的な目的で利用されている.すなわち平面であることにより,素子の取付けやヒートシンクとの接続がより容易でかつ接続による熱抵抗を減少させることができるということと,熱を広げ放熱機能も同時に兼ねてしまうという点が挙げられる.図6・11に,機能面の特徴を熱抵抗を用いて示す.図6・11には従来型と平面型の二つのヒートパイプを用いたヒートシンクの熱抵抗の違いを示している.平面型ヒートパイプの中でも比較的大きなものにおいては,発熱素子を直接的にヒートパイプに取付けられるために,熱源とヒートパイプ間での接触熱抵抗を小さくすることが可能となる.熱源近傍での接触熱抵抗は,熱密度(熱流束)の高い部分での接触であるために,接触部の相当厚さが同じでも,熱抵抗値は高くなってしまう.そのため,この部分での抵抗をなくす意味は大きい.また,放熱側に着目した場合でも,ヒートパイプから放熱フィンへの接触は同様に存在するが,設計次第で放熱フィンに合ったサイズのヒートパイプを作ることが可能となるため,その部分での接触熱抵抗を減少させることが可能となる.以上のようなメリットが平面型ヒートパイプにはある.次に,図6・12に製造方法に由来する各種平面型ヒートパイプの図6・8溝形状図6・9液体パドルの形成図6・10 bond)法が比較的大面積の平面型ヒートパイプを製造すを用いれば結果的に高性能化が可能となる.図6・11 平面型ヒートパイプの熱抵抗平面型ヒートパイプ銅製ヒートパイプ管扁平化法ろう付け法抽伸法アルミニウム製ヒートパイプ管扁平化法ろう付け法押出し法ロールボンド法図6・12 平面型ヒートパイプの種類図6・13 134aやR 123等が用いられるが,これらに加えて炭化水素(HC)系のもの,例えばシクロペンタン等も環境対策のために使用されるようになってきた.
