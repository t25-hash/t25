# 6・2・4 ベーパチャンバ

> **章**: 第6章 熱輸送デバイス

近年、平面型ヒートパイプの中でも特に、CPU 等の発熱密度の高い素子に対応したヒートパイプをベーパチャンバ(vapor cham-ber)と呼ぶ場合がある。この呼称は半導体業界で徐々に広まりつつあるが、発熱素子側の小さな面からヒートシンク側の大きな面へ熱を広げる熱密度の変換素子を呼んだものである。図6・15にベーパチャンバの使用イメージ図を示す。図6・15の場合、上面に対して偏った位置に熱源がある場合の実装例を示している。発熱素子から受取った熱は、伝熱ブロック等の熱伝導材を通して熱を作動液に伝え、そこで作動液は蒸発し、温度の低いほうに向かって蒸気が流れていくのである。そのため、部分的に温度が低いところがあると、そこに向かってたくさんの蒸気が二次元的に移動し、結果的には上面部分の温度が全面にわたって均一となっていくのである。特に、上面のヒートシンク側への空気の流れが不均一の場合、冷却能力が上面の中で分布があったとしても、冷却能力が高い部分に多くの蒸気が移動していくため、放熱効果を減少させることがない。単なる熱伝導による熱輸送の場合、離れた部分で局所冷却があると、そこまでの熱伝導で熱抵抗が増大してしまう。そのため、同じサイズのヒートシンクを用いたとしても、ベーパチャンバ(a) tunnel、風洞)と呼ばれるダクト内にはヒートシンクを設置し、前面風速(体積流量)を変化させて評価を行った。図の有無によって性能差が認められる。これは、風速が速くなるとベース部分の熱抵抗の違いが全体の性能に影響するためである。すなわち、熱源サイズが小さかったり(発熱密度が大きい)、放熱量が大きいためにヒートシンクが大型化する場合には、熱の移動距離が大きくなるのでベーパチャンバは効果的なものとなる。ヒートパイプは、従来の金属を利用した熱伝導体、ファンやポンプを用いた単相流体ループなどの熱輸送方法に比べ、(1)潜熱を利用するため大量の熱を輸送できる、(2)毛細管力や重力を利用するため可動部がなく、かつ無動力で熱輸送できる、(3)中空容器からなるため軽量で、熱容量が小さく熱応答性が良い、(4)蒸気流による熱拡散を行うため均熱化ができる、(5)無重力場でも使用できる(毛細管力利用の場合)、(6)温度の自動制御ができる[可変コンダクタンスヒートパイプ(VCHP: 発熱量の増加業用を中心に家庭用機器から宇宙用まで幅広い分野で利用されている。ヒートパイプの応用をその機能別に示したものを表6・2に示す。ヒートパイプは熱輸送、加熱、冷却、均熱化、熱制御などの機能をもつが、熱輸送機能を生かしたものとして代表的なものは排熱回収用熱交換器(6・3・5項)であろう。最近では、ループ型ヒートパイプ(LHP)のフレキシブル性を利用して人工衛星用展開型ラジエータへの応用が進められている(6・4・2項)。加熱機能の代表例としては路面や屋根の融雪がある。厨房機器への応用もあるが、ヒートパイプの場合は、温度が上がりすぎると作動流体の圧力が上昇して爆発の危険性があり注意を要する。冷却機能としてのヒートパイプの応用が最も多く、電子素子の冷却(6・3・1項)やヒートシンク(7・4節)など産業分野で数多く利用されている。均熱化機能としては均熱板(6・3・4項)や均熱ロール(6・3・6項)への応用が見られる。温度制御機能が利用されている例としては、人工衛星用ラジエータパネルにVCHPが一部利用されている程度で、まだ十分にその機能が利用されている状況ではない。ヒートパイプの利用温度は常温近傍が多いものの、融雪や高温炉など低温から高温まで広い範囲にわたっている。ヒートパイプ形状も円筒状以外にも平板状やループ状があり、大きさもマイクロスケールから数百mまで広い範囲にわたっている。ヒートパイプの熱輸送能力も数W~数kW のオーダまである。以下の各項ではヒートパイプの代表的な応用例として、コンピュータへの応用、密閉電子制御盤、人工衛星用ラジエータ、均熱板、排熱回収熱交換器、均熱ロールの例を示す。
