# 4・7・2 CD（コンパクトディスク）

> **章**: 第4章 入出力装置
> **節**: 4・7 オーディオ機器

a.CDシステムの概要 従来のアナログレコードは，ディ
スクの状態やピックアップの非直線性等の影響を受けるので再生の
忠実度には限界があった．これに対して，コンパクトディスク
（CD:compact disk，compact disc）は音声信号をディジタル信
号の形で記録することによって，高い忠実度で信号を再生できるよ
うにしたものである．ディスクからは1,0のディジタル信号を読み
取るだけでよいので読み取る部分のひずみやわずかな誤りは，再生
信号の忠実度に影響することがないためである．この記録されてい
るディジタル信号には，再生時データを正しく読み取れるように，
音声信号のほかに誤り訂正用のデータも含まれている．そこで，再
生時にはこの訂正用のデータを用いて読み取ったデータをチェック
し，もし誤りがあればこのデータを使用して正しいデータに訂正す
ることができる．
このディジタル信号は，微小ピットの形でディスクに記録してあ
る．再生するにはこのピットを検出し，その有無をディジタル信号
の1,0に対応させることによって読取りを行っている．このため，
12cmのディスクにおよそ5Gb（ギガビット）にのぼる大容量の
データを記録することができ，1時間以上の再生時間を得ている．
さらに上記の信号の読取りは，レーザビームを用いた非接触方式
のため，ディスクは摩耗がなく長寿命である．
b.ディスク 基本構造は透明基盤の片面に信号ピットを形成
し，この面にアルミニウムの反射膜をコーティングしたものであ
る．トラックピッチは1.6mで，ピットの幅は約0.5m，高さ
は約110nmである．ディスクの材質はポリカーボネート樹脂で，
信号の記録は片面のみである．外形寸法と信号の記録されている部
分を図4･57に示す ．
信号トラックはディスクの内周から始まる．トラックのスタート
部にはリードインエリア（lead in area）が設けてあり，ここには
収録されている曲の数やその時間，位置等の種々の情報が記録され
ている．プログラムエリアの記録密度が一定になるように，
1.2～1.4m/ sの一定線速度（CLV:constant linear velocity）で
信号が記録されている．したがって，ディスクの回転数は内周では
速く，外周にいくに従って遅くなる．

からなるデータを2分割してシンボル（symbol ）と呼ばれる8ビ
ット単位の信号にして行う．

d.CDプレーヤーの構成 CDプレーヤーの概略の機能を図
4･59に示す．まず，光ピックアップのサーボ について説明す
る．
ディスクに記録されている信号は，光ピックアップを用いて光学
的に読み取る．ディスクのピット面に正確にレーザスポットを当て
るためには，光ピックアップ本体から可動的に支持された対物レン
ズをコントロールし，光軸や焦点がつねに最適な位置をトレースす
るようにサーボを行う必要がある．光ピックアップからはフォーカ
ス，トラッキング両方向の誤差信号が得られるので，この誤差信号
を打消すように対物レンズと一体となった駆動コイルを動かすこと
でサーボ機構を構成している．
次に再生回路系について信号の流れに沿って説明する．
光ピックアップからは，ピットに応じたEFM信号が再生され
る．まず，このEFM信号からタイミングの基準となるクロック
（clock）を抽出し，さらに同期信号を取出す．このクロックと同
期信号を使って，EFM復調，誤り訂正処理を行い元のオーディオ
信号を得る．
例えば，訂正処理前のランダム性のバイトエラー率が10であ
れば，計算上，訂正処理できないケースは1年に1回未満にすぎな
い．実際はディスクのきずやサーボのアクシデントでエラーが集中
発生して訂正できないケースがあるが，このときは前後の信号から
補間する処理等で再生音のノイズを防いでいる．
オーディオ信号はDAC（digital analog converter ）でアナログ
音声信号に復元される．この信号にはサンプリング周波数（44.1
kHz）の整数倍を中心にサンプリングノイズが含まれるのでロー
パスフィルタで取り除く．
なお，CDには音声信号と一緒に各種の情報を含んでいるサブコ
ード信号（subcode）が記録されている．これを読み取ることによ
り，アナログレコードではできなかった曲の頭出しや経過時間表示
など便利な機能が可能となっている．
