# 6・3・2 密閉制御盤冷却用ヒートパイプ式熱交換器

> **章**: 第6章 熱輸送デバイス
> **節**: 6・3 ヒートパイプの応用

電子機器はその用途も多様化し、機械加工工場、製鋼所、化学工場など、鉄粉・油分・水分・腐食性ガスなどを含む厳しい環境条件下で使用されるようになってきた。この場合、電子機器は密閉された制御盤に収納される。また、屋外に設置される制御盤は雨水の流入や周囲への騒音を防止するため密閉されることも多い。制御盤を密閉した場合、外気と内気が入れ替わらないため、内部の電子機器の発熱により内気の温度が高くなって電子機器からの放熱性能が低下する。そのため、密閉された制御盤の冷却方式としてコンパクトで高性能のヒートパイプ式熱交換器が用いられるようになってきた。a. ヒートパイプ式熱交換器による密閉制御盤冷却方式密閉制御盤へのヒートパイプ式熱交換器(heat pipe heat exchanger)の実装方式として、図6・27,6・28に示す天井取付型と盤側面取付型の2種類がある。それぞれのヒートパイプ式熱交換器の外観を図6・29,6・30に示す。いずれの場合もヒートパイプの一方が盤内空気側に、他方が外気側に設置され、ヒートパイプを介して盤内空気から外気への熱輸送が行われることにより盤内空気が冷却される。図6・27 ヒートパイプ式熱交換器の実装例(盤天井取付型)図6・28 ヒートパイプ式熱交換器の実装例(盤側面取付型)図6・29天井取付型ヒートパイプ式熱交換器図6・30 側面取付型(薄型)ヒートパイプ式熱交換器3―100熱機器ヒートパイプは作動液体の還流力として重力を利用する二相密閉型熱サイホンとして動作するため、その設置形態には制限があり、高温である盤内空気側が低温の外気側より低位置になるよう設置されている。ヒートパイプはアルミフィンが装着された銅管に作動流体として水が充てんされたものからなり、高密度に装着されたアルミフィンにより小型で高性能の冷却能力が達成されている。b. ヒートパイプ式熱交換器による冷却能力計算式図6・31にヒートパイプ式熱交換器を実装した密閉制御盤の熱の流れを熱回路で示す。制御盤内で発生した熱は熱伝導、対流熱伝達、放射熱伝達により制御盤壁およびヒートパイプ式熱交換器に移動し、ここから制御盤の外部に放熱される。制御盤内の空気温度上昇をとすると、盤内発熱量と制御盤表面からの放熱量および熱交換器による放熱量との関係は式(6・31)となる。=+=+(6・31)ここに、:筐体表面を通る盤内空気と外気間の熱通過率[W/(m・K)],:筐体表面の有効放熱面積(m)、:ヒートパイプ式熱交換器の熱コンダクタンス(W/K)である。制御盤およびヒートパイプ式熱交換器の形状が与えられたときの()およびは式(6・32)(6・33)から求めることができる。=1++(6・32)=1욞욠+욄+욂+욞욂(6・33)ここに、:盤内空気、盤内壁間熱抵抗(K/W)、:盤壁の熱抵抗(K/W)、:盤外壁、外気間熱抵抗(K/W)、욞욠:内気、フィン間の熱抵抗(K/W)、욄:ヒートパイプ内蒸発熱伝達に基づく熱抵抗(K/W)、욂:ヒートパイプ内凝縮熱伝達に基づく熱抵抗(K/W)、욞욂:フィン、外気間の熱抵抗(K/W)である。制御盤内の空気温度上昇を所定値욥욄욤以下に下げるための熱交換器の所要能力(熱コンダクタンス)욥욄욤は式(6・31)から以下のように求められる。욥욄욤=욥욄욤-(6・34)盤表面からの熱通過率は、鋼鈑製の制御盤の内外に特別の通風装置を設けない場合には5~6W/(m・K)程度の値となるので、図6・31 密閉制御盤の伝熱回路図6・32ヒートパイプ埋込みラジエータパネルをもつ通信衛星盤内部の発熱量および盤からの放熱表面積がわかれば、式(6・34)から所要の熱交換器の能力욥욄욤を求めることができる。
