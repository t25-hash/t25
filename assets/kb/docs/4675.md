# 6・3・3 衛星用ヒートパイプ埋込みラジエータパネル

> **章**: 第6章 熱輸送デバイス
> **節**: 6・3 ヒートパイプの応用

通信衛星、放送衛星、気象衛星など静止軌道上にある衛星は三軸姿勢制御方式をとっており、その構体は図6・32に示すように一般的に直方体である。その一面はつねに地球を向いており(地球指向面)地球観測機器やレーダなどが搭載されている。一方、人工衛星内部に発生する熱を放熱するためのラジエータは、太陽光線が当たる太陽電池アレイに直交し、太陽光が当たりにくく外部熱環境が安定している向きの構体表面(南/北面)に設けられている。ラジエータパネル(radiator panel)構成を図6・33に示す。ラジエータパネルはその両面を薄いアルミ表皮とした軽量で高い剛性をもつハニカムサンドイッチパネルからなり、その内面にはトランスポンダ(中継器)などの電子機器が搭載され、外面は太陽光線は吸収しにくく、赤外線を放射しやすくしたOSR(optical solarreflector)などの熱制御材が実装された放熱面となっている。ハニカムサンドイッチパネルは表面での平面方向の実効熱伝導率が小さいため、その内部には放熱面全体の温度を一様にするためのヒートパイプがパネル内に二次元的に配列され、熱的に結合されてヒートパイプネットワーク(heat pipe network)が構成されている。パネル内ヒートパイプの配列例を図6・34,6・35に示す。図6・34はパイプ形状がL字状に折れ曲がったL字型ヒートパイプやU字状に曲げられたU 字型ヒートパイプおよびストレート型ヒートパイプが同一平面上で結合された一層配列の例である。図6・35はヒートパイプが格子状に直交して熱結合する二層構造の例であ図6・33 ラジエータパネル構成図6・34 ヒートパイプ二次元配列(一層構造)図6・35 ヒートパイプ二次元配列(二層構造)る。二層構造の場合、放熱板のフィン効率を支配するヒートパイプをラテラルヒートパイプ、ラテラルヒートパイプ間を結合するヒートパイプをヘッダヒートパイプと、その機能面から区別して呼ばれている。しかし、ヒートパイプの基本構造は同一である場合が多い。パネル内部に使用されるヒートパイプの代表的な仕様を表6・3に示す。ヒートパイプは容器にアルミニウム、作動流体にアンモニア、軸方向に設けられた溝をウィックとした軽量で信頼性の大きな軸方向溝型ウィックヒートパイプが用いられている。ヒートパイプの種類にはヒートパイプ1本からなるシングルチャネル型と2本を並行して並べることにより熱輸送能力増大を図ったデュアルチャネル型があり、パネルに搭載される機器からの排熱量に応じて選択的に用いられる。その熱輸送能力は1チャネル当たり100~200W・mである。
