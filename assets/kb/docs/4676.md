# 6・3・4 ヒートパイプ式均熱板

> **章**: 第6章 熱輸送デバイス
> **節**: 6・3 ヒートパイプの応用

高精度の温度均一性が求められる半導体シリコンウェーハ(wafer)や液晶パネルなどの加熱プロセスでは、下面に面状発熱体を接着した均熱板材料(例えばアルミニウムなど)にウェーハを搭載し、この発熱体に発生する熱を均熱板材料に伝え、熱伝導による熱拡散によって全体を均一に加熱する方式が一般に用いられている。この方式の場合、面状発熱体と均熱板材料との接触状態にばらつきが生じやすく表面の温度むらが生じやすいこと、均熱板の熱容量のために熱応答性が悪いことなどの問題点があり、これらの課題を解決するためにヒートパイプを用いることができる。ヒートパイプ式均熱板(heat pipe isothermal heating plate)はヒートパイプの優れた熱輸送特性と高い均熱性を応用して精密な温度履歴を必要とする半導体の加熱プロセスに用いられるものである。その外観を図6・36に示す。円盤状をしたヒートパイプ式均熱板内部には中空部が形成されており、内部を真空排気した後、所定の作動流体が充てんされている。均熱板内部には電気ヒータが設置されており、この発熱をヒートパイプの蒸発部で効率的に作動流体に伝え、作動流体の蒸発と凝縮による熱輸送によって均熱板各部を均一に加熱するものである。表6・3 ヒートパイプ仕様型シングルチャネルヒートパイプデュアルチャネルヒートパイプ容器A 6063 T 6A 6063 T 6作動流体アンモニアアンモニア重量245g/m484g/mウィック軸方向溝軸方向溝熱輸送能力(2.54mmトップヒート時)152W・m(66°C)152W・m/チャネル(66°C)外形状(単位mm)図6・36 ヒートパイプ式均熱板3―101このヒートパイプ式均熱板は、作動流体の蒸気が微小な温度差で移動するため面内の均熱性が高い、蒸気流により瞬時に熱を均熱板各部に拡散させるため応答性が良い、熱容量が小さく温度上昇に要する加熱量が小さくてよいため省エネルギーである、均熱板全体が均一に温度上昇するため熱変形が小さいなどの特徴をもっている。シリコンウェーハ加熱に使用されるヒートパイプ式均熱板の仕様例を表6・4に示す。ヒートパイプ式均熱板は200mmφウェーハ用、300mmφウェーハ用の2種類があり、室温から160°Cの使用温度範囲で±0.15K の面内温度の均熱性能をもっている。直径200mmのシリコンウェーハ加熱用を対象として、従来方式およびヒートパイプ式における温度特性の実測例を図6・37、熱板表面(中心部、中間部、外周部)に温度センサを張り付け、常温から約110°Cまでの温度上昇特性を計測した。なお、昇温は双方とも同一のPDI制御によって行い、ヒータ容量はヒートパイプ方式が500W、従来方式が550W である。図6・37,6・38の均熱板表面温度は設定温度に達する付近の状況を拡大して示したものであり、面内温度差は常温スタートから設定温度に至るまでのばらつきの変化を示している。従来方式では設定温度に対し2K 程度のオーバシュートが見られ、温度が安定するまでの時間も長くなるが、表6・4 ヒートパイプ式均熱板仕様(ウェーハ加熱用)200mmφウェーハ用300mmφウェーハ用寸法φ228×t30mmφ330×t30mm質量4kg7kg使用温度範囲室温~160°C温度分布精度±0.15K 以内(定常時)有効温度範囲200mm以内300mm以内平面度0.02mm以下0.03mm以下図6・37 従来方式均熱板の温度特性図6・38 ヒートパイプ式均熱板の温度特性3―102熱機器ヒートパイプ方式ではオーバシュートもなく安定が早い。また昇温時の面内温度のばらつきは従来方式では昇温時に6K 程度あるが、ヒートパイプ方式では昇温時でも1K 程度のばらつきであり、安定した温度分布が保たれている。また、ヒートパイプ方式は熱容量が小さいために均熱板の温度上昇速度が従来方式よりも速く、ヒータ容量も従来方式よりも10%程度小さいものでよいため省エネルギーで昇温できる。ヒートパイプ式均熱板は半導体ウェーハや液晶パネルなどの加熱プロセスや精密な温度管理を必要とされる接着、硬化などのプロセスに適しており、加熱対象が大型化するほどその特徴が生かされる。
