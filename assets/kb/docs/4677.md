# 6・3・5 排熱回収熱交換器

> **章**: 第6章 熱輸送デバイス
> **節**: 6・38 に示す。試験は均熱板単体を静止空気中に置き、それぞれの均

ヒートパイプを利用した排熱回収熱交換器(exhaust heatrecovery heat exchanger)は多数のフィン付ヒートパイプを千鳥状に配置し、中央に仕切板を設けてその仕切板の一方側に高温排気が、他方側に給気が流れるようにフレームに組込んだ構造とし、高温の排気によりヒートパイプを加熱し、低温の給気に熱輸送するようにしたものである。図6・39にヒートパイプ式排熱回収熱交換器の外観の一例を示す。その用途としては産業用炉(乾燥炉、加熱炉、ボイラ、脱臭装置など)の排熱を回収して元の炉の燃焼用空気や導入外気の予熱、他の装置の熱源あるいは暖房用として使用される例が多い。ヒートパイプ式熱交換器のフィン形式は、図6・40に示すプレートフィン(plate fin)型とラジアルフィン(radial fin)型の2形式に大きく分けられる。プレートフィン型は一般にアルミニウムの薄板をヒートパイプに積層して組込んだものであり、比較的温度の低い領域(200°C程度まで)のクリーンな排ガスの場合に使用される。ラジアルフィン型はヒートパイプの外周にスパイラルのフィンを設けたものであり、材質も鉄やステンレス鋼などを使用できるため、比較的高温であらゆる種別の排ガスの熱回収に適している。作動流体は低温領域ではアンモニア、常温から200°C程度までは水、それより高い温度領域ではダウサム、ナフタレンなどが使用される。ヒートパイプ式排熱回収熱交換器の一般的な特徴は、構造が簡単で単位体積当たりの伝熱面積が比較的大きくとれるために小型・軽量であること、中間の仕切板により排気と給気が混合しない気密構造にできること、熱輸送のための動力を必要としないので省エネルギーであること、機械的な可動部分がないので保守が容易であること、従来の熱交換器に比べて圧力損失が低く、既存の設備にもそのまま取付けられる場合が多いことなどである。ただし、この熱交換器の設置に際しては、ヒートパイプが水平もしくは高温側が低温側図6・39排熱回収熱交換器外観(b) ラジアルフィン型(a) プレートフィン型図6・40 フィンチューブの種類よりも低位置になるように傾斜させる必要がある。ヒートパイプ式排熱回収熱交換器の設計は従来の対向流型熱交換器と同様の考え方で行うことができるが、ヒートパイプの作動流体が熱交換器の使用温度範囲に適したものであること、熱交換器の処理能力がヒートパイプの熱輸送能力の範囲内であることを前提とする。ヒートパイプ式排熱回収熱交換器の交換熱量は式(6・35)で表される。=욆욇(6・35)ここに、:交換熱量(W)、욆욇:両流体間の対数平均温度差(K)、:両流体間の熱抵抗(K/W)でありは式(6・36)で表される。=1욄욠=1욄욠욄욠+욦욠욄욠+욇욠+1욡욠욡욠+1욡욂욡욂+욇욂+욦욂욄욂+1욄욂욄욂(6・36)ここに、:熱通過率[W/(m・K)],욄욠욄욂:高(低)温側管外熱伝達率[W/(m・K)],욡욠욡욂:ヒートパイプ蒸発(凝縮)部熱伝達率[W/(m・K)],욇욠욇욂:高(低)温側ヒートパイプの容器壁熱抵抗(K/W)、욦욠욦욂:高(低)温側汚れ係数[(m・K)/W ],욄욠(욄욂:高(低)温側管外伝熱面積(m)、욡욠욡욂:ヒートパイプ蒸発(凝縮)部伝熱面積(m)である。管外熱伝達率、ヒートパイプの容器の熱抵抗、汚れ係数は一般の熱交換器の場合と同様である。ヒートパイプ式排熱回収熱交換器を乾燥炉(寸法:幅4.1m×長さ7m×高さ4.7m)の排熱回収に適用したときのシステム例を図た乾燥用空気(熱風)の循環により乾燥炉に搬入された製品の乾燥が行われるが、製品の乾燥により発生する水蒸気の濃度を一定以下にするために循環空気の一部が排気ファンにより外部に排出され、同量の給気が炉内に取入れられる。排気と給気のダクトの途中にヒートパイプ熱回収式熱交換器が設置され、給気と排気との間のヒートパイプによる熱交換によって給気が予熱されて循環空気の中に供給される。すなわち、給気の加熱分だけ蒸気ヒータの加熱量を低減することができる。この例では乾燥炉の温度が約150°Cと比較的低温で、排気も清浄なことから、プレートフィン型(24段、12列、長さ3m)を使用している。この排熱回収システムの各部温度と回収熱量の一例を表6・5に示す。温度効率66%で使用され、156kWの排熱が回収されている。図6・41 乾燥炉の熱回収フロー表6・5 乾燥炉の熱回収例排気給気流量N・m/min入口温度°C出口温度°C温度効率%回収熱量kW
