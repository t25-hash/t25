# 4・2・3 燃料および炉心設計

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・2 沸騰水型軽水炉

a.BWR燃料 BWR燃料集合体（fuel assembly）は，多
数の二酸化ウラン（一部の燃料棒については少量のガドリニアをウ
ランと混合）焼結ペレット（sintered pellet ）をジルカロイ-2製の
燃料被覆管に入れ，被覆管の両端を端栓で溶接した燃料棒を組立て
たものである．燃料棒の配列については，BWR初期の7×7型か
ら数回の設計改良がなされ，現在では9×9燃料が用いられている．
9×9燃料（A型）は，2本の太径のウォータロッドを燃料集合
体の中央部に配し，その周囲に74本の燃料棒を9行9列に配して
構成する．このうち8本の燃料棒は，標準燃料棒の約3分の2の長
さとした部分長燃料棒であり，下部の燃料支持板（タイプレート）
に固定されている．また，74本の燃料棒のうち8本の燃料棒（タ
イロッド）が上部および下部タイプレートを結び付ける役目をして
いる．
9×9燃料（B型）は，1本の角管のウォータチャネルを燃料集合
体の中央部に配し，その周囲に72本の燃料棒を9行9列に配して
構成する．72本の燃料棒のうち8本の燃料棒（タイロッド）が上
部および下部タイプレートを結び付ける役目をしている．
9×9燃料（A型）の2本のウォータロッドのうち1本または
9×9燃料（B型）のウォータチャネルは，スペーサの軸方向位置
を決める機能を有し，7個のスペーサを保持している．ウォータロ
ッドまたはウォータチャネルはジルカロイ製であり，内部を冷却材
が通過するようになっている．
スペーサは，ジルカロイ製の外枠，円筒形セルとインコネル製の
スプリングからなっており，燃料棒間の間げきを保つ役割を果た
す．
9×9燃料（A型）の標準燃料棒，9×9燃料（B型）の燃料棒，
ウォータロッドおよびウォータチャネルの上部端栓は，上部タイプ
レートの孔の中を上下に動きうるようになっている．上部タイプレ
ートは，これらの上部端栓にはめてあるインコネル製のエクスパン
ションスプリングによって支えられる．このような構造とするこ
と，およびスペーサのスプリング強さを適切に設計することによ
り，燃料棒，ウォータロッドおよびウォータチャネルは，すべて独
立して軸方向に自由膨張ができるようになっている．
下部タイプレートは，炉心下部から流入する冷却材を各燃料棒に
分配する役割を果たす．また，上部タイプレートには燃料集合体の
移動のためにハンドルが付けられている．
燃料集合体を炉心に装荷する際には，外側にジルカロイ製のチャ
ネルボックス（channel box）をはめる．チャネルボックスは，燃
料集合体内の冷却材流路を定めるとともに，制御棒作動のガイドお
よび燃料を保護する役割を果たしている．
図4･5および4･6に燃料集合体の構造図の例を，表4･2に基本仕
様を示す．
b.設計方針
.燃料の許容設計限界 通常運転時および運転時の異常な
過渡変化時において，原子炉の設計と関連して，燃料の損傷を安全
上許容される程度とし，かつ，継続して原子炉の運転をすることが
できるようにするため，以下に示す燃料の許容設計限界を用いてい
る．
