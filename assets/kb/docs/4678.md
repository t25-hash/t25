# 6・3・6 ヒートパイプ式均熱ロール

> **章**: 第6章 熱輸送デバイス
> **節**: 6・41 に示す.この乾燥システムでは,蒸気ヒータで加熱され

紙,合成繊維,不繊布,樹脂フィルム,金属はくなどの製造過程においては,カレンダロール,ラミネータロール,シーズニングロール,コータロール,予熱ロールなど,加熱された回転ロールが多数用いられている.通常のこれらの回転ロールではロール内部を中空とし,その中に温水や高温蒸気を流すことによりロールを加熱する方法がとられている.しかし,温水を流す方式でロールの熱負荷が小さい場合,その表面温度は均熱化されているものの,熱負荷が大きくなると軸方向や周方向に温度むらが生じるという欠点があり,紙やフィルムの品質に空間的あるいは時間的なむらが生じる問題があった.また高温蒸気を流す方式では,蒸気発生装置が必要で装置が大型化したり,熱負荷変動に対する応答が遅くなるという問題があった.そのため近年,回転ロールの胴部をヒートパイプ化し,かつロールの加熱方式として誘導加熱方式を採用したヒートパイプ式均熱ロールが用いられるようになってきている.ヒートパイプ式均熱ロール(heat pipe isothermal heating roll)の特徴を下記に示す.(1) ヒートパイプおよび誘導加熱方式の採用により大きな熱負荷においても優れた均熱性をもつ.(2) ヒートパイプの作用により負荷変動に対する温度制御応答性が優れている.(3) 誘導加熱方式によりロール自体を加熱するため熱損失が少なく省エネルギーである.(4) ロール自体に加熱源があるため加熱装置・配管などの付帯設備が不要である.このヒートパイプ式均熱ロールの構造を図6・42に,その外観写真を図6・43に示す.ロール内部の中心部にコイルと鉄心からなる誘導加熱(induction heating)装置が組込まれ,その外周胴部に複数本のヒートパイプが設けられている.ロール外径は200~1250mm,胴長は600~7600mm程度のものが一般に製作されている.その表面はロールの使用条件に応じて硬質クロムめっき,テフロン・セラミックコーティング,エンボス加工などが施されている.図6・42 ヒートパイプ式均熱ロール構造図6・43 ヒートパイプ式均熱ロール外観図6・44 ヒートパイプ式均熱ロールの温度分布3―103ロールの材質としては炭素鋼,ステンレス鋼がおもに使用されている.ロールの加熱に用いられている誘導加熱方式とはロールの内部に配設されたコイルに流れる交流電流により交番磁界が発生し,その電磁誘導作用によりロール本体に渦電流が流れて発熱が生じる作用を利用したものである.この誘導加熱方式ではロール自体が発熱するため,極めて効率の良い加熱を行うことができる.ロールの胴部に設けられたヒートパイプとしては,ロールに加工された孔にヒートパイプを設けたもの,あるいはロールの胴部に設けられた孔自身に作動流体を封入して胴自身をヒートパイプとしたものなどがある.作動流体の循環に遠心力と重力が利用できるためヒートパイプはウィックを必要としない.ロール温度の使用範囲は通常80~250°C程度であり,作動流体としては水が多く用いられる.これよりも温度が高い250~400°C程度までの領域ではダウサムなどが使用される.図6・44に均熱ロールの温度分布の実測例を示す.ロール本体に一体結合されたヒートパイプの優れた熱伝達性能により,ロールの高温側から低温側へ効率良く熱が輸送され,ロール表面全域で温度分布が±1K 以下の高い均熱性が実現されている.また,これらのヒートパイプ式均熱ロールではロールに埋め込まれた温度検出センサからの信号によりPID 制御が行われ,熱負荷に応じた温度制御が行われている.特殊ヒートパイプは大きく二つに分類される.一つは基本的に作動原理そのものが同じで,形状・構造や機能面,利用方法の違いによって分類されているもの,もう一つは作動原理そのものが異なるものである.形状・構造に関していえば,容器自体がベローズ構造のように伸縮可能なものや曲げ加工が可能なもの,蒸気の流れと凝縮液の流れを分離して一方向流れとした分離型のものなどが挙げられる.また,作動液の還流に毛細管力を用いるものを厳密にはヒートパイプと呼んでいるが,重力や遠心力を用いたものも同様にヒートパイプと呼んでいる.機能面においては,通常のヒートパイプはその等温性を確保するために極力容器内の不凝縮性ガスを排除しているが,あえて所定量の不凝縮性ガスを注入し,温度に合わせて放熱量を変化させるものがある.可変コンダクタンスヒートパイプ(variable conductanceheat pipe)と呼ぶもので,作動液のもつ飽和蒸気圧に対応し,反応性のない不凝縮性ガス(例えばHeやAr,Nなど)を所定量注入することによって,使用温度およびコントロール範囲を調整することができる.利用面についていえば,ヒートパイプを単なる熱輸送デバイスとして利用するだけではなく,蒸気流路に羽根車を設置しタービンのように回転させて動力を得るために利用されたこともある.作動原理が異なるもので熱輸送デバイスとしての位置付けで呼ばれているものに,化学反応を用いたケミカルヒートパイプや,液プラグと蒸気泡の二相流が振動により熱輸送を行う自励振動型ヒートパイプなどがある.
