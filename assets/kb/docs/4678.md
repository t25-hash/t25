> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・2 沸騰水型軽水炉

原子力プラントでは，原子炉・タービン発電機・工学的安全施
設・プラントの各種補助設備に対する制御系を有している．図4･7
にBWRの制御系の例を示す．これらの制御および，プラントの
運転監視のために，原子炉核計装（neutron monitoring system），
放射線計装（radiation monitoring system），プロセス計装などの
各種計装が設けられている．
a.原子炉の制御
.出力制御 原子炉の出力制御は，炉心の反応度を制御す
ることにより行う．軽水炉の場合には，制御棒と炉心冷却材の流量
あるいは，一次冷却材中のホウ素濃度の調整を組合わせて炉心反応
度を制御している．
BWRでは，一般に原子炉の起動，停止などの大幅な出力変更，
出力分布の調整および，燃料の燃焼に伴う長期の炉心反応度変化の
補償は制御棒により行い，負荷変動に対する急速な出力の調整は，
炉心冷却材の流量を調整する再循環流量制御により行っている．ま
た，いずれの場合も，原子炉を緊急に停止する必要のある場合は，
制御棒を全数同時に急速挿入する．
.水位・圧力制御 原子炉を安定に運転するため，原子炉
出力の制御に加え，水位と圧力の制御を行っている．原子炉水位
図4･7 BWR原子力発電所の制御系
は，原子炉から取出される蒸気量と原子炉に注入する給水量のバラ
ンスにより決まるので，原子炉水位，主蒸気流量，給水流量の三つ
の要素を検出して，原子炉の出力に見合った適切な給水制御を行う
3要素制御（three element control ）により原子炉水位を制御して
り自動化されている．
いる．また，原子炉圧力の制御は原子炉圧力を一定に保つように原
子炉からタービンに供給する蒸気量を制御する．すなわち，原子炉
出力に従って発生する蒸気量をタービン加減弁あるいはタービンバ
イパス弁により制御して，原子炉圧力を一定に保っている．
