# 6・4・1 大型、長尺ヒートパイプとその応用

> **章**: 第6章 熱輸送デバイス
> **節**: 6・4 特殊ヒートパイプ

a. 道路融雪システム長尺ヒートパイプの代表的応用例には、道路の融雪システムがある。特に自然熱や排熱、例えば地熱、温泉熱、排湯を利用した融雪システムは、省エネルギーに寄与している。図6・45に地熱利用融雪システムの構成図を示す。多数のヒートパイプを地中3~18mの深さに埋め込み、地熱エネルギーを集め、路面に熱輸送し、路面の融雪、凍結防止を行うシステムである。土壌温度は地中深さ7m以上では、年間を通じてほぼ一定で、その地方の年間平均気温にほぼ等しい。本州では、土壌が10~14°Cの熱エネルギーを保有していると見てよい。ヒートパイプ1本当たりの熱輸送量(W)と表面熱流束(W/m)3―104熱機器は、式(6・37)(6・38)で表される。例えば路面融雪を例にとると、これらの熱抵抗は、土壌の熱伝導率、ヒートパイプの直径、長さ、作動流体の種類、埋設パターン、(ボーリング孔のピッチ、舗装内の埋設ピッチ)、舗装材(コンクリート、アスファルト等)の厚さと熱伝導率によって決められる。気温低下または降雪により舗装表面温度が下がると、土壌温度と路面温度に差が生じることによって、このシステムは自動的に熱エネルギーを土壌から輸送する。なお、熱負荷が大きくなるほど熱輸送量は増加して、より大きな能力を発揮する。また、冬季でも日中、日射が強いと冠雪のない路面に熱が吸収され、その温度が土壌温度より高くなれば熱輸送は停止し、ヒートパイプは休止状態となる。このシステムは、路面熱流束で100W/m、融雪量で1cm/hが限界である。温泉水、温泉排湯の自然熱や廃熱を熱源とした融雪システムのほかに、ボイラ熱源の融雪システムも実用化されている。b. ヒートパイプによる電柱支線の融雪山間部の豪雪地帯図6・45 地熱利用融雪システムの構成図図6・46ヒートパイプによる電柱支線融雪装置図6・47 ヒートパイプ式地中低温貯蔵庫では、雪の沈降力が原因で、電柱の折損あるいは支線の切断事故が発生することがある。この予防策としてヒートパイプによる倒壊防止装置が開発、実用化された。図6・46にヒートパイプによる電柱支線融雪システムを示す。ヒートパイプは地熱を吸収して上部に熱を伝え、支線周囲の雪を溶かし、直径10cm程度の孔を作る。これによって、雪と支線は接触しなくなり、雪圧で電柱が倒壊したり、支線が切断したりする事故を防ぐことができる。地中の温度は、冬季を通じて5~10°C程度で、1本のヒートパイプは10~15W の熱を運ぶ。c. 地中低温貯蔵庫ヒートパイプ式地中低温貯蔵庫は、冬季の寒気を利用してヒートパイプにより地中に人工凍土を作り、その凍土層によって囲まれた空間を低温貯蔵庫として利用するものである。概念図を図6・47に示す。冬季に外気温度が0°C以下になると、ヒートパイプは地中から熱を吸収して、作動流体は蒸発して凝縮部へ流れ、放熱部より空気中に放射される。そのためヒートパイプ周囲の土壌は冷却され凍結する。このヒートパイプはウィックレスであるため、上部が加熱されても、熱を上から下に伝えないというダイオード特性があり、春以降気温が上昇しても大気中から熱を地中には伝えない。そのため、凍土は長期間保持され、食料の貯蔵用の冷熱として利用できる。ヒートパイプと自然冷熱のみの利用のため運転動力を必要としない。図6・48に積算寒度を示す。d. 大型ヒートパイプによる地熱利用地下から直接地熱を図6・48 積算寒度図6・49シャワーノズル付ループヒートパイプの構造抽出する方法にヒートパイプを利用することが考えられる。長尺のヒートパイプを垂直に布設して、地熱を抽出しようとすると、従来の単管式ヒートパイプでは、熱輸送量が増えると蒸気流速が増大して、飛散限界やフラッディング限界が発生する。そこで、シャワーリングノズルを有するループ式ヒートパイプが開発された。図6・49に構造図を示す。垂直蒸発管内に液体を霧状に噴霧するシャワーノズルを有しており、液体は重力で加圧供給される。したがって、垂直に長い蒸発管でも下部が液体のプールにならず、液膜蒸発が実現できる。蒸発した蒸気は上部に上昇し凝縮器に導入され熱交換される。九州電力大岳地熱発電所で、100kW 級の実証試験が実施された。
