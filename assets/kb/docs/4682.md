# 6・5・1 ヒートパイプの劣化

> **章**: 第6章 熱輸送デバイス
> **節**: 6・5 ヒートパイプの信頼性

一般に何をもってヒートパイプの信頼性(reliability)というかを明確にする必要がある。ヒートパイプは極めて単純な構造であり、基本的に外部動力を必要としない完全に外部と独立した密閉容器である。したがって、密閉する段階と密閉した後に、本来望ましくない状態が起きた場合に信頼性がないということができる。ヒートパイプの内部は本来作動液とその飽和蒸気のみで空間が満たされていなければならない。しかし、密閉する際に不凝縮性のガスが混入したり、容器の密封度が完全でないためにリークしたり、あるいは内部の清浄度が十分でないためになんらかの反応が起こり不凝縮ガス(noncondensable gas)を発生させてしまう場合が劣化として考えられる。いずれの場合も、不凝縮性ガスがヒートパイプ劣化に影響するもので、これにより本来熱輸送すべき領域まで蒸気が移動しなくなるために、等温部分の長さが減少し、ひいては放熱面積が減少して熱輸送素子としての性能低下を起こすのである。特に、径の細いヒートパイプほど不凝縮性ガス発生に対して敏感となっていく。なぜなら、ガスの発生量が内表面積に比例すると考えられ、またガス発生が温度低下に及ぼす影響はその内容積に反比例すると考えられるからである。したがって、内表面積が相対的に大きく、内容積が相対的に小さい、すなわち小径のヒートパイプほどガスの発生とその影響度が大きいということになる。いま、内容積()に対する内表面積()の比(=/)はほぼ径の大きさに反比例することから、例えば3mmφのヒートパイプと10mmφのヒートパイプでは、より径の小さい3mmφヒートパイプの劣化影響度の大きさが重大なものであることが推定される。したがって、ヒートパイプ、特に電子機器に用いられるようなマイクロヒートパイプでは長期の信頼性を十分確認し確保することが重要である。
