# 6・5・2 信頼性の評価方法

> **章**: 第6章 熱輸送デバイス
> **節**: 6・5 ヒートパイプの信頼性

信頼性確認のためには寿命試験(ライフテスト)といわれる試験を行う。これは熱負荷をかけた状態で、しかも通常使用が想定される温度よりもヒートパイプの作動温度を上げて行う一種の加速試験方法である。図6・58に加速試験の装置の構成を示す。ヒートパイプは垂直に立てられ、下部(蒸発部)には一定の熱負荷が加えられる。また上部(凝縮部)はファンで冷却され、ヒートパイプの蒸気温度もしくは断熱部(中央部付近)の温度(作動温度)が一定の温度になるように調整される。ヒートパイプの作動温度に対し、ヒートパイプ上部先端での温度の変化を経時的に測定することによって劣化は確認できる。もし不凝縮性ガスが発生すれば上部先端の温度は低下し、蒸気温度とのは大きくなる。一方不凝縮性ガスの発生がほとんどない場合には1万時間を経たあとでもこの温度差(断熱部温度-上部先端温度)はせいぜい数°Cしかつかず、極めてわずかの差しか認められない。図6・59にこの方法によるマイクロヒートパイプの評価結果例を示す。なお長期信頼性試験は、加速試験を行うために作動温度は例図6・58 加速試験の装置図6・59性能変化の特性例3―107えば150°C,200°Cといったように通常の使用温度よりも高温で行う。また、非凝縮性ガスの影響を見る際、断熱部と上部先端部の温度差を測定するため、内圧を下げてガス検出感度を向上させる目的で、作動温度を50°C程度まで下げている。図6・59においては不凝縮性ガスの経時的な発生パターンの代表例を示している。不凝縮性ガスが多く存在する場合には、温度差は大きく現れている。ここで直線A,B は、封止の際の状態で決まり経時的な変化がないものである。すなわち、十分に脱気ができている場合にはA、そうでない場合にはB のような状態となる。また、D は時間の推移にほぼ比例するように温度差が拡大していく場合である。これはコンテナ容器に微小なリーク箇所があり、一定の条件で使用しているときに空気が外部よりヒートパイプ内部に徐々に侵入していくことで劣化が進んでいく場合が多い。また曲線Cは、ヒートパイプの作動を開始してから初期に急速に劣化したあと徐々に変化が少なくなる場合で、このような曲線を描くときは、ヒートパイプ内部に初めから付着していた不凝縮性ガスが作動に伴い分離されていく場合が多い。一方、曲線Eは最初はほとんど劣化が見られないものの、ある時点から急激に温度差が生じ始めるタイプで、この場合には作動液とコンテナ材料との不適合や容器に付着した汚れ・油分などが作動に伴い分離分解し、不凝縮性ガスが発生していることが多い。いずれにしても、温度差は初期では低い値に抑えられ、長期にわたって変化の少ないものが長期信頼性があるということができる。次項で、信頼性を確保し、またそれを確認するための方法について示す。
