# 4・4・2 新型転換炉（ATR:advanced thermal reactor )

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・4 重水炉

新型転換炉の開発は，エネルギー資源に乏しい日本において核燃
料の有効利用の観点から，1967年以来ナショナルプロジェクトと
して核燃料サイクル開発機構（旧動力炉・核燃料開発事業団）が中
心となって進められてきた．出力165MWeの原型炉「ふげん」
は，1979年に本格運転を開始して以来順調に稼働し，燃料には微
濃縮ウランとMOX（mixed oxide）(ウラン・プルトニウム混合
酸化物）燃料が用いられている．原型炉「ふげん」は，優れた
MOX燃料の利用実績や，ATRでの核燃料サイクルの環の完結等，
わが国の核燃料サイクル政策を先行的に具現化，実証してきたが，
ATR開発のための原型炉としての役割が終了したことから2003
年に運転を終了し，廃止措置段階に移行している．
実証炉については，1982年に原子力委員会が全炉心にMOX燃
料を装荷できる600MWe級プルトニウム燃焼炉の計画を決定し，
諸準備が進められたが，1995年の電気事業連合会からの実証炉計
画見直しの申入れを受け，原子力委員会はおもに経済性を理由とし
て実証炉計画の中止を決定した．

e.工学的安全防護設備 工学的安全防護設備として，急速注
水系，高圧注水系，低圧注水系からなる非常用炉心冷却系ならび
に，蒸気放出プール冷却系，隔離冷却系および余熱除去系が設置さ
れている．急速注水系は，低圧注水系が作動するまでの間に注水す
るもので，常時加圧された蓄圧器の水を下部ヘッダに注水する．高
圧注水系は，中小破断事故時に蒸気ドラム内の減圧を早め，低圧注
水系の作動を促進させるために蒸気ドラムに注水する．低圧注水系
は，大破断事故時に下部ヘッダまたは蒸気ドラムに注水する．
工学的安全防護設備は互いに独立し，おのおの100 %の容量をも
つ2系統から構成されており，外部からの電源供給が得られない場
合でも非常用ディーゼル発電機により電源が確保される設計となっ
ている．

i .プラント設計 新型転換炉のプラント設計の基本的な考え
方は，発電用の軽水炉と同様である．ここでは，熱水力設計と安全
.熱水力設計 冷却材や減速材などの核設計の基本となる
要素は，温度やボイド率等の熱水力設計により定まるパラメータに
よって変化する．このため，炉心設計では，核設計で得られた炉心
内出力分布を基に熱水力設計により炉心内の流量分布とボイド率分
布を求め，それらを基に再び炉心内出力分布を求めるという繰返し
計算を収束するまで行うのが一般的である．熱水力設計では，この
ような核熱水力結合計算により求めた熱的余裕を評価する指標が，
原型炉「ふげん」では，この熱的余裕を評価する指標として，限
界熱流束相関式（CHF相関式，critical heat flux correlation）に
基づく最小限界熱流速比（MCHFR:minimum critical heat flux
ratio）を使用し，プラントの異常な過渡変化が起こった場合にも
燃料要素表面での伝熱形態が核沸騰の範囲にとどまることを基準と
CHF相関式は，核燃料サイクル開発機構大洗工学センターの14
MW大型熱ループにおいて実施した実規模バーンアウト実験のデ
ータに基づいて，限界熱流束と燃料集合体水平断面平均クオリティ
の関係より最下限線をとり求めている．
炉心の最小限界熱流束比を求めるため，核設計で評価された出力
分布と各燃料チャネルの流量が必要となる．各チャネルへの流量配
分は，各チャネルの下部ヘッダから蒸気ドラムまでの圧力損失が等
しいという条件で求められる．この際，熱バランスを考えて各チャ
ネル内部のエンタルピー上昇，したがって，クオリティおよびボイ
ド分布の圧力損失への効果も考慮する必要があるので，チャネル流
量配分計算は繰返し計算となる．各チャネルごとの流量配分および
エンタルピー，クオリティ，ボイド分布等の熱水力特性が計算され
ると，これらの特性値から熱的余裕値が計算される．
