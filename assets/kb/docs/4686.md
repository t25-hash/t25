# 6・5・4 その他の確認内容

> **章**: 第6章 熱輸送デバイス
> **節**: 6・60 に示すように、加える圧力により検出能力は変化するが、0.1

電子部品用のヒートパイプにはJIS 規格やMIL規格などに示される試験が適用される場合がある。その中で比較的よく用いられるものに次の試験がある。(1) 高温動作寿命試験(2) 高温高湿寿命試験(3) 温度サイクル試験(4) 熱衝撃試験このほかには振動、衝撃などによる影響を見る場合もある。ヒートパイプは可動部がなく構造がシンプルなうえ、受動部品であるため基本的には信頼性は極めて高い。そのため銅-水系マイクロヒートパイプでは、これらの試験において問題となる例はほとんどないが、温度サイクル試験での低温側においては凍結に対する影響を検討する必要がある。ヒートパイプが凍結で問題となるのは、水が凍結するときに発生する体積膨張である。凍結時の体積膨張によってヒートパイプの容器が変形する場合があり、このような場合には、作動液を入れすぎない、外部から適当な補強をするなど使用条件に応じた適切な対策をとる必要がある。銅などの熱伝導率の高い材料で作られたフィルムや板あるいは棒などに比べて、同型で多量の熱を拡散あるいは輸送するものを熱輸送デバイスと総称する。閉空間内の加熱部で液体の蒸発を起こし、その蒸気を冷却部で凝縮し、凝縮液を毛管力により加熱部に還流するヒートパイプ(6・1~6・5節参照)は、この代表例である。図6・61 熱音響効果(壁近くの流体要素Cは音響振動により位置H とLとの間で振動する)ウィック型ヒートパイプに代表されるヒートパイプは、外部から動力等を供給せずに作動する優れた自立型熱輸送デバイスであるが、以下のような課題もある。(1) 主として還流能力に基づく熱輸送限界が存在する。(2) 特に細径化に伴い熱輸送限界が低下する。(3) 還流機構を要するため、フレキシブル化が難しい。(4) 不凝縮性ガスの凝縮部への堆積により、熱輸送量が低下する。(5) 熱スイッチなどのように熱輸送量を制御するには、特殊な工夫を要する。これらの課題に対し、そのいくつかを補うことのできる熱輸送デバイスが提案され、一部実用化している。その代表例は、還流構造をもたない熱輸送デバイスであり、振動流型熱輸送デバイスなどがある。振動流型熱輸送デバイスとしては、単相強制振動流型熱輸送デバイスや二相自励振動流型熱輸送デバイスなどがある。前者は、アクチュエータなどにより流路内液体に強制的に振動流を起こさせ、その振動流による熱拡散促進効果を利用する熱輸送デバイスであり、相変化は利用しない。後者は、細い流路内に封入された気液二相に自励的に誘起される振動流や脈動循環流による熱輸送を利用する熱輸送デバイスである。振動流による熱拡散促進効果とは、以下の効果である。図6・61に示したように、静止した非粘性流体中に薄い板が置かれている系を考える。この系には、板の長さ方向(方向)に温度こう配があり、非粘性流体中には音響定在波が加えられている。板の長さは音響定在波の波長より十分に小さいとし、板は温度変動しないとすると、音響定在波により方向に次式で与えられる熱流が流体中に誘起される。式(6・40)の右辺第1項は、正の量であり温度こう配をさかのぼる方向への熱流を表し、ヒートポンプ効果を表す項である。第2項は、温度こう配∇と逆符号の量であり、温度こう配を下る方向への熱流を表し、熱拡散促進効果を表す項である。流体が液体の場合は、βは十分に小さく、したがって、式(6・40)の第1項は第2項に比べて十分に小さい。すなわち、温度こう配をもつ固体に沿って液体に振動流を起こさせると熱拡散促進効果が現れる。単相強制振動流型熱輸送デバイスでは、この振動流による熱拡散促進効果を利用する。
