# 4・5・2 原子炉の構造と安全性の特徴

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・5 高温ガス炉

a.原子炉の構造 高温ガス炉は黒鉛減速ヘリウム冷却型の
原子炉であり，燃料，炉心構成，出力密度，原子炉圧力容器および
一次系主要機器等に特徴がある．炉心は耐熱性に優れる被覆燃料粒
子と黒鉛材料で構成されており，ヘリウムガスで冷却され，軽水炉
に比べて低い出力密度の炉心と相まって高い安全性を有する．燃料
はウランのほか，トリウムも使用できる．平均燃焼度約10万
MWd/ t以上の高い燃焼度が得られ，使用済燃料は再処理せずにそ
のまま廃棄できる特性をもつ．高い炉心温度を達成するため，金属
材料の代わりに黒鉛等のセラミック材料で炉心を構成する．原子炉
格納形式として，コンファインメント方式が用いられる．図4･27
に，被覆燃料粒子の構造と，ペブルベッドおよびブロック形式の燃
料構造を示す．
燃料は，ウラン酸化物等の微小粒子を熱分解炭素，炭化ケイ素で
多重に被覆した直径1mm以下の被覆燃料粒子である．セシウム
137やストロンチウム90等の金属核分裂生成物をも効果的に閉じ
込めるため，炭化ケイ素層を有する四重被覆粒子が用いられる．被
覆燃料粒子の各被覆層の機能は次のとおりである．
(１) 低密度熱分解炭素層：燃料核表面付近から飛び出す核分裂
片の運動を止め，外部の被覆層の損傷を防止するほか，核分裂生成
ガス等をためる空間，燃料核スウェリングの吸収，燃料核移動に対
する緩衝帯としての機能がある．
(２) 内側高密度熱分解炭素層：炭化ケイ素蒸着の際に発生する
不純物と燃料核とが反応するのを防ぐほか，照射収縮により炭化ケ
イ素層に圧縮応力を発生させ，被覆燃料粒子を安定化させる．ま
た，ガス状核分裂生成物に対する拡散障壁となる．
(３) 炭化ケイ素層：ガス状核分裂生成物，固体状核分裂生成物
に対する拡散障壁であるとともに，被覆燃料粒子全体の強度を保
つ．
(４) 外側高密度熱分解炭素層：照射収縮により炭化ケイ素層に
圧縮応力を発生させ，照射下の被覆燃料粒子全体の強度を保つ．
燃料要素にはこれまで棒状燃料，ペブルベッド型燃料およびブロ
ック型燃料の3種類がある．棒状燃料では，軽水炉燃料のように燃
料ペレットを黒鉛スリーブに挿入して燃料棒とし，この燃料棒が黒
鉛ブロックに装荷される．燃料棒内には核分裂生成物トラップ用の
空間を有している．アメリカのピーチボトム炉およびドラゴン炉の
初期炉心で採用された．球状燃料を用いるペブルベッド型燃料で
は，直径60mmの黒鉛素地中の直径50mm内に被覆燃料粒子を
分散させている．ドイツで開発され，ドイツの高温ガス炉AVR，
THTR-300で採用された．中国，ロシアおよび南アフリカ共和国
の高温ガス炉で採用される計画である．ブロック型燃料にはマルチ
ホール型燃料とピンインブロック型燃料とがある．マルチホール型
燃料では，六角柱状黒鉛ブロック中に燃料棒用孔とヘリウムの流路
用孔とが別々に設けられており，フォートセントブレン炉で採用さ
れた．フォートセントブレン炉では，このような黒鉛ブロックの燃
料集合体を六段積み上げて1カラムを構成し，247カラムで炉心を
構成した．ピンインブロック型燃料では，黒鉛ブロック中の燃料棒
周辺に環状流路としてヘリウム用流路が設けられている．ドラゴン
炉の後期炉心および日本のHTTRで採用されている．HTTRで
は，黒鉛ブロックを五段積み上げて1カラムを構成し，30列の燃
料カラムと7列の制御棒案内カラムで炉心を構成している．
高温ガス炉の燃料サイクルでは，開発の当初からトリウムの利用
が積極的に追求され，ドイツやアメリカの発電用原型炉では高濃縮
ウラン，トリウムサイクルが採用された．しかし，核不拡散の観点
から低濃縮ウランまたは低濃縮ウラン/トリウム燃料に変更された．
平均燃焼度が約10万MWd/ tであり，使用済燃料中の核分裂性核
種の残留量が少なく再処理をする必要性が低く，使用済燃料を単純
な処理をして永久貯蔵するワンススルー方式が提案されている．

b.高温ガス炉の安全性 高温ガス炉がもつ優れた特性を活
用し，固有の安全性（inherent safety）を可能な限り高めることに
より，高温ガス炉では事故時の安全確保の基本要件である原子炉を
止め，冷やし，放射性物質を閉じ込めるという機能を動的設備の作
動や運転員の操作に期待せず，可能な限り固有の特性のみで果たさ
れる設計が実現できる．高温ガス炉が固有の安全性に優れているの
は，燃料に被覆粒子燃料（coated particle fuel ），減速材と炉心構
成材料に黒鉛，原子炉冷却材にヘリウムガスを用いていることによ
るものである．その特徴は，以下のとおりである．
(１) 炉心構成材に黒鉛を用いているため熱容量が大きく，出力
密度が小さいため，反応度の異常な上昇や冷却能力の異常な低下な
どが生じても，炉心温度の変化が極めて緩慢である．すなわち，熱
容量は被覆燃料粒子の健全性が損なわれず有意なFP放出の可能性
のない1600℃までとしても，高温ガス炉は軽水炉の約5倍であり，
また，高温ガス炉の出力密度は2～6MW/ mと軽水炉の数十分の
一である．
(２) 炉心が高温になっても燃料や炉心構成物は溶融することな
く，炉心溶融（core melt ）の心配がない．炉心の大部分が黒鉛で
構成されており，黒鉛は3000℃でもほとんどその強度が低下せず，
また，燃料の被覆層を形成する炭化ケイ素も約2400℃まで熱分解
することがない．
(３) 冷却材のヘリウムガスは化学的に不活性であり，事故時に
水のように相変化し蒸気になって爆発的な機械力を発生するという
ことがなく，また，燃料や構造材との化学反応がない．さらに，ヘ
リウムガス，黒鉛とも放射化されにくいので，従事者の被ばく線量
が低く，放射性廃棄物の発生量が少ない．
