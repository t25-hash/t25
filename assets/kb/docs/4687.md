# 6・6・1 単相強制振動流型熱輸送デバイス

> **章**: 第6章 熱輸送デバイス
> **節**: 6・6 その他の熱輸送デバイス

この種の熱輸送デバイスには、図6・62に示したように、ドリームパイプ(dream pipe)とCOSMOS ヒートパイプ(counter-stream-mode oscillating flow heat pipe)とがある(双方とも提案者により名付けられた名称)。a. ドリームパイプドリームパイプは、1本の管でも図6・62のような管群でも成立する。管群の場合は、隣接流路間の振動流は同位相であり、隣接流路内の温度分布は相似であり、隣接流路間の熱的連成は起こらない。したがって、1本のドリームパイプの熱輸送量をとすると、本の管群で構成されたドリームパイプの熱輸送量はとなる。ドリームパイプの熱輸送特性については、流路壁の熱的影響を含めた解析があるが、円形流路で流路壁が断熱材の場合の近似解のみ以下に記す。流路半径、振動流の角周波数ω、液体の動粘性係数νにより定義される次の無次元数はウオマスリー数(Womersley number)と呼ばれる。図6・63に、=50mm,=1mmで、流路壁が断熱材である場合について、ドリームパイプの見掛けの熱伝導率と振動流周波数との関係を三種の液体に対して示した。図6・63よりわかるように、熱伝導率が高い液体が必ずしも最適な作動液体ではない。例えば、流路壁が断熱材である場合は~1程度の液体が、壁材の熱伝導率が液体の熱伝導率に比べて極めて高い場合はが小さい液体が高い見掛けの熱伝導率を実現する。b. COSMOS ヒートパイプCOSMOS ヒートパイプの名称は、counter-stream mode oscillating-flow heat pipeの略称であ図6・62 単相強制振動流型熱輸送デバイス能である。図6・63ドリームパイプにおける見掛けの熱伝導散率の周波数特性3―109る。COSMOS ヒートパイプは、ドリームパイプと同様に振動流の熱拡散促進効果を利用するが、図6・62に示したように密着した蛇行閉ループにより隣接流路間の熱的連成を活用する点で異なっている。COSMOS ヒートパイプにおける見掛けの熱伝導率は、積層された二次元流路で壁厚さが零の場合は次式で与えられる。COSMOS ヒートパイプは、流路壁を介した隣接流路間の熱移動を利用するので、流路壁材料の熱伝導率が低い場合や壁の厚さが厚い場合などは、ドリームパイプとの差が少なくなる。COSMOS ヒートパイプの熱伝導率は流路、液体、振動条件が同じドリームパイプのそれに比べて、作動液体が水の場合は低周波数で、水銀の場合は図6・63に示した周波数全域でかなり高い。c. 単相強制振動流型熱輸送デバイスの最適化単相強制振動流型熱輸送デバイスでは、振動流を駆動するためのエネルギー供給が必要である。したがって、例えば、単位長さで単位温度こう配を有する単相強制振動流型熱輸送デバイスにおける熱輸送量と振動流駆動仕事率との比を基準動作係数η[式(6・48)]と呼ぶと、基準動作係数を最大にするなどの最適化が必要となる。d. 単相強制振動流型熱輸送デバイスの特徴この熱輸送デバイスは、ウィック型ヒートパイプに比べて以下の特徴を有する。(1) 振動流振幅や周波数の制御により見掛けの熱伝導率の制御が容易にできる。(2) 作動液体の凝固点と沸点の範囲内であれば、熱輸送量に限界がない。(3) ウィックなど内部構造をもたないため、フレキシブル化が可3―110熱機器(4) 常識的周波数における最適な流路断面寸法は比較的小さく、細径熱輸送デバイスに適している。(5) 振動流を駆動するアクチュエータ等のオンオフにより、見掛けの熱伝導率が大きく変化でき、熱ダイオードとして機能する。
