# 4・5・3 高温ガス炉熱利用技術開発

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・5 高温ガス炉

高温ガス炉は，最高約1000℃の高温の熱が取出せるので，高効
率発電が可能となるだけでなく核熱エネルギーを多くの工業プロセ
スにも広く利用することができる．
a.高効率発電（high performance power generation) 発
電への利用では，在来火力発電並みの蒸気条件の蒸気タービン発電
で約40%の熱効率が得られるが，原子炉出口冷却材温度が850℃以
上ではガスタービン発電により45～50 %の熱効率が得られる．閉
サイクルのヘリウムガスタービンを高温ガス炉と結合させて，大規
模な発電を行うための研究開発として，ドイツで閉サイクルガスタ
ービンの開発が行われてきた．また，ガスタービンと蒸気タービン
との複合システムとして発電を行うことによっても約50 %の熱効
率が得られる．モジュール型高温ガス炉と閉サイクルガスタービン
を組合わせた発電プラントの概念設計が，アメリカ，ロシアを中心
にフランス，日本の企業の協力を得て実施されてきた．また，南ア
フリカ共和国の国営電力ESKOM社は，モジュール型高温ガス炉
ガスタービン発電プラントの概念設計を終了し，2005年初臨界を
目指した計画を進めている．
b.水素製造（hydrogen production) 水素は現在でも化学
工業の重要な原料であるが，本来燃料としても極めて優れた性質を
もっている．また，原料が水で無限にあり，燃料として使用すると
再び水に戻り有害物質を発生しないのでクリーンであり，貯蔵が容
易であり，家庭用から工業用，宇宙用，さらには自動車や飛行機な
どの燃料にも使用できる，といった特徴をもっている．高温ガス炉
の水素製造への有望な適用法としては，天然ガスの水蒸気改質法，
熱化学法による水からの水素製造，高温水蒸気の電気分解などがあ
る．天然ガスを用いる方法は，高温ガス炉の熱を利用してメタンと
水蒸気を反応させることによって水素と一酸化炭素を生成するもの
であり，日本原子力研究所でHTTRに実規模装置を接続して実証
する計画が検討されている．熱化学法は，水以外の物質を関与させ
て数段階の化学反応を利用して水を分解するものであるが，必要な
反応熱を高温ガス炉から供給するもので，化学反応に用いる反応物
質により，硫黄系サイクル，鉄-ハロゲン系サイクル等に分類でき
る．硫黄系サイクルの代表的なものとしてGA社が提案したサイ
クルおよび日本原子力研究所のISプロセスがあり，鉄-ハロゲン
系サイクルとして，東京大学でUT-3サイクルの研究が行われて
きた．
c .熱電併給システム（cogeneration supply system) 高温
ガス炉は発電以外にも高温の熱エネルギーを産業用熱源として広く
利用できるので，この特徴を最大限活用すれば，原子炉から得られ
た高温の熱を必要温度の高いプロセスから段階的に利用することに
より，極めて効率のよい熱の使い方が可能となる．例えば水素製
造，石炭ガス化等の高温プロセスに利用した後，一般工業，海水淡
水化，地域暖房などに利用し，さらに下がったところで農業，漁業
などに利用するなど，段階的に利用すると，熱の利用効率が飛躍的
に高められ，省資源，省エネルギーに役立つばかりでなく，エネル
ギー多消費産業からの環境汚染物質の放出量を低減でき，かつ，総
合熱利用率が70 %を超えるシステムとすることができる．わが国
では，日本原子力研究所が中心となってこのような効率的なシステ
ムの設計検討が行われている．
