b.原子炉システムの構成
.原子炉の構成 原子炉の構成を一次冷却材の循環方式で
分けると，自然循環方式と強制循環方式となる．自然循環方式は一
次冷却材の密度差により循環させるものであり，動的機器を使用し
ないため，単純化・簡素化が図られる．BWRでは炉心の沸騰によ
りボイドが発生し，これが浮力として作用し上昇力となる．炉心上
部では，気水分離器で蒸気と水が分離され，水は炉心周囲から上方
に伸びるダウンカマ部で給水と混合しながら下降し，炉心入口に達
する．このように，炉心発熱部とダウンカマ部を分けるシュラウド
の内外で密度差（外側で密度が高く，内側が低い）がつくことによ
り，自然循環が行われる．強制循環方式は，原子炉再循環ポンプで
一次冷却材を強制的に循環するため，原子炉圧力容器はコンパクト
となる．
.原子炉システム構成 BWRは，原子炉で直接蒸気を発
生してタービンに送る原子炉システムとなっている．他方，熱供給
（TOSBWR-200L）や一次冷却材との分離（SSBWR）などの目
的で，原子炉内に熱交換器もしくは蒸気発生器を内蔵した間接サイ
クル型のBWRも検討されている．
c.安全システム構成と特徴
.静的安全システム 原子力プラントの安全設計は，事故
を起こさないようにする予防保全を大前提とし，そのうえで事故が
起こった場合に対応する安全設計を行っている．安全システムは多
重化され，プラント外への影響が最小となるよう設計されている．
この安全システムは，止める（原子炉の停止)，冷やす（冷却)，閉
じ込める（放射性物質の格納）の機能に分けられる．
① 止める 原子炉の停止は，制御棒［ボロン（ホウ酸）やハ
フニウム］を炉心に挿入して中性子を吸収することにより，核反応
を停止させるのが基本である．この挿入が何らかの理由により行わ
れない場合，炉心にホウ酸水を注入して核反応を停止させる．現在
運転中のBWRでは，炉心の下部より水圧や電動で制御棒を出し
入れし，緊急停止時のスクラム操作は，蓄圧式のガス加圧系より水
圧で挿入させている．他方，一部のBWRでは，上部挿入の制御
棒駆動方式で重力落下式スクラムを行うことが検討されている．
② 冷やす BWRでは設計想定事故（DBA:design base
accident ）に至らない過渡事象の対応として，加圧防止や水位維持
等のために逃がし安全弁や原子炉隔離時冷却系（RCIC:reactor
core isolation cooling system）を設けてある．このRCIC機能を
静的に行うのがアイソレーションコンデンサ（IC）である．ICは
原子炉蒸気を水プール中に浸漬した熱交換器に導き，蒸気を冷却凝
縮した後に，原子炉に戻す機能を有している．この際，水プールの
水は蒸発し大気に放出される．
原子炉冷却材喪失（LOCA）時に原子炉を停止状態にするために
は，前記の制御棒挿入による核反応の停止のほかに，保有エネルギ
ーと発生する炉心の崩壊熱を冷却除去するため，緊急炉心冷却系
（ECCS）が設けられている．現在運転中の動的安全系を有する
BWRでは，動的な高圧注水系と低圧水系が設けられている．これ
に対し，静的安全系を有するBWRの場合には，保有水量の増加
などにより高圧注水系を削除し，低圧注水系のみとしている場合が
多い（SBWR系，LSBWR，SSBWRほか)．この低圧注水の駆動
力として，炉心上端からGDCSプールまでの水頭差を利用して行
うのが，重力落下式注水系（GDCS）である．LOCA時に格納容器
内に放出された蒸気を，格納容器内の圧力差を駆動力としてPCC
プール内の熱交換器に導き，冷却プールに移行した熱は蒸発して大
気に放出する静的な冷却システムが，静的格納容器冷却系
（PCCS）である．例として，SBWRのGDCSとPCCSの概要を
図4･48に示す．静的格納容器冷却系は，この例のほかに，格納容
器の外壁を水で冷却蒸発させるウォータウォール形式もある．
③ 閉じ込める 原子炉は事故時に原子炉から発生する放射性
物質を閉じ込めるため，PCVを設けている．このPCVの健全性
を維持するためにPCCS他を設けてある．このPCCSはシビアア
クシデント時にも使用される．PCCSのほかに，事故時に発生する
水素をパラジウム触媒により静的に水素と酸素を結合させて水にす
る静的可燃性ガス濃度制御系などが検討されている．
