# 7・3・3 ファン(fan)選定

> **章**: 第7章 ヒートシンク
> **節**: 7・3 電子機器設計とその実際

a. ファンの種類と特徴電子機器冷却に通常使用するファンは、ファンの外径が数cmから十数cmまでのものである。また、タイプとしては軸流ファンが最も一般的であり、その他に遠心ファンや横流ファンなどが使われることがある。図7・17にこれらのファンの風量(volumetric flow rate)・静圧(static pressure)特性を示す。7・3・2項で述べたように、風路を流れる冷却空気の流量は、このファンの風量・静圧特性と流路の圧損カーブとの交点であり、流路の抵抗が大きいときには、遠心タイプのファンが有利であり、逆に流路の抵抗が小さいときには、軸流タイプのファンが有利である。また、横流ファンは両者の中間に位置する。b. ファンの直列、並列運転特性電子機器筐体内の冷却を行う際に、複数のファンを用いて行うことがある。同一性能のファンを2台直列、または並列に使用した場合の風量・静圧特性は図7・18になる。並列に構成した場合の最大風量は、単体のほぼ2倍となる。ただし、実装時の風量は流路抵抗との兼合いにより決まるので、2倍とはならない。直列に構成した場合の最大静圧は、もし二つのファンの間の流路において空気流の漏れや回り込みがなければ、単体のほぼ2倍となると考えてよい。ただし、ファン2台を直接つなげた場合には、上流側ファンの出口で回収損失が発生するため、2倍とはならず約1.5倍に留まる。c. ファン選定手順ファンの選定は、おおむね以下に示すような手順で行う。(1) 冷却しようとする筐体での冷却必要風量と流路の圧損特性を求める(7・3・2項参照)。(2) 流路の圧損が比較的小さい場合→軸流ファンを選ぶ。(3) 流路の圧損が比較的大きく、かつ冷却する範囲が広い場合→横流ファンを選ぶ。(4) 流路の圧損が比較的大きく、かつ冷却する範囲が狭い場合→遠心ファンを選ぶ。このほかの要因としては、ファン騒音(accoustic noise)を考慮しなければならない場合が多い。ファン騒音は、一般的には軸流ファンが最も小さく、遠心ファンが最も大きい。また、風量・騒音特性は図7・19のようになる。軸流、横流ファンでは、風量に対して騒音が極小となる点(サージ点付近)が存在し、遠心ファンでは、風量とともに騒音は増大する。また、ファンが複数ある場合には単体に比べてファン騒音は増大する(例えば、同一性能のファンでは2台で+3.0dB,3台で+4.8dB,4台で+6.0dB である)。d. ファン使用上の留意点(1) 物体をファンの上流側に置く場合、羽根径の2分の1以上離す(騒音の発生の防止)。(2) ファンの吐出側から吸入側への漏れを極力少なくする(流量低下、風温上昇、回転音発生の防止)。図7・17ファンの風量と静圧の関係図7・18 ファンの風量-静圧特性図7・19ファンの違いに対する風量と静圧・騒音の関係(3) ファンのカタログにある騒音値は、通常騒音源から1m離れた点での測定値であり、例えば騒音源から300mmの所では、カタログ値よりも+10.5dB 大きな値となる(この点は、デスクトップやノートサイズの筐体の冷却設計において特に注意を要する)。近年の技術革新の流れはマイクロエレクトロニクスの分野ばかりではなく、パワーエレクトロニクスの分野でも目覚しい進展を見せ、ダイオード、サイリスタ、GTO(gate turn offtransistor)、IGBT(insulated gate bipolar transistor)素子などのパワー半導体素子においても高発熱密度化がますます進んでいる。そのため、これらの素子の冷却を効率良く行うことは装置全体の小型化・高信頼性化の面から非常に重要な技術課題となっている。これらの発熱を効率良く除去する手段として、従来からアルミニウムや銅製のヒートシンク(heat sink)を用いた空冷や水冷、沸騰冷却等がその熱量に応じて使用されてきたが、ヒートパイプによりその高性能化が図られている。可動部がなく無動力で大量の熱を輸送できるというヒートパイプの特徴から、ヒートパイプを使用することにより、構造がシンプルで信頼性が高く、軽量、小型のヒートシンクを得ることが可能となる。以下の各項では、これらパワーエレクトロニクス(power elec-tronics)の分野で使用されているヒートパイプ一体型ヒートシンクの実用例について述べる。
