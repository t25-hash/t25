# 4・7・2 開発の歴史

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・7 高速炉

a.世界における開発経緯と動向 表4･8に各国の高速炉開発
を示す．世界初の高速炉は1946年にアメリカロスアラモスで臨界
を達成した水銀冷却実験炉クレメンタイン（Clementine）である．
次のステップはアルゴンヌ国立研究所の設計になるNaK冷却高速
増殖炉EBR-Ⅰで，1951年12月世界初の原子力発電を行った．ガ
ス炉や軽水炉に先駆け，高速炉が最初の発電を行ったことは特筆に
価する．EBR-Ⅰで高速増殖炉の概念が実証された後，長期にわた
るエネルギー源を確保できる増殖炉の研究開発が基礎研究，国際共
同も含め各国で活発に行われた．EBR-Ⅰより30年後には，イギ
リスのPFR，フランスのフェニックス，ロシアのBN350，
BN600などの原型炉が次々と電力網に組込まれている．
アメリカでは，EBR-Ⅰ以降，実験炉としてフェルミ炉，EBR-
Ⅱ，SEFOR，FFTFが建設され試験された．1972年から原型炉
CRBRPの設計ならびに製作が進められていたが，1977年カータ
ー大統領の核不拡散声明による政策変更に伴い，建設費高騰もあっ
て1980年代初めにプロジェクト中止が決まった．当時は，天然ガ
ス，石炭をはじめとする資源の枯渇に対する脅威が遠のいたことな
ども中止の誘因でもあった．他方，アメリカ原子力規制委員会およ
びアメリカ電力会社のニーズに適応したモジュラ型高速炉プラント
PRISMの設計研究がGE社やRI社により進められたが，結局撤
退のやむなきに至った．その間，長期間安定に運転を継続していた
EBR-ⅡとFFTFも現在は閉鎖することになっている．
1980年代半ばになると，1950年代から積極的に開発を進めてき
たイギリス，原型炉SNR300を建設したドイツなど，経済性への
疑念などの理由から開発撤退が相次ぎ，チェルノブイリ事故も相ま
って，全世界的な高速炉開発のスローダウンは避けられない情勢で
あった．ヨーロッパ各国の実証炉計画も一時ヨーロッパ統合高速増
殖炉（EFR）計画に統合されたが，EFR計画もヨーロッパの諸事
情により中断されている．こうした状況において，1985年に臨界
に達し1986年に出力運転を開始した1240MW電気出力の大型高
速増殖炉スーパーフェニックス（super - phenix）を有するフラン
ス，1956年水銀冷却実験炉BR-2をはじめとする膨大な研究開発
を実施しつつ，1980年に臨界に達した600MW電気出力の原型炉
BN600を運転中のロシア（旧ソ連），1993年に初臨界を達成した
280MW電気出力の原型炉「もんじゅ」を開発した日本は，いわ
ゆる原子力開発先進国の中でも高速増殖炉路線を堅持した例外的な
存在であった．しかし，そのフランスも，現在スーパーフェニック
スの閉鎖を決定し，開発の見直しを行いつつある．一方，原型炉フ
ェニックスは10年間の寿命延長計画に基づいて現在設備改造を実
施中で，高性能燃料開発やプルトニウム，マイナアクチニド，長寿
命核分裂生成物の核変換などの試験研究を行うものとしている．
このように，高速増殖炉を取巻く状況は必ずしも芳しくない．ア
メリカにおける以下の動向は今後の高速炉開発の方向性を示してい
るものといえよう．すなわち，米ソ冷戦の終結による核軍縮からの
プルトニウムの余剰が問題となり，現状で建設費が軽水炉よりも高
い高速炉で，核不拡散上不用なプルトニウムをさらに増殖させるこ
とに疑義がもたれた．むしろ軽水炉使用済燃料の再処理から生じる
プルトニウムとNp，Am，Cm等のマイナアクチニド系物質を含
むTRU燃料を消滅させることこそ緊急の課題であるとして，増殖
炉よりもTRU消滅炉として目的を変更して方向転換することが打
ち出されてきた．そして小型固有安全炉の概念とともに，アクチニ
ドリサイクル炉に重点がおかれるようになってきている．このこと
は各国経済の好不況，財政状態，国民の意識という事情に依存して
高速炉の果たす役割が変化しうるということであり，現状では大容
量高速増殖炉の建設計画は見合わせているにすぎない．むしろ大事
なことは，高速炉というシステムが時代の要求の変化に応じてその
性能を発揮できるという，ポテンシャルの高さに注目すべきであろ
う．
b.日本における開発経緯と動向 わが国では，1960年代前半
から高速増殖炉の調査活動を開始し，精力的な調査研究の結果，
1966年プルトニウムとウランの混合酸化物燃料（MOX）を用いる
ナトリウム冷却高速増殖炉の開発のため，国産技術によって高速実
験炉および高速増殖炉原型炉（prototype reactor ）の建設プロジ
ェクトを発足させることになった．そしてその自主開発の一元的な
責任を担う機関として，1967年動力炉・核燃料開発事業団（動燃）
が設立されるとともに，大型の試験研究施設を集中的に設置した大
洗工学センターが開設され，国内の研究者や技術者を集めて，
FBRの炉物理，炉心設計，熱流動，高温構造材料，安全性分野な
どの基礎研究から工学的研究まで幅広く進められた．一方，MOX
燃料の開発は，動燃東海事業所で基礎試験，燃料製造技術，燃料設
計などが実施され，これらの経験を基に高速実験炉「常陽」，原型
炉「もんじゅ」の炉心燃料の製造を行った．一方，燃料や材料の照
射後試験施設が大洗工学センターに建設され，イギリスPFR，ア
メリカEBR-Ⅱ，フランスのフェニックス炉，「常陽」などの高速
炉で照射した燃料の照射後試験を行い，燃料の健全性や信頼性など
燃料の性能向上のための研究や試験が進められた．
高速実験炉（experimental reactor ）「常陽（Joyo）」と高速増
殖原型炉「もんじゅ（Monju）」の主要目を表4･9に示す．高速実
験炉「常陽」は1977年4月に初臨界，2年後に熱出力75MW
（MK-I炉心）に到達した．1983年3月には照射用に改造した
MK-II炉心で熱出力100MWの運転を開始した．「常陽」は初臨
界から現在まで重大なトラブルもなく順調に運転を続け，FBRプ
ラントの設計，建設，運転に関する技術的経験を蓄積するととも
に，設計の妥当性の実証，運転や保守技術の確立，プラント挙動の
把握，燃料や材料の照射挙動の解明などに多くの成果を上げ，

FBRの開発のための大きな役割を果たしている．現在，「常陽」の
照射性能向上を図るために，熱出力を140MWまで増大させるこ
となどを含むMK- III計画に基づき炉心改造が進められている．
FBR開発に関する国のプロジェクトの第2段階である高速増殖
原型炉「もんじゅ」の概念の構築や主要目の検討などの作業は
1968年から始められ，研究開発の成果や「常陽」の設計，建設，
運転で得られた多くの技術的知見や経験，設計データの蓄積，設計
手法や安全評価法の進展が反映されたことはいうまでもない．ま
た，開発を進めるに当たって，安全設計方針，高温構造設計方針，
耐震設計方針などの安全技術指針類が整備されたが，これらは今後
のFBRの開発に対して重要な基盤になっている．
もんじゅ」は1985年福井県敦賀半島の白木地区で建設を開始，
1994年4月に初臨界を，1995年8月29日には初発電（電気出力
5%，熱出力40 %）を達成した．しかしながら，性能試験中の同年
12月8日に二次系主配管温度計部からナトリウム漏えい（sodium
leakage）が発生し（「もんじゅ」二次系ナトリウム漏えい事故），
2005年現在まで，運転を停止中である．ナトリウム漏えいの原因
は，ナトリウムの流れによって温度計さや管の両サイドに生じた双
子渦による連成振動の結果，金属疲労でさや管が折損したものであ
った．また，1997年3月に発生した東海再処理工場のアスファル
ト固化施設における火災爆発事故などを契機に動燃の改革が検討さ
れ，その結果1998年10月1日に特殊法人核燃料サイクル開発機構
（サイクル機構，JNC:Japan Nuclear Cycle Development Insti-
tute）が新たに発足した．
もんじゅ」に続く高速増殖炉実証炉（demonstration reactor ）
およびその延長線上にある実用炉の開発目標は，現在実用化されて
いる軽水炉と同程度の安全性，信頼性，運転保守性を確保し，経済
的に競合できるプラントを実用化することである．こうした目標に
沿って，電気事業者が主体となった実証炉設計が進められ，新たな
革新的技術を積極的に取入れた電気出力約660MWのトップエン
トリー方式ループ型炉の提案が行われた．その後1998年に国が定
めた核燃料サイクル開発機構の業務に関する基本方針を踏まえ，サ
イクル機構の中長期事業の一環となる実用化戦略調査研究に日本の
高速炉開発の関係者（電力会社，メーカ各社）を集め，2015年頃
をめどに実用化の可能性が高い高速炉プラントシステムとサイクル
技術を提示することを目指して，サイクル機構にて一元的な研究開
発が進められている．
