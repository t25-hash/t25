# 7・4・1 大容量半導体素子冷却用ヒートパイプ・ヒートシンク

> **章**: 第7章 ヒートシンク
> **節**: 7・4 ヒートパイプ一体型ヒートシンク

モータの回転数制御用インバータ装置などで使用される大容量半導体素子(IGBT 素子など)の発生熱量は増加の一途をたどっている。一方、これらの素子を内蔵したモジュールを収納する制御盤の小型・薄型化が望まれており、高性能・薄型のヒートシンクとしてヒートパイプを使用したものが用いられるようになってきている。図7・20にヒートパイプ・ヒートシンク(heat pipe heat sink)の構成を示す。ヒートシンクの上部(凝縮部)は空調機に使用されている多数本の銅パイプと薄いアルミフィンからなるクロスフィンチューブ型熱交換器からなり外気で強制冷却される。下部(蒸発部)は半導体素子が搭載されるアルミブロックからなり、熱交換器からの銅パイプが上方から挿入され拡管により密着している。銅パイプは内部の作動流体として水が封入されており、ヒートパイプ⇦6mm左へずらしてある図7・20 ヒートパイプ・ヒートシンクの構成図7・21 放熱能力計算モデル3―118熱機器(二相密閉型熱サイホン、two-phase closed loop thermosyphon)として動作する構成となっている。a. ヒートシンクの放熱能力ヒートシンクの放熱能力を計算するための熱回路網モデルを図7・21に示す。No.1~No.3のヒートシンクの中では凝縮部および蒸発部における伝熱面積が大きく、凝縮部での風量も最も大きなNo.3が最も小さな熱抵抗を示している。比較したくし型フィンは全長300mm、幅265mm、フィン高さ50mm、ベース板厚み10mm、フィン厚み1mm、フィン枚数88枚で熱抵抗0.04K/W のものである。図7・23からヒートパイプ・ヒートシンクの送風動力当たりの冷却能力はくし型フィンヒートシンクと同等であるものの、ヒートシンクの全体積当たりの冷却能力ははるかに大きく、ヒートパイプ・ヒートシンクはコンパクト性能に優れていることがわかる。
