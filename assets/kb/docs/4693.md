# 7・4・2 ヒートパイプ・ヒートシンクの応用例1

> **章**: 第7章 ヒートシンク
> **節**: 7・4 ヒートパイプ一体型ヒートシンク

従来のデスクトップPC(パーソナルコンピュータ)には,ファンをヒートシンクの上部に搭載した,いわゆるCPU(centralprocessing unit)クーラが用いられ,サーバやワークステーションには,押出しやダイキャストによって作られたヒートシンクをMPU(micro processing unit)上に取付け,シャシに取付けたシステムファンにより,系外に排熱されてきた.近年,MPU の性能は,周波数がGHz オーダに達し,発熱量も70~100W に達する.この領域では,MPU の発熱量が大きくなるのに対して,MPU のサイズは大きくはならない.回路設計も0.25~0.18 mと,線サイズはますます微細化している.しかしながら,最終放熱に使われるヒートシンクは,放熱量に比例して大きくならざるを得ない.この場合,ヒートシンクの端部はMPU からの熱が十分伝わらずに,たいへん効率の悪い部分が生じる.そこで,ヒートシンクのベースに二次元的に伝熱拡散可能な平板型ヒートパイプを組込んだヒートパイプ・ヒートシンクが有用と考えられる.平板型ヒートパイプの熱性能は,加熱部の熱流束で50W/cm,固有熱抵抗で0.05K/Wを必要とする.厚さはせいぜい5mm以下とすべきである.まず,平板型ヒートパイプのウィック構造としては,加工性の良い銅粒をヒートパイプコンテナ内面に焼結し使用する.また,ヒートパイプは,どのような傾斜角度に設置しても,作動する必要がある.熱源の面積とヒートシンクのベースの面積の比を基準に,熱拡散抵抗を評価することが重要である.つまり,どういう比率では,平板型ヒートパイプの効果がどの程度現れるかを,モデリングと実験的検証の両面から考察を加える必要がある.図7・24に平板型ヒートパイプを組込んだヒートシンクの実施例を示す.図7・25に示すように,垂直にヒートパイプを配置して,下部にはヒートシンクのベースを接合し,凝縮部には多数のフィンを挿入したヒートパイプ・ヒートシンクが実用されている.ヒートパイプは,軸方向に均一な温度であるため,大型のサーバ等の発熱量が大きく,冷却風の圧力損失を小さく抑える装置の冷却に適している.図7・26に示すものは,1U サーバのように,高さ方向にスペースのない筐体内の素子を冷却する場合に適している.発熱素子の直上スペースでは,十分な能力のヒートシンクは取付けられない.そ図7・24 平板型ヒートパイプを利用したヒートシンク図7・25タワー型ヒートパイプ・ヒートシンク図7・26 ヒートパイプ式複合ヒートシンクの場合ヒートシンクのベースにヒートパイプを埋め込み,ヒートパイプ片端を取出して,別のヒートシンクに接続すると,熱抵抗を低減し,冷却能力を増加することができる.今後ますます発熱する,MPU の冷却に対しては,個々の冷却部品の高性能化,熱的結合の低熱抵抗化は,いうまでもなく,複雑かつ緻密な複合設計も必要であろう.
