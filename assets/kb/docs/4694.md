# 4・7・4 ナトリウム冷却高速炉システム概要と安全設計

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・7 高速炉

高速増殖炉には，炉心を収納する原子炉容器，一次ナトリウム循
環ポンプおよび中間熱交換器を配管で接続するループ型（loop-
type reactor ）と，これら一次冷却系全体を容器の中に納めるタン
ク型と呼ばれる二つの形式がある．タンク型（tank or pool type
reactor ）は，一次系機器の保守点検，自然循環などによる熱輸送
経路の特定，耐震上やや難点があるが，わが国を除いた実証炉の設
計では，タンク型炉の採用が圧倒的に多い．このほか両形式を併せ
たハイブリッド型，二重タンク型や二次冷却系の削除も検討されて
いる．「常陽」や「もんじゅ」，アメリカのFFTF，CRBRP，ドイ
ツの原型炉SNR300などはループ型である．タンク型の代表的な
ものは，フランスのフェニックス，スーパーフェニックス，アメリ
カのEBR-Ⅱ，イギリスのPFR，旧ソ連のBN600などがある．
高速増殖炉の原子力発電所プラントを構成する系統設備のうちの
主要なものについて，ループ型高速増殖炉原型炉「もんじゅ」を例
にとって説明する．プラントシステムは基本的に原子炉本体と炉
心，原子炉冷却系，発電設備，工学的安全設備，補助冷却系，ナト
リウム補助系，アルゴンガス系，燃料取扱系と貯蔵設備と計測制御
系などから構成される．プラント全体構成図を図4･53に示す．
原子炉は炉心，その支持および炉心の流量配分を行うための炉内
構造物，制御棒駆動機構等の案内および支持のための炉心上部機構
からなる．炉心は燃料集合体（炉心燃料，ブランケット燃料），制
御棒集合体，中性子反射体（中性子遮へい体）などによって構成さ
れる．炉容器の上部には遮へいプラグを設置し，ナトリウム液面上
部をアルゴンガス雰囲気に保つ．遮へいプラグには炉心上部機構，
燃料交換機等が取付けられる．さらに，万一，原子炉容器でナトリ
ウム冷却材漏えい事故を想定しても炉心の崩壊熱除去に必要なナト

る．
補助冷却系は，原子炉を通常停止したときや運転中の異常および
事故により原子炉が停止したときに，核分裂生成物の崩壊熱や他の
残留熱を強制循環または自然循環で除去し，炉心を冷却するもので
ある．補助冷却系の運転時は，中間熱交換器において一次冷却材か
ら二次冷却材に伝達された熱が二次冷却系より分岐して蒸気発生器
に並列に配置された空気冷却器により大気中に放散される．
ナトリウム補助系としては，原子炉容器などの一次冷却系と二次
冷却系のナトリウム液位の制御，ナトリウムの純化，機器や配管へ
の充てん・ドレンおよびドレンしたナトリウムの貯蔵を行うもの
で，一次系と二次系のそれぞれにある．
高速増殖炉の安全設計（safety design）の考え方は，他の原子
炉施設と同様，深層防護（defense-in-depth）の考えに従う．第1
のレベルは異常状態の発生を防止することで，異常が起こらないよ
うに万全を期し，第2のレベルではたとえ異常が発生しても，事故
に拡大させないことで早期に異常を検出し，対策を講じる方策が施
されている．特に，原子炉を確実に停止すること，また原子炉停止
後，崩壊熱を確実に除去することが重要である．第3のレベルは，
万一事故が起こったとしても，一般公衆に影響を及ぼさないよう，
放射性物質を格納容器によって封じ込める対策を講じる．工学的安
全防護施設（engineering safety guard system）は，原子炉設備の
損傷，故障等の原因で原子炉内の燃料の破損等により多量の放射性
物質が環境に放散される可能性がある場合に，これらを抑制または
防止するための機能を備えた施設である．原子炉本体や一次冷却系
等の原子炉施設の主要部分を収容する原子炉格納施設，原子炉格納
施設とそれを取り囲む外部遮へい建物との間のアニュラス部の空気
を循環，排気するアニュラス循環・排気装置ガードベッセル，補助
冷却系および一次アルゴンガス漏えい事故のときに放出される放射
性物質の量を抑制するための活性炭吸着塔を備えた一次アルゴンガ
ス系収納施設などがある．なお，低圧系である高速炉システムで
は，軽水炉のように高圧注水系などのECCSを必要としない．そ
の代わり冷却材喪失を防ぐという意味でガードベッセルがECCS
の機能を果たす．
