# 7・5・2 筐体内の性能

> **章**: 第7章 ヒートシンク
> **節**: 7・31 )を取上げ、その自然空冷と強制空冷それぞれの場合の代表的

筐体(cabinet)内のヒートシンク(heat sink)の性能は他の電子部品や隣接する基板、ケーシング等の影響を受ける。これは冷却空気の流れ場が他の部品の影響を受けるためであるが、特に強制空冷下にある圧力損失の大きいフィンの場合、フィンの性能は周囲の空間の有無に影響される。これは冷却空気がフィンの中を通過する流れとフィン周囲の空間に迂回する流れとに分かれるからである。ここでは図7・35に示すように、強制空冷されるプレートフィン(plate fin)の両側に空間があり、冷却空気に迂回流が生じる場合のプレートフィンの性能予測法について述べる。なおフィン先端には隣接する基板やケーシング等があり、フィン先端での迂回流はないものとする。プレートフィンの冷却性能は、フィン間の平均流速の関数として式(7・35)で与えられる。それゆえ、問題はフィン間平均流速を求めることに帰着する。以下に強制空冷されるプレートフィンの両側に空間があり、冷却空気に迂回流が生じる場合のフィン間平均流速の予測法を述べる。フィン間およびフィン外部の流れのエネルギー収支を考えるために、図7・36に示す検査領域A,B をとる。なお図7・36ではフィン側壁面と側方に存在するフィンあるいは筐体壁とのすきまの半値である。領域A,B の間で質量の流入出がないように両領域間の境界は流線に完全に一致させる。また両領域の上流側境界における速度、圧力はともに,とする。流れに平行な上下境界は対称境界とし、エネルギーの流入出はない。以上より式(7・37)(7・38)を解くと次の関係が得られる。したがって、上式を満足するを求めることによって、式(7・35)からプレートフィンの熱伝達率を評価することができる。
