# 7・6・3 機械加工法

> **章**: 第7章 ヒートシンク
> **節**: 7・6 節の製造法に関する記述では,素材としてアルミニウムを想定

機械加工(machining)は切削,研削に大別できる.工具コストは比較的低く,形状,寸法とも選択範囲は広いが除去加工のため材料歩留りが低い.研削は微細フィン(ピッチ0.4mm,アスペクト比50)の成形を可能とするが低加工能率のため工業製品への適用図7・44 鋳造加工図7・43 切削加工図7・45 接合法(ろう付け)は少ない.a. 切削加工(cutting,図7・43)フライス盤,旋盤が代表的で前者によるプレートフィン,後者によるラジアルフィンが主たる成形品である.自動化によりコスト削減が進んでいる.成形可能なフィンピッチ下限は1.5mm程度,アスペクト比は最大約30である.ピンフィンではアスペクト比10以上のものは成形が困難である.b. スカイブ加工(skiving)特殊切削工具により素材ブロックからわん曲したプレートフィンを切り起こす.切削加工と塑性加工の組合せ.厚さ0.3mm程度の薄いフィンを50以上の高アスペクト比で成形できる.
