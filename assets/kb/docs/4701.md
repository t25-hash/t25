# 4・9・4 燃料加工（fuel fabrication）

a.燃料加工技術の概要 燃料加工では，各種原子炉の装荷に
適した燃料集合体の製造を行う．燃料の化学形態から，大きく金属
（合金）燃料，セラミック燃料（酸化物，窒化物，炭化物），液体燃
料に区別できる．また，燃料の形状からは，棒状のスラグ燃料，ペ
レット燃料（pellet fuel ），粉末を充てんする振動充てん燃料，被
覆粒子燃料等に，燃料の成分によって，ウラン燃料，プルトニウム
燃料（ウランとプルトニウムを混ぜたもの），トリウム燃料等に分
類することもできる．
現在，世界の原子力発電の主流は軽水炉であり，その燃料は，ウ
ラン酸化物（UO ）でペレット形状である．このほか，軽水炉再処
理サイクルではMOX燃料のペレット形を，高速炉サイクルでは
金属燃料や酸化物燃料を中心に開発を進めているが，さらに窒化物
燃料や振動充てん燃料等多様な燃料概念の研究も進めている．
わが国では，1970年頃の軽水炉の本格的な導入に合わせて，
1970年前後から濃縮ウラン燃料（U濃縮度3～5 %のペレット形
燃料）の製造が始まった．その後，自主技術の積極的な採用を進
め，燃料性能の向上や品質保証の向上に努め，世界的にも最も破損
率の低い水準を保っている．
代表的な軽水炉用ウラン燃料の加工はおもに次の工程から構成さ
れている．
① 濃縮工場から払い出されたUFの酸化物（UO ）への再転換
② UO粉末を成形，焼結してペレットへの加工
③ ペレットの燃料棒への装てん
④ 燃料棒の燃料集合体への組立
このおもな工程以外に，各製造工程が終了するごとに検査工程が
控えており，原材料から最終製品に至るまで品質保証を行ってい
る．燃料加工の工程フローを図4･61に示す．

d.ガス炉燃料の製造 被覆粒子燃料（coated particle fuel ）
はガス炉用の燃料として開発され，燃料核と呼ばれる直径約
200～600mの微小のセラミック燃料球を熱分解炭素（PyC）や炭
化ケイ素（SiC）等で被覆したものでできており，代表的なものに
三重ないし四重被覆で構成されたTRISO形燃料がある．被覆層は
核分裂生成物（FP:fission products ）ガスの閉込め，燃料核のス
ウェリングの吸収，被覆粒子全体の強度保持等の機能を有してい
る．各被覆層の厚さは，数十～60m程度である．
