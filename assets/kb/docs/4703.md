# 4・9・6 使用済燃料および高レベル廃棄物（high level waste）

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・9 核燃料サイクル

蔵（interim storage）した後，最終処分場の真上にあるコンディ集合体を封入する．
中間貯蔵方法としては，キャスタ貯蔵方式，ボ理サイクルの場合は，ガラス固化体を同じく30～50年冷却貯蔵しクはエレベータにより地下数百mよりもさらに深い処分場に運び，使用済燃料またはガラス固化体（vitrified waste）中の放射性物化体の場合はガラス，ステンレス製キャニスタ，オーバパック，緩アを組合わせた多重バリアの概念が用いられている．
圧縮ベントナてんする自己シール性，放射性核種を吸着し移行を遅延させる性て優れた性能を有している．
サイクル機構を中心に関係機関が協力して検討を進めてきた高レ成果を受けて，2000年には地層処分基準の整備，サイト選定，処電環境整備機構が設立された．
西村友宏・ほか2名,サイクル機構技報,12別冊,日本原燃株式会社パンフレット,六ケ所再処理工場の概要,ワンススルーでは，使用済燃料（spent fuel）を30～50年中間貯ショニング施設に搬入され，処分容器にBWRで10～40体程度のールト貯蔵方式，サイロ貯蔵方式，水プール貯蔵方式がある．
再処た後，炭素鋼オーバパックに収納する．
処分容器およびオーバパッ埋設する．
ベルガラス固化体の地層処分（geological disposal）の研究開発の分施設の建設を行う機関として電力業界の出資による(財)原子力発指している．
