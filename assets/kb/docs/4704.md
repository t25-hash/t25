# 7・6・6 その他の加工法

> **章**: 第7章 ヒートシンク
> **節**: 7・6 節の製造法に関する記述では、素材としてアルミニウムを想定

エッチング等LSI製造技術の応用によるマイクロマシニング技術で、微細寸法フィンをもち単位体積当たりの伝熱面積が従来よりはるかに大きなヒートシンクの加工も可能となりつつある。その伝熱特性や工業的な製造可能性を含め、冷却部品としての有効性が検討されつつある。ヒートシンク(heat sink)の形状、寸法、使用形態(自然対流か強制対流かなど)は多種多様であり、7・5節で述べた性能予測法でその性能を把握できるヒートシンクは限られている。このため、複雑な形状のヒートシンクはもちろん、単純形状のものでも実際の使用形態に則した性能把握が必要な場合は実測する必要が生じる。設計現場では通常、伝熱(冷却)性能を式(7・43)の熱抵抗によ3―125図7・46 性能測定におけるヒートシンクとヒータの接続部り評価する。=욁-(7・43)ここで욁,,は各々ヒートシンク基板温度、雰囲気温度、ヒートシンクへの単位時間当たり流入熱量で、これらが測定量となる。また、空気流量(流速)やヒートシンクの流動性能として圧力損失が必要な場合は、特定位置の静圧も測定量となる。
