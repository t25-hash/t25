# 4・1・1 形式，特徴，用途

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・1 往復ポンプ

往復ポンプ（reciprocating pump）は容積形ポンプに属し，ピを行うもので，駆動方式により蒸気直動式，クランク式，ダイアフ圧力は約20～60MPa，歯車減速機を内蔵している．
図Ⅱ-4･3は蒸プピストンである．
ダイアフラム式は，補助ピストンの混入の許されない薬液注入ポンプなどに使用される．
MPaに達する高い吐出し圧を有するものも多いが，機構上，高速型となり，振動も大きいのでより強固な基礎が必要である．
繊維工業，合成樹脂工業等の高圧を必要とする用途に多く用いられれ，鉄鋼業でのデスケーリングポンプとして再認識されている．
気直動式複動2連ピストンポンプで， が蒸気ピストン， がポンまたはプランジャによって膜に往復動を与える構造で，漏れや異物往復ポンプは構造上，最も高圧作動に適するポンプで，数十回転が困難で，このため同一容量のターボポンプに比してかなり大往復ポンプは一般送水用のほか，石油精製，石油化学工業，合成ている．
また，最近はアンロード時の軸動力が小さいことが着目さお，油圧用については第Ⅳ部3章を参照のこと．
