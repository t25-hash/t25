# 7・7・1 温度測定とヒータ

> **章**: 第7章 ヒートシンク
> **節**: 7・7 性能測定法

ヒートシンクの性能測定では,自然対流,強制対流にかかわらず,ヒートシンクとその雰囲気温度の測定が必要となる.その際,一般的な温度計は熱電対で,T 型(銅-コンスタンタン)を用いることが多い.これはヒートシンクの場合,測定温度はたかだか100°C台であり,この温度域ではT 型の温度-起電力間の線形性が高く精度が高いためである.ヒートシンクの温度測定点は,目的により異なる.機器の熱設計のため,最高温度のみ必要な場合は熱源付近の一点を測ればよく,基板平均温度が必要なら複数点の温度を測る.複数点の位置はガウスの数値積分点に対応した点,基板面を等分割した各中心点等とすればよいが,アルミニウム等,熱伝導率の高い基板ではこれら測定点配置の違いによる平均値の差は小さい.ヒートシンクへの熱電対固定方法はさまざまで,下記事例を参照していただきたい.ヒータはシート状のものを用い,ヒートシンク基板を等熱流束条件で加熱する場合が多い.熱流束(heat flux)はヒータ入力電力とヒータ面積から求める.この種の実験において誤差原因となるヒータ背面からの熱漏えい(heat leakage)は,ヒータ背面の断熱や2個のヒートシンクをヒータを挟んで面対称に設置することにより回避する.シート状ヒータの作成例を示す.マイカ板(厚さ約1mm)に等間隔かつ密にニクロム線(直径約0.1mm)を巻き回し,一側面から給電用リード線を引出した後,両面をさらにマイカ板(厚さ約0.2mm)で挟みアルミテープ(アルミ厚0.08mm,接着剤厚5mm)で密着固定すれば,厚さ約2mmのほぼ等熱流束条件を満たすヒータが得られる.図7・46にヒータと熱電対を取付けたヒートシンクの一例を示す.この例では,熱電対(T 型,素線径0.076mm)はヒートシンク基板にアルミテープで固定され,ヒータはシリコンコンパウンドの薄層(約0.3mm)を介してヒートシンク基板と密着接合されている.
