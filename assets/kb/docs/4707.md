# 7・7・3 強制対流下での性能測定と測定装置(風洞)

> **章**: 第7章 ヒートシンク
> **節**: 7・7 性能測定法

強制対流下の性能測定では、測定量として温度、入熱量のほかに流量(流速)、圧力損失も加わり性能測定装置として風洞(windtunnel)が必要となる。ここでは冷却空気がその基板と平行に流入する場合(以下、平行流と呼ぶ)と、基板へ垂直に流入する場合(以下、噴流と呼ぶ)の双方における強制対流下での性能測定可能な図7・48に示す風洞について述べる。風洞は押込み式(吐出し式)である。ここで風洞の押込み式と吸込み式の違いについて簡単に述べる。押込み式では、テストセクション上流のブロワ(ファン)による乱れの影響を受けやすい。吸込み式は外部気流の乱れを整流部で減じてテストセクションに導くことが容易で、流入部乱れの影響を排除した測定が可能である。一般に低乱風洞も吸込み式による。ただし、ヒートシンクの実際の使用条件を考えると押込み式による測定はあながち雑で妥当性を欠くとはいいがたい。図7・48に戻るが、空気体積流量は検定された入口ノズルとマノメータで測定する。この入口ノズルは高精度かつ自身の圧力損失が極めて小さいことを特徴とし、流量に応じて複数用意する。ブロワはインバータ制御で流量の制御応答性、安定性の点で優れているが、計測器へのノイズ対策を要する。平行流、噴流用いずれのテストセクションも防振ダクトを介してブロワ下流に取付ける。測定精度悪化の原因となる流量計-テストセクション間での空気漏えいがないよう注意する。ブロワを小型化してコンパクトな風洞とするためには、テストセクション以外での圧力損失を極力抑えることが有効である。図7・49は平行流用のアクリル樹脂製テストセクションで、外側を発泡スチロールで断熱している。空気出口温度測定部上流にはミキサを設け、混合平均温度が測定できるよう配慮する。ヒートシン(c)(b)(a)(f)(e)(d)(a)~(c) 自然対流でのヒートシンク姿勢の違い(d)自然対流でのヒートシンクと壁面との距離(e)~(f) ファン付ヒートシンクにおけるファンの向きの違いとファン-ヒートシンク間距離図7・47 自然対流下およびファン付ヒートシンクの性能測定に影響を及ぼす実装条件図7・48 ヒートシンク性能測定装置[押込み式(吐出し式)風洞]クによる圧力損失は、ヒートシンク上下流に設けた圧力タップ間の差圧をマノメータで測定する。図7・50は噴流用テストセクションで、ブロワと接続したアクリル樹脂製のプレナム端部にオリフィス状噴流ノズルが設けられる。ヒートシンクはフィン先端がノズルと一定距離を保つよう固定される。ヒートシンクとヒータとの接合およびこれらのノズルへの固定はねじ式ジグを用い、ねじの締付け力の調整によりヒートシンクとヒータとの接触圧力が一定となるよう留意する。
