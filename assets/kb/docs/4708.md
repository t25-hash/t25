# 4・2・2 スクリュー形ポンプ（screw pump)

> **章**: 第4章 容積形ポンプおよび特殊ポンプ
> **節**: 4・2 回転容積形ポンプ

ロータの回転に伴い，回転軸方向にポンプ室が移動するポンプで
ある．ユニバーサルジョイントをもち，回転軸と並行偏心回転をす
る一軸のスクリューポンプと，タイミングギヤをもつ二軸スクリュ
ーポンプ，主軸と二つの縦動軸をもちタイミングギヤが不要な三軸
スクリューポンプがある．
二軸，三軸スクリューポンプは，ギヤポンプと同様に剛体のロー
タが直接互いに，または剛体のポンプ室内壁と接触しゅう動するた
めに固形分を含まない高粘度流体の移送に適する（第Ⅴ部2･2節
「油圧ポンプ・モータ」参照）．
一軸スクリューポンプは，一般にポンプ室内壁（ステータ）が弾
性体にて作られるため，水分の少ない脱水ケーキのような個液混相
流も移送可能である．スクリューポンプは，ポンプ室が軸方向に独
立して重なる多段構造であるため，部屋数が多いと1段当たりの差
圧が小さくなるので，長くしてポンプ室を増やし，高圧を得ること
ができる．図Ⅱ-4･13のように一軸スクリューポンプは，ロータ中
心軸に対して円を偏心させてらせん状に配置（軸直交断面が円形の
おねじ）し，偏心量の2倍の半径に内接させて回転させると，円の
軌跡（ハイポサイクロイド曲線）が直線になることから，おのおの
の断面でロータがステータの中を自転しながら往復運動をすること
(ａ)一軸スクリューポンプの作動原理
でポンプ作用をする．ポンプ室はねじれた空間となり，1/2リード
ごとに隣り合って重なる ．
スクリュー形ポンプの一種にアルキメデスポンプがあり，傾斜さ
せた円筒の内壁に接する，らせん状の板により仕切られたポンプ室
をもつ（図Ⅱ-4･14）．他の面に密閉されていないが，空気との自由
境界面によりポンプ室を構成する．傾いてしか使えないが，構造が
簡単であり大量の土砂を含む水の揚水が可能である．
