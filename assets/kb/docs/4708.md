# 7・7・4 性能測定の例

> **章**: 第7章 ヒートシンク
> **節**: 7・7 性能測定法

性能測定装置の具体例として,各種流れ場に対するヒートシンク性能測定用の実験装置の概要を示す.a. 主流中でのヒートシンク単体の冷却性能この場合に対する風洞は吸込み式であり(図7・51),空気流量はJIS 標準オリフィスを用いて測定する.テストセクションは風洞の前縁に位置し,その流路断面形状は90mm角とヒートシンクの幅22mmに比べて十分に大きい.冷却風は入口ノズル部で一様流速化され,テストセクション,ブロアを通って,外気に放出される.ヒートシンクからの熱の漏れを最小とするため薄いシート状のヒータの両面に同じ形状のヒートシンクを2枚張り合わせ,細い針により空中に保持してある.ヒートシンクの温度測定用の熱電対(T 型)は,ベース部中央に埋め込んである.また,風洞入口部に空気温度測定用の熱電対(T 型)が設置されている.b. 拘束された平行流れ場でのヒートシンク群の冷却性能こ図7・49平行流用テストセクション図7・50 噴流用テストセクション図7・51 ヒートシンク単体用実験装置図7・52 パッケージ群用実験装置図7・53 噴流冷却用実験装置の場合に対する風洞も吸込み式であり(図7・52),空気流量はJIS標準オリフィスを用いて測定する.テストセクションは風洞の前縁側に位置し,その流路幅は実際の電子回路基板と同じである.また,その高さは種々の基板間隔での設置状態を模擬できるように可変としてある.テストセクションはアクリル製であり,中に配列したパッケージ群が見えるようになっている.冷却風は入口に設けられた1/4円ノズルで一様流速化され,テストセクション,ブロワを通って,外気に放出される.模擬パッケージはLSIと同寸大の断熱材とその上部にある薄いシート状のヒータからなり,その上面にヒートシンクが接着されている.ヒートシンクの温度測定用の熱電対(T 型)は,ベース部中央に埋め込んである.また,風洞入口部に空気温度測定用の熱電対(T 型)が設置されている.c. 噴流におけるヒートシンク単体の冷却性能この場合に対する風洞は吐出し式であり(図7・53),空気流量は整流洞(出口側プレナム)とJIS 標準ノズルを用いて測定する.テストセクションは風洞の前縁に位置し,その流路断面形状は幅が250mmとヒートシンクの幅60mmに比べて十分に大きい.さらに,その高さはヒートシンクのフィン高さ욞,ヒートシンクの上端部と噴流孔との距離(ギャップ)などの影響を検討できるように可変としてある.また噴流チャンバの形状は250×250mmであり,その中には整流金網が設けられている.冷却風は噴流チャンバ内で一様流速化され,噴流孔から流出し,テストセクション,整流洞を通って,外気に放出される.なお,平板フィンの向きが冷却風の流れの方向と同じになるようにヒートシンクを設置してある.さらに,ヒートシンクと上流側の壁との距離,ヒートシンクと側壁との距離を十分に取り,フィンから出た冷却風がダクトの影響を受けにくくしてある.ヒートシンクはフィルムヒータにより加熱量が一定になるように加熱され,その下部の銅板に9個の温度測定点(校正済みの線径が0.1mmのT 型熱電対)が設けられている.なお,9個の測定点は,平均温度が正確に測定できるようにガウス・ルジャンドルの数値積分点に対応した位置に設けられている.それらの重み付けした平均値をヒートシンクのベース温度として測定する.なお,銅板の下部は断熱してある.本測定装置を用いた伝熱実験での9点の測定値の分布幅(平均値と最大値,最小値の差)は平均温度上昇値に対し,+9%~-6%で3―127あり,60mm角程度の大きさのヒートシンクに対しては,極端な差はない.なお,同一ヒートシンクに対する相対温度の測定値のばらつきは3%程度である.圧力損失としては噴流チャンバとテストセクション下流の静圧差を測定する.微差圧計の分解能は平均差圧500Paに対し,1Pa程度であり,圧力損失の測定値のばらつきは2%程度である.文献(1)
