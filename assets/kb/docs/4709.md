# 8・1・1 蓄熱の目的とその分類

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・1 熱エネルギー需給と熱エネルギー貯蔵技術の現状

a. 蓄熱(heat storage,thermal storage)の目的熱エネルギーの貯蔵は,古くから生活の知恵として行われてきたが,工業や民生用として蓄熱技術が再認識されたのは,そう昔のことではない.蓄熱は,超高温から極低温の広範囲にわたる熱の貯蔵技術で,その温度域,使用目的によって,さまざまな蓄熱材料や蓄熱方式がある.蓄熱材料開発は,多くの候補材料から,その材料の特徴と特性を把握したうえで,さらに新たな特徴をもつ材料への改質や複数の材料の組合せなどを検討することを目的とする.最終的には,安全かつ経済性のある蓄熱材料を選定し,安価で高性能な蓄熱システムを構築することを目指すものである.図8・1は,各種エネルギーの単位質量当たりエネルギー貯蔵量の特徴を示したもので,蓄熱は総合的に判断して,中位のエネルギー貯蔵法に位置付けられる.他のエネルギーに比較して,利用の多いエネルギー形態である熱エネルギーを直接入出力できるので,利用範囲が広く,貯蔵法が比較的簡単な特徴を有する.蓄熱槽は,熱源(供給)側と熱負荷(利用)側との中間に介在する,ある種の伝熱機器である.その機能としては,(1)熱源側の時間的,量的,質的な変動の平準化と負荷側へ安定かつ均質な熱エネルギーの供給,(2)負荷側に生じる一時的かつ過大な熱源需要の平準化と熱源における過負荷の防止,(3)熱発生と熱利用の間に生じる時間差(位相差)の調整,(4)量的な分散化,温度や汚れにおける低質化した熱エネルギーの集中・高質化などがある.蓄熱システムの導入は,第一に捨てられている熱エネルギーを蓄えたり,温度水準や熱量を調整して再活用することができる.第二に,ロードピークに相当する供給源・流通源が不要となるために設備コストが軽減できる.ただし,この場合は入出力装置費,蓄熱槽費,貯蔵材料費などのコストが掛かるのでそれらの兼合いとなる.第三に,随時消費に対して熱エネルギーを備蓄しておくことにより,余分なエネルギーコストを軽減できる.第四に,安価な夜間電力を利用した蓄熱や排熱の蓄熱など,安価なエネルギーを利用できる.さらに,蓄熱システムの導入によってエネルギーシステムにおける利用効率の改善が図られる.また,蓄熱技術の利用として,熱機器に蓄熱部を設けることにより,機器全体の効率を高めることも可能である.したがって,蓄熱技術は,将来性のある極めて高度な省エネルギー技術の一つである.これら蓄熱システム構築に最も重要となるものが,蓄熱材料である.b.
