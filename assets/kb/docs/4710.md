# 8・1・2 各種の蓄熱材開発の現状

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・1 は,蓄熱物質の種類と分類を示したものである.

ここでは,前述の蓄熱材料の特徴とその開発動向について概説する.a. 顕熱蓄熱材料顕熱蓄熱材料には,(1)単位容量当たりの蓄熱量(比熱×質量×温度差)が大きい,(2)単位蓄熱量当たりの蓄熱材料および蓄熱槽容器が安価なこと,(3)化学的に安定な物質で,腐食性および毒性がなく,廃棄に際して公害が生じないこと,(4)熱の出し入れが容易であること,などが主として挙げられる.顕熱蓄熱材は技術的にも取扱いやすく,蓄熱性能も優れており,現在普及している蓄熱材の多くはこの種のものであり,特に水は廉価かつ安全で熱容量が大きく,温度制限はあるものの蓄熱材としては適した物質である.b. 潜熱蓄熱材料物質の相変化に伴う潜熱を主として利用する潜熱蓄熱材料に要求される性質は,(1)単位体積当たりの潜熱量が大きく使用温度範囲内にある,(2)固-液相変化時の体積変化および蒸気圧が小さい,(3)熱伝導率が大きく熱応答性が良い,(4)相変化に伴う過冷却や相分離がない,(5)安全性,耐久性そして経済性に富む,などである.利用に当たって,利用温度に近い相変化温度をもつ潜熱蓄熱材を選び,顕熱分を少なくし,その大きな潜熱量で蓄熱槽の小型化とともに熱損失の低減を図る.熱の取出しは,相変化特性から一定温度で可能となる.潜熱蓄熱材料として,無機系固-液相変化材料,有機系固-液相変化材料,気-液相変化材料や固-気相変化材料などがあり,固-液相変化材料などは球,円筒そして矩形状のカプセルに充てんされて使用される場合が多い.c. 濃度差型蓄熱材(concentration difference type heat stor-age material)濃度差型蓄熱材は,溶媒としての吸収剤溶液の濃度や吸着剤の濃度と溶質(一般に蒸気)の濃度差に基づく自由エネルギーの変化により,熱エネルギーを貯蔵するものである.溶媒と溶質を分離することにより,熱損失なしに長期間の蓄熱が可能となる.濃度差型蓄熱材には,吸収剤を利用する吸収型と吸着剤を利用する吸着型がある.吸収型においては,吸収剤溶液に対する溶質蒸気の吸収発熱・加熱蒸発と溶質蒸気の蒸発吸熱・凝縮発熱過程を,吸着型では,吸着剤に対する溶質蒸気の吸着発熱・加熱脱着と溶質蒸気の蒸発吸熱・凝縮発熱過程を,それぞれ組合わせることにより,吸収式や吸着式ヒートポンプまたは冷凍機を形成できる.濃度差型蓄熱材は,排熱や太陽熱などの自然エネルギーなどを大量に短期・長期間にわたって蓄熱が可能である.d. 蓄熱・蓄冷熱機能を有する熱流体従来の固-液相変化潜熱蓄熱材を蓄熱槽内に固定充てんして,熱の授受を熱媒体で行うスタテック型蓄熱システムに代わり,固-液潜熱蓄熱材を他の液体と3―129混合して,輸送機能をもたせたダイナミック型蓄熱システムが出現し,蓄熱システムの技術的ブレイクスルーが展開しつつある.現在,輸送機能に富む固液潜熱蓄熱材として,氷水スラリー,潜熱球カプセル,潜熱エマルジョンそしてクラスレートスラリーがある.e. 化学型蓄熱材料(chemical type heat storage material)物質の化学反応に伴う吸熱や発熱現象を利用した熱エネルギーの貯蔵法が化学型蓄熱である.また,熱エネルギーの出し入れを可逆的に行える化学物質を化学型蓄熱材料と呼ぶ.化学型蓄熱は,エネルギー密度が大きく断熱材なしで長期間の蓄熱が可能である.さらに,化学型蓄熱は,低温度水準・低密度で分散している熱エネルギーをケミカルヒートポンプの利用で,昇温そして高密度化することが可能である.化学型蓄熱材には,おもに水和・脱水反応熱,アンモニア化物の生成熱,金属水素化物の生成熱や有機反応熱などに関連する化学物質がある.f. その他の蓄熱材料. 光化学反応蓄熱材(opto-chemical type heat storagematerial)ある種の有機化合物に太陽光などの光エネルギーを作用させると,化合物の構造にひずみ変化を生じ,可逆反応を行わせて熱エネルギーを取出せる蓄熱方式が可能となる.ノルボルナジェンは,紫外線を受けると結合異性体のクオドリシクレンに異性化する.高ひずみとなったクオドリシクレンは,触媒の基で1040kJ/kg もの熱を放出する.これらの物質は,蒸気圧が高く,引火性そして不快臭があり,密閉系での利用となる.その他にナフタレンとナフトバレン系等があり,変換効率向上のために光増感剤の利用が検討されている.. バイオマス(biomass)バイオマスは,生物体を構成する有機物であり,有機物を燃焼させ,熱エネルギーとして利用する.このようなバイオマスエネルギーは,植物の光合成作用により炭酸ガスと水から,太陽エネルギーを有機物に変換して得られるものである.現在,バイオマス資源に関して,廃木材,サトウキビ繊維,稲わらなどのセルロース成分が燃料用発酵アルコール原料として検討されている.さらに,促成植物であるユーカリやサトウキビの効率的生産技術の研究も行われている.
