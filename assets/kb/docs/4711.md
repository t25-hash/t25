# 8・1・3 蓄熱に関する伝熱問題

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・1 は、蓄熱物質の種類と分類を示したものである。

a. 蓄熱技術に必要とされる指標蓄熱を具体化するうえで必要な性能指標に蓄熱密度がある。蓄熱密度は、単位質量(蓄熱材を輸送する場合)や単位体積当たり(固定蓄熱槽の場合)どのくらいの熱エネルギーが蓄えられるかというもので、この値が大きいほど同じ熱エネルギーを蓄えるのに必要な質量または体積が少なくて済むことになる。さらに、蓄熱の本質的な性能指標には蓄熱効率(heat storage efficiency)がある。蓄熱効率は、入力の何%が出力できるかということで、入力熱エネルギーに対する出力熱エネルギーの比で定義される。蓄熱効率を下げる要因としては、蓄熱時における熱損失と入出力における熱損失がある。図8・2に蓄熱モデルの概念を示す。蓄熱材(蓄熱前の熱エネルギー状態は)に、熱エネルギー읉윰を与えると、熱エネルギーの高い状態となる(蓄熱)。ある時間の後、熱エネルギー状態はから욈に低下する。一方、蓄熱材の熱エネルギー状態が蓄熱前と同じとなるまで熱エネルギーを取出すとき(熱エネルギー図8・2 蓄熱モデルの概要3―130の放出)、放出される熱エネルギー윱윲읏は通常蓄熱槽に流入した熱エネルギー읉윰より少なくなる。蓄熱および放熱時、ならびに蓄熱期間にさまざまな原因で熱エネルギーが散逸し、それぞれ、=읉윰--,=욈--윱윲읏,-욈である。放出される熱エネルギー윱윲읏を蓄熱槽に流入したエネルギー읉윰で割った値を蓄熱効率η=윱윲읏/읉윰=욈--윒/-+と呼ぶ。蓄熱期間中の熱損失の大きなものは、原理的に長期間の貯蔵には適していない。熱損失の低減には、断熱方法や蓄熱や採熱方法を工夫する必要がある。季節間の蓄熱に地下帯水層による蓄熱が用いられるのは、規模を大きくして蓄熱槽体積当たりの表面積を減少し熱損失の割合を減らすことができるのに加えて、自然状態を利用するところの経済性が成り立つことによる。一方、化学蓄熱は、熱損失がないとはいえ、季節間の蓄熱には向いていない。蓄熱時間が長いということは、稼働率が低いということを意味し、経済的に成り立たないからである。蓄熱期間は、経済性を大きく左右する。さらに、入出力のエクセルギー比で定義されるエクセルギー効率も一つの性能指標である。同じエネルギー量でも高温ほどエクセルギーは大きい(利用価値が高い)ので、熱源の温度になるべく近い温度で蓄熱するほうが有利である。ある状態の系が周囲環境と平衡するまで変化するとき、外部の系に与えうる最大仕事をある状態の系の有効エネルギー(availableenergy)またはエクセルギー(exergy)という。蓄熱熱交換器において、比エンタルピーJ/kg、比エントロピー[J/kg・K]の状態にある系の比エクセルギーJ/kgは、周囲環境の温度が욉K、比エンタルピー욉、比エントロピー욉であれば、=-욉--욉욉となる。熱媒体の蓄熱槽入口(in)から出口(out)へ状態が変化したときの単位質量当たりに得られる有効仕事は、=읉윰-윱윲읏となる。したがって、エクセルギー効率は、η욄=/읉윰=읉윰-윱윲읏-욉읉윰-윱윲읏/읉윰-욉-욉읉윰-욉となる。このエクセルギー効率は、カルノー効率η=읉윰-윱윲읏/읉윰を上回ることはない。エクセルギー効率の向上には、断熱を工夫して熱損失を小さくする必要がある。顕熱蓄熱や潜熱蓄熱で温度成層を作ることはエクセルギー効率を高めるための工夫である。蓄熱効率の低下はエクセルギー効率の低下も意味するが、蓄熱効率がそれほど低下しなくても、エクセルギー効率が大幅に低下することもある。伝熱過程は有限の温度差によって成り立ち、伝熱性能が悪いと温度差が大きくなるのでエクセルギーの低下が大きくなる。実用化する面での性能指標に、温度レベルがある。温度レベルは、入出力できる熱エネルギーの温度がいくらかということであり、実際にこの温度レベルによって利用先が限定される。욦욡:蓄熱材の蓄熱(放熱)における初期温度、욦욞:蓄熱(放熱)終了時の蓄熱材温度とし、읉윰:熱媒体の蓄熱槽入口温度、윱윲읏:熱媒体蓄熱槽出口温度とすると、蓄熱過程の熱媒体の温度効率は、φ욠욦=읉윰-윱윲읏/욦욞-욦욡となり、一方、放熱過程の温度効率は、φ욠욥=윱윲읏-읉윰/욦욡-욦욞となる。これらの温度効率が大きいほど、高水準の温度レベルの熱を蓄熱または放熱可能となる。また、この温度効率は、熱媒体の水当量(質量流量×比熱)、熱媒体と蓄熱槽間の熱通過率、伝熱面積および蓄熱量に関連する。特に、熱通過率は、熱移動速度を規定することになる。b. 蓄熱に関する伝熱問題熱応答性はどの程度の速度で熱エネルギーが入出力できるかということで、要求される速度は場合に応じて異なるが、一般に速い入出速度が要求される。基本的には、入出速度がコントロールできるものが望ましい。蓄熱密度や応答速度が小さいと蓄熱技術を具体化するのは困難となる。特に、潜熱蓄熱槽(latent heat storage tank)の開発や設計にとって、最も重要な技術問題の一つが、必要な放熱および蓄熱速度を得ることが困難なことである。図8・3は、放熱時における熱交換器壁近くでの状況と温度分布を示したものであるが、潜熱蓄熱材側では、凝固層の形成が時間とともに進行する。この凝固層の伝熱形態は伝導伝熱となり、その熱抵抗は大きく、伝達熱量がその厚さとともに減少することになる。また、液相で存在した自然対流の影響も時間とともに減少する傾向となり、ますます抽出熱量の低減となる。この問題を解決するための、潜熱蓄熱器の熱交換法には、パッシブとアクティブな熱交換法がある。パッシブ熱交換法とは蓄熱材を容器に充てんして、容器壁を介して熱媒体と間接的に熱交換を行い、潜熱材料を強制的に動かすことなく熱交換を行うものである。例えば、シェルアンドチューブ型やカプセル型などがある。パッシブ熱交換法においては、潜熱蓄熱材の固体相における熱伝導率が低いので、その伝達熱量向上のために、(1)見掛けの熱伝導率を向上させる:金属粉や金網やヒートパイプを入れる、(2)伝熱管の伝熱面積を増す:伝熱管にフィンを付ける、(3)伝熱面での凝固相の形成を防ぐ:伝熱面に凝固層の付着を防ぐコーテングを施す、などの対策がある。一方、アクティブ熱交換法とは蓄熱材を熱交換器内で強制的に動かすか、あるいは直接接触させるものである。具体的には、(1)固化した潜熱蓄熱材を伝熱面からはがしたり、かき取る、(2)ガスを吹込み機械的振動あるいは超音波などにより蓄熱材を振動またはかくはんする、(3)蓄熱材自身を循環する、(4)蓄熱材を熱媒体と直接接触させる、などの方法がある。c. 代表的な固-液相変化蓄熱槽の伝熱問題図8・4は、代表的な潜熱蓄熱材を用いた蓄熱槽の例を示したものである。. シェルアンドチューブ型通常のシェルアンドチューブ型熱交換器のシェル側に蓄熱材を充てんしており、チューブ側を熱媒体が流れる構造である。. カプセル型蓄熱材が円筒状、矩形状や球状の比較的小さな容器(カプセル)の中に収納されており、多数配置されたそれら容器の外側を熱媒体が流れる。カプセル型の特徴は、(1)蓄熱材の熱伝導率の小ささによる熱交換器性能の悪さを改良するには、蓄熱材または蓄熱槽単位体積当たりの伝熱面積を大きくできるカプセル型が適している、(2)蓄熱材を単純形状のカプセル内に密閉することにより、蓄熱材の漏れ防止とその取扱いが容易となる、(3)蓄熱材相変化時の体積変化に対して、シェルアンドチューブ型よりも図8・3 潜熱蓄熱槽からの採熱時の伝熱モデル(a) カプセル型(b) シェルアンドチューブ型(c) 直接接触型図8・4 潜熱蓄熱槽の例対応が容易である、などの特徴がある。. 直接接触型蓄熱材と熱媒体を隔てることなく、両者を直接接触させる。この型では、容器の熱抵抗がなくなり、小さな分散相として蓄熱材または熱媒体をそれぞれ熱媒体または蓄熱材中に供給、移動させて直接接触させるので、蓄熱材単位体積当たりの出熱面積を飛躍的に増大させることができる。特に、放熱時の熱交換の悪さを改善しようとするものである。しかし、この方法では運用できる材料の組合せが限られることのほかに、直接接触させうるのは放熱過程のみで、蓄熱過程では伝熱壁を介して熱交換させることや、分離・輸送のことなどの技術的問題が残る。また、直接接触はさせるものの、相変化しても形の変わらない架橋ポリエチレンなどの形状安定処理とした潜熱蓄熱材を充てんした直接接触法もある。
