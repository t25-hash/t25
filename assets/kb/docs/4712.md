# 4・1・1 技術分類および発展の経緯

> **章**: 第4章 検査・診断機器
> **節**: 4・1 概説

検査・診断機器の基本は計測技術であり，最近では各種のトラン
スデューサ（変換器）を用いてさまざまな生体情報が計測されてい
る．検査・診断技術は大きく検体検査（sample test ）用と生体
（生理または患者）検査（phisiological test ，patient test ）用に分
けられ，前者は血液，尿，生体組織などあらかじめ採取された検体
を対象として生体外（in vitro）で分析を行うものを指す．一方，
後者は生体内（in vivo）で形態や機能を計測するものを指し，
種々の生体信号（主として電気・磁気的情報や機械的情報などの物
理情報）が計測される．最近では生体検査技術は医用画像として可
視化される方向にあり，他の産業技術分野と比較して画像技術が著
しく進んでいる点が特色といえよう．検体検査のうち血液生化学検
査は内科領域で極めて基本的な検査であり，1960年ごろまでは担
当医が用手法によって行うことが多かった．第二次世界大戦を契機
に臨床医学はアメリカ流（多量の検査項目に基づいた科学アプロー
チの医学）が主流となり，それに伴って検査量は飛躍的に増加し
た．大量検体処理のニーズに対応して医療機関における検査の集中
化が始まり，わが国では1950年に厚生省（現・厚生労働省）の指
示に基づいて中央検査室制度が発足し，その後続々と病院内に中央
検査部が設置された．同時に大量・迅速処理を目指した自動分析技
術の研究・開発が進み，例えばスケッグス（Skeggs ）による単チ
ャネル自動化学分析装置（フロー方式オートアナライザ）の試作は
1953年ごろに試みられ，やがてラボラトリーオートメーション学
会（Japan Society for Laboratory Automation）などが設立
（1973年）され，1980年代に入ると自動化学分析装置は必需品とし
て導入されるに至った．
一方，心電計や脳波計に代表される生体物理信号の計測は，生体
を生きた状態で即時的に観察するもので，各種のトランスデューサ
を介して生きたまま（in situ），実時間（real time）で連続して電
気信号出力として実時間計測される．この特色は，検体（試料）を
対象とした検体検査と大きく異なり，1960年以降の電子技術の急
速な発達に伴い医用電子工学（medical electronics ）として発展し
た．これらの技術が基本となり，患者監視装置（patient monitor-
ing system）が登場し，さらにはそれらを集中的に使用して術後の
生命維持管理を施すICU (intensive care unit ，集中治療室）や
CCU( coronary care unit ，冠動脈集中治療室）が誕生した．
生理（患者）検査においては，電極やトランスデューサ部位にお
ける一点計測から，より高度な生体情報を引出すための多点化が図
られ，さらには多数電極から同時計測された信号をコンピュータで
解析して内部状態を推測しようとする試み（心電図逆問題やコンピ
ュータトモグラフィーなど）が出現した．その後，X線画像のデ
ィジタル化（X線CT，X-ray computed tomography）や超音波
診断装置，MRI(magnetic resonance imaging)，PET( positron
emission tomography）などが次々に誕生した．1990年代に入る
と1980年代の医用画像技術の飛躍的進歩に伴い，疾患の早期診断
と患部の正確な部位同定が可能になったことから，外科治療の侵襲
をより低減化して患者の負担を軽減させることを目指した低侵襲治
療（minimumly invasive treatment ）が強く求められるようにな
った．これらの技術動向の背景として内視鏡装置の多機能化，およ
び1970年代後半から盛んになったカテーテルを用いた血管造影法
を基盤にした経カテーテル的治療法（インタベンショナルラジオロ
ジー）の勃興が挙げられる．1980年代にはバルーンカテーテルを
用いた血管形成術（アンジオプラスティ）やレーザ血管形成術が始
まり，これらのガイド役として微細径血管内視鏡や超音波血管内視
鏡などが出現した．1990年代には各種医用画像三次元構築技術が
提供されるようになり，それらの画像を術野に重ねて手術を行う画
像支援技術（image guided surgery）や，立体内視技術やVR
( virtual reality）技術と合体させたロボット手術へと発展してい
る．近年ではそれら内視鏡下治療の拡大に伴い，内視鏡検査・診断
を行うための光ファイバと結合させたバイオオプティクス検査・診
断技術も発展しつつある．
