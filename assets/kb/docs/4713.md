# 8・1・5 具体的蓄熱システムの例

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・1 は,蓄熱物質の種類と分類を示したものである.

蓄熱システムといえば,従来から水を基礎とした温水や冷水蓄熱が一般的であるが,ここでは最近のトピックス的な蓄熱システムと将来予想される新型蓄熱システムの例に関して述べる.3―132熱機器a. 最近の蓄熱システムの開発動向. 電力負荷平準化を目的とした氷蓄熱システム(ice heatstorage system)2000年末までにビルや業務用の冷房用として大規模氷蓄熱システムやエコアイスという小型氷蓄熱システムが累計で約1万件も導入している.氷蓄熱システムは顕熱蓄熱システムである水蓄熱に比較して設備費などで経済性に劣る面があるが,各種の補助金のもとで普及している現状にある.氷蓄熱槽の大きさは水に比較して大幅に小さくて済む利点があり,土地借料や効率的な運転維持などを加味すると氷蓄熱は有利な蓄熱システムとなる.冷却伝熱面に成長した氷層の状態のままで採冷熱するスタテックスアイス方式よりも,流動性に富むスラリー状氷を利用するダイナミックアイス方式に関してさまざまなものが提案されている.このダイナミック方式の氷蓄熱システムが全氷蓄熱システムの約4分の1を占めており,その熱負荷への応答性の良さやハンドリングの良さなどから,将来その普及が有望とされている.この方式では氷結晶粒子の凝集などが問題となっており,その解決が大きな開発課題である.. 潜熱蓄熱システムを利用した負荷追従発電大規模火力発電所に潜熱蓄熱システムを設置すると,オフピーク時に高温蒸気を一部抽気して,潜熱蓄熱槽に蓄熱(200~500°Cの溶融塩潜熱蓄熱材)し,ピーク時に給水または低圧低温蒸気の一部を潜熱蓄熱槽からの熱回収で蒸気発生または高圧蒸気に改質し,新たに設けたピークタービンを回して発電量を増やす発電システムが検討されている.. 宇宙熱機関発電低軌道の宇宙基地は高度500kmで周回すると一周期の約3分の1は蝕となり,この間の電力補償が必要となる.太陽熱を利用した熱機関発電システム(ブレイトンサイクル,有機ランキンサイクル)と潜熱蓄熱を組合わせたものが,太陽光発電と電池の組合せよりも経済的に有望である.蓄熱材としては,日射や蝕のサイクルが短いので,伝熱の応答性が重要となり,NaF(融点996°C),LiH(融点689°C)などの溶融塩(moltensalt)による潜熱蓄熱が優れている.集光採熱部内の円筒状内壁に蓄熱材を取付けて開口部よりの太陽熱で融解させ,蝕に入ったときにその潜熱放熱で熱機関へ熱エネルギーを供給し発電を継続する熱機関発電システムが検討されている.. スターリングエンジン用蓄熱再生器(heat storageregenerator)LiFの550~848°Cにわたる温度差の顕熱と848°Cにおける相変化潜熱を用いて,数kW のスターリングエンジンを数時間動かすシステムである.ヒートパイプの熱媒体金属ナトリウムの蒸発や凝縮熱を利用した迅速な熱伝達とカプセル蓄熱を併用している.. 自動車の暖気運転用潜熱蓄熱寒冷地では自動車の始動時にエンジンを暖める暖気運転を行っている.この暖気運転の熱源として,走行中の冷却水や排ガス(ヒートパイプにより熱抽出)の熱を利用して,潜熱蓄熱材に蓄熱して利用する.酢酸ナトリウム3水和塩やチオ硫酸ナトリウム5水和塩などの無機水和塩を蓄熱材とする.これらの蓄熱材は,水分の少ない結晶相が融点以上に加熱されると結晶の周囲にある水分を吸収してゲル状になる.完全にゲル状化したものは温度が低下しても過冷却状態を保ち,蓄熱材の塩の結晶核(発核剤)や機械的刺激を加えて,ゲル状態から結晶化して潜熱を外部に取出す.したがって,断熱材がなくても数日間潜熱を保存できる.. 糸状コーティング蓄熱材の土壌混入による太陽熱の蓄熱糸状に形成された樹脂または天然素材からなる担体にワックスなどの蓄熱材を含浸させ,さらに浸みだし防止用の樹脂を形成した糸状の蓄熱材を用いる.この糸状の蓄熱材をペレット状にし,土壌に混合することにより,日中蓄熱されたペレットが夜間相変化して放熱する.この作用で冬季の植物の保温を行う.このように,氷蓄熱システムをはじめとして固-液の相変化潜熱蓄熱システムは,住宅・ビル空調,車両の暖気熱源や人工衛星の宇宙発電用などさまざまな分野ですでに利用され,一部は利用が計画されている.しかしながら,いずれも潜熱蓄熱材は蓄熱槽内に充てんされ,二次熱媒体により蓄熱や採熱操作が行われる形式のものが大半を占めている.わが国においては,電力負荷平準化のための氷蓄熱システムの目覚しい普及以外に,他の分野への利用は遅れぎみである.b. 将来の新しい蓄熱システム将来にわたって蓄熱システムに要求されるものは,高密度化,小型化,長周期化や利用温度領域の拡大などである.以下に将来に要求される蓄熱システムの考え方を列記する.. 大規模蓄熱公園や道路などの地下地盤(深さ50m程度)の大きな熱容量や地下水などの滞留水を利用して排熱,自然エネルギーや深夜電力エネルギーを大規模かつ長期間にわたり蓄熱する.. 化学蓄熱対象温度:-100~200°C程度;排熱などの熱源を用いて化学物質の熱分解反応を行わせ,熱が必要なときに分解反応生成物から再合成反応を行わせ,そのときの反応熱を利用する.60種類程度の組合せが報告されており,反応系としては,水素吸蔵合金/水素,無機水和塩/水またはアンモニア,クラスレートなどがある.実質的反応条件として反応が100%進み,副反応が見られないことなどクリアすべき課題が多いようである.. 躯体潜熱蓄熱住宅壁などに室温と外気温度の平均温度の融点を有する潜熱蓄熱材で,凝固・融解に伴う熱の出し入れを利用した,断熱材フリー防熱潜熱蓄熱壁が可能である.. 小型高密度蓄熱槽一般住宅などへの深夜電力利用潜熱または顕熱高密度中温度コンパクト蓄熱槽の開発が志向されている.100~200°C程度(給湯,暖房,乾燥熱源)の潜熱蓄熱材(高分子材料や溶融塩)の開発や500~600°Cの顕熱れんが・セラミックスへの蓄熱などが考えられている.. 潜熱蓄熱機能を有する機能性熱流体による蓄熱材の管搬送潜熱蓄熱材が熱媒体に分散混合し,管輸送が可能なダイナミック蓄熱システムである.例えば,氷水スラリー,潜熱マイクロカプセル,潜熱エマルション,クラスレートなどが提案されており,配管径の減少や搬送動力当たりの輸送熱量増大が見込める.. 10°C前後の環境に優しい安価な潜熱蓄熱材の開発露点を回避した冷房空調用向け潜熱蓄熱材で冷凍機の成績係数向上等のメリットがある.. 100°C前後の環境に優しい安価な潜熱蓄熱材の開発高密度蓄熱による小型蓄熱槽の開発と給湯,暖房以外に乾燥など利用目的の拡大に寄与する.. 人工合成型の新しい蓄熱材の開発利用温度帯の拡大や蓄熱量の増大などを意図して,環境負荷の小さなかつ安価な複合潜熱蓄熱材を化学合成物質で製造する.. オフラインによる熱エネルギーの輸送(カセット型蓄熱技術)現状ではわが国における配管敷設費は高価であり,顕熱や潜熱型蓄熱材の中長距離輸送は経済的に不可能である.配管に代わって,タンクローリー,はしけや鉄道により,化学型や濃度型蓄熱材を輸送する技術開発が望まれている.
