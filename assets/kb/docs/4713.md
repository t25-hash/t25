# 4・1・2 今後の技術動向

> **章**: 第4章 検査・診断機器
> **節**: 4・1 概説

21世紀に入り，医療技術は1990年代に進展した分子生物学や細
胞工学などのバイオテクノロジーを取り入れた新しい技術革新を迎
えようとしている．主体は遺伝子（ゲノム）情報を活用した新たな
検査・診断技術であり，本書の7章で詳細に取り上げられている．
臨床医学的には個人の遺伝子情報に立脚したオーダーメイド，ある
いはテーラーメイド医療と称される個体差を考慮した薬剤投与や治
療方針の決定に期待が寄せられ，SNPs （single nucleotide
polymorphisms ，一塩基多型）を詳細に解析することで将来実現
可能になるだろう．これらの大量な遺伝子情報の解析はバイオイン
フォマティクス（bioinformatics ，生体分子情報学）と呼ばれ，今
後は医学情報の基盤を構築するものと思われる．加えて今後最も進
展が期待される再生医療に関しても，検査・計測技術の進歩が予測
される．再生医療は主として生きた細胞を薬剤として用いる細胞注
入療法（cell transplantation）と，細胞を培養して生体組織に仕
立てるヒト組織工学（tissue engineering）が基盤になる．それら
を臨床技術として用いるためには，単なる細胞培養技術だけでなく
細胞の活性度（viability）や安全性の品質保証，さらには体内に移
入した後のトラッキング技術が必要であり，それらの支援技術が完
備して初めて本格的な臨床応用が開花する．生体分子情報の測定は
今後の医療診断を行ううえで極めて重要である．これらを迅速かつ
選択的に測定する方法として，生体自身が行っている選択的反応を
利用することは極めて有効であり，いわゆるバイオセンサを用いた
計測法が期待される．測定対象となる生体分子情報を分類すると，
以下のようになる．
在宅診断がキーワードになるだろう．この実現の一環として，極微
量の血液の分析から健康マーカを測定し，在宅で健康診断ができる
ヘルスケアチップの創製が期待される．すでにMEMS (micro-
electro-mechanical systems ）技術を応用した化学・生化学分析シ
ステムの小型化研究は続けられており，ヨーロッパではμ- TAS
ノロジーにより微細化されてPOS( point of care system）に変化
(１)遺伝情報：遺伝情報物質（DNA:deoxyribonucleic acid，
RNA:ribonucleic acid)，複製，転写，翻訳に関わる酵素

することが期待される．自動分析装置により集中化された検査の時
代から，患者個人や健常者が日常生活において低侵襲で容易に使用
できる身体情報モニタ機器を携帯し，自らの健康を定期的にチェッ
