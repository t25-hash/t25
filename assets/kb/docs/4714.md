# 8・1・6 今後の蓄熱技術の展開

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・1 は、蓄熱物質の種類と分類を示したものである。

わが国におけるエネルギー供給源の多様化の観点からは、天然ガス(メタンハイドレードを含む)、再生可能な自然エネルギー、リサイクル型エネルギー、排熱を含む未利用エネルギーなど、いわゆる新エネルギー依存率を大幅に増大する必要がある。特に、熱エネルギーを循環利用する熱カスケードシステム(thermal cascadesystem)の拡大が今後の大きな課題である。すなわち熱エネルギーの発生から最終利用までの高効率化のための、熱エネルギー変換、輸送、蓄熱そして利用技術の複合化やそのシステム化技術を展開することにより、新たな熱エネルギーシステムの構築が可能となる。その中核をなす蓄熱技術の果たす役割は大きいものといえる。特に、変動性に富む太陽熱をはじめとする自然エネルギーや各種の排熱等の有効利用には、蓄熱システムが必要不可欠である。環境負荷の小さい、搬送性に富む高密度蓄熱材の開発は、熱エネルギー変換、輸送、蓄熱そして利用技術を統合化し、効率的な熱エネルギー利用システムを展開するためには重要である。
