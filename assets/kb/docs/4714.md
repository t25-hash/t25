# 4・10・1 血糖値検査

> **章**: 第4章 検査・診断機器
> **節**: 4・10 光計測装置

光を利用した血糖値（blood glucose，血液中のグルコース濃度）
検査法の開発は，その優れた非（低）侵襲性から，大きな期待が寄
せられている．特に，世界中で約1億人ともいわれるインスリン依
存型と非依存型の糖尿病患者の治療に際して，患者の血糖値を常時
モニタしていくことは非常に重要である．このため，痛みも感染性
もない，非侵襲（non-invasive）的な測定法の開発には大きな期
待が寄せられているのである．
この開発途上にある光を使った血糖値測定システムとして，以下
に述べるようにおもに赤外領域の計測技術を紹介する．
a.赤外吸収分析（infrared absorption spectroscopy) ここ
で使われる赤外線の吸収測定技術は，通常赤外（あるいは中赤外）
領域（波長：2.5～25m，波数：4000～400cm ）と近赤外領域
（波長：700nm～2.5m，波数：14000～4000cm ）で適用され
る．この波長領域でも定量測定では通常のランベルト・ベール
（Lambert -Beer ）の法則が適用され，吸光度（absorbance）と溶
質濃度の比例関係により計算される．
通常，赤外領域では分子振動スペクトルを測定できるので，グル
コースなどの糖質の分析ではOH基の伸縮振動由来の3400cm
付近の吸収やC- O結合由来の1040cm付近の吸収などがおもに
観察できる．血液中のグルコースなどのように水に溶けている場
合，HO自体のOH基などの赤外吸収が大きいので，その中の糖
質の赤外吸収を測定する場合は背景としての水の赤外吸収を差引く
（subtraction）必要がある．このときは，ATR（attenuated total
reflectance，全反射）測定法を使うことで，糖を含む溶質の赤外
スペクトルを有効に測定できる．このATR法では，セレン化亜鉛
（ZnSe）やゲルマニウムなどの結晶の中を赤外線が全反射しながら
通過するときに，結晶の外面に生じたエバネッセント波（にじみ出
た赤外線のエネルギー）でその結晶に接している物質（血液や血
清，皮膚など形態を問わない）の赤外吸収を測り，水の吸収を最小
限にして，吸収スペクトルの引算処理（subtraction）によって精
度よく溶質成分の赤外スペクトルを測定することができる．この
ATR法では，ZnSeなどの結晶表面に接している物質の表面から
せいぜい数 m以内の化学的組成の情報しか得られないが，口腔粘
膜（oral mucosa）や組織細胞表面の物質代謝の計測には有効であ
る．図4･63(ｂ)に，ZnSe結晶のロッドに口腔粘膜を押し当てて測
定した赤外スペクトルを示した．
糖のC-OH結合由来の吸収が1000～1100cmの領域に，OH
結合の変角振動由来と考えられる弱い吸収が1350～1450cmの
領域に見られる．糖の赤外吸収は非常に弱いので，血清中の脂質や
タンパク質などのグルコース以外の成分による赤外吸収からの干渉
が大きくなる．

b.その他の分析方法 上記以外では，ラマン散乱分光分析
（Raman spectroscopy)，偏光分析（polarimetric analysis )，蛍光
分析（fluorescence analysis ）などが利用される．特に偏光分析で
は，眼房水を対象に糖分子のキラリティに基づく旋光分散を測定
し，糖の定量分析が行われ，非侵襲的には前眼房に光を当てて，透
過光を使って糖量の計測も行われた．蛍光分析はグルコース酸化酵
素を使う特異的な検出法であり，発生する過酸化水素を蛍光物質で
検出し間接的に糖を定量する．
