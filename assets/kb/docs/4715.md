# 8・2・1 固体による蓄熱

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・2 顕熱蓄熱

固体の顕熱を利用した蓄熱で最も一般的なものは，排熱回収に用
いられる蓄熱式熱交換器（storage type heat exchanger）である．
高温ガスからの排熱を蓄熱体に蓄え，蓄熱体からの熱を用いて低温
ガスを加熱する．蓄熱式熱交換器は，回転式蓄熱熱交換器（rotary
storage type heat exchanger）とバルブ切換式蓄熱熱交換器
（periodic- flow type heat exchanger）に分類されるが，回転式蓄
熱熱交換器は図8･6に示すように，軸流式と半径流式に大別される．
図8･6中でマトリックスと示された部分が固体蓄熱体であり，蓄
熱体内部を熱媒体であるガスが貫流するため，蓄熱体は金属，セラ
ミックスなどの薄板を波状に積層した構造や，貫通孔を有する流路
が加工された構造などとなっている．高温ガスによって加熱された
マトリックスが回転して低温ガス流路で，低温ガスを加熱する．図
8･7は，ボイラの空気予熱器などに用いられるユングストローム式
蓄熱熱交換器であり，蓄熱体は，1～4rpm程度の速度で回転する．
固体による顕熱蓄熱は，上記の空気予熱器とともに，土壌を利用
(ａ) 軸流式 (ｂ)半径流式
図8･6 回転式蓄熱熱交換器
図8･7 ユングストローム式蓄熱熱交換器
図8･8土壌表面温度を±10℃変えたときの土中温度分布
