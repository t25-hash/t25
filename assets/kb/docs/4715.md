# 8・2・1 固体による蓄熱

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・2 顕熱蓄熱

固体の顕熱を利用した蓄熱で最も一般的なものは、排熱回収に用いられる蓄熱式熱交換器(storage type heat exchanger)である。高温ガスからの排熱を蓄熱体に蓄え、蓄熱体からの熱を用いて低温ガスを加熱する。蓄熱式熱交換器は、回転式蓄熱熱交換器(rotarystorage type heat exchanger)とバルブ切換式蓄熱熱交換器(periodic-flow type heat exchanger)に分類されるが、回転式蓄熱熱交換器は図8・6に示すように、軸流式と半径流式に大別される。図8・6中でマトリックスと示された部分が固体蓄熱体であり、蓄熱体内部を熱媒体であるガスが貫流するため、蓄熱体は金属、セラミックスなどの薄板を波状に積層した構造や、貫通孔を有する流路が加工された構造などとなっている。高温ガスによって加熱されたマトリックスが回転して低温ガス流路で、低温ガスを加熱する。図蓄熱熱交換器であり、蓄熱体は、1~4rpm程度の速度で回転する。固体による顕熱蓄熱は、上記の空気予熱器とともに、土壌を利用(a) 軸流式(b) 半径流式図8・6 回転式蓄熱熱交換器図8・7 ユングストローム式蓄熱熱交換器図8・8土壌表面温度を±10°C変えたときの土中温度分布3―133した地中蓄熱が徐々に普及しつつある。土壌の温度変化が四季を通じてわずかなことを利用し(図8・8)、冬期の温熱源、夏期の冷熱源(放熱源)として地中蓄熱が、冷暖房、融雪等に利用されている。特に、冬期のヒートポンプの機能低下が防げ、大気を熱源としたヒートポンプが適用できなかった寒冷地でも適用可能となることから、地中蓄熱とヒートポンプとを組合わせたシステムが注目されている。地中蓄熱は、熱媒体が流動するパイプを地中に埋設するパイプ方式が最も一般的であるが、地中蓄熱を有効に利用するうえで、土壌の熱物性値を十分に理解しておく必要がある。土壌は土質により、その物性が変化するが、熱伝導率は含水分が高いほど高い値となり、密度にほぼ比例することが知られている。表8・2は、おもな土質とその熱物性値である。図8・9はアメリカの太陽熱発電所(solar thermal power sta-tion、発電出力:10MW)における、砕石を用いた充てん層型の顕熱蓄熱システムであり、直径19mの蓄熱器内には10000tの砕石と1200mの有機熱媒体(カロリアHT 43)が充てんされ、蓄熱容量は180MWhである。太陽熱で発生した343°Cの蒸気と有機熱媒体が熱交換し、蓄熱器内で砕石と有機熱媒体の顕熱として蓄熱される一方、蓄熱器からの高温有機熱媒体は、蒸気発生器で発電用の274°Cの蒸気を発生させる。
