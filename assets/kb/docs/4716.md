# 4・10・3 酸素飽和度

> **章**: 第4章 検査・診断機器
> **節**: 4・10 光計測装置

酸素飽和度は血液中のヘモグロビンが酸素と結合している割合を素化ヘモグロビンとなり，肺から取込んだ酸素を脳や心臓，腎臓，だが，高齢になると正常でも95～96 %に低下する．
そして90%以こし，30 %以下になると細胞の死を招く．
原理として用いられているランベルト・ベールの法則を利用する．
外光を図4･67のように測定対象となる血液に照射し，酸素化ヘモロビンは赤外光よりも赤色光をよく吸収するという特性を利用するを求めることが可能である．
外にバンドやクリップを巻き付けて測る簡単な検査法で動脈血酸素メータでは，体外より光を照射するため，照射波は動脈血で吸収さる．
また血管壁のような境界面では光が屈折，反射を起こし，ランように，血管以外の組織と静脈血による光の吸収は心拍に関係なくを数値的に差引けば，組織や静脈血による吸収の成分は除去され，素飽和度を示す．
パーセント表記したものである．
ヘモグロビンは酸素と結合して酸筋肉など全身に運ぶ役目をする．
飽和度を計測する機器がパルスオキシメータである．
パルスオキシれるだけでなく，皮膚をはじめとする組織や静脈血でも吸収を受けベルト・ベールの法則が当てはまらなくなる．
そこで，図4･68のほぼ一定であることを利用し，吸光全体から変動しない不変な成分血管内で脈動している動脈血による吸収成分のみが残る．
