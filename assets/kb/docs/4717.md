# 4・10・4 光トポグラフィー（optical topography）

> **章**: 第4章 検査・診断機器
> **節**: 4・10 光計測装置

a.概説 近赤外光（near infrared light ）を用い，脳血液や
代謝の変化を非侵襲かつ簡便に計測し画像化することにより機能診
断を目的とした光トポグラフィーがある．光による脳機能計測法は
従来より使用されているfMRI ，PETやMEG（脳磁図）等に比べ
て，以下の特徴がある．
(１)無侵襲な計測のため被検者に負担がかからず何回でも計測で
きる．
(２)小型で安価．ベッドサイドでの計測も可能．
(３)計測時の制限が少なく，少し動いても計測可能なため，複雑
な負荷刺激を与えることができる．
(４)他の計測法との干渉が少ないため，MRIやEEGとの同時計
測が可能．
図4･69に装置の概観を示す．
b.計測原理 近赤外光は皮膚や頭蓋骨の透過性が比較的高
く，頭蓋骨のすぐ内側にある大脳皮質の血液変化を頭皮上から計測
することができる．血液中に含まれるヘモグロビンの光吸収特性を
利用して，二つの異なった波長の光を同時に照射することで，酸素
化および脱酸素化ヘモグロビン量変化を計測する（図4･70，4･
71) ．
照射と検出位置が30mmの場合，光の広がりは半値幅で約25
mm，計測中心深さは20～30mmと考えられている．
光トポグラフィー装置は光の広がりを考慮し，帯域の狭いレーザ
光を使用している．多チャネルの受信信号を計測する方法として照
射位置ごとに時間分割照射をする方法があるが，チャネルを増すご
とに計測時間が長くなるため，光トポグラフィーでは照射光に変調
を加える方法を採用している．変調方式では外乱光に対しても強

c .画像表示 光トポグラフィー装置は，データとしてタイム
コース（時間変化をグラフで表示）とトポグラフィー画像を表示す
る．
トポグラフィー画像はタイムコースに表したデータを二次元画像
として表したもので，時間軸に変化させることで動画として表示す
ることができ，増加を赤，減少を青の濃淡で表示する．これにより
負荷刺激に対する血液量の変化をリアルタイムで動画像で観察でき
るため，活性化部位や代謝のダイナミックな変化をとらえることが
容易になる（図4･72) ．
