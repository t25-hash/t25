# 8・2・3 帯水層による蓄熱

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・10 のように変化する.一方,温度成層型は,槽内に人為的に維

a. 帯水層(aquifer)大地を掘り下げていくと,帯状の地層が現れる.地表近くは表土,腐植土,その下が粘土,砂れきまた粘土という具合に帯状模様が基盤まで数百m続く.この帯状模様は,河川のはんらんにより作られたものである.地下水で満たされた帯状砂れき層を帯水層と呼ぶ.b. 蓄熱原理と特徴井戸は帯水層の地下水をくみ上げる設備であるが,水のよく出る井戸は,水のよく入る井戸でもある.ここで,温水を井戸に注入すると,図8・13に示すように,温水は砂れき層の間を流れ去る.その際,温水は熱を砂れき層に伝達しながら自身の温度を下げる.温水注入を続けることにより,井戸周りの大量の砂れき層は暖められる.地下水自然流速の小さい帯水層では,熱対流が抑えられ,なかなか冷却しない.ここで井戸から揚水すると,冷たい地下水が周囲から押し寄せ,暖かい砂れき層の間を熱交換しながら流れる.その結果,温かい水がくみ上げられ,温熱エネルギーが回収される.な3―135図8・15 冷熱井戸と温熱井戸の熱回収率の変化図8・16 蓄熱適正地選定手順お,冷熱の蓄熱機構も温熱同様,いわゆる岩石蓄熱であって地下水自体は熱を運ぶ媒体である.帯水層蓄熱の特徴をまとめると以下のようになる.(1) 自然地下水流の小さな帯水層でのみ実現可能.(2) 蓄熱温度レベルは低い(水質保全の立場から,温熱で20~30°C,冷熱で5~10°Cが望ましい).(3) 蓄熱容量はTJ 規模で巨大である.(4) 蓄熱規模を大きくするほど効果が上がることから,季節間蓄熱に適する.c. 蓄熱利用システム蓄熱利用システムは,図8・14に示すように2本井戸を備え,一方を温熱蓄熱井戸,他を冷熱蓄熱井戸として活用する季節間蓄熱システムが基本である.季節蓄熱運転の基本サイクルは次の4行程からなる.15月から9月までの5箇月間が冷熱回収・温熱注入行程.210月の1箇月が停止行程.311月から3月までの5箇月間が温熱回収・冷熱蓄熱行程.44月の1箇月が停止行程.蓄熱井戸の性能は,熱回収率ηで評価される.ここで熱回収率ηは,注入熱量に対する回収熱量の比と定義する(ただし,注入水量をすべて揚水した時点での数値である).熱回収率ηは,その実験結果を整理すると図8・15に示すように,温熱蓄熱,冷熱蓄熱にかかわらず,無次元注入量の単調増加関数である.ここで無次元注入量は:帯水層厚さ(m),욄:見掛け熱拡散率(m/s),:注入量(m/s)により次式で与えられる.=/욄サイクルを繰返すにつれ,井戸周りに鉄コロイドダムが造成さ3―136熱機器れ,見掛け熱拡散率욄が低下し,熱回収率ηが顕著に向上することが確認されている.したがって,帯水層はカナケ井戸地帯のほうがむしろ蓄熱に向いている.d. 蓄熱適正地選定手順熱回収率を支配する最大の要因は帯水層内の自然地下水流速である.蓄熱適正地選定手順は,地下水の流れていない帯水層の選定という観点で図8・16のようにまとめられる.帯水層の見掛け熱拡散率욄は,自然地下水流がなければ,水で充てんされた砂れき層の値욦を示すが,自然地下水流速の増加に伴い急激に増大する.蓄熱適正地の最終判定は,試験井戸の見掛け熱拡散率の大小で判断することができる.試験井戸による必要最低測定項目は,1帯水層厚さ,2地下水自然水位,3地下水の全鉄濃度,4地下水自然温度および地温こう配,5透水量係数(透水係数×帯水層厚さ),6安全揚水量,7見掛け熱拡散率の7項目である.
