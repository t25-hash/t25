# 4・10・5 血栓計測方法

> **章**: 第4章 検査・診断機器
> **節**: 4・10 光計測装置

血栓とは人の体内で生じる血液の凝固物である．人体は血管に何
らかの傷害を受け出血すると，その傷口では血液が固まりすみやか
に止血を行う止血機構を持っている．血小板，白血球，赤血球およ
び繊維状の固形物を生成するフィブリン（fibrin，不溶性タンパク
質，繊維素）からなる物質である血栓を傷口に生じ，これを閉そく
するのである．近年，脳塞栓症や肺血栓，エコノミークラス症候群
などの原因として改めて血栓の危険性が指摘されてきている．これ
は血栓が正常な止血機能としてではなく，凝固系バランスの崩壊や
血管壁の異常，狭窄（さ)，うっ血などでも生成されてしまうため
であり，たとえそれが微小な血栓であっても，さらに細い血管，つ
まり脳や肺の毛細血管を塞栓することで，組織に致命的な損傷を与
えるためである．血栓の対処法として血液凝固阻止剤を用いた抗凝
固療法を適用しているが，過分な投与は逆に出血を抑止することが
困難になるなどの問題もあり，これら薬品を効果的に使用する指針
としても血栓形成の連続計測は重要となっている．
血液中の血栓を検出するために，光学的手法，特に近赤外光を利
用した計測手法は連続計測も可能であることから効果的である．血
栓を形成する物質はそのほとんどが血液中に存在する物質ではある
が，その組成（密度，形状等）は通常の血球成分とは異なることか
ら，光吸収特性を左右するヘモグロビン密度や，光散乱特性に影響
する粒子径などに大きな変化が生じる．これを利用し，血液層にレ
ーザを照射，その透過（前方散乱）光量の経時的変化を観測するこ
とにより，血栓形成の有無を確認することが可能となる．波長に
は，ヘモグロビンの酸素結合状態によって光吸収率が異なるという
性質があるため，ヘマトクリットの計測と同様に脱酸素化ヘモグロ
ビンと酸素化ヘモグロビンの等吸収点である800nm付近の赤外光
を使用する．これにより動脈血・静脈血，運動前・後といった血液
の酸素飽和度の変化に影響を受けない計測が可能となる．
図4･73は連続流型人工心臓Bio-Pump BP-80（メドトロニック
バイオメディカス社）と，人工透析用血液チューブより構成される
血液を満たした模擬循環系にて，ポンプ内のインペラ部に付着した
血栓である．この血栓を検出するため光ファイバをポンプ外壁表面
に45°の角度をなして取付け，ポンプを回転させながらポンプ外部
からインペラに向けてレーザを照射した際の反射強度の変化を計測
した．図4･74に示すように血栓の位置（A-D）と光の反射強度の
変動とが一致しているように，赤外光による血栓計測を行うことが
図4･75光CTの測定系概念図
できる．
