# 8・3・1 低温の蓄熱

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・3 潜熱蓄熱

低温用潜熱蓄熱材として分類される水和物としては無機塩類(無機水和塩)が挙げられ,温熱,冷熱蓄熱材として研究されている.このほか,広義の水和物として包接型水和物があり,主として冷熱蓄熱材として研究されてきている.特に最近では,潜熱材をボール等のカプセルに密封して間接的に用いる方式のほか,スラリー化して直接熱交換媒体として用いる方式も研究されてきている.a. 水和物(hydrate)一つの化合物が水分子といっしょに別種の固体分子を形成する場合,その化合物を水和物と呼ぶ.水溶液から析出する塩類の多くは水を付加して水和物となっている(水和塩).水和塩には,金属イオンに配位した結晶水をもつもの,あるいは水分子が格子内の空間を満たし弱く結合しているものなどがあるが,蓄熱材の候補である無機水和塩中の水のほとんどは金属イオンに配位した結晶水である.無機水和塩は有機物に比べ,腐食性,過冷却性に注意を要するが,単位体積当たりの潜熱などで有利である.無機水和塩(salt 包接水和物(clathrate hydrate)包接水和物は水分子が構成するかご状構造(ホスト構造)の中に他の原子または分子が入り込んだ結晶構造をもつ物質であり,氷とよく似た外観をもつ.包接水和物はゲスト分子によって次の4種類に分類,大別される.. 気体(気体水和物)メタン,エタン,プロパンなどの炭化水素,フロンガスの一部,二酸化炭素などの常温,常圧で気体の物質.表8・6にその一例を示す.水和物は臨界分解点以下の温度,圧力で生成できる.. 水溶性物質(エーテル型包接水和物)エーテル,ケトンなどの極性の強い低分子有機物.常温,常圧で液体.一例としてテトラハイドロフラン(THF:
