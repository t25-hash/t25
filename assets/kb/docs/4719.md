# 4・11・1 CT

> **章**: 第4章 検査・診断機器
> **節**: 4・11 その他の映像検査装置

X線CT（コンピュータ断層撮影法，computed tomography）
はX線が生体を直進して透過し，その減衰が通過した直線上のX
線減衰係数の積分になることを利用し，ラドン変換によってX線
減衰係数分布の生体内断層像を描き出す技術である．光や電気も生
体内を透過して再び体表面に現れ，それを検出することによって光
や電気の特性値に関する断層像を描き出すことができる．しかし，
光や電気は生体内をまっすぐには進まないため，X線CTとは異
なったアルゴリズムが必要となる．以下では，光，電気およびマイ
クロ波を使ったCTについて述べる．

b.インピーダンスCT（impedance CT) インピーダンス
CTは，組織の導電率分布あるいは複素誘電率分布の断層像を得よ
うとするものであり，装置が簡単，安価，取扱いやすいなどの利点
がある．周囲の組織と導電率が著しく異なる組織（血液や脂肪な
ど）の画像化が可能であるため，血行動態などの診断に役立つと考
えられている．インピーダンスの周波数特性から組織の性状をとら
える多周波数インピーダンスCTの研究により，体脂肪量の測
定 も行われている．

動による誤差が大きいこと，低い空間分解能，電極のインピーダン
ス問題などにより，臨床的な応用にはなお課題が多い．
c.その他 マイクロ波CT（microwave CT）は，X線の代
わりに1～2GHzの電磁波を生体に照射して，生体によるマイクロ
波の減衰率などの断層像を得るものである．インピーダンスCTに
比べ，電磁波の周波数が格段に高いため，直進透過成分が多い．し
かし，生体組織によるマイクロ波の回折や散乱が避けられないた
め，断層像を得るためには何らかの方法で直進成分を取出してX
線CTのアルゴリズムを用いるか，あるいはマイクロ波の伝搬経路
を特定して逆問題アルゴリズムを用いる必要がある．
マイクロ波の伝搬をマクスウェル方程式により記述してボルン
（Born）近似やリトフ（Lytov）近似を用いて逆問題を解く方式は
回折波トモグラフィーと呼ばれるが，近似が生体中では成立せず適
用が困難である．
電磁波の可干渉性を利用するチャープレーダの技術を用いたマイ
クロ波CT では，チャープパルス信号と信号処理技術を利用し
て送受信アンテナ間の直線経路を伝搬した波動成分のみを抽出し，
X線CTのアルゴリズムを用いる．その結果得られる断層像は複
素誘電率分布であるが，複素誘電率の温度依存性を利用して体内の
温度分布の変化を計測する（サーマルCT）ことができると考えら
れている．がんの温熱療法では，体内の温度上昇を1℃よりもよい
精度で計測することが求められるが，マイクロ波CTをいわゆるサ
ーマルCTとして利用できれば，マイクロ波で加温と温度測定が同
時に可能となる．
