# 1・1・2 気体燃料

> **章**: 第1章 火花点火機関
> **節**: 1・1 燃料

a. 気体燃料の性質と機関性能気体燃料は従来国内では主としてバーナ燃焼で用いられてきたため、性質のうち燃焼速度とウォッベ指数(Wobbe index)が重視されてきた。ウォッベ指数は、発熱量を比重の平方根で割った数値で、バーナノズルから噴出する気体燃料の発熱量を示す指数である。気体燃料の種類、組成が異なっても、燃焼速度とウォッベ指数が同一であれば、バーナの燃焼性能は変化しない。ガス機関においては、燃焼速度の機関性能へ及ぼす影響は、点火時期の調整により容易に対応できる。また、ウォッベ指数の変化は、ガスミキサの種類によっては、供給空燃比に影響するが、これも調整は容易である。気体燃料の真発熱量は燃料によりかなり異なるが、機関出力、特に無過給機関の最大出力に影響するのは燃料自体ではなく混合気の発熱量である。後者は前者ほどには、気体燃料の種類による相違は大きくない。また、気体燃料の燃焼限界、特に希薄限界は希薄燃焼ガス機関の希薄側の運転限界と関係する。しかし、機関の運転限界はトルク変動の許容値等から決められるとともに、機関内の燃焼条件は気体燃料の燃焼限界が決められる通常の温度、圧力条件とは異なるために、気体燃料の燃焼限界がそのまま機関の運転限界となるわけではない。一方、気体燃料の性質のうち、耐ノック性は機関の平均有効圧力、圧縮比、点火時期等の制約因子であるため、機関の出力、効率に最も大きく影響する。また、耐ノック性は気体燃料の種類により大きく異なるため、最も重視すべき性質である。耐ノック性の指標としては、メタン価(methane number)、ブタン価(butane num-ber)等が用いられる。メタン価とは、それと同等の耐ノック性を示すメタン(炭化水素で最も耐ノック性が高い)と水素の混合気におけるメタンの体積割合であり、ブタン価は、同様のメタンと-ブタンの混合気におけるブタンの体積割合である。したがって、メタン価は数値が大きいほど、ブタン価は数値が小さいほど耐ノック性が高い。メタン価は多くの種類の単体気体と混合気体について測定されており、任意の混合気体について成分ガスのメタン価から算出可能だが、算出は多少複雑である。ブタン価に対する算出式はいくつか提案されており、いずれも算出は簡単であるが、これらが気体燃料真発熱量25°CMJ/m理論混合比における燃料空気混合気発熱量MJ/mH9.8832.920.125CO11.5793.43―CH32.873.12CH51.684.01――CH54.423.56―CH58.813.320.2CH80.173.58―CH85.063.430.38n-CH112.323.52i-CH111.313.4910.50.85天然ガス例1(国内産)31.93.12CH;97,N;1,CO;2天然ガス例2(国内産)34.73.151.38CH;94,CH;5,CH;1天然ガス例3(LNG)36.53.172.72CH;90,CH;6,CH;4都市ガス13A38.13.194.47CH;89.5,CH;4.6,CH;5.4,CH;1.5都市ガス6C15.33.06―H;46,CO;5,CH;22,CH;4,CH;1,N;10,O;2,CO;10液化石油ガス(い号)85.63.4439.7CH;1,CH;96,CH;3液化石油ガス(ろ号)3.5110.699.4CH;1,CH;994―5適用できるのは天然ガス系の気体燃料に限られる。各種気体燃料の性質を表1・4に示す。b. 天然ガス天然ガス(natural gas)は、油田、ガス田から産出するメタンを主成分とする可燃性気体である。油田から産出するガスは炭素数2以上の炭化水素を含み、発熱量が高く、随伴ガス、油田ガスと呼ばれる。石油と共存しない純ガス田からのガスは、ガス田ガス、構造性ガスと呼ばれ、炭素数2以上の炭化水素が少なく発熱量が低い。国内では地下水中に溶存している水性ガスがあるが、これは構造性ガスの一種である。天然ガスは、海外では発達したパイプライン網により、ガス井から消費地まで長距離輸送されて使用されているが、国内では生産量が少ないため、LNG(liquefied natural gas、液化天然ガス)として輸入されるものが非常に多い。LNG は、気化されてそのまま発電に使われるとともに、都市ガスの原料とされる。c. 都市ガス都市ガスは、パイプラインにより供給されるが、国内ではウォッベ指数と燃焼速度の組合せにより7種類に分類されている。供給に際して、総発熱量とともに燃焼安定性を考慮し、ウォッベ指数および燃焼速度が一定範囲に保たれる。都市ガスの原料として、以前は石炭、石油が多かったが、最近ではLNG が原料の大部分を占めている。LNG は産地により発熱量が異なるため、プロパン、ブタンを加え総発熱量を46.0MJ/N・m一定として13A ガスという呼称で供給することが多い。13A ガスには主成分のメタンのほかにプロパン、ブタンが含まれるため、LNG や海外の天然ガスに比べて耐ノック性が低い。d. 液化石油ガス液化石油ガスは通常LPG(liquefied petro-leum gas)、プロパンガスなどと呼ばれており、主成分は炭素数が3および4の飽和および不飽和炭化水素である。常温で加圧すると容易に液化するため、ボンベによる輸送、貯蔵に便利である。耐ノック性は、天然ガスに比べるとかなり低い。また、消費時プロパンが先に蒸発するため、発生する混合気体の組成が時間とともに変化する。なお、都市ガス、液化石油ガスには、保安のため付臭剤が加えられているが、その量は極めて微量であり、ガス機関の触媒等への影響は無視しうる。e.
