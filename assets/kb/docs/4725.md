# 4・13・1 眼圧検査機器

> **節**: 4・13 その他の検査機器

眼圧は眼内の房水により作られる圧力で，正常眼圧は10～21
mmHgである．検査目的は緑内障疾患の診断がおもなものである．
眼球は球体に近似することができ，検査法は圧入法と圧平法とに大
別される．
a.圧入法 一定の荷重をかけた可動杆（かん)（plunger ）の
沈下量から換算して眼圧を求めるものである．最も代表的な器具と
してツェッツ（Schiotz）眼圧計がある．角膜表面に装置を保持す
るための外筒の中におもりを載せた軸棒（可動杆）があり，可動杆
の沈下量に対応する目盛（0.05mm沈下/1目盛）から換算表によ
り眼圧を求めるものである．測定値は角膜の曲率（基準値：7.8
mm）と眼球壁硬性（0.0215）に影響される．基準値よりも角膜
曲率が小さい場合，または眼球壁硬性が高い場合，換算値は過大評
価となる．測定は表面麻酔を行ったうえ，仰臥位で行う．
b.圧平法 弾性が無視できる球体の内圧 と外圧 および
