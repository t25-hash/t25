# 1・2・1 点火装置(ignition equipment)

> **章**: 第1章 火花点火機関
> **節**: 1・2 点火系

a. 概説点火装置はおもに、高電圧発生部(点火コイル、電流断続部)、配電部、点火プラグ(spark plug)および点火時期制御部から構成される。図1・7に概略の構成を示す。高電圧発生部の方式はいろいろあるが、いずれも電磁誘導作用を利用して、点火コイルに15~35kV の高電圧を発生させ、点火プラグの電極間で火花放電を起こして混合気に点火する。各気筒への配電は、1個のコイルからディストリビュータを介して、高電圧を複数の気筒に配電する高圧配電方式と低電圧で配電し、気筒ごとに高電圧に昇圧する低圧配電方式がある。なお、電流遮断部(断続部)は、古くはポイント接点方式が多く用いられていたが、最近は半導体による無接点遮断方式が一般的である。また、エンジン性能を最適化するために、点火時期の電子制御が多く用いられてきている。b. マグネト方式永久磁石が配されたロータを、磁石発電機に内装された点火コイル周りで回転することで、交流電圧を発生させる。これを一次コイルの電流のピーク値近傍で遮断して、高電圧を発生する方式である。発電部分を低電圧の発電コイルのみとし、別体の点火コイルで昇圧する方式もある。前者を高圧マグネト方式、後者を低圧マグネト方式と呼ぶ。図1・8に低圧マグネト方式の構成を示す。マグネト方式はバッテリが不要で簡単な構造のため、小型エンジンに用いられることが多い。. バッテリ方式バッテリから一次コイルに通電する方式で、自動車をはじめ多くのエンジンで用いられている。図1・9に示すように直流電圧で一次コイルに通電するため、マグネト方式に比べるとコイルに蓄えられる点火エネルギーが安定し、点火タイミングの自由度を大きく取れる利点がある。. コンデンサ放電(CDI)方式あらかじめコンデンサに400V 熱特性点火プラグは高温の燃焼ガスにさらされるため、絶縁体受熱部の熱を中心電極や取付金具から放散するように設計されている。放散する熱量が少ないと、点火プラグ先端部の温度が高くなり、ついには火花放電前に着火が起こってしまう。これを過早着火(プレイグニッション、preignition)と呼び、着火時期が早まるため、異常燃焼を起こしてエンジンを損傷させる。一方、発火部の温度が低すぎると、燃焼ガス中のカーボンなどの汚損物質が絶縁体の表面に付着堆積し、絶縁抵抗が低下して高電圧が漏えいし、火花が発生しなくなる。点火プラグの選定に当たっては、エンジン運転中につねにプレイグニッションが起こらない温度以下にすることが必要で、なおかつ、運転中に自己清浄温度(絶縁体表面のカーボンは500°C前後で消失するので、これを自己清浄温度と呼ぶ)以上にすることが望ましい。受熱量が少なく放熱度合いが大きい点火プラグは温度が上がりにくく、これを冷え型(コールドタイプ)と呼び、その逆で、温度が上がりやすいものを焼け型(ホットタイプ)と呼ぶ(図1・18)。プレイグニッションで決まるプラグの使用上限を熱価(heat プラグコード(plug cord)、プラグキャップ(plug cap)点火プラグコード、プラグキャップ(図1・22)は、高圧配電方式の場合にコイルで発生した高電圧を点火プラグまで送電するのに用いる。電波雑音防止のため、通常カーボンや金属抵抗細線により数kΩの抵抗を有している。プラグコードが長いと二次電圧は低下し、また浮遊静電容量が増加しても同様に低下するため、配線には注意が必要である。プラグキャップはおもに二輪、はん用エンジンに用いられ、プラグコード切断端面にねじ込んで接続できるようになっている。プラグキャップには、電波雑音防止のため、通常、数kΩの抵抗が内装されている。i. 点火時期制御装置点火時期は、エンジンの回転、負荷の状況により、出力/燃焼効率向上、排出ガスの発生抑制等をねらって制御される。制御方法は、機械式と電子制御式に大別され、機械式の場合はe項のディストリビュータ内のガバナ機構や、吸入負圧を利用する。. 機械式進角装置機械式進角制御は、遠心式と吸入負圧式がある。遠心式は、ディストリビュータ内のガバナウエイトの遠心力を利用し、エンジンが高回転になるにつれて点火時期を進角側に調節する機構である。また吸入負圧式は、吸入負圧を利用してダイヤフラムに連結した可動台板を移動させ、スロットル開度が開くに伴い、点火時期を遅角側に調節する機構である。通常、この二つの機構を組合わせて、エンジン特性に合わせた点火時期に制御する。機械式の場合、細かな部分までは最適時期に合致させることが難しく、理想的な点火時期からは多少のずれが生じる。.
