# 4・13・2 眼底検査機器

> **章**: 第4章 検査・診断機器
> **節**: 4・13 その他の検査機器

眼底検査は眼底の網膜，脈絡膜の病変を検査することを目的とす
るもので，眼科臨床の基本的なものの一つである．検査法としては
外でどのように像を結ばせるか（眼外のレンズ系）により，眼底像
がそのままの像として見える直像法と，点対称像となる倒像法とに
分けられる．
a.直像鏡 眼底からの光を被検者の眼底に対称に結像させる
装置である．直像鏡からの照明光を瞳孔の一部から眼底に照明し，
被検者は直像鏡の光源軸に近いところに自分の眼を置き眼底を観察
する．散瞳せずに眼底の観察が可能であることなどから，眼科医以
外の領域で用いられることが多い．眼科領域では高い倍率での観察
が可能であるため，視神経乳頭などの眼底後極部の観察に使用され
る．
