# 1・3・1 潤滑系および潤滑油の役割

> **章**: 第1章 火花点火機関
> **節**: 1・3 潤滑系

内燃機関の潤滑系および潤滑油は、性能および耐久性の向上を担っており、以下に示す諸作用を同時に行っている。a. 摩擦損失および摩耗の低減作用潤滑部のしゅう動面間に油膜形成による流体摩擦を生じさせて、摩擦損失、摩耗を低減する。油膜形成機構に関してはβ4編第II部1・4節を参照されたい。b. 冷却作用しゅう動部で発生する摩擦熱および燃焼室内の高温気体から伝達される熱を油膜および潤滑油の流れで吸収し、しゅう動部の温度上昇を抑制する。c. 密封作用ピストンとピストンリングおよびシリンダ間の微小すきまを潤滑油で満たし、気体の漏えいを抑制する。d. 衝撃力の分散・吸収作用燃焼室内で混合気が燃焼すると、瞬間的に数MPaの圧力が発生する。各軸受部を満たしている潤滑油は、この衝撃力を油全体に分散、吸収して、局部的な圧力上昇を軽減する。これにより、金属相互の直接接触時に発生する打音(機関騒音となる)を防止する。e. 防錆および防食作用潤滑油膜が金属表面を保護するため、燃焼に由来する水、亜硝酸ガスなどによるさびや腐食作用を低減する。f. 防じんおよび清浄作用潤滑油膜により機関内に吸入されたほこりがしゅう動部に入り込むことを防ぐとともに、油膜中に侵入したすすや摩耗金属粉などの異物を洗い流す。g. 作動油としての働きいくつかの油圧作動部品が弁機構に使用されており、潤滑油はその作動に使用されている。
