# 1・3・3 潤滑系の構成

> **章**: 第1章 火花点火機関
> **節**: 1・3 潤滑系

潤滑系の構成は多様であるが,一般的には図1・25に示すようにオイルパン,オイルストレーナ,油ポンプ,オイルフィルタ,オイルギャラリ,各潤滑部で成り立っている.以下に潤滑油の流れに沿って主要部品について説明する.a. オイルパン(oil pan)潤滑油はオイルパンに集められており,ここから油中の比較的大きな異物を取り除くためのオイルストレーナ(oil strainer)を介して油ポンプを通って機関各部に供給され,潤滑を終了して再びここへ戻ってくる.オイルパンに戻ってくる油は,軸受やピストンなどのしゅう動部分で発生する摩擦熱や燃焼に由来する熱により相当高温になっており,これを放熱して潤滑油の温度を下げてその劣化を抑える機能も受け持っている.図しており,排気量とともに潤滑油量が増加することがわかる.潤滑油量は,時間経過とともに減少する.消費が進み潤滑油量が不足すると機関の焼付きが生じる.逆に,潤滑油量が多すぎると,クランク軸と干渉してかくはん抵抗の増加を招く.そこで,オイルパン内に適正量の油量が存在することを確認するため,油面計(oil level gauge)が設けられている.また,運転席から油切れがわかるように,メインギャラリ油圧の表示計や警告灯が装備されている.オイルパン内の潤滑油を有効に使用するため,極力低い位置にオイルストレーナが設置され,その吸込み口には金網状のフィルタが取付けられている.オイルパン底の沈殿物を吸い込まないよう吸込み口と底面との間にはすきまを設ける必要があるため,車両旋回時など大きな加速度が加わる場合や傾斜路などでは,大きな油面変動が生じ吸込み口から油面がずれて多量の空気が吸込まれる可能性がある.これを防ぐためオイルパン内に図1・27に示すバッフル板(baffle plate)を設けた仕様がある.b. 油ポンプ(oil pump)油を機関各部に圧送する油ポンプ図1・28 外接式インボリュート歯車ポンプの例(a) 多数歯トロコイドロータの例(b) 4葉5節トロコイドロータの例(c) スローアウェイ式オイルフィルタの例付ける内接式歯車ポンプ[図1・29(a)(c)]は収納上の利点があるが,インナおよびアウタロータの側面がハウジングと相対運動を行うため,この部分の摩擦損失は図1・30に示すように大きな割合を占め,吐出し量を増すためのロータ径増加にはこの点の考慮が必要となる.ポンプの大きさ(吐出し量)は,機関が老朽化し,各部のすきまが増した場合や高油温で使用される場合なども考慮して,一般的に必要循環量の2倍程度の余裕をもたせて設計されている.しかし,この吐出し量を常時供給すると,通常は油圧が必要以上に上昇し,機関各潤滑部の油漏れや破損を招くことがある.特に,低温始動時には潤滑油の粘度が大きいので油圧が非常に高くなり,油ポンプの負荷が増し,駆動歯車などの破損や摩耗損失を起こす.そこで,油ポンプの吐出し口には圧力調整弁(pressure regu-lator valve)が設けられ,どのような使用条件でも適正な油圧を保持し,問題の発生を防いでいる.このような系による潤滑油の供給状態の一例を図1・31に示す.なお,近年では弁機構において,1・3・3・f項で記述する油圧駆動可変機構が採用されており,機関の低回転域から油圧・油量を確保する必要が生じているが,この要求をポンプの吐出し量増加で対処すると駆動損失の増加を招く.そこで,2個の歯車式ポンプと制御弁を組合わせて,低回転域では両方のポンプを駆動して油量を確保し,高回転域では制御弁の切換えによって一方の吐出し口と他方の吸入口を接続するように経路を変えて,むだな吐出しのための駆動損失を低減した形式も採用されている.c. オイルフィルタ(oil filter)機関の運転によって潤滑油には,ほこり,燃焼生成物,油の劣化分解による不溶性物質,金属粉などが混入するが,これらを取り除きしゅう動部を保護するろ過装4―12内燃機関置がオイルフィルタであり,一例を図1・32に示す.オイルフィルタの設置法には図1・33に示す方式がある.(1) 全流量方式機関を循環する潤滑油の全量がフィルタを通る方式であり,ほとんどの火花点火機関に採用されている.(2) バイパス方式機関の循環とは異なる別経路を設けて,機関循環量の5~25%程度がフィルタを通る方式である.フィルタ素子(filter element)は,図1・34に示すように,ろ紙およびろ紙に樹脂を含浸補強させたものを波状に折り込んで十分なろ過面積をもたせてある.ただし,完全に目づまりを起こしたときに機関各潤滑部に潤滑油が供給されないと支障が生じるため,安全弁としてフィルタ素子の前後の圧力差がほぼ0.1MPa以上になると逃がし弁(relief valve)が開き,潤滑油がフィルタを素通りするようになっている.また,機関停止後にフィルタ内の潤滑油が抜けてしまうと,始動直後に潤滑油がフィルタを満たすまで各潤滑部に供給されないので,つねにフィルタ内に潤滑油を保持できるように,潤滑油が抜け落ちることを防止するドレーンバック防止弁を設けている.なお,オイルフィルタは機関を保護するために定期的な交換が必要である.近年では,この際の分別廃棄が容易な構成になってきており,図1・35にその一例を示す.d. オイルクーラ(oil cooler)過給機を装着した機関では熱負荷が大きく,オイルパンだけでは潤滑油を十分に冷却することができないため,適正な供給油温を保持するためにオイルクーラを設ける場合がある.冷却方式により水冷式と空冷式,配置によりブロックへの直接取付式と別置き式,構造により多板式と多管式がある.図1・36にブロックとオイルフィルタ間に取付ける水冷多板式の例を示す.e. 機関内の潤滑経路オイルフィルタを通過した油はメインギャラリ(main gallery)を通り,クランク軸の主軸受部および弁機構へ供給され,これらの部分の潤滑を行う.主軸受部に供給された潤滑油はさらにクランクピンを潤滑し,飛まつとなった油がシリンダ壁を潤滑するとともに,ピストンなどを冷却する.弁機構においては直接駆動式(直動型)のカム-フォロワ間,バルブリフタ部およびバルブステム-バルブガイド間はカムのベース円に設けた小孔からの油またはカム軸受部から流れ出た油を利用する.一方,ロッカアーム式ではロッカ軸ブラケットからロッカアームの軸受を潤滑した油,または給油パイプからの滴下油により,カム-フォロワ間およびバルブステム-バルブガイド間を潤滑する.そして,これらの部分を潤滑した油は自由落下によりオイルパンへと戻る.(a) 油圧作動部品弁機構では,一般的にいくつかの油圧作動部品が使用されている.クランク軸の回転をカム軸にチェーンを用いて伝える方式の場合,チェーンの張りを確保するために油圧を利用したチェーンテンショナが設置されている.また,カムとバルブリフタまたはフォロワのすきまをなくすために,油圧によるすきま調整機構が利用されている場合がある.近年では,機関の燃費と出力性能の両立を図るために可変動弁機構が開発され,油圧が利用されている.代表的な例として,図1・37のようにカム軸を運転条件によって回転させて作動時期を変更(揚程は同一)する形式と,図1・38のように作動時期・期間と揚程の異なる二つのカムをもち,運転条件により使用するカムを変更する形式を示す.図1・37(a)では,A 通路に油圧が作用しB 通路から油が流出するので,ピストンは図中の右側に移動し,そのガイドの形状がヘリカル歯車となっているため,カム軸はカムプーリに対してひねられて,作動時期は進角する方向に切換わる.図1・37(b)では,逆にB 通路に油圧が作用しA 通路から油が流出するので,作動時期は遅角する方向となる.図1・38では,高速カムと副ロッカアームが,その両側に設けられた一対の低速カムと主ロッカアームに接しており,弁は主ロッカアームによって作動される.低速域では,副ロッカアームは主ロッカアームと分離されているため空作動し,弁は低速カムに従って作動する.一方,高速域では,主ロッカアームと一体になっている切換えレバーが油圧により副ロッカアームの下に入り込み,主ロッカアームと一体構造となる.これにより,ロッカアーム全体は揚程の大きい高速カムに従って作動するようになる.
