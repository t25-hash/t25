# 4・13・5 骨密度検査機器（bone densitometer )

> **章**: 第4章 検査・診断機器
> **節**: 4・13 その他の検査機器

a.はじめに 骨密度検査装置は骨塩量測定装置とも呼ばれ，
人体の骨中に含まれるカルシウムを主体とするミネラル成分を平面
骨密度として，すなわちg/ cmを単位として，非侵襲的に定量す
ることを目的とした装置である．骨粗鬆（こつそしょう）症
（osteoporosis ）罹患人口の増大につれ，近年，急速に普及しつつ
あり，現在では国内で約8000台以上の装置が稼働しているといわ
れている．
骨密度検査装置の歴史は比較的新しく，1986年ごろ，日本とア
メリカでほぼ同時に開発された装置に端を発する．国内では，1988
年に橈（とう）骨専用機が製造承認を受け，次いで翌年，腰椎専用
機が輸入承認を受けた．これらの装置および検査方式は，X線が
人体を透過する際に生じる減弱を測定論拠としていたために，現在
ではDXA（dual X-ray absorptiometry，二重X線吸収法）と呼
ばれている．
骨密度検査装置は人間の成長・退行過程に応じさまざまな用途が
あり，その代表的なものとして以下のものがある．
(１)小児，学童期における骨成長の評価
(２)若年成人期の最大骨量の評価
(３)退行期における骨粗鬆症の検診
(４)退行期における骨粗鬆症の診断
(５)投薬効果の判定
このほか，二次的に骨密度の増減をきたす他の諸疾患，例えば以
下の場合にも骨密度をモニタすることを目的として用いられる．
(１)腎疾患（人工透析などを含む)
(２)副甲状腺疾患
(３)胃切除に伴う骨軟化症
(４)ステロイド剤の長期投与
ここでは，骨密度検査装置の測定原理，検査例などについて概説
する．
b.測定原理 骨密度検査装置は，前述のとおりX線などの放
射線が人体を透過する際に生じる減弱を測定論拠としている．人体
は放射線の減弱という観点で見たとき，骨および軟組織からなる2
成分系と考えることができる．これら各成分の線吸収係数（すなわ
ち各成分の単位長さ当たりで生じる放射線の減弱率）は放射線のエ
ネルギーに依存して相違しており，この相違の程度もまたエネルギ
ーに依存している．この性質に着目すると，2種のエネルギーをも

し，透過放射線量を計測することにより，各走査点における各成分
の厚み，さらには重量を決定することができる．

f .測定結果のスコア化 骨密度検査装置による測定結果は，
平面骨密度，すなわち単位面積当たりの骨重量（g/cm ）として得
られる．しかしながら，実際の診療に際し，この結果のみから診断
を下すにはかなりの困難を伴うのが実態である．なぜなら，その数
値の妥当性を性別，年齢，病態などを考慮に入れ，総合的に判断す
るにはかなりの熟練と経験を必要とするからである．
このため，得られた測定値を，データベースとしてあらかじめ装
置に蓄えられている標準値（正常値）と比較してスコア化して用い
ることが広く行われている．また，学会，公的機関などから提示さ
れている診断基準，検診基準なども多くはこのスコアに基づいてい
る．この意味で測定結果のスコア化は極めて重要な概念といえる．
.Zスコア 被検者の測定値と同性別，同年齢の標準値の
比較により定義されるスコアである．具体的には次の2種がある．
(１)Zスコア＝(測定値－標準値)/標準偏差
(２)%Zスコア＝(測定値/標準値)×100（%)
このZスコアは，若年成人において，その人の骨密度を同年齢
の標準値と比較して，食生活，運動の指導などに役立てる場合によ
く用いられる．
.Tスコア これに対してTスコアは，同年齢の標準値
ではなく，同性別の若年成人の標準値との比較によって定義される
スコアである．具体的には次の2種がある．
(１)Tスコア＝(測定値－標準値)/標準偏差
(２)%Tスコア＝(測定値/標準値)×100（%)
