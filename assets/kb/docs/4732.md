# 1・4・3 排出ガスの排出特性と低減手法

> **章**: 第1章 火花点火機関
> **節**: 10・15 モード(Hot)

図1・43にアメリカの排出ガス試験モード(FTP: federal testprocedure,LA #4モード)を走行したときの積算HC排出量を示4―16内燃機関図1・43 LA #4モード走行時のHC排出特性図1・44 HC排出量低減手法の概念図表1・9おもな排出ガス低減技術機関冷間時(触媒活性前)機関暖機後(触媒活性後)機関排出ガス低減・/希薄化・燃料の微粒化・燃料噴射時期最適化・混合気乱れの促進・二点点火・細径プラグ機関排出ガス低減・外部EGR・内部EGR・燃料噴射時期最適化・ピストンクレビス縮小機関触媒暖機促進・冷間アイドル回転数上昇・点火時期遅角・/希薄化・排気マニホルド,排気管の低熱容量化・触媒の機関近接配置・触媒の低温活性向上触媒の浄化性能向上・/制御性の向上・触媒の三成分(HC,CO,NO)浄化領域の拡大・触媒の耐久性向上触媒・二次空気・HC吸着・HC吸着型三元触媒・電気加熱式触媒補助機構す.HCは走行開始初期に排出する割合が非常に大きく,規制値がLEV からSULEV へと厳しくなるに従いこの傾向はさらに強まる.機関や車両が冷えている状態から始動し走行を開始する冷間始動運転では,触媒が浄化を開始する温度に達するまでの間に未浄化のまま排出されること,低温のため燃料の気化が悪く,暖機後に比べ過剰の燃料を供給する必要があること,機関や車両の摩擦損失が大きく,吸入空気量が増加することがおもな要因である.排出ガス浄化の着目点は,図1・44に示すように触媒が浄化を開始する前の運転域では,1機関から排出する量そのものを低減する,2触媒を短時間で活性化させる,3補助的な手段を使うことであり,触媒活性化後は,4触媒の性能向上,5触媒の特性を十分に活用するための空燃比制御性の向上などがおもな技術手段となる.課題は自動車の運転性を犠牲にすることなく排出ガスを低減することであり,そのために,表1・9に示すようなさまざまな技術が規制値や機関の排気量,自動車の大きさなどに応じて使い分けられている.
