# 1・4・3 排出ガスの排出特性と低減手法

> **章**: 第1章 火花点火機関
> **節**: 10・15 モード(Hot)

HCは走行開始初期に排出する割合が非常に大きく、規制値がLEV からSULEV へと厳しくなるに従いこの傾向はさらに強まる。機関や車両が冷えている状態から始動し走行を開始する冷間始動運転では、触媒が浄化を開始する温度に達するまでの間に未浄化のまま排出されること、低温のため燃料の気化が悪く、暖機後に比べ過剰の燃料を供給する必要があること、機関や車両の摩擦損失が大きく、吸入空気量が増加することがおもな要因である。排出ガス浄化の着目点は、図1・44に示すように触媒が浄化を開始する前の運転域では、1機関から排出する量そのものを低減する、2触媒を短時間で活性化させる、3補助的な手段を使うことであり、触媒活性化後は、4触媒の性能向上、5触媒の特性を十分に活用するための空燃比制御性の向上などがおもな技術手段となる。課題は自動車の運転性を犠牲にすることなく排出ガスを低減することであり、そのために、表1・9に示すようなさまざまな技術が規制値や機関の排気量、自動車の大きさなどに応じて使い分けられている。
