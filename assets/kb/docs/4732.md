# 4・2・3 心音計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

心臓が拍動する際に発せられる音を心音という．医師は聴診器で
心音を聞き，患者の症状を診断している．聴診器は心臓内の弁の開
放・閉鎖に伴う音や血流の流出音を聞くことができる．弁が正しく
開かなかったり，通常より狭い流路があることを音の違いにより診
断する．この心音をマイクロホンでとらえ記録する装置が心音計
（phono cardiograph）である．心音計の構成は通常，図4･5のよ
うになっている．心音は図4･6のように心臓が収縮し血液が動脈に
送り出されるときにⅠ音，血液流出が終わるときにⅡ音が聞こえ
る．それ以外にも心臓内に血液が入り込んでくるとき，流れ出すと
きなどにⅢ，Ⅳ音が記録される．最近では，心音マイクロホンは加
速度型が通常用いられる．これは心音を胸壁の振動としてとらえ，
小型・高感度を実現できるためである．心音計では聞きたい心音の
音域によってローカットフィルタを選択して用いる．心音を正しく
記録するためには，計測する環境の騒音や振動を少なくすることが
必須である．
