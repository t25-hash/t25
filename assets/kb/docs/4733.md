# 4・2・4 脳波計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

脳はわれわれの生命および感覚をつかさどっている複雑な器官で
あり，その脳の大脳皮質から発生する自発性の電気変動を頭皮上に
取付けた電極から導出し増幅を行って記録する装置を脳波計
（EEG:electroencephalograph）という．この記録波形は脳波
（electroencephalogram）と呼ばれ，大脳皮質における多数の神経
細胞の活動電位の総和と見られている．波形の振幅，周波数成分，
位相などから病的異常の診断を行っており，いまや臨床検査の一つ
として欠くことのできないものとなっている．図4･7に脳波計のブ
ロック図を示す．脳波計には光で生体に刺激を与えることで脳波の
変化を見るための光刺激装置も内蔵されている．最近では，多量の
脳波データをディジタル化し電子媒体に保存するファイリング機能
も搭載されている．
a.脳波関連装置 データおよび画像処理技術の発達により複
雑な脳波データを少しでも判読しやすくするため，高速フーリエ変
換（FFT:fast Fourier transform）を用いた周波数分析機能や，
電位分布や周波数成分分布をディスプレイ上の頭図に色分けし，あ
たかも地図のように表示する電位マップ，周波数マップ表示機能な
どを搭載した解析プログラムや解析装置などが実用化されている．
また，無線伝送（EEGテレメータ）もしくは携帯型記録装置
（EEGホルタ）を用いて被検者を無拘束状態で長時間にわたりモニ
タリングする装置や，必要に応じその動態をビデオテープレコーダ
（VTR）におさめ，脳波と行動の関係を調べる脳波ビデオモニタリ
ングシステムもある．最近では脳波と映像をディジタル化して電子
媒体に保存し，時刻同期させて再生表示できる装置も開発されてい
る．
