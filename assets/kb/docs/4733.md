# 4・2・4 脳波計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

脳はわれわれの生命および感覚をつかさどっている複雑な器官で取付けた電極から導出し増幅を行って記録する装置を脳波計（electroencephalogram）と呼ばれ，大脳皮質における多数の神経位相などから病的異常の診断を行っており，いまや臨床検査の一つロック図を示す．
脳波計には光で生体に刺激を与えることで脳波の脳波データをディジタル化し電子媒体に保存するファイリング機能a.脳波関連装置 データおよび画像処理技術の発達により複たかも地図のように表示する電位マップ，周波数マップ表示機能なまた，無線伝送（EEGテレメータ）もしくは携帯型記録装置タリングする装置や，必要に応じその動態をビデオテープレコーダングシステムもある．
最近では脳波と映像をディジタル化して電子る．
あり，その脳の大脳皮質から発生する自発性の電気変動を頭皮上に細胞の活動電位の総和と見られている．
波形の振幅，周波数成分，として欠くことのできないものとなっている．
図4･7に脳波計のブ変化を見るための光刺激装置も内蔵されている．
最近では，多量のも搭載されている．
雑な脳波データを少しでも判読しやすくするため，高速フーリエ変電位分布や周波数成分分布をディスプレイ上の頭図に色分けし，あどを搭載した解析プログラムや解析装置などが実用化されている．
