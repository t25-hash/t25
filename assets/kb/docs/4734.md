# 1・4・5 自動車用触媒

> **章**: 第1章 火花点火機関
> **節**: 10・15 モード(Hot)

触媒(catalyst)とは、機関から排出されるHC,CO,NOを触媒反応により浄化する部品であり、ここでは自動車用触媒の主流として使われ、HC,CO,NOの3成分を同時に浄化できる三元触媒(three way catalyst)について解説する。希薄燃焼機関用触媒については1・4・8項を参照。a. 触媒の構造図1・48に触媒コンバータ(catalytic con-verter)の一例を示す。セラミックス製でハニカム構造のモノリス基材を、耐熱容器内にアルミナ繊維製のマットを用い保持した構造である。モノリス基材には耐熱ステンレスはく製のものもあり、この場合には容器にろう付けされる。触媒に使われる貴金属(Pt,Pd,Rh)は、アルミナなどの表面積が大きく耐熱性のある材料をモノリス基材の表面に塗布した後に担持される。b. 貴金属と浄化特性図1・49(a)~(c)に示すように、それぞれの貴金属(noble metal)の浄化特性には特徴があり、単独で使用した場合には高い浄化率が得られる空燃比範囲(浄化ウインド)は狭く、実用上十分な性能を得ることはできない。実用触媒では、図1・50のように複数の貴金属と助触媒材料(アルミナ、セリア、ジルコニアなど)を組合わせることで、浄化ウインドウを拡大して使われている。c. 三元触媒の浄化機構三元触媒は図1・51のように、おもに貴金属とアルミナ(AlO)と助触媒であるセリア(CeO)などから構成されている。セリアは酸化雰囲気下では酸素を貯蔵し、還元雰囲気下では貯蔵した酸素を触媒表面に放出する特性(酸素貯蔵能、oxygen storage capacity)がある。機関から排出されて触媒に流入するガスは、酸素センサを用いたフィードバック制御により理論空燃比(stoichiometric air-fuelratio)を中心に酸化雰囲気と還元雰囲気につねに変化しているが、触媒表面上では、上記のセリアの特性により雰囲気変動が吸収され、空燃比は理論空燃比近傍に保持される。このため、同時に高いHC,CO,NO浄化性能を得ることができる。d. velocity)空間速度は、単位時間当たりに流れるガスの容積(v)と触媒容積(V)の比で定義され、触媒単位体積当たりの処理能力の目安となる。空間速度が大きくて高い浄化率を得られる触媒ほど性能が優れていることになる。一般に、空間速度が大きくなると浄化性能は低下するため、触媒容積は、空間速度が数万から10万Hrになるように設定される。したがって、機関の排気量が大きく車両の重量が増大すると、触媒への流入ガス量が増加するため容積を増加する必要がある。実際の自動車では、機関の近くや車両の床下などに複数の触媒を搭載している。f. 触媒性能の劣化(catalyst degradation)。 熱・雰囲気による劣化貴金属を担持するアルミナなどの担体は、高温で使用すると相変化を起こして表面積が減少する。一方貴金属は、高温に酸化雰囲気が重なると凝集が進行し、活性点が減少する。その結果、活性開始温度が上昇し、浄化性能が低下する。触媒内部は1273K 近くの高温になる場合もあり、高温耐久性を向上させるために担体材料や担持方法等の改良が続けられている。. 被毒劣化触媒の被毒成分としては燃料や潤滑油中に含まれている硫黄、リン、鉛などがある。硫黄は、低温(873K 以下)の酸化雰囲気で貴金属や担体と硫黄化合物を作り、浄化性能を低下させるが、一時的な被毒であり高温還元雰囲気にすることで回復する。一方、リンや鉛は触媒上に逐次蓄積し、貴金属の被覆や細孔閉塞等による永久被毒現象を引き起こして、浄化性能を低下させる。各国とも被毒成分の低減を進めており、主要国のガソリンはすでに無鉛化されている。
