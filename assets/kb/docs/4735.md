# 4・2・6 血圧計（sphygmomanometer ，blood- pressure meter ）

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

心臓の収縮に伴い拍出された血液は全身を循環する．この循環系
の診断・解明のため生体情報として欠かせないものとして，血流量
の測定と血圧の測定が挙げられる．血圧は心臓から押し出された血
液が血管壁に及ぼす圧力である．血圧の単位は通常mmHgという
水銀柱の高さで表される．圧力は国際単位系（SI ）ではPa（パス
カル）で示す必要があるが，血圧では特例としてmmHgを使用し
てよい．1mmHg＝133Paである．動脈の血管内圧は通常，心臓
収縮期（収縮期圧，最高血圧，sys ＝systolic pressure）では120
mmHg程度，拡張期（拡張期圧，最低血圧，dia＝diastolic pres-
sure）では80mmHg程度を示す．また血圧波形の時間的積分の平
均を平均血圧（mean＝mean pressure）と呼ぶ．血圧測定装置は
血圧計といわれ，測定方法により観血式血圧計と非観血式血圧計に
大別できる．
a.観血式血圧計（直接法，invasive blood pressure monitor )
直接法に用いるセンサの例を図4･9に示す．圧力を受けてダイアフ
ラムの面が凹面状に変形し，中央に接続された可動ロッドが右側に
動く．この可動ロッドと周囲固定部との間にはひずみゲージがあ
り，ロッドの右移動に対して電気抵抗が高くなる抵抗体2本と，低
くなる抵抗体2本が四辺のブリッジ回路を構成するようにシリコン
基板上に拡散されている．このブリッジ回路により圧力に比例した
出力信号を得ることができる．実際に血圧測定するためには，水を
満たした細いチューブ（カテーテル）を動脈内に挿入留置し，カテ
ーテルのもとに圧力計を接続して測定する．最近では半導体技術の
進歩に伴い，シリコン基板にダイアフラムをエッチング技術で形成
し，このダイアフラム上の周囲に抵抗4本を拡散技術で作製するダ
イアフラム型圧力センサが広く使用されるようになった．このセン
サを応用して，血圧測定用の滅菌済み使い捨て圧力センサが供給さ
れるようになっている．
b.非観血式血圧計（間接法，non- invasive sphygmomanome-
ter ) 血圧測定の一般的な方法は体に侵襲を与えない間接的測定
法（非観血式血圧測定法）である．間接法として聴診法（aus-
cultatory method）とオシロメトリック法（oscillometric）の2
図4･11 電磁血流計原理
方式が広く使用されている．聴診法はごく一般的に行われる血圧測
定法である．上腕を腕帯（カフ）で加圧し，カフ部より下流の血管
からの血管音（コロトコフ音，Korotkoﬀsound）を聴診器やマイ
クロホンで聴き取る方法である．血圧よりも高い圧力をカフに印加
すると血流が遮断されるが，カフ圧を最高血圧より低くすると血管
音が発生し，また，カフ圧を最低血圧より下げると血流を遮断しな
くなるので，再び血管音が消失する．一方，オシロメトリック法で
はカフに圧力を印加することは聴診法と同様であるが，動脈の拍動
によってカフ内圧にわずかな脈圧振動を発生させる．この脈圧振動
とカフに印加した圧力との関係により，最高血圧・最低血圧・平均
血圧を求める．
現在では図4･10に示すオシロメトリック法が血圧計として広く
普及している．脈圧振幅が最大になるカフ内圧が平均血圧であり，
その約50 %を示す圧を最高血圧，約60 %を示す圧を最低血圧とし
ている．
