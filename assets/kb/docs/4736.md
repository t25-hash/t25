# 4・2・7 血流計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

血流測定は心臓のポンプ機能のうち極めて重要な血液の流量（流
速と管路断面積の積）の測定を血流計（blood flow meter ）を用い
て行う．ただし，血流計は通常露出された血管にプローブを装着す
る必要があるので侵襲的であり，臨床的にはあまり使用されない．
実用化されている血流計には電磁血流計と超音波血流計がある．

b.超音波ドップラー血流計 図4･12に示すように速度 で
移動している血流に超音波を照射し，その反射波を受信すると，血
流速度 によりドップラー効果を受け周波数が変化する．したが
って，この周波数の変化を計測すれば血流量を求めることができる．
センサを小型化できるが，装着状態により安定性に影響を受ける．
c.超音波トランジットタイム血流計 図4･13に示すように，
超音波が血管を透過する時間（トランジットタイム）は血流（血球
の速度）により変化する．血管とプローブの径が一致していなくて
も定量測定できるが，プローブが大きくなるので適用部位が限られ
る．
