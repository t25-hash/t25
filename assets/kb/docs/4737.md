# 1・4・5 項で述べたように,現在のガソリン自動車の排気浄化装置

> **章**: 第1章 火花点火機関
> **節**: 10・15 モード(Hot)

は三元触媒を使い,排出ガス中の三成分(HC,CO,NO)を同時に浄化する方式が主流であり,触媒の機能を十分に発揮させるために電子制御技術が駆使されている.図1・57に示すように,機関に吸入される空気量は空気流量計で計量され,それに基づいて機関に供給される燃料量が計算され,燃料噴射弁から供給される.しかし図1・50に示すように,排気三成分を同時に浄化できる空燃比の範囲は非常に狭いため,この制御のみでは高い浄化率を得ることは難しい.このため,排気マニホルドには排出ガス中の酸素濃度や空燃比を検出するセンサが取付けられ,最大の浄化性能が得られるように燃料量がフィードバック制御される.a. 吸入空気量の計量機関が吸入する空気量の計量方法には,吸気管圧力と機関回転速度から計算する(speed density)方式と空気流量計(air flow meter)で直接計量するマスフロー(mass flow)方式とがある.マスフロー方式には,熱式,ベーン式,カルマン渦式があるが,現在では熱式が質量流量に対応した出力を直接得ることができ,空気密度による補正が不要で,可動部分がなく,圧力損失が小さいことからおもに使われている.b. 酸素(O)センサ(oxygen sensor),空燃比()センサ(air-fuel ratio sensor)図1・58に酸素センサの構造を示す.ジルコニア素子(ZrO)には,高温状態で素子の内外面に酸素濃度差があると起電力が発生する性質がある.図1・58(b)4―19(a) 本体(c) 素子断面(b) 素子図1・58 酸素センサの構成(a)単一酸素センサ構成(b)酸素-酸素センサ構成(c)空燃比-酸素センサ構成O(or /)センサO(or /)センサ図1・59センサ位置と触媒の構成(c)に示すように,素子(固体電解質)の両面に白金電極を設け,内側には酸素濃度の高い大気を,外側には酸素濃度の低い排出ガスを導入する.排出ガス中の酸素濃度は理論空燃比を境に変化するため,起電力が変化し理論空燃比を検出できる.また,ジルコニア素子には,電圧を印加すると酸素濃度に比例した酸素イオンの移動が起こり,電流が流れる特性がある.空燃比センサは,この特性を利用して空燃比を計測する.酸素センサに比べ,広い濃度範囲の排出ガスの状態を判別することができる.センサ素子の外側は,電極の保護のためセラミックスで被覆され[図1・58(c)],さらに素子全体は保護管で保護されている.c. 触媒と排出ガスセンサの配置図1・59に示すような触媒と排出ガスセンサの配置が,排出ガス規制値の厳しさや1・4・10項で述べる触媒劣化検出等の要求により使い分けられている.図1・59(a)の単一酸素センサ方式では,機関の運転状態とセンサ信号から,下流側の触媒の酸素吸蔵量(1・4・5・c項参照)の状態を推定しながら,高い浄化率が得られるように供給空燃比を制御する.図1・59(b)の触媒を挟んでセンサを置く方式では,触媒後の酸素センサ信号により触媒の状態を検出できるため,図1・59(a)の方式に比べて正確な空燃比制御が可能になる.最近では,機関の吸入空気量や触媒の酸素吸蔵量をモデル化することで,さらに制御精度を高める技術が開発されている.
