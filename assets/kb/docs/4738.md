# 4・3・2 脳磁計

> **章**: 第4章 検査・診断機器
> **節**: 4・3 磁気測定装置

脳神経活動に伴って頭の周りに生じる磁場は脳磁図（MEG:
magnetoencephalogram）と呼ばれ，脳機能研究や脳疾患診断の
臨床において重要な役割を果たしている．脳磁図は非常に微弱なも
のであり，測定には超高感度のSQUID磁束計が必要である．脳磁
図測定の目的の大部分は，測定した脳磁図データより脳内部の磁場
発生源，すなわち脳活動源を推定することにある．そのために，頭
部の多くの場所からの磁場を同時に計測できる多チャネルSQUID
システムが脳磁計には必要である．多チャネルSQUID磁束計が本
格的に実用化になったのは，1990年代に入ってからである．1990
年にアメリカで37チャネルSQUIDシステムが開発されると，日
本国内の病院にも導入され，脳磁図の臨床応用に関する研究が本格
的に始められた．さらに頭部全体をカバーして測定できるようなホ
ールヘッドタイプの64チャネルや122チャネルシステム，さらに
日本においても超伝導センサ研究所で256チャネルシステムも開発
された．現在では，国内において数十台のSQUID磁束計が大学病
院や脳研究機関に設置され，脳機能研究や脳磁図の臨床応用に使用
されている．
図4･17に，エレクタニューロマグ社製の306チャネル脳磁計を
示す．この脳磁計は，204チャネルの平面型グラディオメータと
102チャネルのマグネットメータで構成されている．脳磁計は非常
に微弱な磁場を計測するため，磁気シールドルームの中に設置する
ことが不可欠である．特に検出コイルがマグネットメータの場合だ
と，ノイズの影響を直接受け，SQUIDが動作しないこともある．
脳磁図は脳波と同じように脳内の電気現象をとらえたものであ
り，X線CT，MRI ，PETなどの画像診断装置とは異なった特徴
を有しており，以下のような特徴がある．
(１)機能的情報が得られる
脳磁図計測は神経細胞の電流変化を測定対象としているので，脳
の機能的情報がミリ秒の時間分解能で得られる．
(２)無侵襲計測である
X線CTやPETなどの診断装置は，無侵襲とはいっても体外か
ら生体に何らかの物理エネルギーを加え，生体との相互作用を測定
して体内の様子を推定するものである．この点，磁気計測は体内か
ら発生する磁気を検出するのであって真に無侵襲である．
(３)電源局在推定が数mmの精度でできる
生体組織では比透磁率は真空の透磁率にほぼ等しい．このため磁
気計測では，脳波測定での欠点である頭蓋骨の影響による信号の減
衰が無視できるので，特に大脳皮質の電源推定に有効である．例え
ば，聴性誘発脳磁図では2～3mmの精度で音の周波数に対応した
ニューロン群の反応部位が推定できる．
(４)脳波で測定できない脳内電源が検出できる
極性が互いに逆向きにある二つの電源は閉ループ電流を作り電源
の近傍だけを短絡的に電流が流れるので，脳波では検出しにくい．

しかし，脳波に信号となって現れてこない閉ループ電流はループ面
に垂直に磁場を生成するので，脳磁図で検出することができる．
ヒトを対象とした非侵襲脳機能計測において，現在のところms
オーダの時間分解能で脳機能局在を推定する手法がほかにないた
め，脳磁図計測は脳機能研究に必須の手段であることは間違いな
い．脳磁図は脳の基礎研究や言語活動，記憶，認知といった高次脳
機能の研究のみならず臨床診断にも有用であり，そこでは，てんか
んスパイク波の検出をはじめ，脳腫瘍や脳血管障害に伴って出現す
る徐波，さらには事象関連電位等に伴う超低周波の脳電気活動の磁
気計測が行われている．脳磁計は，ハードウェア的には多チャネル
化，低ノイズ化も進み，ほぼ満足する域に達してきた．しかし，脳
磁図を用いて特定の高次脳機能をよりよく抽出するためのパラダイ
ムや，脳のダイナミックな機構を理解するための新しいパラダイム
および計測の手法がまだ十分には開発されていない．また，脳磁図
計測で雑音の大きい，すなわち信号対雑音比が極端に小さい信号の
電源推定や，広がりをもつ電源推定のための信号処理技術や逆問題
手法，およびそのためのよりよい脳内電源モデルの構築はまだ不十
分であり，今後の発展が望まれる．
