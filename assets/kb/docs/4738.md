# 1・4・8 希薄燃焼機関(lean burn engine)の排気処理

> **章**: 第1章 火花点火機関
> **節**: 10・15 モード(Hot)

希薄燃焼機関は、均質燃焼方式と成層燃焼方式に分類される。均質方式はポート噴射機関でおもに使われ、均質で希薄な混合気を使い、シリンダ内の乱れを強化することで安定した燃焼を実現している。成層方式はシリンダ内に直接燃料を噴射し、全体としては点火不可能な超希薄混合気でありながら、プラグ付近に点火可能な濃い混合気層を作ることで安定した燃焼を可能にしている。成層燃焼機関(stratified charge engine)は、燃料消費量の低減効果が大きく、過去長い間研究されてきたが、1990年前半に希薄燃焼ガス中のNOを浄化できるNO吸蔵還元型触媒が日本で開発され、さらに電子制御により精密な空燃比制御が可能になったため、1996年に日本の自動車会社により世界に先駆け量産化された。酸素が過剰な雰囲気で使える触媒として、NO選択還元型触媒やNO吸蔵還元型触媒が開4―20内燃機関発されてきたが、現在では浄化性能の面から後者が広く使われている。図1・60にNO吸蔵還元型触媒の浄化機構を示す。空燃比が希薄のときにはPt上でNOがNOに酸化され、吸蔵材(Ba,K,Naなど)と反応し硝酸塩として吸蔵される。吸蔵されたNOは還元雰囲気下で分解し、Pt上で還元成分と反応してNに還元される。このように吸蔵と還元を繰返すことによりNOが浄化される。実際の機関では図1・61に示すように、触媒に吸蔵されたNOの量を推定し、設定量以上になると空燃比をスパイク的に濃くし、NO還元に必要な量の還元剤を供給する。この触媒がNOを吸蔵還元できる温度範囲はおおよそ550~820K にあるため、機関からやや離れた位置に置き使用される。b. 吸蔵還元型触媒の劣化三元触媒と同様に熱劣化と硫黄被毒劣化が発生する。硫黄による被毒は、NOを吸蔵する吸蔵材の特性上避けられない。硫黄は燃料や潤滑油中に含まれており、硫酸塩として吸蔵され、その量が増加するとともにNO浄化性能は低下する。被毒を抑制する方法として、触媒内の温度を約870K以上の還元雰囲気に制御して還元脱離する方法や、チタニアなどの酸性物質を担体に添加し、硫黄の被毒を抑制する方法が使われている。また燃料中の硫黄量の低減は、被毒による性能低下を抑制することができるため、将来の進展が期待される。c. 排気浄化システム図1・62に示すように、成層希薄燃焼で運転できる領域は全運転域ではなく、高負荷時や始動暖機過程では、空燃比は理論空燃比あるいはそれより濃い側で運転される。このため図1・63のように、三元触媒が機関に近接配置され、NO吸蔵還元型触媒は下流側に配置されるのが一般的である。三元触媒は、希薄運転時のHC,COとそれ以外の空燃比で運転時のHC,CO,NOを、吸蔵還元型触媒は希薄運転時のNOを浄化する。また機関は成層希薄燃焼を実現するために、ポート噴射機関に対し、燃料噴射弁やピストン形状や吸気ポート形状の変更、気流制御弁の採用などの改良が行われ、機関自身から排出するNOを低減するために電子制御式EGR 弁が使われている。
