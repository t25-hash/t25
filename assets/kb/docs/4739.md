# 1・4・9 燃料蒸発ガスの処理

> **章**: 第1章 火花点火機関
> **節**: 10・15 モード(Hot)

自動車から排出する蒸発ガス(evaporative emission)は、図図1・60 NO吸蔵還元機構図1・61空燃比の過濃化制御とNO浄化特性料が、駐車中や走行中に機関や外気の熱によって蒸発し、大気中に漏れ出すHC,2シールや配管系のゴム部品から透過し、大気中に漏れ出すHC,3給油時に燃料タンクの空間部分にある燃料蒸気が、給油燃料と置換し大気中に放出するHC(以上燃料系HC)、4塗料やプラスチックやタイヤから放出するHC(非燃料系HC)に分けられる。カリフォルニア州のPZEV 規制(表1・8)に対応する自動車では、駐車中に大気に漏れ出す燃料蒸発ガスを零にすることが義務付けられている。a. 燃料蒸発ガス低減システム(evaporative emission controlsystem)図1・65に系の構成例を示す。駐車中に燃料タンクから蒸発する燃料蒸気は活性炭が詰められた容器(キャニスタ、charcoal canister)に導かれ、吸着貯蔵され大気中への放出が防止される。走行時には機関の吸気系へ通じる通路(パージ通路)が開かれ、吸気管負圧を使ってキャニスタ内に空気を流すことでHCを脱離し、吸着能力を回復させる。走行中に燃料タンクから蒸発する燃料は、キャニスタに吸着されると同時に機関にも吸入されるが、キャニスタには、続く駐車時や給油時に発生する燃料蒸気をすべて吸着できる能力が残されている必要があり、走行中に十分な脱離量が得られるように設計される。給油中に放出するHCも同じキャニスタに吸着されるが、この場合には給油口からの放出を抑制しつつキャニスタに導く必要がある。一般には、給油する燃料で給油管の先端を密閉し(リキッドシール)、給油口と給油ノズル先端は給油燃料により発生する負圧を利用し、外部から空気を巻込み放出を制御する。タンク内の蒸気図1・63 直噴ガソリン機関の排気浄化システム構成図1・64 蒸発ガス発生部位図1・65 燃料蒸発ガス低減システム(アメリカ規制対応)図1・62 成層運転領域図1・66 駐車中の燃料蒸発ガス評価(DBL)図1・67 走行中の燃料蒸発ガス評価(running loss test)は、給油口とタンクの油面差を利用してキャニスタに送り込まれる。ゴム部品からの透過量の低減に対しては、内面にフッ素コートを施したり、ナイロンホースなどの透過しにくい材料が使われる。以上のような低減技術の採用により、燃料蒸発ガスは未規制時に比べて95~99%が削減されている。また、給油時の蒸発燃料の低減にはガソリンスタンドで対応する方法もあり、アメリカの一部の州で実施されていたが、現在では自動車での対応が義務付けられている。b. 燃料蒸発ガス規制(evaporative emission regulation)と評価方法燃料蒸発ガスの規制と評価方法(evaporative emissiontest procedure)について、アメリカを例に解説する。蒸発ガスの規制には、駐車中(DBL: diurnal breathing loss)と停車中(hot soak loss)の規制、走行中(running loss)の規制、給油時(ORVR)の規制がある。いずれも密閉された部屋(SHED :sealed housing for evaporative determination)で評価される。これらの評価は決められた手順に従って連続して行われるため、キャニスタが燃料蒸気を吸着する能力と、機関がその蒸気を吸入できる能力の両方が評価される。走行中と給油時の規制はアメリカでのみ実施されている。. 駐車中の燃料蒸発ガス評価図1・66に示すように、夏季の屋外に3日間駐車されたことを想定した試験法である。密閉された部屋に温度制御装置を設け、1日の気温の変化を模擬し、3日間の評価を行う。規制に適合しているか否かは、車両から放出されるHCの1日当たりの最大量で判定される。日本やヨーロッパもほぼ同じ評価方法を実施しているが、1日の駐車を想定している。. 走行中の燃料蒸発ガス評価夏季の日中を想定した走行を模擬するために、密閉された部屋にシャシダイナモメータを設置し(図1・67)、評価モードを走行する。車両の走行とともに燃料タンク内の温度は上昇するため、蒸発ガスの量は増加するが、評価期間内に規定量以上のHCが大気中に放出してはならない。この試験では、走行中に機関が吸入できるHC量の処理能力が評価される。. 給油時の燃料蒸発ガス評価ガソリンタンドと同じ設備を備えた給油専用の部屋(ORVR SHED)で給油を行い、給油口とキャニスタから漏れ出るHCの量と給油性能が評価される。
