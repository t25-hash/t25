# 4・4・1 ベッドサイドモニタ

> **章**: 第4章 検査・診断機器
> **節**: 4・4 生体情報モニタ

ICU，CCU，救急病棟などに収容されている重症患者をはじめ，
電図や呼吸数などのバイタルサインを監視する必要がある．この目
的に対して使用されるのが生体情報モニタ（単にモニタと呼ばれ
る）であり，大まかにはベッドサイドで使われる一人用生体情報モ
ニタと，ナースステーションなどで使われる多人数用セントラルモ
ニタに分類される．当初，重症患者用を主目的として使用されてき
たモニタも，最近では中間病棟や一般病棟用としても用途が拡大さ
れ，臨床上必須の機器として位置付けられるようになってきた．
一人用生体情報モニタ（ベッドサイドモニタ，bedside monitor ）
では，基本的なバイタルサインとしての心電図，血圧，呼吸数，体
図4･ 18 ベッドサイドモニタ 図4･19 4人用無線式セントラルモニタ
では計測技術の進歩が目覚しく，非観血的，無侵襲でかなり多くの
生体情報が得られるようになってきている．特に指先などにトラン
スデューサを装着するだけで，簡単に連続的に測定できる動脈血酸
素飽和度（SpO ）は基本パラメータの一つになっている．非観血
血圧測定においても連続的に血圧を測定できるようになり，患者の
負担をより軽くし，かつ質の高いデータが測定できるようになって
きている．さらには脳波やそのスペクトル解析を行い患者の意識レ
ベルを測定したり，連続で心拍出量を測定できるものもある．ベッ
ドサイドモニタでは測定できない外部機器のデータ，例えば人工呼
吸器のデータや新生児，未熟児を対象にした経皮酸素分圧モニタの
データ等もベッドサイドモニタに取込み，同じ患者のデータとして
一元管理できるようになってきており，用途に応じて使われる．
このように測定され処理された生体情報のデータは，CRT
（cathode ray tube）あるいは液晶ディスプレイに波形および数値
で表示される．ここでベッドサイドモニタに血圧，心拍数値などの
上限および下限を設定しておけば，この範囲を超えたとき，音また
は光により警報を発し，医療スタッフに緊急事態を知らせるように
なっている．さらに連続記録部がある場合は，警報発生時に自動的
に数十秒間心電図が記録される．また，発生した不整脈を保存して
一覧表として表示したり，患者の治療経過をレビューするために，
心電図波形や数値データを長時間保存してトレンドグラフや波形の
長時間圧縮表示をすることにより診断治療に有効利用されている．
このベッドサイドモニタのうち最も一般的なものが，心電図（心
拍数を含む）の監視を行う心電図モニタ（ハートモニタともいう）
であり，一般病棟をはじめとして，無線テレメータの簡便さととも
に広く使われている．多パラメータ用のベッドサイドモニタやさら
に多機能のポリグラフは，一部には無線テレメータの方式がある
が，有線方式が主流である．また，得られたデータをセントラルモ
ニタなどに送ることなどもモニタの重要な役割の一つであり，最近
ではセントラルモニタとの間をネットワークで接続し，ネットワー
ク上のどこからでも患者情報（ベッドサイドモニタの情報）を得る
ことが可能になってきている．
