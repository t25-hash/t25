# 1・5・2 手動変速機(MT : manual transmission)

> **章**: 第1章 火花点火機関
> **節**: 1・5 動力伝達系

手動変速機は上記3機構を備え,運転者の操作により作動する.図1・70に一般的な5速MT 断面図と機構例を示す.a. 動力遮断機構手動変速機の遮断機構は,内燃機関と変速機の間に設けられた乾式単板の摩擦クラッチが主流である.一部にはクラッチディスクを複数枚使用し,クラッチ容量を増大したものもある.図1・71に一般的な乾式単板クラッチの構造を示す.クラッチ(clutch)は,フライホイール,クラッチカバー,クラッチディスクから構成され,クラッチペダル操作による内燃機関と変速機間の動力遮断および接続,ならびに半クラッチ操作による円滑な発進機能を有する.またクラッチには,内燃機関の回転速度変動を低減するフライホイール機能も有する.b. 変速機構手動変速機の変速は,車両走行状態に応じた適切な駆動力を得るために,運転者の意志により最適な変速段を選択する.手動変速機内の歯車列は,平行軸上に同期装置(synchron-izer,synchromesh)を備えた変速歯車を数個配列した構造となっ図1・69走行性能線図ている.乗用車の例をとると,前進4~6段の歯車列と後進1段の歯車列で構成されるものが主流である.さらに,より大きな駆動力が必要な車両においては,極低速段や副変速機構が追加された例がある.前進段の場合,それらの変速段を形成する歯車どうしは常時かみあっており,運転者がその歯車段を選択したとき,その歯車段に設置されている同期装置の作用により回転速度が同期させられることで,円滑な変速操作が可能となる.通常,黄銅製の同期リングと鉄製の同期コーンをもった同期装置が主流で,それぞれの被同期歯車に1個設置されているが,操作性および耐久性の向上を目的として,その摩擦面を複数有するマルチコーンタイプの同期装置や,摩擦面をカーボン材などにして高容量化と耐久性を両立させたものもある.図1・72に同期装置の構造例を示す.c. 前後進切換機構手動変速機の前後進切換機構は,常時歯車がかみあっている前進歯車とは別に専用の歯車列をもち,後進時に後進専用の歯車どうしがかみあう選択しゅう動方式と,常時後進用歯車どうしがかみあって,かつ同期装置を備えた常時かみあい方式がある.図1・73に常時かみあい式後進歯車の一例を示す.
