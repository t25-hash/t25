# 1・5・3 自動変速機(AT : automatic transmission)

> **章**: 第1章 火花点火機関
> **節**: 1・5 動力伝達系

変速機の遮断ならびに変速操作を自動化したものを自動変速機と呼ぶ.乗用車用の自動変速機としては,前進3~7段,後進1~2段の有段変速機および変速比を無段階に制御する無段変速機(CVT :continuously variable transmission)が実用化されている.有段変速機は,平行軸式(parallel axes type)と遊星歯車式図1・71 クラッチ図1・72 同期装置(ダブルコーンタイプ)図1・70 5速MT 断面図1・73 常時かみあい後輪歯車(planetary gear type)に大別される.図1・74に平行軸式5速ATの断面図を示す.平行軸式は入力軸と出力軸上に配置された常時かみあい式の各速歯車列と,おのおのに専用に設置された湿式クラッチを有し,各クラッチを締結および開放することで変速を行う.したがって,各速の変速比を歯車列ごとに選択できるため,変速比の設定自由度が大きいことを特徴とする.一方,遊星歯車式は,複数の遊星歯車列(planetary gear sets)とクラッチおよびブレーキを組合わせて構成され,複数のクラッチおよびブレーキの締結および開放により変速を行う.一軸上に機構を構成できることや,複数の遊星歯車列の組合せで多段化できることを特徴とする.図1・75に遊星歯車式5速AT の断面図を示す.本例では,3個の遊星歯車列を用いて前進5速,後進1速のAT を構成している.また,手動変速機の歯車列を基に,クラッチおよび変速操作部に電動モータや油圧アクチュエータを設置し,これらをコンピュータ制御で操作させる変速機もある.無段変速機は,入出力軸にベルトやチェーンで駆動される一対のプーリを設け,片側のプーリの可動部を電動モータや油圧により軸方向に連続的に動かすことでプーリへの巻付き半径を変化させ,無段階な変速比を得る金属ベルト式が主流である.図1・76に一例を示す.他の無段変速機としては,ベルト式より大容量に適するとされるディスクとパワーローラ間のトラクションドライブを利用したトロイダル式CVT がある.a. 動力遮断機構自動変速機における遮断機構は,流体を介図1・74 平行軸式5速AT 断面4―23して動力の伝達を行うトルクコンバータ(torque converter)を採用するものが多い.図1・77にトルクコンバータの断面図を示し,図1・78に3要素一段二相式トルクコンバータの性能特性図を示す.トルクコンバータは,内燃機関に直結したポンプと変速機の入力軸に直結したタービン間を,流体の運動エネルギーを利用して動力を伝達する機構である.内燃機関の回転速度が低いときは動力の伝達は少なく,車両が停止していても内燃機関は回転可能である.回転速度を上げ,ポンプとタービン間の速度差が大きい(速度比が小さい)と,タービンの流れの出口にあるステータの働きでトルクを増幅し駆動力を増す.さらに回転速度を上げていくと,ポンプ,タービン,ステータが一体で回転しトルクの増幅はなくなる.内蔵された直結クラッチ(lockup clutch)を締結すると,内燃図1・76 金属ベルト式CVT 断面図1・77 トルクコンバータ図1・75 遊星歯車式5速AT 断面図1・78 トルクコンバータの性能特性図4―24内燃機関機関の動力は流体を介さずに直接変速機の入力軸に伝えられ,トルクコンバータによる動力損失がなくなるので伝達効率が向上する.この場合,内燃機関のトルク変動吸収のため,ダンパ機構が必要となる.また,トルクコンバータのトルク増幅機能をもたない流体継手や,図1・76に示すような湿式多板クラッチや,電磁クラッチを用いた遮断機構もある.流体を用いた機構は動力を完全に遮断できないので,車両停止時変速用のクラッチを開放する方式も採用されている.b. 変速機構自動変速機の変速機構は,各種の動力伝達部と変速制御部から成る.変速制御部は,油圧制御回路部,変速指令制御部,クラッチおよびブレーキ部から構成される.図1・79に変速制御部の概念図を示す.油圧制御回路部は,オイルポンプ,各種油圧弁(hydraulic valve),電磁切換弁(solenoid control valve)および比例電磁弁(servo solenoid valve,linear solenoid valve),油路から構成され,オイルポンプで作られた油圧を調圧し,変速指令に基づき的確にクラッチおよびブレーキを作動させる役割がある.油圧の制御性は,温度や作用圧力および回路構成要素数などで大きく左右されるので,フィードバック制御が採用されているものもある.変速司令制御部は,セレクトレバーの位置,内燃機関および変速機の各軸の回転速度,内燃機関の発生トルクや運転者の加減速意志の信号となる内燃機関スロットル開度などを各種センサにより検出し,コンピュータで変速時期,クラッチおよびブレーキへの油圧の切換え時期および油圧調圧値を算出し,各電磁弁に駆動信号を出力している.クラッチおよびブレーキ部は,油圧によりピストンを押して摩擦係合要素を締結させる役割がある.ここでは内燃機関の動力を確実に伝える締結力を発生させることと,切換え時にスムーズな変速が行えるような摩擦特性およびクラッチ機構を有する必要がある.電子装置の進歩により,油圧制御回路部の油圧弁の代わりに電磁弁(solenoid valve)が直接クラッチおよびブレーキを制御するものや,コンピュータが油圧制御回路部に一体で構成されるものもある.温度や部品精度による油圧制御性のばらつきは,コンピュータの計算速度や記憶容量の増大により著しく改善されている.また,高回転時の遠心油圧(centrifugal hydraulic pressure)による油圧制御性の悪化を防止するため,ピストン室の裏側にオイルだまりを設ける油圧キャンセラ付きクラッチ機構も変速ショックの低減に寄与している.c.
