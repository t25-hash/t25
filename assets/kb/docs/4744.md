# 1・5・4 駆動方式の種類

> **章**: 第1章 火花点火機関
> **節**: 1・5 動力伝達系

自動車の駆動方式は,大きく二輪駆動(2WD : 2wheels drive)と四輪駆動(4WD : 4wheels drive)に大別される.2WD はFR,FF,MR,RR の形式があり,また4WD は駆動力の配分方式により大きくパートタイム,フルタイムに分かれる.図1・81にその体系図を示す.2WD とは前後どちらかの車輪を駆動させる方式で,以下のように分類される.FF(front engine front drive):内燃機関を前部に配して前輪を駆動する形式で,推進軸などの駆動用部品が少なく軽量,高効率が特徴である.FR(front engine rear drive):内燃機関を前部に配して後輪を駆動する形式で,前輪が舵取り機能,後輪が駆動機能と機能が分担されているため操舵性は良い傾向にある反面,推進軸などの駆動伝達部品が増えるため重量増加や効率低下が課題となる.また,特殊な車両(スポーツカーなど)に採用されている駆動形式で,内燃機関を車両中央に配して後輪を駆動するMR(midengine rear drive)や,内燃機関を車両後部に配して後輪を駆動させるRR(rear engine rear drive)がある.図1・82に駆動方式とその代表的な構成を示す.4WD は,前後輪に駆動力を伝達する装置を備え,悪路や荒地も(a) 前進時(b) 後進時図1・80 遊星歯車式前後進切換機構駆動形式2WDFRFFMRRR4WDパートタイムフルタイム固定配分可変配分図1・81 2WD/4WD 体系図2WDFFFRMRRR図1・82 2WD の駆動方式4WDフルタイム4WDFF固定配分FR可変配分FF可変配分パートタイム4WDFR図1・83 4WD の駆動方式図1・84 FF固定配分式フルタイム用動力分配装置容易に走行できることを目的とした駆動方式である.近年では雨天や積雪時の走行安定性向上や,4WD 時の走行抵抗の低減を目的としたさまざまな機構,構造が開発されている.パートタイム4WD:前輪または後輪の二輪駆動と四輪駆動を任意に選択できる方式で,高い駆動力を得られる反面,駆動系の重量増加や駆動効率の悪化が生じる.フルタイム4WD:前輪と後輪の間の駆動力を伝える部分に回転差を吸収する機構を備え,駆動力を常時一定の割合で四輪に伝える固定配分式と,駆動力が必要になったときに適切な駆動力配分を自動的に行う可変配分式(パッシブタイプ,トルクオンデマンドなどとも呼ばれている)に大別できる.これは,ビスカスカップリングや湿式クラッチを用いた継手にて車輪の差回転などを感知する.近年では,FFの長所を生かした4WD が増加している.図1・83に基本的な駆動方式とその代表的な構成を示す.4WD において駆動力は動力分配装置を介し,前輪または後輪に伝達される.図1・84にFFを基にしたフルタイム固定配分用動力分配装置の一例を示す.また,駆動力配分を電子制御によりきめ細かく可変制御させる機構が開発されている.図1・85に電磁コイルおよび湿式クラッチを用いた電子制御式FF用後輪駆動装置の一例を示す.4―25図1・85 FF 可変配分式フルタイム用後輪駆動装置図1・86 内燃機関のBSFC図1・87 AT の内部損失要因
