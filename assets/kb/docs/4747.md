# 4・5・4 断層撮影装置

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X線装置

断層撮影装置（tomography system）は，X線管装置とフィル
ムの移動方式により，平行平面運動方式と円弧運動方式に分けられ
る．これらの両方式の概要を図4･31 に示す．また，X線管装置
（焦点）の軌道により，X線管球とフィルムが一方向に運動する直
線軌道方式と，回転中心を中心として曲線軌道運動を行う多軌道方
式に分類される．
ここでいう断層撮影はCTのようなコンピュータを用いた方法と
は異なり，直接フィルムに断層像を写し込む撮影法である．単純撮
影によるX線写真は，撮影部位のX線方向にあるすべての組織，
器官が重なった状態でフィルム上に写し込まれているが，断層撮影
では断層像の上下に位置する組織，器官の像をぼかし，見ようとす
図4･30一般撮影用システム例
る断層面を相対的に強調したX線像が得られる．これにより病巣
の位置，大きさ，形状などの立体的な状態を知ることができる．
CTが人体の横断面像を得るのに対し，断層撮影では縦断面像が得
られるのが大きな特徴で，一般撮影像との対比も容易なため，かつ
ては多用されていた．最近は画像処理技術の飛躍的な発達とMRI
装置の普及により立体画像や任意の断層像が容易に得られるため，
断層撮影装置の需要はほとんどなくなっている．
