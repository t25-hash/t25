# 4・5・5 循環器用X線システム

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X線装置

循環器系診断用システムは，血管内に挿入したカテーテルから造
影剤を注入してその流れをX線像でとらえる血管造影検査を行う
システムである．全身の血管の疾患を対象としているが，特に疾患
の多い心臓と頭部，および肝臓，下肢を対象とした検査が多い．最
近では透視像を確認しながらカテーテルを用いて血管内で患部の治
療を行うインタベンショナルラジオロジー（IVR:interventional
radiology）が急速に広まってきており，システムの用途が診断支
援から治療支援にも拡大している．
これらの検査では，被検者をできるだけ動かすことなくあらゆる
方向から血管の観察ができることが求められる．このためX線管
装置と映像系装置をC形のアームの両端に取付け，アームの回転
とスライドの組合せにより，X線の投影方向を自在に変えられる
ような構造を採用しているシステムが多い．
この診断では，他のX線診断に比べ，造影剤の速い流れを追跡
して撮影するために，短時間ばく射と高速連続撮影の技術がハード
ウェアおよびソフトウェアの両面から要求される．このためこのシ
ステムでは，一般のX線診断機器に比べ，より高精度のものが要
求されると同時に，専用のユニットや周辺機器も多岐にわたってく
る．
このシステムの技術面における特殊性として，動きが早く，吸収
コントラスト差の少ない臓器の診断，微細血管の診断，立体的な血
管内の血流の流動状態の診断などが挙げられる．このため微細かつ
高度な技術を要することに加えて，つねに検査に危険を伴う面が
