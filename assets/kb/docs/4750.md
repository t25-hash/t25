# 1・7・1 はん用機関

> **章**: 第1章 火花点火機関
> **節**: 1・7 はん用機関等

火花点火機関の中でも自動車や二輪車のように道路を走る輸送用機械ではなく、定置または移動しながら使われる多種多様な機械の動力源として利用されているものははん用機関(engine for gen-eral use)と呼ばれており、表1・15に示すような農業・林業、土木・建設、電気・産業やレジャー・家庭用などの機械に広く搭載されている。はん用機関の中で最も多く生産されているのが、出力19kW 以下の小型はん用ガソリン機関であり、わが国では2001年に2ストローク、4ストローク合わせて約600万台が生産され、80%以上が直接あるいは機械に搭載され世界中に輸出されている。小型はん用ガソリン機関は各種作業機械への搭載性と廉価なことから、軽量コンパクトな空冷単気筒およびV 形二気筒が主流となっており、用途に応じて2ストローク機関と4ストローク機関が選択されているが、近年では環境問題への対応のため、これまではほとんど2ストローク機関が搭載されていた手持ち式の機械においても4ストローク機関が採用されるようになってきた。小型はん用ガソリン機関の使用条件と使用環境は用途や地域によってさまざまであり、家庭用として年間で数時間しか使われないものから、業務用としてほとんど毎日、長時間連続して使われるもの、負荷条件においても軽負荷から高負荷まで一定または大きく負荷が変動するものがある。使用環境の面では258K 主機部分クランクケースとシリンダは、アルミダイカストの一体成形で特殊鋳鉄製のシリンダライナを鋳込み、ピストンしゅう動部の耐摩耗性向上を図る構造が一般的に採用されている。シリンダの外周には冷却フィンを付けて冷却を行うが、OHV 式やOHC式のシリンダ部はSV 式に比べて吸排気のバルブやポートがないため構造が簡単で冷却性に優れ、シリンダの熱変形が小さいことから、燃焼室への潤滑油上がりを大幅に低減できる。したがって、潤滑油消費量の低減と出力や排出ガス性能の劣化が少ないといった長所がある。クランクケースは燃焼による爆発力に加え、搭載機械から加わる外力にも十分に耐えられるだけの剛性をもたせる必要があるが、一方では軽量化の追求も必要であり、リブを効果的に配置してその両立を図っている。ピストンとコネクティングロッドはともにアルミニウム製で軽量化を図り、ピストンは低圧鋳造品に熱処理を施してアルミニウムの機械的性質を向上させ、コネクティングロッドは高圧のアルミダイカストで油スクレーパを一体成形するものが大多数を占めているが、一部には鍛造品も採用されている。一般的なアルミダイカスト製法では鋳造時に巻き込まれた空気が鋳巣となって製品内部に残り、熱処理を施すことができないという問題があったが、ダイカスト技術の進歩により、鋳巣がなく熱処理が可能な製法が開発されたため、大排気量の機関においても鍛造からダイカスト化が進んでいる。コネクティングロッドの大端部は本体とキャップに二分割され合わせ加工により仕上げられている。その際、クランクピンへの組立時にズレが生じないように本体とキャップの合わせ面にインロー加工を施すことが多い。クランク軸は炭素鋼の鍛造またはダクタイル鋳鉄を使った鋳造の一体構造で、一端を搭載機械への動力取出しに利用しているため、搭載機械の要求に応じて何種類もの出力軸形状が準備されている。主軸受にはボールベアリングを採用して爆発荷重のみならず、搭載機械を駆動する際に発生する曲げ荷重やスラスト荷重に耐えられる強度・剛性をもたせている。クランクピン部はアルミニウム製コネクティングロッドの直軸受方式を採用するため、高周波焼入れが施されている。単気筒機関ではピストンなどの往復質量が発生する慣性力を釣合わせることはできないが、クランクピンと反対側に釣合いおもりを付けて上下方向の慣性力の一部を左右方向へ置換え、機関の振動を低減している。比較的排気量の大きい機関ではダイナミックバランサと称して不釣合いおもりをもったバランサ軸をクランク軸と逆回転させることにより、一次の慣性力を釣合わせ、機関の振動低減を図っている。カム軸は鋳鉄製の鋳物でカム歯車を一体化し、カムローブ部には耐摩耗性を向上させるために高周波焼入れを施したものが一般的に採用されているが、騒音低減や軽量化などを目的に樹脂製のカム軸も一部の小排気量機関には採用されている。軸受はクランク室のアルミニウム部分をそのまま利用する直軸受方式である。シリンダヘッドはアルミダイカスト製でOHV 式やOHC式では吸排気弁とポートおよび点火プラグを配置するため、SV 式に比べて構造が複雑である。燃焼室形状はバスタブ型、多球型やペントルーフ型が採用されているが、設計に当たっては弁配置や冷却性のほかに、燃料タンク、マフラ、空気清浄器などの補機部品レイアウト、点火プラグの保守性なども考慮する。シリンダヘッドの設計は、出力・排出ガス性能に加えて耐久性にも重大な影響を及ぼすため、十分に検討することが必要である。. 始動装置(starting system)始動にはロープ始動方式が一般的に用いられている。始動ロープを引くことにより始動装置の内部にあるラチェット式の爪が飛び出し、はずみ車に取付けた起動プーリを介してクランク軸を回転させる。機関の初爆に必要な点火エネルギーを発生させるためには、クランク軸の回転速度をおよそ300~350rpm以上にする必要があり、始動時の圧縮乗り越しを軽くしてクランキング速度を高めるため、自動圧縮抜き装置(オートデコンプ)をカム軸に備えることが多い。始動ロープを放せばぜんまいばねの戻し力により、引出したロープは始動装置内部のリールに巻き戻される。排気量が大きい機関や用途によってはセルモータによる始動方式も採用されている。. 点火装置点火装置はフライホイールマグネト(flywheel magneto)方式で、ほとんどの機関が無接点のトランジスタ点火方式を採用し、保守点検を不要としている。はずみ車に張り付けた永久磁石により点火コイルの一次側を発電するこの方式は、ロープによる始動方式と組合わせることにより電池を不要とし、小型はん用ガソリン機関を軽量コンパクトなものにしている。点火コイルははずみ車外側のシリンダ部に直接取付ける方式が一般的に採用されているが、点火コイルには一次側と高電圧を発生する二次側のコイルに加えトランジスタが内蔵されているため、353K以上の高温にならないように配慮する必要がある。点火時期は出力性能やノックなどの運転調子、排出ガス性能などを考慮して決定するが、上死点前0.1~0.14πradの範囲で固定した点火時期を採用する機関が多い。また、始動回転域での点火時期を上死点付近まで遅角させて始動時のキックバックを防止するものや、低中速域でのノッキングと高速域での出力性能を両立させるために、機関回転数におおむね比例する電子式進角方式を採用することもある。小型はん用ガソリン機関は用途や使用者によって運転状態がまちまちであり、点火プラグの熱価の選定には注意が必要である。特に、アイドリングや軽負荷での運転を多用する場合には、焼け型の点火プラグに交換して中心電極温度を自己清浄温度以上に保ち、電極の汚損を防止しなければならない。. 燃料供給装置小型はん用ガソリン機関は図1・182~1・184に示すように、機関の最上部に燃料タンクを配置し、気化器(carburetor)への燃料供給は重力による自然供給方式を採用している。燃料タンクを機関に直接取付けるため、タンク容量は定格出力負荷で約2時間の無補給運転を目安に設定することが一般的であるが、長時間の無補給運転を必要とする発電機では、大型の燃料タンクを発電機フレームに搭載したり、車両などに搭載される場合には、別置きの燃料タンクから燃料ポンプを使って燃料を供給する方式も採用される。燃料タンクを直接機関に取付けることから、機関からの伝熱やふく射熱によりタンク内のガソリンが温まらないようにすることや、機関振動に耐えうる強度とすることが必要である。また、搭載機械によっては傾斜状態で運転されることもあり、注油口の位置を高くしてふたから燃料が漏れるのを防止したり、Y 字配管と称して燃料タンクの両側から燃料を取出すものもある。気化器はフロート式で固定ベンチュリをもつサイドドラフトタイプが一般的に採用されており、チョーク弁による始動系統以外は、主系統と低速系統のみをもつシンプルな構造ですべての運転条件に適応している。低燃費化や排ガスの浄化のためには空燃費(/)を大きくする必要があるが、始動・加速やハンチングなどの運転性能やシリンダヘッドの冷却を考慮して適合を行う必要がある。気化器は機関からの伝熱を防止するために熱きり用のインシュレータを介して取付けられている。燃料系にまつわる問題として燃料供給通路内で発生するベーパロック、フロートチャンバ内のガソリンが沸騰するパーコレーションおよび絞り弁などが結氷するアイシングがあり、機関の搭載状態や運転状況により対策が必要となる。また、田植機のような農業機械では使用する期間が短く長期間保存されるため、気化器のフロートチャンバにガソリン抜き用のドレンを追加し、腐食ガソリンによるジェットの詰まりを防止するほか、内部に特別な防錆処理を施すこともある。. 調速装置(governor)小型はん用ガソリン機関では、負荷変動に対して機関の回転速度を一定に保つ調速装置が付いているのが一般的である。クランク軸と一定の回転比で回転するガバナ歯車やカム歯車などにガバナウエイトと称するおもりを付け、おもりの発生する遠心力と操作リンク系に取付けたガバナスプリングとの力の釣合いにより、気化器の絞り弁開度を自動的に調整する方式が機械式調速装置である。ガバナ特性は、定格出力で運転している状態から急激に負荷を抜いた場合の瞬間的な回転上昇の割合を示す瞬時変動率と、安定時の回転速度との割合を示す整定変動率、回転が安定するまでの時間を示す整定時間で表すことができる。整定変動率は無負荷高速回転速度と定格負荷運転時の回転速度の変化量であり、一般的には5~7%程度としている。なお、調速装置は機関の過回転防止にも役立っている。. 潤滑装置潤滑はコネクティングロッドの大端部にある油スクレーパによる引っかけ飛まつ方式が一般的であるが、一部の単気筒やV 形二気筒機関では油ポンプによる強制圧送式が採用されている。油スクレーパによりはねかけられた潤滑油が霧状になってクランク室内に充満することにより、シリンダ、ピストン、クランク軸やカム軸などのしゅう動部を潤滑する。OHV 式やOHC式機関ではロッカアームなどの動弁系の部品がシリンダヘッドのロッカ室にあるため、ブローバイガスを空気清浄器に排出するブリーザ機構をロッカ室に設け、ブローバイガスとともに霧状の潤滑油をロッカ室へ導いて潤滑する。潤滑油はしゅう動部の焼付きを防止するほか、機関を冷却する役割も果たしているが、飛まつ量が多すぎる場合にはブリーザからの潤滑油の吐出しに注意する必要がある。特に単気筒はクランクケース内圧力の変動が大きいため、ブリーザ機構の遮油構造に工夫が必要である。また、クランク室内に潤滑油をためる構造のため十分な潤滑油量を確保することができず、使用者の点検不足による焼付きを未然に防止するため、規定油量以下になると機関を自動的に停止させる油面センサの装着率が増えている。. 冷却装置(cooling system)冷却ははずみ車と一体成形したシロッコファンによる強制空冷式が一般的に採用されている。ファンカバーとシリンダバッフルにより、シリンダヘッドやシリンダの高温部へ冷却風を導き効率的に冷却する。冷却不足は機関性能の低下のほか、ヘッドガスケットからのガス漏れやしゅう動部の焼付きなど機関そのものの耐久性に影響するため避けねばならず、馬力当たり1mの風量を目安に設計する。冷却風は始動装置4―55にあけられた窓から取り入れるが、搭載機械によっては機関が筐体などで囲われることがあり、冷却風の取入れ口の確保や温められた冷却排風が筐体内に滞留しないように工夫する必要がある。
