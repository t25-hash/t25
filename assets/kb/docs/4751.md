# 1・7・2 ロータリ機関

> **章**: 第1章 火花点火機関
> **節**: 1・186 に最高出力と機関質量の関係を示す。

a. 概要まゆ型の二節ペリトロコイド(peritrochoid)曲線から成る内周面をもつロータハウジング(rotor housing)の内部でロータ(rotor)が回転する、バンケル型ロータリ機関(rotaryengine)が自動車用として実用化されている。ロータリ機関は、往復運動部分や動弁機構をもたないため、高回転まで振動、騒音が少なく、また、基本構造が簡単で小型、軽量という特徴をもっている。b. 作動原理図1・187にロータリ機関の基本構造、図1・188にその作動を示す。ロータはロータ歯車と固定歯車とのかみあいにより自転しながら公転する。ロータの回転に伴い、その外周とロータハウジングに囲まれた三つの部屋が回転しながら容積変化し、吸入、圧縮、膨張、排気の四つの行程を繰返す。ロータに加わる爆発荷重はロータに挿入された偏心軸(eccentric shaft)に伝わり、動力として取出される。ロータが1回転自転し、ロータの周りの各部屋が吸入から排出までの四行程を完了する間に、出力軸は3回転(1080°)する。すなわち、一行程は出力軸が270°回転する間に行われる。往復動機関の一行程は出力軸180°の間に行われるため、同一出力軸回転数で比較すれば、ロータリ機関の一行程の時間は往復動機関の1.5倍である。そのため、吸気行程期間が長くとれ、高回転での体積効率が高めやすく、高回転運転で長所が引出せる機関である。また、行程期間が長いことにより、トルク変動も滑らかである。一方、行程が長いこと、全体に作動室が扁平で、表面積/容積比(/比)が大きいことにより、ガス漏れ損失、冷却損失が多い傾向がある。c.
