b.ガンマカメラ ガンマカメラ（gamma camera）は，厚さ
約1cm，直径25～50cmの1枚のシンチレータ［通常はNaI
( Tl ）結晶］に多数の光電子増倍管を光学接続したものである．
1958年にH.O. アンガーにより設計されたのが最初である．γ線
（gamma ray）により発光した結晶中の位置は，光電子増倍管の出
力信号の演算により決定される．すべての光電子増倍管の出力信号
の和によりγ線のエネルギーが判別できる．
140keVのγ線に対するカメラの空間分解能は3mm程度を得
ることができる．しかし，実際にはカメラの前面に装着するコリメ
ータから線源までの距離により空間分解能は劣化する度合いが増
す．
c .コリメータ γ線はレンズによる屈折作用を受けることが
ないため，放射能の分布を撮像するには，γ線の飛来する方向を特
定するためのコリメータ（collimator ）をガンマカメラの正面に装
着する必要がある．
コリメータの種類には，ピンホールコリメータ，平行多孔コリメ
ータ，コンバージングコリメータ（焦点型コリメータ），ダイバー
ジングコリメータ，スラントホールコリメータ，ファンビームコリ
メータなどがある．用途に応じて孔の長さ，大きさ，隔壁の厚みが
異なるいく種類ものコリメータが用意されており，高エネルギー用
や低エネルギー用だけでなく，高解像力用や高感度用等の特定性能
に的を絞ったコリメータも作られている．一般に，コリメータの幾
何学的な分解能と効率とは相反する関係にあり，一方を立てれば他
方が立たない傾向が見られる．
d.走査法 断層像を再構成するための計測投影データを得る
ために，ガンマカメラの正面を被検体に向けながら体軸を回転中心
として被検体の周りを回転走査する．ガンマカメラのシステム空間
分解能は，線源が検出器のコリメータ面から離れるほど劣化するの
で，解像度の低下を防止するために回転半径はできるだけ小さくす
る必要がある．通常の回転走査は円軌道であるが，だ円軌道を行う
場合もあるのはこのためである．
ガンマカメラ回転型SPECT装置の欠点は，検出器を回転しなが
ら投影データを収集するため収集時間が比較的長くかかり，短時間
で動態画像を得ることが難しいことである．ガンマカメラを円周上
に3個もしくは4個と複数配置することでこの欠点を克服し，同時
に感度を向上した装置も使用されている．
