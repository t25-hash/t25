# 1・7・3 航空用機関

> **章**: 第1章 火花点火機関
> **節**: 1・186 に最高出力と機関質量の関係を示す。

航空用ピストン機関の全盛期には出力2500kW 程度の星型二十八気筒エンジンも存在したが、現在では高出力機関はガスタービンにとって代わられ、出力150~250kW クラスの四気筒または六気筒の水平対向型機関が小型航空機用用途または無人機(UAV : un-manned aerial vehicle)用途に用いられている。表1・16にアメリカのテレダイン・コンチネンタル社、テキストロン・ライカミング社およびオーストリアのロータックス社の製品例を示す。機関形式としては正面面積が小さく、釣合いの点で有利な水平対向式が主流であり、小型機に用いられる水平対向式機関の構造および構成上の特徴は、以下のとおりである。(1) クランク軸を挟むように左右にクランク室およびシリンダを配置することにより、クランク室周りの剛性の確保と、クランク軸が直列型に比べて短くできることによるねじり振動に対しても強い構造を有する。(2) 星型機関に比べて正面面積が小さく、シリンダの張出しも翼の方向に向いているため、飛行抵抗を減らす面でも有利である。(3) 燃料系統はフロート式気化器に代わり、シリンダ吸入口の近くに燃料噴射装置(fuel injection system)を備えるものが今日では主流であり、高空性能に対する空燃比制御の適正化を図っている。(4) 各シリンダに独立した二重の点火系統をもち、点火の信頼性を確保している。(5) 過給機(turbocharger)をもつ場合は、排気タービン駆動の過給機を装備するのが主流となっている。これまで大きな技術的変化がなかった航空用ピストン機関であるが、昨今NASA GAP(General Aviation Propulsion)プロジェクトに代表されるような、航空用有鉛ガソリンに代わり、安価で、環境負荷も小さいジェット燃料を使用するディーゼル機関の研究が4―58内燃機関行われている。これらの機関形式は水平対向式であり、騒音低減と小型化をねらった水冷式を採用し、電子制御技術を生かした機関操作性の向上が検討されており、今後の研究開発が注目される。文献(1)
