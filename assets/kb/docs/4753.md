# 2・1・1 燃料噴射と噴霧特性

> **章**: 第2章 圧縮点火機関
> **節**: 2・1 燃焼の基礎

a. 燃料供給圧縮点火機関(compression ignition engine)とは、円筒形往復動ピストンとシリンダとで囲まれた閉空間に存在する気体に、燃料の燃焼によって生成する熱エネルギーを与え、気体の状態変化を利用して熱エネルギーの一部を仕事として取出す容積形内燃機関の一種である。ピストンの圧縮作用により生じた高温高圧の気体中へ燃料を噴射し、自発点火により燃焼を行わせることから圧縮点火機関と称する。発明者の名前にちなみ、一般にディーゼル機関(diesel engine)と呼ばれている。熱力学的には、大型低速機関は定圧サイクル(constant pressure cycle)、小型高速機関は複合サイクル(combined cycle)を理想サイクルとする。固体、液体、気体いずれの形態の燃料も利用可能であるが、一般に液体燃料が利用される。液体燃料は、噴射ノズル先端の小孔を通じて高速燃料噴霧として間欠的に燃焼室内へ供給され、高温高圧空気からの熱輸送により蒸発しながら空気と混合する。形成された燃料蒸気-空気混合気中において酸化反応が進行し、やがて自発点火、燃焼過程へと移行する。燃焼室内混合気の総括当量比は1以下で空気過剰状態にあるが、燃料と空気との混合が不十分で燃料過濃あるいは過薄な領域が生じることが多く見受けられる。このような局所当量比の空間的不均質性は、不完全燃焼あるいは有害燃焼排出物増加の要因となる。これを防ぐためには、噴霧に適度の運動量を与え空気との混合を促進し、空気利用率を高める必要がある。一般に、混合気形成ならびに燃焼過程に利用できる時間は短く、特に高速機関ではこれが顕著となる。蒸発および混合を含む物理現象に関わる特性時間は化学反応時間に比べて極めて長く、これが燃焼過程を律則する支配因子となる。したがって、燃焼期間を短縮するためには、運動量、熱および物質移動速度の向上が肝要である。燃料の微粒化促進は、高温空気と接触する燃料の表面積増大による蒸発速度の上昇をもたらすことから、燃焼促進において重要な役割を果たす。混合気形成、点火遅れ、燃焼、有害物質生成、動力性能に多大な影響を及ぼす燃料供給過程において重視される特性は、噴射時期、噴射期間、噴射圧力、噴射量、噴射率、噴射安定性、噴霧の巨視的ならびに微視的構造などが挙げられる。最適な燃料供給を実現するため種々の燃料噴射装置が開発され実用に供されてきたが、ボッシュ式噴射装置が最も広く普及している。コモンレール式などの新しい方式も登場してきており、これらについては2・3節で詳しく説明する。b. 燃料噴射広く利用されているボッシュ式噴射装置の系統図を図2・1に示す。この装置主要部は、高圧で燃料を供給する噴射ポンプ(injection pump)、燃料を微粒化しながら高速で燃焼室内へ噴射する噴射ノズル(injection nozzle)およびそれらを連結する噴射管(injection pipe)で構成される。噴射ポンプ内では、クランク軸により駆動されるカムと戻しばねにより往復運動するプランジャの下降行程で燃料が吸入され、上昇行程で燃料の圧縮および送出が行われる。噴射ポンププランジャ室上部に取付けられた吐出し弁はばねの作用により逆止弁として働くほかに、吸戻し弁としての効用を発揮する。噴射終期にプランジャ室内燃料圧力が低下して吐出し弁が下降すると、噴射管内に存在する燃料の占める容積が増大し、噴射管内圧力の急激な低下が生じる。この吸戻し作用により、急速な噴射終了を実現することができる。噴射ノズルは針弁とそれを押さえるばねを有する自動弁となっており、燃料圧力が上昇しばねの力以上に達すると針弁が上昇し燃料噴射が生じる。このような噴射系において見受けられる典型的な噴射率(injec-tion rate)、針弁揚程(needle lift)、噴射ノズル入口圧力および噴射ポンプ吐出し弁出口圧力の時間変化を図2・2に示す。噴射率とは単位時間あるいは噴射ポンプカム軸単位角度当たりに噴射される燃料の質量であり、針弁揚程は針弁の着座位置から上方への移動距離を示す。針弁揚程が正の値となり、噴射ノズルから噴射が行われる期間を噴射期間(injection period)という。噴射開始以前には噴射管内に前サイクルの噴射終了後高圧に達した燃料が存在し、この燃料圧力を残留圧力と称する。噴射ポンププランジャ室内に閉じ込められた燃料はプランジャの上昇とともに圧縮され、その圧力が残留圧力と吐出し弁ばね圧力の和に達すると吐出し弁が開き、吐出し弁出口圧力の上昇(点a)とともにノズル方向への燃料圧送が始まる。プランジャ断面積とプランジャ揚程の積である幾何学的燃料送油量を圧送期間で除した値を送油率と呼び、噴射ポンプ性能の指標として用いられることがある。吐出し弁出口で発生した圧力波は音速でノズル方向へ伝ぱし、短い遅れ時間後(点b)ノズル入口圧力が上昇し始める。その後、圧力波は反射波となって吐出し弁方向へ伝ぱし、以後噴射管内を往復する。プランジャがさらに上昇すると、噴射管内に圧送された燃料量が増加してノズル入口圧力も上昇する。ノズル入口圧力が針弁を押さえるばね力以上に達すると(点c)、針弁が上昇してノズル先端から燃料の噴出が始まる。このときのノズル入口圧力を開弁圧力と称する。また、プランジャによる燃料の圧縮開始から針弁の上昇を伴う噴射開始までの期間を噴射遅れという。噴射開始直後、針弁上昇により噴射管内に燃料の占める容積が増大するとともに燃料噴出により燃料の質量が減少するため、ノズル入口圧力図2・1 噴射過程は一時的に低下する。ノズル入口における圧力波は吐出し弁出口に達するまでに減衰するため、吐出し弁出口圧力波形にはこのような一時低下は現れない。その後プランジャ室から燃料の圧送が続くため、吐出し弁出口圧力およびノズル入口圧力は上昇し続ける。圧力波は噴射管の両端で反射され、管壁の摩擦により減衰しながら往復運動を続ける。このような圧力波の挙動によりノズル入口圧力は時間的に大きく変動し、針弁の運動と相まって、噴射率波形にも顕著な時間的変動が現れる。プランジャによる燃料の圧送が終了すると、吐出し弁が閉じ吐出し弁出口圧力は急激に低下する。それから少し遅れてノズル入口圧力の低下が始まり、ばね力以下になると(点d)針弁が着座し燃料噴射が終了する。このときのノズル入口圧力を閉弁圧力と称する。ノズル開弁時と閉弁時との受圧面積の差により、閉弁圧力は開弁圧力よりも低い。吐出し弁およびノズル針弁が閉じた後、圧力波が両者間を伝ぱしながら減衰して残留圧力に収束する。この過程が図2・2に示すノズル入口圧力および吐出し弁出口圧力波形に反映されている。残留圧力が毎サイクル同一であれば、定常で安定した燃料噴射が持続する。定常噴射のうち、1サイクル当たり一度の燃料噴射が生じる場合を正常噴射と呼び、これに対して異常噴射がある。噴射終了後、ポンプ側から伝ぱした反射波がノズル側に達し、ノズル入口圧力が開弁圧力以上になると再び燃料噴射が生じる。これを二次噴射(secondary injection)と称する。二次噴射が生じると、高温で低酸素濃度の雰囲気中へ燃料が噴射されるため、不完全燃焼、すす生成の増大、未燃炭化水素排出量の増加、粘着性炭素質の析出の原因となり、重大な機関運転の阻害要因となる。運転条件によっては、三次噴射が生じることもある。残留圧力、燃料噴射量ならびに噴射率波形がサイクルごとに変動する場合、これを不整噴射と称する。これらの異常噴射に影響を与える因子として、燃料噴射量、噴射ポンププランジャ速度に関連する機関回転数、噴射管長、ノズル開弁圧力、ノズル噴孔面積と噴射管通路面積との比、ノズル針弁質量、針弁上部ばねの強さ、ノズル形状、燃料性状などが挙げられる。噴射終期において、ノズル入口圧力が低下するため燃料が大粒径の液滴として低速で燃焼室内に噴出することがある。これが後だれ(dribbling)と呼ばれ、すす発生の要因となる。噴射ポンププランジャ室上部吐出し弁が下降すると、吸戻し作用により噴射管内の圧力が急激に低下する。急速な噴射終了により、低速での燃料噴射を短縮し、後だれ現象の発生を抑制することができる。このほか、噴射系に関わる異常現象として、噴射圧力降下時にノズル針弁の下降動作が不十分なため燃焼ガスが噴射系内に入り込む逆流現象、ポンプ吐出し弁の吸戻し量の過大や激しい燃料圧力変動によって噴射管やノズル針弁に生じるキャビテーションエロージョン(cavitationerosion)などが挙げられる。c.
