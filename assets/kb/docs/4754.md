# 2・1・2 燃焼特性と排出物の発生機構

> **章**: 第2章 圧縮点火機関
> **節**: 2・1 燃焼の基礎

a. 燃焼特性圧縮着火機関の燃焼は,2・1・1項で説明した燃料噴射装置を用い,圧縮された空気中に高圧噴射した燃料が空気と混合して反応し自己着火して始まる燃焼である.一般的には以下のような過程を経ることを特徴としており,それらの過程と関連する因子を図2・5に,またそれに対応する自動車用直接噴射機関(圧縮比17,排気量2.2L)の燃料噴射率,燃焼圧力,熱発生率(rateof heat release)の例を図2・6に示す.. 着火遅れ通常,燃料は上死点直前の適切な時期に噴射される.燃料は,高温高圧の周囲空気との速度差で生じるせん断力によって微粒化されて液滴群となり,さらに高温の空気によって加熱され蒸発する.これによって,空気との混合,拡散が進行しながら酸素との化学反応により着火に至る.このような燃料噴射の開始から着火に至る時間を着火遅れ(ignition delay)と呼ぶ.この時期の圧力線図を見ると,噴射燃料の蒸発によって熱が奪われ,また初期の反応が吸熱反応であることから,非燃焼の場合に比べて温度と圧力がわずかながら低下する傾向がある.さらに発熱反応に至って熱発生が本格的に起こり,圧力が非燃焼時の圧力を上回ることになり,この時点を着火時期とみなすことができる.このように,着火の時期に関しては,燃焼によって圧力が上昇する時期と図2・5 圧縮着火機関の燃焼過程と諸因子図2・6 圧縮着火機関の燃焼特性して圧力線図から特定するのが一般的であるが,燃焼室内を観察し,燃焼反応に伴う発光を検出する時期とすることもできる.着火遅れに影響する因子としては,燃料の圧縮着火性の指標であるセタン価(cetane number)が挙げられる.また,空気の温度や密度,酸素濃度,不活性ガスの種類と濃度も影響因子である.このように着火遅れは,燃料の微粒化や分散,加熱・蒸発,混合・拡散等の物理的な遅れと燃料と酸素による化学的な遅れからなる.両者は重なり合っているが,低温では化学遅れが支配的であり,高温になると化学反応が十分早くなるので,物理的な遅れが支配的になる傾向がある.実用的な観点から,着火遅れτを以下のようなアレニウス型の式で表すことができる.τ=욗exp욀ここに,:空気の絶対温度,:空気の圧力,,:燃料固有の値,욀:活性化エネルギー,:一般ガス定数.. 初期燃焼初期燃焼とは,着火遅れの期間中に蓄積された燃料と空気の予混合気の一部が反応によって自己着火する燃焼であり,爆発的な燃焼となって急激な圧力上昇を生じる.この燃焼は着火遅れと密接な関係があり,着火遅れが大きいほど着火可能な予混合気の量が増えるので,その燃焼量も増加する傾向がある.また,このような急激な圧力上昇は,エンジンの振動・騒音の原因ともなる.これが過大になる現象をディーゼルノック(dieselknock)と呼び,エンジンの焼付きや破損に至ることもあるので,設計上,おおむねクランク角度1°当たりに500kPa程度以下の圧力上昇に抑える必要があるとされている.過大な初期燃焼,さらに振動・騒音を抑えるには,着火遅れを抑制するとともに,初期の噴射量を抑制することが有効である.このためには,適切な値のセタン価の燃料を使用することと,初期噴射図2・7 噴射量(負荷)の影響量を抑制することが有効である.また,少量の先立ち噴射(パイロット噴射,pilot injection)を行って,着火しやすい混合気を形成する方法も効果的で,自動車用エンジンでは騒音・振動対策として広く用いられている.. 主燃焼初期燃焼によって予混合気が燃焼すると燃焼圧力と温度が上昇し,引き続き噴射される燃料は蒸発しやすくなる.また,反応速度も上がるので,これと乱れを伴った空気との拡散・混合がおおむね律速となる乱流拡散燃焼の形態が主体となる.初期燃焼の特徴である予混合気燃焼が終わると熱発生率は極小値をとるが,その前から主燃焼は開始している.主燃焼では,燃料の噴射率と後述する空気流動とが燃焼速度を左右する極めて重要な因子となる.初期燃焼で誘起される乱れも拡散燃焼の促進に役立つ効果がある.また,図2・7に示すように,燃料の噴射量を増すと初期燃焼はあまり変化せず,拡散燃焼が主体の主燃焼量が増加することになり,これが負荷の増大による圧縮着火機関の燃焼の特徴となっている.. 後燃え拡散燃焼が進行して燃焼室内の空気が消費されて不足気味になり,またガス流動に伴う乱れも高温の状態や時間経過によって減衰する.さらに,膨張に伴って温度が低下することにより燃焼が緩慢になることがあり,これを後燃えという.この時期の燃焼は上死点からも遠ざかり,熱効率の悪化の原因にもなるので,混合を活発にして燃焼を促進する必要がある.また,後燃えはすでに発生した燃焼ガス内の黒煙・微粒子が再燃焼する時期であり,その低減対策の観点からもこのような燃焼の遅延を改善する必要がある.b. 空気流動と過給の役割ガス流動特性や過給は,空気の利用促進を通じて燃焼特性を,さらに熱効率や排出物の生成を左右する極めて重要な因子である.図2・8に示すように,空気流動にはスワール(swirl)とスキッシュ(squish)の二つの形態がある.吸気行程において吸気ポートに方向付けられてシリンダ内に流入し,圧縮行程でシリンダ軸まわりに旋回流となるのがスワールであり,吸気ポート形状によってその旋回速度を設定することができる.エンジンの回転速度に対する旋回速度の比をスワール比と呼び,空気流動を決める重要な設計因子とされている.スワールは,圧縮行程で燃焼室の中に押込められると旋回速度を増す効果があり,燃焼室径の小さい燃焼室のほうが増速効果が大きい傾向がある.半径方向に向かって噴射された燃料が噴霧を形成し,これに対してスワールは円周方向に旋回するので効果的に空気が導入される.ただし,スワール比が大きすぎると隣接する噴霧が重なり合い,その領域が空気不足の状態となって熱効率の低下と黒煙・微粒子の悪化を招く.このような現象をオーバスワールと呼ぶ.噴口数が多いほど起こりやすいので,空気混合気の形成には噴4―63図2・8 燃焼室内における流動口数とスワール比との組合せには適切な関係が存在し,設計者によって経験的に決められている.また,より高い圧力で噴射を行う方式では噴霧がもつ運動量が大きく,これによって空気導入が増すので,スワールの役割は相対的に小さくなる傾向がある.圧縮行程が終了する直前では,ピストン頂面の間げき部にあった空気が燃焼室内に流入する流れがあり,これをスキッシュ,また膨張行程で間げき部に流出する流れを逆スキッシュと呼ぶ.このような流れは,燃焼室口径が小さいほど強くなるので,スキッシュの効果を高めるために燃焼室の口縁部の径を小さくした構造のものをリエントラント型燃焼室と呼び,自動車用の機関では広く利用されている.このような巨視的な流動は,噴霧内への空気導入を全体的に高める一方乱流渦を伴うので,これによって空気と蒸発燃料の微視的な混合が促進され,燃焼を促進する効果がある.また,実際の燃焼室内の流れは,吸気管と吸気ポートを通って流入する空気の流動に始まり,圧縮行程においてスワール,シリンダ軸方向の流動(タンブル),スキッシュが複合的に影響し合う形態をとり,燃料噴霧によっても流動と乱れが誘起されるのでさらに複雑なものとなる.このような流動と乱流特性に関しては,最近各種のレーザ計測法によって把握されるようになっている.また,後述するようにコンピュータを使った数値流体力学的な手法で予測できるようになっており,吸気系や燃焼室形状の設計に役立てられている.過給(super charge)が燃焼に及ぼす効果を調べた例を図2・9に示す.このように,過給によって着火遅れが短くなるとともに,拡散燃焼を行う主燃焼が促進される効果があり,熱効率の向上にも寄与する.ターボ過給では排気タービンで圧縮機を駆動して吸気圧を高めるので,排気エネルギーの一部を回収する効果があり,ここでも熱効率の改善効果をもたらす.さらに,後述する排出ガス対策で悪化する燃焼を改善するのにも有効である.c. 粒子状物質(PM)粒子状物質(PM : particulatematter)は局所的に当量比がおおむね2を超える燃料過剰な燃料噴霧内で生成される.そのような領域では,蒸発した燃料が熱分解と脱水素反応によって低級化してアセチレン等の不飽和成分となり,さらにそれらが重合したポリアセチレン等を経て多環芳香族化し,微粒子の前駆物質を形成する.これが2nm以下の核となり,同様の炭化水素成分が表面に凝集して層状に成長して固体粒子を形成するに至る.さらに粒子が10~30nm程度の粒径へと巨大化が進むとともに,そのような粒子どうしが房状に集合して全体として数百nm程度の粒子群を形成する.その一方,各過程で酸素によって酸化し消滅する過程も同時に進むため,結果的に排出される微粒子は,生成されたもののうちのごく一部にすぎない.また,膨張行程の後半や排気過程,さらには大気中に放出される過程でこれに周囲の炭化水素が凝縮し吸着する.これらは極めて高速で複雑な生成経路をとるので,未解明の点も多いのが実情である.排出された粒子状物質を分析すると,黒煙の主体である固体炭素図2・11 炭化水素(HC)と一酸化炭素(CO)HC(hydrocar-bon)は,燃料噴霧の中心部や壁面に当たる燃料過剰な領域や,逆に空気が極めて過剰な領域では,酸化反応が遅くなり未燃焼のまま排出される.また,エンジンの冷始動時や暖機の際に,燃焼室壁面に当たって液膜となり蒸発が遅れた燃料や不整噴射による燃料,燃焼ガスの混入する潤滑油等もHCの排出の原因となる.また,このようなHCは酸化されて酸化物やCO(carbon monoxide)となるが,酸化が遅れるとそのまま排出されることになる.ディーゼル機関の燃焼は全体として空気過剰の条件で運転されるので,このようにして排出されるHCやCOは高温で酸化されやすく,対策はそれほど難しくない.また,排気系に酸化触媒を設置することによって大幅な低減が可能である.d. ディーゼル機関のガス流動と燃焼の数値シミュレーション.
