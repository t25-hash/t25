# 4・7・1 磁気共鳴装置

> **章**: 第4章 検査・診断機器
> **節**: 4・7 NMR・MRI装置

磁気共鳴装置（magnetic resonance system）とは，核磁気共鳴
現象［NMR（nuclear magnetic resonance）現象］を利用し，緩
和時間を介して物質や生体組織の性質を調べる装置である．固体の
物性や化学物質，アミノ酸・タンパク質などの生体物質の構造を調
べるための高分解能NMR装置と，生体内部の構造や生体機能を
画像計測するための磁気共鳴画像装置（MRI装置）に大別される
が，両者の機能をもつ研究用の装置もある．前者は薬品開発など
に，後者は画像診断や治療システムの一部として用いられる．
