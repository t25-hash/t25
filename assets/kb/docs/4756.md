# 2・2・2 合成系燃料

> **章**: 第2章 圧縮点火機関
> **節**: 2・2 燃料

石油代替および低公害性の観点から天然ガスからの合成系燃料が開発されており、図2・14に示すように天然ガスを改質した合成ガス(CO+H)からGTL(gas to liquid)軽油、DME、メタノールが製造される。これらの性状を表2・8に示す。GTL軽油(GTL diesel fuel)はまず合成ガスをFT(fischer-tropsch)プロセスによりワックスに変換し、次にこれを水素化分解したものを分留して製造する。GTL軽油は蒸留性状は軽油と同等であるが、硫黄分、芳香族分、オレフィン分を含まないために排ガス性能面からは圧縮点火機関用燃料として理想的である。すでに数箇所で実用化されているが、今後も多くのプラント建設計画がある。DME(dimethyl ether)は常温で気体であるが、圧縮により簡単に液化することができLPG(liquefied petroleum gas)とほぼ同様な取扱いが可能である。セタン価が高く、また燃焼時にすすを生成しないため良好な圧縮点火機関用燃料と考えられる。メタノールはセタン価が低いため圧縮点火機関用燃料には適していないが、スパークアシスト法、グローアシスト法等の燃焼技術改良との組合せが検討されている。合成ガスは天然ガスからだけではなく、石炭、重質油、バイオマス、廃棄物のガス化によっても製造できるため、合成系燃料は資源的な制約が少ないことも特徴である。
