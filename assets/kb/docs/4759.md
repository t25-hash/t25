# 2・3・1 燃料噴射機構

> **章**: 第2章 圧縮点火機関
> **節**: 2・3 燃料噴射系統

a. 燃料霧化機構燃料噴射ノズル(fuel injection nozzle)はポンプより送られてくる高圧燃料を霧化状態で機関燃焼室へ噴射するものであり、噴射装置がジャーク式、コモンレール式いずれの場合でも燃焼室との接点としての役割は同様である。一般に直接噴射式機関にはホール型ノズル(図2・15)、副室式エンジンにはピン型ノズルの一種であるスロットルノズル(図2・16)がおもに使われる。ホール型ノズルではおもに噴口数、噴口径、噴射方向の選択により燃焼との適正化が行われる。スロットルノズルではピン形状(ス図2・15 ホール型ノズル図2・16 スロットルノズル(a) サック式ノズル(ミニサック)(b) VCOノズル図2・182スプリング方式ノズルホルダ[(株)デンソー]図2・17ホール型ノズルロットル部のすきま、長さを含め)と噴口形状の組合せで、ノズルニードル弁のリフトに応じ開口面積が変えられ、燃焼に影響を与える。ホール型ノズルはコモンレールインジェクタにも使用され、ノズル先端部の形状には噴口直前にサックと呼ぶ燃料だまり室をもつものと、シート直下が直接噴口となるVCO(valve coveredorifice)ノズルとがある(図2・17)。噴口部に燃料だまり室のないVCOノズルは噴霧の後だれや燃料の吸出しによるスモークやHCの低減に効果がある。ノズルホルダ(nozzle holder)はノズルの保持と位置決めを行うとともに、噴射圧力や噴射期間などを左右するノズル開弁圧および弁揚程を制御する。現在は排ガスと騒音への対応から、二段の弁揚程と開弁圧をもつ2スプリングの方式が自動車用ノズルホルダの主流となっている(図2・18)。b. 燃料加圧機構燃料噴射装置は、機関に最適な燃料噴霧を供給するためのものであり、図2・19に示すようにプランジャで燃料を圧送するジャーク式と蓄圧式とに分類される。ジャーク式における列型噴射ポンプはポンプ本体、ガバナ、フィードポンプ、タイマにより構成されており、用途に応じて種々の形式のものが生産されている。分配型噴射ポンプ(図2・20)は、ポンプ本体に上記構成部品が一体となって組込まれており、列型噴射ポンプ(図2・21)に比べて小型・軽量である。カムシャフトレス型噴射ポンプは、ポンプ本体のみで構成されており、それ以外はエンジン側の構成部品となっている。列型、分配型およびカムシャフトレス型噴射ポンプとも、燃料は噴射管を通り、ノズルから噴射される。また、ユニットインジェクタ(unit injector、図2・22)は、噴射管がなく、一般に圧送、調量、噴射時期制御のポンプの機能とノズルの機能を一本化したものである。一方、コモンレール式は燃料を蓄圧室に蓄えておき、各気筒に送る方式であり、噴射圧力制御や噴射率制御が可能図2・19噴射装置の分類図2・20 分配型噴射ポンプ[(株)デンソーV 3]図2・21 列型噴射ポンプ[(株)デンソーNB 型]図2・22 ユニットインジェクタ(Bosch社)な新しい噴射装置として1995年より実用化されている(3・3・4項参照)。c. 燃料高圧噴射管系ジャーク式噴射装置のユニットインジェクタ以外では、ポンプで発生させた高圧エネルギーを噴射ノズルに伝達するのに燃料噴射管(injection pipe)を用いる。噴射管内径が大きすぎると噴射圧力が上がらず、また調量できる最小噴射量が大きくなり、逆に小さすぎると流体抵抗が増して圧力損失が大きくなる。高圧化傾向において噴射管内の圧力変動により気泡が発生し、崩壊することによるキャビテーションエロージョンの問題が注目されている。噴射管内を正圧に保ち、気泡発生を抑えることが対策として有効である。d. 給油系定置機関などの場合は、燃料タンクを高所に配置し重力を利用してポンプへの給油を行う場合もあるが、その他では燃料供給用のポンプを燃料噴射ポンプに取付けたり、または内蔵するもの、あるいは別置きにしているものもある。
