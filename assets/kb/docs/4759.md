# 4・8・2 エラスチックイメージング

> **章**: 第4章 検査・診断機器
> **節**: 4・8 超音波診断装置

超音波組織性状診断（ultrasonic tissue characterization）は超
音波を利用して体内組織の性質を把握（例えば，がんや腫瘍の良
性・悪性の判断）する診断技術である．Bモード診断装置やカラー
ドップラー装置などは，生体の平均音速でパルス飛行時間から距離
を推定し，画像分解能や周波数依存特性から使用する超音波の搬送
周波数を決定しているが，超音波と生体組織の相互作用は非常に複
雑であり，音速の変化，減衰の様子，反射の強度などの音響特性と
生体組織の物理的および化学的変化の間には，いまだに明確な対応
関係が発見されていない．
エラスチックイメージング（elasticity imaging）は，超音波で
生体内部組織の硬さ分布を映像化する手法であり，静的加圧法と動
的加圧法に大別できる．前者は，体表から静圧を加える前後の体内
組織変化をBモード画像やエコー信号の相関処理によって検出し
てひずみ量を測定するが，計測対象部位での正確な圧力の大きさの
計測が困難であることから，組織間での相対的な硬さ分布の評価に
とどまっている．一方，後者は，200Hz程度の低周波振動を体表
から体内に加え，体内を伝ぱするずれ弾性波（横波）の伝ぱ速度が
組織のずれ弾性係数の平方根に比例することを利用して，横波で組
織内部の構造を探る手法である．現在，ずれ弾性波が体内を伝ぱす
る様子を，振動振幅像，振動位相像，伝ぱ速度像，伝ぱ方向像とし
て観察可能であるが，臨床用装置の開発には至っていない．
