# 4・9・1 硬性内視鏡（rigid endoscope）

> **章**: 第4章 検査・診断機器
> **節**: 4・9 内視鏡（endoscope）

硬性内視鏡（硬性鏡）は光学視管とカメラヘッド，および光源な
どからなる．光学視管は円筒形であり，内部に画像伝達系としてリ
レーレンズ系とライトガイドを有する［図4･55(ａ)］．また，カメ
ラヘッドから容易に着脱でき，交換や滅菌・洗浄が可能である．カ
メラヘッドにはピント調節用光学系と，単板式あるいは三板式のカ
ラーCCDカメラユニットが備わっている．光源としてはおもにキ
セノンランプが用いられ，ライトガイドによって外部から光学視管
内に導入され，先端より臓器を照明する．撮影された臓器の映像は
モニタ上に表示される．画像の保存手段としては，近年はVTRや
画像ファイリングシステムによる記録，またはビデオプリンタによ
る印刷が一般的である．
硬性内視鏡は屈曲機構を有しないため，操作が単純であるほか，
保持装置による機械的操作にも適している．また，受像素子である
CCDは比較的大型のものが利用できるため，重量は重くなるもの

の，他の方式に比べて画質に優れる．一方で先端での視野方向を自
由に変えることができないため，側方の観察が必要な場合には斜視
鏡あるいは側視鏡（oblique view endoscope）と呼ばれる斜め方向
の観察が可能な硬性内視鏡が併用されることもある．
硬性内視鏡は本体が屈曲しないため，消化管などの管腔臓器内部
の観察ではなく，体腔あるいは切開した体表からの内部観察に用い
られる．医療現場では硬性内視鏡は目的臓器によって腹腔鏡
（laparoscope)，胸腔鏡（thoracoscope)，関節鏡（arthro-
scope)，尿管鏡（ureteroscope）など，さまざまな別名をもって
呼称されるが，機構的には同一である．腹腔や胸腔の場合はスリー
ブ様のトロッカール（trocar ）にて体表を貫通した後に，スリーブ
内に内視鏡を通過させ目的臓器手前まで到達させる．近年，加速的
に普及している腹腔鏡下手術は，機械的に（つり上げ法）または空
気圧で（気腹法）腹腔内に空間を確保し，複数のトロッカールから
内視鏡や手術器具を挿入して行う手術である．
硬性内視鏡の外径は11mm（腹腔用）から4mm（鼻腔用，図
4･56)，1.9mm（新生児用尿道鏡，カールストルツ社K27301
AA）とさまざまな種類がある．視野は一般的に80°から100°であ
る．また，ズーム機能を備えたものも多い．
