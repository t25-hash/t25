# 2・3・2 噴射制御機構

> **章**: 第2章 圧縮点火機関
> **節**: 2・3 燃料噴射系統

プランジャによる燃料圧送工程が一定の場合でも、ポンプの回転数が変化すると噴射特性が変るため、噴射する燃料の量は必ずしも一定にならない。このため回転領域によっては、図2・23のようにエンジンの負荷トルクのこう配よりも機関の発生トルクのこう配が大きくなることがある。このため図2・24のように機関トルクを制御する調速機が必要である。調速機、噴射ポンプ、機関の制御系は閉ループを構成する。したがって、機関の負荷特性、噴射ポンプの噴射量特性などを考慮して、調速機の仕様を決める必要がある。燃料ポンプが圧送を開始してから、圧力波がノズルに伝わり噴射が開始するまでに噴射遅れ期間があり、一方、機関自体には着火遅れ期間がある。これらの遅れ期間は機関の回転数にそれほど影響されないため、回転数が高くなると着火時期(クランク角度)が遅く4―69図2・23 調速をしないときの機関・負荷トルク特性図2・24 調速時の機関・負荷トルク特性(a) 高・低速調速機(b) オールスピード調速機(c) ハーフオールスピード調速機図2・25 調速機の制御特性なる。このため使用回転域の広い機関では、回転数に応じて最適な着火時期が得られるような噴射時期制御装置が必要になる。また始動性改善、騒音低減のために回転数だけでなく負荷に対しても噴射時期を制御する装置が必要になってきている。a. 噴射量制御機構調速機(speed governor)はアイドリング回転数から最高回転数まで、あらゆる回転数範囲で制御を行うオールスピード調速機、アイドリング回転数制御と最高回転数制御を行い中間域の回転数では制御を行わない高・低速調速機、さらに、中間域の回転数制御を細かく行うハーフオールスピード調速機がある(図2・25)。車両用としてはガソリン機関の運転特性に類似しているハーフオールスピード調速機や高低速調速機が一般的であり、建設機械、発電機などには負荷変動に対して回転変動の少ないオールスピード調速機が使用される。発電用などの定速エンジンでは、次式で定義される瞬時回転数変動率(instantaneous speed change)と整定回転数変動率(perma-nent speed change)δの規格を満足するように調速機の仕様を定める必要がある。さらに外部条件、運転感覚などに合った細かい制御特性を得るため、電子技術を利用した調速機も実用化されており、図2・26に電子制御調速機の原理図を示す。b. 燃料噴射時期制御機構(automatic timing device)機構的には、機械式、電気式などがある。図2・27に機械式タイマの一例を示す。二つのフライウェイトは、回転数によって増減する遠心力により変位する。遠心力とばね力とが釣合ったところで二つの偏心カムを介して機関駆動軸と噴射ポンプ駆動軸との位相差が定まり、噴射時期が決定される。回転数のみでなく、負荷やその他の外部条件に応じた複雑で精度の高い精密な制御特性を得るには電子技術を利用したタイマが用いられている。図2・28に模式図を示す。4―70内燃機関図2・26 電子制御調速機の原理図図2・27 機械式タイマー[(株)デンソー]図2・28 噴射時期電子制御の模式図
