# 4・9・2 軟性内視鏡（flexible endoscope）

> **章**: 第4章 検査・診断機器
> **節**: 4・9 内視鏡（endoscope）

硬い光学視管を有する硬性内視鏡に対して，全体の屈曲が可能な
内視鏡を軟性内視鏡（軟性鏡）と呼ぶ．軟性内視鏡にはファイバス
コープ（fiber scope）と4･9･3項で後述する電子スコープがある
が，文献によっては両者をまとめてファイバスコープと呼んでいる
場合も見受けられる．なお，胃カメラと俗称されることもあるが，
本来の胃カメラは小型の銀塩カメラを胃中に挿入して撮影するもの
であり，現在は用いられていない．
ファイバスコープは，画像伝達系として光学視管の代わりに極細
径のグラスファイバあるいはプラスチックファイバを束ねて用いら
れ，手元のCCDカメラにより画像化される．照明は硬性内視鏡と
同様にライトガイドによって導入される［図4･55(ｂ)］．
ファイバスコープはファイバの径と数により外径と画質が決定さ
れる．一般的なファイバ数は数千から数万であるが，より少数のフ
ァイバを用いることで，外径1mmを切るような細径のスコープ
が可能となる．一方でファイバ数は得られる画像の実質的解像度に
なるため，細径化は画質の低下に直結する．またファイバの網目模
様や色吸収の影響も存在する．さらに外力によるファイバの屈曲破
壊という耐久性の問題が敬遠され，かつてさまざまな臓器の観察に
用いられていたファイバスコープは，電子スコープの画質の急速な
向上に伴って，一般的な検査における役割を電子スコープに譲りつ
つある．
現在のファイバスコープの用途は，電子スコープの挿入が不可能
な細い管腔臓器内部の観察に移っている．気管支や膵管，胆管，血
管内の観察においては主流であり，例えば膵管鏡として外径0.8
mmのファイバスコープが存在する［オリンパス(株) PF type
8P］．電子スコープのチャネル内にファイバスコープを組込んだデ
ュアルスコープを構成し，観察対象付近まで電子スコープで誘導し
た後にファイバスコープで観察するという利用方法も多い．
