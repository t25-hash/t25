# 4・9・3 電子スコープ（video endoscope）

> **章**: 第4章 検査・診断機器
> **節**: 4・9 内視鏡（endoscope）

ビデオスコープ，電子内視鏡あるいは電子スコープと呼ばれてい
る．外形的にはファイバスコープに類似しているが，機構的には全
く異なり，受像素子であるCCDカメラは内視鏡の先端にあり，画
像伝達は電気信号として行われる［図4･55(ｃ)］．内視鏡内には信
号ケーブルとライトガイド，および先端屈曲用のワイヤが備わって
いる．先端部の直径は一般消化器用では10mmであるが，超小型
のモノクロCCDと回転するRGBカラーフィルタを組合わせた
RGB面順次方式による，先端直径4.2mmのものも市販されてい
る［ペンタックス(株) VNL-1330，図4･57］．また，小型化によっ
て発生したスペースを利用して，鉗子（かんし）やカテーテル用の
チャネル（通路）を内蔵することができ（消化管では直径2～4
mm)，電子スコープを用いた組織検査や治療が行えるようになっ
た．カテーテルに関しては5･5･13項に示す．
電子スコープはファイバスコープと同様に内視鏡自体が屈曲し，
体腔に挿入することができるため，消化管や気管支などの診断に幅
広く用いられている．また，先端のみを屈曲させることによって，
管腔臓器の内壁全体を自由に観察することが可能である．その一方
で挿入には高度な技術を要し，患者に苦痛や合併症を与えずに観察
を行うためには熟練が必要である．
電子スコープの視野は鼻腔・腹腔用で80°から120° ，消化管用
で120°から140°である．把持部のコントローラを操作することに
より，先端のCCDカメラ部を上下あるいは左右に屈曲させること
ができる．屈曲角は各方向100°程度あり，硬性内視鏡より広範囲
を観察することができる．先端のCCDカメラのみ屈曲し，途中を
曲げられないようにした硬性タイプの電子スコープも登場し［フジ
ノン(株)EL2-R410］，腹腔鏡として手術に用いられている．
