# 4・9・4 立体内視鏡（stereoscopic endoscope）

> **章**: 第4章 検査・診断機器
> **節**: 4・9 内視鏡（endoscope）

立体内視鏡は術野を立体画像として観察できる硬性内視鏡であ
る．その基本的構造は硬性内視鏡と同一であるが，光学系を2系統
有し，わずかな輻輳（ふくそう）角をもたせることにより両眼立体
視を行うものである．立体内視鏡の方式としては，光学視管と
CCDカメラをともに2系統有する2眼2カメラ式［図4･55(ｄ)］，
1系統の光学視管を共有する1眼2カメラ式などがある．
立体内視鏡画像の表示装置はバーチャルリアリティなどで応用さ
れる立体ディスプレイが用いられる．二つのディスプレイをもつゴ
ーグル型のものと，単一のディスプレイ上で左右画像を時分割方式
で表示し偏光眼鏡で観察するものがある．手術室内で用いられる場
合，複数の観察者が同時に内視鏡像を見る必要があることと，ゴー
グルの装着感や疲労の問題から，時分割方式が主流である．
立体内視鏡は機構上，通常の硬性内視鏡より重量・外寸法が倍増
することになるが，光学系の改良とCCDの小型化により，通常の
硬性腹腔鏡と同じ10mm前後まで細径化されている．
立体内視鏡は1990年代前半に各社から発表されたものの，当時
は本格的な普及には至らなかった．その大きな理由として立体画像
観察時の観察者の疲労が挙げられている．当時の立体ディスプレイ
技術では時分割によるフリッカが避けられないうえ，時分割シャッ
タと偏光眼鏡によって内視鏡像の明度が著しく減少し，暗室内での
観察を余儀なくされた点が原因と考えられる．また，立体内視鏡自
図4･58立体内視鏡システム［(有)新興光器製作所SK-1057-3D］
身の不自然な輻輳角も疲労の原因と考えられる．
近年の立体ディスプレイ技術の進歩と立体内視鏡自身の改良によ
り，明視下でも自然な画像が得られる立体内視鏡が市販されている
［(有)新興光器製作所SK-1057-3D，図4･58］．またインテュイテ
ィブサージカル社の手術ロボット（surgical robot ）da Vinci TM
でも立体内視鏡が用いられている．
立体内視鏡は，通常の内視鏡では得られない両眼視差（binocu-
lar parallax）による三次元情報を得ることができることから，縫
合操作など内視鏡下手術での微細作業に効果が高いとされ，腹腔内
手術や内視鏡下脳外科手術への応用が期待される．ただし，通常の
単眼式の内視鏡でも，運動視差や遮へい効果などの奥行き手がかり
（depth cue）によりある程度の奥行き情報が得られており，両眼視
差の付与だけによって著しく奥行き認知機能が向上するわけではな
い．遠隔手術（tele- surgery）のように特に実体感の乏しくなる手
術への応用や，超音波断層画像など他の術中三次元画像機器との画
像統合に有効であると考えられる．
