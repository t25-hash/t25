# 2・5・2 後処理技術

> **章**: 第2章 圧縮点火機関

a. 酸化触媒粒子状物質(PM : particulate matter)は可溶性有機成分(SOF)、黒煙、サルフェートと結合水の三つの成分からなり、このうち未燃燃料やオイルからなるSOF分および排気中のHC,COが酸化触媒(oxidation catalyst)により浄化される。その際、燃料中の硫黄より生じるSOガスも触媒により酸化され、サルフェートを生成して触媒によるPM 低減効果を阻害する。サルフェートの生成は燃料中の硫黄量はもちろん、触媒の酸化力や排気温度の高さに大きく影響される。そのため、酸化触媒がサルフェートを生成することなくPM 中のSOFを低減する必要がある。担体はガソリン機関で使用されるものと同様なハニカム形状の一体型(モノリス)構造を有するセラミック材であり、表面にウォッシュコートと呼ばれる多孔・高表面積のアルミナ材を担持している。この担体に金属触媒を担持するほか、助触媒として他の成分を担持するケースが多い。ディーゼルエンジンの排気ガス温度は100~500°Cと広範囲に変化し、低温側ではSOF成分はいったん触媒表面に吸着され、排気温度の上昇に伴い触媒が活性化し酸化される。排気温度がさらに上昇すると排気中SOが酸化されサルフェートを生成するため、結果としてPM 低減効果が少なくなる。このような状況からディーゼル用酸化触媒には・高いSOF酸化性能・高温でのサルフェート生成の抑制・硫酸や高沸点HCによる触媒被毒の抑制といった機能が求められる。現在は低温活性に優れガス状HC,CO浄化効果の高い白金(Pt)とサルフェート抑制に優れたパラジウム(Pd)とを基本にした触媒が使用されている。酸化力の異なる酸化触媒と、硫黄含有量の異なる軽油の組合せに4―9 13モードで評価した。ベンゼン、ホルムアルデヒド、アセトアルデヒドの酸化触媒による低減効果を図2・111に示す。触媒のベンゼン低減効果はPt担持量とともに向上することから、ベンゼンの低減に対してはPtが非常に効果的である。ホルムアルデヒドおよびアセトアルデヒドに対して触媒のPt担持量の増加に伴い低減効果は向上することから、アルデヒド類低減にもPtが非常に効果的である。その理由として、酸化力の強い触媒では排ガスが低温の場合でも未規制物質を含むHCの完全酸化が進むが、酸化力の弱い触媒ではHCの酸化が完全に進行しないため、HCの部分酸化物であるアルデヒド類の生成が促進されるものと推察される。以上の結果から、D 13モードで硫黄含有率0.046wt.%の燃料aを使用した場合、酸化触媒によるPM と今回検討したすべての未規制物質の同時低減は困難である。したがっ4―9 4内燃機関て、PM と未規制物質の同時低減のためには燃料の低硫黄化が必要であり、その効果を以下に述べる。(3) 低硫黄燃料によるサルフェートの抑制効果触媒によるサルフェートの生成を抑制するため、燃料中の硫黄含有率を減らした低硫黄燃料を使用してD 低減効果に対する影響はなかった。これに対し、Pt担持量の多い触媒C(Pt:0.2g/L)と触媒D(Pt:2g/L)では、燃料の硫黄含有率の減少とともにサルフェートの生成が減少して、燃料c(=0.011wt.%)では触媒CがPMを低減できるようになり、燃料d(=0.005wt.%)ではPt担持量2g/L(触媒D)を含めたすべての触媒でPM 低減率20%であった。また、本組合せでは、未規制物質の大幅な低減も実現できている。2・2節で述べたように、わが国では、軽油については2007年まで10ppm以下に低硫黄化されることになり、サルフェート生成の問題は解消される。b. DPFディーゼル車の低公害化の一方策として、大幅なPM 低減を可能とするDPF(diesel particulate filter)があり、20年以上前から欧米や日本において開発が実施されてきた。DPFシステムは、PM を高効率で捕集可能な耐熱フィルタとフィルタに捕集されたPM を除去するフィルタ再生装置とからなる。これまで研究、開発されてきたものは、PM の再生装置として、電気ヒータあるいは軽油バーナ等により捕集されたPM を焼却除去し、フィルタを再生するものが主流である。こうしたシステムは電子制御を駆使した複雑で精緻なものになりがちであり、価格(維持費含む)、信頼性、また寸法、重量等において一般への普及が難しいのが現状であった。一方、近年、酸化触媒とDPFとを組合わせ、酸化触媒でNOを酸化し生成するNOにより捕集したPM を酸化除去、フィルタを再生する連続再生式DPFシステムが検討されている。本DPFシステムの概要を図2・113に示す。本システムは、従来のDPFシステムに比べて大幅な簡素化と高信頼性、低価格が可能となるた図2・112 D 13モードにおける酸化触媒のPM 低減効果図2・113 NOを用いた連続再生型DPFめ、有力なPM 浄化システムとして実用化されている。本システムのPM 低減効果について、D 13モード条件での評価結果を図2・114に示す。これまでの燃料では酸化触媒により硫酸分に起因するPM が激増し、PM 排出量が2倍以上となった。一方、硫黄分を40ppmまで低減した燃料を使用した場合、大きなPM 低減効果が得られたが、サルフェートによるPM 増加により、例えばヨーロッパで予定されている大型車の排出ガス規制、EuroIV への適合には十分とはいえない。また、本DPFシステムのNOには、すすの連続再生には約300°C以上の温度が必要であり、市内走行中のディーゼル車の排気温度は200°C以下であるため、連続再生には不十分である。市内だけを中心とした車両への適合には、なんらかの強制再生装置との組合せが必須となる。c.
