# 3・1・3 圧縮機(compressor)

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・7 に示す。記号の最初のグループの数字は圧縮機とタービンと

ガスタービンは多くの空気流量を必要とする速度型内燃機関なので、その圧縮機には空気流量を大きくとれる軸流式や遠心式など速度型圧縮機が適しており、両者はガスタービンの用途および構造に応じてそれぞれの特徴を生かし、あるいは結合させて用いられている。a. 遠心圧縮機(centrifugal compressor)遠心圧縮機は軸流式と比較して、構造が簡単かつ頑丈で高速に適しており、1段当たりの圧力比が高くとれる(図3・10)。作動範囲もかなり広く安定しており、汚れに対する性能低下も少ない。異物の侵入に対しても割合丈夫であり、また製作費も比較的安いなどの長所がある。短所としては、前面面積当たりの空気流量が多くないので大容量のものにはあまり適さないこと、効率が軸流式よりも数%低いことなどである。このような長所短所を検討した結果として、遠心圧縮機は比較的小型の陸・舶用および航空用ガスタービンに広く用いられている。通常のガスタービン用圧縮機は圧力比4程度以上であり、段数は1または2段である。圧力比6程度までは単段が普通であり、ポリトロープ効率85%程度である。単段で圧力比10を超えるくらいになると、通常は効率が急激に低下し、また作動範囲が狭くなる。高い圧力比に対しては、2段にして1段当たりの負荷を下げることもある。同一の羽根車周速に対しては、径向き羽根車(radialvane impeller)が強度上有利で負荷も大きくとれ圧力比が高いの図3・10 圧縮機1段当たりの圧力比1:1段後向き羽根車2:1段ディフューザ3:曲り流路4:2段後向き羽根車5:2段ディフューザ図3・11 2段遠心圧縮機で、軽量小型を重視する航空用などに適している。後向き羽根車(backward vane impeller)は羽根の後向きの角度に応じて負荷が減少し、圧力比も少し低下するが、効率や作動範囲は径向き羽根車の場合より良いので、陸・舶用など効率重視の場合に有利である。陸・舶用は重量よりも効率が重視され、作動範囲の広いことも要求されるので、羽根車周速は600m/s程度までとし、後向き羽根車を用いることが多い。非常用・防災用などに設置する発電用ガスタービンの圧縮機は圧力比8前後が多い。単段にすると構造が簡単で軽量となるが、高速高負荷となるので所要の性能および作動範囲を得るためにはかなりの技術的困難を伴う。2段にすると構造は複雑となるが、羽根車の周速および負荷を無理のない程度まで下げることができ、性能および構造・強度の面で余裕のある設計製作ができる。図3・11は後向き羽根を有する2段圧縮機(圧力比8)の例である。航空用では、初期には遠心圧縮機がエンジン全般に用いられたが、エンジンの大容量化ならびに軸流圧縮機の性能改善が進むとともに、中型・大型エンジンでは軸流式が主流となり、遠心式は小出力のターボプロップ、ターボシャフトエンジンに用いられている。b. 軸流圧縮機(axial compressor)軸流圧縮機は遠心式と比較して、図3・10に示したように1段当たりの圧力比が小さいので、所要の圧力比を得るには多段にしなければならない。このため構造が複雑で翼枚数が多く、翼の製作に高い工作精度を必要とするので製作費が高い。また、異物が侵入したとき翼が破損しやすい。長時間運転すると気流中のほこりが翼面に付着して性能が低下するので、特に陸・舶用の場合は空気ろ過器を設置したほうがよい。このようなさまざまな難点をもちながらも空気流量当たりの重量および容積が小さく、大きな空気流量を処理できること、多段にして高圧力比とした場合も高い効率が得られることなどの特徴のために、大型のガスタービンには広く用いられている。航空用では航空機に搭載する立場から、前面面積当たりの空気流量が大きいこと、飛行中のラム圧を効率良く利用できることなどの利点が効果的であり、多段軸流圧縮機が圧倒的に主流を占めている。比出力および熱効率はサイクルの最高最低温度比が高くなるほど4―101向上し、それに応じて最適圧力比も高くなるので、圧縮機も高圧力比化への傾向をたどっている。多段軸流圧縮機の作動範囲は、低速では流量の小さいときに初段の正失速、流量の大きいときは終段の負失速で制限され、高速では流量の小さいとき終段の正失速、流量の大きいときには終段のチョーキングによって制限される。このような作動範囲への制限は圧力比が高いほど厳しくなり、作動範囲の狭小化とともに部分負荷特性の低下、加速特性の悪化を招くので、高圧力比の場合は対策が必要である。1軸で得られる圧力比は5程度までであり、それ以上の圧力比を得るためには、以下のような手法が必要である。(1) 抽気(bleeding)主として低速時に中間段のケーシング壁から空気を抽出し、前段を通る空気流量を増して、低速における前段の正失速を抑制する。構造は簡単であるが、抽出する空気に加える仕事量だけタービンの仕事量を増すため、タービン入口温度が上昇する。(2) 可変静翼(variable stator vane)入口案内翼およびそれに続く前段の静翼取付け角を可変とする。低速時に翼を寝かせて動翼への迎え角を減少させ、作動点での流量減少に合わせるとともに前段翼列の失速を抑制する方法で、作動範囲の拡大と効率および起動性の改善が期待できる。取付け角度の変化量は回転数に対して一定の関係で制御する。(3) 多軸圧縮機(multi shaft compressor)1軸の圧力比を対策なしでも十分な作動範囲を有する圧力比4程度までに制限し、全圧力比に応じて低圧系・高圧系の2軸あるいは3軸に分割する。陸・舶用では低圧圧縮機と高圧圧縮機の間で中間冷却したり、高圧タービンと低圧タービンの間で再熱したりする自由度が増すので都合がよい。複合発電システムの高効率レヒートガスタービン用の軸流圧縮機では、低圧圧縮機(圧力比、約5)・中間冷却器・高圧圧縮機(圧力比、約12)の配置で全圧力比は55~60にも達する。航空用では軽量で前面面積当たり空気流量の大きいことが要求され、入口ボス比0.35~0.5程度、入口の軸流速度150~210m/s程度になっている。圧縮機段数を減らすため1段当たりの仕事量が大きく、平均1段当たり温度上昇が40°Cを超すものもある。このため動翼先端の周速は350~430m/sにもなり、初段動翼の相対流入マッハ数は高く遷音速領域にある。したがって、効率は陸・舶用よりもいくぶん低い傾向にあるが、遷音速翼形の開発および翼端すきまの縮小など効率改善への努力が払われて、ポリトロープ効率90%程度が得られている。ターボファンエンジン用の圧縮機では、高バイパス比(5程度以上)の場合は図3・12(a)に示すように2軸または3軸で、最前列にアスペクト比の大きい(3以上)ファン(圧力比1.4~1.7)が配置され、低バイパス比(1前後)の場合は2軸でファンは2段以上(圧力比2前後)である。用途別では、高バイパス比が亜音速飛行用、低バイパス比は主として超音速飛行用である。陸・舶用は重量や容積の制限が厳しくない代わりに経済性が重視され、作動範囲が広く安定かつ高効率を保持することに重点がおかれている。したがって、その圧縮機は航空用に比べて周速、軸流速度がともに低めの傾向で、入口ボス比0.6以上のものが多い。ポリトロープ効率は91%を超す程度である。図3・12(b)に示すように、航空用ターボファンエンジンを陸・舶用に応用した航空転用型のガスタービンも多数製造されている。これは信頼性の高い高性能のターボファンエンジンのファンを取り去り、ファン駆動に消費していた動力を出力として利用するもので、量産化された高性能圧縮機がほとんどそのまま応用されている。c.
