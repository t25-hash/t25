# 3・1・4 タービン(turbine)

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・7 に示す.記号の最初のグループの数字は圧縮機とタービンと

タービンは,流体のもっているポテンシャルエネルギーを速度エネルギーに変換し,その角運動量変化に相当する回転機械仕事を取出すものである.内部流体が回転軸に沿って流れるものを軸流タービン(axial flow turbine),半径方向に流れるものをラジアルタービン(radial flow turbine)という.図3・13に軸流タービンの例を示す.一般に航空用ガスタービンや産業用ガスタービンのように大出力,高効率のものは軸流タービンが用いられる.タービンは圧縮機や発電機などと組合わせて用いるので,被駆動体の回転数や出力特性などとのマッチングを考慮した設計を行うことが大切である.すでに述べたように,サイクル効率と比出力を増大させるために,圧力比を高めるとともにタービン入口温度(turbine inlet tem-perature)を高めることが必要となる.タービン入口ガス温度は,タービン冷却技術の進歩によって,タービン部材温度より400°C以上も高くすることができるようになっている.特に航空用ガスタービンは,離着陸時の短時間ではあるが,タービン入口平均ガス温度が1600°Cを超えるものもある.また,発電用ガスタービンは長時間,定負荷運転でかつ高い信頼性をもつように設計されるため,タービン入口平均ガス温度は航空用エンジンより低いが,それでも1500°C程度に達するものも現れてきた.タービンは,圧縮機で圧縮した空気を燃焼器に導き,燃料を燃した高温高圧ガスを作動流体としているのが一般である.このため,タービンは圧縮機や燃焼器の影響を強く受けている.燃焼器ライナの中で燃料と一部の圧縮空気を用いて燃焼させ,質量流量は大きいが,体積流量の少ない圧縮空気で希釈混合してタービンに供給されている.この希釈混合長が短いため,半径方向,円周方向に温度むらが見られるのが普通である.このため航空用エンジンでは,タービン入口の局所最高ガス温度は平均ガス温度より300°C以上も高いと考えられるものがある.a. タービンの作動条件と材料燃焼器の直後にあるタービンの部材は,ガスタービンの中で最高のガス温度にさらされるとともに,高強度特性をもつことが要求されている.一般に初段静翼(ノズル翼,nozzle blade)はケーシングから両端支持が可能なため応力値は低いが,燃焼器による温度むらのためタービンの入口平均ガス温度より高い局所最高ガス温度に耐えることが必要である.このためノズル翼は耐食耐酸化性に優れたCo基合金が用いられている.一方,動翼(rotor blade)は高速で回転しているので流入速度は低くなり,動翼にとっての全温は静止系のノズル翼に比して下がる.さらに,円周方向の温度むらも平均化されるので,動翼の温度環境は楽になるが,極めて高い遠心力が働くため高い応力に耐える必要がある.また,2段ノズル翼は1段タービンで仕事が吸収されているのでガス温度は下がるが,作動ガスが体積膨張するので翼高さが増すだけでなく,構造上ケーシング外径からの片持ち支持となるため,高い曲げ応力に耐えることが必要になる.さらに2段動翼は1段動翼と同様の原因でいっそうガス温度は下がるが,空力性能向上のため一般にシュラウドが取付けられるので応力は高くなる.このような状況から1段動翼,2段ノズル翼,2段動翼は高温クリープ強度の高いNi基合金が用いられている.Co基合金およびNi基合金の冷却タービン翼は,鋳造技術の向上により,精密真空鋳造技術とセラミック中子技術によって中空翼化が行われている.この鋳造は従来,冷却が急速に行われていたため多結晶化していたが,最近は一方向結晶制御(uni-directionalcrystallization)を行うことで柱状晶化や単結晶化(single crys-tallization)を実現し,材料特性の向上を得ている.タービン動翼を保持するタービンディスクは高速で回転しているので,高い遠心力と熱応力のため低サイクル疲労に強い鍛造材が用いられてきたが,最近はより優れた粉末冶金(powder metal-lurgy)製のものが利用されている.b. 冷却タービン翼(cooled turbine blade)タービン翼を冷却(cooling)すれば,部材の温度より高いタービン入口ガス温度を採用することができ,ガスタービンの性能が向上する.このため高温強度部材である1段および2段のノズル翼,動翼,さらにタービンケーシングやディスクを圧縮機の出口空気などで冷却している.翼の冷却法として,翼内部に流路を設けて冷却空気を流す対流冷却(convection cooling),冷却空気を高温物体に吹き付けるインピンジメント冷却(impingement cooling)が広く利用され,また高温ガスから熱の流入を防止するために,タービン部材の表面に冷却空気の薄いフィルムを吹き出すフィルム冷却(film cooling)も使用されている.インピンジメント冷却を含む対流冷却は,熱流束の増大をもたらすために熱応力が生じる.フィルム冷却は熱流束の減少をもたらすので材料強度上好ましいが,主流ガスとの混合などのため圧力損失等が生じやすい.近年はまた,冷却空気量を増やさずに熱流束を減少させるため,高温タービン部材の表面に熱伝導率の低いジルコニアなどの材料を,プラズマ溶射などにより被覆する遮熱コーティング(TBC: thermal barrier coating)の技術が進歩し,積極的に利用されている.図3・14に冷却タービン翼の構造例を示す.第1段タービンノズル翼には翼前縁にシャワーヘッド型フィルム冷却孔を設け,翼の背側と腹側にもフィルム冷却孔を設けている.さらに翼後縁からも吹出しを行っている.翼内部にはインサートを入れてインピンジメント冷却と対流冷却を行っている.タービン動翼では,翼中央部は翼(a)
