# 4・3・2 クライオスタット

> **章**: 第4章 極低温機器
> **節**: 4・3 極低温応用機器

クライオスタット（cryostat ）とは，超伝導コイルなどの極低温
機器を収納する容器のことをいう．クライオスタットの設計に当た
っては，その用途や温度レベルによってさまざまな設計手法が取ら
れるが，いずれの場合も，低温を安定に長期にわたって維持できる
ように，高度な断熱技術が駆使されている．さらに，最近では医療
用MRI （magnetic resonance imaging）のように，研究者のみで
はなく一般のユーザにまで極低温機器が利用されるようになり，使
いやすさもクライオスタットの設計に対して重要な要素となってき
ている．ここでは，小型極低温冷凍機を利用した超伝導磁石用クラ
イオスタットについて述べる．
図4･29(ａ)は，GM冷凍機（GM refrigerator ）を用いてふく射
シールドを冷却するクライオスタットの構成を示す．超伝導コイル
は液体ヘリウム容器（4.2K）の中に浸漬され，GM冷凍機は真空
容器と液体ヘリウム容器の間に設置された2枚のサーマルシールド
を約80Kおよび約20Kに冷却している．このような構成により，
室温からヘリウム容器内にふく射および熱伝導によって流入する熱
は，80Kレベルと20Kレベルでそれぞれ一，二段ふく射シールド
を介してGM冷凍機に取り去られる．そのため，ヘリウム容器へ
の入熱は20Kから4.2Kへの熱侵入のみとなる．一般に普及して
いる超伝導MRIはこの構成のものが主流であり，液体ヘリウムの
蒸発量も数十cc /h程度となっている．
図4･29(ｂ)は4K- GM冷凍機のみによって超伝導コイルを冷却
し，液体ヘリウムを必要としないクライオスタットの構成を示す．
超伝導コイルは4K-GM冷凍機の二段冷却ステージ（4Kステー
ジ）と伝熱板によってつながれ，4Kステージからの熱伝導によっ
て冷却される．一段冷却ステージは，4Kレベルへの熱侵入を低減
させるために，ふく射シールドを約60Kに冷却している．4K-
GM冷凍機によって，冷凍機にスイッチを入れるだけでクライオス
タットを運転することが可能となった．停電や冷凍機の故障時にク
ライオスタットが使用不可能になる欠点はあるが，液体ヘリウム容
器がないために小型化が可能となり，ヘリウムの注液作業がいっさ
い不要という簡便さもあって，液体ヘリウムの入手が困難な場所や
極低温の知識のないユーザにも使用が容易となった．さらに，形状
や使い方の自由度が増すというメリットによって，オープン型
MRIや角度を自由に変えて利用可能なクライオスタットが実用化
されている．
図4･29の構成以外にも，液体ヘリウムで超伝導コイルを冷却し
ながら4K-GM冷凍機やその他の4K冷凍機によって蒸発したヘ
リウムを再凝縮するクライオスタットも利用されている．始めに液
体ヘリウムを注液さえすれば，その後は液体ヘリウムを補給する必
要がない．停電などで冷凍機が停止した場合も液体ヘリウムがある
ため，ある程度の時間は超伝導コイルの運転を続けられるというメ
リットも有する．
