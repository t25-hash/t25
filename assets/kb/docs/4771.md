# 3・1・5 燃焼器(combustor)

a. 要求性能ガスタービン燃焼器に要求される性能としては、高い燃焼効率(combustion efficiency)、広い火炎安定範囲、確実な点火、圧力変動や燃焼に起因する不具合のないこと、小さい圧力損失、タービン翼の長寿命化に適した出口ガス温度分布、スモーク・未燃焼成分・NO等の大気汚染物質の排出が少ないこと、低コストで維持費の少ないこと、エンジンに適した大きさと形状、耐久性に優れることが挙げられる。航空用ガスタービンでは加えて、高空再着火性が必須である。また、常用・非常用兼用の産業用ガスタービンでは、気体燃料を常用するが非常時には液体燃料に切換えるなど、多種燃料に対する適応性の高いことが求められることもある。b. ディフューザ燃焼器での圧力損失を許容範囲に抑えるためには、圧縮機からの空気流を低損失で減速しておくことが必要である。ディフューザは、この作用に加えライナに対してスムーズで安定な空気流れを与え、空気量配分を決めるという役目ももつ。従来からの動圧の回復を重視した空気力学的ディフューザに代わっ4―104内燃機関て、出口に急拡大部を設けたダンプディフューザが用いられる傾向にある。後者は、圧力損失はやや大きいが、圧縮機出口の速度プロファイルの影響を受けにくく、エンジンの軽量化にもつながる点が評価されている。この急拡大部において抽気を行い、渦により流れをいっそう安定化する方式が、作動温度の高温化によりタービンの所要冷却空気量が増大する状況にあって広まっている。d. 燃焼室ディフューザで減速された空気は燃焼器ライナの壁面にあけられた開口から燃焼室内に流入する。燃焼室の上流部の空間、燃焼器ライナドームにはスワーラと呼ばれる保炎器を通って燃焼用空気が流入する。スワーラは燃焼用空気に旋回を与え、その旋回は再循環流を形成し、保炎を確実にする。燃焼器ライナの上流部壁面にあけられた穴から流入する空気噴流も、再循環領域の形成を助ける。この再循環領域を含む部分は一次領域と呼ばれる。燃料は燃料ノズルから放射状に噴射され、スワーラからの空気と混合し、再循環領域の高温既燃ガスとの混合により点火され、燃焼する。一次領域での燃焼ガス中の未燃焼成分は、その下流において流入する空気と混合して燃焼する一方、熱解離により存在していたCOなどは再結合が進む。この領域を二次領域と呼ぶ。もちろん、作動条件によっては一次領域の延長として機能することになる。二次領域の下流で導入される希釈空気は、単に平均的な温度を下げるだけでなく、タービンノズル入口において適切な温度分布を形成する役割をもっている。そのため、希釈空気孔の形状や大きさ、配置、タービンノズルにつながる流路形状などは適切な温度パターンが得られるように設計される。燃焼ガス温度の半径方向分布は、翼根側で低く、翼端に近い位置で最高になるのが最適とされる。一方、周方向には一様なことが、タービン翼に周期的な熱負荷を与えないことから望ましい。e. 燃料噴射航空用ガスタービンには各種のジェット燃料が使用される。産業用ガスタービンには天然ガス、灯油、A 重油のほか、高炉ガスや汚泥分解処理などで発生する消化ガスなどの利用も進んでいる。これらの気体燃料の単位体積当たりの発熱量は幅が非常に広い。燃料噴射には、気体燃料の場合は単純な孔を複数個配列した、いわゆる多孔ノズルが用いられる。液体燃料の場合は、かつては燃焼室内に延伸した管内に燃料を吐出させて壁面からの熱により蒸発させ、過濃混合気として燃焼室内に噴出させる蒸発管方式が用いられたことがあるが、現在では下記の微粒化ノズル方式により噴霧として供給する方式が用いられている。. 圧力スワールノズル[pressure swirl nozzle、図3・17(a)]液体燃料の微粒化手段として最も広く使用されている。燃料は渦室に流入することで旋回が与えられ、噴射圧がほぼ0.4MPaを超えるとオリフィスから中空円すい状の液膜となって流出する。この液膜はオリフィスから離れるに従って薄くなり、分裂して液糸となり、最後に微細な液滴となる。これが噴霧である。基本は渦室、オリフィスがそれぞれ1個のシンプレックスノズルである。燃料流量のターンダウン比を大きくするために大小2個の渦室とオリフィスを同軸に配置したのがデュプレックス(デュアルオリフィス)ノズルである。また、渦室の背面の中心に燃料を逃がす通路を設けることで、この課題を解決したのがスピル-リターンノズルである。逃がし量を増やせば、噴射流量が小さい場合でも十分強い旋回が渦室に形成され、微粒化の悪化を防止できる。. エアブラストノズル[air-blast nozzle、図3・17(b)]ジェットエンジンの作動圧力の上昇が始まったころから離陸時のスモークが顕著になった。その対策として生まれたのがこの方式の燃料ノズルである。その後、排出規制の対象になったHCやNOの抑制にも有効なことから現在、大型航空用ガスタービンでは主流になっている。毎秒百メートル前後の流速で燃焼室内に流入する燃焼用空気が微粒化に利用される。燃料をフィルマと呼ばれる薄肉円筒の内壁面上で膜状に形成する方式や、微細孔から気流中に噴射する方式が用いられる。液膜方式の場合、その両面を旋回気流に接触させることが良好な微粒化に有効である。. エアアシストノズル(air-assist nozzle)圧力スワールノズルの微粒化を空気あるいは蒸気により促進するもので、外部の空気源、蒸気源を使用するもの、始動時において圧縮機最終段から抽気した空気を補助圧縮機で加圧して使用するものなどがある。エアブラストに比べ微粒化用空気量は少ないが、燃料噴射圧が低いときの微粒化改善効果が大きい。. 遠心微粒化器エンジンの主軸あるいはそれに付けた円盤に半径方向にあけた微細孔から、燃料を遠心力により噴出させることによって微粒化を行わせる方式である。使用は、小型エンジンの逆流アニュラ燃焼器に限定される。高圧燃料ポンプを必要としないので構造が簡単で安価である。f. 点火装置混合気の点火には、高エネルギー型点火装置が用いられる。この装置に使用される点火栓は表面放電型と呼ばれ、中心電極と外周電極とはセラミックス製円筒で絶縁されており、その円筒の先端には両電極をつなぐ半導体の薄膜が形成されている。この半導体は電極間のイオン化を容易にする働きがあり、放電はこの面に沿って起きる。小型ガスタービン用には蓄積エネルギー2J、繰返し放電数毎秒4回、大型エンジン用には4~12J で毎秒1回が一般的である。点火栓位置は確実な点火と点火栓の寿命とを考慮して選定される。通常は、一次領域において燃料噴霧の外縁で、高温のスパーク核が再循環により上流に運ばれる、600°C以上にならない場所が選ばれる。火花放電では点火の難しい低質油燃料の場合には、グロープラグあるいは軽質燃料やガスによる火炎でのトーチ点火が採用される。
