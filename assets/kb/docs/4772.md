# 3・1・6 熱交換器

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・7 に示す.記号の最初のグループの数字は圧縮機とタービンと

a. 概要ガスタービン用熱交換器は,サイクル効率向上にとり不可欠であり,大型発電用から自動車用小型ガスタービンまで広範囲に利用されている.熱交換器の基本構造自体には大きな進歩は見られないが,特に環境負荷軽減の社会的要請,小型ガスタービン(自動車用,マイクロガスタービン,舶用)の実用化研究や超小型ガスタービン関係のプロジェクトの推進により,技術的に大きな進歩を遂げている.以下では,ガスタービンでの熱交換器の応用例として,サイクル効率上重要でかつ高い設計・加工技術を必要とする再生器について説明する.b. 熱交換器に求められる性能再生器に求められる性能は,(1)高温度効率,(2)低圧力損失など通常の熱交換器に共通な要件に加えて,高圧作動流体を用いることが多いことから,(3)低リーク量が求められる.加えて,ガスタービンでは起動・停止が頻繁に起きることから,(4)耐久性が必要となる.一般に熱交換器は,複雑に板金加工された薄い構造体を何層にも接合した構造で,熱応力に弱いセラミック材などを多用することから,そこで発生する熱応力に十分配慮した設計が肝要である.材料開発,有限要素解析ツールによる熱応力解析に加え,熱流体解析ツールおよび格子生成ツールの進歩,各ツール間の連携の強化,実験的手法の進展などにより,耐久性の向上が図られている.c. 熱交換器の分類ガスタービン用の再生器は,伝熱式熱交ビンおよびジェットエンジン章ガスター第34―1図3・19セラミック直管を用いた熱交換器図3・20 積層型熱交換器換器(recuperator)と蓄熱式熱交換器(regenerator)に大別することができる.. 伝熱式熱交換器隔壁式熱交換器とも呼ばれ,高温ガスと低温空気とが伝熱壁を介して熱交換を行う形式のものであり,実用上最も多く用いられている.伝熱壁の形状で,多管式,板式などに分類できる.多管式の場合,金属製のU 字細管が多く用いられるが,複雑形状の加工が困難なセラミック管を用いる場合は,タービンの出口直後に直管のセラミック管を用いる形態の熱交換器が提案されており(図3・19),温度効率86%,圧損2.9%が達成されている.板状の場合には,熱通過量を増加させるため,伝熱面積の拡大と熱伝達率の増加を目的とした伝熱面の複雑化が一般的であり,フィン,波状壁(corrugated wall)などの加工が行われる.フィンとしては,高熱伝達率が得られるオフセットフィンが多用される.オフセットフィンは,フィン高さ,フィン幅,オフセット長さなどをパラメータとできることから,所定の圧力損失に合わせたパラメータ群の組合せを選ぶことのできる自由度が特徴である.例えば,高温ガス側は密度が小さく流速が大きくなることからフィン高さを大きく,低速流となる低温空気側にはフィン高さを小さくするなどの設計上の工夫が可能である.図3・20は積層型の伝熱式熱交換器の一例であり,空気側に高さ2.2mmのオフセットフィンを採用することで,積層高さ3m以内で温度効率85%以上を達成している.このほかにもコルゲートフィンをコアに採用したプレートフィン型熱交換器の開発例もある.. 蓄熱式熱交換器おもに自動車用ガスタービン向けに開発されたもので,回転するディスク状の蓄熱体に高温ガスと低温空気が交互に通過することで,熱の交換を行う仕組みになっているものである(図3・21).低容積で比較的高い温度効率を得られるが,高温ガスと低温空気とのシールが極めて重要である.蓄熱体としてセラミックハニカムコアを用いた場合,1000°Cを超える高温ガスの利用が可能である.車両用ガスタービンの需要が乏しいこと4―106内燃機関図3・21 回転蓄熱式熱交換器を装着したガスタービンの例図3・22 MIT 型熱交換器表3・1 ガスタービンの制御要求・タービン入口温度制限・燃焼器圧力制限・実/修正回転速度制限・圧縮機サージ制限・安定燃焼制限エンジン保護エンジン安定性・出力変動幅(例:±1%以下)・圧縮機サージ余裕(例:0.15以下)・出力調整・公称出力/燃料消費率の保証・再現性・微調整機能・エンジン性能劣化,搭載法,入口条件変化に対する制御感度定常性能過渡性能・出力応答・燃焼安定性・加減速(例:アイドル→最大出力=5秒以下)効率向上・ティップクリアランス制御起動停止・起動制御・停止制御もあり,蓄熱式熱交換器の応用例は多くはない.d. 超小型ガスタービンMIT の提案による超小型ガスタービンでの熱交換器を図3・22に示す.解析によれば,ボタンサイズのMIT 型の場合の温度効率は0.5程度で,流路パターンの複雑化,約2倍のサイズ拡大により温度効率0.8の達成が可能であるとされている.
