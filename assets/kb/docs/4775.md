# 4・1・4 水処理法

> **章**: 第4章 水処理技術
> **節**: 4・1 概説

水の処理法は，水道用水（浄水）・工業用水等用水の処理法と，
下水・工場排水等排水の処理法とに大別され，初期においては沈
殿・ろ過といった共通の単位操作が利用されてきたが，利水目的の
理施設の整備が推進されている．下水道および浄化槽によるトイレ
の水洗化人口の推移を図4･5 に示す．また，平成15（2003）年
度末には，生活排水対策の基本となる下水道処理人口普及率は
66.7%に達している ．
今後は，流域別下水道整備総合計画や都道府県構想（全県域汚水
適正処理構想など）に基づく下水道・合併浄化槽等の整備，単独浄
化槽の合併浄化槽への移行，浄化槽汚泥対策の推進とし尿処理施設
（汚泥再生処理センター）の整備，合流式下水道の雨天時水質改善，
窒素・リンを高度に除去できる高度処理の普及など，生活系汚濁負
荷削減対策の推進が求められている．
多様化や汚濁の複雑化とともに特有の用水・排水処理技術が開発さ
れ，利用されている（表4･12）．
