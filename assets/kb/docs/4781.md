# 3・3・1 ターボジェットおよびターボファンエンジン

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・3 航空用ガスタービンおよびジェットエンジン

a. 種類と特徴ターボジェット(turbojet)およびターボファンエンジン(turbofan engine)は、ジェットノズルから噴出するジェットのエネルギーをスラスト(推力)として航空機の推進用に使用する形式のエンジンであり、固定翼航空機の推進用に使用される。ターボジェットエンジンは圧縮機、燃焼器、タービン、空気取入れ口、ジェットノズルという基本構成要素とそれら全体をシステムとして作動、機能させるための燃料制御、潤滑、内部空気・冷却空気、電気、油圧系のシステムあるいはコンポーネント(補機類を含む)により成り立っている。この中心部となる圧縮機、燃焼器、および圧縮機を駆動するタービンから構成される部分を、通常コアエンジンというが、このコアエンジンの前後にファンおよびファン駆動タービン(低圧系と呼ぶ)を追加した形態のエンジンを、ターボファンエンジンという。ターボジェットおよびターボファンエンジンは以上のような構成要素の形式や組合せによって種類を細分できるが、ここでは形態上最も代表的な特徴といえる圧縮機の形式と軸構造の面から分類してみる。. 圧縮機の形式による分類高効率が追求でき、また前面面積も抑えることができることから、航空エンジンでは現在軸流圧縮機が主流となっている。最新のもので段当たりの圧力比は1.3~1.5程度であり、単軸で圧力比5程度のものから2軸以上のターボファンエンジンでは全体圧力比が40を超えるものまでその範囲は広い。一方、遠心圧縮機は効率面で見劣りするものの単段で圧力比が3~10程度と高く取れ、エンジンの構造を簡単にできることから小型エンジンで使用されている。特に小型エンジンの場合、軸流圧縮機では翼高さが小さくなりすぎる後段側を遠心型に置換え、軸流型と遠心型を組合わせる形で遠心圧縮機の特徴を生かしているエンジンが見られる。なお、遠心圧縮機を使用した場合、燃焼器も逆流型を採用するケースがよく見られる。. 圧縮機とタービンを組合わせた軸の構成による分類(1) 単軸形式最も基本的な形式であり簡潔な構造にできるが、単軸では圧力比が高く取れないため、最近では低価格の特殊用途のターボジェットに限定されている。なお、単軸でもなるべく圧力比を上げようとすると、始動時の失速余裕の不足などに対応するために可変静翼(variable stator vane)や始動時抽気が必要となる。(2) 多軸形式圧縮機を低圧系・高圧系(2軸形式)あるいは低・中・高圧系(3軸形式)に分割して、それぞれ低・高圧タービンまたは低・中・高圧タービンによって独立に駆動する形式である。中空の軸の中に他の軸系を通すので軸系の支持構造は複雑になるが、回転数の自由度が増し、少ない段数で高い圧力比を取ることができるので、かえってエンジンを短くまとめることができる。燃料経済性のうえから全体圧力比を上げることが望ましいので最近はほとんどのエンジンが多軸式となっている。なお、多軸式でも個々の圧縮機の圧力比が高くなると可変静翼や抽気が必要となる。(3) ターボファンエンジン多軸形態の一つであるが、圧縮機、燃焼器およびタービンから構成されるコアエンジンの前に大量の空気を吸い込むファンを取付けたエンジンである。ファンで圧縮された空気はコアエンジン側に入り、燃焼に関与する空気とバイパス流れとしてそのまま排気ノズルから排出される空気に分かれ、このバイパス空気流量とコアエンジンに入る空気流量の比をバイパス比(bypass ratio)という。同じスラストを出力するにあたり大量の空気を吸い込んで(すなわちバイパス比を大きくして)バイパス流を低排気速度で噴出した場合、推進効率が良くなり燃料経済性に優れたエンジンとなりうる。エンジンの基本設計の際には、搭載される航空機のミッションに合わせて、バイパス比を全体圧力比やタービン入口温度等とともに最適値に設定する。中・長距離飛行に供される中・大型の旅客機や輸送機では、燃料経済性が重視されるため高バイパス比のエンジンが使用される。ゼネラルエレクトリック社のCF6,GE90、プラット& ホイットニー社のPW 2037,PW 4000、ロールスロイス社のRB 211,TRENT,CFMI社のCFM 56,IAE社のV 2500などがその代表で、バイパス比は4.5~9レベルに設定されている。バイパス比をこれ以上大きくしようとするとファンの外径が大きくなり、それと同時に周速が稼げない低圧圧縮機や低圧タービンの段数がどんどん増えて結果的にエンジンが重くなり、航空機全体の経済性は頭打ちか悪化の方向となる。これをブレークスルーするためには超軽量材の採用や減速ギヤ(ファンと低圧タービンの間に設置)の採用が必要となる。なお、高バイパス比ターボファンエンジンは上述のようにジェットノズルからの排気速度が低いため、低騒音化という点でも優れている。高空を高亜音速で飛行するビジネス機の用途には、中位のバイパス比(2.5~4)のハネウェル社のTFE731、ロールスロイス社のBR 710などが搭載されている。前述のようにこのビジネス機用の小さいエンジンでは遠心圧縮機が使用されるケースがあり、また軽量化のためにファンと低圧タービンをつなぐ軸に減速ギアを配置して(低圧タービンの段数を減らして)いるエンジンもある。比較的短距離の区間を運航する中・小型旅客機では、1980年代ごろまではロールスロイス社のSpey,Tayやプラット& ホイットニー社のJT 8D 等のバイパス比の小さな(1~3レベルの)エンジンが使われていたが、騒音規制が厳しくなったことおよび燃料経済性も求められる中で、現在は小型旅客機でもゼネラルエレクトリック社のCF34、ロールスロイス社のAE3007等の高いバイパス比のエンジンが使用されるようになっている。なお、小型でも練習機の場合は低バイパス比(1~2)のエンジンが多く使用されている。4―119超音速域で高いスラストが必要な超音速機には、低バイパス比(1前後)のターボファンにアフタバーナ(afterburner)を付けたエンジンが使用される。ゼネラルエレクトリック社のF404,F414,F101,F110、プラット& ホイットニー社のF100、ロールスロイス社のAdour、ターボユニオン社のRB 199、ユーロジェット社のEJ 200などが代表である。なお、超音速旅客機であるコンコルドには2軸ターボジェットにアフタバーナを付けたOlympus593エンジンが搭載されたが、次世代の超音速旅客機では騒音規制を満足させるために低バイパス比ターボファン形態(アフタバーナなし)を中心に研究開発が進められている。日本ではターボジェットのネ-20(1945年)、J 3(1959年)に引き続き高バイパス比のFJR 710(1983年)、低バイパス比のF3(1985年)および超音速機用の実証エンジンであるHYPR エンジンが開発されている。さらに民間機用エンジンの国際共同開発では高バイパス比のV 2500(1988年)、CF34-8C(1999年)等の開発に参画している。表3・5に代表的なターボジェットとターボファンエンジンのおもな項目を示す。. V/STOL機用エンジンV/STOL機(vertical/shorttake offand landing aircraft、垂直、短距離離着陸機)には各種のものが研究されているが、大別すると1離着陸と巡航にプロペラを用いるもの、2離着陸と巡航に同一のジェットエンジンを用いるもの、3離着陸と巡航を別個の機構で行うものに分けられ、それに応じてエンジンの種類、特徴が異なる。1のプロペラ機では左右主翼端に取付けたエンジンを転向させるティルトロータ機(例:アメリカV 22Osprey)が実用化されつつある。2および3ではターボファンエンジンが中心となるが、2では強力なターボファンエンジンの排気ジェット(コアおよびファン排気)を離陸時に下方に向けて上昇力を得る(ロールスロイス社のペガサスエンジンがその代表)か、あるいはファンからの排気の代わりに圧縮機から高圧の空気を抽気して下方に噴出す形式のものがある[アメリカJSF(Joint Strike Fighter)機X 32用エンジン].一方、3の場合、巡航用のエンジンとは別のリフトエンジンで離着陸する方法と、巡航用エンジンのパワーを圧縮機からの抽気かあるいは軸駆動馬力の形で抽出して、このパワーによってリフトファン(lift fan)を駆動することにより揚力を得る形式などが考えられる。アメリカJSF機X 35では上記リフトファンによる揚力と巡航エンジンの排気を下方に向けることを組合わせ、垂直離着陸を実現している。b. 性能。 翼部圧縮機の翼部の形状は、空力的な要求と振動に起因する高サイクル疲労破損防止の点から決まるので、翼の固有振動数を予測し、エンジン運用範囲に有害な共振点がないように設計する。高負荷対応の三次元薄肉翼化により、高次の固有振動数の共振を避けるのが困難な場合は、設計段階から空力的な励振力や減衰力を推定し、共振時の振動レベルを予測する必要がある。また、ファン動翼やファン出口案内翼には、空気取入口からの異物吸込みによる損傷(FOD : foreign object damage)に対する要求がある。FAR 33.76では、エンジン空気取入口面積に応じて1.85~3.65kg の鳥を吸い込んでも、終局荷重条件を超えるマウント荷重が発生するなどのエンジンに致命的な破損が生じないこと、同様に0.35~1.15kg の鳥を空気取入口面積によって決まる数だけ吸い込んでも、75%以上の推力を維持することといった要求がある。数値シミュレーションにより損傷状態を予測して翼形状を設計し、要素試験やエンジン試験で実際に鳥を打ち込んで健全性を確認する。タービン翼では、防振設計に加えて、高温環境での使用に対応した冷却設計や、遠心力によるクリープ強度、遠心力や圧力に加えて、熱応力の繰返しによる低サイクル疲労強度を満足するような設計が必要となる。. ロータ部翼を支持しているのがディスクである。ディスクは破損時の飛散エネルギーが大きいため、このエネルギーをケースで吸収して貫通させないようにしようとすると重量が過大となってしまう。そこで、万一破損した場合にエンジン外部へ飛散するおそれのあるディスクやシャフトなどのロータに対しては、安全性の点から、他の部品に比べて厳しい設計ならびに製造上の要求が課せられている。ディスクは、起動停止の繰返しによる低サイクル疲労寿命が、エンジン要求寿命を満足するように設計される。起動停止のサイクルには遠心力や圧力荷重とともに、特にタービン部では熱応力の繰返しが重要である。このため、ディスクの定常、非定常の温度分布を精度良く予測する必要がある。寿命の算出には、材料の疲労試験データを用いた安全寿命設計(safe life design)が行われる。すなわち、ロータごとに使用制限寿命を定め、寿命に至る前に交換するように運用管理されている。さらに、材料や製造時の欠陥を想定して、損傷許容設計の点からも寿命の検討がなされる。またディスクは、制御システムの故障や、シャフトの損傷などにより過回転状態になっても破損しないように設計され、試験で確認することが規格でも求められている(FAR 33.27)。. level)dB で規定されている。例えば機体質量が5700kg 以上のジェット機およびプロペラ機に対しては、従来Chapter3[FAA (Federal Aviation Agency)のStage3と同等]が適用されてきたが、2006年以降の新規登録機に対しては、この3点の累計値が10dB 強化されたChapter4が適用されている(図3・70)。このICAO騒音基準は航空機の型式承認に関する規定であるが、これとは別に夜間の飛行制限や、騒音レベルに応じた着陸料徴収など、独自の運航規制を設ける空港が増えてきており、さらに厳しい低騒音化が要求されている。ジェットエンジンの騒音源は排気ジェットとファンによるものが通常支配的であるが、タービン、低圧圧縮機、燃焼器からの騒音も問題になる場合がある。ジェット騒音(jet noise)は乱流混合により発生し、排気速度の7~8乗に比例する広帯域騒音であるが、アフタバーナ作動時などの超音速ジェットでは衝撃波に起因するスクリーチと呼ばれるトーンノイズが卓越してくる。一方、ファン騒音(fan noise)は動翼と静翼や入口乱れと動翼などの空力的な干渉により発生し、翼通過周波数(動翼枚数×回転数)およびその高調波からなる。さらに離陸上昇時には、動翼の衝撃波に起因するバズソーと呼ばれる騒音がこれに加わる。また翼面境界層などによる広帯域音にも注意する必要がある。ターボファンエンジンの高バイパス比化は騒音低減に大きく寄与したが、さらなるジェット騒音低減には、ミキサノズルによる混合促進やロングカウルが有効である。ミキサの一種でシェブロンノズルと呼ばれるギザギザ状のノズルは重量や性能への影響も少ない。ファン騒音は動静翼間隔の拡大と動静翼枚数比の最適化により低減される。タービンの騒音対策も基本的に同様であるが、翼枚数を増やして可聴域外に追い出す方法が取られることもある。これらエンジン内部で発生した騒音をさらに確実に低減するため、ダクト内壁などに吸音材(noise absorbing material)が張られる。吸音材はハニカムに穴あき表面板を張った形態が用いられ、複合材による軽量化や空力損失の少ない表面板が工夫されている。これら騒音低減対策の一覧を図3・71に示す。.
