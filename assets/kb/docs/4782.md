# 4・3・2 下水処理技術（sewage treatment process )

> **章**: 第4章 水処理技術
> **節**: 4・3 処理技術の事例

a.概要 下水処理は，水質汚濁の防止から閉鎖性水域の富栄
養化防止，近年では処理水の有効利用など社会的要請の変化に伴
い，有機物除去から高度処理へと処理の高度化が進んでいる．ま
た，大都市での用地不足や中小市町村での技術者不足など地域特性
を踏まえた各種処理方式が開発されており，時代とともに下水処理
技術はますます多様化してきている．ここでは，標準活性汚泥法と
オキシデーションディッチ法に代表される有機物除去を目的とした
処理技術と，栄養塩類除去を目的とした高度処理技術についてその
概要を述べる．
b.施設の構成と処理機能
.標準活性汚泥法（conventional activated sludge proc-
ess ) 下水処理では，微生物の有機物分解作用を利用した活性汚
泥法が主流である．標準活性汚泥法の水処理施設は，最初沈殿池，
反応タンク，最終沈殿池，消毒設備で構成されている．最初沈殿池
において沈殿処理された下水は，反応タンク（HRT＝6～8h）に
おいてエアレーションにより活性汚泥と混合され，下水中の有機物
は生物学的に処理される．その後，最終沈殿池において活性汚泥混
合液は固液分離され，上澄み水は塩素等で消毒して放流される．標
準活性汚泥法は中大都市を中心に広く採用されており，日本の下水
処理の主流処理方式として位置付けられている．
.オキシデーションディッチ法（oxidation ditch process )
オキシデーションディッチ法は，オランダで考案されヨーロッパに
おいて古くから普及してきた処理方式である．日本国内では，24
時間の滞留時間をもつ反応タンク（無終端水路）と最終沈殿池で構
成された施設として，日本下水道事業団において水量ごとの施設設
計がすでに標準化されている．オキシデーションディッチ法は敷地
面積を広く要するが，維持管理が極めて容易であることから国内の
小規模下水の主流処理技術として普及しており，2002年度現在で
すでに700箇所以上の稼働実績がある．
.高度処理 1972年の瀬戸内海での赤潮の大量発生など
閉鎖性水域での富栄養化が社会的問題となり，下水中の窒素，リン
の除去を目的とした高度処理の開発が進められた．窒素除去法とし
ては生物学的窒素除去法が一般的であり，循環式硝化脱窒法や多段
ステップ流入法が開発され実用化されている．また，リン除去法と
しては生物学的方法として嫌気好気法が，物理化学的方法として凝
集剤添加法が採用されている．さらには，窒素・リンの同時除去法

の下水処理場を中心に普及してきている．近年では下水中の難分解
性有機物除去を目的としたオゾン・生物活性炭処理法も開発され，
