# 3・4・1 種類と特徴

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・4 車両用ガスタービン

車両用ガスタービンには、鉄道車両用あるいは自動車用などのガスタービンがある。これら移動用に対するガスタービン採用の利点は、小型軽量で出力が比較的大きくとれることや、振動、騒音、排出ガスなどの低公害性に優れていることが挙げられる。一方で、ディーゼル車に比べて燃費(燃料消費率)が良くないなどの問題もある。a. 鉄道車両用ガスタービン鉄道車両用ガスタービンは、非電化区間の高速化のために開発が行われた。重量当たりの出力がディーゼル車に対して大きくとれるので、車両の軽量化によって車両速度を向上させるとともに、曲線区間の遠心力の増加を抑制することをねらいとした。一方で、車両の軽量化は動輪のスリップを起こしやすくするなどの問題がある。近年では軽量でしかも振動がなく低騒音の利点を生かしたものとして、磁気浮上式鉄道(リニアモータカー)の車両に搭載する予備電源用発電装置として開発された例がある。b. 自動車用ガスタービン自動車用ガスタービンは、欧米では1940年代から、日本では1960年代から開発が始められた。開発例では大型トラック、高速バス、レーシングカー、乗用車などガスタービンの重量当たりの高出力の利点を生かす用途が多い。1970年代初め、環境問題がクローズアップされ、排ガス性能の良いガスタービン乗用車などの開発が進められた。しかし、オイルショック以降、低燃費化の要求から高効率ガスタービンの開発が重要(b) 2軸式(a) 1軸式図3・90ガスタービンの形式となり、従来の金属製ガスタービンでは燃費向上に限界があることから、高温化により熱効率の向上を図ったセラミックガスタービンの開発が欧米、日本で行われた。また、金属製ガスタービンに高速発電機を直結したタービン発電機と駆動用モータとを組合わせ、ハイブリッド化することにより燃費向上を図るハイブリッド自動車の開発も行われている。. 形式自動車用ガスタービンでは、熱効率の向上を図るために熱交換器で排気熱を回収する再生式が採用されている。形式は図3・90に示すように、1軸式および2軸式で大別される。1軸式は圧縮機とタービンを結合する軸が同時に出力軸になる形式で、構成が簡単である。車両を駆動する動力を取出す場合には無段変速機(CVT : continuously variable transmission)を介して取出し、電気を取出す場合には出力軸に高速発電機を直結して取出す。これに対して、2軸式は圧縮機駆動タービンと車両を駆動する出力タービンがそれぞれ独立に存在し、2本の軸を有する形式である。この場合、二つのタービンの間のガスの流れが無段変速機の機能を有するのでCVT を必要としない。従来の自動車用ガスタービンでは2軸式が多く採用されてきたが、今後、ガスタービンを自動車に搭載する場合には、燃費向上の観点からハイブリッド化が不可欠であり、1軸式に高速発電機を直結する形式が採用される傾向にある。. 基本構成基本構成は、図3・90に示すように圧縮機、タービン、燃焼器、熱交換器からなる。1軸式を例示すると圧縮機はサイクル上すべて遠心式であり、タービンは1段で膨張比が大きく取れるラジアル式のほか軸流式も使用される。熱交換器は、従来伝熱効率の良い回転蓄熱式(rotating regenerator)が多く使用されてきたが、シール部の漏れや耐久性などの問題が十分解決されておらず伝熱式も使用されている。燃焼器は低NO燃焼方式が採用されている。すなわち、燃焼器内でなるべく均一な混合気を形成して燃焼させる方法、あるいは均一な予混合気を燃焼器に供給して予混合燃焼させる方法である。. 特徴ガスタービンの利点としては、1構造が簡単で部品点数が少なく保守点検が容易である、2冷却水が不要である、3回転機械であるため振動がほとんどない、4連続燃焼であるため爆発音がなく音は高周波で消音しやすい、5比較的低圧の連続燃焼であるため排気の低公害化に対応しやすく幅広い燃料が使えるなどが挙げられる。一方、欠点はディーゼルエンジンに比べて熱効率が低いことである。この欠点を克服するために、タービン入口温度を高温化し熱効率の向上を図ったものとして後述のセラミックガスタービンの開発がある。さらに、燃費を向上させるとともに、ガスタービンの低公害性の利点を生かしたものとして、ガスタービンハイブリッドバスがある。
