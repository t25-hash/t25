# 3・4・1 種類と特徴

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・4 車両用ガスタービン

車両用ガスタービンには，鉄道車両用あるいは自動車用などのガ
スタービンがある．これら移動用に対するガスタービン採用の利点
は，小型軽量で出力が比較的大きくとれることや，振動，騒音，排
出ガスなどの低公害性に優れていることが挙げられる．一方で，デ
ィーゼル車に比べて燃費（燃料消費率）が良くないなどの問題もあ
る．
a.鉄道車両用ガスタービン 鉄道車両用ガスタービンは，非
電化区間の高速化のために開発が行われた．重量当たりの出力
がディーゼル車に対して大きくとれるので，車両の軽量化によって
車両速度を向上させるとともに，曲線区間の遠心力の増加を抑制す
ることをねらいとした．一方で，車両の軽量化は動輪のスリップを
起こしやすくするなどの問題がある．近年では軽量でしかも振動が
なく低騒音の利点を生かしたものとして，磁気浮上式鉄道（リニア
モータカー）の車両に搭載する予備電源用発電装置として開発され
た例がある．
b.自動車用ガスタービン 自動車用ガスタービンは，欧米で
は1940年代から，日本では1960年代から開発が始められた．
開発例では大型トラック，高速バス，レーシングカー，乗用車など
ガスタービンの重量当たりの高出力の利点を生かす用途が多い．
1970年代初め，環境問題がクローズアップされ，排ガス性能の良
いガスタービン乗用車などの開発が進められた．しかし，オイルシ
ョック以降，低燃費化の要求から高効率ガスタービンの開発が重要
となり，従来の金属製ガスタービンでは燃費向上に限界があること
から，高温化により熱効率の向上を図ったセラミックガスタービン
の開発が欧米，日本で行われた．また，金属製ガスタービンに
高速発電機を直結したタービン発電機と駆動用モータとを組合わ
せ，ハイブリッド化することにより燃費向上を図るハイブリッド自
動車の開発も行われている．
.形式 自動車用ガスタービンでは，熱効率の向上を図る
ために熱交換器で排気熱を回収する再生式が採用されている．形式
は図3･90に示すように，1軸式および2軸式で大別される．1軸式
は圧縮機とタービンを結合する軸が同時に出力軸になる形式で，構
成が簡単である．車両を駆動する動力を取出す場合には無段変速機
（CVT:continuously variable transmission）を介して取出し，
電気を取出す場合には出力軸に高速発電機を直結して取出す．これ
に対して，2軸式は圧縮機駆動タービンと車両を駆動する出力ター
ビンがそれぞれ独立に存在し，2本の軸を有する形式である．この
場合，二つのタービンの間のガスの流れが無段変速機の機能を有す
るのでCVTを必要としない．従来の自動車用ガスタービンでは2
軸式が多く採用されてきたが，今後，ガスタービンを自動車に
搭載する場合には，燃費向上の観点からハイブリッド化が不可欠で
あり，1軸式に高速発電機を直結する形式が採用される傾向にあ
る．
.基本構成 基本構成は，図3･90に示すように圧縮機，
タービン，燃焼器，熱交換器からなる．1軸式を例示すると圧縮機
はサイクル上すべて遠心式であり，タービンは1段で膨張比が大き
く取れるラジアル式のほか軸流式も使用される．熱交換器は，従来
伝熱効率の良い回転蓄熱式（rotating regenerator）が多く使用さ
れてきたが，シール部の漏れや耐久性などの問題が十分解決されて
おらず伝熱式も使用されている．燃焼器は低NO燃焼方式が採用
されている．すなわち，燃焼器内でなるべく均一な混合気を形成し
て燃焼させる方法，あるいは均一な予混合気を燃焼器に供給して予
混合燃焼させる方法である．
.特徴 ガスタービンの利点としては，①構造が簡単で部
品点数が少なく保守点検が容易である，②冷却水が不要である，③
回転機械であるため振動がほとんどない，④連続燃焼であるため爆
発音がなく音は高周波で消音しやすい，⑤比較的低圧の連続燃焼で
あるため排気の低公害化に対応しやすく幅広い燃料が使えるなどが
挙げられる．
一方，欠点はディーゼルエンジンに比べて熱効率が低いことであ
る．この欠点を克服するために，タービン入口温度を高温化し熱効
率の向上を図ったものとして後述のセラミックガスタービンの開発
がある．さらに，燃費を向上させるとともに，ガスタービンの低公
害性の利点を生かしたものとして，ガスタービンハイブリッドバス
がある．
