# 3・4・2 自動車用セラミックガスタービン

> **章**: 第3章 ガスタービンおよびジェットエンジン
> **節**: 3・4 車両用ガスタービン

ガスタービンの熱効率は、基本的には図3・91の再生サイクルの熱効率に示すように、タービン入口温度を上げることにより向上させることができる。セラミックガスタービン(ceramic gasturbine)は高温部分にセラミックスを用い、タービン入口温度を高温化することにより熱効率を向上させるものである。25~300kW の金属製ガスタービンの定格時の熱効率は25~30%であるが、ディーゼルエンジンと同等レベルの熱効率40%を得るためには、タービン入口温度(TIT : turbine inlet temperature)は1350°C、圧力比は5程度である。自動車用として開発された100kW セラミックガスタービンは図減速させCVT を介して駆動力を取出す構造である。定格時の設計条件は出力が100kW,TIT が1350°C、圧力比が5、タービン回転数が10万minである。このときの各ステージにおける状態量(計算値)は図3・93のようになる。出力と熱効率の特性を図3・94に示す。ここで、TIT はタービンの回転数が5万minのとき、1214°Cである以外は1350°Cである。最良燃費はタービン回転数9万minの運転条件のときに得られる設計である。図3・91 再生サイクルの熱効率図3・921軸式セラミックガスタービンの構造図3・95主要セラミック部品4―131セラミックガスタービンでは、セラミック部品と金属部品を組付ける必要があるが、セラミックスはボルトによる締結が困難である。そこで、静止部品の組付けに際しては、セラミック部品の平面部分を接触面として部品を積み重ねていく構造とし、全体をばねと空気圧を利用して金属ベローズによって弾性的に強く押し付けて支持する構造が採用されている。タービンロータや高温静止部品などのセラミック部品は、燃焼器ライナが炭化ケイ素、熱交換器コアがMAS(コージェライト)で造られている以外は、ほとんどが窒化ケイ素で造られている。主要セラミック部品を図3・95に示す。熱交換器には回転蓄熱式が用いられている。ハニカム状の熱交換図3・93定格時における各ステージの状態量図3・94出力および熱効率の特性(計算値)4―132内燃機関図3・96 ロータの外観器コアが回転数20minでゆっくりと回転しながら排気熱側で受熱し、空気側で放熱することにより排気熱を回収する形式である。燃焼器には予蒸発予混合希薄燃焼方式と拡散燃焼方式が併用される。始動時には拡散燃焼させ、常用運転時には予蒸発予混合燃焼させる。急加速時には予蒸発予混合燃焼している中に加速燃料を噴射させ燃焼させる方式である。開発結果は、目標出力100kW に対して92.3kW、目標熱効率40%に対して35.6%が得られている。熱効率の向上策として、ハウジングからの熱損失の低減やハウジングの変形に起因する空気漏れ量の低減などの課題がある。
