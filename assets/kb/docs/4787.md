# 4・1・1 ロケットの推進原理

> **章**: 第4章 ロケット機関
> **節**: 4・1 基本性能

ロケット機関の基本原理は,ロシアの数学教師ツィオルコフスキー(1875~1935年)によって,ライト兄弟が初飛行に成功した1903年に提唱された.ツィオルコフスキーは,論文「ロケットによる宇宙空間の探索」を発表し,推進原理の解明を行うとともに多段式の液体水素ロケットを提唱した.その発案どおり,多段式で上段用に液体水素を用いたサターンV型ロケットがアメリカで開発され,1969年には人類初の月面着陸という20世紀最大の快挙がもたらされた.a. ロケット機関の特徴一般に地表から高度100km以上を宇宙空間と呼ぶが,地球の引力(重力)に対抗して長時間にわたって宇宙空間に滞在するには,周回飛行による遠心力を利用する以外に有効な方策はなく,宇宙機(人工衛星,宇宙船,資材など)を第1宇宙速度(秒速7.9km)付近まで加速して,空気抗力が極めて小さい高真空の宇宙空間を高速飛行させる必要がある.打上げ用ロケットの役割は,宇宙機を軌道上に投入することであるから,その大きな特徴は超高速の加速機械という点にある.空気中を飛ぶ飛行機は,音の壁や熱の壁を克服する必要があり,実用化されている最も速い軍用のジェット機でも最高速度は時速3000km程度である.ロケットには,その約10倍(時速2万8000km以上)もの速度が要求されることから,空気が希薄となる高度まで上昇し,真空中で本格的な超高速加速を行う.したがって,ロケット機関には,真空中推進という際立った特徴もある.真空中推進では,物質(質量)を後方に高速で噴出し,その運動量を利用して,ロケットを前方に加速する.また,真空中では,大気に含まれる酸素は利用できないので,化学ロケットでは燃料に加えて酸化剤を機体に積込む必要があり,このタンクも装備する.ロケットは,大気層を通過する際に大きな空気抗力を受けることから,空気による揚力の積極的な利用は難しく,翼をもたない方式を主流として発達してきた.このため,発射重量を上回る強力な推力を発生するロケットエンジンを装備して,地上から垂直姿勢で上昇することになり,飛行初期の加速を担うロケットにおいては,その特徴に大推力機関が加わる.b. ロケット機関の種類ロケット機関は,搭載した推進薬(推進剤ともいい,推薬と略称される)を熱源や電磁場等を用いて効率よく高速まで加速することにより,推進薬を運動量として後方に噴出して,推進力を発生する装置である.ロケット機関は熱源の種類等によって分類されるが,代表的なものとして,化学ロケット,電気ロケット,原子力ロケットがある.化学ロケット(chemical rocket)は,推進薬(燃料や酸化剤)の化学反応をロケット機関の熱源として利用するもので,燃焼や分解の発生熱で加熱し,ノズルで膨張させた生成ガスを噴出して推進する.化学ロケットの特徴は,大きな推力が得られることであり,地上からの打上げに用いる大推力のロケット機関は,現状のところ化学ロケットに分類される液体ロケットと固体ロケットに限られる.液体ロケットの大きな特徴は,タンクの大きさとは無関係にエンジンの定格推力を選択でき,小推力エンジンと大型タンクの組合せが可能なことから,比較的に穏やかな加速度で軌道速度まで増速できる点にある.加速度に制限のある有人飛行や柔構造の大型人工衛星の打上げでは,最後まで加速を担うロケット本体(コア機体)には,液体ロケットが採用される.さらに,液体ロケットには,エンジンの始動・停止が自在なこと,推力の制御が容易なこと,推進薬が単位重量流量当たり発生する推力(比推力)が大きいことなどの利点がある.一方,固体ロケットには,短秒時の作動ながら大きな推力を容易に発生できること,構造が簡単なことなどの利点があり,重量が重く大きな推力を必要とする発射初期の推力増強用や小4―133型ロケット用として利用される.電気ロケット(electrical rocket)は,電力を用いて物理学的な作用で推進薬を加速する推進機関であり,電熱器等を熱源として加熱しノズルで膨張させた推進薬を噴出して推進する方式や,電磁場を利用してイオン化した推進薬を加速して推進する方式がある.電気ロケットの特徴は,比推力の高さにあるが,現状での推力レベルは小さく,宇宙機の姿勢制御用や軌道制御用として,イオンスラスタが実用化されている程度である.原子力ロケット(nuclear rocket)は,原子炉を熱源として推進薬を物理的に加熱しノズルで膨張させた推進薬を噴出して推力を得る推進機関である.比推力が高いことや長時間の作動に適していることから,将来の惑星間飛行用として有望視されているが,現在は研究段階にある.c.
