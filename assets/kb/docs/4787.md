# 4・1・2 水圧機器

> **章**: 第4章 水圧システム
> **節**: 4・1 水圧システムの特徴

水圧システムは，添加剤をいっさい用いない清水あるいは水道水
を作動流体とした駆動システムである．水圧システムに使用される
水圧機器（water hydraulic component ）は，主として従来の油圧
機器に相当する機能を実現する機器として開発されている．しか
し，水を作動流体とすることから，水の物理的あるいは化学的な特
性によって，油圧機器とは異なる観点からの設計が必要とされる．
例えば，水の粘性は油圧作動油より低いため，しゅう動部や漏れ流
量の条件がより厳しい．また，水に対する耐食性のある材料を選択
しなければならない．最新の材料技術やコンピュータによる解析・
制御技術などを駆使して，さまざまな水圧機器が開発されている．
a.ポンプ 遠心ポンプは低圧大流量に適したポンプとして，
およそ2MPa程度の圧力範囲で広く使用されている．効率は，お
おむね40～50%とそれほど高くない．また，ベーンポンプ（vane
pump）も低圧用に用いられており，効率は50～60%である．従来
から用いられてきたこれらのポンプは，おもに低圧領域における水
圧システム用のポンプとしての発展が期待されている．
より高圧の領域における水圧システム用には，ピストンポンプ
（piston pump）が適している．アキシアルピストンポンプ，ラジ
アルピストンポンプ，レシプロケーティングポンプに大きく分類さ
れるが，アキシアルピストンポンプとレシプロケーティングポンプ
が大半を占めている．レシプロケーティングポンプは100MPa程
度の超高圧を発生させることができ，ウォータジェットなどに従来
から利用されている．しかし，クランク室とピストン室の間を遮断
するためのシールの寿命や吐出し圧力の脈動に注意が必要である．
中高圧領域の水圧システム用のポンプとしては，アキシアルピス
トンポンプが代表的である．定格圧力は14MPa前後のものが典型
的である．ピストンなどのしゅう動部には，プラスチックとステン
レスの組合せが使用されている．また，しゅう動部の面圧を低下す
ることを目的として，機構的な工夫を施したポンプも開発されてい
る．効率面では，油圧ピストンポンプと同程度か若干下回る．
b.制御弁（control valve) 絞り弁やボール弁は，従来から
他の分野で使用されていたものを転用できる可能性がある．チェッ
ク弁（逆止め弁）も同様である．
方向切換弁は，比較的低い圧力範囲であればさまざまな種類のも
のが入手できる．中高圧の範囲では，小流量のものに限られるよう
である．高圧力では，ボール弁を電動モータや空気圧によって開閉
する弁がある．高速に大流量を切換える弁は水圧駆動回路を構成す
るうえで重要な制御弁であるが，ポペット弁やボール弁のタイプの
ものが開発されている．
リリーフ弁は，オーバライド特性に優れたバランスピストンタイ
プのものが，水圧システム用に開発されている．キャビテーション
やエロージョンへの対策として，材料の選択，流路形状の工夫など
がある．
比例制御弁やサーボ弁では，先進技術を駆使した開発が行われて
いる．スプールを静圧軸受で支持し，しゅう動特性を改善したサー
ボ弁がある．また，ボールねじ駆動形やムービングコイル形などの
サーボ弁が開発されている．スプールには，ステンレスやセラミッ
クスが利用可能である．スプールの変位制御系をマイナループとし
て内蔵し，油圧と同程度の流量特性と周波数特性が得られ，高速高
精度な制御を実現する回路に用いられる．
c .アクチュエータ（actuator ) シリンダは，ステンレスを材
質とするさまざまな仕様のものが利用できる．ロッド部やピストン
部でのシールやウェアリングの選択は重要で，耐久性や摩擦力，負
荷能力を考慮して選択する．また，ストロークエンドでクッション
効果を得るために特別な設計をすることもある．
水圧システム用のモータとして利用できるものは限られている
が，ピストンモータやベーンモータが開発されている．これらの水
圧システム用モータでは起動トルクが大きく，最低回転数が比較的
高いものが多く，位置や速度などを高精度に制御する場合には注意
が必要である．また，低速度で駆動するために水潤滑ギヤボックス
を取付ける場合がある．
d.補器類 アキュムレータは，ステンレス仕様のものが従来
から利用可能である．フィルタやクーラも同様である．タンクは，
ステンレス製のものやプラスチック製などが利用できる．タンクの
材質は遮光性に優れたものが水管理上有利である．配管は，高圧の
部分にはステンレス鋼管や高圧用の水用ホースが利用できる．低圧
部には，ふつうのビニルホースなどを利用することもある．管継手
は，高圧用のステンレス製継手がある．低圧では空気圧用の継手類
が転用できる．
