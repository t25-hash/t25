# 4・1・2 理想ロケット機関

> **章**: 第4章 ロケット機関
> **節**: 4・1 基本性能

打上げ用に利用されている化学ロケットを例にして、ロケット機関の推進原理について述べる。実用のロケット機関では、高温高圧の燃焼ガスをノズルで膨張させて推進力を得るが、実際の燃焼ガスは膨張の際に、状態方程式が完全ガスとは異なること、状態の変化に応じてガス組成が変化すること、壁面には冷却に伴う熱境界層や粘性に伴う速度境界層があり、これらは流れに沿って発達すること、流れには三次元分布や衝撃波の発生があることなど、多様で難解な現象を伴う。ここでは、ロケット機関の基本的な特性を理解するために、流れの単純化を図り、理想気体の準一次元断熱膨張流を想定する。a.
