# 4・1・2 理想ロケット機関

> **章**: 第4章 ロケット機関
> **節**: 4・1 基本性能

打上げ用に利用されている化学ロケットを例にして、ロケット機関の推進原理について述べる。実用のロケット機関では、高温高圧の燃焼ガスをノズルで膨張させて推進力を得るが、実際の燃焼ガスは膨張の際に、状態方程式が完全ガスとは異なること、状態の変化に応じてガス組成が変化すること、壁面には冷却に伴う熱境界層や粘性に伴う速度境界層があり、これらは流れに沿って発達すること、流れには三次元分布や衝撃波の発生があることなど、多様で難解な現象を伴う。ここでは、ロケット機関の基本的な特性を理解するために、流れの単純化を図り、理想気体の準一次元断熱膨張流を想定する。a. 推力(thrust)ロケットは後方に質量を放出したとき、前方に加速されることから、前進力(推力)を受けたことになる。燃焼圧力や流量を増加すると、大型の液体ロケットには必須となるターボポンプの駆動軸出力が増大し、特に流体密度の低い液体水素エンジンでは、高出力のターボ機械が必要となる。発射初期の推力を担うロケット機関では、推力は特に重要な特性である。発射推力が大きければ、発射重量の重い大型ロケットの打上げが可能となり、軌道投入用ロケットにおいて最も重要な性能である打上げ能力を向上することができる。b. 比推力(specific impulse)ロケットの基本性能である増速量は、式(4・4)に示したように噴出速度[正確には式(4・14)に示す有効排気速度]に比例して向上する。c. 特性排気速度(characteristic exhaust velocity)圧縮性流体では、流速を音速との比であるマッハ数で整理すると、流れは無次元化され、相似的に取扱うことができる。ロケット機関では、スロート部での音速(ここでの流速は断熱膨張流では音速に等しい)が速いことは、噴出速度が速いことを意味する。音速は流れの静温を計測すれば求められるが、これは難しいので、質量流量と燃焼室圧力を計測してこれらから音速を求める。ここで、準一次元断熱膨張流を仮定すれば、音速と質量流量には式(4・10)の関係があることから、燃焼室内の音速を計算で求めることができる。この場合、計測が容易な圧力、流量、断面積から求めた式(4・15)で定義される量を選べば、音速に比例(γ=1.4では1.23倍)する量が得られ、これを特性排気速度と呼ぶ。したがって、優れた推進薬の条件は、燃焼温度が高いこと、燃焼ガスの分子量が小さいことの2点に集約される。実用の液体水素エンジンでは、燃焼温度をやや犠牲にして分子量の小さい未燃水素を適切に含ませる混合比(酸化剤と燃料の質量流量の割合)を採用し、大きな特性排気速度を得ている。d. 推力係数(thrust coefficient)ロケット機関のスロート部を音速で通過した燃焼ガスは、ノズルで超音速膨張を続け、速度を増加してから噴出される。この増加の度合いを推力係数と呼び、式(4・13)~(4・15)から、次式で定義される。大きな開口比のノズルを装着したエンジンは、高い膨張圧力比が得られ、推力係数に比例する大きな噴出速度を得ることができる。この場合、周囲圧力比が1を超えると流れは過膨張となり、実際のエンジンにおいては、4・1・5項に示すように2~3を超える領域で流れのはく離を考慮する必要がある。
