# 4・1・3 ノズル形式

> **章**: 第4章 ロケット機関
> **節**: 4・1 基本性能

ノズル性能は、式(4・17)で定義した推力係数で評価する。準一次元流れでは、ノズル出口圧力は、推進薬と燃焼室圧力を一定とすればノズル開口比εのみで決まり、εが大きいほどは低下する。周囲圧力は、ロケットの上昇に伴って低下することから、ノズルのをつねに最大に保つためには、飛行高度に合わせてεを増加させる必要がある。εが一定の通常のノズルでは、εが大きいほど高高度では性能は上昇するが、低高度では過膨張となり性能は逆に低下する。このため、ロケットの飛行経路を考慮して飛行中の平均比推力が最大となるようにεの値を選ぶ必要がある。
