# 4・1・1 設計項目

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

低真空（10～10Pa）では，真空容器内の気体の排気が主であり，中真空（10～10Pa）および高真空領域［10～10（あるいは10）Pa］では，真空容器内壁面に吸着しているガス分子の放出が支配的である．
特に，HO分子の吸着脱離挙動を考慮した排気が必要になる．
超高真空（10～10 Pa）から極高真空（10 Pa以下）では，拡散や透過によるガス放出も考慮しなければならない．
以下，真空容器の漏れはないものとし，それぞれの排気過程について記す．

真空容器にはつねに，大気圧1atm（10.13N/ cm）の外力が作用している．
また，フランジ部の変形は漏れ（リーク）に結びつく場合が多いので，大気圧による変形と強度を考慮した，圧力容器としての設計が前提となる．
使用条件として決めておくべき点は，(ａ)到達圧力，(ｂ)残留ガス組成，(ｃ)使用環境（特に温度），(ｄ)化学反応の有無（特に腐食性ガスの使用の有無），(ｅ)プラズマや高エネルギー粒子の存在，(ｆ)特殊条件として，磁場の有無および電磁波の許容レベルや許容振動レベルなどが挙げられる．
真空装置とは，真空雰囲気を作成して種々の用途に使用する装置である．
魔法瓶，蛍光灯，ブラウン管などのように密閉容器から成る真空装置と，真空容器と真空ポンプで構成される真空装置の両者がある．
前者は真空雰囲気を長期間にわたって維持するため，放出ガスを十分に低減しなければならない．
そのため，容器を封止する前の脱ガス処理が欠かせない．
また，チタンなどのガス吸着材料（ゲッタ材）を内部に設置した装置もある．
後者は一般的な真空装置である．
装置構造に加えて，真空材料，真空排気系，真空計測系（真空計，質量分析計など），ガス導入系，試料導入と搬送系，その他の装置固有の系などが設計対象である．
真空装置の設計において考慮すべき項目は，用途，装置なお，HO分子が多く含まれる気体を排気する場合は，の排気時間より長くなることが知られており，経験的な目b.吸着（adsorption）ガス分子の排気 低真空から高真空および超高真空領域では，材料表面に吸着しているガス分子の脱離現象が支配的である．
