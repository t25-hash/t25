# 4・2・2 燃焼特性

> **章**: 第4章 ロケット機関

a. 点火(ignition)固体モータ主推薬への点火は、点火器から供給される点火噴流でグレイン表面を加熱することで行う。点火噴流の対流加熱によって推薬表面のAP粒子とバインダが一部熱分解し、発生する酸化性および燃料成分ガスが表面近傍の気相中で拡散・混合し、燃焼反応を開始するのが着火の第1段階である。通常、点火噴流による加熱開始から初期着火によるモータ内圧の上昇開始までの経過時間を着火遅れ(ignition delay)時間τと定義する。着火を確実にするには、点火噴流による加熱時間をτに比べて十分長く設定する必要がある。点火器の主装薬としてペレット状に成形した点火薬を用いる場合、その火炎持続時間は、ほぼ200ms一定である。これを点火モータと呼び、その主装薬にモータ主推薬の友材を利用することが多い。ただし、グレイン初期着火面が成形時のマンドレル離型面でなく切削面である場合、表面粗さとAP粒子の露出による対流加熱・熱分解効率の改善によってτが大幅に短縮され、上記の相関から外れる。通常の前方着火の場合、初期着火は対流加熱による局所熱伝達率が最大となるグレイン先端からポート径の2倍程度の位置で起きる。その後モータ内圧が上昇して平衡圧力に達するまでの着火遷移期間の間に、既着火面からの燃焼ガスによる蓄圧過程と未着火面への火炎伝ぱが並行して進行する。着火起点から後流部位への火炎伝ぱは非常に早く、所要時間はほとんど無視できる。このため、図4・12から読み取れるように、大細長比モータの着火遅れ特性は長さがその3分の1規模のモータと同等となる。モータが定常燃焼に移行する直前の着火遷移期間の終了期には、点火噴流と主推薬燃焼ガスの重畳と蓄圧過程の慣性現象により内圧曲線にのこ歯状の点火ピークが現れる。モータの高性能化や点火システムの簡素化を図る目的で、細長比の小さい上段用モータのノズルスロート直後流部位に装着した点火モータからスロートを介して点火噴流をモータ内に噴入し、着火遷移期間の末期に点火器自身をモータ内圧で吹き飛ばす形式の投棄式後方着火点火方式、観測ロケットのように細長比の大きいモータのグレイン先端部に点火モータ主装薬に相当する燃焼面積をもつ空孔(cavity)を設けておき、小型の点火ブースタによってまず同部内面に着火し、そこから発生する燃焼ガスによって残りのグレイン全面に着火する形式のキャビティ点火方式等が考案・実用化されている。図4・12 特殊条件下における燃焼。 加速度の影響Al充てん推薬の場合、飛翔中の姿勢安定化スピンおよび推力加速度の大きい機体では、加速度が燃焼特性に与える影響に注意する必要がある。通常、推薬中のAl粒子は、気相にわき出して化学反応に関与する以前に、燃焼表面に滞留・凝集して焼結し、それらの溶融成分が合体して集塊(agglomerate)を形成する。スピン燃焼に伴って燃焼面に直角に加わる加速度成分は、固相からのわき出し気流の揚力によって気相に飛び出そうとするAl粒子を推薬表面に押し付けて集塊の成長を助長する結果、固相への熱伝達率を高めて局所線燃速の増大をもたらす。スピン安定推力飛翔中のモータ内部では、燃焼面に滞留しているAl集塊および気相中に燃焼しつつ浮遊しているAl粒子の双方にスピンによる遠心加速度と推力飛翔に伴う軸加速度が重畳作用する。滑落した集塊粒子は、ノズル埋没部外周および周辺のケース断熱材表面に滞留・蓄積してスラグ(slag)を形成する。気相中の浮遊粒子に作用する体積力は、これらをノズル入口近傍で急激に偏向する気相流線から逸脱させ、の方向に沿ってノズル埋没部外周とその周辺部に衝突・滞留させてスラグの成長に加算的に寄与する。Al集塊特性の強い推薬を充てんしたモータの場合、地上でののみを加えたスピン燃焼試験と比べて、が重畳する飛翔環境では、上述のAl集塊粒子の掃拭効果によって燃速の増大効果が減殺される代わりに、未燃Alを大量に含むスラグの形成により比推力が低下し、モータ質量比が劣化する。高温・大熱容量スラグの大量残留は、ふく射加熱により間接に、熱伝導により直接に、耐・断熱材の焼損・熱分解を促進して後述の残留内圧特性の劣化を招く。推薬のAl集塊特性を緩和するには、AP粒度配合を複数分布とし、中粒以上のAP粒径を小さく、細粒APを多くする、充てんAl粒子の粒径を極力小さくするなどの配慮が必要である。. limit)と呼ぶ。真空中を飛翔する上段モータでは、充てん推薬の燃焼終了後、赤熱した黒鉛材スロートインサートを含むノズル埋没部からのふく射加熱によるケース断熱材の熱分解ガスにより、長時間にわたって6kPa以下の微小な圧力が発生し続ける。これを残留内圧(residualchamber pressure)と呼ぶ。充てん推薬の燃焼終了期から残留内圧域への過渡期間における圧力減衰挙動は、モータの形態、ノズル埋没部の形状と熱容量に依存し、グレイン形状設計によるスライバ(sliver)残留形態、推薬の低圧燃焼特性、ケース断熱材の熱分解特性によって決定される。細長比が小さく高ノズル埋没率で、燃焼温度の高い高性能推薬を充てんした上段モータほど残留内圧の減衰が遅く、持続時間が長い。e. 振動燃焼(oscillatory burning)固体モータのフリーポート(free port)は、音響学的には、気柱振動の場(acoustic cav-ity)である。そこに発生する微小な圧力擾乱が音響波として伝ぱする過程で増幅されるか減衰するかは、おもに推薬表面からの音響エネルギーの補給による増幅効果とノズル排気によるエネルギー散逸や不活性内壁面との摩擦等による減衰効果との大小関係によって決定され、これが増幅される場合には燃焼が不安定になる。圧力振動(pressure oscillation)には、表4・4に示すように、多様なモードが存在するが、最近の高Al充てん推薬では、燃焼ガス中に浮遊する大量のアルミナAlO凝縮相(condensed phase)粒子の内部摩擦効果によって横モード(transverse mode)の圧力振動は抑制され、縦モード(axial mode)の振動のみが観測される。図4・13 実用推進薬燃焼限界圧と同近傍の線燃焼速度特性表4・4 coupling oscillatory burn-ing)と呼ぶ。圧力振動が強まって振幅の大きい波動状態になると、燃焼ガスが燃焼表面に沿って平均流速に波動による速度成分が重畳する形で往復流動するようになり、ついには、前述の侵食燃焼効果によって燃焼速度が増大して平均内圧が大幅に増大する。このような激越な振動燃焼現象を速度連成振動燃焼(velocity coupling os-cillatory burning)と呼ぶ。は時刻=0における振幅、ωは角振動数で、αはモータ増幅定数(motor growth-constant)と呼ばれ、以下のような寄与項の線形結合で与えられる。このことは、振動燃焼の発生が観測されるのが観測ロケットに代表される形態的に細長く、推薬充てん率の高い小型モータに限られる、という経験的事実と合致する。最も普遍的な手法は、その形態的特徴からT バーナ(T-burner)と呼ばれる両端部に試料推薬片を設置した円筒形燃焼器内における気柱振動の増幅および減衰係数を測定して、それらの差としてαを求める方法である。内圧振動は、燃焼器中央に設けたパドルを急速に開放することによって誘起する。T バーナによる測定精度は、300Hz 以上の高周波帯域では十分高いが、これ以下の低周波帯域では燃焼器が長くなり内壁面への熱損失が大きくなって音速が低下するため急激に劣化する。実用上、内圧振動の発生が懸念される細長比の大きい観測ロケットや探査機打上げ用ロケットの下段モータの場合、一次縦振動の周波数は50~150Hz の範囲にあり、T バーナは有効でない。このような低周波帯域での測定には、熱損失を抑制するため内壁面を断熱材で覆い、より大型の内面燃焼型試料推薬ブロックをケース内前後端部に装てんした実用モータの形態に近い形状の細長い円筒形燃焼器が利用される。内圧振動は、ケース頭頂部に装着した火工品の作動による圧力スパイクの印加によって誘起する。この方法でも高Al充てん推薬の場合、αの高精度測定は難しい。近年、燃焼圧力の周期的変動による推薬固相へのフィードバック熱量変化をレーザ照射によるふく射加熱によって模擬して得られるふく射応答関数を圧力応答関数に換算する方法や、ロータリバルブ等によって強制的に燃焼器内圧を任意の振幅、周波数で振動させて圧力応答関数を直接測定する方法など、実験室規模の装置でより高精度に低周波帯域でのαを測定するための研究が進められている。モータ内圧の平均値が一定に保たれる圧力連成振動燃焼と異なり、モータ内圧そのものが数Hz から数十Hz の低周波で増減を繰返す不安定燃焼現象がまれに観測される。低周波燃焼振動(lowfrequency burning oscillation)と呼ばれるこの現象は、圧力変動に応答して起きる燃焼ガス発生量の変動とノズル排気量の変動との位相のずれに起因する。これが低い燃焼室圧で発生すると、燃焼が間欠的に起きる息つき燃焼(chuffing)の現象を呈する。
