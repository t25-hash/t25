# 4・2・3 固体モータのノズル性能

> **章**: 第4章 ロケット機関
> **節**: 4・12 に示す規模・形態の異なる多くの固体モータの点火モータに

固体モータのノズル推力損失は、おもに、ノズル出口開口角による余弦損失(divergence loss)とアルミナ凝縮相の存在による混相流損失(two-phase flow loss)に起因する。ノズル膨張流内部で進行する凝縮相粒子と気相間の運動量・熱交換の過程で発生する遅れ(lag)により流れが非平衡となるために損失が生じる。また、粒子のもつ慣性によって粒子流線がガス流線から逸脱することによって、凝縮相の膨張・加速効率に関わる二義的な余弦損失が発生する。宇宙航空研究開発機構が実施した規模・形態・充てん推薬の異なる各種固体モータの地上燃焼試験において捕集したノズル排出アルミナ粒子のデータも、図4・15に示すように、上式の相関によく従う。現在、宇宙航空研究開発機構が比推力の推定とノズル内面形状4―142内燃機関図4・15 ノズル排出アルミナ粒子粒径のスロート径依存性図4・16混相流ベルノズルの設計パラメータ(contour)の最適設計に利用している二次元軸対称混相流解析ルーチンでは、超音速領域を特性曲線法によって解く。ノズル入口部からの遷音速領域には実績のある近似解を適用し、気相を理想気体、凝縮相を粒径が一定・不変の一様粒子群として扱う。α=0.464,β=0.297が最適値とされており、これによる比推力の推定精度は、モータ規模によらず、±0.3%(~±1s)程度と高い。固体モータでも、コンタ母線長を極小化しつつ比推力を最大化することを目標に、ベルノズルの最適形状設計が行われる。宇宙航空研究開発機構における設計手法では、ベルコンタに放物線アークを仮定して図4・16に示すような設計パラメータを定義する。ノズル内を流動するアルミナ粒子群はコンタに即応して膨張する気相より遅れて膨張するが、それらがに対応する幾何学的平均粒径を中央値とする対数正規分布をもつことから、ノズル後流部位の断面では中心軸近傍に大粒径粒子が集中し、周辺部に微小粒径粒子が偏在する分布となる。図4・18は、幾何学的に相似な混相流ノズルでは、規模が小さくなるほど比推力特性が劣化して粒子衝突が起きにくくなること、>150mm規模の大型のノズルでは、粒子衝突回避の条件としてθ<12°を厳守する配慮が必要であることを示しており、これらは実験的にも検証されている。現在、数値流体力学的手法によってモータ内部からノズル出口に至る全混相流過程を一貫して解くシミュレーション技術の研究開発が進められており、近い将来、ノズル超音速域における流動粒子の三次元的挙動の詳細解明に加えて、フリーポート内の亜音速域からスロートを通過するまでの遷音速域における凝縮相の挙動と粒度分布再形成過程の定量的解明、ケース内スラグ残留特性およびノズル構成部材への局所熱伝達率特性とこれによる耐・断熱材焼損特性の一括数値解析が可能となるものと期待されている。
