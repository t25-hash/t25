# 4・1・3 ガス放出量（outgassing rate）

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

ガス放出量は，材料の種類，加工条件，洗浄条件，保管条件，排
気の履歴などの影響を強く受ける．真空溶解したステンレス鋼
SUS316Lのガス放出特性を，図Ⅳ-4･2に示す ．最初の排気①
の後に，250℃×24hのベーキング処理（baking）を実施した②の
ガス放出量は，①に比較しておよそ3けた減少している．次に大気
開放して再び真空排気を開始すると，①と比較し③は約1けた近く
ガス放出量が低下した．さらに150℃×15hのベーキング処理を実
施した後のガス放出量④は，②よりも低い．この後，大気開放とベ
ーキング処理を繰返しても，⑤および⑥に示されるように，ガス放
出量の変化は認められない．この現象は，ステンレス鋼の表面に，
CrやFeの酸化物が形成される挙動と考えられている．ガス放出
量が表面状態や排気過程の影響を受けることを活用し，事前に真空
ベーキングや表面析出などの特殊な処理を実施することにより，ガ
ス放出量が低減できる．
図Ⅳ-4･3，Ⅳ-4･4はベーキング処理の最高温度と，単位面積当
たりのガス放出量の関係 を示したものであるが，ステンレ
ス鋼およびアルミニウム合金のいずれにおいても，特殊な前処理の
有無でガス放出量が1けた近く異なっている．なお，単位面積当た
りのガス放出量をガス放出速度という．
表Ⅳ-4･1 に各種金属材料のガス放出量を示す［文献(31)
(34)(36)～(42)］．表Ⅳ-4･2 に非金属材料のガス放出量を示
す［文献(５)(21)(43)～(45)］．概して非金属材料は，金属材料に
比較してガス放出量の大きいものが多いので，事前にガス放出量を
把握しておく必要がある．
ガス放出量は，材料固有の物性値ではなく種々の影響を受けるの
で，ある程度の誤差を許容したうえで真空装置を設計する．洗浄条
件によってもガス放出特性が影響を受ける ．
