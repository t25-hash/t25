# 4・2・4 推薬の力学物性

> **章**: 第4章 ロケット機関
> **節**: 4・12 に示す規模・形態の異なる多くの固体モータの点火モータに

推薬グレインは,成形後の残留熱応力,保管中の自重負荷と環境温度の日周変化,推薬の燃焼に伴う熱と圧力,飛翔加速度の影響等さまざまな荷重条件にさらされる.その成分である粘結材は劣化が避けられない高分子材料であり,APは吸湿性をもち,Alは酸化しやすい.このため,長期保管には燃焼特性・力学物性の変質について留意が必要で,恒温・恒湿に加えて自重による垂れ込み(slump)の向きが偏らないような保管環境の整備・管理が要求される.このような荷重条件が原因となって起こりうるき裂や接着はく離などグレインの力学的局所破壊は,燃焼面積の増大によりモータ内圧の異常昇圧を誘起してその推力プログラミングを損ない,過度の場合はモータ構造部材を破壊する.a. 単軸引張特性と破壊過程コンポジット推薬は,材料力学的には粘弾性挙動を示す粘結材を基材とする粒子分散型の複合材料であるから,温度・ひずみ速度に強く依存する複雑な力学物性・破壊特性を示す.推薬の単軸引張試験において,温度・ひずみ速度を変えて得られる力学物性・破断特性データは,線形粘弾性(linearviscoelasticity)理論と時間-温度等価則(time-temperaturesuperposition principle)に基づく換算変数法(reduced variablescheme)によって,一元的に整理することができる.推薬の単軸引張応力-ひずみ特性は,図4・19に見るように,たかだか二つの極大点が現れる複雑な挙動を示し,その様相が温度・ひずみ速度によって大きく変わる.これは,巨視的な破断に先行してAP粒子が粘結材からはく離するデウエッティング(dewet-ting)と呼ばれる微視的な破壊過程が進行することに起因する.試験片に引張ひずみが加わると,まずAP粒子近傍の粘結材相内で応力集中による凝集破壊(cohesive failure)が起こり,これがやがてAP粒子と粘結材界面の接着破壊(adhesive failure)に発展する.この破壊過程はAP粒子の粒径が大きいほど起きやすい.はく離したAP粒子は粘結材の空孔中に浮遊した状態となって材料補強効果を喪失するから,荷重は引き伸ばされた粘結材フィルム群によってのみ支えられることとなる.さらなる引張ひずみによるそれらの断裂が試験片の巨視的な破断につながっていく.応力-ひずみ曲線に現れる二つの極大点は,それぞれ,粒子のはく離と粘結材フィルムの破断という異なる微視的破壊過程による応力緩和の結果である.複数粒度分布のAP粒子を充てんした実用推薬では,通常,これら二つの応力緩和過程の重畳によって,応力-ひずみ曲線の極値は一つ,すなわち,最大応力点(ε욇,σ욇)しか現れない.粒子はく離が起きる以前の低ひずみ域では推薬は完全な線形粘弾性挙動を示す.この領域では,近似的にポアソン比0.5,すなわち,体積弾性率無限大とみなしてよい.この領域で広範な温度・ひずみ速度に対して求めた応力-ひずみデータは,換算変数法によって,図4・20に示すように,薬種ごとに固有の換算応力-ひずみ曲線に集約される.図4・20中,λ=1+ε,はひずみ速度,は時間-温度等価則における変換パラメータ(shift strain)は,最大応力点ひずみε욇の0.5~0.7倍の範囲にある.図4・19に示したように,推薬の引張強さも温度・ひずみ速度に大きく依存する.換算応力-ひずみ平面にプロットした破断点(ε욁,σ욁)を結んで得られる曲線は,ひずみ履歴によらないという性質があり,破壊包絡線(failure envelope)と呼ばれる.破壊包絡線の形状も充てんAP粒子の量・粒径によって大きく変わる.上述のような推薬の力学的破壊過程の理解から,グレインの構造解析を行ううえでの破壊基準はひずみ評定で設定される.これまで,ε,23ε욇もしくは12ε욁等の採用が提案されているが,以下の理由によりいずれも決定的でない.ε以上,ε욇以下の水準に達するひずみ履歴をもつ推薬試料も,除荷後十分な時間が経過すれば,処女材と変わらない物性および燃焼特性を取り戻す.この水準のひずみ状態に保持されている試料の燃焼態様も,処女材と同様安定した爆燃(deflagration)で質量燃速に有意差はない.燃焼特性に異常が現れるのは,ε욇を超えるひずみ状態に達して粘結材フィルムの断裂が進行し,粗粒APのデウエッティングによって生成した多数の空孔が連成して連続気泡状態になってからである.この状態では,十分な熱および圧力衝撃の印加によって爆ごう(deto-nation),あるいは,初期静穏な爆燃から爆ごうに移行するDDT(deflagration to detonation transition)現象を起こす可能性がある.このような破壊過程がさらに進行して巨視的なき裂が生成すれば,高温高圧の燃焼ガスの侵入によって燃焼面積が増大して加速度的にモータ内圧が上昇する.圧力容器内において推薬試料の単軸引張試験を行うと,印加圧力の上昇につれて,そのヤング率,最大応力点応力-ひずみ,破断点応力-ひずみが連動して増大する.これは,流動性の高い粘結材が4―144内燃機関圧力媒体として作用し,粒子はく離に至る微視的破壊過程の進行を阻害するからである.また,グレイン内部で発生しうる局所的な応力集中場の近傍では,隣接する領域で粒子はく離が発生して急速に応力-ひずみが緩和されるから,周辺健全部へのき裂の伝ぱが防止される.このような特徴的な破壊特性により,現用のコンポジット推薬充てんモータでは,モータの機能破壊につながるような重大な推薬局所破壊の発生例は皆無である.b. グレインの応力-ひずみ解析モータ構造設計では,システム要求仕様から保管条件,運用環境温度,燃焼圧力,飛翔加速度等の荷重条件が設定され,各荷重条件における破壊モードを想定したうえで対応した検討を進める.グレインに加わる荷重条件と破壊モードは多様であるが,破壊部位はモータの構造・グレインの形状に依存し,直てんモータでは,ほぼ変形が最も集中し応力-ひずみ集中度の高いグレインの内表面および端面部に限られる.グレインには,通常,いくつかの荷重条件が重畳した複合荷重が加わる.例えば,直てんモータでは,グレイン内部に生起する推薬硬化温度と運用温度との差に起因する応力-ひずみがモータの全運用期間にわたって残留し,燃焼時には圧力荷重による応力-ひずみが加わる.細長い円筒内面燃焼型グレインでは,温度差および圧力荷重による内表面最大主ひずみ(接線ひずみ)はいずれも引張ひずみで,弾性近似解はそれぞれ次式で与えられる.ε=β
