# 4・1・5 真空計測

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

大気圧から超高真空までの圧力領域を，単一の真空計（vacuum
gauge）でカバーすることはできない．そこで，(１)大気～10
Pa程度，(２)10～10Pa，(３)10Pa以下（10Pa程度ま
で）の3領域に対応した真空計を選定する．用途別では，(１)高精
度圧力測定，(２)通常の圧力測定，(３)高精度圧力モニタ（圧力制
御用），(４)圧力モニタ（大気か真空の判定）に分類できる．要求
第Ⅳ部 真空機器 第4章 真空システム
仕様別では，(１)高精度（圧力の測定精度が高い），(２)長期安定
性（測定値の変動量が小さい），(３)耐久性（故障しにくい），(４)
耐食性に優れている，(５)ガス放出量が少ない，(６)操作性に優れ
る，などに区分される．
圧力計の測定精度が必要な場合は，真空計を校正 しながら
使用しなければならない．スピニングロータ真空計 は長期の
安定性に優れていることから，低真空領域の基準真空計として広く
使用されている．高真空用の基準真空計は，標準マクラウド真空計
にて校正された三極管型熱陰極電離真空計VS1- A が，日本真
空協会から供給されている．
超高真空から極高真空領域では電離真空計が使用されるが，軟
X線効果の影響で，測定可能な真空計が限定される．市販の真空
計ではエクストラクタゲージ が最も測定下限圧力が低く，2×
10Paである．
