# 4・2・6 推力方向制御装置(TVC: thrust vector control)

> **章**: 第4章 ロケット機関
> **節**: 4・12 に示す規模・形態の異なる多くの固体モータの点火モータに

固体モータの推力方向制御装置には,固定ノズル内に二次液体を噴入して発生する斜め衝撃波によって生じた内壁面の圧力不平衡を利用する液体噴射LI(liquid injection)TVCとノズル全体を機械的に偏向させる可動ノズルMN(movable nozzle)TVCがある.最近では,耐圧性を備えたゴムエラストマと金属シムの積層構造をもつフレキシブルジョイント(可撓継手)を介してケースに結合されたノズル全体を駆動・偏向させるMNTVCが一般的である.この方式では,つねに主推力とノズルの偏向角δによって決まる横推力sinδを得ることができ,大きな制御力が必要な場合,二次噴射量に対して制御力が飽和する傾向をもつLITVCより有利である.下段モータでは,偏向角に依存するフレキシブルジョイントのばね反力が大きいことと大型のノズル構造全体を偏向させる必要から,大駆動力の油圧駆動アクチュエータを用いる.小型の上段モータでは,空気力が無視できるため所要偏向角が小さく,またノズル規模・フレキシブルジョイントのばね反力ともに小さいことから,小駆動力ながら軽量・高精度で整備性・価格性に優れた電動アクチュエータを用いて高精度な制御を可能にしている.フレキシブルジョイントはノズルの偏向方向だけでなく機軸方向にも自由度をもち,燃焼内圧の上昇に伴いエラストマが圧縮されてノズル全体が後方に押し出される.これをPcバイアス(Pc-bias)と呼ぶ.ノズルには,通常,2基のアクチュエータが90°位相で装着されるから,それらのストロークが固定されていると,Pcバイアスによってノズルは,相対的にアクチュエータ装着側に引き寄せられて不整偏向し,制御上のミスアライメントとなる.こ4―146内燃機関の制御偏差を極小化するため,図4・22に示したように,ノズル軸に関してアクチュエータと対称位置にその部位での変位を計測するポテンショメータを装着する.その出力をアクチュエータの駆動指令信号に加算フィードバックしてアクチュエータの伸展量を修正し,両者のストロークが等しくなるように制御する.剛体の場合,この原理は両者のストロークの平均値をフィードバックすることと等価であるが,実際のノズルでは,構造共振等の影響により単純な平均値フィードバックとはならない.内圧上昇速度が大きいモータ着火時には,アクチュエータが急激に増大するPcバイアスに起因するノズル偏向速度に追従できず,ノズルは慣性による揺動や構造振動モードの変化を起こす.このときアクチュエータ軸にはこれらに応じた大荷重が作用するから,アクチュエータの荷重設計にはこれを十分考慮する必要がある.
