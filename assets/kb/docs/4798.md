# 4・1・6 真空シール（vacuum seal )

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

真空シールは真空を維持するための封止方法であり，固定シール
と軸シールに大別される．真空シールの仕様は，漏れ量を流量で表
示したリークレート（単位はPa･m /s ）で表され，ベース圧力と
真空排気系の仕様から許容リークレートが設定される．
固定シールは真空フランジのシールに代表される．溶接やろう付
け，ガラス封止などの取外しができない封止方法も，固定シールと
呼ばれる．真空用のフランジには，エラストマシール（Oリング）
用フランジとメタルガスケット用フランジがあり，JIS B2290：
1998「真空装置用フランジ」，JIS B8365：1988「真空装置用クラ
ンプ形継手の形状及び寸法」，および日本真空協会規格JVIS
003：1982「真空装置用ベーカブルフランジの形状・寸法」におい
て規格化されている．その他，金線やインジウム線，アルミニウム
線などの軟らかい金属線をシール材とする方法や，薄いステンレス
板やニッケル板を両側から締付けてシールする方法などがある．
軸シールはベローズやOリングシールなどの，運動伝達機構の
シールを指す．Oリングシールは潤滑とシール性能向上のため，真
空グリースを用いることが多い．磁性流体シール はシール部に
磁性流体を使用したもので，磁性流体を永久磁石の磁力でシール部
に局在化させている．真空用に使用されている磁性流体は，マグネ
タイトやマンガン亜鉛酸化鉄の微粒子を，有機溶媒の中に懸濁させ
たコロイド溶液である．しゅう動抵抗が小さいので，高速回転や微
小角回転に使用される．ベローズシールは完全封止型のシール方法
であり，高温にも耐えるので，ベーキング処理が必要な超高真空装
置に使用できる．ベローズには，ステンレス鋼など（SUS304や
SUS316，インコネルやハステロイ）の薄板を溶接接合したもの
と，薄肉パイプを加圧成形したタイプのものがある．伸縮量により
ベローズの寿命が決まるので，伸縮量に余裕をもった設計が必要で
ある．
