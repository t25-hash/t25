# 4・1・7 駆動機構および真空部品

> **章**: 第4章 真空システム
> **節**: 4・1 真空装置の設計

真空中の試料駆動方法には，真空中に設置した駆動源を用いる場
合と，大気中に駆動源を設置し，運動導入機構により真空中の試料
を駆動する方法がある．
真空中にモータ を設置して使用する場合，ガス放出と放熱
対策が必要である．真空中では気体による熱伝導が期待できないた
め，モータの放熱対策が難しい．その結果，モータ温度が上昇して
ガス放出量も増加する．発熱の比較的少ないステッピングモータが
市販されているが，低出力のものが多い．
大気中から真空中への動力伝達は，大気中の駆動源の動きを，軸
シールを用いた運動導入機構を介して真空中に導入する方法が一般
的である．軸シール部の漏れを防止するため二重にシール部を設
け，その中間部を差動排気する場合もある．
磁力を利用した動力伝達は磁気カップリングとも呼ばれる．磁力
を利用した運動伝達機構であるため，大気側と真空側の運動が必ず
しも一致しないが，回転や直線導入を同じ機構で導入することがで
きる．
運動機構の設計では，真空中の摩擦抵抗が大気中に比較して数倍
大きいことに注意しなければならない．しゅう動部を避けて転がり
接触にするような構造上の工夫や，接触部の材料は異種材料を選定
するなどの配慮を要する．潤滑材は，蒸気圧が低ければ使用可能
で，フッ素系のグリースや真空ポンプ用の潤滑油などが使用でき
る．フッ素系樹脂やポリイミド系樹脂などの高分子材料もしゅう動
部に使用される．また，しゅう動部の母材には，Au，Ag，Inなど
の金属薄膜や，MoS ，WSをコーティングした固体潤滑も使用さ
れている．真空用ベアリングには，AgコーティングやMoSコー
ティングを処理したものもある．
電流導入端子には，電流導入の金属線をガラスやセラミックスで
封止したものや，Oリングなどのエラストマシール方式のものが市
販されている．耐電圧や許容電流により種々のタイプがある．
光の導入としては，硬質ガラス，石英，サファイアなどのフラン
ジ付き窓が市販されている．透過させたい光の波長に応じて窓材料
を選定する．そのほか，X線用のベリリウム窓やマイクロ波用の
セラミック窓などもある．
真空中で爆発性のガスを使用する場合は，真空容器に破壊しやす
い部分を設けて大事故を防止する．金属薄膜を用いたラプチャディ
スクが市販されている．ラプチャディスクの取付位置は，周囲の状
況を十分考慮して決めなければならない．
