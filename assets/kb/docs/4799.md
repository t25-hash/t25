# 4・3・1 液体ロケット推進システム

> **章**: 第4章 ロケット機関
> **節**: 4・3 液体ロケット

ロケットの推進系基本仕様は,推進薬の種類,搭載量,エンジンへの供給方式(ガス押し式,ターボポンプ方式),使用するエンジンの仕様(推力,性能,混合比,エンジン基数,エンジンサイクル)等があり,全機システムの基本要求(ペイロード要求,射点制約,保安距離等)を踏まえて設定されるが,その詳細仕様が全機システム仕様(機体質量,形状,性能等)に影響するため,設計,開発の過程で適宜フィードバックをかけながら開発作業が進められる.本項では,液体酸素,水素を推進薬とし,ターボポンプをエンジンに用いた方式の液体ロケット推進システムに関し,その概要を述べる.液体ロケットの推進系システムの例として,H-IIA ロケット第一段推進系システム系統図を図4・23に,また,推進系システムの主要構成品例として,H-IIA ロケット第一段推進系の例を表a. 推進薬タンク推進薬タンクは,機体質量に占める割合が大きいため,ロケットの基本仕様に最適な容量,使用圧力,断熱性能等を設定する必要がある.図4・23 H-IIA ロケット第一段推進系システム系統図. 容量タンク容量は,搭載推進薬量から推進薬量構成(有効/無効推進薬量),アレッジ体積,エンジン混合比等を考慮して,酸化剤タンク/燃料タンクの容量が設定される.(1) 有効推進薬量:ロケットの推進に寄与する推進薬(2) 無効推進薬量:ロケットの推進に寄与しない推進薬量で,フライト中のエンジン予冷等で消費される分,燃焼終了後にタンクおよび機体内に残留する推進薬量等から構成される.(3) アレッジ体積(ullage volume):おもにタンク加圧初期等で発生する圧力オーバシュートを軽減するために,全体の数%の体積(通常3%程度)が推進薬を搭載しない上部ガス領域(アレッジ)として設定される.(4) エンジン混合比:燃焼中のエンジンの燃料/酸化剤使用比率.タンク容量は,通常エンジンのノミナルの混合比を用いて設定される.混合比のエンジン個体差により,最適の推進薬搭載量比率が個々の機体で異なってくるが,エンジン混合比のフライト中制御(推進薬有効利用制御,propellant utilization control)や,エンジンの領収燃焼試験で得られる混合比タグ値(実際に得られた値)により,アレッジ体積を変化させ搭載量をコントロールすることで対表4・6 推進系構成(H-IIA ロケット第一段の例)推進薬供給系推進薬タンク(タンク断熱構造)液体水素タンク液体酸素タンクタンク加圧系LHタンク加圧系LOX タンク加圧系推進薬供給系(フィードライン系)LHフィードライン系LOX head,圧力から推進薬液温相当の飽和蒸気圧を引いたものであり,推進薬流体のヘッドで表示される)を確保するため,飛行中の推進薬供給系圧力損失,飛行加速度,推進薬量から求まるヘッド圧力,各種センサ誤差等を考慮して,タンク圧力が設定される.推進薬の飽和蒸気圧は,おもに推進薬充てん後のタンク内の液温で決まり,液温は入熱と蒸発したガスが排気されるまでの圧力損失に支配されるため,これらを考慮して断熱仕様が設定される.通常,液体水素タンクの断熱材には,軽量で断熱性能の高い,ウレタン系断熱材のPIF(ポリイソシアーヌレートフォーム,H-II,H-IIA ロケットで使用),塩化ビニルフォーム(アリアン,セントールで使用)等が使用される.. タンク内ぎ(艤)装品推進薬タンクには推進システムとして必要な次のようなものがぎ装されている.(1) スロッシュ防止バッフル(slosh baffle):液面のスロッシング振動を抑制し,制御上の安定性を確保するためのリング状のじゃま板.(2) 渦防止バッフル(vortex baffle):タンク出口付近の渦発生を抑えるとともに,燃焼末期の液面の落込みを抑制するため,十字形状にじゃま板を組立てたもの.(3) タンク出口スクリーン:タンク内のごみが流出するのを防止するため,出口付近にぎ装される金網を用いたフィルタ.(4) センサ類:液位を計測するレベルセンサと,液温,タンク内表面温度を計測するための温度センサが装着される.レベルセンサのうち,タンク底部に複数個(通常3個)取付け,推進薬が枯渇したことを検知する目的に使用するセンサを,ディプリーションセンサと称する.(5) ディフューザ:タンク加圧ガスの流速を抑え,加圧ガスが推進薬表面をたたくことにより生じる液の飛散,ならびに液への入熱量増加,加圧ガス量の増大を防止する加圧ガス出口.b. タンク加圧系フライト中,設定したタンク圧力を保持するため,加圧ガスを用いて推進薬タンクを加圧する系統であり,加圧方式としては,加圧ガス流量を制御することでタンク圧力を制御する方式と,固定オリフィスによる方式がある.一般的に,前者はタンク圧力を適正な範囲に制御できる反面,バルブ・オリフィス等の部品,バルブを開閉制御するためのシステムを要するため,後者に比べてコスト高なシステムとなる.一方,後者はフライト中のタンク圧力変化が大きくなる傾向があるため,限られたシステムにしか適用できない場合が多い.適用例としては,前者がH-IIA ロケット第一段液体水素タンク加圧系,後者がH-IIA ロケット第一段液体酸素タンク加圧系がある.加圧ガスとしては,推進薬をエンジンでガス化して使用する方式と,あらかじめヘリウムガス等を気蓄器等の圧力容器に搭載して使用する方式があり,システムとして要求される性能(加圧ガスに使用する質量,上段系の場合はコースト中のガス凝縮の考慮),コスト等を考慮して選定される.前者の適用例としては,H-IIA ロケット第一段/第二段液体水素タンクおよび第一段酸素タンク加圧系,後者の適用例としては,H-IIA ロケット第二段液体酸素タンク加圧系(液体水素タンクに搭載した極低温気蓄器のヘリウムガスを加温して使用)がある.なお,酸素ガスをタンク加圧ガスに使用する場合には,使用する配管,バルブ等の材質,ガスの流速等の酸素ガスに対する適合性に関して慎重な検討が必要である.c. 推進薬供給系(フィードライン系)推進薬タンクからエンジンまで推進薬を供給する系統(フィードライン)は,下記の観点を考慮する必要がある.. 配管径フィードラインの配管径は,発生する圧力損失による推進薬タンクの質量の増分と,配管サイズに対するフィードライン質量の全体として最適になるように選定される.. ゲイサリング(geysering)極低温推進薬で,配管径に4―147比べて長いフィードラインを使用する場合には,ゲイサリング現象(配管下部の液体が入熱により間欠泉的に蒸発,吹き上げる現象.吹き上げた後,配管内のガスが凝縮,推進薬が落ち込むことにより,水撃的なピーク圧力が発生する)に注意する必要がある.特に,液体酸素等の密度の大きい推進薬で,ヘッド差の大きい長尺配管を使用する場合には,ゲイサリング発生に伴う水撃ピーク圧力が大きくなるため,十分なゲイサリング防止策をとる必要がある.防止策としては,フィードライン中にヘリウムガスを吹き込むことで冷却する方式(スペースシャトルで使用),推進薬を一定流量ブリード等することにより流動を与え,配管下部に高温の液が停留しないようにする方式(H-II,H-IIA ロケット第一段で採用)等がある.. POGO(ポゴ)液体ロケット燃焼末期に,全機振動に伴う推進系の圧力変動と,この圧力変動によるエンジン推力変動がクローズドループとなって発生する全機振動現象(POGO)が認められる場合がある.N-I,N-IIロケット第一段POGO発生時のフィードライン内脈動状況を図4・24に示す.POGO発生の有無については,安定性解析を実施することにより予測することが可能である.また,POGOを抑制するための装置として,フィードライン中にアキュムレータとオリフィスをもつPSD(POGO抑制装置)が使用される.d. 配管,バルブ類各システムラインの目的に応じて口径,材質,バルブの様式等を選定して使用する.極低温推進薬を用いたロケットの場合,材料の低温特性,溶接性を考慮して,300系のステンレス,5000系/6000系のアルミ合金,ニッケル合金,チタン合金等が用途に応じて使用される.また,一般に配管類に加わる変形量が大きくなるため,ベローズや配管自身の変形による可撓性を用いて吸収させる配慮が必要である.配管接続部の継手には,シールや管継手が用いられる.常温の液体推進薬には推進薬との適合性に十分注意したうえで,ゴム系,テフロン좲(デュポン社の登録商標)系のシールを選ぶ.極低温推進薬に対しては,金属Oリングもしくは断面形状がU 字形,K 字形などの機械加工した金属リングにテフロン좲をコーティングまたは,金・銀めっきしたもの,あるいはテフロン좲,三フッ化塩化エチレン樹脂のカバー材を金属スプリングでバックアップしたシールが用いられる.軸シールには,テフロン좲またはマイラのシートを用いたリップシールが用いられる.極低温配管用管継手類としては,フレア付き管継手も用いられるが,より確実なものとしてはダイナチューブ좲(レジストフレックス社の登録商標)継手,小さな金属Oリングを挟み込む継手などがある.バルブ類には,空圧弁,直動型またはパイロット型の電磁弁が使用される.通常,口径が大きく低温になるバルブには空圧弁,空圧弁の駆動ガス(通常Heガスまたは窒素ガス)をコントロールする目的には電磁弁が使用される.e.
