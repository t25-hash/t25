# 4・3・2 液体推進薬

> **章**: 第4章 ロケット機関
> **節**: 4・6 に示す.

液体推進薬には燃料と酸化剤を燃焼させる二液推進薬(bi-propellant)と,一液のみで分解ガスを発生させる一液推進薬(monopropellant)がある.さらに二液に固体または液体の第3成分を加えたり,2種類の燃料を切り換えて使用する3成分(三元)推進薬(tripropellant)も研究が進められている.貯蔵性の観点からは,常温で貯蔵できるものを貯蔵性推進薬(storable propel-lant),極低温での貯蔵が必要なものを極低温推進薬(cryogenicpropellant)と呼ぶ.なお,極低温推進薬のうち,沸点の比較的高いもの(例えば123K 以上)は,宇宙貯蔵性(space storable)推進薬と呼ぶ場合がある.貯蔵性推進薬のうち,ヒドラジン,モノメチルヒドラジンや四酸化二窒素は長期間の保管でも劣化が少なく,システムを簡単にできることからロケットの姿勢制御,宇宙機/人工衛星の推進系を中心に広く使用されてきた.しかし,これらの燃BFT射場カウントダウンCFT*:BFT はBatteleship Firing Test(厚肉タンクステージ燃焼試験),CFT はCaptive Firing Test(実機型タンクステージ燃焼試験)を示す料は毒性が強く,射点や着陸場,宇宙ステーションなどに対する安全性確保のために,適切な機体のハードウェア設計,防護設備の準備が必要となり,コスト増大/運用制約要因となっている.このような状況から,近年,四酸化二窒素/ヒドラジン系に代わる無毒性(non-toxic)推進薬(グリーンプロペラント)の研究が盛んになっている.推進薬を選定するには,理論的に発生しうる性能のほかに,推力室の冷却特性,材料との適合性,応答性,密度,毒性,取扱いの難易,貯蔵性,入手性,価格等さまざまな面からの検討が必要となる.a. 二液推進薬大型から小型のエンジンまで幅広く利用されている.代表的な二液推進薬の性質を表4・8に,性能を表4・9に示す.液体水素と液体酸素の組合せは,高比推力かつ無毒なため,上段用,一段用ともに使用例が増えている.一段用としては,液体酸素との組合せでRP-1等のケロシン燃料も多く用いられている.近年は液体水素やケロシンの代替として,メタンやプロパンなどの炭化水素系燃料の使用も検討されている.四酸化二窒素とヒドラジン/モノメチルヒドラジンのような,混合するだけで発火する推進薬を自己着火性(hypergolic)推進薬と呼ぶ.自己着火性推進薬は点火装置が不要で,システムを簡素化できるため,ロケットおよび宇宙船,人工衛星の軌道変換・姿勢制御用に広く用いられている.毒性の強いヒドラジン/四酸化二窒素系の代替として,近年過酸化水素とアルコール等の炭化水素の組合せで自己着火性をもつエンジンシステムの研究が盛んに行われている.b. 一液推進薬貯蔵時は安定であって,加圧・加熱・触媒などの作用によって,容易に高温ガスを生成できる性質が要求される.この条件を満たすもので現在実用段階にあるものは過酸化水素とヒドラジンのみであり,性能/安全性の観点から現状はヒドラジンの触媒分解式スラスタがおもに使用されている.表4・10に示すように比推力は低いものの,システムが簡単になるため,ロケット/衛星の姿勢制御等の比較的低推力を要求される場合によく用いられる.衛星用等で,より低推力で高い욦욣が必要な場合には,分解ガスにさらに熱を与える仕組みとして,ヒータで加熱する電熱式スラスタ,アーク放電で加熱するアークジェットスラスタ等が実用化されている.近年,毒性の強いヒドラジンの代替として,従来から使用されている過酸化水素をはじめ,酸化剤と燃料をあらかじめ混合し,触媒で分解ガスを得るタイプのHAN(hydroxylammonium nitrate)系混合推進薬や,ADN(ammonium dinitramide)系混合推進薬等の研究が盛んに行われている.
