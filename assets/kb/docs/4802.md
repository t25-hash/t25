# 4・3・4 液体ロケットエンジンコンポーネント

> **章**: 第4章 ロケット機関
> **節**: 4・31 )

a.エンジン組立 液体ロケットエンジンの代表例として，わ
が国で開発されたLE-5およびLE-7を取上げる．両エンジンとも
推進薬は液体酸素・液体水素であり，前者はH-Ⅰロケットの二
段，後者はH-Ⅱロケットの一段として使われた．表4･12に両エ
ンジンの主要諸元を示す．
.エンジン混合比 液体酸素・液体水素エンジンの比推力
が最大になるのは一般に混合比4程度であるにもかかわらず，実際
の混合比は5～6に設定されることが多い．図4･34にLE-5の比推
力と混合比の関係を示す．低混合比では機体タンクに大量の液体水
素を搭載しなければならず，機体の容積・重量がかさんでしまうの
で，エンジン性能を若干犠牲にしてでも，混合比はやや高めに設定
して，機体重量とエンジン性能の最適化を図るのである．
.LE-5エンジン（LE-5engine) 図4･35，4･36に
LE-5の外観図および配管系統図を示す．LE-5は二軸式ターボポ
ンプをもつガス発生器サイクルである．ポンプで昇圧された液体水
素（液水）は主燃焼室を冷却しながらガス化して噴射器に入る．主
燃焼室出口の水素ガスの一部は高膨張ノズルの冷却にも用いられ
る．液体酸素（液酸）はポンプで昇圧されて直接噴射器に入る．主
燃焼室で生成された燃焼ガスは高膨張ノズルを通って超音速で噴出
されて推力を発生させる．
エンジン始動にはまず始動弁を開けて，主燃焼室を冷却した水素
ガスでターボポンプを駆動する．エンジンの出力がある程度立ち上
がった時点で，ガス発生器に着火して始動弁を閉じて定格運転に至
る．この方式は始動時のみエクスパンダブリードサイクルとして作
動するものであり，LE-5の特徴の一つである．LE-5にはまた，
真空中での再着火能力がある．高膨張ノズルに取付けた熱交換器を
通してガス化した状態の酸素・水素を点火器に供給することによっ
て，主燃焼室およびガス発生器に着火するのであるが，高膨張ノズ
ルは初回燃焼の後でも高温状態を保っているので，その熱容量を利
用して再着火が可能なのである．
