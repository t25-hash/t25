種類と特徴液体ロケットエンジンの主要部品としてターボポンプ(turbopump)はほぼ唯一の高速回転機械とみなしてよい。その構成はポンプ部とタービン部に大別される。ポンプの形式は、運転範囲が広い・軽量・信頼性が高い・構造が簡潔等の理由により遠心ポンプが選択されることが多い。タービンの形式は、エンジンシステム影響やポンプの必要とする消費動力により決まるが、軸流タービンが主となっている。ペイロード重量を最大にするため、ターボポンプ自身には軽量化が要求され、かつ推進薬タンクの重量軽減のために高い吸込性能も併せて要求される。軽量化要求のためにターボポンプは小型/高速回転となる。高速回転と高い吸込性能要求は互いに矛盾する要求であるが、両者を成立させるために羽根車(impeller)の前にインデューサ(inducer)を設置することが一般的である。また、低速の予圧ポンプを別に設け、高速回転ポンプとして必要な入口圧力を確保することもある。高速回転を選択せざるを得ないことによりターボポンプの技術的難度は一般に高くなる。液体水素と液体酸素の組合せはクリーンで比推力()の高い優れた推進薬であるが、液体水素の比重が小さいので、ポンプ吐出し圧力を確保するためには羽根車材料の限界強度に近い周速や多段化が必要であり、構造が複雑となるため技術難度はさらに高4―156内燃機関まる。一般にロケット用ターボポンプは起動から数秒で全開出力まで立ち上がる。液体水素のような極低温推進薬を採用した場合は、ポンプ側は約20K の極低温であり、タービン側は1000K 近い高温となるため非常に狭い領域で大きな温度分布をもつことになる。また、立上りが速いため過渡時に大きな熱負荷を受けるなど非常に厳しい運転条件となる。. 基本構成ターボポンプのおもな構成要素は、インデューサ・羽根車・タービン動翼(turbine blade)およびこれらを結合するシャフト等の回転軸系ならびに回転軸系を支持し耐圧容器となるケーシング類および軸受や軸封シール等の機能部品である。. 設計技術ターボポンプの設計には、ポンプおよびタービンの流体力学、回転体およびケーシング類の構造強度解析や疲労寿命評価、回転軸系のロータダイナミクス等あらゆる工学的要素が必要である。これらは一義的には決定されず、さまざまな複合要素間の調整結果により最終設計が確定される。. 材料遠心力による高い平均応力を受けるポンプ側および遠心力とともに大きな熱衝撃を受けるタービン側の材料の選択は非常に重要である。現有技術では液体水素用の羽根車には比強度の高いチタン合金、タービン動翼には一方向凝固材や単結晶材も採用されている。. 軸受/軸封シール極低温用ターボポンプでは油潤滑は採用できず、軸受の冷却には推進薬自身が用いられ、潤滑はテフロン等の固体潤滑により行われることが多いため、軸受にとっては厳しい使用条件となっている。タービン側とポンプ側の分離のため軸封シールが必要となるが、高速回転用のものが必要であり同じく厳しい使用条件となる。. 軸系の危険速度ターボポンプの運転領域と回転軸系の危険速度(critical speed)は十分に離す必要がある。スロットリングが要求されるエンジンではターボポンプの運転領域は広い範囲にわたるため危険速度位置を避けることが難しくなる。液体水素用のターボポンプでは高速回転のため、運転領域は一次危険速度のみならず高次の危険速度を超える場合もある。軸系の剛性を大きくし、高次の危険速度を避けることが振動問題を防ぐために好ましいが、軸受の限界周速やばね常数その他との兼ね合いで最終的な危険速度位置が決定される。. 流体力の釣合せターボポンプ内部には流体の圧力分布により、半径方向推力および軸方向推力が発生する。これらの流体力を釣り合わせ、軸受に過大な荷重が負荷されないようにする必要がある。軸方向推力の釣合せには、バランスホール方式やバランスピストン(balance piston)方式が採用されている。. 代表的なターボポンプ例ターボポンプの代表的な例として図4・47にLE-7A 液体酸素ターボポンプ断面図軸受冷却は液体水素(LH)で行い、潤滑のためにテフロンを含浸させたガラス繊維製の保持器を採用している。リフトオフシールは、起動前のポンプ予冷時には閉状態であり、タービン側への漏出を完全に防いでいるが、運転後は開となり、液体水素がディスク表面と内部を冷却する。ポンプ部のケーシング外表面には、エンジンを予冷する際ターボポンプ内部への熱侵入を抑え、液体水素予冷量を節約する目的で、断熱材が取付けられている。液体酸素(LOX)ターボポンプの回転数は約18300minであり、約230kg/sの液体酸素は0.7MPaから18MPaに昇圧され、そのうち約47kg/sはさらに約27MPaに昇圧されてプリバーナに送られる。残りの酸素は主噴射器に送られる。構造および材料はほぼ液体水素ターボポンプと同じであるが、二段羽根車の代わりにプリバーナ用羽根車が装着され、主羽根車とプリバーナ用羽根車は耐熱合金の精密鋳造品である。タービンの作動ガスは高温の水素過濃の燃焼ガスであるためポンプ内の液体酸素と分離する必要があり、軸封シールは三重の複雑な構造となっている。さらに中央部に位置する軸封シールにはヘリウムパージも併用して完全な分離を果たしている。h. バルブ類LE-5,LE-7とも、主弁、予冷弁、点火器弁等を装備している。LE-5の主弁は酸素・水素とも同構造であり、最も小型化できるベンチュリ方式をとっている。開閉はヘリウムガスによるニューマチック駆動とし、メインシールにはKEL-Fを採用している。ポペットおよびベローズはInconel718であるが、ケースはAl合金である。LE-7の主弁も酸素・水素とも同構造でありヘリウムガスのニューマチック駆動であるが、全開時の圧損が少なく、途中開度のコントロールが容易なボールバルブ(ball valve)を採用している。アクチュエータが二重になっており、バルブは全閉から全開まで四つの開度をとれる構造としている。LE-7では流量が大きくポンプ吐出し圧力が高いので、エンジンの始動・停止時の混合比を適性に保つために、バルブを途中開度で保持する必要があるためである。メインシールはKEL-Fであるが、開閉時の損傷を防止するために、3個のベローズに組込まれたシールをヘリウムガスで加圧して浮かせてから、ボールを回転させる構造としている。ボールおよびケースはInconel718であるが、アクチュエータはAl合金である。i. 制御系LE-7は各種バルブ駆動用のヘリウムガスをコントロールするための10個の電磁弁および点火器エキサイタスパークプラグ励起用の電源回路からなるニューマチックパッケージを装備している。これらはシーケンスボックスからの指令信号を受けて開閉を行い、エンジンを安全に始動・停止させる。シーケンス機能は点火器の着火、燃焼圧力の立上りおよびプリバーナガス温度の上昇を検知しながら、8ビットのマイクロプロセッサによって実現される。エンジン始動シーケンス(engine start sequence)に従って各バルブの動きを説明すると以下のようになる(図4・38のLE-7配管系統図参照)。エンジン始動に先立って、酸素・水素予冷弁を開けてポンプ部を完全に液温まで冷却しておく。次に予冷弁を閉じて、主点火器弁およびプリバーナ点火器弁を開け、スパークを飛ばして点火器に着火させ種火を作っておく。この状態で主液水弁を徐々に開けると、液体水素は燃焼器の中でガス化して、プリバーナを通ってターボポンプを駆動し始める。少し遅れて主液酸弁を半開にすると、主燃焼器はエクスパンダサイクルモードでの低圧燃焼を開始する。次にプリバーナ液酸弁を半開にしてプリバーナに着火すると、タービン駆動ガスは低温水素ガスから燃焼ガスに切り換わり、エンジンは急速にパワーアップしていく。この時点で主液酸弁とプリバーナ液酸弁を全開にすると、エンジンは二段燃焼サイクルモードで定格運転状態に到達する。エンジンの始動に要する時間はわずか5秒である。
