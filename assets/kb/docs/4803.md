# 4・3・1 吸着ガスの挙動と脱ガス処理

> **章**: 第4章 真空システム
> **節**: 4・3 極高・超高真空システム

分子流領域での真空排気は，ガス分子が真空ポンプ排気口に飛び
込んでくる頻度に依存する．あるガス分子に着目すると，壁面への
吸着と再脱離を繰返しながら，真空ポンプ排気口に飛び込んで排気
される．脱離して再吸着するまでの時間を，真空容器の大きさとガ
ス分子の平均速度から推定すると，数十msのオーダである．吸
着・脱離を何回繰返して排気されるかは，真空装置の大きさや構造
に依存するが，気相中に存在している時間は非常に短い．したがっ
て，排気時間は脱離の活性化エネルギーに大きく依存する．高真空
と超高真空の違いは，真空容器内壁面に吸着しているHO分子の
有無（多少）とみなせる．4･1･2･b項で述べたように，壁面に吸着
しているHO分子は，脱離の活性化エネルギーが92～100kJ/ mol
である．脱離の活性化エネルギー と，平均滞在時間（mean
residence time，表面に吸着したガスが再脱離するまでの平均時
間）τの間には，次式が成立する．
τ＝τexp (Ⅳ-4･16)
ここで，τは10 s程度の値で，吸着分子の垂直方向の振動周期
に対応し， ＝8.31J/K･molはガス定数である． とτの関係
を図Ⅳ-4･12 に示す．室温（300K）におけるHO分子の吸着時
間は ＝92～100kJ /molに対し17min～7hであり，排気される
までの時間は非常に長いことがわかる．平均滞在時間τは温度
に大きく依存する．ベーキング処理とは，壁面に吸着しているガス
分子を強制的に脱離させる処理であるが，吸着から再脱離するまで
の滞在時間を短縮することにある．ベーキング温度を200℃とする
とHO分子の平均滞在時間は，17min～7hであったものが1.4～
11msまで短縮される．ベーキング前後の排気特性を図Ⅳ-4･13に
示す ．
図 -4･13脱脂処理したSUS304のベーキング処理と排気特性
図Ⅳ-4･13のHO分圧の減少は，ベーキング処理により吸着の
滞在時間が短縮されたことに対応している．脱離の活性化エネルギ
ーを ＝100kJ/ molとし，20℃から150℃に昇温されることで，
HO分子の平均滞在時間がどれだけ短縮されるかを図Ⅳ-4･12から
見積もると，約5けた短縮される．図Ⅳ-4･13では，150℃のベー
キング処理によりHO分子のガス放出量は約5けた減少している．
HO分子が排気された後のおもな残留ガス成分は，HとCOで
ある．Hは真空容器材料内部に含まれている主要なガス成分であ
り，あらかじめ高温脱ガス処理を施すか，H拡散防止の被膜を表
面に形成するなどの方法で，低減することができる．
真空容器の加熱が困難な場合の脱ガス処理として，放電洗浄
が使用される．放電洗浄とは，真空容器内にArなどのガスを導入
してプラズマを作成し，真空容器壁面をイオンで照射することによ
り，吸着ガス分子を強制的に脱離させる手法である．
