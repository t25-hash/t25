# 4・3・5 推力方向制御システム

> **章**: 第4章 ロケット機関
> **節**: 4・31 )

a. ジンバル制御液体ロケットの推力方向制御は、エンジン燃焼室上部に取付けられたジンバル(gimbal)と呼ばれる自在継手を中心に、2本の直動アクチュエータにより、エンジン・ノズルの向きを変えることにより行われるのが一般的である。舵角は数度程度であり、アクチュエータには油圧方式と電動方式が用いられる。b. 油圧アクチュエータ方式徐々に電動アクチュエータ化されつつあるが、現時点でもほとんどの大型液体ロケットエンジンのジンバル制御は油圧アクチュエータ方式が採用されている。油圧アクチュエータ本体は、サーボバルブ、油圧シリンダピストン、位置検出センサ(ポテンショメータ、リニア可変差動トランス等)から構成されている。油圧アクチュエータに作動油を供給する油圧源には、油圧ポンプ方式による作動油循環方式(図4・49)とガス押し式(ブローダウン方式、図4・50)がある。液体ロケットエンジンの場合、エンジンのターボポンプに油圧ポンプを直結、あるいはエンジンからタップオフしたガスにより駆動する専用タービンで油圧ポンプを駆動する作動油循環方式が一般的であったが、油圧ポンプ駆動系、油圧ポンプ、アキュムレータ、リザーバ、熱交換器(オイルクーラ)等の多くの機器が必要な複雑なシステムとなっていた。一方、ガス押し式は、タンクに充てんした作動油をガスで加圧してアクチュエータに供給するというものであり、システムを構成する機器の数が大幅図4・49作動油循環方式4―157に削減でき、コスト的メリットが大きいが、作動油循環方式と異なり、作動油搭載量でロケット飛行中の可能操舵量が制限され、また作動油搭載量によってはシステム質量がやや増加する方向であるため、作動油の搭載量に適正なマージンをもたせつつ極力少なく抑える必要がある。液体ロケットエンジンでは、アリアンV ロケットの第一段、H-IIA ロケットの第一段でガス押し式が採用されている。c. 電動アクチュエータ方式パワーエレクトロニクス素子の大電力化、およびディジタルサーボ技術の進歩に伴い、近年、従来の油圧アクチュエータが電動サーボ駆動アクチュエータに置換えられつつある。電動アクチュエータは、サーボモータ、減速ギヤ、ボールスクリュー、位置検出センサ(ポテンショメータ、リニア可変差動トランス)、サーボアンプ、電源から構成される。古くはスペースシャトルのOMS(軌道操作システム)のエンジンのジンバル制御に適用されていたが、最近では諸外国を含めて主力ロケットの上段エンジンのジンバル制御はほとんど電動アクチュエータ方式になってきており、国内でもH-IIA ロケット第二段は電動アクチュエータ(図4・51)がすでに採用され、運用中である。またH-IIA ロケット第一段用も試作フェーズを完了しており、実機適用のめどを得ている。油圧アクチュエータ方式と比較した場合の電動アクチュエータ方式の長所は、(1) 機器の数が少なく、質量・コストの両面でメリットあり(2) 漏えい点検、清浄度管理が不要となり、点検・整備作業が簡素化でき、整備性が向上するなどが挙げられる。
