a.ジンバル制御 液体ロケットの推力方向制御は，エンジン
燃焼室上部に取付けられたジンバル（gimbal）と呼ばれる自在継
手を中心に，2本の直動アクチュエータにより，エンジン・ノズル
の向きを変えることにより行われるのが一般的である．舵角は数度
程度であり，アクチュエータには油圧方式と電動方式が用いられ
る．
b.油圧アクチュエータ方式 徐々に電動アクチュエータ化さ
れつつあるが，現時点でもほとんどの大型液体ロケットエンジンの
ジンバル制御は油圧アクチュエータ方式が採用されている．
油圧アクチュエータ本体は，サーボバルブ，油圧シリンダピスト
ン，位置検出センサ（ポテンショメータ，リニア可変差動トランス
等）から構成されている．
油圧アクチュエータに作動油を供給する油圧源には，油圧ポンプ
方式による作動油循環方式（図4･49）とガス押し式（ブローダウ
ン方式，図4･50）がある．液体ロケットエンジンの場合，エンジ
ンのターボポンプに油圧ポンプを直結，あるいはエンジンからタッ
プオフしたガスにより駆動する専用タービンで油圧ポンプを駆動す
る作動油循環方式が一般的であったが，油圧ポンプ駆動系，油圧ポ
ンプ，アキュムレータ，リザーバ，熱交換器（オイルクーラ）等の
多くの機器が必要な複雑なシステムとなっていた．一方，ガス押し
式は，タンクに充てんした作動油をガスで加圧してアクチュエータ
に供給するというものであり，システムを構成する機器の数が大幅

c .電動アクチュエータ方式 パワーエレクトロニクス素子の
大電力化，およびディジタルサーボ技術の進歩に伴い，近年，従来

電動アクチュエータは，サーボモータ，減速ギヤ，ボールスクリ
ュー，位置検出センサ（ポテンショメータ，リニア可変差動トラン
ス），サーボアンプ，電源から構成される．
古くはスペースシャトルのOMS（軌道操作システム）のエンジ
ンのジンバル制御に適用されていたが，最近では諸外国を含めて主
力ロケットの上段エンジンのジンバル制御はほとんど電動アクチュ
エータ方式になってきており，国内でもH-ⅡAロケット第二段は
電動アクチュエータ（図4･51）がすでに採用され，運用中である．
またH-ⅡAロケット第一段用も試作フェーズを完了しており，実
機適用のめどを得ている．
油圧アクチュエータ方式と比較した場合の電動アクチュエータ方
式の長所は，
(１)機器の数が少なく，質量・コストの両面でメリットあり
(２)漏えい点検，清浄度管理が不要となり，点検・整備作業が簡
素化でき，整備性が向上する
などが挙げられる．
