# 4・3・6 液体ロケット推進系開発の進め方

a. 開発の基本思想液体ロケットエンジンは通常、原型エンジン、実機型エンジン、認定型エンジンの3フェーズを通して開発される。エンジンを構成するコンポーネントに新規技術が要求される場合は、これらのエンジンシステムの開発フェーズの前に、さらにコンポーネント基礎開発フェーズをおくこともある。液体ロケットエンジンに限らず、原子力発電システムや大型化学プラント等の複雑なシステムの開発に当たっては、概念設計、基礎設計、詳細設計のステップを踏むことがある。ただし、液体ロケットエンジンの開発では設計だけではなく、製造工程の検討、供試体の試作、コンポーネントの機能試験、エンジンシステム全体の燃焼図4・50 各エンジン開発フェーズの確認項目。 原型エンジン(prototype model engine)開発フェーズ原型エンジンとしては、まず大まかな設計仕様に基づく概念設計および基礎設計を実施したうえで、噴射器、燃焼室、ターボポンプ、主弁等の主要コンポーネントのみをぎ(艤)装したプロトタイプのエンジンを試作し、燃焼試験を行う。このフェーズではエンジンの耐久性よりは、まず最低限必要な性能、所定の機能が発揮できるかどうかを確認することを主眼とする。原型エンジンはプロトタイプとはいえ、このフェーズでエンジンの基本構造が定まると考えなければならない。ロケットエンジンには宿命的に軽量化構造が要求されるが、一度各コンポーネントの形状が決められるとその後の重量軽減は困難な作業となるので、主要コンポーネントについては原型エンジンの段階で要求重量を満足するめどを得ることも重要な確認項目である。. 実機型エンジン(engineering model engine)開発フェーズ次に、これらの燃焼試験データを評価・解析して、各コンポーネントに対する詳細要求仕様を設定する。これに基づいて詳細設計を実施し、実機型エンジンを試作する。実機型エンジンは主要コンポーネントはもちろんのこと、エンジンのジンバリング機構、残留推進薬のパージシステム、機体タンクの加圧ライン、機体との電表4・13 engine)開発フェーズこれらの実機型エンジンの試験データはさらに次の認定型エンジンの設計にフィードバックされる。認定型エンジンはフライトエンジンと全く同じコンフィギュレーションでなければならず、着火判定やフライト時のテレメータデータ取得用の各種センサまで装備したものになる。認定型エンジンの試験は各種性能の再現性やばらつき、寿命耐久性データ取得を中心に実施される。上段エンジンの場合はさらに高空燃焼試験設備を用いて、真空中での着火特性や高膨張ノズルの性能等が確認される。また場合によっては安全余裕を確認するために、オフリミット試験と称する規定寿命を上回る長秒時試験や規定作動範囲外での運転を行うこともある。最終的に認定型エンジンはフライトタイプの薄肉の機体タンクと組合わせた実機型タンクステージ燃焼試験(CFT : Captive Firing Test)に供されて、機体とエンジンの総合性能・機能が確認される。エンジンの燃焼試験ばかりではなく、エンジン全体での振動試験や音響試験も実施される。各種バルブ類、電装品類の機能部品は単体として、規定の寿命確認試験および振動試験等に供される。一連の認定試験完了後には、クリティカルなコンポーネントを切断検査して、有害な損傷がないことを確認することも重要である。. 審査会による評価各開発フェーズの節目には審査会を開催して、第三者の有識者が設計および供試体製造工程の妥当性、試験データの有効性等をチェックする。最終的には認定試験終了時の審査会によって、必要なデータがすべて取得され、所定の性能・機能・耐久性が満足されていることを確認して開発完了となる。c. 開発上の留意事項前述した三つの開発フェーズをとることが理想的ではあるが、現実にはスケジュール上の問題で各開発フェーズを完全に繰返すことが困難であることが多い。すなわち、一つ前のフェーズの試験データが十分取得されていないにもかかわらず、やむを得ず次の供試体の製作に着手しなければならない場合も出てくる。この場合はやみくもに製作を進めるのではなく、試験データをタイムリーに評価して、前のフェーズで発生した不具合が再発することのないように確実に設計、製造工程に反映させなければならない。また、いくつかの開発フェーズを繰返すと当然、開発コストもか項目年度初フライト⬆さむことになるので、杓子定規に開発フェーズを積み重ねるのではなく、場合に応じて柔軟に開発計画を設定する必要がある。例えば、既存エンジンの改良開発の場合は、原型フェーズを省略して開発スケジュールおよび開発コストを削減することも考えられる。実験計画法を有効活用し、燃焼試験回数および開発フェーズのトータルのエンジン台数をミニマム化することも、低コスト化のためには有効である。LE-7は世界の他のエンジンに比べると、少ない燃焼試験回数で高信頼度を達成しているよい例である。
