# 4・3・7 発射整備・打上げ

> **章**: 第4章 ロケット機関
> **節**: 4・31 )

工場での組立作業を完了した機体を、射場に持ち込んで機体の最終組立、機能点検等の整備作業、打上げ作業を実施する。ここでは、H-IIA ロケットを例に、射場にて実施される作業を紹介する。H-IIA ロケットでの発射整備作業スケジュール(固体補助ブースタのない形態での例)を図4・52に示す(H-IIA ロケットの場合、機体起立作業以降、射場整備作業期間として作業が実施される)。a. 打上げ射場ロケット打上げ射場には、打上げに使用するための各種設備が設けられている。例としてH-IIA ロケットで使用されている打上げ射場を図4・53に示す。ロケットの打上げに使用する射座(LP: launch pad)、機体の最終組立を実施する組立棟(VAB : vehicle assembly building)、機体を組立棟から射座まで移動して使用する移動発射台(ML: mobile launcher)、打上げ時に機体の管制を実施する発射管制棟(B/H : block house)、機体4―159に推進薬、高圧ガス、空調ガス等を供給するための設備等が射座まわりに配置されているほか、打上げ後の管制を含む、打上げ作業全体を管制する総合指令棟(RCC: range control center)が種子島宇宙センター内に配置されている。b. 機体組立作業工場にて各部位ごとに製作された機体は、射場で最終組立作業を実施する。組立作業は、H-IIA ロケットの場合、実際にロケットを打ち上げる射座から500mほど離れた、大型ロケット組立棟(VAB)で実施される。機体の組立作業は、図4・53 H-IIA ロケット射場概要[提供:(独)宇宙航空研究開発機構]図4・52 発射整備作業スケジュール(H-IIA ロケット固体補助ブースタなし形態)図4・54 自動カウントダウンシーケンス[提供:(独)宇宙航空研究開発機構]4―160内燃機関固体燃料を搭載したブースタ、導爆線等の火薬類のハンドリングを伴うため、安全確保のため人員の立入り規制等を適宜実施して作業が実施される。また、これらの作業の中で、機体と射場設備とのインタフェース部である、アンビリカル関連の結合作業も並行して実施される。c. 機能点検機体組立後、推進系、電気系等の各サブシステムの機能点検が実施される。一般的に、各組立単位での機能点検は工場での組立作業の中で実施されるため、射場における機能確認は、射場での組立作業に伴うインタフェース部の機能確認、各サブシステムの機能確認に主眼をおいて実施される。フライト時の重要機能については、最終組立状態でのend to endの確認も実施される。また、フライト中のシーケンス等に関する最終確認を目的として、実際の飛行時を模擬したフライトシミュレーションも実施される。d. 極低温点検(WDR)おもに極低温推進薬充てん時の機体・設備系全体確認を目的として、ロケットのタンクに推進薬を充てんし、実際にカウントダウンを実施する極低温点検が実施される。実際の打上げを模擬して実施されるため、リハーサルまたはWDR(wet dress rehearsal)とも呼ばれる。極低温点検で確認される機体・設備の機能に関しては、打上げ当日でも確認できることが多いため、機体・設備の成熟に伴い、打上げ期間・費用節約の観点からH-IIA ロケットでも削減する方向である(H-IIA ロケットの前身であるH-IIロケットでは、フライト4号機以降極低温点検が削除された)。e. 衛星系作業ロケット系の整備作業と並行して、衛星関連の機能確認、推進薬充てん作業が実施され、最終的にロケットのフェアリング内に格納される。近年、衛星を衛星フェアリング内に格納するところまで、ロケットとは別に作業を実施し、カウントダウン直前にフェアリングごと機体に結合する方式を採用するロケットが多くなってきている(アリアンIV、アリアンV,H-IIA 等)。f. カウントダウン衛星結合を実施後、打上げ数日前から秒読み作業(カウントダウン)を実施する。カウントダウンの中では、定められたタイムテーブルに従い、機体・設備系最終確認、火工品等のアーミング作業(結線作業)等が実施される。H-IIA ロケットの場合、打上げ当日に、機体を移動発射台ごと組立棟から射座に移動して、設備との結合作業を実施後、推進薬充てん、最終的な秒読み作業(ターミナルカウントダウン)が実施される。秒読み最終段階では、自動シーケンスに従い、機体・設備の健全性を確認しつつ最終的なアーミングが実施され打ち上げられる。自動カウントダウンシーケンス時の状況を図4・54に示す。
