# 4・1・1 航空機の分類

> **章**: 第4章 航空機
> **節**: 4・1 航空機一般

機に関して，それぞれの代表的な原理を以下に整理する．
a.揚力 揚力（lift）は固定翼，回転翼機が自重を支える力となるばかりではなく，プロペラやジェットエンジンファンの推力となる．
揚力を生む翼（wing）の設計基本は，今日でもNACAのデータに基づいている．
揚力は迎え角に比例して増加するが，大きな迎え角では失速する．
失速特性は，基本的には風洞実験によって確認される．
飛行機が，一定速度 で一定高度（密度ρ）を飛行する場合（巡航という），揚力 は自重 と釣合えばよいので，までになる．
回転翼を用いるヘリコプタも揚力を利用するため，重という関係が成り立つ．
ここで， は翼面積， は揚力係数である．
これより飛行速度は航空機（aircraft）は，大気中を飛行して人や物を移動させることができる乗り物の総称である．
最初に人を乗せて飛行できたのは，熱気球（1783年，フランス）であり，浮力を利用した軽航空機が誕生した．
気球は19世紀半ばに動力飛行船となり，20世紀初頭に硬式飛行船として実用化したが，重航空機の発達によって急速に衰退した．
揚力を利用して飛行する重航空機は，19世紀のケイリー（George Cayley，イギリス）やリリエンタール（OttoLilienthal，ドイツ）のグライダを経て，1903年，ライト兄弟属製機体は1915年に登場し，ジェットエンジンの開発により今日の発展が築かれ，超音速旅客機コンコルドが就航（1976年）する航空機に分類される．
