# 4・4・1 ハイブリッドロケット推進システム

> **章**: 第4章 ロケット機関
> **節**: 4・4 ハイブリッドロケット

ハイブリッドロケットは推進薬に液体と固体の組合せを用いるロケットで、液体推進薬を用いる液体ロケット、固体推進薬を用いる固体ロケットに対して、両者の混成(hybrid)であることからハイブリッド推進薬ロケットエンジン(hybrid propellant rocketengine)、略してハイブリッドロケットといわれる。液体酸化剤と固体燃料、および液体燃料と固体酸化剤の2通りの組合せが考えられるが、通常使用されるのは、前者の液体酸化剤と固体燃焼の組合せである。a. 特徴ハイブリッドロケットは、以下に示すような長所をもつ。(1) 安全:燃料と酸化剤が完全に分離されているため、事故等による爆発の危険性がない。(2) 推力制御・停止および再点火:液体酸化剤の流量を制御することにより、推力制御やオンオフ制御を行うことが可能である。(3) 低公害:燃焼ガスに塩酸や酸化アルミニウム等の汚染物質を含まない推進薬の組合せを選択することが容易である。図4・55 ハイブリッドロケットの概念図4・56 ポリエチレン-液体酸素およびポリエチレン-HAN の比推力(燃焼室圧力5MPa、開口比100、真空中)(4) ロバスト性:固体燃料にクラックが生じても不感である。(5) 低温度感度:推進薬温度が燃焼速度にほとんど影響を及ぼさないため、エンジン性能が打上げ環境温度の影響を受けない。(6) 安価:燃料、酸化剤ともに安価であり、供給系が1系統で済むため、エンジンが安価である。また、エンジンの安全性により安全対策の簡略化や市販レベルの部品の選択が可能であり、システム全体でも低コストとなる。一方、短所をまとめると以下のようになる。(1) 低推力:固体燃料の燃焼速度が低いため、大推力発生には不向きである。(2) 低充てん密度:固体燃料内部に液体酸化剤および燃焼ガスの流路を設ける必要があるため、固体燃料の充てん密度が低くなる。(3) 低燃焼効率:燃料と酸化剤の混合不十分および燃え残りの発生により、低燃焼効率になりやすい。(4) 酸燃比シフト:燃焼の進行により固体燃料の形状が変化するため、酸燃比が燃焼経過時間に従って変化し、比推力の低下を招く。b. 原理代表的なハイブリッドロケットの概念を図4・55に示す。液体酸化剤は加圧され、制御弁を経て噴射器から燃焼室に噴射される。燃焼室には固体燃料が充てんされ、点火器により燃焼が開始する。酸化剤および燃料ガスは固体燃料の軸方向に設けられたポートを流れ、ポート側壁が燃焼面となる。
