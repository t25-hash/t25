# 4・4・1 ハイブリッドロケット推進システム

> **章**: 第4章 ロケット機関
> **節**: 4・4 ハイブリッドロケット

ハイブリッドロケットは推進薬に液体と固体の組合せを用いるロ
ケットで，液体推進薬を用いる液体ロケット，固体推進薬を用いる
固体ロケットに対して，両者の混成（hybrid）であることからハ
イブリッド推進薬ロケットエンジン（hybrid propellant rocket
engine)，略してハイブリッドロケットといわれる．液体酸化剤と
固体燃料，および液体燃料と固体酸化剤の2通りの組合せが考えら
れるが，通常使用されるのは，前者の液体酸化剤と固体燃焼の組合
せである．
a.特徴 ハイブリッドロケットは，以下に示すような長所を
もつ．
(１)安全：燃料と酸化剤が完全に分離されているため，事故等に
よる爆発の危険性がない．
(２)推力制御・停止および再点火：液体酸化剤の流量を制御する
ことにより，推力制御やオンオフ制御を行うことが可能であ
る．
(３)低公害：燃焼ガスに塩酸や酸化アルミニウム等の汚染物質を
含まない推進薬の組合せを選択することが容易である．
(４)ロバスト性：固体燃料にクラックが生じても不感である．
(５)低温度感度：推進薬温度が燃焼速度にほとんど影響を及ぼさ
ないため，エンジン性能が打上げ環境温度の影響を受けな
い．
(６)安価：燃料，酸化剤ともに安価であり，供給系が1系統で済
むため，エンジンが安価である．また，エンジンの安全性に
より安全対策の簡略化や市販レベルの部品の選択が可能であ
り，システム全体でも低コストとなる．
一方，短所をまとめると以下のようになる．
(１)低推力：固体燃料の燃焼速度が低いため，大推力発生には不
向きである．
(２)低充てん密度：固体燃料内部に液体酸化剤および燃焼ガスの
流路を設ける必要があるため，固体燃料の充てん密度が低く
なる．
(３)低燃焼効率：燃料と酸化剤の混合不十分および燃え残りの発
生により，低燃焼効率になりやすい．
(４)酸燃比シフト：燃焼の進行により固体燃料の形状が変化する
ため，酸燃比が燃焼経過時間に従って変化し，比推力の低下
を招く．
b.原理 代表的なハイブリッドロケットの概念を図4･55に
示す．液体酸化剤は加圧され，制御弁を経て噴射器から燃焼室に噴
射される．燃焼室には固体燃料が充てんされ，点火器により燃焼が
開始する．酸化剤および燃料ガスは固体燃料の軸方向に設けられた
ポートを流れ，ポート側壁が燃焼面となる．
4・4・2ハイブリッドロケット用推進薬
液体酸化剤は通常の液体ロケットエンジンに使用される酸化剤と
同じものが使われる．固体燃料としては，ポリメチルメタクリラー
ト（PMMA)，ポリエチレン（PE），ポリウレタン等の一般的な高
分子材料や，固体推進薬の燃料としても使用される末端水酸基ポリ
ブタジエン（HTPB)，アジ化ポリマ（GAP）等も使われる．比推
力は燃料と酸化剤の組合せによりさまざまだが，一般的に固体ロケ
ットと同等以上の比推力を示し，液体酸素-HTPBの場合で，最大
理論比推力は370秒（燃焼室圧力5MPa，開口比100，真空中）と
なる．その他の代表的な例として，燃料にポリエチレン，酸化
剤に液体酸素（LOX）もしくはHANの95%水和物を用いた場合
の比推力を酸燃比の関数として図4･56に示す．燃焼室圧力5
MPa，開口比100，真空中での値である．
