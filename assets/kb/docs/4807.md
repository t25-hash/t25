# 4・4・2 ハイブリッドロケット用推進薬

> **章**: 第4章 ロケット機関
> **節**: 4・4 ハイブリッドロケット

液体酸化剤は通常の液体ロケットエンジンに使用される酸化剤と同じものが使われる。固体燃料としては、ポリメチルメタクリラート(PMMA)、ポリエチレン(PE)、ポリウレタン等の一般的な高分子材料や、固体推進薬の燃料としても使用される末端水酸基ポリブタジエン(HTPB)、アジ化ポリマ(GAP)等も使われる。比推力は燃料と酸化剤の組合せによりさまざまだが、一般的に固体ロケットと同等以上の比推力を示し、液体酸素-HTPB の95%水和物を用いた場合の比推力を酸燃比の関数として図4・56に示す。燃焼室圧力5MPa、開口比100、真空中での値である。固体ロケットでは、推進薬にアルミニウムの粉末を添加することにより燃焼ガスの温度を上げ、推進薬の比重を高くするとともに比推力を増加させることがよく行われる。ポリエチレン燃料と液体酸素の組合せを用いたハイブリッドロケットの場合で、アルミニウム添加が比推力に及ぼす影響を図4・57に示す。図4・57中の数字は、燃料に占めるアルミニウムの重量割合である。図4・57に示すように、燃料にアルミニウムを添加しても比推力は向上せず、むしろ最大比推力は平均分子量の増加によりやや低下する。これは、もともと断熱火炎温度が高いため、アルミニウムを添加しても、固体ロケットの場合ほどは燃焼ガス温度の上昇が比推力増加に寄与しないためである。a.
