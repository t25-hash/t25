# 4・4・3 性能予測

> **章**: 第4章 ロケット機関
> **節**: 4・4 ハイブリッドロケット

初期燃料グレイン形状および酸化剤流量履歴から、ロケットの推力および比推力の履歴を算出する。a. 解析解酸化剤流量が一定で、初期ポート断面が円の場合は以下のように解析的に解くことができる。まず、ポート内径の履歴を求める。圧力指数を0.75としたときのポート径および酸燃比の履歴を図焼時間でそれぞれ無次元化してある。燃焼中の酸燃比シフトに及ぼす圧力指数の影響を図4・63に示す。圧力指数が大きいほど、酸燃比シフトの幅が大きくなることがわかる。圧力指数が0.5の場合は、燃焼中の酸燃比は一定となる。b. 数値的な解法ポート形状が円ではない場合や、酸化剤流量が一定ではない場合も含めた、より一般的な性能予測は、数値計算により逐次的に行う。計算の流れをまとめたものを表4・14に示す。
