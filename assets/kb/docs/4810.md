# 4・5・1 全般

> **章**: 第4章 ロケット機関
> **節**: 4・5 制御用ロケット

本章は,ロケットおよび人工衛星の制御用ロケットシステム(control rocket system)について記述する.ここで取上げる制御用ロケットシステムとは,ロケットの推力飛行中またはコースティング中に,主エンジンとは独立して姿勢制御用に推力を発生させるもの,および人工衛星の軌道・位置・姿勢制御用に用いられる比較的小推力のロケットエンジンシステムであり,以下にその分類を示す.制御用ロケットロケット制御用バーニアエンジン(ロール制御用),ガスジェット装置(RCS)等人工衛星制御用軌道変換用[アポジキックエンジン(apogee kick engine)等]姿勢制御用(RCS 等)4―164内燃機関制御用ロケットの機体への搭載例を図4・66に例示する.これはH-IIA ロケット二段の制御用ガスジェット装置(RCS : reactioncontrol system)の例である.これらの用途に用いられるロケットシステムは,常温貯蔵性推薬(storable propellant)を使用し,推薬を加圧圧送するといった簡素なエンジンサイクルを用いているものが主流である.これは,この種のシステムがその用途から,大推力/高性能(高比推力)よりも,軽量/高信頼性を要求されることによる.単一液体推薬を使用する一液式システム(monopropellant system),二元推薬を用いる二液式システム(bipropellant system)がその中心である.その他の例としては,不活性ガスを単に加圧噴射するコールドガスジェットがあり,一部ロケットの姿勢制御に用いられている.構造は最も簡単であるが比推力が低いため,一般にはロケット上段系・衛星系には用いられない.また構造は複雑であるが,比推力の高い電気推進系も制御用ロケットとして実用化された例がある.電気推進系には推薬を電気加熱するタイプと電離した推薬を静電または電磁加速する方式があり,最近は後者が主流である.固体モータを衛星軌道変換に用いる例もあり,構造が簡素で大きなインパルス噴射が可能であるため,おもにアポジキック等に使用される.これら制御用ロケットを,使用される推薬により分類すると表
