# 5・2・4 重油の品質

> **章**: 第5章 舶用機関
> **節**: 5・2 船舶燃料

重油は用途に応じて動粘度、残留炭素、硫黄分および流動点などを調整して製品にする必要があり、また、挟雑物や水分などの混入を避ける必要もあることから、品質に十分な注意が払われている。舶用ディーゼル機関等の内燃機関用のA 重油の品質は、次の項目が重要となる。(1) 着火性特に小型のディーゼル機関は高回転のため、着火しやすく良好な燃焼を得るために、ある程度のセタン指数が必要となる。(2) 動粘度重油の分類のみならず、用途の面からも重要な性状である。例えば、燃料をスムーズに供給し安定した燃焼を起こさせるために、燃料を噴射する際の状態が重要になる。エンジンのタイプによって適正な動粘度が要望されている。(3) 蒸留性状燃焼過程において燃料の気化は重要な因子であり、蒸留性状により気化の度合いは異なる。例えば、沸点の高い留分が多い場合は、気化が進みにくいため、効率良く燃焼させるためには、燃料の加温などの対策が必要となる場合がある。一方、低沸点分が多い場合は燃焼場での燃料拡散が大きくなる。このため、空気などの供給が不十分になると、燃焼が不安定になることも考えられる。(4) 低温流動性軽油の場合と同様に、温度が降下するにつれてワックスの結晶が析出し、さらには析出量の増加に伴いワックス結晶の凝集が起こる。これらの凝集物が燃料フィルタに堆積し、閉そくに至る。この温度を低温フィルタ目づまり点(CFPP: cold filter pluggingpoint)という。この対策としては、貯蔵設備の加温や流動点降下剤の添加などが挙げられる。(5) 硫黄分燃料中の硫黄分は燃焼時にSOになり、燃焼ガスとして排出されるため、燃焼機器の腐食や環境規制の観点から重要視されている。船舶燃料用のC重油の品質としては、下記の項目が重要となる。(1) 着火性大型のディーゼル機関は低速回転のため、着火性はA 重油ほど問題にはならない。C重油のセタン価を測定することはできないため、密度と動粘度から計算で求めるCCAI(calculated carbonaromaticity index)が燃料の着火性を示す指数としてよく利用されている。(2) 動粘度流体力学上では粘性抵抗に当たり、ポンプ送油の際の大きな要素でもあるため、最も重要な性状である。C重油はA 重油よりも粘度が高いため、加熱により適当な動粘度に調整し使用する。一般に移送・貯蔵時は40~70°Cに加熱し、燃料噴射時にはさらに加熱している。(3) 硫黄分A 重油の場合と同様に燃焼の際にSOとなるが、その一部はSOとなり、それが燃焼ガスの露点以下の温度ではガス中の水分と反応し硫酸となる。硫酸は金属を著しく腐食させるので、硫酸生成を防止するために燃焼ガスの温度を露点以上に保つことが必要である。なお、IMO(International Marine Organization、国際海事機関)では「船舶からの大気汚染防止に関する規則」の中で、燃料油中の硫黄分の上限を4.5%としており、SO排出規制海域(バルト海およびIMOが指定する海域)ではさらに1.5%以下、もしくは排出ガス洗浄装置の設置を必要としている。(4) 固形物C重油の主成分は残油のため、原油に含まれている泥分、鉄錆などの金属分が濃縮される傾向にある。特に接触分解装置のFCC(fluid catalytic cracking)触媒は硬質のシリカアルミナ(SiO・AlO)でFCCの残油であるスラリー油に混入されていることが4―169ある。FCC触媒など硬い固形物を含む燃料がディーゼル機関に使用されると、ピストンやシリンダのしゅう動面を激しく摩耗することが報告されている。(5) 安定性貯蔵中に泥状、もしくは粉状の固形物が発生、沈殿することがある。これをスラッジと呼び、スラッジの生成しにくさを安定性と呼ぶ。スラッジはフィルタ目づまり、燃焼不良、エンジンの損傷などのトラブルの原因となる。スラッジの要因はさまざまであるが、アスファルティックスラッジ、ワックス析出によるスラッジ、水分および挟雑物によるスラッジ、微生物によるスラッジなどが挙げられる。(a) アスファルティックスラッジ:C重油にはアスファルトの主成分の一種であるアスファルテンやマルテン(油分)、レジン分(樹脂分)などが含まれている。アスファルテンはレジン分の働きによりマルテン中に安定に分散しているが、これらが他の油種と混和されるとき、混和性が悪いとアスファルテンが互いに凝集・成長し、アスファルティックスラッジが生成する。(b) ワックス析出によるスラッジ:重油には高沸点パラフィン系成分であるワックスが含まれている。温度低下によりワックスが析出、結晶化する。C重油では加温されているため問題は少ないが、冬季には特に補油後の流動性に注意する必要がある。(c) 水分および挟雑物によるスラッジ:重油は精製直後には水分は含まれないが、貯蔵や輸送を通して水分が混入される。水分は重油と水-重油エマルジョンを形成するが、水分量が多くなると水-重油エマルジョンスラッジが生成する。それ以外に、重油には無機物や有機物の挟雑物の混入によるスラッジも生成する。(d) 微生物によるスラッジ:適当な温度と水分およびエネルギー源がそろうと、微生物が繁殖しやすい条件となり、微生物によるスラッジが生成する。C重油は加温貯蔵されるため微生物の成長がしにくく、トラブルは少ないが、A 重油では常温で貯蔵されるため、条件がそろえば繁殖可能となり発生する例が多い。
