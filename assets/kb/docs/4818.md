b.循環型社会の構築と廃棄物処理 地球温暖化やダイオキシ
ン類問題等への関心が高まるにつれ生活様式と環境問題に対する議
論が高まり，2000年6月には循環型社会形成推進基本法が公布さ
れた．これはわが国の国民生活（ライフスタイル）や経済活動（ビ
ジネススタイル）を見直し，適量生産，適量消費を通じて廃棄物等
の発生を抑制し（リデュース），再使用（リユース），再生利用（リ
サイクル）を進め，最終的に排出される廃棄物については適正に処
理処分を行う，環境への負荷を低減した循環型社会を形成しようと
いうものである．
この中で廃棄物処理に求められるものは，再資源化を適正に行う
と同時に，廃棄物処理本来の目的である安定化，減容化，無害化を
行って，適正処理費用で最終処分量を最小にすることである．
一般廃棄物のうち可燃ごみについては，焼却・溶融等の熱処理操
作を加え，減容化，安定化，無害化を行うが，焼却・溶融処理で発
生する大量の熱エネルギーを回収し，発電や熱利用を行うこと（サ
ーマルリサイクル）が重要である．不燃ごみ，資源ごみについて
は，破砕・選別・回収の操作を行い，鉄，アルミニウム，紙，ガラ
ス等の資源はリサイクルルートに還元する．表5･16に一般廃棄物
処理技術の一覧を示す．
一方，産業廃棄物については品目別に回収されるため，リサイク
ルが容易である．できるだけリサイクルを行い，その他のものにつ
いては各品目に適した方法で中間処理を行い，最終処分する．
c.一般廃棄物焼却施設（municipal waste incineration facil-
ity) 一般廃棄物を対象とするごみ焼却施設とは，熱分解・燃
焼・溶融等の単位反応を単独または組合わせて適用することによ
り，ごみを高温酸化して容積を減じ，残渣または溶融固化物に変換
する施設をいう．ストーカ式燃焼装置，流動床燃焼装置，回転炉式
燃焼装置等を有するごみ焼却施設のほか，ガス化溶融施設，ガス化
改質施設を含む．
.施設規模 施設規模は，計画処理量，計画月変動係数，
施設の稼働体制，既存施設能力，施設補修時等における対応方法等
を勘案して決める．
炉形式は，原則として24時間連続稼働する連続運転式とし，安
定燃焼によるダイオキシン類の生成抑制，立上げ・立下げ時のダイ
オキシン類排出削減および余熱の有効利用を十分に行う．また，ご
み処理の広域化，発電の効率性，経済性等の観点から，施設規模は
ごみ処理量300t / d程度以上が望ましく，少なくとも一施設100t /
d程度以上とすることがごみ処理に係るダイオキシン類発生防止等
ガイドラインで求められている．ただし，離島等で規模が小さい場
合には，24時間連続稼働しない間欠運転式とすることが認められ
ている．
焼却炉の数は原則として2炉または3炉が一般的であるが，炉の
補修点検時の対応，経済性等に関する検討を十分に行い決定する．
炉補修点検時に既設炉等の他の施設にてごみの処理が可能な場合
は，1炉構成の施設も考えられる．なお，連続運転式焼却炉の1炉
当たりの処理規模については，最小20t / dから最大600t /dまでの
国内実績がある．
.焼却施設の構成 焼却施設は，受入れ供給設備，燃焼設
備，燃焼ガス冷却設備，排ガス処理設備，通風設備，灰出し設備，
焼却残渣溶融設備，余熱利用設備，給水設備，排水処理設備，電気
設備，計装設備のほか，建築物，建築設備，外構施設等から構成さ
れている．
