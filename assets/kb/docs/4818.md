# 5・3・1 概要

> **章**: 第5章 舶用機関
> **節**: 5・3 大型低速ディーゼル機関(2ストローク)

大型低速ディーゼル機関は、社会的インフラストラクチャを支える物流の中核である海上輸送船舶の主推進機関として、経済性と高い信頼性のみならず海洋汚染防止上、低公害性が求められる。トランクピストン型機関(trunk-piston engine)も過去には製作されていたが、低質燃料油(heavy fuel oil、重質燃料油)対応が難しいことから、現在は低質燃料油対応可能な、燃料経済性に優れたクロスヘッド型機関(crosshead engine)が製作されている。クロスヘッド型機関は燃焼時ピストンに作用する側圧をクロスヘッドで受ける構造となっている。基本構造はクランク軸を含めた台板、架構、シリンダの三つの主要ブロックからなり、タイボルト(tiebolt)により各ブロックが締結されている。
