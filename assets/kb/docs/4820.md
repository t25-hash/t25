.離着陸性能 運用される空港により要求が決められ，所
期の離着陸滑走路長での運用性を確認する．離陸時はエンジン推力
と揚抗特性が，着陸時は高揚力装置を作動させた状態での最大揚力
が性能を左右する．耐空性審査要領では離陸上昇こう配の規定があ
り，通常はエンジン推力を得にくい高温かつ高地での運用が標定と
.上昇性能 燃料消費率最小で巡航するために必要な上昇
限度まで上昇して巡航可能であることを確認する．また双発機で
は，山脈越えの運用に備えてエンジン片発停止状態における上昇高
度要求も満たす必要があり，エンジン選定において配慮を要する．
.速度性能 航空機の運用に従って最大巡航速度等の速度
性能に関わる要求がある．目標速度までは有害な抵抗増加を防ぐと
.その他 航空機の運用に従って固有の空力性能要求があ
り，個々に確認を必要とする．例えば，一般的に戦闘機では低速度
から高速度への加速時間に関わる要求や，旋回半径または旋回時間
e.空力特性 空力特性は航空機の安定性および操縦性（合わ
せて飛行性と呼ぶ）に深く関与し，飛行中および地上において良好
な飛行性を確保するため，各国ではそれぞれに定めた耐空性基準に
おいて規定を設けている．以下に代表的な飛行性の例を挙げるが，
これらの特性推算は，空力係数，機体重量，機体慣性能率の推算値
を用いて行われ，飛行試験により確認される．
.静安定特性 航空機が釣合い状態で飛行中に何らかの擾
乱を受けたときに，元の釣合い状態に戻ろうとする特性を有してい
る場合，航空機は静安定（static stability）を有しているという．
機種の上げ下げや速度の増減に関わる縦静安定特性と，機軸まわり
のバンク角（横揺れ）や水平面内の機首振り（偏揺れ）に関わる横
揺れおよび方向静安定特性とがある．特性は機体の形態（脚や高揚
図4･37超音速空力抵抗の低減（エリアルール)
力装置の状態）や速度，高度に左右されるため，一般には全運用飛
行領域において十分な静安定を有する必要がある．最近では電気式
操縦システム（FBW）による航空機も実用化されてきており，静
安定余裕を緩和することにより運動性を高めたり，尾翼等の安定板
面積を減少させて軽量化を図っている機体もある．
.動安定特性 航空機が釣合い状態で飛行中に擾乱を受け
たときに元の釣合いまで戻る力やモーメントが作用する場合を動安
定（dynamic stability）があるといい，復元力，慣性能率，減衰
力等に影響を受ける．動安定に関する要求は，操縦者の制御能力と
密接に結びついていて，人間の知覚および動作に不可避の時間遅れ

動安定特性として，急速な縦揺れ操舵により生じうる迎え角変動で
ある縦短周期モード，擾乱を受けた後に安定した速度状態に落ち着

.航続性能 搭載可能な有償重量（ペイロード，pay-
load）と航続距離（レンジ，range）との組合せで表される性能で
ペイロード・レンジ性能ともいう．機体の揚抗特性，機体重量，エ
ンジン性能の3要素に支配されるため，機体の低抵抗化，軽量化が
つねに求められる．
.失速特性 一般的には，迎え角の増大により発生する気
流のはく離に起因する現象であり，はく離の仕方により失速特性の
善し悪しが定まる．主翼後退角の小さい航空機では主翼内翼からの
気流のはく離に伴い水平尾翼による機首下げの傾向を生じ，主翼後
退角が大きい場合は，主翼外翼からの気流はく離に伴う機首上げの
傾向が生じる．特に翼端失速を生じる場合は補助翼利きの低下によ
り失速（stall ）からの回復操舵に困難をきたすことがあるため，
極力これを避けるよう設計する．失速に至る前にパイロットが明り
ょうに識別できる空力的または人工的警報を装備することが規定で
定められている．

た場合の操縦性に関わる非対称推力時操縦性，失速後のきりもみ状
態に関わるスピン特性等各種空力特性があり，航空機の用途により
必要な特性が規定により定められている．
