# 5・3・3 機関部システム概要

> **章**: 第5章 舶用機関
> **節**: 5・3 大型低速ディーゼル機関(2ストローク)

2ストローク低速ディーゼル機関の制御は従来,空気式入力や機械レバー式入力の機械式ガバナが多く使われていたが,一部を除き,最近はコンパクトな電気アクチュエータが導入され,応答の速い電子制御ガバナがほとんど使用されている.このため,機械式ガバナでは,電気式の船内制御装置と空気などの媒体への信号変換が必要であったが,電子ガバナでは直接,電気信号のやりとりが可能となり,船橋からの指令に対する機関の応答も迅速化される.さらに,電子制御機関の場合は極低速回転や低NOモードおよび低燃費モードでの運転など船舶のニーズに対応した運転が可能となるので,フレキシビリティの高い機関運転制御が可能となる.図5・41に従来カム式機関と増圧ピストン式電子制御機関との基本システム構成の比較を示し,図5・42にコモンレール式電子制御機関の概略システムを示す.
