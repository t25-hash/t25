# 5・3・4 高効率化

> **章**: 第5章 舶用機関
> **節**: 5・3 大型低速ディーゼル機関(2ストローク)

低質油が焚けて低燃費であることが2ストローク低速ディーゼル機関の最大の特徴である。現在、機関単独で熱効率が50%を超えている原動機はほかにない。図5・43に機関出力率の変遷を、図内爆発圧力の上昇による燃焼サイクルの改善、過給機の高効率化、燃料噴射系の最適化、掃気効率の高効率化などが実施されてきた。今後とも燃料経済性は大切であるが、船の安全な運航が大前提であるため、燃焼サイクル改善の有力手段であるシリンダ内爆発圧力は抑制せざるを得ない。特に海洋汚染防止上、排出ガス中のNO削減には燃焼温度抑制など従来の燃費低減手法とは反対の燃焼制御が図5・42コモンレール式電子制御機関の概略システム図5・43 機関出力率の変遷図5・44 燃料消費率の変遷必要なため、排出ガス規制を満足させつつ、熱効率の低下をいかに抑制していくかが今後の課題といえる。
