# 5・4・1 概要

> **章**: 第5章 舶用機関
> **節**: 5・4 4ストローク低速ディーゼル機関

4ストローク低速ディーゼル機関は、わが国において歴史的に内航船用、漁船用ディーゼル機関に始まり、古くから独特の発達をしてきた。特に他分野の4ストロークディーゼル機関と比べ、行程-内径比が大きいこと、すなわち長行程(long stroke)であることは海外において比較対象となる同種の機関は存在せず、オリジナル性が高い。機関の開発はその時代に建造される船舶の動向に合わせて行われ、現在に至るまで内航貨物船、内航コンテナ船、内航タンカー、内航フェリー、作業船、漁業練習船、調査船ならびに遠洋・近海漁船等の主推進機関(main propulsion engine)として幅広く使用され続けている。シリンダ径は240~500mm、定格出力450~3600kW、定格回転数200~450min程度の機関が実用化されている。正味平均有効圧力は1.9~2.2MPa、行程-内径比は約2.0、シリンダ内最大圧力は13~15MPa程度の機関が多い。図5・50にシリンダ径330mm、出力1600kW の機関断面を示す。構造は堅牢、簡素であることを特徴とするが、これは機関トラブル要因を最小限とし、容易に船内での保守整備ができることを意図した設計配慮によるものである。同一のシリンダ径を有する4ストローク中・高速ディーゼル機関との比較では、4ストローク低速ディーゼル機関のほうが行程は長く、機関回転数が低い。したがって、シリンダ内に噴射した燃料を時間をかけて燃焼させることができ、C重油等の低質燃料油(heavy fuel oil、重質燃料油)の利用に有利である。また、ピストンの膨張仕事が大きく本質的に熱効率に優れる。さらに低速機関図5・50 4ストローク低速ディーゼル機関の機関断面図(AX 33型)[(株)赤阪鐵工所](a) 直結型推進システム(b) 減速(逆転)機装備図5・51 4ストローク低速ディーゼル機関の推進システム事例は、減速装置なしで直接プロペラ軸に機関の動力を伝達する、簡素で経済的な直結型推進システムを構成できる[図5・51(a)].4ストローク低速ディーゼル機関の特徴を総括すると、長行程、高効率、低質燃料油に強い、直結型推進システムが可能ということであり、2ストローク大型ディーゼル機関と同一の指向性をもった4ストローク機関といえる。一方、同径の4ストローク中・高速ディーゼル機関と比較し長行程であるため、機関全高が高い分重量が増し、また回転数が低いので出力は低く抑えられる。
