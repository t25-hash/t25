# 5・5・1 概要

> **章**: 第5章 舶用機関
> **節**: 5・5 4ストローク中速ディーゼル(主機関,発電機)

a. 中速ディーゼル機関(medium speed diesel engine)の定義,用途ディーゼル機関は各種原動機の中で最も熱効率が高いため広い分野で使用されており,自動車以外の用途では現在1機当たり0.4~102960kW の出力範囲,7000~54minの機関回転数の範囲で機関が製造されている.中速ディーゼル機関を普遍的に定義することは困難であるが,現在生産されている機関要目を踏まえると4ストローク中速ディーゼル機関の基準は概略以下のようにまとめられる.シリンダ内径:≒200≦≦640mm機関回転数:1500≧>320min平均ピストン速度(mean piston speed):11≧욇>7.6m/sまた,中速ディーゼル機関の用途としては,舶用の主機や補機,陸用での常用発電・非常用発電,機械駆動,コージェネレーションシステムなどに必須な原動機として幅広い分野で使用されている.b. 代表的な機関の要目国内および海外での代表的な中速ディーゼル機関の主要目を表5・3に示す.最近の中速ディーゼル機関の仕様としては,正味平均有効圧力(BMEP: brake meaneffective pressure)が約2.5MPa,욇(平均ピストン速度)が約10m/s,윯율융(シリンダ内最高圧力)は約20MPa以上に達している.
