# 5・6・2 主要部の構造と特徴

> **章**: 第5章 舶用機関
> **節**: 5・6 4ストローク高速ディーゼル(主機関,発電機)

動力発生および伝達部品の強度に最も関係するのは最高燃焼圧力である.次に記す主要部品は,燃焼室で発生した力をいかに損失を小さくして機関出力端に伝え,動力に伴って発生した熱をいかに効率良く冷却するかに工夫を凝らしている.舶用の特殊性に関わりの深いものを特記する.a. ピストン,ピストンリング基本構造は他用途のものと何ら変わらないが,過給と無過給,出力率の高低により最適な材質・構造・形式のものが使い分けられている.ピストン燃焼室形状としては,ヘンシェル,トロイダル,リエントラント等が主流で,その他の特殊な形状としてはスキッシュリップ等がある.材質はアルミ合金,鋼,ダクタイル鋳鉄(FCD)等が一体型または組立型で使い分けられているが,一体型アルミ合金製ではニレジスト鋳鉄のリングトレーガ入りが多く(図5・116),そのほかに燃焼室部に繊維強化複合材を鋳込み熱き裂対策を図っているものもある.ピストンリングは圧力リングとオイルリングからなり,圧力リング2本とオイルリング1本もしくは圧力リング3本とオイルリング1本が一般的である.圧力リングのうちでトップリングは燃焼ガス圧力をシールするうえで特に重要であり,舶用ディーゼル機関においても矩形リング以外のキーストンリングが使われ,矩形リングでは合い口部を特殊形状にしてガスシール性を向上させているものもある.また,トップリングの位置では,ピストン頂面からの距離寸法をシリンダ径の10%程度として燃焼室むだ容積低減を図った,ハイトップリングと称するものもある.b. シリンダヘッド全シリンダ一体型と分割型の一般的な特徴比較は割愛するが,舶用高出力機関用としては吸排気弁間の機械加工の容易さによる冷却性の優位さ,剛性上の優位さ,シリーズでの部品共用による生産性とともに船内での整備性と経済性も考慮された結果,分割型が多くを占めている.吸気ポートは燃料高圧噴射を有する機関においても,燃焼促進のために若干の吸気スワールを発生する形状としている.吸排気弁の数は出力率によって変わり,各シリンダの吸排気弁が各1個の二弁式と,各2個の四弁式がほとんどであるが,特殊なものとして吸気弁が各2個で排気弁が各1個の三弁式がある.無過給で出力率の低い機関は二弁式,過給で出力率の高い機関は四弁式が一般的である.三弁式は中速ディーゼル機関で採用されていた例があるほか,無過給機関で吸気慣性過給と吸気スワールを併用しての出力率向上策として採用されている例がある.材質はねずみ鋳鉄が一般的だが,出力率の高い機関ではねずみ鋳鉄より強度で勝るダクタイル鋳鉄を使用している.材質選定では鋳造性,機械強度以外に耐熱性,熱伝導率を考慮して選定される.基本構造は他用途のものと変わらないが,長時間使用と信頼性を要する舶用として,特に潤滑油の燃料油による希釈防止に留意した構造のものが多い.c. injector)も高圧噴射の特性を生かして使われており(図5・118),自動車用で使われているコモンレール(common 潤滑系統基本的な経路,機能等は他用途のものと同じであるが,大きく異なるのはメンテナンス部品である.フィルタは近海用小型機関ではペーパフィルタを使用しているが,比較的大型機関の遠洋用は逆洗式ノッチワイヤ等のエレメント交換不要の形式のものを使用している.また,船内設備として遠心こし器等の大容量フィルタと併用されるのも特色である.潤滑油冷却器は海水冷却式が多い関係で多管式がほとんどであり,多板式も積層組立型または一体型が使われているが,一体型多板式は基本機関が自動車用の小型機関に多い.i. 冷却系統他用途のものと機能,経路が最も異なっているのが冷却系統であって,ごく一部の小型海水直接冷却式機関を除き海水間接冷却式がほとんどであり,海水と清水の二系統の冷却系統を有している.水ポンプは,自吸性に優れた回転ゴム式が小型機関の海水ポンプとして使われている以外は,海水ポンプも清水ポンプも遠心式が一般的である.熱交換器としては清水冷却器,潤滑油冷却器,空気冷却器があり,燃料油冷却器を装備する機関もある.各冷却器は温度が低く冷却に有利な海水で冷却しているが,一体型多板式潤滑油冷却器は清水で冷却する機関が多い.また,各冷却器は機関装備が一般的であるが,船内設備として機関に装備しない場合もある.配管を含む海水経路部材の防食法としては,銅系材料等の使用,防錆塗装を行うとともに,防食材として純鉄,アルミニウム,亜鉛等を被防食材の材質により使い分けている.シリンダヘッド,ライナ,シリンダブロック等の水室は清水で冷却されており,海水直接冷却式に比べ電気腐食やカルシウム,塩類等の堆積に伴う局部的冷却阻害に基づく熱応力の増大,あるいはライナ等の過冷却に基づく硫酸腐食等が除去され,機関の無開放運転時間の延長に寄与している.また,清水による高温冷却化は燃料消費率の低減をもたらしている.一般的な舶用ディーゼル機関は冷却系統部品をすべて機関に装備し,おのおの独立したシステムとなっているが,舶用ディーゼル機関であることの特色を生かし船内の設備の一部として冷却システムを構成し,複数機関での冷却系統部品の兼用,清水二系統システムとするなどのいわゆるセントラルクーリングシステム(centralcooling 過給システム過給方式としては静圧か動圧,一段式か二段もしくは多段式があり,過給機の形式としては排気ターボ過給機か機械過給機,水冷式か空冷式,自己給油式か外部給油式があって,用途,出力率等の条件により使い分けられている.一般的に空気冷却器と併用されるが,過給機のうちで圧倒的に多く使われている排気ターボ過給機の圧力比は3~4のレベルに達しており,機関の高出力化および低燃料消費率化が過給機の高性能化に負うところが大きいのは他用途の場合と同じである.過給システムとしては機関全体の小型化を考慮したうえで過給度向上のために二段過給にしたり,過給効率を上げるために排気脈動圧を効率良く利用する方策等に工夫を凝らしている.なかでもMTU 始動装置電気式と空気式が用いられており,電気式は他用途のものと同様に電気スタータ,リングギヤ,オルタネータ,バッテリ等から成り立っている.空気式にはエアモータ式と直入れ式がある.エアモータ式はバッテリが高価でメンテナンスも必要なことから,圧縮空気を用いてエアモータのピニオンでフライホイールのリングギヤを回転させることにより機関を起動している.直入れ式は始動弁,分配弁,操縦弁,そく止弁等から成り立っているが,シリンダヘッドに取付けられている始動弁部が,燃焼室のむだ容積となって燃焼上の不利になることもあり,エアモータ式が増える傾向にある.m. 減速逆転機基本構造は減速機部とクラッチ部からなり,機関の動力を船の推進器であるプロペラに回転数を減速して伝えて4―200内燃機関いる.クラッチ部の形式としては機械式か油圧式であり,減速機部の形式としては平行歯車式がほとんどで,特殊な形式として遊星歯車式や傘歯車式がある.機械式クラッチは小型機関用に限られ,ほとんどが油圧によりかん脱を行う摩擦多板式である.機関との直結に使われる弾性継手は,機関のトルク変動を緩衝して歯車を保護するためのもので,舶用ディーゼル機関としてはゴム継手かばね継手がほとんどである.漁船からの基本要求は船速とえい力であるが,船の軽量,高速化と機関の高速化に伴い,単一の減速比では,満足させることができない.この場合は,低船速・高えい力を要するときは高減速比で,高船速・低えい力を要するときは低減速比でといった使い方のできる二速式が用いられており,結果として機関の最もよい性能を引き出す効果もある.また,漁船の操業パターンの複雑化,高度化は定点操業や微速航行の要求となり,クラッチの油圧を変えることにより任意に摩擦板を滑らせ,機関回転数と関係なくプロペラ回転数を変えることのできる,トローリング装置なるものも使われている.緊急ボルトは,油圧系統に異常があり摩擦板をかん入できない状態になっても,強制的に摩擦板をかん入状態にするためのものであるが,これも船級規則で要求されているフェイルセイフ機構のうちの一つであり,舶用機関の信頼性確保に必須のものである.
