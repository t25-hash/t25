# 5・6・3 高効率化、排気対策、振動騒音対策

> **章**: 第5章 舶用機関
> **節**: 5・6 4ストローク高速ディーゼル(主機関、発電機)

高速舶用ディーゼル機関の高効率化と排気エミッション対策技術を図5・123に示し、代表的な適用技術について以下に述べる。a. 高効率化。 燃焼系の改善(1) 直噴化シリンダ径200mm以下の機関は、1970年代中ごろまでは予燃焼室式が主流であったが、オイルショックを契機にシリンダ径の大きいものから燃料消費率の優れた直接噴射式へ逐次移行し、現在ではシリンダ径70mmクラスまで直噴化されている。燃料消費率は予燃焼室式に比べてサイクル効率の良い直接噴射式が過給機関で約10%、無過給機関で約15%以上低減する。(2) 燃焼系諸元の最適化小型直噴機関では噴霧の燃焼室壁面への衝突拡散とスワール、スキッシュなどの空気流動により混合気形成をしている。燃焼室形状はトロイダル型が主流で、シリンダ径が小さくなるに従い開口比(燃焼室直径/ピストン呼び径)を小さくして燃焼室内流動を強化噴射系の改善噴射時期の最適化タイマ進角量の最適化燃料噴射時期遅延ロングストローク化燃焼室容積比の増大圧縮比の増大高効率化(低燃費化)燃焼最高圧力の増大圧縮比、爆発度、過給圧増大等圧燃焼化冷却水温度の最適化高恒温清水冷却充てん効率の増大四弁化、高過給化、給気冷却吸排気系の改善過給機の高圧力比・高効率化コンプレッサ・タービン効率の改善ガス交換損失の低減弁開閉時期、排気マニホルド構成の最適化摩擦損失の低減各部しゅう動抵抗の低減ピストン速度の最適化、ローラタペットの採用潤滑油粘度の最適化潤滑油温度の最適化、マルチオイルの採用図5・123 高効率化と排気エミッション対策技術し、燃焼改善を図っている。燃焼性能に影響する因子は上死点すきま、圧縮比、容積比、開口比、スワール比、噴口数、噴口径、噴射角度、噴射圧、噴射期間など多々あり、それぞれが複雑に関連しているので、実験により製造ばらつきも考慮して最適諸元が決められる。. 噴射系の改善噴射ポンプの送油率増大による高圧化だけでも燃料消費率の改善は図れるが、噴口径を縮小し高圧化と同時に噴射期間の短縮と燃焼室形状も最適化すれば、より大きな効果が得られる。. 吸排気系の改善(1) 充てん効率(charging efficiency)の増大充てん効率は吸気弁径の拡大や三弁化、四弁化、吸気慣性効果の利用、高過給化、給気冷却器などにより増大を図り高出力化に対応している。高過給化と給気冷却器の大容量化による燃料消費率の改善効果は大きく、無過給機関に比べて10%以上低減できる。これは燃焼改善効果と機械効率向上などによる。(2) ガス交換損失の低減高過給機関では高負荷時の排気押出し抵抗が大きいので、排気弁の開き時期を進めるか、開弁速度を上げて排気押出し損失を低減している。どちらも排気弁開き時の動弁系各部の接触面圧が増加するので、耐久性を損なわないように配慮が必要である。. 摩擦損失(friction loss)の低減摩擦損失は、機関回転数もしくはピストン速度が速くなるにつれて増え、その割合はピストンリング系が最も大きく、クランク軸系、動弁系噴射ポンプ、補機駆動系の順である。おもな低減策としてはピストンリング本数や張力の低減、クランク軸径・メタル幅の最小最適化、ローラタペットの採用、潤滑油温度の高温化、潤滑油のマルチ化などであるが、いずれも摩耗や潤滑油消費量を増大させる傾向にあるので採用には十分な注意が必要である。b. 排気対策。 白煙(white smoke)、青煙(blue smoke)小型直噴機関では冬季やセタン価の低い燃料を使用した場合、始動直後やアイドリング時の燃焼が悪く白煙や青煙を発生しやすい。対策は圧縮比アップや噴射時期の最適化による着火遅れの短縮、開口比の拡大による燃焼室壁面への付着燃料の低減、小噴口径化による噴霧の微粒化、サック容積縮小による後だれ低減などである。始動後、冷却水温度が上がるまでの間、電気式ヒータで吸気を加熱したり、減筒運転ができる機関もある。要素方策おもな適用技術排気エミッション対策の付加技術直噴化トロイダル燃焼室リエントラント、スキッシュリップ燃焼系の改善燃焼室形状の最適化開口比・深さの最適化、むだ容積低減ハイトップリング化スワール比の最適化吸気弁径・弁シート・ポート形状の最適化低スワール化燃料噴射弁の最適化噴口数・径・角度の最適化、サック容積の低減小噴口径・多噴口化高圧・短期噴射化高圧ポンプ、ユニットインジェクタの採用電子制御燃料噴射システム。 加速時黒煙過給機関においては、低速トルクが低いため、加速時の黒煙発生や加速性の悪化が問題になる。その改善にはブーストコンペンセータによる過剰燃料の抑制やツインターボによる応答性の改善、ウェイストゲートや機械過給とターボ過給機の併用あるいはシーケンシャルターボなどで積極的に空気量を増加させたり、自動切換え二速式クラッチでプロペラ回転数を切換え制御するなど、種々の方法が適用されている。. 排気エミッション(1) IMO規制と対策IMO(国際海事機関)では130kW を超える舶用主機・補機に対するNO規制が2005年5月に発効され、段階的に強化の方向にある。現在の規制に対するおもな対策は給気温度の低減や燃料噴射時期遅延(タイミングリタード)、燃焼室-噴射系の最適化であるが、給気温度低減以外は燃料消費率低減とトレードオフ関係にあるものが多い。タイミングリタードは過給圧や圧縮比の上昇と併用し、等圧燃焼に近づけたほうが燃料消費率悪化を抑制できてNO低減に有利である。(2) アメリカEPA、ヨーロッパ規制と対策アメリカではEPA(Environmental Protection Agency、アメリカ環境保護局)によるマリン規制があり、ヨーロッパではRCD(Recreational Craft Directive、プレジャボート搭載エンジンの規制)、NRMM(Non-Road Mobile Machinery、内陸水路航行船舶搭載エンジンの規制)などがある。これらはNOだけでなく、PM やHC,COも規制しており、段階的に規制が強化されつつある。これらの規制対応にはさらなるタイミングリタードやリエントラント燃焼室、電子制御燃料噴射システムなどで、将来的には後処理装置も視野に入る。c. 振動(noise)騒音(vibration)対策振動騒音対策は燃焼起振力の低減と構造対策で行う。燃焼起振力は圧力上昇率を低くすることで低減され、排気エミッション対策などと併せて最適化する。構造上のおもな対策としては、(1) 往復運動部分の慣性力低減:二次バランサ(四気筒機関)、大型バランサ(2) 機関の剛性向上による放射音低減:ライナ中間支持、補助ボルト付き主軸受キャップ(3) 機関の歯車音低減:ねじり振動ダンパ、ギヤトレーンのフライホイール側配置(4) 減速逆転機の歯車音(ガラ音)低減:クラッチを微小スリップさせる油圧制御機構(5) 排気音の低減:排気消音器の大型化、アクティブ消音器(6) 船体への機関振動の伝達低減:防振支持などが適用されている。
