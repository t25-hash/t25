# 5・7・2 主要部の構造と特徴:2ストローク船外機機関

> **章**: 第5章 舶用機関
> **節**: 5・127 )。

a. 機関潤滑方式2ストローク船外機機関の潤滑方式は、潤滑オイルを燃料のガソリンにあらかじめ混合して使用する混合潤滑方式と、潤滑オイルを燃料とは個別に使用する分離潤滑方式に分けられる。混合潤滑方式は機関の潤滑構造を簡素化することができ、これによって高い耐久性が得られることから、十分な整備点検が望めない開発途上国などの漁業や交通手段として使用される船外機に採用されることが多い。混合潤滑における混合比はガソリン対オイルの容積比で50:1が一般的である。分離潤滑方式はオイル消費量の削減を目的に1970年代に開発が進められ現在に至っている。オイルの供給量は機関の回転数と負荷によって可変する機構が採用されている(図5・130)。b. 燃料供給方式船外機用2ストローク機関は構造が簡素であり軽量で高出力を得ることができる。しかし掃気行程における燃料の吹抜けから燃料消費率は4ストロークの燃料消費率に比較し劣るものとなっている。近年、これらを改善する技術手段の一つとして燃焼室に直接燃料を噴射する機構を採用する傾向にある。燃料を燃焼室に噴射する方法として、燃料を加圧したうえで圧縮空気で噴射するエアアシスト噴射方式と、燃料を7MPa程度の高圧で直接図5・130 分離潤滑方式図5・131 燃料噴射方式による燃料消費率の比較図5・132 高出力船外機のクランク軸構造噴射する高圧燃料噴射方式が採用されている。前者のエアアシスト噴射方式では成層燃焼を行い、特に低速運転時の燃料消費率の改善を実現している。高圧燃料噴射方式では特に高速運転時の燃料消費率の改善を実現している(図5・131)。c. エンジン構造の特徴2ストローク船外機機関の種類は、小出力は1.47kW(50mL)から高出力の184kW(3.1L)までが生産されている。クランク軸の方式は小出力船外機では組立方式が、高出力船外機では各気筒が一体になったクランク軸が採用されており、気筒間の混合気シールのためにクランク軸外周にピストンリングに似たシール用のリングが採用されている(図5・132)。また一体型のクランク軸用の連接棒は大端部と小端部にニードルベアリングを採用しており、大端部はコンロッドキャップ部を自然割りにて製造し、真円度を確保している。シリンダの製法は小出力船外機においてはアルミダイカスト工法により製造され、高出力船外機では低圧アルミ鋳造を用いて製造されている。2ストローク機関の高出力化技術としては排気パルスを利用するのが一般的であり、高出力船外機では排気通路をシリンダ内に形成し他気筒のブローダウン圧力を利用して掃気時に素通りした混合気を押し戻す工夫をしている。一般的な船外機用機関ではクランク軸上部にフライホイールを備えており低速時のスムーズな運転を可能にしている。またフライホイールの内部には発電用のコイルと点火装置用のコイルが設置されている。ほとんどの船外機用機関の点火方式はCDI方式を採用し、連続低速運転における点火プラグ電極の汚損耐力を向上させている(図5・133)。図5・133 船外機用機関の点火システム図5・134 船外機用4ストローク機関の弁機構と潤滑油の循環機能
