# 4・4・1 宇宙からの地球観測の展開

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・4 高精度地球観測衛星の姿勢制御

宇宙からの地球観測では，1990年代後半から量的な拡大と質的
な多様化が進展している ．地球観測衛星（earth observation
satellite）が続々と打ち上げられる一方，観測センサやミッション
の多様化が進んだ．また地球観測の各国への浸透が進み，さらに民
間へと拡大した．最近では，地球科学分野でアメリカのTerra
（1999年）やAqua（2002年），ヨーロッパのEnvisat （2002年）
が，実利用分野ではアメリカのLandsat7（1999年），フランスの
SPOT 5（2002年），民間のIkonos -2（1999年）やQuickBird-
2（2001年）が打ち上げられ，その画像が配布されている．
こうした中で観測衛星の高空間分解能化が姿勢軌道制御に影響を
与えている．空間分解能は長い間10～30mにとどまっていたが，
1990年代に1～5mの分解能をもつシステムが次々と発表され，そ
の運用が開始されつつある．高分解能データを利用する場合に重要
となるのが，取得画像の地表位置決定精度，画像の幾何精度である
が，これらが姿勢制御に厳しい要求を課す．本書では日本の高分解
能衛星ALOSを取り上げ，その例を用いて高精度地球観測衛星の
姿勢制御について紹介する．
