# 5・7・4 推進システム構造

> **章**: 第5章 舶用機関
> **節**: 5・127 )。

機関の動力は、クランク軸からドライブ軸を介してドライブギヤに伝達される。前進ギヤと後進ギヤは、常時ドライブギヤとかみあっているため、プロペラ軸上で互いに反対方向に空転している。シフト操作によって、クラッチドックが前進ギヤ、または後進ギヤにかみあい、プロペラ軸が右回転または左回転する。プロペラが各方向に回転し推力を得ることで、船体が前進、後進する。a. ギヤケースユニット(図5・135)ギヤケースユニットは、おもにドライブ軸、ギヤ、プロペラ軸、シフト機構、ウォータポンプから構成されている。このユニットにプロペラが取付けられる。ギヤケース内部は、ギヤオイルが充てんされている。そのオイルは、ドライブ軸に取付けられたオイルポンプ機能をもつ部品によって循環される。ドライブ軸やギヤは、複数のベアリングによって支持される。また、ギヤオイルが充てんされている部分以外の空間は、エンジ4―204内燃機関ン側から排出される排気通路になっており、プロペラのボス部から水中排気される。ギヤケースの外観形状は、水中抵抗を軽減する形状になっている。ギヤケースに取付けられたプロペラが水面近くで回転すると、水面から空気を吸い込んで船速が急激に減速したりプロペラ回転数が急激に上昇したりする場合がある。それを抑えるためにアンチベンチレーションプレートが取付けられている。また、走行時にギヤケースによって発生する水飛まつを抑えるために、スプラッシュプレートが取付けられている。船外機は、走航中にどちらかに舵をとられるような現象が起こる。これは、プロペラが回転することによって生じる左右不均衡な力である。この力を軽減し操舵しやすくするために、トリムタブが取付けられる。この力は、船外機のトリム角度によって傾向が変化するので、最も使用頻度の高いトリム状態で調整することが望まれる。b. ドライブ軸ドライブ軸は、機関の動力をギヤへ伝達する中間経路で、水中に触れる部分は耐食性に優れたステンレス材、ギヤケース内では鋼材を使用しており、異材質を摩擦溶接によって結合している。クランク軸とドライブギヤは、スプラインかん合で結合している。特にドライブギヤについては、テーパとスプラインかん合によって結合している。ドライブ軸は常時回転していることから、ウォータポンプの駆動やギヤケース内のオイル潤滑機構の役目として利用されている。c. ギヤドライブギヤおよび前進ギヤと後進ギヤで構成され、低回転から高回転域まで対応する。変速機構はなく、船外機特有の高負荷高速連続運転のような過酷な使用条件に耐えるため、高強度材を使用し、まがりばかさ歯車を採用する場合が多い。前進ギヤや後進ギヤには、クラッチドックとかみあうつめ部があり、シフト操作時にクラッチドックのつめ部とかみあう。d. プロペラ軸ドライブ軸と同様に、ステンレスと鋼材との摩擦溶接で構成されている。クラッチドックやプロペラをかん合するためのスプラインが加工されている。また、プロペラの推力を受けるテーパ部がある。e. シフト機構(図5・136)船外機は、前進と後進に切り換えるシフト機構をもち、シフトレバーを倒すと、シフトレバーからリンクされた各部品が作動し、最終的にクラッチドックが前進ギヤまたは後進ギヤのつめ部とかみあう。小型機種の中には、前進構造しかなく、後進時には、船外機自身を180°回転させて走行するものもある。クラッチドックに直結された部品が、シフトカムのプロフィルに沿って移動すると、それに伴ってクラッチドックも移動するために、中立、前進、後進の位置が決定される。シフトカムの種類には、シフトカムが上下に移動するタイプと、シフトカムがシフト軸を中心に回転するタイプがある。一般的に、回転シフトタイプのほうがシフトフィーリングが良いとされ、大型機種で採用されている例が多い。f. ウォータポンプ(図5・137)ドライブ軸にキー結合されたインペラが回転すると、ポンプケースとインペラによってできる空間容積が変化し、水がエンジン側へ圧送される容積型ポンプである。吸水口は、ギヤケースに設置されている。1:シフトカム、2:前進ギヤ、3:クラッチドック、4:後進ギヤ図5・136 シフト機構(前進、中立、後進の状態)g. プロペラプロペラは、エンジン動力を推進力に変換する重要な役割がある。プロペラ選択は、走航性能に多大な影響を与え、振動や燃費にも影響する。船外機の大半のプロペラは、ハブ内部に排気通路となる排気口がある。材質は、アルミニウムやステンレスのほかに樹脂製のものもある。プロペラには、プロペラダンパが圧入されており、水中で暗岩や浅瀬に衝突したときにブッシュが滑るため、ギヤなどの部品への衝撃を緩和する緩衝機構となる。翼枚数を増やすとトルクの伝達効率が良くなるため振動や騒音が低減し、燃費や加速性がよくなるが、最高速が低下する。バランスを考慮した中で、三翼か四翼プロペラが最も使用されている。h. カウンタローテーション外洋向けの大型艇では、大型船外機を2機装備する場合が多く、その場合、2機のプロペラトルクを相殺して、艇の水平姿勢と直進性を確保するためにカウンタローテーション仕様のギヤケースが提供される。その構造は、レギュラローテーション仕様のシフト機構を反転させて、後進ギヤを前進ギヤとして使用し、また、前進ギヤを後進ギヤとして使用する。プロペラ軸回転方向がレギュラローテーション仕様と逆回転になるため、プロペラは、翼の向きが逆ひねりになっている専用プロペラが必要になる。
