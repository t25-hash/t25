# 5・7・5 懸架システム構造

> **章**: 第5章 舶用機関
> **節**: 5・127 ).

a. 懸架システム(図5・138)船外機のマウント構造は,クランプブラケット,スイベルブラケット,ステアリングブラケット,マウントラバー,ドライブシャフトハウジングにより構成され,クランプハンドル(小型船外機)またはボルトとナットにてボートの船尾に固定する.b. 操船ステアリングブラケットに結合されたテイラーハンドルまたはリモートコントロールケーブルにより操舵を行う.船外機の場合は船に舵はなく,船外機本体の向きを左右に回動させることでプロペラの推進力の向きを変えて旋回する.c. マウントラバー一般的に上下各2個のマウントラバー1:インペラ,2:ポンプケース図5・137 ウォータポンプ1:スイベルブラケット,2:クランプブラケット,3:チルトピン,4:クランプハンドル,5:マウントラバー,6:ステアリングブラケット,7:ドライブシャフトハウジング,8:パワートリム・チルト装置,8-1:トリムロッド,8-2:チルトロッド,8-3:モータ/オイルポンプ,8-4:オイルタンク,9:テイラーハンドル図5・138 懸架システム(筒形防振ゴム)によりステアリングブラケットとドライブシャフトハウジングを結合し船外機本体を支えている.船外機の場合,機関の防振とともにプロペラの推力も伝達しなければならず,緩衝特性の良いことと強さの両方が要求される.d. トリム角度の選定航走時のボートの姿勢が最適となるトリム角度をチルトピンまたはパワートリム・チルト装置により調整する.e. チルトピン小型船外機においてはチルトピン取付穴位置を変更して最適なトリム角度に調整する.f. パワートリム・チルト装置(図5・124)大型船外機においてはパワートリム・チルト装置を採用している.パワートリム・チルト装置の下部はクランプブラケットに固定され,チルトロッドがスイベルブラケットに結合されている.バッテリを電源とするモータによってオイルポンプを駆動し,オイルポンプにて発生した油圧をシリンダに導きトリムおよびチルトロッドを上下させて船外機をトリム(チルト)アップまたはダウンする.通常航行時はトリムロッドがスイベルブラケットを支えて推力を保持し,浅瀬航行時や船外機をチルトアップするときはチルトロッドがスイベルブラケットを保持する.航行中,船外機が障害物に衝突した場合,船外機およびボートの破損を防止するためにショックアブソーババルブを備えている.また衝突後は,衝突前のトリム位置となるようにフリーピストン機構を備えている.
