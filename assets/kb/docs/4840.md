# 5・8・3 高効率化、排気対策、振動騒音対策

> **章**: 第5章 舶用機関
> **節**: 5・8 舶用ガスタービン

a. 高効率化市場ニーズに合わせ、既存機種の改良により舶用ガスタービンの大出力化や高効率化が進められている一方、最新の航空用高バイパス比ターボファンエンジンを転用した大出力高効率の舶用ガスタービンも新たに開発されており、大型船の主機用や発電用として今後採用されていくものと思われる。その一例としてRR 社の航空用トレント(Trent)800をベースとする産業用トレントを舶用化したMT 30(Marine Trent 30)ガスタービンを図ントから低圧圧縮機を除いた中圧および高圧軸系をガス発生器とし、低圧タービンを出力タービンとしたもので、熱効率は単純サイクルでありながら40%を超えている。一方では、排ガス中の排熱回収により高効率化を目指す動きも活発で、排熱回収ボイラを装備して複合サイクル(combind cycle)化が図られたり、1970年代以降途絶えていた再生サイクル(regenerative cycle)や中間冷却再生サイクル(intercooledrecuperated cycle)を導入した舶用ガスタービンも開発されており、舶用ガスタービンとディーゼル機関との燃料経済性の差は徐々に縮まりつつある。図5・145 大出力高効率ガスタービンの一例(RR 社のMT 30ガスタービン)図5・146 再生サイクルガスタービン本体(スーパーマリンガスタービン)。再生サイクルガスタービンわが国では、燃料消費率は舶用高速ディーゼル機関と同等でありながらNOの排出量を10分の1に抑えることを目標として再生サイクルを取り入れたスーパーマリンガスタービン(SMGT)が開発された。SMGT は、軸流四段、遠心一段の圧縮機、A 重油焚きに対応する4本のマルチキャン型燃焼器、各二段の圧縮機駆動タービン、出力タービンとプレートフィン式再生器から構成される産業型で、ガスタービン本体部分の断面図を図5・146に示す。. 中間冷却再生サイクルガスタービン艦艇用ガスタービンの燃料消費量低減を主目的として、ノースロップグラマン社(Northrop Grumman)、RR 社が中心となって開発した中間冷却再生サイクルガスタービンWR-21の断面図を図5・147に示す。WR-21は、RR 社の航空用RB 211とトレントをベースとした航空転用型ガスタービンである。ガス発生器は、低圧軸系と高圧軸系の二軸構成で、低圧圧縮機と高圧圧縮機の間に水冷のプレートフィン式中間冷却器(intercooler)が装備されている。中間冷却を行うと、高圧圧縮機入口温度の低下に伴い高圧タービンにて消費される高圧圧縮機駆動動力が減少するため、出力タービンから取出しうる出力が増加する。中間冷却だけでは燃焼器入口温度も下がるため、サイクル最高温度への加熱に要する燃料投入量も増えることから直接的な高効率化は望めないが、再生器を併用することで燃焼器入口温度が出力タービン出口温度近くまで予熱され、中間冷却による燃料投入量の増加を抑制できる。その結果、中間冷却器による出力増加と再生器の排熱回収による燃料投入量低減効果が重畳されて大幅な高効率化が達成される。圧力比に関しては、SMGT の圧力比8.0に対してWR-21は16.2と高くとられている。これは圧縮機出口温度が低いほうが再生器での排熱回収量が大きくなるため、再生サイクルでは最高熱効率が得られる圧力比も低くなるが、中間冷却再生サイクルでは、中間冷却により圧縮機出口温度が下がることから、圧力比を高くとっても大きな排熱回収量と高い熱効率が保たれることによる。図5・148に、WR-21とスペイSM 1Cの燃料消費率特性を示す。単純サイクルガスタービンの燃料消費率は部分負荷で単調増加する図5・147 中間冷却再生サイクルガスタービン(ノースロップグラマン社/RR 社のWR-21ガスタービン)図5・148 燃料消費率図5・149乾式低NO燃焼器の一例(スーパーマリンガスタービン燃焼器)のに対し、WR-21は広い出力範囲で低燃料消費率を維持しており、低出力域での運転時間の長い艦艇用途では特に優れた燃料経済性が期待できる。この特性は、艦艇同様部分負荷での運転時間がほとんどを占める自動車用の再生サイクルガスタービンでも多用される可変ノズル(variable area nozzle)機構を出力タービン入口部に装備したことによるもので、低出力時には可変ノズルを絞ることで再生器入口温度が高温に維持され、再生器での排熱回収量を大きいままに保つことができる。また、SMGT でも部分負荷時の燃料消費率改善を目的として可変ノズル機構が用いられている。b. 排気対策大気汚染物質の中で現在特に問題とされているのが窒素酸化物(NO)と硫黄酸化物(SO)である。ディーゼル機関と比較すると、ガスタービンはNOの生成要因となる燃焼部最高温度や圧力比が低いうえに、硫黄分をほとんど含まない燃料を使用するため、NO,SOともに、国際海事機構(IMO)が舶用機関に要求する規制値に比べて十分低く、何らかの排気対策が早急に必要とされるレベルにはない。しかし、今後海上においても一般産業用と同様に、環境にいっそう配慮した舶用機関が必要とされることを念頭において、液体燃料に対する乾式低NO燃焼器(drylow NOcombustor)の開発が各社で進められている。その一例として、SMGT においても図5・149に示す希薄予混合・予蒸発+追だき燃焼方式の燃焼器を装備し、1g/(kW・h)のNO排出量を達成する低NO化が図られている。c. 振動騒音対策純粋な回転運動を行うガスタービンの振動レベルは往復運動の舶用機関に比べて十分小さい。したがって、防振ゴム製のマウントでガスタービンパッケージ全体を弾性支持することで船体への振動伝ぱを通常問題ないレベルまで低減できるうえに、船殻のたわみや船体側からの衝撃荷重、振動も緩和することができる。なお、弾性支持を行う場合は、過大変位を防止するためのストッパを併用するのが一般的である。ガスタービンからの騒音は高周波の流体音が主体であるので、ガスタービン本体を適切な防音エンクロージャで囲い、吸排気ダクト中に消音器を装備することで問題ないレベルに低減することは比較的容易である。ただし、吸気消音器(intake silencer)は部品が脱落して異物としてガスタービンに吸い込まれた場合、甚大な被害(FOD : foreign object damage)を招くことになる。また、排気消音器(exhaust silencer)は吸音材を充てんしたスプリッタが高温高速の排ガス気流中にさらされるため、消音器の構造、使用材料の耐腐食性や強度、排気消音器に関しては特に熱膨張にも十分配慮した設計を行う必要がある。文献(1) 坂部寛典、日本マリンエンジニアリング学会誌、36-11(2001)、1.(2) Jensen,T.、日本マリンエンジニアリング学会誌、37-2(2002)、41.(3)
