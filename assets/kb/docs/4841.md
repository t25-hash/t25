# 6・1・1 概説

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・1 ハイブリッドシステム

a. ハイブリッド自動車の定義従来の内燃エンジンに加えて他の動力源とそれ用の再充電可能なエネルギー貯蔵装置(RESS :rechargeable energy storage system)を搭載した車両を、一般にハイブリッド自動車(HV : hybrid vehicle)と呼ぶ。他の動力源としては、電気モータ、油圧モータ、機械式フライホイール等があるが、近年開発されたHV は、効率と制御性の高さから、ほとんどの場合が電気モータを搭載した車両、すなわち、ハイブリッド電気自動車(HEV :hybrid electric vehicle)となっている。なお、燃料電池車の場合も、バッテリ等のRESS を搭載した場合は、燃料電池とRESS とのHEV ということができる。b. HEV の分類HEV は、走行エネルギーの供給形態の面から、外部充電が必要なタイプと不要なタイプに分類できる。前者のタイプの多くは、電気自動車(EV : electric vehicle)に小型の内燃エンジンを搭載したものである。そのコンセプトは基本的にEV であり、走行に必要なエネルギーはおもに外部から充電された電気によって供給される。RESS の充電状態(SOC: state ofcharge)が設計上の下限値になるまではおもに電気を消費してEV走行するが、下限値に達した後はエンジンで発電した電力で走行するものである[図6・1(a)].このタイプは、EV の航続距離延長が目的であること、あるいは外部から充電されることから、レンジエクステンダあるいはプラグインHEV とも呼ばれる。バッテリの寿命、コストの問題があるため、2006年3月現在量産車は存在しないが、後者のタイプよりもCO排出量を低減できるポテンシャルを有しており、今後の技術開発が期待されている。一方、後者のタイプは、1997年に世界初の量産HEV として発売されたトヨタプリウスに代表されるように、日本が世界に先駆けて開発・市販化を進めてきたHEV である。そのコンセプトは、電気駆動系の効率の高さと内燃エンジンのコンパクトネス、軽量性、利便性といった長所を組合わせ、従来の内燃エンジン自動車(ICEV : internal combustion engine vehicle)と同等の利便性をもたせながら燃料消費率(以下、燃費)の大幅な向上と排出ガスの低減をねらったものである。搭載されるRESS は、充電と放電を繰返しながらSOCがある一定レベルに保たれる[図6・1(b)].そのため、外部充電が不要であり、ICEV と同様に燃料の補給のみで走行できる。欧米の自動車メーカも2005年ごろから市販化を進めており、少なくとも台数上はこちらのタイプが主流になっていくと考えられる。ここでは、特に断りのない限り、HEV とは外部充電不要なタイプを指すこととし、解説したい。(a) 外部充電が必要なHEV(b) 外部充電が不要なHEV4―211c. HEV の長所と短所HEV の長所としては大きく3点が挙げられる。すなわち、(1)車両停止時のエンジンストップ、(2)減速エネルギーの回生、および(3)エンジン運転条件の最適化、である。. 車両停止時のエンジンストップHEV は車両停止時のエンジンストップを容易に行うことができる。HEV では、従来型ICEV のスタータよりも強力なモータを使用するため、停止しているエンジンを瞬時に再始動できるためである。エンジンストップによる燃費の向上効果は、平均車速が低くなるほど車両停止時間が長くなるため、平均車速が低くなるほど大きくなる(図6・2)。. 減速エネルギーの回生ICEV において、摩擦ブレーキおよびエンジンブレーキによって熱として消費していた減速エネルギーを、HEV では駆動用モータをジェネレータとして作動させる、すなわちモータジェネレータ(MG : motor generator)とすることによってバッテリ等のRESS に一時的に貯蔵し、加速時に利用する。図6・3は、典型的な都市バスが市街地運転サイクル(M 15モード)で走行した際の駆動輪出力を示した例であるが、駆動エネルギーを1.0とすると、減速エネルギーは0.6にも及ぶ。したがって、この0.6の減速エネルギーを100%回収、再利用したと仮定すると、駆動エネルギーを0.4に低減できることになる。このように、減速エネルギーの回生によって、燃費を向上させることができる。また、加速時のエンジン負荷を軽減できることから、排出ガスの低減も可能である。. エンジン運転条件の最適化内燃エンジンは図6・4に示すように、低負荷になるほど効率が悪くなる特性をもっているため、HEV では可能な限り低負荷の運転を行わず、高効率領域で運転する。また、エンジンの最大負荷領域では、モータで補助することによって負荷を軽減し、NOや粒子状物質(PM : particulatematter)を削減することが可能である。また、燃料電池システムの場合においても、HEV 化することによって、効率が低下する極低負荷および高負荷運転を回避することができ、高効率化が期待できる。以上3点の長所によって、HEV は燃費向上(最大で従来車の2倍程度)と排出ガスの低減が可能となる。HEV の短所としては、まず、質量が大きく機構が複雑になりがちな点が挙げられる。これは、従来の内燃エンジンに加えてRESS、モータ、ジェネレータ、インバータ等の装置が必要になるためである。また、HEV では一般に、瞬間的に取出せる最大出力を従来車と同等としているが、燃費向上のためエンジンを小型化して連続的に取出せる出力を抑えた設計が多い。そのため、質量増加の影響と相まって登坂路が続く走行条件では従来車よりも動力性能が低下する。こう配がきつくなるに従って減速エネルギー回生の効図6・1 エネルギー供給形態から見たHEV の分類図6・2 平均車速と車両停止時間率およびアイドリング燃料消費量が走行燃費に占める割合4―212果も減少するため、従来車に対する燃費の向上率も低下する。HEV は本質的に以上のような短所を有するが、これらの問題は、RESS の出力密度・エネルギー密度の向上やモータの比出力の向上、用途に応じた出力(搭載量)の最適化およびシステム設計の工夫等によって、現状の市販車ではほぼ問題ないレベルに達している。
