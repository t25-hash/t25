# 4・1・2 選定

> **章**: 第4章 軸流タービンおよびラジアルタービン
> **節**: 4・1 形式と選定

a.衝動形軸流タービン（impulse type axial flow turbine）
(１)動翼の周速度の割に段当たり圧力比（仕事量）を大きく取れ
る．
(２)ノズル翼のみで圧力降下させるため流量を調節しやすい．
(３)反動形に比べて効率はあまり高くない．
(４)構造が比較的簡単で軽量化できる．
(５)軽量，高圧力比の点でロケットエンジン用タービンなどに適
用されている．
(６)段当たり大きな圧力比を吸収できるので蒸気タービンの初段
などに使われている．

(３)ラジアル形に比べて容易に多段化することができ，全体とし

(４)各翼列，各段での圧力降下の配分，翼形状の改善，二次流れ

ビンに多く採用されている．

れていない．
d.内向き半径流タービン
(１)反動度を50 %付近で選定することが多く，小型でも効率が
高い．
(２)構造上頑強であるから高い周速度にすることができ，1段当
効率が低下する傾向がある．

(３)構造が簡単なため軽量，安価である．
(４)軸方向長さを短くできるが，多段にすると流路形成が複雑に
なり効率が低下しやすい．
(５)エンジンの効率向上の目的からタービン入口ガス温度を高く
する要求があり，軸流タービンと同様，冷却タービンを採用
する研究も行われている．また，自動車用など小型のエンジ
ン用として，翼やケーシング材を耐熱セラミックスで作るセ
ラミックタービン（ceramic turbine）の研究開発も行われ
ている．
