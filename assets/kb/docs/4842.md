# 6・1・2 システム構成

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・1 ハイブリッドシステム

HEV は、システム構成上から、(1)シリーズ方式、(2)パラレル方式、および(3)シリーズ・パラレル方式、の3種に大別できる。HEV はエンジンとモータおよびRESS を組合わせたシステムであるため、設計のねらいによって、従来のICEV 寄りの構成やEV 寄りの構成とすることができる。図6・5では、ICEV に近い順図6・3 駆動エネルギーと減速エネルギーの比の例図6・4内燃エンジンの負荷-熱効率特性とHEV における運転目標範囲に各方式を並べた。a. シリーズHEV(series HEV)シリーズHEV はエンジンの駆動力を、ジェネレータを介して電気に変換してから用いる方式である。モータのみで駆動するため、EV に近い方式といえる。エンジンと車軸が機械的に連結していないため、他の方式に比べエンジンの運転領域を最適化しやすいのが長所である。一方、エンジン-ジェネレータ-RESS-モータと種々の要素を通過してエネルギーを伝達するため、損失が大きくなる短所がある。シリーズHEVの燃費は、内燃エンジンの効率が低下する低速低負荷ではICEVよりよくなるが、高速高負荷では前述の長所を短所が上回り、悪くなることがある。シリーズHEV はその特性から、低速でストップアンドゴーを繰返す都市内走行に向いており、三菱ふそうトラック・バスや日産ディーゼル工業等がこの方式の都市バスを発表している。b. パラレルHEV(parallel HEV)パラレルHEV の定義は後述のシリーズ・パラレルHEV との区別が不明りょうなところもあるが、一般的には、専用の発電機をもたず、RESS に蓄えられたエネルギーのみでモータを駆動する車両を指す。この方式は、エンジンの駆動力を直接用いるエネルギーパスがあるため、高速高負荷でも燃費が悪化せず、平均車速によらずつねに従来車よりもよい燃費が得られるのが長所である。日野HIMRやホンダインサイトのように、エンジンのフライホイールを比較的低出力のモータ1機で置換えた方式の場合、従来のスタータとオルタネータを1機のモータで兼用することになり、非常にシンプルな構造とすることが可能である(ただし、エマージェンシー用のスタータを装備する場合が多い)。この方式は、変速機-駆動輪間の動力伝達系にICEV のものを流用できることや、最もコスト高となるRESS の搭載量を削減できる長所があり、低コストでHEV 化できる。ただし、エンジンにモータが直結しているため、回生ブレーキ作動時にエンジンの摩擦損失の影響を受け、減速エネルギーの回収効率が低下するという短所がある。2001年には、さらに簡素化されたパラレルHEV として、定格36V のバッテリ(42V 電源系システム)を使用するシステムがトヨタクラウンに搭載され発売された。これは、従来型ICEV のパワートレインをそのまま使用し、従来のスタータとオルタネータを1機のモータジェネレータで置換えたものである。マイルドHEV(mild HEV)とも呼ばれ、ICEV に最も近いHEV ということができる。最大の長所は、専用部品とRESS 搭載量が少ないため最小限のコストアップでHEV 化できることである。短所としては、モータ出力が小さいため、減速エネルギーの回生量およびアシスト量も少なくなることであり、結果として燃費向上効果も小さくなる。燃費向上を、車両停止時のエンジンストップに頼ることになるため、車両停止時間比率の大きくなる渋滞走行において効果を期図6・5 ICEV、各種HEV およびEV のシステム構成待できる。c. シリーズ・パラレルHEV(series-parallel HEV)シリーズ・パラレルHEV は、上記2種のハイブリッド方式を組合わせたものである。低速低負荷ではモータで走行し、高速高負荷ではエンジンの駆動力を機械的に用いて走行するため、走行条件によらず効率がよいのが長所である。多くの場合、ハイブリッド専用の変速機を装備する必要があり、トヨタプリウスでは遊星歯車を利用した無段変速機(CVT : continuously variable transmission)を、日産ティーノハイブリッドやトヨタエスティマハイブリッドではベルト式CVT を採用している。変速機の設計と制御がシリーズ・パラレルHEV の鍵になると考えられ、今後種々のトライがなされるものと思われる。HEV のシステム構成はさまざまな車両特性に影響を及ぼす。一般的には、表6・1に示すように、EV 寄りになるほどRESS およびモータジェネレータ出力が大きくなるため、回生エネルギーが大きくなり、低速低負荷においてICEV 寄りのHEV より高い燃費向上率が得られる。一方で、車両質量が大きくなる傾向となる。なお、シリーズ方式の都市バスの中には、高速走行をしないという用途を考えてエンジン排気量を小さくし、変速機の省略と併せてベースディーゼルバスと同等の車両質量に抑えた例も存在する。
