# 6・1・3 システム作動

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・1 ハイブリッドシステム

HEV の作動形態は、システム構成やSOCによって異なるものの、基本的には図6・6に示すようになる。(1) 車両停止中は基本的にエンジンを停止させてアイドリング運転によるむだな燃料消費を防止する。RESS を充電させるためやエアコンディショナ(空気調和機)のコンプレッサを作動させるなどの必要がある場合に限り、エンジンを作動させる。(2) 発進加速時は、RESS に蓄えられたエネルギーから優先的に使用し、モータの駆動力を利用する。シリーズ方式およびシリーズ・パラレルHEV では、エンジンが停止したままモータのみで発表6・1 HEV システムと車両特性HEV システム構成ICEV 寄りEV 寄りマイルド~パラレル~シリーズ・パラレル~シリーズRESS およびMG 出力小さい大きい回生エネルギー少ない多い効率特性高負荷型低負荷型車両質量小さい大きい4―213進する。エンジンとモータが直結された多くのパラレルHEV では、停止したエンジンを引きずりながらモータで加速するか、あるいは、発進前にエンジンを始動し、エンジンとモータの両者を用いて加速することになる。モータのみで加速する時間は、EV 寄りのHEV になるほど長くなる傾向にある。加速途中において、加速に必要とする出力やエネルギーがRESS からの供給のみでは不足した場合、エンジンを始動して不足分を供給する。エンジンの運転は、燃費向上の観点からは、HEV システム全体の効率が最良となる条件であることが望ましい。(3) 定常走行時は、走行負荷が加速時よりも低下するため、走行負荷分のみのトルクを発生しようとするとエンジン効率の低い領域で運転せざるを得ない。そこで、SOC制御が充電モードになっている場合、エンジンは高効率条件で運転し、余剰トルクで発電しながら走行する。シリーズHEV およびシリーズ・パラレルHEVにおいてSOC制御が放電モードになっており、なおかつRESS のみで十分な出力を供給できる場合、エンジンを停止して走行する。(4) 減速時は、従来の摩擦ブレーキをできるだけ使用せず、回生ブレーキ(regenerative braking)にて減速する。減速エネルギーを高効率に回収するためには、駆動モータの出力とRESS の回生受入れ性が十分にあり、各要素の効率が高く損失が抑えられ、また、回生ブレーキのみでも自然なブレーキ感覚を実現できる制御が必要とされる。
