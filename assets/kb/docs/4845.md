# 6・1・5 燃費・排出ガス評価法

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・1 ハイブリッドシステム

a. 外部充電不要なHEV の評価法ほとんどの国産HEV のように外部充電不要でRESS のSOCが維持されるタイプでは,RESS の充放電を繰返しながら走行する.そのため,試験終了時のSOCが開始時より高くなった場合は,放電で利用した量よりもエンジンで充電した量のほうが多くなるので燃費は悪くなり,逆に,試験終了時のSOCが開始時より低くなった場合は,エンジンで充電した量よりも放電で利用した量のほうが多くなるので燃費はよくなる(図6・9).真の燃費を測定するためには,試験開始時と終了時のSOCが一致している(蓄電量差零とする)必要がある.実際には,蓄電量差零となるように試験するのは困難である.そこで,蓄電量差の異なる燃費を4点程度以上測定し,蓄電量差に対して燃費(ここでは,L/kmやMJ/kmなど燃料消費量が分子にくる単位を用いる)を直線回帰することにより補正する方法(図6・10)が日本から提案されている.この方法は,2006年にISO化される見込みである.ただし,信頼性の高い回帰直線を得るには複数回の試験が必要となる短所があるため,アメリカのSAE J1711:1999「小型HEV 排出ガス・燃費試験法」が提案する,式(6・1)~(6・3)で規定した公差内であれば,1回の試験値を認める考え方も併記される見込みである.この公差は,エンジンの発電効率にもよるが,±3%程度の燃費測定誤差を許容することになる.윩율읏읏읆윲윬윮×100≦1(%)(6・1)윩율읏읏=윩율읏읏×윩율읏읏×3600 J(6・2)읆윲윬윮=읆윲윬윮×γ읆윲윬윮×읆윲윬윮J(6・3)윩율읏읏:バッテリの蓄電量差(J)윩율읏읏:バッテリの電気量収支(A・h)윩율읏읏:バッテリの定格電圧(V)읆윲윬윮:消費燃料エネルギー(J)읆윲윬윮:燃料消費量(L)γ읆윲윬윮:燃料の比重(kg/L)읆윲윬윮:燃料の低位発熱量(J/kg)直線回帰により補正する場合の蓄電量差は,SOCが変化しても比較的電圧変動の小さいバッテリの場合は,試験中の電気量積算値(電気量収支)によって代用される.そのため,バッテリのクーロ(a)(b)図6・8 パワーウエイトレシオと加速性能図6・9 HEV のSOC変化が燃費に及ぼす影響図6・10 HEV の電気量収支と燃費との関係ン効率が98%程度以上ない場合は誤差が大きくなるので,注意が必要である.電圧変動の大きなキャパシタの場合は,試験開始時と終了時の開放電圧差から式(6・4)によって算出される.なお,排出ガスに関しては,エンジン形式やHEV システムによって,SOCとの相関がある場合とない場合があり,相関がある場合は直線回帰とする方向で検討されている.=12욡-욞(6・4):キャパシタの蓄電量差(J):キャパシタの静電容量(F)욡:キャパシタの試験開始時電圧(V)욞:キャパシタの試験終了時電圧(V)b. 外部充電が必要なHEV の評価法このタイプのHEV では一般に,バッテリのSOCがあるレベルに低下するまでは電気のみで走行し,SOCが低下してからはエンジンで発電しながら走行する.そのため,電気のみで走行する場合はEV としての電力消費率や航続距離の評価を行い,エンジンが始動してからは燃費と排出ガスの評価を行う必要がある.このタイプは現段階では市販車がないため,試験法の検討は机上で行われている.
