# 6・2・1 概説

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・2 ハイブリッド用エンジン

近年、地球環境保護や省資源の観点から、自動車の低燃費化と排気エミッション低減に対する要求はますます強くなってきている。すなわち、地球温暖化を促進するCOの削減、黒煙、粒子状物質(PM : particulate matter)、NO,SOによる環境汚染防止や酸性雨対策である。これまで自動車用内燃機関は、継続して広い運転領域での性能・効率の向上、および排ガスの浄化を図ってきた。しかし、内燃機関は定格出力発生条件下では最高効率を示すものの、低速低負荷領域、全負荷近傍領域では効率が低下する特性をもつ。したがって、さまざまな走行環境で使われる自動車においては、そのほとんどが熱効率の悪い部分負荷領域での運転となる。また、減速時には機関で得られたエネルギーが熱として捨てられてしまうため、総合効率の低下は避けられない。一方、上記環境負荷原因物質の低減に関しても、ほぼ定常状態で運転される機関であれば浄化が比較的容易になるが、運転状態が多様に変化する場合であれば浄化が困難になる。低燃費化と排気エミッションの最適化は相反する関係にあり、これを両立することは難しい。捨てられているエネルギーの回収ができ、内燃機関を効率のよいところで運転することができる手段を組合わせれば燃料消費量を大幅に減らすことができ、それに伴って排出される環境負荷原因物質の総量も削減できる可能性がある。しかしながら、内燃機関単独ではこのようなシステムの構築は大変困難である。ハイブリッドシステムは、複数の動力源を組合わせることによってこのような要求を満たそうというものである。一方、内燃機関以外の動力源についても開発が進められており、電気自動車や燃料電池車が代表として挙げられる。電気エネルギーはエネルギーとして最もクリーンであり利便性も高いが、現状ではバッテリ性能への依存度が高く、特にエネルギー密度の点で化石燃料に比べて大きく劣っており、主流にはなりきれていない。こうした長所・短所をもつ内燃機関と電気エネルギーの組合せによって、それぞれ単独使用される場合の欠点を補いながら、低燃費化、低エミッション化を図る手法の一つが、内燃機関・電気ハイブリッドシステムである。ハイブリッド本来の意味からいえば、組合わせる複数の動力源についてはどのようなものでもかまわないのであるが、一部大型機関と油圧モータを組合わせた蓄圧式ハイブリッドシステムの適用例がある以外は、内燃機関と電気モータを組合わせたものが大部分である。内燃機関としては電気モータの力を借りてさらなる低燃費、低エミッションを図り、電気モータはエネルギー密度の低さを内燃機関により補うことができる。さらに、電気モータを減速時に発電機として使用し、制動エネルギーを回生して蓄えることで航続距離を伸ばすことが可能になる。ハイブリッドシステムを大別すると、シリーズ方式、パラレル方式、シリーズ・パラレル方式に分類される。シリーズ方式では、内燃機関の出力を電気エネルギーに変換しこれを走行用に使用するため、内燃機関としては効率のよい領域で使用できる反面、エネルギー形態の変換に伴う効率低下、走行用モータの大型化やバッテリ容量の増加が必要となる。パラレル方式では、走行条件により内燃機関とモータからの動力を使い分けることが可能で、構造は複雑になるがシステムの自由度は拡大する(図6・11)。シリーズ・パラレル方式では、それぞれの長所を組合わせることが可能である。ハイブリッドシステムは上記の方式に大別されるが、最近では燃費向上率に対する機能から区別される例も出てきている(図6・12)。内燃機関・電気ハイブリッドシステムにおいては、走行のための駆動エネルギーはシリーズ方式、パラレル方式にかかわらず主として内燃機関により供給される。したがって、内燃機関の効率がシステムとしての効率に大きく寄与する。ハイブリッド用エンジンの選4―215択は、これら採用するシステム、車の使用環境等に応じて行われる。一般的な内燃機関形式に対して効率、クリーン度合い、燃料特性、コスト等により得失を比較すると表6・2のようになる。また、現在までに市販されたハイブリッド車、および適用されているエンジン群をまとめると表6・3,6・4のようになる。(a)
