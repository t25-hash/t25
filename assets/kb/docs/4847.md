# 6・2・2 ガソリン機関

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・2 ハイブリッド用エンジン

前述したとおり、ハイブリッドシステムにおいては内燃機関の効率が全体の効率に大きく影響を与えるため、内燃機関の効率向上を図る必要がある。現在実用化されているガソリン機関・電気ハイブリッドシステムにおいて特徴的な熱効率向上のための主要技術をまとめると、以下のようになる。a. 小型軽量化一般的に、ハイブリッドシステムはモータやバッテリ等の搭載により重量が増す。一方で、モータにより出力を補えることでガソリン機関としては低排気量・低出力とすることができる。ハイブリッドシステムとしてどの方式を選択するかにより差はあるが、基本的には1ランク以上小型の機関の採用が可能となる。機関が小型化することによって得られる軽量化に加えて、使用条件の緩和により往復回転部品(ピストン、コネクティングロッド、クランクシャフト等)、動弁系部品の軽量化が図れる。また、機関単体の出力が小さくなることで総発熱量も小さくなり、冷却系(ラジエータ、冷却水量)の小型、軽量化にもつながる(図6・13)。b. 熱効率向上ガソリン機関単体の効率向上手段として、以下のようなものが挙げられる。. 高膨張比(high expansion ratio)サイクル[ミラーサイクル(miller cycle)、アトキンソンサイクル(Atkinson cycle)]ハイブリッド用として、運転領域を通常機関に対して限定できる場合(低い最高回転数)、出力、トルクの低下を伴うため、これまでは採用が難しかった高膨張比サイクルの適用が可能となる。ガソリンエンジンのような等容サイクルでは、圧縮比を上げることで理論熱効率は向上するが、ノッキングの発生により圧縮比は制限されたものとなる。その対策の一例として高膨張比サイクルがある。機械圧縮比は上げるが吸気弁閉じ時期を遅らせて吸入空気量を抑え、実効圧縮比を従来エンジン並みに抑えることで、ノッキングを回避して実質的に熱効率を支配する膨張比(機械圧縮比)を高めることができるサイクルである(図6・14)。. 摩擦損失(friction loss)の低減(低フリクション化)ガソリン乗用車で市街地を走行する場合、全エネルギーの40%以上がエンジンの摩擦損失で失われているといわれており、効率向上の観点では摩擦損失の低減も重要な因子である。ハイブリッドシステム用として使用するうえで、低い最高出力、低い最高回転数という条件緩和により各部のしゅう動抵抗低減を図ることができる。例えば、フラッタリング等が発生しない程度までのピストンリング張力の低減、動弁系挙動を満足するバルブスプリング荷重の低減、各部しゅう動面積の低減などによって同一出力時の摩擦損失は小さくできる。また、オフセットシリンダを採用することで、ピストンサイドフォースと時間損失の低減を図ることができる。通常のガソリン機関にて適用しうる向上策は、ハイブリッドシステムにおいても当然採用される。上記を含め、主として採用されている技術を簡単にまとめ、表6・5に示す。内燃機関単体で熱効率向上を図ったとしても、通常は捨てざるを得ない減速時のエネルギーを回生し再利用できることが、ハイブリ車名トヨタコースタートヨタプリウストヨタエスティマトヨタクラウンホンダインサイトホンダシビック日産ティーノ構成シリーズシリーズ・パラレルシリーズ・パラレルパラレルパラレルパラレルシリーズ・パラレル気筒配置直列4気筒直列4気筒直列4気筒直列6気筒直列3気筒直列4気筒直列4気筒図6・13 その他効率向上と合わせて、排ガス浄化についてもハイブリッドシステムを有効に使用することができる。エンジン冷間時は触媒の暖気が不十分であり、高い浄化性能が得られない。このようなとき、車両の駆動力としてモータを最大限に活用し、エンジンからの駆動力を最小限にして排ガス浄化に最適な運転を行うことができる。エンジンの吸入空気量を一定にすることで空燃比のばらつきを抑え、エンジン排ガスエミッションの抑制を図るとともに、空燃比の最適化、点火時期の遅角化により触媒の急速暖気を実現することが可能である。
