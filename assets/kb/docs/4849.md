軸構成では，産業用ロボットとして高い作業性を確保することに
留意する必要がある．すなわち，動作領域が広いこと，特異点が動
作領域内で少ないこと，座標変換が容易であることなどが求められ
る．実際には，ロボット台座側からの3軸で先端の位置を決め，手
首部の構成軸で姿勢を決めるように役割を分担させ，手首部の構成
軸を一点で交わるように構成することが多い．このため，手首軸の
構成はねじり，曲げ，ねじりの順番とするのが一般的である．
構造体はアクチュエータや伝達機構を保持し，配線・配管を内蔵
し，関節間のリンクを構成する．材質は通常，鉄またはアルミニウ
ム合金である．ロボットの性能向上のため，軽量性と高強度・高剛
性の両立が要求される．
アクチュエータは，初期には油圧シリンダが多用されたが，簡易
型では低価格の点から空気圧シリンダも使用された．油圧アクチュ
エータは，当時のサーボモータに比して小型軽量で大出力であるこ
とから，油圧サーボ弁と組合わせて使用された．また塗装用では，
防爆性の観点から使用された．しかし油圧型ロボットは，サーボ用
作動油管理の煩雑さ，エネルギー効率の悪さ，特性の温度変化や非
線形性による制御の困難さなどの問題点を抱えていた．
電動サーボモータは，当初は制御性のよい高出力のものがなかっ
たが，性能の進歩が著しく，現在ではほぼ完全に油圧アクチュエー
タに取って代わった．高出力が可能になったこと，応答性が高いこ
と，制御性がよいことが電動化を可能にしたおもな要因である．サ
ーボモータの形式はDC（direct current ）ブラシ型から保守性の
よいDCブラシレス型，AC（alternating current ）型へと進化し
た．

こと，ロストモーションという非線形性をもつがバックラッシがな
一方，ハーモニックドライブの伝達トルクに限界があること，剛
性もそれほど高くないことから，大トルクの伝達にはサイクロ減速
機，RV（rotary vector ）減速機等がよく使用される．これらの減
速機も，従来の歯車を組合わせた減速機に比して軽量，無バックラ
ロボットの各関節軸にモータ・減速機を設置すると，アームの先
端側の重量が増加し，可搬重量の低下，動特性の悪化，振動，たわ
みの増大等の悪影響を及ぼすことになる．このため，重量物である
モータ・減速機はなるべくロボット基部側に設置して伝達機構で各
関節部に動力を伝達する必要が生じる．伝達機構の要素としては，
歯車，歯付きベルト，リンク，トルクチューブ等がある．これらを
組合わせて伝達系を作るが，手首部関節では複数の軸の伝達要素が
ロボットアームの先端で作業を行う部分をエンドエフェクタとい
う．ワーク（work-piece，被加工部品）をハンドリングするロボ
ットではハンドが，溶接ロボットや塗装ロボットでは溶接トーチや
スプレーガンがそれに当たる．ハンドやツールは作業に応じて専用
のものが用意され，複数のワークを扱う場合は，それぞれのワーク
に対応したハンドやツールを状況に応じて交換する機構（ツールチ
ェンジャ）が使用される．1台のはん用ハンドであらゆるワークや
ツールをハンドリングするのが理想であるが，作業精度や確実性の
要求を満たすものはまだなく，当面は現状方式が続くであろう．
通常，産業用ロボットは，教示データを指令値とした位置サーボ
駆動を行っている．センサは，位置検出器として各モータに接続さ
れたエンコーダが基本になっている．速度は位置信号を微分するこ
とで，駆動トルクはモータ駆動アンプの電流検出回路で得るのが一
般的である．
近年，作業に応じて外界センサを使用する例が増えてきた．視覚
センサが多用されるが，ワークの種類，位置・姿勢の判断，教示デ
ータと実際のワーク位置との差の補正用等に使用される．高速処理
と高精度の両立が課題となるが，その認識能力は急速に進歩してお
り，多数のセンサが実用化されている．しかし，現状では実時間で
の処理は比較的単純な形状に制限されている．
力制御等でワーク反力を検出するには，エンドエフェクタ部に力
センサを設置するのが一般的で，ひずみ計を使用した6軸力覚セン
サ（3軸方向の力とトルクを検出する）が多用される．今後の発展
が期待される分野であるが，センサの精度，頑健性等，広く使用さ
れるためには課題が残されている．
