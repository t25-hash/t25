# 6・3・2 直流モータ(DC motor)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

a. 概要直流モータは励磁方式により永久磁石励磁形と巻線励磁形に分類される.巻線励磁形には界磁巻線(field coil)と電機子巻線(armature)が直列に接続される直巻モータ(seriesmotor)と並列に接続される分巻モータ(shunt motor)および両方の巻線方式を有する複巻モータ(compound motor)がある.直流モータの特徴は,原理的に大きな始動トルクと負荷の変動に適応したトルクが得られること,車載電池の直流電源をそのまま利用できること,速度制御,回転方向の切換が容易な点である.また,制御装置の構成が安価であることから,古くより電気自動車の駆動モータとして用いられてきた.直巻および分巻直流モータは古くは電気自動車で多く用いられてきたが,最近では一人乗りの電気自動車など小容量用途に永久磁石式直流モータの採用が見られる(表6・8).一方,電気自動車,ハイブリッド電気自動車の範ちゅうには該当しないが,日産e-4WD で採用された他励式直流モータと減速機で構成された後輪駆動システムがある(図6・26).このシステムでは後輪駆動用車名エルフリゾートハイゼットバンアルトサンバーEVEV-1ルーキー製造者いすゞ自動車ダイハツ工業スズキ富士重工業タケオカ工業種類分巻モータ分巻モータ分巻モータ直巻モータ永久磁石式直流モータ最大出力kW/min70/125020.9/345022.3/348025/36000.6/4000最大トルクN・m/min542/059/342061/348070/3250(0.15)/4000制御装置IGBT チョッパトランジスタチョッパMOSFET チョッパMOSFET チョッパPWM 制御駆動方式後輪駆動後輪駆動前輪駆動後輪駆動後輪駆動主変速機前後進一段マニュアルマニュアル5速マニュアル5速ベルト式無段変速モータはエンジンに補機ベルト駆動を介した発電機出力を電機子(回転子)電流とし,4WD コントロールユニットから出力されるモータ界磁電流により作動する.b. 構造と特徴直流モータは界磁方式により図6・27のように巻線界磁形と永久磁石界磁形に分類される.直流モータは,磁束を作る界磁鉄心(field core),界磁巻線,起電力を生じ,主電流を通じる電機子巻線および電機子鉄心(arma-ture core),電機子巻線に接続された整流子(commutator),整流子に接触し電流を流すブラシ(brush),およびそれらを支持する構造部から構成される.. 直流直巻モータ界磁巻線と電機子巻線を直列に結線した接続方法を用いる.起動時に大きなトルクが得られるが,電機子の回転速度の増加とともに急激にトルクが減少する.端子電圧の極性を逆にしても,界磁電流と電機子電流の極性が同じであるため回転方向を変えることができないことから,変速機と組合わせて用いられる.. 直流分巻モータ界磁巻線と電機子巻線が並列に結線した接続方式を分巻モータと呼ぶ.界磁巻線と電機子巻線が独立した電源に接続されるモータを含めて称される場合もある.界磁電流を調整することにより広範囲の回転速度での運転が可能であることと,界磁電流あるいは電機子電流のいずれかを正負とも流せるようにすれば容易に正逆両方向での回転が実現できることが特徴である.. 直流他励モータ電機子電流とは別の直流電源から励磁電流を受けて界磁磁束を作る方式であり,2種類の直流電源が必要である.図6・26の例では,エンジンによりベルト駆動された発電4―221表6・8 直流モータの適用例(a) 適用例(b) 構造図6・26 他励式直流モータの適用例と構造(日産マーチe-4WD)(a) 巻線励磁形(b) 永久磁石界磁形図6・27 直流モータの構造機の出力電流を電機子(回転子)電流とし,固定子の界磁コイルにはコントローラの制御電流を用いている.
