# 6・3・3 交流誘導モータ(AC induction motor)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

a. 概要交流誘導モータは,構造が堅牢であり保守が容易なことから,産業用として広く用いられている.誘導モータの可変速駆動技術は,比較的速度制御精度を必要としない適用分野では電圧と周波数比を制御する制御が,また高精度な速度制御や高速応答制御が必要な分野では磁束,トルクを個別に制御するベクトル制御が用いられる.電気自動車,ハイブリッド電気自動車への適用は,インバータに用いる車載可能なパワー素子が開発されてきた1990年代前半から始まっている.誘導モータの特徴として,(1)回転子の構造が堅牢,(2)10000minを超える高速回転が可能,(3)高速・低トルク領域において弱め界磁制御,最大効率制御などを併用することにより高効率制御が可能,などが挙げられる.一方,(4)励磁電流が必要なため,特に低速領域の力率が低い点が短所である.誘導モータの適用は,乗用車用途ではより高効率,小型化への要求が高まり,希土類磁石の実用化が進みPM モータが移行してきたことから,現在では大型車クラスに多く見られる.b. 構造と特徴誘導モータは回転子構造から,巻線形(wound rotor)とかご形(squirrel cage rotor)に分類される.電気自動車には,高速運転と小型軽量化の設計要求からかご形が用いられている.かご形は固定子(一次側)に三相巻線が施されており,周波数に比例した速度の回転磁界を作る.回転子(二次側)のスロットにバー(bar,導体)を装着し,その両端をエンドリング(end ring,端絡環)により短絡したかご形の回転子をもつ.同モ4―222内燃機関ータの構成例を図6・28に,適用例を表6・9に示す.
