# 6・3・5 電磁鋼板(electrical steel sheet)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・31 ).

モータに用いられる電磁鋼板には,モータの高効率化,小型化するために低鉄損で高磁束密度な材料が要求される.それらの磁気特性を得るため,炭素や窒素などの不純物を減らすことで透磁率を高め,また,薄板を用いることやSiなどの添加により固有抵抗を上げることで,鉄損低減を行った無方向性電磁鋼板が用いられる.しかし,一方でSi量の増加は素材のコスト増加につながり,それ以外にも合金の硬さや脆性を増し,加工性を悪化させる.これら磁気図6・33 埋込磁石形の磁石配置の検討例図6・34 焼結拡散接合を用いたロータ4―224内燃機関(b) 集中巻方式(a) 分布巻方式図6・35 ステータ巻線方式(分布巻方式,集中巻方式)(a) ホンダインサイト(b) ホンダシビック図6・36 集中巻方式での非対称配置例特性や加工性以外にも,機械強度や鋼板の形状,それに絶縁抵抗をもたせるために付与されている被膜の諸特性なども重要な特性となってくることから,モータの製造に当たってはこれらすべての諸特性とコストを総合的に判断し,最も適した電磁鋼板が決定される.家電製品に使用されるモータや工業用のはん用モータなどでは,商用周波数の50Hz または60Hz での運転が前提であり,電磁鋼板の種類と磁気特性は,表6・12のJIS C 2552:2000「無方向性電磁鋼帯」で規定されている.種類記号の最初の2けたは呼称厚さの100倍を表し,下4けたの数字は周波数50Hz での鉄損保証値を示す.それに対して,自動車駆動モータでは,発進から高速走行に至るまでの広い運転領域で損失の低いことが必要であり,0Hz から1000Hz 以上の広い周波数範囲で低鉄損であることが要求される.図6・37はハイブリッド車用電磁鋼板の磁気特性の最近の動向を示したもので,400Hz での鉄損が指標として用いられている.
