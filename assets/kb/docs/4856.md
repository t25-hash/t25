# 5・4・1 サービスロボット（service robot )

> **章**: 第5章 ロボティクスの応用展開
> **節**: 5・4 人間共存型支援システム

a.サービスの内容と機能 家庭内の日常生活において期待さ
れるロボットのサービス機能にはさまざまな種類があり，また公共
施設や商業施設でも不特定多数の人を対象とする各種のサービスロ
ボットの機能が考えられる．手術や介護，災害救助など特定分野の
高度なスキルアシスト以外に限っても，サービスロボットの種類や
用途を限定することは難しいが，おもな機能を大別すると以下のよ
うなものが挙げられる．
(１)家事支援（housekeeping support )
･居室や風呂場，浴槽などの掃除
･洗濯から乾燥，収納までを含めた衣類の管理
･調理や皿洗い，後片付けから生ごみ処理などの食事支援
(２)セキュリティ支援（security support )
･戸締り確認，侵入者監視などのための巡視や通報
･消灯確認や防犯照明の管理
(３)ヘルスケア支援（health care support )
･トイレ，風呂，ベッドなどの場での健康管理情報の検出と提示
･医療機関との通信をもとにした健康アドバイス情報の提示や通
報
(４)自立行動支援（self - care support )
･自律移動車いすや歩行器などによる移動支援
･ベッドやトイレなどでの移乗や姿勢変え支援
･生活用具や重量物などの供給，搬送支援
(５)コミュニケーション支援（communication support )
･情報提供のためのエージェント
･話し相手やチャイルドケア
･接客や案内
以上のようなサービスの内容による分類のほかに，屋内/屋外，
家庭/施設などのようにロボットが活用される場による分類や，人
との接触の有無，移動やマニピュレーションなどの基本機能，形態
などによる分類も考えられる．
b.サービスロボットの要素技術 機能や形態の多様性から，
サービスロボットに必要な要素技術もまた広範囲に及ぶ．自律移動
やマニピュレーション，マンマシンインタフェースやさまざまな情
報通信など多くの技術のインテグレーションが求められるが，日常
的にさまざまな人と接するというサービスロボットの大きな特徴か
ら，対話や個人識別などのヒューマンインタフェースの機能が特に
重要である．また，産業用ロボットや特定のスキルアシストロボッ
トと違って，その活動環境が限定しにくく変化しやすいという特徴
から，さまざまなセンシング手段を駆使した環境認識技術と，それ
に応じた安全で信頼性の高いアクチュエーション技術も必要であ
る．こうした要素技術の中でも大半のサービスロボットの基本機能
となる自律移動について，その機能を構成するために必要な要素技
術のうち代表的なものを以下に述べる．

建物のCADデータや各種の入力ツールを用いてあらかじめロボッ

止し効率的な移動ができなくなるため，この停止エリアの大きさは

現するためには，障害物の種類を認識しその特性に応じた回避行動
をとることも必要である．障害物が人である場合には，まず安全性
の確保が優先されなければならないため，すれ違いや接近において
もある程度の間隔と急停止に対応できる速度を維持しなければなら

うことによって，より高度でスムーズな障害物回避が可能になる．
