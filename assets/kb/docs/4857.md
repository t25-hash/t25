# 6・3・7 冷却装置

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・31 ).

ハイブリッド電気自動車用モータは出力密度の向上に伴い,冷却効果を上げるために液体を用いて冷却する設計傾向にあり,水冷以外にも油による冷却が用いられる場合もある.水冷を用いる構造では,モータのハウジングにウォータジャケットを設け,インバータなどの制御装置を含め冷却するシステムが用いられる場合が多い.冷却はモータ,インバータに内蔵された温度センサにより電動ポンプ制御を行う.小型の電気自動車では,発熱量が比較的少ないことから,空冷方式が多く用いられるが,トランスアクスル部の潤滑油を利用したオイル循環冷却方式を併用する例も見られる.
