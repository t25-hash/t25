# 6・3・8 回転検出器

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・31 )。

モータの回転角度検出器は、出力信号の形態によりアナログ検出器とディジタル検出器に大別される。前者には、DCタコジェネレータ、シンクロレゾルバが、後者はロータリエンコーダがある。PM モータの制御では、トルク制御部、電流制御系、-軸への座標変換系および位置検出器からなる制御系とPWM(pulse width modulation)インバータの組合せが用いられる。磁極位置を正確に検出する方法が必要であり、位置検出器としてレゾルバ(resolver)と呼ばれる電磁誘導型角度検出器が用いられる例が多い(図6・39)。レゾルバの信号変換器にはRD 変換器(resolver digital transformer)と呼ばれるIC(半導体集積回路)が用いられ、図6・39中の制御誤差εを零に近づけるよう閉ループ負帰還制御することでアナログ角度情報がディジタル値に変換される。
