# 5・5・1 アミューズメントロボット（amusement robot ）

> **章**: 第5章 ロボティクスの応用展開
> **節**: 5・5 アミューズメント・セラピーロボット

ロボットは従来からアミューズメント目的にも用いられてきた
SDR（後のQRIO) といった二足歩行ロボットがこれに分類され
る．
(３) ペットロボット（pet robot )
外見は動物に似せたものが多い．しっぽを振る，音を出すなどの
表現により，人間からの働きかけに自律的に反応し，また自動的に
さまざまな表現をするといった方法によって人間との相互作用をす
るロボットである．例えば，ソニー(株)のアイボ などがこれに
該当する．ペット動物を飼っている場合と同様に，生き物のように
自律的にふるまうロボットとかかわることで人々を楽しませる．ま
た，精神的な癒し効果などをもたらす場合もある．
(４) ソーシャルロボット（social robot )
相互作用により人と長期的・社会的な関係を築き，人間の日常生
活を幅広く支援するようなロボットのことである．擬人的な外見を
もつものが多く，また高い自律的な相互作用能力が特徴である．コ
ミュニケーションロボット（communication robot ），パートナー
ロボット（partner robot ）とも呼ばれることがある．アミューズ
メントにとどまらず，コミュニケーションを通じたさまざまなタス
クが期待されている．商品として実用化されているものはまだない
が，研究開発が進められており，プロトタイプとして例えば本田技
研工業(株)のアシモ，ソニー(株)のQRIO（キュリオ），ATRの
Robovie（ロボビー) などのロボットが発表されている．家庭や
個人での所有が期待されているパーソナルロボット（personal
robot ）もこれと同様のコンセプトによるものである．
b.ヒューマンインタフェース アミューズメントロボットの
インタフェースは他の機械装置などと大きく異なる．通常の機械装
置では，順番にボタンを押すといったように，人は一定の取決めに
従った操作を行う．また，操作に対して機械装置が完全に正しい反
応を行うことが求められる．
これに対して，多くのアミューズメントロボットはユーザの入力
をほとんど，あるいは全く取り決めない．また入力手段も，ロボッ
トが適切に反応できるかどうかはともかくとして，音声，身振り手
振り，ふれあいといったさまざまな方法が可能である．これは，む
しろ人間どうしのコミュニケーションや人間が動物に働きかける場
合に近い．また，人間側がロボットに期待することも，指示に対す
る完全に正しい反応というよりはむしろ，時に間違えるのも愛嬌の
うちとして，人間の反応になにかしらリアクティブに反応し，ある
いはロボットがアクティブに何か行動をしてみせることでロボット
の知的なふるまいを見せ，言葉や非言語のやりとりによる相互作用
を行うことにある．
c .アクチュエータ，表現系 自由度は，犬型のペットロボッ
トや上半身のみが人間型のロボットで20程度，二足歩行人間型の
もので40程度である．特にペットロボットやソーシャルロボット
では高速な動作よりも，柔らかい制御，安全性，小型であることが
重視される．手先などの可動範囲よりも，むしろ表現能力の点で動
物や人間に近いかどうかという視点で設計されることが多い．これ
に加えて，LED（light emitting diode）などにより喜怒哀楽の顔
表情を作り出すものもある．ロボットの内部には金属製の部品が使
われても，表面にはプラスチック，シリコーンゴム，布，毛皮など
を利用することで柔らかさや暖かさを実現するものも多い．
d.センサ アミューズメントロボットの内界センサにはポテ
ンショメータ，ジャイロなどが用いられるが，むしろ特徴的なのは
外界センサである．アミューズメントロボットには，人間からの働
きかけに対して反応するための多様な外界センサが搭載される．利
用されることが多い外界センサの種類と用途について以下に示す．
(１) 視覚
人の位置，動き，ジェスチャ，顔の表情といった情報の取得のた
めに，CCDカメラが多くのアミューズメントロボットに搭載され
ている．たいていの場合，ロボットの眼に該当する位置に1～2台
取付けられる．これに加えて，より視野の広い魚眼カメラや全方位
カメラも用いられる場合がある．
(２)聴覚
マイクロホンも多くのアミューズメントロボットの耳の位置に搭
載され，音声認識に用いられている．やや高価であるが，多数の同
期入力可能なマイクロホンから構成されるマイクロホンアレイが胴
体部などに搭載される場合もある．マイクロホンアレイは，音源方
向の検出や音源分離による雑音の低減に利用される．
(３)触覚
人とふれあう状況を想定したロボットでは，ロボットをなでる，
たたくといった入力が期待される．そのため，多くのペットロボッ
トと一部のソーシャルロボットでは触覚センサが利用されている．
単純なプッシュスイッチ式のもののほか，圧式のセンサを利用する
ものもある．
e .知能化アルゴリズム 特にペットロボットやソーシャルロ
ボットに該当するアミューズメントロボットにとって，人間との相
互作用機能は最も本質的な機能である．そのためには，人間の働き
かけに応じた知的に見えるふるまい（ビヘービア）が重要である．
アーキテクチャは人間の動作に反応するといった短時間の知能化に
関連し，いかにビヘービアを遷移させるかといった行動制御が重要
となる．
(１)アーキテクチャ（architecture)
アーキテクチャはロボットの知的なふるまいの基本設計である．
アミューズメントロボットに関しては，人間の行動といった非常に
動的な要素に反応するために，高い反射行動能力が必要とされ，多
くの場合，サブサンプション（subsumption）アーキテクチャに代
表されるような行動ベースド（behavior based）制御（4･7･3項）
を基本としたアーキテクチャが採用される．
(２)ビヘービアの遷移と行動制御
多くのアミューズメントロボットは，インタラクティブなふるま
いを実現する要素行動としてビヘービアを用いている．ビヘービア
難であり，人工知能分野での二人ゲーム問題といったプランニング
過去の行動や状態と照らし合わせて一見矛盾しないような行動を行
表すことが可能だが，より長期的にロボットの行動を変化させ，ビ
は，しっぽを振る，ボールをける，お辞儀をする，握手をするとい
ったロボットの一つの具体的なふるまいに対応する．そして，この
ビヘービアを逐次的に切り換えるような制御によって知的なふるま
いや相互作用が実現されている．
ビヘービアの遷移に当たっては状況依存性の利用が重要になる．
ロボットの多くのビヘービアは特定の状況下では適切に実行できる
が，すべての状況において実行できるものではない．また逆に，特
定の状況を仮定することで効率的で信頼性の高いセンシングが可能
になる．さらに，ロボットがビヘービアを行うことによって状況が
限定される．例えば，人間型ロボットが人間と握手をするというビ
ヘービアを例にとると，ロボットが人に握手を求めるという状況
は，人がロボットの前方の一定距離にいる状況であると考えられ
る．また，この状況で人がロボットの手先に触れると，それは握手
が成立したと考えることができる．しかし，一般的な状況でロボッ
トの手先が触れられたからといって握手が行われたと認識すること
はできない．このように，ロボットが現在の状況を利用し，また自
ら状況を作り出すことで，状況に依存した期待待受け型のセンシン
グを行い，一般的な状況では困難な人間行動認識を実現することが
できる．
ビヘービアの遷移に関しては状態遷移モデルやその拡張形式が利
用されることが多い．遷移は，本来は行動のプランニングに関する
問題であるが，人との相互作用を行うアミューズメントロボットに
とって，相手となる人間の将来的な行動を予測することは非常に困
が適用可能な状態にはなかなかならない．むしろ，ロボット自らの
うための状態遷移モデルや，ルールベースの制約メカニズムによっ
て遷移行動が設計される．
f .相互作用の設計モデル 数分から数時間程度のロボットの
ビヘービアの遷移は，上記の状態遷移モデルやルールによって書き
ヘービア遷移のバラエティを作り出すことも，ロボットと相互作用
する人間を飽きさせないために重要である．このようなロボットの
ふるまいの決定は，ややアートに近い部分もあるが，どのような相
互作用のモデルを設計ポリシーに採用するかは工学の領域であろ
う．以下に，アミューズメントロボットに用いられることの多い相
互作用の設計モデルについて解説するが，複数のモデルが同時に用
いられることも多い．
(１) 行動モデル
ロボットの行動が人間に直感的に理解できるように，しばしば実
際の動物や人間の行動をモデルとしてロボットの行動が構成され
る．ペット型ロボットの実現に際しては，動物の典型的な行動パタ
図5･47人工物の設計における応用分野別主観的評価と客観的評価
ーンを見いだす動物行動学（ethology）の手法が利用されること
がある．例えば，アイボは動物行動学により解明されたイヌの典型
的な行動（周囲の探索，摂食，休息，遊びなど）をビヘービアとす
ることにより，イヌと類似したふるまいをするように設計されてい
る．
(２) 感情モデル
感情（emotion）とは，喜怒哀楽といった人間の気持ちや心的な
内部状態のことを指す．人間どうしのコミュニケーションの際に
は，顔の表情や発話の韻律，ニュアンスなどに感情が表出する．ア
ミューズメントロボットでも，たたかれると怒る，なでられると喜
ぶなど感情モデルに沿ったビヘービアをもたせ，人間や動物と似た
反応を返すことで，実際の生き物のように見せるために感情モデル
が利用されている．また，その場での直接的な反応のみでなく，例
えば，なでられるとしばらくは機嫌よく歩き回るというように，感
情モデルに関する内部パラメータを変化させて状態遷移モデルの遷
移を制御する手法が用いられる．ソニー(株)のアイボ，オムロン
(株)のNeCoRo（ネコロ），日本電気(株)のPaPeRo（パペロ）な
ど，多くのロボットに感情モデルが実装されている．類似のモデル
として，食欲や睡眠欲といった欲求をモデルに取り入れることで，
たくさん歩き回ると眠くなって休む行動が頻出するようになるとい
った欲求モデルによる状態遷移モデルの遷移制御が行われることも
ある．
(３) 学習モデル
長期の相互作用で重要となるのが学習や適応のメカニズムであ
る．自分にだけなついている，だんだん言葉を話すようになった，
お手を教えたらお手を覚えたなどのように，ロボットが何か新しい
ことを覚え，成長することは，人間の興味を引き続けるために有効
な手法である．実際には，このために見掛け上の学習が行われる場
合が多い．すなわち，ロボットは実は多くのビヘービアや機能があ
らかじめ実装されているが，初めのうちはこれらのビヘービアや機
能を表出させず，学習や成長のトリガが生じた場合にこれらのビヘ
ービアを新たに表出させることで，あたかもロボットが成長し，新
しいことを学んだかのようにふるまわせるのである．
