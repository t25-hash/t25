a. リラクタンスモータ(reluctance motor)PM モータがリラクタンストルクを補助的に利用するモータであることに対して、永久磁石をもたずリラクタンストルクのみを利用したモータをリラクタンスモータと呼ぶが、これにはスイッチトリラクタンスモータ(switched reluctance motor,SR モータ)、シンクロナスリラクタンスモータ(synchronous reluctance motor,SynR モータ)がある。これらはリラクタンストルクの活用度合いによる違いと見られることから、最近ではPM モータを含めてリラクタンス応用電動機という分類で扱われるようになった(図6・40)。SR モータの構造上の特徴はステータ、ロータともに突極構造をもつ点である。ロータの位置情報に基づいた電流をステータの突極に集中巻きした巻線に供給する。ロータの突極が近づいてきたステータのコイルに通電を順次切り換えることにより磁気吸引力が発生し、ロータの回転トルクとなる。これに対して、SynR モータはロータにだけ突極があり、ステータはPM モータと同様の構成である。b. インホイールモータ(in-wheel type motor)車輪の独立駆動を実現するため、車輪の内部にモータを取付ける構造をインホイールモータと呼ぶ。トランスミッション、デファレンシャルギヤなどが不要となり、トルク伝達効率が向上するとともに、車両部品配置の設計自由度が増す利点がある。トルク伝達方式によりダイレ図6・39振幅変調型レゾルバの構成とRD 変換器表6・13 PM 同期モータでの振動騒音の強制力と対策例ステータスロット数とロータ極数の最小公倍数×回転周波数スキュー、補助溝電流の基本周波数の6倍(三相全波駆動の場合)誘起電圧の正弦波化、正弦波駆動正弦波駆動、剛性向上4―226内燃機関スイッチトリラクタンスモータ(SR モータ)表面磁石形(SPM)埋込磁石形(IPM)シンクロナスリラクタンスモータ(SynR モータ)マグネットトルクのみで回転マグネットトルクとリラクタンストルクの両方で回転リラクタンストルクのみで回転図6・40 リラクタンスモータ図6・41 インホイールモータの構造例クト駆動方式、減速機付き方式がある。図6・41に電気自動車用に開発されたインホイールモータの構造例を示す。
