# 5・1・2 風力エネルギー

> **章**: 第5章 再生可能エネルギー
> **節**: 5・1 概説

地球上の大きな風の流れとして赤道近辺の貿易風，南北半球の緯
度が約40°の偏西風がある．このような大局的な風の流れから誘因
されたり，あるいは地上の地勢などにより地域で得られる風を風力
として利用することができ，おもに発電に利用されている．
a.風力資源 わが国では，風力エネルギーの利用拡大のため
に設置の容易さなどの観点から山深い内陸部よりもむしろ沿岸部で
風速測定が行われている．それによれば，北海道の日本海側，東北
地方北部，伊吹山系，南四国に多くの最適地があり，特に宮古島に
は各所に適地があり，これらの地点に数々の風力発電設備が設置さ
れている．
b.風力エネルギー利用技術 風力は風車によって動力として
得られ，エネルギー量は風車の回転面積（受風面積）を単位時間当
たりに通過する空気量と風の運動エネルギーの積で求められ，受風
面積に比例し，風速の三乗に比例する．そのため，風況のよいとこ
ろに受風面積の大きい風車を設置して風力エネルギーを最大に利用
することになるが，風車で得られる量は最大でこのエネルギーの約
6割である．
風力発電に使用される風車は風車の羽根の軸が水平か垂直かの向
きによって二分され，さらに回転力の相違で分けられている．現
在，最も普及している形式は3枚羽根の水平軸プロペラ（horizon-
tal axis propeller ）式である．発電出力は大型化することによって
経済性が高まる（発電コストの低下）ため，単機で500kW，1
MW級のものが実用的に使用されている．
わが国では，北海道の苫前，東北北部の竜飛に大規模の風力発電
設備が設置されており，近年，風力発電が急速に普及し，2003年
末には発電設備の容量が約500MWに達している．そのために，
わが国では，2010年の風力発電の導入規模を300MWとしていた
が，目標値を3000MWに修正する見込みである．
