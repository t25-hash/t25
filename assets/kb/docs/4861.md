# 6・4・1 概説

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・4 電力変換器と制御装置

電気自動車、ハイブリッド電気自動車ではモータ走行を前提にした駆動力制御設計が必要である。モータの使い方で見ると、図6・42に示す可変速ドライブシステムに相当する。自動車駆動用モータに求められるおもな特性は以下のようになる。(1) アクセル開度に応じたトルク制御が可能なこと(2) 低回転時(低速時)一定トルク特性(3) 高回転時(高速時)一定パワー特性(4) 減速時の負トルク(エンジンブレーキに相当するエネルギー回生)(5) ゼロ回転時のトルク発生(6) 微低速時のクリープトルク電力変換器はこれらのモータ駆動力を制御するためにモータシステムへの電力変換と制御を行う装置である。電力変換方式により、直流から直流へ変換するチョッパ回路、直流から交流に変換するインバータ(inverter)、周波数変換を用いたサイクロコンバータ(cycloconverter)がある。二次電池を用いる方式の電力変換方式には、表6・14に示す種類があり、直流から直流へ変換する直流チョッパ回路、直流から交流に変換するインバータ、周波数変換を用いたサイクロコンバータ、マトリックスコンバータなどがある。交流誘導モータ、PM モータではPWM 制御を用いた三相電圧型インバータが一般的に用いられている。モータ制御以外の電力変換器では、ハイブリッドシステムの高電圧電池から補機電源(低圧)への電力変換用のDC-DCコンバータがある。また、燃料電池自動車では、燃料電池の発電電力(直流)と車載二次電池(直流)との間で電圧調整を行う必要性から、駆動モータ用インバータのほかに、大容量のDC-DCコンバータが用いられる。このほか、電気自動車では、外部電源から車図6・42 可変速モータドライブシステムと電力変換器、制御装置表6・14 電力変換器の種類出力入力DC(直流)AC(交流)DC(直流)直流変換・直流チョッパ逆変換・インバータAC(交流)順変換・整流回路・高力率コンバータ周波数変換・サイクロコンバータ・マトリックスコンバータ図6・43 直流チョッパ回路によるモータ制御載充電器を介して電池を充電する。この充電器も車載電力変換器の一つである。これらの電力変換器、制御装置を一つの筐体に実装したものをパワーコントロールユニット(PCU : power controlunit)と呼ぶ例が多い。
