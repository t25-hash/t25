# 6・4・2 直流モータ用チョッパ

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・4 電力変換器と制御装置

直流モータを用いた駆動システムはおもに小型電気自動車に用いられ、その制御には抵抗制御、界磁制御、電圧制御の三つの方式がある。抵抗制御は電機子回路に直列に抵抗を挿入し、この抵抗値を変えることで出力トルクの制御を行う。界磁制御は界磁抵抗器またはチョッパ(chopper)が用いられる。電圧制御はモータの端子に印加される電圧を変化させる方法である。電圧制御には、電圧切換え制御、直流チョッパ制御がある。電圧切換え制御は車載の組電池のタップ切換えをする方法であり、ダイオードとスイッチを用いた回路構成がとられる。直流チョッパ制御は半導体スイッチを用いてオンオフさせることでモータに印加される直流電圧の平均値を制御する。図6・43はチョッパ回路の基本構成である。力行(加速)動作の場合は、図6・43中の矢印が示すようにパワー素子IGとダイオードDが動作する。IGがオンの場合はチョッパ出力電圧=、オフの場合には、平滑リアクトルLの電流が連続するため、Dが導通し、=0となる。平滑リアクトルで平均化された電圧は次式となり、パルス幅比+を変化させることで電圧制御する。=+ここで、:IGオンの時間、:IGオン(IGオフ)の時間。制動時の回生制御では、IGを動作させ、直流モータを電源とした直流チョッパが形成され、電源電池に電力が回生される。後退走行ではモータの回転方向を逆にする必要があり、電機子端子電圧、界磁巻線の切換えを行う方法のほか、ブリッジ型可逆チョッパを用いる。直流チョッパに用いるパワー素子はIGBT,MOSFET が使われる。
