# 6・4・4 モータの制御法

a. 直流モータの制御直流モータの可変速運転を行うためには、(1)電機子電圧を変える、(2)界磁電流を変える、(3)電機子抵抗と直列に抵抗を挿入する、などの方法がある。抵抗制御は損失が大きくなることから、現在では用いられることは少ない。IGBT,MOSFET など半導体電力変換方式による電機子電圧(電流)制御法、界磁電流制御法が主流である。b. 交流誘導モータの制御。 ベクトル制御(vector control)誘導モータはインバータで駆動し、その周波数を変えることによって容易に速度制御が可能となる。通常、電圧と周波数の比を一定に変化させる一定制御が広く用いられている。しかし、この方法では定常トルクを制御することはできるが、電気自動車で必要となる瞬時トルクを制御する精度が得られないことから、より精度の高いベクトル制御が広く用いられている。適用に当たっては、回転速度の検出センサと-軸と三相電圧욊,욋,욌の間の座標変換が必要と図6・46 パワーコントロールユニットの水冷構造例図6・45パワーコントロールユニットの構成例(a) 電池パック(b) インバータケース内部図6・47 電池パックとパワーコントロールユニットの空冷方式の構成例[(株)本田技術研究所提供]図6・49誘導モータ弱め界磁制御のブロック図図6・50 最大効率制御のブロック図なる。図6・48は制御ブロック図の一例である。. 弱め界磁制御(field weakening control)一般のベクトル制御誘導モータでは、励磁電流が一定に制御されるため、高速回転になると誘導起電力が増大し、インバータの出力可能な最大電圧とバランスした点で最高回転速度が決定する。このため、一定電圧のもとに広範囲の定出力運転を行うような場合には、回転速度がある程度増加したら励磁電流を小さくして界磁を弱めるような制御が必要となる。この制御を弱め界磁制御と呼んでいる。電気自動車用モータでは広範囲の定出力運転が要求される場合が多く、弱め界磁は重要な制御技術の一つとなる。図6・49は弱め界磁制御を実現するための一方式であり、回転速度の関数として励磁電流を変化させている。基底回転速度以下では一定の励磁電流を流し、基底回転速度以上では速度に反比例させて励磁電流を流して弱め界磁を行う。弱め界磁制御は磁束の変化を伴うため高速応答が得られにくく、電圧にある程度の余裕をもって制御する必要がある。誘導モータは励磁電流を自由にコントロールできるので、定出力運転に有利なモータである。. 最大効率制御電気自動車では一充電走行距離の観点から駆動モータの効率が重視される。一般に平坦路で定速走行する場合、モータは軽負荷となり効率が低下する。特に、通常のベクトル制御では励磁電流を一定に制御するため、軽負荷時に必要以上の励磁損が発生し効率が低下する問題がある。この点を解決するため、負荷に応じて励磁電流を調整し、つねに最高の効率点で運転しようとする制御方式が最大効率制御である。図6・50に制御ブロック図を示す。,はそれぞれ励磁分電流、トルク分電流で、両者の比を損失最小になるように決定することでモータ効率を最大にすることができる。. 速度センサレス制御ベクトル制御に必要な速度センサはモータに直結されるため、振動、じんあいなどの悪環境下におかれる。速度センサレス技術を用いると信頼性の向上、コストダウン、さらにモータ軸長の短小化が可能となり、一般産業では積極的4―229図6・48誘導モータベクトル制御のブロック図図6・51 PM モータのベクトル図に採用されるようになった。しかし、一般的な速度センサレス制御では低速域に駆動限界があり、停止の状態から高トルクが要求される電気自動車、ハイブリッド電気自動車用量産モータへの適用に至っていない。c.
