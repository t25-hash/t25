# 6・5・1 ハイブリッドシステムから見た電池への要求性能

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・5 蓄電装置

ガソリンエンジンに電気モータを組合わせたハイブリッド自動車(HV : hybrid vehicle)では、エンジンとモータの出力配分や、発電用あるいは駆動アシスト・回生用といった各モータの運転・制御方法によって、電池への要求性能は大きく異なる。電気自動車(electric vehicle)では1回の充電当たりどれだけ走れるかが最も大きな課題であり、電池性能としてはエネルギー密度(energy density)が最も重要視されるが、パラレルハイブリッド自動車では使用した電力を発電により補充し、つねに充電状態をある一定の範囲に保持しながら用いるため、電池容量はそれほど要求されない。代わって、発進加速アシスト時や減速時の回生制動等による瞬発的な充放電ができる高出力特性が要求される。まず電池に要求される出力は、各モータの最大出力とそれらの運転方法により決まる。電池の出力を決定する因子の中で、内部抵抗(inner resistance)は最も重要である。この内部抵抗は、出力のみならず入出力時の電池発熱量を決定する因子ともなる。ハイブリッドシステムの中で電池の冷却機能に割り当てられるスペースや重量は極めて限られたものにならざるを得ないため、電池に許容される発熱量は従来の電池と比較して大幅に小さくする必要がある。つまり発熱量の面でも従来にない出力密度の達成が必須である。次に、上記入出力が可能な電池の容量であるが、これは、モータによる発進加速での放電電力量および減速回生による充電電力量、さらにジェネレータでの発電電力量で決まる。これはドライバの運転方法にも大きく依存し、加速頻度が高ければ電池は放電傾向になり、長い下り坂等で回生頻度が高ければ充電傾向となる。それらをどれだけ吸収できるようにするかにより電池の必要容量が決まる。図6・55は、あるシステムを前提とした場合の種々の運転モードに対して、電池の充放電量がどの程度になるかをシミュレーションにより求めた一例である。これらの走行シーンに対応できるシステムとするためには、約300W・hの出し入れが可能なエネルギーの器として電池が機能しなければならないことを意味している。もちろんそれぞれの走行条件下で必要とされる入出力要求を満たせることが必要であるので、現実的には要求入出力を満たせるSOC(state of charge、充電状態)の範囲内でこのエネルギー容量を確保する必要がある。システムにより影響度は異なるが、このエネルギー容量を満たせない場合はシステム全体の高効率運転に制限を受ける頻度が高くなるため、燃費とのトレードオフとなる傾向にある。一方、電池の搭載スペースやコストといった視点からは極力搭載量を極小化する必要があるため、市場における走行パターンのど図6・55 HEV の走り方と充放電電力量図6・56 HEV 走行時の組電池への入出力とSOCの変化こまでをカバーしようとするかが現実には容量設計のポイントとなる。組電池の熱性能は、ハイブリッド電気自動車(HEV : hybridelectric vehicle)のように電池搭載量を極力制限したい場合においては特に重要で、その要求性能は最大入出力要求よりも電池の内部抵抗に厳しい要求を課す場合が多い。HEV における電池の発熱は電池の充放電効率の損失部分に相当する。図6・56は、実際のHEV の走行時の組電池への入出力とSOCの変化の一例を示している。スタート時点と同じSOCに戻る(時間)までの出力電力量の総和と入力電力量の総和の比が組電池の充放電効率であり、このケースでは約95%であるがこれは組電池に入力されたエネルギーの5%が熱に変換されていることを意味しているにほかならず、この発熱量をいかに抑えられるかが非常に重要である。一般に電池の発熱は、その内部抵抗によるジュール発熱、充放電反応時の化学変化による吸発熱、副反応に起因する発熱との総和で決まる。SOCをある範囲内に収まるように入出力を管理するHEVでの条件では、充放電反応時の化学変化による吸発熱はほぼ相殺されるため、電池発熱は他の2要因によりほぼ決定される。水溶液系の電池の場合には、SOCおよび温度の高い領域では充電時に水の電気分解が副反応として伴いやすく、充電効率が悪化する。密閉化されたこれらの電池では、電気分解されて発生した酸素、水素を水に戻すいわゆる密閉化反応によりこの充電効率のロス分がすべて熱に変換されるため、内部抵抗のジュール発熱に加えて、発熱量が増加する。特にニッケル水素などの電池では、高温領域で充電効率が低下する固有の現象があるため、高温雰囲気下では発熱量がいっそう増加するという悪循環が生じる。この現象は電池性能のばらつきの助長につながるため、高度な均一冷却構造などが必要となる。一方、リチウムイオン電池のような非水系電解液を用いる電池では、充放電時に副反応を伴わず充放電効率がほぼ100%に近いため、その発熱は内部抵抗のジュール発熱によりほぼ決定され、上記のような悪循環が生じることがない。しかしながら、その一方でリチウムイオン電池の場合には密閉化反応を有していないため後述(6・5・4・c項)のバイパス機能が必要であり、これらの点がニッケル水素電池とリチウムイオン電池のHEV に適用する場合の大きな特徴となっている。以上述べてきたように、HEV 用電池に求められるおもな要求性能は出力、容量、発熱であり、さらにはその性能を長期に保証するための寿命と組電池の制御しやすさが重要となる。そのほか、車両システムに適合させるための重要な用件としては、極低温時エンジン始動を可能にするための低温出力特性、モータに電力を供給するインバータ回路とのインピーダンス適合性なども非常に重要であるが、これらはHEV システム構成およびエンジンの仕様により大幅に異なり、HEV 用電池の一般論としては適さないため、ここでは省略する。
