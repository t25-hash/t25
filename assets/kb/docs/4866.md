# 6・5・2 鉛酸蓄電池(lead-acid battery)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・5 蓄電装置

鉛酸蓄電池(鉛蓄電池,鉛酸電池,鉛電池ともいう)は,エンジン始動用の電源として長年にわたってその座を他の電池に明け渡すことなく大量に普及してきた.自動車の寿命に比較して短い寿命しか有していないにもかかわらず,その安価なコストにより消耗部品として認知され,エンジン始動用途としては不動の地位を築いてい4―231図6・57 ニッケル水素電池の充放電反応機構図6・58 ニッケル水素電池の容量バランス模式図る.しかしながら,HEV 用の蓄電装置としては,その寿命の短さに加え,充電状態の検知性や劣化状態の把握が困難であることが実用化を難しくしている.ただし,その安価さは魅力的であり,寿命の改良や状態検知の技術開発が精力的に行われており,42V 電池での簡易型ハイブリッドシステムも一部市販されるに至っている.簡易型ハイブリッドシステムにおいても,回生入力に対する受入れ性が要求されるため,充電状態は中間SOCに維持することが必要となる.そのため,一般の始動用鉛酸蓄電池とは異なり,密閉型鉛蓄電池(valve regulated lead-acid battery)が採用されている.
