# 6・5・3 ニッケル水素電池(nickel-metal hydride battery)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・5 蓄電装置

a. ニッケル水素電池原理ニッケル水素電池の反応式は一般に次式で示される。正極:NiOOH+HO+e ⇔Ni(OH)+OH負極:MH욀욁+OH⇔M+HO+e電池:NiOOH+MH욀욁⇔M+Ni(OH)ここでM は水素急増合金、MH욀욁は合金中に吸蔵された水素を意味する。ニッケル水素電池では、充電時には水素が正極から負極へ、放電時にはその逆に移動するだけで電解液の量やその濃度の増減は伴わない。反応生成物は、充電時には正極では水酸化ニッケルNi(OH)が酸化されてオキシ水酸化ニッケルNiOOH と水が生成する(図6・57)。一方、負極では水が還元されて水素吸蔵合金の表面で水素原子が生成し、この水素原子が水素吸蔵合金中に吸蔵されて金属水素化物になる反応が起こる。放電反応はその逆である。ニッケル水素電池の密閉化原理について記す。正極と負極の容量バランスの模式図を図6・58に示す。密閉化原理は、基本的にはニッケルカドミウム電池と同様である。水素吸蔵合金負極の容量を正極のそれより十分大きく(負極の過剰量を充電リザーブと呼ぶ)することにより、過充電時に正極から発生する酸素ガスについては次式の化学反応式により水素吸蔵合金負極が還元して水に戻り、電池の内部圧力を一定に保つ結果、電池の密閉化が可能となる。正極:OH→1/4O+1/2HO+e負極:MH욀욁+1/4O→M+1/2HO正極で発生した酸素ガスを負極で迅速に吸収させるために、ガス透過性の優れたセパレータが使用されている。また、ガス透過性や負極表面での反応性を良好にするため、密閉電池内の電解液量の制限が必要である。また、図6・58に示したように負極は、その一部をあらかじめ充電状態(放電リザーブと呼ぶ)にしておく電池構成法が一般的に採用されている。放電リザーブを設けることにより、放電末期に正極容量規制となり、水素吸蔵合金負極の酸化やそれによる溶出を防止することができる。組電池化の際、直列セル間で充電4―232内燃機関状態の不均一が発生しても、この密閉化反応が充電末期に充電電流の一部消費を行い、結果として各セル間で充電状態を均一化するという極めて重要な機能を果たしている。b.
