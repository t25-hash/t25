# 5・3・1 地熱発電の歴史と現状

> **章**: 第5章 再生可能エネルギー
> **節**: 5・3 地熱

a.国外 天然蒸気を用いて世界初の地熱発電（geothermal
power generation）を行った場所はイタリアのラルデルロである．
当時ラルデルロでは天然蒸気中からホウ酸を取出していたが，1904
年にジノリ・コンチ公爵がこの蒸気を用いて0.75馬力の発電機に
より5個の電灯をともした．その後，同地区では1913年に出力
250kWの地熱発電所を建設して，工場の照明用に利用しており，
これが世界で初の地熱による商用発電である．同地区における発電
量は徐々に増大し，第二次世界大戦時には一時壊滅的な打撃を受け
たが，現在では世界第3位の設備容量をもつ．それ以降，1958年
にニュージーランドのワイラケイにおいて6.5MW，1960年にア
メリカのガイザーズにおいて10MW，1967年に旧ロシアのパウジ
ェトカにおいて11MWの地熱発電所が誕生している．このうち，
ニュージーランドにおける地熱発電技術は第二次世界大戦終了後に
イタリアに駐留していたニュージーランド軍によりもたらされたと
されている．2001年には世界中で休止中であるものも含めて23箇
国が地熱発電設備を有している．全世界の総発電設備容量は約3
191GWであり，地熱発電設備容量の合計は約8.3GWであること
から，後者は前者に対して約0.3%の割合を占めていることにな
る ．地熱発電設備容量の最も大きな国は，アメリカ（2228
MW)，次いでフィリピン（1931MW)，イタリア（923MW)，メ
キシコ（890MW)，インドネシア（590MW）であり，日本（549
MW）は世界第6位である．また，総発電設備容量に対して地熱
発電の占める割合が高い国は，順番にエルサルバドル（16.8 %)，
フィリピン（16.0%)，ニカラグア（15.8%)，アイスランド
（13.9%)，コスタリカ（9.8 %）となっている．
b.国内 国内における地熱発電の研究は1918年の大分県朝
日村において深さ約24m，直径約12cmの坑井を掘削し，この場
所で東京電灯(株)研究所長の太刀川年治博士が1925年に1.12kW
のタービンによる発電に成功したことから始まる．地熱発電に関す
る研究は第二次世界大戦により中断するが，戦後1948年には(株)
利根ボーリング［現：利根地下技術開発(株)］が湯ノ沢試験場で出
力3kWのタービンによる発電，1951年には工業技術庁（現：独
立行政法人産業技術総合研究所）が別府実験場で30kWの発電，
さらに1952年には(株)利根ボーリングが宮城県鳴子町において出
力20～30kWの発電に成功している．これらの試験的な発電の成
功を受け1953年には九州電力(株)が大分県大岳，1956年には東化
工(株)［現：日本重化学工業(株)］が岩手県松川において本格的な
地熱開発の調査を開始した．これらの調査の後，松川では1966年
に日本で初めての9.5MWの地熱発電所，さらに大岳では1967年
に11MWの発電所の運転を開始している．最近では1998年に八
丈島で3300kWの地熱発電所が運転を開始しており，現在の総発
電設備容量は事業用と自家用を合わせて約549MWに達している．
