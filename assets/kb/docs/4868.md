# 6・5・4 リチウムイオン電池(lithium ion battery)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・61 )の仕様として表している。

ハイブリッド自動車用のリチウムイオン電池は、その高出力密度とコストダウンの可能性から活発な研究開発が行われている。最初に市販されたリチウムイオン電池搭載ハイブリッド自動車は2000年4月に日産自動車より発売されたティーノHEV である。当時このティーノHEV に搭載されたリチウムイオン電池パックの最大出力は25kW であったが、その後の開発の進展により2005年時点では最大出力40kW 級、50kW 級のパックの開発が行われ、燃料電池車等に搭載されるに至っている。ここではこれらのリチウムイオン電池をおもな例として紹介する。a. 反応メカニズム例としてティーノHEV 用リチウムイオン電池の構成で説明する。この電池の正極にはスピネル型マンガン酸リチウム(LiMnO)、負極にはハードカーボン(非晶質系炭素)が用いられている。これらの材料はいずれも層構造を有しており、その層間にリチウムを電気化学的にドープ/脱ドープすることを利用して電池が構成されている。この原理を模式的に図6・66に示す。電池反応式は次式で示される。正極:LiMnO⇔Li+Li욪MnO+e負極:C+Li+e⇔LiC電池:LiMnO+C⇔Li욪MnO+LiC層間に存在するリチウムはイオン状態であることが化学分析により確認されている。これらの正負極材料は、リチウムイオンという有限の大きさのイオンが挿入脱離するため、リチウムイオンの収まるべきサイトと拡散可能なチャネルが必要である。このような条件を満たす正極材料としてはLiCoO,LiNiO,LiMnOなどが代表的で、HEV 用途としてはコストの観点から、LiNiO,LiMnOを主体とする材料の開発が主流となっている。一方、負極の炭素材料についてはグラファイトとハードカーボンの2種に大別できる図6・65 走行時のニッケル水素電池パックの入出力と温度変化図6・66 リチウムイオン電池の充放電機構4―233が、HEV 用としてはハイレートの充電受入れ性や後(6・5・4・c項)に述べる充電状態の検知性の観点でハードカーボンがより有効である。b. HEV 用リチウムイオン電池構造図6・67にはティーノに搭載されたセルおよびモジュールの写真とその仕様を示す。セルは捲(けん)回された電極を内蔵する円筒形の構造であり、3.6A・hの定格容量を有する。HEV 用として活物質粒子の微粒化、電極厚みの低減と大面積化など出力重視の設計とすることにより、セルの出力密度は2000W/kg(SOC50%)を達成している。上記モジュールには48個のセルを直列に接続し、セルの電圧監視機能と容量バランス調整機能を有するセルコントローラ(6・5・4・c項参照)が内蔵されている。このリチウムイオン電池の車両への搭載構造を図6・68に示す。電池パックは、居住性やスペースユーティリティを犠牲にしないため、車両の前席下および後席足元の床下に搭載されている。床下搭載のため、モジュールの搭載構造や冷却風の経路などにおいてエンジンからの排熱の影響を受けにくくするさまざまな工夫がなされている。また、モジュールの筐体はコネクタを含め防水構造がとられている。セルの構造としては、当初はニッケル水素電池と同様に電極を捲回した円筒形が主流であったが、その後角形の缶構造なども開発され、さらには図6・69に示すようなラミネートタイプと呼ばれる樹脂とアルミニウムの積層フィルムを外装として用いるセルの開発も進められている。このようなラミネートタイプの構造は完全な密閉構造となるため、内圧の上昇につながる電解液の分解などの副反応寸法260×540×160重量20kg電圧120~200V容量3.6A・h最大出力>12.5kW図6・67 FCV)に搭載されている。c. 組電池(battery pack)管理リチウムイオン電池の組電池モジュールには、全セルのSOCを監視し、充電状態を均一化させる機能を備えた各セルの状態を監視するセルコントローラが内蔵されている。図6・71にはセルコントローラの回路の概念図を示す。別体として設けられているバッテリコントローラは、このセルコントローラからの情報と組電池総電圧、電流の情報からリアルタイムにSOCと入出力可能パワーを演算し、車両コントローラに情報を伝達している。車両コントローラはこの情報をもとに、各モータの出力、発電量、回生許容電力等を統合的に制御している。HEV 仕様のようにエネルギー容量の少ない電池を最大限に用いて車両性能を引き出すためには電池の正確なSOC管理が必要であるが、このリチウムイオン電池は開放電圧とSOCが図6・68に示すような相関関係にあるため、電圧値からSOCを正確に検出することができる。これは負極にハードカーボン系の材料を用いているリチウムイオン電池特有の特性である。また図6・72に示すように、各SOCにおいて放電電流と電圧降下代はほぼリニアな関係にある。走行中の電流電圧の情報をサンプリングし、その一次回帰直線を電流零に外掃することによりその時点での開放電圧、結果的にSOCを精度よく推定することが可能である。また、逆に最大電流許容値に外掃することにより最大出力許容値の推定もリアルタイムで可能である。リチウムイオン電池は、6・5・1項で前述したように水溶液系の電池のような水の電気分解を副反応として伴わないため、高温領域で充電効率の悪化により発熱が助長されるというような悪循環がなく、また充放電時の効率がほぼ100%のため本質的にばらつき図6・70 ラミネートタイプリチウムイオンモジュール図6・71 セルコントローラ概念図は進行しにくい電池である。しかしながら、その反面、わずかな温度の違いによる自己放電量の差が蓄積すると電池自体にはそれを修復する機能を有してはいない。そのため、セルコントローラには、バイパス回路によるばらつき補正の機能をもたせており、組電池の能力をつねに最大限に引き出せる状態に保つことを可能としている。図6・73には、HEV 車両にて約7万km走行後の組電池96セルすべての電圧データの分布を示しており、セルコントローラの機能によりセル電圧の分布を一定範囲内に収めることが可能である。
