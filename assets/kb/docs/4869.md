# 5・3・2 地熱プラントの概要

> **章**: 第5章 再生可能エネルギー
> **節**: 5・3 地熱

a.地熱貯留層（geothermal reservoir ) 地下深部から上昇し
てきたマグマは，数十kmから数kmの深さにマグマだまりを作
る．このマグマだまりは通常600℃以上の温度があり，数万年から
数十万年にわたって安定して多量の熱を放出する．一方，地表から
地下深部へ浸透した雨水や海水はマグマだまりからの熱によって加
熱され，高温の地熱流体（熱水や蒸気）として岩盤の割れ目やすき
ま（貯留構造）に蓄えられる．これらの熱，地熱流体，貯留構造は
地熱貯留層を構成する3要素と呼ばれる．
b.地熱井（geothermal well ) 地熱貯留層から地熱流体を取
出すために地表より生産井が掘削される．地熱井掘削のための技術
や機器は石油鉱業で用いられるものが転用されているが，石油井と
比較すると地熱井は高温で硬質の岩盤を対象とする，あるいは地熱
貯留層は高温・高圧で腐食性の流体を含むなど，地熱井特有の問題
がある．生産井の掘削深度の範囲はおおよそ500mから3000m
程度であるが，近年，掘削深度がより大きくなる傾向にある．ま
た，生産井から得られた地熱流体の一部は蒸気として大気中に放散
されるが，残りは熱水となる．わが国においてはこの熱水は全量地
下に戻されるが，そのための坑井を還元井（reinjection well ）と
呼ぶ．
c .地熱流体（geothermal fluid) 生産井坑口において熱水を
伴わず乾燥飽和蒸気あるいはわずかに過熱した蒸気が得られる蒸気
卓越型の地熱貯留層と，飽和蒸気と熱水とが混合して得られる熱水
卓越型の地熱貯留層とがある．両者の違いは地熱貯留層への水と熱
の供給バランスによって説明でき，水の供給量が熱の供給量と比較
して多いと熱水卓越型になりやすい．わが国では松川等一部の地熱
発電所を除いてほとんどが熱水卓越型である．地熱流体は地盤から
種々の成分を取込むため，場所によって濃度は異なるが生産井から
の蒸気中にはCOやHS等の不凝縮性ガスが含まれる．また熱水
中にはCl ，SO ，Na，K，Ca，Mg，Fe，Al ，Si ，B，As等の化
学成分が含まれる．これらの成分のうちCaやSiは熱水温度が低
下するあるいはpHがアルカリ側に変化すると析出することがあ
り，配管内や還元井で目づまりを起こす原因となる．また熱水の中
には強酸性のものもあり配管や機器の腐食を引き起こすため，これ
らの除去技術や対策技術の開発が大きな課題となっている．
d.発電方式 蒸気タービンは背圧式（back pressure tur-
bine）と復水式（condensing turbine）とに大きく分類することが
できる．背圧式では生産井からの乾燥蒸気でタービンを回し，ター
ビンからの排気は直接大気へ放出する．一方，復水式はタービンの
排気を復水器により復水することで高真空とし，タービン内で蒸気
を十分に膨張させることで効率を上げることができる．背圧式は復
水設備が不要であるために安価ではあるが，単位出力当たりの蒸気
消費量が復水式の2倍以上となるため，特別の場合を除いて用いら
れない．図5･33に天然蒸気背圧タービン方式を示す．
生産井から得られる地熱流体が熱水卓越型の場合には，地熱流体
を気水分離器に入れ蒸気と熱水とを分離した後に，分離した蒸気で
タービンを回す．排気は復水器に導いて冷却水で冷やすことにより
凝縮し，凝縮水は冷却水の補給水として使用する．この方式を図
5･34に示す．わが国ではこの方式による地熱発電所が一般的であ
り，冷却水を供給するために設置される冷却塔は地熱発電所に特有
のものである．図5･34で示した熱水分離型蒸気復水発電方式にお
いて気水分離器から分離された熱水の量が多く，温度が十分に高い
場合には，この熱水をさらにフラッシャで減圧沸騰させることによ
り低圧の蒸気を得ることができる．この低圧蒸気をタービンの中間
段に導入することにより，発生する電力を増加させることができ
る．図5･35で示すこの方式はダブルフラッシュ蒸気復水タービン
方式と呼ばれる．なお，図5･34で示した発電方式をダブルフラッ
シュと対応させる意味でシングルフラッシュ蒸気復水タービン方式
と呼ぶこともある．
熱水の温度が低くフラッシュ方式を用いても少量の蒸気しか得ら
れない場合には，図5･36で示す低沸点媒体発電方式が用いられる．
これは水よりも沸点の低いブタンやプロパンを低温熱水によって加
熱し，媒体蒸気でタービンを回す方式である．この方式はオーマッ
ト工業(株)により商業化され，同社の0.25～10MW規模のモジュ
ラ型発電機が国外で稼動している ．
