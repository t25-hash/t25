# 6・5・5 キャパシタ(capacitor)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・61 )の仕様として表している。

電気二重層キャパシタは、分極性電極と電解液界面に形成される電気二重層と呼ばれる極めて薄い電化配列層の容量を利用したものである。電気二重層キャパシタの素子内の充放電要素は、正負極に用いた活性炭の電極表面に形成されるイオン吸着層の吸脱着を利用したものである(図6・74)。最大の特徴は、電池のような化学反応を伴わないイオンの吸脱着という物理現象のみを応用しているため、電荷移動において化学反応の抵抗が伴わず、内部抵抗が電池に比較して温度の影響を受けにくい点にある。また、化学反応を伴っていないため、電解液の分解が発生しない電位範囲(電位窓という)に動作電圧をコントロールできれば、長寿命が期待できる。一方、欠点としては電池などに比較して大幅に蓄電できるエネルギーが小さいことであり、この蓄電装置の冒頭で述べたような300W・hもの最大入出力可能なエネル図6・72 HEV 用リチウムイオン電池(組電池)の開放電圧-SOC、および放電I-V 特性図図6・73 HEV 車両の走行距離による組電池内全セルの電圧分布の推移図6・74 電気二重層キャパシタの動作原理図6・75 大容量電気二重層キャパシタ表6・18 大容量電気二重層キャパシタセルの仕様(例)静電容量1300F定格電圧2.5V直流内部抵抗2±0.5mΩ形状寸法150mm×70mm×23.6mm重量390gギー容量レンジを必要とするHEV システムに対応することは不可能である。しかしながら、低温出力重視であるが蓄電エネルギーを大量に必要としないシステムや、あるいは鉛蓄電池とハイブリッドされた複合電源システムにおいてはその特徴を有効に発揮できる可能性があり、その方面でのアプリケーションをねらった開発に焦点がおかれている。電解液としては、水溶液系電解液、有機溶媒電解液、無機固体電解質、有機ポリマゲル電解質などが提案されているが、HEV 用としては有機電解液を用いたタイプが多く開発されている。理由は、キャパシタの耐電圧を高くできることにあり、電解液の種類にもよるが2.3~3.3V/cell程度の他動電圧が実現できる。有機電解液として具備すべき特徴としては、酸化側、還元側の分解電圧が高く、広い電位窓をもつことと、広い温度範囲で高い電気伝導度をもつことが要求される。電極材料は、それ自身は電気化学的には不活性で理想分極性を示し、電子導電性に優れ、さらに比表面積が大きいことが要件である。実用化された有機電解液系電気二重層キャパシタでは、おもに、やしがら、コークスやピッチなどの天然由来物、フェノールなどの合成樹脂を原料とする大表面積の活性炭材料が用いられている。図6・75には、高出力パワーパルス電源、HEV 用途向けに開発された電気二重層キャパシタの写真、表6・18には、代表的な仕様を示す。
