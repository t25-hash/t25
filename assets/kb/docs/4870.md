# 6・6・1 フライホイールバッテリ(flywheel battery)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・6 電気以外のエネルギー蓄積装置

フライホイールは慣性モーメントの大きさを利用した機械要素であり、内燃機関のフライホイールがトルク変動の平滑化を目的としているのに対し、エネルギーの蓄積を目的としたものをフライホイールバッテリと呼ぶ。エネルギーを回転体の運動(回転)として蓄え、回転の加減速によりエネルギーの入出力を行うという原理である。エネルギーの入出力方法には電気的方法と機械的方法の2とおりが考えられ、その機構として前者ではモータ(motor)/発電機を用い、後者ではクラッチなどが用いられる。エネルギー蓄積の原理からフライホイールバッテリは従来の化学バッテリに対して機械式バッテリ、あるいは電気機械式バッテリと分類される。回転体がもつ運動エネルギーは慣性モーメントと回転数の二乗との積に比例する。回転数の上昇に伴い蓄えられるエネルギーが増加するため、同一の体積、重量でも回転数が高いほど多くのエネルギーを蓄えられる。図6・76に比重7.8の回転体がもつエネルギーのグラフを示す。4―235図6・76 フライホイールの回転エネルギー図6・77 フライホイールバッテリの構成要素図6・78 炭素繊維複合材料製ロータ従来はおもに低回転なフライホイールと機械的な入出力方法が利用されたが、現在行われている技術開発では多くの場合、数万から数十万minに至る高い回転数を目標としており、高強度な材料と電気的入出力方法を利用したフライホイールバッテリが主流である。フライホイールバッテリの利点は(1) 寿命が長い(2) 環境温度に対する性能変化が少ない(3) 容量検知が容易であるなどであり、電圧が自由に設計できることや容量と出力が独立して選択できることも利点である。今後解決すべき技術課題としてバッテリ(battery)の自己放電に当たる無負荷時損失の大きいことが挙げられ、車両等の移動体に搭載する場合に考慮すべき項目としてジャイロモーメントの影響が挙げられる。a. フライホイールバッテリの構成要素フライホイールバッテリの構成要素は図6・77に示すようにロータ、モータ/発電機、ケース、軸受の大きく四つに分類できる。. ロータエネルギーを貯蔵する主体であり、高回転に耐えられる強度と構造が求められる。回転体の単位重量に蓄えられるエネルギーは材料の引張強度を密度で除した値である比強度に比例するため、エネルギー密度の高いロータを得るためには比強度の高い材料を選択することになる。ロータ材料として従来は鉄系材料が使用されていたが、現在では多くのフライホイールバッテリで炭素繊維複合材料(carbon fiber reinforced plastics)が使用されている(図6・78)。その製法としては繊維を周方向に巻くフィラメントワインディング成形と呼ばれる手法が一般的である。現在、周速で4―236内燃機関図6・79エリコン社のジャイロバス1000m/sを超えるロータの設計・製造が可能になってきており、一部では2000m/sを超える例もある。. モータ/発電機電気エネルギーと運動エネルギーの相互変換を行う。通常、ロータと同一軸上に配置されるため、モータの回転部が高回転に耐えられることが重要となる。また、フライホイールバッテリの充放電効率はインバータを含めたモータ/発電機システムの効率が大きく支配するため、効率の高いDCブラシレスモータを採用する場合が多い。. ケースケースの重要な機能は真空の維持である。フライホイールを空気中で高速回転させると空気との摩擦損失が生じ、エネルギーの消費とそれに伴う温度上昇が起こるため、回転部を真空雰囲気に保ちこの損失を低減する。長期にわたる真空維持を実現するためのケースおよびシール技術が重要となる。. 軸受フライホイールバッテリにおいて難度の高い技術要素の一つである。超高速回転部を低損失で安定して支持することや真空中での機能が要求されるため、磁気軸受(magnetic bear-ing)が多く使用される。b. 車両への適用車両へのフライホイールバッテリの適用には、1940年代後半、スイスのエリコン社が電気自動車路線バスに使用した例がある(図6・79)。直径1.6m、重量約1500kg,3000minのフライホイールバッテリを搭載し、約800mごとに約80秒の充電を行って走行した。自己放電の大きいフライホイールバッテリの活用法として、一定時間ごとに充電を行える路線バスへの適用は有効な手法である。近年になり、乗用車タイプの電気自動車(electric vehicle)やハイブリッドレーシングカー、バスへの搭載、さらには鉄道車両への適用が提案された。また、アメリカではPNGV(Partnership fora New Generation Vehicle)の中でフライホイールバッテリは先進バッテリ(battery)、キャパシタ(capacitor)と並び主要な要素技術として研究が進められ、1997年の報告書の中で「ポテンシャルは高いが開発に時間を要する技術」と結論づけられた。現時点で車両に搭載され実用化されているフライホイールバッテリはないが、上記提案の一部は現在も研究が継続されている。
