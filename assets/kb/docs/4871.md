# 6・7・1 概説

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・7 燃料電池

a.原理 燃料電池（fuel cell）は，燃料および酸化剤が外部
から連続的に供給され，燃料がもつ化学エネルギーを電気化学反応
によって直接電気エネルギーに変換し，連続的に発電する装置と定
義される．固体高分子形燃料電池を例にとると，電解質膜として厚
さ数十 mから百数十 m程度のフッ素系イオン交換樹脂膜を使
い，その両側に触媒（catalyst）としての白金の超微粒子を分散さ
せて電極を形成する．図6･80に示すように，まず燃料極（fuel
electrode，負極，水素極）では，白金の触媒作用により，水素は
イオン化されてプロトンと電子になる．ここで，電解質膜はプロト
ンのみを選択的に透過し，電子は透過しない性質をもつため，電子
は外部回路を通って空気極（air electrode，正極，酸素極）に到達
する．一方，空気極では，白金の触媒作用によって，酸素と，電解
質膜を透過してきたプロトンと，外部回路を経由してきた電子とが
結合して水を生成する．
燃料極の反応は，H→ 2H＋2e
空気極の反応は，1/2O ＋2H ＋2e→ HO
燃料電池の全体での反応は，H ＋1/2O→ HOとなる．
燃料電池は，燃料がもつ化学エネルギーを電気化学反応によって
直接電気エネルギーに変換する点では，密閉式乾電池と同じである
が，燃料および酸化剤が外部から連続的に供給される限り，半永久
的に電気エネルギーを取出すことが可能な点が異なる．つまり，密
閉式乾電池が電気エネルギーを蓄える機能をもつのに対して，燃料
電池は化学エネルギーを電気エネルギーに変換する機能をもつもの
とみなすことができる（図6･81)．
