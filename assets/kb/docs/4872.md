# 6・7・2 燃料電池スタック

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・7 燃料電池

燃料電池が発生する電圧は,単セルで1V 程度であり,実際には大きな電圧を得るために,この単セルを積み重ね直列につなぐ必要がある.このように単セルを複数個積層したものを燃料電池スタック(fuel cell stack)と呼ぶ.実際の燃料電池スタックは,単セル4―239図6・85 固体高分子形燃料電池の単セルの構造図6・86 燃料電池スタックの構造に加え,セパレータ,冷却板,集電板,絶縁板,締付板,マニホルドなどの付属品が一体となって積層されて燃料電池スタックを構成している.図6・85に固体高分子形燃料電池の単セルの構造を,また,図6・86に燃料電池スタックの構造を示す.a. セパレータ(separator)
