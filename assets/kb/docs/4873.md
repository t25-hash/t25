# 5・4・11 風車用翼型

> **章**: 第5章 再生可能エネルギー
> **節**: 5・4 風力

風車はその作動原理によって揚力型と抗力型に分類される．しか
し，発電事業を目的とする風車はエネルギー変換効率が高い揚力型
が適している ．この揚力はブレードの翼型断面に発生する．し
たがって高性能風車の開発には高性能翼型の開発が不可欠であり，
ドイツのヴォルツマン翼型，アメリカのSERI ・NRELシリーズ，
オランダのデルフト工科大学TUシリーズ，デンマークRISO研
究所の翼型ファミリー，スウェーデンのFFA翼型，産業技術総合
研究所のMEL翼型などが挙げられる．風車用翼型に求められる特
性は，高揚力係数，高揚抗比，安定した失速特性，レイノルズ数非
依存性，高性能圧翼などである．つまり，空力性能と構造強度の双
方を担う翼型が求められる．
