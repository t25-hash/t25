# 6・7・1 c項で説明したように、

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・7 燃料電池

燃料極側には水素あるいは水素を含む改質ガスが、空気極には酸素あるいは空気が流されるため、単セルを積層する場合においては、燃料極のガス(燃料ガスと呼ぶ)と、空気極のガス(酸化剤ガスと呼ぶ)の混合を防ぐために二つの単セルの間に仕切板を設ける必要がある。このような仕切板をセパレータと呼ぶ。セパレータは、隣り合う単セル間を電気的に接続するコネクタとしての役目も担っている。燃料電池のセパレータには、低い電気抵抗、良好な熱伝導性、低い水素透過率、高い耐衝撃性、高い耐割れ性などが求められるが、さらに燃料電池スタックの小型化・軽量化のためには、セパレータは薄くて軽いことが望まれる。また、セパレータはその両面あるいは片面に、ガス流路あるいは冷却水流路を形成することになるため、加工性、成形性に優れることも求められる。固体高分子形燃料電池セパレータとしては、焼成グラファイトカーボン板を切削加工したものが使われてきたが、先に述べた特性を満たすさまざまなセパレータが新たに開発されている。具体的には、カーボン粉をフェノール樹脂などの熱硬化性樹脂に混練したうえで熱プレス加工により成形したセパレータ、カーボン粉をポリフェニレンサルファイド(PPS)などの熱可塑性樹脂に混練したうえで熱プレス加工により成形したセパレータ、ステンレスやチタンなどの金属板の表面に耐腐食性の表面処理を施したセパレータなどである。多くの場合、セパレータは燃料ガスと酸化剤ガスの混合を防ぐだけでなく、膜-電極接合体(MEA : membrane-electrode assem-4―240内燃機関bly)に燃料ガスと酸化剤ガスをそれぞれ均一に流すための、ガス流路のための溝が設けられている。このようなガス流路が設けられたセパレータはリブ付きセパレータとも呼ばれる。リブ付きセパレータにおいては、単セル面内を流れる燃料ガスと酸化剤ガスを流す方向の組合せがいくつかある。(1) 直交流(クロスフロー):単セル面内で、燃料電池ガスと酸化剤ガスを直交する方向に流す方法(2) 対向流(カウンタフロー):単セル面内で、燃料電池ガスと酸化剤ガスを逆向き方向に流す方法(3) 並行流(コフロー)または平行流(パラレルフロー):単セル面内で、燃料電池ガスと酸化剤ガスを同じ方向に流す方法(4) リターンフロー:単セルの面内で、流れの方向を変え、ガス入口から入った方向と逆方向に流す方法(5) サーペンタインフロー:単セルの面内で、流れ方向を何回か反対方向に変えて流す方法b. 冷却板燃料電池の電極を冷却するための冷媒を流す溝をもった板で、単セル間に挿入される。空冷式燃料電池においては、冷却用の空気が流される。c. 集電板燃料電池スタックにおいて、積層した単セルの端部に設置し、燃料電池スタックの外部に電流を取出すための板。一般に電気抵抗の低い金属板により作られる。燃料電池スタックの端子電圧は、この集電板間の電位差であるが、単セルにおけるセル電圧と混同するのを防ぐために、スタック電圧あるいはセルスタック電圧と呼ばれる。d. マニホルド燃料電池スタックにおいて、各単セルへ反応ガスを供給するためのヘッダ部分、または、各単セルから排出される反応ガスをまとめるためのヘッダ部分をマニホルド(mani-fold)と呼ぶ。ここでマニホルドが燃料電池スタックの内部に設置されている場合を内部マニホルド、燃料電池の外側にマニホルドが取付けられている場合を外部マニホルドと呼ぶ。e. 締付板複数の単セルの積層体からなる燃料電池を一体とするために、積層体の両端部に設置し、締付圧力を均一に加えるために用いる。燃料電池を締め付けて一体とする方法としては、積層した単セル全体を貫通するボルトとナットで締めて固定することが一般に行われるが、金属製のベルトで一体化する方法や、接着剤や溶接により複数の単セルを一体化する方法も行われている。f. 絶縁板燃料電池スタックの単セル積層体の端部に設けられた集電板と、締付板の間に挿入し、両者の電気絶縁を行う。g. 収納容器一般に燃料電池スタックは金属製あるいは樹脂製の容器に収納される。これは、燃料電池スタックの運搬のしやすさ、取付けや設置のしやすさといった便宜上の点だけでなく、燃料電池スタックへの接触による感電からの保護という役割もある。以上に述べた燃料電池スタックの構造は、固体高分子形燃料電池を例に説明したが、燃料電池の形式によっても異なっている。例えば、リン酸形燃料電池では電解質として液体のリン酸(HPO)を用いるが、このリン酸を保持するために多孔質の炭化ケイ素(SiC)板にリン酸をしみこませた状態で使っている。またリザーバと呼ばれるリン酸を保持した板を設け、電解質のリン酸が反応により減少したときに毛細管現象で電解質に対してリン酸を補給する機能をもたせている。このようにリン酸形燃料電池では液体の電解質を取扱うために、単セルは平面状に積層される構造になっている。固体酸化物形燃料電池は、運転温度が900~1000°Cの高温である一方、電解質のイットリア安定化ジルコニア(YSZ)の熱伝導性が低いため、温度分布による熱応力の影響を強く受けることから、平板形のセル構造に加え、円筒形のセル構造も用いられている。
