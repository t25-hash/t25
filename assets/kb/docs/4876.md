# 6・7・4 改質装置

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・7 燃料電池

燃料電池は燃料極の燃料ガスとして水素を使うが、水素は地球上に単体ではほとんど存在しないため、化石燃料、水、バイオマスなどの原燃料から水素を製造する必要がある。燃料、製造方法、製造場所の組合せにより、基礎研究レベルのものも含めれば、さまざまな手法の検討が進められている。この中で炭化水素系の原燃料から化学反応によって水素濃度の高いガスを得ることを改質といい、改質反応を起こすための装置を改質器(reformer)と呼ぶ。なお、燃料電池の燃料極に供給するガスを燃料ガスと呼ぶことから、それと区別するために、水素を得るためのもとになる燃料は原燃料あるいは原燃料ガスと呼ばれる。a. 水蒸気改質(steam reforming)炭化水素などの燃料を、触媒を用いて水蒸気と改質反応させ、水素濃度の高い混合ガスを得る方式。この混合ガスは改質ガスと呼ばれる。改質ガスには、水素のほか、二酸化炭素、一酸化炭素などが含まれる。この方法で用いる反応は吸熱反応であり、反応を継続させるために改質器に熱を供給し続ける必要がある。メタンの水蒸気改質反応を次に示す。CH+HO→CO+3Hb. 部分酸化改質(partial oxidation reforming)炭化水素などの燃料に空気を混合し、燃料の一部を燃焼させたときに生じる熱を熱源として、同時に発生する水蒸気を残りの燃料と改質反応させ4―242内燃機関て水素濃度の高い改質ガスを得る方式。この改質ガスには、水素のほか、二酸化炭素、窒素、一酸化炭素などが含まれる。この方法では、水素の一部が燃焼に用いられるために、水蒸気改質に比べて改質効率が低くなるとともに、得られた燃料ガス中の水素濃度も低くなるが、反応が発熱反応であるため、外部から熱を加える必要がなく、改質器を短時間で昇温させることができるので、改質器の起動時間が短いという特徴をもつ。メタンの部分酸化改質反応を次に示す。CH+1/2O→CO+2Hc. 併用改質あるいはオートサーマル改質(auto thermal reform-ing)吸熱反応である水蒸気改質と発熱反応である部分酸化改質の両方を組合わせて、熱的に自立した運転条件で改質反応させて水素濃度の高い改質ガスを得る方式。この改質ガスには、水素のほか、二酸化炭素、窒素、一酸化炭素などが含まれる。この方法では、改質器が熱的に自立し、改質器の外部との熱のやりとりが不要になるため、改質器の加熱装置や冷却装置を省く、あるいは小型化することができ、コンパクトな改質器を作ることが可能になる。改質装置には次のような特性が求められる。(1) 重量が軽く、体積が小さいこと。(2) 起動、停止が速やかに行われること。(3) 負荷変動に追従して幅広い動作領域で高い効率で運転できること。(4) 得られる改質ガス中の一酸化炭素濃度が低いこと。このような特性を実現するために、実際の改質装置は図6・90に示すように、脱硫器、改質器、一酸化炭素変成器、一酸化炭素除去器および制御系統の部分から構成されており、燃料と製造方法の違いにより、その中の一部をもたないもの、あるいは一酸化炭素変成器が複数の変成器からなるものがある。. 脱硫器(desulfurizer)燃料に硫黄分が含まれる場合、硫黄分が改質器の改質触媒や燃料電池の燃料極触媒を被毒させるため、あらかじめ除去しておく必要がある。硫黄分は、軽油、ナフサなどの石油系燃料に含まれるほか、都市ガスや液化プロパンガスの付腐臭剤としても含まれるため、こうした燃料を使用する場合には、脱硫器が必要となる。一方、合成燃料やメタノール、ジメチルエーテル(DME)などには硫黄分が含まれないため、こうした燃料を使用する場合には、脱硫器を設ける必要はない。(1) 水添脱硫器:ニッケル-モリブデン酸化物触媒やコバルト-モリブデン酸化物触媒によって燃料中の硫黄分と水素を反応させて硫化水素とし、さらに酸化亜鉛と反応させて硫化亜鉛とすることで硫黄分を吸着除去する脱硫器(2) 吸着脱硫器:燃料中の硫黄分を活性炭や酸化マンガンあるいは酸化亜鉛などの吸着材に吸着させて除去する脱硫器。 一酸化炭素変成器改質反応によって発生した一酸化炭素を、改質器の後段で水蒸気とシフト反応させて二酸化炭素と水素に変換する反応器。シフト反応器とも呼ばれる。反応温度により、高温(400~450°C)で反応させる高温一酸化炭素変成器と、低温(200~250°C)で反応させる低温一酸化炭素変成器に分けられる。. 一酸化炭素除去器運転温度の比較的高いリン酸形燃料電池では、改質ガス中の一酸化炭素濃度は1%程度まで許容される。一方、運転温度の低い固体高分子形燃料電池においては、燃料極の触媒として、耐一酸化炭素被毒性に優れた合金触媒(例えば白金ルテニウム系触媒)を用いた場合でも100ppm以下、白金触媒を用いる場合には1ppm以下にまで一酸化炭素濃度を低減させる必要がある。このため、固体高分子形燃料電池を用いた燃料電池システムにおいては、改質装置に一酸化炭素変成器に加え、一酸化炭素除去器を設ける必要がある。一酸化炭素除去器は一酸化炭素選択酸化部とも呼ばれ、改質ガス中に、ごく少量の空気を吹き込み、白金触媒上で一酸化炭素を低温(100~200°C)で酸化させて、燃料極の触媒に影響を与えない二酸化炭素に変換している。また、一酸化炭素分離膜を使って改質ガス中の一酸化炭素を選択的に分離除去する方法も使われる。
