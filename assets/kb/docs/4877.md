# 6・8・1 乗用車の例1(トヨタハイブリッドシステム)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・8 ハイブリッド電気自動車の適用例

a. 概要トヨタ自動車では、世界初の量産ハイブリッドシステム、THS(Toyota Hybrid System)を搭載したプリウスを1997年12月に生産・販売を開始した。2000年にはハイブリッド用電池として搭載性に優れた角形電池を開発するとともに、排出ガス低減・燃費性能と動力性能を両立させたハイブリッドパワートレインに改良し、国内のほか北アメリカ、ヨーロッパへの輸出を開始した。2001年にはCVT を組合わせたTHS-Cと電気式4WD であるE-Fourを搭載したエスティマハイブリッドならびに42V 系電源システムによるマイルドハイブリッドシステムTHS-M を搭載したクラウンの販売を開始し、多様な形でのハイブリッド技術の製品化を進めた。2003年には、昇圧回路を採用し、モータシステムの電源高電圧化(500V)と制御系の改良を行うことにより動力性能の向上とシステム効率をさらに高めたTHS IIを開発、新型プリウスに搭載し、日本、北アメリカ、ヨーロッパへの導入を拡大した。このTHS IIでは排出ガス低減、燃費性能の環境性能だけでなく、動力性能を大幅に向上することにより、クルマ本来の楽しさであるFun to Driveとの両立を高め、従来車と遜色のない車両性能を実現している。2005年にはSUV(sport utility vehicle)用THS IIとして、モータシステム電源の高電圧化(650V)、リダクションギヤの採用ならびにモータの高回転化による新型ハイブリッド用トランスミッションを開発し、V 6エンジンと組合わせたハリアーおよびクルーガーのハイブリッド仕様の販売を開始した。これらのハイブリッドシステムの開発と搭載車種の拡大により、トヨタ自動車におけるハイブリッド自動車は2005年3月現在で9車種、累積販売台数は36万台になっている。トヨタハイブリッドシステムのおもな仕様、諸元の推移を表6・23に示す。b. THS2003年に発売されたTHS IIは、THS システムにインバータ回路と電池の間に昇圧回路を追加し、システム電圧の高電圧化(500V)によるモータシステムの高出力化を行うことにより、モータシステムの高効率化を図った(図6・98)。モータ制御では、従来の低速域と高速域のPWM 制御方式に加え、中速域に可変調制御を追加し、中速域での出力を最大約30%向上している。また、昇圧制御では、システム損失を最小限とするため、昇圧電圧範囲を最適制御する可変電圧制御を採用した。この可変電圧制御はモータおよび発電機の動作状態に応じて電圧を連続的に可変している。エンジン出力および効率の向上と併せて出力性能の向上を達成した。d. SUV 用THS2005年に発売を開始したハリアーハイブリッドおよびクルーガーハイブリッドは、V 形6気筒3.3Lの3MZ-FEエンジンと新開発のFFハイブリッド用トランスミッションを搭載したSUV 用ハイブリッドシステムである。おもな特徴は、1高電圧化設計(650V)されたモータおよび発電機ユニットならびにパワーコントロールユニットからなる電気システム、2動力分割機構にリダクションギヤを追加することで高速・高出力化設計された駆動ユニット、3高出力性能、小型化設計のNi-MH(ニッケル水素)電池システムである。後輪駆動用パワーユニットには、エスティマで採用されている電気式4WD(E-Four)の出力性能を改良し、前輪駆動システムと組合わせて4WD 仕様としている(図6・99,6・100)。電池には従来の樹脂型電槽を金属型電槽に変更し、冷却性能の向上と電池内部抵抗の低減により、高出力性能を実現した密閉式角形ニッケル水素電池を採用した。e. THS-C2001年にエスティマに搭載し、発売を開始したTHS-Cはエンジンとモータ、CVT(continuously variabletransmission、無段変速機)遊星歯車機構による動力切換機構からなるパラレルハイブリッドシステムである(図6・101,6・102)。後輪をモータ駆動する電気式4WD の構成をもち、前輪駆動用のモ図6・98 THS IIシステム(プリウス、2003年)4―245ータで発電した電力をリヤモータに使うという意味でシリーズハイブリッドの機構を有している。CVT とパワーステアリングの油圧源として、両方のオイルポンプを駆動するモータと、エンジン始動のためのスタータモータを備えている。走行中のエンジン始動は、補機駆動用ベルトを介して行われる。f. THS-M(トヨタマイルドハイブリッドシステム)2001年にクラウンに搭載発売されたTHS-M は、補機駆動ベルトにより駆動される小型モータ(発電機兼用)と駆動用インバータおよび42V 電源で構成されている。駆動モータはエンジン駆動力の補助動力を供給するほか、回生時にエネルギー回収を行う。停車時のみにエンジン停止(エコノミーラン)することから、走行中のエンジン停止(モータ走行)を行うプリウス、エスティマのシステムと区別するため、マイルドハイブリッド(mild hybrid)と呼ばれている(図6・103)。
