# 1・1・2 人類初期のエネルギー変換

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・1 機械工学におけるエネルギー変換とは

人類は古代より長く奴隷や牛馬などの力学エネルギーを利用していたが,文明の進歩とともに帆船(BC2000年ころ)や簡単な水車(BC100年ころ)やスペイン風車(700年ころ)の利用が始まった.またアルキメデスが多数の反射鏡で敵船を焼いた故事(BC201年)はまさにソーラエネルギー利用の始めといえよう.また2世紀初めころにギリシャのヘロンが簡単な反動蒸気タービンを作ったのが今までの熱から力学へのエネルギー変換の主力である熱機関の創始といえよう.それ以後の中世にはあまり著名な機械工学的変換の発明はないが,11世紀ころに中国で発明された黒色火薬は,まずヨーロッパに広がり,リニア熱機関の原形としての大砲やその製作用金属工作技術の急速な発展に貢献した.わが国には1543年に種子島に鉄砲が伝来したが,数年のうちに堺,国友等で急速に大量生産化されて,その生産性と技術は当時の世界に例を見ない.しかし17世紀から18世紀にかけての自然科学や,起電機等を含む物理学,電気学,化学,冶金技術等の急速な発展は蒸気機関による産業革命の下地を作った.
