# 1・1・7 おわりに

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・1 機械工学におけるエネルギー変換とは

以上のように各種の新旧エネルギー変換史を極めて簡単に述べた。今後を考えると従来の化石燃料や原子力などを熱源とする機械工学的な発電所や自動車や熱機関によるエネルギー変換から、地球環境に影響の少ない太陽電池や燃料電池やバイオマス発電といった電気化学的、生物化学的エネルギー変換に研究開発の主力は移ると予想できる。5―4エネルギー供給システムまた動的なものから静的かつクリーンな変換へ、また集中型から分散型へと変化するであろう。またそれらの製作も今までの古典的機械加工からミニロボットによるミクロやナノ的な膜やチップの大量生産へと移行するであろう。また制御技術も進歩し、安全、無公害再生可能型エネルギー変換技術の開発が期待されている。
