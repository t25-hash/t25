# 1・2・1 エネルギーとは

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

エネルギーは文明を発展させる『起動力』であるとともに、文明を滅亡へと向かわせる破壊力でもある」。昨今の地球環境問題や世界情勢を眺めると、この言葉ほど、エネルギーの本質をついたものはないだろう。確かに、人間が築き上げた文明を生かすも殺すも、エネルギーの使い方次第である。歴史を振り返ると、エネルギーは、古代ギリシャで、作用・活動・勢力などを表すエネルゲイア(ενερχεια)を語源として、イギリスの物理学者トーマス・ヤング(Thomas Young,1773~1829)が科学的な意味で使い始めたといわれている。物理現象において、エネルギーは時間や空間とともに最も本質的な概念である。まさに、エネルギーは仕事をする能力といえる。中国語でエネルギーを意味する能源は、エネルギーの本質をよく表しているといえよう。
