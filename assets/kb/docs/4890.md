# 1・2・2 エネルギーの種類

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

エネルギーには,さまざまな種類がある.おもなものを以下に説明する.a. 力学エネルギー(mechanical energy)質量の物質が速度で運動する場合は運動エネルギー(kinetic energy)=2をもつ.また,質量の物質が重力場(重力の加速度)で基準面より高さにあれば,相対的にの位置エネルギー(potential energy)をもつ.重力場は,鉛直方向座標をとするときポテンシャル=の微分=-옿옿=-により規定される保存力場である.この意味で,位置エネルギーはポテンシャルエネルギーともいう.ばねやゴムなどの弾性体(あるいは圧縮性のある気体)は,伸縮ひずみ(あるいは膨張収縮)を与えることにより弾性エネルギー(elastic energy)を蓄える.弾性エネルギーは位置エネルギーの一種であり,例えばばねの場合に,ばね定数を,伸縮長さをとすると,=2と表される.気液界面では,表面張力γに対して面積Δが増加するときγΔの仕事が必要で,結果的にこの分の表面エネルギーが増加すると考えられる.b. 電磁気エネルギー(electromagnetic energy)静電場で,二つの点電荷A(電荷)とB(電荷′)が距離だけ離れている場合,クーロン(Coulomb)力(引力または斥力)により,=′4πεの位置エネルギーが蓄えられている(ε:真空の誘電率).また,容量のコンデンサに電荷がためられているときは,2の静電エネルギーが保持されている.より一般的には,静電場(電界,電束密度,誘電率ε)における単位体積当たりの静電エネルギーは=ε2=2εと表現される.一方,静磁場(磁界の強さ,磁束密度,透磁率μ)には,単位体積当たり=2μ=μ2の静磁界エネルギーが蓄えられている.これらをまとめて,電磁場のエネルギー=1ε+μd=1+d(1・1)は,速度=1εμで空間中を運ばれる(d:体積素片).マクスウェル(Maxwell)の方程式を用いて変形すると,ρを電荷密度としてdd=-ρd-×d=-ρd-d(1・2)となる.ここで,右辺第1項はジュール(Joule)熱への変換,第2項は面積素片dから外部に流出する電力エネルギーを表し,=×はポインティング(Poynting)ベクトルと呼ばれる.c. 光エネルギー(light energy)光は電磁波の一種であるので,b項と明確な境界はないが,電波(波長10m程度のマイクロ波以上の電磁波)よりも波長が短い領域,すなわち,赤外線,可視光,紫外線,X 線などの放射線に対応する.なお,波長の短い紫外線やX 線に対しては,波動とする扱いは適当でなく,光量子の概念を導入する必要がある.熱エネルギーとの関連では,赤外線領域の熱ふく射(thermalradiation)が重要であり,黒体に関するステファン(Stefan)・ボルツマン(Boltzmann)の法則욁=σ(욁:黒体の全放射能,σ:ステファン・ボルツマン係数,:絶対温度)や,放射スペクトルが最大となる波長λ윯율융と黒体温度との関係を示すウィーン(Wien)の変位則λ윯율융=2898mKが重要である.d. 化学エネルギー(chemical energy)物質を構成する原子や分子が,外周電子の作用で,結合,分解,電離などの反応を生じるときに放出あるいは吸収されるエネルギーである.したがって,化学反応に伴うエネルギー変化は1分子当たり1eV のオーダである.化学エネルギーにかかわる化学結合をミクロな視点から分類すると,ファン・デル・ワールス(van der Waals)力結合,イオン結合,共有結合,金属結合,水素結合などがある.化学反応では,例えば燃焼のように熱エネルギーを放出・吸収する場合と,電気分解のように電気エネルギーが関係する場合がある.e. 熱エネルギー(thermal energy)物質は内部エネルギー(internal energy)と呼ばれる熱エネルギーをもっている.ミクロに見ると,固体状態では原子や分子が振動運動しているが,温度が上昇して相変化により液体になると分子どうしが互いのまわりを回転する運動もできるようになり,さらに温度が上昇して相変化により気体になると並進運動もできるようになる.相変化に伴って吸収・放出される熱エネルギーは潜熱と呼ばれ,相変化がない場合に物質の温度変化に伴う顕熱と区別される.そもそも,温度そのものがエネルギーを定量化した指標であり,原子や分子の1自由度当たり=2(:ボルツマン定数)のエネルギーが分配されている.比熱容量は単位温度当たりの比内部エネルギー変化と定義される.気体のエネルギーについては注意が必要である.アルゴンのような単原子気体では並進自由度が3であるが,水素のような2原子分子の場合には,回転自由度の2が加わり,合計5となる.通常,われわれが扱う温度域の水素の定積比熱容量が욋52(:気体定数)であるのは,以上の理由による(さらに厳密には,比熱容量は温度によって段階的に変化する).熱は,伝導・対流・ふく射などにより高温側から低温側に伝達される.f. 核エネルギー(nuclear energy)原子核そのものの分裂や融合,いわゆる原子核反応に起因するエネルギーが核エネルギーである.原子核の質量変化と発生するエネルギーの間には=(:光速)という法則がある.見掛け上は運動エネルギーの表現に似通っているが,1反応当たりのエネルギー発生は,MeV のオーダになる.具体例としては放射性元素の崩壊時に放出されるX線,原子炉での核分裂反応U+hUP+P+n(ここで++=236,=2または3で平均2.42程度)がある.また,未来のエネルギー源の一つである核融合炉で重要となる反応として,重水素と三重水素の核融合反応D+THe+n+17.6MeV がある.
