# 1・2・2 エネルギーの種類

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

エネルギーには、さまざまな種類がある。おもなものを以下に説明する。この意味で、位置エネルギーはポテンシャルエネルギーともいう。ばねやゴムなどの弾性体(あるいは圧縮性のある気体)は、伸縮ひずみ(あるいは膨張収縮)を与えることにより弾性エネルギー(elastic energy)を蓄える。気液界面では、表面張力γに対して面積Δが増加するときγΔの仕事が必要で、結果的にこの分の表面エネルギーが増加すると考えられる。ここで、右辺第1項はジュール(Joule)熱への変換、第2項は面積素片dから外部に流出する電力エネルギーを表し、=×はポインティング(Poynting)ベクトルと呼ばれる。c. 光エネルギー(light energy)光は電磁波の一種であるので、b項と明確な境界はないが、電波(波長10m程度のマイクロ波以上の電磁波)よりも波長が短い領域、すなわち、赤外線、可視光、紫外線、X 線などの放射線に対応する。なお、波長の短い紫外線やX 線に対しては、波動とする扱いは適当でなく、光量子の概念を導入する必要がある。d. 化学エネルギー(chemical energy)物質を構成する原子や分子が、外周電子の作用で、結合、分解、電離などの反応を生じるときに放出あるいは吸収されるエネルギーである。したがって、化学反応に伴うエネルギー変化は1分子当たり1eV のオーダである。化学エネルギーにかかわる化学結合をミクロな視点から分類すると、ファン・デル・ワールス(van der Waals)力結合、イオン結合、共有結合、金属結合、水素結合などがある。化学反応では、例えば燃焼のように熱エネルギーを放出・吸収する場合と、電気分解のように電気エネルギーが関係する場合がある。e. 熱エネルギー(thermal energy)物質は内部エネルギー(internal energy)と呼ばれる熱エネルギーをもっている。ミクロに見ると、固体状態では原子や分子が振動運動しているが、温度が上昇して相変化により液体になると分子どうしが互いのまわりを回転する運動もできるようになり、さらに温度が上昇して相変化により気体になると並進運動もできるようになる。相変化に伴って吸収・放出される熱エネルギーは潜熱と呼ばれ、相変化がない場合に物質の温度変化に伴う顕熱と区別される。比熱容量は単位温度当たりの比内部エネルギー変化と定義される。気体のエネルギーについては注意が必要である。アルゴンのような単原子気体では並進自由度が3であるが、水素のような2原子分子の場合には、回転自由度の2が加わり、合計5となる。熱は、伝導・対流・ふく射などにより高温側から低温側に伝達される。f. 核エネルギー(nuclear energy)原子核そのものの分裂や融合、いわゆる原子核反応に起因するエネルギーが核エネルギーである。見掛け上は運動エネルギーの表現に似通っているが、1反応当たりのエネルギー発生は、MeV のオーダになる。また、未来のエネルギー源の一つである核融合炉で重要となる反応として、重水素と三重水素の核融合反応D+THe+n+17.6MeV がある。
