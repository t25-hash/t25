a.海流エネルギー（ocean currents energy) 海面に降り注
ぐ太陽熱エネルギーは，地球表面において一様ではない．それは低
緯度地域ほど大きく，高緯度地域になるにつれて小さくなる．した
がって，海水には熱エネルギーによる力の不均衡（低緯度地域と高
緯度地域間の海水温度差）が生じて対流現象が誘起される．さら
に，地球の自転による慣性力が加わり，海洋に地球規模での海水の
大循環が発生することになる．この海水の流れが海流である．世界
のおもな海流分布を図5･87 に示す．低緯度海域で太陽熱で加熱
された海水（暖流）が高緯度海域へ移動し，氷などで冷却された高
緯度海域の海水（寒流）と交換され，それが低緯度海域へ環流す
る，いわば太陽熱の平均化現象といえる．海流は一般に，北半球に
おいては時計回りに，南半球では反時計回りに流れている．
日本の南岸沿岸には北半球の二大海流である黒潮が流れている．
日本近海における海流の様子を図5･88 に示す．黒潮の規模は，
流量が3000～5000万m /s ，流速が1～5ノット（0.5～2.5m/s ）

b.海流発電（power generation from ocean current ) 海流
からエネルギーを回収する試みは今までに実例はなく，構想の段階
に終わっている．それは，流速が比較的遅く（2～3m/s ）エネル
ギー密度が低いことと，陸地からかけ離れた沖合に巨大な海洋構造
物を設置あるいは係留するなど，技術的，経済的に困難な問題が多
いためである．
海流のエネルギー変換システムは海水の運動エネルギー利用であ
るから，海水の流れのエネルギー変換を考えればよい．海流をエネ
ルギー源として利用する場合，エネルギー輸送に有利であり，エネ
ルギー利用の自由度が高い電気エネルギーへの変換システムすなわ
ち発電システムが考えられる．
発電する方法としては，水車と発電機を組合わせた方法と電磁流
体発電による方法とがある．水車・発電機による方法としては，海
流の運動エネルギーをいったん水車により回転エネルギーに変換
し，発電機を駆動する方法が一般的に考えられる．今までに提案さ
れている方法を図5･89に示す．
c .潮流エネルギー（tidal currents energy) 潮流は潮汐現象
に起因した周期的な海水の流れである．潮汐と同じように潮流は外
洋では弱く，海岸の近くで強くなり，特に湾の入口が狭まった場所
や陸地に挟まれた海峡，水道または瀬戸で流速が速くなる．したが
って，潮流の流速が速い地域分布は潮汐差の大きい地域分布と合致
する．日本における潮流の流速は，鳴門海峡で最大流速約10ノッ
ト 5m/s，来島海峡では最大流速約9ノット 4.5m/sであるか
ら，海流の流速に比較し2倍以上速い．日本の海域別潮流最大流速
とエネルギー賦存量を表5･11に示す．わが国の潮流エネルギーの
図5･87世界のおもな海流
賦存量の推定値は2500万kWと試算され，年間約2190億kWh
と見込まれている．潮流の流速は潮汐と同様に12時間25分の周期

d.潮流発電（tidal power generation) 潮流発電では堤防を
造らずに自然の流れから直接発電しようとする方法であるので，潮
汐発電のような，湾の入口に貯水用のダムを造るための建設費など
を考慮すると，コストの面で有利である．ただ，海流とは異なり潮
流は約6時間ごとに流向が反転するために，発電装置が有効に作動
するための方位制御機構あるいは流向に依存しない発電装置の開発
が必要である．
また，潮流の変動するエネルギーを有効に利用するするために
は，効率のよい変換システム（電気エネルギー，熱エネルギー，機
械エネルギーへの変換）あるいはエネルギー貯蔵システム（揚水，
水素貯蔵，圧縮空気，フライホイールなど）を検討する必要があ
る．
図5･92 発電特性
潮流発電では潮流の時間的流速変化の予測が可能，建設費がダム
式より安価，環境インパクトが少ないなどの利点のため種々の試み
がある．しかし，現在は実験段階で実用化の例はなく，実際の潮流
による実験は，ダリウス型水車を用いたものがカナダと日本に一例
ずつあり，クロスフロー型水車を用いたものが日本に一例あるだけ
である．ダリウス型水車は，構造が簡単，流れの方向が反転する潮
流に対して水車はつねに一方向に回転し，翼の周速は流速の2～3
倍などの特徴がある．日本大学では，3枚翼のダリウス型水車を試
作して，1983～1988年の間，来島海峡の実際の潮流で3回にわた
り実験を行った．海底設置型の発電装置を図5･91に示す．実験装
置設置場所の最大流速は1.7m/sである．翼の材質をGFRP，
CFRPに変更して軽量化した第3回めの実験では流速0.6m/sで
自起動し，流速1.6m/sで発電電力2.3kW，変換効率は43.9 %
の結果が得られた．この装置の発電特性を図5･92に示す．
徳島大学では，1986年に潮流の流れ方向が反転しても回転方向
が同方向で，効率がよいクロスフロー水車を用い，鳴門海峡の実際
の潮流で実験を行った．水車は，直径31.5cm，長さ1mであり，
長さ5m，幅2.5mの浮体に取付けたものである．大鳴門橋の工
事用桟橋にロープで固定して行い，最大800Wの発電を行った．
