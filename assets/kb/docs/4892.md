# 1・2・4 エネルギーの保存とエネルギー変換

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

工学的に厳密な意味ではエネルギーは決して消費されず、ただ変換されるだけである。前述のように、核エネルギーの場合は質量保存則も含めて考える。人間はさまざまなエネルギーを、あるときは無意識的に、あるときは意識的に他のエネルギーに変換して生活している。最も身近な例として、公園の滑り台で遊ぶ子供について考える。もとはといえば、食事によって得た食物の化学エネルギーが、子供の筋肉に作用して滑り台上の出発点に立つ子供の位置エネルギーに変換される。その位置エネルギーは滑り下りる子供の運動エネルギーに変換される。しかし、子供が地面に着地し終わったときには、そのエネルギーは子供と滑り台との間の摩擦仕事、空気抵抗に対する仕事、着地時に靴底によって飛散する土の運動エネルギーなどに変換されている。そしてしばらくすると、すべて熱エネルギーとなって環境中に拡散して埋もれてしまう。このように、エネルギーはつねに変換されながら、最終的には環境と同じ温度の熱エネルギーに落ち着く。一般的にいわれる「エネルギーを消費する」ということの工学的に厳密な表現は、周囲環境の状態(温度や圧力など)に対して相対的に定義されるエクセルギー(exergy:有効に取出しうる仕事量)の語を用いた「エクセルギーを消費する」である。
