# 5・7・1 バイオマス（biomass ）とは

> **章**: 第5章 再生可能エネルギー
> **節**: 5・7 バイオマス

地球上の生物は植物，動物，菌，原生動物，モネラ（細菌）から
構成されており，これら生物体の量をバイオマス（biomass ）とい
う．しかし，エネルギー源や工業原料の対象としてのバイオマス
は，一般には，光合成により蓄積された植物体の重量あるいはエネ
ルギー量を指す．
地球上の植物体は，光合成により水とCOから有機物を合成し，
葉や根，茎や実などとして蓄積する．光合成により植物が1年間に
蓄積した有機物量を純一次生産量といい，理論上は，再生可能な資
源あるいはCOニュートラルな資源と考えられる．
地球上のバイオマスの総量（植物体）は乾燥重量で1800Gt ，純
生産量は170Gtといわれている．表5･13に，地球上の異なる生態
系におけるバイオマスの蓄積量と純一次生産量を示す ．熱帯多
雨林をはじめとする森林がバイオマスの蓄積量と生産に大きな役割
を果たしている．

バイオマスを資源として利用する場合のおもな利点と欠点は以下
のようにまとめられる．
おもな利点
･太陽エネルギーの変換貯蔵であり，再生産が可能である．
･微生物により分解される．
･燃焼による環境負荷が小さい．
･燃焼後に残る灰は資源としてリサイクルできる．
おもな欠点
･広い範囲に拡散しており，効率的な利用には集積が必要にな
る．
･化学的，物理的に複雑な構造をしている．
･利用に際しては，脱水，乾燥，粉砕などの前処理が必要にな
る．
･単位重量当たりの発熱量が小さい．
