# 1・2・5 エネルギー変換(energy conversion)の目的別分類

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

前者には化石燃料や太陽・水力・風力エネルギーなど、後者としては電気エネルギーや水素などがある。エネルギー変換は、目的別に分類すると4種類ある。まず、化石燃料のようなエネルギー資源中に含まれている化学エネルギーを燃焼させて熱エネルギーを発生する場合、資源側変換という。次に、エネルギー源の獲得とエネルギーの消費とは、空間的にも時間的にも異なる点で行われることが多いので、輸送しやすさを目的とする輸送変換や貯蔵しやすさを目的とする貯蔵変換が必要になる。輸送変換の代表例は、一次エネルギーから電気エネルギーおよびガスへの変換であり、この結果、それぞれ電線・ガス配管網により容易に輸送されることはいうまでもない。また、深夜に供給過剰となる電力で氷を作り、その冷熱を昼間の冷房に用いるのは、貯蔵変換である。一方、火力発電所で化石燃料の燃焼熱を用いて、ガスタービンや蒸気タービンを回し、さらに発電機を回すことは、熱エネルギーを使いやすい電気エネルギーとして利用するための利用変換である。
