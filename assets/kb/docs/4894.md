# 1・2・6 各種エネルギー相互間の変換

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

前述のようにエネルギーには、力学エネルギー、電磁気エネルギー、光エネルギー、化学エネルギー、熱エネルギー、核エネルギーなど、さまざまな種類がある。これらのエネルギーでは、核エネルギーを除き、多くの場合に相互変換が可能である。表1・2には、エネルギー変換前と変換後の組合せと、変換に関与する現象、効果、変換の媒介となる装置につきまとめた。この表のマトリックスを繰返し循環することにより、エネルギー変換の連鎖ともいえる一連の過程を理解することができる。例えば、現代の文明を支えている電気エネルギーをさかのぼると、太陽の核エネルギーが電磁気エネルギー・光エネルギーとして地球に到達して、生物体のもつ化学エネルギーとなる。生物体は死後何億年もの間に炭化作用で化石燃料の化学エネルギーとなり、火力発電所における燃焼反応による熱エネルギーおよび熱伝達でボイラ内の水蒸気の熱エネルギーとなり、蒸気タービンで力学エネルギーとなって、最終的に発電機の電磁誘導現象で電気エネルギーとなる。そして、この電気エネルギーを、われわれは機械エネルギー、熱エネルギー、光エネルギー、音エネルギーなどに利用しているのである。
