# 5・2・2 マイクロ波加熱を利用した機器

> **章**: 第5章 加熱・冷却機器
> **節**: 5・2 マイクロ波加熱

マイクロ波加熱を応用した機器はその利点を生かし，家庭用から
工業用と幅広くある．身近なマイクロ波加熱装置としては家庭用電
子レンジがある．電子レンジはそのマイクロ波出力が1kWくらい
であるが，工業用マイクロ波加熱装置では数十kW，なかには数百
kWと大電力のものが多い．
また，医療分野では比較的低電力のマイクロ波電力（100～200
W）が用いられ，治療や手術用マイクロ波加熱装置が開発されて
いる．マイクロ波加熱の利用分野は食品，化学，医療，印刷，窯
業・セラミックス，木材・紙工業，繊維工業分野と多分野にわた
り，それぞれに専用のマイクロ波加熱装置が開発され実用化されて
いる．ここでは食品，化学，医療分野でのマイクロ波加熱装置の例
を示す．
a.食品分野 食品の調理では電子レンジ（microwave oven)
がマイクロ波加熱装置の代表である．電子レンジは家庭で主として
食品の加熱調理に利用されているが，電気ヒータを併用したもの
（オーブンレンジ）が日本国内では主流となっている．図5･5に家
庭用電子レンジ（オーブンレンジ）を示す ．工業用としては，食
品の熟成，殺菌，防かび，スナック食品などの膨化乾燥，茶葉の乾
燥用マイクロ波加熱装置がある．殺菌や防かびではアプリケータ内
の圧力を上げることにより沸点を上昇させて，高温菌を死滅させる
加圧式マイクロ波加熱装置がある．また，茶葉の乾燥用マイクロ波
加熱装置では，従来の外部加熱方式と比較して，茶葉の葉緑素の安
定化が図られ長期保存による変色がなく甘味を引出すことができ
る．さらにはビタミンCの残存率量が高いので広く実用化されて
いる．茶葉の乾燥用マイクロ波加熱装置は，一般的にはコンベヤ式
が多く利用され，連続的に茶葉を乾燥するようになっている．
b.化学分野 化学分野では，顔料・染料の乾燥用，樹脂の発
泡用マイクロ波加熱装置がある．特殊なものとしては，図5･6に示
すようなマイクロ波化学合成反応装置がある ．この装置は複数の
化学薬品を混合し，これにマイクロ波を照射して化学反応を促進す

らに生成された物質も新しい性質や機能をもったものが得られるな

c .医療分野 癌細胞は約43℃に加温すると死滅することが実
験的に確かめられている．そこで低電力のマイクロ波をホーンアン
テナなどを利用して癌細胞に局部的に照射して加温し，癌細胞を死
滅させるマイクロ波加熱装置（ハイパサーミア装置）が開発され，
実用化されている．
また，人体内の病変組織に電極を刺入れてマイクロ波を供給し，
病変組織の止血凝固を図り，切除するマイクロ波メスもある．図
5･7，5･8にマイクロ波メス本体装置と電極部を示した ．マイク
