# 1・2・11 わが国のエネルギーフロー

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・2 エネルギー変換の基礎

図1・3に、わが国のエネルギーフローを示す。左側は一次エネルギー、右側が二次エネルギーである。大きな流れとしては、石炭・石油・天然ガス等の化石燃料を主体とする一次エネルギーのうち約41%が発電用に用いられている。発電所の大部分を占める火力・原子力発電ではランキンサイクルやブレイトンサイクルが主であるの図1・2 ペルチェ効果図1・3 わが国のエネルギーフロー5―8エネルギー供給システムで、平均的な発電効率は40%程度であり、残りは温排水や大気中への放熱となっている。すなわち、発電所だけで、一次エネルギー換算にして27%もの量がむだになって環境に捨てられることになる。発電に向けられる割合は年々増加の一途をたどっているので、発電の効率が向上しない限り、むだに捨てられるエネルギー量が増加していくというのが現状である。このために、発電に伴って発生する熱エネルギーを有効に利用することを目的としてコージェネレーション(熱電併給、cogenera-tion)が省エネルギー(energy conservation)の一つの柱とされてきた。ただし、電気エネルギーと異なって熱エネルギーの長距離輸送は困難であるため、熱需要のある箇所で小規模のマイクロコージェネレーションを行う分散型電源システムが、近年普及し始めている。このほか、発電効率の高い燃料電池の開発と普及、風力・太陽等の自然エネルギーの利用促進も重要な課題である。
