# 1・4・2 ライフサイクル評価

> **章**: 第1章 エネルギー変換の歴史
> **節**: 1・4 エネルギー供給システムのライフサイクル評価

ISO 14040は、LCA がいかなる目的や考え方に基づいて作られ、どのような方法があるかを規定している。それによるとLCA は、目標の設定と範囲(goal definition and scoping)、インベントリー分析(inventory analysis)、影響評価(impact assessment)、それに解釈(interpretation)から構成されている(図1・9)。LCA を実施していくためには、適用の意図に合った目標の設定と範囲が明確に定義されていなければならない。目標には、適用の意図、調査理由、そして結果の報告主体があいまいさのない文言で述べられていなければならない。範囲とは調査の広がりと深さを意味しており、その内容は目標にかなったものになっている必要がある。インベントリー分析とは、製品(財、サービス)のライフサイクルにわたり資源消費と環境負荷の投入と産出がどのように行われているかを明らかにするもので、ライフサイクルアセスメントの中で最も基本となる分析である。このことからインベントリー分析は、これまでのLCA 研究において最も多くの研究が行われてきている。社会における環境負荷は資源消費、すなわち原料、素材、エネルギーの消費によって発生する。=影響評価は、対象製品等から排出する環境排出因子のインベントリーが、社会や人にどのような影響を与えているかを明らかにするものである。一般に影響評価は、分類化(classification)、特性化(characterization)、価値化(valuation)といった3段階で評価される。影響評価の最初の集計作業となる分類化では、環境排出のインベントリーデータはいくつかの環境影響因子に割り当てられる。影響因子は場所や時間によって健康や生態系に与える大きさが異なるために、まだ統一のとれた分類方法がない。分類例としては、資源枯渇、健康影響、生態系影響の三つに大きく分類し、それらをさらに化石燃料消費量、希少資源消費量、地球温暖化、酸性雨、オゾン層破壊、砂漠化、光化学酸化物形成、あるいは埋立て容量、景観破壊、有害物質、騒音、悪臭、従業員リスク、生物資源、過密化などの細目に分けていく方法がある。一つの環境影響因子に含まれている環境排出因子であるいくつかの化学物質は、種類によって影響因子への影響度が異なる。特性化とは、分類された細目の環境影響因子に対してその影響度を定量化するものである。例えば地球温暖化影響について考えてみると、温室効果ガスであるCO,CH,NO,CFCは、それぞれが異なる温暖化影響をもつ排出因子である。その影響の大きさは、赤外線吸収力を指標化した温暖化ポテンシャル(GWP: global warmingpotential)で評価される。このように各温室効果ガスの温暖化影響としてGWPのような共通の評価指標または評価関数を設定し、インベントリーの影響因子への影響度を定量的に表すのが特性化である。価値化とは、各影響因子がエンドポイントに対してどれだけ重要であるか、それぞれの重要度を判断するものである。重要度とは重み付けであり、その判断は自然科学よりも政策や倫理など社会科学の観点に基づいて評価される。最終的にはその重みを利用することで、ある製品あるいは技術のライフサイクルにわたるすべての環境影響が統合化されることになる。EPI=EPIはスカラ量であり、それを決める重要度は、専門家、政策決定者、あるいはその他の利害関係者の判断によって求められる。
