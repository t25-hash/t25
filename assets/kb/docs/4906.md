# 5・2・1 保管，収集，輸送のあり方

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・2 廃棄物の収集，輸送

都市ごみ処理システムでの保管・収集・輸送の位置付けを示すと
図5･7のようになる．保管とはごみが発生し収集されるまでの作業
をいい，収集とは収集機材にごみを積み込むまでの一連の作業，輸
送とは積み込まれたごみを目的地まで運ぶ作業をいう．目的地に
は，ごみの積換えを行う中継施設や焼却工場などの中間処理施設，
これらの要素によって，収集頻度，収集容器指定，収集場所，収
集するごみの種類などが決まるが，各要素は互いに影響し合うこと
も考慮する必要がある．一方，要素選択の制約条件となるものの多
くは地域特性と呼ばれるもので，各自治体特有のものである．それ
らの制約条件を以下に示す．

これらの制約条件の中で最も大きいのは費用の問題，すなわち自
治体の財政であろう．例えば，いつでもごみが出せるようにしてほ
しいという住民の要望に対しては，パイプ輸送方式（pipe convey-
ance system）の採用，車両収集頻度の向上による排出時間指定の
解除，コンテナ等の保管容器設置によるごみ投入方式の採用などの
解決策が考えられる．また，ごみを出す際の利便性を高めてほしい
という要望に対しては，パイプ輸送方式の採用や，家庭から収集場
所までの持出し距離を短くするなどの方法がある．しかし，その実
施に当たっては，費用が制約となることが多い．
また，これらの制約条件も，各条件が互いに影響し合うことが多
い．ここに挙げたサービス内容の決定要素と制約条件の項目には重
ごみ排出容器
(一時保管容器)
複するものもある．気候についていえば，気候はハエや悪臭の発生
に関係するもので，必要収集頻度を決定する要素であるが，降雨降
雪量の多少によっては無がいトラックの使用を制約する条件にもな
りうる．
一般に，ごみ処理施設建設に対する反対理由として，収集輸送車
の出入りがよく指摘されるように，ごみ処理に関わる環境影響の中
で，収集輸送車の運転に伴うものが問題とされることが多い．収集
そして埋立てなどの処分を行う最終処分場がある．
都市ごみ処理事業において保管・収集・輸送は住民との接点とい
う重要な役割を担っており，住民の目に触れる機会が多い極めて重
要な工程である．家庭で発生したごみは速やかに収集して保管場所
（居住空間）から取り除かないと，(１)公衆衛生の低下，(２)集積
したごみによる空間の占拠，(３)環境の美観低下等の問題が引き起
こされる．よって，発生したごみを速やかに取り除く必要があり，
また保管中にもコンテナ（container ）等の収集容器を置き，ごみ
集積所の衛生管理，美観保持を容易にする必要がある．一方で，保
管容器としてポリ袋を指定し，その他の収集容器を使用しない自治
体が増えてきているが，それは，(１)収集場所でのごみの容量を小
さくできる，(２)作業員が収集しやすいため収集時間が短縮でき
る，(３)住民が容器を引き取る必要がない，などの理由による．
都市ごみ処理経費の中で収集・輸送の占める割合が最も高く，こ
の作業に携わる労働力は全体の50%以上となることから ，この
工程の適正化が都市ごみ処理事業全体にもたらす効果は大きい．保
管・収集・輸送は機材，方法等のさまざまな要素からなるが，その
代表的なものは表5･12のようになる．収集・輸送サービスの量と
質は，そのサービスに対する要求と，サービスを提供する自治体の
負担等の制約条件との兼合いで決まる．
したがって，その計画策定に際しては，表5･12に列挙した方法
と機材の組合せの中から，(１)各種要求を満足する，(２)最も経済
的である，の二つの条件を満たすものを選択することになる．サー
ビスの内容を決定する要素の例を次に示す．
(１)住民の要望（環境保全，快適性，美観，時間的・経済的負担
など）
(２)ごみ発生量
(３)ごみ質，種類
(４)気候（特に気温）
作業においては収集場所周辺での環境影響が考えられる．また，車
両走行に際しては排ガス，騒音，振動，交通渋滞，交通事故，道路
への汚水垂れ流し，悪臭等の環境影響が予想される．作業員に対す
る影響として，作業時の交通事故，転倒，転落，無理な動作，障害
物との接触による外傷，疾病あるいは作業中にさらされる排ガスに
よる影響等に関しても検討する必要がある．
都市ごみ収集・輸送計画の策定に当たっては，これらのサービス
内容の決定要素と制約条件との相互関係を把握し，整理する必要が
ある．保管・収集・輸送計画の策定手順を示すと次のようになる．
(１)調査，分析
(２)保管・収集・輸送におけるサービス内容の設定
(３)調査，分析の範囲の設定
(４)収集機材，方法に関する組合せの選定
(５)収集機材，方法の決定
(６)収集経路の設計
(７)評価，修正
計画をいったん策定した後も，サービスの内容を決定する要素は
つねに変化するため，各種要求と収集・輸送の実態に隔たりが生じ
ることがある．したがって，つねに見直しのための調査，評価を行
うことが必要である．
