# 2・5・2 ウラン資源の探鉱・採鉱・製練・転換

> **章**: 第2章 エネルギー資源
> **節**: 2・5 原子燃料

a. ウランの探鉱自然放射線(γ線)を測定し放射能異常を探索する放射能探査(radioactive survey)が従来のウラン探査の主要な方法であった。1960年代未から1970年代前半にかけて、高感度のシンチレーションカウンタを航空機に搭載した空中放射能探査システムが普及したため、効率的に広域が調査されるようになり、その結果、鉱床発見が相次いだ。γ線は、岩石の中を50cmほど通り抜ける間にほとんど吸収されるため、放射能探査では、ウラン鉱床の一部が地表に露出している露頭鉱床(outcrop deposit)しか直接とらえることはできない。このため、世界中の主要地域の調査が進んだ1980年ごろには露頭鉱床の発見がまれになり、探査対象は露頭鉱床から地下深部の潜頭鉱床(sub-surface deposit)へと移行した。潜頭鉱床を対象とした探査では、対象とする鉱床タイプの特徴に応じ、磁気探査や電磁探査等の複数の方法を組合わせて、鉱床付近の地質の特徴(周囲の岩石とは異なる性質)をとらえつつ段階的に有望地を絞り込むのが一般的である。ウラン鉱床の特徴は鉱床タイプによって大きく異なるうえ、深部情報を得るのに阻害要因となる地表部の状態に地域性があるため、有効な探査手法と手順は鉱床地域ごとに異なったものとなる。リモートセンシング、放射能探査(エアボーン、カーボーンなど)、地表化学探査、磁気探査、電磁探査、直流比抵抗探査、重力探査、試すい・検層などが組合わされて利用されている。カナダ、アサバスカ地域における不整合関連型鉱床を対象とした探査手順を図2・26に示す。図2・26 アサバスカ地域の探査手順図2・27 ボックスホールストーピング法によるウランの採鉱b. ウラン鉱床の採鉱ウラン鉱床を採掘・回収する手法は、露天採鉱法、坑内採鉱法、および原位置侵出法の三つに大別される。前二者は、基本的にウラン以外の金属鉱床等の採鉱技術を利用したものであるが、原位置侵出法はウラン鉱床に固有の技術である。. 露天採鉱法(open pit mining)露天採鉱法は土木および土質工学の応用であり、ベンチ高や法面こう配等は岩質により決定される。経済的な面から比較的浅い鉱床に対して適用可能であり、現在稼動中の露天掘り鉱山の多くは、深度150m以浅となっている。近年の露天掘り鉱山は、開発当初から、採掘終了後の埋戻し(原状回復)が義務づけられている。. 坑内採鉱法(underground mining)露天採鉱法では経済的に回収できない深部の鉱床に対して、通常は坑内採鉱法が用いられる。方法は特に一般の金属鉱床と変わるところはないが、金属鉱山と比較した場合、特有な点は、放射線管理、特にラドンガス濃度の管理上、金属鉱山などよりも換気量を多くすること、ラジウムに関する鉱山廃水管理・処理を行う必要があることである。最近では、放射線防護という観点から、カナダ、アサバスカ地域の高品位ウラン鉱床採鉱のために、作業員が鉱床内部の切羽に立ち入らずに採掘するいくつかの手法が開発されておりその総称をノンエントリー法(non-entry mining)という。今後、世界のウラン生産の5割近くがノンエントリー法による生産となる見込みである。図2・27にその一つであるボックスホールストーピング(box-hole stop-ing)法の概要を示した。5―37図2・28 溶液採鉱法。 原位置侵出(ISL: in-situ leaching)法地表から井戸を使って、鉱床を含む地層に酸やアルカリなどの溶液(浸出液)を浸透させて、鉱床中の有用元素を回収する方法で、溶液採鉱(solution mining)とも呼ばれる(図2・28)。鉱石の採掘・運搬・破砕・侵出の諸工程を省けるため他の採鉱法に比べて低コストで生産できる。また、鉱山保安と環境保全の観点からも本方法は優れている。砂岩型ウラン鉱床は一般に0.1~0.2%U と品位が低いため、他の採鉱法ではまったく採算が合わない鉱床でも、本方法が適用できれば生産が可能となっている。アメリカでのウラン生産の大部分がISL鉱山のもので、鉱床深度は100~300mである。アメリカ以外では、カザフスタン、ウズベキスタン、中国、オーストラリアで操業中である。生産終了後に地下水の水質復旧を行う必要があるアメリカでは、従来硫酸やアルカリ溶液であった浸出液にかえて、近年は酸素と炭酸ガスを用いて、pH を変化させないで侵出を行っている。一つの生産単位の井戸の配列は、一辺の長さが15~30mの正方形の頂点に注入井を、その中心に生産性を配するパターンが一般的である。. 環境管理先進諸国では近年鉱山開発に関する環境審査が強化されてきた。開発を申請する企業は、膨大な環境影響調査報告書の作成を求められ、そのための事前の環境調査(ベースラインスタディ)が必要となる。また、審査期間も長期化しており、鉱床発見後生産開始までのリードタイムは10年以上となっている。環境管理の主要な項目として、鉱山排水処理、捨石の処置、製練廃滓の管理、閉山後の原状回復等がある。このうち、高品位鉱石を処理するカナダ、サスカチワン州では廃滓管理施設(沈殿池)に対する要求が厳しくなり、露天掘り跡を廃滓管理施設に改造・利用する方式が主流となってきた。これは、露天掘りの底から横坑を掘り、これにつながる縦坑ないし井戸およびポンプを露天掘りの横に設置して、廃滓管理施設とその周辺から地下水を集中回収して製練所の用水として再利用(ないし廃滓管理施設へ循環)するもので、汚染水の環境への流出を防止するものである。また、周囲の地層の透水率が低く導水こう配が比較的大きい場合、廃滓を取り囲むように露天掘り外壁部分に透水性のよい砂れき層の壁を作り、廃滓管理施設の導水こう配を低くして地下水が廃滓を浸透・通過するのを防いでいる。c. 製錬・転換。 概要(図2・29)ウラン製練とは、ウラン鉱石からウランを分離・精製し、四フッ化ウラン(UF)を製造するプロセスであり、粗製練工程、および精製練工程(後述する転換工程を含む場合もある)からなる。ここで粗製練工程とは、鉱石からウランを抽出しイエローケーキ(yellow cake、主として重ウラン酸塩、以下YCという)を製造するまでの、また精製練工程とは、YCから5―38エネルギー供給システム図2・29各種のウラン製練・転換フローUFを製造するまでの工程のことである。. 粗製練工程以下の小工程よりなる。1破砕・粉砕:破砕により採掘したウラン鉱石を小粒塊に砕く。破砕した鉱石は、粉砕工程においてアルカリ溶液などで0.15~1mm程度の粒径をもつスラリー状にする。2浸出:スラリー状の鉱石を加熱し、浸出液(硫酸等)にウランを溶解させる。この含ウラン浸出液を貴液と呼ぶ。前述の方法のほかにも粉砕した鉱石を野積みし上部から浸出液を散布するヒープリーチング(heap leaching)法や、鉱床に直接浸出液を注入するインプレースリーチング(inplace leaching)法等がある。3固液分離:貴液中の懸燭固体をろ過機等を用いて除去する。4溶媒抽出(solvent extraction)・イオン交換:貴液から不純物を取り除き(精製した貴液を溶出液と呼ぶ)、さらにウラン濃度を高める。溶媒抽出法またはイオン交換法が用いられている。5YC沈殿・乾燥:溶出液中のウランを中和処理等によりYCとして沈殿させ、フィルタで脱水後、乾燥する。6焙焼:YCを加熱して八酸化三ウラン(UO)を製造する。. 精製練工程精製練工程の方式として、1湿式法[湿式一貫製練法として動力炉・核燃料開発事業団(現核燃料サイクル開発機構)が開発、PNC法],2乾式法(旧アライドケミカル社が開発した方法)および3湿法と乾式法を組合わせた方法がある。3が商業規模の精製練工場で最も一般的に用いられており、BNFL社(イギリス)、コミュレックス社(フランス)、カメコ社(カナダ)等において実績がある。以下の小工程からなる。1YC溶解:YCを硝酸に溶解して硝酸ウラニルUO(NO)溶液とする。2溶媒抽出:UO(NO)溶液を溶媒抽出法により精製する。3脱硝工程:精製したUO(NO)溶液を加熱して硝酸根を取り除き、粒状のUOを製造する。脱硝法としてはUO(NO)溶液とアンモニアとの反応によりYCを生成させ、これを熱分解することによりUO(UOの場合もある)とする方法もある。水素還元およびHFフッ化は乾式工程と同様である。. 転換(conversion)工程HFフッ化工程で生成したUFは、用途に応じてUFまたは金属ウランに転換される。(1) UF転換UF転換は、ウラン濃縮用の原料を製造する場合に行われる。UFは、大気庄における昇華点が低く56.5°Cであり、ウラン化合物をガス化して濃縮を行う遠心分離法やガス拡散法に適した特徴を備えている。UFは、UFとフッ素ガスの反応により生成させる(UF転換と呼ぶ)ことができ、発生したUFガスは-50~0°Cの温度で冷却、固化される。この操作を捕集という。捕集されたUFは、コールドトラップ内で液化されUFシリンダに充てんして最終製品とする。(2) 金属ウラン転換金属ウラン転換は、イギリスのマグノックス炉のように、燃料に金属ウランが必要な場合に行われる。一般的には、UFをマグネシウム等で還元させることにより製造する。ほかにも、溶融塩電解法等の金属ウラン製造技術が検討されているが、工業的には用いられていない。. 回収ウランの転換原子力発電所で燃焼させた使用済燃料中のウランを再処理工場で分離・精製したものを回収ウランと呼び、再度、核燃料として用いることにより、資源の有効利用が期待できる。基本的なプロセスは、精製練工程(乾式法)と転換工程の組合せよりなる。
