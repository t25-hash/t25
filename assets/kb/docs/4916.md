# 2・6・2 水力エネルギー(hydropower energy)

> **章**: 第2章 エネルギー資源
> **節**: 2・6 再生可能エネルギー

a. 水力エネルギーの利用水力エネルギーは再生可能な自然エネルギーの中でも古くから人類が利用してきたエネルギーであり、古代文明はすべて大河流域に都市国家を形成することで発展した。中世に入ると、人間は水の流れを利用して水車を回し、水車を利用して粉をひく方法を考え出した。それまで人間の体力にエネルギー源を求めていたものが、より大きな力を出す水力を利用することにより、生産性の向上が図られた。そして19世紀後半には、水力エネルギーを電気エネルギー(electrical energy)に変換する水力発電所(hydroelectric power station)が建設され、その後技術の発展に伴い次第に大容量化がなされ、水力エネルギーは現在でも重要なエネルギー源として位置付けられている。b. 水力エネルギーの特徴一次エネルギーである水力エネルギーは、電気という利用しやすい形態の二次エネルギーへの変換が表2・18 system in nature)の一部であり、元をたどれば太陽エネルギーを間接的に利用していることとなる。水力エネルギーは太陽エネルギーを起源とし、それが枯渇しない限り利用できる再生可能エネルギー(renewable energy)である。c. 水力エネルギーと水力発電水力エネルギーは、重力場において水のもつ位置エネルギー(potential energy)、速度エネルギー(velocity energy)として利用される。現在ではこれらのエネルギーにより水車と発電機を回転させ、電気エネルギーに変換して利用されることがほとんどである。ここで、ηは水車と発電機を合わせた全効率、ρは水の密度、は単位時間当たりの流量、は水路などで生じる損失ヘッド(損失エネルギーを水の高さに換算したもの)である。現存する一般的な水力発電所では、効率η(水車と発電機を合わせた全効率)が80%を超え、他の発電方式と比較して圧倒的に高効率となっている。d. 今後の水力エネルギー水力エネルギーの利用は、水力発電という形で自然のクリーンなエネルギーを活用することにより今日まで発展してきているが、反面、経済的に利用しうる開発地点は先進国では少なくなってきている。国土の狭いわが国において、特にその傾向が強く、人工的に開発地点を選定できる揚水発電を除き、水力の利用は中・小規模のものがかなりのウェートを占めるようになってきた。水力発電は、初期投資は大きいが、一度出来上がれば設備補修のみで理論的には永久に発電が可能であり、COによる地球温暖化が問題となっている現在、再び注目されるエネルギー源である。また、構造が簡単、高効率、制御が容易、長寿命、環境に優しいなど水力エネルギーの利点は多く、今後とも一つの重要なエネルギーとして位置付けられることとなろう。
