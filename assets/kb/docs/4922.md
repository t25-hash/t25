# 3・1・3 コージェネレーション(cogeneration、熱電併給)

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・1 に示す。電力比率はほぼ単調に増加傾向にあり、過去30年に

発電だけでなくトータルの変換効率を上げる技術として期待されているのがコージェネレーションである。コージェネレーションとは発電と同時に、その際に低温熱源に放出される熱を暖房や給湯などの熱需要に使うことを目指した技術である。通常の蒸気ランキンサイクル発電所ではタービン出口に復水器を置き、海水で冷やすことによりタービン背圧を下げて発電出力の増加を図っている。言い換えれば、低温熱源である海水に電力に変換されない熱を放熱して図3・1 日本における最終需要に占める電力比率5―55いる。この場合、排熱温度は環境温度+数°Cであり、給湯・暖房には利用不可能である。コージェネレーションは発電効率を多少犠牲にしても排熱温度レベルを高くとり、温熱として利用可能にすることによって、トータルの一次エネルギー利用効率を70~80%と高く保持することを基本的な考え方としている。わが国におけるコージェネレーションの導入状況を図3・2に示す。過去30年にわたり、導入量が急拡大している様子が見て取れる。コージェネレーションは次のように原動機によってタイプ分けされる。(1) ディーゼルエンジンコージェネレーション(2) ガスエンジンコージェネレーション(3) ガスタービンコージェネレーション(4) 燃料電池コージェネレーションいずれもエンジンジャケット冷却水あるいは排ガスからの熱回収(heat recovery)として温熱を得る。原動機によって排ガスの温度レベルが異なるので、回収される形態も中圧~低圧蒸気、温水と分かれる。一般的に、コージェネレーションには排熱が不足する場合に補うボイラが付属する。業務系などでは温熱の需要よりも冷熱の需要(冷房需要)のほうが多いので、回収された温熱を冷熱に変換する吸収冷凍機が装備される。70°C程度の温水回収の場合には通常の吸収冷凍機が動作しないため、吸着冷凍機が組合わされることもある。発電効率の観点から各種プラントを整理したものを図3・3に図3・2 コージェネレーションの普及状況図3・3コージェネレーション機種別発電性能5―56表3・1 石油系燃料の性状例重油(C)比重(15/4°C)0.952~0.9550.8520.88~0.8870.8530.8590.6970.796――発熱量(kJ/kg)(高位)42000~46200―43100~4410045600―4740022400―29500粘度(CST at 50°C)250~6.910~1311.90.533(30°C)0.77(20°C)(30°C)硫黄分(%)3.5以下1.722.6~3.00.1010.10.001≒05.32.4~2.9引火点(°C)70以上0以下-14以下-190以下――流動点(°C)8~11-15以下-15以下32.5-30-20(融点) 15―バナジウム(ppm)6~960.61以下―――280~350示す。近年ガスエンジンでも系統発電所に匹敵するような発電効率が高いものも開発されている。一方、大型のガスタービンとは異なる考え方に基づき小型化されたマイクロガスタービンが開発され、現在その応用について検討が進められている。ここでは、コージェネレーションについて原動機別に解説を行う。ただし、燃料電池については3・7節により詳細な概説があるので、そちらを参照していただきたい。石炭を燃料として始まった火力発電は、経済性や環境対応から、石油系燃料の使用が始まり、当初は重油が中心であったものが、さらなる経済性の追求から、原油の生焚きや残油が採用され、また、高環境性への要求の高まりから、ナフサ、メタノールといった軽質油も使用されており、立地や状況に応じて多岐にわたる石油系燃料が使用されている。石油系燃料(petroleum fuel)の性状の一例を、表3・1に示す。
