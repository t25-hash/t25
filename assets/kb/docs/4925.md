# 5・5・2 欧州指令（WEEE，RoHS)

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・5 環境負荷低減材料および製品開発

認定したシンボルマークを表し，タイプⅡは事業者が自主的に環境
同じ世界産業である電気・電子産業では，主要市場であるEU
配慮を宣言するものである．タイプⅢは定量的環境情報開示型とい
われ，LCA手法により生産から使用・廃棄までのCO排出量など
の環境負荷を定量的に表示するもので，産業環境管理協会のエコリ
ーフ環境ラベルが運用を開始している．そのほかにグリーン購入の
上位概念であるサステイナブル購入の評価基準としてエコプレミア
ム を商品購入の枠組にする活動もある．
2003年3月EU官報に，廃電気電子機器指令（WEEE:waste
環境負荷低減材料および製品開発においては，規制が技術進歩を
促し，商品の情報開示が消費者や社会の認知・認証を受けることが
技術的な常識となる．技術開発には地球環境の目線と消費者の目線
が必要とされ，環境リスク対策が重要な商品化の要素となる．
