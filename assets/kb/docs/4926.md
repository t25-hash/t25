# 3・2・4 重質油火力発電(residual oil fired power generation)

発電の経済性を主に、従来の重油原油とは異なる重質系の燃料の活用が注目され実用化が行われている。いくつかの例を以下に示す。a. 残渣油(residual oil)石油精製の最終過程で生じる高粘度、高硫黄分の重質残渣である。重質残渣をさらに減少するため、減圧蒸留装置による精製が進められており、この過程での残渣は真空残渣油と呼ばれ、硫黄分、窒素分が多く、環境対応や安定運用に配慮が必要である。これら重質油は粘度が高いため、200~250°C程度に加熱し、搬送性を高め、バーナ噴霧での微粒化を促進し、良好な燃焼が得られるよう計画される。重質油はバナジウム、ナトリウム等のアルカリ成分や硫黄分が多く、これらに起因するボイラ伝熱管の腐食が懸念され、対策が検討されている。NO低減対策としての還元燃焼と燃料の高硫黄分と相まって、火炉水冷管に還元硫化腐食が懸念され、部分的に耐腐食性の合金を溶射する、といった対策がとられる。アルカリ硫酸塩とバナジウムが反応し、500~600°C程度の低融点化合物が生成される。ボイラの過熱器、再熱器では、伝熱管の外表面温度がこの融点以上となるため、腐食に対し注意を要する。対策としては、伝熱管の材質の選択、伝熱面の配置と蒸気温度の選定、化合物の融点を上げる添加剤の注入等が図られる。アルカリ硫酸塩の融点は800~900°C程度で、伝熱管の表面温度が高くなる火炉に面した過熱器では配慮が必要で、材質、蒸気温度の選定などに注意する(表b. オリマルジョン(orimulsion)オリマルジョンとはオリノコタールと呼ばれる天然タールに、水(約30%)と界面活性剤を加えてエマルジョン化し、常温でハンドリングを可能にした燃料である。オリノコタールは、ベネズエラのオリノコ川流域に豊富に表3・3 燃料中の成分と腐食形態バナジウム化合物腐食形態と特徴腐食因子の溶融点例:2NaO・3VO565°C特徴・低融点化合物生成による溶融塩型腐食・経験上、V 分、25ppm以上、V/Na比、3以上で問題になる場合が多いアルカリによる腐食腐食形態と特徴アルカリ硫酸塩アルカリ硫酸塩錯化合物腐食因子の溶融点例:NaSO884°C特徴・硫化腐食・オーステナイト系ステンレス鋼の使用では腐食量軽減腐食因子の溶融点例:NaFe(SO)624°C特徴・低融点化合物生成による溶融塩型腐食5―59埋蔵され、約2670億バレルの可採埋蔵量があると推定されている。水分が30%含まれ、せん断力を加えると粘度が変化する非ニュートン流体である。水分があることから重油に比べ、火元は多少離れるが安定した燃焼が確認されている。硫黄分が高いため、バーナゾーンでの硫化水素による硫化腐食を防ぐ必要がある。バーナゾーンの水冷管に耐腐食性の金属溶射を行う、燃焼空気の一部を還元性雰囲気の火炉壁に沿って導入しカーテンエアを作って腐食を抑制するバウンダリーエアシステムを採用する、などの対策がとられる。燃料中にバナジウムが多く、バナジウムアタックといった高温腐食が懸念されるため、発生蒸気温度を抑える、過熱器には耐高温腐食性に優れたステンレス系材料を使用する、などの配慮が必要である。灰分が最大0.25%程度含まれ、伝熱面への付着を考慮してすす吹器を適正に配置する、詰まりを考えて伝熱管間スペースを十分確保するなど、従来の重原油焚きボイラよりも灰に対する配慮が必要となる。エマルジョン燃料であり局部加熱や急速加熱、せん断によってエマルジョンを壊す可能性があるため、燃料系統には次のような考慮を要する。・弁にはせん断のかからないボール弁を使用する。・重原油の加熱器は蒸気加熱が多いが、過熱防止のため温水加熱方式とする。・流量調節はせん断のかかる調節弁方式をやめ、噴燃ポンプの回転数制御方式とする。
