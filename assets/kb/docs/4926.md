# 5・5・3 環境負荷低減製品開発

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・5 環境負荷低減材料および製品開発

EUの規制有害物質の主用途は，電機・電子部品に一般的に使用･カドミウム：リレー接点，プラスチック安定剤･水銀：スイッチ，センサ･PBB，PBDE：プリント基板，外装部品フリーはんだであり，各社が導電性接着剤，ペースト，新合金，基はんだの性能まで改良が求められている．
日本化学物質排出把握管理促進法（PRTR）：対象物流通商品が10万種類といわれる化学産業では，自主活動としてと，つまり経済社会活動全体で環境効率（ファクタ）を4倍にする家電・OA（oﬃce automation）の新製品の一指標になりつつあ市場メカニズムを活用した環境改善認証として，環境ラベルがあされている．
リサイクルコストの削減に結びつくとともに，消費者への製品PRにもつながる．
また，製品・サービスが生み出す質的価値や経済効果の拡大とその生産や使用段階での環境負荷の低減を対比させる指標が環境効率であり，1990年にドイツのワイツゼッカーが提唱したファクター4 が代表例である．
地球規模での平均生活水準を2倍に向上する一方，資源消費を2分の1に抑えるような経済社会を構築するこ構想である．
