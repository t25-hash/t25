# 5・1・1 携帯電話の通信方式について

> **章**: 第5章 情報通信機器
> **節**: 5・1 電話（携帯電話）

2001年の主流は，第二世代通信方式のディジタル携帯電話であ
った．全世界的に見ると，ディジタル携帯電話といってもいろいろ
な方式が存在する．ここではまず各方式の紹介と，全世界での占有
率について概要を示す．
a.第二世代通信方式
.GSM（global system for mobile communications )
1982年に採用されたディジタル携帯電話のヨーロッパ統一標準方
式である．アジア，アフリカ，アメリカなど世界の130箇国以上で
採用されている．世界占有率は，約55%である．
.IS-95（cdma one） CDMA（code division multiple
access ：符号分割多元接続）技術を使ったアメリカ標準の規格であ
る．日本，韓国，香港の方式もIS-95を基にした仕様で使われて
いる．この方式の世界占有率は，約15%である．
.IS-136 アメリカで使用されているTDMA（time
division multiple access ：時間分割多元接続）方式の第二世代シス
テムの一つである．TDMA方式は，GSMやPDCなどでも使用さ
れている技術である．IS-136は，D-AMPS（digital - advanced
mobile phone service)とも呼ばれる．世界占有率は，約15%である．
.PDC（personal digital cellular ) 日本のディジタル携
帯電話方式の規格である．NTT移動体通信網が中心になって開発
し，1992年にサービスを始めた．世界占有率は，約10%である．
以上がおもな第二世代の通信方式であるが，各通信方式とも，同
一の方式が採用されている地域でしか，携帯電話が使用できない．
このような課題を解決し，よりいっそうの高速通信を実現しようと
しているのが，第三世代の通信方式である．
b.第三世代通信方式 第三世代通信方式では，世界標準化の
取組みが行われている．それがIMT-2000（international mobile
telecommunication2000）という規格である．IMT-2000は，国
際電気通信連合（ITU:International Telecommunication
Union）が標準化を進めている第三世代携帯電話の世界標準規格
で，高品質の音声通話や最大2Mbpsの高速データ通信が可能であ
り，第二世代に比べて約40倍の高速回線になる．
.W- CDMA（wideband code division multiple access )
第三世代携帯電話システムで採用されている標準の一つで，日本や
ヨーロッパから提案された規格である．日本では，2001年10月か
らサービス（FOMA:freedom of mobile multimedia access ）が
開始されている．
.cdma 2000 現在のcdma oneを基に，第三世代携帯電
話システムで採用されている標準の一つで，アメリカから提案され
た規格である．日本では，2002年4月にサービス（cdma20001x）
が開始された．
このように，第三世代通信方式でも二つの標準があり，GSM，
IS-136，PDCは，W- CDMA方式へ，またIS-95は，cdma2000
方式へ進化している．第二世代から第三世代になることで，より高
速通信が可能となり，音声や文字，静止画だけでなく，動画像まで
扱えるようになる．また，一つの携帯電話で，日本国内だけでな
く，海外との通話もできるようになる．
