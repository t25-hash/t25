# 3・3・1 LNG 汽力発電

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・3 LNG 火力発電

a. LNG 燃料。 LNG の特徴液化天然ガス(LNG : liquefied naturalgas)とは、天然ガスを-162°Cの超低温まで冷却して液化したものである。LNG を火力発電用燃料として使用する場合、石油、石炭と比較して以下のような特徴がある。・LNG の主成分はメタンなどの分子量の小さい炭化水素であるため、燃焼性が良く、ばいじんが発生しない。また液化する際に硫黄分を除去するため、燃焼してもSOが発生しない。さらに窒素分も含まないため、NOの発生を抑制できることから、石油や石炭と比較して優れた環境特性を有している。・石油や石炭と比較して発熱量当たりの炭素分が少ないため、発電電力量当たりのCO発生量が少なく、地球温暖化への影響が少ない燃料といえる。・産地が比較的世界に広く存在するため、中東に偏在する石油と異なり、安定的な供給が期待できる燃料といえる。. LNG の導入経緯日本が高度経済成長時代を迎えた1960年代、電気事業者は急増する電力需要に対応するために多くの電源立地を計画するとともに、社会問題となり始めた環境問題に対応するためにLNG の導入が検討され、1970年に東京電力(株)南横浜火力発電所で初めて導入された。その後、数多くのLNG 専焼プラントが建設されるとともに、環境問題への対応から、既設石油焚きプラントのLNG 焚きへの改造も多数行われた。この結果、LNG の使用量は着実に増加し、現在LNG による発電電力量は日本全体の約4分の1を占めるに至っている。b. LNG 汽力発電設備LNG 汽力発電設備の一例を図3・16に示す。LNG を燃料とした汽力発電設備では、石油、石炭を燃料とした汽力発電設備と比べ、燃料基地設備、ボイラ設備、排煙処理設備が異なっている。ここではこれらの相違点について説明する。. LNG 基地設備LNG 基地には天然ガス産出地で液化5―60エネルギー供給システム図3・17 ORV 構造図されたLNG が輸送されてくるが、発電設備ではガス燃料として利用するため、払出し時にLNG を気化させる必要がある。LNG の気化はオープンラック式気化器(ORV : open rackvaporizer)にて海水と熱交換させて行う方法が一般的である。ORV は図3・17に示すとおり、アルミニウム製のフィン付チューブを数十本並べたパネルを複数枚並列につなげた構造で、下部からLNG を流し、パネル外側に海水をかけることによりLNG を気化させている。. LNG 焚きボイラ設備(1) バーナガス燃料は液体燃料や固体燃料と異なり、その燃焼過程における蒸発過程を省略することができるので、LNG の燃焼性は石油、石炭より一般的に良い。そのため、燃焼装置も比較的簡単で、バーナからガスを高速噴射するだけで燃焼させることができる。バーナはガスと空気との混合方式の違いにより、拡散混合型バーナと予混合型バーナに大別することができる。前者は、バーナノズルからガスだけを噴射し、空気を周囲から供給して燃焼させる方式で、後者は、空気の一部または全部をあらかじめガスと混合させて燃焼させる方式である。(2) 火炉寸法LNG は燃料中にバナジウムなどを含まないことから、火炉出口ガス温度は高くしても高温腐食の懸念がないため、火炉をコンパクトにすることができる。しかし、火炉を小さくするとNO発生量が増加してしまうため、火炉の大きさは石炭焚きよりは小さいが、石油焚きと同程度となる。(3) 伝熱面LNG は燃焼生成ガス中にばいじんを含まない図3・16 LNG 汽力発電設備表3・4 ボイラ効率と熱損失の一例LNG 焚きボイラ石油焚きボイラ石炭焚きボイラAH 出口ガス温度99°C140°C135°Cボイラ熱損失乾排ガス損失2.7%4.3%4.3%燃料中の水素分による損失10.2%6.5%4.0%空気中の湿分による損失0.1%0.1%0.1%放散熱損失0.2%0.2%0.2%未燃分損失0.0%0.0%0.5%その他損失1.0%1.0%1.5%ボイラ効率85.9%87.9%89.4%ため、スートブロワを設置する必要がない。また、ばいじんによるチューブ摩耗の懸念もないため、石油焚き、石炭焚きと比較すると後伝部のガス流速を速くすることができる。(4) ボイラ効率LNG は燃料中に硫黄分がなく、SOによる低温腐食の懸念がないため、空気予熱器の出口温度を低くすることにより顕熱損失を少なくすることができる。しかし、燃料中の水素分が多いため、燃焼生成ガス中の水分が多く潜熱損失は大きい。したがって表3・4に示すように、石油焚き、石炭焚きと比較するとボイラ効率は若干低くなる。(5) その他LNG の主成分はメタンであり、そのガス比重は約0.6と空気より軽く、爆発限界は5~15%と低いため、十分なガス漏えい対策が必要になる。. 排煙処理設備LNG は燃焼生成ガス中にSO、ばいじんを含まないため、脱硫装置(desulfurization equipment)、集じん装置(dust collector)は設置する必要がない。ただしNOは、燃料中に窒素分を含んでいないが、燃焼空気中の窒素が高温状態で酸素と反応することによりサーマルNO(thermal NO)が発生するため、国内の新設プラントでは脱硝装置(NOremovalequipment)を設置するのが一般的である。汽力発電設備では、サーマルNOの発生を抑制するため、低空気比燃焼、二段燃焼、排ガス循環、低NOバーナなどの技術が採用されており、一例として炉内脱硝を図3・18に示す。これは、炭化水素系燃料の還元脱硝性を利用した燃焼方式であり、主燃焼域の上段(後方)に還元性燃焼ガスを吹込み、この領域を還元域としてNOを還元させ、さらにその後段で空気を加えて完全燃焼させる方法である。c.
