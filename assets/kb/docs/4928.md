# 3・3・2 LNG 複合発電

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・3 LNG 火力発電

a.LNG複合発電の動向 LNGを燃料とする複合発電（com-
bined cycle power generation）では，ガスタービンを中心とした
ブレイトン/ランキンサイクルが用いられている（図3･20)．燃焼
ガスを作動流体とするガスタービンのブレイトンサイクル
（Brayton cycle）を高温側のサイクルに用い，その燃焼排ガスを，
低温側における水蒸気を作動流体とするランキンサイクル（Ran-
kine cycle）の加熱流体として利用するものである．すなわち，汽
力発電における蒸気タービンの高温化に対する制約が克服され，か
つ，ガスタービンの排ガスのエネルギーを有効利用することで，プ
ラント総合熱効率向上を実現している．わが国においては，
1100℃級ガスタービンによる本格的なLNG複合発電が1984年か
ら運転を開始し，発電機端熱効率で約46 %に達している．また，
1995年ごろからは，1300℃級以上のガスタービンによる複合発電
が順次導入され，55 %以上の熱効率を達成している．
b.LNG複合サイクルの種類 各種LNG複合サイクルの特徴
について以下に記述する．
.排熱回収サイクル（図3･21) ガスタービンで仕事をし
た後の排ガスを排熱回収ボイラに導き，その排熱を回収して蒸気を
発生し，蒸気タービンを駆動する方式である．近年のガスタービン
高温化により，ガスタービン単体での熱効率が上昇し，また排ガス
も高温化していることから，最も高い熱効率が得られる複合発電方
式である．さらに，制御系をはじめシステム構成が単純であるこ
と，運用特性に優れていることなどの利点もあるため，今日の複合
発電の主流となっている．
.排気助燃サイクル（図3･22) 排気助燃式は，ガスター
ビン排ガスに燃料を追加投入し，排ガス中の残存酸素を使って燃焼
させ，排ガス温度を高めて排熱回収ボイラに導く方式である．排ガ
ス温度が低いガスタービンを使用する場合には，排熱回収ボイラの
耐熱温度範囲内において採用可能であり，出力，効率の向上を図る
ことができる．
.排気再燃サイクル（図3･23) 排気助燃式と同様にガス
タービン排気に燃料を追加投入する方式である．ボイラ押込み通風
機の代わりにガスタービンを設置し，その排気をボイラに導きその
余熱を回収するとともに，ボイラ燃料の燃焼用空気として利用する
ものである．蒸気タービンサイクルが主体となり，ガスタービンの
出力割合は少ない．既設汽力発電にガスタービンを追加設置するリ
パワリングとして採用されている．
