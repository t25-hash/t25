# 5・1・4 第三世代携帯電話と関連技術の開発動向

> **章**: 第5章 情報通信機器
> **節**: 5・1 電話（携帯電話）

2001年10月から，第三世代携帯電話であるW-CDMA方式の
サービスが開始された．その伝送スピードは，384kbpsという高
速であり，第二世代の9600bpsと比べると40倍となる．このス
ピードによって動画像も見られるようになり，携帯テレビ電話のよ
うな使い方が可能になる．ここでは，第三世代携帯電話と関連し，
今後開発が進むと考えられる技術について紹介する．
a.カメラ技術 前述したように，静止画だけでなく，動画像
を扱えるようになるため，カメラ付き携帯電話の高機能化が求めら
れる．静止画用では，11万画素のCMOS（complimentary metal
oxide semiconductor ）カメラを使用しているが，今後35万画素
以上の画素数が求められ，さらに，画像を記録するためのメモリ機
能の充実も検討されている．ディジタルカメラやビデオカメラのよ
うな光学ズーム機能の小形化も要求されてくるだろう．カメラ機能
の使用頻度が増えれば，携帯電話の形もその使用形態に適した形状
に変化していくと考えられる．
b.個人認証技術 携帯電話の新たな利用方法として，電子商
取引（EC:electronic commerce）が考えられている．IMT-2000
の規格では，ICカードの搭載が規格化されており，UIM（user
identity module）カードが採用されている．UIMカードによって
その端末が正規なものであると確認でき，さらに，使用者が本人で
あるかどうかを非接触ICカードにより確認する方法が検討されて
いる．非接触ICカード以外の個人認証技術としては，バイオメト
リクス（biometrics ）を使った技術開発も行われている．
バイオメトリクスには，指紋認証，虹彩認証，声紋認証，署名認
証などがあり，それぞれ研究が進められている．特に，指紋認証
は，携帯電話への搭載を意識したモジュールが開発されつつある．
また，カメラ付きが主流となれば，虹彩認証も実用化しやすくな
る．今後いっそう開発に拍車がかかると予想される．
c .三次元高密度実装技術 第三世代携帯電話では，第二世代
に比べ，さらに多くの機能を盛り込んでいくことになる．しかし，
携帯電話としては，小形，軽量化が求められる．これらを実現する
ために，さらなる半導体の高集積化が求められるとともに，複数の
部品を一つのモジュールとして小形，一体化する三次元高密度実装
技術の開発が行われている．例えば，LSIや抵抗，誘電膜を回路基
板の内層に埋め込んでしまう技術である．ほかにも，ベアチップ実
装やCSPのさらなる微細化なども検討されている．
高密度実装技術の進化は，本体の構造設計においても，十分に着
目し，考慮しなければならない重要なポイントである．
