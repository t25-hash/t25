c .伝送技術 ファクシミリにおける伝送技術は，変復調技術
（電話網用ファクシミリに対応）と冗長度抑圧技術の二つに分けら
れる．
電話網用のファクシミリに対応した変復調技術はITU（国際電
気通信連合）により標準化されており，使用されているのはV. 27
ter （4800bps ），V.29（9600bps），V.17（14400bps），V.
34（33600bps ）およびそれに付随する規格である．近年，ビジネ
ス用途のファクシミリでは，V.34に適合した，通称スーパーG3
といわれる機種が主流となっている．
冗長度抑圧技術は，送信する画情報の主走査方向の白または黒の
連続ビット数に対し，あらかじめ発生頻度を想定して定められたコ
ードを与えることにより冗長度を抑圧する一次元符号化と呼ばれる
方式と，主走査方向のデータと副走査方向のデータ（次ライン）の
画情報の相関を基に符号化する二次元符号化と呼ばれる方式が用い
られている．これら符号化方式はITUにより標準化されており，

huﬀman）符号化方式，二次元符号化方式のMR（modified
read）符号化方式が標準化されている．またITU-T勧告T.6で
MR方式を拡張したMMR（modified modified read）符号化方式
が標準化されている．これは，伝送路でのデータ誤りを再送により
