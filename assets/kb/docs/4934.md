# 3・5・2 ディーゼルコージェネレーション

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・5 熱併給発電

a. ディーゼルエンジンの特徴ディーゼルエンジン(dieselengine)は往復動内燃機関の一種であり、化石燃料を使用する各種熱機関の中で、最も高い熱効率を有する。また、その信頼性、耐久性も卓越していることから、発電用はもちろんのこと、船舶の推進用、建設機械の大多数、大半のトラック、バス、一部は乗用車にと、広く利用されている。このディーゼルエンジンは19世紀末ドイツのルドルフ・ディーゼルの発明になるもので、以来100年以上にわたって改良が施され、軸出力1kW 以下のものから7万kWに近いものまでが生産されている。往復動内燃機関(reciprocating engine)では、シリンダ内をピストンが往復運動することにより吸入-圧縮-燃焼・膨張-排気を行い、これをクランク機構によって軸の回転運動に変換する。このように燃焼が間欠的であるため、特に高耐熱の材料を使わなくても高温高圧での燃焼が可能であり、高温であればあるほど、熱力学上高効率となるところから、連続燃焼方式のガスタービンに比べ一般的に熱効率が高い。さらに理論上、圧縮比を増すほど、また空気過剰率を増すほど、熱効率を高くすることができるため、燃料の自己着火のため高い圧縮比を設定し、かつ空気過剰状態で燃焼させるディーゼルエンジンの熱効率を高いものにしており、大型の数万馬力のディーゼルエンジンでは50%を超える熱効率に達している。また、使用燃料は軽油または大型の機関では重油で、比較的燃料費が安いこともあってさらに経済的である。動作サイクルは四サイクルと二サイクルがあるが、今日では発電機の駆動用では特に大型のものを除けばその熱効率のよさや、排ガス成分の問題でほとんどが四サイクル機関である。発電機駆動用としてのディーゼル機関の特徴としては、(1) 起動後短時間で負荷を投入可能ごく大型のものを除いて暖機運転は短時間でよく、必要時にただちに負荷を投入できる。これは往復動内燃機関に共通する特徴で、この特徴を使って防災用の発電セットとして10秒起動のものもディーゼルエンジンで駆動されている。(2) 負荷変動に追従性がよい発電機を駆動する場合、一部の定電力制御を行っている設備を除5―81表3・8 排ガス成分と環境問題環境問題排ガス成分地球温暖化CO,NOオゾン層破壊NO酸性雨NO,SO光化学スモッグHC,NO呼吸器系障害NO,SO,PM循環器系障害CO図3・57 排ガス成分き、電力負荷は常時変動している。これに見合って機関出力を加減する必要があるが、ディーゼルエンジンでは他の熱機関と比べて容易に出力調整が可能である。(3) 軽負荷でも効率低下が少ない一般に熱機関は定格の出力に対して負荷が軽いと急激に効率が落ちるが、ディーゼルエンジンの場合は定格出力の30%程度まで出力を落としても大幅な効率低下は生じない。b. ディーゼルエンジンの排ガス1997年、京都で開かれた「気候変動に関する国際連合枠組み条約」の「第3回締約国会議(COP3)」で二酸化炭素をはじめとする温室効果ガスの削減目標が決められた。ディーゼルエンジンは他の熱機関と比較しても熱効率が高いため燃料の使用量、すなわち二酸化炭素の排出量も少なく、この点で非常に有利である。そのほかに、エンジンから排出される排ガス中に含まれる種々の有害物質による地球環境汚染も問題となっている(表3・8)。燃料中の炭素分(C)、水素分(H)、少量の硫黄分(S)が、大気中の酸素(O)、窒素(N)とが燃焼して高温となり、さまざまな有害物質が生成される(図3・57)が、これら生成物が近年の石油燃料の消費増大により、酸性雨や光化学スモッグ、オゾン層破壊を招くとして排出の抑制に向けて種々対策が執られている。ディーゼルエンジンでは特に窒素酸化物と粒子状物質(suspended particulate matter、黒煙を含む)および燃料中の硫黄分による酸化硫黄(SO)の排出割合が多く、他の熱機関に対してかなり劣っている。近年、アメリカをはじめとする先進国はもとより、各国でこれら排ガス中の生成物に対し量的規制をかけてきている。現在の規制の対象は、国によっても異なるが、窒素酸化物(NO)、炭化水素(HC)、一酸化炭素(CO)、粒子状物質(PM)、および黒煙である。コージェネレーション用のディーゼルエンジンのような定置式の機関の場合、日本では、窒素酸化物の排出量のみが法的には規制されており、全国一律の規制がかけられている。ただし、大都市部の人口密集地域では、さらに厳しい指導基準が定められている。これら規制は、将来的にはますます厳しくなる方向であり、対象とする有害物質の範囲も広がることが予想される。この規制に対し、ディーゼルエンジンの改良が続けられている5―82エネルギー供給システムが、ディーゼルエンジンだけの改良では限界があり、アンモニアや尿素を還元剤として、触媒を使って排ガス中の窒素酸化物を窒素と酸素に分解する排ガス後処理装置も使われるようになっており、黒煙や粒子状物質を排ガスからこし取るDPF(diesel particulatefilter)も使われるようになると思われる。さらに、これら有害物質の生成に大きく関与しているのが燃料である。酸化硫黄を生じさせ、粒子状物質の増大や触媒の使用を制限する燃料中の硫黄分は、できるだけ少ないことが求められており、排ガス中の有害物質の生成を抑制する燃料の組成が種々な方向で検討されている。c. ディーゼルコージェネレーションコージェネレーションシステムでは、電気を主として使用することを目的とするか(電気重視)、熱の回収を重視するか(熱重視)で原動機の選択が行われるが、一般的にはエネルギーとして使いやすい電気重視が多い。電気重視の場合、発電機の駆動用原動機は、b項で述べたように熱効率に優れたディーゼルエンジンを選択することが多い。ディーゼルエンジンでの発電効率は(1基当たりの発電出力の規模によるが)一般的には35~50%であり、よく使われる500kW クラスではよいものは45%前後である。熱回収に関してはエンジン冷却水の排熱だけを使用する温水回収型、排ガスの熱を排ガスボイラで回収して蒸気を取出す型などがあるが、電気出力と熱回収を最大限行った場合にはエネルギー回収効率は85%ほどに達する。ディーゼルコージェネレーションは電気出力の効率はよいが、b項で述べたように排ガスを無害化するのが非常に難しいため、大都市圏など規制が厳しいところに設置する場合には慎重な検討を要する。図3・58にディーゼルのコージェネレーションシステムフロー図を示す。一般に往復動内燃機関を用いる発電機やこのようなコージェネレーションシステムでは、内燃機関の故障や定期整備の場合に停止させる必要から、この例のように複数台を組合わせてシステムを組むのが一般的である。この例のシステムでは、熱を使用しきれないときのために温水用の熱交換器のほか、エンジン用の冷却塔を別にもち、排ガスボイラで蒸気を取出して吸収冷凍機に供給している。また、このシステムはNO削減のための脱硝装置も設置してある例でもある。
