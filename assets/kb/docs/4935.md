# 5・3・2 モバイル情報機器の構成

> **章**: 第5章 情報通信機器
> **節**: 5・3 モバイル情報機器（PDA）

キーボードとペン入力を備えた本格的なモバイル情報機器の構成
例を図5･15に示す．入力デバイスとして，タッチパネルやキーボ
ード，表示用にLCD（liquid crystal display) を装備するのが通
常である．プログラム用とデータ記憶用のメモリと処理用のCPU
（central processing unit ）などから構成される．ネットワーク接
続用ポートに携帯電話などの通信モジュールを接続すれば，通信機
能付きのPDAとなる．
