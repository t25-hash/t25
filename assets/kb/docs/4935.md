# 3・5・3 ガスエンジンコージェネレーションシステム

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・5 熱併給発電

a. ガスエンジンコージェネレーションシステムの概要。 特徴ガスエンジンコージェネレーションはガスエンジン(gas engine)を原動機とした発電機に熱回収機器を組合わせたシステムである。ガスエンジンは排ガスがクリーンであることと熱回収がしやすいという二つの特徴をもつため、ガス管が埋設されている市街地で普及が進んでいる。また2時間分の燃料タンクをもつ図3・58 ディーゼルエンジンコージェネレーションシステムフロー図図3・59ガスエンジンコージェネレーション主要機器配管系統図(蒸気回収システム)[出典:三菱重工業(株)技術資料から抜粋]か都市ガス管から燃料を供給されていれば非常用発電機として認可されることから、非常用発電兼用機としての導入事例も増加している。. マイクロコージェネレーション(micro-cogeneration)10kW 未満のエンジン発電機は電気事業法の規則により一般用電気工作物との扱いとなるため、設置と運用が簡便なので、発電出力が1~9.9kW 級のものが製造され、マイクロコージェネレーションと称される。おもに家庭用、小型店舗向け等で使用される。. ディーゼルコージェネレーションとの相違ディーゼルエンジンとガスエンジンは燃料が異なるのみで構造はよく似ている。その結果、熱回収回路等その他も全体的に相似なものとなっているが、燃料が都市ガスの場合、燃料タンクがないという点が大きく異なっている。それに付随する燃料移送ポンプもないため燃料系統の配管は簡素なものとなっている。その代わりに発電機盤の中を見ると点火装置、ガス流量制御装置などが取付けられて複雑なものとなっている。外観的に変わるところはほとんどない。. ガスエンジンの効率ガスエンジンは低NO化の方式により熱効率が異なり、三元触媒方式では30~35%、希薄燃焼方式においては38~42%の効率が得られている。さらにミラーサイクル(miller cycle)を希薄燃焼に組合わせた燃焼方式を採用したガスエンジンでは、もう一段の高効率化が得られる。発電効率は最高で40%程度となる。. ガスエンジンコージェネレーションの総合効率効率を計算するのに損失割合を推定し、その合計を100%から差引く方法で求める。(1) 排ガス損失排ガス熱量を150°Cまで熱回収すると、排ガスによる放熱量は希薄燃焼方式で排ガス熱量の30~40%となる。排ガス熱量が燃料入熱量の30%程度を占めるとすれば、全体としては9~12%くらいの損失となる。(2) ヒートリゼクションの値ヒートリゼクションの値はエンジン単体では3~5%の範囲であるが、コージェネレーションとした場合は表面積が大きくなるので7~10%程度となる。(3) 発電機損失発電機効率を94%とすると損失は6%であり、発電機出力は燃料入熱量の40%なので損失割合は2.4%である。(4) ガスエンジンコージェネレーション特有の構成機器。 ガスエンジンガスエンジンはメタン、プロパン等の気体燃料を燃焼させて回転動力を発生させる。排ガス中のNO濃度が低く、排ガス中のすすもないため市街地で使用するのに適している。燃料のC-H 比が軽油、重油等の液体燃料より小さいので、二酸化炭素の発生量が約20%少なく、地球温暖化防止にも役立つ。外観的にはディーゼルエンジンとほとんど同じである。(1) ガスエンジンの分類・低NO化による分類ガスエンジンを低NO化方式により分類すると、希薄燃焼方式と三元触媒方式に分かれる。希薄燃焼(lean burn)には副室式と単室式があり、過給機付きが一般的である。エアクーラを二段に分けて熱回収量を増やしたものもある。三元触媒方式は空気過剰率が1のストイキ(stoichiometry)燃焼をさせるため排気温度が高く、熱回収率が高い。過給機付きのものとなしのものがある。・燃料混合方式による分類燃料ガスと空気の混合方式で分けると、過給機の前で混ぜるものと後で混ぜるものに分けられる。前で混ぜるものを予混合方式という。各気筒の入口に噴射して混合するものをポートインゼクション方式という。. ガスエンジン発電機ガスエンジンはNOを抑制しつつ熱効率の高い燃焼をさせるために、ノックセンサや燃料ガスの電子制御機器等を取付けて燃焼制御の電子化を行っている。それらの制御をフィードバックするためにエンジンには種々のセンサが取付けられている。そのため制御盤とエンジン間の配線が多い。発電機そのものはディーゼルと全く同じものが用いられる。. ガス圧制御弁燃料ガス供給ラインに入れるガス圧制御弁はエンジンへの燃料ガス供給量を制御するためのものである。これをエンジンから離れたところに設置するとガスの流速により制御の遅れが生じるので発電機の直近に取付ける。. ガスコンプレッサ希薄燃焼ガスエンジンには主燃焼室のほかに副室を備えて燃焼のコントロールを行っているものが多い。副室に燃料ガスを送り込むためにはエンジンの過給圧より高い圧力が必要なことから副室ガス用コンプレッサが副室ガスラインに設置されている。. ガス配管ガス会社の配管には敷地への入口にガスメータと遮断弁を備えたメータボックスが付いており、その先構内配管を経て発電機室近辺まできたところに区分のバルブが設置される。区分のバルブまでの配管はガス供給者が施工するので、その先はコージェネレーション設置者が設計施工しなければならない。ガスの配管はすべて溶接してつないでいるので配管のフラッシングを施工後に行うことは困難であるから、目の細かいフィルタを電磁弁の前に取付けておく。d. ガスエンジンコージェネレーション使用上の留意点ガスエンジンはプロパンタクシーのエンジンとして街中を多数走り回っているものと同じであり、基本的には普通のガソリンエンジンやディーゼルエンジンと同じに取扱うことができるが、エンジン冷態時は起動しにくい、専用オイルを用いる、排ガス中に水分を多く含むなどの点に留意する。. 冷態時の起動都市ガス燃料の主成分はメタンであり、これは不活性ガスに分類されるものである。したがって、特に気温が低くなる季節には起動しにくくなるので、ウォータヒータにて冷却水を加温して良好な起動性を保っているが、年末年始の長い休みにウォータヒータの通電をストップすると休み明けの起動が困難なものとなる。これを1日前くらいから通電しておけば起動の問題を回避できる。. エンジンオイルエンジンオイルにはアルカリ価を保つためにカルシウム等のアッシュ分が配合されているが、ガスエンジ5―84エネルギー供給システムンの場合にはアッシュが燃焼室内に堆積するとノッキングを引き起こす原因となる。これを防ぐためにアッシュ分の少ないエンジンオイルをガスエンジン用として設定し使用しているので、他のオイルを転用しないように留意する。. 排ガス中の水分メタンの主成分が水素であることから燃焼ガス中には水分が多く含まれている。高温時には水蒸気として出ていってしまうので問題になることはないが、低温時には水蒸気が水分となるため排気管中に水が大量にたまったり、排ガスが白色となったりすることがある。排気管のドレンバルブを開放しておけば実質的な問題は生じない。
