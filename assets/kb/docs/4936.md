# 5・3・3 モバイル情報機器用OSの動向

> **章**: 第5章 情報通信機器
> **節**: 5・3 モバイル情報機器（PDA）

モバイル情報機器用のOSとしては，消費電力の小さなCPUで
動作し，必要とするプログラムサイズの小さいものが必須であった
が，半導体技術の進展により必ずしもこれらが最重要ではなくなっ
てきている．むしろ，多くの機能を容易に追加できることやオープ
ン性が重要となっている．携帯情報端末はPIM機能などに加えて
多くの高機能を実現するため，CPUは8ビット→16ビット→
RISC（reduced instruction set computer ）へと高性能化し，プロ
グラム容量は年々倍のスピードで増加している．現在，携帯情報端
末用OSとしては下記のものが開発されている．その概要を以下に
記述する．
a.PalmOS アメリカのUSロボティクス社が1996年5月
にPilotとして発売したPDAに初めて搭載されたOSである．全
メモリ128kB（キロバイト）でPIM機能を実現していた．現在で
は，アメリカのパームソース社が提供している．バージョンアップ
が進められ，世界のPDA市場の7～8割のシェアをもつといわれ
る（1999年IDC調べ）．OSの機能を拡張できることなどが特徴で
ある．マルチスレッドが可能で，画面制御と通信制御などを同時に
行うことができる．シングルタスクOSであり，アプリケーション
の応答性の速さやシンプルなヒューマンインタフェースが特徴であ
る．アプリケーション開発環境が各種プラットホーム上に整備され
ており，そのため，シェアウェアソフト，フリーウェアソフトが充
実している．
b.EPOC（エポック) イギリスのPsion（サイオン）社の
開発したOSでSymbian（シンビアン）プラットホームのOSで
ある．16ビットから始まり，32ビット対応のEPOC32となって
いる．現在は，シンビアン社が提供しており，移動体通信デバイス
向けへの改良がなされている．高信頼性，機能ごとにコンポーネン
ト化（オブジェクト指向），モジュラ指向と柔軟性，ミドルウェア
が充実，すべてのソースコードの公開を特徴としている．PDAだ
けでなく，携帯電話用組込みOSとしても採用されている．
c.WindowsCE（ウィンドウズシーイー) マイクロソフト
社がPegasusというコードネームで開発していた，家電製品向け
に開発されたOSである．WindowsNTと同様にマルチプラット
ホームをサポートしていることが特徴である．日立製作所のSH
シリーズ，MIPS系CPU，PowerPC，ARM，x86アーキテクチ
ャのサポートが決定している．マルチタスク/マルチスレッドをサ
ポートした32ビットOSである．周辺ライブラリ，ドライバなど
の機能が豊富で，マイクロソフト社がコンパイラからデバッガまで
の開発環境を提供し，Windows PCとのデータ互換性があり，強
力なグラフィカルユーザインタフェースや通信機能，さまざまな
MPU（micro processing unit ）で利用可能であることなどが特徴
とされている．リアルタイム性は組込み向け専用リアルタイムOS
より多少低い．
現在は，手帳型PC向けに特化されている．Pocket PCと呼ば
れるものは，このOSと電子ブックリーダやストリーミングメディ
ア再生プレーヤーソフトなどをバンドルした手帳型PC向けソフト
のことである．
d.ザウルスOS シャープが開発したわが国唯一のモバイル
情報機器用OSである．わが国におけるモバイル情報機器市場の約
5割のシェアをもつといわれる．マイクロカーネルをベースとした
リアルタイム性のあるマルチタスクOSである．オープン性の乏し
い独自OSであったが，1997年にPIMスクリプト技術とアプリケ
ーションソフトの開発環境を公開しており，2001年にはJavaによ
るアプリケーション開発環境のオープン化が行われた．現在，ザウ
ルスOSは開発中止となり，ザウルス本体にはLinux（リナック
ス）が搭載されている．
