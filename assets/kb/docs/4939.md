# 3・7・1 概説

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・7 燃料電池

昨今、燃料電池という名が大衆を魅了する新車のショーや新聞、雑誌、テレビなどで盛んに取上げられている。燃料電池というと、一般には乾電池や鉛蓄電池の類に解釈されることが少なくない。通常の電池は電気の池つまり蓄えておくものである。本来なら燃料発電槽のような表現のほうが望ましく、外部から供給するエネルギーを電気に変換する静的な装置と考えれば正しい。燃料を供給していれば装置性能が維持している間は電気を生産できることが最も大事な特徴であり、例えば発電装置とするほうが無難といえよう。燃料電池は化学から電気への直接変換で、しかも静的な機構がないので、小型でも変換効率が高く、静粛性に優れたメンテナンスフリーの発電器を実現する期待がもたれるわけである。燃料電池は最先端技術の代名詞とされているものの、1839年という昔にイギリスのグローブ卿が発明し、原理の発見に至ってはすでに2世紀も前にさかのぼる。1965年に宇宙船ジェミニ5号に搭載されたのが最初の実用化であり、しかもそれが今をときめく固体高分子形燃料電池であった。この事実は乾電池や鉛蓄電池のように古くから利用されているものとは異なり、技術的な障壁がそれほど低くはなく、かなりの部分で高度な技術が要請されることを物語っている。開発が進められている燃料電池の種類には、ほぼ実用段階のリン炭酸塩LiCO,KCO(液体)安定化ジルコニアZrO(固体)・開発が最も進み実績が多い・電解質消失あり・内部改質が可能・炭酸ガス濃縮への応用可・電解質消失あり・炭酸ガスリサイクルが必要・起動停止に長時間を要する・ニッケル短絡の懸念あり・高出力密度・内部改質が可能・メンテナンス容易・起動停止に長時間を要する・長期性能や昇降温サイクルに対する不安あり・100kW 級電池を開発・1MW パイロットプラント実証試験終了・数百W~数kW 級電池スタックの開発中・セラミックス材料、製造法を含め基礎研究段階5―9 2エネルギー供給システム酸形(PAFC: phosphoric acid fuel cell)、開発が盛んなものに溶融炭酸塩形(MCFC: molten carbonate fuel cell)、固体酸化物形(SOFC: solid oxide fuel cell)、さらに現在大きな注目を集めている固体高分子形(PEFC: polymer electrolyte fuel cell)などがある。表3・11にそれら4種類の燃料電池の特徴をまとめた。順序は作動温度の低いタイプから並べている。大雑把に特徴を総括すれば、温度が高くなると白金など高価な金属を用いた触媒が不要になり、燃料改質プロセスで生じる一酸化炭素が燃料になるものの、起動停止に伴う昇降温の制御に熱応力等の問題から神経を遣う必要が生じる。また、PAFCとMCFCは電解質が液体なので、散逸による減少とその補給に関するメンテナンス、さらには加圧した場合に差圧制御を行う必要性が指摘される。現状の開発レベルとしてはPAFCが実用段階に突入しており、MCFCが次を追っている。構成としては両端のPEFCとSOFCが望ましいように思われるが、PEFCはCO被毒および水分管理に極端にデリケートであること、SOFCは高温であること自体が、技術開発における大きな壁となっている。どの種類の電池に対しても、電極、電解質、さらにカーボンやチタンを使わなければならないセパレータはいわゆるハイテク材料であり、このあたりがやっと現代になって燃料電池が発電装置として現実味を帯びてきた理由であろう。これら種類はそれぞれの特性や規模、そして排熱利用の考え方に応じて小規模から大規模、さらにはさまざまな適用分野が提案されている。先に挙げた種類以外には、アルカリ形燃料電池(AFC: alkalinefuel cell)および直接メタノール形燃料電池(DMFC: directmethanol fuel cell)の名前をよく聞くことがある。AFCは安価でもありアポロ宇宙船に搭載された実績がある。しかしアルカリ電解溶液は空気中の二酸化炭素によって劣化するため、最近あまり注目はされていないものの一部で復活する動きもある。DMFCは液体であるメタノールをそのまま投入するだけで発電が可能であり、その意味ではかなり優れた方式であるが、実用性能にはほど遠く基礎研究が進められているにすぎない。DMFCが実現するとコストターゲットが低いうえに高い利便性をもたらすと予想されるモバイル機器への応用や、自動車用などでも他の電池にはない取扱いの容易さによって大きな技術革新が期待される。
