# 3・7・2 固体高分子形燃料電池(PEFC)

> **章**: 第3章 化石エネルギー技術・システム
> **節**: 3・7 燃料電池

a. まえがき固体高分子形燃料電池は電解質として高分子膜の一種である陽イオン交換膜(proton exchange membrane)を用いるもので,1電極単位面積当たりの発電出力が大きい,2室温から80°C程度までの比較的低温で動作するなどの特徴を有している.そのため小型高出力と同時に短時間起動が要求される家庭用の電源や電気自動車の動力電源などへの応用が期待されている.英語ではproton exchange membrane fuel cell(PEMFC)あるいはpoly-mer electrolyte fuel cell(PEFC)と呼ばれる.固体高分子形燃料電池の研究の歴史は比較的古く,1965年にアメリカでジェミニ宇宙船の電源に適用された実績がある.その後も研究は続けられてきたが,開発の初期はイオン交換膜の性能が低く,その結果電池特性が他の燃料電池に劣ったため,その他の長所にもかかわらず注目されなかった.しかし1970年代の後半にデュポン社の陽イオン交換膜(商品名ナフィオン)が市場に出るようになって,特性,寿命ともに改良された.さらに1987年にバラード社がダウ・ケミカル社の新しい膜を用いて,特定の条件下ではあるが電流密度5A/cmを超える驚異的な高出力密度を発表した.この結果,世界的に開発熱が高まり,特にアメリカ,カナダ,ドイツ,イタリア,韓国,中国,日本などにおいて家庭用や電気自動車用などに開発が進められている.b. 原理固体高分子形燃料電池は電解質であるプロトン(H)導電性を有する陽イオン交換膜と,この膜の一方の面に接合した多孔質の水素電極と,他方の面に接合した多孔質の酸素電極か図3・81 発電原理らなり,水素電極に純水素(H)または水素を含む改質ガス,酸素電極に酸素(O)または空気(空気中の酸素が反応)を供給して,室温から80°C程度で動作させるものである.図3・81にその発電原理を示す.Hは水素電極内を拡散し,水素電極と電解質膜の界面で電子(e)を放出してHに酸化し,Hは膜内を移動して酸素電極に到達する.一方eは外部電気回路を通って電気的な仕事をした後,酸素電極に達する.酸素電極ではOが電極内を拡散し,酸素電極と電解質膜の界面で膜内を移動してきたHおよび外部回路からのeと反応して水(HO)に還元する.Hが膜内を移動する際には,膜に含まれているHO分子を数個伴って移動する.これは電気浸透効果と呼ばれるものである.したがって膜の水素側は乾燥しがちになり,このことが膜の電気伝導度を低下させ,その結果電池性能が低下する.これを防ぐ方法として水素には水蒸気を含ませて電池に供給する(改質ガスの場合は,ガス中に水蒸気を含むので,加湿は不要である).一方酸素電極では水が生成するが,排空気といっしょに,水蒸気としてあるいは一部は液体の水のまま電池外に持ち出される.空気の流量が多い場合は,膜中の水分の空気への蒸発量が多くなり膜が乾燥する.この場合は空気も加湿して供給する.c. 特徴固体高分子形燃料電池の特徴を列挙すると次のようになる.. 長所1高出力密度で,コンパクト化が可能.2室温から発電できるので,起動が早い.3電解質が固体で,移動あるいは蒸発することがないので,本質的に寿命が長く,また起動停止による劣化がない.4膜の耐圧が高いので,水素と酸素間の差圧の精密な制御が不要.5電解質の蒸発や飛散がないので,腐食の問題が少ない.6電解質が酸性なので,COを含む改質ガスを燃料として使える.7運転温度が低いので,材料選択の幅が広く,安価なプラスチック等が使える.. 短所1運転温度が低く白金触媒がCOに被毒されるので,改質ガス中のCOをほぼ完全に除去する必要がある.2電池の排熱温度が60~80°Cと低く,熱利用しにくい.3運転温度を低く保つために,大きな冷却装置を必要とする.4イオン交換膜の性能を維持するために,水分量制御が必要である.5イオン交換膜がNaイオンやその他の金属イオンに汚染されやすく,それによる劣化が生じる.6イオン交換膜のコストが高い.d. スタック(stack)の構造一つの電池(セルと呼ばれる)の電圧は,発電している状態では通常0.6~0.8V 程度なので,必要な高電圧を得るためには多数のセルを積み重ねる(積層する).積層されたものはスタックと呼ばれる.スタックはセルを構成する各部材を繰返し積み重ね,これらを一体に締付けて作られる.各部材の構成の一例を図3・82に示す.図3・82において,膜-電極接合体(membrane-electrode as-sembly)はイオン交換膜の両面に白金系触媒からなる多孔質の水素電極と酸素電極を接合したものである.支持集電体は多孔質のカーボンペーパあるいはカーボン布で,電流を電極から集電する役割と同時に,薄い膜-電極接合体を支持する役目も果たす.膜-電極接合体と熱圧着で一体化している場合もある.セパレータ(separator)は一面にガスを流す溝を有し,他面に冷却水を流す溝を有する導電性の緻密なカーボン板(または耐食性金属板)である.両面ともガス溝(一面は水素,他面は空気)の場合もあるが,この場合も冷却のため,数枚ごとに片面が冷却水溝のセパレータが挿入される.セパレータはバイポーラプレート,隔壁板,インタコネクタなどとも呼ばれる.セパレータで膜-電極接合体を挟んで,一つのセルを構成し,さらにこのセルを複数個重ねてスタックとする.ガスや冷却水のシールのためにはゴム系のシールリング等が用いられる.スタックの両端には金属の集電板を配置して,外部電流取出し端子とする.さらにその外側に絶縁板を介して端板を配置し,全体をボルト・ナットで締付けて一体化する.絶縁板と端板の間に加湿部(ガスを湿らせ図3・82 スタックの構成5―9 3てスタックに供給する部分)を組込む場合もある.固体高分子形燃料電池は他の燃料電池と異なり,電極面が鉛直になるように水平方向に積層化する場合が大部分である.これは発電に伴って生成する水が,ガス流路に沿って上から下へ移動して,セル下部から外に排出しやすくするためである.また固体高分子形燃料電池は他の燃料電池と比べて冷却板の数が多く,通常1~4セルごとに冷却水が流れる.これは運転温度を80°C程度以下に低く保たなければならないこと,膜の湿潤度を適正に保つために温度を厳密に制御する必要があること,電流密度が高く発熱が大きいことなどによる.冷却水を用いない空冷方式のスタックもある.空冷の場合はスタック構造がシンプルになる利点がある.ただしスタックの体積が大きくなることや,排熱を利用しにくいなどの欠点がある.e. 膜-電極接合体の製法膜-電極接合体は発電をつかさどる最も重要なコンポーネントである.膜-電極接合体の製作方法は種々あるが,代表例として次のような方法がある.すなわち電極材料である白金系触媒粉末,ポリ4フッ化エチレン(PTFE)懸濁液(結着剤兼発水剤)および有機溶剤を混合してスラリー状にする.これを電解質膜の両面に,スクリーン印刷法あるいはスプレー法などで数十mの厚さに塗布する,あるいはスラリーをキャリアシート上にいったん塗布し,電解質膜上に転写する.その後熱圧着を行って一体化する.図3・83は上述の方法で作られた膜-電極接合体の断面SEM 写真である.触媒(catalyst)である白金は,カーボン粉末を担体として,その表面に数nmの白金微粒子を均一に分散させる.白金使用量は水素電極で電極面積1cm当たり0.1~0.4mg,酸素電極で0.2~0.7mg まで低減している.改質ガスを燃料ガスとして用いる場合は,ガス中に微量に含まれるCOによる白金の被毒を防ぐために,CO被毒に強い白金-ルテニウム合金触媒が用いられる.100 m図3・83 級発電システムを例に挙げて,図3・84に示す.図3・84において供給された都市ガスは,蒸気発生器・改質器・CO変成器・CO除去器で構成される改質系反応器により,CO濃度10ppm以下の水素リッチなガス(H:約80%,CO:約20%)に改質され,電池本体に送られる.電池本体では,発電に伴い改質ガス中の水素が70%程度消費され,未反応水素は改質器のバーナで燃焼して,改質器および蒸気発生器の熱源となる.電池本体からの電気出力はインバータ(図3・84中にはなし)により交流電力に変換される.システムからの排熱は,改質器の燃焼排ガス,電池本体冷却水および空気極排空気であるが,これらの排熱は熱交換器を介して60~70°Cの温水として回収され,熱利用される.このシステムはリン酸形燃料電池発電システムとほぼ同様な構成であるが,固体高分子形特有の次の補機が加わる.1蒸気発生器燃料改質に必要な蒸気はリン酸形では電池運転温度が高いため電池冷却水系から得られる.しかし固体高分子形は運転温度が低いため,蒸気発生は改質器の燃焼排ガスの熱を利用する.そのため改質器で燃焼させる未反応水素量を多くする必要があり,電池本体の水素利用率が低くなる.このことは発電効率を下げる原因となっている.2CO除去器CO変成器を出た改質ガス中に1%程度含まれるCOを10ppm以下にする必要があり,このためCO除去器を設ける.除去の方法は微量の空気を改質ガスに混合してCO除去器に供給し,COを白金触媒上でOと反応させて無害なCOに変える.g. 用途. コージェネレーション(cogeneration)固体高分子形燃料電池は低温作動のため,起動が容易である,安価な材料が使えて低コスト化が期待できるなどの特徴があるので,家庭用コージェネレーションとして,一般住宅,商店,集合住宅などでの利用が想定されている.作動温度が80°C程度以下なので,高温の排熱は得られないが,排ガス系からの低温排熱で温水を作り利用できる.導入のイメージは図3・85に示すようなもので,都市ガスを燃料に発電し,同時に出る熱を使って温水を作り,貯湯槽にためて,お風呂の湯,キッチン・洗面所の給湯などに使うことが考えられている.すでに試作機が数社で作られ,ガス会社などによりフィールドテストされている.家庭用燃料電池コージェネレーションシステムの試作機の一例を図3・86に示す.システムは1kW 終わりに固体高分子形燃料電池は他の燃料電池にない数々の優れた特徴をもっており,排熱利用を重視しない用途に対しては優れた発電手段になると考えられる.用途としてはまず家庭用コージェネレーションシステムや燃料電池自動車などが考えられ,さらに業務用発電などにも一部適用されていくであろう.また食塩電解工場などで副生する水素や,将来の水素エネルギー時代の水素を利用した燃料電池発電に対しては,固体高分子形は最適な燃料電池となろう.なおパソコンや携帯電話の電源として注目されているマイクロ燃料電池も固体高分子形燃料電池の一種であるが,燃料のメタノールを直接電池本体に供給する方式の直接メタノール形燃料電池として別分類されることが多いので,本項では取り上げなかった.
