b.定位脳手術からナビゲーション手術へ 定位脳手術（ster-
eotactic neurosurgery）では患者の頭部に定位脳手術フレームを
装着し，これを座標系の基準として用い，手術も定位脳手術フレー
ムを着けたまま行う．しかしこれでは撮影が煩雑で，患者の負担も
大きいうえに，開頭手術の場合にはアプローチの方向や開頭範囲が
限定される欠点がある．そこで，定位脳手術フレームの代わりに骨
もしくは頭皮に小さなマーカをいくつか装着して座標系の基準とす
る（定位脳手術）フレームレスナビゲーションが開発された（頭部
固定用フレームは使用）．定位脳手術フレームを着けた定位的脳腫
瘍生検（biopsy）や脳内血腫除去と比べるとさまざまな術式と容
易に組合わせて種々の目的に利用することができるため，現時点で
はフレームレスナビゲーション（frameless navigation）が脳神経
外科手術の主流になっている．
c .ナビゲーションのための三次元位置計測法 ナビゲーショ
ンに用いる位置計測は，再現性があって一定の精度を保っているこ
とが保証できるものが望ましい．さまざまな方式があるが特に機械
式・光学式・磁気式の3方式は精度がよく，また互いに相補的な性
質をもっているため，目的に応じて組合わせることによって，適用
対象が広く，かつ精度のよい計測が可能になる．
(１) 機械式（mechanical tracking）
6以上の自由度をもっている関節をもつアームの先端にポインタ
を有し，各関節の曲がり角をエンコーダで計測することによってポ
インタの位置を求める．最新のモデルでは小開頭部での使用におい
て0.1～1mmの精度に達している．アームの操作が煩雑な場合が
あること，一度に一つの対象物の位置しか測れないことが弱点であ
る．
(２) 光学式（optical tracking）
ポインタに取付けた光学マーカ（発光ダイオードなど）や反射型
マーカを複数のテレビカメラでとらえて，その位置を計測する．精
度は0.3mm程度である．多数の器具の位置を同時に測定できる
が，光学マーカとカメラとの間に遮へい物があると測定できなくな
る．
(３) 磁気式（magnetic tracking）
術野外部に置いた複数のコイルで磁場を交互に発生し，ポインタ
に取付けたセンサで磁場強度を計測して位置を知る．光学式と異な
り遮へい物があっても計測できる利点がある．ナビゲーション以外
に，三次元超音波装置の中にも組込まれて使われている．一方，術
野近くにある磁性体の影響などで計測値に誤差が生じていても気づ
かないおそれがあるが，最近では動的磁場はプログラム的にキャン
セルされるようになっている．
d.術中画像によるナビゲーション 手術操作による臓器の変
形や移動に追随するには術前画像だけでは不十分であり，術中にボ
リュームスキャンを行ってナビゲーション情報を更新する必要があ
る．実際，術中にCTでモニタリングすることは従来から行われて
いる．最近では毎秒6枚のCT像を撮影しながらほぼリアルタイム
で表示できるCTフルオロスコピー（fluoroscopy）が市販されて
おり，脳手術・生検・IVR（interventional radiology）などに応
用されている．
一方，インタベンショナル（interventional ）MRIに代表される
手術室でのMRIの使用が今後の主流となる．CTと違って医療ス
タッフのX線被ばくがなく，またボリュームスキャンが容易であ
る．磁場を乱す手術機器が使用できず，非磁性体で構成された特殊
な顕微鏡や手術器具が必要ではあるが，長年脳外科医を悩ませてき
た硬膜切開後の手術操作による脳の変形・移動に対処するリアルタ
イムアップデートナビゲーションとして，その有用性は高く評価さ
れている．
