# 5・2・5 手術顕微鏡（surgical microscope）

> **章**: 第5章 手術・治療機器
> **節**: 5・2 手術周辺支援機器

a.概要 手術に顕微鏡を使用することで，より細かな神経や
血管の治療，腫瘍の摘出などを短時間に的確に行うことが可能とな
る．近年，顕微鏡を導入する手術は増える傾向にある．手術顕微鏡
には，脳神経外科，整形・形成外科，耳鼻咽喉科，産婦人科，眼
科，歯科などの手術の形態に特化した専用の機種が用意されてい
る．
b.構成と機能
.顕微鏡部 手術の作業上，立体感（奥行き）を必要とす
るので実体顕微鏡の構造となっている．手術は通常二人の医師で行
うため，助手用の観察鏡筒が付く．また，同一視野をビデオカメラ
やスチルカメラで記録するため，ビームスプリッタで光路が分配さ
れる．おもな構成として，対物レンズ，焦点調整装置，変倍装置，
記録用カメラ，助手用鏡筒，照明光学系からなる（図5･14）．高級
機種では電動により手元もしくはフットスイッチによって焦点位置
や倍率調整，視野の移動などが行える．
.照明装置 術野をかげりなく観察するために顕微鏡部か
ら照明を当てる．観察光路と照明光路の角度差が大きいと，狭く深
い術部の場合，照明が奥まで照らせない．しかし，観察光路と照明
光が同一角度の同軸0°照明では，かげりはなくなるが立体感が乏
しく見える場合がある．外科用顕微鏡の場合，光学系レイアウトの
制約もあり，2° ～4°程度のものが多い．眼科用顕微鏡は，眼から
の反射の影響が大きいために照明角度を観察光路に対して0° ～6°
程度に変えられる構造になっている．また，網膜等に影響がないよ
うに少ない照明光量でも観察できる明るい光学系が必要である．光
源の配置には顕微鏡にランプを組込んだものと，光源部を顕微鏡か
ら離れた位置に置き，光ファイバで光を導く方式とがある．前者の
ものは比較的発熱も少なく電球自体も小型のハロゲン光源用で，後
者は高輝度，高発熱の光源装置に採用されることが多い．高い輝度
で自然な色温度をもつキセノンランプを使用した機種も増えてき
た．照明光路には，患部への熱影響を防ぐため高性能な熱線カット
フィルタが組込まれている．また術中の球切れに備えて予備ランプ
に即座に切り換えられる機構を備えている．
.懸架装置 顕微鏡を支持する懸架装置の操作性も手術顕
微鏡システムの使いやすさに大きく関わる．顕微鏡の視野をスムー
ズかつ自在に動かせることと，振動を極力抑えることが重要であ
る．懸架装置の大まかな構造として，アーム部をスプリングやガス
圧シリンダによって支える方式と，カウンタウエイトによりバラン
スを平衡状態に保つ方式（図5･15）等があり，手元のスイッチで
関節軸をロックまたはフリーにできる．簡易型は手動クランプ軸と
手動微動装置からなっている．眼科では観察範囲が狭いことや顕微
鏡を大きく傾ける必要がないので，電動で視野を水平移動できる
微動装置が重要視される．脳神経外科では手術の時間が長い
うえに術中頻繁に顕微鏡を動かすことが多いので，広い可動範囲と
動きのスムーズさが重要となる．動きのよさではカウンタウエイト
式が有利であり，脳神経外科用にはこの方式が多い．
