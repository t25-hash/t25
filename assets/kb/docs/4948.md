# 4・2・4 制御および計装

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・2 沸騰水型軽水炉

原子力プラントでは,原子炉・タービン発電機・工学的安全施設・プラントの各種補助設備に対する制御系を有している.図4・7にBWR の制御系の例を示す.これらの制御および,プラントの運転監視のために,原子炉核計装(neutron monitoring system),放射線計装(radiation monitoring system),プロセス計装などの各種計装が設けられている.a. 原子炉の制御. 出力制御原子炉の出力制御は,炉心の反応度を制御することにより行う.軽水炉の場合には,制御棒と炉心冷却材の流量あるいは,一次冷却材中のホウ素濃度の調整を組合わせて炉心反応度を制御している.BWR では,一般に原子炉の起動,停止などの大幅な出力変更,出力分布の調整および,燃料の燃焼に伴う長期の炉心反応度変化の補償は制御棒により行い,負荷変動に対する急速な出力の調整は,炉心冷却材の流量を調整する再循環流量制御により行っている.また,いずれの場合も,原子炉を緊急に停止する必要のある場合は,制御棒を全数同時に急速挿入する.. 水位・圧力制御原子炉を安定に運転するため,原子炉出力の制御に加え,水位と圧力の制御を行っている.原子炉水位は,原子炉から取出される蒸気量と原子炉に注入する給水量のバランスにより決まるので,原子炉水位,主蒸気流量,給水流量の三つの要素を検出して,原子炉の出力に見合った適切な給水制御を行う3要素制御(three element control)により原子炉水位を制御している.また,原子炉圧力の制御は原子炉圧力を一定に保つように原子炉からタービンに供給する蒸気量を制御する.すなわち,原子炉出力に従って発生する蒸気量をタービン加減弁あるいはタービンバイパス弁により制御して,原子炉圧力を一定に保っている.b. 原子炉の計装原子炉の計装には核計装と呼ばれる中性子の計測装置と,放射線計装と呼ばれるプラントの各所の放射線を測定する計測装置と,プロセス計装と呼ばれる圧力,温度,流量などの計測装置とがある.このうちプロセス計装は,通常の工業プロセスの計装と同等の測定方式であるが,原子炉の制御保護などに用いられるので信頼性,耐環境性,耐震性,耐放射線性など,原子力特有の要求仕様を満足するよう設計されている.核計装は,計測範囲が非常に広いために1種類の計器で全領域をカバーすることは困難で,図4・8のように数種類の計測器をオーバラップさせて検出する.中性子レベルの差に応じ,中性子源領域モニタでは核分裂電離箱またはパルス計数管の出力パルスをカウント(パルス計数)する方法,中間領域モニタではγ線補償型電離箱または核分裂電離箱の出力電流変動と中性子揺らぎの相関性を基に中性子束を計測する方法(キャンベル法),出力領域モニタではγ線非補償型電離箱または核分裂電離箱の出力電流を測定する方法を採用することが多い.なお最近は,測定回路にてパルス計数とキャンベル法の自動切換えを行うことにより,中性子源領域と中間領域の二つの領域を計測する起動領域モニタが実用化されている.放射線計装には,プロセス放射線計装とエリア放射線計装があり,前者はプラント保護用,排水排気の放射能監視に使用され,後者はプラント建屋内の放射線レベル監視に使用される.c. 計算機応用原子力発電所における計算機の活用範囲は広がりつつあり,炉心の性能計算や,燃料の燃焼度計算およびプラントのデータロガーとして使われることはもちろんプラントの自動運転,カラーCRT(cathode ray tube)やFDP(flat displaypanel)の表示およびタッチ操作機能を活用した運転監視,プラント診断,プラントの管理システムなどに使用されている.. プラント管理システム原子力発電所では膨大な量の各種データが使用され,管理されているが,計算機を使用して正確に,効率的に管理する各種計算機システムが設置されている.核計装からの炉心データなどを基に炉心性能計算や燃料燃焼度計算などを行う炉心管理システム,放射線関係のデータの管理を行う放射線管理システム,被ばく管理システムなどがその例である.. プラントの自動運転計算機技術の進歩に伴いプラントの自動運転範囲は拡大しており,最新のプラントでは,制御棒の自5―113動操作を含めたプラントの起動停止操作,スクラム後の定形操作等が自動化されている.また,供用期間中検査装置,燃料取替装置,制御棒駆動装置交換機などのプラント補助設備の制御も計算機により自動化されている.. プラントの運転監視計算機とCRT やFD を活用した運転状態の表示・監視操作技術の進歩および,光伝送ネットワークとディジタル制御装置の広範な導入を背景に,運転性および信頼性の向上が図られている.最新の中央制御盤では,タッチ操作機能を有するデバイス(CRT 等)の採用に伴う盤のコンパクト化,大形表示盤の採用による情報共有の容易化等を実現しており,プラントパラメータの表示記録と,それに基づいた運転員の操作にとどまらず,運転員の支援を広く行える設備としている.. 将来の動向原子力発電所における計算機応用はいっそう拡大発展する方向にある.運転制御の自動化範囲の拡大,プラント異常時の運転支援システムやプラント全体の各種データの管理などへの活用が今後も進められる.
