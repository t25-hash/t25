# 5・3・2 訓練用シミュレータ（training simulator ）

> **章**: 第5章 手術・治療機器
> **節**: 5・3 シミュレーション・治療計画・ナビゲーション

a.治療シミュレーション概説 治療支援を目的としたシミュデータに基づくものに大別される．
一般的モデルに基づくものは，患者ごとのデータに基づくものでは，一般的な訓練のみならず，実に関連してくるので，5･3･3項で述べる．
鏡，関節鏡，気管支鏡など内視鏡手術による低侵襲治療が普及して一方で，術者にとっては限られた視野，限られた操作空間における練した医師にとっても再訓練が必要であり，訓練用シミュレータののではなく，テレビモニタを通して見る．
仮想現実感技術の観点かべ，テレビモニタを通しての知覚効果をシミュレートすることは比は，臨床的必要性と技術的実現性の両面の要因から実用化が進んでいる．
以下では，内視鏡手術シミュレータを中心に説明を行う．
のおもな機能は，内視鏡・術具の模擬操作機構と操作情報のコンピ（haptic sensation）提示，操作に伴って時々刻々変化する内視鏡血，痛みなど）のシミュレーションとその提示である．
内視鏡・術体知覚を得るための視覚提示装置が組合わされる．
レーションは，一般的（典型的）モデルに基づくものと患者ごとの仮想的に治療行為を体験する訓練用のシミュレータに代表される．
際に特定患者の手術前のリハーサルが可能になり，手術計画と密接b.訓練用内視鏡手術シミュレータ概説 近年，腹腔鏡，大腸きている．
内視鏡手術は，患者の身体的ダメージを大幅に軽減する新しい術具の操作法の習得が必要となる．
よって，通常の手術に熟必要性は高い．
一方，内視鏡手術では，手術部位を肉眼で直接見るら，術野を直接見るという知覚効果をシミュレートすることに比較的容易である．
以上のように，内視鏡手術訓練用シミュレータまれる．
