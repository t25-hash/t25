# 4・3・1 基本的特徴

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・3 加圧水型軽水炉

加圧水型軽水炉(PWR : pressurized water reactor)は、原子炉設備(一次系)とタービン設備(二次系)が蒸気発生器で分離された構成となっている。減速材兼冷却材の軽水は炉心内で沸騰を起こさないよう一次系で加圧(通常15.5MPa)されており、炉心内の燃料により加熱された冷却材は、一次系と二次系の接点である蒸気発生器で熱交換を行い、二次系で蒸気を発生させた後、一次冷却材ポンプを経て炉心へ戻る。特徴としては、冷却材に中性子吸収材のホウ酸を入れて炉心反応度を制御(ケミカルシム制御)でき、中性子分布を平坦化できること、一・二次系はおのおの独立した水質管理ができること、タービン建屋は非管理区域に設置でき保守管理が容易であること、負荷追従性が良好なこと、などが挙げられる。図4・8 中性子測定器の使用範囲図4・9原子炉圧力容器および炉内構造物5―114エネルギー供給システム炉心は図4・9に示すように、格子状に配列した燃料集合体と軽水から構成され、原子炉容器に収容されている。炉心反応度を制御する制御棒は燃料集合体内に挿入または引抜きされ、原子炉容器上ふたには制御棒駆動装置が取付けられている。一次冷却材ポンプ、蒸気発生器、一次冷却材配管で1ループを構成しており、出力によってループ数を選んでいる。日本における発電用軽水炉の開発は、1970年に商業用原子力発電を開始して以来、順次大容量化を図るとともに信頼性、運転・保守性、経済性の向上が図られてきた。この間、国を中心とした改良標準化計画が1980年からスタートし、信頼性向上、定期検査の効率化をおもなねらいとした第一次改良標準化計画、信頼性向上に加えて、保守性・運転性向上をおもなねらいとした第二次改良標準化計画、さらに、改良標準型炉であるAPWR(advanced PWR)の開発を行った第三次改良標準化計画が遂行された。現在は、APWR に続く、2010年以降の導入が考えられるPWR 型将来炉(次世代PWR)について開発が進められている。
