では、あらかじめ与えられた原子炉出力に対して、燃料濃縮度、燃料寸法・配列、冷却材流量・温度・圧力、バーナブルポイズン、制御棒配置、燃料装荷パターンなどに関してサーベイを行い、安全上の制約や基準を満たす範囲で経済的な炉心を設計する。. 核設計原子炉に所要の機能と性能をもたせるための核反応に関する設計を核設計(nuclear design)と総称している。核設計は静特性、動特性および燃焼の三つの面に分かれる。まず、静特性として、炉心は臨界に達することはもとより、出力運転、停止、再起動などが行えるものでなければならず、これらの機能と性能の達成が設計目標となる。設計内容としては、炉心の燃料・材料の組成や配置、炉心寸法・形状などを適切に選ぶこと、温度上昇や燃焼などによる反応度低下を補償するために適切な反応度余裕をもたせること、十分な中性子吸収能力のある制御系を設けること、ゼノンオーバライドのための反応度余裕をもたせることなどが含まれる。動特性の面では負の出力係数、出力振動の減衰など炉心が安全かつ安定に作動するための条件が要求される。燃焼(burning)に関しては、計画された燃焼期間を通じて、経済的な燃焼ないしは転換が行えることが目標とされる。核設計に対して課せられるおもな安全上の制約としては、(1)最大反応度価値を有する制御棒クラスタ1体がまったく挿入できなくても、他の制御棒クラスタにより、高温停止状態で少なくとも所定の反応度停止余裕を有し、さらにケミカルシムにより、低温状態でも所定の反応度停止余裕を維持できること、(2)制御棒クラスタが手順に従い最大速度で引抜かれても、炉心や原子炉冷却材圧力バウンダリーの健全性が損なわれないこと、(3)ドップラー係数は負であり、かつ高温出力運転状態で減速材温度係数は負であること、である。経済性を向上させるためには、燃料の到達燃焼度を高めることや中性子経済を改善することが有効であり、核設計においてこれらの因子が考慮される。PWR の核設計においては、炉心パラメータを入力条件とし、核設計コードを用いて炉心特性などの核設計データを求め、さらに運転条件を考慮したうえで原子炉性能の評価を行っている。また、群定数計算コード、拡散燃焼コード等の核設計用コードでは、制御棒効果や出力分布等を合理的な三次元手法にて計算している。. 熱設計U 原子核1個の核分裂で約200MeV のエネルギーが放出される。U やPuプルトニウム239などの場合でもほぼ同様である。燃料棒内での核分裂による発熱は中性子束に比例し、濃縮度や燃料物質密度、核分裂断面積に依存する。棒状燃料の場合は、燃料棒単位長さ当たりの発熱量を線出力密度(linearheat generation rate)という。熱はおもに燃料ペレットおよび被覆管中の熱伝導と、冷却材への熱伝達とを通じて外に取出される。燃料ペレットと被覆管の間にギャップまたは不完全接触部があると、そこでの伝導、熱放射などに基づく熱抵抗も考慮する必要がある。燃料棒内の垂直な断面内の定常温度分布は、燃料棒の長さが十分に長く軸方向の熱移動が無視できるとすれば定常熱伝導方程式の解として求まる。燃料棒の軸方向の温度分布は中性子束分布や冷却材温度分布等から計算することができる。熱的に最も厳しい冷却流路での発熱量やエンタルピー上昇などの値と、平均的流路でのそれらとの比をホットチャネル係数(hotchannel factor)という。要因別には、設計に基づく炉心発熱の不均一性を表す核的ホットチャネル係数と、燃料製造に関する不確かさを考慮した工学的ホットチャネル係数があり、例えば発熱のホットチャネル係数(F음)は、F음N(核的)とF음E(工学的)の積である。熱設計に対する安全上の要件として、通常運転および運転中の異常な過渡変化に対して、炉心の冷却が適切に行えることが必要である。PWR のおもな熱的制限値としては、(1)最小DNBR(departure from nucleate boiling ratio)が限界値を下回らないこと、(2)燃料中心最高温度は燃料の融点未満であること、である。PWR 燃料ではその健全性確保のため被覆管の平均塑性ひずみなどが制限され、また安全上の観点から冷却材喪失事故時の被覆管最高温度なども制限される。これらを考慮して最大線出力密度は40kW/m前後に制限されている。
