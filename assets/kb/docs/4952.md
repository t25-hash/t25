a. 制御および計装。 原子炉の制御原子炉の制御は、原子炉の出力制御、および水位・圧力制御が中心となっている。(1) 出力制御原子炉の出力制御は、制御棒挿入位置と一次冷却材中のホウ素濃度の調整を組合わせて炉心反応度を制御することにより行う。キセノンの蓄積、燃料の燃焼度等による緩やかな長期的反応変化は一次冷却材中のホウ素濃度の調整により行い、原子炉の起動、停止時および負荷変動時の出力および出力分布の調整は、制御棒とホウ素濃度の組合せによって行っている。(2) 水位・圧力制御PWR では一次系保有水量を加圧器水位により制御する。加圧器の水位制御は一次冷却材平均温度から決まる基準水位に一致するように、一次冷却材の充てん・抽出量を調整して行う。蒸気発生器の水位制御は、蒸気発生器の水位、蒸気流量、給水流量の3要素制御(three element control)により行う。原子炉の圧力は、加圧器の加熱器、スプレーなどにより一定に制御する。蒸気発生器で発生する蒸気は飽和蒸気であり、その蒸気圧はタービン出力により変化するので一次冷却材平均温度の制御により調整する。この一次冷却材平均温度はタービン出力で決まる基準温度になるように、制御棒を用いて原子炉出力を制御している。図4・12にPWR プラントの制御系概要を示す。. 原子炉の計装原子炉の計装にはプロセス計装と呼ばれる圧力、温度、流量等の計測装置と、核計装と呼ばれる中性子の計測装置、放射線計装と呼ばれるプラントの各所の放射線を測定する計測装置がある。このうち、プロセス計装は、通常の工業プロセスの計装と同等の測定方式であるが、原子炉の保護・制御等に用いられるため信頼性、耐環境性、耐震性など、原子力特有の要求仕様を満足するよう設計されている。核計装は、炉の中性子束密度を測定するが、計測範囲が広範であり、1種類の検出器で全領域をカバーすることは困難であるため、図4・13のように中性子源領域、中間領域、出力領域と3種類の計測系をオーバラップさせて検出する。中性子源領域系ではBF比例計数管の出力パルスを、また、中間領域系ではγ線補償形電離箱により電離電流を計測する。出力領域系ではγ線非補償形電離箱により原子炉出力に比例した相対出力を計測し、中性子束密度の急変、あるいは異常上昇時には原子炉を緊急自動停止させるための安全保護機能を担う。放射線計装には、プロセス放射線計装とエリア放射線計装があり、前者はプラント保護、排水・排気の放射線監視に使用され、後図4・13 中性子検出器の測定範囲者はプラント建屋内の放射線レベル監視に用いられる。プロセス放射線計装にはシンチレーションカウンタ、エリア放射線計装にはGM 計数管等が用いられているが、近年は半導体式放射線検出器が採用されている。. 計算機応用(1) プラント管理システム原子力プラントでは膨大な量の各種データが使用され、管理されているが、これらのデータを計算機を使用して正確かつ効率的に管理する計算機システムが設置されている。(2) プラントの自動運転計算機によるプラントの自動運転は、タービンの起動停止から、原子炉の昇温、昇圧などに実用化されており、今後もその利用範囲が拡大されていく方向にある。(3) プラントの起動操作は、加熱開始から完了まで、臨界操作、炉出力上昇および並列に大きく分けられる。加熱開始から完了までの操作は、一次冷却材系を加圧器ヒータ等により昇温昇圧し、2.75MPaで冷却材ポンプを起動する。冷却材ポンプの入熱と加圧器ヒータの入熱により加熱時制限曲線を遵守しながら、所定の加熱率にて昇温昇圧を行う。加圧器圧力が15.4MPaになれば、加圧器ヒータおよびスプレー制御を自動にする。臨界操作は、停止グループ制御棒を臨界前に完全に引き抜き、臨界になると推定される所定のホウ素濃度になれば、制御グループ制御棒を定められた順序で、かつ中性子束の増倍率制限値以内になるよう引抜き、臨界を達成する。その後、原子炉出力を数%に上昇させ、約5%の負荷にて送電系統に並列し、発電機出力の所定の上昇率で上昇させ、それと並行して原子炉出力を増加させていく。発電機出力が15%以上になれば給水制御を自動制御に切換え、所内電力を外部電源から発電機負荷に切換える。以降プラントは自動制御状態となる。プラントの停止は、定格出力状態からタービン負荷減少に追従して原子炉出力を低下させ、発電機出力が約15%になれば制御棒制御を手動とし、給水制御を給水バイパス弁による制御に切換える。その後、発電機出力が約5%になれば発電機を解列し、引き続きタービンをトリップさせる。. 通常運転PWR プラントの通常運転は、負荷変化に際して軸方向出力分布を負荷変動前の平衡状態での分布にできるだけ近く保つようにする。軸方向オフセットの制御は、ホウ素濃度による調整を主とし、制御棒制御(手動)による調整を従とする。. 運転制御PWR プラントの運転制御は、火力発電所と同様にタービン優先方式であり、タービン発電機の出力状態が原子炉側に伝えられて原子炉出力をそれに追従させる運転方式である。また運転操作は、中央制御室から集中監視および遠隔操作が行われ、プラントの異常な過渡状態が発生した場合に原子炉を保護するための安全保護系が設けられている。PWR プラントにおける通常の制御は、おもに制御棒制御系、加圧器圧力・水位制御、蒸気発生器水位制御、蒸気ダンプ制御等により制御される。5―118エネルギー供給システム。 保守管理原子力発電所では設備の状態を把握し、トラブルの未然防止に努めてプラントをつねに良好な状態に維持するため、定期検査を実施している。機器・設備の定期検査については、電気事業法、原子炉等規制法などに基本的な事項が定められており、これに基づいて具体的な実施要領、実施時期が決められる。原子力発電所設備の保守は、定期検査、定期点検、運転中の保守・点検、不具合機器の補修作業、設備改善などからなる。通常運転中の保守点検は、大別して巡視点検と故障修理に分類される。運転中の機器・設備について定期的なパトロールなどの巡視点検、あるいは定期的な作動試験により機能が正常であることを確認し、異常があればただちに修理あるいは取替えなどの処置が行われる。定期検査では、発電所の主要な機器・設備を定期的に点検整備し、安全運転を確保するため原子炉の燃料取替えとともに行われる。検査の頻度は電気事業法の規定により原子炉およびその付属設備については毎定期検査、蒸気タービンおよびその付属設備については2回の定期検査で1回行うこととなっている。
