# 5・4・3 手術用レーザメス

> **章**: 第5章 手術・治療機器
> **節**: 5・4 手術用装置

レーザを組織などに当てたときの吸光・散乱特性が，波長と組織
や体液により異なることを利用したものである．適切に波長を選ぶ
ことで，意図する組織だけに作用を与えることができる．
レーザメスはレーザ光をファイバなどで導光し，ハンドピースか
ら照射するものである．散乱が少ない波長と組織の組合せでは，狭
い領域を深く焼灼することでシャープな切断面を得ることも可能で
ある．逆に散乱が大きい組合せでは，その散乱の様子に応じた範囲
を焼灼することができる．
手術用レーザメスは，一般外科では病変部の焼灼に用いる．眼科
ではシャープな断面を利用して切開に用いるほか，精密なアブレー
ションにより眼の光学特性を改善する利用法も行われている（メス
以外の用法については5･7･5項参照)．
a.安全性 レーザメスは，吸収あるいは散乱されないと減弱
しない．特に他に併用する器具に当たって意図しない部位にレーザ
が反射することがないように注意する必要がある．また，術者や周
囲のスタッフへの安全配慮も必要である ．
