# 4・5・1 開発の経緯

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・5 高温ガス炉

黒鉛(graphite)を減速材とし、炭酸ガスまたはヘリウムを冷却材として用いる原子炉をガス冷却型炉といい、炭酸ガスで冷却するマグノックス型炉と改良型ガス炉AGR およびヘリウムで冷却する高温ガス炉がある。これまでマグノックス型37基、AGR 15基、高温ガス炉7基を合わせ総計59基が建設された。高温ガス炉(HTGR またはHTR:high temperature gas-cooled reactor)は、冷却材温度を700°C以上にするため、炉心構成材料に金属材料をいっさい使用せず、出力密度を約6MW/m程度まで高めるとともに、燃料要素の基本単位として微小燃料核を熱分解炭素や炭化ケイ素で多重に被覆した被覆燃料粒子(coated fuel particle)で、核分裂生成物を直径1mm以下の粒子内に閉じ込めるのが特徴である。a. 海外における開発経緯と動向高温ガス炉の開発は、1956年イギリス、ハーウェル研究所で開発された高温に耐える独特な被覆燃料粒子に始まる。同年実験炉ドラゴン炉が建設着手され、研究開発を目的としたDRAGON 計画が英国を中心にOECD 11箇国の参加で開始された。1964年の初臨界後、燃料および黒鉛の開発や健全性実証のための広範囲な照射試験と高温ガス炉の運転の貴重な経験を積み、1976年運転を終了した。ドイツは、反射体を兼ねた下部を円すい状にした黒鉛製の円筒状炉心に、直径6cmの球状燃料を詰めて構成する独特のペブルベッド炉心(pebble bed core)の原子炉を開発した。冷却材のヘリウムガスは燃料の球と球のすきまを流れ、原子炉を運転しつつ燃料球を炉心上部から連続して装荷し、自然落下させつつ燃焼させ炉心下部から取出すので、燃料交換のため原子炉を停止する必要がない。電気出力15MW の発電用実験炉AVR がユーリッヒ研究所に建設され、1967年炉心出口温度850°Cの全出力運転に入り、1967~1978年の間約77%という極めて高い稼動率で順調に運転され、1974年炉心出口温度を950°Cまで上げることに成功した。1988年に20年余にわたる運転を終了した。ペブルベッド炉の概念を発展させた発電用原型プラントとして建設された電気出力約300MW のTHTR-300は、一次冷却系全体を単一のプレストレスコンクリート製原子炉容器PCRV に収容し、大口径配管がない設計である。THTR-300は、営業運転に入ってからの稼動率は50~60%で高温ガス炉の原型炉としての目的を果たしたが、1988年高温ダクト内の断熱板のボルトの頭が破損したトラブルをきっかけに財政的理由で運転終了となった。AVR およびTHTR-300の経験を基に、発電および熱電併給用高温ガス炉HTR-モジュール等の開発が行われた。HTR-モジュールは炉心の寸法、出力密度等に制限を加えることによって、固有の安全性を最大限高め、炉心溶融事故が起こらない原子炉としたものである。このほか、豊富な石炭、褐炭を原子炉の熱でガスや液体にして使いやすく環境汚染の少ない燃料に変える計画、また原子炉の熱を遠方の利用者まで供給し回収する計画を進めたが、近い将来経済的に成立する見通しがなく1980年代半ばで中断した。2005年現在、安全性研究以外の開発は中断されており、わずかに南アフリカ共和国、オランダ、インドネシア等の計画に協力が行われている。アメリカでは、オークリッジ国立研究所やGA 社が主体となって開発が進められ、GA 社が設計した実験炉ピーチボトム炉は棒状燃料を用い、直径89mmの804本の黒鉛燃料要素からなるブロック型炉心が鋼製圧力容器に収められた。1967年全出力運転に入ったが、初期に燃料コンパクトの照射膨張により黒鉛スリーブにクラ図4・26 燃料集合体5―124エネルギー供給システムックが発生したため、一層の熱分解黒鉛層で被覆された燃料粒子から二重に被覆した燃料粒子に交換された。その後は、高い稼働率で順調な運転実績を残して、1974年運転を終了した。次いで、世界初の原型炉としてフォートセントブレン炉が1974年に初臨界となった。マルチホール型のブロック型燃料(block type fuel)とPCRV を使用しているのが特徴である。出力上昇試験の過程で、冷却ポンプから一次系への水の漏込み、炉心出口温度の変動等の初期故障などを経験した後1979年に営業運転に入り、1981年に100%出力運転を達成したが、稼働率が悪く1989年運転を終了した。1978年から電力会社およびユーザ会社で構成されるガス冷却炉協会GCRA が政府とともに資金援助を行い開発計画が進められた。中小電力会社が数多くあるアメリカでは、電力需要の伸び悩み等に加え、TMI事故、チェルノブイリ事故等の経験を踏まえ、事故時に周辺住民の緊急避難を必要としない小型モジュール型炉の開発へとつながった。現在、プルトニウムを燃焼しガスタービンで発電するGT-MHR の開発がロシアおよびフランス、日本の企業と共同で進められている。旧ソ連では、蒸気供給やアンモニア製造を目的に、球状燃料を用いたペブルベッド型炉の研究開発を行ってきた。特にチェルノブイリ事故以後、ドイツのHTR-モジュールをベースとして熱出力200MW のモジュラ炉の開発を進めてきたが、現在、ロシアはアメリカとGT-MHR の開発を進めている。中国では、重油回収や石油化学工業用の熱源として高温ガス炉利用の検討がドイツの協力のもとに進められてきたが、清華大学が熱出力1万kW のペブルベッド炉心をもつ試験炉HTR-10を、2001年臨界とした。南アフリカ共和国には現在PWR 2基が運転中であるが、次期原子力発電プラントとして、小型の高温ガス炉ガスタービン発電プラントPBMRの開発計画を進めている。国営電力ESKOM 社が中心となり、設計にはドイツが協力してイギリス、アメリカ、日本の企業が参加している。また、オランダが小型モジュラ炉の検討を、インドネシアは自国の石炭資源、天然ガス資源等の有効活用のため熱源や分散形電源として高温ガス炉導入の検討を行っている。b. わが国における開発経緯と動向1969年わが国の新たなプロジェクトとして、発電用のみならず製鉄、化学工業分野等の熱源として核熱のプロセス利用を目指した高温ガス炉開発が日本原子力研究所で開始された。この多目的高温ガス炉(multi-purposeHTGR)プロジェクトでは、実験炉の設計および関連要素技術の開発が行われた。関連要素技術の開発として、大型構造機器実証試験ループ、高温ガス炉臨界実験装置、燃材料照射ガスループ等を用いた大型構造機器の健全性、炉物理、被覆粒子燃料、耐熱材料、高温機器等の研究開発や原子炉設計、建設に必要なすべての分野での研究開発が進められた。他方、通産省工業技術院では実験炉に接続する直接製鉄パイロットプラントの完成を目指した大型プロジェクト「高温還元ガス利用による直接製鉄技術開発」が1973年から6年計画で開始され、1980年に所期の目的を達成して終了した。この時点までの成果は日本原子力研究所に移転された。1986年核熱プロセス利用(nuclear process heat application)の需要の動向などに鑑み、高温ガス炉研究開発の意義、現状ならびに今後の研究開発の検討が原子力委員会で行われた結果、高温ガス炉技術の基盤の確立と高度化を図るとともに、高温工学に関する先端的基礎研究を行う試験研究施設として、熱出力30MW、最高出口冷却材温度950°Cの高温工学試験研究炉(HTTR : high temperature engi-neering test reactor)を建設することとなり、1991年建設に着手した。1998年に初臨界を達成し、その後段階的に出力上昇試験が行われ、2001年末に熱出力30MW・炉心出口温度850°Cの定格運転条件が達成された。その後高温ガス炉の特性試験が行われ、2004年4月、世界で最初の950°Cの熱を原子炉容器の外に取出すことに成功した。表4・4および4・5に、それぞれこれまで運転された高温ガス炉および現在運転中と計画中の高温ガス炉の主要諸元を示す。表4・4
