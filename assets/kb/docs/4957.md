# 5・5・10 コンピュータによる手術支援

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

コンピュータは医療の質の向上にも大きく貢献してきた．手術支
援においても，すでに術者を補助する第二の脳として，治療計画か
ら治療（手術）の実施，術後管理，アウトカム評価まで，陰に陽に
活躍している．
以下に述べる手術ナビゲーションシステム，手術マニピュレータ
システムがその代表である．
コンピュータを上手に利用することでケアレスミスなどヒューマ
ンエラーを未然に防止し，より精密，より確実な医療を実現するこ
とが期待される．一方，治療のポイントが医師の手を離れ機械によ
って左右されることに対する危うさも指摘されている．新しい技術
は，その効果と安全性，そして限界が広く医師に認識されるまで
は，たとえそれが安全のためであっても未知のリスクをもつことを
忘れてはならない．
a.手術ナビゲーションシステム（surgical navigation system)
低侵襲化の進行に伴い，いま操作している部位はどこかの判断が容
易でなくなってきた．そこで，位置センサにより座標計測して解剖
的な知識との対応を得る手術ナビゲーションシステムが開発され
た．発想および原理的な構成は自動車のナビゲーションシステムと
同じである．GPS（global positioning system）に相当する位置セ
ンサ，地図データに相当する断層画像からなる．ナビゲーションで
は，地図としての画像データと位置センサによる実測値との間のレ
ジストレーションを行う（レジストレーションに関しては5･3節参
照）．レジストレーションで生じる誤差は，ナビゲーションの精度
を大きく左右する．
すでに関節置換術や脊椎修復などの整形外科手術，腫瘍切除術な
どの脳神経外科手術では，高精度な手術を行うことで治療成績が向
上することが示唆されており，普及しつつある．
.CTベース（CT-base ） ，CTフリー（CT-free) 近年，
断層画像データとのレジストレーションを必須としないナビゲーシ
ョン法が考案されている．ナビゲーションシステムに含まれる位置
センサだけを用いて，対象とする手技で求められる幾何学的な拘束
条件が満たされるように術具操作を補助するものである．断層画像
を必須としないことからCTフリーナビゲーション（CT- free
navigation) などと呼ばれ，これに対し画像とのレジストレーシ
ョンを行うものをCTベースナビゲーション（CT-base naviga-
tion）と呼ぶこともある．
一例として，膝の人工関節置換術で大腿骨側の関節を取付ける際
の位置決めが挙げられる．正立時の荷重方向に従い，股関節側の大
腿骨頭から膝関節の中心を結ぶ線分に垂直になるように人工関節を
取付ける．
CTベースナビゲーションでは術前のCT画像で大腿骨頭中心，
膝関節中心を計測して荷重線分を同定し，術中に画像と大腿骨をレ
ジストレーションして荷重線分に垂直になるように，損傷した膝関
節を除去する．レジストレーションのため，大腿骨に位置計測マー
カを固定する．
CTフリーナビゲーションでは，膝関節の中心は切開後に直接ポ
インティングして取得する．大腿骨頭の中心は，これが中心となる
ように大腿部を動かして，大腿骨に固定した位置計測マーカの位置
を計測して円の方程式に当てはめて決定する．これによって荷重線
分が得られ，あとはCTベースナビゲーションと同じである．レジ
ストレーションの誤差を含まないため，CTベースナビゲーション
よりも精度的に有利と考えられている．
b.手術ロボット（surgical robot ) いわゆるロボット機構を
治療に用いるシステムはいくつか発表されているが，2007年現在
の段階では多くが基礎研究の段階である．ロボットの役割は人間の
素の能力を超えることと総括できるが，以下がそのおもなものであ
る．
(１)低侵襲手術支援
内視鏡手術の不便を補う機構．狭所での結紮（けっさつ）を容易
にしたりする．マスタスレーブ式マニピュレータによる製品が海外
で市販されている．5･3･11項にて詳説する．
(２)高精度手術
1990年代初めにアメリカで開発が行われた股関節置換術におけ
るステム部挿入のための開削を行うNC（numerical control ）機
構が ，欧米で市販され，多数の臨床実績を上げている．ステム
の形状に合わせた穴を所定の位置に正確にあけることができる．こ
のほか，定位脳手術を行うシステム などが開発されている．高
精度化は医療の安全などにも寄与する．
(３)医療の合理化
低侵襲手術では術者とは別の1人が内視鏡の視野を操作する役割
を担うが，術者と息が合わないと不自由であるうえ，それだけにス
タッフを充てるのは不経済でもある．そこで，術者が内視鏡の視野
を操作する機構が開発され，国内でも販売されていた実績があ
る ．
そのほか，放射線治療装置と組合わせて線源位置を精密に操作す
るものなどが実用化されている．
