# 4・5・3 高温ガス炉熱利用技術開発

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・5 高温ガス炉

高温ガス炉は,最高約1000°Cの高温の熱が取出せるので,高効率発電が可能となるだけでなく核熱エネルギーを多くの工業プロセスにも広く利用することができる.a. 高効率発電(high performance power generation)発電への利用では,在来火力発電並みの蒸気条件の蒸気タービン発電で約40%の熱効率が得られるが,原子炉出口冷却材温度が850°C以上ではガスタービン発電により45~50%の熱効率が得られる.閉サイクルのヘリウムガスタービンを高温ガス炉と結合させて,大規模な発電を行うための研究開発として,ドイツで閉サイクルガスタービンの開発が行われてきた.また,ガスタービンと蒸気タービンとの複合システムとして発電を行うことによっても約50%の熱効率が得られる.モジュール型高温ガス炉と閉サイクルガスタービンを組合わせた発電プラントの概念設計が,アメリカ,ロシアを中心にフランス,日本の企業の協力を得て実施されてきた.また,南アフリカ共和国の国営電力ESKOM 社は,モジュール型高温ガス炉ガスタービン発電プラントの概念設計を終了し,2005年初臨界を目指した計画を進めている.b. 水素製造(hydrogen production)水素は現在でも化学工業の重要な原料であるが,本来燃料としても極めて優れた性質をもっている.また,原料が水で無限にあり,燃料として使用すると再び水に戻り有害物質を発生しないのでクリーンであり,貯蔵が容易であり,家庭用から工業用,宇宙用,さらには自動車や飛行機などの燃料にも使用できる,といった特徴をもっている.高温ガス炉の水素製造への有望な適用法としては,天然ガスの水蒸気改質法,熱化学法による水からの水素製造,高温水蒸気の電気分解などがある.天然ガスを用いる方法は,高温ガス炉の熱を利用してメタンと水蒸気を反応させることによって水素と一酸化炭素を生成するものであり,日本原子力研究所でHTTR に実規模装置を接続して実証する計画が検討されている.熱化学法は,水以外の物質を関与させて数段階の化学反応を利用して水を分解するものであるが,必要な反応熱を高温ガス炉から供給するもので,化学反応に用いる反応物質により,硫黄系サイクル,鉄-ハロゲン系サイクル等に分類できる.硫黄系サイクルの代表的なものとしてGA 社が提案したサイクルおよび日本原子力研究所のIS プロセスがあり,鉄-ハロゲン系サイクルとして,東京大学でUT-3サイクルの研究が行われてきた.c. 熱電併給システム(cogeneration supply system)高温ガス炉は発電以外にも高温の熱エネルギーを産業用熱源として広く利用できるので,この特徴を最大限活用すれば,原子炉から得られた高温の熱を必要温度の高いプロセスから段階的に利用することにより,極めて効率のよい熱の使い方が可能となる.例えば水素製造,石炭ガス化等の高温プロセスに利用した後,一般工業,海水淡水化,地域暖房などに利用し,さらに下がったところで農業,漁業などに利用するなど,段階的に利用すると,熱の利用効率が飛躍的に高められ,省資源,省エネルギーに役立つばかりでなく,エネルギー多消費産業からの環境汚染物質の放出量を低減でき,かつ,総合熱利用率が70%を超えるシステムとすることができる.わが国では,日本原子力研究所が中心となってこのような効率的なシステムの設計検討が行われている.軽水炉は,これまでの多面的な研究開発と運転実績を設計・運転に反映することによって改善を重ね,すでに現在その技術は十分に熟成したといえる.しかし,軽水炉は,経済性を高めるために大型化が進められ,また安全性をより確実なものとするため,多様化・多重化が要求され,システムが複雑化していった.原子力の社会的受容性を向上するために安全システムが確実に機能することを簡単な論理で説明することが将来の原子力に求められており,わかりやすいシンプルな設計,単純な運転を念頭においた軽水炉が開発されつつある.安全の基本的考え方を原子炉の設計にどのように組込んでいくかについては二つの考え方がある.その一つは自然力を利用した受動的安全性を生かして,原子炉を設計し運転しようとするもので,受動的安全炉という名前で呼ばれている.この安全の考え方を取入れた最初の原子炉がスウェーデンから提案された軽水炉PIUS で,その後いくつかの原子炉について受動的安全のアイディアが提案されており,出力および出力密度の大きくない小型炉でこのような考え方が実施されやすいことがわかっている.この考え方に沿って現在の軽水炉のもつ複雑さを改善し,設備,運転の両面から単純さを積極的に導入しようとするのが単純化軽水炉である.また,安全系は受動的安全系または静的安全系と呼ばれ,アメリカをはじめ,次世代の軽水炉開発の一環として研究開発が進められている.受動的安全炉の安全上の特徴を端的に表現すれば核分裂の連鎖反応に直接関連した事象が急峻なものでなく,かつ,原子炉自体のもつ固有の特性で異常が拡大することなく抑え込まれることと,原子炉の異常な状態に対しても崩壊熱の除去が自然力によって行われることである.このような安全上の特徴および運転管理上の特徴を有する原子炉は,開発途上でいまだ原子力技術を十分もっていない国々のエネルギー供給に好適と考えられる.また,局地やへき地で人口も少なく自然条件にも恵まれていない地域のエネルギー源としても好適と考えられる.受動的安全炉で利用される自然力として,次のものが考えられる.・流体の温度差に基づく密度差による水圧の差・重力ポテンシャルによる水圧の発生・温度上昇による磁力の喪失・温度上昇による燃料体および制御棒の伸び・温度上昇による金属の溶融・自然対流および伝導・放射による熱伝達
