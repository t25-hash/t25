# 5・5・11 マニピュレータシステム

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

内視鏡手術を補助するマスタスレーブ型マニピュレータは最も普
及に近いと考えられている（図5･48)．それは，内視鏡手術の進展
とともに顕在化した問題点をうまく解決しているからである．医師
は，開腹下での従来の手術では，術部に手首を近づけて手首から先
図5･50 種々のカテーテルの先端形態
(ａ)スレーブ部
う．これからの手術ロボット研究は医療上の研究とノウハウの集積
も併せて進めることが不可避で，そのような医工連携が不可欠とな
る．
(ｂ)マスタ部
図5･ 49 内視鏡手術用マニピュレータ，ダ・ヴィンチのマスタ部とスレ
ーブ部（提供：インチュイティブ・サージカル社）
を効果的に使って結紮など微細な作業をこなしてきた．内視鏡手術
では細長い鉗子を用いて手術を行うが，腹腔などに挿入したトロッ
カが支点として作用する．鉗子の先端を1cm動かすには，てこ比
に応じて術者の腕全体を動かす動作が必要となる．この状態で微細
な作業を行うことは容易でなく，また従来の術式と比較して疲労し
やすいことを示唆している．
手術マニピュレータシステムは，手首を使う動作を復活させた．
有名なダ・ヴィンチ（da Vinci ）システム（インチュイティブ・サ
ージカル社）は，ワイヤ駆動で動作する4自由度（鉗子動作を含
む）の手首機構をもつ（図5･49)．術者は操作卓に座って手首の操
作に集中することができる．さらに，術者動作の震えをフィルタし
たり，術者動作を縮小して鉗子を動作させることができ，微細作業
がより容易になっている ．
このほか，脳神経外科手術用マニピュレータも開発が進んでい
る ．
