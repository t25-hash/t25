# 5・5・12 能動鉗子（active forceps ）

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

低侵襲手術を支援するマニピュレーション技術としては，内視鏡
処置具の一種である小型の鉗子，血管内カテーテルなどがある．現
在用いられているものは2～3自由度程度で，手元で手動操作する
簡単な機構であるが，駆動源を有し多くの自由度をもち複雑な形状
をとる能力をもつもの，ある程度の保持力を要する能動鉗子が研究
されている．将来的には，現在臨床研究が進められているカプセル
内視鏡に小型の能動鉗子を取付けたものも構想されている．
今後の手術ロボット技術は，マイクロ化，ピンポイント化，ユビ
キタス化の方向をたどるものと予想される．その際に重要なのは人
間と機械がお互いの得意分野を分担することである．
すでにCTやMRIなどの断層画像システムと組合わせて画像誘
導手術を行うシステムが多数発表されている．画像上で計測された
座標値どおりに手術を行うことは人間の術者にとって必ずしも容易
なことでなく，理にかなった応用といえる．ユビキタス化は患者に
とってのユビキタスと医師にとってのユビキタスの両方が考えられ
る．マスタスレーブ型のシステムでは，両者間の距離を長距離にす
ることに関して遅延以外の技術的困難はない．大西洋を挟んでの遠
隔手術の実験などが行われている．遠隔手術は患者と医師の双方が
お互いの住居のそばで双方の満足度の高い医療を実現する可能性が
ある．
一方で，単純に人間の動作を機械で置換えるだけのシステムは，
それが医療の真のニーズに合うものであるか厳しく問われるであろ
