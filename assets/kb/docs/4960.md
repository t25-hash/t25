# 5・5・13 腹部領域におけるカテーテル

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

a.概略 本項においては腹部領域で使用されるカテーテル
（catheter ）とともに，消化器内視鏡において内視鏡の鉗子孔を介
して使用される処置具の一部について論じる．
b.消化器病領域での血管カテーテルとその他のカテーテル
血管内カテーテルは血管造影という診断目的で使用が開始された．
当初は皮膚を切開し直視下に露出した血管からカテーテルを挿入し
ていたが，1953年のSeldingerによる経皮的血管穿刺法の発明に
より，局所麻酔下に皮膚を直接穿刺することでカテーテルの挿入が
可能となった．また血管造影も大動脈内に挿入した直線カテーテル
から造影剤を注入する亜選択的大動脈造影から始まったが，目的血
管の分岐形態に合った先端形態のカテーテルを使用し，目的とする
血管分枝内までにカテーテルを挿入することで，選択的血管造影が
可能となった（図5･50）．
またカテーテルの先端からガイドワイヤを目的とする末梢側の血
管分枝内に進め，このガイドワイヤを支索としてカテーテルを押し
込むことにより，より末梢側の血管分枝に至るまで選択的にカテー
テルを進めることが可能となった．このような選択的カテーテル挿
入技術によって，選択的造影や抗がん剤・塞栓物質等を対象臓器の
みに選択的に血管内腔から投与することも可能となった．さらにバ
ルーン付きカテーテルやステントによる血管狭窄の拡張も可能とな
り，血管造影の手技は診断のみならず重要な治療手技ともなった．
これらの血管造影カテーテル等を駆使した治療をinterventional
radiologyと総称している．
担がん臓器を支配する動脈内に留置したカテーテルから抗がん剤
を直接注入する動脈内注入化学療法とは目的臓器のみに集中的に抗
がん剤が投与できるので，治療効果の増強と副作用の軽減の双方を
可能とする治療法である．従来は全身麻酔下の開腹手術によって目
的の血管を露出しカテーテルを留置していたが，interventional
radiologyの技術により開腹することなく局所麻酔下にカテーテル
を留置できるようになった．
これは開腹手術によらずにカテーテル留置が可能となったという
だけでなく開腹手術においては腫瘍の浸潤等の理由で血管の露出自
体が困難である場合でも，安全に血管内にカテーテルを留置できる
ことや，腫瘍に行く複数の血管を血管内塞栓物質などで閉そくさ
せ，1本のカテーテルからの選択的な薬剤投与が可能となるなど，
interventional radiologyならではの福音も大きい（図5･51) ．
図5･51の症例では上腸間膜動脈（SMA:superior mesenteric
artery）から肝動脈が分岐する症例に対し，肝動脈の分枝である胃
十二指腸動脈（GDA:gastroduodenal artery）を塞栓用コイルで
閉そくさせたのち，留置用カテーテルをここに挿入，留置カテーテ
ルの側面にあけた側孔から肝動脈内に薬剤投与を図っている．従来
の開腹手術では対応困難であるような症例である．

カテーテルは血管内だけでなく，胆管・尿管等にも使用される．
悪性腫瘍，結石，炎症等による胆管・尿管の狭窄，閉そくに対して
従来は開腹手術により狭窄の解除やバイパス手術を行うほかはなか
った．しかし超音波断層診断装置によるリアルタイムの画像誘導下
に狭窄部より上流の拡張した管腔を経皮的に穿刺し，血管造影と同
様にガイドワイヤを支索としてカテーテルを留置することで，胆汁
や尿を体外に誘導することも可能となった（PTCD:per-
cutaneous transhepatic cholangiodrainage，経皮的経肝胆道ドレ
ナージ）．
さらに血管と同様に，拡張用バルーン付きカテーテルや形状記憶
合金を用いたステントによる狭窄部位の拡張を局所麻酔下に経皮的
に施行することも可能となった．これらの技術もinterventional
radiologyの一分野として広く臨床に供されている（図5･52)．
このようなカテーテルを用いた治療の進歩と普及には，カテーテ
ルやガイドワイヤの先端形状，太さ，材質等の改良による操作性の
向上によるところも大きい．
しかし今日カテーテル操作における手技上の困難性はカテーテル
やガイドワイヤの先端形状を血管や胆管内で自由に可変できないこ
とと，カテーテルの先端に挿入のための駆動力がなく，数十cmか
ら1m前後離れた術者の手元での操作（出し入れとねじり）に頼
っていることに起因している．このためカテーテルのたわみからト
ルクの伝達が不十分となったり，挿入中のカテーテルが手元の操作
方向に反して目的の血管から逸脱することなどがしばしば見られ
る．これらの問題点は術者の技術により補われているのが現状であ
り，この面でのハードウェアの研究開発が強く望まれている．
c .消化器内視鏡処置具 消化器内視鏡には種々の処置具の通
路となる鉗子孔があり，常用される消化器内視鏡においては3mm
前後の直径を有するチャネルが一般的である．
この鉗子孔を介して使用される器具としては，組織採取用の生検
鉗子（5･5･6･b項参照）のほかに，内視鏡的治療のための種々の処
置具がある．例えば，ポリープ切除のための電気通電可能な切除用
スネア，結石破砕用スネア，止血用クリップとクリップアプライ
ヤ，薬剤注入用穿刺針，ヒートプローブとユニット，止血やがんの
レーザ治療用のレーザ用ファイバ等である（図5･53)．これらは早
期消化器がんの内視鏡による根治切除や，結石の低侵襲治療を可能
図5･54大腸内視鏡下のスネアによる大腸ポリープ切除
としている（図5･54）．
特に電子内視鏡の普及と進歩は従来の光学内視鏡に比べて大型デ
ィスプレイ上の高画質の映像を複数の人数で共有できることもあ
り，内視鏡による消化管病変の治療に大きく寄与している．
今日の消化器内視鏡における治療器具の操作上の問題点は，一部
の内視鏡を除き鉗子孔は1本であり，1本の処置具で病変を処置せ
ざるを得ないこと，そして処置具の先端形状のコントロールが困難
であるので，出し入れ以外の方向への処置具の操作には内視鏡の先
端自体の屈曲を利用せざるを得ないことにある．このため，通常の
外科手術のように複数の手を用いた繊細な操作が，内視鏡において
は困難となっている．
消化管内腔に挿入する消化器内視鏡では，その口径に必然的な限
界が生じるので，鉗子孔の数と口径にも限界が存在することも自明
であるが，血管カテーテルと同様に自律的な先端の制御と駆動が可
能な処置具の開発も待たれるところである．
