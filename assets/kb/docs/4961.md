# 5・5・14 心細動除去装置（defibrillator )

心室細動とは，周期の短い，ランダムな相互に同期していない心
筋の収縮である．正常な心臓では洞房結節に発生した興奮が心房を
経て房室結節，次いでヒス束，左右脚，プルキンエ線維網からなる
心室内刺激伝導系組織を介して左右心室のすみずみまで伝えられ
る．心筋組織はいったん興奮したのち200～500msの不応期に入
る．このため心臓の興奮伝ぱは威信周期ごとに完結し，そこで消失
する．しかしある特殊な条件下では興奮波が1心周期の間に消失せ
ずに，再び元の部分に戻ってきて心臓を再興奮させることがある．
この現象を興奮の再進入（リエントリー）と呼ぶ．心房，心室にお
ける期外収縮や頻脈性不整脈のある種のものはリエントリーが原因
となっていることが確認されている．このような旋回する興奮波が
複数に分裂し，多くの互いに同期していない興奮波に分裂すると，
心筋はもはや同期して収縮することはできない．このような現象が
心室で生じる心室細動では，心臓の血液を送り出すポンプ機能は失
われ，致命的な事態に至る．
ズムを停止するために，患者に強い電気ショックを加える装置であ
る．強い電気ショックを心室に加えることにより，心筋の大部分を
脱分極させ，心室全体の興奮の同期を正常なものに戻すことが，電
気的除細動の簡単な原理の説明になるが，現在でもその過程は完全
には理論的に説明されていない．しかしながら除細動を行うには，
心臓に加える電気ショックがある一定以上の強さと持続時間をもた
なければならない．電気ショックの持続時間（パルス幅）が短い場
合には一般に大きなエネルギーの電気ショックを加える必要があ
り，電気ショックの持続時間（パルス幅）が長くなるにつれ，必要
となるエネルギーは小さくなる．

a.自動体外式除細動器（AED:automatic external
defibrillator ) 頻脈性不整脈の停止では，患者に心電図計測用
の電極を装着し，心電図波形を計測して患者に電気ショックを印加
すべきかどうか，また印加すべき場合のタイミングを自動設定す
る．心電図のT波付近は受攻期と呼ばれ，この期間に電気刺激を
心臓に加えると容易に心室細動を誘発することが知られており，こ
のタイミングで除細動刺激を加えてはならない．わが国において
も，医師以外の人間による電気的除細動装置の使用が認められ，救
命率の向上が図られている．図5･58に半自動体外式除細動器の一
例を示す．
b.植込み型除細動器（ICD:implantable cardioverter
defibrillator ) 植込み型除細動器は心室不整脈を検出し，自動
的に高電圧（～750V）の電圧を心臓に加え，心拍を正常に戻す治
療デバイスであるが，心臓ペースメーカとしての機能をもつものも
ある．植込み型除細動器の使用により，心臓突然死の発生率が有意
に小さくなることがわかっている．
植込み型除細動器は，心電図信号を増幅・信号処理し頻拍・心室
細動を検出するユニット，心電図信号処理結果に基づき治療を制御
するユニット，治療用の電気エネルギーを印加するユニット，低電
圧の電池電圧を高圧（600～700V）に昇圧する電圧昇圧ユニット，
不整脈発生とそれに対する治療履歴を記憶するメモリユニット，外
部と電波によりこれらの情報を通信する，あるいは外部から除細動
パラメータの変更を可能とする無線通信ユニットなどから構成され
る．
頻拍の治療の場合，まず頻拍より少し速くペーシングをすること
で治療を行う．もしこの治療が不成功である場合には，より大きな
電気エネルギーを安全なタイミングで印加することで，脈を正常に
戻すことを行う．心室細動などの重篤かつ致命的な不整脈の治療に
は，さらに強いエネルギーの電気ショックを加える．
