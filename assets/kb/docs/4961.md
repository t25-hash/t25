# 4・7・1 高速炉とウランの有効利用

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・7 高速炉

おもに高速中性子を核分裂反応に利用する原子炉を高速炉(fastreactor)と定義している。速い中性子を利用する理由は、原子炉の中でU が核分裂を起こしやすい物質であるPuにより効率よく変換するためである。このことは、運転中に炉心に生成されるプルトニウムをそのまま同時に燃焼する燃料無交換炉心または燃料交換期間の大幅な延長の可能性をも示している。また、ある程度燃料が燃焼した後に、炉心およびブランケット燃料を再処理して核分裂性物質のみを取出し、新たな燃料に加工して利用することによるウラン資源利用率の大幅な向上を達成することができる。なお、生成された燃料物質と消費されたそれとの比を転換比(conversion ratio)と呼ぶが、この値が1より大きいときは、増殖比(breeding ratio)と呼ぶ。現在運転されている熱中性子炉では転換比は0.6程度であるが、わが国で開発された「もんじゅ」のような高速炉では1.2以上の増殖比が期待されている。このように消費した燃料物質より多く燃料物質ができることが、高速増殖炉(FBR : fast breeder reactor)と呼ばれるゆえんである。U は天然ウラン中に0.7%しか存在しない。そのU を3~5%に濃縮した燃料を熱中性子炉である軽水炉で燃焼し、その使用済燃料を長期保管するだけの方式をワンススルー方式と呼んでいる。現在のアメリカの原子力政策ではこの方式を採用している。この方式では、使用済燃料に含まれる未反応のU やU が転換して生成されたPuを将来にわたって再利用しない。この場合、天然ウランを資源として考えると、その資源利用効率は1%に満たず、極めて低いといわざるを得ない。99.3%のU を有効な資源とみなすか否かが高速増殖炉導入の是非につながり、原子力エネルギーの将来が大きく変わってくる。すなわち、U を積極的にPuに転換して資源利用効率を飛躍的に高めるのが高速炉技術であり、その実用化はわが国のエネルギー問題を一挙に解決するものとして期待されている。しかしながら世界的に見ると、最近の潮流として資源枯渇への脅威が遠のき、高速増殖炉によるプルトニウムを増殖させる意義が一時的に後退してきた。むしろ米ソ冷戦の雪解けによる核軍縮からの余剰プルトニウムの処理、核不拡散対策などから、高速炉をプルトニウムおよびアクチニドの専焼炉にすべきだという主張が強くなってきている。さらに高速炉は、原子炉で発生する放射性物質を核変換し安定な非放射性物質に変える能力が高く、放射性廃棄物の低減という観点からの注目も集めるようになった。いずれにしても、増殖炉はやがて来るエネルギー問題に対する最強の切り札である事実に変わりはなく、技術開発の継続と醸成が重要である。本節では以下、最も開発が進んでおり、将来ともに実用化に最も近いNa冷却MOX
