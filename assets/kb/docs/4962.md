# 4・7・2 開発の歴史

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・7 高速炉

a. 世界における開発経緯と動向表4・8に各国の高速炉開発を示す。世界初の高速炉は1946年にアメリカロスアラモスで臨界を達成した水銀冷却実験炉クレメンタイン(Clementine)である。次のステップはアルゴンヌ国立研究所の設計になるNaK 冷却高速増殖炉EBR-Iで、1951年12月世界初の原子力発電を行った。ガス炉や軽水炉に先駆け、高速炉が最初の発電を行ったことは特筆に価する。EBR-Iで高速増殖炉の概念が実証された後、長期にわたるエネルギー源を確保できる増殖炉の研究開発が基礎研究、国際共同も含め各国で活発に行われた。EBR-Iより30年後には、イギリスのPFR、フランスのフェニックス、ロシアのBN 350,BN 600などの原型炉が次々と電力網に組込まれている。アメリカでは、EBR-I以降、実験炉としてフェルミ炉、EBR-II,SEFOR,FFTFが建設され試験された。1972年から原型炉CRBRPの設計ならびに製作が進められていたが、1977年カーター大統領の核不拡散声明による政策変更に伴い、建設費高騰もあって1980年代初めにプロジェクト中止が決まった。当時は、天然ガス、石炭をはじめとする資源の枯渇に対する脅威が遠のいたことなども中止の誘因でもあった。他方、アメリカ原子力規制委員会およびアメリカ電力会社のニーズに適応したモジュラ型高速炉プラントPRISM の設計研究がGE社やRI社により進められたが、結局撤退のやむなきに至った。その間、長期間安定に運転を継続していたEBR-IIとFFTFも現在は閉鎖することになっている。1980年代半ばになると、1950年代から積極的に開発を進めてきたイギリス、原型炉SNR 燃料を消滅させることこそ緊急の課題であるとして、増殖炉よりもTRU 消滅炉として目的を変更して方向転換することが打ち出されてきた。そして小型固有安全炉の概念とともに、アクチニドリサイクル炉に重点がおかれるようになってきている。このこと5―136エネルギー供給システムは各国経済の好不況、財政状態、国民の意識という事情に依存して高速炉の果たす役割が変化しうるということであり、現状では大容量高速増殖炉の建設計画は見合わせているにすぎない。むしろ大事なことは、高速炉というシステムが時代の要求の変化に応じてその性能を発揮できるという、ポテンシャルの高さに注目すべきであろう。b. 日本における開発経緯と動向わが国では、1960年代前半から高速増殖炉の調査活動を開始し、精力的な調査研究の結果、1966年プルトニウムとウランの混合酸化物燃料(MOX)を用いるナトリウム冷却高速増殖炉の開発のため、国産技術によって高速実験炉および高速増殖炉原型炉(prototype reactor)の建設プロジェクトを発足させることになった。そしてその自主開発の一元的な責任を担う機関として、1967年動力炉・核燃料開発事業団(動燃)が設立されるとともに、大型の試験研究施設を集中的に設置した大洗工学センターが開設され、国内の研究者や技術者を集めて、FBR の炉物理、炉心設計、熱流動、高温構造材料、安全性分野などの基礎研究から工学的研究まで幅広く進められた。一方、MOX燃料の開発は、動燃東海事業所で基礎試験、燃料製造技術、燃料設計などが実施され、これらの経験を基に高速実験炉「常陽」、原型炉「もんじゅ」の炉心燃料の製造を行った。一方、燃料や材料の照射後試験施設が大洗工学センターに建設され、イギリスPFR、アメリカEBR-II、フランスのフェニックス炉、「常陽」などの高速炉で照射した燃料の照射後試験を行い、燃料の健全性や信頼性など燃料の性能向上のための研究や試験が進められた。高速実験炉(experimental Japan Nuclear Cycle Development Insti-tute)が新たに発足した。もんじゅ」に続く高速増殖炉実証炉(demonstration reactor)およびその延長線上にある実用炉の開発目標は、現在実用化されている軽水炉と同程度の安全性、信頼性、運転保守性を確保し、経済的に競合できるプラントを実用化することである。こうした目標に沿って、電気事業者が主体となった実証炉設計が進められ、新たな革新的技術を積極的に取入れた電気出力約660MW のトップエントリー方式ループ型炉の提案が行われた。その後1998年に国が定めた核燃料サイクル開発機構の業務に関する基本方針を踏まえ、サイクル機構の中長期事業の一環となる実用化戦略調査研究に日本の高速炉開発の関係者(電力会社、メーカ各社)を集め、2015年頃をめどに実用化の可能性が高い高速炉プラントシステムとサイクル技術を提示することを目指して、サイクル機構にて一元的な研究開発が進められている。
