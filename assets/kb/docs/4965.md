# 4・8・1 まえがき

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・8 核融合炉

a. 核融合反応とプラズマ閉込め(plasma confinement)核融合炉(nuclear fusion reactor)とは、核融合反応(nuclearfusion reaction)を制御して持続させ、発生したエネルギーを取出す装置である。核融合炉はいまだ実現されていないが核融合研究は、核融合炉の実現を目指し着実に進展してきた。核融合反応にはいくつかの軽元素同士の反応がある。最も反応を起こしやすいのは、重水素(D)と三重水素(T)を燃料とする核融合反応で、る。DT 反応は、次式で表される。D+T→He(3.5MeV)+(14.1MeV)DT 反応の結果17.6MeV のエネルギーが発生し、その約80%は中性子の運動エネルギーとして放出される。他方U の核分裂反応では、1反応当たり200MeV 程度のエネルギーが発生し、その約80%が核分裂生成物の運動エネルギーである。核融合反応を実現するためには、燃料粒子をクーロンポテンシャルに逆らって近づくぐらいの高温にし、十分な反応が起こる程度に粒子密度を高める必要がある。この状態をある領域の中で起こさせるために、二つの5―139図4・54 主要なプラズマ閉込め方式方法がある。一つは、粒子を超高温にして電離した状態、すなわちプラズマ状態にし、磁力線のかごの中に閉じ込める方法である。この方法を磁場閉込め(magnetic confinement)方式と呼び、磁力線の構造によって種々の方式がある。もう一つは慣性閉込め(inertial confinement)方式と呼ばれる方法で、短時間ではあるが固体粒子の密度を高めて核融合反応をパルス的に起こさせる。燃料粒子の氷の粒を強力なレーザで照射して爆縮させ、中心部に高温高密度プラズマ領域を生成して核融合反応を起こすレーザ核融合がその例である。このように、核融合反応を実現させるためのプラズマを閉じ込める方式として、磁場閉込め方式と慣性閉込め方式とがあり、それぞれの方式はまた、いくつかの炉型に分類することができる。図4・54には核融合炉の実現を目指した主要なプラズマ閉込めの方式を整理した。b. 核融合炉の形式図4・54に示したおもなプラズマ閉込め方式の特徴は以下のとおりである。. トカマク(Tokamak)型プラズマに流す電流(プラズマ電流)が作る磁場とトロイダル磁場コイルが作る磁場とが合成してできる軸対称のらせん状磁力線構造によってドーナツ状容器内にプラズマを閉じ込める方式である。他の方式に比べ、極めて高いプラズマ性能を実現しており、国際協力による建設が予定されている国際熱核融合実験炉(ITER: international thermonuclearexperimental reactor)もこの方式である。プラズマに電流を流すために中心ソレノイドコイルをもつが、定常運転を実現するためには電流駆動装置が必要である。. ヘリカル型ヘリカルコイルにより形成された非軸対称のらせん状磁力線構造によりプラズマを閉じ込める方式である。トカマクと異なりプラズマに電流を流す必要がないので、定常運転に有利であるという特徴を有する。代表的な装置は、核融合科学研究所の大型ヘリカル装置(LHD : large helical device)である。. 逆磁場ピンチ(reversed field pinch)型トカマク型と同様な方式でプラズマを生成した後、プラズマの中心部と周辺部とでトロイダル磁場の向きを反転させる方式。トカマクより相対的に大きなプラズマ電流が必要である。必要なトロイダル磁場の強さがポロイダル磁場と同程度であるため、トロイダル磁場が低くて済む5―140エネルギー供給システムという特徴をもつ。代表的な装置は、産業技術総合研究所のTPE-RX 装置である。. ミラー型プラズマ両端を閉そくした筒状の磁力線によって円筒状容器内にプラズマを閉じ込める方式である。端部からの損失防止に電場の効果を併用する。プラズマ粒子が両端部で跳ね返される磁気ミラー構造であることからこの名称となった。筑波大学のGAMMA-10が代表的な装置である。. レーザ型強力なレーザを球状の燃料ペレットに四方八方から照射し、一気に爆縮する。これによりペレット中心部が強力に圧縮され、高温、高密度のプラズマが生成して核融合反応を起こす方式。外部に噴出した燃料片の慣性力により強力な圧縮力を発生させるのでこの名称となった。わが国では大阪大学の激光12号装置がある。c. 核融合研究の発展核融合の研究開発は、プラズマの高温化とその効率的な閉込めを目指し、さまざまな装置を工夫しつつ進められてきた。1950年代にロシアでトカマク型装置が開発され、それ以後世界中でトカマク型による研究開発が活性化した。順次装置規模を大きくしてプラズマ性能の向上を進め、1980~1990年代に日本のJT-60、ヨーロッパのJET など大型トカマク装置によって、当初の目的であった臨界プラズマ条件(break-even plasmacondition)の達成に成功した。臨界プラズマ条件とは、高温プラズマを発生するために必要な入力とそのプラズマ中での核融合反応による出力とが等しくなる条件である。この出力と入力との比をQ値(Q=出力/入力)と呼び、臨界プラズマ条件はQ=1に相当する。図4・55にプラズマ性能の進歩を示した。縦軸にプラズマの中心密度と閉込め時間の積をとり、横軸にプラズマ中心温度をとったもので、ローソン図(Lawson diagram)と呼ばれる。右上に行くに従ってプラズマ性能が向上していることを意味する。図4・55中に、臨界プラズマ条件と自己点火条件(self-ignition condition)を示した。自己点火条件とは、プラズマへの外部からの入力なしに、核融合反応で生じた3.5MeV のα粒子(Heの原子核)による加熱で核融合反応を持続するために必要なプラズマ状態が維持される条件、すなわち、Q=∞となる条件である。図4・55では、ほぼ10年ごとにプラズマ性能の著しい向上を見ることができる。JT-60やJET による臨界プラズマ条件達成を踏まえ、21世紀のエネルギー源として貢献することを目指し、核融合開発は新しい段階図4・55 プラズマ性能の進歩図4・56 ITER 鳥瞰図に入った。次なる目標は、DT 反応による高いQ値の実現、DT 反応の長時間維持、核融合エネルギー生産に向けた炉工学の確立である。1988年から開始された日本、ヨーロッパ、アメリカ、ロシアの国際協力による国際熱核融合実験炉計画(ITER 計画)ではすでに装置機器の実規模級試作開発を含む工学設計と研究開発、既存のトカマク装置を用いたプラズマ物理および計測に関する研究協力を進め、その建設活動に移行する段階に至っている。他方慣性核融合研究は、近年のレーザの進展と相まって大きな発展期を迎えている。例えばアメリカにおいてはレーザによる核燃焼点火の基礎物理確認を目指した大型試験装置(NIF: nationalignition facility)の建設が進んでいる。大阪大学における高速点火実験の構想もあり、近い将来にはレーザ核融合炉への展望が開かれることも期待される。これまでの核融合研究開発において、トカマク方式は他の方式に比べて優れた性能を示しており、現状では核融合炉の姿を具体的に描くことのできる方式である。図4・55に示したさまざまな磁場閉込め装置においては、炉を構成する機器にトカマクと異なるものもあるが機能的に類似のものが多い。トカマク装置における研究開発の現状を認識することは、他の炉型での構成機器やその課題を理解するうえでも、大きな助けになろう。それゆえ本書では、磁気閉込め方式の代表としてトカマク型核融合炉について、その特徴、核融合炉を構成する機器の機能や構造および、研究開発の現状について具体的に解説する。
