# 5・5・5 組織接着剤（surgical adhesive）

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

a.概略 外科手術の基本操作は切開，止血，縫合の三つに大
別されるが，縫合および止血の一部を接着により行おうとする試み
は比較的古くから見られた．しかし接着剤の組織障害性，接着力の
強度不足などの問題点から普及には至らなかった．現在においても
止血の基本はクリップ等の使用を含めた縫合操作であり，一定以上
の大きさの血管からの出血に対してクリップを含めた縫合手技に取
って代わりうるような止血方法は，電気凝固や組織接着剤等を含め
て存在しない．
また切開・切離された組織が癒合・治癒するには，治癒機転を成
立させる生体自身の創傷治癒過程が根本にある．このためには癒合
治癒すべき局所にかかる物理的な張力を減圧して，適切な癒合面の
間隔と安静が保たれることが，組織の血行とともに重要である．古
くから行われていた外科医による縫合操作も，あくまでこの生体自
身の創傷治癒力や血液凝固能を発揮させるための場を作っているに
すぎないことを見過ごしてはならない．
外科手術に内在するこのような性質の中で組織接着剤の開発が続
けられている理由としては，次のようなことが挙げられる．手術の
最終的操作である皮膚の閉鎖においては，縫合糸の締付けによる血
流障害による創治癒の遅延や，縫合糸による瘢痕（suture mark）
あるいは縫合糸の針孔からの感染等の問題が存在することである．
また脆弱な組織からの気体（呼吸器）や液体（血管縫合部の針孔，
肝臓や膵臓の切離面からの血液，胆汁，膵液等）の漏出に対して
は，やみくもな縫合糸による縫縮によっては組織を損傷するばかり
となる状況も存在する．
このような状況への解決策の一つとして組織接着剤の開発が数十
年来行われてきた．組織接着剤による接着法には，接合する創面に
直接接着剤を塗布する直接接着法と，創面を他の方法（皮下縫合
等）で接合させ，創面にかかる張力を十分に減張させたのちにその
周囲に接着剤を塗布する間接接着法がある．また，現在使用されて
いる医療用の組織接着剤は，フィブリン接着剤等の天然材料の接着
剤とシアノアクリレート等の合成接着剤に大別される．
b.天然材料による接着剤 フィブリン系組織接着剤に代表さ
れる天然材料による接着剤は，同種（ヒト）およびウマ・ウシ等の
異種血しょう製剤由来のフィブリノーゲン，トロンビン等の凝固因
子を用い，フィブリンの形成を促進することにより生理的な凝固，
治癒作用の原理に基づく止血，接着，閉鎖作用を行うものである．
形成されたフィブリンの中で，線維芽細胞の増殖に始まる組織修
復過程により生体固有の組織に置換され，約1箇月で器質化し，3
箇月後には瘢痕化するとされている．この過程の中でフィブリンの
り自体は拒絶反応なしに吸収されることが特徴であり，縫合糸によ
る縫合操作のみでは止血，縫合に難渋する脆弱な体内臓器の切離面
や接着面に対して頻用されている．
現在わが国で使用されているフィブリン系組織接着剤の剤形に
は，液体とシートの2種類がある．液状組織接着剤としてはティシ
ール ，ベリプラスト ，ボンヒール の3種が市販されており，
対象の性状と目的に応じ，直接塗布もしくはスプレー状噴霧にて使
用されている．液状であるので投与対象物の形状を問わず塗布が可
能であるが，溶解性にやや難があるため，使用時の調剤に十数分の
時間がかかるという難点もある．
またシート状組織接着剤には，ウマコラーゲンの基盤の上に，接
着面にヒトフィブリノーゲン，ウシトロンビン，ウシアプロチニン
の粉末状成分を固着したタココンブ が発売されている．シート
状組織接着剤の特徴としては圧迫が可能なシート状であり，また使
用時に混合調剤が不要であるという点が挙げられる．凹凸に富んだ
り，狭小な部位には貼付しにくいが，逆に一定の面積をもった局面
にそのまま使用できるという利点がある．タココンブ において
は，基盤のコラーゲンシートに対する異物反応の結果として吸収過
程で線維・瘢痕化が生じる点が液状フィブリン系組織接着剤と異な
る．しかし肝臓等の実質臓器の切離面等への使用には瘢痕化による
影響があまり問題とはならないため，液状フィブリン系組織接着剤
とともにそれぞれの長所を生かして使い分けられている．
現在フィブリン系組織接着剤は，これらなしには遂行困難な止血
や縫合のために臨床上不可欠なものとして普及・使用されている．
しかしいずれも同種および異種血しょう製剤を使用していることか
ら，エイズやクロイツフェルト・ヤコブ病等が昨今問題となる中
で，未知のウイルスやプリオン等による医原病発生の不安は完全に
は否定できない．この問題への対応として，光合成性プレポリマ
（AdvaSeal）等の吸収性合成接着剤や，遺伝子工学的手法による
フィブリンのり等の生産方法も研究開発されている ．
c .合成組織接着剤 瞬間接着剤として普及しているシアノア
クリレートの組織接着剤への応用は，わが国でも古くは1950年代
半ばから東京大学木本外科において研究されていた ．シアノア
クリレート系組織接着剤の作用機序は，シアノアセテートとホルム
アルデヒドが結合した液状モノマであるシアノアクリレートモノマ
が組織の水分を重合開始剤として，ポリマに架橋・硬化することに
よっている．
シアノアクリレートモノマは，そのアルキル基が短鎖であるほど
強固な接着力を有するが組織毒性も高まることが知られている．シ
アノアクリレート系組織接着剤の接着力は強靱ではあるものの，組
種々の画像診断の進歩にもかかわらず，病変部の組織を光学もしく
織障害性や発がん性の問題から生体内での使用は一部を除いて断念
され，その後普及した天然材料によるフィブリン系組織接着剤にそ
の任を譲っている．しかしその強靱な接着力を生かして，皮膚の上
から使用する間接接着法としての使用法が確立された．
現在わが国ではエチルエステルが医用アロンアルファ として，
またブチルエステルが血管内カテーテルを介した血管内塞栓物質用
のヒストアクリルブルー として医用アクリレート系組織接着剤
が市販されている．また8個の炭素鎖を有しつつも組織接着力と柔
軟性を高めた2-octyl - cyanoacrylateがダーマボンド として
2000年からわが国でも発売されている．
一部の血管造影専門医によって，ヒストアクリルブルー が抗
がん剤の選択的動脈注入用血管内留置カテーテルの固定や止血を目
的として血管内に投与されていることを除けば，シアノアクリレー
ト系組織接着剤は間接接着法による皮膚接着剤として使用されてい
る．
この皮膚接着剤による皮膚接着の利点としては，縫合に際する局
所麻酔の必要性を減少させること，（症例によっては）ガーゼによ
る創部の保護が不要で早期のシャワー浴も可能であること，抜糸を
要さず通院回数も減少できること，縫合糸による瘢痕形成がないこ
となどが挙げられる．しかし感染・炎症が認められる創部や粘膜表
面およびその近傍，毛髪が密集する皮膚などには禁忌とされてい
る ．また中縫いにより筋層，脂肪層，真皮層等が接合されてい
ることも前提である．
このように組織接着剤とは，従来からの縫合・止血手法との適切
なる組合せにおいて有効なのであり，従来の手技を完全に代替する
ようなものではないことに留意しなければならない．
