# 5・5・7 フォトン放射線治療装置（PRS:photon radiosur

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

-
放射線がさまざまな腫瘍の治療に有効であることは明らかであ
図5･43 PRSの内部構造
図5･44 PRS線量分布
は正確にターゲット管先端の金板まで到達する．先端部分はX線
透過性のベリリウム製（beryllium window）で，X線はほぼ球形
の線量分布となる．半径1.5cmでは毎分0.6Gy（グレイ）照射，
すなわち36Gyまでなら1時間以内で照射することが可能である．
ここでGyとは放射線の単位の一つで，従来の放射線治療は1日当
たり2Gy，1週間では10Gyしか照射しないのに対し，このとき
の腫瘍中心部は500Gyにも達するのである（図5･44）．PRSから
発生するのは軟X線であるので，距離の三乗に反比例する．半径
1.5cmの辺縁線量を15Gy照射した場合（40keV，40Aで20
分），半径1cmの部位で52Gy，半径0.5cm以内では500Gy以
上の照射となるが，半径2cmで6.4Gyまで減少する．安全な照
射のため，照射直前にさまざまな測定をPCで施行する．
PRSはすべての脳腫瘍に対し効果があると思われるが，現在お
もに悪性脳腫瘍に用いている．悪性脳腫瘍とは治療に反応しない脳
腫瘍であり，正常の脳組織の中を浸潤していく神経膠腫（グリア細
胞から発生した脳腫瘍）が代表的である．正常組織との境界がはっ
きりせず，手術で完全に摘出することは困難である．PRSにおい
ては，CTなどを用い，脳の特定の場所に針などを正確に到達させ
る装置である定位脳手術装置に取付け脳深部の病変に照射する方法
と，手術で腫瘍をできるだけ摘出したのち残存腫瘍に対してPRS
の先端を挿入し手術中に照射する方法とを用いている．一度放射線
をかけた患者はさまざまな副作用のため二度めは照射できないが，
PRSは追加照射ができる．また小児や老人患者の場合，通常の放
射線治療では知能障害や痴呆の可能性があることから，この方法は
通常の放射線治療後に再発してきた患者，小児や老人患者に対して
もよい適応と考えられる．1995年より2001年まで110例以上に治
療が施行されているが，明らかな合併症は認められず，短期間に腫
瘍の消失が認められた症例も見られた．
PRSは持運びが可能で大量の放射線を正確に照射できるという
夢の装置を，高度な工学技術と最先端のコンピュータ技術によって
実現した医療機器である．今後は脳神経外科領域だけでなく，子宮
がんや膀胱がんの治療にも応用可能であると思われる（国内未承
認．現在ドイツで販売）．
