# 4・9・2 採鉱,製錬(milling and refining),転換

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・9 核燃料サイクル

天然ウラン資源は,オーストラリア,カザフスタン,カナダ等に多く存在しており,確認資源は約460万tU,未確認資源も含めるとおよそ1440万tU 存在している.日本では規模は小さいものの人形峠(鳥取県)や東農(岐阜県)でウラン鉱床が発見されているが,探鉱活動は1988年に終了した.鉱石の採掘方法には,露天掘り,坑内掘り,インプレースリーチング法等がある.採掘した鉱石は粉砕後,酸またはアルカリによってウランを溶出させたものをイオン交換法や溶媒抽出法を用いて濃縮・精製した後,イエローケーキ[重ウラン酸塩NHUOほか]と呼ばれる中間製品を生成する.これを粗製錬という.イエローケーキは再び溶媒抽出法にて不純物の除去を行った後,Hガスによる還元工程およびHFガスとFガスによるフッ化工程を経て,最終的に濃縮に適したUFガスを得る.ここまでの工程を精製錬,転換という.このほか,サイクル機構によって開発された一貫精錬法(PNC法)がある.一貫精錬法では,AMEX 抽出,電解還元,フッ化沈殿・脱水工程を採用し,中間生成物であるイエローケーキの生成およびその前後の工程を省略し,直接UF粉末を製造する.UF粉末はFガスによるフッ化工程を経て最終的にUFガスになる.
