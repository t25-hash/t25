# 4・9・4 燃料加工(fuel fabrication)

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・9 核燃料サイクル

a. 燃料加工技術の概要燃料加工では,各種原子炉の装荷に適した燃料集合体の製造を行う.燃料の化学形態から,大きく金属(合金)燃料,セラミック燃料(酸化物,窒化物,炭化物),液体燃料に区別できる.また,燃料の形状からは,棒状のスラグ燃料,ペレット燃料(pellet fuel),粉末を充てんする振動充てん燃料,被覆粒子燃料等に,燃料の成分によって,ウラン燃料,プルトニウム燃料(ウランとプルトニウムを混ぜたもの),トリウム燃料等に分類することもできる.現在,世界の原子力発電の主流は軽水炉であり,その燃料は,ウラン酸化物(UO)でペレット形状である.このほか,軽水炉再処理サイクルではMOX 燃料のペレット形を,高速炉サイクルでは金属燃料や酸化物燃料を中心に開発を進めているが,さらに窒化物燃料や振動充てん燃料等多様な燃料概念の研究も進めている.わが国では,1970年頃の軽水炉の本格的な導入に合わせて,1970年前後から濃縮ウラン燃料(U 濃縮度3~5%のペレット形燃料)の製造が始まった.その後,自主技術の積極的な採用を進め,燃料性能の向上や品質保証の向上に努め,世界的にも最も破損率の低い水準を保っている.代表的な軽水炉用ウラン燃料の加工はおもに次の工程から構成されている.1濃縮工場から払い出されたUFの酸化物(UO)への再転換2UO粉末を成形,焼結してペレットへの加工3ペレットの燃料棒への装てん4燃料棒の燃料集合体への組立このおもな工程以外に,各製造工程が終了するごとに検査工程が控えており,原材料から最終製品に至るまで品質保証を行っている.燃料加工の工程フローを図4・61に示す.b. 軽水炉および重水炉燃料の製造軽水炉用として微濃縮ウラン燃料を,重水炉(CANDU 炉)用としては天然ウラン燃料を製造する.再転換の方法は,大きく湿式法と乾式法の2種類がある.ADU法はわが国やアメリカが広く採用している湿式法で,UFを加熱,蒸発させ,純水およびアンモニア水と反応させるとADU の沈殿が生じる.ろ過,ばい焼還元を行い,UO粉末を得る.工程はやや複雑であるが,製造した粉末の特性は安定している.一方,乾式法はイギリスやフランスで採用されており,装置がコンパクトのうえ,液体廃棄物の発生が少ないなどの長所を有している.ペレット製造工程では,UO粉末をペレットと呼ばれる直径1~1.5cm,高さ1~2cmの円柱状のセラミックスに焼き固める.おもな装置は,成形装置,焼結炉(sintering furnace),研削装置である.再転換工程より受け入れたUO粉末は,流動性を高めるため必要に応じて粗成形や造粒等の前処理を行った後,多孔金型付プレスやロータリプレス等の成形装置を用いて加圧・成形して,グリーンペレットと呼ばれる成形体を作る.グリーンペレットは耐熱性の金属ボードに載せ,バッチ式あるいは連続式の焼結炉に搬入する.400~800°Cの予備焼結で結合材や潤滑材を除去した後,約1700°Cの高温水素雰囲気中で数時間焼き固める.図4・62にプッシャ式連続焼結炉の概念図を示す.焼結ペレットはパーツフィーダにより連続的にセンタレスグラインダと呼ばれる研削装置に供給され,設計仕様どおりの寸法になるように研削する.外観,寸法,表面粗さ,不純物の検査を受けた後,燃料棒加工工程へ送る.燃料棒加工工程では,下部端栓溶接済みの燃料棒の中に所定の量のペレットおよびスプリング,水素・水分吸収用ゲッタ等の要素部材を装てんし,Heガスを充てん(BWR では約3atm,PWR では約30atm)した後,上部端栓をTIG(tungsten inert Gas)溶接を行う.溶接部の健全性,燃料棒全体の外観,寸法,封入したUOの濃縮度等について全数検査を行った後,BWR ではタイプレートやスペーサ,PWR ではノズルや支持格子等の集合体構成部品と合わせて集合体に組立てる.燃料棒(被覆管)の材質には,中性子吸収が少なく,優れた耐食性と機械的強度を有するジルカロイ-2(BWR 用)とジルカロイ-4(PWR 用)と呼ばれるジルカロイ合金(ジルコニウムに少量の鉄,クロム,スズ等を添加したもの)が用いられている.プルサーマル用MOX 燃料の製造工程は,基本的には微濃縮ウラン燃料と同じであるが,プルトニウムを扱うため装置をグローブ図4・63 成形整列装置の概要図5―145ボックス内に設置するなどの対応が必要である.詳しくはc項で述べる.c. 高速炉燃料の製造高速炉ではプルトニウムを燃料とし,酸化物(MOX)を中心に金属や窒化物等の燃料開発が進められている.形状としては,ペレット燃料のほかにスラグ燃料,振動充てん燃料,被覆粒子燃料等が開発対象となっている.これまでに,日本をはじめ,アメリカ,フランス,ロシアの高速炉を中心に,酸化物ペレット燃料や金属スラグ燃料を装荷した実績がある.MOX ペレット燃料の製造方法はb項のUOペレット燃料の製造方法と基本的に同じである.湿式再処理工程でマイクロ波加熱転換したプルトニウム富化度調整済み(プルサーマル用MOX 燃料の数~10%に対して15~30%)のMOX 原料粉を受け入れた後,成形,焼結,研削,検査工程を経て,燃料棒の加工,燃料集合体の組立を行う.被覆管の材質にはおもにオーステナイト系ステンレス鋼を用いている.成形整列装置の概要を図4・63に示す.潤滑剤を粉末に混合するためのデルタ型ミキサとレシプロ型プレスおよびペレット整列機から構成されている.潤滑剤と混合したMOX 粉末は,4箇所程度のダイスに充てんし約5秒間圧縮する.成形したグリーンペレットは,ペレット整列機により整列させMO製の焼結皿に載せた後,焼結設備へ払い出す.MOX 燃料を扱う場合,各装置に対しては臨界,遮へい,熱,耐震および閉込めの観点から,UO燃料製造の場合より高い安全設計基準が要求される.これは,プルトニウムがウランに比べ,最小臨界質量が小さい,比放射能が高い,発熱量が高いなどの理由によるもので,粉末等を扱う装置は質量管理や形状管理により厳しく臨界管理するとともに,負圧管理した密封性の高いグローブボックスの中に設置し,プラントとしてプルトニウムが周辺環境に漏えいしないように閉込め機能を有している.また,作業中の被ばく低減のため遠隔自動化での運転方法が採用されている.振動充てん燃料(vibration packing fuel)製造の場合,その工程は,湿式再処理工程からのU/Pu混合溶液の受入れ,プルトニウム富化度の調整,ゲル化法による顆粒製造,焼結,振動充てんによる燃料棒の加工,燃料集合体の組立から構成される.図4・64に示す外部ゲル化装置では,プルトニウム富化度の調整後のU/Pu混合溶液に,粘性等の調整のためにポリビニルアルコールとテトラハイドロフルフリルアルコールの添加物を加えた後,ノズル先端に液滴を作り,先端を振動させることにより液滴を切り離し,水酸化アンモニウム溶液中に滴下してゲル球を生成する.ゲル球はゲ図4・62 プッシャ式連続焼結炉の概要図5―146エネルギー供給システム図4・65 振動充てん装置の概要図ル化の際に発生した硝酸アンモニウム溶液とともに,醸成・洗浄・乾燥装置に移送され,昼夜をかけて外側からゲル化反応を進める.形成されたゲル球は,アンモニア等を除去するために洗浄し乾燥させた後,焼結工程に搬出する.焼結工程ではさらに還元雰囲気中にて1300°C程度に加熱することにより高密度の焼結球を得る.焼結を終えた粒子はその粒径に応じて分級され,振動充てん装置に送られる.振動充てん装置の概要を図4・65に示す.上部のホッパに充てんした粒子は,定量フィーダを通して一定量がマガジンに縦型に装着した燃料要素(あらかじめ下部端栓溶接された燃料棒)に供給され,周波数が数百Hz 程度の条件下でおよそ5~10min振動充てんし,所要の充てん密度を得る.d. ガス炉燃料の製造被覆粒子燃料(coated particle fuel)はガス炉用の燃料として開発され,燃料核と呼ばれる直径約200~600 mの微小のセラミック燃料球を熱分解炭素(PyC)や炭化ケイ素(SiC)等で被覆したものでできており,代表的なものに三重ないし四重被覆で構成されたTRISO形燃料がある.被覆層は核分裂生成物(FP: fission products)ガスの閉込め,燃料核のスウェリングの吸収,被覆粒子全体の強度保持等の機能を有している.各被覆層の厚さは,数十~60 m程度である.被覆粒子燃料の製造工程は,燃料核製造,被覆,燃料棒加工,集合体組立の大きく四つの工程に分かれる.燃料核の製造方法は,振動充てん燃料の粒子製造にも用いられているゲル化法がおもに採用されている.燃料核への被覆方法は,流動床と呼ばれる蒸着炉において化学蒸図4・64 外部ゲル化装置の概要図図4・66 再処理工程のフロー図(ピューレックス法の場合)着(CVD)によって行われる.流動層と呼ばれる場所に燃料核が入り,1100~1600°C高温に加熱された粒子流動ガス(Arガス)と原料ガス(アセチレンやプロピレン等)の混合ガスが下部のノズルから吹上げ,原料ガスの熱分解で生じた炭素が燃料核に蒸着する.TRISO燃料の場合,バッファ層にはアセチレンを,PyCにはプロピレンを,SiC層にはメチルトリクロロシランを原料ガスとして使う.燃料要素の加工は燃料コンパクトやボール燃料の製造を行う.燃料コンパクトの製造ではオーバコート法や射出成形法が,ボール燃料の製造ではオーバコート法が用いられている.集合体組立では,燃料コンパクトの場合,さらに直接またはピン(黒鉛スリーブ)にいったん挿入し,黒鉛ブロックに装荷する.ボール燃料の場合はそのまま炉内に装荷するため,集合体組立工程は特にない.
