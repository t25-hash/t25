# 4・9・5 再処理(reprocessing)

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・9 核燃料サイクル

a. 再処理技術の概要再処理とは使用済燃料を化学的に処理し、廃棄物となるFP等と再利用可能な核分裂性物質(U,Pu等)や燃料親物質(U 等)を分離・精製し、回収することである。そもそもアメリカにおいて軍事利用目的で始まった再処理技術開発は、その後、軽水炉燃料を対象として開発が進み、湿式法として溶媒抽出法[ピューレックス法(purex process)、レドックス法、ブテックス法等]、沈殿法(リン酸ビスマス法)、イオン交換法が、乾式法として高温冶金法(フッ化物揮発法等)や高温溶融塩法(溶融遠電解法等)等の研究が行われた。1950年半ばからリン酸トリブチル(TBP: tri-butyl phosphate)を溶媒としたピューレックス法が工業化の主流となり、現在は高速炉再処理への適用も行っている。日本においては、1977年、日米交渉の成立を受けてサイクル機構の東海再処理工場のホット試験が始まり、さらに2006年ころには日本原燃(株)が六ヶ所再処理施設(処理能力800t U/y)の操業を開始する予定である。両施設ともピューレックス法を採用している。代表的な再処理工程のフローを図4・66に示す。b. 軽水炉燃料の再処理軽水炉から取出した使用済ウラン燃料は、再処理施設に搬入し、貯蔵プールで冷却貯蔵される。一定期間貯蔵後、使用済燃料は、せん断、溶解、清澄、分離(共除染、分配)、精製、転換(脱硝)の各工程を経て、UO粉末あるいはMOX 粉末(UO-PuO粉末)製品となる。せん断機(chopping machine)の概要を図4・67に示す。水平にセットされた燃料集合体は、燃料送出し装置によりせん断ホルダのほうへと送り出され、4~5cm程度の寸法でせん断される。せ図4・67 集合体せん断機の概要図図4・68 連続式溶解槽の概要図ん断した燃料片は溶解槽(dissolver)に供給する。図4・68に連続式溶解槽の概要を示す。溶解槽には東海再処理工場で採用している回分式と六ヶ所再処理施設で採用している連続式がある。連続式は槽内の燃料インベントリーが少なく臨界管理が容易であるうえ、連続処理を行うため大容量化に適している。材質は強い耐食性を有するステンレス鋼等が用いられている。燃料せん断片は一定角度ごとに回転するホイールの軸中心付近から、ホイール内に仕切られたバスケット内に供給され、バスケットがホイール下部にさしかかった際、一定時間、高温で比較的濃度の高い硝酸溶液と接触して燃料が溶解する。溶け残った被覆材はハルと呼ばれ、溶解槽より取出し洗浄後、高レベル固体廃棄物として貯蔵する。一方、溶解液は清澄・調整工程に送られ、パルスフィルタあるいは遠心式の清澄機によりmオーダの粒子状の不溶解残渣を取り除いた後、次の分離工程に適した硝酸およびウラン濃度を調整する。再処理の中心である分離(共除染、分配)工程では、ウラン、プルトニウム、FP等の分離を行う。ピューレックス法では、有機溶媒のTBP(有機相)と硝酸水溶液(水相)の間において、各元素の溶媒に対する分配係数の違いを利用して、二相分配(溶媒抽出)を行う。共除染工程では、原子価が6価のウランと4価のプルトニウムを有機相に抽出し、水相に残った大部分のFPとマイナアクチニド元素(Np,Am,Cm等)等不要なものはラフィネート(廃液)と呼ばれ、高レベル廃液処理工程へ送られる。分配工程では、有機相に抽出したウランとプルトニウムのうち、プルトニウムの原子価を4価から溶媒に抽出されにくい3価に還元し、水相に移行させ分離す5―147図4・69ミキサセトラの概要図る。有機相中のウランは濃度の低い硝酸溶液と接触させ水相中に逆抽出し、プルトニウムとともに次の精製工程へ送る。還元法としては、還元剤(Uやヒドロキシルアミン)の添加や電解還元がある。溶媒抽出にはミキサセトラ(mixer-settler)、パルスカラム(pulse column)、遠心抽出機と呼ばれる、連続・多段・向流式の抽出装置を用いる。抽出剤のTBPには希釈剤としてドデカンまたはケロシンを加える。図4・69にミキサセトラの概要を、図4・70にパルスカラムの概要を示す。ミキサセトラでは、ミキサ室のかくはん羽根により混合された水相と有機相は、セトラ室で上下に分離され、仕切板の上から有機相、下から水相が次の段のミキサ室へ流れ込む。特長としては、設計が容易で運転が安定している。パルスカラムでは、カラムの上から水相、下から有機相を導入し、二相の向流接触により抽出を行う。シャフト部の環状部には多孔板が設置され液体をパルス的に塔内に送り込むことにより、接触効率を上げている。混合した二相はカラムの上部と下部にあるセトラ部分で分離され、上部からは有機相、下部からは水相が取出される。溶液の滞留時間および滞留量が少なく、溶媒の放射線劣化や臨界管理上の緩和が期待される。東海再処理工場ではミキサセトラ、六ヶ所再処理施設ではパルスカラムを採用している。精製工程はウラン精製設備とプルトニウム精製設備から構成される。精製方法は分離工程と同じ溶媒抽出法であり、製品スペックを満たすようにFP等を除去する。六ヶ所再処理施設では、ウラン精製設備にミキサセトラ、プルトニウム精製設備にパルスカラムを採5―148エネルギー供給システム図4・70 燃料の再処理については、おもにピューレックス法をベースに改良・合理化を加えた湿式法とアメリカ、ロシアにて開発された乾式法が、サイクル機構や電力中央研究所を中心に研究されている。先進湿式法と呼ばれる再処理概念では、簡素化溶媒抽出法、晶析法、マイナアクチニド回収工程(SETFICS+TRUEX プロセス)等を採用し、従来のピューレックス法より経済性や資源有効利用性等に優れたシステムの開発を目指している。一方、おもに金属燃料用に開発された乾式法(酸化物電解法、金属電解法、フッ化物揮発法)の酸化物燃料への適用も検討されており、動作温度や処理時間等の運転条件の改良、廃棄物発生量の低減、マイナアクチニド回収効率の向上、白金族FPの分離等の改良に取組んでいる。酸化物電解法における電解槽の概念を図4・72に示す。このほか、金属燃料や窒化物燃料の再処理への乾式法の適用も研究されている。
