# 5・6・3 ガンマナイフ

> **章**: 第5章 手術・治療機器
> **節**: 5・6 放射線治療

ガンマナイフ（gamma knife）はコバルト60（cobalt -60）を線
源としたγ線を用いた定位的放射線治療装置で，正式名はガンマ
ユニット（図5･64）である．定位的放射線治療（stereotactic
radiosurgery）とは放射線照射の焦点が目標位置に正確に一致する
ように，定位的に位置決めと固定を行って照射する方法である．ガ
ンマユニットではあたかも外科で使うメスで病変を切り取るように

図5･67右後大脳動脈より栄養される動静脈奇形は，ガンマナイフ治療
により4年8箇月後の検査では完全に消失した
放射線照射が行われるため，この機器はガンマナイフまたはガンマ
メスと呼び慣らわされており，学術用語としてもガンマナイフが用
いられている．
この機器の原型は1968年にスウェーデンのレクセル（Leksell ）
により開発された．従来の外部照射との違いは，ガンマナイフでは
線量の集中度が極めて高く，一度に高線量を照射して目標部位を破
壊することである．
ガンマナイフの構造（図5･65) は，201個のコバルト小線源が
半球状に配列されたものである．おのおののコバルト線源からの線
量は比較的弱く，皮膚・頭蓋骨・脳実質などに吸収されそのエネル
ギーは半分以下になるが，各線源から発生するγ線は個々のコリ
メータにより散乱しない細いビームとなって半球の中心に焦点を作
るように設計されており，焦点部ではこれらの集合で強い照射線量
を得ることができる．コリメータは，本体内の固定部分（図5･65
中②）と交換可能なヘルメット部分（図5･65中③）からなり，後
者を病変の大きさに合わせて4種類の中から選ぶ．4種類のコリメ
ータによる線量こう配は急峻であり，焦点から少し離れると急激に
γ線照射線量が軽減することから，周囲の正常脳組織への照射を最
小限に抑えながら，目標の病変にのみ十分な線量を照射し，その結
果として境界明りょうな破壊巣を作ることができる．このことがガ
ンマナイフの安全性につながっている．
治療計画は，患者の頭部にレクセル式定位脳手術用フレームを4
点固定する．そのままの状態でCT，MRI ，または血管撮影を行
い，これらの機器とガンマナイフ専用コンピュータシステムを直結
し，各画像データによる三次元画像再構成も含めて病変の正確な位
置と大きさ，照射線量および線量分布を決定し，必要なコリメータ
ヘルメット（図5･65中③，図5･66）を選択する．照射対象となる
病変が不整形であれば，異なる大きさのコリメータを用いて線量分
布を専用のコンピュータソフトウェアにより描く．境界線量として
は線量こう配の最も急峻な50～70 %を用いる．50%を用いた場合
に最大のコリメータ1回で照射される範囲の直径は25.2mmであ
り，これより大きな病変では複数のコリメータを組合わせて必要な
線量を得ることとなる．しかしながら，縦・横・高さがすべて30
mmを超えるような病変では，境界部の線量こう配が急峻でなく
なるため，周辺の正常脳組織への照射量が大きくなり，ガンマナイ
フとしての本来の目的から外れてしまう．
ガンマナイフのおもな適応疾患は脳動静脈奇形，手術困難あるい
は高齢者の悪性および良性腫瘍であり，その作用機序は高線量一括
照射として腫瘍血管や血管腫の血管内皮細胞に対する高度の障害か
ら血管壊死を引き起こすことである．脳動静脈奇形および腫瘍の栄
養血管の内皮細胞は障害により壊死を起こしながら細胞質の膨化が
起こり，血管内腔を閉そくする．脳動静脈奇形では血管腫内に凝血
が起こり閉そくする（図5･67）．腫瘍では栄養供給が絶たれるため
腫瘍の発育は止まり壊死に陥る．脳動静脈奇形の閉そく機転には数
箇月から2～3年が必要であるが，ごく小さな転移性腫瘍であれば
短期間に治療効果が得られている．
最近では，機能性疾患へのガンマナイフ治療が有効であることが
わかり注目されている．難治性てんかん，三叉神経痛，がん性疼
痛，パーキンソン病などに応用されているが，これらに対する作用
機序はまだ十分には解明されておらず，機序の解明と新たな適応疾
患について今後の研究が期待されている．
