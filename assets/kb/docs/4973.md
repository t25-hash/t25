# 4・9・6 使用済燃料および高レベル廃棄物(high level waste)

> **章**: 第4章 原子力エネルギー技術・システム
> **節**: 4・9 核燃料サイクル

の処分ワンススルーでは、使用済燃料(spent fuel)を30~50年中間貯蔵(interim storage)した後、最終処分場の真上にあるコンディショニング施設に搬入され、処分容器にBWR で10~40体程度の集合体を封入する。中間貯蔵方法としては、キャスタ貯蔵方式、ボールト貯蔵方式、サイロ貯蔵方式、水プール貯蔵方式がある。再処理サイクルの場合は、ガラス固化体を同じく30~50年冷却貯蔵した後、炭素鋼オーバパックに収納する。処分容器およびオーバパックはエレベータにより地下数百mよりもさらに深い処分場に運び、埋設する。使用済燃料またはガラス固化体(vitrified waste)中の放射性物質を長期間にわたって人間の生活環境から隔離するよう、ガラス固化体の場合はガラス、ステンレス製キャニスタ、オーバパック、緩衝材(圧縮ベントナイト)の人工バリアおよび地層による天然バリアを組合わせた多重バリアの概念が用いられている。圧縮ベントナイトはいわゆる粘土で、非常に低い透水性、自らすきまやき裂を充てんする自己シール性、放射性核種を吸着し移行を遅延させる性能、高い熱伝導性、化学的および物理的緩衝性等の人工バリアとして優れた性能を有している。サイクル機構を中心に関係機関が協力して検討を進めてきた高レベルガラス固化体の地層処分(geological disposal)の研究開発の成果を受けて、2000年には地層処分基準の整備、サイト選定、処分施設の建設を行う機関として電力業界の出資による(財)原子力発電環境整備機構が設立された。2030年後半の最終処分の開始を目指している。文献(1) 通商産業省資源エネルギー庁公益事業部原子力発電安全管理課編、
