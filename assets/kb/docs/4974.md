# 5・1・1 太陽エネルギー

> **章**: 第5章 再生可能エネルギー
> **節**: 5・1 概説

大気圏外において太陽光に垂直な面の太陽エネルギーの密度は1.395kW/mであり,地球に入射する大気圏外の量は173×10kW になる.そのうち,約30%は大気圏外で反射され,残りの70%のうち,67%(81×10kW)が大気,地表,海面に吸収されて熱になり,33%(40×10kW)が蒸発,降雨,海洋循環のエネルギーになる.海流や波力に変換されるエネルギーは370×10kW,光合成に寄与する分は40×10kW と推定されている.a. 太陽エネルギー資源地球上で太陽エネルギーを利用する場合,地球は地軸が傾き,自転しながらだ円軌道を描きながら太陽を周回し,大気圏内の気象変化の影響を受けるために大気圏外と異なって太陽からの放射エネルギー量は地球上の地域(位置)によって異なる.そのため,赤道地域が最も日射がよく,極地が最も低いことになるが,地表面全体における単純に平均した日射量は,散乱光[diffused solar radiation;雲や周囲から反射された太陽光.太陽から直接くる光を直達光(direct solar radiation)という]を含表5・1 太陽熱利用太陽温水器で知られるように太陽光を黒色の面で吸収して太陽熱を得ることができる.この太陽熱を得る装置を集熱器あるいはコレクタといっている.集熱器には,平板に太陽熱を吸収する水や空気が通る管を取付けた平板型と管を真空の円筒ガラス管内に封入した真空管型の2種類が市販されている.太陽光を受ける平板や管の表面は,太陽光をよく吸収し,自ら放射する光が少なくなるように特殊加工が施されている.このような特徴を有する面を選択吸収面(selective sur-face)という.これらの集熱器を用いて,集熱面積が3m以上のものをソーラシステムといい,それには,1給湯システム,2暖房・給湯システム(暖房の方式として,床暖房,空気集熱暖房がある),3冷暖房・給湯システムなどがあり,用途に応じていろいろな種類のシステムを構成することができる.太陽熱で給湯,暖房を行うことは比較的容易であるが,太陽熱で冷房する場合は,太陽熱を熱源に駆動する吸収式などの冷凍機を用いる必要がある.しかし,使用期間が夏だけに限られ稼動率が低く,経済性に優れないために太陽熱で冷房するソーラシステムは少ない.このように,太陽光を直接受けて冷暖房・給湯する100°C程度の熱は太陽から容易に得ることはできるが,それよりも温度の高い高温の熱を得ようとするには,太陽のエネルギー密度が低いためにレンズや反射鏡などを用いた光学機器でエネルギー密度を上げなければならない.このような光学機器を用いて高温の太陽熱で発電するシステムが太陽熱発電システムである.発電方式は,蒸気を用いる通常の火力発電と同様の方式で,蒸気発生に太陽熱を用いる点が異なっているだけである.しかし,太陽からくる光は昼間だけであり,天候に左右されるため安定に電気を供給するために蓄熱装置を設置,あるいは天然ガスなどの補助熱源をもつことが多い.また,太陽は時々刻々位置が変化する.ソーラシステムを構成する集熱器は屋根などに固定して設置しても効率的に太陽熱を得ることができるが,高温の熱を得るには,太陽から直接くる直達光しか利用できないために,光学機器は太陽を追いかけて光を集めるようになっている.これを太陽追尾(sun tracking)という.太陽熱発電の技術開発はわが国で行っていないが,アメリカでは,発電出力が約354MW の太陽熱発電システムが稼動している.また,日射に優れた地中海沿岸地域では,ギリシャのクレタ島で50MW のプラントが建設中であり,スペイン,モロッコ,エジプトなどにも建設する計画が報告されている.c. 太陽光利用この代表的な例は半導体による光起電力効果により電気を発生する太陽電池である.太陽電池はp形とn形の半導体を接合し,両面に電極を設けて,太陽光に照射すると,電子と正孔の対ができ,両電極を結線するとそこに電子が流れ,直流の電気を得ることができる.太陽電池の種類はシリコン系と化合物系に大別される.シリコン系太陽電池は,結晶系(単結晶,多結晶)と非晶質のアモルファス系に分けられる.化合物系太陽電池には,II-VI族(CdS/CdTe),III-V族(GaAs,InP)がある.この電池を数十枚,直並列に接続したパネルを太陽電池モジュール(solar cell module)といい,各種の太陽電池モジュールが市販されている.太陽電池モジュールの大きさはだいたい,幅1m,長さが2m以内で,出力は数十W である.これらの太陽電池モジュールを必要とする枚数を直並列に組合わせたものを太陽電池アレイ(solar cell array)といい,それから得られる電気を利用したシステムが太陽光発電システムである.太陽光発電システムは基本的に,1日射があるときだけ直流の電気を負荷に供給するシステム,2蓄電池を設置し,直流から交流に変換するインバータにより交流の電気を供給するシステム,32と同様に交流の電気を供給するシステムであるが,蓄電池をもたないで,太陽から電力が得られない場合は商用の配電線から電力をまかなうシステムの三つの構成がある.12のように商用電力と連係しないで,単独で太陽からの電気を供給するシステムを独立型,3を系統連係型という.また,蓄電池の有無,独立型では特に負荷が直流か,交流かによっても太陽光発電を区分することもある.これらの区分によれば,1は独立型直流負荷蓄電池無システム,2は独立型交流負荷蓄電池有システム,3は系統連係型蓄電池無システムとなる.わが国で住宅用として普及している3kW 級の太陽光発電システムの多くは3の方式である.太陽電池モジュールの効率は,化合物半導体太陽電池で30%のものもあるが,市販されている結晶系シリコン太陽電池の効率は約15~18%である.非晶質のアモルファス太陽電池は10%程度である.太陽光発電システムを普及させる最も大きな要因は太陽電池の低コスト化である.そのために薄膜シリコン太陽電池や色素増感太陽電池などの新型太陽電池の開発が進められている.一般に太陽光発電システムの電気は照明や電化製品の電源として利用されるが,電気を水電解に利用すれば酸素・水素の燃料を製造することができる.このように太陽光から得られる電気により燃料を製造する方法のほかに,TiOなどの酸化物半導体光触媒に太陽光を照射すると,水を酸素・水素に分解することができる.この方式での太陽エネルギーの利用率は,現在非常に低い値であるため効率向上のための研究開発が進められている.
