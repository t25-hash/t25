# 5・1・3 海洋エネルギー

> **章**: 第5章 再生可能エネルギー
> **節**: 5・1 概説

わが国は四面を海に囲まれているため,海流,潮流や波力の運動エネルギー,表層と深層の海水温度差熱エネルギーなどの海洋エネルギーに恵まれた国である.a. 海流エネルギー海流のエネルギーの例として,わが国近海を流れる黒潮のもつエネルギーは平均流速0.5m/s,水深1km,流域幅120kmとすると約1872GW になる.しかし,海流は沿岸部から遠く,発電設備の設置技術などが未開発であるために大規模海流発電の実績は世界的に見ても皆無である.b. 潮流エネルギー潮流は太陽エネルギーに起因した現象でなく,潮汐現象に起因した周期的な海水の流れである.潮流は湾の入口が狭まった場所や陸地に挟まれた海峡,水道で流速が速くなり,その流れのもつ運動エネルギーを利用することができる.わが国では潮流の流速は12時間25分の周期で正弦波状に変化し,半周期ごとに流れの向きが反転する.わが国で最も有望な利用地域は,鳴門海峡,来島海峡などの瀬戸内海にあり,それらの潮流エネルギーの賦存量は2500万kW と試算されている.c. 波力エネルギー波力,あるいは波浪のエネルギーは年平均海岸線1m当たり日本海側で9~10kW/m,太平洋側で6~9kW/mであり,賦存量は3600万kW と試算されている.波のエネルギーを力学エネルギーに変換するための方式として,1可動物体方式(movable column;海洋中に物体を置いて上下方向,水平方向,揺動などの運動によってエネルギーを吸収する方式),2振動水柱方式(oscillating water column;容器を水中に沈め,上部に閉じ込められた空気の流れを操作してエネルギーを吸収する方式),3水流方式(water flow;波を堤で集めて流路に流れを作り,その流れを利用する方式),4受圧面方式(directreceived;波を直接,圧力,流れの形態で受ける方式)がある.また,これらの変換装置の設置の相違から洋上に設置される浮遊式,海岸に固定設置される固定式に分けられる.わが国で開発されている方式は水柱振動型がおもで,基本的に空気を閉じ込めるA,Bの2室からなっている.波で2室が圧縮されるとき,B 室の弁が開いて,A 室の空気がタービンを回転させながらB 室に入る.波が去って海面が下がると2室の圧力が低下して逆にB 室の弁が閉じ,A 室の弁が開いて,外部から空気が侵入し,その空気がA 室からB 室に流れるとき,タービンを回転する.このように波が上下しても弁の操作で一定方向の空気の流れを作り,その空気の流れで発電するようになっている.波力を利用して実用化されているのは,現在,航路標識用の電源とした数十W 程度の出力をもつ振動水柱型のブイだけである.大型のシステムは,これまでに各所で波力発電を設置して実証・開発が行われてきたが,現在,わが国で稼動している波力発電は,海洋科学センターが三重県で行っている浮遊式振動水柱型のマイティホエール(50kW が1基,30kW が2基)と東北電力の福島県にある原町火力発電所の防波堤に設置した固定式振動水柱型の水弁集約式波力発電システム(130kW)である.d. 海洋温度差エネルギー海洋温度差エネルギーは,表層と深層の海水温の温度差のエネルギーで,そのエネルギーを利用して発電する方式が海洋温度差発電であり,OTEC(ocean thermalenergy conversion)ともいわれる.これには,アンモニアなどの低沸点媒体を表層の海水で蒸発させ,その蒸気でタービン・発電機を駆動し,タービンから排出された蒸気を深層水で凝縮し,低沸点媒体を閉じた系内で作動させるクローズドサイクル(closedcycle)と低圧の蒸発器で表層の海水を蒸発させ,その蒸気でタービン・発電機を駆動し,その蒸気を深層水で冷却して海底に放出するオープンサイクル(open cycle)がある.これらの設備は設置によって洋上式,陸上式に分けられる.わが国やアメリカでOTECの開発が行われたが,現在,日本では佐賀大学の75kW 級の陸上式実験設備とインド洋で佐賀大学が指導し開発した1000kW の洋上式クローズドサイクル(アンモニアと水の混合液が作動媒体)の発電システムがあり,このシステムは2002年の初頭から本格的な運転が行われている.これらはすべてクローズドサイクルのシステムである.
