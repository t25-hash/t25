# 5・1・4 水力エネルギー

> **章**: 第5章 再生可能エネルギー
> **節**: 5・1 概説

高所にある水の位置エネルギー(落差)を利用して水車などによって得られる動力が水力であり、おもにその動力は水力発電として利用される。a. 水力資源世界では、カナダ、アメリカ、ノルウェー、スウェーデンなどで開発が進められている。中国、アジア、南アメリカなどでも水力エネルギー資源は潜在的に膨大であると推定されているが、未開発であるため、利用できる量は未知である。わが国では、雨量、地形などから水力資源量が推定されており、その量を理論包蔵水力といっているが、そのうち、技術的・経済的に開発が可能と考えられる量が包蔵水力(hydraulic potentialenergy,water resources)である。水力発電の方式は後述するが、1998年3月時点、事業用としての水力発電の地点、発電出力は、それぞれ、一般水力(conventional hydraulic power)が1521地点、1992万kW、純揚水発電が22地点、1747万kW、混合水量が20地点、573万kW であり、発電規模として1発電所の出力は平均数万kW 以上である。100kW 以上の水力は266地点、140万kW である。しかし、わが国には、一般水力として33000MW、混合揚水として、14000MW の包蔵水力がある。また、未開発地点も多くあり、技術の進展と経済性の尺度の変化によって開発できる可能性は高い。一次エネルギー供給の水力の割合は1999年において約4%であり、比率は小さいが、純国産エネルギーを確保する観点から重要なエネルギーではある。b. 水力発電水力発電は、一般に落差を得るための土木設備の構造、河川流量の運用上の特性から分類されている。しかし、最近は、規模の大きな適地が少なくなり、中小規模の水力発電が多くなっているため発電規模と水力発電所の運用上の観点から分類されることもあり、発電出力が1000kW を境に、それより大きいものを大規模、それ以下を中小規模とする場合もある。さらに、大規模発電は、河川水量を利用して、その運用の仕方によって一般水力と揚水式に分類される。一般水力には、河川水量を調節する池をもたないで、河川水を緩やかなこう配の水路によってヘッドタンクに導いて、落差を利用して発電する方式の流れ込み式と、高所にある自然湖水や河川ダムを造って水をせき止め、その水を直接または水路に導いて落差を利用する貯水池(調整池)式に分けられる。一方、これらの方式に対して落差を得るための土木設備の構造から水路を用いる方式を水路式、ダムを構築する方式をダム式と分類することもある。揚水式(pumped storage)は、深夜などの余剰電力を利用し下部貯水池の水を上部の貯水池にくみ上げるか(純揚水)、あるいは、一般水力と組合わせて(混合揚水)昼間の電力ピーク時に発電する揚水発電がある。中小規模の発電が中小水力発電であり、発電出力が100kW 以下を小水力(ミニ水力、mini hydraulic power)、100kW から1000kW 程度が中水力(マイクロ水力)であり、流れ込み式で発電する方式が多い。これらは発電規模が小さいので、水路などの工事を行わないで、自然の落差を利用して発電する場合が多い。これらの水力発電の中で、わが国における代表的な方式は貯水池式(ダム式)発電方式である。ダムとは、河川をせき止めて流水を貯水、あるいは取水する目的で建設された構築物である。ダムは治水やかんがい用のために利用されることもあるが、発電用のダムのうち、水路式発電所の取水のために設けられた高さ15m以下の低いダムは取水ダムあるいは取水ぜきといわれ、貯水を目的にしたダムと区別されている。ダムにせき止められた水は導水路を通してサージタンクに導入される。サージタンクは、負荷変動(発電量変動)によって生じる水量の急激な変化による水圧変動を吸収し、導水路に及ぼす圧力変動を軽減する働きをする。水圧管路は、サージタンクから水力発電所の取水口に通じ、水車に直接導水するための水路であり、サージタンクと発電所取水口の落差の圧力がかかっている。水圧管の材料は一般に鋼管であるが、落差が100m程度の中小水力では、強化プラスチック管が採用されている。
