# 5・1・5 バイオマス

> **章**: 第5章 再生可能エネルギー
> **節**: 5・1 概説

バイオマスとは、COと水の無機物から光合成によって有機物が合成され、植物、動物、微生物などの生物に形態変化した有機体の総称である。したがって、バイオマスは、森林資源の産物である木材、海で生育する海藻類、食料となる植物、植物の未利用部分や家畜ふん尿などの有機廃棄物など、さまざまな種類がある。また、バイオマスは、直接燃料に使用することもできるが、ほかに、メタン発酵や菌による水素の発生などによりエネルギーに変換することができる。それらをバイオ燃料(biomass fuel)、バイオエネルギー(biomass energy)といっている。バイオマスを燃料などに使用した後、COを大気中に放出するが、同量のCOがバイオマスの再生産により固定される限り正味の大気中のCO濃度は変わらない。これをバイオマスの炭素中立性(カーボンニュートラル、carbon neutral)といい、太陽が存在する限り再生産されるのでバイオマスは再生可能なエネルギー資源である。a. バイオマス資源バイオマス資源には、多くの種類があるが、それらを生産資源、未利用・廃資源に大別することができる。これらすべてのバイオマス資源量を算定することは困難であるが、地球上のバイオマスは乾燥重量にして、陸地で、1兆8370億t、海洋で約40億t、合計1兆8410億tが存在すると推定されており、毎年太陽エネルギーによって再生産されるバイオマスの光合成量は陸地で1150億t、海洋で550億t、合計1700億tといわれている。また、現存するバイオマスのうち、樹木が1兆7000億tと全体の約92%を占め、樹木、すなわち、森林バイオマスが主要なバイオマス資源になっている。わが国の森林バイオマスの蓄積量は約35億mで、未利用のバイオマス資源は約4300万t/年である。この量が有するエネルギーは、約1800万tの石油に相当する。農業・畜産などからの有機廃棄物の総量は約2億7000万t/年であり、わが国における廃棄物の総量の約60%を占めている。b. バイオマスエネルギー変換バイオマスをエネルギーとして使用するために、・バイオマス生産資源の栽培(プランテーションバイオマス、plantation biomass)・未利用・廃資源の有効利用(廃棄物の有効利用)の二つの方法が考えられている。前者の場合は、将来のエネルギー生産を目指したエネルギー作物の栽培である。休耕田などの有効利用も考えられるが、食糧生産と競合する資源であれば問題となる。後者は、直接燃焼させて燃料に使用するのが最も簡便な方法であり、発展途上国では日常生活の燃料として利用されている。特に、廃棄物資源を利用する場合、ごみ発電で利用されているように、わが国では、都市ごみで代表される未利用・廃資源の利用が有望と思われる。ごみ発電は、通常、ごみを燃焼させて約250°Cの蒸気で発電しているが、蒸気温度を330°C以上にすると塩化水素などで管が高温腐食を引き起こし、発電設備の寿命が低下し、温度が低いため効率はせいぜい15%程度である。そのため、効率を向上させるために、スーパーごみ発電(wasted combined power generation)が開発されている。このシステムでは、ごみは従来どおりの温度の蒸気を発生するために使用されるが、ガスタービンで発電する設備を併設して、ガスタービンから排出されるガスが高温であるので、この排熱を利用してごみの燃焼で発生した蒸気の温度を約450°Cに加熱して蒸気タービンの効率を向上させて発電するシステムになっている。このように二つの発電設備を組合わせて発電する方式を複合発電(combined cycle power generation)という。以上のようなバイオマスを直接利用した技術のほかに、他の形態の燃料に転換する技術がある。バイオマスの多くは、固体であり、連続的に使用したり、移動させるのが容易でない。そのため、これらの欠点を取り除くために気体や液体の燃料に転換する技術がある。すでに実用化されている技術から基礎的な研究段階のものまでさまざまな燃料転換技術があるが、これらの技術が開発されれば、バイオマス資源を含むエネルギー資源に乏しいわが国では、これらの技術を利用し、バイオマス資源の豊富な海外諸国において気体、液体の燃料を製造すれば、輸送、貯蔵が容易であるため、輸入しやすい石油に代わる燃料を得ることができ、環境問題を引き起こさないクリーンな燃料を得ることが可能になる。
