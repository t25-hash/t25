# 5・7・4 皮膚・形成外科治療用レーザ手術装置

> **章**: 第5章 手術・治療機器
> **節**: 5・7 レーザ治療装置

進行がんに対しては，食道がん，胃噴門がんによる狭窄の解除に用いられる．
蒸散凝固治療にはNd:YAGレーザが用いられることが多い．
近年では，小型・軽量でNd:YAGレーザと同等の効果をもつ半導体レーザも用いられる．
このほか，光線力学的療法（PDT）も行われる．
低出力Nd:YAGレーザによるレーザ温熱療法も開発されたが，消化管で用いられることは少ない．
あざの原因となる色素に強く吸収される波長を選択することと，パルス幅を特性熱緩和時間よりも十分に小さくすることで，周囲の健常皮膚に熱的影響を及ぼさない選択的レーザ治療が可能になる．
黒あざ（母斑性母斑）ではメラニンに，一方赤あざ（血腫）は微小血管を破壊するためにヘモグロビンに光を吸収させる．
メラニンと周囲の健常組織（コラーゲン）との吸収差が大きい波長帯は700nm近傍であり，ルビーレーザ（波長694nm）が使用される．
深部あざ治療には光侵達度が長いNd:YAGレーザが用いられる．
