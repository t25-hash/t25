# 5・1・6 地熱エネルギー

> **章**: 第5章 再生可能エネルギー
> **節**: 5・1 概説

地球の内部には高温のマグマという熱源があり，その熱が地表に
向かって流れている．地表から約3℃/100mの割合で温度が高く
なっているが，地下2km程度では約200℃を超える温度になって
いる．地表から水が断層などを通って地下に浸透し，この高温の熱
で加熱され，熱水となって地下に蓄えられている．熱水が一部の断
層を通って上昇し，温泉あるいは蒸気としてわき出してくる．この
ように地下に存在する熱水のもつ熱エネルギーが地熱エネルギーで
ある．地下に存在する高温の岩体の熱を利用する方法も開発中であ
るが，一般には熱水を利用している．発電には，蒸気や坑井を通し
て地上に取出した熱水が利用され，低温の熱は温泉やその熱を直接
利用している．
a.地熱資源 地熱資源として，地下3kmまでの深さに蓄え
られている熱を利用できるものとし，その熱量を積算すると地熱発
電に利用できる量は20540MWと試算されている．現在，地熱の
発電量は540MWであり，利用している量はわずか2.6%程度で
ある．わが国は火山列島であるため，条件が整えば，地熱は大きな
エネルギー資源になり得る．
b.地熱利用 地熱は発電や直接利用することができる．地熱
発電は，既存の火力発電と同様な方式で，ボイラで蒸気を発生する
代わりに地下からの蒸気を使って発電する点が異なっているだけで
ある．発電方式には，坑井から蒸気だけが取出される場合は，その
蒸気を直接利用して発電する蒸気発電（steam turbine power gen-
eration）と坑井から蒸気と熱水が得られる場合は，熱水を減圧し
て沸騰させて（これをフラッシュという），これらの蒸気で発電す
るフラッシュ発電（flashed steam turbine power generation）が
ある．坑井内を上昇中に圧力が低下してフラッシュする蒸気を一次
蒸気といい，地上で蒸気と熱水を分離（気水分離）し，熱水を減圧
沸騰させてできた蒸気を二次蒸気という．一次蒸気を利用する発電
方式をシングルフラッシュ発電，二次蒸気も利用する方式をダブル
フラッシュ発電という．現在，わが国で実用的に稼動している地熱
発電は18基あり，その総設備容量は54万3600kWである．その
うち，工場やホテルの自家用の設備は3万6800kWである．発電
方式は，13基がシングルフラッシュ，ほかがダブルフラッシュ方
式である．地熱発電設備は北海道に1箇所，ほかは東北と九州にあ
るだけである．
直接利用は，地下から取出された熱水を温泉のようにそのまま利
用することもあるが，熱水に含まれている成分が不都合な場合，清
水と熱交換してその熱を利用し，熱水は地下に還元される．地熱は
温度によって工業，農林・水産業，民生用など多様な用途があり，
多くの地域で利用されているが，大規模に利用するところまでは至
っていない．

a.中小水力の分類 中小水力エネルギーの利用としては，水
力発電によるものが大部分で，出力10万kW以下のものを中小水
力発電といっている．このうち10万kW以下1万kWまでを中水
力としているが，それ以下の小水力（small hydraulic power）に
ついては，一般的に1万kW以下1000kWまでを小水力，1000
kW以下100kWまでをミニ水力（mini hydraulic power），100
kW以下をマイクロ水力（micro hydraulic power）と分類してい
る．
電気事業法上，発電機器から除外されている10kW未満のもの
は，一般はん用機器として取扱われるので，簡易な設備が可能であ
り，特に東南アジア等発展途上国などではピコハイドロと呼ばれて
普及している．
