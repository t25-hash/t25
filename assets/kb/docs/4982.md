# 5・3・1 地熱発電の歴史と現状

> **章**: 第5章 再生可能エネルギー
> **節**: 5・3 地熱

a. 国外天然蒸気を用いて世界初の地熱発電(geothermalpower generation)を行った場所はイタリアのラルデルロである。当時ラルデルロでは天然蒸気中からホウ酸を取出していたが、1904年にジノリ・コンチ公爵がこの蒸気を用いて0.75馬力の発電機により5個の電灯をともした。その後、同地区では1913年に出力250kW の地熱発電所を建設して、工場の照明用に利用しており、これが世界で初の地熱による商用発電である。同地区における発電量は徐々に増大し、第二次世界大戦時には一時壊滅的な打撃を受けたが、現在では世界第3位の設備容量をもつ。それ以降、1958年にニュージーランドのワイラケイにおいて6.5MW,1960年にアメリカのガイザーズにおいて10MW,1967年に旧ロシアのパウジェトカにおいて11MW の地熱発電所が誕生している。このうち、ニュージーランドにおける地熱発電技術は第二次世界大戦終了後にイタリアに駐留していたニュージーランド軍によりもたらされたとされている。2001年には世界中で休止中であるものも含めて23箇国が地熱発電設備を有している。全世界の総発電設備容量は約3191GW であり、地熱発電設備容量の合計は約8.3GW であることから、後者は前者に対して約0.3%の割合を占めていることになる。地熱発電設備容量の最も大きな国は、アメリカ(2228MW)、次いでフィリピン(1931MW)、イタリア(923MW)、メキシコ(890MW)、インドネシア(590MW)であり、日本(549MW)は世界第6位である。また、総発電設備容量に対して地熱発電の占める割合が高い国は、順番にエルサルバドル(16.8%)、フィリピン(16.0%)、ニカラグア(15.8%)、アイスランド(13.9%)、コスタリカ(9.8%)となっている。b. 国内国内における地熱発電の研究は1918年の大分県朝日村において深さ約24m、直径約12cmの坑井を掘削し、この場所で東京電灯(株)研究所長の太刀川年治博士が1925年に1.12kWのタービンによる発電に成功したことから始まる。地熱発電に関する研究は第二次世界大戦により中断するが、戦後1948年には(株)利根ボーリング[現:利根地下技術開発(株)]が湯ノ沢試験場で出力3kW のタービンによる発電、1951年には工業技術庁(現:独立行政法人産業技術総合研究所)が別府実験場で30kW の発電、さらに1952年には(株)利根ボーリングが宮城県鳴子町において出力20~30kW の発電に成功している。これらの試験的な発電の成功を受け1953年には九州電力(株)が大分県大岳、1956年には東化工(株)[現:日本重化学工業(株)]が岩手県松川において本格的な地熱開発の調査を開始した。これらの調査の後、松川では1966年に日本で初めての9.5MW の地熱発電所、さらに大岳では1967年に11MW の発電所の運転を開始している。最近では1998年に八丈島で3300kW の地熱発電所が運転を開始しており、現在の総発電設備容量は事業用と自家用を合わせて約549MW に達している。国内の地熱発電所の概要(事業用12地域で14発電機、自家用6箇発電所名所在地発電部門蒸気供給部門設備容量MW*1 DS はドライスチーム(天然蒸気背圧タービン方式)、SFはシングルフラッシュ、DF はダブルフラッシュを表す。*2 /は、ダブルフラッシュにおいて一次蒸気/二次蒸気を示す。また圧力はゲージ圧である。所)を表5・3にまとめた。
