# 5・7・8 心臓・血管系レーザ治療装置

> **章**: 第5章 手術・治療機器
> **節**: 5・7 レーザ治療装置

パルスレーザの熱影響を推定するには，熱が熱伝導により周辺に散逸する時間の目安として，特性熱緩和時間が定義される．
形状による補正係数（平板の場合4，円筒の場合は8），χは熱伝導熱緩和時間よりも十分に短いパルス幅，および十分に長いパルス間ることができる．
疹などの除痛用として日常的に使用されている．
するもので，脈絡血管からの眼底出血に対する止血も行える．
Arレーザ（波長647nm）も用いられる．
後者は角膜表面の形状をレ角膜に照射すると，熱損傷のない精密なレーザ蒸散が生じる．
この程が含まれているが，組織除去の主たる過程は短時間に生じる熱蒸る．
このほか，同じ装置を用いて角膜の実質表層にできる病変（角atectomy）も可能である振気泡に達したとき，破綻的に圧壊することがある．
症の心筋梗塞治療法として，レーザで心筋壁に直接貫通孔をあけて光ファイバを束ねたマルチファイバカテーテルを用いて動脈硬化にで示され，δは入熱部分の寸法（光加熱の場合は光侵達長）， は率（ほぼ水の熱伝導率と考えると1.3×10 cm /s）である．
特性隔でレーザ照射すると，周囲健常組織への熱影響を最小限にとどめ（5～100mV）が使用されている．
副作用がないことから，帯状疱者はレーザにより網膜をスポット状に熱凝固して網膜はく離を治療レーザ（波長514.5nm）が一般的だが，網膜深部の凝固にはKrーザ蒸散で変化させ眼の屈折力を矯正する治療装置で，PRKArFレーザ（波長193nm）が用いられる．
パルス幅10ns程度で作用には紫外光エネルギーによって分子結合が切断される光解離過散である．
よる血管狭窄を除去するレーザ治療装置が開発されており，冠状動脈治療に多く用いられる．
同期させた出力800WのCOレーザ治療装置などがある．
