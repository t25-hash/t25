# 5・3・3 地熱プラント計画

> **章**: 第5章 再生可能エネルギー
> **節**: 5・34 に示す.わが国ではこの方式による地熱発電所が一般的であ

地熱発電所の運転開始までは,対象とする地熱地域で開発が可能かどうかを調査する段階と,それ以降の生産井や還元井掘削,発電所の建設等の具体的な開発という段階を踏むのが一般的である.a. 調査段階調査は通常,概査,精査,構造試すい井掘削,調査井掘削という手順で行う.概査は精査地域を絞り込むための調査であり,精査は対象とする地域の地質構造と地熱流体との関係を概念的に把握するためのモデルを作ることを目的とする調査である.このために,地質調査,地化学探査(流体地化学,水同位体組成,地化学温度計,土壌ガス・土壌水銀法等),物理探査(リモートセンシング,重力探査,磁気探査,電気探査,電磁気探査,地震波探査等)の調査や探査が実施される.これらの調査や探査は基本的には地表より行われるため,実際に地下の地熱流体の温度を正確に知ることをおもな目的として構造試すい井が掘削される.この際にコアが採取され地質学的に識別されて地質柱状図が作成される.また電気検層や温度検層等の坑内検層も実施される.構造試すい井によって開発の可能性があると判断される場合には調査井を掘削する.調査井では地熱貯留層から地熱流体を噴出させ,その生産量や物理化学的な性質を把握することが主目的である.構造試すい井も広い意味では調査井であるが,構造試すい井では噴気試験は通常行わないという点で異なる.この後,必要に応じて補足精査を行うこともある.この調査井による噴気試験や,生産井や還元井を掘削した場合の成功率,立地条件,環境条件等を総合的に判断して事業化の見通しが立てば,開発に進むことになる.b. 開発段階開発は,計画の作成からはじまり,道路・坑井基地土木工事,生産井・還元井掘削,生産井の流量特性試験,生産井・還元井掘削,生産井・還元井掘削,地表配管・発電所建設工事,全坑井生産還元試験,発電機試運転を経て,運転開始に至る.生産井と還元井の掘削計画を立案するうえで,(1)長期にわたって安定して生産量が維持できる生産井間の間隔,(2)還元井に還元した熱水が地熱貯留層の地熱流体に悪影響を及ぼさないような生産井と還元井の配置,(3)必要とする蒸気量から見積もられる生産井の総数,(4)還元能力から見積もられる還元井の総数,(5)生産井を掘削しても蒸気が出ないことがあるため,この成功率,(6)坑井の傾斜堀による掘削基地の集約化等を十分に検討する必要がある.
