# 5・8・1 音響化学治療システム

> **章**: 第5章 手術・治療機器
> **節**: 5・8 音響化学的原理による治療技術

a.概要 音響化学治療（sonodynamic therapy）は，患部に
対する選択性が薬物のみでは不十分なとき，図5･76に示すように，
超音波照射により薬物を患部でのみ活性化させて，全身的副作用を
抑えながら治療しようというものである．これを広くとらえると超
音波照射による薬物吸収の促進なども含めることができるが，ここ
では，生体に対する超音波照射による微小気泡（microbubble，マ
イクロバブル）の圧壊が生体に投与した化学物質との組合せにより
引き起こす生体作用を利用する治療と定義する．このような治療は
臨床的にはいまだ一般的でないが，研究が盛んに行われるようにな
ってきた．
