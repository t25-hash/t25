# 5・3・4 将来の地熱発電技術

> **章**: 第5章 再生可能エネルギー
> **節**: 5・34 に示す.わが国ではこの方式による地熱発電所が一般的であ

a. 高温岩体発電システム(hot dry rock power generation sys-tem)高温岩体発電システムは,高温ではあるが,地熱流体も貯留構造もない岩体から熱を取出し発電を行うためのシステムとして,1970年ごろにアメリカ,ニューメキシコ州のロスアラモス国立研究所のモートン・スミスとロバート・ポッターが考案した.このシステム作成の手順はまず高温の岩盤中に1本めの坑井(注入井)を掘削し,この注入井に高圧の水を圧入することにより,数十mから数百mの広がりをもつ人工貯留層を高温岩体中に造成する.その後,人工き裂を貫通するように2本めの坑井(生産井)を掘削する.発電は,注入井から送り込んだ水を人工き裂内で加熱し,生産井から蒸気や熱水として噴出させることにより行う.発電に使用した後の熱水は再び注入井に送り込む.本システムはアメリカ,イギリス,EU,オーストラリア,ドイツ等で開発が進められており(一部の国では終了),国内においても新エネルギー・産業技術総合開発機構(NEDO)や(財)電力中央研究所が研究開発のための現場実験を行っていた.これら世界各国において現場実験が開始された当初は注水井と生産井がそれぞれ1本ずつというシステムが一般的であったが,その後,注水井2本と生産井2本からなるシステム(NEDO)や,高透水性の岩盤を対象として生産井からポンプによって強制的に熱水をくみ上げるシステム(フランス)も開発されている.b. 地熱井涵養技術アメリカエネルギー省は1999年にEGS(enhanced geothermal system)と呼ばれる新しい地熱技術開発政策を発表した.この英文名に対応する的確な日本語はないが,内容的には地熱井涵養技術と訳すことができる.アメリカ,ガイザーズ地熱発電所における年間蒸気生産量は1987年の23.1×10tをピークとして徐々に減衰し,1995年には12.6×10tとピーク時のおおよそ55%まで低下した.これはこの地域の地熱貯留層の透水性が低いために生産井近傍に十分な熱水が供給されなくなり生産井が枯渇したためである.このような枯渇現象はガイザーズばかりではなく,イタリアのラルデルロや国内の松川地熱発電所等の蒸気卓越型地熱地域でも発生している.ガイザーズではこのような状況に対応するため,近郊のクリアレーク地域からパイプラインによって処理廃水を運び,それを地熱貯留層に対して積極的に圧入した結果,圧入前と比較して39MW 分の発電量が増加した.EGS はこの経験を踏まえ,対象をガイザーズのような既存の地熱発電地域で枯渇が進行している地域から,その周辺地域,さらには高温岩体まで拡大し,これらの地域において地熱貯留層に積極的に注水を行って,枯渇の進行防止あるいは蒸気生産量の増大を目指す技術開発である.
