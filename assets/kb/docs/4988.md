# 5・1・2 アウトガス

> **章**: 第5章 材料
> **節**: 5・1 宇宙環境が及ぼす材料への影響

アウトガス（out -gassing）は，真空中で材料から発生するガス
のことであり，材料自身の劣化と周辺機器に対する汚染の点で，材
料選定時の重要な評価項目の一つとなっている．
アウトガスの特性指標としては，質量損失比（TML:total
mass loss ），再凝縮物質量比（CVCM:collected volatile conden-
sable materials ），および再吸水量比（WVR:water vapor
regained）があるが ，TMLとCVCMの二つの特性で評価する
のが一般的である．材料選定に当たっての推奨値は，TMLが1%
以下，CVCMが0.1%以下である．なお，上記評価ではアウトガ
ス特性の時間による変化率がわからない，再凝縮温度条件（25℃）
が実宇宙環境温度（－100～＋100℃）を反映していないなどの課題
があるため，より実宇宙環境に即した試験条件を考慮したアウトガ
スレート測定 も行われるようになっている．
アウトガスは，おもに有機材料から発生し，特に接着剤，塗料，
潤滑油，グリース等から生じたアウトガスによる周辺の光学機器や
太陽電池パドル等への汚染が問題となっており，留意する必要があ
る．
