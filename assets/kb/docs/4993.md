# 5・4・8 可変速運転(variable-speed operation)

> **章**: 第5章 再生可能エネルギー
> **節**: 5・4 風力

風力タービンの最適設計理論によると、翼型特性に及ぼすレイノルズ数変化の影響を無視すると、風車空力性能(出力係数、power coefficient)は周速比(=ロータ周速/風速、tip speedratio)だけの関数であり、したがって最適効率はある周速比の値で与えられる。プロペラ型(水平軸型)風車ではブレード枚数によるが、最適周速比は6~10の範囲にある。自然風は絶えず風速を変動させており、最適なロータ回転数はこの最適周速比を維持しながら可変とするのが合理的である。従来は、発電機出力の周波数変動を避けるために定速でロータを駆動したが、パワーエレクトロニクスの進歩を背景としたDCリンクシステム等の導入により、広い風速範囲で効率が高い可変速運転が主流となった。可変速運転は、周波数変動が制御しやすく、電力品質の維持に優れていることに加え、回転系のフライホイール効果の利用によってロータや主軸の機械荷重を緩和する利点もあり、耐久性を要求される機械においては重要である。
