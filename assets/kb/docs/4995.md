# 5・4・10 出力制御(power control)

> **章**: 第5章 再生可能エネルギー
> **節**: 5・4 風力

風車の出力制御は失速制御方式(stall control)とピッチ制御方式(pitch control)に大別される.前者は,突風や強風を受けたときにブレードに発生する失速現象(stall)を利用し,後者はブレード軸をねじって迎え角を減少させ,もって回転数・出力を制御する.失速制御にはブレード先端部を稼動とし,過回転時に遠心力を利用して突出させ,これを空力ブレーキとして機能させるものもある.また,ブレードをねじり,積極的に失速を招いて制御するアクティブストール(active stall)もある.しかし表5・4と表5・5に見られるように,現代の大型風車はより的確な制御を行うためにピッチ制御を採用する傾向が高い.
