# 5・4・11 風車用翼型

> **章**: 第5章 再生可能エネルギー
> **節**: 5・4 風力

風車はその作動原理によって揚力型と抗力型に分類される.しかし,発電事業を目的とする風車はエネルギー変換効率が高い揚力型が適している.この揚力はブレードの翼型断面に発生する.したがって高性能風車の開発には高性能翼型の開発が不可欠であり,ドイツのヴォルツマン翼型,アメリカのSERI・NRELシリーズ,オランダのデルフト工科大学TU シリーズ,デンマークRISO研究所の翼型ファミリー,スウェーデンのFFA 翼型,産業技術総合研究所のMEL翼型などが挙げられる.風車用翼型に求められる特性は,高揚力係数,高揚抗比,安定した失速特性,レイノルズ数非依存性,高性能圧翼などである.つまり,空力性能と構造強度の双方を担う翼型が求められる.
