# 5・4・13 外部条件(external conditions)

> **章**: 第5章 再生可能エネルギー
> **節**: 5・4 風力

外部条件という概念は風車の安全性と性能を保障する出発点であり、設計者のみならず、風車設置者においても重要である。A,B 乱流強度の分類。は風速が15m/sのときの乱流強度。図5・43 伊是名島の離島用風車(100kW×2基)風車の外部条件は、(1)風況(エネルギー源である)、(2)電力系統(出力先である)、(3)その他の自然・社会条件(陸・海、平地・複雑地形など)、の三つである。風車の設計においては、外部条件は、長期的な構造荷重を評価する正常外部条件と、まれではあるが危機的荷重を評価する極値外部条件に分けられる。これらの荷重ケースは、多様な風車の運転状態と組合わせて検討される。
