# 5・4・15 離島用風車

> **章**: 第5章 再生可能エネルギー
> **節**: 5・4 風力

わが国は,台風が襲来し,また山岳部や岬・離島など地形が複雑で,アクセス性が悪く,乱流成分が高く,また電力系統も弱小である外部条件を有するサイトが多い.ナショナルプロジェクト「離島用風力発電システム等技術開発」は,わが国の過酷な外部条件に適合する風力発電技術を開発するもので,2基の22m直径-100kW風車[図5・43,製造業者:富士重工業(株)]が伊是名島で試験運転下にある.設計上の特徴として,耐風速80m/s,耐用年数20年,20t以上の大型クレーンが不要,最大40%の系統投入割合の実現などが挙げられる.
