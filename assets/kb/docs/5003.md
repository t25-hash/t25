# 5・5・2 太陽熱発電

a. 開発されている太陽熱発電システムの集光・集熱方式と特徴高温の太陽熱を得るためにさまざまな方式の集光・集熱方式が提案されているが、現在、開発が進められている太陽熱発電(以下、熱発と称する)は図5・63に示したような四つの方式によって高温の太陽熱が収集されている。各方式の特徴は以下のとおりである。. 集中型(central receiver type,solar tower)平面鏡を取付けて太陽光を反射させるヘリオスタット(heliostat)をタワーの周囲に多数配置し、タワーの頭部に取付けた集熱器(receiver)に集光して熱を吸収する。図5・63ではヘリオスタットがタワーの片面にしか配置されていないが、全周に配置することもある。この方式はタワー型(central receiver)ともいわれる。. 分散型(distributed collector type)図5・63に示したような形状の円筒放物面鏡により太陽光を反射して焦線(focalline)上の集熱管(円筒真空ガラス管内に集熱媒体が流れる管を配置した管)で熱を吸収するコレクタ(collector)を敷地に直並列に接続して太陽熱を集光・集熱する。この方式はコレクタの焦線(集熱管)の向きによって東西型、南北型に区分される。. ディシュ型(dish type)回転放物面の内面に取付けた鏡により太陽光を反射して焦点近傍に設置した集熱器で高温の熱を収集したり、あるいはスターリングエンジンを設置して発電する。特に、スターリングエンジンを用いた場合、ディシュ/スターリング発電といわれる。図5・63のように回転放物面の内面をすべて鏡面にする場合が基本であるが、ストレッチドメンブレン(stret-ched membrane)といわれる直径約1.5mの小型の鏡面(円形の枠の両面に薄膜で覆って太鼓状にして、片面に薄膜のガラス鏡を取付け、内部をわずかに減圧した凹面鏡)を用いて回転放物面を擬した平面状の支持体に取付けて焦点に太陽光を集光する方式もある。. ダブル集光型(double concentration type, beam downtype)集中型でタワー頭部に集熱器でなく反射鏡を取付け、ヘリオスタットで反射した太陽光を再度反射して太陽光を2回にわたって集光して地上に設置した集熱器で熱を吸収する。頭部の反射鏡で集光した光が地上の集熱器に下向きに入射するためビームダウン(beam down)型ともいわれる。これらの四つの集光・集熱方式は以下のような物理的な特徴をもっている。(1) 設置条件i,ii,ivの方式はヘリオスタットやコレクタの設置のために広大で平たんな敷地を必要とするが、iiiの方式はそのような形状の敷地を必要としない。特に、この方式でスターリングエンジン発電を行う場合、集熱媒体が流れる配管がなく配電線であるため設置が容易になる。(b) 分散型(a) 集中型図5・63 太陽熱発電の集光・集熱方式5―173(2) 熱の収集i,iii,ivは集熱器で集中的に熱を吸収するが、iiは広く分散配置したコレクタから熱を回収する。そのため、熱の輸送過程中の配管部からの熱損失は配管長によって変わり、iiiで集熱する場合、集熱部分の配管長に比べて輸送配管長が長いために熱損失は最も多く、次いでii,i,ivの順になる。iiiの方式で発電する場合はその損失がない。また、日本のように緯度の高い地域では季節によって太陽高度が大きく変化するためi,ivは太陽高度が高い夏に集熱量が多いが、冬季は減少する。しかし、ii,iiiはその影響が少ない。(3) 動力変換i,ii,iiiおよびivでは太陽熱で蒸気を発生し、その蒸気で発電する場合は大型のシステムになっている。しかし、iiiでスターリングエンジン発電する場合はエンジン内の気体の膨張・圧縮によって発電するために小型のシステムであり、発電規模は使用台数で調整する。(4) 異常時の運用i,ii,ivは少数のヘリオスタットやコレクタが故障した場合、それらを停止するだけで、システム全体の運用を停止する必要はないが、i,ivで集熱器が故障したときは運用不能になる。iiiの場合は故障したものだけを除外すればよく、全体の運用を停止する必要はない。このような方式で高温の太陽熱を得る場合、太陽光の直達成分(太陽から直接くる光)のみが利用でき、直達光は太陽の位置の変化とともに移動するためにヘリオスタットやコレクタなどは直達光に直面するように太陽を追尾する装置が設けられている。この動作を太陽追尾(sun tracking)という。b. 海外における太陽熱発電システムの概要熱発技術はオイルショックを契機に石油代替技術として、1974年からわが国をはじめアメリカ、フランス、スぺイン、EU 連合、IEA で国家予算、国際協力資金のもとでプラント開発が進められた。1981年には各国で運転が開始され、約2年間の運転により太陽熱による大規模発電技術が実証された。しかし、いずれのプラントも実用化を目指したプラント開発に進展することはなく、わが国とフランスは開発を終了し、アメリカなどでは地道な開発研究が継続された。一方、この間にイスラエルに本社をもつルーツ社は民間資金のもとで分散型のシステムを開発した。このシステムの詳細は後述するが、その1号基をアメリカ、カリフォルニア州のモハベ砂漠に発電端出力14MW を建設し、1985年から運転が開始された。その後、同様の方式のプラントが毎年のように規模を大きくしてプラントが建設され、現在、9基のプラントで総発電出力354MW で稼動している。1990年代に入り、COなどによる温暖化問題が重要な地球環境問題になり、熱発開発が注目され、特に、地中海沿岸で活発な開発計画が進められている。以下にこれまでに開発された主要なプラントと現在開発中のプラント建設計画の概要を示した。.
