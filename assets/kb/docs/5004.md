# 5・5・3 太陽光発電

> **章**: 第5章 再生可能エネルギー
> **節**: 5・53 は、1995年に建設された鉄筋コンクリート3階建て43戸の集

太陽光発電システムは太陽電池を用いて電力を発生させるシステムであり、その基本構成は図5・73に示すように主として太陽電池アレイ、蓄電池、インバータからなる。太陽電池アレイは太陽電池の集合体を意味し、太陽エネルギーを電気エネルギーに直接変換するシステムの心臓部を担っている。また、インバータはパワーコンディショナとも呼ばれ太陽電池アレイの出力である直流電力を交流電力に変換するとともに、システム全体を制御する機能を備えている。蓄電池は負荷に電力を安定的に供給するために使用される。a. 太陽光発電システムを構成する要素技術。 太陽電池太陽電池の歴史は、1876年、アダムスとデイがセレンの光電効果を発見したことに始まる。ただ、当時は光検出用のセンサとして利用する程度であった。その後、1954年ベル研究所のピアソンらは、シリコンのPN 接合を利用した太陽電池の開発に成功した。それ以来、材料、設計理論、測定手法などに大きな進展が見られ、変換効率が大幅に改善された。太陽電池の実用化は、1958年アメリカで打ち上げられた人工衛星に搭載した無線装置の電源として単結晶シリコン太陽電池によるものである。1960年代に入ると人工衛星の電源として、わが国では無人灯台の電源や海上に浮かぶブイなどのへき地に設置される小容量電源として用いられるようになった。第一次石油危機(1973年)以降、国内外で地上用太陽電池開発図5・73 独立型太陽光発電システムの大規模なプロジェクトが推し進められた。また、1990年代に入りエネルギー資源問題とともに地球環境問題が大きく取上げられ、太陽電池に大きな役割が期待されている。太陽電池は太陽光を直接電気エネルギーに変換する素子で、通常は2種類の半導体を接合して構成されている。太陽光が太陽電池に入射すると、半導体内部で電流の担い手である電子ができ、外部に負荷を接続すれば電流が流れ電力を取出すことができる。ここで、外部回路が短絡しているとき流れる電流を短絡電流(short-circuitcurrent)といい、開放したときの両端の電圧を開放電圧(open-circuit current)という。これらの値は太陽電池の性能を示す重要な値である。太陽電池の良否は、太陽電池に入射した太陽光の何%が電気エネルギーに変わるかによって決められることが多い。この値は変換効率といわれており、一般に10~30%程度である。これは、太陽光のスペクトル分布が300~2500nmに広がっているにもかかわらず、各種太陽電池の光感度が限られたスペクトル帯域にあることによる。太陽電池は図5・74に示すように結晶系シリコン太陽電池、非晶質系(アモルファス)シリコン太陽電池、化合物太陽電池に分類される。結晶系シリコン太陽電池は、単結晶と多結晶のシリコン基板を使用した太陽電池で発電効率、信頼性に優れている。非晶質系(アモルファス、amorphous)シリコン太陽電池は非晶質系のシリコンを薄膜状に作成した太陽電池である。アモルファスとは、原子配列に規則性をもたない材料を意味する。この材料を用いたアモルファスシリコン太陽電池は製造工程が簡単である、製造のためのエネルギー量が少ない、集積型などセル構造の工夫により自由に電圧、電流が選択可能であるなどの特徴をもっている。その結果、コストの安い太陽電池が実用化されるものと期待されている。化合物太陽電池は化合物の半導体材料から作る太陽電池で、ガリウムヒ素、カドミウムテルル、銅インジウムセレナイドなどがある。特に、ガリウムヒ素系の太陽電池は効率、信頼性に優れている。太陽電池の変換効率は年々向上しており、研究開発の成果が着実に上がっている。また、各種太陽電池の理論変換効率を求めるシミュレーション技術が進展し、設計に反映されるようになってきた。算定された変換効率の理論限界値は、単結晶型シリコン太陽電池では30%程度、ガリウムヒ素など化合物半導体太陽電池では40%程度である。世界の太陽電池生産量はここ数年順調に伸びており、2002年度に約560MW となっている。わが国の生産量の全体に占める割合は約45%、アメリカは約23%、ヨーロッパは約23%、残りがその他でわが国の生産量は世界のトップとなっている。また、世界全体では結晶系シリコン太陽電池が全体の約70%、残りがアモルファスシリコン太陽電池である。わが国では2010年に約500万kW の導入を目標に国の長期計画のもとで開発が進められている。. 蓄電池時々刻々変化する太陽エネルギーをエネルギー源とする太陽光発電システムでは、日射変動を抑制することや太陽5―177エネルギーの利用できない夜間・雨天時に電力を供給する必要がある。蓄電池は太陽電池アレイの出力と負荷の要請を整合するため重要な働きをする。太陽光発電システム用の蓄電池が備えなければならない条件は、次のとおりである。・多い放電サイクルに耐えること・不完全な充電・放電に耐えること・自己放電が少ないこと・保守の必要がないこと太陽光発電システム用の蓄電池として現在使用されているのは、おもに鉛蓄電池とアルカリ型蓄電池である。鉛蓄電池には多くの種類があり、使用サイドの要求に応じて選択する必要がある。例えば、梅雨時期の太陽電池出力の減少をカバーする蓄電池としては、自己放電の少ないことが重要であり、低自己放電型が望ましい。また、太陽光発電システムに適用される蓄電池は、計画的に使用されるより、ランダムな条件で使用されることが多いことも肝に銘じておく必要がある。そのほかにナトリウム・硫黄、亜鉛・塩素、亜鉛・シュウ素、レドックスフロー型電池、電気二重層キャパシタなどがある。これらの電池は新型電池と呼ばれており、実用化のための研究開発が進められている。. インバータ太陽電池アレイの出力は直流電力であり、一般的な負荷は交流電力である。インバータはパワーコンディショナとも呼ばれ、基本的には直流電力を交流電力に変換する機能を備えている。また、太陽電池アレイと負荷とを連結して、最大電力効率の維持制御、電力品質の維持、運転の安定性、配電線との保護協調の働きがある。1958年にサイリスタが発表されて以来、いろいろなインバータが開発されている。最近、このサイリスタに代わってパワートランジスタやゲートターンオフサイリスタのような、素子自体がオフ機能を有する素子が用いられるようになった。この素子の採用により、次に述べる自励式サイリスタインバータには不可欠であった転流回路が不要となり、またスイッチング周波数が大幅に高められたことによりインバータの性能が向上した。インバータは複数個の半導体素子を組合わせて構成されており、素子間の電流を切換えることにより直流電力を交流電力に変換している。このとき、素子間の電流の移り変わる動作を転流(turn oftides)と呼ぶが、転流を確実に行わせるためには、転流前に通電していた素子をオフ状態にする必要がある。そのためには素子自身で電流をオフすることができる機能をもっているパワートランジスタやゲートターンオフサイリスタではゲートにオフ信号を加えればよいが、サイリスタは逆電圧を加える必要がある。インバータは他励式と自励式に分けられる。他励式(separatelyexcited method)とは、基本的に交流商用電源や負荷が発生する交流電圧を利用してサイリスタに逆電圧を加えて転流を行わせるインバータである。また、自励式(self-excited method)とは、サイリスタでは転流回路により、パワートランジスタやゲートターンオフサイリスタでは素子のもつ機能(自身で電流をオフする)によって転流できるインバータで自分自身制御機能を有する自立型のインバータである。太陽光発電システムでは、商用系統との連系を行うときには他励式、自励式いずれも適用可能であるが、系統と連系しない独立システムのときには自励式に限られる。b. 太陽光発電システム。 分類すでに述べたように太陽光発電システムの基本構成は図5・73に示すとおりである。このシステムは必要な電力を自前で供給するシステムであり独立システムと呼ばれている。システム例として、灯台や海上ブイの電源、山小屋の照明・動力用電源、無線中継所などの遠隔地での電源などが挙げられる。また、この独立システムの中にハイブリッドシステムがある。このシステムは太陽光発電システムと他の電源、例図5・74 太陽電池の分類5―178エネルギー供給システムえばディーゼル発電機、燃料電池などを組合わせ、各々の欠点を補完し合ってシステムを構成している。太陽光発電システムとディーゼル発電機とを組合わせたハイブリッドシステムの例では、ディーゼル発電機は太陽光発電システムをバックアップするように働き、蓄電池のみのシステムに比べ、蓄電池の容量を減らすことができる。また、システムの信頼性を向上することも可能である。独立システムに対して系統連系システムがある。このシステムは商用電源と連系して使用するシステムで、いくつかのタイプがあるが、完全連系システムは次のようである。このシステムは太陽電池アレイの出力が負荷電力を上回ったとき、余った電力商用電源系統へ供給され、逆に不足のときには商用電源系統から不足分を供給してもらうシステムである。わが国では平成5年4月に電気事業関係法令が整備され、一定の基準がクリアされていれば導入が可能となった。このシステムは構成が単純なため、運用によっては経済性の高いシステムであり、実用化が進められている。図5・75に連系型太陽光発電システムの構成図を、図5・76に太陽光発電システム(太陽電池パネルの屋根)を示す。. 系統連系型太陽光発電システムの特性すでに述べたように系統連系型太陽光発電システムの特徴は、太陽光発電システムからの発生電力は負荷に供給されるが、余剰量があればその電力を商用電源側に供給できることにある。例えば、夜間、雨天時には商用電源系統から電力を供給してもらい、太陽電池アレイからの供給電力が負荷電力を上回るとき商用電源系統に供給されることになる。個人住宅用の系統連系型太陽光発電システムについての特性は次のようである。年間の太陽電池アレイ発生電力量と負荷電力量の関係から太陽光発電システムの容量を決定するが、このシステムの容量は、商用電源へ逆潮流させる年間余剰電力量と商用電源から供給される年間不足電力量が等しくなる値とした。個人住宅用の年間消費電力量は約3800kWhであるとし、典型的な家庭での照明、冷蔵庫、空調、コンセントの全消費電力量に占める年間の各負荷割合をそれぞれ、約20,40,20,20%として求めた。その結果、システム定格と太陽電池アレイ面積はそれぞれ3.2kW,35mであり、年間で負荷変動の少ない冷蔵庫負荷への太陽光発電システムから期待できる供給率が最も大きく67%となった。また、この住宅負荷の年間の太陽依存率は40%程度で、商用電源系統との授受の割合は60%程度であることがわかる。このことは図5・75 系統連系型太陽光発電システム図5・76太陽光発電システム(アモルファスシリコン太陽電池パネルの屋根)商用電源網が信頼性の高い貯蔵機能を有しており、太陽光発電システムの出力を有効に利用することが可能であることがわかる。. ソーラカーソーラカーは太陽エネルギーを動力源とする車両であり、最近ソーラカーの性能を競うソーラカーレースが国内外で行われている。ソーラカーは太陽電池アレイ、蓄電池、モータ、車輪で構成される。車のスピードは太陽電池の発電量と走行抵抗とのバランスで決まる。この走行抵抗は車が走ることによって生じる空気抵抗、転がり摩擦による転がり抵抗、電気エネルギーを機械エネルギーに伝えるときに生じる伝達損失からなる。そのため、太陽電池アレイの発電量と走行抵抗のバランスをよくするため、発電量をできるだけ多くするよう、空気抵抗をできるだけ小さくするよう設計するなど工夫がなされている。実例の一つとして、太陽電池アレイの容量が1.2kW、蓄電池が酸化銀・亜鉛電池、モータとしてDCモータを採用したソーラカーがある。. 太陽・水素エネルギーシステム将来、水素エネルギーを中心とした水素エネルギー時代の到来を予想する向きもある。最近、水素製造に必要なエネルギーを太陽エネルギーに求めた、太陽エネルギーによる水素製造の研究が国内外で行われている。また、海外で太陽光発電システムを建設し、この電力で製造した水素と回収した二酸化炭素とを反応させアンモニア、アルコールなどを製造して、エネルギー消費地帯に輸送する地球規模の国際プロジェクトも提案されている。太陽光発電システムと水素をエネルギー源とする燃料電池を組合わせて、つねに電力を供給する太陽光発電・燃料電池ハイブリッドシステムの開発が進められている。このシステムでは、日中太陽電池アレイで発電した電力を負荷に供給しつつ、余った電力を利用して水素発生機により水を電気分解して水素を製造し貯蔵する。夕方から夜間にかけて、太陽電池アレイで電力を十分供給することが困難になったときに、水素を燃料電池に送り負荷に電力を供給することができる。このシステムの特徴は、まず、系統から電力供給を受けず負荷に電力を安定的に供給できること。次いで、燃料電池は発電と同時に熱エネルギーを発生するため、両者を有効に利用することにより、システム効率向上が期待できることである。. 宇宙太陽光発電システム宇宙太陽光発電システムの構想は、1968年グレイサーにより提案された壮大な発電所建設プロジェクトである。地球から約3万6000km離れた宇宙空間に発電プラントを建設して太陽エネルギーを電力に変換し、マイクロ波ビームで地上に伝送するシステムである。1970年代に検討されたシステムは1基500万kW の巨大な構想であった。エネルギー源としてほぼ永久的に利用可能であること、地球上の昼夜、天候に左右されないこと、技術的に実現性があること、技術の波及効果が期待できることなどにより、その後アメリカ、ヨーロッパ、わが国など可能性を求めて基礎研究が進められている。このように太陽光発電システムの適用は多種多様にわたっている。われわれの生活に大量に導入し同化させるには、小容量分散型のエネルギー源として実力を発揮させつつ、継続して技術的・経済的両面から開発を進める必要がある。
