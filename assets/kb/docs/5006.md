# 5・6・2 海流・潮流(ocean currents and tidal currents)

a. 海流エネルギー(ocean currents energy)海面に降り注ぐ太陽熱エネルギーは、地球表面において一様ではない。それは低緯度地域ほど大きく、高緯度地域になるにつれて小さくなる。したがって、海水には熱エネルギーによる力の不均衡(低緯度地域と高緯度地域間の海水温度差)が生じて対流現象が誘起される。さらに、地球の自転による慣性力が加わり、海洋に地球規模での海水の大循環が発生することになる。この海水の流れが海流である。世界のおもな海流分布を図5・87に示す。低緯度海域で太陽熱で加熱された海水(暖流)が高緯度海域へ移動し、氷などで冷却された高緯度海域の海水(寒流)と交換され、それが低緯度海域へ環流する、いわば太陽熱の平均化現象といえる。海流は一般に、北半球においては時計回りに、南半球では反時計回りに流れている。日本の南岸沿岸には北半球の二大海流である黒潮が流れている。黒潮の規模は、流量が3000~5000万m/s、流速が1~5ノット(0.5~2.5m/s)で、流域幅は最大120kmあるとされている。流れのもつ運動エネルギー(W)は、流速を(m/s)、流量を(m/s)、水車の掃過面積を(m)、海水密度をρ(=1025kg/m)とすると、次式で表される。上記の式より平均流速0.5m/s、幅250km、水深1000mの黒潮のもつエネルギーを概算すると、1600万kW current)海流からエネルギーを回収する試みは今までに実例はなく、構想の段階に終わっている。それは、流速が比較的遅く(2~3m/s)エネルギー密度が低いことと、陸地からかけ離れた沖合に巨大な海洋構造物を設置あるいは係留するなど、技術的、経済的に困難な問題が多いためである。海流のエネルギー変換システムは海水の運動エネルギー利用であるから、海水の流れのエネルギー変換を考えればよい。海流をエネルギー源として利用する場合、エネルギー輸送に有利であり、エネルギー利用の自由度が高い電気エネルギーへの変換システムすなわち発電システムが考えられる。発電する方法としては、水車と発電機を組合わせた方法と電磁流体発電による方法とがある。水車・発電機による方法としては、海流の運動エネルギーをいったん水車により回転エネルギーに変換し、発電機を駆動する方法が一般的に考えられる。今までに提案されている方法を図5・89に示す。c. generation)潮流発電では堤防を造らずに自然の流れから直接発電しようとする方法であるので、潮汐発電のような、湾の入口に貯水用のダムを造るための建設費などを考慮すると、コストの面で有利である。ただ、海流とは異なり潮流は約6時間ごとに流向が反転するために、発電装置が有効に作動するための方位制御機構あるいは流向に依存しない発電装置の開発が必要である。また、潮流の変動するエネルギーを有効に利用するするためには、効率のよい変換システム(電気エネルギー、熱エネルギー、機械エネルギーへの変換)あるいはエネルギー貯蔵システム(揚水、水素貯蔵、圧縮空気、フライホイールなど)を検討する必要がある。潮流発電では潮流の時間的流速変化の予測が可能、建設費がダム式より安価、環境インパクトが少ないなどの利点のため種々の試みがある。しかし、現在は実験段階で実用化の例はなく、実際の潮流による実験は、ダリウス型水車を用いたものがカナダと日本に一例ずつあり、クロスフロー型水車を用いたものが日本に一例あるだけである。ダリウス型水車は、構造が簡単、流れの方向が反転する潮流に対して水車はつねに一方向に回転し、翼の周速は流速の2~3倍などの特徴がある。日本大学では、3枚翼のダリウス型水車を試作して、1983~1988年の間、来島海峡の実際の潮流で3回にわたり実験を行った。海底設置型の発電装置を図5・91に示す。実験装置設置場所の最大流速は1.7m/sである。翼の材質をGFRP,CFRPに変更して軽量化した第3回めの実験では流速0.6m/sで自起動し、流速1.6m/sで発電電力2.3kW、変換効率は43.9%の結果が得られた。この装置の発電特性を図5・92に示す。徳島大学では、1986年に潮流の流れ方向が反転しても回転方向が同方向で、効率がよいクロスフロー水車を用い、鳴門海峡の実際の潮流で実験を行った。水車は、直径31.5cm、長さ1mであり、長さ5m、幅2.5mの浮体に取付けたものである。大鳴門橋の工事用桟橋にロープで固定して行い、最大800W の発電を行った。
