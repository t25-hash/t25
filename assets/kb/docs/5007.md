# 5・6・3 潮汐

> **章**: 第5章 再生可能エネルギー

地球上の全海域において、地球と月および太陽との相対的な天体運動が原因で、約半日周期または1日周期で、海面の上下運動が見られる。この周期的な海面の上下運動を潮汐という。潮汐現象はそ図5・91潮流発電装置図5・92発電特性の起こる原因が地球・月・太陽の相対的な天体運動であるから大体の予想が可能である。しかし、地形の影響により地域によって大きな差があり、海洋または海岸で、任意の地点の潮汐の予測には、前もって観測したデータが必要で、それに基づいて正確な予測が可能である。海面の高さを潮位というが、潮位がいちばん高くなったときを高潮(high water)あるいは満潮(high tide)、いちばん低くなったときを低潮(low water)あるいは干潮(low tide)という。潮位の満潮時や干潮時の高さは日によって異なり、満潮と干潮の高さの差を潮位差(tidal range)と呼んでいる。潮位は時間に対して正弦波状に変化しており満潮、干潮が1日に二度ある半日周期(semidiurnal tide)の周期は約12時間25分で、一度しかない日周期(diurnal tide)の周期は約24時間50分である。このような、1日の相次ぐ満潮、干潮の高さや時間間隔の違いを日潮不等(diur-nal inequality)と呼んでいる。図5・93は海上保安庁による愛媛県今治港の潮位と来島海峡の潮流曲線を示す。また、月のうちでは新月(朔)と満月(望)の2回の満潮がいちばん高く大潮(spring tide)という。新月と満月の中間(上弦あるいは下弦)には月のうちでいちばん潮位の低いときがあり、小潮(neap tide)という。この、大潮、小潮も年間では変化し、春秋の彼岸のころに年間でいちばん高い分点潮あるいは彼岸潮がある。図a. 潮汐発電(tidal power
