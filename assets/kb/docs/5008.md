# 5・7・1 バイオマス(biomass)とは

> **章**: 第5章 再生可能エネルギー
> **節**: 5・7 バイオマス

地球上の生物は植物,動物,菌,原生動物,モネラ(細菌)から構成されており,これら生物体の量をバイオマス(biomass)という.しかし,エネルギー源や工業原料の対象としてのバイオマスは,一般には,光合成により蓄積された植物体の重量あるいはエネルギー量を指す.地球上の植物体は,光合成により水とCOから有機物を合成し,葉や根,茎や実などとして蓄積する.光合成により植物が1年間に蓄積した有機物量を純一次生産量といい,理論上は,再生可能な資源あるいはCOニュートラルな資源と考えられる.地球上のバイオマスの総量(植物体)は乾燥重量で1800Gt,純生産量は170Gtといわれている.表5・13に,地球上の異なる生態系におけるバイオマスの蓄積量と純一次生産量を示す.熱帯多雨林をはじめとする森林がバイオマスの蓄積量と生産に大きな役割を果たしている.資源としてバイオマスを分類すると,森林資源の産物である木表5・13 地球上の植物バイオマスの純一次生産量と蓄積量面積×10km純一次生産量kg/m/年蓄積量kg/m熱帯多雨林17.02.2熱帯季節林7.51.6温帯林12.01.3亜寒帯針葉樹林12.00.8疎林・低木林8.50.7サバンナ15.00.9温帯草原9.00.61.6ツンドラ・高山8.00.140.6耕地14.00.65沼沢地2.0外洋332.00.1250.003湧昇流海域0.40.50.02大陸棚26.60.360.01藻場・さんご礁0.62.5入江1.41.5図5・99バイオマス資源の変換利用場面材,海で生育する海藻類,人類の食料になる作物類,アルコールやテルペン系炭化水素などを得るためのさとうきび,とうもろこし,ユーカリなどのエネルギー植物,家庭や産業からの有機廃棄物,農産物残渣や家畜ふん尿,などさまざまなものがある.これらバイオマスの資源としての利用場面を図5・99に示す.バイオマスを資源として利用する場合のおもな利点と欠点は以下のようにまとめられる.おもな利点・太陽エネルギーの変換貯蔵であり,再生産が可能である.・微生物により分解される.・燃焼による環境負荷が小さい.・燃焼後に残る灰は資源としてリサイクルできる.おもな欠点・広い範囲に拡散しており,効率的な利用には集積が必要になる.・化学的,物理的に複雑な構造をしている.・利用に際しては,脱水,乾燥,粉砕などの前処理が必要になる.・単位重量当たりの発熱量が小さい.
