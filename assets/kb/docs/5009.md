# 5・7・2 森林のバイオマス

> **章**: 第5章 再生可能エネルギー
> **節**: 5・7 バイオマス

世界の森林におけるバイオマスの蓄積量は330Gmであると推定されている.世界の木材生産量は3.25Gmで,そのうち1.68Gmが薪炭用,1.57Gmが製材や製紙用パルプに利用されている.わが国の森林面積は約2.5×10haで,森林バイオマスの蓄積量は約3.5Gmである.面積の内訳は,人工林が41%,天然林が53%,竹林1%,ほかである.わが国の森林由来の未利用バイオマスの資源量は約43Mt/年になる.43Mtの森林バイオマスの有するエネルギーは,石油換算で約1.8Mtと見積もられる.森林バイオマスや木質系廃棄物から,燃焼や発酵などによりエネルギーを得る工程を効率的に進めるには前処理が必要になる.木材を破砕処理してチップ化し,減容化して運搬性と加工性を向上させる前処理用機械が開発されている.代表的な方式としては,ベルトコンベヤで廃棄物を破砕室に供給し,破砕室では回転刃と固定刃により所定の粒度になるまで繰返し処理する.処理がすんだ破砕物のチップは磁気選別機によりくぎや鉄くずが取り除かれる.なお,チップの粒度は利用目的に合わせて変えられる.
