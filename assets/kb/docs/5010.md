# 5・7・3 食料と有機廃棄物

> **章**: 第5章 再生可能エネルギー
> **節**: 5・7 バイオマス

20世紀後半の食料用の穀物生産量は2.05Gtで,約61億人の地球上の人間を養っている.わが国の穀物生産量は約19.6Mtである.わが国における農業関連の有機廃棄物の発生量を表5・14に示す.表5・14には農業生産に伴う廃棄物とともに,加工や消費の過程における有機廃棄物およびし尿や下水汚泥の発生量も併せて示してある.有機廃棄物の中では家畜ふん尿の量が最も多い.有機廃棄物の総量は約280Mt/年であり,廃棄物総量の約60%を占める.有機廃棄物中の窒素分の総量は約1.3Mt/年になり,コンポストにして全農地に還元できる容量を超えていると試算されている.
