# 6・1・4 排熱利用発電

> **章**: 第6章 未利用エネルギー
> **節**: 6・1 概説

図6・1のヒートカスケーディングのように化石燃料の燃焼熱から発電するだけではなく,カスケーディングの途中から発電することも可能である.一つの方法は,比較的高温の工場排熱などから低沸点媒体を用いて熱回収し発電する技術である.従来はフロンタービンなどが用いられていたが,近年アンモニア・水混合冷媒を用いた発電技術が実際に運用されている.アンモニア(ammonia)は低沸点媒体であるのみならず,天然に存在する自然作動媒体であり,地球温暖化やオゾン層破壊を引き起こさない点で好ましい冷媒の一つである.第6章ではこのカリーナサイクル(Kalina cycle)と呼ばれる技術について解説する.一方,高温熱から直接電力を得る方法もある.小規模な例としては熱電対と呼ばれる温度センサとして利用されている現象を,熱を電力に変換する素子(熱電素子,thermoelectric device)として大規模化したものである.一般に高炉のような高温プロセスでは反応炉表面から躯体放熱により熱が環境中に逃げてしまう.そのような炉表面に熱電素子を張り付けることによって放出する熱を電力に変換することができる.冷媒を用いて熱を回収するには冷媒の流路,タービン,復液器が必要となり設備が大型化するが,熱電素子では一体化されていること,可動部がないことなどの長所を有している.省エネルギーに向けて今後の展開が期待される技術と位置付けられる.
