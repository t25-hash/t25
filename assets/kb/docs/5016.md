# 5・2・4 重油の品質

> **章**: 第5章 舶用機関
> **節**: 5・2 船舶燃料

重油は用途に応じて動粘度，残留炭素，硫黄分および流動点など
を調整して製品にする必要があり，また，挟雑物や水分などの混入
を避ける必要もあることから，品質に十分な注意が払われている．
舶用ディーゼル機関等の内燃機関用のA重油の品質は，次の項
目が重要となる．
(１) 着火性
特に小型のディーゼル機関は高回転のため，着火しやすく良好な
燃焼を得るために，ある程度のセタン指数が必要となる．
(２) 動粘度
重油の分類のみならず，用途の面からも重要な性状である．例え
ば，燃料をスムーズに供給し安定した燃焼を起こさせるために，燃
料を噴射する際の状態が重要になる．エンジンのタイプによって適
正な動粘度が要望されている．
(３) 蒸留性状
燃焼過程において燃料の気化は重要な因子であり，蒸留性状によ
り気化の度合いは異なる．例えば，沸点の高い留分が多い場合は，
気化が進みにくいため，効率良く燃焼させるためには，燃料の加温
などの対策が必要となる場合がある．一方，低沸点分が多い場合は
燃焼場での燃料拡散が大きくなる．このため，空気などの供給が不
十分になると，燃焼が不安定になることも考えられる．
(４) 低温流動性
軽油の場合と同様に，温度が降下するにつれてワックスの結晶が
析出し，さらには析出量の増加に伴いワックス結晶の凝集が起こ
る．これらの凝集物が燃料フィルタに堆積し，閉そくに至る．この
温度を低温フィルタ目づまり点（CFPP:cold filter plugging
point ）という．この対策としては，貯蔵設備の加温や流動点降下
剤の添加などが挙げられる．
(５) 硫黄分
燃料中の硫黄分は燃焼時にSOになり，燃焼ガスとして排出さ
れるため，燃焼機器の腐食や環境規制の観点から重要視されてい
る．
船舶燃料用のC重油の品質としては，下記の項目が重要となる．
(１) 着火性
大型のディーゼル機関は低速回転のため，着火性はA重油ほど
問題にはならない．C重油のセタン価を測定することはできないた
め，密度と動粘度から計算で求めるCCAI （calculated carbon
aromaticity index）が燃料の着火性を示す指数としてよく利用さ
れている．
(２) 動粘度
流体力学上では粘性抵抗に当たり，ポンプ送油の際の大きな要素
でもあるため，最も重要な性状である．C重油はA重油よりも粘
度が高いため，加熱により適当な動粘度に調整し使用する．一般に
移送・貯蔵時は40～70℃に加熱し，燃料噴射時にはさらに加熱し
ている．
(３)硫黄分
A重油の場合と同様に燃焼の際にSOとなるが，その一部は
SOとなり，それが燃焼ガスの露点以下の温度ではガス中の水分と
反応し硫酸となる．硫酸は金属を著しく腐食させるので，硫酸生成
を防止するために燃焼ガスの温度を露点以上に保つことが必要であ
る．なお，IMO（International Marine Organization，国際海事
機関）では「船舶からの大気汚染防止に関する規則」の中で，燃料
油中の硫黄分の上限を4.5%としており，SO排出規制海域（バル
ト海およびIMOが指定する海域）ではさらに1.5%以下，もしく
は排出ガス洗浄装置の設置を必要としている．
(４)固形物
C重油の主成分は残油のため，原油に含まれている泥分，鉄錆な
どの金属分が濃縮される傾向にある．特に接触分解装置のFCC
( fluid catalytic cracking）触媒は硬質のシリカアルミナ（SiO・
AlO ）でFCCの残油であるスラリー油に混入されていることが
ある．FCC触媒など硬い固形物を含む燃料がディーゼル機関に使
用されると，ピストンやシリンダのしゅう動面を激しく摩耗するこ
とが報告されている．
(５)安定性
貯蔵中に泥状，もしくは粉状の固形物が発生，沈殿することがあ
る．これをスラッジと呼び，スラッジの生成しにくさを安定性と呼
ぶ．スラッジはフィルタ目づまり，燃焼不良，エンジンの損傷など
のトラブルの原因となる．スラッジの要因はさまざまであるが，ア
スファルティックスラッジ，ワックス析出によるスラッジ，水分お
よび挟雑物によるスラッジ，微生物によるスラッジなどが挙げられ
る．
(ａ)アスファルティックスラッジ：C重油にはアスファルトの
主成分の一種であるアスファルテンやマルテン（油分），レジン分
（樹脂分）などが含まれている．アスファルテンはレジン分の働き
によりマルテン中に安定に分散しているが，これらが他の油種と混
和されるとき，混和性が悪いとアスファルテンが互いに凝集・成長
し，アスファルティックスラッジが生成する．
(ｂ)ワックス析出によるスラッジ：重油には高沸点パラフィン
系成分であるワックスが含まれている．温度低下によりワックスが
析出，結晶化する．C重油では加温されているため問題は少ない
が，冬季には特に補油後の流動性に注意する必要がある．
(ｃ)水分および挟雑物によるスラッジ：重油は精製直後には水
分は含まれないが，貯蔵や輸送を通して水分が混入される．水分は
重油と水-重油エマルジョンを形成するが，水分量が多くなると水-
重油エマルジョンスラッジが生成する．それ以外に，重油には無機
物や有機物の挟雑物の混入によるスラッジも生成する．
(ｄ)微生物によるスラッジ：適当な温度と水分およびエネルギ
ー源がそろうと，微生物が繁殖しやすい条件となり，微生物による
スラッジが生成する．C重油は加温貯蔵されるため微生物の成長が
しにくく，トラブルは少ないが，A重油では常温で貯蔵されるた
め，条件がそろえば繁殖可能となり発生する例が多い．
