# 6・2・3 リパワリング(repowering)

> **章**: 第6章 未利用エネルギー
> **節**: 6・16 に示す。

廃棄物発電プラントにおけるリパワリングは、ガスタービンの導入により発電量の増加と総合発電効率を向上させる手法である。国内では、スーパーごみ発電(wasted combined power generation)と呼称されているシステムが、リパワリングの一例として実用化されている。ヨーロッパでは、発電と熱供給の両面でのパワーアップ効果をねらった例もある。廃棄物の焼却により発生する排ガスと飛灰には、塩素(Cl)、アルカリ金属(Na,K 等)、重金属(Zn,Pb等)が含まれている。これらの作用で、ボイラ水管のメタル温度が、付着飛灰の溶融温度に近い300~400°C以上の温度領域では厳しい腐食環境を形成する。廃棄物焼却炉付設ボイラから発生する蒸気温度は、腐食温度領域を避け低く設定されてきた。しかし、廃棄物発電効率を向上させる社会的要求により、蒸気温度の高温化が必要とされている。高温化を目的として、次の方策が実用化されている。(1) ボイラ出口蒸気を腐食の少ない温度領域とし、ガスタービン排ガスにより過熱する。(2) ボイラ出口蒸気を500°C程度まで高める。腐食対策としてボイラ水管、スーパーヒータ管に高合金(Cr,Mo,Ni含有)を用いる。上記(1)は、ガスタービン排気ガスでボイラ発生蒸気を過熱し、蒸気タービン発電効率を向上させるシステムであり、スーパーごみ発電として知られる。また、上記(2)のように、ボイラ発生蒸気温度がガスタービン排ガス温度と同等以上の場合は、ガスタービン排ガスによるボイラ発生蒸気の過熱は不可能である。しかし、さらに発電効率を向上させる方法として、高圧蒸気タービン出口の蒸気をガスタービン排ガスで再過熱し、低圧蒸気タービンでの発電量を増大させるリパワリングシステムの適用が可能である。リパワリングシステムとして、基本的には、通常の廃棄物発電方式で蒸気タービン以降の設備を共用するガスタービン併置方式と、これに加え、過熱器または再熱器をガスタービン排ガスで加熱する5―19 8エネルギー供給システムボイラ蒸気条件ST 蒸気条件廃棄物焼却量国名名称温度圧力温度圧力t/d×炉数°CMPa°CMPaデンマークネスティベッド108×36.86.5GT 併置方式スウェーデンリンチェピング165×1288×21.71.57不明GT 複合方式注GT:ガスタービン、ST:スチームタービン図6・18 ガスタービン併置方式ガスタービン複合方式に分類される。それぞれの構成例を以下に示す。a. ガスタービン併置方式(図6・18)ガスタービン排ガスボイラで過熱蒸気を発生させ、蒸気タービンに供給する。廃棄物焼却炉付設ボイラから発生した蒸気は、別の配管で蒸気タービンに供給される。蒸気タービンには、混気(混圧)タービンを用い、高圧側にガスタービン排ガスボイラで発生させた蒸気が、その下流に焼却炉ボイラで発生させた蒸気が導かれる。b. ガスタービン複合方式(過熱器型、図6・19)廃棄物焼却炉付設ボイラから発生する蒸気温度が、ガスタービン排ガス温度よりも低い場合に適用される。ガスタービン排ガスボイラに、廃棄物焼却炉付設ボイラから発生した、過熱または飽和蒸気の過熱器が設けられる。廃棄物燃焼ガスによるボイラ水管の高温腐食を回避できるので、ボイラ水管材料に低コストの炭素鋼を使用できる。スーパーごみ発電は本方式による。c. ガスタービン複合方式(再熱器型、図6・20)廃棄物焼却炉付設ボイラから発生する蒸気温度が、ガスタービン排ガス温度と同程度か高い場合に適用される。ガスタービン排ガスボイラに、高圧蒸気タービン出口蒸気を導き、再熱器により再過熱し、低圧蒸気タービンに導く。廃棄物燃焼ガスによるボイラ水管の高温腐食対策として、ボイラ水管材料にインコネル等の高コストの高合金を用いる必要がある。国内外の、廃棄物発電リパワリング施設の概要を表6・3に示す。表6・3 廃棄物発電リパワリング施設の概要GT排ガス発電設備容量発電効率稼働時期リパワリング方式温度STGT合計°C年MWMWMW%ホルセンス120×24.74.512.534.5不明GT 併置方式ゾンデルボルグ192×16.26.0不明不明GT 併置方式オランダメルデック636×39.0GT 複合方式イタリアベローナ278×25.25.228.5GT 複合方式群馬県高浜発電所150×31.961.5710.517.534.3GT 複合方式日本堺市クリーンセンター230×22.792.2512.44.116.521.1GT 複合方式北九州市新皇后崎工場270×32.32.0628.38.036.326.5GT 複合方式図6・19ガスタービン複合方式(過熱器型)図6・20 ガスタービン複合方式(再熱器型)
