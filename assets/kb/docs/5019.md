# 6・3・1 LNG

> **章**: 第6章 未利用エネルギー
> **節**: 6・3 LNG 冷熱発電

液化天然ガス(LNG : liquefied natural gas)は大気圧のもとでは沸点が約-160°Cの超低温の液体である。その成分は、メタンが90%前後であり、ほかにエタン、プロパン、ブタン、ペンタンが含まれる。世界のLNG 生産量は、年間約8000万tであり、日本は、約5400万tを輸入している。日本に輸入されるほぼ70%のLNGが、火力発電所の燃料に利用され、残りは都市ガスとして利用されている。LNG は、石油・石炭・原子力に次ぐ基幹エネルギーとして位置付けられており、発電設備の電源別構成比率の約24%を占める。他の化石燃料に比べ炭酸ガスの排出量が少なく、またそのクリーン性から今後も使用量の増加が見込まれている。
