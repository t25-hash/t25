# 5・3・4 高効率化

> **章**: 第5章 舶用機関
> **節**: 5・3 大型低速ディーゼル機関（2ストローク）

低質油が焚けて低燃費であることが2ストローク低速ディーゼル
機関の最大の特徴である．現在，機関単独で熱効率が50%を超え
ている原動機はほかにない．図5･43 に機関出力率の変遷を，図
5･44 に燃料消費率の変遷を示す．燃費率低減のため，シリンダ
内爆発圧力の上昇による燃焼サイクルの改善，過給機の高効率化，
燃料噴射系の最適化，掃気効率の高効率化などが実施されてきた．
今後とも燃料経済性は大切であるが，船の安全な運航が大前提であ
るため，燃焼サイクル改善の有力手段であるシリンダ内爆発圧力は
抑制せざるを得ない．特に海洋汚染防止上，排出ガス中のNO削
減には燃焼温度抑制など従来の燃費低減手法とは反対の燃焼制御が
