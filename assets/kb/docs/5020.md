# 6・3・2 冷熱発電の効率評価

> **章**: 第6章 未利用エネルギー
> **節**: 6・3 LNG 冷熱発電

LNG のもつ冷熱エネルギーは、0°Cを基準にとる熱力学第一法則では負となる。そのため、冷熱利用の評価では、エンタルピーでなくエクセルギーを用いる。エクセルギー(exergy)とは、カルノーサイクル(Carnot cycle)で得られる仕事量Work であり、カルノー効率に利用できる単位熱量(kJ/kg)をかけ、さらに利用できる量をかけて表す。エクセルギー評価では、熱交換器や回転機、さらに燃焼に伴う損失を個別に評価するため、プロセスの各点のエンタルピーとエントロピーを求める必要がある。また燃料のもつ化学エクセルギーも式(6・2)で計算できる。そのため化学エクセルギーの値は、外界温度に応じて変わる。これらの値やプロセス内の各機器のエクセルギー損失は、プロセスシミュレータとリンクさせ、自動的に計算させることができ、LNG の燃料化学エクセルギーは、14400kWh/LNG-tであり、また常温までの冷熱エクセルギーは250kWh/LNG-tとなる。この冷熱エクセルギーは、大気圧-160°CのLNG を外界20°Cの温度に気化させる間でカルノーサイクルにより得られる仕事量である。
