# 6・3・3 冷熱利用

> **章**: 第6章 未利用エネルギー
> **節**: 6・3 LNG 冷熱発電

冷凍機で冷熱を発生させる場合、温度レベルが低くなるほど冷熱発生のための仕事量は増加する。したがって、LNG の冷熱を利用する場合、できるだけ低い温度レベルで利用するほど高い価値がある。LNG は、通常ポンプで加圧され高い圧力で気化される。そのため、250kWh/LNG-tの冷熱エクセルギーは、例えば4MPaに加圧されると、それぞれ125kWh/LNG-tの冷熱と圧力エクセルギーに変換される。表6・4に日本のLNG 冷熱利用状況を示す。空気分離設備では、循環する窒素ガスをLNG により-150°C近くに冷却するため、冷熱エクセルギーを約80%と高い効率で回収できる。しかし、空気分離設備は、24時間の連続運転や製品窒素・酸素の販売に制約があり、LNG 冷熱を大量に利用できない。一方、負荷変動に対応できる冷熱発電は、第二次石油危機後の省エネルギー気運の中で脚光を浴び、相次いで都市ガス事業者や電力事業者により建設され、現在、LNG 基地において、年間850万tのLNGが利用されている。しかしLNG の気化熱源は、大規模LNG 基地では海水であり、地方都市にタンクローリーで輸送されるLNG表6・4 日本のLNG 冷熱利用状況設備名温度レベル°C年間利用量万t冷熱エクセルギー回収率%冷熱発電-120~-4020~34空気分離-150BOG 再液化-120液化炭酸-55冷蔵倉庫-60~-2020~505―19 9は、すべて大気で気化されている。未利用のLNG 量は、現在年間3500万tもあり、その冷熱は海水や大気に捨てられている。この冷熱を積極的に利用することは、省エネルギーや環境問題から非常に意義がある。
