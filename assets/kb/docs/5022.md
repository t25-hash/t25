# 6・3・4 冷熱発電(cold power generation)の方式

> **節**: 6・3 LNG 冷熱発電

図6・21にLNG 冷熱発電方式の分類を示す。サイクルの種類として、ランキン、天然ガス直接膨張(直膨)、ランキン直膨組合せ、およびブレイトンサイクルに大別される。ランキンサイクル(Rankine cycle)は、作動流体によって単一冷媒と混合冷媒に分類される。図6・21中の*印は、現在稼働中の方式である。図6・22に作動媒体がプロパンのランキン直膨組合せ方式のプロセスフローを示す。通常LNG は、6MPa程度にポンプで加圧され、タービン出口温度約-40°Cのプロパンガスによりプロパン凝縮器で気化され、冷熱エクセルギーが回収される。-50°C以上に気化されたLNG は、海水熱源の天然ガス加熱器により0°C以上に加温され、天然ガス膨張タービンで圧力エクセルギーが回収される。膨張により温度が下がった天然ガスは、海水で常温まで加温される。ガスの送出圧力が低い場合には、天然ガス膨張タービンでさらに圧力エネルギーが回収される。ランキンサイクル方式の冷熱エクセルギーの回収率は、冷媒の凝縮温度とその冷媒の再生の有無により大きく変わる。天然ガス循環直膨や炭化水素系混合冷媒ランキンでは、図ンプ吐出しの過冷却液をプロパン凝縮器においてLNG 冷熱と組合わせ、タービンを出た冷媒ガスを凝縮させ、循環量を増やし、効率よく冷熱を回収できる。これらの設備では、海水を熱源とするオープンラック式、シェルアンドチューブ式あるいはプレートフィン式の気化器が用いられる。また図6・23にガスタービンの排熱を熱源として作動媒体に窒素ガスを用いるブレイトンサイクル(Brayton cycle)を示す。ガスタービンの排熱で循環する圧縮窒素を550°Cまで加熱し、窒素タービンで動力を回収する。タービンを出た250°Cの窒素ガスは冷却水で冷やされた後、LNG で-100°C前後に多段冷却され窒素ガスの圧縮動力を低減する。このサイクルは、海水が低温のため熱源として使えないヨーロッパの受入基地で検討されたが、実施されていない。わが国でもLNG の気器に対する信頼性や余剰電力の問題から採用されていない。図6・21 LNG 冷熱発電方式の分類図6・22 ランキン直接膨張組合せ方式のプロセスフロー5―200エネルギー供給システム図6・23 窒素ガスブレイトンサイクルのプロセスフロー表6・5 稼働中のLNG 冷熱発電設備の一覧基地名称運転開始年出力MW方式LNGt/h送出圧力MPa泉北第21.45R3.0知多共同1.0R1.4泉北第26.0R/NG1.7北九州9.4R/NG0.9知多7.2R/NG0.9知多7.2R/NG0.9日本海5.6NG0.9根岸4.0MFR2.4東扇島3.3NG0.8姫路2.8R4.0東扇島8.8NG0.4泉北第12.4NG0.7四日市7.0R/NG0.9東扇島8.8NG0.4岩崎橋(供)1.15NG0.2姫路1.5NG0.7酉島(供)5.0NG0.7
