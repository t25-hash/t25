# 5・4・1 概要

> **章**: 第5章 舶用機関
> **節**: 5・44 ストローク低速ディーゼル機関

4ストローク低速ディーゼル機関は，わが国において歴史的に内
航船用，漁船用ディーゼル機関に始まり，古くから独特の発達をし
てきた．特に他分野の4ストロークディーゼル機関と比べ，行程-
内径比が大きいこと，すなわち長行程（long stroke）であること
は海外において比較対象となる同種の機関は存在せず，オリジナル
性が高い．機関の開発はその時代に建造される船舶の動向に合わせ
て行われ，現在に至るまで内航貨物船，内航コンテナ船，内航タン
カー，内航フェリー，作業船，漁業練習船，調査船ならびに遠洋・
近海漁船等の主推進機関（main propulsion engine）として幅広く
使用され続けている．
シリンダ径は240～500mm，定格出力450～3600kW，定格回
転数200～450min程度の機関が実用化されている．正味平均有
効圧力は1.9～2.2MPa，行程-内径比は約2.0，シリンダ内最大圧
力は13～15MPa程度の機関が多い．図5･50にシリンダ径330
mm，出力1600kWの機関断面を示す．構造は堅牢，簡素である
ことを特徴とするが，これは機関トラブル要因を最小限とし，容易
に船内での保守整備ができることを意図した設計配慮によるもので
ある．
同一のシリンダ径を有する4ストローク中・高速ディーゼル機関
との比較では，4ストローク低速ディーゼル機関のほうが行程は長
く，機関回転数が低い．したがって，シリンダ内に噴射した燃料を
時間をかけて燃焼させることができ，C重油等の低質燃料油
（heavy fuel oil ，重質燃料油）の利用に有利である．また，ピスト
ンの膨張仕事が大きく本質的に熱効率に優れる．さらに低速機関

a.ピストン（piston) ピストンの構造は大別して二分割型と
一体型に分類される．図5･52(ａ)に二分割構造のピストンを示す．
二分割型の本体は，ピストンクラウン（上部）とピストンスカート
（下部）で構成され，シリンダ内圧力によるピストンクラウン外周
部の変位をピストンスカートに伝えない合理的な構造である．ま
た，両部品が担う機能に適した材料の使い分けが可能で，ピストン
クラウンの材質は，シリンダ内の燃焼ガス圧力，熱負荷に対して機
械的性質，耐熱性の高い鍛造鋼が使われ，ピストンスカートにはシ
リンダライナとのしゅう動特性に優れた鋳鉄材料が使われる．なお
図5･52(ａ)のピストンスカートは，鋳鉄材料の中でも機械的性質
の高い球状黒鉛鋳鉄を使って薄肉・軽量化した事例である．
一体型ピストンは，正味平均有効圧力が現在よりずっと低かった
時代に多くの機関で多用されたが，当時のものは厚肉で重量軽減の
余地を残すものであった．現在の一体型のピストンは，鋳造技術の
進歩により，引張強さ600N/ mm以上の機械的性質をもつ球状黒
鉛鋳鉄を使用して図5･52(ｂ)に示すように薄肉化しており，二分
割型よりも部品点数の少ない組立容易な軽量化ピストンが製造さ
れ，回転数の比較的高い機関に用いて慣性力を軽減するのに優位で
ある．ただし，シリンダ内圧力の高い機関に適用する場合，ピスト
ン胴部の外周変形に十分配慮した設計が必要である．
b.シリンダヘッド（cylinder head) シリンダヘッドはシリ
ンダライナ，ピストンとともに燃焼室を構成する部品であり，機械
的・熱的負荷に耐える構造としなければならない．シリンダヘッド
には，燃料噴射弁，排気弁，吸気弁，起動弁，指圧器弁，安全弁が
装備される．これら諸弁の装着部のほか，吸排気孔，冷却水室が造
形されており，構造は複雑で一般的に鋳鉄製である．材質は片状黒
鉛鋳鉄，より強度を要する場合はバーミキュラ鋳鉄や球状黒鉛鋳鉄
が使われる．図5･53にシリンダヘッドの断面図を示す．ガス圧力
の作用する燃焼室壁の剛性を高めるために二重壁構造［図5･
53(ａ)］としたもの，中心部から斜めにリブを配した構造［図5･
53(ｂ)］のものなどがある．
c .シリンダライナ（cylinder liner ) シリンダライナは燃焼
室を形成する部品である．材質は鋳鉄が使用され，機械的性質を高
めるために少量のニッケル，クロム，モリブデン，リン，ボロン等

ゼル機関と比較し長行程であるため，機関全高が高い分重量が増
し，また回転数が低いので出力は低く抑えられる．
