# 6・3・5 冷熱発電の現状

> **章**: 第6章 未利用エネルギー
> **節**: 6・22 のプロパン凝縮器で作動冷媒を-120°C以下に冷却し、冷媒ポ

表6・5に稼働中のLNG 冷熱発電設備の一覧を示す。表6・5中の方式のR,R/NG およびMFR は、それぞれランキン、ランキン天然ガス直膨組合せ、および炭化水素系混合冷媒(multi-compo-nent refrigerant)ランキンを示す。LNG 1t当たり発電出力は、発電方式やガスの送出圧力に応じ、20~60kWhである。単一媒体ランキンの発電出力は、海水温度や冷媒の種類あるいは凝縮温度により異なるが20~30kWhである。炭化水素系混合冷媒ランキンの発電出力は、40kWhである。ランキンサイクルによるLNG 冷熱エクセルギーの回収率は、表6・4に示すように20~34%となる。一方、天然ガス直接膨張(national gas direct expansion cycle)のLNG 1t当たり発電出力は、ガスの送出圧力に応じ大きく変わるが、一般に利用できる圧力エクセルギーの73%を電力として回収できる。表6・5からわかるように1991年以降、エネルギー価格の安定、高い建設費、発電規模が小さい、さらに火力発電所が蒸気タービンからガスタービンコンバインドにシフトしたため、LNG 基地における冷熱発電の新設は、ほとんど進んでいない。今後、従来型のLNG 火力発電設備がガスタービンコンバインドにリプレースされると、送出圧力の低い冷熱発電は順次停止されるだろう。一方、岩崎橋(供)や酉島(供)のように都市ガスの供給所において、ガスの圧力エクセルギーを電力として回収する動きが、電力の自由化の中で今後注目されるであろう。
