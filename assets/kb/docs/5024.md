# 6・3・6 冷熱発電の機器構成

> **章**: 第6章 未利用エネルギー
> **節**: 6・22 のプロパン凝縮器で作動冷媒を-120°C以下に冷却し、冷媒ポ

代表的なLNG 冷熱発電方式のランキンサイクルの機器構成を図の全景を示す。主たる機器は、右からプロパン蒸発器、凝縮器、ポンプ、タービンおよび発電機で構成される。ランキンサイクル設計上のポイントを(a)~(c)にまとめる。図6・24 ランキンサイクルの機器構成図6・25 出力1450kW 冷熱発電商用機の全景図6・26 LNG 流量に対する発電出力特性(a) 凝縮器は、蒸発器より高位置に設置され、サイホン効果を利用して、気化器運転で、ポンプが停止しても、各バイパス弁が全開となり、蒸発器と凝縮器の液面差によりプロパンが循環し、蒸発器で海水から得た熱を凝縮器でLNG に伝える。(b) 凝縮器伝熱管下部の飽和器により、凝縮した過冷却プロパン液とタービン排気ガスを気液接触させ、凝縮液を飽和温度に近づけてプロパン循環量を増やし発電出力を向上できる。(c) プロパンポンプは、モータとポンプが一体となったサブマージドタイプであり、常温から-40°Cに減圧する過程で発生するガスを凝縮器に戻し、キャビテーションを生じることなく運転できる。
