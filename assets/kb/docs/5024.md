# 5・4・3 高効率化，排気対策，振動騒音対策

> **章**: 第5章 舶用機関
> **節**: 5・44 ストローク低速ディーゼル機関

a.高効率化 1970年代の石油危機を契機に省エネルギーが最
大の課題となり，4ストローク低速ディーゼル機関では，燃焼最大
圧力の増大，低回転-長行程化，燃料噴射圧高圧化，機関の高温冷
却化等を実施し，燃料消費率10 %以上の低減を見た．またこれら
の要素技術の変革は，C重油の燃焼改善にも寄与し，安価な低質油
の利用率を高めた．さらに軸系では，プロペラ推進効率を向上させ
る目的で大口径プロペラを低回転で運用する方式が広まり，この軸
系との組合せで，4ストローク低速ディーゼル機関は，減速機を必
要としないプロペラ直結型推進システムとすることが可能であり，
機械的損失を最小限とした簡素なシステム構成で船舶の経済運航に
貢献している．
b.排気対策 舶用から排出される大気汚染物質を低減する目
的で1997年，国際海事機関（IMO）は，従来の海洋汚染防止条約
（MARPOL73/78条約）に附属書Ⅵとして「船舶からの大気汚染
防止のための規則」を追加するための議定書を採択した．この附属
書Ⅵは16項目の規則からなり，この中で舶用ディーゼル機関にと
って特に関連深いものは第13規則「窒素酸化物（nitrogen
oxides ，NO ）」の規制である．これは第一段階としてNOの排
出率を従来機関と比較し約30 %低減することを目標としたもので
あり，5年ごとに規制が見直され強化される．なお，この条約は発
効要件が整い次第実施されるが，窒素酸化物については2000年に
さかのぼって規制が適用される．
上記の国際条約により，これまで規制のなかった舶用ディーゼル
機関のNO低減対策は急速に進展し，4ストローク低速ディーゼ
ル機関においても条約適合型機関の製造比率が2000年以降急速に
高まった．図5･62に条約の規制値を示し，対策前後の4ストロー
ク低速ディーゼル機関の排出率をプロットした．
現在4ストローク低速ディーゼル機関で実用化されているNO
低減手法は，
(１)燃料噴射時期遅延
(２)初期噴射率を抑制したプロフィルをもつ燃料カム
(３)燃料噴射弁ノズル孔の多孔化，小径化
(４)給気温度の低減
などである．これらの手法を幾通りか組合わせてNOを低減させ
ている機関が多いが，この中で特に(１)の燃料噴射時期遅延は機関
の運転全領域にわたって燃料消費率，すすの増加という悪影響をも
たらす．これを抑制する目的で圧縮比を上げてサイクル効率を改善
したり，給気圧力を上げて燃焼用空気量を増加させて燃焼改善を行
っている機関が多く，燃料消費率を同等ないしは数%程度の増加に
とどめている．従来型機関とこれにNO低減対策を施した機関の
シリンダ内圧力を比較すると図5･63のようになる．対策型機関
［図5･63(ｂ)］では，燃料噴射時期遅延により着火点はTDCに近
づき，圧縮比，給気圧力を上げることで圧縮圧力は上昇する．シリ

計を見直しつつ最大燃焼圧力を上げる方策がある［図5･63(ｃ)］．
