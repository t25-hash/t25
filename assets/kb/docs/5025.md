# 6・3・7 LNG 冷熱発電の性能特性

> **章**: 第6章 未利用エネルギー
> **節**: 6・24 に示す。図6・25に世界初の出力1450kW の冷熱発電商用機

図6･26に出力1450kW冷熱発電商用機の異なる海水条件での
LNG流量に対する発電出力特性を示す．プロパンの凝縮圧力はほ
ぼ大気圧である．海水温度が上がると発電出力は増加する．
6・3・8運転課題とその対応
試運転において経験したトラブル対応を列挙する．
(ａ)プロパンタービンへのミストの同伴 タービンに入るガ
ス中のミストをデミスタで除去できず，発電出力が大きく変動し
た．原因は，蒸発器管群での気泡率が大きいため，液面計の液位が
実際の液位より低い値を指示し，デミスタの底が液につかったこと
による．そこで蒸発器の液面を低下させ，プロパンの蒸発伝熱特性
は変えずに，発電機の出力を安定させることができた．
