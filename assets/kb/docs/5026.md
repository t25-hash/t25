a.クランクケース（crankcase) クランクケースは機関の基
本構造であり，機関を基礎に支持する台板部分，シリンダライナを
支持する部分，連接棒大端部とクランク軸が回転する部分から構成
されている．これらの構成部分には燃焼ガス圧力，慣性力や，これ
はこれらの規則を満足しなければならない．
によって生じるピストンスラスト，慣性偶力等が作用するので，こ
れらに対応できる構造となっている．
中速ディーゼル機関クランクケースの材質としては一般的に鋳鉄
が用いられている．当初はねずみ鋳鉄（FC）が採用されていたが，
出力増加に対する強度向上や軽量化対応のため球状黒鉛（ノジュ
ラ）鋳鉄（FCD）も採用されている．また，一部の大型機関では
鋼板溶接を採用しているものもある．
構造としては，図5･67に示すようにクランク軸が主軸受を含む
ベッドの上に取付けられるベッドタイプと，図5･68に示すように
軸受をフレームの隔壁に設けつり下げ軸受としているハンガタイプ
がある．ハンガタイプの場合，ピストンの力が直接軸受キャップに
作用するのでベッドは不要となり，鉄板の油受でよいから軽い構造
になる．また，ベッドタイプよりも軽量・高剛性の設計がしやす
く，列型舶用ディーゼル機関ではベッドタイプを採用するものもあ
るが，最近の高出力機関のほとんどがこのハンガタイプを採用して
いる．
最近のクランクケースは給気トランクを含め冷却水通路や潤滑油
通路の各種配管を鋳ぐるみにして一体化し，部品点数削減とともに
剛性アップを図っている．

d.シリンダライナ（cylinder liner ) シリンダライナは耐摩
耗性，耐腐食性に優れ，高温における強度・硬さが高く，また油膜
保持力の強いことが要求される．シリンダライナとピストンリング
の間は不完全な境界潤滑のもとにしゅう動するので，油膜が切れる
と異常摩耗に進行する．この場合，油膜がすぐに復元すればそれ以
上進行しない．接触する金属の溶融温度差が大きいと溶着は起こら
ないから，溶融温度差が大きいことが望ましい ．材質としては
鋳鉄が最も一般に用いられ，緻密なパーライト組織の基質に黒鉛が
適当な量と大きさをもって均一に分布したものが適当とされ，フェ
ライトの析出は好ましくない ．また，P，Cr ，Ni ，Cu等を加え
基質の改善を図っている．出力やシリンダ内圧の増加に対してシリ
ンダライナの強度向上が要求されているが，内面の潤滑性も維持ま
たは向上しなければならない．このような要求に応えるため基質を
鋼とし内面部を鋳鉄としたバイメタルライナも開発された ．シ
リンダライナの硬さを向上させるため，内面の硬質クロムめっき，
高周波焼入れ，窒化処理等も行われている．さらに，内面の仕上げ
はホーニング仕上げをクロスハッチに行うことで耐焼付き性の向上
を図っている．ホーニング目の頂点を再研磨し，平坦部を形成する
プラトーホーニングを採用する例が増加している．

f .連接棒（connecting rod) 近年，中速ディーゼル機関の
高出力化，高回転化に伴いクランク軸系の剛性向上や軸受性能向上
の要求が増加している．その結果，クランクピン径が増大し，限ら
れた寸法に抑えなければならない連接棒大端部の設計が非常に困難
になってきている．大端部（big end）はクランクピンに取付ける
ための本体とキャップに分割されているが，この割面が水平方向の
場合を水平割連接棒といい，斜め方向の場合は斜割連接棒という．
水平割連接棒はさらに，大端部と幹部が分かれているものと分かれ
ていないものが存在する．図5･80に各種連接棒の形状を示す．
今日に比べ出力がまだ小さかったころの連接棒は，ほとんどがこ
の水平二分割連接棒であった．出力が大きくないため，大端部の外
形もシリンダ内径よりも小さく，分解時にはキャップを外すだけで
ピストンをシリンダライナから抜くことができた．
出力および回転数が上昇し，クランクピンの径が大きくなってく
ると，大端部外径も大きくなりシリンダ内径以上となった．この場
合，連接棒の装着されたピストンがシリンダ上部から抜けなくな
る．これを回避するため，連接棒の大端部を水平方向でなく角度を
もたせて分割することにより，連接棒を傾けた状態でピストンと連
接棒をライナから抜くことができるようにした斜割連接棒が設計さ
れた．また，この種の連接棒は割面にセレーションと呼ばれるかん
（嵌）合部をもっており，そののこ状の歯で滑り方向の位置を正確
に固定している．この連接棒は分割面を斜めにしたため，ボルトね
じ部が爆発力を受けるほぼ真上にくる構造となっている．

抜けなくなってきたため，水平二分割連接棒を幹部と大端部でさら

k.冷却器（cooler ) 潤滑油，冷却水，空気の冷却器には一

プレートをちょうどフィルタプレスのようにガスケットを介して重
ね合わせて締め付け，各プレート間に薄い長方形断面状の通路を形
成し，この流路を1枚おきに高温液と低温液が交互に流れて熱交換
する構造になっている．突起状のプレート例と流れ様式を図5･99

ースを多く必要とする．後者の熱交換器の特徴は，プレート枚数に
より容易に容量を変更できるとともに，前者に比較してより少ない

ケール落としが簡単に行えることである．
形式選定に当たっては配管との位置関係，メンテナンス性（分解
掃除の必要頻度，許容スペース等），温度レベル等多くの選定基準

計が採用されるようになってきたが，このような場合にはプレート
式熱交換器が使用される割合が増加している．
