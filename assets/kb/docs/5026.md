# 6・3・8 運転課題とその対応

> **章**: 第6章 未利用エネルギー
> **節**: 6・24 に示す。図6・25に世界初の出力1450kW の冷熱発電商用機

試運転において経験したトラブル対応を列挙する。(a) プロパンタービンへのミストの同伴タービンに入るガス中のミストをデミスタで除去できず、発電出力が大きく変動した。原因は、蒸発器管群での気泡率が大きいため、液面計の液位が実際の液位より低い値を指示し、デミスタの底が液につかったことによる。そこで蒸発器の液面を低下させ、プロパンの蒸発伝熱特性は変えずに、発電機の出力を安定させることができた。(b) 凝縮液の蒸発器への供給方法蒸発器伝熱管に同一の熱負荷がかかるよう伝熱管とデミスタの間に分散管を設置した。(c) 天然ガス加温器の着氷天然ガス加温器のガス入口部伝熱管に厚い氷が付着し、その後流部で渦流が発生し壊食が生じた。そこでチタン管の壊食に対する耐食性などの確認を行い、伝熱管を黄銅管からチタン管に変更した。(d) タービンのシール機構プロパンタービンではメカニカルシールを使用したが、試運転においてプロパン内にシールオイルが混入し、蒸発器でフォーミング現象を生じた。各種の対応策で安定した運転が可能となった。(e) 凝縮器の不凝縮性ガスによる伝熱阻害LNG と冷媒の伝熱性能が凝縮器の圧力を下げるにつれ、凝縮器出口天然ガスの温度が下がった。原因は、冷媒ガス系に混入した窒素ガスがタービンの運転で凝縮器に集められ、伝熱管表面が不凝縮性ガスで覆われたためである。そこで凝縮器をブロックし、内部ガスを放散することで、設計どおりの伝熱性能を得ることができた。
