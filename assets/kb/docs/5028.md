# 6・3・10 新LNG 冷熱発電システム

> **章**: 第6章 未利用エネルギー
> **節**: 6・24 に示す.図6・25に世界初の出力1450kW の冷熱発電商用機

新LNG 冷熱発電システムは,従来のランキンや直接膨張サイクルで実施されたフロンやプロパンなどの冷媒を用いてLNG 冷熱で発電する方式ではない.図6・29にこのシステム構成を示す.冷媒ポンプ,タービンおよび冷媒用熱交換器の代わりに,はん用の産業用,事業用ガスタービンコンバインドサイクル(gas turbine com-bined cycle)またはマイクロガスタービンとDACで構成される.-100°C吸気に対応できる既存のガスタービンは現在販売されていない.そこで-100°Cの冷却空気と外気を混ぜ,低温吸気で定格出力が得られるガスタービンに供給する.ガスタービンは,外気温度が下がると吸気量が増す.燃料を設計タービン入口燃焼温度まで増やせるので,発電出力を大幅に向上できる.LNG で吸気を冷却するだけなら,t当たり平均140kWhの発電出力アップとなる.このシステムでは,年間を通じて定格の最大出力で運転できるため,年間総発電量が増え,kWhに占める固定費を下げることができる.2003年10月よりNEDOが経済産業省からの交付金を原資とし実施しているエネルギー使用合理化技術戦略的開発「LNG ハイブリッド冷熱利用システムの研究開発」において,マイクロガスタービンを用いた吸気冷却試験が実施されている.
