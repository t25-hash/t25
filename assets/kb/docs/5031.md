# 7・2・1 元素としての水素

> **章**: 第7章 水素エネルギー
> **節**: 7・2 水素の基礎特性

水素は原子番号1の元素であり、最も軽く、小さい。その同位体には質量数1の水素(H:プロチウム、軽水素、protium;原子量1.007825)、質量数2の重水素(D:ジュウテリウム、deute-rium;原子量2.014102)および質量数3の三重水素(T:トリチウム、tritium;原子量3.01605)がある。いずれも原子核に含まれる陽子は1個であり、中性子数が水素では零、重水素では1、三重水素では2になる。これらの同位体では他の元素に比べてH,D,T 間での相対質量比が大きく、同位体効果が最も大きい元素である。水素、重水素は安定に存在するが、三重水素は放射性同位体元素(半減期12.33年)である。自然界での存在比は、水素が99.985%と大部分を占め、重水素は0.015%と少ない。三重水素は天然には10%程度と極めて少量しか存在しないが、人工的にLiを利用するなどした核反応生成物として得られ、トレーサとしても利用されている。重水素、三重水素は夢のエネルギー源として研究が進められている核融合に用いられているが、詳細は4・8節に譲る。宇宙における元素としての水素の存在比は90%以上であり、宇宙はほとんどが水素である。水素の大部分は化合物ではなく原子・分子・プロトンといった単体として存在する。宇宙では最も簡単な元素である水素から出発する核融合反応により、質量数の大きい他の元素(重元素)を作り出している。太陽においても、そこに存在する大量の水素による核融合反応がエネルギーの源になっており、化石エネルギーを過去の太陽エネルギーの遺産と考えると、現在われわれが利用しているエネルギーの大部分がこれの恩恵に浴していることになる。地球上を考えると、大気中に単体として存在している水素分子はわずかに0.5×10vol%程度である。大気には酸素が含まれているため、水素は単体として安定に存在できず、酸素と結合して水となるか、水酸化物の形として鉱物中に存在することになる。また、水素は炭化水素やアンモニア誘導体など他の元素とのさまざまな化合物としても存在している。元素としての水素の地殻存在比は0.14%と小さいが、非金属元素では酸素・ケイ素に次ぐ3番めであり、元素全体では10番めである。水素の地殻存在比には海洋に存在する水の水素は含まれていない。海を考えると、そこに存在する水は無限大の量であり、水素は多量に存在するといえる。
