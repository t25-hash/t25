# 5・7・3 主要部の構造と特徴：4ストローク船外機機関

> **章**: 第5章 舶用機関
> **節**: 5・7 船外機用機関

5･7･2項の2ストローク船外機と比較して，下記の二つの差違が
ある．
(１)弁機構の装備
(２)機関潤滑油の循環機能の装備
以下に，上記2点について，構造と特徴を記載する．船外機用4
ストローク機関の弁機構と潤滑油の循環機能は図5･134に示す．
a.弁機構
.カム軸 船外機は，高速で回転するため，できる限り往
復運動をなくすために，カム軸はシリンダヘッドに内蔵されてい
る．さらに，1本のカム軸で吸気および排気弁を作動させるタイプ
と，2本のカム軸で吸気弁および排気弁をそれぞれ独自に作動させ
るものとがある．
.弁 弁は，各シリンダごとに吸気および排気用に各1
本，合計2本保有しているもの，高出力を得るため吸気用に2本，
排気用に1本と合計3本保有しているもの，さらに，吸気および排
気用にそれぞれ2本，合計4本保有するものもある．
.その他 4ストローク船外機にとって，弁機構は，出力
を引き出す重要な部品であることから，低速領域では1本の吸気弁
が作動し，高速領域では2本の吸気弁が作動する複雑な動きをする
船外機が作られるようになった．
b.機関潤滑油の循環機能 船外機の潤滑は，大きく二つのタ
イプのものが採用されている．
(１)はねかけ式（一部の小型船外機に採用）
(２)ポンプによる強制圧送式（ほとんどの船外機に採用）
.潤滑経路 船外機の場合は自動車などと異なりクランク
軸が縦配置であるため，オイルパンはクランクケースにではなくシ
リンダ下部に設けられて潤滑油がたまる構造になっており，たまっ
た潤滑油はポンプにより吸い上げられ，下記の二つのルートに圧送
される．
(１)クランク軸およびピストンまわりを潤滑するルート
(２)カム軸およびロッカアーム，弁まわりを潤滑するルート
以下に，詳細な通路の説明をする．
.クランク軸およびピストンまわりを潤滑するルート ポ
ンプから吐出された潤滑油は，クランクケースの通路を通り，クラ
ンク軸の軸受に圧送される．さらにクランク軸には，軸からピンま
で通路が設置されているため，潤滑油はクランクピンまで到達し，
クランクピンの動きとともに放出され，連接棒，ピストン，シリン
ダに潤滑油の飛まつがかかり潤滑する．
.カム軸およびロッカアーム，弁まわりを潤滑するルート
ポンプから吐出された潤滑油は，クランクケース，シリンダおよび
シリンダヘッドのオイル通路に従って圧送され，カム軸の軸受に導
かれる．
さらに，カム軸にも軸受部からカムまでオイル通路が設置されて
いるため，潤滑油はカムまで到達し，カムの動きとともに放出され
弁機構の重要部品であるロッカアーム，弁まわりを潤滑する．
.その他，構成部品の説明
(１)オイルポンプ
オイルポンプは一般的にカム軸に装着されており，インナロータ
とアウタロータ，ボデーで構成するトロコイド方式が主流である．
作動原理は，インナロータとアウタロータ間の容積変化により，吐
出する構造になっている．
(２)オイルフィルタ
潤滑油の異物，金属かすなどを取り除き，つねにきれいな潤滑油
が行き渡るよう設置されている．形式は，編目式とろ紙式のものが
多く採用されている．
(３)油圧警報装置
潤滑油の量および潤滑状態をつねにチェック，運転者が簡単に確
認できるよう，オイルプレッシャスイッチおよびインジケータラン
プが装着されている．
ただし，一部の小型船外機は，クランクケース内にのぞき窓を設
け，直接，目視するものもある．
