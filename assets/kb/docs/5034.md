# 7・2・4 水素の電気化学的エネルギー変換

> **章**: 第7章 水素エネルギー
> **節**: 7・2 水素の基礎特性

水素のもう一つの特徴は次式H+12OHO(7・3)を利用して,化学エネルギーと電気エネルギーを直接相互に変換できることである.式(7・3)の右向きが水生成反応,左向きが水電解になる.298K,101.3kPaにおける式(7・3)に対するエネルギーダイアグラムを図7・6に示す.標準状態で考えると,この反応のΔ°は-286kJであり,これは外部に取出しうる仕事Δ°=-237kJ(これを化学エネルギーと呼ぶ)と可逆熱Δ°=-49kJに分けられる.この化学エネルギーが電気エネルギーと直接変換される.すなわち,理論的には燃料電池で水生成反応により,水1mol当たり最大237kJの電気エネルギーと最小49kJ の熱が得られ,水電解の場合には,最小237kJの電気エネルギーと最大49kJの熱が必要となる.この条件における燃料電池で得られる理論起電力(theoretical electromotive force)は1.23V であり,水の理論分解電圧は1.23V となる.実際には系に内在する不可逆性のために,燃料電池で得られる電気エネルギーは化学エネルギーの減少分よりも必ず少なく,水電解で増加する化学エネルギーは加えた電気エネルギーより必ず小さくなる.図7・7に式(7・3)左向きの水電解反応に伴うΔ°,Δ°および理論分解電圧の温度依存性を示す.100°C以上は水蒸気である.吸熱反応である水電解反応は,温度の上昇とともに必要な電気エネルギーが小さい,すなわち,分解電圧が小さくて済む.しかし,逆に燃料電池として考えた場合,発熱反応である水生成反応は温度の上昇とともに得られる電気エネルギーは小さくなることを意味する.理論的には燃料電池は低温で作動させたほうが大きな電気エネルギーが得られ,水電解では高温で行うほど小さな電気エネルギーでの電解が可能となる.5―212エネルギー供給システム
