# 7・3・2 水素の分離・精製(hydrogen purification method)

> **章**: 第7章 水素エネルギー
> **節**: 7・9 に原理を示すが、陽極で水が分解され、酸素と水素イオンと電

水素の製造時には使用する原料やプロセスにより種々の不純物が混じる。例えば天然ガスの水蒸気改質法では水素75%,CO23%,CO1%、その他微量の不純物などの組成になるので、99.99%レベルの純水素を得るためには、精製工程を通して不純物を除去する必要がある。水素の分離・精製法は大別すると吸収法、深冷分離法、吸着法、拡散法などの種類があるが、用途に応じてその特徴を生かせる方法が採用されている。a. PSA(pressure swing adsorption)大容量化学プラントから小規模な水素製造装置にまで広く利用されている実用的精製装置である。吸着剤を充てんした吸着塔内で水素に混合している不純ガスを吸着除去する。吸着剤は原料ガスの組成に応じて選定されるが、他成分に対処するために複数の吸着剤、例えばCMS(カーボンモレキュラシーブス)とZMS(ゼオライトモレキュラシーブス)等が併用される。CMS などの吸着能は、水素に対する吸着量が他のガスと比較して非常に少ないので、高純度の水素ガスを得ることができる。運転は0.5~4MPaくらいで吸着させ、低圧に戻すことによって吸着した不純物を脱着する。製品水素としては99.999~99.9999%の純度まで精製できる。通常4塔の吸着塔を備え、1塔は吸着・均圧減圧・洗浄・均圧昇圧の4段階を1サイクルとして、4塔が1工程ずつずれて作動する。製品ガスは連続的に供給され、1サイクルは15~20minで行われる。減圧後の洗浄は再生用パージガスとして水素ガスを少量流して行う。さらに不純物をppbの単位まで低減させる場合は、モレキュラシーブスや活性炭を充てんした吸着塔を液体窒素中に設置した低温吸着精製装置を使用する。PSA のシステムを図7・11に示す。b. 選択酸化法(partial oxidation method)水素中の不純ガスのうち特定のガスのみを酸化して除去する方法が選択酸化法である。天然ガス改質型燃料電池の改質ガスには0.5~1%のCOが含まれているが、固体高分子形燃料電池の場合、電池触媒に有害なCOを約10ppm程度以下に低減するため選択酸化法が採用される。選択酸化には白金触媒やルテニウム触媒が使用され100~150°CでCOを酸化する。c. 深冷分離法(cryogenic separation method)極低温の冷熱を利用して水素中の不純ガスを液化して除去する方法である。純度が95%程度の水素を数千N・m/h以上の大量に必要とするときに適しているが、寒冷の発生源が必要となる。d. 高分子膜による水素の分離石油精製産業では水添脱硫図7・11 PAS のシステム図7・12 Pd膜における水素透過の原理プロセスに大量の水素を必要とするので高分子膜によるオフガスからの水素分離が行われている。膜の材料としてはポリスルホン膜、ポリイミド膜、ポリアラミド膜などが使用される。この方法は高圧のプロセスで水素の回収率が高く、高純度を要求されない水素濃度95~99%程度のプロセスに適する。e. 金属分離膜による水素の分離高純度の水素が得られる膜分離技術として金属および合金の膜を使用する分離法がある。Pd膜は水素の分離に最適であるが、水素の透過機構を図7・12に示す。水素透過のプロセスは次のとおりである。(1) 水素分子がPd膜表面に吸着する。(2) 水素原子への解離が起こる。この反応は吸熱反応であるため、膜か原料ガスを加熱すると原子状水素を容易に作りうる。(3) 解離した水素原子からPdが電子を奪うことによりプロトンが生じる。(4) 濃度こう配により膜中をプロトンが拡散する。この際、原料ガス側の水素濃度(分圧)が精製水素側の水素濃度(分圧)より大きくなければならない。(5) 精製水素側膜表面においてプロトンと電子が再結合し、原子状となる。(6) Pdの触媒能によって原子状水素は会合して分子となる。(7) 吸着分子が着脱する。水素以外の分子は上述の過程を経ることができないので、原理的には純度100%の水素を得ることが可能である。水素透過量は水素分圧差が大きいほど、温度が高いほど、また膜厚が薄いほど大きくなる。水素分離金属膜には純Pd,PdとAg の二元合金、PdとAg,AuなどのIb族およびPt,RuなどのVIII族との三元合金があるが、三元合金は純Pdの約2倍の透過速度を有する。Pd基合金は高価であること、573K 以下の低温領域においては水素の透過度が低く使用できないなどの課題があるので、673K 以下の低温において純Pdより大きい水素透過度を有するV-Ni合金膜が水素分離膜材料としての可能性を有している。そのほかテフロンまたはポリイミド膜上に水素吸蔵合金のLaNiの薄膜を形成する方法も研究されている。
