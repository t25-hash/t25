# 7・4・1 概要

> **章**: 第7章 水素エネルギー
> **節**: 7・4 水素輸送・貯蔵技術

大規模な工業用水素の輸送・貯蔵は従来から行われているが、最近では、燃料電池自動車や一般家庭用電源としての小規模分散による固定式燃料電池の普及が予測され、小容量の水素輸送・貯蔵技術が必要となってきた。水素の輸送・貯蔵方法としては、(a)気体水素での輸送・貯蔵で、水素ガスを高圧に圧縮し容器等に充てんする方法、または水素ガスを圧縮してパイプラインで圧送する方法、(b)液体水素での輸送・貯蔵で水素を液化して断熱容器に充てんする方法、(c)水素を水素吸蔵材料(メタルハイドライド、ケミカルハイドライドや炭素材料)に吸蔵させて輸送・貯蔵する方法が考えられ、その特徴は表
