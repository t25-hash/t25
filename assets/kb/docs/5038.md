b.排気対策 大気汚染物質の中で現在特に問題とされている
のが窒素酸化物（NO ）と硫黄酸化物（SO ）である．ディーゼル
機関と比較すると，ガスタービンはNOの生成要因となる燃焼部
最高温度や圧力比が低いうえに，硫黄分をほとんど含まない燃料を
使用するため，NO ，SOともに，国際海事機構（IMO）が舶用
機関に要求する規制値に比べて十分低く，何らかの排気対策が早急
に必要とされるレベルにはない．しかし，今後海上においても一般
産業用と同様に，環境にいっそう配慮した舶用機関が必要とされる
ことを念頭において，液体燃料に対する乾式低NO燃焼器（dry
lowNOcombustor ）の開発が各社で進められている．その一例
として，SMGTにおいても図5･149に示す希薄予混合・予蒸発＋
追だき燃焼方式の燃焼器を装備し，1g/(kW･h）のNO排出量を
達成する低NO化が図られている．
c.振動騒音対策 純粋な回転運動を行うガスタービンの振動
レベルは往復運動の舶用機関に比べて十分小さい．したがって，防
振ゴム製のマウントでガスタービンパッケージ全体を弾性支持する
ことで船体への振動伝ぱを通常問題ないレベルまで低減できるうえ
に，船殻のたわみや船体側からの衝撃荷重，振動も緩和することが
できる．なお，弾性支持を行う場合は，過大変位を防止するための
ストッパを併用するのが一般的である．
ガスタービンからの騒音は高周波の流体音が主体であるので，ガ
スタービン本体を適切な防音エンクロージャで囲い，吸排気ダクト
中に消音器を装備することで問題ないレベルに低減することは比較
的容易である．ただし，吸気消音器（intake silencer ）は部品が脱
落して異物としてガスタービンに吸い込まれた場合，甚大な被害
（FOD:foreign object damage）を招くことになる．また，排気
消音器（exhaust silencer ）は吸音材を充てんしたスプリッタが高
温高速の排ガス気流中にさらされるため，消音器の構造，使用材料
の耐腐食性や強度，排気消音器に関しては特に熱膨張にも十分配慮
した設計を行う必要がある．
