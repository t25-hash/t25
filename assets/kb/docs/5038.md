# 7・4・2 水素輸送・貯蔵能力

> **章**: 第7章 水素エネルギー
> **節**: 7・7 のようにまとめられる。

水素の輸送貯蔵能力はその輸送貯蔵方法によって異なっており、以下の二つの方法でその能力を評価することができる。a. 体積水素密度(volumetric hydrogen density)輸送貯蔵容器の単位容積(m)当たりに輸送・貯蔵できる水素重量を示し、kgH/mで表す。表7・7 水素形態による輸送・貯蔵の特徴圧縮水素ガス液体水素水素吸蔵合金炭素系吸蔵材料・常温高圧圧縮・20MPa以下ボンベ・FCV 用水素供給圧力現在35MPa、将来70MPa・金属間化合物、固溶体合金の水素化物・冷却加圧で吸収、昇温減圧で放出・高水素吸蔵量合金の開発中・極低温(20K,0.1MPa)・高密度(70.8kg/m)・カーボンナノチューブ、カーボンファイバ、黒鉛、活性炭・水素の物理吸着・低温、高圧内容・圧力上昇で充てん密度アップ可能・設備的に簡便・重量水素密度が大・体積水素密度が大・大量輸送・貯蔵に適・L N G の輸送・貯蔵技術が適用可能・安全性が高い・体積水素密度が大・常温、常圧近傍で作動・吸蔵、放出速度が大・重量水素密度が大(軽い)・資源的に豊富長所・安全性(高圧、漏えい)・圧力上昇で充てん効率低下・体積水素密度が小・安全性(極低温、漏えい)・極低温断熱容器が必要・ボイルオフロスが大・液化エネルギー、設備コストが大・重量水素密度が小(重い)・体積水素密度が小・高圧充てんが必要・大量生産技術が未・高価短所図7・13 種々の水素貯蔵形態における水素貯蔵能力5―215特に燃料電池自動車の乗用車等では燃料としての水素搭載スペースが制限されるため、自動車航続距離を伸ばすためには、この体積水素密度が大きい方法を選択する必要がある。b. 重量水素密度(gravimetric hydrogen density)輸送貯蔵容器の自重を含む全体重量当たりに貯蔵できる水素重量を示し、kgH/kg またはwt.%Hで表す。燃料電池自動車のような移動体の燃料水素貯蔵容器では、搭載する燃料水素の重量負荷を極力抑えるため、水素容器重量が小さく水素貯蔵量が大きい方法、つまり重量水素密度が大きい方法を選択する必要がある。図7・13は種々の水素貯蔵形態における水素貯蔵能力を、体積水素密度と重量水素密度を用いて表した一例である。
