b.船舶の大きさ，トン数
.総トン数（GT:gross tonnage) 船の内部の総容積
100cf （立方フィート，1000/353m ）を1tとして表したもので，
船の大きさを表す指標として課税，通量の算定に用いられる．詳細
は船舶積量測度規程により規定されている．総トン数には国際総ト
ン数と総トン数があり，国際総トン数は，「船舶のトン数制度に関
する国際条約（1966）」に定める計測方法によって算定され，国際
航海に従事する船舶に適用される．
.純トン数（NT:net tonnage) 純トン数は，総トン数
から機関室，乗組員室，バラストタンク等の運航に直接関わる場所
の容積を差引いた部分の容積を総トン数と同じ単位で表した大きさ
の指標である．旅客，貨物等を積載する場所の大きさを表し，利益
に直接関係する指標であるから，種々の課税，手数料算定の基準と
して用いられる．
.排水量（Δ， :displacement ) ある喫水状態におけ
る船の全重量を表し，船体が排除する水の重量に等しい．満載状態
の排水量を満載重量，軽荷状態の排水量を軽荷重量と呼ぶ．
.載貨重量（DW:deadweight ) 満載重量と軽荷重量の
差で表される重量で，貨物船の積載能力を表すのに用いられる．
図5･13油槽船（タンカー)
DWには貨物重量のほかに燃料油，水，油類，乗組員の重量を含
む．DWの表現はメートル法単位のキロトン（KT）と英国式単位
の英トン（LT）が用いられる．
.載貨容積（cargo capacity) 載貨容積は全貨物倉の容
積でmまたはcfで表す．ベール（bale）とグレーン（grain）が
あり，ベールは梱包（こんぽう）貨物を格納できる容積を表し，貨
物倉内の突起物（肋骨，はり等）の内側までの容積，グレーンはば
ら積貨物積載容積を表し，貨物倉内部の総容積である．
