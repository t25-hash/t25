# 7・5・3 熱利用ほか

> **章**: 第7章 水素エネルギー
> **節**: 7・5 水素のエネルギー利用技術

a. 燃焼熱利用動力変換技術と同様に水素を燃料として燃焼させ、その高温熱を直接利用するボイラ、厨房ガス機器への利用はすでに検討され、NO生成抑制の触媒燃焼利用など基盤は把握されており、これらの実用化のキーは水素供給インフラストラクチャ整備にある。b. 変換媒体利用水素吸蔵合金は、水素吸蔵合金+水素⇔金属水素化物の可逆反応で水素を吸蔵および放出する。この可逆反応の平衡温度・平衡圧力特性や吸蔵・放出時の発熱・吸熱特性を利用する方法である(表7・13参照)。水素吸蔵合金アクチュエータ(metal hydride actuator)は、水素吸蔵合金を熱から機械エネルギーへ変換するための変換媒体に利用するものであり、具体的には、水素吸蔵合金を封入した容器をヒータやクーラで温度制御して、この温度相当の水素平衡ガス圧力を利用してピストン等の駆動仕事に変換する。したがって、本利用は熱機関であり、その効率は低く、応答性は熱源と合金間の熱移動速度で支配されるので遅い。しかし、動作音は小さく、得られる圧力は大きいなどの特徴から、動作音の制約が厳しい条件、かつ繰返し移動するような使用条件での用途が期待され、昇降型福祉機器への実証例が試みられている。ヒートポンプ利用では、吸蔵・放出時の発熱・吸熱を利用して熱をくみ上げるもので、異なる温度特性をもつ吸蔵合金をおのおの封入した二つの容器間を各容器の温度条件を設定することにより、水素ガスを移動させて熱をくみ上げる熱駆動方式と、1種類の吸蔵合図7・26移動用電池の重量エネルギー密度と出力密度図7・27 水素エネルギーシステムの概念図金を用いて少量の熱源とコンプレッサで熱移動するコンプレッサ式がある。その用途には住宅向け冷暖房システム、工場のコージェネレーションシステム、冷凍システム等への応用が実証されている。
