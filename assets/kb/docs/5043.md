# 5・1・5 船舶の計画・設計

> **章**: 第5章 船舶
> **節**: 5・1 船舶一般

船舶の設計に限らず，工業製品の設計のプロセスにおいては，
(１)必要の認識：社会・経済的ニーズの把握，(２)問題の定義：設
計要求事項の存在を見分けて認識する，(３)問題の解析・最適化・
総合による代替案の作成，(４)代替案の評価・統合化：客観的評価
に基づく実行案の絞り込み，が必要である．
船舶の計画・設計の基本的な流れを図5･24 に示す．船舶の基
本計画段階では，(１)周辺の社会・経済状況，運航上の制約条件等
を勘案しつつ，(２)船主と基本設計者が共同して船舶システムや使
命の解析を繰返し行い設計船の機能・性能の計画を策定した後，
(３)(４)造船所設計者が初期設計，契約設計，詳細設計を行うこと
となる．現在ではわが国の多くの造船所では，船舶の使命や所要性
能は船主からの要求事項として受け入れ［プロセス(１)(２)］，初
期設計，契約設計，詳細設計を設計者が行う［プロセス(３)(４)］
が採用されている．
プロセス(１)(２)の基本計画においては，種々の要求・制約事項
の検討は広範な視点に立って行われる必要があり，運航における専
門的技術情報もこの段階で造船所技術者にフィードバックされる．
また，新たな技術情報を必要とする場合には企業の研究所を活用す
ることとなる．基本設計者は船主側と造船所側の間の通訳の役割を
果たすことになり，初期設計から後のプロセスでは構造・機能・性
能に関する熟練解析者（設計者）の役割を果たす．
初期設計段階では，船主要求仕様（船種，載貨重量，航海速力
等）を満足する船型，主要目，構造，概略配置，諸性能，船価等を
複数の代替案に対して見積もり，船主との交渉の基本資料を作成す
る．契約設計においては，初期設計で作成された案を基本に船主側
と交渉し，船主要求，使命に適合した主要目，構造，概略配置，諸
性能，船価を決定し，建造契約書類としての各種図面，仕様書を作
成する．詳細設計においては，資材調達，建造工程管理，品質管
理，工作図面の作成等具体的な建造に必要となる詳細事項の決定を
行う．
