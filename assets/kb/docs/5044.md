# 7・6・2 水素エネルギーシステム概念の拡大

> **章**: 第7章 水素エネルギー
> **節**: 7・6 水素エネルギーシステム

近年の燃料電池実用化促進等の動向を踏まえると,図7・27に示したような再生可能エネルギーベースのシステムの構築に至る中間段階においても,水素に変換・利用することにより,化石燃料の従来より以上の高効率利用が可能となる分野への水素供給を視野に入れて,化石燃料から製造した水素の供給システムを構築すべきである.この意味で,近年水素エネルギーシステムは図7・27に示した基図7・28広義の水素エネルギーシステム(液体)CHOH(l)CO+2H-9464.649.412.521.5CHOH(l)+HOCO+3H18.811.72CHOH(l)HCOCH+H12.43.1311.4NH(l)1/2N+3/2H-77.717.711.0H(l)-259-25370.0(気体)H(g,200atm)-259-25318.0(固体)7iH(s)7i+H4.0434.5本概念に,化石燃料の高効率利用の観点を加えた幅広い概念となってきている.図7・28は再生可能エネルギー,化石エネルギーいずれをとっても一次エネルギーの大部分を海外に依存せざるを得ないわが国に即して描いた構想,すなわち,エネルギーの海洋輸送を含むグローバル(地球規模)システム構想を示したものである.しかし,近年では,一国内から一住宅単位まで,さまざまのサイズの領域内でのローカルシステム,すなわち図7・28に示したシステムの縮小型などのバリエーションも検討されており,それらを包含して,水素エネルギーシステムと呼ぶようになっている.ローカルシステムの例としては,わが国の西南諸島など,全国的な電力グリットから隔離され,島ごとにディーゼル発電をしている離島における,太陽エネルギー/風力/水素システムなどがある.
