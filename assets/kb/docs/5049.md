# 7・6・7 鉄鋼業副生水素供給システム

> **章**: 第7章 水素エネルギー
> **節**: 7・15 に示す。

世界の自動車産業が、2003~2005年の市販を目標に燃料電池自動車(FCV)の実用化へ向かって大きく動いている。経済産業省資源エネルギー庁は、2010年に5万台、2020年に500万台の燃料電池自動車実用化戦略を打ち出している。FCV 実用化の動きを背景として、特に導入初期(2005~2010年)における水素供給をどうするかが緊要の課題となってきた。その答えの一つとして提案されたのが、鉄鋼業副生水素を利用するシステムである。特別な水素5―226エネルギー供給システム表7・17 鉄鋼副生ガス量と水素含有量副生ガス種副生ガス量水素含有量COG115億Nm/年64億Nm/年(+116億Nm/年)BFG840億Nm/年197億Nm/年LDG71億Nm/年53億Nm/年注:a) 2000年度値ベースb) BFG およびLDG の水素含有量は変成反応後の値を記載c) 副生ガス量は回収ガスから製銑用およびコークス製造消費分を差し引いた値d) COG の水素含有量には()内にCOG 改質後の増量分を記載製造プラントを新たに建設する必要性がないので、即応性があるのが利点である。鉄鋼業副生水素源として利用可能な副生ガスは、コークス炉ガス(COG)、高炉ガス(BFG)および転炉ガス(LDG)である。それらの性状を表7・16に示す。COG は水素を約50%含有しており、硫黄分等の不純物除去とPSA(pressure swing adsorption)処理を行うことで容易に高純度水素を回収することができる。BFG およびLDG は、含有している一酸化炭素を水生ガス変成反応を利用して水素に転換することができる。表7・17に鉄鋼業副生ガス量と水素含有量を示す。BFG とLDG は一酸化炭素変成後の値である。また、COG に約30%含まれるメタンを水蒸気改質して水素に転換した場合の水素の増量分も示してある。COG の場合は、水素の精製・分離がシステムとして確立しているので、自動車用燃料水素源としては最も有力な導入初期水素源の一つと考えられる。COG そのものは現在でも製鉄所内燃料として有効利用されているが、全体のエネルギーバランスに対する影響の少ない量として、副生水素64億N・m/年の約8%程度(熱量ベースでCOG の約5%)を仮定すると、FCV 用燃料として供給可能な量は約5億N・m/年となる。この量でまかなえるFCV 台数は約50万台である。COG 副生水素システムの概念を図7・32に示す。図7・32のシステムにおいて、COG の本来原料である石炭由来の炭素はすべて鉄鉱石還元用に供されるので、COG 副生水素製造プロセスでは、水素精製動力分等以外には直接炭酸ガス発生はないとみなすことができる。なお副生水素利用システムとしては、食塩電解によるかせいソーダ、塩素生産プロセスからの副生水素を水素源とするシステムも検討されている。
