# 5・3・1 プロペラ（propeller ）

> **章**: 第5章 船舶
> **節**: 5・3 推進器と動力伝達装置

船を推進する装置としてはオール，帆，外輪，スクリュー，ジェ
ット，ロケット等があるが，一般には普通の船舶に最も多用される
スクリュープロペラ（screw propeller ）を単にプロペラといい，
特殊な形状，構造のスクリュープロペラを含め，その他のものを特
殊プロペラという．以下にスクリュープロペラの形状，構造等につ
いて説明する．
スクリュープロペラの羽根数は，推進性能を主眼に，かつ，船体
および軸系の諸振動を考慮して決めるが，一般には3～5枚である．
巨大船や超高速貨物船では6枚羽根が使用されることもある．図
5･43に代表的な4枚羽根プロペラの形状および各部の名称などを
示す．プロペラの回転方向は，一軸船では船尾から見て時計回りが
普通で，これを右回りという．二軸船では右舷が右回りで左舷が左
回りの組合せを外回り，これの反対を内回りといい，推進，操船性
能等を考慮して選定する．羽根とボスは一体に鋳造されるのが普通
であるが，別個に製作してボルトにより結合する組立型もあり，砕
氷船などに採用される．材質としては強度，鋳造性，海水中におけ
る耐食性などの見地からアルミニウム青銅や高力黄銅などが常用さ
れ，氷海航行船などにはステンレス鋼も使用される．
また，最近はキャビテーションによる船尾振動軽減の見地から，
ハイスキュープロペラ（highly skewed propeller ）のほうが多く
用いられている．これは，羽根のスキュー角（skew angle，図
5･44) を大きくして，羽根が高伴流域を通過する時間を半径位置
でずらしてプロペラ起振力を減少させることをねらったプロペラで
ある．自動車運搬船に用いられているハイスキュープロペラの例を
図5･45に示す．
