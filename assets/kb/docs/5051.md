プロペラの性能は，揚力面理論などの計算によっても実用的な資料

スラスト係数（thrust coeﬃcient ) ＝ρ
トルク係数（torque coeﬃcient ) ＝ρ
前進係数（advance ratio) ＝1ω
プロペラ単独効率（propeller open water eﬃciency)
η＝2π
ここで， は毎秒回転数， はプロペラ直径を示す．
プロペラは直径が大きく回転数の小さいほうが効率は高い．効率
の実例については5･2節の表5･4を参照されたい．
プロペラの設計（要目決定）は，翼形，展開面積，羽根数ごとに
作られているチャートによるのが一般的で，図5･47に一例を示し
た．図5･47は チャートと呼ばれており，馬力 ，毎分回転
