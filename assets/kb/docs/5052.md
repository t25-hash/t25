# 8・1・1 概説

> **章**: 第8章 エネルギー輸送・貯蔵
> **節**: 8・1 エネルギー輸送

動の源がエネルギーであり,特に資源小国のわが国において,その確保は重要でありエネルギー輸送は重要な課題である.前版では,輸送については熱輸送のみが述べられていたが,本書では上記の重要性に鑑みて,エネルギー輸送全般について述べることにした.すなわち,一次エネルギーである石油,ガス,石炭各燃料の輸送,およびエネルギーの一形態である熱の輸送,ならびに,二次エネルギーである電力エネルギーの輸送と,その対象を広くした.これら各技術は,エネルギー輸送という面では共通であるが,それぞれの間には,エネルギーが一次エネルギーか加工された状態かどうかというエネルギー形態,あるいは,エネルギーの利用目的とそれに伴う形態の種類,エネルギーの質,ならびにこれらに伴い輸送形態が大きく異なっており,このため,相互の比較より個々の技術に意味があるので,概説では,8・1節の構成についてふれるのみとし,個々の技術については各項目を参照されたい.まず,燃料輸送としては,石油,ガス,石炭について述べる.資源のないわが国にとって今後とも一次エネルギー,化石燃料資源の確保は重要な課題である.大陸と結ぶような広域パイプライン構想なども立案されており,この実現に果たす機械工学者の役割は大きい.熱輸送に関しては前版で詳しく記載されていたが,本書でも踏襲し,顕熱と潜熱,およびヒートパイプについて詳述した.電力輸送に関しては,今回新しく含めた.内容としては,基本的な輸送技術として,架空送電線による輸送,地中送電線や海底ケーブルによる輸送を解説するとともに,新しい送電技術として,直流送電技術,超伝導送電技術,マイクロ波送電技術についても解説することとした.直流送電技術は,わが国では,北海道と本州間,本州と四国間で実現しており,世界的にも特に長距離送電や海底ケーブル送電に適することから,近年多く建設されている.超伝導送電はまだ実現している技術ではないが,電気抵抗が零になるという,夢の送電技術として各国で開発が進められている.わが国の超伝導技術の技術レベルは高く,現在,高温超伝導線材を使った実証研究が進められている.マイクロ波送電技術もまだ実現している技術ではないが,無線で送電ができるという夢の送電技術であり,また,将来,宇宙での太陽電池から地球に送電する技術として開発が進められている.
