# 5・3・3 動力伝達装置および軸系

> **節**: 5・3 推進器と動力伝達装置

般に減速装置，弾性継手，クラッチなどを指し，主機あるいは動力
伝達装置からプロペラに軸出力を伝え船に推進力を与える機構を総
称して軸系（shafting arrangement ）と呼ぶ．

b.減速装置 減速装置（reduction gear ）は，通常，歯車式
が用いられる．主機で発電して軸系を電動機で駆動する電気推進も
一種の減速方式であるが，ここでは歯車式について説明する．減速
装置の形式は主機の特性，運転形態，機関室配置などに応じて多種
多様であるが，大きくは平行軸型と遊星歯車型に分類される．平行
軸系はあらゆる主機形式の一機一軸，多機一軸，多機多軸の主機・
軸系配置に広く使用される．図5･49 はデュアルタンデムアーテ
ィキュレーテッド型（dual tandem articulated type），通称ロック
ドトレン（locked train type）の例で，大出力蒸気タービン主機用
などに使用する．遊星歯車型は，入出力軸が同心で小型軽量の利点
があり，一機一軸配置のディーゼル船に多く採用する．また，平行
軸型と組合わせて使用する例も多い．
主機がディーゼル機関の場合は回転数が比較的低く，通常は一段
減速である．一機一軸平行軸型は水平オフセットとすることが多い
が，上下オフセットを採用して主機を船体中心に配置することもあ
る．いずれの場合も，機関室配置にとって大小歯車の心間距離は極
力短いほうが好ましいので，表面硬化歯車を使用して歯車を小型化
することが多い．多機一軸平行軸型の場合は，機関の据付間隔を確
保するため小型化の要求は少なく，一般に調質歯車が使用される．
主機が非逆転の場合，後進のための逆転機構が必要である．クラッ
チをかん（嵌）脱して歯車系を切換える機構のものが小出力主機に
使用される．
主機が蒸気タービンの場合は高圧段と低圧段の二つのタービンを
二機一軸型に配置するケースが多い．低圧タービンには後進用ター
ビンを内蔵するのが普通である．タービンが高出力かつ高回転であ
るので，減速装置は大型，高速，高減速比となる．通常，二段減速
で調質シェービング仕上げのやまば歯車が用いられる．
ガスタービンは商船用としての例は少なく，艦艇用が主である．
艦艇では巡航時と増速時の出力差が大きく，このため複数台の主機
を要求出力に応じ組合わせて使う方法をとる．したがって，主機・
軸系配置は一般に多機一軸あるいは多機多軸となる．ガスタービン
の場合も減速比が大きく，一般に二段減速とする．複数台の主機の
統合分割はクラッチと歯車の組合せで行う．また，ガスタービンは
