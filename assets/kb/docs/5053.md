# 5・3・4 プロペラのキャビテーション

> **章**: 第5章 船舶
> **節**: 5・3 推進器と動力伝達装置

プロペラの性能でキャビテーション（cavitation）の影響は大変激しいキャビテーションが発生しないようにプロペラを設計する．
衝撃圧でプロペラ翼面を浸食，破損し，騒音，振動の原因となる場キャビテーションについては理論的な研究のほかにキャビテーション水槽（高速の減圧回流水槽）における模型プロペラの実験的研ーションが最も有害な影響を与えるので，プロペラと船体を組合わーション実験が行われている．
一般商船では，軍用の高速艇のように推力低下を伴うしかし効率に影響はなくても，キャビテーション気泡の消滅による合が多い．
分の損失があるなど，一般にプロペラより効率がかなり悪い．
究も活発である．
