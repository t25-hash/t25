# 5・3・4 プロペラのキャビテーション

> **章**: 第5章 船舶
> **節**: 5・3 推進器と動力伝達装置

プロペラの性能でキャビテーション（cavitation）の影響は大変
重要である．一般商船では，軍用の高速艇のように推力低下を伴う
激しいキャビテーションが発生しないようにプロペラを設計する．
しかし効率に影響はなくても，キャビテーション気泡の消滅による
衝撃圧でプロペラ翼面を浸食，破損し，騒音，振動の原因となる場
合が多い．
キャビテーションについては理論的な研究のほかにキャビテーシ
分の損失があるなど，一般にプロペラより効率がかなり悪い．
ョン水槽（高速の減圧回流水槽）における模型プロペラの実験的研
究も活発である．船尾流れのような不均一流中で発生するキャビテ
ーションが最も有害な影響を与えるので，プロペラと船体を組合わ
せた実験や，金網等で船尾流れを模擬した中でのプロペラキャビテ
ーション実験が行われている．
