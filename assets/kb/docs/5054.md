# 5・3・5 特殊推進装置

> **章**: 第5章 船舶
> **節**: 5・3 推進器と動力伝達装置

a.可変ピッチプロペラ（controllable pitch propeller ) 翼角
を変えること（変節）により，ピッチを前進から後進まで自由に変
えることができる機能をもつプロペラである．変節はプロペラボス
内および軸系内に装備される油圧機構によって行う．船体の推進抵
抗が変化した場合でも翼角を制御することにより主機出力を有効に
利用できる．前後進の切換えが早い，定回転運転が可能などの利点
がある．ボスの径が大きくなるなどの理由によりプロペラ効率は若
干低下するが，上記の利点があるため引き船，トロール船，カーフ
ェリーなどに多用されるとともに一般商船にも採用されている．

c .舵の機能を併せ持つプロペラ この種のプロペラとしては，
古くから鉛直軸の回りに翼形が回転する形式のフォイトシュナイダ
ープロペラが有名であるが，日本ではほとんど使用されず，これに
代わるものとして，ラダープロペラ（rudder propeller ）が普及し
ている．ラダープロペラは，かさ歯車を用いて駆動軸を直交変換
し，ダクトプロペラを360°旋回できるようにした機構をもつプロ
ペラである．固定ピッチプロペラのものと可変ピッチプロペラのも
のがある．操縦性が良いので引き船や作業船等に広く使用されてい
る．最近は大型高出力のものも開発され，砕氷船を中心に採用が増
えてきている．また，かさ歯車による効率ロスやトルクの限界を克
服するため，電動機を小型化して船外に出し，プロペラと直結させ
るというPOD型推進器（POD propeller ）も実用化されつつある．

e .スーパーキャビテーティングプロペラ（supercavitating pro-
peller ）とトランスキャビテーティングプロペラ（transcavitating
propeller ) 船速が50knを超えるような超高速船では通常のプ
ロペラでは大量のキャビテーションの発生により性能が極度に悪く
なるため，スーパーキャビテーティングプロペラが適する．このプ
ロペラは，通常のプロペラと異なり，大きなキャビテーションを発
生した状態で設計する．負圧面の圧力が水の蒸気圧になるため，正
圧面をへこませて，正圧面の圧力を高くすることによっても推力を
得る．現在のところ小型艇にしか実績がないが，原理的には大型船
にも適用可能である．一方，通常型プロペラでは船速35kn程度を
超えるとキャビテーションのため効率が大きく劣化する．船速
35～50knという中間領域に適するプロペラとして，両者の形状を
足し合わせたトランスキャビテーティングプロペラが考えられてい
る（図5･50）．
f .空中プロペラ ACV（air cushion vehicle，通称：ホバー
クラフト）や湿地帯用の滑走艇など，没水部をなるべく少なくする
必要のある特種高速艇では，空中プロペラが使われる．航空機のプ
ロペラと違って荷重度が大きいため，羽根枚数が多く，羽根面積も
大きい．名前もファンと呼ばれる．騒音が非常に大きい．
