# 5・4・1 操縦性能

> **章**: 第5章 船舶
> **節**: 5・4 運動性能

a.操縦性能（ship manoeuvring performance) 船の操縦運
動は，二つの並進運動および一つの回転運動，すなわち図5･51で
示す 軸方向の前進運動と 軸方向の横流れ，および 軸回り
の旋回運動の合成で表される．このとき船の操縦性能は，定常旋回
半径の大きさで代表される旋回性能（turning ability），操舵に対
する船体運動の追従性能および針路安定性能（保針性能）で評価さ
れる．針路安定性能（course keeping ability）とは，船体に風や
波などによる外力が働いて船が航路をそれ始めた（船が方向を変え
始めた）とき，外力を取り去ると元の方向に戻る性質のことをい
う．このように外力を取り去ると元の方向に戻る場合を針路安定ま
たは方向安定といい，外力がなくなっても船の方向がさらに元の針
路からはずれていく場合を針路不安定または方向不安定という．一
般に一つの船型では旋回性能と針路安定性能は相反する性質であ
り，針路不安定な船ほど旋回性能は良くなる．
操縦性能を表すものとして，スパイラル特性が一般に用いられ
る．これは，ある舵角に対して旋回するときの定常旋回角速度を示
したもので，この一例を図5･52に示す．横軸は舵角（δ）で，正
が右舵，負が左舵を表す．縦軸は無次元化した定常旋回角速度（′
＝）であり，正が右旋回，負が左旋回を表す．ここで， は
定常旋回角速度， は船長， は定常旋回中の船速である．この
絶対値が大きいほど旋回角速度が大きく，旋回半径が小さくなり，
旋回性能が良いことを意味している．また，舵角零では一つの舵角
に対して三つの旋回角速度が存在することになり，外乱がある場
合，右旋回になるか左旋回になるかわからないことを意味し，針路
不安定であることを表している．この舵角零付近で三つの旋回角速
度が存在する舵角の範囲をループ幅と呼び，この幅の広さが針路不
安定の強さを表している．針路安定な船の場合，このループ幅は存
在しない．タンカーのように太った船の場合，まず針路不安定であ
るが，極端に針路不安定の場合を除いて操船上問題になることはな
い．
一般に大洋を航行する場合は，針路不安定であっても船の方位を
一定に保つ自動操舵装置により操船上の問題はないが，針路不安定
性能が強まり針路補正のための当て舵が大きくなると，燃費に影響
してくる．一方，湾内など狭い航路を船が進む場合は，衝突や座礁
を避けるために，操舵に対する追従性能が重要となり，許容される
針路安定性能にも限界がある．小さな舵角で容易にコースを保てる
船などは，保針性能が良いといわれるが，これは追従性能に多く依
存している．また，針路安定性能の良い船は追従性能も良く，した
がって保針性能も良い．操縦性能の評価は航路の状況によって異な
るが，舵をとればそれに応じて旋回に入り，中央に戻せばすぐに旋
回を止める追従の早さが，操船を容易にするうえで重要である．
操縦性能の理論的取扱方法は，大きく二つに分けられる．一つ
は，自動制御理論のように考え，操舵量を入力とし船の方位角を出
力として表す方法である．もう一つは，連立運動方程式を立て，そ
の係数が船体に働く流体力をできるだけ忠実に表すようにした方法
である．自動操舵装置の設計には前者が有用であり，船型の違いや
舵の違いなどによる操縦性能の違いを評価するうえでは後者が有用
である．後者の計算法で必要となる船体に働く流体力については，
一般には模型試験により求めるが，船を細長体とみなす細長体理論
を用いる方法や，最近では数値流体力学（computational fluid
dynamics ）を用いて推定する方法が開発されている．
b.制御装置 船の制御装置には一般に舵（rudder ）が用いら
れる．舵に発生する直圧力の大きさは舵面積などが大きく影響する
ため，船型と操縦性能を考慮しながら舵面積が決定される ．
舵にも目的によっていくつか種類がある．特殊な舵の中には，
70°という大舵角もとれるシリング舵や，シリング舵2枚を使って
プロペラ後流の向きを制御する舵もある．この種の舵は，離着桟時
の操船性能を改善する目的で，内航船のように出入港離着桟の回数
が多い船に装備される．また，旋回力を増すためにフラップをもつ
フラップ舵もある．さらには，プロペラと舵をいっしょにしたよう
な推進装置を装着した船も増えてきている．
舵以外にも，離着桟時に横移動を行うための補助装置としてサイ
ドスラスタを装備した船もある．サイドスラスタは，船首あるいは
船尾の水面下に真横方向に設けられたトンネルにプロペラを装備し
たもので，真横方向に水流を発生させ，その反力で船を横方向に動
かす装置である．
c .操縦性能に関する国際基準 操縦性能については，国際海
事機関（IMO:International Maritime Organization）で定めた
操縦性能基準（Standards for Ship Manoeuvrability）［Resolu-
tion,MSC.137(76)］があり，極端に操縦性能が悪い船を排除し，
