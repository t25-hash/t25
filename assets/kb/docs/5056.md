# 5・4・2 停止性能（stopping ability）

> **章**: 第5章 船舶
> **節**: 5・4 運動性能

り，図中の記号で，□はタンカー，◇はバルクキャリヤ，○は旅客船（カーフェリー，RO/ RO船を含む），＋は船種不明船を表している．
急速停止操縦を行う際の問題点は，プロペラ逆転を行うことにより方向安定性が低下し，この結果多少とも外乱があると左右どちらに旋回するかわからないことである．
ただ，急速停止で旋回することにより停止距離および停止時間が短縮される場合もある．
船の停止性能は，車でいえばブレーキ性能に相当する．
車の場合，ブレーキを踏めば車の回転数が減少し，最後には車の摩擦により停止する．
しかし，船の場合はそう簡単ではない．
船舶の質量は，車とはけた違いであり，少々の摩擦では止まらない．
ディーゼルエンジンで固定ピッチのプロペラをもつ通常の船の場合，急速に停止するには，プロペラ回転数を減少させ，速度を減少させる．
