# 8・2・1 概説

> **章**: 第8章 エネルギー輸送・貯蔵
> **節**: 8・2 エネルギー貯蔵

社会のエネルギー需要は時々刻々と変化しており、その変化する需要に合わせてエネルギーは供給されなければならない。エネルギーの中で、石油、天然ガス、熱のようにそのままのエネルギー形態で貯蔵できるものは、需要の変動に対して安定に供給することができる。それに対して、太陽光、風力、それに電気はそのままの状態では貯蔵ができないエネルギーであるために、需要の変動に対しては貯蔵できる他のエネルギーに変換して対応することになる。エネルギーを貯蔵する形態には、力学、熱、化学、磁気などがある。貯蔵するエネルギーを考えるうえで基本になるのが、示強変数に示量変数を掛けたエネルギー量である。表8・12は貯蔵できる各種エネルギーについて示強変数と示量変数を示したものである。石油などの化学物質はエネルギー密度(energy density)が最も大きいエネルギーで、貯蔵特性に優れている。エネルギー密度は、化学、高温熱、運動、磁気、重力の順に小さくなる。図8・29は、各種エネルギーの貯蔵特性をエネルギー密度と貯蔵費用とで比較したものである。図8・29の右下にあるエネルギーほどエネルギー密度が高く安価に貯蔵でき、石油と液体水素が最も優れ、次いで化学物質や熱貯蔵が続いている。それに対して、左上の実線で囲った電力貯蔵用のエネルギーとして利用されているものは、貯蔵エネルギーとしてはあまり優れた特性のものでないことがわかる。石油やLNG(液化天然ガス)はエネルギー密度が高い貯蔵特性に優れたエネルギー形態である。石油やガスは、電気と違いそのままの状態で貯蔵することができる。特に液体である石油は、建設費が比較的安価な貯蔵タンクによって蓄えることができる。需要の変表8・12 storage)技術といい、それには貯蔵施設だけでなく変換装置が必要になる。電力貯蔵技術の場合、どのようなエネルギー形態にして貯蔵すればエネルギーの損失が少なく、かつ経済的になるかを考えていく必要がある。電気を利用して石油系燃料を作り、それを再び電気に戻すシステムは、2回のエネルギー変換によって変換効率は20%以下になりエネルギー損失が大きい。石油系燃料は貯蔵特性においては優れているが、変換時の効率が悪いために電力貯蔵技術として利用していくことは経済的に見て難しい。それに対して、電力貯蔵技術として実用化している揚水発電や蓄電池、あるいは実用化が検討されている圧縮空気、超伝導、フライホイール(flywheel)などは、貯蔵特性の点では劣っているが、変換効率は60%以上になる。貯蔵費用が高くても変換時のエネルギー損失が小さいため、全体から見ると経済的になる。電力貯蔵は、夜間などオフピーク時にベース電源の電気でエネルギーを貯蔵し、昼間のピーク負荷時にそれを電気に変換する、いわゆる負荷の移行によって平準化を行うものである。電力貯蔵の導入によって発電設備の設備容量の削減は図られるが、貯蔵設備の容量が追加されるために供給設備全体の容量を減らすことにはならない。負荷の移行による負荷平準化の効果としては、原子力発電のオフピーク時の電力を電力貯蔵用に利用する方法がある。それは原子力発電の連続運転による利用率向上だけでなく、昼間の化石燃料、特に石油とLNG を節約し、資源に乏しいわが国のエネルギーセキュリティを高め、かつ温暖化の原因となっている二酸化炭素の排出量を削減することになる。電力貯蔵には、そのほかに、その優れた負荷応答特性により電源の運用と計画を容易にする役割もある。例えば電圧・周波数調整あるいは停電時の非常用電源として利用でき、また予備力としての働きによって電力系統の信頼性を向上する。そういった働きは、負荷平準化や信頼性向上のために余分にかかっている費用、起動停止時や負荷追従による効率低下に伴う燃料費、電圧・周波数調整設備費といった費用を削減する経済効果もある。表8・14は実用化、あるいは研究開発中の各種電力貯蔵装置の特徴を比較したものである。
