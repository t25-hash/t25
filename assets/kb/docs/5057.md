# 5・4・3 安定性能

> **章**: 第5章 船舶
> **節**: 5・4 運動性能

車の急ブレーキに相当するが，車と違うのはいつでもプロペラを逆転させることができるわけではないことである．
通常の船の場合，緊急停止の発令後，遊転しているプロペラの回転数が十分に減少したところ（最近のディーゼル主機では，最大回転数の30～40%）でエアブレーキをかけてプロペラ回転を完全に止め，それからプロペラ回転を逆転させ，後進推力を出して停止させる．
このため，緊急停止発令後プロペラ回転が完全に止まり逆転させるまでの主機操作時間が，船が停止するまでの距離および時間に対して大きく影響する．
これに対して可変ピッチのプロペラをもつ船は，プロペラ回転の方向を変えることなく，プロペラピッチを変えることで後進推力を出すことができる．
このため，固定ピッチをもつ船より，後進発令から後進推力を発生するまでの時間が短くなり，停止距離（stop-ping distance）が短くなる．
これ以外にも，5･4･1･b項で紹介したシリング舵2枚を一つのシステムとして制御する舵を装備した船は，プロペラによる前進推力を，2枚の舵を閉じるようにして船首方向に流れの向きを変え，後進推力とすることができる．
この舵を装備した場合，プロペラの回転方向を変える必要がないため，後進推力を得るまでの時間が短くて済み，やはり停止距離が短くなる．
また，船が停止するまでの距離および時間は，前述のプロペラ回転が完全に止まるまでの主機操作時間以外に，初速，排水量，主機型式の違いによる後進推力の差に大きく影響される．
