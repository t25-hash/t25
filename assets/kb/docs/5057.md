# 8・2・2 燃料貯蔵

> **章**: 第8章 エネルギー輸送・貯蔵
> **節**: 8・2 エネルギー貯蔵

a. 石油燃料の貯蔵わが国では,原油については,国家備蓄および石油備蓄法に基づく民間備蓄によって原油タンクに大量に備蓄している.国家備蓄は,1998年3月末時点で約5000万kL[原油(crudeoil)の消費量ベースで約82日分]に達している.これらは,表8・15の国家備蓄基地にておもに貯蔵されている.また1976年に石油備蓄法が施行され,民間備蓄が義務付けられた.1980年度末に90日分の備蓄を達成後,国家備蓄基地の完成に伴い民間の備蓄義務は段階的に軽減され,現在は70日分となっている.石油製品については,製油所で原油から精製された製品を消費地まで効率よく配送するための中継基地として全国網で整備された油槽所において貯蔵される.油槽所は,おもに小型タンカー,貨物船表8・15 国家備蓄基地(1998年時点)基地名容量万kL方式完成年むつ小川原(青森県)陸上1985年9月苫小牧東部(北海道)陸上1990年11月白島(福岡県)洋上1996年8月福井(福井県)陸上1986年7月上五島(長崎県)洋上1988年9月秋田(秋田県)半地下1995年7月志布志(鹿児島県)陸上1993年12月久慈(岩手県)地下1993年9月菊間(愛媛県)地下1994年3月串木野(鹿児島)地下1994年5月で受入れを行う臨海油槽所と,タンク車,トラックにより受入れを行う内陸油槽所に分類される.臨海油槽所は全国に約452箇所あり,タンクの総容量は1413万kLで,全油槽所のタンク容量の96%を占める.一方,内陸油槽所は148箇所で,タンク容量は60万kLとなっている(1995年3月末データ).油槽所の主要設備は,受払設備(桟橋,タンク車積み場,タンクローリー積み場)と貯蔵設備(タンク,商品倉庫,屋外貯蔵所)であり,このほかに保安・消防設備を備えている.臨海油槽所では,桟橋が設置され,タンカー側受払口と接続するためのローディングアームが設置される.桟橋の形態は図8・30に示すように岸壁型,床ホーム型,ドルフィン型,フィンガ型に分けられる.またタンクは,屋根構造によってコーンルーフタンク,フローティングルーフタンクおよびカバードフロートタンクに分類される.各々のタンク形態を図8・31に示す.揮発性の高い石油製品には,大気との接触を避けるためにフローティングルーフタンクが使用され,特に雨水の浸入を嫌う場合や降雪地には,カバードフロートタンクが使用される.また,高流動点の石油類を貯蔵する場合は,ボイラ設備を併設し,タンクへの保温処置を講じる.消費者の石油製品の貯蔵手段としての機能を備える容器は,1850年代ごろから木だるが主要な容器として用いられ始めた.その後,1900年代初めにアメリカのネリーブライにより,金属製ドラム缶が発明され,今日の輸送容器の主流を占めるようになった.また,ドラム缶以外の貯蔵容器として,消費者の要望および用途の拡大とともに容量,材質,形状の異なる多種多様の容器が発明されてきた.国内で流通している代表的な貯蔵容器は,円筒形200L鋼管ドラム缶,円筒形20Lペール缶(鋼製およびポリエチレン製),角形18L缶,角形4L缶,円筒形1L缶の5種類である.また民生用の灯油の貯蔵容器は,ポリエチレン製の18L缶が主体を占めている.これらの大量に流通される貯蔵容器については,「容器包装に係わる分別収集および再商品化の促進等に関する法律」に基づき,廃容器類の適正処理の徹底,リサイクル(recycle)を考慮した容器類の材料選定等を実施することにより,廃油,廃容器類等の廃棄物削減に向けた取組みが求められる.b. 天然ガス(natural gas)の貯蔵
