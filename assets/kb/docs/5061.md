c .船酔いの予防と治療
.予防方法 乗物酔いの予防法として確立された方法はな
い．対症療法として，薬物の内服，薬物の貼付が一般的であるが，
生理的・心理的個人差のため，すべての場合に必ず効果を発揮する
とは限らない．乗物酔い発症の予防には心理療法が効果を発揮する
といわれる．
.設計・運航時の対策 旅客船を設計する際には乗り心地
や船酔い対策も重要な設計条件であるが，必ずしも合理的な対策が
とられているわけではない．外洋を航行する大型旅客船や長距離フ
ェリーにおける上級船室は通常船首側の上層甲板に配置されるが，
一般に船体重心より前方は上下揺れ，縦揺れに誘起される上下加速
度が大きく，上層甲板ではさらに横揺れによる水平加速度も大き
い．設計時には，瘠（せき）型船型の採用，動揺固有周期に対する
配慮，動揺軽減装置の配備，航海速力の選定等によって波浪中船体
運動の軽減が図られるが，船体と外洋波浪の相対的な大きさが同程
度であるため船体運動を零にすることはできない ．船体の減揺

5･72に示されるとおりである．設計時の減揺装置の採用は，船種，
用途，航行海域，航海速力等の条件によって異なる．このほかの対
策としては，旅客区画の快適性に配慮した内装がなされる．
小型の旅客船やフェリーの航行区域は通常内海，近海であるの
で，動揺軽減装置が重視されることは少ない．小型の遊覧船におい
て油圧制御によって乗客キャビンの動揺を制御した例，半没水双胴
船型を用いた低動揺型旅客船の例がある．また，高速交通艇におい
ては双胴船型や全没型水中翼船型を採用して波浪中船体動揺を大き
く軽減している．小型の旅客艇や高速艇においては航走時の船体動
揺は比較的小さいが，出入港の低速時，乗客乗降の停船時の動揺が
乗り心地を悪化させる．
