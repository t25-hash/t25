# 1・1・1 緒論

> **章**: 第1章 全体概要
> **節**: 1・1 交通システムの計画と政策

a. 人間の交流を支える交通機能交流は,多くの生物の中で人間というものを特徴づける極めて基本的な活動である.人と物そして情報についての交流という活動が,生産の特化と交易を通じて経済活動を進展させ,人々の相互的な刺激と競争を通じて新たな創造を生み,移転を通じて文化と文明を広く展開させてきたのは,人類史の物語るところである.文字システムが時間を超えた交流を人間にもたらしたのと同じく,交通機能の発展は空間を超えた交流を可能にした.それは同時に交流の場として都市というものを出現させ,さらには階層的な都市群システムを生んだ.交通機能がもたらした空間の構造化である.b. 交通システム(transport system)の定義とその構成要素➡交通サービスを生産する供給システムが交通システムである.その字間ベタに古典的な要素は,入れ物としての1交通具と,その入れ物を乗せるなるよう22交通インフラストラクチャの二つである.前者には車両や船舶,全角を削除航空機などが該当し,後者には道路,鉄道線路や駅,港湾や空港なし改行してどといった交通路や交通ターミナルが該当する.近代的な交通シスありますテムとなると,上記の二つに加えて,円滑で安全な運行を実現するための信号システムや管制システムなどの3運行制御と,運賃収受や座席予約などのソフトを含めた4営業システムが加わる.c. 交通システムの輸送集約性と環境負荷交通システムは,人工動力を使用するか否か,私的所有か公的所有かなどさまざまな視点から分類することができるが,交通計画上最も重要なのは,輸送集約性(まとめて運ぶ程度)の高低によるものである.陸上旅客輸送では自転車→マイカー→パラトランジット→バス→LRT(ライトレールトランジット)・路面電車→中量輸送システム→鉄道の順で,貨物輸送では小型トラック→大型トラック→鉄道→船舶の順で輸送集約性が向上する.自転車のように人工動力を使用しないものを除けば,一般に輸送集約性が高いほどエネルギー消費や環境負荷が小さい(図1・1,1・2).反面,輸送集約性の高い交通システムには,まとめて運ぶという性質上,1ドアトゥドア性に劣り乗換えや積替えの手間とコストがかかる,2まとまった交通需要がないと運行頻度が低下する,3運行の柔軟性や臨機性に劣る,といった弱点がある.このため,需要密度が低いケース,積替えなどのコストが響く短距離輸送のケース,輸送に臨機性が要求されるケースなどでは,鉄道やバスなどの輸送集約性の高い交通手段は市場的には成立しにくい.d. 交通市場(transport market)とその周辺交通サービスの供給機能を担う交通システムと対になるのが交通需要のサイドである.供給サイドと需要サイドを合わせて交通市場が構成される.消費者や企業は,そのニーズに従って交通サービスの良否を判断し,➡均等づめにしてあります図1・1貨物輸送機関の二酸化炭素排出原単位(2000年度)6―1さまざまの選択行動を行って交通需要を発生させる.運賃・料金に速達性や快適性その他の交通サービスの善し悪しを加味した概念を一般化交通費用と呼ぶが,この一般化交通費用の高低によって交通需要も変動する.交通需要の発生様式には二つのタイプがある.一つは本源需要,すなわち交通サービスを消費すること自体が目的となっているケースである.散歩やドライブ,豪華客船のクルージングなどである.もう一つは,交通サービス自体が本来の目的ではなく発生する交通需要である.原料や部品の調達,業務打合せや営業活動は,生産や流通活動の生産要素としての交通需要,買い物は消費活動に伴う交通需要,通勤は労働力供給に伴う交通需要である.これらを派生需要あるいは連動需要と呼ぶ.このような意味で交通市場は,交通以外のさまざまな財やサービスの市場と密接に連動している.また,交通サービスの良否は土地の価値に極めて大きな影響をもたらすため,土地や建物などの不動産市場とも連関している.このように交通需要は,交通ニーズ,提供される交通サービス,人口や経済活動の密度,地理的条件などに応じて発生する.反対に交通需要の質と量に応じて交通システムの在り方が異なってくる.現に存在する交通システムは,基本的にはこうした交通市場のメカニズムを経て生き残ったものといえる.e. 交通サービスの成果と交通政策(transport policy)交通サービスは,交通市場のみならず周辺のさまざまな市場において,経済的な利得をもたらし,あるいは消費者に満足をもたらす.しかし,交通サービスは,そうした市場的な成果ばかりでなく,市場の外部にさまざまな(正負の)影響をもたらす.いわゆる外部効果である.ここでいう外部効果とは,市場を構成する人々の判断や行動に必ずしも十分に反映されない要素をいう.大気汚染や騒音,振動などの地域的な環境負荷や二酸化炭素のような地球環境への負荷,事故を通じての人的あるいは物的な損害,交通混雑などは外部効果の典型である.理想的な状況下では,自由な市場は広義の経済的利得を最大化することができる(いわゆる神の見えざる手)が,こうした外部効果が存在する場合には,市場をあるがままに任せていたのでは最適状態は達成できない.ここに交通市場への公的セクタの政策的な介入の理由の一つがある.これ以外にも公共財(多数者の消費が競合せず,同時に負担拒否者の消費を排除しがたい財・サービス)の供給,弱者への配慮なども公的政策介入の理由となる.g. 交通政策の動向交通政策としてはどのようなものがあるのか.第1は,安全や環境,弱者保護の視点からの公的な技術的コントロールである.各種の技術基準や環境基準,バリアフリー基準などはこれに相当する.第2は,何らかの政策的意図をもった公的助成である.地下鉄整備に対する公的助成や,海運のような環境にやさしい交通機関の利用促進(モーダルシフト)を目指した公的助成などはその典型である.第3は,同じく何らかの政策的意図をもった公的抑制である.有名なシンガポールのマイカー保有抑制政策や,都市部におけるマイカーの利用抑制を意図したロードプライシングなどに代表される交通需要マネジメントなどはその典型である.第4は,道路整備や港湾整備に代表される公的供給(公共事業)である.前述の公共財や,投資規模や投資リスクの大きさ,非常に長いプロジェクト期間などが公的供給の根拠となる.しかし,従来は公的供給が普通であった分野でも,現在は公的セクタと民間セクタの連携(プライベートパブリックパートナーシップ,PPP)方式が着目されている.このほかにも,事業の安定的継続などを根拠とするいわゆる公的需給調整規制(市場参入規制や退出規制),弱者保護の立場からの公的運賃料金規制なども交通政策の重要な部分をなしてきた.しかし,近年では事業経営の効率化や政府の介入による非効率の発生を排除するため,事業関係については規制緩和を進めることが世界的な潮流となっている.h. 交通システムの計画プロセス交通システムの諸要素を需要面や経済社会,環境への影響を念頭に入れて計画するプロセスは,通常,構想段階→概念計画段階→詳細設計段階といった経過をとるが,多くの場合,政府や地方自治体の何らかの公的計画,特に都市部では都市計画上の計画決定を伴う.これは,交通システムが地域社会に極めて大きな影響を与えることから,公共公益的な視点からの十分なチェックが必要であることや,その公益性に鑑みて交通プロジェクトの実施に対して,用地上の私権制限などを付与することが必要となるためである.また,特に公共事業や規模の大きな公益性の高い交通プロジェクトの場合には,一般市民やユーザの意見を計画や設計に取り入れる市民関与と呼ばれるプロセスがとられることが多くなっている.計画の際には,交通需要予測とプロジェクト評価という作業が非常に重要となる.前者は,社会環境や交通プロジェクトがもたらす交通サービス条件に応じて発生する利用需要を推計することをいう.その結果は,1交通システムの設計内容の妥当性を検証する,2交通システムの事業採算性を検討する,3交通システムのもたらす社会経済的効果をプロジェクトの実施に必要な費用と比較考量する,4交通システムのもたらす環境影響と環境対策の妥当性を検証する,などの作業に用いられる.このうち,2~4を広義のプロジェクト評価と呼んでいる.以下では,まず1・1・2項で交通需要の予測方法を,1・1・3項で交通プロジェクトの評価方法を概説する.
