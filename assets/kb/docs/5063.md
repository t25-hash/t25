# 1・1・2 交通需要の予測

> **章**: 第1章 全体概要
> **節**: 1・1 交通システムの計画と政策

a. 交通需要予測(transport demand forecast)の基本的な考え方交通政策や計画案の工学的妥当性、経済的有効性、事業の採算性等をチェックまたは比較検討するうえで、施策実施に伴う交通量の変化の予測は欠かせない作業である。交通需要予測に当たっては、対象とする交通需要の特性を表現できる計量モデルが用いられる。かなりの数のモデルが提案されているが、政策や計画案の目的、対象地域の特性、目標年次、入手可能なデータの精度等を勘案して、適切なモデルを選定することが重要である。交通需要予測の標準的な方法は次のとおりである。(1) 予測対象の選定検討すべき政策や計画に関して、事前に想定される影響の大きさや範囲、種類等に応じて、対象とする地域、期間、旅行者(あるいは貨物)、交通機関等を決定する。(2) モデルタイプの選定モデルの論理性、操作性を考えながら必要最小限で交通需要動向を表現できるものを選ぶ。モデルに取込む変数は、政策・計画の代替案間で異なる重要な要素(政策変数と呼ばれる)を中心に、必要最小限な要因に絞る。(3) モデルの推定公表される各種統計データや独自の調査を行って得られるデータから、モデルのパラメータを統計的に求める。モデルで仮定される交通行動のロジックに応じて、適切な推定方法を用いる。(4) モデルの検証モデルの現況再現性、他地域や他時点への移転可能性、政策変数の感度等の点から、統計的検定も含めて多角的にモデルの妥当性を検証する。(5) モデルを用いた予測モデルに他の地域や時点のデータを入力して、予測値を求める。b. 交通需要予測モデルの概要交通行動は、人間の種々の営みとは切り離して考えられない事象だが、そもそも人間のアクティビティは無限に多様である。また交通行動だけを取出しても、移動者の個人属性、空間特性、時間特性、手段属性等、かなり高次元の特性をもつ現象である。したがって、これらのすべてを取扱うのは不可能であるし、また政策や計画の検討を行ううえで無意味なことも多い。そこで交通需要予測において、交通行動は、特定の範囲に限定されるとともに、論理性が保持される範囲内で適宜分割され、段階的に取扱われる。その最も典型的な方法が、交通行動を四つの意思決定ステップとして取扱う4段階推定法である。ここでは、一連の交通行動が、発生・集中(出発するか否か)、分布(どこへ行くか)、分担(どの交通機関を使うか)、配分(どの経路を使うか)という四つの段階からとらえられる。4段階推定法では、発生・集中以外の3段階の行動は、いずれも、上位段階の行動結果を制約として次を選択するものとみなされる。したがって、予測作業においても、交通需要を上から順に予測し、上位段階から得られる結果を所与として次段階の需要を求めるというステップを踏む。なお、4段階推定法はあくまでも一つの考え方であって、例えばこれらの段階を統合した予測モデルも提案されている。また、必ずしもすべての段階をモデル化する必要はなく、分析目的や対象に応じて、特定の段階のみに着目し、他の段階の需要に何らかの仮定をおいて予測することが多い。c. 4段階推定法の各モデルの概要。 発生・集中交通量予測モデル(trip generation/attrac-tion model)各ゾーンから出発する、あるいは各ゾーンに到着する交通量を予測するモデルである。各ゾーンの人口指標や経済指標が説明変数として用いられる。多くの場合、計量経済モデル等を援用した、ゾーンを単位とする集計モデル(例:重回帰モデル)が用いられる。. 分布交通量モデル(trip distribution model)各ゾーンから発生した交通が、どこに行くかを予測するモデルである。ここで、発生側のゾーンのことを発ゾーン(origin)、集中側を着ゾーン(destination)、これらの組合せをOD ペアと呼ぶ。分布交通量の予測には、ゾーンを単位とする集計モデルが用いられることが多い。この代表的なものとしては、重力モデルが挙げられる。これは、物理現象において重力が二つの物質の重量と両者の距離の関数によって表現されることを模したものであり、욡욢=욡욢욡욢により求められる。ここで、욡욢:間の分布交通量、욡:発ゾーンの発生交通量(上位モデルから求められる)、욢:着ゾーンの集中交通量(上位モデルから求められる)、욡욢:間の交通抵抗(距離や所要時間等の関数)、,α,β,γ:係数である。なお、実際には、政策や計画の実行によっても交通量がほぼ変化しないと考えられるOD ペアについては、分析時点(現状)での分布パターンと同一という仮定を設ける方法(現在パターン法)がよく適用される。この場合、重力モデルは、発ゾーンあるいは着ゾーンで交通量に大きな変化を生じさせる事態が想定されるOD ペアのみに適用される。. 分担交通量モデル(modal split model)OD ペア間の交通について、どの交通機関を用いるかを予測するモデルである。ゾーンを単位とする集計モデルと個人を単位とする非集計モデルの両方があるが、近年は非集計モデルが多く用いられる。集計モデルとしては、分担率曲線モデルや犠牲量モデルが挙げられる。分担率曲線モデルは、各交通手段の利用率をOD ペア間の距離や時間の関数として表現する回帰式である。犠牲量モデルは、利用者の時間評価値(特定の時間に対する金銭的価値。時間価値ともいう)に分布を仮定し、異なる時間評価値をもつ利用者集団が、各時間評価値に応じた犠牲量(交通による損失を表す言葉。一般化費用と同義)の最小となる交通手段をそれぞれ選択すると考えるモデルである。一方、近年では、個人の意思決定過程を明示的に考慮できる非集計モデルがよく用いられている。これは、経済学で仮定される効用最大化原理と整合的な非集計モデルが提案されて費用便益分析等への理論的拡張が可能となったこと、計算機性能が向上してパラメータ推定が容易になったことなどの理由による。非集計モデルでは、まず個人の効用(utility)を基礎におく。ここで効用は、交通サービス特性(OD ペア間の時間や運賃など)や旅行者の個人属性の関数として表される。次に、効用に対してランダム効用(効用は、確定的効用と特定の確率分布に従う誤差との線形和によって表されるという考え方)を仮定する。そのうえで、人間は、対象とする交通サービスについて、選択可能集合の中から最も効用が大きくなる選択肢を選ぶ(効用最大化原理)と考える。非集計モデルとして多くのタイプのモデルが提案されているが、最も一般的なのは非集計ロジットモデルである。非集計ロジットモデルでは、効用の誤差に対し、選択肢間で無相関かつ同一のガンベルの二重指数分布が適用される。このとき、個人の選択肢の6―3選択確率は、욡욈=exp욡욈∑exp욢욈のように求められる。ただし、욡욈:確定的効用、Ω욈:個人の選択可能集合である。ここで、効用関数中の未知パラメータは、アンケート調査等によって得られる個人の行動データと交通サービス特性データを基に、最尤推定法によって推定される。推定されたモデルは、同一特性をもつと考えられる利用者集団に拡大して適用され、ゾーン単位の交通量に集計化されることにより、ゾーン単位の予測交通量が求められる。. 配分交通量モデル(traffic assignment model)複数の経路が存在するとき、ある交通機関利用者がどの経路を選択するかを予測するモデルである。集計モデルと非集計モデルの両方が用いられる。集計モデルはさらに、経路選択の均衡状態を仮定するか否か(均衡型、非均衡型)、混雑が経路選択に与える影響を考慮するか否か(フロー依存型、フロー非依存型)、利用者の確率的な分布を考慮するか否か(確定型と確率型)等で、いくつかに類型化できる。代表的な配分方法は、次のとおりである。・最短経路一括配分法想定される経路のうち、所与の旅行コストが最小となる経路にゾーン間交通量をすべて配分する方法(非均衡型×フロー非依存型×確定型)。・分割配分法ゾーン間交通量を複数に分割し、各経路の交通量と混雑による旅行コストの増大を考慮して、順に交通量を配分することにより旅行コストを更新していく方法(非均衡型×フロー非依存型×確定型)。・確定的利用者均衡配分法すべての利用者が、混雑による旅行コストの増大を考慮しつつ、旅行コストが最小となる経路を選択することによって、それぞれの利用者の選択可能なすべての経路において旅行コストが同一になる均衡状態となるように、交通量を配分する方法(均衡型×フロー依存型×確定型)。非集計モデルも用いられるが、経路は選択可能集合が膨大であるにもかかわらず経路集合を特定しなければならないという問題があり、経験的に選択肢集合の特定が比較的容易な鉄道経路選択等に、実際の適用は限定されている。なお、近年では、計算機の能力の向上に伴って、地理情報システムやシミュレーションを用いた動的な予測モデルも活用されるようになっている。
