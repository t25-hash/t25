# 1・1・3 交通プロジェクトの評価

> **章**: 第1章 全体概要
> **節**: 1・1 交通システムの計画と政策

a. 交通プロジェクト評価(transport project evaluation)の種類広義のプロジェクトの評価には、1事業の財務的評価、2事業の経済的評価、3環境影響評価(環境アセスメント)、などが含まれる。1の財務的評価は、交通プロジェクトの事業実施者の収入(運賃料金収入や公的助成)が、初期投資費用や金利負担、および営業開始後の経費を十分に賄いうるかどうかという視点からの評価である。独立採算が前提とされる公営事業や私企業による交通プロジェクトではどうしても必要となる基本的な条件である。2の経済的評価は後述する。3の環境影響評価は、わが国では通常、事業の実施段階において行われ、事業のもたらすさまざまな環境影響、大気、騒音振動、動植物、土壌、水質などへの影響を理学的・工学的に予測し、あらかじめ定められた環境基準への適合性を検証するものである。この事業段階の環境影響評価は、わが国では近年ようやく正規に法制度化されたものである。なお、諸外国では計画段階においても環境影響評価に類する手続きが制度化されていることが少なくない。b. 交通プロジェクトの経済的評価経済的評価とは、財務的評価とは異なって、各種の金銭的利得や損失のみならず、利用者の満足や不快、環境上の効果や負荷、事故や災害に伴う損害などといった、プロジェクトがもたらす多岐にわたるインパクトを本来の評価対象としている。これらの各種インパクトを評価技術的に可能な範囲で金銭換算して、プロジェクトの投資費用と比較考量する技法を費用便益分析と呼んでいる。また、金銭換算した一次元的評価の編交通機械―4γ6みならず、より多様な評価視点を取り入れた評価技法を広義の費用対効果分析と称している。前者では、プロジェクトライフを30年から50年とし、その間に生じる各種の費用や便益を、現在の価値と将来の価値の違い(つまり、ある価値を利用する機会の大小)を反映するため、社会的割引率を用いて現在価値に換算してその相対的大小を比較するのが普通である。なお、費用とは通常、プロジェクトの実施に伴って必要となる投資費用を指し、便益とは、プロジェクトを実施する場合と実施しない場合で生じる各種の利得や損失の差を意味している。評価の尺度としては、費用と便益の比(費用便益比、cost benefitratio)などを用いることが多い。交通プロジェクトの場合に生じる便益としては、利用者が享受する直接的な便益である利用者便益[user benefit(時間短縮効果、交通費用の低減、快適性の向上など)]とプロジェクトの事業者が受取る供給者便益(運賃収入と運営費用の差など)の直接便益、環境改善(悪化)や交通安全の効果などの外部効果を算定し合算するのが普通である。金銭換算する際には、交通行動分析などの研究の蓄積から推定された原単位が用いられる。これ以外にも、プロジェクトが実施される地域での交通利便性向上に伴って生じる地域経済上の効果(間接効果)なども発生するが、算定の際には利用者便益などと重複して計算することのないよう注意が必要である。便益の大宗を占める利用者便益の計算法についてはc項に紹介する。費用便益分析は、金銭換算された一次元評価であるため、明快でわかりやすく実務でも多用されているが、環境影響や人の命など金銭換算に当たっては相当大胆な仮定をおかざるを得ないことや、重要ではあるが金銭換算が困難なため無視している事項もあるなど、適用には注意が必要である。こうした欠点を補てんするため、より広義の費用対効果分析が行われるのが一般的である。これは必ずしも金銭換算に固執することなく、各種の定量評価(例えば交通事故率の変化)や定性評価(例えば美観変化の大小)を費用便益分析の結果と併用するのが普通である。なお、いろいろな多次元的評価指標を総合化するための手法として、階層的意思決定プロセス(アナリティカルハイアラーキープロセス、AHP)などといった統計学的手法も整ってきている。c. 利用者便益の推定方法交通プロジェクトの便益の非常に多くを占める利用者便益は、おおむね以下のような方法で推定されることが多い。まず、利用者にとっての一般化費用(generalizedcost、あるいは交通不効用)という概念を導入する。これは、所要時間や乗換えの利便性、快適性などといった交通サービスの善し悪しと、運賃料金のような金銭的負担とを加味した変数である。具体的な関数型や未知パラメータは、交通調査データや交通行動の分析などを通じて推定される。関数型は多くの場合、線形の関数が用いられる。また、この一般化費用上、所要時間の項を金銭換算する係数を時間価値または時間評価値と呼んでいる。次に、プロジェクトを実施する場合の一般化費用と交通需要をそれぞれと、プロジェクトを実施しない場合のそれをととする。これらは、交通需要予測の作業を経て推定される。するとこの場合、このプロジェクトによってもたらされるであろう利国の環境問題である。図1・5
