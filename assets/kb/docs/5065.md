# 5・6・4 ドップラーソナー

> **章**: 第5章 船舶
> **節**: 5・6 装備品

ドップラーソナー（Doppler sonar ）は海底に向けて斜めに超音
波ビームを発射し，海中および海底からの反射信号の周波数を計測
することによって，船速を測定する装置である．図5･78に3ビー
ム方式の例を示す．ビームは船の前方向（ビーム1），右後方向
（ビーム2），左後方向（ビーム3）に発射され，これにより船の前
後，左右の速度を知ることができる ．
現在では単に船速計というだけではなく，多くの機能が付加され
てきている．ドップラーソナーによって対地船速や対水船速が計測
されるが，これらの差をとれば潮流の情報が得られる．このように
して，現在では，船速，真針路，船首方位，潮流，風，キールクリ
アランス，エンジン回転数，回頭角速度，緯度，経度，航程（積算
距離）などを一つの表示器で表示するようになっているものもあ
る．
