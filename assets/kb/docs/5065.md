# 1・2・1 交通機械の恩恵と、環境・エネルギー問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

機械の発明はわれわれに便利で快適な生活をもたらしてくれたが、交通機械はその中でも、特に人間にとって必要性が高い。その理由は、人間が2本足で歩くようになったおかげで移動という能力が4足歩行の動物に比べて劣ってしまったことによる。そのために、交通機械はこれまで大きな発展を遂げ、世界の隅々まで、その恩恵をもたらすことになった。しかし、それがゆえに環境とエネルギーの面でわれわれは深刻な問題を背負うことにもなった。交通機械をいままでのように使い、さらにわれわれの欲求を適切に満たしていくために、環境とエネルギーの問題を乗り越えることは必須である。
