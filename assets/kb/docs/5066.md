# 1・2・2 環境問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

1960年代の高度成長期の時代から,日本では環境問題が顕在化した.当時は公害問題(environmental pollution issue)といわれ⬅ていたが,次第に問題が広がり,深刻さの度合いが増す傾向にある.に均等づめ現在,環境問題と呼ばれている問題は,大きく三つに分類されすまりしてある.それらは地域の環境問題,環境リスク,および地球の環境問題である.地域の環境問題(environmental problems in region)とは,かつて公害問題と呼ばれていたものとほぼ同じ問題を指す.これらは,いわゆる典型7公害に相当する.この問題に含まれるのは,大気汚染,水質汚濁,土壌汚染,騒音,振動,悪臭,地盤沈下の各問題である.これらのうち,いくつかの問題は多くの努力の結果,沈静化している.しかし,大気汚染,水質汚濁,騒音については多くの施策や企業の努力にもかかわらず改善の方向に向かっていない.大気汚染の原因は,大量の硫黄を含む石炭や,重油の燃焼による亜硫酸ガスと,燃料を高温で燃焼させたときに発生する窒素酸化物が主であり,さらにディーゼル車が排出する黒煙もある.1980年代の半ばになると,さらに環境リスク(environmentalrisk)という考え方が一般化し始めた.これは新たに人間社会で利用されるようになった化学物質等が,人間を含めた自然環境に与える影響のことである.これが従来の公害と異なるのは,ごく微量でも影響があったり,急性影響,慢性影響はもとより,世代を超えて影響する可能性があることによる.その代表的な物質として,ダイオキシンが挙げられる.PCB(ポリ塩化ビフェニル)のような化学製品,あるいは農薬の一部も含まれる.これらの物質は,問題が明らかになるたびに生産や販売が中止されるなどの対策がとられてきたが,年々使われるようになる化学物質の数の増加に危険性の評価が追いつかないという状況があった.このように,環境リスクの深刻さが少しずつ増してきていた状況の中で,新たに内分泌かく乱化学物質(endocrine disruptor,通称環境ホルモン)の問題が浮上した.これは,ある種の化学物質が人間の体内で働くホルモンと同じような働きをするというもので,その影響で,人間の精子数が減る,子宮内膜症などの病気の原因になる,子供の発育が阻害されるなどの可能性が心配されている.1988年を境に世界中で認識されるようになったのが,地球温暖化(global warming)である.この問題は,原因と影響が国境を越え,あるいは地球全体にまで広がっている環境問題と定義されている.現在この中には,九つの問題が含まれている.これらを列挙すると,地球温暖化,酸性雨,オゾン層破壊,熱帯林の減少,砂漠化,野生生物の絶滅,海洋汚染,有害廃棄物の越境移動,発展途上表1・1 各環境問題の分類定義具体的問題씗典型7公害>大気汚染水質汚濁土壌汚染騒音振動悪臭地盤沈下地域の環境問題地域に限定された環境問題で,かつて公害問題と呼ばれた環境リスク新しく利用されるようになった化学物質などが,人間を含む生態系や環境に与える影響有害化学物質内分泌かく乱化学物質(環境ホルモン)地球温暖化酸性雨オゾン層破壊熱帯林の減少砂漠化野生生物の絶滅海洋汚染有害廃棄物の越境移動発展途上国の環境問題地球の環境問題原因と影響が国境を越え,あるいは地球全体にまで広がっている環境問題これらの問題のうち,地球温暖化は温室効果ガスの働きにより,地球全体が温暖化に向かうと予想されている問題である.ここでいう温室効果ガスとは,地球から宇宙空間に向けて放散される赤外線を吸収する働きのある気体である.その中で,温暖化に最も大きく寄与するとされているのが二酸化炭素であるが,そのほかにメタン,亜酸化窒素,オゾン,フロン等が挙げられている.地球が温暖化すると,海面上昇や降雨への影響のような一次的な影響と,それにより引き起こされる生態系への影響,農業への影響,人間の健康への影響などが心配されている.二酸化炭素を発生させる最も大きな要因は石油,石炭,天然ガスなどの化石燃料である.酸性雨(acid rain)は大気汚染物質である窒素酸化物や亜硫酸ガスが大気中で酸化され,硝酸や硫酸となって地上に降下する現象である.その結果,植物が枯れる,水中の魚や小動物が死滅するなどの影響が出る.オゾン層破壊(ozone layer depletion)は,フロン,ハロンなど塩素や臭素を含む有機化学物質が,成層圏で太陽の紫外線によって分解されるときに発生する塩素や臭素が,成層圏に存在するオゾンを連鎖反応的に破壊する現象である.これらの環境問題の定義と具体的問題をまとめて表1・1に示す.
