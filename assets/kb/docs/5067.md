# 5・1・1 はじめに

> **章**: 第5章 製鉄機械
> **節**: 5・1 総論

世界の鉄鋼業は20世紀後半になり，生産技術，設備ともに飛躍
的に発展してきた．鉄鋼生産量（steel production）も急激な増加
を示し，1980年以降は年産7～8億t （粗鋼）の水準となっている．
日本では1973年に年間生産量1億2千万t （粗鋼）を記録したが，
1980年代に入って生産量はやや減少したものの，技術面では鉄鋼
製品の高品質化，製造コストの低減，省力化が積極的に進められ，
今日に至っている．図5･1に世界主要国の生産高推移，表5･1に
2000年の日本の生産高を鉄鋼製品別に示す．
鉄鋼製造プロセス（steel making process ）は大別すると，鉄鉱
石から銑鉄を製造する製銑，溶銑または鉄くずから鋼を製造する製
鋼，鋼片から最終製品を製造する圧延，表面処理からなる．
鉄鋼製造プロセスは，1970年以降の製品の高品質化，製造コス
トの低減，生産能力の向上などの目的でプロセスも変革をとげ，特
に製鋼での連続鋳造設備の採用，圧延工程でのコンピュータ制御に
よる製造ライン制御，表面処理での表面加工技術の進歩は著しい．
