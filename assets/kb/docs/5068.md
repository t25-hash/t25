# 1・2・4 廃棄物問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

廃棄物問題は、かつては伝染病予防のような衛生の問題ととらえられてきた。その量の増加と質の変化に伴い、環境やエネルギー問題と関連をもつようになり、その影響の大きさ、深刻度の点から、社会問題として広く認識されるに至っている。廃棄物とは、一般に不要となったもののことを指す。廃棄物は何らかの処分がなされるが、そこに不適切なことがあると、それが問題になる。廃棄物の処分で一般的な方法は焼却、埋立て、リサイクルである。焼却の過程で何らかの排ガスが出れば、大気汚染、環境リスクのそれぞれを引き起こし、さらには酸性雨のような地球環境問題に発展する可能性がある。埋立処分では地盤沈下、土壌汚染、水質汚濁の問題につながった例もある。適正なリサイクルがなされずに廃棄物が放置されることによる問題の例も数多い。リサイクルとエネルギー問題の関係では、廃棄物が適正にリサイクルされれば、新しい材料を作り出すためのエネルギー消費を大いに減らすことができる。焼却の際に出る熱を有効に使うことをサーマルリサイクルと呼ぶが、これもエネルギー消費を抑える効果がある。
