# 1・2・5 交通機械と環境問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

交通機械は過去において多くの環境問題の原因となってきた。そのために多くの対策が試みられてきた。しかし、技術的にその対策が容易でなかったことと、交通についての需要が経済の成長を上回る速度で増えてきたこともあり、必ずしも問題が解決に向けて改善されているわけではない。では、交通機械が及ぼしている環境の問題にはどんなものがあるか。まず、地域の交通問題では騒音(noise)と大気汚染(air pol-lution)がある。騒音については航空機、鉄道、自動車のそれぞれで問題を抱えている。航空機は離発着時に飛行場周辺に大きな被害をもたらすことがある。このため、航空機エンジンの低騒音化、離陸時の急上昇による騒音被害が及ぶ範囲の縮小、夜間の離発着の制限、飛行場周辺の家屋の防音対策などが講じられてきた。鉄道騒音については、かつての蒸気機関車の時代には在来線沿線において問題が顕著だったが、電化と鉄道線路の改善によって沈静化している。しかし、高速で走行する新幹線については問題が解決したとはいえない。この問題については、線路わきへの防音壁の設置、線路とまくら木との間への防振マットの挿入、車体の空力特性の向上、パンタグラフと架線の間で発生する騒音への対策等がとられてきた。また、住宅密集地での減速も行われている。自動車騒音はおもに幹線道路周辺で問題が発生してきた。特に大型トラックによる影響が著しい。これに対して自動車側の対策として、エンジン騒音の低減、低騒音タイヤの採用等が行われてきた。6―6交通機械図1・6交通部門の二酸化炭素排出量の推移また、道路側の対策として、防音壁の設置、複数の車線の道路での大型トラックの中央路線の走行への規制、最高速度の制限などが行われている。大気汚染については、自動車から発生する窒素酸化物とディーゼル排ガス中に含まれる黒鉛が主要な問題である。都市内で発生する窒素化合物は、その80%までが車によるものとされている。これらの排ガスを減らすためにとられてきた手法は、車に対する排ガス規制の強化と、その規制値を満たす車の開発、信号制御、大型車の住宅密集地からのう回規制などである。環境リスクと交通機械の関係では、自動車より排出される排ガスに関しての影響が懸念されている。例えば、燃料中に含まれるベンゼンはエンジン中で燃焼されずに大気中に放出されると肺がんの原因になる可能性がある。また、ディーゼル黒煙の中に含まれるペンゾピレンやニトロアレーンなどの化学物質も、肺がん等への影響が懸念されている。地球環境と交通機械の関係では地球温暖化、酸性雨、オゾン層破壊の各問題が影響する。地球温暖化ではその原因物質としての二酸化炭素(carbon dioxide)、メタンガス、オゾン、フロン、亜酸化窒素のそれぞれが交通機械から発生する。まず、二酸化炭素は化石燃料の消費によって発生するため、自動車、航空機は直接これを発生して走るし、鉄道では発電の際に化石燃料を使うことにより間接的に発生する。その発生量は交通機械全体で全排出量の約25%、自動車のみで20%の影響力をもっている。交通部門の二酸化炭素排出量の推移を図1・6に示す。年々増加し、1998年度には7020万tの二酸化炭素を排出し、日本の二酸化炭素排出量の約22%を交通部門が占めるに至っている。メタンガスと交通機械の関係は現在のところほとんどない。オゾンは成層圏に存在するものと対流圏にとどまっているものとがあるが、成層圏のオゾン層はむしろ破壊が問題になっており、今後これが増える要因は考えられない。対流圏のオゾンは大気中に存在する有機物が太陽の光で分解されてOH ラジカルなどの反応性の強い物質ができ、これがさらに酸素と結合するような化学変化によって発生する。この有機物の発生源としては、車の排ガス中に含まれる未燃焼ガスがある。フロンはかつてクーラの冷媒として使われており、交通機械のエアコン装置にも大量に使われていた時代があった。これらは現在では使用禁止となっており、大気中で分解が早く進む代替フロンに置換えられている。亜酸化窒素は燃料を高温で燃やしたときに発生するが、現在では自動車のエンジンからのものがほとんどである。このように、温暖化と交通機械とは密接な関係がある。そのうち、二酸化炭素の影響が最も大きく、対策が急がれるのはこの分野である。酸性雨と交通機械の関係では、車が排出する窒素酸化物が硝酸に変化して被害をもたらす可能性が指摘されている。オゾン層破壊については温暖化問題と同様、フロンガスの大量放出が原因であるが、現在はその対応は進行中である。かつて、超音速飛行機が成層圏を飛行するとき、エンジンから発生する窒素酸化物がオゾン層を破壊するのではないかとの懸念があったが、その後の研究によってこの問題は大きくないことが確認されている。
