# 1・2・6 交通機械とエネルギー問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

エネルギーと交通機械は極めて密接な関係がある.産業革命の時代に鉄道が広く普及するに至ったのは,イギリスをはじめとするヨーロッパで大量の石炭を産出することができたからであり,20世紀に自動車が世界中に普及したのも,初めにアメリカで,次いで中東を中心に大量の油田が見つかったことによる.しかし,これらの資源の有限性が,その将来に及ぶ利用を制限する可能性がある.これまでの交通機械とエネルギー問題について特徴的なことは,そこで使われる発動機のそれぞれが固有の燃料を必要とするということである.オットー型内燃機関エンジンはオクタン価の高い揮発油,小型のディーゼルエンジンはセタン価の高い軽油,航空機燃料はガソリンと軽油の中間の灯油に近い性質のものが使われる.このため,全エネルギーに占める交通機械の寄与率は10%,全化石燃料に占める割合は25%であるが,全石油消費量に占める交通機関が消費する割合は50%にのぼる.二酸化炭素による温暖化とエネルギー問題はいずれも化石燃料の大量消費がその原因であるから,これらの問題に対する対応策も共通の手法が用いられる.それらの方策は,よりエネルギー消費の少ない技術に置換える,他の手段に変える,ルールや運用法の変更により輸送量を減らすということに大きくまとめられる.新しい技術に置換えることの例として,蒸気機関車がディーゼル車に替わり,さらに電車に替わったことが挙げられる.他の手段への置換えとしては自動車輸送を鉄道や船に置換えるモーダルシフトが代表例である.輸送量を減らす手段としては共同輸配送のシステムなどがその例に当たる.
