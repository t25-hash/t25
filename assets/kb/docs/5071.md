# 5・12・1 自動化の種類

> **章**: 第5章 製鉄機械
> **節**: 5・12 制御装置

製鉄設備で自動化の発達を支えてきたものは，機械設備と制御装
置（control system)がバランスよく融合し，さらに製鋼と圧延技
術のサポートによるところが多い．
自動化システム（automation system）の構成を計画する場合，
設備の規模，制御機能の質と難易度および操業方法などによって，
機能分担やシステム機器の台数，さらに設備効率の検討がなされ総
合的に最適システムが構成される．制御システム構成例を図5･75
に示す．
大規模プラントとして，ホットストリップミルまたは，コールド
ストリップミルの設備が代表的である．プラントが大規模になった
場合でも，管理，監視，操業，保全等が効果的に運用されるよう
に，プロセスの情報（例えば，製品の搬送位置や温度，厚さ，形状
等の計測値）は，ネットワーク経由で任意の監視制御点からその情
報を活用できることが大きな特長となっている．
