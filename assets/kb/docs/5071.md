# 1・2・7 交通機械と廃棄物問題

> **章**: 第1章 全体概要
> **節**: 1・2 交通機械と環境・エネルギー

交通機械と廃棄物問題で密接な関係があるのは,その機械が寿命を迎えたときの処理の部分についてである.特に自動車は年間おびただしい数の廃棄車両が発生する.かつてはこれらは各部品ごとを丁寧に分解し,利用できる部分は中古部品として市場に流れ,金属のように資源としての価値のあるものは回収業者の手によってリサイクル(recycle)が行われていた.しかし,現在では廃棄される車の量が多すぎることと,相対的な価格の低下によって部品として利用できるものが減っていることから,その処理の方法も変化している.一般的処理方法は,廃棄車両をいったんプレスにかけて小型化し,これを1箇所に集めてシュレッダと呼ばれる粉砕機に投入して小片に分解したうえで,金属と非金属に分離する方法がとられている.このうち金属はさらに細かく種類ごとに分けられてリサイクルにまわされ,非金属は埋め立てられるものが多かった.ところが,最終処分場の問題が各地で発生していることから,非金属のうち,プラスチックなどの燃える部分については焼却処理をし,ガラスの破片などの燃えないものだけが最終処分をする方向に変わりつつある.また,設計のときから廃棄物を減らすための試みがなされている.例えば,バンパなどのプラスチック製品を対象として,部品としての再利用を容易にするための取外しのしやすさを向上させたり,部品ごとに材料を明記し,リサイクルを容易にするなどの技術開発が行われていたりする.今後は車体の長寿命化,軽量化,リサイクルしやすい材料の選択などの方向への努力が続けられていくことになるだろう.
