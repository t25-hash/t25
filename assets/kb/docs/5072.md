# 1・3・1 交通機関別の事故死者数等の推移

> **章**: 第1章 全体概要
> **節**: 1・3 交通機械と安全(事故)

交通機関別事故死者数の推移を図1・7に示す.2002年には道路交通事故(traffic accident)による死者数は8326人と他の交通機関に比べて最も多く,次いで鉄軌道交通344人,海上交通321人,航空交通13人となっている.道路交通事故は1970年をピーク図1・8 国内旅客輸送の交通機関分担率の推移図1・9国内貨物輸送の交通機関別分担率の推移(16765人)に減少し,1979年から緩やかな増加に転じ,1992年以降はまた減少の傾向にある.鉄軌道交通は全体的に減少の傾向にあり,前半の減少率が大きい.海上交通は全体的に減少傾向にある.航空交通は全体的に死者数は少ないが,1971年(雫石事故),1985年(御巣鷹山事故),1994年(名古屋空港事故)のように大型旅客飛行機の衝突・墜落が発生すると重大事故となっている.次に,国内旅客輸送の交通機関分担率の推移を図1・8に示す.鉄道の分担率は1950年に約90%を占めていたが減少し,1998年には約28%にまで低下した.これに対して自動車は年々増加し,1980年には鉄道と逆転し,1998年には約62%にまで増加した.バスは1965年をピークに減少し,航空の分担率は低いものの増加傾向が6―7図1・7 交通機関別交通事故死者数の推移図1・10 国内旅客輸送の距離帯別機関分担率続いている.また,国内貨物輸送の交通機関分担率の推移を図1・9に示す.鉄道の分担率は1950年に50%を超えていたが,その後著しく低下し,1997年には約5%にまで低下した.これに対して自動車(トラック)の増加が著しく,1997年には約50%に至っている.内航海運は変動があまりなく,約40~50%で推移している.航空は極めて低い.国内旅客輸送の距離帯別交通機関分担率を図1・10に示す.500km未満の短距離では自動車が,500~750km未満の中距離ではJR が,750km以上の長距離では航空が旅客輸送の多くを分担している.
