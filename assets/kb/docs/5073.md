# 1・3・2 自動車

> **章**: 第1章 全体概要
> **節**: 1・3 交通機械と安全(事故)

自動車事故による年間死者数が増加に転じた1980年から2002年までの間に、自動車保有台数が約3894万台から約7750万台(1.99倍)へ、運転免許保有者数が4300万人から7653万人(1.78倍)へと増加するに伴い、死傷者数もこれにほぼ比例して60万7479人から117万6181人(1.94倍)へと増加した(図1・11)。次に事故率(accident rate)の推移を図1・12に示す。人口10万人当たりの死者数は1970年が16.2人とピークをなし、その後減少し、1979年には7.3人にまで減少したが、2002年は6.5人である。自動車1万台当たりの死者数および自動車1億走行km当たりの死者数は、1980年ごろまで大きく減少したが、その後は少しずつ確実に減少しており、2002年にはともに1.1人となっている。6―8交通機械図1・12 人口10万人・自動車1万台・自動車1億走行km当たりの交通事故死者数の推移図1・13 事故類型別死亡事故発生件数(2002年時点)2002年中の事故類型(accident type)別死亡事故発生件数を図め、次いで工作物衝突15.9%、その他横断中(横断歩道・横断歩図1・11 道路交通事故による交通事故発生件数、死傷者数および死者数の推移図1・14 鉄軌道運転事故の件数と死傷者数の推移図1・15 海難船舶の用途別隻数の推移道付近以外での横断中)12.7%、正面衝突12.2%の割合が多い。
