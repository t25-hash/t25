# 1・3・3 鉄道

> **章**: 第1章 全体概要
> **節**: 1・13 に示す。細分類で見ると出会い頭衝突が最も多く16.8%を占

鉄軌道交通事故の推移を図1・14に示す。1982年に運転事故件数は1893件であったが、2002年には851件まで減少し、死傷者数819人、うち死者数344人であった。また、2002年中の運転事故の事故率は、列車走行100万km当たり0.67件である。事故種類別では、踏切障害が442件(51.9%)、人身障害333件(39.1%)、道路障害53件(6.2%)である(表1・2)。列車事故区分列車衝突列車脱線列車火災小計区分発生件数(件)大型飛行機小型飛行機超軽量動力機ヘリコプタジャイロプレーン滑空機飛行船計年注:(1)国土交通省資料による。(2)各年12月末現在の値である。(3)日本の国外で発生したわが国の航空機に関わる事故を含む(1998年1件、1999年3件)。(4)日本の国内で発生した外国の航空機に関わる事故を含む(1998年1件)。(5)事故発生件数および死傷者数には、機内における自然死、自己または他人の加害行為に起因する死亡等に関わるものは含まない。(6)死者数は、30日以内死者数であり、行方不明者等が含まれる。(7)2000~2002年の負傷者数は暫定値である。
