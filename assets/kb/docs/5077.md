# 1・4・1 交通と人間

> **章**: 第1章 全体概要
> **節**: 1・4 交通機械とバリアフリー

交通は人や物を移動させるものであり,交通機械(車両)と人の関わり合いは非常に大きいものである.安全に,快適に,速く移動することが交通の大きなターゲットであるが,利用者として,運転者としての人間の特性を加味した機能や構造をもつことが必要とされる.運転者・操作者の関係は,いわゆるマンマシンインタフェースの問題として,さまざまな取組みがなされてきているので,ここでは高齢化,障害者のノーマライゼーションに向けたバリアフリー(barrier-free)技術について解説する.利用者にとっての交通機関の利便性・快適性の議論は,これまで健常者をターゲットとしての話が比較的多く検討されてきたが,交通機関,特に公共交通では,利用者が多岐にわたっており,最近増加傾向にある高齢者を中心とした移動制約者(mobility-handicapped)の特性を考慮に入れなければ,使い勝手の悪いシステム・車両となり,利用に困難をきたすこととなる.
