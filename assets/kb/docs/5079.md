# 1・4・3 交通バリアフリー法(Transportation Accessibility

> **章**: 第1章 全体概要
> **節**: 1・4 交通機械とバリアフリー

Improvement Law)交通におけるバリアフリー化を計画的に促進させるために、「交通バリアフリー法」(正式名称「高齢者、身体障害者等の公共交通機関を利用した移動の円滑化の促進に関する法律」)が2000年5月に制定、同年11月より施行された。これは、移動制約者等の交通利用の際の困難さを低減するために、施設・車両等のバリアフリー化を、罰則を伴う義務化として制定したもので、日本が交通のアクセシビリティをあるレベルまで保障するという極めて重大な法律である。同法では、交通事業者の果たすべき項目を制定しているほか、市町村が基本構想を定めて責任をもってバリアフリー化を進展させることも課しており、4省庁(制定当時)が連携して制定したという点も新しい。実際の義務化の内容については、移動円滑化基準ということで、2000年11月に省令・政令が定められている。その主要な部分を表1・5に示す。交通バリアフリー法は、交通の利用の円滑化を図るために、国として定めた最低基準を示すものであり、それを満たせば望ましいレベルということではない。交通バリアフリー法に至るまでに、運輸省(当時)では次のような指針等の設定をしてきている。「公共交通ターミナルにおける身体障害者用整備ガイドライン」策定(1983年)、「視覚障害者誘導用ブロック設置指針」策定(1985年)、「心身障害者・高齢者のための公共交通機関の車両モデルデザイン」策定(1991年)、「鉄道駅におけるエスカレータの整備指針」策定(1991年)、「鉄道駅におけるエレベータの整備指針」策定(1993年)、「公共交通ターミナルにおける高齢者・障害者等のための施設整備ガイドライン」策定(1994年)、「みんなが使いやすい空港旅客施設新整備指針」策定(1994年)、「鉄道駅におけるエレベータ及びエスカレータの整備指針」策定(1999年)など。また、交通バリアフリー法の制定に合わせて、船舶の設計マニュアルの策定、車両モデルデザインや施設整備指針の改訂も行われた。
