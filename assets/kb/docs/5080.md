# 1・4・4 福祉車両(welfare-vehicle)

> **章**: 第1章 全体概要
> **節**: 1・4 交通機械とバリアフリー

自動車にバリアフリー技術を付加して高齢者・障害者の移送に適した構造としたものを福祉車両と呼ぶ。1970年代に施設送迎用の小型バスやワゴン車に、車いすリフトを装備したタイプが登場したのが始まりで、その後、下肢麻痺者を中心とした身体障害者自身が表1・5 移動円滑化基準の主要項目鉄道・乗降口とホームの段差、すきまを可能な限り小さくすること・車いすで乗降できる扉と車内スペースを確保すること・トイレを設ける場合は一つ以上車いす対応とすること・視覚、聴覚障害者のための情報設備を設けること・車両連結部に転落防止棚を設けること路線バス・床高65cm以下の低床バスとすること・車いすで乗降できる設備を設け、乗降口の有効幅は80cm以上とすること・車いす用スペースを1座席以上設け、固定装置を有すること・視覚、聴覚障害者のための情報設備を設けること旅客船・通路幅は80cm以上とし、手すりを設けること・フェリーの障害者用駐車ますは350cm以上の幅を確保すること・乗船口には段差をなくすこと・乗船口と客席が異なるデッキの場合にはエレベータ等の設備を設けること・座席数25に対して1の割合で移動制約者向けの座席を設けること・座席数100に対して1の割合で車いす対応のスペース、固定具を備えること・トイレを設ける場合は一つ以上車いす対応とすること・視覚、聴覚障害者のための情報設備を設けること航空機・座席数30以上の場合は通路側の座席の肘当ては半数以上可動とすること・座席数60以上の場合は機内用車いすを用意すること・通路が2箇所以上の場合は、車いす対応トイレを設けること・視覚、聴覚障害者のための情報設備を設けることターミナル・1日の乗降数5000人以上のターミナルにはエレベータ等を設けること・車いす対応トイレを設けること運転できるように補助装置を取付けた自操タイプが加わり、1990年代に入ると、施設用ではなく個人ユース向けの移送用の車種が拡充された。特に最近は、回転シート、リフトアップシートを装備した車両が新型車発表と同時に用意されるようになり、ラインナップの一つとして位置付けられるようになり、前年比2けた増の増加傾向が続いている。車両技術・構造としては、車いす使用者の送迎用として、車いすリフトを車両後部(あるいは側面)に設けたものや、低床化を図り簡易スロープで乗降が容易に図れるものがあり、車いすの固定装置も数多くのタイプが市販されている。車いすから座席への移乗を容易にしたり、体の弱い高齢者の乗降を助けるために、座席を外部に飛び出させるタイプも数多くある。移乗作業は介助者にとっても負荷の高い作業であるので、座席がそのまま車いすになるタイプも提案されている。自操型では、足の不自由な人向けに手だけで運転ができる補助装置が多数あるほか、サリドマイドで手が利かない人向けの、足だけで操縦する補助装置、重度の脊髄損傷者や脳性麻痺者向けのジョイスティック操縦装置なども実用化されている。現在の福祉車両は、特装メーカが後付け改造で対応するものが多く、価格も高いものが多い。今後は標準化や生産体制の変革による低価格化や、ベース車のユニバーサルデザイン化などが望まれている。文献(1) 国土交通省、 平成14年度国土交通白書、 (2002)、 http://www.mlit.go.jp/hakusyo/mlit/h14/H14/html/E2072330.html(2) 国土交通省、平成13年度国土交通白書、(2001)、22.(3) 運輸省、平成12年度運輸白書、(2000)、大蔵省印刷局。(4) 内閣府編、平成15年版交通安全白書、(2003)、国立印刷局。→ Next Chapter
