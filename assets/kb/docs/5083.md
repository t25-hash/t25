# 2・1・3 自動車の諸元(specification of automobiles)

> **章**: 第2章 自動車
> **節**: 2・1 自動車一般

自動車の寸法,重量,構造,装置および性能などを表したものを自動車の諸元と呼び,仕様書,説明書,カタログ等に諸元表(表自動車の仕様書様式を標準化したJIS D 0001:2001では,諸元表の構成,様式,記載要領を規定しており,自動車の仕様の概要を示す主要諸元と自動車の使用,整備,その他技術的な目的のために仕様の詳細を示す明細諸元が含まれている.また自動車の型式指定などを取得するために国土交通省に提出する申請資料の中にも「諸元表」があり,JIS で規定されている諸元表項目に加え,法規で定められた事項に関する項目が要求されている.具体的な記載内容,項目は,「自動車型式認証実施要領」の附則5「自動車等の諸元表の記載要領」等で規定されている.
