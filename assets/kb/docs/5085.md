# 5・5・2 分塊圧延の主要機器

> **章**: 第5章 製鉄機械
> **節**: 5・5 分塊圧延設備

分塊圧延設備には均熱炉，分塊圧延機，せん断機，スカーファがa.均熱炉（soaking pit) 製鋼工場でインゴットケースからで均一に加熱される．
均熱炉はすべてバッチ式で1基当たりークスガスの混合ガスを使用するものが多く重油燃焼のものもあ延機が使用される．
大型化，高能率化，自動化が要求されるスラブルが多く用いられる．
生産能力は600～1000t / hに及び，大型化に多い．
分塊圧延機の前後にはローラテーブルとマニプレータを設置図5･22にスラビングミルの全体図を示す．
切断するため，大容量のせん断機が設置される．
せん断形式としてップアンドダウンカット式が用いられる場合もある．
刃物の駆動はMN程度となる．
二重逆転式の分塊圧延機のみで圧延することは極めて非能率である延機を通して100mm角程度のビレットを生産する．
この圧延機材料を捻転なしに圧延する．
断されたビレットは冷却床に送られて冷やされる．
生産された角ま目なし鋼管の素材にもなる．
るための装置で，酸素およびプロパンガスにより鋼片が通過中に上手作業に比し能率は高い．
設置され，ビレットを生産する設備では鋼片圧延機が加わる．
型抜きされた鋼塊は，この均熱炉に送られ分塊圧延に適した温度ま20～85t / hの能力の炉が数基設置されている．
燃料は高炉ガスとコる．
ない設備ではスラブとブルーム圧延兼用のハイリフト二重逆転式圧圧延では水平ロール機に垂直ロール機を組合わせたユニバーサルミ伴い主電動機は上下ロールを別々に駆動するツインドライブ方式がして圧延材の搬送と転倒を行う．
さらにこの連続圧延機の下流側には走間せん断機が設けられ，切たは丸ビレットは次工程の棒鋼ミル，線材ミルに送られるほかに継e.ホットスカーファ（hot scarfer) 鋼片の表面傷を除去す下と側面の4面を同時に薄く溶削する．
歩留り低下の欠点があるが表5･4に分塊圧延機の代表的な例を示す．
