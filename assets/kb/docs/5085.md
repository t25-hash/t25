# 5・5・2 分塊圧延の主要機器

> **章**: 第5章 製鉄機械
> **節**: 5・5 分塊圧延設備

分塊圧延設備には均熱炉，分塊圧延機，せん断機，スカーファが
設置され，ビレットを生産する設備では鋼片圧延機が加わる．
a.均熱炉（soaking pit ) 製鋼工場でインゴットケースから
型抜きされた鋼塊は，この均熱炉に送られ分塊圧延に適した温度ま
で均一に加熱される．均熱炉はすべてバッチ式で1基当たり
20～85t / hの能力の炉が数基設置されている．燃料は高炉ガスとコ
ークスガスの混合ガスを使用するものが多く重油燃焼のものもあ
る．
b.分塊圧延機（slabbing mill ，blooming mill ) 生産量が少
ない設備ではスラブとブルーム圧延兼用のハイリフト二重逆転式圧
延機が使用される．大型化，高能率化，自動化が要求されるスラブ
圧延では水平ロール機に垂直ロール機を組合わせたユニバーサルミ
ルが多く用いられる．生産能力は600～1000t / hに及び，大型化に
伴い主電動機は上下ロールを別々に駆動するツインドライブ方式が
多い．分塊圧延機の前後にはローラテーブルとマニプレータを設置
して圧延材の搬送と転倒を行う．
図5･22にスラビングミルの全体図を示す．
c.せん断機（shear ) スラブまたはブルームを所定の長さに
切断するため，大容量のせん断機が設置される．せん断形式として
はスラブ切断ではダウンカット式が用いられ，ブルーム切断ではア
ップアンドダウンカット式が用いられる場合もある．刃物の駆動は
電動式と油圧式とがあり，せん断力はスラブシャーにて25～30
MN程度となる．
d.鋼片圧延機（billet mill ) 小断面のビレットを作る場合，
二重逆転式の分塊圧延機のみで圧延することは極めて非能率である
ため，180～230mmのブルームで分塊圧延を終え，引続き鋼片圧
延機を通して100mm角程度のビレットを生産する．この圧延機
は水平垂直の各ロール単独駆動式のもので4～6基連続に設置され，
材料を捻転なしに圧延する．
さらにこの連続圧延機の下流側には走間せん断機が設けられ，切
断されたビレットは冷却床に送られて冷やされる．生産された角ま
たは丸ビレットは次工程の棒鋼ミル，線材ミルに送られるほかに継
目なし鋼管の素材にもなる．
e.ホットスカーファ（hot scarfer ) 鋼片の表面傷を除去す
るための装置で，酸素およびプロパンガスにより鋼片が通過中に上
下と側面の4面を同時に薄く溶削する．歩留り低下の欠点があるが
手作業に比し能率は高い．
