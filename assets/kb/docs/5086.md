# 5・6・1 加熱炉設備

> **章**: 第5章 製鉄機械
> **節**: 5・6 厚板圧延設備

加熱炉（furnace）は鋼片を圧延可能な温度まで加熱する設備で
あり，その温度は鋼種ごとに異なり通常900～1250℃と広範囲にわ
たる．
加熱炉には，省エネルギー（燃料消費量の最少化），品質の高度
化（加熱温度の均一性），環境保全（NO 等排出量の最少化），生
産性向上，省力化（保守・点検の容易さ）に関わる工夫・技術が多
く採用されている．
炉の能力は一基当たり100～300t / hで，厚板工場能力に合わせ
て数基設置される．炉前のテーブルまで搬送された鋼片はプッシャ
またはチャージャで炉に装入され，油圧または電動式のウォーキン
グビームで炉内を搬送され，エキストラクタで抽出される．これら
一連の搬送は，電算機を用いた自動搬送制御で行われる．
炉内の鋼片は通常二列操業であり，炉幅は8～12m，炉長は
30～50mである．鋼片の昇温率は鋼片厚さ1mm当たり0.7～0.9
分程度である．
燃料は重油，ガス（コークス炉ガス，混合ガス，液化石油ガスな
ど）が使用され，上下複数帯に区分されて配置された複数バーナお
よび自動燃焼制御により鋼片の加熱を行う．また鋼片内部の温度分
布や昇温パターンを数式モデルを用いて電算機で計算させ，省エネ
ルギーや品質の高度化を図っている（図5･24）．
