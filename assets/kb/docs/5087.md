# 2・2・1 自動車に作用する力

> **章**: 第2章 自動車
> **節**: 2・16 に示す重心に原点をおく直交座標系(,,)と結びつ

自動車に作用する力ベクトルを自動車固定座標系の各軸方向成分に分解して、1前後力(成分)、2左右力(成分)、3上下力(成分)、またモーメントベクトルの各軸成分として、4ローリングモーメント(成分)、5ピッチングモーメント(成分)、6ヨーイングモーメント(成分)が定義される。自動車の運動はエンジンなどの駆動力、ブレーキ装置などの制動力によってタイヤと路面間で発生する前後力を制御し、ある前進速度が与えられた状態において、ステアリング装置でタイヤのステア角を操向して横力、ヨーイングモーメントを制御することで、自由な平面運動を行う。さらに車体等に作用する空気力や、路面不整、傾斜による外乱、加速、減速、旋回等の平面運動に伴って重心点に作用する慣性力により、6自由度の運動が発生する。ここで、2と6は主として走行中の操舵による作用力、4はその結果として生じる慣性力が主図2・16 自動車の座標系図2・17 タイヤの座標軸図2・18 転がり抵抗の速度依存性要な成分であるのに対し、135は操舵とは直接的な関係がなく発生する作用力として、大別できる。ただし、近年ではシャシ制御装置により左右輪の制動力や駆動力の差を利用することにより、6を自律的にコントロールすることなども行われており、自動車の運動と作用力の関係は多様化している。
