# 2・2・2 タイヤ特性

> **章**: 第2章 自動車

自動車が各軸方向の運動を行う場合,その際に路面との間に作用する力はタイヤに依存する.タイヤは鉛直荷重,駆動力・制動力,求心力(横力)の三軸方向の力をすべて受け持つ.特に水平面内の力(駆動力・制動力,求心力)は路面との摩擦により発生するため,タイヤと路面の接地部の状態によりさまざまに変化する.図である.a. 転がり抵抗タイヤが路面上を転動することにより発生する抵抗である.転がり抵抗(rolling resistance force)を,鉛直荷重を,転がり抵抗係数(rolling resistance force co-efficient)をμとすると式(2・1)の関係がある.=μ(2・1)このμは,路面の種類・状態,タイヤの転動速度,タイヤ内圧,鉛直荷重,タイヤの構造・材料などによって変化する.図2・18に転がり抵抗の転動速度による変化とタイヤの構造との関係を示図2・19転がり抵抗と空気圧図2・20
