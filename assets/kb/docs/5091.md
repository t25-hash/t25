# 2・2・5 制動性能

> **章**: 第2章 自動車
> **節**: 10・15 モードの走行サイクルを示す。

近年の日常生活での自動車の利用が一般化する中、自動車の安全性に対する意識の高まりが非常に強くなってきている。そのことを背景に自動車の速度を制御する装置の一つであるブレーキ性能はより重要性を増し、短く止まることのできる能力だけでなく、一般の運転者が容易にその能力を使いきれること、操作性が良いこと、車両挙動が安定していることなどが重要視されつつある。ブレーキ性能としては、効き、停止距離(stopping distance)、車両挙動安定性と法規適合性に加え、失陥時のフェイルセイフ等がある。ブレーキの効きは入力に対する出力の関係で示されるが、運転者にとってはブレーキシステムの効きとなり、操作力とブレーキ力を車両質量で除した減速度との比となる。一般の運転者がブレーキの能力を使いきれること、操作性が良いことの点から過大、過小な操作力は避けるべきである。この点を考慮したブレーキアシストシステム(BA : brake assist system)の採用は、アンチロックブレーキシステム(ABS : anti-lock brake system)の普及と相まって、通常使用時には軽すぎず、かつ緊急時には操作力の増加を抑えて必要なブレーキ力を得られることを可能にした。ブレーキ力は路面で吸収されるため、タイヤと路面の特性も効きにおいて重要な役割を占める。タイヤと路面の特性限界を超えてブレーキ力を発生することはできない。効きの安定性も運転者にとって重要である。降坂時にブレーキの使用頻度が高まるときや高速からの制動時に、ブレーキの摩擦部分の温度が上がり摩擦係数が変化する。この現象をブレーキフェード(brake fade)といい、熱による効きの変化を意味し、操作力に対しブレーキ力発生が低下する効きの悪化が一般的である。また、浸水地帯を走行し、摩擦部分が水にぬれた場合も摩擦係数が変化する。この現象をウォータフェードという。ウォータフェードの場合は、水にぬれた直後に効きが低下することが一般的であるが、その後の回復過程で効きが上がりすぎる場合があり注意が必要である。これら効きの変化を小さくして安定性を確保するため、最近はブレーキ力発生の占める割合の大きい前輪のブレーキにディスクブレーキ(disk brake)が採用されている。停止距離性能は、速度を制御することや意図した場所に停止するなど運転者の意図した制御のブレーキの使用に対し、機械装置の緊急停止に匹敵する最短停止距離を意味する場合が多い。安全の面から緊急時において最短距離で止まれることが重要であるが、ブレーキとタイヤの特性を合わせた性能であり、タイヤ-路面間の摩擦係数が最大となるタイヤが回転している状態を得られるよう、前後輪のタイヤがロックせずにタイヤの特性を使いきるブレーキ力のバランスを設定する必要があるが、現実的には非常に難しい。一般の運転者にとって、タイヤがロックせずに最短停止距離を得ようとブレーキを制御することは非常に難しく、これを補うためABS の採用が一般化してきた。車両挙動安定性は、前輪ロック(front wheel lock up)または後輪ロック(rear wheel lock up)を引き起こす前後のブレーキ力バランス(brake force distribution)によるところが大きい。前輪ロックの場合は操舵性が失われ、後者の場合はスピンを引き起こす原因となる。ブレーキ力のバランスをつねに車両の諸元に合わせて最適にすることは不可能に近い。そこで安全性の面から、後輪ロックによる自動車のスピンを避けるためプロポーショニングバルブを用いて前輪ロックを先に発生させるか、ABS を用いてロックを避ける特性としている。また、左右のブレーキ力バランスも重要である。左右のブレーキ力が異なる場合、特に前輪の左右のバランスが大きく異なるとブレーキ力の大きいほうにモーメントが発生してハンドルをとられることになり(片効き)、自動車が思わぬ方向に向かうことになる。自動車関連法規は、自動車を登録するうえで最小限満たしていなければならない要件として、各国でそれぞれ規定されている。ブレーキ性能についても、多くの国で定められている各国の要件を満たさなければならない。1990年代前半までは、アメリカ(FMVSS105)、ヨーロッパ諸国(ECE R 13)、日本(保安基準12条)がそれぞれ独自の内容のブレーキ法規制を採用し、他の国がそれぞれに準拠した法規を採用していた。それぞれの国が独自の法規を採用することは、自動車の輸出入が世界化する中、安全面、経済面の課題となった。1980年に、乗用車の統一法規を検討する会議体がUN/ECE(国際連合/欧州経済委員会)の中に設定され、日本、アメリカ、ヨーロッパ各国の参加により乗用車の統一法規が制定され、批准が進みつつある。失陥時のフェイルセイフは、考えられる最大入力に対し破壊や機能を失わないことと疲労等でも破壊しないことを強度として備えていても、なんらかの要因でブレーキシステムの一部に破損や機能低下等が生じる場合がある。液圧式ブレーキシステムを例にとると、図2・32 車両挙動安定性車両挙動安定性の確保は、急制動時に車線を逸脱することなく停止することの可否につながるが、この特性は前後ブレーキ力配分によるところが大きい。制動時には減速度により荷重移動が起こり(前進時は、前輪軸重が増加し、その分、後輪軸重が減少する)、その動的荷重に促した適正なブレーキ力を算出したのが理想ブレーキ力配分である。理想ブレーキ力配分線の一例を図2・37に示す。図2・37においては、実ブレーキ力配分が後輪ロック限界線を横切る形になっており、後輪ロックの特性を示している。また、実ブレーキ力配分が前輪ロック限界線を横切る場合は、前輪ロックの特性となる。前輪ロックでは、後輪が回転しているために発生する後輪のコーナリングフォースが車両姿勢を進行方向(車両の慣性力の方向)に矯正する力として働き、直進状態で車両は停止するが、前輪のコーナリングフォースは零となり、操舵性は失われる。後輪ロックでは、後輪のコーナリングフォースが零となり、わずかの車両姿勢の変化に対して、前輪のコーナリングフォースが車両をさらに巻き込む方向に働くため、車両のスピンを引き起こす。四輪同時にロックした場合は、前後輪ともコーナリングフォースが零となるため、車両にはヨーが発生せず、外乱がなければ進行方向に平行な状態を維持して停止する。以上のことから、急制動時に車両の挙動が安定する前輪が先にロックするよう、前後のブレーキ力配分を設定することが一般的である。一般の運転者が、タイヤをロック直前の状態でブレーキ力を維持することは非常に困難であり、ロックを避け操舵性を確保しつつ、高い減速度を得られるABS
