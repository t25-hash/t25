# 2・2・6 操縦性・安定性

> **章**: 第2章 自動車

自動車の主として操舵に伴う横方向運動に関連する性能を操縦性・安定性という。この性能は図2・16で示した座標系の軸回りのヨーイング、軸方向の並進運動、およびこれらの平面運動に伴う軸回りのローリングがおもな運動となる。ただし、加速や減速を伴いながら操舵する場合には、前後方向やピッチングとの連成も考慮する必要がある。また、実際の操縦性安定性の評価は、操縦者の意図どおりに自動車が応答するか、横風や路面からの外乱に対して安定しているかなどといった、人間-自動車-環境で構成されるクローズドループ系での総合的な性能である。そのため、ドライバの嗜好や操作の特性、道路環境などにも左右され、自動車の機械的な特性として理論的に最適解を求めることは難しく、経験則の蓄積による部分も少なくない。しかし、線形二輪モデルと呼ばれる、単純化された操舵入力に対する機械系応答としての自動車固有の運動力学は、比較的理論的な扱いが容易であり、かつ操縦性・安定性評価に含まれる種々の特性を支配する基本的な特性として重要である。a.
