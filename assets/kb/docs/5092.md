# 2・2・6 操縦性・安定性

> **章**: 第2章 自動車
> **節**: 2・35 に踏力感知型BA の倍力特性の一例を示す。

自動車の主として操舵に伴う横方向運動に関連する性能を操縦性・安定性という。この性能は図2・16で示した座標系の軸回りのヨーイング、軸方向の並進運動、およびこれらの平面運動に伴う軸回りのローリングがおもな運動となる。ただし、加速や減速を伴いながら操舵する場合には、前後方向やピッチングとの連成も考慮する必要がある。また、実際の操縦性安定性の評価は、操縦者の意図どおりに自動車が応答するか、横風や路面からの外乱に対して安定しているかなどといった、人間-自動車-環境で構成されるクローズドループ系での総合的な性能である。そのため、ドライバの嗜好や操作の特性、道路環境などにも左右され、自動車の機械的な特性として理論的に最適解を求めることは難しく、経験則の蓄積による部分も少なくない。しかし、線形二輪モデルと呼ばれる、単純化された操舵入力に対する機械系応答としての自動車固有の運動力学は、比較的理論的な扱いが容易であり、かつ操縦性・安定性評価に含まれる種々の特性を支配する基本的な特性として重要である。図2・38に示す前後輪横力の合力の着力点をニュートラルステアポイント(NSP)と呼び、重心点からNSPまでの距離をホイールベースで割って無次元化した量をスタティックマージン(SM : static margin)と呼ぶ。NSPが重心点より後方にあるときSM は正の値をとり、ステア特性はUS を示す。また式(2・9)より、オーバステアの場合には、ある速度以上では旋回半径が物理的に意味のない負の値になる限界速度が存在する。US の場合は、式(2・13)で求められる速度で最大値をとり、さらに高速では漸減する。ドライバにとって、適度なUS 特性が運転しやすい背景の一つである。平均的なの値として、0.002s/mの場合、ヨー角速度ゲインが最大となる速度は約80km/hとなる。b. 動的操舵応答特性動的なハンドル角入力に対するヨー角速度の応答特性は下式のような伝達関数で表される。ζωは操安キャパシティとも呼ばれ、ステップ操舵入力でのヨー角速度応答の振動減衰の包絡線を決定する値である。この値が大きいほど、操舵に対する速応性が優れ、定常値に達する収束時間が短くなる(図2・41)。これらの値はともに一般的には大きいほうが操縦しやすいと考えられている。そのためには、車両質量、ヨー慣性モーメントを小さく、タイヤコーナリングパワー、ホイールベースを大きくすればよいことがわかる。また、これらの値は高速になるに従って減少し、特にダンピングの低下を招くので、高速車でのパラメータ設定には注意が必要である。実際には、適度なUS 動的操舵応答特性の試験法一定速走行時にタイヤの線形範囲の横加速度(3~4m/s程度)の発生するパルス状、あるいはランダム周波数の正弦波入力を与え、ハンドル角に対するヨー角速度などの応答を伝達関数(周波数応答)として求める方法が一般的に行われている(図2・42)。また、ステップ状の操舵を行い、ヨー角速度や横滑り角の時系列変化から応答性を評価する手法もある(図2・43)。f. ロールとタイヤ非線形領域での旋回特性e項までの議論では、一定荷重のもとで、タイヤがタイヤスリップ角に比例してコーナリングフォースを発生し、ヨーイングと横方向移動を行うという線形近似二輪モデルでの特性を考察してきた。実際の自動車では、旋回横加速度により、軸回りのロール運動が発生する。ロール角の小さい範囲では、前後ロールセンタを結ぶロール軸回りに車体がロールすると扱える。ロール率の大きさは、操縦性・安定性の性格を左右する重要な特性の一つであり、図2・45に見られるように時代とともに低減されてきている。また、ロール運動と同時に旋回内輪から外輪に荷重の移動が発生し、実際のタイヤコーナリング特性は、図2・46で示されるような荷重依存性を示すことから、特に高い横加速度での旋回時にはステア特性に大きな影響を及ぼす。また、旋回時には、前後のタイヤに対地キャンバ角の変化が発生する。対地キャンバ角は、独立懸架の場合、ロール角とサスペンションリンクの幾何学的キャンバ変化、および横力とサスペンション剛性で決まるコンプライアンスキャンバ変化の和で決定される。通常、旋回外輪は、ポジティブ方向に対地キャンバ角が変化し、タイヤのキャンバスラスト分だけ横力が減少する。したがって、前後のサスペンションのキャンバ変化特性や剛性を変えることでも、ステア特性が変化する。タイヤの横方向グリップ能力を最大限に生かすためには、対地キャンバ角はできるだけ小さくすることが望ましいが、そのためにサスペンションストロークに対する幾何学的なキャンバ変化を大きく設定することは、不整路面での直進性を阻害するため限度がある。一般的には、前輪の荷重移動量を後輪より大きくするなどの手段により、前輪から先に旋回限界に達する、すなわち定常的にはUS 特性を維持し、横加速度とともにUS が強まる傾向の設定がされることが多い。これは乗車人員や積荷の増加や、g 項で述べる旋回中の減速に伴うOS 方向の変化があってもなおUS を維持することが、安定性上望ましいことと関係している。g.
