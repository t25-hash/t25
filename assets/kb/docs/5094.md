# 5・7・4 焼なまし設備（annealing furnace）

> **章**: 第5章 製鉄機械
> **節**: 5・7 薄板圧延設備

冷延鋼鈑の焼なましは，冷間圧延によって加工硬化した金属組織
を再結晶温度以上に加熱して，結晶粒を適度に成長させ機械的性質
を改善することを目的とする．焼なましに使用される炉は，操業方
法によってバッチ式と連続式に分けられる．

b.連続焼なまし ブリキ用連続焼なまし設備（continuous
annealing line）では，処理材は化学清浄，電解清浄を経て焼なま
し炉に入り，加熱，均熱，徐冷，急冷の各帯で所定のヒートサイク
ルにて通板された後巻取られる．ペイオフおよびテンションリール
は各々2基ある．近年では時効性を良くするため，炉において過時
効処理を行ったり，表面粗度と硬度を調整するため炉出側において
テンパーミルにより圧下する場合もある．図5･52にブリキ用連続
焼なまし炉，図5･53に焼なましサイクルの一例を示す．一方冷延
鋼板の連続焼なましは，社会的ニーズである品質安定性と経済性の
追求，さらには冶金学の進歩，設備技術の進歩が相まって急速に進
歩，普及してきた．この方法の特徴は急速加熱，短時間均熱により
再結晶，結晶粒成長をさせた後，急速冷却と過時効処理によって過
飽和固溶C（カーボン）を析出させてバッチ焼なましと同等の材質
を得ることができる．ライン構成は，入側部，焼なまし（加熱，均
熱），冷却，過時効炉，（酸洗），出側部からなっているが，冷却方
式によって多少異なっている．各部の機能，特徴は次のとおりであ
る．
.入側部 冷延コイルを巻きほぐし，次々と溶接し連続走
行，表面を化学洗浄，電解洗浄した後，焼なまし炉へ送り込む．
.焼なまし 加熱，均熱帯からなり，鋼板を所定の温度ま
で昇温し，約1分程度保持する．加熱方式はラジアントチューブに
よる間接加熱が大半だが，最近，熱効率と急速加熱の観点より直接
加熱（ただし，無酸化雰囲気となる燃焼）が研究，実施されてい
る．均熱帯には通常ラジアントチューブか電熱ヒータが用いられて
いる．

.過時効炉 約400℃で数分の過時効処理を行うため炉内
に電熱ヒータが装備されている．過時効炉では，鋼板の繰返し曲げ

制御能力が向上している．
