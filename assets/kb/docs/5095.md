# 2・3・1 動力伝達装置(transmission system)

> **章**: 第2章 自動車
> **節**: 2・3 自動車の構造

a. 概説動力伝達装置とは,エンジン出力を駆動車輪へ伝達させる種々の装置で構成されたシステムであり,その構成要素はクラッチ,変速機(手動変速機,有段自動変速機,無段自動変速機),四輪駆動装置,推進軸,終減速機等である.また駆動形式については,前輪駆動,後輪駆動,四輪駆動(全輪駆動)に分類される.図2・60にFF(front engine-front drive)と呼ばれる前輪駆動方式,図2・61にFR(front engine-reardrive)と呼ばれる後輪駆動方式のレイアウト図を示す.b. クラッチ(clutch)クラッチはエンジンと変速機の間に設けられ,車両停止状態でエンジンの動力を遮断し,あるいは走行時に接続する機能を有する.クラッチの種類としては乾式摩擦クラッチ,湿式摩擦クラッチ,電磁クラッチがある.. 乾式摩擦クラッチ(dry-plate friction clutch)図2・62に代表的な乾式摩擦クラッチ構造を示す.クラッチディスクは乾式単板の円形状が多く,駆動系に伝達される回転変動を吸収して駆動系異音を低減する.発進時には滑りを許しながら動力を徐々に伝える.普通,手動変速機とともに用いられ,接続が円滑,遮断が確実で,慣性モーメントが小さく,十分な伝達トルク容量と滑りによる摩耗に対して十分な耐久性を備えていることが必要である.. 乾式摩擦クラッチの例ラッチに対して係合面に油が介在するため,摩擦係数が安定し,滑りに対しても油による冷却効果が得られる.またピストンも油圧で制御されるためコントロールが容易である.一部の無段変速機において用いられている.. 電磁クラッチ(electromagnetic clutch)電磁力を利用して動力を断続する装置で,トルクを伝達する部分の構造により,圧着式,かみあい式,鉄粉(パウダ)を媒体とする空げき式などがある.一部の無段変速機では空げき式電磁クラッチとして用いられ,コイルの作用により固体状に磁性粉が結合してトルクを伝える.c. 手動変速機(manual transmission). 手動変速機の機能手動変速機は手動操作により,1変速比を変化させる,2回転を逆転させる,3動力伝達を行わない状態(ニュートラル)を作る,という機能を有する.また,発進,走行中にクラッチ操作と変速操作の二つを必要とする.さらに,車両要求性能に応じたあらゆる走行状態で,エンジン出力を最大限に利用できる適切な変速段,変速比が要求されると同時に,小型軽量,操作容易でかつ良好なフィーリング,静粛性,高効率,高い信頼性などが要求される.. 手動変速機の基本構造と特徴図2・63は後輪駆動方式6速手動変速機の断面の一例を示したものである.構造は,入力軸,副軸,出力軸,各変速段の変速比を得るためのギヤ対と,歯車6―34交通機械図2・63 後輪駆動方式6速手動変速機の例図2・64 トルクコンバータの例の回転速度と軸の回転速度を一致させるための同期装置等から構成されている.また,同期装置は,変速の際に同期側と被同期側の結合スプラインの回転速度を摩擦力によって等しくし,円滑な変速操作を可能とする装置である.手動変速機の分類としては,一般乗用車,商用車には,前進4~6速・後進1速が用いられる.かつ変速操作方式はフロアシフトとコラムシフトに大別され,歯車列構造の違いにより次のように分類される.(1) 入出力軸同心型入力軸,副軸,出力軸の三軸で構成されており,入力軸と出力軸が軸を介して同一軸上に配置されている.動力の伝達は,入力軸→副軸→出力軸という順序で行われる場合と,入力軸と出力軸を直結して入力軸→出力軸という順序で行われる場合がある.通常,FR 車用変速機に用いられる.(2) 入出力軸平行型入力軸,出力軸の二軸で構成されているものが多いが,これに副軸を加えて三軸で構成されているものもある.上記各軸が平行に配置されており,動力の伝達は,入力軸→出力軸,または入力軸→副軸→出力軸の順序で行われる.通常,FF車用変速機に用いられている.d. 流体伝導装置(hydraulic power transmission)自動車に一般的に用いられている流体伝導装置には,流体の運動エネルギーを用いて動力を伝達するターボ式の流体継手や流体トルクコンバータ(torque converter)がある.ターボ式流体伝導装置は,流体を仲介としてエンジンの動力を変速機に伝達する装置であり,伝達効率や発熱の問題があるものの負荷に応じて自動的に出力回転速度が変化する自動変速性を有しており,流体を仲介しているため,振動衝撃の吸収ができ,適当な滑りを許して機関停止を防ぐ発進装置としての機能を有している.流体継手は,入力軸トルクをそのまま出力軸に伝えるものである.またトルクコンバータは,トルクの増大作用を有し,このため自動車の自動変速装置として広く用いられている(図2・64).トルクコンバータにおける動力損失を低減するために,トルクコンバータの機能を必要としない領域で入力軸と出力軸を機械的につなぐ機構(直結クラッチ)が設けられているものが増えてきている.e. 自動変速機(automatic transmission). 自動変速機の機能自動変速機は,車両のさまざまな走行条件に必要な駆動力を得るために,発進のためのクラッチ操作と図2・65 FF 方式5速自動変速機の例ギヤシフト操作を自動的に行う装置である.自動変速機では,1変速比を変化させる,2回転を逆転させる,3動力伝達を行わない状態を作る,という手動変速機の機能に加えて,4クラッチの断続操作を自動で行う,という機能を有している.. 有段自動変速機の基本構造と特徴自動変速機の基本諸元として変速段数,ギヤ比がある.変速段数やギヤ比は車両の燃費,排出ガス,動力性能,振動・騒音等の諸特性に最も適合するように決定される.一般的に最高速段のギヤ比と終減速機のギヤ比は最高速度や燃料消費率等の諸特性を考慮して選定される.また,変速機の変速段数とそれぞれのギヤ比は発進,加速,登坂等の要求性能とエンジンの出力特性から最適な諸元が設定される.自動変速機は基本的に以下のような構成要素から成り立っている.(1) 発進装置:流体式,機械式,電磁式(2) ギヤ比を形成する歯車列:遊星歯車式(シンプソン式,ラビニヨ式,CR-CR 式),平行軸歯車式(平行二軸式,平行三軸式)(3) ギヤ比を切換える要素:多板クラッチ・ブレーキ,同期装置(シンクロなど),シフトアクチュエータ(4) 変速タイミングなどを制御する制御装置:油圧制御式,電子油圧制御式エンジンからの駆動力,回転は発進装置である流体継手(トルクコンバータ),多板クラッチ,遊星歯車につながり,目的とする変速比になるよう係合を切換えて最終的に出力軸に伝達される.図2・65にFF方式の5速自動変速機の断面の一例を示す.. 無段変速機の基本構造と特徴無段変速機(CVT :continuously variable transmission)は変速比を無段階連続に変更可能な変速機である.一般的に,有段変速機と比較して,変速比幅を広くとることが可能で優れた低燃費特性を実現できる.また,CVT の特徴としてエンジン回転数を保持することができ,これによりエンジンの最大駆動力を維持したままの加速が可能である.さらに,駆動力特性(エンジン回転数,トルクと車速の関係)を自在に設定することが可能で,これによりエンジン特性を最大限に発揮しながらスムーズで運転しやすい変速特性を実現できる.自動車用無段変速機は,流体式(油圧駆動式,トルクコンバータ式など),機械式(ベルトドライブ式,トラクションドライブ式など)に分類されるが,実用されているものは機械式CVT である.(1) ベルトドライブ式CVT動力伝達用ベルトにより,ゴムベルト式,金属ベルト式,チェーン式に分類される.ゴムベルト式CVT は,発進クラッチに遠心式クラッチを用い,プーリに内蔵された遠心質量とエンジンの吸入管負圧により無段変速を制御するものである.金属ベルト式CVT(図2・66)は,金属ベルトはプーリへの最小巻付半径を小さく設定でき,それによって変速比の幅を大きくと図2・66 金属ベルト式CVT の例図2・67 金属ベルト式無段変速機構の作動図2・68 金属ベルト部れる.変速は,油圧により溝幅が可変する1対のプーリにより金属ベルトの巻掛かり径を可変させることで無段階に行う.また用いる金属ベルトは金属製のエレメントとスチールリング2列から構成される.一般のチェーンやゴムベルトなどが引張力により動力を伝達するのに対して,金属ベルトはコマの圧縮作用(コマの押出し)により動力伝達を行う(図2・67,2・68).チェーン式CVT は,金属ベルトの代わりにチェーンを用いるもので,プーリ機構,変速制御等の基本は金属ベルト式と同じである.(2) トラクションドライブ式CVT無段変速は入出力ディスクに挟まれたローラの傾き角を変え,接点の半径比を無段階に変化させることで実現する.動力の伝達は油膜を介してディスクからローラに力を伝えるトラクションドライブの技術が使われている.トラクションドライブ式CVT は高面圧下で安定したトラクション係数を発生する専用オイルの開発と無段変速部の信頼性・耐久性の確保により,近年実用化され大トルク容量と高応答な変速を実現している.f. 四輪駆動装置四輪駆動装置(four-wheel drive system)とは,四輪駆動車(4WD)等の前後駆動輪に動力を伝達可能にする装置である.必要機能として,当初,雪路,悪路等での脱出性,走破性,登坂能力が重点であったが,近年は高性能原動機との組合せにより操縦性能,直進安定性,限界加速,旋回性能,高速時ブレーキ性能などの大幅性能向上が要求されてきた.四輪駆動装置の種類としては,大別してパートタイム方式(4WD 切換え式),フルタイム方式(常時4WD 式),スタンバイ方式(車輪あるいはタイヤ空転時動力伝達式)がある.. パートタイム方式二輪,四輪の切換えが可能な方式で,四輪駆動時に前後輪を直結する方式と,センタデファレンシャ6―35図2・69プロペラシャフトの例図2・70 フロントドライブシャフトの例図2・71 リヤドライブシャフトの例ルや可変分配装置をもち四輪駆動時には下記フルタイム方式と同じ機能を有する方式とがある.前者は,必要に応じて前後輪を機械的に直結するもので,四輪が同じ摩擦係数の路面上にある場合,駆動力分配は前後輪の動的荷重配分に比例し,前後輪のいずれかがスリップしてもタイヤのグリップ力に応じた駆動力分配となる.. フルタイム方式前後輪の回転差を吸収しながら駆動力を前後輪へ常時伝達するもので,その中でも,差動歯車機構により前後輪の回転差を吸収するとともに前後輪へ一定の比率の駆動力を分配する固定分配式と,走行状態に応じて変化させる可変分配式に大別される.(1) 固定分配式四輪駆動装置に差動装置を内蔵し,旋回時に生じる前後輪の回転差を吸収するとともに,差動歯車の構造により前後輪へ一定の比率の駆動力を分配する.(2) 可変分配式前後輪の回転速度差に応じて駆動力分配が自動的に変わる方式や,油圧多板クラッチを用いてその作動油圧により伝達トルク容量が制御される方式などを単独で前後駆動系に配置したり,あるいはこれらを差動制限機構として差動歯車に並列配置し,前後輪への駆動力分配を可変にする方式などがある.. スタンバイ方式通常時は二輪駆動であるが,駆動輪が空転した場合のみ空転度合いに応じて他方の車軸に駆動力を伝達する方式である.前後輪間に差動制限装置をおくことで可能となる.g. 推進軸エンジンの駆動力を変速機からタイヤに伝達する軸を推進軸と呼ぶ.エンジンおよび変速機が車体に固定され,車輪が懸架装置で支えられているため,相対変位による長さ・角度の変化を吸収する必要がある.変速機と終減速機をつなぐものをプロペラシャフト(propeller shaft,図2・69)と呼び,終減速機と車輪の間をつなぐものをドライブシャフト(drive shaft)と呼ぶ(図2・70,2・71).これらの推進軸は動力を伝達しつつ長さ・角度変化を吸収する必要があり,各種のジョイントと呼ばれる機構を備えている.特にFF車両の前輪に用いられるドライブシャフトのように,タイヤが操舵され大きな角度がついている状態でもスムーズに動力を伝達する必要があることから,ドライブシャフトには等速ジョイントが採用される場合が多い.h. 終減速装置. 終減速機(final gear box)終減速装置は変速機からの駆動力を変換する減速歯車と,左右の駆動輪に駆動力を配分する差動装置とこれらを支持するハウジングで構成され(図2・72),変速機の出力回転数をタイヤの回転数に変換し,かつ,より大きなトルクを伝達する機能を有する.FR 車の場合,エンジンと車輪の回転方向を合わせる必要があるためハイポイドギヤを用いるのが一般的である.横置きエンジンFF車ではエンジンとタイヤの回転方向が同一となるため,ヘリカ6―36交通機械図2・72 終減速装置の例ルギヤ(はすば歯車)を用いるのが主流となっている.また,変速機と終減速機を一体のケースに内蔵する場合が多く,このような装置をトランスアクスル(transaxle)と呼んでいる(図2・60).. 差動(歯車)装置(differential gear)車両が旋回している状態では左右の駆動輪の回転半径が異なるため回転差が生じる.この回転差を吸収しながらタイヤに動力を伝達するために差動装置(図2・72)が用いられている.差動装置は前述の終減速機と駆動輪を結ぶドライブシャフトの間に設けられており,はすばかさ歯車を組合わせて構成されている.滑りやすい路面で片輪が空転することによる駆動力の低下を防いだり旋回時の運動性能を改善するため,条件に応じて差動を制限する装置(リミテッドスリップデファレンシャル,ビスカスカップリング)や,差動を完全になくす装置(デファレンシャルロック)を備える場合もある.
