# 2・3・2 ハイブリッドシステム(hybrid system)

> **章**: 第2章 自動車
> **節**: 2・3 自動車の構造

ハイブリッドシステムの歴史は古く、1900年代初頭の内燃機関の揺籃期に自動車に適用された例がある。その後の内燃機関の発達により、しばらく自動車用としては顧みられることがなかったが、1970年代、大気汚染問題の顕在化、オイルショックなどを契機にハイブリッド技術が見直され、さまざまな研究が進められてきた。1990年にはアメリカ、カリフォルニア州の大気浄化法の成立により、ZEV(zero emission vehicle)、すなわち電気自動車(EV)の実用化が推進されることとなった。しかしコスト、一充電当たりの走行距離不足など、従来の自動車の代替としてのEV の成り立ちが困難なことが明らかになると、ハイブリッド車などもZEV の代替として認められるようになった。一方、地球温暖化の防止策としてCO削減の国際的な合意が形成されつつある中、COの排出量に少なからず影響を与えている自動車に対しても、日本国内における2010年の燃費規制、ヨーロッパにおけるCO排出量自主規制などの形で低減要請が厳しくなってきている。このような状況の中、ハイブリッドシステムは、排出ガス・燃費低減効果の高いシステムとして注目を集めており、ハイブリッド車を含む低公害車導入に対するインセンティブも各国で導入されるなど、これからの普及が期待されている。a. ハイブリッドシステムの概要ハイブリッドシステムは、技術的には作動原理の異なる2種以上の原動機を有するかあるいはそれらを組合わせて作動させるシステムと定義される。自動車用ハイブリッドシステムにおける原動機の一つは、一般的に用いられるエンジンであるが、もう一つの原動機としては車両エネルギーを回生利用する目的から、電気モータ、油圧モータなどが使われる。近年、EV の開発に伴いモータの小型化技術、システム制御技術、バッテリ技術などが進展し、電気モータシステムの車両適合性が向上したことにより、エンジンと電気モータを使ったハイブリッドシステムの実用化が進んできている。ハイブリッドシステムのねらいとするところは、複数の動力源それぞれの長所を生かし、単一動力源でのシステムをはるかに上回る諸性能を発揮することである。反面、電気システムの追加による質量の増加、コストの上昇が避けられず、商品としての位置付けが問図2・73 ハイブリッドシステムの分類われるところである。燃費を目的としたハイブリッドシステムの基本機能は、エンジンの運転点の最適化と運動エネルギーの回生の二つである。ハイブリッドシステムにおけるエンジン運転点最適化は、熱効率の悪い軽負荷域でのエンジン運転を避けることが基本である。そのために、軽負荷時のエンジン停止やEV 走行(バッテリ電力による走行)、高効率領域運転の多用化などができるよう、システムが構成される。一方、運動エネルギーの回生は、車両減速時にモータを発電機として作動させ、電気的に制動力を得ることにより行われ、回収した電気エネルギーはバッテリなどに蓄えられる。蓄えられた電気エネルギーは、EV 走行や、エンジン駆動力のモータによるアシスト、補機類の駆動などで消費され、エンジンの消費する動力を低減する。ハイブリッドシステムを構成する要素としては、エンジン、変速機構に加え、駆動・回生可能なモータおよびその制御装置、充放電可能なバッテリ、およびこれら全体を制御するシステム制御装置が必要になる。またエンジンを車両の運転中に停止するシステムの場合は、これらに加え、エンジンの始動装置の高性能化や、エンジンで駆動される補機類の機能をエンジン停止中にも維持するための装置または代替手段が必要となり、従来システムの大規模な変更が必要となる。b. ハイブリッドシステムの分類電気モータを併用する自動車用ハイブリッドシステムを動力伝達の形態から分類すると、まず次の2種に大別できる。一つはエンジンからの動力をいったんすべて電力に変換し、電気モータで車両を駆動するシリーズハイブリッドシステム(serieshybrid system)で、動力はエンジンから発電機、バッテリを経由し、駆動軸につながったモータへと直列(シリーズ)に伝達される。このシステムは、バッテリ電力を主たる動力源とし、エンジン駆動の発電機で効率良く電力を補いながら走行するもので、EV に近いシステムといえる[図2・73(a)].もう一方はエンジンの動力and/or電気モータの動力で車両を駆動可能なパラレルハイブリッドシステム(parallel hybrid system)であり、動力はエンジンからの経路と、バッテリからの経路の二つが並列(パラレル)して駆動軸に伝達されるものである。このシステムは主としてエンジンの動力により走行するもので、従来の車両の延長上に位置するシステムといえる。この方式にはいろいろな種類が考えられるが、エンジン動力主体で運転することから、変速機構が必要となる。変速機構としては、図2・74 シリーズハイブリッド構成例機械式変速機と、電気-機械式変速機とが考えられる。前者の変速機機構としては、無段変速機(CVT)、多段式自動変速機や、手動式変速機など、従来型の変速機が使用できる[図2・73(b)].後者では、二つのモータと動力分割機構により、エンジン動力の伝達経路を機械動力による伝達経路と電気動力による伝達経路に可変的に制御し、無段変速機構を構成することができる[図2・73(c)].このタイプは、電気的な動力手段をもつハイブリッドならではの機構であり、シリーズ、パラレル双方の動力伝達機能を有することから、シリーズ・パラレル式ハイブリッドもしくはパワースプリット式ハイブリッドと呼ばれる。パラレルハイブリッドにおける他の分類として、ストロングあるいはフルハイブリッドや、マイルドハイブリッドなどの表現が使われることがある。両者の技術的な定義は明確ではないが、前者はハイブリッド機能を優先し電気系のパワー・エネルギーを理想的に用いたシステム、後者はハイブリッド性能を限定し電気系のパワー・エネルギーを極力リーンにしたシステムとしてとらえられている。前記の分類の中では、機械式変速機を用いる方式が比較的電気系をリーンにしやすく、マイルドハイブリッド構成に適していると考えられる。c. シリーズハイブリッドシステムシリーズハイブリッドシステムの構成例を図2・74に示す。このシステムではエンジンが増速機を介して発電機と接続されており、発電機の電気出力は整流器を介してバッテリに接続されている。また、バッテリはインバータを介して駆動用モータに接続されており、モータ出力は減速機を介して出力軸に接続されている。このように、シリーズハイブリッドシステムではエンジンが駆動軸と機械的には結合しておらず、エンジンの動力を電気エネルギーに変換して動力伝達するということや、バッテリを駆動エネルギーのバッファとしても利用できるため、エンジンは負荷変動の少ない運転ができるという特徴をもつ。またエンジンの出力を車両の平均的な要求動力に設定することにより、エンジンをつねに高効率な負荷状態で運転でき、燃費や排出ガス性能を向上させることが期待できる。車両の運転に必要なパワーは直接的にはバッテリから供給される。バッテリへの充電は、充電状態(SOC: state of charge)がつねに所定の値となるように、エンジン動力で発電し充電制御する。また、回生制動によりブレーキエネルギーを回収することも可能である。d. パラレルハイブリッドシステムパラレルハイブリッドシステムの例を図2・75に示す。図2・75の例は、一つの駆動用モータとベルト式CVT を組合わせたもので、CVT の入力軸上に駆動用モータが結合され、エンジンからの動力は電磁クラッチを介して駆動用モータおよび入力軸に結合できるように構成されている。また、エンジン始動および高電圧バッテリ充電のため、駆動モータとは別にエンジンと結合されたモータジェネレータを有する。軽負荷時は電磁クラッチでモータとエンジンを切り離し、エンジンを停止してバッテリ電力によりEV走行を行うことができる。回生時にも同様にモータとエンジンを切6―37図2・75 走行時や回生時に駆動モータとエンジンを切り離したり、駆動モータとは別に、エンジン始動・充電用モータジェネレータを有することも先の例と同じである。一方、システム構成の簡素化をねらいに、モータ駆動時にエンジンとの切離し機構をもたない構成のものも実用化されている。これらの例では、エンジンとモータはクランク軸に直結もしくはベルトで結合されており、機械式の変速機を経由して車両を駆動する。エンジン動力のアシスト、EV 走行、回生などのHV の基本機能に加え、エンジンの始動機能をもちうるが、EV 走行時や回生時のモータの損失仕事にエンジンを空転させる仕事が加わるため、モータ利用時のエネルギー効率は低下する。そのため、エンジンを使用しない場合に部分的に気筒休止(バルブ閉固定)させ、ポンプロスを低減する機構を採用する例がある。以上は、機械式変速機をもつものであるが、電気-機械式の変速機を構成した例を図2・77に示す。この例では、プラネタリギヤによる動力分配機構と、二つのモータジェネレータで変速機構が構成されている。エンジンからの動力は動力分配機のキャリヤに入力され、サンギヤは発電機に、リングギヤは出力軸と駆動用モータにつながっている。おのおののモータは、インバータ、バッテリを介して、電力が授受できるよう制御される。また、エンジン動力は、電力として伝達される部分[式(2・29)第1項]と、機械動力として伝達される部分[式(2・29)第2項]とに分割されることも示しており、いわゆるシリーズ・パラレル変速機を構成している。また、この機構ではジェネレータのトルクを零に制御することにより、エンジンと駆動モータとの切離しが可能であり、EV 走行、回生時には電気系の損失を低減できる。また、発電機を使ってエンジンの始動も可能であり、走行中のエンジン停止を容易にしている。e. パラレルハイブリッドにおける燃費向上手段。 エンジン運転点の最適化エンジンの熱効率は負荷によって変化し、ガソリンエンジンでは特に軽負荷時の熱効率低下が大きい。そのため軽負荷運転状態では、エンジンで直接駆動するときのエネルギー効率よりも、エンジンの高負荷域で発電しバッテリへ蓄えた電気エネルギーを再びモータで機械的駆動エネルギーに変換するときのエネルギー効率のほうが優れる領域が存在することになる。図2・78は模式的にその状況を示した例である。パラレルハイブリッドにおけるエンジン運転点の最適化は、このような考え方を基本にEV 領域(エンジンを使用しない領域)を設定することにある。EV 走行の領域は、走行条件や回生によるエネルギーの割合、エンジンで発電するときのエンジン熱効率などによって異なるが、小型乗用車では概略10kW 程度の領域である。パラレルハイブリッドのバッテリは一般的には外部充電機構をもたず、エンジン動力による発電と回生により、充電状態(SOC)が維持される。したがって、パラレルハイブリッドにおけるEV 走行は一種のシリーズハイブリッド運転と考えることができる。. 回生(regeneration)回生時、駆動用モータを発電機図2・79協調ブレーキ制御として作動させ、モータトルクを制御することにより制動力として利用する。最大限のエネルギー回生のためには、要求制動力を油圧ブレーキと電気的な回生制動とに分割し、協調的に制御可能なブレーキシステムが必要となる。図2・79に、一制動間の制動力分担の時間的推移例を示す。バッテリあるいはモータのパワーの制約から、制動時間の推移とともに回生による制動力を変化させる必要があることを示している。また、油圧ブレーキとの協調制御をもたず、減速時にエンジンブレーキ機能相当の回生を行うことも行われている。回生によって得られたエネルギーは、前記のEV 走行や補機の電力として消費することにより、エンジンの消費動力をより低減できる。これまで発表されたハイブリッドシステム搭載車では、エンジンの高効率化や無段変速機能によるエンジンの燃費最適運転、車両の各種損失低減などを含めて、市街地走行モードでの燃料消費量を従来車比最大で約半減する。しかし、平均車速の高い使用域では、EV 走行、回生の頻度の減少により燃費向上効果は減少する。f. システム構成要素。 エンジン燃費重視の観点から、高膨張比エンジン、リーンバーンエンジンなど、高効率エンジンが選ばれる。バッテリパワーの併用により、特にエンジン低回転時の動力性能を向上させたり、高効率化によるエンジンパワーの低下を補助することが行われている。. モータ高効率でコンパクトな設計が可能なPM シンクロナスモータが選ばれることが多い。モータに必要な性能はシステムのコンセプト、適用車両により異なるが、数kW から百kW程度の出力が要求される。作動電圧はおおむね出力に比例し、42~650V 程度が選ばれている。自動車用として生産性の良い構造が求められるため、ステータコアを分割式としたり、コイルの巻線方式に集中巻きを採用する例がある。. バッテリ(電源系)高電圧系、低電圧系(12V)の2系統のバッテリをもつ。低電圧系は従来車同様、ECU、照明等の電源であり、高電圧系は車両駆動用モータの電源である。発電機能を効率の良い高電圧系にまとめ、低電圧系にはDC-DCコンバータで電力供給し充電するのが一般的である。高電圧バッテリとしては、比較的低電圧(42V)やエネルギー容量の大きな適用例には鉛電池、より高電圧な適用例にはニッケル水素電池、リチウム電池が使われている。. インバータ(inverter)直流電源をPWM 制御により交流に(あるいは逆の制御も)変換し、PM シンクロナスモータを駆動する機能をもつ。電流を直接スイッチングするものがIGBT等のパワートランジスタであり、近年のモータ技術の進展はこのパワー素子によるところが大きい。
