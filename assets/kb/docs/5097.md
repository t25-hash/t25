# 5・8・1 形鋼製造設備のレイアウト

> **章**: 第5章 製鉄機械
> **節**: 5・8 大中型形鋼圧延設備

形鋼製品は品種および寸法が多種多様であり，工場によっても生
産品種が種々ことなるので，他の鋼材の圧延設備のように工場のレ
イアウトがあまり定型化していない．その中でも代表例を挙げると
図5･58 に示すとおりである．
(ａ)並列式圧延機配列工場
(ｂ)直列式圧延機配列工場
(ｃ)直列X-H式圧延機配列工場
(ｄ)連続式圧延機配列工場
図5･58(ａ)は並列式圧延機配列工場であり，H形鋼以外の比較
的小さな通常の形鋼（steel shapes ，steel sections ）を製造する工
場である．
図5･58(ｂ)は多品種の大型形鋼を製造する直列式圧延機配列工
場で，可逆式圧延機であるブレイクダウン圧延機（break-down
mill ）および中間の圧延機群（図中の記号RUMおよびEM）で鋼
材を何度も往復させて圧延するので，圧延機の台数が少ない．一
方，冷却床以降の精整設備は，多品種の形鋼に対処するため比較的
大きな面積を占有している．
図5･58(ｃ)は(ｂ)の直列式圧延機の仕上圧延機を中間の圧延機
群の直後に配置したものである．
粗ユニバーサルミル（RUM）と仕上ユニバーサルミル
（FUM）によるタンデムリバース圧延を行い生産性向上とレイアウ
トのコンパクト化を図っている．
図5･58(ｄ)はH形鋼（H-steel ，H- section）のほか一部の通常
の形鋼の多量生産型連続式圧延機配列工場で，多くの圧延機台数を
必要とするが，ブレイクダウン圧延機以外は可逆式圧延機がないの
で，形鋼を高能率で圧延できる特徴を有しているが，工場の長さが
長くなる．
