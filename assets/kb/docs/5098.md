# 5・8・2 形鋼圧延機の種類

> **章**: 第5章 製鉄機械
> **節**: 5・8 大中型形鋼圧延設備

a.ブレイクダウン圧延機 ブレイクダウン圧延機（break-
down mill ）は圧延機の原点となる構造の二重圧延機で，上下の水
平ロールで可逆圧延するものである．ロール表面には各製品，サイ
ズに応じた複数の穴型（caliber ）が配置されている．素材断面形
状（スラブ，ブルーム，ビームブランク）から次工程のユニバーサ
ル圧延に必要な中間断面形状まで圧延する．図5･59は素材と製品
の寸法関係例を，図5･60はブレイクダウン圧延機の全体を示す．
b.ユニバーサル圧延機 ユニバーサル圧延機（universal
mill ）はH形鋼を圧延するための特別な構造の圧延機で，図5･61
に示すように同一平面内に上下の水平ロールと垂直ロールの軸心が
ある圧延機である．
ユニバーサル圧延機では，材料は上下の水平ロールの間および上
下水平ロールの側面と左右の垂直ロールとの間に形成される空間で
材料を圧延するので，H形鋼のフランジの先端部は圧延されず，

先端部は丸くなり，またフランジの幅も規定寸法にするのが難しい
ので，図5･62に示すようにフランジ先端部を圧延するエッジング
圧延機（edging mill ）と組合わせて使用される．
c .オープンヨーク式ユニバーサル圧延機 ユニバーサル圧延
機のもう一つの特色は，4個のロール軸が同一平面内にあるためロ
ール交換がしづらいことである．このため種々のロール交換装置が
考案されている．図5･63に示す例は圧延機本体の内垂直ロールの
圧延荷重を支持する部分を開閉できるように設計された圧延機で，
オープンヨーク式ユニバーサル圧延機（open yoke type universal
mill ）という．ロール交換の際には，図5･64に示すように4個の
ロールを同時に圧延機から出し入れすることができ，ロール交換が
短時間でできる特長を有する．
