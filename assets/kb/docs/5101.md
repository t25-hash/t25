# 5・9・1 加熱設備

> **章**: 第5章 製鉄機械
> **節**: 5・9 小型条鋼線材圧延設備

めにローラ矯正機（roller straightener）によって矯正される．
図ッシャ式加熱炉とウォーキング式加熱炉に分類される．
自在である，鋼片にすり傷が付かない，鋼片単重や炉容量なり，さらに品質向上のための均熱への要求が高まり，ウォーキンが一般的に採用されている．
また，省エネルギーを目的としたリジいる．
冷却された形鋼は形鋼としての真直性と正しい断面形状を得るた矯正された形鋼は，形状，寸法，外観などを検査した後，表示，加熱設備は，素材である鋼片（ビレット）の送り機構により，プウォーキング式加熱炉は，均熱性が良い，装入，抽出がに制限が少ない，などの利点がある．
