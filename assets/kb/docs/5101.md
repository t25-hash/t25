# 2・3・7 車体構造

> **章**: 第2章 自動車
> **節**: 2・139 )。システムの機能は、車線検出センサ(カメラ)から得

a.フレーム架装車体（body shell with frame) シャシフレ
ームに，別に作った車体を架装する方式で，走行時の主要な荷重は
フレームに依存するため，あらゆる形式の車体に適用でき，バン，
ピックアップへの変形も容易である．この構造の長所は，路面から
の衝撃がフレームを介して車体に伝わるため，静かでソフトな乗り
心地が得られる点である．
反面，重量増加，コスト高，車高が高くなるなどの欠点がある
が，小型車に比べてこれらの欠点が致命的とならない大型車では，
前述の長所を高く評価して，この形式を採用していることが多い．
フレームの分類とその特徴を見ると次のようになる．
.はしご型フレーム 2本の縦材と数本の横材をおもに溶
図2･143プラットホーム型フレーム

ため乗用車では用いられなくなったが，ピックアップやトラックの

.X形フレーム 平面形をX字状にしたものである．曲
げにより縦材が車体へのねじり荷重を受けるため，ねじり剛性が高
く，また床面足元付近を下げることができる．しかし，中央部の接
合方法によってねじり剛性が左右されることや衝突時のフレームつ
ぶれ特性が調節しにくいことなどにより，現在は用いられていない
（図2･141)．
.バックボーン型フレーム 中央に1本の大きな背骨を通
し，車体を支えるアウトリガを出したものである．車体の支え方が
難しく，また構造により生産性も悪化するためほとんど用いられて
いない（図2･142)．
.プラットホーム型フレーム 床板を車体から離し，フレ
ームに一体化したものである．これをボルト締めして車体を形成す
るが，今日の乗用車の単体構造への移行の過渡的形式である（図
2･143)．
.ペリメータ型フレーム 客室床のサイドシル付近に縦材
を配したものである．このため床面全体を低くすることができる．
前輪の後と後輪の前で縦材がクランクするため，フレーム単体の剛
性は低下するが，車体との結合により剛性を補うよう工夫されている．
今日フレーム付きの乗用車はほとんどこの構造を採用している
