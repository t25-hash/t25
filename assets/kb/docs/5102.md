# 2・3・8 装備品

> **章**: 第2章 自動車
> **節**: 2・149 に示す。

a.概要 従来からのバッテリ，オルタネータ，スタータ，エ
アコン，ヘッドライト，メータなどといった装備品に加え，自動車
への要求の高まり（環境の保全，安全性の向上，快適性の向上）に
応えるため，エレクトロニクスを活用した装備品が増加している
（表2･7)．また，運転者の負荷軽減を目的とした走行支援や，イン
フラストラクチャとの協調により迅速な移動や道路の効率的な利用
を図るITS（intelligent transportation systems ：高度道路交通シ
ステム）関連装備品の装着も進んできている．
さまざまな新しい装備システムが増加すると，ドライバに提供さ
れる情報や操作項目が増加するため，運転に支障を与えない安全な
操作/表示技術，すなわちHMI （human machine interface）技術
の開発が重要となる．図2･150はインストルメントパネルの構成例
である．情報の重要度によって提供する方法や表示位置を決めてい
る．また，複雑な操作は音声認識や，操作機能が集約されたマルチ
スイッチを手元に置き，操作性を向上させている．
複数のコンピュータが協調して制御を行うシステムが増加してお
り，コンピュータ間のデータ通信量が増加している．そのため車内
LAN（local area network）が整備されている．
b.エアバッグ（air bag restraint system) シートベルトに
よる乗員保護性能を補助する装置であり，通常SRS（sup-
plemental restraint system）エアバッグと呼ばれる．乗員の胸や
腰を拘束するシートベルトの働きに合わせて，瞬時にクッションと
なるよう膨らみ，乗員の頭部や胸部のステアリングホイールやイン
ストルメントパネル，ピラー等への衝突の可能性を低減する．
.システム構成 エアバッグ，衝撃検出センサ，衝突判定
とエアバッグの作動およびシステムの故障診断を行うセンタエアバ
ッグセンサアセンブリー，故障を知らせるウォーニングランプ等で
構成されている（図2･151)．
.作動メカニズム（例：カーテンシールドエアバッグ)
まず，衝突による強い衝撃がボデー骨格を介してサイドエアバッグ
センサに伝わる．サイドエアバッグセンサは，センサ部に発生した
減速度を内蔵コンピュータで演算し，演算された値があらかじめ設
定されているしきい値を超えた場合，衝突と判定し，センタエアバ
ッグセンサアセンブリーへ点火信号を送る．センタエアバッグセン
サアセンブリーは点火信号を受信すると，インフレータの点火装置
に点火電流を流す．すると，インフレータからガスが発生し，瞬時
にSRSカーテンシールドエアバッグが展開する（図2･152)．
