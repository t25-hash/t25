# 2・4・2 連結車(trailer)

> **章**: 第2章 自動車
> **節**: 2・161 )である.岩石,土砂,チップ等をショベルローダやベルト

動力源と運転を主体としたけん引車としてのトラクタと,荷物搭載を主体とした被けん引車トレーラからなる連結車は,相互に自由に結合,分離できる方式で,大量輸送,荷役分離による輸送効率向上のほか,分割不可で長大な荷物輸送に用いられる輸送形態である.欧米では長い歴史を有していたが,海上コンテナの導入と同時に日本はもちろん,世界中で急激に普及した.したがってその荷物である海上コンテナの大きさ,重量から車両に関する法規制が,車両全高が3.5mから3.8mに,車両総重量も20tからホイールベースによっては28tに変更されるなどの影響を受けている.それでも海上コンテナの大きさ,重量は規制をオーバしているものがあり,その場合には個別に基準緩和を受けて使用されているのが現状である.したがって今後とも法規制の変更が予測される.連結車の方式としてセミトレーラ,フルトレーラの2方式に大別される.さらにその組合せ,多重連結方式やその他の専用変形の連結車がある.しかし,海上コンテナの多さから連結車のほとんどがセミトレーラ式連結車である.a. セミトレーラ式連結車(semi-trailer,図2・168)この方式のトラクタは前後方向の傾きを可動とした一軸カプラか,前後・6―59(a) セミトレーラ式(海上コンテナ)(b-1) フルトレーラ式(バン型)(b-2) フルトレーラ式(センタアクスル型)(c) ダブルストレーラ式(バン型)(d) ボールトレーラ式(橋げた搭載時)図2・169トレーラ式連結車各方式左右方向の傾きを可動とした二軸カプラを架装し,トレーラのコントロールのため,ブレーキ用エア供給用(サプライライン)とコントロール用(コントロールライン)のエアカップリング,さらに各種電気信号を伝える電気のカップリングが装着されている.カプラはトレーラとの連結器で,第5輪とも呼ばれている.その連結時カプラに加わる上下荷重は第5輪荷重と定義されていて,駆動方式4×2のトラクタでは第5輪最大荷重が4~11.5t,駆動方式6×4の場合は11~18tが一般的である.一方トレーラとしては走行装置・懸架装置と制動装置等が装着されたシャシに荷台が架装され,前輪がなく格納可能なランディングギヤ(補助足)とキングピン(連結器),トラクタからのコントロールを受けるエアカップリングおよび電気のカップリングが装着されている.キングピンは万国共通になっており,トラクタのカプラと合わせ相互に互換性がある.荷台の種類により,海上コンテナトレーラ[図2・169(a)],バン型トレーラ,重量物用低床トレーラ,ダンプトレーラ,タンクトレーラ等に分類できる.海上コンテナトレーラには万国共通のコンテナの緊締装置(ツイストロック)がシャシに装着されている.ダンプトレーラの場合にはトラクタ側にPTO駆動の油圧ポンプを装着し,カップリングを介してトレーラ側に送られるようになっている.土砂ダンプトレーラが2000年1月に日本でも認められ生産が開始された.b. フルトレーラ式連結車[full-trailer,図2・169(b)]この方式のトラクタは,けん引用装置として通常のトラックの車両後端にピントルフックを装着し,被けん引トレーラのコントロールのため,ブレーキ用エア供給用(サプライライン)とコントロール用(コントロールライン)のエアカップリング,さらに各種電気信号を伝える電気のカップリングが装着されている.ピントルフックにはけん引時の前後荷重はもちろん,旋回時には横荷重が加わることを配慮し十分なフレーム強度にしておかなければならない.また,トレーラのドローバ先端の連結環(ルネットアイ)とのガタによる衝撃や発進時,減速時,走行時のトレーラ前後動による衝撃を緩和するため,ばね組込み式や圧縮空気を利用した6―60交通機械ショックレス型の構造が一般的である.ルネットアイ,ピントルフック取付け部の寸法はISOやJIS に定められている.一方このトレーラは,ドローバの自重の一部を除き全重量が前輪,後輪で支持され,セミトレーラのようにトラクタ側には重量が加わらないのが一般的である.前軸はターンテーブルで回転する構造である.そのため関節点が2点(ルネットアイ部とターンテーブル部)となり後退運転が難しくなるため,ターンテーブルには回転を拘束するジャックナイフ防止装置を備えるのが一般的である.また,トレーラをセミトレーラとして転用可能とするため,前軸部をトレーラから分離できるドーリー方式を採用したものもある.また,台数は少ないが連結スペースが少なく,荷台長を拡大し,関節点が1点のセンタアクスル型フルトレーラ[図2・169(b-2)]も製品化されている.c. ダブルストレーラ式連結車[doubles trailer,図2・169(c)]セミトレーラの後部にドーリーを介してもう一つのセミトレーラを連結したものをダブルストレーラという.アメリカでは27フィート(8.2m)×2→全長約20m,40フィート×2→全長33mのダブルス,さらに1両加わった3両連結のトリプルスが特定道路で運行されている.日本では1971~1981年に試作運行が行われたが,現状では本格運行まで至っていない.d. ポールトレーラ式連結車[pole trailer,図2・169(d)]長大物を運搬するためターンテーブルをトラクタ,トレーラに設け,積荷の両端をそれに固定する方式の連結車である.トラクタはフレーム後端にピントルフックを有し,トレーラはターンテーブルのみ架装できるフレームと車軸で構成され,トレーラの前端には収縮可能な連結用ポールを有する.積載物によっては連結ポールを省略する場合がある.
