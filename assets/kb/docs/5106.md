# 2・4・3 特殊車

> **章**: 第2章 自動車
> **節**: 2・161 )である.岩石,土砂,チップ等をショベルローダやベルト

作業目的に合った特殊構造・装置または機能を有する自動車で,一部はん用自動車部品を流用している以外は既成シャシと構造が異なるものが多い.その一例を以下に記載する.(1) 農業用としては耕起,運搬用の営農用トラクタや,樹木の抜根,岩石の除去,不整地の整地などに使用される開墾用トラクタや,作物の刈入れ用のコンバイン車等がある.(2) 建設作業用の油圧式ショベル,トラクタショベル,ブルドーザ,スクレーバ,ダンプトラック,道路整備のスノープラウ車,ロータリ除雪車,塩カリ散布車等がある.また,荷役作業用としてはフォークリフト,運搬台車等がある.(3) 人員輸送用のバスの外観と,車としての走行機能を使用し,内装を各種装置・設備を設定した特殊車として,商品展示・販売車,放送中継車,宣伝車,診療車,移動銀行,図書館車,引越し専用車等がある.(4) 自動車としての機動性を生かして,電車等の軌道上をも走行可能なようにした軌陸車(図2・170)がある.軌道上を走るときにはタイヤを外し鉄輪を前,後輪に装着するか,車体内に格納していた鉄輪を自動的に下げるなどで軌道上の走行を可能にしている.車のタイヤで駆動させるタイヤ駆動方式とT.M.PTOを使い,油圧ポンプ,油圧モータで鉄輪を駆動して軌道上を走る方式がある.
