# 2・5・1 原動機

> **章**: 第2章 自動車
> **節**: 2・5 二輪自動車

二輪車の原動機は、排気量が50cmの単気筒から1800cmの水平対向六気筒まで非常に多岐にわたっている。また、4サイクルと2サイクル機関がある。四輪車と大きく違う構造上の特徴は、クランク系とトランスミッション系が同じオイルを使用する一体構造であること、および商品の特性上高回転・高出力が要求されることである。実用車は、経済性の面から単気筒が主流である。スポーツ車は、ユーザニーズが多彩なため、単気筒から六気筒まで多くの種類が存在する(図2・178)。一方、実用車からスポーツ車まで環境改善としてエミッション・燃費に対する要求が近年大変強まっている。そのため、出力・燃費・エミッションを両立させるための動弁系可変技術の導入が見られる。図2・179は、バルブ作動を油圧制御にてコントロールする直押しバルブ休止機構である。図2・180は、吸排気の脈動効果を各回転レンジで制御する構造である。原動機冷却方式については、従来は空冷式が多く採用されていたが、近年はエミッション低減の面から水冷式が数多く採用されるようになってきた。燃料供給系も従来主流だったキャブレタ方式から四輪車で主流のFI(fuel injection)方式に変わりつつある(図6―62交通機械図2・180 排気管長および集合方式可変機構図2・181 フューエルインジェクション二輪車の原動機には高回転・高出力が要求されるため、4サイクル機関の動弁機構は頭上カム軸型式(OHV・OHC)が一般的に用いられている。2サイクル機関は構造がシンプルであったが、やはり環境問題に対応するため排気タイミングコントロールバルブなどの可変技術を投入しているので、構造が複雑になってきている(図kW/L以上が普通とされている。
