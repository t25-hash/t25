# 2・5・2 変速機

> **章**: 第2章 自動車
> **節**: 2・182 ).比出力は,市販車では50~90kW/L,競走車では150

二輪車の変速機は,小型軽量化を図るためクラッチ室を含めてクランク室と一体構造となっている場合がほとんどで,クランク軸から歯車を介して駆動される構造が一般的である.変速機は,左足で変速レバーを踏み下げたり引き上げたりして行うことで,歯車列のかみあいを一段ずつ順次に切換える構造となっている.四輪車のように中立位置を中心に任意の位置へ変速できる構造ではない.変速段数は,商品特性から実用車では三段から五段,スポーツ車では五段から六段変速の構造が多い.しかし,スクータの変速機は簡単な操作が要求されるため,ベルト式の無段変速機を組込んだ構造が多い(図2・183).クラッチは変速機軸に設けるのが一般的であるが,遠心式自動クラッチを用いる場合と,車の進行方向と平行にクランク軸を配置した,いわゆる縦置きクランク軸方式では,クランク軸に直結の構造も少なくない.クラッチ形式は,ノイズが少なく耐久性に優れた湿式多板式が一般的である.しかし,競走用車両のクラッチ構造は,軽量化などの要求から乾式多板式が使われている.一方,スクータ等の遠心式自動クラッチにはシュー拡張型の乾式と,ディスク型の湿式が用いられている.クラッチ操作は,ハンドルの左側に設けられたクラッチレバーを握って行う.図2・182 2サイクル機関の排気タイミングコントロールバルブ図2・183 ベルト式無段変速機
