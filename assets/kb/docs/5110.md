# 2・5・3 フレーム

> **章**: 第2章 自動車
> **節**: 2・182 )。比出力は、市販車では50~90kW/L、競走車では150

フレームは、原動機、動力伝達装置をはじめ座席、荷物積載装置などを保持するとともに、前後輪をステアリング、サスペンション機構を介して連結する車両の基幹となる部品である。また、運転者や荷物の荷重、走行中の路面からの衝撃力に耐えなければならない強度部材であると同時に、操縦性・安定性の観点から、軽量で高剛性であることが要求され、二輪車全体のスタイルを決める重要な要素でもある。フレームの形態は、使用目的、原動機の形態・大きさによりさまざまな種類がある。構造的には複数の部材で空間を構成するスペースフレーム構造と、1本の主部材で構成されるはり構造のものがある。スペースフレーム構造にはクレードル型とダイヤモンド型がある。一方、はり構造にはバックボーン型とステップスルー型がある。a. クレードル型フレーム[cradle frame、図2・184(a)]原動機全体を上下の部材で囲む揺りかご状の形状のフレームである。原動機下部を抱き込む部材(ダウンチューブ)の形状によりシングルクレードル、セミダブルクレードル、ダブルクレードルに分類される。フレーム単体で比較的強度・剛性が高く、オフロード車や大型車に用いられる。b. ダイヤモンド型フレーム[diamond frame、図2・184(b)]ダウンチューブを省略しているのが特徴であり、原動機を強度・剛性部材の一部として利用する形式のフレームである。フレーム下部が省略できる分、軽量化が図れる利点があるが、反面、フレームに(b) ダイヤモンド型フレーム(a) クレードル型フレーム(c) ダイヤモンド型フレーム(ボックスチューブタイプ)(d) ステップスルー型フレーム図2・184 フレームの形態加わる力が原動機にも及ぶため、ケースがひずむなどの欠点もある。これまではおもに中軽量車に用いられていたが、図2・184(c)に示すように、ボックス型断面の太いチューブでヘッドパイプとスイングアームピボットを、原動機の両サイドを通り直線的につなぐなどの適切な設計手法により、十分な強度・剛性を確保できるようになり、最近では大型の高性能車に使われることも多い。また、同様な考えをダブルクレードルに適用したものや、スイングアームピボットをエンジン側に設け、メインフレームにピボットを付設しないタイプもある。c. バックボーン型フレーム(back bone frame)原動機を抱き込む部分がなく、ヘッドパイプから後部にかけ背骨状の部材で構成するはり構造のフレームである。部材は主として鋼板プレス成形にて製造されることが多い。d. ステップスルー型フレーム[step through frame、図2・184(d)]主としてスクータやファミリー車に用いられる形式で、乗降の容易さを目的としてヘッドパイプとシートの間に空間を設けている。主部材は下部を通ることから、アンダボーン型とも呼ばれる。フレームを構成する部材は一般的にはスチールのパイプ材やプレス材が用いられるが、最近では軽量化と高剛性の両立を目的としてアルミニウムの鋳造材と押出し材を組合わせた構造のフレームが多く採用されている。主部材にバフ仕上げやアルマイト処理など施すことにより、高品位で美しい外観が可能で、デザイン要素としても重要な位置付けになっている。
