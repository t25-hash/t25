# 3・1・1 社会的背景

> **章**: 第3章 鉄道
> **節**: 3・1 鉄道一般

鉄道とは一般に,左右2本のレール上を,鉄の車輪を用いて走行する車両により,人または物を輸送するシステムをいう.索道,モノレール,無軌条電車(トロリバス)なども,特殊な鉄道として分類される.イギリスで1825年に世界で初めて鉄道が開業して以来,線路と車輪を用いる構造にあまり変化はないが,システムとして鉄道を見た場合,それぞれの時代の最新技術を取り入れ,最も効率的な輸送機関として今日まで発展してきた.その後,鉄道はヨーロッパ,アメリカで発展し,日本にはイギリスから輸入された.日本で鉄道が初めて開通したのは1872年,新橋~横浜間である.関西地区では1874年に神戸~大阪間が開通している.輸送システムとして鉄道を見た場合,その得意分野は,大量輸送の必要な,大都市の通勤輸送,高速鉄道による都市間輸送である.車両は車両限界と呼ばれる枠の中から,はみ出さないように設計,製作され,線路によって案内されるので,周りの建築物との間隔を小さくできる.よって鉄道は,狭いスペースで,大きな車両を何両も連結して運行できるので,大都市の大量輸送に適している.また,線路の直線性を極めて精度良く整備することができ,その線路に確実に案内されるため,乗り心地も良いので高速鉄道にも適している.鋼鉄のレールの上を鋼鉄の車輪で走行するため,極めて走行抵抗が少ない.このため大量輸送を少ないエネルギーで行うことができる.これらの特徴により,省エネルギー,排ガスなど環境面から見ても他の輸送機関よりも勝っている.一方,鋼鉄どうしの動摩擦を利用して加減速を行うため,雨,雪などを考慮してあまり高い加減速度はとれない.したがって,こう配(grade)についても一般的には3.5%程度が最大である.車輪と線路が鋼鉄どうしであるため,電流が流れることを利用し,走行するための電気のマイナス側をレールからとるとともに,信号システムにも利用している.a. 安全性(safety)鉄道は極めて安全なシステムである.新幹線が開業以来,乗客の死亡事故が零という記録からも推測できるように,信号,運行システムなど,故障しても安全側になるようにシステムが組まれている.これらのシステムは,長い歴史の中で,事故を教訓に改良されてきたもので,現在は,ほぼ完璧な安全システムができあがっている.1人・km当たりの乗客死亡事故発生率も,自動車,飛行機に比べ,低い値となっている.b. 日本の鉄道日本の鉄道技術については,開業当時はイギリスの技術が輸入されたが,これを手本に国産技術を徐々に確立してきた.初めての国産蒸気機関車は1893年に鉄道庁(当時)神戸工場で完成している.初期の鉄道は,機関車で客車をけん引する動力集中方式であり,機関車が蒸気機関車,電気機関車,ディーゼル機関車と発達する間も,その主たる方式は1950年代まで変わらなかった.1955年以降になると,長距離列車に初めて動力分散方式の電車が用いられるようになった.それまでは,電車は,床下にさまざまな機器を取付けているため,騒音,乗り心地,快適性の点で客車列車に劣ると考えられていたからである.その後,動力分散方式の電車は,技術の発達により,快適性の面でも客車に劣らず進歩したため,また,その使い勝手の良さの点で客車列車に勝ることから急速に発達した.1964年に東海道新幹線が開業したときの列車形態は電車であったが,これは軸重の平均化,折返し設備の簡素化などの理由から,今後の高速化には動力分散方式が適していると判断したためである.これに対し,ヨーロッパ,アメリカでは,高速列車も動力集中方式が主流となっている.最近,ドイツでは日本の新幹線を見習って電車方式高速列車の運行を開始している.営業走行速度は,在来線の場合,最高速度から停止までのブレーキ距離(braking distance)が600m以下という決まりが以前あったため,それに対応して地上設備が作られており,このことが制約になって130km/hが最高速度である.しかし,踏切のない区間では例外で160km/hで営業している区間がある.新幹線では,現在300km/hまで営業運転しているが,さらに他交通機関との競争力をつけるため,速度向上の技術開発を推進している.この場合のいちばん高いハードルは環境対策である.世界的に見ても鉄道の営業最高速度は,現在300km/hである.
