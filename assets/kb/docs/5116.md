# 3・1・4 レール、架線との相互作用

> **章**: 第3章 鉄道
> **節**: 3・1 鉄道一般

a. レール・車輪との相互作用レール・車輪系の相互作用に影響を与える要素としては、レールと車輪の間に作用するクリープ力(creep force)または粘着力(adhesion,adhesive force)がある。また、車輪踏面とレールの形状も相互作用に影響を与え、これにより、接触点の等価踏面こう配が決まり、輪軸の幾何学的挙動が決まる。このクリープ力と輪軸の幾何学的挙動が相まって、車両の加速・減速の性能、高速走行での走行安定性、曲線でのステアリング、車輪・レールの摩耗・疲労損傷などの現象が影響を受ける。. 車輪-レール間のクリープ力50kN の力で直径860mmの標準的な車輪をレールの中央部付近に押付けるとその接触部には車輪とレールの弾性的たわみにより、直径10mm程度の円に近いだ円が生じる。また、車輪がレール上を運動しながら転動する場合、この接触だ円部には前後方向の滑りが生じる。車輪が左右にずれると左右方向の滑りが生じる。このとき車輪踏面のこう配により、一方の車輪半径は大きくなり、他方の車輪半径は小さくなる。一対の車輪は同じ速度で進むので、径の大きい車輪は後ろ向きに滑り、径の小さい車輪は進行方向に滑ることになる。さらに車輪はその垂直軸の回りに回転滑り(スピン)も生じる。このような滑りにより接触面には前後方向接線力、左右方向接線力、垂直軸回りのモーメントが発生し、これらをクリープ力と総称する。図3・7に車輪・レール接触部の滑りと力の関係を示す。クリープ力は滑り速度が小さいときは滑り速度に比例して増大していくが、さらに滑り速度が増していくと滑り率1%程度において最大値を示し、この最大クリープ力を粘着力という。図3・8に滑り率と接線力の関係を示す。図3・7 力行、ブレーキ時の粘着問題車輪が駆動回転する(力行)ときに、滑り率1%程度以上の早い滑り速度で車輪を回転させると、空転が激しくなり、粘着力は減少し、モータのエネルギーが空転に費やされ、前進力が低下する。ブレーキ時には、過大なブレーキ力をかけると車輪の滑り速度が増大し、車輪滑走(wheelslide)が生じる。この粘着力の変化の傾向については、レール・車輪表面の水、油、汚損物質、接触面の粗さなど多くの要因が関連し、特に、水の存在と表面粗さが大きな役割を果たす。2物体間の接触をミクロに見ると、凸部どうしが接触しているが、そこに水が入ると水膜の厚さが増して凸部どうしの接触が減り、真の接触面積を減少させるために、粘着係数が低下すると解釈される。図3・9に水膜と表面粗さの概念を示す。この理由により、降雨時は滑走が増え、編成の中では多量の水にさらされる先頭車の車輪に滑走の発生が多い。粘着力を増すためには、表面に微小な突起(波長:50~100 m、突起高さ:1~2 m)を積極的に多数形成することによって、降雨時においても粘着係数の低下を抑えることができる。このために、車輪に増粘着研磨子を押付けたり、踏面制輪子に研磨成分を入れて車輪表面に適度な表面粗さを形成させている。増粘着研磨子の場合は、車輪の駆動回転時の空転を防止するためにも動作させている。. ステアリング(操舵、steering)挙動の問題車輪踏面(wheel 曲線通過性能(curving performance)が良く、レールに与える横圧が小さいこと曲線を通過するときは、等価踏面こう配の大きいほうが、左右の車輪回転径差が大きく得られて車輪-レール間の横圧が減り、フランジ摩耗やレール摩耗が減る。しかし、等価踏面こう配が大きいと蛇行動が起こりやすくなる。このように、走行安定性と曲線通過性能それぞれに要求されることがらは相反している。(4) 摩耗しても、踏面形状の変化が小さいこと車輪とレールの接触により、車輪踏面は次第に摩耗し踏面形状は変化する。それに伴い、走行安定性や曲線通過性能は変化するので、ある時期において車輪踏面形状を削正し、性能を回復させることが必要になる。摩耗した踏面は車両の形式の差異や線区によるレールの相違により多少の相違はあるが、ほぼ類似の摩耗形状に落ち着く。これを摩耗踏面(worn wheel profile)と称する。よって、新製時や踏面削正後の踏面形状を摩耗踏面になるように設計し、所要の走行安定性や曲線通過性能を満足すれば、経年によっても、車輪の性能は大きな変化がなく車輪削正周期を延伸できることになる。そのような摩耗踏面に近い形状の踏面を円弧踏面といい、近年多用されている。これまでの踏面は円すい踏面であり、摩耗による形状変化が大きい。b. 架線・パンタグラフとの相互作用車両のパンタグラフは架線をしゅう動しながら進行し、架線はそれにより上下波動を発生する。その状況を図3・11に示す。相互作用の弊害を克服するための工夫が必要になるが、架線・パンタグラフの接触しゅう動の問題について以下に説明する。. パンタグラフの追随性と架線の波動伝ぱの問題パンタグラフが架線を押上げながら走行することにより、架線には上下変位の波が発生して車両より早く前方に伝ぱし、ハンガで反射してパンタグラフのほうに戻り、そこにパンタグラフが進入することにより、接触力変動が生じ離線(contact break)の源となる。離線が発生すると、すり板の摩耗が増大し、騒音、電波障害の問題が発生するので、パンタグラフ側では、集電舟体のばね系を追随性が向上するように設定する。架線側は、パンタグラフと架線の接触力変動を少なくするために、架線の張力を増大したり、支持点付近にばね特性のある防振ハンガなどを用いて架線の押上量をできるだけ均一にする。. 架線とパンタグラフすり板とのしゅう動摩耗の問題集電系の摩耗の形態は、架線の波状摩耗や局部摩耗、パンタグラフのすり板の摩耗のように、架線とパンタグラフの動的相互作用に大きく影響されるが、そこに使用される材料の効果も無視できない。パンタグラフのすり板は自身の摩耗低減の観点からだけではなく、架線の摩耗が少なくなるようにも配慮されて材料が選定されている。すり板の材料は大別して、焼結合金とカーボン材料がある。焼結合金はカーボン材料に比べて機械的強度や電気伝導性では優れているが、架線の摩耗を低減する機能は劣っている。カーボン材料はグラファイト(黒鉛)成分が摩擦面に膜を形成し、架線の摩耗が少ないので、架線のメンテナンスから見れば大きなメリットがある。図3・9水膜と表面粗さの関係6―72交通機械
