# 3・1・5 環境問題

> **章**: 第3章 鉄道
> **節**: 3・1 鉄道一般

鉄道車両の環境問題は、鉄道沿線の環境保全に関するものと、地球環境保全(environmental protection)に関するものに大別される。a. 沿線環境保全鉄道車両に関わる沿線環境問題は、騒音、振動のほか電波障害や電磁誘導障害などがあるが、新幹線を代表とする高速鉄道において騒音・振動が特に問題とされてきた。. 騒音・振動に関する法体系1993年に公布された「環境基本法」を基本とし、環境を保全するうえで維持することが望ましい基準として、「新幹線鉄道騒音に関わる環境基準について」「環境保全上緊急を要する新幹線鉄道振動対策について」「在来鉄道の新設又は大規模改良に際しての騒音対策の指針について」が示されている。これらの環境基準の達成に向け、さらに暫定基準が示され、これに沿って沿線環境対策が進められている。. 騒音発生源と対策列車の走行に伴う騒音(noise)は、その発生源によって、転動音、集電系音、空力音、構造物音、機器音に分類でき、それぞれについて騒音低減策がとられている。転動音は、車輪がレールの上を転がることによって発生する騒音で、双方の接する面の凹凸が大きく影響する。低減策としては、滑走検知装置による車輪のフラット発生防止、車両の軽量化、特にばね下質量の軽量化が行われている。集電系音は、トロリ線とパンタグラフのすり板のしゅう動音、離線によるアーク破裂音、パンタグラフの風切音であり、高速走行をする新幹線で特に問題となる。低減策としてパンタグラフの周囲に壁を構成する構造がとられている。また、翼型、シングルアーム型など低騒音型のパンタグラフも開発されている。アーク破裂音については、離線を抑制するパンタグラフ構造や、複数のパンタグラフをケーブルで並列接続する方法がとられている。空力音は、車両の走行によって生じる空気の乱れから発生する騒音で、ほぼ速度の六乗に比例するため、高速走行する新幹線で特に問題となる。車体高さを低くして防音壁で遮音できない部分を減らすほか、車両の表面を平滑化することで空気の乱れを低減する構造が採用されている。構造物音は、車輪がレールの上を転がることによって発生する地上構造物の振動が発生原因であり、低減策は転動音と同様である。機器音は、車両の機器類から発生する騒音である。低減策としては機器自体の低騒音化や車体の遮音性の向上が行われている。以上のほか、新幹線においてはトンネル突入時に入口で発生した圧縮波がトンネル内を伝ぱし、出口でトンネル微気圧波として放出する際に発生する騒音がある。新幹線車両では圧縮波の圧力こう配を小さくするため、急激な車体断面積の変化がない先頭形状を採用している。. 振動発生源と対策列車の走行によって発生した振動が地盤振動として伝わる。低減策としては、構造物音、転動音と同様に車輪のフラット発生防止、車両の軽量化、特にばね下質量の軽量化が有効である。b. 地球環境保全近年、地球規模での環境破壊が大きな問題となっている。なかでも、地球温暖化の原因となるCOなど温室効果ガスは削減目標が設定されている。鉄道は他の交通機関と比較すると単位輸送量当たりのCO排出量が少なく、環境保全面で優れている。特に電車、電気機関車については電気を動力源とするため、運転中に直接COを排出しない。さらに、車両の軽量化やVVVFインバータ制御・電力回生ブレーキの採用により、消費電力を低く抑えることが可能となり、間接的なCO排出量も低減されてきている。また、地球を取り巻くオゾン層の保護のため、フロン(fluoro-carbon)の製造、使用が制限されている。鉄道車両では空調装置や機器の冷却用の冷媒としてフロンが使用されてきた。特定フロン(CFC: chloro fluorocarbon)については生産全廃となり、代替フロン(HCFC: hydro chloro fluorocarbon,HFC: hydro fluoro-carbon)が多く使用されているが、HCFCについては、オゾン層破壊と地球温暖化防止の点から全廃される計画であり、HFCについても地球温暖化ガスとして、COと同様に排出が規制されている。c. 循環型社会への対応地球の資源と環境を保護するため、廃棄物を再資源化する循環型社会への対応が求められている。鉄道車両については、老朽化して廃車となった車両を解体、材料ごとに分離し再資源化を図っている。200系新幹線電車(アルミニウム車体)では重量比で99.7%が再資源化されている(このうち5.3%は固形燃料として使用)。
