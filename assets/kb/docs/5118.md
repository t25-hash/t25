# 3・1・6 鉄道法規(railway regulation)

> **章**: 第3章 鉄道
> **節**: 3・1 鉄道一般

鉄道を取り巻く法規関係は,行政改革の方針に沿って大幅な見直しが行われた.手続きについては,2000年5月の鉄道事業法改正で認定鉄道事業者制度が発足し,技術力の認定された事業者はその程度に応じて手続きの簡素化が図られた.技術基準については鉄道の構造,性能,取扱い等を規定していた五つの省令を統合し,「鉄道に関する技術上の基準を定める省令」が2002年3月に施行されている.a. 新省令の内容従来の省令(ministerial ordinance)が,車両の各部位の具体的な構造(例:通路幅は550mm等)を定めた仕様規定としていたのに対し,性能規定ではもつべき機能(例:通路は安全かつ容易に通行できる等)を定めることで,鉄道事業者の技術的自由度を高め,新技術の導入や線区の個別事情への柔軟な対応を可能とした.性能規定化された省令に対し,鉄道事業者の技術的判断や国土交通省での許認可等の判断基準を明確にするため,拘束力のない解釈基準を策定している.新省令の施設および車両に関わる体系を図3・12に示す.b. 車両の基本機能に関わる内容. 車両限界(rolling stock gauge)車両限界は鉄道事業者が定めることとし,従来の車両限界は,解釈基準で標準として例示された.なお,車両限界は,車両が直線軌道の中心に位置した状態で超えてはならないエリアを示す.曲線では,車両の偏倚(へんい)に応じて側部の限界を拡大するとしているが,この数値も事業者が定める.. 車両の重量等軌道等の負担力は,鉄道事業者により異なるため,具体的な数値の規定はない.鉄道事業者では,地上設備の規定と連携し,線区・速度・車軸配置等を考慮した軸重に関する規定や数値を実施基準で定めている.. 車両の走行装置走行装置に対しては,1軌道の損傷防止,2走行線区の最小曲線半径に対応した車軸配置,3軌道からの衝撃に耐える懸架装置,4列車の最前部での排障機能を有すること,が規定されている.最小曲線半径や軸配置は鉄道事業者により異なるため,解釈基準にも通過可能な曲線半径の数値は記されていないが,最大の固定軸間距離や車輪各部の寸法は普通鉄道,新幹線鉄道別,各軌間別に明示されている.. 車両の動力発生装置など電気回路に対し,1絶縁破壊による感電・火災防止,2誘導障害防止(鉄道関係の電気回路限一般車両施設との適合車両の性能線路係員車両の構造停車場車両の設備施設および車両の構造および保全電気設備異常時の対応道路との交差運転運転保安設備施設および車両の保全経過措置特殊な構造を有する鉄道図3・12 新省令の体系定),3円滑な集電,4パンタグラフの一斉降下,5過電流保護と回路の開放機能,6雷電圧等からの保護,が規定されている.また,機関等に対し,1壁等の発熱発火防止,2燃料系統の漏えい引火防止と旅客への影響防止,3排気管のガス,熱の旅客等への影響防止,4過熱等の異常時の保護措置,5排出物による火災防止,が規定されている.いずれも,具体的な対処方法については解釈基準に示されている.. 車両のブレーキ装置等ブレーキでは,次の基本機能のみが規定されている.1確実に減速,停止が可能.2乗務員室からの操作で連動して作用.3列車が分離した際に自動的に作用.4急速な停止が可能(非常ブレーキ機能).5振動,衝撃などによる作用への障害の防止.6制動力を連続して作用できる機能.7制動力の供給確保ができない場合の発車防止.8常用ブレーキのバックアップ(車種に応じて).9留置中の転動防止車(車種に応じて).解釈基準では,常用,留置,保安の各ブレーキの機能と,車種別に設けるべきブレーキの種類が示されている.また,車種別,ブレーキ種類別の具体的なブレーキ率も示されている.新幹線では,速度帯ごとの減速度が示されている.また,いくつかの事故例を参考に,両運転台で単車走行を行う車両に対するブレーキ装置の二重系化等の措置が新たに示されている.c. 車体の構造および設備に関する内容いずれも省令は概要のみで詳細は解釈基準に記載がある.. 著しい騒音を軽減するための構造新幹線車両が対象だが,事故復旧や保守用車は例外となる.. 乗務員室の構造操作を妨げない構造と,旅客が容易に触れないような設備の保護および視界の確保が求められる.. 客室の構造窓の機能,換気,照明,通路,座席および立席,便所の設置を規定している.座席と立席は,解釈基準にも具体的寸法はないが,参考数値は,JIS E7103:1994「通勤用車両の車体の設計通則」に記載がある.. 旅客用乗降口の構造バリアフリー対応で,すきま・段差の縮小,床面の滑りにくさ,戸の開閉する側が容易にわかることを規定している.. 貫通口・貫通路編成内他車への避難のほか,側面から避難できない区間を走行する場合の前頭部からの避難を規定している.. 乗務員室の設備力行・ブレーキ制御に必要な装置と,列車間の安全を確保する装置(ATS 等の保安装置)の車上装置の搭載を規定している.. 火災対策各車種共通の火災発生防止の機能と,旅客車に限った火災発生,延焼防止の機能を定めた.各部の耐燃焼性および車両用材料の試験方法については,解釈基準に記載されている.これらは,鉄道別・車種別・部位別に一覧表でまとめられ,チェックが容易となっている.. 停電時の設備の機能火災対策同様,停電時の安全の確保と避難に必要な機能が集約された.対象の装置等は解釈基準に一覧表で記載している.d. その他の構造,取扱い,検査に関わる規定車両に関連する軌道等の構造物,信号・電車線などの構造,運転取扱い,車両等の検査方法等も「鉄道に関する技術上の基準を定める省令」にまとめられ,対応した解釈基準も作られた.車両等の検査周期については,省令の記載はないが,国土交通大臣の告示で示されている.また,普通鉄道における非常ブレーキの停止距離の規定(旧:運転規則)は,新省令では「支障箇所までに停止できる非常制動距離を考慮」となり,解釈基準で「600m以下とすることを標準」とされたが,併せて「迅速な列車防護の方法による場合は,その方法に応じ6―73た非常制動距離とすることができる」とされ,在来線の速度向上に配慮した記述となった.
