# 3・2・1 駆動電動機(traction motor)

> **章**: 第3章 鉄道
> **節**: 3・2 動力発生装置と制御装置

電気車の運転に用いられる電動機は主電動機または駆動電動機と呼ばれる。つねに衝撃を受けるので耐振性が要求され、かつ台車に取付けるため寸法上の制約を受けること、雨水、じんあいによる汚損がはなはだしいので防水・防じんが要求されることなどが通常の電動機と大きく異なる。使用される回転型電動機の種類であるが、直流電動機としては主磁極の界磁巻線と電機子巻線が直列に接続された直巻電動機、並列に接続された分巻電動機、並列および直列に接続された複巻電動機がある(図3・13)。交流電動機としては交流整流子電動機、誘導電動機、同期電動機などがあるが、近年では誘導電動機をインバータ制御する方式が主流となってきた。また交流整流子電動機は古く16(2/3)Hz 電化のドイツ、スイスなどでおもに採用されたが、車両性能や整流面で不利であり、最近では整流器式直流電動機駆動から、誘導機・同期機駆動に移っている。a. 直流電動機[direct current (DC)motor]. 直巻電動機制御技術が未熟であった時代、駆動電動機の性能がそのまま車両の性能となるため、起動トルクが大きく、速度の上昇とともにトルクが減少するという速度-引張力特性が適しており、電圧急変などに対する過渡整流も優れているので、古くから駆動電動機として多用されてきた。反面、整流子、ブラシというしゅう動部分があり、b・ii項の誘導電動機に比べて信頼上、保守上の問題があるため、誘導電動機をインバータ制御する方式が主流となってきた。. 分巻電動機界磁制御装置が必要であり、その過渡応答性や過渡整流など車両用としては不利な面があるが、電機子・界磁を独立にチョッパ制御や位相制御して欠点を補い、速度制御や回生ブレーキを行う車両も見られる。. 複巻電動機直巻界磁による整流の過渡応答性と並列運転時の電流不平衡を改善し、分巻界磁のチョッパ制御などにより速度制御・回生ブレーキなどを容易に行うことができるが、直巻電動機に比べて構造が複雑で整流性能も劣る。b. 交流電動機[alternating current (AC)motor]交流電動機は、定常運転状態でその回転運動がエアギャップの回転磁界と同期するかしないかによって、同期機と非同期機に分けられる。この回転速度を同期速度という。誘導電動機は非同期機に分類される。. 同期電動機(synchronous motor)同期電動機は、固定子と回転子からなり、回転子に永久磁石または電磁石でNS の磁極を作り、固定子コイルに三相交流を流して回転磁界を作ると、磁極は回転磁界に引っ張られて回転する。固定子の電源周波数を変えることにより、回転数を制御する。直流電流をインバータで三相交流に変換するが、電動機の回転数はインバータの出力周波数に一致しているので、車輪径の差によりそれぞれの電動機の回転数が異なることのある車両用には、1台のインバータで複数の電動機を駆動するシステムを採用できない。そのため、1インバータ、1電動機制御となる。また、回転子に磁極を作るために、スリップリングで電源を供給しているので、メンテナンスの面でやや不利となる。この方式は特にフランス[TGV-A(Train 定格(rating)定格とは回転機に保証された使用限界をいう、と定義されており、出力に対する使用限度を定めるとともに、電圧、回転速度、周波数などを指定する、と規定されている。定格の種類の中で、連続定格と短時間定格があるが、鉄道車両では通常連続定格と1時間定格が用いられている。. 温度上昇と絶縁種別駆動電動機がどれだけの負荷に耐えられるかは車両の性能を左右するが、これを決める要因が温度上昇限度である。電動機の内部は銅コイルを絶縁物で絶縁した構造となっている。温度上昇によって影響を受けるのは絶縁物である。絶縁構成物は長時間高温にさらされるため、機械的振動や冷・熱サイクルを受けると絶縁劣化が進む。そのため温度上昇限度が決められている。絶縁物の温度は試験走行中測定するが、荷重の多い営業走行中の値を測定することはできないため、通常は温度シミュレーションも行う。駆動電動機の負荷は断続負荷であり、電動機の温度上昇を推定する場合は、その損失の大部分が銅損によるものとみなされるため、その温度上昇が電流の二乗に比例するとして等価的な電流を算出する。=욡dここで、:瞬時電流、:走行時間。は電流実効値で、rms電流(二乗平均平方根電流)と呼ばれる。走行条件に応じたrms電流を算出することにより、温度上昇を推定し、電動機が負荷に耐えうるか判定できる。温度上昇限度は、電動機に用いた絶縁材料の種別に固有な耐熱特性により定まる。現在規定されている絶縁種別と温度上昇限度を表
