# 3・2・2 電気車の制御装置

> **章**: 第3章 鉄道
> **節**: 3・6 に示す。

鉄道車両は速度範囲が広く、かつ、けん引重量の変化、線路条件による所要けん引力の変化があるので、これに対応するために駆動電動機の回転数と回転力を制御しなければならない。制御の方法を定める大きな要素としては、駆動電動機に直流電動機を使うか交流電動機を使うか、架線から受電する電力が直流であるか交流であるかの2点である。最近では新製される電気車の制御は三相誘導電動機をインバータで制御する方法であるが、簡単に直流電動機を用いた制御方法も述べる。a. 直流電動機の制御(DC motor control)直流(直巻、分巻、複巻)駆動電動機は、原理的に電機子電流とトルク、誘起電圧と速度が対応するので、一般に電流制御、電圧制御、界磁制御の三つの基本制御方式の組合せによって制御される。電流制御は、起動時や発電ブレーキ時など、所要トルク(したがって、所要加速度・減速度)を制御するのに用いる。また電圧制御は、所要の速度が得られるように電動機電圧を制御する。起動時は電流を制御しつつ速度の増加とともに電圧を高めていき、発電ブレーキ時は同じく電圧を低くしていく。界磁制御は、界磁のアンペアターン(界磁電流に界磁巻線を乗じたもの)を変えることによりトルクや出力を制御する。表3・6 駆動電動機の温度上昇限度(JIS E 6102:1999)絶縁種別BFH固定子巻線同期主電動機の回転界磁巻線抵抗法130K155K180K200Kスリップリング電気温度計120K120K120K120Kかご形回転子およびダンパ巻線電気温度計温度上昇はすべての巻線やその他の部分を危険にするほど大きくしない。駆動電動機としてはおもに直流直巻電動機が用いられ、抵抗制御と組合わせるのが一般的である。また、電機子チョッパ制御を用いて低損失かつ回生ブレーキが可能な制御もある。直流複巻電動機の分巻界磁を界磁チョッパにより制御する方法や、直巻電動機のままで界磁添加励磁制御を行うことによる強め界磁によって力行性能向上や回生ブレーキを行う例もある。b. 交流電動機の制御(AC motor control)半導体、特に大容量GTO,IGBT の発達により可変電圧可変周波数(VVVF:variable voltage variable frequency)インバータによる誘導電動機が実用化されている。インバータ制御車の主回路を図3・16に示す。この方式の特徴は、保守の大幅低減、駆動電動機の高出力化・小型軽量化、粘着性能向上、省エネルギー、信頼性向上などである。(1) 保守性直流電動機のようなブラシがなく、取換えが不要である。整流子がないため、整流子の保守が不要。また、整流不良によるフラッシュオーバが発生しない。制御装置の可動部分の保守が大幅に削減できる。(2) 小型・軽量化整流機構がないため、回転数を高くできる小型化が可能。台車内に大容量モータを搭載できる。特に新幹線のように大出力が必要な車両に有利となる。(3) 粘着性能向上空転、滑走制御が容易となり、期待粘着性能を高くとれる。電動車比率を下げることができる。また、ブレーキ時の電気(回生)ブレーキ負担を多くすることが可能となり、省エネルギー率向上とともに、付随車の制輪子摩耗の低減が可能である。わが国の電気車で一般的な誘導電動機のインバータ制御について説明する。誘導電動機は交流電源の周波数によってほぼ一定の回転速度で回転しようとする。これの回転数、回転力を制御するためには、可変電圧可変周波数(VVVF)の駆動電源が必要である。誘導電動機のトルク特性式は次のようになる。=욦ここで、:係数、:電動機線間電圧、:インバータ出力周波数、욦:滑り周波数。インバータのトルク制御は図3・17、表3・7に示すように、定トルク、定出力、特性の領域に分けられる。
