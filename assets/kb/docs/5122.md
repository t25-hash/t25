# 3・2・4 機関およびトルクコンバータ

> **章**: 第3章 鉄道
> **節**: 6・3 節に述べる.

a. 機関(engine)鉄道車両用機関の使用上の特徴は,駅間において高負荷,アイドルを頻繁に繰返して使用することである.しかもこう配線区においては長時間にわたり連続して最大の出力を出し続ける能力も合わせてもたなければならないので,機関にとっては過酷な使用条件である.したがって機関の出力決定に当たっては,使用条件に合わせた試験方法も定められており(JIS E5303:1989「鉄道車両用ディーゼル機関試験法」),機関の出せる最大出力よりいくらか下回った定格で使用することが多い.駆動機関のほかに,客室照明等に供するサービス用電源機関があるが,この場合は一定回転数で長時間にわたり連続して使用するので,使用条件は駆動用よりも楽である.しかし,保守のことを考慮して,一般的には同系列の機関を使用することが多い.表3・8に現在わが国で使用されている代表的な鉄道車両用ディーゼル機関を示す.1970年代前半のオイルショック以後,ディーゼル機関については燃費の優れている直接噴射式が従来の予燃焼室式に取って代わってきているが,鉄道車両用機関でも最近開発されたものはすべて直接噴射式となっている.機関のシリンダ配列は,ディーゼル機関車用,客室に搭載する発電用についてはV 形,縦型であるが,動力分散式のディーゼル動車では,床下に機関をつり下げる必要があるので,機関の全高を抑える意味から横型機関が主体となっている.シリンダ数の多いものでは水平対向型機関である.質量当たりの出力を向上させるため,最近の機関は排気タービン過給器付きであり,機関車用とディーゼル動車用の一部ではさらに給気を冷却する中間冷却器付きのものもある.中間冷却器の冷却には,冷却専用回路をもつ一部の機関車以外,すべて機関冷却用の冷却水を共通に用いており,冷却水温は約80°C前後である.b. トルクコンバータ(torque converter)液体変速機は機関からの入力を油の流れるエネルギーに変換し,これを羽根車に当てて再び回転エネルギーに変換して出力する装置である.特性は入力回転が一定であっても出力回転は一定でなく,回転数が小さいときは出力トルクが大きく,回転数が上昇するに従って出力トルクが小さくなる.鉄道車両の動力装置は,車両を起動開始するときにその駆動力が大きく,加速するにつれて駆動力が小さくなることが望ましいことを考えれば,液体変速機の出力特性がいかに鉄道車両に適しているかが理解できる.特に車両の起動時において,機関は停止からスタートできないので,液体変速機と機関を組合わせることにより,初めて鉄道車両用の動力装置として使用できる.しかし一方では,液体変速機の効率は最大でも85%程度であり,最大効率点より出力軸の回転数が大きくても小さくても効率が低下するので,実際の使用は以下のように工夫している.一つは車両の起動時から比較的低い速度域の間に液体コンバータ形式頭上弁,横型,水冷頭上弁,横型,水冷頭上弁,横型,水冷頭上弁,横型,水冷頭上弁,V 形,水冷を介して動力伝達を行い,高い速度域では液体コンバータを介さず機械伝達を行う方式であり,他の一つは,いくつかの液体コンバータの組合せまたは一つの液体コンバータと歯車変速機を組合わせて,それぞれの速度域に対して液体コンバータの最も効率の高い領域を使用する方式である.前者は比較的構造が簡単であることからディーゼル動車に用いられることが多い.この方式で以前は変速一段,直結一段の合計二段を有するものが一般的であった.図3・20にこのタイプのDB 型変速機を示す.気動車は車速に応じて,変速(コンバータ運転)と直結を使い分けて走行している.エンジンのもつ最大出力を,いかに有効に利用できるかを考えた場合,変速走行ではトルクコンバータの性能によって決まるのに対し,直結走行ではほぼ車速に比例しエンジンの回転数によって決まる.前述のDB 型のように直結1速だけの変速機では,最大出力を利用できるのはその車両が最高車速で走行するときだけである.c. 多段変速機高速度域だけでなく,中速度域でもエンジンの最大出力を引出せれば加速力の向上とこう配登坂時の速度の向上を図ることができるはずである.図3・21に直結二段付き変速の性能曲線の一例を示す.直結に高低2速の切換えを設けることにより,中速度域から直結運転が可能となり加速力の増加とともに変速運転域が少なくなるので,燃費の向上も図られる.10%こう配での均等速度を直結1速だけの場合と直結2速付きの場合とで比較すると,前者では図3・21により66km/h(A DW 6型変速機の効率曲線液体変速機の制御は,それぞれの速度に応じて速度段と直結段の切換え,また,使用液体コンバータの切換えを行うことが主体である.ディーゼル動車においては,従来は変速段と直結段の切換えは主幹制御器での手動操作で行うのが主流となっていたが,最近は操作の熟練度によらずスムーズにできる方法として入出力軸の回転数が一致するときに電子コントローラを用いて,自動的にクラッチを投入する方式を採用してきている.この方式によれば,乗客に与える変直切換え時のショックも少なくでき,また,液体変速機自体の耐久性にもプラスになる一石二鳥の効果がある.機関車の場合は,使用液体コンバータの切換えはすべて入出力軸の回転数比を検知して自動的に行う方式である.この場合,クラッチはないのでディーゼル動車のようにクラッチ投入時のショックについては問題ないが,充排油のタイミングをうまく制御することが切換え時のショック低減対策に結びつくことになる.図3・26にDW
