# 3・2・5 駆動装置

> **章**: 第3章 鉄道
> **節**: 6・3 節に述べる。

一般に、鉄道車両の駆動装置とは、電動機や内燃機関などの動力源から車輪までの、車両走行のための回転力伝達の入出力を含む系の全体をいう。一方、動力伝達装置とは前記の駆動装置から動力源である電動機、内燃機関などと車輪を除くものである。これらは、電気車の場合と液体式ディーゼル車の場合とではその機構が大きく異なっている。電気車の場合は、駆動装置という呼称はこの動力伝達装置の範囲を表しているが、ディーゼル車の場合は、動力伝達装置の主要な機能を占めるトルクコンバータ(液体変速機)が3・2・4項で詳述されているので、3・2・5項の駆動装置では電気車とディーゼル車の減速装置および軸継手についての概略を述べる。電気車の駆動装置の機構は、一般的に電動機を台車に組込む形式であり、車体その他台車以外に組込む形式と区別して、電動機台車装架式と呼ぶ。さらにこれを電動機から車輪に動力を伝達する機構によって分類したのが図3・29である。一方、ディーゼル車の場合は、駆動装置への入力源であるトルクコンバータ(液体変速機)は車体にぎ装されているため、トルクコ6―79電動機つり掛け式滑り軸受式転がり軸受式カルダン式平行カルダン式短カルダン軸式電動機中空式中空カルダン軸式直角カルダン式垂直カルダン式クイル式電動機つり掛けクイル式電動機つり掛けリンク式無歯車式無変速式電動機台車装架式図3・29電動機台車装架式の駆動装置の分類図3・30 電動機つり掛け式駆動装置ンバータ出力軸と台車装架の減速装置入力軸は、レール方向に設置されるプロペラシャフト(推進軸)で結合されている。電気車の場合とその形態は大幅に異なっている。駆動装置の機能は、電動機軸端およびトルクコンバータ出力軸に生じるトルクと回転数を、歯車、たわみ軸継手などによって車輪に伝達して、車両を駆動する。一般に歯数比だけ車両は減速されて走行する。この場合、歯車による減速の目的は次の事項である。(1) 原動機の効率の良い回転数領域はかなり高いので、その回転数を低下させ、車両として使用可能な速度に減じること。(2) 原動機のトルクを増大し、車輪に作用する引張力を増すこと。また、電動機やトルクコンバータと輪軸との間には、走行中に相対変位が必然的に発生するので、この変位を許容しつつトルクを伝達しなければならない。この機構により装置の分類が生じる。a. 電気車の代表的な駆動装置。 電動機つり掛け式駆動装置この形式の駆動装置は古くから使用されており、電気機関車等に使用されている。図3・30に示すように、電動機の一端を平軸受により車軸上で、他端を台車枠横ばりで支える・構造簡単・取扱い容易な駆動装置である。しかし、車両が高速化されるに従い、走行時車軸からの振動、衝撃が直接電動機に加わることに対する考慮が必要である。すなわち、平軸受のすきま、ノーズ部のばねなどによる電動機の車軸と台車枠に対する運動があり、また、小歯車が電機子軸に片持に取付けられていることなどによる軸のたわみなどもあって電機子軸と車軸間、すなわち大小歯車間の平行度を正確に保持することが困難なため、歯車の歯すじ方向の寸法修正をしたり、歯車を強くするための形状を大きくしたり、特殊な材料、熱処理などを改善しなくてはならない。また、取付誤差が大きく、歯車かみあいから生じる騒音、振動も大きいので、高速車両用としては不適格な構造である。. カルダン式駆動装置高速車両用駆動装置として、電動機を完全に台車に装架して、ばね下質量である車軸との結合部にたわみ軸継手を用いるように考案されたのが各種カルダン式駆動装置である(図3・31)。カルダン式の一種である短カルダン軸式の場合、歯車装置は図けられ、小歯車は剛性の大きい歯車箱(鋳鋼製またはアルミニウム合金製)に高精度の軸受を介して取付けられている。歯車箱は小歯車側で、台車枠より緩衝ゴムを介してリンク装置によりつられ、重量を支えるとともにトルク反力を受けている。歯車箱の荷重は台車枠と車軸に半分ずつ負荷されている。ここ6―80交通機械図3・31 無歯車式駆動装置これは、車軸に電動機を直接取付ける方式である。これまで、電動機を直接車軸に取付けると、回転数を低くせざるを得ないため、十分な電動機出力が得られなかった。近年、LRV(light rail vehicle、軽量軌道車両)において、独立回転車輪を駆動する方式に採用されるようになった。また、DDM(direct drive motor、直接駆動電動機)と称して大出力の電動機が開発され、一般の電車にも採用されてきている。高速回転をする歯車装置がないので、走行騒音の低減とメンテナンスの簡素化が期待される。b. ディーゼル車の代表的な駆動装置ディーゼル動車の駆動装置も種々の方式があるが、図3・36に代表的な事例を示す。機関の出力は車体に装架されたトルクコンバータ(torque converter、液体変速機)により速度に応じた所要トルクに変換される。その後、自在継手を有する推進軸により、台車に装架された歯車装置に伝達される。推進軸が車体-台車間の変位を吸収しながらトルクを伝達する機能を有する。歯車装置には、前後進のための逆転機能と、歯数比による回転数変換とトルク変換機能があり、車輪に所要トルクが伝達される。最近のディーゼル車では、逆転機能は液体変速機内部に組み込まれる例が多い。c. 動力伝達系の振動車両用駆動装置は、その特徴として使用速度域が広く、振動の共振速度を常用速度域から避けることが難しい。それゆえ、駆動軸系の振動については十分な検討が必要である。振動系の概要は図3・31に示されている。電動機(電機子욇θ)から軸継手(욧)、歯車装置(大歯車、小歯車)、車軸と結ばれるねじり振動系(トルク욇)のほかに、歯車箱(욁)が歯車箱つりゴム(욁)につられ、車軸を中心として振り子運動をするモードが存在する。d.
