# 6・2・4 電気方式，その他

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・2 ゴムタイヤ式新交通システム（AGT）

新交通システムの電気方式についても，小型の車両に搭載するたである．
わが国の標準化仕様では原則として直流給電とされているり，採用例はほぼ拮抗している．
電車線（power rail）についてないような配慮が求められることから，鉄道で一般的な架空線方式った災害から旅客を保護している．
めのさまざまな工夫はなされているが，基本的には電車技術の応用が，路線の長さや運行頻度などから三相交流にも有利な場面があは，できるだけ占有空間を小さくし，また都市美観上の妨げとならではなく，側方または中央走行面の下に設置するいわゆるサードレール方式がとられている．
