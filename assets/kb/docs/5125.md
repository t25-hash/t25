# 3・3・1 鉄道車両のブレーキシステム

> **章**: 第3章 鉄道
> **節**: 3・3 ブレーキ性能

鉄道車両のブレーキ(brake)は列車を減速し、または停止させる機能を有する。ブレーキは連結して運転する車両に連動し、ブレーキ力を連続して作用させること、組成された車両が分離しても自動的に作用すること、通常使用するブレーキとは別に独立して作用するブレーキを有することとされている。在来線用車両は通常使用する常用ブレーキ(service braking)装置およびこれとはブレーキ指令系統等を独立させた保安ブレーキ(security braking)装置を備えている。常用ブレーキ装置には電源が絶たれた場合にも作動する非常ブレーキ(emergency brak-ing)機能が備わっている。新幹線車両は保安ブレーキ装置に代わり、これに相当する装置として緊急ブレーキ(urgent braking)装置を備えている。また、ブレーキは運転保安装置に連動している。ATS(自動列車停止装置)が動作すると、列車には非常ブレーキが作用する。6―82交通機械ATC(自動列車制御装置)が動作すると常用ブレーキが作用して、列車を段階的または一段で減速させる。
