# 6・3・2 常伝導（常電導）磁気浮上式鉄道

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・3 磁気浮上式鉄道

a.外国の動向 車両の支持を常伝導磁石または永久磁石で
行う浮上式鉄道を，一般には常伝導磁気浮上式鉄道と呼んでいる．
この種のシステムが初めて実用化されたのはイギリスで，1984
年にバーミンガム空港とバーミンガム国際展示場（国鉄）駅を結ぶ
約600mの単線2本の路線（バーミンガムピープルムーバ）であ
った．これは，車上の電磁石と地上の鉄レールとの吸引力による磁
気支持・案内を行い，推進は車上一次方式のリニア誘導モータ
（LIM:linear induction motor ，電源コイル側を車両に搭載し，二
次側のリアクションプレートを地上に敷設する方式）で行う，最高
速度が50km/ h程度のシステムである．このシステムは順調に稼
働していたものの，交換部品調達の困難さから10年程度で運行を
停止し，現在はケーブル駆動のシステムが代わって運行されている．

ドの実験線（全長31.5km）での走行実験を行ってきた．この間，

れ，リスクなしで実用化が実現できると判定された．このシステム

b.日本の動向 日本では，日本航空が1974年からHSSTと
して開発を開始し，それらの成果を受けて，中部エイチ・エス・エ
ス・ティ開発が1991年から名古屋市大江の全長約1.5kmの実験
線で走行試験を行った．1993年に運輸省（当時）により，安全性
に問題がないシステムであると判断され，特殊鉄道構造規則にも記
載された．このシステムは，車両の支持・案内は車両に搭載した電
磁石と鉄レールとの吸引力により行い，駆動は，車上一次方式のリ
ニア誘導モータにより行う，バーミンガムピープルムーバと同様の
方式である．ただし，バーミンガムが車体中央下部にリニアモータ
一次側を搭載しているのに対し，HSSTは車体の両側のモジュー
ル内に搭載されている（図6･14)．
このシステムの特徴は，車体の支持を逆U字形の電磁石とU字
形のレールとの吸引力により行うのであるが，電磁石とレールの間
隔が8mmとなるように制御される点にある．また，案内方向に
ついては，U字形電磁石と逆U字形レールの配置によって，電磁
石が左右どちらかに変位しても復元力が働くことにより非接触で案
内力を生じる仕組みとなっている．しかし，高速になるにつれて，
案内ギャップ変動は大きくなる傾向があるため，万一の場合に備え
て案内スキッドを有しており，モジュールがレールと衝突すること
のない構造となっている．また，万一浮上制御が故障した場合は，
モジュールがレール上に落下することになるが，モジュールに搭載
しているスキッドにより受けることとなり，リニアモータやモジュ
ール本体がレールに衝突することはない．
駆動に関しては，車上一次方式のリニア誘導モータで行うが，リ
ニアモータとリアクションプレート（レール上面に張付け）間に働
く垂直力が電磁石とレールとの間に働く吸引力と反対方向なので，
この垂直力を極力小さくするために，滑り周波数を大きくとる制御
を行っている．また，リニア誘導モータは高速域になると端効果に
より性能が落ちる（推進力の低下）が，HSSTは最高速度が100
km/ h程度なので，それほどの推進力の低下は見られない．
このシステムは愛知県の東部丘陵線（藤が丘～八草間約8.9
km）に導入することが決まり，2000年に設立された愛知高速交通
（第三セクタ）により，軌道法に基づく特許を取得して2005年3月
から営業開始した．図6･15に東部丘陵線で走行している車両（リ
ニモ，Linimo）の概観を示す．
