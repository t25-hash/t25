# 3・3・2 ブレーキの種類

> **章**: 第3章 鉄道
> **節**: 3・3 ブレーキ性能

鉄道車両のブレーキは、通常、車上の動力を利用して作用させる。ブレーキ力を作用させる方法として車輪とレール間に働く粘着力を利用する粘着ブレーキと、粘着力を利用しない非粘着ブレーキに分けられる。ブレーキの構造別分類を図3・41に示す。a. ブレーキ力源機械ブレーキは、空気力または油圧を動力源として運動エネルギー熱をエネルギーに変換する。電気ブレーキは、主電動機を発電機として使用し運動エネルギーを電気エネルギーに変換してブレーキ力とする。流体ブレーキでは、流体の運動エネルギーは熱エネルギーに変換される。手ブレーキは車両を留置させる場合に、側ブレーキは貨車を低速から停止または留置させる場合に使用する。b. 機械ブレーキ機械ブレーキは、圧縮空気や油圧を用い車輪やディスクに制輪子を圧着させて得た摩擦力をブレーキ力とするブレーキ方式である。動力源別に空気ブレーキ、油圧ブレーキまたは総称して摩擦ブレーキとも称される。空気ブレーキは動力源として980kPa未満の空気圧を利用し、油圧ブレーキは15MPa程度の油圧を動力源とする。機械ブレーキは車両停止時にも継続してブレーキ作用を維持できるため、バックアップブレーキとしても不可欠である。. 踏面ブレーキ(tread brake)車輪の踏面に制輪子(brake block)を圧着し、発生する摩擦力によってブレーキを作用させる方式。制輪子にはレジン系、鋳鉄系、焼結合金が使われる。また、雨等の湿潤時にも摩擦係数の低下を防ぎ、レール-車輪間の粘着力を高めるため、硬質金属粒子、セラミックス等を分散複合させたり、金属ブロックを埋め込んだ増粘着制輪子がある。空気シリンダ、てこ、すきま調整器等を小型化して鋼製枠に収納した制輪子の圧着装置をユニットブレーキという[図3・42(a)].図3・43に、制輪子等の速度に対する摩擦係数の変化(瞬間摩擦係数)を示す。一般に、制輪子の摩擦係数は低速で高くなる。電気発電抵抗ブレーキ電力回生ブレーキ渦電流ディスクブレーキ粘着ブレーキ機械踏面ブレーキ(空気)ディスクブレーキ(空気、油圧)鉄道車両のブレーキ流体エンジンブレーキ排気ブレーキコンバータブレーキ電気電磁吸着レールブレーキ渦電流レールブレーキ電力回生ブレーキ(リニアモータ)非粘着ブレーキ機械圧着レールブレーキ図3・41 電気ブレーキ駆動用モータ(主電動機と称する)を発電機として使い、回転を止めようとする電磁力をブレーキ力とする方式で、発電抵抗ブレーキ(rheostat brake)、(電力)回生ブレーキ(regenerative brake)がある。主電動機には直流直巻電動機、直流複巻電動機、交流誘導電動機等がある。主電動機コイルを励磁するため、ブレーキ力の立上りは機械ブレーキに比べて遅い。また、制御電源が絶たれるとブレーキ力が出力されない欠点があるが、制輪子の摩耗を大幅に低減できる。. 発電抵抗ブレーキ主電動機の発生電力を車両に搭載する抵抗器に消費させてブレーキ力を得る方式。直流電動機では、主電動機を直巻発電機として使用する界磁自励式と、界磁を電動発電機等により励磁する他励式がある。. 電力回生ブレーキ主電動機の発生電力を、電車線を介して他の電気車で消費させてブレーキ力を得る方式で、省エネルギー効果が大きい。一方、直流区間では電車線電圧が限度値より高くなる場合や他列車等の負荷がない場合には、回生失効が起こりやすい。直流電気車の回生ブレーキ制御方式では、直流直巻電動機の電機子電流を制御する電機子チョッパ方式、直巻界磁を制御する界磁添加励磁方式、誘導電動機によるVVVFインバータ制御方式等がある。交流電気車の回生ブレーキでは、直流電動機の場合はサイリスタブリッジ回路と他励インバータを用いたサイリスタ位相制御方式が使われる。誘導電動機の場合は発電された交流電力をインバータで直流に変換し、さらに別のコンバータで交流に変換して変圧器で昇圧し、回生電力を架線に戻す。. 誘導電動機によるブレーキ作用GTO(ゲートターンオフ)サイリスタ、IGBT(絶縁ゲート双極型トランジスタ)等の高電圧、高電流をスイッチングする素子を用いたインバータ(直流-交流電力変換装置)、パルス幅変調(PWM)方式等により、三相交流周波数制御を容易に行えるようになり、省保守で軽量な交流誘導電動機を主電動機として利用できるようになった。車輪に連結している誘導電動機回転子の回転速度に対し、固定子に付加する回転磁界の速度をわずかに低くするよう制御すると、回転子を止める方向のトルクが発生し、これを回生ブレーキ力としている(図停止間際までブレーキ力を出力させることができる。d. 渦電流ディスクブレーキの例(新幹線)渦電流を誘起させて、発生する磁気抗力をブレーキ力とする方式を渦電流ディスクブレーキ(図3・45)という。台車に取付けた電磁コイルをレールに相対させ、レール面に渦電流を発生させてブレーキ力を得る方式を渦電流レールブレーキという。e. 流体ブレーキ(hydraulic brake)内燃車両では、常用ブレーキの補助またはこう配区間における降坂時の抑速ブレーキとして、流体ブレーキが使われることがある。. エンジンブレーキディーゼル機関のピストンとシリンダ間の摩擦力をブレーキ力とする方法である。. 排気ブレーキ機関の排気管出口に設けた排気シャッタを閉じ、ディーゼル機関のシリンダ内空気圧を高めることによりブレーキ力とする方法である。. コンバータブレーキ内燃車両のトルクコンバータ内の油の摩擦やかくはん抵抗を利用したブレーキ方法である。ブレーキ時に車輪から回転力を伝えられているロータ内に作動油を充てんすると、油はポンプ作用で外周のステータに送り出され、油相互の摩擦によって作動油の速度が失われてブレーキ力を生じる。f. レールブレーキ(rail brake)非粘着ブレーキの一種である。台車に装架された電磁コイルによってブレーキシューをレールに吸着させて、摩擦力をブレーキ力とする電磁吸着式レールブレーキ、電磁コイルでレールに渦電流を発生させて非接触状態でブレーキ力を得る渦電流レールブレーキ、ばね力によりシューをレールに圧着させる圧着レールブレーキ等がある。レールブレーキの高速列車への実用化例はドイツ鉄道に見られるが、わが国には実用化例はない。
