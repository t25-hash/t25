# 3・3・3 ブレーキ性能と制御

> **章**: 第3章 鉄道
> **節**: 3・44 )。電動機の特性により高速域ではブレーキ力が低下するが、

a. ブレーキ指令伝達方式ブレーキ指令を列車の各車両に伝達する方式には空気圧による方法、電気信号による方式がある。. 電磁直通空気ブレーキ(electromagnetic straight airbrake)列車に引き通された直通管の空気圧を増減させてブレーキ力の制御を行う方式である(図3・46)。ブレーキハンドルを扱うと、先頭車の電磁直通制御器によって全車一斉にブレーキ電磁弁と緩め電磁弁が励消磁して直通管圧力が高められ、これを指令値とし、各車の中継弁の作用で元空気だめの高圧空気をブレーキシリンダに送気する。また、直通管圧力を電流値に変換して電気ブレーキを作用させている。列車分離時にブレーキが作用するように、常時加圧されている非常ブレーキ電磁弁回路または自動空気ブレーキを6―83図3・46 電磁直通空気ブレーキ併用している。. 電磁自動空気ブレーキ(automatic electro-magnetic airbrake)列車に引き通されたブレーキ管に圧力空気を込め、圧力を減圧させることによりブレーキ制御弁を働かせてブレーキ力を制御する方式。自動空気ブレーキでは列車最前部から順次ブレーキが作用するが、本方式では各車両にブレーキ電磁弁と緩め電磁弁を設け、ブレーキハンドルによるブレーキ管の減圧と並行して、電気のオンオフ指令により全車両一斉にブレーキ管の圧力制御を行う。列車分離時には自動的に最大ブレーキ力が作用する。. 電気指令式ブレーキ(electric command brake)先頭車からのブレーキ指令を電気的に各車両に伝達し、各車両の読換え装置でブレーキ力に変換して列車全体の空気ブレーキの制御を行う方式。電気信号を用いるため電気ブレーキの制御を精度良く行える。空気ブレーキと電気ブレーキとのブレンディング(協調)、速度・粘着特性に沿ったブレーキ力制御などのきめ細かな応答性の高いブレーキ制御を行うことができ、かつ、空気配管を大幅に削減できるため、最近の車両に多用されている。電気指令の伝達方法としてディジタル式とアナログ式の2方式がある。ディジタル式では常用指令7ノッチ分を3本の指令線で伝送し、ブレーキ制御器で回生ブレーキ力と空気ブレーキ力の分担、応荷重演算を行ってブレーキ力指令を出力する(図3・47)。アナログ式では電流または電圧に応じた連続的なブレーキ力制御が行える。常時加圧の非常ブレーキ線を列車に一往復させる方法により、列車分離時には自動的に非常ブレーキが作用する。b. ブレーキ性能車両のブレーキ性能とは、ブレーキを作用させたときに所定の列車運転速度から目標の速度に減速または停止させる性能をいい、空走時間、ブレーキ距離、実平均減速度等で表す。. 空走時間(idle running time)ブレーキハンドルを扱った後、列車の有効な減速が開始されるまでの時間をいう。目標ブレーキ力の63.2%が出力される時間で表すこともある。ブレーキ指令伝達方式により異なるが、非常ブレーキの場合、電気指令式空気ブレーキ列車では約0.5~1.0s、自動空気ブレーキコンテナ列車では約4sである。. 停止ブレーキ距離列車のブレーキ初速度(m/s)としたとき空走時間に走行した距離(m)とレールに作用するブレーキ力および列車抵抗を勘案して演算したブレーキ距離(brak-6―84交通機械ing distance)を加算して停止ブレーキ距離とする。ブレーキ力、抵抗値等は速度により異なるので、平均した値を用いる場合が多い。一例として、走行抵抗、こう配抵抗(上りこう配を+とする)および曲線抵抗が作用する場合のブレーキ距離は、=+=+2+욥±+욂で示される。ここで、列車の総質量(t)、列車からレールに作用する合計ブレーキ力(N)、列車の単位質量当たり走行抵抗욥(N/t)、こう配抵抗(N/t)、曲線抵抗욂(N/t)、車輪・主電動機等の回転慣性係数(1.05~1.15)。. 実平均減速度通例、平坦区間における空走時間を除いたブレーキ時間での列車の減速度を表す数値として実平均減速度が用いられる。実平均減速度β(m/s)はβ=2で表される。ブレーキ初速度39m/s(140km/h)から600m以内で停止させるための実平均減速度はおおむね1.45m/sである。c. ブレーキの制御ブレーキの制御とは、ブレーキ力の分担率や荷重に応じたブレーキ力の演算、滑走制御、異常や故障に対する防護など多くの機能を駆使し、目標となるブレーキ力を出力させるために所要の操作を加えることである。. 応荷重制御(load compensating brake control)乗客や貨物の荷重に応じてブレーキ力の増減を行い、荷重にかかわらず同じ速度からのブレーキ距離または減速度を同一とする制御である。. 電気ブレーキ優先制御(遅れ込め制御)電車において電動車の電気ブレーキを優先的に使用し、不足するブレーキ力を電動車および付随車の空気(摩擦)ブレーキを作用させて補うブレーキ力分担制御方式である。. 速度-粘着パターン制御車輪フラット発生防止のため、速度に対する粘着係数の変化を表す粘着曲線(adhesion curve,-μパターン)に沿って、粘着係数が低い高速度域ではブレーキ力を低下させ、速度の低下に伴いブレーキ力を徐々に増加させる制御方法である。. 編成全体のブレーキ力分担制御雨天時等に編成前部に発生しやすい車輪滑走を極力少なくするために、前部の車両のブレーキ力を若干低減し、後部車両のブレーキ力を高めて、列車の減速に必要なブレーキ力を得る方法である。
