# 3・3・4 粘着力(adhesion,adhesive force)と滑走再粘着

> **章**: 第3章 鉄道
> **節**: 3・44 )。電動機の特性により高速域ではブレーキ力が低下するが、

制御a. 粘着力と車輪滑走粘着ブレーキでは車輪-レール間の摩擦力を利用する。ブレーキ力と車輪-レール間に作用する粘着力を図3・48に示す。粘着係数μは滑走発生の目安値としてブレーキ力の設定に使われる。一般に、湿潤状態にある車輪-レール間の粘着係数は、速度が高くなるに従い低い値となる速度依存性を有する。最近の新幹線車両で測定された粘着係数の分布の例を、先頭車両および中間車両に分けて図3・49に示す。b. 滑走再粘着制御(wheel slide readhesive control)車輪フラット防止等のため、車輪滑走時に当該車輪のブレーキ力を一時的に低め、車輪を再粘着させるため滑走再粘着装置(アンチスキッドブレーキ)が導入されている。日本における滑走再粘着制御は、新幹線では1964年開業時の車両に、在来線では1987年651系特急電車に装備されてから本格的な使用が開始された。空気ブレーキ用滑走制御装置では、車軸端部や主電動機駆動装置に取付けた速度検出器からの速度パルスをディジタル処理して滑走を検知し、オンオフ制御による電磁給排気弁を2個組合わせた滑走防止弁により、ブレーキシリンダ内の高圧空気の排気、保持、復帰の動作を行う。電気ブレーキに起因する車輪滑走に対しては、主電動機の端子電圧や電流値を比較する方式により、滑走が継続する間、電気ブレーキ力を低下させる。滑走制御(slide control)の目標は、車輪-レール間の粘着力を最大限に有効利用することであり、必要以上にブレーキ力を低下させず、必要以上に長時間ブレーキを緩めないことにより、ブレーキ距離を延伸させないことが必要である。このような方法の例として、車輪の滑り率を5%以下に制御して、粘着係数が高い状態で滑走を維持させる、滑り率滑走制御方法が、新幹線車両、在来線車両に使用されている。また、ファジィ推論を適用した滑走制御方法が実用化されている。. 滑走検知(slide detection)と制御図3・50に滑走再粘着制御方法の例を示す。基準軸速度(1両4車輪のうちの最も高い速度値)と各軸速度間の速度差、車輪の滑り率(sliprate)、車輪減速度βを演算し、式(3・1)~(3・3)の一条件を満たした場合を滑走と判定し、両電磁弁を励磁して高圧空気の供給を止め、ブレーキシリンダ内空気を必要量排気させブレーキ力を低減させる。滑走収束傾向検知β<(m/s)を満足すると滑走が収束傾向にあると判断して、吐出し電磁弁を非励磁としてブレーキ力を保持する。
