# 3・4・2 空気ばね(air spring)

> **章**: 第3章 鉄道
> **節**: 3・4 乗り心地性能とサスペンション

近年,新製される旅客車の多くで車体-台車間のサスペンションに空気ばねが用いられている.空気ばねには客車,在来線特急電車などに使用されてきたベローズ型と,新幹線電車など現在広範囲に使用されているダイアフラム型(図3・55)がある.ダイアフラム型は,外周部でダイアフラムが外筒と内筒の間に挟まれていて,横方向の変位に対してダイアフラム内の空気圧による復元力が得やすい構造である.その剛性は空気圧のほか外筒の面板傾斜などにも依存するため,実験に基づいた設計が行われる.前後と左右の方向で外筒の面板傾斜を変えることで,それぞれの方向で異なる横剛性を与えることもできる.ダイアフラム型空気ばねの採用により,揺れまくら機構の省略など台車構造の単純化が可能になった.空気ばねの利点は,良い乗り心地のために必要な柔らかい剛性が1台車枠,2軸ばね,3軸箱,4板ばね,5軸はり,6緩衝ゴム,7リンク図3・53 代表的な軸箱支持方式図3・54 JIS E 4207に示された応力限界線図6―86交通機械図3・55 空気ばねのモデル化図3・56 空気ばね諸元の数値例と周波数特性の計算例利用でき,しかも乗客の乗り降りで荷重が変動しても内圧の調整により車体高さを一定に保つことができることにある.また,補助空気室を設け,空気ばね本体との間をオリフィス(絞り)で結ぶことによりばねが伸縮する上下方向の動きに減衰を与えることができる.ただし,ばねの伸縮を伴わない横方向の動きにはオリフィスによる減衰を利用できない.このような空気ばねの特性は上下方向に関して図3・55の右に示すようにモデル化されており,その剛性は本体部分に相当するのばねと補助空気室の効果を表すのばねとの直列結合で表される.=/욀N/m(3・4)ここで,:ポリトロープ指数(=1.4),:有効受圧面積m,:標準高さにおける空気ばねの絶対内圧N/m,욀:空気ばね本体の容積mであり,は空気ばね本体と補助空気室の容積比=욀/욁を表す.空気ばね本体と補助空気室の間のオリフィスによる減衰は,剛性のばねと並列なダンパで表され,その減衰係数は次式で与えられる.=γNs/m(3・5)ここで,γ:空気の比重量N/m,:空気の流量抵抗係数[実用的な範囲では=12.6×10/s/m]で近似される.mはオリフィスの直径である.との直列ばねと並列に作用するのばねは,ばねの伸縮で変化する有効受圧面積の効果を表す剛性である.=-율읏d/dN/m(3・6)ここで,율읏:大気圧,d/d:ばね伸び方向における有効受圧面積変化率である.空気ばね諸元の数値例と振動変位振幅の周波数特性の一例を図3・56に示す.
