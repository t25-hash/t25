# 3・4・4 制御サスペンション

> **章**: 第3章 鉄道
> **節**: 3・4 乗り心地性能とサスペンション

近年,コンピュータの演算速度が著しい向上を遂げたことにより,現代制御理論を用いた車両振動の制御技術が進展している.そこでは,高速化で増大する走行振動の抑制が求められるだけでなく,軌道に対する負荷の軽減も期待されている.ここで,外部からエネルギーを供給して振動を抑制する方式がアクティブサスペンション(active suspension)であり,外部からのエネルギー供給を必要としない方式がセミアクティブサスペンション(semi-activesuspension)と呼ばれる.a. セミアクティブ方式減衰力の切換えが制御可能なダンパを使用する方式であり,制御系のフェイル時にもパッシブなダンパとして機能する本質的にフェイルセイフな系である.すでに,高速の新幹線電車で左右振動の抑制に使用されている.そこで使われる可変ダンパの一例を図3・60に示す.車体-台車間で横方向に配置される可変ダンパは,3個のオリフィスの組合せによって6段階図3・59乗り心地レベルの等感覚曲線6―87の減衰係数が選択できる.車上のコントローラは車体左右加速度とダンパストロークを観測し,スカイフック則に対応した減衰力が発生できる減衰係数段を10msのサイクルで可変ダンパに指示する.その切換えは高速ソレノイド弁によって行われる.ただし,その時点でダンパの伸縮方向により実現可能な減衰力とスカイフック系の減衰力の向きが一致しないときは無負荷が選択され,このことがセミアクティブ方式における制振効果の制約となる.なお,セミアクティブサスペンションによる左右振動の制御効果として,走行速度300km/hで約30%の減少が報告されている.b. アクティブ方式1990年代に,左右振動の抑制を目的とする各種方式のアクティブサスペンションが,新幹線の試験車で検討された.その構成例を図3・61に示す.アクチュエータはそれと並列の左右動ダンパとともに車体-台車間で横方向に配置されている.エネルギーを供給するアクチュエータには,空気圧式および油圧式のものが試験されているが,国外ではボールねじを用いた電気機械式アクチュエータの使用も検討されている.コントローラは車体の左右加速度とロール加速度を観測し,則などのロバスト制御理論に基づく制御力を最小5msのサイクルでアクチュエータに指示する.試験車では,1~5Hz の周波数域での効果的な振動抑制により,300km/hを超える走行速度域で50%以上の振動低減が確認されている.振動低減効果の大きいアクティブ方式は,セミアクティブ方式に比較して運用コストが高いとされている.また,高速で曲線を走行する場合,超過遠心力による車体の偏りでストッパに当たりを生じ,制振効果を阻害することがあるため,振動制御に加えて車体を中心方向に引き戻すセンタリング制御の付加が必要である.
