# 3・4・5 車体傾斜(body tilting system)

> **章**: 第3章 鉄道
> **節**: 3・4 乗り心地性能とサスペンション

曲線の多い在来線で列車の到達時分を短縮するためには,曲線通過速度の向上が効果的である.しかし,曲線を高速で走行すると,乗客は遠心力により体が横へ引張られる感じを受け,この力が大きいとよろけたり,不安を感じるようになる.乗客がどの程度の左右力を受けたときに不快と感じるかについては,過去に走行試験でのアンケート調査等が行われ,定常的な左右加速度が0.8m/sより大きいと不快感を覚える人が多いとされてきた.ただし,これは一つの目安であり,定常加速度と振動加速度が同時に加わるときなども含め,現在もこの評価法自体が研究のテーマとなっている.曲線軌道には,遠心力を打ち消すためにカントと呼ばれる傾斜が図3・60 可変ダンパの構成例図3・61 空気式圧アクティブサスペンションの構成例6―88交通機械図3・62 車体傾斜車両の例(ころ式制御付き振り子装置)取付けられている.軌道面に平行な左右方向の定常加速度を超過遠心加速度(unbalanced centrifugal acceleration)といい,半径,カントの曲線を速度で車両が走行するとき,超過遠心加速度α욊は近似的に次式で表される.α욊=-(3・10)車両が曲線を高速で走行すると,超過遠心力によりサスペンションがたわみ,車体が外側へ角度φだけ傾くので,乗客は超過遠心加速度よりさらに大きな左右定常加速度αを感じることになる.このような一般車両のばね特性を係数<0で表すとき,左右定常加速度αは以下のように表現できる.α=1-α(3・11)ここで,:車体質量.在来線のカントは,最大105mm(角度:約5.6°)で,貨物列車を含む各種の通過列車に適した値としているので,特急電車がさらにスピードアップするのに十分ではない.そこで,曲線で車体を内側に傾けて乗客の受ける左右力を減じ,乗り心地向上を図った鉄道車両が開発されてきた.このような機構を設けた車両を総称して,車体傾斜車両という.車体傾斜車両とは,式(3・11)において,機構的にもしくは強制的に≒1/とし,左右定常加速度αを零に近づけることをねらった車両である.図3・62にころ式制御付き振り子装置を装架した車体傾斜車両の例を示す.車体傾斜車両が実用化された当初は,回転中心を車体重心より高い位置におき,超過遠心力によって車体が傾斜する自然振り子であったが,緩和曲線での振り遅れによる乗り心地悪化を改善するため,傾斜角を能動的に制御する制御式車体傾斜が現在では主流となっている.具体的な車体傾斜機構には,ころ式やベアリングガイド式の振り子のほか,リンク式や空気ばねの圧力制御により車体を傾斜させるものなどがある.また,これら傾斜機構の制御には,地点情報を基に緩和曲線の手前から車体傾斜を始めて一定の角度に制御するもの,車上のセンサで車体ロール角や左右加速度等を感知してフィードバック制御を行うものなどの各種方式がある.外国でも種々の車体傾斜車両が導入されてきており,最大傾斜角は±5°~7°程度である.回転中心が車体重心より上にある場合には,車体傾斜により重心が曲線の外側へ移動して内軌側の輪重が抜けやすくなるので,車体重心を下げることが特に重要となる.強制的に車体傾斜制御を行う車両では,回転中心を下げることによっても,このような内軌側輪重の減少を防止することができる.また,左右定常加速度だけでなく,緩和曲線における大きな車体ロール角変化が乗客の不快感を誘発する原因となるので,車体傾斜角を抑え気味に制御する方式が国内では広く用いられている.
