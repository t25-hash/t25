# 3・5・1 蛇行動安定性(走行安定性、running stability)

> **章**: 第3章 鉄道
> **節**: 3・5 走行性能と走り装置

鉄道車両には、高速走行時に車体や台車が激しく左右に揺れ出す蛇行動(hunting)という不安定な自励振動現象がある。蛇行動が発生すると、列車が水平面内を蛇のようにくねりながら進むので、この名がついたといわれている。蛇行動は、乗り心地を害するだけでなく、軌道破壊や脱線にも至る危険な現象であり、この安定性を確保することが鉄道車両の設計に際し必須の要件となっている。鉄道車両では、軸箱・台車枠間、台車枠・車体間に適切なばねおよび減衰要素を付加して蛇行動を防止している。a. 幾何学的蛇行動鉄道車両では一般に、踏面にこう配を有する車輪が車軸に圧入された一体輪軸を用い、自ら軌道中心に沿って走行する自己操舵機能を活用している。しかし、逆にこの輪軸(wheel set)の基本的な運動特性が蛇行動の発生要因となっている。まず、蛇行動の基本的な性質を理解するために、線形化した輪軸の運動方程式を用いて幾何学的な運動を調べる。. 輪軸の幾何学的蛇行動レール上を転がる輪軸には、左右動とヨーイングψの連成振動が発生する。輪軸が速度で転走するとき、慣性力を無視すると、車輪がレールに対して滑らずに転がる以下の関係式が得られる。等価踏面こう配は、左右車輪の回転半径差をとして、次式で定義される線形化定数である。 剛体台車の幾何学的蛇行動2本の輪軸が台車枠に固定された剛体台車を考えると、輪軸と同形式の運動方程式が得られる。剛体台車の幾何学的蛇行動波長は、軸間距離の半分をとして次式で与えられる。実際の鉄道車両では蛇行動を防止するために軸箱-台車枠間に弾性復元力を与えるので、その場合の安定性について以下に記す。. 弾性支持された輪軸と剛体台車の安定性弾性支持された輪軸の蛇行動安定性については、文献(22)に詳細に論じられているので、ここでは結果のみを記す。運動方程式の近似解から、輪軸は一定波長で振動し、走行速度が次式の限界速度以上になると運動が不安定となることがわかる。なお、式(3・16)では、ばね定数はおのおの1輪軸当たりの支持ばね定数を示し、前後、左右の線形クリープ係数は等しいとしている。剛体台車の場合は、式(3・16)において次の置換えを行えばよい。式(3・16)(3・17)から、蛇行動限界速度を向上するには、軌間や軸間距離を広くする、輪軸や台車の質量を小さくする、支持剛性を硬くすることなどが有効であることがわかる。. 鉄道車両の蛇行動安定性実際の鉄道車両では、車体-台車間の支持剛性や減衰要素も蛇行動防止に重要な要素であるが、運動自由度が大きくなり、安定性解析が難しくなる。この場合、実用的な方法として、微小振幅を仮定した線形運動方程式を立て、固図3・63 固有値解析の結果例有値解析により安定性を判別することが行われる。ダンパ取付ゴムの弾性を考慮し、ダンパとばねの直列系としてモデル化する場合には、質量の自由度が必要となる。固有値解析の結果例を図3・63に示す。実用的な検討の際には、減衰比ζ≦0.1を低安定領域の目安とする場合もある。車両の振動モードは、各固有値に対応した固有ベクトルを見ることで把握できる。固有ベクトルは、変位間の相対的な関係を表すので、例えば、先頭輪軸の左右変位を(1,0)として正規化し、複素平面上に並べれば、卓越した振動要素や各自由度の位相関係を知ることができる。
