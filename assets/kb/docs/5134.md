# 6・7・1 ガイドウエイバス（guideway bus ）

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・7 その他の新しい交通システム

わが国には，新交通システムを導入するほどではないが路線バス
では対応しきれないような中間的な交通需要をもつ地域も多い．
1983年の新交通システムの標準化に当たって，そのような地域に
は軌道構造物いわゆるインフラ部分を先行整備し，十分な需要が見
込まれるまで簡易な案内装置を取付けたバスを走らせるという，段
階整備の構想が示され，ガイドウエイバス開発の契機となった．
このシステムを特徴付けるものは，道路と軌道の両方を走行でき
る，いわゆるデュアルモード（dual mode）機能であり，一般の路
線バスに新交通車両と同様の案内装置を取付けることで，軌道上は
ハンドル操作なしに走行できる．これにより路線バスの簡便さと，
軌道系交通の安全性や確実性を併せ持つことが可能になった．
ガイドウエイバスの開発は1985年から官民共同で開始され，
1989年の福岡市アジア太平洋博覧会での運行を経て，2001年3月
に，名古屋ガイドウエイバスの志段味（しだみ）線が開業した．同
線は名古屋市の東北部にあって，志段味地区の開発に伴い新たに発
生する交通需要への対応と，流入する都心部での交通混雑の緩和の
ため，わが国初の実用路線として導入された ．
高架区間には軌道法が適用され，全長約6.8kmに9駅が設けら
れている．表定速度は30km/ h，最高速度は60km/ hで，25両の
車両が最小3分の間隔で運転されている．
起点の大曽根駅には，端末折返しのための転回場が設けられ，運
転士のハンドル操作と併用して車両の入替えが行えるようになって
いる．ただし，ここには一般道路への出入り口はない．いっぽう，
図6･31に示すように，小幡緑地の先，軌道と道路の境界部には，
車両確認や案内装置の伸縮などを行うためのモードインタチェンジ
（MIC:mode interchange）があり，登録車以外の進入を阻止でき
るよう遮断機や監視カメラなども設置されている．
車両は図6･32に示すような2.5m幅の伸縮式案内装置を備えて
いる．シャシについては2車種あるが，前車軸を共通化して同じ案
内装置を用いている．なお，開業時点で運用される車両はディーゼ
ルエンジン駆動の一般低床バスであるが，今後，低公害化やノンス
テップ化が図られることになる．
軌道は全線複線高架で，走行路はコンクリート製，案内レールは
H形鋼製のシンプルな構成となっている．
駅はすべて相対式であり，エレベータ，エスカレータと階段によ
ってホームと地表とを結んでいる．ホームには安全柵，接近案内表
示器，監視カメラなどが設けられているが，運賃収受については車
内で行うことを基本にしており，運賃設備は端末の大曽根駅にのみ
設けられている．
MICに併設する形で本社建家と簡易整備場が設けられている．
本社には，運転指令室があり，車両の運行やホームの状況，施設の
稼働状態，気象などを監視し総合的な運行管理を行うことができ
る．さらに，MICでの車両の進入出を監視し，出発の指示など簡
単な運転整理も行えるようになっている．なお，軌道法のもとで
60km/ h運転が認められているが，無軌条電車の運転規則が準用
され，軌道には特別な信号設備はない．
MICの一般道路への出入り口には，車両の運行に連動した交通
信号機が設けられ，一般道路ではPTPS（公共車両優先システム）
