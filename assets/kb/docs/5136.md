# 3・5・3 脱線に対する安全性

> **章**: 第3章 鉄道
> **節**: 3・5 走行性能と走り装置

脱線(derailment)は、その形態および車輪-レール間に働く力に着目して、乗り上がり脱線(flange climb derailment)、滑り上がり脱線、飛び上がり脱線の三つに分類されている。ただし、現実に発生しやすいのは、転走する車輪のフランジが自分でレールをよじ登る乗り上がり脱線である。アタック角正の車輪が乗り上がりを開始したときの車輪-レール間作用力を図3・66に示す。乗り上がり開始状態では、横圧を輪重で除した/と車輪-レール接触面に作用する力との間に以下の関係が成り立ち、ナダル(Nadal)の式と呼ばれている(図3・66)。したがって、式(3・27)でμ=μとしたナダルの式の値より/が小さければ、車輪は絶対に乗り上がり開始状態にない。このように/は脱線を判断する重要な指標であるので、脱線係数(derailment quotient)と呼ばれ、式(3・27)で計算される限度値を限界脱線係数という。実際の評価には、例えばフランジ角度60°の車輪の場合、式(3・27)にα=60°,μ=0.3を代入した計算値に安全率を見込んだ0.8等の数値が目安として用いられる。鉄道車両の脱線防止のためには、曲線通過時のアタック角および横圧の低減、カントや軌道面のねじれによる輪重抜け防止が重要である。新幹線電車等では、限界脱線係数の値を大きくして安全余裕を確保するため、フランジ角度70°の車輪を用いている。急曲線に続く緩和曲線部を低速で走行した場合に、曲線外軌側の輪重が減少し、軸重が小さい軸において高い脱線係数が観測される。このような状況での乗り上がり脱線に対する安全性評価は、軌道・車両条件の組合せを考慮した推定脱線係数比を用いて行う。推定脱線係数比=限界脱線係数/推定脱線係数(3・28)この値が1以上であれば、脱線に対し一定水準の余裕が確保されていると考えられるが、諸条件の不確定性を見込んだ1.2という値を安全性評価の基準とする。乗り上がり脱線に対する安全性を確保するには、左右静止輪重のアンバランスが生じないように、車両の静止輪重管理を行うことが重要である。
