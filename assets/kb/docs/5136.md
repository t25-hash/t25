# 6・1・1 真空環境

> **章**: 第6章 トライボロジー
> **節**: 6・1 宇宙機器が遭遇するトライボロジー問題

c .潤滑剤蒸気の広範囲な飛散 真空中においては，分子密度が小さくなるため，蒸発した分子の平均自由行程（平均自由走行距離：他の分子と衝突するまでの平均距離）が長くなる．
これは，蒸発した潤滑剤の分子が相当な距離にも容易に達しうることを示しており，したがって，汚染が広範囲に広がる可能性を示唆している．
d.しゅう動部の温度上昇 真空中では対流熱伝達による放熱が期待できないために，しゅう動面の温度が高温となりやすい．
油，グリースにおいては，蒸発を促進し，また，熱的劣化を促進する場合がある．
宇宙空間の真空レベルは，高度100～800kmの低軌道では10Pa，高度3万6000kmの静止軌道では10 Pa，また，惑星間では10Pa以下にまで達する．
ただし，人工衛星内部では衛星自体からの吸着気体の脱離や蒸発などがあり，宇宙空間より1～2けた高い圧力となっていると思われる．
このような高真空環境は，機構しゅう動部に対して以下に示すような影響を及ぼす．
a.潤滑剤の蒸発 雰囲気圧力が低下すると流体あるいは固体であっても蒸発し始め，さらに蒸気圧以下の圧力となると沸騰を生じる．
