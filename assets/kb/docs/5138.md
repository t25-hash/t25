# 3・5・5 操舵台車

> **章**: 第3章 鉄道
> **節**: 3・5 走行性能と走り装置

従来の台車では、蛇行動安定性と曲線通過性能との両立を図るものの、急曲線での転向性能は十分でなく、著大な横圧による軌道狂い進み、車輪フランジやレールの摩耗、きしり音等を生じていた。そこで操舵性能改善を目的として、従来と異なる構造の台車が開発され、実用化されている。この操舵台車には、主として輪軸の自己操舵機能を最大限に引出す自己操舵台車(self-steering bogie)と強制的に輪軸の舵をとる強制操舵台車(forced steering bogie)の二つのタイプがある。前者の代表例として、2輪軸をクロスアンカリンクで結合し軸箱を柔支持とした貨車用シェッフェル台車や、軸箱前後支持剛性を前軸柔支持・後軸剛支持とした前後非対称方式台車があり、後者の例としてボギー角連動強制操舵台車がある。図では、車輪踏面こう配を大きくしたほうが、むしろ蛇行動安定性が図3・67 実用化された操舵台車の例図3・68 タルゴ客車の台車向上する場合がある[ウインストック(Weinstock)の効果].
