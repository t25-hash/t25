# 3・6・1 車体構造

> **章**: 第3章 鉄道
> **節**: 3・6 車体

a. 構体の要件構体は車体(carbody)の基本構造物であり車体の最外殼を構成し、車体全体の重量を支え、強度、剛性を負担する筒状の複合構造物である。この複合構造物を理解するためには、まず構体の構造に求められる機能や特性の項目とともに、それらの相関性を明確にし、次に材料自体の機能や特性を明確にする必要がある。. 構体構造の機能要素と技術要素図3・69に構体の技術要素(technical factor of carbody)の相関図を示す。構体構造のコストを支配する要素として部品点数と接合線長がある。いずれも少数をもって良しとなる項目である。このほかに構体の機能要素として質量、強度、剛性の3要素がある。これらの機能要素は環境負荷・環境保全、各種荷重、形状保持性の3環境要素からそれぞれ制約を受ける。さらに、高性能な車両を実現するために必要な高速性、快適性、安全性の3車両要素からも影響を受ける。また、これらの全要素の良好なバランスを実現することにより達成される車両全体の効率性・経済性の合計11要素を考慮しなければならない。図3・69中の+,-記号は構体の各機能要素に対し環境要素と車両要素が増方向に作用するのか、減方向に作用するのかを示している。特に質量に関する要素には矛盾の関係がある。効率性・経済性の高い車両を開発するためには、この相反する関係を解決しなければならない。すなわち、質量は軽く、強度、剛性が大きくできて、そのうえに構体の部品点数が低減し、接合線長の短縮または接合線を単純化できる構体構造の開発と、それを実現する材料およびそれらの生産技術の開発が必要であり、図3・69ではそれが構体技術の発展の方向であることを示している。. steel)が主流となっている。それらの時代時代で、使用される構体材料の基本機能と加工の可能性およびイニシャルコストを主体に有用性が判断され使用されてきた。現在においてもその機能が重要であることに変わりはないが、LCA(life cycle assessment)や乗客に対する安全性や車内環境を尊重する設計思想が拡大し、基本機能以外の機能も同じレベルで重要視され、それらのバランスのうえで有用性が判定されるようになった。さらに、将来の構体用材料として非金属複合材が検討されるようになってきた。その場合、金属材料としては金属であること自体で良好な機能を具備しているため、いままで特別に検討要素としては注目していなかった熱、安全、メンテナンス、リサイクル等を十分検討する必要がある。図3・70に構体材料に望まれる機能を示す。b. 構体構造、製造法現在、実用されている構体構造(car-body structure)としては普通鋼製構体とアルミニウム合金製構体およびステンレス鋼製構体がある。それぞれの特徴が発揮できる車両の分野として、大きく分けて高速車はアルミニウム合金製構体、在来線はステンレス鋼製構体と普通鋼製構体、地下鉄はアルミニウム合金製構体とステンレス鋼製構体が使用されている。最近ではアルミニウム合金製構体の製作両数が多くなっている。. 構体の構成と組立法構体は図3・71に示すように、使用される金属材料にかかわらず、まず、最初に台枠(1面)、側構体(2面)、屋根構体(1面)、妻構体(2面もしくは妻構体1面と先頭構体1面)の合計6面のブロックを製作し、その後六面体に組立てる。これらはブロック分割組立方法(building block build upsystem)と呼称される。6―9 2交通機械。 アルミニウム合金製構体構造開発の変遷日本のアルミニウム合金製車両の開発は在来線車両で誕生し、私鉄・地下鉄車両で育ち、より荷重条件の厳しい新幹線車両まで進化した。また、アルミニウム合金はそれ自体リサイクル性の高い材料であるが、リサイクル上問題をもつAl-Zn-Mg 系合金A 7N 01の有用性が低下し、強度的には低いが、リサイクル性で優位なAl-Mg-Si系合金A 6N 01の有用性が向上するという事態が生じている。これに対し、従来の構体構造は外面にのみ表皮があり、内面に補剛材として縦通材と横断材を組合わせた骨組を設けた構造であり、シングルスキン構造(single-skin structure)と呼ばれるものである。構体の内面側は当然補剛材の幅面のみで荷重を負担するので、高強度の部材が必要になる。一方、強度的に低いA 6N 01が、構体構造を中空形材のように内外面に表皮をもち内部にせん断力に耐えるコア構造をもつ、いわゆるダブルスキン構造(double-skin structure)とすることにより、構体内面の荷重負担部が広くなり、局部応力の集中度が緩和されて低減する。この結果A 6N 01合金の強度でもA 7N 01と同等に構体に使用できるようになった。この例のように構体構造と材料の開発は相互に深く関連して変遷したといえる。図3・72にアルミニウム合金製構体技術の変遷を示した。. リベット接合による構体日本における最初のアルミニウム合金製構体としては1946年に戦後の鋼材不足の中で、当時、使い道のなくなった航空機用ジュラルミンを使用して在来線電車6両、客車5両などが製造された。ジュラルミンであるために接合技術はリベット構造であり、骨組は普通鋼材である。いわゆるスキンアルミニウム合金・普通鋼製構体ということができる。しかし、ジュラルミンは腐食に弱いため、長い期間は使用されなかったが、最近一部の高速試験車両および超伝導磁気浮上車両の構体構造としてジュラルミンリベット構造が採用された。これはシングルスキン構造の特徴である軽量性の極致としての航空機の機体構造を取り入れたものである。図3・71 溶接接合による構体本来シングルスキン構造の特徴は軽量性のほかに部品点数が多いという宿命をもっている。鉄道車両用の技術としては多少軽量性を犠牲にしてもコストを重視し、リベット接合よりも部品点数を縮減できる溶接による接合を目指した。溶接技術による普通鋼製構体は1938年ころには開発・実用化されていたが、アルミニウム合金製構体は形材の押出技術とアーク溶接技術の研究およびそれらを可能とする合金の開発、さらに当時確立していた車両用張殼構造の考え方を取り入れて開発された。この考えのもと、1962年に山陽電気鉄道2000系電車が、日本で最初の溶接接合によるアルミニウム合金製構体を用いて製造された。図3・73に同車両の構体構造を示す。. 押出形材の大型化・薄肉化の進展鋼材に比較してアルミニウム合金は母材強度・溶接部強度が低く、剛性もヤング率の関係で低くなりがちである。したがって、その後の開発努力は普通鋼製構体の数値を目標に、機能要素のうち強度の向上は新合金の開発により、剛性の向上は効率性・経済性とともに構体構造を工夫することにより行われ、その手段として大型押出形材とその製造技術の開発を促進した。また、質量の軽減は形材の薄肉化により実現されたが、そのこともまた新合金の開発と製造技術の開発を促した。. シングルスキン構造の事例300系新幹線電車の車体軽量化のため押出し性の良好なA 6N 01合金の薄肉形材に、T 字形の縦通材を一体押出成形することで部品点数を低減し、また、薄肉形材は24.5mにわたり一体物とし、接合線を車両長手方向のみに配置した構造となった。溶接はいずれも車両の内面側の一方向から自動溶接が可能な構造(ワンサイド工法)とすることにより、シングルスキン構造でありながら質量と部品点数およびコストを極小とした。図3・74に300系新幹線電車の構体構造を示す。. ステンレス鋼製車両構体構造開発の変遷日本における最初のステンレス鋼製車両は1958年に製作された東京急行電鉄5200系電車である。この車両は外板にSUS 304のステンレス鋼板を使用し、台枠や構体骨組を普通鋼製としたスキンステンレス(いわゆるセミステンレス)構造である。以後この構造の車両は多数生産されたが、内部の普通鋼が腐食して補修が必要になるなど、耐食性と軽量化の点で不十分であった。そこで1959年にオールステンレス鋼製車両の製造技術を有していたアメリカの車両メーカと日本の車両メーカの間で技術提携の契約を行い、日本におけるオールステンレス鋼製車両の開発と生産を開始した。当時、構造用ステンレス鋼として適当なものがなかったので、そのときはアメリカの車両メーカが研究開発していた独自の規格によるステンレス鋼(現在のSUS 201に近い材料)を使用した。その後ステンレス鋼製構体に適した材料を開発するために、日本の製鉄メーカと車両メーカが協力して高抗張力ステンレス鋼(high tensile strength stainless steel,SUS 301)の試作を行った。また、それらのステンレス鋼板を構体の部品に加工する成形技術6―9 3としてロール成形とストレッチ成形技術を研究開発するとともに、接合技術としてアーク溶接法だけでなく抵抗スポット溶接法も開発され、さらに、高抗張力鋼の特性を生かした構体構造が研究された。それらの結果として、台枠の一部(端台枠)を除き他の構造部材や外板などのすべての部品をステンレス鋼とした、いわゆるオールステンレス鋼製車両として1962年に東京急行電鉄7000系電車が完成した。この構体構造の車両はそれまでの普通鋼製の車両よりも約2tの軽量化を実現し、保守費低減の効果も大きく、約20年間にわたり約1000両が生産された。1973年の中東戦争に端を発した第一次オイルショックは全分野で省エネルギー方策の推進を求める原動力となった。鉄道車両においても省エネルギー=軽量化という形で要望が高まった。ステンレス鋼製車両に対しても、アルミニウム合金製車両並みの質量を目標とする軽量化と、普通鋼製車両並みの価格を目標とするコスト低減が求められた。これらの推進のために大型コンピュータと有限要素法による三次元強度解析技術の発展が大きく貢献した。この解析により信頼性の高い計算結果が得られるようになり、軽量化とコスト低減のための新しい構想の構体構造の開発と実用化を推進した。これらの開発の結果を反映して1985年に日本国有鉄道(現:JR)205系軽量ステンレス鋼製電車が完成した。さらに、これらのステンレス鋼製構体に使用される材料の合金開発は、粒界腐食割れ(intergranular
