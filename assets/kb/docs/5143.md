# 3・6・4 連結装置

> **章**: 第3章 鉄道
> **節**: 3・6 車体

一般に用いられている連結装置は、連結器(coupler)と緩衝器(draft gear)で構成される。a. 連結器。 自動連結器(automatic coupler、図3・85)自動的に錠のかかるナックルと呼ばれる部分でかみあう構造の連結器である。ナックルは先端形状に工夫が施され、連結時にナックルピンを中心に回転して互いに相手の連結器を受け入れることができるようになっている。連結状態で互いのナックル間にすきまがあるため、加減速時に列車への衝撃が大きい。機関車や一般の貨車などに用いられている。. 密着連結器(tight lock coupler、図3・86)連結面どうしが密着できるようにして、連結状態でのすきまをなくした連結器である。先端の連結面にはガイド兼用の中空突起があり、円弧状の錠が中に装備されている。連結接触面は機械加工が施されたうえ、錠が回り込みながら効くので密着性が良く、列車のブレーキ用空気管のコネクタや、制御用引通し電気配線のコネクタなどが同時に組込まれることもある。新幹線をはじめとする各種電車に用いられている。6―9 6交通機械。 密着式自動連結器(tight lock type automatic coupler、図3・87)密着性を向上させ、連結時のすきまを小さくするための機構をもつ自動連結器である。従来の自動連結器とも連結可能である。高速の貨物列車、特急寝台をはじめとする客車、ディーゼル動車などに用いられている。. 固定連結器(fixed coupler)連結を頻繁に開放する必要のない列車編成中間の連結部などに用いられる連結器である。棒連結器とも呼ばれる。b. 緩衝器連結器の根元に取付けられて、連結時・列車走行中の前後方向衝撃力を吸収・緩和する装置である。特性として適度な弾性とダンピング性が要求されるので、最近では両方の性質を併せ持つゴム緩衝器がよく用いられている。緩衝器としては、このほかに緩衝器自体の金属製胴体部分にも緩衝機能をもたせたものがある。これは、もし車両どうしが衝突した場合でも、その衝撃を緩和できるように考慮されたもので、緩衝器の胴体部分が衝突時に圧縮力を受けて一種の塑性変形を起こすことにより、ゴム緩衝器だけでは緩和しきれない衝突のエネルギーを吸収できるようになっている(図3・88)。海外製の連結装置に採用されている例が多い。
