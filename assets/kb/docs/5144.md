# 3・6・5 旅客車の室内設備

> **章**: 第3章 鉄道
> **節**: 3・6 車体

装置内の圧縮機は，三相あるいは単相交流を電源とする電気式

pump）式がおもなものである．ただし，新幹線でも寒冷地向けの
車両についてはヒートポンプ式では暖房能力が不十分のため，ヒー
タを内蔵した冷暖房装置としている．ヒートポンプ式では冷房時と
同様に温風を天井部から吹出し，空気の強制対流により客室を暖め
るが，それ以外の方式は，通常腰掛下にヒータ等を設置し自然対流
により客室を暖めている．冷暖房の制御は，客室に設置された温
度・湿度センサからの情報をもとにON，OFF制御を行ったり，
圧縮機の回転数を変化させるインバータ制御としている．
b.腰掛 鉄道車両用の腰掛（シート，座席）は大別して固定
式と転向式に分けられる．固定式には，地下鉄車両を代表とする通
勤型の車両にあるような窓際に車両中央方向に向いて設置されるロ
ングシートタイプと，近郊型車両にあるような4人分のボックス席
を構成しているクロスシートタイプがある．転向式には腰掛の背ず
りのみを動かす転換式と，座布団から上部を回転させて方向を変え
る回転式がある．回転式のほとんどはリクライニング機構を備えて
いる．近年では，乗り心地等のサービス向上のため，前述の固定式
のクロスシートが回転式あるいは転換式に代えられる傾向にある．
腰掛の一人分の幅は450mm程度であり，クロスシートの場合，
腰掛の車両長手方向の取付ピッチは普通車では1000mm前後，グ
リーン車では1160mmが一般的である．
腰掛の骨組み構造は通常鋼製であるが，新幹線のように軽量化を
要求される車両においては部分的にあるいは全体にアルミニウム合
金製が採用されるようになってきている．また，固定腰掛において
は軽量化や部品の簡素化によるコストダウンのために，FRP（繊
維強化プラスチック，fiber reinforced plastic）の一体成形として
いるものもある．以前は座布団および背ずり布団のクッションに金
属ばねを用いていたが，最近では詰め物をクッション性のあるポリ
ウレタンフォーム等の成形品として座り心地の向上とコストダウン
を図っている．通常使用される表地には平織りとモケットがある
が，耐久性の点でモケットが大部分を占めている．新幹線あるいは
在来線の特急車の腰掛は，腰掛背ずり背面に大型のテーブルを設け
たり，車内ラジオ放送用のイヤホンジャックを設置してサービスの
向上を図っているものもある．また，高齢者や身体障害者等の移動
制約者のために優先席を設置することが一般的になり，特急車にお
いては腰掛に車椅子の緊締装置を装備している．
c .便洗面所 車両の便洗面所は，腐食防止および清掃の容易
さの観点から全体をFRP製のユニットとし，車両外部である程度
組立てられた後，車両に搬入されることが多い．また，腰から下の
水にぬれやすい部分のみFRP製とし，上部を化粧板等で構成し意
匠性を向上しているもの，あるいはタイル張りとして清潔感を高め
ているものもある．
