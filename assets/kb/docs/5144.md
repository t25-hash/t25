# 6・2・1 協調型ディジタル機器ネットワークの分類

> **章**: 第6章 ネットワーク接続機器
> **節**: 6・2 協調型ディジタル機器

協調型ディジタル機器という呼称は，まだ定義が定着していない
用語であり，ネットワークでつながるという部分ですら，さまざま
な形式のものが，共存，あるいは競合関係にある．理想をいえば，
ある1規格に統一されたネットワーク環境に，すべてのディジタル
機器が接続，利用可能であるべきである．しかし，そうした標準化
は，規格を決めている間に技術が進歩してしまうこと，そして機器
や使用場所によって求められるものが違うという2点において困難
を伴う．
規格の統一には案の作成，検討，規格書の作成にそれぞれ最低1
年は要するが，すべてを網羅しようとすればするほど，関係者が増
えるためさらに時間がかかり，決まるころには陳腐化してしまうの
が実情である．
また，ネットワークに求められるものは，情報の転送レートから
接続手段まで多岐にわたる．接続が有線か無線かで求められる方向
性が異なり，個人利用とオフィス利用では接続する機器数や配線の
長さが異なる．またテレビや据置き型のパソコンでは問題ない大き
さの端子が，ポータブル機では採用できない場合があったり，利便
性を追求して電源供給までサポートしようとすると，電池駆動の機
器には不利になる場合がある．また有線の場合，新たに設置工事の
必要があるか否かは，家庭での普及にとっては重要な要素になる．
以上のような背景で，全機器が一堂に会する一元化されたネットワ
ークというのは理想的だが実現が難しい．そこで，何かに特化して
利害の一致する機器だけの共通規格が推進されることとなるが，そ
の特化の方向性は，大まかに次のようになる．
(１)個人規模のモバイル（mobile）ネットワーク．
(２)パソコンを中心とした，オフィス，家屋内ネットワーク．
(３)テレビなどAV家電を中心にした家屋内ネットワーク．
ここで(１)は個人が持ち歩く機器の協調を図るためのネットワー
クで，屋外へ移動しても使えることが重要であり，パソコンがその
場になくても機能することも重要である．(２)はパソコンに簡単に
周辺機器や別のパソコンをつなぐことが第一で，持ち歩きよりも，
高速性やつなげる機器数の多さが重要である．一方(３)はディジタ
ルコンテンツを扱うAV機器で，データの配信，受信，編集，ラ
イブラリ管理，視聴および機器の遠隔操作を連携して行うためのネ
ットワークである．これらの棲み分けを利用場所軸とAV/情報家
電軸で表したのが図6･13である．以下では，具体的にどのような
機器，どのようなネットワーク技術が，現状製品として登場し利用
されているかについて言及し，それぞれがどう関わり合って上記の
ような棲み分けを形成しているかを解説する．
