# 3・7・1 自動列車停止装置(ATS : automatic train stop)

> **章**: 第3章 鉄道
> **節**: 3・7 運転保安システム

列車が停止信号に接近しても正常に停車しない場合などに、運転士に警報を鳴らして注意を喚起したり、自動的にブレーキを作動させて、信号の手前で停止させる装置である。通常の列車運転は、運転士に任せ、緊急時に運転士のブレーキ動作をバックアップするものである。ATS の種類は、鉄道会社の歴史により各社各線区各様の方式を使用しているが、大別すると、停止信号に近づいたときに警報を発して、所定の操作をしないと一定時間後にブレーキをかける警報式と、列車速度をチェックして制限速度をオーバした場合にブレーキをかける速度照査式がある。JR で多く採用されているATS-S 型は警報式であるが、この方式では、所定の操作(確認扱い)をすれば非常ブレーキはかからないので、操作後の運転士の扱いミスに対しては防護できない。一方、速度照査式には、特定の地点数箇所で速度をチェックする方式、停止までの数段階の速度を連続的にチェックする方式、停止までの減速曲線を基に設定した速度照査パターンに対して連続的にチェックする方式などがあり、警報式に比べ安全度は高くなる。JR では高密度線区の列車に、車両側の制御装置内に速度照査パターンをもち、その速度を超えた場合にブレーキをかける速度検知式のATS-P型を導入している。地上から車上装置に対してATS 信号を情報伝送する方式には、あらかじめ定めた複数の地点から車上にATS 信号を伝送する点制御方式と、レールなどを用いて連続的に車上に伝送する連続制御式図3・91山陽新幹線駅中間停車時のATC 動作例がある。点制御において、地上側の信号情報を伝送する装置を地上子というが、地上子にはATS-S 型のように周波数変化をさせるコイルを置いたものやATS-P型のようにディジタル情報を送るトランスポンダ(多情報伝送装置)を置いたものなどがある。連続制御式では、レールにATS 信号を流す方式が一般的で、民間鉄道会社で広く採用されている。点制御では、列車通過後に信号現示変化があった場合に、次の地上子の位置に進むまで現示変化に追随できない。
