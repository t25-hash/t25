# 3・7・3 自動運転装置

> **章**: 第3章 鉄道
> **節**: 3・7 運転保安システム

自動運転は,自動的に運転を行うものの総称であり,ATO(automatic train operation)装置のようにすべてを自動で行う装置だけでなく,定速運転装置や定位置停止装置等運転の一部機能を自動的に行う乗務員支援的な装置も含まれる.自動運転装置は,運転保安装置としてのATCに前述のような機能を追加した装置である.一部の自動運転装置としては,定速運転装置や定位置停止装置があり,新交通システムや地下鉄等で採用されている.定速運転装置は,定められた目標速度に追随しながら運転を行うもので,目標速度と実際の速度との差を検知し,速度差に合わせた力行やブレーキの制御を行うものである.ATO装置は,ATCの機能に発車制御,定時運転制御,定位置停止制御(TASC: train automatic stop control)の機能を追加したすべての運転士業務を自動で行う装置である.新交通システムや比較的最近開業した地下鉄等で採用されており,無人運転を行っているものもある.発車制御は駅停車時分制御と出発制御機能をもつ.駅停車時分制御は,運行管理システムの情報より停車時分を適正化するものであり,出発制御は信号条件とドアの閉じたことを検知して出発を行うものである.定時運転制御は,列車の加速・惰行・減速を行い,所定の時間に次の駅に到着するように制御するもので,通常運転モード,回復運転モード,遅延運転モード等をもつ.制限速度を超えるとATCの制御によりブレーキがかかるため,できるだけ制限速度を超えずに円滑に走行するよう制御する.また,定位置停止制御は,地上からの地点情報を基に停止制御パターンに沿って定位置に停止させるものであるが,このパターンには地上パターン式と車上パターン式がある.特に無人運転を行っている新交通システムでは,ホームドアを設置しており,停止位置が狂うとドアをあけられないため,停止位置を正確にすることが必要となっている.最近の定時運転制御や定位置停止制御では,正確性だけでなく,車両の乗り心地も考えた制御が必要であり,ファジィ制御やエキスパートシステム等AI技術も応用されている.
