# 3・8・1 検査体系と検査手法

> **章**: 第3章 鉄道
> **節**: 3・8 車両の保守

鉄道車両の保守（maintenance）は，旅客および貨物を安全，正
確，迅速かつ快適に輸送できる状態に車両を保持するために行わ
れ，定期検査による予防保全を基本としている（表3･9）．
a.定期検査 車両の定期検査は，車両の使用と密接に関連
し，それに対応した検査体系が整えられている．車両の種類に応
じ，それぞれの検査について，検査の名称，周期，内容が規定され
ており，大別すると次の三つである．
.状態と機能の検査 比較的短い期間ごとに，車両の種類
や運行状況に応じ，動力発生装置，走行装置，ブレーキ装置，車体
等の状態，作用および機能等について在姿の状態で行う検査であ
る．
.重要部検査 比較的長期間使用した後，車両の種類およ
び使用状況に応じ，機能確認のため，動力発生装置，走行装置，ブ
レーキ装置その他重要な装置の主要部分について行う検査である．
.全般検査 長期間使用した後，車両の種類および使用状
況に応じ，車両の主要部分全般について行う検査であり，最も上位
の検査である．

3・8・3輪重管理
2000年3月に発生した帝都高速度交通営団（当時）日比谷線中
目黒駅構内列車脱線衝突事故は，曲線に続く緩和曲線の始端付近
で，輪重（wheel load）減少や横圧増加を引き起こす複数の因子の
影響が複合的に積み重なったことにより，事故車両の第一軸車輪の
脱線係数が増大し，車両がレールに乗り上がって脱線したことによ
るもの（乗り上がり脱線）であった．因子が複数ある中，事故当
時，第一軸に大きな静止輪重のアンバランスを有していたことがお
もな因子であった．
この事故を契機として静止輪重の管理が導入され，車両新製時お
よび全般検査，重要部検査等の定期検査時には静止輪重の測定を行
い，静止輪重比が一定の管理値を超えた場合には調整を行ってい
る．
