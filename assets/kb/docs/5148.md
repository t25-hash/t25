# 3・8・1 検査体系と検査手法

> **章**: 第3章 鉄道
> **節**: 3・8 車両の保守

鉄道車両の保守(maintenance)は、旅客および貨物を安全、正確、迅速かつ快適に輸送できる状態に車両を保持するために行われ、定期検査による予防保全を基本としている(表3・9)。a. 定期検査車両の定期検査は、車両の使用と密接に関連し、それに対応した検査体系が整えられている。車両の種類に応じ、それぞれの検査について、検査の名称、周期、内容が規定されており、大別すると次の三つである。. 状態と機能の検査比較的短い期間ごとに、車両の種類や運行状況に応じ、動力発生装置、走行装置、ブレーキ装置、車体等の状態、作用および機能等について在姿の状態で行う検査である。. 重要部検査比較的長期間使用した後、車両の種類および使用状況に応じ、機能確認のため、動力発生装置、走行装置、ブレーキ装置その他重要な装置の主要部分について行う検査である。. 全般検査長期間使用した後、車両の種類および使用状況に応じ、車両の主要部分全般について行う検査であり、最も上位の検査である。b. 検査の組合せと検査周期定期検査は、数種類の検査を組合わせて実施しているが、その組合せ方は、期間の短い軽度な検査表3・9車両保守体系状態と機能の検査定期検査重要部検査予防保全全般検査車両保守臨時検査随時検査運転中の検査事後保全故障の処理6―9 9を繰返し行って、一定期間(または一定走行距離)後の検査時期に、次の上位の検査を行い、さらにそれを繰返した後、最終的に主要部分全般について行う全般検査を行う方式で、図3・92に示すとおりである。また、これらの各検査を行うべき周期は、車両の種類および使用状況に応じ、構造、重要度、経済性を加味して定めている。c. 臨時検査車両の臨時検査は、新製した車両および改造または修繕した車両を初めて使用する場合、脱線その他の運転事故が発生した車両で故障の疑いがあるもの、使用休止した車両を初めて使用する場合等に行う検査である。
