# 3・8・2 モニタリング装置

> **章**: 第3章 鉄道
> **節**: 3・8 車両の保守

車両間に伝送線を引き通し,各機器の動作状態を遠隔監視するためのモニタリング装置(monitoring equipment)は,1970年代から導入されてきた.当初は列車乗務員に対する支援装置としての位置付けであったが,近年の電子技術の発展に伴って多くの機能が盛り込めるようになり,列車全体の情報を管理できるようになってきた.こうした背景の中,モニタリング装置を車両の保守支援装置として利用する例が増えてきており,下記の分類に分けられる.(1) 定期検査時に,モニタリング装置を通じて各機器の機能を検査する.自動検査装置としての利用例もある.(2) 故障発生時に,モニタリング装置に蓄えられた故障発生時の動作記録を用いて原因を調査する.(3) 試運転時に,モニタリング装置によって自動的にデータを収集する.このほかにも,使用中の機器の動作状態から機能を随時監視し,故障の兆候を予知して検査・修繕する(状態監視保全とも呼ばれる)という利用方法も一部で進められている.モニタリング装置に蓄えられた情報を地上側で使用するための出力手段としては,以前は車内にある中央局に読出し用の端末を接続して読み出す方法が多かったが,その後ICカードに情報を転送しこれを地上側の装置で読み出す方法が一般的になった.さらに最近では,車上と地上との間に無線の伝送装置を設置し,機器の動作情報を地上の保守システムに自動的に転送することも実用化されている(図3・93).
