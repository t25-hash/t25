# 3・8・3 輪重管理

> **章**: 第3章 鉄道
> **節**: 3・8 車両の保守

2000年3月に発生した帝都高速度交通営団(当時)日比谷線中目黒駅構内列車脱線衝突事故は、曲線に続く緩和曲線の始端付近で、輪重(wheel load)減少や横圧増加を引き起こす複数の因子の影響が複合的に積み重なったことにより、事故車両の第一軸車輪の脱線係数が増大し、車両がレールに乗り上がって脱線したことによるもの(乗り上がり脱線)であった。因子が複数ある中、事故当時、第一軸に大きな静止輪重のアンバランスを有していたことがおもな因子であった。この事故を契機として静止輪重の管理が導入され、車両新製時および全般検査、重要部検査等の定期検査時には静止輪重の測定を行い、静止輪重比が一定の管理値を超えた場合には調整を行っている。・静止輪重静止状態の車両の1個の車輪を通じてレールに加図3・92検査の組合せと検査周期6―100交通機械図3・93モニタリング装置の構成例わる荷重。・静止輪重比片方の車輪の輪重をその軸の平均軸重で除した値。1に近いほど左右の車輪にかかる輪重のバランスが良い。
