# 6・3・4 今後の動向

> **章**: 第6章 ネットワーク接続機器
> **節**: 6・3 セットトップボックス

今後，家庭内での映像エンタテイメントとして，ディジタル放送
と通信との融合が加速することが予想され，セットトップボックス
ないしは家庭内のテレビ受像機に要求される機能は急速に進化して
いくと考えられる．従来の放送を受信するために必要なセットトッ
プボックスとしてだけではなく，通信ネットワークから配信される
映像エンタテイメントやさまざまなサービスを可能にすることが求
められるであろう．
いままでのディジタル放送はエムペグ2（MPEG-2:moving
picture experts group phase 2）という圧縮技術に支えられて発展
してきたが，エムペグ4（MPEG-4）に代表される，より高効率な
圧縮技術が，放送およびブロードバンドで使用され，発展していく
と予測される．MPEG-4に代表される新しい圧縮技術を使用する
ことにより，例えば，ハイデフィニション映像を7Mbpsと従来の
4分の1程度のスピードで伝送できる．高効率な圧縮技術は，送信
側も含めて，限られた帯域をより有効に活用することができるの
で，多チャネル化，ハイデフィニション映像送信など，大きな変革
をもたらす可能性を秘めているといえる．
特に，マイクロソフト社は，独自の圧縮技術および著作権保護を
基に，放送・通信の新しい仕様としてさまざまな提案をしてきてお
り，標準化あるいはデファクトスタンダード化に向けて，激しい競
争が予想される．
ホームネットワーキングと著作権保護技術の進化により，いまま
で単品で機能していた家電製品は相互につながるようになり，コン
テンツを他の機器にコピーしたり，どこでもコンテンツを楽しめる
ようになるだろう．自宅から離れていながら自宅の機器にアクセス
し，機器をコントロールしたり，コンテンツを転送してユビキタス
で楽しめるようになるだろう．この際に重要なのは，機器の接続互
換性を保つためのホームネットワークのプロトコルの標準化，およ
びネットワーク上でやりとりされるコンテンツの保護技術である．
プロトコルがすべての機器間で統一できなくても，複数の異なるプ
ロトコルで動作する商品をつなげて，それぞれのプロトコルを制御
する上位概念としての新たなプロトコルも提案され始めている．
したがって，セットトップボックスは，従来の放送や通信のサー
ビスを受信するための箱から，エンドユーザがユビキタスに映像コ
ンテンツを楽しむことを可能にする製品になっていくと思われる．
