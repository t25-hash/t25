# 3・9・1 鉄道の新技術

> **章**: 第3章 鉄道
> **節**: 3・9 新技術

日本の鉄道の歴史は,1872(明治5)年に新橋~横浜間の開業に始まり,以来130年以上がたった.鉄道の技術は,車両,軌道,電力,信号・通信等,あらゆる分野で新しい技術開発がなされ進歩してきた.例えば,車両の動力は,蒸気機関からディーゼル機関,そして電気機関(電気動力)へと近代化が進み,軌道は継目のないロングレールとコンクリートまくら木により乗り心地の改善のみならず保守の省力化にも貢献した.保安システムでは,タブレットによる通票閉そく式はすでになくなり,自動列車制御装置(ATC:automatic train control)に代わっている.信号機も腕木式信号機から3色現示信号機,そして現在では車内信号機とともに自動運転も実現しつつある.種々のシステムに対して新技術が導入されてきたが,鉄車輪が鉄レールの上を転がるという基本的な鉄道車両の構成は変わっていない.架線から集電して軌条(レール)のない道路を走るトロリバスは自動車に属するが,ゴム車輪(タイヤ)を用いてレールのない案内軌道を走るモノレール等の新交通システムや,鉄の車輪もレールも使わずガイドウエイを浮上走行する超伝導(超電導)磁気浮上式鉄道は,従来の鉄道車両の構成や鉄道システムを根本的に変えた新技術であるが,第6章にあるように,もはや鉄道とは別の分野として取扱われている.鉄道の新技術として,車輪-レール間の摩擦力(いわゆる粘着)によらない駆動および制動(ブレーキ)システム,異なる軌間(ゲージ)の路線を直通運転できる車両システム,線路(レール)と道路の両方を走ることができる車両等が挙げられる.
