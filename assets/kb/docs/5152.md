# 3・9・2 リニアモータ駆動小型地下鉄

> **章**: 第3章 鉄道
> **節**: 3・9 新技術

車輪-レール間の摩擦力によらない、いわゆる非粘着駆動・制動システムを用いた鉄道車両として、リニアモータ(linear motor)駆動小型地下鉄電車が実用化されている。リニアモータにより駆動する車両は、主電動機により車輪を回転させて車両を走行させるのではなく、台車を直接動かして(けん引または推進して)車両を走らせる駆動システムであるので、車輪は荷重を支持するだけ、線路は車両の進行方向を案内するだけである。この電車は停止直前まで電気ブレーキが作動するので、低速でも車輪の回転を止めて車輪-レール間の摩擦力によってブレーキをかける必要はないが、停車中には電気ブレーキが効かないので車輪を機械的に押付けて車輪の回転を止めている。リニアモータ駆動により車輪径を小さくした薄い形状の台車を構成することができるので、車両の高さを一般の電車より低くすることができる。車両の小断面化は地下鉄トンネル建設費の低減に有効である。現在、リニアモータ駆動小型地下鉄は、大阪市の長堀鶴見緑地線、東京都の大江戸線、神戸市の海岸線、福岡市の七隈線などで営業運転している(横浜市は建設中)。
