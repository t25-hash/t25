# 3・9・3 軌間可変電車

> **章**: 第3章 鉄道
> **節**: 3・9 新技術

軌間(ゲージ,gauge)の異なる路線を直通運転する車両として,広軌のスペインと標準軌のフランスからスイス,ドイツ等へとヨーロッパ各国間を走る「タルゴ列車」が知られている.日本でも新幹線の標準軌と在来線の狭軌路線を直通運転できるシステムを開発しており,軌間可変電車(GCT : gauge change train)と呼ばれる3両編成の試験電車が1998(平成10)年に製造された.a. 開発の経緯軌間可変試験電車は,1994(平成6)年に運輸省(現:国土交通省)の運輸技術審議会の答申(21世紀に向けての鉄道技術開発のあり方について:SUCCESS 21計画)で「標準軌の路線と狭軌の路線の乗り換えをなくし,鉄道ネットワークの利便性の向上と所要時間の短縮を図るためには,相互に乗り入れ可能な軌間可変台車の開発が重要である」と提言され,日本鉄道建設公団(現:(独)鉄道建設・運輸施設整備支援機構)からの委託を受けた(財)鉄道総合技術研究所が「新幹線/在来線直通運転システムの開発」として,この軌間可変システムを実証試験するために製作した車両であり,軌間可変する電車としては世界初である.試験電車は,1999(平成11)年から約2年間にわたり,山陰本線で確認走行の後,アメリカ,コロラド州プエブロ実験線で200km/h以上の速度で約60万km走行の高速耐久試験を行った.2001(平成13)年に交直流電車化・新幹線対応に車両を改造し,九州の日豊本線および四国の予讃線で在来線での速度130km/h走行と本則+25km/hの曲線通過性能試験,新下関試験基地で軌間変換試験,交流25kV/直流1500V の交直セクション直通試験を行った.2004(平成16)年には,山陽新幹線新山口~新下関間で速度210km/hまでの走行性能試験を行った.b. 軌間可変電車の特徴と概要スペインのタルゴ車両が動力をもたない客車であるのに対して,軌間可変電車は電動機付きの軌間可変台車を各車両に装備した全電動電車である.タルゴ車両が広軌(1668mm)と標準軌(1435m)間233mmの変換に対し,軌間可変電車は標準軌(1435mm)と狭軌(1067mm)間368mmの変換である.軌間可変電車の車体寸法は,新幹線と在来線を直通運転するため,秋田新幹線「こまち」と同じ在来線の車両限界を適用した.設計最高速度は,新幹線で300km/h,在来線で130km/hである.電気方式は,新幹線の交流25kV(50/60Hz)と在来線の交流20kV(50/60Hz),直流1500V に対応した交直流電車であり,交流25kV と直流1500V 間の交直切換システムを初めて採用した.パンタグラフは新幹線用高速パンタグラフと在来線用交直流用パンタグラフを上げ下げして切換えている.主回路機器は,1台のPWM コンバータと個別制御の同期電動機1車両分8台のVVVFインバータを組込んだ電力変換装置(主変換装置),真空遮断器(VCB),交直切換器,主変圧器,負荷接触器,断流器,リアクトル,高速度遮断器等,一般の交直流電車に相当する機器で構成されている.軌間可変台車には2種類あり,永久磁石同期電動機を用いた主電動機を車輪に直結させた直接駆動(DDM)・独立車輪方式にリンク式の操舵装置を装備した台車(軸間距離2200mm)と,台車枠装架の誘導電動機を用いた一般的な平行カルダン駆動方式台車(軸間距離2500mm)とがある.いずれの台車も狭軌での狭いスペース図3・94アメリカ,コロラド州プエブロ実験線で高速耐久試験中の軌間可変電車[(財)鉄道総合技術研究所提供]6―101に必要な駆動機構を構成し,新幹線並みの高速性能と在来線の曲線通過性能を有することが要求される.軌間の変換は,特殊な軌道(軌間変換装置)を約10km/hの速度で自走し通過しながら,軸箱体に設けられた車輪を固定するロック装置のロック(鎖錠)を解除し,ガイドレールによって車輪の間隔を変換して再びロックする方法をとっている.現在,軌間変換装置は,山陽新幹線と山陽本線を結ぶ新下関駅構内,九州新幹線と鹿児島本線とをつなぐ新八代駅構内に設置されている.図3・94に電車の外観,図3・95に軌間変換装置を示す.
