# 3・9・4 デュアルモードシステム

> **章**: 第3章 鉄道
> **節**: 3・9 新技術

ここでいうデュアルモード(dual mode)とは,鉄道と道路のことで,デュアルモードシステムとは,線路(レール)上と道路の両方を走ることができる車両システムを意味する.鉄道車両ではないが,軌道や架線の保守作業に用いる軌陸車という鉄道用車輪を装備した自動車があり,鉄車輪を油圧で上げ下げして線路上と道路を自走することができるものがある.a. ロードレーラアメリカの鉄道にはロードレーラ(roadrailer)という貨車があり,20年以上前から営業運転されている.道路を走ってきたトレーラやコンテナ車は,取付けられた鉄車輪を油圧装置で下げることにより道路モードから鉄道モードに切換えることができる.日本でも鉄道と道路の両方を走ることができる車両システムを検討しており,今後の貨物輸送の新しい技術として注目されている.b. デュアルモードビークル(DMV)北海道旅客鉄道は,2004(平成16)年にデュアルモードビークルという道路も鉄道も走ることができる試験車両を試作した(図3・96).ガソリンエンジンのマイクロバスを改造したDMV 試験車は28人乗りで,道路用のゴムタイヤと油圧で上下する鉄道用車輪があり,切換えは約15秒間で行える.踏切のような場所でも切換えは可能である.前のタイヤを浮かし後ろのゴムタイヤがレールと接触して駆動力を出し,速度70km/hで走ることができる.レールの上をゴムタイヤで走り,鉄車輪が進路を案内するというイメージである.もちろん,道路走行用のハンドルは付いている.図3・97にモード切換えの仕組みを示す.現在,連結運転の試験を行っており,将来は40~50人乗りを目指し,実用化に向けた開発を行っている.図3・96デュアルモードビークルの試験車両[北海道旅客鉄道(株)提供]図3・95
