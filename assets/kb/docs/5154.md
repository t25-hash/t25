# 6・1・4 システム性能

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・1 ハイブリッドシステム

車両停止中は基本的にエンジンを停止させてアイドリング運転によるむだな燃料消費を防止する．
RESSを充電させるためやエアコンディショナ（空気調和機）のコンプレッサを作動させるなどの必要がある場合に限り，エンジンを作動させる．
発進加速時は，RESSに蓄えられたエネルギーから優先的に使用し，モータの駆動力を利用する．
回生エネルギー 少ない 多い車両質量 小さい 大きいジンストップと減速エネルギー回生の効果が減少すること，および率の高い領域となること，による．
したがって，HEVは平均速度ICEV寄り EV寄りマイルド～パラレル～シリーズ・加速性能を維持しつつ，定常時はエンジンを高負荷高効率領域で運的小型のエンジンを搭載している場合が多い．
そのため，加速性能性上，最高速度性能が重視される用途には適さない車両といえる．
また，RESSのSOCによって，動力性能に数秒程度両と同等の性能を確保するよう設計される．
RESSの充放電を繰返しながら走行する．
そのため，試験終了時のンジンで充電した量のほうが多くなるので燃費は悪くなり，逆に，電した量よりも放電で利用した量のほうが多くなるので燃費はよく時のSOCが一致している（蓄電量差零とする）必要がある．
