# 6・3・2 直流モータ（DC motor ）

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

a.概要 直流モータは励磁方式により永久磁石励磁形と巻線
励磁形に分類される．巻線励磁形には界磁巻線（field coil ）と電機
子巻線（armature）が直列に接続される直巻モータ（series
motor ）と並列に接続される分巻モータ（shunt motor ）および両
方の巻線方式を有する複巻モータ（compound motor ）がある．直
流モータの特徴は，原理的に大きな始動トルクと負荷の変動に適応
したトルクが得られること，車載電池の直流電源をそのまま利用で
きること，速度制御，回転方向の切換が容易な点である．また，制
御装置の構成が安価であることから，古くより電気自動車の駆動モ
ータとして用いられてきた．
直巻および分巻直流モータは古くは電気自動車で多く用いられて
きたが，最近では一人乗りの電気自動車など小容量用途に永久磁石
式直流モータの採用が見られる（表6･8) ．一方，電気自動車，

b.構造と特徴 直流モータは界磁方式により図6･27 のよ
うに巻線界磁形と永久磁石界磁形に分類される．
直流モータは，磁束を作る界磁鉄心（field core），界磁巻線，起
電力を生じ，主電流を通じる電機子巻線および電機子鉄心（arma-
ture core），電機子巻線に接続された整流子（commutator ），整流
子に接触し電流を流すブラシ（brush），およびそれらを支持する
構造部から構成される．
.直流直巻モータ 界磁巻線と電機子巻線を直列に結線し
た接続方法を用いる．起動時に大きなトルクが得られるが，電機子
の回転速度の増加とともに急激にトルクが減少する．端子電圧の極
性を逆にしても，界磁電流と電機子電流の極性が同じであるため回
転方向を変えることができないことから，変速機と組合わせて用い
られる．
.直流分巻モータ 界磁巻線と電機子巻線が並列に結線し
た接続方式を分巻モータと呼ぶ．界磁巻線と電機子巻線が独立した
電源に接続されるモータを含めて称される場合もある．界磁電流を
調整することにより広範囲の回転速度での運転が可能であること
と，界磁電流あるいは電機子電流のいずれかを正負とも流せるよう
にすれば容易に正逆両方向での回転が実現できることが特徴であ
る．
.直流他励モータ 電機子電流とは別の直流電源から励磁
電流を受けて界磁磁束を作る方式であり，2種類の直流電源が必要
である．図6･26の例では，エンジンによりベルト駆動された発電
機の出力電流を電機子（回転子）電流とし，固定子の界磁コイルに
はコントローラの制御電流を用いている．
