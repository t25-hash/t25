# 4・3・1 運動方程式

> **章**: 第4章 航空機
> **節**: 4・3 操縦・安定性・飛行性

航空機の運動を論じる場合に使用する、座標系の定義と機体軸系での航空機の運動方程式を示す。4・3節で使用するおもな記号を記しておく。添字としては、釣合い飛行状態には零、昇降舵、補助翼、方向舵に関するものにはそれぞれ,、推力に関するものにはを使用する。a. 座標系航空機の運動を論じるときに使用される座標系としては、機体軸系、安定軸系、風軸系、地面固定座標軸系がある。b. オイラー角(Euler angle)航空機の姿勢は、地面固定軸系に対する機体軸系の向きとしてオイラー角を用いて定義される。6―111図4・25 機体軸と記号の定義すなわち、地面座標系と一致していた機体軸系が、,軸の順序でその軸まわりにΨ,Θ,Φだけ回転して現在の機体軸の方向になったとする。c. 運動方程式航空機は剛体であり平面に関し左右対称であると仮定すると、次の6自由度の運動方程式が導かれる。次に、航空機の運動が釣合い飛行からの微小擾乱であると仮定して運動方程式を線形化する。基準となる航空機の釣合い状態としては横滑りなしの定常直線飛行を考え、そこからの各変数の擾乱量を一次の微少量と仮定すると線形化された運動方程式は縦の運動(状態変数,α,θに関するもの)と横・方向の運動(状態変数β,φに関するもの)とに分離され、線形化された運動方程式は次のようになる。なお無次元化した微係数を無次元安定微係数(stability deriva-tive)と呼ぶ。
