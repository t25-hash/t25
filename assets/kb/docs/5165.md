# 4・3・2 安定性と釣合い

> **章**: 第4章 航空機
> **節**: 4・3 操縦・安定性・飛行性

安定性(stability)には静的安定性と動的安定性の2種類がある.釣合い飛行をしている航空機が,何らかの原因で平衡状態から変化したとする.このとき発生した力やモーメントが,航空機を元の釣合い状態に戻す向きに作用するとき,航空機は静的に安定であるという.静的な安定とは,元の状態へ戻そうとする復元力や復元モーメントが働くかどうかというだけで,それからの運動には関係しない.これに対し,航空機に復元力や復元モーメントが作用するだけでなく,元の釣合い状態まで戻るように力やモーメントが働くとき航空機は動的に安定であるという.静的な安定,動的な安定は,それぞれ縮めて,静安定,動安定とも呼ばれる.本節では静安定性と釣合い状態について述べ,動安定性については4・4節で述べる.a. 縦の静安定性航空機の縦運動における静安定性として,迎え角安定と速度安定がある.. 迎え角安定航空機が釣合い状態から迎え角のみが変化したとき,迎え角を元の状態に戻すようなピッチングモーメントが働くとき,迎え角安定は正(安定)という.このときピッチングモーメントの発生を表す安定微係数が욇であり,욇<0ならば迎え角安定が正である.迎え角安定を表す安定微係数욇は,迎え角に対する揚力係数の変化を表す微係数により近似的に次式で与えられる.욇=-욈(4・73)ここで,重心位置が位置욈にあるとき,迎え角変化によりピッチングモーメントは変化しないので,욈は縦安定中立点(neutralpoint)と呼ばれる.式(4・73)は迎え角安定が揚力変化により発生し,重心位置と揚力変化の作用点욈の位置関係により決まることを示している.迎え角安定を有するためには,一般に>0であるから-욈<0が必要である.すなわち,重心位置は욈より前方にある必要がある.. 速度安定釣合い状態から速度が増大したとき機首上げモーメントが発生すれば,これは速度変化を復元する向きに働くので速度に関し静的に安定である.これは一般に縦の静安定と呼ばれるが,速度安定と呼ばれることもある.航空機が速度욂욉で水平釣合い飛行しているとする.スロットルを不変とし昇降舵のみを微少量変化させたとする.このとき,操縦かん押し(dδ욄>0)によって定常状態での速度が増加(d>0)する,すなわち,dδ욄/d>0ならばパイロットには操縦かん固定の縦の静安定性があると感知される.また,操縦力の押し(d욄>0)により定常状態の速度が増加(d>0)する,すなわち,d욄/d>0ならば,パイロットには操縦かん自由の縦の静安定があると感知される.操縦かん固定と自由の違いは,操縦かん固定が昇降舵角に対する速度変化を表すことから航空機そのものの特性を表しているのに対し,操縦かん自由は航空機そのものの特性に操縦系統特性や安定操縦性増大装置などによる制御を含めた特性を表しているといえる.b. 横・方向の静安定性横・方向の静安定は横滑りによるローリングモーメントとヨーイングモーメントの発生により決定される.航空機は基本的にバンク角変位に対して復元性をもたない.バンク角変位により横滑りが発生し,横滑りによりロール復元モーメントを生じ横の安定性をもつ.航空機が釣合い状態から横滑り角のみが変化したとき,横滑りの方向と反対方向にロールする復元モーメントを発生させるとき上反角効果(dihedral effect)が正,または横の静安定性があるという.上反角効果は,横滑りにより発生するロールモーメントを表すもので,安定微係数욆で表される.上反角効果に影響を与える要因としては,主翼の上反角,後退角,主翼の上下取付位置や垂直尾翼形状や面積がある.上反角効果がある場合,욆<0が成り立つ.釣合い状態から横滑り角のみが変化したとき,横滑りの方向に機首を向ける復元モーメントを発生させるとき方向安定もしくは風見安定があるという.方向安定性は安定微係数욈で表され,욈>0のとき方向安定があるという.方向安定には,垂直尾翼面積と重心-垂直尾翼空力中心間の距離の積が大きく寄与する.c. 縦の釣合い定常直線飛行中の航空機において揚力方向の力および抗力方向の力が釣合っており,さらに重心まわりのピッチングモーメントが零のとき,縦の釣合い状態にあるという.縦の釣合い状態を決定する変数は,迎え角α,昇降舵角δ욄および推力である.ピッチングモーメントの釣合いを考える.航空機の重心位置が前後に移動したとき,空力特性の揚力と抗力については変化しないが,ピッチングモーメント特性は変化する.つまり,重心まわりのピッチングモーメント係数욇は,욈位置での空力特性(,욇)を用いると,욇=욇+-욈(4・74)と表される.욇がδ욄に対して線形であるとすると,δ욄=δ욄+-욈/욇욄(4・75)となる.これは釣合い(욇=0または=0)を実現するとδ욄の関係を示し,図式的には図4・26に示すようである.d. 横・方向の釣合い航空機の横と方向の運動特性は互いに連成しており,両者を分離して取扱える場合は少ない.さらに,横・方向の運動は縦の運動も誘起することもあるが,ここでは簡単のため縦への連成は考慮しない.横・方向の釣合いとして,定常横滑り飛行,非対称推力時の定常直線飛行および水平定常旋回飛行について簡単に述べる.定常横滑り直線飛行は,航空機を一定の横滑り角で滑らせバンク角をとりながら直線飛行するものである.横滑りによる横力と上反り角効果によるローリングモーメントを風上側にバンク角をとるようにエルロン操舵し,ヨーイングモーメントと横力を打ち消すためにラダー操舵して釣合い状態を作り出す.釣合い状態を決定する変数は,横滑り角β,エルロン舵角δ욀,ラダー舵角δ욥,バンク角φの関係図4・26 釣合い昇降舵角とである.非対称推力時の定常直線飛行は,多発機のエンジン1基が故障した状態で定常直線飛行をしている場合である.このとき,エンジン故障による左右非対称推力によりヨーイングモーメントが発生する.これをラダー操舵と横滑りにより打ち消して釣合わせる.また,横滑りによるローリングモーメントを打ち消すためにエルロン操舵とバンク角をとる.この釣合い状態は,横滑り角β,バンク角φ,エルロン舵角δ욀,ラダー舵角δ욥により決定されるが,解は一意には決まらない.しかし,通常の航空機の場合,エルロン舵角は小さく問題になることはないため,近似的にδ욀=0として釣合い状態を求めることができる.水平定常旋回は,航空機に働く横力が零となっている旋回飛行状態である.このとき旋回により機体に作用する遠心力は,バンク角をとり揚力面を旋回中心に傾かせることにより釣合わせる.水平定常旋回時の旋回率をψとすると,定常旋回時には角速度は=0,=ψcos φ,=ψcos φの関係があり,縦の運動と横・方向の運動が連成した運動となる.釣合い状態を決める変数は,バンク角φ,迎え角α,エレベータ舵角δ욄,エルロン舵角δ욀,ラダー舵角δ욥である.横滑り角βは零である.
