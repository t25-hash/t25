# 6・3・5 電磁鋼板（electrical steel sheet ）

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

モータに用いられる電磁鋼板には，モータの高効率化，小型化す性を得るため，炭素や窒素などの不純物を減らすことで透磁率を高げることで，鉄損低減を行った無方向性電磁鋼板が用いられる．
し外にも合金の硬さや脆性を増し，加工性を悪化させる．
もたせるために付与されている被膜の諸特性なども重要な特性とな性とコストを総合的に判断し，最も適した電磁鋼板が決定される．
