# 4・3・3 運動モードと操舵応答

> **章**: 第4章 航空機
> **節**: 4・3 操縦・安定性・飛行性

縦の運動は、通常の航空機では短周期モード(short-period mode)と長周期モード(long-period mode)あるいはフゴイドモード(phugoid mode)と呼ばれる運動に分類できる。このモードは通常、周期が数秒程度と短く、減衰が比較的大きい。長周期モードは、減衰が小さく周期の長い運動である。軸方向速度やピッチ角変化に比べて、迎え角変化が小さいことが特徴であり、迎え角安定の大きな機体が速度とピッチ角をゆっくりと変化させる運動である。また、長周期運動は機体の運動エネルギーと位置エネルギーの交換により発生するともいえる。操舵応答の例として、エレベータをパルス入力して縦の短周期モードを励起した場合の運動時歴を図4・27に示す。横・方向の運動は、通常の航空機ではスパイラルモード(spiral mode)、ロールモード(roll mode)およびダッチロールモード(dutch-roll mode)と呼ばれる運動に分類できる。 スパイラルモードスパイラルモードは、一般に時定数の大きなゆっくりとした運動である。発散する場合もしばしばあるがパイロットによる修正は容易である。このモードではおもにロール角と方位角が変化し、これらに比べて横滑り角は小さい。スパイラルモードが安定か不安定かを調べる飛行試験では、まずバンク角を若干とる操作をしてから手放しして、その後のバンク角の挙動を見る。このとき、バンク角が次第に深くなるときスパイラルモードは不安定であり、バンク角が零(水平)に収束すれば安定である。スパイラルモードのメカニズムは次のようである。機体が擾乱を受けて右バンクして横滑りしたとすると、方向安定(>0)により機首を右方向に向けるヨーイング運動が発生する。逆の場合はバンク角は零に戻され、スパイラルは安定となる。. ロールモードロールモードは軸まわりのロール角速度が減衰していく運動であり、減衰モーメントはおもに主翼から発生する。ロールモードは通常安定であり、時定数は-1/′で決まる。. ダッチロールモード航空機が方向安定を有するために発生するヨーイング運動とこれによって引き起こされる横滑りの振動が、上反角効果を介してロール角の振動を連成させる運動である。典型的なダッチロール運動では、ロール運動に対し少し遅れて横滑りが発生し、方位角はほぼψ-βの関係を保ちながら変化する。ダッチロール運動のメカニズムは、次のように説明される。ラダー操舵により右ヨーイング運動が励起されたとする。ヨーイングモーメントにより機首は右を向き、それと同程度の左横滑りが発生する。この横滑りは上反角効果による右ローリングモーメントを発生し右へバンクする。同時に、方向安定により機首は左ヨーイングモーメントを発生し、機首を元に戻そうとし横滑りは小さくなる。この繰返しにより、機首方位の変動とバンク角変動がある位相をもって振動的になる運動がダッチロール運動である。操舵応答の例として、ラダーをパルス入力してダッチロールモードを励起した場合の運動時歴を図4・28に示す。
