# 6・3・6 磁石材料

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

（プラセオジム），Dy（ジスプロシウム）など希土類元素をバランも使用可能な磁石が実用化され，自動車用途に使うための障害では面への絶縁処理を施すことも有効な手段とされている．
コ磁石，サマリウム・コバルト磁石（Sm-Co磁石），ネオジム・38)．
高価なサマリウム，コバルトの希土類元素を用いた磁石が開発されたことから，自動車駆動用モータへの適用が進んだ．
環境温度が高いことから高温で減磁しないことが要求される．
Nd-が低いという欠点をもっていた．
スよく使うことで，現在では使い方によっては200℃以上の高温でなくなっている．
