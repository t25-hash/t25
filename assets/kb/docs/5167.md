# 4・3・4 飛行性と操作性

> **章**: 第4章 航空機
> **節**: 4・3 操縦・安定性・飛行性

飛行性(flying qualities)は、航空機自体の安定性・操縦性(stability and control)にパイロットによる操縦や操作のしやすさを示す操作性(handling qualities)を加えたものと定義される(図4・29)。6―114交通機械航空機の操縦は、与えられたミッションを遂行するためにパイロットと機体から構成される閉ループシステムによって実行される。このため、機体の固有特性が不十分であってもパイロットがこれを補償することにより十分にミッションを遂行することができる場合がある。この補償のために要求されるパイロットワークロードの程度で飛行性を評価することができ、この尺度をパイロットレイティング(PR : pilot rating)という。よく知られた尺度として、クーパー・ハーパー・レイティング(CHR : Cooper-Harper ratingscale)がある(図4・30)。CHR はパイロット評価という定性的な基準を定量化して評価できる点で優れており、航空機開発でしばしば使用される。CHR は、最高が1となる10段階の評点からなり、さらに次の三つのレベルで評価される。レベル1 1≦≦3.5 :満足、改善の必要なし。レベル2 3≦≦6.5 :不満であるが許容される。改善が望ましい。図4・28中央より5ミリ左にずらしている。図4・28 横・方向の操舵応答の例レベル3 6.5≦≦10:許容し得ない。改善が必要である。
