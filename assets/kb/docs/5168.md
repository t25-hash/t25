# 6・3・8 回転検出器

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

モータの回転角度検出器は，出力信号の形態によりアナログ検出
器とディジタル検出器に大別される．前者には，DCタコジェネレ
ータ，シンクロレゾルバが，後者はロータリエンコーダがあ
る ．PMモータの制御では，トルク制御部，電流制御系， -
軸への座標変換系および位置検出器からなる制御系とPWM
（pulse width modulation）インバータの組合せが用いられる．磁
極位置を正確に検出する方法が必要であり，位置検出器としてレゾ
ルバ（resolver ）と呼ばれる電磁誘導型角度検出器が用いられる例
が多い（図6･39) ．レゾルバの信号変換器にはRD変換器
（resolver digital transformer ）と呼ばれるIC（半導体集積回路）
が用いられ，図6･39中の制御誤差εを零に近づけるよう閉ループ
負帰還制御することでアナログ角度情報がディジタル値に変換され
る．
へのスキュー（skew），ロータ磁石配置での位相差の設置などがあ
る ．誘導モータでは，ステータとロータ間のエアギャップの
不均一，ステータ巻線の不平衡などによる磁気吸引力による振動騒
音現象が知られている．PMモータでは，直流モータと交流誘導モ
ータのそれぞれの振動強制力が存在することになる．このほか，イ
ンバータのPWM駆動による騒音現象として搬送波（キャリヤ）
周波数の電磁騒音があり，この上限周波数はスイッチング素子によ
り決まる．PMモータの振動騒音現象と強制力および対策例を表
6･13 に示す ．
