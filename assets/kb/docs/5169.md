# 6・3・9 モータの振動騒音

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・3 モータ

直流モータ，PMモータでは無通電状態でも永久磁石の磁束が存ルク脈動が回転に伴って発生する．
構造体の固有振動数と共振した場合に騒音が増大する．
マトリックスコンバータクト駆動方式，減速機付き方式がある．
図6･41に電気自動車在することから，コギングトルク（cogging torque）と呼ばれるト数と固定子突極数の最小公倍数で決まる脈動周波数をもつ．
その低極があり，ステータはPMモータと同様の構成である．
イールモータと呼ぶ．
トランスミッション，デファレンシャルギヤ配置の設計自由度が増す利点がある．
用に開発されたインホイールモータの構造例を示す．
