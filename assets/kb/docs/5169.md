# 4・3・6 ヘリコプタ

> **章**: 第4章 航空機
> **節**: 4・3 操縦・安定性・飛行性

a. ヘリコプタの種類と操縦舵面ヘリコプタは回転翼が作る空気力を揚力,推進力,操縦力として利用するロータ形式の航空機である.ロータ形式でヘリコプタを分類すると,機体重心付近に駆動軸を垂直に取付けた大直径のメインロータを配置し,尾部にメインロータ回転によるトルクのキャンセルおよび方向操縦用の小直径のテールロータをもつシングルロータヘリコプタ,互いに反転する2個のメインロータを機体の前後に配置するタンデムロータヘリコプタ,互いに反転する2個のメインロータを機体の左右に配置するサイドバイサイドヘリコプタ,互いに反転する2個のロータを同軸上下に配置する同軸反転ロータヘリコプタがある.図4・32にシングルロータヘリコプタの構成要素を示す.b. 操縦装置ヘリコプタの操縦装置は操縦かん(サイクリックスティック),ペダルおよびコレクティブピッチレバーで構成される.サイクリックスティックは,ロータ面をスウォッシュプレートを介して前後左右に傾斜させ,ピッチとロール運動を発生させる.ペダルはシングルロータヘリコプタではテールロータのピッチ角を変更してヨーイング運動を発生させ,タンデムロータヘリコプタでは前後ロータを左右反対方向に傾斜させてヨーイングモーメントを作り出す.コレクティブピッチレバーはスウォッシュプレート6―116交通機械を上下させ,メインロータのピッチ角を変更してロータ揚力を増減させる.c. ヘリコプタの運動方程式ヘリコプタの重心まわりの運動方程式は,基本的に式(4・67・a)~(4・67・f)と同じである.しかし,ヘリコプタの場合,回転体であるロータが主要な力を発生しているため,ロータのダイナミックスと空気力を計算する必要がある.ロータに働く空気力を計算する方法としては,ロータの上流と下流の運動量の変化から空気力を推定する運動量理論があるが,その最も簡単な方法としては,ロータ面を一つの円板翼と考え,誘導速度がロータ面で一様で時間的に変化しないと仮定する方法がある.また,ロータ面内を円周方向と半径方向に分割し,各微小扇形要素に運動量理論を適用する方法もある.他方,各ブレードを半径方向の翼素に分割して各翼素に働く空気力を求める翼素理論がある.ロータの運動を記述する座標軸系としては,ロータハブ中心を原点にとったロータ軸系,ロータ軸系をロータ軸まわりに回転させたハブ軸系やブレード軸系が使用される.d. 安定性と操縦性ヘリコプタの安定性は,固定翼機に比べて劣っている.その原因のおもなものは,ロータが前進飛行時の迎え角変化に対して静安定性をもたないことと,機体速度変化に対し重心まわりのモーメントが連成することにある.ヘリコプタの安定性について特徴的なものを以下に簡単に示す.縦の静安定性(速度安定性)はホバー近傍では安定であるが,前進飛行時には不安定化することが多い.縦の迎え角安定はホバー時にはほぼ中立,前進飛行時には水平安定板のテールボリュームにより安定になったり不安定になったりする.縦の短周期モードは迎え角安定に影響を受け,ホバー時には安定であるが前進飛行時には不安定化することもある.固定翼機の長周期モードに相当する中周期モードは速度安定と迎え角安定に影響され,不安定になる機体も少なくない.ヘリコプタの操縦性上の特徴としてカップリング特性がある.角速度による運動へのカップリングとしては,ピッチ-ロールカップリングおよびロール-ピッチカップリング,操舵によるカップリングとしては,コレクティブピッチ-ヨーカップリングがある.ヘリコプタは固有の不安定性やカップリング特性を有しているため,安定した飛行を遂行するにはパイロットがそれらを補償しながら操縦するか,安定操縦装置により補償する必要がある.このため,コントロールパワーを含む操縦性の確保が重要である.e. 飛行制御ヘリコプタは本来不安定な特性を有するため,飛行制御装置は,古くから必須のものとして発達してきた.ヘリコプタの飛行制御装置としては,安定性増大装置(SAS : stabilityaugmentation system)や安定性操縦性増大装置(SCAS : stabil-ity and control augmentation system),縦静安定特性の増強のための静安定機能増大装置やAST(automatic speed trim)装置,パイロットアシスト機能としてのASE(automatic stabilizationequipment)や自動操縦機能をもったAFE(automatic flightequipment)やAFCS などがある.ヘリコプタに特徴的な自動操縦機能の一例としては,自動アプローチ/ホバリング機能がある.これは,巡航状態から目標点へ設定されたアプローチプロファイルに従って減速・降下して目標点の上空で自動ホバリングする機能である.
