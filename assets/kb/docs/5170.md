# 4・4・1 空力

> **章**: 第4章 航空機
> **節**: 4・4 機体・製造技術

a. 全般航空機は空気力を利用して大気中を飛行するため、空力は航空機設計の中でも重要な分野の一つであり、高い性能や効率とともに安全性への配慮が求められる。空力設計には大別して諸元策定、空力形状、空力性能、空力特性に関する各作業があり、設計の進捗に従って空力に関わる機体形状、諸性能、諸特性を定義し、確認していく。b項以下に固定翼航空機を例にとってそれぞれの概要を述べる。b. 諸元策定(aircraft sizing)要求性能に基づいて、主翼面積、主翼平面形、エンジン、機体重量等の機体主要諸元を定める作業で、設計作業の最上流に位置する。要求性能の種類は対象とする航空機により異なるが代表的には航続性能、速度性能、高度性能、離着陸性能、燃料消費率等がある。図4・33は旅客機について各要求性能を満たすエンジン推力と主翼面積の組合せに関する諸元策定結果の例である。図4・33中の塗りつぶした領域が930km(500海里)を1座席当たり24kg 以下の燃料消費で飛行できるエンジン推力と主翼面積の存在領域を表している。この諸元策定プログラムはせた構成となっている。c. 空力形状(aerodynamic configuration)策定された主要諸元に基づいて細部の外形形状を定義する作業である。主翼、尾翼、胴体、エンジン等を適切に配置するとともに主翼等の各空力構成要素の三次元的な形状を定義する。主翼形状定義では、諸元策定にて定まった平面形を基に断面形を定義する。断面形の決定に際しては、離着陸を主体とした低速度域と巡航を主体とした高速度域の両方を設計点に選ぶことが多く、戦闘機等ではこれに高機動に関わる設計点が加わる。図4・34に代表的な翼型を示す。特に超臨界(supercritical)翼型は遷音速性能に優れるため、最近の民間旅客機に多く用いられている。従来はこの中から翼型を採用していたが、最近では、目標とする空力性能を圧力分布に置換えてこれを満たす翼型を求める逆解法もよく用いられる。図4・35に逆解法を用いた主翼形状の設計例を示す。図4・36に代表的な高揚力装置の形態とその特性を示す。次に胴体形状の定義では機軸方向の胴体断面積分布や主翼との結合部の形状が性能や特性を定めるうえで重要となる。図4・37は胴体と主翼の断面積分布の総和の変化を滑らかにすることにより、遷・超音速空力抵抗を減少させる手法の例でエリアルール(area rule)と呼ばれている。空力形状の設定や結果の確認手段としては風洞試験やCFD(computational fluid dynamics、計算流体力学)が用いられる。図4・38にCFD により機体まわりの流れの様子を解析した例を示す。d. 空力性能設定された空力形状に基づいて揚力、抵抗の推算を行い、別途推算する重量やエンジン推力等と併せ用いて4・2節に示した性能計算を実施し、機体が要求性能を満足していることを確認する。. 航続性能搭載可能な有償重量(ペイロード、pay-load)と航続距離(レンジ、range)との組合せで表される性能でペイロード・レンジ性能ともいう。機体の揚抗特性、機体重量、エンジン性能の3要素に支配されるため、機体の低抵抗化、軽量化がつねに求められる。6―117. 離着陸性能運用される空港により要求が決められ、所期の離着陸滑走路長での運用性を確認する。離陸時はエンジン推力と揚抗特性が、着陸時は高揚力装置を作動させた状態での最大揚力が性能を左右する。耐空性審査要領では離陸上昇こう配の規定があり、通常はエンジン推力を得にくい高温かつ高地での運用が標定となるが、氷結滑走路での運用への配慮も必要である。. 上昇性能燃料消費率最小で巡航するために必要な上昇限度まで上昇して巡航可能であることを確認する。また双発機では、山脈越えの運用に備えてエンジン片発停止状態における上昇高度要求も満たす必要があり、エンジン選定において配慮を要する。. 速度性能航空機の運用に従って最大巡航速度等の速度性能に関わる要求がある。目標速度までは有害な抵抗増加を防ぐとともに、揚力や姿勢角の急変も避けるような設計を心がける。. その他航空機の運用に従って固有の空力性能要求があり、個々に確認を必要とする。例えば、一般的に戦闘機では低速度から高速度への加速時間に関わる要求や、旋回半径または旋回時間に関わる要求が付け加わる。e. 空力特性空力特性は航空機の安定性および操縦性(合わせて飛行性と呼ぶ)に深く関与し、飛行中および地上において良好な飛行性を確保するため、各国ではそれぞれに定めた耐空性基準において規定を設けている。以下に代表的な飛行性の例を挙げるが、これらの特性推算は、空力係数、機体重量、機体慣性能率の推算値を用いて行われ、飛行試験により確認される。. 静安定特性航空機が釣合い状態で飛行中に何らかの擾乱を受けたときに、元の釣合い状態に戻ろうとする特性を有している場合、航空機は静安定(static stability)を有しているという。機種の上げ下げや速度の増減に関わる縦静安定特性と、機軸まわりのバンク角(横揺れ)や水平面内の機首振り(偏揺れ)に関わる横揺れおよび方向静安定特性とがある。特性は機体の形態(脚や高揚力装置の状態)や速度、高度に左右されるため、一般には全運用飛行領域において十分な静安定を有する必要がある。最近では電気式操縦システム(FBW)による航空機も実用化されてきており、静安定余裕を緩和することにより運動性を高めたり、尾翼等の安定板面積を減少させて軽量化を図っている機体もある。. 動安定特性航空機が釣合い状態で飛行中に擾乱を受けたときに元の釣合いまで戻る力やモーメントが作用する場合を動安定(dynamic stability)があるといい、復元力、慣性能率、減衰力等に影響を受ける。動安定に関する要求は、操縦者の制御能力と密接に結びついていて、人間の知覚および動作に不可避の時間遅れのため、時定数の短い運動ほど強い安定性を要求される。代表的な動安定特性として、急速な縦揺れ操舵により生じうる迎え角変動である縦短周期モード、擾乱を受けた後に安定した速度状態に落ち着こうとするときに起こる縦長周期(フゴイド)モード、横揺れと偏揺れとが組合わさった揺動運動を繰返すダッチロールモード等がある。. 失速特性一般的には、迎え角の増大により発生する気流のはく離に起因する現象であり、はく離の仕方により失速特性の善し悪しが定まる。主翼後退角の小さい航空機では主翼内翼からの気流のはく離に伴い水平尾翼による機首下げの傾向を生じ、主翼後退角が大きい場合は、主翼外翼からの気流はく離に伴う機首上げの傾向が生じる。特に翼端失速を生じる場合は補助翼利きの低下により失速(stall)からの回復操舵に困難をきたすことがあるため、極力これを避けるよう設計する。失速に至る前にパイロットが明りょうに識別できる空力的または人工的警報を装備することが規定で定められている。. その他着陸形態における横風に対する舵利きに関する横風着陸特性、離陸滑走中および上昇中に1基のエンジンが停止した場合の操縦性に関わる非対称推力時操縦性、失速後のきりもみ状態に関わるスピン特性等各種空力特性があり、航空機の用途により必要な特性が規定により定められている。
