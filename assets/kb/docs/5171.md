# 6・4・2 直流モータ用チョッパ

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・4 電力変換器と制御装置

ゼロ回転時のトルク発生微低速時のクリープトルク電力変換器はこれらのモータ駆動力を制御するためにモータシステムへの電力変換と制御を行う装置である．
電力変換方式により，直流から直流へ変換するチョッパ回路，直流から交流に変換するインバータ（inverter），周波数変換を用いたサイクロコンバータ（cycloconverter）がある．
二次電池を用いる方式の電力変換方式には，表6･14 に示す種類があり，直流から直流へ変換する直流チョッパ回路，直流から交流に変換するインバータ，周波数変換を用いたサイクロコンバータ，マトリックスコンバータなどがある．
交流誘導モータ，PMモータではPWM制御を用いた三相電圧型インバータが一般的に用いられている．
モータ制御以外の電力変換器では，ハイブリッドシステムの高電Dが導通し，＝0となる．
平滑リアクトルで平均化された電圧がある．
また，燃料電池自動車では，燃料電池の発電電力（直流）と車載二次電池（直流）との間で電圧調整を行う必要性から，駆動モータ用インバータのほかに，大容量のDC- DCコンバータが用いられる．
このほか，電気自動車では，外部電源から車制動時の回生制御では，IGを動作させ，直流モータを電源とし行ではモータの回転方向を逆にする必要があり，電機子端子電圧，いる．
直流モータを用いた駆動システムはおもに小型電気自動車に用いられ，その制御には抵抗制御，界磁制御，電圧制御の三つの方式がある．
抵抗制御は電機子回路に直列に抵抗を挿入し，この抵抗値を変えることで出力トルクの制御を行う．
界磁制御は界磁抵抗器またはチョッパ（chopper）が用いられる．
電圧制御はモータの端子に印加される電圧を変化させる方法である．
電圧制御には，電圧切換え制御，直流チョッパ制御がある．
電圧切換え制御は車載の組電池のタップ切換えをする方法であり，ダイオードとスイッチを用いた回路構成がとられる．
直流チョッパ制御は半導体スイッチを用いてオンオフさせることでモータに印加される直流電圧の平均値を制御する．
