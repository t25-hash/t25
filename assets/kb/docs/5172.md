# 4・4・3 装備(system)

> **章**: 第4章 航空機
> **節**: 4・2 節の各性能を計算する機能に空力モデルや重量モデルを組合わ

a. 主要装備航空機の装備品には、おもに下記のような各種の制御系統、操作装置および機器がある。(1)動力系統、(2)降着系統、(3)操縦系統、(4)油圧および空気系統、(5)空気調和(空調)および与圧系統、(6)防除氷、防曇、防雨系統、(7)計器、(8)電気系統、(9)電子系統、(10)室内装備、(11)火災報知系統、消火系統、(12)保命装備。b. 動力系統航空機用動力装備は、ターボジェットエンジン、ターボファンエンジン、ターボプロップエンジン、ピストンエンジンなどに代表されるエンジン本体および、それらを連続に正常に作動させるための各種装置、または系統(空気取入および吸気系統、エンジン制御装置、燃料系統、潤滑系統、冷却系統、始動装置、エンジン火災検知および消火装置、スラストリバーサ、ノイズサプレッサ、補助動力装置)からなる。. エンジン制御エンジンへ供給される燃料の制御や、推力をパイロットが要求する値へ保持する機能、また可能な限り速やかにエンジンを加速、減速する機能をもつ。従来は、索、リンケージで機械的にエンジン出力を制御していたが、最近は制御装置が電子化され、電子信号で制御(throttle by wire)する方式がとられつつある。. 燃料系統機体に燃料を貯蔵して、その燃料をエンジンの要求に応じて間断なく供給する機能をもつ。構成品としては、燃料タンク、燃料ポンプ、流量コントロールバルブ、圧力スイッチ、燃料系、ベント口、燃料ヒータ、警報装置がある。. 補助動力装置航空機の運用に必要な動力がエンジンから抽出できない場合の補助動力源、駐機中の電源・圧縮空気源として使用され、ガスタービン式とラムエアタービン式がある。最近の趨勢として、電子技術を駆使したFADEC(full author-ity digital electronics control)をエンジンに装備して、外気温度、排気温度、圧力、エンジン回転数等の値を自動的に計算し、エンジンを所定の出力にするまでの制御を最適化している。これにより燃費の節減、レスポンスの向上、乗員の仕事量の軽減(オートスロットル)を果たしている。c. 降着系統降着装置の機能は、地上移動・地上滑走の役割と着陸時に機体に加わる衝撃力を軽減することである。おもな構成要素は、脚、緩衝器、揚降機構、操向装置、ブレーキ系統、車輪、タイヤおよび非常脚下げ系統である。ブレーキ系統は、ドラムブレーキとディスクブレーキがあるが、ディスクブレーキは小型、軽量の割に吸収エネルギーが大きいので最近の航空機ではディスクブレーキが主流である。アンチスキッド装置は、着陸時、ブレーキを急激にかけた際の車輪ロックによるタイヤ破裂の危険を防ぐため、最近ほとんどの機体へ装備されている。この装置は車輪回転状況を検知して最適のブレーキ圧にコントロールする装置である。最近の趨勢としては、以下の3つが挙げられる。(1)超高抗張力鋼の使用・軽量化、(2)高性能ブレーキ、カーボンブレーキ、(3)電子技術を用いた操向装置(SBW : steering by wire)、ブレーキ装置(BBW : brake bywire)。d. 操縦系統操縦装置は航空機における最も基本的な装置であり、その目的は各舵面を作動させることにより、(a)機体姿勢の制御、(b)静安定・動安定の改善を行うこと、である。機体の大きさ、使用目的により多少差異はあるが、基本的には操縦かん(輪)とフットペダルからなり、前者は縦と横の操縦に、後者は方向操縦とブレーキ操作に用いられる。舵面としては、無尾翼機などの一部例外を除いて、補助翼・昇降舵・方向舵があり、三軸回りの制御を行うのが通例である。操縦方式は、大別して、(a)人力操縦(直接操縦)、(b)機力操縦(間接操縦)、に分かれる。亜音速機や小型機には人力操縦方式、遷・超音速機や中・大型機6―120交通機械人力操縦とは、操縦かん(輪)またはフットペダルから各舵面までを操縦索およびリンケージでつなぎ、すべてを人力で操縦する方式である。人力操縦の場合、パイロットが加速度や姿勢変化を検知する良いセンサであるとともに、操舵を行う動力源であったが、機体の大型化、多数、多様の機体が低速から高速までの広い速度範囲を飛ぶようになると、パイロット(人間)の能力では広域を網羅できなくなった。操舵力の動力源としては、応答性、必要な作動力が得やすいことから油圧が用いられている。その他のパイロット能力を補うものが飛行制御装置である。機体の動的安定を改善する安定増大装置(SAS : stability augmentation system)、操縦性を改善する操縦性増大装置(CAS : control augmentation system)がその代表である。そして、コンピュータ等電子技術の目覚ましい進歩により、電子式制御を主体とするFBW(fly by wire)方式が一般化しつつある。FBW とは機力操縦系統で用いられていた機械的リンケージを電線に置換えて電気信号のみで舵面操舵アクチュエータを制御する機力操縦の一方式である。後述する電子系統(アビオニクス)の構成部品の一つである飛行管理装置(FMS : flight management sys-tem)からの信号が自動飛行制御システム(AFCS : automaticflight control system)に入力され、アクチュエータ制御コンピュータ(ACC: actuator control computer)を介し舵面アクチュエータを制御する。e. 油圧および空気系統。 油圧系統航空機のエンジン、その他の動力源より引出した機械的な力を、油圧ポンプにより作動油を昇圧し、配管を通して機体各部に送りアクチュエータや油圧モータ等の機械的な出力に変換する力の伝達装置を一般に油圧系統という。主要構成部品として以下の機器がある。(1) リザーバリザーバは油圧系統の作動・温度変化・漏えいによる油量の変化の吸収、ポンプの入口圧力の確保、系統内で発生する空気の排出をおもな目的とする。(2) ポンプポンプは構造上多くの種類があるが、航空機に要求される高圧、高効率およびコンパクトを満足するアキシアルピストンタイプ(axial piston type)の可変吐出し容量型ポンプが多用されている。(3) アキュムレータアキュムレータはポンプの吐出し量が不足した場合の短時間の油圧供給や脈動吸収、動力がないときの蓄圧に使用する。また、油圧系統の重量軽減、容積縮小を目的とした作動圧力の高圧化が実用化段階にあり、4000~5000psiの油圧系統をもつ実用機が登場している。. 空気圧系統空気圧源としてエンジン、補助動力装置(APU : auxiliary power unit)などの抽気圧の利用、機内に搭載するポンプがある。油圧系統に比べ作動媒体である空気は、圧縮性が高く、蓄圧が容易で、かつ環境に優しいことから飲料水システムの加圧、扉シール装置等で使用されている。その他、油圧リザーバおよびアキュムレータの加圧等に用いられている。f. 空気調和および与圧系統空調・与圧系統は、航空機の乗員および乗客に快適な居住環境を与えるため、および搭載電子機器の冷却を行うために、機内の温度、湿度、流速、換気、および絶対圧力の制御を行う。冷暖房の熱源としては、ガスタービンエンジンの抽気、燃焼加熱器、電熱器等の高温熱源や外気、燃料、水、極低温冷媒(液体窒素)などの低温熱源が利用される。また、圧力源としては、ガスタービンエンジンの抽気がおもに使用される。温度や圧力の制御には従来空気圧制御方式のバルブ類が用いられてきたが、最近では電子制御によるバルブ等による制御が主流である。g. 防除氷、防曇、防雨系統航空機が過冷却の水滴を含む着氷気象中を飛行すると、機体の各よどみ点付近に氷が付着し、抵抗の増加、揚力の減少が発生し、特に翼前縁、およびエンジンの空気取入れ口、操縦席風防ガラスの外側等に付着すると操縦安定性、視界不良が生じ飛行安全に影響を与える。防氷、除氷の方式として、加熱空気式(エンジンの抽気)や、電熱式(ヒータマット、電熱ガラス)、機械式(ラバーブーツ)および除氷液式等がある。防曇としては、操縦席の風防ガラスの内側に水滴や霧が付着することによる視界不良を避けるために、上記防氷用電熱ガラス以外にも、加熱空気または、乾燥空気を吹付ける方式がある。防雨としては、特に離着陸時の雨中の飛行に対しパイロットの視界を守るために、ワイパや加圧空気の吹付けの方式がある。h. 計器系統飛行計器として機体の姿勢、速度、高度を表示する装置、また航法計器として水平面における機体の場所、飛行方向および目的地までの距離、時間等を表示する装備が搭載される。機体内の各システムの状況を乗組員に表示する装置としてはエンジンの回転、温度、振動を表示する計器、燃料の量、油圧の圧力・量、電源の電圧・電流や操縦舵面の角度状態を示す計器類を装備する。近年は、従来型の単独計器は大型のディスプレイ表示器に統合表示され、かつカラー表示によりパイロットへの視認性が向上している。i. 電気系統電気系統としておもに電源系統、照明系統があり、また電気を扱うさまざまな系統がある。電源装置はエンジンの回転を利用した発電機と独立したバッテリーが電力源となり、また電源の交流を直流に、あるいは直流を交流に変換する装置や各電力源の切換装置からなり、信頼性の高い制御と配電にて各電力利用機器に交流115V,400Hz および直流28V の電力を供給する。照明系統には大別して外部と内部照明系統がある。外部照明としては自機の位置、飛行方向を示す赤、青および白色の航法灯があり、ストロボライトのような衝突防止灯、さらには着陸灯や地上走行時の走行灯が装備される。内部照明としてはコックピット内や計器類、また客室内の照明灯および非常時の脱出灯が装備される。電源については特に電源の品質、信頼性、照明については飛行に関する照射角度、輝度等それぞれ航空規定を満足するように装備される。j. 室内装備航空機の客室は乗客が快適に過ごせ、手荷物が効率よく収納できるよう座席とオーバヘッドビンが配置されている。また、洗面所や食事等乗客へのサービスの準備をするギャレーが運行に合わせ装備されている。内張りや座席のクッションには軽量かつ難燃性の材料が使用されており、外板と内張りのすきまには防音・断熱材が詰められている。最近では座席ごとに液晶画面が装備されるなど、電子娯楽設備が急速に充実している。k. 電子系統(avionics system)通称アビオニクス系統として電波を用いた地上局・空港および機体間の音声交信装置や気象レーダ装置、地上との電波高度、地上局および人工衛星の電波を利用した位置・方位の測定装置が装備される。電子機器の発達でコンパクト化・高性能化された機器として、地形および他機の位置情報を検出し衝突を防止する回避装置や、速度、高度、位置情報および自機のエンジン、操縦装置等と連接した自動操縦装置がある。電子系統の代表的な技術として、以下のものが挙げられる。. FMS(flight management system)慣性航法装置(INS : inertial navigation system)と性能管理装置(PMS : per-formance management system)を一つにまとめ、その機能と精度を向上させたもの。機体重量、外気温度、気圧等の情報を基に、最も経済的な上昇速度、巡航速度・高度、最適降下開始地点等を自動的に算出する。. TCAS (traffic alert and collision avoidance system)衝突の危険の生じる可能性がある航空機の接近を検知し、パイロットに対して、その航空機の位置情報や衝突回避のための上下方向回避警告を提供する。l. 火災報知系統、消火系統火災報知系統は、一般的に火災加熱感知系統、煙感知系統、および警報系統から構成されている。消火系統は、エンジン火災、補助動力装置火災、貨物室火災などに対し、ハロン等を貯気した消火ボトルが装備されている。m. 保命装備高高度を飛行する航空機において、酸素装置は与圧装置の故障、機体の破損に対して、乗員、乗客に地上とほぼ同等の濃度の酸素を供給する装置である。酸素源としては、貯蔵する方式、化学反応方式および大気中の酸素を分離、濃縮する方式がある。
