# 4・4・4 設計製造技術

> **章**: 第4章 航空機
> **節**: 4・2 節の各性能を計算する機能に空力モデルや重量モデルを組合わ

a. 設計技術航空機の開発の流れを図4・41に示す。航空機の開発は、運用要求に基づいて設計の各段階でシステムや機器の機能設定を行い、地上における各種試験や飛行試験を通じてその機能を確認して顧客に納入し運用を支援していく一連の活動である。設計は航空機の機能や性能を定義し、それを実現するよう機体の仕様を設定する作業であり、大別して構想設計、基本設計、詳細設計のフェーズに分かれる。最近ではコストの低減、スケジュールの短縮をねらって図4・42に示すようなコンカレントエンジニアリング(concurrent engi-neering)の手法を開発に適用することが多い。これは設計部門からサービス部門まで開発に携わる全員が設計データを共有して同時に作業を進めていく方法である。共有されるデータは、コンピュータを用いて定義される三次元の機体および部品形状(ディジタルモックアップ)で、重量や材質等の部品固有の情報も盛り込まれている。これにより、従来下流工程とされてきた製造や顧客サービスの情報も設計に反映が容易となり、製造に入る前の事前検証が可能となった。したがって、製造工程以下の不具合をあらかじめ防止でき、設計変更などの戻り作業を極限まで抑え込んだコスト、スケジュールの最適化が図られる。この手法は近年のEWS(engineer-ing work station)およびコンピュータネットワーク技術の進歩により可能となった。以下に設計の各フェーズについて概説する。. 構想設計ここでは機能・性能の要求に従い、まず設計基準を設定し、次に目標コスト、スケジュールを勘案して機体諸元、構造様式、システム主要構成等の概略の機体仕様を設定する。そのために機能・性能、コスト、スケジュールを評価対象として空力、構造、各システムにまたがる広範なトレードオフスタディが実図4・41 航空機開発の流れ6―121施される。これらの検討に資するため、風洞試験や基礎的な構造試験等一部の開発試験もこのフェーズにスタートする。機体三面図、機体構造線図、システム構想に加えて開発全体日程、経費計画等が作られる。. 基本設計構想設計を受けて100万点ともいわれる部品すべてにわたる機体細部の仕様設定を行う。詳細な空力形状と、それに基づく性能/特性/空力荷重の検討を実施し、構造および各システムの細部仕様を設定するフェーズである。飛行シミュレーション強度計算に供する空力データ取得のための風洞試験、飛行特性を調べ飛行制御則を設定するためのシミュレーション試験、機器の操作性、整備性等を検討するための木型試験等が実施される。各種試験と並行して数値シミュレーションによる解析も幅広く行われる。CFD(computational fluid dynamics)による空力解析や有限要素法による構造解析が代表的な例であるが、図4・42に示す共有データを基に解析が実施され、結果は再び共有データに反映されて更新されていく。通常は主契約会社(プライムメーカ)のもとに構造、装備、エンジン等の各協力会社(パートナ)が参集して共同作業を行うため、計画図フェーズ(joint definition phase)とも呼ばれる。基本設計の主要の成果物は機体計画図であるが、コンカレントエンジニアリングによる最近の開発では二次元の図面としてではなく、三次元設計データとして定義されるようになった。図4・43は2001年に就航したボンバルディア社の70人乗り民間機CRJ-700の後部胴体の計画図で、構造および各系統装備の全部品が三次元の電子データとして定義されている。. 詳細設計基本設計を受けて製造用の図面(drawing)を作成する作業である。各製造工程に適した加工方法や組立方法に関する情報が製造図面に盛り込まれる。製造に必要なツーリング(治工具、tooling)の設計も併せて行われる。操縦系統等の各システムに関するソフトウェアの整合性を確認するシステムリグ試験もこのフェーズにスタートとする。通常は半年から1年の間に全部品の図面を作成するため、全開発期間を通じて最も多くの設計者を必要とするフェーズである。詳細設計が終了すれば実機の部品加工、組立へと進み、並行して静強度試験、疲労強度試験が行われ、初飛行、飛行試験、初号機納入へとフェーズが進んでいく。b. 製造技術航空機は代表的な総合組立工業製品である。航空機は膨大な種類と量の部品から構成されているので、効率的な組立作業を行うためいくつかの単位モジュールに分割し、モジュールごとの組立を行ったあとで最終組立が行われる。技術要求に従いさまざまな材料の部品加工を行い、構造組立作業では組立ジグを使用して各種の部品を位置決めし、相互の部品の穴あけ、ファスニング作業を行う。その後、機体の気密シール、表面処理、機体内部塗装作業が行われる。動力装置、降着装置や操縦、空調、計器類など機器類のぎ装品が、管、索、電線などとともに機体構造に組込まれる。各系統の調整を行った後、機能試験として装備品単体機能試験、全機系統機能試験、さらに全機塗装作業や飛行試験による最終的な確認を実施し、完成機となる。各製造工程では、技術要求(図面、スペック)より展開された作業指示書に従い図4・42 コンカレントエンジニアリング図4・43 計画図の例(CRJ-700後部胴体)6―122交通機械図4・44 航空機製造工程の流れ作業が行われる(図4・44)。. ツーリングツーリング(治工具)は、航空機を製造するために必要となる道具であり、航空機の精度、品質を保証し図面要求を満たすために必須なものである。治工具は、一般的に、各製品の製造に共通として使用されるはん用治工具と、特定製品の製造のみに使用される専用治工具とに区分される。治工具の種類には、ある製品や構成する部品の製作およびその組立、調整などに使用されるジグ、工具や検査具、測定具、機能試験装置、運転装置、運搬具、作業台などがあり、航空機の生産時期に合わせて設計・製作される。. 部品加工航空機の部品工作法の基本には、板金加工、機械加工、接着複合材成形加工、特殊加工、熱処理、溶接、表面処理・塗装、ワイヤハーネス製作加工などがある。板金加工は、アルミニウム合金、チタニウム合金などの成形加工技術が重要であり、ゴムプレス成形、ストレッチング成形、セコスタンプ成形やホットサイジング、超塑性成形、ショットピーンフォーミングなどの成形法が適用される。航空機の機械加工は、軽量化や比強度向上のため一体構造の薄肉加工と五軸制御ねじれ面加工、チタニウム合金等の難削材加工などの特徴を有している。そのために航空機特有の機能をもつ工作機械が必要となる。ねじれ面や角度加工を必要とする部品など複雑な形状の部品加工を目的としたプロファイラ、主翼外板などの加工用のスキンミラー、長尺物のけたフランジを加工するスパーミラーなどの工作機械が使用される。接着複合材成形加工は、良好な接着品質を得るために被接着面を活性化する接着前処理工程を必要とする金属接着加工法と、複合材料の積層作業が中心となる複合材成形加工法に大別される。積層には自動レイアップマシン、成形硬化にはオートクレーブが使われる。また、本成形加工に不可欠な積層内部の検査のためX 線、超音波、磁粉探傷などの非破壊検査が行われる。このほかにハニカムコア加工法などが必要となる。特殊加工としては、金属の接触面に母材より融点の低い金属を溶融添加して接合するろう付け法、母材接合界面の原子の拡散現象を利用する拡散接合法、ロストワックス法などの精密鋳造加工法、ワイヤカットなどによる放電加工法、化学的な侵食により金属を削り落とすケミカルミーリング加工法がある。熱処理は、加熱・冷却の操作により目的とする材料の諸性質を得るために行う処理であり、鉄鋼材料、アルミニウム合金、ステンレス鋼などに使用目的に応じた処理が適用されている。溶接加工法には、TIG(tungsteninert gas)溶接、電子ビーム溶接、電気抵抗溶接などが用いられる。加工の完了した部品は、適切な検査を実施した後、耐食性や耐摩耗性などの目的のため、アルミニウム合金や鋼などの材料の種類や目的別により各種の表面処理や塗装を実施し、完成品となる。ワイヤハーネス製作加工は、電線加工(測長、切断、印字)、端末処置、コネクタ組立を行った後、荷姿図に合わせて電線のハーネス組立を行う。さらに、各電線が図面どおりに正しく結線されていることの検査を、自動回路試験装置などを使用して導通/絶縁確認を行い、ワイヤハーネス製作の完成となる。. 構造組立構造組立作業には航空機の外形を形づくるパネル組立や、強度部材のフレーム組立、また大きなものでは旅客機の胴体組立などがあり、その主要作業は位置決め、穴あけ、ファスニング、シーリングなどがある。航空機の厳しい空力特性や強度特性などにより位置決め作業には、高い精度が要求されるが、これを満足するため組立ジグによる方法と部品穴基準による方法がとられる。穴あけ作業は、入念な品質管理のもとに組立ジグのガイドブッシングや単体部品にあらかじめあけられた案内穴を利用して、正規サイズの穴あけを行う。特に仕上精度が要求される精密穴は、リーマ仕上げが行われる。ファスニング作業は、使用目的によってリベット取付作業、ボルト・スクリュー・ナットの取付け、ブラインドファスナの取付け、ハイロックボルトやテーパロックファスナなどの特殊ファスナの取付けを行う。シーリング作業は、気密性や水密性を保持するためのプレッシャシーリング、作動油による腐食防止などのために行う周辺シーリング、機体内部の空間を燃料タンクとして有効に活用するためのインテグラル燃料タンクシーリングなどに大別できる。. ぎ装品取付けぎ装品とは動力装置、降着装置、油圧、操縦、空調、与圧、防水、潤滑油、空気、計器、電気、電子、保命などの機器類とその関連部品の総称である。これらの取付位置はコックピット、エンジンナセル、客室床下などの局部的に機体内の狭い空間に集中することが多く、機体構造組立の小組立段階から順次、配管、索、電線などとともに機体構造に組込まれる。機体構造、ぎ装品の取付けが完了したのち、各系統としての所定の性能を発揮すべく調整を行う。調整に先立ち各機能部品の動力源となる系統の漏えい試験、フラッシング、電気電子回路の導通絶縁試験、回路確認などを実施し、機体外から、油圧、電気を供給し可動状態にする。例えば操縦系統は、操縦かん、ペダルと各動翼の作動の関係を、機械的リンケージのケーブルとロッドの長さにより調整したり、電気的に中立位置を調整したりする。. 機能試験(functional test)航空機の製造過程で機能部品および、それらを組合わせた各系統の機能、性能を確認するための試験であり、装備品単体機能試験、全機系統機能試験に大別される。装備品単体機能試験は、油圧、電気、計器、電子などの機能装備品を、単独または系統的に組合わせ、試験室(テストベンチ)で実施する試験である。全機系統機能試験は、組立ラインで機外の油圧源、電源を使用して行う試験と、整備ライン(屋外)で機体自身の各動力を作動させた状態で行う試験に大別される。従来、屋外作業であったエンジン運転も、作業環境の確保や騒音対策のため消音装置を備えた特殊な建物の中で実施するようになってきている。これらの機能試験に使用される装置や計測器材は、精度保証のため定期点検が行われ、一連の試験を少人数で実施できるよう自動化が進められている。. 飛行試験(flight test)飛行試験は、飛行状態での各系統機能および各系統相互間の機能を確認するための試験であり、専門のテストパイロットにより機体に装備された計器類を基に実施される。飛行試験で機体としての機能や性能を最終的に確認し、完成機となる。c. 製造技術の最近の動向航空機工業がもつ先端技術的性格から、必要性能や機能を達成するのに大きな研究努力が払われるため、材料や構造・ぎ装様式の進歩が著しい。このような新材料や新しい構造・ぎ装様式が導入されれば、既存の設備や加工技術では製造不可となり、厳しい技術要求を満たすために機体の開発と並行して、新しい加工技術の研究開発が行われている。さらに、より複雑で高性能化した航空機の開発を、高品質を保ちながら開発費用や期間を抑制するための研究や、IT 技術を駆使した研究も盛んに行われている。フロントローディング(FL: front loading)により開発の初期段階に各種資源を投入することで後々の不具合を未然に防いだり、コンカレントエンジニアリング(CE: concurrent engi-neering)により、三次元設計ツールとIT 技術を用いて、研究、設計、工作、資材、品質保証、サービスの全部門で同時に同一のデータ情報を共有することで、品質の良い航空機を短期間に製造する方法が研究されてきている。工作部門では、FLやCEにより生産事前検証の徹底(例:組立性、整備性の検討)と迅速生産展開(例:NCプログラム作成の自動化、作業指示書の自動作成)が可能となる。ツーリングでは、三次元設計データを基に治工具図面を設計し、干渉検証、強度計算に展開している。部品加工では、実機で行っていた配管や電線の成形、配線データの作成を三次元設計データから自動作成している。また、板金加工では三次元設計データをインクリメンタル成形、フレキシブル引張成形、レーザトリムなどの成形技術に適用しており、機械加工では、自動作成したN/C加工プログラムのカッタ軌跡の適否を検証している。また、三次元レーザ測定器を部品の検査工程に組込み、成形中の部品形状を測定しながら、成形プログラムに自動フィードバック反映することにも三次元設計データが用いられている。また、部品点数を削減して部品成形後の組立工数を減らす効果のある一体成形、積層の不要なインゼクションモールド加工を使った複合材加工、高速切削加工に対応したマシニングセンタを活用した機械加工など部品を安価に早く作る新加工法も研究されている。構造組立やぎ装の現場ではNC機械やロボットの導入が進み、穴あけやびょう打ちなどにロボット(オートマチックリベッタ)が使用されている。ここでも設計部門や生産技術部門の三次元データの直接利用によるオフラインティーチングでの時間の削減がポイントとなっている。また、組立作業性向上や組立ジグ費削減のため単品部品加工時に三次元設計データを基にして高精度の基準穴を設け、組立作業は組立ジグではなく部品の穴基準で行うことも取り入れられてきた。サブ組立では、三次元データを直接利用して、専用ジグだけでなくフレキシブルはん用ジグによりロボットやNC機械で行う技術が開発されており、今後の発展が期待される。機能試験では、機体上のシステムのディジタル化に伴い、整備性向上のために機体に装備されるようになった機能を製造過程にも利用するケースが増えてきている。システムをディジタル化すると、エンジンのモニタリングと不具合兆候の検出、操縦系統の故障検出などのシステム作動状態の監視および記録が容易になり、これらの機能が機体上のシステムに付加(ビルトイン)されている。将来的には、よりいっそうのディジタル化が進み、試験装置の統合、省人化がさらに拡大すると考えられる。
