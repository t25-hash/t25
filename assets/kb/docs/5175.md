# 6・5・2 鉛酸蓄電池（lead-acid battery）

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・5 蓄電装置

鉛酸蓄電池（鉛蓄電池，鉛酸電池，鉛電池ともいう）は，エンジ
ン始動用の電源として長年にわたってその座を他の電池に明け渡す
ことなく大量に普及してきた．自動車の寿命に比較して短い寿命し
か有していないにもかかわらず，その安価なコストにより消耗部品
として認知され，エンジン始動用途としては不動の地位を築いてい
る．しかしながら，HEV用の蓄電装置としては，その寿命の短さ
に加え，充電状態の検知性や劣化状態の把握が困難であることが実
用化を難しくしている．ただし，その安価さは魅力的であり，寿命
の改良や状態検知の技術開発が精力的に行われており，42V電池
での簡易型ハイブリッドシステムも一部市販されるに至っている．
簡易型ハイブリッドシステムにおいても，回生入力に対する受入れ
性が要求されるため，充電状態は中間SOCに維持することが必要
となる．そのため，一般の始動用鉛酸蓄電池とは異なり，密閉型鉛
蓄電池（valve regulated lead- acid battery）が採用されている．
