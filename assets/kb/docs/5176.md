# 4・5・3 航空機の安全性

> **章**: 第4章 航空機

a.定期航空の事故 世界の定期航空の乗客死亡事故発生率
は，近年は改善傾向が鈍化しているものの（表4･4)，現代の定期
航空の安全性は，他の交通機関等と比較して遜色のない水準に達し
ている（表4･5）．事故の主原因は，乗員の判断・操作の誤り等が
過半を占め，発生時の運航区分は，離陸上昇・進入着陸に集中して
いる（表4･6)．
b.小型機の事故 小型機の事故発生率は，大型機による定期
航空に比較すると，1けた以上高くなっている（表4･7)．事故の原
因は，定期航空と同様に，乗員によるものが多い．
c .安全対策 現代の民間航空機の設計においては，すべての
重要システム・構造等について故障・破壊モード解析等が行われ，
重大な結果が発生する確率は極微となるよう設計・整備上の配慮が
なされ，実運航経験からの見直しも絶えず行われている．CFIT
（controlled flight into terrain，航空機の機能は正常であるまま地
表等に激突する事故形態）や空中衝突のような繰返し発生している
重大な事故に対しては，GPWS（ground proximity warning sys-
tem，対地接近警報装置），EGPWS（enhanced ground proximity
warning system，機能強化型GPWS），TCAS（traﬃc alert and
collision avoidance system，衝突防止装置）等の特別の安全装置
が開発され，その装備義務化が図られている．
また，事故原因の大半を占めるに至っている乗員の過誤等のヒュ
ーマンファクタ（human factor）については，人間中心の自動化
の原則の開発，CRM（crew resource management，クルーリソ
ースマネジメント：安全で効率的な運航の達成のためすべての利用
可能な人的リソース，情報等を効果的に活用するための訓練）の実
施などが行われている．
さらに，事故のみならず事故に至らないインシデントを調査し，
それらの情報を航空関係者で共有し，事故防止に役立てようとする
ことなど，安全情報の共有化の促進も図られている．
d.航空保安 ハイジャック，航空機爆破等の航空機に対する
不法行為は，近年では，航空事故とともに民間航空の安全に対する
重大な脅威となっており，1980年代には，旅客機が空中で爆破，
撃墜される事件が多発し，多くの人命が失われた．ハイジャック等
を防止するために，国際条約が締結され，空港における保安検査等
が強化されてきたが，2001年9月11日にアメリカで発生した同時
多発テロは死者が約3000人とみられる未曾有の大惨事となり，民
間航空に対するテロの根絶のために，航空保安対策の徹底的強化が
国際的に図られていくこととなった．
