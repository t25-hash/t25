# 5・1・1 物流における船舶の役割

> **章**: 第5章 船舶
> **節**: 5・1 船舶一般

a. 物流における船舶の位置付け国際貿易を支える輸送機関としての船舶の位置付けは極めて重要で、総輸送重量の99%は船舶によって運ばれている。各国の国内物流においては、旅客部門では自動車が主流であるが、貨物部門では海上輸送も重要な役割を果たしている(図5・1)。国際物流においては、乾貨物輸送に関してコンテナ化が進み、特に経済発展の急なアジアからアメリカ、ヨーロッパへの製品輸送のコンテナ化が顕著である(図5・2)。b. 貿易における船舶の役割四方を海に囲まれ国土が狭いわが国では、多くの物資を海外に依存しなければならない(図5・3)。島国であるわが国においては輸出入は海上または空路に頼らざるを得ないが、貿易額、輸送重量ともに船舶の果たす役割は大きい(図5・4)。図5・1 各国国内物流における輸送機関の分担率図5・2 貨物輸送におけるコンテナ化の推移6―131c. 物流におけるエネルギー消費各種輸送機関による輸送効率をエネルギー消費の観点からとらえると、単位輸送力当たりの消費エネルギーは鉄道が最も優れている(図5・5)。また、各機関の輸送エネルギー総消費割合は概略自動車88%、船舶5%、鉄道2%、航空機5%である(図5・6)。
