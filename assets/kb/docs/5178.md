# 5・1・2 船舶の定義と種類

> **章**: 第5章 船舶
> **節**: 5・1 船舶一般

a. 船舶の定義法的には船舶の定義は存在しないが、社会通念上船舶とは、水上(中)において人または荷物を載せ、これらを運搬するための航行能力を有する構造物、と述べることができる。b. 船舶の種類船舶の分類は、用途、船型、航行区域、運航形態、支持方式、機関、推進方式、構造・材料、法規等によって分類される。通常は船舶として商船および特殊船を指すことが多い。表5・1に用途による船舶分類の概要を示す。船型による分類では、三島型船、平甲板船、遮浪甲板船、凹甲板船、全通船楼船、低船尾楼船、長船尾楼船に分類されるが、三島型図5・4 輸出入における船舶、航空機の分担図5・5 単位輸送力当たりの消費エネルギーの比較図5・3 エネルギー・食料の海外依存度図5・6 輸送機関別エネルギー総消費割合6―132表5・1 用途による船舶の分類船種船舶用途/貨物旅客船純客船旅客貨客船、フェリー旅客、貨物、自動車(定員:13名以上)一般貨物船一般貨物船、LO/LOコンテナ船、RO/ROコンテナ船雑貨、コンテナ船油槽船、ケミカルタンカー、LPG 船、LNG 船、プロダクトキャリヤ液状貨物(原油、薬品、石油製品、液化ガス)自動車専用船自動車冷凍(蔵)貨物船冷凍(蔵)貨物重量物運搬船重量物漁船トロール船、きんちゃく網漁船、はえなわ漁船等漁労カニ工船、サケ・マス工船加工冷凍(蔵)運搬船、活魚運搬船運搬特作業船引き船、砕氷船、救難船、消防船、ケーブル敷設船、自航しゅんせつ船、自航クレーン船殊調査船海洋調査船、気象観測船、水路測量船、漁業調査船船取締船巡視船、税関監査船、漁業取締船運搬船給油(水)船、渡船、はしけその他灯台船、水先船、練習船、検疫船、プレジャボート、ヨット(a) 三島型船(b) 平甲板船(c) 遮浪甲板船(d) 凹甲板船図5・7 船型による船舶の分類表5・2 航行区域の種類航行区域適用制限平水区域湖沼、河川および港内の水域および安全法により指定された特定の区域沿海区域陸岸より20海里の範囲内の水域および安全法で指定された特定の水域近海区域東経175°、南緯11°、東経94°、北緯63°の線によって囲まれた水域遠洋区域すべての水域限定沿海区域沿海の範囲内で、平水区域から当該船舶の最高速力で2時間以内に往復できる水域。この区域には、母港または母港を含む平水区域から当該船舶の最高速力で2時間の範囲内にある避難港まで、およびその避難港から片道1時間の範囲内の水域を含めることができる。船型は新造船には採択されなくなっている(図5・7)。各船型はそれぞれ用途、船の大きさと関係がある。航行区域による分類では、平水区域、沿海区域、近海区域、遠洋区域および限定沿海区域に分類されるが、遠洋区域を除いていずれも航行区域に制限がある(表5・2)。特殊貨物船鉱石専用船、穀物専用船、セメント運搬船、砂利運搬船、チップキャリヤ撒積貨物(鉱石、穀類、石炭、セメント、砂利、木材チップ)商図5・8 支持方式による分類図5・9船体主要寸法支持方式による分類では、主として浮力支持型(排水量型の単胴船、双胴船、半没水双胴船)、揚力支持型(水中翼船)、空気圧支持型(エアクッション船)があるが、これらの方式を複合的に組合わせた方式(滑走艇等)もある(図5・8)。支持方式は積載能力、航海速力、耐航性能等との関連で選ばれる。推進方式では、櫓(ろ)、櫂(かい)、風力、外車、水中プロペラ、空中プロペラ、ウォータジェットがある。ウォータジェット方式の中には超伝導電磁推進方式も含まれる。
