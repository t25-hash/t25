# 6・5・5 キャパシタ（capacitor )

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・5 蓄電装置

電気二重層キャパシタは，分極性電極と電解液界面に形成される
電気二重層と呼ばれる極めて薄い電化配列層の容量を利用したもの
である．電気二重層キャパシタの素子内の充放電要素は，正負極に
用いた活性炭の電極表面に形成されるイオン吸着層の吸脱着を利用
したものである（図6･74）．
最大の特徴は，電池のような化学反応を伴わないイオンの吸脱着
という物理現象のみを応用しているため，電荷移動において化学反
応の抵抗が伴わず，内部抵抗が電池に比較して温度の影響を受けに
くい点にある．また，化学反応を伴っていないため，電解液の分解
が発生しない電位範囲（電位窓という）に動作電圧をコントロール
できれば，長寿命が期待できる．一方，欠点としては電池などに比
較して大幅に蓄電できるエネルギーが小さいことであり，この蓄電
装置の冒頭で述べたような300W･ hもの最大入出力可能なエネル

しかしながら，低温出力重視であるが蓄電エネルギーを大量に必
要としないシステムや，あるいは鉛蓄電池とハイブリッドされた複
合電源システムにおいてはその特徴を有効に発揮できる可能性があ
り，その方面でのアプリケーションをねらった開発に焦点がおかれ
ている．
電解液としては，水溶液系電解液，有機溶媒電解液，無機固体電
解質，有機ポリマゲル電解質などが提案されているが，HEV用と
しては有機電解液を用いたタイプが多く開発されている．理由は，
キャパシタの耐電圧を高くできることにあり，電解液の種類にもよ
るが2.3～3.3V/cell程度の他動電圧が実現できる．有機電解液と
して具備すべき特徴としては，酸化側，還元側の分解電圧が高く，
広い電位窓をもつことと，広い温度範囲で高い電気伝導度をもつこ
とが要求される．電極材料は，それ自身は電気化学的には不活性で
理想分極性を示し，電子導電性に優れ，さらに比表面積が大きいこ
とが要件である．実用化された有機電解液系電気二重層キャパシタ
では，おもに，やしがら，コークスやピッチなどの天然由来物，フ
ェノールなどの合成樹脂を原料とする大表面積の活性炭材料が用い
られている．
