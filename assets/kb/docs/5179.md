# 5・1・3 船型と主要目

> **章**: 第5章 船舶
> **節**: 5・1 船舶一般

a. 主要目船体の全長、垂線間長、型幅、型深さ、型喫水を主要寸法と呼び、これに船型、船級、肥せき係数、満載排水量、軽貨重量、載貨重量、総トン数、主機関、航海速力等の船の種類、機能・性能を表す諸要目を加えて主要目と呼ぶ。. 全長(: length overall)船体の最前端から最後端までの水平距離(図5・9)。i」は改。水平距離(図5・9)。文字が増。型深さ(: moulded depth)船体中央において上減した場甲板ばりの舷側における上面(船側外板の内面と上甲板下面の交合は、2点)から基線(base line、キール上面)までの垂直距離(図5・9)。字アキで。ださい。b. 船舶の大きさ、トン数。 総トン数(GT : gross tonnage)船の内部の総容積100cf(立方フィート、1000/353m)を1tとして表したもので、船の大きさを表す指標として課税、通量の算定に用いられる。詳細は船舶積量測度規程により規定されている。総トン数には国際総トン数と総トン数があり、国際総トン数は、「船舶のトン数制度に関する国際条約(1966)」に定める計測方法によって算定され、国際航海に従事する船舶に適用される。. 純トン数(NT : net tonnage)純トン数は、総トン数から機関室、乗組員室、バラストタンク等の運航に直接関わる場所の容積を差引いた部分の容積を総トン数と同じ単位で表した大きさの指標である。旅客、貨物等を積載する場所の大きさを表し、利益に直接関係する指標であるから、種々の課税、手数料算定の基準として用いられる。. 排水量(Δ,: displacement)ある喫水状態における船の全重量を表し、船体が排除する水の重量に等しい。満載状態の排水量を満載重量、軽荷状態の排水量を軽荷重量と呼ぶ。. 載貨重量(DW : deadweight)満載重量と軽荷重量の差で表される重量で、貨物船の積載能力を表すのに用いられる。―1336図5・11 コンテナ船図5・12 ばら積船(ばら積貨物船、鉱石運搬船)図5・13 油槽船(タンカー)DW には貨物重量のほかに燃料油、水、油類、乗組員の重量を含む。DW の表現はメートル法単位のキロトン(KT)と英国式単位の英トン(LT)が用いられる。. 載貨容積(cargo capacity)載貨容積は全貨物倉の容積でmまたはcfで表す。ベール(bale)とグレーン(grain)があり、ベールは梱包(こんぽう)貨物を格納できる容積を表し、貨物倉内の突起物(肋骨、はり等)の内側までの容積、グレーンはばら積貨物積載容積を表し、貨物倉内部の総容積である。
