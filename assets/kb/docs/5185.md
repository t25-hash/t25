b.日産ハイブリッドシステム（Nissan Hybrid System）の構
成 本システムの構成を図6･105に示す．おもな構成要素はガソ
リンエンジンと駆動・回生用モータの二つの動力源，動力伝達経路
にCVTと電磁クラッチ，発電・エンジン始動用モータ，バッテ
リ，インバータである．
エンジンとモータはどちらもダイレクトにタイヤに動力を伝達す
る．モータは作動点を任意に選択でき，変速によるトルク増幅を利
用してモータを小型化するという考えから，CVTの上流に配置し
た．
車両の走行状態にかかわらずエンジンは独立に始動停止および発
電をするために，モータとエンジンの間には電磁クラッチを配置し
た．これはモータ走行および回生時にエンジンフリクションをカッ
トする機能ももつ．また，発電とエンジン始動を兼ねるモータはエ
ンジンフロント側に配置し，クランクシャフトからチェーンスプロ
ケットにより駆動する方式とした．
モータおよび発電機の駆動は効率に優れたインバータを用いた．
インバータを介して電力を供給，充電するバッテリは，高い出力で
充放電を繰返し行う場合でも効率が高く，床下に搭載することで居
室空間を確保するために，耐熱性も高いものが必要と考え，リチウ
ムイオン電池を選択した．各コンポーネントのスペックを表6･24
に示す．
c .制御システム 制御システム構成を図6･106に示す．六つ
の各ユニット用ECUは，それぞれのユニットごとの制御・状態監
視・診断を行う．統合ECU（system coordinate electronic con-
trol unit ）は，各ECUを取りまとめ，ハイブリッド自動車として
の性能を最大限に引き出すように各システムを統合制御する．統合
ECUのおもな機能としては，システムの起動停止，運転者の制駆
動力要求実現，電力の配分，動力源の配分，バッテリのエネルギー
マネジメント，システムの診断・監視・保護・フェイルセイフ等が
ある．また，制御システムを成立させるためには，各ECU間のデ
ータの授受を高速，かつ高い信頼性を確保した通信によって行う必
要がある．本制御システムでは，通信プロトコルとしてCAN
（control area network）を用いた．

d.まとめ
(１) 現行の同クラスガソリン車に対して，代表的市街地走行燃
費を2倍以上改善し，動力性能，居住空間は現行車と同レベルを確
保したハイブリッドシステムを開発した．
(２) パワートレイン部にはCVTを採用し，伝達動力を現行車
並みに確保した．またトルクコンバータ，前後進切換機構をモー
タ，電磁クラッチに置換え，CVTに内蔵・一体化することで，従
来車のエンジンルームへの適用を可能とした．
(３) パワー密度，エネルギー密度ともに優れたリチウムイオン
電池をハイブリッド用に適合し，高い信頼性と長寿命化も確保し
た．
(４) 動力性能，燃費性能，運転性等の諸性能を高次元で両立さ
せるために，統合ECUにより全体制御を行う，エネルギー最適制
御システムを開発した．
