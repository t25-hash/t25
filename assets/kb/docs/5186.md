# 6・8・3 乗用車の例3（ホンダシビックハイブリッド)

> **章**: 第6章 ハイブリッドシステムおよび燃料電池
> **節**: 6・8 ハイブリッド電気自動車の適用例

ここでは，ハイブリッドの適用例として低燃費とクリーン化を両
立した前輪駆動式小型乗用ハイブリッド自動車，シビックハイブリ
ッドについて紹介する（図6･107）．
a.基本構成 システムの基本構成は，フロントに搭載したエ
ンジンとCVTの間に永久磁石式交流同期モータを，リヤシートバ
ック背面にニッケル水素電池とモータ駆動ユニット（PCU:
power control unit ）を統合一体化した高圧電装ユニット（IPU:
intelligent power unit ）を配置している（図6･108）．
ハイブリッド形式はシリーズ方式とパラレル方式と両方を合わせ
たシリーズ・パラレル方式に大別される．シビック用ハイブリッド
システムは，二つの異なる動力源が機械的に駆動輪に接続されてい
るパラレル型ハイブリッドである．
エンジンを主動力源とし，モータを加速時の補助動力源と減速時
の回生による電気エネルギーの回収装置として利用している．ま
た，このモータは，通常のエンジン始動およびアイドリングストッ
プ状態からのエンジン再始動にも使用する．
通常のエンジンとトランスミッションで構成されたエンジンは低
負荷領域で運転されると，ポンピングロスなどの原因により燃料消
費率，すなわち燃費が悪化する．シビックのハイブリッドシステム
は，低・中速域においてモータトルクを追加することにより，通常
のガソリン車に搭載されているエンジンの排気量に対して排気量を
低減しても同じ出力性能を得ることができる．このように排気量を
低減することで，同一要求駆動力において相対的にエンジン側の負
荷割合が上がり，燃費が向上する．
シビックのハイブリッドシステムではつねにエンジンを運転しつ
つ，特殊な動力分配機構をもたずに，エンジンとモータおのおのの
発生駆動力の比率により1個のモータで燃費に優れたハイブリッド
パワープラントを構成している．
b.エンジン エンジンは直列4気筒1.3L SOHC，1気筒当
たり2プラグのエンジンを用い，低負荷領域の燃費向上を図った
（図6･109）．ハイブリッド車は減速時に回生する電気エネルギーを
走行時に使用することで，燃費を向上させる．回生量をさらに増加
させるために，減速時のエンジンフリクションを低減する必要があ
る．その手法として気筒休止機構を採用した．この機構は減速時に
給排気バルブを停止することによりポンピングロスを低減し，気筒
休止機構をもたないエンジンに対して回生量を50%増加させた．
c.モータ 永久磁石式交流同期モータをエンジンとCVTの
間に配置するためには，極力モータの幅を狭くする必要がある．ま
た，減速時の回生による電気エネルギー回収のためには，十分なト
ルクが必要である．幅狭化のために，ステータのコイル部にはコイ
ル間の渡り線部がコンパクトな巻線形式の突極集中巻を用いた．ま
た，コイルに電力を供給する配電部は樹脂を二重にモールド成形し
たバスリングによりコンパクト化を図った．トルク向上のために，
ステータコイルの占積率向上を中心に改良を施した．現状の手法で
ステータ巻線径を太くしてコイルを製作すると，隣接するコイルど
うしが接触し絶縁性確保が困難となる．インシュレータ形状のコイ
ル内側を薄肉化し，インシュレータの重なる部分を中央からずら
し，コイル巻太りが中央部で重ならない非対称のコイル形状とした
ことで，線径拡大とティース幅拡大を実現し，占積率を向上させた
（図6･110，6･111）．

ロータは，クランクシャフトに直接締結されるため，エンジンと
モータのトルクを同時に伝達する強度を必要とし，かつマグネット
のバックヨークとして，高い透磁率を確保する必要がある．これら
の相反する要求を満たすために，カーボン含有量が部位によりコン
トロール可能で，かつ異種金属の結合信頼性が高い焼結拡散接合を
用いたロータとした．バックヨーク部には軟磁性材，トルク伝達部
には高強度材を適用することにより，飽和磁束密度の向上と，容積
当たりの重量を削減できた（図6･112）．
以上のようなステータおよびロータの改良により，最大回生トル
クと実用域の平均効率の向上を実現した．
d.IPU IPUは内部に電力変換を行うPCUと高電圧を蓄え
るNi - MH電池，および，その他の高圧周辺機器と冷却部品で構
成されており，それらを総称してIPU（intelligent power unit ）
と呼んでいる．PCUはモータ駆動装置（PDU，パワードライブユ
ニット），12V供給用DC- DCコンバータ，冷却フィンで構成され
ている．またIPUは，高電圧に対応した大型のリレー，制御ユニ
ット（ECU）も内部に配置され，全構成部品を支えるケース本体
の機能と冷却ダクトの機能も確保している．冷却は，一つの冷却フ
ァンにより各機器とバッテリの冷却を同時に行っている（図6･
113）．
