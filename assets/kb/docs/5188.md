# 5・3・1 プロペラ(propeller)

> **章**: 第5章 船
> **節**: 5・3 推進器と動力伝達装置

船を推進する装置としてはオール,帆,外輪,スクリュー,ジェ6―143ット,ロケット等があるが,一般には普通の船舶に最も多用されるスクリュープロペラ(screw propeller)を単にプロペラといい,特殊な形状,構造のスクリュープロペラを含め,その他のものを特殊プロペラという.以下にスクリュープロペラの形状,構造等について説明する.スクリュープロペラの羽根数は,推進性能を主眼に,かつ,船体および軸系の諸振動を考慮して決めるが,一般には3~5枚である.巨大船や超高速貨物船では6枚羽根が使用されることもある.図示す.プロペラの回転方向は,一軸船では船尾から見て時計回りが普通で,これを右回りという.二軸船では右舷が右回りで左舷が左回りの組合せを外回り,これの反対を内回りといい,推進,操船性能等を考慮して選定する.羽根とボスは一体に鋳造されるのが普通であるが,別個に製作してボルトにより結合する組立型もあり,砕氷船などに採用される.材質としては強度,鋳造性,海水中における耐食性などの見地からアルミニウム青銅や高力黄銅などが常用され,氷海航行船などにはステンレス鋼も使用される.また,最近はキャビテーションによる船尾振動軽減の見地から,ハイスキュープロペラ(highly skewed propeller)のほうが多く用いられている.これは,羽根のスキュー角(skew angle,図でずらしてプロペラ起振力を減少させることをねらったプロペラである.自動車運搬船に用いられているハイスキュープロペラの例を図5・45に示す.
