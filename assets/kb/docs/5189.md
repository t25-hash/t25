# 5・3・2 プロペラの性能

> **章**: 第5章 船舶
> **節**: 5・44 )を大きくして、羽根が高伴流域を通過する時間を半径位置

一様流中のプロペラの性能は、揚力面理論などの計算によっても実用的な資料が得られるが、一般には、模型プロペラを用いて5・2・2・b・ii項の単独性能試験を行い、図5・46のような試験結果が得られる。ここで、試験結果の各無次元値は次のとおりである。プロペラは直径が大きく回転数の小さいほうが効率は高い。効率の実例については5・2節の表5・4を参照されたい。プロペラの設計(要目決定)は、翼形、展開面積、羽根数ごとに作られているチャートによるのが一般的で、図5・47に一例を示した。図5・47からは、最適なプロペラ直径(点)などを容易に知ることができる。
