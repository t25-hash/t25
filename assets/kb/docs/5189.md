# 6・1・1 ステッパ（stepper ）

a.ステッパの概要 半導体露光技術は，半導体デバイスプロ
セスの中心的な技術であり，半導体の高速化・高集積化・コストダ
ウンをけん引する最も重要な技術である．半導体プロセスは，シリ
コンウェーハ上に回路パターンを形成し，デバイスの要素を構成す
る技術である．また，近年では大型液晶テレビのパネル製造のプロ
セスにも応用され，高性能なパネル製造の最重要技術となってい
る．ここでは，その中で最も重要な役割を果たすステッパ（露光装
置）について，近年の技術動向と今後の方向を踏まえ解説する．
図6･1に半導体プロセスにおけるリソグラフィー技術 の位置

sure）技術により微細構造を形成する際に用いる，露光技術を中心
とする技術全般を指す．その際，微細構造パターン形成に用いる装
置を一般にステッパと呼ぶ．ステッパは，光，電子線，X線等の
光源を用い，ウェーハ（wafer ，半導体基板）に回路パターンを露
光する装置を指す．ステッパはその年代や用途とともに種々の方式
に分けられるが，(１)光を用いる方式，(２)電子線を用いる方式，
(３)X線を用いる方式に大きく分類される．図6･2に露光装置の
分類を示す ．
光露光装置（photo lithography）は，密着・近接露光装置と投
影露光装置に大きく分けられる．投影方式の装置は，その光学的処
理の違いから等倍方式と縮小方式に分けられるが，特に縮小方式が
近年の微細化の主流として発展してきた．また，露光に伴う装置の
動作方式も，当初のステップアンドリピート動作からスキャンアン
ドステップ動作に移り，今日のスキャンステッパに発展してきてい
る．

d.スキャンステッパ（step-and-repeat scanning stepper ）
.スキャンステッパの構成 1995年までのチップは，64
MBのDRAM（dynamic random access memory）で0.35m解
像度，チップサイズ10 ×20mmであったが，1998年には256MB
のDRAMの供給が始まり，0.25mの解像度で12×24mmのチ
ップサイズが想定されていた．このころより，露光エリア25×25
mmが必要とされ，レンズ口径では露光フィールドとして36mm
の大口径レンズが必要となった．しかし，最も高い精度が必要とさ
れるレンズ系の大口径化は，コスト面，製造面での現実性がなかっ
た．このために考案された方式がスキャンステッパ と呼ばれる
方式であり，現在の半導体露光装置の主流となっている．光源はお
もにKrFやArF等のエキシマレーザ光源が用いられている．
図6･9は，露光方式の違いを示している．スキャン方式では，レ
ンズのほぼ最大口径のスリット状ビームをレティクルに対してスキ
ャンし，同時に反対方向にウェーハステージが同期駆動されること
でパターンの転写が可能となる．一方で，従来の方式では同じサイ
ズのパターンを露光する際のレンズ口径が大きくなり，これが光学
系のコストや精度品質上の障害となる．
