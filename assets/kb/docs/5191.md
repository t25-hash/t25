a.カメラにおけるメカトロニクス カメラ（camera）の起
源は紀元前までさかのぼるという話もあるが，今日われわれが頭に
描くような形態のカメラが使われ始めたのは19世紀になってから
のことである．そして20世紀前半まで，カメラは精密機器の代名
詞の一つとして，もっぱらメカニクスにより構成されていた．その
後，電子技術が積極的に取り入れられ，カメラは大きな発展を遂げ
た．カメラの機能のうち，露光調節，焦点調節，フィルム送り，手
ぶれ補正などは，フィルムカメラの時代からメカトロニクス化によ
り著しく向上してきた．そして今日急速に普及してきた電子カメラ
では，さらに高機能化が進んでいる．

c .露光調節 露光調節とは，フィルムまたは電子カメラの場
合は撮像素子に適正な露光（exposure）を行わせることをいい，
カメラの中では最も古くからメカトロニクス化が進んだ部分の一つ
である．
図6･ 29 絞り制御機構
.測光 被写体の輝度を測定して適正な露光量を演算する
ことをいう．被写体からの光を直接あるいは撮影レンズを通して測
光素子（光電変換素子や電荷蓄積素子）に取込み，得られた電気信
号から被写体輝度を知り，さらに演算を行ってシャッタと絞りをど
う制御すべきかを決定する．画面内の輝度の平均を求める単純なも
のから，画面中央部を重点的に計るもの，画面内を複数のブロック
に分割してそれぞれのブロックの輝度を求め，複雑なアルゴリズム
によって最適な露光量を演算する評価測光方式も採用されている．
.絞り（aperture) 露光調節を行う手段の第一は絞りで
あり，通常は絞り羽根5～9枚を環状に配置して虹彩を形成し，リ
ングの回転とカム機構によりその口径を調節することで，レンズを
透過する光量を決定する．その一方で，絞りは被写界深度調節の機
能も併せ持っている．
メカトロニクス化以前の絞り制御機構では，使用者が撮影に先立
ち絞りリングなどであらかじめ設定した絞りにしか制御できなかっ
た．しかしメカトロニクス化により，カメラが演算で決定した絞り
に，自動的にしかも精度よく制御することが可能となった．
図6･29は一眼レフカメラに組込まれた絞り制御機構の一例であ
る．シャッタボタンが押下げられると，第1ソレノイドに通電され
て絞り連動レバーの係止が外され，付勢ばねにより絞り連動レバー
が下降し始める．絞り連動レバーはレンズの絞りレバーと係合して
いて，絞り連動レバーの動きに伴い，レンズの絞りが開放状態から
絞込み方向に動き始める．一方，絞り連動レバーの動きは扇形ギ
ヤ，増速ギヤを介してラチェットを回転させる．このラチェットの
回転がフォトインタラプタでモニタされ，制御しようとしている絞
りに対応する所定のパルス数がカウントされると，第2ソレノイド
に通電する．これにより係止レバーが動き出し，その係止部がラチ
ェットに係合して係止し，絞り連動レバーが停止する．このように
してレンズの絞りが制御すべき口径に制御される．
.シャッタ（shutter ) 露光調節を行う第2の手段はシャ
ッタである．絞りが露光の光量を制御するのに対し，シャッタは露
光時間を制御するものである．電子カメラでは，撮像素子が電子シ
ャッタ機能を有する場合は，これにより露光時間の制御を行う．電
子シャッタとは，光を電荷として蓄積する時間を制御して露光時間
を決定するもので，1/16000sのような高速シャッタスピードも可
能である．一方，撮像素子がこの電子シャッタ機能をもたない場合
は，フィルムカメラ同様に機械式シャッタが必要となる．機械式シ
ャッタは，撮影レンズの近傍または内部に配置されるレンズシャッ
タと，焦点面近傍に配置されるフォーカルプレーンシャッタに大別
される．フォーカルプレーンシャッタの場合は，シャッタスピード
の高速側限界は1/8000s程度となっている．シャッタも古くは時
計同様にメカニクスだけで構成されていたが，電子技術との融合に
より精度が向上し，高度な制御が可能となっている．

であり，撮影の可能性の拡大に大きく貢献している．

d.焦点調節（focusing) 焦点調節とは，フィルム面あるいは
撮像素子面にレンズの最良像面を結像させることであり，多くの場
図6･30一眼レフカメラのフォーカルプレーンシャッタ

も，先の露光調節と並んでカメラの基本的機能であるが，自動焦点
調節（AF:automatic focusing）を目指して早くからメカトロニ
クス化がなされてきた．AFが登場するまでの焦点調節は，例えば

しAFの著しい進化により，現在では速くて正確な焦点調節が可能
.AFの方式 AFの一般的な方式としては以下のものが
(ａ)コントラスト検出方式 露光前にレンズを動かし，像の
コントラストが最大になる位置を探す方式である．比較的簡単な機
構で実現できるため，小型化・製造コストの面で有利である．しか
し，合焦までに時間がかかるためシャッタボタンを押してから露光
までの時間が長くなり，また低コントラストや低輝度の被写体では
(ｂ)位相差検出方式 図6･32は位相差検出方式の原理を示
している．撮影レンズを通った光を撮像面と等価な位置で結像させ
た後，左右に分離して二つのCCD（charge coupled device）ライ
ンセンサ上に再結像させる．この二つの像のラインセンサ上での結
像位置を比較することで，ピントが撮像面の前側にきているのか後
ろ側にきているのか，さらにそのずれ量までも知ることができ，こ
れに基づいてレンズを駆動してピントを合わせることができる．こ
の方式の利点は，機械的駆動機構のないセンサによる検出だけで瞬
時にピント外れの方向と量がわかることで，特に速写性が要求され
る一眼レフカメラに適した方式である．
(２)アクティブ（能動的）方式
赤外線や超音波などをカメラから被写体に向かって照射し，それ
が被写体に反射して戻ってくる角度から三角測量の原理で被写体ま
での距離を求め，レンズを対応する位置に制御する方式である．

ッタスピードの限界は，35mmフィルムフォーマットでは1/焦点

しかし程度の差こそあれ，手ぶれにより夕暮れ，夜景，スポーツ，

し，露光調節，焦点調節と並んでカメラにとって大きな課題であっ
た手ぶれについても，カメラのメカトロニクス化により大きな前進
