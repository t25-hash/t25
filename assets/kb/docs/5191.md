# 5・3・4 プロペラのキャビテーション

> **章**: 第5章 船舶
> **節**: 5・44 )を大きくして、羽根が高伴流域を通過する時間を半径位置

プロペラの性能でキャビテーション(cavitation)の影響は大変重要である。一般商船では、軍用の高速艇のように推力低下を伴う激しいキャビテーションが発生しないようにプロペラを設計する。しかし効率に影響はなくても、キャビテーション気泡の消滅による衝撃圧でプロペラ翼面を浸食、破損し、騒音、振動の原因となる場合が多い。キャビテーションについては理論的な研究のほかにキャビテーション水槽(高速の減圧回流水槽)における模型プロペラの実験的研究も活発である。船尾流れのような不均一流中で発生するキャビテーションが最も有害な影響を与えるので、プロペラと船体を組合わせた実験や、金網等で船尾流れを模擬した中でのプロペラキャビテーション実験が行われている。
