# 6・1・4 DVD装置

> **章**: 第6章 メカトロニクスの応用展開
> **節**: 6・1 精密メカトロニクス

DVD（digital versatile disk）は，一層4.7GBの容量をもつ直
径120mmの光ディスク（optical disk）である．こうした光ディ
スクは，1972年のレーザビジョン発表などを契機に，1980年代に
入って盛んに研究開発が始められた ．1982年のCD（com-
pact disk）商品化およびその普及の時期を経て，DVDが1995年
にSD規格として提案される．そして現在では，DVDは再生専用
型，追記型，記録型など多様な種類の媒体でDVDファミリーを形
成している ．この発展の歴史は，LD（laser diode）の単波長化
の歴史と歩みを同じくしている．半導体レーザの開発の歴史とCD
やDVDの開発フェーズの相関をまとめたのが図6･37 である．
830nmの波長のレーザ（laser ）が室温で発振した数年後に780
nm波長対応のCDが開発され，600nm台のレーザの室温発振に
よってDVDの開発が行われている図式である．また近年発表され
ている400nm台の青紫色レーザの開発により，次世代DVDと呼
ばれる領域の光ディスクも開発・製品化されている．DVDに代表
される光ディスクは，記憶媒体に求められる省スペースを実現し，
可搬媒体である利便性をもつ．特に再生型のDVDは，コンテンツ
供給用のメディアとして，テープメディアよりも安価に製造できる
利点を生かし大きな市場を形成している．また記録型のDVDは，
テープメディアにはないアクセス性能を実現して普及しつつあり，
DVDは複数の規格でさまざまな使用用途にこたえている．
DVDの媒体は，0.6mmの基板を2枚張り合わせた構造である．
DVD装置は，650nmの赤色レーザをこの媒体の情報記録面に集
光し，集光スポットの戻り光から情報を読み取る装置である．情報
の記録が可能なDVD装置では，集光したレーザ光によって光ディ
スクの局部加熱を行って情報を記録する．このため，再生・記録さ
れる情報の密度を高くするには，集光される光学スポット径を小さ
くする必要がある．スポットサイズがλ/ NA（λ：レーザ波長）に
図6･ 37半導体レーザと光ディスクの歴史（CW:continuous wave)
比例する関係から，レーザ波長を短くする，またはNAを高くと
る必要がある．このため，より短い波長のレーザの開発が，より高
密度な光ディスクの開発を後押ししてきた．一方，この再生・記録
動作を行う光スポットは，数十nmのオーダで位置決めされる．
DVD- RAMの位置決めサーボ系に求められる仕様を，参考として
表6･3に示す．こうした広帯域の高精度位置決め（high accuracy
positioning）は，共振周波数の高いアクチュエータ開発やディジ
タルサーボの採用など位置決め機構・サーボ技術や，対物レンズで
の絞り角に相当するNAを高くする光学系の設計技術，再生信号
を処理する信号処理技術，位置決め誤差検出方式などの進化によっ
て実現されている．また量産装置であるDVD装置は，機構や回路
のばらつきを吸収する自動調整などの補償技術が不可欠で，多くの
メカトロニクス技術の集大成としてDVD装置のシステムは成り立
っているのである．本項では，このDVDにおけるメカトロニクス
技術の応用について，機構，位置決めサーボ，位置決め誤差検出方
式の観点から述べる．

c.スピンドルモータの機構と制御 スピンドルモータ（spin-
dle motor ）は，DVDなどのディスクを把持して所定の回転数で
回転させるモータである．最近ではブラシレスモータがおもに用い
られ，DVDの種類に応じて，回転角速度一定または回転線速一定
となるように制御される．1倍速で回転させる場合の回転数は，
DVD- RAMの場合が最も速く46Hz程度である．しかし実際には
高倍速での再生・記録が行われ，スピンドルモータの回転数として
は100Hz程度の回転制御が実現されている．このように高速回転
されるスピンドルモータでは，その回転トルクが小さいと所望の回
転数に整定するまでの時間がかかることになり，結果としてDVD
装置のアクセスタイムが遅くなる．光学スポットが形成される半径
位置に応じて回転数を変化させる場合，この整定時間が問題となる
スピンドルモータに光ディスクを装着する方法は，磁石によって
光ディスクを取付台座に挟み込むマグネットチャックが一般的であ
るが，この装着時にディスクの中心とスピンドルモータの回転中心
の間に偏差が発生する．さらにディスク規格として，ディスクの中
心と情報記録トラックの中心は±30～50mの偏心をもつことが許
されており，トータルの偏心量は±100mを超える値となる．こ
の偏心は，スピンドルモータが回転することによって情報記録トラ
ックへの位置決め外乱として作用する．同様にディスク規格は，デ
ィスク自体の反り量として±300m程度の変動を許している．偏
心同様，この反りやスピンドルモータ装着時のディスク傾きなど
は，スピンドルモータの回転によって面振れ外乱となる．また，デ
ィスク自体の偏重心によって，スピンドルモータの回転に必要な回
転トルクは変動する．こうした外乱成分は，スピンドルモータを安
定に高速回転させることを妨げるだけでなく，位置決め制御に大き
な影響を与えることになる．このような外乱の影響を抑えて，スピ
ンドルモータは安定した回転制御を実現するように設計される．軸
受も高速回転に対応し，安定かつ機械ロスの少ない回転を実現する
ために，流体軸受が採用され始めている．DVD装置のAV
（audio visual ）機器への応用やPCへの搭載が進むにつれて，静粛
性も重視され始めており，こうした軸受の開発が進んでいる．
このスピンドルモータの回転制御は，速度制御によって実現され
る．速度制御のための回転速度検出は，以下の2方式に分別され
る．
(１)スピンドルモータの回転エンコーダによる速度検出
(２)ディスクからの再生情報による速度検出
スピンドルモータに回転エンコーダが取付けられている場合，回
転角速度一定制御［CAV（constant angular velocity）制御］は，
このエンコーダ出力を用いて制御される．回転エンコーダはホール
素子を用いた磁気エンコーダで，回転角に応じて正弦波状の信号が
エンコーダから出力される．この正弦波状の信号は2値化され，パ
ルス状の信号となって，このパルスの立上りに対してPLL（phase
locked loop）制御がかけられる．このPLL制御は，次のような
AFC制御（auto frequency control ）とAPC制御（auto phase
control ）によって実現される．AFC制御は，所定の回転周波数が
目標回転周期として与えられると，パルスの立上り間隔が測定さ
れ，目標回転周期との周期誤差がフィードバックされる制御であ
る．さらに，そのパルス立上り位相の誤差成分までフィードバック
するのがAPC制御で，このAFC制御とAPC制御によってスピ
ンドルモータは回転制御される．また最近では，エンコーダを用い
ずに，モータドライバがモータ逆起電力などから回転周期を推定し
てエンコーダに相当する信号を生成する方式が主流となっている．
ディスクからの再生情報による回転制御は，同様のPLL制御で
あるが，ディスクに埋め込まれた回転同期信号を用いる点が異な
る．回転線速一定制御［CLV（constant linear velocity）制御］
を行う場合，光学スポットが形成される半径位置に応じて目標とす
べき回転周波数は変化する．この変化をスムーズに行うために，す
でにディスクに埋め込まれた同期信号を用いたPLL制御が行われ
る．
例えばDVD-Rでは，情報トラックが蛇行するウォブルトラッ
クという構造を採用している．このウォブル周波数は140kHzで，
トラック位置決めには影響を与えない高い周波数であり，トラック
誤差信号をBPF（band pass filter ）でフィルタ処理することによ
って抽出できる．抽出されたウォブル信号はディスクの回転同期信
号として利用でき，半径位置に応じたCLV制御は，この信号を用
いて回転エンコーダ信号と同様なPLL制御で実現される．また
DVD- ROMの再生信号に含まれる情報の区切りを表す信号も，同
様に回転同期信号であり，半径位置に応じたCLV制御に用いられ
る．
d.フォーカストラッキングアクチュエータの機構と制御 フ
ォーカストラッキングアクチュエータは，光学スポットを精密位置
決めするアクチュエータである．DVD装置の光学ヘッドは，半導
体レーザからの出射光をコリメートレンズで平行光として，その平
行光を立上げミラーによってDVDの情報記録面に垂直入射させ
る．このとき，DVDの情報記録面上に適切な光学スポットが形成
されるように対物レンズを位置決めするのが，このアクチュエータ
の役割である．
DVDの場合，DVDの表面から0.6mmの位置に形成された情
報記録面に焦点が合うように，ディスク面法線方向に対物レンズが
位置決めされる．これがフォーカス位置決めである．また情報記録
面には，らせん状または同心円状に情報トラックが設けられてお
り，この情報トラックの中心に光学スポットが形成されるように，
半径方向に位置決めされる．これがトラッキング位置決めである．
従来，このフォーカストラッキングアクチュエータとしては，光学
ビームを振ってトラッキング位置決めを実現するタイプのミラーア
クチュエータ，フォーカス方向の位置決めのみを実現する対物レン
ズを平行板ばねで支持するアクチュエータ，トラッキング方向とフ
ォーカス方向の二軸への対物レンズの駆動位置決めを可能にした二
軸の対物レンズアクチュエータなどが開発されてきた．この二軸対
物レンズアクチュエータのトラッキング位置決めは，図6･41のよ
うに対物レンズをトラッキング方向に移動させることにより，光学
スポットが変位する原理 を用いている．現在のDVD装置では，
機構がコンパクトであるため，この二軸対物レンズアクチュエータ
がおもに採用されている．
二軸対物レンズアクチュエータも，対物レンズを保持する対物レ
ンズ保持体の支持方法によって，複数のタイプが開発されてきた．
対物レンズ保持体がヒンジ機構で支持されるアクチュエータ，軸し
ゅう動型アクチュエータ（shaft -guided lens actuator ），ワイヤ支
持型アクチュエータ（wire-suspended lens actuator ）などであ
る．現在最も主流となっているのが，ワイヤ支持型の二軸対物レン
ズアクチュエータである．

軸しゅう動型二軸対物レンズアクチュエータは，軸に対してしゅ

発生し，確実なしゅう動動作が保障されるために，変位しても対物
レンズの傾きは発生しにくいとされている．しかし一方で，可動部

方向の駆動感度が低くなることがある．また微小変位を実現する際

なり，高精度位置決めに適さないという課題もある．さらに，可動
部が重いために，共振周波数を高くとることにも限界があり，二次

の傾きについては別の方法で補償するなど工夫がなされ，現在，
DVD装置ではワイヤ支持型がアクチュエータの主流となってい
