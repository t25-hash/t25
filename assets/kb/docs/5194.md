# 5・4・2 停止性能(stopping ability)

> **章**: 第5章 船舶
> **節**: 5・4 運動性能

船の停止性能は、車でいえばブレーキ性能に相当する。車の場合、ブレーキを踏めば車の回転数が減少し、最後には車の摩擦により停止する。しかし、船の場合はそう簡単ではない。船舶の質量は、車とはけた違いであり、少々の摩擦では止まらない。ディーゼルエンジンで固定ピッチのプロペラをもつ通常の船の場合、急速に停止するには、プロペラ回転数を減少させ、速度を減少させる。さらにプロペラを逆転させ、後進推力により船速を零にして停止させる。車の急ブレーキに相当するが、車と違うのはいつでもプロペラを逆転させることができるわけではないことである。通常の船の場合、緊急停止の発令後、遊転しているプロペラの回転数が十分に減少したところ(最近のディーゼル主機では、最大回転数の30~40%)でエアブレーキをかけてプロペラ回転を完全に止め、それからプロペラ回転を逆転させ、後進推力を出して停止させる。このため、緊急停止発令後プロペラ回転が完全に止まり逆転させるまでの主機操作時間が、船が停止するまでの距離および時間に対して大きく影響する。これに対して可変ピッチのプロペラをもつ船は、プロペラ回転の方向を変えることなく、プロペラピッチを変えることで後進推力を出すことができる。このため、固定ピッチをもつ船より、後進発令から後進推力を発生するまでの時間が短くなり、停止距離(stop-ping distance)が短くなる。これ以外にも、5・4・1・b項で紹介したシリング舵2枚を一つのシステムとして制御する舵を装備した船は、プロペラによる前進推力を、2枚の舵を閉じるようにして船首方向に流れの向きを変え、後進推力とすることができる。この舵を装備した場合、プロペラの回転方向を変える必要がないため、後進推力を得るまでの時間が短くて済み、やはり停止距離が短くなる。また、船が停止するまでの距離および時間は、前述のプロペラ回転が完全に止まるまでの主機操作時間以外に、初速、排水量、主機型式の違いによる後進推力の差に大きく影響される。したがって、大型船の停止距離を推定するに当たって、船の慣性力と後進推力である主機の制動力の影響を考慮し、停止距離がこれらの比で表せるとして、船の排水量、主機の最大連続出力(MCR)、MCR での船6―147図5・55 停止距離(a) 中立(b) 安定(c) 不安定図5・56 釣合いの前進速度、および船の試験速度でのフルード数(=,:船速、:重力の加速度、:船長)を用いる推定法もある。実際に計測された停止距離を図5・55に示す。横軸に船の長さ、縦軸に実測された停止距離を船長で割った無次元値を表している。停止距離は船の大きさにもよるが、大きなものでは船長の20倍くらいになることがわかる。図5・55中のIMO MSC.137(76)は
