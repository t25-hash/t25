# 6・3・1 複写機（copying machine，copier )

> **章**: 第6章 メカトロニクスの応用展開
> **節**: 6・3 オプトメカトロニクス機器

PCおよびその周辺機が充実した今日，文書を複写する方法は多
種多様に存在する．しかし1枚当たりのコストを低く抑え，手軽で
すぐに複写を得るというニーズを満たす技術は少ない．ジアゾ化合
物を塗布した感光紙を用いるジアゾ複写機（diazo copying ma-
chine）や，酸化亜鉛感光紙を用いる直接式電子写真法（エレクト
ロファックス，electrofax）などが実用化されてきたが，専用紙を
使用する難点があり，現在は事務用複写機のほとんどがPPC
（plane paper copying machine）と呼称される電子写真複写機
（electrophotographic copying machine）となっている．
a.電子写真（electrophotography) 電子写真プロセスでは，
感光紙に代わり感光体（photoconductor ）と呼ばれる円筒または
無端ベルト状部材を媒体として静電潜像を作り，帯電したトナー
（toner ）で現像し，これを用紙に転写することで普通紙に連続的に
複写を行う ．
有機感光体はアルミニウム素管など導電性の基体表面に有機材料
を用いた電荷発生層（carrier generation layer ）と，透明な電荷
輸送層（carrier transport layer ）がコートされており，暗中では
表面が均一に帯電可能であり，光を選択的に当てると当該部分の表
面電荷が除電され静電潜像が形成できる機能を有している．耐久性
に優れるアモルファスシリコン感光体を採用したものもある．
トナーは粒径5～15mの粉体で，バインダ樹脂に着色剤，帯電
制御剤，離型剤などを混練，粉砕，分級して作られる．モノマを水
媒体中で重合させてトナー粒子を作る方法もある．
画像形成のプロセスを図6･68に示し，各工程について説明する．
.帯電（charging）工程 感光体表面に放電に伴うイオ
ンあるいは電子を付着させて，一定電位かつ均一に帯電する．
150m以下の微小径のワイヤを金属ケース内に懸架して5～10
kV程度の高電圧を加えてワイヤ周辺に強電界を形成し，コロナ放
電に伴うイオンによって帯電するコロナ帯電器や，感光体表面に導
電性ゴムローラなどの電極部材を設け直接放電させることで帯電を
行う接触帯電器を用いる．
.露光（exposure）工程 感光体の光導電現象を利用し
感光体表面に選択的に光を照射することで，原稿像に対応した静電
潜像を形成する．
図6･69のように，アナログ複写機では原稿像をスリット状光源
で照らし，可動ミラーおよび結像レンズを用いて直接感光体表面に
導いて露光を行う．これに対しディジタル複写機では，原稿像をい
ったん電気信号に変換，処理した後，変調光を照射し露光を行う．
ディジタル露光方式についてはc項で詳述する．
.現像（development ）工程 静電潜像を着色した帯電微
粒子であるトナーを用いて可視像とする．
トナーは現像装置内部でかくはんすることで摩擦帯電し，現像ロ
ーラ上に薄層付着したうえ，感光体と対向することで画像部分に静
電気力により選択的に付着する．トナーの付着量は感光体と現像ロ
ーラ間に電位差を設け，現像工程近傍の電界を適切に設定すること
で調整される．キャリヤ（carrier ）と呼ぶ磁性粉体をトナーと混
合し，現像ローラ上へマグネットを用いてトナーを搬送する2成分
磁気ブラシ現像器が広く用いられる．
.転写（transfer ）工程 感光体上のトナー像を直接また
は中間転写体を介して用紙上に転写する．
転写領域に電界を付与し，静電気力を主体に帯電しているトナー
を転写する静電転写法が用いられる．用紙を帯電させることで強い
転写電界を与えるコロナ転写法と，電極となるローラまたはベルト
を当接させ，電界を形成するローラ転写法，ベルト転写法などがあ
る．

b.アナログ複写機の問題点とディジタル化による改善 図6･
70のジョーンズプロットは露光サブシステム（露光光学系），潜像
サブシステム（感光体，帯電工程，露光工程），トナー像サブシス
テム（トナー，現像，転写，定着）の入出力特性を各象限に配置
し，調子再現曲線（tone reproduction curve）を求める方法であ
る．
アナログ複写機では，原稿濃度に対し忠実なコピー濃度再現を得
ることが極めて難しい．潜像サブシステム，トナー像サブシステム
がいずれも飽和型のS字特性を有しているため，トータルのリニ
アリティ確保が難しいこと，加えて画像パターンによってトナー像
サブシステムの感度曲線が変化することがその理由として挙げられ
る．
また，露光サブシステム，潜像サブシステムの感度は波長により
異なるので，カラー原稿の濃度を再現することも難しい．
ディジタル複写機ではCCDを用いた読取り光学系を用いて原稿
像をいったん電気信号に変換し，ディジタル回路技術を駆使して多
様に処理した後，レーザ光やLEDを応用した書込み光学系を用い
て電気信号に応じて露光する方法をとる．露光光学系を電気的に調
整することで調子再現曲線が直線となるよう補正が可能となる．読
取り光学系をカラー対応とすることで電気的に調整が可能となり，
カラー原稿にも対応できる．このようなオプトメカトロニクス技術
の登場によってコピー濃度再現が制御可能となり，シアン（C）
色，マゼンタ（M）色，イエロー（Y）色を用いた減色混合による
高品位フルカラー複写機が実現可能となった．
c .ディジタル露光光学系の構成 高品位のディジタル露光を
得るには，人間の視覚特性に対応できる十分な解像度（単位長さ当
たりのドット数），ドット階調数（階調表現ビット数）が必要とな
る．複写機では解像度600dpi ，階調数2bit程度の描画力を備え
る．その情報量はA4サイズ単色1枚で8MB，フルカラーでは4
倍の32MBほどになる．1分間にカラーA4サイズ30枚の生産性
を生むためには，少なくとも16MB/ s程度の画像データフローを
処理できるディジタル回路技術が必要となる．
.読取り光学系（image scanning optical system) 図6･
71のように，照明系，第1走査ユニット，第2走査ユニット，結
像レンズ系，CCD駆動回路から構成される．原稿をスリット状に
照明する光源はハロゲンランプ，蛍光管などが用いられる．張り合
わせのある原稿でも影を生じないように，ミラーを対向配置し2方
向より照明する．走査系は固定原稿面に対してはアナログ露光系と
同様の走査ミラーで構成する．極めて低振動，等速かつ起動停止時
間を短縮するため，プーリとワイヤ，高起動トルクのステッピング
モータあるいはサーボモータを用い，駆動回路もマイクロステップ
駆動などの工夫を行う．自動原稿送り装置を備えたものは，原稿自
身を搬送することで複写速度に見合う高速走査を行う．
結像レンズ系およびCCD駆動回路はCCDレンズブロックと呼
ばれる一体化モジュール構成とし，焦点および光軸調整を容易とす
る．CCDは量産化が進み，一般的に3ラインカラーCCDが用いら
れる．読取り感度は光量変動，結像レンズ系のコサイン四乗則によ
る原稿幅方向のシェーディングを補正するため，原稿近傍に白色基
準板を設け，感度分布の調整を行う．アナログ複写機では結像レン
ズ系に変倍機能をもたせるが，ディジタル複写機では変倍機構を不
要とし，電気信号処理によって変倍を行う．

optical writing system) 用紙幅に対応する長尺のLEDアレイ
を屈折率分布型レンズを用いた等倍結像素子と組合わせ，感光体に
対向配置する．LEDアレイの高密度化，各露光スポットの均一性，
整列性をどう実現するかが重要課題であるが，これらが解決される
ことで今後実用化がさらに進むと考えられる．
d.フルカラー複写機の構成例と技術課題 フルカラー画像を
得るにはディジタル露光光学系を用いた濃度再現に加え，シアン，
マゼンタ，イエロー，ブラックの4色を精度よく色位置合せ
（color registration）する装置技術が必要である．人間の視覚特性
は50～150mの色ずれを認識できるので，高い機械精度が求めら
れる ．

.シングル方式（single engine) 図6･74のように同一の
露光サブシステム，潜像サブシステムを用い，現像工程を各色ごと
に切り換えて中間転写体上で色重ねを行い，一括して用紙に転写，
定着することでフルカラー画像を得る．共通の潜像形成までの工程
を利用するので，比較的容易に精度よく色位置合せができる．しか
し単色の場合に比べ，カラー生産性は4分の1に低下する大きな欠
点がある．
.タンデム方式（tandem engine) 四つの独立した露光
サブシステム，潜像サブシステム，定着工程を除くトナー像サブシ
ステムを用い，用紙に直接色重ね，あるいは中間転写体上で色重ね
後，用紙に二次転写することで色位置合せを行い，共通の定着工程
を経てフルカラー画像を生産性の低下なく出力する ．図6･75に
用紙に直接色重ねするタイプの装置例を示す．
色位置合せを精度よく実現するために4色の画像位置，傾き，お
よび倍率を調整する必要がある．この場合も制御性に優れるディジ
タル露光サブシステムの特徴を生かし，レーザビーム書込み光学系
であれば露光タイミングを調整することで水平，垂直位置を調整
し，さらに画像信号周波数を微調整することで倍率を調整するよう
に構成する．ビーム走査線の相互の傾き調整は画像信号調整では行
えないので，機械的に精度を調整するか，モータを備えた調整機構
を設け自動調整とする．
色ずれの原因は露光サブシステムの不ぞろいだけでなく，感光体
および中間転写体の不整列，偏心などによる速度変動も大きな問題
となる．駆動源，伝達機構，被駆動体までの系全体で最適な制御を
行う必要がある．
.イメージオンイメージ方式（image on image engine)
図6･76にイメージオンイメージ方式の装置例を示す．同一感光体
上で連続して帯電，露光，現像工程を4回続けて行うので，生産性
の低下がない．2色め以降の潜像形成，現像を前工程のトナー像の
上から重ねて行うため，前の画像を乱さず次の露光・現像を行う方
法が課題となる．この課題が解決できれば，露光サブシステムの整
列だけで色重ねが極めて精度よく実現できる大きな利点が得られ
る．
