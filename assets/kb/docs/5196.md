# 5・4・3 安定性能

> **章**: 第5章 船舶
> **節**: 5・4 運動性能

a. 船の横安定性能安定性能には、5・4・1項で述べた針路安定性能のほかに、表5・6の軸回りの縦安定性能、そして軸回りの横安定性能がある。一般に船舶の安定性能を論じる場合、針路安定性能は転覆などの危険と直接関係せず、縦安定性能は横安定性能と比べるとはるかに安定であり、横安定性能が転覆などの危険と最も強く関係しているから、横安定性能を指すことが多い。したがって、ここでは横安定性能について述べる。図5・56(a)に示すように、平面に置かれた球は重心に働く重力と球と平面の接点に働く反力の、大きさが等しく反対向きの力の釣合いで静止している。この釣合い状態からある外乱によって重心が移動し、また元の位置に戻ろうとする場合を安定な釣合い[図5・56(b)]といい、ますます離れていく場合を不安定な釣合い[図5・56(c)]という。また、図5・56(a)のように重心がいくら移動しても釣合っている場合を中立の釣合いという。船の安定性能は安定な釣合いにあり、不安定な釣合いの場合は次の安定点まで傾斜する。b. 復原力(restoring force)ある外乱に対して元の釣合い状態に戻ろうとするとき働く力のことを復原力という。正確には復原モーメントと呼ぶべきであるが、通常は復原力と呼ぶことが多い。船の横安定性能を考える場合、平水中、非損傷時の横復原性能が基本となる。図5・57に示すように、船が傾いたときの浮力と重力により生じる偶力が復原モーメントである。図5・58に復原力曲線の典型例を示す。水面下の形状が急激に変わらない傾斜角(図5・59の上甲板の舷端が没する傾斜角、または船底のビルジ部が水面に出るような傾斜角)のうち小さい角度まで、すなわち傾斜角が10°~15°では、浮力の作用線は一定点を通り、このをメタセンタと呼ぶ。このときをメタセンタ高さ(metacentric height)と呼び、傾斜初期のいわゆる初期復原力を表す。この初期復原力が復原力評価の基礎となる。横傾斜角が、初期復原力の適用限界角より大きい角度になると、浮力の作用線とメタセンタは一致しないが、復原てこは重心から浮心の作用線に下ろした垂線である。図5・58中で、傾斜角がφを超えると船は転覆することを意味している。復原力曲線は、重心高さはもちろん、船の幅、舷側の高さ、上部構造、開口位置によって大きく変化する。また、船内に滞留水等の自由な水面があると復原力は悪くなる。また、復原力曲線の積分値は、横揺れにおける風波等による外力の仕事量を示すことになるので、積分曲線を動的復原力曲線(dynamic stability curve)と呼ぶ。船は風波による横傾斜だけでなく、操舵旋回時に遠心力によって図5・60 横揺れの応答特性も横傾斜する。初期復原力が小さい場合には横傾斜角が大きくなり、荷崩れを誘起し転覆に至る場合も考えられるため、適切な復原力の確保は重要である。以上は平水中での安定性だが、波浪中では船と波の相対的な位置関係で、すなわち船が波の山や谷にあるときで復原力は異なり、時々刻々復原力は変化する。c. 復原力に関する基準わが国では、船舶の安全確保のため復原力の基準が定められている。旅客船の場合は、船の大きさ、航行区域によって基準値が異なり、横風による定傾斜モーメント、乗員のかたよりによる傾斜モーメント、さらに突風波浪に基づく横揺れの動的効果、海水打込みによる余裕などを考えて定められている。貨物船に対しても船の大きさ、航行区域によって適用範囲は異なるものの、旅客船と類似した基準が設けられている。以上は非損傷時の場合であるが、船舶が衝突等の事故により損傷した場合の安定性は損傷・浸水した区画の復原力が喪失すると考えて求めることができる。損傷した場合の安定性に対しては国際基準が定められており、国際航路に従事する船舶はこの基準を満足する必要がある。このため、水密区画の配置、乾舷高さなどが制限される場合がある。RO/RO船のように広大な全通の車両甲板をもつ船は、短時間に大量の海水がこの車両甲板に滞留したりすると転覆する危険があり、こうした場合の復原性の基準も地域によっては設けられている。
