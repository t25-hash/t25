# 5・4・4 耐航性能(seakeeping ability)

> **章**: 第5章 船舶
> **節**: 5・4 運動性能

船の性能は、平水中の性能で判断されることが多いが、航海中の安全性などを考えると波浪中での性能も重要である。船は、一般に船体振動の周波数範囲が船体運動のものと大きく離れているために剛体と考えられる。したがって、船の運動は図5・51のように座標軸をとり、慣性主軸に沿って各軸方向の直線運動と軸回りの回転運動の合成として表せる(各運動の名称は表5・6参照)。船の動揺は、おもに波の力によって生じるが、規則波中の周波数応答特性が基本となる。不規則波中の動揺は、この応答特性を波のスペクトラムに乗じて動揺スペクトラムを求め、それから求まる統計的な性質で表される。規則波中の動揺の応答特性は、粘性影響のある横揺れ減衰の問題を除いて船を細長体と仮定した細長船理論を応用して計算することができる。細長船理論の代表的なものの一つに二次元ストリップ法(strip method)がある。この方法では、船体形状は左右対称で、長さに比べて幅、喫水が小さな細長体で、前後の対称性も大局的には保証されると仮定する。そうすると船体運動は、船体長手方向の軸を含む鉛直面内の運動である上下揺れと縦揺れの連成運動、水平面内の運動である左右揺れ、船首揺れと横揺れの連成運動、および前後揺れの3種類に分けて計算できる。これらの運動を、海洋波を微小振幅波で一方向からくる規則波と仮定し、さらに船長方向に輪切りにした各ストリップを二次元の物体と仮定すると、このストリップに作用する波浪外力を求めることができる。これを動揺の運動方程式の係数に代入し運動を求める。したがって、これらの係数を各波周波数に対して求めておけば、各周波数に対する船体運動の応答特性が求められる。ある波高の波に対する横揺れの応答特性の一例を図5・60に示す。図5・60中、욦は船の横揺れ固有周期であり、욌は波の周期を表している。また、μは横揺れ振幅と波傾斜図5・61 出合周期の比を表しており、船の横揺れ固有周期と波の周期が一致すると横揺れが大きくなることがわかる。なお、動揺の強制力の周期は、船と波との出合周期욄=λ욌욌-cos χで示される。図5・61に示すように、λ욌は波の波長、욌は波の進行速度、は船速、χは出合角である。縦揺れと上下揺れは推進性能を低下させるだけでなく、その運動が大きくなるとスラミング(slamming)と呼ばれる現象を起こしたり、船首への打込み水により、船首甲板上や貨物倉のハッチ等に損傷を生じたりする。スラミングとは、船首船底がある瞬間波面から離れて、次の瞬間波面をたたき、大きな衝撃圧を船首船底に受ける現象である。したがって、縦揺れは減少させることが望ましいが、そのモーメントは大きいこともあり一般には減少させることは難しい。なお、船の減揺装置に関しては5・5節を参照されたい。
