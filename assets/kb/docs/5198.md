# 5・5・1 乗物と乗り心地

> **章**: 第5章 船舶
> **節**: 5・5 船舶の乗り心地

a.乗物酔いの定義 乗物酔いは医学の分野では，自動車，列
車，飛行機，船舶等の乗物で一定時間揺られることによって引き起
こされる一過性の自律神経失調状態をいい，その代表的症状は不快
感，眠気，生つば，頭重感，悪心，吐き気，嘔（おう）吐である，
と定義されており，動揺病と呼ばれることが多い．また，乗物
上における外的刺激は動揺加速度であることから，加速度病とも呼
ばれる．
b.船酔い発症のメカニズムと症状 船酔い発症のメカニズム
は完全には解明されてはいない．古くからの乗物酔いに関する医学
的研究成果として過剰刺激説，体液移動説，感覚混乱説等さまざま
な説があるが，現在では感覚混乱説 が支持されている．列車や
自動車のような周波数の高い振動刺激と異なり，船舶の場合は周波
数1Hz以下の低周波動揺であるため，感覚混乱説による解釈が適
していると考えられる．通常の1Gの重力の加速度のもとでは三次
元空間における人間の空間識（位置感覚）は，(１)重力ベクトルと
直線加速度に関する前庭の情報，(２)三半規管による角加速度情
報，(３)身体の位置感覚に関する視覚情報，(４)深部知覚による体
性感覚情報，の4情報に基づいている．感覚混乱説では，視覚，平
衡感覚，深部知覚として日常の重力環境下で中枢神経系内に蓄積さ
れた空間識に対して，通常は非日常的な動揺環境下においても目，
前庭，三半規管等の感覚受容器からの信号情報が比較され円滑な運
動がなされるが，動揺による強い外部刺激，長時間乗船による持続
的外部刺激・ストレスの場合には急にあるいは徐々に不適合が起こ
り，自律神経失調状態が誘発されると解釈されている（図5･62)．
また，自律神経失調状態に対しては心理的影響が大きいことも指摘
されている．
乗物酔い発症時の症状には個人差や場合差が大きいが，統計的な
調査によれば，発症頻度は頭重，悪心，生つば，頭痛，ふらふら
感，嘔吐，胃部膨満感，顔面蒼白，無力感，冷汗，欠伸，便意の順
に多く，発症の順序は，頭重，欠伸，頭痛，生つば，冷汗，便意，
胃部膨満感，顔面蒼白，悪心，ふらふら感，嘔吐であると報告され
ている．
5・5・2乗物酔い発症に影響する要因
a.乗物酔いに対する感受性
.年齢 生後2歳くらいまでは酔いにくいが，2歳から小
学生年齢は酔いやすい．その後加齢とともに酔いにくくなり，50
歳以降では酔うことは少ない．
.性差 女性は男性より酔いやすい．思春期前の少女は少
年より酔いやすい．
.生理的要因 消化器官（胃，腸）に不調のある人，頭痛
を起こしやすい人，迷路受容器に過敏性のある人は酔いやすい．
.心理的要因 性格，ストレス，酔いの経験，不安，予感
等が酔いの感受性に影響することが知られている．
b.外的刺激およびその許容限界
.動揺モード 船体の運動モードに関しては上下揺れ，縦
揺れ，横揺れの順に船酔いを起こしやすい．また，加速度に関して
は，上下加速度が酔いの発症に強く関与している．ただし，姿勢，
活動状況，身体的・精神的体調および個人差による差がある．
.加速度レベル 上下加速度では0.1～0.2Gで酔いやす
くなるといわれる．また，加速度の時間微分（力度）を指標とする
説もある．加速度および動揺角度の許容限界に関しては造船学の分
野においては古くから種々の提案があるが，よく用いられているの
はノルウェーの基準である（表5･7)．
.動揺周波数 上下動揺では，0.1～0.6Hzで酔いやすい
ことが知られているが，個人差が大きい．
.加速度および周波数の複合効果 船酔いの場合には上下
加速度とその周波数は個別にも効果があるが，加速度レベルと周波
数は図5･63に示されるように複合効果をもつ．
.動揺持続時間の影響 船酔いの場合には動揺持続時間が
大きな影響をもつ．図5･64 中の曲線より上側の加速度レベルで
は10%の乗客が船酔いを起こすことを表している．また，一般に
人は動揺刺激暴露に対する適応性（学習効果）をもつため，乗船後
長時間経過することによる慣れの存在が示されている．
.その他の外的刺激の影響 読書，細かい作業等一点に集
中すると酔いやすい．これは感覚混乱説によって説明できる酔いの
発症である．しかし，他人の嘔吐の目撃，船内特有の臭いによって
酔うことはよく知られているが，解明されてはいない．

c .船酔いの予防と治療
.予防方法 乗物酔いの予防法として確立された方法はな
い．対症療法として，薬物の内服，薬物の貼付が一般的であるが，
生理的・心理的個人差のため，すべての場合に必ず効果を発揮する
とは限らない．乗物酔い発症の予防には心理療法が効果を発揮する
といわれる．
.設計・運航時の対策 旅客船を設計する際には乗り心地
や船酔い対策も重要な設計条件であるが，必ずしも合理的な対策が
とられているわけではない．外洋を航行する大型旅客船や長距離フ
ェリーにおける上級船室は通常船首側の上層甲板に配置されるが，
一般に船体重心より前方は上下揺れ，縦揺れに誘起される上下加速
度が大きく，上層甲板ではさらに横揺れによる水平加速度も大き
い．設計時には，瘠（せき）型船型の採用，動揺固有周期に対する
配慮，動揺軽減装置の配備，航海速力の選定等によって波浪中船体
