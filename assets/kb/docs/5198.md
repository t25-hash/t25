# 5・5・1 乗物と乗り心地

> **章**: 第5章 船舶
> **節**: 5・5 船舶の乗り心地

a. 乗物酔いの定義乗物酔いは医学の分野では、自動車、列車、飛行機、船舶等の乗物で一定時間揺られることによって引き起こされる一過性の自律神経失調状態をいい、その代表的症状は不快感、眠気、生つば、頭重感、悪心、吐き気、嘔(おう)吐である、と定義されており、動揺病と呼ばれることが多い。また、乗物上における外的刺激は動揺加速度であることから、加速度病とも呼ばれる。b. 船酔い発症のメカニズムと症状船酔い発症のメカニズムは完全には解明されてはいない。古くからの乗物酔いに関する医学的研究成果として過剰刺激説、体液移動説、感覚混乱説等さまざまな説があるが、現在では感覚混乱説が支持されている。列車や自動車のような周波数の高い振動刺激と異なり、船舶の場合は周波数1Hz 以下の低周波動揺であるため、感覚混乱説による解釈が適していると考えられる。通常の1G の重力の加速度のもとでは三次元空間における人間の空間識(位置感覚)は、(1)重力ベクトルと直線加速度に関する前庭の情報、(2)三半規管による角加速度情報、(3)身体の位置感覚に関する視覚情報、(4)深部知覚による体性感覚情報、の4情報に基づいている。感覚混乱説では、視覚、平衡感覚、深部知覚として日常の重力環境下で中枢神経系内に蓄積された空間識に対して、通常は非日常的な動揺環境下においても目、前庭、三半規管等の感覚受容器からの信号情報が比較され円滑な運動がなされるが、動揺による強い外部刺激、長時間乗船による持続的外部刺激・ストレスの場合には急にあるいは徐々に不適合が起こり、自律神経失調状態が誘発されると解釈されている(図5・62)。また、自律神経失調状態に対しては心理的影響が大きいことも指摘されている。乗物酔い発症時の症状には個人差や場合差が大きいが、統計的な調査によれば、発症頻度は頭重、悪心、生つば、頭痛、ふらふら感、嘔吐、胃部膨満感、顔面蒼白、無力感、冷汗、欠伸、便意の順に多く、発症の順序は、頭重、欠伸、頭痛、生つば、冷汗、便意、胃部膨満感、顔面蒼白、悪心、ふらふら感、嘔吐であると報告されている。6―149図5・62 乗物酔い発症メカニズム(感覚混乱説)表5・7 加速度および横揺れ角度の許容限界(NORDFORSK,1987)上下加速度G横加速度G横揺れ角度°適用0.020.032.0客船乗客0.050.042.5交通艇0.100.053.0頭脳労働0.150.074.0重作業0.200.106.0軽作業注:RMS Criteria
