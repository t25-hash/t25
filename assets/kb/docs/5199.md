# 5・5・2 乗物酔い発症に影響する要因

> **章**: 第5章 船舶
> **節**: 5・5 船舶の乗り心地

a. 乗物酔いに対する感受性. 年齢生後2歳くらいまでは酔いにくいが,2歳から小学生年齢は酔いやすい.その後加齢とともに酔いにくくなり,50歳以降では酔うことは少ない.. 性差女性は男性より酔いやすい.思春期前の少女は少年より酔いやすい.. 生理的要因消化器官(胃,腸)に不調のある人,頭痛を起こしやすい人,迷路受容器に過敏性のある人は酔いやすい.. 心理的要因性格,ストレス,酔いの経験,不安,予感等が酔いの感受性に影響することが知られている.b. 外的刺激およびその許容限界. 動揺モード船体の運動モードに関しては上下揺れ,縦揺れ,横揺れの順に船酔いを起こしやすい.また,加速度に関しては,上下加速度が酔いの発症に強く関与している.ただし,姿勢,活動状況,身体的・精神的体調および個人差による差がある.. 加速度レベル上下加速度では0.1~0.2G で酔いやすくなるといわれる.また,加速度の時間微分(力度)を指標とする説もある.加速度および動揺角度の許容限界に関しては造船学の分野においては古くから種々の提案があるが,よく用いられているのはノルウェーの基準である(表5・7).. 動揺周波数上下動揺では,0.1~0.6Hz で酔いやすいことが知られているが,個人差が大きい.. 加速度および周波数の複合効果船酔いの場合には上下加速度とその周波数は個別にも効果があるが,加速度レベルと周波数は図5・63に示されるように複合効果をもつ.. 動揺持続時間の影響船酔いの場合には動揺持続時間が大きな影響をもつ.図5・64中の曲線より上側の加速度レベルでは10%の乗客が船酔いを起こすことを表している.また,一般に人は動揺刺激暴露に対する適応性(学習効果)をもつため,乗船後長時間経過することによる慣れの存在が示されている.. その他の外的刺激の影響読書,細かい作業等一点に集中すると酔いやすい.これは感覚混乱説によって説明できる酔いの発症である.しかし,他人の嘔吐の目撃,船内特有の臭いによって酔うことはよく知られているが,解明されてはいない.
