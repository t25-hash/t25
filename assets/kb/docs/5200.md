# 5・5・3 船舶の乗り心地評価と船酔い対策

> **章**: 第5章 船舶
> **節**: 5・5 船舶の乗り心地

a. 乗物酔い・乗り心地の計測と解析乗物酔い発症の原因が動揺による物理的刺激のみではなく,視覚・嗅覚等の五感によって6―150交通機械図5・63 嘔吐率に対する上下加速度と動揺周波数の複合効果図5・64 乗物酔い発症に対する加速度,周波数,暴露時間の影響図5・65上下加速度の暴露許容限界(ISO 2631-3)感知される刺激と,これに生理的・心理的・遺伝的・精神的要因による個人差の大きな影響が複雑に関連しているため,その計測は極めて難しく,確立された方法は存在しない.よく行われる生理計測として心電図,脈拍,脳波,発汗が挙げられ,時間領域,周波数領域,揺らぎ解析などが行われる.酔いの程度や乗り心地のような心理的変化は主観的であるから,アンケート調査や愁訴に頼らざるを得ないが,個人差に基づくあいまいさが計測や解析を困難にしている.総合的な解析には,統計的手法である主成分分析,判別分析,数量化理論,重回帰分析等を用いることが多い.あいまいさを考慮に入れたファジィ理論の適用も試みられている.b. 予防方法乗物酔いの予防法として確立された方法はない.対症療法として,薬物の内服,薬物の貼付が一般的であるが,生理的・心理的個人差のため,すべての場合に必ず効果を発揮するとは限らない.乗物酔い発症の予防には心理療法が効果を発揮するといわれる.. 設計・運航時の対策旅客船を設計する際には乗り心地や船酔い対策も重要な設計条件であるが,必ずしも合理的な対策がとられているわけではない.外洋を航行する大型旅客船や長距離フェリーにおける上級船室は通常船首側の上層甲板に配置されるが,一般に船体重心より前方は上下揺れ,縦揺れに誘起される上下加速度が大きく,上層甲板ではさらに横揺れによる水平加速度も大きい.設計時には,瘠(せき)型船型の採用,動揺固有周期に対する配慮,動揺軽減装置の配備,航海速力の選定等によって波浪中船体運動の軽減が図られるが,船体と外洋波浪の相対的な大きさが同程度であるため船体運動を零にすることはできない.船体の減揺図5・68 受動型減揺タンク図5・67 ビルジキール図5・70 フィンスタビライザ図5・69制御型減揺タンク図5・71 ハイブリッド式減揺装置図5・72 各種減揺装置の性能比較のためにほとんどすべての商船においてビルジキール(bilge-keel)が装備されている(図5・67).ビルジキールはある程度の減揺効果をもつものの,抵抗性能との関係で効果を大きくすることはできない.そこで,客船などのようにさらに減揺の必要な場合には,減揺タンク[anti-rolling tank,受動型(図5・68),制御型(図5・69)],フィンスタビライザ(fin stabilizer,図5・70)等の減揺装置を用いて船体動揺を軽減する方策がとられる.さらに,近年ハイブリッド式減揺装置(図5・71)が開発され,海洋地球研究船「みらい」に搭載された.各種減揺装置の相対的性能比較は図用途,航行海域,航海速力等の条件によって異なる.このほかの対策としては,旅客区画の快適性に配慮した内装がなされる.小型の旅客船やフェリーの航行区域は通常内海,近海であるので,動揺軽減装置が重視されることは少ない.小型の遊覧船において油圧制御によって乗客キャビンの動揺を制御した例,半没水双胴船型を用いた低動揺型旅客船の例がある.また,高速交通艇においては双胴船型や全没型水中翼船型を採用して波浪中船体動揺を大きく軽減している.小型の旅客艇や高速艇においては航走時の船体動揺は比較的小さいが,出入港の低速時,乗客乗降の停船時の動揺が乗り心地を悪化させる.
