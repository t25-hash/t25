a.ジャイロコンパス（gyrocompass) オートパイロット
（autopilot）はジャイロコンパスから情報を入手して適切な船首方
位にするために操舵機に信号を送る装置で，第二次世界大戦後の
1950年代以降，日本の商船の再興に伴って普及していった．
ジャイロコンパスという名前は，1852年にフランスのフーコー
がジャイロを使って地球の回転運動を証明したときの，地球の回転
（gyros）を見る機械（scope）に由来している．その後，ドイツの
アンシュッツ（Anschutz）とアメリカのスペリー（Sperry）がコ
マを利用したジャイロコンパスの製品化に成功し，船舶で使用され
るようになった．
ジャイロコンパスは図5･73のように制振作用をする軸，ジャイ
ロの回転方向，ジャイロの重心位置などによって分類できる．いず
れの方式においても船に搭載されたジャイロコンパスが検出するの
は船体運動によって発生する運動と，地球の自転による運動の合成
されたものなので，ジャイロコンパスとして使用するためには船体
運動による成分を分離したり，緯度により変化する地球の自転によ
る地盤の動きを補正する必要がある．
振り子式は民間の商船用として多く用いられており，比較的安価
に製造できるが，加速度などの外力の影響が大きく，周期的な運動
に対してもフィルタなどで減衰させることができないので，加速度
が加わる状態では誤差を減らすことが難しい．
プラットホーム式は図5･74に示すように通常2個のジャイロを
二軸自由に支持し，ジャイロの重心と支点を一致させたフリージャ
イロと加速度計2個を組合わせて安定した水平面を作る．一方のジ
ャイロを振り子式と同様のトルクを加えて北を向かせ，他のジャイ
ロは南北ジャイロと直角にかつ水平になるように制御させる．これ
により，計算過程において補正が簡単で外力の影響の少ない高精度

b.制御方式 現在のオートパイロットの制御方式は1950年
代当時と大きな違いはなく，図5･75に示すように比例制御（P），
微分制御（D），針路偏差が不感帯内にあるときには舵をとらない
ような不感帯機構による天候調整機能をもっている．
その後，1960年代になり，上記のほかに長時間一定方向から風
や潮流を受けたときに生じる針路偏差をなくすための積分制御
（I），波浪により発生する周波数の高い外乱に対し不感帯を設けて
舵をとらないのではなく，小さな針路偏差に対しては小舵角で舵を
とり，大きな偏差に対しては大きな舵をとるような非線形二重ゲイ
ンの採用，急変針しても大舵角をとらないよう舵角リミットを設け
るなどの機能が追加されている．
さらに，1990年代以降になるとヒューマンインタフェースの改
善が求められるようになり，設定したコースだけではなくルートか
らの偏位に対応するためのルートトラッキング制御，特にヨーロッ
パで強く要求された河川航行や諸島間航行において変針時に船の航
跡を確実に制御するためのレート制御変針，トラッキング中にモー
ドを切換えることなく避航操船を行えるようなオーバライド制御な
どの機能が加えられるようになってきた．
国際海事機関（IMO）では2002年7月1日に発効した改訂
SOLAS条約第Ⅴ章（航行の安全）の中で，従来の針路を保持する
装置をHCS（heading control system），航路を保持する装置を
TCS（track control system）と呼び区別している．また10000
GT（gross tonnage，総トン数）以上の船舶はHCSかTCSのい
ずれかを装備することとした．
5・6・2レーダプロッティング装置
レーダプロッティング装置（radar plotting aids）とは船舶の航
海用レーダを使って，他船またはそれらしい物標の位置を時々刻々
プロッティングすることにより，危険な関係にあるかどうかの判断
を援助する装置である．改訂SOLAS条約第Ⅴ章ではレーダプロ
ッティング装置を電子プロッティング装置，自動物標追跡装置，自
動衝突予防援助装置の3種類に分けて，それぞれを船の総トン数に
応じて装備することにした．
a.電子プロッティング装置 電子プロッティング装置
（EPA:electronic plotting aids）は手動で選択された物標を電子
的にプロットを続けて，その物標が危険な関係にあるかどうかのデ
ータを表示することによって判断を援助する装置である．300GT
以上の全船舶および総トン数に関係なくすべての旅客船には航海用
レーダを備え，そのレーダにはEPAを必ず装備することになって
いる．
b.自動物標追跡装置 自動物標追跡装置（ATA:automatic
tracking aids）は自動的に10以上の物標を選択して追尾し，危険
の程度を予報し，データを表示させて判断の援助をする装置であ
る．500GT以上の全船舶のレーダにはATAを装備することにな
っている（図5･76)．
c .自動衝突予防援助装置 自動衝突予防援助装置（ARPA:
automatic radar plotting aids）は自動的に20以上の物標を選択
して追尾し，危険の程度を予報し，データを表示し，より安全な関
係になるような自船の操船方法をシミュレーションして判断の援助
をする装置である．10000GT以上の全船舶の航海用レーダにはこ
れを装備することになっている（図5･77)．
5・6・3電波航法システム
電波航法システム（radio navigation system）としては，地上
系のシステムとしてロランA，ロランC，デッカ，オメガなどがあ
り，衛星系のシステムとしてGNSS，GPSなどがある．現在船舶
用として最も使用されているシステムが地上系ではロランC，衛星
系ではGPSである．
a.ロランC ロランC（LORAN- C:long range navigation
C）はマイクロコンピュータの採用により高度な信号処理が可能に
なって普及した．このシステムの最大の特徴は絶対精度はそれほど
良くはなく数百mの誤差があるにもかかわらず，再帰精度が良い
ということである．そのために以前に行った地点に再度行っても前
回とほとんど同じロラン時間差になるので，漁船にとって漁場のポ
イントを決める際には好都合である．計測位置はロラン時間差
（LOP）で表示されるため，緯度経度に変換する必要がある．しか
し，小型漁船などでは現在もロラン時間差表示のまま使用してい
る．最近ではロランC受信機は全自動で緯度経度に変換し，目的
地までの距離，方位や船速，針路を演算しブラウン管上や液晶表示
器に電子参考海図などを使って，現在位置，過去の航跡，予定航路
などを表示することができるようになっている．
b.広域測位システム 広域測位システム（GPS:global
positioning system，公式名称は全世界的測位システム）は地球全
体規模でどこでも24時間測位可能なシステムで，高精度かつ速度
情報も得ることができるなどの特徴をもっている．GPSシステム
の精度向上対策としては，ディファレンシャルGPS（DGPS）が
考えられている．DGPSは地上の既知の位置に基準となるGPS受
信機を設置し，視野内の衛星の電波を受信してその誤差分を推定
し，無線によってその値を船舶側のGPS受信機に伝達して誤差成
