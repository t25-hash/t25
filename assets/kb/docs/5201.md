# 5・6・1 オートパイロット

> **章**: 第5章 船舶
> **節**: 5・6 装備品

a. ジャイロコンパス(gyrocompass)オートパイロット6―151図5・73 ジャイロスコープ・ジャイロコンパスの分類図5・74 プラットホーム式ジャイロコンパス(autopilot)はジャイロコンパスから情報を入手して適切な船首方位にするために操舵機に信号を送る装置で,第二次世界大戦後の1950年代以降,日本の商船の再興に伴って普及していった.ジャイロコンパスという名前は,1852年にフランスのフーコーがジャイロを使って地球の回転運動を証明したときの,地球の回転(gyros)を見る機械(scope)に由来している.その後,ドイツのアンシュッツ(Anschutz)とアメリカのスペリー(Sperry)がコマを利用したジャイロコンパスの製品化に成功し,船舶で使用されるようになった.ジャイロコンパスは図5・73のように制振作用をする軸,ジャイロの回転方向,ジャイロの重心位置などによって分類できる.いずれの方式においても船に搭載されたジャイロコンパスが検出するのは船体運動によって発生する運動と,地球の自転による運動の合成されたものなので,ジャイロコンパスとして使用するためには船体運動による成分を分離したり,緯度により変化する地球の自転による地盤の動きを補正する必要がある.振り子式は民間の商船用として多く用いられており,比較的安価に製造できるが,加速度などの外力の影響が大きく,周期的な運動に対してもフィルタなどで減衰させることができないので,加速度が加わる状態では誤差を減らすことが難しい.プラットホーム式は図5・74に示すように通常2個のジャイロを二軸自由に支持し,ジャイロの重心と支点を一致させたフリージャイロと加速度計2個を組合わせて安定した水平面を作る.一方のジャイロを振り子式と同様のトルクを加えて北を向かせ,他のジャイロは南北ジャイロと直角にかつ水平になるように制御させる.これにより,計算過程において補正が簡単で外力の影響の少ない高精度6―152交通機械図5・75 オートパイロット制御ブロック図のものができるので,軍事用,航空機用に用いられている.その他に水平プラットホームを用いず,ジャイロや加速度計を直接船体に固定するリングレーザジャイロやファイバジャイロがある.これらは,船体運動による姿勢の変化に対応して,機械的な水平面ではなくコンピュータの内部に仮想的な水平面を作り出す.初期値が得られれば基準の三軸回りの角速度と三軸方向の加速度を計測して,それらの積分値を初期値に加算して方位,ロール,ピッチ,緯度,経度を求めることができる.b. 制御方式現在のオートパイロットの制御方式は1950年代当時と大きな違いはなく,図5・75に示すように比例制御(P),微分制御(D),針路偏差が不感帯内にあるときには舵をとらないような不感帯機構による天候調整機能をもっている.その後,1960年代になり,上記のほかに長時間一定方向から風や潮流を受けたときに生じる針路偏差をなくすための積分制御(I),波浪により発生する周波数の高い外乱に対し不感帯を設けて舵をとらないのではなく,小さな針路偏差に対しては小舵角で舵をとり,大きな偏差に対しては大きな舵をとるような非線形二重ゲインの採用,急変針しても大舵角をとらないよう舵角リミットを設けるなどの機能が追加されている.さらに,1990年代以降になるとヒューマンインタフェースの改善が求められるようになり,設定したコースだけではなくルートからの偏位に対応するためのルートトラッキング制御,特にヨーロッパで強く要求された河川航行や諸島間航行において変針時に船の航跡を確実に制御するためのレート制御変針,トラッキング中にモードを切換えることなく避航操船を行えるようなオーバライド制御などの機能が加えられるようになってきた.国際海事機関(IMO)では2002年7月1日に発効した改訂SOLAS 条約第V章(航行の安全)の中で,従来の針路を保持する装置をHCS(heading control system),航路を保持する装置をTCS(track control system)と呼び区別している.また10000GT(gross tonnage,総トン数)以上の船舶はHCS かTCS のいずれかを装備することとした.
