# 5・6・2 レーダプロッティング装置

> **章**: 第5章 船舶
> **節**: 5・6 装備品

レーダプロッティング装置(radar plotting aids)とは船舶の航海用レーダを使って、他船またはそれらしい物標の位置を時々刻々プロッティングすることにより、危険な関係にあるかどうかの判断を援助する装置である。改訂SOLAS 条約第V章ではレーダプロッティング装置を電子プロッティング装置、自動物標追跡装置、自動衝突予防援助装置の3種類に分けて、それぞれを船の総トン数に応じて装備することにした。a. 電子プロッティング装置電子プロッティング装置(EPA : electronic plotting aids)は手動で選択された物標を電子的にプロットを続けて、その物標が危険な関係にあるかどうかのデータを表示することによって判断を援助する装置である。300GT以上の全船舶および総トン数に関係なくすべての旅客船には航海用レーダを備え、そのレーダにはEPA を必ず装備することになっている。図5・76 自動物標追跡装置図5・77 自動衝突予防援助装置機能内蔵レーダb. 自動物標追跡装置自動物標追跡装置(ATA : automatictracking aids)は自動的に10以上の物標を選択して追尾し、危険の程度を予報し、データを表示させて判断の援助をする装置である。500GT 以上の全船舶のレーダにはATA を装備することになっている(図5・76)。c. 自動衝突予防援助装置自動衝突予防援助装置(ARPA :automatic radar plotting aids)は自動的に20以上の物標を選択して追尾し、危険の程度を予報し、データを表示し、より安全な関係になるような自船の操船方法をシミュレーションして判断の援助をする装置である。10000GT 以上の全船舶の航海用レーダにはこれを装備することになっている(図5・77)。
