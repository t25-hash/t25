# 5・6・3 電波航法システム

> **章**: 第5章 船舶
> **節**: 5・6 装備品

電波航法システム(radio navigation system)としては、地上系のシステムとしてロランA、ロランC、デッカ、オメガなどがあり、衛星系のシステムとしてGNSS,GPS などがある。現在船舶用として最も使用されているシステムが地上系ではロランC、衛星系ではGPS である。a. ロランCロランC(LORAN-C: long range navigationC)はマイクロコンピュータの採用により高度な信号処理が可能になって普及した。このシステムの最大の特徴は絶対精度はそれほど良くはなく数百mの誤差があるにもかかわらず、再帰精度が良いということである。そのために以前に行った地点に再度行っても前回とほとんど同じロラン時間差になるので、漁船にとって漁場のポイントを決める際には好都合である。計測位置はロラン時間差(LOP)で表示されるため、緯度経度に変換する必要がある。しかし、小型漁船などでは現在もロラン時間差表示のまま使用している。最近ではロランC受信機は全自動で緯度経度に変換し、目的地までの距離、方位や船速、針路を演算しブラウン管上や液晶表示器に電子参考海図などを使って、現在位置、過去の航跡、予定航路などを表示することができるようになっている。b. 広域測位システム広域測位システム(GPS : globalpositioning system、公式名称は全世界的測位システム)は地球全体規模でどこでも24時間測位可能なシステムで、高精度かつ速度情報も得ることができるなどの特徴をもっている。GPS システムの精度向上対策としては、ディファレンシャルGPS(DGPS)が考えられている。DGPS は地上の既知の位置に基準となるGPS 受信機を設置し、視野内の衛星の電波を受信してその誤差分を推定し、無線によってその値を船舶側のGPS 受信機に伝達して誤差成(a) 平面図(b) 側面図図5・783ビーム方式ドップラーソナーの原理分を補正する。これらの測位システムは、電子海図情報表示装置(ECDIS :electronic chart display and information system)や統合船橋システム(IBS : integrated bridge system)の主要センサとして使用されている。
