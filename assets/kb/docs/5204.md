# 5・6・4 ドップラーソナー

> **章**: 第5章 船舶
> **節**: 5・6 装備品

ドップラーソナー(Doppler sonar)は海底に向けて斜めに超音波ビームを発射し、海中および海底からの反射信号の周波数を計測することによって、船速を測定する装置である。図5・78に3ビーム方式の例を示す。ビームは船の前方向(ビーム1)、右後方向(ビーム2)、左後方向(ビーム3)に発射され、これにより船の前後、左右の速度を知ることができる。現在では単に船速計というだけではなく、多くの機能が付加されてきている。ドップラーソナーによって対地船速や対水船速が計測されるが、これらの差をとれば潮流の情報が得られる。このようにして、現在では、船速、真針路、船首方位、潮流、風、キールクリアランス、エンジン回転数、回頭角速度、緯度、経度、航程(積算距離)などを一つの表示器で表示するようになっているものもある。
