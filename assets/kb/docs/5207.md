# 6・3・2 建設廃棄物処理技術

a.概要 建設廃棄物とは，建物の建築工事と道路建設や港湾
工事などに代表される土木工事のそれぞれから発生する廃棄物を指
.プラスチックの分別 プラスチックは，廃家電処理を行
す．建設廃棄物の発生量は，1999年度を例にとると，産業廃棄物
の全排出量の約2割，最終処分量においては約3割に当たる．すな
わち，建設廃棄物は産業廃棄物の大きな部分を占めており，発生抑
制と再資源化が強く求められていることがわかる．こうした状況を
踏まえて，2000年5月には「建設工事に係る資材の再資源化等に
関する法律（建設リサイクル法）」が公布され，2002年5月から施
行されている．この法律は，特定の建設資材を用いた一定規模以上
の建築物などの新築工事や解体工事に対して，分別排出や分別解
.熱処理による廃家電処理 これまでに述べてきた技術は
ことを目的としている．具体的には，建設廃棄物の大半を占める，
コンクリート，アスファルトコンクリート，木材の3種類について
再資源化を義務付けている．ここでは，建設廃棄物の代表的な発生
源，種類，処理技術を紹介する．なお，建設工事や土木工事では廃
棄物には分類されない発生物（そのまま原材料となる再生資源や建
設発生土など）もあり，これらと廃棄物をまとめて建設副産物と呼
ぶ．
b.廃棄物の発生源と種類
.建築工事 建築物としては，木造建築，RC（rein-
forced concrete）・SRC（steel framed reinforced concrete）建
築，コンクリート構造物があり，発生する廃棄物の種類に相違があ
る．また，建築工事は一般に，基礎・掘削工事，軀体工事，設備工
事，内装・仕上工事，外構工事の各工程からなっており，各工程で
発生する廃棄物にも違いがある．ここでは工程別に発生するおもな
廃棄物を示す．
(１) 基礎・掘削工事
建設汚泥，金属くず，伐採材の発生がある．量的には建設汚泥が
多い．建設汚泥とは，基本的に土に水を加えたもので廃棄物である
が，掘削した土砂そのものなどは廃棄物ではなく，建設発生土と呼
ばれている．汚泥と発生土の識別は，発生時点での含水率，土の粒
径，泥の状態の3点でなされる．工法の関係で発生時点で水が添加
された状態の泥は汚泥となる．
(２) 軀体工事
建築物の種類によって異なるが，おもに木くず，コンクリート，
金属くずなどが排出される．木くずとしては，型枠廃材や新築端材
がある．コンクリートとしては，工法の関係で発生するはつりや捨
てコンクリート，生コンクリートのポンプ車などの装置に残留する
余分なコンクリート，配管スリーブのためのコア抜き，強度試験の
テストピースなどがある．金属くずとしては，鉄骨や鉄筋に起因す
るものなどがある．
(３) 設備工事
おもに金属くず，プラスチックなどが発生する．配管工事や配線
工事などに起因する金属配管やプラスチック配管（硬質塩化ビニ
ル），廃電線などである．
(４) 内装・仕上工事
さまざまな廃棄物が発生する．梱包（こんぽう）材，ボード類，
断熱材，内装材などに起因するプラスチック材，木材，石こうボー
ド，軽量気泡コンクリート（ALC:autoclaved lightweight con-
crete）材，ロックウール，グラスウールなどである．建具取付け
や塗装に起因する金属くず，木くず，廃有機溶剤なども発生する．
また，これらの材料の梱包材や包装材に起因する衝撃吸収材のプラ
スチック，木枠，包装紙などもある．しかもこの工程では分別が困
難なものも多く，これらが混合された廃棄物は混合廃棄物（ミン
チ）と呼ばれる．
(５) 外構工事ほか
屋根材の陶器類やスレート，外壁のサイディング材などの端材，
それらの材料の梱包材なども廃棄物として発生する．塀や造園など
の工事からは，コンクリート，木くず，せん定枝などが発生する．
.土木工事 シールド工法に伴う建設汚泥，各種工作物の
建設工事に伴うコンクリート，道路工事からのアスファルトコンク
リート，造成工事に伴う木くず（伐採・除根）等がおもなものであ
る．
.解体工事 以上のⅰ項とⅱ項では新しい建設工事に伴い
発生する廃棄物を取り上げた．そして，これらの建設物は補修や改
造，解体されるときにはすべて廃棄物となる．特に解体時には，建
設した構造物そのものが廃棄物となるため，構造物本体に起因する
廃棄物が中心となる．木造建築では廃木材，RC・SRC建築ではコ
ンクリートがら，鉄骨くず，木くず，コンクリート構造物ではコン
クリートがら，鉄骨くずが主体となる．また，これらの構造物に伴
う設備類も廃棄物となる．
.解体工事に伴う有害物 解体工事において，着工前から
存在する法律に基づく処理が必要な廃棄物もあり，これらは着工前
に工事発注者が責任をもって実施する必要がある．
PCBを使用したトランス，コンデンサ，蛍光灯安定器は，PCB
を処理委託できる施設が完成するまで発注者が保管する義務がある

冷蔵庫，エアコン，洗濯機は家電リサイクル法に基づいて，発注者
が処理する義務がある．業務用エアコンや冷蔵庫・冷凍庫の冷媒フ

c .処理技術
.汚泥 建設汚泥は，脱水操作で土として再利用できる可
能性がある．脱水の困難な汚泥は，その原因が土粒子表面の帯電に
よる土粒子の凝集妨害と水分分離の困難性にあるので，この帯電を
中和すれば造粒と水分の分離が可能となり，良質土とすることがで
きる．この改良方法としては凝集剤の添加が考えられ，凝集効果を
もつアルカリ性の石灰質の添加による土質改良プロセスなどが実用
化されている．
