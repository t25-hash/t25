# 6・1・2 新交通システムの分類

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・1 一般

広義の新交通システム,すなわち都市交通向けに開発された新しい交通システムの分類を試みる.新交通システムは,既存の都市鉄道や路線バスなどと異なる方式の都市交通システムの総称であるから,極めて多くの方式のものが考えられる.それらの中から,これまで研究開発されてきたシステムを中心に形態別に分類してみる.分類としては,大きく分けて以下のようになる.(1) 輸送形態として・車両単位に輸送を行う:車両(または搬器)単位輸送型...略して車両型・連続的に輸送を行う:連続輸送型...略して連続型(2) 輸送モードとして・専用のガイドウエイ(線路もガイドウエイの一種)か道路のみを走行する:シングルモード(道路型・軌道型)・両方を走行できる:デュアルモード(dual mode)型(3) 動力源および駆動方式として・動力源を地上に置く:地上動力型...ロープけん引,ベルト駆動,地上一次リニアモータなど・車上に置く:車上動力型...回転型電気モータ,内燃機関,車上一次リニアモータなど(4) 走行形態(支持・案内)による分類として・車輪支持・空気支持・磁気支持(磁気浮上という用語は磁気支持と磁気案内の双方を含めた表現)・車輪案内・磁気案内・非接触案内(磁気ネイル式,光学式など)以上の分類方法に基づき,これまで開発されてきた新交通システムを分類すると,表6・1および6・2のようになる.表6・1,6・2では,備考欄に実用化の現状とシステムの特徴を簡単に注記した.6―155
