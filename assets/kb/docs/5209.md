# 6・1・4 リニアモータ駆動システム・磁気浮上式鉄道

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・1 一般

リニアモータ(linear motor)による推進は,レールと車輪間の粘着を利用しない非接触駆動のため,滑走や空転がなく低騒音なので,急こう配や急曲線が多く,環境性も要求される都市交通システムに適している.車両の支持・案内の仕方により車輪支持式,磁気浮上式等の方式があるが,現在,実用化が最も進んでいるのは鉄車輪式である.鉄車輪式リニアモータカーは,車両の推進・制動をリニアモータで,車両の支持・案内は鉄車輪/鉄レールで行うもので,車輪は用いているものの駆動・制動は粘着によらないので,急こう配や急曲線走行が可能といったリニアモータ推進の利点を発揮できる.また,リニアモータが偏平であるため,車両の低床化やトンネル断面の縮小化ができ,建設コストの低減を図ることができる.1985年,カナダのUTDC社が開発したシステムがトロントで開業した後,バンクーバーのスカイトレイン,デトロイトのピープルムーバとして開業している.日本では,トンネル断面積の縮小による地下鉄の低コスト化を目標とした研究開発プロジェクトが運輸省(当時),地下鉄協会等によって行われ,リニアモータ地下鉄(lin-ear system)は,車輪支持式に比べると開発要素が多いため,都市交通システムとしての本格的な実用化はやや遅れている.イギリス,バーミンガム空港のピープルムーバ,ドイツのM-バーンなどの運行は中止され,海外では,現在,運行しているシステムはほとんどなくなってしまったが,日本のHSST(high speed surface transport)は,2005年3月,愛知万博のアクセス交通機関ともなった東部丘陵線において実用化した.HSSTは,常伝導(電導)浮上式で車上一次リニアインダクションモータ方式のシステムで,名古屋市内に建設された実験線での走行試験の結果,1993年3月に公共交通機関として実用化できるとの結論を得,その後,地下鉄の終点駅藤が丘から愛知万博会場方面を経て,愛知環状鉄道線と結ぶ路線が建設された.磁気浮上・リニアモータ(linear motor)駆動のシステムとしては,超高速・都市間輸送向けで超伝導(超電導)誘導浮上型のJRマグレブ(MAGLEV : magnetic levitation)の実用化試験が,山梨実験線で進められている.1999年度までに基本的安全性の確認を終え,現在,信頼性,耐久性の確認とコスト低減のための試験を実施中である.また,リニアモータ駆動システムではないが,2005年には,自動車のITS 技術を利用し,専用のガイドウエイでは無人運転のバスが隊列走行を行い,一般道路では運転士による通常の走行を行うIMTS(intelligent multi-mode transit system)が,愛知万博会場内で磁気誘導式鉄道という位置付けで営業運転を行った.現在,このシステムをより実用的なシステムに発展させる研究が進められている.このほか,空気浮上のシステムとしてオーティス・リニアシャトルが,日本オーティス・エレベータの実験線での走行実験の結果,実用化可能との評価委員会の結論を得たが,現在のところ建設の予定はない.また,ジェットコースタの支持構造を応用して,重力とリニアモータを利用した,コストの安い短距離システムを実現しようという試みもある.
