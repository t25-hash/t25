# 6・1・5 ライトレールシステム(LRT : light rail transit sys

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・1 一般

tem)-LRT はlight rail transitの略称で、軽量で小型の車両を使用することにより、輸送需要があまり多くない路線において、比較的安価に利便性の高い都市交通システムを実現することができる。LRT の特徴は、乗降が容易な街路走行、自動車交通の輻輳している都心部での地下走行、高速で走れる郊外部での高架走行などを自由に組合わせることができ、かつ、必要に応じて段階的に地下化、高架化などを行っていける柔軟性にある。旧来の路面電車を改良しただけという見方もとれるが、超低床化など車両の近代化のみならず、地下化など軌道の近代化も図られており、新しい交通システムと考えるほうが妥当である。これまで、日本国内では、車両のみを新しくする例しか見られなかったが、2006年には、JR 富山港線の転用と新規の併用軌道による新しいライトレール路線が、富山市に開業する予定である。a. 超低床式LRV(light rail vehicle、軽量軌道車両)旅客の乗降を容易にするため、客室床面の超低床を図ったLRT 用の車両であり、ヨーロッパでは、各車両メーカ、各都市が近年競うように急速に導入している。LRT は、全線が専用軌道で鉄道のような専用の高いホームをもつ路線を除いて、街路面を走行する部分をもっているのが一般的である。このため道路面からステップを使わずに乗り降りできれば、身体障害者や高齢者はもちろん一般客にも利用しやすい乗物となる。超低床式(ultra low floor)LRV は、床面の高さが30cm程度しかなく、ステップなしで乗降することが可能である。通常の車軸のある鉄道の台車を用いていては、このような超低床化を図ることは不可能なので、独立車輪(independently rotatingwheel)を用いた車軸のない台車、ハブモータ等による車輪の直接駆動、VVVF(variable voltage variable frequency)方式による小型化された誘導電導機駆動などの新しい斬新な機構が取り入れられている(3・5・6項参照)。低床式(low floor)LRV は、以下の三つに分けられる。1連接車構造の台車と台車の中間の客室部を低床化した部分低床タイプ(低床化率は普通10%程度)2連接車の両端の台車をモータ付きの動台車とし、中間の台車をモータのない付随台車としてこの部分の低床化を図った半低床タイプ(低床化率60~70%)3動台車も含め、すべての台車を車軸のない独立回転車輪方式にし、客室の床面を100%低床化した全低床タイプ2005年には、技術研究組合での研究成果を利用して、国産型で初めての100%低床タイプのLRV が広島市で運行を始めた。
