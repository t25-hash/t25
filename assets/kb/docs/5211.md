# 6・2・1 一般

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・2 ゴムタイヤ式新交通システム(AGT)

ここでは、狭義の新交通システム(AGT : automated guide-way transit)、特にバスと鉄道の輸送力のすきまを埋めるものとして開発されたゴムタイヤ式中量軌道輸送システムについて述べる。これは急曲線や急こう配に強く、振動や騒音が少ないというゴムタイヤ車両の特徴を軌道系交通に取り入れたものである。1970年代の開発当初には、単車あるいは2~3両単位で連結された比較的小さい車両がネットワーク状に形成された軌道上を高頻度かつ目的地直行で自動運行するという、より自動車に近い形態のものが構想されていた。実用化の過程で、おもに経済性の制約から、直線やループといった単純な線形の軌道上を多数両連結された列車がシャトル運行するという、より鉄道に近い形態へと集約されていった。さらにわが国では、1983年になって当時の運輸・建設両省から「新交通システムの標準化とその基本仕様」が通達され、必要かつ最小限の基本方針として次の3方式9項目が定められた。(1) 方式・案内方式:側方案内方式(outboard guidance system)・分岐方式:水平可動案内板方式(lateral switch rail system)・電気方式:原則として直流750V(2) 車両・車両限界:図6・1参照・満車重量:18t以下(3) 軌道・建築限界:図6・2参照・案内面寸法:左右間隔中心高さ走行面より300mm・乗降場高さ:走行面より1070mm図6・1 車両限界および走行装置限界図6・2 建築限界および乗降場限界図6・3 設計荷重・設計荷重:図6・3参照この標準化によって、新交通システムの普及促進が図られることになったが、同時に中量軌道輸送システムとしての性格が決定付けられたといえよう。なお、コンピュータ管理による全自動運転は当初からのねらいであり、わが国においても約半数の路線で実用に供されているが、交通需要などによって多様な運転方式が想定されるため、標準化には織り込まれなかった。いっぽう、海外ではAGTの字義通り無人運転システムが定着しており、特に欧米では都市内交通や空港内の移動手段として数多く採用されている。新交通システムの車両はいわばゴムタイヤ式の電車であり、技術的には鉄道車両の応用といえるが、走行輪がゴムタイヤゆえ、台車に相当する部分には自動車技術に特有の機構なり構造なりが見受けられる。特に、ゴムタイヤそのものには転動方向を制御する機能が備わっていないため、荷重を支える走行輪とは別に水平の案内輪を設けて、荷重支持と誘導案内との協調をいかに実現するかに工夫が凝らされている。軌道についても、分岐装置を含めて、ゴムタイヤ車両の走行に適した設計が行われており、走行路、案内軌条、電車線などが設置される。走行路は一般的にコンクリート舗装が採用されるが、軽量化が要求される長スパンの橋りょう部などでは鋼製げたに合成樹脂系の舗装材を敷くことも行われている。軌道の形態としては、海外も含め、高架構造がほとんどであり、一部、地下の例もある。自動車交通との干渉を避け、全自動運転のもとで運行の安全を確保するという意味で有効であり、乗客にとっても快適性に通じる爽快感や展望の楽しみが期待できる。表6・3および6・4に、内外のおもな新交通システムの主要目を掲げる。また車両の例として図6・4に「ゆりかもめ」の形式図を示す。
