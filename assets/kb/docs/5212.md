# 6・2・2 案内方式

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・2 ゴムタイヤ式新交通システム(AGT)

6･2･1項でも述べたように，新交通システムの車両は走行輪とし
てゴムタイヤを採用している．ゴムタイヤでは，鉄道車両の車輪の
ようにフランジ付きタイヤを構成できないので，走行輪の転動方向
を規制し，車両を軌道に沿って誘導するための何らかの案内装置が
必要となる．これに対し通常，車両に水平に取付けた案内輪
（guide wheel）を軌道設備として設けた案内軌条（guide rail）に
沿わせることで案内する方式がとられている．この案内方式は，走
行路の左右両側に案内軌条を設けた両側案内方式（both-side guid-
ance system）と，走行路間の中央に1本の案内軌条を設けた中央
案内方式（center guidance system）に大別できる．中央案内方式
の場合，車両の床下機器のぎ（艤）装スペースを確保するために，
案内面は走行路面より下方の位置に配置されるのが普通である（図
