# 6・5・2 PCB処理技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・5 その他の先端技術

ガス化処理（gasification）技術 イオン交換樹脂中のを1000℃以上で酸化分解するプロセスである．
図6･20 にそのれた．
各技術の概要を表6･5 にまとめる．
化により生じる残渣はフィーダ出口で容器に回収される．
系元素を含む難燃性ガスが発生する場合には，あらかじめガスの一蒸気と有機ハロゲン化合物を反応させることで，ハロゲンを除去す中の炭素に対し残渣中の炭素の除去率は，イオン交換樹脂では国内では廃ポリ塩化ビフェニル（PCB熱媒油，トランス，コンデンサ等の容器・部材，感圧複写紙等）について処理技術が開発さ有機成分を500～700℃に加熱した水蒸気中でガス化し，このガスプロセスを示す．
ガス化はスクリューフィーダ型の移送機内で蒸気の流れとイオン交換樹脂の移行を対向させることで行われる．
ガスガス中の放射性核種は，フィルタにより除去される．
除去後のガスは主反応器中で，空気および高圧空気により酸化される．
ハロゲン部を空気量を低減させた状態で反応させ，その際の反応熱により水る．
脱ハロゲン後のガスは高圧空気雰囲気で酸化される．
