# 6・5・6 フロン処理（chloro- fluorocarbon removal ）技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・5 その他の先端技術

a.背景 フロンにはオゾン層破壊と地球温暖化という二つの
環境問題がある．オゾン層破壊に関しては，1985年のオゾン層保
護のためのウィーン条約において，CFC（chloro- fluorocarbon）
の規制の方向が打ち出され，1987年のオゾン層を破壊する物質に
関する第1回モントリオール議定書締約国会議にてCFCの全廃が
決議された．その後，1992年の第4回締約国会議において，CFC
全廃の前倒しとHCFCの全廃が決議され，これらの回収・再利
用・破壊を促進すべきことがはっきりした．第11回締約国会議
（1999年）では，先進国はCFC管理戦略の策定が求められ，環境
庁（現・環境省）が国家CFC管理戦略 をとりまとめた．また，
環境庁は処理技術の評価を実施し，1996年に策定したCFC破壊処
理ガイドラインを1997年に改訂 した．また，通商産業省（現・
経済産業省）も2000年，特定フロン等の破壊技術に関する手引
き をまとめた．そして，2002年2月，環境省と経済産業省が共
同でフロン類の破壊に関する運用の手引き（第1版) を発行し
た．また，地球温暖化に関しては，第3回気候変動枠組条約締約国
会議（地球温暖化防止京都会議，COP3）（1997年）にて，温暖化
に影響があり，削減すべき物質としてHFC，PFCがその対象とな
った．こうした背景を踏まえて，わが国では1988年に「特定物質
の規制等によるオゾン層の保護に関する法律（オゾン層保護法）」
が制定され，CFCの生産の規制が始まった．また，2001年からは
特定家庭用機器再商品化法（家電リサイクル法）が施行され，家庭
用の冷蔵庫とエアコンの冷媒（CFC，HCFC，HFCが中心）の回
収が義務付けられた．そして，2002年からは「特定製品に係るフ
ロン類の回収及び破壊の実施の確保等に関する法律（フロン回収破
壊法）」が施行され，業務用冷凍空調機器やカーエアコンに使われ
ているCFC，HCFC，HFCの回収と，再利用または破壊が義務付
けられた．このように，生産側と回収・破壊処理側の両面での規制
が進行している（カーエアコンに関しては「自動車リサイクル法」
に引き継がれた)．
b.発生源と種類 フロンとは，炭素，フッ素，塩素，水素か
らなる有機化合物である．炭素，フッ素，塩素からなるものをクロ
ロフルオロカーボン（CFC），これに水素が加わったものをハイド
ロクロロフルオロカーボン（HCFC），炭素，フッ素，水素からな
るものをハイドロフルオロカーボン（HFC）と呼ぶ．オゾン破壊
に関しては塩素が原因物質となるためCFCとHCFCがおもな対象
で，特にCFCについては特定フロンと呼び，先進国における生産
は1996年に停止されている．地球温暖化に対しては，これに加え
てHFCが対象となる．HCFCとHFCを代替フロンと呼ぶ．この
ほかに炭素，フッ素からなるパーフルオロカーボン（PFC）があ
り，COP3ではHFCとともに対象物質に指定された．フロンのお
もな用途は洗浄剤，冷媒，発泡剤，噴射剤であることから，これら
の用途で使われた製品の使用済み後が発生源となる．
c .処理技術 環境省と経済産業省によるフロン類の破壊に関
する運用の手引き によれば，フロン類（CFC，HCFC，HFC）
を対象とした処理技術の破壊能力として，分解率が99 %以上であ
りかつ最終排ガス中のフロン類の濃度が1ppm以下であること，
または，分解率が99.9 %以上でありかつ最終排ガス中のフロン類
の濃度が15ppm以下であることが要求されている．そして処理技
術の具体例として，廃棄物混焼法方式，セメント・石灰焼成炉混入
法方式，液中燃焼法方式，プラズマ法方式，触媒法方式，過熱蒸気
反応法方式を取り上げ，施設の構造，運用方法のあり方をまとめて
いる．また，国連環境計画（UNEP:United Nations Environ-
図6･ 36 コロナ放電によるNO分解特性例
ment Programme）では世界中45種類の技術を評価して，次の12
種類の技術を推薦している．それらは，リアクタクラッキング法，
ガスヒューム酸化法，ロータリキルン法，液体注入法，都市ごみ焼
却炉法，セメントキルン法，アルゴンプラズマ法，高周波プラズマ
法，マイクロ波プラズマ法，窒素プラズマ法，気相触媒法，過熱水
蒸気反応法である．以下にこれらの技術を燃焼技術，非燃焼技術，
プラズマ技術に分類して，国内で実用化しているものを中心に紹介
する．
.燃焼技術
(１)ロータリキルン法
産業廃棄物処理で用いられている既存のロータリキルン（rotary
kiln）型焼却炉を利用する方法である．耐火れんがで内張りされた
鋼鉄製の円筒容器（キルン）を軸を水平からやや傾斜させて回転さ
せることで，ごみを移動させながら加熱する．そこで発生するガス
を二次燃焼室で焼却する．フロンはおもにこの二次燃焼室で，
850～1200℃で焼却される．フロンを内包する材料をそのまま処理
できる可能性があり，断熱材などはフロンを分離回収しなくても処
理できる利点もある．フロン処理の生成物である塩化物，フッ化物
の処理を確実に行うことが求められる．
(２)都市ごみ焼却炉法
やはり産業廃棄物処理で用いられている既存の立て型溶融炉，固
定床炉，移動床型のストーカ炉などの都市ごみ焼却炉を利用する方
法である．いずれもごみは一次燃焼室でおもに熱分解され，発生す
るガスを二次燃焼室で焼却する．立て型溶融炉は炉の下部が
1500℃以上の温度となるので，ここにフロンを導入し分解する．
その他の炉では，フロンは二次燃焼室で処理される．特に固定床炉
やストーカ炉では炉内温度が850～950℃程度なので，灯油の助燃
により加熱して高温を維持するなど，より厳密な温度管理が要求さ
れる．また，フロン処理の生成物である塩化物，フッ化物の処理を
確実に行うことも求められる．フロンを内包する材料からフロンを
取出すことなくそのまま処理できる可能性がある．
(３)セメントキルン法
セメントの製造装置である既存のキルンを利用して処理する方法
である．セメントキルンは1500℃程度の温度で運転されているの
で，ここへフロンを注入して熱分解する．分解生成物の塩化物もフ
ッ化物もセメント原料のカルシウムと反応してセメント製品の中に
混入させることができるので，排ガス処理に特別な配慮がいらない
ことが利点である．しかし，セメント製品中の塩素含有量に制限が
あるため，これが処理量の制約条件となる．また，フロン内包材料
では，フロンのみを分離回収する必要がある．
(４)石灰焼成炉法
生石灰製造用の既存の焼成炉を利用して処理する方法である．石
灰焼成炉の高温部は約1200℃あるので，ここでフロンを分解する．
しかし，導入できる塩素量には限界があり，これが処理量の制約条
件となる．また，フロン内包材料では，フロンのみを分離回収する
必要がある．
(５)液体注入法
燃料を用いる燃焼炉に，液体状または気体状のフロンを注入して
火炎の中で燃焼させる方法である．燃料そのものにフロンを混入さ
せる場合と，炉内の火炎にフロンを導入する方法がある．
(６)液中燃焼法
燃焼部の原理は液体注入法と同じであるが，液体を入れた容器の
中にバーナまたは燃焼室を設け，そこで燃料や空気とともに燃焼
し，排ガスを燃焼直後に液体と接触させる方法である．酸性の排ガ
スを発生直後に中和処理できること，排ガスの熱が固体壁を通さず
に熱交換（冷却）できることなどから，腐食性の生成物の発生が避
けられないフロンの処理には適した方法である．
(７)ガスヒューム酸化法
高温水蒸気法とも呼ばれる．LPGなどの燃料を燃焼させて高温
（1200～1250℃）に維持した燃焼炉にフロンと水蒸気を吹き込んで
高温熱分解する．原理的には液体注入法と同じであるが，水蒸気を
導入し加水分解反応を利用する．
(８) リアクタクラッキング法
(３)熱容量が大きい．
酸素と水素による燃焼で，約2000～2200℃の高温の火炎を形成
し，その中へフロンを注入して分解する．原理的には液体注入法と
同じであるが，酸水素炎を利用し，燃焼生成物が水中心となる．こ
の分解処理で得られる排ガスから55%のHFと33%のHClを回収
できる．ドイツで開発され，UNEPの推薦技術であるが，国内で
の実績はない．
.非燃焼技術
(１) 気相触媒法
触媒反応を利用して水蒸気存在下でフロンを加水分解する方法で
ある．予熱器で水蒸気とフロンを混合し，約400～500℃に加熱さ
に述べる．
れた触媒層を通すことで，加水分解する．触媒としては，二酸化チ
タン（TiO ）やリン酸アルミニウム（AlPO ）を使う方法などが
ある．PFCをアルミナ系の触媒により約750℃の温度条件で分解
する技術も開発されている．
(２) 過熱水蒸気反応法
過熱水蒸気を利用する点ではガスヒューム法に似ているが，反応
器内ではフロンと水蒸気と空気を接触させるだけで，燃焼はさせ
ず，熱分解反応のみを利用する．ガスヒューム法よりも低温の
800～1000℃程度を反応温度とする．
(３) 化学的熱分解法
600～1000℃に加熱された反応器内で，ケイ素系の薬剤との反応
によりフッ素が四フッ化ケイ素となる反応を利用して分解する方法
である．排ガスはアルカリ洗浄され，最終的にはフッ素はCaF
に，塩素はNaClとなる．
