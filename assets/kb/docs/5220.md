c .原動機 移動型の建設機械の場合，一般に大型の機械はデ
ィーゼルエンジン，ハンドガイド型（walk behind type）や小型の
搭乗機械はガソリンエンジンを用いている．トンネル，地下，都市
型土木の分野で排気ガスや騒音などの環境問題に対処するために，
表6･7に採用の得失を示した電動機も採用されている．
電動機は，電源の確保の容易な定置式機械（骨材プラント，コン
クリートプラント，ケーブルクレーンなど）のほか，場内の換気設
備の負荷を軽減するためにトンネルや立坑の掘削で採用されてい
る．管埋設など都市土木での静粛な施工を目的に，電源ケーブル巻
込み型電動建機も検討された．
建設機械の電動機の型式や出力などの選定に際しては，①交流か
直流か，②周波数・電圧，③馬力・トルクなど負荷の大きさと変
動，④建機の主動力，⑤作業装置・ポンプ（油圧・水など)・コン
プレッサなど負荷の種別，⑥直結・クラッチ（cluch）・流体継手
（fluid coupling）・歯車・ベルトなど負荷との取合い，⑦温度・湿
度・腐食性や起爆性ガス・じんあいなど稼動環境，⑧手動運転・自
動運転・遠隔操作など運転の条件，などを考慮する必要がある．
往復運動がなく効率的で使用燃料の幅が広いガスタービンの使用
も検討された．さらに未来型システムとして蓄電型電動建機，高圧
の水素ガスと酸素が清浄な燃焼をする水素エンジン（hydrogen-
fueled engine）なども視野におさめる必要がある．
