# 6・5・2 国内で使用されている一般的な索道

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・5 索道

索道の形式については、6・5・1項の索道の種類と方式を組合わせ、索条の使用目的別に単線、複線、複式を冠して呼んでいる。国内で使用されている一般的な索道の形式としては、以下のものが挙げられる。a. 複線交走式普通索道複線交走式普通索道は、起点停留場と終点停留場の間に張られた支索に、懸垂されたそれぞれの閉鎖式搬器をえい索でつなぎ、えい索を動かすことで、それぞれの搬器は交互に行き交い、同時に起点、終点停留場に停止する方式の索道(図6・21)である。この方式は、従前から多く建設されロープウエイと呼ばれている。日本では大型のもので166人乗りが建設されている。また、2階建の搬器で運送するものもある。b. 単線自動循環式普通索道単線自動循環式普通索道は、起点停留場の滑車と終点停留場の滑車間の支えい索を、閉鎖式搬器の握索装置が停留場で自動的に放索または握索する方式の索道(図多数の搬器が一定の間隔で握索して循環するため、輸送量は大きい。停留場には、到着した搬器を支えい索から離すための装置と、放索後に旅客を乗降車させながら反対側へ移動させるための押送装置、乗降車が完了した後に支えい索を再び握索させるための装置が設けられるのがこの方式の特徴である。図6・23 24人乗りDLM図6・24 DLM の構造c. 単線固定循環式特殊索道単線固定循環式特殊索道は、起点停留場の滑車と終点停留場の滑車間の支えい索に固定されたいす式搬器を循環させる方式の索道である。スキー場で運転されるものがスキーリフト、夏山や山岳地のレジャー施設として運転されているものが夏山リフトや登山リフトと呼ばれている。単線自動循環式特殊索道とともに、一般に搬器の乗車定員により、1人乗りをシングルリフト、2人乗りをペアリフトまたはロマンスリフト、3人乗りをトリプルリフト、4人乗りをクワッドリフトなどと呼んでいる。d. 単線自動循環式特殊索道単線自動循環式特殊索道は、起点停留場の滑車と終点停留場の滑車間の支えい索を、いす式搬器の握索装置が停留場で自動的に放索または握索して旅客を運送する方式の索道で、施設は単線自動循環式普通索道と同じで、おもな違いは搬器の構造にある。一般にデタッチャブルリフトまたは高速リフトと呼んでいるが、単線固定循環式特殊索道と同様に搬器の乗車人員による呼び方もされる。
