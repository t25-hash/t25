# 6・10・1 建設分野でのロボットの活用

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・10 建設ロボット（robot for construction work）

建設ロボットとは，①周辺の状況を判断して最適な動きを自律的
に決定し行動する機械，②あらかじめ設定した手順に則って動くシ
ーケンス制御型機械，③直接または離れた所から操縦できる建設機
械や機械搭載の作業装置の総称である．活用分野として建設作業，
計測，維持管理がある．表6･18に土木・建築ロボットの分類と定
義例を示した．
自律型建設機械（autonomous construction machinery）は自ら
作業環境の認識と最適化の判断をして移動や作業を行う．実用段階
で全操作を自律化したものはない．しかし地下や水中施工機械など
オペレータが視認できない部分をセンサとコンピュータの判断に委
ねる部分的な自律機能をもつ機械はすでに存在する．機械の運転操
作の効率化や安全性確保のために，センサやコンピュータシステム
を用いた最適操縦システムも開発された．複数機械での施工時の最
適協調行動自律化の研究も行われている．
トンネルボーリングマシン，シールド掘削機，ケーソン等立孔掘
削システム，地下連続壁掘削機，捨石均し機械などはプロセス制御
型の自動化システムを用いている．締固め機械やダンプトラックの
無人走行もあらかじめ設定した手順で動くシステムである．遠隔操
縦型の建設機械は製鉄所，鉱山，水中作業など危険・苦渋な現場で
使用されていた．揚上機械やコンクリートポンプ車など機械と作業
装置操作を一人で行うための運転補助システムも開発された．
1992年の長崎県雲仙普賢岳の噴火に伴う火砕流災害の復興工事
は安全対策として，全作業をIT＋ロボット技術で施工する無人化
施工（unmanned construction system）が採用された．遠隔操縦
型建機（remote controlled construction machinery）を中心に，
締固めや搬送に半自律型の機械を投入した新工法である．以来10
年間，全国で50件を超える無人化施工が実施されている．
全作業の機械化が必要な無人化施工では，標準型のラジコン建機
で行ってきた，掘削，集土，積込み，運搬などに加え，従来人手を
介していた把持，揚上，設置，整形，出来形計測などの遠隔施工を
実現した．事例として，①遠隔操縦建機で整形した土砂を型枠とし
