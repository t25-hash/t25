# 6・5・3 新しい形式の索道

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・22 )である.一般にゴンドラまたはゴンドラリフト等と呼ばれ,

ヨーロッパ等で運行されている新しい形式の索道がいくつかある.複式単線自動循環式普通索道,複線自動循環式普通索道といった索道がこれである.複数のロープで搬器が支持されていることで,耐風性の高い構造をしていると考えられる.日本国内でも一部導入が始まっているこれらの索道の概要は,次のとおりである.a. 複式単線自動循環式普通索道複式単線自動循環式普通索道は,複数の支えい索をそれぞれ握索した閉鎖式搬器が移動する方式で,そのうちDLM(double loop monocable,図6・23およびした方式の索道である.また,DMC(double monocable)は,エンドレスに張られた2条の支えい索を架設し,循環させる方式の索道である.b. 複線自動循環式普通索道複線自動循環式普通索道は,起点停留場と終点停留場の間に固定された支索上を,閉鎖式搬器が移動し,循環する方式の索道である.えい索が1条であるか2条であ図6・25 30人乗り3S システムロープウエイるかによって,1支索1えい索式,1支索2えい索式に区分され,支索2条とえい索1条で構成されている方式は,3S システム(3seil system,図6・25)といわれている.最近まで,この種類の索道は,技術基準の関係で国内に導入することが困難な状況にあったが,1997年5月,技術開発の促進,海外で開発された機種の導入の容易化等を図る観点から,この技術基準の大幅な改正が行われた.これにより,耐風性の高い構造をしていると考えられる上記のような索道の導入が容易になると考えられ,風の特に強い地域では,大きな期待が寄せられている.
