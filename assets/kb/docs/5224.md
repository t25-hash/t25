# 6・10・4 建設ロボットの今後

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・10 建設ロボット（robot for construction work）

今後は，ロボット導入による危険・苦渋作業の回避や，経済効果
を実証するためにロボット化すべき施工分野と開発目標を明確にし
て取組む必要がある．ロボット化が期待される施工現場として，災
害救援・復旧，高所作業，長大法面，狭隘現場，高温・多湿現場，
粉じん・悪臭・有害汚染物質が存在する現場などがある．特に，焼
却場や原子力施設の再開発，汚染土壌の浄化工事などのロボット化
は次世代に課せられた課題である．建設投資の抑制と循環型社会へ
の転換で維持管理ロボットへの期待も大きい．
人間の腕（ブーム&アーム)，手（ハンド）の形をした作業機構
をもつ油圧ショベルがはん用建設機械の過半を占めるようになった
いま，その手を自由に動かしてさまざまな作業を行う建設ロボット
の実現も期待される．
図6･118無線LANを用いた無人化施工（システムテクニカル株式会社）
超高速無線LANを用いた機械の遠隔制御，現場映像，機械や施
工データなどを一元的に伝送することによって工事全体を一つのシ
ステムとして高度な情報化施工を実現する試みが始まっている．
