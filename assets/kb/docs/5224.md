# 6・6・2 エスカレータ(escalator)

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・28 ).油圧エレベータはその構造上高速・長行程には適さない

エレベータが間欠輸送であるのに対し,エスカレータや動く歩道は連続輸送で,待ち時間がなく,多数の乗客の輸送に適している.このため建物内だけでなく,駅舎,歩道橋,各種展示場など利用者の多い場所で使用される.エスカレータは踏段間に段差があるのに対し,動く歩道は踏段間に段差がなく,表6・12に示す違いがある.a. エスカレータの構造図6・30にエスカレータの構造例を示す.上部機械室に電動機や減速機で構成する駆動機があり,駆動機と上部ターミナルギヤは駆動鎖で連結される.上下のターミナルギヤに踏段鎖を掛け,これに踏段(step)が取付けられる.踏段は4個の踏段ころで支持され,踏段面を水平に保つように配置された踏段レール上を走行する.エスカレータのこう配と速度は表6・12に示すとおりであり,踏段の幅は1.1m以下,両側の欄干の移動手すりは踏段と同方向に同一速度で移動するように規定されている.踏段の幅は0.6mと0.8mのものが多く,速度30m/minの場合の公称輸送能力は,前者が4500人/h,後者が9000人/hである.踏段の積載荷重(N)は,=2600で計算した数値以上としなければならない.ここで,は踏段面の水平投影面積である.構造部材の安全率はその使用状況に対応して表6・13のように規定されている.このほかに,高低差が高い場合の中間踊場付エスカレータ,車いす利用者も利用できる車いす用ステップ付きエスカレータや曲行するエスカレータなど,特殊な構造のものもある.表6・12 エスカレータと動く歩道こう配θ定格速度V備考θ≦8°V ≦50m/min動く歩道θ≦15°で踏段が水平でないV ≦45m/min8°<θ≦30°踏段が水平エスカレータ30°<θ≦35°V ≦30m/min図6・30 エスカレータの構造例6―172交通機械表6・13 エスカレータの安全率常時安全装置作動時置かれた状態部材設置時使用時設置時使用時鋼製その他の金属製の踏段3.0―2.0―鉄骨造りのトラスまたははりの鋼材の部分3.0―2.0―踏段をつる鎖その他これに類するものおよびその端部7.04.02.52.5ベルト7.04.04.02.5b. 安全装置(safety gear)エスカレータでは利用者が固定物と移動物の両方に接するので,そのすきまに引き込まれたり挟まれたりする危険性が高い.このため以下の安全装置の設置が義務付けられている.(1) 制動装置(電磁ブレーキ)動力が切れた,駆動装置が故障した,人や物が挟まれたなどの場合に,加速度1.25m/s以下で踏段の昇降を停止させる.(2) 非常停止ボタン上下の昇降口の近くで,できるだけ見やすく容易に近づける位置に設ける.(3) 踏段鎖および駆動鎖安全スイッチ駆動鎖や踏段鎖が切断したあるいは異常に伸びた場合にこれを検出して停止させる.(4) スカートガード安全スイッチ人または物が踏段側面とスカートガードとの間に強く挟まれた場合にこれを検出して停止させる.(5) インレットスイッチ人または物がハンドレールの入込口に入り込んだ場合にこれを検出して停止させる.(6) 保護板手すりに近接して建築物の天井やはり,他のエスカレータの下面等と交差する場合に,それらとの間に頭部等を挟まれないよう,利用者に注意を喚起する.c. 動く歩道動く歩道は踏板面に段差のない特殊なエスカレータで,踏板の構造によりパレット式とゴムベルト式がある.こう配と速度の関係は表6・12に示すとおりで,踏板の両側には同一方向に同一速度で移動する手すりを設ける.強度部材の安全率や安全装置はエスカレータに準じて計算あるいは設置される.移動距離が長くなると移動時間が長くなるので,これを短縮するために速度が途中で変わる可変速式動く歩道もある.
