# 6・7・1 ガイドウエイバス(guideway bus)

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・7 その他の新しい交通システム

わが国には,新交通システムを導入するほどではないが路線バスでは対応しきれないような中間的な交通需要をもつ地域も多い.1983年の新交通システムの標準化に当たって,そのような地域には軌道構造物いわゆるインフラ部分を先行整備し,十分な需要が見込まれるまで簡易な案内装置を取付けたバスを走らせるという,段階整備の構想が示され,ガイドウエイバス開発の契機となった.このシステムを特徴付けるものは,道路と軌道の両方を走行できる,いわゆるデュアルモード(dual mode)機能であり,一般の路線バスに新交通車両と同様の案内装置を取付けることで,軌道上はハンドル操作なしに走行できる.これにより路線バスの簡便さと,軌道系交通の安全性や確実性を併せ持つことが可能になった.ガイドウエイバスの開発は1985年から官民共同で開始され,1989年の福岡市アジア太平洋博覧会での運行を経て,2001年3月に,名古屋ガイドウエイバスの志段味(しだみ)線が開業した.同線は名古屋市の東北部にあって,志段味地区の開発に伴い新たに発生する交通需要への対応と,流入する都心部での交通混雑の緩和のため,わが国初の実用路線として導入された.高架区間には軌道法が適用され,全長約6.8kmに9駅が設けられている.表定速度は30km/h,最高速度は60km/hで,25両の車両が最小3分の間隔で運転されている.図6・31 ガイドウエイバス車両とモードインタチェンジ図6・32 案内装置起点の大曽根駅には,端末折返しのための転回場が設けられ,運転士のハンドル操作と併用して車両の入替えが行えるようになっている.ただし,ここには一般道路への出入り口はない.いっぽう,図6・31に示すように,小幡緑地の先,軌道と道路の境界部には,車両確認や案内装置の伸縮などを行うためのモードインタチェンジ(MIC: mode interchange)があり,登録車以外の進入を阻止できるよう遮断機や監視カメラなども設置されている.車両は図6・32に示すような2.5m幅の伸縮式案内装置を備えている.シャシについては2車種あるが,前車軸を共通化して同じ案内装置を用いている.なお,開業時点で運用される車両はディーゼルエンジン駆動の一般低床バスであるが,今後,低公害化やノンステップ化が図られることになる.軌道は全線複線高架で,走行路はコンクリート製,案内レールはH 形鋼製のシンプルな構成となっている.駅はすべて相対式であり,エレベータ,エスカレータと階段によってホームと地表とを結んでいる.ホームには安全柵,接近案内表示器,監視カメラなどが設けられているが,運賃収受については車内で行うことを基本にしており,運賃設備は端末の大曽根駅にのみ設けられている.MICに併設する形で本社建家と簡易整備場が設けられている.本社には,運転指令室があり,車両の運行やホームの状況,施設の稼働状態,気象などを監視し総合的な運行管理を行うことができる.さらに,MICでの車両の進入出を監視し,出発の指示など簡単な運転整理も行えるようになっている.なお,軌道法のもとで60km/h運転が認められているが,無軌条電車の運転規則が準用され,軌道には特別な信号設備はない.MICの一般道路への出入り口には,車両の運行に連動した交通信号機が設けられ,一般道路ではPTPS(公共車両優先システム)図6・33 スカイレールの基本構成表6・14 スカイレールの主要諸元路線営業キロ1.3km最急こう配27%駅数3駅最小曲線半径100m高低差160m輸送力(開業時)600人/h (成熟時)1200人/h車両(開業時)6両(成熟時)13両定員25人最大乗車人員37人空車質量2.5t最大質量4.72t運行方式全自動運行運転速度18km/h最小運転間隔75s主要設備主駆動装置直流モータ300kW/基ロープ異形線ストランド6×WS(26)外径φ28mm加減速装置地上一次LIM45kW/個車両用電源AC100V 1φ剛体電車線を用いて運行の効率化が図られている.なお,海外では,オーストラリアのアデレード市や,ドイツ,イギリスなどで同様のシステムが運行されている.
