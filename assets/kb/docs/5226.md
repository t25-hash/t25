# 6・7・2 スカイレール(Sky Rail)

> **章**: 第6章 その他の軌道交通システム
> **節**: 6・7 その他の新しい交通システム

短いとはいえ徒歩では抵抗感のある距離や上下の移動は従来の軌道系システムに不得手の領域であり、より使いやすく小回りのきく、いわゆる短距離交通システム(short haul transportation sys-tem)の実現が期待されている。スカイレールはこのようなシステムの一つであり、ロープ駆動と懸垂式車両とを組合わせたユニークな形態をもち、急こう配に強く、導入空間が限られた地域での輸送手段として開発された。1998年8月、その実用線として広島市安芸区のみどり坂ニュータウンに広島短距離交通瀬野線(通称:スカイレールみどり坂線)が開業した。同線は、延長わずか1.3kmであるが、急傾斜地にある住宅街と最寄りのJR 駅とを結び、25人乗りの小型車両が27%のこう配を75秒間隔で自動運行することができる。システムの基本構成を図6・33に、主要諸元を表6・14に示す。図術は単線自動循環式ロープウエイと懸垂式モノレールである。軌道げたはローマ数字のIIに似た箱型断面である。軌道げたの上に張りわたされたロープはつねに一定速度で循環駆動されており、車両は駅の出発、到着の際に握放索を行う。軌道げた、台車、握索機の関係を図6・34に示す。握索爪はシーブ(索輪)から離れた位置に台車によって保持されるため、握索中はつねにロープを引き出した状態で走行でき、シーブに接触することはない。駅部ではロープを放し、地上一次式リニア誘導モータ(LIM :linear induction motor)によって加減速走行する。軌道げたの下には一次コイルが設けられ、リアクションプレートは車両の台車上に設けられている。各コイルの間には非接触式の車両位置検出センサがあり、つねに車両の位置をモニタすることによって、コイルごとのオンオフ制御を行うとともに、検出位置の移動から車両の走行6―173図6・34 軌道、台車、握索機の関係図6・35 スカイレールみどり坂線速度を演算している。なお、最終段の停止にはLIM でなく機械ブレーキを用いている。駅間はロープで走行するので追突の心配はなく、ロープ駆動系の保安システムもロープウエイと同じである。駅内はLIM で制御されるため、鉄道でいう閉そくの考えを取り入れ、車両検知器とリレーロジックによる追突防止システムを構成している。運行管理システムは駅内、駅間の両保安システムを統合して監視、制御するものであるが、必要に応じて指令員による介入操作もできる。車両、駅および沿線の状況は、運行管理所から集中監視制御される。スカイレールにはいわゆる車庫はない。両端の駅構内にループ状の折返し部を設け、車両留置にも兼用している。通常運行中は本線路内にあるすべての車両がつねに順繰りに移動しつつ運用される。また、折返し部には分岐器が設けられ、保守点検などの際に車両を待避線に引き込めるようにしている。各駅は無人化されていて、車両ドアと連動するホームドアが設けられている。また、自動改札機は各駅の乗車口のみに設けられており、切符はその場で回収し、降車口はフリーである。定期券には、わが国の鉄軌道分野としては初めて非接触式ICカード(contact-free IC card)が導入された。いずれも簡素化と保守の容易化をねらったものである。スカイレールは、小型車両を単車で自動運行させることから、多6―174交通機械客時にも即座に高頻度運転で対応できるという特徴をもつ。そこで、同線では待受け運転と称して、閑散時に車両をつねに両端駅に待機させておき、乗客の到着を運行管理所のモニタで確認してドアの開閉や出発の操作を行うという、デマンド運転に近いサービスを提供している。このような運行の柔軟性も短距離交通システムに期待される要件の一つである。図6・35にスカイレールみどり坂線の沿線風景を示す。文献(1)
