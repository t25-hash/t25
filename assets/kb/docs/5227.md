# 1・1・1 語源

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

ロボット（robot）という言葉は，1920年にチェコの作家カレ
ル・チャペックが書いた戯曲『R.U.R.(ロッサム・ユニバーサ
ル・ロボット)』に使われたrobotaに起源をもつ．Robotaはチ
ェコ語で労働を意味する語であり，それに対する英語がrobotで
ある．この戯曲は著名なジャーナリストでもあったチャペックが，
当時の解剖学を背景とする臓器保存技術の進展に対する危惧と，資
本主義体制の進展により工場の機械化が進み，人が非人間的に機械
に使われて働く状況が見られるに至り，社会を風刺する意味で書い
たものである．ロボットは化学プロセスによって作られ，人に忠実
に仕える人間のように働く機械として描かれた．
戯曲は好評で当時は何回も上演されたと聞くが，戯曲が上演され
なくなってからもロボットという言葉だけは残り，社会で日常的に
使われるようになった．その使われ方は，人のように働く高度な機
械や高度に自動化された機械，あるいはオートメーションの代用
語，また機械のようにしか働かない融通性のない人を揶揄する言葉
としてなど，さまざまである．後述するように，このさまざまな使
われ方が，後にロボットが学術的に取り上げられるようになったと
きにその定義にも混乱を引き起こす原因となった．
一方，ロボットがいわゆる技術の分野で正式名称として使われた
のは1960年代の初めである．1962年発行のアメリカの業界誌
American Metal Marketで，アメリカの2企業ユニメーション
（Unimation）社，AMF社から初めて発売され，現在では工場の
自動化に普及している教示再生型ロボットの商品であるユニメート
（Unimate），バーサトラン（Versatran）が産業用ロボット（in-
dustrial robot）として紹介されたのが，技術的な分野でのロボッ
トという用語の使用の始まりといわれる．その翌年，この用語はロ
ボットとともに日本に輸入され普及した．はじめはその訳語として
工業用ロボットが使われたが，その後，工業よりもより広く産業全
般に波及する技術であるとの認識と期待から産業用ロボットが正式
訳語となり，今日に至っている．
