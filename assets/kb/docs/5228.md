# 1・1・2 ロボットの定義

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

上述したようにロボットという語はまず戯曲作家の造語であり，
その戯曲のイメージのもとでさまざまな使われ方がなされてきた．
科学技術の分野においても例外ではなく，さまざまな解釈がある．
産業用ロボットの分野では，教示再生型ロボットという具体的な機
種にその名前がつけられたこともあり，JIS B0134：1998や
ISO/ TC184/ SC2/WG1に明確な定義がある．しかし，現在の
ロボットの学問がカバーする領域を考えるとそれは極めて狭い定義
であり，学術分野でのロボットの定義としては適切とはいえない．
要するに学術的にロボットとは何かと問われたときに，現状では
統一的な回答が存在しない．ロボットの研究者各人が自らの有する
定義のもとで，研究開発を進めているのが実状である．
しかし，こうした混乱した状態の中でも，ロボット技術に従事す
る関係者の間で漠然とした合意が得られている．それによればロボ
ットとは，「何らかの人間あるいは生物的な特徴を備えた，実世界
に作用を及ぼす能力をもつ機械」ということであろう．
チャペックの戯曲のロボットはいわゆる人造人間であるが，人の
形をしていることは問わないまでも，従来の機械に比べて，人がも
つ作業機能，知的機能を工学的にできるだけ模倣して構築された，
実世界で行動する機械あるいは機械システムをロボットとする傾向
がある．また，ロボット技術においては，人のみならず昆虫，動物
など生物全般をも模倣の対象とするのが一般的である．
