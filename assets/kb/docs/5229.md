# 1・1・3 ロボット研究の動機

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

戯曲で描かれた,またその後のサイエンスフィクション(SF:science fiction)などで描かれたロボットを具体的に科学として追7―1求する試みが生まれたことにより今日のロボット技術が存在するわけであるが,それではわれわれはなぜロボットを研究する,あるいはしてきたのであろうか.そこにはさまざまな動機が存在する.まず,最も単純な動機は,SFや人および生物の機能に触発されて,そのようなものが科学的に構築できないかという興味本位の動機である.こうした試みは古代ギリシャの時代から存在し,蒸気によって駆動する人間の構想や,機械仕掛けで動く精巧なからくり人形などを生み出した.しかし,これらは科学というよりも職人の世界での試みで,学術の体系を後世に残す学術的営みとは異なる.重要かつ考慮すべきは,科学的および工学的とも呼ぶべき二つの動機である.前者は,人や生物の機能を解明するためにロボットを研究するという立場である.例えば,人の脳機能や人・生物がもつ知能の解明は,人や生物の未知の機能を知るという純粋な科学的願望を満たすための中心課題の一つである.また,脳機能の解明は医学の進展にも貢献することが明らかで,さらにはそうした従来の人工物にはない生物の優れた機能が技術によって構築できれば現状の多くの機械やシステムの性能を格段に改善できることから,工学的にも興味深い課題である.従来,脳機能の研究は生理学,生物学の中で脳単体を扱う学問として,また知能に関しては人工知能(AI:artificial intelligence)としてコンピュータ科学の一部でソフトウェアや論理的アルゴリズムの研究として行われてきた.しかし近年,脳や知能はそれを備えた人・生物の個体の形状・挙動とも密接な関係があること(身体性)が指摘され,脳の生理学的分析や人の知的行動から明らかになった脳機能や知能にかかわる知見を,人や生物に類似した具体的な工学モデルの中に実装し評価するということが試みられるようになり,こうした研究の中でロボットが重要な役割を果たしつつある.また,人や生物の巧妙な仕組みの解明において,その挙動などの観察からその仕組みに対しある種の仮説を立て,それに基づいて具体的な人や生物の機械モデルであるロボットを構築し,その挙動と元の人や生物の挙動とを比較評価し,その結果から仮説を検証して生体の仕組みを探るということも試みられている.この方法は生体機能解明に対する構成論的アプローチとも呼ばれ,人・生物機能を解明するうえでの重要手法の一つとなっており,ロボットの学問の重要な一分野を構成している.こうした研究はシミュレーションの世界でそのすべてが行われることもあるが,一般には実際にロボットを作製したり,既開発ロボットが活用される.この場合に構築されるロボットは仮説を検証するための実験装置であり,研究の最終成果物は検証された仮説そのものになることに注意する必要がある[図1・1(a)].検証された仮説は科学的知識として蓄積され,次の研究の展開のベースになる.また,仮説の内容によっては,新たな生物的知見を備えた機械システムを構築する際の知識としても活用される.一方,もう一つの工学的動機は,実際に社会に役立つ人工物を提供したいという願望に基づくものである.この背景には,さまざまな社会問題が提起するニーズ(needs)に応えようという使命感がある.その社会問題とは,例えば生産性を高めるための工場の自動化,危険作業からの人の解放,高齢化社会(ageing society)での一人暮らし高齢者の日常生活支援等々である.この種の研究では,まずニーズから出発し,それに応えるロボットが満たすべき仕様を特定した後,その仕様の実現に必要な技術の探索が行われ,必要な技術を統合して具体的なロボットシステムが構築される.このように技術を統合してニーズに応えるシステムを構築することもロボット技術の重要な研究開発であり,こうした取組みはシステムインテグレーションと呼ばれ,それを効率的に行う手法なども検討されている.このインテグレーション研究で構築される最終成果物が,構築されるロボットそのものであることは言うまでもない[図1・1(b)].7―2したがって,インテグレーション研究は,初期に設定したニーズに応える本当に役立つロボットが成果物として構築されたかどうかによって評価されることになる.ニーズに応えるロボットの仕様を満たすうえで活用できる要素技術がそろっていない場合は,必要な要素技術の開発が同時に推進される必要があり[図1・1(b)中の新要素技術開発のルート参照],現に多くの要素技術研究が行われている.むしろ,インテグレーション研究よりも,今後必要とされるロボットを想定し,それに必要な要素技術を先行研究する例が数多く存在し,ロボットの学問において大きな領域を構成している.この要素技術研究では,開発した要素技術の効果を検証するため(a) robotics)産業用ロボティクス(industrial robotics)人間共存型ロボティクス(human friendly robotics)メンテナンスロボティクス(maintenance robotics)エンターテイメントロボティクス(entertainment robotics)リハビリテーションロボティクス(rehabilitation robotics)にロボットが構築されるが,この場合のロボットも要素技術検証の実験装置として位置付けられるもので,この種の研究においては効果が検証された要素技術が成果であり,ロボットそのものが最終成果物ではないことに注意する必要がある[図1・1(c)].近年,ロボット研究の成果の解釈は混乱しており,科学的動機のもとでの研究や,要素技術の開発において開発技術の検証に使われたロボットが興味深い動きを見せるとあたかもそのロボット自体が開発対象であるかのごとく宣伝される傾向が見られるが,それはロボット技術に誤った評価を与えると同時にロボット技術の健全な応用展開にも障害をもたらすことになるので,注意する必要がある.
