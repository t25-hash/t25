# 1・1・4 学問の名称

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

ロボットは実際に構築された人工物であるが,こうしたロボットに関し,ロボットとは何かということからそれをいかに作るかなど,さまざまな考察・研究・開発を展開するのがロボットの学問である.一般に,対象とするロボットが扱う領域,応用分野などの相違によりさまざまな学問的名称が使われるが,その代表的な名称は本書のタイトルの一部に使われているロボティクスである.ロボティクスの英語はroboticsだが,-ticsはphysics(物理学),math-ematics(数学)などに見られるように~学と訳される接尾語である.したがって,roboticsは一般に日本語ではロボット学と訳され,工学的な問題だけではなく社会科学的課題も含む,ロボットにかかわる学問全体を意味する言葉として使われる.しかし,工学としてのロボット以外の学問がそれほど活発に行われていないこともあり,ロボティクスを工学的なロボットのみを扱う学問と解釈する場合も多い.本書もその例に従っている.一方,日本語ではロボット工学という名称を特にロボティクスと分けて工学的問題のみを扱う場合に使うことも多いが,外国ではこの用語は一般的ではなく,例えばrobot engineering などという言葉はあまり使われていない.これは,欧米ではロボティクスが工学(engineering)というよりも科学(science)の一分野ととらえられていることによると思われる.欧米の大学では,一般に,コンピュータサイエンス学科の中にロボティクスの講座がおかれていることが多い.ロボティクスではさらに,扱う研究分野や応用分野によって,生物型ロボティクス(bio robotics),医用ロボティクス(medicalrobotics),宇宙ロボティクス(space robotics),水中ロボティクス(underwater robotics),産業用ロボティクス(industrialrobotics),マイクロロボティクス(micro robotics),人間共存型ロボティクス(human friendly robotics)等のようにさまざまな名称が使われている(表1・1).これらの名称は新応用分野の出現とともに案出される傾向にあり,今後も,研究分野,応用分野の展開に応じて増えていくことが予測される.
