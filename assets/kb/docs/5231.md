# 1・1・5 ロボットの構成要素

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

センサ・センシング要素:実世界で効果的に仕事をするために実世界のようすを把握する必要性があるが,そのために用いる見る・聞く・触るなどの各種感覚機能をつかさどる要素.(3) 知能・制御要素:使用者である人の指令を聞いて理解し,それに基づき,センサからの情報により理解した実世界の状況下で適切な行動を生成して運動系に指令を発する要素.(4) 環境構造要素:知能・制御要素およびロボットの機能を発揮しやすくするように工夫された環境としての要素.以上に述べた構成要素が統合化されたシステムがいわゆるロボットであるが,このロボットがチャペックの戯曲のように社会において人の役に立つ働きをするには,さらに忘れてはならない重要な要素がある.それは,人に具体的にサービスを提供するソフトウェアや応用技術である.一般的には,ロボットは上記4種の要素のうえにサービス機能が実装された工学システムあるいは機械システムとして構築される(図1・2).サービスの実装については後述することにして,各要素をさらに詳細に見ていく.まず運動要素としては,物をつかんだり操る人や生物の手・腕に相当する機能をつかさどるマニピュレーション要素と,脚の機能をつかさどる移動要素とに大別され,それを実現するさまざまな機構や制御技術,例えば腕や脚の運動学,動力学,多関節協調制御理論,安定歩行理論等々が開発されている.移動要素については,最近では路面移動だけでなく飛行機能も取り上げられる傾向にある.一方,センサ・センシング要素で検討が行われるのは,さまざまな情報を検知する各種センサとその信号処理技術である.一般に人や生物は,自身の体の状態を制御するための内界センサ(internalsensor)と,自身の周囲の状況を検知したり他とコミュニケーションするために他の個体の発する情報を検知する外界センサ(exter-nal sensor)の2種をもつが,ロボットにおいてもこれら2種のセンサとその処理技術が検討されている.ここで,内界センサとして重要なものは運動要素機構の可動部の位置や速度,加速度などを測るセンサ,また外界センサとしては視覚,力覚,触覚,聴覚センサなどがある.また,ロボットの応用分野によっては,人や生物にはない情報を検知するセンサ(例えば原子力プラントメンテナンスロボットなどの放射線検知センサなど)も検討されている.知能・制御要素は,ロボットを設計者の要求どおりに知的に動かすために必要な,ロボットがもつべき仕組みである.ロボットが一般には人に使われる機械であることを前提とするとき,人にどのように操作されるかによってその仕組みは異なる.人が遠隔で逐一操作するタイプと,人が人に接する場合のように極力簡単な指令を発することでロボットがその意をくみ取り自身の知能によって行動するタイプの2種類が検討されており,前者は遠隔操作型ロボット,後者は自律型ロボットと呼ばれる.両者の相違は,ロボットの行動図1・2 ロボットの構成要素7―3生成における人の介入の度合いの違いであるが,この度合いがどの程度のとき遠隔,自律になるかの区別はあいまいであるうえ,両者の技術を含むシステムが構築されることも多く,遠隔・自律融合型ロボットという名称が使われることもある.一般に遠隔型ロボットの技術においては,複雑な動きをするロボットを人が自然に効率的に操作できるようにするためのヒューマンロボットインタフェースを構築することが課題である.また自律型ロボットにおいては,人が行動を生成する際の知能を科学的に理解し工学的に実現することが究極的には必要になり,学習,言語理解,行動計画機能,作業計画機能などが検討されている.以上はロボット本体の構成要素であるが,ロボットをその機能に応じて有効に活用するには,環境をロボット向きに改変する技術も重要である.例えば,教示再生型産業用ロボットは位置決め機能しかもたないため,教示された位置に作業対象がないと何もできない.そこで,一般に工場では,パーツフィーダ,コンベヤを設計・配置し,決まった位置に正確に作業対象(把持する対象物,溶接する対象など)を供給する.このように,ロボットがもつ機能に応じて環境側を工夫し,所望の作業が達成できるようにすることが一般的に行われている.もちろんロボットをより知能化すれば環境側を細工しなくても作業が達成できるわけであるが,現状の学術技術レベルから見て知能化には限界があるため,このような方法がとられる.一般に,ロボットの環境をロボット向きに設計することを環境を構造化するというのに対し,対象物の位置がロボットの性能に合わせて厳格に管理された環境を,位置をベースとして構造化(された)環境などと呼ぶ.移動ロボットの空間位置計測を行うためにGPS(global positioning system)が設置された環境も,構造化環境の一例である.最近はIT 技術の進歩により,RFID(radio frequencyidentification)タグなどを環境に配置することで環境からロボットにさまざまな知識を提供し,ロボットに知能的動作を行わせることも試みられている.こうしたIT 技術を活用してロボット向きの環境を構築することを環境の情報構造化,またその環境を情報構造化(された)環境と呼ぶことがある.ロボットは,コンピュータの超小型化,無線ネットワークの進展などを背景に,本体だけでなく情報化された環境とも統合されて機能の高級化,知能化が図られる状況にある.以上述べた構成要素に応じて研究が行われている技術・課題例を表1・2に示す.これらの各構成要素に応じてその実現法を追求し,表1・2
