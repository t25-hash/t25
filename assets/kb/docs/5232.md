# 1・1・6 ロボットの形と機能

> **章**: 第1章 総論
> **節**: 1・1 ロボティクス

ロボットは、1・1・5項で述べた構成要素の組合せで具体的には構成されるわけであるが、その構成法にはいろいろな考え方がある。ロボットは人が使うために人が作るいわゆる人工物(artifact)であるので、工学的視点からいえば、人が何に使うかというニーズ(needs)に応じて仕様を定め、それを満たすに十分な機能要素群をもてばよい。もし、人の要求がものをある場所からある場所に移動させることだけであれば、手・腕に相当する機能は不要で、移動機能とそれを制御する機能のみがあればよい。その場合ロボットは、その目的に応じて1・1・5項に述べた(1)~(3)の機能の中から必要なものを選択した構造をもつことになる。人や生物はすべての要素をもって生まれるので、世の中に存在するものは種が同一であればその構造は基本的に同じ要素をもつが、ロボットの場合は異なった要素の組合せからなるさまざまな構造のものが存在する。特に、ニーズに応じて機能本位にその構造が構築されるロボットを自由型ロボットと呼び、人型ロボット(humanoid、ヒューマノイド)あるいは蛇型や昆虫型ロボットなどの生物型ロボットのように、機能だけでなく人や生物の形にもこだわって設計されるロボットと区別されることがある。また、形にこだわらないロボットでは、機能が満たされていれば要素はハードウェアでつながっている必要はないとして、機能要素を空間的に分散配置し、近年発展の著しい通信ネットワークを介して情報的にそれらを接続して統合的に機能させることでサービスを提供するシステムの構築も試みられている(図1・3)。このようなシステムを部屋型ロボット、あるいは空間型ロボット、またそのような空間を、その空間の種類に応じてインテリジェントスペースあるいはインテリジェントルーム、インテリジェントハウス、ロボティクススペースなどと呼んでいる。
