c .架装方式による分類
.定置式コンクリートポンプ（stationary concrete pump)
定置式コンクリートポンプは，原動機とポンプを一体としてスキッ
ドの上に架装したもので，一現場で長期間使用するような場合に適
する．また，長距離圧送や，高所圧送の中継用としても使用され
る．最近では，バッチャプラント船に，デストリビュータブームと
組合わせて搭載され，海洋土木工事などに使用されている．
