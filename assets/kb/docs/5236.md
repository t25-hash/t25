# 6・5・1 舗装道路構築作業と施工機械の特徴

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・5 路盤関係（締固め，舗装など)

交通車輌を走行させるための舗装された道路構造は，一般の車輌
ばかりではなく飛行場の滑走路駐機場また港湾の荷役ヤードなどに
も採用されており，荷重を担った車輌を円滑に走行させる機能を有
し，走行に伴って生じる繰返し荷重に耐える強度と，車輪の回転を
スムースに保つ平滑度を備えた路面形状が要求される．
走行車輌からの荷重を分散させる機能をもつ路盤の施工には，使
用する材料を所定の厚さ（高さ）と幅に敷広げる作業と，その材料
が所定の密度を得るための締固作業が必要となる．さらに直接走行
車輪と接する舗装表面を構築する作業においては，一般土木作業と
は異なり，使用するセメントコンクリートまたはアスファルト混合
物の硬化が進行する状況下で，定められた順序に各作業を連続して
行わなければならない時間的制約を受ける．さらにこれらの作業は
図6･55 に示すように，製造業とは異なり，不動の作業対象
（work）面上を処理装置（tool ）が移動しながら作業を行う形態で
あり，作業装置には現場間の移動を容易にするため，制約を受けた
寸法重量内で必要な各種機能が搭載されている．さらに操作は，あ
らかじめ測量作業によって設置された基準杭（丁張り）を目標とし
て行われるが，表層部の施工においてはミリメータオーダの精度が
要求される．そのため，舗装用施工機械の開発時期と平行して，早
くから各種センサを用いた位置制御機構が開発され採用されてい
る．
