# 6・5・1 舗装道路構築作業と施工機械の特徴

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・5 路盤関係（締固め，舗装など)

交通車輌を走行させるための舗装された道路構造は，一般の車輌も採用されており，荷重を担った車輌を円滑に走行させる機能を有スムースに保つ平滑度を備えた路面形状が要求される．
あり，作業装置には現場間の移動を容易にするため，制約を受けたらかじめ測量作業によって設置された基準杭（丁張り）を目標とし要求される．
そのため，舗装用施工機械の開発時期と平行して，早る．
構築に使用される装置は表6･15のように機能別に分類される．
ばかりではなく飛行場の滑走路駐機場また港湾の荷役ヤードなどにし，走行に伴って生じる繰返し荷重に耐える強度と，車輪の回転を走行車輌からの荷重を分散させる機能をもつ路盤の施工には，使が所定の密度を得るための締固作業が必要となる．
さらに直接走行は異なり，使用するセメントコンクリートまたはアスファルト混合行わなければならない時間的制約を受ける．
さらにこれらの作業は（work）面上を処理装置（tool）が移動しながら作業を行う形態で寸法重量内で必要な各種機能が搭載されている．
