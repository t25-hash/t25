# 1・2・3 メカトロニクス的問題解決の構造

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

メカトロニクスが注目されるのはそこに新しい問題解決の構造が存在するからであるが、それはどのような構造であろうか。それは、新しい機能をもつ半導体素子やマイクロコンピュータ(micro-computer)とソフトウェアが異分野の人にも容易に活用できる状況が用意され、新しい機能が比較的容易に実現できるようになったことに関係がある。CCD(charge coupled device)に代表される光電変換素子、磁気・電気変換などを可能にする磁気抵抗素子、セラミックスなどの力学的量(ひずみ)-電気量間の変換を行う圧電素子群、光の透過度を電気的に制御する液晶などが出現し、新しい機械やシステムを設計したり、現行システムがもつ課題や性能限界を解決し突破する製品の開発を目指す技術者、研究者に問題解決の強力な新手段を提供している。また、マイクロコンピュータとソフトウェア技術は、複雑な制御を容易に構築する手段を提供した。われわれが設計する人工物は、古典的には静的構造物と運動機構、駆動機構(アクチュエータ、actuator)、制御機構、動力源か(a) 古典的機械システム(b) 最近の機械システム図1・9機械システムの構造の変遷ら構成されてきた[図1・9(a)].例えば、橋のような建築物の場合は静的構造物のみ、初期の自動車、電動地下水くみ上げポンプのような機械では前述の5要素を含む。動力源は、歴史的には人力、家畜力、水力(水車)、蒸気力などが使われてきたが、近年では電気力(電動機)が動力機構の主力となったことは歴史の示すところである。一般に各種動力は制御されて初めて機械システムにおいて活用可能になるため、動力を備えた機械には何らかの制御機構が存在するが、それは蒸気機関の調速機や油圧機構のフィードバック機構に見られるように、機械的なメカニズムを使用していた。そのため複雑な制御は困難な技術的背景が存在したが、マイクロコンピュータの出現はこの状況を一変した。図1・9(b)に示されるように、最近のシステムは制御部がマイクロコンピュータで構成されており、コンピュータとアクチュエータ、また運動機構の動きや外界の状況をモニタするセンサなどが接続され、コンピュータ内のプログラムによって制御アルゴリズムが構成される。この方式の特徴は、複雑な処理をコンピュータのプログラムという形で容易に実現できること、さらには、必要であればハードウェアを変更することなくプログラムの書換えによって容易に設計内容の変更ができることである。この機能は、技術者・研究者に強力な問題解決手段を提供している。マイクロコンピュータの導入は、制御アルゴリズムだけでなく、電気信号に変換された情報をプログラムを介してさまざまに処理する手段を提供する。こうした機能は、上記のような物理的な仕事をする動力型の機械システムだけではなく、半導体技術で開発された各種センサ素子と組合わされて情報を処理して人に提示する、いわゆる各種情報機器の出現をも促した。マイクロコンピュータの導入は従来なかった新製品を生み出すほかに、機械の構成要素を標準化する技術も提供する。すなわち、複数の金物(ハードウェア)の要素の組合せで実現していた制御機能などを、マイクロコンピュータという標準的な要素とプログラムというソフトウェアで実現することを可能にしたことにより、構成部品の簡素化、標準化を実現している。それは、例えばディジタル時計に端的に見ることができる。機械式の時計に比べて、電子式時計のパーツは非常に少ない。また、こうしたメカトロニクス機器では、ハードウェアはほぼコンピュータ一つに標準化され、多様な機能はそのソフトウェアによって実現できることから、多品種少量生産型の製品を大量生産ラインで生産することをも容易にする。こうしたメカトロニクス製品がもつ特質は生産行程の単純化につながり、生産性を向上させる点でも産業にメリットを与えている。現に、メカトロニクス製品のこうした特質が生み出す生産性向上効果は、多品種少量生産型製品をもつ産業の発展に重要な役割を果たしてきた。近年、マイクロコンピュータの小型化はいっそう進み、ネットワーク技術と結びついて、ネットワーク上に分散配置されたコンピュータ群を統括管理することにより1台のコンピュータでは実現できない機能を引出すユビキタスコンピュータ技術が、さまざまな新システムを生み出す強力な手段を提供しつつある。メカトロニクスにおいても、今後、こうしたユビキタスコンピュータ技術の融合が新しい製品開発、問題解決の世界を開くものと考えられる。
