# 1・2・4 期待されるメカトロニクス技術者像

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

産業活動に伴う種々の問題を解決するうえでメカトロニクス技術の重要性が高まるにつれて、その人材をいかに育成するかが重要になる。メカトロニクス技術者にはどのような素養が必要であろうか。すでに述べたように、メカトロニクスにおいて必要なのは、従来の方法では解決困難なある問題に直面したときに、異なる専門分野の技術を用いてその問題の解決を図れる能力である。異分野の技術を使った方法が見い出され、その技術が定着すれば、その後はその技術の専門家が問題に対処できる。例えば、腕時計は従来機械工学技術者の独壇場であったが、それがディジタル時計の技術に置換わった後は、その技術の維持・発展は電子技術者の仕事になる。メカトロニクス技術者とは技術の転換部分に貢献する人である。メカトロニクスは、機械工学、電気・電子工学、情報科学(工学)、計算機科学(工学)の融合した学問領域とマクロには定義できるから、メカトロニクス技術者にはこれらの個別工学の知識をもつことがまずは求められよう。しかし、個別工学の知識をもつだけでは不十分で、さらに重要なことは、ある分野に対する深い知識を少なくとも一つはもつということである。というのは、1・2・2項でも述べたように、ある問題の解決が従来の方法では困難と判定することは、その分野に対する深い知識をもち技術を極めた人によって初めてできることだからである。技術者が自身の専門ではこれ以上の進化進展は困難と判断したときに別の解法の探索が行われ、問題解決における異分野間の技術の意義ある転換・融合は起きる。根拠もなしに異分野を融合してシステムを構築しても、それは機・電融合システムを構築したということだけで、ベストの問題解決法は得られない。この点で、単に機械工学、電子工学、情報工学などのひととおりの知識しかもたない技術者には真のメカトロニクス製品は開発できない。企業では、T 型人間、Π型人間という言葉が使われている。T,Πの縦線|はそれぞれ専門を意味し、また横線―はその専門をもとに周囲に知識を広げて学際的に問題が考えられるようになることを意味している。Π型人間は基幹となる専門が二つあり、T 型人間はそれが一つであることを示しているが、いずれも複数の個別工学の知識をもち、そのうちの一つないし二つはより深い知識を有する専門分野をもっており、それをベースに他分野にも目を配れる人を意味している。深く技術を極めた専門分野を一つか二つもつ人が、その極めた自己の知識では問題解決ができないときに別の分野の解を探索することで初めて意義ある成果が生まれる。こうした意味で、メカトロニクス技術者にはT 型人間、Π型人間が不可欠である。ここで、周辺領域の知識は必ずしも深いものである必要はない。必要なのは、自身の問題を異分野の人にも理解できる言葉で説明することだけである。メカトロニクス技術者には、自己の専門では解決不可能な問題に直面したときに自身がもつ周辺知識をにらみ、異分野の個別工学分野の技術者に問題を提起し、異分野の技術者との討論を通して問題解決の方法を見つけ、それを個別工学の問題に分解してそれぞれの個別工学の技術者にミッションを明確にして振り分けることが期待される。このとき、メカトロニクス技術者は個別工学の詳細まで必ずしも立ち入る必要はない。問題が個別工学の問題に分解された後は、個別工学の研究者・技術者の成果を統合する視点から支援して、結果を得るべく努力すればよいのである。産業界では、こうした人材をいかに育成するかに多くの努力を費やしている。一般に大学を卒業したての技術者は出身学科に応じた個別工学の知識しかもたないため、メカトロニクス技術者を育てるには各企業の独自努力による新たな社内教育を必要としているのが7―11実情である。
