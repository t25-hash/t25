a.モータグレーダ（motor grader ) モータグレーダは路面
地表などを車体中央に取付けられたブレード（blade）により，平
滑に切削したり砕石などを敷均し成形することを目的とした車輪式
建設機械である．道路の構築（おもに路床路盤の整形）作業，砂利
道などの維持補修，法面の切削，整形作業，さらにこの機構を利用
して除雪作業にも使用される．
そのためブレードは上下，回転，チルト左右シフトが可能な構造
となっている．また移動を容易にするため，長い軸間を生かして車
造となっている．これらの関係を図6･60に示す．
体幅より広いブレードを車体フレーム長手方向に回転し車体幅内に
格納することができる．図6･59に示す．
.おもな構造と特徴 モータグレーダはブレードの長さ
（幅）によって分類される．大型4～5m，中型3.4～3.7m，小型
2.8～3.1m，車輌質量（重量）は約5～30t ，エンジン出力が約
50～210kWである．

b.アスファルトフィニッシャ（asphalt finisher ) アスファ
ルトフィニッシャはアスファルト合材を所定の幅と高さに（厚さ）
に敷均す装置であり，材料を均一な密度で下面の凹凸の影響を受け
にくい機構により，平滑に一定の高さで敷均す機能を備えており，
用途上つねに移動をするために，運搬上の制限寸法質量内で多くの
要素，機能を搭載している．図6･62に示す．
.おもな構造と特徴 アスファルトフィニッシャは，敷均
し装置とそれをけん引するトラクタ部分で構成されている．トラク
タは移動性の良いホイール式とけん引力の強いクローラ式があり，
用途に応じて使用される．トラクタの型式を図6･63に示す．敷均
し装置の厚さ調整装置（スクリード装置）は独特のフローティング
機構を採用しており，トラクタに左右2本の先端をピンジョイント
されたビーム（レベリングアーム）によりけん引される整形用プレ
ート（スクリードプレート）に角度をもたせることにより，路面に
敷広げられたアスファルト合材を下面にくさび状に流入させ，垂直
方向の分力と装置の質量とのバランスを取っている．ゆえにプレー
トの角度を変化させることにより，厚さのコントロールが可能とな
る．一方，舗装幅員は補助スクリードを油圧装置により伸縮して行
う機種が多く生産されている．
図6･64に舗装厚調整機構とスクリード装置の原理図を示す．ま
た，この機構に距離計測センサを組合わせた自動厚さ調整機構が使
用されている．
c .コンクリートフィニッシャ（concrete finisher ) コンクリ
ートフィニッシャはセメントコンクリートを所定の幅と厚さに敷均
す装置であり，道路用の硬練りコンクリートを使用する場合には，
材料に振動を与えることにより流動性を高めて充てん度の高いコン
クリート版を成形する．
.おもな構造と機種 コンクリートフィニッシャは粗均し
装置，締固め装置，仕上げ装置で構成されており，締固め装置は起
振装置を備えた振動板を50～70Hzで振動させ，コンクリートを
締固める．また棒状のインナバイブレータで締固める機種もある．
また施工方法として，路上にレール付の鋼製型枠を設置しフィニッ
シャをレール上で移動させながら打設するセットフォーム方式と，
型枠の設置を不要とする本体に，締固め装置と成形用のスライドフ
ォームを内蔵して，クローラにより自走するスリップフォーム方式
があり，この方式はコンクリート舗装において世界的な普及が進ん
でいる．図6･65にスリップフォームペーバを示す．
