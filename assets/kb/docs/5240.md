# 6・5・5 締固め機械

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・5 路盤関係（締固め，舗装など)

締固め機械は締固め方式から次のように分類される．
① 静的荷重方式：ロードローラ，タイヤローラ
② 動的荷重方式：振動ローラ，振動コンパクタ
③ 衝撃的荷重方式（突固め)：タンピングローラ，タンパ
またこれらの締固め機構を組合わせた機種も使用されている．一
方締固め機械を機械の形態から分けると，ロール状のものとプレー
ト状のものに大別できる．さらに用途別に分類すれば，おもに盛土
工などの土工工事用とアスファルト舗装の締固め作業に使用される
舗装用とに分けられる．
a.ロードローラ（road roller ) ロードローラは平滑胴ロー
ルを車輪とし，その配列形式によりマカダムローラ（三輪ローラ），
タンデムローラなどの別がある．マカダムローラの自重は8～12t
のものが最も多く使用されている．最近ではマカダムローラの形状
は車体屈折式で前後輪とも駆動し，かつ同一直径のロールで質量配
分も1：1としたマカダムローラが舗装現場では使用されている．
タンデムローラは前後二つの平滑ロールを車輪とし，3～10tの自
重のものがあり舗装材の締固め作業に使用されるが，この範囲の質
量機種は最近では振動ローラに取って代わっている．締固め能力の
目安となる線圧（車輪の接地質量をその車輪の接地幅で割った値：
単位はN/ cm）は10～12tの車体屈折式で430～550N/cm程度で
ある．
b.タイヤローラ（pneumatic tire roller ) タイヤローラは空
気タイヤの特性を生かして，ニーディング作用（こね返し）により
敷均されアスファルト混合物の砕石のまわりにモルタルを詰める効
果があるので，舗装表面のシール性を高めることができる．さらに
砕石を破砕することがなく，舗装体中の砕石の方向を整えることに
より，舗装体の安定度を増すこともできる．またバラストを増減さ
せ質量を変えることも可能である．用途としては，路床路盤の転圧
からアスファルト舗装の表層の転圧まで，広範囲に使用されてい
る．車体の質量は3～25t位まで用途に合わせて各種あり，前後輪
のタイヤの配列は踏み残しのないように奇数と偶数に配置されてい
る．大型の機種は他の締固め機械と異なり，車体に水タンクを備え
用して前後進を可能とした機種も開発されている．
ており，砕石路盤転圧時に散水できる機能をもっている．
