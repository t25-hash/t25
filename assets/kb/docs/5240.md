# 1・2・3 項に述べた問題解決の結果としてさまざまなメカトロニク

> **章**: 第1章 総論
> **節**: 1・2 メカトロニクス

ス機器が生まれ,社会に貢献している.ここでは具体的に製品を挙げて,それを構造に注目して分類し,メカトロニクスが社会にどのように貢献しているかを整理する.メカトロニクス機器は一般に,採用された問題解決手段を反映した構造を備えている.したがって,この構造を見ることによって,現在どのような手法がメカトロニクス的問題解決の際に採用されているかを逆に知ることもできる.(財)機械振興協会経済研究所は,1・2・1項で述べたように1978年に行ったメカトロニクス技術の調査研究の中で,これら製品群をその構造に着目して四つに分類した.これは現在でも有効と考えられるので,従来からの4種についてはこの分類を採用することにする.一方,特に1990年代以降の技術動向として,電子・情報工学の世界ではコンピュータの超小型化とネットワーク技術が進展し,ユビキタスコンピューティング(ubiquitous computing),すなわちコンピュータの分散化が進んでいる.また,従来,おもに電子・情報工学の進展に支えられてきたメカトロニクス技術が,機械工学側の進展にも支えられる状況が生じている.それは,マイクロ・ナノ加工技術を背景としたセンサ,アクチュエータ,機構の超微細化製造技術の進展である.こうした最近の技術の進展は,従来の分類に属さない新たな製品群を生み出しつつある.以下では従来からの4種の分類に加え,二つの新しい分類を考慮して,現状のメカトロニクス製品群を整理する.(1) 分類1:従来,機械部品のみで構成されたシステムの一部がエレクトロニクス要素で置換えられたものこの分類に属する機器としては,カメラ,電子ミシン,自動車など多数ある.カメラは従来,機械部品のみで構成されていたが,半導体技術の進展とともに絞り機構やシャッタ機構が電子化された.ミシンは従来,カムと人力あるいはモータで縫製機能を実現していたが,最近は複数のモータとそれを協調的に制御するマイクロプロセッサを組込んだものが標準的である.また,自動車でも一部要素機器の電子技術による置換えが進んでいるが,電子燃料噴射装置はその一例で,従来,純粋に機械式であったキャブレタが電子式に変わり,機械システムとしての自動車に組込まれている.(2) 分類2:従来,機械部品のみで構成されていたシステムのほとんどがエレクトロニクス要素で置換えられたものこの分類の代表例は電子式時計,電卓などである.時計は,古くはぜんまい,歯車といった機械部品で構成されていたが,マイクロプロセッサや高集積電子回路技術の進歩および液晶表示技術の進歩により,フレーム以外はエレクトロニクスで構成される時計が出現した.また卓上計算機も,純機械式のものから電子式にほぼ完全に世代交代した.こうした機構のすべてがエレクトロニクスで置換わることにより,従来製品をいっそう高精度,高機能化するという問題解決が可能になり,製品の低価格化・大衆化も促進された.(3) 分類3:エレクトロニクスが主体の機器で機械部分もあるもの.また,エレクトロニクスが含まれているが機械自体は簡単なものワードプロセッサ,ファクシミリ,CD(compact disk)プレーヤー,自動改札機,ATM(automatic teller machine)など,この分類に属する機器は多い.ワードプロセッサはコンピュータのパーソナル化およびワープロソフトの開発と機械電子部品で構成されるプリンタとが融合して生まれた,メカトロニクス以前の時代には存在しなかった新しい製品である.また,ファクシミリやCD プレーヤー,ATM,自動改札機なども半導体イメージセンサやオプトエレクトロニクスデバイス,磁気記憶・センシングデバイスなどのエレクトロニクスの新デバイス素子をベースとして生まれた新製品であり,いずれもエレクトロニクスが機能の心臓部を構成し,機械7―12メカトロニクス・ロボティクス部分がその補助要素として機器の中に存在している.エレクトロニクスの進歩によって生まれた新機能のセンサ素子を活用することで,従来は存在しなかった数々の新製品が生み出されている.(4) 分類4:従来の機械製品にエレクトロニクスを応用し,いままでに得られなかった高い性能や多くの機能をもつ機械装置としてまとめたもの産業用ロボットやNC工作機械がこの分類に属する.従来の機械的メカニズムにコンピュータをはじめとするエレクトロニクスを接続することで,手動に頼る要素が多かった機器操作の自動化を達成し,機器そのものの性能向上や省力化,省人化,あるいは操作性の向上を図った多くの機器が普及している.この分類のメカトロニクス製品は工場の自動化,省人化の推進等に多大に貢献し,産業の発展を支えている.(5) 分類5:機能要素を通信ネットワークを介して結合・統合化することによって,時間的,空間的制約を超えて人にサービスを提供する製品(ネットワークメカトロニクス製品)ネットワークに接続されたディジタル家電製品,インタネットやその他の通信ネットワークで遠隔操作あるいは自律制御されるロボット,ネットワークと通信しながら走行する高度道路交通システム(ITS : intelligent transport systems),ウェアラブルコンピュータと通信ネットワークで制御される各種機器,空間に多数添付されたRFID タグ群とロボットが連携してその空間にいる人にさまざまなサービスを提供するインテリジェントルームなど,個々のメカトロニクス機器が単独で使われるのではなく,通信ネットワークを介して有機的に結合して機能を発揮する,いわゆるネットワークメカトロニクス技術が新たなメカトロニクス製品の世界を形成しつつある.(6) 分類6:機構のマイクロ化,ナノ化が進行して,マイクロのレベルでエレクトロニクス効果と機械効果が融合し,外見上は機械部分とエレクトロニクス部分の要素が識別できない新しい機電融合製品機械をマイクロ・ナノ化する技術の進展あるいはこうした技術や機能材料への注目は,機械とエレクトロニクスを物理的に結合することで構築されるメカトロニクス製品とは異なる,機・電が分離できない形で融合した新たなメカトロニクス製品を生み出している.各種機能材料を用い,外部情報によって構造物の状態を制御できる知的構造物,ミクロに構造物(液体・気体を流すパイプ,航空機の翼など)の表面構造をアクティブ制御して流体抵抗を低減する知的構造物などがその例である.こうした技術は,静的構造物の世界にもメカトロニクス技術を普及させている.以上,多様なメカトロニクス製品が開発され,われわれの生活に貢献している.本書の6・2節では,これら機器の具体例をいくつか解説している.
