# 2・1・1 アナログとディジタル

> **章**: 第2章 基礎技術
> **節**: 2・1 ディジタル論理の基礎

a.アナログ量とディジタル量 連続的なすべての値を考える
のがアナログ量で，離散的な値しか考えないのがディジタル量であ
る．温度，変位，速度，圧力，角度などの物理量，硬さ，表面粗
さ，機械強度などの工業量はすべてアナログ量で，これらの量をセ
ンサで変換して得られた電圧，電流，抵抗値など，また音声や画像
などもアナログ量である．
測定値の処理（ある複数の値どうしを比べ，加・減算し，ある条
件を満足する値を選別する，その他の操作）は，アナログ量で行う
よりもディジタル量で行うほうが精度がよい（時間は少しかかる）．
図2･1は，センサによりとらえた測定値を処理し，その結果を取
出してアクチュエータ［DC（direct current）モータ，油圧サーボ
弁，XYレコーダなど］を作動させるシステムの例で，アナログ量
をディジタル量に変換するA-D変換器（AD converter)，その逆
のD-A変換器（DA converter）を含んでいる．
b.ディジタル信号処理の特徴 アナログ信号のままで，例え
ば増幅という処理を行う場合には，同一の増幅器でも信号の振幅の
大小により増幅度に違いがある［違いを小さく保てる振幅が大きい
ほどダイナミックレンジ（dynamic range）が広いという］．また
小さい雑音が混入すると，そのまま増幅されて出力される．ディジ
タル信号によるならば，これらの難点はかなり改善される．
例えば音をディジタル信号の形で記録しているCD（compact
disk，コンパクトディスク）やDAT（digital audio tape
recorder，ディジタルオーディオテープレコーダ）は，アナログ信
号のまま記録する録音テープやレコードに比べ，優れた音質を保持
することがよく知られている．
