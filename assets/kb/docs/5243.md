# 2・1・2 ブール代数

> **章**: 第2章 基礎技術
> **節**: 2・3 項参照のこと。

a. 論理学とブール代数例えば、受けた2回の2進コードが(a) アナログ信号とサンプリング(b) 量子化とディジタル信号図2・2 アナログ信号からディジタル信号へ7―14一致していれば次の動作を起こすという仕様に従う装置を設計するには、ブール代数(Boolean algebra)による解析が必要である。ブール代数は、もともと形式的論理学における命題(ある事柄の記述)が真か、偽か、すなわち1か0かだけを問題にするシステムの解析に使われていた。どんな複雑な命題(あるいは論理)でも、この(1)~(3)の命題(あるいは論理)を組合わせて構成できることが知られている。式(2・1)(2・2)はそれぞれ論理和(logical sum)あるいはOR の論理、論理積(logical product)あるいはAND の論理といわれる。式(2・3)は否定あるいはNOT の論理である。表2・2にこれらを含めた論理演算の記号を示す。式(2・1)(2・2)は(1)(2)の論理を表すものであるので次式を得る。c. 論理シンボル式(2・1)~(2・3)をシンボルを使って図2・3(a)~(c)と書く。図2・3(d)はAND の否定(NOT-AND)でNAND、図2・3(e)はOR の否定(NOT-OR)でNOR の論理シンボルである。丸印は否定を表し状態表示記号と呼ばれる。d. 真理値表(truth table)真理値表は各論理変数の状態(1,0)により論理式の状態(1,0)はどうなるかを表の形で示すものである。表2・3(a)は、二つの入力,のとる状態に対する出力の状態を、OR 論理、AND 論理について示すもので、式(2・4)(2・5)を表の形にしたものとなっている。表2・3(b)はNOT 論理の真理値表である。
