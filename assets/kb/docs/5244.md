# 6・7・2 基礎工事用機械

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・7 ボーリング，基礎工事用機械等

建築構造物あるいは土木構造物の荷重を支え，良質な支持地盤等
に伝達させる役目をもった構造物が基礎である．この基礎構造物を
築造あるいは地盤を改良するために使用する機械を基礎工事用機械
と称する．基礎工事用機械は，基礎の種類，仕様，地盤条件，施工
個所の周辺環境など合わせてさまざまな種類の機械が使用されてい
る．
a.ディーゼルハンマ（diesel pile hammer ) ディーゼルハン
マは機動性，経済性，施工効率などが優れており付帯設備も不要な
ため，既製杭打込み工法の主力機械として古くから普及している．
しかし，騒音，振動音が大きく，油煙の飛散などもあることから近
年は海上や山間部でおもに使用されている．作動原理は2サイクル
ディーゼル機関と同じで，ラムの落下によりシリンダ内の燃料と空
気の混合ガスが圧縮され爆発を起こし，打撃力ならびに爆発力によ
って杭を打込むと同時にラムを上方に押上げ次工程に入る．ラム質
量は1.3t級から8t級の大型まであり，水冷式と空冷式の冷却方
式がある．
b.気動ハンマ（air hammer ) 蒸気または圧縮空気を動力源
としてピストンまたはシリンダを上昇，落下させて杭を打込む．作
動方式により打撃体を自由落下させる単動式と強制落下させる複動
式に分けられる．ラム質量は5tから125tと大型のため，海上構
造物などの大型杭の打込みに使用される．
c .油圧パイルハンマ（hydraulic pile hammer ) 油圧ユニッ
トから送り出される作動油によりラムを引上げた後，ラムを自由落
下あるいは強制落下させて杭を打撃し，打込む．ラム質量は2tか
ら180t級まであり，落下高さを自由に選択できるため杭の選択範
囲も広く，地盤の状態に応じた施工ができる（図6･86)．
d.振動杭打ち機（vibration hammer ) 振動パイルハンマあ
るいはバイブロとも呼ばれ，主として鋼矢板（シートパイル)，H
鋼，鋼管などの打込みあるいは引抜きに使用される．電気式と油圧
式があり，いずれもチャックで杭頭部を把持し杭に振動を与えるこ
とにより周辺の摩擦抵抗を減少させて施工を行う．作動原理は本体
内部の偏心おもりを同一回転数でかつ反対方向に高速回転させて振
動を発生させる方式（図6･87）と油圧シリンダのピストンロッド
を高速に往復運動させて振動を発生させる公害対策方式とがある．
この機械の利点は騒音が打撃式のハンマに比べて小さい，杭頭を傷
めない，打撃能率が高いなどである．欠点としては打込み力が比較
的弱い，振動が大きいことなどが挙げられるが近年高周波タイプが
開発され改善されてきている．
e .杭圧入引抜機 油圧ユニットを動力源とし，油圧ジャッキ
の上下動により鋼矢板等を静的な圧力を加えて地中に押込むあるい
は引抜く装置で，機械の重量，アンカーあるいはすでに打込まれて
いる杭を反力として施工する．振動・騒音がほとんどなく，杭の全
数に対してその耐力が確認できるという利点はあるが，圧入可能な
地質，杭長，杭形状に制限があるため事前の検討が必要である．

k.地盤改良機械（soil stabilizing equipment ) 軟弱地盤の
改良工法は，対象となる深度により次のように大別される．
(ⅰ) 浅層地盤改良 表層改良や重機，車両走行時の足元の支
持耐力確保などを目的とした数m程度の深度を対象とする．
(ⅱ) 中層・深層地盤改良 液状化防止対策や耐圧版下の地耐
力改善，根切り時の山留土圧低減などを目的としたものから海底の
軟弱土を対象とした40mにもなる大深度まで対応する（図6･94）．
また，地盤改良の仕組みとしては現位置土の状態を化学的に変化
させるものと，物理的に変化させるものとに分けられる．
前者は石灰やセメントなどの安定処理材を地中の軟弱粘土と混合
かくはんして固結化する工法などがある．機械装置としては，バケ
ットにかくはん装置を装着した油圧ショベルや，リーダタイプのベ
ースマシンによる中層・深層改良機，改良材を地中に高圧で注入か
くはんするジェット方式など，さまざまなタイプの装置が開発され
ている．
後者のタイプとして，砂を詰めたガイドパイプを地中に貫入した
後，パイプを引上げて砂杭を造成するバーチカルドレーン用機械
や，このガイドパイプを上下振動を加えながら引上げて造成するサ
ンドコンパクション用機械などが挙げられる．どちらも砂杭を通じ
て地盤内の含有水を排水して圧密を促進して地盤を改良するもので
ある．
また，ロッド先端に取付けた起震機を地中に貫入し，振動させて
地盤を改良する振動締固め機も開発されている．
