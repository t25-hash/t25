# 2・1・4 論理回路

> **章**: 第2章 基礎技術

複数のディジタル信号の入力を論理変数とし、それらの間の論理演算を実行する回路を論理回路という。a. 組合せ回路と順序回路すべての論理回路は、AND,OR,NOT の論理回路(デバイス論理ゲートという)を組合わせて作ることができる。また一般に、論理回路は組合せ回路(com-binational circuit)と順序回路(sequential circuit)に分類され、前者はすべての入力の除去と同時に出力も消滅する回路、後者はすべての入力を除去しても出力が残存する、あるいは複数入力を与える順序により出力の状態が変わる回路である。順序回路は組合せ回路にフィードバックループを付加して作られる。その例はフリップフロップ(FF: flip-flop)である。ゲートもFFもIC(integratedcircuit)の形で市販されている。b. ICゲート最も多く用いられているICゲートはTTL(transistor transistor logic)、CMOS (complementary metaloxide semiconductor、シーモス)ゲートである。TTLは高速動作が可能であるが、スレッショルド電圧(その値を超える電圧が入力すると入力1が入ったことになる)が約1.4V で雑音に弱い(小さい雑音で誤って動作する)。CMOS は消費電力が小さく(腕時計などによく使われる)、またスレッショルド電圧が約2.5V(電源電圧5V の半分)で雑音には強いが、低速動作(周波数特性が劣り1MHz以上では使用困難)であり、さらに出力インピーダンスが高い(プリント基板のパターンやコネクタ配線が長いと雑音を拾いやすい)。TTLゲートとCMOS ゲートとをそのまま接続したのでは正常動作が得られない。図2・6はICゲートのブロック図(ピン接続)を示す。図2・6(a)は6個のNOT ゲート(別名インバータ)が組込まれたもの(例:SN 7404)、図2・6(b)は2入力NAND ゲート(例:SN 7400)である。ほかに排他的論理和(exclusive OR)ゲート(入力,の一方が1のときだけ出力が1、例:SN 7486,CD 4030)、2組の数の大小を比較する回路(例:SN 7485,MC 14585)、BCD コードを10進数に変換する回路(例:SN 7442)、2組の数の加・減算を行う回路(例:SN 7483,CD 4008)などが市販されている。(a) インバータ(b) 2入力NAND図2・6 ICゲートのブロック図c. フリップフロップFFは2個の出力端子をもち、一方の出力が1のとき他方の出力は0となるので、出力を,と書く。SR-FFは、図2・7(横軸は時間)のように入力が1になると出力は1、入力が1になると出力は0となる。をセット入力、をリセット入力という。,がともに1になることは禁止されている(誤動作を生じる)。図2・7はタイムチャートと呼ばれる。SRT-FFは入力,のほかにクロック入力を与える。ディジタルシステムでは一定の周期のクロックパルスの発振器[例えば8ビットマイクロコンピュータ用Z 80CPU(central processingunit、中央処理装置)では2.5MHz]を備え、クロックがくるごとに論理演算が順次行われるようになっている。SRT-FFは図2・7(b)のような動作を行う。すなわちが1でありが1のときは1となり、が1でが1のときは0となる(その他ではの変化がない)。この場合も,がともに1となることは禁止されている。JK-FFは二つの入力が同時に1となってもよいFFである。そのタイムチャートは図2・8(a)である。入力,がともに1のときクロックパルスが1となれば出力の状態(1,0)が反転する。JK-FFのブロック図を図2・8(b)に示す。ここで、,を入力として使えばSR-FFの動作を行い、,,に入力を与えればSRT-FFの動作を行う(例:SN 7473,7476)。そのほか、入力をクロックに同期して出力するD-FF(例:SN 7474,7475)、2個のパルスを入力し1個の方形波を出力するT-FFがある。d. カウンタとレジスタ。 カウンタ(counter)入力パルスの個数を進数の形で計数できるのがカウンタである。FFを個接続すれば=2욗であるが、任意ののものも設計できる。入力パルスの計数を10進数で表すICの例としてSN 7490がある。(b) SRT-FF(a) SR-FF図2・7 SR,SRT フリップフロップのタイムチャート(a) タイムチャート(b) ブロック図図2・8 JK フリップフロップ。 レジスタとシフトレジスタパンチテープの孔から読み取られたデータ(孔のありは1、なしは0)を蓄えるには、そのビット数だけの個数のFFを使えばよい。入力を与えておいて書込みパルスを与えると、AND ゲートを通って各FFに入力が入り、入力の1,0に応じて出力が1,0となる。これがレジスタ(register)である。読出しパルスを与えると出力が外部に取出される。乗除算を実行する場合には、レジスタの各ビットの内容を順番に右または左に移していく(シフトパルスを与えて行う)動作が必要である。これを行うものをシフトレジスタ(shift register)と呼ぶ(例:SN 9495)。またシフトレジスタを使うと、パンチ孔から与えられたコードをシフトパルスを与えるごとにディジタル波形に変えて出力することができる。
