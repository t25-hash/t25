# 6・8・1 道路維持修繕用機械

> **章**: 第6章 土木建設機械・鉱山機械

道路の機能を常時確保する必要から効率的な機械と工法が求めらめの清掃作業などの用途別に，それぞれの作業に適した機械類が考道路巡視用機械は，道路パトロールカー，路面補修機械を行車両からの荷崩れなどによる落下物の処理作業に使用される．

排水管清掃車（suction cleaner）は，高圧水噴射により路面下横断管などの清掃をするもので，加圧した水をホースの先端に取付けた特殊型状ノズルより前方および後方噴射してその反力によりノズルが前進する．
これを引戻し前進を繰返すことで管内を清掃する．
いい換えれば前方噴射は，堆積した土砂を射掘し，後方噴射は射掘した土砂の後方送りと合わせて，ノズルとホースを前進させる推力となり，管内土砂を開口部までかき出して，側溝清掃車により吸上げて処理する．
また，一部には使用する水を回収し，水と泥・砂に分離して浄化した水を再利用する機械もあり，この機械では，従来型の作業に必要な給水車，吸引車がなくとも作業が可能である．
この清掃は高圧水を噴射しての作業のため圧力水が外部に噴射しないよう流末などを十分調整して作業することがたいせつである．
側溝清掃車は集水桝や側溝に堆積した汚泥を吸汚管を介して，ホッパに吸込む方式で，ロータリブロア式，真空ポンプ式，ターボブロア式がある．
トラックシャシに吸込み装置とホッパを架装したものが一般的であるが，吸引車とホッパ車を専用化したものもある．
なお，ターボブロア式のものは吸込管経が200～250mmと粗大ゴミの吸込みが可能であるが，吸引距離は短い．
ロータリブロア式，真空ポンプ式のものは遠距離の作業が可能であるが，吸込管経は75～125mmと小さくなるが，この方式は吸込車1台に対してホッパ車を数台組合わせることにより連続作業や大量処理を行うことができる．
道路路面維持補修用機械には通行車両の大型化と交通量の増大に伴い轍掘れや，摩耗した舗装部分の切削りや全面切削りをする路面切削機，段差のすり付けや，ポットホールの補修，小規模な機を作るプラント，区画線を引くラインマーカなどがあり，道路輸れば常時道路機能を確保する必要があり，維持補修作業において求められている．
