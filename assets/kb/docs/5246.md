# 2・2・1 ディジタル制御と離散時間制御

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

a. ディジタル制御制御系を構成する際、補償器をA-D 変換およびD-A 変換を介してディジタルコンピュータを用いて実装することが広く行われており、ディジタル制御(digital control)と呼ばれる。ディジタル制御は、アナログ回路を用いた実装と比較して安価で信頼性が高く、複雑な制御アルゴリズムの実装が容易であり、またその変更も容易である。ディジタル制御系の例を図2・9に示す。図2・9のシステムは制御対象と補償器による構成に関しては単一フィードバック系にほかならないが、補償器がアナログ回路ではなく、アンチエリアシングフィルタ(AAF: anti-aliasing filter)、A-D 変換器(A-D)、ディジタルコンピュータ、D-A 変換器(D-A)から構成されている。(1) 本質的な情報は離散的な時刻における値で表されるため、離散時間信号として取扱うことができる。(2) とる値はディジタルコンピュータ内部で表現可能な値であるので、その集合は離散的である。結果として、ディジタル制御系の理論的な取扱いは連続時間系と比較して困難である。b. A-D 変換とD-A 変換ディジタル制御系に混在する2種類の信号はA-D 変換器とD-A 変換器により相互変換される。A-D 変換器(AD converter)はサンプリングと量子化を行う。サンプリングとは連続時間信号から離散時間信号への変換であり、量子化とは連続的な値からディジタル計算機内部で表現可能な値への変換である。ここでは量子化の影響は無視する。多くの場合、サンプリングは周期的に行われ、その周期>0はサンプリング周期と呼ばれる。これを理想サンプラ(ideal sampler)という。ただしは整数の集合である。一方、D-A 変換器(DA converter)は離散時間信号を連続時間信号へ変換する。これを零次ホールド(zero order hold)という。以上より、ディジタル制御系には以下の制約があり、一般に制御系の性能に関して不利にはたらく。(1) サンプル時刻以外は開ループ制御である。(2) サンプル時刻間における制御入力は定値である。この現象をエリアシング(aliasing)という。ωはサンプリング角周波数(sampling angu-lar frequency)と呼ばれる。エリアシングはディジタル制御系の取扱いを困難とし、性能に悪影響を与える。エリアシングの影響を小さくするためA-D 変換器の前にはローパスフィルタが置かれ、アンチエリアシングフィルタと呼ばれる(図2・9)。エリアシングは周波数領域において以下のように解釈できる。しかし、十分な事前情報があればからを再構築することが原理的には可能である。ただし、ω=ω2はナイキスト角周波数(Nyquistangular frequency)と呼ばれる。これをサンプリング定理(sam-pling theorem)という。また、連続時間信号に上の帯域制限を仮定するのはしばしば現実的でない。
