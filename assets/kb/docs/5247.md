# 2・2・2 -変換とパルス伝達関数

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

a. -変換(-transform)-変換は、離散時間信号を取扱うための基本的な道具の一つであり、連続時間信号のラプラス変換に対応する。ただしは整数の集合である。-変換は数列の母関数と本質的には同じもので、差分方程式への応用は少なくとも1940年代前半までさかのぼる。制御工学への応用はHurewiczに始まり、Ragazzini-Zadehにより-変換と呼ばれた。-変換の定義域は、解析接続により特異点を除く全平面とすることができる。-変換は以下の性質をもつ。ただし、は収束域内の、原点を囲む閉曲線である。すなわち、逆-変換には収束域に関する情報が必要である。逆-変換により負の時刻において零をとる離散時間信号を求めるとき、無限級数はテイラー展開となり、は半径の十分大きな円とすることができる。したがって、留数定理などを用いて上の積分を計算することができる。また、を伝達関数(c項参照)とするシステムのインパルス応答として計算することも可能である。特に、初期値と(存在するならば)最終値に関しては以下の公式がある。また、離散時間インパルス信号の-変換は1であることから、この離散時間システムのインパルス応答はである。これらの性質は連続時間システムのものに完全に対応している。
