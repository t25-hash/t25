# 6・9・3 木質系破砕機械（wood grinder ）

> **章**: 第6章 土木建設機械・鉱山機械
> **節**: 6・9 環境保全・リサイクル機械

建設廃棄物の種類別排出量において，全体の6%（1995年度）を発された機械である．
し，野焼き禁止，焼却処理によるダイオキシン発生問題等，木質系自走式木材破砕機は，これらの背景を受け急速に市ミルタイプが一般的であり，いずれもロータを高速回転させ，ローする．
原料の供給方法は上方投入タイプと横入れタイプがある．
家庭からの粗大ゴミ（洗濯機，自転車等），産業廃棄物等の幅広これら廃棄物の処理は1箇所集中化，大規模化が進めば進むほどのできるだけ小規模な段階で処理することにより，周辺環境への影動性に富む自走式タイプが脚光を浴びている．
自 律 型 感覚制御，適応制御，学習制御，人工知能によっシーケンス制御型 設定，教示された手順で動く機械れた所からすべての作業を行う遠隔操縦型の施工せん断力により破砕効率を向上させている．
タ構造と，一体型のモノカッタ構造がある．
占める建設発生木材，および伐根，廃木材は，リサイクルがあまり進んでおらず，再利用率は建設発生木材で4割程度である．
しか廃棄物の処理問題解決は急務となっている．
場を拡大している．
操 作 型 作業機械の一部またはすべてを直接操作する．
