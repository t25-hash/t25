# 2・2・5 PID 制御(PID control)

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

本項では制御対象は1入出力系であるとする。a. PID 制御と調整則定数ゲイン、積分器、微分器の並列結合を補償器として用いた制御系をPID(proportional-integral-derivative)制御系という。なおγを微分ゲインといい、10前後が用いられることが多い。例えばZiegler-Nicholsの限界感度法は、P制御実験により求めたゲイン余裕と位相交差角周波数ωを用いた調整法であり、閉ループ系の減衰率をほぼ4分の1にすることを目指しているといえる。その他の調整則を含めた詳細は、例えば文献(6)などを参照していただきたい。PID 制御は調整則が単純であり、また各パラメータの物理的な意味が明らかであるため再調整も容易であると考えられ、制御系の構成において最初に試みられることが多く、実際、十分な性能を達成することも少なくない(現実の制御系の90%以上がPID 制御系であるとの報告もある)が、より高い性能を求める場合は固執すべきでないだろう。PID 制御の調整則のほとんどは、ステップ目標値応答およびステップ外乱応答しか考慮しておらず、またそれ以上の緻密な要求に対して必ずしも直感的に調整できるものではない。ただしはサンプリング周期である。b. 2自由度PID 制御標準的なPID 制御で十分な性能を得ることができない場合、2自由度PID 制御はしばしば有効である。まず、2自由度制御系がもつ一般的な性質を示す。この意味で単一フィードバック制御系には自由度が1しかない。次に図2・19(b)の制御系を考える。すなわち、この系には自由度が2あり、2自由度制御系(two-degree-of-freedom controlsystem)と呼ばれる。制御系を2自由度制御系として構成することは、しばしば著しい効果を発揮する。2自由度制御系の有効性はHorowitzによって論じられ、1980年代に理論・応用に関する研究が飛躍的に進んだ。詳細は『機械工学便覧』β6編2・3節などを参照していただきたい。2自由度制御系における補償器をPID 補償器に限った系を2自由度PID 制御系(荒木)という。2自由度PID 制御の調整則を含めた詳細は、例えば文献(6)などを参照していただきたい。
