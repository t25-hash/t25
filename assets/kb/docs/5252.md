# 2・2・5 PID 制御(PID control)

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

本項では制御対象は1入出力系であるとする。a. PID 制御と調整則定数ゲイン、積分器、微分器の並列結合を補償器として用いた制御系をPID(proportional-integral-derivative)制御系という。PID 制御の歴史は非常に古く、少なくともMinorskyまでさかのぼる[例えば文献(6)].PID 制御の伝達関数は1+1で表現され(図2・18)、を比例ゲイン、を積分時間、を微分時間という。微分器は実装できないので、の代わりに、例えば1+γが用いられるが、簡単のため以下では無視する。なおγを微分ゲインといい、10前後が用いられることが多い。PID 制御器のパラメータ,,の適切な値を選ぶ調整法は、おもに実験的に得ることができる時間領域の指標を用いて数多く提案されている。例えばZiegler-Nicholsの限界感度法は、P制御実験により求めたゲイン余裕욂と位相交差角周波数ω욣욂を用いた調整法であり、閉ループ系の減衰率をほぼ4分の1にすることを目指しているといえる。具体的には、P制御のみの場合は=0.5욂,PI制御の場合は=0.45욂,=0.833욂,PID 制御の場合は=0.6욂,=0.5욂,=0.125욂とすればよいとしている。ただし、욂=2πω욣욂である。その他の調整則を含めた詳細は、例えば文献(6)などを参照していただきたい。PID 制御は調整則が単純であり、また各パラメータの物理的な意味が明らかであるため再調整も容易であると考えられ、制御系の構成において最初に試みられることが多く、実際、十分な性能を達成することも少なくない(現実の制御系の90%以上がPID 制御系であるとの報告もある)が、より高い性能を求める場合は固執すべきでないだろう。PID 制御の調整則のほとんどは、ステップ目標値応答およびステップ外乱応答しか考慮しておらず、またそれ以上の緻密な要求に対して必ずしも直感的に調整できるものではない。なお、PID 制御のディジタル実装(2・2・1項参照)においては、微分器を後進差分で置換え、伝達関数(2・2・2項参照)を1+-1+-1で定めることが多いようである。ただしはサンプリング周期である。b. 2自由度PID 制御標準的なPID 制御で十分な性能を得ることができない場合、2自由度PID 制御はしばしば有効である。まず、2自由度制御系がもつ一般的な性質を示す。図2・19(a)の単一フィードバック制御系(1自由度制御系)において、目標値から出力までの伝達関数욥、外乱から出力までの伝達関数욃はそれぞれ욥=1+욃=1+図2・18 PID 制御系(a) 1自由度制御系(b) 2自由度制御系図2・192自由度制御系で与えられる。このとき両者は욥=1-욃で関係付けることができる。すなわち、補償器を固定すれば욃と욥が同時に定まり、単一フィードバック制御系の外乱に対する特性と目標値に対する特性を独立に決めることはできない。この意味で単一フィードバック制御系には自由度が1しかない。次に図2・19(b)の制御系を考える。ただし、は2入力1出力系で=とする。このとき単一フィードバック制御系と同様に、目標値から出力までの伝達関数욥、外乱から出力までの伝達関数욃を求めれば、それぞれ욥=1-욃=1-となる。したがって両者は욥=욃で関係付けられ、単一フィードバック制御系と異なり、욃を決めた後も욥を調節することができる。すなわち、この系には自由度が2あり、2自由度制御系(two-degree-of-freedom controlsystem)と呼ばれる。2自由度制御系は図2・19(b)以外の構成も可能であり、例えば=+=-++であるから、図2・19(a)の実装を拡張することで2自由度制御系を実装することを考えるならば、補償器として=-を考えるのが自然であろう。制御系を2自由度制御系として構成することは、しばしば著しい効果を発揮する。例えば욃が不安定零点をもたなければ、욥の極は自由に設定できる。2自由度制御系の有効性はHorowitzによって論じられ、1980年代に理論・応用に関する研究が飛躍的に進んだ。詳細は『機械工学便覧』β6編2・3節などを参照していただきたい。2自由度制御系における補償器をPID 補償器に限った系を2自由度PID 制御系(荒木)という。2自由度PID 制御系は=1+1=とした微分先行型PID 制御、==1+としたI-PD 制御(北森)などの統一的な表現となっている。2自由度PID 制御の調整則を含めた詳細は、例えば文献(6)などを参照していただきたい。
