# 2・2・6 制御

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

制御(control)は,閉ループ系の伝達関数のノル制御およびそれをベームを評価規範とする準最適制御である.スとするロバスト制御理論は1980年代以降,理論・応用に関する研究が飛躍的に進み,産業界への応用も少なくない.ここでは制御の動機付けの一つを与え,問題設定など応用上の注意を中心に述べる.詳細は『機械工学便覧』β6編2・4節などを参照していただきたい.離散時間問題,ディジタル制御問題も完全に解かれている.a.
