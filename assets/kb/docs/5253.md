# 6・1・1 未利用エネルギーとヒートカスケーディング

> **章**: 第6章 未利用エネルギー
> **節**: 6・1 概説

未利用エネルギー（unexploited energy）の利用法は大きく二つ
の種類に分けられる．一つは，まだ利用可能な温度の熱が使用され
ずに廃棄されているものを回収して使用する場合であり，もう一つ
は，環境中に排出され薄まった熱を回収して高温化し，利用する場
合である．前者は熱を多段階利用するヒートカスケーディングの枠
組みであり，後者はヒートポンプ技術を用いて大気や河川水・海水
などの環境中にある内部エネルギーを回収する形態と言い換えるこ
とができる．第6章では前者の場合について取上げる．
ヒートカスケーディング（heat -cascading）とは，化石燃料を
燃焼させて得た高温（1500℃～）の熱を，温度レベルが徐々に低
下する順に熱利用プロセスに投入し，最終的に環境温度に低下する
まで熱を利用する考え方である．ヒートカスケーディングの概念図
を図6･1に示す ．高温の熱はエクセルギーが高く，効率的に発電
を行うことが可能である．そこで，高温の燃焼熱は発電プロセスや
工業加熱プロセスに利用され，温度が低下した排熱は続いてボイ
ラ，空気調和用熱源に用いられる．さらに温度が低下した熱は給
湯，暖房など低温の熱負荷に投入され，熱としての利用価値を使い
尽くされる．例えば，鉄鋼プラントでは高炉において非常に高温の
熱を利用する一方，300℃程度の熱は使い道があまりない．それに
対し，紙製造プラントでは300℃以下の熱を多用する．すなわち，
鉄鋼プラントから出る排熱を回収して紙製造プラントに利用すれ
ば，熱を多段階に利用できるとともに，後者のプロセスに消費して
いた一次燃料を削減することが可能となる．各工場単位での省エネ
ルギーが広く普及しているわが国においては，工場をまたがった未
利用熱エネルギー有効利用の連携によっていっそうの省エネルギー
を図ることが求められている．なお，近年普及が進んでいるコージ
ェネレーション（cogeneration，熱電併給）やコンバインドサイ
クル（combined cycle，複合サイクル）も熱を多段階に利用して
おり，ヒートカスケーディングの一環として理解される．
図6･1には熱の多段階利用として環境温度を境にして高温側と低
温側が描かれている．低温側の出発点はLNG冷熱（－160℃）が
位置付けられている．LNGは天然ガスを液化する際に注入された
冷熱を保持している．すなわち，メタンとしての化学エネルギーだ
けでなく，－160℃での潜熱変化分と環境温度まで上昇するまでの
顕熱変化分を併せ持っている．この冷熱エネルギーを有効に活用す
ることも将来的に大きな課題と考えられるので，第6章において解
説する．
