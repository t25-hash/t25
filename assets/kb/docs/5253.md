# 2・2・6 制御

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

制御(control)は、閉ループ系の伝達関数のノル制御およびそれをベームを評価規範とする準最適制御である。スとするロバスト制御理論は1980年代以降、理論・応用に関する研究が飛躍的に進み、産業界への応用も少なくない。ここでは制御の動機付けの一つを与え、問題設定など応用上の注意を中心に述べる。詳細は『機械工学便覧』β6編2・4節などを参照していただきたい。離散時間問題、ディジタル制御問題も完全に解かれている。a. ループ整形法ここでは古典制御の枠組における設計法の一つであるループ整形法を通じて制御を動機付ける。簡単のため、図2・19(a)の単一フィードバック制御系を考える。感度関数はから偏差、外乱から入力への伝達関数と一致し、また定義からも、ゲインが小さいことが望まれる。したがってのゲインが小さいことが望まれる。図2・20にループ整形法のる求め補償器を模式図を示す。したがって、ループ整形法においては安定性を確保できるとは限らない。仕様式(2・17)を満たす補償器のシステマティックな設計が制御の一つの動機であった。b.設計問題設計問題は図2・21を用いて定式化される。. 問題(設計問題)図2・21のフィードバック制御系を考える。与えられたとγ>0に対して次の二つの仕様を満たすを求めよ。(1) 閉ループ系は内部安定である。(2) からまでの伝達関数のノルムはγ未満である。制御系設計においてノルムを評価規範にとることはZamesによって提案された。最適化問題ではなく準最適化問題であることに注意していただきたい。設計問題のもう一つの特徴はを用いた定式化であり、は一般化プラント(generalized plant)と呼ばれる。設計問題の解法には、リカッチ方程式に帰着する方法、線形行列不等式に帰着する方法などが知られている。理論的な考察をするのでない限り、設計問題の求解にはRobust Control Tool-box/MATLAB などの制御系CAD(computer aided design)ソフトウェアを用いるのが一般的であるので、ここでは省略する。『機械工学便覧』β6編2・4節などを参照していただきたい。一般化プラントは制御対象以外の部分(上の例題においては,)を含んでおり、それらを含めた一般化プラントの構成によって設計問題を解くことで得られる制御系の特性が決まる。したがって、設計による制御系設計においては、満たすべき仕様をいかに一般化プラントに反映するかに最も注意を払うべきである。例えば、上の例題の一般化プラントに固執して,の選定に注力するのは本末転倒であり、避けるべきである。また、適当に重み付けした状態をすべて評価出力に含めることで特性が改善されることがある。直接観測できない信号は評価出力に含めることができないと誤解しないよう、注意が必要である(入力についても同様)。
