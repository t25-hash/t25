# 6・1・2 廃棄物発電（waste to energy）

> **章**: 第6章 未利用エネルギー
> **節**: 6・1 概説

国土の狭いわが国においては廃棄物を焼却処理して減容化するこ
とが不可避である．焼却の際にごみの燃焼熱が発生するので，それ
を有効利用することが進められている．排熱を温水プールの加温な
どに利用するのも一つであるが，発電に利用するものも普及してい
る．廃棄物の処理能力別の発電設備をもつごみ処理設備件数および
発電出力合計を図6･2に示した ．処理能力300～600t / d規模の設
備が中心的であることがわかる．個別の自治体ではごみ処理規模が
小さくて発電設備をつけることが難しい場合でも，広域的な対応を
図6･1ヒートカスケーディングの体系
図6･2 廃棄物発電の件数および発電出力
とができる．その一つの方法がRDF化（固形燃料化）である．固
形化することによって，一般廃棄物の状態よりも輸送，貯蔵，ハン
ドリング面で大幅に取扱いがしやすくなる．
一方，ごみ発電の発電効率は必ずしも高くはない．1990年代前
半までは高くても10～15 %（発電端）であり，多くは5%程度の発
電効率に留まっている．これは腐食などの制約条件によって蒸気温
度を十分高くとれないことに起因している．1990年代後半になっ
て発電効率20 %以上となる高性能なごみ発電設備が導入されてい
る．特にガスタービン発電と組合わせたスーパーごみ発電は
25～35%もの高い効率を実現している．
ごみのもつ熱エネルギーを利用した分は一般の燃料を使わなくて
すむため，ごみ発電は直接一次エネルギーの削減に結びつく．家庭
から排出されるごみそのものの発熱量も年々増加傾向にあるので，
その熱量を効果的に電力に変換し有効利用することは，その重要性
を増している．将来的には水分を多く含む生ごみなどを一般のごみ
から分離して，燃焼熱を十分引出せる社会システムの必要性も示唆
されており，廃棄物発電は将来的に注目すべき技術と期待される．
