# 2・2・8 予測フィルタ理論

> **章**: 第2章 基礎技術
> **節**: 2・2 メカトロニクスにおける情報と制御の理論

a. カルマンフィルタ(Kalman filter)動的環境や不確かな環境で適切に行動するためには、環境の状態の推定(estimation)や予測(prediction)が必要である。例えば、移動物体をマニピュレータや可動台に取付けられたカメラで追跡することを考える。このとき、追跡対象の現在の位置を適切に推定し、さらに将来の移動方向を適切に予測することが、確実な追跡の鍵となる。しかし、一般にカメラやその他のセンサから入力された情報は不確かさを含み、また追跡対象の未来の行動も完全に予測することはできない。したがって、そのような不確かな情報をもとに推定・予測を行う必要がある。また、推定・予測の誤差を見積もることも重要である。一方、ロボットが動作するとき、その動作に不確かさがあれば、自分自身の状態の推定・予測が適切な行動計画のために必要である。以上のような問題は、過去から現在に至るまでの時系列データから現在の状態を推定し、あるいは将来の状態を予測する問題と考えられる。そのような推定や予測を行うための手法として、カルマンフィルタがある。カルマンフィルタは誤差の確率モデルを用いて線形システムの推定を行うための手法であり、観測値の線形結合で推定値を計算する。得られる推定値はそのような線形推定の方法の中では分散最小であり(線形最小分散推定値)、誤差分布が正規分布のときには、すべての推定法の中で分散最小の推定値を計算することができる。カルマンフィルタは以下のように定式化できる。共分散行列についても同様の添字表記を用いる。こ図2・27 distance)を用いたイノベーションの検定を用いる。マハラノビス距離とは分散で正規化した二乗距離であり、以下ので与えられる。次に、実際に観測が得られなかったときの扱いであるが、その場合にはもちろんフィルタリングを行うことができないため、式(2・52)(2・53)による予測のみを行う。その結果、状態量の不確かさは単調に増加する。このとき、実際には得られていない仮想的な観測値(例えば観測の予測値)を想定し、それを用いてフィルタリングすることはできない。b. カルマンフィルタの物体追跡(object tracking)への応用例簡単な例について実際にカルマンフィルタを構成してみる。誤差項はセンサの誤差に起因する、観測された位置の誤差を表す。例えば、誤差が物体までの距離に依存するセンサを用いる場合には、観測された距離に応じてこの誤差の大きさ(共分散行列)を調整すればよい。また、物体の初期位置とその誤差は問題に応じて適当に与える。例えば、最初に物体を発見したときの位置とその観測誤差を用いることが考えられる。以上のシステム方程式、観測方程式、初期値を用いて式(2・51)~(2・56)を適用すれば、カルマンフィルタによる追跡が実現できる。予測、観測、フィルタリングの関係を図2・28に示す。実際の追跡問題では、センサの誤り等により、一つの追跡物体に対し二つ以上の観測値が得られることがある。そのような場合に7―30メカトロニクス・ロボティクスは、とりあえず可能な対応の候補をすべて保持してそれぞれをカルマンフィルタで独立に追跡し、一定時間対応する観測が得られない場合に追跡を中止するtrack-splitting filter(分割追跡フィルタ)などが用いられる。また、複数物体を同時に追跡する場合には、複数の観測値と複数の追跡対象との対応付けが必要となる。その対応が一意に決められないときには、観測値と観測対象との対応の確率分布を用いるjoint probabilistic data association filter(同時確率対応付けフィルタ)などが用いられる。c. 拡張カルマンフィルタ(extended Kalman filter)カルマンフィルタは線形確率システムを対象としており、非線形システムにはそのまま適用することができない。そのような場合には、拡張カルマンフィルタを用いることが多い。d. 拡張カルマンフィルタの移動ロボットの位置推定(localiza-tion)への応用例移動ロボットが、位置が既知のランドマークを移動しながら観測し、自己位置を逐次的に推定することを考える。指令値とロボットへ与えられる実際の操作量との間には誤差があり、それが動きの不確かさを生じさせるとする(実際には車輪の滑りなどの影響も含まれる)。これは線形化された観測方程式である。この場合、観測を重ねると、ロボットの位置・姿勢だけでなくランドマーク位置もより正確になる。このような問題をSLAM(simulta-neous localization and mapping)問題と呼ぶ。e. データのサンプル集合を用いたフィルタリングカルマンフィルタおよびその拡張は、誤差が正規分布である[あるいは少なくとも単峰性(unimodal)である]ことを想定している。したがって、誤差の分布が多峰性をもつときには適切なフィルタリングが行えない。そのような場合に用いるフィルタリング手法の一つとして、確率分布をサンプル点の集合として離散的に近似する方法[逐次モンテカルロ法(sequential Monte Carlo method)、パーティクルフィルタ(particle filter)といった名称で呼ばれる]がある。サンプルの数が十分多いときには、最適フィルタのよい近似となる。以下に、逐次モンテカルロ法の最も基本的な計算法を示す。推定すべき状態をとし、対象とするシステムは一次マルコフ性をもつとする。この計算をサンプルの集合を用いて近似的に行う。このとき、より重みの大きいサンプルが取出される回数がより多くなる。以上を繰返すことによってフィルタリングを行う。そのままでは一つのサンプルに確率1.0が与えられる可能性があるので、実際にはある程度より小さい確率のサンプルは消去するなどの処理が必要となる。
