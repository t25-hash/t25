# 2・3・1 ハードウェアの基礎

> **章**: 第2章 基礎技術
> **節**: 2・3 制御用コンピュータの基礎

昨今，コンピュータの小型化が進み，かつてはスーパーコンピュ

2・3・2入出力インタフェースと通信インタフェース
ロボティクス・メカトロニクス要素を制御するためには，多くの
インタフェースを利用する．ここでは，代表的な入出力インタフェ
ースと通信インタフェースについて示す．
a.入出力インタフェース
.ディジタル入出力（DIO:digital input outpnt) DIO
はHigh/ Low（高/低）の信号で外部機器を制御する．この機能は
スイッチ的な用途で利用可能であり，LED（light emitting diode）
の点滅のように単純な機能もこれだけで実現される．さらには，ポ
ートのHigh/ Lowを周期的に実行することによる独自プロトコル
による通信にも利用可能であることから，広い用途で利用できる機
能であり，メカトロニクス機器の制御における基本的なインタフェ
ースの一つといえる．
.A-D変換器（AD converter) 光センサや加速度セン
サ，超音波センサ，ポテンショメータなどから取得される電圧値を
量子化してコンピュータで取扱える情報にする処理である．使用す
るA- D変換器により，A- D変換器の分解能が異なる．センサなど
から取得できる電圧値を忠実にディジタル化するためにはA- D変
換の分解能が大きく影響するため，正確なデータを必要とする場合
は高分解能のA- D変換器が求められる．また，データの取得周期
（サンプリングタイム）もA-D変換器の選定において重要な要素
である．
これらのマイクロコントローラはプログラムによってその動作を
変更でき，その開発においては各メーカから開発環境が無償で提供
されているものもあれば有償のものもあるが，いずれにせよ第三者
によって作られている開発環境を利用している．また，このとき利
