# 6・2・1 廃棄物発電（waste to energy）

> **章**: 第6章 未利用エネルギー
> **節**: 6・2 廃棄物発電

a.廃棄物の種類と組成 廃棄物発電は，サーマルリサイクル
ともいわれ，ごみの焼却処理に加え，その有するエネルギーを利用
して発電を行うもので，ごみの処理と発電を同時に達成できるた
め，わが国の環境・エネルギー事情に極めて適した技術である．そ
のこともあり，わが国エネルギー政策上，新エネルギーとして位置
付けられ，その中心的技術として期待されている．本項では，その
ような廃棄物発電の技術面からの特徴と概要を述べる．
b.廃棄物発電の種類
.廃棄物利用形態の相違による分類 廃棄物利用形態・方
法から見た分類としては，従来型発電方式，RDF発電方式，リパ
ワリング方式，および新発電方式の4方式に大別される．本項で取
上げるのは，そのうち従来型発電方式と新発電方式である．従来型
発電方式（conventional waste to energy type）は，廃棄物を焼却
炉で燃焼して同燃焼エネルギーをボイラで回収して蒸気タービンで
発電する方式で，廃棄物発電の導入以来，今日でも廃棄物発電の主
体となっている方式である．新発電方式は，ガス化溶融発電技術
（pyrolysis and combustion type waste to energy），ガス変換発電
技術（pyrolysis and gasification type waste to energy），あるい
は新型（次世代型）ストーカ燃焼発電技術（advanced stoker
firing type waste to energy）など，実績は乏しいものの環境面，
効率面での特徴を有し，近年注目を浴びてきている技術である．
.蒸気の回収方法（蒸気タービン形式）の相違による分類
発電を前提とすると発生した蒸気は蒸気タービンで発電することに
なるが，蒸気タービンの形式により，背圧タービン方式と復水ター
ビン（抽気復水タービンを含む）方式に大別される．
① 背圧タービン方式は，タービンの背圧を正圧（通常
130～250kPa程度）にして運転する方式で従来わが国で多く採用
されてきた．その理由は，発生電力で所内動力をまかなうには同形
式で十分であること，系統構成が簡単なこと，さらに排気を熱源用
蒸気として利用可能なことなどによった．
② 復水タービンあるいは抽気復水タービンは，排気を冷却し真
空圧まで下げる方式で，発電出力が大きくなるので近年その採用が
増加してきた．なお，冷却方式としては，空冷と水冷に大別される
が，その排圧は空冷の場合では約30kPa，水冷の場合は冷却塔使
用による冷却水循環方式では約10kPa，海水冷却方式では5kPa
が採用される．なお，わが国の廃棄物発電としては，立地場所，経

f .新発電方式
.廃棄物ガス化溶融発電技術 廃棄物発電所からの灰は減
容化および不活性化の点から溶融固化処理法が推奨されている．溶
融固化する場合，従来技術では取出した灰を別置き装置で電気等を
使って灰の融点以上に温度を上げる方式が採用される．そのための
装置が必要であることに加え，加熱用に電気等のエネルギーが必
要，溶融炉（melting furnace）の耐火材・電極・バーナの損耗が
大きい，さらに高温下で再飛散する微粒子の捕集と処理の問題等を
抱える．廃棄物ガス化溶融発電技術は，廃棄物を熱分化し，発生し
たガス，チャーおよびタール等を後流の溶融炉で高温燃焼し，蒸気
タービンで発電するシステムである．その特徴は，①灰をごみのエ
ネルギーのみで溶融化するので効率が高く経済性に優れる，②高温
燃焼のためにダイオキシン類を分解する，③金属類を低温ガス化後
に分離するのでリサイクル特性に優れた金属類の回収が可能である
などが挙げられる．同技術は近年ヨーロッパで開発が進められ，わ
が国でもダイオキシン類規制の実施に伴い急速に普及してきている
技術である．同技術の分類としては，特に熱分解炉（pyrolysis
reactor ）の形式を中心として，シャフト炉型（shaft furnace
type），ロータリキルン型（rotary kiln type），および流動床型
（fluidized bed type）に大別される．図6･10 に同システムの概念
を示す．
