# 6・2・3 リパワリング（repowering）

> **章**: 第6章 未利用エネルギー
> **節**: 6・2 廃棄物発電

廃棄物発電プラントにおけるリパワリングは，ガスタービンの導
入により発電量の増加と総合発電効率を向上させる手法である．国
内では，スーパーごみ発電（wasted combined power generation）
と呼称されているシステムが，リパワリングの一例として実用化さ
れている．ヨーロッパでは，発電と熱供給の両面でのパワーアップ
効果をねらった例もある．
廃棄物の焼却により発生する排ガスと飛灰には，塩素（Cl ），ア
ルカリ金属（Na，K等），重金属（Zn，Pb等）が含まれている．
これらの作用で，ボイラ水管のメタル温度が，付着飛灰の溶融温度
に近い300～400℃以上の温度領域では厳しい腐食環境を形成す
る ．廃棄物焼却炉付設ボイラから発生する蒸気温度は，腐食温
度領域を避け低く設定されてきた．しかし，廃棄物発電効率を向上
させる社会的要求により，蒸気温度の高温化が必要とされている．
高温化を目的として，次の方策が実用化されている．
(１)ボイラ出口蒸気を腐食の少ない温度領域とし，ガスタービ
ン排ガスにより過熱する．
(２)ボイラ出口蒸気を500℃程度まで高める．腐食対策として
ボイラ水管，スーパーヒータ管に高合金（Cr ，Mo，Ni含有）を
用いる ．
上記(１)は，ガスタービン排気ガスでボイラ発生蒸気を過熱し，
蒸気タービン発電効率を向上させるシステムであり，スーパーごみ
発電として知られる．また，上記(２)のように，ボイラ発生蒸気温
度がガスタービン排ガス温度と同等以上の場合は，ガスタービン排
ガスによるボイラ発生蒸気の過熱は不可能である．しかし，さらに
発電効率を向上させる方法として，高圧蒸気タービン出口の蒸気を
ガスタービン排ガスで再過熱し，低圧蒸気タービンでの発電量を増
大させるリパワリングシステムの適用が可能である ．
リパワリングシステムとして，基本的には，通常の廃棄物発電方
式で蒸気タービン以降の設備を共用するガスタービン併置方式と，
これに加え，過熱器または再熱器をガスタービン排ガスで加熱する

図6･19ガスタービン複合方式（過熱器型)
ガスタービン複合方式に分類される．それぞれの構成例を以下に示
す．
a.ガスタービン併置方式（図6･18) ガスタービン排ガスボ
イラで過熱蒸気を発生させ，蒸気タービンに供給する．廃棄物焼却
炉付設ボイラから発生した蒸気は，別の配管で蒸気タービンに供給
される．
蒸気タービンには，混気（混圧）タービンを用い，高圧側にガス
タービン排ガスボイラで発生させた蒸気が，その下流に焼却炉ボイ
ラで発生させた蒸気が導かれる．
b.ガスタービン複合方式（過熱器型 ，図6･19) 廃棄物焼
却炉付設ボイラから発生する蒸気温度が，ガスタービン排ガス温度
よりも低い場合に適用される．ガスタービン排ガスボイラに，廃棄
物焼却炉付設ボイラから発生した，過熱または飽和蒸気の過熱器が
設けられる．廃棄物燃焼ガスによるボイラ水管の高温腐食を回避で
きるので，ボイラ水管材料に低コストの炭素鋼を使用できる．スー
パーごみ発電は本方式による．
c.ガスタービン複合方式（再熱器型 ，図6･20) 廃棄物焼
却炉付設ボイラから発生する蒸気温度が，ガスタービン排ガス温度
と同程度か高い場合に適用される．ガスタービン排ガスボイラに，
高圧蒸気タービン出口蒸気を導き，再熱器により再過熱し，低圧蒸
気タービンに導く．廃棄物燃焼ガスによるボイラ水管の高温腐食対
策として，ボイラ水管材料にインコネル等の高コストの高合金を用
いる必要がある．
国内外の，廃棄物発電リパワリング施設の概要を表6･3に示す．
