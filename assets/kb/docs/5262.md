# 2・4・3 応用別ネットワーク技術

> **章**: 第2章 基礎技術
> **節**: 2・4 情報通信ネットワーク制御の基礎

これまでに説明したインタネットを代表とするネットワークの要素技術は、その大部分が業務用、オフィスネットワーク向けに開発されてきた。しかし、余暇や生活支援のために開発されたホームネットワークに代表されるように、ネットワークの要素技術を利用して目的別にそれぞれ設計されたネットワークが実現されている。以下にその概要について説明する。a. ホームネットワーク技術ホームネットワーク(homenetwork)とは、特定の方式によって実現されたネットワークではなく、家庭で利用されるネットワークの総称である。古くは映像(AV : audio visual)機器間の映像データ転送や、住宅設備機器間の制御データの転送に用いることで進化した。前者ではIEEE1394規格などによる高速映像転送、後者では無線や電力線による制御システムなどが例として挙げられる。従来は家庭内に閉じたクローズドシステムとして設計されていたため、ネットワークにはレイヤ2相当の機能のみが要求されてきた。またそれらも、機器のカテゴリ別にAV 系ネットワーク、情報機器系ネットワーク、生活家電および住宅設備系ネットワークがそれぞれ敷設されることが一般的であった。図2・37にその概要を示す。近年は機器のディジタル化やインタネット技術の導入が進み、ホームネットワークも家庭内ネットワークと、家庭とISP上のサービスプロバイダとを結ぶアクセス系ネットワークから構成されるようになってきた。さらに近年、放送を用いたデータ配信、通信による映像配信など、放送システムと通信システムの融合も進みつつある。図2・38にその概要を示す。家庭内ネットワークのデータリンクは、新たな伝送路の敷設が困難なため、無線や電力線の利用が前提とされてきた。今後、高品位画像転送向けに、より高速なデータリンクの導入も加速されると思われる。b. アドホックネットワーク技術アドホックネットワーク図2・37ホームネットワーク技術の概要(DVD :digital versatiledisk)7―39(ad hoc network)とは、移動を前提とする機器へのネットワークとして固定的な設備を利用せず、ユーザ機器や一時的な設備機器のみで構成したネットワークである。機器の移動に伴い機器間のデータリンクの状況が変化するため、それに応じたネットワークのトポロジー制御を行う。例えば、無線LAN にはアドホックモードと呼ばれる、アクセスポイントを必要とせず機器間で直接相互通信できるモードが広く搭載されているが、それを用いてユーザ機器のみで構成したネットワークを形成することができる。システムを集中制御する固定的なインフラストラクチャを必要としないため、自立分散型ネットワークと呼ばれることや、可搬性が前提であることが多く無線の利用が前提とされているため、無線アドホックネットワークと呼ばれる。データリンクには、それに適した方式が必要であり、現在は無線LAN,Bluetoothなどの利用がその主流である。2・4・1・a項にて前述したPAN は通常、アドホックネットワークの要素技術から構成される。P2P(peer to peer)と呼ばれるユーザレベルの直接通信形態のネットワークとして利用されることもある。ここで、アドホックネットワークを単一のデータリンクで構成する場合、可搬型ノードでは無線ノードの消費電力を抑える必要があり、無線通信で直接通信可能な範囲が小さいことが多い。無線の通信範囲を拡大するのではなく、無線ノード間でデータを転送することでネットワークの規模を拡大するための技術として、マルチホップネットワークが検討されている。これは不安定な通信路に対し、動的に目的地へのルートを選択しながらデータを転送する方式であり、ルートを選択する手段によりプロアクティブ方式とリアクティブ方式に大きく分類される。前者はあらかじめ経路表を作成しておく方式であり、通信要求時に即応して通信が開始できる。OLSR(optimized link state routing)、TBRPF(topology broadcastbased on reverse-path forwarding)と呼ばれる方式によって実現されている。後者は、通信の要求発生時に適切な経路表を作成する方式であり、ノードの負担を極力減らすことができる。DSR(dynamic source routing)、AODV(ad hoc on-demand distancevector)と呼ばれる方式によって実現されている。c. センサネットワーク技術センサが収集した情報を転送・流通させるためのネットワークを一般にセンサネットワーク(sen-sor network)と呼ぶ。これらは、社会インフラストラクチャとして固定的に配備されたセンサより単に情報を収集する程度のものから、多数、小型、可搬、長時間稼働が要求されるノードを接続してデータを効率的に収集することを目的としたネットワークまで広く存在する。特に、後者の場合のみをセンサネットワーク技術と呼ぶ場合も多い。その転送プロトコルは、一般に無線データリンクとアドホックネットワーク技術により構成される。さらに、効率的にセンサデータを収集するためのセンサの配置方法、データの管理機構図2・38 ホームネットワークの構成例(CATV :cable television,IPv6:internet protocol version6,xDSL:x digital subscriber line)-7―40メカトロニクス・ロボティクスなどの研究が進められている。
