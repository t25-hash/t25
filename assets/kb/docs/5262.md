# 2・4・3 応用別ネットワーク技術

> **章**: 第2章 基礎技術
> **節**: 2・4 情報通信ネットワーク制御の基礎

これまでに説明したインタネットを代表とするネットワークの要
素技術は，その大部分が業務用，オフィスネットワーク向けに開発
されてきた．しかし，余暇や生活支援のために開発されたホームネ
ットワークに代表されるように，ネットワークの要素技術を利用し
て目的別にそれぞれ設計されたネットワークが実現されている．以
下にその概要について説明する．
a.ホームネットワーク技術 ホームネットワーク（home
network）とは，特定の方式によって実現されたネットワークでは
なく，家庭で利用されるネットワークの総称である．古くは映像
（AV:audio visual）機器間の映像データ転送や，住宅設備機器間
の制御データの転送に用いることで進化した．前者ではIEEE1394
規格などによる高速映像転送，後者では無線や電力線による制御シ
ステムなどが例として挙げられる．
従来は家庭内に閉じたクローズドシステムとして設計されていた
ため，ネットワークにはレイヤ2相当の機能のみが要求されてき
た．またそれらも，機器のカテゴリ別にAV系ネットワーク，情
報機器系ネットワーク，生活家電および住宅設備系ネットワークが
それぞれ敷設されることが一般的であった．図2･37にその概要を
示す．
近年は機器のディジタル化やインタネット技術の導入が進み，ホ
ームネットワークも家庭内ネットワークと，家庭とISP上のサー
ビスプロバイダとを結ぶアクセス系ネットワークから構成されるよ
うになってきた．さらに近年，放送を用いたデータ配信，通信によ
る映像配信など，放送システムと通信システムの融合も進みつつあ
る．図2･38にその概要を示す．
家庭内ネットワークのデータリンクは，新たな伝送路の敷設が困
難なため，無線や電力線の利用が前提とされてきた．今後，高品位
画像転送向けに，より高速なデータリンクの導入も加速されると思
われる．

されることもある．
ここで，アドホックネットワークを単一のデータリンクで構成す
る場合，可搬型ノードでは無線ノードの消費電力を抑える必要があ
り，無線通信で直接通信可能な範囲が小さいことが多い．無線の通
信範囲を拡大するのではなく，無線ノード間でデータを転送するこ
とでネットワークの規模を拡大するための技術として，マルチホッ
プネットワークが検討されている．これは不安定な通信路に対し，
動的に目的地へのルートを選択しながらデータを転送する方式であ
り，ルートを選択する手段によりプロアクティブ方式とリアクティ
ブ方式に大きく分類される．前者はあらかじめ経路表を作成してお
く方式であり，通信要求時に即応して通信が開始できる．OLSR
（optimized link state routing），TBRPF（topology broadcast
based on reverse- path forwarding）と呼ばれる方式によって実現
されている．後者は，通信の要求発生時に適切な経路表を作成する
方式であり，ノードの負担を極力減らすことができる．DSR
（dynamic source routing），AODV（ad hoc on-demand distance
vector）と呼ばれる方式によって実現されている．
