# 2・4・5 分散コンピューティング技術

> **章**: 第2章 基礎技術
> **節**: 2・4 情報通信ネットワーク制御の基礎

クライアントサーバアプリケーションの応用として,それぞれ分散した環境上で動作する複数のプログラムが相互にデータ交換をすることで協調動作を行うアプリケーションの実現要求も高い.それを支える技術が分散コンピューティング(distributed comput-ing)技術である.これらは2・4・2・e項にて前述した上位層プロトコルとして構築されている.分散コンピューティング技術は,ソフトウェア工学の分野で進展したオブジェクト指向システムのネットワーク化との関連が深い.プログラミング言語JAVA やその応用,CORBA と呼ばれる規格により,分散オブジェクトの設計およびその連携を支援する環境が提供されてきた.近年はインタネット(ウェブ)技術との親和性の高さが重視されており,ウェブサービスやUPnP(universal plugand play)など,はん用のインタネットプロトコル,およびデータ形式としてXML(extensible markup language)を用いた標準化技術が中心となり構成されている.a. 分散コンピューティングに必要な機能分散コンピューティングのミドルウェアとして必要とされる一般機能がアプリケーションに対してどのように提供されているかを,機能別に整理する.. アドレッシング機器をネットワークに接続する際に,その機器またはサービスの識別子(ID : identification)を割り当てる機能である.ID にはネットワーク構成に依存しない不変なものと可変なものがある.. 機能・サービス記述ネットワークに接続された機器が,どのような種類のものでどのような機能をもつか,また,それにアクセスするにはどのような手段が必要かを記述した情報である.. デバイス・機能発見ネットワークに接続されている機器およびその機器がもつ機能や情報を,任意の時点で検索できる機能である.. メッセージ伝達サービスを提供するモジュールと利用するモジュールは,同一機器内に存在する場合もあるし,ネットワーク上に分散する場合もある.どのような状況でも,モジュール間で通信可能にする機能のことを指す.. 機器導入通知・アナウンス機器やサービスがネットワークに接続した際に,あるいはその後必要に応じてその存在を他の機器に通知する機能である.. イベント通知機器の状態変化等の情報を非同期的に通知する機能である.クライアントは必要に応じて,興味あるイベントのsubscribe/unsubscribe(登録/解除)を行うことができる.. 一貫性管理ネットワークが不安定な状況でも,機器やサービスの一貫した情報管理を行う機能である.. ユーザインタフェース(操作インタフェース)機器をどのように操作するかを,より簡単にユーザに提示する機能である.機器の位置に依存せず,機器の制約からも解放される必要がある.. 既存機器対応対象プロトコルをサポートしていない機器や,異なるプロトコルのネットワークに接続されている機器とも相互接続可能にする機能である.. セキュリティサービスへのアクセスを制限,またはサービス利用に際して認証を行う機能である.必要に応じて暗号化処理を行う.b. メカトロニクス応用例ネットワーク技術のメカトロニクスへの応用として,物理的なセンサを仮想化した機能オブジェクト,アクチュエータを仮想化した機能オブジェクトなどをそれぞれ図2・39ユビキタスネットワークの構成要素(例)図2・40 分散コンピューティングのメカトロニクス応用例設計し,それらを遠隔プログラムから制御・情報収集するシステムなどが盛んに検討されている.これは,各機器が保有する機能オブジェクトに対し,ネットワークを介して離れたクライアントからデータを入出力することでシステム制御や情報連携を行うものである.それらの応用に分散コンピューティング技術が利用されている.図2・40に,家庭向けの応用例として,ホームロボットやネットワーク家電の連携システム例を示す.センサやアクチュエータを搭載するホームロボットやネットワーク家電などの対象機器と,ユーザ端末やアプリケーションプログラムなどのコントローラとの間で,XML記述された機器や搭載機能の仕様や制御メッセージを,ウェブサービスやUPnPなどの分散コンピューティング用のミドルウェアを用いて送受信することで制御を行う.このようなミドルウェアを利用することにより機器側のセンサやアクチュエータをオブジェクト化し,その操作方法および仕様を定義する程度で,コントローラがネットワーク家電やホームロボットの搭載機能の仕様を理解したうえで必要な制御を行うことが可能になる.上記ミドルウェアのプロトコルは,HTTPやSOAP(simpleobject
