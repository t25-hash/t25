# 6・3・6 冷熱発電の機器構成

> **章**: 第6章 未利用エネルギー
> **節**: 6・3 LNG冷熱発電

代表的なLNG冷熱発電方式のランキンサイクルの機器構成を図
6･24に示す．図6･25に世界初の出力1450kWの冷熱発電商用機
の全景を示す．主たる機器は，右からプロパン蒸発器，凝縮器，ポ
ンプ，タービンおよび発電機で構成される．
ランキンサイクル設計上のポイントを(ａ)～(ｃ)にまとめる．
図6･26 LNG流量に対する発電出力特性
(ａ)凝縮器は，蒸発器より高位置に設置され，サイホン効果を
利用して，気化器運転で，ポンプが停止しても，各バイパス弁が全
開となり，蒸発器と凝縮器の液面差によりプロパンが循環し，蒸発
器で海水から得た熱を凝縮器でLNGに伝える．
(ｂ)凝縮器伝熱管下部の飽和器により，凝縮した過冷却プロパ
ン液とタービン排気ガスを気液接触させ，凝縮液を飽和温度に近づ
けてプロパン循環量を増やし発電出力を向上できる．
(ｃ)プロパンポンプは，モータとポンプが一体となったサブマ
ージドタイプであり，常温から－40℃に減圧する過程で発生するガ
スを凝縮器に戻し，キャビテーションを生じることなく運転でき
る．
