# 3・1・1 センサの分類

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・1 センサ技術

ここではメカトロニクス機器、ロボットの開発に使われるセンサの典型的な構造と機能を記述する(図3・1)。ロボットによる自律的な作業の実行は、基本的にはロボットに与えられた一連の作業手順に基づいて行われるが、この作業の実行においてロボットはロボット自体に関わる状態(内界状態)や、また必要に応じて作業の対象ならびにその作業環境の状態(外界状態)について把握し、時としては作業内容を変更しなければならない。例えば、ロボットハンドの制御には、腕の各関節制御のためのサーボ系に関節角度、角速度、角加速度の値が内界状態量として必要とされるが、これらの値は関節軸に付けられたセンサによって検出され制御される。また、ハンドによる物体の把持作業などの実行には、あらかじめその物体の位置などの外界状態量の検出が必要となる。例えば、山積みされた部品の中から1個の部品を取出すためには、その部品の位置、姿勢が識別されなければならない。このように、ロボットを用いて目的とする作業を行わせる場合には、上記のようにロボット自体を制御するための内界状態量による制御と、作業内容に応じた外界状態量による制御があり、ロボット作業の実行のためには作業内容に応じた状態量を獲得するためのセンサシステムが必要不可欠となる。上記の意味からセンサは、ロボット自体の内界状態量を検出する内界センサと、物体あるいは作業環境における外界状態量を検出する外界センサとに大きく分けられる。a. 内界センサ(internal sensor)内界センサとしては、ロボットハンドの関節角度、角速度、角加速度を検出するためのロータリエンコーダなどが多く用いられている。特に最近では、モータユニットの価格を下げる目的などのため、センサレスモータが提案されている。このセンサレスモータでは、モータにかかる電流を内界状態として直接検出し、モータにかかるトルクを検出する技術も多く用いられている。b. 外界センサ(external sensor)一方、ロボットが作業対象とする物体や作業環境などの検出に用いられる外界センサとしては、三次元空間に存在する作業対象物体や環境などの情報を検出することから、領域的・空間的な情報獲得が必要となる。従来、このような空間的な情報獲得手法として多く用いられているものは画像センサであるが、得られた画像から必要な特徴量を抽出する必要がある。また、外界センサは、ロボットと接触することで外界状態量を獲得する接触型センサと、画像センサに代表される非接触で外界の状態量を獲得する非接触センサに大別される。. 接触型センサ(contact sensor)接触センサは、ハンドのための触覚センサ、ならびに手先効果器のための接触情報を取得するセンサである。移動ロボットの障害物回避や、歩行ロボットの接触情報検出に用いられるタッチセンサなどもこの範ちゅうに入る。図3・1 センサの分類a項で述べたように、センサレスモータは基本的には内界センサとしての働きをもつが、得られる内界状態がそのまま外界状態を表すこともあることから、外界センサの代替を行うことが可能な内界センサの一例として挙げることができる。ハンドに用いられる接触型センサは、触覚センサと力覚センサに大別される。前者はハンドによって物体を把持したとき、指と物体との間に働く力を検出するセンサであり、把持状態、把持された物体の形状識別などの検出、識別を行う。触覚センサはさらに、接触覚センサ、圧覚センサ、すべり覚センサなどに分類される。力覚センサは、指またはハンドに加わる力ならびにモーメントを検出するセンサである。特に手首部分に付けられ、ハンドに加わる力・モーメントを検出するセンサをリストセンサという。力覚センサは穴に棒を挿入するはめあわせ作業などに応用され、軸に加わる力・モーメントをハンド座標系の各成分として検出し、穴と軸との接触力を検出するために用いられる。. 非接触型センサ(non-contact sensor)非接触型センサの代表としては、昨今の撮像素子の普及からカメラを用いた二次元非接触型センサが数多く使われている。特に、ロボットの視覚機能として必要とされる物体認識、位置計測などのためのセンサとして広く使われている。非接触型センサによる情報の取得形態には、工業用テレビカメラあるいはCCD(charge coupled device)カメラなどを用いてワークなどの対象物、作業環境を画像として取得し、その画像処理から必要な情報を抽出し識別するもの、また対象となる物体に光、超音波などを照射して、その反射波などから、その伝搬時間、位相変化などから距離情報、物体表面の特徴を見るものなどがある。この非接触型センサはさらに、視覚センサと効果器センサに分類される。視覚センサは作業環境の領域、空間情報の取得を主目的とし、200~1000mm程度離れた対象について扱うことが多いが、例えば移動用ロボットの場合、1000mm以上離れた環境の情報を取得する場合もある。この視覚センサは、カメラの視野のようにセンサがある領域の検出範囲を有しているうえ、走査による領域の情報が取得可能である。さらに非接触であるために、対象にきずをつけるなどの影響を与えないといった特徴がある。しかし、視覚センサによって取得された画像の処理は、処理内容によっては処理時間が膨大となり、ロボットのリアルタイム制御への応用が困難となる。このため、センサの情報取得とその処理時間の短縮を考慮したシステムの構成が求められる。c. その他の分類上記のようなロボットの内界状態、外界状態をそれぞれ検出するためのセンサの分け方のほかに、センサの物理的に配置されている位置関係による、ロボットに搭載したセンサと環境に付随したセンサという分け方もある。. 搭載センサ(robot sensor)ロボットへの搭載型のセンサにはa項で前述した内界センサも含まれるが、外界センサの一部も含まれる。例えば、ロボットアームに取付けられた力覚センサ、触覚センサ、さらには視覚センサにもハンド・アイシステムなどに代表されるロボットに搭載された視覚センサや、障害物回避を行うための超音波センサ、レーザレンジファインダなどがある。. 環境センサ(environmental sensor)一方、最近のロボット技術では、すべてのセンサを搭載しないで環境側にインフラストラクチャ技術として入れ込み、それらのセンサデータを通信でロボットに送り制御する手法が広く行われている。この場合、環境センサにはセンサの位置をあらかじめ知っておく必要があるものの、その環境の中では搭載センサのもつ限界(例えば画像などの場合、視野の制限など)を超えた計測情報を用いてロボットを動かすことが可能であるなど、ロボットからでは計測できない広域な情報を用いて制御を行うことが可能であるなどの利点がある。
