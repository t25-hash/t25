# 3・1・2 内界センサ

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・1 センサ技術

ロボットなどメカトロニクス機器を制御して所用の機能を実現するためには、機器の内界状態とその周囲の状態を精度よく検知し、認識することが欠かせない。例えば、マニピュレータで物を把持するために腕を伸ばすには各関節の角変位を知り、それを適切な角度に安定して位置制御する必要がある。移動ロボットでは現在位置の認識が自在な移動の基礎となるが、外界を観察してその位置を知る処理をサーボレベルのような高頻度で行うことは難しく、車輪回転速度などの情報を積分して位置を推定する方法が一般的にとられている。このように内界センサ(internal sensor)は、おもにメカトロニクス機器の内界状態を検知するために用いられる。すなわち、内界センサは位置、変位、速度、加速度、力・トルク、温度などを検出するセンサで、その主要なものは表3・1に示すように分類される。ここで、並進運動では速度センサが、回転運動では角加速度センサに対するものは一般的には用いられない。また、ロボットの力・トルクや方位・姿勢など純粋には内界状態ではないものがあるが、環境に対するロボットの状態などを検知することから、これらのセンサも内界センサに含めて扱うことがある。内界センサの一部は、変位センサや速度センサなど、ロボットの関節のようにメカトロニクス機器の自由度の運動制御用サーボセンサとしても使用されることが多い。a. 位置センサおもに運動自由度の可動範囲を規制する用途に用いられるセンサで、リミットスイッチとフォトインタラプタが代表的なものである。.
