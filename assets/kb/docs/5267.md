# 6・3・8 運転課題とその対応

> **章**: 第6章 未利用エネルギー
> **節**: 6・3 LNG冷熱発電

試運転において経験したトラブル対応を列挙する．
(ａ)プロパンタービンへのミストの同伴 タービンに入るガ
ス中のミストをデミスタで除去できず，発電出力が大きく変動し
た．原因は，蒸発器管群での気泡率が大きいため，液面計の液位が
実際の液位より低い値を指示し，デミスタの底が液につかったこと
による．そこで蒸発器の液面を低下させ，プロパンの蒸発伝熱特性
は変えずに，発電機の出力を安定させることができた．
(ｂ) 凝縮液の蒸発器への供給方法 蒸発器伝熱管に同一の熱
負荷がかかるよう伝熱管とデミスタの間に分散管を設置した．
(ｃ) 天然ガス加温器の着氷 天然ガス加温器のガス入口部伝
熱管に厚い氷が付着し，その後流部で渦流が発生し壊食が生じた．
そこでチタン管の壊食に対する耐食性などの確認を行い，伝熱管を
黄銅管からチタン管に変更した．
(ｄ) タービンのシール機構 プロパンタービンではメカニカ
ルシールを使用したが，試運転においてプロパン内にシールオイル
を凝縮分離させ除湿器の負荷を下げる．
が混入し，蒸発器でフォーミング現象を生じた．各種の対応策で安
定した運転が可能となった．
(ｅ) 凝縮器の不凝縮性ガスによる伝熱阻害 LNGと冷媒の
伝熱性能が凝縮器の圧力を下げるにつれ，凝縮器出口天然ガスの温
ェネレーションの排熱も利用できる．
度が下がった．原因は，冷媒ガス系に混入した窒素ガスがタービン
の運転で凝縮器に集められ，伝熱管表面が不凝縮性ガスで覆われた
ためである．そこで凝縮器をブロックし，内部ガスを放散すること
冷器で冷却する．
で，設計どおりの伝熱性能を得ることができた．
