# 3・1・5 センシング技術

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・1 センサ技術

ここでは,センサの情報処理技術,信号処理技術,センサ融合技術に関わる理論,ハードウェア,信号処理機能組込み型センサなどを解説する.a. 信号処理とセンサ信号処理(sensor signal processing)ロボットは基本的にはセンサから情報を得て,処理システムで処理し,アクチュエータに制御信号を発することで運動を制御している(図3・47).まず,センサ情報に含まれない情報は処理をしても得ることができないが,通常,センサ情報には制御情報に直接関係のない情報やノイズが多く含まれることから,信号処理を行うことで実際にアクチュエータを動かすのに有効な制御情報が抽出可能となる.本項ではおもにセンサに依存する信号処理のハードウェア構成について議論したい.また,最近の信号処理の多くはディジタル信号処理であることから,ここではディジタル信号処理に限定することとする.センサデバイスには,ディジタル信号をそのまま出力することが可能であるセンサデバイスも存在する.例えば,従来の画像処理システムでは,カメラはNTSCなどの規格に基づいたアナログ信号を出力しており,計算機に入力するためには画像入力ボードなどを用いてA-D 変換し,計算機に取込んで処理を行っていた.最近ではカメラインタフェースとしてIEEE1394などのディジタルインタフェースが標準化され,画像劣化がなく取込めるディジタル入出力による画像取込み,信号処理技術が標準となりつつある.さらには,カメラの撮像素子レベルから考えると,画像の撮像素子から出てきたアナログおよびディジタル信号を画像の補正を行うための処理装置にそのまま直接入力し,修正された画像をディジタル入出力インタフェースを通して通常のPCに取込む手法が一般的となりつつある.このように信号処理のレベルは,センサデバイスから信号を効率よく取るための信号処理と,その上位で制御信号を出すための処理の二つに大きく分けられる(図3・48).b. センサ統合・融合(sensor fusion)ここでは,複数のセンサを統合したり融合したりする技術について議論する.センサにはそれぞれ計測種類,計測範囲や計測精度などの制約があることから,センサ統合・融合技術を用いることでそれらの制約をなくすことを目的としている.. 計測範囲統合・融合例えば,画像センサは光学レンズを通して撮像素子に写った画像を処理することから投影型のセンサであり,視野としてはレンズの焦点距離に依存して無限長の四角すい(錐)の形状をとる.これに対し,超音波センサは超音波の反射を見るセンサ原理であることから基本的にはすいの形をとるが,画像センサが周波数の高い光であることから無限遠まで取れるのと異なり,超音波は周波数が低いことから計測レンジは超音波の減衰によって決定する.異なる計測レンジのセンサを複数用い,実際にロボットを動かす環境を広く網羅するため,センサの計測範囲を考慮したセンサ融合が必要となる(図3・49).. 計測精度統合・融合同じ計測精度を有する複数のセンサ,もしくは異なる計測精度を有する複数のセンサを用いて,単体のセンサより高い精度での計測を可能とする技術である.例えば,画像センサを複数台用いたステレオビジョンの場合,単体のカメラではその計測原理が透視投影であることから奥行きの計測は不可能であるが,複数のカメラを用いることで奥行きの計測が可能となるだけでなく,空間解像度を上げることも可能となる.最近では,特に撮像デバイスに関しては撮像素子自体を微小に動かして複数枚の画像を獲得し,それを融合することで計測精度を上げようとする試みや,絞りを変えながら撮られた複数枚の画像を融合することで感度の高い画像を獲得する手法,焦点距離を変えながら得られた複数枚の画像を融合することで焦点深度の深い画像を得る方法など,撮像素子レベルでのインテグレーテッドビジョン技術の開発が行われている.この件に関しては,c項で詳しい事例を挙表3・3
