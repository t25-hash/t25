# 3・2・2 アクチュエータの分類とその特徴

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・2 アクチュエーション技術

アクチュエータの分類は動作原理によるものが一般的であり,従来型と新原理アクチュエータに大別される.従来型アクチュエータは電気や流体のエネルギーを機械エネルギーに変換するものであり,それぞれについて小型・軽量化,高効率化,制御性能向上などが進められている.新原理アクチュエータは新しい材料や現象を用いたアクチュエータであり,圧電素子や形状記憶合金などを用いたアクチュエータが実用化されている.さらに,機能性流体や高分子材料を用いたアクチュエータの研究開発が進められている.a. 電気アクチュエータ(electric actuator)電気アクチュエータは電流と磁界の相互作用(フレミングの左手の法則)により生じる電磁力により機械的運動を得るものであり,回転運動を行う電気モータと直線運動を行う電気リニアモータに大別される.電気モータは単なる動力用あるいは運動制御用として使用され,後者の用途に用いるモータはサーボモータと呼ばれる.電気サーボモータには,直流電源で動作する直流(DC: directcurrent)サーボモータ,交流電源で動作する交流(AC: alternat-ing current)サーボモータおよびステッピングモータがある.ステッピングモータは一つの入力パルスに対して一定の角度だけ回転するモータである.その特徴は開ループ制御ができることであり,位置検出器を必要としない.入力パルスに同期して回転し,停止時にはモータ自身に高い保持能力があるため,簡便に高精度の速度や位置の制御を行うことが可能である.これに対し,DCサーボモータやACサーボモータを用いた制御系は位置検出器を備えたフィードバック制御方式となる.DCサーボモータは入力電流と発生トルクが比例し,また速度制御が容易であることから長い間電気サーボモータの主流を占めていたが,ブラシと整流子のしゅう動摩耗部が存在することによりメンテナンスやノイズの点で不利である.これに対して,サーボドライバの高機能化と低価格化により,機械的な整流機構を有しないACサーボモータがメンテナンスフリーを特徴として急速に普及している.電気リニアモータでは,リニア誘導モータ(LIM : linear induc-tion motor),リニアパルスモータ(LPM : linear pulse motor)およびリニア直流モータ(LDM : linear direct current motor)がよく使用される.LIM は磁気浮上鉄道用として注目されて以来,搬送分野で多く使用されている.LPM は位置保持機能を有し開ループ方式による制御が可能であり,簡易な位置制御機構としてOA(office automation)機器などへの応用が進んでいる.LDM はDCサーボモータと同様の特徴を備え,頻繁な正逆転や変速などに強いサーボモータとしての使用が有利である.b. 油圧アクチュエータ(hydraulic actuator)油圧アクチュエータは圧油が有する流体エネルギーを直進,回転および揺動運動の機械エネルギーに変換するものであり,それぞれシリンダ,モータおよび揺動型アクチュエータと呼ばれる.出力パワーが大きい,パワー密度が大きく小型化が可能,力-質量比およびトルク-慣性比が大きく高速な応答が可能,高精度の位置や速度の制御が可能などの特徴を有する.アクチュエータの動作速度はアクチュエータに供給する圧油の流量により制御でき,油の流量は油圧制御弁によって調整される.油圧制御弁に電気油圧サーボ弁を用いた制御系は電気油圧サーボ系と呼ばれ,各種の産業用ロボット,工作機械,建設機械,航空機や船舶,自動車など広範な分野で使用されている.作動油の温度や汚染管理の必要性,油漏れによる環境汚染,油圧タンクなどの付帯設備によるスペース効率の低下などの欠点があり,使用環境が制限されることがある.c. 空気圧アクチュエータ(pneumatic actuator)空気圧アクチュエータは圧縮空気のエネルギーを機械エネルギーに変換するものであり,直線運動をする空気圧シリンダが代表的である.また,空気圧モータ,揺動型アクチュエータ,ベローズ,ダイアフラムなども使用される.空気圧アクチュエータは手軽に利用できるアクチュエータとして生産システムの自動化・省力化,各種産業機械,輸送機械など多分野で使用されている.空気圧アクチュエータは発生力の調整が容易であり,空気漏れがなければエネルギー消費なしでクランプ状態を維持できるため,ワークの搬送や端点位置決め,クランプ作業,ロボットハンドなどに多用されている.空気圧シリンダが代表的であり,基本構造は従来から大きな変化がないが,小型・薄型化が進むとともに,ロッドレスシリンダや低摩擦型,低速型などの特殊用途向けシリンダが開発されている.また,方向切換弁やガイド機構,ロック機構などを組込んだ複合型シリンダも一般的になっている.ガイド付きシリンダはテーブルやガイドとシリンダが一体化され,設置時の平衡度などを気にすることなく搬送装置や位置決め機構が構成できる.空気圧アクチュエータは作動流体がその圧縮性により柔軟なばね特性を有し,圧力の調整によって発生力を容易に制御できる利点がある.また,シリンダなどの従来型アクチュエータに加えて空気圧ゴム人工筋が福祉・介護分野などへの応用を目的とした次世代アクチュエータの一つとして注目されている.d. 新原理アクチュエータ(new actuator)ロボットやメカトロニクス機器の開発が進むにつれて,より小型・軽量かつ制御性に優れた駆動機構に対する要望が高まり,新原理を利用したアクチュエータの開発が行われている.圧電アクチュエータ,超音波アクチュエータ,形状記憶合金(SMA : shape memory alloy)アクチュエータなどが実用化されている.このうち,1 m以下の制御精度が要求される微動位置制御機構に圧電アクチュエータが用いられる.その応用分野は多岐にわたり,顕微鏡用ステージ,超精密直進案内機構,工作機械の切削誤差補正装置,油圧および空気圧制御弁の電気-空気圧変換部,さらにVTR(videotape recorder)ヘッドなどに使用されている.また,圧電素子の変位を拡大して使用する手段として,てこの原理を利用した拡大機構やインチングワーム機構,圧電素子の急速変形に伴う慣性力と摩擦力の相互作用を利用した駆動機構などが用いられている.超音波モータも圧電素子を用いた摩擦駆動型のアクチュエータである.形状記憶合金アクチュエータは,Ni-Ti合金など特殊成分の合金の形状記憶効果を利用したアクチュエータである.これは合金を変形させても加熱により変形前の形に回復する現象であり,温度制御によりアクチュエータの変位と発生力を制御できる.Ni-Ti系のSMA は電気抵抗が大きいため,直接通電加熱(ジュール加熱)が可能である.このアクチュエータは力-質量比が大きく,小型・軽量でしゅう動部のない駆動機構が構成できる.その応答速度は熱系の応答に支配されるため,加熱・冷却時間が重要である.アクチュエータの理想形態の一つは生体筋肉であり,それに類似した機能と性能を有する人工筋肉の研究開発が進められている.それらはソフトアクチュエータと総称され,高分子材料やゴム材料を用いたアクチュエータが開発されている.高分子アクチュエータとしてはゲルアクチュエータが代表的であり,将来の人工筋肉の実現を目指している.一方,ゴム材料を用いたアクチュエータは空気圧ゴム人工筋として実用化されている.マッキベン(McKibben)型人工筋に加えて種々の形式のゴム人工筋が開発されているが,これらは小型・軽量・柔軟で,出力-質量比が極めて大きい.
