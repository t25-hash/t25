# 3・3・2 ハウジング

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・3 実装技術

a. ハウジング設計の注意事項ハウジング(housing、筐体)はメカトロシステムの内部と外部を分離し、その干渉を最小化するものであるが、以下の点を考慮する必要がある。また、ハウジングの例を図3・137に示す。. 内部の保護メカトロシステムが置かれる環境から内部の機構、制御装置を保護する必要がある。外部からハウジング内部への侵入を防ぐ対象としては、水、粉じん、電磁ノイズ、熱、振動等が考えられる。水、粉じんに関しては表3・9に示すような保護等級が定められている。. 外部への妨害低減内部の機構が発する音や振動の遮へいと、内部の電子機器が発する電磁ノイズのふく射を低減させる。. 安全性の確保内部に高速回転部、高温部、高電圧回路図3・137 ハウジングの例(コントローラ、ロボットアーム)表3・9 IP保護等級記号番号内容保護なし50mmより大きいものが侵入しない手足の保護12mmより大きいものが侵入しない指先の保護第1記号2.5mmより大きいものが侵入しない1.0mmより大きいものが侵入しない粉じんの侵入に対する保護多少、侵入しても影響を受けない粉じんが侵入しない保護なし鉛直に落下する水滴に対する保護±15°以内で傾斜したときに、鉛直に落下する水滴に対する保護鉛直から60°以内の噴霧水に対する保護第2記号飛まつに対する保護任意方向からの噴流水に対する保護任意方向からの強い噴流水に対する保護水中への浸漬に対する保護連続的な水没に対する保護注:IPの表記方法は二つの数字の組合せによる。第1記号は固体物に、第2記号は水に対する保護の程度を表す(例:IP65)。等が存在する場合、その部分に人体が容易に接触しないように保護する必要がある。また、内部の機構の破壊や大電力部品の破裂等から人体を保護する必要がある。保守点検用の扉等が存在し、内部が危険である場合は、扉が開いたことを検出して動作を停止させる必要がある。表3・10に代表的な安全規格を示す。これら以外に、経済性、保守性、美観、環境コスト等を十分に考慮する必要がある。b. ハウジングの材料ハウジングの材料は金属と樹脂に大別され、以下の特徴がある。. 金属樹脂に比べて頑丈であり、電磁ノイズの遮へいが可能であるが、重い、加工がしにくいという短所がある。. 樹脂軽く、成形用の型により量産が容易で、美観に優れるが、電磁ノイズの遮へいが行えない、熱に弱いという短所がある。ただし、電磁ノイズの遮へいについては内壁に導電塗装を施すことである程度の対処が可能である。
