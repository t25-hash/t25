# 6・3・4 ヒートパイプ式均熱板

> **章**: 第6章 熱輸送デバイス
> **節**: 6・3 ヒートパイプの応用

高精度の温度均一性が求められる半導体シリコンウェーハ
（wafer ）や液晶パネルなどの加熱プロセスでは，下面に面状発熱
体を接着した均熱板材料（例えばアルミニウムなど）にウェーハを
搭載し，この発熱体に発生する熱を均熱板材料に伝え，熱伝導によ
る熱拡散によって全体を均一に加熱する方式が一般に用いられてい
る．この方式の場合，面状発熱体と均熱板材料との接触状態にばら
つきが生じやすく表面の温度むらが生じやすいこと，均熱板の熱容
量のために熱応答性が悪いことなどの問題点があり，これらの課題
を解決するためにヒートパイプを用いることができる．
ヒートパイプ式均熱板（heat pipe isothermal heating plate）は
ヒートパイプの優れた熱輸送特性と高い均熱性を応用して精密な温
度履歴を必要とする半導体の加熱プロセスに用いられるものであ
る．その外観を図6･36 に示す．円盤状をしたヒートパイプ式均
熱板内部には中空部が形成されており，内部を真空排気した後，所
定の作動流体が充てんされている．均熱板内部には電気ヒータが設
置されており，この発熱をヒートパイプの蒸発部で効率的に作動流
体に伝え，作動流体の蒸発と凝縮による熱輸送によって均熱板各部
を均一に加熱するものである．

図6･36ヒートパイプ式均熱板
このヒートパイプ式均熱板は，作動流体の蒸気が微小な温度差で
移動するため面内の均熱性が高い，蒸気流により瞬時に熱を均熱板
各部に拡散させるため応答性が良い，熱容量が小さく温度上昇に要
する加熱量が小さくてよいため省エネルギーである，均熱板全体が
均一に温度上昇するため熱変形が小さいなどの特徴をもっている．
