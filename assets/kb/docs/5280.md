# 3・3・4 項で説明する。

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・3 実装技術

c. プリント基板仕様の注意点メカトロシステムで特に注意するべきボードの仕様(ボード固有の仕様以外)を以下に示す。. 動作温度範囲産業用途では周囲の気温が-20~60°C程度より広い範囲で動作可能であること。. 耐振・耐衝撃性機構部の振動がボードに伝わることが多いため、耐振性が高いこと。. 耐ノイズ性モータ、サーボアンプ等が大きなノイズを発生することが多いため、耐ノイズ性が高いこと。. 長期供給性産業用途では10年以上装置を使用することが少なくないため、ボード故障時の交換、修理が長期間可能なこと。. 鉛フリーはんだEU(European Union、欧州連合)による電子機器への有害物質の規制が2006年7月から開始されたが、今後、鉛(Pb)等の有害物質を含まないボードを選定する必要がある。
