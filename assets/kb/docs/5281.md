# 3・3・4 配線・接続

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・3 実装技術

配線(wiring)は個々の部品間を接続し,信号,電力を伝達するために使用される.半導体チップ内のトランジスタ間,プリント基板上の電子部品間,ハウジング内のプリント基板間,ハウジング間,インタネットというさまざまなレベルの配線・接続があり,全く異なる技術が使用される.ここではプリント基板間,ハウジング間の配線を中心に説明する.a. コネクタ(connector)コネクタは部品,配線の脱着を容易にするためのものであり,電圧,電流,周波数,信号数,かん合強度等を考慮して選定する必要がある.また,メカトロシステムではコネクタの接点に振動,衝撃が加わらないように注意する必要がある.表3・11にコネクタの種類を示す.b. 電線(electric wire,electric
