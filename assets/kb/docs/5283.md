# 3・3・6 EMC対策

> **章**: 第3章 ロボット・メカトロニクス要素
> **節**: 3・3 実装技術

EMC(electromagnetic compatibility)とは電磁環境適合性と翻訳されており、電磁ノイズ(electromagnetic noise)の放出に関するEMI(electromagnetic interferences)と電磁ノイズに対する耐性に関するEMS(electromagnetic susceptibility)からなる。a. EMC対策、規制の必要性メカトロシステムにはセンサ信号のような数V という微小信号を取扱うものから、モータ用インバータのように数kV の電圧を取扱うものまでが存在し、後者の発生する電磁ノイズで前者が誤動作するという状況に陥りやすい。このためノイズの放出量を減少させ、ノイズに対する耐性を向上させる対策が必要である。また、電磁ノイズはテレビ、ラジオ、携帯電話等の機能を妨害する可能性が高く、その影響が大きいため、表b. 対策方法、設計の留意点電磁ノイズは空間を伝わることと、図3・142のように配線、電子部品等が理想的でないことが原因で発生することが多いうえ、回路図を一見しただけでは発生を予想できないことがその対策を困難なものとしている。効率のよいノイズ対策のために電磁気学の習得を推奨する。EMC対策はノイズの発生部、伝搬部、受容部のどの部分で対策を施すかにより、また空間を伝搬するノイズか、電線を伝搬するノイズかによって最適な対策が多様に分かれる。以下に対策の考え方、例を示す。. ノイズ発生部の対策ノイズ発生部の対策を以下に挙げる。表3・17 EMC 試験電磁波放射量装置から空中に放射される電磁波強度が規格に適合するかCISPR,VCCI,FCC注CISPR :International Special Committee on Radio Interference,VCCI:Voluntary Control Council for Interference byInformation Technology Equipment,FCC:Federal Communications Commission7―88メカトロニクス・ロボティクス(1) 発生する電磁ノイズの量は電流、電圧の変化速度に比例するため、適度なスイッチング速度を保有する部品を採用し、不必要に急峻な電圧、電流の変化を避ける。(2) 放射される電磁ノイズの量は、急峻に電流・電圧が変化する導体のサイズ増加とともに増大するため、これらの部分を最小化する。また、逆方向に変化する部分を近接させることにより、電磁ノイズを打ち消し合うようにする。具体的には配線長を短くすること、ツイストペア線を使用すること、バイパスコンデンサ等により高周波電流の流れる経路を小さくすることなどである。. ノイズ伝搬部の対策ノイズ伝搬部の対策を以下に挙げる。(1) ノイズは伝搬距離とともに減衰するため、ノイズ発生部と受容部の距離を大きくする。具体的にはインバータとセンサはハウジングを分離すること、電線を離すことなどである。(2) 電磁波を反射する金属でノイズ発生部、または受容部を取り囲む。具体的にはハウジングに金属を使用すること、電線にはシールド線を使用することなどである。また、最近では電波吸収シートをノイズ発生場所に張り付けるという対策も増えてきた。電線を伝搬するノイズに対してはノイズフィルタを挿入する。. ノイズ受容部の対策ノイズ受容部の対策を以下に挙げる。(1) 受信するノイズの量は受信部導体のサイズとともに大きくなるため、微小信号を取扱う回路のサイズを極力小さくする。具体的にはセンサ直近にアンプを設けることなどである。(2) 信号の周波数帯域幅を必要最小限とし、かつ帯域通過フィルタを用いる。ノイズのエネルギーは周波数帯域が狭いと小さくなるためである。(3) 回路のインピーダンスを下げる。信号線が受けるノイズのエネルギーが同じでもインピーダンスが低ければノイズ電圧が小さくなること、またはノイズ源から回路までのインピーダンスと回路のインピーダンスでノイズ源の電圧が分圧されるためである。従来、ノイズ対策は試作後に行うか、経験に基づき設計時に実施していたが、最近では設計時にEMIシミュレータを利用して実施する例も見られる。文献(1) (株)村田製作所ホームページ、 http://www.murata.co.jp/arti-cles/ta02bl.html(2) 多摩川精機(株)カタログ、T12-1090N13、(2004)。(3) 日本ロボット学会編、 新版ロボット工学ハンドブック、 (2005)、108-127、コロナ社。(4) 長谷川健介・増田良介、 基礎ロボット工学、 制御編、 (1994)、 105-151、昭晃堂。(5) 篠田裕之、日本ロボット学会誌、20-4(2002)、35.(6) Ciddor,P.E.,Appl.
