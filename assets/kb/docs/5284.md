# 4・1・1 ロボットの機構

> **章**: 第4章 ロボティクス
> **節**: 4・1 機構

ロボットの構造は一般に，作業を遂行するためのマニピュレータ
と，マニピュレータを作業点近傍の適切な位置まで移動させるため
の移動機構からなる．マニピュレータは，物体を把持するなど作業
対象と直接相互作用する手（ハンド）と，そのハンドを作業空間に
移動させるための腕（アーム）から構成される．移動機構はアーム
の機能を拡大する役割を果たしているとみなすこともできる．作業
範囲が限定されるロボットでは移動機構を含まず，据付け型とする
ことが多い．
ロボットでは種々の柔軟な運動機能が求められるため，ハンドや
アームは一般に複数の節からなるリンク機構で構成される．また移
動機構でも，脚型にはリンク機構が用いられる．そして，節と節の
結合部に駆動機構を組込み，それを制御することによって必要な運
動を作り出す．動力化された節と節の結合部を関節（joint）と呼
ぶが，JIS（Japanese Industrial Standards，日本工業規格）では
リンクの連結部分をジョイントと呼び，直進（直動）するジョイン
トを直進ジョイント，回転するジョイントを回転ジョイントあるい
は関節と呼ぶ．しかし実際には，ロボットのジョイントは関節と呼
ばれることがほとんどである．

4・1・3アームの機構
a.アーム（arm）の自由度 いくつかのタイプの関節を適宜
連結することでアーム（あるいは脚）を構成することができる．ア
ームの先端には一般にハンドが付いており，アームの役割はそのハ
ンドを作業空間内の適切な場所に移動させることである．ハンドを
一つの剛体と考えると，三次元空間内におけるその位置・姿勢はそ
れぞれ三つの変数を含む座標，,,，α,β,γで規定でき
る．ハンドの位置・姿勢を変えるために，アームはこれら六つの変
数を操作できなければならないことから，結局，アームには少なく
とも六つの自由度が必要になる．
このようなことから，一般的な自由度構成としてロボットのアー
ムは6自由度を有する．ロボットの用途によっては限られた作業空
間を移動できればよいこともあり，そうした場合には6以下の自由
度が採用される．通常の産業用ロボットは3～6自由度のアームを
有することが多い．
