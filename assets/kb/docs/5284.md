# 6・5・1 ヒートパイプの劣化

> **章**: 第6章 熱輸送デバイス
> **節**: 6・5 ヒートパイプの信頼性

一般に何をもってヒートパイプの信頼性（reliability）というかり，基本的に外部動力を必要としない完全に外部と独立した密閉容くない状態が起きた場合に信頼性がないということができる．
ヒーていなければならない．
しかし，密閉する際に不凝縮性のガスが混は内部の清浄度が十分でないためになんらかの反応が起こり不凝縮て考えられる．
いずれの場合も，不凝縮性ガスがヒートパイプ劣化動しなくなるために，等温部分の長さが減少し，ひいては放熱面積で，作動温度を50℃程度まで下げている．
なぜなら，ガスの発生量が内表面積に比例すると考例すると考えられるからである．
したがって，内表面積が相対的にどガスの発生とその影響度が大きいということになる．
いま，内容さに反比例することから，例えば3mmφのヒートパイプと10mmφ影響度の大きさが重大なものであることが推定される．
したがっトパイプでは長期の信頼性を十分確認し確保することが重要である．
を明確にする必要がある．
ヒートパイプは極めて単純な構造であ器である．
したがって，密閉する段階と密閉した後に，本来望ましトパイプの内部は本来作動液とその飽和蒸気のみで空間が満たされ入したり，容器の密封度が完全でないためにリークしたり，あるいガス（noncondensable gas）を発生させてしまう場合が劣化としに影響するもので，これにより本来熱輸送すべき領域まで蒸気が移が減少して熱輸送素子としての性能低下を起こすのである．
