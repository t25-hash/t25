# 4・1・1 ロボットの機構

> **章**: 第4章 ロボティクス
> **節**: 4・1 機構

ロボットの構造は一般に,作業を遂行するためのマニピュレータと,マニピュレータを作業点近傍の適切な位置まで移動させるための移動機構からなる.マニピュレータは,物体を把持するなど作業対象と直接相互作用する手(ハンド)と,そのハンドを作業空間に移動させるための腕(アーム)から構成される.移動機構はアームの機能を拡大する役割を果たしているとみなすこともできる.作業範囲が限定されるロボットでは移動機構を含まず,据付け型とすることが多い.ロボットでは種々の柔軟な運動機能が求められるため,ハンドやアームは一般に複数の節からなるリンク機構で構成される.また移動機構でも,脚型にはリンク機構が用いられる.そして,節と節の結合部に駆動機構を組込み,それを制御することによって必要な運動を作り出す.動力化された節と節の結合部を関節(joint)と呼ぶが,JIS(Japanese Industrial Standards,日本工業規格)ではリンクの連結部分をジョイントと呼び,直進(直動)するジョイントを直進ジョイント,回転するジョイントを回転ジョイントあるいは関節と呼ぶ.しかし実際には,ロボットのジョイントは関節と呼ばれることがほとんどである.
