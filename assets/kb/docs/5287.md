# 6・5・4 その他の確認内容

> **章**: 第6章 熱輸送デバイス
> **節**: 6・5 ヒートパイプの信頼性

電子部品用のヒートパイプにはJIS規格やMIL規格などに示さものに次の試験がある．
ヒートパイプは可動部がなく構造がシンプルなうえ，受動部品でロヒートパイプでは，これらの試験において問題となる例はほとん響を検討する必要がある．
ヒートパイプが凍結で問題となるのは，よってヒートパイプの容器が変形する場合があり，このような場合条件に応じた適切な対策をとる必要がある．
れる試験が適用される場合がある．
その中で比較的よく用いられる高温動作寿命試験温度サイクル試験このほかには振動，衝撃などによる影響を見る場合もある．
あるため基本的には信頼性は極めて高い．
そのため銅-水系マイクどないが，温度サイクル試験での低温側においては凍結に対する影水が凍結するときに発生する体積膨張である．
