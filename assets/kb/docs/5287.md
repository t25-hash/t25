# 4・1・4 ハンドの機構

> **章**: 第4章 ロボティクス
> **節**: 4・8 に示すように,直動シリンダは人間の筋-骨格系と同様にリン

a. ハンド(hand)の分類ハンドはアームの先端に取付けられ,対象物に直接作用を及ぼす装置であり,エンドエフェクタ(end-effector)とも呼ばれる.この装置は人の手に対応する.人の手は多様な作業能力をもつが,そのすべての能力を備えたハンドを設計することは困難であり,そのようなハンドは研究段階にある.実用的観点からは,ロボットの用途に応じて必要な機能を抽出し,それらを発揮できるハンドが設計される.最も機能が限定されたハンドは特殊ハンドと呼ばれるもので,専用工具を用いるタイプである.例えば,溶接ロボットでは溶接ガン,塗装ロボットではスプレーガンであり,平らな金属板や壊れやすいガラス板をハンドリングする専用ロボットでは,磁気吸着装置や真空吸着装置などが使われる.産業用として多く用いられている特殊工具以外のハンドは,グリッパ(gripper)あるいはメカニカルハンドと呼ばれ,独立した関節のない2指あるいは3指による挟み機能をもち,特定の形状のものを把持することに用いられる.これに対し,3指以上を備え,それぞれの指に関節を有する,人の手に近い機能の実現を目指したハンドを多指ハンド(multi-fingered hand)あるいはユニバーサルハンド(universal hand)と呼ぶ.多指ハンドは,関節を有する3指以上の指を柔軟に制御することで多様な形状の物体を把持した(a) 原理図図4・16 自重補償アームの例り,対象物を操る機能を実現することを目指している.b. グリッパグリッパは挟み,つかみ,つまみなどの単機能に特化したハンドであり,対象物の形状に合わせて設計されることが多い.指の本数も2指から複数指まであるが,ほとんどが2指である.駆動用のアクチュエータは,電気式であればモータを一つだけ搭載し,ラックとピニオン,ボールねじ,平行リンクなどを使用した並進型機構が多い(図4・17).また,工場の組立ライン用の場合,複数台のロボットを同時に制御することと,組立作業においてコンプライアンス(柔軟性)をもたせるために,圧縮性流体である空気を用い,1台のコンプレッサから圧縮空気を送り込んで動作させる空気圧式機構が採用されることが多い.c. 多指ハンド人の手には5指あるが,各指が果たす機能に注目すると,小指,薬指を除く3指で十分な場合が多い.義手の研究でも,日常生活での作業の70%が3指でカバーされると報告されている.多指ハンドは3~5指で構成されており,いずれのタイプも各指に3~4の自由度を有する.ロボットのハンドの自由度を人間の指と同等にするには,根元のMP関節(metacarpal phalangeal joint)に2自由度,PIP関節(proximal interphalangeal joint)とDIP関節(distal interpha-langeal joint)のそれぞれに1自由度ずつの計4自由度となるが,先端のDIP関節に相当する自由度は省略されることが多い.なお,拇指の場合は根元からCM 関節(carponetacarpal joint),MP関節,IP関節(interphalangeal joint)となり,自由度は順に2,1,1である.多指ハンドは1970年代から多数が試作されており,岡田による3指ハンド,サリスバリ(Salisbury)らによる3指ハンドのスタンフォード大学/JPL(ジェット推進研究所)ハンド,ジェイコブソン(Jacobsen)らによる4指ハンドのUTAH(ユタ大学)/MIT(マサチューセッツ工科大学)Dexterous
