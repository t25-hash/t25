# 6・5・4 その他の確認内容

> **章**: 第6章 熱輸送デバイス
> **節**: 6・5 ヒートパイプの信頼性

電子部品用のヒートパイプにはJIS規格やMIL規格などに示さ
れる試験が適用される場合がある．その中で比較的よく用いられる
ものに次の試験がある．
(１)高温動作寿命試験
(２)高温高湿寿命試験
(３)温度サイクル試験
(４)熱衝撃試験
このほかには振動，衝撃などによる影響を見る場合もある．
ヒートパイプは可動部がなく構造がシンプルなうえ，受動部品で
あるため基本的には信頼性は極めて高い．そのため銅-水系マイク
ロヒートパイプでは，これらの試験において問題となる例はほとん
どないが，温度サイクル試験での低温側においては凍結に対する影
響を検討する必要がある．ヒートパイプが凍結で問題となるのは，
水が凍結するときに発生する体積膨張である．凍結時の体積膨張に
よってヒートパイプの容器が変形する場合があり，このような場合
には，作動液を入れすぎない，外部から適当な補強をするなど使用
条件に応じた適切な対策をとる必要がある．
