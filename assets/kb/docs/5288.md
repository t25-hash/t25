# 4・1・5 移動機構

> **章**: 第4章 ロボティクス
> **節**: 4・8 に示すように,直動シリンダは人間の筋-骨格系と同様にリン

移動機構(mobile mechanism)には,大別して車輪型(wheeled type),クローラ型(crawler type),脚型(leggedtype)の3種がある.車輪型には自動車のような4輪のほかに2輪,3輪があり,脚型にも1脚(あるいは足)から8脚まで多くの種類がある.車輪の数や脚の本数により機構も異なり,さらに車輪と脚を組合わせたハイブリッドタイプである脚車輪型も試作されている.4・1・3・e項の多自由度機構も移動機構の一つに含めることができる.脚の機構は基本的にアームの機構と同等であり,2脚であれば一般的に6自由度以上のアームを2本並べた機構となる.ただし脚の先端すなわち足部(足底)の機構は,アームにおけるハンドの設計方法とは異なり,環境適応性や安定性を実現するための歩行形態や歩行制御方式に大きく関係してくる.移動機構については,4・3節で移動機構のメカニズムと制御の詳細が述べられているので,そちらを参照されたい.本節では全体的な概要をまとめるにとどめることとする.表4・1に3方式(脚式は2脚と4脚以上)の特徴をまとめる.車輪型は,自動車や自転車と同様に,平坦な場所における直進移動に関しては経済性,安定性,制御性がいずれも非常に高く,高速移動も容易に可能である.ただし,段差や不整地など環境が整っていない場所での移動は,不可能もしくは著しく制約を受ける.クローラ型は,建設現場など不整地や泥地,急斜面など環境条件が厳しい場所での作業を想定した移動機構である.そのため,不整地などに対する適応性に特に優れているが,大型になるうえ,機敏に動くことが難しい.また,クローラの構造上,じゅうたんなど柔らかい床材の場合にはきずをつけるおそれがあり,生活空間への導入には適していない.脚型のうち2脚(2足,biped)は人間と同様の形態であり,不整地,段差などあらゆる環境に適応が容易である.また,占有面積を小さくできることからロボット全体をコンパクトにまとめられる反面,安定性の維持が難しく,安全なロボットとするには多くの技術課題がある.また,アームと同様の多自由度駆動・制御となるため,車輪型と比較すると制御性,経済性(コスト)は低くなる.4脚以上になると,2脚よりも安定性は増し制御性もよくなるが,2脚ほど運動の自由度は得られない.また,ロボット全体が大型化する傾向がある.ただし,4脚はクローラ型に比べると移動の自由度が大きく,山などの自然環境下での作業用に適している.移動機構の種類によって性能の差が大きいことから,それぞれの特徴を組合わせた移動機構も検討されている.車輪の移動性能と脚の環境適応性の両特徴を組合わせた脚車輪型の移動機構はその代表である.これは脚の先端に車輪を取付けたロボットであり,平面内での移動は車輪で行い,不整地や段差に対しては脚での移動に変更する方式である.2脚では不安定であるため,4脚に適用された例が多い.表4・1 移動機構方式(車輪型,クローラ型,脚型)の比較種類速度経済性安定性制御性方向調整環境適応性専有面積車輪◎◎◎◎△×○クローラ×○◎◎△○△2脚(足)○△×△◎◎◎4脚以上△△△○○◎○注:◎が優位であることを示す.7―9 6メカトロニクス・ロボティクス
