# 4・2・3 運動分解制御と動的制御

> **章**: 第4章 ロボティクス
> **節**: 4・2 マニピュレーションの力学と制御

ロボットアーム(マニピュレータ、manipulator)は先端部に装着されたエンドエフェクタ(end-effector)やハンド(hand)に運動(力を含む)を与えることで作業を遂行する。マニピュレータは通常、多リンク・多関節機構から成り立ち、各関節(自由度)の運動を制御することで全体の運動を実現する。作業のやり方はエンドエフェクタの運動で記述されるから、その運動を実現するためにはそれを各関節の運動に変換(分解)し制御することが必要である。ここではその方法について述べる。マニピュレータの制御の精度を考えたとき、フィードバック制御(feedback control)だけでなく、フィードフォワード制御(feed-forward control)を加味することが非常に有効である。そのためのマニピュレータの動力学を考慮した動的補償制御について述べる。a. 速度分解制御(resolved motion rate control)エンドエフェクタの速度(位置・姿勢の時間変化率)を各関節の回転速度(またはスライド速度)に変換し、制御する方法である。エンドエフェクタの速度,,,α,β,γと関節速度θ,θ,θ,θ,θ,θとの間には次のような線形関係が成り立つ。=(4・169)ここで、は変数変換(ヤコビ)行列である。これにより、エンドエフェクタ速度から関節速度を求める次の式が得られる。=(4・170)速度分解制御系の構成を図4・29に示す。ここでは、実現すべきエンドエフェクタの速度を分解して得た速度を、関節速度サーボ系の目標値として与えている。関節速度サーボ系の代わりに関節位置サーボ系を用いても構成できる。この場合は、分解された速度を積分して、関節位置サーボ系の目標値として与えればよい。この制御法はもともとテレオペレータの制御法として考案されたものであり、オペレータが意図する方向にエンドエフェクタを導くために使われた。産業用ロボットの応用に必要なエンドエフェクタの位置・姿勢や力を制御する場合は、本制御系の外側にもう一つの制御系を設定する必要がある。速度分解制御と同じことは、エンドエフェクタの位置・姿勢を関節角度(またはスライド量)に変換する方法でも実現できる。しかしそれは多次元の代数方程式を解く問題に帰着され、式(4・170)のような一般解は存在しない。本方式は複雑な自由度配置をもつマニピュレータにも適用できる点が特徴である。b.
