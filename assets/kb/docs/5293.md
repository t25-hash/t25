# 4・2・5 閉リンクシステムの制御

> **章**: 第4章 ロボティクス
> **節**: 4・39 にその概念を示す。

本項の目的は、ロボットマニピュレータと対象物とで運動学的・力学的に閉じたループ構造(closed loop structure)を形成する双腕ロボットおよび多指ハンドによる単一対象物の把持と操りのための動的協調制御則について解説することである。閉リンク機構(closed link mechanism)を有するロボットマニピュレータには、アクチュエータのベースへの集約による可動部軽量化や高剛性を図った平行リンクロボット(parallel link robot)や、4・2・1,4・2・2項で記述されているパラレル機構(parallelmechanism)を有するロボットなどがある。これらはもし全関節がアクチュエータを有する能動関節(active joint)の場合、一般にその数が閉リンク機構の運動自由度(この場合はエンドエフェクタの作業自由度)より多いため、駆動冗長性を積極的に利用する場合以外は、しばしばその構造可制御性を損なわない範囲で機構内に適切に受動関節(passive joint)を含む構造となっている。これに対し、複数ロボットマニピュレータ(アームもしくは指)を用いた対象物の協調制御において、特に複数アームによる場合、一般に全関節が能動関節である閉リンク機構が構成される。この場合、機構全体の駆動冗長性(actuational redundancy)はしばしばアーム手先もしくは指先の力や運動の任意性として表れ、その制御が協調制御という閉リンクロボット(closed link robot)の制御を特徴付ける問題の一つとなる。このような複数ロボットマニピュレータによる対象物の協調制御法は文献(44)(45)による研究にさかのぼる。2本アームによるものから始まり、マスタスレーブ(リーダ・フォロワ)方式やその区別をしない対称な方式、剛性(コンプライアンス)制御法や位置・力の動的制御法、機構の運動学的ないし動力学的冗長性を内力制御に用いる方法や与えられた評価指標の最適化に用いる方法、アームやハンドの制御を共通の枠組みで統一的に扱う方法などが開発されている。また、受動性に基づく制御法も提案されている。以下ではまず、双腕ロボットによるグリッパを用いた対象物の把持・操りと、多指ハンドによる対象物の把持・操りとについて、これら閉リンク機構の駆動冗長性によるアーム手先もしくは指先の力や運動の任意性およびその制御則の記述を容易にしそれらを統一的に扱うために、拘束行列という概念を導入し、これに基づいてそれらの運動学、動力学方程式を与える。次いで、把持・操り作業を実施するための位置軌道と力軌道を実現する動的協調制御則を与える。最後に、協調制御機構を、駆動冗長性をもつ一般的な閉リンク(a) 双腕ロボット(b)多指ハンド図4・51 複数マニピュレータシステム機構としてとらえたときの計算量の少ない動力学計算法について触れる。なお、本項では特に記さない限り、位置は(並進)位置と(回転)姿勢、力は(並進)力と(回転)モーメントをそれぞれ総称したものとする。a. 双腕ロボットと多指ハンド単一の対象物を複数のロボットマニピュレータで把持して協調的に操るのに当たって、双腕(多腕)ロボット(dual arm robot)による場合と多指ハンド(multi-fingered hand)による場合とは、作業における負荷能力や器用さを除き、本質的にはその手先と対象物間接触の運動学的、力学的関係によって区別される。図4・51(a)に示すように、双腕ロボットは通常グリッパで対象物を確実に把持し、手先と対象物間には相対運動がなく手先力がそのまま対象物に加えられるので、接触部には転がりや滑りの問題がなく、通常の閉リンク機構として扱うことができる。このようなシステムにおける一つの課題は、対象物に余計な力がかからないように対象物に加わる内力をいかに小さくするかである。他方、多指ハンド[図4・51(b)]は比較的小さな対象物を巧妙に操るのに適している。しかし通常、指と対象物の接触は各指の押付け力のみによって保たれているのに加え、転がり接触、滑り接触など多様な接触がありうる。この接触の安定性を保持しながら対象物を操らなければならない点が課題である。また、接触条件によって対象物を把持し任意の加速度で操るためには、一定数以上の指が必要になる。以上のような相違点があるが、制御則の理論的導出においては同じ枠組で扱うことができるので、以下ではこの二つの場合を複数ロボットマニピュレータによる対象物の動的協調制御(cooperativecontrol)として統一的に記述することにする。b. grasp)が維持できない。なお、本手法のように代数学的に内力を表現する以外に、その物理的意味をより重視した内力の表現手法もある。図4・54な7―114メカトロニクス・ロボティクスらびに式(4・257)は後者の例となっている。なお、対象物位置を記述する六次元ベクトルの姿勢部分の表現としては、オイラー角やロール角、ピッチ角、ヨー角がある。これはマニピュレータシステムに運動学的な冗長性がないことを意味する。よって、制御則[式(4・263)]によって系が線形非干渉化されることがわかる。対象物が環境との干渉によりその運動が拘束を受ける場合や、対象物の持替えなどのために対象物表面上で指先(finger-tip)に滑り運動(sliding motion)を行わせながら協調制御を行う場合も、同様に扱うことができる。広がりをもつ手(指)先が相対姿勢の変化に伴い対象物上を転がり、把持点位置が変化する場合については、文献(70)などを参照されたい。f. 一般的な閉リンク機構としての動力学計算一般に複数マニピュレータによる協調制御系は、アームもしくは指と対象物を一体として見れば、駆動冗長性(actuational
