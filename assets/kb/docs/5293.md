# 4・2・5 閉リンクシステムの制御

> **章**: 第4章 ロボティクス
> **節**: 4・39 にその概念を示す.

本項の目的は,ロボットマニピュレータと対象物とで運動学的・力学的に閉じたループ構造(closed loop structure)を形成する双腕ロボットおよび多指ハンドによる単一対象物の把持と操りのための動的協調制御則について解説することである.閉リンク機構(closed link mechanism)を有するロボットマニピュレータには,アクチュエータのベースへの集約による可動部軽量化や高剛性を図った平行リンクロボット(parallel link robot)や,4・2・1,4・2・2項で記述されているパラレル機構(parallelmechanism)を有するロボットなどがある.これらはもし全関節がアクチュエータを有する能動関節(active joint)の場合,一般にその数が閉リンク機構の運動自由度(この場合はエンドエフェクタの作業自由度)より多いため,駆動冗長性を積極的に利用する場合以外は,しばしばその構造可制御性を損なわない範囲で機構内に適切に受動関節(passive joint)を含む構造となっている.これに対し,複数ロボットマニピュレータ(アームもしくは指)を用いた対象物の協調制御において,特に複数アームによる場合,一般に全関節が能動関節である閉リンク機構が構成される.この場合,機構全体の駆動冗長性(actuational redundancy)はしばしばアーム手先もしくは指先の力や運動の任意性として表れ,その制御が協調制御という閉リンクロボット(closed link robot)の制御を特徴付ける問題の一つとなる.このような複数ロボットマニピュレータによる対象物の協調制御法は文献(44)(45)による研究にさかのぼる.2本アームによるものから始まり,マスタスレーブ(リーダ・フォロワ)方式やその区別をしない対称な方式,剛性(コンプライアンス)制御法や位置・力の動的制御法,機構の運動学的ないし動力学的冗長性を内力制御に用いる方法や与えられた評価指標の最適化に用いる方法,アームやハンドの制御を共通の枠組みで統一的に扱う方法などが開発されている.また,受動性に基づく制御法も提案されている.以下ではまず,双腕ロボットによるグリッパを用いた対象物の把持・操りと,多指ハンドによる対象物の把持・操りとについて,これら閉リンク機構の駆動冗長性によるアーム手先もしくは指先の力や運動の任意性およびその制御則の記述を容易にしそれらを統一的に扱うために,拘束行列という概念を導入し,これに基づいてそれらの運動学,動力学方程式を与える.次いで,把持・操り作業を実施するための位置軌道と力軌道を実現する動的協調制御則を与える.最後に,協調制御機構を,駆動冗長性をもつ一般的な閉リンク(a) ステップ1本来の協調制御系[図4・55(a)]に等価な閉リンク機構[図4・55(b)]において,冗長な能動関節욀욥を適当に選んで受動関節とみなし,非冗長駆動系にして考える[図4・55(c)].その際の冗長能動関節は,非冗長駆動系にしたとき受動関節の速度が一意に定まるように選ぶ.残りの能動関節は非冗長駆動系の能動関節욀욄となる.(2) ステップ2(a) 適当な受動関節で仮想的に切断し仮想木構造[図4・55(d)]を作る.残りの受動関節に仮想的なアクチュエータ욧욣を仮定する.(b) 閉リンク構造に要求される運動と全く同じ運動をするものとして,開ループリンク機構に対する手法を用いて仮想木構造の関節トルクを求める.(a) 閉リンク双腕機構(b) 冗長駆動系(c) 非冗長駆動系(d) 仮想木構造機構◎:能動関節윥:受動関節図4・55 閉リンク機構と仮想木構造機構(c) 仮想木構造の関節トルクから拘束条件を考慮することにより,もとの非冗長駆動系の関節トルクを求める.(3) ステップ3非冗長駆動系の関節トルクから各関節間の拘束関係を考慮することにより,もとの閉リンク機構の関節トルクを求める.なおこの方法は,駆動の冗長性を閉リンク機構の最適負荷分配や運動の最速化などには利用できるが,このままでは式(4・256)の右辺第2項に相当する手先力の内力成分を明示的に与えることはできないという点に注意されたい.
