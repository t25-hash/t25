# 6・2・5 その他のバイオセンサ

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・2 診断

a.表面プラズモン共鳴（SPR:surface plasmon resonance)
表面プラズモン共鳴は金薄膜で反射される光が金表面に微量に存在
する物質により微妙に変化する現象を用いる．表面プラズモン共鳴
は表面の状態に敏感であり，表面にプローブを固定してターゲット
がハイブリダイズしたか否かで散乱光の角度が微妙に変化すること
をとらえる検出方法である．簡便で感度の高い方法としてDNA，
タンパク質などの検出を実時間で行える利点がある．図6･19に原
理図を示した．プローブとターゲットの相互作用を実時間で調べる
ことができる利点がある．
b.モレキュラビーコン（molecular beacon) DNAプローブ
に蛍光標識とクエンチャ（消光剤）とを結合させ，ターゲットにハ
イブリダイズしたときだけ蛍光を発するように工夫したプローブが
種々開発されている．プローブの構造の一例を図6･20に示した．
通常の状態では自分自身でハイブリダイズしてループ構造をとる．
ターゲットにハイブリダイズしない状態では蛍光標識とクエンチャ
は近い位置にあり，光を吸収して励起状態になった蛍光体はクエン
チャにエネルギー移動して失活して光を出さない．しかし，ターゲ
ットにハイブリダイズすると両者の距離が離れ，エネルギー移動は
起こらないようになるので蛍光を観測することができる．ターゲッ
トに付くか付かないかによって光を出したり出さなかったりするの
でモレキュラビーコンと呼ばれ，いろいろな構造のものが報告され
ている．例えばPCR増幅に伴って蛍光が観測されるようになるも
のもある．
c .電気伝導度センサ 通常のDNAチップでは蛍光標識ター
ゲットをハイブリダイズさせて蛍光検出するが，DNAがハイブリ
ダイズすることで二つの電極間にDNA架橋を作り電気伝導度の変
化を観測するタイプのDNAチップが開発されている．簡便で小型
の遺伝子検査ツール（ハンドヘルドDNAセンサ）が実現でき，救
急医療などに威力を発揮すると考えられる ．
