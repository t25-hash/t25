# 4・3・1 車輪型移動機構

> **章**: 第4章 ロボティクス
> **節**: 4・3 移動機械の力学と制御

ロボットの移動機構には車輪や脚などさまざまなタイプのメカニズムが考案されているが,一般的に移動ロボットというと車輪で走行するものを指す場合が多い.車輪型の移動ロボットであっても,車輪の配置の仕方・操舵の方法によりさまざまなタイプの移動機構が開発されている.図4・67に,さまざまな車輪配置・操舵方法の機構の例を示す.それぞれの車輪の構成により,車輪の制御方法は異なる.平面を移動するロボットが安定して地面に接地するためには最低三つの車輪が必要であるが,必ずしもすべての車輪が動力をもつ必要はなく,動力をもたない従輪によって自重を支えることも多い.近年では左右二つの動輪しかもっていない倒立振り子のような移動台車が製品化されている例も見られる.また,図4・68に示す特殊な車輪7―118メカトロニクス・ロボティクス(a)(b)(c)(d)(e)(f)(h)(i)(g)図4・67移動ロボットの車輪配置と舵取り機構(a) 車輪の配置例(b) オムニホイール(c) メカナムホイール図4・68 全方向に移動可能な車輪機構と配置機構を利用することで,本体の姿勢にかかわらずいつでもどの方向へも直接移動することができる移動台車も開発されている.これらの移動機構を用いることで,通常の自動車のような車輪配置ではできない真横への移動を切返しすることなく行える.ここでは,自律移動ロボットでよく利用される独立二輪操舵型[図4・67(g)(h)]と,自動車のようなステアリング操舵型[図4・67(a)(b)(e)(f)]という二つの代表的な機構について,メカニズムと基本的な制御方法を紹介する.a. 独立二輪操舵型独立二輪操舵型の構造は,車体中心に左右の動輪を配置し,それぞれの動輪の回転数を独立に制御できるようにしたものであり,移動ロボットプラットホームとして最も一般的なタイプである.この構造がよく用いられる理由として挙げ図4・69ステアリング操舵のモデルられるのは,(1)ロボットの動力は電動モータが利用されることが多く車輪回転数が直接制御しやすいこと,(2)ステアリングのメカニズムやモータを用意する必要がなく機構がシンプルであること,(3)並進用モータが同時に旋回用モータの役割も担うためエネルギー効率が高いこと,といった点である.左右の動輪の回転角速度をおのおのω욥,ω욆,左右動輪の間隔をとし,車輪の半径をとすると,本体中心の目標並進速度と目標角速度ωは以下の式を満たす.ω욥ω욆(4・274)ω =-2∴ω욥ω(4・275)ω욆=2これは,移動台車中心部の目標並進速度と目標回転角速度ωが与えられたとき,左右の動輪をどれくらいの速さで回転させればよいかを算出する式となる.b. ステアリング操舵型ステアリング操舵型は,自動車や三輪車のように,ハンドル操作により車輪の向きを変えることで進行方向の制御を行うものである.ここでは,簡単のため図4・69に示すような三輪車のモデルを考える.舵をとる前輪と推進力を生む二つの後輪から構成され,前輪と後輪のホイールベースを,前輪のステアリング角をσとする.このとき,すべての車軸の延長線は旋回中心で交差する.この旋回中心と左右動輪の中央との距離を回転半径とすると,=tanσと表される.一般に,半径の円周上を回転するときの運動は=ωの関係があるため,ω=tanσとなる.なお,σが小さいときはtanσ≒σと近似できるので,ω=σ∴σ=ωと書ける.これは,移動台車中心部の目標並進速度と目標回転角速度ωが与えられたとき,どれくらいステアリングを切ればよいかを算出する式となる.
