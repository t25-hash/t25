# 4・3・3 その他の移動機構

> **章**: 第4章 ロボティクス
> **節**: 4・3 移動機械の力学と制御

移動の形態は、車輪のような接地点が連続変化するもの、歩行のように固定接地点が入れ替わるもの、スケートのように接地部が滑るものがある。ここでは、車輪の派生としてクローラ型、スケートの原理によるヘビ型、歩行の特例であるミミズ・ナメクジ型の移動形態について述べる。a.
