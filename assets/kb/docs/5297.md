# 6・3・3 バイオ人工臓器

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・3 治療

a.バイオ人工肝臓 人工腎臓や人工心臓は工学的な手法を用
いて構築されており，物理的な機能を代行することで多くの患者の
救命に役立ってきた．しかし，より高度な代謝機能をこのような工
学的手法のみで達成することは非常に難しく，肝臓，膵臓などその
機能を細胞機能に帰着させることのできる臓器では，細胞の培養系
で臓器の代謝機能を代行させるバイオ人工臓器の開発が進められて
いる．
バイオ人工肝臓は，工学的装置の中に生きた肝細胞を組込んだも
ので，肝不全患者の血液を体外で浄化するなど一時的な肝臓として
代用できる．劇症肝炎に対して，患者の肝臓が回復するまでの間，
人工肝臓で生命維持することができれば，移植に頼らないで治療効
果を上げることができる．
このような人工肝臓は，細胞ソース，機能保持時間，モジュール
デザイン，培養足場材，培養液，培養条件などで特徴付けることが
でき，いくつかの要素で分類される．国内開発例では，ブタ肝細胞
にひも状集合体構造をもたせることで機能保持の延長を図り，中空
糸モジュールに組込んだ船津らのモデル，ブタ肝細胞に肝細胞スフ
ェロイド構造をとらせることで機能保持の延長を図り，箱型モジュ
ールに組込んだ藤堂，松下らのモデルなどがある．アメリカでは
FDA（Food and Drug Administration，食品医薬品局）認可を得
た製品例がある（表6･6) ．
細胞にはヒト肝細胞を用いることができれば最良である．しかし
ヒト肝細胞は体内では旺盛な再生能力を見せるが，in vitroで培養
増殖することにはいまだ成功しておらず，ヒト肝細胞をモジュール
に充てんするだけ集めるのは容易ではない．したがって現状では異
種動物細胞であるブタ肝細胞をモジュール内に充てんして使用する
ことが多いが，将来は幹細胞，ES細胞より出発するヒト肝細胞に
置換わることが予想される．
バイオ人工肝臓の究極の姿には，幹細胞から出発して試験管の中
で作製されたクローン肝臓が予想されるが，その完成にはさらに多
くの時間と労力を必要とすると考えられる．この一段階手前とし
て，ブタの肝臓を使用することが考えられている．ブタは肝臓など
主要な臓器の大きさや形が人間のものに近く，成長も早いので移植
には都合がよい．遺伝子技術でうまくヒト遺伝子を組み入れた臓器
を作り，異種間拒絶反応を回避することが課題となっている．補体
制御遺伝子の導入あるいは主要組織適合抗原系遺伝子を破壊したい
わゆるノックアウトブタの作製が考えられる．この場合，ブタの細
胞に組込まれたレトロウイルスをはじめとする異種動物経由の感染
症に対する懸念も完全に解消しておかなければならない課題であ
る．
b.バイオ人工膵臓 糖尿病の根本的な治療法をねらった膵島
移植が検討されており，膵島細胞を免疫抑制剤を併用しながら肝臓
に移植する治療の良好な結果が報告されている．しかしドナーが不
足していることや，免疫抑制剤の長期使用による問題は大きい．こ
のため，幹細胞やES細胞から膵島細胞を誘導することや，拒絶反
応の防止のための遺伝子操作が研究されている．また，膵島細胞を
工学的に選択的透過膜で包み，ヒトの免疫系から隔離した状態で人
工膵臓として機能させる試みが行われている．各種のバイオ人工膵
臓の特徴を表にまとめる（表6･7) ．
