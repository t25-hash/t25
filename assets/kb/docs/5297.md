# 4・3・3 その他の移動機構

> **章**: 第4章 ロボティクス
> **節**: 4・3 移動機械の力学と制御

移動の形態は、車輪のような接地点が連続変化するもの、歩行のように固定接地点が入れ替わるもの、スケートのように接地部が滑るものがある。ここでは、車輪の派生としてクローラ型、スケートの原理によるヘビ型、歩行の特例であるミミズ・ナメクジ型の移動形態について述べる。a. クローラ(crawler)クローラ型走行車両(ここでは単にクローラという)は通常のタイヤ型に比較して、軟弱地盤走行、段差、階段等の障害のある地形での走行に有利である。また、その場旋回や小半径の旋回能力が優れている。クローラがスリップせずに発生できる推進力の車両重量に対する比は、一般にタイヤ型よりも大きい。ただし、は履帯の接地面積、は粘着係数で平均15kPa程度、φは土の摩擦角で通常30°程度である。つまり、接地圧/が小さいほど実効摩擦係数が大きい。なお、接地圧は人間の歩行(片足に体重+α)で40kPa程度、軟弱地盤の車両走行には70kPa以下である必要があるといわれている。また、文献(108)では、グローサ(つめ)のあるクローラの、より精密な推進力が導出されている。クローラが旋回を行うには、左右履帯に速度差を与えるが、前後に長い形状のクローラは旋回しにくい。図4・76のように、履帯長さ、左右間隔のクローラで、履帯と地面間の実効的摩擦係数を前後方向がμ、左右方向がμとする。車両重量を、接地圧分布は均等であるとする。ここでは履帯の幅を無視して、履帯各部(セグメントあるいは一つのグローサ部分)が横滑りすることだけを考えている。幅広の履帯をもつ場合には、さらに履帯各部を地面に対してヨー軸回転させるモーメントが必要であるから、より条件は厳しくなるが、実際の多くのクローラでは=1.0~1.7になっている。この条件を満たさないクローラでは、左右に小さな速度差を与えても左右それぞれの履帯が前後に滑り、車両の進行速度が左右の平均的なものとなるだけで、旋回が行われない。ただし、大きな速度差を与えれば横滑りが生じて旋回できるのが普通である。前後に長いクローラで良好な旋回を行うためには、図4・77(a)のような跳ね上げのできる4クローラ型、図4・77(b)のように転輪が突き出して中央部の接地圧を高くするもの、連接車体でクローラが左右にステアリングできるものなどの工夫が行われている。クローラが段差を乗り越える性能は、摩擦やグローサによるグリップ力のほかに、クローラ形状と重心位置に依存する。一般に図4・78(a)のように、前縁車輪のセンタ高が越えられる段差の限界であることが多い。この高さを大きく得るためには、図4・78(a)のように車両前後の高い位置に車輪をもつ逆台形形状のクローラが有効であることは明白である。しかしそれとは別に、図4・78(b)のように、乗越え中に重心が段差部分を越えないと傾くだけで前に進めない。傾斜が増して図4・78(c)のように重心が接地部後端より後になると転倒する。このため、クローラ前後長に対して重心位置を低く設計することが有効である。文献(114)では、この条件を満たさないクローラを前後につなげて互いに補助する方法が提案されている。また、転輪の位置を制御して履帯の下部をたわませ、車体傾斜を小さく抑えることも有効である。b.
