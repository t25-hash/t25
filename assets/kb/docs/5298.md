# 6・3・4 胎児性幹細胞（ES細胞)

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・3 治療

細胞を培養増殖して組織を再建する再生医療の確立には，細胞ソる場合には，免疫問題に加えヒト組織の採取に関わる倫理問題を避が可能な場合にはこの問題は生じない．
このために，再生医療の技は，体のあらゆる組織や臓器などの細胞に分化する多分化能をもつ能であり，ES細胞を再生医療の細胞ソースに用いることに期待がン大学が初めて作製に成功した．
