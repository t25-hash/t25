# 4・3・5 ナビゲーション技術

> **章**: 第4章 ロボティクス
> **節**: 4・3 移動機械の力学と制御

a. 自己位置推定移動中のロボットが自分自身の位置をつねに知っていることは非常に重要である.位置を特定するために空間中に座標系を定義し,ロボットや周囲の物体の位置と姿勢を数値として記述する.環境の形状や物体配置などの情報が同じ座標系で定義されていれば,現在の位置姿勢からロボットが地図上のどこにいるかも容易に知ることができる.移動ロボットの自己位置推定方法は,デッドレコニングによる位置推定,外界センサ情報を用いた位置推定に大別できる.いずれの推定手法にもぞれぞれの長所と短所が存在するため,通常は両方の推定方法を融合することで信頼度の高い位置を推定する.. デッドレコニングデッドレコニング(dead reckon-ing)とは,移動体の位置と姿勢の微小変化をエンコーダ等の内界センサを用いて遂次計測し,それを積分することによって初期状態からの変化量を推定する手法であり,オドメトリとも呼ばれる.車輪移動型ロボットの場合は,車輪の回転数を観測することでボデーと地面の相対速度を得る手法が多くとられる.また,浮遊移動物体の場合においては,加速度センサにより移動速度と方向を計測する.この方法は,センサに計測誤差や量子化誤差が存在すると徐々に誤差が累積してしまうため,長距離の移動をする場合には定期的に誤差を検出して補正しなければならない.
