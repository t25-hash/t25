# 4・4・1 ヒューマノイドのハードウェア

> **章**: 第4章 ロボティクス
> **節**: 4・4 ヒューマノイド技術

a. 代表的なヒューマノイドロボット工学史上特に重要なヒューマノイドを図4・97に示す.図4・97(a)は,1973年に早稲田大学の加藤一郎研究室によって開発された世界初のヒューマノイドWABOT-1である.WABOT-1は,視覚により対象物を認識し,聴覚と人工の口により人間とコミュニケーションし,触覚を有する両手で物体を操作し,静歩行ではあるが二足歩行(biped locomotion)することもできるなど,ヒューマノイドの構成要素をひととおり備えていたという点で画期的であった.次にヒューマノイドの研究開発の一大転機となったのは,本田技研(株)によるP2の発表であった[図4・97(b)].本田技研(株)は,1986年から秘密裏にヒューマノイドの研究開発を実施し,1996年12月にその成果である身長180cm体重210kg のヒューマノイドP2を発表した.P2は体内にコンピュータとバッテリを搭載し,高い信頼性で二足歩行を行える自立型ロボットとして世界初のものであり,国内外のロボット研究者に大きな衝撃を与えた.本田技研(株)はその後,1997年に身長160cm体重130kg のP3,2000年に身長120cm 体重43kg のASIMO[アシモ,図4・97(c)]を発表し,小型軽量化を進めている.また,ソニー(株)は2000年11月に身長50cm体重5kg の小型ヒューマノイドSDR-3X,2002年3月には身長58cm体重6.5kg のSDR-4X を発表,その発展型であるQRIO(キュリオ,SDR-4XII,身長58cm体重7kg)は2003年3月に発表された[図4・97(f)].しかし,2006年1月に事業再編の一環としてQRIOの新規開発の中止がアナウンスされている.7―126メカトロニクス・ロボティクス(a) WABOT-1(1973)(c) ASIMO(2000)(b) P2(1996)(f) QRIO(2003)(e) H
