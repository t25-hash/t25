# 7・3・3 ファン（fan）選定

> **章**: 第7章 ヒートシンク
> **節**: 7・3 電子機器設計とその実際

a.ファンの種類と特徴 電子機器冷却に通常使用するファン
は，ファンの外径が数cmから十数cmまでのものである．また，
タイプとしては軸流ファンが最も一般的であり，その他に遠心ファ
ンや横流ファンなどが使われることがある．図7･17にこれらのフ
ァンの風量（volumetric flow rate）・静圧（static pressure）特性
117
を示す．7･3･2項で述べたように，風路を流れる冷却空気の流量
は，このファンの風量・静圧特性と流路の圧損カーブとの交点であ
り，流路の抵抗が大きいときには，遠心タイプのファンが有利であ
り，逆に流路の抵抗が小さいときには，軸流タイプのファンが有利
である．また，横流ファンは両者の中間に位置する．
b.ファンの直列，並列運転特性 電子機器筐体内の冷却を行
う際に，複数のファンを用いて行うことがある．同一性能のファン
を2台直列，または並列に使用した場合の風量・静圧特性は図7･
18になる．並列に構成した場合の最大風量は，単体のほぼ2倍と
なる．ただし，実装時の風量は流路抵抗との兼合いにより決まるの
で，2倍とはならない．直列に構成した場合の最大静圧は，もし二
つのファンの間の流路において空気流の漏れや回り込みがなけれ
ば，単体のほぼ2倍となると考えてよい．ただし，ファン2台を直
接つなげた場合には，上流側ファンの出口で回収損失が発生するた
め，2倍とはならず約1.5倍に留まる．
c.ファン選定手順 ファンの選定は，おおむね以下に示すよ
うな手順で行う．
(１)冷却しようとする筐体での冷却必要風量と流路の圧損特性を
求める（7･3･2項参照)．
(２)流路の圧損が比較的小さい場合→軸流ファンを選ぶ．
(３)流路の圧損が比較的大きく，かつ冷却する範囲が広い場合→
横流ファンを選ぶ．
(４)流路の圧損が比較的大きく，かつ冷却する範囲が狭い場合→
遠心ファンを選ぶ．
このほかの要因としては，ファン騒音（accoustic noise）を考慮
しなければならない場合が多い．ファン騒音は，一般的には軸流フ
ァンが最も小さく，遠心ファンが最も大きい．また，風量・騒音特
性は図7･19のようになる．軸流，横流ファンでは，風量に対して
騒音が極小となる点（サージ点付近）が存在し，遠心ファンでは，
風量とともに騒音は増大する．また，ファンが複数ある場合には単
体に比べてファン騒音は増大する（例えば，同一性能のファンでは
2台で＋3.0dB，3台で＋4.8dB，4台で＋6.0dBである)．
d.ファン使用上の留意点
(１) 物体をファンの上流側に置く場合，羽根径の2分の1以上
離す（騒音の発生の防止)．
(２) ファンの吐出側から吸入側への漏れを極力少なくする（流
量低下，風温上昇，回転音発生の防止)．
(３)ファンのカタログにある騒音値は，通常騒音源から1m
離れた点での測定値であり，例えば騒音源から300mmの所では，
カタログ値よりも＋10.5dB大きな値となる（この点は，デスクト
ップやノートサイズの筐体の冷却設計において特に注意を要する)．
