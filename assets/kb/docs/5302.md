# 4・4・2 ヒューマノイドで実現されている動作・作業

> **章**: 第4章 ロボティクス
> **節**: 4・4 ヒューマノイド技術

現状のヒューマノイドでどのようなことが実現されているか、HRP-2を例にとって見てみる。まず、移動機能の例を図4・99に示す。図4・99(a)は平地上の二足歩行(1.35km/h,1歩0.3m,0.8s)のようすであり、内蔵バッテリによって1時間以上の二足歩行を行える。図4・99(b)は不整地歩行のようすであり、1歩間の高低差±4cm、最大傾斜5%までの不整地に対応することができる。図4・99(c)は片手で手すりをつかんでバランスを確保しつつ一段28cmの段差を上っているようすである。さらに図4・99(d)は、二足では移動できない狭隘空間を腕、腰、脚の協調による四つんばい歩行によって1km/hで移動しているようすである。図4・100は実現されている作業の例である。図4・100(a)は人間の作業者と長尺の板を協調搬送しているようすである。手首の力センサの情報をもとにして前進、後退、方向転換を行うことで、作業者の意図に沿って歩行する。図4・100(b)では床面上に置かれた物体を押して移動させている。ロボットは約5kgfの力を発生させ、摩擦や対象物の重量に応じて歩行速度を調整する。図4・100(c)は、奥まった位置にあるバルブを操作するため、片手でバランスをとりつつ身を乗り出しているようすである。図4・100(d)では、頭部のカメラにより物体を認識し自律的にテーブル上の物体を持ち上げている。図4・101は転倒実験(falling experiment)のようすである。ヒューマノイドは直立した状態が標準なので、何らかの原因で転倒事(a) 平地歩行(b) 不整地歩行(c) 大段差乗越え(d) 四つんばい歩行図4・99ヒューマノイドの移動機能(b) 押し作業(a) 長尺物の協調搬送(d) 自律的物体把持(c) バルブ操作図4・100 ヒューマノイドの作業機能故が発生することは避けがたい。このような場合でも転倒による障害を最小限にとどめ、再び起き上がって作業を続行できることが望ましい。図4・101はHRP-2のプロトタイプ機を用い、転倒開始を検出後、速やかに腰を落として臀部と背中に装着されたクッション材で衝撃を吸収し、いったんあおむけ状態で静止、さらにその後再び立ち上がるまで実験の模様を示している。
