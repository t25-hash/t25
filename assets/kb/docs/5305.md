# 4・4・5 シミュレータ

> **章**: 第4章 ロボティクス
> **節**: 4・4 ヒューマノイド技術

ヒューマノイド開発においてシミュレータ(simulator)は以下の目的に用いられる.a. 機構設計関節軸構成,アクチュエータ,減速比等を決定するため各種動作をシミュレートし,消費電流,アクチュエータにおける発熱,関節応力等を評価する.b. 動作パターン設計歩行や各種作業中のZMPの予測,関節最大速度等を予測し,動作の実現可能性や転倒の危険性の予測を行い,動作パターンのチェックや改善に利用する.c. 安定化制御系設計安定化制御系の設計や動作チェックに用いる.床面の想定外の凹凸やスリップ,各種外乱のもとでのロボットや安定化制御系の動作を評価し,アルゴリズムの評価や改善に利用する.特に等身大に近い大型のヒューマノイドではその開発に多大の資金,労力,時間を要するため,シミュレーションによる検討を設計時に十分行っておくことが重要である.多くの商用シミュレータ(例えばMSC. visualNastran 4D)はこのような要求に応えられるほか,HRPで開発されたOpenHRPなどもある.
