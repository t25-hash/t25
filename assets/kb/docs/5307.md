# 4・5・1 テレロボティクスとは

> **章**: 第4章 ロボティクス
> **節**: 4・5 テレロボティクス

テレロボティクス(telerobotics)とは,ロボティクスに遠距離の意を表すtele(テレ)をつけた語であり,従来のテレオペレーション(teleoperation)の技術とロボティクスの技術とを融合した概念として1980年代に提唱されたものである.テレロボティクスのもととなったテレオペレーション技術,すなわち遠隔操縦の技術の歴史はさらに古く,1940年代にまでさかのぼることができる.テレオペレーションでは,原子炉内や深海,宇宙空間などの極限環境での作業を,人間が遠隔からマニピュレータ(manipulator)などを操縦することで実現するものである.このような環境で要求されている作業は複雑で不定形なものが多く,コンピュータによって制御される完全自律ロボットで行うのはまだまだ難しい.テレオペレーションでは,これらの高度な認識・判断・作業指令を操縦者である人間自らが行う.プログラムにより決められた動作を自動的に再現したり,センサ信号をもとに自律的に動作したりする自動機械をロボットと呼ぶとするならば,遠隔操縦されるマニピュレータなどは厳密にはロボットと呼ぶべきではないかもしれないが,遠隔操縦される機械を操縦型ロボット(manually controlled robot)と呼ぶことが多い.テレオペレーション技術の進化の方向は大きく二つに分けられる.その一つは,ロボットと操縦者との感覚系・運動系の結合をさらに密にしていく方向である.これはテレイグジスタンス(telex-istence)やテレプレゼンス(telepresence)と呼ばれ,究極的にはあたかも操縦者自身がロボットに成り代わって作業を行えるようになる,すなわちロボットが人間の分身となる方向である.このテレプレゼンス,テレイグジスタンスについては4・5・6項で詳しく述べる.もう一つは,操縦されるロボットを知能化させる方向である.ロボット側にある程度の知能をもたせれば単純な動作はロボットが自律的に行えるので,操縦する人間は動作を事細かに指令する必要はなく,より高度なレベルでの指示を与えるだけでよい.究極的にはロボットが下僕や有能な秘書のようになることを目指している.このような考え方は管理制御(スーパーバイザリコントロール,supervisory control)としてフェレル・シェリダン(Ferrell-Sheridan)によって早くから提案されており,1980年代に提唱されたテレロボティクスの概念へとつながっている.つまりテレロボティクスは,狭い意味では操縦されるロボットを知能化させるアプローチを意味するが,もう一方の進化の方向であるテレプレゼンス,テレイグジスタンスの実現にもロボティクスの技術が用いられることは事実であり,いまではテレロボティクスという語が広く操縦型ロボットに関する技術を指すと考えても差し支えない.1940年代に放射性物質を扱う要求から発達したテレオペレーションの技術は,現在でもさまざまな場面で用いられており,以下にその一例を挙げる.図4・110は,原子力関連施設内において放射性物質を扱う作業用空間であるホットセルで用いられている機械式のマニピュレータである.このマニピュレータは,基本的には1950年代に用いられたものと変わりはない.アメリカのインテュイティブ・サージカル社が1997年に開発したダビンチ(図4・111)は,内視鏡手術を行う操縦型ロボットであり,2002年の時点で日本国内で数台,海外では欧米を中心に約100台が普及している.これまでに心臓の冠状動脈のバイパス手術などの高度な手術を含めて世界で7000件以上の手術実績をもっている.最近では,前立腺の内視鏡手術に有効であることが認められ,普及のけん引力となった.これら以外でもさまざまな分野でテレオペレーションの技術が用いられている.
