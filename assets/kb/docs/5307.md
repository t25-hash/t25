# 4・5・1 テレロボティクスとは

> **章**: 第4章 ロボティクス
> **節**: 4・5 テレロボティクス

テレロボティクス（telerobotics）とは，ロボティクスに遠距離
の意を表すtele（テレ）をつけた語であり，従来のテレオペレーシ
ョン（teleoperation）の技術とロボティクスの技術とを融合した
概念として1980年代に提唱されたものである．テレロボティク
スのもととなったテレオペレーション技術，すなわち遠隔操縦の技
術の歴史はさらに古く，1940年代にまでさかのぼることができる．
テレオペレーションでは，原子炉内や深海，宇宙空間などの極限
環境での作業を，人間が遠隔からマニピュレータ（manipulator）
などを操縦することで実現するものである．このような環境で要求
されている作業は複雑で不定形なものが多く，コンピュータによっ
て制御される完全自律ロボットで行うのはまだまだ難しい．テレオ
ペレーションでは，これらの高度な認識・判断・作業指令を操縦者
である人間自らが行う．プログラムにより決められた動作を自動的
に再現したり，センサ信号をもとに自律的に動作したりする自動機
械をロボットと呼ぶとするならば，遠隔操縦されるマニピュレータ
などは厳密にはロボットと呼ぶべきではないかもしれないが，遠隔
操縦される機械を操縦型ロボット（manually controlled robot）
と呼ぶことが多い．
テレオペレーション技術の進化の方向は大きく二つに分けられ
る．その一つは，ロボットと操縦者との感覚系・運動系の結合をさ
らに密にしていく方向である．これはテレイグジスタンス（telex-
istence）やテレプレゼンス（telepresence）と呼ばれ，究極的には
あたかも操縦者自身がロボットに成り代わって作業を行えるように
なる，すなわちロボットが人間の分身となる方向である．このテレ
プレゼンス，テレイグジスタンスについては4･5･6項で詳しく述べ
る．
もう一つは，操縦されるロボットを知能化させる方向である．ロ
ボット側にある程度の知能をもたせれば単純な動作はロボットが自
律的に行えるので，操縦する人間は動作を事細かに指令する必要は
なく，より高度なレベルでの指示を与えるだけでよい．究極的には
ロボットが下僕や有能な秘書のようになることを目指している．こ
のような考え方は管理制御（スーパーバイザリコントロール，
supervisory control）としてフェレル・シェリダン（Ferrell -
Sheridan）によって早くから提案されており，1980年代に提唱
されたテレロボティクスの概念へとつながっている．つまりテレロ
ボティクスは，狭い意味では操縦されるロボットを知能化させるア
プローチを意味するが，もう一方の進化の方向であるテレプレゼン
ス，テレイグジスタンスの実現にもロボティクスの技術が用いられ
ることは事実であり，いまではテレロボティクスという語が広く操
縦型ロボットに関する技術を指すと考えても差し支えない．
1940年代に放射性物質を扱う要求から発達したテレオペレーシ
ョンの技術は，現在でもさまざまな場面で用いられており，以下に
その一例を挙げる．図4･110は，原子力関連施設内において放射性
物質を扱う作業用空間であるホットセルで用いられている機械式の
マニピュレータである．このマニピュレータは，基本的には1950
年代に用いられたものと変わりはない．アメリカのインテュイティ
ブ・サージカル社が1997年に開発したダビンチ（図4･111) は，
内視鏡手術を行う操縦型ロボットであり，2002年の時点で日本国
内で数台，海外では欧米を中心に約100台が普及している．これま
でに心臓の冠状動脈のバイパス手術などの高度な手術を含めて世界
で7000件以上の手術実績をもっている．最近では，前立腺の内視
鏡手術に有効であることが認められ，普及のけん引力となった．こ
れら以外でもさまざまな分野でテレオペレーションの技術が用いら
れている．
4・5・2マスタスレーブマニピュレータ（master -slave
manipulator）
ロボットを遠隔操縦するにはさまざまな方法が考えられるが，操
縦方法としては図4･112に示すように，(１)各軸スイッチ方式，
(２)ジョイスティック（joystick）方式，(３)マスタスレーブ方式
の三つに分けることができる．
各軸スイッチ方式は各関節の増分を指令する方式なので，アーム
の手先に望みの動きをさせるように操作するのはかなり難しい．ジ
ョイスティック方式では，アームの手先位置や姿勢の増分を直接指
令することができるので，より直感的な指令が可能であるが，依然
