# 4・5・3 ユニラテラル制御とバイラテラル制御

> **章**: 第4章 ロボティクス
> **節**: 4・5 テレロボティクス

a. ユニラテラル制御(unilateral control)サーボマニピュレータは,機械式マニピュレータと違って何らかの制御系を組込まなければ動作しない.マスタスレーブマニピュレータの制御方式として最も簡単なものは,図4・115に示すようにマスタアームの関節の角度をセンサで読み取って,それをスレーブアーム側の位置サーボ系の目標値とする方式である.この方式は,サーボ系(servosystem)がスレーブアーム側にのみ存在し,目標値はマスタからスレーブへ送られるだけなので,次に述べるバイラテラル制御と対比させてユニラテラル制御と呼ばれている.ユニラテラル制御では,操作者の動作指令をスレーブ側に送るだけなので,機械式マニピュレータでは可能であったスレーブ側の作業状況を操作者が力感覚として感じることができない.このためユニラテラル制御では,マスタアームに位置センサのみを取付ければよいのでアームの構造が簡単軽量になるものの,環境との接触を伴う作業は反力が感じ取れないため困難である.ただし,環境との接触作業を主としない単純な作業であればユニラテラル制御でも十分可能である.b. バイラテラル制御(bilateral control)機械式マニピュレータと同様にスレーブ側の作業状況を操作者が力感覚として感じることができるようにするためには,まずマスタ側とスレーブ側の双図4・114 電動式サーボマニピュレータ,モデルE3(a) 対称型バイラテラル制御(b) 力逆送型バイラテラル制御(c) 力帰還型バイラテラル制御図4・116 代表的なバイラテラル制御法方にそれぞれのアームを制御する制御系を配置し,次に各アームの制御系への目標値が相手側のアームからのセンサ情報をもとに決定されるようにしなければならない.a項で述べたユニラテラル制御と違い,マスタアームの制御系とスレーブアームの制御系が双方に目標値を出し合いながら結合していることから,この制御方式をバイラテラル制御と呼ぶ.図4・116にバイラテラル制御の代表的な方式を示す.バイラテラル制御のもつ特徴は,ユニラテラル制御のように単に操作者の動作指令をスレーブ側に送るだけでなく,スレーブ側の作業状況が力感覚として操作者に伝えられる点である.例えば,ユニラテラル制御では,たとえスレーブアームが環境中の何かと接触しても,操作者には力感覚を通してはその状況が伝わらないのに対して,バイラテラル制御ではそれが可能となり,クランク回しなど環境からの拘束を受ける作業では作業能率が飛躍的に向上する.逆にバイラテラル制御では,マスタアームにもアクチュエータと制御系を配置する必要があるので,装置が大掛かりになるのが欠点である.以下では,図4・116に示したそれぞれのバイラテラル制御について詳しく説明する.. 対称型(symmetric type bilateral control)図4・116(a)に示したように,対称型では両方に位置制御系を配置し,相手側のアームの位置が目標値となるような閉ループ系を構成している.同じ位置制御系がマスタ側とスレーブ側で対称に配置されていることから,こう呼ばれている.対称型は,以下に述べる力逆送型や力帰還型では必要な力センサを必要とせず,また安定性の観点からも性質のよい制御法である.ただし制御系の構成上,オペレータが感じる見掛けの慣性は,マスタアームの慣性にスレーブアームの慣性が重畳されるため操作感が重くなりがちであり,スレーブからマスタへの力フィードバックを位置制御系の偏差信号に頼るのでアームの駆動系の摩擦の影響を受けやすいのが欠点である.. 力逆送型(force reflecting type bilateral control)力逆送型は,マスタアームの位置を目標値としてスレーブアームが位置制御されるのと同時に,スレーブアーム側で検出された環境との接触力がマスタ側に逆送されてマスタアームを駆動する方式である.そのためマスタ側には力制御系が配置され,スレーブ側には環境との接触力を検出する力センサが必要となる.同様の方式に力帰還型があるが,力逆送型ではマスタ側の力制御系が開ループとなっているのが大きな違いである.このため力逆送型ではマスタ側に力センサを必要としないが,マスタアームの駆動系での摩擦が大きいとスレーブ側の力が正しく操縦者に伝わらないという欠点がある.. 力帰還型(force feedback type bilateral control)力帰還型は,マスタアームの位置を目標値として,スレーブアームが位置制御されるのと同時に,スレーブアーム側で検出された環境との接触力を目標値としてマスタアームが力制御される方式である.力逆送型との違いは,マスタアームにも力センサが備えられ,力目標との誤差を帰還する閉ループが存在することである.この力帰還ループの追加により,力感度,逆動可能性(backdrivability)などが向上し,アームの見掛けの慣性が小さくなって操作感が向上する.以上三つの代表的なバイラテラル制御法は1950年代に提案されたが,その後も基本的な部分は変わることなく用いられてきた.1980年代後半になってようやく,これら従来のバイラテラル制御とは違った新しい制御法の提案がなされ始めた.一つは操作性を向上させる方向であり,もう一つは通信時間遅れに対処する方向である.これらの最近のバイラテラル制御の研究動向については文献(163)(164)を参照されたい.
