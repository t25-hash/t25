# 4・5・3 ユニラテラル制御とバイラテラル制御

> **章**: 第4章 ロボティクス
> **節**: 4・5 テレロボティクス

a.ユニラテラル制御（unilateral control) サーボマニピュ
レータは，機械式マニピュレータと違って何らかの制御系を組込ま
なければ動作しない．マスタスレーブマニピュレータの制御方式と
して最も簡単なものは，図4･115に示すようにマスタアームの関節
の角度をセンサで読み取って，それをスレーブアーム側の位置サー
ボ系の目標値とする方式である．この方式は，サーボ系（servo
system）がスレーブアーム側にのみ存在し，目標値はマスタから
スレーブへ送られるだけなので，次に述べるバイラテラル制御と対
比させてユニラテラル制御と呼ばれている．
ユニラテラル制御では，操作者の動作指令をスレーブ側に送るだ
けなので，機械式マニピュレータでは可能であったスレーブ側の作
業状況を操作者が力感覚として感じることができない．このためユ
ニラテラル制御では，マスタアームに位置センサのみを取付ければ
よいのでアームの構造が簡単軽量になるものの，環境との接触を伴
う作業は反力が感じ取れないため困難である．ただし，環境との接
触作業を主としない単純な作業であればユニラテラル制御でも十分
可能である．
