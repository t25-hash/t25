# 7・6・2 塑性加工法

> **章**: 第7章 ヒートシンク
> **節**: 7・6 製造法

塑性加工（plastic forming）は，加工能率が高く材料歩留りも
高い反面，金型，ロール等の初期コストが大きい．このため塑性加
工法は少品種大量生産に適した製造法である．被加工素材の温度に
より熱間（再結晶温度以上)，冷間（室温程度，加熱なし)，温間
（熱間と冷間の間の温度域）の各加工に分けられる．熱間，温間加
工では加工力が低減されるが，加熱装置とエネルギーコストを必要
とする．
a.押出し加工（extrusion，図7･41) 熱間加工で高速であ
る．コンテナに装てんした加熱材をラムで加圧し，金型から均一断
面の成形品を連続的に加工する．現在，市販されるプレートフィン
型の大半およびピンフィン型の一次加工品の多くは押出し加工品で
ある．金型強度の面からフィンピッチ下限が3～4mm程度，アス
ペクト比（＝フィン高/フィン厚）は最大15程度である．切断等の
二次加工が必要であり，設備，金型コストは中程度である．
b.鍛造加工（forging，図7･42) フィン寸法，寸法精度等に
応じ，熱間，温間，冷間加工の別がある．素材をフィン形状孔型を
もつ金型へ油圧力あるいは衝撃力によりプレス成形する．金型強度
の面からフィンピッチ下限が3mm程度，アスペクト比は最大25
程度．プレス荷重は100～2000tに及ぶ．高加工度のため成形フィ
ンの剛性が高く寸法精度も高い．フィン高さをそろえるため高度な
技術が必要である．素材背面（金型と反対側）にメタルフローによ
るくぼみが生じるため仕上二次加工が必要である．設備，金型コス
トは非常に高い．
c.転造加工（rolling) 冷間加工．円柱（円筒）状素材をそ
の中心軸周りに回転させ工具（一般にロール形状）を押付け塑性変
形させる．ラジアルフィン型の製造法である．工具強度の面からフ
ィンピッチ下限は2～3mm程度であり，切断，仕上げに切削工程
が必要である．設備，工具コストは中程度であり，押出しに比べ加
工能率はかなり低い．
