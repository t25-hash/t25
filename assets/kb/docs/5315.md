# 4・5・7 通信時間遅れを含む遠隔操作システム

> **章**: 第4章 ロボティクス
> **節**: 4・5 テレロボティクス

レーブの間の距離が大きくなると，その間の通信に要する時間が無
視できなくなる．通信時間の遅れがあると，通常のバイラテラル制
御では系が不安定になることがよく知られている．時間遅れの
ある通信環境下でも安定性が保証されるバイラテラル制御について
はb項で詳説するが，時間遅れが大きすぎると元来バイラテラル
制御がもつ直接作業に近い作業形態という特徴が失われてしまうこ
とになる．例えば極端な例として，火星と地球との間の通信時間遅
れ（約30分）のもとでも安定なバイラテラル制御を構成すること
は理論上は可能であるが，実際のところバイラテラル制御では作業
が不可能なのは明らかである．
