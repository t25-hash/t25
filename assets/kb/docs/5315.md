# 7・7・1 温度測定とヒータ

> **章**: 第7章 ヒートシンク
> **節**: 7・6 製造法

ヒートシンクの性能測定では，自然対流，強制対流にかかわら
ず，ヒートシンクとその雰囲気温度の測定が必要となる．その際，
一般的な温度計は熱電対で，T型（銅-コンスタンタン）を用いる
ことが多い．これはヒートシンクの場合，測定温度はたかだか
100℃台であり，この温度域ではT型の温度-起電力間の線形性が
高く精度が高いためである．ヒートシンクの温度測定点は，目的に
より異なる．機器の熱設計のため，最高温度のみ必要な場合は熱源
付近の一点を測ればよく，基板平均温度が必要なら複数点の温度を
測る．複数点の位置はガウスの数値積分点に対応した点，基板面を
等分割した各中心点等とすればよいが，アルミニウム等，熱伝導率
の高い基板ではこれら測定点配置の違いによる平均値の差は小さ
い．ヒートシンクへの熱電対固定方法はさまざまで，下記事例を参
照していただきたい．
ヒータはシート状のものを用い，ヒートシンク基板を等熱流束条
件で加熱する場合が多い．熱流束（heat flux）はヒータ入力電力
とヒータ面積から求める．この種の実験において誤差原因となるヒ
ータ背面からの熱漏えい（heat leakage）は，ヒータ背面の断熱や
2個のヒートシンクをヒータを挟んで面対称に設置することにより
回避する．シート状ヒータの作成例を示す．マイカ板（厚さ約1
mm）に等間隔かつ密にニクロム線（直径約0.1mm）を巻き回
し，一側面から給電用リード線を引出した後，両面をさらにマイカ
板（厚さ約0.2mm）で挟みアルミテープ（アルミ厚0.08mm，
接着剤厚5mm）で密着固定すれば，厚さ約2mmのほぼ等熱流束
条件を満たすヒータが得られる．
