# 7・7・2 自然対流下とファン付ヒートシンクの性能測定

> **章**: 第7章 ヒートシンク
> **節**: 7・7 性能測定法

これらの場合，自然対流，強制対流の違いがあるものの，測定量
はヒートシンク基板と雰囲気の温度および入熱量のみである．その
際，実装条件に応じて次の事項を考慮する必要がある（図7･47)．
a.自然対流下の場合
(１)ヒートシンクの姿勢
(２)使用されるのは密閉容器内か開空間か
(３)壁面からの距離（流動抵抗および放射に影響）
b.ファン付ヒートシンクの場合
(１)ファンの位置
(２)ファンの向き（押込みあるいは吸込み/平行流あるいは衝突
流）
(３)ファン流入部（流出部）の空間
(４)壁面からの距離（流動抵抗に影響）
流量（流速)，圧力損失も加わり性能測定装置として風洞（wind
tunnel ）が必要となる．ここでは冷却空気がその基板と平行に流入
する場合（以下，平行流と呼ぶ）と，基板へ垂直に流入する場合
（以下，噴流と呼ぶ）の双方における強制対流下での性能測定可能
な図7･48に示す風洞について述べる．風洞は押込み式（吐出し式）
である．ここで風洞の押込み式と吸込み式の違いについて簡単に述
べる．押込み式では，テストセクション上流のブロワ（ファン）に
よる乱れの影響を受けやすい．吸込み式は外部気流の乱れを整流部
で減じてテストセクションに導くことが容易で，流入部乱れの影響
を排除した測定が可能である．一般に低乱風洞も吸込み式による．
ただし，ヒートシンクの実際の使用条件を考えると押込み式による
測定はあながち雑で妥当性を欠くとはいいがたい．図7･48に戻る
が，空気体積流量は検定された入口ノズル とマノメータで測定
する．この入口ノズルは高精度かつ自身の圧力損失が極めて小さい
ことを特徴とし，流量に応じて複数用意する．ブロワはインバータ
制御で流量の制御応答性，安定性の点で優れているが，計測器への
ノイズ対策を要する．平行流，噴流用いずれのテストセクションも
防振ダクトを介してブロワ下流に取付ける．測定精度悪化の原因と
なる流量計-テストセクション間での空気漏えいがないよう注意す
る．ブロワを小型化してコンパクトな風洞とするためには，テスト
セクション以外での圧力損失を極力抑えることが有効である．
を発泡スチロールで断熱している．空気出口温度測定部上流にはミ
キサを設け，混合平均温度が測定できるよう配慮する．ヒートシン
