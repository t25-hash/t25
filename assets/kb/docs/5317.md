# 7・7・3 強制対流下での性能測定と測定装置（風洞）

> **章**: 第7章 ヒートシンク
> **節**: 7・7 性能測定法

強制対流下の性能測定では，測定量として温度，入熱量のほかに
図7･49は平行流用のアクリル樹脂製テストセクションで，外側
クによる圧力損失は，ヒートシンク上下流に設けた圧力タップ間の
差圧をマノメータで測定する．
図7･50は噴流用テストセクションで，ブロワと接続したアクリ
ル樹脂製のプレナム端部にオリフィス状噴流ノズルが設けられる．
ヒートシンクはフィン先端がノズルと一定距離 を保つよう固定
される．ヒートシンクとヒータとの接合およびこれらのノズルへの
固定はねじ式ジグを用い，ねじの締付け力の調整によりヒートシン
クとヒータとの接触圧力が一定となるよう留意する．
