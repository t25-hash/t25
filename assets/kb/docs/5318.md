# 4・6・2 ロボットの種類

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a. ペットロボット(pet robot)ペット型ロボットに関しては、人間にとって日常的に親しみがあるもの(例えばイヌやネコ)、日常的にはあまり知られていないもの(例えばアザラシの子供)、全く新しいキャラクター[日本電気(株)のPaPeRoなど]といったさまざまな動物型のペットロボットがこれまでに作られている。特に人間に親しみがある動物を対象とした場合には、実際の動物と似ていることが重視される。例えば、イヌがしっぽを振るときは喜んでいるときであるという知識は多くの人がもつものであり、ペットロボットにも同様のふるまいが期待される。これに対して、あまり日常的には親しみがない動物を用いた場合には、むしろ積極的にデフォルメした動作表現が用いられることが多い。人間がペットとしない動物、例えばアザラシの子供の喜び表現について知っている人も少なく、実際のアザラシの喜び表現をアザラシ型ロボットに期待する人も少ない。どちらかといえば、このようなロボットでは実際のアザラシの行動とは無関係に、アザラシ型ロボットが喜んでいるように見えることが重視される。実際には未知の動物(または動物型ロボット)に接する場合のほうが、人はその動物の動作を素直に受け入れる。いずれのペット型ロボットにおいても、おもに頭の向きやしっぽ、手足を動物に似せて動かすことで、情報伝達よりもむしろ感情表現といった情緒的交流を行うことが多い。b. 日常活動型ロボット図4・135(a)に示すRobovieは、ATR 知能ロボティクス研究所で開発された日常活動型ロボットである。視線制御可能な二つの眼、車輪による移動機構、4自由度のペイロードの小さい腕、体を覆う接触センサなどコミュニケーションに必要な最低限の機能をもつ、自律型ロボットである。図4・135(b)に示すのは、同じくATR 知能ロボティクス研究所で開発されたRobovieIIS である。このロボットの特徴は、全身を覆う柔らかい皮膚である。この皮膚はまた高感度の接触センサとなっている。人間とかかわるロボットでは、この柔らかい皮膚は非7―142メカトロニクス・ロボティクス(a) RobovieII(b) RobovieIIS図4・135 コミュニケーションロボット(a) Repliee R 1(b) Repliee Q1図4・136 アンドロイドロボット常に重要で、今後多くのロボットが標準的に備える機能になることが期待される。c. アンドロイド(android)ロボットのその姿形は単なるデザインの問題ではなく、コミュニケーションにおける情報媒体として大きな意味をもつ。コミュニケーションロボットの姿形はどうあるべきか、人間らしい姿は人間とロボットのかかわりにおいてどのような役割をもつかという基本問題に挑戦するために大阪大学と(株)ココロの共同研究で開発されたのが、図4・136に示すアンドロイドロボットReplieeである。
