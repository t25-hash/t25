# 4・6・3 感覚機能

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a. 視覚(vision)人とかかわるロボットにおいては,人と同様の感覚機能,特に視聴覚情報が重要となる.ATR 知能ロボティクス研究所で開発されたロボットRobovieでは,人を発見するための全方位カメラと,人の顔を発見するための通常のカメラを組合わせている.b. 聴覚(audition)人とかかわるロボットの聴覚処理において重要なのは,言語情報を抽出する音声認識のみではない.ロボットは実空間で活動するためにセンサ情報処理上の問題を多く抱えており,音に関する信号レベルでの処理がむしろ重要になる.例えば,ロボットから少し離れた位置から話者が発話するだけでも従来以上に背景雑音の影響が大きくなるため,既存の音声認識ソフトウェアをそのまま動作させても正しく動かない場合が多い.ロボット聴覚には2本以上のマイクロホンが必要とされるが,複数のマイクロホンを利用して音源方向や雑音処理を行うことも,実空間ロボットに必要となる聴覚処理である.また,聴覚処理から抽出されることが期待される結果も,音声認識から得られるような言語情報のみでなく,音源方向のような空間性をもった信号レベルの情報や,音の高低などの非言語情報も重要となる.c. 触覚(tactile sensing)工場など限られた環境で活動する産業用ロボットには,ロボットの活動のための安全領域が確保されており,周囲の人間とは安全のため直接触れ合わない距離がとられる.しかし,人間とのコミュニケーションに主眼をおくロボットの場合は,人間と物理的に接触する可能性が非常に高くなる.そのため,ロボット表面の柔らかさと敏感さは,コミュニケーションの相手となる人間への安全性という面において重要であり,産業用ロボットではあまり考えられていない人間との接し方に対する考慮が必要となる.またそれだけではなく,スキンシップという言葉どおり,体の表面は最初に触れるインタフェースであることから,コミュニケーションの相手がどのようにどの場所を触れているかを知ることは相手の状態を知ることにつながる.ATR 知能ロボティクス研究所で開発した人とかかわるロボットRobovieIIS[図4・135(b)]は柔らかく敏感な全身触覚をもっている.RobovieIIS では,この全身皮膚感覚と多感覚とのマッピングとして,三次元動作計測システムによるコミュニケーション相手の位置・姿勢とのマッピングを行っている.マッピングの方法は,被験者にRobovieIIS とコミュニケーションをとってもらい,そのときの皮膚センサ出力と,三次元動作計測システムによって計測されたロボットの動作と被験者の動作を同期させて記録する.その後,記録した皮膚センサ出力をISO-DATA 法等によってクラスタリングし,各クラスタに含まれている皮膚センサ出力に対応するコミュニケーション相手のロボット座標系での位置・姿勢の分布を解析する.この分布がある範囲にまとまっているときは,そのクラスタを代表する皮膚センサ出力に近い値が得られた場合に,相手の位置・姿勢の推定を高い確率で行うことが可能となる.
