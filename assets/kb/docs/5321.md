# 4・6・5 人とかかわるロボットの評価

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a. 主観評価(subjective evaluation)ロボットの対人評価において評価の中心となるのは,ロボットを観察し,また相互作用した人間がどのように感じたかといった主観の評価である.主観評価は,質問紙や聞取りによってのみ得ることができる.特に,定量的評価では質問紙の利用が不可欠である.質問紙を準備するうえで注意が必要となるのは,ワーディングと尺度である.ワーディングは簡潔でわかりやすいものが望ましい.また,複数の解釈が可能であったり,誘導的であったりしてはならない.極端な悪い例では,ロボットの可愛らしさについて評価する際に,このロボットは可愛らしいふるまいをしますが,このロボットは可愛いですかといった質問をすると,被験者の判断に不要な偏りを与えてしまう.用いる尺度については評価実験後に利用する統計手法を考慮して,適切なものを設定する必要がある.最も基本的なのは間隔尺度を用いた評価である.通常は,図4・138のように等間隔の数直線や数字の両端に意見や態度,評価を表す語彙を用いて質問文を用意する.心理学分野では,これまでに人間の性格や態度などについて標準的な質問紙が作成されてきた.例えば,性格テスト,不安尺度といったものである.また,対象のイメージを一般的に測る方法としてはSD(semantic differential)法(意味差判別法)がよく用いられる.SD 法はオズグッド(Osgood)が考案した印象の調査方法である.明るい-暗い,早い-遅いといった数十の幅広い形容詞対を用いて,評価対象について繰返し主観評価を求めることにより,単純な優劣の判別よりもむしろ印象の違いを判断することを目的とする.通常,SD 法による質問紙の結果には,因子分析が適用される.また,過去の心理学分野での研究から,どのような対象についても適切な形容詞対を用いた場合には,評価結果を因子分析するとE-P-A(evaluation:評価,potency:潜在性,activity:活動性)の3要素に基本的に分解されることがわかっている.b. 客観評価(objective evaluation)人間とかかわるロボットの対人評価では,主観的評価のみでなく,ロボットとかかわる人間の行動や反応を観察するといった客観的評価も行える.例えば,ロボットが人にものを頼むことを考えると,この依頼が成功したかどうかは,人が実際に頼んだ行動を実行したかどうかによって客観的に判断することができる.このように,あるタスクの成否を評価尺度とするのは,基本的な客観評価法である.一方,質問紙によって直接被験者に質問する代わりに,被験者の行動,体の動き,顔の表情,ロボットの行動に対する反応時間などの非言語情報を計測する方法もある.このような非言語情報のうち最も典型的なものは対人距離(personal distance)である.また,やや装置が大掛かりになるが,アイマークレコーダ(eye-markrecorder)やモーションキャプチャシステム(motion capture sys-tem)により視線や身体動作といった行動も計測が可能である.このような身体動作は,人間とロボットとのかかわりにおいて,主観的評価と相関関係にあることがわかっている.また,人間の外面に現れる行動のみでなく,人間の生体情報も客観的評価の指標となる.例えば,脳活動,脈拍,ホルモンによるストレス測定といった指標が利用されている.. モーションキャプチャシステムモーションキャプチャシステムとは,物体の正確な移動軌跡を計測するための装置である.従来,人間の動きを再現したコンピュータグラフィックスやリハビリテーションに利用されてきた.赤外線の反射を利用するもの,カラーマーカを利用するもの,磁気式のものなどがある.例えば,赤外線反射式のモーションキャプチャシステムの一つであるVICON システムでは,120Hz の速度で,10×10m程度の大きさの部屋で誤差1mmの計測が可能である(図4・139).. アイマークレコーダ人間の眼球の動きを計測することにより,人間の注視点を分析する装置のことである.車を運転中のドライバの視線分析など,インタフェースを操作中の視線移動の分析に利用されてきた(図4・140).頭部に撮影用のカメラを取付けるものが多いが,自由な相互作用がやや制約されてしまうという問図4・139モーションキャプチャシステムによる人-ロボット相互作用の身体動作計測図4・138 主観評価に用いる尺度図4・140 計測された視線(右側のロボットの手先付近の四角形と十字がそれぞれ右眼と左眼の見ている点)題もある.. 脳活動計測脳活動の計測をロボットの開発と関連付ける研究アプローチが近年徐々に増えてきている.脳活動計測装置には,fMRI(functional magnetic resonance imaging,機能的核磁気共鳴映像法),脳波計測,光トポグラフィーなどが用いられる.各装置には次のような特徴がある.fMRIは空間解像度が高く(数mm),時間分解能は中程度(1s程度)である.磁気を利用するためロボットのような電子機器を同時に使用するのは困難で,遠隔地から映像にして提示する必要がある.また,頭部を装置に固定しなければならない.脳磁図(MEG : magnetoencephalogram,脳磁場計測)は時間分解能が高く(1ms程度),空間解像度も高い(数mm)が,計測箇所は通常は1点であり,事前にMRIなどを利用して計測位置を決定する必要がある.また,頭部を装置に固定しなければならない光トポグラフィーは,時間分解能は中程度(0.1s程度)で,空間解像度も中程度(数cm)である.計測用のプローブを頭部に取付ける必要があるが,計測中に姿勢を変えたり身体を動かしたりしても問題ない.
