# 4・6・6 人とかかわるロボットの諸問題

> **章**: 第4章 ロボティクス
> **節**: 4・6 人間・ロボットインタフェース

a. 認知科学研究(cognitive science)人間の認知機構を計算論的・構成論的に調べることを特徴とする認知科学では,提案するモデルの検証はおもに計算機上でのシミュレーションによって行われてきた.しかし,人間の認知機構を解明するためには,環境との相互作用や身体との関連を明らかにすることが必要である.これらについて,従来の認知科学は実証的な研究手段をもたなかった.このような認知科学において,ロボットは実世界において提案したモデルを検証するための有効な方法論を提供する.これら認知研究で得られた成果はロボット研究へフィードバックされることになる.b. 不気味の谷(uncanny valley)ロボットの擬人化は,親しみやすいインタフェースやロボットを作る際の最も単純で効果的な開発方針である.非常に機械的な産業用ロボットよりは人間に少し似たおもちゃやロボットに,人は親しみを感じるものである.しかし,ロボットが人間に似れば似るほど人はその違いに敏感になり,ちょっとした違いを不気味に感じるようになる.例えば,生きている人間に対しわれわれは不気味さは感じないが,死体がそのまま起きて歩き出すと非常に不気味な感覚を覚えるであろうし,文楽人形も人間に似ているがゆえに,時におもちゃの人形以上に不気味さをもつ.また,このような傾向は対象が動くことによって際立つ.これは不気味の谷と呼ばれる.1970年に森によって提唱された不気味の谷は,発表当時はそれほど注目されなかったものの,近年コンピュータグラフィックスによる人間顔エージェントやヒューマノイド,アンドロイドといった擬人的なインタフェースが研究されるにつれて注目を集めている.c. メディアイクエーション(media equation)擬人化のように人工物を人間的に取扱う現象は,表面的な外見についてのみ生じるわけではない.スタンフォード大学のリーブス(Reeves)とナス(Nass)は,人間がコンピュータを扱う際にも,意図せずとも対人的に,そして社会的にふるまってしまうことを多数の心理学実験を通じて明らかにし,著書の中に示している.例えば,人はたとえコンピュータ相手であってもおせじを言われるとうれしく思い,おせじを言うコンピュータに好意をもってしまう.また,自画自賛するコンピュータよりは,他のコンピュータが賞賛するコンピュータのほうが,人は有能であるように感じる.このような知見はそもそも従来のメディアを介したコミュニケーションに当てはまってきたものである.人は,テレビに映るニュースキャスターがまるで自分の目前で話しているかのように,現実に生じた出来事のように感じる.そして,メディアイクエーションで紹介されるコンピュータに関するこのような体系的な積上げは,人間がコンピュータを,これまでのテレビやラジオといったメディアと同様に取扱うことを示してきた.人とかかわるロボットについても,人間はロボットという新しい7―145メディアを,見掛けの人間との類似性(例えば,丸に二つの点があるだけで顔のように見える)だけでなく,内面やふるまいについても,無意識のうちに擬人的に扱う傾向があると考えられる.d. 身体性(embodiment)従来の人工知能に関する研究では,実世界と切り離されたコンピュータ上の知能を実現すれば,実世界で動作する知能システムが実現すると考えられていた.これに対して身体性とは,ロボットの研究においては,体をもっているがゆえに可能になることや,必要となる性質を広く指す.体をもつことで,環境とのかかわり合いの中から知的な行動が生じるといった考え方を意味する場合もある.人とかかわるロボットにおいても,人間に似た身体をもっていなくとも,移動したり体の向きを変えたりといった身体を利用した対話を行うことが可能である.英語には身体性という語に直接対応する言葉はなく,embodiment,body propertyといった用語が用いられたりする.コンピュータ上の対話エージェントでも,手足や顔といった身体をもつ場合にはembodied agentと呼ばれることがある.
