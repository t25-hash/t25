点到達・単発作業このクラスの問題の中で最も代表的なものは,群ロボットの動作計画問題である(図4・164).これは,ロボットが初期コンフィギュレーションから目標コンフィギュレーションまで障害物を回避しつつ移動する経路を導出する,通常の動作計画問題を群ロボット系に拡張したものである.この場合,自分以外のロボットが移動障害物となるため,問題が複雑化する.最もシンプルな方法論の一つは,すべてのロボットをまとめて1台の超々多自由度ロボットと考え,その動作を決定するものである.すなわち,次元をロボットの台数分だけ掛け合わせた複合コンフィギュレーション空間(composite configuration space)での動作計画問題として解く.ホップクロフト(Hopcroft)らは,当該問題が領域計算量の観点からPSPACE(polynomial SPACE)で困難であることを示している,これは,ロボット台数が増大すると当該方法が破綻を招くことを意味する.これに対して,探索に確率的な概念を導入している一般的な動作計画アルゴリズムは群ロボット系にも適用できる.例えば,ランドマイズドポテンシャル法や確率的ロードマップ法が挙げられる.もう一つの考え方は,ロボットごとに動作計画を行い,実際に動作させて相互に衝突する危険性がある場合に局所的に調整を行う方法論である.その一つである速度調整法とは,以下の手順に基づく方法を意味する.(1)各ロボットは他のロボットの存在を無視した経路計画を行う.(2)各ロボットがある決まった速度で動作するとロボットどうしが干渉する危険性がある場合には,各ロボットの速度を調整することでロボット間回避を実現する.その一例として作業完了ダイアグラムがある.そこでは,2台のロボットの問題を対象としている.(1)で得られた各ロボットの経路の線長を縦軸横軸のパラメータとして,二次元平面に展開する.各ロボットの初期コンフィギュレーションを左下,目標コンフィギュレーションを右上に対応させる.各ロボットがそれぞれのコンフィギュレーションをとった際に干渉する場合のセルを黒く,そうでない場合のセルを白くする.初期・目標コンフィギュレーションを右上がりで連結する経路を導出する.これにより,2台のロボットがうまく速度調整しつつ目的地に向かう動作が生成できる.ここで,右上がりの制約はロボットが経路上を逆表4・3 混在・単発作業点到達作業と面被覆作業が混在している作業を混在作業と呼ぶ.存在する場所が未知である物体を適当な場所まで搬送する問題は,物体の探索(面被覆)と搬送(点到達)を含むため,この範ちゅうに含まれる.パーカー(Parker)は,機能の異なる群ロボットから構成されるフォールトトレラントな群ロボットアーキテクチャALLIANCEを提案している.障害物回避等の低レベルの行動(下位レイヤ),各作業を実現するための行動セット(中位レイヤ),どの行動セットを排他的に活性化するかを決定する動機行動(上位レイヤ)から構成されている.動機行動の構成として,いくつかの時変パラメータを導入し,それらを適切に調整することで,系全体の耐故障性を向上させている.マタリック(Mataric썝)らは,オークションベースド(auction-based)の作業割付け則を提案している.これは,以下の三つのステップから構成される.(1)発生した作業に対して各ロボットが,自己の評価関数に基づいてその作業を請け負う意思を示すかどうかを決定する.(2)オークション機構がそれらの中から適切なロボットを選定する.(3)選定されたロボットは作業を実行する.この(1)(2)のメカニズムのもち方についていくつかの手法を提案し,その有利不利について論じている.実時間性が要求される複数個の作業を迅速に実現するアーキテクチャが提案されている.複数移動ロボットにより,未知環境において外部環境を分担して計測し,じゃまな可動障害物を除去しつつ大型対象物を目的地まで搬送する作業を実現している.群ロボットが移動しながら実時間で(観測,搬送,障害物除去等の)作業割付けを行うことにより実現している.個々の作業の重要度の時間的変化と整数計画法に基づく割付け手法に基づき,割付けの迅速化を図っている.d.
