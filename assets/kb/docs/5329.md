# 4・8・2 産業用ロボットの安全

> **章**: 第4章 ロボティクス
> **節**: 4・8 ロボットの安全

a. マニピュレータ(manipulator)いわゆる産業用ロボットの国際規格は,上述の機械に関する国際安全規格類の整備に伴い,これらの規格体系の中の個別規格として2006年6月に改定された.改定以前は,国内の安全指針と同様に,人間とロボットとのガード等による隔離,または人間がロボットに近接するときはロボットが停止することを安全原則とし,教示や保守等の非定常作業時はおもに運用上の対策を求めていたが,この改定規格は人間とロボットとの共存・協調作業まで考慮していることが特徴である.以下におもな項目を紹介する.(1) リスクアセスメントに基づく安全設計リスクアセスメントに基づいた適切なリスク低減措置を要求する.(2) 停止機能非常停止機能と保護停止(インタロックによる停止)機能を有し,これらの停止は,動力源遮断か制御装置による停止後に動力源を遮断することにより達成する.(3) 人間工学に配慮した装置ロボットの安全防護空間内(例えばガード内)での教示作業においては,ペンダント制御装置にホールドトゥラン(手を離したとき機能停止する)および3ポジションイネーブル(中央位置のみ有効で,この位置を超えるか手を離すと機能停止する)装置を適用する.(4) 減速制御ロボットの教示,保守等の自動モード以外では,TCP(toolcenter point,ツールセンタポイント)において250mm/sを超えてはならない.安全防護空間内でプログラム検証のためにこの速度を超える場合は,追加方策を要求する.(5) 協働運転定常運転モードとして,例えば,運転中のロボットのハンドガイド装置を人間がロボット近傍で直接操作するための要件が規定され,動力および力の制限を本質安全設計(inherent safetydesign)による(TCPにおいて80W もしくは静的に150N)か制御システムで実現するか,もしくは速度および位置監視を制御システムで実現し,いずれも制限を超えると保護停止することを要求する.7―160メカトロニクス・ロボティクスなお,この規格において,停止機能,動力・力制限機能,速度・位置監視機能ならびにソフトウェアによる軸制限機能等の安全制御システムに対して要求される安全性能は,リスクアセスメントに基づき決定されることが基本である.b. 移動ロボット(無人搬送車)移動ロボットに関する国際規格は存在していない現状にあるが,関連する参照に値する規格としてJIS D 6802:1997,SEMI-S 17-0701:2001がある.前者は無人搬送車(AGV : automated guided vehicle)を中心とする搬送システムの導入計画段階から設置運用までの安全確保に関する一般事項について規定している規格であり,後者は半導体製造工場において用いられる無人搬送車に適用される安全規格で,通常の無人搬送車以外にも軌道上を動く搬送車や天井走行型の自動走行システムも含まれている.以下に,上記二つの規格において記載されている,おもに安全方策応用観点のセンサ技術についてその概要を紹介する.設計および計画段階における安全方策としてJIS D 6802では,外形形状,警報装置,非常時の操縦装置,障害物接触バンパの装備,接近検出装置の装備,走行速度,非常停止などについて規定されている.安全確保のためのセンサとしては,障害物接触バンパの装備が明記されており,(1)作業者および周囲に危害を与えない安全な形状および構造であること,(2)バンパ幅は進行方向の車体幅と同等以上であること,(3)バンパの反力,ストロークおよびスイッチの作動点を無人搬送車の安全に停止できる速度,距離を勘案して選定し,接触時に停止させる構造であること,との記載があり,障害物接触バンパの安全設計について規定されている.さらに,障害物接触バンパだけで安全に停止することが困難な車速の場合は,これを事前に検知する接近検出装置を装備し,安全許容内に減速または停止しなければならないとある.また,接近検出装置の作動範囲は車体幅および車体高さと同等以上とすることも明記されている.一方,SEMI-S 17-0701でも同様に障害物接触バンパの搭載,衝突回避のための非接触型接近検出装置の搭載について規定されており,特に障害物接触バンパについてはJIS D 6802での記載以外に,床との距離が15mm以下に設置すること,障害物との衝突時にバンパのストローク内で移動体が停止することなどが記載されている.無人搬送車においても,その本体が人間と接触する危険性のある場合には,国際標準の観点における安全技術性能として,a項のマニピュレータの(5)協働運転時と同等の速度・位置監視機能が要求されるとみなされる.しかし,本項で述べた無人搬送車の場合には,いわゆる移動マニピュレータが移動しながらマニピュレータを動かすような局面が想定されていないので,注意を要する.
