# 5・1・3 産業用ロボットの構造

> **章**: 第5章 ロボティクスの応用展開
> **節**: 5・1 産業用ロボット

産業用ロボットの構成を図5・1に示す。アーム本体と制御装置、教示器が基本構成であり、外界センサや周辺装置が接続される場合がある。アーム本体は、構造体、アクチュエータ、伝達機構、エンドエフェクタ、センサ等の構成要素からなる。自由度はその対象作業によって必要最小限に構成される。6自由度が標準であるが、電子基板への部品挿入やパレタイジング用では4自由度のものがあり、溶接や塗装では5自由度のものもある。また、広い動作領域を求められる場合には、ロボット本体のほかに、移動のための自由度を追加した構成もある。軸構成では、産業用ロボットとして高い作業性を確保することに留意する必要がある。すなわち、動作領域が広いこと、特異点が動作領域内で少ないこと、座標変換が容易であることなどが求められる。実際には、ロボット台座側からの3軸で先端の位置を決め、手首部の構成軸で姿勢を決めるように役割を分担させ、手首部の構成軸を一点で交わるように構成することが多い。このため、手首軸の構成はねじり、曲げ、ねじりの順番とするのが一般的である。構造体はアクチュエータや伝達機構を保持し、配線・配管を内蔵し、関節間のリンクを構成する。材質は通常、鉄またはアルミニウム合金である。ロボットの性能向上のため、軽量性と高強度・高剛性の両立が要求される。アクチュエータは、初期には油圧シリンダが多用されたが、簡易型では低価格の点から空気圧シリンダも使用された。油圧アクチュエータは、当時のサーボモータに比して小型軽量で大出力であることから、油圧サーボ弁と組合わせて使用された。また塗装用では、防爆性の観点から使用された。しかし油圧型ロボットは、サーボ用作動油管理の煩雑さ、エネルギー効率の悪さ、特性の温度変化や非線形性による制御の困難さなどの問題点を抱えていた。電動サーボモータは、当初は制御性のよい高出力のものがなかったが、性能の進歩が著しく、現在ではほぼ完全に油圧アクチュエータに取って代わった。高出力が可能になったこと、応答性が高いこと、制御性がよいことが電動化を可能にしたおもな要因である。サーボモータの形式はDC(direct current)ブラシ型から保守性のよいDCブラシレス型、AC(alternating current)型へと進化した。特殊用途向けにモータ出力軸で直接駆動するダイレクトドライブ図5・1 産業用ロボットの構成7―167方式もあるが、一般に電動モータは、減速することでロボットに要求されるトルクを発生している。減速機として望まれることは、軽量、高剛性、無バックラッシ等である。小型のロボットや手首部のように比較的小さな出力トルクの領域では、ハーモニックドライブがよく使用される。多用される理由は、大減速比で小型軽量であること、ロストモーションという非線形性をもつがバックラッシがないことなどである。一方、ハーモニックドライブの伝達トルクに限界があること、剛性もそれほど高くないことから、大トルクの伝達にはサイクロ減速機、RV(rotary vector)減速機等がよく使用される。これらの減速機も、従来の歯車を組合わせた減速機に比して軽量、無バックラッシ等の特徴をもつ。ロボットの各関節軸にモータ・減速機を設置すると、アームの先端側の重量が増加し、可搬重量の低下、動特性の悪化、振動、たわみの増大等の悪影響を及ぼすことになる。このため、重量物であるモータ・減速機はなるべくロボット基部側に設置して伝達機構で各関節部に動力を伝達する必要が生じる。伝達機構の要素としては、歯車、歯付きベルト、リンク、トルクチューブ等がある。これらを組合わせて伝達系を作るが、手首部関節では複数の軸の伝達要素が集中するため非常に複雑な構造となる。ロボットアームの先端で作業を行う部分をエンドエフェクタという。ワーク(work-piece、被加工部品)をハンドリングするロボットではハンドが、溶接ロボットや塗装ロボットでは溶接トーチやスプレーガンがそれに当たる。ハンドやツールは作業に応じて専用のものが用意され、複数のワークを扱う場合は、それぞれのワークに対応したハンドやツールを状況に応じて交換する機構(ツールチェンジャ)が使用される。1台のはん用ハンドであらゆるワークやツールをハンドリングするのが理想であるが、作業精度や確実性の要求を満たすものはまだなく、当面は現状方式が続くであろう。通常、産業用ロボットは、教示データを指令値とした位置サーボ駆動を行っている。センサは、位置検出器として各モータに接続されたエンコーダが基本になっている。速度は位置信号を微分することで、駆動トルクはモータ駆動アンプの電流検出回路で得るのが一般的である。近年、作業に応じて外界センサを使用する例が増えてきた。視覚センサが多用されるが、ワークの種類、位置・姿勢の判断、教示データと実際のワーク位置との差の補正用等に使用される。高速処理と高精度の両立が課題となるが、その認識能力は急速に進歩しており、多数のセンサが実用化されている。しかし、現状では実時間での処理は比較的単純な形状に制限されている。力制御等でワーク反力を検出するには、エンドエフェクタ部に力センサを設置するのが一般的で、ひずみ計を使用した6軸力覚センサ(3軸方向の力とトルクを検出する)が多用される。今後の発展が期待される分野であるが、センサの精度、頑健性等、広く使用されるためには課題が残されている。
