# 7・5・2 バイオ人工肝臓の必要性

> **章**: 第7章 人工臓器
> **節**: 7・5 ハイブリッド型人工臓器

肝臓はしばしば人体の化学工場にたとえられるように，高度で複
雑な代謝反応をつかさどる，体内では最大の臓器である．その主要
な機能は，(１)有毒物質の解毒，(２)タンパク質，凝固因子などの
有用物質の生産，(３)水分，イオンの代謝，調節などである．一
方，何らかの原因でこの肝臓の機能が広範に障害された重症の肝不
全患者に対しても，その急性期に一時的に肝機能を代行できれば，
多数の患者が救命できると考えられている．これは，肝臓は他の臓
器とは異なって旺盛な再生能を有するためであり，急性期を乗り切
れば患者自身の肝再生能によって救命・延命を期待できるからであ
る．したがって，人工肝臓が目的としているのは週単位の短期間の
肝機能補助であり，万一その間に患者の肝機能の再生が見られない
場合にも，肝臓移植へのつなぎとしての役割が期待されるわけであ
る．このように人工肝臓の開発目標が限定されているにもかかわら
ず，現在の進歩した血液浄化法をもってしても劇症肝不全患者の約
30%程度しか救命することができないところから，生体肝臓の細
胞，組織を利用する新しい形式のハイブリッド型ないしバイオ人工
肝臓（bioartificial liver ）の開発が強く求められている．
肝臓にコラゲナーゼという酵素を含む溶液を灌流すると，肝細胞
をばらばらにして分離，摂取することができる．バイオ人工肝臓
は，これを体外で（in vitroに）長期に培養して物質移動装置の中
に組込み，肝臓の機能を代行させようとするものである．肝臓自身
は，すでに述べたように高度の再生能を有しているが，in vitroの
培養系では肝細胞はほとんど増殖しないので，これがバイオ人工肝
臓を実現するうえでの最大の障害になっている．このような肝細胞
の特殊な性質に加えて，肝臓は人体で最大の内臓であるので，この
機能を人工的に代行するのは装置工学的に見ても極めて困難な課題
細胞密度（10個/cm -基材程度）を維持しつつ，なおかつ2週間
程度の期間，肝細胞の代謝機能を維持できる培養方法あるいは培養
装置（バイオリアクタ）を開発する必要があり，しかもそのリアク
タはスケールアップや無菌操作の容易さなどの困難な要件が満たさ
れなければならない．この問題点を克服するためにこれまでにさま
ざまな試みが行われ，一部では大胆な臨床応用も報告されている．
紙数の関係で，以下ではバイオ人工肝臓の開発に関する代表的な試
みのいくつかについて簡単に触れることとし，技術の詳細について
は文献(14)～(16)などを参照いただきたい．
