# 5・1・5 産業用ロボットの教示(teaching)技術

> **章**: 第5章 ロボティクスの応用展開
> **節**: 5・4 の例では、ダイナミックフィードフォワード部でロボットの逆

産業用ロボットは通常、あらかじめ決められた条件、データに従って作業を行う。これをティーチング・プレイバック(教示・再生)方式という。教示方法の分類を図5・5に示す。教示方式を大別すると、実ロボットを使用するオンライン方式と、使用しないオフライン方式になる。7―170メカトロニクス・ロボティクスオンライン方式とは、ロボットを実際に動作させながら教示データを作成、記録するものであり、さらにダイレクトティーチ(di-rect teach)とリモートティーチ(remote teach)に分けられる。ダイレクトティーチは、操作者がロボットアームの先端を持って必要な位置に動かす方式である。直接ロボットを目標位置に動作させることから、間違った教示をするおそれはない。また、重力によってアームが降下するのが防げる構造であれば、動力を切った状態で教示(パワーフリー教示)できるので安全性が高いなどの利点がある。一方、欠点としては、高精度な教示が困難であること、モータ駆動方式ではクラッチ等の動力遮断装置を設けることで構造が複雑化することなどが挙げられる。また、パワーフリー教示ができない場合はサーボ状態で教示することになり、教示のための指令器の設置の必要性や暴走の危険性の問題が出てくる。このため過去において、重力補償ばねをもつ塗装ロボットの教示に使用されたことがあるが、現在はあまり採用されない。リモートティーチの代表は、図5・6に示すようなティーチングボックスあるいはティーチングペンダントと呼ばれる教示器で、ボタン操作によりロボットを必要な位置に移動させるもので、現在最も多用されている教示方式である。操作方法は押しボタン操作が一般的で、ロボットの各軸を単独に動作させるモード(各軸モード)、ロボットの取付面に固定された座標系でのモード(ロボット座標または絶対座標モード)、エンドエフェクタに設置された座標系のモード(ツール座標モード)等を選択して動作できるようになっている。本方式は精確な教示が利点であるが、操作員がロボットの動作範囲内で作業することが多いので、十分な安全対策を必要とする。なお、上記以外のリモートティーチ方式として、マニピュレータの形状をしたものを操作者が動かしてロボットへの指令とするマスタスレーブ方式や、ジョイスティック方式がある。操作は容易であるが、精確な指令には不向きであるので、教示用としては現在はあまり使用されない。オフラインティーチ(offline teach)とは、作業条件やデータをもとに、あらかじめロボットの動作プログラムを作成することをいう。有効なオフラインティーチを行うには、三次元の作業・環境データ、作業性のよいロボット言語と、高機能のロボットシミュレータが必要である。ロボットの動作プログラムを作るには作業対象や環境の三次元データが必要となるが、現在は三次元CAD(com-puter aided design)が一般的になってきており、この点でオフラインティーチの基盤は整っている。単純な環境で簡単な作業を行うのであれば、人間が手動で現状のロボット言語を使用してプログラミングすることは十分可能である。しかし、ロボットは通常、入り組んだ狭い作業環境で最短時間の動作を要求される。他の装置との干渉を避けながら動作可能か、ロボットの各軸にはどれほどのパワーが必要となるか、作業のサイクルタイムはどれほどかかるかなどを勘案して、作業プログラムの作成を遂行するのは大変な手間である。このため、従来は有効なオフ図5・6 教示器の例ラインティーチが困難であったが、最近はロボットシミュレータが進歩して、人間と対話しながら最適の条件抽出ができるようになってきた。すなわち、図5・7に示すように、仮想空間上でロボットに動作をさせて、機構的に必要な位置と姿勢がとれるか、装置間の干渉はないか、ロボットの各軸の速度やトルクは定格内に収まっているか、さらには制御システムとしても実現可能かなどを検討する。そして、最終的にはロボットの動作プログラムを自動生成する。このような機能をもつシミュレータが実現したことによって、オフラインティーチが実用化されることになった。なお、ロボットシミュレータは、ロボットの教示のみでなく、作業の可能性の事前検討、さらにはシステム設計の段階で、ロボットの選定や装置の設置位置の検討を行う際にも必須なものとなっている。また実際の教示では、オフラインティーチだけでは十分ではなく、オンラインで修正・調整を行うのが一般的である。その理由は複数あるが、最大のものはロボットの絶対精度の不足である。今日の産業用ロボットは、繰返し精度は0.5mm以下が一般的であるが、絶対精度は数mm以上である。このため、オフラインティーチのデータをそのまま実行しても精度の高い作業をすることはできず、ロボット1台ごとにオンラインで教示データの修正をする必要がある。これが教示時間を長くし、ロボットのスムーズな交換を困難にしていることから、最近は絶対精度の向上が強く求められている。ロボットを交換してもつねに必要な精度が保証されるようになるには、ロボット機構の加工や組立における精度向上も必要であるが、それらには限界があり、各ロボットの個体差を計測、評価してその誤差を補正するシステムが必要となるだろう。
