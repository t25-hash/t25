# 5・1・6 応用(application)事例

> **章**: 第5章 ロボティクスの応用展開
> **節**: 5・4 の例では、ダイナミックフィードフォワード部でロボットの逆

産業用ロボットを生産ラインに導入するに当たっては、事前に慎重な検討を要する。ロボットの特性を理解し、それを生かす計画を立てないと、所期の経済効果を得ることはできない。ロボットと専用機、人間の一般的な特性比較を表5・4に示す。図5・7 ロボットシミュレータの例表5・4 ロボットの特性比較比較項目ロボット専用機人間はん用性がある○×◎作業速度が速い△◎○(限定付き)24時間連続稼働◎◎×悪環境下での作業○◎×重量物が扱える○◎×職業、判断能力がある△△◎臨機応変に対処できる××◎表5・4からわかるように、産業用ロボットは決して万能ではない。ロボット化で作業が高速化するかというと、必ずしもそうではない。重量物のハンドリングでは人間以上の速度を出すが、判断や前後の作業とのインタフェースでは人間のほうが高速の場合が多い。ロボットで作業効率が上がるというのは、24時間連続作業が可能なことから、トータルで人間以上の効率を得ていることが多い。逆にいうと、長時間作業を必要としない少量生産では、ロボット化のメリットが得られない場合が多い。また、ロボットのはん用性についてよくいわれるが実際は限定されたものであり、そのロボットの性能が十分発揮できる範囲で使わないと作業効率が落ち、投資効果も悪化する。はん用性をもたせるためにはソフトウェア体系が複雑になり、維持・管理が困難になる。また、認識部を含めた外界センサの機能・性能も、一般には人間に比してかなり低いのが現状であり、経済性を確保してのはん用性は限られたものであると認識すべきである。結局、産業用ロボットはある程度のはん用性をもった中・大量生産向け専用機械であると考えて、導入を計画すべきである。その他、導入に当たっては、ロボットメーカ間の互換性がないこと、メーカとユーザが協力して設計しないと満足のいくものにならないことなどに留意すべきである。また、ロボットを実際に導入するには、目標設定、構想、基本設計、詳細設計、製作、搬入、組立、教示、試運転、運転教育、稼働、保守と多くの段階を経る必要がある。それら各段階での検討も多岐にわたり、速度や精度等のロボット単体性能評価のほかに、作業性や干渉問題等のレイアウトの検討、作業時間やインタロックの検討、ハンド・ツール設計、付帯機器・周辺機器設計、安全の検討等を行う必要がある。また場合によっては、生産する商品をロボット導入システムに合わせて再設計する必要が生じることもある。現在これらの検討は、5・1・5項で紹介したロボットシミュレータによって詳細かつ定量的に評価され、ロボット導入計画に反映されるようになっている。以下、代表的応用事例を示す。a. ローディング(loading)・アンローディング(unloading)本作業は広い意味ではハンドリング(handling)であるが、単なる把持とセット動作ではなく、工作機械や生産ラインのジグとのワークの授受作業であって、位置や姿勢において高精度を必要とする。また、ワークをハンドリングしている時間は生産のための加工は行っていないので、作業時間ゼロが理想である。このため、生産工程時間の短縮の点から高速性を要求されることが多い。図5・8に、ロボットがパレットに積載された素材を取出し、工作機械に投入するシステムを例示する。ここでは、ロボットに搭載したカメラ(視覚センサ)でパレット内に乱積みされた素材ワークの位置、傾きを認識し、これを把持して取出し、工作機械に投入している。従来のロボットでは、対象となるワークをあらかじめ位置決めしておく必要があり、このためにワークのストッカ装置や整列・位置決め装置などが不可欠であった。これを解決するために、パレット内をカメラで見て、捕らえた画像をもとにワークの種類、位置、傾図5・8ローディング・アンローディングの例[提供:(株)不二越]図5・9スポット溶接の例[提供:川崎重工業(株)]7―171きを認識し、これを把持して取出す方式を採用した。また、照度の変化やワークに付着したごみ、汚れなど、外乱要因からの影響を受けにくくするため、複数のカメラと照明を使用するなどの対策を施している。図5・8のように、画像処理機能を組込んだシステム構成が採用されるようになって、ロボットシステムの知能化が可能となり、より複雑な人間に近い作業の自動化が実現できる傾向にある。また、これにより従来必要であったストッカ装置や整列装置が不要となり、省人化、省スペース化と設備コスト低減が可能となる。b. 抵抗スポット溶接(spot welding)抵抗スポット溶接は初期の産業用ロボットの最も成功した適用例といえよう。溶接ガンなど抵抗スポット溶接機器をロボットに付帯させることにより、抵抗スポット溶接を行い、ホワイトボデーの組立を行うものである。最大のユーザは自動車産業であり、ボデーの製造ライン等に大量に投入されている。本作業に使用されるロボットは、質量150kg 前後のスポット溶接ガンを把持するため、130kg から200kg までの可搬重量の能力をもつことが求められ、ガン用冷却水やガントランス用電源ケーブルなどをロボットアームに内蔵し、抵抗スポット溶接専用ロボットとして使用されるケースが多い。また、おもに自動車ボデーを作業対象とするため、広い動作範囲が必要であると同時に、連続スポット溶接作業でサイクルタイムを短くするため、加減速の速い動作特性が必要となる。その他、自動車工場のラインでは限られたスペースの中で複数台で使用されることが多く、周辺ロボットとの動作干渉をチェックするなど相互インタロックをもつ。適用事例を図5・9に示す。スポット溶接作業においては、特殊な部位を除いて要求位置決め精度は±1mm程度とそれほど高くなく、溶接点間の移動は軌跡精度が要求されないPTP動作で済まされる。しかし、溶接点数が多いため、点間のタクトタイム削減が最重要視され、現場では0.1秒の増減が問題となる。一方でイニシャルコストやランニングコストも低く抑える要求があるため、単純にモータの容量や駆動電流を上げるわけにはいかない。このため、最短の加減速時間となる速度パターンの生成や振動の抑制が制御上の課題となり、各種の現代制御や非線形制御の導入も試みられている。教示に関しては、現場での立上げ時間の短縮が望まれる中、ロボット1台当たりの作業点が多いことから、オフラインティーチの要求が強い。オフラインティーチは実行されつつあるが、ロボット各機に差があるため現場での補正教示が行われているのが現実である。生産ラインの立上げ時間を少しでも短縮するため、ロボットの絶対位置精度の向上が強く求められつつある。また、予防保全の見地から、故障する前に事前に警告が出せるような仕組みが考えられている。一例として、各軸のモータ電流をモニタし、電流値の変化により減速機など駆動系の異常を察知するシステムが挙げられる。現在一応の仕組みはできたが、今後精度の向上が望まれる。システム全体を見ると、従来、溶接ガンはエア駆動であり、ロボ7―172メカトロニクス・ロボティクスット動作とガンによる溶接は原則的に独立していたが、最近はロボットのコントローラで制御されるサーボモータで溶接ガンを駆動するタイプが増えており、ロボットが溶接品質の向上、管理に寄与している。また、昨今の電子技術の進歩により、インタロック盤、工程制御盤など従来は周辺機器で行っていた機能をロボットが取込むなどロボットの守備範囲(役割)が広くなると同時に、設備のトータルコスト削減に貢献している。c. 塗装(painting)塗装にはスプレー塗装と塗布塗装とがあるが、産業用ロボットの塗装はスプレー方式がほとんどである。通常の塗装ではシンナ等の可燃性の溶剤を用いるため、塗装用ロボットには防爆性が要求され、これが大きな特徴となっている。また作業の性質上、高精度の位置決めはそれほど要求されないが、軌跡の制御(CP制御)が行われ、広い手首姿勢や高速動作が要求される。軸構成はスプレーパターンにもよるが、5軸で済ます場合もある。おもに防爆性の観点から、初期には油圧駆動ロボットが塗装用に使用された。サーボ弁を工夫することで、本質安全防爆が実現できたためである。その後、サーボモータの進化によりACサーボモータが出現して安全性が向上し、電動方式が主流となった。現在はロボット構造体の内部圧力を外部より高圧に保つことで、爆発性のガスを電気品から隔離する内圧防爆方式が一般的になっている。この方式は、塗料がロボット構造内部に入るのを妨げる働きも兼ねている。図5・10に塗装ロボットシステムの例を示す。ワークをハンドリングするポジショナと協調動作を行うことで、高効率の作業を実現している。また、ロボット構造体内部に塗装ホースを設置して、作業性を向上させている。現在はワークの外面の塗装が主となっているが、今後は内面塗装への発展が求められるであろう。そのためには、ドア等のふた物をあけるオープナの開発や、関連機器の配置問題等が課題となる。また、今後オフラインティーチへの要求が強くなると思われるが、教示の簡便化も実用上重要となる。なお将来、水性塗料の採用が広がることも考えられる。その場合、ロボットは防爆性を免除されるが、水性塗料特有の塗装条件への対応が求められる。d. 組立(assembling)組立作業では、ワークどうしの位置決め精度を高くする必要がある。組立てられる部品間のすきまが大きければ通常の位置決め精度でよいが、少しすきまが小さくなると、組立作業は不可能になる。このため、初期の産業用ロボットでは、ごく粗い組立作業しかできなかった。近年、視覚センサの発達により組立ワークの相対位置・姿勢が計測できるようになり、センサ信号による補正で高精度の組立が実現されるようになった。さらに、締まりばめや視覚センサが使用できない場合には、力・トルクセンサを使用し、力・トルクの制御によって組立作業を行う事例も増えつつある。組立システムの例を図5・12に示す。リニアモータで駆動される走行軸上を分離・合体しながら動く2台の知能ロボットが協調してミニロボットを組立てるシステムである。知能ロボットは手首に力センサと三次元ビジョンセンサを搭載している。このシステムでは、ベアリング、減速機、モータ等の精密機械部品をロボットが自図5・11 その他(シーリング、レーザロボット、バリ取り、半導体、検査、分解・解体)上記の代表例のほかに、種々の作業に産業用ロボットが適用されている。ローディング・アンローディング関連では半導体分野に適用事例が増えており、将来はバイオテクノロジーや製薬関連にも進出すると考えられる。溶接関係ではレーザを使用しての溶接・切断の実用化が進みつつある。塗装関連では自動車分野を主としてシーリング作業への適用が多い。その他、バリ取り作業が実用化しつつあり、リサイクル社会の到来に対して分解・解体用ロボットも実用化していくであろう。
