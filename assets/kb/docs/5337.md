# 5・2・1 ロボットセル(知能ロボット応用生産セル)

> **章**: 第5章 ロボティクスの応用展開
> **節**: 5・2 知能ロボット応用生産システム

本項では、産業用知能ロボット(industrial intelligent robot)を応用した生産システムとしてロボットセル(robot cell)を紹介する。初めに、このロボットセルが登場した背景について述べる。今日、先進工業国の機械工業は次に示すような大きな課題を抱えている。(1) グローバルな国際競争の中でコスト競争力を維持しなければならない。(2) ユーザからの多様な要求に迅速に対応しなければならない。(3) 少子・高齢化等により熟練作業者の数が大幅に減りつつある。このような課題を抜本的に解決する手段として、産業用知能ロボット[以下、知能ロボット(intelligent robot)]を応用した生産システムであるロボットセルが開発された。ロボットセルは周辺機器やジグの多くが不要であり、設備費を大幅に抑制する。さらに、生産する対象物が変更された場合でも基本的に知能ロボットの動作プログラムを入れ替えるだけで済み、短時間でシステムを再構築することができる。すなわち、高い再構築性(reconfigurability)と敏捷性(agility)をもつ。a. 知能ロボット知能ロボットはロボットを知能化することにより生まれた。ここでいう知能化とは、人間のように考えることではなく、熟練作業者に匹敵する技能を機械にもたせることである。知能化されていない普通のロボットは、工場を自動化して生産性を高め、生産コストを低減し、仕事を教え込めばそれを正確に繰返すことで製造品質の安定に大きく役立っているが、細かい動作を人がすべて教える必要があるため手間がかかるという問題があった。また、部品供給装置など専用の周辺機器、あるいは部品を正確に位置決めするための多くのジグを準備しなければならず、設備費増加につながっていた。加えて、事前にワーク(work-piece、被加工部品)を決められた位置に人が正確に並べるという、非常に単純な労働が毎日の日課として発生し、ロボットが働くために人が下働きをしなければならないという状況が生じた。そこに、普通のロボットから進歩した知能ロボットが登場した。知能ロボットは、眼としての視覚センサ(vision sensor)と手の感触を得る力センサ(force sensor)を搭載している。ただし、知能ロボットとは、単にセンサが付いたロボットではない。センサとロボットとの間の情報のやりとりにより作業に関する自律性(autonomous func-tionality)を獲得し、熟練作業者に匹敵する作業能力をもつことが7―173できた。以下に、知能ロボット実現のためのキーテクノロジーである視覚センサと力センサについて詳述する。. 三次元ビジョンセンサ(three-dimensional vision sen-sor)三次元ビジョンセンサは図5・13に示すように、CCD(charge coupled device)カメラと十字形のレーザ光であるストラクチャ光(structured light)を組合わせたセンサで、対象物の三次元位置および姿勢を検出することができる。まず、対象物にストラクチャ光を投光し、検出対象面を三角測量法で求める(三次元処理機能)。次に、その面上にある検出対象の特徴(例えば円の中心)を検出する(二次元画像処理機能)。このように、三次元ビジョンセンサはこの二つの機能を組合わせて対象物の検出を行う。. 力センサ機械部品の多くは、JIS のH 効果加工ロボットセルでは、ワーク取付け・取外し工程に知能ロボットを採用することで、第二世代機械加工システムと比較して人件費、加工費および初期設備投資費を大幅に削減し、43%もの加工コストの節減を達成した。c. 組立ロボットセル(assembly robot cell)加工分野と同様に、組立分野においても組立ロボットセルが実稼働を始めた。これは、知能ロボットが作業者に代わって組立作業を行うロボットセルである。. 組立ロボットセルの登場機械工業の工場は、大きくは機械加工工場、機械ユニット組立工場、さらに機械を完成させる総合組立工場の三つに分けられるが、これらの全作業工数の多くは組立工数であるといわれており、従来、機械工業における組立作業は人手に頼る部分が多かった。1995年に、経済産業省が主導する知的生産システムIMS(Intelligent Manufacturing System)プロジェクトにおいて、双腕の知能ロボットがミニロボットを組立てるシステムが試作された。. 組立ロボットセルの構成組立ロボットセルは、機械部品の組立を行う知能ロボット、搬送・ボルト締めを行う知能ロボット、および部品棚、部品供給台より構成される。図5・19に組立ロボットセルを示す。. 効果知能ロボットの導入により、多くの精密機械部品からなる産業用ロボットの自動組立が可能となり、ロボットがロボットを作ることができるようになった。この組立ロボットセルでは、人手作業の工数を従来に比べ約40%削減できた。d. ロボットセルの今後の展望知能ロボットの導入により、加工から組立まで一貫したロボットセルによる生産が実現され、そのコスト競争力は従来に比べ大幅に高まった。これにより、世界のどこでも良好な品質とコストで生産できるので、企業は研究開発、製造、販売、サービスの総合力といった企業本来の力によって国際競争を行う時代となり、知能ロボットによる新しい生産の時代が始まった。
