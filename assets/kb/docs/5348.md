図6･49マイクロファクトリの生産形態
a.マイクロファクトリのメリット
.省のつく効果 生産システムが小型化することにより，
機械自身を駆動するための直接的な省エネルギーだけでなく，恒温
室やクリーンルームといった環境の制御，照明などの間接的な省エ
ネルギー効果も得られる．むしろ後者のほうが割合が大きく，工場
全体の消費エネルギーの70 %を超えるケースが多い．また，設置
面積が小さくなることで省スペース化が図られるうえ，小型軽量な
機械は頑強な基礎を必要としないことから，配置の自由度が上が
る．立体的なライン配置や，マンションの一室を工場として利用で
きる可能性もある．さらに，工場建設や生産設備を構築する資材の
削減，小型部品の生産に適した工程による原材料や産業廃棄物の削
減といった省資源効果も期待できる．
.多様な生産形態 生産システムの小型軽量化による配置
自由度の高さから，これまで固定的な概念でとらえられてきた工場
のイメージを超えて，さまざまな生産スタイルおよび利用形態が想
定される．図6･49は，それらを生産方式と生産・利用場所で分類
し，具体例を示したものである．ここでは，生産システムを分
散型と集中型の二つに分類している．
分散型とはデスクトップサイズのマイクロファクトリがいろいろ
な場所に設置されて生産を行う，あるいは利用される形態である．
小型化された生産システムは，工場以外のさまざまな場所に設置で
きる．店頭など消費者により近い場所に設置することで，必要なと
きに（オンデマンド，on demand），ニーズに合わせて（カスタマ
イズ）製品を作ることができる．また，交換部品の供給など需要変
動が大きかったり，多品種少量の生産をしたりする場面でメリット
集中型とは，マイクロマシンが多数集まり，協調作業によって自
身よりも大きなものを生産あるいは解体・再生する形態，いわば小
さな大工ロボットのイメージである．具体的には，多数のマシンが
地中に潜って配管工事等を行うモグラ型生産システムなどが考えら
マイクロマシンに限定せずとも，近年の携帯情報機器に代表され
れる．
新たな生産形態が生まれれば，そこで作られた製品の売り方も変
わってくる．つまり，新しいビジネスモデルが生まれる．カタログ
だけでなく，営業担当者が製造設備を持ち歩き，その場で注文を受
けて製品を作るスタイルや，ベンチャー企業に工場設備を一定期間
だけ貸し出すレンタルファクトリなど，アイデア次第で多彩なビジ
ネスを展開できる可能性が広がる．
.技術的メリット 工学的な視点から小型化による機構や
材料特性の変化を見ると，小型軽量な機構は高速駆動や超高速回転
ここでは，最初に生産設備をマイクロファクトリ化するメリット
に有利であり，固有振動数が高くなることから外乱に強いメカニズ
ムを実現できる．また，熱的な特性では，熱変形量は寸法に比例し
て小さくなり，小さな熱容量により短時間の加熱冷却が可能にな
る．設計においても，理想的な特性を有する高価な材料を適用する
