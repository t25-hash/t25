# 6・2・1 マイクロ工作機械とマイクロファクトリ

> **章**: 第6章 メカトロニクスの応用展開
> **節**: 6・2 マイクロメカトロニクス

マイクロファクトリ(micro-factory)とは、小型の部品・製品に見合ったサイズの小型生産システムのコンセプトである。この考え方は、マイクロマシン(micro-machine)の製造システムとして、どんなスタイルが適しているかという議論の過程で生まれてきた。マイクロマシンに限定せずとも、近年の携帯情報機器に代表されるように製品自身の小型化・高機能化に伴い、実装密度が高まり、構成部品のサイズも小さくなっている。このような状況で従来の設備をそのまま使用すると、加工対象に対して生産システムのサイズが相対的に大きなものになる。具体的には、米粒大の部品の加工に、人の背丈よりも大きな加工機械が用いられることもありうる。マイクロファクトリでは、このような場面において、生産設備を小型化することでさまざまなメリットが得られることを提唱している。ここでは、最初に生産設備をマイクロファクトリ化するメリットを述べ、続いていくつかの具体例を紹介しながら、マイクロメカトロニクス技術の現状と課題を整理する。生産方式生産・利用場所具体例分散型生産システム移動体生産船内魚介類調理システム(物流を兼ねたファクトリ)デスクトップサイズファクトリによる生産特殊環境下生産宇宙実験システム、無菌環境医療機器製造システム無重力下食品・薬品製造システムマイクロファクトリ一般生産携帯機器製造、化学合成マイクロファクトリ超小型電池等生産、細胞クローン培養マイクロマシンの協調作業による生産図6・49マイクロファクトリの生産形態a. マイクロファクトリのメリット。 省のつく効果生産システムが小型化することにより、機械自身を駆動するための直接的な省エネルギーだけでなく、恒温室やクリーンルームといった環境の制御、照明などの間接的な省エネルギー効果も得られる。むしろ後者のほうが割合が大きく、工場全体の消費エネルギーの70%を超えるケースが多い。また、設置面積が小さくなることで省スペース化が図られるうえ、小型軽量な機械は頑強な基礎を必要としないことから、配置の自由度が上がる。立体的なライン配置や、マンションの一室を工場として利用できる可能性もある。さらに、工場建設や生産設備を構築する資材の削減、小型部品の生産に適した工程による原材料や産業廃棄物の削減といった省資源効果も期待できる。. 多様な生産形態生産システムの小型軽量化による配置自由度の高さから、これまで固定的な概念でとらえられてきた工場のイメージを超えて、さまざまな生産スタイルおよび利用形態が想定される。図6・49は、それらを生産方式と生産・利用場所で分類し、具体例を示したものである。ここでは、生産システムを分散型と集中型の二つに分類している。分散型とはデスクトップサイズのマイクロファクトリがいろいろな場所に設置されて生産を行う、あるいは利用される形態である。小型化された生産システムは、工場以外のさまざまな場所に設置できる。店頭など消費者により近い場所に設置することで、必要なときに(オンデマンド、on demand)、ニーズに合わせて(カスタマイズ)製品を作ることができる。また、交換部品の供給など需要変動が大きかったり、多品種少量の生産をしたりする場面でメリットがある。集中型とは、マイクロマシンが多数集まり、協調作業によって自身よりも大きなものを生産あるいは解体・再生する形態、いわば小さな大工ロボットのイメージである。具体的には、多数のマシンが地中に潜って配管工事等を行うモグラ型生産システムなどが考えられる。新たな生産形態が生まれれば、そこで作られた製品の売り方も変わってくる。つまり、新しいビジネスモデルが生まれる。カタログだけでなく、営業担当者が製造設備を持ち歩き、その場で注文を受けて製品を作るスタイルや、ベンチャー企業に工場設備を一定期間だけ貸し出すレンタルファクトリなど、アイデア次第で多彩なビジネスを展開できる可能性が広がる。. 技術的メリット工学的な視点から小型化による機構や材料特性の変化を見ると、小型軽量な機構は高速駆動や超高速回転に有利であり、固有振動数が高くなることから外乱に強いメカニズムを実現できる。また、熱的な特性では、熱変形量は寸法に比例して小さくなり、小さな熱容量により短時間の加熱冷却が可能になる。設計においても、理想的な特性を有する高価な材料を適用する店先、オフィス内家庭内生産パーソナル・オリジナルグッズ製造(化粧品、コンタクトレンズ)廃物再生機(生ごみ→有機肥料)教育用小型生産システム、ラボサイドファクトリオンサイト生産交換部品生産システム(自動車修理)故障解析装置、分析装置、飲料水製造機集中型生産システムモグラ型生産システム(油田・鉱物探査、ケーブル敷設)環境保全マイクロファクトリ(害虫駆除、オイル回収)建築物建造および解体マイクロマシンなど従来の常識から大きく離れた独創的なアイデアを積極的に取り入れ、低コストで試作機を開発できる。これらのメリットを生かして、従来のシステムでは不可能な加工条件を付与し、より高精度な加工を実現できる可能性がある。技術開発に成功して量産体制に入る際にも、小型機を複数台並べることで必要な生産能力を得ることができ、需要に応じた生産調整にも柔軟に対応できる。b.
