# 6・2・2 マイクロメカトロニクス機器

> **章**: 第6章 メカトロニクスの応用展開
> **節**: 6・2 マイクロメカトロニクス

マイクロメカトロニクス(micro mechatronics)とは、マイクロエレクトロニクス、マイクロ加工技術、マイクロ機構等の融合で生まれた新しい技術である。代表的なマイクロメカトロニクス機器として、(1)機械量検出センサ(ジャイロセンサ、加速度センサ、圧力センサ等)とそれらを用いたシステム(エアバッグ駆動システム、カメラ手ぶれ補正、マイクロ血圧モニタ等)、(2)光学デバイスとそれらを用いたシステム(ディスプレイ、スキャナ、マイクロカメラ用レンズ駆動等)、(3)流体制御デバイス(ポンプ、バルブ等)とそれらを用いたシステム[micro total analysis systems(μTAS)、マイクロリアクタ、マイクロ燃料電池等]、(4)プローブ精密位置決め機構とその応用システム(プローブ顕微鏡等)、(5)情報機器用マイクロメカニズム(マイクロディスク、ヘッド駆動機構等)、(6)マイクロロボット(医療用マイクロロボット、狭隘箇所点検マイクロロボット等)等が挙げられる。このように、マイクロメカトロニクスの応用範囲は極めて多岐に及ぶが、その実現のための基盤技術は、マイクロセンサ、マイクロアクチュエータ、マイクロ電池といったデバイス技術、マイクロ機構の設計と加工技術、これらを統合するシステム設計技術等、かなり共通する部分が多い。ここでは、特にマイクロロボットを中心に取り上げ、これまでの開発例を紹介する。a. 管内検査マイクロロボット(micro robot)原子力発電所や化学プラント内の配管事故防止のため、管内部検査の重要性が高まっている。一般には、配管外部からの超音波検査や人手による分解検査、工業用内視鏡を用いた検査が行われているが、検査に時間がかかったり、曲がり管が多く挿入距離が長い場合には内視鏡がうまく挿入できない場合がある。そのため、管内を自走して内面を非分解で点検できる配管内検査ロボットが各種開発されてきたが、従来のメカトロニクス技術では、1~2インチ管といった小径管を対象としたものは実現できなかった。しかし近年、マイクロモータ(micro motor)、マイクロカメラ、マイクロ減速機(micro reduc-tion gear)などの要素技術が開発され、小口径管用の検査ロボットが実現できるようになった。図6・60,6・61に、1インチ管を対象として開発された管内点検ロボットの例を示す。ロボットは直径23mm、長さ110mm、質量16g カメラを搭載したワイヤレスの内視鏡で、薬のように飲み込むことで、食道から胃、小腸、大腸の内壁を撮影して外部に画像を送信することができる。飲み込んだ後は、被検者の苦痛がほとんどないうえ、小腸等、従来の内視鏡では検査できなかった箇所の検査も行えることから、大きな期待がもたれている新しい医療マイクロメカトロニクス機器の一つである。図6・66,6・67に示すカプセル内視鏡は、外径9mm、長さ23mmのピル型の外形をもっており、内部にCCD カメラのほか、照明用のLED、ピント調整用のマグネットコイル、姿勢制御用のロータコイル、外部から電力を受けるための電力受磁コイル、撮影画像を外部に転送するためのマイクロ波ビデオトランスミッタ、蓄電コンデンサ等を内蔵している。また、バルブやタンクの搭載により薬液投与や採取生検も可能である。被検者はコイルを内蔵したベストを着用し、このコイルを通して体内のカプセル内視鏡からの画像データ受信、およびカプセル内視鏡への電力送信を行うことができる。図6・68 電子写真プロセス図6・67 カプセル内視鏡の構成[提供:(株)アールエフ]カプセル内視鏡が必要とする電力は、コイルを通した電力電送によってまかなわれる。バッテリを搭載していないため、万一、事故でカプセル内視鏡が体内にとどまっても危険性が少ない。また、CMOS 撮像素子ではなくCCD を用いているため、色再現性が高く、ダイナミックレンジの深い画質が得られる。41万画素のCCDを用い、30フレーム/sの動画を体外に送ることができる。ピント調整機構にはボイスコイルモータが用いられ、体外のモニタを見ながらレンズを駆動してピントを合わせることができる。また、必要な部位を撮影できるように、カプセル内視鏡の姿勢を外部から制御する機能も実現できている。図6・67に示すように、カプセル内には三つの姿勢制御用ロータコイルA,B,Cが120°間隔で配置されており、これが通常のモータでいうロータに相当し、ベストに埋め込んだ三つのコイルがステータとして機能することにより動作する。体外から伝送された電力をカプセル内視鏡内のコンデンサにいったん蓄積し、これをステータコイルと同期して瞬時に流すことにより大きな磁力を発生させ、カプセル内視鏡が動作する。カプセル内視鏡の位置と傾きはロータコイルとステータコイルに流れる電流を体外で監視することで、位置に関しては5mm以内、傾きに関しては15°以内の精度で制御することができる。
