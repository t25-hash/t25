# 7・2・2 衛星構造の設計手法

> **章**: 第7章 構造力学
> **節**: 7・2 構造設計の手法

衛星構造（spacecraft structure）の設計は，ミッション要求に
対してソリューションを提供する総合衛星システム設計と密接な関
係があり，衛星機械系システムの要となるものである．各種ミッシ
ョン要求や打上げロケットからの制約条件を吟味し，機器配置を含
めた衛星コンフィギュレーション設計の中で，基本的な衛星構造の
形状と構造様式を決定する．ミッション機器や搭載推進薬にできる
だけ多くの質量を配分するため，衛星構造は最軽量で高強度，高剛
性が要求される．
衛星構造の主要な機能は，以下のとおりである．
(１) すべてのミッション期間を通じて，搭載機器を構造的に安全
に保持すること（搭載機器保持機能)．
(２) ロケット打上げ時の荷重に耐える強度，剛性を保有すること
（強度，剛性保持機能)．
(３) 地上における製造，組立，輸送，試験，射場整備の段階か
ら，打上げ，軌道上全ミッション期間を通じて，搭載機器に
対して安全な機械環境を提供すること（機械環境維持機能)．
(４) 光学センサの視野，取付精度，アライメント等の幾何学的安
定性を確保すること（幾何学的安定性確保機能)．
これらの機能要求を満足するため，衛星構造の設計手法を以下の
とおり設定している．
(１)設計条件の設定（荷重条件，剛性要求，その他）
(２)材料部材の強度
(３)構造設計，解析
(４)試験評価
以下に詳細を述べる．
a.設計条件の設定 衛星構造設計の荷重条件は，次のフェー
ズについて規定される．
(１)製造（保管，輸送を含む）段階
(２)試験段階
(３)打上げ段階
(４)軌道上での運用段階
この中で地上における製造段階での荷重が構造設計を支配するこ
とのないように，製造，取扱い，輸送，組立および保管時の環境を
配慮する．また軌道上では，温度変化による熱サイクルを受ける
が，熱応力が問題となるアンテナ等の特殊な機器を除けば，衛星主
構造は温度が安定しており，クリティカルな荷重条件とはならな
い．したがって，衛星構造に対する荷重条件はロケットによる打上
げ段階の荷重と，それを地上で模擬する試験時の荷重が最も厳しい
ものである．以下，具体的な荷重条件を示す．
.荷重条件（loads condition）
(１)準静的加速度荷重（quasi - static acceleration）
衛星主構造の評定荷重となるもので，H-ⅡAの場合，打上げ時，
第1段主エンジン停止直前，第1段主エンジン停止過渡時がクリテ
ィカルイベントである．加速度荷重は，定常的な加速度（静的加速
度）と低周波トランジェントの加速度（動的加速度）を合成し，準
静的加速度荷重条件として，剛性基準を満足する衛星に対して規定
されている．
(２)動的環境条件
主として衛星の二次構造部材，搭載機器，アンテナ，太陽電池パ
ドル等の耐振動要求のベースとなるもので，次の項目で規定されて
いる．
(ａ)正弦波振動（sinusoidal vibration) ロケット打上げ時
の低周波振動を包絡したレベルが衛星分離面で規定されている．振
動試験時においては，ロケット打上げ時と境界条件が異なるため，
特に衛星の主共振点において衛星内部に過大な荷重が加わる可能性
がある．これを回避するため，ロケットとの結合荷重解析（CLA:
coupled loads analysis ）で打上げ時に加わる衛星内部の最大加速
度，荷重を求め，試験時の応答がこのフライト予測を超えないよう
に，入力レベルを制限（ノッチング）する方法がとられる．
M-Ⅴロケットにおいては，ランダム振動で低周波振動もカバー
している．
(ｂ)ランダム振動（random vibration) 主として音響によ
る衛星分離部の応答を包絡したレベルが規定されている．衛星の大
型化に伴って，衛星下端入力のランダム振動では衛星上部まで適切
に励振できないため，高周波振動に対しては音響だけで衛星を開発
する傾向にある．
(ｃ)音響（acoustics ) おもにリフトオフ時と遷音速飛行時
に生じる高周波振動環境で，衛星外部に一様に作用するため，外部
に露出する大面積のアンテナ，太陽電池パネル等にとってクリティ
カルな環境であり，また搭載機器のランダム振動環境のベースとも
なる．
(ｄ)衝撃（shock) パイロ衝撃として，衛星分離時および
太陽電池パドル等分離機構のリリース時に対して規定されている．
パイロ衝撃は，ピーク加速度は高いが持続時間が短く，発振源から
離れるに従って急速に減衰するので，通常局所的環境と考える．
M-Ⅴロケットにおいては，パイロ衝撃ではなく，むしろ打上げ
時の過渡的荷重をシミュレートしている．

.その他
(１) 衛星システム設計との適合性
電気設計や熱設計等からの構造部材の材料選定，配置等に対する
制約に適合する構造設計を行う．また，搭載機器を適切な環境条件
のもとに支持または収納できること．
(２) 信頼度
構造系の信頼度向上のため，構造はできるだけシンプルな構成と
し，強度・剛性上の機械的特性や，ロードパス等が正確に把握可能
なコンセプトを採用すること．
(３) 製造性
製造は，品質を確保するため，認定済みの確立された製造方法に
よるものとし，設計と製造の適合性について考慮すること．
(４) アクセス性
構造設計に際しては，衛星の取扱い，輸送，組立・分解，補修，
および検査時の環境を考慮し，それらが安全かつ容易にできるため
の手段を講じること．

考慮した設計許容値（例えば近似的な3 σ下限値）を設定し，構造
設計に使用する．

c.構造設計，解析 衛星構造の設計・解析は，一般的に図7･
5に示す作業フローで実施される．衛星の機械系システム設計にお
いて，搭載機器のレイアウトやロケットインタフェースを考慮し
て，打上げ時および軌道上の衛星コンフィギュレーションを設定す
る．この過程において，衛星構造の寸法，衛星構造計画を立案す
る．次に，衛星全体の強度・剛性，熱設計を考慮し，具体的構造部
材の選定，サイジング等を行い，構造様式を決定する．一次設計が
できると，図7･5のフローのとおり，衛星レベルの数学モデルを作
成して構造解析を行う．有限要素法（FEM:finite element
method）による構造解析プログラムで，動的応答解析（固有振動
数，振動モード），静的解析（準静的加速度応力解析），熱ひずみ解
析等を行う．これにより強度・剛性・クリアランス評価等を行い，
a項の設計要求条件を満足しているかどうかを確認し，不十分な場
合は部材諸元の再設定，構造部材の再選定作業を繰返し，軽量化を
図りつつ，要求条件に対して最適化された衛星構造を設計する．
一般に，大型衛星の開発においては，衛星全体の横剛性が設計標
定になるケースが多く，比剛性の高い材料を選択し，曲げ方向の断
面二次モーメントを大きく設計することが多い．構造の強度要求に
対しては，比強度の大きい材料を選定するほうが望ましいが，断面
諸元（肉厚等）を修正してローカルに強度不足を補うことが可能で
あり，比較的容易に対応可能である．また，打上げ時の荷重に対し
ては，部材の引張強度よりも，圧縮強度や座屈（buckling）など
の形状安定に対する強度がクリティカルとなるケースが多い．
衛星の骨格を構成し，所要の強度・剛性を確保する基本構造を一
次構造という．衛星は一次構造により打上げ時の荷重をロケット側
に伝達し，衛星全体の剛性を確保する主構造を構成する．一方，ア
ンテナや太陽電池パドル，タンク支持トラス構造や機器取付けブラ
ケット等，二次的な荷重を分担する構造を二次構造という．
衛星の一次構造の構造様式には，トラス等の骨組を主構造に採用
したトラス構造様式（図7･6），中央にセンタシリンダを採用し主
要ロードパスを構成するシリンダ構造様式（図7･7），ハニカムパ
ネル（honeycomb panel ）に搭載機器を実装しパネルのみで荷重
を支持するパネル構造様式（図7･8）がある．
