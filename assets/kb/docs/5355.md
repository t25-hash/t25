# 7・3・3 軌道上での振動制御

> **章**: 第7章 構造力学
> **節**: 7・3 構造動力学

a.制御系と柔軟構造物の干渉 衛星がロケットから分離され
b.柔軟構造物のダイナミクスの同定 適切な制御系の設計の
ためには柔軟構造物の振動特性が事前にわかっていなければならな
いが，そのような柔軟な構造を地上で試験することは重力と大気の
問題で困難である．部分構造の試験により数学モデルを作り上げて
いく方法はあるが，軌道上で同定し，それを反映して制御系パラメ
ータの修正をコマンドで送ってやるのが最も現実的である ．た
だし，そのためにはある程度地上で同定できていることが条件であ
る．
