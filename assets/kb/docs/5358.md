知能化の分類本項では、このようなマイコン化の歴史を踏まえ、現在の家電での特徴的なマイコン利用用途として以下の3種の機能実現による知能化を扱う。(1) システム的機能実現による知能化:センサと機器要素をマイコン制御で連携し機能実現をする。(2) 機器性能向上の機能実現による知能化:従来機械的に実現させた機能を、高速マイコンでのソフトウェア処理により機能実現。(3) 通信利用によるマイコン連携制御での知能化:機器内部での連携と機器外部とのネットワーク化による機器どうしの連携による機能実現。家電機器を例にとり、各知能化が融合して使われている状況を解説する。図6・147 冷蔵庫でのシステムブロック図7―237c. 知能化の実際例1:冷蔵庫(refrigerator)初期の機械式制御から、センサ値をディジタル化して所定の目的を実現させるプログラムにより制御を実現する中で、複数のセンサを利用したシステム的機能実現の例を示す。冷蔵庫のシステム図を図6・147に示す。センサとしては、各室空気温度センサ、冷媒温度センサなどがあり、各部に取付けられ入力値となり、メインコントロール部のマイコンへ入る。メインコントロール部では、庫内負荷を算出し、冷却用アクチュエータ機器(各ファン)を制御する。電力量モニタ部からの冷媒漏れ判定をして問題がなければ、冷蔵庫制御部が負荷に合わせたコンプレッサの回転数指令をコンプレッサ回転制御部(DSP制御部)へ出す。DSP制御部は、回転数指令にのっとりコンプレッサモータの回転を制御する。従来は単純なインバータであったが、最近では電流検出を行うことで回転中のトルクに合わせた電力最小電流値を算出する省エネルギーモータ回転制御であるベクトル制御も実施して、省エネルギー化を進めている。ベクトル制御(FOC: field oriented control)では、検出した電流値とコンプレッサへ出力する電圧値を用いて瞬時電力を算出することができる。この瞬時電力値の変化量により、扉開閉などで冷蔵庫の負荷が増したことを検出した場合は、コンプレッサと冷却ファンの回転数制御およびプラズマユニットを動作させてパワフルな冷却と脱臭を行い、鮮蔵制御する。また、同様に電力変化をモニタすることにより、冷媒漏れを検出する。モニタにより急激な電力変化が認められた場合、冷媒漏れかどうかを冷媒漏れ判定部が検証し、冷媒漏れと判定した場合は、安全対応システムを作動させる。冷蔵庫の扉を開けると開閉センサがあり、開値をマイコンが認識すると内部照明をつけたり、内部扉部分の気流ファン風量を制御したりというように、きめ細かな対応ができるようになった。また、内部に設置したにおいセンサにより、においを検知すると脱臭器のプラズマユニットをオン制御したり、従来に比べ野菜が長期間保存できるようになるといったシステム的機能提供によるメリットが追加されている。これらの機能などは、従来の機械式ではできないマイコン化の恩恵と考えられる。機械性能の向上機能実現での知能化の例としては、単独機器内での向上として、最新の機器においては家電としての機能の強力化かつ省エネルギー化を実現するために、モータ制御と機器制御をそれぞれ独立に分担したCPU により制御することを実施している。特に冷蔵庫では、DCモータをベクトル制御することにより省エネルギーと性能向上を実現している(ノンフロンthe鮮蔵庫GR-NF415G)。ベクトル制御を搭載した冷蔵庫のシステム概要を図よりコンプレッサモータを高効率で駆動することで、消費電力の低7―238メカトロニクス・ロボティクス減を実現した。また、DSPで瞬時電力(コンプレッサ入力)変化量を演算により求め、冷却、除菌・脱臭の鮮蔵制御を行うとともにノンフロン冷媒の漏れも検出する。ベクトル制御は、モータ電流値(三相)を検出し、この電流値をモータのロータとともに回転する直交座標系に変換(変換)して、トルクに寄与する成分욤としない成分욃とに分けて制御する方式である。モータ固有のトルク特性方程式を用い、必要トルクを得るための電流値を最小に制御することで、コンプレッサモータの高効率駆動が可能になり、従来の矩形波方式に対し、より省エネルギー化を図ることができる。このベクトル制御をDSPという高速演算機を用いて構成し、あらゆる負荷条件に対して瞬時に対応できるシステムとなっている。d. 知能化の実際例2:洗濯機(laundry machines)洗濯機は、1977年にマイコンを搭載した全自動洗濯機が初めて登場し、1991年には洗濯量、汚れ具合に合わせ洗濯の仕方を細かく調整する制御としてニューロアンドファジィ理論を利用した制御の全自動洗濯機が導入され、本格的ディジタル処理による知能化時代が到来した(システム的機能実現による知能化)。その後、日本の家屋事情を反映し、静音と省エネルギー(energy conservation)の実現として個別部品の知能化が進み、DD(direct drive、直接駆動)モータ化とその最適回転制御として本洗乾一体型ドラム洗濯機では、c項で前述したDSPによるベクトル制御が同様に導入されている。DD モータ(図6・149)を洗濯機に搭載したときのメリットとして、まず静粛性と制御性の向上が挙げられる。静粛性については、従来方式ではベルトとギヤ機構によって脱水や洗いの回転数まで減速駆動され、また脱水停止時にはバンドブレーキで機械的に水槽に回転制動をかけていたため、ギヤ音や摩擦音が発生していた。DDモータ方式ではこれらの機構部分の騒音源をなくし、大幅な静音化を達成している。1998年から全自動洗濯機、2000年からはドラム式洗濯乾燥機に相次いで搭載され、進化を続けており、機構部分の薄型化によるドラム容積の拡大と本体サイズの小型化に貢献している。このDD モータは、全周24極の磁極をもつため、1/24回転の分解能でドラムの位置および回転むらの検知を行うことができる。これにより、前述のベクトル制御を有効に生かす回転センシングと、きめ細かな制御が可能となった。洗濯機においては、システム機能の知能化として、各種センサ値の利用が挙げられる。まずは高精度重量センサの利用である。最近の洗濯機は、水位や洗浄時限の設定や洗濯工程の最適化などを行うため、洗濯開始時に布重量を検知している(重量センサ)。全自動洗式はパルセータ(水槽底面のかくはん翼)の上にある布の回転抵抗を負荷として検知するため、DD モータ搭載機では、回転抵抗をモータの電流値変化により把握することで10段階以上の分解能で布量を識別できるので、これをベースに、全自動洗濯式では布負荷図6・148 冷蔵庫でのDSPインバータブロック図(PID :proportional-integral-derivative,PWM :pulse width 知能化の実際例3:エアコンディショナ(空気調和機、airconditioner)通信による知能化の例としてエアコンディショナを示す。日本で一般的であるスプリットタイプエアコンディショナは、室外機と室内機に独立しており、それぞれの制御部には独立したマイコンをもち、室内・室外機間は通信によって連携するように制御されている。また、無線リモートコントローラにもマイコンをもち、室内機本体とリモートコントローラとの通信によりコントロールを実現している。図6・151にエアコンディショナの全体制御システム構成を示す。室内機制御メイン基板では、室内機としてのセンサと動かすべきルーバ等、室内ファンなど状況に合わせた温度調節制御を実施している。室外機温度など制御上の必要情報は、室内機へ通信伝達される。室内機が指定した要求回転数指示を室外機マイコンへ通信することにより、室外機マイコンでは回転波形生成指示をDSPマイコンへ伝達し、コンプレッサ回転を制御する。室外機マイコンは室外機各冷媒温度やコンプレッサ温度をセンサ値として把握してサイクルの最適化を図り、コンプレッサの保護制御を実施している。リモートコントローラのマイコンからの情報を赤外線通信で受光部を通じ通信取得することで、人の意思に合わせた動作をするようになっている。現在は、さらに外部インタフェースとしてIT 端子と記載したように、マイコン情報を設備系通信規格ECHONETに準拠した外部通信プロトコルへも対応したエアコンディショナが普及し始めた。f.
