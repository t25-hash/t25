# 7・5・1 全般

> **章**: 第7章 構造力学
> **節**: 7・5 H-/ H-Aロケットの構造

H-Ⅱ/ H-ⅡAロケットは実用衛星を打ち上げるために宇宙開発
事業団（現：宇宙航空研究開発機構，JAXA）が開発した2段式液
体ロケットである．実用衛星打上げ用のロケットはN-Ⅰ，N-Ⅱ，
H-Ⅰロケットと順次大型化し国産化率を高めて，H-Ⅱロケット
（H-Ⅱ launch vehicle）で純国産となった．H-Ⅱロケットの開発
理念は，静止衛星2トン級の打上げ能力をもつロケットを全段自主
技術により開発する，第1段機体を最重点項目とし第2段機体は
H-Ⅰロケットの第2段を活用して大型化することにより開発項目
を抑える，などであった．2006年9月現在，H-Ⅱロケットのシス
テムを簡素化，高信頼性化したH-ⅡAロケット（H-ⅡA launch
vehicle）を開発して9機の打上げに成功している．H-ⅡAロケッ
トの開発理念は，静止衛星2トンから3トン級の多様なミッション
に対応する，H-Ⅱの開発成果に基づきシステムを簡素化すること
により打上げコストの低減を図る，などである．
