通信の終了(解放)一方の電話機の受話器が下ろされるなど、通話の終了を検出した加入者交換機は、通話が終了したことを伝える呼制御信号を各交換機に送出し、設定された通話路を解放する。また、発信側の加入者交換機は、課金を終了する。b. アナログ電話通信方式。 音声信号帯域電話網では音声の特徴を考慮して、伝送すべき帯域などを規定している。人間の可聴帯域は一般に、20Hz~20kHz であるが、音声通信品質の評価結果から、会話を行うためにはおおむね4kHz 以下の帯域が伝達されれば十分である。標準的な電話機であるNTT の600形電話機の場合、機械音響系などを総合して、約300Hz~3.5kHz の音声信号を通話信号として送受信する能力をもつ。. フッキング信号通話中に、交換機に信号を送出する方式として、短時間、ループを切断するフッキング信号が用いられる。切断や誤操作による一時的なフッキングと識別するため、フッキング信号を用いる付加サービスを利用している加入者に対しては、最小値(0.1s)および最大値(2.2s)が規定され、最小値以下は誤操作による一時的なフッキング、最大値以上は切断、その中間がフッキング信号と判断される。c. 電話通信に対する要求条件電話網は、エンド・トゥ・エンドで接続し、安定した通信サービスを実現するため、さまざまな要求条件を満たすように構築されている。ここでは、おもに品質に関わる要求条件について述べる。電話網における通信品質としては、通話品質、接続品質、安定品質などが規定されている。. 通話品質(transmission performance)通話品質とは、通話中の通話のしやすさ、わかりやすさ、正確さなどを表すもので、音声情報の減衰量、ひずみ、雑音、エコー、遅延などの値で規定される。これらは、途中の伝送路の品質や経路長、中継する交換機の段数などにより定まる。(1) ラウドネス定格電話網の通信品質はラウドネス定格(LR : loudness rating)により規定される。ラウドネス定格は音の大きさを評価する尺度の一つで、原理的には網の特性に近い中間系(IRS : intermediate ref-erence system)と、被測定系の音量を比較することにより求めるものであるが、被測定系の感度周波数特性から計算により求めることも可能である。事業用電気通信設備規則(1985年郵政省令第30号)では、アナログ電話端末相互間の総合ラウドネス定格(OLR :overall LR)は21dB 以下と規定される。OLR は、送話ラウドネス定格(SLR : sending LR)、受話ラウドネス定格(RLR :receiving LR)、接続ラウドネス定格(JLR : junction LR)の和で表される。これらの関係を、図1・5に示す。(2) その他の通話品質(参考)通話品質として、ラウドネス定格以外に以下のような品質が参考的な値として用いられる。8―3図1・5 総合ラウドネス定格図1・6 ハイブリッド回路におけるエコーの発生(a) 伝送損失通信網の各区間における、伝送信号の損失量で規定される。(b) 減衰ひずみ加入者回線の周波数特性や各種フィルタなどの特性により、周波数によって減衰量が変動する。これを減衰ひずみという。(c) 雑音雑音としては、無通話時に回線に混入する無通話雑音、交換機におけるディジタル化の際の量子化雑音、パルス雑音、商用電源などからの誘導雑音などがある。(d) 群遅延ひずみ変復調時のフィルタなどの特性により、周波数ごとに遅延時間が変動する。これを群遅延ひずみという。群遅延ひずみは、減衰ひずみと合わせて、音声信号の周波数ごとの波形のひずみや位相のずれとなり、音声信号上のデータ通信品質や最高伝送速度などに影響を与える。(e) 伝搬遅延伝搬遅延は、双方向の会話を行う際に、会話のタイミングのずれという形で主観品質に影響を与え、長距離通信や衛星通信回線を用いる場合に問題となる。(f) エコー(反響)2線式の加入者回線では、音声信号は上り/下りが重畳されて伝送されているが、交換機入口でハイブリッド回路により上り/下りの信号が分離され、おのおの交換処理が行われる。その際、インピーダンスの不整合により、一部の音声信号が逆方向に流れエコーとなる。伝搬遅延時間が大きい場合、エコーは主観品質に影響を与えるため、エコーキャンセラによりエコーを除去する処理が行われる。図1・6にエコーの発生例を示す。. 接続品質(call processing performance)接続品質とは、発信時の接続のしやすさなどを表すものである。着信端末が通話中でないにもかかわらず通信網のリソース不足などにより通話できない(呼損)確率や、受話器を上げたときに交換機が検出してダイヤル音を送出しダイヤル信号を受信可能となる時間、複数の交換機を介して着信側の交換機が呼出しを開始したことを発信側の端末に伝える時間などが規定されている。これらは、交換機の処理能力、交換機の容量(一つの交換機内で同時に接続できる回線数)、中継交換機の段数、交換機間の回線数などにより定まるもので、一定品質を満足し、かつ最小の設備で通信網を構築することが要求される。事業用電気通信設備規則で規定されるおもな接続品質を表。
