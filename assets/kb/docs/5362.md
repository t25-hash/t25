# 7・6・1 概要

> **章**: 第7章 構造力学
> **節**: 7・6 M- ロケットの構造

M- Vロケットのおもな構造要素を，1号機を例として図7･34に
示す．本ロケットは3段式の全段固体燃料ロケットであり，キック
モータはオプションである．以下に，主な各部構造の特徴について
述べる．
口元周りの層間せん断応力の緩和を図っている．
なお，5号機以降はさらなる打上げ性能向上と低コスト化を目指
して機体の改良が実施され，構造としては，第2段モータケースの
CFRP化とそれに伴う1・2段接手および2・3段接手の設計変更
が行われている．
技術を適用し，高性能化を達成している．
