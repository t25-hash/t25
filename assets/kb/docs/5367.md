# 1・3・1 インタネットとは

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・3 インタネット

インタネット(internet)とは、インタネットプロトコル(IP:internet protocol)を用いて多種多様なコンピュータを相互接続することによりさまざまなコンピュータ資源やサービスを共用するネットワーク基盤である。インタネットは世界に広く分布するコンピュータ資源の利用を可能とし、現在では単なる物理的な通信ネットワークという枠を超えて、そのうえでさまざまな知識、情報ベースが形成され、また物理的位置の制約を克服する協調作業環境としての世界最大の情報基盤となっている。インタネットはアメリカ国防総省の実験用コンピュータネットワーク(ARPANET)をその起源としている。これは、戦時下における攻撃破壊にあっても確実な通信の実現を目的としたものであり、この結果、コネクションを必要とせず、柔軟な経路制御が可能な通信プロトコルであるインタネットプロトコル(IP)が開発された。さらに、IPの上位レイヤにTCP(transmission controlprotocol)を設けることによって、信頼性の高い通信が容易に実現できるようになった。1980年代に入ると、IP技術は、NSFNet等のアメリカ政府が所有する他のコンピュータネットワークだけではなく、それらとは無関係な商用コンピュータにも実装されるようになった。特に、低価格化の進みつつあったワークステーション上のバークレー版UNIX にTCP/IP(transmission control protocol/internet pro-tocol)通信プロトコルとそれをアプリケーションから容易に利用できるインタフェースが標準搭載されたことにより、大学、研究機関を中心に爆発的に普及した。この結果、研究用ネットワーク間の相互接続が急激に進んで、現在のインタネットの母体が誕生したのである。その後、TCP/IPはスーパーコンピュータやマイクロコンピュータにも搭載されるようになり、マルチベンダコンピュータネットワークの事実上の標準として利用されることになった。現在では、既存のIP(IPv4)の次世代版であり、IPv4よりもはるかに広大なアドレス空間をもつIPv6の導入も盛んになりつつあり、IP技術の普及をさらに促進するものとなっている。
