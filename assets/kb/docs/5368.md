# 1・3・2 パケット交換機としてのルータ

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・3 インタネット

インタネットは,複数のネットワークをIPルータ[以下ではルータ(router)と呼ぶ]というIPパケット用の交換機により相互接続した形態をとる.この形態は物理層(イーサネット,FDDI,専用線等)に依存しない.また,ネットワークが構内に閉じたもの(LAN : local area network)や私設網であっても,インタネット通信業者が提供する公衆網であっても基本的に変わらない.このように,インタネットは階層のないフラットな接続関係をもち,個々のネットワークは管理主体の異なる組織が運用している.すなわち,インタネットは基本的に分散管理のシステムであり,IPという共通の技術を用いつつ自律的にネットワークが変化していくところが,従来の通信網の構成と大きく異なる.インタネットの大きな特徴は,この階層性のなさ,言い換えると一つの技術(インタネットプロトコル)によるあらゆるネットワークの接続にある.また,インタネットは,個別に管理されている組織(サービスプロバイダ,企業ネットワーク,学術ネットワーク等)や地域単位のネットワークの集合体という形をとるが,遠距離にわたるトラフィックを効率的に運ぶために,通常,高速のバックボーンネットワークで結ばれた構成となっており,これもルータによる相互接続によって実現されている.以上より,ルータによるIPパケット配送がインタネットの本質であるととらえることができる.
