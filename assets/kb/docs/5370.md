# 1・4・1 移動通信の概要

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・4 移動通信

現在の移動通信システムには、第一世代、第二世代、および第三世代セルラシステム、PHS(Personal Handyphone System)、無線呼出システム、コードレス電話、航空・列車電話、高度道路交通システム(ITS)さらに業務用・個人用移動通信システムなどさまざまな方式が存在する。通信に用いられる無線周波数は、相互の干渉・混信を防ぐためあらかじめ配置が厳格に決定されている。表を示す。この周波数配置(frequency allocation)が、最もたいせつな基本である。利用されている周波数帯は広く、MF帯からEHF帯にわたる。移動(無線)通信の歴史はより高い周波数帯域の開拓の歴史ともいわれ、低い周波数帯域が利用されて飽和状態になると、次に高い周波数帯域の開拓(技術開発)が行われ、それが繰返される構造となっている。その結果、低い周波数帯には昔からのシステムが、高い周波数帯には新しいシステムが割当てられている。表1・12において800MHz帯から準マイクロ波帯にかけて、最近の移動通信システム(携帯・自動車電話、PHS,IMT-2000)が割当てられている。ここより低い周波数帯はすでに満杯状態であり、逆に高い周波数帯域は技術開発により将来の移動通信に使われる可能性をもつ。移動通信の歴史を、技術に着目して簡単に振り返ってみる。実は、移動通信の歴史は無線通信と同じくらい古い。無線通信は1895年のマルコーニの無線電信機の発明から始まった。そして、1899年にはすでに、マルコーニが起こした会社(Marconi’s Wire-less Telegraph Company)により船舶通信用の電信装置が商用化されていた。電信から始まった移動通信が、電話の機能をもつようになったのは第一次世界大戦のころからである。ただし、利用された周波数帯は中波(MF帯)や短波(HF帯)であり、変調もAM方式のため効率が悪かった。1946年には、150MHz帯を用いた自動車電話(大ゾーン方式)がアメリカで開始された。大ゾーン方式は、一つの基地局で都市の全域(現在のシステムと比較するとかなり広域)をカバーするシステムであるが、広域をカバーするために欠点をもっていた。ユーザ容量が小さく、かつ大きな送信電力が必要となってしまう点である。大ゾーン方式の欠点を克服できる本格的な自動車電話が小ゾーン(第一世代セルラシステム)方式である。小さい無線エリアを細胞のようにすきまなく詰めて配置しサービスエリアをカバーしているさまから、セルラシステムとも呼ばれる。小ゾーン方式の自動車電話は1979年に、世界に先駆けて日本で800MHz帯を用いてサービスが開始された。小ゾーン方式はその後、1981年に450MHzを用いてスウェーデンで、1982年には800MHzを用いてアメリカでサービスが開始された、これが、セルラシステムの幕開けである。なお、船舶電話は1953年、列車電話が1957年、無線呼出が1963年に、それぞれ日本で商用化された。以上、移動通信の歴史は100年以上になる。しかし、社会に広く普及が始まったのは最近である。最初に大きく普及した公衆サービスが無線呼出(ポケットベルなど)である。1980年代に急速にユーザ数を伸ばし、1980年代末には日本で1000万以上のユーザを抱える大きなサービスに成長した。無線呼出は一般的に、単一の県または数個の隣接する県をサービスエリアとし、サービスエリアは一つの無線呼出中央局と複数(10~70程度)の無線呼出基地局から8―12情報・メディア機器表1・12移動通信で用いられる周波数帯とおもな用途高周波数帯おもな利用周波数移動通信におけるおもな用途EHF(ミリ波)帯:30~300GHz60.76GHz帯など特定小電力無線局ミリ波レーダなど準ミリ波帯19~20GHz帯など無線LAN
