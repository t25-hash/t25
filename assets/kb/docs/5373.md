# 1・4・4 移動通信の将来

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・12 に,移動通信システムとそこで用いられている周波数帯と

図1・33に,将来の移動通信(新世代移動通信)の領域について示す.横軸は通信速度,縦軸は利用環境(移動する速度)である.2010年ごろ以降の移動通信には,第四世代と呼ばれるセルラ環境でのサービスと,高速移動無線アクセスと呼ばれる構内やスポットエリアを対象としたサービスとが,ともに重要な要素と考えられている.さらに,ユーザの利便性の観点からは,これらのサービスのシームレス化(seamless correctivity)が技術課題となる.シームレス化とは,ユーザにどのサービスを利用しているかを意識させることなく,通信も連続性を保ち,品質も安定していることである.図1・34には,将来の移動通信サービスのイメージを示す.利用する環境やスペースにおいて,それぞれに適した移動通信方式が提供される.衛星を用いた広いエリアを対象とする移動通信方式から,オフィスやパーソナルスペースなどの狭いエリアを対象とする方式までさまざまである.これらの方式はシームレス化により,相互切換えなどをユーザが意識することなく,連続的に利用できる点が特徴である.このシームレス化は,固定のネットワークやプライベートのネットワークとの間でも実現することが重要である.
