衛星通信の概要衛星通信は電波を利用して情報を伝達する無線通信の一種として発展してきた.しかしながら,従来の地上無線通信に比べて際立って大きな特徴をもっており,宇宙開発の急速な発展に伴って,最近では独自の位置を占めるようになっている.衛星通信は,広義には人工衛星(satellite)と他の地点または物体とを結ぶ通信を指すといえるが,一般的には人工衛星を中継点として地球上の複数地点間に無線回線を設定して通信を行うことを指す.最近では人工衛星間の通信も重要になりつつある.無線通信では電波の経路に障害がないように,高い塔や山上に中図1・33新世代移動通信システムの領域(2010年)[出典:情報通信技術審議会答申]図1・34 利用目的別の分類衛星通信サービスは国際電気通信条約でさらに,固定衛星通信業務(FSS : fixed satellite service),移動衛星通信業務(MSS : mobile satellite service),放送衛星業務(BSS : broadcasting satellite service)の3種類に分類されている.固定衛星通信業務は地球上に設置した地球局(earth station)と衛星が情報の授受を行うもので,電話,画像,データなどの伝送を目的として国際,国内の通信に現在最も多く使われている.地球局は大形のアンテナを備えて大容量,高速の通信を行うものから,非常災害時に運搬して迅速に通信回線を設定する小形で簡易なものまで多くの種類がある.移動衛星通信業務は船舶,航空機,自動車などに設置した地球局と衛星間で通信を行うもので,大形船舶に対するサービスから,漁船などの小形船舶,さらには自動車,航空機へと着実にその領域を広げている.放送衛星業務は衛星から多数の受信局に対して行われる一方向の通信で,受信装置を簡易なものにするために衛星は大きな送信電力を必要とする.放送の性格上,国あるいは比較的小さな地域を対象としたサービスが多い.使用できる周波数は,他の無線通信サービスと同様,混信防止のため国際電気通信条約により割当てられている.この中で,Cバンドと呼ばれる4GHz,6GHz 帯は早くから利用が進んだが,地上無線システムでも多用しているため,新しく衛星通信に用いることは困難になってきている.このため,Kuバンド(11GHz,14GHz 帯)やKaバンド(20GHz,30GHz 帯)の利用が進められている.10GHz 以上の周波数では電波が大気中を通過する際の降雨減衰(rain attenuation)が大きいので,電力を大きくしてマージンを確保したり,激しい降雨時には違った地球局をう(迂)回路に使ったりするダイバーシティ技術などが必要になる.日本は「さくら2号(CS-2)」によってKaバンドを,また「ゆり2号(BS-2)」による衛星放送を世界で最初に実用に供した国である.10GHz 以下の周波数としては,Lバンド(1.5GHz,1.6GHz 帯),S バンド(2.5GHz,2.6GHz 帯),X バンド(7GHz,8GHz 帯)の利用が進められており,電波伝ぱ時の損失が少ないLバンド,Sバンドは船舶,陸上移動,航空機などの移動体衛星通信に広く利用されている..
