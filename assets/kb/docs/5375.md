# 1・5・2 通信衛星の分類と特徴

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・5 衛星通信システム

a. 静止通信衛星衛星通信システムで用いる衛星は目的に応じて実験衛星,国際通信衛星,移動体通信衛星,軍事衛星,国内通信衛星,アマチュア衛星,科学衛星,データ中継衛星などに分類できる.なかには衛星間通信を行うものもあるが,ここでは表1・15に示すおもな4分類について述べる.実験衛星は,搭載機器や衛星バス,あるいは地上系も含めた衛星通信システムの検証や実証のために打上げられる.衛星通信の初期には実験衛星と実用衛星の区分が必ずしも明確ではなく,実験衛星として企画されたものを引き続き実用衛星として運用する例も見られたが,近年では,実験衛星と実用衛星は明確に分けられている.また,実験衛星は少なくなり,最初から実用衛星として開発される例(N-STAR,SuperBird,JCSAT など)が多い.. 実験衛星初期にはEcho,Telstar,Relay衛星などの低軌道衛星で,電波の中継による衛星通信の実用性を確認する実験が行われてきた.その後,静止衛星の開発とともに,多様化・高度化する技術の実証が求められ,実験衛星といえども衛星自体が大形化・複雑化する傾向にある.その中でアメリカは1966年から1974年にかけてATS(applications technology satellite)シリーズで通信衛星として要求される基本的な技術の宇宙実証を重ねてきた.わが国では,1970年代からETS(engineering test satellite)シリーズやCOMETS(communications and broadcast engineeringtest satellite)などで独自技術の蓄積を重ねてきた.. 国際通信衛星(international communication satellite)広域での通信に向いている衛星通信は国際通信に適している.このために国際機関であるINTELSAT が設立され,国際間の通信のための衛星が数多く開発されてきた.1965年に打上げられたIntel-sat Iから1997年に打上げられたIntelsat VIIIは,衛星の大形化・標準化・量産といった道に大きく貢献した.現在,アジア・太平洋上,インド洋上,大西洋上に20機以上の静止衛星を配置し,加盟国のほか多くの国が利用している.国際組織以外にもPanAmSatやアジアにおけるAsiaSatなど多くの国際通信衛星シ表1・15目的による通信衛星の分類(静止衛星)分類諸外国の衛星例わが国の衛星例Echo,Telstar,Relay(周回衛星)ETS-V,VI,VIIICOMETSATSOICETS(周回衛星)ACTSIntelsatpanAmSat IIシリーズや,同じく1996~1998年のInmarsat IIIシリーズなど多くの衛星が打上げられた.高速データ通信用ミニM型携帯移動地球局によるサービスなどが行われている.移動体衛星通信では可搬型地球局を使い,また通信周波数も低い(LバンドやS バンド)ために衛星搭載アンテナの大形化が欠かせない.このような観点から1995年と1996年に打上げられたMSAT は開口径約5mの軽量な折り畳み式アンテナが搭載されている.また2000年に打上げられたACeS(ASEAN cellular satellite system)のGaruda衛星やThuraya衛星では開口径約12mの展開メッシュアンテナが搭載されている.この寸法のアンテナを搭載して初めてハンドヘルド端末での衛星通信が可能になる.. 国内通信衛星(domestic communication satellite)国内の基幹通信回線や迂回通信回線に衛星通信を使うことを目的に,多くの国が静止通信衛星を保有するに至っている.わが国では1977年から1988年にかけてCS,CS-2,CS-3が打上げられ,また1995年と1996年にはN-STAR が打上げられた.また通信の自由化に伴い,JCSAT やSuperBirdといった静止衛星を調達・保有し,通信サービスを行う事業者が出現した.b. 非静止通信衛星(non-geostationary communication satel-lite)目的による分類とは別に手段(軌道)に応じた分類もできる.これが静止衛星・非静止衛星であり,非静止衛星は表1・16に示すように,軌道により中低高度軌道(250~1000km)・高高度軌道(1000~20000km)あるいは太陽同期軌道などさまざまな軌道に打上げられる.開発の初期には静止軌道(36000km)への投入や軌道維持の困難があるため低軌道衛星(aのi項で述べたEcho,Telstar,Relay衛星など)から実験が始まったが,その後の打上げ技術の進歩により通信衛星の主流は静止衛星へ移っていった.しかし近年では,多数の非静止衛星を使うIridiumやICOといった移動体衛星通信システムが開発されている.また,衛星インタネットを目標として開発が始まったTeledesicはKaバンドを用いた低軌道衛星通信システムである.これらは,多数の衛星が地球の周りを周回し,通信ビームの走査や回線切換えの技術を使ってシームレスな通信サービスを実現するものである.
