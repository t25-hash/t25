# 1・5・3 衛星本体の基本構成要素

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・5 衛星通信システム

通信衛星はミッション機器(mission equipment)である通信系と衛星バス(satellite bus)からなる。図1・36に例としてCS-3の構成と外観を示す。a. 電源系(electric power systems)電源系は太陽電池による一次電源系、バッテリによる二次電源系、そして電源制御回路によって構成される。太陽電池セルにはおもにSi,GaAs材料が用いられる。スピン安定型の衛星では円筒状の外壁に装着するのに対し、三軸安定型では展開したパドルに装着して太陽の方向をつねに向いているように回転駆動する。スピン安定の場合は太陽光を受ける面積が限られているため、Intelsat を超える衛星も打上げられており、このような大電力衛星では損失を抑えるために100V という高いバス電圧を採用している場合もある。b. テレメトリコマンド(telemetry and command)通信ミッション機器とは別にテレメトリコマンド用の通信回線を設定する。テレメトリは衛星の姿勢、機器の温度、動作状態などをおもにモニタし、コマンドは姿勢、軌道変更などの各種指令、運用中の機器の切換えなどを行う。地上運用の作業を軽減するとともに、より8―19安全な運用を行うためにオンボードデータ処理(onboard datahandling)の中で自律運用を行う例もある。c. 推進系(propulsion systems)および姿勢制御(attitudecontrol)衛星は打上げによって投入された軌道からミッションに必要な軌道までの飛行、ミッション軌道位置の保持、姿勢制御のために推進系をもつ。衛星に使用される推進系は打上げ機に対して二次推進系(secondary propulsion system)と呼ばれ、固体推進(solid propulsion)、ガスジェット(cold gas jet)、一液推進(mono-propellant)、二液推進(bi-propellant)、電気推進(elec-tric propulsion)がある。通信衛星の場合、固体推進は、ロケットによって投入された遷移軌道から静止円軌道へ遷移するためにアポジ点で噴射するアポジキックロケットに使用させることが多い。近年ではアポジキックロケットに二液推進エンジンを用いて複数回に分けて噴射する例もある。ガスジェットや一液推進はおもに軌道位置の保持と姿勢制御に用いられ、また最近では姿勢制御に電気推進を用いる例もある。電気推進の推力は数10mN とごく小さいが、比推力(specific impulse)が2000s以上と際立って大きいのが特徴である。姿勢制御の方式はスピン安定と三軸安定の形式に分類できる。スピン安定衛星はバイアスモーメンタム方式である。衛星全体をスピンさせるので、軸対称の形状をしており、CS-2,CS-3のような円筒形状の衛星が典型的な例である。通信衛星の場合はアンテナをつねに地球の方向に向けることが必要であるため、アンテナ部分だけ、あるいは通信機器を搭載したプラットホーム全体をモータによって逆回転(de-spin)させる。一方、三軸安定では、衛星の姿勢をジャイロ等の情報から把握し、ガスジェット、イオンエンジン等のスラスタで制御するゼロモーメンタム方式と、リアクションホイールの回転のみで姿勢を安定させるバイアスモーメンタム方式、そしてリアクションホイール、ジャイロ、スラスタという両方式の機器を備えるハイブリッド方式とがある。d. 構体および熱制御(spacecraft structures and thermal con-trol)衛星構体の役割はバス機器、ミッション機器が打上げ環境に耐え、軌道上環境において動作するよう支持固定することにある。また搭載したアンテナ、センサ類を要求された位置精度で支持する役割がある。打上げ環境に対しては荷重経路(load pass)を構体のどの部分に設けるかで、中央円筒方式、パネル方式の二つの方式に分けることができる。スピン安定型の衛星では中央円筒方式が採られ、三軸安定型の衛星では中央円筒方式あるいはパネル方式が採られる。このほかに中央円筒方式ではバッテリやテレメトリコマンド機器、姿勢制御機器等を搭載するミッションパネルが設けられる。また推進薬のタンクやアポジキックモータのような重量物は中央円筒内に収納する。パネル方式ではバス機器とミッション機器とを別モジュールとし、組立の簡素化と衛星バスの標準化とを図る。熱制御は搭載機器の性能を保証された温度範囲に維持するために必要であり、能動熱制御と受動熱制御とに分けることができる。受動熱制御には白色ペイント、OSR(optical solar reflector)、多層断熱材(multi-layer insulator)、ヒートパイプ(heat pipe)がおもに用いられる。能動熱制御としては熱電対とヒータによる制御がある。また衛星の電力は太陽電池による発生電力とともに廃熱能力によっても強く制限されており、大形衛星では廃熱効率を高めるための南北連結ヒートパイプや廃熱面積を拡大するための展開型ラジエータを用いる例もある。e. 通信系サブシステム(telecommunications)。 中継器(transponders)中継器は受信した電波を送信用周波数に変換し、増幅する機能をもつ。N-STAR 中継器の構成を図1・37に示す。N-STAR は、四つの周波数(Ka帯、Ku帯、C帯、S 帯)、五つのミッション(Ka帯マルチビーム、Ka帯成形ビーム、Ku帯成形ビーム、C帯成形ビーム、S 帯マルチビーム)の通信機器を搭載している。Ka帯マルチビーム通信系では、8―20情報・メディア機器図1・37 N-STAR 搭載中継器構成受信ビームと送信ビーム間をサテライトスイッチにより時分割接続する。Ka帯、Ku帯、C帯の成形ビーム通信系では、それぞれ最大2チャネルずつを異なる送受信周波数帯間で接続可能なクロスストラップ機能を有している。S 帯マルチビーム通信系では、電力増幅としてマルチポート増幅器を使用している。マルチポート増幅器の構成要素である単位増幅器では、各ビームからの通信信号をすべて共通に増幅しており、これによりビーム間にトラヒック変動があってもトータルの容量が下がることがなく、衛星電力を有効に利用できる。. アンテナ(antennas)搭載用反射鏡アンテナにはパラボラやカセグレン形式、およびそのオフセット形式が多く用いられる。初期の通信衛星であるCS-2,CS-3にはオフセットパラボラの変形であるホーンリフレクタ形が、現在のN-STAR a,bではオフセットパラボラ形式を使っている。また、これらの反射鏡面アンテナは開口形状が円であるため、日本列島のように縦に長い領域でサービス領域をむだなく覆うビームを作るためにビームの照射領域を整える成形ビーム(shaped beam)が多く用いられる。これには鏡面形状をパラボラから変形させる方法と、複数の給電ホーンを配置してその幾何学配置に応じて空間でビームを合成する方法がある。各ホーンから異なった電波を放射すると、ビームを合成されることなくマルチビームになる。実際のマルチビームアンテナでは低サイドローブ化のために位相、振幅を調整したさらに複数の給電ホーンの集まりで一つのビームをつくる。一方、フェーズドアレイ等の直接放射アンテナは時々刻々電気的なビーム成形、指向制御を行うことが可能であり、おもに衛星間通信や合成開口レーダに用いられているが、大開口面を形成するためには多くの素子を精度良く配置する必要があり、衛星の寸法、重量制限、ビーム形成回路の複雑さから、開口面の大きさが限られている。ビーム形成回路や大開口アンテナの開発により適用範囲を広げていくことが可能となる。アンテナの性能指標である利得はアンテナ開口面積、使用する電波の波長λに対し、=4πλで表される。反射面の表面形状が設計パラボラ形状からδRMS(root
