# 1・6・2 ディジタル放送

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・6 放送システム

a.ディジタル放送システムの構成とサービス 放送のディジ
タル化は，すでに実用化が進んでいる放送局内設備に加え，送信・
伝送系まで含めた放送システム全系のディジタル化を指す．放送局
で制作された番組（コンテンツと呼ばれる）は，情報源符号化
（source coding，映像・音声信号の圧縮およびデータ符号化），多
重化，伝送路符号化（channel coding，ディジタル変調）の過程を
経て放送波となる．これにより放送局から送り出される番組や各種
情報が家庭まで直接届けられるようになる．ディジタル放送の送信
系の基本構成を図1･48に示す．
ディジタル放送では表1･21に示すように1080i，720p，480
p，480i （数値は有効走査線数，iは飛び越し走査，pは順次走査）
の四つの映像フォーマットによるHDTVおよびSDTV（stan-
dard definition television，標準テレビジョン並みの解像度のディ
ジタルテレビジョン）放送を行うことができる．技術的実現性が検
証できれば1080pも認められる．
ディジタル放送で実現するサービスには，高画質HDTVのほ
か，多チャネルのSDTV，HDTVとSDTVのマルチ放送（図1･
49），階層変調を利用した降雨対応放送（図1･50）などの柔軟な番
組編成，EPG（electronic program guide，電子番組ガイド）やデ
ータ放送による各種マルチメディアサービス，ダウンロードサービ
ス，双方向サービス，サーバ型放送サービスなどがある．
