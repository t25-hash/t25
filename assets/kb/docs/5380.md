# 1・6・3 放送設備・機器

> **章**: 第1章 情報通信ネットワーク
> **節**: 1・6 放送システム

放送設備・機器には放送局や局外制作で使われるものや,家庭で使われる受信機器がある.放送局は番組の制作や運行・送出を行う演奏所(studio)と,放送波として電波を送信する放送所(trans-mitting station)からなる.a. 番組制作機器演奏所における番組制作系統の例を図1・57に示す.(1) などの映像を切換え・合成したり文字を重畳して番組を作り上げる.主要パーツは映像スイッチャである.通常の制作ではコンポジット信号で処理するが,ポストプロダクションなど,複雑な処理を行い画質を重視する場合にはコンポーネント信号を扱う.(5) ノンリニア編集機VTR に収録された映像を編集する場合,必要なシーンを早送り,巻戻しをしながら頭出しして別のVTR に収録していくため手間がかかる.ノンリニア編集機は映像を高速アクセスが可能なハードディスクに取込んで処理する.これにより任意の場所を自由に指定して効率的に編集が可能になる.VTR を使ったテープ編集とノンリニア編集の違いを図1・59に示す.(6) 照明機器スタジオなどで被写体にテレビカメラで撮像するのに十分な明るさを与えるとともに,心理的・芸術的効果を与える.光源と照明器具,位置姿勢を決める懸架装置,明るさや色を制御する調光装置からなる.光源にはハロゲン電球やメタルハライドランプなどが使われる.照明器具は目的によりスポットライト,フラッドライト,ホリゾントライト,エフェクトライトなどがある.b. 局外制作機器局外制作は,小形・軽量で機動性のある機器を使用して行う.このための機材には小形のテレビカメラ,録画機器,映像・音声調整卓などがある.番組素材や現場で編集した番組はFPU(field pick-up unit)で演奏所に送る.これらを搭載した中継車の例を図1・64に示す.局外でのニュース取材はENG(electronic news gathering)と呼ばれ,バッテリだけで動作する録画機器一体型カメラを使用する.e. 放送所の設備制作された番組を電波に乗せて送信する場所を放送所という.放送所の設備には放送機,送信アンテナ,電源設備,監視・制御装置などがある.放送所の系統を図1・65に示す.8―31図1・67 サーバ型放送のイメージ放送所の立地は,効率よく電波を届ける観点から選定される.電波の直進性が強いVHFやUHFを使うテレビジョンやFM 放送では,放送所はサービスエリアを見渡せる高い場所に設置する.一方,中波ラジオでは送信アンテナの効率を高めるため,大地導電率の良い平坦な場所に設置する.一般的に演奏所とは離れた場所にあり,その間はマイクロ波回線のSTL(studio-transmitter link)で結ぶ.放送所は無人で自動運転することが多いため,有人の演奏所から送信設備の運転を制御したり監視する遠隔監視制御装置を置く.f. 受信機器放送の受信機器には受信機,家庭用録画機器,ディスプレイなどがある.CATV やディジタル放送の受信のため,チューナ,映像・音声復号器,CAS のデスクランブラ等の機能をもつセットトップボックス(STB : set-top box)も使われる.ディジタル放送受信用のSTB はディジタルチューナとも呼ぶ.その構成を図1・66に示す.家庭用録画機器の記録媒体には磁気テープ,光ディスクおよびハードディスクがある.磁気テープを用いる家庭用ディジタルVTRにはD-VHS 方式があり,ディジタル放送のTS をそのまま記録できる.ハードディスクに記録するものはPVR(personal video8―32情報・メディア機器recorder)とも呼ぶ.ハードディスクとテープあるいは光ディスクを組合わせた製品もある.ハードディスクは高速転送,高速アクセスが可能なことから,録画番組の瞬時の頭出し,番組を録画しながら同時に別の番組や再生など,これまでの録画機器にない機能が実現できる.PVR の機能を充実させ,インタネットやホームネットワークに接続し,放送以外のコンテンツや情報も取込む家庭内の情報センタをホームサーバ,ホームサーバを前提とした放送をサーバ型放送と呼ぶ.サーバ型放送のイメージを図1・67に示す.文献(1) 秋山稔, 近代電子工学大講座7 近代通信工学, (1973), 14, 電気書院.(2) 電話網を利用するための技術参考資料第5版,24,日本電信電話(株).(3)
