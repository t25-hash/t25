# 2・2・1 基本構成

> **章**: 第2章 コンピュータ
> **節**: 2・2 コンピュータの基礎

コンピュータの基本的な構成を非常に簡略化して示すと図2・1のようになる。CPU(central processing unit)は計算を行う装置で、機械命令(machine instruction)によって動作が規定される。機械命令(以下、単に命令と記す)の種類や形式はCPU によって固有であり、それらを総称して命令セットアーキテクチャ(ISA :instruction set architecture)と呼ぶ。代表的なISA に、IBM 社の370アーキテクチャ、インテル社の8086アーキテクチャ、サン・マイクロシステムズ社のSPARC(スパーク)アーキテクチャなどがある。主記憶装置(main memory unit)は命令や演算対象となるデータを格納する装置である。コンピュータの動作は、主記憶装置から読み込んだ命令に従って、主記憶装置からデータを読み込み、演算を施し、結果を再び主記憶装置に書き戻すことの繰返しととらえることができる。これらの動作を正しく行うためには、主記憶装置に格納された命令やデータの位置を一意に指定する仕組みが必要になる。これは、主記憶装置内のすべての位置に1対1に対応する番地(アドレス:address)を付与することで実現される。すなわち、CPU と主記憶装置との間のデータの授受は、必ずアドレスを指定して、そのアドレスによって定まる主記憶装置内の位置を対象に行われる。
