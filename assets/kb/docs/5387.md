# 2・3・3 ローエンドコンピュータ

> **章**: 第2章 コンピュータ
> **節**: 2・3 実際のコンピュータ

a. 概説1971年にマイクロコンピュータ(microcomputer)が誕生して以来,その高性能化が進むとともにその小形化も急速に進んできた.1980年代に卓上を独占していたコンピュータも,今日ではポケットに入る携帯端末がその性能を上回るほどである.このような背景には,半導体集積回路の急速な技術発展とともに,その応用製品を受け入れる市場からの要求とが相まって,これら技術発展の進化を推し進めてきた.この進化は,マイクロコンピュータにとどまらず,あらゆる情報機器に波及している.とりわけ,携帯電話においては,1980年代には大きなセカンドバックほどの大きさであったものが,現在では,手のひらに納まるほどになった.また,進化は筐体の大きさだけではなく,その性能向上から,各パーソナル小形情報端末が提供するサービスの多様化,高度化を生じている.性能の向上はパーソナルコンピュータ利用を一昔前のスーパーコンピュータの利用領域に広げ,そして,サービスの多様化は,各パーソナル小形情報端末間の利用領域を超え始めている.ゲームやインタネットを使ったメールやWeb閲覧はパーソナルコンピュータの独占であったものが,携帯電話でも行えるようになり,また,音声通話は携帯電話だけのものではなくなり,パーソナルコンピュータなどでも行えるようになってきた.b. パーソナルコンピュータ(personal computer)パーソナ図2・23 初期のパーソナルコンピュータ[左:PC-8001(日本電気),右:AppleII(アップルコンピュータ)]図2・24 クラスタコンピュータ(ブレード型)ルコンピュータは,1970年代後半から1980年代初頭に誕生した.当時は現在と異なり,各社独自の仕様の基づいたパーソナルコンピュータがいくつも存在していた.日本では,日本電気がPC-8001を発表,海外に至っては,アップルコンピュータ社のAppleII,IBM 社のIBM PCが市場の拡大を図っていた(図2・23).その後,仕様の完全オープンアーキテクチャが起因し,IBM 社以外の新興企業が低価格のパーソナルコンピュータを生産できるようになった.さらにこれに大きく貢献したのが,マイクロソフト社のWindowsである.各社の仕様の違いを隠ぺいし,各種アプリケーションの普及,拡大を推し進めた.これらの要因が相互に相まって,IBM PCが主流になり現在のAT 互換機がパーソナルコンピュータの大勢を占めるようになった.このような経緯をたどり,現在のパーソナルコンピュータは,CPU の処理性能やアプリケーションの高度化もあり,従来専門の技術者が行っていた情報処理サービスを個人でも行えるようにした.ワードプロセッサ(word processor)ソフトの普及により写植作業は,一般の人でも行えるようになり,IEEE1394などの周辺機器とパーソナルコンピュータとの高速接続インタフェースの普及により,ディジタルビデオカメラで撮影された映像をビデオ編集のプロのように編集し,個人的なDVD(digital versatile disk)の作成を行えるまでに至っている.また,ネットワーク技術の発達により,クラスタ(cluster)コンピュータ,グリッドコンピューティング(grid computing)などの新たなパーソナルコンピュータの利用が進んでいる.クラスタコンピュータは,並列計算機の一種であり,その種類は,パーソナルコンピュータをネットワーク接続したものや,パーソナルコンピュータの部品を流用した,ラックマウント型,ブレード型(図2・24)などがある.グリッドコンピューティングは,複数のコンピュータをネットワークを介して接続し,仮想的に一つのコンピュータに見せる技術である.例えば,パーソナルコンピュータは1台当たりの処理能力が低く,大規模な処理をこなせないが,多くのパーソナルコンピュータをネットワークで接続することにより,全体としてスーパーコンピュータ並みの処理能力をもつことが可能となる.また,パーソナルコンピュータは実際には使われていない時間も多く,その時間を利用することにより,計算リソースの有効利用も図られる.狭義の意味で,このような利用方法もグリッドコンピューティングと呼ぶ図2・25 PDA(personal digital assistance)マイクロコンピュータが誕生する少し前の1970年初頭,日本国内では電卓の低価格化,小形化の競争が起こっていた.この競争はやがて,電卓の高機能化という方向に向かい,プログラマブル(programable)電卓や住所録機能が搭載された電卓が誕生,シャープのザウルス等,PDA の片鱗を示した.一方海外では,1990年代の半ばころに,数社からPDA が販売された.アップルコンピュータ社からはNewton Mes-sagePad,US ロボティクス社からはPilot,そしてマイクロソフト社のOS,WindowsCEを搭載したPDA を各社が販売した(図このころのPDA は機能こそ現在のベースとなるものが搭載されていたが,処理が重くなかなか軽快な使い勝手とはいかなかった.その中で,Newton MessagePadは1998年2月に開発中止が発表され,しばらくPDA の話題は落ち着いた.その後,OS やCPUなどに改良が加えられ,また,半導体集積回路の急速な技術発展により,その使い勝手は向上していった.2000年になり,再びUSロボティクス社を中心にPDA の話題が上がってきた.ハードウェアの性能向上もさることながら,ソフトウェアの操作性も向上し,その実用性を増してきた.CPU はインテル社のStrongARM SA-1110 206MHzが広く使われるようになった.アーキテクチャの改良と動作周波数の向上は継続しており,StrongARM の後継であるXScaleプロセッサを搭載したPDA が現れた(図2・26).このような半導体集積回路の発展は,PDA の新たなアプローチを提供することになった.それは,従来,搭載機能を必要最小限に絞り小形化を図るアプローチであったが,パーソナルコンピュータそのものを小形化するアプローチである.このようなコンピュータはマイクロPCやUPC(ウルトラPC)と呼ばれる.その一例が,OQO社が開発したOQOウルトラパーソナルコンピュータ(図2・27)である.OQO社は,アップルコンピュータ社のPowerBook8―43図2・27 phone)携帯して移動しながら電話がかけられる電話機が出たのは1970年代後半である.このころは,携帯といっても大きなショルダーバック位の大きさであり,重さも2~3kg はあった.それが,やはり半導体集積回路の発達により,ようやく片手でも持てるサイズになったのが,1980年代後半である(図2・28).その後,携帯電話のサイズはますます小さく,それに伴い重さもずいぶん軽くなった.また,通信技術の発展も急速に進み,音声通話中心であった初期の携帯電話に比べ,データ通信が行えるようになった.さらに通信方式もPDC(personal digital cellular)方式からW-CDMA(wideband code division multiple access)方式へと,より高速なデータ通信が行える方式に移行が進んでいる(図このようなハードウェアの発展は,携帯電話の利用形態も変えてきた.音声のやり取りだけでなく,Webの閲覧,メールのやり取り,双方向のTV 電話さえ扱えるようになった.また,共通のOSを搭載することにより,簡単なアプリケーションも携帯端末で操作可能になっている.このような傾向は,今後,携帯電話の利用領域がPDA の利用領域へと拡大していく予兆である.文献(1) 情報処理学会歴史特別委員会編, 日本のコンピュータ発達史,8―44情報・メディア機器(1998),オーム社.(2)
