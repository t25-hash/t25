# 3・1・1 情報処理システムにおける記憶装置(storage unit)

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・1 記憶装置概論

の役割人類が今日の高度な文明を築き得たのは、言語を発明し、過去から現在までに多くの人々が得た情報や知恵を記録し、先人の知識遺産を現在のすべての人々のものとすることによって、より高度な技術を開発し社会の生産力を高めてきたからである。大量の情報を高密度・高速に蓄積する記憶装置は、人間のこのような知識生産力を高めるための機械である。人類の知識生産技術の歴史の中で画期的な出来事は、紙の発明、印刷機械の発明、通信技術の発明、そして20世紀半ば以降に生まれた電子計算機と大量の情報を高密度・高速に蓄積できる記憶装置の発明であった。情報処理システムは、図3・1にパーソナルコンピュータ(パソコン)の構成例を示すように、機能の異なる五つの装置から構成される。すなわち、数字、文字、図形、音声、画像などの自然情報を機械的に処理できる2値符号の機械情報に変換する、マウス、キーボード、スキャナ、ディジタルカメラ、ディジタルビデオカメラなどの入力装置、データを処理・加工する中央処理装置(CPU : cen-tral processing unit)、処理中のデータを一時的に記憶する内部記憶装置[internal storage unit、主記憶装置(main memoryunit)]、データおよび処理プログラムを蓄積する外部記憶装置[external storage unit、補助記憶装置(auxiliary storage unit)]および処理したデータを人間の理解できる自然情報に変換するプリンタなどの出力装置である。情報処理システムには、パソコン、ワークステーション、サーバ、電算機センタ、携帯情報機器などがあるが、ここでは現在生産額の大半を占めているパソコンにおける記憶装置に注目する。内部記憶装置には電気的に高速ランダムアクセスが可能な半導体メモリが使用される。その主体は構造が簡単で集積度が高いが記憶保持動作が必要なDRAM(dynamic random access memoryの略、ディーラムと呼ぶ)である。CPU との直接の演算処理には、集積度は低いが最も高速アクセスが可能で記憶保持動作が不要なSRAM(static random access memoryの略、エスラムと呼ぶ)が用いられる。半導体メモリは通称メモリとも呼ばれる。一方、外部記憶装置には通称ストレージと呼ばれる機械的装置が使用される。まずランダムアクセス可能な大容量記憶装置として磁気ディスク装置(通称、ハードディスク装置ないしHDD と呼ばれる)がある。また小容量データの保存・配布・交換媒体としてフレキシブルディスク装置(1.4MB の3.5インチフロッピーディスク装置をFDD、ディスク媒体をFD と呼ぶ)が、大容量プログラムデータ、画像データ等の保存・配布・交換媒体としてCD,MO,DVD などの光ディスク装置が用いられる。さらにワークステーション、サーバ、データウェアハウスなどの大規模データのバックア8―45ップに磁気テープ装置(MTD)が用いられる。一方、ディジタル機器一般の基本ソフトの格納、携帯情報端末、携帯電話、ディジタルカメラの外部記憶装置およびパソコン用情報交換媒体として半導体メモリの一種であるフラッシュメモリが用いられている。一般に記憶装置に要求される性能は、ランダムアクセス時間が短く、データ転送速度が高く、かつビットコストが低いことである。しかし、これらすべての面で優れた記憶装置は存在せず、例えばDRAM に対してHDD はアクセス時間が5けた以上遅いがビットコストは2けた以上低い。そこで図3・2に示すように、CPU に対して各種記憶装置を階層的に構成し、演算処理のためCPU と常時データをやり取りするメモリとしては、ビットコストは高いがアクセス時間が最も小さいSRAM を用い(キャッシュメモリという)、そこにデータがなければその約3けた大容量のアクセス時間はやや遅いDRAM からデータをもってくる。また、そこに処理すべきデータがなければ、アクセス時間はmsオーダであるがビットコストがさらに小さく2けた以上大容量のハードディスク装置からデータをDRAM に移動させるようにしている。さらにコンピュータシステムへのデータ、プログラムの入出力・保存・交換等には、媒体価格の安いフラッシュメモリ、フレキシブルディスク(ドライブや媒体の絶対価格が小さいことも重要)を小容量用に、容量の増大に応じてCD,MO,DVD を用い、大規模データのバックアップには体積蓄積密度が高く、データ転送速度が比較的高い磁気テープを用いている。用途に応じて各種記録媒体を使い分け、全体的にシステムの性能・価格比を高める、このような使用形態を記憶装置の階層構成という。また携帯情報機器では、ビットコストは比較的高いがアクセス時間が短いフラッシュメモリが外部記憶として用いられている。以下に記録原理が異なる半導体メモリと機械的なストレージについて技術の特徴と動向を述べる。
