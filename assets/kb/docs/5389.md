# 3・1・2 半導体メモリ(semiconductor memory)

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・1 記憶装置概論

一般に、メモリは1,0を区別できる1ビットの状態を記憶する領域を作る必要があり、これを記憶細胞(ビットセル)という。記憶細胞には番地がふられ、任意の番地を選択し、これに書込み読出しできる仕組みが必要である。SRAM,DRAM では記憶細胞がシリコン基板上にマトリックス状に形成され、この記憶細胞を交点として、書込み読取り用のセンス線とワード線と呼ばれる番地指定と書込み読出し用のリード線が縦横に張られている。SRAM の記憶細胞は3個のトランジスタによるフリップフロップ回路からなり、高速に動作し、通電を切らなければ記録が失われない利点をもつが、回路が複雑なので集積度を上げにくい。これに対してDRAMは1個のトランジスタによりコンデンサ部に電荷を蓄える構造なので、SRAM に比して集積度も十数倍高くでき、価格が安いため主記憶に用いられる。しかし、電荷が時間とともに失われるので、一図3・1 パソコンシステムの構成例図3・2 記憶装置の階層構成(2004年11月現在)8―46図3・3 DRAM、フラッシュメモリ(NAND 型)、HDD、光ディスク装置の面密度の年次推移定時間ごとに記憶保持のため再書込み(リフレッシュ)を行う必要があり、このためダイナミックRAM と呼ばれる。これらのパソコンの主記憶装置に使用される半導体メモリは電源を切ると記憶が失われる揮発性メモリであるが、一度記録すれば通電しなくても記憶が失われない不揮発性メモリもある。一度記録して読取り専用のROM と記憶の消去書込みが何度でも行えるEPROM(erasable programable read only memory)があり、後者には消去に紫外線を用いるUV-EPROM と電気的に記憶を消去できるEEPROM(electrically erazable programable read onlymemory)がある。EEPROM のうち、フラッシュメモリ(flashmemory)は記憶細胞ごとのランダムアクセス機能をもたずブロック単位でデータを書込み消去する構造をもつのでDRAM より高密度化が可能である。フラッシュメモリにはNOR 型とNAND 型があり、NOR 型はディジタル機器一般の基本ソフトの記憶に使用され、NAND 型は書込み速度が速く大容量化しやすい特徴をもつので、携帯情報機器や情報交換用記憶媒体として広く使用されるようになってきた。図3・3にDRAM とフラッシュメモリの面密度向上の年次推移を示す。内部記憶装置は1970年代の初めまで磁気コアが用いられた。1972年にシリコン基板上に写真露光技術とエッチング技術に基づき微小な回路を形成するLSI技術により、16kb(キロビット)/チップ(約1500mm)の半導体メモリが実用化されて以来、3~4年で4倍の高集積化が図られてきた。2004年11月現在の主流は256Mb(メガビット)/チップ、512Mb/チップ製品で、1Gb(ギガビット)/チップも製品化されている。これらは線幅110nmのプロセスの技術で作られており、最大の面記録密度は3Gb/in(平方インチ)程度である。線幅90nm以下は技術的な困難が増大すると予想されており、また不揮発性メモリ実現への期待から、記憶素子としてコンデンサの代わりに、強誘電体、TMR 記憶素子、光ディスクに用いられている相変化膜などを用いた不揮発性メモリも代替メモリ候補として研究されている。またフラッシュメモリでは多層記録や多値記録により高密度化しようとする動きもある。
