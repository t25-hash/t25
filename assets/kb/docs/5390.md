# 3・1・3 ストレージ(storage)

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・1 記憶装置概論

内部記憶装置はコンピュータのスイッチを切ると記憶内容が消えてもかまわないが,外部記憶装置は通電しなくても永久に保存されることが必要である.このため磁気媒体の残留磁化,物質の有無や相変化などを記録原理に用いている.また内部記憶装置より圧倒的に記憶容量が大きくビット当たりのコストが低いことが重要である.このため半導体メモリのように1ビットごとにランダムアクセスできる記憶細胞を形成するのではなく,記録再生変換器(一般にヘッドと呼ぶ)と記録媒体の精密な相対運動と記録信号の制御によってトラック状に記憶細胞を配置し,一定量のデータブロックをまとめて書込み・読出しを行うようにしている.図3・3に磁気ディスク装置(magnetic より大きくなり,2002年度より年率30%程度に減速したものの,2004年11月現在約100Gb/inの最も高密度な記録装置が実用化されている.図3・3中に示すように,磁気記録によるビットサイズはヘッド・媒体の相対運動方向に小さく,また媒体とヘッドの記録再生速度も速いので,データ転送速度が高いのも特徴である.光ディスクはビットサイズがレーザ波長に支配され,後述する青色レーザを用いた記憶容量27GB(ギガバイト)の高密度化DVD 装置でも,面記録密度は19.5Gb/inである.同じ記録原理でも,記録媒体の形式やヘッドと媒体との相対運動の形式によって,機構形態,機能・性能およびビット当たりのコストが異なる各種の記憶装置を実現できる.特に磁気記録を原理とする記憶装置は各種の形式がある.表3・1は記録媒体の形式と記録媒体とヘッドの相対運動の形式の組合せ方と,これまでに使用されたことのある記憶装置の名前である.まず記録再生のための相対運動には,等速回転運動か等速直進運動が用いられる.装置内で実現できる剛体の等速運動は回転運動しかなく,また回転運動は高精度で高速運動が実現しやすいので,記録再生のための相対運動には媒体かヘッドを回転させる場合が多い.ドラム装置は円筒表面に多数の固定ヘッドを用いて記録する装置で,データへの平均アクセス時間が回転周期の2分の1となり最も速い.しかし高価なヘッドが多数必要になるのと,小形にすると記録表面が広くとれないので大容量化しにくい欠点がある.磁気ドラム装置は1950~1960年代に主記憶装置として使用され,1970年代初めまでスワップメモリとして利用されたが,その役割は半導体メモリとハードディスク装置により代替された.複数枚の剛体ディスクを回転させ,ディスク1面当たり1個のヘッドを半径方向に移動させて多トラック記録する剛体ディスク装置は,ドラムより大容量化可能でビットコストが大幅に低減でき,アクセス時間もあまり増加しない.このため磁気ディスク装置と光ディスク装置にこの機構形態が用いられている.磁気ディスク装置は高密度・高速・小形化が進み,3.0~3.5インチ形がサーバ用やデスクトップパソコン用に,また2.5インチ形がノートパソコンの外部記憶装置として用いられている.サーバ用はアクセス時間の短縮と高転送速度が重要で,このため回転速度は7200rpm(回転/分),10000rpmと増大し,1999年に15000rpmの機種が開発された.2004年11月現在の最高位機種の性能は平均アクセス時間5.3ms,転送速度147MB(メガバイト)/s,3.3インチディスク4枚による記憶容量は147GB(ギガバイト)となっている.デスクトップパソコン用3.5インチ形HDD は回転速度が5400~7200rpmで,記憶容量40~80GB/枚,転送速度100MB/s程度,平均アクセス時間13~15ms程度である.ノートパソコン用の2.5インチ形は回転速度4200~5400rpm,容量20~40GB/枚,平均アクセス時間17~20ms,転送速度100MB/s程度となっている.近年,回転速度7200rpmの機種も現れ,また10000rpmの機種も商品化されようとしており,2.5インチ形の高性能化による3.5インチ形代替の兆しも現れている.一方では,ノートパソコン,携帯ディジタル機器用として1.8インチ形(20~30GB/枚)が用いられ,ディジタルカメラ,携帯ディジタル機器に1インチ形(2~5GB)が用いられている.また2005年には携帯電話用として0.85インチ形(2~4GB)が商品化される予定である.光ディスク装置(optical disk drive)は1990年代後半よりはん用プログラム入力にCD-ROM が,大容量データ保存や普及用にCD-R(recordable)が,書換え可能な保存媒体としてMO(mag-neto optical disk)が用いられてきた.CD-ROM は音楽用CD[compact disk(disc)の略,シーディと読む]と同様に,直径120mmの光ディスク媒体に型成形により微小なピットを形成し,読出しはレーザ光の反射率の相違により行うもので,最大約800MB の容量があり,すでにパソコンに標準装備されている.CD-Rはレーザの熱により媒体の相変化を用い,記憶容量は650MB または700MB あり,媒体価格が安いのでよく利用されている.最近のパソコンにはCD-R 媒体に記録でき,また容量650MB のCD-RW(rewritable)媒体にも繰返し記録再生できるCD-R/RW が搭載されるようになってきた.MOは3.5インチ形ディスクにレーザビームの熱と磁界で磁気記録し,レーザ光で読み出すので比較的高速に記録再生できる特徴を有し,230MB から540MB,640MB,1.3GB,2.3GB と大容量化されてきている.一方,動画像をも記録できる,片面4.7GB,両面9.4GB の大容量光ディスク装置として,読取り専用のDVD-ROM および一度だけ書き込めるDVD-R がある.また,書換えできる装置にはDVD-RAM,DVD-RW およびDVD-ROM との互換性を重視したDVD+RWがあり,これらを搭載したテレビや動画を楽しむパソコンおよび大容量HDD と組合わせたHDD/DVD レコーダが普及し始めている.これらのDVD には赤色レーザが用いられているが,青色レーザを用いた記憶容量27GB(片面1層)/54GB(片面2層)のブルーレイディスク,および現行DVD と互換性を重視した15GB(1層)/30GB(2層)のHD(high density)DVD の商品化が2004年から2005年にかけて活発に行われている.フレキシブル媒体はハード媒体より面記録密度は低いが安価で保存時の蓄積効率がよい.このため記録媒体だけをカートリッジ化して1台の記録装置で記録再生すれば,大容量・低ビットコストが実現できる.フロッピーディスク装置(floppy disk drive)はランダムアクセスできるが,小容量なので個人の文章用ファイルに適している.従来から利用されてきたフロッピーディスクの容量1.4MBは小さいので,同じ3.5インチ形フレキシブルディスクにトラックサーボ技術を用いて高密度記録した記憶容量100MB,250MB のZipドライブが商品化された.しかし,フラッシュメモリやCD-R/RW,DVD-R/RW,DVD-RAM のドライブと媒体の性能と価格がこれらのフレキシブル磁気ディスクをしのぐようになってきたので,フレキシブルディスク型はフラッシュメモリと光ディスクにより代替されつつある.磁気テープ装置(MTD : magnetic tape drive)にはオーディオ用カセットテープレコーダと同じように,固定した1チャネルまたは多チャネルヘッドで記録する形式と,ビデオテープレコーダ(VTR)と同様に回転ヘッドで記録する形式とがある.磁気テープ装置は記録時のデータ転送速度が光ディスク装置より速く,情報蓄積効率がよいので,現在も画像データやハードディスク内の全データなどの連続したデータを保存・蓄積するのに用いられている.カードは小容量媒体としては便利なので,キャッシュカード,切符,定期券などに用いられる.このように記録原理は同じでも,媒体の形式,ヘッドの配列,運動形式などを変えるとさまざまな特徴をもった記憶装置が実現でき,それぞれ異なる情報処理の用途に利用できる.また,同じ記憶装置形態でも用途によって大きさ,回転速度,記憶容量,性能の異なるさまざまな装置が商品化されている.
