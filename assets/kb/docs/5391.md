# 7・6・6 プロトンアイランズ構想（Proton Islands project )

> **章**: 第7章 水素エネルギー
> **節**: 7・6 水素エネルギーシステム

中部国際空港近接地域における新たな都市形成の場で，環境に配
慮した新たな社会システムを形成するための先導的拠点整備を基本
理念として，①次世代エネルギーシステムの創造，②環境面から見
た都市・生活のあり方の提案，③エネルギー関連産業の創造・育成
を目標に，愛知県支援のもと，産官学メンバーよりなるプロトンア
イランズ基本計画策定委員会で検討が進められている（2001年6
月時点）．プロトンアイランズ全体構想を図7･31に示す．構想は三
つの分野からなる「プロトンアイランズ実証・実験プロジェクト」
を統合した形となっている．中部国際空港近接地域という限定され
た領域ではあるが，水素の長距離洋上輸送部分を除いて，図7･27
および図7･28に示した水素エネルギーシステムの基本部分を抱含
し，さらに，メタノール，ジメチルエーテル（DME:dimethyl
ether ），GTL（gas to liquid）供給・利用システムも組合わせた意
欲的な構想である．
