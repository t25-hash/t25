# 3・2・1 記憶システムの階層構成

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・2 記憶システム

計算機システムにおける記憶システムは、プログラムならびにデータの格納場所を提供する部位として位置付けられる。一般に記憶システムにおける記憶媒体は、その性能、すなわち高速なアクセスタイムを提供するものほど、単位容量当たりのコストが高価であり、低性能な記憶装置は容量当たりのコストが低い。このことから、複数の記憶媒体を組合わせて階層的な記憶システムを構築することが多い。これを記憶の階層構成と呼ぶ。記憶媒体には、CPUに近接する高速・小容量な半導体記憶装置(SRAM,DRAM)や、低速・大容量な磁気ディスク装置(magnetic disk drive)、磁気テープ装置、光ディスク装置などが含まれる(図3・4)。レジスタ、キャッシュメモリ(cache memory)は極めて高速な半導体記憶素子によって構成され、主記憶(main memory)上の高頻度でアクセスされるデータが一時的に格納される。キャッシュメモリは、CPU と主記憶装置の速度格差を解消するため、二次から三次レベルの階層構成が採られることが多くなってきている。レジスタ、一次、二次キャッシュメモリはCPU 内部に置かれ、三次キャッシュメモリは通常CPU チップ外に置かれる。主記憶装置は、プログラム、データなどを格納する記憶装置であり、揮発性記憶が用図3・4 階層記憶(hierarchical memory)8―48情報・メディア機器いられる。磁気ディスク装置、磁気テープ装置は、プログラムやデータの永続的な保存に利用され、不揮発記憶が用いられる。マルチメディアデータなどデータの容量増大とともに、磁気テープや、光ディスクなどの可搬媒体をレコーダとともにジュークボックス筐体に装着した超大容量ファイルシステムが最下層オンライン記憶システムとして利用されている。
