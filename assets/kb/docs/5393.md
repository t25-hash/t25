# 7・6・8 その他の水素エネルギーシステム

> **章**: 第7章 水素エネルギー
> **節**: 7・6 水素エネルギーシステム

7･6･7項までに記述したもの以外に以下のような多様な水素エネ
ルギーシステムが研究されている．
(１) 離島または遠隔地水素エネルギーシステム：アラスカやわ
が国西南諸島のように，エネルギーネットワークが未整備の地域に
おける太陽エネルギー，風力，水力等を利用し，水素を二次エネル
ギーとする自立型エネルギーシステム．
(２)天然ガス・水素エネルギーシステム：天然ガスから水素を
製造し，FCV等へ供給するオンサイトまたはオフサイト型システ
ム．
･オンサイト型：都市ガスグリットを利用し，最終需要サイトで小
規模水素製造し，ユーザに供給．
･オフサイト型：需要地から離れた場所で相対的に大規模プラント
で水素製造，水素需要サイトへ配送．
(３)水素を用いるエネルギー自立型住宅システム：住宅単位で
の太陽エネルギー利用自立型水素エネルギーシステム．
