# 7・2・2 わが国の動き

> **章**: 第7章 継続・循環型社会の構築
> **節**: 7・1 概説

省資源・加工貿易立国でありながら国際競争力を担保するわが国の製造技術は，資源生産性，エネルギー原単位の国際比較においてトップレベルである．
これを支える機械工学は各分野でより強い，早い，軽い，安い技術を追求してきた．
これは個々には生活・環境に対してもおおむね良好なものであったが，製造物の大量普及が引き起こした環境問題は使われ方，廃棄のあり方を含めた検討段階に入ってきている．
日本機械学会でも2002年にRS-3環境技術将来展望検討分科会 を設置し，以下の七つの視点を提言している．
生産製造プロセスと環境技術の一体化 機械工学としての環境要素技術の研究と教育 環境技術の有機的抽出 環境モニタリング，データベースの構築 環境システム技術の多様性a.環境基本計画 わが国においては，従来の「公害基本法」等から新たな環境政策の枠組として，1993年に「環境基本法」，1994年には環境基本計画が制定された．
環境基本計画では，四つの目標である循環，共生，参加および国際的取組みを明言しており，環境負荷の少ない循環を基調とする経済社会システムを実現するために，環境施策の大綱，各主体の役割，政策手段を定めている．
