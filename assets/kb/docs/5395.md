# 3・2・4 二次記憶システムの高機能化

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・2 記憶システム

ディスクアレイの出現により,二次記憶システムは,従来の単なる補助記憶装置から脱皮し,大きな処理機能を装備すると同時に,膨大なコードからなるソウトウェアを有する装置へと変身してきた.データとパリティの変更は,透過性を満足するトランザクションでなければならず,また,障害ディスクの回復を通常サービスと並行して行うには高度なソフトウェアが必要とされた.この流れは,ディスク装置をより高付加価値な装置へと変ぼうさせるきっかけとなっていくと考えられる.インタネットを介したビジネスが広く浸透するにつれ,バックアップによるシステム停止が無視できなくなってきたことを背景に,ディスクシステム内での二次データの作成(スナップショット)取得機構と非同期バックアップという形態が提案されている.さらに,システム障害への対処として,ディスクシステム自体にリモートコピー機構を導入し,ホスト介在のない高信頼なファイルシステムを実現する手法が導入された.最近では,メインフレームとオープン系システムとの処理コードの違いをディスクシスム上で自動変換することにより,ファイル共有を可能にするなど,極めて高度な機能が実現されつつある(図
