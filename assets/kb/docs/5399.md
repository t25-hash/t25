# 3・2・8 ストレージサービス

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・7 )。

ディジタルデータの増大に伴い、その管理負担が重荷となりつつあることから、ストレージ管理のアウトソース化が試みられている。ネットワーク技術の進展とそのインフラストラクチャの整備に連動し、ユーザは、ネットワークを介して接続したストレージを利用し、スペース管理、機器の更新、バックアップなどをサービスとしてサービスプロバイダが提供するビジネス形態が出現しつつある。アプリケーションレベルでのサービス提供と同時に、バックヤードとしてのストレージ管理も一つの重要なサービスとなろう。情報通信における高速ネットワークの進展により、2010年以降には社会インフラストラクチャとして、一般家庭でも平均100Mbpsの速度のネットワークを利用できるようになると予想されている。また、情報のディジタル化の進展とともに衛星放送が拡大し、2010年には高精細画像を数多くのチャネルで見ることができるようになると推定されている。このような高速情報通信のインフラのもとでは行政、教育、医療、レジャーなどの分野で個人あるいは企業の社会活動がマルチメディア情報を大量に活用すると予想され、大容量映像情報のダウンロードなどによって端末系の情報ストレージ量は今後ますます増大の一途をたどると思われる。一方、ビジネス分野では、電子商取引や高速イントラネットが一般化し、さらに社会インフラとして電子医療や遠隔医療、電子行政、遠隔教育、電子図書館、電子美術館などのシステムが普及するものと考えられる。これらのシステムでは、通信系と情報処理系が統合されたシステムとなることが特徴であり、これには高速ネットワークに接続された高度なサーバシステムがそのインフラとなる。このサーバシステムの実現には超大容量のファイルシステムが必要となり、2010年には現状の1けたから2けたのデータ蓄積量の増加が生じると予想される。このような社会的背景を受けて、磁気メモリ技術および光メモリ技術においてもさらなる高密度化が要求される。磁気ディスクおよび光ディスクを超高密度化の視点で現在から2010年代までの技術的発展を予想すると図3・9の推移を示すことが考えられる。1900年代の磁気ディスクでは、再生出力の高いMR ヘッド(magneto-resistive head)、PRML(partial response maximum likelihood)信号処理技術、さらには、GMR ヘッド(giant magnetoresistivehead)で再生感度向上、FDB(fluid dynamic bearing)スピンドルモータでNRRO(non repeatable runout)改善、反強磁性結合媒体で低ノイズと熱安定性の両立、などの技術開発により年率図3・9磁気および光記録の密度トレンド図3・10 スピンバルブGMR ヘッド100%を超える記録密度の向上が図られた。今後はさらに、マグネティックトンネルジャンクションGMR ヘッド、垂直磁気記録技術の適用あるいは熱アシスト磁気記録、パターン媒体などにより、1平方インチ当たり数百Gbの記録密度の達成が図られていくものと思われる。光ディスクについては、従来おもに、レーザの短波長化とヘッドのレンズ開口数(NA)を高める技術などにより高密度化が図られてきた。また、光磁気(MO)の超解像技術(MSR)やクロストーク低減などの信号処理技術の適用により高密度化が進展した。今後はさらにマスタリング技術における電子ビーム露光技術、近接場記録技術、あるいは三次元多層・多重記録技術の導入により、1平方インチ当たり数百Gbの記録密度の達成が図られていくものと思われる。以下では、磁気記録および光記録の高密度化技術について概説する。
