# 3・3・1 磁気記録の技術動向

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・3 磁気記録と光記録

近年,磁気ディスクの記録密度は,年率100%を超える伸びを示している.2004年末には,100Gb/inの面記録密度が量産レベルで実現され,130Gb/inを超える勢いで研究開発が進められている.このような面記録密度の向上は,ヘッド,メディアおよび高密度信号処理等に関わる技術の集大成で実現されており,特に年率100%の記録密度の向上は,狭トラック化,高感度化等のヘッド技術(GMR ヘッド)と媒体の低ノイズ化によるところが大きい.ただし,21世紀に入ってから100%の伸びは止まっており,30%程度に落ち着いた.このような高密度化において媒体粒子を微細化していくにつれ,熱揺らぎ(thermal decay)によって常温においても記録された磁化が減衰してしまい,高密度化の大きな障害となることが指摘されている.そこで,このような問題を回避するための技術として,熱アシスト磁気記録やパターン媒体などが注目されるようになってきている.a. 磁気ヘッド(magnetic head)磁気抵抗効果型(MR)ヘッドは,大きな再生出力が得られることから年率100%を超える記録密度の向上の原動力となった.この技術は,さらに新しい巨大磁気抵抗技術(GMR)へと発展した.現在のGMR 素子はスピンバルブと呼ばれる構成である.スピンバルブGMR ヘッドの構成を図3・10に示す.スピンバルブ膜の構造は2枚の磁性膜を挟んで非磁性層を設け,片側の磁性層を磁気的に固定する構造である.磁気抵抗の変化は2枚の磁性層の間の相対的な磁化の方向の変化から発生する.最近では,薄膜界面で電子を散乱させ,さらにMR 効果を高められるスペキュラスピンバルブ技術の開発が進められている.このようなスピンバルブGMR の改善により100Gb/inが実現されている.さらにトンネルジャンクションGMR 技術も検討されている.トンネルジャンクションGMR ヘッドは,薄い電気絶縁層を挟んで二層の磁性膜を有し,この磁性膜の磁化が平行,反平行の場合に絶縁膜を通るトンネル電流の値が異なることを利用している.トンネルジャンクションGMR においては,30%を超えるMR 効率を期待できるが,トンネル接合として電気絶縁層を用いるため,その低抵抗化が実用上のポイントとなる.今後の高密度記録を実現していくための過程では,再生ヘッドの開発以上に書込みヘッドの開発がキーポイントとなる.書込みヘッドの改良にはおもに二つの開発課題が検討されている.一つは高保持力媒体へのオーバライト特性の改良であり,他の一つは高周波書込みの改良である.ディスク媒体の保磁力向上に対応するための高磁極材料,高速記録に対応するための短磁気回路,磁極材料,低インピーダンスコイル,および0.1 mレベルの狭トラック幅磁極を形成する高アスペクト加工プロセス技術が今後必須となる.将来的にはナノメータ制御のMEMS 技術の使用も必要となると考えられる.一方,前述したように面記録密度が高まるにつれて,媒体の磁化が熱揺らぎにより消えていく熱揺らぎ限界が現れてくる.この対策として,書込みヘッドの磁路の中に媒体を入れて書込みを行う,二層構造垂直磁気記録媒体を用いた垂直磁気記録が記録密度の向上には有望である.二層構造垂直磁気記録媒体においては原理的には,面内記録の1.5倍程度の書込み効率の向上が期待できる.しかし,実際には高密度記録での媒体の保持力もさらに大きくなり,かつトラック幅の減少および記録線密度の向上に対応した書込みポールのポール長の減少により,やはり書込みの限界が出てくる.書込み効率の良いヘッド設計が必要である.b. 磁気記録媒体近年の磁気記録媒体は,磁性結晶粒の微細化と粒間の交換相互作用を低減する成膜プロセスの改善によって低ノイズと高分解能を実現してきた.しかし,結晶粒の微細化,記録層の薄膜化がなされるにつれて,熱揺らぎによる磁化の不安定性を一層高める結果となってきている.しかし,この熱揺らぎによる密度限界論を打破する方法として水平記録の場合,反強磁性結合媒体(synthetic ferromagneticmedia)が提案された.反強磁性結合媒体は,二層あるいは三層の媒体構成であり,記録層にRu層を介して薄い磁性層を反強磁性結合させた媒体である.長手記録媒体の熱安定性を高められるだけでなく,媒体SN を向上させることができる媒体として研究が進められている.この媒体は高分解能と低ノイズ特性の両立を可能にし,面内媒体の熱揺らぎによる記録密度の限界は,200~300Gb/inまで改善されると予想されている.一方,面内記録の熱揺らぎ限界の懸念から,垂直磁気記録(per-pendicular magnetic recording)の研究が盛んになってきている.垂直磁気記録の利点としては,(1)高密度記録状態で隣接記録磁区からの反磁界が記録磁化を安定化する方向に作用する,(2)面内媒体に比べて記録層を厚くできるので,磁性粒子の体積を比較的大きくできる,(3)軟磁性裏打ち層の付与により,記録磁界が大きくできるので,熱安定性に優れた高磁気異方性記録材料を使用できるなどのメリットがある.垂直磁気記録媒体において,記録密度の向上を実現するうえでの課題は媒体ノイズの低減である.現状では,面内記録媒体と同様に磁化遷移領域からのノイズが主体であり,磁性粒子間の交換結合の低減と磁性粒子の微細化が必要である.さらに二層構造垂直磁気記録媒体では,軟磁性下地層からのノイズ低減も重要な課題である.磁気記録媒体の構成をまとめて図3・11に示した.高密度記録媒体の開発では,テラビット級の磁気記録実現に向け図3・11
