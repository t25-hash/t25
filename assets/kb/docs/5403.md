# 3・4・2 可換媒体型装置(removable disk storage drive)

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・3 に,シーク時間の変遷を図3・18に示す.1990年代後半から

可換型の磁気記録媒体としてフロッピーディスクFDC(flexibledisk cartridge)がよく知られている.これは,2軸に伸延した約80 m厚のPET(polyethylene terephthalate)フィルム基材の表面にCo被覆したγ-FeOなどの磁性体をバインダや潤滑材などとともに塗布したものが用いられている.現在は,JIS X 6225:1995などで規定される3.5インチ形で記憶容量1.44MB のものが主流となっている.駆動装置はFDD(flexible disk cartridge storage drive)と呼ばれ安価で標準化されている媒体を用いるため,近年PC(personalcomputer)の普及とともに手軽なデータ交換用に多用されてきた.しかしながら,扱われるデータ量の増加とともに記憶容量の大きなFDD が求められてきた.3.5インチ形で記憶容量4,16,21MB などの提案がなされ,規格化が行われたが,線記憶密度(linear density)とトラック密度(track density)を高めるために必要なコストと記憶容量の釣合いが良くなかったために,一般的に普及するには至らなかった.一方,1995年にIomega좲(アイオメガ)からZip좲(ジップ)100,1996年には,松下寿電子工業などからLS-120[後のSuperDiskTM(スーパーディスク)]が相次いで発売され,100MBクラスの記憶容量をもつFDD が脚光を浴びるとともに一般的にも普及を始めた.以下,3.5インチ形程度の大きさで記憶容量が100MB 以上のFDD を実現するための技術的な概要について説明する.まず,表a. 記憶容量(storage capacity)記憶容量は,記憶媒体の面表3・2 各種FDD の仕様項目Zip250SuperDisk2HDDisk m程度としてある.一方,磁気ヘッドは単結晶フェライトのリング型で媒体を十分飽和記録できるようにR/W(read/write)Gapには,飽和磁束密度の高いFe-Si-Al,Co系アモルファス,FeN 系などの材料を付加したMIG(metal in gap)と呼ばれる形態のものが用いられている.スライダの形状は,安定した媒体との機械的インタフェースが得られるようにABS(air bearing surface)面に工夫が施されている.Zipに用いられているスライダは,ナノスライダ(3・4・1・c項参照)でABS 面には安定して数nmの浮上量が得られるように走行方向と直角にクロススロットと呼ばれる溝が設けられ,正負圧を発生する4パッド型の構造をしている.トラック密度は,磁気ヘッドを所要トラックにいかに正確に位置決めできるかによって決まる.トラック方向のずれの主要因としては,媒体の着脱による1回/回転の偏心成分と,フィルム基材を2軸に伸延したために起こる温湿度変化による2回/回転の異方性成分である.ヘッドの位置決めには,セクタサーボ方式が多用されている.トラック内にバーストパターンによるサーボマークを配置して磁気ヘッドで読み出し,オフトラック量を補正するようにVCMを制御する.オフトラック量については,通常トラックピッチの数%程度に抑えられている.一方,SuperDisk においては,媒体の片側に2トラックおきにレーザによりサーボステッチと呼ばれる反射率の低い溝が設けてある.光学系により二相の信号を生成してオフトラック量としてサーボを掛けている.オフトラック量の情報は,セクタサーボによれば間欠信号であるが,サーボステッチでは,ほぼ連続信号とすることができる.さらには,角速度一定CAV(constant angle velocity)で動作8―57するFDD においては,媒体全面をより有効に使うために内周に比べて外周側のトラック当たりのセクタ(sector)数を増やすZBR(zone bit recording)方式を用いて媒体全周にわたって同程度の線記録密度として記憶容量の増加を達成している.Zip100では4ゾーンに分け,トラック当たりのセクタ数は40~72としてあり,Zip250では8ゾーンで60~120としてある.このようにして面記録密度を高めることで記憶容量の増大がなされている.記憶容量の増加とともに転送速度の向上や,ECC(error correc-tion code)を用い,エラー訂正や欠陥セクタの代替処理などが用いられて信頼性・パフォーマンスが高められている.これらは,HDD(hard disk drive)で用いられている技術をFDD に応用したものである.Zipにおいては,512バイトのデータセクタに対して,4バイトのCRC(cyclic redundancy code)と48バイトのECCが付加され強力なエラー訂正がされている.このおかげで,2HD でのエラーレートが1×10程度であるのに対して1×10程度と向上されている.b. 下位互換性(downward compatibility)可換媒体型装置では,基本的に装着できる媒体に対して互換性が求められる.Zipでは,カートリッジの形状を変えて従来の2HD が装着できないようにして下位互換性のない構成としてある.これにより,機構部分のみならず回路構成も簡素化されている.Zip以外のFDD においては,2HD の媒体に対してのR/W 互換性を有している.媒体の磁気特性と,トラック幅が異なるために,磁気ヘッドをそれぞれの媒体に対応するようにABS 面に2種類搭載してある.2HD の媒体に対しては,磁気ヘッドが接触することが必要であり,特に高容量媒体で浮上する形態の装置では,HDI(head disk interface)に十分な検討が必要となる.また,磁気ヘッドの移動には直動型VCM(linear voice coilmotor)が用いられる.これは,互換のときに磁気ヘッドと記録トラックとの間で発生するアジマス損失(azimuth loss)によるエラーの発生を低減するためである.トラックの位置決めについては,2HD 媒体にはトラック位置を示す位置信号が書かれていないために別途スケールを搭載し,これを基準として位置決めがなされている.磁気ヘッドのトラック方向の位置調整(radial alignment)や,トラック00(最外周の基準トラック)の位置調整も行われる.その他,従来型のFDD と同様の調整もしくは確認が必要で,互換性確保のためのコストがかさみやすい.さらには,高容量媒体が従来型のFDD に装着されたときには,書込みが禁止されるようにWP(write protect)ノッチが設定されているとともに,高容量FDD 間で互換性がないのでそれぞれ異なる位置にWPノッチを設け書込みができないようなっている.回路系も高容量と2HD 媒体とに対応できるように設定されている.
