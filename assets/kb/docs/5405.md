# 3・5・1 光ディスクの動作原理と装置の課題

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・5 光ディスク装置

a. はじめに光ディスク(optical disk)は1980年初頭に音楽再生用のCD(compact disk,compact disc)、映像再生用のVD(video disk,video disc)として製品化された。特にCD は従来のレコードより音質、操作性に優れていたため5年間でレコードを置換えてしまった。さらに、1980年代後半からは、1回記録可能な追記型、続いて書換え型が登場した。これらは文書あるいは図面の電子ファイリングには好適な記録媒体といえる。最近のコンピュータ環境は分散処理化(オープン化と小形軽量化)、ネットワーク化、取扱う情報のマルチメディア化の方向に急速に動いており、大容量でランダムアクセスが可能な可換媒体であることを特徴とする光ディスクが必要とされている。さらに、低価格化を実現できたCD-ROM は、OS、アプリケーションソフト、あるいは百科事典等のデータベースの頒布用媒体として普及している。CD の後継として大容量のDVD(digital versatile disk)が登場し、情報化社会を支える可換媒体として期待されている。ここでは、光ディスクの概略とこれを支える機械技術について述べる。b. 光ディスクの記録再生原理と種類図3・30にCD 光ディスク装置の構成図3・31に、光ディスク装置の基本構成と原理を示す。大別して、光ディスクと、データを記録再生するための光ヘッド、およびそれを駆動してアクセス、トラック追従を行うためのアクチュエータから成り立っている。ここでは一例として、コンピュータ用として世界で初めての製品となった、追記型の12インチ光ディスクについて述べる(図3・32)。ディスクは直径300mmで、表裏2面の基板がスペーサを介して空気を挟んだサンドイッチ構造となっている。基板は、厚さ1.2mmのガラスであり、その上に情報記録面を有する。情報記録面はサンドイッチ構造の内側に封じ込められた形となっており、情報の記録再生はガラス板を通してレーザ光を絞り込むことによって行われる。ディスク面上にはピッチ1.6 m、幅約0.4 m、深さ約0.1 mで、トラック案内溝(プリグルーブ)があらかじめ形成されており、この上にTeを主体とする厚さ約300Åの記録膜が蒸着されている。トラックの1周は64セクタに分割されており、各セクタの頭にはアドレス信号、同期信号などがあらかじめ凹凸の穴(プリピット)の形で記録されている。データはプリグルーブ上にレーザ光を絞り込んで熱的に約0.8 mの穴をあけることによって記録され、穴の有無による反射率の違いによって再生される。光ヘッドは、半導体レーザを光源としており、大きく分けると、図3・32 光ディスク装置の全体構成データ記録再生(W/R : write/read)系、焦点信号検出系、トラッキング信号検出系の三つの部分から成り立っている。W/R 系は半導体レーザから出たレーザ光束をディスク上に直径約1.6 mの微小スポット(spot)として絞り込むものであり、半導体レーザ、カップリングレンズ、絞り込みレンズなどから成り立っている。通常、レーザ出力は、情報記録時には約15~20mW、再生時には3~5mW 程度で用いられる。ディスクは回転中に±200 m程度上下に振れており、つねにディスク上に安定して光スポットを絞り込むためには、絞り込みレンズを動かして上下振れに追従しなければならない。焦点信号検出系はこのための焦点ずれ(focusing error)信号の検出を行う。焦点ずれ信号検出系は、ディスクからの反射光束中に、球レンズ、円柱レンズ、ナイフエッジ、2等分割光検出器などを配置した構成となっている。さらにプリグルーブも、回転中に偏心しているために±60 m程度左右にずれるので、つねにプリグルーブ上に安定して光スポットを位置決めするためには、ガルバノミラー(garvano-mirror)を動かして追従しなければならない。トラッキング信号検出系は、このためのトラックずれ信号の検出を行う。トラックずれ(track-ing error)信号検出系は、ディスクからの反射光束中に、プリグルーブからの反射回折光を検出する光検出器を配置した構成となっている。データの記録再生を行うときは、以下の手順で行われる。まずデータを記録すべきトラック番地、セクタ番地が指定されると、リニア(コース)アクチュエータと光ヘッド内のガルバノミラーの組合せ動作によって、光スポットがアクセス(access)される。所定の位置へアクセスされると、トラックアドレス信号およびセクタアドレス信号を読み取って、正しいトラックかどうかを確認する。その後、レーザが高出力で発振し、所定セクタのプリグルーブ上にピットが形成される。データ再生時には、同様に所定のトラック、セクタへ光スポットがアクセスされ、ピットの読取りが行われる。次に、光ディスク装置の開発を行ううえでポイントとなる五つの課題を示す。まず、1番めは微小光スポットの安定形成技術である。±200 mで上下動するディスクに、半導体レーザからの光束を直径約1.6mの微小スポットとして焦点誤差±1 m以下で安定に絞り込む技術である。基本となる要素技術としては、W/R 光学系および焦点信号検出光学系、アクチュエータ、光点制御系が関連してくる。2番めは、光スポット高精度位置決め技術である。プリグルーブの偏心に対して±0.1 mの高精度で追従させるものであり、従来の磁気ディスクに比較してトラック密度の高い光ディスクにおいて生命線ともなるべき重要な課題である。ディスクの傾き、アクチュエータの摩擦などによってトラッキング誤差が大きくなるが、トラックずれ信号検出光学系、精粗の2段アクチュエータ方式、光点制御方式などにより、±0.1 mのトラッキング精度が実現されている。3番めは、高速アクセス技術である。光ディスクにおけるアクセスの最大の課題は、磁気ディスクの場合と違って、必要分解能が1.6 mと非常に高精細になっている点にある。このため通常は、外部スケールまたはトラックのカウントによるマクロシークと、光スポットをトラックジャンプさせるときのパルス数をカウントするミクロシークの組合せから成り立っている。さらにトラックをスポットが横切る数をカウントする方式が開発されている。4番めは、高信頼度のデータ記録再生を行うために必要な、S/N40dB 以上の高品質記録膜形成技術である。膜組成、下地膜などを工夫した高S/N 記録媒体が開発されている。5番めは、高精度案内溝形成技術である。案内溝の役割は光スポットを高精度に位置決めするための基準スケールとなるものであり、正確なトラックずれ信号を発生しなければならない。そのためには、光スポットが照射される溝または凹凸ピットの形状が高精度に形成されていなければならない。溝またはピットの形状を決める要因である原盤記録条件(レーザパワー、ホトレジスト厚など)の関係を明らかにして、記録条件を制御することにより最適トラックずれ信号、データ信号が得られる溝およびピット形状を高精度に形成する必要がある。以上、光ディスク装置の開発を行ううえでポイントとなる五つの課題を示した。ここでは、上記課題のうち、機械技術に関する第二と第三の課題の詳細およびその解決に必要な基本技術の開発について、以下に詳述する。
