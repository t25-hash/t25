# 3・5・2 光点制御技術

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・5 光ディスク装置

a.光点制御系の課題 記録可能な光ディスクにおいて，情報
の記録/再生を行うためにはディスク面上に光スポットを維持する
焦点合わせ制御と，特定の溝（トラック）の真ん中に光スポットを
位置付けるトラッキング制御の二つの制御機構が必要となる．
光ディスクの制御信号はディスクからの反射光量を用いて検出す
る．したがって記録時には半導体レーザからの発光光量が変化する
ため，記録時，再生時の利得切換えを行っている．再生時において
は，ディスクからの反射信号のピークのみを検出してこの制御信号
の変動を防いでいる．
b.自動焦点制御（autofocusing control）系 この制御系は
ディスクの上下振れに追従して，対物レンズを駆動し，対物レンズ
の焦点位置をディスク面に維持するものである．サーボ系としての
プロック図を図3･33に示す．大きくは制御信号検出部と電気補償
部と対物レンズの駆動部から構成されている．
焦点ずれの許容値は焦点ずれによって生じる信号振幅の限界値か
ら決める．記録，再生はそれぞれ独立動作であるので，それぞれの
焦点ずれによる信号振幅の限界値を求め，焦点ずれ許容値を例え
ば，±1.0mとする．この許容値内に自動焦点合わせの全体追従
精度を入れなくてはならない．
ディスク上下振れは，例えば最大振れ量±0.8mmと最大加速
度0.25G（Gは9.8m/s）の二つの表現で規定される．これを図
3･ 34に斜線で示す．この上下振れに追従するために，制御系の開
ループ特性は同図の一点鎖線のようにゲインを確保する．
