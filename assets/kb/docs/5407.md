# 3・5・3 アクチュエータ(actuator)

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・38 のブロック図を基に説明する。ディスク面上のトラック中心

ここでは光スポットをディスク上のトラックに焦点合わせし、アクセスおよびトラッキングするアクチュエータと機構について方式の概略と特徴について述べる。なお、以下の説明ではトラッキングはアクセスのためのディスク半径方向の広域移動とトラックフォローイングのための精密位置決め動作を含むものとする。a. アクチュエータの設計方法アクチュエータは一般的に所定の範囲内を任意に移動できること、必要な加速度を出せること、制御が容易なこと、所定の精度で姿勢を保つことなどが要求される。光ディスク装置に用いられるアクチュエータも同様であり、剛体を、入力に比例した駆動力により、摩擦なくまっすぐに移動させることを理想として設計しなくてはならない。ヘッドアクチュエータを設計する場合のフローを図3・47に、駆動部に注目したフローを図3・48に示す。実際のアクチュエータを実現するうえでは駆動あるいは、支持・案内するうえでの考慮すべき事項がそれぞれあるが、それらの技術を評価・比較する際の基本と考えられる4指標について説明する。第1の指標として加速度が挙げられる。加速度は移動時には所定の距離を目標時間内に移動できることが要求され、動作モード切換え時には所定の時間あるいは所定の距離の範囲内で通常の追従誤差以内に静定できる値でなければならない。また追従時にはトラックなど目標点の移動加速度以上の加速度を出す能力がなければならない。キャリッジアクチュエータではシーク時間の要求から最大加速度が決定される。第2の指標として副共振周波数が挙げられる。これは最高応答周波数を決定する指標と考えられるもので、おもに、アクチュエータ可動部の曲げ(bend)、ねじり(twist)モードなどの弾性振動(elastic vibration)の周波数であることが多い。ヘッドアクチュエータでは、ディスク回転速度が低い場合は20kHz図3・47ヘッドアクチュエータの設計フロー図3・48 駆動部の設計フロー以下の場合もあるが、一般には20kHz 以上にする。キャリッジアクチュエータでは、シーク時の速度制御系やトラッキング時のコースアクチュエータ制御系の応答周波数帯域に依存する。第3の指標としてクロスアクションが挙げられる。クロスアクションとしては焦点合わせアクチュエータを駆動したときの焦点合わせ方向の変位に対するトラッキング方向の変位の比をいうことが多いが、ここではキャリッジアクチュエータを駆動したときの、可動ヘッドの加速度に対する対物レンズのトラッキング方向の加速度の比、あるいはガルバノミラー(garvano-mirror)の回転による光スポットのトラッキング方向加速度の比も考えることにする。第4の指標として8―65アクチュエータの合計重量が挙げられる。可動部と固定部を合わせた合計重量はアクチュエータの移動範囲や最大加速度を大きくする場合に大きくなる。キャリッジアクチュエータの固定部は通常、スピンドルモータなどと同じベースに固定されるため、問題になることは少ない。ヘッドアクチュエータは可動ヘッドに搭載されているため、特に高速シークを目指すときに重要な指標になる。b. アクチュエータの方式選択(1) 駆動力の選択焦点合わせとトラックフォローイング用アクチュエータは、c項で述べた光点制御仕様を満たして、重量0.05~1g の対物レンズを駆動することが要求される。そのために焦点合わせとトラックフォローイングアクチュエータの駆動機構は0.1~1mm以上の可動範囲と50m/s以上の加速度、そして2kHz 以上の応答周波数を実現できるものでなければならない。電わい効果や磁わい効果を利用する推進機構では以上の仕様を満たすことは現在のところ困難であり、電磁力を利用した磁気アクチュエータが用いられている。磁気アクチュエータには可動鉄心型、可動磁石型、可動コイル型、の3種類があるが、多くは欠点の少ない可動コイル型アクチュエータ(VCM)を採用している。(2) 各アクチュエータ方式(a) 焦点合わせアクチュエータ焦点合わせアクチュエータは移動すべき範囲が±0.5mm程度と狭いため、小形の磁気アクチュエータで、必要な加速度と振動特性を比較的容易に実現できる。また、広域位置決め機構(キャリッジアクチュエータ)の負荷になるため極力小形軽量にすることが要求されている。(b) トラッキングアクチュエータトラッキング機構はリニアタイプとロータリタイプに大別される。ロータリタイプはオーディオ用の一部の製品と、コンピュータ用に試作された例があるが、大部分はリニアタイプである。トラッキングは16~75mmの範囲でトラックを±0.1 mの精度で追従しなければならず、広い移動範囲と高い応答性を一つの機構で満たすことは困難である。そのため広域位置決め用のキャリッジアクチュエータ上に焦点合わせアクチュエータと精密トラッキングアクチュエータ(ヘッドアクチュエータ)を搭載した構成をとっている。コンピュータ用などの短いアクセス時間を要求される装置では、トラッキングアクチュエータは全体として高い加速度と、剛性の高い構造が要求される。そのため、広域移動時に加速度が加わっても焦点合わせアクチュエータと精密トラッキングアクチュエータが振動しにくい構造にした2段型アクチュエータを採用する。ヘッドアクチュエータには焦点合わせアクチュエータと精密トラッキングアクチュエータを個々に組合わせる方式と、両者を一体化した二次元アクチュエータ(two-axis actuator)方式がある。前者は初期の光ビデオディスク装置から用いられている方式であり、トラッキングはガルバノミラー、焦点合わせは可動コイル型のリニアアクチュエータを用いるものが一般的である。その例を図3・49に示す。c. キャリッジアクチュエータキャリッジアクチュエータ(carriage 個別アクチュエータ個別アクチュエータ方式に用いる焦点合わせアクチュエータは可動コイル型リニアモータをばねで支持するものが大部分である。支持ばねは4本程度の細長いばねを均等な角度に平面配置したものを上下に設けたタイプと幅広の平行板ばねのタイプなどが用いられている。精密トラッキング用アクチュエータとしてはガルバノミラー(garvano-mirror)が多く用いられる。可動磁石型のものもあるが、大部分は可動コイル型であり、可動部の先端に力発生部を設けると駆動トルクを大きくできる。ガルバノミラーは対物レンズの焦点距離にもよるが、一般的に駆動部での等価質量が小さいため、光スポットの加速度感度を大きくすることができ、高速応答に適している。また等価質量が小さいため、低周波数の範囲に対して電磁ダンピングを用いることができる。(2) ベース・スピンドル機構光スポットをディスク上のトラックに正確に位置決めするには、光スポットの追従性を上げるだけでなく、ディスクをできるだけ振れないように回転させることと、可動ヘッドが移動したときにディスクが振動しないように、ベーススピンドル機構を構成することが重要である。以下、考慮すべき点8―67をまとめる。ディスクを回転させるスピンドルモータは回転軸の上先端部にディスクのセンタを合わせ、取付けるハブが設けられており、ボールベアリングで支えられている。ハブにセットされたディスクのセンタ穴とスピンドルセンタ軸とのすきまによりディスク中心が偏心し、スピンドル回転数に同期したディスク半径方向のトラック変位と加速度が発生する。これに加えて薄型の装置では上下のボールベアリングの間隔が短いため、軸の倒れに対する剛性が小さく、軸の倒れによる振動が出やすい。トラックの半径方向の数百Hz 以上の成分にはボールベアリングの玉通過振動や回転駆動力のアンバランスなどが影響することもある。ディスクは基板の材質、板厚、直径がその振動特性に大きく影響する。現在の光ディスクの基板の材質はポリカーボネートが多く、ガラスも一部に用いられている。ポリカーボネートのディスクの最低次の固有振動数は100~200Hz であり、4~500Hz 以上では多くの振動モードが存在している。そのためスピンドルモータやベースと連成した振動が発生する。ガラスでは剛性が高く、固有振動数が高いため比較的振動が問題になることは少ない。ベースは可動ヘッドが移動したときの反力を受け振動する。ベースの振動はベースのみでなくスピンドルモータやディスクも合わせて振動するため、結果的に可動ヘッドが移動すると、ディスクも振動する。ずれの許容範囲が焦点合わせ方向に比べて小さいトラッキング方向には、この付加的なディスクのトラック方向の振動が影響する。スピンドルモータがベースの先端に設けられることが多いため、ベースが曲がるモードではスピンドルモータを加振する形になり、振動が問題になることが多い。
