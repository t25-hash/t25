# 7・2・5 作業機装着・昇降および制御システム

> **章**: 第7章 農業・水産機械
> **節**: 7・2 トラクタ

ここでは作業機としておもに直装タイプのロータリを想定する．
ロータリは三点リンクヒッチ（three-point link hitch）を介してト
ラクタに装着され，昇降される．日本の水田作での耕うんは大半が
ロータリ耕であり，しかもほ場が狭いため，代かきも含めてほ場へ
の出入りや枕地旋回のときにロータリ部があぜや畝に当たらないよ
うに，すばやく高く上げねばならない．そのため，JISやISOな
どによる規格とは別の三点リンクヒッチが考案されている．図7･
11 はトラクタ側上部リンク取付点と上部リンクの間に回動アー
ム（リンクアーム）を取付けたものである．こうすることによって
リフトアームを上げるとリンクアームを介して上部リンクは本機側
に引き寄せられ，ロータリの地上高を大きく稼ぐことができる．
三点リンクへの作業機の脱着・交換作業は熟練を要し，時間のか
かる厄介なものであったが，これをトラクタに乗ったままワンタッ
チで行う自動ヒッチ部が開発され，作業機の脱着のみならずPTO
軸からの回転軸の接続も自動的に行うことができるようになってい
る．回転軸のクラッチ部には爪クラッチが採用され，これによって
装着時に自動的にクラッチが入り，作業機をつり上げるだけで作業
を開始することができる．

油圧を利用したポジションコントロールのリンク機構作用図（図
7･12）において，ポジションレバーをAの位置からBの位置へ

その結果スプールアームを介して油圧制御弁のメインスプールが右

したリフトアーム回転変位を得ることができ，作業機の位置が定ま
る．

いて自動的に油圧レバーを上下させて耕深を一定に保つ方式が用い
られている．
