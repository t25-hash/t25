# 3・6・1 概説

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・6 次世代記憶装置

現行の主たる外部記憶装置としての磁気ディスク装置、光ディスク装置の性能を凌駕する記憶原理や機構に基づく新記憶装置出現の兆しはまだない。次世代記憶開発のおもな動機は、いうまでもなく磁気記憶における常温下の熱揺らぎ(thermal decay)に起因する磁化消失や、光記憶の回折限界に伴う集光スポットの狭小化限界を見据えての記憶性能の格段の向上要求による。加えて、1980年代初頭に発明された走査型トンネル顕微鏡(STM : scanning tunne-ling microscopy)の要素技術を基に、トンネル電流以外のさまざまな物理量を検出・可視化できる走査型プローブ顕微鏡(SPM :scanning probe microscopy)が発展してきていることが挙げられる。ここでは先鋭なセンサ探針により表面修飾された微小な点状パターンをディジタル記憶情報に対応させることで超高密度記憶を予感させる最先端成果を種々の研究機関が示し始めたことにより、SPM を活用した記憶応用諸研究が相互に刺激し合いつつ深化している。装置化における基本的な要件は、単に記憶密度のみならずデータ授受の速度(転送速度)、総記憶容量、補器類を含む体積記憶密度、データ(記憶)の安定保持性、耐衝撃性、さらに、装置の用途によっては書換え・追記性、また読出し専用(ROM)であれば媒体の作製や複製の容易性などが挙げられる。図3・50に磁気ディスク装置、光ディスク装置の代表性能としての面記録密度とデータ転送速度を、次世代、次々世代の記憶として検討されている各種方式と比較して示した。現行装置は一般に高密度かつ高速のもの(図3・50の右上)ほど新しいが、次世代記憶方式については実証形態にさまざまな特徴がありこの傾向が必ずしも当てはまらないため、図3・50中に文献などの発表年次も併記した。次世代方式の動作原理は
