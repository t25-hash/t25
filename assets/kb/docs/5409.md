# 3・6・2 項および3・6・3項にて詳細に述べるが、1990年代後半から特

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・6 次世代記憶装置

に性能向上が著しい磁気記録(磁気ディスク)装置においては、一時期、年率120%ものまさに爆発的な面記録密度の上昇を示した経緯があり、技術難度が指数関数的に高まってきた今日でもなお、30~60%程度の密度増加を堅持している。2004年時点ではすでに8―68情報・メディア機器商用機において60Gb/in弱の密度が実用され、また実験室段階では150~200Gb/inの密度が検証され始めている。この記憶密度の技術動向に従えば、2005年中には100Gb/inを超える商用機の登場も十分考えられる。一方、次世代の記憶方式に関しては、多くがSTM に準拠した形で基礎検討がなされており、それらの代表的なものの面密度、転送速度を可能な範囲で文献に忠実に見積もり記載した。しかし、既述の通り磁気記憶の過激な密度向上により、かえって次世代記憶方式の実証性能が現行装置の性能を下回る事例も出てきている。これは、必ずしもそれらの原理的な潜在性能の低さを示唆しているのではなく、個々のセンサ・ヘッド機構や記録媒体などの要素部品の試作・開発において、商用磁気装置のような格段の安定性や性能の極限追求を可能とする底辺技術基盤がまだ十分成熟していないか、あるいは利用できないためと考えられる。次世代記憶も、STM や原子間力顕微鏡(AFM : atomic force microscopy)など原子・分子オーダに及ぶ実空間分解能を有する手法を用いた検討例では、それらの密度において現状の1.5けた以上に及ぶ圧倒的に高い性能上の優位性を示している。しかし、その一方でデータ転送速度など応答性能は現行所要値の1~3けた以下と極度に低く、実用性能としては極めて不十分なものである。書換え性のあるもの、またはディスク回転系による検討例のあるものは、実証密度性能では現行装置レベルにとどまっているが、速度的には1MHz から数MHz前後であり、近未来の装置化に向けた潜在能力としてはかなり高いものもある。
