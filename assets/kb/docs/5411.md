# 3・6・3 再生方法

> **章**: 第3章 記憶装置・記憶メディア
> **節**: 3・6 次世代記憶装置

図3・54に再生方法を整理して示す.これらは図3・51に示す記録方式とは必ずしも一対一の対応はしていない.まず,追記方式または再生専用(ROM)方式として,原子オーダの凹凸や原子クラスタ(塊)については導電性探針を用いたトンネル電流検出方式が,また絶縁材料を含めた凹凸に関してはAFM 探針による表面形状の8―69(a) 原子移動(b) 原子はぎ取り(c) 原子またはクラスタ堆積(d) 切削(e) インデンテーション(e′) ナノインプリント追従方式(片持ばりのたわみ検出)が用いられる.また,光の波長を下回る超微小開口(aperture)近傍に定在する光の場(近接場,near-field)を用いた再生も可能である.書換え方式のうち,導電率変化方式に関してはトンネル電流検出と,導電性探針を接触しゅう動させての接触コンダクタンス検出の二つの原理が用いられている.誘電分極や電化蓄積方式では,導電性プローブ(探針)と対向電極となるp型半導体基板との間の空乏層の大小に起因する静電容量の変化を検出する.光磁気方式では,微小開口近在の近接場を点光源とする散乱伝搬光の偏光回転を,垂直磁気方式では磁気力こう配検出などの原理が用いられている.また,相変化媒体に透過率(反射率)変化として記録されたピットは,微小開口による近接場の散乱・透過光強度変化により検出できる.
