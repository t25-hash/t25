# 4・1・1 マンマシンインタフェースとは

> **章**: 第4章 入出力装置
> **節**: 4・1 マンマシンインタフェース

マンマシンインタフェース(MMI: man machine interface)とは、機械と人間との情報の共有、伝達手段を指す。保守管理も含めて完全に自動化された機械システムを除いて、すべての機械は必ず人間からの指示、操作を必要とし、それに基づいて行われた動作の結果を人間に報告しなければならない。したがって、機械を使って効率的な作業を実現するには、人間から機械への指示の伝達、あるいは機械から人間への作業結果の報告において、最も人間にとって快適な手法、手順を開発する必要がある。マンマシンインタフェースとはこのための学問体系である。特に対象とする機械をコンピュータと考えた場合、マンマシンインタフェースの同意語として、ヒューマンコンピュータインタフェース、あるいは単にユーザインタフェース(user interface)という言葉も使われている。一方、近年、マンマシンインタフェースを拡張した概念として、ヒューマンヒューマンインタフェース(HHI: human humaninterface)が提唱され、使われ始めている。これは、マンマシンインタフェースでは、人間と情報を共有する対象は機械であり、機械への快適かつ確実な情報伝達に研究の主眼がおかれる場合が多いのに対し、ヒューマンヒューマンインタフェースでは人間と人間、人間と社会などの情報の共有を目的とし、機械はそのための手段にすぎないという考え方である。このため、単なる操作性の向上だけではなく、人間特有の不確実性やそれを予見する仕組み、あるいは情報伝達により生じるお互いの心の動きなど、人間特有の感覚評価も取入れる必要がある。
