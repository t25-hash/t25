# 7・4・2 刈取・搬送部

> **章**: 第7章 農業・水産機械
> **節**: 7・4 コンバイン

自脱コンバインの断面を図7･37に，刈取部詳細を図7･38に示
す．引き起こし装置は対地角約70度に取付けられ，ピックアップ
タイン（pick- up tine）によりイネをかき揚げるようにもち上げ，
スターホイールで茎の下部をまとめ，株元搬送チェーンで保持して
図7･ 37 自脱コンバイン構成要素（ヤンマー農機)
往復動刃で切断する．通常，固定式の受刃と動刃の組合せが用いら
れるが，振動低減のため両方の刃が移動するものも用いられる．ま
た，柔らかいイネの茎の切断に適するようにのこぎり刃が使用され
る．刈取後は，搬送チェーンで挟んで後部に送り，フィードチェー
ン（feed chain）に受け渡し，茎の端部を挟持したまま穂先だけを
こぎ室内に供給する．自脱はこぎ胴と穂の相対位置が重要なため，
フィードチェーンが最適の位置で茎を挟持できるよう搬送チェーン
の位置を自動制御して，こぎ室内での穂の位置を調整するこぎ深さ
制御が行われる．また，多くの機種に刈高さ制御装置や方向制御装
置が装着されている．倒伏しやすいイネを刈り取り，茎の端部を挟
持しながら搬送チェーン，フィードチェーンと受け渡していく機構
は世界に例のないもので，開発初期には大変な苦労があった．
これに対して，普通およびインディカイネコンバインには通常，
リール（reel ）が使用される．現在は偏心スパイダ機構によりリー
ルが回転してもタインはつねに垂直に保たれるピックアップリール
が使われる．刈り取られた作物は，プラットフォームオーガで中央
に集められ，コンベアでもち上げられ，脱穀部に供給される．投げ
込み式であるため構造は簡単である．
