# 4・1・3 ロボット(robot)におけるマンマシンインタフェース

> **章**: 第4章 入出力装置
> **節**: 4・1 マンマシンインタフェース

コンピュータ以外のマンマシンインタフェースの例として、人間とロボットとの間のインタフェースを取上げる。人間とロボットが直接的に接する例として、以下のような場合が考えられる。(1) 組立ロボットへの作業教示(2) ロボットと人間の共同作業(3) 医療、介護、福祉分野でのロボットによるサービス組立ロボットへの作業の教示手法としては、教示者が直接ロボットに接し、作業内容を教示するダイレクトティーチング、ティーチングペンダントやコマンド入力によるリモートティーチングが代表的である。一方、近年ではロボットに学習能力をもたせ、テレビカメラなどを通じて教示者が行う作業を観察、理解し、自動的に行動プログラミングを作成する模倣学習(teaching by showing)の研究が盛んになってきた。一方、医療、介護、福祉分野において、人間に代わり患者や要介護者などへさまざまなサービスを行う、いわゆるサービスロボットが注目されている。これらのロボットで最も重要な機能は、不確実で予測困難ないかなる人間の行動に対しても、つねにサービスを受ける者の安全を100%保障する機能である。このためには、操作者による指令の発行と処理を基本とした従来のフィードバックタイプのマンマシンインタフェースだけではなく、サービスを受ける側の行動、状態をつねに監視し、必要に応じて操作者の指令に介入し、行動を制限するフィードフォワードタイプのマンマシンインタフェースの実現が不可欠となる。
