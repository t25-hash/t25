# 4・1・4 良いマンマシンインタフェースの設計

> **章**: 第4章 入出力装置
> **節**: 4・1 マンマシンインタフェース

機械と人間との良いマンマシンインタフェースを設計するには、操作性、安全性、デザイン、保守性などさまざまなファクタを考慮する必要がある。例えば操作性を向上するには、機能を直感的に推測できる記号付けや提示場所、あるいは色の選定にも注意を払わなければならない。またコンピュータが処理できる情報量と人間が処理可能な情報量との差が次第に大きくなってきており、人間が状況を判断するのに必要十分な情報だけを選別し、提示する仕組みも重要な開発項目となっている。ポイント入力機器とは、コマンドやデータを音声によらず直接的にコンピュータに伝達するマンマシンインタフェース機器の一種である。身近なところでは、文字を入力するキーボードや画面上のカーソル位置を制御するマウスが該当するが、近年、コンピュータの多様化に伴い、さまざまなタイプのものが開発されている。ポイント入力機器を、文字入力とポインティングという2大アプリケーションで大別すると図4・1のようになる。
