# 7・5・2 青果物選果包装施設

> **章**: 第7章 農業・水産機械
> **節**: 7・5 農業施設

選果包装施設（packing house）とは生産者が収穫したさまざま
な大きさや品質の混ざり合った青果物を集荷し一定の品質等級と階
級規格に従って選別し，所定の荷姿に包装して販売するための商品
を作る施設である．選果包装の工程は自動化されており，すべてコ
ンピュータにより管理されている．一般的な工程は，荷受，一時貯
留，供給，前処理，等級選別，階級選別，包装，箱詰，計量印字，
封函，搬送，出荷で，大型施設では製品判別自動出荷の立体倉庫を
備えているものも増えている．前処理には青果物の種類によって異
なるが，洗浄・清浄，除袋・輪切り，ワックス処理などがある．
1975年頃には多くの選果場が建設されその数は大小合わせると5
000以上もあったが，現在は統合され，中・大型の自動化された施
設が建設されるようになり，約1500施設に減少している．特に
1992年以降は近赤外分光法を用いた内部品質測定装置が開発され，
この内部品質測定装置を設備した施設がこの10年間で急増してい
る．また，これらの施設の大半はカラーカメラとコンピュータ画像
処理技術により，IPX0.5mm角以下の傷害まで計測できる外部品
質測定装置と合わせて設備されている．階級選別はRGB分光カメ
ラのコンピュータ画像処理によって色や傷害計測と同時に測定判別
が行われており，これまでの電子天秤を用いた重量選別機は極めて
少なくなってきている．
内部品質測定装置による計測の対象青果物は，柑橘類，落葉果
樹，メロン，スイカ，トマトとほとんどすべての青果物が対象とし
て計測できるようになり，糖度，酸度，熟度，水浸果，蜜入り果，
褐変果などの青果物に応じた項目の測定が行われている．
a.選別機 現在（2002年）の選別機の種類を見ると，日本国
内では大別して以下のように分類される．
.フリートレイ方式 果皮の薄い，傷みやすい落葉果樹や
つるの付いた高級メロン等を中心にトマト，スイカ等の各種果実
は，皿状の容器（フリートレイ）を用いるフリートレイ方式が主流
となっている．この方式は，測定されたデータを容器に取付けられ
たバーコードや磁気メモリと制御コンピュータで等階級を管理し所
定位置に排出する方式である．排出後はロボット箱詰機による自動
箱詰が主流である．フリートレイ方式の特徴は，果実がトレイに載
ったままで選別，仕分，搬送されるので，選別工程で完熟した果実
も損傷を受けることがなく，測定，選別できるところにある．
.PKコンベア方式 柑橘類で使われている．ホルダによ
って支持されたピアノの鍵盤に似た片持式のプレートコンベアで，
人の指幅程度に分割されたキーが，連続した直線プレートコンベア
を形成し，その上に柑橘類が自動供給される．ホルダを押すことに
よって支持板が傾動する構成となっており，果実の大きさによって
キーを必要枚数傾動して果実を排出することができる．このタイプ
は，フラットでカラーカメラに対し反射防止機能を設けたものと，
果実を回転させて計測させるためのローラタイプのものとがある．
柑橘用のこれらの選別機にも，RGB分光カメラと近赤外分光測定
機が設備されている．
.その他の方式 バケット式選別機は10年程前までは，
落葉果樹を中心とした選別機の主流であったが，現在は個人農家用
や，小さな施設にのみ用いられているだけでほとんどなくなってい
る．機械式形状選別機もバケット式と同様主流ではなくなった．
b.近赤外分光法による内部品質測定 この技術は，青果物に
測定光を照射し，その内部拡散光をスペクトル分析し，コンピュー
タの統計解析法により内部品質を定量分析，または定性分析を行う
技術で，初期1990年から1995年までは反射光測定法が中心であっ
たが1994年に開発された透過光測定法（図7･49）が以降中心にな
った．近赤外分光法とは，一定の測定光を青果物に照射し，その内
部透過光を分光スペクトルとして計測し，その青果物を破砕し，実
測データを得る．この分光スペクトルと化学分析により得た実測デ
ータをコンピュータによる統計解析を行い，目的の成分と相関の高
い分光波長を選択し成分分析を行う統計解析数式（検量線）を得る
方法である．一成分当たり約300程度果実がサンプル果として検量
線を作るために用いられるが，その後は，測定装置を通過させるの
みで内部成分を非破壊計測できる技術である．この測定法により，
青果物の糖度，酸度，熟度，渋，水侵果，褐変果等の計測ができる
ようになった．現在，オンライン計測で毎秒5～7果実が計測可能
である．
c.外部品質測定技術 RGB分光カメラによるコンピュータ画
像処理法による選別方式がとられて7～8年が経過しているが，現
図7･50差圧予冷庫（中央吸込方式）
図7･ 51 播種プラント
在は，RGBまたは，RGIR分光カメラを使って，形状だけでなく
青果物の傷や色判別までも同時に行うものが通常である．
d.その他 落葉果樹類から玉ネギ，バレイショのような土物
まで選別する各種の選果施設には対象物によりさまざまな施設があ
り，また，各種のレイアウト設備が予算や規模によって設けられて
いる．例えば，汚れ，ほこり，皮等を除く洗浄・清浄機械や，果実
につやを出すためのみがき機，ワックス処理機，乾燥機，段ボール
箱を組立てる製函機，その他ロボットによる自動箱詰機，自動封凾
機，オートダンパ，デパレタイザ，パレタイザなどの各種の機械が
使われている．
