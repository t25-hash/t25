# 4・2・3 ポインティング機器の方向性

> **章**: 第4章 入出力装置
> **節**: 4・3 )ではキーを打つたびに予測とあいまい検索を行い、入力語候

以上，さまざまなポインティング入力機器に関しての現状を述べ
た．今後の方向性としては，以下の点が挙げられる．
(１) ユビキタス（ubiquitous）化
今後のIT端末はモバイルやウェアラブルが中心となるため，省
スペースでユーザの意思を伝達可能なポイント入力機器およびシス
テムがますます重要となる．特に文字入力に関しては，子供から高
齢者までが使えるユニバーサルなデザインが望まれる．
(２) 多機能化
触覚マウスのように，ポインティング機能だけでなく，他のモー
ダリティを提示したり入力する機能との組合せも，臨場感を高めた
り，無意識的にユーザからプロファイル情報を取得するうえで重要
な技術となる．
