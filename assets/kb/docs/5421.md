# 7・5・5 育苗施設

> **章**: 第7章 農業・水産機械
> **節**: 7・5 農業施設

共同で多量かつ省力的な作物の育苗を行うのが育苗施設（nurs-
ery house）である．育苗は種子の前処理，播種，出芽，緑化，硬
化などの作業工程からなっている．前処理では種子の選別，消毒，
浸種，催芽が行われる．
a.催芽機 種子の発芽をよくし，そろえるために30℃程度に
加温して鳩胸状に発芽させる．温水式と蒸気式がある．
b.出芽装置 育苗箱を台車などに積み重ねたものを保温カバ
ーで覆い，適切な温度，湿度に保って均一な発芽を促す．
c .播種プラント（図7･51) 育種箱への播種作業を行う．育
苗箱をコンベヤ上で移動させ床土詰め，播種，覆土，潅水を能率的
に行う．作業は自動化されている．
d.その他 床土乾燥機，砕土器，肥料混合機，脱ぼう（芒）
機，育苗箱消毒洗浄機などが使われている．
