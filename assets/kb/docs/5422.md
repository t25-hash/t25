a.自動化のニーズ 農業機械は，大自然のもとで変化に富ん
だ土壌や作物を対象に，広大なほ場で営まれるという農業の特質に
対応して，走破性に優れた移動機能，変化への柔軟な適応性，1人
でより広い面積を処理する作業能率などを併せもつように進化して
きた．反面作業者は，場所ごとに作物や土壌の条件に合わせて作業
機の調整を行いながら，なるべく早い速度で作物列などの目標経路
（target path）に合わせて機械を操縦することとなった．このため
作業者は，ともすれば超人的な負担を課せられることになり，その
解決のため，作物列に沿った運転操作や作業機の調整などが自動化
されるようになった．また，シーズン中は昼夜連続して運転される
穀物乾燥機のような屋内で使用される機械も，プラント設備などと
同様に作業者による監視や操作を必要としないように自動化が進め
られてきた．以下に，代表的な農業機械における自動化の例を示
す．

c.コンバインにおける自動化 コンバインのオペレータ
は，稲株列（条）に合わせた操舵操作ばかりでなく，脱穀部の負荷
た統括的制御を行いうるよう世界的にはCAN（control area net-
work）の導入が進みつつある．おもな自動化項目を図7･52，表7･
2に示す．
トラクタは多段の変速機を備え，作業はおおむね一定の速度で行
われる．このため負荷が変動しても設定したエンジン回転数を維持
するようガバナが備えられている．ガバナは今日，機械式に代わっ
てエンジン回転，スロットル開度，ガバナラック位置などを検出し
て適正値を演算し，噴射料等の制御を行う電子式が用いられるよう
になった．これにより機械式では考えられないきめ細かな制御が可
能になり，トルクカーブの設定に自由度が増して限界負荷特性の向
上と高出力，低燃費，排出ガスのクリーン化を両立させることが可
能になった．変速機は，作業速度の細かい設定が要求されるため，
クラッチ操作の自動化とともにHSTによる無断変速機も増加傾向
にある．クラッチ操作の自動化は，レバー操作されるシフタギヤの
動きに感応して油圧制御される電子制御（electronic control ）ク
ラッチによって，主変速操作，二輪/四輪駆動切換や前後進切換
（シャトル変速）を足踏みクラッチ操作なしで行えるものである．
電子制御化により，エンジン回転センサおよびスロットル開度セン
サからエンジン負荷を推定して，クラッチ圧力を最適に制御し，シ
ョックの少ない変速を行うことも可能になった．また，旋回時に前
輪回転数を増速させて旋回半径を小さくする倍速機構の操作も同様
な機構で電子制御化され，舵角が一定値以上になったときの自動倍
速機能による作業能率向上とともに，車速が一定以上になったとき
の自動倍速切断機能による安全性の向上が図られている．
トラクタ側からの作業機制御は，おもに作業機昇降用油圧機構
（三点リンク）のリフトアーム昇降用とリフトロッド伸縮用の2系
統の油圧シリンダを制御することによって行われる．作業機制御
は，作業の目的に応じて位置制御（position control ），耕深制御
（tillage depth control ），水平制御（leveling control ），けん引力
制御（traction control ）などが自動化されている．位置制御では，
に応じた速度変更，刈取部クラッチの入/切，作物やほ場の状況に
合わせた刈高さやこぎ深さの調整など多くの操作を同時にしなけれ
ばならず，作業者にとって負担の大きいものであった．これら複雑
な操作を適切に，かつ組合わせて自動的に行えるよう各種自動化機
構が開発されてきた．また自動化と同時に，ランプ，ブザー，ディ
スプレイなどによって各部の状態をモニタし，異常を警告するとと
もに，統合的な制御が行われるようになったことはトラクタの場合
と同様である．さらにコンバインでは，カッタや回転部などによる

作動するよう特に配慮されている．自脱コンバインにおけるおもな

d.乾燥機における自動化 処理能力の高い共同乾燥施設な
どでは，荷受から貯蔵，搬出までの一連の作業が自動化されプラン
ト化されているが，ここでは個別農家や小規模な共同利用施設で使
用される比較的小型の米麦用循環式乾燥機における自動化について
述べる（図7･54）．
循環型乾燥機で最適な乾燥を行うためには，水分センサ（mois-
ture meter ）により水分計測を行い，熱風温度と穀粒の循環量（休
止時間）などを制御する必要がある．水分センサは，バケットコン
ベヤ途中からサンプリングされた穀粒がシュートによりローラ電極
間に導かれ，電極間の電気抵抗を測定することで水分を推定する．
熱風炉内には温度センサ（thermal sensor ）が設置され，バーナの
ON/ OFFを繰返すことで熱風温度の制御を行う．バーナには炎の
有無を検出する炎センサ（flame sensor ），灯油の噴霧圧力を検出
する圧力スイッチ（pressure sensor ）があり，失火した場合の自
動再点火が行える．運転の際には，穀粒の種類（もみ，コムギ，オ
オムギ，ビールムギ等），仕上り水分を指定することで最適な乾燥
温度，乾燥時間が設定され自動運転を行うことができる．
