# 4・3・2 項の電子スチルカメラに対する基本的な違いは,原稿の画

> **章**: 第4章 入出力装置
> **節**: 4・3 映像入力機器

像情報を詳細に読み取るため原稿と読取りの構成要素とを機械的に相対移動(走査)する場合がほとんどであること,原稿を所定位置に設置すること,原稿を照明する専用の照明系を搭載することである.a. 構成要素と走査読取りは,図4・10に示すように照明系,結像光学系,光電変換素子,信号処理部という構成要素で行われ,二次元画像の読取りのために機械的な走査と光電変換素子の配列とが組合わされる.それぞれの構成要素は読取り密度(解像度),読取り速度,濃淡情報の精度,倍率精度(寸法精度)などについて,各用途でスキャナに要求される性能により選択される.機械的な走査を行わずに光電変換素子を二次元に配列したエリアセンサ(area sensor)を用いるカメラ型と呼ばれる方式は,光電変換素子を配列する密度に限界があることから簡易的な用途に使われるのみであり,光電変換素子を一次元に配列したリニアセンサ(linear sensor)を用い,機械的に一次元の走査を行うものが主流である.一部の高性能が要求されるものについては二次元の機械的走査が行われる.一次元または二次元の機械的走査を行う場合,読取りの構成要素は,ライン状の領域,または1点の読取りに特化した専用設計がされ,機械的走査と同期して時分割で読取りを繰返す.読取りが時分割で行われるため,機械的走査の際に生じる振動が読取り画像の品質に大きく影響する場合があり,スキャナの機械系について位置精度とともに振動が重要な評価パラメータとなっている.b. 照明系スキャナの照明系には,原稿の濃淡情報を正確に読み取ることができる照明方向であること,読取り範囲を均一に照明すること,機械的走査の時間内に光量が安定していること,高い光量であることなどが要求される.紙などに記録された反射光で観察をする原稿に対し,照明系は光が斜め方向から照射するように配置される.これは光電変換素子に原稿面の正反射光が入射するのを防ぎ,正確な濃淡情報を読み取るためであり,照射角度は測色計と同様に45°程度が用いられる.フ図4・10 スキャナの構成要素ィルムなどの透過光で観察する原稿に対しては,照明系は原稿に対して光電変換素子の反対側に結像光路を延長した位置に配置される.機械的に一次元の走査を行うスキャナでは,一度に読み取るライン状の領域を均一に照明するため,ラインハロゲンランプ,蛍光ランプなどのライン状の光源が一般的に用いられる.また小さな発光部の光源を多数配列してライン状に照明するLED アレイなども用いられる.また発光面積が小さい光源から,ライン状の領域に光を導く導光体を利用した方式も実用化されている.導光体を利用する場合,透明な棒状のガラスあるいは樹脂の導光体に端面から光を入射させ,導光体の側面の全反射により光を伝達させ,その導光体の一部に設けられた拡散材で光を散乱させて照明を行う.機械的に二次元の走査を行うスキャナでは,1点を強力に照明するためにハロゲンランプなどの発光点を小さくできる光源と集光レンズが照明系として用いられる.c. 結像光学系スキャナの結像光学系では読取り領域の全体にわたって解像力,光量伝達率,わい曲などの結像性能が均一であることが求められる.リニアセンサを用いる場合,所定の位置に設置された原稿のライン状の領域を結像する光学系が必要であり,その方式は縮小結像方式と等倍結像方式に分類される.. 縮小結像方式図4・11(a)に示すように結像レンズを用いて原稿を縮小してリニアセンサに結像する方式であり,結像レンズには球面レンズを組合わせたものが用いられる.結像レンズは設定された光路長と倍率での原稿幅全体の結像性能を高めるように最適化がされる.縮小結像方式はリニアセンサを小型とすることができ,また光路長や倍率,および結像レンズのF値を含めた光学設計によって結像性能を高めたり被写界深度を深めたりすることができるなど,設計の自由度が高い方式である.ただし,画角ωが大きくなるほど両端部でのレンズの収差の影響が大きくなり,また結像レンズの光量伝達率は光軸に対する傾角θの余弦(cos θ)の四乗に比例して低下するため(コサイン四乗法則),十分な性能を確保するうえで画角ωを小さくすることが望ましく,長い光路長が必要となる.なお,g・i項に示すミラー移動方式を含め,縮小結像方式では長い光路を小さな装置に収める目的で,ミラーによる光路の折返しが行われる.折返しに使用するミラーの平滑度やわん曲は結像性能に影響するため,高い平面精度が必要である.. 等倍結像方式レンズアレイを用いて等倍結像を行う方式であり,等倍結像素子としては図4・11(b)に示すような屈折率分布型レンズを多数配列したロッドレンズアレイ(rod lens array)が主流である.屈折率分布型ロッドレンズではロッドに入射した光が正弦波状に蛇行することで結像が行われる.等倍結像方式では原稿と同じ幅のリニアセンサが必要であり,レンズアレイの特性により被写界深度が浅いものとなるが,光路長を短くできるため,小形のスキャナに用いられる.(a) 縮小結像方式の断面図(b) 等倍結像方式の例8―79d. 光電変換素子一次元に光電変換素子を配列したリニアセンサとしては,光電変換素子のフォトダイオードを多数配列し,各フォトダイオードの出力信号をCCD やCMOS デバイスなどで読み出す電気的走査機能をもたせたものが用いられる.リニアセンサは通常のICプロセスで製造が行われるが,原稿と同じ幅のリニアセンサを1チップで製造することは困難であるため,等倍結像方式ではチップを多数並べるマルチチップセンサが用いられる.マルチチップセンサの場合,チップのつなぎ目の不連続性を少なくすることが製造上の課題である.光電変換素子の評価パラメータは感度,感度均一性,出力信号のリニアリティ,雑音(ノイズ)特性などであり,温度上昇とともに増大する熱雑音を少なくするために冷却を行う場合もある.高性能が要求されるスキャナでは,高感度で広いダイナミックレンジを備えた光電変換素子である光電子増倍管(フォトマルチプライヤ)が用いられる.e. 信号処理部光電変換素子から出力されるアナログ信号は,信号処理部でアナログ/ディジタル変換(A-D 変換)がされ,そのディジタルデータに必要な補正演算を行って出力される.光電変換素子の出力信号は,正確に入力光量に比例するものではなく基準値による補正が必要である.その補正を行うため,光が入射しない状態の出力信号と,基準となる濃度を読み取った際の出力信号とを用いた補正演算が行われる.リニアセンサを用いるスキャナでは,リニアセンサに多数配列されている光電変換素子の出力信号を基準となる濃度の出力信号によって個々に補正演算することで,個々の光電変換素子の感度むら,照明系のむら,および結像光学系の光量伝達率のむらについての影響が補正される.この補正はシェーディング補正(shading cor-rection)と呼ばれ,スキャナで行われる最も一般的な補正演算である.光電変換素子の出力信号は,シェーディング補正のほかに,階調性を調整するγ補正,精鋭度を調整するシャープネス処理などが,そのスキャナに用いられている構成要素の特性に応じて行われる.また,濃淡のデータを二値のパターンに変換する二値化処理なども用途によって実施される.信号処理部で使用される素子については,A-D 変換を行うA-Dコンバータや各種補正演算処理をする素子,さらにスキャナの制御を行う回路素子を集積するIC化が行われており,また補正演算の一部をスキャナ側で行わずデータを受取ったコンピュータ上のソフトウェアにより実施することも行われている.f. カラー画像読取りカラー画像の情報は,一般に赤(R),緑(G),青(B)の3色の濃度情報により構成され,その3色に色分解した信号を読み取る方式には,一つの光電変換素子に入射する色を順次切換える方式と,3色の色ごとに光電変換素子を用いる方式とがある.一つの光電変換素子に入射する色を切換える方式としては,照明する光源の色を順次切換える光源切換え方式や,結像光路内のフィルタを順次切換えるフィルタ切換え方式などがあるが,3色を時分割で切換えるため読取り速度の点では不利であり,光源やフィルタ切換え機構に高速の応答が要求される.3色の色ごとに光電変換素子を用いる方式としては,プリズムやダイクロイックミラーを用いて色分解した結像位置にそれぞれ光電変換素子を配置する方式も用いられるが,3色用の光電変換素子とカラーフィルタを一体に並べて形成したもの(カラーセンサ)を用いる方式が主流となっている.カラー画像読取りでは色分解する際の分光的な特性(色分解特性)を用途に合わせたものとする必要があるが,フィルタなどの光学要素により理想的な分光特性が実現できない場合が多く,それを補正する信号処理として色補正演算が組合わされる,g. スキャナの種類スキャナには各種用途で要求される性能に合わせた多種多様のものがあり,走査の形態や用途などから分類図4・11 結像光学系8―80情報・メディア機器がされる.. 原稿固定式スキャナ(stationary document scanningtype)最も一般的なのは透明な平面状の台に原稿を置くフラットベッドスキャナと呼ばれる原稿固定式のスキャナであり,固定された台に原稿を置く方式であるために原稿の形態を選ばず,多くの用途に使用される.原稿固定式は原稿幅のライン状の領域を読み取る構成要素について原稿長さの機械的走査をする必要があり,装置は構成要素の移動空間を含めた大形のものとなる.読取りの構成要素を走査する方式としてはキャリッジ一体方式とミラー移動方式などがある.リニアセンサと結像光学系,照明系を一つの筐体に固定し,その筐体全体をキャリッジとして機械的に走査させる方式がキャリッジ一体方式であり,結像方式を問わず用いられる方式である.構成要素を一つの筐体に固定する構造は結像光学系の位置精度が保ちやすく,また機械的走査を行う構造も単純なものとなる.一方,長い光路が必要な縮小光学系を採用する場合に,照明系,結像光学系,リニアセンサの一部を分割して移動させる走査方式もあり,代表的なものが図4・12に構造を示すミラー移動方式である.ミラー移動方式は,結像光路を折返すミラーと照明系を搭載した第一キャリッジと,角度90°の2枚のミラーを搭載した第二キャリッジを備え,第一キャリッジの走査に対して第二キャリッジを2分の1の速度で走査することにより,結像光路の長さを一定に保ったまま,固定された結像レンズとリニアセンサに原稿の像を導いている.機械的走査構造は複雑となるが,長い光路を比較的小形の装置に収めることができ,また走査する第一,第二キャリッジを軽量なものとできるため高速な走査を行うことにも適している.. 原稿移動式スキャナ(moving document scanning type)固定した読取り構造の読取り位置に原稿を移動させるスキャナであり,読取りの構成要素を走査する必要がないため小形化に適している.原稿はシート状のものに限定されるが,シートを連続的に搬送する機構により連続読取りを行う用途などに適している.. 手動式スキャナ(handy scanner)読取り構造を手動で移動させるスキャナであり,小形で携帯性に優れ,部分的な読取りが容易であることが特徴である.スキャナが手動で操作されたことを検出する検出機構により読取りを行い,その位置検出により読み取った複数の画像を合成して広い面積の画像を生成することなども行われる.表4・2 各種のディスプレイ方式の分類と原理分類略称英語表記表示原理発光型CRT cathode ray
