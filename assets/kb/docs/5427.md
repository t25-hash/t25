# 4・4・4 プラズマディスプレイ(PDP: plasma display panel)

> **章**: 第4章 入出力装置
> **節**: 4・4 映像表示機器

プラズマディスプレイ方式の開発の歴史はLCD より古く,1950年代より始められていたが,1990年代になって大画面のフルカラーディスプレイを目指す方向の開発と利用が活発化した.表示の基本原理はガス放電に伴う発光現象を利用するものであり,モノクロ表示の場合は,ネオンガス等の放電により発生した可視光を直接表示に利用できるが,カラー表示用としては希ガス放電に伴う紫外光を3種類の蛍光体により3原色の可視光に変換して表示を行わせている.カラー表示方式は大きくはDC型とAC型の二つに分類することができる.図4・18に2種類のAC型基本構造のものを含む3種類の基本構成と動作原理を示す.DC型では対向する電極の露出表面間にDC電圧を印加して放電を発生させる.一方AC型では電極は誘電体と保護膜により覆われておりDC型のように電極表面が露出していないのが特徴であり,放電は絶縁体を挟んだ電極間にAC電圧を印加することにより発生させる.AC型には対向する電極間に放電を起こさせる対向型と片面の面内放電を起こさせる面放電型があり,特に面放電型は放電に伴うイオン衝撃を蛍光体が受けにくく蛍光体を長寿命化できる長所を有する.AC型カラーPDPに用いられるパネル構造の例を図4・19に示す.溝状に隔壁によって仕切られた放電ガス室が形成されており,背面電極側の溝内に蛍光体層が形成されている.放電は前面電極側で発生し,背面電極側に到達して可視光を発生させ,透明な前面電極側に可視光を取出して表示させる.
