a.魚群探知機 魚群探知機（fish finder ）は，漁船の船底に
れが魚群や海底に当たって反射して戻るまでの時間を測ることによ
り，魚群の存在とその距離，海底間での距離などを計測する機械で
ある．魚種や漁業種別にそれぞれ適したものが開発されているが，
ようなミリ波を併用してそれぞれの特色を発揮させて用いる方法も
ある．計測距離レンジも50m以下の浅海用のものから，1000m
以上の深海用のものまで漁業種類に応じて使用されている．スルメ
イカなど超音波の反射損失の大きいものには，超音波の発射エネル
ギーと受信感度の大きいものが，底魚を海底から分離して発見する
には分解能の高いミリ波が用いられている．なお，原理は魚群探知
機と同じであるが，コンピュータによるエコー信号処理により魚の
量を解析できる計量魚群探知機がある．
b.サーチライトソナー サーチライトソナー（search light
sonar ）は，送受波用振動子に適当なふ（俯）角をもたせて旋回さ
せ，その指向方向における目標物の反射波を検知する方式のもので
ある．超音波の指向方向は垂直面（ふ角0～90° ），水平面（全周
360° ）で変化させられるから，広範囲の探索が可能である．ただ
し，超音波を発射し，設定された距離範囲を往復するのに要する時
間を待って，次の超音波が少し角度を変えて発射される．このた
め，探索を一巡するのに，ある程度時間がかかる欠点がある．
c .スキャニングソナー スキャニングソナー（scanning
sonar ）はサーチライトソナーの欠点である探索時間を低減させる
ため，多数のビーム発信用振動子群を備え，送信時は均一レベルの
超音波を広範囲に瞬時に送波し，受信時は尖鋭な受信ビームを電子
的に切替えて，高速度で旋回させて反射波をとらえ，CRT画面に
表示させるものである．レーダとは異なりスイープの方向も直線状
ではなくスパイラルである．このため，その繰出されるスパイラル
状のラスタは，送信されてから超音波伝ぱの速度と方位が同期して
いるので，距離と方位がレーダを見るように探知でき，反射波の強
弱により輝度が変化して魚群の分布と密度がわかりやすく表示でき
るようになっている．現在，このスキャニングソナーは魚群を探索
するのに必要不可欠な漁業計測機器になっており，漁船の大小にか
かわらず，イワシ・サバ巻網漁船，カツオ巻網漁船，サンマ棒受網
漁船に設置されている．
d.ネットゾンデ ネットゾンデ（net zonde）は水中にある漁
具の状態を連続測定し，その情報を船上に伝達する装置である．本
器は発信器と受波器および受信器よりなり，発信器を網に取付け，
発信器内に内蔵された音響測深器によって測定した値を超音波FM
方式で伝達し，この信号を船外に垂下した受波器で受け，さらにこ
れを有線で船内の受信器に導いて紙上に記録するものである．信号
伝達距離は2000mぐらいである．中層びき網の深度測定，底びき
網の網口の左右および上下距離や入網魚群量の測定，巻網の網裾の
沈降状況を観測するのに使用されている．また，水中を超音波FM
方式で伝達しないで，空中を超短波で伝達するテレサウンダ
（tele- sounder ）方式のネットゾンデもある．これは，定置網に入
網する魚群量を測定したり，巻網の船団操業において，付属船の情
報を主船に送るのに使用されている．
