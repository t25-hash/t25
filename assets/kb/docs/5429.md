# 4・5・1 画像形成技術

> **章**: 第4章 入出力装置
> **節**: 4・5 プリンタ(printer)

a. 画像形成法歴史的には、電動タイプライタ式の活字プリンタが初期の計算機端末のプリンタとして華やかなる時期もあったが、現在ではほとんどディジタル画素によって画像を形成する方式が用いられている。このディジタル画像形成方式は、図4・24に示すように、ドットの集合で画像を形成するものであり、その解像度はdpi(dot per inch,dot/25.4mm)で表示される。数百dpi以上の解像度であれば、肉眼ではほとんどドットを視認できないが、オフセット印刷や写真並みの高画質を実現するために、1200dpi以上のものも市販されている。フルカラー画像は、1画素にシアン(cyan、青)、マゼンタ(magenta、赤紫)、イエロー(yellow、黄)の3原色と黒(black)を重ねて作成する(CMYK 減法混色:色を混ぜるほど明度が下がって黒に近づく)。原理的には、3原色の混合で黒色を得ることも可能であるが、濃い純粋な黒色を得るために黒も併用される。フルカラーの場合、解像度だけでなく階調性や各色の相対的な位置ずれ(registration)も画質を決定する重要な要因となる(4・6・2・f・iii項参照)。高画質のフルカラー画像を印刷するためには膨大な量の情報処理と高度な精密技術を必要とするが、ソフト・ハード両面での技術の進展により、安価で高速・高画質のプリンタが普及してきた。b. 種類表4・7に示すように、プリンタの印字方式はインパクト方式とノンインパクト方式に大別される。インパクトプリンタは、ノンインパクトプリンタに比べて騒音が大きい、高速化が困難、高画質のカラー画像が得られないなどの欠点があるため、次第に用いられなくなった。インパクト方式で現在使用されているのは、ワイヤドットプリンタのみである。ノンインパクトプリンタには種々のものがあるが、普及しているのは、インクジェット方式、サーマル方式(感熱方式、熱転写方式)および電子写真方式である。図4・25に全世界でのプリンタの出荷台数の変遷を示す。台数ではインクジェットプリンタが圧倒的に多い。これはカラー印刷を低コストで実現するのに最も適した方式であることがおもな要因である。一方、プリント枚数では電子写真方式のレーザプリンタが80%近くを占めている。これは、電子写真が複写機と同様に普通紙への高速印刷に適した方式であるため、オフィス用途を中心に利用されているためである。一部に例外はあるが、各方式のカラー化の図4・24 ディジタル画像形成と解像度表4・7 プリンタの分類印字方式技術分類小分類カラー化の可否印刷速度インパクトワイヤドット△低速インクジェットサーマル○低速ピエゾ○低速感熱×低速ノンインパクト熱転写○低速電子写真レーザ○高速LED○高速8―84情報・メディア機器可否と印刷速度はおおよそ表4・7のようである。
