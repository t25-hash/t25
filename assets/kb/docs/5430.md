# 4・5・2 ワイヤドットプリンタ(wire dot printer)

> **章**: 第4章 入出力装置
> **節**: 4・5 プリンタ(printer)

ワイヤドットプリンタは,インクリボンに含浸させたインクをワイヤのインパクトによって記録紙に写すことにより画素を形成するものである.図4・26に示すように,ワイヤは電磁的に駆動され,構成画素数などにより,複数のユニットを一組とする.複数のユニットを集めた印字ヘッドはシリアルに移動しながら画素を印字する.1980年代までは,プリンタの大半はこのワイヤドットプリンタであったが,インクジェットプリンタやレーザプリンタなどの低騒音,高速,高画質のノンインパクトプリンタの出現により,現在では,同時に複数枚の印字が可能という特徴が重視される用途などに限られている.
