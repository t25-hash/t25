# 4・5・3 インクジェットプリンタ(ink jet printer)

> **章**: 第4章 入出力装置
> **節**: 4・5 プリンタ(printer)

a. 種類インクジェット方式のプリンタは,ノズルからインク滴を飛翔させて印画紙上に直接画素を形成するものである.インクの吐出方式には,インクを連続的に噴射する連続噴射型と,必要なときにだけインクを飛ばすオンデマンド(on demand)型がある.オンデマンド型には,加熱により発生した気泡の圧力でインク滴を吐出するサーマル方式と電圧の印加によって変形するピエゾ(圧電素子)を用いるピエゾ方式がある.下記におもな吐出方式の分類を示す.連続噴射型オンデマンド型サーマル方式たわみ変形式縦振動式ピエゾ方式ずり変形式連続噴射型は低価格化や小形化が難しいため,コンピュータ出力のプリンタとしては普及していないが,製品にその製造年月日などを印刷する産業用のマーキングプリンタなどに用いられている.コンピュータ出力のプリンタとして現在広く普及しているのはオンデマンド型のサーマル方式とピエゾ方式である.ピエゾ方式には圧電素子を使用する機構の違いによりいくつかのタイプがある.初期から利用されているたわみ変形式や縦振動式のほか,1990年代半ばからずり変形モード(shear mode)を利用したピエゾ方式も実用化されている.最近のインクジェットプリンタは,通常の4色インクに加えてライトシアンとライトマゼンタの淡インクを用いること,インク吐出量を減らしてドットサイズを小さくすること[1984年に1000pl図4・25 全世界のプリンタの出荷台数(ピコリットル)以上あったインク吐出量が,1999年には3plに低減されている]や専用光沢紙の開発などにより写真画質を実現している.b. サーマル方式. 原理・基本構成サーマル方式は1970年代後半にキヤノン社が開発した方式で,ノズル内にヒータ素子を埋め込み,その発熱によってインクを沸騰させて気泡を発生し,その圧力によりインクを吐出するものである.通常液体はゆっくり加熱すると核沸騰を起こすが,これでは高圧を高速に繰返し発生させることができない.しかし,マイクロ秒オーダで急速加熱をすると膜沸騰現象が起こり,10MPaもの高圧のバブルを非常に短時間だけ発生する.この原理を利用したサーマル方式はバブルジェット方式(キヤノン)やサーマルインクジェット方式(ヒューレットパッカード)とも呼ばれ,ほかにもいくつかのメーカで採用されている.サーマル方式はc項に述べるピエゾ方式に比べて,可動部分がないこと,10倍以上の高い圧力が発生可能であること,微小なヒータで液体を大きく変位できること,などの特徴から構造が簡単で小型のノズル設計が可能である.そのため,製造コストが低く,高密度化や多ノズル化が容易である.ピエゾ方式のように吐出量を変調することは難しいが,たくさんのノズルを高密度に配置することで大幅な高速化と高画質化を達成している.最新の機種では,1200dpiピッチで3072ノズル(512ノズル×6列)を配列したヘッドも実用化されている.図4・27にサーマル式ヘッドの断面模式図を示す.これは,半導体の成膜工程をノズル製法に応用したものであり,シリコン基板の上にノズル材料を成膜してノズルを形成するフォトファブリケーション技術によって製作される.このヘッド部分にインク供給部材と駆動用の電気供給部材を実装することで,プリントヘッド部分が完成する.サーマル方式はインクジェットの低価格化を進め,1990年代前半にインクジェットプリンタが広く普及するための原動力となった.サーマル方式の技術的課題の一つは,発熱体の耐久性を上げることである.バブルジェット方式では発熱体の温度は数マイクロ秒以内で400~500°Cにまで加熱され,大きな熱ストレスがかかる.また,気泡がつぶれるときに生じるキャビテーションも発熱体腐食の原因となる.さらに発熱体や電極は高熱下でインクにさらされることになるため,化学的な腐食に対する安定性も要求される.この課題に対して,ヒータ表面の保護膜の改良やノズル設計の最適化により,現在では実用的には十分な耐久性を達成している.また,高温にさらされたインク中の成分が熱分解してヒータ表面に発生するこげもサーマル方式特有の課題であるが,インク成分の最適化により実用的な信頼性を達成している.. インク吐出高精度化技術通常のサーマル方式の吐出メカニズムは以下のようである.(1) 発泡工程:ヒータをごく短時間通電すると,近傍のインクが急激に加熱され,微小な気泡が発生し,高圧でインクを押す.(2) 分離・吐出工程:インクがノズルから押出されると同時に,気泡はインクの慣性により膨張させられるが,やがて最大気泡に達し,ノズル内のインクの運動は止まり,インク滴は分図4・26 ワイヤドットプリンタの印字ヘッド(1画素分原理図)図4・27 サーマル方式インクジェットノズルの断面模式図[キヤノン(株)資料より]離される.(3) 収縮・消泡工程:分離したインク滴が吐出されるとともに,内部がほぼ真空状態の気泡は急激に収縮し消泡する.(4) リフィル工程:毛管現象により,ノズル内に再びインクが充てんされる.これに対してMFDT(micro fine droplet technology)と呼ばれる微小ドットをより安定して高速に吐出できる新しいメカニズムでは,ヒータ部をよりノズル開口部に近づけて配置されている.図MFDT 方式では,従来方式にあった分離および収縮・消泡工程をなくして,発泡,成長・吐出,リフィル工程だけを繰返すよう単純化されている.従来方式では最大泡に達したあと液滴がちぎれて分断されるが,このちぎれる位置がインク粘度や発泡圧力の揺らぎなどの影響で微妙に変化するため,インク滴の大きさが変動してしまうという問題点があった.特にインク滴を微小化して高画質化しようとする場合には,この影響は無視できなくなる.MFDT では,インク滴は気泡によってつねに決まった位置で分断され,そのまま気泡が外気と連通する形で発射されるので,インク滴のサイズの精度は大幅に向上する.また,吐出のエネルギー効率も良くなるため,インク滴の吐出速度を速くすることが可能となり,微小インク滴で問題となる空気抵抗の影響による着弾精度の悪化を抑制することができる.これらの特徴からMFDT 方式は高画質化に求められるインク滴の微小化に伴う問題点を解決し高画質化を達成している.c. ピエゾ方式. 原理・基本構成初めて提案されたインクジェットは圧電体の変形を利用してインクを吐出させるピエゾ方式であった.しかし,セラミックスや金属材料の複雑な加工が必要なために量産が困難であったため長い間本格的な実用には至らなかった.この問題を解決するために新しい発想で開発されたピエゾ方式がセイコーエプソンのマッハ(MACH : multi-layer actuator head)ジェット方式で,1993年に登場した.マッハジェットヘッドには図4・29に示すような二つのタイプがある.その一つはシリコンの異方性エッチングによって形成したインク室に積層構造のピエゾを配置したMLP(multi layer piezo)タイプで,ピエゾを縦振動モードで使用する.もう一つはピエゾとジルコニア(ZrO)をセラミック一体焼成で積層形成し,セラミックスの積層でインク室を形成するMLChips(multi layer ceramic with hyper integrated piezosegment)タイプで,ピエゾ素子と振動板とのバイメタル効果によるたわみ変形モードを利用する.MLChipsは二次元的な積層構造のため,自動組立に適し量産性に優れている.一方,MLPはインク室がコンパクトでノズルの高密度化に適しており,MLChipsと製品コンセプトに応じて使い分けられている.. ドット制御技術インクジェットプリンタ高画質化の手段の一つはマイクロドット化であるが,単純にノズル径を小さくすると,インクの目づまりなどの信頼性が低下するだけでなく,インク滴吐出後インクが再充てんされるまでのリフィル時間が長くなり,駆動周波数を高くするのが難しくなる.これらの問題を解決す図4・28 バブルジェット吐出メカニズム8―85るために,同じノズル径から大きなサイズのインク滴と,より小さなインク滴を吐出する技術が開発された.図4・30にその制御イメージを示す.まず吐出前にいったん電圧を降下させてインク室を広げる方向にピエゾを制御し,メニスカスを後退させるpull制御を行う.pull制御を素早く行うほど,後退したメニスカスが回復する時間的余裕がなくなるのでメニスカス後退量は多くなる.次に正電圧を印加して急激にインクを吐出するpush制御を行い,小さいインク滴を十分なスピードで吐出させる.インク吐出後には,電圧を下げることで再びメニスカスを後退させてドットの切れを高め,サテライトドットと呼ばれる余計な派生ドットの発生を抑制するとともに,メニスカスの振動を速やかに収束させて次のインク吐出までの時間を短くし,高い応答性を実現している.図4・30の例ではpull制御タイミングのみを変更しているが,印加電圧の変更でも吐出量が制御できる.通常は電圧とタイミングの両方を最適化してインク滴が最も安定して吐出される条件に設定する.図4・31にドット径とプリンタ解像度の関係を示す.図4・31でドット中央に描いた四角形は,その解像度に対応した本来の画素サイズを示している.解像度に対してドットサイズはかなり大きい.原(a) インク量,ドット径,解像度の関係(pl:ピコリットル)8―86情報・メディア機器理的にはドットの直径が画素の対角線サイズより大きければ印画メディア表面を100%覆うことができるが,インクジェットプリンタではそれよりもかなり大きめのサイズのドットを用いるのが普通であり,直径が画素幅の2倍以上になることもある.その理由の一つは十分な出力濃度を確保するためであり,隣接するドットどうしが重なり合い始めてからは濃度階調的に出力濃度が増加する.もう一つの大きな理由は,ドットの飛行曲がりやプリンタの機械精度に対するマージンを確保するためである.ドットサイズに余裕がないと,特定のノズルから吐出されたドットの着弾位置が少しずれただけでもドット間にすきまが生じて,筋状の濃度むらとなって画質を劣化させる.特に720dpi以上のような高解像度モードでは,それに応じた機械精度とインク滴の吐出方向精度を確保するのは容易ではなく,これらの精度が高いほどドット径のマージンは少なくできる.さらに,出力濃度と着弾位置精度の点からそのドットサイズに最適な出力解像度が決まったとしても,現実の解像度は機械的制約などから1440,720,360dpiなどの飛び飛びの値しか選べないことが多く,最適な解像度より高い解像度に設定される.必要以上に高解像度化しても画質はあまり向上せず,印画速度の低下やデータ処理量の増大を招くだけなので,解像度は必ずしも画質を正確に反映していないことになる.これに対して吐出インク量が変えられると,使用するメディアや解像度に応じて,効率よく高画質が引き出せる最適のドット径を選択できるようになる.ドット径は印画メディアによっても大きく異なるので,同じ出力解像度でもメディアに応じてインク吐出量を変えている.さらにドット微小化を図るために3種類のサイズのドットを画素単位で使い分けるMSDT(multi-sized dot technology)が開発され,1998年に製品に導入された.図4・32にMSDT のドット制御イメージを示す.1周期の中で小ドット用の駆動波形と中ドット用の駆動波形をもち,両方を続けると大ドットが吐出される.小ドットの吐出タイミングが中ドットよりも早いが,小ドットのほうが吐出液滴の飛行速度が遅いため,用紙上では両者はほぼ同じ位置に着弾する.同様に,大ドット吐出時は小ドットと中ドットが用紙上で合体して一つの大ドットになる.MSDT によって,印画速度を落とすことなく最小ドットを微小化し,さらなる高画質化が可能になった.この例では大ドットのインク量は小ドットと中ドットの合計インク量よりも多くなっている.これは連続して吐出するときには,小ドット印画後のメニスカスの振動を利用して次に続く中ドットの吐出量を増やせるためで,より広い幅でのドロップ量変調を可能にしている.ドット径変調の幅は設定次第でさまざまにでき,製品と印画モードに応じて(4,7,11pl),(3,10,19pl)など,さまざまな組合せが実現されている.d. 画像処理技術画像データを基に各色インクのドット配置データを生成する画像処理技術も画質に大きな影響を及ぼす.通常のPC用プリンタではホストPC上のソフトウェアであるプリンタドライバがこの処理のほとんどを実行する.RGB イメージ生成以降の具体的な処理項目には,(1)RGB 色データをプリンタのインク量データに変換する色分解,(2)原画像の色とプリンタ出力色を一致させるためのカラーマッチング,(3)誤差拡散法などによってインク量データを実際のドットのオン/オフに置換えていくハーフトーニングがある.なお,このような画像処理技術は,レーザプリ図4・32ピエゾ方式インクジェットのMSDT 制御図4・33 インクジェットプリンタの構造ンタなどにも共通するところが多い.e. インクインクジェットプリンタ用インクには,水性と油性,染料タイプと顔料タイプ,特殊なものとしては紫外線硬化タイプや,ホットメルトタイプなどがある.民生用インクジェットプリンタで用いられるのはおもに水性の染料インクで,一部には黒インクにのみ水性顔料インクを使うものもある.顔料タイプのインクはインクが独自の質感をもつために,専用紙での画質は染料系インクに劣ることが多いが,耐光性などの耐環境性や長期保存性に優れるという特徴があり,装飾展示ポスター用の大判プリンタなどで採用されている.水性染料インクには,溶媒の水,染料,保湿剤(グリコール類,アルカノールアミン類など),表面張力調整剤(界面活性剤,アルコール類など)などが含まれおり,染料材料としてはアゾ染料,フタロシアニン染料,アントラキノン染料など各種の染料が色再現性や耐光性の向上を目指して開発されている.f. 用紙インクジェットプリンタはPPC用紙に代表される普通紙にも印画可能である.しかし液体インクゆえのにじみや耐水性などの問題があり,その能力を最大限に発揮するためにはインクジェット専用に開発された記録用紙が必要になる.特にフルカラー出力では,3色以上のインクを重ねて打つため,単位面積当たりに印画されるインク溶媒量はかなり多くなる.このため用紙には,多量のインクをにじむことなく速やかに吸収し,ドット形状が真円にそろい,なおかつ十分な濃度で発色する,という特性が求められる.このため,インクジェットプリンタ用のコート紙や写真印画紙と同等以上の質感と光沢をもった光沢紙が開発されている.g.
