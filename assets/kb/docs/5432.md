# 4・5・4 感熱・熱転写プリンタ

> **章**: 第4章 入出力装置
> **節**: 4・28 に通常方式とMFDT 方式による吐出メカニズムを示す.

a. 感熱プリンタ(thermal printer)直接発色型の感熱プリンタは,熱によって発色する感熱記録層を紙やフィルムの上に塗布し,多数の抵抗発熱体ドットからなるサーマル印字ヘッドを紙に押付けて感熱記録層を加熱することにより,画素を印画するものである.通常は単色のみの印刷であるが,一部の用途で赤・黒2色発色型も使われている.ヘッドには,薄膜技術によるものと圧膜印刷技術を使ったものの2種類がある.またプリンタの機構には,シリアル型とライン型がある.数mm程度の小さいヘッドを用い,往復操作させながら,1行ごとに記録するのがシリアル型である.低コストでできることから,パーソナル用途に多い.一方,印字幅(数十mmから数百mm)分のヘッドを固定し,用紙の走査によって記録するのがライン型である.1回の走査で1ページの印刷を行うため速度が早い.印刷サイズが大きいとヘッド製造が困難となってくるが,A 3幅のヘッドをつないで1mを超える大サイズのラインヘッドも開発されている.感熱プリンタは,超小型プリンタ,ファックス,バーコードプリンタ,券売機,計測器,CAD プロッタ,医療画像プリンタの出力などに用いられている.b. 熱転写プリンタ(thermal transfer printer)熱転写プリンタは,インクリボン上のインクを発熱体を有するサーマルヘッドの発熱を利用して記録紙に転写するものである.溶融熱転写でのレジンインクの転写プロセスは大きく分けて図4・34に示すような4工程からなる.サーマルヘッドにより加圧,加熱されたインクがヘッドのエッジ位置でリボンをはく離することによって紙へ転写される.原色を重ねることによってカラー印刷が可能である.転写剤に顔料を使う溶融型と昇華染料を使う昇華型がある.溶融型は,普通紙から平滑なコート紙,合成紙,フィルムまで広く転写用受容紙として使用できるが,昇華型は記録紙の表面に必ず昇華染料を受容できる熱可塑性樹脂をコーティングした専用紙が必要となる.
