# 4・7・2 CD(コンパクトディスク)

> **章**: 第4章 入出力装置
> **節**: 4・7 オーディオ機器

a. CD システムの概要従来のアナログレコードは、ディスクの状態やピックアップの非直線性等の影響を受けるので再生の忠実度には限界があった。これに対して、コンパクトディスク(CD : compact disk,compact disc)は音声信号をディジタル信号の形で記録することによって、高い忠実度で信号を再生できるようにしたものである。ディスクからは1,0のディジタル信号を読み取るだけでよいので読み取る部分のひずみやわずかな誤りは、再生信号の忠実度に影響することがないためである。この記録されているディジタル信号には、再生時データを正しく読み取れるように、音声信号のほかに誤り訂正用のデータも含まれている。そこで、再生時にはこの訂正用のデータを用いて読み取ったデータをチェックし、もし誤りがあればこのデータを使用して正しいデータに訂正することができる。このディジタル信号は、微小ピットの形でディスクに記録してある。再生するにはこのピットを検出し、その有無をディジタル信号の1,0に対応させることによって読取りを行っている。このため、12cmのディスクにおよそ5Gb(ギガビット)にのぼる大容量のデータを記録することができ、1時間以上の再生時間を得ている。さらに上記の信号の読取りは、レーザビームを用いた非接触方式のため、ディスクは摩耗がなく長寿命である。b. ディスク基本構造は透明基盤の片面に信号ピットを形成し、この面にアルミニウムの反射膜をコーティングしたものである。トラックピッチは1.6 mで、ピットの幅は約0.5 m、高さは約110nmである。ディスクの材質はポリカーボネート樹脂で、信号の記録は片面のみである。外形寸法と信号の記録されている部分を図4・57に示す。信号トラックはディスクの内周から始まる。トラックのスタート部にはリードインエリア(lead in area)が設けてあり、ここには収録されている曲の数やその時間、位置等の種々の情報が記録されている。プログラムエリアの記録密度が一定になるように、1.2~1.4m/sの一定線速度(CLV : constant linear velocity)で信号が記録されている。したがって、ディスクの回転数は内周では速く、外周にいくに従って遅くなる。c. 信号処理フォーマットデータの構成を図4・58に示す。アナログ音声信号は44.1kHz のサンプリング周波数で16ビットに量子化されてディジタル信号になる。信号処理は、この16ビットからなるデータを2分割してシンボル(symbol)と呼ばれる8ビット単位の信号にして行う。まず、きずなどのために生じる連続したデータの欠落(burst図4・57 CD の形状と信号の記録部分図4・56 カセット式図4・58 CD のデータ構成図4・59 CD プレーヤーのブロックダイヤグラムerror)によって誤りが生じるのを最小限にするために、データはシンボルを単位として時間的に前後に分散して並び方を変えてある(interleave)。そして誤り訂正用としてはC,Cからなる2組のリードソロモンコードを24シンボルごとに付加している。このような信号処理の方法をCIRC(cross interleave reed-solomoncode)と呼ぶ。この信号をディスクに記録し安定して再生できるようにするために、EFM(eight to fourteen modulation)と呼ばれる変調方式で変換される。この方式は、ビットの反転が3ビット以上、11ビット以下(記録信号の基本波成分は最高700kHz、最低200kHz)に収まり、DCバランスのずれも少なくなるように工夫されている。実際に記録されているデータは、以上の信号のほかに同期信号とサブコード信号を加えて1フレームとして構成されている。d. CD プレーヤーの構成CD プレーヤーの概略の機能を図る。ディスクに記録されている信号は、光ピックアップを用いて光学的に読み取る。ディスクのピット面に正確にレーザスポットを当てるためには、光ピックアップ本体から可動的に支持された対物レンズをコントロールし、光軸や焦点がつねに最適な位置をトレースするようにサーボを行う必要がある。光ピックアップからはフォーカス、トラッキング両方向の誤差信号が得られるので、この誤差信号を打消すように対物レンズと一体となった駆動コイルを動かすことでサーボ機構を構成している。次に再生回路系について信号の流れに沿って説明する。光ピックアップからは、ピットに応じたEFM 信号が再生される。まず、このEFM 信号からタイミングの基準となるクロック(clock)を抽出し、さらに同期信号を取出す。このクロックと同期信号を使って、EFM 復調、誤り訂正処理を行い元のオーディオ信号を得る。例えば、訂正処理前のランダム性のバイトエラー率が10であれば、計算上、訂正処理できないケースは1年に1回未満にすぎない。実際はディスクのきずやサーボのアクシデントでエラーが集中発生して訂正できないケースがあるが、このときは前後の信号から補間する処理等で再生音のノイズを防いでいる。オーディオ信号はDAC(digital analog converter)でアナログ音声信号に復元される。この信号にはサンプリング周波数(44.1kHz)の整数倍を中心にサンプリングノイズが含まれるのでロー8―9 3パスフィルタで取り除く。なお、CD には音声信号と一緒に各種の情報を含んでいるサブコード信号(subcode)が記録されている。これを読み取ることにより、アナログレコードではできなかった曲の頭出しや経過時間表示など便利な機能が可能となっている。
