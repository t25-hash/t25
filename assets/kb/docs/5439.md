# 4・7・4 新しいオーディオ機器

> **章**: 第4章 入出力装置
> **節**: 4・59 に示す。まず、光ピックアップのサーボについて説明す

オーディオ機器の今後を占う意味で，ここではディジタルオーデ
ィオの二つの新しい流れ について触れておく．
a.高機能・高音質化 大容量，高密度ディスクの実用化や，
高速信号処理技術，半導体技術の進展により，次世代オーディオデ
ィスクとしてSACDやDVDオーディオが新たに登場した．これ
らはオーディオの性能限界を大幅に拡大するとともに，高品位マル
チチャネルオーディオ再生を実現し臨場感の向上をもたらし，次世
代オーディオに相応しい内容を備えている．
SACD（super audio CD）は，現行CDの発展型ともいえるもの
で，音声信号を高サンプリングレートのΔΣ（デルタシグマ）変調
器によりパルス波形の密度で表現する方式であり，周波数帯域100
kHzと可聴帯域のダイナミックレンジ120dBを実現するものであ
る．
DVDオーディオは，DVD（digital versatile disk）のオーディ
オフォーマットとして規格化されたものであり，DVDビデオと互
換性を確保しつつ，4.7GB（ギガバイト）の大容量ディスクを利
用してサンプリング周波数最大192kHz，最大量子化24ビット
（ダイナミックレンジ144dB），最大6チャネルを実現する．
SACD，DVDオーディオはともに，ウォータマークと呼ばれる
著作権保護用の識別信号が記録されており，違法な複製を防止でき
るように対策されている．なお，両方式は互いに互換性はなく専用
のプレーヤーが必要であるが，一部には両方式のディスクを再生で
きるプレーヤーも発売されている．
b.新しいオーディオの形態 パソコンの普及に伴うインタネ
ットの発展と，伝送効率の向上を目指した音声圧縮方式の進展を背
景に，音楽配信サービスがスタートしている．音声圧縮技術として
は，MP3（MPEG-1audio layer -3）をはじめ数種類の音声圧縮
技術が実用化 されており，音声データを1/10～1/20に圧縮する
ことができる．
一方記録メディアはこれまでテープやディスク記録方式が主流で
あったが，最近はメカニズムを使用しない半導体メモリが注目され
ている．現在，半導体メモリのフォーマットは数種類が存在してい
る段階であるが，容量2GBも発売されており45時間程度の音楽
を記録できる．テープやディスクと比較すると高価格であるが，メ
カニズムをもたないために，小形軽量で消費電力が少なく振動に強
いという利点があり，ポータブル機器を中心に発展が期待される．
これらのシステムが発展していくために，技術面での向上・発展
もさることながら，①利便性が高いゆえの著作権保護と②魅力ある
コンテンツの提供を充実させていくことが重要である．
