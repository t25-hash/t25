# 5・1・2 項で,ストレートタイプと折り畳みタイプの構造例を一つ

> **章**: 第5章 情報通信機器
> **節**: 5・1 電話(携帯電話)

ずつ示したが,次に,共通する主要構成部品について説明する.a. 電子回路基板部電子回路基板部は,おもに回路基板と高密度実装部品により構成されている.. 回路基板電子部品が高集積化され,回路基板との接続部も高密度化が要求されている.従来は,表層IVH(inner viahole)構造の基板が使用されていたが,現在は,全層IVH 構造の多層プリント配線基板が主流となっている.図5・3に両基板の構造を示す.図5・3のように,従来の表層IVH 多層基板は,貫通スルーホールが必要であり,そのため,有効面積が制限される.現在の全層IVH 多層基板は,内層での層間接続が可能となり,表層への高密度部品の実装面積がより広く使用できる.. 高密度実装部品小形,高機能化を実現するには,半導体の高集積化と,その半導体を納めた半導体パッケージを小形化し,実装面積を減らすことが必要である.またチップ部品の寸法も,現在は1.0×0.5mmサイズが主流であるが,さらに小形化が進められている.従来の半導体パッケージは,パッケージからはんだ接続用のリードが形成されていたが,最近の主流は,CSP(chipsize package)という小形パッケージである.これらは,パッケージの下面に回路基板との接続部を有し,実装に必要な面積を減らしている.CSPには,その接続部の端子形状により,BGA(ballgrid array),LGA(land grid array),QFN(quad flat non-lead-ed package)などの種類がある.図5・4にBGA とQFN の断面構造を示す.b. 筐体部携帯電話における筐体の役割は,外装部品として商品の魅力を発揮することと,筐体内部に納められた電気回路部品,表示部品,音響部品などを,外部の負荷から保護することである.一方で,商品として小形化,軽量化が求められる.ストレートタイプの場合,筐体部品はフロントカバー,リヤケースの2部品となる.材料はおもにABS 樹脂,PC樹脂,ABS+PC樹脂が使用されている.これらの材料は,コストが安価であり,成形性も良く,表面の塗装やめっきなどが可能である.国内の製品では,材料自身への着色を行い,その上に下地塗装を行い,さらにその下地塗装を保護するためのオーバコート塗装(透明)を行うという方法が多く見られる.折り畳みタイプの場合は,筐体部品としては4部品となり,そのうち,ヒンジユニットを保持する部品には,マグネシウム合金が使われている.チクソモールディングという薄肉成形技術により,樹脂の射出成形と同じように0.8mm程度の薄肉の成形が可能である.表5・1にマグネシウム合金とABS,PCの物性値の比較を示す.マグネシウム合金は,比重1.8と実用構造金属の中では最も軽く,剛性面では,ヤング率がABS の約20倍,PCの約7倍と高い.強度面では,樹脂材料に比べ有利であるが,外装部品としては,成形性,塗装品質,表面処理品質が樹脂材料に比べやや劣っており,現在もこれらについて改善の取組みが行われている.携帯電話は,今後も小形,軽量,薄型化が求められ,一方で電気部品は高密度化されていく.内部部品を保護するための筐体の内部構造設計は,市場での落下衝撃や曲げなどの負荷を想定し,CAE(computer-aided engineering)による強度解析を駆使した取組みも行われている.c. 入力部おもな入力操作はキー操作で行われている.操作内容は,ダイヤル入力と文字通信のための文字入力である.これらは,数字キーに文字も割付けられている.ほかに,電源のON,図5・3 表層IVH 基板と全層IVH 基板の構造比較図5・4 BGA とQFN の断面構造図5・2 折り畳みタイプの構造例T:ガラス転移点図5・5 表示部現在は,カラー液晶パネル(LCD : liquid crystaldisplay)が主流になってきている.文字情報から画像情報へと,より情報量が増えてきたためである.今後も静止画像から動画へと進むにつれて,表示部にはさらに高度化が求められていく.具体的には大画面化や高精細化,一方で小形化のための内部配線部の微細化や薄型化などである.LCD 以外の表示素子として,高輝度で,薄型構造である有機EL(electro luminescence)の開発も活発に進められている.e. アンテナ部主流は,スライド式のアンテナ構造であり,伸ばした状態での曲げに対して強い形状記憶合金が多く使用されている.このスライド式アンテナと本体内部にある内蔵アンテナを組合わせて使用されている.アンテナ部は,本体の筐体部から突出しているため,落下衝撃を受けやすい部分であり,強度面も十分に考慮して設計されている.しかし,今は,意匠デザイン的にも突出部がないほうがよく,内蔵アンテナ化の研究も行われている.また,人体への電磁波の影響も考慮した設計が行われている.f. バイブレータ着信を音ではなく振動によって使用者に伝えるためのものである.主流は,コアレスモータの先端に分銅を付け,回転させ振動を発生させている.最近では,薄型化のため,コイン状のフラットなモータを使用しているものもある.本体の質量により,分銅の質量とモータ回転数を調整し,適切な振動を発生させる.g. 音響部品主たるものは,マイク,レシーバ,スピーカである.マイクは音声入力部で,寸法は,直径約6mm×厚さ1.6mmまで小化されている.レシーバは音声出力部で,寸法は,直径約10mm×厚さ約2.7mmのものができている.スピーカはおもに着信音の出力部として使用され,寸法は,直径16mm×厚さ約3.5mmのものができている.これらの部品はつねに小形化が要求され,今後もその取組みが進められる.また,本体の限られたスペースに組込むため,異形のレシーバや超薄型レシーバなどの開発も活発である.h.
