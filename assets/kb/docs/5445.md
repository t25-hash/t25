# 5・1・4 第三世代携帯電話と関連技術の開発動向

> **章**: 第5章 情報通信機器
> **節**: 5・5 に示す。

2001年10月から、第三世代携帯電話であるW-CDMA 方式のサービスが開始された。その伝送スピードは、384kbpsという高速であり、第二世代の9600bpsと比べると40倍となる。このスピードによって動画像も見られるようになり、携帯テレビ電話のような使い方が可能になる。ここでは、第三世代携帯電話と関連し、今後開発が進むと考えられる技術について紹介する。a. カメラ技術前述したように、静止画だけでなく、動画像を扱えるようになるため、カメラ付き携帯電話の高機能化が求められる。静止画用では、11万画素のCMOS(complimentary metaloxide semiconductor)カメラを使用しているが、今後35万画素以上の画素数が求められ、さらに、画像を記録するためのメモリ機能の充実も検討されている。ディジタルカメラやビデオカメラのような光学ズーム機能の小形化も要求されてくるだろう。カメラ機能の使用頻度が増えれば、携帯電話の形もその使用形態に適した形状に変化していくと考えられる。b. 個人認証技術携帯電話の新たな利用方法として、電子商取引(EC: electronic commerce)が考えられている。IMT-2000の規格では、ICカードの搭載が規格化されており、UIM(useridentity module)カードが採用されている。UIM カードによってその端末が正規なものであると確認でき、さらに、使用者が本人であるかどうかを非接触ICカードにより確認する方法が検討されている。非接触ICカード以外の個人認証技術としては、バイオメトリクス(biometrics)を使った技術開発も行われている。バイオメトリクスには、指紋認証、虹彩認証、声紋認証、署名認証などがあり、それぞれ研究が進められている。特に、指紋認証は、携帯電話への搭載を意識したモジュールが開発されつつある。また、カメラ付きが主流となれば、虹彩認証も実用化しやすくなる。今後いっそう開発に拍車がかかると予想される。c. 三次元高密度実装技術第三世代携帯電話では、第二世代に比べ、さらに多くの機能を盛り込んでいくことになる。しかし、携帯電話としては、小形、軽量化が求められる。これらを実現するために、さらなる半導体の高集積化が求められるとともに、複数の部品を一つのモジュールとして小形、一体化する三次元高密度実装技術の開発が行われている。例えば、LSIや抵抗、誘電膜を回路基板の内層に埋め込んでしまう技術である。ほかにも、ベアチップ実装やCSPのさらなる微細化なども検討されている。高密度実装技術の進化は、本体の構造設計においても、十分に着目し、考慮しなければならない重要なポイントである。8―100情報・メディア機器
