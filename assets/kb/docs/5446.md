# 8・2・1 機械インタフェース

> **章**: 第8章 機構
> **節**: 8・2 システムインタフェース設計

機構設計の中でシステムの機械インタフェースに関わる項目としシステムあるいは他のアペンデージとの干渉，が挙げられる．
星ETS-Ⅵのアンテナ反射鏡面の場合には，跳ね上げ機構とアンテンタフェース面荷重を超過するようなラッチ速度で展開を完了したる．
技術試験衛星Ⅷ型（ETS-Ⅷ）のような巨大なアンテナ反射鏡る荷重に加え，展開の開始時，展開途中のシステム姿勢制御への擾面構造を固縛している保持解放機構が展開を開始する場合の反動ト乱トルクを超えないように，ダンパ等の速度制御要素を利用して初b.アライメント 展開構造が機構を介してシステムに固定さション性能に大きな影響を及ぼす．
展開構造が機構を介してシステイメント基準となる．
このアライメント基準面を機構の取付面その計測コンフィギュレーションを考慮した適当な位置にアライメントト精度で保証するといった方法がとられる．
