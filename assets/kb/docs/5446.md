# 8・2・1 機械インタフェース

> **章**: 第8章 機構
> **節**: 8・2 システムインタフェース設計

機構設計の中でシステムの機械インタフェースに関わる項目とし
ては，(１)インタフェース荷重と姿勢擾乱，(２)アライメント，
(３)システムあるいは他のアペンデージとの干渉，が挙げられる．
a.インタフェース（interface）荷重と姿勢擾乱 技術試験衛
星ETS-Ⅵのアンテナ反射鏡面の場合には，跳ね上げ機構とアンテ
ナタワーとのインタフェース面荷重が規定された．反射鏡がこのイ
ンタフェース面荷重を超過するようなラッチ速度で展開を完了した
場合，アンテナ反射鏡面を片持状態で支えるタワーの根元が破壊す
る．技術試験衛星Ⅷ型（ETS-Ⅷ）のような巨大なアンテナ反射鏡
面を搭載する衛星では，システムへの取付インタフェース点におけ
る荷重に加え，展開の開始時，展開途中のシステム姿勢制御への擾
乱も考慮することが必要となる．ETS-Ⅷでは，打上げ時に反射鏡
面構造を固縛している保持解放機構が展開を開始する場合の反動ト
ルク，展開を終了する場合のラッチ衝撃トルクがシステムの許容擾
乱トルクを超えないように，ダンパ等の速度制御要素を利用して初
期展開速度およびラッチ速度の適正化を行っている．
b.アライメント 展開構造が機構を介してシステムに固定さ
れる場合，そのインタフェース面のアライメント規定と管理がミッ
ション性能に大きな影響を及ぼす．展開構造が機構を介してシステ
ムに固定される場合，そのインタフェース面は展開構造全体のアラ
イメント基準となる．このアライメント基準面を機構の取付面その
ものにしてしまうと組立時に基準を設定することができないため，
計測コンフィギュレーションを考慮した適当な位置にアライメント
ミラーを取付けたり，取付精度をリーマボルトで決まるアライメン
ト精度で保証するといった方法がとられる．
が大型になってくると，展開途中の経路で他の展開構造や衛星バス
構造との干渉がないように，あるいは展開完了時に他の通信機器の
レイパスとの干渉，姿勢制御用センサの視野への干渉を起こさない
ように慎重に設計を行うことが必要である．
源（駆動回路，ヒータ），コマンド（駆動回路，ヒータ），テレメト
リ（駆動回路，温度，マイクロスイッチ），電磁干渉（EMI:elec-
tromagnetic interference），静電気放電（ESD:electrostatic dis-
charge）等が挙げられる．電源については，使用電圧，リップル
電圧の許容量，電圧ドリフト変動の許容量，電力等について，シス
テムの仕様，制約条件との調整が必要であり，コマンド・テレメト
リに関しては通信プロトコル，信号規格，チャネル数，サンプルレ
ートなどの調整が必要であり，データを蓄積伝送する場合にはその
運用のタイミングや蓄積容量の調整が必要となる．さらに，システ
ムに有害な影響を与える電気のノイズの発生の防止，また駆動回路
の異常動作時のシステム保護のための遮断回路の設置が求められ
る．
