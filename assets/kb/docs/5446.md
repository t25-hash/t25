# 5・2・1 ファクシミリ(facsimile,fax)の原理

> **章**: 第5章 情報通信機器
> **節**: 5・2 ファクシミリ

ファクシミリは,送信原稿を読み取って離れた場所にある受信側へ送り,受信側は受取った信号を記録紙上に再現する装置である.現在,市場ではITU-T 勧告で定められたグループ3(G 3機)が広く用いられている.G 3ファクシミリは以下の1~7のプロセスで動作する.1送信原稿を読み取り電気信号に変換する.2冗長度抑圧符号化により送信するデータ量を削減する.3削減した電気信号を変調して受信側に向け送信する.4受信した信号を復調して符号化されたデータに戻す.5符号化されたデータを復号化して元の電気信号に戻す.6電気信号を記録に適した形に変換する.7変換された信号を記録紙上に再現する.
