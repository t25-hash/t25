# 5・2・2 ファクシミリの主要技術

> **章**: 第5章 情報通信機器
> **節**: 5・2 ファクシミリ

a. 読取り技術原稿の濃淡を電気信号に変換する方法として、初期には図5・6に示すように、原稿を巻付けた円筒ドラムを回転させ、光源で照射した原稿の濃淡をヘリカル状に走査し、レンズ系を介して1画素に相当する光電管またはフォトダイオードで光電変換する円筒走査方式(機械式走査)が用いられた。しかし、近年半導体技術の発達により、図5・7に示すような多数の光電変換素子および周辺回路を高密度に集積し、光電変換、走査機能を内蔵(電子式走査)したライン型センサ素子(イメージセンサ)が用いられるようになった。走査機能を構成する半導体の種類によりCCD(charge coupled device)型、MOS(metal oxidesemiconductor)型がある。イメージセンサ上に、原稿幅に相当する1ライン画像をレンズ系で縮小結像させる縮小光学系が用いられている。縮小光学系は光路長が長く、装置をコンパクトにするために折返しミラー等が使われている。電子式走査の採用により機構部は主走査が不要となり、原稿を送る副走査だけになった。その結果、装置の高信頼性、低コストが実現された。さらに光学系の小形化のため、図5・8に示すような、複数の発光ダイオード等を並べた光源、レンズアレイによる等倍光学系、センサ素子、周辺回路を一体化し、原稿に密着して読み取ることができる密着型イメージセンサ(CIS : contact image sensor)が開発された。この密着型イメージセンサには薄膜型とマルチチップ型がある。薄膜型は光電変換素子にCdS-CdSe膜、アモルファスシリコン等が用いられ、走査回路にはハイブリッドIC,TFT が用いられ図5・6 円筒走査方式図5・7 電子式走査方式(縮小光学系)ている。マルチチップ型はCCD 型、MOS 型イメージセンサを複数配列して長尺化したものである。また、近年原稿のカラー化に対応すべく密着型カラーセンサが実用化されている。密着型カラーセンサには、1本の光源とRGB 3色のオンチップでカラーフィルタを備えた密着型イメージセンサにより同時に読み取るオンチップ方式、RGB 3色にフィルタを切換え1個の光源、1個のイメージセンサでフィルタを機械的に交換して読み取るフィルタ切換方式、RGB 3色の光源をもち1個の密着型イメージセンサで光源を切換えて順次読み取る光源切換方式などがある。b. recording)は、熱溶融性または熱昇華性インクを塗布したフィルムの背後から感熱ヘッドで加熱し、溶融または昇華したインクを紙に移行させるもので、装置が簡単で信頼性が高く、カラー化が容易な記録方式である。しかし、転写フィルムのランニングコストが高く普及に至らなかった。業務用ファクシミリでは図5・11に示す記録速度の速い電子写真記録(electrophotographic recording)が主流となり、今日のファクシミリ、コピー、プリンタを兼ね備えた複合機の主要な記録方式として採用されている。電子写真記録はドラム状の感光体を均一に帯電させ、細く絞ったレーザ光を高速回転するポリゴンミラーで走査し、感光体上に静電潜像を形成し粉体トナーで現像した後、紙に転写定着させるものである。感光体、帯電器、現像器、クリーナ等を一体化してカートリッジとすることで容易に交換することができるようになり、メンテナンスの煩雑さが改善されて普及が進んだ。加えて最近の家庭用ファクシミリにおいて、プリンタとして使われることで信頼性が格段に向上したこと、またカラー化が容易に達成できるという理由から、従来の感熱記録、熱転写記録に代わってインクジェット記録が採用されている。インクジェット記録(ink jetrecording)では、図5・12に示す圧電素子の体積変化を利用したピエゾ方式と、図5・13に示す液体の膜沸騰を利用したサーマルジェット(バブルジェット)方式の2方式が主流を占めている。さらに事務用複合機においてはカラー化が始まっている。1組の感光体、レーザスキャナと4本の現像器、転写ドラム(転写フィルム)を備え、色ごとの画像を順次転写ドラムに形成して転写ドラム上で色合成を行い、最後に転写ドラムから紙に転写する方式と、高速化を求めて、4組の感光体、レーザスキャナ、現像器を備え、一度に4色の画像を紙に重ね合わせていくタンデム方式のカラー電子写真記録方式がある。このように、従来白黒2値であった記録も、図5・9感熱記録図5・10 伝送技術ファクシミリにおける伝送技術は、変復調技術(電話網用ファクシミリに対応)と冗長度抑圧技術の二つに分けられる。電話網用のファクシミリに対応した変復調技術はITU(国際電気通信連合)により標準化されており、使用されているのはV.27ter(4800bps)、V.29(9600bps)、V.17(14400bps)、V.34(33600bps)およびそれに付随する規格である。近年、ビジネス用途のファクシミリでは、V.34に適合した、通称スーパーG 3といわれる機種が主流となっている。冗長度抑圧技術は、送信する画情報の主走査方向の白または黒の連続ビット数に対し、あらかじめ発生頻度を想定して定められたコードを与えることにより冗長度を抑圧する一次元符号化と呼ばれる方式と、主走査方向のデータと副走査方向のデータ(次ライン)の画情報の相関を基に符号化する二次元符号化と呼ばれる方式が用いられている。これら符号化方式はITU により標準化されており、ITU-T 勧告T.4で一次元符号化方式のM H(modifiedhuffman)符号化方式、二次元符号化方式のMR(modifiedread)符号化方式が標準化されている。またITU-T 勧告T.6でMR 方式を拡張したMMR(modified modified read)符号化方式が標準化されている。これは、伝送路でのデータ誤りを再送により訂正することを前提に、MR 符号化におけるライン同期信号を省略するとともに、1ラインの最短処理時間0msを認め圧縮率の改善を図った方式である。このほかITU-T 勧告T.82でJBIG(jointbi-level image experts group)符号化方式が定められている。JBIG 符号化方式は、参照画素と、その周囲10画素を基に、1024図5・12 インクジェット記録(ピエゾ方式)図5・13
