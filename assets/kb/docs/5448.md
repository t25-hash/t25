# 5・2・3 ファクシミリ装置の種類

> **章**: 第5章 情報通信機器
> **節**: 5・2 ファクシミリ

a. 装置の国際分類ファクシミリ装置はITU-T 勧告のT.0で分類が定められ,T.1~4などで各グループごとに標準化されているが,現在では電話網を用いたファクシミリ規格のうち,グループ3(G 3機)が市場のほとんどを占めている.グループ1(G 1機)やグループ2(G 2機)は勧告として残存しているのみで,実機を見ることはない.ディジタル網を使用するグループ4(G 4機)機能をもつファクシミリも出荷されているが,G 3機の機能拡張用オプションとして搭載されることが多い.また近年では,ファクシミリ通信をIPネットワーク上で行うインタネットファクシミリが出荷されている.送信原稿の画情報を電子メールの添付ファイルにして送信するもので,通信手順は電子メールと同等である.このインタネットファクシミリの標準化はIETF(Internet Engineering Task Force)により行われており,RFC2301~RFC2305に記述されている.このRFCを基にITU-T で勧告化がなされている.表5・3に勧告番号を示す.モバイル情報機器(PDA : personal digital assistance)は,個人の住所録・電話帳を電子的に置換えた電子手帳から,スケジュール機能だけでなく携帯電話などとの接続による通信機能を得て,電子メール/インタネット端末へと変ぼうし,単なるPIM(personalinformation manager)デバイスからパーソナルな情報ツール,すなわちパーソナルIT 端末として大きな市場を形成しようとしている.ここでは,モバイル情報機器を概説し,デファクトスタンダードへ向かう技術として,主要なOS,また,インタフェースとして重要なIrDA,Bluetoothの状況について述べる.
