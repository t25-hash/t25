# 5・3・1 モバイル情報機器概説

> **章**: 第5章 情報通信機器
> **節**: 5・3 モバイル情報機器(PDA)

携帯型モバイル情報機器は電卓がメモリ容量の拡大により数値のみでなく文字を記憶できる段階に始まり、半導体技術の進展によって漢字をも表示できる段階から電話帳・住所録やスケジュール機能をもつような電子手帳へと進化し、携帯電話などとの接続による通信機能を得て、電子メール/インタネット端末へと変わりつつある。表5・4にPDA のおもな機能を示す。モバイル情報機器は個人がつねに携帯して使用するためのツールであることから、小形軽量であることが必要とされる。ノートPCの小型版としてキーボードを有するタイプのものと、キーボードをなくしてペン入力にし、より小形化したタイプの2種類があった。近年では、キーボードを備えペン入力機能をもつものや、ペン入力を主体とするが超小形なキーボードを備えるものも登場している。キーボードタイプ(以降、これをハンドヘルドPC型と称す)で注目されたものは、ヒューレットパッカード社のHP100/200LXであり、MS-DOS をROM に搭載し、パーソナルコンピュータで開発されたアプリケーションが利用できるハンドヘルドPCとして表5・4 モバイル情報機器の代表的な機能機能内容備考電話帳いわゆる電話帳(会社、自宅、携帯、ファクシミリ番号など)PIM 機能の一つ住所録住所、電話番号やEメールアドレスなども含まれるPIM 機能の一つスケジュールカレンダーに対応した予定表、アラーム機能との連動PIM 機能の一つ備忘録作業すべき項目、期限など。スケジューラとの連動PIM 機能の一つメモ帳各種のメモ通信機能パソコンとの通信、携帯電話等との接続機能などその他辞書(英和、国語、漢字など)、電卓、タイマ、世界時計、音楽再生機能、画像(静止画・動画)再生機能、電子ブックビューア、ゲームなど図5・15 モバイル情報機器の構成例現在でも根強い人気を保っている。この流れはWindowsCEをOS とするモバイル情報機器に受け継がれている。一方、ペン入力を基本として、手帳サイズにこだわったもの[以降、これをPalm(パーム)型と称す]として、わが国においてはシャープのザウルスがあり、世界的にはUS ロボティクス社(現在はパームソース社)のPalmがある。いずれも独自OS であるが、ザウルスはPCに置換わる個人の電子ツールからスタートしたのに対し、Palmは初めからPCコンパニオンやビューアとしてスタートした。形状や生い立ちに差はあるものの、機能としては表5・4に示したものを有しており、個人のビジネス用のPIM 機能のためのツールから、音楽再生や動画像再生機能などをもった、個人のエンタテイメントツールへと変わりつつある。
