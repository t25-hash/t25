# 8・3・2 形状精度計測方法

> **章**: 第8章 機構
> **節**: 8・3 試験評価法

形状精度（surface accuracy）測定には，表面形状測定とアライ
メント精度測定がある．近年の宇宙機の構造は大型で，かつ柔軟な
構造になっているため，これらの測定を行う場合には，従来の剛体
の測定方法に加え，重力による変形を考慮した測定方法の工夫が必
要になる．
a.表面形状の測定 望遠鏡やアンテナの反射鏡面などでは，
表面形状が性能を支配するため，高精度な表面形状の測定が必要で
ある．表面が比較的硬い構造の場合には接触式のプローブを
テーブル上で走査して，鏡面の凹凸を測定する方法がよくとられ
る．表面が金属メッシュなどの軟らかい面の場合や大型の構造の場
合には，写真測量を利用した三次元形状測定装置（photometry
system）やビデオカメラを利用した三次元形状測定装置（video-
metry system）を使用して鏡面形状を測定する．このとき，重力
の影響で大きく自重変形するような柔軟で大型の構造の場合には，
ミューゼス（MUSES）Bに搭載された高精度メッシュアンテナの
ように，鏡面精度をいくつかの境界条件で測定，数学モデルを綿密
にアップデートして無重力の形状を推定する方法 や，技術試験
衛星Ⅷ型（ETS-Ⅷ）に搭載された大型アンテナのように構造をモ
ジュール分割し，それぞれのモジュールを重力方向，重力と反対方
向それぞれの姿勢で測定し，その計測結果から無重力の形状を算出
し，かつ各モジュールの変形量を数学モデルに反映して全体構造の
形状精度を推定するという方法 がとられる．
b.アライメント精度（alignment accuracy）測定 アライメ
ントの測定は角度の高精度な測定が基本である．測定対象を高精度
に位置決めするロータリテーブル，基準面となるコーナキューブ，
レベル計，オートコメリメータやセオドライドといった高精度角度
測定装置を利用して行う．これらの機器を用いた測定結果に基づ
き，シム（調整分解能に対応した基準厚みをもつ金属片）を利用し
てアライメント調整を行う．アライメント測定，調整の場合も構造
物が大型で柔軟になった場合，重力の影響を大きく受けることにな
る．この場合，構造物を高い剛性のジグでつり下げ，変形を最小に
すると同時に，最終的なアライメント精度は構造解析モデルにより
推定する．
