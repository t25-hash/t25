# 8・4・1 ETS- 搭載展開アンテナ跳ね上げ・展開機構

> **章**: 第8章 機構
> **節**: 8・4 機構設計の実例

図8･26に技術試験衛星Ⅵ型（ETS-Ⅵ「きく」6号，engineer-
ing test satellite Ⅵ）に搭載したアンテナモジュールを示す．
ETS-Ⅵには20GHz -S帯用開口直径3.5mの展開・跳ね上げ式ア
ンテナ主反射鏡面（20GHz-S帯主反射鏡面）と30GHz-C帯用開
口直径2.5mの跳ね上げ式アンテナ主反射鏡面（30GHz- C帯主反
射鏡面）が搭載された．ロケットによる打上げ時には20GHz- S
帯主反射鏡面は全体を三つの部分に分割して両翼を折り畳み，かつ
鏡面全体を根元で折り曲げてタワーモジュールに固定し，30GHz -
C帯主反射鏡面は鏡面の根元のみで折り曲げて固定して収納する．
これらのアンテナ鏡面は，軌道上においてアンテナタワーとの固定
部が解放されると，ヒンジ部に取付けたばねの力によって展開を開
始し，所定の位置においてラッチ固定される．図8･27，8･28には
それぞれの主反射鏡の構造および外形寸法を示す．反射鏡面はアル
ミニウムハニカムコアをCFRP（炭素繊維強化プラスチック）の
スキンで挟んだ薄いサンドイッチ構造になっており，パラボラ面を
形成している．この薄い反射鏡面構造に剛性を与えるため，円柱形
状を近似したトラス構造体で多数のスタンドオフを介して支持して
いる．
図8･29に20GHz - S帯主反射鏡面の跳ね上げ機構を示す．機構
は二つで一組になっており，①ハウジング，②回転軸，③転がり軸
受，④ねじりコイルばね，⑤コイルばね案内筒，⑥ラッチアーム，
⑦ラッチ爪から構成されている．展開力はねじりコイルばねによっ
て与え，展開の終了時にはラッチ爪によって鏡面支持トラス構造体
を構成するトラス部材の一部をつかんでアンテナ鏡面を固定する．
図8･30には30GHz - C帯主反射鏡面の跳ね上げ機構を示す．二つ
で一組となって取付けられ，①ハウジング，②回転アーム，③滑り
