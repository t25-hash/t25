c .アンテナ指向（antenna pointing）角度調整機構 アンテ
ナは展開ブームによって衛星からの所定の位置に離隔保持される
が，製造公差の積上げや排湿変形あるいは無重力解放変形などによ
って指向角度に誤差が生じるため，その大きさによっては指向角度
を調整する必要がある．そこで，初期運用に限る条件でアンテナを
微小角回転するアライメントアジャスト機構（AAM:alignment
adjustment mechanism）を備えている．この機構は二軸ジンバル
機構であり，動作範囲が狭くてよいが，バックラッシがないことと
高剛性であることの要求があるため，弾性ヒンジ（elastic hinge）
を用いた方式を採用している．
図8･36はその構成を示したものであり，駆動されるフランジを
三角形の頂点に配置した三つの砂時計型弾性ヒンジで支持してい
る．うち1本Aは固定ヒンジであり，残り2本B，Cを直動させ
ている．B，Cを同じ方向に駆動するとフランジは固定ヒンジを支
点に 軸回りに傾斜し，B，Cを逆方向に駆動するとフランジは
固定ヒンジを支点に 軸回りに傾斜する．
ロック機能が必要であることから，ステップモータ・減速機・ねじ
機構で駆動している．また，ねじ部のバックラッシ除去のため，軌
道上で発生する荷重以上のプリロードをスプリングで付与してい
る．なお，高真空で使用するためねじ機構の潤滑には固体潤滑とし
て金めっきを施し，真空グリースを併用して固着を防止している．
