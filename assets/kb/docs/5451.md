# 5・3・3 モバイル情報機器用OS の動向

> **章**: 第5章 情報通信機器
> **節**: 5・3 モバイル情報機器(PDA)

モバイル情報機器用のOS としては,消費電力の小さなCPU で動作し,必要とするプログラムサイズの小さいものが必須であったが,半導体技術の進展により必ずしもこれらが最重要ではなくなってきている.むしろ,多くの機能を容易に追加できることやオープン性が重要となっている.携帯情報端末はPIM 機能などに加えて多くの高機能を実現するため,CPU は8ビット→16ビット→RISC(reduced instruction set computer)へと高性能化し,プログラム容量は年々倍のスピードで増加している.現在,携帯情報端末用OS としては下記のものが開発されている.その概要を以下に記述する.a. PalmOSアメリカのUS ロボティクス社が1996年5月にPilotとして発売したPDA に初めて搭載されたOS である.全メモリ128kB(キロバイト)でPIM 機能を実現していた.現在では,アメリカのパームソース社が提供している.バージョンアップが進められ,世界のPDA 市場の7~8割のシェアをもつといわれる(1999年IDC調べ).OS の機能を拡張できることなどが特徴である.マルチスレッドが可能で,画面制御と通信制御などを同時に行うことができる.シングルタスクOS であり,アプリケーションの応答性の速さやシンプルなヒューマンインタフェースが特徴である.アプリケーション開発環境が各種プラットホーム上に整備されており,そのため,シェアウェアソフト,フリーウェアソフトが充実している.b. EPOC(エポック)イギリスのPsion(サイオン)社の開発したOS でSymbian(シンビアン)プラットホームのOS である.16ビットから始まり,32ビット対応のEPOC32となっている.現在は,シンビアン社が提供しており,移動体通信デバイス向けへの改良がなされている.高信頼性,機能ごとにコンポーネント化(オブジェクト指向),モジュラ指向と柔軟性,ミドルウェアが充実,すべてのソースコードの公開を特徴としている.PDA だけでなく,携帯電話用組込みOS としても採用されている.c. WindowsCE(ウィンドウズシーイー)マイクロソフト社がPegasusというコードネームで開発していた,家電製品向けに開発されたOS である.WindowsNT と同様にマルチプラットホームをサポートしていることが特徴である.日立製作所のSHシリーズ,MIPS 系CPU,PowerPC,ARM,x86アーキテクチャのサポートが決定している.マルチタスク/マルチスレッドをサポートした32ビットOS である.周辺ライブラリ,ドライバなどの機能が豊富で,マイクロソフト社がコンパイラからデバッガまでの開発環境を提供し,Windows PCとのデータ互換性があり,強力なグラフィカルユーザインタフェースや通信機能,さまざまなMPU(micro processing unit)で利用可能であることなどが特徴とされている.リアルタイム性は組込み向け専用リアルタイムOSより多少低い.現在は,手帳型PC向けに特化されている.Pocket PCと呼ばれるものは,このOS と電子ブックリーダやストリーミングメディア再生プレーヤーソフトなどをバンドルした手帳型PC向けソフトのことである.d. ザウルスOSシャープが開発したわが国唯一のモバイル情報機器用OS である.わが国におけるモバイル情報機器市場の約5割のシェアをもつといわれる.マイクロカーネルをベースとしたリアルタイム性のあるマルチタスクOS である.オープン性の乏しい独自OS であったが,1997年にPIM スクリプト技術とアプリケーションソフトの開発環境を公開しており,2001年にはJavaによるアプリケーション開発環境のオープン化が行われた.現在,ザウルスOS は開発中止となり,ザウルス本体にはLinux(リナックス)が搭載されている.
