# 8・4・4 ALOS搭載合成開口レーダ（PALSAR）

> **章**: 第8章 機構
> **節**: 8・4 機構設計の実例

7･7節に示すALOSに搭載している合成開口レーダ（PAL-
SAR）は1枚の長さが約2.2m，幅約3.1mのアンテナパネル4
枚を屛風のように折り畳んで衛星に搭載している．展開後には衛星
構体に対して約40°傾斜するように展開する必要があり，特殊な展
開シーケンスと展開機構を採用している．
図8･39は展開シーケンスを示したもので，(１)4枚を重ねたま
ま衛星に対して直角に起こす90°展開，(２)4枚を重ねたまま全体
を地球指向方向に傾けるオフナディア展開，(３)4枚パネル保持を
解放するパネル展開，により保持解放展開が完了する．このパネル
は1枚の質量が約100kgあり，ばね展開では対応できないため，
いずれのシーケンスも基本的にはモータで低速展開する方式をとっ
ている．
a.90°展開 4枚のパネルを重ねて保持した状態で，後にオフ
ナディア展開するベースと機構を一緒に衛星構体に対して立てる展
開である．大きな構造物であるために約1.5m離れた2箇所のラ
ッチポイントを設けている．ともに確実なラッチを行う必要がある
ことから，1台の駆動部からの展開トルクをトーションバーで2方
向に分け，どちらが先に展開してもトーションバーがねじれること
で他方がラッチするトルク差動方式を採用している．
b.オフナディア展開（oﬀ-nadir deployment ) PALSARの
特徴的な展開であり，この段階でパネルは4枚が折り畳まれた状態
で衛星に対して約40°傾けられる．後のパネル展開のベースになる
部分も兼ねているため，後述のパネル展開を考慮して図8･40に示

すようにアームを屈曲する方式としている．パネル展開では剛性付
与のためにトラスを立ち上げる必要があり，その駆動部がアーム部
を自走する必要がある．そこで，収納時に屈曲してずれていても展
開時にはまっすぐになるラックアンドピニオン（rack and pinion）
とリニアガイドレールをアームに併設している．
c .パネル展開 図8･41に示すようにオフナディア展開完了後
のパネルは屛風のように折り畳まれており，パネル保持機構を解放
後，パネル間のヒンジに装着したコイルばねと展開トラスに装着し
たコイルばねおよびパネル間展開同期機構で初期自由展開を行い，
自由展開停止後にモータ駆動展開する．モータ駆動展開では，オフ
ナディア展開でまっすぐにつながったオフナディア展開アーム上の
ラックとガイドレールを利用し，ラックアンドピニオンで駆動され
るトラスガイドが移動して，パネル背面に折り畳んでいたトラスを
立ち上げる方式を採用している．
パネル展開は，メインステム駆動機構が第2アームの終端でラッ
チし，サブステムがまっすぐになってラッチして完了する．このと
き中央部のパネルが90°展開し，同期機構でリンクされている端部
のパネルが中央部パネルに対して180°展開する．
