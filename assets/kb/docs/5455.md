# 5・4・2 ウェアラブルPC

> **章**: 第5章 情報通信機器
> **節**: 5・22 に示すような通話時には金属バンドを開き,手にもって使用

PCを自然な形で身に付けて行動し,快適かつインテリジェントな活動をしたい,という願望がウェアラブルPCを生む原動力となった.このようなウェアラブルPCの基本構成は,1981年に高校生のマン(Mann)が作ったWear Compで具体的に示された.このシステムはAppleIIで使われていた6502を搭載したコンピュータを背負ってもち,これでカメラやフラッシュなどの撮影装置を制御するものであり,表示装置としてはヘルメットに付けたビューファインダ用CRT,入力装置としてはマイクロスイッチを用いている.その後,PCの小形軽量化が進み,種々の高性能なHMD(head mounted display)が開発され,入力装置としてもキーボード,タッチパネル,音声入力,運動検知など幅広い選択肢が出てきた.またネットワーク環境も発展したので,現時点ではウェアラブルPCの基本形は,小形PCを肩や首から下げるか腰に装着し,マイク,ヘッドフォン,HMD のマルチメディアインタフェースとGPS,カメラなどのセンシングツールを身に付け,小形PCがワイヤレス通信を介してネットワークに接続されている,というものになっている.新たな可能性を秘めたウェアラブルPCではあるが,その有力な活用法については多くの研究者,開発者による模索が続いている段階といえる.今のところ,限定領域内での実用化が先行しそうであり,以下には小規模工場におけるウェアラブルPCの応用検討例について述べる.応用検討前提は小形CNC(computer numerical control)旋盤が数十台稼働する小規模な工場である.機械オペレータは,図とカメラ,マイクを首から下げ,工場内を自由に動き回れるようにしてある.入力は簡単なボタン操作と音声認識とを併用した.表示装置はHMD のほうが行動性に優れるが,今回のケースは機械工場ということであり,安全性を考慮してPCの液晶パネルをそのま図5・26 ウェアラブルPC の外観ま利用している.工場内でのウェアラブルPCのアプリケーションは以下のようであり,情報管理とコスト削減において大きなメリットが期待できる.1サーバに蓄えられた旋盤のマニュアル情報等を閲覧し,高度な機械操作を進める.2現場の旋盤トラブルやノウハウなどのレポートを作成し,サーバにその情報を蓄える.3ネットワークを通じ,遠隔地の熟練者や機械メーカの専門家からの作業支援を受ける.4旋盤や他の装置の保守管理や操作段取り等を行う.
