# 8・5・2 アクチュエータ

> **章**: 第8章 機構
> **節**: 8・5 機構に関する最近の話題

衛星に使われるアクチュエータはモータかばねがほとんどである
が，保持解放機構では近年新しい機構が使われ始めている．また，
光学センサの微調整に電歪素子が使われた例もある．
a.形状記憶合金 変位は小さいが大きな力を発生できること
から，保持解放機構に使われている．これはフランジボルトという
製品であり，図8･43に模式的に示すようにUノッチを入れたボル
トで締結しておき，解放するときにはリング状の形状記憶合金を動
作させてボルトを破断するものである．解放衝撃は小さいが，ヒー
タによる加熱を必要とするため，複数の箇所で使用しても同期はと
れない．
b.パラフィンアクチュエータ パラフィンが溶融して膨張す
る相変化の体積力を利用するもので，やはり保持解放機構に使われ
ている．図8･44はその構造を模式的に示したもので，基本的には
油圧ピストンと同じ動作を行う．比較的ストロークが大きく，力も
あるので，ピンの押込みや引抜きなどに使われる．形状記憶合金と
同様に解放衝撃は小さいが，ヒータによる加熱を必要とするため，
複数の箇所で使用しても同期はとれない．
