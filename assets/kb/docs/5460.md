# 6・2・1 協調型ディジタル機器ネットワークの分類

> **章**: 第6章 ネットワーク接続機器
> **節**: 6・2 協調型ディジタル機器

協調型ディジタル機器という呼称は、まだ定義が定着していない用語であり、ネットワークでつながるという部分ですら、さまざまな形式のものが、共存、あるいは競合関係にある。理想をいえば、ある1規格に統一されたネットワーク環境に、すべてのディジタル機器が接続、利用可能であるべきである。しかし、そうした標準化は、規格を決めている間に技術が進歩してしまうこと、そして機器や使用場所によって求められるものが違うという2点において困難を伴う。規格の統一には案の作成、検討、規格書の作成にそれぞれ最低1年は要するが、すべてを網羅しようとすればするほど、関係者が増えるためさらに時間がかかり、決まるころには陳腐化してしまうのが実情である。また、ネットワークに求められるものは、情報の転送レートから接続手段まで多岐にわたる。接続が有線か無線かで求められる方向性が異なり、個人利用とオフィス利用では接続する機器数や配線の長さが異なる。またテレビや据置き型のパソコンでは問題ない大きさの端子が、ポータブル機では採用できない場合があったり、利便性を追求して電源供給までサポートしようとすると、電池駆動の機図6・12 従来の負荷分散、クラスタ構成による信頼構成図6・13 ネットワークの棲み分け器には不利になる場合がある。また有線の場合、新たに設置工事の必要があるか否かは、家庭での普及にとっては重要な要素になる。以上のような背景で、全機器が一堂に会する一元化されたネットワークというのは理想的だが実現が難しい。そこで、何かに特化して利害の一致する機器だけの共通規格が推進されることとなるが、その特化の方向性は、大まかに次のようになる。(1) 個人規模のモバイル(mobile)ネットワーク。(2) パソコンを中心とした、オフィス、家屋内ネットワーク。(3) テレビなどAV 家電を中心にした家屋内ネットワーク。ここで(1)は個人が持ち歩く機器の協調を図るためのネットワークで、屋外へ移動しても使えることが重要であり、パソコンがその場になくても機能することも重要である。(2)はパソコンに簡単に周辺機器や別のパソコンをつなぐことが第一で、持ち歩きよりも、高速性やつなげる機器数の多さが重要である。一方(3)はディジタルコンテンツを扱うAV 機器で、データの配信、受信、編集、ライブラリ管理、視聴および機器の遠隔操作を連携して行うためのネットワークである。これらの棲み分けを利用場所軸とAV/情報家電軸で表したのが図6・13である。以下では、具体的にどのような機器、どのようなネットワーク技術が、現状製品として登場し利用されているかについて言及し、それぞれがどう関わり合って上記のような棲み分けを形成しているかを解説する。
