# 6・2・2 協調型ディジタル機器の機能分類

> **章**: 第6章 ネットワーク接続機器
> **節**: 6・2 協調型ディジタル機器

まず、ネットワークを構成する機器おのおのの役割について整理する。機器には単体で十分機能するものから、何らかの機器につながらない限り、機能を発揮できないものまでさまざまであるが、機器全体で見て必要な機能は、大まかに次の三つの要素に分解できる。(1) ネットワークの外部からデータやコンテンツを入力する。(2) 入力したデータを処理(整理、加工、保存)する。(3) 処理されたデータを外部に対して出力、表示する。この三つの要素それぞれを担当する下記のような機器が存在している。a. データ入力機器協調機能をもったデータ入力装置には、静止画、動画など画像を入力するもの、他のネットワークから画像、文字、および音楽データなどを取込むための機器がある。. ディジタルスチルカメラ、ディジタルビデオカメラ静止画像データを得るための機能を備え、単体動作可能な機器で一番ポピュラーなのは、ディジタルスチルカメラである。1995年ごろから普及を始め21世紀に入って動画のビデオカメラに比肩する台数が生産されるようになった。その背景には、パソコンやプリンタの普及と進化によって、静止画の扱いが容易になってきた事情に負うところが多い。撮影した画像ファイルはおもにUSB 接続もしくは、記録メディアの交換によって他に受渡される。またPCを介さずともプリント処理ができるものも登場してきている。また動画をおもに扱うビデオカメラの場合には、動画ストリームをディジタルで他の機器に転送するために、IEEE 1394:1995規格の端子が、8―117テープに保存してある動画のダビングや、チューナ、ディジタルセットトップボックスとの連携による録画システムの構成に利用される。. スキャナレンズを介してではなく、リニアセンサを走査して光学的に画像情報を取込む機器である。紙に描かれた情報をディジタル化するために用いられる。画像取込み機能専門のものから、単体でプリンタを内蔵しているファックス兼用タイプまである。また、スキャン方式では本体を据え置いて原稿を読取り面に固定し、走査するフラットヘッドタイプと読取り部を手で動かして読み込ませるハンディタイプがあリ、使われる場所と守備範囲が異なる。接続にはUSB を使うのが一般的であるが、扱う画像サイズの大きいものではIEEE1394を採用しているものもある。. ディジタル録音機メモリICにディジタルで音声、音楽を録音する装置である。取込んだ音声は、パソコン等で再生できるファイル形式で保存されており、自在に他の機器とのやりとりが可能なことが、従来のカセットテープレコーダと異なる点である。録音した音声データのやりとりにはUSB 端子やメディアそのものの交換によるものが多い。. ディジタルチューナセットトップボックス(set-topbox)とも呼ばれる。TV 放送のディジタル化に伴い、受信した動画をディジタルデータのまま他機へ受渡しできるチューナが登場してきた。IEEE1394端子が多く用いられ、ディジタルビデオ録画機器との間で著作権情報も伝達し、複製や視聴に制限を設けることができる。従来のTV チューナ同様に、アナログのビデオ信号を出力できるものが多いが、圧縮されたディジタルデータを他の機器でデコード(decode)してアナログ変換することを前提に、アナログの出力をもたず、単体で画像の再生はできずに他との協調動作に依存するものも登場してきている。b. データ処理機器先のデータ入力機器も、これから述べる出力機器も、何らかの処理機能はもっているが、それらは機能が限定されているものである。ここで述べるデータ処理機器は、単機能にとどまらないはん用性や拡張性のあるものを指している。また、ここでいう処理とはファイリングから検索、データの修正加工まで含めた全般的な意味合いである。. パーソナルコンピュータパーソナルコンピュータ(personal computer、以後パソコンと記す)は、インタネット接続や、各種ディジタル機器から得た情報の管理に欠かせない機器である。ただ、一般の家電品に比べ、使いこなすためには専門知識や経験が必要な部分も多い。特に接続する機器の構成を自由に変更できるのがパソコンの特徴の一つであるが、新しい機器をパソコンが認識し、使えるようになるまでの設定をいかに容易にするかが重要である。パソコンはここで解説した機器や、ネットワークのすべてに対応可能なものであり、最近ではAV のネットワークとも接点をもち、パソコンのモニタTV でビデオ素材の編集作業を行ったり、放送の視聴をする使われ方が家庭にも浸透してきている。. PDA(personal digital assistance)PDA は携帯情報端末と翻訳されることが多い。その意味するところは、機能的には一部パソコンに匹敵する能力をもちながら、軽さ、小ささ、駆動する電池のもち時間の長さが小型のパソコンよりも総合的に勝っている端末を指す。表示は液晶パネルを用い、文字の入力はキーボードが使えるものからタッチパネルによる簡略文字入力式まで多岐にわたるが、蓄えられる情報量やデータ入力のしやすさなどはパソコンよりも劣るので、他の機器との連携が欠かせない。多くの場合、USB にデータを変換するアダプタ(クレイドル)との併用でパソコンと連携するが、ほかに赤外線によるPDA 間の情報交換が可能なものや、Bluetooth(ブルートゥース)に対応してパソコンや携帯電話機とつながるものなども登場してきた。. 携帯電話単なる移動電話から、回線処理のすべてをディジタル化したものに置換わって以降急速に協調型ディジタル機器8―118情報・メディア機器としての地位が向上し、パソコン、PDA に並ぶ存在になってきたのが携帯電話である(ここでいう携帯電話は特定の通信方式のものを指すのではなく、持ち歩ける電話機一般を指し、PHS なども含めている)。PDA 機能をもった携帯電話は、独立して一つのジャンルを形成しつつあるので、あえて項目を分けた。直接インタネット上のコンテンツにアクセスできるものから、他の端末とインタネットをつなぐ通信機として動作させる使い方まで、電話機としての通話以外の用途も幅広い。c. データ出力機器ディジタルデータやコンテンツ(con-tents)を人間が利用するには、視角、聴覚で処理可能なアナログ情報にしなければならない、そこで専用の出力装置が必要になってくる。. モニタパソコンやビデオ機器の画面表示を行う機器を総称してモニタと呼ぶ。大別してパソコンの画面を映すためのものと、ビデオやTV チューナの映像を映すためのもの、および双方の機能を併せもつものがある。パソコンとの接続にはD-sub15pinのアナログRGB 接続型が主流であるが、徐々にDVI(digital visual interface)規格に基づくディジタル接続端子搭載のものが増えつつある。現時点ではパソコン、モニタとも両方の接続に対応している機器が多い。一方、ビデオ機器との接続用のものでは、IEEE 1394,HDMI(high-definition multimedia interface)規格の端子も装備しているものがある。. プリンタパソコンのデータ出力装置が主目的のものが多いが、ディジタルスチルカメラの普及と高画質化に伴って、写真印刷をアピールするものが増えてきている。また、動画のビデオの画像をプリントする、家庭電化製品寄りの製品もある。パソコンなしにプリンタ単体で印刷できる環境は、かつてはアナログ入力のビデオプリンタのみであった。それはTV 放送画面の解像度は全世界的に見ても数種類しかなく、統一がとれていたからである。一方、ディジタルスチルカメラでは年々扱える画像サイズが大きくなり、パソコンで加工すれば任意のサイズにできてしまうため、それを印刷するにしてもパソコンから細かく設定する必要があった。しかし、1998年に制定されたDCF(design rule for cam-era file system)および1999年に制定されたDPOF(digital printorder format)という、写真などの画像をファイリングするデータ構造と、印刷指示手段を規格化したものが広く採用されたため、ディジタル系の画像においても、あえてパソコンを使わない印刷が可能になった。現在ディジタルスチルカメラの多くが対応しているため、DCFで異なるメーカ間のファイル認識、受渡しや協調動作が容易になり、DPOFで、カメラ側であらかじめ設定しておいた印刷指示付きの画像データを対応プリンタに読み込ませる、あるいは写真店などの現像所に提出するだけで指定どおりの印刷が可能になる。. ディジタル音楽再生機ディジタルの音楽ソフトウェアはCD(compact disk,compact disc)およびMD(mini disk,mini disc)といったパッケージメディア主流であったが、ディジタルの音楽圧縮技術が一般に普及する一方でパソコンの能力向上、インタネット接続環境の向上と相まって、個人でディジタル音楽コンテンツを作成、インタネットで交換できるようになってきた。MPEG-1 audio layer-3という圧縮形式は通称MP3(エムピースリー)と呼ばれ、一般に普及した圧縮形式の標準的存在である。こうした圧縮音楽の再生環境は、当初パソコンやワークステーション上のみであったが、専用の携帯プレーヤーが登場してから普及が加速した。専用プレーヤーは一般にパソコンと連携して音楽データをダウンロードし、再生する機能をもつ。そのデータのやり取りはUSB 接続によるものが多いが、内蔵するメモリが取外し可能なカード式になっており、直接メモリを交換するものも少なくない。また、記憶容量的に有利なハードディスクドライブやCD-R を扱えるものもある。ただしMP3形式の音楽ファイルは、容易にコピー、配布が可能なために著作権保護上の問題が提起されている。こうしたコンパクトな音楽データはディジタルデータゆえに、音質を劣化させることなく、原理的には無限にコピーが作成可能かつ、配布が容易な点に配慮し、音楽の圧縮に併せて、複製、配布に制限を設ける著作権保護技術も加えた規格がいくつか登場している。
