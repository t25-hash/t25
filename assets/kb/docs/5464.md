# 6・3・2 業界の主要なプレーヤー

> **章**: 第6章 ネットワーク接続機器
> **節**: 6・3 セットトップボックス

PC業界と異なり、技術の業界標準というものが存在せず、各国の放送方式仕様や、個別のサービス事業者の規定する仕様に基づいてセットトップボックスが設計されているので、さまざまな種類のセットトップボックスが存在している。特に、アメリカやヨーロッパでは、新しいサービスを可能にするための技術を搭載した商品開発が家電業界の枠を超えて展開されている。セットトップボックスという最終商品としての開発競争もさることながら、セットトップボックス、ひいては次世代テレビを支える技術の覇権争いが、業界の垣根を超え、従来の家電メーカだけでなく、通信業界、パソコン業界をも巻込み、全世界規模で展開されている。従来のテレビ受像機は、日本の家電メーカの独壇場であったが、このような放送と通信の融合は、全世界規模で開発競争を巻起こし、トムソン、三星、LG、フィリップスといった外国家電メーカはもとより、モトローラ、サイエンティフィック・アトランタのような通信機器メーカ、インテル、マイクロソフトといったパソコン業界、ブロードコム、エーティーアイ(ATI)、東芝などの大規模システムオンチップメーカ、オープンティービー(OpenTV)、ディジオ(Digeo)といったミドルウェアベンダ、エヌディーエス(NDS)、ナグラ(Nagra)などの課金システムベンダ、放送局や放送事業者に加え、電話会社、ブロードバンド事業者などが競争的優位性を保つために買収や提携を繰返し、次世代テレビの技術の覇権を制しようとしているのが現状である。
