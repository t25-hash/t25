# 6・3・4 今後の動向

> **章**: 第6章 ネットワーク接続機器
> **節**: 6・3 セットトップボックス

今後、家庭内での映像エンタテイメントとして、ディジタル放送と通信との融合が加速することが予想され、セットトップボックスないしは家庭内のテレビ受像機に要求される機能は急速に進化していくと考えられる。従来の放送を受信するために必要なセットトップボックスとしてだけではなく、通信ネットワークから配信される映像エンタテイメントやさまざまなサービスを可能にすることが求められるであろう。いままでのディジタル放送はエムペグ2(MPEG-2: movingpicture experts group phase 2)という圧縮技術に支えられて発展してきたが、エムペグ4(MPEG-4)に代表される、より高効率な圧縮技術が、放送およびブロードバンドで使用され、発展していくと予測される。MPEG-4に代表される新しい圧縮技術を使用することにより、例えば、ハイデフィニション映像を7Mbpsと従来の4分の1程度のスピードで伝送できる。高効率な圧縮技術は、送信側も含めて、限られた帯域をより有効に活用することができるの8―121で、多チャネル化、ハイデフィニション映像送信など、大きな変革をもたらす可能性を秘めているといえる。特に、マイクロソフト社は、独自の圧縮技術および著作権保護を基に、放送・通信の新しい仕様としてさまざまな提案をしてきており、標準化あるいはデファクトスタンダード化に向けて、激しい競争が予想される。ホームネットワーキングと著作権保護技術の進化により、いままで単品で機能していた家電製品は相互につながるようになり、コンテンツを他の機器にコピーしたり、どこでもコンテンツを楽しめるようになるだろう。自宅から離れていながら自宅の機器にアクセスし、機器をコントロールしたり、コンテンツを転送してユビキタスで楽しめるようになるだろう。この際に重要なのは、機器の接続互換性を保つためのホームネットワークのプロトコルの標準化、およびネットワーク上でやりとりされるコンテンツの保護技術である。プロトコルがすべての機器間で統一できなくても、複数の異なるプロトコルで動作する商品をつなげて、それぞれのプロトコルを制御する上位概念としての新たなプロトコルも提案され始めている。したがって、セットトップボックスは、従来の放送や通信のサービスを受信するための箱から、エンドユーザがユビキタスに映像コンテンツを楽しむことを可能にする製品になっていくと思われる。文献(1)
