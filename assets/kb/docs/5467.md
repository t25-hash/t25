# 7・1・1 オーディオ・音声圧縮技術

> **章**: 第7章 情報圧縮・セキュリティ技術
> **節**: 7・1 圧縮技術

a. はじめにオーディオ信号や音声信号をディジタルの符号に変換して伝送あるいは記録するための情報量を削減する技術が,オーディオ・音声圧縮符号化技術である.信号をコンパクトにディジタル化することで,音声,音楽,映像など異なる種類の情報を統合的に扱うマルチメディア通信や蓄積などの応用を容易にするとともに,無線周波数資源や蓄積資源などを効率的に利用することが可能となる.音声符号化としては,32kbps(bits/s)程度を対象としたADPCM(adaptive differential PCM)や,4~16kbps程度を対象としたCELP(code excited linear prediction),別名VXC(vector excitation coding)が波形符号化方式として主流を占め,ディジタル携帯電話などの標準に多く用いられてきた.最近ではさらに低ビットレート化をねらった方式としてパラメトリック符号化の研究も盛んに進められている.パラメトリック符号化は,音声を極限までモデル化してその情報を構造化し表現しているため,2~8kbpsの低ビットレート符号化のみならず,ピッチ変換,スピード変換といった音声の変形処理にも向いた分析合成手法である.一方オーディオ符号化では,サブバンド符号化(sub-band cod-ing)方式,変換符号化(transform coding)方式を中心に標準化が進められ,蓄積系や放送系のアプリケーションに広く応用されている.さらに最近では,オーディオ符号化に対してもパラメトリック符号化を用いたものが標準化されている.
