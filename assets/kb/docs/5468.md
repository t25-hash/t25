# 8・4・2 ファシリテーションと可塑性

> **章**: 第8章 機能回復訓練機器（医学的リハビリテーション機器)
> **節**: 8・4 その他

中枢神経系（脳，脊髄）におけるファシリテーション（facilita-
tion）とは，神経の興奮性インパルスが反射中枢に達して，神経活
動が強化されることである．また，可塑性とは外界からの刺激によ
って神経網の構造や重み付けに変化が生じることをいう．可塑性は
外界の変化への適応や，外傷などの構造変化に対応する機能として
大きな役割を果たしている．本項では，中枢神経系の可塑性を外部
からファシリテートする研究について，ヒトの運動機能に関する部
分を中心として概説する．
大脳皮質運動野をファシリテートする方法としては，経頭蓋的磁
気刺激（TMS:transcranial magnetic stimulation）が代表的で
ある．すなわちコイルへ瞬間的に電流を流し，発生した変動磁場に
よって脳内に渦電流を発生させる．さらに，この電流によって並行
して走行する軸索を刺激する．この軸索は他の脳細胞とシナプスを
形成しているため，シナプスの可塑性を誘発することが可能であ
る．
