# 7・2・1 情報セキュリティ技術概要

> **章**: 第7章 情報圧縮・セキュリティ技術
> **節**: 7・2 情報セキュリティ技術

近年、インタネットに代表される情報技術分野が大きく発展してきており、不特定多数の利用者がこれらの技術を無意識に利用する機会が増加してきている。しかし、インタネット上を流れる情報は、第三者が容易に盗み見たり、改ざんしたりできるため、インタネットで商取引を行う場合には注意が必要である。これらのこともあり、1990年代から情報セキュリティに関する学会が活発化し、その重要性も徐々に認識され始めてきた。一方、情報セキュリティといっても、その技術分野はかなり広い。以下に一例を示す。(1) 暗号技術、電子署名および改ざん防止に関する技術分野(2) 鍵配信や鍵共有に関する技術分野(3) 著作権保護に関する技術分野(4) 電子透かしなどの不正複製を検出する技術分野(5) PKI(public key infrastructure)に関わる技術分野(6) PMI(privilege management infrastructure)に関わる技術分野(7) コンピュータウイルスに関する技術分野(8) ネットワークセキュリティの技術分野(9) セキュリティ機能をもつ半導体の技術分野以下に、これらの中から抜粋し、その概要を説明する。
