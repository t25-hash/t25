# 8・2・2 自動化機器の種類

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATMシステム

ATMは，従来営業店のテラー（顧客応対職員）が行ってきた預いを利用者が自ら操作する装置であり，CD等の単機能装置も含め自動化機器は，一般的にその機能により次のように分類される．
を行う現金自動預金機（AD:automatic depository），通帳への記行う現金自動預金支払機，振込処理を行う振込専用機などである．
型，営業店の外壁に顧客操作面を出して設置する外壁型，顧客利用融機関の店舗以外の身近なところにCD/ATMを設置してほしい集まるさまざまな場所にCD/ATMが設置されるようになった．
住宅街や主要道路沿い等，多様化が進んでいる．
金や為替などの業務処理と，それに伴う現金や通帳等の現物の取扱て自動化機器と呼ぶ．
現金の支払い処理のみを行う現金自動支払機，現金の預金処理のみ帳処理を行う自動記帳機，入出金処理や通帳記帳，振込等の処理をまた，設置形態による分類としては，ロビーに設置するロビー設置場所まで移動できる車載型などもある．
