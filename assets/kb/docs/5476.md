# 8・2・3 ATMの機能

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATMシステム

ATMが備える基本機能としては，磁気ストライプ付きキャッシリードおよび発行機能，カード上のエンボスを取引明細票にインプ能，通帳に取引情報をプリントする機能，通帳発行機能，紙幣を計貨の真偽を鑑別する機能などがあり，そのほかに，ディスプレイ機記録機能，防犯対策としての各種センサ機能がある．
今後，さらな携帯電話対応機能も検討されている．
別な観点から，バリアフリーュカードの磁気ストライプの内容を読み取る機能，振込みカードのリントする機能，取引明細票に必要な取引情報をプリントする機数する機能，紙幣の真偽を鑑別する機能，硬貨を計数する機能，硬能，音声ガイダンス機能，キーボード機能，通信機能，ジャーナルる利便性の向上とセキュリティ向上に向け，ICカード対応機能や，機能の検討も進められている．
