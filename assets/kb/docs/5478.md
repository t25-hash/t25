# 8・2・5 現金関連機能

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATMシステム

現金での入出金・振込みサービスを提供している日本のATM
の特徴として，現金関連機能が非常に発達していることが挙げられ
る．
第一に挙げられるのは，紙幣および硬貨の種類や真偽を鑑別する
機能である．24時間稼働が進むにつれ，人目の少ない深夜におけ
る犯罪に対抗するためにも，鑑別機能はますます重要になってきて
いる．
次に，ATMの現金収納ボックスには，現金が直接出入りする収
納ボックスと，収納ボックスの現金が一定量を割った場合や，一定
量を超えた場合に現金を補充または回収する補充・回収ボックスが
ある．ATMの運用において，金融機関または金融機関に委託され
た企業は，補充・回収ボックスの現金が大きく過不足になった場合
のみ，これに現金を補充・回収すればよいことになる．また，補
充・回収ボックスを交換する際に，収納ボックスの現金を用いて
ATMのサービスを継続できるため，利用者に不便をかけなくて済
む．
