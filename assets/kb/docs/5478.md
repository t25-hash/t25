# 7・2・6 個人認証(user authentication)

> **章**: 第7章 情報圧縮・セキュリティ技術
> **節**: 7・2 情報セキュリティ技術

近年、一般へのコンピュータ、インタネット接続環境の普及に伴い、利用者を特定する手段としてパスワードを使う機会が急速に増加している。コンピュータ、ネットワークを利用するためにはさまざまなパスワードを入力する必要があり、一人のユーザが必要とするパスワードの数が急速に増加し、その結果、利用のための手続きが煩雑になり多くのユーザの負担となっている。またパスワードさえ知ることができれば、第三者が容易にユーザに成り済ますことが可能であり、従来型のパスワードによる個人認証の安全性が懸念され始めている。個人認証の方法は、大きく分けて知識による認証、所有物による認証、身体的特徴による認証の三つに分類できる。知識による認証とはパスワード等ユーザの記憶する知識に基づいた認証である。所有物による認証とはICカード等ユーザの所有するものに基づいた認証である。身体的特徴による認証とは指紋等ユーザの生体情報に基づいた認証である。これらの認証方式はそれぞれ単独で用いるのではなく、複数の認証方法を組合わせることでよりセキュリティの高いシステムを構築している。また、上記認証技術のうち、特にその利便性から身体的特徴による認証技術の研究、開発が活発に行われている。身体的特徴もしくは行動様式で人を自動的に認識する方法はバイオメトリクス(biometrics)と呼ばれ、個人を特定する特徴を定量化し、事前に登録したデータと認証時に認識したデータとの比較で認証を行うシステムである。本人であるのに認証できない割合のことを本人拒否率(FRR : false rejection rate)、他人であるのに本人と認証してしまう割合のことを他人受入れ率(FAR :false acceptance rate)と呼び、FRR,FAR ともに低いシステムが信頼性のあるシステムとされている。しかしFRR とFAR はトレードオフの関係にあり、そのバランスは認証に用いられる特徴やシステム全体のセキュリティポリシーに従って定められている。認識に用いられる特徴や行動とは指紋、声、顔、眼球の虹彩、網膜、手のひら、サイン、手首の血管、手の模様などさまざまであるが、どの方式が利用されるかはFAR,FRR の精度以外にも利用者の認証行為に対する受容性や装置の規模等などの要因を考慮して決定されている。また、本人に固有の情報であるという特徴から、バイオメトリクス技術とPKIとを組合わせたシステムの構築も多く行われている。以下にバイオメトリクスの代表例と一般的な精度(FAR,FRR)を示す。(1) 指紋認証(FAR:0.5%,FRR:0.01%)あらかじめ採取した指紋画像との類似度を比較する方式(パターンマッチング)と、隆線の端点や分岐点など、指紋の模様に含まれるそれぞれ固有の特徴点から本人か他人かを識別する方式(特徴照合)がある。多くのアプリケーションにおいて最も普及が進んでいる技術で、アメリカ、フィリピンなどにおいて社会福祉への大規模な適用が進んでいる。デバイスの低価格化、高精度化が進んでいる。(2) 顔認証(FAR:1%,FRR:1%)カメラで撮影した顔の特徴データを基に、本人認識を行う認証方式。認識プロセスが顔を見るという人間の行為と同じであることや非接触であることという特徴がある。(3) 声紋認証(FAR:1%,FRR:0.1%)声紋をディジタルデータ化し照合する認識方式。クライアント側にはマイク以外に特別な装置が不要という導入、利用面での利点があり、電話を用いるアプリケーション等に用いられている。(4) 筆跡認証(FAR:0.2%,FRR:0.6%)人の署名のスピードやストローク、筆圧など、動的な行動そのものを解析し照合する認証方式。サインという自然な行為で認証が行えるという特徴がある。(5) 虹彩認証(FAR:2.8%,FRR:0%)人の目の瞳孔から外側に現れるしわの形状を照合する認証方式。個人に固有で偽造が困難であり、人間の状態による変化が少ないため高精度の認証が可能である。しかし認証装置のコストが高いため、高セキュリティを求められる分野に利用されている。
