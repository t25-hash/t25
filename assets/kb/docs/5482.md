# 8・1・1 通常郵便物処理のしくみ

> **章**: 第8章 社会情報システム
> **節**: 8・1 郵便物処理システム

一口で郵便物といっても通常郵便物(葉書、一般封書等)から小包に至る広範囲なものである。8・1・1項では通常郵便物における機械化を中心として述べる。図8・1に日本の一般的な郵便物処理の基本的な流れを示す。(1) ポストに投函された郵便物は定められた時刻に専用車によってその地域の取集局に集められる。(2) 取集局では、まず郵便物の切手に取扱局および取扱時刻入りのスタンプを押し、かつ普通郵便物と速達郵便物の区分をする。(3) 次に、郵便物上に記載されている郵便番号あるいは住所に基づき、指定配達局ごとに区分けをする[これを差立区分(dispatch sorting)と称する].(4) 区分けされた郵便物はおもに輸送ケースに収納され、定められた時刻に、定められたルートを経て指定配達局へ輸送される。図8・1 郵便物の基本的な流れ(5) 各配達局では郵便物上に記載された住所および宛て名に基づき各配達人の配達区域単位に区分けをする[これを配達区分(delivery sorting)と称する].(6) 区分けされた郵便物は各配達人が、自分の配達ルートに沿って郵便物を順番に並べる[これを道順組立(carrier walksequence sorting)と称する].次に、取集局および配達局内の郵便物の流れを基本作業に分解してみると下記となる。○取集局内における基本作業(図8・2)(1) 取り集められた郵便物の中から厚物も含め大形の郵便物を取り除く。..〔選別作業〕(2) 同一面に郵便物をそろえる。..〔取りそろえ作業〕(3) 切手上に取扱局名および取扱時刻入りのスタンプを押印する。..〔消印作業〕(4) 郵便物上の郵便番号あるいは住所に基づき指定配達局ごとに区分する。..〔差立区分作業〕(5) 同一方面に区分された郵便物束に宛て先の配達局が書かれている札(これを紙札と称する)を載せて縛る。..〔把束作業〕○配達局内における基本作業(6) 郵便物上に記載された住所および宛て名に基づき各配達人の配達区域単位に区分けをする。..〔配達区分作業〕(7) 各配達人は自分の配達ルートの順に郵便物を並べ直す。..〔道順組立作業〕図8・2 郵便物処理の基本作業表8・1 機械処理可能な通常郵便物の大きさの制約内容ケース長さmm幅mm長さ/幅の比厚さmm重さgおもな採用国1140~90~~50日本ほか最も多くの国が基本としている2140~90~1.4~2.5~6~80北ヨーロッパほかヨーロッパ内に広く普及している3約127~292約90~155~約85アメリカ*:日本において定形郵便物として差出すことができる最大厚さは10mmである。
