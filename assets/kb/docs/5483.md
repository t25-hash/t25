# 8・1・2 郵便物に対する制約

> **章**: 第8章 社会情報システム
> **節**: 8・1 郵便物処理システム

各国が必ずしも共通ではないが機械処理効率を高めるため、取扱い対象となる郵便物に基本的な制約を与えている。a. 機械処理可能な通常郵便物の大きさの制約長さ、幅、長さと幅の比、厚さ、重さを制約している(表8・1)。各国とも比較的共通化されている。b. 機械で読取可能な切手の制約取りそろえ作業の効率化のための制約である。切手素材、切手印刷、切手貼付位置などが制約されている(表8・2)。各国まちまちであり統一が十分図られていない現状にある。c. 郵便番号の制約差立・配達区分・道順組立作業の効率化のための制約である。郵便番号の表記構成すなわち表記文字種およびけた数の制約、表記方法すなわちタイプライタを含む印刷活字および自由手書の制約、表記の位置および、郵便番号が表記する情報範囲すなわち差立区分どまりか配達区分作業を包括するものか等が制約されている。特にこの制約は各国における文化、歴史風俗の影響が大であり、それぞれ独自の制約を有する傾向が強く、特に異文化間の相違は大きい(表8・3)。
