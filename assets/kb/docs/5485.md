# 8・1・4 郵便物処理の機械化概況

> **章**: 第8章 社会情報システム
> **節**: 8・1 郵便物処理システム

各国が必ずしも同じ足並みで機械化が推進されているわけではない。8・1・4項では世界において機械化推進の代表格とされている日本の機械化を中心として述べる。日本における機械化は、1967年に郵便物自動選別取りそろえ押印機が開発・実用化され、選別作業・取りそろえ作業・消印作業の機械化が実現し、1968年に自由手書郵便番号自動読取区分機による差立区分作業の機械化がスタートした。また、日本の漢字表記の住所が読み取れる郵便物宛て名自動読取区分機が1989年に開発・実用化され、配達区分作業の機械化が実現した。その後、1997年にはさらに高い読取り機能・性能と住所データベース検索機能を併せ持つ新型区分機が実用化され、道順組立作業までの機械化が実現した。
