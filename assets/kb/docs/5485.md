# 8・3・3 ローカウンタ相談端末

> **章**: 第8章 社会情報システム
> **節**: 8・3 金融機関店舗機器システム

金融機関の営業店では，従来ハイカウンタで行っていた預金の入
出金業務や振込み業務を，人手のかからない自動化機器にシフトし
てきた．これに伴い，ハイカウンタの業務は，新解約，定期預金，
為替などに絞られ，従来に比べ業務量が減ってきている．一方，ロ
ーン相談，資金運用相談などの，顧客に着座してもらい，より高度
な金融商品の営業活動を行うローカウンタを充実させ，営業店を営
業の戦略拠点に衣替えさせようとする動きが活発化している．
ローカウンタ相談端末はこのような背景で生まれたもので，ロー
カウンタにおいて，テラーと顧客が相対し，対話しながら金融商品
の紹介，シミュレーションによる資金運用相談などが行えるととも
に，従来ハイカウンタで行っていた勘定系取引をも含めて処理でき
るようにしたものである．
営業店のローカウンタにそのまま設置できる形態であり，顧客と
の対話用にタッチセンサ付きのカラーディスプレイを備える．顧客
は，ディスプレイ上で商品の説明や，シミュレーションを確認しな
がら，相談を進めていく．テラー側には，窓口端末装置が組み込ま
れており，相談内容から派生する勘定系の取引処理も，席を立たず
に処理することができる．
