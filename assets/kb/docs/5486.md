# 8・1・5 郵便物自動処理機械

> **章**: 第8章 社会情報システム
> **節**: 8・1 郵便物処理システム

以下は最新の機能・性能を有する新型区分機の機能を中心に説明する。a. 新型区分機(advanced letter sorter、図8・3)郵便局内における区分作業(差立区分、配達区分、道順組立)すべてを自動化できる機能を有する。新型区分機は、郵便物の取出・搬送・区分機能を有する本体部、郵便物に記載された郵便番号を含む宛て名を認識する読取り部、読取り部で認識できなかった郵便物を処理するためのVCS(ビデオコーディングシステム)、認識結果をバーコードに変換して郵便物上に印刷し再度読取り確認するバーコード印字・読取り部、郵便局の配達原簿情報を管理し新型区分機の道順組立用データに変換する配達総合情報システムで構成される。以下に各部の主要な機能・性能について説明する。(1) 新型区分機本体部処理対象となる郵便物を受け入れて郵便物を一通ずつ取出し、搬切手貼付位置おもな採用国2特に規定なしりん光インクによるバーフランス、イギリスほか4りん光紙アメリカ、オーストラリア、オランダ、日本ほか表8・3 郵便番号の制約印刷活字自由手書表記位置表記の情報範囲差立区分配達区分主要な採用国1数字4けた○住所ブロック内の最下行最右端が基本○―ハンガリー指定「赤枠」内の記入27けた○住所ブロック内の最上行最左端が基本○△日本35けた○住所ブロック内の最下行最左端が基本○―フランスほか○○カナダオランダオランダ最下行最左端(例)1000NA AMSTERDAM○○イギリス8―138図8・3新型区分機(本体部、350口)[幅:16.3m、奥行:1.7m、高さ:2.0m]図8・4 ビデオコーディングシステム(外観)図8・5 VCS を使った読取り不能郵便物の処理送し、読取り部またはバーコード読取り部の認識結果に基づいて郵便物を区分する機能を有する。道順組立処理のときは配達総合情報システムのデータベース情報より、郵便物を配達順に並び替えることができる。処理するモードおよび処理対象郵便物によって処理能力(3万~5万通/h)が変わる。(2) 読取り部本体部の搬送路途中に設置され、郵便物の宛て名記載面の画像を撮るスキャナ部と、撮像された画像を処理して郵便番号・宛て名を認識する判別部(別置)とで構成される。判別部は、郵便番号だけでなく宛て名(町名、街区番号、棟室番号、氏名)を読み取ることができる。また、郵便物差出人によって宛て名といっしょに印刷されるカスタマバーコードを読み取る機能も有する。(3) VCS(ビデオコーディングシステム、図8・4,8・5)読取り部で認識できなかった郵便物の画像を人手で処理するための装置である。VCD(ビデオコーディング端末)のディスプレイ上に郵便物の画像を表示し、オペレータがキーボードを使って必要な情報(郵便番号、住所番号など)を入力する。図8・6 郵便物自動選別取りそろえ押印機(大形機)[幅:9.7m、奥行:3.3m、高さ:2.3m]図8・7 郵便物選別台付き自動取りそろえ押印機(小形機)[幅:4.4m、奥行:3.5m、高さ:1.2m](4) バーコード印字・読取り部読取り部の認識結果またはVCS の入力結果に基づいて、郵便物上にバーコードを印刷し、その後印刷したバーコードを読み取って正しくバーコードが印刷されたかどうか照合・確認するバーコード読取り部とで構成されている。日本の郵便局では人の目に見えにくい透明な蛍光バーコードを使用している。このバーコードは郵便局内だけで使用される専用バーコードである。(5) 配達総合情報システム郵便局が管理する配達原簿の情報を電子化し、管理する機能を有する。新型区分機本体部で道順組立機能を実現するため、配達総合情報システムに登録された情報を変換し、住所とその配達する順番などのデータを抽出したファイルを作成する。新型区分機本体部ではこのファイルを読み込み、その情報に従って供給された郵便物の区分先を一通ごとに細かく指定している。b. 郵便物選別取りそろえ押印機(culler facer canceller、図8・6,8・7)郵便物上の切手を検知し、表裏、上下をそろえて切手に消印後、普通郵便物、速達郵便物の区分をして集積する。設置の局の規模に応じ大形・小形機がある。処理能力は大形・小形機おのおの3万通/hおよび1.5万通/h程度である。
