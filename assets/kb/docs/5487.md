# 8・2・1 ATM の歴史

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATM システム

1965年ごろに始まった金融機関の第一次オンライン化を背景に,現金の支払いのみを行う現金自動支払機(CD : cash dispenser)が出現した.1975年ごろには,第二次オンライン化が始まり,預金機能をもつ現金自動預金支払機(ATM : automated tellermachine)が開発され,設置台数の著しい増加を見た.1985年ごろに入ると従来のCD/ATM の機能に加え,振込機能や入金された紙幣/硬貨を出金に使用できる紙幣/硬貨還流機能を備えたCD/ATM が開発されるに至った.1998年ごろに入ると,利用者のニーズに応えるため,駅,コンビニエンスストア,スーパーマーケット,住宅街等,金融機関以外の場所にATM が設置されるケースが増え,複数の金融機関に対応する機能,公共料金や税金を収納する機能,定期預金の預入れ・解約機能をも有するものに発展した.このような自動化機器の機能向上および発展の背景としては,電子工学,特に半導体技術や機械工学の進歩,銀行の大衆化,リテール志向の強化,省力化/合理化努力に,利便性を求める利用者のニーズが合致したことが挙げられる.国内市場においては,民間金融機関の店舗数が1993年を境に減少を続けているのに対し,CD/ATM の設置台数は増加傾向にある.ただし,現金払出し機能のみのCD は,平成に入ると減少傾向にあるため,CD/ATM の設置台数増加のほとんどが,ATM であるといえる.民間金融機関のCD/ATM は,1998年時点の推定でおよそ12万台が稼働している.
