# 8・2・1 ATM の歴史

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATM システム

1965年ごろに始まった金融機関の第一次オンライン化を背景に，
現金の支払いのみを行う現金自動支払機（CD:cash dispenser）
が出現した．1975年ごろには，第二次オンライン化が始まり，預
金機能をもつ現金自動預金支払機（ATM:automated teller
machine）が開発され，設置台数の著しい増加を見た．1985年ご
ろに入ると従来のCD/ATMの機能に加え，振込機能や入金され
た紙幣/硬貨を出金に使用できる紙幣/硬貨還流機能を備えたCD/
ATMが開発されるに至った．1998年ごろに入ると，利用者のニ
ーズに応えるため，駅，コンビニエンスストア，スーパーマーケッ
ト，住宅街等，金融機関以外の場所にATMが設置されるケース
が増え，複数の金融機関に対応する機能，公共料金や税金を収納す
る機能，定期預金の預入れ・解約機能をも有するものに発展した．
このような自動化機器の機能向上および発展の背景としては，電子
工学，特に半導体技術や機械工学の進歩，銀行の大衆化，リテール
志向の強化，省力化/合理化努力に，利便性を求める利用者のニー
ズが合致したことが挙げられる．
国内市場においては，民間金融機関の店舗数が1993年を境に減
少を続けているのに対し，CD/ATMの設置台数は増加傾向にあ
る．ただし，現金払出し機能のみのCDは，平成に入ると減少傾向
(C)日本機械学会
にあるため，CD/ATMの設置台数増加のほとんどが，ATMであ
るといえる．民間金融機関のCD/ATMは，1998年時点の推定で
およそ12万台が稼働している．
8・2・2自動化機器の種類
ATMは，従来営業店のテラー（顧客応対職員）が行ってきた預
金や為替などの業務処理と，それに伴う現金や通帳等の現物の取扱
いを利用者が自ら操作する装置であり，CD等の単機能装置も含め
て自動化機器と呼ぶ．
自動化機器は，一般的にその機能により次のように分類される．
現金の支払い処理のみを行う現金自動支払機，現金の預金処理のみ
を行う現金自動預金機（AD:automatic depository），通帳への記
帳処理を行う自動記帳機，入出金処理や通帳記帳，振込等の処理を
行う現金自動預金支払機，振込処理を行う振込専用機などである．
また，設置形態による分類としては，ロビーに設置するロビー設置
型，営業店の外壁に顧客操作面を出して設置する外壁型，顧客利用
場所まで移動できる車載型などもある．また，近年においては，金
融機関の店舗以外の身近なところにCD/ATMを設置してほしい
という利用者のニーズと金融機関のリテール志向がマッチし，人の
集まるさまざまな場所にCD/ATMが設置されるようになった．
スーパーマーケット，百貨店，コンビニエンスストア，駅，郊外の
住宅街や主要道路沿い等，多様化が進んでいる．
