# 8・2・5 現金関連機能

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATM システム

現金での入出金・振込みサービスを提供している日本のATMの特徴として,現金関連機能が非常に発達していることが挙げられる.第一に挙げられるのは,紙幣および硬貨の種類や真偽を鑑別する機能である.24時間稼働が進むにつれ,人目の少ない深夜における犯罪に対抗するためにも,鑑別機能はますます重要になってきている.次に,ATM の現金収納ボックスには,現金が直接出入りする収納ボックスと,収納ボックスの現金が一定量を割った場合や,一定量を超えた場合に現金を補充または回収する補充・回収ボックスがある.ATM の運用において,金融機関または金融機関に委託された企業は,補充・回収ボックスの現金が大きく過不足になった場合のみ,これに現金を補充・回収すればよいことになる.また,補充・回収ボックスを交換する際に,収納ボックスの現金を用いてATM のサービスを継続できるため,利用者に不便をかけなくて済む.
