# 8・2・6 運用形態

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATM システム

従来、自動化機器の運用時間は金融機関の営業時間にほぼ一致していたが、利用者の生活時間の拡大や、より一層のサービス向上などを求める要求により、自動化機器の運用時間も年々拡大の一途をたどり、時間外運用や休日運用などの対応が必要になってきた。一方、金融機関は自動化機器の運用のために休日や時間外応対の人員をかかえることができない背景がある。このため、無人化運用を可能にするためのサポートシステム開発が進められている。この基本的な考え方は自動化機器の集中管理思想であり、現在実用化されているシステムでは、集中管理センタからの制御により、自動化機器の電源オン/オフから状態管理(明細票、紙幣、硬貨の残量状態の把握)、障害時における診断情報の収集などを行うことができる。
