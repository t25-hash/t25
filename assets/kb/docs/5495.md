# 8・2・9 機能の多様化

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATM システム

自動化機器は従来、人手によって行われてきた作業を利用者自身の操作により行うもので、金融機関にとっては事務効率化のメリットが大きいため、今後ともいっそう対象となる取引が増えると考えられる。このためには、より簡単なマンマシンインタフェースをもつことが必要となろう。また、各種公共料金の払込み処理機能をも図8・9ソフトウェア構成例つ公金ATM が実用化されつつある。電力、電話、ガス、水道等の公共料金のほか、税金の払込みが可能になれば、金融機関にとっても利用者にとってもその利便性は大きい。ただし、この実現とサービス範囲の拡大のためには、OCR(optical character reader)入力を可能とする公共料金振込用帳票フォーマットの統一等の、払込みを確実に確認する処理(消し込み処理)の実現が前提となる。
