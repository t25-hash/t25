# 8・2・10 ATM と社会

> **章**: 第8章 社会情報システム
> **節**: 8・2 金融ATM システム

自動化機器の時間外・休日運用への移行は、社会生活の質的変化の結果であり、金融機関はこれらの変化に対応し、社会的ニーズを満たすことによりサービスの向上に努めてきた。同時に、これら自動化機器の時間外・休日運用への移行は、生活時間拡大の要素となっている。このように、自動化機器の利用時間は今後ますます拡大していくと思われるが、それに伴い自動化機器に要求される信頼性、多機能性はますます高度化していくであろう。金融機関の営業店は、次のような店舗機器システムから構成されている。
