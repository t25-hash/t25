# 8・3・2 窓口端末装置

> **章**: 第8章 社会情報システム
> **節**: 8・3 金融機関店舗機器システム

主として営業店のロビーに面したハイカウンタ(一線)およびその後ろに位置する二線に設置され、預金、貸付、為替等の銀行取引業務を行う。具体的には、顧客が記入した伝票に基づき入力された取引内容をホストコンピュータに送信し、その処理結果を通帳や伝票上に印字する機能をもつ端末装置である。キーボード、ディスプレイ、通信機構、通帳・証書・伝票の印字ができるプリンタが接続され、さらに、現金(紙幣・硬貨)を扱う現金処理装置が接続される。通帳証書発行機と連動することにより、新規の通帳発行、繰越時の新通帳発行、証書の印刷発行が窓口端末装置のオペレーションにより自動的に行える。(1) 構成営業店のさまざまな設置条件に対応できるモジュールタイプや、窓口装置の制御部やプリンタ部をデスクの中に組み込んだシステムタイプがある。各社から提供されている窓口端末装置の外観の例を以下に示す(図8・11)。1キーボード取引種別の選択と数値データの入力を行うファンクションキーボードと、英字やかなを入力できるはん用キーボードがある。一般的に、一線ではファンクションキーボードが、二線や後方でははん用キーボードが使われることが多い。2ディスプレイ取引種別の選択により入力用画面を表示し、操作手順をガイダンスしたり、入力データのモニタとしての機能を実現する。技術の進歩に伴い、モノクロからカラーへ、英数字に加えて日本語表示へと機能が向上しており、最近では、営業店のスペース効率や電力消費、発熱に対して優れた特性をもつ液晶ディスプレイが主流になりつつある。図8・10 機能窓口端末装置の機能の概略を、主として一線で行われる普通預金の入金処理を例にとり、操作の流れに沿って説明する。1顧客の依頼する取引をキーボードから選択すると、その取引のための入力画面がディスプレイに表示される。2顧客の通帳を通帳・伝票プリンタに挿入すると、その通帳に張り付けられた磁気ストライプに記録された口座番号が自動的に読み取られ、顧客の口座情報が画面に表示される。また、磁気ストライプに記録されている、通帳の最終印字ページ、印字行の情報により、通帳のページをめくり、次に印字が開始される行位置に自動的にセットされる。3画面から、必要なデータ(預金する金額や金種など)を入力する。4現金処理装置(この場合は紙幣入金機、硬貨入金機)に顧客から受取った現金を投入する。現金処理装置はそれを計数し、計数結果を窓口端末装置に通知する。窓口端末装置では、テラーがキーボードから入力した金種、金額と現金処理装置で計数された金種、金額の照合チェックを行う。5完了キーを押下すると、窓口端末装置とホストコンピュータとの間で取り決められた電文フォーマットに従って、取引データが送信される。6ホストコンピュータでは取引処理を行い、その結果に基づいて元帳を更新した後、顧客の通帳および伝票への印字データを電文に編集して窓口端末装置に送信する。7窓口端末装置ではその電文を受信すると、通帳・伝票プリンタにセットされている通帳に取引結果を印字するとともに、最終印字行等の磁気ストライプ情報を更新し、通帳を排出する。8窓口端末装置のハードディスク上に設定されたジャーナルファイル(電子ジャーナル)に取引内容を記録する。近年の窓口端末装置は、預金、為替、貸付等のいわゆる勘定系業務のほか、ホストコンピュータの情報を参照する機能、ホストコンピュータの機能を利用する機能(エミュレータ機能)、日本語文章の作成・編集機能、スプレッドシート作成・編集機能など、多様な機能を備え、顧客に対するサービス性向上、業中、業後の営業店事務の効率化をねらったものが多い。(3) 通帳・伝票プリンタ通帳・伝票プリンタのおもな機能を以下に説明する。1印字機能通帳、証書、伝票等、種々の厚みのある媒体に印字することや、複写伝票への印字が要求されることから、ドットインパクト式のプリンタが用いられる。技術的にはプラテン機構、印字ヘッド支持機構から構成され、通帳のような厚みのある媒体に印字するためのさまざまな工夫がなされている。2媒体搬送機能媒体を、吸入・排出口から吸入し、印字位置まで搬送する。また、印字の際には改行動作を行う。さまざまな大きさや厚さの媒体をスムーズに搬送するための工夫がなされているが、万が一、媒体の斜行やジャムが発生した場合は、媒体搬送路の各所に配置された媒体センサがこれを検知し、テラーに通知する機能を備えている。3磁気ストライプ読取り・書込み機能通帳には、顧客の口座番号、通帳の最終印字ページ・行等を記録した磁気ストライプが張り付けられており、通帳が通帳・伝票プリンタにセットされた時点でこれを読み取り、取引終了後にこれを更新する。4自動ページめくり機能取引結果データや未記帳データを通帳に印字している途中で最終印字行に達する場合がある。自動ページめくり機能とは、このとき、通帳を排出し、人手でページをめくるのではなく、装置内部でページを自動的にめくる機能であり、本機能を備えた窓口端末装置が徐々に増えつつある。さらに、通帳を閉じた状態で挿入し、表紙も自動的にめくる機能を備えた装置もある。これらの機能は、テラーが機械の操作に煩わされずに、顧客サービスに専念できることを目的に、発達してきたものである。(4) 現金処理装置各社から発表されている現金処理装置の外観の例を以下に示す(図8・12)。現金処理装置は窓口端末装置に接続され、一線、二線ではテラー業務を、後方では出納業務を支援するために使用される装置である。現金処理装置は、特に、一線に設置された窓口端末装置に接続することで、伝票や現金を二線や後方に回付することなく一線で取引を完結できること(一線完結処理)、および、現金を金種ごとに自動的に管理することで、業後の締上げ処理が自動化され、金融機関の事務合理化に効果がある。現金処理装置の導入効果としては、次の点が挙げられる。・現金の鑑別・計数処理の自動化による省力化・テラーの負担軽減による事務処理の効率化・顧客サービスの向上・営業店内出納業務の省力化・営業店における資金運用の効率化1種類現金処理装置をその使用形態から分類すると、一線、二線に設置されてテラーが使用するものと、後方に設置されて主として出納係が使用するものとに分けることができる。これらには、窓口端末装置に接続され、オンラインで使用されるタイプと、他の機器とは接続されずにオフラインで使用されるタイプがある。ロビーには顧客が直接使用するタイプのものもあるが、これらは単なる両替機能のみの単機能型か、入金額に応じた領収レシートのみを出力するオフライン(窓口端末装置に接続されない)のものである。現金処理装置は大別して入金機と出金機に分けることができるが、それぞれにおいて、紙幣を扱うものと硬貨を扱うものがある。営業店のテラー周りのスペース効率上、二人のテラーが両側から使用できる双頭型(二台の窓口端末装置に対して一台の現金処理装置が接続された状態)の装置や、入出金の両方の機能を実現した装置、入金した現金を出金にも利用できる還流型の装置も多くなってきている。2機能現金処理装置のおもな機能は、次のとおりである。図8・12 通帳証書発行機通帳証書発行機は、営業店の一線、二線において、窓口端末装置と連動し、通帳や証書の自動発行処理を行う装置である。窓口で発生する通帳や証書の新規発行、通帳の更新に伴う手続き作業を効率化するための装置であり、おもな機能は以下のとおりである。1通帳の発行窓口端末装置との連動により新約通帳および繰越通帳の表紙に、顧客氏名、店番号、口座番号を印字し、通帳内部に取引内容の印字を行う。さらに、口座番号、最終印字ページ、最終印字行を磁気ストライプに記録し、通帳を放出する。2証書の発行新規証書に取引内容を印字し、放出する。3通帳未記帳への記帳処理無通帳取引の後処理として、通帳に取引内容の印字を行う。通帳を1冊ずつ処理する方法と、カセットに複数通帳をセットして大量一括処理する方法がある。4媒体管理通帳証書発行機内部に収容している通帳や証書の在庫管理を行う機能である。通帳証書発行機自身および窓口端末装置から、媒体の管理データを照会することもできる。通帳証書発行機の収容庫は、金庫性が高く、オペレータカードによるセキュリティ機能と合わせることにより、通帳や証書の不正発行の防止を行うことができる。
