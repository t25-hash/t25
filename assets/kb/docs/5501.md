# 8・1・6 今後の蓄熱技術の展開

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・1 熱エネルギー需給と熱エネルギー貯蔵技術の現状

わが国におけるエネルギー供給源の多様化の観点からは，天然ガ
ス（メタンハイドレードを含む)，再生可能な自然エネルギー，リ
サイクル型エネルギー，排熱を含む未利用エネルギーなど，いわゆ
.糸状コーティング蓄熱材の土壌混入による太陽熱の蓄熱
る新エネルギー依存率を大幅に増大する必要がある．特に，熱エネ
ルギーを循環利用する熱カスケードシステム（thermal cascade
system）の拡大が今後の大きな課題である．すなわち熱エネルギ
ーの発生から最終利用までの高効率化のための，熱エネルギー変
換，輸送，蓄熱そして利用技術の複合化やそのシステム化技術を展
開することにより，新たな熱エネルギーシステムの構築が可能とな
このように，氷蓄熱システムをはじめとして固-液の相変化潜熱
る．その中核をなす蓄熱技術の果たす役割は大きいものといえる．
排熱等の有効利用には，蓄熱システムが必要不可欠である．環境負
荷の小さい，搬送性に富む高密度蓄熱材の開発は，熱エネルギー変
換，輸送，蓄熱そして利用技術を統合化し，効率的な熱エネルギー
利用システムを展開するためには重要である．
