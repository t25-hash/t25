# 8・3・6 ポータブル端末

> **章**: 第8章 社会情報システム
> **節**: 8・3 金融機関店舗機器システム

金融機関営業店の営業、得意先係の事務効率化、顧客サービス向上を支援する端末機である。おもに、客先を巡回し現場で取引を行うためのデータ入出力を行う。小形プリンタを内蔵しており、顧客に取引結果を印字したレシートを残すことができる。客先の巡回が終了後、営業店に戻ると、ポータブル端末を親端末機(一般には窓口端末装置または、パーソナルコンピュータ)に接続し、取引データを親端末装置に送信し、取引処理を実行する。客先での取引の比重が大きい地域金融機関において事務効率化に果たす役割は大きい。
