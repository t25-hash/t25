# 8・3・7 高速銀行券取扱機器

> **章**: 第8章 社会情報システム
> **節**: 8・3 金融機関店舗機器システム

市中金融機関が顧客向けに提供するATM のような機器ではなく、現金が大量に集まる場所、特に各国の中央銀行における銀行券取扱機器である。各国の中央銀行、日本でいえば日本銀行は、その役割の一部として銀行券の発行を行う。発行された銀行券は流通の過程で劣化していくため、その品質維持が必要となる。中央銀行は、市中金融機関から銀行券が戻された時点で、銀行券の券種、真偽をチェックするとともに、券の劣化、汚れ具合を調べ、再使用可能な券と不可能な券に分類する。再使用不可能な券は、不正な再使用を防ぐため、原形をとどめないように処分され、不足分は新券を補充する。上に述べた業務を機械化した高速銀行券取扱機器の概要を以下に示す。(1) チェックの内容1券種処理中の該当券種以外の券あるいは異物が混入している場合はそれを排除する。裏券、他銀行券、旧券の場合も同じ。2真偽真券以外の券を排除する。また、有効面積が十分でない券、異なる番号の券を張り合わせた券も排除する。(2) 正確さに対する要求1取扱うものが金銭であること、信用が第一であることから、システムに対して極めて高い精度・信頼性が要求される。2オペレータ等の不正が入り込めないセキュリティを考慮したシステム設計が要求される。3異常があった紙幣の小帯、大帯が、正確に特定できることが要求される。(3) 高速銀行券処理機の基本構成1ホッパこれから処理する銀行券を100~1000枚セットする部分である。2取込装置ホッパ内の銀行券を高速に1枚ずつ取出す装置である。正確に1枚取出しを実現するために、摩擦を利用する機械的な方法よりも精度の高い真空(バキューム)を利用する方法が一般的である。正確なセンシングを行うために、銀行券が傾いて取込まれることも防ぐ必要がある。3移送装置取出された銀行券を一定速度でセンシング装置、区分け装置を経由して集積装置まで移送する。センシングの結果により、銀行券を再発行可能券、再発行不可能券、排除券に分類し、それぞれの集積装置に移送する。移送速度は2~10m/sに達し、高速度移送による銀行券の損傷を防ぐため、スムーズな移送路となっている。また、移送路から銀行券が飛び出さないように、真空ドラム、ベルト、ピンチローラが使用される。4センシング装置券種、真偽、再発行の可否、2枚重ねを図8・14 印鑑照会・照合システム構成検出し、その結果により、それぞれの集積装置に区分けするための信号を、区分け装置に対して送信する。センシングの方法は種々あるが、おもに光学的、機械的手段を組合わせて感知する。移送中の銀行券に対して、複数の角度からセンシングして、信頼性の高い結果が得られるように工夫されている。5区分け装置センシング結果の信号により、移送路を切換える装置である。6集積装置再発行可能券・不可能券、および、排除券用に3種類の集積装置がある。再発行可能券は100枚ごとに結束装置に移送される。集積装置においては、高速度で移送されてくる銀行券をいかにスムーズに減速するかが、集積状態の良否を決定する。さまざまな状態の銀行券を確実に集積する方法として、バキュームドラム方式等が用いられている。7結束装置再発行可能券100枚を小帯で結束する。銀行券が容易に抜け落ちない結束力が必要となる。8裁断装置再発行不可能券を再生不可能な状態に裁断する装置である。裁断片を空気輸送することにより、機械室外に大量の裁断片を持ち出すことができる。
