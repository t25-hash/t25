# 8・4・2 金属水素化物（水素吸蔵合金）

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・4 化学蓄熱

a.蓄熱材としての水素吸蔵合金 図8･28に水素吸蔵合金
（hydrogen absorbing alloy）の温度・圧力特性を示す．水素吸蔵
合金は，多量の水素を可逆的に吸蔵放出する性質とともに，水素を
吸蔵すると金属水素化物（metal hydride）となって発熱し，水素
を放出すると吸熱する特性をもっていることから，蓄熱材料として
以下の特徴をもつ．(１)水素化反応が他の固体-気相反応に比べ可
逆性に優れていること，(２)反応速度が速いこと，(３)水素吸蔵合
金の種類または合金成分の比率を変えることにより－60℃前後から
1000℃程度の温度位までの熱源温度に対応できること，(４)水よ
りも大きな蓄熱密度が期待でき，システムのコンパクト化が可能と
なる．課題としては，水素吸蔵合金のコストが比較的高いことがあ
る．蓄熱材として必要となる水素吸蔵合金の比熱，熱伝導率などの
物性値は，合金の種類，操作条件（温度，水素ガス圧力，水素吸蔵
量など），合金の粒度により変化するため，各合金に対して実験的
に求められている．

c .蓄熱容器構造 代表的な水素吸蔵合金を充てんした蓄熱容
器の構造例を図8･30に示す．水素吸蔵合金と熱交換する熱媒体は
金属管内を流れる隔壁式熱交換器構造となっている．
水素吸蔵合金は，水素ガスとの接触面積を大きくするため粉体の
状態で容器内に充てんされる．水素吸蔵合金は，水素を吸蔵すると
体積膨張する．LaNi合金では，合金自身の体積が約20%程度膨
張する．また，合金は一般的に割れやすい性質をもち，水素の吸蔵
放出操作を繰返すと膨張時に合金粉末が割れて，数十 m程度の微
粉末となる（微粉化)．
このため，蓄熱容器の合金充てん部構造に対して，他の熱交換器
と異なり合金充てん容器設計時に，以下の対策が行われている．
(１)合金の膨張を緩和する緩衝材の配置と合金充てん方法の最適
化，(２)合金微粉末の水素の流れや外部振動により移動偏在しない
合金充てん部の構造設計，(３)微粉化した合金が外部へ流出しない
ような構造およびフィルタの採用，(４)合金層内の水素の通気性を
確保するための通気材の選定と配置などがある．
