# 8・4・3 吸着

> **章**: 第8章 蓄熱・蓄冷機器
> **節**: 8・4 化学蓄熱

a.吸着現象 固体による気体の吸着は界面現象の一つで，固
体表面からの引力により分子が引きつけられ表面近傍に拘束される
現象である．吸着する固体を吸着材，吸着される物質を吸着質と呼
ぶ．この吸着には固体表面の引力が化学結合によるものと分子間力
（ファン・デル・ワールス力）によるものとがあり，前者を化学吸
着，後者を物理吸着と呼ぶ．いずれの場合も発熱（吸着熱）を伴う
が，化学吸着では化学反応を伴うため吸着熱が大きく束縛が強固で
不可逆性が強く，物理吸着では前者に比べて吸着熱は小さく（吸着
される気体の凝縮熱よりいくぶん大きい程度）可逆的である．この
逆現象，すなわち吸着された分子がエネルギーを吸収して離脱する
現象を脱着という．この気体の吸脱着現象は，気体の分離・精製・
回収・調湿など多くの工業プロセスに利用されてきた．また，吸着
熱を利用・操作する熱機器・システムにおいては可逆性が必須であ
るため，主として物理吸着が利用されている．
b.吸着材 前述のように，吸着は界面現象であるため吸着材
としては表面積の大きい多孔性の材料が適しており，代表的なもの
にはゼオライト，シリカゲル，活性炭，活性アルミナなどがあり，
それぞれ種々の吸着材が開発・商品化されている．
ゼオライト（zeolite）は，アルミナケイ酸塩の結晶材料であり
結晶水が放出された後に分子サイズの細かい三次元網目構造の均一
な細孔を有している．細孔の径が均一であることがゼオライトの最
大の特徴であり，細孔より小さな分子しか吸着しない性質により気
体を分離する分子ふるいの機能を有している．この細孔に，アンモ
ニア，水素，二酸化炭素などをはじめ各種の分子を吸着する．もと
もと，ゼオライトは天然の鉱物であるが，1945年に初めて合成ゼ
オライトが開発されて以来，いまでは100種類を超える天然および
合成ゼオライトが知られている．ゼオライトの細孔径は数Å，比表
面積は400～750m / g程度である．
シリカゲル（silicagel ）は，二酸化ケイ素のコロイド粒子が三次
元的に乱雑に結合してできた，無定型，多孔質のゲルで粒子間の空
げきが各種の分子を吸着する細孔となっている．粒子の凝集状態の
違いによってA型とB型に大別され，A型はコロイド粒子が密に
凝集しているため，表面積が大きく細孔容積は小さい．低湿度での
吸着力が大きいのが特徴である．B型では粒子の凝集は粗で，か
つ，粒子径が大きいので表面積が小さく細孔容積が大きい．高湿度
で多量の水分を吸着するという特性を有している．シリカゲルの平
均細孔径は10～140Å，比表面積は300～830m / g程度である．
活性炭（activated carbon）は，石炭やヤシ殻などの炭素物質を
原料として高温でガスや薬品と反応させて作られ，微細孔をもつ．
この微細孔は，炭素内部に網目状に構成されており，その微細孔の
壁が種々の物質を吸着する．活性炭の平均細孔径は20～50Å，比
表面積は500～3500m / g程度である．
活性アルミナ（activated alumina）はアルミナ水和物を加熱脱
水して得られる中間アルミナであり，アルミナ微結晶の結晶水の脱
水によって生じる細孔が吸着の活性中心となっている．活性アルミ
ナの平均細孔径は40～120Å，比表面積は95～350m /g程度であ
る．

吸着材の吸着特性は，相対圧（吸着質の圧力と飽和蒸気圧の比）
に対する平衡吸着量をプロットした吸着等温線により表される．図
8･32に代表的な吸着材の吸着等温線を示す．吸脱着の操作は相対
圧の操作，具体的には圧力もしくは温度を変化させることによって
なされる．
c .吸着による蓄熱の応用 物理吸着は完全に可逆的な現象で
あり，熱サイクルを構成するのに適している．吸着による蓄熱を利
用した熱サイクルの例として，吸着式ヒートポンプ（adsorption
heat pump）を紹介する．吸着質（冷媒）としては水，アンモニ
ア，メタノールなどが用いられる．
図8･33は吸着式ヒートポンプの原理を模式的に示したものであ
る．系は吸着材を充てんした吸脱着器，凝縮器および冷熱を生成す
る蒸発器とからなる．図8･33(ａ)は脱着（蓄熱）過程を示す．吸
着材を外部熱源で加熱することにより冷媒を脱着する．脱着された
冷媒蒸気は凝縮器で冷却され凝縮・液化する．脱着が完了すると，
系を吸着（放熱）操作に切換える［図8･33(ｂ)］．吸着熱が発生す
るので冷却水（常温）で除熱しながら吸着させる．このとき，吸着
される冷媒蒸気は蒸発器での蒸発により供給されるので，蒸発器で
蒸発潜熱が奪われて冷熱が生成され，この冷熱を冷房などに利用す
る．吸着が完了すると再び脱着操作に移り，これを繰返す．
この操作では，冷熱の生成が間欠的になるので吸脱着器を2基設
け，上記の操作を交互に繰返すことで連続的な冷熱の生成が可能と
なる．
この吸着式ヒートポンプは，すでにその一部が実用化されている
が，100℃以下の比較的低温の熱源が利用できるという特徴を有し
ており低温排熱の有効利用技術として着目され，その高性能化・実
用化に向けた研究開発が行われている．
