# 2・1・3 リハビリテーション工学

> **章**: 第2章 福祉工学
> **節**: 2・4 から、年々増加傾向にあることがわかる。また、2005年度調査

リハビリテーション工学(rehabilitation engineering)とは、レズウィック(Reswick)によれば、障害をもった人のQOLの向上を目指した科学技術の応用である。すなわち、何らかの障害をもった人に対して、科学技術・工学技術を駆使してその人の生活を支えるための工学といえる。リハビリテーション工学発展の最初の契機は、アメリカにおける退役軍人のための義肢開発や、日本でのサリドマイド児に対する電動義手の開発に見られるように、人の手脚を人工的に作ることをテーマとしたものであった。このテーマは、機械工学、電気工学、制御工学等を駆使したものであり、リハビリテーションの分野に工学が入り込むにはよいきっかけとなった。これらのテーマは、後のロボット工学の発展にも大きな影響を及ぼした。その後、リハビリテーション工学は、上肢下肢の欠損のみではなく、その他のすべての障害に対する工学的支援へとその範囲を広げていった。そのような経緯の中で、単なる工学的技術開発ではなく、障害者に本当に役立つ技術の開発に主眼がおかれるようになっていった。これにより、リハビリテーション工学は障害をもった人の生活を重視して考えることとなり、上述のレズウィックの定義に合致したものとなった。リハビリテーション工学の重要なキーワードは福祉機器(assis-tive technology)である。役立つ福祉機器の開発とともに、福祉機器の適合、すなわちそれぞれの使用者に適した福祉機器を選択すること(assistive technology service)も重要なパートを占めている。その背景には、福祉機器のもつ難しさがある。一つは、障害者の特徴が一人ひとり異なっているという個別性の問題である。ある人に有効な福祉機器が、別の人にそのまま導入して使えるかというと、そうではない場合が多い。これに対して、福祉機器開発の立場では、個別対応でのものづくり、モジュール化やアジャスタブル化による調整機構をもった福祉機器の開発、ヒューマンインタフェースの工夫など種々の対応策を講じている。そのような機器を障害のある人に導入しようとした場合、必要不可欠なのが適合である。その人それぞれに合わせて、機器の組合せ、調整、改造、インタフェースの製作などを行い、その人が本当に使えるものに仕上げていく技術がこれに当たる。ここでは、機器の知識のほかに、医学的知識や社会制度の知識も要求される。それらを体系化して、よりよい適合技術を構築していくことも、リハビリテーション工学の役割となる。さらに最近のリハビリテーション工学では、福祉機器の利用効果を明確に示していくための手法の開発も行われている。福祉機器を使用することによる効果を、使用者の身体機能、心理面さらに社会学的な面から評価する試みが行われている。その重要な評価指標がQOLとなる。これらが整理されることにより、真に役に立つリハビリテーション工学が構築されると考えられる。一方、福祉機器という特殊解を模索するのではなく、一般解を求めようとする考え方も広がりつつある。ユニバーサルデザイン(UD : universal design)がそれであり、一般製品を障害をもった人にも使いやすく設計するというものである。ノーマライゼーションの理念に合致するとともに、大量生産によるコストダウンが図れ9 ―9ることもユニバーサルデザインの有効な点である。リハビリテーション工学では、福祉機器に関すること以外にも重要な研究分野がある。麻痺した手脚の機能を再建する技術として、ニューロリハビリテーションやニューロエンジニアリングが注目されている。さらに、人の身体の動きや力を解析する技術もリハビリテーション工学に大きく関係しており、運動計測技術の開発やモデル解析による筋力の推定手法の開発、計算機シミュレーションを駆使した障害動作の解析および訓練手法の開発等も行われている。以上のようにリハビリテーション工学には機械工学や電気工学、建築工学等の従来の工学に加えて、医学による人の身体の理解、心理学による人の心の理解、社会福祉学による社会全体の理解など複数の学問知識を必要とする。そして、最終的には障害をもった人、個人個人のQOLをどのように向上させるかを考えることになる。そのためには、これまでの工学にはない新たなパラダイムの構築も望まれるところである。
