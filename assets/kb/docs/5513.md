# 2・2・1 高齢者とその現状

> **章**: 第2章 福祉工学
> **節**: 2・2 高齢者とジェロンテクノロジー

高齢者とは、人口統計上では65歳以上の人々のことを指す。そして、人口全体に占める高齢者の比率(高齢化率)が7%以上で高齢化社会(ageing society)、14%以上を高齢社会(aged society)という定義がされている。わが国は1970年に高齢化率が7%を超え、社会の高齢化という問題が大きく取り上げられるようになった。そのわずか20年後の1991年には14.1%となり、高齢社会に突入した。今後は2014年には25%台に達し、その後も増え続けて2050年には35.7%にまでなると予測されている(図2・2)。高齢化社会の定義は、1956年に発表された国連の人口問題に関するレポートによるものである。発表された当時は、高齢化率が14%を超えてもまだまだ増大し続けるような社会のことは考慮されていなかった。そこで最近では、わが国のように14%を超えてなお高齢化が進む社会を超高齢化社会などと呼んでいることもある。わが国の特徴として、高齢化がこのように急速であることが挙げられる。例えば、高齢化率が7%から14%になるまでの期間は、フランスで130年、福祉先進国といわれるスウェーデンでは85年、ノルウェーで90年であり、これらと比較すればわが国の高齢化はたいへん急激で、人類がいまだかつて経験したことのない社会的変化であることがわかる。そのためわれわれは、社会的・経済的に迅速な対応を迫られている。社会・経済全体から議論されるとき、高齢者人口は生産年齢人口(18~64歳)との対比で議論される。つまり、高齢者すなわち65歳以上とは定年を迎えた人々であり、その生活を生産年齢人口に当たる人々が支えるという図式である。1990年ごろには高齢者1人当たり生産年齢人口6人であったのが、2020年ごろには高齢者1人につき生産年齢人口2人という、従来の社会・年金の仕組みのままではたいへん厳しい状況が待ち受けていることが以前から指摘されていた。図2・2 ―10医療・福祉・バイオ機器こうした状況を受け、かつては高齢者といえば庇護されるべき弱者としてとらえられていた社会的認識は、高齢者自身の意識とともに大きく変化してきた。つまり、なるべく元気で自立して生活していこう、働ける人は働こうという考え方にシフトしてきたのである。一口に65歳以上といっても、元気に生活している人もいれば、病院や施設に入って要介護状態となっている人もいる。そこで人口統計上も高齢者を、65~74歳の前期高齢者(young old)と75歳以上の後期高齢者(old old)とより詳しく分類するようになっている。この後期高齢者は前期高齢者よりも増加していき、2020年ごろには高齢者全体の半数を占めるようになると予測されており、したがって要介護の状態にある人々も増えると考えられている。高齢者世帯の状況(表2・7)を見ると、2000年と2025年では、世帯主65歳以上の世帯が1113万6000世帯から1842万6000世帯へと増加するが、そのうち単独(一人暮らし)と夫婦のみの世帯数はほぼ倍増する。さらに、世帯主が75歳以上の世帯については2.6倍に増え、そのうち単独世帯数は3倍近くまで増加する。このように高齢者のみの世帯は今後とも急速に増加する傾向にあり、かつてのように家族に依存したケアではもはや立ち行かない状況にあるといえる。人々の考え方を見ると、かつては介護するほうもされるほうも、人手でなければ嫌だという意見が絶対的だったが、介護にまつわる肉体的負担・精神的ストレスの問題が表面化し広く指摘されるようになって、こうした考えは変化してきている。団塊の世代を中心として、高齢化のピーク時に高齢者層となる年代層には、がんばって親を介護してきた結果、子供世代にはこんなつらい思いはさせたくないと考える人が多くなっている。高齢者の定義についても、意識調査などでは70歳以上と考える人が多くなってきており、生活費も年金などの公的支援だけでなく就業による収入を考え、自助努力したいという考えの人が増えてきている。なるべく健康で長生きして、子供に負担をかけたくないという考えの表れであろう。こうした考えは、介護の主体を担わざるを得なかった女性に顕著に見られる。そのため自分が介護を受ける立場になった場合にも、女性のほうがヘルパーなどの外部サービスなどを希望する傾向が強い。これからは、介護が楽になって気持ちにゆとりができるのなら、機械を使ったり外部の人手を利用したほうがいいという時代になってきている。高齢者の生活状況については、1人当たりの所得そのものは他の年代と大差はなく、消費支出などはむしろ全年齢層の中でも多いため、経済力は向上してきているといえる。支出の中でも教養・娯楽や交通費の占める割合は比較的大きく、地域の活動や趣味(特に旅行)に活発に取り組んでいることが見て取れる。ただし健康面については、高齢者の半数以上が何らかの自覚症状を訴えており、「平成14年度医療費の動向」では、高齢者の医療費は1人当たり73.1万円で、一般の12.8万円に対し5.7倍となっている。高齢化に伴い急速に増大する医療費は、社会経済的に大きな問題となってきている。疾病を早期に予防する診断技術や、患者への身体負担を軽減して入院期間を短縮する新しい治療技術などは、医療費を削減するための解決策として重要である。このような高齢者の生活を支援するため、1989年に高齢者保健福祉推進10ヵ年戦略、通称ゴールドプランが策定された。ここで表2・7 technology、情報通信技術)の急速な進展についていけない情報弱者という問題が注目されてきている。高齢者の中にも最新の情報機器を使いこなす人はいるが、大多数は身体的な状況に機器が適合しなかったり、機械は苦手という心理的な抵抗感があったりで、なかなか活用できていないのが現状である。1990年にアクセシビリティ指針が制定され、さまざまな不自由をもつ人々にも情報機器のインタフェース部分を使いやすくするための取組みが行われてきた。またパソコン教室などにより、高齢者に情報機器の活用に興味をもってもらう活動もなされている。今後は、ハードウェア面だけでなくソフトウェアも併せて、高齢者にとって使いやすい情報機器の開発と普及が期待されている。このように、現在の高度技術社会を高齢者にもいっそう適合させるため、高齢者の心身特性についての知見を深めるとともに、それを実際の製品に応用できるような研究開発、およびそのための人材が必要となってきている。
