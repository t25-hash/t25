# 2・2・2 老化

> **章**: 第2章 福祉工学
> **節**: 2・2 高齢者とジェロンテクノロジー

a.老化の特徴 高齢者の加齢に伴う不可逆的な機能低下を老
化（ageing）と呼ぶ．多くの身体機能は20代をピークに徐々に低
下するが，50代ごろまではあまり顕著ではなく，それほど問題に
はされない傾向にある．しかし60代のころから機能低下による日
常生活の不便さや外的ストレスへの耐性低下が顕著となり，老化と
して認識されることとなる．
老化の特徴の一つは，身体のあらゆる機能が低下する点にある．
一部の機能が欠損・低下しているが残りの機能には問題のない身体
障害とは，その点で大きく異なる．そのため，身体障害の場合では
残存機能を用いた機能代行が可能となるが，老化の場合には他の機
能を用いて代行できない場合が多い．ただし，機能によって低下の
割合や低下し始める時期は大きく異なる．例えば，一般に精神機能
の低下は身体機能に比較して遅く，特に総合的な判断能力は高齢期
にピークがあるといわれる．筋肉によっても機能低下の割合は異な
る．また，個人差が大きい点も老化の特徴である．
b.老化研究 人間の加齢に伴う経時変化は，各機能ごとに統
計的に調べられているものも多い．調査方法によって，横断的研究
と縦断的研究に分かれる．横断的研究とは，ある時点におけるさま
ざまな年齢の集団を対象とした研究であり，調査の容易さから多く
の研究がある．しかし問題点も多く，データの信頼性に注意して取
扱う必要がある．第一に個体の経時変化は多様であり，これを平均
すると必ずしも各個体の経時変化の特徴を残すとは限らなくなるこ
とである．例えば，各個体で異なる年齢に鋭いピークをもつ特性を
示す機能であっても，これを平均するとなだらかなカーブになって
しまう．このため，これらの平均値のデータから各個体の老化特性
を推定することは困難である．また，環境因子の影響と老化とを混
同する危険性があることも問題点の一つとして挙げられる．
一方，縦断的研究とは，同一個体で長期にわたって繰返し測定す
ることにより，各個体の加齢変化を測定する方法である．一般に縦
断的研究は横断的研究よりも信頼性が高いが，時間とコストがかか
ることから研究はまだ少ない．また縦断的研究は調査機関のある特
定地域でなされることがほとんどであり，地域性の影響が出る可能
性も考慮に入れる必要がある．
c .老化と機能低下 老化の原因はまだ明確にはなっていない．
複数の仮説が立てられているが，その中のいくつかは実際に老化に
関係していることが証明されている．例えば，正常細胞の細胞分裂
回数には限界があることから，遺伝学的考え方による細胞レベルの
老化仮説が生まれた．酸化によって細胞組織が傷害を受けることに
より老化が起こることは，ある程度立証されている．また，一つま
たは複数の特定の組織や臓器が老化に関わっているとする説も，説
得力のある根拠がある．しかし，これらはいずれも老化の一側面だ
けを表したものであり，現実の老化現象は多くの要因が複雑に絡み
合って多様に発現すると考えられている．
運動機能について見ると，老化によって筋線維は減少し，筋力も
筋持久力も低下する．特に膝伸展力や背筋力などは老化により大き
く低下し，このことが起上り・立上りの困難を招く一因となる．一
方，握力などの低下はあまり顕著ではない（図2･3）．一般に静的
筋力に比較して動的筋力の低下が大きいことが知られている．その
ため，高齢者はゆっくりとした動きは可能であるが，俊敏な動きは
しにくくなる．また筋の萎縮や関節の拘縮などにより柔軟性が低下
すること（図2･4）も動きにくくなる原因である．

れないことが多い．
内臓機能に関しては，血圧上昇，最高心拍数の減少，心拍出量の
減少，肺活量や最大換気量の低下，胃の消化機能低下，腎臓機能の
低下，血中ホルモン値の変化など各臓器で機能低下が起きる．
脳・神経機能では，末梢神経の神経伝達速度が軽度に低下するほ
か，脳でさまざまな機能低下が見られる．脳血流量は低下し，脳波
振幅も低下する．また，脳波の基本アルファ波の出現が多くなる．
これらはいずれも脳の活動が低下していることを示している．
この結果，精神機能や知的能力の低下が引き起こされる．意欲の

骨折が治っても，すぐに転倒して再び骨折することが多い．骨折に
至らなくても体調が悪くなって数日寝ているだけで体力が低下し，
同様の悪循環に陥ってしまうことも多い．このような悪循環が起こ
ると，筋力低下などから自分では立ち上がれなくなって寝たきりに
なり，外界からの刺激もなくなるため精神機能も低下する．このよ
うに元気な高齢者であっても老化によって体力の余裕がなくなって
いるため，一度体調を崩すと一気に悪化してしまい，寝たきりや認
知症になってしまう危険性がある．
d.老化予防 現在の技術では老化を止めることはできない
が，老化を遅らせる方法はいくつか明らかになっている．喫煙や有
害物質の摂取を避けること，カロリー・各種ミネラル・タンパク質
等が欠乏したり，逆に過剰摂取になったりしないように適切な食生
活を心がけること，適切な体重を維持すること，適度な運動をする
ことなどである．またストレスを減らすなど心理的な方法も重要で
ある．
e .まとめ 以上述べてきた通常の老化現象以外に，高齢者に
多発する病気による機能低下がある．高齢者に特徴的な病気には，
認知症，脳血管障害，心筋梗塞，糖尿病，リウマチ様関節炎，骨粗
鬆症，白内障などがある．これらは厳密な意味では老化ではない
が，高齢者に高い頻度で発生する疾病であり，高齢者の機能低下を
考える際には無視できない．このような病気にかかると，一般的な
老化よりも急速に機能低下を引き起こしてしまうこととなる．
老化による機能低下データを調べる場合には文献(41)～(50)など
を参考にされるとよい．
