# 2・2・2 老化

> **章**: 第2章 福祉工学
> **節**: 2・2 高齢者とジェロンテクノロジー

a. 老化の特徴高齢者の加齢に伴う不可逆的な機能低下を老化(ageing)と呼ぶ。多くの身体機能は20代をピークに徐々に低下するが、50代ごろまではあまり顕著ではなく、それほど問題にはされない傾向にある。しかし60代のころから機能低下による日常生活の不便さや外的ストレスへの耐性低下が顕著となり、老化として認識されることとなる。老化の特徴の一つは、身体のあらゆる機能が低下する点にある。一部の機能が欠損・低下しているが残りの機能には問題のない身体障害とは、その点で大きく異なる。そのため、身体障害の場合では残存機能を用いた機能代行が可能となるが、老化の場合には他の機能を用いて代行できない場合が多い。ただし、機能によって低下の割合や低下し始める時期は大きく異なる。例えば、一般に精神機能の低下は身体機能に比較して遅く、特に総合的な判断能力は高齢期にピークがあるといわれる。筋肉によっても機能低下の割合は異なる。また、個人差が大きい点も老化の特徴である。b. 老化研究人間の加齢に伴う経時変化は、各機能ごとに統計的に調べられているものも多い。調査方法によって、横断的研究と縦断的研究に分かれる。横断的研究とは、ある時点におけるさまざまな年齢の集団を対象とした研究であり、調査の容易さから多くの研究がある。しかし問題点も多く、データの信頼性に注意して取扱う必要がある。第一に個体の経時変化は多様であり、これを平均すると必ずしも各個体の経時変化の特徴を残すとは限らなくなることである。例えば、各個体で異なる年齢に鋭いピークをもつ特性を示す機能であっても、これを平均するとなだらかなカーブになってしまう。このため、これらの平均値のデータから各個体の老化特性を推定することは困難である。また、環境因子の影響と老化とを混同する危険性があることも問題点の一つとして挙げられる。一方、縦断的研究とは、同一個体で長期にわたって繰返し測定することにより、各個体の加齢変化を測定する方法である。一般に縦断的研究は横断的研究よりも信頼性が高いが、時間とコストがかかることから研究はまだ少ない。また縦断的研究は調査機関のある特定地域でなされることがほとんどであり、地域性の影響が出る可能性も考慮に入れる必要がある。c. 老化と機能低下老化の原因はまだ明確にはなっていない。複数の仮説が立てられているが、その中のいくつかは実際に老化に関係していることが証明されている。例えば、正常細胞の細胞分裂回数には限界があることから、遺伝学的考え方による細胞レベルの老化仮説が生まれた。酸化によって細胞組織が傷害を受けることにより老化が起こることは、ある程度立証されている。また、一つまたは複数の特定の組織や臓器が老化に関わっているとする説も、説得力のある根拠がある。しかし、これらはいずれも老化の一側面だけを表したものであり、現実の老化現象は多くの要因が複雑に絡み合って多様に発現すると考えられている。運動機能について見ると、老化によって筋線維は減少し、筋力も筋持久力も低下する。特に膝伸展力や背筋力などは老化により大きく低下し、このことが起上り・立上りの困難を招く一因となる。一方、握力などの低下はあまり顕著ではない(図2・3)。一般に静的筋力に比較して動的筋力の低下が大きいことが知られている。そのため、高齢者はゆっくりとした動きは可能であるが、俊敏な動きはしにくくなる。また筋の萎縮や関節の拘縮などにより柔軟性が低下すること(図2・4)も動きにくくなる原因である。9 syndrome)と呼ぶ。このため高齢者では骨折が治っても、すぐに転倒して再び骨折することが多い。骨折に至らなくても体調が悪くなって数日寝ているだけで体力が低下し、同様の悪循環に陥ってしまうことも多い。このような悪循環が起こると、筋力低下などから自分では立ち上がれなくなって寝たきりになり、外界からの刺激もなくなるため精神機能も低下する。このように元気な高齢者であっても老化によって体力の余裕がなくなっているため、一度体調を崩すと一気に悪化してしまい、寝たきりや認知症になってしまう危険性がある。d. 老化予防現在の技術では老化を止めることはできないが、老化を遅らせる方法はいくつか明らかになっている。喫煙や有害物質の摂取を避けること、カロリー・各種ミネラル・タンパク質等が欠乏したり、逆に過剰摂取になったりしないように適切な食生活を心がけること、適切な体重を維持すること、適度な運動をすることなどである。またストレスを減らすなど心理的な方法も重要である。e. まとめ以上述べてきた通常の老化現象以外に、高齢者に多発する病気による機能低下がある。高齢者に特徴的な病気には、認知症、脳血管障害、心筋梗塞、糖尿病、リウマチ様関節炎、骨粗鬆症、白内障などがある。これらは厳密な意味では老化ではないが、高齢者に高い頻度で発生する疾病であり、高齢者の機能低下を考える際には無視できない。このような病気にかかると、一般的な老化よりも急速に機能低下を引き起こしてしまうこととなる。老化による機能低下データを調べる場合には文献(41)~(50)などを参考にされるとよい。
