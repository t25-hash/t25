# 8・1・5 気相エピタキシャル成長装置（epitaxial reactor )

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・1 基板材料製造装置

エピタキシャル成長とは，基板結晶の結晶軸をそのまま伸ばす結
晶成長であり，ウェーハの表面に形成された膜をエピタキシャル
膜，得られたウェーハをエピタキシャルウェーハと呼ぶ．半導体シ
リコンではシラン系のガス（SiH ，SiHCl ，SiHCl ）を原料とし
て使用する．成長時の雰囲気は水素ガスであり，その圧力は0. 1～
0 .01MPa，温度は1100℃付近である．これにより，CZ結晶より
高品質のシリコン結晶膜を表面に形成できる．

シリコンエピタキシャル成長に用いられているおもな装置を次に
示す．
図8･10(ａ)と(ｂ)はシリコン基板を1枚ずつ成長させる方式
（枚葉式）であり，大きなシリコン基板に大きな成長速度でエピタ
キシャル膜を形成する．一方，図8･10(ｃ)と(ｄ)は複数のシリコ
ン基板を同時に成長させる装置であり，図8･10(ａ)と(ｂ)に比較
して成長速度は小さいものの一度に複数枚のエピタキシャルウェー
ハを作れる利点がある．シリコン基板の直径が150mm以下では
図8･10(ｃ)と(ｄ)が主であるが，200mm以上の直径では図8･
10(ａ)と(ｂ)が主である．いずれの装置においても，シリコン基板
およびサセプタ（基板の支持台）が高温で，反応容器（cham-
ber ）壁が低温のコールドウォール型である．
図8･10(ａ)および(ｂ)のように1枚ずつ成長するリアクタにお
いては生産性を高めるために，図8･11のようにロボットの周りに
複数台のリアクタを配置して同時に運転することが多い．このよう
な配置をクラスタ構造と呼ぶ．
