# 2・2・3 高齢者福祉の基本理念

> **章**: 第2章 福祉工学
> **節**: 2・2 高齢者とジェロンテクノロジー

急速な社会の高齢化に伴い,高齢者福祉の整備が急ピッチで進められている.1995年に制定された「高齢社会対策基本法」では,長寿をすべての国民が喜びの中で迎え,高齢者が安心して暮らすことのできる社会の形成が望まれるとして,雇用,年金,医療,福祉,教育,社会参加,生活環境等を高齢社会に適したシステムとして形作るための施策が定められている.高齢者福祉の基本理念は,社会においてすべての人に認められる基本的な人権に根ざしている.イギリスの高齢政策センターがまとめた老人ホームとナーシングホームを対象とした運営基準では,ケアの原理として以下の10点を挙げている.(1)プライバシーと尊厳(dignity)性の尊重,(2)自尊心の維持,(3)自立支援(in-dependent living support),(4)選択とコントロール,(5)個性の認識,(6)安全,(7)自己責任で危険を冒すこと,(8)市民としての権利,(9)身内の人々や友人との人間関係,(10)余暇活動の機会である.これらの原理は,とかく世話が主たる目的と考えられがちな高齢者福祉とは異なり,個性の尊重と自己決定,自立といった考え方が前面に押し出されている.したがって,介護といっても,高齢者の自立を支援するための介護という位置付けになる.日本においても,2000年に策定されたゴールドプラン21では以下の4つを基本方向(基本的な目標)として位置付けている.(1)活力ある高齢者像の構築,(2)高齢者の尊厳の確保と自立,(3)支え合う地域社会の形成,(4)利用者から信頼される介護サービスの確立である.これらによって,介護や支援を必要とする高齢者に対しては在宅,施設の両面においてサービスを提供できる基盤を整備し,施設から在宅へ,また要介護から自立へという方向性を示している.この中で,介護は従来の措置制度から,介護を受ける側と介護サービスを提供する事業者との間での契約により成り立つということが定められている.これは,自己決定や自立を前提とした制度であり,介護を受ける側にとっても,提供する側にとっても大きな転換を必要とする.2000年から実施されている介護保険(long-term care insurance)は,このような考えのもとに作られた象徴的な制度の一つといえる.さらにゴールドプラン21では,元気な高齢者の健康作りや生きがい活動に対する支援を行うことで,高齢者の積極的な社会参加を促進することを目標としている.その基盤として,すべての高齢者やその家族が生きがいをもって暮らすことができる,地域での支援体制作りがうたわれている.また,それらを支える社会的基礎として長寿科学技術,介護に対する理解,高齢者・障害者に配慮されたまちづくり,国際交流を挙げている.科学技術に対する期待も大きく,福祉機器の開発・普及の促進,バリアフリーやユニバーサルデザインの考え方を取り入れたまちづくりは,高齢者の自立を促進し,高齢者福祉を考えるうえで必要不可欠な項目となっている.高齢者福祉は,社会の急速な高齢化に伴い,急激に変化を遂げている.そのため,制度と現場とのギャップを生み,至る所で混乱を招いている.その背景には,社会全体が高齢者福祉に対する考え方を変える必要性に迫られていることが挙げられる.これは,機械工学の立場から考えれば,逆に好機ともとらえられる.手でやるのが介護という旧来の考え方から,介護者にも被介護者にも安全で質の高い介護が求められるようになり,介護機器の利用は広がるものと期待される.さらに自立を促進するための支援機器の活用も模索されている.このような状況の中で,機器を使うことによる効果をしっかり示し,高齢者福祉に機械工学が貢献できることを示すことで,機械工学の新たな発展が期待できる.
