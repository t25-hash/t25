# 8・3・2 エッチング装置（etching apparatus )

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・3 半導体チップ製造装置

a.概要 半導体デバイスの微細化がどこまで可能かは，リソ
グラフィーとともにエッチング技術がその鍵を握る．エッチング
は，マスクと呼ばれる耐食性のパターンをリソグラフィーなどで基
板上に作り込み，基板を腐食加工することにより，マスクを写真で
いうネガとして回路パターンを基板に転写する技術である．マスク
はフォトレジストと呼ばれる樹脂や，あるいはハードマスクと呼ば
れるSiN（シリコン窒化膜）やSiO （シリコン酸化膜）で形成さ
れる．エッチングはドライエッチングとウェットエッチングの2種
類に大別される．ドライエッチングは微細パターン形成のキー技術
であり，プラズマエッチング（plasma etching）装置が用いられ
る．ウェットエッチングは薬剤溶液によるエッチングで，アンダカ
ットと呼ばれるマスク下のえぐれを生じるために微細加工には向か
ないが，生産性が高いので，現在でも酸化膜や金属膜の全面はく離
など工程によっては利用されている．ここではドライエッチングに
ついて述べる．
b.エッチング装置に要求される特性 材料が多様化し，デバ
イス構造も複雑化する中で，エッチングに要求される特性も多岐に
わたるが，以下は基本的に重要な項目である．
.加工形状 マスクパターンから設計どおりのパターンを
基板にエッチングする．そのために高度な異方性加工が要求され
図8･19にスキャン露光装置の利点を示す．
スキャン露光装置ではスリット状の露光光束をレチクル状に照射
し，レチクルとウェーハを相対移動させてデバイスパターンを焼付
ける．スリット状の露光光束はレンズの一部しか通過しないので，
収差の調整領域は小さく低収差のレンズ製造ができる．また，スキ
ャン露光によってスリット内の収差は平均化効果で低減できる．
フォーカス制御はスキャン中にウェーハのショット面に対してリ
アルタイムにオートフォーカス制御を行う．ウェーハにある程度の
ウェーハフラットネスが存在する場合でも露光面はウェーハの凹凸
に追従することができるため，フォーカスマージンは広がる．
スキャン露光装置はスキャン中のステージ位置を制御することで
ショット内（intra-field）誤差が補正できる．ショット内非等方倍
率や直交度は従来のステッパでは補正できなかった．このショット
内誤差補正により重ね合わせ精度は向上し，重ね合わせ精度向上に
より複数の露光装置の混合運用（mix&match）が容易になって
生産性が向上する．
このように微細化対応に要求される，高 ，低収差，フォーカ
スマージン拡大，重ね合わせ精度向上を満足すべく，スキャン露光
装置はステッパに代わって全盛期を迎えるに至った．

.速度 微細加工の観点から速度は犠牲になりがちだが，
生産性の観点からは高速性が要求される．
.選択比（selectivity) マスクや他の材質とのエッチン
グ速度に差をつけることによりエッチングが自動的に停止するよう
にすることが重要である．
.均一性 基板が大径化するに従い，エッチングの基板面
内均一性も素子の歩留り向上のために要求される．
.低ダメージ 高度な異方性をイオンにもたせるために高
速のイオンを用いる傾向があり，素子部分のチャージアップとも絡
んでダメージ低減は重要な要求である．
c .ドライエッチング装置の分類とその機構 エッチング機構
による分類と，プラズマの発生機構による分類がよく用いられる．
エッチング機構による分類では
(１)物理的なイオンの衝撃によるスパッタエッチング
(２)化学反応によるケミカルドライエッチング（CDE)
(３)両者を組合わせた反応性イオンエッチング（RIE)
に分けられる．スパッタエッチングはエッチング速度が遅く，半導
体製造プロセスではあまり用いられることはない．以下RIEと
CDEについて述べる．
.反応性イオンエッチング装置（RIE:reactive ion etch-
ing) 基本的にはプラズマ発生とエッチングが同一のチャンバ内
で行われ，プラズマによる分解反応の結果生じる活性な反応種の表
面への吸着と，イオンの衝撃エネルギーによる吸着サイトに対する
エッチングをその機構とする装置をいう．圧力の高い領域で行われ

この方式は，イオン照射表面のみエッチングされるので，イオン
の直進性により高度な異方性エッチングが可能である．したがって
LSI製造で必要とされる極微細なパターン形成には必須の技術であ
る．反応性イオンエッチング装置はプラズマを発生させる機構の違
いにより，次の型に分類される．
(１) 容量結合型（CCP:capacitively coupled plasma) 図
8･21(ａ)に示すような平行平板を電極とする電界型放電を利用した
装置．プラズマ密度はあまり高くないため，低圧力でのエッチング
速度に問題がある．比較的大面積にわたって均一なプラズマを作り
やすい．上部または下部電極にプラズマ発生電源を印加するだけで
基板に直流負電圧が発生し（自己バイアス），正イオンを基板に垂
直に引き込むことができるため，微細加工に必要な異方性エッチン
グが可能である．しかし，ダメージも考慮したさらに高度なイオン
エネルギーの制御を行うためには，バイアス制御用電源をもう一つ
基板側に設けるのが普通である．このバイアス電源はプラズマ発生
電源周波数より通常低めに設定される．
(２) 誘導結合型（ICP:inductively coupled plasma) コイ
ル電流が形成する磁界の時間的変化で電界を誘導し，その電界で加
速された電子がプラズマを形成するため誘導結合型と呼ばれる［図
8･21(ｂ)参照］．高度な異方性のエッチングを達成するためには，
イオンの直進性が要求され，イオン加速領域（シース領域）付近に
おける気体分子との衝突による散乱を避ける必要がある．そのため
エッチング装置は低圧力化を進めてきたが，プラズマ密度も同時に
低下するのでエッチング速度が低下し，生産性が低下する問題もあ
った．ICPは数パスカル以下の低圧力でも高い密度のプラズマ
10cmが生成可能である．CCP型と同様に，イオンを基板に
垂直に加速するためのバイアス用電源が基板側に設けられる．
(３) その他 高密度のプラズマを発生させる目的でさまざま
なプラズマが提案されている．例えば(ａ)プラズマの表面波をマ
イクロ波で励起する表面波プラズマ（SWP:surface wave
plasma)，(ｂ)磁場の周りの電子サイクロトロン運動を利用し
CCPの弱点であるプラズマ密度を改善したマグネトロンエッチン
グ装置［図8･21(ｃ)参照］，(ｃ)電子サイクロトロン共鳴を利用す
る電子サイクロトロンエッチング装置（ECR:electron cyclotron
resonance）［図8･21(ｄ)参照］などがあり，マグネトロンエッチ
ング装置は特にシリコン酸化膜のエッチングに多用されている．
ECRエッチング装置も実用化されている．
.ケミカルエッチング装置（CDE:chemical dry etching)
代表的な例は図8･22に示すように，プラズマを発生させてラジカ
ル種などの化学的に活性な反応種を作り出す部分と，基板を処理す
る部分が別々になっている．この形態の装置はリモートプラズマエ
ッチング装置とも呼ばれ，イオンを使わない等方的エッチングなの
で，微細加工には不向きであるが，荷電粒子がエッチングに寄与し
ないので，ダメージの心配がない．またこの方式の装置は，酸素プ
ラズマを利用してフォトレジストをはく離するアッシング装置とし
ても使われている．
d.使用ガス RIE装置もCDE装置も，エッチングする対象
となる材料により使用するガス種が異なる．配線材料（Al ，W等）
やポリシリコンなどのエッチングに対しては，Cl ，BCl ，HBrな
どのハロゲン系ガスを用いることが多い．シリコン酸化膜のエッチ
ングにはフロロカーボン系のガスを用い，側壁に形成されるポリマ
状の保護膜を利用して高いアスペクト比のホールを形成することが
できる．また，フロロカーボン系のガスのように下地のシリコン基
板や，ポリシリコン，あるいはゲート側壁のシリコン窒化膜とのエ
ッチング選択性が高いことも重要な要素である．初期にはCFが
多く用いられてきたが，このような選択性の観点や，地球温暖化の
問題からCFやCF等のガスが使用されることが多くなってき
ている．
