# 2・2・4 ジェロンテクノロジー

> **章**: 第2章 福祉工学
> **節**: 2・2 高齢者とジェロンテクノロジー

a. ジェロンテクノロジーの役割従来,高齢者の病気については医学が,身体障害や介護技術については福祉工学がもっぱら対応してきたが,健康高齢者を含めた高齢者全体が抱える生活上の問題を解決するための明確な技術的分野は存在しなかった.しかし,高齢社会の急速な進行に伴い,高齢者と製品・情報・環境等の社会システムとの不適合が顕在化してきたため,健康な高齢者を含むすべての高齢者が社会的に適合することを促進する技術分野の創出が必要となった.その結果生まれたのがジェロンテクノロジー(ger-ontechnology)といえる.この研究分野における役割について,国際ジェロンテクノロジー学会のホームページでは,「高齢者が良好な健康状態を保ち,快適かつ安全に自立的な生活と社会参加を可能にするための,技術及び環境の設計」と示されている.b. 歴史的経緯ジェロンテクノロジーという言葉は,英語のgerontologyとtechnologyの複合語として,1980年後半にオランダのアイントフォーヘン工科大学で誕生した.この技術分野の目指すものは,二つの言葉が示すように老年学とテクノロジーの合体であり,老年学等で得られた科学的な知見を,人間生活の向上に役立つテクノロジーとして社会に発信しようとするものである.欧米では,日本に比べ人口の高齢化が早くから進行しており,1956年の「国際連合経済社会理事会報告書」では,7%以上の高齢化率をもつ社会を高齢化社会とするとの定義がなされるなど,すでに高齢化社会への対策が重要なテーマとなっている.そのような社会背景の中で,1960年前後から高齢者特性とテクノロジーを結合させた論文が人間工学や心理学の分野で出現し始めた.1970年代から1980年代にかけて,人口の高齢化問題が社会的に重大な問題として広く認識され始め,アメリカ人間工学会の学術誌における加齢に関する特集号の企画や,国際人間工学会における加齢技術部会の形成など,この分野の研究が盛んになった.1991年にジェロンテクノロジーに関する最初の国際会議が開催され,2回めの国際会議の翌年1997年に国際ジェロンテクノロジー学会が誕生した.わが国においては,高齢化率が1970年に7%になった後も急激に増加を続け,2000年には17%を超える世界一の高齢化国家となった.そのような状況の中で,1999年に日本人間工学会の部会として,日本で初めてジェロンテクノロジーの名前を冠した組織が誕生した.2000年にはジェロンテクノロジー国際ワークショップが筑波で,さらに2005年に国際ジェロンテクノロジー大会が名古屋で開催され,この分野に関する期待が急速に高まった.c. 成果基盤的な成果としては以下のことが挙げられる.(1) 高齢者特性の理解に関する多くの研究が実施された結果,高齢者特性データベース(characteristics database of the elder-ly)の整備が世界的規模で飛躍的に進展した.わが国でも,(財)人間生活工学研究センターが3万人規模の人体寸法・形態データとともに,数百人から数千人規模の視聴覚や動作特性に関するデータベースを構築中である.(2) 高齢者特性データをディジタルヒューマンモデルに入力することにより,人と環境,製品とのインタラクションの評価がコンピュータ上で可能になった.(3) 高齢者対応製品の設計指針となる国標準作りが盛んになった.2001年には,わが国主導で提案された,高齢者・障害者のニーズに配慮した規格の策定に関する指針がISO/IECガイド71(ISO/IEC Guide 71)として承認された.また,応用的な成果としては以下のものがある.(1) 住宅や公共的環境,公共交通機関等におけるバリアフリーが進み,高齢者の生活安全や社会参加の促進が見られた.(2) 製品におけるユニバーサルデザイン化が進み,高齢者にも使いやすい製品が増加し,高齢者の行動の自立促進に役立った.(3) 人間特性に配慮した福祉機器の設計が可能となった.(4) 高齢労働者が定年後も就労可能な職場改善がなされた.
