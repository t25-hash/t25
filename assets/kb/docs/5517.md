# 2・3・1 福祉機器を取り巻く問題

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

福祉機器は、使用者の生活に密着に結びついている。したがって、使用者の障害や特徴、介護者の状況、使用環境、社会制度など多くの問題が複雑に絡み合い、福祉機器の開発や設計、使用に対して影響を及ぼす。福祉機器を考えるには、これらを総合的に考慮しなければならない。最も基礎的なことは、障害の医学的な理解である。使用者は何らかの身体的障害をもっているため、それを高進させないことが福祉機器には要求される。また、障害の特徴を把握するとともに、障害の個別性(individuality)を考慮する必要がある。例えば、頸髄損傷の人が使う電動車いすを、そのまま脳性麻痺の人が使えるかというと、そんなことはない。頸髄損傷の人には軽い力で動かすことのできるジョイスティックが適しているのに対し、脳性麻痺の人には不随意運動の影響を受けにくい硬めのジョイスティックが適しているかもしれないからである。9 ―13障害の個別性は、工業製品の製造過程において問題を生じる。すなわち、大量生産によるコストダウンが図れず、むしろ多品種少量生産が求められる。究極の福祉機器は、頸髄損傷の人用、脳性麻痺の人用というように、オーダメイドで作ることになってしまう。しかし、これでは経済効率が上がらない。そこで、モジュール化やアジャスタブル化を図ることが行われている。これにより、共通の部品としてある程度の量を確保しながら生産することができる。また、CAD(computer aided design)/CAM(computer aided man-ufacturing)システムを利用した個別生産システムを構築し、一人ひとり違ったものを効率よく生産する試みもなされている。個別性に対応するもう一つの重要な項目は、福祉機器の適合である。それぞれの使用者の特徴や使用目的、使用環境に合わせて、適した福祉機器を選択する作業である。ここでは、機器の知識、医学的知識以外に、使用者の生活をどのようにとらえるか、また支給制度のような社会制度をどのように利用するかといった知識が必要になる。セラピストや看護・介護職員のような中間ユーザがこの作業に関わることが多いが、彼らの存在もまた福祉機器に関わる問題の複雑さを増す要因となる。公的支給制度により機器を手に入れようとする場合、購入にかかる費用を行政が負担することになる。ここで、問題となるのは行政の判断であり、それによって必ずしも十分な機器の供給がなされない場合がある。特に新しい機能をもった高価な機器はなかなか支給の対象にならないため、支給の枠内という制約によって機器の進歩が停滞する可能性がある。使用者側は支給されるものを第一に考え、大きな費用負担をしてまで高機能の機器を手に入れようという考えが薄れていく。製造者側も給付の枠内で同型の機種を製造していればよく、新たな機器を開発する意欲を感じなくなる。公的支給制度には、このような悪循環を生じる可能性を含んでいる。介護保険制度や障害者自立支援法による制度においては本人の費用負担を求めており、このような問題に対する効果が期待されている。近年、ユニバーサルデザインが注目されている。これは、障害のある人でも使いやすいように一般製品を設計しようというものである。障害の程度が軽度であれば、個別性への要求はあまり強くなく、ある程度共通の特徴をつかむことができる。それらの問題は、ユニバーサルデザインにより解決可能である。以上のように福祉機器は特有かつ複雑な問題を多く有している。福祉機器に関わるエンジニアは、これらの点を十分考慮しなければならない。
