# 8・3・3 不純物導入（doping） ・拡散装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・3 半導体チップ製造装置

不純物の導入は半導体素子の電気的特性を決定する工程であり，
素子の高性能化と微細化に伴って，その導入，活性化および拡散に
高度な制御が要求されるようになってきた．最近では不純物の導入
は，イオン注入装置によって不純物を注入し，その後の熱処理（ア
ニール）工程によって不純物の拡散と電気的な活性化を行うのが主
流である．

a.イオン注入装置（ion implantation) イオン注入装置とは
注入する元素をイオン化し，電場で加速して基板に注入する装置の
ことで，加速電圧により注入深さを精密に制御することができる．
装置の構成は図8･23に示すように通常，イオンの発生部，引出部，
分離部，加速部，レンズ，スキャン電極，イオン中性化部，基板処
理室からなる．イオン発生室はプラズマによるイオン生成をガス原
料から行い，出口外側にあるイオン引出電極によってイオンが取出
され，予備加速される．次に磁場中で所望の元素のイオンのみが質
量分離され，所定のエネルギーに加速され，磁気レンズでビーム制
御された後，基板に照射される．スキャン電極によってビームを振
る場合もある．またイオンにより基板が帯電し，素子にダメージを
与えるため，処理室直前にイオンのニュートライザが設けられる場
合もある．基板処理室には注入イオン量を計測するためにファラデ
カップを備えた装置が多い．
注入するイオン種としてはp型（アクセプタ型）としてボロン
が多用されるほか，ガリウムやインジウムなども用いられることが
ある．n型（ドナー型）としてはリンやヒ素等が多く用いられる．
b.イオン注入の問題点と動向 イオンにより，基板に原子を
注入する（イオン打込み）と，打込んだ原子が無衝突で（電子との
衝突はある）基板の結晶格子の空間を進むチャネリングといわれる
現象が起こる．この原子は所望の打込み深さより深いところまで達
するため，不純物の分布に問題が起こる．これを防ぐには基板をイ
オンの入射角度に対して若干（7° ～10 ° ）チルトさせることで対応
する．しかしボロンのように小さい原子ではチャネリングはいまだ
に問題である．イオン注入深さはますます浅くなる方向にある．そ
のためイオンの打込みエネルギーを従来の20～200keVから2～10
keV程度に下げる必要がある．しかし，打込みエネルギーを低下
させると，チャネリングに対する基板の限界チルト角度が大きくな
る，あるいは基板損傷の低下によりチャネリングが起こりやすくな
るなどの不利な点もある．その解決方法の一つとして，目的元素の
注入前にあらかじめゲルマニウムやシリコンなどの元素を打込むこ
とで基板をアモルファス化しておく方法がある．また，低エネルギ
ーイオン打込みには，高速イオンを減速して低速イオンを作る方法
があるが，電気的に中性化したイオンが減速されずに基板に到達す
るといわゆるエネルギー汚染が起こる．エネルギー汚染はこのほか
導入したガス（例えばBFをBFとして注入する場合）が加速
中に他の分子と衝突するなどして分解した場合にも起こりうるの
で，装置内の圧力はできるだけ，低圧に保つ必要がある．
金属汚染も深刻な問題を引き起こすが，これは2種類ある．一つ
は質量分離部の問題で，例えばSiとFeは質量分離では区別
がつかず，鉄が混入し金属汚染となる．もともとの原因は汚染物質
イオンがビームに混入することであるが，ハロゲン化ソースの使用
などでイオン化室が活性反応種にさらされると，MoやFeなどが
構造材から放出され，これは今後ますます厳しい問題になると予想
されている．もう一つは，基板処理室に多用されているアルミニウ
ムがイオンによりスパッタされ，それが基板を汚染するケースであ
る．これは，ヒ素など重い元素の打込みにおいて，特に顕著であ
る．
c .その他の不純物導入装置 不純物の導入には，拡散ソース
を用いる方法がある．この方法は大がかりなイオン注入法に比べ
て，装置が単純で安価である利点があり，精度をそれほど必要とし
ない不純物導入に使用される．代表的な装置は図8･24に示すよう
にヒータ，処理室，ガス供給系からなるバッチ式の炉で，多数枚の
基板が一度に処理できる．原理は，不純物源として気相にBF ，
BHやPOClなどのガス状原料を供給し，次に基板の表面になん
らかの方法で析出させ，それを基板中に固体拡散させるものであ
る．表面への析出の方法としては，(１)基板表面とガスソースと
の表面反応を利用するもの，(２)気相中でのガスソースの分解に
よる表面析出を利用するもの，(３)プラズマによるガスソースの
分解とイオン化による打込み効果を利用するもの，(４)CVD
（chemical vapor deposition:化学気相成膜法）等により不純物を
高濃度に含む薄膜を堆積させるもの，などがある．基板をエキシマ
レーザ照射することにより，ガスソースの気相分解，表面析出，固
相拡散を同時に行う方法もある．最近注目される応用例としては，
不純物を高濃度に含むトランジスタゲートの酸化膜に高濃度Bド
ープし，そこからの固相拡散によって，浅い接合を作るものがあ
る．
d.拡散装置 導入された不純物は，イオン注入の場合にはそ
のままでは電気的な活性化率が低いため，アニール（anneal ）し
て活性化する必要がある．アニールの過程で不純物は同時に固相中
を拡散するために，拡散深さを浅く制御する必要のある素子では，
いかに拡散の程度を制御するかが重要な問題である．一般的な拡散
炉は基本的には図8･24に示すような不純物導入装置と同型式のも
ので，昇降温度を厳密に制御し，不活性ガス雰囲気中で行われる．
最近ではさらに拡散時間を短くすることを目的として，高温で短時
間のアニールを行う装置が多くなってきている．代表的なものは，
図8･25に示すランプアニール（RTA:rapid thermal anneal ）装
置と呼ばれる装置で，ハロゲンランプにより基板を1枚ずつ
100℃/秒のオーダの速度で加熱する．また冷却水などによって炉壁
の温度が上昇しない構造になっているので，図8･24に示す装置に
比べて冷却速度も大きい．最近ではさらに数ミリ秒から数十ミリ秒
