# 2・3・2 福祉機器の適合

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

福祉機器は、使用者の身体的特徴や使用環境に適合して初めて有効に使用される。したがって、機器・機種の選定は慎重に行うべきである。また、市販の福祉機器に改造や変更を加えて使用する場合も多い。使用者に適した機器はどのようなものであるかを明確にし、最適な機器・機種を選定していく作業を福祉機器の適合という。福祉機器の適合をどこで誰が行うかについては、まだ明確な答えが出されていない。現状では更生相談所、病院、リビリテーションセンター、施設、養護学校、福祉機器販売店などで、医師、セラピスト、義肢装具士、介護職員、リハビリテーションエンジニア、教師、業者等福祉機器の知識のある者が行っている。福祉機器の適合では、機器の知識と医学的知識、福祉制度の知識、そして使用者の状況から適切な機器選択をすることができる能力などが必要とされる。これらは徐々に体系化が進められている段階である。適合の手順は実施している場所や実施者の立場によって異なるが、おおむね以下のようなものである。(1) 使用者、家族等のニーズの確認(2) 身体状況の評価(3) 生活状況、機器の使用環境の評価(4) 方針の決定(5) 仮適合機器の製作と適合調整9 ―14医療・福祉・バイオ機器(6) 試用評価(7) 購入・申請手続き(8) 導入(9) 再評価これらのプロセスを、本人、家族、場合によっては施設や学校の職員等福祉機器の使用に関わる者と共同で進めていくことが重要である。使用者の生活の中で一つでも大きな不都合があると、その機器は使用されなくなる可能性が高い。使用場所、使用状況を実際に見て、確認することも重要である。
