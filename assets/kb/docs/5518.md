# 8・3・4 薄膜形成装置（thin film deposition reactor )

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・3 半導体チップ製造装置

a.概要 半導体の素子構造はおもにシリコン基板上に立体的
に形成された10層に及ぶ薄膜から構成されている．この素子構造
を形成するために用いられるのが薄膜形成装置およびエッチング装
置である．したがって，電子素子，特に，半導体製造工程において
は薄膜形成が必須である．素子構造の中にトランジスタ，ダイオー
ド，コンデンサ，抵抗などに相当する回路を作り込むため，用いら
れる薄膜も，半導体，金属，絶縁性などの各種の物質が用いられ
る．また，その形成方法も物理的方法，化学的方法など多岐にわた
っている．表8･3に薄膜の種類と応用を示す．
b.目的 薄膜形成装置の目的は半導体ウェーハ上に各種の材
料の薄膜を形成させることにある．単結晶膜（エピタキシャル膜）
成長，多結晶膜成長，酸化膜成長，金属膜成長，絶縁膜成長などが
行われる．ウェーハの大口径化が進んでいるため，いずれの薄膜形
成についても
(１)膜厚，膜組成の均一性
(２)大きなスループット（単位時間当たりの処理量）
(３)不純物，ダストなどの汚染が少ないこと
が要求される．薄膜の種類に応じて形成方法，装置が固有の形式で
開発されている．
c .各種の方式 薄膜の形成方法は大別してドライ方式
（PVD:物理的気相堆積法，CVD:化学的気相堆積法，など）とウ
ェット方式（塗布法，電解めっき法など）がある．ここではドライ
方式の代表例としてスパッタリング装置，CVD装置，ウェット方
式の例として電解めっき装置について説明する．
.スパッタリング（sputtering）装置 スパッタリング法
は一般的な薄膜形成法として幅広い材料に用いられている．真空容
器内でプラズマによりイオン化したArをターゲット電極表面に衝
突させ，生成したスパッタ粒子を対向して配置したウェーハ上に堆
積させる．スパッタ粒子はイオン化され，バイアスを印加したウェ
ーハ側に引き込まれる．さらにウェーハ側に印加したバイアスによ
り形成膜は再スパッタされるため，その結果，微細孔の側壁と底部
にも膜が形成される．近年はCu配線形成の前処理プロセスへの適
用が図られている．配線にCuを用いることが進みその形成工程と
してダマシンプロセス（絶縁膜に溝を掘りその溝内に電解めっきな
どの方法でCuを堆積し配線を形成する方法）が用いられ始めた．
スパッタリングはCuを堆積するに先立ってCu配線のシード形成，
段差部へのTa，TaN系バリア層の形成に用いられる．
.CVD装置（chemical vapor deposition reactor ) 目的
元素を含有する原料（水素化物，塩化物，有機物など）の気体をウ
ェーハに供給し気相または表面での化学反応を利用して目的物質を
堆積させる．このとき用いる化学反応の励起手段として熱，プラズ

図8･27枚葉式CVD装置
マなどが用いられ熱CVD，あるいはプラズマCVDと呼ばれる．
適用材料としてバリアメタル，Cu，Al ，W，層間絶縁膜などがあ
る．金属膜形成には熱CVD装置というように膜に応じて用いられ
る．装置形式としてはバッチ方式あるいは枚様方式が用いられる．
装置図を図8･26，図8･27に示す．プラズマCVDでは大面積に均
一にプラズマを形成することが必要であるため枚葉式（single
wafer ）がよく用いられる．
原料ガスとして金属原料の有機物（有機金属）を用いる場合を
MOCVD（metal organic chemical vapor deposition）と呼ぶ．青
色レーザなどの材料であるGaN（チッ化ガリウム）の単結晶薄膜
成長などで実用化されている．装置としては熱CVD装置が多く用
いられる．
.電解めっき（electroplating）装置 電解めっき装置は
上述のCuを堆積させる装置として多く用いられるようになった．
ウェーハにはCu堆積する前に，あらかじめCuシード層が全面に
形成されている．ウェーハは外周で接触するコンタクト電極からの
給電によりカソード電極となっている．Cu析出の面内均一性を確
保するため回転機構を有している．めっき液はめっき槽とタンクの
間を循環し，噴流式にウェーハに接触する．アノード電極はウェー
ハに対向してめっき液中に設置されている．アノード電極はリンを
0.04～0.08%含有した含リン銅アノードもしくは純銅アノードが用
いられ，めっき液中に溶解して析出により消費されたCuを補給す
る．LSIに用いられるめっき装置はスループット確保のため複数の
めっき槽とスピンリンス/ドライ槽を有している．また，滴定や
CVSが組込まれた自動分析・薬液供給装置が接続されめっき液の
液組成，添加剤濃度が自動で管理される．さらに，後工程でのクロ
スコンタミメーションやCuの自己拡散を防止するためウェーハの
裏面洗浄，エッジ部のCu除去機能，めっき後のアニール処理まで
を同一装置内で行うことのできる装置も市販されている．
