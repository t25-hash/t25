# 2・3・3 ハイテクノロジー福祉機器と道具

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

工学の先端技術を取り入れた福祉機器は障害者・高齢者のQOL向上に大いに役立つものである。ロボット技術や制御技術、センサ技術、VR(virtual reality)技術などを取り入れた福祉機器の開発は盛んに行われている。パワーアシスト技術を応用した車いすは、日本が世界に先駆けて開発し、市販に至っている福祉機器であり、現在、世界的にもシェアを伸ばしている。パワーアシスト技術の応用は、パワーアシストスーツ(power assisted suit)などその他の機器でも研究・開発が進められている。新しいコンセプトの移動機器として注目されているものに、ジャイロセンサにより姿勢を検出し、倒立振り子の制御技術を応用した電動車いすがある。2輪で立ち上がって高いところの物を取ることができたり、階段の昇降、段差の乗越えを可能としている。また、音声認識技術の応用も進んでおり、四肢麻痺者でも音声によりテレビやエアコンといった家庭電化製品を操作することができる環境制御装置などが実用化されている。福祉ロボットの分野でも、四肢麻痺者の自立支援を行うロボットアームの開発が行われている。スイッチ式のジョイスティックを操作することで、四肢の麻痺した障害者でも介助者なしで食事をすることができる食事支援ロボットが開発され、改良を重ねて、実用的な製品として市販されている。先端技術を使った福祉機器でも、そのハイテクノロジーの部分のみを強調した機器は、結局は使われることなく消滅してしまう。ここが福祉機器の難しいところである。リハビリテーション工学の創生期に盛んに研究が行われた高機能な電動義手が、結局はあまり普及していないことは象徴的である。例えば、マウススティックという棒がある。四肢麻痺者が口にくわえて、日常生活におけるさまざまな作業に使用する道具である。その棒で、パソコンを打つ、新聞を読む、電話をかける等々いろいろなことができてしまう。衛生上の問題や歯茎を痛めるなどの問題もあるが、多くの四肢麻痺者が使用している。その理由は、手軽にそれだけのことができる福祉機器がいまだにできないからであろう。そのほかにも、障害者が自ら工夫して作ったものは、生活に密着して実によくできている。輪ゴム一つをとっても、スプーンを手に固定したり、つまみを回しやすくするなど、有効な使い方がいろいろできる。ハイテクノロジーな福祉機器も、最終的にはこれらの道具のレベルまで手軽さや使い勝手を向上させなければ、生活に浸透していくことは難しい。
