# 8・4・1 グラインド装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

バックグラインドは，①テープ貼付け，②バックグラインド，③
テープはがし，④純水洗浄/乾燥，⑤厚み測定の各工程からなる．
近年用いられている先ダイシングの工程は，①ハーフカットダイシ
ング（溝深さは最終チップ厚み＋20m程度），②デバイス面保護
テープマウントウェーハローディング，③オリフラ/ノッチ検出，
④裏面研削（この最中にチップセパレーション），⑤ウェーハ洗浄
と乾燥，⑥転写装置へ搬送である．粗研削，仕上げ研削，ウェーハ
アンローディングが行えるようになっている．バックグラインドは
ウェーハの厚さの均一性が要求されるがほとんど装置およびテープ
の選定でその精度は決定される．テープ貼付け工程ではウェーハ表
面を保護するために保護テープを張る．テープは基材100～200
m，粘着剤1020mからなる．バックグラインド装置は通常二
軸の砥石からなる．粗研磨から仕上げ研磨を行い，ウェーハ裏面を
鏡面仕上げする．一軸めに#360メッシュのダイヤモンド砥石を使
い，約50ms程度の周速度で回転させながらウェーハに押付け仕
上げ厚み＋20m程度のところまで削り込む．二軸めでは#2000
メッシュの砥粒を含んだレジノイドボンド砥石で約58ms程度の
周速度で回転させながらウェーハに押付け最終仕上げ厚みまで削り
込む．元厚さ700m程度のウェーハを通常200m程度に仕上げ
る．テレホンカード等ICカードに搭載されるLSIでは50mまで
薄くする技術が求められている．
