# 2・3・5 海外の福祉機器との相違

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

福祉機器の普及が促進されてきている今日,外国製の福祉機器も日本国内で多く流通している.10年ほど前までは,文化の問題や家屋構造の問題,体格の問題などで日本には適さない機器もあるとされていたが,最近では機器のモジュール化などにより,日本の状況でも使用できる外国製品が市販されるようになってきている.また,外国製品の穴を埋める形で開発・市販される国内製品も多くあり,その意味で,日本の福祉機器市場においては国内製品と外国製品の融合がうまくなされてきているといえる.福祉機器の分野にもアメリカやヨーロッパを中心としたグローバリゼーションの波が訪れており,日本もその中に飲み込まれつつある.日本の福祉機器メーカにも国際競争力が求められる時代になってきている.
