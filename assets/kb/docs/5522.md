# 8・4・3 ダイボンド装置

> **章**: 第8章 電子機器製造機械・装置
> **節**: 8・4 パッケージング装置

ダイボンド工程ではカットされたチップがAgペーストなどのダ
イボンド剤を用いてリードフレームのダイパッドに固定される．工
程は図8･33に示すとおりである．
ボンディングステージ上でAgペーストをリードフレームのダイ
パッドに塗布する．ピックアップステージにてダイシング工程でキ
ャリヤテープ上に個々に分割されたチップをキャリヤテープ下面か
らニードルで突き上げる．テープから分離されたチップは直上で待
機しているバキューム穴をもつピックアップツール（コレット）に
吸着される．ピックアップされたチップはボンディングステージに
移動しAgペーストの塗布されたダイパッド上にボンディングされ
る．Agペーストを硬化させチップをダイパッド上に固着させる．
図8･34にダイパッド上でのボンディング工程の模式図を示す．
最近三次元実装（ダイの上にダイを重ねる）が進むに伴いダイの
薄型化がますます進められている．そのためダイボンディング装置
としてはダイシングテープからのピックアップを中心に工夫を凝ら
されている．具体的には上からコレットで吸い付けるタイミングと
下からニードルで突き上げるタイミングを完全同期化する（どちら
か一方が過剰な場合ダイに荷重がかかり結果的に破損する），ボン
ディングヘッドを軽量化する，などである．
