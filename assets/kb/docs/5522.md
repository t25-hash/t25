# 2・3・6 福祉機器開発手法

> **章**: 第2章 福祉工学
> **節**: 2・3 福祉機器に関する注意点

福祉機器開発でよくいわれることは,しっかりとしたニーズ(needs)を把握して機器開発に臨むことである.これは非常に重要であり,ニーズをつかみ損ねたために,全く見当違いの機器を開発してしまう場合も多く見受けられる.しかし,ニーズばかりを考えて機器開発を進めてしまうと,工学技術の進歩は減速し,技術の限界を迎えてしまう.特に,重度障害者を対象とした福祉機器開発では,技術シーズの進歩がなければ越えられない問題が多くある.したがって,工学としての技術シーズをしっかり開発することも重要なファクタとなる.以上を踏まえたうえで福祉機器開発を次の三つの段階に分けて考えることができる.(1) 基礎開発(2) 実用化開発(3) 商品化開発この三つの段階は,必ずしもクリアに分けられるものではないが,それぞれの段階は福祉機器開発における重要な特徴を含んでいる.基礎開発では,おもに技術シーズの開発に主眼をおく.技術シーズの福祉機器への応用を考えながら,その技術シーズのさらなる開発を進めるものである.実用的な福祉機器までまとめ上げることよりも,技術シーズの応用の可能性やさらに解決しなければならない問題点などを整理し,将来のビジョンを示すことが重要な役割となる.実用化開発では,ニーズとシーズを合致させ,実用的な福祉機器までまとめ上げることに主眼をおく.その流れを図2・7に示図2・7 実用化開発の流れす.使用者のニーズから開発を始める場合と,技術シーズから開発を始める場合が考えられる.いずれにしても使用者の特徴や使用環境,社会状況,社会制度,適合技術等の背景因子を十分に分析し,ニーズに見合った技術シーズを組合わせて福祉機器の開発目標を定め,コンセプト作りを行う.これらを行うのに有効な手法として,使用者を開発グループに含めるユーザパーティシペーションの手法や,逆に開発者自身が想定される使用対象者や使用現場に密着して問題分析を行う手法などがある.次の段階では機器を試作し,試作機を実際に使用してもらう試用評価,さらにはその評価結果から指摘された問題点の改良を行う.ここでは,しっかりとした試用評価が重要である.まず試用評価に耐えうるだけの安全性やユーザビリティ等の完成度を確保した機器を試作する必要がある.そのうえで,想定した使用対象者に実際に生活の場面で使ってもらうことにより,機器の実用性を高めていく.さらに,長期的に試用することによって,機器の効果を試用者のADL,QOL等の観点から明確化することも,実用化開発において重要な項目である.これらにより,実用的な機器の構成と,その試用により得られる効果を明らかにすることができる.商品化開発は,福祉機器開発における最終段階であり,商品として仕上げることが主眼となる.まず,開発機器の使用対象者を把握し,その範囲を明確にしたうえで,それに向けてのはん用化を行う.モジュール化やアジャスタブル化による対応などもはん用化に向けての解決策の一つとなる.同時に機器の適合に必要なノウハウの蓄積や,その教育なども必要となる場合がある.安全性の確保も重要であり,人体への危害をなくす配慮や,JIS 規格やISO規格で定められた試験評価を実施する必要がある.また,生産コストや市場性の検討,社会制度への対応といった経済性の考慮も,商品として成功するかどうかに関わることであり重要である.基礎開発・実用化開発・商品化開発は,1箇所ですべてを行うことがほぼ困難である.大学,公的研究機関,企業のそれぞれで目指すものは異なっている.そこで,それぞれの開発段階での成果を次の段階に引き継いでいく技術移転は,福祉機器開発において重要な課題である.これをスムーズにかつ有効に実行していくための政策も,福祉機器の促進に大きく影響を及ぼすこととなる.ここで示した福祉機器開発手法は一つの例であり,開発する機器の種類,対象者,使用目的等によって異なる手法はいくらでも存在する.重要なことは,最終的には障害者や高齢者に役立つ機器として使われることを目指して,福祉機器開発に取り組むことである.それには,開発者の独りよがりな考え方のみでは成功することはなく,使用する人の立場に立って考えることが重要である.また,基9 ―15礎開発では実用化や商品化にはほど遠い機器を開発する可能性もある.大切なことは,その機器開発が使用者の手に届くまでのどの段階にあり,解決しなければならない問題は何であるかというビジョンを的確に把握することである.有用な事実を積み重ねていくことにより,社会全体として福祉機器を進歩させることが重要である.(1) 福祉(rehabilitation,well-being,healthcare)福祉を英訳してwelfareとする日本人は多いが,多くの場合適切ではない.この語は,施し与える福祉という意味であり,最近はあまりよくない意味に使われることも多い.日本語の福祉概念と一致する英語はなく,rehabilitation(障害者福祉),well-being(福利),healthcare(健康管理)などの語をその意味に合わせて使い分ける必要がある.(2) 障害者(person with impairments,person with disabili-ties)障害者の害という漢字は,じゃま者という否定的な印象を与えるため,古い漢字である障碍者を使おうとする運動もあるが,まだ一般的にはなっていない.英語では,その障害の意味をよく考えたうえでperson with impairments,person with disabilitiesなどの表現が使い分けられている.このような,障害という特性をもった人という表記法は現在の福祉概念の現れであり,the disabledのような過去分詞による表現はいまはあまり使われない.また同じ障害でもhandicapは個人の特性というより社会が原因として生まれた不利益という概念であるため,あえてそのような意味を含めたい場合以外には使わない.また最近はthe challengedという表現もよく使われるようになったが,科学技術論文では見掛けることは少ない.障害の概念の変化によって使われる表現が変化しており,注意を要する.(3) 福祉機器(assistive technology)福祉機器は福祉用具とも呼ばれる.福祉用具は行政用語であり,おむつなどまで機器と呼びたくないという発想であろうが,基本的に指すものは変わらず,どちらを使ってもよい.英語ではassis-tive technologyやassistive deviceが一般的である.前者は福祉機器のほかに支援技術という意味で使われることもあるので注意を要する.Technical aidという用語は現在ではほとんど使われない.(4) 介護・介助介護と介助の区別は必ずしも明確ではなく,人によって使い方が異なる.使い分けの指針の一つとして,介助は実際の作業が明確にできる場合,介護は一般的な用語と考えるとわかりやすい.例えば,介助者は特定の人に対して決まった援助をする人を指し,介護者はさまざまな援助作業をする人を指す一般用語であるというものである.英語ではassist,care,aid,help,supportなどが使われる.Assistは被介護者が何かをしようとするのを助けるという意味であり,一方的に介護者が何かする場合には当てはまらない.Supportも支える,支援するという意味でassistに近い.Aidはよく使われるが,近年はHIV 陽性との混同を避けて使用頻度が減っている.Careは治療的意味合いも含んでいる.Helpは一般語である.(5) 高齢社会WHOの定義によれば,高齢化社会(ageing society)とは全人口に対する高齢者の割合(高齢化率)が7~14%の社会をいう.高齢化率が14%を超えている現在の日本はすでに高齢化社会ではなく,高齢社会(aged society)と呼ぶのが正しい.(6) 高齢者(elderly person,the elderly)老人という表現は嫌う人も多く,高齢者と表記するほうがよい.英語ではelderly personが多く,aged personも使われる.国連による定義は65歳以上の人であるが,国連の統計では60歳以上の統計が多くとられている.9 ―16医療・福祉・バイオ機器(7) 差別的表現文章を書いたり話をしたりする際には,差別的表現をしないように心がける必要がある.例えば,つんぼ,めくら,おし,いざりなどはすべて差別用語であり,使うべきではない.ろうは適切な用語であるが,それを漢字表記した聾はつんぼとも読めるのでこの漢字は使わず,平仮名表記が望ましい.基本的には用語の問題ではなく,使っている人が差別意識をもっているかどうかが問題ではあるが,嫌っている人がいる表現は避けることが望ましい.自分の意志がはっきり表せない重度障害者であったとしても軽視したりせず,その人の人格を尊重することが重要である.文献(1)
