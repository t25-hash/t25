# 3・1・1 生活工学のコンセプト

> **章**: 第3章 生活工学
> **節**: 3・1 生活工学のコンセプトと歴史

生活工学（life engineering）は，われわれが生活の中で遭遇す
る製品，環境，情報に関するさまざまな問題を工学的手法を用いて
解決することにより，生活の安全，快適，自己実現などの質的な向
上や新しい生活価値の創出を目指す，非常に幅広い研究領域であ
る．具体的な研究分野としては，12～14章に示されるような健康
工学，感性工学，スポーツ工学があるが，10，11章に見られるジ
ェロンテクノロジー，製品，環境におけるバリアフリーとユニバー
サルデザイン等に関する研究に加え，まちづくりに関わる都市工
学，社会システム工学，安全工学，住居学，衣服に関わる繊維工
学，水質や空気の汚染問題に関わる環境工学やリサイクル技術等
も，広義の生活工学に含まれる．
これらのことからわかるように，この分野は医学，生理学，心理
学，社会学，経済学などの人間とその生活に関わる研究領域を中心
として，物理学，化学，生物学等の基礎科学と機械工学，電子工
学，材料工学，生体工学，バイオテクノロジー等の工学的手法が融
合して構成される学際的な研究領域である．
