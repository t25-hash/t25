# 3・1・4 生活工学において重要な基盤研究

> **章**: 第3章 生活工学
> **節**: 3・1 生活工学のコンセプトと歴史

生活工学において人間中心設計をするためには,人間特性の十分な理解が必要であるが,日本におけるデータベース整備の現状は身体的寸法を除いては十分とはいえない.そこで,人間の生理,形態,感覚,認知,動作,行動等の人間特性に関する知識を利用しやすいように体系的に整理したデータベースを構築し,それを公開する事業が経済産業省の主導で活発に行われている.また,データベースを応用的な場面で使いやすくするために,人間特性を簡明に解説したガイドラインやJIS(Japanese Industrial Standards)やISO(International Organization for Standardization)などの規格の作成が,関連学会とともに産学官を挙げて進行中である.さらに製品,環境,情報における人間適合性評価を迅速かつ効果的に行うための,人間特性を計算機内でモデリングするディジタルヒューマン技術の精緻化が世界的な課題である.特に工学的な側面を取り上げると,人間特性データベースのための新規な計測法の開発がある.例えば,12章の健康工学の分野では,各種の生体の状態をモニタリングするための計測法の開発が必要不可欠である.また,13章の感性工学の分野において,感性に合った生活関連製品をデザインするには,単なる寸法データだけではなく,身体構造の三次元形状変化の計測に加え,五感の交互作用の計測が必要となる.前者については,身体形状は姿勢や時間とともに微妙に変化するので,瞬時に計測できるシステムが必要不可欠である.最近,1秒程度で,全身の三次元形状が計測可能な装置が開発されたばかりである(14・3節参照).一方,後者の五感の交互作用に関する計測はこれからの課題であろう.例えば,われわれは食事をするとき,暗闇で食べるのと明るい部屋で食べるのとでは明らかに味が異なる.あるいは,歯が丈夫なときと虫歯の治療中などで調子の悪いときでも味は異なる.このように味覚,視覚,触覚等の五感は複雑な交互作用をもっており,これらの関係をモデリングすることが今後の感性工学発展の基盤研究として必要不可欠である.
