# 3・1・5 生活工学への社会的な期待

> **章**: 第3章 生活工学
> **節**: 3・1 生活工学のコンセプトと歴史

生活工学の研究領域が1990年代以降重視され始めた理由の一つとして、それ以前の高度経済成長時代に対する反動を挙げなければならない。高度成長期においては、産業界はひたすら新機能の開発と低コストを目標として使い捨ての生活財の生産を進めることが多かった。そのため、広くは資源の大量消費とそれによる空気、水などの環境インフラストラクチャの悪化に代表される地球環境問題が生じるとともに、家庭内では使いにくい機能をもつ生活財があふれていた。時を同じくして、バリアフリーやノーマライゼーションなどの福祉的な取組みが欧米の先進諸国を中心として広がりを見せつつ、人口の高齢化による切実な問題が顕在化し始めた。このような事態の中で、障害者や高齢者の自立的な行動を確保するとともに、広く人間生活の質を問い直す社会的な機運が高まり始めた。その結果、生活の安全、安心や快適さなど、生活の質的向上を目指した工学的な取組みの必要性が社会的に認知され始めた。ここに生活工学発祥の原点があるといえる。生活工学の領域は、すでに述べたように人間生活が関わる膨大な範囲にわたるが、その中から特に生活の安全、安心、快適性、自己実現などの人間特性に深く関係している分野として、12~14章で詳述されている健康工学、感性工学、スポーツ工学に関する技術について以下に概説する。9 ―18
