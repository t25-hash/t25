# 9・2・3 各種製版

> **章**: 第9章 印刷機械
> **節**: 9・2 製版

a.凸版 凸版印刷の刷版には金属凸版，ゴム版，感光性樹脂
版などがある．金属凸版は，感光液を塗布した亜鉛板にネガフィル
ムを密着して露光し，水洗，バーニングの後，硝酸などで腐食して
作られる．感光性樹脂版の処理方法は版材によって異なるが，感光
性樹脂層は紫外線で硬化または不溶化する性質があり，画線部を光
硬化させた後，非画線部である未露光部を水や溶剤で洗い出した
り，エアで吹き飛ばすことにより，露光された部分を残し，凸版レ
リーフを形成する．最近では，フレキソ印刷の製版のようにネガフ
ィルムを用いることなく，感光性樹脂版材上に直接レーザで出力さ
れる．
b.平板 平版印刷の刷版としては，通常0.2～0.3mm厚さ
のアルミニウムの平板に感光剤をコーティングしたPS（presensit-
ized）版が用いられる．光が当たった部分が画線部となるネガタイ
プと，光が当たらない部分が画線部となるポジタイプがある．フィ
ルム原版とPS版を焼付機で密着，露光後，現像処理することによ
って製版されていたが，最近では，フィルムを使用せず製版データ
から直接刷版にレーザで製版する装置が多く使用されている．
c .グラビア 製版方式には二つの形式があり，一つは，製版
画像データを出力し，画線部を薬品で腐食させる方法である．この
方法での凹部の製作には，次の三つの方法がある．①グラビアでの
濃淡表現の種類に対応して，凹部の大きさを一定にし，深度を変化
させるコンベンショナル法，②深さを一定とし凹部の大きさを変化
させる網グラビア法，③大きさと深さの両方を変化させる方法であ
り，印刷物によって使い分けられている．もう一つは，シリンダを
回転させながら，レーザ等で直接に版を作る方法である．凹部の面
積と深さがともに変化して濃淡を表現する．
d.スクリーン印刷（screen process printing) スクリーン印
刷の製版方法は．手工的なカッティング法，描画法，写真製版法，
フォトエッチング法，めっき法，電子製版法などいろいろな方法が
ある．現在おもに使われているのは写真製版法であり，直接法と，
現像して得られた版膜をスクリーンに転写する間接法とがある．
