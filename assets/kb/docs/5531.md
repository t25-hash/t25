# 3・2・2 項で、二つのモニタリングシステムに言及したが、今後、

> **章**: 第3章 生活工学
> **節**: 3・2 健康工学

健康工学の研究を発展させていくために必要不可欠な基盤技術が、図3・1 健康への不安図3・2 余命に占める自立期間と要介護期間(1995年、一部改編)生体モニタリング技術と、時系列的に得られたデータから心身の不具合をいち早く検出する解析技術である。健康維持を目的とする日常生活のモニタリング技術においては、特に実現しなければならない計測のための重要な四大要素がある。(1)非侵襲、(2)非拘束、(3)無意識、(4)継続性である。非侵襲は言うまでもなく、測定により痛み、出血等の身体的な侵害を被らないことである。非拘束とは、測定のために身につけるセンサや記録・解析システムにより身体的活動の負荷が増えないことである。無意識とは、被測定者が測定されていることを意識することなく測定できることである。そして、継続性とは長期間にわたり、容易にモニタリングが続けられることである。近ごろ、マイクロエレクトロニクス等の発展により、ウェアラブルセンサとしての超小型軽量のシステムが増加し、生活モニタリングが容易になりつつある(12・1・2項参照)。また、無意識なモニタリング技術については、12・1・3項に詳細があるように、排便、排尿、入浴、睡眠などの生活に欠くことができない行動を利用して、日常生活行動の中で無理なく計測する技術が研究されている。今後、さらにナノテクノロジー等の微小化技術の進化が予想され、錠剤サイズのナノカメラや、痛みや出血を伴わないナノ注射針等の開発が進めば、健康に関わる臓器内の生理的データを含めて、無意識かつ長期にわたり安定して抽出できるセンサの出現が期待できる。最近の新たな生体モニタリング技術の動向としては、バイオテクノロジーの急速な進化に伴うバイオセンサの研究が盛んであることが挙げられる(6・2・5項参照)。例えば、血糖値センサとしてグルコースセンサの使用が検討されている。マイクロ化されたグルコースセンサは、従来の電流検知方式や吸光度方式の血糖値センサのように、測定のたびに採血針で皮膚を突き刺し、侵襲的に出血させることなく、数箇月の長期にわたり非侵襲的な血糖値モニタリングが可能とされている(12・1・2項参照)。
