# 3・2・4 高齢者の生活行動モニタリング技術

> **章**: 第3章 生活工学
> **節**: 3・2 健康工学

健康維持のためのモニタリングの対象として，従来から盛んであ
った心拍，心電図，血圧，呼吸，体温，体重，体脂肪等に加え，近
ごろでは，独居高齢者の生活行動モニタリング技術の研究が盛んで
ある（12･1･3項参照）．近年，増加の一途をたどっている独居高齢
者は，事故や病気などにより室内で動けなくなったとき，発見が遅
れれば死に至る危険性が高くなるので，できるだけリアルタイムで
身体的な異常を検出する必要がある．この異常を生活行動から検出
するには，当然のことながら平常の生活行動パターンを基盤データ
として記録しておかなければならない．しかし，日常生活のモニタ
リングは生活者のプライバシーを侵す可能性がある．したがって，
監視カメラのように生活行動をそのままに撮影してしまう視覚的な
測定手法は適切ではない．そこで，日常生活に必ず現れる動作を間
接的に反映する指標を検出する手法が広く研究されている（12･1･3
項参照）．例えば，朝起きれば，通常は顔を洗い，炊事をするので，
水道の蛇口を必ずひねる．そしてガスをつける．冷蔵庫を開ける．
お茶を飲むためにポットを押す．あるいは，部屋の扉を開け閉めす
る．夜寝るときには照明を消す．そこで，人にセンサを付けるので
はなく，人が動作の対象とするものにセンサを付け，その動きから
間接的に人の行動を予測する．もちろん，人そのものにセンサを取
付ける手法もある．例えば，人の姿勢の変化を詳細に検知したいと
きには，複数の加速度センサを腰部に付けることにより，人の立
位，座位，横臥位などが判定できる．
