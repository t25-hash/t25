# 3・2・5 睡眠に関わる研究:睡眠工学

> **章**: 第3章 生活工学
> **節**: 3・2 健康工学

食欲、性欲と並ぶ人の三大欲求の一つである睡眠は健康を維持するために重要な要素である。通常、人は睡眠に1日の生活時間の約4分の1以上をかけており、このことからも睡眠が生活に占める重要性は高い。従来は眠ってしまえば後は同じなどと、睡眠環境に十分な配慮を払わないことが多かったが、現在では睡眠科学の進歩により、睡眠の質的な側面の重要性が明らかになってきた。したがって、不眠症の対策にとどまらず、快適な睡眠の確保は生活工学の重9 ―19要なターゲットとなっている。わが国では睡眠障害に悩む人は成人人口の約20%といわれている。夜型社会、興奮やストレスの過多、日中における運動不足、時差などの要因があるが、特に高齢者では、寝つきが悪く、眠っても睡眠深度が浅く、目が覚めやすいことが問題となる。このような睡眠の質と量に関する問題を解決し、医薬品以外の方法で制御する研究を睡眠工学(sleep engineering)と呼び、健康工学の重要な分野である。睡眠の促進のため、ベッド、枕等の睡眠用具の研究に加え、寝室および寝具内の温熱環境、寝室の照明、音、におい環境に関する研究が行われている。また、福祉工学においても褥瘡(じょくそう)防止のためのベッドや車いすの研究が盛んである(9・9・2項参照)。そのほか、より直接的に生体に働きかける手法として、睡眠にも影響する生体のもつ日内周期を光刺激により積極的に統制して、睡眠に入りやすくする高照度光療法などがある。
