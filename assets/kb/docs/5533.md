# 9・3・2 印刷インキ（printing ink)

> **章**: 第9章 印刷機械
> **節**: 9・3 印刷資材

a.印刷インキの組成 印刷インキは色材である顔料とこれを
分散し流展性を与え，固着させる媒質（vehicle，ベヒクル）が主
成分である．これに乾燥性，流動性，皮膜特性などを調整する補助
剤が加わっている．色材はフレキソグラフィで特に透明性を要求さ
れるとき，あるいは布印刷など特殊な場合は染料が用いられるが，
一般的には顔料の型で使用される．

b.印刷インキの分類 印刷インキの分類方法にはさまざまな
ものがあり，(１)色別によるもの，(２)印刷方式によるもの，(３)
被印刷物の用途によるもの，(４)媒質の相違によるものなどがある
が，おもには次の7種に分類されている．
(１)平版インキ，(２)凸版インキ，(３)グラビアインキ，(４)フ
レキソインキ，(５)スクリーンインキ，(６)凹版インキ，(７)新聞
インキ．
c .特殊なインキ 印刷技術の進歩に伴い開発された特殊イン
キの例として，以下のようなものがある．
.紫外線硬化型インキ（UV-cure ink) 感光性樹脂と光
反応性希釈剤からなるベヒクルに，光増感剤を加えインキ化したも
のである．紫外線（UV）の照射により瞬時に硬化し，強い耐摩擦
性が得られるため，紙器印刷，金属印刷，ビジネスフォーム印刷な
どに使われている．
.蛍光インキ 蛍光顔料を使ったインキで，雑誌の表紙，
ポスタ，広告宣伝物などにおいて人目を引くために使われる．
.磁気インキ（magnetic ink) 磁性酸化鉄の粉末を使っ
たインキで，印刷された文字を磁気リーダで自動読取りするために
使用される．小切手，通帳等の証券類の記号，番号の印刷に使われ
る．
