# 3・3・1 感性工学への社会的要請

> **章**: 第3章 生活工学
> **節**: 3・3 感性工学

最近,生活に関連して感性工学(kansei engineering)が言及されることが多い.われわれは20世紀において,過去にない著しい物質文明の洗礼を受け,次から次へと新しい機能をもつ製品の洪水の中で暮らしてきた.その中で,新しい製品を追い求めるあまり,長い間,履き心地の悪い靴を履き,風合いが悪く体型に合わない服を身につけ,座り心地の悪い椅子に座り,寝心地の悪いベッドで眠り,居心地の悪い住宅に住み,住み心地の悪い都市で暮らすことに我慢してきた.多くの製品や環境の設計は,人の感性に十分に配慮しない設計者によってなされてきたといっても過言ではない.言葉を換えれば,新機能,効率化,スピード化という利便性のために,感性を犠牲にしてきたといえる.しかし,人は高度成長期の終末において,ようやくわが身を振り返り,新機能の追求よりもむしろ生活における快適性の確保や自己実現に生活価値を移行させつつあるといえる.内閣府の国民生活に関する世論調査によると,生活における心の豊かさを重視する人の割合は,1977年には物の豊かさを重視する人の割合とほぼ同じ40%程度であったが,その後,前者を重視する割合が増加する一方で後者を重視する人が減少を続け,2002年の調査では,心の豊かさの重視が約61%,物の豊かさの重視が約27%と,明確な心の豊かさ重視の傾向を示すようになっている(図3・3).このような国民の生活価値の変化が感性工学発展の基盤にある.
