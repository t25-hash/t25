# 9・3・3 ゴム，ブランケット

> **章**: 第9章 印刷機械
> **節**: 9・3 印刷資材

印刷を行ううえで，ゴムやプラスチックなどの高分子物質はなく
てはならない重要な材料である．代表的なものとして，ゴムローラ
とブランケットが挙げられる．
a.ゴム（rubber ) 平版や凸版印刷では，版面にインキを薄
く均一に供給するため，数多くのゴムローラが使われている．イン
キや湿し水および洗浄剤への耐性が必要であり，またインキの受
理・転移性の良いことが要求されニトリルゴムなどの耐油性合成ゴ
ムが多く使用される．硬さはJIS A型硬度で20～40度が多い．ま
た，グラビアや凸版印刷の圧胴には，ムラのない印刷を行うために
ゴムがライニングされている．グラビア圧胴では硬度70～90度の
ものが使われる．
b.ブランケット（blanket ) オフセット印刷では，版面のイ
ンキはゴムブランケットに転写された後，紙に転移される．ブラン
ケットはゴム層と補強布とを交互に重ね合わせて作られた一種のゴ
ム引き布であり，印刷機械のブランケット胴に装着される．その厚
みは約2mmであり，硬度はJIS A型硬度75～85度である．
