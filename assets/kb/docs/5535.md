# 3・3・2 感性工学の歴史

> **章**: 第3章 生活工学
> **節**: 3・3 感性工学

感性工学は情緒工学(emotion engineering)研究から誕生した.情緒工学研究は,1969年に日本情緒工学研究会の発足とともに始まり,1972年には日本人間工学会の機関誌「人間工学」において情緒工学の特集が組まれている.その後,1988年まで,情緒工学のキーワードを使って多くの製品の感性的な側面を重視する研究が行われてきた.感性工学という言葉の登場は,1986年に自動車メーカであるマツダ(株)の山本健一会長がミシガン大学での特別講演で「車の文化論」を展開した際に使用したのが最初であるとさ図3・3 生活における心の豊かさ,物の豊かさ9 ―20医療・福祉・バイオ機器れている.その後,1988年の第10回国際人間工学会で,長町が1970年以降に行ってきた情緒工学研究を感性工学研究と改めて発表することにより,感性工学の世界的な知名度が高まったといえる.長町によると,感性工学とは,人間がもっている願望としてのイメージや感性を物理的なデザイン要素に翻訳し,具体的に設計する技術とされている.感性工学はその後,生活において物質的なものに代わって精神的なものに価値を見いだす傾向,生活の個性化,快適さの追求の風潮およびそれらと並行して進展してきた情報化技術の生活への関与の増大に伴い,産業界を含めて幅広く研究が行われるようになった.その結果,大学にも感性工学関連学科が新設されるようになっている.例えば,1995年の信州大学繊維学部感性工学科に続き,山口大学工学部感性デザイン工学科,広島国際大学の感性情報工学科などが誕生している.また,1998年には日本感性工学会が誕生し,この分野の研究が加速している.
