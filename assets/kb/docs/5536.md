# 3・3・3 感性的な感性の定義

> **章**: 第3章 生活工学
> **節**: 3・3 感性工学

そもそも感性という言葉は定義が難しい。2000年の日本感性工学会主催のシンポジウムでも、感性工学とは何かということがいまさらながら話題となった。そういう意味で、俳諧で重要とされるわびやさびと同様、まさに感性的な理解が要求される言葉である。英語においても、日本語のままにkansei engineering が使用されるようになった理由もここにある。多様な定義があるが、少なくとも感性的な反応とは、入力された個々の感覚情報を理性的、論理的に検討したり、意識的に思考・判断を積み重ねたりした結果としての反応ではなく、もろもろの入力刺激に対して直感的に脳(心)に現れる総合的反応といえるのではないか。したがって、個々の感覚器官の単なる感受性だけではなく、過去の生活の中で学習され、混然一体として心の奥底深く形成・蓄積されてきたものの見方、態度、さらには文化を含めた価値観を枠組みとする五感の交互作用の結果としての、ほとんど無意識的な反応の総体を指すといえる。したがって、このようなあいまいな特性をもつ感性を工学的に有効に処理するためには、今後、感性の定量化を可能とするモデル化の研究が重要となる。
