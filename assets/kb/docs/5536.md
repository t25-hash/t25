c .オフセット印刷機における自動化機器
.インキ量自動プリセットシステム（automatic ink preset-
ting system) 熱練と時間を要するオフセット印刷のインキ供
給量の設定を自動的に行うシステムであり，(１)刷版絵柄面積率
計，(２)インキプリセット入力装置，(３)インキ供給量リモコン調
整装置から構成される（図9･11)．
刷版絵柄面積率計は，刷版の画線部の量を読取り，ディジタル信
号化する．これをインキプリセット入力装置に送ることにより，イ
ンキ供給量を決定するインキキーの開度が自動的に最適状態に設定
され，インキ供給量のプリセットが完了する．最近では，製版デー
タから刷版の画線部データを演算し，印刷機のインキプリセット入
力装置にネットワークを介して直接送られるシステムが開発され，
刷版絵柄面積率計は不要になりつつある．
.ブランケットウォッシャ（blanket washer ) オフセッ
ト印刷では，一定印刷枚数ごとにブランケットを洗浄し，紙粉やイ
ンキカスを除去する必要がある．これを自動化するためにブランケ
ット洗浄装置が開発された．
ブランケット胴に溶剤を吹付け回転するブラシでこすり取る方式

.カットオフコントロール装置（cut - oﬀcontroller )
印刷された紙を折り機で断裁（カットオフ）するとき，紙面の天地
方向の断裁位置は絵柄に対し所定の位置となるように調整する必要
がある．これを自動的に調整するもので，特殊マークをスキャニン
グヘッドで検知し，基準パルスと比較し，誤差をなくするように印
刷部から折り部の断裁位置までの紙経路長さを変える．具体的に
は，図9･12のようにU字型に紙を巻付けたコンペンセータローラ
をねじの回転により移動させて自動修正する．
.エッジガイドコントロール装置（web edge controller )
印刷中，巻取紙の巻きの不ぞろいなどで紙が左右にずれることがあ
り，これを自動的に検出・修正するものである．修正は2本の平行
ローラを油圧シリンダまたはモータでひねることによって行われ
る．紙端位置の検出は，光電管方式とエア方式とがある．
