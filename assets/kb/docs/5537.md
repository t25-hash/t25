# 9・4・3 グラビア印刷機（gravure press )

> **章**: 第9章 印刷機械
> **節**: 9・4 印刷機械

a.枚葉グラビア印刷機（sheet -fed gravure press ) 枚葉グラ
ビア印刷機の版は図9･14のように版胴に巻付けて固定する．印刷
された紙は紙取胴に移り，排紙爪で乾燥装置に運ばれた後，排紙部
に積み上げられる．
b.グラビア輪転機（rotary gravure press ) 巻取式のグラビ
ア輪転機の場合は，版胴自体に製版される．印刷部は版胴，圧胴，
インキ供給装置，ドクタ，乾燥装置から構成される．印圧を加える
機構としては，単に1本の圧胴を用いる方式のほかに，バックアッ
プロールを用いる方式もある（図9･15）．
インキを版に供給する方式には，ドブ漬け方式，着けローラ方
式，カスケード方式などがある（図9･16）．
乾燥装置は，蒸気，電熱による加熱ドラム式，赤外線ランプ式も
あるが，火災に対する安全性と乾燥効率の点から現在では，ほとん
どが電気，ガス，蒸気などにより加熱された熱風を用いる方式とな
っている．
