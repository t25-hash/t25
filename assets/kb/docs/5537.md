# 3・3・4 感性工学の対象の広がり

> **章**: 第3章 生活工学
> **節**: 3・3 感性工学

感性工学の対象は、初めはどちらかといえば、生活で使用する個々の製品の快適性、例えば、衣服、家電、住宅、自動車などのものづくりが主たる対象であった。そこでは、製品を使用する個人の多くが満足するような解を企業が出すことに、感性工学的手法が広く使用されている。しかし、近年になって、人はより広い生活空間に対しても快適性を求めるようになってきた。例えば、歩くだけで五感に楽しさを感じさせる街並みの設計を目指す景観、音風景、香り空間、あるいは街路の歩き心地の創出のための工学的取組みなどが増えてきた。街には年齢、身体条件、あるいは文化的基盤の異なる多くの人が住んでいるので、その街で表現する感性のコンセプト作りが難しい。感性の合わない製品に対しては、それを選択しない、あるいはオプションをつけるなどで対応できるが、自分が住む街が感性に合わなくなったからといって転居することは容易ではない。さらに街には、そこに在住する人とそこへの訪問者とがあり、両者の街並みに求めるものが異なる場合がしばしばある。これらの複合的な事柄を総合して、機能的、快適、個性的なまちづくりをするために、今後ますます、感性工学的取組みが必要である。
