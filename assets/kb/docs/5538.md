# 9・4・4 スクリーン印刷機（screen process press )

> **章**: 第9章 印刷機械
> **節**: 9・4 印刷機械

スクリーン印刷は，スキージといわれるゴムのヘラを使って版の置いた被印刷物にインキを転移することによって行われる．
ずつ印刷する枚葉印刷方式と，紙，布，プラスチックフィルムなど図9･17に枚葉スクリーン平面印刷機の機構を示す．
台の上に被き，スキージで版を押付けながら左から右へ動かして印刷する．
刷に用いられる．
の腰のない素材を印刷する巻取印刷方式がある．
印刷物を位置決めして置き，その上にスクリーン枠を少し離して置b.曲面印刷機 プラスチックや金属などの円筒容器側面の印で，被印刷体は版面に上って密着し，回転させながら印刷する．
けた円筒スクリーンの内側にスキージを置き，被印刷物を円筒スクなる．
