# 9・4・4 スクリーン印刷機（screen process press )

> **章**: 第9章 印刷機械
> **節**: 9・4 印刷機械

スクリーン印刷は，スキージといわれるゴムのヘラを使って版の
上にのせたインキをかき，版にあけられた微小孔を通して版の下に
置いた被印刷物にインキを転移することによって行われる．
a.平面印刷機 ポスタ，プラスチック板，金属板などを1枚
ずつ印刷する枚葉印刷方式と，紙，布，プラスチックフィルムなど
の腰のない素材を印刷する巻取印刷方式がある．
図9･17に枚葉スクリーン平面印刷機の機構を示す．台の上に被
印刷物を位置決めして置き，その上にスクリーン枠を少し離して置
き，スキージで版を押付けながら左から右へ動かして印刷する．
b.曲面印刷機 プラスチックや金属などの円筒容器側面の印
刷に用いられる．スキージが固定しスクリーン枠が移動するもの
で，被印刷体は版面に上って密着し，回転させながら印刷する．
c.ロータリ印刷機 エレクトロフォーミングにより小孔をあ
けた円筒スクリーンの内側にスキージを置き，被印刷物を円筒スク
リーンと圧胴の間に通して印刷するもので，連続高速印刷が可能と
なる．
