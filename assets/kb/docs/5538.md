# 3・4・1 遊びと癒しの工学

> **章**: 第3章 生活工学
> **節**: 3・4 情緒工学

情緒工学(emotion engineering)の起源は,感性工学のところで触れたように1970年ごろに始まる.ただし,当時の情緒工学の研究内容は,後に感性工学として分かれた,製品の感性的デザインに関連するものが中心であった.情緒あるいは情動は,心理学辞典によると「自己の心理的,身体的な存在が脅かされたり,促進されたりすると感じたときに急激に生じる心理的,身体的な激動,混乱図3・4 仕事や職場生活に関することで強い不安,悩み,ストレス等を感じる労働者の割合状態」と定義されるので,当時の研究対象を中心とする研究分野が情緒工学から感性工学へと名称を変更したのは妥当なことといえる.他方,残された情緒工学は,情緒の定義によりふさわしい研究対象へと向かっていった.すなわち,遊びや癒し等によって感情の高揚,沈静,平安を図る工学的手法の研究である.すでに述べたように,20世紀末において,高度経済社会が破綻していく中で,一方では人口の高齢化が進み,他方では高度情報化が急速に進展するという未曾有の社会的激変の中で,強い不安やストレスを体験する機会が多くなった.労働省(当時)の「労働者健康状況調査報告」によると,仕事や職場生活に関することでストレス等を感じる労働者の割合は年々増加しており,1982年の51%に比べ,1997年では約63%に達している(図3・4).人は快適な生活を送るためには,睡眠や食事だけではなく,感情のバランスを調整しなければならない時代になったといえる.これらの精神的ストレスや不安に対する悩みを解決したいとする強い社会的要請に対して,さまざまな遊びや癒しの研究が盛んとなった.遊びとしては,東京ディズニーランドやユニバーサル・スタジオ・ジャパンなどの情緒工学を巧みに生かした巨大システムから,家庭で容易に体験できる各種パソコンゲームまで,開発が盛んである.これらの遊びでは,非日常性の体験,すなわち日常生活では経験しにくい喜び,悲しみ,驚き,恐怖,あるいは攻撃・破壊感や達成感などの強い感情を体験することにより,生活における精神的な彩りを豊かにする.今後,感覚や運動に関するバーチャルリアリティ技術の進化に伴い,この分野での新展開が期待される.他方,癒しとしては,視覚・聴覚・嗅覚・触覚等の五感に働きかけストレス解消を目指すリラクゼーションやセラピーの研究が盛んである.また,近年新たに出現してきた癒しとしてペットロボット(pet robot)がある.従来,ロボットは実用的な作業をするためにさまざまな工夫がされてきたが,1990年代になってソニー(株)のアイボの誕生とともに,人の情緒的側面を支援する新たなロボットの概念が作られた.現在では,多くのタイプのペットロボットがある(アイボは2006年に生産中止).ペットロボットは通常,動物の形をしており,犬や猫などの代わりにペットとしての機能を果たす.しかし,同様な機能を果たすものとして,従来からあった人形や動物のおもちゃとは異なり,人間の声に反応して動いたり,声を出したり,あるいは表情を作ることができることにより,情動的反応をする.また,学習機能を備えているものが多く,飼い主の個性によって異なる成長をする.これらのペットロボットは,孤独な高齢者あるいは入院中の子供たちの精神安定にも効果があるといわれている.
