# 3・4・2 情報化社会における情緒工学

> **章**: 第3章 生活工学
> **節**: 3・4 情緒工学

人間の生活において、他者とのコミュニケーションはたいへん重要な意味をもつ。過去の歴史においては、多くのコミュニケーションは対面でなされることにより、好むと好まざるにかかわらず、そこには話し手の情緒的な情報が必ず介在した。例えば、多くの表情、声音、身振りは話し手の怒り、憎しみ、悲しみ、喜び、慈しみなどの情緒的な態度を相手に伝えることになる。そもそも、われわれが日常のコミュニケーションで多くの場合知りたいのは、百科事典のような純粋な知識ではなく、話し手の話す内容に対する態度、価値観である。それは通常、情緒的な要素として、表情や身振りなどの中で表現される。したがって、感情は社会的信号としての機能を強くもっているといえる。ただし、情緒的な表現には文化による表示・解読規則があり、それを理解していないと話し手の気持ちを正確に読み取ることができない。しかし、高度情報化が顕著に進展しつつある現在、コミュニケーションは対面的に果たされるのではなく、電子媒体を通じて文字により伝達されるケースが増加した。その典型例が電子メールである。電子メールでは、多くの人は自分の感情を、自然と表情に出てくるようにはうまく文字で表現することができないので、会話における互いの情緒的共有が困難になる。これらの問題を解決することを目指して、表情や身振りなどの感情の表現と理解に関するノンバーバルコミュニケーションなどの情緒工学的な研究がなされている。この研究が進めば、電子メールなどで、表情や身振りなどを絵記号により表現することにより、コミュニケーションにおける情緒の共有の増進が期待できる。
