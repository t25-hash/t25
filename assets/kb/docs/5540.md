# 3・5・1 スポーツ工学の体系

> **章**: 第3章 生活工学
> **節**: 3・5 スポーツ工学

スポーツには,記録に挑むような競技スポーツから,健康づくり,気晴らしやストレス解消目的などのレジャースポーツまで多様なものが含まれる.これらを総合した現在のスポーツは一つの大きな産業となっており,プロ野球やサッカーJ リーグなどのプロスポスポーツ年注:2001/2000年欄の単位は%9 ―21ーツから,ゴルフ,テニスなどに代表されるアマチュアスポーツまで,スポーツ産業の年間売上げは国内で約4兆7000億円とされている.この内訳としては,スポーツ施設使用料や各種スポーツクラブの収入が約50%を占め,次いで各種スポーツ用品が45%となっている(表3・1).また,オリンピックやサッカーワールドカップの日本向け放映権料はそれぞれ約39億円(第18回冬季オリンピック長野大会),5億5000万円(第16回サッカーワールドカップフランス大会)に達する.スポーツをよりよくするための工学的アプローチとなるのがスポーツ工学(sports engineering)であり,その対象は,(1)スポーツのための用具,設備,施設,(2)運動能力の計測評価機器,(3)トレーニング機器や教示デバイスなどである.表3・1によると,スポーツ産業全体の中で,スポーツ工学関連の売上げは総額2兆1400億円である.内訳としては,ゴルフ用品が約22%といちばん大きく,以下,釣り具,ウィンタースポーツ用品(それぞれ約11%)と続いている.スポーツ工学に関連する工学分野は幅広く,おもに以下のような分野が関係している.まず用具,設備,施設については,構造力学,流体力学,振動学や材料科学が生かされる.特に用具の進歩は,新しい材料の登場がきっかけとなる場合が多い.次に計測評価については,筋電位といった生体信号の計測技術や,画像分析点の座標算出技術,血液や呼気ガスなどの化学分析技術が必要となる.最後にトレーニング機器では,トーイングなどの運動アシスト機器や,等速性筋力負荷装置のためのサーボ制御技術などが求められる.また教示デバイスでは,運動時のダイナミクスを計算し提示するためのシミュレーション技術が必要になるであろう.表3・1 スポーツ市場の推移(単位:億円)2001/球技スポーツ用品6890 ▲1.0ゴルフ用品4630 ▲1.3テニス用品0.0卓球・バドミントン用品0.0野球・ソフトボール用品990 ▲1.0球技ボール用品0.0山岳・海洋性スポーツ用品9640 10020 10060 10260 10520 104507840 ▲4.9スキー・スケート・スノーボード用品2330 ▲2.5登山・キャンプ用品1540 ▲6.1釣り具2420 ▲6.9海水中用品1550 ▲3.7その他のスポーツ用品0.6スポーツ自転車2.6その他のスポーツ用品1730 ▲1.1スポーツウエア等3420 ▲2.3トレーニング・競技ウエア2020 ▲2.4スポーツシューズ1400 ▲2.1スポーツ施設・スクール21550 24780 27580 30650 32960 31850 30560 30260 29660 29350 28160 27150 25670 24380 ▲5.0ゴルフ場12090 13730 15600 17750 19610 18430 18060 17880 17630 17580 16840 16320 15010 14000 ▲6.7ゴルフ練習場1840 ▲8.0ボウリング場1150 ▲1.7テニスクラブ・スクール4.2スイミングプール2770 ▲3.1アイススケート場0.0フィットネスクラブ0.0スキー場(索道収入)990 ▲2.9スポーツ観戦料4.9合計42150 47280 52140 57380 60530 59170 57620 57510 56970 55760 53300 51170 48860 47100 ▲3.69 ―22医療・福祉・バイオ機器
