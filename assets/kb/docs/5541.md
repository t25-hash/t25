# 3・5・2 スポーツ工学の歴史

> **章**: 第3章 生活工学
> **節**: 3・5 スポーツ工学

スポーツ工学の歴史は,スポーツ用具改良の歴史である.用具をより軽くより強くするために,その素材は天然のものから金属を経て,CFRP(carbon fiber reinforced plastics,炭素繊維強化プラスチック)やGFRP(glass fiber reinforced plastics,ガラス繊維強化プラスチック)等をおもな素材とする先進複合材料に移行してきた.またシューズやスポーツサーフェスには,優れた衝撃緩衝性や柔軟性をもつ高分子発泡材が積極的に取り入れられている.このように機械的特性に優れた素材の活用は,プレーヤーのパフォーマンスを向上させ,記録更新の大きなきっかけとなってきた.用具そのものの機械的特性を向上させることによる改良法のほかに,最近では人体の特性を考慮した用具開発について検討がなされてきている.例えば長野オリンピックで有名となったスラップスケートは,ブレードと靴が踵部分で脱着する構造により下肢の筋負担を減らし,多くの記録を塗り替えた.また,シドニーオリンピックで多用されたフルボデースーツタイプの競泳用水着は,筋の形状や動きを考慮して,水から受ける形状抵抗を削減するように意図されている.今後は,人間にとって使いやすい用具の特性についての研究が望まれる.計測や評価のうち,三次元の動作解析について,1980年代には,半導体位置検出器(PSD : position sensitive device)を利用したポジションセンサシステムによって,輝点の空間座標を非接触で計測することができるようになり,大量のデータ処理が可能となった.しかし,従来からあるビデオや16mmフィルムによる画像解析も,身体に分析用マークを付けることができない競技会での動作解析や,屋外あるいは水中といった多様な環境下での分析には必要不可欠である.なお現在までに,赤外線反射マーカを用いた自動追尾方式や,画像計算処理方式の各種計測装置が商品化されている.また,アイソキネティックマシン(Cybex machine)は,走る,跳ぶ,蹴る等のスポーツ動作において非常に重要となる膝関節について,その等速運動時の筋力を測定・評価するのに非常に有用であると同時に,リハビリテーションやトレーニングにも利用できるため,多くの施設に設置されている.
