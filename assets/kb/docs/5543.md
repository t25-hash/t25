# 3・5・4 今後の課題

> **章**: 第3章 生活工学
> **節**: 3・5 スポーツ工学

スポーツ産業は、1992年をピークに若干縮小傾向にはあるが、現在も高い水準を維持しており、そのためスポーツ工学分野への要求もますます高まることが予想される。一方で、国内のスポーツ施設は充実してきており、ゴルフ場やスキー場は過剰状態にある。また、少子高齢化の影響で若年者の比率が低下するともに、スポーツになじみのない若者が増加していることや、携帯電話によるコミュニケーションやゲームに集中してスポーツに興味をもたない若者が増えていることなど、懸念すべき点は多い。今後は、中高年齢者の健康志向に合致したスポーツの導入や、技術レベル向上を促進させるような用具の開発が求められる。そして、スポーツから遠ざかっている若者の興味を引きつけるようなゲーム性のあるスポーツの創成を図る必要があるだろう。文献(1) 座談会人間生活工学の10年、人間生活工学、1 (2000)、2.(2) 持丸正明・ほか2名、計測自動制御学会第19回センシングフォーラム予稿集、(2002)、47.(3) 健康・体力づくり事業財団、 健康づくりに関する意識調査、(1996)、23.(4) 厚生省監修、平成12年版厚生白書、(2000)、62、ぎょうせい。(5) 健康日本21ホームページ、http://www.kenkounippon21.gr.jp/(6) 内閣府、国民生活に関する世論調査、 (2002)、 内閣府大臣官房政府広報室。(7) 特集「情緒工学」、人間工学、8-5(1972)。(8) 長町三生・ほか2名、人間工学、10-4(1974)、121.(9) 長町三生、電子情報通信学会誌、71 (1988)、245.(10) 長町三生、 感性工学―感性をデザインに生かすテクノロジー、(1989)、25、海文堂。(11) 鈴木邁監修(大澤光編)、感性工学と情報社会、(2000)、森北出版。(12) 労働省、労働者健康状況調査報告、 (1997)、 労働大臣官房政策調査部。(13) Campos,J.J.、ほか2名、Develop.
