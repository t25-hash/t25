# 4・2・1 バイタルサイン計測

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

生体現象測定記録装置は生体から発する情報を計測するものと，
生体に働きかけを行いその応答を計測するものとに分けられる．こ
こでは生命現象の直接情報であるバイタルサイン（vital sign）計
測について述べる．バイタルサインとは，ヒトの生命（vital）の
基本的な兆候（signs）のことだが，通常，脈拍・呼吸・体温・血
圧の四者をバイタルサインと呼んでいる．これらの四つの兆候こ
そ，ヒトが生きているという身体的状況を的確に反映しているもの
である．ただし，急患などでは意識の有無，心臓の拍動確認が最も
重要な兆候になるので，救急では意識や心音聴取などもバイタルサ
インに含めることもある．
バイタルサインの中でも最も重要なパラメータは脈拍（pulse）
であり，心臓が拍動していることを示す．心臓の周期的な収縮によ
って，血液が動脈に駆出されることにより，脈拍として触れる．簡
単には手首の動脈に触れる方法があるが，計測し記録するために
は，手足または胸部に電極を装着して心電図を得る方法がある．近
年，指先にパルスオキシメータといわれる光電センサを装着するこ
とで動脈血液の拍動分をとらえ，さらには血液中のヘモグロビンの
酸素との結合により光の吸光特性が変化することを利用して，動脈
血液の酸素飽和度も無侵襲に連続的に計測することのできる方法も
利用されている．
要な二炭化炭素を排出するために肺に空気を出し入れすることであ
る．呼吸を計測するには，胸郭の動きを胸部に装着した電極間の電
気的インピーダンスの変化でとらえる方法のほか，鼻からの空気の
出入りを鼻孔に温度センサを装着して温度変化としてとらえる方法
もある．

血圧（blood pressure）は心臓の力と血管の緊張性によって保た
とができる．血管に直接的にカテーテルといわれるチューブを装着
して，チューブの根元に設置した圧力センサで測定する方法があ
る．一方，簡便にはカフといわれる袋状の腕帯を上腕に巻き付け

4･2･2項以降では，バイタルサインを計測する装置として心電
計・心音計・血圧計・血流計について説明する．またバイタルサイ
ンの大もとをつかさどる脳の活動と，身体の動きである筋肉の活動
を計測する脳波計と筋電計についても説明する．また，重症患者で
はバイタルサインの何種類かを同時に連続的にベッドサイドモニタ
により計測するが，これらの装置やモニタではコンピュータを用い
たディジタル処理により雑音対策・体動対策などが高度化し，表示
や記録の方法もよりビジュアルになってきている．
