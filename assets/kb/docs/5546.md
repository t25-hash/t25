# 4・2・1 バイタルサイン計測

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

生体現象測定記録装置は生体から発する情報を計測するものと、生体に働きかけを行いその応答を計測するものとに分けられる。ここでは生命現象の直接情報であるバイタルサイン(vital sign)計測について述べる。バイタルサインとは、ヒトの生命(vital)の基本的な兆候(signs)のことだが、通常、脈拍・呼吸・体温・血圧の四者をバイタルサインと呼んでいる。これらの四つの兆候こそ、ヒトが生きているという身体的状況を的確に反映しているものである。ただし、急患などでは意識の有無、心臓の拍動確認が最も重要な兆候になるので、救急では意識や心音聴取などもバイタルサインに含めることもある。バイタルサインの中でも最も重要なパラメータは脈拍(pulse)であり、心臓が拍動していることを示す。心臓の周期的な収縮によって、血液が動脈に駆出されることにより、脈拍として触れる。簡単には手首の動脈に触れる方法があるが、計測し記録するためには、手足または胸部に電極を装着して心電図を得る方法がある。近年、指先にパルスオキシメータといわれる光電センサを装着することで動脈血液の拍動分をとらえ、さらには血液中のヘモグロビンの酸素との結合により光の吸光特性が変化することを利用して、動脈血液の酸素飽和度も無侵襲に連続的に計測することのできる方法も利用されている。呼吸(breath)とは時々刻々に身体に必要量の酸素を与え、不要な二炭化炭素を排出するために肺に空気を出し入れすることである。呼吸を計測するには、胸郭の動きを胸部に装着した電極間の電気的インピーダンスの変化でとらえる方法のほか、鼻からの空気の出入りを鼻孔に温度センサを装着して温度変化としてとらえる方法もある。体温(body temperature)は一般的には腋(わき)の下に検温器を入れて測るが、連続的には直腸などに温度センサを挿入留置して計測する。血圧(blood pressure)は心臓の力と血管の緊張性によって保たれているので、血圧を測ることによって心臓や血管の機能を知ることができる。血管に直接的にカテーテルといわれるチューブを装着して、チューブの根元に設置した圧力センサで測定する方法がある。一方、簡便にはカフといわれる袋状の腕帯を上腕に巻き付けて、このカフに伝達されるカフ内の脈圧の大きさから上腕動脈に伝わってくる圧力を推定してとらえ、計測する方法がある。
