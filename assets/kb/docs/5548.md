# 4・2・2 心電計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

バイタルサインの脈拍とは,心臓が血液を体の中に送り出した拍動のことである.心臓はヒトが生命活動を行っていくために必要な酸素や栄養分を血液を使って体の各部の細胞に供給したり,細胞の新陳代謝により生じた二酸化炭素や不要物質を運び去るために,生きている限り休むことなく動き続けている.心臓のポンプ的な動きは,自動車のエンジンに例えられる.自動車ではエンジンを動かすために,スパークプラグを間欠的に点火させて,ガソリンと空気の混合気を爆発させることによって大きな力を生み出している.心臓図4・1 心臓の模式図図4・2 心電計の構成例図4・3 心電図の例では図4・1に示す心臓の右心房近くにある洞結節からリズミカルに微弱な電気的興奮を作り出し,この興奮をトリガとして心臓全体が収縮を起こし,血液を全身に送り出す.心電図(electro cardiogram)とは,図4・2に示すような構成をした心電計(ECG : electro cardiograph)によって,心臓の電気的興奮を胸部や手足に装着した電極により電圧として取込み,増幅し図4・3のように記録するものである.心電図は誘導電極,心電図信号バッファ部,誘導切換え,増幅部を介し,データ処理装置によってドリフト補正をしたり,不整脈などを解析して記録する.心電図は1mV 程度の微弱な信号なので,電極やリード線から発生する雑音や外部からの交流障害を防ぐために,バッファ部分は高性能な差動増幅器が用いられている.心電計には次のような種類がある(図4・4).(1) 一要素心電計および多要素心電計心電図は体表面に装着された数個の電極から2電極を選択誘導して記録される.誘導を切換え,一誘導ずつ記録していくものが一要素心電計で,いくつかの誘導を同時に記録するものが多要素心電計である.(2) ホルタ心電計心疾患や循環器系の異常が疑われる場合に,日常生活の仕事や睡図4・4 心電計の外観例図4・5 心音計の構成例眠中に心電図がどのように変化するかをチェックするためにホルタ心電計(長時間の携帯型記録装置)が利用される.現在では非常に小型化され,大容量のデータも半導体メモリに記憶されるものになっている.(3) ストレステスト心電計軽度の心臓障害では安静時の心電図では異常が現れず,心臓に負荷を与えたときに異常が発生することがある.このようにストレステストを行いながら心電図を記録するために,体動による雑音除去手法が強化されたストレステスト心電計が用いられる.(4) 解析機能付き心電計心電図に現れる異常な波形の不整脈解析(arrhythmia analy-sis)を自動的に行う機能を付けた心電計のことであり,コンピュータの発達に伴って最近の心電計の6割以上に搭載されている.
