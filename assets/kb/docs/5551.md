# 4・2・5 筋電計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

腕や足を動かすためには筋肉の収縮、弛緩が行われている。これらの筋が神経からの興奮を受けて収縮する際、活動電位が生じる。この電位を細胞外から導出し増幅記録する装置を筋電計(EMG :electromyograph)といい、記録された波形を筋電図(electro-myogram)という。特に筋電図とは一般に骨格筋の活動電位を記録したものをいい、消化器、血管、膀胱、心臓などを構成している平滑筋の活動電位とは区別している。図4・8で示すように筋電計は筋からの活動電位を針電極などで導出し、増幅器を通してディスプレイに表示される波形やスピーカからの音で観察し記録する。最近では音、光、電気刺激を与え、それによって誘発される筋電図や、誘発電位を記録する機能を搭載した筋電図・誘発電位検査装置が実用化されている。これらには、誘発電位とノイズを分離する9 ―26医療・福祉・バイオ機器図4・8 筋電計のブロック図図4・9観血式血圧計構造ための同期加算回路および記憶回路が組込まれている。
