# 4・2・7 血流計

> **章**: 第4章 検査・診断機器
> **節**: 4・2 生体現象測定記録装置

血流測定は心臓のポンプ機能のうち極めて重要な血液の流量(流速と管路断面積の積)の測定を血流計(blood flow meter)を用いて行う。ただし、血流計は通常露出された血管にプローブを装着する必要があるので侵襲的であり、臨床的にはあまり使用されない。実用化されている血流計には電磁血流計と超音波血流計がある。a. 電磁血流計(electromagnetic blood flow meter)図4・11に原理を示す。このプローブを血管に引っかけて測定する。励磁コイルに矩形波状の電流を印加する。コイル内部にはコの字形に鉄心図4・12 超音波ドップラー血流計原理図4・13 超音波トランジットタイム血流計原理があり、コイル電流により発生した磁場を血管に直角に導き出す。ここに電気的に導体である血流が流れると、磁場との相互作用で起電力が誘起されるので、これを電極により検出する。この起電力は血流量に比例して発生し、以下の式で求められる。=ここで、:起電力、:磁場の強さ、:血管外径、:血流速度。この計測法は精度が高いが、侵襲的に血管にプローブを直角に装着させなければならず、各血管径に合わせて何種類かを用意しなければならない。b. 超音波ドップラー血流計図4・12に示すように速度で移動している血流に超音波を照射し、その反射波を受信すると、血流速度によりドップラー効果を受け周波数が変化する。したがって、この周波数の変化を計測すれば血流量を求めることができる。センサを小型化できるが、装着状態により安定性に影響を受ける。c. 超音波トランジットタイム血流計図4・13に示すように、超音波が血管を透過する時間(トランジットタイム)は血流(血球の速度)により変化する。血管とプローブの径が一致していなくても定量測定できるが、プローブが大きくなるので適用部位が限られる。
