a.点字ピンディスプレイと携帯用点字入出力機器 PCの操
作内容およびデータを点字として確認できることは，点字使用者に
とって重要な機能であるといえる．しかし，点字ピンディスプレイ
（braille pin display unit ）で完全に確認が行えるのはほぼデータ部
分だけである．音声確認の代わりに点字ピンディスプレイだけで
PCを操作できる状態とはほど遠いのが現状であるが，とりわけ点
字が読める視覚障害者で後に耳が不自由になった盲ろう者にとって
は，不可欠といえる重要な出力形態である．なお，テキストファイ
ルを自動点訳して点字ピンディスプレイ等で読むことは広く行われ
ている．
点字ピンディスプレイはピンの駆動部を含めて原価が高いため，
マス数によって用途が分かれていることが多い．例えば，全国の点
字図書館等ネットワークであるないーぶネット（9･3･5･ e項参照）
では，標準点字器のマス数と同じ32マスが標準とされているため，
行情報や制御コードなどを表示する部分と合わせて40マスの点字
ピンディスプレイが一つの標準品となっている．
22マスや18マスなどとマス数が少ない点字ピンディスプレイの
小型機種は，PCの端末としてだけではなく，単体で携帯用点字入
出力機器として手軽に使用できる機能をもつようになってきてい
る．点字キーで入力でき，メモリに記録したデータを点字ピンディ
スプレイや，機種によっては音声でも確認できる小型の機器が，携
帯用の点字メモ機や音声電子手帳として利用されている．また，
PDA（personal data assistance）ともいえる幅広い機能を付加す
る機種も登場しており，点字使用者のニーズに適合した装置として
注目されている．
b.点字プリンタ 点字データを確実に読むには，点字プリン
トされたハードコピーが最もよい．現在の点字プリンタ（braille
printer ）は，厚みのある点字用紙に機械的に凹凸をつけていくタ
イプのため，紙の固定が重要で，両サイドの連続穴によって紙が固
定されるトラックフィードタイプが圧倒的に多く，単票紙に印字す
るタイプはわずかである．また，動作方式では，点字の1マスごと
に打っていくシリアル方式と，1行の上位点・中位点・下位点の順
に高速で打っていくラインプリンタがある．なお，日本式の点字を
中心としたタイプはマス間がやや狭く，欧米型のものはマス間がや
や広いが行間は狭い．増加している中途失明者用に，マス間・行間
ともに広げた点字プリンタも開発されている．
点字を正確に大量に製作するには，薄い亜鉛板やプラスチック板
を二枚重ねにして点字を打刻した原版を作り，その2枚の間に点字
用紙を挟んでローラに通す，エンボス印刷と呼ばれるプレス印刷方
式が日本の点字制定の数年後から用いられている．その原版を点字
データから自動的に製作する点字自動製版機は1980年に開発され，
点字印刷を行っている点字出版所等で使用されている．
c.点字読取装置 紙に凹凸で書かれた点字の文をデータ化す
ることが必要なとき，または点字を知らない人が点字を読まなけれ
ばならないときのために，点字OCR（optical character reader ）
ともいうべきシステムがある．
点字の凹凸を光の陰のパターンとしてとらえてパターン認識し，
点字データとして処理するシステムで，通常のOCRでも凹凸の陰
影が明りょうに認識できる機種を利用してシステム構成しているも
のもある．
点字が片面（一つの面だけ凸）のみに書かれている場合は処理精
度も高く，実際に活用されているが，両面（表側が凸の点字行の行
間に裏側が凸の点字行があるものが多い）に書かれている場合は処
理精度が低くなることが多く，実用に堪えないことすらある．いず
れも，点字プリンタや点字製版機により点字プレス印刷されたもの
は読取り精度が高いが，手書きのものは精度が低くなる．
