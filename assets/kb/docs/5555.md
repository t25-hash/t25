# 4・3・2 脳磁計

> **章**: 第4章 検査・診断機器
> **節**: 4・3 磁気測定装置

脳神経活動に伴って頭の周りに生じる磁場は脳磁図(MEG :magnetoencephalogram)と呼ばれ,脳機能研究や脳疾患診断の臨床において重要な役割を果たしている.脳磁図は非常に微弱なものであり,測定には超高感度のSQUID 磁束計が必要である.脳磁図測定の目的の大部分は,測定した脳磁図データより脳内部の磁場発生源,すなわち脳活動源を推定することにある.そのために,頭部の多くの場所からの磁場を同時に計測できる多チャネルSQUIDシステムが脳磁計には必要である.多チャネルSQUID 磁束計が本格的に実用化になったのは,1990年代に入ってからである.1990年にアメリカで37チャネルSQUID システムが開発されると,日本国内の病院にも導入され,脳磁図の臨床応用に関する研究が本格的に始められた.さらに頭部全体をカバーして測定できるようなホールヘッドタイプの64チャネルや122チャネルシステム,さらに日本においても超伝導センサ研究所で256チャネルシステムも開発された.現在では,国内において数十台のSQUID 磁束計が大学病院や脳研究機関に設置され,脳機能研究や脳磁図の臨床応用に使用されている.図4・17に,エレクタニューロマグ社製の306チャネル脳磁計を示す.この脳磁計は,204チャネルの平面型グラディオメータと102チャネルのマグネットメータで構成されている.脳磁計は非常に微弱な磁場を計測するため,磁気シールドルームの中に設置することが不可欠である.特に検出コイルがマグネットメータの場合だと,ノイズの影響を直接受け,SQUID が動作しないこともある.脳磁図は脳波と同じように脳内の電気現象をとらえたものであり,X 線CT,MRI,PET などの画像診断装置とは異なった特徴を有しており,以下のような特徴がある.(1) 機能的情報が得られる脳磁図計測は神経細胞の電流変化を測定対象としているので,脳の機能的情報がミリ秒の時間分解能で得られる.(2) 無侵襲計測であるX 線CT やPET などの診断装置は,無侵襲とはいっても体外から生体に何らかの物理エネルギーを加え,生体との相互作用を測定して体内の様子を推定するものである.この点,磁気計測は体内から発生する磁気を検出するのであって真に無侵襲である.(3) 電源局在推定が数mmの精度でできる生体組織では比透磁率は真空の透磁率にほぼ等しい.このため磁気計測では,脳波測定での欠点である頭蓋骨の影響による信号の減衰が無視できるので,特に大脳皮質の電源推定に有効である.例えば,聴性誘発脳磁図では2~3mmの精度で音の周波数に対応したニューロン群の反応部位が推定できる.(4) 脳波で測定できない脳内電源が検出できる極性が互いに逆向きにある二つの電源は閉ループ電流を作り電源の近傍だけを短絡的に電流が流れるので,脳波では検出しにくい.しかし,脳波に信号となって現れてこない閉ループ電流はループ面に垂直に磁場を生成するので,脳磁図で検出することができる.ヒトを対象とした非侵襲脳機能計測において,現在のところmsオーダの時間分解能で脳機能局在を推定する手法がほかにないため,脳磁図計測は脳機能研究に必須の手段であることは間違いない.脳磁図は脳の基礎研究や言語活動,記憶,認知といった高次脳機能の研究のみならず臨床診断にも有用であり,そこでは,てんかんスパイク波の検出をはじめ,脳腫瘍や脳血管障害に伴って出現する徐波,さらには事象関連電位等に伴う超低周波の脳電気活動の磁気計測が行われている.脳磁計は,ハードウェア的には多チャネル化,低ノイズ化も進み,ほぼ満足する域に達してきた.しかし,脳磁図を用いて特定の高次脳機能をよりよく抽出するためのパラダイムや,脳のダイナミックな機構を理解するための新しいパラダイムおよび計測の手法がまだ十分には開発されていない.また,脳磁図計測で雑音の大きい,すなわち信号対雑音比が極端に小さい信号の電源推定や,広がりをもつ電源推定のための信号処理技術や逆問題手法,およびそのためのよりよい脳内電源モデルの構築はまだ不十分であり,今後の発展が望まれる.
