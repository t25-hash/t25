# 4・3・3 心磁計

> **章**: 第4章 検査・診断機器
> **節**: 4・3 磁気測定装置

心臓病はがんや脳血管疾患などと並んで死亡原因の上位に挙げられている。心臓病の初期診断のためには心電図の計測が有効である。心臓の筋肉や神経の活動によって生じる電流が作る体表の電位差を計測したものが心電図である。この電流は、また、磁場(心磁図、MCG : magnetocardiogram)を生成する。心磁計はこの磁場を計測する装置である。心磁図の大きさは約10T と脳磁図より2けた以上大きい信号であるが、フラックスゲート磁束計では測定感度が不足しており、SQUID 磁束計が必要である。従来のチャネル数が少ない測定面が平らなSQUID 磁束計の場合は、脳磁図計測にも心磁図計測にも同じSQUID 磁束計を兼用して用いることができるが、頭全体を覆うホールヘッドタイプの脳磁計が主流となると、心磁計とは共用できず、心磁図専用の磁束計が必要となる。SQUID 磁束計を用いた生体磁気計測では脳磁図計測による脳機能計測が中心であり、心磁図計測はその臨床的有用性がまだはっきりと確立していないこともあって、まだこれからの分野である。このため、心磁図計測専用の磁束計もあまり存在していない。ここでは、(株)日立製作所が開発した64チャネル心臓磁場計測システムに関して簡単に紹介する。この装置では、磁気センサと胸部または心臓との位置関係が正確にとれるように、,の直行3方向にベッドを動かすことができる。計測は簡易磁気シールドルームの中で行い、患者は着衣のまま心磁図計測が行われる。この心磁計に搭載したSQUID は1~100Hz の周波数大域で10fT/ Hz の高いSN 比をもち、検出コイルはグラディオメータとなっている。磁気センサアレイは25mmのセンサ間隔で8×8の格子状に配置し、175mm×175mmの測定面積をカバーすることにより、成人の心臓を1回で計測できる。各チャネルの時間波形は時間分解能として最小0.5msまで計測できる。64チャネルの心磁図波形は各点ごとの時間波形表示や重ね合わせた波形として表示できる。磁場は胸部に対して垂直な法線方向成分を計測するが、そのマッピングでは法線成分の等磁場線図と、それに垂直な成分と成分を合成した接線成分と等価な等磁場線図を解析的に表示することができる。接線成分表示では、心臓内の電流強度が大きいところと計測面での磁場強度の大きいところが1:1に対応し、心臓内のどの部位が活動しているのか直感的に理解することが可能である。心磁計は臨床の場でまだ実際に用いられてはいないが、不整脈や虚血性心疾患などへの応用が期待されている。不整脈には多くの疾患要因があり、伝導障害部位や異常興奮部位などを知ることが診断上重要であり、伝ぱ過程の可視化と部位の三次元推定の画像化技術が求められている。心磁図を測定することにより、不整脈の局在する電流源の位置と大きさを明らかにすることができる。心磁計を用いた心磁図測定の利点の一つとして、胎児の心臓の発9 ―29達状態を知ることができることが挙げられる。母体の腹部から胎児の心電図を測定しようとすると、電気絶縁層となる胎児の影響で明りょうな信号を得にくいが、心磁図ではその影響を全く受けずに測定することができる。
