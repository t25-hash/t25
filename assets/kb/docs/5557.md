# 4・3・4 肺磁計

> **章**: 第4章 検査・診断機器
> **節**: 4・3 磁気測定装置

肺磁計とは、肺に蓄積した磁性微粒子が作る磁場(肺磁図、MNG : magnetopneumogram)を測定し、肺内に蓄積している粉じんの量を測定する装置である。磁場を計測する磁束計、肺内の磁性微粒子を磁化させるマグネットから構成される。粉じんの多い工事現場で働く労働者や鉱山労働者の中には、長年の労働によって排出されない粉じんが肺内に多く蓄積している者がいる。粉じんが肺内に蓄積すると、珪肺症などの塵肺症を引き起こすので、その蓄積には注意を要する。塵肺の多くは現在X 線によって診断が行われているが、X 線では肺内に長く蓄積しているような細かい粉じんはほとんど検出されず、塵肺が進行し、肺の組織に変化が生じるまで病気は発見されない。X 線で塵肺症が発見されてもすでに手遅れで、蓄積している粉じんのために病気は進む傾向にある。早期発見のために微小量の粉じんの発見が要求され、どうしても病気になる前に肺内の粉じんの量を知る必要がある。これに応えるのが、高感度磁束計を用いた肺磁界計測である。多くの粉じんは強弱の差はあるが磁気的な性質をもっているため、肺内蓄積粉じんに外部から直流磁界を加えれば粉じんは磁化され、外部磁化を取り去った後でも残留磁気として外部に微弱磁界を発生する。この残留磁気により体外に発生する磁界を体表面上で測定することによって、肺内に蓄積している粉じんの量と分布を求めることができる。さらに、残留磁界の時間変化を測定すれば、蓄積している個々の粒子の回転によって磁化が減少する様子をとらえることができる。印加磁界を取り去ると、個々の粒子は周囲の細胞の運動によって方向を乱していく。個々の磁気微粒子が方向を乱しランダムな方向を向いていくに従って、体外での検出磁界は小さくなる。また、微粒子は肺胞マクロファージに取込まれ排泄されていくので、微粒子群全体の磁化は徐々に消失していく。これらの時間変化、すなわち緩和過程を測定すれば、肺の活動度を知ることができる。肺内粉じんの分布と総量を求めるために、肺全体を一様な強さの外部直流磁界で磁化させる。このとき、大きなヘルムホルツコイルを用いてできるだけ一様な磁場を作り磁化させる。粉じんの種類によって粉じんの完全磁化に要する外部直流磁界の強さは異なるが、100mT あれば十分である。計測される磁束密度は、粉じん1mg当たり、0.1m離れた点で数nT 程度である。肺磁界計測にはSQUID 磁束計も用いられるが、冷却の必要もないフラックスゲート磁束計が、取扱いが簡便なため多く用いられている。しかしながら、磁性が弱い粉じんの場合にはフラックスゲート磁束計では感度不足であり、このような場合にはSQUID 磁束計を必要とする。この場合でも、液体窒素温度で動作する高温超伝導デバイスで作られた高温超伝導SQUID 磁束計を用いると維持費が安くできる。
