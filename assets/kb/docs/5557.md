# 9・4・1 補聴器

> **章**: 第9章 支援機器
> **節**: 9・4 聴覚言語障害者用機器

補聴器（hearing aid）は難聴者を対象とした福祉機器として最
も知名度の高い補装具の一つであるが，その実態は意外に知られて
いない．工業製品としての補聴器は，JIS C5512：2000により規
定されており，国際規格ではIEC60118-0～14に規定されている．
補聴器の最も基本的な機能は音を増幅して出力するという単純な
ものであるが，実際の製品となるためには多くのハードルが存在す
る．補聴器は，原則として1本（個）の電池で動作させるために電
設備としての規格が定められている．
源電圧が制限され，その条件下で十分な増幅度と出力音圧を確保
し，長時間の動作をする必要があるため回路設計は決してやさしい
ものではない．また，音響障害を防ぐための出力制限回路や，聴覚
の特性に合わせるための周波数特性の調整回路を内蔵するのが一般
的である．そして，これらすべての構成部品を耳の穴に収まるサイ
ズにしなくてはならないのである．
形状から見た補聴器の種類は，箱形，耳掛け形，耳穴形（図9･
12）に分類でき，国内で出荷された補聴器の数は46万台（2006
年，全国補聴器メーカー協議会）ほどである．その中では耳穴形補
聴器の出荷台数が最も多く，全体の54 %程度を占めている．内部
の処理回路で分類すると，アナログ式，プログラマブル（ハイブリ
ッド）式，ディジタル式の3種類に大別できる．アナログ式補聴器
は，音の入口であるマイクロホンから出力となるレシーバ（イヤホ
ン）までアナログ回路で構成され，真空管，トランジスタ，ICと
変わっているものの，開発当初から受け継がれてきた方式である．
プログラマブル式は，増幅回路はアナログのままであるが音質など
の調整部分に従来使われてきたトリマ（半固定抵抗器）を半導体に
置換えてディジタル制御（設定）できるようにした補聴器であり，
外部の専用調整装置と接続して設定を行うものである．トリマがな
くなることで，小型化のニーズに対応できるようになり，また故障

販売店や医療施設が限られてしまうという問題もある．ディジタル
式はマイクロホンやレシーバ周辺部分以外をディジタル化し，周波
数特性の調整などをディジタル信号処理によって実現している．デ
ィジタル化することで細かな周波数特性が設定可能となり，設定状
態の再現性も高くなった．また，補聴器の小型化によりマイクロホ
ンとレシーバとの距離が短くなるために発生しやすくなるハウリン
グを抑制するための処理も，安定した動作が実現されたことで実用

hearing aid）と呼ばれているものがある．これは，FM受信回路

無線局の一種として位置付けられ，補聴補助用ラジオマイク用無線
補聴器は一般の家庭用電気機器とは異なり，使用する個人ごとに
異なる聴覚の状態に合わせて調整する必要がある．この補聴器を特
定個人に適合することをフィッティングといい，医療機関で十分な
耳の検査をしたうえで行うのが原則である．基本的なフィッティン
グの考え方は，低下した聴力しきい値をもとにして周波数帯域ごと
の増幅特性を難聴者の生活環境を考慮しながら決定する．実際には
何度か実生活内で試聴したうえで設定状態を確定するものである．
しかし，多くの眼鏡店で補聴器が販売されていることもあって，眼
鏡の調整と補聴器の調整が同様なものとして考えられているのが現
実である．眼鏡は水晶体の物理的なひずみを補正するのに対して，
補聴器は聴神経機能の低下，つまり感覚を補正するものであり，両
者は全く違うものといえる．補聴器の装用状態は他覚的な検査や評
価をしにくいものであるため，熟練した担当者が実施することが望
ましい．
