# 4・4・1 ベッドサイドモニタ

> **章**: 第4章 検査・診断機器
> **節**: 4・4 生体情報モニタ

ICU,CCU、救急病棟などに収容されている重症患者をはじめ、手術中や救急搬送中で病態が急変する可能性がある患者は、常時心電図や呼吸数などのバイタルサインを監視する必要がある。この目的に対して使用されるのが生体情報モニタ(単にモニタと呼ばれる)であり、大まかにはベッドサイドで使われる一人用生体情報モニタと、ナースステーションなどで使われる多人数用セントラルモニタに分類される。当初、重症患者用を主目的として使用されてきたモニタも、最近では中間病棟や一般病棟用としても用途が拡大され、臨床上必須の機器として位置付けられるようになってきた。一人用生体情報モニタ(ベッドサイドモニタ、bedside monitor)では、基本的なバイタルサインとしての心電図、血圧、呼吸数、体温といったものが監視されている(図4・18)。これらに加え、最近9 ―30医療・福祉・バイオ機器では計測技術の進歩が目覚しく、非観血的、無侵襲でかなり多くの生体情報が得られるようになってきている。特に指先などにトランスデューサを装着するだけで、簡単に連続的に測定できる動脈血酸素飽和度(SpO)は基本パラメータの一つになっている。非観血血圧測定においても連続的に血圧を測定できるようになり、患者の負担をより軽くし、かつ質の高いデータが測定できるようになってきている。さらには脳波やそのスペクトル解析を行い患者の意識レベルを測定したり、連続で心拍出量を測定できるものもある。ベッドサイドモニタでは測定できない外部機器のデータ、例えば人工呼吸器のデータや新生児、未熟児を対象にした経皮酸素分圧モニタのデータ等もベッドサイドモニタに取込み、同じ患者のデータとして一元管理できるようになってきており、用途に応じて使われる。このように測定され処理された生体情報のデータは、CRT(cathode ray tube)あるいは液晶ディスプレイに波形および数値で表示される。ここでベッドサイドモニタに血圧、心拍数値などの上限および下限を設定しておけば、この範囲を超えたとき、音または光により警報を発し、医療スタッフに緊急事態を知らせるようになっている。さらに連続記録部がある場合は、警報発生時に自動的に数十秒間心電図が記録される。また、発生した不整脈を保存して一覧表として表示したり、患者の治療経過をレビューするために、心電図波形や数値データを長時間保存してトレンドグラフや波形の長時間圧縮表示をすることにより診断治療に有効利用されている。このベッドサイドモニタのうち最も一般的なものが、心電図(心拍数を含む)の監視を行う心電図モニタ(ハートモニタともいう)であり、一般病棟をはじめとして、無線テレメータの簡便さとともに広く使われている。多パラメータ用のベッドサイドモニタやさらに多機能のポリグラフは、一部には無線テレメータの方式があるが、有線方式が主流である。また、得られたデータをセントラルモニタなどに送ることなどもモニタの重要な役割の一つであり、最近ではセントラルモニタとの間をネットワークで接続し、ネットワーク上のどこからでも患者情報(ベッドサイドモニタの情報)を得ることが可能になってきている。
