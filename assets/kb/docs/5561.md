# 4・5・1 X 線イメージングシステム

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X 線装置

X 線診断とは、広義に解釈すれば、後述するX 線透視あるいは撮影によって得られた濃度差のある画像を観察し疾病に関する有効9 ―32医療・福祉・バイオ機器な情報を取出すことにある。この画像を構成する過程において多くの周辺器機が必要とされる。これらの諸要素は、その時点で得られる新技術を導入してシステム全体の改良が加えられている。X 線による診断装置として最終のアウトプットであるX 線画像に関しても、新材料の開発やエレクトロニクスの進歩を取り入れ、新しい画像情報も得られるようになってきている。特に近年、X 線CT に始まったディジタル化を軸に各種の技術革新が行われ、診断の手法を含めてその変革が著しい。循環器系の診断では、ディジタルサブトラクションアンギオグラフィー(DSA : digital subtraction angiography)、一般撮影ではコンピューテッドラジオグラフィー(CR : computed radiography)の導入がある。さらに最近では、液晶パネルの技術をベースに開発されたディジタルX 線平面検出器(フラットパネルディテクタ、FPD :flat panel detector)が各種の部位別診断装置に利用・開発されるなど、X 線画像の分野では、X 線の利用を始めてから数十年続いたアナログの時代からディジタル化の時代への移行が顕著になりつつある。このようにX 線診断機器は、究極の目標である人体の情報を最大限に最高の画質で、かつリアルタイムで診断に供するため、今後も引き続いて改良が図られていくことが予想される。特に周辺の要素技術の向上と、メカトロニクス、マイクロコンピュータなどの利用による新しい制御技術の導入、さらにはディジタル化によって高度化された画像処理技術によるより正確な診断情報、あるいは従来得られなかった知見の獲得などがある。また、コンピュータネットワークの整備・進展などで必要な画像がタイムリーに、かつ迅速に提供されるなど、X 線診断の領域は変化・拡大していくものと思われる。a. X 線透視撮影法の原理。 原理X 線は波長の短い電磁波であり、光と同様直進する性質をもち、物質中を減弱しながら透過する。X 線の減弱は光電効果、コンプトン散乱(Compton scattering)およびトムソン散乱(Thomson scattering)によるが、診断X 線のエネルギー領域では、このうち前二者が主体である。X 線の減弱は式(4・1)により表すことができる。X 線が人体を透過するとき、人体の構造に応じた減弱を示すので、このX 線を検出することにより人体構造を影像化することができる。図4・23に骨組織、軟部組織、脂肪組織の線減弱係数を示す。減弱係数はX 線のエネルギーが大きくなるほど小さくなる。図4・23人体組織の線減弱係数のX 線管電圧依存性特に骨組織では変化が大きい。したがって低圧撮影では、各組織での減弱が大となりコントラストの大きな像が得られるが、被ばくの影響も大となる。一方、高圧撮影では各組織での減弱が小さくなるので得られた画像でのコントラストも小となるが、それだけ被ばくの影響は小さくなる。つまり、高圧撮影はコントラストの大きい被写体を撮影するのに適していることがわかる。. X 線の発生図4・24に示すように、回転陽極X 線管(rotating anode X-ray tube)の陰極フィラメントから発生した熱電子が陽極と陰極の間に加えられた高電圧に加速され陽極ターゲットに衝突し、陽極中のターゲット材料の原子と相互作用して運動エネルギーを失う。このとき、エネルギーの一部が陽極から電磁波として放出される。この電磁波のうち、振動数が約10s以上がX線である。X 線は電磁波としての波動性と光(量)子としての粒子性を示す。このエネルギーの一部が陽極での相互作用によってX 線光子のエネルギーとして放出される。放出されるX 線の波長を調べると連続なスペクトルとターゲット材料に特有な線スペクトルが現れ、それぞれ連続X 線(白色X 線)、特性X 線(固有X 線)と呼ばれる。このうち、一般の診断用X 線管では連続X 線を、マンモグラフィーのような軟部組織の診断には特性X 線を用いる。連続X 線の管電圧による変化を図4・25に示す。この図より次のことがわかる。(1) 管電圧が高くなるほど、曲線と横軸で囲まれた面積が大きくなる。この面積はX 線の全強度を表し、管電圧のほぼ二乗に比例して急速に増大する。(2) 最高強度を示す波長および最短波長λは、管電圧が高図4・24 回転陽極X 線管図4・25 連続X 線(白色X 線)のスペクトルくなるほど短波長側に移動する。次に、管電圧を一定にして管電流を変化させると、λや最高強度を示す波長は全く変化しないが、曲線で囲まれた面積、つまり連続X 線の全強度は変化し、その変化は管電流の大きさに比例して増大する。陽極ターゲットの材料を変えてもλや最高強度を示す波長は変化しないが、管電圧と管電流が同じでもターゲットの材料が変わると、発生するX 線の全強度は変化する。そしてこの全強度は材料の原子番号に比例する。タングステン(=74)、=100kV の場合にはη=0.7%となり、残りの99%以上は熱に変換される。このため、ターゲット表面の材料には融点が高く、原子番号が高いタングステンW、またはモリブデンMoが使用されることが多い。b. 基本構成図4・26にX 線撮影系の基本構成を示す。X線管から発するX 線は、被写体により減弱されX 線フィルムを露光する。X 線管では、加速した電子をターゲットに衝突させてX線を発生させる。このとき、X 線源の焦点サイズが小さいほど鮮明な画像が得られる。X 線絞りによりX 線束を必要視野に制限し、被ばくおよび散乱線の発生を最小限にする。被写体では、コンプトン散乱により散乱線が発生し画像のコントラストを低下させるので、通常は散乱線を防ぎ直接線のみを通過させるグリッド(X-raygrid)をフィルムの直前に配置する。X 線の検出には、図4・26のように蛍光体よりなる増感紙(スクリーン)とX 線フィルムを組合わせたスクリーン・フィルム系が用いられるほか、X 線イメージインテンシファイア(XRII: X-ray image intensifier)などがある。増感紙(intensifying screen)はX 線検出効率を増し、感度を向上させる役目を果たす。フィルムはX 線により露光されるだけでなく、増感紙からの発光により露光される。XRIIでは入力面でX 線検出が行われ、出力面に輝度増倍された光学像が得られる。この光学像はスポットカメラやテレビカメラにより撮像される。X 線画像を得る手段としては、短時間に強いX 線を照射して静止画像を得る撮影と、弱いX 線を連続的、あるいはパルス状に照射して画像を動画として得る透視があり、どちらか一方または両者を併用することで、診断に最適な画像を得る。診断用X 線システムは診断目的から見ると細かく分類されるが、いずれのシステムも基本的に図4・27のようなユニットの組合せ図4・26 X 線撮影系の基本構成9 ―33で構成される。. X 線発生装置X 線発生装置は、X 線の発生というX線システムにとって最も基本的な機能を担うユニットである。X線を発生させるためには数万V から十数万V の高電圧が必要であり、その高電圧を作るX 線高電圧装置と、この高電圧によってX線を発生させるX 線源装置および両ユニットを電気的に接続するプラグ付きX 線高電圧ケーブルからなる。. X 線機械装置X 線機械装置は、被検者を位置決めしX 線の透過方向を定める機能を担っている。この装置は透視と撮影の両方に対応し、消化器や循環器用として多用される透視撮影台と、撮影だけに対応したX 線撮影台、およびX 線源装置とX 線影像装置を保持して位置決めする保持装置に大別される。しかしながら、被検者の位置決めに必要な機能は診断目的により異なるため、X 線機械装置はそれぞれのX 線システムで大きく異なる形態となっている。例えば消化器用システムでは、消化管の内部に造影剤(contrast medium)であるバリウム製剤を一様に付着させるために被検者の体の向きを容易に変えられる機構を備えている。一方、循環器用システムでは、被検者をできるだけ動かさずにX 線の投射方向を自在に変えられるような機械装置になっている。また最近では、多くの診断目的に対応できるような多目的診断用システムも急速に普及してきている。. X 線映像装置X 線映像装置は、人体を透過したX 線をとらえてフィルムやモニタに画像として表示する機能を担っている。従来から透視用にはXRIIとテレビカメラを組合わせたX 線テレビカメラ装置が、撮影用にはスクリーン・フィルム系が用いられている。XRIIはX 線像を可視化するための大型の真空管であり、微弱なX 線像も可視化できることやリアルタイムでX 線画像を得られる特徴を有しており、透視を行うシステムでは不可欠なユニットである。X 線テレビカメラ装置は、このXRIIの出力光学像をCCD(charge coupled device)カメラでとらえ、最終的にはCRT(cathode ray tube)モニタや液晶ディスプレイなどに映像として表示する。最近はX 線映像装置の形態はX 線画像のディジタル化に対応して大きく変化している。高精細化したXRIIとテレビカメラを組合わせた高精細X 線テレビカメラからのビデオ信号をディジタル化する高精細実時間DR(digital radiography)方式、あるいはX線像を潜像化したIP(imaging plate、イメージングプレート)をレーザ光で走査して光ファイバで集光し、フォトマルチプライヤで電気信号に変換後ディジタル化するCR 方式がある。また最近では、急速に革新が進む半導体技術を取り入れたディジタルX 線平面検出器を利用したFPD 方式がある。これらはすべて電気出力をディジタル化して何らかの画像処理を施し、高精細のX 線像を提X 線装置X 線発生装置X 線源装置X 線管装置プラグ付きX 線高電圧ケーブル照射野限定器X 線高電圧装置高電圧発生装置X 線制御装置X 線機械装置透視撮影台X 線撮影台保持装置X 線映像装置X 線イメージインテンシファイア装置X 線間接撮影用ミラーカメラ装置X 線テレビカメラ装置X 線画像処理装置ディジタル撮影装置ディジタル透視装置その他関連機器図4・27 診断用X 線システムの基本構成9 ―34医療・福祉・バイオ機器供する。特にFPD の出現は、単にX 線検出器の形状が薄くなるだけでなく、シャープでひずみのない画像がリアルタイムで得られることから、X 線システムの診断用途を広げるものと期待されている。. X 線画像処理装置X 線画像処理装置は、X 線映像装置からの映像信号をディジタルデータに変化させてメモリに記憶するとともに、CRT モニタなどに表示するユニットである。ディジタル化された映像信号に対して、診断目的に応じてさまざまな処理を加えることで、より診断能に優れた画像を提供する。X 線画像処理装置はディジタル撮影装置とディジタル透視装置に大別される。
