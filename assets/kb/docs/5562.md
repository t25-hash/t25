# 4・5・2 透視撮影装置

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X 線装置

透視撮影装置(radiography/fluoroscopy system)で最も広く一般的に用いられているのは,消化器系診断用システムと循環器系診断用システムである.ここでは消化器系診断用システムについて述べ,循環器系診断用システムについては4・5・5項で述べる.消化器系診断システムは,食道,胃,大腸などの消化管を中心に肝臓,胆のうなどの疾患に対する画像診断に用いられるシステムである.消化管の診断は,硫酸バリウムなど陽性造影剤(positive con-trast medium,X 線の吸収を目的とする液剤)と空気や炭酸ガスなどの陰性造影剤(negative contrast medium,X 線の透過を目的とする発泡剤など)を同時に用いた二重造影法の普及により著しく進歩し,システムも二重造影法に適した形態で発展してきた.最近では消化管診断が内視鏡検査に移行してきているが,X 線検査では対象とする臓器全体を一度に観察できることや,病変部の大きさと形状を正面から把握できるという特徴があり,依然として幅広く利用されている.胆のう,肝臓の検査では,内視鏡や直接穿刺(せんし)針を併用する検査などが,このシステムを用いて一般的に行われている.消化器系診断用システムの基本構成は図4・28に示す.このシステムは,X 線映像装置とそれらを装着するX 線透視撮影台からなっており,造影剤の流れをコントロールしながら,体位変換と位置決めを行うため,被検者を載せて起倒や上下左右スライドなどの動作を行う.X 線管球をテーブルの上方あるいは下方のどちらに配置するかにより,オーバ式とアンダ式があり,オーバ式は多目的診断に適し,アンダ式では良好なX 線画像が得られる.速写撮影装置は,X 線テレビモニタ上の透視像を見ながら撮影したい部位をタイミングよくX 線フィルムに撮影するための装置である.現在では,省力化や診断能率の面で,シートフィルムを用いる遠隔式のカセッテレス方式が普及している.以下にカセッテ,およびカセッテレス方式について説明する.カセッテ(film cassettes for medical radiography)はX 線フ図4・28消化器系診断用システムの基本構成(図はオーバ式で,X 線管球と映像受像系の配置が入れ替わったものがアンダ式)ィルムを収納する容器であり,カセッテ内の両面または片面には増感紙が張られ,この中にX 線フィルムを密着させて使用し,X 線画像を形成するものである.X 線フィルムを光から遮光する機能をもち,フィルムの出し入れが容易でX 線吸収が少なく,軽量であることが要求される.構造はフィルム収納部と裏ぶたからなり,収容部の枠は軽量で,反りがなく,表面の材質はX 線吸収が少なく均一なものが必要である.裏ぶたはフィルムと増感紙の密着を保つため,クッション層を設けている.また多くの場合,カセッテ背後の後方散乱X 線を防ぐため,裏ぶたに鉛板が張り付けられている.カセッテレス方式(cassetteless type)は撮影枚数の多い消化器系診断のため開発されたもので,X 線フィルムを収納する容器であるカセッテを用いない方式で,機能的には循環器系診断システムで用いられるフィルムチェンジャと似ている.X 線フィルムは50~100枚程度,供給マガジンと呼ばれるところに収納されており,このマガジンから撮影に応じて1枚ずつ引出され,増感紙が張り付けられたフィルムホルダと呼ばれるものの間にセットされる.X 線フィルムと増感紙が一体となったフィルムホルダは撮影開始と同時に撮影位置まで移動し,画像のぼけを防止するため,移動中にフィルムと増感紙の密着が増すよう各種の工夫が施されている.撮影終了とともにフィルムホルダは元の位置に戻り,撮影済みX線フィルムは収納マガジンに格納される.このようにして予定された被検者の一連の撮影が終了すると収納マガジンは外されて自動現像機にセットされ,X 線フィルムは現像される.一般的には,このような一連の撮影動作は遠隔操作により行われる.速写撮影装置はまた,診断目的に対応する形態と機能を有する透視撮影台の性質によって,近接操作式透視撮影システムと遠隔操作式透視撮影システムに分類できる.両方式の概略図を図4・29に示す.近接操作式透視撮影システムは,操作者が被検者が載っている透視撮影台に接近して操作する手法で,操作者の注入操作が必要な造影検査や精密検査などに用いられる.一方,遠隔操作式透視撮影システムは,透視撮影台を設置している撮影室と対向して設けられた(a) 近接操作式(b) 遠隔操作式図4・29透視撮影システムの概略図操作室から,鉛ガラスの窓を介して被検者の状態を観察可能な場所に操作卓を配置し,透視はX 線テレビで行い,モニタに映った透視像を連続的に観察しながら診断に最適のタイミングで速写撮影する.このシステムでは,被検者が撮影室に一人でおかれるため,装置の安全性に対する配慮と透視撮影台の各部の移動操作に細心の注意が必要である.
