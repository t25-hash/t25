# 4・5・3 直接撮影装置

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X 線装置

直接撮影装置（direct radiography system）は患者に造影剤を
注入するといった特別な処理をすることなく，また特殊な装置を用
いることなく行う単純撮影を中心とした診断システムで，一般撮影
用システムともいわれる．診断部位としては特に限定されていない
が，消化器系や循環器系などの診断がそれぞれ専用の装置で行われ
ているため，おのずから胸腹部と骨格系の診断が中心になってい
る．比較的単純な撮影で豊富な診断情報が得られることや，検査に
苦痛を伴わないこと，胸腹部と骨格系の異常を訴える患者が多いこ
となどから，この分野での検査件数は非常に多くなる．また，得ら
れた撮影フィルムから直接診断を下す以外に，他の画像診断や検査
を計画するための情報を得る役目も併せ持っている．
要求される機能としては，患者を所定の状態に位置決めする，X
線の照射範囲を決める，X線を照射して撮影像を得るといったX
線装置にとって最も基本的な機能があればよく，一般的には透視機
能は備えていない．
一般撮影用システムは，用途に応じて次のような基本ユニットを
組合わせて構築される．
(１)水平式撮影台などの患者を位置決めする機器
(２) X線管保持装置，可動絞りなどのX線照射位置を決める機
器
(３) X線高電圧装置，X線管などX線発生系の機器
(４)ブッキー装置，フィルムチェンジャなどX線受像系の機器
床上式X線管保持装置を中心に構成したシステム例を図4･30に
示す．なお，この分野にはCR装置がかなり導入されているが，最
近ではFPDを使用したシステムも導入され，X線画像のディジタ
ル化において最も注目される分野となっている．
