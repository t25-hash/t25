# 4・5・3 直接撮影装置

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X 線装置

直接撮影装置(direct radiography system)は患者に造影剤を注入するといった特別な処理をすることなく、また特殊な装置を用いることなく行う単純撮影を中心とした診断システムで、一般撮影用システムともいわれる。診断部位としては特に限定されていないが、消化器系や循環器系などの診断がそれぞれ専用の装置で行われているため、おのずから胸腹部と骨格系の診断が中心になっている。比較的単純な撮影で豊富な診断情報が得られることや、検査に苦痛を伴わないこと、胸腹部と骨格系の異常を訴える患者が多いことなどから、この分野での検査件数は非常に多くなる。また、得られた撮影フィルムから直接診断を下す以外に、他の画像診断や検査を計画するための情報を得る役目も併せ持っている。要求される機能としては、患者を所定の状態に位置決めする、X線の照射範囲を決める、X 線を照射して撮影像を得るといったX線装置にとって最も基本的な機能があればよく、一般的には透視機能は備えていない。一般撮影用システムは、用途に応じて次のような基本ユニットを組合わせて構築される。(1) 水平式撮影台などの患者を位置決めする機器(2) X 線管保持装置、可動絞りなどのX 線照射位置を決める機器(3) X 線高電圧装置、X 線管などX 線発生系の機器(4) ブッキー装置、フィルムチェンジャなどX 線受像系の機器床上式X 線管保持装置を中心に構成したシステム例を図4・30に示す。なお、この分野にはCR 装置がかなり導入されているが、最近ではFPD を使用したシステムも導入され、X 線画像のディジタル化において最も注目される分野となっている。
