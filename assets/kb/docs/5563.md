# 9・5・2 手動車いす

> **章**: 第9章 支援機器
> **節**: 9・5 移動用機器

手動車いす（manually propelled wheelchair ）は，9･5･3項で
後述する電動車いす等と比べ価格も比較的低いこともあり，各種車
いすの中で最も普及している．手動車いすは，おもに自走式と介助
式に大別される．その構造，寸法，外観，試験方法等については，
JISによる規格（JIS T9201：2006）が存在する ．
自走式の一般的な構造を図9･19 に示す．自走式は，手でハン
ドリム（hand rim）を押して動力をタイヤに伝えることにより操
作する．介助式には自走用のハンドリムは付いておらず，比較的小
さい後輪が付いている．なお，図9･19のような自走式車いすは介
助用としても用いることができる．車いすのフレームは通常軽量な
パイプで造られ，材質は近年ではアルミニウム合金が用いられるこ
とが多い ．また，保管場所などの面から，折り畳み式の製品が
多い．座面や背もたれについては，折り畳みが可能なように布張り
のものが多い．
手動車いすについては，上述したような標準型だけでなく，背も
たれが倒れるリクライニング型や座る部分の高さが低い低床型な
ど，非常に多くの種類が市販されている．また，座面の布等，素材
の改良により座り心地がかなり改善されている．金属にはない温か
みを出すために，素材に木を用いた商品も開発されている．バスケ
ットボールやマラソン等の競技専用に用いられるスポーツ車いす
も，多くの種類が市販されている ．
手動車いすの問題点としては，リフトの際と同様にベッド等から
の移乗が困難なこと，段差を登るのが困難なことなどが挙げられ
る．また，長時間たわんだ布張りの座面に座ることにより，骨盤の
後傾等の変形が生じることが指摘されている ．
