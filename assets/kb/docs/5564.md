# 9・5・3 電動車いす

> **章**: 第9章 支援機器
> **節**: 9・5 移動用機器

a.電動車いす 電動車いす（electric wheelchair ，powered
wheelchair ）についても，現在さまざまな種類が市販されている．
その標準的なものを図9･20 に示す．基本的な構造は手動車いす
と同様であるが，駆動力源として電池を用い，タイヤの駆動トルク
をモータにより生成するという特徴がある．その最高速度は「道路
交通法」上6km/hと規定されており，電池としては自動車用の鉛
蓄電池を用いたものが多い．駆動用のモータは2個積んでいるもの
図9･19自走式車いす各部の名称
が多い．電池およびモータを搭載するため，質量が約50kgほどに
なり，介護者1人では持ち上げることが困難なことがある ．ま
た，電動車いすは，価格が高いという点が問題の一つになってい
る．しかし，十分な操作力を車いすに伝達できない障害者にとって
は非常に有用な移動機器であるといえる．
電動車いすの操作方法としては，ジョイスティックにより進行方
向を指示するものが一般的である．上肢障害者のために，頭部の動
きなどで操作するタイプも開発されている．
前述の質量の問題を軽減するために，図9･21 に示すような自
走式車いすにユニットを取付けるタイプも販売されている．このタ
イプの質量は約23kgであり，かなりの改善が見られる ．
近年ではおもに高齢者用に，ハンドルで運転する三輪車や四輪車
タイプのものも市販されている（図9･22) ．これは，電動車いす
に準じた性能をもち，道路交通法上車いすと同じ扱いとなってい
る ．
b.パワーアシスト車いす（power assisted wheelchair ) 電
動車いすは，走行中，タイヤの動力をモータによりつねに与えてい
るが，これには多くの電力が必要であり，バッテリの質量の増大に
図9･20標準的な電動車いす
もつながる．また，上肢健常者にとっては，より積極的に上肢を用
いる機器のほうが，残存機能を維持するという点でよいと考えられ
る．そこで，ハンドリムに力を加えたときだけモータによる補助ト
ルクを生成する仕組みにしたのが，パワーアシスト式と呼ばれる方
式である．補助トルクは，入力された力に応じて出力されるように
なっている ．そのほかにも，介助者の握る取っ手にスイッチが
あり，介助者をアシストする方式の車いすも販売されている．な
お，片麻痺者のために，片方のハンドリムを動かせば両輪に補助ト
ルクが発生する，片漕式のものも販売されている．
このタイプには，手動車いすに取付けることができるものもあ
り，価格は電動車いすよりも低いが，あまり広まっていない．これ
は，ブレーキ時に振動現象を起こすなど，乗り心地が若干悪いこと
も原因の一つとして考えられる ．
