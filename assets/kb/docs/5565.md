# 4・5・5 循環器用X 線システム

> **章**: 第4章 検査・診断機器
> **節**: 4・5 X 線装置

循環器系診断用システムは,血管内に挿入したカテーテルから造影剤を注入してその流れをX 線像でとらえる血管造影検査を行うシステムである.全身の血管の疾患を対象としているが,特に疾患の多い心臓と頭部,および肝臓,下肢を対象とした検査が多い.最近では透視像を確認しながらカテーテルを用いて血管内で患部の治療を行うインタベンショナルラジオロジー(IVR : interventionalradiology)が急速に広まってきており,システムの用途が診断支援から治療支援にも拡大している.これらの検査では,被検者をできるだけ動かすことなくあらゆる方向から血管の観察ができることが求められる.このためX 線管装置と映像系装置をC形のアームの両端に取付け,アームの回転とスライドの組合せにより,X 線の投影方向を自在に変えられるような構造を採用しているシステムが多い.この診断では,他のX 線診断に比べ,造影剤の速い流れを追跡して撮影するために,短時間ばく射と高速連続撮影の技術がハードウェアおよびソフトウェアの両面から要求される.このためこのシステムでは,一般のX 線診断機器に比べ,より高精度のものが要求されると同時に,専用のユニットや周辺機器も多岐にわたってくる.このシステムの技術面における特殊性として,動きが早く,吸収コントラスト差の少ない臓器の診断,微細血管の診断,立体的な血管内の血流の流動状態の診断などが挙げられる.このため微細かつ高度な技術を要することに加えて,つねに検査に危険を伴う面が(a) 平行平面運動方式(b) 円弧運動方式図4・31 断層撮影装置の概略図9 ―36医療・福祉・バイオ機器多々あることから,システム構成を計画する際には十分な検討が必要である.このシステムには,循環器系で最も動きが激しい心臓および冠動脈の造影を目的とした心臓血管撮影システム(cardiovascularradiography system)と,心臓を除く全身の血管を撮影する頭・腹部血管撮影システムがある.図4・32に二方向からの撮影が可能な心臓血管撮影システムの概略図と概観図を示す.
