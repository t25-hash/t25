a.自動車 近年，障害者や高齢者の自動車への乗車に対する
要望が高まっている．これらの人々の乗降に便利なように，一般の
自動車を改造や仕様変更した福祉車両の需要が高まり，その開発お
よび販売が盛んに行われている．各メーカの多くの車種が対応して
おり，オプションとして選択できるようになっている．その種類と
しては，助手席回転シート車，シートリフトアップ車，車いす仕様
車に大別される．
助手席回転シート車では，車いすから助手席へのスムーズな移乗
を目的に，座席を外側に回転させる機構を取付けたものが一般的で
ある（図9･23) ．これに加えて，回転した座席を車外に出す機構
を加えたものもある．また，助手席だけでなく，後部座席を回転さ
せるものもある．シートリフトアップ車は，シートの回転および車
外への移動機構を電動モータにより駆動するものであり，比較的サ
イズの大きな車が対応している場合が多い．車いす仕様車は，車い
すごとの乗車を実現するために，後部にスロープやリフトを取付け
ることのできるものである（図9･24) ．このタイプでは，乗車
後，車いすの固定が必要である．また，車のシートの一部がそのま
ま簡易的な車いすになるものもある ．
これらは，いずれも使用者の状態に応じて個々に対応する必要が

c .運転補助装置 福祉車両の一つとして，上肢や下肢が不自
由な障害者のための運転装置をオプションとして選択できる車種が
増えてきている．上肢不自由者のための運転装置として，足によっ
てアクセル，ブレーキ，ステアリングなどの操作ができるシステム
が開発されている（図9･25) ．下肢不自由者のための運転装置と
しては，手でアクセルとブレーキを操作するコントロールグリップ
を取付けたものが開発されている．これらの装置は，個々の身体障
害者の状況に応じて適宜使用する機器を選定する必要がある．ま
た，これら運転装置そのものよりも，車いすなどから運転席への移
乗のほうが困難であるとの指摘がある ．
