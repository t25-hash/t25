# 9・6・1 障害者用入力装置

> **章**: 第9章 支援機器
> **節**: 9・6 コミュニケーション機器

障害者用入力装置としてさまざまなスイッチが市販されている．
しかしながら，どの操作スイッチを選択したらよいのかは，使用者
の残存機能ばかりでなく，使用目的，使用環境，使用頻度，介助者
の状況，他の生活行為との関連など，障害が重度であればあるほど
考えなければならないことが増え，適切な検討や試行錯誤が必要と
なる場合も多い．ここでは障害者用入力装置を，a.腕や足や頭や
息などの大まかな動きで操作できる入力装置，b.指先，唇，まぶ
たなどのわずかな動きで操作できる入力装置，c.脳波や脳血流な
どの身体的動きを伴わないで操作できる入力装置に分けて紹介す
る．
a.大まかな動きで操作できる入力装置 脳性麻痺による不随
意運動などに対応した比較的大型の操作端であり，足や腕での操作
の場合は作動力を大きくしたり機械的な遊びを大きくする．図9･
26は作動力の大きい押スイッチのベッド柵への設置例である．図
9･27は息を吹きかけると作動する呼気スイッチである．
b.わずかな動きで操作できる入力装置 高位頸髄損傷，筋ジ
ストロフィー症など多くの疾病では，操作力や可動域の減少を伴
う．わずかな動きを検出するためには，操作スイッチを可動部分の
そばの身体部位に固定し，固定位置と可動部の変化分を検出する方
法が多い．この方法であれば体全体が動いても検出に影響を与えな
いので，姿勢が変化したり不随意運動が出現する場合にも有効であ
る．図9･28は指先の動きの検出例，図9･29 は赤外線近接セン
サによるまばたきスイッチである．
c .身体的動きを伴わないで操作できる入力装置 難病の一つ
である筋萎縮性側索硬化症は，進行とともに四肢麻痺や呼吸麻痺を
