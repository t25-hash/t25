# 9・6・4 電話機

> **章**: 第9章 支援機器
> **節**: 9・6 コミュニケーション機器

c.重度障害者用意思伝達装置 極めて重度の肢体不自由者が，
1個ないし数個のスイッチにより操作可能な文章作成装置である．
PC画面上に50音文字盤が提示され，1個ないし数個のスイッチを
使い分けることで行と列を指定し，一つの文字を選択する．これを
繰返して文章を作成していくシステムである．文字選択法は行と列
を指定する方法が一般的であるが，そのほかにもモールス信号で指
定したり，b項で述べたらくらくマウスやトラッカープロ，ジョイ
スティックなどの接続も可能である．文章作成のみならず，インタ
ネットへ接続しメールを送ったり，周辺電化製品のオンオフや非常
が提供されている．
呼出機能などを有するものもある．オペレートナビ，キネックス，
伝の心などが市販されている．
