PETの特徴PET 検査ではまず、C-11,N-13,O-15,F-18などの陽電子放射性核種により薬剤を標識合成する(表4・2)。炭素、窒素、酸素は生体を構成する元素であり、フッ素は化学的性質が水素に似ているため、生体活性物質などに標識することが容易である。次にこの陽電子放射性薬剤を生体に投与した後、体外に放出する放射線をPET 装置で検知し、目的とする臓器における薬剤の分布や代謝動態を分析することで、がんを検出したり中枢神経疾患部位を発見したりする。検査を終えた後の体内放射能はすみやかに減衰し、被ばく量はX 線診断と比べて同程度かそれ以下である。陽電子(ポジトロン、positron)は、電子と同じ質量および電荷量をもつが電荷は正であり、放射性同位元素の壊変により生成する電子の反粒子である。陽電子は体内ですぐに止まり、直接検出することは難しいが、近くの電子と結合して消滅する際に、電子の静止質量分のエネルギー(511keV)に相当する極めて短い波長(2.43pm)の電磁波(光子)を二つ放出する(図4・42)。エネルギーの大きいこの光子は消滅放射線(annihilation radiation)と呼ばれ、互いに反対方向に飛び去るため、対向する二つのγ線検出器により一対の消滅放射線を同時に検出すれば、消滅した陽電子はγ線9 ―40医療・福祉・バイオ機器検出器を結ぶ線(同時計数線)上にあったことが同定できる。あらゆる方向の同時計数線を計測して得たデータから、X 線CT と類似の方法で陽電子放射性薬剤の体内三次元分布の画像を再構成する。陽電子放射性薬剤の比放射能が高いことから、非侵襲的な検査法として、他の映像法では達成不可能な高感度の測定が可能である。近年、がんの診断にF-18標識フルオロデオキシグルコース(FDG : fluorodeoxyglucose)検査の有用性が明らかになってきたことから、臨床での急速なPET の普及が見込まれる。b. 同時計数PET における同時計数には、真の同時計数、偶発同時計数、散乱同時計数の3種類が含まれるが、信号源は真の同時計数のみであり、偶発同時計数および散乱同時計数はノイズとなる。偶発同時計数とは、それぞれ独立した事象により発生した二つの信号が偶然、同時計数の時間窓に飛来する場合の計数である。検出器1および検出器2のシングル計数率をそれぞれ,とし、時間窓をとすると、偶発同時計数率は=である。同時計数回路をもう一つ用意して、入力信号を片方のみ遅延回路に通すと偶発同時計数が測定できる。放射能の強度に比例して真の同時計数率は増加するが、偶発同時計数率は放射能強度の二乗に比例する。散乱同時計数とは、消滅放射線(annihilation radiation)の一方もしくは両方が同時計数される前に散乱され、真の同時計数線からずれる場合である。検出器のエネルギー分解能を向上してエネルギーしきい値を上げることで散乱線を低減できる。c. 検出器PET 用検出器には、511keV の消滅放射線(光子)に対して高い検出効率と時間分解能をもつ必要性から、ゲルマニウム酸ビスマスBiGeO(BGO)などの実効原子番号の高いシンチレータ(scintillator)が用いられる。ちゅう密に素子を配列するために、4行8列や8行8列のシンチレータ素子配列を四つの光電子増倍管に光学結合した検出器ユニットが開発され使用されて図4・42 消滅放射線の同時計数による計測(a) 2D モード(セプタ繰出し)のデータ収集(b) 3D モード(セプタ撤収)のデータ収集いる。光電子増倍管の出力信号を用いて、アンガー方式の位置演算を行い結晶素子を同定するとともに、素子で吸収された放射線のエネルギー情報および時間情報が引出して、SN 比の向上を図る。PET 専用装置では、このような検出器ユニットをリング状に配置する。d. データ収集モードPET のデータ収集モードには、2D モードと3D モードの2種類がある(図4・43)。2D モードPET とは、体軸に直交する放射線のみを検出してスライス画像を得る平面計測法である。体を囲む多角形もしくは円形に沿って検出器をちゅう密に配列した検出器リングが一般に使用される。検出器リング内の検出素子間で同時計数データを収集し、投影データの計数を稼ぐ。検出器リングを多層に重ねることで同時に連続したスライスの投影データが得られるため、スライスごとに二次元画像再構成をして三次元画像を得ることができる。ただし、体軸と直交しない大部分(99%以上)の放射線が検出されない点は、多層検出器リングにしても同じである。各スライスごとに線源の検出領域を制限して、散乱同時計数や偶発同時計数を低減するため、検出器リングの間に円環状の鉛板(セプタ)を挿入する。3D モードPET とは、どの向きの放射線でも検出しようとするPET 独自の立体計測法である。PET の原理に立ち戻り、X 線CTの方法とは異なる三次元画像再構成法が研究開発されたため利用可能となった。
