# 9・7・1 排泄用機器

> **章**: 第9章 支援機器
> **節**: 9・7 排泄用機器と入浴用機器

人の排泄動作は，尿意や便意，括約筋のコントロールなどの神経
系の能力，衣服の脱着や後始末動作などに必要な体幹バランス機能
や上肢機能，トイレまでの移動や便器への移乗に使う下肢や体幹バ
ランス機能などの組合せで行われる．これらの能力のどこに障害が
あるかで，使用される支援機器が異なる．排泄支援機器（toilet
aid）は，排泄に伴う移動や移乗，衣服の脱着などの関連動作を支
援する機器，排泄に伴う上肢動作を支援する機器，身体の排泄機能
を支援する機器などに分けられ，脊髄損傷者などは一般に市販され
ている排泄関連機器や住宅設備機器，福祉用具，排泄方法に合わせ
た機器などを生活方法に合わせて選択し生活している．排泄にかか
わる動作は設計者にとって見えにくい部分であるため，開発が遅れ
ているのが現状である．ここでは，排泄行為を排便と排尿に分け
て，排泄支援機器の考え方とその例を記述する．
a.排便用機器 座薬挿入器，浣腸挿入器，洗腸器など排便を
促すための薬剤などを肛門に挿入するための支援機器，洋式便器へ
の移乗を支援する機器，便器周辺機器，身体装着用機器などがあ
る．

.臀部洗浄器 排便後の臀部を洗浄するために湯を噴射す
る機能をもつ便座や携帯用洗浄器などがある．
.ストーマ（stoma）用品 人工肛門に装着する用具で，
清潔・安全，無臭的に便を処理することができる．
b.排尿用機器 身体装着型収尿器，導尿用カテーテル，留置
カテーテル，手持ち式収尿器，排尿用の設備機器などがある．
.身体装着型収尿器 尿を漏らさずに集めることを目的に
作られたもので，男性用は多く使用されているが，女性用は実用に
堪えるものがないのが現状である．コンドーム型のような陰茎固定
型やベルト固定型，パンツ固定型などがある．
.導尿用カテーテル 尿意を感じたときや排尿時間ごと
に，カテーテルを尿道口から挿入し尿を出す導尿に用いる．頸髄損
傷者のような四肢麻痺で手指が任意に動かない者でも，手関節まで
動けば図9･43のような導尿セットを用いて導尿動作を自立して行
うことができる．自分でできない場合は，介助者が行うことにな
る．カテーテルを消毒容器に入れて携帯するタイプと，毎回使用後
に破棄する使い捨てのタイプがある．
.留置カテーテル 尿道口から挿入し，バルーンと呼ばれ
る袋に生理食塩水を入れて膨らませ外れにくくして，尿を体外に出
すカテーテルのこと．
.手持ち式収尿器 しびんのように受尿部と蓄尿部が一体
となったタイプと，受尿部と蓄尿部をチューブで連結したタイプが
ある．後者では，受尿部のセンサで尿を感知し吸引するものもあ
る．受尿部は男性用と女性用がある．
.設備機器 一般の小便器などの設置方法を変えること
で，自己導尿を自立させた男性用の例を図9･44に，女性用の自己
導尿設備を図9･45に示す．市販の幅の狭い小便器などを図9･44の
図9･45女性用の自己導尿設備
ように設置することで，自立性を向上し，排尿にかかる時間を軽減
させることができる．
