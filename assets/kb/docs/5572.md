# 9・8・1 自助具

> **章**: 第9章 支援機器
> **節**: 9・8 操作用装置

日常生活においては，例えば食事で箸やスプーンを使うように道
具を操作したり，小物を扱う作業は多い．そのため，そのような作
業がうまくできない上肢障害者にとっては，この不便を解消する操
作用装置は日常生活において必要不可欠なものである．このような
用途では，ハイテク機器や大規模な装置でなくても，簡単な道具で
十分に日常生活の補助に使える場合が多い．これらは自助具
（low-technology aids for manipulation）と呼ばれ，数百円から
数千円程度の価格の安さと便利さから広く普及している．
自助具は市販されているもののほかに，使用者やボランティアら
によって個別に製作されるものも多く，製作方法に関する書籍も数
多く出版されている．一般に市販品のほうがデザインに優れ，軽量
であるが，手作り品は材料費だけなので価格が安く，また個々のユ
ーザへの適合が容易であるという利点ももっている．
食事は日常生活活動（ADL:activities of daily living）の中で
も比較的実現が容易な機能であり，自立できる人は多い．しかし自
立できていない人にとっては食事自立のニーズは高く，各種の食事
用自助具はQOL（quality of life，生活の質）向上には欠かせな
い．食事が自立できない人のうち，握力が足りなかったり手先の細
かな作業ができない人には，柄を太くして握りやすくしたスプーン
［図9･49(ａ)］や，握らなくても手に固定して使えるスプーン［図
9･49(ｂ)］などが役に立つ．特に湯で変形させることができる形状
記憶樹脂は，使用者の手の形状に簡単に合わせることができるので
便利に使われている．すくいやすいように取付角度を工夫したスプ
ーン［図9･49(ｃ)］や，簡単に先が合うよう配慮された箸［図9･
49(ｄ)］，食器の縁が立っていてすくいやすい皿なども便利な道具
として使用されている．書字のためにペンを把持するための道具
も，食事用自助具と同様の考え方で実現された自助具が多く使われ
ている．一方，腕が持ち上げられないために食事ができない人の場
合は，BFO（balanced forearm orthosis ）やバランサ（図9･50）
を使って腕を支えることによって，自力で食事ができるようになる

ことも多い．
調理・炊事のための自助具では，片手で食材を切ることができる
ようにするため，まな板に食材を固定するためのピンが付いている
ものなどが使用されている．
比較的軽度な障害の人でも，更衣に苦労している人は多い．その
ため，更衣を補助する自助具もいろいろ使われている．ソックスエ
イドやストッキングエイドを使えば，足先まで手が届かない人でも
ソックスなどを簡単に履くことができる．指先がうまく使えない人
のためにボタンを簡単に着脱するためのボタンエイド（図9･51），
手が背中に回せない人でも背中のファスナを閉めることができる道
具などがある．
整容も問題のある人が多く，腕が上がらない人でも髪をとかせる
長柄のブラシや，細かい作業ができなかったり握力がなくても使え
る爪切りなどは，自助具の代表的な例である．
寝たきりの人や車いすユーザなどは，床に落ちたものや高いとこ
ろにあるもの，少し離れたところにあるものなどが取れないで不便
を感じることも多い．このようなときにはリーチャ（reacher ，図
9･52）やマジックハンドと呼ばれる自助具が役に立つ．さまざまな
製品があるが，軽いこととうまくものを取扱えることが選択の際の
要点となる．福祉用具として売られているものだけでなく，左ハン
ドル車用として売られているものの中にも便利に使える製品があ
る．
リウマチ患者などでは，ねじる動作が苦手でうまくできない人が
多く，そのような人のためにも各種の自助具がある．例えば握力が
不足してビンのふたが開けられない人には，軽い力でふたが開けら
れる自助具が各種販売されている．また，ねじらずに押すだけで開
閉できるふたをもつ保存容器も便利である．その他のねじる動作を
補助するものとして，ドアノブに固定したり巻き付けることによっ
て，ねじらなくてもドアの開閉ができるレバーや万能ハンドル（図
9･53）などがある．この万能ハンドルは，ばねのついた多数のピン
をねじるレバーに押し当てることにより，握力が弱い人でもレバー
を確実に固定してねじる動作が可能となる．また，ガスレンジの火
力調整レバーでは，ねじらずにスライドして火力調節ができる方式
のものも販売されている．この方法は直感的な火力判断の面ではね
じるタイプに劣るが，高齢者には負担が少なく使いやすい．
