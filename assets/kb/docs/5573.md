# 9・8・2 自立用ロボット

> **章**: 第9章 支援機器
> **節**: 9・8 操作用装置

食事や整容などの自立を図るためのロボット（robot ）は，欧米
を中心に若年重度四肢麻痺者に使われ始めている．ロボットの使用
によってあらゆることが自立できるようになるわけではないが，そ
れまで一人ではできなかった作業がロボットを使うことによってで
きるようになるだけで使用者のQOLが高まり，精神面にもよい影
響を与える．このようなロボットの基本的な考え方は自助具と同じ
であるが，自助具と呼ぶのはローテクノロジーによる機器だけであ
ると考える人が多く，ロボットは自助具と呼ばないことが多い．
代表的な自立用ロボット（assistive robots ）にオランダ製の
MANUS（図9･54）がある．ハンドの開閉を含めて8自由度をも
つはん用マニピュレータで，通常は車いすに取付けて使用される
が，簡単に着脱もできる．価格は約350万円であり，発売以来
2007年までの約15年でオランダ国内を中心に累計約300台が販売
された．操作は4×4個のスイッチの組合せで行うが，ジョイステ
ィックも使える．はん用で多機能な分だけ操作が少し難しく，意欲
のある若年障害者であっても習得には練習が必要不可欠であり，高
齢者には使いこなすことが難しく，高齢者の症例は報告されていな
いようである．実際の使用状況調査によれば，食事，歯磨き，ひげ
そり，軽量物の操作などさまざまな用途に使用されており，使用者
にとってはこのロボットなしの生活など考えられなくなるほど毎日
の生活に必要不可欠な道具になっているという．
HANDY-1（図9･55）はイギリス製の食事用マニピュレータで
ある．価格は約60万円であり，全世界で約300台が使われている．
トレー上の7箇所に分けて置かれた食物を好きな順で食べることが
できる．スイッチ1個だけと簡単な操作で食事ができるため，重度
障害児でもこれを使って食事が可能であるが，食べられるものは刻
み食だけに限定されており，例えばごはんをこれで食べようとする
と大きな塊が持ち上がってこぼしてしまう．オプションを使えば，
歯磨き，ひげそり，化粧などもスイッチ一つでできるようになる．
MANUSよりも機能は制限されるが，スイッチ一つで使いやすい
