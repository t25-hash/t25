# 4・7・4 磁気共鳴機能画像法(fMRI)と超高速撮影法

> **章**: 第4章 検査・診断機器
> **節**: 4・7 NMR・MRI 装置

磁気共鳴機能画像法(fMRI: functional magnetic resonanceimaging、機能的核磁気共鳴映像法)は、MR 画像を用いておもに脳機能を計測する手法を意味する。fMRIにはBOLD 法(bloodoxygen level dependency)とAST 法(arterial spin tagging)の二つの方法があるが、前者が主流である。BOLD 法は、神経細胞の活動に伴い毛細血管や細静脈で常磁性体である脱酸素化ヘモグロビン量が変化し、磁化率の変化が生じるため、見掛けの横緩和時間()が変化して信号強度の変動となって現れることを利用し、ある脳活動と関連する信号変化を検出する方法である。AST 法では神経活動に伴う血液の流速変化を検出する。fMRIでは、被検者に運動・認知機能を調べるための課題を実行させながら、に敏感なグラディエントエコー法に超高速撮像のためのEPI法(echo planar imaging)か、その変法であるスパイラル法(spiral imaging)を組合わせた撮像法を用いて連続撮影を行う。EPI法(図4・49)では、エコー信号が発生する時期に、断層面の,それぞれの方向に高速で方向転換する傾斜磁場を加えることにより、1回の励起で1枚の画像を生成するために必要な空間周波数領域のサンプリングを行う。EPI法を用いてfMRIを行うためには、以下の性能が必要となる。図4・50 スパイラル法のパルスシーケンス図4・51 指運動時の脳機能マップ図4・52 視聴覚提示装置と被検者応答記録装置(1) 1.5T 以上の静磁場(2) 超高速撮影が可能な性能:1回の励起により1画像が撮影(single shot)可能な傾斜磁場性能(>100mT/m/ms)、傾斜磁場の渦電流を抑制するアクティブシールド、および高速レシーバ(16bits,1MHz)(3) 十分な静磁場の均一性を確保するシム調整機構(4) 数千の画像が保存可能なデータベーススパイラル法(図4・50)は空間周波数領域において、低周波数領域を重点的にサンプリングするように傾斜磁場が操作され、空間周波数領域の二次元方向に対してらせん状にサンプリングを行う。データ収集後、それぞれのデータ収集点を格子状の空間周波数座標に変換したうえで、二次元フーリエ変換を行い画像を生成する。被検者の体動によるデータの劣化に強く、空間的な画像のひずみが少ない、傾斜磁場装置への負荷が少ない、原理的にコントラスト変化に対してより感度が高いなどの利点がある。被検者が課題を実行中に連続撮影を行って得られた画像データは、個々のピクセルの信号強度変化と課題実行との因果関係をANOVA(analysis of variance)などを用いて統計的に評価し、有意の相関が認められた部位が脳機能マップに表示される(図4・51)。解剖画像に重ね合わせたり、ボリュームレンダリングによる三次元表示も行われる。被検者に認知・運動課題を行わせるために、視聴覚提示装置や応答記録装置が必要となる(図4・52)。MRI装置内で使用するためには非磁性の材料を用い、撮影室内の回路には電磁シールドを施すか、光ファイバによる信号伝送を行うなどの対策が必要である。
