# 4・8・1 二次元・三次元スキャン

> **章**: 第4章 検査・診断機器
> **節**: 4・8 超音波診断装置

a. 概要人間の体は超音波に対して比較的透明であるから、体表から超音波を入射すると体内組織の音響特性インピーダンス(acoustic characteristic impedance、媒質の密度と伝ぱ速度の積)のわずかな相違によって一部が反射し、残りは透過する。この透過波は再び音響インピーダンスに差がある組織に到達すると反射と透過現象を繰返すので、原理的には吸収(熱拡散と散乱)によって完全に消滅する領域までの体内組織の情報を非侵襲で計測することができる。なお、通常の臨床用超音波診断装置には、体積弾性波(縦波)として体内を伝ぱするMHz 帯の超音波が利用されている。なお、超音波の伝ぱ速度は体内組織の違いや温度によっても異なるが、臨床用装置では固定値(=1530ms)が利用されている。パルスエコー法の基本はインパルス送波にあるが、超音波パルスエコー法の場合は、数波長分の超音波パルスを送波し受波信号と送波信号との積をとり(マッチドフィルタまたは共役検波)、ローパスフィルタを通してエコーの振幅成分のみを抽出している。例えば、超音波の送受波位置が同じ場合、超音波ビームと角度θ(ドップラー角度)をなして移動する赤血球や血しょう板などが生じるドップラー偏移周波数は次式で表現できる。これらの技術は、アレイ状に並べた微小振動子からの波動伝ぱの様子を説明したホイヘンスの原理によって理解することができる。まず、ビームフォーカス技術は、微小振動子の駆動時刻を調整することにより、各振動子からのパルス超音波が同時に収束点Pに到達するように制御する技術である。一方、ビームスキャン技術は、アレイ直下にビームフォーカスしながらアレイの端から端まで駆動する振動子群を変化させながら順にビーム送波するリニア走査と、アレイを扇の要として扇の竹骨の方向にビーム送波するセクタ走査に大別できる。. 画像表示法計測したエコー信号の表示法には、横軸に時間経過を縦軸にエコー信号の強度(amplitude)を表示するA モード、送波位置を固定しパルス飛行時間を縦軸として横軸にパルス送波時刻をとりエコー信号強度を輝度変調して反射体の動き(motion)位置を表示するM モード、ビームフォーカス方向とパルス飛行時間を対応させた二次元平面上にエコー信号強度を64~128階調のグレーレベルに輝度変調(brightness modula-tion)して表示するB モードがある。9 ―43また、画像の分解能には、縦分解能と横分解能の概念がある。パルスエコー法の場合には、前者は超音波の進行方向の距離分解能、後者は超音波進行方向に直交する方位分解能に相当し、それぞれパルス幅と超音波のビーム径に依存する。距離分解能の向上にはを高くして送波波長を短くするとよいが、到達可能距離は減少する。b. 2D(二次元)スキャナ。 B モード診断装置(B-mode diagnostic equipment)パルスエコー法とビームフォーカス技術、ビームスキャン技術、Bモード表示を応用したB モード診断装置は、体表から体内組織(臓器や腫瘍など)の形状を非侵襲的に二次元断層像として観察するために、最もよく利用されている。最近の装置は、フォーカスした超音波パルスを体内に送波し、体内からのエコーを受波して電気信号に変換するために小さな圧電素子(128個以上)を並べて一次元アレイ構造にしたプローブ(探触子)、各圧電素子を駆動するための発信器、圧電素子の駆動時刻を制御するための電子スイッチ、ビームを形成するビームフォーマ、計測した断面情報を画像として再構成するためのコンピュータ、画像化結果を表示するディスプレイからなる(図4・53)。このほか、計測の高精度化や計測目的また診断目的に合致した超音波パルス送波や画像生成のためのエンコーダや、デコーダおよびコントローラなどがある。また、音響インピーダンスが臓器や血液と極端に異なる骨や肺は超音波の伝ぱを阻害するので、体表アプローチの場合、音響窓(超音波を体内に伝ぱすることのできる体表上の部位)が狭い心臓検査にはセクタ走査、また音響窓が比較的広い腹部検査にはリニア走査(最近ではより広い領域が走査できるように振動素子を凸形に並べてリニア走査するコンベクス型プローブ)のプローブが使用される。このほかに、おもに血管内超音波(血管壁断面像が観察可能)分野で利用する体腔内アプローチのために、アーク走査を採用したカテーテル型プローブもある。. コントラストエコー装置(contrast echo equipment)超音波エコーの強度を増強して明りょうな画像を得るための超音波エコー増強剤とB モード診断装置を併用する手法に、コントラストエコー法がある。従来の装置は、微弱超音波と組織の相互作用を基礎とした線形現象(伝ぱに伴う超音波の搬送周波数は変化しな図4・53 超音波診断装置のブロック図9 ―44医療・福祉・バイオ機器い)を応用していたのでその効果はあまりなかったが、最近、送波パワーを増大すると伝ぱする波形がひずむ非線形現象(搬送周波数に高調波成分が含まれる)を応用したハーモニックモード診断装置が開発されている。まず、組織ハーモニック法(tissue harmonicmethod)は、送波パワーがフォーカスビームの周辺部より強い中央付近に搬送周波数(基本周波数)の高調波成分が重畳されることから、二次高調波を選択検出してよりシャープなビームを形成して画像方位分解能の向上に寄与することを目的としている。なお、この効果を強調するために位相が異なるパルスを順次送波する手法(パルスインバージョン技術)も考案されている。一方、バブルハーモニック法(bubble harmonic method)は、含気性超音波造影剤(バブル)を体内に静脈注射し、送波超音波との相乗効果でバブル自体が発生する高調波成分を検出する手法である。これらの手法は、B モード診断装置の画像分解能や画質の向上を目的として開発され、未統一の技術名称や学術用語が残る新しい分野であるが、今後、急速に普及することが予想される。. ドップラー診断装置非侵襲で血流速度が計測可能な血流ドップラー法を応用した臨床用装置には、連続波ドップラー血流計、パルスドップラー血流計、カラードップラー装置がある(図には減少するため、これらの装置には、移動方向を知るために直交位相検波方式と周波数分析法(高速フーリエ変換、FFT)を組合わせた方向判別手法が利用されている。まず、連続波ドップラー血流計(continuous wave Doppler flowmeter)は、送波用プローブから超音波を連続的にビーム送波し、受波用プローブの検出ビームとの交差域上で移動している血流速度情報を計測する装置である。FFT 分析に必要な時系列は2以上のサンプル周波数で収集できる。また、ドップラー効果は1波長以上のパルス超音波を送波する場合も観測できるため、パルスエコー法と組合わせて計測場所を限定し、同じ方向に繰返し超音波パルス(パルス繰返し周波数)を送波して血流速度情報を計測する装置がパルスドップラー血流計(pulsed Doppler flowmeter)である。パルス送波の後は、計測場所までのパルス飛行時間が経過した後にエコーをサンプリングすればよいので、送受波用振動子を一体化したプローブを利用する。さらに、パルスドップラー血流計のエコーをきめ細かくサンプリング(150箇所程度)し、各反射場所に対応する時刻のデータに並べ替えることにより一次元上に分布するパルスドップラー信号を得て、次にビームスキャニング技術を応用して観察方向を変化させながら断層面内での血流パターン(探触子に近づく流れを赤色系、遠ざかる流れを青色系のカラー輝度情報に変換)をリアルタイムで計測する装置がカラードップラー装置(Doppler color flow equipment、図4・54)である。超音波パルスの同方向への繰返し送波、超音波ビームの右から左への順次走査という二つの操作が必要となることから、周波数分析に必要な時系列が短くて済む瞬時周波数法が利用されているが、雑音低減のために同一方向に約8回程度のパルス発射が必要であるため、走査方向数をB モードエコー法の半分に減少しても画像計測レートは4分の1程度となる。画像表示法には、各計測部位の平均流速に対応する平均ドップラー偏移周波数を表示するドップラーモードと、ドップラー角度や流速の遅さに依存しないドップラー信号強度を表示するパワーモードがある。c. 3 D(三次元)スキャナ三次元スキャナには、コンピュータ処理法、リアルタイム超音波ビームトレーシング法、拡散音響レンズ法を採用した方式がある。まず、多くの断層像(B モード像やカラードップラー像)を並べて立体化するコンピュータ処理方式の装置は、三次元像の表示、任意断面の表示、三次元空間での距離計測・面積計測・体積計測、血流情報から推定した血管走行、結果の保存・複写・伝送などの諸機能を有している。しかし、B モードでも約30ms(走査本数150)のリアルタイム画像が限界であるから、超音波ビームスキャン断面の厚さを考慮して30枚程度のB モード像から三次元立体像を生成する、関心領域を狭く設定するなどの工夫により、短時間で計測・処理が可能なリアルタイム化を実現している。また、心臓など動きの速い臓器を対象とする場合には、心電同期をかけながら二次元アレイプローブによってランダムな方向を走査して計測ビーム数を減少させ、かつ以前の情報で補完して徐々に鮮明な画像を構築していく方法もある。一方、すべての空間を収集して立体モデルを作成しないで、必要な情報のみの表示を目的に情報処理時間を短縮した手法が、超音波ビーム上の最強エコー部位を表示するリアルタイム超音波レイトレーシング方式で、現在、毎秒4フレーム程度の立体画像の計測表示に成功している。しかし、現在の超音波医用画像診断装置に利用されている送波ビーム形成方式を使用する限り、X 線や光の伝ぱ速度に比べて伝ぱ速度が極端に遅い超音波においてはパルス飛行時間が無視できなくなることから、われわれが日常的に眼で観察しているような三次元情報の瞬時映像方式の研究開発もなされている。d. アーチファクト実際の超音波画像にはアーチファクト(実際とは異なる状態を画像化した表示エラー)が含まれるため、十分に注意する必要がある。B モード診断装置の代表的アーチファクトには以下のものがある。(1) フォーカス音場パターンが目的方向(メインローブ)だけでなく他方向(サイドローブ)にも広がるため、サイドローブ領域に存在する反射体からのエコーがメインローブ内に存在するものとして表示される。(2) 走査断面厚が厚くなり、異なる断層面情報が重なって表示される。(3) 反射強度が大きい物体が超音波の進行方向と垂直に存在すると多数回の反射が起こり、パルス飛行時間に対応した場所に表示される。(4) のう胞や血管などは鏡面反射するため、円形断面や斜めに走行する血管像が表示されない。
