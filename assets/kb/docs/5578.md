# 4・9・2 軟性内視鏡(flexible endoscope)

> **章**: 第4章 検査・診断機器
> **節**: 4・56 )、1.9mm(新生児用尿道鏡、カールストルツ社K 27301

硬い光学視管を有する硬性内視鏡に対して、全体の屈曲が可能な内視鏡を軟性内視鏡(軟性鏡)と呼ぶ。軟性内視鏡にはファイバスコープ(fiber scope)と4・9・3項で後述する電子スコープがあるが、文献によっては両者をまとめてファイバスコープと呼んでいる場合も見受けられる。なお、胃カメラと俗称されることもあるが、本来の胃カメラは小型の銀塩カメラを胃中に挿入して撮影するものであり、現在は用いられていない。ファイバスコープは、画像伝達系として光学視管の代わりに極細径のグラスファイバあるいはプラスチックファイバを束ねて用いられ、手元のCCD カメラにより画像化される。照明は硬性内視鏡と同様にライトガイドによって導入される[図4・55(b)].ファイバスコープはファイバの径と数により外径と画質が決定される。一般的なファイバ数は数千から数万であるが、より少数のファイバを用いることで、外径1mmを切るような細径のスコープが可能となる。一方でファイバ数は得られる画像の実質的解像度になるため、細径化は画質の低下に直結する。またファイバの網目模様や色吸収の影響も存在する。さらに外力によるファイバの屈曲破壊という耐久性の問題が敬遠され、かつてさまざまな臓器の観察に用いられていたファイバスコープは、電子スコープの画質の急速な向上に伴って、一般的な検査における役割を電子スコープに譲りつつある。現在のファイバスコープの用途は、電子スコープの挿入が不可能な細い管腔臓器内部の観察に移っている。気管支や膵管、胆管、血管内の観察においては主流であり、例えば膵管鏡として外径0.8mmのファイバスコープが存在する[オリンパス(株)PF type8P].電子スコープのチャネル内にファイバスコープを組込んだデュアルスコープを構成し、観察対象付近まで電子スコープで誘導した後にファイバスコープで観察するという利用方法も多い。図4・55 各種内視鏡の機構図4・56 硬性内視鏡(カールストルツ社HOPKINS II、上:直視鏡、下:斜視鏡)9 ―46医療・福祉・バイオ機器図4・57 鼻咽喉用電子スコープ[ペンタックス(株)VNL-1330]
