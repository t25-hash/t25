# 10・1・2 ジェロンテクノロジーの研究対象

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・1 概説

ジェロンテクノロジーの研究対象として，以下の5点が言及され
ることが多い ．
(１)加齢の解明
(２)機能低下の防止
(３)機能低下の補償
(４)高齢者にふさわしい活動の向上
(５)介護者の支援
加齢の解明は，加齢に伴う高齢者の機能低下の理解や高齢者特有
の生活行動特性を明らかにする基礎的な研究である．今後，特に社
会の情報化が進展することが予想されること，および高齢者の認知
特性の低下が外出等の社会参加の減少に関連しているとの報告があ
ることから，高齢者の認知特性解明に対する取組みが重要とな
る ．
それに対して(２)以下は，(１)の知見を利用して具体的に高齢者
の生活を改善するための応用的な研究といえる．まず機能低下の防
止では，身体的な機能低下を防止する技術を開発することを目指
す．例えば，高齢者が日常的に行う運動処方を確立することによ
り，筋力等の低下を防ぐ研究をする．機能低下の補償では，低下し
てしまった機能を補償する技術を開発する．例えば，高齢者の低下
した視覚機能を補償するために効果的な眼鏡の研究や，聴力低下に
対する補聴器の研究をする．また，このような身体的な機能そのも
のの補償だけではなく，高齢者が使用する製品や環境を高齢者に使
いやすく行動しやすいものにする研究もここに含まれる．近ごろよ
く話題になるユニバーサルデザイン（UD:universal design）は，
高齢者にも使いやすいデザインを考えることにより，間接的に高齢
者の機能低下を補償することをねらいとしている．高齢者にふさわ
しい活動の向上に関しては，高齢者に独特な生活行動支援に関する
技術開発が関係するが，高齢者に配慮した自動車などの移動システ
ムや就労環境の創出なども，間接的に高齢者の生活行動に広がりを
もたらすことからこのカテゴリーに入るといえる．介護者の支援に
ついては，福祉工学と重なるところであり，介護が容易な浴室やベ
ッドの設計など多くの該当例がある．
