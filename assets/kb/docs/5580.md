# 10・1・3 ジェロンテクノロジーにおける基盤研究

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・1 概説

10･1･2項で述べた5項目の研究対象はすべて重要なものである
が，それだけで十分とはいえない．特に，加齢の解明の結果として
得られた知見を(２)以下の応用技術に生かしていくには，両者をつ
なぐ基盤的な研究が重要な要素となる．基盤的な要素として以下の
3点が重要である．
(１)知識を利用しやすいように構成したデータベースの構築
(２)上記のデータベースを応用的な場面で使いやすくしたヒュー
マンモデリング技術の開発
(３)データベースや人間特性測定法を簡明に解説したガイドライ
ンや規格の作成
データベースの構築では，高齢者特性の解明のための実験や調査
で得られたデータを系統的に整理・管理することにより，必要な統
計処理をした情報の高速検索を可能にする．適切なデータベースを
利用することで，高齢者特性に配慮した製品や環境等の設計が容易
となる．
ヒューマンモデリング技術は，計算機上で動く人間モデルを構築
し，同じ計算機上の製品・環境モデルの中で行動させることによ
り，製品や環境が人間特性に適合しているかを評価するものであ
る．これによって，データベースをより組織的に使えるようにな
る．この技術は現在発展途上にあるが，完成すると人間によるモニ
タ実験では安全上できない評価も可能となり，製品設計が迅速かつ
効果的なものになる．
また，ガイドラインや規格の作成では，データベースの内容を簡
使いやすい台所の設計，全体として段差をなくして将来の車いす使
潔にまとめ，設計の基準を具体的に示すことや仕様の統一化によ
り，誰でも簡単に人間特性に配慮した設計ができることを目指して
いる．製品流通の国際化の観点からも，高齢者の特性を盛り込んだ
グローバルな規格作成に取り組むことは，今後のジェロンテクノロ
ジーの重要な基盤要素になる．
