# 4・9・5 内科以外で使われる内視鏡(顎関節鏡など)

a. 内視鏡の分類内視鏡は,体内の管腔を通して光学的に直接観察する医療機器である.体腔以外でも組織げき(隙)などに気体,水を注入して空げきを広げて検査,診断を行うことのみならず,近年は内視鏡で病変部位を観察しながら処置を行う低侵襲手術(minimally/less invasive surgery)用の機器として治療にも広く応用されている.その構造上,広義には軟性内視鏡と硬性内視鏡に大別される(表4・3).軟性内視鏡は,グラスファイバを束ねた導光ファイバを通して体外の光源装置から熱をカットした光(コールドライト)を送って,先端の視野を照明下に観察する構造である.一方,硬性内視鏡は,直達鏡ともいわれ,金属管の中に光学レンズ系あるいはファイバを挿入したものがある.両者とも管径は約1.0mmから数mmまで,その用途部位により各種あり,最近ではスコープ先端に個体撮影像素子CCD を組込んで,モニタテレビを観察する電子内視鏡(電子スコープ)が普及しつつある(図4・59).食道,胃,十二指腸,大腸などの消化器を対象にした内科や,鼻咽腔,喉頭,副鼻腔を対象にした耳鼻咽喉科,さらに気管,気管支を対象とした麻酔科で使用される内視鏡は,ファイバスコープと呼ばれる屈曲可能な構造で,生体内にもともとある体腔の形態に適合させやすく,口腔,鼻腔,副鼻腔,消化管に適用され,軟性内視鏡に属する.一方,軟性内視鏡が適用されない膝関節,股関節などを対象とした整形外科や,顎関節を対象にした歯科口腔外科で使用される関節鏡(図4・60),あるいは消化器,膀胱,子宮などを対象とした腹腔外科で使用される腹腔鏡(図4・61)や,気管,心臓,肺などを対象にした胸部外科で使用される胸腔鏡は,硬性内視鏡に属する.これらは観血的に体内の組織内や臓器内へ小皮切開を通し穿刺,挿入して用いられる手術器具としての性格をもち,まっすぐな円筒状の表4・3 硬性内視鏡よって開発され,改良を重ねて1877年にニッチェ(M.Nitze)によって完成された.関節鏡は膀胱鏡を膝関節に適応したのが最初で,1918年にわが国で高木によって創始された.当初は,腹腔や胸腔などの消化管以外にも関節鏡や膀胱鏡が応用されていたが,その後,周辺機器を含めて構造に改良がなされ,1990年代に内視鏡下手術用の硬性鏡の製品化に至った.硬性内視鏡の構造上の特徴として,生体の組織内,臓器内への穿刺,挿入操作が可能な強靱性,耐久性があり,滅菌操作ができることが挙げられるが,先端に照明が内蔵され術野の腔を拡大するための気体,液体などの導入孔などが一体化されているものもある.硬性内視鏡の体内への挿入は,膀胱鏡,膣鏡以外は皮膚切開ののち組織内を経て関節腔,あるいは腹腔,胸腔へ到達させる.関節腔は生理食塩液を灌流させながら,また腹腔,胸腔は空気などを注入し,術野を拡大し操作する.内視鏡での手術は低侵襲手術で行えるという特徴から,現在では各管腔臓器にその適応が広がっている.内科以外で使われる内視鏡のうち,顎関節鏡についてb項で解説する.b. 顎関節鏡顎関節の内視鏡検査は,顎関節鏡を使用して顎関節腔内を観察する検査法で顎関節鏡視法と呼ばれ,1974年に大西によって開発された.さらに,その治療的応用として関節腔内を観察しながら病変部の外科的手術を行う処置が顎関節鏡視下手術に9 ―48医療・福祉・バイオ機器よって実現化された.関節鏡視,関節鏡視下手術に必要な器具として,関節鏡(針状硬性鏡)本体のほかに,照明のための光源装置,映像用の内視鏡テレビ装置,テレビモニタ,穿刺のための付属器具,内針と外套管があり,鏡視下手術器具としては,鉗子(かんし),鉤(こう)類,各種関節鏡メス,レーザメス(Nd:YAG,Ho:YAG など)のレーザ導光用クオーツファイバのほかに,削除用ドリル(いわゆるシェーバで,削除組織片の吸引装置付き)などがある(図4・60).内視鏡下手術器具は,一般関節鏡以外の他の外套管を通して関節内で使用される.また顎関節の手術用関節鏡として,関節鏡筒と手術器具が1985年に開発された.さらに関節鏡と並行に手術器具が同時挿入可能な2チャネルの外套管を組合わせた関節鏡も2003年に開発されている(図4・62).顎関節疾患の外科的治療として,従来より行われている皮膚切開による関節開放手術の施行症例の中にも顎関節鏡視下手術で代替可能となる対象が広がり,大きな手術侵襲を回避できることになったのは福音である.特に顎関節の開放手術は,病変部に到達する際,解剖学的に顔面神経麻痺の合併症の危惧が懸念されるほか,関節包の切離におけるはんこん(瘢痕)拘縮による二次的な顎運動障害などの後遺症が発症する可能性が高い.顎関節のように狭小かつデリケートな構造で限局した病変においては,その病変部の切離や切除処置よりも関節開放のための外科的侵襲のほうが大きい.したがって,内視鏡の刺入により必要な部分のみの選択的な処置が可能な内視鏡下手術が安全確実に行われれば,その有用性は極めて高い.c. 内視鏡下手術の臨床的役割前述のように顎関節鏡視下手術を例としてその意義を述べたが,内視鏡下手術の治療の対象が関節のみならず腹腔,胸腔,膀胱尿路などをはじめとする最近の外科系各科での内視鏡下手術の発展とも相まって,低侵襲手術としてその有用性が注目されている.内視鏡下手術の用途をまとめると,病変部の観察,病変部の生検などによる診査・診断,および病変部そのものの切除のための電気焼灼,蒸散処置による外科的治療がある.(a) 関節鏡全体像[オリンパス(株)と大西らの共同開発]と関節鏡先端(右上).外套管には関節鏡と手術器具を並行して挿入することができ,鏡視時に器具先端は常時,有視下にあり,確実に操作しうる(b) 2チャネルの顎関節鏡による鏡視下手術のシェーマ軟性内視鏡による内視鏡下手術は,小さな病変に適応して体腔そのものを経路し,病変部を観察しつつ処置を行え,切開は不要である.一方,硬性内視鏡による内視鏡下手術は,切開が小さく,したがって術後の切開創が目立たないなどの利点がある.以上の内視鏡下手術に共通する利点は,術後の痛みが少なく,早期の社会復帰が可能なことなどであり,関連各科での適用,さらに内視鏡および周辺機器の積極的な改良が産学共同で行われている.このように内視鏡は,低侵襲の検査・診断および治療法として脚光を浴びているが,その一方で視野が狭いなど適用条件に限界もある.疾患の適応症を厳密に選択し,十分な手技を習得した術者によって行われて初めて正確で安全な操作が可能となり,低侵襲手術という目標が達せられるものであることを忘れてはならない.
