# 10・1・4 ジェロンテクノロジーの具体的な内容

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・1 概説

ジェロンテクノロジーの内容を模式的に3層に構成し，要約した
ものを図10･1に示す．最上層はジェロンテクノロジーの精神であ
り，究極的な目標を示している．すなわち，高齢社会を支えるに
は，自立と生きがい寿命の確保が必要不可欠で，それを人間関係の
豊かな共生社会の中で実現しようとするものである．
その下の中層に応用技術モジュール群がある．ここでは，上層の
目標を実現するために，10･1･3項で前述した下層の基盤研究の成
果を利用しつつ，生活全般に役立つ高齢者の低下した機能を補助・
支援する技術開発を行うとともに，生活活動の向上に役立てる具体
的な製品・環境・情報を開発する．例えば，健康の維持・確保に始
まり，高齢者に適合した住居，移動に関連する施設・機器，就労環
境，製品・情報の設計に加え，高齢者の生きがいを支援するシステ
ムの開発などがある．住居に関しては，事故の起こりにくい浴室や
図10･1 ジェロンテクノロジーの模式図
用時に備えた設計などがある．移動に関しては，高齢者特性に合っ
た自動車の運転操縦システムの設計や，見やすい交通標識を含めた
道路設計がある．また，就労関連では，低下した視力や聴力に対応
できる情報提示装置や，筋力低下や操作の巧緻性低下に対応するた
めの操作機器の設計が，職務の再設計とともに行われている．
