# 4・10・1 血糖値検査

> **章**: 第4章 検査・診断機器
> **節**: 4・10 光計測装置

光を利用した血糖値(blood glucose、血液中のグルコース濃度)検査法の開発は、その優れた非(低)侵襲性から、大きな期待が寄せられている。特に、世界中で約1億人ともいわれるインスリン依存型と非依存型の糖尿病患者の治療に際して、患者の血糖値を常時モニタしていくことは非常に重要である。このため、痛みも感染性もない、非侵襲(non-invasive)的な測定法の開発には大きな期待が寄せられているのである。この開発途上にある光を使った血糖値測定システムとして、以下に述べるようにおもに赤外領域の計測技術を紹介する。a. 赤外吸収分析(infrared absorption spectroscopy)ここで使われる赤外線の吸収測定技術は、通常赤外(あるいは中赤外)領域(波長:2.5~25 m、波数:4000~400cm)と近赤外領域(波長:700nm~2.5 m、波数:14000~4000cm)で適用される。この波長領域でも定量測定では通常のランベルト・ベール(Lambert-Beer)の法則が適用され、吸光度(absorbance)と溶質濃度の比例関係により計算される。通常、赤外領域では分子振動スペクトルを測定できるので、グルコースなどの糖質の分析ではOH 基の伸縮振動由来の3400cm付近の吸収やC-O結合由来の1040cm付近の吸収などがおもに観察できる。血液中のグルコースなどのように水に溶けている場合、HO自体のOH 基などの赤外吸収が大きいので、その中の糖質の赤外吸収を測定する場合は背景としての水の赤外吸収を差引く(subtraction)必要がある。このときは、ATR(attenuated totalreflectance、全反射)測定法を使うことで、糖を含む溶質の赤外スペクトルを有効に測定できる。このATR 法では、セレン化亜鉛(ZnSe)やゲルマニウムなどの結晶の中を赤外線が全反射しながら通過するときに、結晶の外面に生じたエバネッセント波(にじみ出た赤外線のエネルギー)でその結晶に接している物質(血液や血清、皮膚など形態を問わない)の赤外吸収を測り、水の吸収を最小限にして、吸収スペクトルの引算処理(subtraction)によって精度よく溶質成分の赤外スペクトルを測定することができる。このATR 法では、ZnSeなどの結晶表面に接している物質の表面からせいぜい数m以内の化学的組成の情報しか得られないが、口腔粘膜(oral mucosa)や組織細胞表面の物質代謝の計測には有効である。図4・63(b)に、ZnSe結晶のロッドに口腔粘膜を押し当てて測定した赤外スペクトルを示した。糖のC-OH 結合由来の吸収が1000~1100cmの領域に、OH結合の変角振動由来と考えられる弱い吸収が1350~1450cmの領域に見られる。糖の赤外吸収は非常に弱いので、血清中の脂質やタンパク質などのグルコース以外の成分による赤外吸収からの干渉が大きくなる。このATR 法を用いて血清中の糖の量や、口腔粘膜を通して血糖値を測定する方法などが試みられてきた。図4・64(a)にグルコースの赤外吸収スペクトルの一部を示す。この通常赤外領域では、糖のスペクトルの微細構造が把握できるが、一方、近赤外領域で図4・62 2チャネルの顎関節鏡(a) 光学系実測例(b) 口腔粘膜実測例図4・63 ATR の光学系と口腔粘膜実測例(a) 赤外(b) 近赤外図4・64 グルコースの吸収スペクトルは、図4・64(b)に示すようにかなりブロードな吸収になる。近赤外分析による血糖値測定では、700~2500nm(14000~4000cm)の波長領域を用い、C-H やO-H 結合伸縮振動由来の基準振動の倍音や結合音振動スペクトルを測定する。この領域の近赤外の透過あるいは拡散反射光を測定するに際し、皮膚や口腔粘膜の近赤外光の透過性がよく、比較的深部を流れる血液からの化学的情報をとらえることが可能であるが、皮膚などの生体組織成分(タンパク質、脂質等)からの赤外吸収による干渉が大きく、またそれ9 ―49らの個人差も大きく、比較解析するうえで困難が生じる。現在、臨床用の非侵襲的血糖値検査機器としては、口腔粘膜を対象にしたものや、指や上腕部内側を対象にしたものなどが開発されている。b. その他の分析方法上記以外では、ラマン散乱分光分析(Raman spectroscopy)、偏光分析(polarimetric analysis)、蛍光分析(fluorescence analysis)などが利用される。特に偏光分析では、眼房水を対象に糖分子のキラリティに基づく旋光分散を測定し、糖の定量分析が行われ、非侵襲的には前眼房に光を当てて、透過光を使って糖量の計測も行われた。蛍光分析はグルコース酸化酵素を使う特異的な検出法であり、発生する過酸化水素を蛍光物質で検出し間接的に糖を定量する。
