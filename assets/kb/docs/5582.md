# 10・1・5 ジェロンテクノロジーの今後の展開

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・1 概説

ジェロンテクノロジーの技術基盤が整備され，人間特性データベルヒューマンモデルが開発できれば，それを駆使することで，われ高齢者も若者と同様，生活機能を高める新製品を自由に使いこなし，快適な生活を送ることが期待できる．

その結果，従来の高齢者に対する社会的な行動規る．
