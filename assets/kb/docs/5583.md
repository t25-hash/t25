# 4・10・2 ヘマトクリット

> **章**: 第4章 検査・診断機器
> **節**: 4・10 光計測装置

血液の全容積に占める赤血球の相対的容積比率をパーセントで表したものをヘマトクリット値(hematocrit)といい、通常の血液検査では末静脈血、耳朶血などの末梢血で測定する。正常値は成人男性で39.6~50.8%、成人女性で32.8~44.7%(赤血球容積演算法)である。ヘマトクリット測定のおもな目的は、貧血および多血症の診断や、赤血球絶対値の計算である。多くの場合、ヘマトクリット値はそのまま貧血あるいは多血のパラメータと考えられるので、ヘマトクリット値が低い場合には酸素の運搬を行っている血液中の赤血球の相対的容積が低下していることを意味するため、生体には貧血などの症状が疑われる。一方、ヘマトクリット値が高い場合には、生体には多血症、赤血球増多症などの症状が疑われる。ただし、血液の希釈(過剰輸液、浮腫などの水血症)、あるいは血液の濃縮(激しい嘔吐、下痢、強い発汗などによる脱水状態)を伴う病態があって、体の循環血しょう量が大きく変動すると見掛け上の貧血や多血を呈するので、診断を行う場合にはヘマトクリット値のみならず赤血球数とヘモグロビン濃度等も測定し、総合的に判断する必要がある。臨床で用いられるヘマトクリット値の測定方法は、遠心分離機を使用する遠心法と自動血球計測器を使用する方法に大別される。遠心法は、さらにウイントローブ(Wintrobe)法と毛細管ヘマトクリット法に分けられる。このうち最も標準化されているのはウイントローブ法であるが、測定操作が簡便で能率がよいことから、毛細管ヘマトクリット法のほうがより広く使われている。ウイントローブ法は、採血した血液をウイントローブ管にとり、凝固阻止剤を加えて回転数3000minで30分間遠心分離を行う。そして、遠心分離後に血球層と血しょう層に分離した液柱の高さを100とし、赤血球柱の高さをパーセントで表すことでヘマトクリット値を計測することができる。毛細管ヘマトクリット法は、採血した血液を毛細管にとり、専用の高速遠心機を用いて回転数12000minで5分間遠心分離を行う。そして、ウイントローブ法と同様に遠心分離後に血球層と血しょう層に分離した液柱より赤血球の容積比率を求める。毛細管ヘマトクリット法は、ウイントローブ法と比べて簡便で遠心時間が短くて済むため、広く臨床検査に用いられる。自動血球計測器を用いたヘマトクリットの計測は、電気抵抗方式がおもに用いられている。電気抵抗方式は、血液を一定量に希釈したのち、電解液中に血球を浮遊させ、電流の流れる小孔(機種により異なるが通常100 m)を絶縁体とみなせる血球が通過する際に、血球の容積と電気抵抗の変動が比例していることを利用し、赤血球と血小板を識別して赤血球容積を電気的なパルスとしてとらえる。そして、計測された赤血球容積の総和を試験に使用した血液量で除すことで、ヘマトクリット値を求める方法である。さらに採血を必要としないヘマトクリットの連続計測を行うために、赤外光を用いた光学的計測方法も開発されている。これは、血液チューブを赤外光の発光部および受光部間に挟み、発光部から光が受光部に達するまでどの程度減衰するかを測定することでヘマトクリットの計測を行う方法である。この透過光量の減衰は赤血球に含まれるヘモグロビンによるものである。波長には、図4・65のよ9 ―50医療・福祉・バイオ機器図4・65 酸素化ヘモグロビン(HbO)と脱酸素化ヘモグロビン(Hb)の光吸収率図4・66 血液の光学濃度(O.D.:optical density)とヘマトクリット(Hct)うにヘモグロビンの酸素結合状態によって光吸収率が異なるという性質があるため、酸素化ヘモグロビンと脱酸素化ヘモグロビンの光吸収率が等しくなる800nm付近の赤外光を使用して、酸素飽和度の影響を受けずに透過光の光学濃度を連続計測する。このとき、図光がヘマトクリット15~85%の間ではほぼ一定値であるので、計測された透過光の光学濃度よりヘマトクリット値を一意に求めることができる。
