# 4・10・3 酸素飽和度

> **章**: 第4章 検査・診断機器
> **節**: 4・66 のように,吸収光がヘマトクリット値の増加に比例し,散乱

酸素飽和度は血液中のヘモグロビンが酸素と結合している割合をパーセント表記したものである.ヘモグロビンは酸素と結合して酸素化ヘモグロビンとなり,肺から取込んだ酸素を脳や心臓,腎臓,筋肉など全身に運ぶ役目をする.酸素飽和度の正常値は97~99%だが,高齢になると正常でも95~96%に低下する.そして90%以下になるとチアノーゼを呈し始め,50%以下になると組織損傷を起こし,30%以下になると細胞の死を招く.酸素飽和度の測定は,光を吸収する物質の濃度を測定するものの原理として用いられているランベルト・ベールの法則を利用する.つまり,波長の異なる640nm付近の赤色光と,800nm付近の赤外光を図4・67のように測定対象となる血液に照射し,酸素化ヘモグロビンは赤色光よりも赤外光をよく吸収するが,脱酸素化ヘモグロビンは赤外光よりも赤色光をよく吸収するという特性を利用することで,酸素化ヘモグロビンと脱酸素化ヘモグロビンの相対的濃度を求めることが可能である.さらに,この原理を応用し,血液を測定器に近づけることなく体外にバンドやクリップを巻き付けて測る簡単な検査法で動脈血酸素飽和度を計測する機器がパルスオキシメータである.パルスオキシメータでは,体外より光を照射するため,照射波は動脈血で吸収さ図4・67 光センサによる酸素飽和度計測センサ部図4・68 パルスオキシメータの計測原理れるだけでなく,皮膚をはじめとする組織や静脈血でも吸収を受ける.また血管壁のような境界面では光が屈折,反射を起こし,ランベルト・ベールの法則が当てはまらなくなる.そこで,図4・68のように,血管以外の組織と静脈血による光の吸収は心拍に関係なくほぼ一定であることを利用し,吸光全体から変動しない不変な成分を数値的に差引けば,組織や静脈血による吸収の成分は除去され,血管内で脈動している動脈血による吸収成分のみが残る.これが酸素飽和度を示す.
