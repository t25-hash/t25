# 4・10・4 光トポグラフィー(optical topography)

> **章**: 第4章 検査・診断機器
> **節**: 4・66 のように、吸収光がヘマトクリット値の増加に比例し、散乱

a. 概説近赤外光(near infrared light)を用い、脳血液や代謝の変化を非侵襲かつ簡便に計測し画像化することにより機能診断を目的とした光トポグラフィーがある。光による脳機能計測法は従来より使用されているfMRI,PET やMEG(脳磁図)等に比べて、以下の特徴がある。(1) 無侵襲な計測のため被検者に負担がかからず何回でも計測できる。(2) 小型で安価。ベッドサイドでの計測も可能。(3) 計測時の制限が少なく、少し動いても計測可能なため、複雑な負荷刺激を与えることができる。(4) 他の計測法との干渉が少ないため、MRIやEEG との同時計測が可能。図4・69に装置の概観を示す。b. 計測原理近赤外光は皮膚や頭蓋骨の透過性が比較的高く、頭蓋骨のすぐ内側にある大脳皮質の血液変化を頭皮上から計測することができる。血液中に含まれるヘモグロビンの光吸収特性を利用して、二つの異なった波長の光を同時に照射することで、酸素化および脱酸素化ヘモグロビン量変化を計測する(図4・70,4・71)。照射と検出位置が30mmの場合、光の広がりは半値幅で約25mm、計測中心深さは20~30mmと考えられている。光トポグラフィー装置は光の広がりを考慮し、帯域の狭いレーザ光を使用している。多チャネルの受信信号を計測する方法として照射位置ごとに時間分割照射をする方法があるが、チャネルを増すごとに計測時間が長くなるため、光トポグラフィーでは照射光に変調を加える方法を採用している。変調方式では外乱光に対しても強図4・69光トポグラフィー装置図4・70近赤外光による頭皮上からの大脳皮質計測図4・71 ヘモグロビン(Hb)の光吸収スペクトルく、計測時間はチャネル数に依存しない。c. 画像表示光トポグラフィー装置は、データとしてタイムコース(時間変化をグラフで表示)とトポグラフィー画像を表示する。トポグラフィー画像はタイムコースに表したデータを二次元画像として表したもので、時間軸に変化させることで動画として表示することができ、増加を赤、減少を青の濃淡で表示する。これにより負荷刺激に対する血液量の変化をリアルタイムで動画像で観察できるため、活性化部位や代謝のダイナミックな変化をとらえることが容易になる(図4・72)。
