# 10・3・1 温熱環境（thermal environment ）

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・3 高齢者に適した環境設計

a.体温調節（thermoregulation）の特徴 ヒトの体温調節は
体温を一定に保つための機構であり，身体の表面や内部にある温度
受容器により絶えず情報が検出され，体温調節中枢に伝達される．
体温調節中枢は統合された情報と基準値との差に従って，体温調節
の効果器を制御する．すなわち，高温環境で温受容器が活性化され
て脳に伝達されると，血管が拡張し血流量が増え，身体からの放熱
を促進する．さらに高温になると，発汗が始まり，潜熱による熱放
散も始まる．一方，寒冷環境では冷受容器が温度の低下によって活
性化されると，血管は収縮し血流量を減らして放熱を抑制する．さ
らに，鳥肌が立つなど，震えによる熱産生を生じる．これらの体温
調節を自律性体温調節と呼ぶ．
自律性体温調節機能は加齢に伴って低下する．暑熱や寒冷の環境
でなくとも，人がふつう生活する中等度温熱環境においても，高齢
者と若年者の体温調節反応を比べると差が認められる ．夏期に
23℃，27℃，31℃の環境下で安静状態をとった場合の生理反応を比
較すると，23℃では高齢者の皮膚温は1℃近く若年者よりも高くな
り，代謝量は温度条件にかかわらずほぼ一定であるが，高齢者の代
謝量は若年者よりも2割少ない．湿性放熱量は温度が高くなるにつ
れて大きくなるが，年齢差は認められない．つまり，湿性放熱量に
年齢差が見られず，乾性放熱量は高齢者のほうが大きいことから，
23℃での熱収支量はマイナス側，すなわち体内温は下がりやすいこ
とを示している．

和（空調）など目的をもった活動を行動性体温調節と呼ぶ．行動性
体温調節は温冷感や快適感など人の温熱感覚に従って生起すること
が多い．温冷感は全身的なものと局所に関するものの2とおりがあ
る．局所温冷感の例として，冬期における中性温度下での身体各部
位の温覚・冷覚を生じるしきい値を示す（図10･11）．このしきい

と考えられる．胸部，臀部，足部は感受性が鈍いことが示されてお
り，それらの部位においては年齢差が大きいが，頭部では年齢差が
認められない．

c.温熱環境の設計 高齢者の体温調節反応は中等度温熱環境
ですら血管運動の衰えが示され，感覚的にも感受性の低下が見られ
る．よって，環境設計を行うに当たっては，それらの高齢者特性に
配慮する必要がある．すでに提案されている住宅熱環境評価の基準
これまでの独立住宅の温熱環境実測値 に比べると，この基準
値においては冬期の着衣量が少ないために室温は高い．夏期につい
ては，午後から夜半にかけてこの基準値を遵守しようとするなら，
空調による冷房が必要な値である．しかし，実態調査によれば，高
齢者は冷房の使用を好まず，冬期には暖房使用のみならず，こたつ
を併用して手足を温めることが多く，さらに衣服を着込んで対応し
公共空間や施設などでは，高齢者ではなく若年者が室温設定する
場合が多い．上記で述べたように，寒冷や暑熱は高齢者にとって体
温調節がついていけず大きな負担となる．反対に中等度温熱環境で
あっても，冬期の高温側や夏期の低温側の環境では高齢者の感受性
は低下している．よって，着衣の状態や活動状態を見極めて中等度
に設定する必要がある．冬期に厚着でない場合には，代謝量の少な
さを補うため，若年者の快適と感じる室温よりも2～3℃高めに設
定することが一つの目安となろう．また，脱衣所やトイレなど裸や
薄着になる場所には暖房を入れるなどの配慮が必要である．
