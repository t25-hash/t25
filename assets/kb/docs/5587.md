# 4・11・1 CT

> **章**: 第4章 検査・診断機器
> **節**: 4・11 その他の映像検査装置

X 線CT(コンピュータ断層撮影法,computed tomography)はX 線が生体を直進して透過し,その減衰が通過した直線上のX線減衰係数の積分になることを利用し,ラドン変換によってX 線減衰係数分布の生体内断層像を描き出す技術である.光や電気も生体内を透過して再び体表面に現れ,それを検出することによって光や電気の特性値に関する断層像を描き出すことができる.しかし,光や電気は生体内をまっすぐには進まないため,X 線CT とは異なったアルゴリズムが必要となる.以下では,光,電気およびマイクロ波を使ったCT について述べる.a. 光CT(optical CT)光のうち,波長がおよそ700nmから2000nm(2 m)までの近赤外光は,他の波長の光に比べ生体組織に対して高い透過性をもつ.これを利用して直径10cm程度の生体の断層像を描き出すのが光CT(拡散光トモグラフィー)である.生体の光に対する特性は基本的に散乱係数μ욦と吸収係数μ욀で記述される.入射光の強度を(W/mm),組織を距離(mm)だけ進んだときの検出光の強度を(W/mm)とすると,組織の減衰係数はランベルト・ベールの法則からμ=1ln(mm)で与えられ,減衰係数は散乱係数μ욦と吸収係数μ욀の和である.ただし,この式が成立するのは散乱が1回程度しか起こらない薄い媒体に対してのみである.近赤外光に対する生体組織のμ욦は10~100mmであり,μ욦の逆数で与えられる散乱の平均飛行距離は数十mであるため,数十mの厚さの生体試料でのみ有効である.このとき,散乱した光の角度分布は前方に強く偏っている.数mmよりも厚い組織では多重散乱が支配的となり,散乱は等方散乱に近似でき,散乱係数も換算散乱係数μ욦′=1-μ욦で記述される.ここでは異方散乱パラメータであり,生体組織ではおよそ0.95の値である.したがって,μ욦′は0.5~5mmとなる.数mmよりも厚い組織では生体組織の光学特性はμ욦′とμ욀で表され,組織内の光伝ぱは次の時間依存の光拡散方程式(4・5)によって記述される.옿φ옿=∇∇φ-μ욀φ+(4・5)ここで,φは積分強度(W/mm),は時間(ps),は媒体内光速(mm/ps),は拡散係数[=13μ욦′](mm),(W/mm)は光源の強度である.表面での測定値は積分強度の表面における流束Φ=-옿φ옿(は表面から外に向かう単位ベクトル)で与えられる.近赤外域においてはμ욦′は波長に対して単調に減少し,一方μ욀はヘモグロビンおよび水の吸収係数に支配されるが,組織のμ욀はおよそ0.002~0.02mmの範囲にある.ヘモグロビンの吸収係数スペクトルは酸素化ヘモグロビンと脱酸素化ヘモグロビンで異なり,原理的にはその差を2波長あるいは3波長で測定することにより血液の酸素飽和度,ひいては組織の酸素飽和度や血液量を測定することができる.したがって,μ욀の分布を断層像として描き出すことができれば,酸素飽和度や血液量の断層像を得ることができ,いわゆる光CT となる.図4・75 光CT の測定系概念図光CT のアルゴリズムはX 線CT よりも複雑で,式(4・5)を基礎方程式として,μ욀の分布を未知数とする非線形逆問題を解く方式となる.連続光を用いる場合,図4・75のように,個の照射点それぞれに対して組織表面で個の検出点があるとすると,全測定数は=×個であり,個のピクセルからなる断層像では各ピクセルの吸収係数が未知数となる.したがって,逆問題アルゴリズムは=(4・6)と表される.,はそれぞれ次元,次元のベクトル,は行×列の行列である.は各検出点での測定値が,各ピクセルの吸収係数の微小な変動により変化する感度,いわゆるヤコビアン옿Φ옿μ욀を表し,式(4・5)を解いて得られる.一般に光CT では<であり,悪条件となることが多い.そのため,吸収係数分布に拘束条件を与えたり,散乱係数があらかじめわかっているなどの先験情報を活用することが必要である.また,光伝ぱは三次元であるため,三次元断層像を得るアルゴリズムも研究されている.基本的には式(4・6)はニュートン法を用いて解くが,ベイズ推定法を利用した最大帰納確率法や一般化パルススペクトル法など,各種の逆問題解法が適用されている.光CT 装置は各国で開発され,日本では図4・76のような極短パルス光と超高速光検出器を用い,時間分解能が約100psで64チャネルの時間分解システムが開発された.使用波長は761,791,830nmである.また,イギリスでは32チャネルの時間分解システム,アメリカでは,連続光を数百MHzで強度変調した20チャネルのシステム,連続光で約20チャネルながら毎秒3画像取得可能な高速システムなどが開発されている.図4・76のシステムと一般化パルススペクトル法を用いて,直径80mmの円柱ファントムから得られた吸収係数に関する光CT 画像を図4・77に示す.2個の吸収体は直径10mmであり,空間分解能はおよそ20mmとなっている.成人の前腕の光CT 画像も得られている.なお,拡散光ではなく,入射光が干渉性を失わない数mmの深さの組織から後方散乱した光を検出し,散乱光の減衰に関する断層像を得る光干渉CT,いわゆるOCT(optical coherence tomogra-phy)も研究開発され,眼科ではすでに臨床応用されて威力を発揮している.この技術はさらに発展してさまざまな臨床応用が期待されている.b. インピーダンスCT(impedance CT)インピーダンスCT は,組織の導電率分布あるいは複素誘電率分布の断層像を得ようとするものであり,装置が簡単,安価,取扱いやすいなどの利点がある.周囲の組織と導電率が著しく異なる組織(血液や脂肪など)の画像化が可能であるため,血行動態などの診断に役立つと考えられている.インピーダンスの周波数特性から組織の性状をとらえる多周波数インピーダンスCT の研究により,体脂肪量の測定も行われている.対象とする周波数は1kHz から数MHz であり,その領域での各種組織の電気的特性を考える.血液は含水率が大きいため電気抵抗率が低く,ヘマトクリット40%,37°Cの血液で約1.4Ω・m,含水率が中程度の骨格筋では筋繊維方向で2~3Ω・m,繊維に図4・77 円柱ファントムでの吸収体を描き出した光CT 画像(背景はμ욦′=0.863mm,μ욀=6.6×10mm,吸収体は直径10mmでμ욦′=2.36mm,μ욀=5.7×10mm)直角な方向で7~10Ω・mである.皮膚は含水率が非常に小さいため電気抵抗率は大きく数MΩ・mにもなり,肺は空気を含むため数10Ω・m,脂肪組織は数十Ω・mから100Ω・mである.これらの抵抗率は周波数が1kHz から数MHz まで増加すると70~50%まで減少する.また,比誘電率は含水率が高い組織で10程度,低い組織ではこれより1~2けた小さい.比誘電率も1kHz から数MHzまでの周波数増加で70~50%減少する.インピーダンスCT では,体表に置かれた一方の電極から電流が流れ出て生体中に三次元的に広がり,再び対極に向かって集まって対極に達する.また,電気特性値としての導電率や複素誘電率は複雑な三次元分布をもっているため,断層像を得ることは困難である.画像再構成アルゴリズムは基本的には光CT の場合と同様に,非線形逆問題を解くことに帰着する.初期に研究された方式はX線CT でも用いられた逆投影法である.図4・78のように,2電極間に電流を流したとき,隣り合った等電位線に囲まれた領域の抵抗率を,その等電位線の終端における測定電位の変化に対応して変化させる(逆投影).2電極の位置を順番に移動させて各領域の抵抗率の変化を重ね合わせることにより,抵抗率に関する断層像を再構成することができる.逆投影法のほかにもニュートン法によるアルゴリズムなどが開発されている.しかし,前述した問題のほかに,安定した5けたの測定が必要であるがそれが困難で測定誤差が大きいこと,体位変動による誤差が大きいこと,低い空間分解能,電極のインピーダンス問題などにより,臨床的な応用にはなお課題が多い.c. その他マイクロ波CT(microwave CT)は,X 線の代わりに1~2GHzの電磁波を生体に照射して,生体によるマイクロ波の減衰率などの断層像を得るものである.インピーダンスCT に9 ―53図4・76 64チャネルピコ秒時間分解光CT システムの概念図図4・78 逆投影法によるインピーダンスCT の概念比べ,電磁波の周波数が格段に高いため,直進透過成分が多い.しかし,生体組織によるマイクロ波の回折や散乱が避けられないため,断層像を得るためには何らかの方法で直進成分を取出してX線CT のアルゴリズムを用いるか,あるいはマイクロ波の伝搬経路を特定して逆問題アルゴリズムを用いる必要がある.マイクロ波の伝搬をマクスウェル方程式により記述してボルン(Born)近似やリトフ(Lytov)近似を用いて逆問題を解く方式は回折波トモグラフィーと呼ばれるが,近似が生体中では成立せず適用が困難である.電磁波の可干渉性を利用するチャープレーダの技術を用いたマイクロ波CTでは,チャープパルス信号と信号処理技術を利用して送受信アンテナ間の直線経路を伝搬した波動成分のみを抽出し,X 線CT のアルゴリズムを用いる.その結果得られる断層像は複素誘電率分布であるが,複素誘電率の温度依存性を利用して体内の温度分布の変化を計測する(サーマルCT)ことができると考えられている.がんの温熱療法では,体内の温度上昇を1°Cよりもよい精度で計測することが求められるが,マイクロ波CT をいわゆるサーマルCT として利用できれば,マイクロ波で加温と温度測定が同時に可能となる.図4・79は,サーマルCT のファントム実験の結果である.比誘電率が高含水率組織と同様な食塩水で満たした直径約240mmの円柱を32°Cに保ち,初期画像を得る.次に,温度が±1.0°Cおよび±0.5°Cだけ異なる直径64mmの円柱が存在する場合の画像を取得して初期画像との差を求めた結果が図4・79である.空間分解能は10mm程度であるが,0.5°Cの温度分解能が実現されている.
