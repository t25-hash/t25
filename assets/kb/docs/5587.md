# 4・11・1 CT

> **章**: 第4章 検査・診断機器
> **節**: 4・11 その他の映像検査装置

X線CT（コンピュータ断層撮影法，computed tomography）
はX線が生体を直進して透過し，その減衰が通過した直線上のX
線減衰係数の積分になることを利用し，ラドン変換によってX線
減衰係数分布の生体内断層像を描き出す技術である．光や電気も生
体内を透過して再び体表面に現れ，それを検出することによって光
や電気の特性値に関する断層像を描き出すことができる．しかし，
光や電気は生体内をまっすぐには進まないため，X線CTとは異
なったアルゴリズムが必要となる．以下では，光，電気およびマイ
クロ波を使ったCTについて述べる．

の生体の断層像を描き出すのが光CT（拡散光トモグラフィー）で
ある．
