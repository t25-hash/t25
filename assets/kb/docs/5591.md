# 4・12・2 簡易血液分析装置

> **章**: 第4章 検査・診断機器
> **節**: 4・12 検体検査装置

簡易血液分析装置のうちグリコヘモグロビン測定装置について,以下に記述する.a. グリコヘモグロビン(HbA1c)測定装置HbA1cはヘモグロビン(Hb)のβ鎖N 末端のバリン残基のアミノ基と糖のアルデヒド基がアルデマイン結合して,さらにアマドリ転位を経てケトアミンになった糖化物質である.1~2箇月前の平均血糖値を反映する代表的な糖尿病マーカである.HbA1c測定装置DCA 2000システム(バイエルメディカル社製)は測定機器と試薬カートリッジで構成されている.試薬カートリッジ(図4・88)には,血液中図4・88 HbA1c試薬カートリッジ図4・89測定原理のHbA1cを測定するものと,尿中の微量アルブミン・クレアチニンを測定するものとの2種類がある.b. 測定原理まず,総へモグロビンをチオシアンメトヘモグロビン法により測定する.血液溶血ヘモグロビン酸化FeCNメトヘモグロビン錯体形成SCNチオシアンメトヘモグロビン(531mm測定)次に,ラテックス凝集阻止反応(図4・89)によりHbA1cを測定する.凝集剤(HbA1cハプテン)はラテックス(HbA1c抗体)を凝集させる.この反応により透過光の散乱が大きくなり,吸光度が増大する.血液と試薬を反応させると,血液中のHbA1cと凝集剤が競合し,凝集が阻害され透過光の散乱が小さくなるので,これを531nmにおける吸光度減少として測定する.この減少の程度をHbA1c濃度とし,HbA1c(%)=[HbA1c]/[Hb]としてHbA1c(%)値を示す.c. 測定操作全血1 Lを専用のキャピラリーにとり,カートリッジに挿入する.カートリッジのバーコードを読取り機器にセットし,プルタブを引き抜いてドアのふたを閉じる.緩衝液が測光窓に流れて,ブランクが測定される.カートリッジが回転し,緩衝液,血液,酸化試薬(フェリシアン化カリウム)が混和される.赤血球は37°Cでかくはん中に溶血され,ヘモグロビンがメトヘモグロビンからチオシアンメトヘモグロビンとなる.カートリッジが回転し,測光窓で総ヘモグロビンが測定される(531nm).カートリッジがさらに回転し,抗体・ラテックスと凝集試薬がさらに混和される.繰返しカートリッジが回転し,測光窓でHbA1cを測定する(531nm).
