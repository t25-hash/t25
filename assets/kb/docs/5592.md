# 4・12・3 その他の検体検査装置

> **章**: 第4章 検査・診断機器
> **節**: 4・12 検体検査装置

化学発光物質であるアクリジウムエステルを用いた化学発光免疫測定装置ADVIA Centaur(バイエルメディカル社製)について以下に述べる。a. 測定原理。 化学発光物質化学発光物質であるアクリジニウムエステルの特徴は下記のとおりである。(1) 抗原・抗体等にマイルドな条件下で直接標識可能であるため、抗原・抗体等にダメージを与えず安定した抗原性・免疫反応性を維持することができる。(2) 光粒子収率が大きいため、増強剤(エンハンサ)等を使用しなくとも高い感度を得ることができる。(3) 発光は2秒で終了するため、測定の迅速化が可能である。(4) RI(radio isotope)のように半減期等を考慮することなく、また放射線による被ばくがないため、どの検査室でも使用できる。. 化学発光免疫測定法(CLIA : chemiluminescent immunoassay)CLIA は目的成分(抗原・抗体)を検出するために、化学発光物質(アクリジニウムエステル等)を標識した試薬と固相物質(常磁性鉄微粒子等)を結合した試薬の両者を使用している。なお、発光量は光電子増倍管により検出測定する。b. 測定機器ならびに測定試薬。 測定機器の特徴測定機器の特徴は下記のとおりである。(1) 1時間当たり240テストの測定が可能である。(2) 最初のテストの結果は17分で算出できる。(3) 30検査項目を同時にランダムアクセス測定できる。(4) バーコード対応はもちろん、多くの種類の採血管をそのまま使用可能であり、自動搬送ラインと接続できる。(5) 自動再検査・希釈検査機能を有する。(6) 測定結果は内蔵メモリに取込まれるとともにオンラインにより外部コンピュータに転送できる。(7) 機器の点検等において自己判断機構をもっており、また外部から電話回線による管理も可能である(リモートアクセス機能)。. 測定試薬の特徴試薬の特徴としては次の2点が挙げられる。(1) 2ポイントキャリブレーションイムノアッセイにおいては、抗原抗体反応等を用いるために6~10ポイントの標準液を測定し標準曲線を設定する必要がある。ADVIA Centaurにおいてはあらかじめ、登録している標準曲線(マスタカーブ)を2点の校正剤(キャリブレータ)を用いて補正し測定するため、簡便性のみならずランニングコストの低減が可能である。また校正剤は数種の検査項目が一緒であるため、各種項目が一度に校正できる。(2) 常磁性鉄微粒子の使用ADVIA Centaurに用いている常磁性鉄微粒子は、次の特徴を有する。(a) 平均直径1 mの粒子であり大きい表面積を有する(80~150m/g)ため、多くの抗原・抗体を結合することが可能である。(b) 浮力密度が低いために、溶液の均一性を保てる(なおADVIA Centaurはつねに試薬を自動かくはんしている)。(c) 常磁性であるため、磁場中のみ磁性をもつが、磁場を取り除くと即時に元の状態に戻る。これは、反応物質の洗浄において磁石を利用し分別する際に必須である。. 測定項目測定項目は以下の31項目である。・甲状腺・下垂体検査:TSH,FT 4,FT 3,T 4,T 3,TU,TPOAb・性腺・下垂体検査:LH,FSH,PRL,E2、プロゲステロン、テストステロン、HCG・腫瘍マーカ検査:CEA,AFP,ePSA,CA 19-9,CA 125,CA 15-3・貧血関連検査:フエリチン、VB 12、葉酸・アレルギー関連検査:IgE・TDM 検査:ジゴキシン・糖尿病関連検査:インスリン、CPR・心筋関連検査:CKMB、トロポニンI、ミオグロビン・副腎関連検査:コルチゾール
