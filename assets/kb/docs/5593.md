# 4・13・1 眼圧検査機器

> **章**: 第4章 検査・診断機器
> **節**: 4・13 その他の検査機器

眼圧は眼内の房水により作られる圧力で、正常眼圧は10~21mmHg である。検査目的は緑内障疾患の診断がおもなものである。眼球は球体に近似することができ、検査法は圧入法と圧平法とに大別される。a. 圧入法一定の荷重をかけた可動杆(かん)(plunger)の沈下量から換算して眼圧を求めるものである。最も代表的な器具と9 ―59してツェッツ(Schiotz)眼圧計がある。角膜表面に装置を保持するための外筒の中におもりを載せた軸棒(可動杆)があり、可動杆の沈下量に対応する目盛(0.05mm沈下/1目盛)から換算表により眼圧を求めるものである。測定値は角膜の曲率(基準値:7.8mm)と眼球壁硬性(0.0215)に影響される。基準値よりも角膜曲率が小さい場合、または眼球壁硬性が高い場合、換算値は過大評価となる。測定は表面麻酔を行ったうえ、仰臥位で行う。眼球(角膜)表面には涙液が存在し、角膜を圧平する測定チップを表面張力で角膜に引き付ける力として働く。測定チップの接触面積を直径3.06mmとすることによりインバート・フィックの法則に従う関係が得られることをゴールドマン(Goldmann)が報告し、標準的な眼圧測定法(Goldmann applanationtonometry)として最も広く用いられている。本測定原理に基づいて、圧縮空気を角膜に吹き付けて角膜の圧平状態から眼圧を求める空気眼圧計、ゴールドマン眼圧計を手持ち装置に組込んだパーキンス(Parkins)眼圧計などがある。. ゴールドマン圧平眼圧計(Goldmann applanationtonometer)細隙(げき)灯顕微鏡装置に付属させて使用するため、座位で測定を行う。表面麻酔薬点眼後、フルオレセイン液を点眼し、測定用チップを角膜中央に接触させ、直径3.06mmの円を押し当てるのに必要な力になるようにダイヤルを回転させる(チップに対するおもりをてこの原理で移動させる。1目盛が1mmHgに相当するようになっている)。チップの中には上下に分割されたプリズムが入っていて、チップと角膜との間のフルオレセイン液で作られる上下の半円弧を巴(ともえ)状にずらすことにより、3.06mmの距離を計測できるようになっている。キャリブレーションはチップの基部にてこの原理によるバランサを取付けることにより容易に行うことができる。. 空気眼圧計角膜に垂直に圧縮空気を吹き当て、それによって得られる角膜の圧平状態を左右の圧平モニタが検知し、一定面積に圧平するのに要する時間または空気圧から眼圧を測定する。実測定時間は1~3msである。測定には専用装置が必要であるが、非接触測定法(non-contact tonometer)であるため表面麻酔薬の必要はない。また、オートアライメントまたはモニタによる誘導方式であるため測定は容易であり、検診などでのスクリーニングには有用である。しかし、測定時間が短いため眼圧の脈波による影響、および体動などの影響による測定変動がある点に注意を要する。また正常眼圧の範囲外では測定誤差が大きいとされ、精密眼圧測定には問題があることもある。圧縮空気の衝撃が問題になることもある。. パーキンス眼圧計(Parkins tonometer)ゴールドマン眼圧計(Goldmann tonometer)を手持ち式に改造したものである。したがって、仰臥位での測定も可能であるが、操作に練習を要する。また、座位よりも仰臥位の場合は眼圧が高い(3mmHg 程度)ことにも留意する必要がある。c. 電気眼圧計圧入法に類似した外筒と内筒(外筒よりもわずかに先端が出ている)からなる測定チップを角膜に接触させ、内筒の微動量を電気波形として検出し、波形から眼圧を測定する。先端が細いこともあり、角膜移植後や角膜の変形がある場合でも測定することが可能である。装置としては、マッケイ・マーグ(Mack-ay-Marg)眼圧計およびこれを発展させた手持ちタイプのトノペンなどがある。後者は測定信頼限界なども表示する機能が付加されている。
