# 10・5・3 暦年齢と機能年齢

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・5 高齢者特性に適した就労支援技術

65歳以上を高齢者というが，高齢者はすべて諸機能や能力が低
図10･18加齢と機能年齢
(１)高齢者は，一般に勤勉で注意深い面があり，長年従事した
職場で雇用延長で働いてもらい，若年者に対する指導的役割をする
ことで，若い作業者への模範となり，作業する態度や技能の伝承に
もなる．
(２)50歳以降の中高年齢者は，職場では経験が長く深い技能
をもっていることが多い．このような熟練作業者（skilled
worker ）に制御工学，コンピュータ，ソフトウェア工学などの先
端的技術を学習させることで，ロボットなどの自動機械を製作させ
るなど，生産技術の戦力として鍛え上げることができる．各職場か
ら優れた熟練者を集めて最先端技術を学習させ，その後，元いた職
場の改善ニーズを発掘させてそれらを機械化・自動化し，生産の効
率化を実現するというやり方で高齢者を活用した企業は少なくな
い．
下するのではなくて，能力や体力を十分にもっている高齢者も存在
する．加齢の特徴の中に，個人差が大きくなるという特性もある．
そこで加齢に絡んだ重要な考え方を以下に示す．A職務を遂行す
る要件が図10･18の水平線であるとする．誰でもこの水平線近くの
能力があればA職務を遂行できる．図10･18中の横軸は年齢を，
斜線はB作業者の能力を表し，それは年齢がたつにつれて斜線の
ように下がっていく．能力線である斜線と職務要件である水平線と
が交わった点はB作業者にとってA職務遂行がそろそろ困難にな
った時期を表し，それ以降は彼の生産性が低下する．この交点はす
べての人にとって同一年齢で同一ではなくて人によって異なる．つ
まり，斜線はその人の能力によって決まるのであり，これを機能年
齢（functional age）という．若い時期に体力や能力を磨いた人の
機能年齢との交点は，かなり遅くなって発生するが，そうでない人
の交点は早くやってくる．運動を嫌う若年者の早期老年化はこのこ
とを物語る．
そして，交点以降は生産性が低下するので，管理者は高齢者を嫌
がることになる．その打開策として以下のことが考えられる．
(１)高齢者の能力開発を実施して機能年齢を向上させ交点を右側
へ遅らせる．
(２)職務再設計を実施して，B作業者の機能年齢よりも職務要件
を下げ，生産性に問題が生じないように対策を立てる．
高齢者の能力を向上させることは不可能ではないが，費用対効果
を見るとそれほど容易ではない．それに対して職務を改善して高齢
者に楽に従事してもらえるように職務再設計をすることは割と簡単
であり，効果も大きい．このことから，高齢者雇用を推進し，経営
者や管理者からも高齢者を喜んで雇用してもらえる方策として，職
務再設計が重視されている．
