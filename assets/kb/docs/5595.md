# 10・5・4 職務再設計の実際

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・5 高齢者特性に適した就労支援技術

職務再設計とは，職務が要求する職務用件が作業に従事する作業
者に適合するように職務を再設計（改善）することをいう ．こ
こでは高齢者を対象としているので，高齢者がもつ能力や機能に合
うように職務側を改善することであり，職務と高齢者の融合を図
り，彼らを生産性の担い手に位置付けることである．高齢者全員に
共通した再設計も可能だが，ほとんどの場合，個々の高齢者に該当
する再設計を実施することが望まれる．そのためには，以下の点に
配慮する必要がある．
(１) 視力等の生理的機能低下を拡大鏡・顕微鏡などの他の機能
で代替したり，視的機能を音機能に変換したり，より認知しやすい
道具・情報に置換える．
(２) 筋力の低下に対して筋力を用いなくて済むように，クレー
ン，ホイスト，台車，リフト，コンベヤ等の活用，ジグの製作等で
職務要件を下げる．
(３) エネルギー負担となる屈曲作業姿勢を，作業点を楽な適正
位置まで変更することで疲れない正常な作業姿勢に変更する．
一般に高齢者にとって高負荷となる作業であっても，(１)～(３)
の条件に関する配慮で負荷が軽減化するように再設計すると，高齢
者はほとんどの再設計された職場に就労できるようになる．また，
このような再設計に活用できるツールもたくさん開発されている．
