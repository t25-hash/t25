# 4・13・4 赤外線検査計

> **章**: 第4章 検査・診断機器
> **節**: 4・7 のごとくである.

光計測技術の医学応用は従来から低侵襲性と機能診断への適用に適している点で注目されていた.1970年代に生体に対して透過性の高い近赤外光(near infrared light,波長700~1200nm)を用いて脳血流変化を計測できることが報告されている.1980年代には反射型の光学式酸素モニタが開発され,1990年代になると脳の活動に伴う血液量の変化を反射型の生体光計測により測定する実験で,被検者の脳の活動に同期した信号が得られた.こうして,脳機能計測を生体光計測によって行うことが可能であることが示唆された.ここでは,大脳皮質の血液量の変化をトポグラフィーとして画像化するETG-100光トポグラフィー装置(図4・92)について紹介する.本装置の臨床適用上の特徴は以下のとおりである.(1) 長時間計測または繰返し計測が可能であること.(2) 被検者が多少動いても大丈夫なので,小児計測用に適する.(3) 装置が小型可搬でベッドサイド計測が可能.(4) 外来ノイズに強く,特別な計測室を必要としない.a. 計測原理光トポグラフィー(optical topography,4・10・4項参照)においては,多点同時計測を行うことによりある範囲にわたる脳機能(brain function)を非侵襲的に計測できる.大脳皮質には多数の血管が分布し,血液中のヘモグロビンによって運ばれる酸素を脳組織に供給している.脳内の特定領域が一時的に活動すれば,その部位では酸素消費量が増大するが,やがて元へ戻る.これに伴い同じ部位で脱酸素化ヘモグロビン(デオキシヘモグロビン,Hb)の増加,続いて酸素化ヘモグロビン(オキシヘモグロビン,HbO)の増大,さらに元の水準への回復などが起こる.光トポグラフィーは大脳皮質で起こるこれらの現象を非侵襲的に計測するものである.血液中の酸素化ヘモグロビンと脱酸素化ヘモグロビンの量の変化は,それぞれの近赤外光の吸収特性の違い(前出図4・71)を用いて決定される.ETG-100光トポグラフィー装置は780nmと830nmのレーザ光源を使用している.近赤外光が頭皮上に照射され,その反射(皮下へ浸透した光の頭皮上への戻り光)を2~3cm離れた場所で観測すると,測定された光の光路分布は前出図4・70に示すように大脳の一部に達するので,この光を測ることにより大脳皮質の血液の状態を知ることができる.ETG-100光トポグラフィー装置は照射点-検出点間距離を3cmとしており,この条件で頭皮下2cm前後の脳の計測が可能である.b. システム構成光トポグラフィー装置は光源部,照射用光ファイバ,検出用光ファイバ,計測回路部およびデータ変換,処理,表示部よりなる.光源部でチャネルと波長ごとに特定周波数で変調されたレーザ光は,照射用光ファイバで導かれ,被検者の頭皮上に照射される.光ファイバ先端は頭髪にじゃまされないよう先端が細くなっており,アダプタで頭皮上に固定される.検出光は検出用光ファイバにより導かれて計測回路の高感度光検出器(フォトダイオード)によって検出される.計測回路では,検出器の出力信号は,隣接する照射光源の変調周波数に同期したロックイン増幅器で分離(検波)される.これらの信号は全48チャネルの個別回路で同時に最大毎秒10回計測され,計測位置(照射-検出点の中点)ごと,波長(780,830nm)ごとに処理され,ヘモグロビン濃度変化に関連する値に変換される.得られたヘモグロビン濃度変化データはタイムコースとして表示されるほか,スプライン補間を使用したトポグラフィー(二次元動画像)としてリアルタイム表示,計測後の再生表示を行う.c. 計測信号の処理データ変換・信号処理装置(PC)では,酸素化ヘモグロビン,脱酸素化ヘモグロビンの変化が吸収スペクトルを用いて2波長計測の信号から算出される.本装置は二つの計測モードで使用される.第1は連続計測モードで,連続的に信号の変化を計測記録する.第2のモードは積算モードで,計測中被検者にタスク期間とレスト期間を交互に与える.レストの時間に得られた信号は参照信号としてタスク時信号から差引かれ,繰返されるタスク時信号は加算・平均化される.この方法により,ノイズ信号と判別の難しいわずかな信号の差違でも的確に抽出することができる.左手を動かした際に右半球脳から得られる反応を4×4構成プローブで測定した一例を前出図4・72に示す.右運動野付近の血液量増加による酸素化ヘモグロビン量の増加を,タイムコースとトポグラフィー表示で見ることができる.d. 光トポグラフィーの臨床適用現在までに言語発達,言語認知などの高次脳機能研究,脳神経外科の術前診断(言語優位半球同定,てんかん焦点診断)を含む分野で高い実績を上げている.今後も,精神科疾患の診断,小児の発達や,脳循環系疾患の診断などの分野への応用範囲の拡大が期待されている.
