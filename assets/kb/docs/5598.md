# 5・2・1 手術台

> **章**: 第5章 手術・治療機器
> **節**: 5・2 手術周辺支援機器

手術台(operating table)とは、患者をテーブルトップに乗せて手術を行うものであるが、その際、目的とする手術に合わせて患者の体位をとり、テーブルトップを移動、変形させていろいろな体位に変えることができるうえ、主柱を昇降させて手術を行いやすい位置に設定することができる。この場合、患者の安全はもちろんのこと、術者を中心とした手術スタッフの環境をよりよいものとすることも要求されている。さらに、手術の種類や術中の体位変化の頻度、術中X 線撮影の有無などにより、各科別に適した手術台を選択することも可能である。その基本構造、機能、性能上の特徴を説明すると、テーブルトップとコラム(昇降主柱)およびベース(脚台)から構成される。基本的な動きとしては昇降、骨盤低位・高位(縦転)、横転左右下げ、背板上下、さらに最近ではスライド機能を備えたものがある。このような手術台の作動には、油圧シリンダを使ったりギヤを使ったりしている。また、動力は手動(足踏みも含む)を用いる場合と電動で行う場合がある。構造的特徴としては、術中の患者の安全確保を第一とするため、各作動部は堅牢で確実な保持機構を有していることが挙げられる。以下に手術台の種類と簡単な概要を示す。はん用マニュアル手術台、はん用電動手術台、多機能手術台、整形外科用(下肢けん引透視)手術台、X 線透視用手術台、脳神経外科用手術台、マイクロサージェリー手術台、内視鏡用手術台、術中画像診断用手術台[イメージング、CT(computed tomogra-phy)対応、MRI(magnetic resonance imaging)対応]、分離式電動手術台などがある。a. はん用マニュアル手術台、はん用電動手術台通常はん用手術台と呼ばれている一般的な手術全般に対応可能な手術台では、油圧シリンダとギヤ方式の両者の駆動源としての手動、電動の組合せにより、各部の動きを構成している。これらいずれの手術台も基本作動を有し、操作も簡単であるため、はん用手術台(シリーズ)の需要は70%以上となっている。電動手術台としてバッテリを内蔵したバッテリ駆動手術台もあるが、通常は電源コードレスで使用する。b. 整形外科用手術台、スパイラル手術台整形外科用の手術台は、主として下肢のけん引手術用として術中のX 線透視および撮影を行いやすくしている。スパイラル手術台は椎間板ヘルニアの手術を目的として作られたもので全身の透視を可能とし、特に煩雑で患者負担の大きい体位の設定も容易に行え、患者への圧迫も少ないため、患者の負担を大幅に軽減できるものである。c. X 線透視用手術台、イメージングテーブル術中、テーブルトップを脚側または頭側へスライドするとテーブルトップ下にX 線透視を妨げるものがなくなるため、CアームなどのX 線の透視が簡単に行える特徴をもっている。最近ではテーブル板全体にカーボン板を使用し、Cアームなどの360°X 線全身透視にも威力を発揮しており、血管カテーテルの手術に最適である。d. 脳神経外科用手術台、マイクロサージェリー手術台マイクロサージェリーの進歩発達に伴って脳神経外科、眼科、耳鼻咽喉科、整形外科、形成外科等に広まりつつある。顕微鏡下の手術においては、テーブルトップの高さ最低位48cmを可能とすることにより、術者にとっても快適な手術環境が得られる手術台である。e. 内視鏡用手術台近年、低侵襲の手術として各科に広がってきた内視鏡手術に対応した機能をもったもので、特に腹部臓器への手術が行いやすいように横転作動角度が左右70°まで可能となっている。また、暗くした手術室でもコントロールボックスの各作動ボタンが照光し作動を確実に行えるようになっているとともに、患者の体位固定保持のための専用アクセサリーにも対応できるようになっている。f. 術中画像診断用手術台術中画像診断とは手術の信頼性、安全性および確実性の向上のため術中に手術部位を中心としたCT,MRIなどによる画像情報を入手し、その情報をもとに手術を進めるもので、その信頼性、安全性および確実性は飛躍的に向上した。現在は脳神経外科の手術での適用が中心となっているが、将来的には全身の各部が対象となるであろう。g. 分離式電動手術台分離式電動手術台はコラム、テーブル、ストレッチャから構成されている。コラムベースを基盤とし、各科手術対応のテーブルを装着することにより単能台としての機能を有することになる。ストレッチャから手術台への患者の移し替え時の安全を確保しながら、介助者の労力を軽減することができる。
