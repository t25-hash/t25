# 5・2・2 フレーム(頭部固定装置と定位脳手術装置)

> **章**: 第5章 手術・治療機器
> **節**: 5・2 手術周辺支援機器

脳神経外科領域において、フレーム(frame)には二つの目的がある。一つは、頭部を確実に固定することで手術操作による位置のずれを防ぐことである(確実な頭部固定保持)。脳外科においては、円座という馬蹄形の形をした台座に頭を載せて頭部を固定し開頭手術を行っていた。長時間の手術が行われるときには、床ずれとして知られている褥瘡(じょくそう)が円座に当たっている部分に発生することと、固定されていない頭部が手術操作によってずれることがあった。現在わが国では、これらの問題点を解消するために、頭部固定用のピンの数で3点固定[メイフィールド(Mayfield)フレーム]や4点固定(杉田フレーム)が一般に使われている。もう一つは、固定された頭部の脳内の決められた目標に手術器具を正確に誘導する定位脳手術装置である。これは、現在では一般的には(フレームレス)ナビゲーション(navigation)技術として発達している。フレームは、頭部を確実・安全に固定する器具として今後も脳外科手術において必要な道具であり続けるであろう。a. 固定用のフレームの材質手術用フレームは、アルミニウム合金製が一般的である。しかし、術中に使用する画像のモダリティーにより、画質に影響を与えにくい材質が要求されるようになった。術中血管撮影用には、X 線透過性の優れているカーボンフレームが一般的である。モバイルCT(mobile CT)用には、スキャン範囲がカーボンフレームである特殊な形状の固定フレームが製作されている。オープンMRI(open MRI)用のフレームは、初期のころには磁場に影響を与えない材質ということで、グラスファイバによって製作されていた。東京女子医科大学では、オープンMRI手術を遂行するうえで耐久性や操作性などの問題があったため、現在では固定フレームとコイルが一体型となったアルミニウム合金製フレーム型コイルを開発し、使用している。このフレームは、画質の点でも手術操作の面でも従来の診断用のコイルおよび手術フレームと比べて性能的に遜色がない。このように、手術用固定フレームは手術手技の進展とともに変化しているのであるが、基本的には頭部を安全にかつ正確に固定するという機能には変化はない。b. 定位脳手術からナビゲーション手術へ定位脳手術(ster-eotactic neurosurgery)では患者の頭部に定位脳手術フレームを装着し、これを座標系の基準として用い、手術も定位脳手術フレームを着けたまま行う。しかしこれでは撮影が煩雑で、患者の負担も大きいうえに、開頭手術の場合にはアプローチの方向や開頭範囲が限定される欠点がある。そこで、定位脳手術フレームの代わりに骨もしくは頭皮に小さなマーカをいくつか装着して座標系の基準とする(定位脳手術)フレームレスナビゲーションが開発された(頭部固定用フレームは使用)。定位脳手術フレームを着けた定位的脳腫瘍生検(biopsy)や脳内血腫除去と比べるとさまざまな術式と容易に組合わせて種々の目的に利用することができるため、現時点ではフレームレスナビゲーション(frameless navigation)が脳神経外科手術の主流になっている。c. ナビゲーションのための三次元位置計測法ナビゲーションに用いる位置計測は、再現性があって一定の精度を保っていることが保証できるものが望ましい。さまざまな方式があるが特に機械式・光学式・磁気式の3方式は精度がよく、また互いに相補的な性質をもっているため、目的に応じて組合わせることによって、適用対象が広く、かつ精度のよい計測が可能になる。(1) 機械式(mechanical tracking)6以上の自由度をもっている関節をもつアームの先端にポインタを有し、各関節の曲がり角をエンコーダで計測することによってポインタの位置を求める。最新のモデルでは小開頭部での使用において0.1~1mmの精度に達している。アームの操作が煩雑な場合があること、一度に一つの対象物の位置しか測れないことが弱点である。(2) 光学式(optical tracking)ポインタに取付けた光学マーカ(発光ダイオードなど)や反射型マーカを複数のテレビカメラでとらえて、その位置を計測する。精度は0.3mm程度である。多数の器具の位置を同時に測定できるが、光学マーカとカメラとの間に遮へい物があると測定できなくなる。(3) 磁気式(magnetic tracking)術野外部に置いた複数のコイルで磁場を交互に発生し、ポインタに取付けたセンサで磁場強度を計測して位置を知る。光学式と異なり遮へい物があっても計測できる利点がある。ナビゲーション以外に、三次元超音波装置の中にも組込まれて使われている。一方、術野近くにある磁性体の影響などで計測値に誤差が生じていても気づかないおそれがあるが、最近では動的磁場はプログラム的にキャンセルされるようになっている。d. 術中画像によるナビゲーション手術操作による臓器の変形や移動に追随するには術前画像だけでは不十分であり、術中にボリュームスキャンを行ってナビゲーション情報を更新する必要がある。実際、術中にCT でモニタリングすることは従来から行われている。最近では毎秒6枚のCT 像を撮影しながらほぼリアルタイムで表示できるCT フルオロスコピー(fluoroscopy)が市販されており、脳手術・生検・IVR(interventional radiology)などに応用されている。一方、インタベンショナル(interventional)MRIに代表される手術室でのMRIの使用が今後の主流となる。CT と違って医療スタッフのX 線被ばくがなく、またボリュームスキャンが容易である。磁場を乱す手術機器が使用できず、非磁性体で構成された特殊な顕微鏡や手術器具が必要ではあるが、長年脳外科医を悩ませてき9 ―67た硬膜切開後の手術操作による脳の変形・移動に対処するリアルタイムアップデートナビゲーションとして、その有用性は高く評価されている。
