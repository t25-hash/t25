# 5・3・1 概説

> **章**: 第5章 手術・治療機器
> **節**: 5・3 シミュレーション・治療計画・ナビゲーション

近年,仮想現実感や医用画像解析の技術を応用した治療支援機器の進歩が著しい.これらの治療支援機器の目的は,治療行為のシミュレーション,治療前の治療計画立案,および治療計画を正確に実行するためのナビゲーションに大別される.コンピュータ,感覚フィードバック機能を実現する入出力機器,および実時間三次元位置計測機器の目覚しい発展により,これらの支援機器の性能は急速に向上しており,治療訓練,治療の品質管理,治療精度および客観性向上の観点から,今後普及していくと思われる.
