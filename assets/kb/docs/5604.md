# 5・3・2 訓練用シミュレータ(training simulator)

> **章**: 第5章 手術・治療機器
> **節**: 5・3 シミュレーション・治療計画・ナビゲーション

a. 治療シミュレーション概説治療支援を目的としたシミュレーションは、一般的(典型的)モデルに基づくものと患者ごとのデータに基づくものに大別される。一般的モデルに基づくものは、仮想的に治療行為を体験する訓練用のシミュレータに代表される。患者ごとのデータに基づくものでは、一般的な訓練のみならず、実際に特定患者の手術前のリハーサルが可能になり、手術計画と密接に関連してくるので、5・3・3項で述べる。b. 訓練用内視鏡手術シミュレータ概説近年、腹腔鏡、大腸鏡、関節鏡、気管支鏡など内視鏡手術による低侵襲治療が普及してきている。内視鏡手術は、患者の身体的ダメージを大幅に軽減する一方で、術者にとっては限られた視野、限られた操作空間における新しい術具の操作法の習得が必要となる。よって、通常の手術に熟練した医師にとっても再訓練が必要であり、訓練用シミュレータの必要性は高い。一方、内視鏡手術では、手術部位を肉眼で直接見るのではなく、テレビモニタを通して見る。仮想現実感技術の観点から、術野を直接見るという知覚効果をシミュレートすることに比べ、テレビモニタを通しての知覚効果をシミュレートすることは比較的容易である。以上のように、内視鏡手術訓練用シミュレータは、臨床的必要性と技術的実現性の両面の要因から実用化が進んでいる。以下では、内視鏡手術シミュレータを中心に説明を行う。c. 内視鏡手術シミュレータの機能内視鏡手術シミュレータのおもな機能は、内視鏡・術具の模擬操作機構と操作情報のコンピュータ入力、内視鏡操作・術具操作に伴う反力計算とその力覚(haptic sensation)提示、操作に伴って時々刻々変化する内視鏡像生成と視覚提示、および操作に伴って生じる患者の生理反応(出血、痛みなど)のシミュレーションとその提示である。内視鏡・術具の模擬操作機構に関しては、一般には実際の内視鏡、手術器具の操作部と同一の機構が組込まれ、その操作における運動情報がシミュレーション処理用のコンピュータに入力される。反力計算および内視鏡像生成は、通常、コンピュータシミュレーションにより行われる。コンピュータ内には、手術対象となる人体解剖構造の三次元形状モデル、および術具の三次元モデルが格納されている。さらに、クリッピングや切断などの手術操作をシミュレートするために生体組織の力学特性がモデリングされ、術具操作に伴う運動情報と解剖構造三次元形状モデルに基づいて、組織の形状変形および反力が計算される。内視鏡や術具の運動情報および形状変形に連動して、内視鏡像がリアルタイムに生成される。このようなコンピュータによる内視鏡像生成機能は仮想内視鏡(virtual endoscopy)と呼ばれる。患者の生理反応に関しては、出血は出血パターン像を内視鏡像に重畳提示、痛みはコンピュータモニタ上の患者表情アイコンなどで視覚的に提示される。図5・16に腹腔鏡手術シミュレータの外観を示す。d. 内視鏡手術シミュレータの要素技術内視鏡手術シミュレータの基本的な要素技術は、形状変形および反力計算、力覚提示、および視覚提示である。重要な点は、これらをすべてリアルタイムで行う必要がある点である。形状変形および反力計算に関しては、厳密な有限要素解析で解こうとすると膨大な計算量が必要となり、リアルタイムで行うことが極めて困難になる。現状では、実用上問題がない程度の精度で簡易な近似計算を行うことにより、リアルタイム性を実現している。力覚提示に関しては、ハプティックデバイス(haptic device)と呼ばれる力覚提示装置が開発されており、これらの装置を応用している。視覚提示に関しては、近年飛躍的に高速化・低価格化されたパーソナルコンピュータ(PC)のグラフィックスボードを使用して、内視鏡のテレビモニタをコンピュータモニタ上でシミュレートする。三次元形状モデルの表面に対して生体組織に極めて近い反射特性のテクスチャをマッピングすることにより、非常にリアルな内視鏡像を生成している。図5・17に腹腔鏡手術シミュレータにおける視覚提示例を示す。訓練用シミュレータの可能性を高めるための要素技術としては、さまざまな症例を含む解剖構造三次元形状モデルおよび生体組織力学特性モデルのデータベース、および訓練者の熟練度評価を行うソフトウェアの充実が重要である。患者の生理反応は、現在は簡易シミュレーションであ9 ―77り、特に出血に関しては精密なシミュレーション法の開発が今後望まれる。e. その他の治療用シミュレータ内視鏡手術のほかにも、顕微鏡手術やX 線透視下で行われるカテーテル治療(IVR : inter-ventional radiology)など、一般に画像誘導治療(image guidedtherapy)に関しては、内視鏡手術シミュレータと同様の構成要素により訓練用シミュレータが実現されている。通常の開腹、開胸、開頭手術などでは、立体視も含めた現実に近い視覚効果を得るためにはコンピュータモニタでは不十分であり、臨場感のある空間的立体知覚を得るための視覚提示装置が組合わされる。
