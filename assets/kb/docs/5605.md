# 5・3・3 治療計画システム(surgical planning system)

> **章**: 第5章 手術・治療機器
> **節**: 5・3 シミュレーション・治療計画・ナビゲーション

a. 治療計画システム概説治療計画は、形態形成を目的とした治療計画、および腫瘍の切除・(放射線や温熱などによる)壊死を目的とした治療計画に大別される。また、後者において放射線治療を用いる場合に関しては、特別の治療計画システムが必要となる。治療計画では、訓練用シミュレータと異なり、治療前に撮影したCT,MRI、核医学画像などの患者ごとの三次元データに基づいて、患者の治療部位および周辺の構造を詳細に知ることにより、血管や神経など重要な組織の配置の確認、およびシミュレーションによる予後予測などに基づき、安全かつ治療効果の大きい治療計画の立案を目的とする。以下に、それぞれの治療計画について述べる。b. 形態形成を目的とした治療計画人工関節設置や血管へのステント留置など、人工部品を設置することにより形態形成して関節や血管機能を回復させる手術、あるいは骨切り術のように、患者自身の骨から骨片を切り出して、その骨片を適正な位置に移動・固定することにより形態形成して機能を回復させる手術などが対象となる。これらの手術では、患者の骨や血管の三次元構造に基づき、人工部品の種類・サイズと設置位置、骨片の切出し形状と移動量を最適に選択・計画することが必要になる。ここでは、人工股関節手術を例に、治療計画の実例を述べる。(1) 患者の股関節三次元モデルの作成CT データから股関節を構成する骨盤と大腿骨のそれぞれの領域を別個に切り出し、骨盤と大腿骨の三次元形状モデルを作成する。(2) 人工股関節部品設置シミュレーション骨盤と大腿骨の三次元形状モデル上で、人工関節部品モデルの種類・サイズの選択および仮想的な配置を行い、さまざまな方向から視覚的に選択・配置の妥当性を確認する(図5・18)。(3) 人工関節適合性評価カップ(骨盤側に設置する部品)、およびステム(大腿骨の髄腔に挿入する部品)が強固に固定されているかを、カップ被覆率(カップが骨盤に被覆されている割合)、ステム形状と髄腔形状の一致度などを三次元データに基づき計算し評価する。(4) 関節機能評価シミュレーション人工股関節設置時の脚長を三次元データに基づき計算し、もう一方の脚長と比較する(図5・19)。また、関節可動域を骨および人工関節部品の衝突検出アルゴリズムによりシミュレーションし評価する。図5・17 腹腔鏡手術シミュレータにおける視覚提示図5・16 腹腔鏡手術シミュレータの例9 ―78医療・福祉・バイオ機器図5・18 人工関節部品の調整・変更人工関節適合性評価、および関節機能評価の結果に基づき、人工関節部品の設置位置の調整や部品の変更を行い、再び(2)に戻る。現状では、形態データ解析のみにより評価を行う場合がほとんどであるが、今後は生体力学的評価を行う有限要素シミュレーションを導入して治療後の関節機能をより多角的・長期的に予測することにより、治療計画の最適性を向上させることが期待できる。同様に、血管へのステント留置術においても、現状では形態的適合性により手術計画が立てられるが、設置後の血流に関して流体力学的シミュレーションを行うことにより、本来の血管機能に基づく予後予測が期待できる。これらの力学的シミュレーションはまだ研究段階であるが、今後の発展が期待される領域である。c. 腫瘍の切除・壊死を目的とした治療計画腫瘍の手術では、正常組織を完全に保存して、腫瘍組織のみを切除あるいは壊死させることが理想的である。そのためには、腫瘍の広がりをできるだけ正確に知ることが重要になる。治療前に撮影したCT,MRI、核医図5・20 生体肝移植手術計画のための切除肝臓体積計算学画像、脳の場合はMEG(magnetoencephalogram)データなどの情報を統合して、腫瘍組織、通常の正常組織、血管・神経など(脳の場合は言語野、運動野なども含む)の致命的なダメージを与える重要な正常組織を識別し、機能の温存と腫瘍の切除を最適にバランスさせる切除ライン(範囲)を決定する。腫瘍の放射線治療を行う場合には、治療計画において複数の照射位置・方向をどのように配置し、照射量をどう設定するかが検討される。照射位置・方向と照射量が与えられれば、三次元的な線量分布をシミュレーションにより推定することができる。推定された線量分布を術前CT やMRIの三次元画像上に重ね合わせることにより、腫瘍組織のみに線量分布が集中し、正常組織へのダメージが小さく、重要な組織へのダメージが最小限であるかどうかを治療前に確認することができる。これにより、最適な治療計画の立案が可能になる。d. その他の治療計画事例:生体肝移植近年、生体肝移植が普及している。生体肝移植では、肝臓提供者(ドナー)から肝臓の一部を切除し、患者に移植する。肝機能は肝臓の体積に依存すると考えられており、ドナーへの悪影響を最小限に抑え、かつ移植された肝臓が患者の体内で正常に機能する最適量(体積)を決定し、決定された量の肝臓をドナーから抽出する。手術計画で重要な点は、ドナーの肝臓の切除線をどのように定めれば最適量の肝臓を抽出できるかである。そのためには、術前CT データをもとに、正確に肝臓領域のみを切り出し、肝臓の体積を正確に計測することが必要となる。また、CT データ上で任意の切除線をシミュレートして、抽出される肝臓体積を計測することが必要となる。図5・20にそのような手術計画が行えるシステムの例を示す。
