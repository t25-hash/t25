# 10・2・7 地上試験

> **章**: 第10章 ロボティクス
> **節**: 10・3 カナダアームのオペレーション

定盤上で親アーム，子アームの二次元機能性能試験が実施され，
その結果を用いてアームの数学モデルの評価が行われている．定盤
上では，親アームと子アームは重力の影響を補償するため，空気ベ
アリングで浮上させたジグで支持されている．地上試験では確認で
きない軌道上での三次元動作は，試験結果を反映させた数学モデル
を用いた解析で検証されている．
図10･14エンドエフェクタ部（左）およびグラプルフィクスチャ（右)
をアーム先端に載せた状態でのマヌーバ等はマニュアル制御で行
う．マニュアル制御時のアーム先端最大速度はペイロードを把持し
ていない状態で並進0.6m/s ，回転4.7° /sである．自動制御はそ
の軌跡全体にわたって，つねにアームおよび把持するペイロードが
周囲構造物と1.5m以上の距離を維持できる場合に使用可能であ
る．なお地上からのアームの遠隔操作はできない．マニュアル操作
用ジョイスティックは図10･2のように並進用と回転用の二つがあ
る．SRMS操作パネルはシャトル操縦室後部に位置し，シャトル
の船尾を臨む方向および後部操縦室天井部に窓が取付けられてい
る．また，二つのテレビモニタがSRMS操作パネル近傍にあり，
操作者は窓越しやテレビモニタからの視覚情報を確認しながらアー
ムの操縦を行う．テレビカメラはシャトル貨物室に通常四つ以上，
SRMSのひじ部とエンドエフェクタにそれぞれ一つずつ取付けら
れている．また，ISS組立作業時には，ISS側のテレビカメラ映像
もシャトル操縦室内のテレビモニタで使用できる．人工衛星捕獲時
のアーム操作では，エンドエフェクタのテレビカメラによる衛星側
グラプルフィクスチャ上のターゲットの映像が相対位置姿勢の検知
のための主要な視覚情報である．3630kg以上のペイロードを把持
した状態や，人工衛星を捕獲する際のアームのマヌーバはシャトル
側の姿勢制御系を停止して行う．ほとんどのSRMSの運用は2人
の操作者により行われる．主操作者は操作パネルを使用したアーム
の操縦等を行い，副操作者はテレビカメラの操作や操作パネルとは
別の場所にあるコンピュータディスプレイ上に表示されるSRMS
関連テレメトリのモニタ等を行いながら主操作者の操作支援を行
う．
各関節のエンコーダによるSRMS先端部の位置姿勢情報の精度
は並進約5cm，回転1°であるが，把持したペイロード上の参照点
ではこの誤差はさらに大きくなり，SRMSによるISS各モジュー
ルの取付け時には，把持したモジュールに許容される位置姿勢誤差
がSRMS各関節エンコーダによる位置姿勢情報の精度より小さい
ため，SRMSとは独立した，より精度のよい位置姿勢測定システ
ム（取付作業用カメラや画像処理による位置姿勢測定装置等）も使
用されている．
システムや運用手順に関する知識に加え，SRMSの操作訓練に
おいて強調される点は，視覚情報や計器類からのデータによりシス
テム全体の状態を常時把握し，安全，ミッション達成のための適切
な操作を行う能力，ジョイスティックへのスムーズなコマンド入力

および停止や多自由度同時コマンド入力を行えること，ロボティク
ス操縦座標系の理解等を含めたアーム，マヌーバに関する技量，単
軸制御の技量，さらにコックピットリソースマネジメント（主・副
操作者間の協調運用）等であり，航空機の操縦者に要求される資質
に概念的に近いものといえる．
SRMSは1981年の初飛行以来，ハードウェア，ソフトウェア的
にも改修が進められてきている．SRMSのコアシステムからは独
立した形で，SRMSの制御計算機からの関節角データをラップト
ップパーソナルコンピュータに取込むことにより，アームの周囲構
造物との相対位置情報を含めたアーム全体の状態や各関節角の状
態，さらにマニュアル制御時の必要なジョイスティックコマンド入
力量等を操作パネル付近に設置したラップトップパーソナルコンピ
ュータ上にグラフィックス表示するシステムも使用されており，
SRMS運用全体に関する状況把握を促す効果的な情報表示システ
ムとなっている．
