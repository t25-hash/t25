# 5・4・1 回転治療機器

> **章**: 第5章 手術・治療機器
> **節**: 5・4 手術用装置

歯科,整形外科では,硬組織の切開にドリルが用いられ,用途により刃先形状,回転数などが選ばれてきた.駆動源としては高圧ガス,電気が一般的である.また,分解・丸洗いできる部分が分離できる構造となっているのが普通である.機構からの潤滑油の流出は許されないので注油を不要とするか,空気軸受などを用いる.安全のため,過剰負荷を生じると回転が止まる機構や,一定の負荷がかからないと回転しない機構などがある.後者は脳神経外科手術で用いる開頭用ドリルで,頭蓋骨の直下にある硬膜を傷つけないためのものである.いずれも機械的クラッチで構成される.
