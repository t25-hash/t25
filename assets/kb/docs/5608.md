# 5・4・2 電気メスなどの電気手術器

> **章**: 第5章 手術・治療機器
> **節**: 5・4 手術用装置

高周波メスと呼ばれることもあり、周波数300kHz~5MHzの高周波電流を狭い領域に集中して流すことで、その領域を加熱して焼灼する。凝固に特化したRFA[radio frequencyablation、ラジオ波凝固(焼灼)法]システムなども電気手術器の一種である。類似の機器にマイクロ波凝固システムがあるが、作用機序も使用している周波数帯(2.4GHz)も異なる。電気手術器は、出力と波形を切り換えることで切開と凝固(止血)の両方に使用できることから非常に便利であり、広く用いられている。凝固モードでは最大120W 程度、切開モードでは最大300W 程度が用いられる。混合切開(切開しながら凝固できるモード)、スプレー凝固(周辺にアーク放電を発散し、広い範囲を一度に凝固するモード)、ポリープ切除用の低い出力のモードなどを備える機種もある。電気手術器は電流を流すため一対の電極を要する。電極の形状により以下に大別できる。a. 単極式(モノポーラ)片方の電極を刃先として医師が手に持って切開・凝固したい部位にあてがい、もう一方として対極板と呼ばれる幅広の電極を患者身体にあてがうものである(図5・25)。刃先側では電流が狭い領域に集中するのに対して対極板側では広い面積に分散するため、刃先側でのみ発熱が生じる。対極板は普通は導電性の金属はくなどで、患者の身体の下にシーツのように敷いたり、粘着させて皮膚に密着させる。電極表面を誘電体で覆い、コンデンサとして用いる形式のものもある。b. 双極式(バイポーラ)両方の電極を手に持ってあてがうものである(図5・26)。普通はピンセットのような形状をした器具の先端に導通部分が露出していて、非常に狭い領域でのみ作用するようになっている。微細血管の止血などに用いる。c. 電気手術器の原理電気手術器は、多くの外科領域で広く使用されている。その効果によって分類すると以下のようになる。(1) 切開、止血(はん用手術器)いわゆる電気メスであり、はん用手術器として広く用いられる。出力を切り換えることで、切開と凝固(止血)の両方に使用できる。(2) 凝固壊死(腫瘍の治療など)腫瘍などの治療にも用いられ、ラジオ波凝固法(RFA)が代表的である。これらは針状の電極の使用により、経皮的に肝臓など深部臓器の内部の腫瘍を焼灼することで壊死させる。(3) 神経ブロック(疼痛治療など)腰痛などの疼痛で、痛みに関与する神経を焼灼することも行われており、アメリカでは腰痛治療に特化した電極が開発されている。以上のように多岐にわたり、目的に合わせた電極形状が開発されている。手に持って刃先を操作する以外の電極として以下のものがある。(1) 内視鏡処置具内視鏡のチャネルを通して患部まで導かれ、切除などを行う電気手術器がある。スネアは、環状のモノポーラ電極をもつ器具である。大腸ポリープなどに環状部分を引っ掛け、環を小さくして締め上げてから電流を印加してポリープを切除し、傷口を止血する(図(2) 電極針腫瘍治療、神経ブロックなどの目的のため、針形状をした電極を患部まで刺入して作用させる電気手術器がある。d. 安全性電気手術器は数十W から数百W の高周波大電流を印加するが、スパイク状のノイズを発生して付近に置いた電気機器に影響を与えることがある。また、心臓ペースメーカ、埋込み型除細動器など埋込み型の能動デバイスをもつ患者に対しては、電気手術器は注意して使用すべきである。図5・25 モノポーラ型電気手術器図5・26 バイポーラ型電気手術器1. 電極を掛ける2. 電極を絞る3. 電流印加、切除図5・27 スネアによるポリープ切除なお、電気メスでは大電流を通じても生命に危険が及ぶことがないとされる。電流を通じることで神経と筋肉、特に心筋が刺激により正常に機能しなくなるが、その電流しきい値は、周波数が高くなるにつれて高くなることが知られている。1kHz を超す交流電流では、周波数が10倍になると心筋が電撃を受ける電流のしきい値も10倍になるとされている[文献(32)参照。なお、文献(33)の図15ではこの考えに基づいた周波数特性をもたせている].ただし、火花を飛ばすとそのごく短い時間に火花の整流作用により直流に近い電流が生じうる。このため、患者が心室細動などを起こした事例が報告されている。
