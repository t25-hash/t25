# 10・2・2 ロジスティックスの進展と流通構造の変化

> **章**: 第10章 物流機械
> **節**: 10・2 ロジスティックスと物流機械

ロジスティックスの進展は，市場動向に従って商品の生産や原材れ，無駄なコストを抑えることが可能となる．
つまり，ロジスティローコスト化を実現することとなる．
れており，それぞれの段階で物流システムが構築されているため，ーコスト化は，これまで自社のコスト低減を視点に各種施策を実施は，ロジスティックスの進展に伴って，お互いに協力してコスト低も一つの企業のように顧客に対応することが可能である．
体ごとに物流システムを構築する視点から，流通全体を捉えて物流たっては，企業間の情報開示が必要となるなどデリケートな問題もって確実に進んでいる．
スに次のような変革をもたらした．
メーカから小売店までの多段階な流通構造が，簡素化されるよう②アウトソーシングの促進ムの構築や運営が得意な他の業者に任せることで，より効率的な運という仕組みを積極的に導入するようになった．
顧客への配送は，一括物流への取組みを重視するようになった．
物流機能を集約し，売り場の棚別商品をグループにまとめて同時到る．
物流センタでは，保管効率と作業効率は相容れないものである．
作業効率の悪化を防ぐためにセンタ内のエリアを分割し，作業オペためには，作業者の動きを少なくさせることが必要であり，コンピである．
じく，市場ニーズに合わせて，市場の求める量だけ，短時間で，ロ料，部品などの調達が一貫して動き出すので，適正在庫が維持さックスによって必要な在庫が必要量だけ配置されるため，徹底したわが国の流通構造は，メーカ，問屋，小売という多段階で構成さ流通コストを押上げる要因になっていた．
このような構造の中，ロしてきたが，それも限界に近づいていた．
そこで，各段階の企業減を求めるようになってきた．
この考え方は，サプライチェーンマ化の発展により双方が同じデータを見ながら行動するので，あたかサプライチェーンマネジメントは，これまでの流通を構成する主システムを構築する視点へ切替えるものである．
レーションの効率化を図ることが重要である．
