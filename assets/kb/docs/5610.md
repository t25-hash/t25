# 10・2・3 物流機械の活用

> **章**: 第10章 物流機械
> **節**: 10・2 ロジスティックスと物流機械

商品の多くは，消費者の多様化するニーズや，技術革新の速さに
よって，商品価値としての寿命が短くなってきている．このため企
業は，短期間のうちに物流機械を変更する可能性があると考えられ
る商品については，設備投資を慎重にせざるを得なくなってきてい
る．
また，物流システム市場は，景気後退による製造業からの需要の
減少や，生産拠点の海外シフトの増加によって低迷を続けている．
こうした状況では，サプライチェーンマネジメントの進展はますま
す重要となり，単品種生産型から多品種・小ロット生産型への移行
が必要となってきている．企業の中には，大量生産ではなく，多品
種，小ロットに適する屋台方式といわれる方式の採用も増えてい
る．
物流機械は，流通構造に対応する物流システムを構築するうえ
で，システムの中核となる物流センタを効率的に稼働させるための
重要な要素であり，適切な活用は生産性の向上につながる．例え
ば，効率的な保管を実現するための立体自動倉庫，またはセンタ内
作業のスピードと正確性の向上を実現するために，ディジタルピッ
キングシステム，自動仕分け機，さらにはバーコードを活用した検
品システムのような高度情報システムなどがある．
しかし，前述のとおり全般に商品のサイクルが短くなっているこ
とから導入に当たっては，
①今後の市場動向の把握（商品ごと）
②人手を使用した場合のコストと機械化に伴うコストの比較
③人手を使用した場合の作業精度の算定
④機械化する際のレイアウト上の問題
などを検討材料に判断する必要がある．
こうした状況の中，物流機械メーカは，サプライチェーンマネジ
メントの進展に対応するため，機械の性能向上を目的とした技術開
発を推進していくことが大切である．そして，さらに重要なこと
は，これまでのような単体機械の性能向上から，物流システム全体
の仕組みの中で機械の性能を考えることである．例えば，単にコン
ベヤの速度を早めて処理能力を向上させようとしても，物流システ
ム全体を通してみた場合，コンベヤの高速化が適切なことであるの
か，サプライチェーンマネジメントを目指すに当たって有効な方法
であるか否かを検討することが大切である．
物流機械メーカは，流通構造の各段階にどのような性能の機械を
導入することがサプライチェーンマネジメントの進展にとって最も
よい方法であるかを提案するとともに，機械を導入した後であって
も，現状に満足することなくつねに改善提案を行う姿勢が求められ
ている．
