# 5・5・1 低侵襲治療器具

> **章**: 第5章 手術・治療機器
> **節**: 5・5 手術用装置

外科手術の目的は，内部の病的臓器を切除し再建することであ
る．当初は，必要最小限の臓器切除をなるべく短時間に遂行できる
外科医が名医とされていた．しかし，がん治療において，転移の疑
われる部位を可能な限り広く切除すればするほど再発を防止でき治
癒率が向上することが明らかとなった結果，近年の外科学はひたす
ら拡大手術による遠隔成績の向上を目指してきた．けれども，複数
臓器の合併切除や広範なリンパ節郭清に伴う侵襲，ならびに体壁が
切開された状態で臓器が手術室内の低温低湿環境に長時間にわたり
さらされ続けるという侵襲が無視できない限界に達した結果，手術
は成功したが患者は助からないという事態が多発するに及び，ここ
10年余，低侵襲というキーワードが外科治療学の重要なテーマと
なってきている．
1986年にドイツ人のムーアが，次いで1987年にはフランス人の
F.モーレが腹腔内に二酸化炭素を注入し，トロカールを経て内視
鏡と細径の長鉗子（かんし）を挿入して胆のうを摘出した手術は，
その後急速に全世界に普及した低侵襲手術の先駆けとなった．実
際，内視鏡下手術と称される一連の低侵襲手術は，手術の際の必要
悪として長く容認されてきた，体壁を大きく切開し再び縫合修復す
るという操作を不要とし，体壁が大きく切開されない（5～10
mm）ため，腹腔内の高温多湿環境が維持されたまま手術を遂行す
ることができる．また，目的臓器周辺に限られた必要最小限の手術
操作となることにより，従来の大開腹手術と比べ著しく侵襲が少な
い．実際に術後の快復は早く，術翌日より歩行や食事摂取が可能
で，社会復帰も数日以内とされている（図5･28）．
しかし，内視鏡下の低侵襲手術は間接視下の遠隔操作手術となる
ため，適応になる手術はまだまだ限られている．また，そこでは，
患者にかかる侵襲が少ない反面，術者にかかるストレスが過大とな
っていることを忘れてはならない．実際，それに起因した，低侵襲
手術に特異な合併症や重篤な後遺症が，少ないながらも発生してい
ることは見過ごせない．特に気腹法手術においては，致命的な合併
症のため死亡し，訴訟に至るケースも散見される．低侵襲なはずの
治療が悲劇的な結末を招き，社会問題とさえなっている．
内視鏡下の低侵襲手術を安全・確実に遂行するには，腹腔内の手
術に必要な空間をいかなる方法で確保するか，術者の眼となる腹腔
鏡カメラの性能，ならびに術者の手となる細径の特殊長鉗子の性能
ならびに操作性の向上が必要不可欠である．
必要な手術空間確保法としては，当初，欧米で腹腔内に二酸化炭
素を注入する気腹法が開発されたが，高圧に起因するさまざまな合
併症が引き起こされることが問題となる．この点は，わが国の外科
医のアイデアによるつり上げ法によって解決を見ている（図5･
29）．
腹腔内の温度は37～38℃，湿度は95～100%に保たれているが，
手術室の温度は通常22～26℃，湿度は30～50%程度なので，当初，
腹腔鏡を腹腔内に挿入するたびに先端レンズが結露し，モニタ画面
が白濁して手術不能となる問題があった．これを解決するために，
光学視管内の照明用ファイバの一部を先端レンズを保持する金属片
で覆い，照明光の熱エネルギーを利用してレンズ表面を41℃に保
つアイデアによるフォグレススコープがわが国で開発された（図
5･30）．また，二次元カメラの性能は，CCD（charge coupled
device）単板式から3板式，さらにはハイビジョンへと急速に画質
の向上が進んでいる．さらに，三次元カメラも開発され，二眼2カ
メラ方式から二眼1カメラあるいは一眼2カメラ方式等，各種新し
い工夫がなされてきたが，人間の眼の自動輻輳性に比べるとまだま
だの感がある．また，固定視差角に伴う術者の眼精疲労の問題や，
左右の眼に画像を分割して知覚させる分だけ，それぞれの画質が二
次元カメラの画像と比べると若干低下するといった問題などもあ
り，いまだ広く普及を見ていない．

鉗子では，何十年の使用に堪える，手首の屈曲機能を備えた操作
性のよいさまざまなおじぎ鉗子がわが国で開発されている（図5･
31）．それにもかかわらず，保持，はく離，切離といった単純な先
端機能をもつ欧米製のディスポーザブル鉗子が手術室に氾濫してい
る現状は，医療経済の面からも問題がある．鉗子開発の最終目標は
手や指の機能の実現にあるが，鉗子の細径化が患者に与える侵襲の
低減に寄与する反面，機能性が低下するという問題をどう克服する
かという問題も残っている．
