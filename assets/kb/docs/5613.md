# 5・5・3 ガス

> **章**: 第5章 手術・治療機器
> **節**: 5・30 )。また、二次元カメラの性能は、CCD(charge coupled

a. 概略手術室で使用されるガスには、呼吸・麻酔管理に用いられる酸素、笑気等のガス、EOG(ethylene oxide gas)等のガス滅菌に用いられるガス、手術用ドリル等のME(medical elec-tronics)機器の駆動源としてのガス、腹腔鏡下手術における腹腔内作業空間作成のための気腹用のガス等が挙げられるが、本項においては気腹用ガスについて論じる。b. 気腹とガス腹腔鏡検査は、超音波断層診断装置やCT 等による非侵襲的な腹腔内実質臓器の診断が可能となるまで、肝臓や婦人科疾患の重要な検査手法として古くから用いられてきたものである。5・5・2項で論じた皮下鋼線腹壁つり上げ法が創始されるまでは、気腹法は開腹することなく観察と作業のための空間を腹腔内に作成しうる唯一の方法であった。依然として腹腔鏡検査の必要性はあるものの、超音波断層診断装置やCT、あるいはこれらを用いた画像誘導による経皮的穿刺生検法の普及により、その適応症例数自体は減少している。今日諸施設において倉庫の片隅に放逐されつつあった気腹器が再び注目される存在となったのは、腹腔鏡下手術における腹腔内作業空間の作成のためにほかならない。従来、腹腔鏡検査のために使用された気腹器は送気量が最大3L/min程度であったが、現在腹腔鏡下手術用に使用されている気腹器は、手術機器の出し入れや腹腔内の出血の吸引操作等に伴う腹腔内空間の減少を補うために10L/min程度の最大送気量を有するようになり、腹腔内圧の自動調整機能や異常内圧に対する安全装置を完備したものとなった。腹腔鏡検査のために使用された気腹用ガスには歴史的には空気、笑気、酸素、二酸化炭素を含めた数多くの気体が報告されている。しかし空気や酸素は空気塞栓を生じるばかりでなく、電気メス使用時などで引火・爆発の危険がある。また笑気は血液や腸管内腔へ移行しやすいという問題がある。このような理由から現在一般的に用いられているのは二酸化炭素である。二酸化炭素においても気腹圧が上昇すると致命的なガス塞栓を生じうるため、その気腹圧は12mmHg 以下とすることとされており8~10mmHg の低圧気腹が推奨されている。気腹は腹腔内圧を上昇させることによる循環動態の悪化や、後腹9 ―84医療・福祉・バイオ機器膜腔に対する圧力から静脈内における血栓形成の可能性等といった副作用、さらに気腹ガス自体による腹腔内の生理的温湿度環境破壊や腫瘍細胞散布・定着の促進等の問題点を抱えている。これらの根本的な解決には、5・5・2項で論じた気腹を用いない腹腔内空間作成法(皮下鋼線腹壁つり上げ法等)を用いるしか方法は存在しないが、気腹法による腹腔鏡下手術を施行している施設においては、可及的な低圧な気腹法の使用、手術中の抗凝固剤の予防的投与や下肢のマッサージ器の併用、使用ガスの加温、加湿等の対策が施されている。
