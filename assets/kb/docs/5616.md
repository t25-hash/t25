.定置型ベルトコンベヤ（fixed belt conveyor ) 通常，
ベルトコンベヤは地上に設けたコンクリート基礎または建家の床面
に基礎ボルトによって取付け，プーリによって駆動される．コンベ
ヤの駆動は1個のプーリにより駆動されるシングル駆動と，複数の
プーリにより駆動されるマルチプル駆動とがある．長機長コンベヤ
では，コンベヤのヘッド部およびテール部に複数の駆動プーリを設
け，ベルトに発生する駆動張力を分散してベルト最大張力を減少さ
せ，強度の低いベルトを採用して設備費を下げるように計画するの
が望ましい．
下り傾斜のコンベヤでは無負荷運転ではプラス動力であるが，運
搬物を積載し始めると動力が徐々に減少し零となり，全長積載では
マイナス動力となるものがある．電動機は普通の三相誘導電動機
（かご型）を使用し，マイナス動力時には自動的に制動運転となり，
発動した電力は逆送される．通常，大型の下りコンベヤでは停止時
の制動には電気制動と機械制動を併用し，駆動プーリはテール部分
に設ける．

り，居住地域を通過する場合は外部を防音材で遮へいし，防音型ロ

.移動型ベルトコンベヤ（movable belt conveyor ）
(１) シフタブルベルトコンベヤ（shiftable belt conveyor )
主として土砂採掘工事や埋立工事用に使用され，コンベヤ全体を横
方向に移動可能な構造となっている（図10･84)．ヘッド部，中間
部およびテール部の各フレームはそれぞれユニットとして分割さ
れ，そり状として移動に有利な形状となっている．これらのユニッ
トは縦方向のレールに接続され，このレールを特殊な移動装置で横
方向に引張って移動させる．通常，シフタブルベルトコンベヤのシ
フト量は50m程度とし，機長1000m程度の50mシフトに要す
る時間は約8時間ないし12時間程度である．
(２) オートシフトベルトコンベヤ（auto shift belt conveyor )
埋立工事における撒土専用機で，一定の間隔にクローラを配置し横
行，扇形旋回など土砂運搬をしながらでもシフトができる（図10･
85)．
埋立地におけるシフタブルベルトコンベヤや定置型ベルトコンベ
ヤとスプレッダを連絡し連続的に広範囲な埋立ができる．埋立の進
度によって移設が必要なときは，クローラをその場旋回して縦行，
斜行などの移動方向を自在に選択できるため，自走移動が極めて容
易である．
(３)スプレッダ（spreader ) 埋立地までコンベヤで運搬さ
れた土砂を埋立所定の場所に投入するための機械である．移動用ク
ローラの上に受入れ用コンベヤと排出用コンベヤを設けたもので，
通常それぞれのコンベヤはクローラ中間部を中心として旋回できる
構造となっている（図10･86)．一般に土砂投入部分は排出土の積
高さが高いほど崩れやすいため排出用コンベヤのブームを長くして
おく必要がある．
(４) トランスファベルトコンベヤ（transfer belt conveyor )
増大し，30 °程度の角度まで輸送できる．
クローラ上に起伏および旋回の可能なコンベヤを設けたもので，ク
