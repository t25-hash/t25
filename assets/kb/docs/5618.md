# 5・5・8 RFA(radio frequency thermal ablation、ラジオ波

> **章**: 第5章 手術・治療機器
> **節**: 5・41 )。

熱凝固療法)肝腫瘍(liver tumor)に対する低侵襲治療法としてRFA は急速9 ―87図5・43 PRS の内部構造に普及が進んでいる。RFA は電気メスと同様、RF(radiofre-quency)波発生装置と、これを導出して生体に通じさせる穿刺電極、および生体から電流を回収する対極板からなり、RF交流電流および組織インピーダンスによるジュール熱と誘電加熱を利用して生体タンパク質を凝固変性させ、肝腫瘍等を治療する装置である。電気メスが電極接点で高温を発生し組織を切開するのに対し、RFA ではより広く組織を熱凝固させるため、通電中も電流をよく保つことが肝要である。そのため、電流遮断の原因である電極周囲組織の炭化を避けるべく、温度やインピーダンスをモニタして適切なRF出力とすることや、電極を冷水冷却するなどの工夫がなされている。RFA は当初、不整脈等に対するカテーテルRFアブレーションや、微細電極を使ったアブレーションの延長から、バイポーラ電極を用いて2本の電極間の交流電流と組織インピーダンスによる発熱を利用する装置として開発されたが、期待の焼灼領域を得ることが難しく、対極板を使用し、インナフック(inner hook)をもつモノポーラ電極の利用へ移行していった。インナフックは1,2,3個のものが開発されたが、現在ではより球形の熱凝固が得られるよう多数本のインナフックをもつモノポーラ電極が主流であり、これらを用いると直径3~4cmの組織熱凝固が可能である。また、電極先端を冷水冷却するクールチップ(Cool-tip)タイプのモノポーラ電極も遅れて開発が行われ、電流遮断の原因である電極周囲組織の炭化を避けることで、熱伝導により直径2~3cmの熱凝固ができる。肝臓を電気的モデルと考えると、肝実質は比較的高い電気抵抗の固まりであり、その中を電気抵抗の低い血液や胆汁で満たされた血管・胆管が貫いている。単位体積当たりの発熱率は、この立方体の各辺の抵抗と、これを流れる電流から計算される。この抵抗の固まりの発熱も、組織血流によって熱希釈される。肝門部で血流を遮断する方法を併用すると熱希釈が減少し、その結果肝組織温度は早く上昇し、熱凝固も広範囲となる。また、血液と胆汁は肝組織に比較してインピーダンスは低く、RFによる温度上昇は少ない。これらからRFA はマイクロ波など他のモダリティと比較して、胆管損傷や脈管閉塞の危険が少ない安全な方法であるといえる。また、MRIによる組織温度モニタリングにRF波は不適であるが、RF通電を間欠的に行うよう工夫すれば可能である。現在、市販されている3社の装置を比較してみる(表5・6)。いずれの電気手術器でも出力およびインピーダンスのモニタリングが可能で、RITA 社とラジオニクス社製では電極先端部の温度をモニタできる。穿刺電極はRITA 社とラジオセラプティック社製ではインナフックをもち、ラジオニクス社製は、電極内に冷却蒸留水を灌流させることによりインナフックがなくてもだ球状の熱凝固ができる。臨床効果に装置間の差はない。
