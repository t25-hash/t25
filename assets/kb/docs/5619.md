# 10・4・2 棚設備

> **章**: 第10章 物流機械
> **節**: 10・4 保管設備

棚設備は固定ラックと自動ラックに大別することができる．
固定ラックには重・中・軽量ラック，ドライブインラック，スラ
イドラック，カンチレバーラック，ハンガラックなどがあり，自動
ラックには回転ラック，流動ラック，移動ラックがある．

a.固定ラック
.重・中・軽量ラック パレットラックやケースラックに
代表される固定ラックであり，棚に収納できる重量によって分類さ
れる．重量ラックはおもにパレットに積載された荷の保管に用いら
れることの多いラックで，パレットラックと呼ばれることも多い．
これらのラックは，保管している荷の先入れ先出しができるので荷
の性質，品種の多少にかかわらず使用できる非常に汎用性の高い設
備である（図10･151)．ラックは支柱と棚から構成されており荷の
高さに応じて棚を上下に移設することもできる．
.ドライブインラック パレタイズされた保管物をフォー
クリフトにより入出庫する棚であり，フォークリフトの通路がその
まま保管スペースになるため，極めて保管効率の高い棚である（図
10･152)．この棚は入出庫を一方向からのみ行うため先入れ後出し
となり，少品種多量の荷を保管するのに適している．また，上段の
棚に荷を入出庫する場合，フォークリフトを必要高さまでリフトし
たままラック内を走行するため，奥行の長いドライブインラック
は，作業性を良くするためにフォークリフトの下部にガイドロー
ル，ラック下部にガイドレールを取付けたカイドレールシステムを
併用することもある．
.カンチレバーラック 支柱と支柱に接合された片持ばり
で構成されているラックで，保管する荷はこの片持ばりで支持して
保管することになる．通常はパレタイズされていないパイプ，鋼
材，巻物等の長尺物がベニヤ板，鋼板等の保管に使用されることが
多い．
b.自動ラック
.回転ラック 回転ラックは水平形と垂直形に大別され
る．回転ラックは棚が自動的に手元まで回ってくるため，保管やピ
ッキング用の通路が不要という特長から多品種少量，小物部品の保
管効率向上およびピッキング効率向上という目的で利用される．た
だし，荷をラックに補充する場合，出荷作業と同時に並行して行う
ことが困難なことが多いため，設備に際しては，作業スケジュール
を十分に考慮する必要がある．
回転ラックには次のような種類があるが，最近はコンピュータを
使って情報管理機能を有する小物自動倉庫としての使用例がある．
(１)水平型
複数の棚を有する一体構造のラックが水平に回転するもので，小
物部品の大量保管・人手によるピッキングに適した回転ラックであ
る．棚の取付けは自在であり，収納物に応じて変更できる．
またラックを各段ごとに個別に，しかも任意方向に水平回転して
呼出し口に自動的に停止させる型式のものもある．
(２)垂直型
天井空間を最大限に有効利用でき，しかも手元まで自動的に回っ
てくるので，定位置作業ができるのが垂直型の最大の特長である．
事務所の書類保管から工場の小物部品・電子部品等軽量小物，さ
らには最近は食器保管等にも利用されるなど，各分野に適用範囲が
広がっている．
.流動ラック 保管物が棚部分に取付けられたロールコン
ベヤ，ホイールコンベヤなどによって出口へ向かって移動してくる
ラックである（図10･153)．重力で移動するタイプと駆動コンベヤ
によって移動するタイプがある．前者は設備費用が安価だが前面の
荷に後方の荷の重力が加わり，衝撃を与えたり破損したりすること
がある．後者はこの問題はないが前者に比べて高価である．
このラックは先入れ先出しができ，格納効率が高く，しかも入出
庫は棚の前後で行えるので作業性が良い．格納物の底面形状，荷重
のばらつきにより流れ速度が左右されるため，途中で止まったり，
流れすぎて暴走したりする欠点もある．この流動ラックにはパレッ
ト単位の重量用と，カートンケースやプラスチックコンテナによる
ピッキング作業を主体とした軽量用がある．
.移動ラック 移動ラックはラックが一列ごとに独立して
動くため，何列の棚であっても作業通路は一列だけあればよい．そ
のため設置面積が同じならば固定ラックに比べはるかに多くの棚が
設置でき，収容効率は格段に増える（図10･154)．
倉庫や事務所，図書館などで多く使われている事例を見ることが
できる．ラックの移動を電動式で行うものと，動力を用いず人力に
より棚を動かすものに分類することができる．
移動ラックは床に角鋼または軌条レールを埋込み，台座に車輪
（両端にはフランジ付車輪）を取付け，この車輪を動力あるいは人
力によって駆動して，上部に載せたラックを移動させるものであ
る．
動力を使用した移動ラックの場合，人がラックに挟まれるのを防
止するため非常用バンパ，進入検出装置，開通路インタロック，電
源投入インタロック等二重，三重の安全対策が施されているのが一
般的である．
