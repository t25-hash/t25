# 5・5・10 コンピュータによる手術支援

> **章**: 第5章 手術・治療機器
> **節**: 5・41 )。

コンピュータは医療の質の向上にも大きく貢献してきた。手術支援においても、すでに術者を補助する第二の脳として、治療計画から治療(手術)の実施、術後管理、アウトカム評価まで、陰に陽に活躍している。以下に述べる手術ナビゲーションシステム、手術マニピュレータシステムがその代表である。コンピュータを上手に利用することでケアレスミスなどヒューマンエラーを未然に防止し、より精密、より確実な医療を実現することが期待される。一方、治療のポイントが医師の手を離れ機械によって左右されることに対する危うさも指摘されている。新しい技術は、その効果と安全性、そして限界が広く医師に認識されるまでは、たとえそれが安全のためであっても未知のリスクをもつことを忘れてはならない。a. 手術ナビゲーションシステム(surgical navigation system)低侵襲化の進行に伴い、いま操作している部位はどこかの判断が容易でなくなってきた。そこで、位置センサにより座標計測して解剖的な知識との対応を得る手術ナビゲーションシステムが開発された。発想および原理的な構成は自動車のナビゲーションシステムと同じである。GPS(global positioning system)に相当する位置センサ、地図データに相当する断層画像からなる。ナビゲーションでは、地図としての画像データと位置センサによる実測値との間のレジストレーションを行う(レジストレーションに関しては5・3節参照)。レジストレーションで生じる誤差は、ナビゲーションの精度を大きく左右する。すでに関節置換術や脊椎修復などの整形外科手術、腫瘍切除術などの脳神経外科手術では、高精度な手術を行うことで治療成績が向上することが示唆されており、普及しつつある。. CTベース(CT-base)、CTフリー(CT-free)近年、断層画像データとのレジストレーションを必須としないナビゲーション法が考案されている。ナビゲーションシステムに含まれる位置センサだけを用いて、対象とする手技で求められる幾何学的な拘束条件が満たされるように術具操作を補助するものである。断層画像を必須としないことからCT フリーナビゲーション(CT-freenavigation)などと呼ばれ、これに対し画像とのレジストレーシ9 ―89図5・48 脳神経外科手術用の微細操作マニピュレータNeuRobot[提供:(株)日立製作所]ョンを行うものをCT ベースナビゲーション(CT-base naviga-tion)と呼ぶこともある。一例として、膝の人工関節置換術で大腿骨側の関節を取付ける際の位置決めが挙げられる。正立時の荷重方向に従い、股関節側の大腿骨頭から膝関節の中心を結ぶ線分に垂直になるように人工関節を取付ける。CT ベースナビゲーションでは術前のCT 画像で大腿骨頭中心、膝関節中心を計測して荷重線分を同定し、術中に画像と大腿骨をレジストレーションして荷重線分に垂直になるように、損傷した膝関節を除去する。レジストレーションのため、大腿骨に位置計測マーカを固定する。CT フリーナビゲーションでは、膝関節の中心は切開後に直接ポインティングして取得する。大腿骨頭の中心は、これが中心となるように大腿部を動かして、大腿骨に固定した位置計測マーカの位置を計測して円の方程式に当てはめて決定する。これによって荷重線分が得られ、あとはCT ベースナビゲーションと同じである。レジストレーションの誤差を含まないため、CT ベースナビゲーションよりも精度的に有利と考えられている。b. 手術ロボット(surgical robot)いわゆるロボット機構を治療に用いるシステムはいくつか発表されているが、2007年現在の段階では多くが基礎研究の段階である。ロボットの役割は人間の素の能力を超えることと総括できるが、以下がそのおもなものである。(1) 低侵襲手術支援内視鏡手術の不便を補う機構。狭所での結紮(けっさつ)を容易にしたりする。マスタスレーブ式マニピュレータによる製品が海外で市販されている。5・3・11項にて詳説する。(2) 高精度手術1990年代初めにアメリカで開発が行われた股関節置換術におけるステム部挿入のための開削を行うNC(numerical control)機構が、欧米で市販され、多数の臨床実績を上げている。ステムの形状に合わせた穴を所定の位置に正確にあけることができる。このほか、定位脳手術を行うシステムなどが開発されている。高精度化は医療の安全などにも寄与する。(3) 医療の合理化低侵襲手術では術者とは別の1人が内視鏡の視野を操作する役割を担うが、術者と息が合わないと不自由であるうえ、それだけにスタッフを充てるのは不経済でもある。そこで、術者が内視鏡の視野を操作する機構が開発され、国内でも販売されていた実績がある。そのほか、放射線治療装置と組合わせて線源位置を精密に操作するものなどが実用化されている。
