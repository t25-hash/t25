# 10・4・3 貯槽

> **章**: 第10章 物流機械
> **節**: 10・4 保管設備

粉粒体の貯槽（storage vessel）は，原料・中間製品・製品出荷を目的として使用される．
その特徴は，次のとおりである．
倉庫貯蔵に比べて数倍の貯蔵能力がある．
品質管理に必要な温度・湿度・水分・ガス濃度計測，くん 粉じん飛散が少なく，騒音も比較的小さい．
角すい型をし，下部に供給あるいは排出装置が付いている場合をホ粉粒体の輸送方式は，方法によって分類すると表10･29 に示すとおりである．
また，貯槽を計画するに当たって注意しなければならない一般的な事項 は次のとおりである．
位置 搬入，搬出，全製造工程が合理的に行えるよう計画する．
形状 地耐力，地震力，風圧に対する強さ，敷地の形，周囲の状況などから制約されることが多い．
の一時的な保管，物質の物理・化学・生物的な操作および備蓄などで十分考慮する必要がある．
袋づめに比べ，大量の取扱いが可能である．
蒸，入替えが容易なため，変質・虫害が防止できる．
る有効面積をもち，かつ上部に空間をもつように計画する．
地耐力 できるだけ地耐力の大きい場所に設置する．
搬入・搬出装置 粉粒体の性状と，貯槽の使用条件に応じて適切な方式と機器を選定する必要がある．
