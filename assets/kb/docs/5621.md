b.コンテナ（container ) 一般にはすべての容器を指すが，
ユニットロードシステムにおけるコンテナとは，次の条件を満足す
る輸送用容器をいう．
(１)長時間反復使用に耐える十分な強度をもつ．
(２)貨物の積替えなしで，各種輸送機関にまたがって輸送でき
るように特別に設計されている．
(３)積替えを容易にする装置を備えていること．
(４)貨物の積込みおよび取出しが容易である．
(５)内容積が1m以上ある．
