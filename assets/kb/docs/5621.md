# 5・5・11 マニピュレータシステム

> **章**: 第5章 手術・治療機器
> **節**: 5・41 )。

内視鏡手術を補助するマスタスレーブ型マニピュレータは最も普及に近いと考えられている(図5・48)。それは、内視鏡手術の進展とともに顕在化した問題点をうまく解決しているからである。医師は、開腹下での従来の手術では、術部に手首を近づけて手首から先9 ―9 0医療・福祉・バイオ機器(a) スレーブ部(b) マスタ部図5・49内視鏡手術用マニピュレータ、ダ・ヴィンチのマスタ部とスレーブ部(提供:インチュイティブ・サージカル社)を効果的に使って結紮など微細な作業をこなしてきた。内視鏡手術では細長い鉗子を用いて手術を行うが、腹腔などに挿入したトロッカが支点として作用する。鉗子の先端を1cm動かすには、てこ比に応じて術者の腕全体を動かす動作が必要となる。この状態で微細な作業を行うことは容易でなく、また従来の術式と比較して疲労しやすいことを示唆している。手術マニピュレータシステムは、手首を使う動作を復活させた。有名なダ・ヴィンチ(da Vinci)システム(インチュイティブ・サージカル社)は、ワイヤ駆動で動作する4自由度(鉗子動作を含む)の手首機構をもつ(図5・49)。術者は操作卓に座って手首の操作に集中することができる。さらに、術者動作の震えをフィルタしたり、術者動作を縮小して鉗子を動作させることができ、微細作業がより容易になっている。このほか、脳神経外科手術用マニピュレータも開発が進んでいる。
