# 11・1・1 障害者支援としてのバリアフリー技術

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・1 概説

音声案内も最初からつける．
後から追加するより，コストもかからず，デザインも美しく仕上がる．
麗々しく障害者向けといわなくても，子供も，たまたまけがをした人も，海外からの旅行者もみなが助かる．
次第にその考え方は各企業や自治体に広まっていった．
いまでは静岡，埼玉，熊本，岩手，高知などさまざまな自治体が，まちづくりの基本理念にUDを挙げている．
また，ものづくりの基礎としてデザインセンター全員にUDの考え方を学ばせている企業も増えてきた．
UDの7原則は次のようになっている．
物事をデザインするうえでは当たり前のようなことばかりであるが，実際にはこれを守って作られ，アクセシビリティとユーザビリティを兼ね備えた製品は国内ではまだそれほど多くない．
北ヨーロッパなどではUDとうたってはいないが，結果としてUDになっているものも多い．
原則7：アクセスしやすいスペースと大きさを確保すること障害者や高齢者の生活を支援するため，これまでさまざまな技術が開発されてきた．
つえや眼鏡といった身近な日常の生活用品に始まり，駅の階段昇降機や電動車いす，点字プリンタといった大きく高額なものまで，その範囲は広い．
これらは，国内ではバリアフリー機器と呼ばれることが多い．
建築や住宅設計，まちづくりや公共交通も，障害者や高齢者を対象にした製品や設計手法は，総称してバリアフリーと呼ばれる．
かつては障害者向けというとデザインも限られ，コストも高いものが多かったが，いまでは福祉機器展などの展示会場にはカラフルな機器や日用品が並び，多様性を増してきているのがうかがえる．
障害を軽減する研究も多様化しており，さまざまなロボットによる生活支援やバーチャルリアリティによる歩行訓練，携帯電話や最先端の技術を用いた研究も多い．
重度障害者の意思決定を支援するコミュニケーション技術の領域では，一般的なマウスやキーボードによる入力の代わりに視線や脳波を使ったり，音声だけで入出力が可能なインタフェースも開発されてきている．
今後は，医学やリハビリテーション工学，ヒューマンインタフェースや認知心理学など，多くの学問領域をまたがる学際的な研究が求められている分野であるといえよう．
