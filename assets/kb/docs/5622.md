# 5・5・12 能動鉗子(active forceps)

> **章**: 第5章 手術・治療機器
> **節**: 5・41 ).

低侵襲手術を支援するマニピュレーション技術としては,内視鏡処置具の一種である小型の鉗子,血管内カテーテルなどがある.現在用いられているものは2~3自由度程度で,手元で手動操作する簡単な機構であるが,駆動源を有し多くの自由度をもち複雑な形状をとる能力をもつもの,ある程度の保持力を要する能動鉗子が研究されている.将来的には,現在臨床研究が進められているカプセル内視鏡に小型の能動鉗子を取付けたものも構想されている.今後の手術ロボット技術は,マイクロ化,ピンポイント化,ユビキタス化の方向をたどるものと予想される.その際に重要なのは人間と機械がお互いの得意分野を分担することである.すでにCT やMRIなどの断層画像システムと組合わせて画像誘導手術を行うシステムが多数発表されている.画像上で計測された座標値どおりに手術を行うことは人間の術者にとって必ずしも容易なことでなく,理にかなった応用といえる.ユビキタス化は患者にとってのユビキタスと医師にとってのユビキタスの両方が考えられる.マスタスレーブ型のシステムでは,両者間の距離を長距離にすることに関して遅延以外の技術的困難はない.大西洋を挟んでの遠隔手術の実験などが行われている.遠隔手術は患者と医師の双方がお互いの住居のそばで双方の満足度の高い医療を実現する可能性がある.一方で,単純に人間の動作を機械で置換えるだけのシステムは,それが医療の真のニーズに合うものであるか厳しく問われるであろ図5・50 種々のカテーテルの先端形態う.これからの手術ロボット研究は医療上の研究とノウハウの集積も併せて進めることが不可避で,そのような医工連携が不可欠となる.
