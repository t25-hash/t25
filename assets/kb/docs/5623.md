# 5・5・13 腹部領域におけるカテーテル

> **章**: 第5章 手術・治療機器
> **節**: 5・41 )。

a. 概略本項においては腹部領域で使用されるカテーテル(catheter)とともに、消化器内視鏡において内視鏡の鉗子孔を介して使用される処置具の一部について論じる。b. 消化器病領域での血管カテーテルとその他のカテーテル血管内カテーテルは血管造影という診断目的で使用が開始された。当初は皮膚を切開し直視下に露出した血管からカテーテルを挿入していたが、1953年のSeldingerによる経皮的血管穿刺法の発明により、局所麻酔下に皮膚を直接穿刺することでカテーテルの挿入が可能となった。また血管造影も大動脈内に挿入した直線カテーテルから造影剤を注入する亜選択的大動脈造影から始まったが、目的血管の分岐形態に合った先端形態のカテーテルを使用し、目的とする血管分枝内までにカテーテルを挿入することで、選択的血管造影が可能となった(図5・50)。またカテーテルの先端からガイドワイヤを目的とする末梢側の血管分枝内に進め、このガイドワイヤを支索としてカテーテルを押し込むことにより、より末梢側の血管分枝に至るまで選択的にカテーテルを進めることが可能となった。このような選択的カテーテル挿入技術によって、選択的造影や抗がん剤・塞栓物質等を対象臓器のみに選択的に血管内腔から投与することも可能となった。さらにバルーン付きカテーテルやステントによる血管狭窄の拡張も可能となり、血管造影の手技は診断のみならず重要な治療手技ともなった。これらの血管造影カテーテル等を駆使した治療をinterventionalradiologyと総称している。担がん臓器を支配する動脈内に留置したカテーテルから抗がん剤を直接注入する動脈内注入化学療法とは目的臓器のみに集中的に抗がん剤が投与できるので、治療効果の増強と副作用の軽減の双方を可能とする治療法である。従来は全身麻酔下の開腹手術によって目的の血管を露出しカテーテルを留置していたが、interventionalradiologyの技術により開腹することなく局所麻酔下にカテーテルを留置できるようになった。これは開腹手術によらずにカテーテル留置が可能となったというだけでなく開腹手術においては腫瘍の浸潤等の理由で血管の露出自体が困難である場合でも、安全に血管内にカテーテルを留置できることや、腫瘍に行く複数の血管を血管内塞栓物質などで閉そくさせ、1本のカテーテルからの選択的な薬剤投与が可能となるなど、interventional radiologyならではの福音も大きい(図5・51)。図5・51の症例では上腸間膜動脈(SMA : superior mesentericartery)から肝動脈が分岐する症例に対し、肝動脈の分枝である胃十二指腸動脈(GDA : gastroduodenal artery)を塞栓用コイルで閉そくさせたのち、留置用カテーテルをここに挿入、留置カテーテルの側面にあけた側孔から肝動脈内に薬剤投与を図っている。従来の開腹手術では対応困難であるような症例である。さらに血液凝固によるカテーテル閉そくに対する対策として、出月、(株)東レにより開発されたヘパリン化親水性ポリマ(heparin-ized hydrophilic polymer、アンスロン윣윃)はカテーテルの開存性図5・51 Interventional radiologyによる動脈内注入カテーテルの留置図5・52 PTCD とPTCD を利用した狭窄拡張用ステントの留置(出典:Boston Scientific社カタログ)を大幅に高め、血管内留置カテーテルを用いた治療成績とQOLの向上に寄与した。カテーテルは血管内だけでなく、胆管・尿管等にも使用される。悪性腫瘍、結石、炎症等による胆管・尿管の狭窄、閉そくに対して従来は開腹手術により狭窄の解除やバイパス手術を行うほかはなかった。しかし超音波断層診断装置によるリアルタイムの画像誘導下に狭窄部より上流の拡張した管腔を経皮的に穿刺し、血管造影と同様にガイドワイヤを支索としてカテーテルを留置することで、胆汁や尿を体外に誘導することも可能となった(PTCD : per-cutaneous transhepatic cholangiodrainage、経皮的経肝胆道ドレナージ)。さらに血管と同様に、拡張用バルーン付きカテーテルや形状記憶合金を用いたステントによる狭窄部位の拡張を局所麻酔下に経皮的に施行することも可能となった。これらの技術もinterventionalradiologyの一分野として広く臨床に供されている(図5・52)。このようなカテーテルを用いた治療の進歩と普及には、カテーテルやガイドワイヤの先端形状、太さ、材質等の改良による操作性の向上によるところも大きい。しかし今日カテーテル操作における手技上の困難性はカテーテルやガイドワイヤの先端形状を血管や胆管内で自由に可変できないことと、カテーテルの先端に挿入のための駆動力がなく、数十cmから1m前後離れた術者の手元での操作(出し入れとねじり)に頼っていることに起因している。このためカテーテルのたわみからトルクの伝達が不十分となったり、挿入中のカテーテルが手元の操作方向に反して目的の血管から逸脱することなどがしばしば見られる。これらの問題点は術者の技術により補われているのが現状であり、この面でのハードウェアの研究開発が強く望まれている。c. 消化器内視鏡処置具消化器内視鏡には種々の処置具の通路となる鉗子孔があり、常用される消化器内視鏡においては3mm前後の直径を有するチャネルが一般的である。この鉗子孔を介して使用される器具としては、組織採取用の生検鉗子(5・5・6・b項参照)のほかに、内視鏡的治療のための種々の処置具がある。例えば、ポリープ切除のための電気通電可能な切除用スネア、結石破砕用スネア、止血用クリップとクリップアプライヤ、薬剤注入用穿刺針、ヒートプローブとユニット、止血やがんのレーザ治療用のレーザ用ファイバ等である(図5・53)。これらは早期消化器がんの内視鏡による根治切除や、結石の低侵襲治療を可能9 大腸内視鏡下のスネアによる大腸ポリープ切除としている(図5・54)。特に電子内視鏡の普及と進歩は従来の光学内視鏡に比べて大型ディスプレイ上の高画質の映像を複数の人数で共有できることもあり、内視鏡による消化管病変の治療に大きく寄与している。今日の消化器内視鏡における治療器具の操作上の問題点は、一部の内視鏡を除き鉗子孔は1本であり、1本の処置具で病変を処置せざるを得ないこと、そして処置具の先端形状のコントロールが困難であるので、出し入れ以外の方向への処置具の操作には内視鏡の先端自体の屈曲を利用せざるを得ないことにある。このため、通常の外科手術のように複数の手を用いた繊細な操作が、内視鏡においては困難となっている。消化管内腔に挿入する消化器内視鏡では、その口径に必然的な限界が生じるので、鉗子孔の数と口径にも限界が存在することも自明であるが、血管カテーテルと同様に自律的な先端の制御と駆動が可能な処置具の開発も待たれるところである。
