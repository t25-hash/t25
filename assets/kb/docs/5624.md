# 11・1・3 高齢社会日本に紹介されたユニバーサルデザインの概

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・1 概説

b.ユーザの成熟 UDが日本に根付くと思われる二つめの論
拠は，ユーザの成熟である．50歳以上の向上心，向学心に燃えた
ユニバーサルデザイン（UD:universal design）とは，1990年
比較的裕福な世代をターゲットにした雑誌は花盛りであり，それら
の中でも良質で使いやすく配慮され，かつ高級感のある機器や生活
用品が多数紹介されるようになった．このようなマスコミの変化や
消費者文化がようやく成熟するきざしを見せるのであれば，歓迎す
べき傾向といえる．
かつて，日本の消費者は不平を言わなかった．もし購入した機器
が使いにくくても，返品に行くことはまれだった．大量生産，大量
消費の時代には，メーカ側の論理のほうがユーザの希望より強かっ
た．しかし，いまやニーズが多様化したユーザは，他とは違う自分
だけに合うものをメーカに要望してくる．個々人がそれぞれ違うの
だから，ほかならぬ自分に合ったものを要求するのは当然の要望で
ある．多品種少量生産やオンデマンドアセンブリーがそれを可能に
し始めており，企業が多様なユーザの求めに応じるという状況は今
後も加速するものと予測される．
c.ITの進歩 ユーザの多様な要望を可能にしたのは，IT
（information technology）の進展による部分が大きい．車や椅子，
洋服や靴まで，多様な各人の希望を聞きながらオンデマンドで最終
製品を完成させることが一般的になりつつある．いまや企業の意思
決定は，企業側ではなくユーザ側に移っている．ITで主導権を握
ったユーザの意向を無視しては，市場をつかむことはできなくなっ
ている．そのユーザ層が高齢化している以上，メーカはもはや作り
たいものを作るのではなく，ユーザの求めに応じてユニバーサルな
製品を作ることが至上命題となる．
また，ユーザ側がITを使って企業や公企業に意見が出せる環境
が整ってきたことも，UDの進展を加速している．シニア層や女性
層など，UDの必要性のある層は，ITで建設的な意見を述べるこ
とが可能となった．ITが成熟した市民社会や力のある消費者層を
形成する助けとなっているのである．
以上，三つの論点で，今後，日本のUDが進展するという見通
しを述べた．アメリカにおいては「リハビリテーション法508条」
のように，法律で産業界や行政を規制するというアプローチが多
く，ヨーロッパの場合はISO13407：1999のように，デザインの
プロセスを変えていこうという動きが急である．日本においては法
律や国際標準での規制も効果があると思われるが，最も有効なのは
市場の動向であろう．欧米よりもはるかに高齢化率の高い日本にお
いて，ミドルやシニアがITを武器にまちづくりやものづくりに意
見を言い始めるとき，日本独自のUDが根付くものと思われる．
今後の進展が期待される．
