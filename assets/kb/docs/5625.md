# 5・6・1 RI(radioisotope)治療装置

> **章**: 第5章 手術・治療機器
> **節**: 5・6 放射線治療

RI治療装置には,コバルト60遠隔照射装置,リモートアフタローダ,ガンマナイフがある.このうち,コバルト60遠隔照射装置は現在ほとんど使われていないため,記述を省略する.また,ガンマナイフは5・6・3項に述べられている.したがって,ここではリモートアフタローダ(remote afterloader)についてのみ説明したい.図5・59は代表的なリモートアフタローダであるマイクロセレクトロンHDR(high dose rate,高線量率)を示している.図5・59中のレンコン状に穴のあいた部分にイリジウム192(irridium-192)線源37GBqが格納されていて,遠隔操作によりチューブを通して患者に挿入される.線源チューブは細くフレキシブルであり,さまざまな管腔(消化管,鼻腔,子宮など)に挿入するだけではなく,乳房や頭蓋内などの組織へ刺入することも行われる.使用線源は一つであるが,線源チューブは複数使用でき,プログラムにより線源を移動させる.マイクロセレクトロンHDR は日本で100台程度使用されている.リモートアフタローダにはほかにセシウム137を用いる低線量率マイクロセレクトロンや,コバルト60を用いるラルストロンなどがある.
