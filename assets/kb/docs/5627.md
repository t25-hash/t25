# 11・2・2 住宅のバリアフリー，公共建築物のバリアフリー

# 11・2・2 住宅のバリアフリー，公共建築物のバリアフリー

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・2 建築・住宅

a.住宅のバリアフリー わが国の住宅は，柔らかな畳があっ
て木のぬくもりが伝わる，一見高齢者に優しい建築に思われがちで
あるが，段差が多いこと，かつての尺貫法の影響が現在でも極めて
強いため，これまで常識的に決められていた廊下・階段・開口部等
の幅員が高齢者の室内移動に適していないこと，室面積が狭いこと
などに加えて，いわゆる和式の生活様式が高齢者には不向きである
ことが指摘されている．その結果，家庭内事故の発生の多いことが
指摘され，また寝たきり高齢者やおむつ使用者の増加の一因になっ
ているとの指摘もされている．
しかし，仮に身体機能が低下したとしても，在宅でできる限り自
立した生活を少しでも長く継続できるような住宅（バリアフリー住
宅）があれば，これらの問題が解消できるとともに高齢者自身の自
立と意欲の拡大，生きがいにつながり，家族の介護力・量の軽減，
解放となり，家族関係の円滑化にもつながってくると考えられてい
る．
b.バリアフリー住宅の定義 バリアフリー住宅の概念は，高
齢者や障害者ができる限り在宅生活を継続し，安心，安全かつ快適
に生活を送れるように，心身機能に合った住宅といえる．第一に家
庭内事故を少しでもなくすような配慮がなされていること，第二に
心身機能の低下・障害・損失があっても日常生活動作を円滑に行え
るような配慮がなされていること，第三に介助・介護が必要なとき
にそれを行いやすいような配慮がなされていることなどである．
c .住宅バリアフリー化への具体策 住宅バリアフリー化は，
身体面・生理面・心理面からのアプローチが必要であるが，本項で
は身体面からのアプローチについて述べる．バリアフリー化するに
は多くの配慮点が必要であり，総務庁（現・総務省）の住宅・土地
統計調査では，手すりの設置，広い廊下，段差の解消をバリアフリ
ー化への重要項目として挙げている．また，介護保険制度における
住宅改修では，段差の解消，手すりを付ける，床の滑りをなくすな
どの床材変更，引き戸等への扉の変更，便器の洋式便器への変更な
どが掲げられている．表11･1にバリアフリー住宅の基本性能とも
いえる住宅品確法による高齢者居住性能・等級3に基づくバリアフ
リー基準を示す．また，例えば車いす使用者に対するバリアフリー
化を検討する基本としては，図11･1～11･5 のような配慮が求
められる．
このほかにも生理面でのバリアフリー対策もある．例えば，高齢
者の生理面に悪影響を与えるのは室内の温度差（特に浴室）であ
る．浴槽（特に和式浴槽）が深いために，心臓にかかる負担が大き
く危険であることも指摘されている．また，視機能の低下した高齢
者に重要なのは，照明方法である．十分な照度をもった均一化照明
と，眼に刺激を与えない優しい照明方法が求められている．
d.公共建築物のバリアフリー 市役所・劇場・ホテルなどの
公共建築物または公共性のある建築物は，高齢者や障害者が利用し
やすいように設計されることが法律や条令によって求められてい
る．ハートビル法によるバリアフリー基準のおもな内容は表11･2
に示すとおりである．その内容は建築物内を円滑に移動できること
を主眼として，玄関・廊下・階段等について定められている．
