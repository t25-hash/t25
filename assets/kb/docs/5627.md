# 5・6・3 ガンマナイフ

> **章**: 第5章 手術・治療機器
> **節**: 5・6 放射線治療

ガンマナイフ(gamma knife)はコバルト60(cobalt-60)を線源としたγ線を用いた定位的放射線治療装置で、正式名はガンマユニット(図5・64)である。定位的放射線治療(stereotacticradiosurgery)とは放射線照射の焦点が目標位置に正確に一致するように、定位的に位置決めと固定を行って照射する方法である。ガンマユニットではあたかも外科で使うメスで病変を切り取るように(a) 側面(b) 前面1コバルト小線源、2固定コリメータ、3コリメータヘルメット図5・65 ガンマナイフの構造放射線照射が行われるため、この機器はガンマナイフまたはガンマメスと呼び慣らわされており、学術用語としてもガンマナイフが用いられている。この機器の原型は1968年にスウェーデンのレクセル(Leksell)により開発された。従来の外部照射との違いは、ガンマナイフでは線量の集中度が極めて高く、一度に高線量を照射して目標部位を破壊することである。ガンマナイフの構造(図5・65)は、201個のコバルト小線源が半球状に配列されたものである。おのおののコバルト線源からの線量は比較的弱く、皮膚・頭蓋骨・脳実質などに吸収されそのエネルギーは半分以下になるが、各線源から発生するγ線は個々のコリメータにより散乱しない細いビームとなって半球の中心に焦点を作るように設計されており、焦点部ではこれらの集合で強い照射線量を得ることができる。コリメータは、本体内の固定部分(図5・65中2)と交換可能なヘルメット部分(図5・65中3)からなり、後者を病変の大きさに合わせて4種類の中から選ぶ。4種類のコリメータによる線量こう配は急峻であり、焦点から少し離れると急激にγ線照射線量が軽減することから、周囲の正常脳組織への照射を最小限に抑えながら、目標の病変にのみ十分な線量を照射し、その結果として境界明りょうな破壊巣を作ることができる。このことがガンマナイフの安全性につながっている。治療計画は、患者の頭部にレクセル式定位脳手術用フレームを4点固定する。そのままの状態でCT,MRI、または血管撮影を行い、これらの機器とガンマナイフ専用コンピュータシステムを直結し、各画像データによる三次元画像再構成も含めて病変の正確な位置と大きさ、照射線量および線量分布を決定し、必要なコリメータヘルメット(図5・65中3、図5・66)を選択する。照射対象となる病変が不整形であれば、異なる大きさのコリメータを用いて線量分布を専用のコンピュータソフトウェアにより描く。境界線量としては線量こう配の最も急峻な50~70%を用いる。50%を用いた場合に最大のコリメータ1回で照射される範囲の直径は25.2mmであり、これより大きな病変では複数のコリメータを組合わせて必要な線量を得ることとなる。しかしながら、縦・横・高さがすべて309 ―9 5図5・66 実際のコリメータヘルメット(a) 治療前MRIT 1強調像(b) 治療前右椎骨動脈造影正面像(c) 同側面像(f) 同側面像(e) 治療後右椎骨動脈造影正面像(d) 治療後MRIT 1強調像図5・67 右後大脳動脈より栄養される動静脈奇形は、ガンマナイフ治療により4年8箇月後の検査では完全に消失したmmを超えるような病変では、境界部の線量こう配が急峻でなくなるため、周辺の正常脳組織への照射量が大きくなり、ガンマナイフとしての本来の目的から外れてしまう。ガンマナイフのおもな適応疾患は脳動静脈奇形、手術困難あるいは高齢者の悪性および良性腫瘍であり、その作用機序は高線量一括照射として腫瘍血管や血管腫の血管内皮細胞に対する高度の障害から血管壊死を引き起こすことである。脳動静脈奇形および腫瘍の栄養血管の内皮細胞は障害により壊死を起こしながら細胞質の膨化が起こり、血管内腔を閉そくする。脳動静脈奇形では血管腫内に凝血が起こり閉そくする(図5・67)。腫瘍では栄養供給が絶たれるため腫瘍の発育は止まり壊死に陥る。脳動静脈奇形の閉そく機転には数箇月から2~3年が必要であるが、ごく小さな転移性腫瘍であれば短期間に治療効果が得られている。最近では、機能性疾患へのガンマナイフ治療が有効であることがわかり注目されている。難治性てんかん、三叉神経痛、がん性疼痛、パーキンソン病などに応用されているが、これらに対する作用機序はまだ十分には解明されておらず、機序の解明と新たな適応疾患について今後の研究が期待されている。
