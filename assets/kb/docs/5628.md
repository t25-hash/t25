# 5・6・4 サイバーナイフ(Cyber knife)

> **章**: 第5章 手術・治療機器
> **節**: 5・6 放射線治療

コバルト60のγ線あるいは直線加速器(リニアック)のX 線、電子線による放射線治療は手術、抗がん剤と並んでがんの重要な治療方法である。がんはリンパ節転移が多いため、通常の放射線治療はがんと所属リンパ節を含んだ広い範囲を対象としている。このため、広い範囲の正常組織の副作用を考慮して、週5回で6~7週間にわたって治療を行う。正常組織の耐容線量を考慮するため、がんに対する線量にも限界がある。近年、非常に限局した部位に大線量を1回で照射する治療が普及してきた。がんなどの病巣だけに限って照射を行うため、照射領域9 ―9 6医療・福祉・バイオ機器に含まれる正常組織が非常に小さく、重篤な副作用を起こさない。手術でも一部の正常組織はがんと一緒に切除されるが、これを放射線で行う方法であることから放射線手術(radiosurgery)と呼ばれるようになった。このような放射線手術のために開発された装置が、ガンマナイフあるいはサイバーナイフである。サイバーナイフは産業用ロボット(マニピュレータ)に小型のリニアックを搭載した装置である(図5・68)。ロボットを使っているため、通常のリニアックのように一点を中心とした回転運動以外にも任意の位置から任意の方向への照射が可能である。もう一つの特徴は、IPS(image processing system)あるいはTLS(target locating system)と呼ばれる位置認識システムをもっていることである(図5・69)。位置認識システムをもっているため、治療中に患者の位置が変化してもその誤差を数量化し、ロボットが自動的に補正を行う。ガンマナイフやリニアックによる定位照射の場合、通常は患者を固定するために金属製の環状のフレームを頭蓋骨にねじで固定する。サイバーナイフではロボットが自動的に患者の動きの補正を行うために、観血的なフレームを必要としな図5・68 サイバーナイフ本体(サイバーナイフII)(a) サイバーナイフ本体と天井据付けのX 線発生装置(b) アモルファスシリコンディテクタい。位置を認識する方法はリアルタイムのX 線透視画像と治療前に撮影したCT 画像から再構成した画像の重ね合わせによって患者位置の移動量を数量化している。サイバーナイフの特徴の一つがアイソセンタのない治療が可能なことである。リニアックはガントリがアイソセンタを中心に円運動を行う。サイバーナイフはロボットにリニアックが装着されているため、任意の位置から任意の方向への照射が可能である。円運動だけの場合、病巣部に対し複数の円弧状の照射を行えば球状の線量分布となる。だ円体の病巣部を照射する場合は、球状の線量分布の治療を複数使用して行う。この場合、重なる部分でも高線量が照射されない部分では小線量となり、このような線量分布の不均一性が問題となる。これに対してサイバーナイフではアイソセンタにとらわれない治療が可能であり、より病巣の形に近い線量分布を得ることが可能である(図5・70)。サイバーナイフは観血的フレームを使用しないため、頭部以外の部位の治療も機械の構造上は可能である。頭部以外の部位の治療で問題になるのが呼吸性移動である。呼吸をモニタし、呼気のときに限って照射を行うgating を使用する場合もある。サイバーナイフでは呼吸に合わせてロボットを移動させて治療する方法が開発され、日本以外の国では臨床に使われている。患者皮膚に発光ダイオードを張り付け、発光ダイオードの位置をモニタする。発光ダイオードの位置と透視画像で確認したがんの位置を数式化する。治療中は発光ダイオードの位置からがんの位置を推定し、さらに照射までのタイムラグによる位置の変化を予測して照射を行う。ロボットを使うことによって従来のガントリに固定されたリニアックと全く違った治療が可能となった。
