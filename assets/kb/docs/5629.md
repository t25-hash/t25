# 5・6・5 X 線治療

> **章**: 第5章 手術・治療機器
> **節**: 5・6 放射線治療

腫瘍のX線治療には従来，リニアック（linac）装置で患者の体
外からX線を照射する方法が用いられてきた．リニアックは線形
加速器で作った電子ビームを金属標的に衝突させて強いX線を発
生させる，大型の治療装置である．ビームの形状を腫瘍の形に合わ
せて可動絞りで絞り，いろいろな方向から照射を行う原体照射の機
能をもつ装置もある．また最近では，さまざまな方向から患者体内
の一点に向けて細いビームを照射して，ピンポイントの治療効果を
ねらう集光照射もしばしば行われる．
一方，近年になって，患者の体内に針を突き刺し，その先端から
X線を照射して治療を行うPRS（photon radiosurgery system）
が開発された．PRSは直径3.2mm，長さ100mmの照射針の先
端からX線を放出する，特殊なX線管である．本体は約2kgの
滅菌可能な装置で，高電圧発生回路，電子銃を内蔵している．照射
針は中空のベリリウム合金製真空容器で，先端に金の薄膜をもち，
ここに最大40Aの電子ビームを衝突させて制動X線を出す（図
5･71）．X線光子の最大エネルギーは40keV，平均は約20keVで
ある．これはカラーブラウン管の蛍光面で発生するX線と同レベ
ルであり，ガラス板で簡単に遮へいできる程度の透過力しかない．
このため，人体組織中に照射針を挿入した場合，X線は照射針先
端近くでほとんど吸収されてしまい，組織の被ばく線量は先端から

a.他の放射線治療との比較 体外から放射線照射を行う外照
射治療では，正常組織も必ず被ばくする．このため，正常組織には
細胞死を引き起こす線量のしきい値をわずかに下回る量を，また標
的組織にはこれをわずかに上回る量を照射する必要がある．腫瘍の
放射線感受性を高める薬剤を併用したり，多方向から照射（多門照
射・回転照射）しても，正常組織がダメージを受けるのは避けられ
ない．
放射性同位元素を含む針を体内に入れて照射をする内照射
（interstitial irradiation，interstitial brachytherapy）治療では，
針が体外にある間の放射線遮へいが必要である．あらかじめ標的部
位にチューブを挿入しておき，遮へいした室内に患者だけを入れて
遠隔操作で針を送り込む特殊な装置が用いられる．また照射線量が
小さいため，時間がかかる．
PRSは内照射の一種であるが，電源を切ればただちにX線が止
まるので，取扱いが容易で，特別な設備がいらない．患者体内でほ
ぼ完全にX線が吸収されてしまうため，医師の被ばくは極めて少
ない．
b.照射位置と照射線量の計画および誘導 あらかじめCTな
どの画像上で位置を測っておいた標的点にPRSを正確に挿入し適
切な線量を照射するために，二つの方法が用意されている．第一は
定位脳手術装置にPRSを取付けて照射針を挿入する方法である．
第二は光学式位置計測を使った手術ナビゲーションシステムである
PRS Navigatorを使う方法で，このシステムは被ばく線量分布の
計算や，実際の照射の記録をとる機能も備えている．
c.転移性脳腫瘍への適用 肺がんなどが脳に転移して生じた
転移性脳腫瘍は，比較的早期ではほぼ球形で，中心部は壊死してい
て球殻状に腫瘍細胞が集中しているのが典型的である．このため，
腫瘍中心に照射針を挿入して照射することで，一度の照射で治療が
行える．
