# 5・6・5 X 線治療

> **章**: 第5章 手術・治療機器
> **節**: 5・6 放射線治療

腫瘍のX 線治療には従来、リニアック(linac)装置で患者の体外からX 線を照射する方法が用いられてきた。リニアックは線形加速器で作った電子ビームを金属標的に衝突させて強いX 線を発生させる、大型の治療装置である。ビームの形状を腫瘍の形に合わせて可動絞りで絞り、いろいろな方向から照射を行う原体照射の機能をもつ装置もある。また最近では、さまざまな方向から患者体内の一点に向けて細いビームを照射して、ピンポイントの治療効果をねらう集光照射もしばしば行われる。一方、近年になって、患者の体内に針を突き刺し、その先端からX 線を照射して治療を行うPRS(photon radiosurgery system)が開発された。PRS は直径3.2mm、長さ100mmの照射針の先端からX 線を放出する、特殊なX 線管である。本体は約2kg の滅菌可能な装置で、高電圧発生回路、電子銃を内蔵している。照射針は中空のベリリウム合金製真空容器で、先端に金の薄膜をもち、ここに最大40 A の電子ビームを衝突させて制動X 線を出す(図ある。これはカラーブラウン管の蛍光面で発生するX 線と同レベルであり、ガラス板で簡単に遮へいできる程度の透過力しかない。このため、人体組織中に照射針を挿入した場合、X 線は照射針先端近くでほとんど吸収されてしまい、組織の被ばく線量は先端から図5・69 TLS図5・70 髄膜腫に対する治療計画[左の横断面で中央の白い部分(造影剤で造影されている部分)が髄膜腫]図5・71 PRS 装置本体の外観と照射針先端の構造図5・72 PRS 照射針先端からの距離によるX 線被ばく線量率の距離の3.2乗に反比例する(図5・72)。この性質を利用して、照射針の周辺だけに限って十分な線量を照射することができる。照射時間は典型的には5~20分程度である。a. 他の放射線治療との比較体外から放射線照射を行う外照射治療では、正常組織も必ず被ばくする。このため、正常組織には細胞死を引き起こす線量のしきい値をわずかに下回る量を、また標的組織にはこれをわずかに上回る量を照射する必要がある。腫瘍の放射線感受性を高める薬剤を併用したり、多方向から照射(多門照射・回転照射)しても、正常組織がダメージを受けるのは避けられない。放射性同位元素を含む針を体内に入れて照射をする内照射(interstitial irradiation,interstitial brachytherapy)治療では、針が体外にある間の放射線遮へいが必要である。あらかじめ標的部位にチューブを挿入しておき、遮へいした室内に患者だけを入れて遠隔操作で針を送り込む特殊な装置が用いられる。また照射線量が小さいため、時間がかかる。PRS は内照射の一種であるが、電源を切ればただちにX 線が止まるので、取扱いが容易で、特別な設備がいらない。患者体内でほぼ完全にX 線が吸収されてしまうため、医師の被ばくは極めて少ない。b. 照射位置と照射線量の計画および誘導あらかじめCT などの画像上で位置を測っておいた標的点にPRS を正確に挿入し適切な線量を照射するために、二つの方法が用意されている。第一は定位脳手術装置にPRS を取付けて照射針を挿入する方法である。第二は光学式位置計測を使った手術ナビゲーションシステムであるPRS Navigatorを使う方法で、このシステムは被ばく線量分布の計算や、実際の照射の記録をとる機能も備えている。c. 転移性脳腫瘍への適用肺がんなどが脳に転移して生じた転移性脳腫瘍は、比較的早期ではほぼ球形で、中心部は壊死していて球殻状に腫瘍細胞が集中しているのが典型的である。このため、腫瘍中心に照射針を挿入して照射することで、一度の照射で治療が行える。9 ―9 7d. 悪性脳腫瘍への適用悪性脳腫瘍の境界は不明確で、辺縁部では正常組織内に腫瘍細胞が散在している。さらに神経線維に沿って腫瘍細胞が移動し、時には発生したのとは逆側の半球にまで及ぶ。したがって腫瘍細胞を全滅させるには、広い範囲にわたって外科的摘出か、放射線照射を行う必要がある。しかし、このような処置は脳の機能を著しく損なう可能性が高く、半身不随、言語障害などの障害を生じてしまう。このため、悪性脳腫瘍の根治治療は極めて困難とされていた。つまり外科手術はおもに、腫瘍の体積を減らして脳への圧迫(腫瘤効果、mass effect)を取り除き、症状を軽減することを目的として行われ、残存腫瘍に化学療法および放射線治療を適用するのが一般的であった。しかし、腫瘍組織の範囲を正確に見極めてていねいに摘出することによって、5年生存率が有意に上がり、完治に至らないまでも再発までの期間をかなり長くできることが知られるようになった。このような準根治的手術を目指す場合、最も取残しが生じやすいのは腫瘍の最深部である。最深部はまた、神経線維に沿って腫瘍細胞がさらに深部に入り込んでいる可能性の高い部分でもある。そこで摘出手術直後に、腫瘍最深部にPRS 照射を行うのが効果的である。しかし、手術によって組織が変形するために、摘出後には正確に照射位置のねらいを定めるのが難しくなる場合がある。そのような状況が予想されるときには、手術開始直後に照射を行ってもよい。すなわち硬膜を切開する前に、硬膜に小さな穴をあけ、術前のCT・MRI画像上で計画した位置に照射針を挿入して照射する。硬膜を開放する前であれば脳の変形はほとんどないので、計画どおり正確な位置で照射を行える。また、悪性脳腫瘍の一種である星状細胞腫にPRS が高い有効性をもつことを示すデータが得られている。e. 脳以外への適用の可能性肝細胞腫は肝臓の中に腫瘍が多発することが多い。肝組織は再生能があるため、術後に生命維持できる体積を残して切除することが可能だが、腫瘍の分布する位置によっては十分な体積を残すことが難しい場合もある。そこでマイクロ波アンテナや熱源を腫瘍に刺入して、腫瘍とその周辺だけを焼く手術が行われている。しかし、マイクロ波は組織の導電性のばらつきの影響を受け、熱は血流による冷却や熱伝導率のばらつきの影響を受ける。それに対して、PRS を用いれば計画どおりにX 線照射が行える。乳がんは組織が非常に柔らかいので、がん組織だけを摘出する手術は非常に難しい。X 線撮影で乳がん組織の石灰化が観察できる場合には、X 線画像を利用してPRS を正確に標的に挿入して照射できる。また、PRS の先端にX 線照射方向を絞る絞りを取付けて、がん細胞が転移しているリンパ節一つ一つに照射を行ったり、皮膚がんに照射するなどの応用も研究されている。
