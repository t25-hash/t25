# 5・7・1 概説

> **章**: 第5章 手術・治療機器
> **節**: 5・7 レーザ治療装置

レーザ治療装置としては、外科手術全般にわたる手術に使用するレーザ手術装置のほかに、特定の治療目的のために特化したレーザ治療装置がある。前者には生体組織の切開作用に優れた特性をもつ遠赤外線レーザのCOレーザ(波長10.6 m)メスや、凝固作用が強い近赤外線レーザのNd:YAG レーザ(波長1.06 m)を用いたレーザ凝固装置がある。後者は、Arレーザ網膜凝固装置やダイ(色素)レーザ尿路結石破砕装置などに代表される特定の治療目的のために最適レーザ波長を選択して装置化したものである。表5・7に現在使用されているおもなレーザ治療装置の特性と用途を示す。
