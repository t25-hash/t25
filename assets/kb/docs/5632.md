# 5・7・3 手術・治療用レーザ手術装置

> **章**: 第5章 手術・治療機器
> **節**: 5・73 のような現象が生じる。レーザは生体組織により吸収され

a. COレーザメスCOレーザ手術装置は切開を中心としたレーザメスである。現在では封じ切りレーザ管が主流となり、保守の手間が大幅に軽減された。外来治療にも使用できる小型装置も出現している。10.6 mの波長は石英ガラスファイバで導光できないため、金鏡反射を利用した関節型マニピュレータが従来使用されていたが、最近では中空導波路やAgCl/AgBr赤外結晶を用いたフレキシブルファイバが実用化したことにより、操作性が飛躍的に向上した。COレーザは生体組織での光侵達長が数十mであり、創面の熱凝固層が電気メスよりも薄く創傷治癒が早い。連続出力も用いるのが一般的であるが、パルスを重畳させたものや繰返しパルスを用いた装置もある。パルスを発生させるCOレーザ装置では、連続波よりも周囲熱損傷が少なく精密な切開が可能になる。b. Nd:YAG レーザ凝固装置Nd:YAG レーザ凝固装置は石英ファイバ伝送が可能なことから、内視鏡下治療を含め多くの手術に使用されている。接触照射法の開発により組織切開も可能になり、凝固・止血能と相まって臨床で使いやすいレーザ手術装置である。最近は50~60W 級装置では冷却器を内蔵した装置が開発され、小型化が進んでいる。c. Ho:YAG,Er :YAG レーザ手術装置Ho:YAG レーザ手術装置(波長2.1 m)は、水中、あるいは組織内で照射するとファイバ先端に水蒸気気泡が生じる。気泡による圧力で組織を裂くように切開する。硬組織に非接触に照射した場合には、表面近傍の水分の瞬間的沸騰により組織内部に大きな圧力が発生して破壊や切開が生じる。したがって、軟・硬両組織で使用できる特徴があり、(a) 光熱的過程図5・75 レーザによる作用9 ―100医療・福祉・バイオ機器整形外科、耳鼻科領域に有用である。Ho:YAG レーザの発振波長はフレキシブル石英ガラスファイバが使用できる最も長い波長であり、カテーテル、内視鏡、穿刺針などを介した低侵襲手術にも適用する。骨・歯などの硬組織破砕用としてはパルスEr:YAG レーザ(波長2.94 m)も開発されている。歯科領域で歯石やう触除去に用いられ、硬組織の中に含まれる水分が水蒸気となるときに組織除去が生じる。d. Ga-As系半導体レーザ手術装置Ga-As系半導体レーザ(波長約800 m)を用いたレーザ手術装置も最近多く出回っている。特徴は小型、軽量、安価で、装置構成が単純で保守管理は必要ない。半導体レーザビームを光ファイバに入射して伝送するには高NA(numerical aperture、開口数)ファイバが必要であるが、高NA ファイバは通常石英ガラスファイバよりも熱損傷しやすい。800nm付近のレーザ波長の生体作用はNd:YAG レーザ(1064nm)とほぼ同様で、非接触照射による凝固作用が中心となる。805nmに強い吸収をもつICG(indocyanin green)を利用して組織の吸収係数を高めて、半導体レーザを切開蒸散に使用したり、選択的血管凝固に使用する治療法も開発されている。
