# 5・7・4 皮膚・形成外科治療用レーザ手術装置

> **章**: 第5章 手術・治療機器
> **節**: 5・73 のような現象が生じる.レーザは生体組織により吸収され

あざの原因となる色素に強く吸収される波長を選択することと,パルス幅を特性熱緩和時間よりも十分に小さくすることで,周囲の健常皮膚に熱的影響を及ぼさない選択的レーザ治療が可能になる.黒あざ(母斑性母斑)ではメラニンに,一方赤あざ(血腫)は微小血管を破壊するためにヘモグロビンに光を吸収させる.メラニンと周囲の健常組織(コラーゲン)との吸収差が大きい波長帯は700nm近傍であり,ルビーレーザ(波長694nm)が使用される.深部あざ治療には光侵達度が長いNd:YAG レーザが用いられる.パルスレーザの熱影響を推定するには,熱が熱伝導により周辺に散逸する時間の目安として,特性熱緩和時間が定義される.特性熱緩和時間τは,τ=δχで示され,δは入熱部分の寸法(光加熱の場合は光侵達長),は形状による補正係数(平板の場合4,円筒の場合は8),χは熱伝導率(ほぼ水の熱伝導率と考えると1.3×10cm/s)である.特性熱緩和時間よりも十分に短いパルス幅,および十分に長いパルス間隔でレーザ照射すると,周囲健常組織への熱影響を最小限にとどめることができる.そのほか神経内科などの領域では,除痛用に半導体レーザ(5~100mV)が使用されている.副作用がないことから,帯状疱疹などの除痛用として日常的に使用されている.
