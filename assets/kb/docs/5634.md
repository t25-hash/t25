# 5・7・5 眼科治療用レーザ手術装置

> **章**: 第5章 手術・治療機器
> **節**: 5・73 のような現象が生じる。レーザは生体組織により吸収され

おもにレーザ網膜凝固装置と、レーザ角膜切除術装置がある。前者はレーザにより網膜をスポット状に熱凝固して網膜はく離を治療するもので、脈絡血管からの眼底出血に対する止血も行える。Arレーザ(波長514.5nm)が一般的だが、網膜深部の凝固にはKrレーザ(波長647nm)も用いられる。後者は角膜表面の形状をレーザ蒸散で変化させ眼の屈折力を矯正する治療装置で、PRK(photo-refractive keratectomy)と呼ばれる。精密蒸散が可能なArFレーザ(波長193nm)が用いられる。パルス幅10ns程度で角膜に照射すると、熱損傷のない精密なレーザ蒸散が生じる。この作用には紫外光エネルギーによって分子結合が切断される光解離過程が含まれているが、組織除去の主たる過程は短時間に生じる熱蒸散である。近視治療が主体であるが、遠視や乱視の矯正も可能である。このほか、同じ装置を用いて角膜の実質表層にできる病変(角膜ジストロフィー)の蒸散治療(PTK : photo-therapeutic ker-atectomy)も可能である
