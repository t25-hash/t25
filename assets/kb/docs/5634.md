# 11・3・6 JIS Z 8071：2003からの発展

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・3 共用品・アクセシブルデザイン

JIS Z8071：2003が制定された2003年6月，この分野に関するた．
提言書を作成するにあたり同委員会は，高齢者・障害者を含むに依頼し，高齢者・障害者への配慮について標準化が必要な分野をしている．
そのうち2004年度に，「住宅設備機器」，「包装・容器―め方及び光の評価方法」，「情報通信における機器，ソフトウェア及2005年度には「情報通信における機器，ソフトウェア及びサービ公共施設・設備」がJISに制定された．
そして，2006年度には，「触知案内図の情報内容及び形状並びにその表示方法」，「公共トイレにおける便房内操作部の形状，色，配置及び器具の配置」の4テ2003年に提言書が出されてから2007年まで，アクセシブルデザである．
は，中国，日本，韓国で2002年から始まっている東亜標準化協力2007年1月から2月にかけ，3箇国共同での提案書を該当するTCよびプロジェクトリーダーを日本が引き受けることになった．
調査し，その中から優先順位の高い40テーマを同提言書の中で示危険の凸警告表示―要求事項」，「視覚表示物―年代別相対輝度の求びサービス―第3部：ウェブコンテンツ」の4テーマが，またス―第4部：電気通信機器」，「視覚表示物―年齢を考慮した基本色「包装・容器―触覚識別表示」，「包装・容器―使用性評価方法」，ーマが新たにJISに制定された．
イン関連のJISは28テーマとなる．
