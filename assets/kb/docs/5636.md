# 5・7・7 消化管・管腔臓器用レーザ内視鏡治療装置

> **章**: 第5章 手術・治療機器
> **節**: 5・73 のような現象が生じる.レーザは生体組織により吸収され

レーザ内視鏡治療は手術が困難な症例に行われたり,EMR(en-doscopic mucosal resection,内視鏡的粘膜切除術)後の再発例に行われることが多い.進行がんに対しては,食道がん,胃噴門がんによる狭窄の解除に用いられる.蒸散凝固治療にはNd:YAG レーザが用いられることが多い.近年では,小型・軽量でNd:YAGレーザと同等の効果をもつ半導体レーザも用いられる.このほか,光線力学的療法(PDT)も行われる.低出力Nd:YAG レーザによるレーザ温熱療法も開発されたが,消化管で用いられることは少ない.
