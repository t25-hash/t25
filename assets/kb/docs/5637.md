# 5・7・8 心臓・血管系レーザ治療装置

> **章**: 第5章 手術・治療機器
> **節**: 5・73 のような現象が生じる。レーザは生体組織により吸収され

XeClエキシマレーザ(excimer laser、波長308nm)と多数の光ファイバを束ねたマルチファイバカテーテルを用いて動脈硬化による血管狭窄を除去するレーザ治療装置が開発されており、冠状動脈治療に多く用いられる。また、バイパス術などが適用できない重症の心筋梗塞治療法として、レーザで心筋壁に直接貫通孔をあけて血流を確保する心筋蒸散術(L-TMR : laser transmyocardial re-vascularization)も注目されており、ECG(electrocardiogram)同期させた出力800W のCOレーザ治療装置などがある。
