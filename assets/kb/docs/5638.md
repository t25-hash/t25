# 5・7・9 PDT用レーザ治療装置

> **章**: 第5章 手術・治療機器
> **節**: 5・73 のような現象が生じる.レーザは生体組織により吸収され

光線力学的治療(PDT)専用のレーザ治療装置で,腫瘍選択性のある光感受性薬剤[フォトフリン(photofrin)など]の励起にXeClエキシマレーザ(波長308nm)で励起した色素レーザ(波長630nm)が用いられている.わが国では,早期肺がんや早期子宮頸がん等の治療にすでに認可されているが,従来の治療有効深度が浅い(3~5mm)といった欠点を打破するために,生体組織中のレーザ吸収が少ない長波長(650~700nm)側のレーザ励起に移行しつつある.
