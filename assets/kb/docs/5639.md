# 5・8・1 音響化学治療システム

> **章**: 第5章 手術・治療機器
> **節**: 5・8 音響化学的原理による治療技術

a. 概要音響化学治療(sonodynamic therapy)は,患部に対する選択性が薬物のみでは不十分なとき,図5・76に示すように,超音波照射により薬物を患部でのみ活性化させて,全身的副作用を抑えながら治療しようというものである.これを広くとらえると超音波照射による薬物吸収の促進なども含めることができるが,ここでは,生体に対する超音波照射による微小気泡(microbubble,マイクロバブル)の圧壊が生体に投与した化学物質との組合せにより引き起こす生体作用を利用する治療と定義する.このような治療は臨床的にはいまだ一般的でないが,研究が盛んに行われるようになってきた.液体あるいは生体に対する超音波照射により,いわゆるキャビテーション(cavitation)が起きると,生じた微小気泡が成長して共振気泡に達したとき,破綻的に圧壊することがある.この瞬間,周辺の液体または生体に大きな応力を生じるだけでなく,気泡内部の気体が断熱的に圧縮され,条件によって数千°Cに達する高温が生じ,化学反応を導く.この音響化学反応と瞬時応力が,音響化学治療の第一義および第二義的メカニズムである.この一連の現象を引き起こすには,気泡半径に反比例する表面張力に打ち勝って微小気泡を生成する過程が最も重要であり,これに必要な超音波強度がキャビテーションしきい値と呼ばれている.このしきい値は進行波音場で特に大きいことが生体中における再現性のよい音響化学反応を困難にしており,音響化学治療の実現にはこれを克服する必要がある.その方法について以下に記述する.b. 第2高調波重畳法(second-harmonic superimposition)超音波照射によりキャビテーションが起きると,照射した超音波の高調波や分調波が発生することが知られている.そこで,照射する超音波にあらかじめ高調波を重畳しておけばキャビテーションの状況を変えうると考えるのは自然である.図5・77は,1MHz の基本波に2MHzの第2高調波を重畳してヨウ化カリウム水溶液を照射した結果を示したものである.横軸上,基本波と第2高調波の合計強度は一定とし,その比を段階的に変化させ,その音圧波形を模式的に付記した.縦軸には,化学的指標としてヨウ素イオンIが音響化学的に酸化されてヨウ素Iが遊離する反応の速度と,物理的指標として発生する分数倍高調波の強度を示した.キャビテーションが本来起きにくい進行波音場で照射したため,1または2MHzを問わず単一周波数においては,この超音波強度で音響化学的に有効なキャビテーションを発生させることはできない.ところが,第2高調波重畳法によれば,キャビテーションの発生・促進に伴って音響化学反応が効率的に進行している.特に両者の強度比が1:1のとき,最大の音響化学反応速度が得られている.この第2高調波重畳法は,下記の化学物質との組合せにより生体組織に対しても有効であることが確認されている.基本波と第2高調波との相乗効果は,第2高調波の基本波に対する位相に顕著に依存し,負側の最大音圧が強調される位相関係の付近において大き図5・76 音響化学治療に有効な化学物質図5・78(a)および(b)に化学構造を示したある種のポルフィリン(porphyrin)は,超音波照射により活性化して抗腫瘍効果を示す.ヘマトポルフィリンやガリウムポルフィリン錯体ATX-70は,それ自身単独では効果を示さない数十molという低濃度において,水溶液に懸濁した腫瘍細胞に対して超音波単独の数倍の殺細胞速度を示し,また実験動物に移植した腫瘍に対しても,超音波単独ではほとんど効果のない強度において増殖抑制効果を現した.これらの効果は,音響化学的に活性化したポルフィリンからエネルギーを受けて生じる活性酸素の強力な酸化作用によるものと考えられている.これらのポルフィリンは水溶液のキャビテーションしきい値をほとんど変化させないが,図5・78(c)の化学構造をもつある種のキサンテン(xanthene)は,これらポルフィリンと同様の音響化学活性に加えて,数十molという低濃度においてキャビテーションしきい値を有為に低減する効果をもつ.その効果は,第2高調波重畳法と組合わせたとき特に顕著で,ローズベンガルまたはその誘導体を投与したとき,生体組織中のキャビテーションしきい値を1けた以上低減させた.ヘマトポルフィリンの誘導体であるHpD やガリウムポルフィリン錯体ATX-70は,上記の音響化学活性に加えて腫瘍選択的に集積する性質をもっている.これを腫瘍に集束した超音波により活性化して音響化学治療を行えば,薬物と超音波の二重の選択性による高い安全性が期待できる.この性質はローズベンガル自身にはないため,その側鎖を修飾することによりこれを与える研究が進行している.キャビテーションしきい値の問題を解決する全く別のアプローチとして,生体に微小気泡そのものを導入する方法がある.これは,音圧波形:cosω-cos2ω(ω:基本角周波数,≧0,≧0,+=一定)図5・77
