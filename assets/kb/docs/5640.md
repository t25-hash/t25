# 5・8・2 ドラッグデリバリーシステム

> **章**: 第5章 手術・治療機器
> **節**: 5・8 音響化学的原理による治療技術

ドラッグデリバリーシステム(DDS : drug delivery system,薬物送達システム)は,生体に薬物を投与する際に,薬物ができるだけ選択的に望ましい濃度-時間パターンで作用発現部位に到達するように,薬物の生体内における動きを各種の技術で制御し,その結果として治療の効果を高め副作用を軽くすることを目標としている.例えば放出制御を目的としたDDS として,図5・79に示す経口投与型のコントロール製剤などが知られている.これは,外見は一般の錠剤とほぼ同じであるが,消化管内では半透膜を通して水分が浸入し浸透圧が高まるため,放出孔より薬物が長時間にわたり一定速度で放出されるシステムである.また,薬物を病巣部へ特異的に運ぶためのDDS 技術の開発およびウイルスや遺伝子,超微細技術など先端的な研究成果を活用したDDS に対して注目が集まっている.ここでは,現在DDS に使われている代表的な技術について記述する.a. 経皮吸収システム. TTS(transdermal therapeutic system)全身作用を期待して皮膚に適用するための外用コントロールドリリース製剤は,経皮治療システムと総称される.最も実用化が進んでいるのは,狭心症の治療を目的として開発された硝酸化合物[ニトログリセリン(nitroglycerine),硝酸イソソルビド(isosorbide di-nitrate)]のTTS である.トランスダーム・ニトロ(Transderm-Nitro좲)と呼ばれるパッチ製剤は,エチレン・酢酸ビニル共重合体を放出制御膜に用いた膜制御型のTTS で,胸部に張り付けて用いる.図5・80に示すように,薬物貯蔵層にはニトログリセリンを乳糖に含ませ粘性シリコーン中に分散させたものが,接着層中にはニトログリセリンの初期放出量が含まれている.薬物は零次速度過程で放出され,適用後24時間にわたり血しょう中のニトログリセリン濃度が維持される.そのほかに,乗物酔い防止を目的に耳介の後方に適用する膜制御タイプのTTS なども実用化されている.このTTS では低濃度の血中濃度が維持されるので注射剤で生じる眠気,めまい等の副作用が起こりにくいため,宇宙飛行士の宇宙酔い防止にも使われている.最近では禁煙を目的としたニコチン(nic-otine)の各種TTS なども開発されている.. イオントフォレシス(iontophoresis)イオントフォレシスは,電気エネルギーを利用しておもにイオン性薬物の生体膜透過を促進させる方法で,薬物の経皮吸収の促進を目的に利用されている.電極を有する二つのリザーバを皮膚に貼付し,負電荷をもつ薬物(アニオン性薬物)なら陰極槽に,正電荷をもつ薬物(カチオ図5・79経口投与型コントロール製剤図5・80 経皮治療システム断面図ン性薬物)なら陽極槽に封入して両リザーバの電極をつなげ,数V または数百A 流れる電圧を負荷する.その結果,薬物が皮膚に移行し,同時に薬物と対をなす内因性イオンが皮膚からリザーバ中に抽出される.また,もう一方のリザーバでもイオン交換が引き起こされ,電気回路が成立する(図5・81).さらに,皮膚に電圧を負荷すると陽極から陰極に向かって水が移動し,これに乗って解離しない薬物も経皮送達される.すでに数種のイオントフォレシス駆動装置が開発されており,アメリカでIontocaine좲というイオントフォレシス用リドカイン局所製剤が市販されている.また,イオントフォレシスを利用して体内のブドウ糖を検出するグルコウォッチ(Glucowatch좲)や,比較的高分子量のペプチド性薬物への応用研究が進んでいる.. ソノフォレシス[sonophoresis,フォノフォレシス(phonophoresis)]皮膚の表面に超音波を照射することにより薬物の透過性を改善し経皮吸収を促進させる方法である.吸収促進作用は効果的で,皮膚への障害も比較的少ないといわれている.b. 経粘膜吸収治療システム. 眼内治療システム角膜は眼内部への薬物の移行に対してバリアとして働く.このため,眼内への薬物吸収は他の粘膜部位に比較して少ない.また涙液による薬物の希釈と鼻涙管への流失も吸収の少ない原因となっている.したがって眼粘膜吸収を高めるためには,薬物を眼内に長く滞留させておく必要がある.コンタクトレンズのように用いるオキュサート좲はまぶたの内側に挿入するDDS として使用されている.図5・82に示すように,塩酸ピロカルピンとアルギン酸からなる透明な薬物貯蔵部を白いリング(辺縁支持部)内に収め,放出を制御するエチレン酢酸ビニル共重合体の高分子膜で挟んだものであり,塩酸ピロカルピンを1週間にわたって持続的に放出する.緑内障の程度により放出速度の異なるもの(20 g/hと40 g/h)を使い分けることができる.. 鼻腔内徐放システム鼻粘膜では膜面は湿潤しており,繊毛運動も存在するため,薬物を滞留させることが重要となる.アレルギー性鼻炎の治療を目的に,ヒドロキシプロピルセルロース(HPC: hydroxypropyl cellulose)の微粉末をプロピオン酸ベクロメタゾンとともにカプセルに充てんし,専用の噴射器により鼻腔内に適用するDDS が実用化されている(リノコート좲).粘膜付着性と高い粘性をもつHPCにより鼻粘膜上での滞留性が向上する.. 口腔粘膜付着システム口腔粘膜面は湿潤しており,局所作用を期待する薬物を滞留させることが重要となる.アフタ性口内炎の治療を目的として開発された口腔粘膜に適用する製剤(アフタッチ좲)には,粘膜への付着性を付与するために,HPCとカル図5・81 イオントフォレシスの電極と薬物の流れ図5・82 眼科用治療システムボキシビニルポリマ(CVP: carboxyvinyl polymer)が用いられている.これらの高分子の水溶液を混合すると,分子間に水素結合を介した相互作用が働き,不溶性の高分子複合体が形成される.HPCとCVPの微粉末を混合し圧縮成形したマトリックス内に水分が侵入すると,同様の原理でマトリックスは表面から徐々に不溶化する.薬物の放出は表面に形成された強固な網目構造によって制御され,薬物の持続的な放出が達成される.. 子宮内投与避妊システム子宮内に装着し,黄体ホルモンのプロゲステロンを持続的に放出するシステムとしてプロゲスタサート(Progestasert좲)が開発されている.これは,シリコーンオイルにプロゲステロンと硫酸バリウムを懸濁した貯蔵層をエチレン酢酸ビニル共重合体で被覆したシステムである.子宮内に装着すると薬物を1年間以上にわたって放出し,避妊効果が持続される.c. 薬物ターゲティング標的指向(ターゲティング)は,生体の特定組織や病変部位等,必要とする部位のみに薬物を効率よく送達させることを目的としたDDS である.ターゲティングは受動的ターゲティング(passive targeting)と能動的ターゲティング(active targeting)に大別される.受動的ターゲティングは,主として剤形に物理化学的修飾を加え,生体の機能を利用して薬物に標的指向性を付与しようとするものである.これに対し,薬物あるいは薬物を封入したキャリヤ(carrier,担体)に標的部位と特異的な親和性を示す抗体や糖鎖などを導入し,より積極的に標的指向化を図ったものは能動的ターゲティングと呼ばれている.ターゲティングのキャリヤには,以下に述べる高分子物質やd項で取り上げるエマルジョン,リポソーム,マイクロカプセルなどの微小粒子が用いられ,それらの体内移行の特性を利用している.ここでは,おもに高分子キャリヤを利用したDDS について記述する.. 高分子化医薬高分子量の物質は,特異的にがん組織や炎症部位に集積しやすいので,高分子化はこの病巣へのターゲティングにとって重要である.また高分子化により血中半減期が延長すること,水溶性薬物を油剤化することができるため腸管からの吸収の改善や薬剤の安定性の増大を図れることなどが知られている.代表的な高分子化医薬としては,日本で開発されたスマンクス(SMANCS좲)がある.ネオカルチノスタチン(NCS : neocar-zinostatin)とスチレン・無水マレイン酸共重合体(SMA : sty-rene maleic anhydride)との結合体高分子が注入局所に長くとどまる性質を利用して,抗がん剤をがん病巣に持続的に作用させる効果をねらったものである.スマンクスを油性造影剤リピオドールに懸濁させた製剤は,肝臓がんの治療薬として商品化されている.. 免疫ミサイル療法がん細胞等の生体内の特定部位を特異的に認識するモノクローナル抗体を結合させ,これを脈管内に注入すれば,薬物を目的とする部位に集中させることができる.また,特定組織の細胞表面にはガラクトースやマンノース等の糖分子を認識するレセプタの存在が明らかにされている.これらの糖類でリポソーム表面を修飾することにより,特定組織への高い選択性が得られる.. タンパク質ハイブリッドバイオテクノロジーの発展に伴い,本来生体内に存在する生理活性タンパク質やペプチドが容易に生産されるようになり,医薬品化が進められている.これらのバイオテクノロジー応用医薬品は従来の医薬品とは大きく異なる特性をもっているが,なかでも実用化上大きな問題となるのは,その短い生体内半減期である.これを解決するため,現在最も盛んに行われているのは,ポリエチレングリコール(PEG : polyethyleneglycol)やデキストラン等生体適合性に優れた高分子キャリヤを用いてタンパク質の分子構造を修飾し,血液中滞留性を改善するアプローチで,この修飾体をタンパク質ハイブリッドと呼ぶ.立体障害によるプロテアーゼに対する安定性の改善や,分子量増加による尿中排泄の抑制等により循環血液中の滞留時間が延長され,抗原性も9 ―103低下する.特に,PEG は種々の酵素やサイトカインの化学修飾に広く応用され,アメリカにおいてはすでにPEG 修飾したアデノシンデアミナーゼ[adenosine deaminase,アダジェン(Adagen좲)]がこの酵素の欠損により起こる先天性免疫不全症の治療薬として,L-アスパラギナーゼ(L-asparaginase)のPEG 修飾体(オンキャスパー,Oncaspar좲)が抗がん剤として認可されている.インターフェロンαに平均分子量約12000のPEG を共有結合させたC型慢性肝炎治療薬のペグインターフェロン(ペグイントロン좲,pegylated interferon α-2b)は,日本で2004年に承認された.インターフェロンα単独では週3回投与に対し,週1回の投与で効果が持続するため,患者の負担の軽減につながる.. 遺伝子医薬品のDDS最近では,遺伝子レベルで疾病を治療する遺伝子治療が注目されており,既存の方法では治療が困難な種々の疾患の画期的な治療法として期待されている.遺伝子治療では,治療効果をもつ生理活性タンパク質の遺伝子をコードしたプラスミドDNA や病因遺伝子のmRNA に相補的な配列をもち,その発現をブロックできるアンチセンスオリゴヌクレオチド等の核酸が用いられるが,これらを遺伝子医薬品という.治療効果を得るためには,遺伝子医薬品を標的細胞の核など遺伝子発現が起こる特定の細胞内部位に送り込むことが必要であるが,DNA は生体内で速やかに分解するため,その安定性を高め,細胞内への効率のよい導入方法を開発することが大きな課題となっている.こうした問題を解決する方法の一つとして,正電荷をもつ高分子キャリヤ(ポリカチオン)と静電的に結合させた複合体の利用が考えられている.複合体形成により安定性が改善され,さらにこの高分子キャリヤに標的細胞がもつレセプタに特異的な認識を受けるリガンドや糖鎖を導入しておけば,標的細胞に効率的に取り込ませることができる.現在,こうした高分子キャリヤを利用した遺伝子医薬品のDDS 開発が精力的に進められている.d. 微粒子薬物キャリヤシステム. リポソーム(liposome)リポソームは,脂質2分子膜からなる閉鎖小胞で,脂質膜部分と水相部分から成り立っているため,脂溶性,水溶性いずれの薬物も包含することができる.形態学的には,多重層リポソーム(MLV : multilamellar vesicle),小さな一枚膜リポソーム(SUV : small unilamellar vesicle),大きな一枚膜リポソーム(LUV : large unilamellar vesicle)の3種類に大別できる.基本的にリポソームを全身投与した場合には,速やかに肝臓や脾臓の細網内皮系組織やマクロファージに貪食されるので,これらへの受動的ターゲティングを容易に行うことができる.一方,この現象は他の組織が標的の場合には大きな障害となるため,これを回避する目的で粒子サイズや脂質組成を最適化したり,PEG などで表面修飾したリポソームの開発が活発に行われている.このタイプのリポソームは長時間血中に滞留するため,腫瘍組織など血管透過性の亢進した組織に薬物を送り込むためのキャリヤとして利用される.欧米では,抗真菌剤アムホテリシンB[amphotericin B,アムビゾーム(AmBisome좲)],抗がん剤ダウノルビシン(daunor-ubicin,DaunoXome좲)やドキソルビシン[doxorubicin,ドキシル(Doxil좲)]などの血中滞留型リポソーム製剤が実用化されており,アムビゾームとドキシルは日本でも承認・販売されている.また,リポソームの表面を抗体で修飾し,標的指向化を図る試みも行われている.一方,正電荷を有する脂質を用いて調製したカチオン性リポソームは,ポリアニオンである遺伝子医薬品を効率よく細胞内に導入するキャリヤとして広く利用されている.. エマルジョン(emulsion)エマルジョンは,液体中にこれと混合しない他の液体の微粒子が乳化剤の働きにより安定に分散したものであり,o/w型あるいはw/o型のエマルジョンがある.また,安定性の改善を目的に,w/o型エマルジョンの内水相をゼラチンでゲル化したs/o型の多相エマルジョン(multiple9 ―104医療・福祉・バイオ機器図5・83 リピッドマイクロスフェアemulsion)が調製される.リポソームと同様に水相・油相からなるので,水溶性・脂溶性両薬物のキャリヤとなりうる.エマルジョンは局所投与後,選択的にリンパ系に移行する特性を有することから,抗がん剤をリンパ系に送り込むためのキャリヤとしても利用されている.. リピッドマイクロスフェア(lipid microsphere)リピッドマイクロスフェアは,油相として大豆油等を用い,レシチンを乳化剤に用いて乳化したo/w型エマルジョンである(図5・83).これは,平均粒子径が約200nmと小さくかつ均一であるので,安全性が高く,脂溶性薬物のキャリヤとして有用である.臨床において静注用脂肪乳剤として高カロリー輸液療法に用いられる.静脈内投与後,炎症部位や損傷を受けた血管,動脈硬化部位に選択的に移行する性質を利用して,抗炎症剤パルミチン酸デキサメタゾン(dexamethasone palmitate)の製剤(リメタゾン좲)が慢性関節リウマチの治療薬として,プロスタグランジンE(prostaglandinE)を含有する製剤(リプル좲,バルクス좲)が慢性動脈閉塞症や重度の動脈硬化症の治療薬として実用化されている.. 高分子マトリックス高分子マトリックスからなる微粒子性キャリヤを,マイクロスフェア(microsphere)あるいはマイクロパーティクル(microparticle)という.粒子径が1 mより小さいものを区別してナノスフェア(nanosphere)あるいはナノパーティクル(nanoparticle)と呼ぶこともある.脂質からなる微粒子キャリヤと同様にいろいろな形でターゲティングにも利用されている.また,薬物を生体内分解性の高分子マトリックスに包含させた製剤は,マトリックスの分解に伴い薬物を徐々に放出するので,コントロールドリリース製剤として広く応用が試みられている.例えば,黄体形成ホルモン放出ホルモン(LH-RH)の誘導体である酢酸リュープロレリンのポリ乳酸グリコール酸を基剤に用いてマイクロカプセル化した製剤(リュープリン좲)は,生体内分解性高分子が徐々に分解することにより筋肉内注射後4~5週間にわたり零次に近い放出パターンを示す.このため,1箇月1回の投与が可能なコントロールドリリース製剤として,前立腺がんや子宮内膜症などの治療に用いられている.
