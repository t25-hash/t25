# 11・2・3 耐熱構造

> **章**: 第11章 気体力学
> **節**: 11・2 大気再突入飛翔体（reentry vehicle）

空力加熱から機体内部を保護するため，機体表面は耐熱構造にす的なのは，アブレーション（ablation）法と熱放射法である．
熱としてエネルギーが放出されることを利用した方法である．
両者ふく射係数が大きく，また，バランスにより定まる表面温度で破壊るため，断熱材を用いた構造となる．
この方法による耐熱構造を利アブレーション法は，吸収した熱によりアブレーション材を気化上昇に伴い，表面からのふく射熱放出も利用される．
アブレーション材の選択は，損耗の度合いを適度にする必要がある．
第Ⅱ部 基礎編 第11章 気体力学る必要がある．
さまざまな方法があるが，実用的なもののうち代表熱放射法は，空力加熱により表面温度が高まると表面からふく射のバランスにより表面温度が定まるが，表面を構成する耐熱材料はされないことが要求される．
その表面温度を内部に伝達しにくくす用している代表的機体は，スペースシャトルである．
させ，その潜熱放出により冷却する方法である．
また，表面の温度ンによる熱の放出には，表面物質の損耗を伴うため，アブレーショ地球低軌道からの再突入など加熱量の少ない場合のアブレーションーション材の選択は異なる．
