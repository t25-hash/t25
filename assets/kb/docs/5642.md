# 6・2・1 遺伝子診断概説

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・2 診断

遺伝子の種々の変異が病気の原因となることが判明して以来,いろいろな検査方法が発達してきている.遺伝子の変異には顕微鏡などで認識できる染色体レベルの変異と,塩基レベルの変異[配列の9 ―108タンデムリピート,一塩基多型(SNPs: single nucleotidepolymorphisms)など]がある.知られている遺伝子変異と病気の関連の一部を表6・1に示した.染色体レベルの変異を顕微鏡で見る方法はカリオタイピングとして広く診断に用いられており,最近では蛍光標識したプローブを用いた画像測定とコンピュータソフトウェアで染色体形状のタイピングを行う方法が発達してきている.近年進歩が著しいのは塩基レベルの変異の検出で,特にSNPsは人の個性や医薬品に対する感受性に関連しており,世界レベルでSNPsデータベースの構築が進み,SNPsと薬理作用の関係などが明らかにされようとしている.また,種々の検査ニーズに合わせたSNPs解析用の試料調製方法や種々の計測方法が発達してきた.実際のDNA 検査では計測にかけるまでの試料調製の自動化が非常に重要で,ゲノム解析(genome analysis)に関連して種々の装置が実用化されている.6・2節では塩基レベルの変異を調べる計測技術を中心に紹介する.塩基レベルの変異の形としては,(1)繰返し現れる同じ配列(タンデムリピート)の繰返し数を調べる,(2)配列の脱落や挿入を調べる,(3)未知のSNPsを調べる,あるいは,(4)既知のSNPsのタイプを調べる,などがある.変異の検出には,目的に応じて種々の方法が使用される.(1)および(2)の変異の形の場合は,対象とする領域のDNA の長さに変化があるのでゲル電気泳動が用いられる.また,変異部位にハイブリダイズするプローブを用いたDNA チップも変異検査に使用される.一方,(3)のまだ知られていないSNPsを調べるには,ゲル電気泳動を用いたDNA 塩基配列決定や網羅的DNA プローブを用いたDNA 配列の再シーケンシングが用いられる.また,変異DNA 断片を簡単に見つけるには,一本鎖DNA が配列の違いによりゲル電気泳動中に異なる構造をとり移動速度に差が出ることを利用して分析するSSCP(single strand conformation polymor-phism)などが有効である.(4)のすでに知られているSNPsのタイピングには,DNAシーケンシングやSSCP,DNAチップなどに加えて種々の方法が提案され,実際に用いられている.一方,ゲノムDNA を分析対象とするのでなく,活動している遺表6・1遺伝病とその出現頻度[文献(4)より抜粋]病名特徴出現頻度ミスマッチ修復遺伝子異常大腸がん1/200が保因乳がん(BRCA 1)乳がんの45%日本人女性の1/1500~2000が保因シャルコ・マリー病進行性四肢筋萎縮1/2500毛細血管拡張性失調症進行性小脳運動失調免疫不全など保因者1%発生1/40000アルツハイマー痴呆性老人(165万人)の32%筋ジストロフィー(DMD)1/3000~4000神経線維腫症1型1/3000~4000肥大型心筋病1/3000のう胞性線維症(寿命29歳)白人1/2000ハンチントン病欧米1/10000日本1/1000000鎌形赤血球悪性貧血アフリカ人1/500短いDNA シーケンシングおよびSnaPshot ABI PRISM 3100/SnaPshot7500typing/d短いDNA シーケンシングおよびSnaPshot ABI PRISM 3700/SnaPshot23000typing/dDNA チップNano Chip(Nanogen)50サンプル伝子の代表であるmRNA(messenger ribonucleic acid)を分析し,診断に活用することも行われている.これはmRNA の種類と量をモニタして医薬品感受性や疾患の進行の程度などを調べ,診断や治療指針に使用するものである.当初は遺伝子発現プロフィル分析にはゲル電気泳動によるmRNA 断片のフィンガプリント分析が用いられていたが,多くのDNA プローブを固体表面に固定し,種々の遺伝子を一度に検査できるDNA チップ(DNA プローブアレイ)の出現によって大いに発展した.現在,種々のタイプのDNA チップが開発されている.遺伝子診断は本来,各診断項目ごとに整理するとわかりやすいと考えるが本書は機器に関する便覧であり,技術および機器ごとに整理し記述する.診断の中心はSNPsに移りつつある.単純に比較はできないが,一つの目安としてSNPs解析の主要な装置のスループットを表6・2に示した.
