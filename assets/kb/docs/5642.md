# 11・2・4 飛行例および今後の動向

> **章**: 第11章 気体力学
> **節**: 11・2 大気再突入飛翔体（reentry vehicle）

アメリカ，ロシア（旧ソビエト社会主義共和国連邦）における飛
行例は枚挙にいとまがないが，地球低軌道では，スペースシャトル
が有人飛行を行っており，X-38として宇宙基地からの帰還のため
のクルービークルの開発，また，次世代のスペースシャトルの開発
も検討されている．ロシアにおいては，かつてのミール宇宙基地か
らの物資・人の輸送用として，ソユーズ等の機体が利用されてい
る．惑星探査においても，アメリカ，ロシアにおいて，種々の機体
の開発が行われている．最近の例では，火星探査ローバを火星表面
に運んだマーズ・パスファインダ計画や，木星大気に突入してさま
ざまなデータを取得したガリレオ計画がある．
わが国においても，近年HOPE計画や惑星探査計画に刺激され
て，飛行実験が開始されている．
a.HOPE計画関連 アメリカのスペースシャトルと同様に，
有翼で地球的軌道から再突入・帰還を実現するためのHOPE（H-
ⅡOrbiting Plane）計画のHOPEを開発するマイルストーンとし
て，いくつかの再突入飛行計画が実施された．OREX（Orbital
Reentry Experiment ，軌道再突入実験）計画（図11･5）は ，弾
道飛行を行う機体であり，1994年2月飛行実験が行われ，各種デ
ータを取得している．ただ，この実験はデータの取得のみを目的と
しているため，機体の回収は行われていない．HYFLEX（Hyper-
sonic Flight Experiment ，極超音速飛行実験）計画（図11･6）
は ，有翼機であり，制御性の検証などを目指して，1996年2月
にサブオービタル飛行を行った．飛行中のデータは取得できたもの
の，回収に不具合を生じている．
b.エクスプレス 自立帰還型無人宇宙実験システム「エクス
プレス（EXPRESS）」衛星は将来の惑星探査への応用を目的とし
て技術の蓄積を目指し，日独で共同開発されたもので，地球周回か
らの再突入を目指して，1995年1月に飛行実験が行われた ．残
念ながら打上げの不具合のため実験は断念されたものの，その後，
偶然再突入したものが地上で回収されている．回収された機体は丹
念に解析され，さまざまなデータを提供している
c.DASHおよびミューゼスC ミューゼスC（MUSES-C）
衛星（図11･7）は，小惑星の表面の一部を地球に持ち帰る（サン
プルリターン）計画であり，その際，深宇宙から直接地球大気に再
突入する機体を搭載している．打上げは2003年に行われ，2010年
に地球帰還の予定である．この機体の大気突入速度は約12km/s
と地球低軌道からの突入速度（7.8km/s ）に対して際立って大き
く，耐熱構造の開発が眼目であった．DASH（Demonstrator of
Atmospheric Reentry System with Hyper Velocity，高速再突入
実験機）計画（図11･8）は ，ミューゼスC計画のような将来の
惑星探査におけるサンプルリターンに必要な再突入機体の技術蓄積
を目指して，2002年2月飛行実験が行われたが，実験機がロケッ
トから分離せず，再突入実験が未完となったものである．
d.今後の動向 再突入の技術は，地球周辺での宇宙活動に必
須のものであるのみならず，将来の宇宙輸送系の開発においては基
本的な技術であり，今後の発展が待望されている．また，惑星探査
においても，金星等さまざまなターゲットがあり，今後の展開が期
待されている．
