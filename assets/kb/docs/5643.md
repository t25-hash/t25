# 6・2・2 ゲル電気泳動を用いた装置

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・2 診断

ゲル電気泳動(gel electrophoresis)を用いた装置の代表格はDNA シーケンサ(DNA sequencer)である。初期のDNA シーケンサは、2枚のガラスプレートに挟まれた0.3mm程度の間げきにアクリルアミドを入れ、重合させて得たポリアクリルアミドゲル(濃度4~6%)シートをDNA の長さ分離板として用いていた。配列決定原理と代表的な装置の概念図を図6・1に示した。測定しようとするDNA 断片のコピー数をPCR などで増幅し、四つのフラクションに分割する。第1のフラクションには図6・1中赤い蛍光体の付いたプライマを用いて、相補鎖合成反応して種々の長さをもち、A で終わる断片群を作る。第2のフラクションには青い蛍光体で標識されたプライマを用いて相補鎖合成を行い、種々の長さをもちCで終わる断片群を作る。第3および第4についてもそれぞれ異なる蛍光標識をもち、G,T で終わる断片群を作製する。作成後これらを混合してゲル電気泳動する。このようにプライマに蛍光標識を高スループット化のキーポイントは高速泳動と多数泳動路の確保図6・1 DNA シーケンサの配列解析スループットは約20kb/dであったが、ヒトゲノム解析では1~2けた大きいスループットをもち、全自動で操作できるシステムが必要となった。ゲル電気泳動を用いた装置でこれを実現するには、泳動路を多くし、高電圧を印加して高速電気泳動することが必要である。泳動時間は印加した電界強度に逆比例し、ゲル濃度の二乗にほぼ比例する。図6・3はゲル濃度とDNA 断片が一定の長さを泳動する時間、およびゲル濃度と1塩基離れたDNA バンドの距離(バンド間隔)の関係である。図6・3(a)に1塩基離れたDNA 鎖が泳動によりどれだけ分離できるかを種々のゲル濃度で示した。4%以上ではあまり分離に差はない。図6・3(b)はゲル濃度と種々の長さのDNA バンド幅はゲル濃度にあまり依存しないが、バンド間隔はゲル濃度4~5%まで濃度とともに増加し、この濃度以上ではゲル濃度に依存しなくなる。すなわち、ゲル濃度が4~5%以上では泳動時間は長くなるが、バンド間隔は変化しないので分離能は上がらないことがわかる。このことから、なるべく低い濃度のゲルが高速ゲル電気泳動には適していることがわかる。しかし、あまり低い濃度のゲルは強度が弱く、ゲルプレートとして作製しにくい。高速泳動しようとして大きな電界を印加すると、ゲルが発熱して破壊されたり、放熱の関係でゲル平面と垂直方向に温度こう配ができてDNA
