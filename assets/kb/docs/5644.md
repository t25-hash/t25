# 11・1・2 セベソ指令を境にして

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・1 安全規格，法制制定の歴史的展望

防護処置：少なくとも許容しうるリスクに到達するために組込む複数のリスク低減策（低減プロセスの一例を図11･1に示す）．
続く機械指令，圧力機器安全指令，機器安全に関するヨーロッパ規格，国際規格の基礎概念となっている．
その端著となったのは1982年，欧州共同体（当時EC）閣僚理事会が発行したセベソ指令である．
この指令はその加盟国に対して安全・環境に関する厳しい立法処置を求めた．
その結果，加盟国政府は化学工業などが作り出す新たな危険性を制御する一連の法律を整備した．
