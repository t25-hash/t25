# 11・1・3 ISO/ IECガイド51

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・1 安全規格，法制制定の歴史的展望

重防護の思想で安全が確保されており，個々の防護層の安全機能をどの程度の信頼性で設計するかが問題になる．
また最近の安全計装フトウェアを含めた安全規格が必要とされる時代に入っている．
そあり，対象は化学プラントに限らず医療機械，原子力発電への適用翻訳されてJIS C-0508として出版されている．
a.適用範囲 本ガイドは安全がかかわる規格を作成する場はPLC等のプログラム可能な電子装置が使われる傾向にあり，ソの要請に応えて作成された国際規格がIEC61508（機能安全) でも意識した非常に広い，基幹的な概念規格となっている．
