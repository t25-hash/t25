# 11・1・3 ISO/ IECガイド51

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・1 安全規格，法制制定の歴史的展望

重防護の思想で安全が確保されており，個々の防護層の安全機能を
a.適用範囲 本ガイドは安全がかかわる規格を作成する場
どの程度の信頼性で設計するかが問題になる．また最近の安全計装
はPLC等のプログラム可能な電子装置が使われる傾向にあり，ソ
フトウェアを含めた安全規格が必要とされる時代に入っている．そ
の要請に応えて作成された国際規格がIEC61508（機能安全) で

も意識した非常に広い，基幹的な概念規格となっている．本規格は
翻訳されてJIS C-0508として出版されている．
