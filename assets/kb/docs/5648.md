# 6・3・2 再生医療

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・3 治療

再生医療は、細胞培養技術を用いて欠損した生体組織を再生し治療することを目的としており、対症療法の範ちゅうから脱して疾患を根治するもの、治療期間を短縮するものとして注目されている。高齢化社会の進行によって厳しい状況にある医療費事情を好転させる可能性も秘めており、期待も大きい。再生医療は、細胞工学の展開とともに具体化した新しい分野であり、医療と工学が極めて接近して共存する分野である。現在のところはまだ臨床検討が主体であり、産業化されている分野は限られている(表6・3)。再生医療の夢は、すべての組織を臓器の機能ともども再生するところまで延長されるが、その実現はかなり先の技術対象になるであろうし、それまでに解決しておかなければならない社会的、倫理的課題も多い。現在の再生医療技術が対象としているのは、その組織の機能がおもに構造機能によるものであり、皮膚、軟骨、骨、腱、角膜、血管、脂肪組織、膀胱、筋肉などが実用化範囲にある。肝臓、膵臓な表6・3 培養皮膚皮膚は、表皮・真皮・皮下組織から構成されている薄い器官で、表皮・真皮には神経、血管は通っていない。健全な皮膚は、皮下組織から栄養を得ることでつねに再生が行われている状態にある。皮膚は、感染、刺激などから身体を保護すると同時に、体液の制御、体温調節、免疫機能、感覚器、美容などの重要な機能を受け持っているが、身体と外界との界面を構成していることから火傷や外傷などの損傷を受ける機会も多い。皮膚の移植は拒絶反応があるために、自己または一卵性双生児の場合でしか成功しておらず、非常に難しいものとされてきた。しかし、永久生着しないと考えられている他者由来の皮膚も、火傷や外傷などの損傷部位に移植すると、移植した細胞が産生する種々の生理活性物質が皮膚の再生を積極的に促進する効果があるため、移植治療に用いられてきた。グリーンらはヒト表皮細胞を体外でシート状に大量培養する方法を考案し、この方法で培養した表皮を用いた広範囲の熱傷患者の救命が報告された。これを契機に培養皮膚の開発が進められ、アメリカでは医療用製品として認可されて、実際に商品として販売されるようになった。培養皮膚の移植は重度の火傷、慢性皮膚潰瘍、床ずれ、大きくえぐられた外傷、色素性母斑の移植などに適用されている。培養皮膚は、自己の皮膚を培養した自家培養皮膚と、他者の皮膚を培養した同種培養皮膚の2種に分類される。自家培養皮膚は拒否反応がなく、感染症からの懸念も少ないことが利点であるが、必要に応じて生産される製品であることから、緊急の需要に応じにくいことが欠点となる。また、大量生産に対応するものでないことは、産業としての展開に対して厳しい制約となっている。一方、同種培養皮膚は、いわば大量に作り置きしておくものであり、緊急の対応も可能であるが、永久生着しないことが不利な点になる。現在製品化されている培養皮膚の代表例(製品名、会社名)を表に挙げる(表6・4)。日本では(株)ジャパン・ティッシュ・エンジニアリング(J-TEC)などが同種培養表皮の製品化を計画している。(株)J-TECの製品では口腔粘膜に由来する表皮を培養しているが、口腔粘膜は皮膚のおよそ倍の速度で分裂を行うので前出のグリーンらの方法で3週間かかるところをおよそ2週間に短縮が可能で、免疫拒絶反応も少ないという特徴が報告されている。b. 培養人工骨。 骨の再生骨は骨格を構成して、文字どおり全身の荷重を担い、運動機能を有効に作用させる働きをしているほか、造血機能、カルシウム貯槽としての働きなどさまざまな機能を受け持っている。外傷や外科治療などの結果、骨の欠損に由来して機能が欠けることが問題となることは多い。しかし、前述の骨の機能のうち後半部分は骨格上で広く分担することができるもので、機能欠損問題の多くは機械的な機能の欠損、強度に関わるものになる。したがって骨の再生では機械的機能の回復が課題とされることが多く、材料科学的な側面から対応する要素が多い。骨の組織は、骨芽細胞が分泌する細胞外マトリックス(骨基質)とそれに沈着した無機質によって形成されている。骨基質はI型コラーゲンを主とし、プロテオグリカンやカルシウム結合タンパク質、糖タンパク質などで構成されている。無機成分は、ヒドロキシアパタイトCa(PO)(OH)を主成分としており、微細な配向結晶構造をとっている。骨は骨芽細胞が分泌する骨基質から形成されていくが、骨の形成と同時に骨には破骨細胞が作用して骨基質の破壊吸収が進行し、両者は平衡が保たれた状態になっている。破骨細胞の形成と活性化には骨芽細胞が直接関わっており、骨代謝機能の再生には十分な血行と骨芽細胞の供給が必要となる。多孔性ヒドロキシアパタイトセラミックスは生体親和性が高く、機械的強度も骨補てん材料として適当なものが得られることから、骨の欠損を補う材料に用いられている。骨芽細胞は、in vivo でも活発に増殖・遊走するので、細胞を播種しなくても骨欠損部に補てんした多孔性ヒドロキシアパタイトが足場となって骨芽細胞や破骨細胞が集まってくるようになる。その結果、ヒドロキシアパタイト補てん部分は徐々に宿主の骨で置換えられて、欠損部には骨が再生されることになる。骨形成と血管新生の間には密接な関係があり、多孔性ヒドロキシアパタイトセラミックスの細孔は血管が入り込むのに十分な空間を有する連続孔であることが必要になる。ヒドロキシアパタイトの細孔密度を大にすると骨の再生は良好となるが、機械的強度は低下する。また、ヒドロキシアパタイトと生分解性高分製品名会社(国)製品化開始年製品EpicalGenzyme
