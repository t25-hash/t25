# 6・3・3 バイオ人工臓器

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・3 治療

a. で培養増殖することにはいまだ成功しておらず、ヒト肝細胞をモジュールに充てんするだけ集めるのは容易ではない。したがって現状では異種動物細胞であるブタ肝細胞をモジュール内に充てんして使用することが多いが、将来は幹細胞、ES 細胞より出発するヒト肝細胞に置換わることが予想される。バイオ人工肝臓の究極の姿には、幹細胞から出発して試験管の中で作製されたクローン肝臓が予想されるが、その完成にはさらに多くの時間と労力を必要とすると考えられる。この一段階手前として、ブタの肝臓を使用することが考えられている。ブタは肝臓など主要な臓器の大きさや形が人間のものに近く、成長も早いので移植には都合がよい。遺伝子技術でうまくヒト遺伝子を組み入れた臓器を作り、異種間拒絶反応を回避することが課題となっている。補体制御遺伝子の導入あるいは主要組織適合抗原系遺伝子を破壊したいわゆるノックアウトブタの作製が考えられる。この場合、ブタの細胞に組込まれたレトロウイルスをはじめとする異種動物経由の感染症に対する懸念も完全に解消しておかなければならない課題である。b. バイオ人工膵臓糖尿病の根本的な治療法をねらった膵島移植が検討されており、膵島細胞を免疫抑制剤を併用しながら肝臓に移植する治療の良好な結果が報告されている。しかしドナーが不足していることや、免疫抑制剤の長期使用による問題は大きい。このため、幹細胞やES 細胞から膵島細胞を誘導することや、拒絶反応の防止のための遺伝子操作が研究されている。また、膵島細胞を工学的に選択的透過膜で包み、ヒトの免疫系から隔離した状態で人工膵臓として機能させる試みが行われている。各種のバイオ人工膵臓の特徴を表にまとめる(表6・7)。表6・7
