# 6・3・4 胎児性幹細胞(ES 細胞)

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・3 治療

細胞を培養増殖して組織を再建する再生医療の確立には,細胞ソースが厳しい隘(あい)路になっている.他者の細胞を移植に用いる場合には,免疫問題に加えヒト組織の採取に関わる倫理問題を避けて通ることができない.ただし,自分自身の細胞を利用することが可能な場合にはこの問題は生じない.このために,再生医療の技術開発は,幹細胞操作技術の開発と強く結びついている.ES 細胞は,体のあらゆる組織や臓器などの細胞に分化する多分化能をもつと考えられる.未分化のまま人工的に培養し無限に増やすことも可能であり,ES 細胞を再生医療の細胞ソースに用いることに期待が集まっている.ヒトES 細胞は1998年,アメリカのウィスコンシン大学が初めて作製に成功した.日本では,京都大学再生医科学研究所が2003年5月,不妊治療後に不要になった受精卵の提供を受けて作製している.
