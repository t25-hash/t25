# 6・3・5 体外受精

> **章**: 第6章 遺伝子工学応用機器
> **節**: 6・3 治療

生殖医療の進歩はこの20年間に大きな進歩を遂げた。特に不妊症に対する治療法の開発には目を見張るものがある。避妊をしないカップルの80%が1年間で妊娠し、10%が残りの1年で妊娠するため、結婚後2年を経て妊娠しないカップルは不妊症と診断される。したがって10組に1組は不妊症のカップルが存在することとなる。多くの不妊症のカップルは非常に強く治療を要望しており、そのようなカップルに対して、「病気と違うのだから、子供などなくても」という論理は全く通用しない。現在、クローン医療などを含めて不妊治療の倫理がマスコミをにぎわす機会が多いが、実地医療においては倫理で片付けられない面があるのも事実である。妊娠が成立するためには排卵、射精、受精、着床の四つの現象が必要である。受精するためには卵子が女性の卵巣から排卵され、同時に膣内に射精された精子は子宮内を移動しながら受精能を獲得し、両者は卵管膨大部といわれる受精環境の整った部位で接して受精現象が起こる。体外に取出した卵子と精子をin vitro で受精させ、受精卵を子宮内に戻して着床させる医療行為を体外受精・胚移植(IVF-ET : in vitro fertilization and embryo transfer)と呼ぶ。世界最初の体外受精児は1978年にイギリスで誕生している。日本でも増加しており、その数は現在年間5万件を超え、すでに産婦人科一般開業医の技術として定着している。体外受精には採卵(卵胞を穿刺して成熟卵を採取する)、採精(通常は男性が自身で採取)、受精(媒精)、胚移植の四つの過程が必要である。自然排卵では通常1個の卵子しか得られないため、成功率を高めるには排卵誘発剤によって多くの卵子を得る必要がある。自然排卵を抑制して多くの卵胞を発育させるために、酢酸ブセレリン(GnRH アゴニスト)と卵胞刺激ホルモン剤を使用する。これによって5~10個、多いときには20個程度の卵子を採取することが可能である。採卵には麻酔(静脈麻酔)のもと経膣超音波プローブを用いて卵胞を観察しながら膣壁から針(18ゲージ)を卵胞に直接穿刺して卵胞液を吸引する。卵胞液には卵子が含まれてお9 ―121り、これを培養皿に移して倒立顕微鏡下で卵子を分離するが、卵子は光を嫌うため暗室のクリーンベンチ内で施行する。この卵子は排卵前の卵子のため、分離後数時間の前培養を行って受精可能な状態とする。精子は用手法で精液を採取し、遠心分離して精子を精液から分離洗浄して調整する。運動性のよい精子を採取するために、遠心後に培養液を静かに加えて上部に移動してくる精子のみを回収して調整する(スイムアップ法)。最終的に良好運動精子を1~5万個/mLに調整して試験管内の卵子に媒精する。精子濃度が高いと多精子受精が起こるが、自然の卵管内では多精子受精は起こらない。受精は顕微鏡下で卵子に雌雄2個の前核が形成されることで確認できる。2日めには受精卵は4細胞期に達する。通常2~8細胞期に子宮内に移植する。多胎妊娠の予防のため、同時に子宮内に移植する胚数は3個に制限している。胚移植する子宮は着床可能な状態にしておく必要があるため、採卵翌日から黄体ホルモンを2週間連日投与する。胚移植には専用の細いチューブを使用し、選択された胚を吸引して子宮内に注入する。最近は培養条件や培養液の組成の向上によってin vitro で胚盤胞まで受精卵を培養することが可能となり、2~8細胞期の移植では20%前後の移植成功率であったのに対し、胚盤胞移植では成功率が50~60%に上昇している。体外受精は卵子の移送ができない卵管閉塞などの患者の治療法として開発されたが、男性側の因子(乏精子症)にも有用であることが判明した。しかし卵子と精子を培養するだけでは受精しないケースが多くあり、直接卵子内に精子を顕微鏡下で注入する方法が開発された(顕微受精)。顕微受精には卵子とそれを囲む透明帯との間に精子を注入する囲卵腔内精子注入法、卵子の細胞質内に直接注入する卵細胞質内精子注入法があり、現在では細胞質内注入法が一般的な方法である。注入する精子は射精精子のみならず、精巣から抽出した未熟精子(12~18 m)や精子になる前の円形精子細胞(直径6~8 m)でも顕微受精が可能である。顕微受精には、卵子を固定し精子細胞をマイクロピペット内に1個吸引して注入する特殊な精密インゼクタ機器(マイクロマニピュレータ)が必要となる。体外受精はヒトへの応用以前に畜産部門でははん用されている方法であり、近年はクローン動物技術の可否をめぐり論争が続くホットな分野である。これから必要となるのは胚培養技術の向上、着床環境の向上、卵細胞と精子細胞の安全な融合方法である。胚培養はいかにして自然受精が起こる卵管内の状態に近似させるかが焦点で、卵管細胞の生物学的特性の研究が行われている。着床の機構はいまだブラックボックスで、研究は緒に就いたばかりである。工学的な技術開発の期待は顕微受精で使用されるマイクロマニピュレータの分野に寄せられており、三次元の粗動、微動が容易に行える装置の必要性が高まっている。また、卵細胞を傷つけずに安全に安定した技術で精子を注入する機器の開発が期待されている。再生医療の産業化が最も先行しているアメリカでは、1997年にFDA が細胞・組織由来製品の臨床上の安全性と有効性の確保に関するガイドラインを制定し、再生医療関連の製品の許認可制度を開始した。さらに2001年には、最新のヒト細胞/組織に由来する製品の査察および実施に関する基準提案規則(CGTP: Current GoodTissue Practice)を制定し、より本格的な体制の整備を規制面でも実施している。各種幹細胞を用いた研究に関しては、2000年にすでに樹立済みのヒトES 細胞を用いた研究に対してNIH が連邦政府予算の使用を解禁した。その後、2002年にはES 細胞研究に1億ドル、さらに幹細胞研究開発費として2億ドルの巨額の連邦予算を投入した。今後、各種幹細胞の産業化に関しても、国家戦略として研究開発のスピードを上げるものと予想される。日本においても、細胞・組織を用いた医薬品等の取扱いならびに製造に関してどのようなルールを作成すべきかの議論が行われ、す9 ―122医療・福祉・バイオ機器年度法規制・ガイドライン-2000 ES 細胞を使用したガイドラインをまとめ、連邦政府助成金を使用したヒトES 細胞の研究を解禁(NIH)でに厚生労働省が関連のガイドラインを制定している。「改正薬事法」の中に細胞・組織を用いた医薬品等の治験制度についての対応も盛り込まれ、欧米同様、再生医療の産業化を法規制面でもサポートする体制が整備されつつある。また、ヒトES 細胞を用いた研究を実施するためのガイドラインも制定され、すでに国産ヒトES 細胞の樹立および輸入ヒトES 細胞を用いた研究開発も開始されている。ES 細胞以外の幹細胞を用いた臨床研究のあり方に関する専門委員会も設置され、2002年度末にガイドラインの策定が行われている(表6・8)。文献(1) 渡辺格・ほか2名編、生物物理化学実験入門、、(1979)、培風館。(2) 吉田和夫・ほか2名編、
