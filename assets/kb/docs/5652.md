# 7・1・1 人工心肺

> **章**: 第7章 人工臓器
> **節**: 7・1 短期運用人工臓器

人工心肺は,心臓を停止して行われる開心手術に際し,その間の全身血液循環を代行する装置である.その主要部分である血液ポンプを中心に,ディスポーザブル製品を用いて,静脈血脱血カニューレチューブ-血液リザーバ-[血液ポンプ]-人工肺-熱交換器-動脈血送血カニューレチューブという生体循環系との一巡回路を形成する.これとともに,心臓内あるいは手術部位の血液を吸引再循環させる吸引・ベント回路が使用されることが多い(図7・1).1953年に人工心肺装置が初めて臨床使用されて以来,血液ポンプには回路チューブをしごいて送血する形式のモータ駆動ローラポンプが用いられてきた.ローラポンプは構造がシンプルで動作の安定性が高く,操作も容易である.ローラポンプは本来非拍動流ポンプであるが,最近では長時間の人工心肺使用時などには,より生理的な駆出様式を得るために,ステップ状回転数制御により拍動を発生する拍動流ポンプの形式で用いられることもある.また,赤血球などの損傷を低減するため,遠心ポンプを送血に用いることもある.これはディスポーザブルのポンプヘッド内のインペラと駆動装置の磁気カップリングにより回転送血する機構であるが,その構造上,前方・後方の圧負荷により同一回転数でも流量が異なるため,特に低回転駆動の際は注意が必要である.人工心肺装置には各種の計測器が組込まれ,使用中のパラメータを表示する.血液ポンプ回転数,回転数より換算される血流量,送血・脱血バランスを検知する血液リザーバ重量,送血温・脱血温・患者直腸温,酸素ガス流量,複数のタイマなどがおもなものであり,単一ディスプレイに統合表示されるものが多い.人工心肺に用いられる各パーツの信頼性は最高のものである必要があるが,各種の操作上の信頼性も保証されなければならない.例えば,人工心肺による体外循環は脱血・送血のバランスで成立している.過剰脱血は生体灌流(かんりゅう)血圧を低下させ,遷延した場合,臓器不全を引き起こす可能性がある.また,脱血量以上の送血を行えば,リザーバ内の空気が送出される危険性もある.これらを未然に防止するため,リザーバ重量の警告表示,赤外線センサによる液面検知システムなどが用いられている.人工肺としては,合成高分子製の人工膜を介して血液と酸素の間でガス交換を行う膜型肺が多用される.最近では中空糸構造の内側あるいは外側に血液と酸素ガスとを灌流する中空糸膜型肺が使用されることが多い.また,心臓手術の際は代謝抑制のため体温を33°C程度まで冷却することが多く,手術手技上の必要によっては血液温20°C以下の超低体温法を併用する場合もある.そのため,血液に損傷を与えない範囲で迅速な冷却・加温が必要であり,人工肺モジュールにコイル状の灌流回路が熱交換器として組込まれている.これは並置する恒温水槽から温冷水を循環させて,血液の冷却・加温を行うものである.人工材料製の血液回路と血液との接触により,血小板の活性化や各種の炎症反応が引き起こされ臓器障害の一因となる可能性が指摘されてきた.近年ではこれを防止するため,ヘパリン,PMEA[poly(2-methoxyethyl acrylate)]などによりコーティングしたディスポーザブル回路製品が数多く登場してきている.
