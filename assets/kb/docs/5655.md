# 7・1・2 項で述べたIABPを上回る補助循環が必要である場合、直

> **章**: 第7章 人工臓器
> **節**: 7・1 短期運用人工臓器

接的流量補助および酸素供給補助を行うPCPS が用いられる。本システムは、遠心式血液ポンプと膜型人工肺とを送血・脱血カニューレにより生体に装着するものである。酸素供給と静脈-動脈バイパスとを併せ行う点では7・1・1項で前述した人工心肺システムと同様であるが、補助循環に特化したコンセプトとして、適用の簡便性、数日オーダの補助が可能な生体適合性、強力な流量および酸素供給補助機能を有している。遠心ポンプは、ポンプハウジング外のモータの回転が磁気カップリングによりピボットベアリングで支持されたインペラに伝達され、血液を駆出するものである。現在使用可能なポンプヘッドの容量は25mLから80mLまでとばらついているが、人工肺と合わせた1セットの血液プライミング量はより小さいほうが有利である。ハウジング材質はポリカーボネートであるが、最近では他のパーツも含め、抗凝固剤であるヘパリンによるコーティングを施したものが各種登場している。血液は異物表面では凝固する性質をもつため、体外循環に際しては一般に大量のヘパリンを要するが、これは同時に出血という合併症を引き起こす。ヘパリンコーティングにより薬剤としてのヘパリン投与量を大きく低減させることが可能であり、数日オーダの補助を行うPCPS にとっては必須の要素となっている。膜型人工肺をPCPS などに長期使用する際の最大の問題点の一つは血しょう漏出である。これは、血液-酸素ガスを隔てる中空糸膜の微細孔に吸着されたタンパク質の親水化作用による。そのため、人工心肺用に開発された膜型肺では最長48時間程度の連続使用が限界とされてきた。この点を改良したPCPS としての使用に耐えうる膜型肺として、緻密層を有するポリオレフィンの使用、中空糸シリコーンコーティングなどによるものが現在、臨床使用可能となっている。本システムは迅速かつ簡便な適用が可能であることが必要である。そのために、血管へのアクセスは切開縫合などを要さない経皮的アプローチが主流となっている。その際に用いられる動脈はそけい部の大腿動脈が多い。また、静脈も大腿静脈が用いられるが、カニューレ自体は右心房まで挿入されて初めて十分量の血流を得ることができる。そのため、血流は比較的長い管路を通過することになり、抵抗低減のため内径が大きく壁厚が薄いものが要求される。一方では血管内留置による末梢側血流不全を防止するため、カニュー9 ―125レ外径はできるだけ小さなものが望ましい。このような相矛盾する要求を満たすために、ワイヤなどにより薄壁を補強したものなども用いられている。システム全体としても、血液充てん量がより少ない、また救急救命の要請から迅速な脱気充てんが可能なものが求められ、臨床使用可能となっている。
