# 11・5・2 知的設備診断技術

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・5 設備診断

知的設備診断技術（intelligent technology for machinery diag-
nosis ）とは計算機を用いた設備診断技術のことで，計算機が人間
の代わりに単独で自動診断を行う場合と，計算機が人間の診断活動
を補助的に助けて診断の精度を高める場合が考えられる．知的診断
の流れは図11･13に示す．
a.信号測定の手法 設備の信号を測定するセンサの種類は設
備の種類および現場状況によって異なる．能動機械設備（特に回転
機械）の診断には音響信号，振動信号およびAE（acoustic emis-
sion）信号が最も多く用いられる ．
音響センサは人間が接近不能な箇所や少数のセンサで多数の設備
状態を広域監視するのに適する．知的点検・診断ロボット ，エ
ンジンの診断 ，パイプラインの診断 およびコンプレッサバル
ブの診断 などの例がある．
振動センサを用いて回転機械を診断する場合，センサの最適な測
定位置および異常信号を検出する最適な周波数帯域についての研究
例がある ．
AEは金属などの材料内部のクラック等によって発生するため，
AEセンサを用いて軸受や歯車等の機械部品の初期異常を鋭敏に感
知することができる ．
b.信号処理の手法 異常の早期発見および異常原因の早期判
定のために，測定した信号からノイズを除去することは設備診断に
とって最も重要な信号処理の内容である．
ノイズの除去法については従来のバンドパスフィルタなどの方法
は設備診断に適用できない場合がある．そこで，異常信号をできる
だけ早期に検出するために，カルマンフィルタによる異常信号の検
出 や，統計フィルタおよび遺伝的アルゴリズム（GA) による
異常信号の自動抽出法 等が提案されている．
c .特徴抽出の手法 異常信号を抽出した後，異常の種類や異
常の程度を判別するために，対象の信号の特徴（設備状態）を示す
有限個のパラメータを計算する必要がある．このように設備状態を
鋭敏に反映するパラメータを診断用の特徴パラメータ（symptom
parameter ）という．
特徴パラメータは，時間領域の特徴パラメータや周波数領域の特
徴パラメータなどがある．どのような特徴パラメータを用いるかは
診断の内容や異常の種類などによって異なる．人工的に試行錯誤で
診断用特徴パラメータを選定する場合，多大な労力が必要なだけで
なく，良好な特徴パラメータが見つかるとも限らない．計算機で自
動的に有意な特徴パラメータを抽出するために，遺伝的プログラミ
ング（GP) による特徴パラメータの自己再組織化が提案されて
いる ．例えば，ある回転機械のシャフト・ミスアライメント
状態を識別するために，図11･14に示すように，従来定義された特
徴パラメータ ～ の異常識別率が低いが，遺伝的プログ
ラミング（GP）による特徴パラメータの自己再組織化で得られた
特徴パラメータの識別率は約100%に達している．
d.自動診断の手法 設備診断は，簡易診断と精密診断に分け
られている．
(１)簡易診断では，診断用の良好な特徴パラメータを選出した
後，設備状態を定量的に評価し，異常の有無を判別する．簡易診断
の場合，判定基準の決定は重要である．統計検定理論や可能性理論
などによる判定基準の決定法が提案されている ．
(２)精密診断では，異常原因（異常種類）や異常部位を特定す
る．現場の複雑な設備条件や外乱などの影響により症状と原因との
関係が不明確になるが，できるだけ明確な診断結論を得るために複
雑な診断アルゴリズム（diagnosis algorithm）を必要とする ．
ニューラルネットワークやファジィ理論などが設備診断に用いられ

ある．この問題を解決するために，図11･15に示すように逐次的に

る ．
さらに，多くの現場設備は稼働中において，運転条件（回転速度
や負荷など）が変動するため，設備診断のために測定した信号は非
定常な信号が多い．可変運転条件における設備の診断では，診断時
に測定した信号から時間-周波数領域の解析手法（ウェーブレット，
ウィグナ分布，短時間FFTなど）により異常の特徴信号を抽出す
ることが重要である ．
