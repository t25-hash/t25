# 11・5・3 化学機械の設備診断

> **章**: 第11章 産業・化学機械の安全
> **節**: 11・5 設備診断

表11･6は化学プラント（chemical plant ）の設備診断技術
（condition diagnosis technology）をまとめたものである．項目と
しては，設備名，診断内容，点検・検査方法，検査機械・器具類に
分類した．設備名としては，搭類（蒸発搭，蒸留搭，乾燥搭，吸収
搭，抽出搭，吸着搭，反応搭など），槽類（貯槽，かくはん槽，ろ
過槽，混合槽，分離槽など），熱交換器類，配管類，回転機械類
（ポンプ，圧縮機，遠心分離機，かくはん機など），電気・計装設
備，安全弁・調節弁，一般構造物に分類 した．
