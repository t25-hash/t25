# 7・2・2 人工弁

> **章**: 第7章 人工臓器
> **節**: 7・2 長期運用人工臓器

心臓には左右の心室の出入口に合計四つの弁があり，構造変化に
よりこれらの弁に機能不全を生じたものが心臓弁膜症である．薬物
治療の限界を超えたものについては，人工弁による弁置換手術の適
応となる．現在臨床使用される人工弁は，材質の点から機械弁，生
物弁の2種類に大別される．クリアすべき問題点は，溶血・血栓形
成などの生体適合性，血流抵抗性・逆流量などの血行動態，弁全体
の大きさを含めた手術時の取扱いの容易さなどである．
a.機械弁 人工材料のみで形成されている人工弁である．近
年ではパイロライトカーボン製のものがほとんどであり，形式とし
ては工夫を施したピボットで2枚の弁葉を支える二葉弁が主流であ
る．従来はケージの中を高分子製のボールが上下するボール弁，1
枚の円板状弁葉の開閉による傾斜円板弁などの形式が用いられた
が，二葉弁は弁口面積，血流抵抗，弁全体の大きさなどの点におい
て優れている．機械弁は耐久性については患者の一生をカバーしう
るが，血流内の人工材料の宿命である血栓形成に問題を残してい
る．過去の人工弁に比べ，現在使用されているパイロライトカーボ

て一生の間，抗凝血薬を服用しなければならない．
