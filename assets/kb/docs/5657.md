# 7・2・2 人工弁

> **章**: 第7章 人工臓器
> **節**: 7・2 長期運用人工臓器

心臓には左右の心室の出入口に合計四つの弁があり,構造変化によりこれらの弁に機能不全を生じたものが心臓弁膜症である.薬物治療の限界を超えたものについては,人工弁による弁置換手術の適応となる.現在臨床使用される人工弁は,材質の点から機械弁,生物弁の2種類に大別される.クリアすべき問題点は,溶血・血栓形成などの生体適合性,血流抵抗性・逆流量などの血行動態,弁全体の大きさを含めた手術時の取扱いの容易さなどである.a. 機械弁人工材料のみで形成されている人工弁である.近年ではパイロライトカーボン製のものがほとんどであり,形式としては工夫を施したピボットで2枚の弁葉を支える二葉弁が主流である.従来はケージの中を高分子製のボールが上下するボール弁,1枚の円板状弁葉の開閉による傾斜円板弁などの形式が用いられたが,二葉弁は弁口面積,血流抵抗,弁全体の大きさなどの点において優れている.機械弁は耐久性については患者の一生をカバーしうるが,血流内の人工材料の宿命である血栓形成に問題を残している.過去の人工弁に比べ,現在使用されているパイロライトカーボン二葉弁は血栓形成性が著しく改善されているが,患者は原則として一生の間,抗凝血薬を服用しなければならない.b. 生物弁ヒト,ブタなどの動物組織を処理して形成される人工弁である.用いられる材料としては,肺動脈弁,大動脈弁,心膜などがあり,グルタールアルデハイド処理したものをステント(外枠)に取付けて成形する.近年では,弁膜と動脈壁の一部を一体としてステントなしで用いるものの臨床使用も広く行われている.生物弁は抗血栓性に関しては非常に優れたものであり,抗凝血薬の服用が不要である場合も多いが,石灰化を含めた耐久性に関しては機械弁に劣る.従来よりステント構造,グルタールアルデハイド濃度,固定圧力などに関して,耐久性向上のためのさまざまな工夫が行われてきた.ステントを完全に取り去るステントレス弁の登場も,優れた血行動態と耐久性とを同時に達成しようとするものである.
