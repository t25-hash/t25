# 7・2・3 ペースメーカ(pacemaker)と除細動器

circuit)化した電子回路と、大容量小型リチウム電池をチタンやステンレスのケースに密封したパルスジェネレータおよび経静脈的に挿入される電極によって構成される。表7・1 NBG
