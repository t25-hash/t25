# 12・1・2 宇宙機の運動のモデル化

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・1 シミュレーション技術

とともにその重要度を増している．
宇宙工学の分野においても数値シミュレーションの役割は大きい．
本節では，特に宇宙機の姿勢制御系の数値シミュレーション解析を行う場合に必要となる基礎事項について述べる．
12･1･1項では，数値シミュレーションの役割・手順について，12･1･2項では特に宇宙機の運動のモデル化と定式化について，12･1･3項では数値シミュレーションの効率化の鍵を握るはん用解析プログラムの役割や実例について述べる．
宇宙機の運動に関する物理モデルや数学モデルの重要性が強く認識されるようになったのは，1950年代からであろう．
初期に打ち上げられたスピン衛星において，軌道上で姿勢が不安定になる不具合が発生し，その原因を解析した結果，現在では常識的に知られている事実であるが，なんらかのエネルギー消散効果を有する衛星の最小慣性主軸回りのスピン運動は力学的に不安定であることが確かめられた．
