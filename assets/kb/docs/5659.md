# 7・2・4 人工肺(oxygenator)

> **章**: 第7章 人工臓器
> **節**: 7・4 )。

人工肺は生体肺のもつ種々の機能のうちガス交換に着目し、その代替を目的に開発された人工臓器である。したがって、ガス交換機能の向上を一義として開発・改良が進められてきた歴史がある。現存する人工肺は、そのガス交換様式によって気泡型肺と膜型肺に大別される。a. 構造。 気泡型肺(bubble oxygenator)人工肺内に導入された血液に酸素ガスを直接吹送し、血液中に生じる無数の微小気泡と血液中の赤血球との間で直接ガス交換を行うように仕掛けた装置である。一般に、酸素加槽、除泡槽、貯血槽の3部分をもつ。. 膜型肺(membrane oxygenator)吹送ガスと血液(赤血球)のインタフェースにガス交換膜が介在する構造をもつ。図7・5除細動放電波形(右が二相性放電の電位変化を表す)図7・6 最新のICD 本体と電極の外観9 ―127ガス交換膜には、その材質から均質膜(シリコーン膜等)、多孔質膜(ポリプロピレン膜等)、複合膜(多孔質膜の微小孔に均質膜材料を充てんしたもの)に分類される(図7・7)。また、膜の構造から平板と中空糸に分けられる。わが国の臨床では、90%以上膜型肺が使用されている。なかでも、内径が300 m前後の中空糸型構造の人工肺が主流であり、血液が中空糸の内部を通過する内部灌流型と、中空糸の外部を通過する外部灌流型が存在する。内部灌流型は除泡が困難で高灌流圧を必要とし、目づまりを起こしやすいため、外部灌流型が主流である。b. 特徴。 気泡型肺少ない血液充てん量、低灌流圧、簡便な除泡操作、高ガス交換能などが大きな利点である一方、気泡が血液に直接接する構造のため血球破壊、血液変性などの問題があり、さらに微小空気塞栓を発症させる可能性も否定できない。これらの理由から、長期の使用には適さない。比較的製造コストが安価なため、短期使用には限定されるが、術中の人工心肺装置用として経済効率の面から見直されつつある。. 膜型肺ガス交換効率、特に二酸化炭素に対する透過性が不良のため、有効膜面積を大きめにとらなければならず、装置が大型化することが欠点であった。また、除泡も気泡型肺より困難であった。多孔質膜の開発でガス交換能は飛躍的に向上し、外部灌流法により灌流圧の低下、易除泡性を獲得した。気相-液相の直接接触がないため、生体適合性が良好であり、長期間使用される呼吸不全治療用のECMO(extracorporeal membrane oxygenator)としても使用されるようになっている(図7・8)。
