a.プログラム作成上の留意点 宇宙機の運動が定式化された
後は，方程式を解くためのアルゴリズムを適用・構築し，これをプ
ログラミングして計算機コードの作成，実行を行う．プログラム
は，CやFORTRANなど非常にはん用性が高い言語を用いて作成
されることもあれば，特定分野の解析機能に特化したアプリケーシ
ョンソフトウェア上の言語あるいは同ソフトウェアが提供する
GUI （graphical user interface）機能を用いて作成されることもあ
る．CやFORTRANで作成するプログラムにもアプリケーション
ソフトウェア上で作成するプログラムにも当てはまることである
が，効率的な数値シミュレーション解析を行うために，以下のこと
に留意すべきである．
.可読性 プログラム上のバグ修正や新たな機能の追加を
効率的に行うためには，プログラムで実現しようとする機能がプロ
グラムから簡単に読み取れるものであることが望ましい．
.信頼性 バグの洗い出しがなされていること，計算結果
ができる限り理論あるいは実験結果から説明できることが望まし
い．実績のあるプログラムを多用する，あるいはプログラム上の共
通要素をモジュール化，ライブラリ化することも信頼性向上につな
がる．
.はん用性，拡張性 共通的，はん用的機能を実現するプ
ログラム要素がモジュール化，ライブラリ化されており，これらを
再組合せすることやパラメータを再入力することではん用的に利用
できること，また，新たなモジュールを追加することで拡張的に利
用できることが望ましい．
.共有性，継承性 宇宙機のシステム設計で利用されるよ
うなプログラムは，システム設計が大規模であることを鑑みると，
複数の技術者で共有できるものであることが望ましい．また，プロ
グラムも一つの技術として継承していけることが望ましい．
b.はん用解析プログラムの利用 アプリケーションソフトウ
ェアとしては，宇宙機の姿勢運動の解析と関連が深いものだけで
も，複数社から販売されているFEMプログラム，多体系の運動
学・動力学解析プログラム，制御系解析プログラム，軌道・視野解
析プログラムなどさまざまなものがあり，必要に応じて利用されて
いる．このようなアプリケーションソフトウェアを利用することの
利点は，特定分野の中で頻繁に用いられる数値計算アルゴリズムが
すでにパッケージ化されており，これらをコード化する手間が大幅
に省けること，グラフィカルな入出力環境が整備されており入力ミ
スの低減，計算結果の視覚化が容易であることである．
また，CやFORTRAN，あるいはアプリケーションソフトウェ
アを出発点としてかかれる宇宙機の特定機能に関するはん用的解析
プログラムの整備が，宇宙関連の研究所や企業で独自に行われてい
る．この場合も，既存の共通プログラムを利用することで信頼性や
作業効率の向上が図られる．
