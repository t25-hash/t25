# 7・3・1 はじめに

> **章**: 第7章 人工臓器
> **節**: 7・3 人工感覚器官

21世紀に入ってから社会の高齢化と情報化が予想以上に進み,それに伴い感覚器官が衰えたり失ったことにより,コミュニケーションや環境認識に支障をきたす情報弱者も急増している.このような人たちが社会から取り残されないようにしていくためには,人工感覚器官などのバリアフリー機器の役割は極めて重要なものになる.感覚補助代行は,必要な情報を信号処理により抽出・変換し,衰えた感覚器官や残された感覚器官を通じて大脳へ送る方法がとられている.最近は,人工現実感など情報表示に関する技術の飛躍的な進歩や,バイオテクノロジーや再生医療技術の登場により人工感覚研究も新しい局面を迎えようとしている.ここでは人工感覚器官の現状を概観し,将来を展望する.
