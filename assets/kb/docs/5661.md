# 12・2・1 環境試験（environmental test ）の歴史

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

ロケットおよび人工衛星等は，打上げからミッション達成までの
間，過酷な環境にさらされる．打上げ上昇時にはロケットエンジン
の排気流や空力的な影響による振動，音響，空力加熱などの環境を
受ける．軌道上においては真空環境とともに，太陽に当たる部分は
ふく射熱を受けて百数十℃になる反面，陰の部分においてはマイナ
ス百数十℃の低温環境となる．これら人工衛星等の環境試験は，ア
メリカにおける宇宙開発の歴史とともに発展してきた．特に
NASA GSFC（Goddard Space Flight Center ）においては，1960
年代から1970年代初頭にかけ，衛星の軌道上の不具合を分析する
ことにより環境試験項目および設計仕様等の妥当性を評価した．こ
れらの活動に基づき，GSFCでは環境試験仕様書（G-320- G-1）を
確立した ．NASA JPL（Jet Propulsion Laboratory）では，深
宇宙探査機の試験および実フライト時の不具合を蓄積し分析するこ
とにより，環境試験に反映させた ．また，ヒューズ社，ボーイ
ング社，TRW社等，民間会社においても，不具合データを分析す
ることにより試験の有効性を評価した．現在に至っても欧米におい
てはフライトデータを取得するとともに，それらに基づく環境試験
を実施することにより試験の信頼性向上に努めている ．
以下では，宇宙航空研究開発機構筑波宇宙センター［旧：宇宙開
発事業団（NASDA）筑波宇宙センター］における環境試験を中心
に紹介する．旧宇宙開発事業団の発足当初，G-320- G-1および
MIL-STD-810 に基づく試験が計画されていたが，試験設備の
規模および試験環境等がアメリカと違うことなどから，レベルや許
容誤差などが必ずしも適合するものでなかった．そのため，1970
年から1980年代にかけ，G-320-G-1をもとに，宇宙開発事業団の
試験設備に適合する試験共通仕様書（人工衛星編：ESPC-7）を制
定するとともに，試験機打上げ時の環境測定を行い，これらの妥当
性を確認した．その後，ロケットの大型化とともに見直され，1994
年，現在の衛星一般試験標準（NASDA-STD-15) となったが，
当初の実現可能な試験スペックという考え方は次第に薄れた．1994
年以降人工衛星やロケットの打上げ失敗が続いたため，データベー
スの活用，試験検証の重要性が唱えられた．これに基づき1998年
より試験データのデータベース化を始めるとともに，2000年から
は試験スペックの裏付けをとるべく打上げ時および地上燃焼時等の
実測データの取得を再開した．この間，人工衛星の打上げ時質量は
82.5kgから4000kg以上に増加するとともに形状も大型化し，初
期の試験設備では対応できなくなった．このため，以下のフェーズ
において増改築を行い運用してきている．
(１)第1期整備（1975年）
N-Ⅰロケットで打ち上げられる静止130kg級の衛星等の試験を
するために音響試験設備，7.9t振動試験設備，8mφスペースチャ
ンバ等を整備した．
(２)第2期整備（1980年)
N-Ⅱロケットで打ち上げられる静止350kg級の衛星などの試験
をするために13.6t振動試験設備，910m音響試験設備等を整備
した．また，衛星の寸法も大きくなったため，天井高の高いクリー
ンルームなどを新設した．
(３)第3期整備（1987年)
H-Ⅰロケットで打ち上げられる静止550kg級の衛星などの試験
をするため，13.6t振動試験設備などの改修を行った．このロケッ
トのフェアリングはN-Ⅱロケットと変わらないため，建物の改修
までは必要としなかった．
(４)第4期整備（1989年)
H-Ⅱロケットで打ち上げられる静止2t級の衛星などを試験す
るために，総合環境試験棟を整備した．この衛星は，質量，寸法と
も大型化し既設の試験設備では試験できなくなったため，試験設
備，建物とも新設した．また，試験を効率化するために試験ごとに
棟間移動することは避け，同じ建物内で試験できるようにした．
(５)第5期整備の検討（1997年)
(４)で整備した総合環境試験棟は，第3期までに整備された試験
設備と併用することにより，静止2t級衛星を年間2機打ち上げる
ために必要な試験を実施できるように整備された．しかしながら，
最近はこれを超える打上げスケジュールが設定され，試験スケジュ
ールが過密化したため，第2総合環境試験棟の整備について検討を
行った．その結果，実現するまでには至らなかったものの8mφス

試験スケジュールを消化するために試験設備等の増設が行われてい
る．わが国において試験設備の整備については，これまでは質量や
寸法などの物理的な問題で必要性を説明できたが，最近はスケジュ
ールという目に見えない問題であるとともに，これらは打上げの数
年前に問題化するものであるため，せっぱ詰まった問題としては考
えられていないのが現状である．
