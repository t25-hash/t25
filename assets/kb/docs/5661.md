# 12・2・1 環境試験（environmental test ）の歴史

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

ロケットおよび人工衛星等は，打上げからミッション達成までのの排気流や空力的な影響による振動，音響，空力加熱などの環境をふく射熱を受けて百数十℃になる反面，陰の部分においてはマイナメリカにおける宇宙開発の歴史とともに発展してきた．
より試験データのデータベース化を始めるとともに，2000年から実測データの取得を再開した．
この間，人工衛星の打上げ時質量は期の試験設備では対応できなくなった．
このため，以下のフェーズするために音響試験設備，7.9t振動試験設備，8mφスペースチャをするために13.6t振動試験設備，910m音響試験設備等を整備ンルームなどを新設した．
H-Ⅰロケットで打ち上げられる静止550kg級の衛星などの試験トのフェアリングはN-Ⅱロケットと変わらないため，建物の改修るために，総合環境試験棟を整備した．
この衛星は，質量，寸法と備，建物とも新設した．
