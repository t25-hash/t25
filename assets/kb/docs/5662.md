# 7・3・2 人工聴覚器官

> **章**: 第7章 人工臓器
> **節**: 7・3 人工感覚器官

人工聴覚というと,人工中耳や人工内耳を指す場合が多いが,鼓膜や耳小骨の機能を圧電型振動子などで置換えた人工中耳と,残存する聴神経を直接電気刺激する人工内耳(cochlear implant)では,治療のアプローチも対象となる障害も異なる.また,広い意味では,脳幹神経を電気刺激する聴性脳幹インプラントおよび大脳の一次聴覚野を電気刺激する方法なども人工聴覚の範ちゅうに入る.ただし,人工内耳に比べると他の方法は適用例が少なく,大脳の一次聴覚野刺激に至っては電極構造,信号処理,生体安全性などがいまだ基礎研究の段階である.ここでは人工聴覚の中でも人工内耳を取り上げ,その原理,効果,および将来について述べる.a. 人工内耳について人工内耳は聴覚受容器である有毛細胞(hair cell)に障害があり,それに続く聴神経系が残存している重度の難聴者の補綴(ほてつ)手段である.聴神経への電気刺激の発想は非常に古くからあったが,1964年にアメリカ・スタンフォード大学のシモンズ(Simmons)が,内耳につながる第8神経に6個の電極を入れて電気刺激を加えると,どこの神経に電極を刺したかということと,その神経の与えるパルスの周波数によって知覚される音の高さが決まることを明らかにしたころから,本格的にこの聴神経への機能的電気刺激の研究が始まった.初期のころは,1本の電極で聴神経にいっせいに電気刺激を加えると刺激周波数に応じてピッチが変わるという周波数ピッチ仮説に基づく単電極法であった.しかし,いろいろな実験からピッチ周波数だけで伝達できる情報は極めて少ないことが明らかになり,多くの研究者は図7・12に示したようなマルチチャネルからなる電極アレイを蝸牛管に挿入し,周波数ピッチだけではなくスペクトルの情報も同時に伝達できる方法をとった.25年ほど前から百花りょうらんともいうべき実にさまざまな方9 ―129式が提案されているが,実用化に先駆けたオーストラリア・メルボルン大学のクラーク(Clark)方式が最も普及している.この22チャネル蝸牛内刺激型人工内耳はオーストラリアのヌクレア社(現在,アメリカ・コクレア社)で製品化され,またたく間に全世界に普及した.わが国では東京医科大学の船坂によって初めて導入され,1988年12月までは治療が受けられるのは2施設(31例)のみであったのが,1998年春には1500例を超え,施設では54例に及んでいる.1994年4月から健康保険が適用されるようになり,それ以後,人工内耳装着者は増え続けている.b. コクレア社製22チャネル人工内耳初期のコクレア社製人工内耳では,電極は0.75mmおきにシリコーンホルダに巻き付けた22個のプラチナからなり,図7・13に示したように電極をリング状にしている.電流分布の重なりを防ぐため,22個の電極のうちのどれか一つの電極だけから電流が発生するようにしている.この電極は内耳の鼓室階内で約24mmまで挿入され,正常な蝸牛の600~8000Hz に対応するところに設置される.信号処理ではピッチ周波数(F0),第1(F1),第2(F2)ホルマント周波数などを抽出し,F0周波数を刺激パルス列に対応させ,ホルマント周波数に対応する電極が選択され通電するという方法をとっている.ただし,F0のない無声音ではランダム状のパルス列が作られる.なお,人工内耳の信号伝送方式では,コクレア社製も含めてほとんどが電磁波結合を採用しており,頭蓋骨の乳頭部に穴をあけ受信回路を埋め込むという方式をとっている.1982年に最初に埋め込まれたヌクレア社製の人工内耳はF0F2型と呼ばれ,ピッチ周波数(F0)と第2ホルマント周波数(F2)のみを抽出し,波形の強度を刺激電流の量に変換していた.1985年に第1ホルマント周波数(F1)も加えた新しい方法であるF0F1F2型が開発され,さらに1989年には小型音声信号処理器を用いてF0F1F2型の高域に3個のバンドパスフィルタを付加したMULTIPEAK と呼ばれる方式に改良された.その後,20個からなるプログラム可能なバンドパスフィルタを用いたSPEAKと呼ばれる方式に代わっている.このフィルタの出力は繰返し走査され,ピークの出力が電流量に変換されて平均6個の電極を介して刺激を発生させる.メルボルン大学病院ではこれらの四つの方式の評価を行っているが,図7・14に示したように,日常会話文章に対する正答率(左),単語に対する正答率(右)ともにSPEAK 方式が最も高い成績となっている.このことから,現在はおもにSPEAK 方式が採用されている.人工内耳装用者の語音聴取時の脳活動をPET(positron emis-sion computed tomography)で調べた研究によると,言語習得後の失聴者の聴覚野を含むPET 脳断面から,聴覚野の血流が増加しており,言語を認知するときにも側頭葉の聴覚連合野が活性化していることが明らかになっている.また,言語脳は,語音刺激のパターンが正常なものと異なっていても,語音であればそれに関連する他の領域を駆使してでも言語認図7・13 メルボルン大学クラークのグループが開発した22チャネル電極アレイの写真図7・12 一般的なマルチチャネル人工内耳の構成図9 ―130医療・福祉・バイオ機器図7・14コクレア社製22チャネル人工内耳の四つの信号処理方法と音声認識率との関係知ができるように柔軟に働くことが次第に明らかにされている.ただし,言語習得前の失聴者が人工内耳をさらに使い続けることで語音認知ができるようになるのか,もっと早い2~3歳のときに人工内耳手術を行った場合にはどうなのかについては現時点ではわかっていない.c. 人工内耳の今後の課題これからは,脳の可塑性と言語習得の臨界期との関連を明確にし,新しい言語概念が構築されるような訓練法を開発する必要がある.また,より優れた人工内耳を設計するうえでは,脳の可塑性に全面的に頼るのではなく,技術的な改良も進める必要があろう.例えば,蝸牛管内は導電性の溶液で満たされているため電流が広がりやすく,電極の数を増やしたからといって情報量はそれほど増加しないことから,電流分布をどのように先鋭化するかということについても研究が必要である.現時点の人工内耳は決して万能ではないことから,補聴器あるいは触覚や視覚を介する聴覚補助代行とどのように併用したらより多くの言語を獲得できるかを追求するための研究も,依然として重要である.
