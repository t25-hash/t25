# 7・3・3 人工視覚器官

> **章**: 第7章 人工臓器
> **節**: 7・3 人工感覚器官

最近、残された視神経を電気刺激する人工網膜や人工視覚が再び注目されてきている。これらの電気刺激法は挑戦的ではあるが、今後定着する方式なのかどうかについては予測がつかない面もあるので、ここではこれらの研究の現状だけを述べる。a. 人工視覚2000年1月に、スイスのドベイユ(Dobelle)が全世界に向けて人工視覚の研究成果について新聞発表を行った。これは1968年にブリンドレイ(Brindley)のグループによって発表された論文に触発された研究にさかのぼるものである。ドベイユらは、1978年に二人のボランティアに電極を埋め込み、彼らについて20年間以上にわたり感染などの問題もなく埋込みが続いていると述べたうえで、電気刺激による知覚特性を示している。二人のうち36歳のときに全盲になり、41歳のときに電極マトリックス(8×8)を右側視覚領野の表面に埋め込んだ62歳の男性(図7・15)については、電気刺激と知覚されるせん光(phosphenes)との関係を永年にわたり調べている。実験結果の詳細は省略するが、視力換算で約20/120の大きさの6パターンを6インチの方向に提示した場合、視力が20/1200でほぼ100%の認識率が得られ、データの再現性も高かったと報告されている。この方式は実用化や倫理面などを考えるといろいろと論議を呼ぶところではあるが、永年の埋込みに基づく結果については事実として受け止めるべきであろう。このような人工視覚の研究はアメリカ・ユタ大学のノーマン(Normann)らのグループによっても進められているが、彼らの電極マトリックスは10×10からなり、その中のおのおのの電極は1.5mmの長さで400 m間隔と高密度で配列されている。b. 人工網膜先天的、後天的にかかわらず視覚障害に陥った場合でも、中枢へ行く視神経が残っていれば、網膜の電気刺激により何らかの情報を伝達できる可能性はある。このような人工内耳と図7・15 ボランティアの盲人[20年ほど前の41歳のときに電極マトリックス(8×8)を右側視覚領野に埋め込んだ62歳の男性で眼鏡にテレビカメラを取付けている]同じような発想で人工網膜と呼ばれる研究も、最近現れてきている。ノースカロライナ州立大学のリュウ(Liu)は網膜に設置できるコンピュータマイクロチップ(バイオチップ)を開発し、テストケースではあるが15人のボランティア患者に対しこのバイオチップの埋込みを行っている。患者の中にはいくつかの画像の動きや形あるいは色まで識別できた者がいることを報告している。一方、マサチューセッツ工科大学のウィヤット(Wyatt)とハーバード医科大学のリッゾ(Rizzo)らによっても網膜プロジェクトが進められており、現在、ウサギへの埋込み実験を通じて人工網膜に適する電極構造や、それによって伝達できる情報量について研究を行っている。わが国でも理化学研究所の八木らと名古屋大学の伊藤・内川らによって、光電素子で受けた情報を培養神経細胞を介して中枢に伝えるという新しい発想の人工網膜が提案されている。これは光覚回復をねらっており、光刺激の欠如によって失われた代謝機能を改善したり、失明後の精神的不安定を軽減しようというのが目的である。人工網膜が成功するかもしれないという期待には、人工内耳が黎明期には多くの批判を受けながらも結局定着し、現代医療方式の一つとして認知されたという背景がある。ただし、人工内耳の場合は、ヒトの言語中枢があまりにも柔軟に情報処理できるため、不完全な言語情報でも学習によって理解できるようになるという側面がある。このような機能が視覚中枢ではどこまで備わっているかが、人工視覚や人工網膜がどこまで活用できるかということを決定付けると思われる。
