# 7・3・4 終わりに

> **章**: 第7章 人工臓器
> **節**: 7・3 人工感覚器官

以上、人工感覚器官として実用化事例を踏まえながら、問題点も述べた。この課題の難しさは、未知の部分の多い大脳の機能と直結していることに起因する。特に言語中枢などは人間特有のものであるため、動物実験もあまり役に立たず、どのように中枢機能を探ればよいのかがよくわからないことによる。しかし、最近のIT(information technology)に加えて脳機能計測法などの進歩により、人工感覚は今後とも改良されるとともに、それを適用することにより大脳生理学についても多くの新知見が得られるであろう。その新知見が再び優れた人工感覚技術を生み出すというループを描きながら、研究が進められるものと考えている。
