# 12・2・2 振動試験（vibration test ）

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

a.振動試験の目的 振動試験設備は，ロケットで打ち上げら
れる衛星等の振動環境を地上で模擬する設備である．振動試験の目
的をおおまかに分類すると，以下のようになる．
(１)設計・試作した構造物が，予想される振動環境に耐えること
ができるかを確認する．
(２)設計・試作した構造物の材料・部品に欠陥がないかを確認す
る．
(３)構造物の応答を確認する．
b.振動試験設備（vibration test facility）の加振方式 振動
設備は，加振方式，設備の構造などにより，以下のように分類でき
る．
(１) 油圧（式）型
サーボバルブを使用して，油圧アクチュエータを動作させる方式
で，構造物の質量が大きく，振幅（ストローク）が長大な設備とし
て建設されている．油圧式は，動電型に比較して大きな変位，大き
な加振力，三次元同時加振が可能なことが特徴である．
加振周波数は，DC（直流)～50Hz前後と低く，振幅は，約
100～300mmで，質量10～1000tの構造物を二次元，または三次
元で同時加振することが可能である．また，大型の加振設備の加振
時間は，油圧アキュムレータの容量により決定される場合があり，
長くても1分程度の加振しか実施できない．
(２) 動電（式）型
筑波宇宙センターに設置されている設備は，動電型振動発生器の
代表的なものである．動電式は油圧式に比べ，高い周波数（5～
2000Hz）まで加振可能であり，加振波形精度が良好であるため，
衛星等などの振動試験設備に採用されている．また，加振時間も特
に制約がなく，長時間の加振が可能である．
.シングル加振（SESA:single exciter,single axis )
加振する構造物に対して一方向に加振を行う仕組みで，おもに1台
の動電型振動発生器を使用した振動試験設備として国内外に多く設
置されている．
水平および垂直振動試験設備は振動台の上に構造物を据え付け，
振動台横（水平振動試験設備の場合）および振動台の下（垂直振動
試験設備の場合）に配置された振動発生器により加振する ．こ
の種の試験設備では振動台に高い剛性が求められるとともに，垂直
振動試験設備の場合には振動台および構造物の質量を振動発生器で
支持する必要があるため，軽量な構造が必要となる．そのため，国
内外のシングル加振設備は小型のものが主流である．
.多点一軸加振（MESA:multiple exciter,single axis )
(１) 水平振動試験設備
水平振動台の上に構造物を据え付け，振動台横に配置された複数
の動電型振動発生器，もしくは油圧アクチュエータにより加振す
る．シンプルな構造であるが，複数台の振動発生器を使用するた
め，位相制御が使用されている．振動台の大きさ，振動発生器の配
置により，国内外の設備でそれぞれ特色がある．図12･8に多点水
平振動試験設備の構成例を示す．
(２)垂直振動試験設備
垂直振動台の上に構造物を据え付け，振動台下に配置された複数
の動電型振動発生器，もしくは油圧アクチュエータにより加振す
る．振動台，構造物の質量を動電型振動発生器，もしくは油圧アク
チュエータで支持する必要があるため，特殊な構造となる．そのた
め，国内外の多軸加振設備はバラエティに富んでいる．図12･9に
多点垂直振動試験設備の構成例を示す．

振動台に対して，複数の振動発生器（油圧式ないし動電式）が接続
され構造物を加振する仕組みで，質量が大きく，転倒モーメントな
どが大きい構造物を加振するための方式として採用される．
