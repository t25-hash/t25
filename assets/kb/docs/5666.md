# 7・4・2 眼内レンズ

> **章**: 第7章 人工臓器
> **節**: 7・4 機能材料

ヒトの眼には平均20D の屈折力を有する水晶体が存在する.これは眼球をカメラにたとえるとレンズに当たる部分である.この部分が加齢現象,糖尿病,外傷,先天性,その他の原因により混濁してくる状態が白内障である.この白内障を除去すると,平均20Dの屈折力の欠落が生じる.これを補う方法としては眼鏡,コンタクトレンズなどがあるが,眼鏡でこれを矯正した場合,術後視力の質は輪状暗転やいわゆるびっくり箱現象などに代表される極めて劣悪なものである.コンタクトレンズは術後視力の質をかなり改善したが,高齢者にこれを強いることは実際上多くの困難がある.約50年ほど前,イギリスのハロルドリドレーがこの問題を解決すべくPMMA でできた眼内レンズを手術後の無水晶体眼に挿入したのが今日の眼内レンズの始まりである.当初は重篤な合併症など多くの問題があったが,白内障手術法等の進歩により,1980年ごろから眼内レンズはほぼ確立した方法として眼科臨床で使用されるようになった.そして壮老年期に多発する白内障術後の視力の質(qual-ity of vision),QOLを画期的に改善することとなった.今日では安定した術後結果が得られるようになり,2000年には約80万眼の利用者数を数えるようになった.以下に最近の手術方法を述べる.今日では,3mm以下の切開創で水晶体を超音波により破砕乳化吸引する超音波乳化吸引法が,大多数の症例で適応されている.麻酔は0.5%のリドカイン点眼のみで,手術操作はすべて顕微鏡下で行われる.切開は角膜,強角膜切開,あるいは透明角膜切開で行われ,切開後ヒアルロン酸製剤を前房内に満たした後,子(せっし)により水晶体前のう(囊)に直径約5mmの前のう切開口を作成する.続いて灌流吸引機構を有するチタン製チップから発信される超音波によって水晶体核を乳化し,吸引・除去して残留皮質の吸引・除去,後のう研磨などを行う.本術式では水晶体のうは全体性を保った状態で眼内に残される.この水晶体のう内にPMMA,シリコーン,ハイドロジェル,アクリル等の材料で作られた眼内レンズを挿入する.近年ではアクリル,シリコーン,ハイドロジェル製の折畳み式の眼内レンズが多用されている.このようにすると,水晶体乳化吸引のために作成した3mmの切開創を大幅に拡大することなく挿入できる.このため,患者の早期リハビリテーションが可能になり,多くの症例で日帰り手術も行われる状況になった.上述の術式により極めて安定した術後成績が得られるようになったものの,なお二,三の問題点が残っている.一つは超音波乳化吸引法を習得するために一定の学習曲線(learning curve)を必要とすることである.レーザなどの新しいテクノロジーにより,より単純で容易な乳化吸引法の術式の開発が待たれる.これに関連して,なお3mm程度の切開創を必要とするために術後の感染症が1000眼ないし2000眼に1例程度発生することも問題点である.このことから,より小さな切開創とする技術の開発が待たれる.現在の術式のもう一つの問題点は,上述のように水晶体のうを残すためにこののうの中に残留した水晶体上皮細胞が術後増殖偽化性を起こし,いわゆる後発白内障を高率に起こすことである.現在,Nd:YAGレーザによってこれを切開する術式が採用され,臨床的には大きな問題とはならないが,今後の術式の進歩のためにはこの後発白内障を最初の手術中に予防,あるいは術後治療する方法の登場が待たれる.眼内レンズの素材としては,PMMA,アクリル,シリコーン,ハイドロジェルなどが使用されるが,これらすべての材料は硬度にはさまざまな差があるものの,術後の調節機能はほとんど期待できない.高齢者においても残っている調節機能が完全になくなり,手術後に強い老視を残すことになる.これを改善すべく,多焦点眼内レンズなども開発されているが,なお術後視機能の質に若干の問題を残している.このため,さらなる軟性の生体材料を使用することによって術後に調節を再現することが最終的な目的となる.これを実現するためには適当な生体材料の開発以外に,上述のようにより小さな切開創からの白内障の摘出手技を開発すること,後発白内障の予防法を確立することなどさまざまな問題が残されている.
