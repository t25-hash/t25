# 12・2・4 衝撃試験（shock test ）

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

a.衝撃試験の目的 人工衛星は，火工品を用いてロケットか
ら分離される．この際に高レベル，高周波の衝撃が人工衛星とロケ
ットの分離面から人工衛星に負荷される．また，太陽電池パドルや
アンテナ類を展開する場合にも火工品が用いられ，同様に展開衝撃
による負荷が人工衛星にかかる．このような高レベル，高周波（場
合によれば300万m/ s以上，1MHz以上) の衝撃負荷に対し
て，人工衛星の各機器が耐えうるかを確認するのが衝撃試験であ
る．
b.衝撃試験設備（shock test facility) 上記のような衝撃環
境を地上で模擬する設備が衝撃試験設備である．筑波宇宙センター
所有の衝撃試験設備のシステム構成図を図12･18に示す．また，表

衝撃試験設備は，おもに爆管制御部・計測部・データ収録部等か
ら構成されている．爆管制御部は，衛星を分離させる火工品に点火
電流を流すとともにデータ収録部および高速度カメラの制御を行う
ことができる．衛星に取付けられた加速度センサからの信号は，デ
ータ収録部に記録され，フーリエ変換・衝撃応答スペクトル解析
（SRS:shock response spectrum）の処理を行うことができる．
また，分離の様子は，高速度カメラによって撮影することができ
る．
c .衝撃試験データ計測の問題点 衝撃試験データ計測時，計
測データに実際には発生していない偽信号（pseudo signal ）が混
入する現象（ゼロシフト，zero shift ）が確認されている．これは
おもにセンサ内部の圧電素子の共振による ．通常衝撃応答波形
は，図12･19(ａ)に示すように衝撃が発生した後，ゼロレベルを中
心に正負同程度の振幅をもったまま減衰していく形となる．これに
対し，ゼロシフトが発生すると，図12･19(ｂ)に示すように衝撃発
生直後に正方向または負方向のどちらか一方に急激なシフト成分が
発生し，時間の経過とともにゼロレベルに戻っていく．この結果，
図12･19(ｃ)(ｄ)に示すように衝撃応答加速度の評価方法である
SRS解析結果において，特に低周波成分で大きな相違を示し，試
験データとしては不適なデータとなる．実際の衛星開発では，コス
ト，累積疲労の面から再試験を行うことが非常に困難である．ま
た，再試験できたとしても有効な対策をとらないと同じようなデー
タが再びとれるだけである．よって，以下に示すようなゼロシフト
を含むデータから偽信号を取り除き，評価に耐えうる試験データと
することが望まれる．
d.ウェーブレット（wavelet ）を用いた衝撃データ補正方
法 偽信号を含む試験データから真の計測信号（true sig-
nal ）を得るためには，ゼロシフトした計測データから偽信号成分
を除去することが必要である．フーリエ変換ベースまたはハイパス
フィルタ（HPF:high pass filter ）を使った信号処理では，真の計
測信号の低周波成分も除去してしまうため，適用ができない．その
ため，時間軸上で処理が可能なウェーブレットを用いる．補正結果
の例を図12･20に示す．
これより，ウェーブレットを用いたデータ処理方法は本問題を解
決するのに非常に有効であることがわかる．
