a.熱環境試験の目的 熱環境試験は，宇宙空間を模擬した熱
真空環境（冷暗黒，高温，高真空環境）下で人工衛星等の耐環境性
の確認や熱設計の妥当性の確認を行うことを目的としている．熱環
境試験は広義な意味で熱真空試験（thermal vacuum test ）と呼ば
れ，狭義な意味では熱真空試験と熱平衡試験（thermal balance
test ）に分類される．
狭義な意味の熱真空試験では，主として人工衛星等の打上げから
軌道上での運用終了までに遭遇する熱真空環境と同等か，より厳し
い条件で人工衛星等の特性および耐環境性の確認が行われる．
熱平衡試験は，主として熱設計の評価，妥当性の確認を行うこと
を目的としている．この試験で人工衛星等は軌道上と同等の熱入力
条件が与えられ，測定された温度応答と熱設計解析での予測値との
比較で，人工衛星等の熱制御装置の熱設計や熱数学モデルの妥当性
が評価，確認される．
熱環境試験で人工衛星等の機能，性能を確認するために用いられ
る代表的なプロファイルを図12･21に示す．
