# 12・2・7 静荷重試験設備（static load test facility)

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

静荷重試験設備は，人工衛星等の構造体にロケット打上げ時等に
加わる準静的加速度を静的荷重として負荷し，その強度および剛性
を測定し確認する設備である．
本設備は，荷重負荷部，計測部，データ処理部から構成されてお
り，荷重負荷部は供試体の構体に，油圧式アクチュエータによる引
張りあるいは圧縮荷重を負荷する．計測部，データ処理部は張り付
けたひずみゲージおよび変位計で計測し，その計測信号をデータ処
理部に伝送し，オンラインで主応力等の計算を行う．この結果を設
計解析結果と比較することにより，構造設計の妥当性を確認し，マ
ージンを評価する．また，制限荷重による有害な残留ひずみのない
こと，終局荷重まで破裂，破損しないことを確認する．静荷重試験
設備の外観を図12･24，システム構成を図12･25に示す．
中に供試体または試験装置に不具合が発生した場合に備え，油圧式
アクチュエータのストロークオーバおよび荷重急増等に対する保護
機能が設けられている．また，静荷重試験は油圧式アクチュエータ
を何本か組合わせて行うため，人工衛星のような複雑な形状に安定
した荷重を負荷する計測制御システムも必要である．設備の概要を
