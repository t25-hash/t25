# 7・5・3 バイオ人工肝臓の開発の現状

> **章**: 第7章 人工臓器
> **節**: 7・5 ハイブリッド型人工臓器

すでに述べたように、1950年代後期に開発研究が始まったバイオ人工肝臓の開発が永らく停滞したのは、肝切片、肝細胞などの肝組織の機能を安定して保持できる生化学的方法あるいは培養方法の研究にあまり大きな進展が見られなかったことによる。1972年にセグレン(Seglen)が肝細胞の分離・培養法を確立してから、ハイブリッド型人工肝臓の開発も加速され始めた。現在までに試みられてきた培養方法あるいは装置形式としては、表7・6に示すように、浮遊培養、マイクロカプセルなどへの包埋培養法、ホローファイバ(hollow-fiber、中空糸)形式や多孔質樹脂を担体とする培養法など多くの形式があり、その一部については欧米では臨床応用が試みられ始めているが、わが国ではこの方式のバイオ人工肝臓はいまだ臨床応用されていない。a. 単層培養による積層型モジュールこのモジュールは、肝細胞を単層培養した多数のプレートを積層したものであり、工学的に問題となる技術要素は多くない。しかし、このモジュールで得られた結果は、バイオ人工肝臓の有効性を実証し、他の多くの形式のモジュールの開発を促した。ただし、このモジュールの構造はかなり複雑で操作性も悪いために、はん用的な人工肝臓としての用途には適していないと思われる。c. 肝細胞の包埋培養やマイクロキャリヤへの固定肝細胞をアルギン酸カルシウムなどのハイドロゲルに固定して培養する方式である。これらのゲルで肝細胞の包埋培養実験を行った結果、肝細胞の代謝機能は長時間維持されることが多くのグループから報告されている。一方、いったんスフェロイド(球状凝集塊)を形成させた後に包埋することで、肝細胞の機能をより長期間維持できることも明らかになった。そこで、肝細胞を包埋したハイドロゲルを充てん層、流動層、あるいは回転円板型のモジュールに組込むことによって、バイオ人工肝臓として利用する試みが多くなされた。しかしながら、このハイドロゲルによる包埋法は、ゲルの機械的強度が弱く、調製が煩雑で、しかも肝細胞がゲル内に包埋されているために酸素や栄養物質の物質移動が効率的でないなどの欠点をもつため、バイオ人工肝臓の開発に関しては、近年ではあまり用いられていない。包埋培養法の欠点を解決する方法の一つに、マイクロキャリヤの使用が挙げられる。すなわち、100~200 m程度の直径をもつマイクロキャリヤに肝細胞を固定化して培養する方法である。この方法の利点の一つとして、細胞が付着したキャリヤは、浮遊状態あるいはホローファイバの外筒側に注入された状態でも培養可能となり、培養形式の自由度が高いことがある。肝細胞培養にこの方法を適用することで、付着性である肝細胞は高い生存率を保ち、代謝能も維持されることが報告されている。また、これらを用いた充てん層型のバイオ人工肝臓によるラットの体外循環実験や、移植による肝機能補助の臨床応用も行われている。この方法の要点は、肝細胞をいかに効率的にマイクロキャリヤに固定化するかにあり、固定化方法についてはさらなる改良の余地があると思われる。d. ホローファイバ型モジュールへの固定化ホローファイバ(中空糸)型モジュールは、人工腎臓では広く用いられているモジュールの形式であり、単位体積当たりの膜表面積が広いことから、肝細胞を含む動物細胞培養用の基材としての応用が盛んになった。ホローファイバ内で肝細胞を培養する人工肝臓は、かなり高い肝細胞密度が達成でき、取扱いが比較的容易であることから、現在では最も頻繁に用いられている方法であるが、欠点として、肝細胞を図7・25 ホローファイバ型モジュールによるバイオ人工肝臓付着させるべきホローファイバ外側の面積には限界があることが挙げられる。しかも人工肝臓として使用される場合に、血しょうまたは血液が流れるホローファイバ内部と、肝細胞が培養されるホローファイバ外側との物質移動は膜を介する拡散のみによって行われるために効率が十分ではなく、肝細胞を大量に培養する場合には、肝細胞への栄養や酸素の供給が不十分になるという問題が生じる。そのため、肝細胞を浮遊状態やホローファイバ外壁に付着させた状態で培養させるだけでなく、マイクロキャリヤに固定化して外筒とホローファイバの間のスペースに充てんする、もしくはホローファイバ内でゲルに包埋させるなど、さまざまな方法と組合わせて培養を行っている報告がある。この方式によるハイブリッド型人工肝臓の一例を図7・25に示す。ここではバイオリアクタとして、2基のホローファイバ型モジュールを並列に用いて、ホローファイバの外腔にマイクロキャリヤに付着させた肝細胞を培養している。この形式のバイオ人工肝臓は、1993年ころからアメリカ、ドイツなどですでに100例を超える臨床応用が試みられている。e. 多孔質樹脂を担体とする肝細胞培養法多孔質の金属、セラミックスや樹脂を担体とする固定化法は、かびやマイクロキャリヤを利用するものに類似しているが、担体の選択範囲が広く、しかも安価であるため、大量培養を行うには有利な方法である。多孔質のポリビニルホルマール(PVF: polyvinyl formal)樹脂を肝細胞培養用の担体として用い、これらを充てんした充てん層型(packed-bed型)のバイオ人工肝臓の開発が10年近く行われてきた。この担体は約90%の高い空げき率をもち、その細孔は三次元的に連なっている。その長所としては、第一にPVF樹脂は細胞を固定化するうえで幅広い孔径の範囲があり(平均孔径8~1000 m)、比較的強度もあり、しかも安価であるなどが挙げられる。図7・26は、大島らが開発中の充てん層型バイオ人工肝臓シ図7・26
