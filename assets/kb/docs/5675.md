# 8・1・3 電気刺激による治療訓練

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・1 概説

古くは低周波治療機に始まるが,それに類する機器として時値測定装置(クロナキシメータ),電気変性反応測定装置等がリハビリテーションの概念導入以前から使われてきた.FES(functionalelectro-stimulation)を含め末梢神経・筋の電気的刺激により筋収縮を随意的に起こさせることで,促通効果に通じる被刺激性を高めたり,筋萎縮の防止,筋訓練効果,さらには痙(けい)性抑制効果を期待できることが古くから知られていたが,コンピュータ組込式(computer-aided)の機器出現で進歩した.
