# 8・1・4 姿勢・バランス・筋力トレーニング

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・1 概説

自動的(active)な運動による筋力トレーニング効果についてはいうまでもなく訓練効果を期待できるが,意欲のない者,歩行のかなわぬ者には無理難題になる.しかし,他動的(passive)な運動ならば無理なくできる.スキーのゲレンデを上から滑り降りる際にスキーヤーは位置のエネルギーを得ているが,その実,ゲレンデの下に到達した折には疲労と発汗を覚えている.このことは自動的な筋活動のあったことを示している.ヨーロッパで行われている乗馬療法がこれに類似する.日本では特別な地域と特別な施設でしか乗馬訓練ができないために開発されたのが,馬のくらの三次元運動をパラレルメカニズムによって再現した乗馬シミュレータである.健康運動機器として家庭用のものも市場に出回っており,姿勢,バランス,下肢筋力訓練機器として使われている.特殊な矯正用機器として挙げられる開閉口訓練装置(一種の関節拘縮除去訓練装置)はこれまでも整形外科・耳鼻咽喉科の領域で扱われていたが,最近では口腔外科・歯科の領域とされている.新しいロボット的リハビリテーション機器として画期的といえる.
