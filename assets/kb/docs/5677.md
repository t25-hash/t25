# 8・1・5 バイオフィードバック

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・1 概説

本来,感覚系神経と運動系神経は対になって機能を果たしている.例えば,ある筋が伸張されればその情報は脳に至って認識されると同時に,脊髄レベルの反射で筋収縮が伸張に抗する動きをする.リハビリテーション分野では求心路経由の情報伝達に支障のある場合に,自己のおかれた生体情報を他の手段で認識し,それを運9 ―140動系(遠心系)に帰還させることで直接情報の代替とするための機器を使用する.頻度として高いものに,深部知覚を傷害された者に,視覚,聴覚などの情報を利用した代替情報として筋放電を音あるいは電影で確認させる手段がある.脳磁図(MEG : magneto-encephalogram)などの情報を直接利用する方法も,近未来の実用化が予測される.
