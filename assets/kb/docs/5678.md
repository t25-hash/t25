# 12・2・2 エルゴメトリー

> **章**: 第12章 健康工学
> **節**: 12・2 生活用品・リラクゼーション機器

エルゴメトリーとは人の作業能力（作業効率）を測定する方法を
指すもので，一般に単位時間当たりの機械的仕事量が求められる．
このエルゴメトリーに利用する装置には，トレッドミル（歩く・走
る装置），スイムミル（泳ぐ装置），クライムミル（登る装置），ス
テアミル（階段装置），固定式自転車，ローイングマシン（こぐ装
置）などがある．これらの装置を利用すると，作業（運動）に伴っ
て変動する生理学的・人間工学的な現象を測定しながら，定量的な
作業量（運動量）を人に与えることができる．本書では，エルゴメ
トリーの代表格であるトレッドミルエルゴメータと自転車エルゴメ
ータについて解説する．
