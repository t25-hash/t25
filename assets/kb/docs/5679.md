# 8・3・1 トルクマシン

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・3 神経・筋・骨格系訓練機器

筋力計測法の世界標準であるトルクマシンは関節を運動中心とした角度,角速度,角加速度,トルクを計測する装置である.日本ではキンコム,サイベックス,バイオデックスが普及している.どの機種もダイナモメータと呼ばれるサーボモータ,トルクセンサ,角度センサ,角速度センサなどが内蔵されている部分に,種々の関節に対応したアタッチメントを取付け,関節の屈曲,伸展の角度,角速度,角加速度,トルクなどを測定する.測定時には図8・4に示すように,アタッチメントのレバーアームの回転中心を測定する関節の回転中心に合わせ,関節の遠位と近位をレバーアームに固定する.純粋に関節のトルクを測定するために,鉛直からの角度を測定してアタッチメントおよび脚の重力補償をしている.関節運動の解析には,大きく分けて次の三つがある.(1) レバーアームを回転させずに関節を伸展または屈曲する等尺性収縮時のトルクの測定.(2) レバーを任意の速度で回転しながら関節を伸展または屈曲する等速性収縮時のトルクの測定.(3) つねにトルクを一定にして関節を伸展または屈曲する等張性収縮時の角度,角速度,角加速度の測定.トルクマシンはそれらの測定ができるように設計されている.最大関節可動域,最大トルク,およびその発揮角度,任意角度におけるトルク,任意時刻におけるトルク,主動筋-拮抗筋比率,最大仕事量,総仕事量などで評価することが多い.制御・演算はPC/AT互換機で,OS(operating system)はウィンドウズ(Windows)のパーソナルコンピュータによって行われている.また,収縮時の筋電図を同時に採取して,筋電図の振幅と筋収縮力の関係などを調図8・4 トルクマシンの概略(バイオデックス社製)査することもある.トルクから筋収縮力を求めるには,筋肉が骨に付着する部分や関節の形状など幾何学的非線形性が存在するので,注意を要する.
