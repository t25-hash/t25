# 8・3・2 TEM,CPM

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・3 神経・筋・骨格系訓練機器

TEM(therapeutic exercise machine)は片下肢の大腿部と下腿部をそれぞれ把持・駆動する二つのロボットアームで構成されているので、股関節と膝関節を矢状面(しじょうめん)内で独立に自由に動かすことができる。ここで矢状面とは、身体の前後上下方向の面(正中面)をいい、真横からの投影図になる。これに対して左右上下方向の面は前額面、水平方向は水平面と呼ぶ。TEM はトルクセンサおよび角度センサを搭載しており、療法士が患者の下肢に対して一度訓練軌道を与えると、設定した周期で設定した回数だけプレイバックするリハビリテーションロボットである。各患者の脚長、重量などのパラメータや教示軌道を記憶しているので、二度めからはID 番号を入力するだけで同じ訓練を繰返すことができる。関節可動域拡大訓練用装置なので、トルクは大きい。しかし、インピーダンス制御を行っているので患者が痛みなどで抵抗力を発揮すると、TEM は自動的に軌道を修正する。繰返し訓練を行う間に痛みや拘縮が低減して抵抗力が小さくなると、最初に療法士が教示した軌道に戻る。関節角度、そのときのトルクを時々刻々測定できるので、拘縮や痙縮の回復状況などがわかる。また、患者の随意的運動を介助する自動介助運動ができる。ここで自動(的)とは患者自身が自らの力で動かすことをいい、他人または機械で動かすことを他動(的)という。英語はそれぞれactiveとpassiveである。自動介助運動では教示軌道に沿った動きは柔らかく、ずれると硬くなるので、患者が正しい脚の動きを学習することができる。逆に、患者の動きの逆向きにトルクをかけることで、筋力トレーニングもできる。さらにTEM は、SLR(straight leg raise、膝伸展位での股関節屈曲)によりハムストリングス[hamstrings、大腿部膝屈筋群(おもに大腿二頭筋)]等の2関節筋のストレッチも可能である。SLR のためにはインピーダンス制御と積分制御を行い、患者が抵抗力を与えると角速度は緩やかになるが、療法士によって与えられた最終目標角度まで股関節を屈曲する(図8・5)。整形外科手術後、特に人工関節置換を施術後に患部をギプスなどで固定し、長期間(3~6箇月)安静にすると、関節が拘縮することがある。その後、拘縮を取り除き、関節可動域拡大訓練をすることは困難が伴う。CPM(continuous passive motion)は非常にゆっくりとした速度で関節を動かし、拘縮を起こさないようにする装置である。同時に関節軟骨修復効果があることが報告され、整形外科領域では広く使われている。手術直後から施術すると、痛みによって自動的には患部を動かせない患者も、他動的にゆっくりと動かすことで痛みが少なくなる。股、膝、肩、手指、肘、顎関節用のCPM が市販されている。TEM とは異なり拘縮予防のために動かすだけなので、トルクは小さく、単純な運動だけを行う。例えば下肢用のCPM は仰臥位で臀部を固定し、足部を矢状面内で動かすだけであり、股関節屈曲時には同時に膝関節も屈曲し、二つの関節は独立には制御されていない。下肢の場合CPM を施行することで、図8・5 TEM の使用例[提供:(株)安川電機]9 ―141深部静脈血栓症が減少したとの報告もある。
