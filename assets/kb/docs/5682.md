# 8・3・4 平衡訓練

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・3 神経・筋・骨格系訓練機器

リハビリテーション関連機器は大きく、保有する能力を評価する目的の機器と能力を保持訓練するための機器に分けられ、さらに感覚系および運動系、反射系に分類されている。運動系の訓練機器は、自発的な筋収縮能力を鍛え上げることが基本となる。すなわち、錘体路を介しての電気的刺激により筋の反復収縮が筋の機能そのものを高めることにつながるわけであるが、もう一つの筋収縮経路として反射に基づく筋収縮がある。本来、生体の防御機構として備わった能力であるが、筋は引き伸ばされると、それに逆らって収縮する。これは伸張反射(stretch reflex)で、脊髄反射路を介して惹起されるが、一般的にはアキレス腱反射、膝蓋腱反射などで知られている機能である。以下に、乗馬を楽しむ際に生体に生じる現象を分析する。馬が動くことによってくらが他動的にまたがっている人間を三次元方向に動かす。くらにまたがっている人間はくらが動くことによってバランスを失わないように、換言すれば落馬しないようにあらゆる機能を駆使する。一つには重心の移動を補正するべく、くらよりも上に位置する上半身の三次元的移動を行う。他方、あぶみに掛けた足部を足掛かりにして体重移動を行い、併せて大腿内転筋を収縮させることで転落を防止する策を講じる。くらにまたがっている本人は自分で訓練を行っている意識をもたない。しかし、結果的に筋は自動運動を行っている。さらにこのことを繰返すことで平衡感覚は促通され、併せて全身の筋は伸張される結果に至り、バランス保持に熟達するための効果をも獲得することになる。馬が並足で歩行する際、ギャロップの際、全力疾走する際、すべて異なったくらの動きを示すが、それぞれのパターンを認識し対応に習熟すれば、乗り手である人間はくらの動きを予測することができるため、落馬することはない。種々のパターンが不規則に組合わされるとロデオのようなくら運動となる。松下電工(株)のジョーバは乗馬シミュレータのリハビリテーション訓練用に開発された簡易版といえる。開発のヒントはヨーロッパで盛んな乗馬療法(horseback riding therapy)にあった。理学療法の一環として脳性麻痺児のバランス訓練に、また筋性腰痛の患者に用いられている。患者も狭い訓練室から広い馬場で馬に触れながら乗馬を楽しんでいる間に訓練が行われていることで極めて快適に訓練時間を過ごすことができ、都市周辺部のリハビリテーションセンター等で有効に活用されている。ジョーバを開発するに当たっては当初くらの動きの要素である、(1)上下動、(2)前後動、(3)左右動、(4)ピッチング、(5)ローリング、(6)ツイストをおのおのモータ駆動し再現させることを試みたところ、他動運動(passivemovement)によって自動運動(active movement)を誘発させうることが確認され、1985年の国立病院療養所総合医学会において報告がなされている。6要素をシンクロナイズさせる技術がないままに放置されていたが、その後のコンピュータ技術の進歩により飛躍的発展を遂げるところとなった。すなわち実際の馬のくらの動きをモーションキャプチャで取り込み、六軸パラレルメカニズムの活用により再現させることで、治療あるいは訓練に効果的な動きの要素を定量的に評価することが可能となった。実際の訓練機器、あるいは治療機器においては、筋電図学的な検索を基礎資料として科学的根拠の検証を試み、シミュレータのくらの動きの中で被検者の各所の筋において最も筋放電の高く検出される動きを選択的に抽出した。検索の結果では姿勢保持に必要な筋として重要な脊柱起立筋群を中心に観察すると、最も筋放電の電位の高く認められる動きは前後動であり、次いで左右動、ローリングであり、ピッチングは前後動で、ツイストは左右動で代償され、上下動もその動きのみでは大きな放電が得られないことが明らかとなり、訓練機器としての機能に付与するべきは前後動を中心とした左右動、ローリングで足りることが判然とした。図8・6に六軸パラレルメカニズムを用いた完全復元型シミュレータを示す。これを用いることで種々の筋電図学的検索を行った。また、図8・7は馬のくらの部分のみを残し他は胴体の中に収納した簡易型訓練機で、高齢者の訓練にも安全に使用できるように配慮したものである。同時に行ったバランス訓練効果検定において、片脚起立能力に36%の改善を見ている(図8・8)。
