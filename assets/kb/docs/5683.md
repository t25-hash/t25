# 8・3・5 開閉口訓練装置

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・3 神経・筋・骨格系訓練機器

物をかむことは単に栄養分を摂取するのみではなく、健康の維持増進に大きな役割を演じており、さらには頭蓋骨形状の形成にも重要な役割を担っている。しかしながら、通常は40~50mm開口できるはずの顎を数mmしか開口できない者が日本全国で数万人から数十万人いるといわれている。このような開閉口運動に障害をもつ者のために1995年から開発が続けられているのが開閉口訓練装置である。ここでは、顎が開けられない開口障害の原因とその分類、従来の訓練方法、開閉口訓練ロボット、ロボットによる開閉口訓練について述べる。a. 開口障害の原因と分類開口障害にはいくつかの原因が考えられる。ここでは大きく原因を4種に分類して述べる。図8・6 六軸パラレルメカニズムを用いた完全復元型シミュレータ図8・7 簡易型訓練機(1) 顎関節部分の癒着上下顎間にある顎関節部分の繊維性癒着がおもな原因である。(2) そしゃく筋の機能不全下顎を駆動する筋に何らかの機能不全があるために、下顎を開けることができない状態。(3) 顎骨の損傷交通事故などで顎骨に物理的損傷を被ったことに起因する開口障害。(4) その他軟組織・骨・筋に異常は見受けられないものの、開口することが困難な場合。b. 従来の訓練方法a項で述べた開口障害の分類のうち、本項では(1)を対象とする。その訓練には洗濯ばさみ型木製訓練器が用いられるが、これは長さ20cmの木製洗濯ばさみの一端を患者の前歯あるいは残存歯列面に挿入し、医師または患者自身が他端を握り締めることにより強制的に下顎を開口させる訓練法である。c. 開閉口訓練ロボットb項で述べた従来の訓練法では、下記に示す問題点があった。(1) 訓練時の定量的データが不明。(2) 木製開口器が接触している歯列部分に大きな力が作用する。(3) ヒトの下顎の自由度は少なくとも3であるのに対して、木製開口器は開口のみの1自由度しか有していない。そこで開閉口訓練装置は1自由度、3自由度、6自由度の3種類が開発された。. 1自由度開閉口訓練装置従来の木製開口器は手動で駆動されていたため、定量的なデータ確保が困難であった。そこで、木製開口器にひずみゲージ式力センサとアクチュエータを付加した1自由度開閉口訓練装置が開発された。. 3自由度開閉口訓練装置下顎は少なくとも3自由度を有していることを反映させた装置である。これにより患者は開閉図8・8 身体的効果計測(筋力)ものと考えられる。図8・9開閉口訓練ロボット9 ―143口・前後・左右への開口訓練を受けることが可能となった。さらにこの装置ではマスタスレーブ方式を採用した結果、患者からの反力を医師が感じることが可能となった。. 6自由度開閉口訓練装置(図8・9)ユニバーサルデンタルロボットと名付けられた本装置は、患者の下顎を任意の位置と姿勢に制御可能な6自由度を有した訓練装置である。d. ロボットによる開閉口訓練図8・9で示した開閉口訓練ロボットによる開閉口訓練を行った結果、複数の患者の開口距離が増大した。
