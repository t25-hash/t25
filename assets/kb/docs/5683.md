# 12・2・7 カロリー（エネルギー）計算

> **章**: 第12章 健康工学
> **節**: 12・2 生活用品・リラクゼーション機器

a.食物からの摂取エネルギーとは 食物のエネルギーは，一
般的に熱量またはカロリー（calorie）という尺度で表される．1
kcalは1000calのことで，1calは1gの水を14.5℃から15.5℃
まで1℃だけ上昇させるのに必要となる熱量を指す．1kcalは
4.184kJ （kilojoules ）に相当する．糖質，脂質，タンパク質の三
大栄養素を試験管内で燃焼させると，1g当たりそれぞれ4.1，
9.3，4.1kcal （窒素成分を差引いた値）のエネルギーが発生する．
生体内では食物の消化吸収率が若干異なるため，それぞれの栄養素
1g当たり4，9，4kcalがエネルギー計算に用いられる．各栄養素
が酸化される量を一定にして比べると，脂肪は糖質よりも2倍以上
のエネルギーをもっていることになる．
生体のエネルギー代謝量は，直接的には完全に断熱材で包囲され
た小さな部屋の中を循環する水温の変化から放熱量を測って計算さ
れる（カロリメトリー，calorimetry）．しかし，一般的には生体内
での酸化に必要となる酸素（O ）量，その結果として発生する二
酸化炭素（CO ）量，それにタンパク質の異化によって尿中に排出
される窒素（N）量から，生体内で燃焼された栄養素の分量比を計
算して間接的に求める方法がとられている．

栄養素の種類によって発生するエネルギー量が異なるため，同一
時間内に消費されるO量と排出されるCO量の比でもって，体内
で酸化される栄養素の割合を推定する．呼吸商（RQ:respiratory
quotient ）とは，この比（CO排出量/O消費量）のことを指して
おり，糖質が有酸素性エネルギー供給機構によって完全に酸化され
る場合のRQは0.75÷0.75＝1.0，脂質がTCA（tricarboxylic
acid cycle，citric acid cycle）回路および電子伝達系（呼吸鎖）を
経て酸化される場合は1.43 ÷2.03＝0.71，Nを含まないタンパク
質では0.76 ÷0.95＝0.80となる．仮に体内で燃焼する栄養素が糖
質と脂質のみであるとすれば，RQの値から両栄養素の代謝率が計
算できる．RQが0.85であれば糖質と脂質が約50 %ずつ，また
RQが0.76であれば糖質が約20%，脂質が約80 %代謝された計算
になる．ヒトを対象としたときのRQは呼吸交換比（RER:respi-
ratory exchange ratio）と呼ばれることが多い．
c.運動中のエネルギー代謝量 1kcal /min＝3.3mL/(kg･
min）として単純に計算すると，平常歩行のような60～90m/ min
ぐらいの軽い運動時のエネルギー代謝量（酸素摂取量）は，
2.1～3.6kcal / min［7～12mL/(kg･ min）］程度となる．体重によ
っても異なるが，平常歩行で1分間当たり平均3.3kcalの消費と
仮定すれば，約30分（総距離にして約2.4km）で100kcalを消
費する計算になる．急歩であれば，4.5～6.0kcal / min［15～20
mL/(kg･ min）］ぐらいまで増える．急歩で約5.3kcal / minを消
費するとすれば，19分（約2.4km）で約100kcal消費できる計
算になる．同じ速さの運動でも個人の走能力によってジョギングで
あったりランニングであったりし，全身持久性体力に劣る人にとっ
て時にはそれが激しい運動ともなる．平均的な体力の成人男性の場
合，通常，ジョギングのような中等度の運動では6.0～7.5kcal /
min［20～25mL/( kg･min）］，かなり激しい走運動では9.0～12.0
kcal / min［30～40mL/( kg･ min）］ぐらいである．ごくふつうの
健康な男性が3kmの距離をレースのつもりで全力を出して走れ
ば，12～20分ほど（一流ランナーでは8分）かかるが，最後の
1～2分間に猛烈なラストスパートをかけることができれば，個人
の酸素摂取量は最大になる．この最大値を最大酸素摂取量（maxi-
mal oxygen uptake）と呼ぶ．短時間の最大運動時において発現す
る最大酸素摂取量は，ごくふつうの健康な20歳代の男性A氏の場
合，40～45mL/(kg･ min）ぐらいになる．この高いレベルの酸素
摂取水準を維持しながら運動を行ったと仮定した場合，A氏のエ
ネルギー代謝量は12.0～13.5kcal /min（せいぜい15kcal / min）
になると考えられる．しかもふつうの人なら，わずか数分間しか続
けることができず，単純に計算すれば12～15kcal /min×5min＝
60～75kcal （100kcalに満たない）となる．実際にはこのような
最大運動を行った場合，運動終了後にも数分間以上にわたって安静
水準を大幅に上回るエネルギーが消費される．
