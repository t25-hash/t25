# 12・2・8 電動歯ブラシ

> **章**: 第12章 健康工学
> **節**: 12・2 生活用品・リラクゼーション機器

電動歯ブラシ（electric tooth brush）は，駆動機構と動力源な
どを内蔵する把柄部と，把柄部に着脱できる頭部より構成され，歯
面の清掃や歯肉のマッサージを目的とするものである．頭部の形
態，運動様式，運動の速度，動力源の違いによりさまざまなタイプ
に分かれる．電動歯ブラシの構成を図12･28に示す．
a.頭部の形態 頭部のはけ部の形態は，手用歯ブラシ様のも
の，円形のものが代表的である．これに加え，歯間を磨く，歯周ポ
ケットを磨くなどの特別の目的に絞った特殊な形態のものが存在す
る．
b.運動様式 頭部の運動様式には，往復運動，アーチ型運
動，反復回転運動，振動運動，複合運動がある．運動様式の種類を
図12･29に示す．往復運動は，頸部と平行な方向に動く．アーチ型
運動は，頸部の軸を回転させて動く．反復回転運動は，頭部が円形
のものと組合わせてはけ面が円を描くように回転する．振動運動
は，はけ部が微振動する．複合運動は，上述の運動が組合わされた
動きをする．また，近年では，頭部の運動と同時に，頭部より超音
波を発生させる方式や，微弱電流を発生させる方式も存在する．
c .運動の速度 機種によりさまざまであるが，1000～4000
minで動くものが多い．機種によっては運動の速度の切換機能を
一つである．

d.動力源 充電式と乾電池式がある．充電式は，電源につな
がっている充電機能をもつスタンドに歯ブラシを立てかけて，把柄
部に内蔵されている蓄電池を充電してから使用する．近年では，電
磁誘導によりスタンドと歯ブラシを電気的に接続することなく充電
を行う無接点充電方式が採用されている．乾電池式は，把柄部に乾
電池を装着して使用する．
