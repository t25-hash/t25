# 8・4・1 バイオフィードバック

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・4 その他

バイオフィードバック(biofeedback)は生体信号を映像,音,皮膚刺激などによって本人に提示することで,身体機能を随意的に制御しやすくさせる方法である.一例を挙げれば,中殿筋に電極を貼付して筋電図(EMG : electromyogram)を記録し,スピーカに接続して音波として被験者へフィードバックしたり,あるいは視覚的に提示したりすることで,歩容改善を促す方法がある.これは一般にEMG バイオフィードバックと呼ばれる.骨格筋の収縮はもともと随意的に行えるため,その制御をバイオフィードバックにより精緻に行えることは容易に想像できるが,心拍数,血圧などの自律神経系支配の生体信号もまた,バイオフィードバックにより一定の制御が可能であることが報告されている.バイオフィードバックは知覚しにくい身体の働きに注意を向かせ,随意的な制御を容易にさせることから,機能障害のある者の治療法として広く用いられている.バイオフィードバックを運動学習法の一つとして考えた場合,生体情報を定量的にかつ即時的に提示できるため,高い学習効果が期待できる.筋力増強効果を促すEMG バイオフィードバックの利用法としては,例えば脊柱起立筋の強化による慢性腰痛の軽減,膝十字靱帯再建術後における大腿四頭筋の強化がある.また,EMG バイオフィードバックでは,通常では知覚しにくい筋収縮状況を容易に把握でき,制御することも可能である.例えば,歩行中に外側広筋をなるべく弛緩させ,内側広筋を意識的に収縮させることができるが,これを利用して膝の骨関節炎による痛みを軽減することが可能である.このほか,ギランバレー症候群,顔面神経麻痺,末梢神経損傷,脊髄損傷など,神経疾患による筋力低下を防ぐ方法として有用である.また,単に筋力増強という観点からだけではなく,神経再支配の際の再教育法としても利用される.例えば,顔面神経と舌下神経の吻合後,舌を弛緩させながら顔面運動を行う訓練や,筋皮神経と肋間神経の吻合後,呼吸に独立させながら上腕二頭筋を随意収縮させる治療法に用いられる.バイオフィードバックは,筋の弛緩を意識させる方法としても利用される.前頭筋の弛緩を促す利用法は,頭痛の緩和に有効であったと報告されている.また,不随意な筋収縮により頭部が片側に傾いたままとなる痙性斜傾に対しても,胸鎖乳突筋の弛緩を促すことで改善が見られる.EMG 以外の生体信号を利用したバイオフィードバックとしては,嚥下(えんげ),排泄不全に対する圧力フィードバック,血管性頭痛や指動脈の攣(れん)縮(レイノー病)に対する皮膚温フィードバック,高血圧や起立性低血圧に対する血圧フィードバック,不眠症に対する脳波フィードバックなど,多岐にわたる.近年新しく提案されているバイオフィードバック法としては,機能的核磁気共鳴映像法(fMRI: functional magnetic reso-nance imaging)を利用して,直接被験者に脳の活動状態を提示するといったものや,視力回復のためのフィードバック装置などが報告されている.このようにバイオフィードバックは,身体の働きを知覚し,その機能を目標に近づけるうえで大きな助けとなる方法である.リハビリテーションを中心として,その臨床応用は今後も広く進められる9 ―144医療・福祉・バイオ機器
