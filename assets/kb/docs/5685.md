.揉捏（じゅうねつ）法 最も重要な手技で，指の腹全体
で圧力を加えながら小さな円を描くように，おもに筋肉をもむ．新
陳代謝をよくし，疲労物質をとる作用がある．マッサージ機では，
もみ玉のもみ上げ，もみ下げなどに当たる．
.強擦法 関節や骨の間に，母指で圧迫しながら中心部に
向かい渦巻状にマッサージを行う．関節などの滲出物をもみ砕き，
ゆ着した組織を引きはがす作用がある．マッサージ機では，関節や
骨に接触する手技は行わない．
.圧迫法 指頭や手掌などで，体表面に垂直に一定の圧力
でゆっくり押していったん止め，またゆっくり力を抜く．いわゆる
指圧（shiatsu）のことをいう．神経や筋肉の機能の高ぶりを抑え，
興奮を鎮める作用がある．マッサージ機では，指圧機能に当たる．
.振せん法 手掌，母指，四指を使い，軽く押しながら震
わせ，細かいリズミカルな振動を与える．神経や筋肉の緊張をとき
ほぐす作用がある．マッサージ機ではバイブレータ機能に当たる．
.叩打法（tapotement ) 施術部位を小指側，こぶし，指
などで軽快に叩く方法で，切打法，拳打法，指叩打法，拍打法の叩
き方がある．新陳代謝をよくし，筋肉の疲れをよくする．マッサー
ジ機では，叩き機能に当たる．
c.治療の順序 疲労回復コース，こり改善コースなどマッサ
ージ師が行う手技や治療の順序に合わせるようにマイクロコンピュ
ータで制御された自動コース機能が作られている．また，従来，治
療部位が肩，背中，腰部などに限られていたが，大腿後面，下腿後
面，上肢をマッサージできる椅子式マッサージ機が開発されたた
め，治療コースの新たなバリエーションが可能となっている．
d.刺激量 もみの強さにはもみ玉の硬さや大きさなどが関連
しているが，脊柱のわん曲の状態も影響しているため，背もたれの
角度により強さを調節している場合が多い．従来のリクライニング
機能とは別に，体圧センサを用いたタイプが開発されている．
また，手指の脈拍，皮膚温度，発汗などから体調を判断して，刺
激量を決める機能をもったものが市販されている．
e.椅子としての機能 椅子式マッサージ機は，マッサージ機
能以外の条件として，キャスタを付け移動が容易であること，椅子
としての見栄えがよいこと，クッションなどクリーニングが可能で
清潔であること，無段階のリクライニングを有していることが重要
である．
