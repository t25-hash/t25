# 8・4・2 ファシリテーションと可塑性

> **章**: 第8章 機能回復訓練機器(医学的リハビリテーション機器)
> **節**: 8・4 その他

中枢神経系(脳、脊髄)におけるファシリテーション(facilita-tion)とは、神経の興奮性インパルスが反射中枢に達して、神経活動が強化されることである。また、可塑性とは外界からの刺激によって神経網の構造や重み付けに変化が生じることをいう。可塑性は外界の変化への適応や、外傷などの構造変化に対応する機能として大きな役割を果たしている。本項では、中枢神経系の可塑性を外部からファシリテートする研究について、ヒトの運動機能に関する部分を中心として概説する。大脳皮質運動野をファシリテートする方法としては、経頭蓋的磁気刺激(TMS : transcranial magnetic stimulation)が代表的である。すなわちコイルへ瞬間的に電流を流し、発生した変動磁場によって脳内に渦電流を発生させる。さらに、この電流によって並行して走行する軸索を刺激する。この軸索は他の脳細胞とシナプスを形成しているため、シナプスの可塑性を誘発することが可能である。TMS を利用した運動野の可塑性に関する報告を概述する。まず、大脳皮質運動野の右拇指支配領域を磁気刺激した。その後、磁気刺激で誘発される動きとは異なる動作を被験者に行わせる訓練をし、その結果、訓練開始から30分後には、磁気刺激で誘発される拇指の動きは訓練した動きと同じになった。この著明な変化は5~20分程度持続したとある。また、感覚系を刺激し、その刺激が運動野に到達した際に運動野を連続経頭蓋磁気刺激(rTMS :repetitive TMS)すると、運動野誘発電位(MEP: motor evokedpotential)は刺激前に比べて高振幅となり、rTMS の刺激時間を長くすることによってその効果も長く残存する。1Hz のrTMSを運動皮質に与えると、伸張反射や運動点電気刺激により誘発される反射の長潜時成分のみが有意に減少することから、各成分の反射中枢を特定する方法としてもrTMS は利用できる。また、末梢感覚神経への電気刺激によっても脳の可塑性は生じ、総腓骨神経へ200Hz の間欠的電気刺激を与えた結果、支配筋である前脛骨筋へのMEPの振幅が増加したとの報告もある。上記のような実験的方法を用いずとも、スポーツ選手では動作習練がファシリテーションの要因となり、利き手の運動野が対側に比して拡大し、その位置も異なり、その変化は長期に残存することが知られている。また、末梢でもファシリテーションがあり、平目筋に誘発されるH 反射が非習練者に比べて有意に増加しているとの報告や、脊髄へのrTMS によってH 反射が抑制され、数十分程度その効果が持続したとの報告もある。中枢神経系の可塑性に関する研究は、神経筋系の機能障害に対する治療法の開発へと向けられているものの、可塑性を人為的にコントロールするにはまだ多くの研究が必要である。リハビリテーション分野の新しいアプローチとして、今後の発展が期待される。文献(1) Fujie,M.,Proc.
