# 9・2・1 義手(upper limb prosthesis)

> **章**: 第9章 支援機器
> **節**: 9・2 義肢装具

下肢が歩行という周期的な運動を基本とするのに対して、上肢は運動的にも感覚的にも多種多様の動作を行い、多関節の協調動作と指等による巧緻動作を行っている。臨床で使用されている現在の義手において機能と外観を同時に満足することは困難であり、基本要素に分解された簡単な動作を個別に行う義手が一般に用いられている。機能を最優先する場合には、能動義手として把持動作を行うかぎ形をした能動フックが使用される。能動義手のうち、ある程度の外観と把持動作を両立させるものとしては手の形をした能動ハンドや電動義手(electrical upper limb prosthesis)が、また外観を最優先する場合には装飾用義手として装飾用ハンドが使用される。これらを図9・1に示す。このほかに専用の作業用に準備された手先を使用する作業用義手を使用する場合も多い。これらの手先は職種に合わせて製作されることが多く、大工作業用、農作業用、漁業用等、使用者が行う作業に合わせて製作される。別の分類方法として、手先や肘を動作させる場合の駆動力源として、使用者自身の身体の動き(筋力)を使用する体内力源義手と、モータなどを使用して駆動し、使用者は信号のみを送る体外力源義手に分ける方法もある。体外力源としては、一時、空気圧による駆動方式が試みられたことがあるが、現在は電池を使用する電動義手のみが製品化されている。また、使用者の信号を筋電から得る筋電義手(myoelectric upper limb prosthesis)と、何らかのスイッチを意図的に押すことなどにより動作させる動力義手とに区別する場合もある。適切に製作・調整された義手を用い、十分な訓練を受けることができた義手使用者は、両手動作を含む細かな作業が可能になる。しかし、片手動作に関しては、片側の手を使うことができれば日常生活の90%は動作可能という報告もあることから、義手を使用しない、または装飾用義手のみを使用する上肢切断者も多く、臨床的なニーズを的確につかむことは容易ではない。a. 体内力源能動義手身体の動きを義手の肘や手先に伝える手段として、ハーネスとケーブルを用いて義手側の反対側の肩の動きを利用する方法が挙げられる。ケーブルは背中を通るように走行させ、背中を丸めるような動作をすると距離が延長する。この動きを手先に伝えることで指の開き動作とするものである。この場合には指を閉じる動作はスプリングやゴムリングの復元力で行う。この方式のものを随意開き式、逆の動作のものを随意閉じ式と呼ぶ。図humeral prosthesis)では、肘も屈伸させる必要があるため、義手側の肩の上下で肘継手のメカニカルロックのメカニカルフリップフロップをオンオフする動作も行う。これらの複雑な動作を習得するためには作業療法士による義手装着訓練が必要であるほか、ハーネスやケーブル走行を的確に配置することができる義肢装具士等が必要である。b. 電動義手ソケット内壁に埋め込んだ電極で検出した皮膚表面の筋電位の変化や、ソケット内の断端の動き等で切換えたスイッチのオンオフ信号をもとにして制御回路を通してモータを駆動し、手先の開閉や手首の回旋、肘の屈伸等を行う。筋電義手の制御は、歴史的には手首の掌屈と背屈を行う筋肉からの筋電を比較することによって駆動信号としてきたことから、2個の電極を使用して多チャネルの動作の識別を行う研究が多く見られたが、近年は4個の電極を用いる研究や、それ以上の数の電極を用いて各指の動きを識別する研究なども行われるようになってきた。また、手の柔らかさを義手に取り入れるためには、制御によって実現する研究と、メカニズムを開発することで実現する方法がある。産業技術総合研究所では、容易に義手使用訓練を行えることを目標に障害者の個々の特徴を柔軟に学習する進化型LSI(large scaleintegrated circuit)を開発し、同時に手の各指と手首にモータを備えた手先のメカニズムを開発した。このメカニズムは図9・3のような製品として市販化されたが、高価であることと、手としての外装部品が準備されていなかったことから、購入して使用していると図9・2 能動義手の一例(上腕義手)図9・1各種の義手手先具[左から能動フック、能動ハンド、筋電(電動)ハンド、装飾用ハンド]図9・3 進化型LSIに対応して開発された筋電(電動)義手(各指と手首にモータを内蔵している)いう例は聞かない。これまで筋電義手として製品化された例は多いが、継続して製造販売されている機種は少なく、アメリカなどで製品化されていた手先も販売が中止され、先進国では現在はドイツのOtto Bock 社とイギリスのSteeper社、およびアメリカのMotion Control社等が製造しているのみである。中国やロシアでも製造されているが、自国内での使用にとどまっているようである。国内では20年ほど前に早稲田大学と(株)今仙技術研究所によって共同開発された義手があるが、使用例は少ない。
