# 9・2・4 開発・設計・製作技術

> **章**: 第9章 支援機器
> **節**: 9・2 節冒頭で述べたように装具の範ちゅうは広

義肢装具は一般的な工業製品と異なる点を多くもつ。ほとんどのものは、「身体障害者福祉法」や「労働者災害補償保険法」をはじめとする福祉関係の法律に基づいて支給される。前者では使用者の自己負担額は製品の価格とはかかわりなく、使用者の前年度の納税額によって決定される。そのため、製品が安価でも高価でも負担額は同一となっている。一例とし、インテリジェント大腿義足は通常80万円程度の価格であるが、使用者の自己負担分は1万円前後であることが多い。残額は国、都道府県、市町村の福祉予算を使用して50:25:25で分担することとなっている。後者では使用者の自己負担はなく、基本的には身体障害者福祉法で製作するものと同じ義肢装具を製作することができる。これらに使用できる部品や製作方法は厚生労働省が認定した価格表に掲載されているものを基準とし、特別の手続きなしに使用できる。価格表に載っていないものを使用する場合には基準外の部品等を使用することとして、理由書等図9・9外骨格型動力装具の例(筑波大学HAL-3)図9・10 ―150医療・福祉・バイオ機器を添えて申請し、市町村が独自に判定することとなっているが、実際に認定される例は少ない。最近は義肢装具も高機能化されてきており、従来に比較して高価な部品も認定されるようになってきたが、法執行上の常識を超えると考えられる金額のものは、基準外での申請を行わなければならない。義肢装具のほとんどは使用する障害者に合わせて義肢装具士が製作する。義肢装具士は3年以上の専門学校等を卒業することで国家試験を受験する資格が与えられるが、現在のところ、わが国では2800名程度が資格をもっている。一部には工学部を卒業後に専門学校へ進んだ者や、資格取得後に工学部へ進んだ者などがいるが、ほとんどの者は専門学校の教科の一部として工学的な科目を学んでいるのみであるため、高度に専門的な内容を含み、取扱いに工学の基礎知識を必要とするようなものを扱うことは難しい場合が多い。使用者である障害者の中にも工学的な背景をもつ者もいるが、ほとんどの場合にはそれを期待することは困難で、しかもメカニズムを理解したうえでマンマシンシステムとして取扱うことができる者は少ない。したがって、使用者にとっては義肢装具は自分自身の生活を送るうえでの道具としての存在であり、その調整などに必要な時間は短いほど歓迎される。ただし、同じ道具を使用しても使いこなしによって結果が変わるように、ほとんどの機能的な義肢装具では装着訓練および使用訓練の有無とその程度によって結果が変わる。このような状況であるため、現在の義肢装具の機能改善に対する潜在的な要求はあるものの、そのために取扱いが不便になったり、調整に長時間を要するようなもの、あるいは使用者自身が障害者であることを強調するような外観をもったものなどは受け入れられない場合が多い。特に、義肢装具士にとっては厚生労働省がそれぞれの部品と製作法に対して定めた金額しか請求できない制度になっているため、製作と調整に必要な時間が短いほど収益性は高いことになる。図9・11にインテリジェント大腿義足の例において、基礎研究モデルを臨床的に受け入れられるために実用モデルに再設計したことによって、義肢装具士が義足を製作し、義足使用者が日常生活で使用するまでに必要なプロセスが簡単化され、短縮された例を示す。図9・11から、研究段階のモデルではステップ数が多く、また最適なデータを得るためのループがあることに比較して、実用モデルでは少ないステップ数でストレートに進むことができるように改善されたことがわかる。義肢装具の機能を研究開発する立場から考えると、新しい家庭用電気製品のように、高機能のものを製品化すれば新しい市場が開けて使用者数が増加するというような市場ではない。また、工場で生産したものを障害者に届けるだけで使用できるものではない。した(a) 研究段階の最適制御データ決定プロセス(b) 実用モデルの制御データ決定プロセスので使いやすい。図9・11研究開発モデルと実用モデルのデータ決定プロセスの比較がって、現在の義肢装具士の技術レベルと義肢装具の支給制度を知り、そこで受け入れられる機能と技術的内容、および価格設定を念頭において開発を進めなければ、障害者のQOL(quality of life、生活の質)の向上に役立つものにはならない。このことは技術的に進んだ義肢装具だけに限るものではなく、例えば発展途上国に対して国際支援として義肢装具を供給することを検討する場合にも、現地で義肢装具士を教育することなく、単にものを届けることだけを考えていては実効性がないことは明らかである。また、安価に製造するためにマスプロダクションの手法を取り入れようとしても、最初からまとまった生産量を見込むことが不可能で、立上げ時の少量生産時から、ある程度まとまった生産量が見込めるようになるまで、長期間、安価に供給できる製造システムを確立することも必要と考えられる。このような状況の中で新しい義肢装具を開発するためには、開発者自身が上記のような現場の状況を熟知する必要がある。また、開発者、製造者および使用者による評価等のシステムができあがっていない分野と考えられることから、開発者が製造から義肢装具士による使用上の指導、さらには使用者の評価にまでかかわってリードしなければ、広く使用されるものにすることは難しいものと考えられる。
