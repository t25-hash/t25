# 9・3・2 視覚障害者用端末機器等の触覚関係機器

> **章**: 第9章 支援機器
> **節**: 9・3 視覚障害者用機器

a. 点字ピンディスプレイと携帯用点字入出力機器PCの操作内容およびデータを点字として確認できることは、点字使用者にとって重要な機能であるといえる。しかし、点字ピンディスプレイ(braille pin display unit)で完全に確認が行えるのはほぼデータ部分だけである。音声確認の代わりに点字ピンディスプレイだけでPCを操作できる状態とはほど遠いのが現状であるが、とりわけ点字が読める視覚障害者で後に耳が不自由になった盲ろう者にとっては、不可欠といえる重要な出力形態である。なお、テキストファイルを自動点訳して点字ピンディスプレイ等で読むことは広く行われている。点字ピンディスプレイはピンの駆動部を含めて原価が高いため、マス数によって用途が分かれていることが多い。例えば、全国の点字図書館等ネットワークであるないーぶネット(9・3・5・e項参照)では、標準点字器のマス数と同じ32マスが標準とされているため、行情報や制御コードなどを表示する部分と合わせて40マスの点字ピンディスプレイが一つの標準品となっている。22マスや18マスなどとマス数が少ない点字ピンディスプレイの小型機種は、PCの端末としてだけではなく、単体で携帯用点字入出力機器として手軽に使用できる機能をもつようになってきている。点字キーで入力でき、メモリに記録したデータを点字ピンディスプレイや、機種によっては音声でも確認できる小型の機器が、携帯用の点字メモ機や音声電子手帳として利用されている。また、PDA(personal data assistance)ともいえる幅広い機能を付加する機種も登場しており、点字使用者のニーズに適合した装置として注目されている。b. 点字プリンタ点字データを確実に読むには、点字プリントされたハードコピーが最もよい。現在の点字プリンタ(brailleprinter)は、厚みのある点字用紙に機械的に凹凸をつけていくタイプのため、紙の固定が重要で、両サイドの連続穴によって紙が固9 ―151定されるトラックフィードタイプが圧倒的に多く、単票紙に印字するタイプはわずかである。また、動作方式では、点字の1マスごとに打っていくシリアル方式と、1行の上位点・中位点・下位点の順に高速で打っていくラインプリンタがある。なお、日本式の点字を中心としたタイプはマス間がやや狭く、欧米型のものはマス間がやや広いが行間は狭い。増加している中途失明者用に、マス間・行間ともに広げた点字プリンタも開発されている。点字を正確に大量に製作するには、薄い亜鉛板やプラスチック板を二枚重ねにして点字を打刻した原版を作り、その2枚の間に点字用紙を挟んでローラに通す、エンボス印刷と呼ばれるプレス印刷方式が日本の点字制定の数年後から用いられている。その原版を点字データから自動的に製作する点字自動製版機は1980年に開発され、点字印刷を行っている点字出版所等で使用されている。c. 点字読取装置紙に凹凸で書かれた点字の文をデータ化することが必要なとき、または点字を知らない人が点字を読まなければならないときのために、点字OCR(optical character reader)ともいうべきシステムがある。点字の凹凸を光の陰のパターンとしてとらえてパターン認識し、点字データとして処理するシステムで、通常のOCR でも凹凸の陰影が明りょうに認識できる機種を利用してシステム構成しているものもある。点字が片面(一つの面だけ凸)のみに書かれている場合は処理精度も高く、実際に活用されているが、両面(表側が凸の点字行の行間に裏側が凸の点字行があるものが多い)に書かれている場合は処理精度が低くなることが多く、実用に堪えないことすらある。いずれも、点字プリンタや点字製版機により点字プレス印刷されたものは読取り精度が高いが、手書きのものは精度が低くなる。d. 立体コピー機触ってわかるように盛り上げた図を簡便に製作する方法で、1980年に日本で開発された。熱吸収のよいトナーを使用している白黒コピー機によって、特殊な感熱発泡樹脂を塗布した特殊用紙に、盛り上げたい部分が黒く書かれた原稿を黒色コピーする。キセノンランプなどによってコピーのトナーがよく熱吸収する波長の光を当て、特殊用紙に付着したトナーの黒い部分のみが熱吸収して、感熱発泡樹脂が大きく発泡することを利用しているのが立体コピー(uprised copy using capsulepaper)という手法である。触る図形を比較的簡単に製作する方法として広く利用されているが、難点は特殊用紙の価格がA 4用紙1枚でも100円近くすることである。なお、図形については、墨字(目で見る表現)の図のまま盛り上げても、触図としてはわからないことが多い。それは、第一に、目では一瞬のうちに全体が見渡せるのに対して、手指では指先が触っては消えていく各瞬間の小さい部分の情報を記憶し、それらの断片情報をつなぎ合わせて頭の中で描く必要があるためである。第二に、目の分解能と手指の分解能は面積比で数百分の1で、単位面積当たりの情報量に大きい差がある。第三に、目では遠近感が得られるのに対し、手指では平面的にしか感じないので、斜視図は展開図や断面図など平面的な図の組合せにしたり、省略して文章表現にしたりする必要がある。このような特性の違いに十分に留意して、墨字情報から触覚情報へ変換する方法を考えなければならない。
