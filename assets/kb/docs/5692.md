# 9・3・3 読書関係等の音声機器

> **章**: 第9章 支援機器
> **節**: 9・3 視覚障害者用機器

a. ディジタル録音再生機器視覚障害者のおもな情報入手方法は、点字と音声である。特に近年では高い比率となっている、中高年以降に視覚障害になった人たちの場合は、点字触読の習得は相当困難であり、音声情報に頼らざるを得ない人が圧倒的に多い。従来から点字図書館では、音声図書が多くのボランティアによって製作され、カセットテープの無料貸出しサービスによって音声情報の提供が支えられてきた。この音声情報については、1990年代に入ってディジタル化が進み始め、圧縮技術も進歩してきた。視覚障害者用のように、音楽デ9 ―152医療・福祉・バイオ機器ータではなく音声のみの場合は大幅な圧縮が可能であるが、視覚障害者用の圧縮方式がさまざまになると視覚障害者自身が不利益を受けることになるので、音声ディジタル化の国際標準規格としてDAISY(digital accessible information system、デイジー)という方式を日本が提唱し採択された。わが国は世界で最初に専用再生装置を開発して発売を開始し、また点字図書館の録音物のDAISY化を国家予算によって推進させるなど先駆的な取組みを行ってきた。このDAISY 方式(DAISY standard)の特徴は、無音に近い間の識別によって編集を行い、インデックスをつければ必要なデータの部分へすぐに移動することができること、および1枚のCD(compact disk)に録音テープ約50巻が入る大容量であることが大きなメリットとなっており、貴重な録音資料の保存劣化も防止できるが、再生機の普及が遅れている。今後は、専用読書機を必要とする視覚障害者全員が入手し利用できるようにすることによって、カセットテープとの世代交代を加速していくことが望まれる。b. OCR 読書システム紙に書かれた日本語を読み取って聞きやすい発音で出力する機能をもつ日本語読書機の普及型は、1996年まで現れなかった。これは、日本語においては、まず漢字の読取りにけた違いの精度が必要なこと、成分の切れ目と複数ある読みの確定には膨大な処理が必要なことなど、日本語自体の問題によるものである。その中で日本語処理の技術も進展し、はん用OCR を活用した日本語読書機、つまり墨字文書をOCR の上に置くと文書を合成音声で読み上げる装置が開発され利用されている。装置には、読書器として視覚障害者が利用しやすいように専用装置となっているタイプと、市販のOCR を利用している機種とがあるが、後者でPC使用の場合には、一般のOCR ソフトウェアと音声読上げソフトウェアの利用が多くなっている。
