# 9・3・4 拡大読書機などロービジョン(弱視)者用機器

> **章**: 第9章 支援機器
> **節**: 9・3 視覚障害者用機器

ロービジョン(low vision,弱視)とは,どのような眼鏡等を使用しても正常な視力や視野とはならない状態をいう.その中でも,少しでも見え方が改善されるように,弱視レンズやフィルタ等のレンズ系での調整は重要であり,眼疾のもたらす視力・視野に関する知識や経験のある眼科医によってかなり改善されることがある.墨字資料を読むとき,障害が軽度の場合はレンズ系によってある程度は文字等を読むことができるが限界があり,その場合は数倍から十数倍までの連続的な拡大率をもつ拡大読書機によってかなり補うことができる.これは,カメラとその拡大機能,およびディスプレイの組合わされた機器であり,代表的なものは一体化されていることが多い.台の上に置かれた資料を上部のカメラでとらえ,ディスプレイに拡大表示する構造で,文字を追いやすいように資料台を横方向または縦方向に軽く移動できるものが主である.この機器は日常生活用具に指定され,19万8000円以下であれば本人の負担が1割以下になるため普及が進んでいる.おもな機能としては,多くの強度弱視者が必要としている白黒反転機能があり,焦点合せが困難なロービジョン者のためにオートフォーカス機能も標準装備されてきている.そのほか,携帯性のよいものや,見やすい液晶ディスプレイが目の前まで自由に移動できるものもある.なお,手元に近い部分を拡大する用途以外に,カメラ部分を回転させて遠くの黒板や掲示なども見えるようにしたものもある.一方,一般のPCの画面は,画面自体は大きく見やすくなってはいるが高密度化が進み,晴眼者(sighted people)でも見にくくなってきている(晴眼者とは視覚に障害のない人をいう).ウィンドウズには標準で障害者用の設定がついており,ある程度の拡大や強調する機能も進んできてはいるが,弱視者用としては大幅に拡大する必要がある.それに伴って,手元の資料の拡大画面と,使用中のPC画面の弱視用の拡大表示とを,一つのディスプレイの中で分割表示できる拡大読書機も出されており,離れた画面を比較することの困難な弱視者にとって便利になっている.
