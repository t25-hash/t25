# 13・3・2 情緒と機械

> **章**: 第13章 感性・情緒工学
> **節**: 13・3 情緒工学

情緒は情動と同じ意味合いであるが，その定義は多様である．こ
こでは感情と情緒を区別するため，認知心理学での定義を用いて，
情動とは，感情と，感情に伴う身体的運動変化，自律神経変化，心
理変化のすべてを包含する過程であるとする．
精神医学者の小田は，1984年に開催された「情緒ロボットの世
界―機械とヒトとの新しいコミュニケーションを求めて」におい
て，情緒ロボットを三つに分類した ．ここではロボットを機械
に拡大して，情緒工学における機械を次の三つに分類する．
(１)感情的（情緒的）動きを埋め込まれた機械
(２)人間の情緒を自律神経，言語，表現，動作等から読み取るこ
とができる機械
(３)人間の情緒的欲求を満たす機械
この分類を次のように解釈することができる．まず，分類(１)に
関しては，機械の設計者や製作者が情緒的であると考えた動作を表
現させる機械とする．この場合，この機械から作用を受ける人間
が，機械の動作を設計者が意図した情緒表現と同じように認識した
かどうかで，この機械は評価を受ける．設計者は，心理学などで作
られた情緒モデルを工学的にアルゴリズム化して機械に埋め込むこ
とにより，この種の機械を実現することができる．しかし，情緒モ
デルは多様にあり，その正しさを議論することは難しい．また，複
数の人間（被験者たち）からの認識率によって評価しようとして
も，属性（国籍，文化，経験などさまざまな性質）が異なる被験者
一人ひとりの主観に依存し，解釈や評価の仕方が異なるため，統計
的な評価しかできない．
この分類では，エンタテイメント，案内・説明などを目的に，顔
の表情，身振り・手振りなどによって情緒表現を行うロボットが研
究開発され，実用化もなされている．
次に分類(２)に関しては，設計者が心理学などで作られた情緒モ
デルから情緒を分類したり，ある種のパターンとしてモデル化した
りすることで，機械と相互作用する人間からの情報を接触や非接触
のセンサにより得て，人間の情緒を認識しようとする機械である．
この場合，機械に作用する人間が表現した情緒が，機械によって認
識されたかどうかが評価を左右する．人間が情緒として申告する表
現に一般性はなく，人間の属性に大きく依存しているため，機械の
認識率を統計的にしか評価することはできない．
この分類では，人間の情緒をモニタする方法として，脳波，脳血
流，筋電，心電などを接触式のセンシングで計測したり，ビジョン
により人間の瞬き，顔の表情，身振り，手振りや，マイクロホンに
よる音などの非接触のセンシングで情緒を認識しようとしたりする
センサシステムの研究開発が行われている．そこから人間の快・不
快などを判断し，その結果に基づき機械の制御パラメータなどを変
えることによって，人間の情緒を考慮した機械の制御を行う．
分類(３)に関しては，上記の分類(１)と(２)の要素を含めた機械
である．人間と共存し，短期・長期的に相互作用することにより互
いに刺激しつつ，機械が人の情緒に合わせたり機械が情緒的に動作
したりすることによって，その人間の情緒を刺激し，人間の心理に
働きかけ，快・不快などの気持ちを引き起こす．人間の情緒的欲求
を満たすことは，機械が人間から主観的に高い評価を受けることで
ある．そのため，万人を満足させる普遍的な情緒的動作を作り出す
ことは難しいが，相互作用を通して個人の特性に合わせた機械の動
作を生成することにより，人間の情緒的欲求にこたえることが期待
される．
には人間が制御ループに入っているシステムであり，また機械から
観測すると環境の中に非線形で不確かな人間を含む制御といえる．
システムの設計が非常に重要である．
