# 9・4・1 補聴器

> **章**: 第9章 支援機器
> **節**: 9・4 聴覚言語障害者用機器

補聴器(hearing aid)は難聴者を対象とした福祉機器として最も知名度の高い補装具の一つであるが、その実態は意外に知られていない。工業製品としての補聴器は、JIS C 5512:2000により規定されており、国際規格ではIEC 60118-0~14に規定されている。補聴器の最も基本的な機能は音を増幅して出力するという単純なものであるが、実際の製品となるためには多くのハードルが存在する。補聴器は、原則として1本(個)の電池で動作させるために電源電圧が制限され、その条件下で十分な増幅度と出力音圧を確保し、長時間の動作をする必要があるため回路設計は決してやさしいものではない。また、音響障害を防ぐための出力制限回路や、聴覚の特性に合わせるための周波数特性の調整回路を内蔵するのが一般的である。そして、これらすべての構成部品を耳の穴に収まるサイズにしなくてはならないのである。形状から見た補聴器の種類は、箱形、耳掛け形、耳穴形(図9・12)に分類でき、国内で出荷された補聴器の数は46万台(2006年、全国補聴器メーカー協議会)ほどである。その中では耳穴形補聴器の出荷台数が最も多く、全体の54%程度を占めている。内部の処理回路で分類すると、アナログ式、プログラマブル(ハイブリッド)式、ディジタル式の3種類に大別できる。アナログ式補聴器は、音の入口であるマイクロホンから出力となるレシーバ(イヤホン)までアナログ回路で構成され、真空管、トランジスタ、ICと変わっているものの、開発当初から受け継がれてきた方式である。プログラマブル式は、増幅回路はアナログのままであるが音質などの調整部分に従来使われてきたトリマ(半固定抵抗器)を半導体に置換えてディジタル制御(設定)できるようにした補聴器であり、外部の専用調整装置と接続して設定を行うものである。トリマがなくなることで、小型化のニーズに対応できるようになり、また故障(b) ディジタル式耳穴形補聴器(a) クリアケースに収めた耳掛け形補聴器9 ―153原因の多くがトリマであったことから、故障によるクレームも減っている。しかし、専用の設定装置が必要となるために、対応できる販売店や医療施設が限られてしまうという問題もある。ディジタル式はマイクロホンやレシーバ周辺部分以外をディジタル化し、周波数特性の調整などをディジタル信号処理によって実現している。ディジタル化することで細かな周波数特性が設定可能となり、設定状態の再現性も高くなった。また、補聴器の小型化によりマイクロホンとレシーバとの距離が短くなるために発生しやすくなるハウリングを抑制するための処理も、安定した動作が実現されたことで実用化されている。今後は、雑音抑制処理、音声強調処理などの実用化が期待される。なお、ディジタル式補聴器(digital hearing aid)が出荷台数に占める割合は76%程度(2006年、全国補聴器メーカー協議会)である。特殊な補聴器として、FM 補聴器(frequency transpositionalhearing aid)と呼ばれているものがある。これは、FM 受信回路を内蔵した補聴器であり、耳掛け形のサイズで短い受信アンテナが装着されたものが市販されている。補聴器のマイクロホンを経由した話者の声には部屋の反響や周囲の雑音が付加され、難聴者にとって大変聞きづらいものとなってしまうが、FM 補聴器に対応するワイヤレスマイクを話者(学校では授業を行う教師)が持って発話することで、声を直接伝えることができ、SN 比を大幅に改善することができる。ここで使用されている電波は75MHz帯に割り当てられた専用の周波数および169MHz帯の周波数で、変調方式はFM(周波数変調)が使用されている。このシステムは特定小電力無線局の一種として位置付けられ、補聴補助用ラジオマイク用無線設備としての規格が定められている。補聴器は一般の家庭用電気機器とは異なり、使用する個人ごとに異なる聴覚の状態に合わせて調整する必要がある。この補聴器を特定個人に適合することをフィッティングといい、医療機関で十分な耳の検査をしたうえで行うのが原則である。基本的なフィッティングの考え方は、低下した聴力しきい値をもとにして周波数帯域ごとの増幅特性を難聴者の生活環境を考慮しながら決定する。実際には何度か実生活内で試聴したうえで設定状態を確定するものである。しかし、多くの眼鏡店で補聴器が販売されていることもあって、眼鏡の調整と補聴器の調整が同様なものとして考えられているのが現実である。眼鏡は水晶体の物理的なひずみを補正するのに対して、補聴器は聴神経機能の低下、つまり感覚を補正するものであり、両者は全く違うものといえる。補聴器の装用状態は他覚的な検査や評価をしにくいものであるため、熟練した担当者が実施することが望ましい。
