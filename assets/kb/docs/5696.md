# 13・3・4 情緒工学を応用した機械

> **章**: 第13章 感性・情緒工学
> **節**: 13・3 情緒工学

13･3･2項(１)に分類されたように，明示的な情緒のモデルを埋
め込むことにより人間に対して情緒を表現しようとする機械につい
ては，さまざまなものが研究開発されており，実用化されている．
例えば，アミューズメントパークにおけるロボットなどがある．ま
た，13･3･2項(２)に分類された，人間の情緒をさまざまなセンサ
から読み取ることによって人間の状態に合わせた動作をしようとす
る機械の研究開発も行われている．例えば，人間の心拍数や皮膚電
位を計測したりビジョンにより顔の表情を認識したりすることで人
間の情動の状態を観測し，それに合わせて機械の挙動を制御しよう
とする機器の研究開発がある．
ここでは，13･3･2項(３)に分類される，人間との相互作用によ
り人間の情動に働きかけて楽しみや安らぎを提供することを目的と
する機械として，メンタルコミットロボットを紹介する．
メンタルコミットロボットは，人間の情動に効果的に働きかける
ために，人間がもともともつ経験や知識を利用している．つまり，
相互作用によって人間の感覚を刺激してさまざまな連想を引き起こ
し，ロボットに関する解釈や記憶だけではなく過去の経験などを想
起させることにより，ロボットがもつ機能以上に人間の情動を刺激
することを意図した機械である．
人間の情緒的欲求を満たすためには，人間の情緒を刺激し，快・
不快などの気持ちを引き起こすことが重要であるが，ロボットの見
た目による刺激だけではなく，身体的なふれあいやコミュニケーシ
ョンなど人間のさまざまな感覚を適切に刺激することが重要であ
る．
人間の感覚は五官（感）と呼ばれるが，心理学では視覚，聴覚，
嗅覚，味覚，皮膚感覚，運動感覚（自己受容感覚），平衡感覚，内
臓感覚の8系統に分類される．皮膚感覚はさらに触覚，圧覚，温
覚，冷覚，痛覚に分けられる ．感覚の種類とそれに即した体験
内容は感覚モダリティである．例えば，視覚と皮膚感覚はモードの
違いであり，視覚における赤，青，黄色などの色彩の違いは同一モ
ードの中での質の差である．
知覚については，人間は感覚からの刺激を受けると脳の中枢神経
系がその刺激による連想と推論から意味付けを行っている．例え
ば，コンピュータ内部の仮想動物と人との相互作用では，主体であ
る人の視覚と聴覚を刺激するだけであり，身体的なふれあいはな
い．そのため，人に対する刺激の質の違いとして情報量が多くて
も，人の知覚におけるモード数は少ない．一方，身体をもつロボッ
トと人間との相互作用においては，実世界の中で人間の視覚，聴
覚，皮膚感覚，運動感覚，平衡感覚などを多様に刺激する．したが
って，身体的な相互作用はモードを多様化する．
そこで，ロボットの見た目の動きや音（言語表現も含む）による
非接触の刺激に加え，触ったり抱きかかえたりするなどの接触によ
る刺激を考慮したロボットの設計が重要となる．つまり，ロボット
の構造も人間の感覚に刺激を与える重要な機能なのである．
ところで，短期的な相互作用では，主体である人間が以前から有
している知識や経験に大きく依存してロボットの動作を解釈した
り，自らの価値観に則してロボットを評価したりする．そのため，
特にロボットの形態からの連想は大きな影響をもつ．ロボットの形
態に関しては，次の四つのカテゴリに分類できる．
