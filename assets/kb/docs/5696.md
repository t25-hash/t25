# 9・4・2 人工喉頭

> **章**: 第9章 支援機器
> **節**: 9・4 聴覚言語障害者用機器

人間が母音などを発声するための音源は、喉頭部分にある声帯と呼ばれるひだが肺から送られた空気の流れによって振動することで生じる声帯原音である。この振動によって、三角波状の波形をもつ音波が発生し、声道と呼ばれる共鳴空を通過することで母音などの有声音声が放出される。そして、音質の微妙な調節は口の開き方や舌の位置などで行うのである。この声帯原音の基本周波数は一般的にいわれる声の高さに対応している。しかしながら、病気などで喉頭摘出手術を受け音源となる声帯をなくしてしまうと、声を出せないことになる。人工喉頭(artificial larynx)は、そのような者を対象とした補装具であり、有声音発声時の音源となる声帯振動の代わりとなる音源を提供するものである。人工喉頭には笛式と電動式(図9・13)がある。前者は笛を内蔵した管の先を気管口につけて肺からの呼吸気で笛を鳴らし、反対側の管から口腔内に音を導いて音源とするものである。熟練すると、ある程度は声の高さを変えられることから、比較的自然な発声が可能となる。しかしながら、この方式では管を口にくわえた状態で発声しなくてはならず、発話に不都合な場合も多い。後者は、電池で振動子を駆動するタイプで、前頸部に押し当てて音源とするものである。一般に振動周波数を変化させられる構造にはなっているが、図9・12 一般的な補聴器の例9 ―154医療・福祉・バイオ機器(b) 電動式(a) 笛式図9・13 人工喉頭の例発話しながら操作することは難しく、単調で機械的な音声となってしまう。最近では、両者のよい点を組合わせて、気管口からの声の高さに関わる情報をもとに振動周波数を変化させる電動式人工喉頭が市販されている。しかしながら、現実的には人工喉頭を持ち歩くことは不便であることから、特別な道具を使用しないで発声する食道発声と呼ばれる方法が多く用いられている。これは、胃から空気を吐き出す際に食道入口部付近を振動させて音源とするものである。この方法では特別な道具を持ち歩く必要がないという利点があるものの、習得までにかなりの時間を要することや体力を必要とするために、すべての対象者に適用できる方法ではない。なお、人工喉頭は補装具として交付される対象であるが、JIS などの工業規格はない。
