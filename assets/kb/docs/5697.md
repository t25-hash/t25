# 9・4・3 手話(sign language)

> **章**: 第9章 支援機器
> **節**: 9・4 聴覚言語障害者用機器

聴覚障害者は、音声と聴覚による情報授受の代行手段の一つとして手話を用いて周りの人とのコミュニケーションを図っている。手話は、三次元空間上で複数の調動器官による運動と提示の継時変化として語が形成される空間的言語である。受信者にとって手話は、視覚系で受容される視覚言語である。このため手話は、線条性をもつ音声言語とは異なった言語的特性(独立した文法体系)をもつ。手話の調動は、手指信号(手指動作、manual signals)および非手指信号(非手指動作、non-manual signals)により構成される。手指信号は、手の型、掌方向、提示位置、大局的な運動により構成され、これらの動きを時間軸上に同時あるいは連続的に提示することにより、おもに語の形成に寄与する。非手指信号は、表情、口型、うなずき、視線などの動作で構成され、おもに統語論的、意味論的な作用があるといわれている。手話は話し言葉と同様に対話型自然言語であるため、効率的な認識や生成装置を構成するためには、その言語的特性を理解しておくことが必要である。a. 相互翻訳システム手話の研究・開発の目標の一つは、日本語と手話の相互翻訳システムの構築である。翻訳システムを構成するためには、一般に次の三つのステップを要する。(1) 入力手話を認識するには、言語的特徴の抽出を考慮してコンピュータ処理可能なデータ形式で手話を入力(トラッキング)しなければならない。入力方法には、Cyber Glove윣윃(サイバーグローブ)に代表される装着型装置を用いる方法と、カメラを用いて手話者の調動を撮影しその画像をもとに調動を認識する非装着的手法とに大別できる。装着型装置による計測方法は、データの入力精度およびコンピュータ処理のしやすさから、コンピュータ向きのインタフェースといえる。一方、非装着なカメラモデルによる方法は、手話の調動を妨げることがなく、手話者に色マーカをつけるなど他の弱い拘束の手法と組合わせることにより有望な手法となりうる。現状では、コンピュータ側の処理が複雑となり、十分な精度を期待できない。(2) 認識自然な手話動作を高精度かつ効率的に認識するためには、単語部図9・14 試験運用された情報提供端末の概観(提供:新情報処理開発機構マルチモーダル機能日立研究室)分と単語から次の単語への遷移動作(渡り)とを分離するセグメンテーション(segmentation)を行う必要がある。セグメンテーションは、対話言語の特性を考慮して行うことが可能である。分離された語の動作に対しては、図形や音声認識に用いられるパターンマッチング、DP(dynamic programming)、HMM(hidden Mar-kov model)などにより認識を行う。しかし、顔の表情などに代表される非手指信号の入力および認識に関しては、いまだ十分な成果を上げていない。(3) 生成翻訳システムでは、認識した結果をもとに対話者とコミュニケーションをとらなければならない。コミュニケーションのためには、手話の調動を何らかの方法で生成し表示しなければならない。調動の生成は、規則合成(synthesis by rule)による方法とモーションキャプチャ(motion capture)による方法に大別できる。規則合成では、手話の開始・変化・終了位置の各指関節角度と手の三次元的位置のみを記号化して登録しておき、逆運動学により動きを生成する。規則合成による方法では、単語生成のためのデータ容量が少なくて済む。また、語形変化などに柔軟な対応が可能となる。しかし、この手法では調動を自動生成するため、描画されたアニメーションが機械的な動作になりやすい。モーションキャプチャによる手法では、人間の調動に近い手話動作を期待できる。しかし、この方式では、語形変化への対応や単語登録の手間などの問題がある。b. 情報提供端末翻訳システムの応用例として、新情報処理開発機構マルチモーダル機能日立研究室では、手話による入出力インタフェースを備えた施設のバリアフリー情報を提供する端末を開発した。図9・14に試験運用された情報提供端末の概観を示す。手話の入力には、Cyber Glove윣윃による方式を採用している。手話認識では、辞書登録されている単語データの動作記述と入力された未知データとの類似度を判定する手法を開発し用いている。生成では、登録単語データを結合して手話文の調動を生成している。c. 対話システム郵政省通信総合研究所[現・(独)情報通信研究機構]、松下電器産業(株)、工学院大学らは、郵便局の窓口を想定した手話と日本語との対話システムを試作した。図9・15に試作された対話システムの概観を示す。手話の入力には、ステレオカメラモデルと、弱い拘束として青色手袋による非装着方式を採用している。認識では、大局的動作特徴による候補単語絞り込みを行い、局所特徴により認識する手法を開発した。生成では、登録単語データを結合して調動を生成している。単語データは、光学式と手袋型センサの併用によるモーションキャプチャ方式で取得している。d. 電子化辞書手話の応用研究の柱となるべきものの一つとして、電子化辞書の構築が挙げられる。電子化辞書の構築は、手話図9・15 試作された対話システムの概観[提供:通信総合研究所(現・情報通信研究機構)]学の進展と記述法の開発とともに近年盛んに試みられるようになっている。手話の表示には、実映像によるものとアニメーションによる方式がある。電子化辞書の応用では、教育システムの構築や、アニメーション生成と組合わせることによるネットワーク通信などが検討されている。今後、さまざまな分野から研究・開発が進められ、聴覚障害者のコミュニケーションギャップが軽減されることが期待されている。
