# 9・4・4 字幕生成

> **章**: 第9章 支援機器
> **節**: 9・4 聴覚言語障害者用機器

難聴者に対する情報補償の手段として、音声による情報を文字として表示するものがある。この字幕はキャプチャといわれ、オープンキャプチャとクローズドキャプチャに分けられる。前者は、外国語音声による映画やドラマ、ニュースのテロップなど映像の一部として表示されるもので、すべての視聴者が見られるものである。これに対し後者は、選択的に画面に文字を表示させるもので、画像や音声信号とともに送られてきた文字コードを専用の回路で文字化して画面に合成表示するものである。一般には、字幕放送(closedsubtitles on TV)というと後者を意味する。字幕放送を行うためには、何らかの手段によって文字情報を作成しなくてはならないが、30分程度のドラマで20000文字程度になるとされており、作成作業には多くの時間と経費がかかる。しかし、難聴者はもちろんのこと、高齢者にとっても大変有用な情報提供手段であると考えられる。テレビ受像機の画面に文字を表示する場合、本来の画面が見にくくならないことや文字がきちんと認識できることが基本である。これらを満たす条件として、現在では2行30文字で1分間に10面以下という原則で行われており、日本語の文字がアルファベットに比べて複雑なために、欧米における字幕の文字数に比べて少なくなっている。表示される文字自体に関する問題とは別に、表示する内容についても多くの問題がある。実際に放送される音声をすべて文字にすると情報量が多すぎて字幕を読みきれないような場合もある。そこで、内容を要約して表示する必要があるうえに音楽などで表現されている状況を説明する必要もでてくるため、多くの作業時間と経費を必要とすることになる。字幕の表示内容を要約する、発話内容を入力するといった作業は、リアルタイムで行うにはかなりの無理があるように思われる。しかしながら、難聴者のニーズとしてニュース番組などへの字幕挿入の希望と期待は大きい。リアルタイムを実現するために、アメリカでは法廷などで使われていた特殊な速記用キーボード(ステノグラフ社製特殊タイプライタ等)を使用して生放送にも対応してきた。そして最近では音声認識の技術が進歩したために、音声入力が使われるようになってきた。しかし、認識精度を高めるために発話者を特定し、発音にも正確さが求められるために、専任のオペレータが聞き取った音声を再度発話して音声入力する必要がある。アメリカでは三大ネットワークでの字幕放送(図9・16)の時間は90%までに達しているといわれているが、日本ではNHK が全放送時間の43%、民放キー局では30%程度(2006年度)であると9 ―155図9・16 アメリカの放送での字幕例(下方の3行が字幕)されている。この違いは、ADA(Americans with disabilitiesact、障害をもつアメリカ人のための法律)を基本とした障害のある人に対する補償に関する法的整備がなされており、公共施設での購入物品に対して義務付けられた条件によるところである。それらの中には、テレビ受像機に字幕表示機能を内蔵させることも含まれており、結果的に市販されているテレビ受像機には字幕表示機能が標準で内蔵されていることとなった。もちろん、日本製のテレビ受像機もその対象であるが、日本国内では(コスト削減のため)この機能は除外されている。これは日本国内では法的には努力目標であり、字幕放送自体も少ないことが大きな原因であると考えられる。しかし、加齢に伴って多くの高齢者は軽度の難聴となりうることから、字幕表示は音声が全く聞き取れない人以外にとっても有用な機能であると思われる。この字幕表示と同様な情報補償手段として、手話通訳者を表示する場合がある。しかし、一般的に考えられているほど手話が理解できる聴覚障害者は多くはなく、全体の10%程度であるといわれている。また、小さな手話通訳画面(sign language on TV)に集中すると本来の画面を見られない、手話を必要としない人にとっては画面全体が見づらくなるなどの問題もある。そのような状況もあり、より多くの難聴者、高齢者に対する情報補償を実現するために字幕表示は大変重要な手段であるとされている。なお、字幕放送表示アダプタや字幕放送表示機能内蔵テレビ受像機を聴覚障害者が購入する際には、障害等級にかかわらず、その一部または全額が補助されることになっている。
