# 9・5・2 手動車いす

> **章**: 第9章 支援機器
> **節**: 9・5 移動用機器

手動車いす(manually propelled wheelchair)は、9・5・3項で後述する電動車いす等と比べ価格も比較的低いこともあり、各種車いすの中で最も普及している。手動車いすは、おもに自走式と介助式に大別される。その構造、寸法、外観、試験方法等については、JIS による規格(JIS T 9201:2006)が存在する。自走式の一般的な構造を図9・19に示す。自走式は、手でハンドリム(hand rim)を押して動力をタイヤに伝えることにより操作する。介助式には自走用のハンドリムは付いておらず、比較的小さい後輪が付いている。なお、図9・19のような自走式車いすは介助用としても用いることができる。車いすのフレームは通常軽量なパイプで造られ、材質は近年ではアルミニウム合金が用いられることが多い。また、保管場所などの面から、折り畳み式の製品が多い。座面や背もたれについては、折り畳みが可能なように布張りのものが多い。手動車いすについては、上述したような標準型だけでなく、背もたれが倒れるリクライニング型や座る部分の高さが低い低床型など、非常に多くの種類が市販されている。また、座面の布等、素材の改良により座り心地がかなり改善されている。金属にはない温かみを出すために、素材に木を用いた商品も開発されている。バスケットボールやマラソン等の競技専用に用いられるスポーツ車いすも、多くの種類が市販されている。手動車いすの問題点としては、リフトの際と同様にベッド等からの移乗が困難なこと、段差を登るのが困難なことなどが挙げられる。また、長時間たわんだ布張りの座面に座ることにより、骨盤の後傾等の変形が生じることが指摘されている。
