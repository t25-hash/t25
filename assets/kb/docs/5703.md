# 13・2・3 その他の惑星に関する話題

> **章**: 第13章 月・火星などへの展開
> **節**: 13・2 惑星に関する最近の話題

･外惑星（木星，土星，天王星，海王星）にはリングが存在する．
･木星の衛星イオには火山が存在する（ボイジャーによる観測)．
･木星の衛星イオから溶岩の吹上げ，巨大なカルデラとクレータが
とらえられた（木星探査機ガリレオのカメラ，赤外線マッピング
スペクトロメータ)．
･土星の衛星タイタンには濃い大気（約2atm）が存在する．
･土星の衛星テレスト（L4），テティス，カリプソ（L5）はラグ
ランジュ点に位置する．
･冥王星の質量は1948年にバビネットが地球質量の12倍と推定し
て以来，減少の一途をたどっている（実は彗星ではないかといわ
れている)．
･木星の衛星エウロパの表面の氷の下に水の海がある（ガリレオの
磁気観測データ)．
･木星の衛星カリストとガニメデにも水が存在する可能性が出てき
た．
･第10番惑星発見：セドナ（Sedna，2003VB12）．冥王星の2分
･冥王星は海王星の外側を回る太陽系の第9番惑星とされ，唯一探
識が得られることが期待できる．
