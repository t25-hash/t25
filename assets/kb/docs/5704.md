# 9・6・1 障害者用入力装置

> **章**: 第9章 支援機器
> **節**: 9・6 コミュニケーション機器

障害者用入力装置としてさまざまなスイッチが市販されている。しかしながら、どの操作スイッチを選択したらよいのかは、使用者の残存機能ばかりでなく、使用目的、使用環境、使用頻度、介助者の状況、他の生活行為との関連など、障害が重度であればあるほど考えなければならないことが増え、適切な検討や試行錯誤が必要となる場合も多い。ここでは障害者用入力装置を、a.腕や足や頭や息などの大まかな動きで操作できる入力装置、b.指先、唇、まぶたなどのわずかな動きで操作できる入力装置、c.脳波や脳血流などの身体的動きを伴わないで操作できる入力装置に分けて紹介する。a. 大まかな動きで操作できる入力装置脳性麻痺による不随意運動などに対応した比較的大型の操作端であり、足や腕での操作の場合は作動力を大きくしたり機械的な遊びを大きくする。図9・26は作動力の大きい押スイッチのベッド柵への設置例である。図b. わずかな動きで操作できる入力装置高位頸髄損傷、筋ジストロフィー症など多くの疾病では、操作力や可動域の減少を伴う。わずかな動きを検出するためには、操作スイッチを可動部分のそばの身体部位に固定し、固定位置と可動部の変化分を検出する方法が多い。この方法であれば体全体が動いても検出に影響を与えないので、姿勢が変化したり不随意運動が出現する場合にも有効である。図9・28は指先の動きの検出例、図9・29は赤外線近接センサによるまばたきスイッチである。c. 身体的動きを伴わないで操作できる入力装置難病の一つである筋萎縮性側索硬化症は、進行とともに四肢麻痺や呼吸麻痺を図9・28 指先スイッチ図9・29まばたきスイッチ[提供:パシフィックサプライ(株)、ファイバースイッチ]呈する。さらに症状が進行すると表情筋や眼筋にも麻痺を生じ、身体のどの部分も動かすことができなくなる。しかしながらその感覚機能は残存しているといわれ、そのような人を対象として特定の脳波の出現や脳内の血流変化などによってイエス・ノーの判断やコミュニケーションの拡大を図るものである。脳波(β波)の検出システムであるマクトスModel Dx が市販されている。
