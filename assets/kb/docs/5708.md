# 9・7・1 排泄用機器

> **章**: 第9章 支援機器
> **節**: 9・7 排泄用機器と入浴用機器

人の排泄動作は,尿意や便意,括約筋のコントロールなどの神経系の能力,衣服の脱着や後始末動作などに必要な体幹バランス機能や上肢機能,トイレまでの移動や便器への移乗に使う下肢や体幹バランス機能などの組合せで行われる.これらの能力のどこに障害があるかで,使用される支援機器が異なる.排泄支援機器(toiletaid)は,排泄に伴う移動や移乗,衣服の脱着などの関連動作を支援する機器,排泄に伴う上肢動作を支援する機器,身体の排泄機能を支援する機器などに分けられ,脊髄損傷者などは一般に市販されている排泄関連機器や住宅設備機器,福祉用具,排泄方法に合わせた機器などを生活方法に合わせて選択し生活している.排泄にかかわる動作は設計者にとって見えにくい部分であるため,開発が遅れているのが現状である.ここでは,排泄行為を排便と排尿に分けて,排泄支援機器の考え方とその例を記述する.a. 排便用機器座薬挿入器,浣腸挿入器,洗腸器など排便を促すための薬剤などを肛門に挿入するための支援機器,洋式便器への移乗を支援する機器,便器周辺機器,身体装着用機器などがある.. 座薬挿入器座薬は小さく,体温で溶融するため,手指の巧緻性の乏しい者は自立して挿入することが難しい.この挿入器は,指が全く動かない者でも図9・38に示す姿勢ができ,挿入器の先端を肛門に当てて押し出すことができれば,座薬挿入を自ら行え9 ―161(b) リーチャタイプ(c) てこ挿入タイプ(d) 指装着タイプ図9・38 座薬挿入器のタイプと挿入姿勢(a) 挿入前(b) 挿入後図9・39座薬挿入器での座薬挿入イメージる(図9・39).. 浣腸挿入器肛門に触覚があれば,図9・38(c)のてこ挿入タイプと同様の姿勢で挿入器先端を挿入し,浣腸の容器を押して直腸内に液を挿入する浣腸挿入器もある.. 洗腸器体温程度の温水を直腸内に挿入して排便を促すもので,洋式便器で図9・40の姿勢で使用する.. 昇降便座洋式便器への立座りを補助し,転倒防止と自立性を高めるために使用する(図9・41).9 ―162医療・福祉・バイオ機器図9・40洗腸器図9・41 昇降便座図9・42 トイレッタブル. 便器への移乗台四肢に障害のある者の車いすと便器間の移乗を補助し,転落防止と自立性を高め,トイレッタブルという台上での衣服の脱着や便器への移乗動作の自立性も向上させる(図. 臀部洗浄器排便後の臀部を洗浄するために湯を噴射する機能をもつ便座や携帯用洗浄器などがある.. ストーマ(stoma)用品人工肛門に装着する用具で,清潔・安全,無臭的に便を処理することができる.b. 排尿用機器身体装着型収尿器,導尿用カテーテル,留置カテーテル,手持ち式収尿器,排尿用の設備機器などがある.. 身体装着型収尿器尿を漏らさずに集めることを目的に作られたもので,男性用は多く使用されているが,女性用は実用に堪えるものがないのが現状である.コンドーム型のような陰茎固定型やベルト固定型,パンツ固定型などがある.. 導尿用カテーテル尿意を感じたときや排尿時間ごとに,カテーテルを尿道口から挿入し尿を出す導尿に用いる.頸髄損傷者のような四肢麻痺で手指が任意に動かない者でも,手関節まで動けば図9・43のような導尿セットを用いて導尿動作を自立して行うことができる.自分でできない場合は,介助者が行うことになる.カテーテルを消毒容器に入れて携帯するタイプと,毎回使用後に破棄する使い捨てのタイプがある.. 留置カテーテル尿道口から挿入し,バルーンと呼ばれる袋に生理食塩水を入れて膨らませ外れにくくして,尿を体外に出すカテーテルのこと.. 手持ち式収尿器しびんのように受尿部と蓄尿部が一体となったタイプと,受尿部と蓄尿部をチューブで連結したタイプがある.後者では,受尿部のセンサで尿を感知し吸引するものもある.受尿部は男性用と女性用がある.. 設備機器一般の小便器などの設置方法を変えることで,自己導尿を自立させた男性用の例を図9・44に,女性用の自己導尿設備を図9・45に示す.市販の幅の狭い小便器などを図9・44の(a) 男性用シリコーンカテーテル(b) 女性用シリコーンカテーテル(上からケース,頸髄損傷者用,手指が使える人用)図9・43 自己導尿セット図9・44 自己導尿用小便器図9・45 女性用の自己導尿設備ように設置することで,自立性を向上し,排尿にかかる時間を軽減させることができる.
