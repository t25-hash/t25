# 13・4・2 火星（Mars ）利用の将来構想

> **章**: 第13章 月・火星などへの展開
> **節**: 13・4 惑星利用の将来構想

a.現状と将来 火星は太陽系の成り立ちだけではなく生命の
誕生と進化に関わる科学探査の対象として注目されている．21世
紀初頭の現在，火星に人類を送り込んでさまざまな探査を行うこと
（有人火星探査）が当面の目標となっており，そのための予備調査
としてすでにいくつかの無人探査機（自走式のローバなど）が火星
表面に送り込まれ，さまざまなデータの収集が行われている．
火星の大きさは地球のほぼ半分で，表面重力は地上の約3分の1
である．表面温度は130～300Kで自転速度や自転軸傾斜角は地球
とほぼ同等である．わずかながら二酸化炭素を主成分とする大気が
存在し，水の存在も示唆されている．ただし，大気が希薄で磁気圏
がないため，火星表面には宇宙線が降り注いでいる．このように火
星表面はそこで人間が活動するためには非常に過酷な環境である
が，月や他の惑星表面の環境と比較するとはるかに地球環境に近
い．そのため，将来の人類の移住先として取り上げられることが多
い．さらに遠い将来には火星の大気構造を操作して温暖化させ，地
球と同等の環境を作り上げる（テラフォーミング，terra form-
ing）構想も提案されている．
火星の利用形態は，以下のように基本的には月の場合とほぼ同様
である．
(１)位置の利用
太陽系探査のための前哨基地
地球からの遠さを生かした商業利用（観光）
(２)火星環境（希薄大気，低重力，低温，低ノイズなど）の利用
火星天文台（火星からの観測）
火星内部の科学探査
材料などの新技術開発研究拠点
b.火星利用の形態
.位置の利用
(１)太陽系探査のための前哨基地
火星の脱出速度は約5.0km/ sであり，火星から物資や探査機を
打ち上げるために必要なエネルギーは，地球から打ち上げる場合の
約5分の1である．そのため，太陽系探査の中心基地として火星は
有効に利用できる．
(２)観光
旅行という観点からは火星は少し遠すぎるが，地球以外の惑星に
降り立って景色を見たいという願望をもつ人は多いかもしれない．
このような火星観光を実現するためには，物資や人を運ぶための巨
大な宇宙機が必要となる．直接人間が火星まで行くことなく，環境
センサ（カメラ，マイク，温度計など）のみを火星表面の数箇所に
設置し，地上からその様子をモニタして楽しむ仮想観光なども考え
られている．
.火星環境の利用
(１)火星天文台
火星にはわずかながら大気が存在し，砂嵐なども発生している．
その表面に天文台を建設することはあまり得策ではないかもしれな
い．ただし，VLBI （very long baseline interferometry，超長基
線干渉計）観測の拠点として火星を利用することによって，より精
度の高い位置計測ができる可能性がある．
(２)火星内部の科学探査
月の場合と同様に，太陽系の進化を明らかにするための科学探査
ならびに火星利用のための地形学，力学，地質学，鉱物学的な探査
が行われる．
(３)新技術開発研究拠点
火星の表面環境は月よりも地球環境に近いので，月の場合ほどそ
の環境を利用した技術開発は期待できないかもしれない．ただし，
火星基地の自律性は月面基地よりも高いと予想されるので，火星で
はより長期にわたる研究が可能となる．
.資源の利用 火星の鉱物資源に関してはまだ十分なデー
タが得られていないが，同じ太陽系に属していることから，鉱物学
的には月や地球と大きく変わらないことも予想される．したがっ
て，月の場合と同様の資源利用が可能（ヘリウム3を除く）と考え
られる．さらに炭素や水素を抽出すると，これらを用いた莫大な種
類の有用物質を使えるので，地球上と変わらないレベルの資源利用
が期待できる．
.精神世界としての利用 月の場合と同様に，火星に対す
る畏敬感や神秘性を具現化する手段の一つとして火星を利用するこ
ともありうる．
