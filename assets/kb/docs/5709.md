# 9・7・2 入浴用機器

> **章**: 第9章 支援機器
> **節**: 9・42 ).

移動や移乗が困難な障害者,高齢者にとって入浴動作は日常生活活動(ADL)の中で最も身体的負担が大きく,また事故の発生しやすい環境である.このため各種の入浴用機器(bathroom aid)が開発されているが,大きく分類すると自立型と介助型に分けられる.a. 自立型つえや手すりを使って歩行が可能なケース,または車いすで移動しているケースでは,自力での入浴が可能であるが,環境改善と補助器具の導入が必要である.まず,開閉が容易で開口部の広いドアと段差のない滑りにくい床,浴槽までの手すり,腰掛けて入りやすくした浴槽や椅子などが代表的な器具として使われている.しかし,最も入浴を困難にしているのが浴槽への出入りの動作である.そこで,浴槽内に上下昇降するリフトを設置することがある.バスブースタとも呼ばれるが,コンパクトで浴槽内への設置や取外しが容易に行える機種が多い(図9・46).昇降機構は油圧,水圧,空気圧,機械式,電動式があり,一般的なストロークは400~450mmである.b. 介助型障害が重度化すると介助負担が増加する.介護者の二次障害を防止するために補助器具の活用が勧められる.家庭用としては設置型のリフトがよく使われている.天井走行型は大規模な工事が必要だが,設置型は支柱を床に埋め込んで固定するか,天井の四方にはりを渡して工事なしで設置する方法がある(図9・47).浴室用リフト(bath lift)には支柱を水圧で昇降するタイプと,電動アクチュエータでアームを押し上げる方式がある.アームが2関節になっていると,狭い浴室での取回しが容易で,脱衣室からつり上げて浴槽に直接入れることもできる.昇降幅は900~1800図9・46 浴槽内据置型リフト図9・47 浴室用水圧式リフト図9・48 浴室用スライド式リフト[提供:TOTO(株)]図9・49食事用自助具9 ―163mmで,アームの支点は高さ調節が可能である.脱衣室から浴槽まで,椅子に座ったまま水平方向にスライドし,浴槽の中で昇降する入浴補助装置も開発されている(図9・48).施設向けに設計されている装置は,一般に特殊浴槽と呼ばれている.使用者をストレッチャやシャワーキャリーに乗せたまま,入浴装置の中に入れる方式が多い.ストレッチャを使うタイプは,人が乗る担架を昇降させる機種と,湯張りした浴槽を昇降させる機種がある.シャワーキャリーの場合は座位のまま浴槽に入れることになるので,浴槽のドアを開閉式として出入りを容易にしているが,そのたびに給湯と排水が必要になる.また,シャワーやミストサウナで入浴を代替する装置もある.座位または背臥位の姿勢で頭部を除いた全身を樹脂製のカバーで覆い,その中でシャワーやミストで身体を温めたり,シャワーで汗を流して洗体することができる.
