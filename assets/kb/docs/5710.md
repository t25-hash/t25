b.競技用水着（swim wear ) 泳者にとって最大の敵である
流水抵抗には，表面摩擦抵抗および形状抵抗があり，それぞれ抵抗
全体の1割および9割を占める．より速く泳ぐことに特化した競泳
水着では，これらの抵抗をいかに軽減するかが課題となる．
近年の競泳用水着は，このうちおもに表面摩擦抵抗を軽減するこ
とを目的として開発が進められてきた．水着表面付近の乱流境界層
の発生を抑えて摩擦抵抗を低減するために，水着表面の平滑化が行
われてきた．例えば，ポリエステル，ナイロン，ポリプロピレン等
の合成繊維から作られた極細糸によって表面粗さを低減し，さらに
撥水剤のプリント加工により摩擦抵抗を軽減している．また，生地
も薄いものでは0.35mmとなるため，軽量化も可能としている．
しかしながら，水着表面の平滑化だけによる摩擦抵抗低減には改良
の効果が見えにくくなってきたため，なかには図14･12に示すサメ
の皮膚状の微小な凹凸を利用し，整流効果をねらった水着や，水着
表面に取込みやすくした空気によって摩擦抵抗の低減を図った水着
が登場している．

c .アルペンスキー（Alpine ski ) スキーの性能を決定する
機械的特性には，剛性特性（曲げ，ねじり），慣性特性（質量，慣
性モーメント），形状特性，および振動減衰特性等がある．曲げ剛
性は，その剛性分布を変化させることによりスキー荷重時の雪面反
力分布を調整でき，エッジの雪面への食込み量および範囲を調整す
る．なお，ねじり剛性が大きいほど，エッジは雪面へ食い込みやす
くなる．慣性特性はスキーの質量分布により決定され，これにより
スキーの操作性や直進安定性が変化する．振動減衰特性では，滑走
中に発生するスキーの振動を抑制し，板がばたつくのを抑えるとと
もに，アイスバーンなどにおける高次振動によるエッジの外れを低
減する．
スキーの曲げ剛性によって変化する雪面反力の分布は，ターン技
が容易に実現できるようになった．
