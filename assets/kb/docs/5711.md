# 9・8・2 自立用ロボット

> **章**: 第9章 支援機器
> **節**: 9・53 )などがある.この万能ハンドルは,ばねのついた多数のピン

食事や整容などの自立を図るためのロボット(robot)は,欧米を中心に若年重度四肢麻痺者に使われ始めている.ロボットの使用によってあらゆることが自立できるようになるわけではないが,それまで一人ではできなかった作業がロボットを使うことによってできるようになるだけで使用者のQOLが高まり,精神面にもよい影響を与える.このようなロボットの基本的な考え方は自助具と同じであるが,自助具と呼ぶのはローテクノロジーによる機器だけであると考える人が多く,ロボットは自助具と呼ばないことが多い.代表的な自立用ロボット(assistive robots)にオランダ製のMANUS(図9・54)がある.ハンドの開閉を含めて8自由度をもつはん用マニピュレータで,通常は車いすに取付けて使用されるが,簡単に着脱もできる.価格は約350万円であり,発売以来2007年までの約15年でオランダ国内を中心に累計約300台が販売された.操作は4×4個のスイッチの組合せで行うが,ジョイスティックも使える.はん用で多機能な分だけ操作が少し難しく,意欲のある若年障害者であっても習得には練習が必要不可欠であり,高齢者には使いこなすことが難しく,高齢者の症例は報告されていないようである.実際の使用状況調査によれば,食事,歯磨き,ひげそり,軽量物の操作などさまざまな用途に使用されており,使用者にとってはこのロボットなしの生活など考えられなくなるほど毎日の生活に必要不可欠な道具になっているという.HANDY-1(図9・55)はイギリス製の食事用マニピュレータである.価格は約60万円であり,全世界で約300台が使われている.トレー上の7箇所に分けて置かれた食物を好きな順で食べることができる.スイッチ1個だけと簡単な操作で食事ができるため,重度障害児でもこれを使って食事が可能であるが,食べられるものは刻み食だけに限定されており,例えばごはんをこれで食べようとすると大きな塊が持ち上がってこぼしてしまう.オプションを使えば,歯磨き,ひげそり,化粧などもスイッチ一つでできるようになる.MANUS よりも機能は制限されるが,スイッチ一つで使いやすい図9・56 マイスプーン[提供:セコム(株)]ことと価格が安いことから,使用者はMANUS よりも多い.同様のロボットは日本でも開発されている.例えばセコム(株)のマイスプーン(図9・56)は,HANDY-1よりも多種多様の食事を食べられるよう設計されたマニピュレータである.HANDY-1がスプーンですくい上げるだけであるのに対して,マイスプーンではスプーンとフォークで挟み上げるため,大きな固形物や千切りにした食物なども扱うことができる.
