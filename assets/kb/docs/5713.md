a.レジスタンストレーニング（resistance training）機器
図14･31トーイング機器によるトレーニング
転車（自転車エルゴメータ）等は，能力や目的に応じた運動強度の
設定が可能であるとともに，天候に左右されず定期的に実施できる
点で優れている．これらの機器については，12･2節にその詳細が
記載されているので，ここでは割愛する．
c .スキルトレーニング（skill training）機器 身体動作は，
脳からの指令が中枢神経，末梢神経を通じて最終的に筋肉に達する
ことで開始され，その信号が筋紡錘など筋肉内の感覚器官による情
報とともに再度中枢神経にフィードバックされ動作の調整が行われ
る．この神経系の働きによる調整力を高めるスキルトレーニングを
行うことにより，正確さ，素早さが養われ，パフォーマンスを向上
させることができる．ここではスキルトレーニング機器の一例とし
て，陸上短距離種目におけるトーイング（towing）機器について
説明する ．
ランニングのスピードは，ストライド（歩幅）とピッチ（単位時
間内歩数）との積によって決まる．トーイング機器は，強制的に走
者をけん引することによって，この両者あるいはどちらかを向上さ
せ，スプリント能力を向上させるためのものである．その仕組みを
図14･31に示す．走者の腰部にベルトを装着し，それに取付けられ
たロープを電動モータでけん引する．モータはけん引力の大きさを
設定できるように，ロープ張力のフィードバック制御が行われる．
けん引力は30～60N程度である．これにより自力で到達可能な状
態を超えた水準に筋や神経をおき，それらを刺激することによっ
て，より高いパフォーマンスレベルに対する神経筋組織の適応を促
すことが期待できる．なお，けん引力に頼った受動的なランニング
では，ピッチを増加させることはできない．そのため，合理的なラ
ンニング動作を理解し，ピッチを速くするという意識を高くもって
トレーニングを行わなければ効果は期待できない．
