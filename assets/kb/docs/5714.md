# 14・4・1 スポーツ医学の基礎領域

> **章**: 第14章 スポーツ機器
> **節**: 14・4 スポーツ医学

スポーツ活動は，生体へさまざまな負荷を与える．図14･32に示
すように，生体は負荷が増すに従い適応していくが，あるラインを
超えると適応できずに破綻する．スポーツ医学の基礎領域では，こ
のプロセスを研究対象としている．どの程度の負荷を与えると生体
が破綻するかを知る指標を一般化して特定することは個体差もあり
難しいが，多面的なアプローチがなされている．
運動生理学領域は，スポーツ医学において重要な位置を占めてい
る．特に運動負荷に対する心肺機能の応答についての研究成果は，
競技力向上ばかりでなく健康の維持・増進を図る運動を考えるうえ
での科学的根拠を与えている．
スポーツバイオメカニクス領域は，運動を行う際の生体の動きを
おもに力学的な手法を用いて解析することにより，効率的な運動方
法や生体への力学的な負荷を明らかにする学問領域である．トップ
アスリートのパフォーマンス解析などにより，効率のよいフォーム
などに関する知見を与えてくれる．

スポーツに使用する用具の開発についても，スポーツバイオメカ
ニクスからのアプローチが欠かせない．なかでもシューズと競技サ
ーフェスについての研究は最近特に進んでおり，記録の更新などス
ポーツの進歩に貢献している．コンピュータサイエンスの進歩によ
り大きく展開した領域である．
スポーツ栄養学領域では，運動に必要となるエネルギー消費につ
いての知見をもとに，有効な栄養摂取方法等について研究する．競
技スポーツでは，スポーツに伴う疲労対策，そしてスポーツに必要
となる体作りを効率よく行うための栄養補給方法などがおもな研究
対象となる．マラソンなどの持久的運動におけるカーボローディン
グ方法は，スポーツ栄養学の成果としてよく知られているが，最近
ではサプリメントの功罪が論議されている．
スポーツ心理学領域は，スポーツ活動に伴う心理学的な変化を扱
うとともに，よいパフォーマンス成果をもたらすための心理学的な
アプローチ方法が研究対象となる．特に最近ではイメージトレーニ
ングなどのプログラムが開発されており，競技力向上の視点からそ
の重要性が高まっている．
