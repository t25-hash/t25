# 14・4・2 スポーツによる医学

> **章**: 第14章 スポーツ機器
> **節**: 14・4 スポーツ医学

スポーツによる医学は，種々の疾病に対する予防あるいは治療と
して運動プログラムを用いるものである．また，すでに確立されて
いるリハビリテーションもこのカテゴリに含めて考えられる．
最近では，スポーツ診療科を設けている医療施設も多い．しか
し，予防医学的運動療法は，現行の医療制度の中ではあまり評価さ
れていない．近年ようやく高血圧，糖尿病，高脂血症などに対する
運動療法が制度の中で位置付けられるようになるなど，この領域が
正当に評価される動きがでてきた．
a.運動療法（exercise therapy) 運動療法は，食事療法，薬
物療法と並んで糖尿病の治療法として，内科領域の中でも従来から
重視されてきた．最近では，高血圧や高脂血症などいわゆる生活習
慣病の治療手法の一つとして，その重要性が高まってきている．肥
満は，国際的にも課題とされる病態であるが，運動療法の効果が期
待されている領域である．
整形外科領域においては，骨粗鬆（こつそしょう）症（オステオ
ポローシス）などに対する運動療法の必要性が指摘されている．
b.リハビリテーション（rehabilitation) 内科領域では，心
筋梗塞後のリハビリテーションの効果について研究が進んでいる．
時間といった要因を適切に組込む．
以前は安静が主体であった心筋梗塞の治療に，この分野は新たな光
をもたらしている．整形外科領域では，種々のスポーツ外傷，特に
膝前十字靱帯損傷に関する手術法が進歩し，これに対応したリハビ
リテーションに関する研究が多くなされ，スポーツ医療の進歩に貢
献している．アスレチックリハビリテーションは，スポーツ医学の
領域として重要な位置を占めている．
脳血管疾患に対するリハビリテーションは，すでに領域として確
性も，転倒予防などの視点から注目されている．
立されていることが知られている．
