# 10・1・1 ジェロンテクノロジー(gerontechnology)の目標

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・1 概説

高齢者の生活行動支援に関するこの分野の最終的な目標は、自立寿命と生きがい寿命をできるだけ長く維持することである。自立寿命の考え方は、近ごろよく使われるようになってきた健康寿命の考え方と発想の点では同じである。すなわち、健康寿命重視の視点は、人間生活の幸せは単に寿命を長くすることではなく、苦痛の少ない、より健康な期間を長くすることだとする考え方である。それに対して自立寿命重視のコンセプトは、健康寿命の視点をさらに一歩進めたもので、加齢に伴い、たとえ健康状態が低下しても、自立性が保たれていれば生活に必要な機能や生きがいのための行動が可能になり、高齢者の豊かな生活が確保できるという考え方である。他方、生きがい寿命の視点は、高齢者が生きていることを楽しみ、満足した生活を送るためには、単に自立できるだけではなく、精神的に充実した生活を実現するための生活システムが必要であるというものである。この二つの寿命が確保されている限り、人は希望をもって人生を生き続けることができる。
