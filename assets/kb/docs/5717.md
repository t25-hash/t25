a.ゼロプレッシャ気球（zero- pressure balloon) 気球の底
部に浮力ガスの排気孔を設けた気球である．実際には，少し高い位
置に孔をあけ，ダクトを底部の位置まで垂らす．総重量を上回る浮
力（自由浮力，free lift ）を与えられて上昇する気球は，満膨張と
なった後には自由浮力分のガスを排気孔から排出し，上昇が止ま
る．このとき，気球底部の内圧と大気圧の差は零であり，そこから
気球上の任意の高さ における圧力差 は，大気密度と浮力ガ
スの密度の差をρ，重力の加速度を とすると，
＝ρ (14･1)
である．
この方式の気球は，日没で浮力ガスの温度が低下して浮力が減少
すると，高度が下がっても浮力は回復しない．一定高度を保つには
バラストを投下する必要がある．これを日没補償と呼ぶ．その量
は，中緯度ではおおよそ総浮力の7～10 %となる．このため，ペイ
ロード質量のほとんどすべてをバラストに充てても，1週間ほどの
飛翔が限界である．

b.ロードテープの導入 上記a項で述べた設計概念では，頭
部と底部は，経線方向の張力が集中して無限大となる特異点であ
る．この問題を解決するために，図14･1の右半分に示すように，
ロードテープと呼ぶ高張力繊維の入った補強テープを経線に沿って
等間隔に挿入する．ロードテープは頭部と底部に集中するので，特
異点問題は解消する．かつ底部につるされたペイロードの集中荷重
は，ロードテープを通じてフィルムの分散荷重へとスムーズに引き
渡される．
通常のゼロプレッシャ気球は，製作工程の容易さから，14･2･3･
a項で述べるように，平面状のフィルムの上に補強テープを取付け
た構造となる（平面ゴア法）．その結果，立体形状は図14･3(ａ)の
ように，横断面が多角形の形状となり，フィルムには二軸の張力が
発生し，静定問題として扱えなくなる．ロードテープによる補強効
果も不十分である．
c.三次元ゴア設計法（3- D gore design concept ) ロードテ
ープを挿入する効果を最大限に高める方策である三次元ゴア設計法
の概念を図14･3(ｂ)に示す ．フィルムは，ロードテープの間で
伸びによらずに周方向に小さな局所曲率半径をもった膨らみ（バル
ジ）を形成している．かつ，経線方向にはフィルムに余剰があり，
しわが発生しているとする．このような条件を作れば，フィルムの
張力は，余剰でしわのある経線方向には発生せず周方向のみとな
る．この条件を満たす形状設計は，a項で述べたロードテープがな
い場合と同一の静定問題となり，形状設計問題は統一的に理解でき
る．
