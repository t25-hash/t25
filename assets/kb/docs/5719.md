# 14・2・3 製作

> **章**: 第14章 科学気球
> **節**: 14・2 成層圏気球

a.強度 平面ゴア法により製作されるゼロプレッシャ気球で
は，フィルム張力は二軸方向に複雑に発生し，気球としての実質的
強度は同じ容積の球形気球とほぼ同一となる．標準的な例として，
容積10万mの気球が高度35km（大気密度0.0085kg/m ）で水
平浮遊する場合を考える．球に換算した気球の半径はおおよそ30
mであり，20mの気球用フィルムの低温における破断強度は表
14･1 より約1100N/mであるので，気球が破壊する圧力はおお
よそ70Paとなる．一方，気球頭部付近のフィルムの内外圧力差は
約5Paとなり，14程度の安全率が得られる．
ゼロプレッシャ気球の場合，頭部に加わる圧力は大気圧の高い地
上が最も大きく，地上と上空での大気圧の比の三乗根に比例して増
大し，概略6倍強となる．地上では気球の頭部にのみガスが詰まっ
た状態であるから，もし気球全体を地上強度に合わせて製作すると
むだな重量増をもたらす．そこで，大型気球では頭部のみフィルム
を二重，三重にして強度を確保する．
三次元ゴア設計法を適用すれば，フィルムの幅が約3mである
から，ゴアの膨らみの周方向局所半径 は1m程度に小さくでき
て，気球張力は大幅に減少する．かつ，気球の容積が増大してもゴ
アの枚数が増加するのみで，局所半径は変わらない．すなわち，気
球の耐圧性は気球の容積に依存しない．
b.被膜材料 つり下げるペイロード質量が数百kgから1tを
超える大型ゼロプレッシャ気球の被膜材料には，厚さ20m程度
のポリエチレンフィルムが用いられる．単位面積当たりの質量は約
20g/ mである．ペイロード質量が数kg程度と軽く，40kmを超
える高い高度まで上昇することを主目的とする場合には，5mや
3m強の薄く軽量なフィルムが用いられることもある ．
フィルムに要求される特性は，十分な強度，高い伸び率，高い引
裂強度（tear strength），および脆性温度（brittleness tempera-
ture）が十分低いことである．最後の事項は，上層大気が成層境界
面（中緯度で約10km）で最も低くなるため，そこを通過する際
にも柔軟性を失わない必要があるためである．最近のフィルムでは
－100℃程度まで十分耐えられる．代表的なアメリカの大気球用フ
ィルムの特性を表14･1に示す．低温域でも破断までに400%以上
という大きな伸び率をもっていることが特徴である ．
c .製作工程 平面ゴア方式によれば，ゴアと呼ばれる構成要
素は，14･2･2･a項で求めた自然型形状を縦に 等分した紡錘型形
状であり，ロールに巻かれたポリエチレンフィルムから切り出す．
次にその縁どうしをロードテープとともに熱接着により接合して製
作する．当然ゴアの最大幅は材料としてのフィルムの最大幅で決ま
る．代表的例として，20m厚のフィルムを用いた大型気球の諸元
を表14･2に示す．

線の長さとなり，平面法によるゴアより一回り大きい．その縁の長
