# 14・2・5 飛翔の安全・保安

> **章**: 第14章 科学気球
> **節**: 14・2 成層圏気球

科学気球は，上昇と飛翔終了時に航空機の運行高度を横切る．ま
た気球バースト等，不測の事態では重量物が落下するおそれがあ
る．このため，安全・保安には十分注意して飛翔させなければなら
ない．
航空管制関係の規制は，国際民間航空機関（ICAO:Interna-
tional Civil Aviation Organization）が定めた，国際民間航空条約
第2付属書の3.1「人身および財産の保護」中の3.1.8で無人自由
気球への基本規定が記され，同書の付録-4で細かい条項が定めら
れている ．各国とも気球実験では，この規則に準拠して対応し
ている．わが国でも，気球実験に際しては，国土交通省航空局およ
び関連航空機関との事前協議の後，フライト計画を提出する．航空
局はノータムに記載し，担当機関に周知を図り，飛翔中は航空管制
の対象とする．
海上の安全管理は海上保安庁の管轄であり，事前説明が行われ
る．おもに，ペイロードの海上着水時の安全対策と回収が対象とな
る．1971年に海洋汚染および海上災害の防止に関する法律が制定
されたことにより，気球本体の海上での回収も義務付けられること
となった．欧米のように，陸上で回収するものも気球本体の回収に
は努力を払っている．
