# 10・1・5 ジェロンテクノロジーの今後の展開

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・1 概説

ジェロンテクノロジーの技術基盤が整備され、人間特性データベースが完成し、かつ、そのデータベースを載せる効果的なディジタルヒューマンモデルが開発できれば、それを駆使することで、われわれの周りにある製品・情報・環境の使い勝手は飛躍的に上昇し、高齢者も若者と同様、生活機能を高める新製品を自由に使いこなし、快適な生活を送ることが期待できる。特にこの数十年における寿命の顕著な延長は、高齢者の生きがいやニーズに大きな変化をもたらしつつある。その結果、従来の高齢者に対する社会的な行動規範が大きく変化し、高齢者の新しい生き方への模索が行われている。今後、ジェロンテクノロジーは、高齢者の新しい生き方の可能性を広げていくための新技術を数多く生み出す研究分野になることが期待されている。
