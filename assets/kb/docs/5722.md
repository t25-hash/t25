# 10・2・1 視覚情報

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・2 高齢者特性に適した情報設計

人間の視覚機能は年齢とともに変化する。これらの機能変化の定量的データは、現在徐々に整備されつつあるものの、まだ十分とはいえない。ここでは、主要な視覚機能である分光感度、色の見え、視力について、加齢による変化とその応用について述べる。a. 分光視感効率(spectral luminous efficiency function)人間の視覚はおよそ400~700nmの波長の電磁放射に応答し、その応答特性は分光視感効率と呼ばれる。眼球を構成する水晶体(レンズ)は加齢によって短波長光(およそ400~500nm)の透過率が低下し、そのため視感効率は年齢とともに低下する。図10・2は10歳代から70歳代までの分光視感効率曲線を10歳代ごとに平均して表した年代別データである。データは570nmで規格化されている。視感効率の変化は500nm以下の短波長域(青や紫の領域)に見られ、すなわち高齢者にとっては青や紫はより暗く見える。例として、図10・3に示すような暗い黄色の背景に青の指標のある視覚表示物のコントラストを、次式の輝度計算式に当てはめて計算してみる。図10・3に与えられた背景と指標の分光放射輝度データと、図10・2の分光視感効率を用いて式(10・1)に従って各年代ごとに輝度(cd/m)を求めることができる。ちなみに、図169.2cd/m、背景(黄)は108.1cd/mとなり、それらの輝度コントラストを(指標輝度-背景輝度)/(指標輝度+背景輝度)の定義に従って計算すると約22%となる。一般的に22%のコントラスト値は見やすい。一方、60歳代の高齢者の分光視感効率を用いて計算すると、指標の輝度は低下して131.8cd/mとなり、背景はあまり変化せず111.2cd/mとなる。そのコントラスト値を計算すると約8%となる。すなわち、高齢者にとっては青の指標の輝度は低くなり、そのため指標の視覚的コントラストは8%と、かなり見づらくなる。このように年代別の視感効率を用いて正しく輝度計算を行うことによって、高齢者に配慮した見やすい標識の設計が可能となる。b. 基本色の領域複数の色を視覚表示物に活用する際は、色の見間違いや混同を避けることが重要である。このとき有効なデータがカテゴリカル色知覚(categorical color perception)データである。色は類似した色をグループとして認識する特性を有する。図10・4はマンセル色空間明度5の平面に示した基本色の類似色領域を表す。基本色サンプルとして選択した11色(赤、橙、黄、黄緑、緑、青緑、青、青紫、紫、赤紫、灰)に対して、若年者(20歳代)、高齢者(60歳代)が、それぞれ基本色に類似していると判断する色の領域を表したものである。重なりを避けるために四つの部分に分けて表した。実線は若年者、点線は高齢者を示す。高齢者のほうが類似色の領域が狭いことがわかるが、領域の形状等の基本特性は年齢による差がない。図10・5はデータの使い方である。標識における背景や指標の色を、それぞれ図10・5のように異なるグループから選択すればよい。図10・3 輝度計算のサンプル指標図10・2 年代別分光視感効率図10・4 基本色の領域(明度5平面)グループが異なるので混同はなく、簡単に識別できる。また、こうした配色は選ぶ領域に幅があるので選びやすく、正確に色を指定する煩雑さもない。もちろん、異なるグループであれば、何色でも見やすい色の設計ができる。c. 視力(visual acuity)眼球レンズの弾性や調節の筋肉系が年齢とともに劣化するため、視力は一般的に年齢とともに低下する。図10・6は輝度100cd/mにおいて、5mから30cmまでの視距離における視力の変化を10歳代から70歳代まで年代ごとに示したものである。5mはいわゆる標準的な視力の計測条件で、ここでは70歳代の視力はやや劣るものの、他の10歳代から60歳代の観測者では、すべて視力1.0以上とよい。この視力は10~30歳代までは30cmの近距離まで維持される。しかし40歳を超えると近距離で急激に視力が低下する。すなわち、40歳を超えると近くのものが見づらくなることを示す。図10・7は、100cd/mと0.5cd/mの2段階の明るさにおいて50cmと2mの距離で、実際に高齢者が平仮名と漢字(5~10画)を読むことのできる最小の文字サイズ(太字タイプ)を示したものである。例えば50cmで100cd/mであれば平仮名は8ポイント、漢字は10ポイントまで読めるが、暗くなるとそれぞれ20ポイント図10・5 基本色領域の活用例図10・6 年代別の視距離と視力の関係9 ―171および24ポイントまでとなる。視力が変化しなければ、見る距離と文字サイズは比例関係にあるが、実際は図10・7中の点線で示すように比例関係は成り立たず、近距離ほど視力は低下する。視力や文字の読みやすさにおいて、距離と明るさの影響は十分考慮することが必要である。
