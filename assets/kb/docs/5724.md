# 14・3・2 火星気球

> **章**: 第14章 科学気球
> **節**: 14・3 惑星気球

a.火星の大気 火星大気の97 %は二酸化炭素である．表面に
おける平均気圧は約7hPaで，地球上での高度33.5kmに相当す
る．表面の平均温度は約220Kで，昼夜の温度差は100℃にもな
る．また，火星の軌道がだ円で軸が傾いているため，場所，季節に
より変化が大きく，赤道上での夏の昼間と極地方の冬では160℃も
の差がある．
これまでのランダ等による計測では，通常の風は数m/s程度で
あるが，しばしば砂嵐が起こり，そのときの風速は100m/ s以上
になる．
b.火星気球（Mars balloon）の条件 火星では，大気が希薄
であるため，地球上の成層圏気球でも表面付近に浮かぶことができ
る程度である．昼間のガス温度は使用する被膜にもよるが
30～60℃上昇すると考えられ，昼夜のガス温度差は150℃程度と予
想される．これは，圧力差に換算すると大気圧の50 %以上になり，
ゼロプレッシャ型気球ではガスの排気量が大きくて実用にならな
い．地球上におけるスーパープレッシャ気球より格段に厳しいこの
圧力差に耐えられる軽量のスーパープレッシャ気球が要求され，被
膜はこの温度範囲で性能が劣化しないことが求められる．
そこで，c項に示すゼロプレッシャ型の気球で長時間浮遊可能な
ガイドロープ（landing snake）付き気球 が提案されたこともあ
る．しかし，14･2･2項に示す三次元ゴア設計法によるスーパープ
レッシャ気球を用いれば，この圧力差に十分に耐えられる軽量気球
を実現できる．
c .ガイドロープ付き火星気球 夜間，ガス温度が低下し，
表面近くまで降りてきたところで，気球から長く（数十m以上）
垂れたガイドロープがまず着地するため，浮力低下分が相殺され
る．夜が明けガス温度が上がれば再上昇して，ガイドロープは気球
からぶら下がる．したがって，昼間は数kmの高度に，夜間は表
面すれすれに浮く（soft landing）気球となる．
d.スーパープレッシャ型火星気球 例えば，高度5km（大
気圧4.5hPa，大気密度0.01kg/m ）に浮遊する場合で，夜間の
ガス温度を180Kとし，昼夜間のガス温度差を140℃と仮定する．
昼間の気球内外の圧力差は夜間の圧力差を零として約3.5hPaに
なる．三次元ゴア設計法により，容積1000mの気球を質量6kg
程度で実現可能である．この気球の総浮遊質量は10kgであり，ガ
スを除いた2kgのゴンドラをつり下げ可能となる．
