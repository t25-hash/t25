# 10・3・1 温熱環境(thermal environment)

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・3 高齢者に適した環境設計

a. 体温調節(thermoregulation)の特徴ヒトの体温調節は体温を一定に保つための機構であり、身体の表面や内部にある温度受容器により絶えず情報が検出され、体温調節中枢に伝達される。体温調節中枢は統合された情報と基準値との差に従って、体温調節の効果器を制御する。すなわち、高温環境で温受容器が活性化されて脳に伝達されると、血管が拡張し血流量が増え、身体からの放熱を促進する。さらに高温になると、発汗が始まり、潜熱による熱放散も始まる。一方、寒冷環境では冷受容器が温度の低下によって活性化されると、血管は収縮し血流量を減らして放熱を抑制する。さらに、鳥肌が立つなど、震えによる熱産生を生じる。これらの体温調節を自律性体温調節と呼ぶ。自律性体温調節機能は加齢に伴って低下する。暑熱や寒冷の環境でなくとも、人がふつう生活する中等度温熱環境においても、高齢者と若年者の体温調節反応を比べると差が認められる。夏期に23°C,27°C,31°Cの環境下で安静状態をとった場合の生理反応を比較すると、23°Cでは高齢者の皮膚温は1°C近く若年者よりも高くなり、代謝量は温度条件にかかわらずほぼ一定であるが、高齢者の代謝量は若年者よりも2割少ない。湿性放熱量は温度が高くなるにつれて大きくなるが、年齢差は認められない。つまり、湿性放熱量に年齢差が見られず、乾性放熱量は高齢者のほうが大きいことから、23°Cでの熱収支量はマイナス側、すなわち体内温は下がりやすいことを示している。b. 温冷感(thermal sensation)の特徴衣服の調節や空気調和(空調)など目的をもった活動を行動性体温調節と呼ぶ。行動性体温調節は温冷感や快適感など人の温熱感覚に従って生起することが多い。温冷感は全身的なものと局所に関するものの2とおりがある。局所温冷感の例として、冬期における中性温度下での身体各部位の温覚・冷覚を生じるしきい値を示す(図10・11)。このしきい値の部位差は温受容器と冷受容器の分布密度の違いを反映していると考えられる。胸部、臀部、足部は感受性が鈍いことが示されており、それらの部位においては年齢差が大きいが、頭部では年齢差が認められない。全身温冷感については、欧米では中性温度下においては年齢差が認められないという報告があり、快適になるように自分で温度を制御する場合にも、快適温度に年齢差は認められていない。夏期に中等度温熱環境で全身温冷感を比較した結果を図齢者はやや涼しい、若年者は涼しいと、高齢者が若年者よりも1段階中性側の申告であった。この23°Cにおける温冷感の差は皮膚温の違いによる影響と考えられる。環境に対する評価のうち、許容度から算出した不満足率を示す(図10・13)。不満足率は、温冷感でどちらともいえないと申告された27°Cで最も低く、若年者では23°Cと31°Cでは25%である。しかし、高齢者では23°Cも27°Cと同様に不満足率は5%にすぎない。これらどちらともいえないという中性温度下では、欧米での結果と同様に年齢差は認められないが、わずか4°C低温になっただけで温冷感にも評価にも加齢による影響が現れている。一方、冬期の結果は、温冷感で23°Cと27°Cではほとんど年齢差を認めないが、31°Cでは若年者が暑い、高齢者が暖かいと1段階高齢者が中性側の申告であった。つまり、中性温では温冷感についての年齢差が認められず、夏期には23°Cの低温側条件、冬期には31°Cの高温側条件において、高齢者が若年者よりも夏期には寒さを感じにくく、冬期には暑さを感じにくいことを示している。これは、季節によりばく露される頻度が低い受容器の感受性が減退して図10・11身体局所の温覚・冷覚の生じるしきい値温度図10・12 温熱環境の設計高齢者の体温調節反応は中等度温熱環境ですら血管運動の衰えが示され、感覚的にも感受性の低下が見られる。よって、環境設計を行うに当たっては、それらの高齢者特性に配慮する必要がある。すでに提案されている住宅熱環境評価の基準値を表10・1に示す。これまでの独立住宅の温熱環境実測値に比べると、この基準値においては冬期の着衣量が少ないために室温は高い。夏期については、午後から夜半にかけてこの基準値を遵守しようとするなら、空調による冷房が必要な値である。しかし、実態調査によれば、高齢者は冷房の使用を好まず、冬期には暖房使用のみならず、こたつを併用して手足を温めることが多く、さらに衣服を着込んで対応している。公共空間や施設などでは、高齢者ではなく若年者が室温設定する場合が多い。上記で述べたように、寒冷や暑熱は高齢者にとって体温調節がついていけず大きな負担となる。反対に中等度温熱環境であっても、冬期の高温側や夏期の低温側の環境では高齢者の感受性は低下している。よって、着衣の状態や活動状態を見極めて中等度に設定する必要がある。冬期に厚着でない場合には、代謝量の少なさを補うため、若年者の快適と感じる室温よりも2~3°C高めに設定することが一つの目安となろう。また、脱衣所やトイレなど裸や薄着になる場所には暖房を入れるなどの配慮が必要である。
