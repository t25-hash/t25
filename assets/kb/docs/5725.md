# 14・3・3 金星気球

> **章**: 第14章 科学気球
> **節**: 14・3 惑星気球

a.金星の大気 二酸化炭素を主成分とする金星大気には，高
度50～70kmにかけて硫酸を含む雲があり，地表を覆い隠してい
て，これより上からの可視領域での観測はできない．大気圧，温
度，密度はそれぞれ，表面で755K，9.2MPa，65kg/m ，高度
20kmで580K，2.2MPa，20kg/ m ，高度50kmで地球上と同
程度の0.1MPa，350K，1.6kg/ mである．
金星の自転周期は243日と非常に長いが，風は東から西に非常に
強く，雲の最上層部，高度70kmで100m/ s以上，高度35kmで
40m/ sもの風が定常的に吹いている．これは4日で循環するスー
パーローテーション（super rotation）と呼ばれる金星大気のなぞ
の一つである．この風により非常に広範囲を短時間で飛翔すること
ができる．
b.金星気球（Venus balloon）の条件 金星大気が濃密であ
るため，高度を低くとればかなりの小容積の気球でも十分な浮力
（十分な搭載能力）が得られるが，その分高温環境となり，気球被
膜の耐熱性が要求される．また，長時間の浮遊には耐硫酸性を考慮
しなければならない．
c .高高度膨張型気球（VEGA) VEGA-1およびVEGA-2
はランダから切り離された後，ヘリウムガスによって膨張させら
れ，赤道付近（それぞれ北緯7.1°および南緯6.5° ）の真夜中側に
浮遊した．その高度54kmの圧力540hPa，温度305Kは地球表
面とほぼ同じである．2機の気球は直径3.4mのテフロン製の球形
スーパープレッシャ型気球である（標準圧力差30hPa）．浮遊総質
量21kgのうち観測器とテレメトリシステムを含むゴンドラ（図
14･10）の質量は6.5kgである．周囲の大気圧，温度，相対鉛直風
速，後方散乱，太陽照度が測定された．信号は地球とフライバイモ
ジュールの両方で受信され，地球上のアンテナを動員したVLBI
（very long baseline interferometry）により気球の位置と速度が求
められ，スーパーローテーションが確認された．1kgのリチウム
電池により，飛翔時間46時間を達成し，その飛翔距離は1万1000
km（金星を1/3周）であった．
d.中高度膨張型気球 35km程度の高度（圧力0.58MPa，
温度450K）では，気球被膜として使用可能な耐熱フィルムの選択
肢も多い．300℃以上の耐熱温度をもつものとしてはポリイミド
（polyimide），液晶ポリマ（liquid crystal polymers ）等がある．
雲の下であることから，可視領域や高精度の近赤外線観測も可能で
ある．VEGAと同様にヘリウム等の浮揚ガスを高圧容器に入れて
輸送して膨らませる方法も考えられるが，高度42km以下では水
蒸気の蒸気圧が大気圧より高くなることを利用し，水を浮揚ガスと
して使用することが可能である．
水を浮揚ガスとする利点は，(１)輸送時は容積の小さい液体，
(２)水をあらかじめ気球内部に封じ込めておけばガス注入装置が不
要，などである．ただし，大気中をパラシュート降下する短時間の
うちにすべての水を気化できないと目標高度以下に至り，気球本体

や搭載機器の耐熱限界を超える．例えば，高度35kmに総浮遊質
量10kgの気球を浮遊させるのに必要な水の量は4.3kgである．
これを1時間程度で気化する必要がある．これまでの熱交換器を用
いる方式に対し，水をあらかじめ気球内面に吸着させておき，降下
中に周囲の高温大気を直接利用する方式 が提案されている．こ
の方式の気球投入シーケンスを図14･11に示す．大気からの熱流入
を大きくするため，アスペクト比の大きい円筒形気球が適する．
e.高度可変方式 例えば，水を用いると高度42km以下では
気体となり浮力が増加し，42km以上の高度では液化して浮力を
失う．このような性質を利用すれば，適当なバルブ操作により気球
浮遊高度を制御することが可能となる．このために，大型の熱交換
器を搭載する．表面まで降下してサンプルを採集する計画も提案さ
れている ．
f .ドロップゾンデ方式 高高度に浮遊している気球から複数
のドロップゾンデを投下する方式である．気球はゾンデのプラット
ホームであると同時にゾンデと地球間の通信の中継を行う ．
g.低高度金属球方式 金星大気は濃密であるため，高度20
km以下の高温高圧中では，小型の気球で十分な浮力が得られる．
耐熱フィルムによる膨張型気球の代わりに，最初から浮揚ガスを封
入した薄い金属球の気球を運搬する方式 が適用可能である．高
温，硫酸大気環境下でも気球の強度やガスバリア性に問題がなく，
長期間飛翔が可能である．ただし，一重の気球では最初から球形高
圧容器として運ばなくてはならないため，気球本体重量が増加す
る．これを大幅に減らすため，気球本体は保護のため耐圧容器に入
れて運び，浮遊高度で分離させる二重カプセル方式 が考案され
ている．
例えば，直径1mのチタン製の気球は厚さが0.1mm程度で強
度が保たれ，気球の質量は2kg程度となる．高度13kmに飛翔さ
せるとすると浮遊総質量は17kgで，浮揚ガスを除いた13kgの機
器を搭載可能である．
