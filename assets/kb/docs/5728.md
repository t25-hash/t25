# 10・4・3 高齢運転者の運転特性

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・4 自動車および交通に関わるジェロンテクノロ

高齢運転者の特性に関しては,ドライビングシミュレータを用いた運転行動調査なども含めて国内外で研究が行われている.高齢運転者は,長期間にわたって安全運転を実践してきた実績のある運転者が多いが,加齢によって身体,意識の変化も生じている.そのため,運転特性にはいくつかの特徴が表れてくると考えられる.特徴は概して表10・2のようである.また,高齢者は加齢により特性値の個人差が拡大すること,危険状態の事前警報は回避に有効であることなどが明らかになっている.
