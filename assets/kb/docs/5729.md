# 15・1・2 操作性

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・1 医療・福祉情報システムとしてのインフラスト

医療情報を電子化することにより，医師等医療従事者にこれまで
以上の負荷を与えるようであれば，いくらその効果を示してもシス
テムの普及ははかどらない．そこで，従来の作業フローから大きく
逸脱することのないシステム，もしくは従来システムよりもより効
られる．
率的に処理が可能なシステムを構築しなければならない．例えば，
いままでフィルムで読影していた画像診断をCRT（cathode ray
tube）診断とした場合，その読影方法はドラスティックに変化す
ると考えられるが，CRT読影により効率的な処理，読影精度の向
上が期待できれば，その導入のスピードは速まるであろう．しかし
ながら，読影作業に伴う新たな操作環境がそのスピードを左右する
である．
と考えられるため，キーボード操作等をなるべく意識させない入力
