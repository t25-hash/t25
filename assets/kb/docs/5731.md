# 10・5・1 高齢者の課題

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・5 高齢者特性に適した就労支援技術

1970年に65歳以上の老年人口比が7%を超えて高齢化社会と認定されて以来、日本の高齢化は急速に進み、いまや高齢社会になった。日本の高齢社会の最大の課題は、年金支給開始年齢の65歳への引上げ決定により、高齢者をどのように60~65歳まで雇用継続の実現をさせるかという労働問題である。失業率が戦後最高となり高齢者の有効求人倍率が最低となった現在、高齢者の雇用問題は、(1)企業の定年延長、(2)企業の雇用延長・再雇用、(3)高齢者による起業化の3点に集約される。法律により60歳定年が規定されているので、最大の雇用効果の推進は(2)の雇用延長・再雇用であり、また年齢による雇用の差別をしない(年齢指針)ことである。これには、高齢者は企業経営にプラスになるもののマイナスにはならないとの認識を雇用主が深める必要があると同時に、現実にそうなるための雇用技術やノウハウおよび高齢者雇用のツールが準備される必要がある。それが職務再設計(job redesign)である。
