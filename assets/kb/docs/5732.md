# 10・5・2 加齢による生理的・心理的変化

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・5 高齢者特性に適した就労支援技術

効果的な職務再設計を推進するには、加齢(年をとること)の作用を正確に知り、その知識を活用することが不可欠である。元気な高齢者もいるが、下記が加齢の原則である。(1) 生理的機能は比較的早い時期から低下する。(2) 筋力の低下は足・腰・腕・手の順に生じる。(3) 動作や巧緻性も年齢とともに鈍くなる。(4) 認知的機能はかなり年をとってから低下する。(5) 人工的につけた能力(スキルなど)は年をとっても残存する。(6) 若い時期から継続して学習する機会を与えると、高齢になっても学習能力は落ちない。(7) 年をとるにつれて性格的に固執性が強く頑固になる。一般的に高齢者雇用は生産性低下をもたらすと考えられているが、上記のポジティブ・ネガティブ両面の変化を理解したうえで、ネガティブな機能変化は機械化などの別の機能で代替し、ポジティブな機能を積極的に活用するように職場の仕組みを変えていくと、高齢者は十分に戦力になりうる。
