# 10・5・3 暦年齢と機能年齢

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・5 高齢者特性に適した就労支援技術

65歳以上を高齢者というが、高齢者はすべて諸機能や能力が低9 ―176医療・福祉・バイオ機器図10・18 加齢と機能年齢下するのではなくて、能力や体力を十分にもっている高齢者も存在する。加齢の特徴の中に、個人差が大きくなるという特性もある。そこで加齢に絡んだ重要な考え方を以下に示す。A 職務を遂行する要件が図10・18の水平線であるとする。誰でもこの水平線近くの能力があればA 職務を遂行できる。図10・18中の横軸は年齢を、斜線はB 作業者の能力を表し、それは年齢がたつにつれて斜線のように下がっていく。能力線である斜線と職務要件である水平線とが交わった点はB 作業者にとってA 職務遂行がそろそろ困難になった時期を表し、それ以降は彼の生産性が低下する。この交点はすべての人にとって同一年齢で同一ではなくて人によって異なる。つまり、斜線はその人の能力によって決まるのであり、これを機能年齢(functional age)という。若い時期に体力や能力を磨いた人の機能年齢との交点は、かなり遅くなって発生するが、そうでない人の交点は早くやってくる。運動を嫌う若年者の早期老年化はこのことを物語る。そして、交点以降は生産性が低下するので、管理者は高齢者を嫌がることになる。その打開策として以下のことが考えられる。(1) 高齢者の能力開発を実施して機能年齢を向上させ交点を右側へ遅らせる。(2) 職務再設計を実施して、B 作業者の機能年齢よりも職務要件を下げ、生産性に問題が生じないように対策を立てる。高齢者の能力を向上させることは不可能ではないが、費用対効果を見るとそれほど容易ではない。それに対して職務を改善して高齢者に楽に従事してもらえるように職務再設計をすることは割と簡単であり、効果も大きい。このことから、高齢者雇用を推進し、経営者や管理者からも高齢者を喜んで雇用してもらえる方策として、職務再設計が重視されている。
