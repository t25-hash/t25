# 10・5・4 職務再設計の実際

> **章**: 第10章 ジェロンテクノロジー
> **節**: 10・5 高齢者特性に適した就労支援技術

職務再設計とは、職務が要求する職務用件が作業に従事する作業者に適合するように職務を再設計(改善)することをいう。ここでは高齢者を対象としているので、高齢者がもつ能力や機能に合うように職務側を改善することであり、職務と高齢者の融合を図り、彼らを生産性の担い手に位置付けることである。高齢者全員に共通した再設計も可能だが、ほとんどの場合、個々の高齢者に該当する再設計を実施することが望まれる。そのためには、以下の点に配慮する必要がある。(1) 視力等の生理的機能低下を拡大鏡・顕微鏡などの他の機能で代替したり、視的機能を音機能に変換したり、より認知しやすい道具・情報に置換える。(2) 筋力の低下に対して筋力を用いなくて済むように、クレーン、ホイスト、台車、リフト、コンベヤ等の活用、ジグの製作等で職務要件を下げる。(3) エネルギー負担となる屈曲作業姿勢を、作業点を楽な適正位置まで変更することで疲れない正常な作業姿勢に変更する。一般に高齢者にとって高負荷となる作業であっても、(1)~(3)の条件に関する配慮で負荷が軽減化するように再設計すると、高齢者はほとんどの再設計された職場に就労できるようになる。また、このような再設計に活用できるツールもたくさん開発されている。以下に、高齢者の職場での活用例を挙げる。(1) 高齢者は、一般に勤勉で注意深い面があり、長年従事した職場で雇用延長で働いてもらい、若年者に対する指導的役割をすることで、若い作業者への模範となり、作業する態度や技能の伝承にもなる。(2) 50歳以降の中高年齢者は、職場では経験が長く深い技能をもっていることが多い。このような熟練作業者(skilledworker)に制御工学、コンピュータ、ソフトウェア工学などの先端的技術を学習させることで、ロボットなどの自動機械を製作させるなど、生産技術の戦力として鍛え上げることができる。各職場から優れた熟練者を集めて最先端技術を学習させ、その後、元いた職場の改善ニーズを発掘させてそれらを機械化・自動化し、生産の効率化を実現するというやり方で高齢者を活用した企業は少なくない。
