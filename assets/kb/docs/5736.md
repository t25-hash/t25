# 15・3・2 電子カルテの開発の歴史

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

a.レセプトコンピュータ（clinical accounting system) 医
療機関では毎月，健康保険支払機関に対して診療報酬請求を行う．
この作業の合理化のために開発されたのがレセプトシステムで，一
般にレセコンと呼ばれ，十数社程度のベンダが供給している．レセ
プトコンピュータによって医事処理は合理化できたが，処方，注
射，処置などの医療行為は伝票ベースで残っているため，レセプト
コンピュータにデータを入力する部署に作業が集中し，業務全体の
ボトルネックとなっている．特に大規模病院ではその傾向が顕著と
なり，患者の待ち時間は短くならなかった．
b.オーダエントリーシステム 円滑な処理を実現するため
に，データ入力を診療現場に受け持たせ，処方，検査などが発生し
た段階で入力すれば入力作業の集中が避けられる．このシステムを
オーダエントリーシステムと呼ぶ．このシステムにより患者の待ち
時間は大きく短縮され，伝票の読み違いなどによるミスも少なくな
り，病院業務の改善につながった．しかし，このシステムは単にコ
ンピュータによる伝票処理システムの入力が分散したにすぎず，重
要な医学的記述は紙のカルテに書かれていた．
