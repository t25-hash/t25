# 11・1・1 障害者支援としてのバリアフリー技術

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・1 概説

障害者や高齢者の生活を支援するため、これまでさまざまな技術が開発されてきた。つえや眼鏡といった身近な日常の生活用品に始まり、駅の階段昇降機や電動車いす、点字プリンタといった大きく高額なものまで、その範囲は広い。これらは、国内ではバリアフリー機器と呼ばれることが多い。建築や住宅設計、まちづくりや公共交通も、障害者や高齢者を対象にした製品や設計手法は、総称してバリアフリーと呼ばれる。かつては障害者向けというとデザインも限られ、コストも高いものが多かったが、いまでは福祉機器展などの展示会場にはカラフルな機器や日用品が並び、多様性を増してきているのがうかがえる。障害を軽減する研究も多様化しており、さまざまなロボットによる生活支援やバーチャルリアリティによる歩行訓練、携帯電話やPDA(personal digital assistance)による情報受発信支援など、最先端の技術を用いた研究も多い。重度障害者の意思決定を支援するコミュニケーション技術の領域では、一般的なマウスやキーボードによる入力の代わりに視線や脳波を使ったり、音声だけで入出力が可能なインタフェースも開発されてきている。今後は、医学やリハビリテーション工学、ヒューマンインタフェースや認知心理学など、多くの学問領域をまたがる学際的な研究が求められている分野であるといえよう。なお、日本でいうところの、いわゆるバリアフリーとは、欧米ではアクセシビリティ(accessibility)という概念で語られることが多く、障害者が建築空間や製品情報などにアクセスできるようデザインすることを指す。欧米では、バリアフリーという言葉はあまり使われていない。
