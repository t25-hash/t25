# 11・1・2 バリアフリーの限界

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・1 概説

なぜバリアフリーという言葉が海外では使われないかといえば、建物や製品に内在するバリアを後付けで外していく方法では解決できない問題が多いという認識からである。この状況は日本も同じである。近代日本は、戦後の混乱期を何とか乗り越えようとして、健康な若い世代を中心に街やものを作ってきた。いわば、高齢者や障害者にとってバリアフルなものを前提とし、そのバリアを後から外していくやり方をとっていたのである。しかし高齢者が増え、成人人口の50%が50歳を超すという未曾有の事態に突入した日本では、その方法では間に合わなくなってきた。後付けのスロープは距離が長く車いす利用者にはこう配がきつかったり、障害者専用トイレやエレベータは使用頻度が低くて施錠されたままという事態も発生した。莫大な費用をかけて開発された障害者支援技術が、技術的には優れているのに、高価さや地域における不十分なサポート体制のために、利用者が増えず普及しないというケースも少なくなかった。いわば、特殊な人々のための特殊解では全体の満足感の底上げにはつながらないということが、次第に認識されてきたのである。バリアフリーの方法では、障害者は人とは違う特別なルートをたどることを要請され、格別に目立ったり逆に不本意に隠されたりしながら生活することを余儀なくされる場合もある。車いす専用昇降機には、ベビーカーや妊産婦、高齢者を乗せることは想定されておらず、同じようなニーズがある人々を遠ざける結果になりかねないのである。
