# 15・3・5 電子カルテに決まった形はない

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

a.データの互換性 電子カルテは，相互にデータを交換する
以上の問題を解決すれば電子カルテは現実のものとなる．しか
し，ここで注意すべき点がある．コンピュータという新しいメディ
アを使った医療記録を考えるとき，従来の紙のカルテのイメージに
影響されがちになるが，そのような古い概念は捨て去ったほうがよ
いということである．ユーザインタフェース（目に見える電子カル
テの形）を最初から議論すると，どうしても紙のイメージの電子的
な再現になる．電子カルテの本質は，広域で情報交換が可能である
ことと，対象（患者，医療スタッフ）によって見せ方を変えること
ができることであり，百人百様のユーザインタフェースがあっても
よい．今後提供される電子カルテでは，まず標準的なインタフェー
スが提供されたうえで，ユーザの事情に応じて個人レベルでどのよ
うにでも変更できるようにするのが望ましい．
