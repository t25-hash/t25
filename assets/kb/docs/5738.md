# 11・1・3 高齢社会日本に紹介されたユニバーサルデザインの概

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・1 概説

念ユニバーサルデザイン(UD : universal design)とは,1990年ごろからアメリカのメイスによって紹介された概念である.子供,女性,外国人,障害者,高齢者などの多くの人々が,より使いやすくなるよう最初から考慮してデザインするという新しい考え方であった.新規に駅を開発するのであれば,できればエレベータを最初から設置する.ボタンも最初から車いすユーザの手が届く位置におく.音声案内も最初からつける.後から追加するより,コストもかからず,デザインも美しく仕上がる.麗々しく障害者向けといわなくても,子供も,たまたまけがをした人も,海外からの旅行者もみなが助かる.次第にその考え方は各企業や自治体に広まっていった.いまでは静岡,埼玉,熊本,岩手,高知などさまざまな自治体が,まちづくりの基本理念にUD を挙げている.また,ものづくりの基礎としてデザインセンター全員にUD の考え方を学ばせている企業も増えてきた.UD の7原則は次のようになっている.物事をデザインするうえでは当たり前のようなことばかりであるが,実際にはこれを守って作られ,アクセシビリティとユーザビリティを兼ね備えた製品は国内ではまだそれほど多くない.北ヨーロッパなどではUD とうたってはいないが,結果としてUD になっているものも多い.・原則1:誰にでも公平に利用できること・原則2:使ううえで自由度が高いこと・原則3:使い方が簡単ですぐわかること・原則4:必要な情報がすぐに理解できること・原則5:うっかりミスや危険につながらないデザインであること・原則6:無理な姿勢をとることなく,少ない力でも楽に使用できること・原則7:アクセスしやすいスペースと大きさを確保すること
