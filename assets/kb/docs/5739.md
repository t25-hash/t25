# 15・3・6 電子化によるメリット

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

病歴情報がネットワーク上で電子化された場合，従来考えることテシステムだけからもたらされるものではなく，コンピュータが広ものが大きなメリットを生む．
医療レベルの向上 現在の医療は，ある意味で標準化さ方（医学的背景）などによって異なる治療が行われる．
病気に対す発揮するだろう．
のできなかったメリットが生まれる．
そのメリットは単に電子カル域ネットワーク（インタネット）につながっているという事実そのa.電子カルテシステムがもたらすメリットれていない．
