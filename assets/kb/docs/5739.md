# 11・1・4 バリアフリーとユニバーサルデザイン

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・1 概説

一方で,バリアフリーからユニバーサルデザイン(UD)へというよくある言い方は,あまり現実的でないように思われる.現在の日本には,和風建築をはじめまだまだバリアフルな街やものが多数現存する.それをどのようにアクセス可能にしていくかは,克服すべき大きな課題であり,バリアフリーの研究も非常に重要であるといえる.そのことを理解したうえでUD の感覚を身につけることは,今後,企業や政府関係者にとって必須のマナーであり,高齢者市場に生き残るための常識になっていくと思われる.もはやUD は,社会貢献やイメージアップのために行うものではない.その根拠を3点,以下に紹介する.a. 人口構成の劇的な変化日本は2015年には人口の25%が65歳以上となるという,人類の歴史始まって以来の超高齢国家であるが,産業界などのターゲットはまだ若い人が中心であり,需給のギャップが大きくなってきている.アメリカのシニアの定義である50歳以上という観点で見れば,日本人全体では50歳以上が成人人口の50%を超えたのは2005年である.有権者,納税者,消費者と呼ばれる過半数を50歳以降が占める,まさに未曾有の事態である.また,それに対する備えも少ない.高齢化率が7%(高齢化社会)から14%(高齢社会)になるまでの時間は,フランスでは114年かかったが,日本は24年である.社会の意識,まちづくり,ものづくり,社会インフラストラクチャそのものがこの事態を認識しておらず,対応もできないまま,いまに至っている.今後,この膨大な量のアクティブなシニア層を顧客としてとらえ,その意見を製品に反映できる企業でなければシェアは確保できず,顧客満足度も低く,ユーザサポートのコストは増大するばかりという事態を招くであろう.b. ユーザの成熟UD が日本に根付くと思われる二つめの論拠は,ユーザの成熟である.50歳以上の向上心,向学心に燃えた比較的裕福な世代をターゲットにした雑誌は花盛りであり,それらの中でも良質で使いやすく配慮され,かつ高級感のある機器や生活用品が多数紹介されるようになった.このようなマスコミの変化やアクティブなミドル,シニアの増加によって,未成熟だった日本の消費者文化がようやく成熟するきざしを見せるのであれば,歓迎すべき傾向といえる.かつて,日本の消費者は不平を言わなかった.もし購入した機器が使いにくくても,返品に行くことはまれだった.大量生産,大量消費の時代には,メーカ側の論理のほうがユーザの希望より強かった.しかし,いまやニーズが多様化したユーザは,他とは違う自分だけに合うものをメーカに要望してくる.個々人がそれぞれ違うのだから,ほかならぬ自分に合ったものを要求するのは当然の要望である.多品種少量生産やオンデマンドアセンブリーがそれを可能にし始めており,企業が多様なユーザの求めに応じるという状況は今後も加速するものと予測される.c. ITの進歩ユーザの多様な要望を可能にしたのは,IT(information technology)の進展による部分が大きい.車や椅子,洋服や靴まで,多様な各人の希望を聞きながらオンデマンドで最終製品を完成させることが一般的になりつつある.いまや企業の意思決定は,企業側ではなくユーザ側に移っている.IT で主導権を握ったユーザの意向を無視しては,市場をつかむことはできなくなっている.そのユーザ層が高齢化している以上,メーカはもはや作りたいものを作るのではなく,ユーザの求めに応じてユニバーサルな製品を作ることが至上命題となる.また,ユーザ側がIT を使って企業や公企業に意見が出せる環境が整ってきたことも,UD の進展を加速している.シニア層や女性層など,UD の必要性のある層は,IT で建設的な意見を述べることが可能となった.IT が成熟した市民社会や力のある消費者層を形成する助けとなっているのである.以上,三つの論点で,今後,日本のUD が進展するという見通しを述べた.アメリカにおいては「リハビリテーション法508条」のように,法律で産業界や行政を規制するというアプローチが多く,ヨーロッパの場合はISO 13407:1999のように,デザインのプロセスを変えていこうという動きが急である.日本においては法律や国際標準での規制も効果があると思われるが,最も有効なのは市場の動向であろう.欧米よりもはるかに高齢化率の高い日本において,ミドルやシニアがIT を武器にまちづくりやものづくりに意見を言い始めるとき,日本独自のUD が根付くものと思われる.今後の進展が期待される.
