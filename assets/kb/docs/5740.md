# 11・2・1 バリアフリー環境と法制度

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・2 建築・住宅

バリアフリーな生活環境を整備するための法制度は,おもに次のような法令によって構成されている.a. 住宅の品質確保の促進に関する法律(住宅品確法)年制定.本法の中で,高齢者等への配慮に関する評価方法基準が示され,加齢等に伴う身体機能低下を考慮した移動行為の安全性および介助行為の容易性を考慮して,住宅がもつべき性能を5段階に分けて評価することになっている.評価については居住者の任意によるが,共同住宅の共用部分にも適用される.同法によってホームエレベータの性能が規定されている.b. 高齢者の居住の安定確保に関する法律(高齢者居住安定法)2001年制定.高齢社会の急速な発展に対応し,民間活力の活用と既存ストックの有効利用を図りつつ,高齢者向けの住宅の効率的な供給を促進することを目的にしている.特に,民間活力を活用して高齢者・夫婦世帯向けのバリアフリー化された優良賃貸住宅,高齢者自らによる持ち家のバリアフリー化の推進を図るために,住宅バリアフリー化の基準を示している.c. 高齢者,障害者等の移動等の円滑化の促進に関する法律(ハートビル新法)2006年制定.不特定または多数の高齢者や障害者等が利用する共同住宅,学校,デパート,病院,劇場,ホテル,博物館・美術館,郵便局等(法では特定建築物という)で対象面積2000m以上の建築物や公共交通機関,道路,公園,路外駐車場が円滑に利用できるようにするための措置を講じなければならないと9 ―179している.同法告示において,エレベータや便所におけるバリアフリー機器についての基準が示されている.d. 福祉のまちづくり条例等高齢者や障害者が建築物・道路・公園・公共交通機関を安全・安心して利用できるように,その設置および技術的基準等に関して定めた条例を全国ほとんどの都道府県において制定している.また,これとは別に,住宅については高齢者を主対象とした住宅設計マニュアル等を発行している自治体が多い.
