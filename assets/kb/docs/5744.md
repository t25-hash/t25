# 11・3・2 共用品誕生の背景

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・3 共用品・アクセシブルデザイン

最初の発端は,1972年に日本の工業デザイン事務所4社が集まり,肢体不自由者のリハビリテーション機器の研究を始めたことにさかのぼる.そのときの問題意識は,障害者は日常生活の90%において福祉用具以外の一般製品を使っているが,一般製品は障害者が使うことを想定して設計されていないということであった.図11・6共用品の集合図(UD :universal design)そこで彼らは,障害があってもなくても使える製品をグレーゾーンと名付け,その研究を始めた.1991年4月には,トイレタリー,家電メーカ,および前述の工業デザイン事務所のデザイナーたち16人からなる共用品普及のための研究会,E& Cプロジェクト(enjoyment and creation project)が活動を開始した.
