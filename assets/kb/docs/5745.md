# 11・3・3 不便さ調査

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・3 共用品・アクセシブルデザイン

E& Cプロジェクトは、共用品の原点は障害者の日常の不便さを把握することにあるとし、さまざまな調査を行った。最初に行ったことは、目の不自由な人たちが感じている不便さプリ調査で、メンバーが分担して視覚障害(visual impairment)者のいる20世帯を訪問し、家の中および外で不便さを感じていること、不便さを解消するために工夫していること、そして企業・行政等に望むことについて聞いて回った。その結果、各種製品やサービスの不便さが山のように指摘され、調査を担当したメンバーは皆一様に目からうろこが落ちたとの感想をもった。そのうちの一人は、訪問先で自らの企画した家電製品に出会い、「目の見えない自分にとっては、この平らなスイッチではどこが何のスイッチなのかわかりません」と言われ大きなショックを受けたことを語ったが、その後、所属する会社でバリアフリーの観点から製品企画ができる環境作りに努め、3年後にはスイッチに点字や凸表示など視覚障害者でも操作しやすい製品を開発するに至った。定量調査では、点字、大きな活字および音声カセットテープによるアンケート用紙を送付し、そのうち約300人から回答を得た。定量調査の結果は訪問調査の結果を裏付けるとともに、これにより地方特有の不便さも明らかになった。例えば、視覚障害者の歩行の補助となっている誘導ブロックは、冬の雪国では雪に埋もれ役に立っていないことがわかったが、これは訪問調査で挙げられた誘導ブロックがないという不便さとは異なる回答である。定量調査におけるおもな回答は、同形容器の中身の識別ができない、ポストの中の郵便物等の識別が困難などであり、全体的に情報の入手、機器の操作に関して不便さを感じている人が多いことがわかった。また、定量調査では、不便さとともに工夫していることも聞いた。その中で印象的だったのは、シャンプーとリンスの識別方法の工夫であった。現在では、多くのメーカのシャンプー容器の側面には手で触ってリンス容器と区別がつくようにきざみが付けられているが、調査当時はそれがなかった(図11・7)。調査の中でも二つの容器を間違えると答えた人が多くいたが、その問題を克服するための工夫として、シャンプーの容器に輪ゴムを巻いてリンスと識別しているという回答や、シャンプーは右、リンスは左と置き場所を決めて家族全員で守っているといった回答があった。アンケート結果の集計を行う一方、シャンプーを製造している10社ほどのメーカとの間で情報交換を行ったところ、二つの容器の識別に関しては各社とも過去に何らかの検討を行っていることがわかった。例えば、シャンプー、リンスと書かれた点字のシールを9 ―183図11・7 シャンプー容器側面のきざみ作って配布したり、容器の大きさを変えるなど、企業ごとに独自の工夫を行っているとのことであった。そのような企業の中には、この問題に取り組むために1年間のプロジェクトチームを作り、盲学校等の協力も得て、シャンプー容器の側面にきざみを付けることを計画しているところもあった。そしてその企業は、他の容器(例えばリンス)にきざみが付けられると困るので実用新案を取得するが、この工夫は会社間の壁を越えて広めていくことが重要であるため、無料で公開するので利用してもらいたいとの提案を同業他社に対して行った。その提案から15年後の現在では、日本のほとんどのシャンプー製造企業が容器にきざみを付けている。その後行った他の障害に関する調査の中で出された意見を、以下に障害の種類ごとにまとめて示す。(1) 聴覚障害(hearing impairment)者・音で知らせるもの(例えば玄関のチャイム等)が聞こえない。・名前を呼ばれてもわからない。・相手の説明や質問がわからない。(2) 車いす使用者(wheelchair user)・高いところにあるものが見えない、取れない。・低いところにあるものが拾いづらい。・段差がある場所に昇れない、降りられない。(3) 高齢者(elderly person)・小さな文字が読みづらくなってきた。・力がだんだん弱くなり、びんのふたを開けたり、重いものを持ったり操作したりすることが困難になってきた。
