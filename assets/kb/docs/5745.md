# 15・5・1 はじめに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・5 ホームネットワークを利用した在宅システムと

ホームネットワークは，インタネットの普及による家庭内情報端物家電がネットワーク化された情報家電と呼ばれる端末機器の提案きた．
これらの家庭内情報端末はホームネットワーク内のゲートウンの制御等をはじめ，遠隔保守，ネットワークによるバージョンア医療・福祉分野における在宅診断，介護システムでは，家庭内でラストラクチャとしてこのようなホームネットワークが重要な役目訪問看護支援，在宅介護支援，健康モニタリングなどのシステムが考えられる．
これらのシステムがネットワークと接続されることにより，より効率的，効果的な診断，診療，介護支援が可能となる．
情報インフラストラクチャを利用した在宅での診断は，初診以降の電話診断でも保険点数の加算が可能となった時点から始まった．
しかしながら，会話による問診は可能であるが外見情報や生体情報が得られないため，電話での診断には限界があり，その利用は一部の過疎地や一人暮らしの高齢者宅などに限られていた．
現在，15･2節で述べたブロードバンドネットワークが急速に普及し始めており，そのネットワークと最新のセンサや画像通信技術を組合わせた新しいシステムを用いることで，遠隔地からの在宅診断が可能となってきている．
遠隔地からの診断にはネットワークインフラストラクチャが必須であるとともに，家庭内からの情報発信も重要なポイントである．
医師は患者との会話による直接的な情報と，患者の状態を示す生体情報（バイタルデータ）や様態情報（患者の行動や体調など）などを総合的に判断し，診断を行っている．
患者の生体情報や様態情報は，家庭内のネットワークを介して取得可能である．
よって，今後の在宅診断をより効果的，効率的に発展させるには，患者の生体情報を取得するセンサシステムだけではなく，患者と会話できるコミュニケーション機能や患者の様態情報も取得できる機能を具備させたシステム構築が必要である．
情報システムによる在宅システムとの連携は単に遠隔診断にとどまらず，一人暮らしの高齢者への安心感を提供するシステムや，ヘルパーによる在宅での介護支援，訪問看護支援などもその範ちゅうに含まれる．
ここでは，遠隔地からの診断を支援するためのインフラストラクチャとしての家庭内情報ネットワークの種類とその在宅診断への応用，安心感通信について述べる．
末の増加，従来の冷蔵庫，電子レンジ，エアコン，洗濯機などの白等により，それらの情報端末間を接続させる必要性から検討されてェイからインタネットに接続され，ネットワーク外部からのエアコップなどが可能になる．
療養中の患者を遠隔地から診断し，介護支援するシステムのインフを担う．
在宅での医療を支援するシステムとして，在宅診断支援，ンサ系などには大変効果的なネットワークシステムである．
b.高速LAN，無線LANなどの利用 15･2･2項で紹介した各種構内ネットワークは，ホームネットワークとしても利用可能である．
特に無線LAN環境はa項で述べたHomePlugと同様に，各情報端末から有線によるネットワークアクセス回線を必要としないという大きな特徴がある．
無線LANについてはすでに15･2･2項にて述べているため，ここではホームネットワークに利用可能な，無線LANとは異なる他の無線方式と高速ネットワーク技術について述べる．
バイルPCまたは通常の固定PC，携帯電話，その周辺機器間の短距離間（数mから10m程度）を無線で接続する技術である．
これにより，例えばいままではPCとプリンタ間をUSB（universalserial bus）ケーブルなどで接続していたのがワイヤレスで接続可能となり，プリンタが必要なときは，15･1，15･2節で紹介したホットスポットサービス提供環境などで容易に接続できることになる．
また，ディジタルカメラやPDA，携帯電話などとの接続も容易で，カメラの画像を簡単にネットワークに流すことができる．
現が検討されている．
本技術は，1994年にエリクソン社の社内プロジェクトとして，あらゆる携帯端末どうしを無線で接続する目的のもとに開発がスタートした．
1998年には通信業界，コンピュータ業界大手の5社（IBM，インテル，エリクソン，東芝，ノキア）がBluetooth SIGを設立し，今日に至っている．
技術資料が公開され，ライセンス料無料で利用できるオープンな規格であるため，現在，2500社を超える企業，団体が参加している．
