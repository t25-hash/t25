# 11・3・4 共通化・標準化の必要性

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・3 共用品・アクセシブルデザイン

不便さ調査の最終的な目的はその不便さを解決することであるが、その場合、可能な限り異なる角度から考えることが必要となる。視覚障害者の意見のみを聞いてそれだけを配慮すると、聴覚障害者、車いす使用者、高齢者等がいままで以上に使いづらくなってしまう可能性がある。そのような事態を避けるためのポイントは、目標とする便利さをより多くの人たちが使える位置に据えることである。その位置はそれぞれの製品によっても異なるが、工夫を続けることでその時点での妥当な位置は必ず見つかるはずである。不便さの解決の際にもう一つ重要なこととして、共通化が挙げられる。11・3・3項で述べたシャンプー容器のきざみも、もしそれぞれの会社が思い思いに工夫を凝らし、ある会社がリンスのほうにきざみを付けてしまったら、せっかくの工夫も意味がなくなり、逆にいままで以上に不便になってしまう。
