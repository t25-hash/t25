# 11・3・6 JIS Z 8071:2003からの発展

> **章**: 第11章 バリアフリーとユニバーサルデザイン
> **節**: 11・3 共用品・アクセシブルデザイン

JIS Z 8071:2003が制定された2003年6月,この分野に関する提言書「高齢者・障害者への配慮に係る標準化の進め方」が日本工業標準調査会(JISC: Japanese Industrial Standards Commit-tee)の消費者政策特別委員会から出され,政策の方向性が示された.提言書を作成するにあたり同委員会は,高齢者・障害者を含む消費者団体および幅広い業種にわたる工業会,研究機関等200機関に依頼し,高齢者・障害者への配慮について標準化が必要な分野を調査し,その中から優先順位の高い40テーマを同提言書の中で示している.そのうち2004年度に,「住宅設備機器」,「包装・容器―危険の凸警告表示―要求事項」,「視覚表示物―年代別相対輝度の求め方及び光の評価方法」,「情報通信における機器,ソフトウェア及びサービス―第3部:ウェブコンテンツ」の4テーマが,また2005年度には「情報通信における機器,ソフトウェア及びサービス―第4部:電気通信機器」,「視覚表示物―年齢を考慮した基本色領域に基づく色の組合せ方法」,「点字表示原則及び点字表示方法―9 ―186医療・福祉・バイオ機器公共施設・設備」がJIS に制定された.そして,2006年度には,「包装・容器―触覚識別表示」,「包装・容器―使用性評価方法」,「触知案内図の情報内容及び形状並びにその表示方法」,「公共トイレにおける便房内操作部の形状,色,配置及び器具の配置」の4テーマが新たにJIS に制定された.2003年に提言書が出されてから2007年まで,アクセシブルデザイン関連のJIS は28テーマとなる(表11・5).また,提言書の中に書かれているアジア諸国との連携に関しては,中国,日本,韓国で2002年から始まっている東亜標準化協力セミナーにおいて,中日韓アクセシブルデザイン委員会が同セミナーのもとに発足し,ISOへ共同でNWIP(New Work Item Pro-posal,新規作業項目提案)を提出すべく検討を重ねた.その結果,2007年1月から2月にかけ,3箇国共同での提案書を該当するTC(technical committee)/SC(subcommittee)に提出した.投票結果は,提出した5テーマすべてが承認され,それぞれコンビナーおよびプロジェクトリーダーを日本が引き受けることになった.五つのテーマの担当TC/SC/WG(working
