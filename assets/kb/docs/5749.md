# 15・5・5 在宅医療システムの課題

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・5 ホームネットワークを利用した在宅システムと

在宅医療システムでは，画像によるコミュニケーション手段が必
要条件であり，高速のアクセスネットワークが必須である．ホーム
ネットワークはアクセスラインの高速化が普及した後に，各家庭で
検討されるべきシステムであろう．通常，各家庭には電話線と100
Vの電灯線は配線済みであるが，それに加えて情報ネットワーク
ブロードバンドネットワークとホームネットワークを利用した遠
隔診断における課題は，やはりセキュリティであろう．無線LAN
は，設定時にセキュリティに配慮しないと，無線であるがゆえ情報
が容易に戸外にも出ていってしまう．安易な設置が自らのプライバ
シーの暴露につながってしまう．特に，医療情報は個人の最大のプ
ライバシーの一つであり，その安全性は必ず担保されなければなら
ない．医師との会話による音声情報や画像情報もセキュリティの対
象になりうるものと考える．
また，在宅医療システムでは，高齢者や要介護者の使用が多いこ
とを考慮すれば，そのシステムの使用しやすさは大きな課題であ
る．15･9節でも述べているが，将来的には，端末はユビキタス性
を有し，ユーザが意識しないで病院や診療所と同じように自然な形
で医師や看護師，保健師，ヘルパーと接することができるシステム
が理想である．
