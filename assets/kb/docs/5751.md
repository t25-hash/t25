# 15・6・2 遠隔医療システムに関する今後と問題点

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・6 地域医療と遠隔医療システム

わが国における遠隔医療システムは，1970年代の初めに稲田ら
が実施した和歌山県山間部における実験に端を発する．当時は，画
ている．
像をはじめとする情報伝送など技術的制約が大きかったため実験に
終わってしまったが，この30年ほどの間におけるコンピュータを
はじめとする情報技術，あるいは広帯域通信回線や衛星通信といっ
た通信技術などの各種技術の進歩と，冒頭に述べた高齢化の進展な
どによる医療機関連携のような地域医療に関するニーズの拡大は遠
隔医療の実現を促し，その試行や実用化が進められつつある．した
がって，今後は遠隔手術といった高度のものについても，国内はも
ちろん国際的にも遠隔医療システムの開発と利用がなされることが
期待される．しかしながら，こうした遠隔医療が広く普及するため
には，まだ解決すべきいくつかの課題が存在する．ここでは技術的
課題以外のものも含めて，このような諸課題を表15･6に掲げ（詳
細は割愛），本節の結びとする．
