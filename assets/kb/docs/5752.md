# 15・7・1 はじめに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・7 遠隔リハビリテーション

近年，わが国では急速な少子高齢社会の進行により，保健，医
療，福祉の連携システムの構築が課題になっている．家族構成は二
世代同居世帯の率が低下し，高齢者世帯，一人暮らしの高齢者の増
加，老老介護など介護力の低下が大きな社会問題となっている．一
方，情報通信技術の革新，進歩により，大容量の音声や，画像の双
方向通信が高速で可能となっている．
これらの状況から，高齢者や障害者の身体機能の回復，日常生活
動作能力の向上，介護者に対する介護指導など生活の自立を目指し
た取組みが必要となっている．こういった事情を反映し，病院，医
療機関以外の遠隔地でもリハビリテーション診療を受けられる遠隔
リハビリテーション（rehabilitation distance）の構築について述
べる．

供し，身体機能の回復，日常生活能力の向上を図り，生活の自立を

図15･ 17遠隔リハビリテーションシステム
このシステムは病院，医療機関に設置されるセンター装置と，患
者宅，保健センターに設置される端末装置から構成されている．セ
ンター装置と端末装置はNTTが提供するISDN回線（INS64）
で接続され，双方向画像と音声および端末側から生体医療情報デー
タの通信を行う．また，センター装置は回線を経由して複数の端末
装置と接続され，任意に選択された端末装置との通信が可能となっ
このシステムにさらに，(株)日立製作所が独自に開発した2ベル
ト型歩行訓練機を連携させ，身体機能の回復を図るのが遠隔リハビ
リテーションシステムである（図15･17) ．
この歩行訓練機の特徴は，歩行能力に応じた歩行条件を自動的，
他動的に設定できることである．また，歩行面が2ベルトになって
おり，片麻痺患者等，左右下肢の能力差に応じて最適の歩行条件が
設定でき，歩容改善訓練を行う．テレビモニタからは，歩行状態の
観察，評価ができるように，歩行速度，距離，歩数，左右の重心の
移動，左右下肢にかかる力の表示が可能であり，また，各種生体情
報の表示，データの保存，再生ができる．これらにより，遠隔地の
病院，医療施設から送られてくる映像，生体情報，音声会話による
適切で効果的な遠隔指導，リハビリテーションが可能である．
このシステムは，ブロードバンドインタネット回線で送受信可能
であり，生体情報としては，各種センサを付けることにより，血
圧，脈拍，心音，呼吸，酸素飽和度，体温等のバイタルサインをは
じめ，歩行速度，バランス，患部の状態，動作の指導，確認が可能
である．また医師，理学療法士，訪問看護師，ホームヘルパー，訓
練者，家族等が双方に会話をしながら意思疎通を図り，訓練を実施
することが可能である．センター装置では，リモートコントロール
カメラをコントロールしながら，またマイクを通して会話をしなが
ら指示を与え，訓練状況を確認し，指導することができる．生体，
訓練情報データはデータベースに記録保存することで訓練効果の判
定，訓練プログラム設定に利用できる．
