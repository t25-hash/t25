# 12・2・1 概説

> **章**: 第12章 健康工学
> **節**: 12・2 生活用品・リラクゼーション機器

現代の少子高齢社会においては,高齢者一人ひとりが精神的に,かつ身体的にも自立できることが望ましい.人が活力のある寿命(活力寿命)を延伸していくためには,歩行や自転車こぎなど日常生活における基本的な動作能力を保持することが不可欠である.生活の質(QOL)を良好に保持し,日常生活を快適に送るための手段として,運動,栄養(食事),休養(リラクゼーション)が挙げられる.ここでは,初めにエルゴメトリーの代表格であるトレッドミルエルゴメータと自転車エルゴメータについて詳述する.なお,エルゴメトリーとは人の作業能力(作業効率)を測定する方法を指すもので,一般に単位時間当たりの機械的仕事量が求められる.これらの装置を利用すると,作業(運動)に伴う生理学的な現象を測定しながら,定量的な作業量(運動の量)を人に与えることができる.次いで,パワーアシスト自転車(power assisted bicycle)について説明する.ここで紹介するパワーアシスト自転車とは,脚力の弱い人でも坂道を駆け上ることができるように設計された生活支援機器の一つである.そして,ストレス解消に利用できるアミューズメント機器について,さらに食物のエネルギーや運動によって消費するエネルギー(カロリー)の計算方法などについて解説する.
