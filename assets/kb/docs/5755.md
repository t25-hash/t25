# 12・2・2 エルゴメトリー

> **章**: 第12章 健康工学
> **節**: 12・2 生活用品・リラクゼーション機器

エルゴメトリーとは人の作業能力(作業効率)を測定する方法を指すもので、一般に単位時間当たりの機械的仕事量が求められる。このエルゴメトリーに利用する装置には、トレッドミル(歩く・走る装置)、スイムミル(泳ぐ装置)、クライムミル(登る装置)、ステアミル(階段装置)、固定式自転車、ローイングマシン(こぐ装置)などがある。これらの装置を利用すると、作業(運動)に伴って変動する生理学的・人間工学的な現象を測定しながら、定量的な作業量(運動量)を人に与えることができる。本書では、エルゴメトリーの代表格であるトレッドミルエルゴメータと自転車エルゴメータについて解説する。
