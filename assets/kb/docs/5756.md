# 12・2・3 トレッドミルエルゴメータ

> **章**: 第12章 健康工学
> **節**: 12・2 生活用品・リラクゼーション機器

a. 機器の種類トレッドミルエルゴメータ(treadmill er-gometer、以下、トレッドミル)にはおもに運動負荷テストに利用される電動式のタイプと、自力でゴムベルトやパイプローラの上を歩いたり走ったりする自走式(自歩式)タイプがある。低体力・有障害によって運動量が著しく制限されている場合には、自分の体力に応じて運動の量を調節できる後者の自走式(自歩式)タイプを利用するのもよい。電動式タイプは研究用やリハビリテーション用に使われるもので、その走行面(接地面)には1枚のゴムベルトを用いるものと多数の特殊な弾力板をつなぎ合わせたキャタピラ方式のものがある。キャタピラ方式のベルトは左右へのぶれが少なく、走行面の安定性に優れており、接地時の衝撃を柔らかく吸収する工夫がなされているが、反面、低床トレッドミルを実現することは構造上難しい。b. 機器の概要電動式のトレッドミルは、回転するベルトの図12・17 赤外CCD カメラによる呼吸運動のモニタリング上を人が回転方向と逆方向に走る(歩く)ことにより、運動中の生理的反応を容易に定量することができる代表的な運動負荷装置である。そのほとんどは、ベルトの速度と傾斜角を自由に変えることができるように設計されており、負荷量を最小限から最大限まで段階的に与えることができる。最近では、コントローラ部分に各種の負荷プロトコルを組込んだものや、経過時間・心拍数が表示されるもの、さらには心拍数に応じてベルトの速度を調整するシステムを組込むなどの工夫がなされている。c.
