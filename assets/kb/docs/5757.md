# 12・2・4 自転車エルゴメータ

> **章**: 第12章 健康工学
> **節**: 12・21 )。

a. 機器の概要自転車エルゴメータ(bicycle ergometer)は、機械的あるいは電気的に制動のかかる自転車で、運動負荷テスト装置としては簡便で場所をとらない利点を有している。負荷量は制動力と回転速度の積であり、単位はW ないしkpm/min[欧米や日本で使用、kp(キロポンド)]で表す。b. 機器の構造エルゴメータは、ペダルおよび負荷ユニット1マルチファンクションキー、2リセットキー、3表示切換キー、4ストップキー、5ピッチ音キー、6負荷キー、7スタートキー、8プリンタ関連キー図12・24 負荷ユニット機構のある本体部(図12・22)と機器をコントロールする操作部(図。 負荷ユニットの機構負荷ユニットはクランク軸に固定したロータと、適当な間隔をおいて設けた電磁コイルとの相対運動により、ロータ面に誘起される渦電流によって制動力を得る(図この装置の特徴は、負荷を発生する部分が非接触であるので耐久性に優れていることと、電気的に制御が可能で負荷値を設定できること、また渦電流を発生するロータ部分がフライホイールとして作用するのでスムーズなペダリングが可能になることである。c. 留意点負荷の調整には、ベルトの摩擦やおもりを用いて機械的に行う機械式と、電磁ブレーキによって電気的に行う電磁式がある。機械式では摩耗などにより負荷値が変化してしまうためキャリブレーションが必要となる。一方、電磁式は耐久性に優れており、負荷設定などがコンピュータ制御される。肥満者や高齢者のトレーニング機器として使用する場合には、体重が負荷として直接に膝関節や下肢の筋にかからないよう仰臥位で使用するものを選び、有酸素性トレーニング用や有酸素性能力測定用として用いる場合は座位で使用するものを選ぶとよい。また、体力測定法などのプロトコル面で何種類かの方法が存在するため、体力測定を行う場合は同一プロトコルで統一するべきである。
