# 12・2・5 パワーアシスト自転車(power assisted bicycle、電

> **章**: 第12章 健康工学
> **節**: 12・24 )。

動アシスト自転車)従来の自転車では、乗り手のペダル踏力のみによってその推進力が生み出されている。そのため、自転車を安全かつ快適に利用するには、発進時および加速時に必要となる慣性力、登坂時に作用する重力の傾斜方向成分、および走行時の空気抵抗力等の力を生み出す安定したペダル踏力を、乗り手が発揮できることが必要となる。しかしながら、乗り手の体力や脚力が乏しい場合、および上り坂のこう配が大きな場合には、安定した走行に必要なペダル踏力を発揮することが困難となるため、走行時によろけやすいこと、長距離走行および登坂走行が困難になることなどの諸問題が生じ、その行動範囲が限られることになる。パワーアシスト自転車は、モータをアクチュエータとする電気の力によって、ペダル踏力に応じた推進力を補助するものであり、ペ図12・26 ステッピングセレクション9 ―19 5ダル踏力不足に起因する上述の問題点を解決する一つの策として考案された。図12・25にその構成図を示す。クランク軸部に設けられたペダル踏力センサにより乗り手の踏力をセンシングし、同時にシャフト部等に設置された角速度センサにより現在の車速を測定し、コントローラではこれらの情報をもとに補助推進力を算出し、電流制御型モータのトルク指令値とすることにより、推進力のアシストを行う。このときモータによる推進力の補助割合をアシスト比と呼び、その定義式はα=-で表される。ここではクランク回転出力、は後輪駆動出力である。「道路交通法」により、補助推進力は乗り手が発揮した踏力までとなっているため、アシスト比の最大値は1である。これを超えると原動機付き自転車の扱いとなる。また、補助力の適用範囲は走行速度にも依存し、アシスト比は15km/hから減少し、24km/hで零となるように定められている。パワーアシスト自転車ではバッテリ駆動のため、貯蓄電力がなくなった際にはアシスト力が得られなくなるだけでなく、通常の自転車に比べて、アシスト機構部の重量に応じてより大きな車体重量の自転車を扱うことになる。よって、パワーアシスト自転車を快適に扱うためには、蓄電量が多く軽量なバッテリ、および変換効率の高いモータが理想的には必要となる。現在、バッテリにはおもにニッケル・水素電池あるいはリチウムイオン電池が、モータにはDC(direct current)モータが用いられている。
