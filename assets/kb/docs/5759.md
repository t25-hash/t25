# 15・8・4 保健指導支援システム

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・8 健康モニタリングシステム

NTTは，社員の健康管理の一環として，ネットワーク上に従業
員の健康を管理する健康管理センターを設け，健康管理指導を受け
ている社員，もしくはダイエットや禁煙プログラムへの参加を希望
する健常な社員を対象にした健康プログラムを実験的に行ってき
た．従来の保健指導では，直接的な面談による指導を受けるには社
員が直接健康管理センターに出向かなければならず，継続的な指導
が困難であったため，結局病気になってから指導することになり，
健康管理の面や経済的な面で課題があった．本システムでは，あら
かじめネットワークを利用して指導することにより，個人の状態に
応じた指導が可能になり，またユーザ自身も健康管理センターに何
回も通うことなくプログラムに参加できるメリットがある．冒頭で
述べたように，このようなシステムも広義の健康モニタリングシス
テムである．本システムの考え方を図15･25に示す．システム利用
者はネットワークを介して多種の情報を受け取り，リプライを返
す．医療職側では，バイタルデータや体調情報をもとにユーザの健
康状態を把握し，各種アドバイスなどを返答する．このように，ユ
ーザと医療職側が電子メールや電話などによりインタラクティブに
コミュニケーションをとることにより健康への意識が高まり，効果
的な健康，保健指導が可能となる．
本システムのおもな特徴を以下に挙げる．
(１)有所見の内容に沿った問診項目の自動選択
(２)問診内容に基づいた指導内容決定
(３)個々人の可能な生活改善項目の選択
(４)電子メールなどによる継続的なフォロー
(５)個別指導内容のシナリオ化
(６)実施内容のスケジュール管理
(７)行動科学に基づく指導内容の選定
現在，これらの指導内容に基づき，健康管理センターの医師，保
健師をネットワーク上のセンターに配置し，希望する社員による実
証実験を継続中である．
