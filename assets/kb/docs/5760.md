# 12・2・7 カロリー(エネルギー)計算

> **章**: 第12章 健康工学

a. 食物からの摂取エネルギーとは食物のエネルギーは,一般的に熱量またはカロリー(calorie)という尺度で表される.1kcalは1000calのことで,1calは1g の水を14.5°Cから15.5°Cまで1°Cだけ上昇させるのに必要となる熱量を指す.1kcalは4.184kJ(kilojoules)に相当する.糖質,脂質,タンパク質の三大栄養素を試験管内で燃焼させると,1g 当たりそれぞれ4.1,9.3,4.1kcal(窒素成分を差引いた値)のエネルギーが発生する.生体内では食物の消化吸収率が若干異なるため,それぞれの栄養素1g 当たり4,9,4kcalがエネルギー計算に用いられる.各栄養素が酸化される量を一定にして比べると,脂肪は糖質よりも2倍以上のエネルギーをもっていることになる.生体のエネルギー代謝量は,直接的には完全に断熱材で包囲された小さな部屋の中を循環する水温の変化から放熱量を測って計算される(カロリメトリー,calorimetry).しかし,一般的には生体内での酸化に必要となる酸素(O)量,その結果として発生する二酸化炭素(CO)量,それにタンパク質の異化によって尿中に排出される窒素(N)量から,生体内で燃焼された栄養素の分量比を計算して間接的に求める方法がとられている.b.
