# 15・8・5 今後の課題

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・8 健康モニタリングシステム

健康モニタリングシステムは，毎日変化するバイタルデータや食
事，運動状況などを自動的に，もしくはユーザが定期的に入力する
ことによりその状態を管理することができるシステムである．ここ
では，これらのシステムにおける課題について述べる．
a.バイタルセンサと情報転送ネットワーク バイタルセンサ
を使用してデータを入力するシステムの場合，使用するバイタルセ
ンサの大きさや電池寿命，体に直接取付けた場合の不快感など，バ
イタルセンサ自身に起因する課題が存在する．健康モニタリングに
使用されるバイタルセンサ類は，バイタルセンシングするセンサ機
能部と，センシングしたデータをネットワークを介してデータセン
ター等に転送するネットワーク機能部に分類できる．センサ部の詳
細は4･2節にて述べられており，ここでは高齢者などが違和感なく
使用できるための課題を列挙する．
(１)小型化
(２)電池寿命
(３)ユーザが意識しない装着の仕方
(４)プライバシーへの配慮
次に，ネットワーク部については，これらのセンサにより取得さ
れたデータをどのように転送するかによってその構成が変わる．ネ
ットワークインフラストラクチャとしては以下のものが考えられ
る．
(１)携帯無線ネットワーク
(２)ホームネットワーク（家庭用無線LANに直接送信，もしく
はバイタルセンサから微弱無線により中継器を介して家庭用
有線LANに送信）
(１)は家の中だけではなく，外出中にもモニタリング可能であ
る．しかしながら，データ発生時に携帯電話網に発呼することにな
り，データ転送のたびに課金される．(２)は家庭内の使用に限定さ
れるが，装置の小型化が可能となる．また，インタネットなどに常
時接続していることを前提とした場合，モニタリングデータの転送
のたびに課金されることはない．逆にいえば，常時接続しているた
め，いつでも高齢者などの状態を把握することが可能であり，上述
したプライバシーの面を十分に考慮したうえで，効果的なモニタリ
ングが可能となる．
b.バイタルデータ以外のモニタリング系 バイタルデータは
各種センサにより自動的に取得可能であるが，高齢者の状態情報や
健常者の食事内容の情報などを取得するには，画像による認識や各
種状態情報取得センサとの融合が必須である．例えば，高齢者の状
態を把握するために，電気ポットの使用状況をモニタリングしてそ
の回数をセンターに送出し，遠隔地に住む家族などが携帯電話など
でセンターに確認することで，その使用状況から高齢者の状態を判
断するシステム（みまもりほっとライン) があるが，毎日の電気
ポットの使用を前提としているため，暑い夏場で使用しなかった場
合はどうなるのかなどの問題があり，一つの情報のみを状況判断の
材料とすると誤認識の要因となる．別の状態把握手段として，カメ
ラからの画像情報により高齢者の行動パターンや食事の内容，量な
どを判定し，状態情報や食事内容データとする方法も考えられる
が，これも画像データのみで判定すると誤差を生じる．このよう
に，環境から所望の状況を取得するためのデータ取得系に関して
は，複数のセンサを使用し，それらのデータから総合的に判断する
アルゴリズムが必要である．
