# 12・2・8 電動歯ブラシ

> **章**: 第12章 健康工学
> **節**: 12・24 ).

電動歯ブラシ(electric tooth brush)は,駆動機構と動力源などを内蔵する把柄部と,把柄部に着脱できる頭部より構成され,歯面の清掃や歯肉のマッサージを目的とするものである.頭部の形態,運動様式,運動の速度,動力源の違いによりさまざまなタイプに分かれる.電動歯ブラシの構成を図12・28に示す.a. 頭部の形態頭部のはけ部の形態は,手用歯ブラシ様のもの,円形のものが代表的である.これに加え,歯間を磨く,歯周ポケットを磨くなどの特別の目的に絞った特殊な形態のものが存在する.b. 運動様式頭部の運動様式には,往復運動,アーチ型運9 ―19 8医療・福祉・バイオ機器動,反復回転運動,振動運動,複合運動がある.運動様式の種類を図12・29に示す.往復運動は,頸部と平行な方向に動く.アーチ型運動は,頸部の軸を回転させて動く.反復回転運動は,頭部が円形のものと組合わせてはけ面が円を描くように回転する.振動運動は,はけ部が微振動する.複合運動は,上述の運動が組合わされた動きをする.また,近年では,頭部の運動と同時に,頭部より超音波を発生させる方式や,微弱電流を発生させる方式も存在する.c. 運動の速度機種によりさまざまであるが,1000~4000minで動くものが多い.機種によっては運動の速度の切換機能を図12・28 電動歯ブラシの構成(a) 往復運動(b) アーチ型運動(c) 反復回転運動(d) 振動運動(e) 複合運動図12・29運動様式・もみ上げ・もみ下げグイッと押し,クイッと締め,スッと引く.微妙なリズムの変化によって筋肉をしっかりほぐしていく.・ぐいもみこりのポイントをこねるように反転しながらグイッともみ込んでいく.・ソフトもみ少しずつ反転しながら,徐々に刺激が強くなっていく.筋肉を優しくほぐす.・指圧両手の母指がこりのポイントをジワッと押していく感覚.・ソフト指圧左右交互にリズミカルにこりのポイントを押していく.(a) 指圧もち,複数の速度で運動することのできるものもある.また,近年,15000min付近で高速運動するタイプも存在する.d. 動力源充電式と乾電池式がある.充電式は,電源につながっている充電機能をもつスタンドに歯ブラシを立てかけて,把柄部に内蔵されている蓄電池を充電してから使用する.近年では,電磁誘導によりスタンドと歯ブラシを電気的に接続することなく充電を行う無接点充電方式が採用されている.乾電池式は,把柄部に乾電池を装着して使用する.
