# 13・1・1 感性(kansei,sensibility)

> **章**: 第13章 感性・情緒工学
> **節**: 13・1 概説

個性の時代といわれて久しい。高度経済成長を遂げた1970年代以降における大量生産は、同じ物を大量に生産することで低コストを実現し、家庭用電気製品をはじめとするさまざまな工業製品の一般への普及を促進した。しかし社会が豊かになると人々は自分だけのライフスタイル、デザインを求めるようになり、他人との差別化を図って工業製品においても独自性や個性を求めるようになってきた。このとき、人々が物に対して抱く価値観の源となるのが感性である。さらに、美術工芸品、建築のデザイン、音楽など美的センスの基準となるのも感性である。インタフェースの観点から説明すれば、人間が外からの刺激(入力される情報)を受けて反応(出力する情報)する際の評価基準が感性であるといえる。感性工学は、感性や感覚の計測と定量化、快適性、官能検査、認知、脳、揺らぎ、複雑性、インタラクションなどのテーマを含み、機械工学、人間工学、情報工学、認知心理学、脳科学など多分野を含む学際領域である。適用範囲も広く、アパレル、食、教育、音楽、メディア、工業デザインなどが挙げられる。ただし、感性工学に関する一般理論はいまだ確立されておらず、実証的研究としては、上述したテーマの中では快適性が主として注目されているのが現状である。感性に対応する英語として最も近いのが感受性を示すsensibilityであるが、日本語の感性は感覚のみならず価値観も含むため適切な英語訳がなく、英語でもkanseiと表現され世界的に使用されている。また、感性を専門とする学会である日本感性工学会(JapanSociety of Kansei Engineering)が1998年に設立され、この学問領域は着実に拡大しつつある。本章では、感性工学の代表的適用例として足にフィットする靴、顔にフィットする眼鏡、着心地、座り心地、触り心地を取り上げて、感性工学を具体的に紹介する。
