# 15・9・5 その他の位置情報取得システム

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

上述したように，GPS内蔵型携帯電話，PHSそれぞれ位置検出介する．
同程度の位置精度を有するシステムとして，小型送信機を多数配備提案を行っている．
本システムの位置付けは，図15･29に示すしている．
具体的な利用形態としては，徘徊高齢者のみではなく，図15･30に提案している簡易マーカ方式を示す．
マーカは送信機能その受信レベルから自己の位置を推定する．
ユーザ端末はマーカにした位置情報交換も可能となる．
これらの提案方式は，試作品を通b.微弱電力無線による位置取得法 上述してきた手法はいる．
しかし，ローカルな施設内に設置した各種センサにより徘徊高てもその周辺にいるであろうとの仮定のもとに，微弱無線を使用されている．
これらの機器は，最初はトンネル内や工場内での作業り，徘徊高齢者への適用をも目指したものである．
その特徴は以下腕時計型で携帯に便利呼出し機能により，利用者を呼び出すことが可能サービスエリア内での個人の特定が可能機能に特徴がある．
以下にこれらの方式以外のシステムについて紹a.簡易マーカによる位置取得法 NTTでは屋内でもGPSとする複数マーカ方式により高精度の位置情報を取得するシステムのように，ある領域ではGPSよりも高精度な位置情報を提供可能と美術館やデパート，大規模商品展示会などでの利用が考えられる．
のみを有しており，ユーザはマーカから発信される信号を受信し，対する受信機能と公衆端末との通信機能の両方を有し，公衆網を介して評価，検証実験が行われている．
ずれも，最終的には公衆網を介して位置情報を取得する方法であ齢者の出入りを把握し事前に検知する方法や，出口から出てしまっし，半径数百m程度での位置を検出可能にするシステムも実用化員の位置検出用に開発されていたが，機器の小型化・低価格化によのようである．
外部アンテナにより受信エリアの調整が可能設置地域の出入り監視が可能この方法は，特定のエリア内であれば有効な手段といえる．
