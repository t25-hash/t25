# 13・1・2 情緒

> **章**: 第13章 感性・情緒工学
> **節**: 13・1 概説

情緒も感性と同様に日本独特の言葉である。以前は感性工学を情緒工学と呼んでいた時期もあったが、現在では異なる使い方がされている。英語では感情をemotion,affection,feeling,moodなどさまざまな語で表現するが、情緒に合致する語は見当たらない。一般的な用語であるemotionは感情の急激な変化を表し、情動という訳語が適切である。それに対して情緒は感性に近く、感性に加えて感情的表現も含んだ語である。むしろ心(mind)の意味に近いといえよう。感性に基づく人間の行動(身体の運動や顔の表情変化など)を外部から観察したときに、観察者が感じる被観察者の心の様相・変化の総体を情緒と呼ぶことができる。情緒は、感性と同様に近年の個性重視の中で、個人へのサービスが中心となるエンタテイメントおよびアミューズメント分野で特に注目されている。テレビゲームや癒し系のおもちゃなどは、個人の情緒に作用する機能を付加することで感情移入を促進し、人間との親和性を高めている。しかし、その方法論は確立されておらず、試行錯誤が重ねられている。この情緒に関する学問である情緒工学もいまだ模索の段階にあり、さまざまな研究が多方面で行われている。そこでは、機械工学や人間工学における情緒や感情をベースにしたコミュニケーションとインタラクション、脳科学や人工知能における学習に作用する情緒[生理学的(physiological)な意味での自律系]の機能がテーマになっており、心理学、感情社会学、哲学などにおいても同様の関心がもたれている。ただ、情緒は定量化が難しいため、機械の設計においてはその効果をどのように評価するかが課題となっている。現状では、感性と同様に、実験とアンケートを組合わせた実証試験と統計的解析が情9 ―201緒の唯一の評価方法となっている。本章では、情緒ロボットの代表であるメンタルコミットロボットを例として、情緒の工学的アプローチを紹介する。
