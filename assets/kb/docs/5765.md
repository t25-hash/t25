# 13・1・2 情緒

> **章**: 第13章 感性・情緒工学
> **節**: 13・1 概説

情緒も感性と同様に日本独特の言葉である．以前は感性工学を情
緒工学と呼んでいた時期もあったが，現在では異なる使い方がされ
ている．英語では感情をemotion，aﬀection，feeling，moodな
どさまざまな語で表現するが，情緒に合致する語は見当たらない．
一般的な用語であるemotionは感情の急激な変化を表し，情動と
いう訳語が適切である．それに対して情緒は感性に近く，感性に加
えて感情的表現も含んだ語である．むしろ心（mind）の意味に近
いといえよう．感性に基づく人間の行動（身体の運動や顔の表情変
化など）を外部から観察したときに，観察者が感じる被観察者の心
の様相・変化の総体を情緒と呼ぶことができる．
情緒は，感性と同様に近年の個性重視の中で，個人へのサービス
が中心となるエンタテイメントおよびアミューズメント分野で特に
注目されている．テレビゲームや癒し系のおもちゃなどは，個人の
情緒に作用する機能を付加することで感情移入を促進し，人間との
親和性を高めている．しかし，その方法論は確立されておらず，試
行錯誤が重ねられている．
この情緒に関する学問である情緒工学もいまだ模索の段階にあ
り，さまざまな研究が多方面で行われている．そこでは，機械工学
や人間工学における情緒や感情をベースにしたコミュニケーション
とインタラクション，脳科学や人工知能における学習に作用する情
緒［生理学的（physiological）な意味での自律系］の機能がテー
マになっており，心理学，感情社会学，哲学などにおいても同様の
関心がもたれている．
ただ，情緒は定量化が難しいため，機械の設計においてはその効
果をどのように評価するかが課題となっている．現状では，感性と

13・2・2顔にフィットする眼鏡の開発
眼鏡フレームにもサイズがある．ただし，靴や衣服のようにJIS
規格化されているわけではなく，あくまでも業者独自のサイズ規格
である．また，実際に眼鏡販売店でフレームを購入する際に，別の
サイズを試してみることを勧められた経験はないし，サイズがある
ことを知らない消費者も多いと思われる．これは，眼鏡フレームそ
のものが店頭で消費者個人の顔形状（face form）に適合するよう
に調整できること，そもそも適合性よりも意匠デザインやブランド
が重視される製品であることによる．その一方で，意匠デザインや
新材料だけに頼らずフィット性を前面に打ち出した商品が現れてい
る．
図13･2足形状の個人差に基づく靴型設計
された空間格子を量産靴型に適用することで，コンピュータ上で個
人に適合する靴型を作り出すことができるようになる．
ただし，靴型そのものを個人用に設計し削り出すのは，コストの
面から現実的ではないという考え方もある．特にスポーツシューズ
などでは靴底に金型が使われており，この部分の個別修正は極端な
コスト高につながる．そこで，スポーツシューズでは，個人の足底
形状に応じてインソールを製作する装置が開発されている．
これらの装置はいずれも，従来，靴型職人やシューフィッターが

ジタル化された足形状情報をネットワークで通信することで，適合

ほとんどの眼鏡フレームは，鼻根部と耳で眼鏡を顔に固定する設
計となっている．国内大手メーカの調査によれば，ユーザ苦情の第
1位が鼻や耳の部分の痛みであるという．そこで，これらの部分の
形状を三次元計測し，それに基づいて顧客ごとに適合する眼鏡フレ
ームを製作する装置が開発されており，大きく三つのアプローチが
とられている．第一はフルカスタマイゼーションで，顧客の顔寸
法・形状に応じて個別にフレームを製作する方策である．意匠デザ
インへの対応も多様である．ただし現状では，金型を必要としない
プラスチックフレームに限られている．第二は，鼻当てや耳当ての
部分のパーツを複数個用意しておき，それを顧客個人の三次元形状
データに基づいて選択する方策である．店頭に設置された専用機械
で鼻や耳の部分の寸法や位置関係を自動的に計測し，それに応じて
最適なパーツの組合せで眼鏡フレームを構成する．パーツは量産品
であるので金型の使用も可能であるが，特殊な仕様であるため価格
が非常に高くなってしまうのが難点である．
第三のアプローチは，対象となる顧客集団の顔形状分布を数学モ
デル化し，それを効率よくカバーするように量産品を再設計する方

形状を計測しきれなかった．問題は，フレーム設計に欠かせない耳
たぶの上部，後部の形状が，頭髪と耳たぶそのものによって隠れて
しまうことにあった．ただ，多数台のカメラとパターン光プロジェ
