# 13・2・2 顔にフィットする眼鏡の開発

> **章**: 第13章 感性・情緒工学
> **節**: 13・2 感性工学

眼鏡フレームにもサイズがある。ただし、靴や衣服のようにJIS規格化されているわけではなく、あくまでも業者独自のサイズ規格である。また、実際に眼鏡販売店でフレームを購入する際に、別のサイズを試してみることを勧められた経験はないし、サイズがあることを知らない消費者も多いと思われる。これは、眼鏡フレームそのものが店頭で消費者個人の顔形状(face form)に適合するように調整できること、そもそも適合性よりも意匠デザインやブランドが重視される製品であることによる。その一方で、意匠デザインや新材料だけに頼らずフィット性を前面に打ち出した商品が現れている。ほとんどの眼鏡フレームは、鼻根部と耳で眼鏡を顔に固定する設計となっている。国内大手メーカの調査によれば、ユーザ苦情の第1位が鼻や耳の部分の痛みであるという。そこで、これらの部分の形状を三次元計測し、それに基づいて顧客ごとに適合する眼鏡フレームを製作する装置が開発されており、大きく三つのアプローチがとられている。第一はフルカスタマイゼーションで、顧客の顔寸法・形状に応じて個別にフレームを製作する方策である。意匠デザインへの対応も多様である。ただし現状では、金型を必要としないプラスチックフレームに限られている。第二は、鼻当てや耳当ての部分のパーツを複数個用意しておき、それを顧客個人の三次元形状データに基づいて選択する方策である。店頭に設置された専用機械で鼻や耳の部分の寸法や位置関係を自動的に計測し、それに応じて最適なパーツの組合せで眼鏡フレームを構成する。パーツは量産品であるので金型の使用も可能であるが、特殊な仕様であるため価格が非常に高くなってしまうのが難点である。第三のアプローチは、対象となる顧客集団の顔形状分布を数学モデル化し、それを効率よくカバーするように量産品を再設計する方策である。この場合はまず、顧客層を代表する顔形状データベースが必要となる。大量のデータを蓄積するには人体三次元形状計測装置の利用が欠かせないが、既存の装置では眼鏡設計に必要となる顔形状を計測しきれなかった。問題は、フレーム設計に欠かせない耳たぶの上部、後部の形状が、頭髪と耳たぶそのものによって隠れてしまうことにあった。ただ、多数台のカメラとパターン光プロジェクタを適切に配置することでこれらの問題を解決し、耳裏まで計測できる仕様の形状計測装置が開発されている。多数台のパターン光プロジェクタを用いる場合、パターン光の干渉を避けるためプロジェクタの時分割発光が不可欠であるが、これによって計測時間が長くなり、体動揺の影響が出る。しかし、パターン光の工夫によりこれらの問題も解決され、1秒以下で頭部全周が計測できる(図13・3)。このような装置で計測された頭部形状を解剖学的な対応に基づいてモデル化することで、形状間距離を計算できる。多数の頭部形状データについて相互の形状間距離が得られれば、その距離行列から多次元尺度法によりデータの布置(配置)を示すマップを得ることができる。図13・4は、日本人男性の顔形状の三次元マップである。図13・4中にあるように、このマップをいくつかのサイズグループに分割し、それぞれのグループの平均形状を計算した後、それをCAD(computer aided design)に持ち込んで適合眼鏡フレームを設計した。この方策は、基本的に金型を使う量産生産にのっとっ図13・4 日本人男性の顔形状マップているため材料やデザインの制約がなく、価格も普通のフレームと同程度に抑えることができる。反面、完全な個別対応ではないため、極めて特異な顔形状をもつ人に対するフィット性までは保証できない。また、現状では、店頭で顧客の顔形状を計測し、それに応じてフレームサイズを選ぶ、あるいは修正する装置が備わっていない。眼鏡フレームの場合は、主目的である視力補正に眼科医の処方せんが必要であるため、完全な電子商取引には向かない。一方、眼鏡補正を必要とする顧客は、自分の眼鏡を外して別のフレームを試し掛けしたとき、鏡を見てもよく見えないという問題がある。そういう意味では、ディジタル顔形状データを単に形態的なフィット性向上に使うだけでなく、仮想ミラーに顧客の顔をCG 提示するシステムにも利用できる。この場合、顧客は自分の眼鏡を掛けたまま、いろいろなフレームを試し掛けする自分の姿を見ることができる。
