# 13・2・4 座り心地

> **章**: 第13章 感性・情緒工学
> **節**: 13・2 感性工学

a. 座り心地の概念とその数値化椅子に腰掛けたときに人は,ぴったりや心地よいという感覚的体験をする.座り心地とは,椅子の構造(特に座面)と身体特性(体重,動き,腰部・臀部・大腿の三次元形状,下腿高など)といった因子が互いに関係(作用)したときに,その人間がもつ感覚である.座り心地(seat comfort)は,おもに臀部(でんぶ,buttocks)・大腿部が着座により圧迫されることで発生する筋肉内の感覚神経から大脳中枢への伝達信号によって生起される.臀部筋群への圧力は30~50mmHg 程度が快適で,それを超えると不快な感覚となり,70mmHg の圧力で2時間以上圧迫すると組織壊死が起こるといわれている.臀筋は,このような特性をもつ一方で60kg 前後の荷重に耐えることができる強靱な筋組織である.大腿筋への圧力は,増加すると不快な感覚を生じ,快適な感覚を生起することは認められない.また,大きな荷重に耐えることはできない.大腿内部には生理的に重要な血管・神経があり,これを圧迫することは健康上好ましくない.以上のように,座り心地は生理的にも重要である.また,座位姿勢の人体はその体重の90%前後を座面によって支えられるため,座り心地は座面とのかかわりが主となる.したがって,以下ではおもに座面について扱う.臀部で生起した座り心地の感覚は,脊髄中心管内の中枢神経を伝わり大脳中枢で言語表現される.大脳中枢レベルでは視覚的なデザインといった外乱の要素が大きく,座り心地の言語表現はかなりの誤差を含むことになる.座り心地(言語性)の調査ではアンケート調査・官能評価などを行うが,誤差の点で限界があり,身体性ないしは生理的な座り心地の測定と評価が重要となる.座り心地の表現用語と座り心地のよさ(言語表現)との相関が高いものは,座面・背当てのフィット感と,座面・背当ての硬さであった.クラスタ分析から抽出された要素は,サイズの満足度,フィット感の満足度,そして腰椎支持の満足度であった.以下に座り心地の要素を整理して示す.(1) フィット感:大きさ・硬さ・反発力が人体と影響し合い,ぴったり,この椅子は自分に合っているという感覚的体験(上位概念)(2) スケール感:奥行き,座面幅についての感じ(3) 体圧感:座ったときに感じられる反力や弾性座り心地の上位概念として挙げられるフィット感は,臀部の三次元形状と椅子の形状とのフィット感に関係すると考えられ,着座時に臀部への圧力(体圧)を均等化させることがフィット感の向上につながる.フラットな座面と臀部周辺の三次元形状に近似した座面(以下,三次元座面)の比較を次に示す.このような座面をもつ椅子をコンケーブチェア(concave chair)と称する.まず,フラットな座面と三次元座面における体圧の比較を図13・6に示す.被験者は身長174cm,体重64kg の30代男性.フラット座面の場合,背当ての角度105°,座面後傾角度2.5°,二つの体圧分布ともに測定の上限値は220mmHg である.二つの体圧分布ともに最も高い圧力値は,臀部の坐骨結節(硬い骨のでっぱりで,座位のときに体重がかかる部位として知られている)で認められるが,三次元座面のほうが分布の広がりが大きい.これを体圧分散が大きいともいう.図13・6では省いてあるが,背面体圧分布の比較でも三次元座面のほうが接触面積が大きい(活動セル数が多い).次に,これらの体圧分布(pressure distribution)のヒストグラムと統計値を比較してみる.図13・5 人間と衣服の関係図13・6 フラット座面と臀部周辺の三次元形状に近似した座面の体圧分布の比較(測定:岐阜県生活技術研究所)図13・7によれば,三次元座面のほうが接触面積が大きい(活動セル数が多い)ことがわかる.平均圧力値は三次元座面が平板の60%である.b. 体圧分布と座り心地の関係についての実験結果座り心地の構成因子である体圧感は,体圧分布として計測される.体圧分布特性として分布(mmHg)の最大値で示すと,それは図13・8のように着座時間によって変化する.図13・8は,50分間着座中の座面における体圧分布(mmHg)の最大値の推移を示している.実験は,コンピュータ入力作業を行ったときの測定結果の一例であり,20歳代の男女3人に対して行った実験の代表例である.この図では,座り直しの回数の観察によって0~50mmHg の範囲を快適な座り心地のゾーンとしており,50mmHg 以上の最大圧力を示す場合は不快(discomfort)な状態としている.50分間の作業中の特に後半に,それが顕著に発生している.c. 座り心地に関する留意事項. 椅子の構造と身体特性との間の相互作用(安定性と可動性)椅子や床に座ると姿勢反射はほとんど機能しなくなり,姿勢が崩れやすくなる.したがって,座位時の姿勢の安定性を確保することは,椅子の設計上重要である.安定性を高める手段として,臀部のサポート(ペルビックサポート,pelvic support)がある.. 前ずれ着座時に臀部が前にずれ,それにより座面との身体接触面にせん断応力(shear force)が生じることがある.また,結果として腰部から上の姿勢変形を生じる.これらの現象はいずれも不快感を生じるなど,座り心地を低下させる.d. 座り心地の開発座り心地のよい椅子やシートの開発について五つの点を述べておく.. クッションとインピーダンスマッチング臀部の形状に近似した座面を作ることが難しい場合,椅子の形と人体の三次元形状との差を埋めるために化学素材系(ウレタンフォームなど)を椅子の支持部に重ねて座面を構成する.これにより座り心地が向上する.椅子の構造上,クッションの厚みは50mm前後となる.この図13・7圧力値のヒストグラム(比較).二つの統計値の比較表も示す(測定:岐阜県生活技術研究所)図13・850分間着座中の座面における体圧分布(mmHg)の最大値の推移と快適な座り心地のゾーニング9 ―205厚さで着座時の衝撃を吸収し,かつ良質な座り心地を保つことが必要である.クッションの特性は押込み圧とたわみの関係で示され,これをヒステリシス曲線と呼ぶ.座面に接する人の体表面(主として筋群)も一定の押込み硬さをもっているため,多様なクッションから適当なものを選ぶことは,概念的にはインピーダンスマッチングが最大となったとき最大の座り心地が実現されることに相当する.図13・9は,適当に調整したクッションが人体の荷重により圧縮されたときの曲線である.中心部の厚さ50mmのクッションに着座(女性,体重63kg)したときの沈み込み曲線であり,最も沈み込んだ部分は,臀部坐骨結節部に位置する.50mmの厚さのクッションが最大50%圧縮されているが,この場合はマッチングが適当であると評価される.. 人体形状の三次元採型と座面の輪郭(contour)三次元曲面の採型にはいくつかの方法がある.例えばスピンドル式は,身体形状に応じてスライドする測定棒を面状に配列して座面や背面の輪郭を求めていく方法である.. 高機能座面座り心地のよい座面とするためには,座面が適切に人体,特に臀部・大腿部を支持しなければならない.座面には図13・10に示すような四つの機能があるが,これらを満足する座面を高機能座面(HPS : high performance seat-pan)と呼ぶ.図13・10中に示す四つの機能を有する座面は高機能座面である.通常,臀部周辺の三次元形状に近似した形の座面である.. 最小座面顕微鏡下手術執刀医用の椅子や人間搭乗型二足歩行ロボットなど,臀部だけを載せる小サイズの座面の場合,大臀筋のサイズに合わせるとよい.図13・9沈み込み曲線図13・10 高機能座面9 ―206医療・福祉・バイオ機器. 座り心地品質の表示座り心地も品質の一つであり,椅子に表示することが望ましい.(1) 体圧分散(pressure distribution)値:体圧分布測定装置の活動センサ数.この値が増加すると座り心地が上昇する.ただし,座面に特殊加工しているような場合は適用外.(2) 平均圧力値:この値が高いと,長く座り続けていられない.e. まとめ. 座り心地の構造モデルa~d項で述べた座り心地には多様な要因がかかわっており,それらは身体性と言語性の二つに分類される.図13・11は座り心地の構造モデルである.身体性のブロックは臀部に相当する.そこから発した座り心地情報は,脊髄を通り大脳中枢において必要な処理が行われる.この部分が言語性のブロックに相当する.. 高次の座り心地椅子に腰掛けたとき,人は心地よいとか包み込まれる感じという感覚的体験をする.身体が受ける圧力を最大に分散させた座面をもつ椅子に腰掛けると,包み込まれる感じを体験することができる.この感じを英語ではcradle(動詞)という.ウェブスター(M. Webster)によれば,cradleの意味はsupport protectively or intimately(保護的に,あるいは親密に支持する)というものである.包み込まれる感じが座り心地として高次の感覚的体験であることがよく示されている表現である.
