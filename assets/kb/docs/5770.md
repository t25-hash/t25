# 13・2・5 その他

> **章**: 第13章 感性・情緒工学
> **節**: 13・2 感性工学

本章の表題でもある感性という言葉で表現される対象としては,感覚器官でとらえた場合に触覚(皮膚感覚,体性感覚)に関連するものが多いように見受けられる.この一つの理由としては,バーチャルリアリティの技術開発領域で先行している視覚や聴覚といった他の感覚系と比較したときに,触覚(tactile sensation,senseperception)の機序の解明が進んでおらず,それに伴って触覚が処理する情報の体系化が不十分であることが挙げられる.視覚や聴覚の場合には情報源とは非接触で情報を得ることができるため,それぞれ両眼立体視やステレオ音場環境を再現できるディスプレイを身につけたり周辺に配置すれば仮想的に情報空間を再現でき,実用上十分なリアリティを備えたディスプレイを構築することが可能となる.これに対して触覚では,情報を取得する際に受動的ではなく能動的な触動作を伴うことが多く,また力学的な文字どおりの接触現象を伴う感覚系であるために,仮想的な対象物ではなく実在する機械システムによってディスプレイを構築しなければならない.このためには人間の触覚,特に皮膚感覚の受容器によって知覚さ図13・11 座り心地の構造モデル図13・12 点字の違いれる情報に基づき,対象物のさまざまな属性(硬さや柔らかさといった材質感,表面の状態を表すざらつき感やすべすべ感など)や,いわゆる触り心地が認知される機序を明らかにすることがまず必要である.さらにそれらの知見に基づくディスプレイの設計仕様が明らかになれば,視覚・聴覚ディスプレイと組合わせることで,よりリアリティの高いディスプレイの構築が期待できる.例えば応用例としては,医師による触診情報の定量化あるいは遠隔診断治療といった医療への応用も期待できる.本項では,触り心地が皮膚感覚を介した情報の授受に及ぼす影響の一例として,視覚障害者のコミュニケーションツールとして普及している点字(braille)の特徴に関する事例を紹介する.点字は約4×6mmの範囲に3行2列のマトリックス状の6点の有無のパターンによって仮名が構成される.従来から点字は,基本的には厚手の専用紙に道具を使って点を打ち出すことによって製作されてきた.これらは製本され,点字図書として視覚障害者に読まれている.近年,多くの点字用プリンタが開発され実用化されているが,点の断面形状,高さ,間隔等が機種ごとにわずかずつ異なっており,これが触読時の触り心地に影響し,その結果として人により好みや読みやすさが異なってくる.図13・12に異なる点字の典型的な例として,高いおわん型と低い円すい型の例を示す.いずれも一般的に普及している点字打出し装置で製作した点字であり,(a)(b)はそれぞれ点字の写真と非接触の変位センサで測定した断面形状である.図13・12(a)は断面が円弧状で高さは0.4mm以上だが,図13・12(b)は断面が三角形で高さはわずかに約0.2mmである.高いおわん型に比べて低い円すい型は写真で見ても点自体がほとんど目立たないが,実際に触ってみても両者の違いは歴然としている.図13・12(a)の高い点字のほうが触った際に刺激が強く,図13・12(b)の低い点字と比較して点の存在は明らかにわかりやすい.しかし,実際に視覚障害者による触読実験を行って評価すると,必ずしも触読速度が速いのは高い点字とは限らない.高いはっきりした点字は刺激が強く,どの被験者にも比較的安定して識別される.点字触読に十分に習熟していない読者は刺激の強いほうが識別しやすいが,習熟した読者は逆に刺激の弱い点のほうが触読速度が速いという傾向が認められた.低い点字に対する内観報告でも,前者の被験者は刺激が弱くて読みにくいという回答であったのに対し,後者の被験者は刺激が強すぎないため疲れにくく読みやすかったという好意的な回答が多かった.このように点字を触って評価する場合には,同じ対象物でも触る主体によって全く異なる評価がありうるという難しさがある.触り心地といった場合,個人ごとの経験や好み等の主観的な指標に大きく左右されることがこの例からもわかる.ここでは高さの要因に関して述べたが,さらに点間隔や断面形状の因子が加わればますます触り心地や読みやすさに影響を与えることになる.例えば刺激が強(b) 紫外線硬化樹脂製の点字い場合には点間隔が狭いと異なる2点と識別しにくいが,刺激が弱く特にピークが存在する円すい型の場合には比較的2点と識別しやすいといった傾向が認められる.このように人の皮膚感覚特性を考慮したうえで,適正な点字評価の手法を検討していく必要があると思われる.同様に点字の例として,近年普及してきた樹脂製の点字を図13・13に紹介する.前述の紙に打ち出す点字は,型で点字を製作するという点字打出し装置の機構上,高さや間隔を調節することはできず,機種によって固定されてしまう.他方,近年普及してきた樹脂製の点字は,紫外線硬化樹脂を用いたスクリーン印刷の手法によるため,点の断面形状に関しては樹脂インクの表面張力によってほぼおわん型に限定されるが,版のパターンを変更するだけで高さや点間隔を容易かつさまざまに変更することができる.さまざまな点字が製作できるゆえに,よい点字の標準を規定すべきであり,これが樹脂製点字に対するJIS 制定化の理由の一つである.また,樹脂製点字の利点として,紙に点を打ち出すのと異なり点字を付ける材料にバリエーションがあり,紙はもちろん金属やプラスチック,ガラスといったさまざまな材料でできた製品に点字を付けることができることも挙げられる.ところが触り心地の観点からは,樹脂製の点は微細な繊維で構成される紙の点字と異なり,点字表面が滑らかな樹脂で構成されるため,摩擦抵抗が大きく接触面でのせん断方向の刺激が大きいことが指摘されている.また,紙の場合には発汗時にも繊維が吸収して滑りを保証してくれるが,樹脂の場合にはそのまま摩擦抵抗の増大につながり,ますます刺激を増大させることになる.このため,疲労を抑えるために,あらかじめパウダを振り掛けて摩擦抵抗を抑える等の工夫をしている視覚障害者の例もある.このようにさまざまな可能性を秘めた樹脂製点字も,触り心地から評価するとまだまだ検討課題が残されており,人の皮膚感覚受容器とのマッチングに関する研究,およびその知見に基づく樹脂製点字生産関連技術の開発の必要性が指摘されている.
