# 13・3・1 はじめに

> **章**: 第13章 感性・情緒工学
> **節**: 13・3 情緒工学

人間と共存・共生する機械(ロボットなど)には、人間の動作の補助、協調作業など人間に対する物理的な作用を目的にするものと、アミューズメント、エンタテイメント、癒しなど、楽しみや安らぎのような心理的な作用を目的とするものがある。前者の場合、人間の情緒の変化として人間の生理・心理状態をモニタすることにより、人間に対する安全や安心、快適さによる作業効率の向上などに役立てる機械といえる。一方、後者は人間の心や情緒へ作用し、その結果として、その人間からの主観的な評価を受ける機械である。そのため、産業用ロボットのように事前に整えられた環境と機械という閉じたシステムではなく、機械と共存する人間の情緒や未知の環境を考慮する、開いたシステムである。つまり、人間の主観や感性を考慮し、人間と機械との相互作用(interaction)のダイナミクスを考慮した複雑なシステムである。情緒に関する研究は心理学、脳科学など多彩な分野で行われているが、それらを工学的に応用する情緒工学はまだ確立された体系と9 ―207はいえない。また、人間の主観を扱うため、客観性を前提とする科学としての体系化には困難がつきまとう。しかし、また、機械の目的や条件、動作する環境を限定することなどにより、人間と共存・共生する機械を設計し製作するうえで必要な知識、技術、理論を体系立てようとする研究が行われている。本節では、まず人間と機械の関係を分類し、情緒と機械のかかわりについて述べる。次に、人間と機械の相互作用について述べる。そして、情緒工学を応用した機械として、メンタルコミットロボット(mental commitment robot)のパロと人間との相互作用について述べ、最後にまとめる。
