# 13・3・3 人間と機械の相互作用

> **章**: 第13章 感性・情緒工学
> **節**: 13・3 情緒工学

人間と機械の相互作用を考える場合、機械、その設計者、主体としてその機械と相互作用する人間の違いについて考慮する必要がある。まず図13・14のように人間と人間との相互作用においては、両者がそれぞれの意図や価値観をもって言語や非言語によるコミュニケーションを行う。相手の行動・反応の解釈に当たっては、自らの主観的な解釈によって行われる。言語でのコミュニケーションでは、より明確に自分の意図を相手に伝えることが可能であるが、ジェスチャや表情など非言語の情報伝達も非常に重要である。しかし、相互作用し合う人間の文化に共通性がある場合とそうでない場合とでは、情緒の解釈に大きな違いが生じることも指摘されている。例えば、従来、感情は比較的低次の、進化的に固定された皮質下の組織であり、その主要な機能は生物的適応を達成し生態的ホメオスタシスを維持することであると考えられてきた。エクマン(Ekman)は、基礎感情理論として、喜び、驚き、恐れ、怒り、軽蔑、嫌悪、悲しみの七つが含まれ、それに対応した内的、神経生理的構造があると仮定した。しかし、これらの七つの感情は文化的に普遍なものではなく、アングロサクソン文化圏のものであり、例えば東洋における感情の表現や解釈は異なっていることが指摘されている。つまり、人は快・不快の脳機能、接近・回避の反射、闘争・逃避の反射、性的反射など、進化の過程により生物学的に固定された適応のメカニズムを共通してもつものの、感情などの場合には生物学的、生得的メカニズムだけでなく高次の神経機構による認識のプロセスがかかわり、さらには、成長・学習過程においてその人が生まれ育った社会的、文化的な背景が大きく作用している。このことは、主体である人が対象となる人の行動を認識、解釈するうえで、社会や文化に依存した自らの経験や知識が大きな影響を及ぼすことを意味している。次に、図13・15のように人間と自然物(動物、植物、石など)との相互作用では非言語のコミュニケーション(nonverbal commu-nication)が主であるが、人間が話しかけるなどの行為があり、言語でのコミュニケーション(verbal communication)も見られる。自然物でも動物や植物には意図や解釈がありうるが、その内容を詳細に確認することはできず、人間からの刺激に対する反応などを分析して行動や動作の意図を解釈しているにすぎない。しかしながら、情緒や生命感を感じて愛玩の対象とする人間や、動植物との相互作用から自らの情緒を変化させる人間は多い。静的な石に対してさえ、環境の変化(雨や湿度による水分、光、温度の変化など)を反映する場合もあるが、相互作用によって情緒を変化させる人間は図13・14 人間と人間との相互作用存在する。図13・16の人工物には、彫像や絵画のように静的なものや、機械やロボットのように動的なものがある。中間的な存在として庭園や建築物がある。それらの人工物には作者(製作者、設計者など)が存在し、作者の意図や価値観が人工物に埋め込まれている。人間が主体として人工物と相互作用するときには自らの感覚により知覚するが、その解釈においては主観的に自らの経験や知識に依存している。そのため、人工物そのものに依存して解釈・評価される場合と、人工物に加えその作者に関する経験や知識にも影響を受ける場合とがある(芸術品などでは、例えばピカソの絵というだけで高い評価が与えられる)。一般に人間と共存する機械については、おもに前者である。また、作者により人工物に埋め込んである意図や価値観は主体から全く同じように解釈されるのではなく、主体が自らの意図や価値観に基づいて主観的に解釈するため違いが生じる。例えば、作者が情緒を表現するようなロボットを意図して、情緒的な行動やその生成アルゴリズムをモデル化しあらかじめ埋め込んだ際に、それを主体に認識させようとして認識率を議論してもあまり意味がない。これは、作者が作る感情的な行動(モデル)に普遍性がないことと、主体と機械との相互作用の連鎖の中で発現する行動は、主体の意図や経験に基づく連想などによって主観的に解釈されるためである。そのため作者にとって、機械との相互作用の中に、主体が機械の反応や行動に情緒的な意味や意図を見出せるような誘導をある程度埋め込むことは可能であるが、むしろ主体と機械との相互作用の状況やコンテクスト(context)に依存した主観的な解釈が強く働くことや、時間とともに主体と機械とが新たな関係を徐々に構築していくことを想定することが重要である。短期的な相互作用では、作者の意図に基づく機械の情緒的な表現そのものが、主体にどのように解釈されるかが重きをなす場合がある。しかし、長期的な相互作用では、主体は機械について、機械との相互作用を通して、同時期に発生するさまざまなイベントとともに記憶、学習を行う。その途中で、その機械への関心がなくなれば飽きられ、相互作用が終了する。一方、その機械への関心が持続すれば、主体が機械との相互作用の中で連想したり機械との関係を想起したりすることにより、徐々に安定した関係が構築され、機械に対して情緒を感じたり、自らの情緒的欲求を満たしたりするようになり、愛着がわく対象となる。図13・15
