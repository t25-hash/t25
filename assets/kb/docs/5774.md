# 15・2・5 ソフトウェア

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・2 ライフサイエンス実験装置（細胞培養装置）

装置のソフトウェアは制御部に格納されており，三つの部分から
(１)BIOS（basic input output system）
アプリケーションソフトウェアからサブルーチンとして呼び出さ
(２) アプリケーションソフトウェア
BIOSの機能を利用して，培養部の実験条件制御，実験データバ
換えが可能．
(３) ユーザプログラム
アプリケーションソフトウェアに定義されているコマンドを用い
て，実験個別にシーケンスを記述し，実行することができる．実験
データバスを経由してアップロードすることが可能．
