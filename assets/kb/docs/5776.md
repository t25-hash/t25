# 13・3・5 終わりに

> **章**: 第13章 感性・情緒工学
> **節**: 13・3 情緒工学

情緒工学は人間の主観にかかわる領域であるため,精度,速度などの客観的尺度だけでは評価できず,客観性を基本とする科学にあっては扱いにくい領域である.例えば,自律ロボットの場合には,環境と人間との相互作用から発現する行動が,観察者である主体の経験や知識に依存する主観的解釈によって意味付けされ,主体のもつ価値観で評価されるため,ロボットの設計者・製作者が埋め込んだ意図や価値観とは必ずしも一致を見ない.本節では人間と機械の関係を分類し,その相互作用について述べた.そして,機械と相互作用する主体による主観的解釈について説明し,人間の情緒に働きかけるために主体の感覚を適切に刺激し,連想記憶を引き起こす手法などについて説明した.アザラシ型メンタルコミットロボットのパロと人間との相互作用の事例を紹介し,ロボットの行動について,情緒を意味する言葉により表現されること,人間の情動に変化を与えることができること,そしてロボットに心や情緒があるかのような解釈をされることを述べた.文献(1) 日本感性工学会のホームページ,http://www.soc.nii.ac.jp/jske/(2) Kouchi,M.,Anthropol.
