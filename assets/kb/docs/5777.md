# 15・2・8 計画されている実験

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・2 ライフサイエンス実験装置（細胞培養装置）

2006年現在，複数の候補実験テーマが選定され，それぞれ地上
において実験計画の作成，予備試験による実験条件のつめや，キャ
ニスタに収納する個別実験対応部分の設計・製作などが進められて
いる．これらは細胞を培養する，植物を種から発芽させる，微生物
の増殖を行う，などを通して宇宙環境の影響を調べる実験である．
各実験テーマで，微小重力環境の効果，放射線環境の効果などそれ
ぞれの着目点から研究が行われる．
