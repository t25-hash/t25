# 15・2・9 実験供試体と搭載例

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・2 ライフサイエンス実験装置（細胞培養装置）

CBEFはJEMとともに軌道上に打ち上げられ，付属品の取付け
後，実験テーマごとに固有の実験供試体を利用して15･2･8項で述
べた実験を行う予定である．実験供試体の一例を下記に示す．
図15･11は，中型のキャニスタを搭載したところで， G部に6
個，1G部に4個がそれぞれ搭載できる．キャニスタの中には個別
の実験ユニットが格納されている．キャニスタと実験ユニットを組
合わせて実験供試体と呼んでいる．
図15･12に植物実験供試体の概念を示す．植物の栽培部分と観察
機構および制御機構からなっている．自動制御により種子に注水
し，生育の様子を画像観察しながら実験を実施することができるよ
うに設計されている．
