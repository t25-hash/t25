# 14・3・1 動作の解析技術

> **章**: 第14章 スポーツ機器
> **節**: 14・3 トレーニング機器

スポーツ時の人間の動作を解析することは、模範となる動きや技術の抽出を可能とする。動作解析(motion analysis)は、優れた選手の運動の実体をとらえ、スポーツ技術を究明することをねらいとする場合と、目標とする技術を習得、改善するための情報を得ることをねらいとする場合に大別できる。このうちスポーツ技術の究明をねらいとする場合には、一流選手を対象として、その身体運動を測定・分析することが多い。特に公式試合における測定では、競技規則、競技運営などのさまざまな制約があるため、選手の身体運動の三次元的な動作を記録し解析するには、現在のところ映像解析が最も適している。ここでは選手の身体運動を複数台のカメラ情報から算出するDLT 法の原理や、関節角変位を測定するためのゴニオメータ等について述べたのち、歩行や走行時等の地面反力を得る9 ―217図14・29実空間座標系およびレンズ座標系ために用いられるフォースプラットホームについて述べる。a. DLT法(direct linear transformation method)複数台のカメラを用いて任意の方向から撮影した各カメラの画像情報を用いて、対象の三次元画像を得る代表的な画像分析方法であるDLT法について述べる。これは、実空間座標とディジタイジング画面上の座標との変換係数をあらかじめ求めておき、これらの係数を含む変換式に画像から得た計測点の座標を代入して、実空間座標を再構築する方法である。なお、対象点の三次元座標値を自動で追尾する計測器も存在するが、身体各部へのマーク付けが必要、また屋外の自然光では使用できないなどの公式試合における制約のため、競技会におけるその適用範囲は限られている。b. 電気式ゴニオメータ(electro-goniometer)隣接する体節の表面に装着し、運動時の関節角度変化を電気信号に変換する器具をゴニオメータと呼ぶ。可変抵抗器に二つの体節への固定用アームを取付けた同心型の一軸のものは、構造が簡単であり、また直線性やヒステリシス特性の優れたものを容易に作製できる。しかしながら被験者の自然な動作を妨げる場合があるため、柔軟性に富み、動作への影響が小さいフレキシブル型が現在の主流となっている。このフレキシブル型ゴニオメータには、導電性ゴムあるいはひずみゲージを利用したものがあり、その特徴として、20g 以下と小型・軽量、関節とゴニオメータの回転中心を合わせる必要がないなどが挙げられる。ゴニオメータの利点は、対象とする関節が画像分析法におけるカメラレンズ画角のような空間的制約を受けずに、角度データを電気信号として直接処理できることである。欠点としては、身体に直接装着する必要があるため、求められる変量が相対的な関節角度に限られること、および激しい動きには不向きであることがある。c. フォースプラットホーム(force platform)運動中の身体は重力、地面反力、空気抵抗などの外力を受ける。これらの外力は、それ自体が考察の対象となるだけでなく、関節力や合成筋トルクなどを推定するために不可欠である。フォースプラットホームは、歩行動作、跳躍動作および打撃動作などの解析において、地面反力測定装置として用いられる。これは被測定物を載せるための板と、それを支える4本の柱からなり、各柱に力を測定するためのフォーストランスデューサがある。このトランスデューサには、ひずみゲージ式、および水晶リングに生じる圧電効果を利用した圧電式があり、各柱に作用する鉛直方向および水平2方向の力を検出する。各トランスデューサで検出した三軸方向の力から、圧力中心における力およびモーメントが算出可能となる。
