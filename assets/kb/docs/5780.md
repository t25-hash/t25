# 14・3・2 トレーニング機器

> **章**: 第14章 スポーツ機器
> **節**: 14・3 トレーニング機器

スポーツトレーニングには、筋力、筋パワー、筋持久力等の筋機能を鍛えるレジスタンストレーニング、動作の素早さ、正確さ、滑らかさ等の身体動作の調整力を鍛えるスキルトレーニング、最大酸素摂取量、最大心拍数、最大呼吸数等の心肺機能を鍛える生理的機能トレーニングがある。ここでは、パフォーマンス向上のために必要なこれらのトレーニングに用いられる機器について解説する。a. レジスタンストレーニング(resistance training)機器レジスタンストレーニングは、マシン、フリーウエイトおよび自重などによって筋に負荷を与え、筋肥大および神経系の活性を促すことにより、筋機能を向上させるトレーニングである。このうちマシンによるトレーニングは、ウエイトの落下による危険が少なく、負荷の設定も容易である利点がある。負荷の方法としては、(1)ウエイトスタック式、(2)油圧式、(3)空気圧式、(4)電磁式がある。このうち(1)のウエイトスタック式は、ウエイトを滑車やてこなどを使って持ち上げることにより負荷を発生させるもので、定滑車のみを使用して負荷が一定に保たれるダイレクト負荷方式、関節可動域に合わせて負荷を変化させる可変負荷方式がある。(2)(3)の油圧、空気圧式では、油圧や空気圧の抵抗を利用して負荷を発生させ、ダイヤルやペダルなどによって負荷の増減を調節するものであり、ウエイトスタック式と比較して負荷抵抗値を細かく設定できることが利点である。(4)の電磁式は、角速度を変えることによって筋パワーや筋持久力などさまざまな筋機能を高めることができる。また、非常に小さな負荷を与えることも可能で、かつ筋出力を自分自身で随時調節することができることなどから安全性にも優れており、リハビリテーションの分野でも広く使われている。ここでは電磁式トレーニング機器の代表例として、アイソキネティックマシンについて説明する。筋収縮の様式には、一定の負荷条件のもとで、関節の屈曲・伸展を行うように筋を収縮させる等張性収縮、関節の屈曲・伸展を行わずに力のみを発揮する等尺性収縮、および関節の屈曲・伸展速度を一定として力を発揮する等速性収縮(isokinetic contraction)がある。アイソキネティックマシンは、等速性収縮運動下のトレーニングを可能にした機器であり、図14・30に示すようにサーボモータ、ウォームギヤ等から構成されており、被験者の運動肢にセットされた入力アームが運動とともに回転し、選択した速度に達するとどんなに力を増してもそれ以上の速度では動かないようになっている。このとき入力アームを回そうとする力が運動肢に対する負荷となり、時間経過および関節角度に応じた筋発揮トルクを計測・評価することが可能となる。なお、短期間のレジスタンストレーニングでは、おもに神経系の適応によるところが大きいといわれ、筋肥大等の筋の形態的な変化によってもたらされる効果が出現するには、長期間のトレーニングが必要となる。また、対象筋が局所的な部位に偏って拮抗筋や左右の筋力のバランスが崩れると、肉離れやねんざなどの整形外科的な障害を招く危険があるため、全身をバランスよくトレーニングする必要がある。さらにトレーニングの効果は、負荷の設定によって異なる。すなわち、大きな負荷で反復回数の少ないトレーニングでは、筋力が養われ、小さな負荷で反復回数が多いトレーニングでは、筋出力の持続性を示す筋持久力が養われる。中程度の負荷でのトレーニングでは、筋パワーが向上するといわれている。このようにレジスタンストレーニングでは、どのような筋機能を高めるのかに応じて負荷の設定を変える必要がある。b. 生理的機能トレーニング機器心拍数をある程度高めた状態での運動を長時間維持する有酸素運動は、全身の血液循環を高め、肺や心臓、血管などの心肺機能を向上させることができる。それゆえ心肺機能向上のためのトレーニング機器は、一定の負荷強度で長時間運動を続けられることが絶対条件となる。自転車こぎやランニングは長時間の運動が可能であり、さらに室内で利用できる機器の代表例である室内ランニング装置(トレッドミル)、固定式自図14・30 アイソキネティックマシンの構造図14・31 トーイング機器によるトレーニング転車(自転車エルゴメータ)等は、能力や目的に応じた運動強度の設定が可能であるとともに、天候に左右されず定期的に実施できる点で優れている。これらの機器については、12・2節にその詳細が記載されているので、ここでは割愛する。c. スキルトレーニング(skill training)機器身体動作は、脳からの指令が中枢神経、末梢神経を通じて最終的に筋肉に達することで開始され、その信号が筋紡錘など筋肉内の感覚器官による情報とともに再度中枢神経にフィードバックされ動作の調整が行われる。この神経系の働きによる調整力を高めるスキルトレーニングを行うことにより、正確さ、素早さが養われ、パフォーマンスを向上させることができる。ここではスキルトレーニング機器の一例として、陸上短距離種目におけるトーイング(towing)機器について説明する。ランニングのスピードは、ストライド(歩幅)とピッチ(単位時間内歩数)との積によって決まる。トーイング機器は、強制的に走者をけん引することによって、この両者あるいはどちらかを向上させ、スプリント能力を向上させるためのものである。その仕組みを図14・31に示す。走者の腰部にベルトを装着し、それに取付けられたロープを電動モータでけん引する。モータはけん引力の大きさを設定できるように、ロープ張力のフィードバック制御が行われる。けん引力は30~60N 程度である。これにより自力で到達可能な状態を超えた水準に筋や神経をおき、それらを刺激することによって、より高いパフォーマンスレベルに対する神経筋組織の適応を促すことが期待できる。なお、けん引力に頼った受動的なランニングでは、ピッチを増加させることはできない。そのため、合理的なランニング動作を理解し、ピッチを速くするという意識を高くもってトレーニングを行わなければ効果は期待できない。スポーツが盛んになるとともに、スポーツ医学(sports medi-cine)の概念も変化している。運動生理学を中心とする基礎医学から臨床医学へ、そして多くの診療科がかかわるようになり、学際的な側面が目立ってきている。スポーツ医学が扱う領域を整理したものが表14・1である。ここでは、これをもとにスポーツ医学について概説する。
