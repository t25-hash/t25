# 14・4・1 スポーツ医学の基礎領域

> **章**: 第14章 スポーツ機器
> **節**: 14・4 スポーツ医学

スポーツ活動は,生体へさまざまな負荷を与える.図14・32に示すように,生体は負荷が増すに従い適応していくが,あるラインを超えると適応できずに破綻する.スポーツ医学の基礎領域では,このプロセスを研究対象としている.どの程度の負荷を与えると生体が破綻するかを知る指標を一般化して特定することは個体差もあり難しいが,多面的なアプローチがなされている.運動生理学領域は,スポーツ医学において重要な位置を占めている.特に運動負荷に対する心肺機能の応答についての研究成果は,競技力向上ばかりでなく健康の維持・増進を図る運動を考えるうえでの科学的根拠を与えている.スポーツバイオメカニクス領域は,運動を行う際の生体の動きをおもに力学的な手法を用いて解析することにより,効率的な運動方法や生体への力学的な負荷を明らかにする学問領域である.トップアスリートのパフォーマンス解析などにより,効率のよいフォームなどに関する知見を与えてくれる.9 ―219表14・1 スポーツ医学領域の整理1. スポーツ医学の基礎領域(1) 運動生理学(2) スポーツバイオメカニクス(3) スポーツ栄養学(4) スポーツ心理学(5) その他2. スポーツによる医学(1) 運動療法1内科的運動療法2整形外科的運動療法(2) リハビリテーション1内科領域2整形外科領域3. スポーツのための医学(1) メディカルチェック(2) 運動プログラム(処方)(3) メディカルコンディショニング(4) スポーツ傷害・障害の治療(5) 薬物使用(アンチドーピング関連)図14・32 適応と破綻スポーツに使用する用具の開発についても,スポーツバイオメカニクスからのアプローチが欠かせない.なかでもシューズと競技サーフェスについての研究は最近特に進んでおり,記録の更新などスポーツの進歩に貢献している.コンピュータサイエンスの進歩により大きく展開した領域である.スポーツ栄養学領域では,運動に必要となるエネルギー消費についての知見をもとに,有効な栄養摂取方法等について研究する.競技スポーツでは,スポーツに伴う疲労対策,そしてスポーツに必要となる体作りを効率よく行うための栄養補給方法などがおもな研究対象となる.マラソンなどの持久的運動におけるカーボローディング方法は,スポーツ栄養学の成果としてよく知られているが,最近ではサプリメントの功罪が論議されている.スポーツ心理学領域は,スポーツ活動に伴う心理学的な変化を扱うとともに,よいパフォーマンス成果をもたらすための心理学的なアプローチ方法が研究対象となる.特に最近ではイメージトレーニングなどのプログラムが開発されており,競技力向上の視点からその重要性が高まっている.
