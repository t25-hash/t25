# 14・4・2 スポーツによる医学

> **章**: 第14章 スポーツ機器
> **節**: 14・4 スポーツ医学

スポーツによる医学は,種々の疾病に対する予防あるいは治療として運動プログラムを用いるものである.また,すでに確立されているリハビリテーションもこのカテゴリに含めて考えられる.最近では,スポーツ診療科を設けている医療施設も多い.しかし,予防医学的運動療法は,現行の医療制度の中ではあまり評価されていない.近年ようやく高血圧,糖尿病,高脂血症などに対する運動療法が制度の中で位置付けられるようになるなど,この領域が正当に評価される動きがでてきた.a. 運動療法(exercise therapy)運動療法は,食事療法,薬物療法と並んで糖尿病の治療法として,内科領域の中でも従来から重視されてきた.最近では,高血圧や高脂血症などいわゆる生活習慣病の治療手法の一つとして,その重要性が高まってきている.肥満は,国際的にも課題とされる病態であるが,運動療法の効果が期待されている領域である.9 ―220医療・福祉・バイオ機器整形外科領域においては,骨粗鬆(こつそしょう)症(オステオポローシス)などに対する運動療法の必要性が指摘されている.b. リハビリテーション(rehabilitation)内科領域では,心筋梗塞後のリハビリテーションの効果について研究が進んでいる.以前は安静が主体であった心筋梗塞の治療に,この分野は新たな光をもたらしている.整形外科領域では,種々のスポーツ外傷,特に膝前十字靱帯損傷に関する手術法が進歩し,これに対応したリハビリテーションに関する研究が多くなされ,スポーツ医療の進歩に貢献している.アスレチックリハビリテーションは,スポーツ医学の領域として重要な位置を占めている.脳血管疾患に対するリハビリテーションは,すでに領域として確立されていることが知られている.
