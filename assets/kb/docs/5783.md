# 14・4・3 スポーツのための医学

> **章**: 第14章 スポーツ機器
> **節**: 14・4 スポーツ医学

スポーツ医学は、健康の維持・増進などのためにスポーツを楽しむ一般人も、そして競技スポーツを行うアスリートも、ともにその対象とする。次の項目がスポーツ医学として特に重要な事項である。a. メディカルチェック(health check)スポーツ医学の役割の一つとしてメディカルチェックがある。スポーツにはリスクがつきものであり、スポーツを始める前のメディカルチェックは、突然死やスポーツ外傷等を防止するうえで重要とされている。メディカルチェックは、内科的なものと整形外科的なものに大別される。内科的なメディカルチェックとしては、一般的な健康診断で行われるような項目をベースにして、循環器系のチェックが重要となる。循環器系のチェックとしては、循環器系疾患の家族歴、既往歴の確認の後、心電図や胸部レントゲン写真などのチェックが基本となる。異常が疑われたり、詳細なチェックを要する場合は、運動負荷試験や超音波による検査が実施される。整形外科的チェックとしては、外傷や障害歴を確認後、骨関節系のチェックが行われる。X 脚やO脚の有無など下肢のアライメント、関節の柔軟性を見る関節の可動域チェックなどがおもに徒手検査により行われる。異常が疑われる場合はレントゲン検査が実施され、さらに詳細なチェックを必要とする場合はMRI(magneticresonance imaging)検査などが併用される。以上のチェック結果を総合的に判定して、スポーツを行うことの可否が決められ、具体的なプログラムが組まれる。b. 運動プログラム(運動処方、exercise program)スポーツ活動に際して、どのような運動を行ったらよいか、具体的なプログラムを提示していくことは重要である。健康の維持・増進のためのプログラム作成をしばしば運動処方と呼ぶことがある。プログラムの作成に当たっては、年齢による身体的機能の変化を念頭におく必要がある(図14・33)。成長期にあっては特に運動プログラムの考え方が重要であり、表施していく。図14・33 身体諸機能の推移また、成人ではa項で述べた医学的なメディカルチェックとは別に、フィットネスレベルをチェックして個々に応じたプログラムを作成する。プログラムの作成に当たっては、運動の種類、強度、時間といった要因を適切に組込む。国民医療費の増大が社会問題化していることもあり、中高年期の生活習慣病に対する運動が重視されている。中高年期の特徴は老化現象であり、プログラムの作成に当たっては、予備力の低下、反応の鈍化、回復の遅延、再生力の減退などに留意する必要がある。表有酸素運動が重視されるが、最近では筋力面のトレーニングの重要性も、転倒予防などの視点から注目されている。c. メディカルコンディショニング(medical conditioning)スポーツ医学の重要な役割の一つが、コンディショニングの領域である。その代表的な領域の一つが疲労対策である。スポーツにおける疲労は、生理的疲労と病的疲労に分けて扱われる。生理的疲労は運動生理学の領域で扱われるが、図14・34に示すようにトップアスリートの場合は、トレーニング負荷を限界まで高めるため、病的な疲労のエリアにしばしば入ってしまう。このため、スポーツ現場では、図14・35に示す疲労と休養そして回復というプロセスを理解しておくことが重要となる。スポーツにおける疲労の原因としては、エネルギーの枯渇、疲労物質の蓄積、内部環境の失調、中枢性調節力の失調などが挙げられる。病的な疲労に陥らないように、エネルギー源の適切な補給、乳酸などの疲労物質への対応、適切な水分補給などによる脱水などへの対応などが重視される。また、アスリートのコンディションの把握において、図14・36に示すように、メディカルコンディションの動きは他の要因の動きと異なることを念頭に入れておく必要がある。競技負荷後のリカバリ表14・2 薬物使用[アンチドーピング(anti-doping)関連]ドーピングは、競技スポーツにおいて厳しく禁じられている。国際的なアンチドーピング活動は、1960年代から国際オリンピック委員会がリードしてきたが、現在は1999年に設立された世界アンチドーピング機構が国際的な合意のもとに世界統一の「世界アンチドーピング規程」を策定し、種々の活動を展開している。ドーピング検査は、試合などの競技時ばかりでなく、トレーニング時やシーズンオフ時の競技外検査、いわゆる抜き打ち検査が重視されている。禁止薬物リストは、世界アンチドーピング機構により毎年更新される。表14・5に、世界アンチドーピング機構により定められている禁止物質と方法を示す。アスリートがこれらの禁止薬物を過って使用しないよう、正しい情報の提供と教育が重要である。遺伝子医療の進歩により、遺伝子ドーピングが懸念されている。文献(1)
