# 15・4・4 観測データ処理

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・4 船外プラットホームの利用（超伝導サブミリ波

a.軌道上データ校正 大気の信号はアンテナで集められ，受
信機による周波数変換と増幅の後，マイクロ波分光計で多数の周波
数チャネルに分割・検出されディジタル化される．ディジタル出力
はこれらすべての機器の性能パラメータ（利得，雑音の周波数特性
等の装置関数）の影響を受ける．大気の輝度スペクトルを正確に評
価するためには，基準となる白色雑音（常温の黒体ふく射源からの
サブミリ波雑音）を定期的に受信機に入力し，機器パラメータの変
動を校正する必要がある．軌道上のSMILESは，約1分周期で地
球大気を高度方向に走査するとともに，受信電力強度と受信周波数
の校正を行う．輝度スペクトルデータの発生量は約110kbpsとな
る．
b.地上データ処理 輝度スペクトルからO ，ClO，HCl等の
分子種ごとに大気中の混合比（高度分布）を求めるのは，地上デー
タ処理の役割である．混合比の先験値ベクトル（高度分布）と観測
ジオメトリーおよび装置関数を用いて予想される輝度スペクトルを
計算し（フォワードモデル)，これを観測された輝度スペクトルと
比較することで，より正確な混合比ベクトルを推定する（この処理
をリトリーバルと呼ぶ)．具体的には，フォワードモデルを先験値
ベクトルの近傍で線形化し，混合比ベクトルの微小な変化に対応す
る輝度スペクトルの変化を係数マトリックスとして求め，これを用
いて混合比ベクトルを推定する方法が一般的である［ロジャーズ
（Rodgers ）法］．その際，フォワードモデルの正確さが混合比の
推定精度を左右するのは当然だが，サブミリ波観測では通常，観測
システムの雑音による測定誤差が大きな比重を占める．超伝導セン
サを用いるSMILESの場合は，この誤差が非常に小さく，短時間
に取得されたデータからも高い精度で微量分子の混合比を決定でき
るのが特徴である．
