# 15・1・2 操作性

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・1 医療・福祉情報システムとしてのインフラスト

医療情報を電子化することにより、医師等医療従事者にこれまで以上の負荷を与えるようであれば、いくらその効果を示してもシステムの普及ははかどらない。そこで、従来の作業フローから大きく逸脱することのないシステム、もしくは従来システムよりもより効率的に処理が可能なシステムを構築しなければならない。例えば、いままでフィルムで読影していた画像診断をCRT(cathode raytube)診断とした場合、その読影方法はドラスティックに変化すると考えられるが、CRT 読影により効率的な処理、読影精度の向上が期待できれば、その導入のスピードは速まるであろう。しかしながら、読影作業に伴う新たな操作環境がそのスピードを左右すると考えられるため、キーボード操作等をなるべく意識させない入力法が必要である。ただし、15・1・1項で前述したセキュリティと操作性は相反する関係にあり、セキュリティを高めれば、その処理にかかる時間や入力のために操作性が落ちる。操作者にセキュリティに対してそれを保つための意識をもたせながら、操作性に大きく影響を与えないシステムの構築が望ましい。
