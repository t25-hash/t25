# 15・1・4 ユビキタス

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・1 医療・福祉情報システムとしてのインフラスト

次の課題は端末のユビキタス化である.これにより,医師はどこにいても医療情報ネットワークにアクセス可能であり,いつでもどこでも問診,画像診断が可能となる.これは究極的な遠隔診断の姿であり,救急医療や災害医療において医師不足による不幸を未然に防ぐことができる.この技術課題は,高速なネットワーク環境をいつでも享受できるホットスポットと呼ばれる無線LAN 環境と,PDA(personal digital assistance),ポケットPCなどと呼ばれるハンドヘルドタイプのPCの発展と同時に,端末タイプを判断して情報処理,送受信処理を行う新規ネットワークサービスなどが考えられる.以上,医療・福祉の情報インフラストラクチャとしてのシステム化構築要件を挙げたが,これらの要件は後述するネットワーク,電子カルテ,医用画像,遠隔診断・リハビリテーション,健康モニタリング・高齢者モニタリングにも適用されるべきものであり,今後のシステム開発時にはこれらの要件を参考にしていただければ幸いである.
