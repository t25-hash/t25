# 15・2・1 はじめに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・2 ネットワーク

世界的なインタネットの普及に伴い,ネットワークのブロードバンド化が期待されている.韓国ではいち早くADSLを利用したブロードバンドネットワークが導入され,急速に普及が始まった.ADSL技術については15・2・3・b項で詳細に述べるが,既存のメタルネットワークを利用した高速回線である.日本では先に普及したISDN との混信など技術的問題により普及が遅れていたが,最近ではそれもクリアされ,急速に普及が始まっている.また,次世代の高速化技術にも対応可能な究極的なブロードバンドネットワークであるFTTH と呼ばれる光ネットワークも,2001年からサービスが開始された.これらのブロードバンドネットワークはパケット通信の一つであるTCP/IP(transmission control protocol/internetprotocol)と呼ばれるインタネットプロトコルが使用され,現在ではデファクトとして計算機間の通信のみではなく他のほとんどのデータ通信に使用されている.医療分野では病院内の各種情報通信からメディカル機器間の通信,病院内での放射線画像などを転送する際のDICOM 通信まで,幅広くTCP/IPが使用されている.情報ネットワークは,オフィスや病院内で使用する構内ネットワーク(LAN),外部に出ていくためのアクセス系ネットワーク,電話局間を接続する中継系ネットワークに大別できる.図15・1にそれぞれのネットワークにおける代表的な技術進歩に応じた伝送速度を示す.構内ネットワークは,アクセス系と比較して2けた高速なネットワークであることがわかる.以下に今日のIT 化を推進する重要な技術である,構内ネットワークとアクセス系ネットワークについて述べる.図15・1 高速ネットワークの動向
