# 15・5・3 宇宙医学

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・5 与圧部の利用

吉冨進,電子情報通信学会宇宙航行エレクトロニクス研究会技術宇宙医学（space medicine）の分野では，基礎医学分野の研究と臨床応用を視野に入れた研究が計画されている．
前者は，基礎科学的な側面を重視した基礎医学関連を中心とした研究，ならびにヒトの生物学に関する生命科学現象としてのバイオメディカル研究である．
後者は，骨からのカルシウムの溶け出しや尿路結石など，人体の宇宙環境への適応や，宇宙から地上に帰還した際の再適応を支援する研究など，宇宙飛行士の活動を支援する応用的側面が強調された研究として遂行が予定されている．
