# 16・2・1 基本的構造

> **章**: 第16章 テザーシステム
> **節**: 16・2 ひも付き衛星システム（TSS:shuttle/tethered sat

ひも付き衛星に関する大がかりな実験はスペースシャトル宇宙でテザーを使った実験は，1966年に初めてジェミニ11号とポリエステルのテザーでつながれたアジェナ宇宙船をジェミニ11引っ張られた．
ジェミニ12号での実験の中で，ジェミニとアジェbilization（重力こう配を利用したシステム安定化）の予備的実験1974年，ひも付き衛星（英語でtethered satelliteといい，こので，今後この言葉を用いる) による低高度観測の可能性がスミソの提案の中で彼らは低高度大気の研究のためにスカイフックと呼ばも付き衛星を展張するのに大きな風船を利用することを提案した．
ャトルオービタより後方に向かって押し流されるであろうというこ表された．
マーシャル飛行センターで行われた研究は，大気抗力を利用するのの万有引力の差を利用することによってひも付き衛星の伸展，安定した．
さらに衛星の飛行安定，低高度飛行に伴う空力加熱，スペー験を阻止する技術的な困難は見出し得ないと結論するに至った．
(ｂ)テザーシステムの詳細第Ⅲ部 宇宙空間への進出 第16章 テザーシステムうる．
実験の目的によって衛星にはいろいろな型のものが考えられう（後出図16･6参照)．
衛星は，ブーム（長さ20～50m）の先端に取付けられている衛ぐテザーは直径約1mmで，合成繊維を材料としている．
ブームの目的は，シャトルからの衛星放出時に衛星がスペースシャトルに（space shuttle）によって行われたので，ここではこれに従い説明ジェミニ12号で行われた．
ジェミニ11号では，長さ30～48mの号の周りに回転させることによって生じた遠心力によってお互いがナはお互いに垂直の位置に投入されて初めてgravity gradient sta-が行われた．
語の日本語訳は小山が初めて日本で紹介して以来，定着しているのニアン天文台のコロンボのグループによって示された．
スペースシャトルの上空にも，あるいは地球に向かっても伸展されるが，先端衛星の高度の下限は空力加熱を考えると120kmであろ衛星に絡みつく可能性のあることが指摘されている．
星捕獲のためのアダプタに固定されている．
衛星とシャトルをつなードベイの中にブームが引き寄せられ全行程が終了する．
