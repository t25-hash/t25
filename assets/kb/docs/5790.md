# 15・2・3 アクセス系ネットワーク

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・2 ネットワーク

アクセス系ネットワークは、病院や自宅から電話局までを結ぶネットワークであり、足回り回線とも呼ばれている。ネットワークの高速化は、ユーザからの大量の情報を効率的に伝送するため、最初に中継系ネットワークの高速化が検討されてきた。しかしながら、いくら電話局間の中継系ネットワークが高速になろうとも、このアクセス系ネットワークが高速化されない限り、ユーザにとってのスピードはこのネットワークで律速されてしまうことになる。アクセス系ネットワークはネットワーク構成からいえば最後の回線区間であるため、ラストワンマイル問題として、その高速化が課題であった。以下に、高速化の最初の手段であったISDN 技術と、最近のDSL(digital subscriber line)、FTTH 技術について述べる。a. ISDN 技術従来の電話はアナログ通信路を使用しており、音声による利用が主であった。しかし、ファックスの登場と計算機の普及によるネットワークを利用した計算機処理の需要により、アナログ信号にディジタル信号を重畳するモデムが利用されるようになった。そのモデムは変調特性により安定性や速度に限界があることから、有効にネットワークを利用するためのディジタル通信技術としてISDN が通信標準化組織によって検討されてきた。日本では1984年にサービスが開始され、その通信速度は64kbps,1.5Mbps(または2Mbps)、150Mbpsの三つである。ここでは、ホームユーザ向けの64kbpsのものについて述べる。ISDN は、アナログ網で使用されてきたメタルケーブルを使用してディジタル通信を提供するサービスである。ISDN は、一つの物理回線に対し二つの通信チャネル(64kbps×2)と一つの制御チャネル(16kbps)を有している。ユーザは同時に2チャネルの通信が可能であり、インタネット普及時には、電話とインタネットを同時に通信可能とし、またデータ通信時には2チャネルを同期させ同時に使用して最大128kbpsの通信を可能とするISDN の普及が目立っていた。しかしながら、現在ではb項に後述する既存のアナログ電話線を用いたDSL技術に押され、ISDN の利点が少なくなり、加入者数が減少傾向となっている。b. DSL技術DSL(digital subscriber line)は、既設メタル線を用いてISDN より高速の通信を行うための技術である。代表的なものがHDSL(high speed DSL)、SDSL(symmetricDSL)、ADSL(asymmetric DSL)およびVDSL(very highspeed DSL)である。各方式の頭文字をx で総称して、xDSLとも呼ばれる。これらの比較を表15・3に示す。HDSLとSDSLでは、上り下りの伝送速度が対称である。ADSLはメタル線のもつ伝送能力を上りよりも下りに多く割り当てた方式であり、インタネットでデータをダウンロードするような利用形態に適する。VDSLは動画等のより高速の信号を伝送するものである。DSLは伝送速度が速いため、伝送距離は短い。したがって、ユーザの自宅と電話局との距離がサービス形態に大きく影響するため、サービス提供業者はそのおおよその距離を推定するためのウェブサイトを用意し、サービス提供が可能かどうかをあらかじめユーザに知らせている。現在では、インタネット普及に大きな役割を果たしたISDN に取って代わり、ADSLがインタネットの高速通信環境の実現手段として貢献し、大幅に加入者数増大の傾向を示している。c. CATV ネットワークCATV ネットワークは映像を配信するためのネットワークであるため、高周波の伝送が可能な同軸ケーブルが使用されており、高速化が容易であった。もともとテレビ放送波が微弱な地域で、放送波を有線ネットワークで再配信するためのものであったCATV ネットワーク事業者が、通信の開放政策により技術的に可能であった電話への接続事業を最初に開始し、その後、インタネットの発展とともに高速インタネットアクセスラインとして発展してきたという経緯がある。今後は、同軸ケーブルから光ファイバへの展開が予想され、通信業者の提供する光ファイバとの競合になるだろう。. 無線アクセス系ネットワーク
