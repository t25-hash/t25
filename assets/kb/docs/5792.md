# 15・3・1 はじめに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

1990年代に主として大学病院や大規模病院から始まった病院業務の電子化の波は、オーダエントリーシステム(order entry sys-tem)として中規模病院にまで波及し、患者の待ち時間の短縮や処方ミスの低減など、さまざまな成果が生まれている。一方、1999年4月の旧厚生省通達(診療録の電子化容認)をきっかけとして一気に電子カルテ(electronic patient record)導入の気運が高まりつつあり、大規模病院だけでなく、診療所レベルでも電子化が始まった。これらの動きは、近い将来の日本の医療システムを大きく変える可能性を秘めているが、問題点も多い。ここでは、これらの問題点と、電子カルテシステム構築の考え方について述べる。
