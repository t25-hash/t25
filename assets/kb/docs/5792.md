# 16・3・1 軽量柔軟構造物

> **章**: 第16章 テザーシステム
> **節**: 16・3 システムの工学への応用

軌道上の大型構造物は，国際宇宙ステーションに見られるよう
に，現在ではトラスがその主要な構造部材として使用されている．
宇宙ステーションの次の世代のより大型の構造物では，軽量で大き
な構造を実現できるテザーが主要な役割を果たすと考えられる．地
上でも大型構造物は，つり橋や大型ドームなどの例に見られるよう
にワイヤがその構造における主要な役割を果たしている．またテザ
ーは，軌道上の施設本体から離して保持することが望ましい危険物
貯蔵庫などの係留の手段としても有用である．この場合16･2･2項
で示すように，軌道上の係留物は鉛直下方または上方に伸展した状
態で安定する．テザーは剛性が低く構造体との接合部での運動の自
由度が大きいため，トラスなどの剛性の高い構造物と比較して機械
的な擾乱に対して耐性が高いという利点がある．
