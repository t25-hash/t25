# 15・3・2 電子カルテの開発の歴史

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

a. レセプトコンピュータ(clinical accounting system)医療機関では毎月,健康保険支払機関に対して診療報酬請求を行う.この作業の合理化のために開発されたのがレセプトシステムで,一般にレセコンと呼ばれ,十数社程度のベンダが供給している.レセプトコンピュータによって医事処理は合理化できたが,処方,注射,処置などの医療行為は伝票ベースで残っているため,レセプトコンピュータにデータを入力する部署に作業が集中し,業務全体のボトルネックとなっている.特に大規模病院ではその傾向が顕著となり,患者の待ち時間は短くならなかった.9 ―228医療・福祉・バイオ機器b. オーダエントリーシステム円滑な処理を実現するために,データ入力を診療現場に受け持たせ,処方,検査などが発生した段階で入力すれば入力作業の集中が避けられる.このシステムをオーダエントリーシステムと呼ぶ.このシステムにより患者の待ち時間は大きく短縮され,伝票の読み違いなどによるミスも少なくなり,病院業務の改善につながった.しかし,このシステムは単にコンピュータによる伝票処理システムの入力が分散したにすぎず,重要な医学的記述は紙のカルテに書かれていた.
