# 15・3・5 電子カルテに決まった形はない

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

以上の問題を解決すれば電子カルテは現実のものとなる.しかし,ここで注意すべき点がある.コンピュータという新しいメディアを使った医療記録を考えるとき,従来の紙のカルテのイメージに影響されがちになるが,そのような古い概念は捨て去ったほうがよいということである.ユーザインタフェース(目に見える電子カルテの形)を最初から議論すると,どうしても紙のイメージの電子的な再現になる.電子カルテの本質は,広域で情報交換が可能であることと,対象(患者,医療スタッフ)によって見せ方を変えることができることであり,百人百様のユーザインタフェースがあってもよい.今後提供される電子カルテでは,まず標準的なインタフェースが提供されたうえで,ユーザの事情に応じて個人レベルでどのようにでも変更できるようにするのが望ましい.
