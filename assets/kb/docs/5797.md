# 15・3・6 電子化によるメリット

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

病歴情報がネットワーク上で電子化された場合、従来考えることのできなかったメリットが生まれる。そのメリットは単に電子カルテシステムだけからもたらされるものではなく、コンピュータが広域ネットワーク(インタネット)につながっているという事実そのものが大きなメリットを生む。a. 電子カルテシステムがもたらすメリット。 医療レベルの向上現在の医療は、ある意味で標準化されていない。同じ条件で病院にかかっても、地域特性、医師の考え方(医学的背景)などによって異なる治療が行われる。病気に対する治療の方法は必ずしも一つではないので間違ったことではないが、少なくとも現在最も治療効果が高く、しかも費用が安い方法は、いくつか存在するはずである。そのような最新の知識を道具として電子カルテシステムに組込むことは可能であり、それによって全国どこででも最新の医療が受けられることになれば、患者にとって大きなメリットとなるだろう。. 診療データを医学の進歩に役立てる臨床医学ではヒトを対象とした実験は困難であり、臨床の現場で得られた事実を貴重なデータとして今後の診療方法の改善につなげていく必要がある。したがって、カルテに記載された事実は、医学の進歩に寄与する大変貴重なデータとなる。b. ネットワーク化のメリット。 患者と医療機関電子カルテは患者にも閲覧させることが可能である。家庭からインタネットを通じて閲覧することができ、病状把握に大きな力となる。慢性病は病状を正しく知ることが治療上大変重要なので、糖尿病、心臓病などを治療する際、大きな意味をもつ。また、従来は病院に行くことが治療の基本であったが、電子カルテによってネットワークを通じた受診が可能になる。薬局、検査機関などとも接続されるので、特に慢性病の場合など有効である。患者はネットワーク上で主治医と相談し、検査、処方などの指示を近くの薬局などに出してもらい、遠くの病院を受診する必要もなく、快適な医療を受けることができる。ネットワークでつながっているということで、いつでも相談できる、いつでも守ってもらっているという安心感が得られ、治療にもよい効果をもたらすことと思われる。. 横のつながりコンピュータネットワークは、旧来の縦割り組織を超えて横方向のつながりを発展させる力をもっている。従来、医療関係者どうしの協力関係は、ある意味で限定されてい(対象)(観測手段)(画像-種類、解像度等)(適用対象の例)(情報の種類)人体表面視察一般視診一般検診、外傷、皮膚形態可視投影光モアレ像脊椎、体型、形成外科形態赤外線サーモグラム乳房、血行、神経系機能電磁界体表面電位図、心・脳磁図心臓、脳機能、形態人体内部視察内視鏡、眼底写真気管支、胃、大腸、眼底形態X 線直接・間接・DR(静止透過像)ほぼ全器官、部位形態、機能テレビ・シネアンジオグラフィー・DSA(動画透過像)CT(断面像、三次元像)RI(γ線・陽電子線)シンチグラム(静止画、動画)心臓、肝臓、脳等形態、機能PET・SPECT(断面、三次元像)超音波エコー像乳房、心臓、腹部等形態、機能、組成(A,B,M モード、断層、ドップラー;二・三次元静止画、動画)NMRMRI(断面、三・四次元動画)心臓、脳、骨等形態、組成アンジオグラフィー(三次元、動画)組織標本可視光光学顕微鏡像(二・三次元静止画)細胞診、染色体、血球形態電子線電子顕微鏡像(二・三次元静止画)病理組織標本等形態9 ―229た。電話、ファックスなど効率の悪い通信手段しかなかったためだが、文字、音声、静止画、動画などマルチメディアを駆使することのできるコンピュータネットワークはその壁を破ることができる。. 知識の共有化ii項で述べた、医療関係者どうしのコミュニケーションは個人対個人のものであるが、個人対多、すなわち大規模データベースを利用したり、地球規模での電子掲示板システムを日常的に利用することが可能になる。これは、全人類の知識を共有することを意味する。最新の知識は、インタネットから最も早く得られ、紙の出版物は遅れるのが現状である。ネットワークの情報をいち早く公開できる特性は、今後さまざまな局面で生かされていくと思われる。特に、薬剤の副作用情報などの分野でその威力を発揮するだろう。
