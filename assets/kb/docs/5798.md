# 15・3・7 終わりに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・3 電子カルテ

電子カルテの目的と概念,そしてその効果について述べた.電子カルテは,医療の分野で利用される通信環境であるといえ,これまでの医療のあり方を劇的に変えることと思われる.患者にとってはこれまで遠くにあった病院が身近なものになり,いつも支えてくれる力強い味方になるだろう.また,医療従事者にとっても手放せない道具になることは間違いない.電子カルテの時代の到来,それは漫然と病院で患者を待っていればよかった時代の終わりでもあり,医療はそのあり方の劇的な変革を問われることだろう.
