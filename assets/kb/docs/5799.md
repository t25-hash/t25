# 15・4・1 医用画像(medical image)とは

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・4 医用画像

医療診断に用いられる画像を、広く一般に医用画像と呼んでいる。狭い意味では直接に診療に使用する目的で患者から撮影した画像を意味するが、広くはそのほかにもカルテなどのドキュメント、標本の記録のような研究用の画像も含まれる。医用画像の始まりは、おそらくX 線発見に伴ってレントゲンやその同時代の人たちによって撮られた人体の一部分のX 線像であろう。その後、科学・技術の進歩とともにさまざまの画像が得られるようになっており、この種類をモダリティ(modality)と呼ぶ。そのおもな例を図15・5に示す。また、それぞれが用いられるに至る、いわば医用画像の発展の要因を以下に示す。(1) 視点:人体外観(通常の視診)→内部透視像(単純X 線像)→断面像(CT 像)(2) 意義:可視化(+記録)→可視化+計測(+記録)(3) データの次元:二次元→三次元→四次元(4) エネルギー源:体外→体内→仮想図15・5 おもな医用画像9 ―230医療・福祉・バイオ機器(5) 処理方式:アナログ→ディジタル(6) モデル:実人体(個別患者)→仮想化人体→実人体+仮想化人体(7) 実(R)・仮(V):標本(実物体、R)、仮想化人体(V)、被験者人体(R)、実人体+仮想化人体(R+V)(8) スケール:分子レベル以下→細胞レベル→組織レベル→器官レベル→人体レベル
