# 15・4・2 可視化と計測

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・4 医用画像

医用画像に共通する最終的目標は、人体に関する可能な限りの情報を収集することである。それを一口にいえば、人体を見る(見えるようにする)こと、すなわち可視化(visualization)であり、続いて定量的情報を得る(人体を測る)こと、すなわち計測(mea-surement)である。もちろん、その過程において人体への侵襲は最小限にとどめる、すなわち最小侵襲でなくてはならない。医用画像の歴史を見ると、まず初めに新しい可視化技術が実現し、続いて定量化への道が開かれるというプロセスが繰返されている。現在は両者が相まって急速な発展のさなかにある。とりわけ、人体の三次元映像を取得できるようになってきたこと、磁気共鳴映像法(MRI: magnetic resonance imaging)やアイソトープを用いた画像[PET(positron emission tomography)など]による生体組織の活動状況を示す画像が加わってきたことが大きい。今後は分子レベルの現象をとらえる分子イメージング(molecular imaging)も可能性がある。さらにこれらの具体化にあってはイメージング装置やシステムが必要である。また、画像として人が見るためには、フィルムに焼き付け、さらにそれを観察するビューワ(viewer)やシャウカステンなどの装置やコンピュータディスプレイとそれを稼働するソフトウェアが必要となる。X 線写真などのフィルムを見る伝統的手段に対して、ディスプレイ装置で見る方式をCRT 診断と呼ぶ。人が所望の情報を見るということのみが目的であれば、医用画像を得る時点で可視化は終わるが、実際はイメージング装置の出力を単純に写真などにするのみでなく、さらにさまざまな処理を加えて目標とする情報を見えるようにする。この部分を指して、特に可視化ということが多い。それは技術的にはコンピュータグラフィックス(CG : computer graphics)の領域の技術であるが、一般のCGとは異なる特徴もある。医用画像処理における可視化技術の性格を以下に記す。(1) ユーザ:医師=新たな診断情報を得る、画像認識システムの開発者=認識実験の結果としての三次元画像表示が必要、イメージング装置の開発者=表示ソフトウェアも具備すべき機能の一部をなす、一般の人=医師の説明の理解(インフォームドコンセント)に役立てる。(2) 手法:CG と画像処理を実際の応用の中でつなぐ役割。(3) 仮想化人体(仮想環境)体験ツールとして重要。(4) 実人体の存在:患者、被験者としての実人体が必ず存在し、それとの対応が正確になされること。(5) 実体標本:摘出標本や組織標本のように必ず実物が伴う領域があり、その可視化も重要である。(6) インタラクション:対話操作が可能なら、必要性が高い。(7) 知的操作の組込み:セグメンテーションやレジストレーションのような認識を伴う処理の組込みが望まれる。
