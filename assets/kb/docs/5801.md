# 16・12・2 JISにおける医療機器の規格

> **章**: 第16章 医療福祉に関わる法律・規格
> **節**: 16・12 規格

JISは19の部門に分かれ，各部門はアルファベット1文字で表
される．医療機器に関する規格は医療安全用具としてTで表示さ
れる．部門はそれぞれ内容に応じて九つ程度に分類される．これを
分類番号と呼び，医療安全用具には八つの分類がある（表16･6)．
また，医用放射線機器に関しては，二部門の「放射線（能）関係」
に記載されている．

CD-ROM版のJIS総目録を年3回有料で発行している．また，詳
細情報として規格番号，課題，価格，規格概要，履歴等がオンライ
ンで検索可能である．さらにISO，IEC，海外規格の邦訳も入手が
可能である．
その他，医療関係のJISについての資料は，厚生労働省医薬局
審査管理課で入手が可能である．
以下に各分類について簡単な説明を加える．
(１)一般：福祉関連機器用語と義肢・義足の強度試験方法，その
他医療用具，材料の試験方法に関する規格，医用電気機器の
安全性に関する規格．
(２)医療用電気器械類：医用電気機器，用語，医用差込接続器，
病院電気設備の安全基準，各種機器に関する規格として，心
電計，血圧計などの各種生体計測装置，レーザ光凝固装置，
内視鏡など光学器械の規格．
(３)一般医療器械：針器，管器，容器や糸類，電気を用いない計
器，光器，音叉の規格．
(４)歯科器械：歯科に用いられる器械類．
(５)歯科材料：歯科に用いられる材料．
(６)医療用設備・器械など：医療ガス配管設備，麻酔システム，
滅菌器や手術用顕微鏡などの医療用設備，インプラント材
料．
(７)労働安全：医療業務に従事する人のための安全防護用品に関
する規格，安全靴や化学防護服など．
(８)福祉関連機器・その他の医療用具・衛生用品：ゴム手袋など
の衛生用品，電動車いす，義手・義足，移動・移乗支援用リ
フトなどの福祉機器．
