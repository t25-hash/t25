# 15・4・3 医用画像の使われ方

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・4 医用画像

一口にいえば診療への利用に尽きるが,詳しくはさまざまの利用法があり,それらを実現するまでに必要とされる技術もそれぞれに異なる.以下に利用のタイプとしておもなものを挙げる.a. 画像診断通常の診断では,X 線写真に代表される患者の身体を記録した何らかの画像(時にはモダリティの異なる数種類)を参照しつつ診断を行う.それを表すために画像診断(imagediagnosis)という言葉が1970年代から使われるようになった.個々の診断においては,図15・5の各種画像を,静止画,動画,経時変化,造影などを含めて組合わせて用いる.b. 計算機支援診断人体を記録した(イメージングの出力の)画像を実際に人が見る前にかなりのコンピュータ処理(ディジタル画像処理)が行われるようになってきた.極端な場合,特徴量の数値やある種の病気の可能性を示す数値を示したり,画像上で異常が疑われる陰影にマークを付けたりすることもある.これらを含めて,画像から診断に至るプロセスにさらに計算機処理を積極的に利用する診断を計算機支援診断(CAD : computer aided diagno-sis)という.CAD の応用としてX 線像によるスクリーニング(集団検診)への適用が期待され,乳がん,肺がん,大腸がんなどのスクリーニング用のX 線像CAD の研究が盛んに行われている.c. 仮想化人体イメージング系の進歩によって,特定の計測値の範囲ではあるが人体各部の特性値を1mm以下の単位で計算機内に取込めるようになった.これを個々の人体を仮想化したという意味で,仮想化された人体[あるいは簡単に仮想化人体(vir-tualized human body,virtual human body)]と呼ぶ.仮想化人体を用いて,人体内を自由に移動し,見る位置,方向を自由に選んだ画像を作ることができる.特に管腔様臓器の内部を自由に移動することで内壁の状態の診断を行うものを仮想化内視鏡システム(バーチャルエンドスコピー,virutual endoscopy),さらに管腔様臓器にこだわらず人体内部を自由に移動して診断するナビゲーション診断(navigation diagnosis)が提案されている.d. 手術ガイドとシミュレーションc項で述べた仮想化人体を用いれば外科手術のシミュレーションができ,手術前の計画立案やトレーニングに利用できる.また,手術中にも手術対象に続く見えない部分を重畳表示したり,現在処置中の部位を画面上に表示したりすることもできる.これらは時には手術中の実人体の上に重畳して表示され,また操作中の手術用器具類も同時に表示したりする.これらは術者を補助,支援する効果が大きく,術中支援(intraoperative aid)または手術ナビゲーション(intraoperativenavigation)といわれる.術中支援に際しては,外科医,患者,装置,手術用器具等の正確な位置合せ(レジストレーション,regis-tration)が重要な技術的課題となる.また,必要なら手術中の患者の画像も改めて撮影することや,継続的に画像を表示してモニタすることも行われる.e. 画像データベース画像データベースは,個人の過去の画像から研究用,保管用の大規模データベースまでさまざまなものがある.各個人のものはフィルムによる保管,CRT 診断されたものはディジタル画像の保管という共存が実情であろう.今後は,各個人が自分の記録としてのディジタル画像を保存,管理する方向(個人健康データ,PHD : personal health data)の進展が期待される.
