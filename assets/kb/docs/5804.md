# 16・9・2 電波防護指針

> **章**: 第16章 医療福祉に関わる法律・規格
> **節**: 16・9 電波法

1950（昭和25）年に「電波法」（radio law）が制定されたが，そ
電波の人体への影響について，電気通信技術審議会から「電波利
の電波法では，電線路に10kHz以上の高周波電流を通じる電信，
用における人体の防護指針」［1990（平成2）年6月］および「電
電話，その他の通信設備および10kHz以上の高周波電流を使用す
波利用における人体防護の在り方」［1997（平成9）年4月］が出
る工業用加熱設備，医療用設備，各種設備については，原則として
されている．この答申は，動物実験等の結果に基づき，影響が生じ
個別の設置許可を受けるよう定めている．そのため，高周波電流を
るしきい値に約50倍の安全率を見て，人体に影響を及ぼさない電
利用するMRI装置等の高周波利用設備は，設備から電波が発射さ
波の強さの指針値等（電波防護指針）が示されている．また，今後
れ放送や無線通信に妨害を与えることが予想されるので，規制の対
の電波防護指針の取扱いおよび電波の人体への影響に関する今後の
象となっている．この許可は総務大臣から設置者（医療機関）自ら
研究項目等についてもとりまとめられている．
が受けるものである．
