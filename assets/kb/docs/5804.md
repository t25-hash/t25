# 15・5・2 家庭内情報インフラストラクチャ技術

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・5 ホームネットワークを利用した在宅システムと

家庭内に情報ネットワークを配備させる方法として、次の三点が考えられる。(1) 既存の電灯線ネットワークを利用(2) 高速LAN を設置(3) 無線環境の利用以下にそれぞれの技術の概要を述べる。a. 電灯線を利用したホームネットワークークの概要について述べてきた。その中で、アクセスラインを実現する技術の一つとして電力線ネットワークを紹介したが、このネットワークは家庭から基幹網へ接続するアクセスライン系である。ここではアクセスライン系ではなく、電力線が家庭内にすでに存在し、各部屋間とも接続されている現状から、そのネットワークをデータ転送に利用するホームネットワークの一手法について述べる。このネットワークをここでは電灯線ネットワークと称する。この電灯線ネットワークの利用を推進する約90の企業が参加する業界コンソーシアムであるホームプラグ・パワーライン・アライアンス(HomePlug Powerline Alliance)が、2001年6月にHomePlug1.0を発表している。HomePlug は4.3~20.9MHz の周波数帯を使い、最大13.75Mbpsの伝送速度を実現する。本規格はすでに北アメリカでは実証実験が開始され、そのパフォーマンス、信頼性、互換性、他の機器への無干渉性などが確認されている。本規格のメリットとしては以下のものが挙げられる。(1) すでに各部屋に配線されているアウトレットを使用可能(2) 低コスト(3) 全世界での使用が可能(4) 利用者が容易に利用可能(5) インストールが容易各種情報家電は、通常どおり100V のアウトレットに接続するだけで、電力供給と同時にネットワークアクセスが可能になる。このことは、無線LAN(local area network)がネットワーク接続ケーブルを必要としないという大きな特徴をもつことと類似しており、数種類の外部ケーブルを必要とする在宅診断システムの生体センサ系などには大変効果的なネットワークシステムである。b. 高速LAN、無線LAN などの利用
