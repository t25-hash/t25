# 15・5・3 家庭内情報インフラストラクチャを利用した新アプリ

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・7 に既存無線システムの領域とMMACの目指す領域を、図15・

ケーションマイクロソフト社はブロードバンド、無線LAN を用いた次世代ホームネットワーク構想であるeHomeを発表している。eHomeは、メディアセンタPCと呼ばれるPCを中心にテレビ、AV 機器、PC周辺機器など家庭内のあらゆる情報機器が接続され、自分用にパーソナライズされた映像などを楽しむことができる環境である。これにより、テレビとPCとの連携動作や、PCへのテレビ放送の記録などが容易に可能になったが、その他のアプリケーションも今後発表されてくるであろう。
