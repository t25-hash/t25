# 15・5・4 在宅医療システムとの連携

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・7 に既存無線システムの領域とMMACの目指す領域を,図15・

これまで述べてきたようなホームネットワーク環境とブロードバンドアクセス環境を使用すれば,遠隔医療や訪問看護環境が変化してくる.ここでは,ブロードバンドネットワークの出現の前にISDN により行われた実験例と,最近行われた実験例を示し,最後に在宅医療システムの課題について述べる.a. 福島県双葉郡葛尾村1997年,福島県双葉郡葛尾村は,マルチメディアビレッジ構想のもとにネットワーク環境を整備し,高度情報通信ネットワークを活用した村づくりを開始した.NTT はマルチメディアビレッジ共同実験に参加し,テレビ電話を使用した図15・7 MMAC の目指す領域9 ―234医療・福祉・バイオ機器遠隔医療の試みを行った.図15・9に接続構成例を示す.本実験で使用されるユーザ端末は,おもにISDN のテレビ電話と,テレビ電話に接続してバイタルデータを送信できる付加装置から構成される.ユーザは定期的にバイタルデータを送信し,必要に応じて医師とテレビ電話での対面通信が可能である.医師と対面で会話することにより,ユーザは医師との距離を感じずにコミュニケーションすることが可能となる.また,自治体各機関がネットワーク化されているため,多種の医療福祉サービスが可能となる.本実験の一環では,高齢者宅まで遠隔診断により処方された薬を郵便局員が配達するサービスも計画された.これにより,一人暮らしの高齢者などは在宅のままで診断と薬の入手が可能となる.また,薬の配達の際には人が介在しているため,受取り時の会話などによる実際のコミュニケーションも可能である.本システムでは,相手の状態を画面で確認できるため,電話に比較して安心感を提供できる.しかしながら,ISDN のテレビ電話品質では,顔色や患部の状態を見るような診断は困難であったと思われる.b. e-ライフアメニティーシステム高速ネットワークを使用した遠隔医療は,ニューメディア,マルチメディアと呼ばれた時代から検討され,各種実験も行われてきた.しかし,当時はネットワークの速度が遅く,診断に不可欠な画像を伝送することができなかった.その後,ブロードバンドネットワークの普及とともに高品質な画像が伝送可能となり,患者の状態の把握や診断も可能となってきた.ここでは,単に医師と在宅患者を結んだ在宅医療の実験ではなく,高品質な動画像を使用して入院中の小児患者のアメニティーを支援し,心の安定を与え,ひいては治癒力の向上を目指す試みについて紹介する.長期入院中の小児患者は,家族,特に母親と密に接することができないことによってストレスが生じる可能性がある.長期入院患者だけでなく短期入院の場合でも,小児患者とその母親にとっては,お互いそばにいられないことは心理的につらいものである.そこで,藤野らは小児患者のコミュニケーション環境と教育環境を支援するための患者アメニティーを支援するシステムを提案してきた.本システムでは高速ネットワークにて患者の自宅と病院を常時接続し,ベッドサイド端末と家庭用端末を介して通信を行う.通常は定期的に相手の静止画像を提示する静止画像通信モードとなり,例えばベッド上の画像と家庭居室の画像とを1分ごとに送り合う.これにより,家にいるかベッド上にいるかなどの相手の状態をある程度事前に把握することが可能となり,会話をしなくともつねにそばに図15・9マルチメディアビレッジ共同利用実験における遠隔医療の試みいてくれる感覚を提供できる.必要に応じて小児患者が「お母さん」と呼べば,その音声が相手に送信される.ここで,音声認識技術を適用して送出音声のフィルタリングを行う.通信相手がその場に存在し,呼びかけに返事をした時点から動画像通信モードに遷移する.動画像通信モードでは通常のテレビ電話のように,相手の動画像を見ながら話をすることができる.話が終了し,システム側が音声の入力を検知しなくなれば,最初の静止画像通信モードに戻る.図15・10に上記通信時の画面構成を示す.自分と相手の動画像が表示されている.本通信方式は,親と子のコミュニケーションを支援するものであり,直接の医療情報システムとはいえない.しかしながら,小児患者と親のコミュニケーションを支援することにより,小児患者の精神的安定をもたらし,それが治癒力の向上へつながるのであれば遠隔医療の一手法となりうると考えている.本システムを用いて,2002年春に,福岡市立こども病院にて実際の小児患者の家と病院を結んだコミュニケーションの試みが行われた.小児患者らは,図15・11に示すような病院に設置されたクライアント端末を使用して,見舞いに来られない家族との会話を楽しむことが可能である.これにより,小児患者らは親しい家族や友人と会えないことによるストレスを軽減することができる.本システムでは,高速のホームネットワークとアクセスラインが実現の条件である.動画像はMPEG-4と呼ばれるVHS ビデオ並みの品質を有し,臨場感豊かに会話することが可能である.常時接続された高速ネットワーク環境下であれば,通信以前にお互いの状態情報を確認でき,その後,必要に応じて高精細動画像による通信を実現すれば,お互いの安心感につながる.本システムは,一人暮らしの高齢者とその離れた家族やかかりつけ医との通信にも適用可能であり,高齢化時代での安心感通信として期待される.また,映図15・10 遠隔地の家族との通信時の画面構成図15・11福岡市立こども病院に設置されたクライアント端末図15・12 訪問看護支援システムの構成像品質もISDN テレビ電話に比較して大きく進歩しており,適用領域にもよるが,ある程度の診断も可能と思われる.c. 訪問看護支援システム医療制度の改革や介護保険の導入などにより,今後訪問看護を受ける患者が増加することは明白である.しかしながら,病院看護と異なり現場において参照可能な情報が少なく,看護師個人の判断に依存するため,その質の格差が生じやすい.また,緊急時の対応においても院内と異なり,アドバイスを得られないため適切な看護ができない場合もありえる.これらの問題点を解決するため,看護用携帯情報端末,管理用端末,データベースサーバから構成される訪問看護支援システムが提案されている.本システムの構成を図15・12に,携帯端末でハンドリング可能な情報データベースの例を表15・4に示す.本システムではカメラ付きの携帯情報端末を使用しているため,患者の容体を示しながら医師の助言を得ることができる.また,情報データベースセンターには,主治医の指示書や観察記録などの患者情報と患者記録情報が格納され,この情報に基づいた適切な対処を可能としている.訪問看護師は,患者の家に着いた時点でネットワークに接続し,各種情報の送受信を行うことができる.ここで示したシステムでは,携帯端末にPHS カードを挿入してネットワークに接続しているが,15・境を適用できればより効率的に運用できることは言うまでもない.
