# 15・5・5 在宅医療システムの課題

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 5・2 項で述べた各種ホームネットワーク環境、例えば無線LAN 環

在宅医療システムでは、画像によるコミュニケーション手段が必要条件であり、高速のアクセスネットワークが必須である。ホームネットワークはアクセスラインの高速化が普及した後に、各家庭で検討されるべきシステムであろう。通常、各家庭には電話線と100V の電灯線は配線済みであるが、それに加えて情報ネットワーク9 ―235表15・4 携帯端末でハンドリング可能な情報例データベーステーブル項目指示書主治医、医療機器、傷病等観察記録看護分野、医療機器、傷病患者情報看護段階看護分野の正常・異常、医療機器・傷病の有無看護分野栄養、脱水等看護情報医療機器経管栄養、人工肛門等傷病高血圧、糖尿病、肺炎等のラインを新たに考慮しなければならない。そのため、ネットワークの形態としては無線LAN や電灯線ネットワークが有力である。特に無線LAN は、各種情報端末との接続のみではなく健康モニタリング系のセンサなどとの接続にも使用可能であり、今後の主流になると考えられる。現在では、新築マンション等で各部屋にブロードバンドのホームネットワークをあらかじめ設置している例もある。ブロードバンドネットワークとホームネットワークを利用した遠隔診断における課題は、やはりセキュリティであろう。無線LANは、設定時にセキュリティに配慮しないと、無線であるがゆえ情報が容易に戸外にも出ていってしまう。安易な設置が自らのプライバシーの暴露につながってしまう。特に、医療情報は個人の最大のプライバシーの一つであり、その安全性は必ず担保されなければならない。医師との会話による音声情報や画像情報もセキュリティの対象になりうるものと考える。また、在宅医療システムでは、高齢者や要介護者の使用が多いことを考慮すれば、そのシステムの使用しやすさは大きな課題である。15・9節でも述べているが、将来的には、端末はユビキタス性を有し、ユーザが意識しないで病院や診療所と同じように自然な形で医師や看護師、保健師、ヘルパーと接することができるシステムが理想である。近年のわが国における高齢化の進展、生活習慣病の増加など疾病構造の変化、医療技術の高度化、あるいはそれらに基づく医療費の高騰化などに伴い、医療サービスの形態も変化してきた。すなわち、ひと昔前は病院、医院を問わず一つの医療機関のみでの閉鎖的な形での医療で十分であったが、この10年ほどの間に、医療法の改正による特定機能病院など病院の類型化、在宅医療の充実化、あるいは老人保健センターや画像診断センターなど新しい保健・医療機関の出現など、医療機能の集積と分化が進んできた。その結果、病院や医院が地域の他の保健・医療機関との間で種々の連携を保ちつつ、患者の診療を行う必要性が増してきた。さらに、高齢者保健福祉推進10ヵ年戦略(ゴールドプラン)の制定により、保健・医療・福祉の統合化サービスも要請されるようになり、これらの地域の関係機関間の連携活動がいっそう必要となってきた。このように地域における保健・医療・福祉の諸機関間での連携が行われるようになると、医療のサービス範囲は必然的に広域化することになるが、地域医療はこうした医療を意味するものである。地域医療に関する明確な定義はないが、地域住民がその生活基盤の中で自らの健康を維持し、増進させ、生活の向上を図っていくために必要な保健・医療の諸技術を組織的・体系的に提供し、人々の健康生活を支持していくための一連の活動過程であると考えられている。このような活動を進めるには、複数の保健・医療機関における異なる職種の医療職員によるチームに基づき、保健・医療サービスを組織的・体系的に実施すること、さらに高次にはその地域における保健・医療のニーズを明確にし、人、物、資金の重複投資を避けて限られた資源を有効に配置することが求められるため、サービスのシステム化を図る必要がある。この地域医療のシステム化のた9 ―236医療・福祉・バイオ機器めには、これを促す技術的基盤として地域医療における情報の流れのシステム化、すなわち地域医療情報システム(regional medicalinformation system)が不可欠ともいうべき役割を果たすものである。本節では、上記の地域医療情報システムのうちでも、地域の医療機関と他の医療機関や患者宅を通信線で結び、情報のやりとりにより診断・治療を行おうとする遠隔医療システム(telemedicine sys-tem)に焦点を絞り、若干の事例の紹介とともに概説する。
