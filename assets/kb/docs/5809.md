# 15・6・1 遠隔医療システムと遠隔診断・治療

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・6 地域医療と遠隔医療システム

旧厚生省研究班の報告書によると、遠隔医療の定義は、映像を含む患者情報の伝送に基づいて遠隔地から診断、指示などの医療行為および医療に関連した行為を行うことをいう、となっている。すなわち、地理的に離れた患者と医師や、主治医と主治医を支援する他の医師ないし医療機関との間などで行われる医療である。このような遠隔地間での情報伝達のためには、上述の定義にある映像をはじめとする何らかの情報・通信機能を必要とするが、そのためのシステムが遠隔医療システムである。このシステムの目的は、へき地や離島で勤務する医師の診療活動や、医院や一般病院の医師の要請に基づく基幹病院の専門医によるコンサルテーションを支援すること、あるいはd項で後述する在宅医療システム(homemedical care system)における高齢者を中心とする在宅患者を遠隔地の医療機関の医師が診察することなどであるが、広義には看護師など医師以外の医療従事者による医療支援や、医師間などでの臨床検討会、卒後教育なども相当すると考えられる。遠隔医療システムの具体例には表15・5に示すように診断や治療に関する各種のものがあるが、以下では遠隔診断システムとしての遠隔放射線診断(テレラジオロジー)および遠隔病理診断(テレパソロジー)と、遠隔治療としての遠隔眼科医療、および在宅医療とその一環である在宅酸素療法に関する各システムについて概説する。a. 遠隔眼科医療システム(5) 遠隔内視鏡診断システム(6) 遠隔循環器画像診断システム(7) 遠隔リハビリテーションシステム(8) 遠隔歯科診断システム(9) 遠隔手術システム(10) 遠隔健診システム路により遠隔地の専門医のもとへ伝送し、読影に関するコンサルテーションを依頼しようとするもので、遠隔医療システムのうちでも最もポピュラーなものである。この目的には大別して次の三つがある。すなわち、(1)病院に勤務する放射線専門医が自宅にいるときなど病院を離れている際に、スーツケースに格納可能な携帯型送受信装置と画像モニタを必要時に電話回線などに接続し、送られてきた放射線画像の診断を行う場合、(2)中小病院における非専門医など診断能力のレベルに格差のある医師が、大学病院などの専門医に診断を依頼する場合、(3)緊急時に患者を大病院へ輸送する前にあらかじめ放射線画像などの医用画像を伝送しておき、輸送先の病院での手術などの処置の準備に役立てる場合などである。(2)に関連するものであるが、最近では料金を支払って読影を依頼する商用システムも登場している。システムとしては、PC通信による簡単な構成のものも用いられている。b. テレパソロジー(telepathology)テレパソロジーは、患者から採取した細胞や組織を顕微鏡観察が可能な標本の状態にしたうえで、病理専門医のいる大病院へ通信手段を用いて標本画像を伝送して診断を依頼するシステムである。このようなシステムが必要となるのは、わが国の病理専門医が現在2000人にも満たないため、病理専門医のいる大学病院や大病院以外の多くの病院では緊急を要する病理診断ができないからである。ここで緊急性を要するというのは、例えば胃潰瘍の診断のもとに手術を始めたものの、胃がんとの鑑別が困難なため、手術中に患者から採取した組織についてただちに顕微鏡診断を行うことにより確定診断を必要とする場合などである。これを術中迅速診断というが、迅速診断が必要なのは、確定診断の結果、悪性腫瘍であるかないかにより手術の方法が大幅に異なるからである。すなわち、従来のように組織を郵便で病理専門医に送り数日後に返事を受け取る方法では、胃潰瘍の手術を施した後、すなわち患者の腹部をいったん閉じた後、悪性腫瘍ならば再手術というやっかいなことになるが、もしテレパソロジーにより迅速診断ができれば、ただちに胃の病巣部切除と周囲のリンパ節郭清など適切な手術を行うことが可能となる。テレパソロジーのシステム構成図の一例を図15・13に示す。本システムの通信手段には、ISDN のほか光ファイバを用いたもののほか、コストは別にして実験段階ながら衛星通信によるものもある。また、テレビ画像としては通常のNTSC(National Tele-vision System Committee)方式のものも用いられているが、システムにより映し出される画像としては、顕微鏡で観察されるものと同じ程度かそれ以上の画質であることが要求されるため、高品位テレビ(HDTV : high definition TV)カメラとモニタが使用されているシステムが少なくない。c. 遠隔眼科医療(teleopthalmology)旭川医科大学には1994年から遠隔医療センターが設置されているが、この施設を利用して遠隔眼科医療システムが機能している。本システムにより、図15・13テレパソロジーシステムの構成関連病院で実施する眼科手術を円滑に進めるために、手術適応の判断のような術前診断、あるいは手術がうまくいったかどうかについてや術後合併症が生じたことに対するコメントなどの術後診断が実施されている。このため、旭川医科大学と関連病院との間をISDN(INS 1500)で結び、細隙(げき)灯顕微鏡、検眼鏡、手術用顕微鏡などの眼科画像情報のやりとりをする伝送システムを使用している(図15・14)。d. 遠隔在宅医療システムと在宅酸素療法(home oxygen ther-apy)システム在宅医療の必要性についてはいまさら言うまでもないが、昨今の著しい少子高齢化により、在宅医療の主対象である高齢者の増加に対して医療マンパワーが相対的に減少し、今後、医師や看護師などによる訪問医療・訪問看護は次第に困難となり、訪問回数を減らさざるを得ない。そこで、医用工学(ME: medi-cal engineering)や情報通信技術(ICT)などを応用することにより、マンパワー不足への対処のみならず、在宅医療の質的向上を図図15・14 旭川医科大学附属病院遠隔医療センターの関連病院眼科情報伝送システム図15・15 PHS を用いた心疾患患者テレモニタリングシステム(telemonitoring system for patients with heart diseases)の構成9 により伝送することのできる機能を有している。PHS を用いているため、患者の位置情報も同時に送ることが可能なことから、救急医療が必要な場合、患者の位置特定にも有用である。このため本システムは、発作を恐れるあまり、家に閉じこもりがちな高齢心疾患患者QOL(quality oflife)の向上に資する点が少なくないと考えられる。遠隔在宅治療システムに関する事例としては、在宅酸素療法システムを挙げることができる。在宅酸素療法の対象患者は、結核後遺症や塵肺などによる慢性呼吸不全症が主であるが、最近の高齢化の進展により患者数が増加している。これには適切な濃度の酸素を患者に効率よく供給可能な装置が必要となるため、取扱いが容易で、必要な酸素吸入を的確に行うことのできる酸素濃縮装置が開発・使用されているが、在宅酸素療法の安全な実施には、患者の状態の把握に基づく的確な管理が不可欠である。そこで、パルスオキシメータと電話回線を用い、リアルタイムに、あるいはメモリカードを介して在宅患者の経皮的動脈血酸素飽和度を評価するシステムが用い表15・6
