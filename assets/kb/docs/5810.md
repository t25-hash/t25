# 15・6・2 遠隔医療システムに関する今後と問題点

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・6 地域医療と遠隔医療システム

わが国における遠隔医療システムは,1970年代の初めに稲田らが実施した和歌山県山間部における実験に端を発する.当時は,画像をはじめとする情報伝送など技術的制約が大きかったため実験に終わってしまったが,この30年ほどの間におけるコンピュータをはじめとする情報技術,あるいは広帯域通信回線や衛星通信といった通信技術などの各種技術の進歩と,冒頭に述べた高齢化の進展などによる医療機関連携のような地域医療に関するニーズの拡大は遠隔医療の実現を促し,その試行や実用化が進められつつある.したがって,今後は遠隔手術といった高度のものについても,国内はもちろん国際的にも遠隔医療システムの開発と利用がなされることが期待される.しかしながら,こうした遠隔医療が広く普及するためには,まだ解決すべきいくつかの課題が存在する.ここでは技術的課題以外のものも含めて,このような諸課題を表15・6に掲げ(詳細は割愛),本節の結びとする.
