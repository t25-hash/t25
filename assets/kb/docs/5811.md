# 15・7・1 はじめに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・7 遠隔リハビリテーション

近年、わが国では急速な少子高齢社会の進行により、保健、医療、福祉の連携システムの構築が課題になっている。家族構成は二世代同居世帯の率が低下し、高齢者世帯、一人暮らしの高齢者の増加、老老介護など介護力の低下が大きな社会問題となっている。一方、情報通信技術の革新、進歩により、大容量の音声や、画像の双方向通信が高速で可能となっている。これらの状況から、高齢者や障害者の身体機能の回復、日常生活動作能力の向上、介護者に対する介護指導など生活の自立を目指した取組みが必要となっている。こういった事情を反映し、病院、医療機関以外の遠隔地でもリハビリテーション診療を受けられる遠隔リハビリテーション(rehabilitation distance)の構築について述べる。
