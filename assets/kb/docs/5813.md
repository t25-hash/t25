# 15・7・3 装置について

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・7 遠隔リハビリテーション

(株)日立製作所では、1995年から遠隔医療システム(telemedicine system)を応用した遠隔リハビリテーションの送信テストを行ってきた。このシステムは病院、医療機関に設置されるセンター装置と、患者宅、保健センターに設置される端末装置から構成されている。センター装置と端末装置はNTT が提供するISDN 回線(INS 64)で接続され、双方向画像と音声および端末側から生体医療情報データの通信を行う。また、センター装置は回線を経由して複数の端末装置と接続され、任意に選択された端末装置との通信が可能となっている。このシステムにさらに、(株)日立製作所が独自に開発した2ベルト型歩行訓練機を連携させ、身体機能の回復を図るのが遠隔リハビリテーションシステムである(図15・17)。この歩行訓練機の特徴は、歩行能力に応じた歩行条件を自動的、他動的に設定できることである。また、歩行面が2ベルトになっており、片麻痺患者等、左右下肢の能力差に応じて最適の歩行条件が設定でき、歩容改善訓練を行う。テレビモニタからは、歩行状態の観察、評価ができるように、歩行速度、距離、歩数、左右の重心の移動、左右下肢にかかる力の表示が可能であり、また、各種生体情報の表示、データの保存、再生ができる。これらにより、遠隔地の病院、医療施設から送られてくる映像、生体情報、音声会話による適切で効果的な遠隔指導、リハビリテーションが可能である。このシステムは、ブロードバンドインタネット回線で送受信可能であり、生体情報としては、各種センサを付けることにより、血圧、脈拍、心音、呼吸、酸素飽和度、体温等のバイタルサインをはじめ、歩行速度、バランス、患部の状態、動作の指導、確認が可能である。また医師、理学療法士、訪問看護師、ホームヘルパー、訓練者、家族等が双方に会話をしながら意思疎通を図り、訓練を実施することが可能である。センター装置では、リモートコントロールカメラをコントロールしながら、またマイクを通して会話をしながら指示を与え、訓練状況を確認し、指導することができる。生体、訓練情報データはデータベースに記録保存することで訓練効果の判定、訓練プログラム設定に利用できる。
