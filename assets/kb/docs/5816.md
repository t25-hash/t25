# 15・8・2 Health Hero Platform

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・8 健康モニタリングシステム

アメリカでは、糖尿病、喘息、心臓疾患などの慢性病にかかっている人が約9000万人に達しており、医療費の約75%は慢性病に費やされているとされる。ヘルス・ヒーロー・ネットワーク社は、Health Hero Plat-form(ヘルス・ヒーロー・プラットホーム)というネットワークと専用端末を使用したヘルスケア商品を発表している。図15・20にケアサービス側であるケアプロバイダとクライアント側である患者のコネクションの様子を示す。ケアサービス側にはケアマネージャや医師がいて、患者への質問や回答、患者から入力されたデータの分析を行う。図15・21に専用端末の外観を示す。この端末を電話線に接続すると、ネットワークを介していろいろな質問や入力事項がケアサービス側から送信され、ユーザは毎日端末からバイタルデータや質問に対する回答などを入力する。データの入力は、センターから送信される簡単な質問に答える形で付属の4ボタンを押すだけで行えるため、高齢者でも問題なく使用することが可能である。データは専用端末のほか、携帯電話、PDA,PCなどからも転送可能であり、またそのデータ接続にはセキュリティを考慮したネットワークを使用している。図15・22にネットワークセキュリティの概要を示す。各端末からのデータは暗号化されたセキュアコネクションで接続され、データセンターではネットワーク事業者による24時間体制でのセキュリティを確保している。本システムはバイタルセンサが自動的に情報を収集してセンター表15・7デジタルエンジェル社健康モニタリングシステムにおけるおもな仕様警報の種類機能起動方法緊急コール手助けが必要ユーザ自身転倒コール起き上がることができない自動徘徊情報設定領域からの逸脱自動異常体温異常体温自動充電警告充電警告自動に送信しているタイプではなく、センターから送信される各種質問に答える形でデータを入力する方法をとっており、冒頭で述べた分類に従えば広義の健康モニタリングシステムに含まれる。本方法の特徴である四つボタン方式により、機器の操作は極めて簡単になるが、逆に体温、脈拍などのバイタル情報を得るためには他の機器を使用しなければならず、それらのデータを四つのボタンを用いて入力する必要がある。ユーザのコメントとしては、毎日の歯磨きやヘアブラッシングと同じように、慣れてしまえば毎日データを入力することの困難さはほとんどないとの報告がある。また、毎日センターから質問事項が送信されてくるため、それに答えることにより自分の健康管理意識が高まる効果もあると考えられる。簡易化された自動的な入力手段と、ユーザ意識が高まる手動入力と、どちらが望ましい手段であるかは現段階では判断が困難であるが、ユーザの適性に応じてどのような手段を用いるかを選択することが重要であろう。
