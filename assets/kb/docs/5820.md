# 15・9・1 はじめに

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

21世紀の日本は5人に1人が高齢者という世界にも類をみない高齢化社会を迎える.高齢化時代には,IT 技術により高齢者の健康状態や行動状況をモニタリングするシステムが必須である.健康状態をモニタリングするシステムは15・8節にて述べられた健康モニタリング技術によりカバーすることが可能であるが,痴呆症等により徘徊する高齢者に対処するシステムとしては,緊急時に速やかに徘徊高齢者の位置を確認し保護できる位置検出システムが基本技術となる.本節では,徘徊高齢者の位置を検出するシステムとして,現在試験運用や商用化されているシステムについて紹介する.
