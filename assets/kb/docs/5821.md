# 15・9・2 位置検出方法

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

位置検出機能は簡易型携帯電話として開発されたPHS やカーナビゲーションに用いられているGPS を利用し,それらの位置情報取得機能と情報伝達機能を組合わせたシステムが開発されている.表15・8にGPS 内蔵型携帯電話とPHS 単体による位置検出法の特徴を示す.これらの特徴を簡単にいえば,GPS 型は,位置精度は高いが屋内の使用が困難であること.PHS 型は,位置精度は低いが屋内,特に地下街などでの使用も可能であること,かつ小型化が容易なことである.GPS 型の欠点改良型として,GPS 電波の届かない建物内での測位に,複数の携帯基地局の位置情報からの電波の遅延差を利用する方法もある.これらの方法以外に,各種認識技術やセンサを組合わせて,高齢者がどのような状態にあるかを把握するシステムも提案されている.このような高齢者の状態を把握する機能は15・8節にて述べられた健康モニタリングシステムの範ちゅうに入る.これら位置検出機能は,携帯電話やPHS などの既存ネットワークの利用と,15・2節で述べた高速ホームネットワークや表15・8 GPS 内蔵携帯電話とPHS による位置検出法の特徴GPS 機能により測位高い(条件がよければ数m)つねに衛星に向かう必要あり屋外のみで,かつ衛星が見通せる位置GPS 内蔵型携帯電話(GPS 機能のみで位置検索)屋外ではGPS 機能により測位し,衛星電波不感時には複数の携帯基地局位置情報から現在位置を計算GPS の場合は高い(条件がよければ数m),基地局情報からの推定時には100m程度GPS と携帯電話(基地局情報で位置検索)PHS 基地局情報により位置を検出低い(100m前後) ポケット,かばん内でも可能地下街,駅構内PHS 受信時の最終基地局情報であり,移動による誤差のみPHS(基地局情報で位置検索)アクセスネットワークを組合わせたシステムとして実現される.以下に,位置検出システムの概要と実施例について述べ,これらのシステムの課題を解決するため,既存のシステムに新技術を付加した新規位置情報システムや,GPS とPHS 方式以外のシステムについて簡単に紹介する.最後に,近年注目されているユビキタスコンピューティング環境と徘徊高齢者用モニタリングシステムとの関連性について述べる.
