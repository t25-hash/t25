# 15・9・3 GPS を基本にしたサービス

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

GPSは，地球の周回軌道を回る24個のGPS衛星から地上に放
射される位置測定用の電波を用いて，利用者の現在地（緯度・経
度・高度）を得るためのシステムである．カーナビゲーションに使
用されているGPSはアメリカ国防総省が軍事用として開発したも
のを民間に開放しているもので，従来は安全保障上の問題から精度
を意図的に落としたデータが提供されていたが，2000年5月2日
からこの処理を取り止めるとアメリカ政府より発表があり，その措
置によって精度が最大で10倍ほど向上することとなった．測位法
としては，1個の受信機にて4個以上の衛星から観測点までの距離
を同時に知ることにより観測点の位置を決定する単独測位法，2台
以上の受信機にて同時に4個以上の衛星から受信機に到達する時間
差を測定して2点間の相対的位置関係を測る相対測位法，位置が既
知の基準局と位置を求めようとしている観測点で同時にGPSによ
る測位を行い，ネットワークを介してリアルタイムに情報交換をし
て観測点の正確な位置を得るDGPS（diﬀerential GPS）などがあ
る．カーナビゲーションなどはおもに単独測位法が使用され，数十
mの精度で位置を測ることができる．
近年では，携帯型のGPSだけではなく，GPS受信機が内蔵され
た携帯電話も市販されている．また，GPS機能のみの測位では建
物内での測位が困難であるため，その欠点を解消した方法として，
建物内に入ったときには複数の携帯基地局と情報交換し，その電波
の遅延差から位置を推定する機能を付加した方法も実用化されてい
る．
徘徊高齢者の位置検出システムとしては，この携帯電話型GPS
を使用するものと，ペンダントなどにGPS受信機を内蔵させた専
用型に大別できる．以下に，携帯電話と専用受信機を用いた位置検
出サービスについて概説する．
a.携帯電話を基本にしたサービス (株) NTTドコモはDLP
（DoCoMo location platform）と称した位置検出サービスを2001
年に開始した．本サービスによって提供されるのは，はん用的
な位置情報提供とアプリケーションの作成を支援するASP（appli-
cation service provider）の2種類である．
.位置情報提供 本サービスは以下の機能を有している．
(１)自己位置検索機能
(２)自己位置通知機能
(３)自己位置登録機能
(４)第三者位置検索機能
このサービスは，個人会員や高齢者施設などが利用できる．
.ASP機能 本サービスは以下の機能を有している．
(１)グループ情報管理機能
(２)ゾーン監視機能
(３)プッシュ型情報配信・管理機能
このサービスは，これらの機能を利用したASPサービスの提供
を検討している法人ユーザを想定している．
KDDI (株)は2001年に，同社の携帯電話にGPSを搭載させた位
置サービスを開始した．屋外では平均6個のGPS衛星からの電波
を受けて位置を測定するもので，最もよい条件のもとでは数mの
誤差で測位可能という．ただし，屋内や地下街などGPS衛星の電
波が受信できない場所では，携帯電話の基地局の位置から場所を特
定することになり，この場合の誤差は数百mから数kmに及ぶこ
ともあるため，徘徊高齢者の位置特定には課題が残る．
セコム(株)では，GPSと携帯電話基地局情報をもとにした位置
検索，緊急対処サービスを始めており，緊急時にはセコムの警備員
が携帯電話の位置情報を頼りに駆け付けることも可能である．
b.位置検出と状態認識機能を有するサービス (株)日立製
作所は携帯電話機能とGPS受信機能を有した専用端末に体動セン
サを内蔵させ，端末を装着した人の歩く，走る，立ち止まる，倒れ
るなどの状態を受信側のPCでリアルタイムに確認できるシステム
を実用化している．位置情報を取得したい高齢者には，携帯電話と
それに接続された専用端末を携帯してもらい，家庭などから探索依
頼があった場合，管理センターから専用端末へ電話をかける．その
着信を受けた専用端末はGPSアンテナから位置情報を受け取り，
位置情報と端末利用者の状態を携帯電話の電波を使って自動的に管
理センターに転送する．管理センターは，PCにより高齢者の位置
情報と状態を確認することができる．図15･26にPCでの位置情報
の表示例と本システムの概観を示す．高齢者の状態表示は立ち止ま
る，歩く，走る，倒れるの4様態を示すことができる．携帯電話を
除く専用端末本体のサイズは幅62mm，厚さ36mm，高さ97mm
で，重さは160gと比較的小型，軽量化が図られている．
