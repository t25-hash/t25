# 15・9・3 GPS を基本にしたサービス

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

GPS は、地球の周回軌道を回る24個のGPS 衛星から地上に放射される位置測定用の電波を用いて、利用者の現在地(緯度・経度・高度)を得るためのシステムである。カーナビゲーションに使用されているGPS はアメリカ国防総省が軍事用として開発したものを民間に開放しているもので、従来は安全保障上の問題から精度を意図的に落としたデータが提供されていたが、2000年5月2日からこの処理を取り止めるとアメリカ政府より発表があり、その措置によって精度が最大で10倍ほど向上することとなった。測位法としては、1個の受信機にて4個以上の衛星から観測点までの距離を同時に知ることにより観測点の位置を決定する単独測位法、2台以上の受信機にて同時に4個以上の衛星から受信機に到達する時間差を測定して2点間の相対的位置関係を測る相対測位法、位置が既知の基準局と位置を求めようとしている観測点で同時にGPS による測位を行い、ネットワークを介してリアルタイムに情報交換をして観測点の正確な位置を得るDGPS(differential GPS)などがある。カーナビゲーションなどはおもに単独測位法が使用され、数十mの精度で位置を測ることができる。近年では、携帯型のGPS だけではなく、GPS 受信機が内蔵された携帯電話も市販されている。また、GPS 機能のみの測位では建物内での測位が困難であるため、その欠点を解消した方法として、建物内に入ったときには複数の携帯基地局と情報交換し、その電波の遅延差から位置を推定する機能を付加した方法も実用化されている。徘徊高齢者の位置検出システムとしては、この携帯電話型GPSを使用するものと、ペンダントなどにGPS 受信機を内蔵させた専用型に大別できる。以下に、携帯電話と専用受信機を用いた位置検出サービスについて概説する。a. 携帯電話を基本にしたサービス(株)NTT ドコモはDLP(DoCoMo location platform)と称した位置検出サービスを2001年に開始した。本サービスによって提供されるのは、はん用的な位置情報提供とアプリケーションの作成を支援するASP(appli-cation service provider)の2種類である。. 位置情報提供本サービスは以下の機能を有している。(1) 自己位置検索機能(2) 自己位置通知機能9 ―243位置検出法位置精度受信機利用できるロケーション電波不感時の位置検索通信機能大きさ携帯基地局の情報となり、誤差は数百m携帯電話と組合わせた場合は通話可能、専用端末の場合には不可携帯電話との組合せでは携帯電話並みの大きさ、専用端末の場合は小型化可能衛星不感地域でも可能衛星が見通せる位置に加え、建物内GPS 不感時には、複数の携帯基地局情報から位置を計算し、推定対応PHS の場合は通話可能、専用端末の場合は不可重さ数十g、ペンダント型が可能(3) 自己位置登録機能(4) 第三者位置検索機能このサービスは、個人会員や高齢者施設などが利用できる。. ASP 機能本サービスは以下の機能を有している。(1) グループ情報管理機能(2) ゾーン監視機能(3) プッシュ型情報配信・管理機能このサービスは、これらの機能を利用したASPサービスの提供を検討している法人ユーザを想定している。KDDI(株)は2001年に、同社の携帯電話にGPS を搭載させた位置サービスを開始した。屋外では平均6個のGPS 衛星からの電波を受けて位置を測定するもので、最もよい条件のもとでは数mの誤差で測位可能という。ただし、屋内や地下街などGPS 衛星の電波が受信できない場所では、携帯電話の基地局の位置から場所を特定することになり、この場合の誤差は数百mから数kmに及ぶこともあるため、徘徊高齢者の位置特定には課題が残る。セコム(株)では、GPS と携帯電話基地局情報をもとにした位置検索、緊急対処サービスを始めており、緊急時にはセコムの警備員が携帯電話の位置情報を頼りに駆け付けることも可能である。b. 位置検出と状態認識機能を有するサービス(株)日立製作所は携帯電話機能とGPS 受信機能を有した専用端末に体動センサを内蔵させ、端末を装着した人の歩く、走る、立ち止まる、倒れるなどの状態を受信側のPCでリアルタイムに確認できるシステムを実用化している。位置情報を取得したい高齢者には、携帯電話とそれに接続された専用端末を携帯してもらい、家庭などから探索依頼があった場合、管理センターから専用端末へ電話をかける。その着信を受けた専用端末はGPS アンテナから位置情報を受け取り、位置情報と端末利用者の状態を携帯電話の電波を使って自動的に管理センターに転送する。管理センターは、PCにより高齢者の位置情報と状態を確認することができる。図15・26にPCでの位置情報の表示例と本システムの概観を示す。高齢者の状態表示は立ち止まる、歩く、走る、倒れるの4様態を示すことができる。携帯電話を除く専用端末本体のサイズは幅62mm、厚さ36mm、高さ97mmで、重さは160g と比較的小型、軽量化が図られている。
