# 15・9・4 PHS を基本にしたサービス

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

a. PHS を利用した徘徊高齢者位置検索サービス(1)(株)NTT ドコモはPHS の位置情報を利用した位置検索システムを提供しており、徘徊高齢者や子供の位置把握に利用できるとしている。専用端末は幅34mm、厚さ16mm、高さ51mm、重さ27g(電池を含む)で、図15・27に示すような小型、軽量化が図られたペンダント型である。位置精度は100m程度であり、位置情報結果はインタネットを介した地図上への位置情報表示、iモードによるアクセスでの位置情報表示、ファックスによる位置情報提供など9 ―244医療・福祉・バイオ機器図15・26高齢者の位置情報表示例と状態表示、端末外観図15・27 PHS を利用した携帯型位置検出用端末例(1)緊急押しボタン付きペンダントLC501サイズW 47×D 61×H 17.9mm質量約50g連続通信時間約6時間連続待受時間約300時間図15・28 PHS を利用した携帯型位置検出用端末例(2)によって知ることができる。b. PHS を利用した徘徊高齢者位置検索サービス(2)東芝(株)が開発したシステムを用いた徘徊高齢者位置検索サービスは、専用のPHS 端末を高齢者に所持してもらい、PHS 端末の機能を用いた位置検索サービスによって高齢者の位置情報を通知するシステムであり、すでに2000年度から神奈川県川崎市などでサービスが開始されている。位置検索時のサービスの流れは以下のとおりである。(1) 高齢者の位置がわからなくなったとき、家族が位置検索センターに連絡し、位置検索依頼を行う。(2) 位置検索センターのオペレータは検索PCにより該当高齢者が所有する専用PHS の位置を検索し、電話にて依頼者に返答する。(3) 依頼者は位置情報に基づき、高齢者を保護する。本システムは、全国のPHS サービスエリアを利用できること、地下街や駅構内での位置も検出可能なこと、位置精度は平均約70m、専用端末は約50g 程度で緊急通報ボタン付きの小さなペンダント型であることなどの特徴を有する。図15・28に本専用端末の概観を示す。
