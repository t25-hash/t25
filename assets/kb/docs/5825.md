# 15・9・6 RFID を利用したユビキタス環境

> **章**: 第15章 医療福祉の情報インフラストラクチャ
> **節**: 15・9 徘徊高齢者用モニタリングシステム

近年,RFID(radio frequency identification)はユビキタスコンピューティングと呼ばれる環境の基本技術として注目を浴びている.ユビキタスコンピューティングとは,従来はあらゆるコンピュータが世界に遍在し,それらを自分のコンピュータと同じ環境で使用できる環境という意味であったが,最近では,あらゆるものが簡単な情報処理機能とネットワーク機能を有し,それらが独自に情報交換可能な環境として近未来のコンピュータ環境を意味するようにリーダ側から電磁波を送出し,タグ側で検波して動作電力とする.その電力を用いてID を返信する図15・32 パッシブ型RFID の動作原理なってきた.すなわち,ユビキタスコンピューティング環境とは,もの自身が無線などにより情報を発信可能であるような環境である.RFID は無線タグと呼ばれ,無線により自分のID を発信してその存在を確認するものであり,ユビキタス環境下ではあらゆるものに具備されるものと想定される.RFID は無線によるバーコード認証に近い考え方であり,15・9・4項で示した専用のペンダント型端末や微弱電力無線によるシステムはまさにRFID そのものである.なお,ペンダント型システムのインフラストラクチャはPHS や携帯電話ネットワークを使用しており,これらの携帯電話やPHS 電話も広義な意味でRFID の一つと考えることもできる.RFID には,自身に小型電池を有し定期的に電波を発信してその存在を示すアクティブ型と,電池をもたずに外部レシーバからのマイクロ波や電磁誘導を受けて電波を発するパッシブ型の2種類がある.図15・31に代表的なRFID の種類を示す.上述してきた携帯電話とGPS の組合せシステムやPHS システムなどはいずれもアクティブ型に属し,自分の位置を積極的に知らせることが可能である.ユビキタス環境では,これらRFID が基本技術の一つであり,活発な開発が続けられている.図15・32にパッシブ型のRFID の動作原理について示す.図15・32からわかるように,パッシブ型では電源を必要としないため,あらゆるものに埋め込むことが可能と考えられる.しかしながら,徘徊高齢者用システムとしてはID を比較的広範囲に送出可能なアクティブ型タグの採用が適当である.9 ―245図15・31 代表的なRFID の種類タグの装着方法としては,高齢者に意識的に装着してもらうものではなく,衣服や腕時計,携帯電話などにあらかじめ埋め込まれ,無意識的に装着するようになるであろう.このようにユビキタス環境として,タグなどのもの自身が無意識のうちに情報を発信し通信することの可能な環境では,徘徊高齢者位置モニタリング専用の特別なシステムではなく,一般の位置検出システムのサービス分野の一つとして高齢者をモニタリングする機能を提供することになる.こうしたシステムは高齢者だけでなく,幼児や児童,ペットも検知可能であり,果ては営業員や流通物品までも管理可能なシステムとなる.来るべきユビキタス社会は,その使い方の是非は別として,モニタリング機能が当たり前のものとなるだろう.文献(1)
