# 16・9・1 医用電気機器への電波の影響を防止するための携帯電

> **章**: 第16章 医療福祉に関わる法律・規格
> **節**: 16・9 電波法

話端末等の使用に関する指針医療機関内に持ち込まれる携帯電話等について,不要電波問題対策協議会医用電気機器作業部会から「医用電気機器への電波の影響を防止するための携帯電話端末等の使用に関する指針」が出されている.この指針は,1995(平成7)年度から1996(平成8)年度にかけて使用されていた機器について実施した実験データをもとに作成している.具体的には,医療機関内の屋内における無線設備の利用,植込み型心臓ペースメーカ装着者の注意事項,医療機関以外での医用電気機器の使用,植込み型心臓ペースメーカ装着者および補聴器使用者への配慮等について情報提供を行っている.
