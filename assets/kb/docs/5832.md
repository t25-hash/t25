# 1・2・2 酸性雨の実態

> **章**: 第1章 地球規模の環境問題
> **節**: 1・1 に世界各国の(a)SO、(b)NOの人工的発生源の排出量を

酸性雨問題(acid rain problem)は最初に産業革命がなされたヨーロッパで始まり、次いで北アメリカ北東部、さらには日本、中国等を含む東アジアで問題となりつつある。図1・3は環境省の酸性雨対策調査による日本の酸性雨の状況を示している。1・2・1項で述べたように環境の酸性化問題を考えるうえで硫酸塩、硝酸塩の大気中濃度、乾性沈着量も併せて見る必要があるが、ここでは紙幅の関係で雨の酸性度のみを示している。日本においてはヨーロッパ東部、北アメリカ東部に比べてpH はやや大きいものの、西日本から中部日本にかけてpH 4.7程度となっている。図1・4にヨーロッパ、北アメリカ、中国の酸性雨の状況を示している。ドイツから東ヨーロッパ・北ヨーロッパにかけてpH 4.5以下の地域、アメリカの北東部とカナダ南東部との国境付近にpH 4~5の地域が大きく広がり、越境汚染として問題になっている。中国内陸部の工業地帯にはpH 5以下の地域があり、今後産業の発展に伴い硫黄分の多い中国産石炭の使用量が増大すると、中国内の酸性雨問題と併せて風下側の日本や韓国への影響が懸念される。これを未然に防ぐためには、発生源での脱硫等の対策が不可欠である。
