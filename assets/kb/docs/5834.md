# 1・3・1 オゾン層破壊(ozone layer depletion)の現状

> **章**: 第1章 地球規模の環境問題
> **節**: 1・3 オゾン(O)層問題

オゾン層(ozone layer)の破壊が初めて観測でとらえられたのは1982年10月のことで、場所は南極の昭和基地上空、発見者は日本の気象研究所の忠鉢繁研究官である。その後、南極では人工衛星による観測も加わって詳しい調査が繰返し行われ、南極上空では毎年9~11月ごろにオゾンホール(ozone hole)と呼ばれるOが極端に少ない領域が現れることがわかった。オゾン量の変化は、一般にオゾン全量と呼ばれる地上から上空まですべての大気に含まれるOの総量の変化で表す。オゾン全量は大気層すべてのOを0°C,1atmにしたときの厚みで測ることになっており、全球平均的には約3mmの厚みとなる。気象学ではオゾン全量の単位としてセンチでの厚みの1000分の1(ミリアトムセンチメートル、m atm-cm)を使うので、3mmは約300m atm-cmということになる。大気全量に比べれば極めて量の少ない成分であるが、その大部分は成層圏の地上から高さ20~30kmのところに存在し、オゾン層と呼ばれる厚さ10km程度の層を形成している。気象庁では、南極のオゾン層がどのように破壊されてきたかを量的に示すために、三つの指標を定義して毎年発表している。一つはオゾンホールの大きさを示す値で、オゾン全量が220m atm-cm以下になった領域の広さで表す。この値は1980年以前にはなかったといわれている低い値である。次は壊されたオゾン量を示すもので、オゾンホールを300m atm-cmに戻すのに必要なオゾン総量で表す。三つめはオゾンホールでの最低のオゾン全量である。これらを示すと図1・6となる。この10年間、次第にオゾンホールが大きく、深くなっていることがわかる。ちなみに南半球のオゾン全量は16億5000t(トン)程度なので、オゾンホールで消失するオゾン量はその5%にも達することになる。最近では、南極圏だけでなく北極圏でもOが減少していることが観測されており、南極ほどではないがオゾンホールができているようである。さらに、観測により南北両半球の中緯度でも成層圏のOの減少が確認されている。それでは日本の上空はどうなっているのであろうか。国内では4地点で観測が行われている(図1・7)。その結果を見ると、南の那覇や鹿児島ではこの20年で大きな変化はないが、北に行くほど減少していることがわかる。
