# 1・3・2 オゾン層破壊のメカニズム

> **章**: 第1章 地球規模の環境問題
> **節**: 1・3 オゾン(O)層問題

ここで、オゾン層破壊のメカニズムを考える。対流圏の酸素分子は次第に上空へと拡散して成層圏に達し、成層圏下部において100~200nmと波長の短い日射紫外線に当たり、酸素分子は二つの酸素原子に分解する。分解された原子はすぐに酸素分子と結合してOを生じる。生成したOは酸素原子と反応して再び酸素分子に戻る(図1・8)。これらの反応が成層圏でバランスをとりながら行われて、オゾン濃度が平衡状態になり、オゾン層を形成する。こうして形成されたオゾン層にクロロフルオロカーボン(フロン、chloro-fluorocarbon)が混入してきたとしよう。フロンは安定で紫外線の弱い対流圏ではあまり分解されず、徐々に成層圏に拡散する。成層圏に達したフロンは波長の短い210nmの強い紫外線に当たって分解し、自由な塩素原子を生じる。塩素原子は周囲にあるOと反応して酸素分子と一酸化塩素になるが、一酸化塩素は酸素原子と反応して再び塩素原子に戻る(図1・9)。こうしていったん作り出された塩素原子はあたかも触媒のように繰返しOを破壊していく。ハロンに含まれる臭素も同様な作用をすると考えられている。このような反応はオゾン層をどんどん破壊していくが、実際には南極を除いて急激なオゾン減少は生じていない。これは遊離した塩素原子の大部分がOと反応しない不活性塩素、塩酸や硝酸塩素に図1・7 日本のオゾン全量年平均値の変化(●印は札幌、つくば、鹿児島、那覇におけるオゾン全量の観測開始から2002年までの年平均値、直線は全期間の長期的な傾向を示す)図1・6 オゾンホール3指標の経年変化図1・8 成層圏でのオゾン発生・消滅なっているためと考えられている。それでは、南極だけでなぜオゾンホールのような激しいオゾン層の破壊が生じているのだろうか。南極のオゾンホール形成のようなOの急激な分解には上記の分解反応のほかに、さらに別のメカニズムが働いていることがわかっている。南極大陸の上空には冬季に安定した気流の渦(極渦)が生じて、大気が孤立する。その大気は極夜の条件下で赤外線放射冷却により気温がどんどん下がり、-80°C以下にまでなって、南極大陸上空に極成層圏雲といわれる硝酸や水が結晶化した氷晶核からなる雲が生成される。この氷晶核の核の表面で塩酸や硝酸塩素の反応が進行して硝酸が氷晶核に取込まれ、大気中に塩素分子が放出される。一方、氷晶核は成長して成層圏下部へと重力沈降し、塩素分子がオゾン層に取り残される。このような塩素分子の多い状況下で、南極がやがて春(9~10月)になり太陽光が当たるようになると、紫外線によって塩素分子は分解し原子になる。この塩素原子が前述のプロセスで大量のOを分解して、オゾンホールが形成される。夏が近づくと極渦が消滅して大気の南北交換が進み、やがてオゾンホールも解消する。
