# 1・4・2 地球温暖化の仕組み

> **章**: 第1章 地球規模の環境問題
> **節**: 1・4 気候変動(climate change)と地球温暖化

COの温室効果(greenhouse effect)を説明する前に、大気の温度がどのようにして決まるのかを考えてみる。日中は太陽からの日射エネルギーにより地面が暖められ、それに接している大気も暖められる。また、夜間は地表面が赤外線の形で熱エネルギーを上空に放出することにより冷え、気温も下がる。このように地面、大気等の地球系と宇宙空間のエネルギーのやりとりは日射(短波放射)、赤外線(長波放射)などの放射エネルギーの形で行われ、地球系の温度は大局的には太陽からの日射と地球系からの赤外線エネルギーの出入りのバランスにより決まっているといえる。以上のことを簡単な全球平均エネルギー収支(global energybalance)式により数量的に調べてみる。エネルギー収支は人間活動に起因した、あるいは地熱などの外部から供給されるエネルギーを無視すると次のようになる。π1-=4πεσ욦(1・1)ここで、:地球半径、:太陽定数、:日射に対する全球平均反射率(アルビード)、σ:ステファン・ボルツマン定数、욦:全球平均地表面温度、ε:地表面からの赤外線の宇宙への逸出割合(εは大気の温室効果により1よりも小さい)である。式(1・1)の左辺は太陽エネルギーのうち、地面や雲により反射されて宇宙へ戻る分を差引いた量、右辺は赤外線として宇宙に逸出するエネルギー量である。式(1・1)から全球平均地表面温度は次のようになる。욦=1-4εσ(1・2)式(1・2)に=1367W/m,=0.3(アルビード30%)、ε=1.0(温室効果がない場合)、σ=5.67×10W/(m・K)の値を代入すると욦=255K となり、現在の平均地表面温度288K より33K も低い値となる。反対に温室効果ガスの増大によりεが小さくなると、地上気温は高くなる。なお、現実の大気のεは0.6程度の値である。以上から、全球平均の地上気温約15°Cは、地球が太陽から受けている日射エネルギーによる加熱と地球から宇宙への赤外線放出による冷却のバランス、および大気の赤外線吸収能力すなわち温室効果から決まっていることがわかるであろう。温室効果ガスの赤外線吸収作用について以下にもう少し詳しく説明する。太陽表面温度6000K から放出される日射の波長は0.2~2 mの間にあり、そのエネルギーの大部分は0.4~0.8 mの可視領域に集中している。これに対して、約300K の地球表面から放出される赤外線の波長は4~30 mの範囲にある。地球大気は可視光線に対してはほぼ透明(吸収なし)であるが、赤外線に対しては8~12 mの波長帯(大気の窓といわれ、人工衛星の赤外線による観測に利用)を除いてはほぼ不透明である。人工衛星の赤外線センサを下に向けて、地球から宇宙に放出される赤外線強度の波長分布を測定した事例を図1・13に示す。測定値と300K に対応する黒体放射エネルギー波長分布を比較すると13~17 m,9~10 m,8mより短波長側において大きく減衰していることがわかる。これはこれらの波長帯にCO,O,HO等の吸収帯があるからである。また、フロン11,12は11~12 m,CHとNOは7~8 mに吸を外に逃さない作用が温室に似ていることから、これを大気の温室効果という。
