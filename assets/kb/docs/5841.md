# 2・1・1 異常音表現における擬音語

> **章**: 第2章 快適環境技術
> **節**: 2・1 快適音場の創生

a.はじめに 機械の騒音は何dBの連続音であるとか，何
Hzの音が突出しているが，それはファンの羽根音だなどといわれ
るように，世の中ではだいたいの騒音の周波数特性とその原因がわ
かるようになってきている．そして，それぞれの騒音問題に対して
そのレベルがかなり下げられ，静かな機械が多くなっている．しか
し，そのため逆にわれわれは，ときどき発生する音や変動する音，
そして気持ちよく聞こえてきてほしいと思う表示音，すなわち機器
の作動操作状況を示す報知音が気になるようになった．
異常音と呼ばれる音の場合は，計測点で待ち構えていてもそれら
を計測できないことも多く，一部の人が聞いて，ジージー鳴ったと
か，ガタガタしたとか，他人に言葉で伝えることになる（図2･1）．
特に問題が多いのは，回転機械の場合と二相流が流れる内部流体系
の場合である．なかでも後者の流動音の場合は，チョロチョロ，シ
ャー，シュルシュル，ボコボコなどさまざまなものがあり，それだ
け問題が多い．生活環境騒音として役所に寄せられるクレームを見
ても，隣近所の機械や工事の非定常音のブンブン，ゴンゴン，カタ
カタ，キューキューがうるさいという訴えが多い．ほかには，機械
のがた衝突音，しゅう動音のことが多い．
このような異常音を表す方法として多くの擬音語（onomato-
poeia）が用いられている．それらは情報を正しく伝えることがで
きるならば有効な方法と思われるが，地域や団体，学校や会社，年
代によって用い方が異なることも多い．そこで，標準語としての擬
音語の必要性を受けて，自動車技術会の振動騒音部門委員会は，く
るまの擬音調査報告 として標準語版を発行している．その中で，
自動車の各部分の騒音発生挙動と擬音語とを対応させている．
現象音を表すのに際して，人はできるだけ聞いた音に近い言葉で
伝えたいという思いから，現象音により近い擬音語を用いているも
のと考えられる．
b.擬音語表現で現象の様子を伝達できるか 人が現象音を
聞いて他の人に伝えたいとき，次のことが考えられる．
(１)音をまねてそっくりの音声を作りだす．
(２)人には出せる音，聞ける音の領域があり，その中でできるだ
け工夫して音声を作りだす．
(３)似顔絵のように，特徴だけをうまくとらえて擬音語を当てて
いる．
(４)音以外のジェスチャや絵図などを加えてマルチモードで伝え
るので，だいたいの音表現となる．
(５)感覚だけに訴えて，言葉でそう思わせてしまう．新造語で変
化したり，流行に従う．
