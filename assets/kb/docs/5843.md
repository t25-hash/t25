# 2・1・3 乗用車の音質改善技術

> **章**: 第2章 快適環境技術

近年、乗用車の車室内音は、静粛性の追求に加え感性品質の面から音質(音色)に対する要求も高く、音質向上は乗用車を開発するうえで重要になっている。ここでは加速時の音質を例にとり、影響の大きいエンジンや吸排気系を中心に、音質の向上手法を述べる。a. エンジン剛性の向上エンジンブロックは、燃焼圧力変動によるクランク軸回転方向のモーメント加振力やピストン往復慣性加振力などにより、ねじり、曲げの振動が励起されるため、ブロック剛性向上などによる振動レベル抑制が重要である。図2・23は、ガセット一体型のアルミニウムオイルパンを用いて、エンジンとトランスミッションの結合剛性を向上することで振動レベルを抑制したエンジンの事例である。クランクシャフトの曲げ変形により発生する振動も影響が大きく、シャフト本体の剛性向上に加え、ラダーフレーム(ladderframe、図2・24)を用いてシャフト支持部の剛性を高め、振動を抑制する事例がある。また、クーラコンプレッサなど補機類の支持剛性向上(補機共振抑制)も重要である。取付ブラケットの剛性向上に加え、剛性の高いラダーフレームバルク部分に補機を支持する方法もある。(2) バランスシャフトピストン往復慣性加振力などによる振動について、バランスシャフト(balance-shaft)を付加してカウンタフォースを発生させコントロールする方法がある。図2・25は、4気筒エンジンの上下方向慣性加振力二次成分を抑制するバランスシャフトの事例である。(3)
