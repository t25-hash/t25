# 2・1・6 建設機械の騒音改善技術

> **章**: 第2章 快適環境技術

a. はじめに建設機械(construction machinery)から発する音波は,受音点へ客観的な物理量である音圧の時間変動として到達する.しかし,受け手の状況や周囲環境の違いにより,主観的な聞こえ方は異なる.受け手によっては,環境を悪化させる騒音となることもあれば,機械の運転状態を伝える情報音である場合,さらには心を豊かにする響きとなっている場合さえもある.このようにさまざまな立場で聞かれ,種々の特徴をもつ機械の音を,環境に優しく,さらには魅力ある音へと合理的に改良していくための技術の開発が,すでに自動車をはじめとする機械の分野で進められており,建設機械の分野でも取組まれてきた.b. 音質・音色改善の必要性単に騒音レベルの低減のみを推し進めても,次のような課題に遭遇することがある.(1)必ずしも喧騒感の改善につながらない場合,(2)これまで埋もれていた音が耳につきだして,新たな喧騒感を生じる場合,(3)防音材料の使用拡大や開口部の面積減少による,ヒートバランスや燃費,コストなどの効率が低下する場合などである.これらの問題を解決するために,音質(sound quality)や音色が心理に与える聴感上の影響も考慮した音質・音色設計が必要となる.c. 建設機械の主要な音源と特徴建設機械は,油圧ショベル,クローラクレーン,ラフテレーンクレーン,ホイールローダ,ブルドーザ,アースオーガ,モータスクレーバなど多種にわたるが,ここでは全建設機械の約40%を占め,土工機械の中心的存在である油圧ショベル(hydraulic excavator)を中心に述べる.主要な音源としては,図2・41に示すファン音,エンジン機関音・エンジン排気音・吸気音などのエンジン音,油圧ポンプ音,油圧モータ音,コントロールバルブ音などの油圧機器音,そのほかに旋回歯車音,履帯の走行駆動音,作業時の打撃音,がた音などがある.図2・41 油圧ショベルの主要音源と伝ぱ経路10―22環境システム自動車と異なり定置で作業することが多く,走行風がない状態でエンジン負荷を高く稼働するため,ヒートバランス上冷却ファンの高速回転が必要となり,ファン音は大きな騒音源となる.エンジンと油圧ポンプは,吸音材を内張りした吸遮音性の高い囲いの中に収められ,冷却風のための吸排気の開口が設けられている.それらの開口部からの放射音を低減するために開口面積を低減すると,冷却風量が不足し,ヒートバランスに問題が生じる.開口面積を増大するとヒートバランスは満足されるが,開口からの放射音量が増加する.この二律背反性に対する合理的設計が課題となる.騒音の機側放射は,エンジン囲いからの開口音や透過音,ブームやアームからの固体音,油圧配管音などであり,運転室への騒音伝ぱは,キャビン開口からの進入,固体伝ぱ,透過音(図2・41)である.図2・42に,防音囲いの開口部と遮へい部などから放出される各音源の音響パワー比率の例を示す.この例では,開口部からの寄与率が50%を超えているのがわかる.d. 騒音改善へのアプローチ. 騒音レベル(A 特性音圧レベル)の低減周囲騒音や運転室内の騒音レベルの値が高かった1970年代当時の建設機械では,まずはレベルの低減に力が注がれた.都市部や夜間工事の増加に対し,建設工事の苦情件数は公害件数全体の二十数%に達していた.騒音レベルの低減を目標に,1970年代から低騒音仕様の建設機械の開発が始まり,その後1983年には低騒音型建設機械の指定制度が発足,1989年には建設作業騒音の規制基準値の強化,および低騒音型より6dB 低い超低騒音型建設機械の指定制度が追加され,騒音レベル低減に対する取組みが加速された.さらに1997年には,対象騒音は,それまでの無負荷ハイアイドル時定常騒音から,作業を模擬した模擬掘削動作の繰返しサイクル中の等価騒音レベルへと,より実態に近づけた運転モードでの評価に進展した.ちなみに,油圧ショベルの模擬掘削動作の繰返しサイクル中の等価騒音音響パワーレベルの低騒音仕様基準値は,例えば55kW以下で100dB (A),206kW 以上で106dB (A)である.このような一連の騒音レベル低減の取組みにより,油圧ショベルの外周環境騒音レベルと運転室内騒音は,共に10年間に約10dBの割合で下がっている.また,低騒音型建設機械の普及台数は,1987年に22万台であったものが,1992年には70万台に達し,超低騒音型機も10万台を超えた.1992年の低騒音型建設機械の指定状況は1260型式であったが,1996年には2737型式,1997年の新基準適合建設機械では1998年の568型式から2001年には1806型式,2007年の3797型式へと着実に普及が進んでいる.これらのことから,建設機械の場合は,騒音レベルの低減が騒音改善の有効な方法となっていることが知られる.. 騒音対策技術主要音源に施されるおもな対策は次のとおりである.(1) エンジン・ファン:軸流から斜軸流や遠心型,前進翼型,低騒音油圧駆動ファン・シュラウド:チップクリアランスの低減,ベルマウスなど・機関音:ピストンスラップ対策,ギヤ音対策,ねじり振動対策,剛性強化など・マフラ:シェル固体音対策カバー,リアクティブ型+吸音型併用,気流音対策テールパイプなど・吸気:共鳴型,サイドブランチ型消音器,吸気管路長調節など(2) 油圧系・油圧ポンプ・モータ:バルブプレートのポート,ノッチ形状の最適化,プランジャ数の増加など・配管・コントロールバルブ:ブランチ型ホース,アキュムレータ,防振支持など(3) 歯車騒音:歯形形状変更,かみあい率増加,歯面粗さ改善,歯車箱のリブ補強や防振支持など以上の機器本体の音源対策のみでは騒音低減量が不十分であり,現在では防音囲いを施さざるを得ない状況にある.囲いの内部に吸音が施される場合が多いが,ヒートバランスや保守点検の容易性も具備した囲い構造が課題となる.しかし,これらの対策により騒音レベル値を下げても,b項で述べた理由により必ずしも改善に至らない場合が生じる.そこに,音質・音色設計技術の必要性が存在する.e. 音質・音色の改善設計. 評価音の提示改善に取り組む手順は,(1)問題となる音質・音色の特定,(2)目標音質・音色の策定,(3)改善効果の事前予測,(4)実現するための対策箇所と目標音響特性の決定,(5)対策設計,(6)試作,(7)評価というように進む.各段階における音色評価では,図2・43に示すような,ディジタル技術による評価音の加工・合成・音提示技術が使われる.音の提示に関しては,収録したソースを再生して用いる場合が多い.評価音の音場感を原音場へ近づけるため,ダミーヘッド(擬似頭)や人間の両耳介の近傍に設けた小型マイクロホンにより収音し,ヘッドホンで再生する方法が多くとられている(図2・44).評価音の作成は,周波数領域だけでなく時間領域での分析,加工,合成が必要で,ディジタル技術により対処される.また,被験者が一方的に聞かされるのみにとどまらず,例えばアクセル操作やレバー操作のような人間の操作に対して相互作用的に発する機械音をシミュレートできるシステムも開発されている.このようなシステムを用いることにより,単なる受動的聴取だけでは得られない音色因子が新たに抽出されている.機械の操作において,その操作音を反応としてとらえ,操作性の向上に反映させる事例は日ごろよく経験するところである.. 終わりに建設機械の音質・音色改善技術について,油圧ショベルを例にとって述べた.騒音の改善には,騒音レベルの低減と同時に音質・音色の向上が必要であり,その音質・音色設計について言及した.今後も,機側周辺およびキャビン内部の騒音レベル値の低減や音質改善の取組みが継続され,上質な音質や音色を追求する開発が推し進められるであろう.
