# 2・1・8 家庭用電化製品の音質設計技術

> **章**: 第2章 快適環境技術
> **節**: 2・22 は、1/2次ごとすべての次数成分の音圧レベルが、特に

家庭用電化製品は社会生活に必要不可欠な機械として普及している。低騒音化はその製品性能の一部としてすでに定着しており、ここ十年で騒音レベルは大幅に低減された。しかしながら、生活時間や住宅様式の変化により、最近は消費者が快適性を求めて音を評価する傾向が強まっている。そこで、音質評価法に着目し、音質改善を図った事例について紹介する。a. 離散周波数音の評価手法家庭用電化製品の多くはモータをはじめとした回転機械で構成され、回転数に同期した周波数成分10―26環境システム図2・57 離散周波数音の分析(TNR 分析)図2・58 エアコン室内機と横流ファンが周波数スペクトル上に卓越する。こうした周波数成分は離散周波数音(discrete tone)と呼ばれ、他の周波数成分から分離して聞こえ、騒音の原因になる場合が多い。このような音の評価手法は離散周波数音分析と呼ばれ、TNR(tone-to-noise ratio)手法としてISO 7779:1999(JIS X 7779:1999)で規格化されている。この値が6dB 以上の場合を顕著な離散周波数音(prominent discrete tone)と規定し、卓越したピーク音がはっきりと知覚され、聴感上不快感が増すとされている。通常のA 特性[OA(over all)値]評価では必ずしも聴感上の効果がレベルに反映されるとは限らないことから、物理量的指標のため安定した評価が可能なうえ、しきい値も示されているTNR は音質改善に有効な尺度として普及すると考える。b. エアコン横流ファンの音質改善。 翼ピッチ音のピークのスペクトル分散エアコン室内機で聴感上問題となる音は翼ピッチ音(羽切り音)である。図2・58に示すように羽根が送風出口近傍の舌部(出口から入口への逆流防止部)に近づくにつれ空気が圧縮され、舌部の下に入り込むとそれが緩和される。次の羽根が近づくと圧縮が起こり、繰返されることで翼ピッチ音が発生する。したがって、等ピッチの場合は回転数()×翼枚数の周波数に一次成分が現れ、実際は倍音の二次成分くらいまでが問題となる。そこで、低減方法として、羽根を等ピッチでなく不等ピッチにして圧力変動の周期性をなくす対策を行った。具体的には周期的な周波数変調とランダムな周波数変調をさせる方法である。ここでは羽根の数である。すなわち、周波数変調とみなすことができ、この角周波数をωとすると、羽根は1回転で元に戻るから、ωもωの整数倍となる。変調をかけることでスペクトルが分散し、なかでもM 系列は広い範囲に分散し抑制効果が期待できる。そこで、実際に羽根枚数31枚のファンを試作し評価した。等ピッチ翼では図2・60のように二次まで卓越しているが、M 系列翼では図2・61のようにピークが抑制され、騒音の大きさ自体は低減しないものの聴感上は良好となった。図2・61 M 系列翼の騒音スペクトル(1200min)図2・63 脱水槽穴列のスキュー化の効果予測図2・62 脱水槽穴列。
