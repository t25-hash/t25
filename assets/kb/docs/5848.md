# 2・1・8 家庭用電化製品の音質設計技術

> **章**: 第2章 快適環境技術
> **節**: 2・22 は,1/2次ごとすべての次数成分の音圧レベルが,特に

家庭用電化製品は社会生活に必要不可欠な機械として普及している.低騒音化はその製品性能の一部としてすでに定着しており,ここ十年で騒音レベルは大幅に低減された.しかしながら,生活時間や住宅様式の変化により,最近は消費者が快適性を求めて音を評価する傾向が強まっている.そこで,音質評価法に着目し,音質改善を図った事例について紹介する.a. 離散周波数音の評価手法家庭用電化製品の多くはモータをはじめとした回転機械で構成され,回転数に同期した周波数成分10―26環境システム図2・57 離散周波数音の分析(TNR 分析)図2・58 エアコン室内機と横流ファンが周波数スペクトル上に卓越する.こうした周波数成分は離散周波数音(discrete tone)と呼ばれ,他の周波数成分から分離して聞こえ,騒音の原因になる場合が多い.このような音の評価手法は離散周波数音分析と呼ばれ,TNR(tone-to-noise ratio)手法としてISO 7779:1999(JIS X 7779:1999)で規格化されている.本分析は臨界帯域の考え方に基づくもので,離散周波数音(図2・57中の)を中心周波数として生成される臨界帯域욂内の離散周波数音を除いたエネルギー욧と,離散周波数音のエネルギー욈の比をとったものである.この値が6dB 以上の場合を顕著な離散周波数音(prominent discrete tone)と規定し,卓越したピーク音がはっきりと知覚され,聴感上不快感が増すとされている.通常のA 特性[OA(over all)値]評価では必ずしも聴感上の効果がレベルに反映されるとは限らないことから,物理量的指標のため安定した評価が可能なうえ,しきい値も示されているTNR は音質改善に有効な尺度として普及すると考える.b. エアコン横流ファンの音質改善.
