# 2・1・9 精密機器の機能音の音質設計技術

> **章**: 第2章 快適環境技術
> **節**: 2・65 ).音響パワーレベル低減量は約1dB と少ないが,離散周波

精密機器,自動車,家庭用電化機器など身近な機器は,主機能の高性能化とともに,動作の際に適切な音を発生することで動作状況が確認でき,その音質をよくすることで機器の質感が高められ,使用者の快適性が向上する.個々の機器は新材料を用いて小型,軽量,高速化されるため,さらに音圧を下げる低騒音対策は難しい状況となり,また静粛な音環境の実現だけでは必ずしも望まれる音質とはならない.設計の早い段階で動作音を予測する音響シミュレーションを活用して,感性を考慮した快適な音環境を実現する音質設計(sound quality design)技術が必要である.a. カメラシャッタ音の特徴抽出と目標音質の検討精密機器である一眼レフカメラシャッタ音を例に,過渡的に変化する機能音(function sound)の音質設計技術について述べる.カメラは,撮影する際のシャッタ音を快適に感じている人も多い.近年普及したディジタルカメラでは,シャッタが存在しないタイプもある.静粛性が求められる環境ではよいが,撮影時の確認やカメラらしさを増すために,適宜シャッタ音を付加している.一眼レフカメラでは,光路を切り換えるミラーやシャッタが存在する.図2・67は一眼レフカメラの標準的なシャッタ速度1/60sのシャッタ音であり,発音メカニズムを考慮して三つに分け,A 部のファインダとフィルムへの光路を切り換えるミラーの上昇,B 部のシャッタの開閉,C部の撮影後にミラーが元の位置に下降する機能音からなる.ここで,B 部はシャッタ速度に依存して時間の長短が生じる.また,C部はミラーが下降した際に支持棒から受けた衝撃力により,音圧が徐々に減少しながら数回バウンドする短時間の繰返し音(バウンド音,bound sound)である.各部品から発生する音をそれぞれ独立に変更することで,シャッタ音全体が改善されるとは限らない.各部品の機能を損なわず,また動作機構に対する制約条件を考慮して,音質を主眼にした設計変更を行うことは難しい.そこで,音質に対する寄与が大きい部分を(a) A →B →C(b) B →C→A(c) C→A →B図2・68 動作順序の入替えによる音質評価把握し,その部分の設計変更によって機能音を改善し,シャッタ音全体の音質の印象を変える.過渡音の場合,定常音とは別の音質評価手法(sound qualityevaluation)が必要である.ここでは,20代前半の男性14名の被験者にカメラシャッタ音として提示し,変更前のシャッタ音をオリジナル音(original sound)とし,過渡的に変化するA 部,B 部,およびC部の各音圧のみを順次1/5倍に変更した音と交互に聞いてもらい,主観的な音質評価を実施する.被験者による差異は少なく,変化ありを4ポイント,変化なしを0ポイントとした5段階評価の平均評価値を図2・68(a)に示す.オリジナル音と比較してC部の音圧の変更が最も顕著な差異と被験者に判断されており,C部のバウンド音がシャッタ音全体の音質に大きく寄与していることがわかる.C部は動作音の最後に発生しており,そのことが原因で音質に対する印象が大きい可能性があるので,動作順序を入れ替えた評価音を作成し,同様な音質評価を行う.図2・68(b)に示すB 部→C部→A 部の順や図2・68(c)に示すC部→A 部→B 部の順の場合も,C部のバウンド音がシャッタ音の音質に最も寄与が大きいと判断される.したがって,C部のバウンド音を変更することで,シャッタ音全体の音質の印象を効果的に変えられるのに対し,A 部やB 部を変更してもオリジナル音との変化は小さいと感じられるといえる.次に,心地よいシャッタ音とするために好まれるバウンド音を決定する.一連のシャッタ音でC部の衝突と衝突の間のほぼ無音な部分を削除してバウンド時間を1/2倍および1/4倍に縮めたバウンド音,C部のみ音圧を1/2倍および1/4倍に減少したバウンド音,さらにC部のみ減衰を増したバウンド音を新たに作成し,オリジナル音と比較した音質評価を行う.これらの結果より,バウンド時間が短く,かつ減衰が早いバウンド音とすることで,シャッタ音全体の音質の印象がよくなることが確認でき,これを目標音質(tar-get sound quality)とする.b. 衝撃位置による音質の変化バウンド音を改善するためには,ミラー,支持棒の形状や材質を変更する構造変更を行うか,またはミラーの回転速度や衝撃位置等の機構要素を変更することが考えられる.ミラー動作部の機構は共通性があるので,図2・69に示すアルミニウム製の薄板実験モデルを用いて,比較的容易に設計変更が可能な衝撃位置について検討する.縦横100mm,厚さ3mmの薄板と両端を固定した棒をちょうつがいで結合して,薄板が棒の軸回りを自由に回転できるようにする.薄板を水平な位置から下方向に15°自由に回転させ,支持棒に相当する位置に固定した力センサに衝突させる.衝撃位置の違いによるバウンド音の変化を実験から調べるために,力センサで衝撃力を,マイクロホンで音圧を測定する.衝撃位置1と3の衝撃力の時間軸応答は図2・70(a)(b)であり,衝撃位置の違いにより,第1衝撃力の大きさ,バウンド時間や減衰が異なることがわかる.薄板と力センサとが衝突したときに発生するバウンド音は,シャッタ音と同様に短時間で数回のバウンド後に停止し,衝撃位置の違いも現れている.薄板の実験モード解析から,各衝撃位置で個々の振動モードの励起状態が異なり,バウンド音はいくつかの振動モードの重ね合わせに基づいて発生することが明らかになった.衝撃位置の変更でバウンド音の周波数特性は変化するが,その影響よりバウンド時間や減衰など時間特性のほうが音質に対する寄与が大きい.c. からFを追加し,それぞれのバウンド音を予測する.まず,薄板の有限要素モデルを用いて,各衝撃位置で支持棒に発生する衝撃力を機構解析(mechanism analysis)から求める.バウンド現象では,二つの物体の衝突によりその間に絶対値が等しく反対方向に働く衝撃力が生じるので,二つの物体の接触をばねで表現し,ばね定数,ばね定数の非線形性を示すばね定数に対する指数,および粘性減衰係数を用いる.衝撃位置1における第1衝撃力を基準として,各パラメータを調整し,実験と解析でほぼ同様の傾向を示すモデリングを行い,各衝撃位置での衝撃力を予測する.次に,その衝撃力で発生する薄板の振動速度を振動解析(vibra-tion analysis)より算出する.さらに,ここで求めた薄板の振動速度と,法線方向の音場粒子速度を等しいとして,音響境界要素法(acoustic boundary element method)より各衝撃位置でのバウンド音を予測する.これらの音響シミュレーションから,心地よいバウンド音を発生させる衝撃位置が推定できる.d.
