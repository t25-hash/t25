# 2・1・9 精密機器の機能音の音質設計技術

精密機器，自動車，家庭用電化機器など身近な機器は，主機能の
高性能化とともに，動作の際に適切な音を発生することで動作状況
が確認でき，その音質をよくすることで機器の質感が高められ，使
用者の快適性が向上する．個々の機器は新材料を用いて小型，軽
量，高速化されるため，さらに音圧を下げる低騒音対策は難しい状
況となり，また静粛な音環境の実現だけでは必ずしも望まれる音質
とはならない．設計の早い段階で動作音を予測する音響シミ
ュレーションを活用して，感性を考慮した快適な音環境を実現する
音質設計（sound quality design）技術が必要である．
a.カメラシャッタ音の特徴抽出と目標音質の検討 精密機器
である一眼レフカメラシャッタ音を例に，過渡的に変化する機能音
（function sound）の音質設計技術について述べる．カメラは，
撮影する際のシャッタ音を快適に感じている人も多い．近年普及し
たディジタルカメラでは，シャッタが存在しないタイプもある．静
粛性が求められる環境ではよいが，撮影時の確認やカメラらしさを
増すために，適宜シャッタ音を付加している．
一眼レフカメラでは，光路を切り換えるミラーやシャッタが存在
する．図2･67は一眼レフカメラの標準的なシャッタ速度1/60sの
シャッタ音であり，発音メカニズムを考慮して三つに分け，A部
のファインダとフィルムへの光路を切り換えるミラーの上昇，B部
のシャッタの開閉，C部の撮影後にミラーが元の位置に下降する機
能音からなる．ここで，B部はシャッタ速度に依存して時間の長短
が生じる．また，C部はミラーが下降した際に支持棒から受けた衝
撃力により，音圧が徐々に減少しながら数回バウンドする短時間の
繰返し音（バウンド音，bound sound）である．
各部品から発生する音をそれぞれ独立に変更することで，シャッ
タ音全体が改善されるとは限らない．各部品の機能を損なわず，ま
た動作機構に対する制約条件を考慮して，音質を主眼にした設計変
更を行うことは難しい．そこで，音質に対する寄与が大きい部分を
把握し，その部分の設計変更によって機能音を改善し，シャッタ音
全体の音質の印象を変える．
過渡音の場合，定常音とは別の音質評価手法（sound quality
evaluation）が必要である．ここでは，20代前半の男性14
名の被験者にカメラシャッタ音として提示し，変更前のシャッタ音
をオリジナル音（original sound）とし，過渡的に変化するA部，
B部，およびC部の各音圧のみを順次1/5倍に変更した音と交互
に聞いてもらい，主観的な音質評価を実施する．被験者による差異
は少なく，変化ありを4ポイント，変化なしを0ポイントとした5

ることがわかる．
C部は動作音の最後に発生しており，そのことが原因で音質に対
