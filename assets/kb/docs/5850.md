樹脂複合材(polymer matrix composite)は軽くて強い材料として自動車、航空機、電化製品などさまざまな工業製品に適用されている。特に自動車では軽量化のためにエンジンルーム内の各種カバーにも適用されているが、遮音性能(transmission loss)の観点からは軽い樹脂材料で良好な遮音性能を有するカバーを設計することは難しい。ここでは遮音効果の高い樹脂部品を設計するための遮音効果解析手法として、均質化法を用いた定式化と数値例を示す。均質化法(homogenization method)は周期的なマイクロストラクチャからなる複合材料の平均化された材料定数を求める手法であり、マイクロストラクチャを構成する材料定数や幾何学的形状から複合材料の均質化された材料定数を統一的な手法で求めることができる。樹脂はヒステリシス型減衰を有するが、多くの場合、ヒステリシス型減衰が生じるのは弾性体内部の損失によるもので、構造減衰とも呼ばれ、振動の減衰は周波数に依存しないことが知られている。次に図2・74に示す境界および内部領域で表される構造物を考える。この構造物は、母材となる樹脂と補強材からなる周期的マイクロストラクチャにより構成されているものとする。樹脂は粘弾性体、補強材は弾性体であると仮定する。式(2・5)の複素弾性率を用いた釣合い方程式は次式で表される。なお、以下の式において物体力は無視する。式(2・6)に仮想変位を乗じ領域全体を積分した結果にグリーン(Green)の定理を適用すると、次式が得られる。構造物内部の任意点の変位をユニットセルの周期性より次のように漸近展開する。またεは全体構造に対するユニットセルのスケールε=(2・10)であり、十分小さいものとする。この式は複合材料の均質化された材料定数を表す。実際の解析ではマイクロストラクチャのユニットセルを有限要素に分割し、式(2・15)を周期境界条件のもとに解き、χを求める。得られた結果を式(2・17)に代入し、均質化された材料定数を求める。続いて巨視的な領域を有限要素に分割し式(2・16)を解く。このとき、均質化された材料定数を応力ひずみ関係式として用いる。以下では、この定式化によって樹脂複合材内部に混入された補強材の形状や粘性と遮音性能の関係について検討した三次元定常音場・構造連成解析例を示す。音場・構造連成(vibro-acousticbehavior
