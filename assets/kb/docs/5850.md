# 2・1・10 樹脂部品の遮音効果解析

樹脂複合材(polymer matrix composite)は軽くて強い材料として自動車、航空機、電化製品などさまざまな工業製品に適用されている。特に自動車では軽量化のためにエンジンルーム内の各種カバーにも適用されているが、遮音性能(transmission loss)の観点からは軽い樹脂材料で良好な遮音性能を有するカバーを設計することは難しい。ここでは遮音効果の高い樹脂部品を設計するための遮音効果解析手法として、均質化法を用いた定式化と数値例を示す。均質化法(homogenization method)は周期的なマイクロストラクチャからなる複合材料の平均化された材料定数を求める手法であり、マイクロストラクチャを構成する材料定数や幾何学的形状から複合材料の均質化された材料定数を統一的な手法で求めることができる。樹脂はヒステリシス型減衰を有するが、多くの場合、ヒステリシス型減衰が生じるのは弾性体内部の損失によるもので、構造減衰とも呼ばれ、振動の減衰は周波数に依存しないことが知られている。ヒステリシス減衰では調和駆動力が加えられた場合、ひずみが応力に対して一定の位相だけ遅れることが観測されるため、この関係を複素弾性率(位相遅れ角φ)を用いてσ욡욢=욡욢욅욆옿욅옿욡(2・5)と仮定する。ここで、σ욡욢は応力、욅は任意点の方向変位を表す。次に図2・74に示す境界および内部領域で表される構造物を考える。この構造物は、母材となる樹脂と補強材からなる周期的マイクロストラクチャにより構成されているものとする。樹脂は粘弾性体、補強材は弾性体であると仮定する。式(2・5)の複素弾性率を用いた釣合い方程式は次式で表される。なお、以下の式において物体力は無視する。図2・73
