# 2・2・2 社会的背景

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

空気清浄に対する社会的関心・要請は、いわゆる公害問題から始まったといえる。工業由来のばい煙、硫黄酸化物、窒素酸化物等の大気汚染から始まり、自動車等の移動発生源からの窒素酸化物、硫黄酸化物、粒子状物質の大気汚染へと社会的関心が移った。行政としてはこのような社会的要請にこたえるべく、「大気汚染防止法(Air Pollution Control Law)」を制定して、工業由来の固定発生源、自動車等の移動発生源に対する法的整備を行い、大気汚染の低減に努めている。大気汚染に対する社会的関心が高まるにつれて、室内環境に対する社会的関心も高まってきたと思われる。最初、都市部における住宅の密集に伴い、いわゆる悪臭に対する苦情が増加し、このような社会的要請に応じるために、「悪臭防止法」が制定された。次に、冷暖房における省エネルギー化を目指した住宅の高気密化による室内空気質の低下について、関心が払われるようになった。一例として、局所暖房機器による室内空気汚染が挙げられる。さらに、微量10―32環境システムの化学物質による人体への悪影響が懸念されるようになり、化学物質過敏症候群(いわゆるシックハウス症候群)が広く知られるようになった。これらの対策として、厚生労働省、国土交通省、経済産業省が協力して、住宅等の居住空間でのシックハウス原因物質の濃度基準、建材等の規格制定、住宅建築時の24時間換気方法のガイドライン制定等の法整備等を行っている。最近では、各種アレルゲン、雑菌、ウイルス等へも関心が広がっているようである。これら上述の気運をビジネスチャンスとしてとらえた各業界が、建材、換気システム、空気清浄機等に関する研究開発を積極的に行い、次々と市場に新製品を導入している。一例として、家庭用空気清浄機の市場がある。家庭用空気清浄機は、集じん(タバコ煙、ハウスダスト、花粉等)機能から始まり、脱臭(ペット、調理時のにおい等)機能が加わり、最近では、細菌・ウイルスの抑制、除菌機能を宣伝する商品も市販されている。日本電機工業会民生用電気機器自主統計調査によれば、家庭用空気清浄機の市場規模は、2000年度は201億2000万円(96万9000台)、2006年度は307億3200万円(153万4000台)である。今後も堅調な需要が続くと推定される。
