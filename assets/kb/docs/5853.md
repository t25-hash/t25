# 2・2・3 技術的背景

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

室内環境保全技術は,産業における排ガス処理に用いられるものと同様の技術が用いられる.排ガス処理技術における従来法には,燃焼法,洗浄法,吸着法,生物脱臭法,消・脱臭法等がある.これら従来法においても,日々の改良から,触媒法,生物法における技術革新,そして従来の技術とは質的に異なる新技術による新規な空気清浄法の提案がある.室内環境保全技術においては,産業における排ガス処理と比較して処理対象空気量は少ないが,除去対象ガスの濃度が極めて低いと考えられる.一般に,除去対象ガスの濃度が極めて低いと除去効率が低下するため,除去対象ガスの濃度を高める操作と除去操作を組合わせて用いることによって全体の効率を向上させる.すなわち,濃縮機能と除去機能を組合わせた浄化技術が適しており,この考えに基づいた各種技術の組合せが提案されている.また,運転時に特別な化学物質等(例えばアンモニア,酸性もしくはアルカリ性の中和剤等)を供給する必要があるもの,運転により二次公害(例えば窒素酸化物,硫黄酸化物,オゾンの副生,廃水等の処理等)が発生するおそれがあるものは,本室内環境保全技術としての適用が困難と考えられる.通常は,電気エネルギーを利用できる環境にあると考えられるため,電気エネルギーから変換できる各種エネルギー(例えば熱,光,放電による高速電子など)を用いるか,電気と空気,水等から生成できる活性種を用いる各種空気浄化法に実質上,限定される.これらの技術的背景を踏まえて,新規技術もしくは改良技術として触媒燃焼法,オゾン脱臭法,光触媒法があり,さらに最近では機能性フィルタ,ラジカル反応・活性種・プラズマ分解法,マイナス空気イオンを活用した空気浄化技術が研究開発され,一部,実用化されている.
