# 2・2・4 触媒燃焼法

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

酸化触媒を用いて除去対象ガス成分を酸化することにより、二酸化炭素、水分、窒素等に分解し、空気浄化を実現する方法である。除去対象ガスを燃焼反応により分解除去する点では、工業からの有害物質を酸化分解するのに用いられている直接燃焼法(3・2節参照)と同じである。しかしながら直接燃焼法と比較して、触媒燃焼法(catalytic combustion method)では触媒の存在により、対象とするガス成分の濃度が低濃度でも酸化できる、酸化反応に必要な反応温度を低下させることができるため、特に室内空調設備との整合性がよく、室内環境保全技術として適する。酸化触媒としては、パラジウム、白金等の貴金属触媒(通常は粒子径が数nm程度の貴金属粒子をセラミックス製もしくは金属製ハニカム状構造体に担持した形態で用いる)、もしくは酸化マンガン、酸化コバルト等の金属酸化物触媒を用いる。また、春田は酸化触媒としての金の可能性を見いだしている。金触媒は室温程度における一酸化炭素の酸化除去に優れている。触媒燃焼を効率よく行うためには、除去対象となるガスの成分、濃度、処理空気量に合わせて、最適な触媒成分、触媒構造体の選定を行うことが重要である。また、各種家庭用電化製品に触媒燃焼機能を組込むことにより、高機能化、快適性向上を果たした生活関連機器が市販されている。例として、脱臭機能付き生ごみ処理器、脱臭機能付き洗浄トイレ便座、自己浄化機能付き調理器、脱臭機能付きこたつ等がある。
