# 2・2・5 光触媒法

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

酸化チタンは紫外線照射下で有機性化学物質等を酸化分解する能力(光触媒能)をもつことが知られている(6・2・1項参照)。半導体である酸化チタンが紫外線を吸収することにより光励起電子と正孔が生成し、これらと空気中に存在する酸素と水蒸気から、酸化チタン表面上にO、ヒドロキシルラジカル(hydroxyl radical、・OH)といった活性酸素種が生成する。これらO、ヒドロキシルラジカルは強い酸化力をもつために、有機性化学物質を室温程度でも酸化分解することができる。その酸化分解速度は必ずしも速くはないが、比較的低濃度の化学物質が存在する室内環境浄化に適する技術である。酸化チタンに代表される光触媒(photocatalyst)を室内環境浄化技術として適用するためには、光源の確保、光触媒の高比表面積化、光触媒能の再生操作が必要となる。a. 光源の確保酸化チタン光触媒は紫外線しか利用できない。そのため、電気エネルギーを紫外線に変換する効率を高めることが省エネルギー化をもたらす。一例として、紫外線発光LED(light emitting diode)と光触媒の組合せが提案され、自動車内脱臭機が市販されているほか、冷蔵庫内の脱臭機能として応用されている。また、表面積の大きいフィルタに塗布された光触媒全体に光照射されるように、漏光型ガラスファイバをフィルタ化した材料が提案されている。b. 高比表面積酸化チタン光触媒反応は酸化チタン表面上で進行するため、酸化チタンの比表面積が大きいほど、除去対象物質の処理速度が速くなる。このため、酸化チタンをフィルタ化して、比表面積を大きくする工夫をしている。c. 再生操作酸化チタン光触媒を使用していると、除去対象物質に含まれる窒素分、硫黄分がそれぞれ硝酸イオン、硫酸イオンとして蓄積し、また除去対象物質が完全に分解せずに一部分解した物質が酸化チタン表面に蓄積し、この結果、酸化チタンの光触媒能が低下する。このため、酸化チタンを塗布した光触媒フィルタの再生操作もしくは交換操作が必要となる。自己再生機能付き光触媒フィルタの実現により保守等の操作が不要になるため、その開発が望まれている。現在、光触媒フィルタの再生方法として、水洗いによる除去、天日干しによる再生が提案されている。
