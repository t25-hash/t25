# 2・2・6 オゾン酸化法

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

オゾンは酸素よりも強力な酸化力を有するため、オゾンによる有機性化学物質の酸化除去方法が用いられている。コロナ放電等により発生したオゾンを除去対象ガスと反応させることによって分解除去する。酸化触媒や吸着剤の存在下でオゾンと除去対象ガスを反応させると分解効率が向上し、完全分解(二酸化炭素、水蒸気等に変換する)への生成物選択性が向上する。オゾン酸化法によりすべてのガス成分が分解除去できるわけではない。一例として、酢酸等の低級脂肪酸等に対する分解能力は低いといわれている。また、未反応オゾンの漏洩による二次汚染を抑制するためにも、触媒が用いられる場合がある。オゾン分解触媒としては酸化マンガンが知られており、酸化マンガン触媒フィルタを後段に設置することにより、オゾン濃度を0.1ppm程度まで抑制している。
