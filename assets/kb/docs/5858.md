# 2・2・8 最近,注目されている空気浄化技術,今後の進展が期

> **章**: 第2章 快適環境技術
> **節**: 2・2 室内環境保全技術

待できる技術a. マイナスイオン最近,マイナスイオンの機能をうたった空気清浄機等が市販されている.現状では,マイナスイオンが生体に及ぼす影響については解明されていないようだが,鎮静作用,爽快感,疲労回復等の効果が挙げられている.今後の研究の進展により,その効能が科学的に検証されることが期待される.マイナスイオンの発生法には,コロナ放電法と水噴霧法がある.針状電極に負の高電圧を印加すると,針先の近傍においてコロナ放電が起こってマイナスイオンが発生し,各イオンは気流によって空気中に運ばれる.このコロナ放電法は,静電気除去用に用いられるイオナイザと同様の方法である.水噴霧法(レナード効果)は以下の方法である.気液界面では,水分子の極性により電気二重層が形成される.噴霧ノズルを使って水滴を分裂させると,大きな水滴は正に帯電し,一方,小さな水滴は負に帯電する.小さな水滴は気流によって空気中に運ばれるので,マイナスイオンを含む空気が発生する.これらマイナス空気イオン発生器は,空気清浄機やヘアドライヤの中に組込まれ,市販されている.b. プラズマクラスタイオンプラズマクラスタイオンは浮遊細菌除去,有害化学物質除去,脱臭等に効果があるとされ,これらプラズマクラスタイオン発生器を組込んだ各種家庭用電化製品が市販されている.プラズマクラスタイオンに対する今後の研究により,その効能が科学的に検証されることが期待される.
