# 3・1・1 大気汚染対策の経緯

> **章**: 第3章 大気環境保全技術
> **節**: 3・1 概説

わが国の大気汚染問題は，戦前には足尾銅山（栃木県）における
鉱毒，別子銅山（愛媛県）における煙害のような鉱害問題，セメン
ト製造に伴う粉じん問題等があり，公害問題の原点となっている．
しかし，当時は住民の権利意識が低かったことなどから産業公害問
題は顕在化しなかった．戦後の経済復興を優先した昭和30～40年
代（1955～1974年）のいわゆる高度成長時代に，エネルギー消費
量が急激に増大したこと，また汚染物質を除去するための設備投資
が優先されなかったことから，結果としてばいじん，硫黄酸化物等
による大気汚染が増大した．こうした社会経済状況の中で，いわゆ
る四大公害病（水俣病，新潟水俣病，イタイイタイ病，四日市ぜん
そく）の発生を招くなど大きな社会問題となった．また，モータリ
ゼーションの進展は排出ガスによる大気汚染や騒音問題等を引き起
こした．昭和50年代（1975年～）は二度にわたる石油危機を背景
に高度成長から安定成長に移行し，産業においては省エネルギー化
が進み，産業公害は収束したかに見える．その一方で近年は経済活
動のグローバル化が進み，大量生産，大量消費，大量廃棄型の社会
経済システムへの移行に伴い，地球規模での環境問題が顕在化して
きている．
わが国では昭和37（1962）年に「ばい煙の排出等に関する法律」
を制定し，排出規制を開始した．その後，昭和43（1968）年には
「大気汚染防止法（Air Pollution Control Law)」の制定，昭和
45（1970）年の公害国会（第64回国会）における大気汚染防止法
の改正，さらに二酸化硫黄，二酸化窒素等に関わる環境基準の設
定，数次にわたる排出規制の強化，対策の整備が図られてきている．
