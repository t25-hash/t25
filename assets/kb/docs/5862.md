# 3・1・1 大気汚染対策の経緯

> **章**: 第3章 大気環境保全技術
> **節**: 3・1 概説

わが国の大気汚染問題は,戦前には足尾銅山(栃木県)における鉱毒,別子銅山(愛媛県)における煙害のような鉱害問題,セメント製造に伴う粉じん問題等があり,公害問題の原点となっている.しかし,当時は住民の権利意識が低かったことなどから産業公害問題は顕在化しなかった.戦後の経済復興を優先した昭和30~40年代(1955~1974年)のいわゆる高度成長時代に,エネルギー消費量が急激に増大したこと,また汚染物質を除去するための設備投資が優先されなかったことから,結果としてばいじん,硫黄酸化物等による大気汚染が増大した.こうした社会経済状況の中で,いわゆる四大公害病(水俣病,新潟水俣病,イタイイタイ病,四日市ぜんそく)の発生を招くなど大きな社会問題となった.また,モータリゼーションの進展は排出ガスによる大気汚染や騒音問題等を引き起こした.昭和50年代(1975年~)は二度にわたる石油危機を背景に高度成長から安定成長に移行し,産業においては省エネルギー化が進み,産業公害は収束したかに見える.その一方で近年は経済活動のグローバル化が進み,大量生産,大量消費,大量廃棄型の社会経済システムへの移行に伴い,地球規模での環境問題が顕在化してきている.わが国では昭和37(1962)年に「ばい煙の排出等に関する法律」を制定し,排出規制を開始した.その後,昭和43(1968)年には「大気汚染防止法(Air Pollution Control Law)」の制定,昭和45(1970)年の公害国会(第64回国会)における大気汚染防止法の改正,さらに二酸化硫黄,二酸化窒素等に関わる環境基準の設定,数次にわたる排出規制の強化,対策の整備が図られてきている.表3・1 大気汚染に関わる環境基準1時間値の1日平均値が0.04ppm以下であり,かつ1時間値が0.1ppm以下であること.1時間値の1日平均値が10ppm以下であり,かつ1時間値の8時間平均値が20ppm以下であること.環境上の条件溶液導電率法または紫外線蛍光法非分散型赤外分析計を用いる方法ろ過捕集により重量濃度測定方法またはこの方法によって測定された重量濃度と直線的な関係を有する量が得られる光散乱法,圧電てんびん法もしくはベータ線吸収法測定方法10―35公害関係法令の基本を定めているのは,平成5(1993)年11月に制定された「環境基本法」である.環境基本法は,公害の概念,事業者・国・地方公共団体等の責務,環境基準の設定,排出基準の制定,公害防止計画の策定等,国や地方公共団体のおもな公害防止対策のほとんどをその中に含んでいる.
