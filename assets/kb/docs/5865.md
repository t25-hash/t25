# 3・1・4 大気汚染の発生

> **章**: 第3章 大気環境保全技術
> **節**: 3・1 概説

a. 光化学スモッグ(photochemical smog)光化学スモッグの主要原因物質は、化石燃料の燃焼によって排出されたNOや炭化水素を含んでいる。HC,NO、太陽光による光化学反応では、微粒子とオゾンなどが生成されることがある。b. 酸性雨(acid rain)酸性雨の原因物質はおもに化石燃料の燃焼によって大気中に放出された硫黄酸化物(SO)や窒素酸化物(NO)である。その生成メカニズムには、気相中でのこれらの物質とヒドロキシルラジカル(・OH)との反応、雲や霧の中での反応、粒子状物質上での反応等がある。生成した硫酸、硝酸は雲や雨に吸収されて酸性雨として地上に降下する。c. 成層圏オゾン層の破壊オゾン層は、CFC(chloro-fluoro-carbon)、HCFC(hydro chloro-fluorocarbon)、ハロン、臭化メチル等のオゾン層破壊物質により破壊されている。オゾン層が破壊されると、地上に到達する紫外線が増加し、人体に健康被害を発生させるおそれがある。これらの物質は化学的に安定であるので対流圏ではほとんど分解されず、成層圏で強い紫外線により分解されて塩素原子や臭素原子を放出し、連鎖反応を起こす。d. 地球温暖化(global warming)地球温暖化については1章で記述したので、ここでは省略する。
