# 3・2・2 ばい煙の拡散

> **章**: 第3章 大気環境保全技術
> **節**: 3・2 大気環境保全技術

大気中に排出された排ガスは、それ自身のもつ運動量と浮力により上昇しつつ拡散する。大気中での拡散(大気拡散、atmosphericdiffusion)は大気境界層中の乱流により生じる。したがって、大気乱流の状態によって排煙の拡散は大きく変化する。a. 大気境界層と乱流大気境界層は通常、十分に発達した乱流である。乱流は地表面の粗度による機械的な摩擦と、地表面と大気との間の熱交換による浮力効果の二つの要因により発生する。大気境界層は上下方向の熱輸送の状態により、不安定、中立、安定の3種類に分類される(図3・11)。また、鉛直構造から見ると、地表面から数十m上空までの接地層と、その上の上部境界層に区分される。. 不安定境界層(自由対流境界層、free convective bound-ary layer)晴天日の日中に地表面が太陽からの放射熱で加熱され、対流が発達することにより形成される境界層である。大気は地表面近くの気塊のほうが上空のそれより軽くなるため熱的に不安定な状態にあり、対流運動による鉛直混合が盛んに行われる。このため、混合層とも呼ばれる。混合層は日の出とともに発達し、15時ごろ最盛期となり、その高度は500~2000mに達する。特に、春から夏にかけてよく発達する。混合層中では風の乱れが大きく、拡散は急激に起きる。不安定境界層中での風向、風速、乱流強度の鉛直方向の変化は小さい。10―45図3・11 安定境界層(stable boundary layer)晴天日の夜間には地表面から赤外放射により熱が上空に散逸する、いわゆる放射冷却により地表面付近の気温は上空より低温となり、安定状態が出現する。熱の輸送で見ると上空から地表面に熱が輸送され、自由対流境界層とは輸送方向が逆となる。気温が上空ほど高くなる強い安定層は、逆転層(inversion layer)と呼ばれる。安定層中では下層ほど気塊の密度が高くなるため、上下方向の運動が抑えられて乱流は小さくなり、強い安定状態ではほとんど零となることもある。したがって、安定層中の乱流は小さく、拡散は不安定や中立状態に比較して緩慢である。安定境界層の厚さは100~200m程度であり、地表面近くの風速は上空に向かって急激に増加する。地表面近くの風速分布は対数分布に修正を加えた下記の式(log+linear式)で近似される。=ln 大気安定度(atmospheric stability)と拡散前述のとおり、大気中の乱流の性質は地表面の粗度と大気の熱的な状態、つまり大気安定度により大きく変化する。大気安定度の指標の一つに気温の鉛直こう配(気温減率)がある。乾燥大気を考え、大気中の気塊が上下に移動するときに周囲の空気と熱の交換のない断熱変化を仮定する。このとき、上下移動する気塊と周囲の空気の温度がどこででも等しければ、気塊に対して浮力は働かない。このような大気状態での気温の減率を乾燥断熱減率(dry adiabatic lapse rate)γ욃で表す。γ욃=0.0098(K/m)となる。気温減率がγ욃に近い大気を断熱大気、あるいは熱的に中立状態の大気と呼ぶ。上空への気温の減り方がγ욃よりも急激な大気では、上空へ移動した気塊は周囲の空気より温度が高く上向きの浮力が働き、この気塊はさらに上昇を続ける。このような気温分布の大気を不安定状態の大気と呼ぶ。不安定大気では、乱れは大きい。反対に、気温減率がγ욃より小さい大気では、気塊の運動方向と逆向きの浮力がつねに働くため、気塊の運動は抑制される。このような大気を安定大気と呼ぶ。安定大気では、乱れは小さい。このような大気の安定度(気温こう配)と煙の拡散の関係を模式的に示したものが図3・12である。図3・12中の実線が煙の拡散状態と対応した気温分布を、破線が乾燥断熱減率γ욃の気温分布を表す。c. 光化学オキシダント(photochemical oxidant)光化学オキシダントとは炭化水素類や窒素酸化物などの汚染物質が紫外線の照射を受けることにより生成される二次汚染物質であり、90%以上を占めるオゾンとPAN(peroxyacetyl nitrate、パーオキシアセチルナイトレート)と呼ばれる刺激性の強い成分などからなる。光化学オキシダントの生成には数時間以上の紫外線照射が必要であるため、初夏から盛夏にかけて、正午過ぎから多く発生する。この反応には数十以上の化学反応が関与しており、簡単な予測モデルは実用化されていない。. 浮遊粒子状物質(suspended particulate matter)浮遊粒子状物質とは大気中に浮遊する直径10 mの粒子の総称であるが、その成分は、工場やディーゼル車などから発生するばいじん、ディーゼル排気、金属粒子など産業活動によるもの、砂じんや黄砂、海塩粒子などの自然起源の粒子、さらに硫酸塩や硝酸塩に代表される二次粒子などからなる。このうちの二次粒子に関しては、光化学反応との関係も指摘されており、メカニズムを的確に表現したモデルはない。通常は、大気中での拡散時間に二次物質の生成割合を比例させる便宜的な方法が採用される。
