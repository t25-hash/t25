# 3・2・3 除じん・集じん

> **章**: 第3章 大気環境保全技術
> **節**: 3・2 大気環境保全技術

a. 粒子分離技術気流中に含まれる固体粒子の分離操作を集じん(dust collection)といい,古くから有価物質の回収,有害物質の除去などの目的で,機械的単位操作の一つとして用いられてきた.近年は公害防止,労働衛生,さらに室内空気調和の面から重要視され,除じんの名で呼ばれることもある.集じん操作において,粒子を気流中から分離してダストホッパに捕集するには,重力,慣性力,遠心力,拡散力,電気力,あるいは熱力などが活用される.現在,広く用いられる集じん装置を分類すれば,一般には遠心力集じん装置,洗浄集じん装置,ろ過集じん装置,電気集じん装置の4種になる.ほかに重力集じん装置と慣性力集じん装置があるが,これらは粗大粒子の前処理という役割をもたされる場合が多いので,ここでは割愛する.以下,各機種に共通する粒子の作用力について,基礎的事項を中心に述べる.. 接線流入式サイクロン接線流入式は図3・16に示すように含じんガスを外筒に接線流入させるもので,直進型,渦巻型などがある.通常,入口ガス速度は7~15m/sにとられる.この範囲ではガス速度による集じん率への影響は比較的小さく,内筒径が最も大きく性能を支配することになる.圧力損失は入口ガス流速が12m/s程度のとき,直進型では1kPa前後であるが,渦巻型では20%程度圧力損失が少なくなる.接線流入式は,マルチサイクロンとして使用する場合,含じんガスを均等に単位サイクロンへ配分することが難しい.したがって,単独サイクロンとして使用することが望ましい.(2) 軸流式サイクロン含じんガスを軸方向から流入させるもので,これには反転型と直進型があるが,現在広く用いられているのは反転型である.反転型では,同じ圧力損失で,接線流入式に比較して約3倍のガス量を処理することができ,また含じんガスを単位サイクロンに均等に配分することが容易であるため,主としてマルチサイクロンとして採用されている.入口ガス速度は12m/s前後で,圧力損失は0.8kPa程度である.(3) マルチサイクロン(multi-cyclone)サイクロンにおける分離限界粒子径は,式(3・41・b)からわかるようにに比例するので,内筒が小さいほど細かい粒子を捕集できる.したがって,処理ガス量が多く,しかも高い集じん率を必要とする場合には,通常,図3・17に示すように,小口径の単位サイクロンを多数並列に使用することになる.これをマルチサイクロンと呼んでいる.単位サイクロンの最小内筒径は,実用上灰づまりの点から,燃焼排ガスを処理する場合は一般に100mm,ただし,油燃焼排ガスでは150mmが限界である.. 荷電方式荷電方式には一段式と二段式のものがある.一段式はばい煙防止用として最も広範囲に採用されている形式で,粒子に電荷を与える荷電部と,荷電粒子を集じん極に捕集する集じん部とが同一の電界で行われる.この形式では,再飛散ダストに対する荷電と集じんとが繰返し行われるので,ダストの見掛け電気抵抗率が逆電離領域にある場合,逆電離現象を避けることはできない.一方,二段式は荷電部と集じん部の電界が分かれている形式で,微細な粒子で粒子濃度の低い場合を扱う空気清浄器,あるいは微細なダストでダスト濃度も比較的少ないばい煙の処理に用いられる静電凝集器に採用されている.この場合,逆電離は起こらないが,集じん極つち打ち時の再飛散ダストはそのまま同伴されるので,通常,後段に二次集じん装置が必要となる.(2)
