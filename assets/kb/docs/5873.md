# 3・3・1 分析機器

> **章**: 第3章 大気環境保全技術
> **節**: 3・3 分析機器と悪臭防止技術

種々の発生源から排出される一次汚染物質の濃度,量を測定することは,発生源の適正管理に必須であり,排出抑制基準の定められた事業所には,排ガスの測定義務が課せられている.排ガスからの試料採取方法と特定の排ガス成分(ばいじん,SO,NO,有害物質など)の測定方法,計測機器が定められているが,燃焼を伴う発生源については,ばい煙防止に必要な燃料試験(発熱量,灰分,硫黄分,窒素分など),燃焼管理に必要な測定(CO,O,CO,流量,温度など)も重要である.排ガス中の濃度がppmレベルの汚染物質が大部分であるが,ダイオキシン類(DXNs)ではppbからpptレベルの測定が必要である.自動車排出ガスの測定には,固定発生源とは異なる試料採取方法と分析方法が定められており,NO,炭化水素,CO,粒子状物質の排出量が測定される.大気中には一次汚染物質以外に光化学オキシダント,浮遊粒子状物質などの二次汚染物質も存在している.これらの汚染物質濃度の測定は,環境の状態を把握・監視するうえで重要であり,大気環境基準の設定されている汚染物質については測定方法が定められ,国,地方自治体等の数多くの測定局で継続的に濃度の測定が行われている.また,有害大気汚染物質を含む化学物質については大気環境調査が継続されている.a. 窒素酸化物(nitrogen oxides)窒素酸化物(NO:NO+NO)の化学分析法はJIS K 0104:2000に規定されており,亜鉛還元ナフチルエチレンジアミン吸光光度法,ナフチルエチレンジアミン吸光光度法,フェノールジスルホン酸吸光光度法,ザルツマン吸光光度法,イオンクロマトグラフ法がある.自動計測器としては,JIS B 7982:2002に紫外線吸収方式,赤外線吸収方式,化学発光方式と差分光吸収方式の四つが記述されている.化学発光方式(chemiluminescent
