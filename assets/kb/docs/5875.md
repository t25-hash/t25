# 4・1・1 水質指標

> **章**: 第4章 水処理技術
> **節**: 4・1 概説

水質汚濁の原因物質は、人体や水生生物に対して毒性を有するか、特有の生理作用をもつか、あるいは水環境を悪化させる物質として、有機物、重金属類、無機物質、有害化学物質、農薬、病原性微生物などいくつかの分類方法があり、おもにこれらの濃度をもって水質指標としている。また、水域の特性に応じて生息する水生生物の種類による生物指標の考え方もあり、これらを指標として環境基準・排水基準等の水質基準体系が整備されている。以下に水質等の規制に用いられるおもな水質指標について述べる。a. 有機物有機物は水中で溶存酸素を消費し、腐敗したり他の水生生物の生存環境を悪化させるなどの水質汚濁を引き起こすものであり、その量的指標としてBOD,COD などが多く使われる。. BOD(biochemical oxygen demand、生物化学的酸素要求量)でんぷん・糖類などの炭水化物、脂肪、タンパク質、有機酸などの有機物が河川など水系に流入すると、水中の好気性微生物は溶存酸素を消費してこれらの有機物を酸化分解する。すなわち、有機物の種類にかかわらず酸素消費量が有機物総量の指標となる。これを利用して、一定の条件下(20°C5日間が標準)で微生物が有機物を分解する際に消費する酸素量をもって有機物量の指標としたのが、BOD である。BOD は本来、炭素系有機物量の指標だが、検水中に硝化菌が多く存在するとアンモニア等の窒素化合物を酸化する際に酸素を消費し、見掛け上BOD 値が高くなる。これを避けるため、アリルチオ尿素(ATU : allylthiourea)などの硝化抑制剤を添加して測定したBOD をATU-BOD またはC-BOD と称し、有機物本来の指標として扱う場合もある。. COD(chemical oxygen demand、化学的酸素要求量)BOD と同様、有機物の種類に関係なく一定の条件下で有機物を酸化剤により酸化し、その際に消費する酸素量をもって有機物量の指標とするのが、COD である。重クロム酸カリウムは酸化力が強く、水中に存在する有機物をほとんど酸化するため、全有機物量の指標として用いられることも多い(COD)。わが国では明治時代から過マンガン酸カリウムで酸化される物質量が有機物量の指標として使われているが(COD)、海外ではCODが主流であり、COD値の評価に当たっては注意が必要である。なお、UV(ultra violet ray、紫外線)吸光度によるCOD モニタが総量規制対象施設のCOD 連続監視などに用いられている。これは有機物濃度と波長254nmのUV 吸光度がよい相関関係にある場合にCOD の代替指標とするものであり、薬品を使わない利点がある。. TOC(total organic carbon、全有機性炭素量)TOCは水中に含まれる有機物中の炭素量を表したものである。BOD やCOD が酸素消費量を表すのに対し、TOCは有機物そのものの量を表すものであり、平成15(2003)年に改正された最新の水道基準では長年使用された過マンガン酸カリウム消費量に代わって有機物量の指標に採用された。. TOD(total oxygen demand、全酸素要求量)TODは試料を燃焼させたとき、試料中の有機物の構成元素である炭素、水素、窒素、硫黄、リンなどによって消費される酸素量と定義され、BOD やCOD と同じ概念である。理論的酸素要求量に最も近10―67い値が得られ、測定所要時間が短く(通常3~4分)、測定結果の自動記録あるいは測定値信号の工業的計装への応用が可能であり、水質の常時監視などに用いられている。b. 富栄養化物質富栄養化(eutrophication)とは本来、地殻変動等により生成した湖沼において、周囲から水とともに生物体や栄養塩類・土砂が流入・循環・蓄積し、次第に埋まって湿原化・草地化・森林化していく自然現象である。しかし近年では、窒素・リン等の栄養塩類が工場排水や生活排水とともに湖沼や内湾に流入することによって水中のプランクトンや藻類が成長繁茂し、透明度の低下等によって自然景観としての価値を低下させるだけでなく、かび臭等により上水水源としての利用を困難にしたり、赤潮の発生等により漁業にも悪影響を及ぼすなどの現象を富栄養化と称している。富栄養化した水域ではCOD を指標として排水規制するだけではその進行を防止し得ないので、窒素・リンによる水質規制が実施される。なお、平成12(2000)年度より実施された第五次総量規制では、指定地域内で排水量400m/d以上の特定施設に対して全窒素・全リンの自動測定装置の設置が義務付けられた。そのほか、藻類現存量を示すクロロフィルaや、藻類培養試験で求めるAGP(algal growth potential、藻類潜在生産力)などが富栄養化の指標として用いられることがある。c. 重金属類など飲料水や汚染された魚介類・農産物などを摂取することにより生体内に毒性物質が侵入し、急性・慢性中毒などの害を与えるものとして、水銀・カドミウム・鉛・クロムなどの重金属、メチル水銀などの有機金属化合物、あるいはヒ素・セレン・アンチモンなどの非金属が、おのおのの濃度により規制される。わが国では工場や鉱山の排水に含まれる水銀やカドミウムに起因する水俣病やイタイイタイ病といった重大な健康障害事例があり、こうした蓄積性環境汚染の防止は重要な課題である。d. 有害化学物質重金属類以外にも、シアン化合物や、DDT(dichloro diphenyl trichlorethane)などの有機塩素系殺虫剤、PCB(polychlorinated biphenyl)、農薬などの有害化学物質が規制されるほか、有機溶剤や金属脱脂剤として利用されるトリクロロエチレンなどの有機塩素系化合物も発がん性問題などにより規制される。平成年代に入ると、ダイオキシン類による環境汚染、さらに内分泌かく乱化学物質(endocrine disruptors、いわゆる環境ホルモン)による生態系への影響が問題となり、対策が検討・実施されている。なお、水質規制はおもに個々の物質の濃度をもって行われているが、ダイオキシン類[dioxins,PCDD(polychlorinated dibenzo-para-dioxin、ポリ塩化ジベンゾパラジオキシン)およびPCDF(polychlorinated dibenzo furan、ポリ塩化ジベンゾフラン)にコプラナPCB を含めた200種類以上の有機塩素化合物の総称]の評価は、最も毒性の強い2,3,7,8-TCDD の毒性を1とし、その他の化合物の毒性の強さを2,3,7,8-TCDD に換算し、その換算重量を加算して毒性等量(TEQ: toxicity equivalent quantity)という単位で表現し、評価指標としている。また農薬は、従来チウラムやシマジンなど個々の農薬の濃度をもって規制していたが、最新の水道基準では101種類の農薬の目標値を設定し、各検出濃度/各目標値の比の合計値が1を超えないこととする新たな規制の方法がとられている。e. 病原性微生物生物数を直接・間接的に汚濁物質の指標として使用するものである。大腸菌(Escherichia coli)はコレラ菌などの腸管系伝染病の病原菌よりも環境中での耐性が強いため、ふん便性汚染の指標として用いられたが、初期の水道基準設定当時は大腸菌のみを迅速に検出する技術がなかったため、大腸菌の有する生化学的性状を備える細菌群を大腸菌群とし、ふん便性汚染の指標とした。しかし今日では迅速簡便な大腸菌培養技術が確立されており、平成15年に改正された最新の水道基準では大腸菌が指標として採用された。なお近年、クリプトスポリジウム(cryptospor-idium)が耐塩素性病原性原虫として注目されているが、これを浄水管理の中で常時直接監視するのは技術的に困難なため、大腸菌や嫌気性芽胞菌を人や野生動物のふん便性汚染の代替指標とし、汚染のおそれがある場合は適切なろ過操作を行うとしている。f. 生物指標生物数を直接・間接的に汚濁物質の指標として使用するのではなく、ある環境に特有に生息する生物の種類・個体数をもって水環境の状況を判定する方法を生物指標という。河川の場合、代表的な指標生物として比較的移動性の少ない水生昆虫が利用され、清水域ではカワゲラ・カゲロウ等、汚濁水域ではセスジユスカ等が挙げられる。水質環境基準の生活環境項目では、汚濁が増すのに応じてヤマメ・イワナ等、サケ・アユ等、コイ・フナ等が挙げられている。g. その他の水質指標水の基礎的性状として色度、濁度、味、臭気など、そのほかpH、電気伝導度、硬度、蒸発残留物、酸度、アルカリ度など、さまざまな水質指標が目的に応じて用いられる。
