# 4・1・4 水処理法

> **章**: 第4章 水処理技術
> **節**: 4・1 概説

水の処理法は、水道用水(浄水)・工業用水等用水の処理法と、下水・工場排水等排水の処理法とに大別され、初期においては沈殿・ろ過といった共通の単位操作が利用されてきたが、利水目的の多様化や汚濁の複雑化とともに特有の用水・排水処理技術が開発され、利用されている(表4・12)。a. 工業用水工業用水には上水道、工業用水道を利用するほか、自己水源を確保し必要に応じて処理を行う場合があり、用途に応じて要求水質が異なるためさまざまな処理法が利用されている。使用量の多い冷却用水の場合、地下水等をそのまま使用したり、河川水等では簡易な沈殿・ろ過処理を行って使用することもあるが、一般的には凝集沈殿・急速ろ過法によって処理される。凝集沈殿では、マイクロサンドなどの沈降促進材を添加し従来の数十倍の超高速処理を行う方式、ろ過では各種の繊維ろ材を用いる方法や、連続でろ過とろ材洗浄(逆流洗浄)を行うろ過機なども開発されている。また、高度な濁質除去を必要とする場合には、精密ろ過膜(MF膜: microfiltration membrane)や限外ろ過膜(UF 膜:ultrafiltration membrane)などを用いたろ過法も採用されつつある。MFろ過法やUFろ過法はナノろ過法や逆浸透法の前処理としても用いられる。ボイラや熱交換器などに用いる冷却用水では、スケールを防止するために用水中の硬度除去を目的としてゼオライトやイオン交換樹脂を用いるイオン交換法が一般的であるが、高塩分含有水[鹹水(かんすい)]の脱塩や、より高純度の用水を必要とする製品処理・洗浄水には電気透析法や逆浸透法も用いられる。平成10(1998)年ごろからは、低圧で用いることができる逆浸透膜が開発されており、ナノろ過法として普及しつつある。c. 生活排水生活排水処理においては活性汚泥法、生物膜法などの生物処理が主流となっている。これは生活排水が微生物によ10―78環境システムる分解が比較的容易な有機性排水であること、有機物濃度が比較的低いこと、物理化学的処理では凝集剤などの薬品の使用量が多く、ランニングコストが高価になってしまうことなどの要因による。生物処理で利用する微生物は自然界に存在する好気性および通性嫌気性の微生物であり、生物処理は河川の自浄作用を高密度化、高速化した処理法である。生活排水(下水)の水質は、1人1日当たり汚濁負荷量がそれぞれBOD 58g,COD 27g,SS 45g,T-N 11g,T-P 1.3g と示されており、1人1日当たりの排水量を300Lと仮定して平均水質を求めるとBOD 193mg/L,COD 90mg/L,SS 150mg/L,T-N 37mg/L,T-P4.3mg/Lとなる。最も広く用いられている活性汚泥法は、エアレーションタンクに空気を吹き込み、水中に酸素を供給するとともに汚水と活性汚泥を混合かくはんすることにより、活性汚泥中の微生物が汚水中の有機物を吸着・酸化分解し、汚水を浄化する。活性汚泥混合液は後段の沈殿池で処理された上澄み水と沈降した活性汚泥に分離され、沈降汚泥の大部分は返送汚泥として、再びエアレーションタンクに返送される。分離された上澄み水は消毒後放流される。浮遊生物処理法である活性汚泥法は標準活性汚泥法のほかに種々の変法が開発されており、ステップエアレーション法、オキシデーションディッチ法、長時間エアレーション法、回分式活性汚泥法などが実用化され、広く普及している。従来の生活排水処理においてはBOD を主とする有機物除去に重点がおかれてきたが、近年では富栄養化原因物質である窒素、リンも除去できる処理方法として、循環式硝化脱窒法、嫌気無酸素好気法、嫌気好気法、硝化内生脱窒法などの導入が進められている。また、水需給の逼迫した地域においては、生物処理した処理水をさらにろ過、凝集沈殿、オゾン酸化などにより高度に処理して、水洗トイレ用水、散水用水、修景・親水用水などに再利用している。さらに、身近な親水空間として下水処理水を小河川や水路に導入し、健全な水循環を維持・回復し、快適な水環境を創造するなどの試みも行われている。排水処理施設が普及するにつれて、処理に伴い発生する汚泥量も増大し、その処理が問題となっている。下水汚泥は濃縮、脱水、焼却された後、陸上・海面埋立処分されてきたが、埋立処分地の残余年数も減少してきており、焼却灰を焼成れんが、透水性ブロック、セメント原料、路盤材などの建設資材として有効利用したり、コンポスト、乾燥汚泥として緑農地に還元し、肥料・土壌改良剤として有効利用する方法が開発・実施されている。下水道未整備地域においてし尿や浄化槽汚泥を処理するし尿処理施設は、昭和30~40年代(1960年ごろまで)には嫌気性消化法がおもに採用されていたが、昭和50年代ごろ(1980年代以降)からは公共用水域の水質保全に対応するため、生物脱窒処理方式として低希釈二段活性汚泥法や高負荷脱窒素処理方式が開発された。近年では固液分離工程に膜分離法を採用した処理方式が実用化されているほか、生ごみ等の有機性廃棄物を併せて処理しメタン発酵によりバイオガスとしてエネルギー回収したり、枯渇資源といわれるリンを回収するなど、汚泥再生処理センターとして新たな役割を担っている。d. 産業排水産業排水は業種・発生源によって排水の量や質が多種多様であるため、処理計画に当たっては、排水の特性に応じていくつかの処理法を組合わせるなど、最適な処理方式を選択する必要がある。また、用水・排水の節減、処理・処分対象となる汚濁負荷の低減方法等についてあらかじめ調査・検討しておく必要がある。例えば水収支の明確化、冷却水や特殊な操作・処理を要する油分や有害物質等を含む排水の分離、洗浄方式の検討、製造工程等の最適制御、装置・設備の改良と点検、回収水・処理水の循環利用、処理が容易で有害物質を使用しない原材料への転換も含めた製造工程の見直し、副産物・有価物の回収、排水の量・質の平均化などが、検討課題として挙げられる。処理方式は一般に、有機性排水では活性汚泥法などの生物処理法が、無機排水では中和法や凝集沈殿法が多く採用され、放流先の水質規制状況に応じてろ過や活性炭吸着などの高度処理法が付加される。排水処理に伴って生成する汚泥等の廃棄物は、沈降濃縮・脱水・乾燥等によって減量したあと最終処分する必要があるが、ダイオキシン対策の施行により焼却処分は減少している。高濃度の有機性排水では生物処理の前処理として、凝集沈殿工程や加圧浮上工程、UASB 法(upflow anaerobic sludge blanketprocess)などの嫌気性処理工程、酵母処理などの高負荷処理工程を行うこともあるが、下水道の普及した地域では、これらの除害施設で処理したあと下水道に放流する事例も増えている。今日では循環型社会や環境に配慮した工場のあり方が問われており、使用水量の多い業種では膜分離法など水の回収・再利用技術が、有機汚濁負荷の高い排水ではエネルギー回収や汚泥資源化あるいは汚泥減量化技術が、有害物質を含む排水では無害化技術や管理手法がそれぞれ評価され、ゼロエミッションなど工場全体のプロセス見直しと併せて、新たな技術開発やプロセスの再構築が活発に行われている。e. 土壌・地下水浄化土壌および地下水汚染対策は、汚染土壌と人との接触する機会を抑制するばく露管理法、汚染土壌あるいは汚染物質の移動・地下水汚染を抑制するばく露遮断法、汚染除去法の三つに大別される。ばく露管理法は土壌について、立入禁止、舗装、盛土、土壌入替えなどの措置がとられ、ばく露遮断法は汚染地下水を対象に、封じ込め、遮水・遮断工法による封じ込め、不溶化などの措置がとられる。汚染除去法としては、原位置抽出法(土壌ガス吸引法、地下水揚水法、エアスパージング法)、原位置分解法[酸化分解、還元分解、生物分解法(バイオレメディエーション、原位置土壌洗浄)]などがあり、汚染物質の種類、汚染状況、土地利用方法等に応じて対策が検討される。
