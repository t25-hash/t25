# 4・2・1 浮遊物質の分離

> **章**: 第4章 水処理技術
> **節**: 4・2 水処理に関する要素技術

水中に含まれる浮遊物質(SS : suspended solids)は、和名では懸濁物質ともいい、通常、粒径1 mから2mmまでの水に溶けない懸濁性の物質をいう。浮遊物質の分離は、浮遊物質の大きさ、あるいは水と浮遊物質との密度差を利用した固液分離操作である。大きさを利用した操作としては、スクリーニングやろ過分離があり、密度差を利用した操作としては、沈降分離、凝集沈殿、浮上分離および遠心分離がある。a. スクリーニング(screening)。 概要スクリーニングは、水中の粗大浮遊物を除去する操作の一つである。用排水の分野では、上水や海水の取水口、下水や排水処理施設の流入口、ポンプ場の流入口および水路における粗大浮遊物質の除去などに用いられている。この場合の粗大浮遊物質としては、ごみとして考えられるすべてのものが対象となる。スクリーニングの目的は、後続するポンプなどの機械類の保護、配管などの閉そく防止、水処理工程の円滑化および水域の汚染防止にある。. 分類操作面からは、流れを止め静止状態で沈降させる回分式と、連続流れのもとで沈降させる連続式とがある。形状面からは平面形状により長方形、正方形および円形に分けられる。水理面からは水平流と上向流があり、水平流には平行流(横流)と放射流がある。平行流は長方形および正方形に、放射流は円形および正方形に、上向流は円形および正方形にそれぞれ適用される。. 回分式沈降分離装置懸濁水を槽に満たして一定時間静置し、清澄水と沈殿物に分ける方法である。少量の場合や、流入が間欠的な場合に用いる。. 連続式沈降分離装置(1) 平行流式(横流式)沈殿池[図4・13(a)]懸濁液は池の一端から流入し、清澄化されて他端から流出する。この間に懸濁物質は底部に沈殿し、汚泥かき寄せ機(sludge col-lector)あるいは底部のこう配により集めて排出する。上澄み液を均等に流出させるため、越流せき(overflow weir)が設けられる。汚泥かき寄せ機には、チェーンフライト式、往復動式、走行サイフォン式および水中けん引式などがある。(2) 放射流式沈殿池[図4・13(b)]懸濁液は池の中央より流入し、放射状に流れて池の全周より流出する。流入口の周囲には円筒形の整流板を設ける。越流せきは、普通、全周に設ける。汚泥かき寄せ機には回転式が用いられる。回転式には中央駆動式と周辺駆動式とがある。かき寄せ機の回転により沈殿物を池底中央ピットに集めて排除する。放射流式は汚泥かき寄せ機の機構が単純で電力消費量が少なく、維持管理費が安価である。(3) 浮上分離(floatation separation)。 概要密度差を利用し、重力場において懸濁物質を浮上させて分離する方法を浮上分離という。浮上分離法には、懸濁物質の密度が水より小さい場合に、自然に作用する浮力のみを利用して浮上させる自然浮上法と、密度が水とほぼ等しいかまたは水より大きい場合に、懸濁物粒子に微細気泡を付着させ、見掛けの密度を小さくして浮上させる強制浮上法とがある。浮上分離法は狭義に強制浮上法のみを指す。自然浮上法はオイルセパレータに代表され、排水中の油分の回収・除去に用いられる。強制浮上法は、気泡の作り方により気泡接触型(空気分散法)と気泡析出型(容積析出法)に分類される。気泡接触型は従来より鉱工業分野において選鉱に用いられているが、気泡が大きいため微細粒子を含む懸濁液の固液分離には適さない。このため、水処理分野では微細気泡を発生できる気泡析出型が多く用いられている。. cen-trifuge)円筒状ボウルであるが、この場合、長さに比べて直径のほうが大きい。このため給液により固液界面が乱されやすく、分離効率は一般に低い。回転軸の方向により、立て型と横型がある。一般には分離した固形物の排除は間欠的となる。ボウル内に固形物がたまり、清澄能力が低下してきたら回転したままで給液のみ停止し、スキミングパイプを挿入して残っている清澄な分離液をすくいとった後、固形分はスクレーパで排出する。. デカンタ型遠心沈降分離機(solid bowl centrifuge)ボウル内に固形分をかき出すためのスクリューコンベヤを内蔵した連続排出型の遠心分離機である[図4・31(b)].スクリューはボウルとの間で若干の回転差(5~100min)が設けられている。この回転差を一般に差速と呼ぶ。供給された原液(汚泥)は回転する内胴から外胴に放出される。外胴内部の液深は円筒部端面のせき板により調整が可能である。分離した上澄み液はせき板からいつ(溢)流し、分離液出口から流出する。一方、原液中の固体粒子は遠心力の作用でボウル内壁に沈降し、スクリューにより円すい部に搬送され、脱水後出口より連続的に排出される。ボウルの形状は円筒-円すい形が代表的であるが、このほかに円筒形および円すい形もある。回転軸の方向は水平(横型)が一般的であるが、垂直(立て型)もある。遠心効果は4000以下である。水処理分野においては、汚泥の濃縮および脱水に数多く用いられている。. 分離板型遠心沈降分離機(disk ろ過分離(filtration)。 概要ろ過(filtration)とは、多孔物質(ろ材)を用いて液体中の浮遊物、懸濁物を除去する、固液分離操作の総称で、清澄ろ過(clarifying filtration)とケーキろ過(cake filtration)の二とおりに大別される。清澄ろ過には粒状ろ過、直接ろ過、助剤ろ過、膜ろ過などがある。上水道の浄水工程、排水処理の高度処理工程、あるいは酒、ビールの精製などに使われている。ケーキろ過には、真空ろ過、加圧ろ過、遠心ろ過などがあり、浄水汚泥や下水汚泥の脱水、食品工業などに広く使われている。ここでは清澄ろ過について述べることにする。清澄ろ過は固形分濃度が比較的低い懸濁液を対象とするものである。粒状ろ過は砂、アンスラサイトなどの粒状ろ材を用いる方法で、緩速ろ過と急速ろ過などに分けられる。直接ろ過はセラミックス、網、布、焼結金属などをろ材とするろ過法である。助剤ろ過(pre-coat filtration)は布などのろ布にケイ藻土、粉末活性炭などのろ過助剤をコーティングしてろ過を行う方法で、良質のろ過液が得られる。緩速ろ過は上水処理に古くから用いられているが、良質なろ過水が得られる反面、広大な敷地を要することや処理できる原水の水質が比較的良質でなければならないことから、最近ではあまり用いられなくなっている。急速ろ過は上水処理や排水処理の高度処理に多く用いられている。. 緩速ろ過(slow filtration)砂層に低速(4~5m/d)で水を通し、砂層表面に浮遊物を捕捉すると同時に、砂表面に生成する生物ろ過膜を利用して生物化学的に浄化する方法で、濁度成分、細菌類、溶解性物質(アンモニア、鉄、マンガンおよび水の臭気等)などの除去が可能である。この方式は捕捉した固形物の洗浄機構をもたないことが多いので、濁度の高い原水には不適である。. 急速ろ過(rapid filtration)100m/d以上の高速でろ過する方式で、さまざまの形式のものが実用化されている。分類すると、1ろ過形態からは固定床式と移動床式、2加圧方法からは重力式と圧力式、3ろ過方向からは下向流、上向流、上下向流および水平流(放射流)などに分けられる。(1) 固定床式ろ過工程と洗浄工程が別々に行われ、ろ過工程中はろ層が静止している。重力式は、下向流または上向流で、流入と流出の水位差を利用してろ過する方式である。圧力式はろ過池を密閉し、原水を圧入してろ過する方式であり、下向流、上向流および上下向流がある。(2) 移動床式ろ過工程と洗浄工程を同時に行う方式で、ろ層が絶えず流動していることから流動層型とも呼ばれる(図4・32)。ろ過池は円筒型開放タンクが一般的である。水流方向は上向流であるが、水平流も見られる。. 急速ろ過の設計要素(1) ろ過速度(filtration rate)水道施設では120~150m/dで設計されることが多い。一方、下水等の排水処理の高度処理に用いられるものでは300m/d以下で用いられている。(2) 逆洗速度(back wash rate)固定床によって捕捉した固形物は逆洗(逆流洗浄)によって排除される。逆洗はろ過砂が流動状態となり、砂表面と粒子間に捕捉された浮遊物を系外へ排除するものである。逆洗速度は0.2~0.9m/minで、逆洗時間は5分程度である。(3) ろ材(filter media)天然ケイ砂、無煙炭(アンスラサイト)、人工軽量砂、プラスチック球、ザクロ石およびチタン鉄鉱などが用いられる。最近ではろ過速度を高く、浮遊物捕捉量を大きくするために繊維ろ材、浮上性ろ材などが開発されている。ろ材にはその要求性能により形状(粒子の直径等)および品質が選択される。マンガン除去のためのろ過砂にはマンガン砂が用いられる。(4) ろ層水道施設では、下向流式で厚さを60~70cmとすることが多い。一方、排水処理では、下向流式の場合はろ砂とアンスラサイトの2槽式とし、厚さを60~100cmとしている。また上向流式の場合にはろ砂の単層式とし、厚さは1.0~1.8mが用いられている。. プレコー卜(助剤)ろ過多孔質のろ過エレメントにろ過助剤をコーティングしたものをろ材としてろ過する方式である。装置は密閉型圧力容器内にろ過エレメントをつけた構造である。ろ過エレメントには形状により円筒型と葉状型がある。円筒型にはステンレス製のウェッジワイヤや針金を円筒状に巻いたものや、カーボン、磁器、焼結金属および樹脂製の円筒多孔質材等が用いられる。葉状型にはステンレス金網、合成樹脂網およびろ布などが用いられている。プレコート剤としては、ケイ藻土が一般的であるが、このほかにもパーライト、活性白土、粉末活性炭、炭素繊維やセルロースなどが用いられる。用途としては、用水のコロイド状有機物や大腸菌などの菌体の除去、除鉄、除マンガン処理、排水のABS(alkyl benzene sulfonates)や油分の除去などに用いられる。. 直接ろ過多孔質の合成樹脂、セラミックス、焼結金属や布、網などをろ材として直接ろ過する方式である。ろ過エレメントには、ろ過筒(カートリッジ)、ステンレス製の平板輪状円盤の積層円筒、およびろ紙状の薄膜などが用いられる。大容量の水を扱う場合、マイクロストレーナがある。用途としては、用水の除鉄・除マンガン処理、電子工業における純水製造用ポリッシャ、原子力工業における復水処理や放射性排水処理、および家庭用浄水器などに用いられる。g.
