# 4・2・4 消毒(disinfection)

> **章**: 第4章 水処理技術
> **節**: 4・33 に示す。ここでは限外ろ過(UF)、精密ろ過(MF)について

a. 消毒の定義消毒とは病原性の微生物の感染力をなくすことをいい、滅菌(pasteurization)とはすべての微生物の生活力を物理的または化学的手段で奪うか、フィルタなどで機械的に除去して、文字どおり微生物を完全になくすことをいう。また殺菌(ster-ilization)とは、微生物の生活力をなくすことであり、この意味では消毒とは病原性微生物を選択的に殺菌することともいうことができる。従来の水道水質基準において大腸菌群(coliform group)は検出されないこととなっているのに対して、一般細菌(neterotro-phic bacteria)については1mL中に100個以下と規定されていることからも、消毒の目的は滅菌を目的にしたものではなく、またその必要もないことがわかる。b. 消毒法およびその特徴殺菌力をもつ化学物質としては塩素ガス(chlorine gas)をはじめとする塩素系化合物(chlorinecompounds)、他のハロゲン類、オゾン、過マンガン酸カリウム、過酸化水素などの酸化剤、銀や銅のイオン、逆性石けん、酸および塩基等を挙げることができる。また物理的な方法として、加熱処理、紫外線、ガンマ線、X線の照射、さらにはフィルタによる除去が殺菌法として列挙される。これらの殺菌法は、消毒の目的、効果、持続性、費用、操作性、毒性、安全性、着色、着臭、発がん性物質などの生成などの点から総合的に検討し選択する必要がある。水処理分野では一般に塩素消毒が行われているが、上水分野ではトリハロメタンなどの発がん性消毒副生成物問題やクリプトスポリ10―105ジウム等の耐塩素性病原性原虫問題、下水分野では処理水放流先の水生生物や利水への影響などが指摘されている。これには水系感染症と発がんとのリスクをいかに評価するかという問題が含まれており、塩素に代わる消毒技術として、二酸化塩素、オゾン、紫外線などの代替技術が開発されているほか、送配水管の内面状態や生物スライム付着状態、あるいは水温や滞留時間など、システム全体としての検討が必要である。表4・26に各種の消毒方式の比較を示す。なお消毒効果の指標として、対象微生物の除去率を対数表示(例えば99.99%除去では4log)することがある。また、必要消毒剤摂取量として、CT 値[=消毒剤濃度(concentration)×接触時間(time)]が使用される。c. 塩素(chlorine)水処理分野で塩素が多用される理由は、効果、持続性、費用、操作性、安全性の点にある。主として使用される塩素剤は、液化塩素、次亜塩素酸ナトリウム、さらし粉(次亜塩素酸カルシウム)、有機性結合塩素剤などである。水中で塩素は、ほとんどが次亜塩素酸(HClO)または次亜塩素酸イオン(ClO)として存在するが、その比率はpH により変化する。pH 7付近では80%以上が次亜塩素酸だが、pH 8以上では大部分が次亜塩素酸イオンとして存在している(図4・75)。次亜塩素酸は次亜塩素酸イオンより殺菌力が強い。わが国の水道法では、給水栓で保持すべき残留塩素を規定しており、塩素消毒は必須である。また下水処埋においても、従来の設計指針では塩素の使用を定めており、大規模な下水処理施設ではほとんどが塩素消毒を行っている。しかし近年ではb項に示すような問題により代替消毒技術が開発されているほか、塩素注入率を低減化する技術や、代替法と組合わせた効果的な塩素注入方法も開発されている。d. クロラミン(chloramines)の生成と不連続点塩素処理(breakpoint chlorination)アンモニアを含む水にあっては、次亜塩素酸はアンモニアと反応して結合型の有効塩素(availablechlorine)を生成する。これらの化合物はクロラミンと総称される。クロラミンにはモノクロラミン(NHCl)、ジクロラミン(NHCl)、トリクロラミン(NCl)があり、前二者は殺菌力があるがトリクロラミンは不安定で、窒素はガス化され分解が進む。水に添加された塩素は、次亜塩素酸の形かモノクロラミンまたはジクロラミンの形で残留していれば消毒効果をもつことになる。次亜塩素酸の形で残留する塩素分は遊離残留塩素(free chlorineresidual)と呼ばれ、クロラミンの形のものは結合型残留塩素(combined residual chlorine)と呼ばれる。結合型の残留塩素濃度はアンモニアとの存在比によりクロラミンの存在形が変化し、ある範囲の塩素の添加量に対しては残留塩素濃度がかえって減少してしまう状況が出現する。この関係は定性的には図4・76のように示されるが、図中点B を不連続点と呼び、これを超えて塩素を加え遊離残留塩素を得るように制御することを不連続点塩素処理と呼ぶ。また図4・76中点Cは、水中に存在する有機物かその他の還元性物質によって消費される塩素量を示している。点Cに相当する塩素量は塩素要求量(chlorine demand)と呼ばれる。消毒の効果を維持するうえでの必要塩素量は、水質要因を考慮して決定されなければならない。クロラミンは遊離塩素に比べて消毒力は低いが、残留性に優れ、消毒副生成物を生じにくいという利点があり、地下水や膜ろ過水など病原性原虫の問題が少ない場合には有効である。e. 二酸化塩素(chlorine 紫外線(UV : ultra violet ray)一般にUV-Cと呼ばれる波長200~280nmの紫外線は殺菌効果を有することが知られている。塩素やオゾンが細菌類の細胞壁や細胞膜を破壊して消毒効果を発揮するのに対して、紫外線は遺伝子に直接損傷を与えることにより増殖不能として死滅させるものである。最も殺菌効果が高いのは波長254nm付近であり、低圧水銀ランプはこの波長を効率的に発生し、ランプ表面の温度も低く、汚れが付着しにくいという利点を有する。中圧水銀ランプは紫外部から可視部まで幅広い発光特性を有するため、効率は低下するものの高エネルギーの紫外線を得ることができ、大量の水を対象とした場合に使用される。実用に際しては、必要な紫外線照射量[(mJ/cm)=紫外線強度(mJ/s・cm)×照射時間(s)]、対象水の紫外線透過率、装置の接触効率等のほか、光回復効果を考慮する必要がある。光回復(photoreactiva-tion)とは、殺菌直後の処理水に可視光が照射されると細菌の光回復酵素により損傷を受けた遺伝子の一部が修復される現象をいう。紫外線消毒は数秒程度の照射で効果があり、副生成物を生じず、病原性原虫の不活化にも効果があるといわれる。
