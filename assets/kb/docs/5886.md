# 4・3・3 し尿処理技術

> **章**: 第4章 水処理技術
> **節**: 4・3 処理技術の事例

a. 概要し尿や浄化槽汚泥を対象としているし尿処理は1950年ごろより開始され,下水道の普及した今日においても約5000万人がし尿処理対象人口となっている.し尿処理方式は,嫌気性消化処理+散水ろ床処理法,嫌気性消化処理+活性汚泥処理方式(20倍希釈,1950年~),好気性消化処理方式(20倍希釈,1960年~)を経て,1975年ごろより生物学的硝化脱窒素処理方式(10倍から無希釈)+高度処理方式に移り,今日に至っている.嫌気性消化処理方式は発生するメタンを利用できる利点はあったが,臭気の発生や浄化槽汚泥の混入による処理効率の低下が,一方,好気性消化処理方式はBOD にしか着目していなかったために,富栄養化対策(窒素・リン除去)に対応できなかったことが,それぞれの衰退の原因となっている.生物学的硝化脱窒素処理方式は,10倍希釈の標準脱窒素処理方式(低希釈二段活性汚泥処理方式)に始まり,無希釈・コンパクト化を図った高負荷脱窒素処理方式,さらには固液分離に膜を採用した膜分離高負荷脱窒素処理方式へと開発・実用化が進み,この3方式が現在の主流となっている.なお,1998年度から,処理のみを行っていたし尿処理から,何らかの資源循環機能をもたせた汚泥再生処理センターへと転換が図られてきている.b. 施設の構成と処理機能し尿処理施設(night soil treat-ment plant)は,図4・111に示すように受入貯留設備,主処理設備,高度処理設備,消毒設備,汚泥処理設備,脱臭設備等より構成されている.. 受入貯留設備受入貯留設備はし尿や浄化槽汚泥をバキューム車から受け入れ,し尿等に含まれる挟雑物を除去する設備であり,受入槽,きょう雑物除去装置,および貯留槽から構成されている.. 主処理設備主処理設備は,標準脱窒素処理方式と高負荷脱窒素処理方式(膜分離を含む)等の生物学的硝化脱窒素処理方式が主流である.生物学的硝化脱窒法は自然界に広く分布する硝化菌と脱窒素菌を用いて排水中の窒素化合物を窒素ガスに転換させるもので,アンモニア等の窒素化合物を含む有機性排水を好気的雰囲気におくことでアンモニアを亜硝酸,硝酸に酸化させ,この生成した亜硝酸や硝酸を嫌気的雰囲気下で窒素ガスに転換させる方式である.硝化反応は,亜硝酸菌,硝酸菌の働きによってNHとNOを基質として硝化する反応[式(4・61)(4・62)]である.一方,脱窒反応は,脱窒菌が有機物を嫌気性状態で分解する際に,硝酸あるいは亜硝酸に含図4・110 を利用して脱窒させる方法で,式(4・61)(4・62)の反応を硝化槽で,式(4・63)(4・64)の反応を脱窒素槽で行わせるものである.本法は一般的に硝化液循環法と呼ばれているが,このほかに脱窒素槽でも少量の空気を入れながら硝化脱窒させる方法(混合分解法)や,脱窒素槽,硝化槽を分割し,交互に組合わせて脱窒素槽へし尿を投入する方法(ステップ脱窒法)等の変法もある.いずれの方法でも槽内のMLSS濃度は6000mg/L程度で運転されている.希釈水は10倍で運転することが多かったが,浄化槽汚泥の混入率増加とともに希釈倍率も減少してきている.仕上槽として二次脱窒素槽と再ばっ気槽を設けており,二次脱窒素槽では一部流入する硝酸の脱窒を,再ばっ気槽では残留BOD の除去と汚泥に付着した窒素ガスの脱気を行う.なお,二次脱窒素槽ではBOD 濃度が低いために外部からメタノール等の水素供与体が添加される場合が多い.沈殿槽では菌体濃度を維持するための汚泥返送を行い,上澄み水は必要に応じて高度処理した後,消毒・放流される.高負荷脱窒素処理方式のフローシートを図4・113に示す.高負荷脱窒素処理法は,高容積負荷で運転することにより無希釈化とコンパクト化を行ったものである.MLSS 濃度を高く維持(12000~20000mg/L)すること,高効率のばっ気装置を備えていることが特徴となっている.硝化脱窒の原理は標準脱窒素処理方式と同じである.槽構成とし図4・112 標準脱窒素処理方式のフローシート例図4・113 高負荷脱窒素処理方式(単一槽方式)のフローシート例10―123ては,脱窒素槽(嫌気性槽)と硝化槽(好気性槽)を分離する複数槽方式と,単一の硝化脱窒素槽で時間を区切って嫌気・好気を繰返す単一槽方式の二とおりがある.いずれの方式にしても,MLSS 濃度を高濃度に維持するための固液分離と濃縮が必要になる.固液分離には重力沈殿法や機械分離法,あるいはこれらの併用で対応しているが,分離液のSS 等の除去が要求されている.高度処理設備は主処理の生物学的硝化脱窒素処理方式の種類によって採用する設備構成が異なり,標準脱窒素処理方式の場合には,凝集分離+オゾン酸化+砂ろ過が一般的である.高負荷脱窒素処理方式にあっては凝集分離+砂ろ過+活性炭吸着が,膜分離高負荷脱窒素処理方式にあっては凝集膜分離+活性炭吸着が一般的である(詳しくは4・2・1,4・2・2項参照).. 消毒設備次亜塩素酸ナトリウム注入による塩素消毒が基本となっている(詳しくは4・2・4項参照).. 汚泥処理設備汚泥処理設備は汚泥脱水後,乾燥焼却処理が一般的であったが,近年の資源化の流れを受けて最近では堆肥化を行う例が多くなっている(詳しくは4・2・5項参照).. 脱臭設備し尿処理施設から発生する臭気には,酸性ガス(硫化水素,メルカプタン類等),塩基性ガス(アンモニア,アミン類等),中性ガス(アルデヒド類)が含まれており,その成分や濃度によって脱臭方式が異なる.代表的な脱臭フローを図4・114に示す.受入貯留設備からの高濃度臭気,ばっ気槽や汚泥処理設備からの中濃度臭気,受入室等の低濃度臭気に分け,脱臭処理している.高濃度臭気の脱臭処理には燃焼脱臭による方式と生物脱臭による中濃度臭気化があるが,夜間臭気対策から今日ではばっ気槽の吹込み空気として用いる生物脱臭が多く用いられている.中濃度臭気は酸・アルカリ・次亜塩素酸ナトリウムによる薬液洗浄+活性炭吸着が,低濃度臭気は活性炭吸着処理が主流である.
