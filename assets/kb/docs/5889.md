# 5・1・1 廃棄物処理・再資源化についての法制度

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・1 概説

a. 廃棄物処理・再資源化に関する法律廃棄物処理の今後のあり方として、廃棄物の排出をできる限り抑制し、そのうえでどうしても発生してしまう廃棄物の再使用・再生利用を進め、最終的に廃棄物となってしまうものに対して安全かつ適正に処理することにより、循環型社会(resource circulation society)を形成していくことが求められている。2000年にはこの循環型社会形成のための廃棄物処理(waste processing)・再資源化に関する法律が体系的に整備された。. 循環型社会形成推進基本法(2000年法律第110号)循環型社会形成推進基本法」は、(1)廃棄物最終処分場の逼迫、(2)不法投棄の増大、(3)処理過程での有害物質の発生(特に産業廃棄物焼却施設からのダイオキシン類の発生は社会問題化)等の状況を踏まえ、喫緊の課題となった廃棄物・リサイクル問題に焦点を定めて設定されたもので、生産から流通、消費、廃棄に至るすべての過程において物質の効率的な利用やリサイクルが進められ、資源消費の抑制と環境負荷の低減が図られる社会形成を推進するための基本的な枠組を定めている。法の対象となるものを、有価・無価を問わず廃棄物等とし、廃棄物等のうち有用なものを循環資源と位置付け、その循環的な利用の促進を図るため、優先順位を、(1)発生抑制、(2)再使用、(3)再生利用、(4)熱回収、(5)適正処分とし、国、地方公共団体、事業者および国民の役割分担を定めている。具体的には、事業者・国民の排出者責任を明確化し、生産者が自ら生産する製品等について、使用され廃棄物となった後まで一定の責任を負う拡大生産者責任(EPR : extended producer responsibility)の一般原則を確立している。また、政府は循環型社会形成推進基本計画を策定し、これをもとに、循環型社会を形成するための施策を推進しなければならないとされている。. 廃棄物処理法等の改正(2000年法律第105号)廃棄物の減量化を促進するとともに、安全かつ適正に廃棄物を処理する体制の整備が大きな課題となってきている。こうした状況を踏まえ、廃棄物について適正な処理体制を整備し、適正処理の促進や不法投棄を防止するため、「廃棄物の処理及び清掃に関する法律(廃棄物処理法)」および「産業廃棄物の処理に係る特定施設の整備の促進に関する法律」の一部改正が行われている。廃棄物の減量化・適正処理を計画的に推進するための枠組の整備として、環境大臣が、廃棄物の排出抑制(waste minimization)、再使用(reuse)等による廃棄物の減量その他の適正な処理に関する施策について総合的かつ計画的な推進を図るための基本的な方針を定めることとしている。また、多量の産業廃棄物を生じる事業場を設置している事業者は、産業廃棄物の減量その他の処理に関する計画を作成して、都道府県知事に提出し、計画の実施状況を報告しなければならないとされている。さらに、産業廃棄物の処理に係る特定施設の整備の促進に関する法律を一部改正し、民間施設も含めた優良な産業廃棄物処理施設の整備を促進するため、対象となる特定施設の要件の緩和を行っている。逆に、廃棄物の適正処理のための規制強化として、廃棄物処理の許可取消し等の要件の追加、廃棄物処理施設の設置に関わる許可要件の追加、廃棄物処理施設の設置に関わる許可取消し等の要件の追加、産業廃棄物管理票(マニフェスト)制度の見直し、廃棄物の焼却の規制、適正処分に関する支障の除去等の措置命令の強化、罰則の強化を行っている。. 資源有効利用促進法の設定(2000年法律第113号)廃棄物・リサイクル対策の方向性として1R(recycle)から3R(reduce, reuse, recycle)への取組みの強化を図るため、事業者によるリサイクルの促進をおもな内容とする「再資源の利用の促進に関する法律(リサイクル法)」が改正され、法律の名称が「資源の有効利用の促進に関する法律(資源有効利用促進法)」に改められた。以下にそのおもな改正点を挙げる。(1) 省資源化・長寿命化による廃棄物の発生抑制対策の推進廃棄物の発生抑制対策を推進するため、製品の省資源化・長寿命化を図る設計・製造や修理体制の充実、アップグレードに対応した設計などを事業者に義務付ける指定省資源化製品制度が盛り込まれた。(2) 部品等の再使用対策の推進部品等の再使用対策を推進するため、部品等の再使用が容易な製品設計・製造を行うことや、回収した使用済み製品から取出した部品等を新たな製品において再使用することなどを事業者に義務付ける指定再利用促進製品制度、特定再利用業種制度が盛り込まれた。(3) 事業者による回収・リサイクルの推進製品の回収・リサイクル対策を推進するため、回収・リサイクルが可能であるなどの要件を満たす製品を指定し、回収・リサイクルすることを事業者に義務付ける指定再資源化製品制度が盛り込まれた。(4) 副産物(産業廃棄物)対策産業廃棄物の最終処分量の削減に資するため、工場等でも発生する副産物について、事業者自らが生産工程の合理化等による副産物の発生抑制対策と発生した副産物の利用促進によるリサイクル対策に計画的に取り組むことを義務付ける特定省資源業種制度が盛り込まれた。. 製造業者および輸入業者(製造業者等)指定した取引場所において、自らが製造等した対象機器の廃棄物を引き取る義務を負い、引き取った対象機器の廃棄物について、再利用または破砕処理、処分を行わなければならない。(2) 小売業者小売業者は、自らが過去に小売販売した対象機器の引取りを行う。引き取った対象機器は、中古品として再利用する場合を除き、その対象機器の製造業者等(それが明らかでないときは指定法人)に引き渡す。(3) 消費者消費者は、対象機器の廃棄物を小売業者等に適切に引き渡し、収集・再商品化等に関する料金支払いに応じる。(4) 市町村市町村は、その収集した対象機器の廃棄物を製造業者等(または指定法人)に引き渡すことができる(ただし、自ら再商品化等を行うことも可能)。. 建設リサイクル法(2000年法律第104号)1993~1995年の平均不法投棄量は約39万t(トン)といわれ、そのうちの約図5・1 容器包装リサイクル法の仕組み図5・2 家電リサイクルの仕組み90%が建設廃棄物(construction 処理計画および処理の主体。 一般廃棄物処理計画および処理の主体地域内における一般廃棄物の処理に関する計画は、市町村が定めなければならないとされている。したがって一般廃棄物処理の主体は市町村である。一般廃棄物処理計画には、一般廃棄物の排出抑制のための方策に関する事項等を定めるとともに、関係する他の市町村の一般廃棄物処理計画と調和を保つよう努めるべきことが規定されている。また、その構成としては、基本的計画および各年度ごとの実施計画を定める必要がある。市町村はそれらの一般廃棄物処理計画に従って、生活環境保全上支障が生じないうちにその区域内における一般廃棄物を収集し、これを運搬し、処分(再生を含む)しなければならない。市町村ならびに市町村から一般廃棄物の処理を委託された者や一般廃棄物処理業者が一般廃棄物の収集、運搬、処分を行う場合には、一般廃棄物処理基準(特別管理一般廃棄物の場合は特別管理一般廃棄物処理基準)に従わなければならない。市町村は、多量の一般廃棄物を生じる土地または建物の占有者に対し、減量に関する計画の作成等を指示することができる。市町村は、当該市町村が行う一般廃棄物の処理に関し、条例で定めるところにより手数料を徴収することができる。この市町村が徴収する手数料は、一般廃棄物の特性、処理に要する費用等を勘案して定めなければならない。. 産業廃棄物処理計画および処理の主体産業廃棄物処理の主体はその廃棄物を出した事業者であるが、事業者はその産業廃棄物を自ら運搬または処分するほか、運搬については産業廃棄物収集運搬業者その他の者に、処分については産業廃棄物処分業者その他の者にそれぞれ政令に定める基準に従い委託することができる。委託に当たっては、委託しようとする者の許可内容等の確認を行うとともに、委託した業務が完了したことを確認することが必要である。事業活動に伴って生じる産業廃棄物を処理するために産業廃棄物処理施設が設置されている事業場を有する事業者は、産業廃棄物処理責任者をおかなければならず、また、その処理量等を記載した帳簿を備え、5年間保存しなければならない。都道府県知事または政令市長等は、その地域内において、事業活動に伴い多量の産業廃棄物を生じる事業場を設置している事業者に10―135対し、その処理に関する計画を作成するよう指示することができる。また、事業活動に伴い特別管理産業廃棄物を生じる事業場を設置している事業者は、環境省で定める資格を有する特別管理産業廃棄物管理責任者をおき、帳簿を管理して、これを5年間保存しなければならない。
