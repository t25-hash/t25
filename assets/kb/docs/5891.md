# 5・2・1 保管、収集、輸送のあり方

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・2 廃棄物の収集、輸送

都市ごみ処理システムでの保管・収集・輸送の位置付けを示すと図5・7のようになる。保管とはごみが発生し収集されるまでの作業をいい、収集とは収集機材にごみを積み込むまでの一連の作業、輸送とは積み込まれたごみを目的地まで運ぶ作業をいう。目的地には、ごみの積換えを行う中継施設や焼却工場などの中間処理施設、図5・7ごみ処理システムにおける保管・収集・輸送の位置付け表5・12 都市ごみ保管・収集・輸送の構成要素の例構成要素内容収集場所(保管場所)各戸収集ステーション収集機器積込専用容器を用いたステーション収集その他収集機材圧縮機付きトラック(パッカー車等)無がいトラック天がいトラックパイプ輸送その他収集時間昼間収集夜間収集ごみ排出容器(一時保管容器)ポリ袋ポリバケツその他中継施設コンテナ車(container vehicle)への積替え船舶への積替えそして埋立てなどの処分を行う最終処分場がある。都市ごみ処理事業において保管・収集・輸送は住民との接点という重要な役割を担っており、住民の目に触れる機会が多い極めて重要な工程である。家庭で発生したごみは速やかに収集して保管場所(居住空間)から取り除かないと、(1)公衆衛生の低下、(2)集積したごみによる空間の占拠、(3)環境の美観低下等の問題が引き起こされる。よって、発生したごみを速やかに取り除く必要があり、また保管中にもコンテナ(container)等の収集容器を置き、ごみ集積所の衛生管理、美観保持を容易にする必要がある。一方で、保管容器としてポリ袋を指定し、その他の収集容器を使用しない自治体が増えてきているが、それは、(1)収集場所でのごみの容量を小さくできる、(2)作業員が収集しやすいため収集時間が短縮できる、(3)住民が容器を引き取る必要がない、などの理由による。都市ごみ処理経費の中で収集・輸送の占める割合が最も高く、この作業に携わる労働力は全体の50%以上となることから、この工程の適正化が都市ごみ処理事業全体にもたらす効果は大きい。保管・収集・輸送は機材、方法等のさまざまな要素からなるが、その代表的なものは表5・12のようになる。収集・輸送サービスの量と質は、そのサービスに対する要求と、サービスを提供する自治体の負担等の制約条件との兼合いで決まる。したがって、その計画策定に際しては、表5・12に列挙した方法と機材の組合せの中から、(1)各種要求を満足する、(2)最も経済的である、の二つの条件を満たすものを選択することになる。サービスの内容を決定する要素の例を次に示す。(1) 住民の要望(環境保全、快適性、美観、時間的・経済的負担など)(2) ごみ発生量(3) ごみ質、種類(4) 気候(特に気温)10―141これらの要素によって、収集頻度、収集容器指定、収集場所、収集するごみの種類などが決まるが、各要素は互いに影響し合うことも考慮する必要がある。一方、要素選択の制約条件となるものの多くは地域特性と呼ばれるもので、各自治体特有のものである。それらの制約条件を以下に示す。(1) 自治体の財政(2) 収集地域の地形(起伏・河川等)(3) 気候(降雨・降雪量、風等)(4) 住民の生活習慣(収集方式への協力等)(5) 使用機材(6) ごみ処理方法(混合焼却、可燃物のみ焼却、その他)(7) 労働問題(労働条件、労働衛生等)(8) 道路交通状況(交通量、道路の幅員等)これらの制約条件の中で最も大きいのは費用の問題、すなわち自治体の財政であろう。例えば、いつでもごみが出せるようにしてほしいという住民の要望に対しては、パイプ輸送方式(pipe convey-ance system)の採用、車両収集頻度の向上による排出時間指定の解除、コンテナ等の保管容器設置によるごみ投入方式の採用などの解決策が考えられる。また、ごみを出す際の利便性を高めてほしいという要望に対しては、パイプ輸送方式の採用や、家庭から収集場所までの持出し距離を短くするなどの方法がある。しかし、その実施に当たっては、費用が制約となることが多い。また、これらの制約条件も、各条件が互いに影響し合うことが多い。ここに挙げたサービス内容の決定要素と制約条件の項目には重複するものもある。気候についていえば、気候はハエや悪臭の発生に関係するもので、必要収集頻度を決定する要素であるが、降雨降雪量の多少によっては無がいトラックの使用を制約する条件にもなりうる。一般に、ごみ処理施設建設に対する反対理由として、収集輸送車の出入りがよく指摘されるように、ごみ処理に関わる環境影響の中で、収集輸送車の運転に伴うものが問題とされることが多い。収集作業においては収集場所周辺での環境影響が考えられる。また、車両走行に際しては排ガス、騒音、振動、交通渋滞、交通事故、道路への汚水垂れ流し、悪臭等の環境影響が予想される。作業員に対する影響として、作業時の交通事故、転倒、転落、無理な動作、障害物との接触による外傷、疾病あるいは作業中にさらされる排ガスによる影響等に関しても検討する必要がある。都市ごみ収集・輸送計画の策定に当たっては、これらのサービス内容の決定要素と制約条件との相互関係を把握し、整理する必要がある。保管・収集・輸送計画の策定手順を示すと次のようになる。(1) 調査、分析(2) 保管・収集・輸送におけるサービス内容の設定(3) 調査、分析の範囲の設定(4) 収集機材、方法に関する組合せの選定(5) 収集機材、方法の決定(6) 収集経路の設計(7) 評価、修正計画をいったん策定した後も、サービスの内容を決定する要素はつねに変化するため、各種要求と収集・輸送の実態に隔たりが生じることがある。したがって、つねに見直しのための調査、評価を行うことが必要である。
