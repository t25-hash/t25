# 5・2・3 中継施設

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・2 廃棄物の収集、輸送

収集輸送システムの効率化を図る手段として、収集は小型収集車、輸送は大型の輸送車を用いる中継輸送システム(refuse trans-fer system)がある。このシステムでは収集と輸送の機能が分離さ図5・8 パイプ収集輸送システムれているため、収集車は収集作業に専念させられる。また、ごみを発生する都市の近隣に焼却施設の用地、最終処分場を確保することが難しく、処理の広域化が進んでいる状況において、輸送作業に大量輸送方式を採用できるため、収集輸送費用全体として経済的メリットが得られる。また、その効果は処理・処分地までの距離が遠いほど有利となる。現在までに採用されている中継輸送方式を分類すると、図5・9のようになる。中継施設の採用理由に関する調査結果によると、半数以上が処理・処分場の立地と関連するもので、収集輸送作業の効率化、狭あい地対策、交通事情の悪化などが続いている。また、ごみ量、地域特性、財政事情を考慮した最適の方式が各自治体で採用されているため、中継輸送方式には多くの方式があるが、最近は輸送効率、所要人員、経費、環境保全、作業環境確保の面からコンパクタ・コンテナ方式(compactor container sys-tem)が主流となっている。その実施例を図5・10,5・11に示す。この方式において使用されるコンテナは、ごみが詰め込まれる際の圧力に耐えうる強度が必要であり、さらに輸送前の一時保管設備の要素ももっているため、密閉性が重要となる。また、輸送においては、上記に加えて、(1)最適サイズ、(2)衛生的、(3)高信頼性、(4)荷の積み下ろしが容易、(5)作業の安全性などの条件(c) はしけ:ごみを積む箱型の船で、舵はあるが引船によって引かれるもの(d) コンテナ運搬用バージ船:東品川清掃作業所で不燃ごみを圧縮して詰め込んだコンテナを積む専用船で、押船によって移動する。押船と合体するため舵はない10―145も満たす必要がある。ごみ中継施設を計画する際には主として以下のような事項を検討する必要がある。(1) ごみ中継施設も、焼却場や埋立地同様、迷惑施設と見られるため、施設の環境保全対策上、騒音、悪臭、汚水等に対して適切な処理を行う。(2) 中継輸送システムは、収集-積替え-輸送の各システムからなっており、全体のトータルバランス(経費、環境保全、労働環境等)を考慮した最適なシステムを検討する。(3) 現在の主流であるトラック輸送のみならず、船舶輸送やカプセル輸送システムについても、地域特性、交通事情、地理的条件などを考慮に入れて、地域に適合した中継輸送方式とする。なお、河川や海に面した地域では中継所で船舶にごみを積み替えて輸送を行う場合がある。例えば、東京都内には五つの船舶中継所があり、各区で収集された不燃ごみをそれぞれの中継所にて専用船に積み替えて、所定の処理施設へ運搬を行っている。各中継所へ集められた不燃ごみは、それぞれの積込設備にて専用船に積み替えられる。船舶輸送に用いられる船舶は図5・12のようになってお図5・10 コンパクタ・コンテナ方式図5・11 コンパクタ・コンテナ方式のごみ処理フロー(マンションに併設された地下式中継施設例)(a) 引船:はしけを引く動力船(b) 押船:バージ船と合体する動力船。引船、押船両方の機能を併せ持つものもある(写真はコンテナ運搬用バージ船と接続)(e) クレーン:東品川清掃作業所に設置されたコンテナ用クレーン図5・12 船舶輸送に用いられる船舶の例(提供:廃棄物研究財団)10―146環境システムり、ばら積みの船舶1隻当たりの運搬量は約11t弱、東品川清掃作業所では、コンテナ1個当たりに約6~7tの不燃ごみを圧縮して詰め込み、1隻当たり一度に16個のコンテナを運ぶことが可能である。船舶による廃棄物の輸送は、大量貨物の長距離輸送が可能であるため、輸送効率を向上させられるとともに、陸上交通への影響がなく、環境への負荷が車両に比べて少ないため、廃棄物の臨海地域への輸送に適している。ただし、ガントリークレーン等の港湾施設の準備に多額の投資が必要となるうえ、天候による影響を受けやすいなどのデメリットもある。
