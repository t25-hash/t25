# 5・3・3 焼却技術

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・19 に示す。

a.廃棄物焼却技術の要件の概要 廃棄物処理における焼却技
術には，焼却による廃棄物の減容，減量，安定化のみならず，地球
温暖化防止対策の一環として，また資源循環という立場としての焼
却廃熱回収利用の視点が必要になってきている．また，焼却による
二次公害の防止，特にダイオキシン類の排出防止のための各種要件
を満たすことも重要課題の一つである．
b.焼却処理物の燃焼特性と燃焼装置の考え方 産業廃棄物，
一般廃棄物ともにその種類によって形状，質量等物性が非常に多様
性に富んでいるとともに燃焼特性も廃棄物種類によって大きく異な
るという特徴があり，この特徴を踏まえて焼却炉を選定する必要が
ある．
乾燥域帯の必要性を踏まえて，ストーカ式焼却炉では一般に乾
燥，燃焼，後燃焼の3段階を備えた形式の炉が多い．これらの各段
階に対して必要な機能は以下のとおりである．
(１)乾燥段階（drying stage）
放射伝熱や接触伝熱がごみ層の内部まで及ぶことを目的として，
火床上に送り込まれたごみが適度にほぐされること．特に低質（低
発熱量）ごみに対して，かくはん機能をもつこと．
(２)燃焼段階（combustion stage）
適度にごみ層のかくはんが行われること．また，易燃物が先燃え
して吹抜け（ブローホール）を作りやすいため，ごみ層よりもかな
り大きな通気抵抗を火格子自体にもたせることによって，ごみ層厚
さの不均一やごみ質の相違があっても火床全体として燃焼速度が平
均化され，良好な焼却灰質が得られるように燃焼性の向上を図るこ
.産業廃棄物（industrial waste) 廃棄物処理法で20種
類に分類されており，そのうちの11種類が焼却可能な廃棄物であ
る．燃料的側面を示す発熱量で表現しても変動幅は非常に広く，性
状も液状，泥状，粉状，固形状およびこれらの混合状態まで多様で
ある．このように物性の異なる多種類の廃棄物を，単一の焼却炉で
燃やすことはできない．廃棄物を燃焼特性によって分類すると表
5･19 のように示すことができる．焼却炉形式も各種あるが，お
のおの焼却物に対する適性をもっているため，表5･19に示す廃棄
物の焼却特性に適合する形式のものを選定することが必要になる．
なお，表5･19のうち埋立処分が禁止されている油泥類と廃液類

(３)後燃焼段階（burnout stage）

雰囲気温度を保ちながら適度な少量空気を送り完全焼却する．
