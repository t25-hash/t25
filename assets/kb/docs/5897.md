# 5・3・4 熱回収技術

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・41 )、回転炉に供給されたごみは回転炉後半部および後燃焼部

ごみ燃焼室から出る燃焼ガスの温度は850～1000℃程度である
が，これを後段の排ガス処理設備あるいは通風設備に適した温度に
まで低減させる必要がある．このガス冷却の方法としては，廃熱ボ
イラ式（waste heat boiler type，間接冷却法），水噴射式（water
injection type，直接冷却法）およびこれを組合わせた半ボイラ式
などがある．また，廃熱ボイラ設備などでは高温燃焼ガスとの熱交
換により，蒸気，温水，予熱空気などを生成し，熱回収（heat
recovery）を行うものである．近年，省エネルギーおよび環境問
題の高まりから，ごみの燃焼熱を有効に回収して活用するために，
廃熱ボイラ式を採用する例が増加している．
a.廃熱ボイラ（waste heat boiler) 一般に焼却炉に設置さ
れるボイラは，炉形状に容易に適合可能な水管式がほとんどを占め
ているが，この水管式ボイラはボイラ水の循環方式によって，さら
に自然循環式と強制循環式の二つに分類することができる．自然循
環式はドラムと加熱管および降水管より構成され，加熱管中の気水
混合体と降水管中の水との密度差によってボイラ水の循環が行われ
る．これに対し，強制循環式はボイラ水の十分な循環力が得にくい
場合に循環ポンプを設け，強制的に循環させる方法であり，伝熱面
の配置が比較的自由に行える．しかし，ポンプを駆動するための動
力が必要となるため，現在ではほとんどのボイラが自然循環式とな
っている．
.廃熱ボイラ計画上の留意点 焼却炉のボイラは一般の産
業用ボイラと異なり，燃焼ガス中に飛灰が多く，また金属を腐食さ
せる原因となる塩素（Cl）や灰の融点を下げる亜鉛（Zn）および
鉛（Pb）などが多く含まれているため，運転上支障のないように
これらに対して十分注意する必要がある．
(１) 飛灰の対策
ごみ燃焼によって生じる飛灰（fly ash）は一般に1000℃付近で
軟化現象を呈し，ボイラの伝熱管などに付着して成長する．この飛
灰がボイラの管群などに付着してガス通路の閉そくを起こすと，炉
の運転が不能となり，大きな問題となる．これらの対策として，ボ
イラの燃焼室などの高温部はダストの付着しにくい放射伝熱面で形
成し，ここで燃焼ガスを適当な温度（700℃程度）まで冷却させ，
ダストの付着の少ない低温部を熱伝達率のよい接触伝熱面とするこ
とが多い．
また，スートブロワなどによる効果的な伝熱面のクリーニングを
行うことも重要である．
(２)腐食対策（corrosion countermeasure）
ごみの燃焼ガス中には塩素化合物（HClなど）や硫黄酸化物
（SO）などの腐食性ガスが含まれており，また腐食性成分を含む
ダストも多いことから，ボイラ鋼材の温度を適切な範囲に保つ必要
がある．図5･42 は鋼の表面温度と腐食速度との関係を示してい
る．金属面の温度が130～140℃の露点以下で低温腐食（low tem-
perature corrosion，dew- point corrosion）が起こり，また，
320℃以上の金属面で高温腐食（high temperature corrosion）が
起こる．
廃熱ボイラでは，節炭器管における低温腐食と過熱管における高
温腐食が代表的な例である．節炭器の低温腐食には給水加熱器また
は脱気器を設けることによって，節炭器入口の給水温度をガスの露
点温度以上に高めることが行われている．一方，過熱管の高温腐食
には過熱する蒸気温度の制限で対応し，過熱器出口の蒸気温度は
300℃以下に抑えられていた．しかし，現在では高効率廃棄物発電
の主旨から，過熱蒸気温度を400℃程度とする例が多くなってい
る．高温腐食対策として，過熱器の腐食防止に留意した構造および
配置，高温耐食性を有する適切な材料の採用，伝熱管の腐食しろを
見込んだ肉厚の選定などに配慮する必要がある．
上記のほか，燃焼室水冷壁管部に減耗現象が見られるが，これは
火炎に対面する一定範囲に限られており，還元域腐食（COガスに
よる）と見られている．これはごみ質の高カロリー化に伴う熱負荷
の増加，低空気比運転（低NO対策）などが原因と考えられ，必
要に応じ肉厚管の採用，不定形耐火物による水管表面保護などが望
まれる．
