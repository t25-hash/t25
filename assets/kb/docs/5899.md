# 5・3・6 灰処理技術

> **章**: 第5章 廃棄物処理・再資源化技術

a. 灰処理の概要焼却後の灰は最終処分場に埋立処分される。2000年度には国内の一般廃棄物の77%が焼却処理されて、その焼却残渣(焼却灰と飛灰)は570万tに達し、埋立処分されている。最終処分場は全国に2000箇所余りあるが、新規の確保は難しく、減少傾向にあり、その残容量も厳しい状況にある。そのため、焼却残渣の減容化が各自治体のごみ行政の大きな課題となっている。焼却残渣には有害重金属が微量含まれるため、地下水への溶出が起こらないように安定化する必要がある。特に焼却施設の排ガスから集じんされるばいじん(飛灰)には重金属が多く含まれるため、特別管理一般廃棄物として法で定められた方法による安定化処理を行う必要がある。また、「ダイオキシン類対策特別措置法」の制定により、重金属の溶出の観点に加え、ダイオキシン類の無害化の観点で灰処理方法が見直されている。焼却灰も規制値以上のダイオキシン類が含まれると、飛灰と同様に特別管理一般廃棄物として扱われるようになっている。飛灰安定化処理は、以前はセメント固化が中心であったが、埋立基準の溶出規制値の強化に伴い、薬剤処理が中心となった。さらに、溶融処理やセメント原料化ではダイオキシン類と重金属が同時に処理でき、処理物の有効利用が可能な方法であることから、近年、普及拡大している。また、焼却灰の溶融処理を行う場合には、焼却灰と飛灰を同時溶融する混合溶融処理が普及している。さらに、ごみ焼却を経ずに直接溶融するプロセスが開発され、近年、広く導入されるようになった。処理物(スラグやセメント)を再資源化物として有効利用し、最終処分量を最小化することが望まれることから、溶融スラグやエコセメントの規格化(JIS 化)が進められている。また、溶融炉排ガスから集じんされる溶融飛灰中の重金属を非鉄精錬の原料とする山元還元技術が実用化段階にある。b. 水砕スラグ設備水砕方式は、溶融スラグを冷却水槽に直接落下させ、急冷・固化、細粒化して製造する。また、溶融スラグを噴射水で急冷固化、細粒化後、冷却水槽に落下させ、製造する場合もある。溶融スラグと溶融メタルを一緒に水砕することができる。この場合、砂状のスラグが得られる。針状スラグの除去や、微細クラックによる強度低下を防ぐため、利用に際しては破砕機などで軽破砕・磨砕処理を行う。メタルを一緒に水砕する場合には、金属鉄によるさびの発生防止や密度変動防止のために、磁力選別処理を行う。水砕設備はスラグ出滓部、水砕部[水槽、水噴射装置、吹製とい(樋)など]、水砕水給排水ポンプ系、水砕水冷却塔、ブロー水・排水処理装置、スラグコンベヤ、および水砕蒸気換気装置から構成される。(2)
