# 5・3・10 微生物利用技術

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・41 )、回転炉に供給されたごみは回転炉後半部および後燃焼部

a.微生物利用技術一般 微生物のもつ生化学的反応を利用し
た廃棄物処理・再資源化は，コンポスト化等，酸素の存在下で進行
する好気性分解（aerobic decomposition）と，メタン発酵等，無
酸素下で進行する嫌気性分解（anaerobic decomposition）に大別
される．
好気性分解はコンポスト化処理に代表される方法である．発熱反
応で熱が発生するため原料の温度が60～70℃に上昇し，衛生的な
処理（病原菌，害虫，雑草種子等が死滅）が可能なうえ，温度上昇
が微生物の増殖速度を速めるため有機物の分解が促進される．最初
に糖類等の易分解性物質から分解が行われ，最終的に二酸化炭素や
アンモニアとして大気放散するガスのほか，残存成分としてフルボ
酸，フミン酸等の腐植質と窒素，リン等の有効成分が得られる．同
時に水分も蒸発するため，減容化されコンポスト（compost）とな
る．
嫌気性分解はバイオガスとしてエネルギーを回収することがで
き，酸性発酵期，酸性減退期，アルカリ性発酵期の3段階からな
る．酸性発酵期では，通性嫌気性細菌によりおもに炭水化物が分解
され，酢酸，プロピオン酸，酪酸といった低級脂肪酸が多量に生成
されるため，pHが5～6と急激に低下する．次の酸性減退期では，
タンパク質のような窒素系有機物の分解が起こり，アンモニア，ア
ミン，二酸化炭素，水素，メタン，窒素といったガスのほかに，微
量の硫化水素，メルカプタン等のガスも発生する．アルカリ性発酵
期は，特にメタン発酵と呼ばれ，分解，蓄積された低級脂肪酸，未
分解アミノ酸，アルコール，セルロース等のほとんどがメタン生成
菌により分解され，メタン，二酸化炭素，アンモニアに変換され
る．メタン生成菌の最適生育域には30～40℃（中温消化）と
55～65℃（高温消化）がある．
b.コンポスト化処理 一般的なコンポスト化処理フローを図
5･75に示す．
.前処理（破砕・選別・調質) 前処理段階では，ごみ
（の破砕)，もみがらや稲わらの破砕および鉄分の選別除去等のため
に破砕，選別機を使用する．袋収集されたごみの場合は破袋機を設
置する．
破砕，破袋，選別を同時に行う破砕選別機の例を図5･76に示す．
.発酵 粉砕され，異物を除去し水分を調整した原料を発
酵槽に投入する．発酵反応は急激な熱と炭酸ガスの生成を伴う一次
発酵（primary fermentation）と，緩やかに発酵が進行する二次
発酵（secondary fermentation）に分かれるが，一次発酵は5～10
日，二次発酵では通気しながら堆積放置し，ときどき切返しを行い
熟成させる場合が多い．
発酵プロセスにはさまざまな微生物が関与する．一次発酵期で
は，適当な水分条件と通気により好気性発酵が始まることによって
温度が急上昇し，有機物は好熱菌の同化，異化作用により二酸化炭
素，アンモニア，水に分解される．
