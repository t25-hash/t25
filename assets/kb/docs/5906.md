# 5・3・11 個別産業廃棄物の有効利用技術および事例

> **章**: 第5章 廃棄物処理・再資源化技術

産業廃棄物からの有用物回収は一定の方式があるわけではなく、ケースバイケースの計画が多く、そのプロセスも多種多様である。以下に紹介するのは、おのおのの産業廃棄物の有効利用技術およびその事例である。a. 燃えがら(cinder,combustion residue)一般には最終処分が行われているが、発電用石炭ボイラのボトムアッシュは、路盤材、土壌改良材、下水処理剤、融雪促進剤などに利用されている。そのほか、b項の汚泥の建設材料化またはn項の鉱滓に準じたケースが考えられる。有効利用技術については、溶融や焼成といった熱処理および物理的選別法(ふるい、風力選別、磁力選別、非鉄分離など)がある。b. 廃酸(waste acid)廃酸には無機酸と有機酸がある。無機酸の多くは中和・凝集処理され、固液分離後に放流されているが、冷却法、蒸発法、イオン交換膜法(拡散透析法)、熱分解法などにより硫酸、硝酸、フッ酸などの回収が図られ、再生品として利用される場合がある。例えば、鋼板の酸洗に使用された廃希硫酸は、真空蒸発缶で濃縮し、沈殿した硫酸鉄を飼料添加剤として、また濃縮された硫酸を酸洗に再利用している。また、廃酸に含まれるクロム、ニッケル、アルミナなどを回収する事例がある。図5・91にアルミニウムエッチング廃酸からの高純度アルミナおよび塩酸回収プロセスを示す。アルコール発酵廃液などの有機酸は、海洋投棄の全面禁止を受け、メタン発酵や肥料化による利用が図られている。e.
