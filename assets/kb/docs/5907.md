# 5・4・1 概要

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・4 埋立処分技術

쓕廃棄物最終処分場性能指針」において廃棄物最終処分場を「生活環境の保全上支障が生じない方法で、廃棄物を適切に貯留し、かつ生物的、物理的、化学的に安定な状態にすることができる埋立地及び関連附帯設備を併せた総体の施設という」と明示してあるように、いかなる廃棄物、すなわち産業廃棄物、一般廃棄物を問わず、最終処分場(final disposal site)では廃棄物の安定化と無害化に努めなければならない。a. 関係法令最終処分場を計画し実施するに当たって、廃棄物の処理及び清掃に関する法律などの図5・103に掲げる関係法令を遵守しなければならない。b. 分類最終処分場は産業廃棄物の種類、立地条件、および構造によって下記のように分類できる。一般廃棄物に指定されるものの埋立てについては、産業廃棄物埋立ての管理型処分場に該当する。管理型処分場(controlled landfill site)産業廃棄物の種類安定型処分場(inert waste landfill site)遮断型処分場(strictly controlled landfill site)(法令第7条14号による。図5・104~5・106)立地条件陸上埋立て水面埋立て内水面埋立て海面埋立て改良型嫌気性衛生埋立(improved anaerobic sanitary landfill、埋立層内への通気を阻害する構造)-好気性埋立(aerobic landfill、埋立層内へ強制通気構造する構造)準好気性埋立(semi-aerobic landfill、埋立層内へ自然通気する構造)c. 最終処分場の施設構成最終処分場は図5・107に示すような構成からなる。
