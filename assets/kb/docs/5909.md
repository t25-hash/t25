# 5・4・3 主要施設

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・4 埋立処分技術

最終処分場の主要施設は,貯留構造物,地下水集排水施設(underground water drainage facility),遮水工,雨水集排水施設(run-offdrainage facility),浸出水集排水施設(leachate collec-tion facility),浸出水処理施設,埋立ガス処理施設および被覆施設である.a. 貯留構造物(solid waste retaining structure)貯留構造物は,廃棄物層の流出や崩壊を防ぎ,埋め立てられた廃棄物を安全に貯留・保管するために設けられる.貯留構造物は,廃棄物を貯留することが目的であって貯水が目的ではないが,予想を上回る異常降水時には埋立地内が一時的に滞水状態になることも予想される.また,設計降水の範囲内であっても,浸出水処理設備や浸出水調整設備の維持管理・補修などで浸出水を埋立地内に貯留することも想定されるため,貯留構造物には安全な貯水機能が併せて要求されることになる.b. 遮水工(seepage control work). 遮水工の基本的な考え方廃棄物最終処分場において,埋立地への降水は廃棄物層に浸み込み浸出水となり,やがて底部に敷設された浸出水集排水施設を通じて浸出水処理施設に集まる.この浸出水による地下水質汚染を防止する目的で遮水工を設置する.この目的を達成するためには,次のような各種機能が考えられる.(1) 遮水機能:浸出水による地下水汚染を防止する機能.(2) 損傷防止機能:基礎地盤の凹凸や廃棄物中の異物による損傷を防止する機能.(3) 漏水通過時間確保機能,汚染軽減機能:万一の地下水汚染に対し,その程度を軽減させるための機能.図5・108 廃棄物の分解過程と浸出水(東京都清掃局,1976年)遮水工表面遮水工鉛直遮水工図5・109遮断工の分類(4) 浸出水質の予測浸出水質は,一般に埋立初期は高濃度であるが,経時的に低濃度となる.また,埋立初期は生物処理の容易な汚水であるが,徐々に生物処理の困難な汚水へと変化していく.したがって,浸出水処理設備の設計対象水質は,初期代表水質を採用するなど水処理方式選定に配慮を要するが,埋立後期の生物処理の困難な汚水には,低負荷で対処したり物理化学的処理主体の運転体制に切り換えるなど,維持管理面での対応が重要である.(1) 可燃性廃棄物主体の浸出水質の経年変化BOD(biochemical oxygen demand,生物化学的酸素要求量)は,埋立開始後1~2年で急激に上昇するが,その後すぐに低下し,数年後には100mg/Lを下回り,そのまま直線的に低下する傾向を示す.COD(chemical oxygen demand,化学的酸素要求量)はBODよりも上昇・下降が緩やかであり,100mg/Lレベルに低下するのに約10年の期間が必要である.NH-N は,初期はBOD に遅れて次第に高くなっていくが,年数を経過するに従い徐々に低下する傾向を示す.(2) 不燃性廃棄物主体の浸出水質の経年変化BOD は初期には高い値を示すが,可燃性廃棄物主体に比べ1けた低く数百mg/L程度である.数年で急激に低下し埋立終了後は非常に低い値となる.廃棄物中に含まれる有機物質によって大きなばらつきを示すことがある.COD はBOD に比べ,高くなってもほぼ2分の1の値程度である.年数が経過すると,50mg/L程度でほぼ横ばいとなっている.NH-N は30mg/L程度でほぼ一定の値で推移している.(3) 焼却残渣と不燃性廃棄物の浸出水質の経年変化BOD は,ほぼ200~300mg/Lをピークに,埋立状態によって変動している.COD は,BOD とほぼ同じ傾向を示すが,値としてはBOD より全体的に低い.ただし,難分解性のものはそのまま残り,50mg/L程度で一定の値を示す.10―186環境システムNH-N は,埋立当初に高い傾向にあるが,年数が経過すると50~100mg/Lに低下する.. 浸出水調整設備(leachate control equipment)浸出水処理設備の能力は一定であるが,浸出水量は主として降水により変動する.したがって,その中間に差分量を調整するための設備を設けることが浸出水処理設備の安定稼働のために必要となる.また,浸出水質の急激な変動を緩和し,必要に応じて前ばっ気のような前処理を行う機能を併せ持つ浸出水調整設備が必要である.. 環境保全機能廃棄物最終処分場における環境影響因子は,周辺環境への影響が大きい浸出水,廃棄物の存在,粉じん,臭気および埋立ガスなどである.被覆施設を設けることにより閉鎖空間内での管理と内部環境の制御とが可能となる.(b) 保管・処理機能被覆施設を設けた最終処分場に投入される廃棄物は埋立前に分別,破砕などの前処理により資源化物の回収を極力行うとともに,溶融スラグに代表されるように,需要があればすぐに利用できる材料を保管する機能を有する.廃棄物の処理機能としては,投入された廃棄物の早期安定化と無害化を促進するために,埋め立てた廃棄物層への散水,吸排気などによる制御を行う.また,廃棄物が再資源化物として利用される段階になったときや何らかの不具合で取出す必要が生じた場合のために,被覆施設を設けた最終処分場では,分別保管と搬出との機能をもたせることが望ましい.(c) 地域還元最終処分場では,一般に廃棄物搬入時の運搬車両の通行,騒音の発生,投入時の粉じんおよび臭気の発生などによる環境悪化を全くなくすことは難しい.このため,これらの見返り的な地域還元機能として,被覆施設を設けた最終処分場では人工地盤などによる跡地の先行利用など,その特性を生かした施設整備計画を行い,地域還元振興および環境創造などを考慮した町づくりに寄与することもできる.10―187図5・114 ダイオキシン類分解を採用した浸出水処理フローの例(2) 外部要因の制御被覆施設をもつ最終処分場においては,次のような外部要因の制御が可能となる.(a) 雨水浸出水の発生原因である雨水については,通常の最終処分場にあっては,台風などによる大雨時にも対応できる浸出水処理設備および浸出水調整設備が必要である.このため,施設規模の大型化や多量の浸出水の発生,貯留による漏水リスクへの対策が必要である.しかし,被覆施設を設けることにより雨水を完全に排除し,代わりに人工的に制御された必要量の散水を行うことで,これらのリスクを軽減することができる.(b) 風,積雪風の強い地域,積雪地域では,通常の最終処分場にあっては埋立作業が容易ではなく,また適切な維持管理も困難を有する.このような地域では被覆施設を設けることにより,風および積雪の影響を受けることなく埋立作業が可能となり,周辺環境に対しても悪影響を及ぼすことが少なく,運営面での利点も多い.(3) 内部要因の制御被覆施設をもつ最終処分場においては,廃棄物の埋立地から発生する次のような内部要因の制御が可能となる.(a) 廃棄物の散乱被覆施設をもつ最終処分場では,屋根(または床,人工地盤)と側面の壁によって外部環境と隔離された閉鎖空間になるので,廃棄物の周辺環境への散乱は発生しない.(b) 臭気,ガス閉鎖空間となるため,廃棄物から発生する臭気やガスによる周辺環境への影響は小さいが,内部の作業環境の保全が重要な問題である.臭気についてはこれまでの知見では特にアンモニアが問題とされ,焼却灰と水が接触した1~2日間に発生量が多いとされている.ガスの発生に関しては,爆発や酸素欠乏が懸念されるが,いずれも送風や換気により対応が可能である.(c) 温度被覆材料は外気温の影響を受け,密閉状態となるので,夏期の内部温度は40°Cを超す高温となる場合がある.場合によっては,50°C以上になることも考えられる.このため,埋立てなどの作業中には換気対策により内部温度を低下させる必要がある.. 被覆施設を設けた最終処分場の考え方(1) 施設の立地場所最終処分場の設置による周辺住民の環境影響への不安が根強いことから,i項で前述したように被覆施設を設けた最終処分場は各種の環境影響を制御できるものの一つとして有力であるとともに,山間,谷沢を利用した通常の最終処分場のような立地場所の制限を受けにくい.(2) 施設の構造被覆施設を設けた最終処分場は通常の最終処分場と同様に安全性10―188環境システムと信頼性のために制御ができる構造であるとともに,地域に受け入れられる施設構造が要求される.したがって,住民参加のもとで,周辺環境に適応した構造にすることが必要であるが,一方において被覆施設を設けることにより維持管理費がかさみ,住民の負担が大きくならないような構造にすることも忘れてはならない.(3) 施設管理被覆施設を設けた最終処分場の場合も,通常の最終処分場と同様に,廃棄物が無害化するまでは地下水などを汚染する可能性が残っている.また,最終処分場は管理期間が長くなるので,管理費用を最小化できる最終処分場とすることが肝心である.被覆施設の構造によっては,跡地の先行利用が可能であり,駐車場,倉庫および貸農園にするなどの施設利用によって収入を見込み,管理費を低減するなどの考え方も有効である.. 被覆施設の種類被覆施設は,埋立作業に必要な空間を確保して,埋立地を覆う建築構造物である.地上に突出する形状となるため,周辺環境との調和を図るように配慮することが望ましい.被覆施設は,固定式として常時外界と遮断する方式と,開閉式として必要に応じて降雨および日射の浸入,大気の移動などを許して環境を制御する方式とがある.また,被覆施設の設置方法として,埋立地全体を一つの構造物で覆うタイプと,埋立地を分割して一部の区画のみを覆い,埋立ての進行に合わせて被覆施設を順次移設・移動していくタイプとがある.
