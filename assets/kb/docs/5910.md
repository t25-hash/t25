# 5・4・4 最終処分場の管理

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・4 埋立処分技術

a. 埋立作業(landfill works)廃棄物が最終処分場へ搬入されてから破砕・転圧を経て埋立てられるまでの作業、また、即日覆土や埋立ての進行に伴う法面造成などの付帯的な作業が相互に関連して計画的に進められる。埋立作業は、埋立工法、埋立順序などを勘案して、適切に行う。b. 埋立工法(landfill operation)埋立工法は、陸上埋立てと水面(海面)埋立てに区分され、地形や立地条件に応じた方法で行われる。. 埋立方式(1) サンドイッチ方式(sandwich method)サンドイッチ方式は、図5・116に示すように、廃棄物を水平に敷きならし、廃棄物層と覆土層を交互に積み重ねるもので、狭い山間などの埋立地で用いられる。しかし、埋立面積が広いと、覆土(cover soil)をするために必要な厚さの廃棄物層を確保するには1日のまきだし面積を小さくせざるを得なくなり、結果として廃棄物層に法面が生じる。法面にも覆土が必要となるので、この場合には実質上、次に述べるセル方式となる。(2) セル方式(cell method)セル方式は、図5・117に示すように、埋立廃棄物および法面に覆土を施し、セル状に仕上げるもので、現在最も多く用いられている図5・116 サンドイッチ方式(全国都市清掃会議、1989年)図5・117 セル方式(全国都市清掃会議、1989年)方式である。一つのセルの大きさは、通常1日の埋立処分量によって決まる。埋立層がセルごとに一応独立するので、火災の発生および拡大の防止、廃棄物の飛散防止、悪臭および衛生害虫などの発生を防止する効果がある。一方、セルは埋立層内の埋立ガスや水の移動を阻害するので、浸出水集排水施設や埋立ガス処理施設の設置に際しては、この点に十分配慮した工夫が必要である。. 埋立順序陸上の最終処分場は、山間や開析谷に設置されることが多く、一般に埋立地が傾斜している。このような埋立地における埋立ての順序は、埋立地の地形、降雨、周辺環境、一般道路からのアクセス、廃棄物量、浸出水および雨水の処理などを考慮して定める必要がある。埋立順序は、(1)上流から埋め立てる方法と、(2)下流側から埋め立てる方法に大別できるが、次に示す理由により、下流側から埋め立てる方法が望ましい。図5・118に両者の概念図を示す。(1) 上流からの埋立ては、下流側の未埋立区域内の雨水排除が難しく、浸出水の量が多くなりやすい。また地震、降雨などにより、廃棄物の滑り崩落が起こりやすい。(2) 遮水シートが底面に敷設されている場合、上流からの埋立ては、遮水シートを下流側へ押しやるような形になり遮水シートの破損を招きかねない。(3) 下流側からの埋立ては、順次埋立地を拡張していくとき、遮水シート張り、法面造成などの追加施工が容易である。. 分割埋立て(separate landfill method)異種類の廃棄物を種類や形状などに応じて別々に分割して埋め立てる方法を分割埋立てという。埋立処分地の早期安定化や廃棄物のリサイクル促進の観点から、分割埋立てによって埋立地や埋立跡地の管理が容易になる場合が多い。c. 管理施設と関連施設最終処分場を適切に管理するために、必要に応じて搬入管理施設、モニタリング施設、管理棟(記録保管、閲覧機能を含む)、管理道路、そのほかの施設を設けなければならない。管理施設は、搬入管理施設(搬入廃棄物質の分析を含む)、環境モニタリング施設などの計測・分析施設、最終処分場を管理運営するための機材などの保管施設、管理道路、洗車場および管理要員の執務・厚生施設ならびに各種記録の保管・閲覧施設を適切に組合わせて構成される。. 搬入管理施設(transport control facility)搬入管(a) 上流側から埋立て(b) 下流側から埋立て図5・118 埋立順序の概念(全国都市清掃会議、1989年)理施設は、最終処分場が受け入れる廃棄物の量と質を適切に管理し、最終処分場に持ち込まれるリスクをその入口で管制する機能をもつものである。(1)計量設備、(2)廃棄物質などの分析設備などで構成されている。. 管理棟管理棟は、施設や作業を統合管理するために設置され、管理事務室、試験分析室のほか、必要に応じて作業員控室、シャワー室、更衣室、給湯室、食堂、会議室などからなっている。さらに、住民の見学会などに供する集会室を必要とするところも多くなっている。また、換気装置、通信設備、電話設備、冷暖房設備なども必要な室に設置するよう配慮すべきである。どのような設備・室を必要とするかは最終処分場の規模と管理体系を十分検討するとともに、最終処分場の管理人員および職員の望ましい作業環境も考慮して計画する必要がある。. 飛散防止設備(scattering prevention device)飛散防止の目的は、強風や鳥類などによって飛散・流出し、最終処分場周辺の環境を汚染することの防止である。状況により、立入り防止および目隠しの目的を兼ねる場合もある。廃棄物が飛散・流出する可能性のある経路としては、埋立作業時および日常作業終了後の廃棄物埋立地からの飛散と、運搬作業車両からの流出が考えられる。飛散防止設備は、対策の目的により防止網などの使い分けを行う(図5・119)。散乱対策設備として設置する散乱防止フェンス(高さ1.8~7m程度のものがある)は、必要な耐風性を備えたものでなければならない。忍び返しを埋立地側に倒すことで効果を上げることができるといわれている。散乱対策設備の効果の及ぶ範囲は、網の位置から水平方向に対して網の高さの3~4倍までといわれているが、埋立地の地形、規模および気象条件に大きく左右される。. モニタリング施設(monitoring facility)モニタリング施設は、埋立層モニタリングと環境モニタリングに大別することができる。埋立層のモニタリング方法としては以下のものがある。なお、これらの項目の具体的なモニタリング方法は、「廃棄物最終処分場安定化監視マニュアル」などを参照することができる。(1) 埋立廃棄物の分析(2) 内部温度の調査(3) 沈下量の測定(4) 埋立ガスの分析(5) 浸出水の分析地下水モニタリングについては、基準省令では2箇所以上の観測井または地下水集排水施設により採取した水の水質検査を行うこととなっており、また、検査項目、検査頻度についても設定されているので、地下水モニタリング計画は少なくとも基準省令の維持管理基準に基づき計画することが必要である(表5・35)。d. 跡地利用(utilization after landfill period)最終処分場跡地の有効利用を行うことは、その地域の福祉向上に役立ったり、その地域の活性化・発展を助ける、いわゆる地域還元をもたらすばかりでなく、次の最終処分場の建設に対する住民の理解を得て持続的に最終処分場の立地を推進するために大切なことである。散乱対策設備メッシュフェンスネットフェンス仮設設備暴風対策設備飛散防止設備簡易フェンス(移動式)目隠しフェンスエキスパンドフェンス防風フェンス防風林図5・119飛散防止設備の形式(例)10―189. 跡地利用に関わる基本的留意事項最終処分場跡地の利用に関わる個々の利用行為が経時的に跡地内部の土質力学的、生物化学的組成に影響を及ぼすことが、表5・36のような過去の実績、事例(有機性ごみ主体の最終処分場)により知られており、特に埋立地の廃止前の跡地利用にあっては、あらかじめその影響に留意する必要がある。. 地域還元施設の状況処分場の跡地利用は当該地区に便益をもたらす例もある。最終処分場跡地を地域還元施設として利用する場合は、埋立目的のみで供用された跡地に比べて、利用水準や周辺地との土地利用の適合性で優れているようである。. 地域還元、地域活性化の課題最終処分場の跡地利用により、地域還元を図るための課題として以下の点が挙げられる。(1) 掘削による廃棄物層の露出や廃棄物埋立層の安定化履歴を考慮した土地利用状態を遵守しなければならない利用上の制約。(2) 市街地から遠いため、交通などのアクセスの確保。(3) 最終処分場のもつ負のイメージの払拭。. 平野部における跡地利用公共性の高い施設を建設することによる地域還元への期待度は高いものと予想できるが、最終処分場跡地での施設の適合性では建物重量と遮水構造への影響が最も大きな課題であると考えられる。また、最終処分場の広い敷地利用特性を生かす施設としては、体育館や養護施設、さらに幼稚園、小学校、図書館といった文教施設や研究所などの、平屋建てで広い敷地にゆったりとした施設構成の求められるものが適すると考えられる。. reclamation)近年、最終処分場からの浸出水漏洩による地下水の汚染問題などにより、新規の最終処分場建設が困難な状況になっている。したがって既存の埋立地の延命化が重要となっており、埋立地再生技術の実用化が進められている。環境負荷低減材料や製品を考える条件としては、製造中に使用する資源とエネルギーや中高度有害化物質、使用中の日用品の利便性とそれに含まれる低有害化学物質、および廃棄するときの分解性・リユース率・リサイクル率・最終廃棄量とその化学物質をLCA(life cycle assessment)的にとらえる必要がある。プラントなどの長寿命機器では使用時の環境コストが最も高く、消費財などの短寿命機器は廃棄時の環境コストが高い。これに加えて、重金属や化学物質の環境中の循環・濃縮による環境リスクは、製品リサイクルに重ねて製品設計段階から積極的に配慮し、影響を公開する時代になっている。
