# 5・5・1 製品に使用される化学物質

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・5 環境負荷低減材料および製品開発

人類の利便性を追求するために世界で開発され続けた化学物質は,化学物質登録機関CAS(Chemical Abstracts Service)に2000年末で約2800万種類が登録されており,種類と生産量は増加傾向にある.化学物質の総合管理の代表的な法規制は下記のものである.・USA(1986) 有害化学物質排出インベントリー(TRI:Toxics Release Inventory):対象物質643・EU(2000) ヨーロッパ汚染物質排出登録制度(EPER : theEuropean pollutant emission register):対象物質50・イギリス(1998) 汚染目録(PI: Pollution Inventory):対象物質185・日本(1999) 化学物質排出把握管理促進法(PRTR):対象物質357流通商品が10万種類といわれる化学産業では,自主活動としてレスポンシブルケア活動(化学物質を製造・取扱う事業者が自己決定・自己責任の原則に基づき,化学物質の開発から廃棄に至る全過程で環境・安全・健康面対策の実施・改善を図る自主管理活動.結果の公表と社会との対話も進める)を実施している.世界産業である自動車産業でも,各国の法律や規定に適合したグリーン部品調達を進めている.
