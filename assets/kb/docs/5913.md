# 5・5・3 環境負荷低減製品開発

> **章**: 第5章 廃棄物処理・再資源化技術
> **節**: 5・5 環境負荷低減材料および製品開発

EU の規制有害物質の主用途は、電機・電子部品に一般的に使用されている。・カドミウム:リレー接点、プラスチック安定剤・鉛:はんだ材料、プラスチック安定剤・水銀:スイッチ、センサ・六価クロム:防錆、耐食表面処理・PBB,PBDE:プリント基板、外装部品上記中で代替が難しいのは電子デバイス接合技術に用いられる鉛フリーはんだであり、各社が導電性接着剤、ペースト、新合金、基板やはんだ付け方法の開発を通じ、実用化されているが、従来型鉛はんだの性能まで改良が求められている。環境配慮設計手法に基づく製品開発は、リサイクル、省エネルギー、素材や製造段階での化学物質管理、鉛はんだやフロン不使用などに設計段階から取り組むことで、リサイクル比率向上、解体容易、省電力、梱包材削減などの成果を生み、環境負荷低減と生産・リサイクルコストの削減に結びつくとともに、消費者への製品PRにもつながる。また、製品・サービスが生み出す質的価値や経済効果の拡大とその生産や使用段階での環境負荷の低減を対比させる指標が環境効率であり、1990年にドイツのワイツゼッカーが提唱したファクター4が代表例である。地球規模での平均生活水準を2倍に向上する一方、資源消費を2分の1に抑えるような経済社会を構築すること、つまり経済社会活動全体で環境効率(ファクタ)を4倍にする構想である。テレビ、冷蔵庫、エアコン、複写機、プリンタなどの家電・OA(office automation)の新製品の一指標になりつつある。市場メカニズムを活用した環境改善認証として、環境ラベルがある。環境ラベルにはタイプI~IIIがあり、タイプIは第三者機関が認定したシンボルマークを表し、タイプIIは事業者が自主的に環境配慮を宣言するものである。タイプIIIは定量的環境情報開示型といわれ、LCA 手法により生産から使用・廃棄までのCO排出量などの環境負荷を定量的に表示するもので、産業環境管理協会のエコリーフ環境ラベルが運用を開始している。そのほかにグリーン購入の上位概念であるサステイナブル購入の評価基準としてエコプレミアムを商品購入の枠組にする活動もある。環境負荷低減材料および製品開発においては、規制が技術進歩を促し、商品の情報開示が消費者や社会の認知・認証を受けることが技術的な常識となる。技術開発には地球環境の目線と消費者の目線が必要とされ、環境リスク対策が重要な商品化の要素となる。文献(1)
