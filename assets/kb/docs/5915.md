# 6・2・2 低温プラズマによる大気浄化技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・2 大気・室内環境保全に関する先端研究

a. 低温プラズマの特徴と分解機構。 低温プラズマ発生方式プラズマは、正電気を帯びた粒子と負電気を帯びた電子とがほぼ同じ密度で、ほぼ電気的中性を保って分布している粒子集団と定義されている。低温プラズマ(non-thermal plasma)では、中性粒子が多く存在し、弱電離プラズマと呼ばれる状態になっている。低温プラズマの発生法には、沿面放電、バリア放電(無声放電)、パックトベッド式放電(packedbed style discharge)、パルス放電(pulse electrical discharge)、キャピラリーチューブ式放電がある(表6・2)。沿面放電は、セラミックス中に埋め込まれた接地電極と内壁の棒状電極に数kHz の高周波電圧を印加、放電させることにより円筒内面にプラズマを発生させる。バリア放電は適当な誘電体(ガラスなど)を電極間の片側、あるいは両側に配した構造をもち、その両電極間に高周波電圧を加え、電極間の空げきにプラズマを発生させる。図6・4(a)はパックトベッド放電式プラズマ反応装置の概略であり、強誘電体ペレットを電極間に充てんし、交流高電圧の印加によりペレット間の空げき全体をプラズマ化させる。図6・4(b)はパルス放電式プラズマ反応装置であり、金属ワイヤに対し同軸に金属チューブが配列されている。反応器の電極間に、立上り時間数十ns、持続時間1 クラスのエネルギーで十分分解できる範囲にあることがわかる。プラズマの素過程として、電離(高速電子が分子や原子に衝突してプラスイオンが生成)、励起(高速電子との衝突で分子や原子が内部エネルギーの高い状態になる)、解離(高速電子が化学結合を切断し、中性粒子であるラジカルを生成)および電子付着(マイナスイオンの生成)がある。解離は電子が直接関わる分解反応であり、励起・電離・付着は必ずしも分解が起きているとはいえないが、これらもさらにさまざまな反応過程を経て、VOCや臭気を分解する二次的作用がある。トルエンなどはプラズマ励起した酸素の後段に導入するだけでも容易に分解するという事実や、ベンゼンの分解反応では酸素存在下表6・3
