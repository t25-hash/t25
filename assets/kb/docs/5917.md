# 6・3・2 建設廃棄物処理技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・3 廃棄物処理・再資源化に関する先端技術

a. 概要建設廃棄物とは,建物の建築工事と道路建設や港湾工事などに代表される土木工事のそれぞれから発生する廃棄物を指す.建設廃棄物の発生量は,1999年度を例にとると,産業廃棄物の全排出量の約2割,最終処分量においては約3割に当たる.すなわち,建設廃棄物は産業廃棄物の大きな部分を占めており,発生抑制と再資源化が強く求められていることがわかる.こうした状況を踏まえて,2000年5月には「建設工事に係る資材の再資源化等に関する法律(建設リサイクル法)」が公布され,2002年5月から施行されている.この法律は,特定の建設資材を用いた一定規模以上の建築物などの新築工事や解体工事に対して,分別排出や分別解体,再資源化等を義務付けることにより,資材の有効利用を進めることを目的としている.具体的には,建設廃棄物の大半を占める,コンクリート,アスファルトコンクリート,木材の3種類について再資源化を義務付けている.ここでは,建設廃棄物の代表的な発生源,種類,処理技術を紹介する.なお,建設工事や土木工事では廃棄物には分類されない発生物(そのまま原材料となる再生資源や建設発生土など)もあり,これらと廃棄物をまとめて建設副産物と呼ぶ.b. 廃棄物の発生源と種類. 建築工事建築物としては,木造建築,RC(rein-forced concrete)・SRC(steel framed reinforced concrete)建築,コンクリート構造物があり,発生する廃棄物の種類に相違がある.また,建築工事は一般に,基礎・掘削工事,軀体工事,設備工事,内装・仕上工事,外構工事の各工程からなっており,各工程で発生する廃棄物にも違いがある.ここでは工程別に発生するおもな廃棄物を示す.(1) 基礎・掘削工事建設汚泥,金属くず,伐採材の発生がある.量的には建設汚泥が多い.建設汚泥とは,基本的に土に水を加えたもので廃棄物であるが,掘削した土砂そのものなどは廃棄物ではなく,建設発生土と呼ばれている.汚泥と発生土の識別は,発生時点での含水率,土の粒径,泥の状態の3点でなされる.工法の関係で発生時点で水が添加された状態の泥は汚泥となる.(2) 軀体工事建築物の種類によって異なるが,おもに木くず,コンクリート,金属くずなどが排出される.木くずとしては,型枠廃材や新築端材がある.コンクリートとしては,工法の関係で発生するはつりや捨てコンクリート,生コンクリートのポンプ車などの装置に残留する余分なコンクリート,配管スリーブのためのコア抜き,強度試験のテストピースなどがある.金属くずとしては,鉄骨や鉄筋に起因するものなどがある.(3) 設備工事おもに金属くず,プラスチックなどが発生する.配管工事や配線工事などに起因する金属配管やプラスチック配管(硬質塩化ビニル),廃電線などである.(4) 内装・仕上工事さまざまな廃棄物が発生する.梱包(こんぽう)材,ボード類,断熱材,内装材などに起因するプラスチック材,木材,石こうボード,軽量気泡コンクリート(ALC: autoclaved lightweight con-crete)材,ロックウール,グラスウールなどである.建具取付けや塗装に起因する金属くず,木くず,廃有機溶剤なども発生する.また,これらの材料の梱包材や包装材に起因する衝撃吸収材のプラスチック,木枠,包装紙などもある.しかもこの工程では分別が困難なものも多く,これらが混合された廃棄物は混合廃棄物(ミンチ)と呼ばれる.(5) 外構工事ほか屋根材の陶器類やスレート,外壁のサイディング材などの端材,それらの材料の梱包材なども廃棄物として発生する.塀や造園などの工事からは,コンクリート,木くず,せん定枝などが発生する.. 土木工事シールド工法に伴う建設汚泥,各種工作物の建設工事に伴うコンクリート,道路工事からのアスファルトコンクリート,造成工事に伴う木くず(伐採・除根)等がおもなものである.. 解体工事以上のi項とii項では新しい建設工事に伴い発生する廃棄物を取り上げた.そして,これらの建設物は補修や改造,解体されるときにはすべて廃棄物となる.特に解体時には,建設した構造物そのものが廃棄物となるため,構造物本体に起因する廃棄物が中心となる.木造建築では廃木材,RC・SRC建築ではコンクリートがら,鉄骨くず,木くず,コンクリート構造物ではコンクリートがら,鉄骨くずが主体となる.また,これらの構造物に伴う設備類も廃棄物となる.10―19 9. 解体工事に伴う有害物解体工事において,着工前から存在する法律に基づく処理が必要な廃棄物もあり,これらは着工前に工事発注者が責任をもって実施する必要がある.PCB を使用したトランス,コンデンサ,蛍光灯安定器は,PCBを処理委託できる施設が完成するまで発注者が保管する義務がある[「PCB 廃棄物特別措置法」(2001年)].また,家庭用のテレビ,冷蔵庫,エアコン,洗濯機は家電リサイクル法に基づいて,発注者が処理する義務がある.業務用エアコンや冷蔵庫・冷凍庫の冷媒フロンは,「フロン回収破壊法」に基づいて処理されなければならず,業務用機器の撤去に関しては発注者の責任である.その他の特別管理産業廃棄物や高圧ガスに相当するものなども,それぞれの法律に基づいて処理を行わなければならない.c. 処理技術. 汚泥建設汚泥は,脱水操作で土として再利用できる可能性がある.脱水の困難な汚泥は,その原因が土粒子表面の帯電による土粒子の凝集妨害と水分分離の困難性にあるので,この帯電を中和すれば造粒と水分の分離が可能となり,良質土とすることができる.この改良方法としては凝集剤の添加が考えられ,凝集効果をもつアルカリ性の石灰質の添加による土質改良プロセスなどが実用化されている.. 廃木材(1) 伐採材堆肥化が可能である.しかし,産業廃棄物処理業の許可を得ている処理施設が少なく,実際に処理されている量は少ない.(2) 型枠廃材燃料チップ化,パーティクルボードの原料化が可能である.合板材であることとモルタルやくぎなどの付着物があるために,パルプ原料としてのチップ化は難しい.(3) 解体材分別して解体された木材の一部は,丁寧な補修により再利用可能である.また,パルプ原料用チップ化,ボード材料用チップ化,燃料用チップ化も可能である.しかし,一部の合板材やパーティクルボード,防腐剤処理された木材などは,それぞれに対応する処理技術はあるものの,コスト面の問題から材料としての再利用は困難になっているのが実情である.(4) 新築端材建設リサイクル法の施行とともに分別・再資源化が義務付けられたこともあって,ボード材料用チップ化や燃料用チップ化の割合が増加している.. コンクリート塊建設廃棄物のうち,最大の発生量を占める.しかし,その一方で発生量の95%以上が再資源化されている.分別されたコンクリートはほとんどが再生砕石として利用されている.再生砕石を作るプロセスは,構造物の破壊,コンクリート塊の破砕,分級・分別である.構造物の破壊の方法としては,油圧式の圧砕機による破壊,鉄球などによる打撃,火薬による爆発,削岩機などによる強い加圧と衝撃,高周波プラズマによる破壊などがある.現状では騒音と振動が比較的小さい油圧式の圧砕機による破壊が一般的である.得られたコンクリートがらは粒径を小さく,そしてなるべくそろえるために破砕される.破砕の方法としては,押しつぶす方法で破砕するジョークラッシャ,ハンマでたたく原理のハンマクラッシャ,回転する二軸のローラの間に挟み込むようにして破砕するロールクラッシャなどがある.被破砕物が鉄筋などの鉄金属を含む可能性がある場合には,磁力選別機で分離する.さらにふるい分けの原理で粒径ごとに分級され,0~40mmはクラッシャラン,0~5mmは再生砂,5~40mmは再生砕石などとして,路盤材や埋戻し材として再利用される.. アスファルトコンクリート道路の補修や上下水道,ガスなどの工事で発生するものが主体であり,単独で発生し異物の混入が比較的少ない廃棄物である.道路面などからはがされたアスファルトがら(表層と基層)は,約90%の砕石を含むアスファルト10―200環境システム混合物である.処理としては,このアスファルトがら(アスファルト混合物)をふるい分けして,粒径が40mm以下のものは再生路盤材として再利用され,大きいものは破砕してアスファルト混合物の骨材として再利用される.. 混合廃棄物ほとんどが建築工事(新築,解体)で発生する.この処理自体は極めて難しいことから,ほとんど処分(埋立てなど)されてしまうので,この形態の廃棄物の発生を抑制する努力がなされている.具体的には,ボード類の現場でのカットを減らすためのプレカット,資・機材の全面梱包をやめて局部的な養生のみにする省梱包・無梱包化,多くの建設資材メーカが「廃棄物処理法」に規定されている広域再生利用環境大臣指定制度に基づく指定を受け,自社製品端材を工場に引き取って再原料化するメーカによるリサイクル,極力分別して,スクラップは再生棒鋼,段ボールは板紙にする既存のリサイクルルートの活用,混合廃棄物の4分の1を占める廃プラスチック類を分別してセメント原燃料などにリサイクルするなどの新規リサイクルルートの開発などが行われている.現状で再資源化が遅れているのは廃木材であり,この再生利用技術の開発が建設廃棄物処理の徹底の鍵を握る.現状ではさらなる廃木材からの合板化,チップ化,木材としての再利用の検討が進められていると同時に,再生を行う施設の普及が再資源化のポイントとなっている.コンクリートは,今後廃棄物の分別が進むと,これまで混合廃棄物に回ってしまった分が顕在化してくることも考えられる.そうなると,再資源化率が低下することも予想され,現状では,こうしたことを見通したうえで再資源化を徹底するために,再生砕石へのリサイクルだけでなく,骨材としての回収やその再生骨材の利用技術の開発も行われている.
