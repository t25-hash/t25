# 6・3・3 食品リサイクル技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・3 廃棄物処理・再資源化に関する先端技術

a. 概要食品廃棄物とは,食品の製造,流通,消費の各段階から発生する廃棄物である.産業廃棄物の中で食品廃棄物とされるのは動植物性残渣であり,一般廃棄物の中では厨芥類である.したがって,食品製造プロセスから発生する廃液や排ガスを処理することによって二次的に発生する汚泥などは,食品製造プロセスからの直接の廃棄物ではないとして,食品廃棄物には含めない.また,農業や畜産業からの二次的な廃棄物(農業からの稲わらや畜産におけるふん尿など)もここでは対象としない.産業廃棄物の発生量は年間約4億600万t(2000年度)であるが,その中で動植物性残渣は約400万tを占めている.また,一般廃棄物の発生量は年間約5200万t(2000年度)であるが,その中で厨芥類は1600万tを占めている.産業廃棄物である食品製造業から発生する動植物性残渣の約400万tはその約50%がリサイクルされている.その一方で,一般廃棄物となる流通段階で発生する事業系一般廃棄物の厨芥類は約600万tであるが,リサイクル率は1%に満たない.また,同じく一般廃棄物となる一般家庭から排出される生活系一般廃棄物の厨芥類1000万tはほとんどリサイクルされておらず,厨芥類全体としては約0.3%のリサイクル率となっている.すなわち,食品廃棄物は年間約2000万t発生し,現状ではその中の200万t程度がリサイクルされているにすぎない.このような背景から,「食品循環資源の再生利用等の促進に関する法律(食品リサイクル法)」が2000年6月に制定され,2001年5月から施行された.この法律は,一般家庭からの厨芥類を除く産業界からの食品廃棄物約1000万tを対象として,廃棄物の発生抑制とリサイクルの促進を目的としている.b. 廃棄物の発生源と種類. 製造段階での発生(1) 食品製造工場各種の食品製造工場,食品加工工場からは,プロセス上どうしても発生するロスとしての原材料の廃棄がある.これらは動植物性残渣であり,有機性廃棄物である.(2) 農畜産業や漁業農業や漁業では,出荷できずに廃棄された野菜や果物,魚などがある.. 流通段階での発生スーパーマーケット,コンビニエンスストア,百貨店,食料品店などの食品流通業から,売れ残りや賞味期限切れ,調理くず,調理で発生する食用廃油などの廃棄がある.これらは事業用途であっても一般廃棄物となる.. 消費段階での発生事業系一般廃棄物としては,飲食店,食堂,レストランなどの外食産業やホテル・旅館の施設から発生する調理ロス,調理くず,調理で発生する食用廃油,食べ残し,生ごみなどがある.廃棄物としての性状を考える場合には,調理前後では加熱や味付けなどの結果,含有成分に変化があることを意識しておく必要がある.また,一般家庭からも調理ロス,調理くず,調理で発生する食用廃油,食べ残し,生ごみなどが生活系一般廃棄物として発生する.c. 処理技術. 農業利用食品廃棄物は,製造過程で他の資材の混入がなければ安全であり,栄養バランスに優れており,微生物活性を高めるものもあり,農業利用に適している.農業利用ができる条件としては,作物の成長を阻害するような成分(有機酸やフェノール性酸など)を含まないこと,土壌環境に有害な重金属や病原菌を含んでいないこと,組成が安定していること,取扱いやすい性状(悪臭がなく,水分の含有量が適度で,形状が均一で,貯蔵性がよく,農業機械への適応性が高い)であることなどが挙げられる.農業利用としては,肥料化と堆肥化が挙げられる.生ごみは,牛ふんのような代表的な堆肥の原料に勝る肥料成分が含まれているので,乾燥するだけでも肥料としての利用は可能である.ただし,材料や処理の履歴によっては塩分や油分の調査が必要となる.また,堆肥化も可能である.堆肥化は一般的には堆積方法により行われることが多いが,生ごみは腐敗しやすいため発酵処理装置を用いる.農業や食品製造業からの廃棄物である野菜くず,果汁滓,ビール滓,おから(豆腐滓),コーヒー滓,茶滓などは堆肥化が行われている.それぞれの含有成分により,必要ならば副資材が加えられて堆肥化され,有機肥料や土壌改良材などになっている.. 飼料化飼料化の対象となる食品廃棄物は,おもに野菜くず,魚類,肉類,ご飯・パン・めん類,大豆製品などである.これらを飼料化するに当たっては,収集された廃棄物から金属,ガラス,プラスチック類などの異物の除去と,タンパク質と脂肪の含有量を調整するための素材の選択が必要になる.また,組成を維持するためには貯蔵方法にも配慮する必要がある.そして,2001年の牛海綿状脳症(BSE: bovine spongiform encephalopathy)問題の発生に伴って,肉骨粉などの畜産副産物の家畜への給与が禁止されたため,肉製品由来の素材が混入することを避ける必要もある.食品廃棄物の飼料化の方法としては,残飯養豚方式,リキッドフィーディング方式,乾燥方式などがある.残飯養豚方式は,食品廃棄物を煮沸してかゆ状にして豚に与える方法である.リキッドフィーディング方式は,食品製造副産物,食品廃棄物,配合飼料などを水と混合し,豚の飼槽に液状でパイプを通して圧送する方式である.コンピュータによる工程管理・制御が可能な方式で,規模の大きい施設に向いている.乾燥方式には,熱風乾燥,油温脱水,発酵乾燥などの方法があり,いずれも水分含有量を10%前後に調整する.日本ではいちばん普及している方式である.. エネルギー回収(メタン発酵)有機廃棄物は嫌気性(空気を遮断する)条件下で,微生物(メタン菌)の働きによりメタンを生成する.このプロセスを嫌気性消化,またはメタン発酵という.メタン発酵では,主成分がメタン60~70%,二酸化炭素30~40%,その他に微量成分として硫化水素などの硫黄系ガスと水分で構成されるガスが発生し,これをバイオガスと呼ぶ.バイオガスは高濃度のメタンを含んでいるため,単に廃棄物処理としてだけではなく,新エネルギー源として期待される.メタン発酵は微生物の活動を利用した反応であるため温度条件が重要で,30~37°C(中温発酵)付近と,55°C付近(高温発酵)の2種類の方法がある.中温発酵は滞留時間15~30日でメタンを生成する.高温発酵は中温発酵に比べて12~14日という短い滞留時間でメタンを生成でき,発生量も多く効率的であり,殺菌力が高いなどの利点がある.しかし,中温発酵に比べて反応の安定性が悪く,運転管理に高度な技術を要し,また温度推持のためにより大きなエネルギー投入が必要であるなどの問題がある.いずれのメタン発酵でも,得られたバイオガスはボイラ,ガスエンジン,最近では燃料電池などの燃料として利用可能である.. その他の処理技術(開発中のものを含む)(1) 減量(または減容)脱水,乾燥,焼却などの過程を通して,発生した食品廃棄物の量(重量,容積)を減少させる処理を減量(または減容)という.食品廃棄物は水分含有量の大きいものが多く,乾燥して減量(減容)するだけでも廃棄物処理として有効であり,その後の処理の前処理としても有効である場合が多い.(2) 工業原料化植物性廃棄物(ビール滓,米ぬか,もみがら,トウモロコシ芯など)や水産加工の廃棄物(魚類内臓)を高温高圧(水の超臨界点付近まで)処理すると,加水分解などの反応により工業原料として使える化学物質(糖類,フルフラール,有機酸,アミノ酸など)に変換できる.例えば,生成する糖類はアルコールや乳酸への変換が可能であり,工業原料となる.また,高温の水蒸気雰囲気に置くことでも動植物の細胞内での加水分解反応が起こるので,前述のような工業原料化が可能である.また,有機溶媒と無機溶媒をうまく選定することで,それぞれへの溶解特性を利用して,相分離で化学物質を回収する方法も開発されている.この方法によれば,親水性の炭水化物と親油性のリグニンの分離回収が可能となる.(3) 工業材料化植物性の残渣である各種の滓類を乾留(空気を遮断して熱分解)することにより炭化し,炭化物あるいは活性炭として利用する技術が開発されている.炭化物は調湿,化学物質吸着,断熱,吸音,電磁波遮断などの機能をもち,建築材料,調湿材料,環境浄化材料,土壌改良材などとしての利用の可能性がある.d. 今後の動向食品廃棄物のリサイクルは,これまではおもに肥料化,堆肥化,飼料化であった.しかし,日本の食糧自給率は50%を下回っており,完全な食糧輸入国である.このことは,肥料成分の海外からの日本国内への持込みであり,廃棄物の観点からは有機廃棄物の輸入でもある.すなわち,この有機廃棄物を肥料化や飼料化だけで処理しようとしても,肥料や飼料の完全な供給過剰となることを意味している.したがって,今後の技術開発の方向としては,この廃棄物の肥料化や飼料化により食糧自給の拡大に寄与するとともに,工業原材料の生産やエネルギー変換により,枯渇性の図6・11 熱分解ガス化改質方式によるASR リサイクルプラントのフロー図10―201化石資源から再生性の生物資源への資源転換に寄与することも求められるものと考えられる.
