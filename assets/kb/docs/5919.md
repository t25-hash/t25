# 6・3・4 廃自動車リサイクル技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・3 廃棄物処理・再資源化に関する先端技術

2002年7月に成立した「使用済み自動車の再資源化等に関する法律(自動車リサイクル法)」では,廃自動車の処理工程で発生するフロン類,エアバッグおよびシュレッダダスト(ASR : automo-bile shredder residue)について,自動車製造業者等に対して引取りおよびリサイクル(フロン類については破壊)を義務付けた.フロン類については回収された後,6・5・6項で示すフロン処理技術により処分され,エアバッグは解体された後,金属回収あるいはシュレッダダスト化される.ここでは,難処理物であり,従来ほとんど埋立処分されてきたASR のリサイクル技術について示す.a. ASR の発生量とリサイクルの目標値現在自動車のリサイクル率はおよそ80%にも達しているが,残余の20%の大半をASR が占めており,年間およそ80万t程度発生するASR はもっぱら埋立てによる処理が行われている状態である.国のガイドラインによると,今後の使用済み自動車全体のリサイクル率目標は,2002年以降85%以上,2015年以降95%以上という非常に高い値となっている.この目標の実現には自動車メーカによるマテリアルリサイクルの努力,ならびに難処理物であるASR のリサイクル技術の確立が不可欠となる.b. リサイクル技術の分類ASR のリサイクル技術は方式により次の五つに分類することができる.(1) 素材選別+原料化種々の選別工程により有価物を回収して原料化する.(2) 燃料代替+原料化非鉄金属精錬等の既存設備を活用して燃料利用し,金属を回収する.(3) 焼却処理+熱回収+原料化廃熱ボイラによりASR のエネルギーを回収し,メタルやスラグを回収する.(4) 熱分解ガス化+熱回収+原料化乾留ガス化で得られる熱分解ガスを二次燃焼し,廃熱をエネルギー回収するとともに,メタルやスラグを回収する.(5) 熱分解ガス化+ガス利用+原料化乾留ガス化で得られる熱分解ガスを改質精製して燃料ガスとして回収し,メタルやカーボンを回収する.近年,ダイオキシン類等の無害化性能の高さや,リサイクル性が脚光を浴び,都市ごみ処理プラントに熱分解ガス化(pyrolysisgasification)技術が急速に普及している.これを背景に,ASR 処理にも熱分解ガス化技術が適用されており,ロータリキルン(rotary kiln)方式,あるいは流動床式などのガス化炉によるプラ10―202環境システムントが建設され運用されている.しかしながら,ASR 成分は都市ごみに比べて,プラスチックや重金属類など多種多様な成分を含有し,ボイラ管に付着して閉そくを招きやすい低融点化合物や,また腐食の原因となる塩素の比率が高く,組成が一定しないなどの特殊性を有しているため,都市ごみのガス化溶融炉よりも高度な処理技術が必要と認識されている.c. ガス化改質によるリサイクル技術の例ここでは,先進的なASR リサイクル技術の例として,熱分解ガス化+ガス利用+原料化のカテゴリーに属するプラントを示す.図6・11にシステム概略フローを示す.ASR は破砕機により裁断された後,磁選機で鉄類が回収される.次に間接加熱式の熱分解炉(ロータリキルン)に投入され,ここで低酸素状態にて約550°Cで熱分解されることにより,熱分解ガスと残渣とに分けられる.熱分解ガスは次に内部を約1100°Cに保たれたガスクラッカ(改質器,gas cracker)に移動し,分子構造の大きな有機化合物から水素と一酸化炭素を主成分とする改質ガスに変換される.ガスクラッカを出た改質ガスは,廃熱回収冷却ボイラで急冷されることによりスチームを得るとともに,ダイオキシン類の再合成が抑制される.次にガス洗浄装置で有害な微量成分が除去された状態で系外にクリーンガスとして取出される.取出されたクリーンガスは,熱分解炉加熱用燃料として有効利用されるとともに,ガスエンジン発電機用の燃料として用いられることによりサーマルリサイクルを行っている.また,クリーンガスは所外に燃料として供給することによりマテリアルリサイクルすることも可能である.一方,熱分解炉で生じた残渣からはメタル分等の有価金属を選別回収し,残りは熱分解チャー(カーボン)となる.熱分解チャーは,固形燃料あるいは電炉等の還元材などとしてマテリアルリサイクルされる.
