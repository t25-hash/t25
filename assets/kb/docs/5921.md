# 6・4・2 河川水無害化技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・4 環境再生に関する先端技術

a. 概要河川や湖沼の水質は,生物化学的酸素要求量(BOD : biochemical oxygen demand)または化学的酸素要求量(COD : chemical oxygen demand)を指標として評価する.水域別の環境基準は,河川ではBOD,湖沼・海域ではCOD の数値で定められている.しかし,実際の河川や湖沼・海域の水質環境基準の達成率は,2001年度では河川で81.5%,湖沼は45.8%,全体で79.5%であり,十分とはいえない状態が続いている.水域の水質汚濁に対する河川の水質保全方法として,しゅんせつ,浄化用水導入,汚濁水のバイパス(流水保全水路事業)および汚濁河川,水路の直接浄化(河川浄化施設の設置)がある.ここでは河川の浄化技術として,速効的な水質保全方法である直接浄化の技術を中心に紹介する.b. 汚濁の原因河川の汚濁の原因としては,流域の人口増加や産業の発展に伴い汚濁排水が公共用水域に排出され水域の有する自然浄化作用を上回っていること,下水道整備や排水規制などの遅れ,流域内で対処できない汚濁負荷の発生(下水道整備計画区域外の排水や未規制事業所排水,非特定汚染源からの排水など)が考えられている.c. 処理技術直接浄化の技術としては,浄化対象水域内(例えば河床)に浄化施設を設置する直接浄化方式と,浄化施設は浄化対象水域から離して(例えば河川敷)設置する分離浄化方式に分けられ,それぞれに適した技術,そのどちらにも適用できる技術がある.. 直接浄化方式に適した技術(1) ばっ気法酸素(空気)を供給し,生物による有機物の分解無機化を図る.浄化機能の高い方法である.(2) 希釈法近くに十分な量の清浄な水が確保できる場合には,これを導入して汚濁水を希釈して浄化する.(3) 薄層流浄化法川底に接触材となるれきや木炭などを敷き,河床に河川水を薄く広く流すことで浄化を図る.沈殿や付着の効果のほかに,酸素の供給によりれきなどの表面に形成される生物膜による生物酸化の効果がある.(4) せき構築法せきを作ることにより,上流では流速低下,せきの越流部では河川水の落下により乱れと混合が生じる.これにより,上流部では汚濁物質の沈殿,せきの越流部では汚濁水の落下によるばっ気の効果で生物酸化が促進される.(5) しゅんせつ法汚濁した底泥を除去することによって,底泥による水質の汚濁を防止する.. 分離浄化方式の技術に適した技術(1) 浸透法汚濁水を土壌,土砂,砂れきなどの層に浸透させ,ろ過の原理で浄化を図る.浸透による浄化速度は遅く,広い面積の土地が必要である.(2) 凝集沈殿法凝集沈殿剤の投入によりフロックを形成させて汚濁物質を沈殿分離し,河川水を浄化する.凝集剤の投入施設を必要とし,凝集剤の環境に対する安全性の確認も必要となる.(3) 沈殿池(酸化池)法沈殿池を設けてそこへ汚濁した河川水を導入して,汚泥物質の沈殿による浄化を図る.藻類の増殖に伴い酸素が供給される好気的条件下での有機物の分解も期待される.沈澱池設置のための広い土地を必要とし,藻類の過剰発生を抑制する必要もある.. 直接浄化方式と分離浄化方式の両方に適用可能な技術(1) れき間接触酸化法浄化槽を設け,その中にれきなどの接触材を敷き,そこへ汚濁した河川水を導入することで浄化を図る.れきに形成される生物膜により,生物酸化で浄化する.また汚濁物質の接触沈殿も期待できる.れき間に堆積する汚泥の除去と処理処分が必要である.(2) 接触ろ材充てん法充てん材としてれきに相当するさまざまな形状の樹脂製ろ材を用いて,そこに形成される生物膜による生物酸化,汚濁物質の接触沈殿などによる浄化を図る.ろ材に堆積する汚泥の除去と処理・処分が必要である.(3) 活性炭浄化法(吸着剤等)活性炭を用いて汚濁水中の有機物を吸着除去し浄化する.活性炭のコストが問題である.(4) 水生植物植栽回収法水生植物には栄養塩類の吸収機能があるので,これを積極的に利用して浄化を図る.植物は天候によって能力が変動するので,この点の注意が必要である.d. 今後の動向上記の技術はおもにBOD,SS(浮遊物質,suspended solids)の低減を目的とした技術であるが,人体に有害なトリハロメタンや環境ホルモン物質に対応する除去技術は十分でなく,開発が必要である.また,閉鎖水域では富栄養化による水質悪化の防止が課題であり,窒素,リンの除去技術の開発が期待される.河川の直接浄化で効果が大きい技術はばっ気による生物浄化であるが,この技術では汚泥の発生を伴い,この処理が課題である.河川の浄化は実施されてから日が浅く,これまでは実績データの蓄積が少なく,実績に対して設計・計画の精度が結果的に低いケースがあった.しかし,これからはデータが蓄積されるに従って最適な設計・計画が可能になっていくものと考えられる.こうした実績データの評価・解析による実設計への反映方法も課題である.
