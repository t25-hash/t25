# 6・5・2 PCB 処理技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・18 に示す。イオン交換樹脂を加熱して熱分解ガスを発生させ、

.ガス化処理（gasification）技術 イオン交換樹脂中の
有機成分を500～700℃に加熱した水蒸気中でガス化し，このガス
を1000℃以上で酸化分解するプロセスである．図6･20 にその
プロセスを示す．ガス化はスクリューフィーダ型の移送機内で蒸気
れた．各技術の概要を表6･5 にまとめる．
の流れとイオン交換樹脂の移行を対向させることで行われる．ガス
化により生じる残渣はフィーダ出口で容器に回収される．発生した
ガス中の放射性核種は，フィルタにより除去される．除去後のガス
図6･17湿式酸化分解プロセス
は主反応器中で，空気および高圧空気により酸化される．ハロゲン
系元素を含む難燃性ガスが発生する場合には，あらかじめガスの一
部を空気量を低減させた状態で反応させ，その際の反応熱により水
蒸気と有機ハロゲン化合物を反応させることで，ハロゲンを除去す
る．脱ハロゲン後のガスは高圧空気雰囲気で酸化される．被処理物
中の炭素に対し残渣中の炭素の除去率は，イオン交換樹脂では
98.3 %，ハロゲン化合物では99.9 %，ペーパウエスは96.8%であ
