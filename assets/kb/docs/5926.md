# 6・5・5 プラズマ応用ガス処理技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・29 のような二酸化チタン電極と白金電極からなる電気化学セ

近年,指定化学物質(ホルムアルデヒド等)に関する濃度の指針値や,発電設備,清掃工場,自動車等から排出される種々な有害物質(ダイオキシン類,NO,粒子状物質等)の排出規制が強化されている.これらの環境汚染物質抑制の対策としてさまざまな研究開発が行われており,その一つとして大気圧下での低温プラズマ(non-thermal plasma)を応用したガス処理技術がある.以下に代表例を挙げて説明する.a. 脱臭(deodorization)技術脱臭技術の多くは,活性炭に代表されるような比表面積の大きな物質を使用し,その吸着機能を利用するものが一般的である.これら吸着機能を利用する技術は,初期的には効果があるものの,時間の経過に伴う機能低下を余儀なくされるもので,交換や再生などのメンテナンスが必要であった.低温プラズマを応用した脱臭技術は,化学的に反応性に富んだ活性種(ラジカル)の臭気成分に対する分解機能を利用するため,吸着機能を利用する脱臭技術とは異なる.さらに,低温プラズマから発生する380nm以下の紫外光を二酸化チタン(TiO)の励起に応用したものが放電型光触媒技術である.光触媒(photo-catalyst)は,低温プラズマと同様に臭気物質の分解機能を有するため,これらの融合技術は脱臭処理に対して相乗的な効果を発揮するものとなる.光触媒は紫外線の照射により表面にヒドロキシルラジカル(・OH)やスーパーオキサイドイオン(O)等の強い酸化力を有する活性酸素種を生成する.光触媒は触媒表面反応であるため,二酸化チタンの表面積を大きくする目的で図6・30に示すような三次元網目構造を有する触媒担持体を使用している.担持体表面には,担持体が挟持される構造で金属電極が配置されており,電極間への高電圧印加により図6・31に示すように光触媒近傍で低温プラズマが生じ,臭気物質にそれぞれ作用する.これらの強い分解作用は経時的に安定であるため,メンテナンスの必要がない脱臭技術として注目されている.b.
