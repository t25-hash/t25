# 6・5・7 熱プラズマ(thermal plasma)応用技術

> **章**: 第6章 先端環境対応技術
> **節**: 6・29 のような二酸化チタン電極と白金電極からなる電気化学セ

熱プラズマとは、大気圧付近以上の圧力領域で発生する高温・高密度のプラズマで、重い粒子(原子・イオン・分子)の温度と電子温度とがほぼ等しいプラズマの総称である。熱プラズマを発生させる手段は、正負両電極間に直流もしくは交流で大電流を流すアーク方式と、放電管外部からの高周波電磁界の誘導結合を利用する高周波(RF)プラズマ方式の二つに大別される。熱プラズマの特徴として、以下のものが挙げられる。(1) 1万K 以上の高温が容易に得られる。(2) 10cm程度の比較的高い電子密度が容易に得られる。(3) 熱容量が大きい。a. 熱プラズマの環境技術への適用このような特徴をもつ熱プラズマは、溶射、溶接、微粒子合成などに応用されてきたが、近年の地球環境問題への認識の高まりから、廃棄物や環境汚染物質の分解・無害化に熱プラズマを適用する取組みも積極的に推進されており、特に高濃度の廃棄物・汚染物質を高速に処理する手段として注目を集めている。また、廃棄物から有価値物(金属)を抽出しリサイクルするシステムへの適用も欧米を中心に進められている。6・5・7項ではRF・アークを問わず、環境問題の改善・解決に熱プラズマを適用した事例のうち、実用化されたものを中心に述べる。b. 都市ごみ焼却飛灰の溶融固化処理技術日本において熱プラズマを廃棄物処理に適用した代表例は、アークプラズマによる都市ごみ焼却飛灰の溶融固化処理である。都市ごみ焼却炉にて発生する焼却飛灰(焼却灰を含む場合もある)をアークプラズマにさらし、1500~1600°Cという高温にて灰分を融解させることにより、大幅に減容するものである。灰に含有されるダイオキシン類は、溶融炉内の1200°C以上の高温雰囲気にて完全に分解される。溶融スラグは冷却後、建設資材等にマテリアルリサイクルすることが可能である。灰溶融炉はその熱源が燃料によるものと電気によるものに大別されるが、上記のプラズマ溶融炉は発電設備を備えた大規模な都市ごみ焼却場に多く設置されている。c. 放射性廃棄物の溶融固化処理技術熱プラズマによる溶融処理は、都市ごみ焼却灰だけでなく、原子力関連施設からの低レベル放射性廃棄物の溶融固化処理へも適用されつつある。低レベル放射性廃棄物の封入されたドラム缶ごと溶融固化することも可能であり、廃棄物性状によらず減容比が大きい、処理速度が速いなどの特徴を有している。d. フロン分解処理技術地球温暖化の元凶といわれる特定フロンの分解処理にRF熱プラズマを適用する試みが1988年から始まり、現在、商用プラントとして連続的に稼働している。日本ではフロン分解技術として、ほかにもロータリキルン法、セメントキルン法、触媒法等が開発されているが、RF熱プラズマ処理法はそれらに先駆けていち早く実証・商用レベルに到達した。また、RF法以外に、DCアーク、マイクロ波等の熱プラズマによるフロン分解処理方法もすでに実用化されている。これら熱プラズマを用いたフロン処理方法の特徴として、処理時間、滞留時間ともに短く装置の小型化が可能なこと、排ガス量が少ないこと、装置の起動・停止が短時間でできることなどがある。その処理プロセスは、フロンと水蒸気を熱プラズマ中に導入することによりCO,HCl,HFに分解するものであり、発生したHCl,HF等の酸を中和する後処理系をシステムに組込む場合が大半である。e. PCB、廃油の処理技術被処理物が液体の例としては、PCB や廃油の処理が挙げられる。処理プロセスは、液体のままプラズマに導入し、処理対象により水蒸気、酸素等を供給するなど、気体であるフロン処理の場合と基本的には同じである。北アメリカでは車載型のPCB 処理装置が開発された。f. 金属リサイクル分野技術金属リサイクルの分野では、各種のスクラップや産業廃棄物を二次資源として有価金属を回収するプロセスにて、大型の高温加熱源として熱プラズマ(主としてDCアークプラズマ)を利用した技術が、1980年代から研究・実用化されている。電炉製鋼工程から発生する電炉ダストにはZn,Pb,Cr等が含有されているため、これをリサイクルするために、コークスとともにシャフト炉内に供給したダストをプラズマアーク中で炭素還元し、発生した金属蒸気を凝縮器で捕獲するプロセスがアメリカやスウェーデンで開発され、1984年からスウェーデンで実機が稼働している。文献(1) 小田哲治、静電学会誌、19 (1995)、283.(2) Tachibana,K.、ほか3名、J.
