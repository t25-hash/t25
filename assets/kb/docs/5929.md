# 7・2・1 諸外国の動き

> **章**: 第7章 継続・循環型社会の構築
> **節**: 7・2 循環型社会形成をめぐる環境政策の動向

(１)資源生産性（resources productivity）4割向上
ヨーロッパでは1980年代から環境政策が政治的にも重要な課題
資源生産性＝GDP/天然資源等投入量
2010年度は約39万円/t （2000年度：約28万円/t）と4割向上
させる目標設定で，GDP（gross domestic product）一定の場合，
日本の総物質投入量21.3億t （トン）を15.3億tに削減すること
を示唆したものである．国内資源（土砂，砕石，石灰石等）の使用
削減が実現のためのポイントとなる．
(２)循環利用率（circulation use rate）4割向上
循環利用率＝循環利用量/(循環利用量＋天然資源投入量)
2010年度は約12%（2000年度：約10%）と4割向上させる目標
設定で，廃棄物起因の循環資源化率を現状より4割向上させるもの
であり，循環資源利用プロセス，システムの改善が実現のためのポ
イントとなる．
(３)最終処分量半減
2010年度は約2800万t （2000年度：約5800万t）とおおむね
半減させる目標設定で，各事業者のゼロエミッション化促進が実現
のためのポイントとなる．
また，法律の理念を具体的に実行する法律として，(１)「廃棄物
処理法（改正）」，(２)「資源有効利用促進法」，(３)「容器包装リ
サイクル法」，(４)「家電リサイクル法」，(５)「食品リサイクル
法」，(６)「建設リサイクル法」，(７)「自動車リサイクル法」，
(８)「グリーン購入法」等が成立し，世界最高水準のリサイクル関
連法体系が整備され，これらの数値目標をバックアップしている
（図7･2)．
