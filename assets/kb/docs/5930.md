# 7・2・2 わが国の動き

> **章**: 第7章 継続・循環型社会の構築
> **節**: 7・2 循環型社会形成をめぐる環境政策の動向

a. 環境基本計画わが国においては、従来の「公害基本法」等から新たな環境政策の枠組として、1993年に「環境基本法」、1994年には環境基本計画が制定された。環境基本計画では、四つの目標である循環、共生、参加および国際的取組みを明言しており、環境負荷の少ない循環を基調とする経済社会システムを実現するために、環境施策の大綱、各主体の役割、政策手段を定めている。b. 循環型社会形成推進基本法廃棄物の排出量の高水準での推移、リユース、リサイクルの停滞、最終処分場の逼迫、不法投棄件数の増大といった問題に早急に対応するため、既存の実行法を取りまとめる形で「循環型社会形成推進基本法」が制定され、2001年1月、完全施行された。この法律は、社会物質循環の確保、天然資源の消費の抑制、環境負荷の提言を目的に、基本原則、各主体の責務、国のあるべき姿について、基本的な考え方を示している。2003年3月にこの基本計画に基づいて数値目標が設定されたため、今後は推進が加速されるであろう。以下に主要な3点の数値目標を示す。(1) 資源生産性(resources productivity)4割向上資源生産性=GDP/天然資源等投入量2010年度は約39万円/t(2000年度:約28万円/t)と4割向上させる目標設定で、GDP(gross domestic product)一定の場合、日本の総物質投入量21.3億t(トン)を15.3億tに削減することを示唆したものである。国内資源(土砂、砕石、石灰石等)の使用削減が実現のためのポイントとなる。(2) 循環利用率(circulation use rate)4割向上循環利用率=循環利用量/(循環利用量+天然資源投入量)2010年度は約12%(2000年度:約10%)と4割向上させる目標設定で、廃棄物起因の循環資源化率を現状より4割向上させるものであり、循環資源利用プロセス、システムの改善が実現のためのポイントとなる。(3) 最終処分量半減2010年度は約2800万t(2000年度:約5800万t)とおおむね半減させる目標設定で、各事業者のゼロエミッション化促進が実現のためのポイントとなる。また、法律の理念を具体的に実行する法律として、(1)「廃棄物処理法(改正)」、(2)「資源有効利用促進法」、(3)「容器包装リサイクル法」、(4)「家電リサイクル法」、(5)「食品リサイクル法」、(6)「建設リサイクル法」、(7)「自動車リサイクル法」、(8)「グリーン購入法」等が成立し、世界最高水準のリサイクル関連法体系が整備され、これらの数値目標をバックアップしている(図7・2)。c. 廃棄物処理法廃棄物処理法は1970年に制定された後、何度かの改正を経て、2000年5月、廃棄物の適正処理、廃棄物処理施設の設置規制、廃棄物処理業者に対する規制、廃棄物処理基準の設定等を定め、成立し、2001年4月より完全施行に至っている。廃棄物処理法の改正の大きなポイントは不適正処理への対策強化にある。d. 資源有効利用促進法(resources utilization promotion law)逼迫する最終処分場の問題、資源の将来的な枯渇の可能性等から、大量生産、大量消費、大量廃棄型の経済社会から転換し、循環型社会の形成に取り組むことが喫緊の課題であることを認識し、経済産業省産業構造審議会では1999年7月、循環経済システム構築に向けた提言をまとめた。この提言では3R の拡充、すなわちリサイクル対策(廃棄物の原材料としての再利用対策)の強化に加えて、リデュース対策(廃棄物の派生抑制対策)とリユース対策(廃棄物の部品としての再使用対策)の具体化を図るために、1991年に制定された「再生資源の利用の促進に関する法律」の抜本的な改正を実施し、2001年4月、「資源の有効な利用の促進に関する法律」(資源有効利用促進法)が施行された。
