# 7・3・2 地域(行政)、民間における取組み

> **章**: 第7章 継続・循環型社会の構築
> **節**: 7・10 )。鉄鋼プロセスを活用した手法には、JFEホールディングス

個別の排出事業者ごとのゼロエミッション(zero emission)、個別リサイクル法による資源循環は制度普及とともに急速に進んできた。2010年に向けた取組みではそれぞれの手法を地域ごとの特徴を生かした形で組み立て直す必要がある。地域と産業のゼロエミッション化対策手順(図7・12)で問題になる最大の難関は、コンセンサスと法体系の整備を含む経済的インセンティブとその継続性である。このような多様な地域環境に対応するには、トランスディシプリナリティー(trans-disciplinarity)学術フレーム(自然科学・人文科学・社会科学の融合領域)と組織フレーム(産官学民メディアによるグランドワークトラスト組織等)による運営が不可欠とされている。以下に多様性へ対応するための学術・組織フレーム(堀井清之)の一部と、代表的な事例として北九州エコタウンの内容を紹介する。a. 多様性へ対応するための学術フレームトランスディシプリナリティーとは、環境、エネルギー問題のような多様な複雑系の問題を解明するための学術フレームである。すなわち、自然科学・社会科学・人文科学の横断的融合領域であり、(超)学際領域のことである。この検討基盤となる組織は、産官学民であるが、その特徴は、科学研究を中心とした学問知を基盤とする政治・経済・行政・研究者・企業の連携知をもつエキスパートにより構成される。多様な物理的・文化的背景をもつ地域においてゼロエミッション、環境再生を実施するには、地域の発意があり、市町村のグランドデザインが存在し、評価基盤等を整備する研究者の支援、実施企業の技術が必要である(図7・13,7・14)。環境再生に関して、イギリスの事例では、このような組織として図7・12 地域と産業のゼロエミッション化対策手順10―223公益性を目的としたNPO(non profit organization)グランドワークトラストがある。このトラストは、環境問題を解決し、環境再生活動から経済的・社会的ベネフィットを生み出す組織フレームとして位置付けられている。イギリスのグランドワークは歴史をもち、産業および経済の衰退した、景観が劣悪で生活の質も相対的に低くなっている地域で以下のような事業が実施されている。(1) 地域のパートナーと協力し、町とその周辺にある荒地化した環境の再生を図る。(2) 地域の環境問題への取組みに住民の自覚を促し、教育および参加のための機会を提供する。(3) 地域への経済効果を向上させる。(4) 地域の活動を地球環境問題解決の足掛かりとする。(5) 地域の構成者との協力により、持続可能なライフスタイルを実現する。イギリス型のグランドワークトラストは、商法上の責任を負う機構であり、21世紀には企業とともに経済活動の一翼を担う重要な役割を演じることになると思われる。日本でも福島の環境インフラストラクチャの計画案をもとに、これらに関するビジネスモデルの特許も申請されている。環境に関わる地域特性を明らかにするため図7・13 行政とのパートナシップ図7・14 住民参加型事業の仕組み図7・15 総合環境コンビナート・響リサイクル団地10―224環境システムのエネルギー・物質・環境ポテンシャルマップの創造、地域を熱・物質移動マクロシステムとした環境再生計画、システムアナロジー等を解析する具体的な要素技術は機械工学の範ちゅうに多くある。環境問題への技術支援は、地域の機械工学関係者の使命と考えられる。トランスディシプリナリティー理念のファイナルゴールは、物理的には自然共生地域を具現化することにあり、精神文化的には理解と配慮に伴う共生思想・公共性の創造にある。b. エコタウン(eco-town)事業(国の補助事業)地方公共団体と民間企業が連携して行うリサイクル施設の建設費に対するハードウェア補助と、地方公共団体が行うエコタウンプランの計画策定、環境に関する展示会、情報提供等のソフトウェア面への助成を行う国の事業である。北九州市が公害を克服した貴重な経験を生かし、環境産業(environmental industry)を軸として産業社会構造の転換・再構築を図り、ゼロエミッション構想を推進するために北九州エコタウンプランを策定し、1997年に国の認証(第1号)を得て推進している。実証研究エリアでは、福岡大学資源循環・環境制御研究所を中核に、最終処分技術、遮水機能診断・修復システム(remediation system)、生分解性プラスチック製造技術、溶融スラグ有効利用技術、ガラス、廃コンクリートリサイクル技術、飛灰無害化技術、土壌汚染浄化技術等多くの研究施設が展開されている。また、総合環境工業団地では、周辺のリサイクル工場から排出される産業廃棄物をゼロエミッション化する複合中核溶融炉事業を中核に、ペットボトル、OA(office automation)機器、自動車、家電、蛍光管、医療用具リサイクル事業等が展開されている(図業拠点都市を目指して、環境産業の集積、研究開発・人材育成集積、国際展開、アジアモデル都市化を推進する。この取組みは全国に展開され、2003年現在で18箇所の地域が認証されている。c. その他の国による施策国は経済財政運営と構造改革に関する基本方針2002に基づいて、(1)教育・文化、科学技術、IT(information technology)、(2)都市、地域社会、(3)高齢化社会・少子化対策、(4)循環型社会の構築・地球環境問題への対応を重点課題とし、「都市再生本部/都市再生特別措置法」、「構造改革特区推進本部/特区法」を制定している。これは指定地域を選定し、地域発信型のプロジェクト推進を加速するための施策である。エコタウン事業の推進構造(自治体主導による計画と実行)と同質のものであるため、要素技術と評価法の開発とともにトランスディシプリナリティー理念の構築と日本型のグランドトラストの確立等が、この種の取組みの成否ポイントである。
