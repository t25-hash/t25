# 7・4・3 エネルギーの開発と利用促進

> **章**: 第7章 継続・循環型社会の構築
> **節**: 7・4 今後の関連取組み

日本のエネルギー政策は、1973年に発足した資源エネルギー庁(図7・17)が中心になって策定してきた。同庁の管轄下には七つの審議会があり、各審議会は大臣などの諮問に応じて調査・審議を行う。その結果は法的な拘束力はないものの、政策決定に強い影響力を発揮する。表7・9に石油代替(oil alternative)エネルギーの供給目標(2010年度)を示す。これに示されている新エネルギー(alternative energy)の定義と位置付けは、1998年6月に「新エネルギー利用等の促進に関する特別措置法(新エネルギー法)」において定められている(図7・18)。同法によると、狭義の新エネルギーとは、太陽光発電(photo-voltaic power generation)、太陽熱利用、風力発電(wind powergeneration)、黒液・廃材の利用、温度差エネルギーなどの再生可能エネルギー、廃棄物発電(waste power generation)、廃棄物熱利用などのリサイクル型エネルギーを示している。表7・10に新エネルギーの導入実績と目標、表7・11に新エネルギーの潜在性評価を示す。新エネルギーは既存のエネルギーに対してコスト競争力に欠けている。そのため市場原理に任せていては研究開発が進まず、技術革新も望めない。何らかの助成制度がなければ、石油に依存する現在の経済体質から脱却することは難しい。こうした要請にこたえるため、工業技術院(現・産業技術総合研究所)はプロジェ10―225図7・19ニューサンシャイン計画の体系図7・20 バイオマス・ニッポン総合戦略の目指すものクトを掲げ、1974年サンシャイン計画から、1978年ムーンライト計画、1993年ニューサンシャイン計画と変遷を続けている(図7・19)。しかし、リサイクル型エネルギー利用については再生可能エネルギーとのコスト差(廃棄物処理費が上乗せになる)、通常の燃焼型の発電効率との差(低い発電効率ではCO発生率の高い電気となる)、本来高い発熱量の廃棄物は分別されるべきなどの問題から補助金が制約されている。リサイクル型エネルギーの促進基準が大きな課題である。
