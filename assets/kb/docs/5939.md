# 1・1・3 人工衛星のミッション

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・1 人工衛星開発の歴史と現状

a. 科学観測極めて初期の人工衛星は,宇宙空間で動作する機器の技術確認がおもなミッションであったが,宇宙空間での科学観測を行うことにも同様な重きがおかれていた.1958年に打ち上げられたアメリカの最初の人工衛星であるエクスプローラ(Explorer)1号がバン・アレン帯を発見したのが好例である.科学観測をミッションとした人工衛星は,地球大気圏外の物理現象の観測や,大気圏に影響されない天文観測に多くの成果を上げてきた.その後,軌道上から地球を観測するミッションも加わり,気象図1・2 目的別人工衛星の打上げ実績図1・1 軌道別人工衛星の打上げ実績11―1や災害対策などの実用ミッションとの融合が進んでいる.地球以外の天体や惑星間空間を直接観測することも比較的早くから行われた.地球周回軌道を脱出し,惑星間空間を飛翔した物体は,1959年のルナ(Luna)1号が最初である.その後,月,火星,金星と探査の範囲は広がり,1972年と1973年に打ち上げられたパイオニア(Pioneer)10号,11号は木星と土星(11号のみ)のフライバイのあと,太陽系外に去った.これらは,もはや人工惑星ないし人工天体と称されるべきものであるが,工学的には人工衛星と同様の性格をもっている.b. 実用ミッション科学観測以外のミッションでは通信が早くから着目された.1960年に受動型のエコー(Echo)衛星,1962年には送受信機能をもったテルスター(Telstar)1号が打ち上げられている.さらに,1963年にはシンコム(Syncom)3号が静止軌道に打ち上げられ,その後通信ミッションには静止衛星を用いることが主流となった.これにより,全世界にわたる瞬時の通信が一般化し,人類の生活に大きな影響を与えることとなった.1990年代になると,静止衛星の欠点である通信経路の長大さを克服し,小さな携帯端末などの利用を可能とするために,より高度の低い周回軌道に多数の通信衛星(communication satellite)を配するシステム(コンステレーション)が提唱された.1997年から打上げが行われたイリジウム(Iridium)やオーブコム(Orbcom)は,それぞれ81個,50個の人工衛星からなるシステムを構築したが,商用としての成否についてはいまだ結論が出ていない.静止軌道では通信のほかに,気象観測衛星(meteorological sat-ellite)と放送衛星(broadcast satellite)が多数稼働している.地球観測は土地利用,天然資源,陸域・海域の環境などに関する情報を軌道上から広範囲に収集するもので,人類の生活とその環境を快適にし,かつ持続させるための切り札として期待されている.地球観測には全地球表面を観測でき,かつ観測の時刻が一定となる太陽同期軌道が利用されている.GPS(global positioning system)に代表される測位衛星は,当初軍事利用を中心に開発されたが,現在では一般に利用が開放されている.この目的に使われているアメリカのナブスター(Navstar)は,高度20000kmで異なる6個の軌道面にそれぞれ4個の人工衛星を配置している.このほか,技術実証を行うための人工衛星,重力環境の影響を離れて純粋な物理科学現象を研究する微小重力実験を行う人工衛星などが存在する.これらのミッションは,今後有人宇宙施設で拡大することが期待されている.c. 軍事衛星軍事目的の人工衛星は,アメリカ,ロシア(旧ソビエト社会主義共和国連邦)をはじめとする国々では宇宙開発を推進するうえで大きな位置を占めてきた.そのほとんどは通信や地球観測に密接に関連するものであり,これらの分野では一般に開放されていない高度な技術が存在することが予想される.最近の国際情勢の変化を受けて,このような軍事目的で開発された技術が一部の民生用人工衛星にも適用されるようになってきている.具体的な兵器としての人工衛星は,国際条約で配備が禁止されている.1970年代から1980年代の後半までには,人工衛星を破壊するための人工衛星の開発と実験が行われたこともあるが,現在ではこのような活動は行われていない.
