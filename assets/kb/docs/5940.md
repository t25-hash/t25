# 1・1・4 人工衛星の今後の発展方向

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・1 人工衛星開発の歴史と現状

人工衛星の今後の発展方向を見極めることは極めて難しい。科学観測の分野では、ハッブル宇宙望遠鏡やX 線、赤外線などの非可視領域を使った地球周回軌道上の大型の観測装置が実現し、宇宙科学の知識を急速に蓄積しており、今後もこの観測装置の高度化が望まれるであろう。また、月や惑星の探査は、将来の人類生存域拡大の可能性も含めて大きく推進されることが望まれる。近い将来には火星、金星などの固体型惑星に限定せず、木星以遠のガス型惑星とその衛星への探査も本格化することが期待される。実用分野では、今後も引き続き通信や地球観測が主要なミッションとして継続されるであろう。通信・放送衛星は商用運用の時代に入っている。地球観測衛星の一種である高解像度衛星の画像を販売するビジネスが始まったが、大きな流れになるかどうかは今後の動きにかかっている。日本で打ち上げた実用衛星は、政府機関、地方自治体、研究所、企業が求めるサービスを提供しているが、さらにサービス対象を一般に拡大し、できれば商用運用に移行させることが望まれている。一方、人工衛星の技術発展の方向は比較的明確であるといえよう。まず、科学・実用のどちらに対しても、軌道上の人工衛星がより長期に安定した性能を発揮することが求められる。このため、信頼性を確保する設計・製造手法の強化が求められるとともに、特に大きな施設では軌道上における保守・点検を容易にする技術面の進展が期待される。宇宙基地における有人活動と連動して、ロボットなどの活用が大きな役割を果たすであろう。惑星空間では、地球との通信に長時間がかかることから、装置の知能化・自動化が不可欠である。次に、人工衛星のコストを削減することは大変大きな課題となっている。人工衛星の各部の標準化が有効との論議があるが、この方向はいまだ本格的に進んでいない。むしろ、地上機器の部品や技術を積極的に人工衛星に取込むことが検討されている。衛星の大型化は実用衛星では一般的にコスト的に有利である反面、科学観測や一部の実用ミッションでも単機能化による小型化がコスト低減とシステム信頼性の向上に資することも期待される。いずれの場合でも、部品・機器のマイクロ化、ナノ化は人工衛星の機能向上とコスト低減に大きな影響をもたらす。
