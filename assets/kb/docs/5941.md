# 1・2・1 工学の対象としてのロケット

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・2 ロケット開発の歴史と現状

古くから煙火(花火)・ロケット弾に火薬が用いられていたが、外部・内部弾道学の理論的研究が始められたのは19世紀に入ってからである。ロケット推進が理論的に扱われるようになったのはさらに後のことで、ロケット推進の理論を体系的に展開したのは、ロシアの科学者ツィオルコフスキーであった。彼は、ロケット推進力のみが働き、外力が働かない場合のロケットの最終速度を求める問題(ツィオルコフスキーの第1の問題)の解を、以下の式として求めた。これがツィオルコフスキーの公式(Tsiolkovsky’s equation)であり、ロケット推進の本質を余すところなく示していて、極めて実用性の高いものである。彼は、酸素と水素の組合せが最も大きな有効排気速度を与えることを理解していた。さらに、多段構成ロケットの理論的基礎を確立した。彼の功績は、SF小説の領域にあったロケットに科学としての位置付けを与えたことにある。ロケットが工学の対象として、確固たる地位を確保するうえで功績があった人物の一人は、ドイツの工学者フォン・ブラウンである。彼が、第二次世界大戦中、ドイツ陸軍のために開発したA-4(通称V-2)ロケットは、フォン・ブラウン型ロケットとでも呼ぶべきもので、大推力液体ロケットエンジン、極低温推進薬、慣性誘導、ノズル流れ偏向による姿勢制御、ランチャ上にロケットを直立させる垂直打上げ方式を初めて採用したものである。それとともにシステム的な考え方に基づく組立・点検・整備・打上げ方式を実用化し、近代ロケットシステムの源流となると同時に、今に至るまで、ロケットの基本形としての揺るぎない地位を保っている画期的なロケットであった。
