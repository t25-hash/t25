# 1・2・2 スプートニク以後の展開

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・2 ロケット開発の歴史と現状

第二次世界大戦直後、ロケットの運命が風前のともしびとなった時期がある。A-4ロケットは、システムとしては画期的であるものの、兵器としての評価ははなはだ低く、アメリカでも旧ソ連(ソビエト社会主義共和国連邦)でも軍が興味を示さなかったからである。その事情が一変したのは、核兵器開発が進み、その唯一の運搬手段として、長距離弾道弾が脚光を浴びだしたときである。その折りも折、国際地球観測年の一行事として旧ソ連が人工衛星(スプートニク)を打ち上げた。旧ソ連のスプートニク打上げは、入念な計画で行われた結果とはいいがたく、ロケットの発達に貢献したもう一人の工学者セルゲイ・コロリョフが積み重ねてきた個人的な研究開発活動の成果が、たまたま、外的状況の要求に合致して実を結んだものであるが、結果的には、宇宙の平和利用への門戸を開く快挙となった。1960年代、旧ソ連とアメリカとの間で、熾烈な有人月面着陸飛行の一番乗り争いが行われ(月レース)、サターンロケット・アポロ宇宙船を開発したアメリカが勝ちを収めたが、一代限りで、その後使われることはなかった。もともと、サターンロケットは競争用の特別仕様で、全く市場性を考慮していなかったので、当然といえば当然である。ロケットは、その後、多種・多様なものが出現し、現在は、宇宙輸送システムとして次のように分類されている。宇宙輸送システム打上げロケット宇宙往還機軌道間輸送機月・惑星ロケット打上げロケット(launch vehicle)は地球軌道上へのペイロード輸送用ロケットで、有人・無人の別があるが、いずれも片道飛行である。宇宙往還機(space plane)は、地上・地球周回軌道間の往復輸送手段であり、実際に実現し現在も運用されているのは、唯一スペースシャトルのみである。軌道間輸送機(orbital transfervehicle)は、地球周回軌道間の移動に用いられるロケットであるが、今のところ実例はない。月・惑星ロケット(interplanetaryspace vehicle)は、月あるいは惑星への有人・無人の輸送手段である。以下、打上げロケット、宇宙往還機、月・惑星ロケットについて、歴史と現状の概略を述べる。
