# 1・2・3 打上げロケット

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・2 ロケット開発の歴史と現状

1950年代後半,華やかな宇宙開発競争の陰で,米ソ両国による核兵器運搬を目的とする大陸間弾道ミサイル開発競争が行われた.核兵器を輸送するロケットには,大きな搭載能力と高い信頼度が要求され,これらのロケットが,宇宙開発におけるペイロード打上げに転用されていった.アメリカでは,デルタ,アトラス,タイタン,旧ソ連では,ソユーズ,プロトンなどがそれである.1970年以降,日本,ヨーロッパ,中華人民共和国(中国)等も宇宙開発に参入しだした.わが国は,アメリカから導入したデルタロケットの技術をベースに,N シリーズ,H シリーズのロケットを,またヨーロッパはアリアン,中国は長征シリーズのロケットを開発した.1990年に入って,打上げロケットの世界に大変革があった.地球周回軌道の利用,特に衛星通信が活発化し,民間企業のペイロード打上げ市場が出現したのである.いわゆる宇宙商業化である.これに,1991年の旧ソ連崩解に伴う,ロシアの低価格ロケットの市場参入が相まって,激しい価格競争が始まった.日本ではH-IIAを開発して(2001年8月29日,試験1号機打上げ),市場参入をねらっているが,各国ともロケットを大型化して,打上げ能力を上げながら(GTOに5~10トン級),コスト低減を図っている.2002年に初号機の打ち上げられたアメリカのアトラスV,デルタIV,ヨーロッパのアリアン5改良型などがそれである[アトラスV初号機(2002年8月21日),ならびにデルタIV初号機(2002年11月16日)の打上げは成功したが,アリアン5改良型(2段に極低温推進系を採用)の初号機打上げ(2002年12月11日)は失敗し,明暗を分けた].アメリカでは,競争力を上げるための企業再編も行われ,打上げ専門のサービス会社が作られた.そして,それらサービス会社が,アメリカ・ロシアのロケット製造企業が作ったロケットを使って商業ペイロードを打ち上げるというビジネス方式が確立された.2000年代は,低軌道衛星利用の移動体通信網計画の破綻,衛星技術の進歩による衛星寿命増加などから,1990年代末に平均35機あった商業打上げが,2001年以降半減している.また,商業通信衛星が大型化する一方,中型衛星の需要も堅調で,ロケットのペイロードへの対応の柔軟性がいっそう求められている.今後の打上げ市場は,デルタ,アトラス,アリアン,ソユーズ,プロトン,ゼニートなどの既存勢力に,ロシアのICBM 転用打上げロケット群(ドニエプル,ロコットなど),ロシアで開発中のアンガラ,さらにはH-IIA,長征が加わって少ないパイを奪い合う競争は,ますます激しくなると予想される.競争に勝ち残る条件は,高信頼度と低コストであり,ともに打上げ機会が多いロケットが有利になる.日本ではこれまで,打上げ機数が10を超える同一型のロケットはなかった.不幸にして,H-IIA は6号機で失敗したが(2003年11月29日),7,8,9,10号機の打上げには成功した.今後,打上げ成功実績を,20機,30機と積み重ねていく必要がある.なお,打上げロケットの趨勢として,ここ当分は使い捨て型(expendable)が主流となるであろう.使い捨て型打上げロケットの技術はほぼ飽和点に達しているので,開発・運用ともに民間に比重を移し,設計・製造段階の努力で,信頼性確保と同時に低コスト化を実現していくことが今後の課題となる.
