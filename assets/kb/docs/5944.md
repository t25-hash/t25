# 1・2・4 宇宙往還機

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・2 ロケット開発の歴史と現状

人間が乗って,宇宙,とりあえずは地球周回軌道上に達することは,1960年にアメリカでアメリカ航空宇宙局(NASA : NationalAeronautics and Space Administration)が発足したときの主要な目標の一つであった.1970年代にアメリカはスペースシャトルを開発した.スペースシャトルは,初めて実用化された宇宙往還機として,歴史的意義をもつロケットである.しかし完全再使用型ではなく,一部を再使用する設計となっている.現在,スペースシャトルが抱える問題は,運用段階に入ってからの打上げ機数が予測を大幅に下回っていることと,既存の機体の老朽化である.打上げ機数については,計画当初,政府(含む軍用)ペイロード,民間ペイロード合わせて週1回の運航(年間平均52回)を予定しており,コンフィギュレーションを決定する際のトレードオフ検討も,それをベースに行われた.固体ロケットモータを再使用するようにしたのも,この打上げ機数が根拠となっていた.ところが,スペースシャトルに国防省が興味を示さず,また,再整備に時間がかかりすぎて週1回の運航はとても無理であることがわかり,2006年初頭までの運航実績は,当初計画の20分の1以下となっている.運用コストは打上げ機数に反比例して上昇し,現在,1回の打上げにかか第I部導入編11―3る費用は5億ドルを超えているといわれる(予定は約770万ドル).コロンビア号の事故(2003年2月1日)は,老朽化も含め,信頼性,安全性,その他スペースシャトルの抱える諸問題を改めて浮彫りにした.スペースシャトルの後継往還機としては,完全再使用型で低コストの宇宙往還機を目指して,ゼンガー,ホトル,Aerospace planeなどに始まり,デルタクリッパ(Delta Clipper),X 38に至るまで,これまでに多くの構想が出された.またヨーロッパでは,WLC,FESTIPなどの長期的検討がなされ,旧ソ連では,ブラン,エネルギアが製作され,日本でも観光丸の提案がなされるなどさまざまな活動が今なお行われているが,いまだに決定版が出ていない.その理由は,市場が未成熟で需要の見通しが不鮮明なためミッションが絞れないことと,技術的ハードルが高いことであった.しかし,1996年に発足したサブオービタル宇宙船開発のためのAnsari X prize基金の懸賞に応募したアメリカMAV 社のスペースシップワン(Space Ship One)が2004年10月,2度のサブオービタル飛行に成功し,賞金1000万ドルを獲得して以降,にわかに,民間を含めての有人宇宙往還ロケット開発の動きが活発化してきた.NASA はスペースシャトル代替として,国際宇宙ステーションに搭乗員を送るためのCOTS(Commercial Orbital Transporta-tion Services)開発を2010年を目標に進めることを発表.また,民間宇宙ステーション計画(Nautilus計画)を発表したビゲロウ(R. Bigelow)が輸送手段としての5人乗り輸送手段の2010年までの開発を公募(American prize)するなどして,アメリカでは既存大手メーカのみならず多くのベンチャー企業が参加し,宇宙往還機開発に取組みだした.ロシアでも,2006年に発表された2015年までの宇宙開発計画の中に,宇宙往還機計画(Klipper計画)があり,2006年現在メーカ選定作業中である.2003年10月,自前の有人宇宙船「神舟」を長征ロケットで打ち上げた中国は,独自の宇宙ステーション計画を持っている.その一環として,大型の有人宇宙往還システムを開発することが伝えられており,今後の動向が注目されている.
