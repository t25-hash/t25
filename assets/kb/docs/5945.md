# 1・2・5 月・惑星ロケット

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・2 ロケット開発の歴史と現状

月・惑星探査システムには,はん用の打上げロケットのペイロードとして探査機を搭載し,打上げロケットで地球作用圏からの脱出速度を与えたのち探査機を自力で推進させて目的の惑星に向かう形式と,専用の打上げロケットを開発して探査機を打ち上げる形式とがある.これまでに開発された無人の月・惑星探査ロケットはほとんどが前者の形式で,宇宙航空研究開発機構(JAXA : JapanAerospace Exploration Agency)が開発している月周回衛星計画SELENEもこの方式を採用しており,H-IIA を用いて,ペイロードを月への遷移軌道に投入し,月周回衛星を実現しようというものである.この方式は,今後も無人月・惑星計画の主流であり続け,打上げロケットの大型化に伴って,探査機も大型化していくだろうと考えられる.後者の形式は,1960年代後半の有人月着陸帰還計画(アポロ計画)に採用され,大型の専用ロケットであるサターンV 型ロケットが開発されたが,その後長い間新たな開発計画はなかった.NASA は,初期のころ,スペースシャトルと宇宙ステーションを組合わせて月への航路を開く構想をもっていたが,アポロ計画成功後,国民の月への関心が薄れたことに,再三の宇宙ステーション計画の延期などが重なって,財政的にも余裕がなくなっていたという事情があった.2004年1月14日にアメリカのブッシュ大統領が新宇宙政策を発表し,有人月・火星計画を新たな目標として掲げた.NASA は,国際宇宙ステーション運用を民間へ移行する一方,自らは月・火星計画に比重を移す方針であるかのように見え,そのための有人宇宙船打上げロケット・アレス(Ares)Iと無人の資材打上げロケッ11―4宇宙機器・システムト・アレスVを開発する計画を明らかにしている.ロシアは2015年までの宇宙計画の中では,月無人探査を基調とする方針のように見えるが,アメリカの計画の進展次第では変化があるかもしれない.なお,宇宙科学研究所(ISAS,現JAXA)が開発し,わが国の無人惑星探査機打上げに活躍してきたM-Vロケットは,2006年の打上げをもって引退することが決定された.宇宙開発の先駆者たちの業績からもうかがい知れるように,有人宇宙開発の歴史は,宇宙開発の歴史そのものである.例えば,ロケット推進と多段ロケットの理論的基礎を作り上げたツィオルコフスキーは人工衛星の上での生活について述べ,ドイツ人科学者オーベルトも宇宙ステーションに言及している.もともと月旅行にあこがれていたフォン・ブラウンもドーナツ型宇宙ステーションを構想した.しかし,これらの構想はすべて具体的な技術の裏付けのない夢であった.1957年10月,旧ソ連がスプートニクを打ち上げたときにこの状況が一変した.アメリカは翌年にNASA を発足させ,この新組織の目標の一つに有人宇宙プログラムを設定した.その第一歩として,レッドストーンロケットを用いた有人弾道飛行を行い,次にアトラスロケットを用いた軌道飛行(マーキュリー計画)を行った.これに続く短期目標として月周回有人飛行と軌道上実験室を目指し,長期目標として,有人月着陸から有人惑星飛行,軌道上実験室の延長上に宇宙ステーションを設定した.しかし,1961年4月12日,旧ソ連はヴォストーク1号による有人軌道飛行を実現させ,ガガーリン宇宙飛行士が「地球は青かった」との名言を残した.再び衝撃を受けたアメリカは,1961年5月25日のケネディ大統領による演説で有人月面着陸計画[manned Moon landing program,以後,アポロ計画(Apollo Program)と呼ばれる]を発表した.1969年のアポロ11号による人類初の月着陸は,アームストロング船長によってなされ,世紀の大イベントとして歴史に刻まれることになる.1969年9月,ニクソン大統領に提出されたアポロ計画終了後の宇宙開発に関する計画書(アグニュウ報告と呼ばれる)には,スペースシャトル,宇宙ステーションなどが含まれていたが,大量の失業者を出したアポロ計画を反省したNASA は国防省が興味を示したスペースシャトル開発に絞り,1972年初めにスペースシャトルの開発が決定された.
