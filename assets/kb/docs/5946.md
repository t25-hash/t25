# 1・3・1 スカイラブ(Skylab)

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・3 有人宇宙開発の歴史と現状

1969年9月以後,大統領の決定が下りるまでの間,NASA が検討したアポロ後の作業計画中にスカイラブ計画があった.その構想は,サターンV 型ロケットの第3段(S-IVB)を改造した直径約7m,長さ約20m,質量約90トンのワークショップと呼ばれる本体に3名の宇宙飛行士を収容するものであった.宇宙飛行士を輸送する手段としては,サターンV 型ロケット第3段の代わりに,地上で改装・整備されたワークショップを打ち上げるドライ方式が採用され,1973年5月14日に打ち上げられてスカイラブと名付けられた.スカイラブの技術は,基本的にはアポロ計画で開発された技術の直接の延長上にあり,生命維持システム(CO除去は実施,Oや水の再生は未実施),電源,熱および環境制御装置,姿勢制御システム,通信システム,シャワー,トイレ等,人間の長期滞在に必要と考えられる最低限の機能・装備をすべて備えていた.スカイラブは1974年2月8日にその全計画を終了するまでに,28日間,59日間,84日間の3回のミッションを行い,各3名ずつの宇宙飛行士が合計171日間,延べ513人日間,ワークショップに滞在した.スカイラブは,最初の本格的な有人宇宙活動と位置付けられ,そのミッション期間中,船外活動,宇宙生理学,宇宙ステーション設計に関する多くの貴重なデータを残した.主要な成果は次のとおりである.(1) 船外活動(EVA : extra vehicular activity)打上げ時に破損した隕石防御板により傷ついた太陽電池パネルを船外宇宙活動によって修理した.その後,ジャイロ交換などの修理作業のために,6人の宇宙飛行士が延べ10回,合計約42時間の船外活動を行い,無重力状態下で作業が可能であることを実証した.(2) 宇宙生理学自転車,下半身陰圧負荷装置などを備え,宇宙飛行士の医師による無重力下での人体の各種データの計測,能力の実験を行い,無重力下で90日間の滞在が可能であることが実証された.スカイラブは,NASA が予測していたより早く軌道が低下し始め,1979年7月11日,大気圏に突入した.その残骸はインド洋からオーストラリア大陸西部にかけて飛散し,安全性に関しての教訓を残した.
