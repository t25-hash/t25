# 1・3・2 米ソの共同実験:アポロ・ソユーズ実験計画とシャト

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・3 有人宇宙開発の歴史と現状

ル・ミール計画アメリカと旧ソ連の宇宙開発競争の中,二つの共同実験が行われた.一つはアポロ・ソユーズ実験計画(ASTP: Apollo SoyuzTest Program)であり,もう一つはスペースシャトルとミールのドッキングである.1972年に結ばれた米ソ間の協定に基づいてアポロ・ソユーズ実験は実施された.1975年7月15日,レオーノフ,クバソフ両宇宙飛行士を乗せたソユーズ19が打ち上げられ,また同日,スタフォード,スレイトン,ブラントの3人の宇宙飛行士を乗せたアポロ宇宙船も,サターンIB ロケットで打ち上げられた.7月17日以降,アポロ側とソユーズ側が交互に制御状態でドッキングし,宇宙飛行士が交歓訪問を行った.このときに開発されたドッキングシステムは,その後スペースシャトル・ミールドッキングを経て,宇宙ステーションまでその設計が受け継がれている.1993年,国際宇宙ステーション計画においてロシアの参加が決定されると同時に,NASA はミールの技術の評価と協力関係推進のため,スペースシャトルをミールにドッキングさせる計画を立案した.1995年6月に,スペースシャトルSTS-71のオービタとミールのドッキングが行われ,1995年11月には,STS-74でオービタが再度ミールにドッキングした.
