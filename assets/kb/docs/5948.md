# 1・3・3 旧ソ連の有人宇宙活動:ソユーズとサリュート・ミー

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・3 有人宇宙開発の歴史と現状

ル有人月面着陸競争でアメリカに敗れた旧ソ連は、1960年代後半に開発されたソユーズ(Soyuz)宇宙船とソユーズロケットを組合わせて利用する有人宇宙開発計画への本格的な取組みを開始し、以後、旧ソ連の有人宇宙計画はソユーズを軸に展開された。ソユーズ宇宙船は、軌道・再突入・サービスの3モジュールからなる質量約6.6トンの有人宇宙活動専用の宇宙船である。安定した軌道への乗員の輸送手段となったと同時に、サリュート、ミールなどの宇宙ステーションに発展していったが、その発展は次の3世代に分けられる。第一世代の宇宙ステーションは、短期滞在型のサリュート1号(1971年4月打上げ)からサリュート5号(1976年6月打上げ)までを指し、第二世代は長期間滞在が可能なサリュート6,7号を指す。第三世代宇宙ステーションがミールである。本体は合計6個のドッキングポートをもち、サリュートをいくつか合体させたものである。最初のモジュールが1986年2月に打ち上げられ、その後次々とモジュールが増設されていった。ミールの後継機として大規模なミール2計画も検討されたが、この計画は1991年にキャンセルされ、1993年に国際宇宙ステーションに合流することが決定した。ミールは廃棄処分となり、2001年に約15年間にわたる歴史の幕を閉じた。この間における旧ソ連・ロシアの宇宙飛行士の軌道上滞在期間は、延べ約30年間に達するといわれている。a. サリュート(Salute)サリュート本体はプロトンロケットで打ち上げられ、乗員はソユーズロケットで打ち上げられるソユーズ宇宙船で行き来した。当初のソユーズ宇宙船は3人乗りであったが、サリュート1号からの帰還途中、高度168kmでソユーズ11宇宙船のキャビンの空気が抜け、乗員3名が死亡した事故の結果、乗員を2名として往復には気密服を着用するようにした。第二世代のサリュート6号は、サリュートシリーズの中では最も成功した宇宙ステーションで、当初予定の寿命22箇月をはるかに超えて使用され、長期滞在記録を次々と塗り替え、宇宙医学、宇宙環境利用実験でも多くの成果を上げた。また、サリュートシリーズでは、地球周辺の宇宙空間の研究、宇宙医学、宇宙生理学、長期滞在の人体への影響、天体観測、地球表面の観察、無重力下での実験などが行われた。サリュートは推進機関による高度維持の能力を有し、高度を350kmまで上げ、自然低下(高度減少率142m/day)するとまた上昇することを繰返していた。b. ミール(Mir)ミールは、ミール本体に拡張用のクヴァント1をつなぎ、前方(ミール本体側)ポートに乗員の帰還用のソユーズ、後方(クヴァント側)ポートに無人貨物宇宙船プログレスを結合したものを中心とし、ミール本体前方のノードにある周方向の4個のポートに、クヴァント2、クリスタル、スペクトル、プリローダの4個のモジュールを放射状に展開した構造となっている。これらのモジュールは、サリュートをベースとする同じような基本構造をもった独立のモジュールである。1986年2月19日にミール本体が打ち上げられてから、1996年4月27日に最後のモジュールであるプリローダがドッキングするまで、ほぼ10年を要したが、ここでは有人宇宙活動による多くの宇宙環境実験が実施された。前述の国際宇宙ステーション計画への参加の結果、ミールは廃棄されることになり、2001年3月23日15時00分13秒、3回の制動噴射の後に大気圏に突入、南緯40°、西経160°を中心とする南太平洋海域に落下した。
