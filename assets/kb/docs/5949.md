# 1・3・4 ヨーロッパによるスペースラブ(Spacelab)の開発と

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・3 有人宇宙開発の歴史と現状

日本の参加1969年,ポストアポロ計画への参加を促すアメリカの呼びかけに応えて,ヨーロッパはスペースシャトルのカーゴベイに搭載される再使用型宇宙実験システムを開発することになった.当初は,アメリカへの無償提供分とNASA の購入分を含めて6セット製造する予定であったが,実際にはアメリカは製作試験コストをはるかに下回る価格で1機分を購入したのみで,計画を打ち切った.このため,ヨーロッパのアメリカに対する不信感が生まれ,国際宇宙ステーション計画初期における両サイドの不協和音の原因となった.スペースラブは三つの主要要素(直径約4m,長さ2.7mの与圧モジュール,パレット,トンネル)と二つの補助要素(制御機器収納用のイグルーとミッション固有の実験支持構造部)とからなる.スペースラブはスペースシャトルから電力の供給を受け,自らは搭載している実験装置への電力の分配を行う.また,スペースラブ内には環境制御装置があり,温・湿度の制御,炭酸ガスの除去などを行っている.実験で取得したデータは,スペースシャトル経由で地上へ伝送される.スペースラブは,1974年に開発が開始され,1983年11月に初飛行を行った.このとき,日本の宇宙科学研究所(当時)が粒子加速器による宇宙実験(SEPAC: Space Experi-ment with Particle Accelerators)を行った.また,1992年9月12日,エンデバー号で打ち上げられたSTS-47ミッションには,毛利衛宇宙飛行士が搭乗し,第一次材料実験(FMPT : FirstMaterial Processing Test)を行い,1994年7月28日,コロンビア号で打ち上げられたSTS-65ミッションには向井千秋宇宙飛行士が搭乗し,ライフサイエンス系実験を行った.
