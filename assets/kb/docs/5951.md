# 1・3・6 中国の有人宇宙飛行

> **章**: 第1章 宇宙開発の歴史と現状
> **節**: 1・3 有人宇宙開発の歴史と現状

中国は、独自の有人宇宙船を開発することを宣言してから、着実に無人実験を重ね、ついに2003年10月、神舟(Shenzhou)5号によって世界で3番めに有人宇宙飛行を達成した国となった。その開発の歩みは以下のとおりである。1999年11月の神舟1号に続いて、2001年1月10日に試験機である神舟2号が打ち上げられ、16日に帰還モジュールが中国の内モンゴル自治区に着陸した。神舟2号の軌道モジュールはロシアのソユーズをベースとしているが、ソユーズと違い太陽パネル1組が備え付けられており、推進システムももっている。2002年3月25日、有人宇宙飛行に向けた無人試験機、神舟3号が甘粛省の酒泉衛星発射センターから長征2Fロケットで打ち上げられ、宇宙機の軌道投入に成功した。神舟3号は神舟2号と比べて構造や信頼性など技術的品質が向上している。飛行中にライフサイエンス実験、材料実験、天文研究、物理・微小重力実験を実施した後、帰還モジュールは4月1日に回収された。このとき、関係者は、ロケットと有人宇宙船の安全性と信頼性が確認されれば、2年以内に有人宇宙飛行を開始すること、中国は最初の有人飛行で3人の宇宙飛行士を打ち上げる能力を有していることなどを強調した。2002年12月29日、神舟4号が打ち上げられ、約10分後に所定軌道へ投入された。神舟4号は、有人宇宙船打上げに向けた無人技術実証機で、次の神舟5号は少なくとも1名の宇宙飛行士が搭乗することが明らかにされた。また、神舟5号の安全性についても、不具合自動探知、高速エレベータ脱出装置、自動・半自動スライド式ケーブル、または射出シートによる緊急脱出システム、軌道上での事故に備えた救命設備、信頼性のある精度の高い誘導システム、回収モジュールの安全性、追跡・コマンドを行うための地上および海上の高度ネットワークセンターなど、周到な準備がなされた。2003年10月15日、宇宙飛行士の楊立偉(Yang Liwei)が搭乗した神舟5号は酒泉衛星発射センターから長征2Fロケットによって打ち上げられた。地球を14回周回した後(約21時間30分)、内モンゴル自治区へ着陸した。着陸地点は当初計画されていた地点から約4.8kmずれていた。神舟5号機の質量は7800kg、軌道は高度約343kmの円軌道であった。また、軌道周回モジュールの実験(地上分解能1.6mのCCD カメラを含む)が、打上げから152日めとなる2004年3月9日にすべて完了し、以後は徐々に軌道を下げていき、数週間で大気圏に再突入する予定と報道した。この成功によって弾みのついた中国は、次の神舟6号には2名の宇宙飛行士が搭乗し、そのうち1名は女性である可能性を示唆した。さらに、2020年までに有人宇宙船を月面着陸へという意欲も見せている。具体的には、2007年までに月周回探査機の打上げ、2010年までに無人月探査機の月面着陸、2020年までに有人宇宙船の月面着陸を実施すると述べた。
