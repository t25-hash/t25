# 2・1・1 地球周辺の宇宙環境とその影響

> **章**: 第2章 宇宙環境の最前線
> **節**: 2・1 地球周辺の宇宙環境

宇宙環境は、地上に比べほぼ完璧な真空状態といえるが、同時にそこは希薄な(1cm当たり数個の)プラズマ(plasma)に満たされている。地球周辺には太陽コロナホールから高速(約300~700km/s)で吹き出す太陽風(solar wind)プラズマ(1cm当たり3.2~20個、平均密度8.7個)、太陽からの真空紫外線(オゾン層下の地表には存在しない、波長200nm以下の真空中にしかない紫外線)で大気が電離してできた電離圏プラズマ(高度約250~300kmに密度のピーク)がある。宇宙機や有人活動に影響を及ぼすものとして、電子、陽子、重イオンなどの高エネルギー放射線粒子が、銀河宇宙線(galactic cosmic ray)や太陽フレア(solar flare)粒子および放射線帯[バン・アレン帯(Van-Allenbelts)ともいう]として存在する。この高エネルギー放射線が多いことには注意が必要で、地上で使用して問題のない半導体部品が宇宙で劣化、誤動作や故障を起こしたり、帯電・放電で衛星が故障を起こしたりするため、その防止対策が必要になる。また、宇宙飛行士には被ばく管理が必ず必要となる。低高度(約250~650km)には原子状酸素を主成分とする大気から解離した希薄中性ガス(rarefied gas)があるが、希薄なガスでも、宇宙機が速度約8km/sで飛行する高速飛翔体環境においては、その表面に強い相互作用を与えるので、原子状酸素による表面の切削や酸化、NOガス等によるシャトルグロー発光等の影響が出る。スペースデブリやマイクロメテオロイド(micrometeoroid)においても高速衝突問題があり、加えて低高度では原子状酸素との相乗(シナジー)効果もあることが指摘されている。人工衛星の軌道上故障のうち約20~25%は宇宙環境に起因するという統計が出されている。この統計値は宇宙開発が進んだにもかかわらず、割合を減少させることができていない。その理由は、新しく開発された電子デバイスや高感度センサほど宇宙環境、特に放射線に対する耐性が弱くなっている傾向があること、そしてそれらが宇宙機の基幹部分の制御中枢を担うようになったことにある。耐環境設計だけでなく、部品選定においても耐環境性、耐放射線性などを評価する必要がある。最近の課題として、宇宙機の最適コスト設計のため、不必要な出費を生む宇宙環境の過剰仕様や過剰防御設計を抑えるような、できるだけ精度の高い宇宙環境モデルが要求されている。この精度のよいモデル計算により、高価で購入が困難な耐放射線性半導体部品(radiation-hard)に代えて、安価で高性能・高機能の一般民生用半導体部品(COTS : commercial offthe shelf)の使用が、適切な放射線遮へい厚の採用により可能となっている。a. 大気宇宙環境と定義する高度100km以上の大気は、地表より圧力が約7~14けた低くなる(100~1000km)。その圧力(密度)は11年の太陽活動周期(solar activity cycle)を受け変動する(図2・1)。極大期には加熱された大気が上空まで膨張して圧力が上がり[極小期(solar minimum)との差は1.5~3けた程度]、太陽フレア発生時には短期的に圧力(密度)が急増することがある。極大期(solar maximum)は低高度のスペースデブリ(宇宙ごみ、space debris)や衛星の残骸の落下が顕著になるので、11年に1回の宇宙の大掃除(自然浄化作用)といわれている。希薄大気で、なぜ大気抵抗が問題になるかといえば、衛星が速度約8km/sの高速飛行なので、その速度の2乗に比例する大気抵抗は無視できない力となるからである。大気抵抗で減速して高度が落ちると、高度の減少分に比例して速度が増加し、その2乗の大気抵抗を受けてさらに減速するという正のフィードバックが働き、衛星は急11―7激に落下する。それゆえ低高度衛星の大気抵抗に対する軌道高度保持は重要で、このための姿勢軌道制御系設計、推進系設計、運用計画等は寿命に直接関係する設計事項となる。大気抵抗の計算式を示す。ここで、大気抵抗を(N)、大気密度をρ(kg/m)、衛星の速度を(m/s)、衛星の進行方向からの断面積を(m)とする。大気抵抗係数욃は、衛星の形状で定まる値で、1.9~2.6の範囲を取り、典型的な値としては2.20が使われる。=12ρ욃(2・1)高度約100~500kmの範囲の大気は熱圏(thermosphere)に属する。その名称は、太陽の極端紫外線(EUV magnetism)宇宙環境において、地磁気は銀河宇宙線や太陽フレア粒子等の荷電粒子が地球に進入するのを防ぐ磁気障壁(バリア)または磁気シールドとして働く。その効果は、磁場による曲げられにくさ、すなわち磁気剛度(magneticrigidity)と呼ばれ、放射線粒子の運動量を電荷量で割った値として定義(単位:GV)される。地表のある地点に鉛直方向から入射する荷電粒子が入射可能な磁気剛度を、その地点の切断磁気剛度(cut offrigidity)といい、図2・2に示す。図2・2から切断磁気剛度は、赤道域で14~17GV、東京は11GV、北極や南極域では0.2GV となり、中・低緯度は地磁気シールドで銀河宇宙線の高エネルギー荷電粒子の接近を妨げる一方、高緯度域では低エネルギー荷電粒子まで外部から進入できることがわかる。また、地磁気は放射線帯の要因であり、その形状を決定する。地磁気は自転軸に対し、11.5°傾いた磁気ダイポール(magneticdipole)で近似できる。地心距離、磁気緯度θでの磁束密度(nT)は、次式で表せる。=1+3sinθ(2・7)ここで、の値は7.9×10nTcm、または30400nT×욄、ただし욄は平均地球半径。近似したダイポールは偏心していて、地心から日本の方向に約440kmずれている。したがって、反対側のブラジル側に局所的に低磁場領域ができる。これを南大西洋異常地帯(SAA : South Atlantic Anomaly)と呼び、放射線帯・内帯の陽子がこの領域のみ下層まで降りていて、そこを通過する低高度衛星への放射線障害の要因を作っている。磁気ダイポールは安定したものでなく、約73万年前に極性が逆転したこと、過去7600万年間に171回も極性が逆転したことが、海底地殻や火山岩の残留磁気パターンなどからわかっている。現在は約2000年後に地磁気が零になる方向で減少する時期にある。地球内部磁気の参照モデルとして、=10次のシュミット(Schmidt)球関数展開係数の数値表を5年ごとに改訂しているIGRF(International Geomagnetic Reference Field)モデルが推奨される。地球半径の約4倍以上の高高度では、ダイポール磁場は太陽風の相互作用を受け、非対称にひずみ、彗星の尾のような形状になる。この境界内が磁気圏であり、太陽方向が地球半径の約10倍、反対図2・2銀河宇宙線の切断磁気剛度マップ(単位:GV)側が地球半径の約1000倍以上になる。この磁気圏磁場モデルとしてチガネンコ・ウスマノフ(Tyganenko-Usmanov)モデル、オルソン・フィツラー(Olson-Pfitzer)モデルが推奨される。軌道高度の低い衛星ほど、地球磁場の影響を受け、衛星の姿勢の外乱要因の一つになる。したがって、低高度軌道衛星では衛星のもつ磁気モーメントを少なく(磁性材料部品の排除、電流ループの面積をできるだけ小さくするなどの磁気管理)するのが望ましく、できあがった人工衛星は磁気モーメントを計測し、必要に応じ、消磁したり、永久磁石を取付けてベクトル合成で磁気モーメントを零にする場合がある。逆に地球磁場との相互作用を積極的に使う方法として、衛星内に電磁石(磁気トルカ)を搭載して衛星の磁気モーメントを可変制御する磁気姿勢制御や、姿勢安定用のジャイロのアンローディング(ジャイロに蓄積された角運動量に、逆ベクトルの角運動量を磁気で与え、ジャイロの回転数を落とす)が利用されている。c. 重力環境衛星内は、無重力、正確には重心位置のみで、それ以外は重心位置からの距離に比例した微小重力(マイクロ・ジー、microgravity)があるが、三軸姿勢制御衛星の内部では、固定していないと物体は浮遊する。そのため電気的・物理的接触、コンタミネーション、コールドウェルディング(清浄金属面どうしの凝着)等が起きる可能性がある。対策として、コーティング、ポッティング、可動物への固体潤滑剤の塗布などが必要である。重力場モデルは、球面調和関数の70×70次を使うJGM-2モデルが推奨される[JGM-2の詳細はNASA
