# 2・1・1 地球周辺の宇宙環境とその影響

> **章**: 第2章 宇宙環境の最前線
> **節**: 2・1 地球周辺の宇宙環境

宇宙環境は，地上に比べほぼ完璧な真空状態といえるが，同時に
そこは希薄な（1cm当たり数個の）プラズマ（plasma）に満た
されている．地球周辺には太陽コロナホールから高速（約
300～700km/ s）で吹き出す太陽風（solar wind）プラズマ（1
cm当たり3.2～20個，平均密度8.7個），太陽からの真空紫外線
（オゾン層下の地表には存在しない，波長200nm以下の真空中に
しかない紫外線）で大気が電離してできた電離圏プラズマ（高度約
250～300kmに密度のピーク）がある．宇宙機や有人活動に影響
を及ぼすものとして，電子，陽子，重イオンなどの高エネルギー放
射線粒子が，銀河宇宙線（galactic cosmic ray）や太陽フレア
（solar flare）粒子および放射線帯［バン・アレン帯（Van- Allen
belts）ともいう］として存在する．この高エネルギー放射線が多
いことには注意が必要で，地上で使用して問題のない半導体部品が
宇宙で劣化，誤動作や故障を起こしたり，帯電・放電で衛星が故障
を起こしたりするため，その防止対策が必要になる．また，宇宙飛
行士には被ばく管理が必ず必要となる．
低高度（約250～650km）には原子状酸素を主成分とする大気
から解離した希薄中性ガス（rarefied gas）があるが，希薄なガス
でも，宇宙機が速度約8km/ sで飛行する高速飛翔体環境において
は，その表面に強い相互作用を与えるので，原子状酸素による表面
の切削や酸化，NOガス等によるシャトルグロー発光等の影響が出
る．スペースデブリやマイクロメテオロイド（micrometeoroid）
においても高速衝突問題があり，加えて低高度では原子状酸素との
相乗（シナジー）効果もあることが指摘されている．
