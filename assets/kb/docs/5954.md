# 2・1・2 原子状酸素

> **章**: 第2章 宇宙環境の最前線
> **節**: 2・1 地球周辺の宇宙環境

陽子線、電子線をはじめとする荷電粒子放射線、太陽からの紫外線、熱サイクルなど、宇宙機のさらされる宇宙環境はさまざまである。これらの宇宙環境要因によって宇宙用材料に劣化が引き起こされる。ここでは、これらの中でもスペースシャトルの飛行が始まって以降その重要性が認識された、低高度を飛行する宇宙機において深刻な劣化要因である原子状酸素(AO: atomic oxygen)に焦点を当てて解説する。a. 原子状酸素による宇宙開発への影響高層大気に多くの原子状酸素が存在することは古くから知られていたが、その存在が重要視されるに至ったのは、1983年にスペースシャトル飛行中のシャトルグローという現象が報告されるのと同時に表面化した、原子状酸素による有機材料のエロージョン(erosion)の問題のためである。地球周回低軌道(LEO: low earth orbit)を周回する宇宙機は、この高層大気中を約8km/sで通過するため、原子状酸素が宇宙機の熱制御材料(thermal control material)をはじめとする宇宙用材料に衝突し、その相対エネルギーは約5eV 程度になる。この原子状酸素の衝突によって、多くの宇宙用材料、特に有機材料が著しい劣化を受けることが明らかになっており、現在宇宙開発における重大な懸念事項の一つである。原子状酸素による宇宙用材料のエロージョンが報告されて以来、地球周回低軌道上における耐宇宙環境性評価、特に耐原子状酸素性の評価を目的として、LDEF(Long Duration Exposure Facility)やスペースシャトル利用実験など、数多くの軌道上ばく露実験が実施され、多くのばく露試料が回収、分析された。これら飛行ばく露試験によって原子状酸素によるエロージョンの現象論的把握が進むとともに、高い耐原子状酸素性を有する材料の選定のための参照データを提供する目的で、各種材料の原子状酸素に対する反応効率(reaction efficiency,1個の原子状酸素によって失われる材料の体積によって定義される、単位cm/atm)が報告されてきた。これにより、おおむね各種材料の耐原子状酸素性の傾向は把握できたものの、原子状酸素によるエロージョンの第一義的要因を明らかにするには至らなかった。なぜならば、化学反応がそのエロージョンメカニズムの第一義的な要因と考えられる原子状酸素においては、・Kapton좲:Kaptonはデュポン社の登録商標です。11―9宇宙用材料が同時にばく露される放射線や紫外線の関与によって相乗効果が生じると予想され、飛行ばく露試験のような複合環境下では各種宇宙環境要因の影響を分離できない困難さがあるからである。このため、飛行ばく露試験において紫外線を遮断するカバーを取付け、日陰時のみ原子状酸素を照射するなどの措置がとられている。しかし、軌道上で着目する照射源以外の第三者的照射源による影響を完全に取り去ることは困難であり、エロージョンメカニズムの解明や相乗効果の有無に関する研究のためには、100%原子に解離した酸素ビームの存在が必要であった。このことから、飛行ばく露試験と並行して地上照射試験による耐原子状酸素性評価のためにさまざまな方式の原子状酸素照射装置が開発され、より多くのエロージョンレートの報告、耐原子状酸素性を有するコーティングや新材料の開発が精力的に行われた。c. カプトン膜の質量減少上記に述べた飛行ばく露試験、および地上照射試験において、各種材料における原子状酸素のエロージョンレート、すなわち反応効率を評価する際に用いられている参照試料に、ポリイミドフィルムであるカプトン(Kapton좲)がある。カプトンは高い耐熱性および耐放射線性を有することから宇宙機の熱制御材料として広く用いられており、その反応効率3.0×10cm/atmはさまざまな材料の耐原子状酸素性を評価するに当たり頻繁に引用され、照射場のモニタ材としてよく用いられている。反応効率は、原子状酸素の入射角、フルエンス率、並進エネルギー、照射の対象となる試料の温度、紫外線や放射線などの宇宙空間の各種環境条件との相乗効果など、多くの要因に影響を受けると考えられるが、エネルギー依存性および温度依存性を含めたそのエロージョン過程についての研究は数少ない。ここではモニタ材として用いられるカプトン膜に着目し、質量減少に至るまでの初期エロージョン過程について述べた後、エネルギー依存性と温度依存性についてこれまでに行われている研究結果について概観する。. 初期エロージョン過程原子状酸素によるカプトン膜の初期エロージョン過程は、水晶振動子上に形成した、カプトンと類似した分子構造のポリイミド膜に原子状酸素を照射した際の質量変化を、ナノグラムオーダの質量分解能でその場測定することによって研究されている。その結果、低地球軌道上と同じ5eV のエネルギーを有する原子状酸素を照射した場合、照射初期段階ではカプトン膜の質量は増加し、その後2.0×10atm/cm以上の照射線量で減少に転じることが明らかになっている。その一方で、あらかじめ原子状酸素を照射したカプトン膜では、照射再開直後の質量増加は認められていない。このことから、エロージョンが起きるためにはまず表面の酸化が必要不可欠であると考えられている。したがって、原子状酸素の照射初期段階ではカプトン膜の表面酸化により質量増加が生じ、最表面が酸化されると質量減少が引き起こされると考えられる。上述のように、原子状酸素の照射初期では5eV のエネルギーで質量増加が起きる。この初期過程は、エネルギーが異なる場合どのような差違が生じるのであろうか。木之下らによる実験結果を図2・5に示す。原子状酸素照射の初期過程では5eV の運動エネルギーで質量増加が認められるのに対して、50eV では大きな質量減少が観察されている。一般に5eV のような低運動エネルギーでは物理的なスパッタリング(sputtering)の効果は無視できることから、原子状酸素の照射初期においては30eV 以下のエネルギーでは原子状酸素の吸着による質量増加が、30eV 以上のエネルギーでは物理的なスパッタリングによる質量減少が起きると考えられている。.
