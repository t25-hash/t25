# 2・1・3 プラズマ・放射線環境と帯電・放電

> **章**: 第2章 宇宙環境の最前線
> **節**: 2・1 に真空複合環境試験設備の概要を示す。真空複合環境試験設備

プラズマ・放射線環境の重要性は、宇宙開発の遂行と歩調を合わせて認識されたといってよく、おもなイベントを時系列的に述べる。(1) 地球近傍の放射線帯[radiation belts、バン・アレン帯(Van-Allen belts)]が認識されたのは、人工衛星が打ち上げられてからであり、1958年のエクスプローラ1号、3号衛星の観測結果により初めて確認された。(2) 1962年には高空核実験(スターフィッシュ実験)により人工的に発生(β崩壊)した電子が放射線帯の内帯に注入されて、その後数年間捕捉された。このとき、軌道上を飛んでいた人工衛星のうち、耐放射線対策を考慮していた衛星(テルスター1号)以外は、数日か数週間で機能を失い停止した。この事件を契機として、宇宙放射線の次の2種類の累積的影響への対策の必要性が認識された。第1は放射線粒子の非電離効果で、太陽電池のシリコン結晶に弾性散乱のはじき出し効果(結晶への変位損傷)を与える結果生じる発生電力の減少である。第2は放射線の電離効果で、半導体デバイス中に電荷が蓄積する、総吸収線量(total dose)効果の結果生じる半導体部品のリーク電流の増大や特性劣化による故障である。(3) 1969年に、アポロ11号のオルドリン宇宙飛行士が初めて報告したライトフラッシュ(LF: light flash)現象は、暗い状態で、目を開けても閉じても、無色の発光(星状、直線状等)が見える現象である。その後、多くの宇宙飛行士が確認している。これらの現象は高エネルギー粒子(重イオンや陽子)による影響であることが、マクナルティ(McNulty)により地上のサイクロトロン試験で実証されたが、次に述べるシングルイベントに似た現象であった。(4) 1975年には、インテルサットIV号衛星のフリップフロップ回路ICの誤動作が、宇宙の高エネルギー粒子(重イオンまたは陽子)1個の進入により半導体デバイス内に発生・収集される電荷量に起因するシングルイベント(single event)であることが確認された。(5) 太陽活動極大期の1989年9月29日と10月19日に記録的な大型太陽フレアがあり、「ひまわり3号」衛星が静穏期の約3年分に相当する太陽電池の劣化を一時的に受け、多くの衛星でシングルイベント発生数の急増現象が観測された。このとき、ロシアのミール有人宇宙船内では宇宙飛行士が滞在していて、地上からの指示に基づいて、この太陽フレア中は船内の遮へいの最も厚い部分に避難した。同様な太陽フレアによる障害が、2000年の7月14日と、2003年10月期にあった。(6) 太陽活動極大期の1991年3月24日に、アメリカのクレス衛星(CRRES : combined release radiation effects satellite)や「あけぼの」衛星の観測により、磁気嵐(magnetic storm)の衝撃波によって大量の粒子フラックスが、外部から内帯あたりまで注入する現象が観測された。この現象(内帯のダブルピーク)の発見は、放射線帯はかなり安定したものであるという従来の常識を破る結果となった。また、静止軌道周辺には、高エネルギー電子(10~30MeV)の高密度のフラックスが存在することが判明した。(7) 1994年1月20日にカナダの静止通信衛星アニク-E1,E2号機およびインテルサット機構のインテルサットK 号機の合計3機が同時に、姿勢制御系のモーメンタムホイールの制御回路が故障し運用を停止した。E2は復帰に数箇月以上を要し、数千万ドル以上の損失となった。この故障原因は高速太陽風による高エネルギー電子の線量(フラックス)が急増し、約1週間ほど続いた結果、衛星が内部帯電・放電して故障したと考えられている。以上、宇宙放射線問題の歴史的経緯を概観したが、プラズマ・宇宙放射線が人工衛星に与える影響は、古くて新しい課題である。例えば、人工衛星に使われる半導体デバイスの集積度が高くなり、そのため逆に放射線にセンシティブになった。光学センサ類が高感度になることでは、姿勢制御系の回路やセンサ類が放射線に起因する電磁パルスのノイズ等に鋭敏になった。このように、デバイスの技術進歩が結果的に衛星の頑健性の欠如をもたらし、宇宙放射線粒子1個による誤動作や、その総粒子数の変動に起因する帯電による故障を起こしやすくさせている。a. プラズマ環境地球周辺のプラズマ環境は、地球大気、地球磁場、太陽光(主として大気を電離する紫外線)、太陽風(太陽から流れ出るプラズマ流)の四つの主要な要素で支配されている。地表の大気は高度に対してほぼ指数関数的に減少するが、高度数十kmより上空の大気は太陽紫外線によりその一部が電離される。こうして形成される上空のプラズマの層は電離層と呼ばれ、高度1000km付近まで広がっている。さらに外側は磁気圏と呼ばれ、太陽風と地球磁場との相互作用により地球側を頭部とした巨大な吹流しのような形の領域を形成している。この領域の内部では、電離層から拡散した低温プラズマと太陽風や宇宙線起源の高エネルギープラズマが希薄ではあるが存在している。その外部は太陽風プラズマが高速で流れている。これら地球周辺プラズマ環境の概観を図うち、地球大気と地球磁場は静的であるが、太陽光の電離作用は昼間のみであり、太陽風は太陽活動によって大きく変動するため、地球周辺のプラズマ環境は全体として時間とともにダイナミックに変化する。. 磁気圏(magnetosphere)磁気圏は、地球が固有のダイポール磁場をもっているために形成されるプラズマ領域である。太陽風の圧力と地球ダイポール磁場の圧力が釣り合う境界の内部では太陽風が排除されるが、その領域は空虚ではない。むしろ多様な形態の荷電粒子がダイナミックなプラズマ環境を形成している。太陽風の上流では太陽風の圧力によって地球磁場が押され、その境界は地球半径の10倍程度のところに存在する。太陽風の下流側(磁気圏尾部)では地球磁場が太陽風によって地球半径の1000倍以上も流されている。磁気圏尾部では北極からの磁場と南極からの磁場が平行になり、中央に磁気中性面をもつ反平行磁場構造を構成する。磁気中性面付近には太陽風から供給された1~5keV 程度の高エネルギーのプラズマ(密度10~10/m)がシート状に存在している。また、尾部を含んで南北の領域には電離層起源の5×10/m程度の希薄な低エネルギーのプラズマが存在し、ローブ領域と呼ばれている。磁気圏尾部では太陽風と磁気圏の相互作用によりエネルギーの蓄積・解放が活発に行われる。解放されたエネルギーは磁気中性面付近のプラズマの加速を生じる。加速された高エネルギーの荷電粒子は磁力線に沿って地球の極域に達し、電離層でオーロラを発生させる。地球周辺の磁気圏ではダイポール磁場の閉じたドーナツ状の領域に、電離層から流出した低エネルギー(1eV 程度)のプラズマと、太陽風や宇宙線起源の高エネルギープラズマ(1keV以上)が存在している。低エネルギープラズマの閉じ込められている領域はプラズマ圏と呼ばれている。プラズマ圏のプラズマ密度は、電離層とつながる高度1000km付近では10/m程度であり、地球磁場の強度にほぼ比例して減少し、地球半径の4倍付近のプラズマ圏境界では10/m程度となる。プラズマ圏の外側ではプラズマ密度は急激に減少する。高エネルギープラズマの捕捉されている領域は放射線帯と呼ばれ、地球に近い内帯(高度1000kmから1万km付近まで)と外側の外帯(地球半径の数倍付近)に区分される。内帯は1MeV 銀河宇宙線銀河系内(外も)で、超新星の爆発等により発生し、銀河系内を伝搬中に加速を受け、水素、炭素、酸素、鉄等の元素から軌道電子をほとんどすべてはぎ取られた裸の原子核が、原子番号と同じ電荷をもって光速に近い速度となった粒子を、銀河宇宙線(GCR : galactic cosmic ray)と呼ぶ。エネルギー範囲は10eV(10MeV)~10eV といわれていたが、10eV 以上の粒子も過去7例観測されている。一般的な材料による遮へいは困難だが、地球磁場が荷電粒子の経路を曲げて、あるエネルギー以上の粒子を跳ね返すことで遮へいしている(2・1・1・b項参照)。銀河宇宙線の平均元素組成は、83%が陽子、13%がα線、3%が電子、残りの1%が重イオンである。重イオンで多いのはC,N,O,Ne,Mg,Si、そしてFeで、鉄(Fe)を超える重元素(超重核という)の存在数は鉄に比べはるかに(5~6けた)減少する。シングルイベントの原因になる電離発生量は、電荷の二乗に比例するので、鉄イオンが一般的な最悪条件となる。銀河宇宙線は、惑星空間で定常的に存在するが、地球周辺では太陽活動の11年(厳密には22年)周期の変動を受け、極大期には太陽磁場や太陽からの粒子で遮へいされるのでフルエンスが減少する。反対に極小期では銀河宇宙線のフラックスが最大条件となる。. 太陽宇宙線(SCR : solar cosmic rays,SEP: solar ener-getic particles)太陽宇宙線は、太陽表面の黒点(群)に蓄えられた磁場のエネルギーが、太陽フレアや太陽コロナ質量放出(CME: coronal mass ejection)により解放されることによって加速・放出される、高エネルギー荷電粒子(最大約500MeV)をいう。太陽宇宙線は定常的な銀河宇宙線と違い、突発的に起こる性質と、フレアごとにエネルギーのスペクトラムと組成が異なるという性格をもち、フレアの発生予測が困難という特質をもつ。その平均組成は銀河宇宙線の組成に近い。太陽フレアは、太陽の黒点(sunspot)数[またはウルフ数(Wolf number)]11年周期の中で極大期(黒点数の多い約7年間)に発生する場合が多いが(図2・20)、黒点周期も長期的に変動している。文献(39)より引用した図2・20に黒点数の周期と太陽フレアの発生(棒グラフで示す)の関係を示す。フレアの棒グラフで、破線の長さは10MeV 以上の陽子フルエンス、実線の長さは30MeV 帯電によるコンタミネーション一般にプラズマ中の物体は、周辺のプラズマ電位と違った電位となる。その電位差のある領域をシース(sheath)といい、その厚さは、以下に定義するデバイ長λの数倍程度である。低高度衛星ではデバイ長はcmオーダになるが、静止衛星や高高度衛星では100mオーダとなる(衛星から見て、このデバイ長の数倍の外側は電気的に中性と考えてよい)。したがって、静止衛星や高高度衛星では、衛星からのアウトガスが太陽紫外線でイオン化し、衛星が負に帯電した状態のときに、デバイ長の数倍程度の範囲内で再付着し、コンタミネーションの原因になるので注意が必要である。.
