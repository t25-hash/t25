# 2・2・1 月(Moon)の環境

> **章**: 第2章 宇宙環境の最前線
> **節**: 2・2 月・火星・金星

(1) 基本データ地球と月の平均距離:384400km,公転周期:27.32日,自転周期:27.32日,赤道半径:1738km,質量:0.012300(地球=1,7.35×10kg),平均密度:3340kg/m,重力の加速度:0.17(1.622m/s,1=9.8m/s)脱出速度:2.38km/s(地球では11.18km/s)(2) 大気(組成)大気をもたない.大気密度:昼2×10cm,夜2×10cm(<10atm)He,Na(≦50cm),シリカ,酸化カルシウム等の微粒子(3) 温度環境昼125°C,夜-170°C太陽からのふく射量:1.0[1.37kW/m=1.96cal/(cm・min)]月面には太陽からのふく射は地球上空とほぼ同量届き,大気がないため吸収を受けることなく全量が月面に到達し,月面まで宇宙空間が接近していることになる.(4) 地形月の表側にはクレータで覆われ明るく見える高地と暗く見える海が存在するが,裏側には海がほとんど見られない.海の岩石は高地の岩石より重く,重い物質が集中している面を地球に向けている.惑星の表面が半球ごとに異なった様子を示すのは,火星や地球にも見られる.表面には直径1mmから1000kmに及ぶ隕石の衝突によるクレータが無数に存在する.ティコ・クレータは周りに大規模な光条があり有名である.月の海は17%程度で,最大のものは嵐の大洋で北緯40°から南緯20°に及ぶ.月面にも地球のように山脈・地溝・断崖が見られ,クレータの形成・溶岩流に関連して形成されたと推定される.
