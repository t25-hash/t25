# 3・2・1 M-V

> **章**: 第3章 推進工学
> **節**: 3・2 固体ロケットモータ

文部科学省宇宙科学研究所は,科学衛星打上げ用に独自に固体ロケット(solid rocket)を開発してきた.その経緯は1955年のペンシルロケットの実験開始に端を発し,カッパ,ラムダ,ミューと脈々と続き,打上げ能力の増強のみならず各要素技術の向上に努めてきた.M-V(ミュー・ファイブ)ロケットは現在の主力ロケットで,地球周回低軌道に1.8tの衛星投入能力をもつ全段固体の3段式ロケットである.推進系は,それ以前の主力ロケットM-3SIIから一新され,全段新規開発された.開発は1990年より着手され,1997年に1号機が電波天文観測衛星ミューゼスB「はるか」を,1998年に3号機が火星上層大気観測衛星プラネットB「のぞみ」を成功裏に打ち上げた.2003年に打ち上げられる5号機より,第2段ロケットの新規開発を筆頭に多くの改良が施される.ここでは,5号機の機体構成をもとにM-V ロケット推進系の概略を説明する.M-V ロケットは全長30.8m,代表直径2.5m,全備質量約140tの全段固体3段構成で,高エネルギーミッションに対応してオプションで4段キックステージモータが加えられる.モータケースは1段にのみ高張力鋼製金属ケースを採用するが,2段以上ではすべてエポキシ樹脂を用いた炭素繊維強化プラスチック製フィラメントワインディング(CE-FRP製FW)ケースを採用し,軽量・高性能化を図っている.なお,モータケースに関する詳細は7・6節を参照されたい.ノズルは全段で,スロート部材に従前はグラファイト材を使用していたが,5号機からは,より熱構造的に安定で信頼性の高い三次元織りカーボン/カーボン複合材(3D-C/C)を採用している.開口部ライナにはフェノール樹脂を用いたカーボン・フェノール繊維強化樹脂(CP-FRP),およびシリカ繊維強化プラスチック(SFRP)を使用している.また,3・4段には高性能化のため伸展ノズルが採用されている.ノズルコンタの設計には,アルミナ粒子の飛行を考慮し,開口部ライナへのアルミナ粒子の衝突を避ける工夫が凝らされている.推力方向制御(TVC: thrust vector control)は第1~第3段が可動ノズル方式を採用しており,さらに固体推進薬を用いたガスジェネレータ,ヒドラジンを用いたスラスタと組合わせて三軸制御を行う.第4段はTVC装置は付かずに,小型固体モータを使ったスピン安定化が図られている.推進薬は酸化剤に過塩素酸アンモニウム(AP),金属燃料にアルミニウム,燃料兼結合剤に末端水酸基ポリブタジエン(HTPB)を主3成分としたコンポジット推進薬(composite propellant)を使用している.推進薬は基本的には1・2段用と3・4段用の2種に分けられるが,主3成分の質量分率はすべて同じで,高性能化のためにアルミニウムの含有率が20mass% にまで高められている.AP粒子の大粒子,中粒子,細粒子の配合比は最適化され,モータ燃焼後のケース内スラグ残留物量を極少化している.また,1・2段用推進薬は飛翔中の重力損失を低めるために,燃焼触媒として酸図3・27 リニアエアロスパイクエンジン11―32宇宙機器・システム化鉄を添加し燃焼速度を高めている.点火器は1~4段用すべてモータ型で,点火薬には1・2段モータ用推進薬を共通使用している.1・2段用点火器は同一で前方固定型である.3・4段用点火器は後方着火式で,ノズルスカート内にクロージャとともに内装される.モータ点火後,モータ圧力が上昇し,ある圧力に達するとクロージャをノズルライナ部に固定するシアピンが切断され,点火器はクロージャとともに後方に吹き抜ける.吹抜けに際しては,クロージャが円滑に吹き抜けること,モータ内圧の上昇に悪影響が出ないこと,の2点について設計には細心の注意が払われている.後方着火方式の採用により,モータ前方をすべて推進薬で充てんすることが可能となり,モータの推進性能向上に寄与している.点火器の設計には,モータ着火遅れ時間に関する基礎研究の結果が反映されており,点火器作動時間がモータ予想着火遅れ時間の3倍以上という基準を満たし,着火に関して十分な安全率が確保されている.ケースインシュレーションにはM-3SIIロケットまではアスベスト入りイソプレンゴム(IR)が使われていたが,アスベストが発がん性を有するためM-V ロケットからはアラミド繊維充てんエチレンプロピレンジエンモノマ(EPDM)ゴムが採用されている.全機概要図を図3・28に,各段の主要諸元を表3・6にまとめる.ただし,4段には5号機用キックモータKM-V 2モータの諸元を示す.a.
