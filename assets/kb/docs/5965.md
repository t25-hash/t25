# 3・2・2 H- A ロケット6号機打上げ失敗の原因と対策

> **章**: 第3章 推進工学
> **節**: 3・30 に示す.

2003年11月に打ち上げられたH-IIA ロケット6号機は,右側のSRB-A の分離に失敗したため,衛星の軌道投入に必要な高度および速度が得られないことから,打上げ後約10分53秒に地上からの指令により破壊された.SRB-A の分離ができなかった原因は,打上げ後約62秒にSRB-A のノズルから燃焼ガスが後部アダプタ内に漏洩し,このガスによって前方ヨーブレス分離用密封型導爆線(CDF: confined detonating fuse)が加熱され機能を喪失したためと推定された.ノズルから燃焼ガスが漏洩した原因は,3D-C/C製スロートインサートとイグジットコーンのCFRP製ライナとの表面後退の差によりライナ前端部に発生する一様な段差,推進薬光芒の影響による燃焼ガスの渦による表面後退[surface recession,局所エロージョン(excessive local erosion)],局所エロージョンをさらに加速させる現象(ライナ炭化層の層間はく離,深い溝に流れ込む燃焼ガスによる表面後退)が継起することによりライナが深く削られ,燃焼ガスがアルミニウム合金製ホルダに到達し,その熱によってホルダが溶融・破孔したためと推定された.ノズル部の対策については,局所エロージョンの発生を極力排除し,原因の可能性を否定できなかった要因についても対策を検討し,信頼性の高い設計変更を行うことを基本方針とした.具体的には以下の項目についてトレードオフを行い設計を決定する.(1) 燃焼パターンの見直し:高燃焼圧力化したことが局所エロージョン発生の背景要因になっていることを踏まえ,燃焼圧力の低減等,局所エロージョンの発生の低減効果をもたらすような燃焼パターンを検討する.(2) ノズル形状の見直し:イグジットコーン部の初期立上り角を増加することにより,内壁静圧等流体力学的環境を早期に緩和する.これは層間はく離と脱落の防止にも寄与する.(3) スロートインサート(3D-C/C)範囲の拡大:3D-C/Cの範囲を下流側に広げることによりCFRP製ライナを熱による負荷がより低い側に配置し,ライナの表面後退を低減する.(4) ライナの板厚設計の見直し:地上燃焼試験によりデータを蓄積し,解析による評価を踏まえ,必要な板厚余裕も含む新たな設計基準により板厚を設定する.図3・38 KM-V 2モータ真空推力・内圧パターン図3・37 KM-V 2モータ概要図(5) 製造・検査の改善:CFRPの材料特性を安定化させるため,特性を十分に把握・特定するとともに,性能等の合否判定基準に関しても妥当性を確認する.ライナ等の欠け等の可能性をいっそう低減するため,製造・検査の改善を検討する.また,今回の事故では,SRB-A の分離機構が冗長構成であったにもかかわらず,分離に失敗したことから,CDFを含むぎ(艤)装については,冗長構成の視点から適切な配置であったかどうかについて再検証する.
