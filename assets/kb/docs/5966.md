# 3・2・3 ハイブリッドロケット

> **章**: 第3章 推進工学
> **節**: 3・30 に示す。

ハイブリッドロケット(hybrid rocket)は推進薬に液体と固体の組合せを用いるロケットで、液体推進薬を用いる液体ロケット、固体推進薬を用いる固体ロケットに対して、両者の混成(hybrid)であることからハイブリッド推進薬ロケットエンジン(hybrid propellant rocket engine)、略してハイブリッドロケットといわれる。液体酸化剤と固体燃料、および液体燃料と固体酸化剤の二とおりの組合せが考えられるが、通常使用されるのは前者の、液体酸化剤と固体燃焼の組合せである。a. 特徴ハイブリッドロケットは以下に示すような長所をもつ。(1) 安全:燃料と酸化剤が完全に分離されているため、事故等による爆発の危険性がない。(2) 推力制御・停止および再点火:液体酸化剤の流量を制御することにより、推力制御やオンオフ制御を行うことが可能である。(3) 低公害:燃焼ガスに塩酸や酸化アルミニウム等の汚染物質を含まない推進薬の組合せを選択することが容易である。(4) ロバスト性:固体燃料にクラックが生じても不感である。(5) 低温度感度:推進薬温度が燃焼速度にほとんど影響を及ぼさないため、エンジン性能が打上げ環境温度の影響を受けない。(6) 安価:燃料、酸化剤ともに安価であり、供給系が1系統で済むため、エンジンが安価である。また、エンジンの安全性により安全対策の簡略化や市販レベルの部品の選択が可能であり、システム全体でも低コストとなる。一方、短所をまとめると以下のようになる。(1) 低推力:固体燃料の燃焼速度が低いため、大推力発生には不向きである。(2) 低充てん密度:固体燃料内部に液体酸化剤および燃焼ガスの流路を設ける必要があるため、固体燃料の充てん密度が低くなる。(3) 低燃焼効率:燃料と酸化剤の混合不十分および燃え残りの発生により、低燃焼効率になりやすい。(4) 混合比(O/F)シフト:燃焼の進行により固体燃料の形状が変化するため、O/Fが燃焼経過時間に従って変化し、比推力の低下を招く。b. 燃焼速度ハイブリッドロケットでは、霧化・ガス化した酸化剤および燃焼ガスが固体燃料のポート内を流れ、ポート側壁近傍で燃焼が起こる。基本的には酸化剤とガス化した燃料成分との拡散燃焼である。燃焼面近傍の様子を図3・40に示す。固体燃料表面近傍の乱流境界層内に拡散火炎が形成される。酸化剤は主流から拡散により火炎の外側から供給され、燃料は固体燃料表面のガス化により火炎の内側から供給される。固体燃料の燃焼速度は固体燃料への熱流束および単位体積の固体燃料がガス化するのに必要な熱量に依存し、固体燃料への熱流速はポート内の酸化剤流量密度でほぼ決まる。したがって、固体燃料の燃焼速度(mm/s)は酸化剤の流量密度(単位断面積当たりの酸化剤流量)[kg/(m・s)]を用いて下記のように簡易的に表されることが多い。=욗(3・1)ここで、およびは定数で、実験的に決定される。ポート上流で境界層が十分に発達していない領域では燃焼速度が相対的に高くなる(ブラジウス効果、Blasius effect)が、この効果を含めるために次式が採用されることもある。=욗욖(3・2)ここで、(m)はポート先端からの距離、は実験的に決定される定数である。e.
