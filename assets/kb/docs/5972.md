# 4・1・1 月軌道

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・1 月・火星・L 5軌道

本項では，月（Moon）探査における軌道計画について述べる．
各論に入る前に，まず，月探査軌道計画の概要について述べる．
軌道計画（trajectory planning）とは，探査機の出発地から目的
地までの飛行経路を計画することである．飛行経路の多くは，接続
された複数の軌道からなり，それらの接続点では軌道制御がなされ
ることになる．この観点から，軌道計画とは，目的地までの経路を
構成する各軌道を設計し，それを実現するための軌道制御（タイミ
ングと制御量）を明らかにすることともいえる．
軌道計画には二つの目的がある．第一の目的は，想定している探
査を実現するために必要なエネルギー，速度増分（以下， と呼
ぶ）を算出することである．ロケットに要求されるエネルギーから
は，ロケットにより投入可能な質量が算出され，探査機初期質量の
上限となる．軌道上で必要な からは，探査機に搭載する推進
薬質量が算出される．初期質量，推進薬質量は探査機の重要な設計
条件となる．軌道計画の第二の目的は，運用計画への反映である．
計画で得られた軌道から，軌道制御を実施する場所，タイミングが
明らかになるほか，探査機の地上局からの可視性や日照・日陰，打
上げ機会（ロンチウインドウ）など，運用計画策定に必要な情報が
得られる．
月探査軌道（lunar exploration trajectory）の特徴の第一は，
その多様性である．まず探査形態として，フライバイ探査，周回探
査，着陸探査，地球帰還までを含めた往還探査が考えられ，軌道の
構成がそれぞれ異なる．また，月に到達するまでの月遷移部分をと
っても，地球から月に直接到達する軌道から，エネルギー節約のた
めに遠回りをする軌道まで，いくつかの形態がある．月探査軌道の
特徴の第二は，その複雑性である．地球近傍ミッションでは，地球
との二体問題の範囲でその軌道を概略把握できるが，月探査軌道で
は，地球と月の影響圏にまたがる軌道を計画する必要があり，地
球・月双方から離れた領域では，太陽重力の影響も無視できなくな
る．また，探査計画によっては地球，月，太陽の幾何学的関係まで
考慮しなければならない．
月探査の軌道計画は二つの段階に分けられる．第1段階では，探
査計画に対して軌道をどのように構成するかが検討される．さら
に，月探査軌道を構成する各軌道に対する要求，制約，境界条件
（以下，設計条件と呼ぶ）が検討される．探査計画に応じた多様な
月探査軌道はこの段階で構成される．第2段階では，探査軌道を構
成する各軌道やそれを実現する軌道制御の詳細が検討される．ここ
では，第1段階で設定された各軌道に対する設計条件を満足する軌
道が設計され，それを実現する軌道制御量が算出される．月探査軌
道の複雑な要素が考慮されるのはこの段階である．

スイングバイ（swing-by）というのは重力アシスト（gravity
assist），つまり，意図的に探査機をある天体（この場合は月）の
近くを通らせて，その天体の重力によって探査機を加速して軌道を
変更する技術である．英語ではswing-byと書くが，その名前どお
り，月が探査機をつかんで放り投げていることに相当する．スイン
グバイは，探査機の燃料を使わずに大きな軌道の修正を行うことが
できるという利点がある．図4･20は，「のぞみ」が地球脱出前に予
定していた軌道の略図であるが，最後の地球脱出用の加速以前に，
2回の月スイングバイを行っているということ，2回の月スイング
バイの間の軌道がかなり曲がっているということに気がつく．ここ
では，その2回の月スイングバイの意味と軌道が曲がっている原因
について説明したい．
速度の方向と大きさをベクトルで表す．図4･21は最初の月スイ
ングバイの図であるが，大事なことは，探査機が月軌道の内側から
外側へ，かつ，月の進行方向に対して後ろ側を通っているというこ
とである．月から見ると，接近前と後では，月に対する速度の大き
さは変わらないが，月の重力でその相対速度の方向は曲げられる．
その結果，月接近前後を比較すると地球に対する速度の大きさが増
えていることがわかる．図4･22は2回めの月スイングバイの図で，
今度は，探査機は月の軌道の外側から内側へ，また，月の進行方向
の後ろ側を通っている．1回めと同様に，月接近前後では月に対す
る相対速度の大きさは変化せずにその方向だけ変えられる．その結
果，地球に対する速度の大きさが増えている．このように，探査機
は燃料を使わずにエネルギーを獲得することができる．それでは，
どれくらい燃料を節約できるのであろうか．2回の月スイングバイ
で，最後に地球脱出に必要な増速度量を120m/s程度減らすこと
ができる．つまり，直接火星を目指すと3.6km/ s必要なところを
約3.5km/ sで済むことになる．この結果，探査機質量540kgの
うちの推進薬約20kgを節約できる．科学観測機器の質量は全部合
わせて33kgであるので，少しでも推進薬を節約することが大事で
あることがわかる．
それでは，2回の月スイングバイの間の軌道の激しい曲がり方は
どうして起きるのであろうか．それは，太陽の重力によるものであ
る．正確にいうと，探査機は地球が太陽の周りを回る公転運動と遠
心力との相互作用による力を太陽から受ける（潮汐力）．地球から
近いときには，地球の重力が支配的なためこの力は目立たないが，
2回の月スイングバイの間には地球からの距離は170万kmにも達
し，それくらいの距離になると，この力が地球の重力と同等になっ
てきて，このような曲がり方をする．
