# 4・1・3 L 5軌道

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・1 月・火星・L 5軌道

本項では、太陽-地球系の力学的な平衡点であるラグランジュ点(Lagrange points)を利用するミッションの軌道について述べる。ラグランジュ点は第一天体(本項では太陽)および第二天体(本項では地球)からの重力と遠心力とが釣り合う平衡点(equilib-rium points)であり、五つ存在することがわかっている。a項では、これら五つのラグランジュ点の位置とその安定性について述べる。b項では、太陽および地球と正三角形を形成する第4および第5ラグランジュ点(以下、L4,L5点と呼ぶ)の特徴を利用するミッションの例を述べる。c項では、L5点への遷移軌道について述べ、d項ではL5点近傍での宇宙機の運動について述べる。d項を除き、地球公転軌道を円と近似した円制限三体問題(circularrestricted three-body problem)で考える。a. 五つのラグランジュ点と安定性(stability)五つのラグランジュ点を図4・31に示す。二つの天体からの重力と遠心力との釣合い式を解くことにより、三つの直線解(L1,L2,L3点)と二つの正三角形解(L4,L5点)が存在することを示すことができる。ただし、は太陽を原点とし地球方向を正とし11―59第II部基礎編た座標であり、太陽-地球間距離を1とする。L1点の存在範囲:0<<1L2点の存在範囲:1<<2L3点の存在範囲:-1<<0以下に、各直線解に対する五次方程式とその近似解を示す。. L 5点の特徴を利用するミッション例太陽-地球系のL4,L5点は、太陽と地球を結ぶラインを真横から眺める位置にあり、地球に影響を及ぼす太陽を起源とする諸現象の観測に適した場所である。さらに、L4,L5点は安定な釣合い点であるため、これらの点に宇宙機を軌道保持するのに燃料は不要である。これらの利点を活用するミッションがいくつか構想されている。旧通信総合研究所(2004年4月に通信放送機構とともに情報通信研究機構に統合)と旧宇宙開発事業団は、L5点に宇宙機を配置し、太陽および太陽-地球ラインを観測して宇宙天気予報等に貢献するL5ミッション(L5 mission)を構想している。L4点でなくL5点を選んだのは、約27日周期で自転する太陽の地球に対する面を約4日前にL5点から観測できるからである。図4・34にL5ミッションの構想図を示す。NASA は太陽観測ネットワーク構築プログラムである太陽との共生(Living with a Star)を構想しており、その中で、太陽-地球系のL4,L5点に宇宙機を配置することも検討している。c. 地球公転軌道の離心率による影響地球の公転軌道は約0.0167の離心率(eccentricity)をもっている。離心率があると、太陽と地球の距離は時々刻々と変化する。地球軌道を円と近似した円制限三体問題の場合には、L4,L5点は太陽、地球とともに正三角形を構成する点であった。離心率がある場合の無限小質量物体の運動を扱う問題をだ円制限三体問題という。地球の公転角速度(これも一定ではない)で回転する系で無限小質量物体の運動方程式を作り、変化している太陽-地球間距離を基準長とする無次元化(この座標系を脈動座標系という)を行うと、平衡点であるL4,L5点を表す式は、円制限三体問題の場合と全く同じになる。つまり、脈動座標系で見ると、離心率があっても正三角形の頂点がL4,L5点となっている。L4,L5点近傍の線形運動方程式は、離心率があると変数係数となるために、解析的な解は知られていない。したがって、具体的な運動を求めるには数値積分を使うことになる。地球の離心率による影響の例を以下に示す。円制限三体問題におけるL4,L5点近傍の線形方程式を解くと、周期が約1年の短周期モード(長軸-短軸比が約2)と周期が約222年の長周期モード(長軸-短軸比が約333)が存在することがわかる。離心率が零のときに短周期モード運動をする初期条件を、0.0167の離心率の場合に適用すると、図4・39のように右回りの運動をする1年周期のだ円がずれていく。これは1月1日から10年間計算したものである。しかし、初期速度を約2.5m/sだけ調節することで、このずれを消すことができる。. 木星・金星重力による影響地球軌道の離心率を考慮した場合は、解析的にL4,L5点を定義することができた。木星や金星が存在する場合も、その定義を採用し、時々刻々と変化する太陽と地球の位置とともに正三角形を構成する点をL4,L5点の定義とする。これらの惑星の影響は、L4,L5点付近の宇宙機に直接影響する直接項(direct term)と、惑星の影響を受けた地球の運動が宇宙機に影響する間接項(indirect term)とに分けることができる。地球と惑星の黄経差がいろいろな値となる月日から、惑星が存在しないときにL4点に静止する初期条件を使い、数十年間の宇宙機の軌道計算を行った。木星だけが存在する場合の約35年間の位置のずれを図4・40に、金星だけが存在する場合の約35年間の位置のずれを図4・41に示す。横軸は、初期時刻における地球と木星または金星の黄経差である。この位置ずれは、以下に記すメカニズムによって生じている。惑星の潮汐力(tidal force)により、地球および宇宙機の軌道長半径に短周期変動(short period variation)が発生する。地球と宇宙機の黄経が60°異なるため、両者の短周期変動は一致しない。惑星図4・3910年間の短周期モード運動(離心率=0.0167,1月1日から)が存在しない場合に宇宙機がL4点に静止する初期条件を使って惑星の存在を考慮すると、両者の短周期変動の差が平均軌道長半径(mean semi-major
