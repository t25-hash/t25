# 4・2・1 静止衛星の動向

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・2 先進型静止衛星のバス技術

静止衛星(geostationary satellite)は通信・放送等で商用人工衛星として広く活用されている.静止通信衛星では通信の大容量化とそれに伴う通信コスト低減の傾向があり,そのために静止衛星バスでは大電力化(a項参照)および長寿命化(b項参照)が要求される.これらの課題にこたえるために,近年の静止衛星バスに見られる技術的傾向を概観する.また,商用衛星として低コスト化するためには,静止衛星バスとしてシリーズ化してリカリングコストの低減を図ることが重要であり,代表的な静止衛星バスとしてボーイング社のBS 601,BS 702,ロラール社のLS-1300,ロッキードマーティン社のA 2100,アルカテル社のSpacebus,アストリウム社のEurostarなどのシリーズがある.a. 大電力化静止通信衛星では大電力化のために30kW クラスのものまで開発されている.わが国の静止技術試験衛星は数kW クラスであるから,相当な大型化といえる.このような大型化に対応して必然的に太陽電池パドルは大型化,高効率化が図られている.太陽電池パドルは,パネルを直線状に展開するものでは軌道上サイズが両翼50mに達するものもあり,またパネルを平面状に展開する2段展開パドルも用いられている.大電力化のためには太陽電池セルの高効率化も必要であり,従来のシリコン太陽電池(十数%の変換効率)から高効率のガリウムヒ素太陽電池(20%程度の変換効率)が用いられるようになっている.また,二次電池も従来のニッケルカドミウム電池から,よりエネルギー密度の高いニッケル水素電池を用いて軽量・コンパクト化が図られており,さらにエネルギー密度の高いリチウムイオン電池も実験的に用いられようとしている.大電力化に伴い熱制御の問題も重要である.宇宙空間での排熱はふく射による放熱によらざるを得ないのでラジエータの放11―63第II部基礎編熱面積の確保が課題となるが,大電力型の静止衛星では展開型のラジエータを用いた放熱面積の拡大が図られており,ラジエータまでの熱輸送に用いられるヒートパイプも,ラジエータの展開部に対応できるようにフレキシブルな構造となっている.b. 長寿命化. 姿勢制御コンポーネント静止商用衛星の寿命は15年程度とされており,人工衛星として最も寿命の長い部類に入る.このような長い寿命を達成するためには,衛星バスに用いられる各コンポーネントの信頼性を高めることと,軌道保持に必要となる推進薬量を低減することが重要である.コンポーネントの信頼性を高めるためには,人工衛星バスで用いられる可動部分をなるべく減らす必要がある.可動部分としては姿勢制御用ホイール,ミラースキャン型の地球センサ,機械式ジャイロ,太陽電池パドルなどがあり,ジャイロでは従来のチューンドドライジャイロのような機械式ジャイロに代わって,振動ジャイロの一種である半球形共振ジャイロ(HRG : hemispherical resonator gyro)のような可動部分の少ないジャイロが実用に供されている.また,ノミナル状態で角運動量をもたないゼロモーメンタム静止衛星の場合,その姿勢は,地球センサによるロール・ピッチ角と太陽センサによるヨー角およびジャイロによる三軸の姿勢角速度を衛星のダイナミクスに基づくカルマンフィルタにより処理して決定するのが一般的であるが,姿勢角の検出を地球センサや太陽センサの代わりにスターセンサによって行うこともある.地球センサは地球指向状態において常時ロール・ピッチ角を検出できるが,太陽センサはセンサ視野に太陽の検出できる時間帯が限られており,ヨー角はつねに検出できるわけではない.これに対してスターセンサはつねに三軸の姿勢を検出することが可能であり,かつ可動部分ももたないために信頼性の向上が期待でき,今後静止衛星においても搭載例が増えていくものと考えられる.スターセンサと関連して,静止観測衛星の観測センサで恒星や地表面のランドマークを検出し,姿勢決定誤差やセンサミスアライメントを推定することも行われており,イメージナビゲーションと呼ばれている.コンポーネントの簡素化・軽量化のために,姿勢制御機能と,テレメトリ/コマンド,データ処理,故障検知などの機能を統合化して1台の計算機で処理する統合化制御器の開発も進められている.機能を集約することで高度で柔軟な衛星管理機能が実現できるものと期待されている.. イオンエンジン(ion engine)の利用静止衛星の寿命は軌道保持に要する推進薬量によっても決まる.静止衛星の軌道位置はおもに月・太陽からの重力の影響で南北方向にずれ,地球の偏平性の影響で東西方向にずれるためこれを修正する必要が生じるが,このうち支配的なのは南北方向の軌道修正であり,年間の速度増分として50m/s程度を必要とする.軌道修正に要する推進薬量は用いるスラスタの種類によって異なる.スラスタの代表的なものとしてヒドラジンなどの燃料と酸化剤を用いる二液式スラスタと,キセノンなどの燃料をイオン化し電場で加速して推力を得るイオンエンジンがある.二液式スラスタの比推力が300秒程度であるのに対してイオンエンジンの比推力は3000秒程度となるため,イオンエンジンのほうがはるかに効率的である.静止衛星の初期質量を욡,速度増分を,必要推進薬量を욣,スラスタの比推力を,重力の加速度をとすれば,욣は次式で求められる.욣=욡1-したがって,静止衛星の初期質量を2000kg,を寿命15年に対して750m/sとして必要推進薬量を求めれば,二液式スラスタで450kg,イオンエンジンで50kg 程度に抑えられるが,そのぶん噴射時間は長くなる(1日数時間程度)ので,イオンエンジンの噴射ずれで発生する比較的大きな姿勢外乱が長時間衛星に作用することになり,姿勢制御系ではそのような外乱下での姿勢精度確保が要求される.効率に優れるイオンエンジンもこのようなデメリットをもつので,軌道制御を二液式スラスタとするかイオンエンジンとするかは,静止衛星のサイズや発生電力などを考慮して総合的に判断されることになる.. 姿勢制御方式への影響上述のようにイオンエンジンの採用は姿勢制御系(attitude control system)の構成にも影響を与える.静止衛星の姿勢制御系としては,ピッチ軸方向に角運動量を有するバイアスモーメンタム方式にして,地球センサのようなロール・ピッチ回りの姿勢センサだけで制御を行う方式が主流であった.これはセンサ構成を単純化できるうえに,ピッチ軸回りの角運動量によるジャイロ効果によってある程度の受動的安定性を確保できるからである.いま衛星のロール,ピッチ,ヨー軸回りの慣性モーメントを,,욨,ロール角をφ,ヨー角をψ,ロール軸回りに働くトルクを,ヨー軸回りに働くトルクを욨,軌道角速度をω,ピッチ軸回りの角運動量をとすると,ロール・ヨー系のダイナミクスは次式で表される.φ-++욨-ωψ-ω+욨-ωφ=욨ψ+++욨-ωφ-ω+-ωψ=욨この方程式系ではピッチ軸回りの角運動量によってロール,ヨーが干渉しており,外乱が加わってもニューテーション運動と呼ばれる回転運動を発生して,ロール角,ヨー角が一方的に発散することはない.さらに,ピッチ角運動量による干渉に加えて,軌道運動による干渉も含めた上記のダイナミクスを対象にすると,ロール角を検出するだけで,ロール軸とヨー軸の角度,角速度,外乱などが観測可能となる(ただし,機体座標系で一定値をとるヨー軸回りの外乱を除く).しかし,静止衛星の小さな軌道角速度による干渉をも利用して姿勢制御を行うためには,静止衛星の軌道上外乱が微小であるという前提が必要である.イオンエンジンの長時間にわたる噴射で静止衛星の軌道上外乱が微小とみなせなくなる場合には,上記のような単純化した構成の姿勢制御系とすることは困難であり,衛星の三軸の姿勢を検出することが必要となる.このためバイアスモーメンタム方式でも受動的安定性というメリットはあるが,センサ構成を単純化できなくなる.姿勢を三軸検出可能なスターセンサの利用と相まって,今後の静止衛星においては,バイアスモーメンタム方式でもゼロモーメンタム方式でも姿勢を三軸検出して,各軸をほぼ独立に扱う制御方式が主流となるものと予想される.. スーパーシンクロナス遷移軌道(supersynchronoustransfer orbit)投入方法を工夫することで静止衛星の軌道投入質量を増やすこともできる.通常,静止軌道に衛星を投入する際には遠地点高度をほぼ静止軌道高度とするような遷移軌道に投入して,遠地点においてアポジエンジンと呼ばれる大推力のスラスタを噴射することで軌道傾斜角や離心率を変化させ,静止軌道への投入を図る.ロケットの役割は遷移軌道への投入までであり,遷移軌道から静止軌道への投入に要する推進薬などの質量を差引いた残りが,静止軌道に投入される衛星質量となる.ロケットの遷移軌道投入能力に余裕があれば,遷移軌道の遠地点高度を静止軌道高度より高い軌道とすることで,アポジエンジンの噴射に要する推進薬合計量を減らすことができる.遠地点高度を高めて,そこでアポジエンジンの噴射により軌道傾斜角を変更して第二の遷移軌道に投入し,第二の遷移軌道において再びアポジエンジンを噴射して静止軌道に投入する.このような遠地点高度が静止衛星高度よりも高い遷移軌道のことをスーパーシンクロナス遷移軌道と呼ぶ.一般にだ円軌道において,遠地点距離(地心から遠地点までの距離)を욀,遠地点速度を욀,近地点距離を욣,近地点速度を욣,地球の重力定数をμとするとき,ケプラー運動を仮定すれば욀,욣は次式で表される.욀=2μ욣욀욀+욣욣=2μ욀욣욀+욣したがって,遠地点距離욀を大きくとると遠地点速度욀を小さくすることができ,욀の大きな遠地点において軌道傾斜角を変更することで速度増分を小さく抑えることができる.例えば遷移軌道の軌道傾斜角を30°とするとき,遠地点距離10万kmのスーパーシンクロナス遷移軌道を用いることでアポジエンジンに必要とする速度増分を300m/s程度減らすことが可能であり,いくつかの静止衛星でスーパーシンクロナス遷移軌道を経た軌道投入が行われている.ただし,遷移軌道へ投入するためにロケットに必要となる速度増分を合わせて考えると,スーパーシンクロナス遷移軌道が最適というわけではなく,通常は遠地点高度を静止軌道高度とする遷移軌道のほうが速度増分の和は小さくなる.また,月の重力を利用して静止軌道に投入させた例もある.1997年12月に打ち上げられたヒューズ社の静止衛星は静止遷移軌道への投入に失敗したが,アポジエンジンを用いて遠地点距離が40万kmにも達する軌道に投入し,その間に2回,月の重力を利用して軌道を修正し,最終的に静止軌道への投入に成功している.なお,スーパーシンクロナス遷移軌道とは異なるが,高効率のイオンエンジンを活用して,少ない燃料で静止軌道投入が試みられる例もある.2001年7月に打ち上げられたヨーロッパの静止通信衛星Artemisはやはり静止遷移軌道への投入に失敗したが,アポジエンジンを用いて高高度の円軌道に投入されたあと,1日に15km程度イオンエンジンによって軌道高度を上昇させ,2003年1月に静止軌道への到達に成功した.. 軌道決定静止衛星の軌道決定は,通常,地上局からの電波による距離(レンジ)と距離変化率(レンジレート)の測定によって行われる.これに対して低高度衛星で用いられているようにGPS (global positioning system)によるオンボードの軌道決定が行われれば,衛星運用の省力化につながるので衛星の長寿命化にとっても望ましく,実験的な検討が行われている.GPS 衛星は高度2万kmの円軌道を周回しており,その航法データは地球方向へ発信されているので,それより高度の高い衛星が利用することは難しい.しかし,地球の反対側から来るGPS 衛星のデータを利用することも不可能ではなく,実際静止遷移軌道における高高度での軌道決定が実験的に行われており,静止衛星の軌道決定にもGPS の利用される可能性がある.
