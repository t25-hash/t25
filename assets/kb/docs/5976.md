# 4・2・2 静止衛星の展開

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・2 先進型静止衛星のバス技術

静止衛星は通信,放送,気象観測,データ中継などの分野に幅広く用いられているが,これらに加えて,環境監視,測位,宇宙発電などにも用いられようとしている.静止軌道は,地上局と衛星とのリンクがつねにとれる点で,通信だけではなくエネルギー伝送にとっても有利である.そのため,静止衛星による宇宙太陽光発電の構想は1970年代からあるが,近年再び脚光を浴びている.この背景には,太陽電池の変換効率の向上などで,石油や原子力に代わる基幹エネルギーとして太陽光発電が注目されていることがある.また静止軌道は赤道面上に限られるので,静止衛星だけで測位を行うことはできないが,GPS 衛星や次に述べる準天頂衛星と組合わせて,GPS を補完するシステムが検討されている.a. 準天頂衛星(quasi-zenith satellite system)準天頂衛星とは軌道周期を静止衛星と同じにするが軌道傾斜角をもたせたもので,衛星の地表面軌跡は静止衛星のように赤道上に静止するのではなく,8の字形を描くので8の字衛星とも呼ばれている.ある地域において衛星が常時見えるわけではないが,軌道傾斜角をつけることで高緯度の地域においても衛星を見上げる仰角を高くとることができるので,電波が建物などに遮へいされにくく,静止衛星と軌道面が異なるために周波数帯を有効利用できるなど,通信にとって有利な面をもっている.このような衛星を昇交点経度が異なる軌道面に配置することで,静止衛星と組合わせた測位のほか,移動体通信や放送などへの応用が考えられている.b. 軌道エレベータ(space elevator)静止衛星の可能性の一つとして軌道エレベータも検討されている.これは質量中心を静止軌道高度にもつ直線状に伸びた静止衛星で,その一端を地表面に到達させて,地表面から静止軌道までをエレベータによって輸送するというシステムである.材料強度の確保や長大な衛星の進展法など,実現に向けての課題は多々あるために近年中の実現はありえないが,静止衛星としての利用に加えて宇宙工場,観光,惑星探査基地などの応用が考えられており,NASA も加わって技術的検討が進められている.
