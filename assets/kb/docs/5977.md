# 4・3・1 ホイール(wheel)

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・3 宇宙用ホイール、ジャイロ、加速度計

a. ホイールの原理ベアリング軸受をもつロータにモータを付けて衛星に取付ける。モータを回すとロータが回転し、その反力で衛星本体にトルクを与えることができる。ホイールは人工衛星の姿勢制御に必要なトルクそのものを直接発生することができる。これは他の姿勢制御機器にない特徴である。電気エネルギーさえあれば制御力を供給し続けることができるのも、リソースの限られる衛星にとってありがたい特性である。このため、現在の衛星の姿勢制御にホイールは欠かせないものになっている。b. ホイールの主要構成品回転を支えるベアリング、慣性能率をもったロータ、それにトルクを発生するモータが三大要素である。これにモータコンミュテーションのためのポジションセンサ、モータの駆動回路、回転数を測定するためのセンサ等が加わる(図c. ホイールの主要性能。 出力トルクホイールの能力は制御力、すなわち出力トルクで表される。現在わが国で使われているホイールの最大出力トルクは0.07N・mから0.2N・mくらいのものである。2~3tの衛星用としてはこの程度である。観測衛星では素早く観測器の視線を動かしたいという要求もあり、1N・mを超える出力のホイールも検討されている。. 起動トルクとロストルク出力トルクに対する損失分である。ホイールのロストルクは、ゼロ回転時に起動バイアスをもち、回転数に比例した抵抗係数をもった一次近似式で表される。ロストルク仕様値の例を挙げると、出力トルクが0.07N・mのホイールで、回転数が0~6000minの範囲で0.036N・m以下である。実力値はずっと小さいが、仕様値は出力トルクと比べて無視できないほど大きいことに着目しておく必要がある。. 吸収角運動量吸収角運動量が大きいほど大きな外乱または多量の外乱に対応できる。吸収角運動量は回転速度とホイールロータの慣性能率に依存する。国産ホイールの直径30cm程度の11―65第II部基礎編もので50N・msくらいである。. 最大回転数ホイールの最大回転数が大きいほど吸収角運動量を大きくとることができる。許容最大回転数は、ベアリングのボール走行面とボールの間に油膜が確保される温度条件で定まる。ベアリングのサイズ、予圧あるいはオイル等で差異があるので一概に論じることはできないが、ベアリング温度は最大80°Cくらい、ただし宇宙では安全余裕を確保する必要があるためせいぜい60~70°Cくらいで用いる。この温度条件で最大回転数は6000min程度である。d. ベアリングユニット構成ホイールの信頼性と寿命はベアリングに依存する。回転軸を支えなければならないので、上下にベアリングを配置するが、宇宙用ホイール固有の特異な使い方があるわけではない。e. ベアリング。 内外輪ベアリングそのものは宇宙用の特殊なものではない。素材はステンレススチールで、等級はABEC9である。. ボールステンレススチールであったが、最近、表面にチタンカーバイドのコーティングを施す例や、バルクのセラミックボールを使用する例が出てきた。. 保持器、オイル保持器にはほとんど例外なく、コットンフェノリック材が使われる。理由はオイルの保持である。オイルは在来、実績があるという理由で鉱油タイプが使われてきたが、合成油のほうが組成が明白で性質も把握しやすく、添加剤の調整も容易であるので、合成油に移りつつある。ベアリングのホイール応用で特徴的なことは、ロストルク軽減のためにレース面のオイル量をごく微量に保つこと、その一方でオイル補給ができないことである。f. 磁気ホイール(magnetic wheel)非接触ベアリングであれば、信頼性あるいは寿命の問題は解決する。最短距離にあるのは磁気軸受であるが、ボールベアリングタイプと比較すると、磁気回路が必要、回転しないときのタッチダウンの仕掛け、ロンチロックと呼ばれるロケット打上げ時のロータ固定の仕掛け、設計にもよるが地上試験用の重力キャンセラを必要とするなど、付加部分が多い点もあり、いまだボールベアリングタイプにとってかわるに至っていない。しかし、ボールベアリングタイプでは避けがたい、振動を抑制できるという特徴を生かし、ヨーロッパでは視線のぶれを嫌う観測衛星で使用されている。g. コントロールモーメントジャイロ(CMG : control momentgyro)一定の角運動量をもって回転しているロータの回転軸と直交する軸をトルキングすると、ロータの角運動量と入力したトルクのそれぞれに直交した向きにトルクを生じる。このトルクを利用して制御しようというのがCMG である。角運動量と入力トルクのベクトル積が出力だから、大トルクを容易に発生できることが特徴である。アメリカのスカイラブで、宇宙船に生じる振動を含む不要な運動の吸収に使われた。建設中の国際宇宙ステーション(ISS : Inter-national Space Station)にも使われている。
