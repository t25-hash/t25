リングレーザジャイロ(RLG : ring laser gyro)H-IIロケットの慣性センサユニット用として実用化された。H-IIA ロケットにも使用されており、ロケット用の誘導機器として主流を占めつつある。航空機の慣性航法装置はほとんどがこのタイプのジャイロを使用したものになった。ループ状の経路に光を右回りと左回りと同時に走らせる。経路長は同じだから一周すると同時に戻ってくる。ループを回転させると、光の投入点がループとともに動くから、投入点から投入点まで動く光の距離は右回りと左回りで異なり、投入点に到達する時間に差が生じる。この差を読み取れば回転角速度が検出できる。これが光ジャイロの原理である。利得は経路の囲む面積に比例する。RLG は光路長に依存した発振を利用する。静止時は右回りと左回りのおのおのの発振周波数は同一であるが、角速度が加わると片方の発振周波数は高くなり、片方は低くなる。この周波数差を干渉じまで読み取って角速度を測定する。低い角速度入力の場合、右回りと左回りのおのおのの発振が相手の周波数に引き込まれるロックインという現象を起こす。この解決策としてジャイロの入力軸回りに振動を与える。つまり故意に角速度で変調をかけ両者の周波数差を稼ぐ。g. 慣性航法装置、慣性基準装置実際の応用では、ジャイロを直接利用することはほとんどなく、ジャイロを組込んだ機器を利用するのが普通である。宇宙用ロケットでは慣性航法装置、衛星では慣性基準装置が一般的である。. 慣性航法装置(INS : inertial navigation system)加速度計で加速度を測定し、積分して速度を、さらに積分して位置を得る。加速度の向きを知るのにジャイロを使う。これが慣性航法装置である。慣性航法装置はベトナム戦争時代に急速に発達した。慣性航法装置には大きく二つのタイプがある。プラットホームとストラップダウンである。プラットホーム型はジャイロと加速度計を慣性空間に固定して使う。ジャイロ3台をプラットホームに固定する。プラットホームを三重のジンバルで支え、3台のジャイロの出力がいつも零になるようにジンバルトルカを駆動し、プラットホームをつねに一定の姿勢に保つ。プラットホーム上の加速度計はつねに同じ方向を向くので、出力を単に積分すれば位置が求められる(図4・49)。ストラップダウン型は、ジャイロと加速度計を機体に直接取付け、出力を演算処理して固定した座標系に直す。演算サイクル内では一定の姿勢角だとして機体の姿勢角と位置を更新する。次の演算サイクルでは、この更新された機体座標をもとに演算する。プラットホーム構成と比較するとハードウェアが圧倒的に簡素なので、慣性航法装置はすべてがこの型となった。ジャイロにはダイナミックレンジの大きなものを必要とするが、光ジャイロが実用化された現在、問題はない。計算機の能力の問題もあったが、高速のプロセッサが簡単に手に入る現在、この問題は完全になくなった。.
