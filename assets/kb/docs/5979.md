# 4・3・3 加速度計(accelerometer)

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・48 )。

おもりを付けた振り子を用意する。支点をもって加速度を与えるとおもりは後ろに残る。加速度とおもりの質量の積に比例した力が生じるから、この力を測定すれば加速度が計測できる。a. サーボ加速度計加速度計測は慣性航法になくてはならないものである。国産加速度計はH-Iロケットから使われ始め、11―67第II部基礎編図4・50 TDG IRU[三菱プレシジョン(株)]図4・51 加速度計[日本航空電子工業(株)]H-II,H-IIA ロケット、科学衛星用ロケットM-V の誘導装置にも搭載されている。衛星に航法用として搭載された例はランデブー・ドッキング実験のETS-VIIである。加速度計のもう一つの用途はロケットや衛星の運動の把握である。機体の振動計測、遠心加速度を計測して回転角速度を求める。太陽電池パドルに取付けて振動計測に、衛星構体の要所に配置し衛星の構造解析、制御性解析にも使われる。現在航空機や宇宙で航法用として実用化されている加速度計のタイプは、サーボ加速度計ただ一つといってよい。最大計測値は10G くらいから100G、最小分解能は10~10G くらいだから、ダイナミックレンジは10けた近い(図4・51)。おもりをヒンジで支える。ヒンジはおもりの動く方向を正確に保持する。おもりには変位を測るためのピックオフとトルカを取付け、ピックオフで測定した変位に従ってサーポアンプを介してトルカを駆動し、おもりを中立位置に保つ。トルカに流れる電流は加速度に比例するから、この電流値を読んで加速度を得る。精度を得るには温度制御を必要とするが、五次曲線の補正係数をユーザに提供して数値で補正している例もある。b. 半導体加速度計H-IIA ロケットでは、自動車用のエアバッグ展開のための衝撃加速度計測用の加速度計を利用して振動計測に使っている。半導体加速度計は小型で、利得が大きいのが特徴である。最大の魅力は半導体プロセスを用いて、半導体自身のセンサとしての能力、回路の組込み等のIC技術を駆使できる点にある。
