# 4・4・2 陸域観測技術衛星と指向管理

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・4 高精度地球観測衛星の姿勢制御

a. 陸域観測技術衛星宇宙航空研究開発機構(JAXA)が開発し、2006年に打ち上げた陸域観測技術衛星(ALOS : AdvancedLand Observing Satellite)は、陸域観測分野における実利用の推進を目指した大型高分解能衛星である(図4・52)。特に、高分解能画像のグローバルな取得を通じて、地図作成、地域観測、災害状況把握、資源探査に貢献する。ALOS はミッション要求を実現するため、2.5m分解能のパンクロマチック立体視センサ(PRISM)、マルチバンドの高性能可視近赤外放射計2型(AVNIR-2)、全天候観測が可能なフェーズドアレイ方式Lバンド合成開口レーダ(PALSAR)の三つの大型観測センサを搭載している。衛星としては、質量4000kg、発生電力7kW の大型衛星である。高度691.65kmの太陽同期準回帰軌道に投入され、3年以上の間ミッションを遂行する。b. 高精度指向管理(pointing management)高分解能で全地球の観測データを収集するALOS ミッションの性格から、全地球の取得画像に対し地上基準点(GCP)なしの高精度な幾何補正が必要になる。このために、衛星位置やセンサ指向方向の決定を高精度に行うとともに、指向方向の変動を最小にする必要がある。ここからALOS の指向要求が導かれた。特に、指向安定度[point-ing stability,4.0×10°/5s(p-p)]、指向決定精度(pointingdetermination accuracy、オンボード:4.0×10°、オフライン:2.0×10°)、位置決定精度(position determination accuracy、図4・52 陸域観測技術衛星(ALOS)の外観図4・53周波数領域における要求と擾乱オフライン:1m)において、高い精度を実現することが求められた。ALOS ではこれらの指向要求を、指向管理と呼ぶ総合的な枠組みのもとで、広範なシステムレベルの方策を施し実現している。特に姿勢軌道制御系に対しては、姿勢制御精度、姿勢安定度、姿勢決定精度、位置決定精度として要求を配分している。c. ALOS 姿勢ダイナミクスと指向要求ALOS は、全長22mの太陽電池パドル(PDL)、全長9mのPALSAR アンテナ、全長2mのデータ中継通信アンテナ(DRC)といった大型柔軟構造をもつ。このため各柔軟構造物の低次振動モードが広帯域にわたり存在している。擾乱源としては、おもに、DRCアンテナ駆動機構、AVNIR-2ミラー駆動機構、リアクションホイール、PDL駆動機構、PDLの熱衝撃、環境外乱がある。これらの擾乱が、衛星本体の剛体運動に影響を与えるほか、DRC,PALSAR,PDLの動的モードを励起する。擾乱や動的モードは、それぞれの周波数帯域をもつ。図4・53で、指向要求を周波数領域で表現し、擾乱や動的モードの周波数成分と比較した。指向変動の低周波数成分はおもに姿勢制御系により抑制されるが、その帯域は擾乱や振動モードのすべてを補償するほど広くない。このため、制御帯域外の安定度を低く抑える方策が必要になる。
