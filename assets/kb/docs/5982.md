# 4・4・3 姿勢軌道制御系の構成

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・4 高精度地球観測衛星の姿勢制御

a. 概要今日の地球観測衛星の姿勢軌道制御系(attitudeand orbit control system)に見られる特徴は、高精度化とマヌーバ能力の拡大である。ALOS の姿勢軌道制御系(AOCS)は、このうち高精度化に焦点を当て、高い姿勢安定度、姿勢決定精度、位置決定精度を実現している。このため、高精度恒星センサ、高精度GPS 受信機、高性能計算機、大トルクホイール、恒星同定、拡張カルマンフィルタ、恒星センサ基準姿勢制御、柔軟構造位相安定、高精度協調制御などの新規技術を取り入れている。ALOS AOCS のおもな機能性能を表4・9に示す。AOCS は、観測モードである定常制御では、恒星センサ基準の姿勢決定(高精度姿勢決定)とこれに基づく姿勢制御を行い、姿勢に対する精度要求を満足する。また、観測データの幾何補正に必要な姿勢や軌道に関するデータを常時生成し、低速ミッションデータとして地上にダウンリンクする。b. アーキテクチャALOS AOCS AOCEの搭載ソフトウェア(ACFS)は、民生品のリアルタイムOS 上に高級言語Cで実現された。従来のアプリケーションに加えて、FMM (fault-tolerant multiprocessor manager)、恒星同定、高精度姿勢決定フィルタ、高精度協調制御、高精度軌道モデル、1553B インタフェース、宇宙データシステム諮問委員会(CCSDS)準拠データ処理といった機能が搭載されている。このAOCS の一つの特徴は、定常運用モードでは、STT とIRU を用いた高精度姿勢決定とESA,IRU,FSS を用いた標準姿勢決定を並行して行うことである。このモードでは、高精度系をもとに姿勢制御を行い、標準系は高精度系の監視に用いられる。標準系はまた、高精度系立上げ前や異常時の制御にも用いる。この構成により、高精度姿勢情報のユーザへの配信や、高精度姿勢決定系による高い精度の実現を可能にするとともに、実績のある標準姿勢決定による異常検知でリスク低減が図られている。c. tracker)は、複数の恒星を検出してその方向ベクトルと信号レベルを計測する。この情報を直接出力する方式と、さらに恒星同定、姿勢計測まで行う方式がある。恒星センサは、ミッションの高精度化に伴い地球観測衛星でも広く用いられるようになってきており、機器の高精度化、恒星同定機能を付加した自律化、軽量・低価格化が進んでいる。ALOS 恒星センサ(STT)は、1024×1024ピクセルの電荷結合素子(CCD)を検出素子とし恒星の位置と明るさを出力する高精度姿勢センサで、ランダム誤差9′′、バイアス誤差0.74′′(いずれも3σ)という世界最高の星位置精度を特徴とする。ALOS STTの光学部とアナログ部は3系統搭載され、通常2系統が同時に使用される。精度のために最適化したトラックモードと、既知情報なしに全天の中から初期の恒星同定を行うための捕捉モードがある。トラックモードの場合、AOCEが指定した位置にウインドウを切り、その中の星の位置と明るさを1系統当たり毎秒最大5個、捕捉モードの場合は、星の位置と明るさを明るいほうから1系統当たり毎秒最大10個、AOCEに出力する。ALOS STT はまた、生画像データを出力することもできる。ALOS STT では、高精度の角度検出を実現するために、低熱ひずみ構造、高安定温度制御、低ノイズ高分解能CCD が採用されている。e.
