# 4・4・4 姿勢決定精度

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・4 高精度地球観測衛星の姿勢制御

姿勢決定系(attitude determination system)には,姿勢センサ直接使用方式と状態推定方式がある.後者にはさらに,軌道座標系に対するオイラー角を状態量とし定ゲイン推定器を構成する方式と,慣性座標系に対する姿勢推定誤差に対しカルマンフィルタを構成する方式がある.地球観測衛星では,恒星センサとジャイロの観測量をもとにして後者の方式により推定系が構成されるようになりつつある.ALOS AOCS の定常制御モード時の姿勢決定系もこの方式を採用しており,高精度STT,オンボード恒星同定,拡張カルマンフィルタによる姿勢クオータニオン推定,STT-IRU 間のアライメント変動低減により,高い姿勢決定精度(attitude determinationaccuracy)を実現している(表4・10).a. 恒星同定・姿勢計測恒星同定(star identification)の方式には,パターンマッチング法とダイレクトマッチング法がある.パターンマッチング法は,姿勢情報がない場合に計測した複数の星のパターンに合致する星を恒星カタログから検索する方法である.ダイレクトマッチング法は,既知の姿勢情報がある場合に,恒星センサ視野に入る恒星を予測し計測された星位置に近いカタログ上の星を同定するもので,高速で高精度の星位置精度をもたらす.ALOS 姿勢決定系では,STT 捕捉モード時は,三角法を用いたパターンマッチングにより恒星同定を行い,いったん同定が成功した後は自動でダイレクトマッチングに移行して恒星同定を継続する.移行後,一定時間同定失敗が継続する場合は,自動で再度パターンマッチング法へ移行し同定を行う.このように二つの方法の長所を生かし組合わせて用いている.恒星同定結果から姿勢を計測する方法には,TRIAD 法,q-Method,QUEST 法,特異値分解法などがある.ALOS AOCS の捕捉モードではTRIAD 法を複数星ペアに拡張した方法が用いられている.トラックモードにおけるSTT 計測姿勢角の算出は,姿勢角誤差の微小性を利用した方法によりなされる.b. 恒星カタログ恒星同定に用いる恒星カタログには,BSCカタログ,SAOカタログ,SKY 2000.0カタログ,Hipparcosカタログ,Tychoカタログがある.マスタカタログの選定に当たっては位置精度,等級精度,等級範囲,情報量などが考慮される.ALOS AOCS では,Hipparcosカタログをもとに改訂されたSKY 2000.0Ver.2カタログをマスタカタログとして用いている.これをもとに,離角の小さい恒星対と多重星の削除,検出可能恒星の抽出,固有運動が大きい恒星の削除,変光星の削除,隣接星の削除,星雲と重なる恒星の削除を行い,搭載恒星カタログを作成している.さらに同定した星に対して,固有運動,年周視差,光行差等の補正を行い,高精度姿勢決定フィルタに入力される.c. 拡張カルマンフィルタIRU
