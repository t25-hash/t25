# 4・4・5 姿勢制御精度および姿勢安定度

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・4 高精度地球観測衛星の姿勢制御

ALOS の姿勢制御精度(attitude control accuracy)は、高精度姿勢決定系を用いた姿勢制御、精密な軌道モデルを用いた制御基準座標系により実現された。ALOS の姿勢安定度(attitude stabil-ity)に対しては、柔軟構造の位相安定化、高精度協調制御と軌道上パラメータ同定、アンテナ追尾制御系のアクティブダンピング、高精度姿勢決定系基準姿勢制御、固有値管理、ホイールバイアス運用、パドル共振回避アレイトリム制御が導入された。a. 姿勢制御基準座標系高精度姿勢決定系により算出される姿勢角は慣性座標系に対するものである。地球中心指向姿勢制御を実施する際には、慣性系に対する姿勢から軌道位置に基づき地球指向の基準姿勢を算出する必要がある。基準姿勢の算出には、オンボード軌道モデルによる方法やGPS 受信機出力による方法がある。ALOS AOCS では精密なオンボード軌道モデルを搭載し、これを伝ぱして得られる制御基準軌道座標系を慣性系基準の推定姿勢から差引き制御誤差を算出する。b. フィードバック制御則制御精度が求められる地球観測衛星では、一般にゼロモーメンタム方式がとられる。この場合、各軸独立の姿勢制御系が基本になる。ALOS では、姿勢の各軸に対し図4・55 定常制御ボード線図(パドル回転角90°、ヨー軸)表4・11 姿勢安定度実現方法擾乱対策共振・アンテナ追尾制御系の高アクティブダンピング化・構造、機構の最適化設計・固有値の向上/低減と管理DRCアンテナスルー・協調制御・軌道上パラメータ同定・PDL,PALSARモードの位相安定化・ホイールサーボループの高ゲイン化AVNIR-ミラー駆動・協調制御・ミラー質量特性管理ゼロクロス・バイアス角速度運用インバランス、リップル、ベアリングノイズ・低擾乱ホイールの採用ホイール熱誘導ダイナミクス・ホイールの高トルク化・パドルの高剛性化太陽電池パドル追尾駆動・共振回避駆動制御(駆動パルスランダム化)制御系決定安定度・高精度姿勢決定系基準姿勢制御て、独立した制御則が設計された。設計は周波数領域で古典制御の手法により行われた。制御則はPD 制御と積分補償、リードラグフィルタ、およびホイールスピードサーボループからなる。それぞれのフィルタは、各軸の柔軟構造特性などの違いにより異なったものとなっており、従来の衛星より積極的に柔軟モードの位相安定化(phase stabilization)が図られている。制御帯域は三軸とも0.005Hz程度としているが、補償帯域を0.5~1Hz 程度にまで広げている。例として、ヨー軸のボード線図を図4・55に示す。ホイールスピードサーボとしては、ホイール回転パルスを観測値とした定常カルマンフィルタを構成してスピードサーボの推定を行い、この推定値に基づいてスピードサーボ系を構成している。これは、回転パルスの分解能の粗さによる姿勢安定度の劣化を防止するためのものである。また、スピードサーボループを高ゲイン化して、サーボ系による遅れを改善し高い姿勢安定度を実現している。c. 姿勢安定度実現方法ALOS にて主要な安定度劣化要因に対してとった方策を表4・11にまとめる。DRCアンテナのステップモータによる駆動は、ブームとハーモニックドライブギヤのモードの共振を励起する。これを低減するために、DRCアンテナの追尾制御系はこれらのモードにアクティブダンピングを施している。また、ブーム・ギヤ設計における固有値管理により共振を低減している。DRCアンテナが大きく指向方向を変更する際のスルー運動の角運動量交換により、大きく姿勢が変動する。この影響を低減するため、AOCS はDRCアンテナの運動と協調してフィードフォワー11―71第II部基礎編ド制御を実施する。協調制御はミラー駆動機構により生成される角運動量を補償するためにAVNIR-2に対しても行われる。AOCSによるフィードバック制御も姿勢擾乱の低減に大きく貢献する。ALOS では、太陽電池パドルやPALSAR の低次モードの位相安定を図っており、これにより環境外乱、パドルの熱誘導ダイナミクス等により発生する姿勢振動を積極的に減衰させている。d. 衛星間通信アンテナ協調制御DRCが中継衛星を切換える際のスルー時の協調制御(cooperative control)では、姿勢角の変動を抑えるために、윴읈윬윬윮=-という角運動量をホイールに与える。DRCの角運動量は、アンテナ角度、アンテナ角速度からなる項と、アンテナの質量パラメータからなる項との間に導かれる線形な関係から求められる。ここで、ALOS ではモデル化誤差を低減するため、2剛体モデルに代わり、アンテナモデルに3剛体モデルを用いている。この協調制御の性能は、DRCアンテナの質量特性モデルのパラメータ誤差に依存するため、軌道上でパラメータ同定を行い協調制御誤差を1%以内にする。線形な関係式の導入は、軌道上の計算負荷を低減するだけでなく、パラメータ推定を容易にしている。この関係式を用いて補償角運動量を計算する際、アンテナ角速度には、アンテナ制御系の生成するレートコマンドを用いる。また、ホイール制御系へ入力する際は、ホイールサーボ系での遅れを除去するために差分によりトルクを求めて直接ホイールトルクとして入力する。
