# 4・4・6 姿勢マヌーバ

> **章**: 第4章 軌道・姿勢力学と制御
> **節**: 4・4 高精度地球観測衛星の姿勢制御

地球観測衛星では,観測センサの指向方向を変更するために姿勢マヌーバ(attitude maneuver)を実施することがある.姿勢マヌーバには,ヨーステアリングや,商業高分解能衛星が行うような大角度のロール・ピッチマヌーバがある.ヨーステアリングは,光学センサの観測幅の重なりを確保したり,合成開口レーダのドップラーシフトを最小にするために行われる.ALOS は振幅4°のヨーステアリングを行う機能を有し,姿勢コマンドとして基準姿勢に加えて制御ループに与えられる.大角度姿勢マヌーバを行う場合には,柔軟構造モードを励起する可能性がある.このため,マヌーバによる姿勢変動が整定に至る時間が最小となるような姿勢コマンドプロファイルが研究されている.文献(1) LUNAR-A のホームページ, http://planeta.sci.isas.ac.jp/LUNAR-A/transfer.html(2) 川勝康弘・ほか3名, 第45回宇宙科学技術連合講演会, 01-2A 6(2001),93.(3) 石山仁・ほか5名, 第45回宇宙科学技術連合講演会, 01-3E7(2001),1059.(4) Kawaguchi, J., ほか3名, Acta Astronautica, 35-9/11 (1995),633.(5) Kimura, M., ほか5名, Proc. 21st Int.
