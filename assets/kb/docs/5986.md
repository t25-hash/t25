# 5・1・1 真空

> **章**: 第5章 材料
> **節**: 5・1 宇宙環境が及ぼす材料への影響

宇宙空間は，極めて低密度の中性または荷電粒子（気体分子また
は原子）が存在する空間である．図5･1 のように地表から離れる
につれ，粒子の集合度，すなわち気圧は指数関数的に低下する．
また，気体分子や原子の自由熱運動速度よりも高速で飛行する宇
宙機の後方に形成されるウェーク（航跡）空間には，10Paに達
するより清浄な極高真空環境が存在する（図5･2)．10Paもの
極高真空環境は，地上では数cm程度の容積しか得られないのに
対し，直径10mの円板後流では300mの大容量を得ることがで
きる．
