# 5・1・2 アウトガス

> **章**: 第5章 材料
> **節**: 5・1 宇宙環境が及ぼす材料への影響

アウトガス(out-gassing)は、真空中で材料から発生するガスのことであり、材料自身の劣化と周辺機器に対する汚染の点で、材料選定時の重要な評価項目の一つとなっている。アウトガスの特性指標としては、質量損失比(TML: totalmass loss)、再凝縮物質量比(CVCM : collected volatile conden-sable materials)、および再吸水量比(WVR : water vaporregained)があるが、TMLとCVCM の二つの特性で評価するのが一般的である。材料選定に当たっての推奨値は、TMLが1%以下、CVCM が0.1%以下である。なお、上記評価ではアウトガス特性の時間による変化率がわからない、再凝縮温度条件(25°C)が実宇宙環境温度(-100~+100°C)を反映していないなどの課題があるため、より実宇宙環境に即した試験条件を考慮したアウトガスレート測定も行われるようになっている。アウトガスは、おもに有機材料から発生し、特に接着剤、塗料、潤滑油、グリース等から生じたアウトガスによる周辺の光学機器や太陽電池パドル等への汚染が問題となっており、留意する必要がある。
