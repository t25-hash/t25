# 5・2・1 原子状酸素

> **章**: 第5章 材料
> **節**: 5・2 機構用材料への原子状酸素,宇宙線の影響

高度100km付近の高層大気では,太陽から降り注ぐ紫外線により酸素分子(O)が解離され,原子状酸素(AO: atomic oxy-gen)が発生している.原子状酸素は非常に酸化作用が強く,材料に衝突すると劣化を生じさせる.図5・3に,気体の密度分布と高度との関係を示す.地球周回低軌道の宇宙機は,原子状酸素が浮遊する中を飛行する.原子状酸素の密度は10~10atm/cmと非常に低いが,宇宙機の軌道速度が約8km/s(エネルギー換算で約5eV)と高速であるため,高度約300kmでは約10atm/(cm・s),高度約500kmでは約5×10atm/(cm・s)のフラックス(照射線量率)を受けることとなる.原子状酸素は太陽からの紫外線により生成されるため,太陽活動の変化に伴いその密度は増減する.図5・4に,太陽活動と原子状酸素密度との関係を示す.多くの宇宙機用有機材料や一部の活性金属材料は,原子状酸素により強い酸化を受ける.原子状酸素による酸化作用は温度依存性を有する場合がある.一例として,カプトン100では,233K(-40°C)の環境と473K(200°C)の環境とで反応効率に約1.3倍の差が生じており,宇宙機等へ使用する場合,使用環境に応じた配慮が必要である.
