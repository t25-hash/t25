# 5・2・2 紫外線

> **章**: 第5章 材料
> **節**: 5・2 機構用材料への原子状酸素,宇宙線の影響

宇宙空間を飛行する宇宙機は,厚い大気で保護されている地上に図5・3 気体の密度分布と高度の関係図5・4 太陽活動と原子状酸素密度11―74比べ,はるかに強力な紫外線にさらされる.紫外線は特に有機材料に作用してその分子間結合を変化させ,一般的には劣化を生じさせる.NASA SP-8005では,波長10~300nmの光を紫外線領域と定義している.波長200nm以下の真空紫外線は,分子間結合を変化させる強力なエネルギーを有するが,大気により吸収され地上には到達しない.図5・5に,AM 0(air mass 0,大気圏外)および地表上における太陽光スペクトル分布の比較を示す.
