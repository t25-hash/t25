# 5・3・1 軽量構体パネル

> **章**: 第5章 材料
> **節**: 5・3 機構用材料に関する最近の話題

人工衛星をはじめとする宇宙機の構体には、CFRP(carbonfiber reinforced plastics)またはアルミニウム(Al)合金をスキン材、Alハニカムをコア材とするアルミニウムハニカムサンドイッチパネルが多く使用されている。アルミニウムハニカムサンドイッチパネルは優れた比強度、比剛性を有するが、一方で製造に要するコストが高い。そこで宇宙航空研究開発機構(JAXA : Japan AerospaceExploration Agency)では、宇宙機の軽量化、低コスト化を目的として、スキン材にCFRP、コア材に低密度高分子発泡材であるロハセル(ポリメタクリルイミド発泡材)を使用した軽量構体パネルの研究開発を実施している。通常のサンドイッチパネルが別々に製作されたスキン材およびコア材を接着することにより製造されるのに対し、本軽量構体パネルは、スキン材となるCFRPプリプレグに含まれる樹脂が、多孔質である発泡コア材に含浸することによる一体成形が可能である。そのため、軽量構体パネルは、従来のサンドイッチパネルと比較して、省工数化、接着剤分の軽量化、曲面等複雑形状の成形等が可能なほか、コア材の切削性がよく、加工性に優れるという特徴をもつ(図5・7)。なお、H-IIA ロケットの段間部(図5・8)は、本軽量構体パネルと同様の構成となっており、H-IIA ロケットの軽量化に貢献し、その打上げ実績を重ねて図5・7 軽量構体パネル図5・6 プラズマ環境モデル図5・8 H-IIA ロケットの段間部第II部基礎編いる。
