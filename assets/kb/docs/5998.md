# 6・1・4 微小重力

> **章**: 第6章 トライボロジー
> **節**: 6・1 宇宙機器が遭遇するトライボロジー問題

宇宙環境というと,真空とともにいわゆる無重力(微小重力,microgravity)を思い浮かべる方が多いのではないか.微小重力のため自重による負荷の影響がほとんど無視できるという点ではトライボロジーにはプラスに作用するが,摩耗粉が思わぬところに浮遊して周辺機器の汚染や障害を引き起こすおそれがある.また,液体潤滑剤を使用している場合,油の流れが表面張力により支配される.このため,温度差が生じると表面張力の差による流れ(マランゴニ対流)が生じ,潤滑剤の偏在や,場合によっては潤滑面から潤滑剤が流れ出てしまい潤滑不良に陥るおそれがある.
