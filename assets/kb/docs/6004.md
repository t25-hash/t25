# 7・3・1 ロケットでの動的問題

> **章**: 第7章 構造力学
> **節**: 7・3 構造動力学

a. 動的な荷重ロケットにかかる動的な検討事項としては図与える静的な荷重であるが、カルマン渦励振により曲げの振動荷重が生じる。ロケット発射時には過渡的な力が働き、ロケットに縦と曲げの全体的な振動が誘起される。またこのときエンジン騒音による音響が大きく、特にフェアリングを通してペイロードへの評定荷重となる場合が多い。上昇中、特にジェット気流付近を通過するときのウインドシェア(wind shear)が大きな曲げモーメントを生じさせる。また、そのときの飛行制御力により、機体への曲げ荷重が発生する。ピッチ制御と機体の曲げ一次振動、さらに液体推進薬の横振動(横スロッシング、lateral sloshing)が干渉すると不具合を起こしやすい。ロール制御と機体のねじり振動、さらに推進薬の旋回振動(スワール、swirl)の干渉についても検討しておかなければならない。上昇中にはエンジン騒音と空力騒音によるランダム加振もある。ロケットも空気中を高速で通過するので動圧がかかり、航空機と同様に空力弾性問題を検討しておかなければならない。機体の一部により発生した乱れた後流により引き起こされる不規則振動であるバフェットは特に遷音速で生じやすく、パネルフラッタはロケットの外板が超音速で熱を受けて圧縮応力状態にある場合に生じやすい。尾翼がある場合には通常の揚力面フラッタは検討しておかなければならない項目である。液体ロケットの場合には、各段燃焼末期にポゴ(pogo)という正弦波の縦振動が20~30秒程度起こり、ペイロードに対しての評定振動荷重となる場合が多い。液体に関わる振動については7・4節で述べる。b. 解析のためのモデリング動的な荷重に対する応答としては曲げ一次モードが重要である。簡単な全体の曲げ振動の解析モデルとしてはビームでモデリングされる。ビームとしての断面定数などは詳細な有限要素解析や部分試験によって求める。内部の液体推進薬は7・4・1項に示されるマス-スプリングモデルで置換えられる。縦振動の簡易モデルとしてはバー(棒)要素とマス-スプリングでモデリングする。液体推進薬についてはタンク構造との連成を考慮して図7・21に示すようなモデルが使われてきた。三つのばね定数욡はタンクをシェルと考え、シェルの変形を考慮して計算される。욞は液体の質量、욦は構造の質量である。詳細な動的モデルについては曲げ振動と縦振動の別々のモデル化を行わず、三次元有限要素モデルで解析が行われる。図7・19展開ラジエータ図7・20 ロケットに関する動的問題11―9 2宇宙機器・システム
