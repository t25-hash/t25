# 7・3・2 衛星での動的問題

> **章**: 第7章 構造力学
> **節**: 7・20 に示すような項目がある。地上風は機体に曲げモーメントを

a. 動的な荷重衛星の強度荷重条件はそのほとんどが打ち上げられるロケット搭載時に設定される。基本的には上昇に伴う準静的加速度(静的な加速度に振動加速度を上乗せしたもの)が主構造の強度をだいたい決めるが、7・3・1・a項のロケットの振動・音響による動荷重も標定となる。特に振動・音響によるランダム荷重は二次構造や搭載機器にとって標定荷重となる。振動において最も重要なことは共振を避けることであり、ロケットと衛星の関係についていえば共振を避けるには両者の固有振動数を離しておくことである。図7・22のように設計条件として最低固有振動数を要求してすみ分けをする。この条件を剛性要求条件という。剛性要求は固有振動解析であるため、ある程度解析で追い込み、モード試験を行って確認する。解析による予測が難しいのは音によるランダム荷重と分離衝撃であり、試験により確認する。b. 解析法衛星の固有振動解析については応力解析と同じ詳細な有限要素モデルが使われるようになっている。モード試験との対応については有効モード質量(effective modal mass)の大きいモードを採用する。これは衛星の動荷重がロケットとの結合部(PAF: payload attach fitting)から入ってくるベース加振であるためである。この結合部は衛星の試験に使う振動試験台のような剛な面ではないので、大型の衛星でこの結合面を剛としてロケット側との荷重インタフェースを切ると実際にかかる以上の振動荷重を想定することとなる。このため、結合面を剛としないでロケットと衛星との結合荷重解析(CLA)を行って実際にかかるであろう荷重を算定する。
