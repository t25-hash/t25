# 7・3・3 軌道上での振動制御

> **章**: 第7章 構造力学
> **節**: 7・20 に示すような項目がある。地上風は機体に曲げモーメントを

a. 制御系と柔軟構造物の干渉衛星がロケットから分離されて軌道に投入されると、衛星は基本的に大きな力は受けない。軌道上ではミッションを果たすために軌道制御、姿勢制御が最も重要となる。近年は衛星の大型化、大電力化で大きな太陽電池パドル、放熱板、それに大きなアンテナを装備するようになっているが、これらは柔軟な構造で固有振動数が低く、姿勢制御系との干渉が設計上重要となる。すなわち、姿勢を制御しようとしてガスジェット、モーメンタムホイール等で力を出すと、それによって構造振動が誘起される。その構造振動による衛星の運動を制御するためにまたガスジェットを吹くというような姿勢不安定になるおそれがある。柔軟な付属物の固有振動数を制御周波数から離して高くすればよいが、質量の問題で困難であるので、適切な制御によってこの問題を解決する。ダイナミクスだけでなく鏡面精度制御などを含めてスマート構造化がその一つの解決法である。b. 柔軟構造物のダイナミクスの同定適切な制御系の設計の図7・21 液体の入ったタンクの一次元縦振動モデル(C.G.は液体の重心位置)ロケット衛星構体搭載機器15Hz30Hz100Hz図7・22 固有振動数に関してのすみ分けためには柔軟構造物の振動特性が事前にわかっていなければならないが、そのような柔軟な構造を地上で試験することは重力と大気の問題で困難である。部分構造の試験により数学モデルを作り上げていく方法はあるが、軌道上で同定し、それを反映して制御系パラメータの修正をコマンドで送ってやるのが最も現実的である。ただし、そのためにはある程度地上で同定できていることが条件である。
