# 7・4・2 ポゴ

> **章**: 第7章 構造力学
> **節**: 7・4 液体と機体との連成振動

ポゴ(pogo)とは液体ロケットの燃焼末期に起こる縦方向の自励振動である。図7・26に示すように、機体の質量の大半である液体推進薬が200~300秒程度で消費されるので、機体の固有振動数は燃焼末期に向かってどんどん高くなる。一方、推進薬供給系の固有振動数はほぼ一定であるので、両者の固有振動数が燃焼末期[主エンジン停止前、MECO (main engine cut off)]に一致する。このとき、まず、図7・27のように微小な擾乱による推力変動により、微小に機体の縦振動が誘起され、それが液位の少ないタンク底面で振動流として圧力変動が推進薬供給系に伝達され、その圧力変動で推力変動が増幅される。自励振動ではあるが機体の振動数が急激に図7・25 バッフル=φ욡φ욡=φ욡φ욡:液体の質量行列:φ욡により生じた液体による横力:φ욡により生じた液体による重心軸回りのモーメント:重心軸回りのタンクの単位回転により生じた液体の重心回りのモーメント욃=-ω욡11―9 4宇宙機器・システム図7・26 典型的なポゴの発生現象図7・27 ポゴの発生メカニズム高くなるので発散振動とはならず収まる。これは10~20Hz 程度の正弦波振動で、ほとんどすべての液体ロケットでこの振動が生じる。ポゴという用語は子供の玩具の名称(pogo stick、日本ではホッピング)にちなんでいるが、この振動荷重が衛星の主構造の強度を決める場合が多く、また飛行士が搭乗している場合、その振動加速度に耐えられないので許容量まで抑えなければならない振動現象である。図7・26でわかるように一次の機体の縦振動でも二次の縦振動でも起こりうるし[ミニポゴ(mini-pogo)という],1段だけでなく2段の燃焼末期にも起こりうる。この振動を抑えるには両者の固有振動数が燃焼末期に一致しないようにし、さらに系に減衰が付加できればよい。実際には機体の固有振動数を操作することは困難なので、推進薬供給系の途中にソフトスプリング(例えば図7・27のようなヘリウムガスを入れたアキュムレータ)を入れて推進薬供給系の固有振動数を下げることで解決する。H-IIロケットでは事前検討が十分でポゴサプレッサを導入し、1号機からポゴ抑制に成功した。
