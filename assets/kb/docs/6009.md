# 7・4・3 低重力下での液体挙動

> **章**: 第7章 構造力学
> **節**: 7・4 液体と機体との連成振動

a. 飛行制御との関係微小重力の環境下では液体は表面張力が優位となる.宇宙環境利用実験ではこの微小重力環境を有効に使うが,宇宙機の制御の面から見ると,タンク内にある多量の液体がタンクのどこにあるかわからなくては困る.定常運用に入った衛星内のタンクでは表面張力デバイスを使って推進薬をタンク出口に確保するが,打上げ後の過渡的な状態ではスピン開始,分離,エンジン着火などのイベントの前にガスジェットなどで加速度をかけて液体位置を特定できるようにする.このとき液体の大変形とそのあとの平衡位置周りでのスロッシングが動的な検討課題となる.b. エンジン再着火上段液体ロケットで再着火が可能であれば,複数衛星の軌道投入,推進薬の節約など運用の幅が広がる.これまでタイタンセントール,アトラスセントール,アポロサターンなどで再着火実験が行われており,日本でもH-I/MOS-1bやH-IIで少量残留推進薬での再着火が運用されてきた.今後も複数回再着火の運用が計画されている.慣性飛行(コースティング)においては無重力となるので,液体がタンクのどこにあるかわからない.したがって,この状態でエンジンを起動するとガスを吸い込むおそれがある.このようなことを避けるため,補助エンジンで加速度をかけて液体を燃料吸込口のある下部に集める.このとき液体の過渡的な運動により機体にかかるモーメントを機体姿勢制御の立場から検討しておかなければならない.また,液体はタンク壁に沿って移動し,吸込口に集まり,慣性によりそこを通り過ぎて振動を始める.このスロッシングについて適切な減衰(リングバッフルや十字形バッフルにより)を付加し,微小振動になって吸込口を完全に推進薬がぬらす時点で再着火を行う.
