# 7・5・1 全般

> **章**: 第7章 構造力学
> **節**: 7・5 H- /H- A ロケットの構造

H-II/H-IIA ロケットは実用衛星を打ち上げるために宇宙開発事業団(現:宇宙航空研究開発機構,JAXA)が開発した2段式液体ロケットである.実用衛星打上げ用のロケットはN-I,N-II,H-Iロケットと順次大型化し国産化率を高めて,H-IIロケット(H-IIlaunch vehicle)で純国産となった.H-IIロケットの開発理念は,静止衛星2トン級の打上げ能力をもつロケットを全段自主技術により開発する,第1段機体を最重点項目とし第2段機体はH-Iロケットの第2段を活用して大型化することにより開発項目を抑える,などであった.2006年9月現在,H-IIロケットのシステムを簡素化,高信頼性化したH-IIA ロケット(H-IIA launchvehicle)を開発して9機の打上げに成功している.H-IIA ロケットの開発理念は,静止衛星2トンから3トン級の多様なミッションに対応する,H-IIの開発成果に基づきシステムを簡素化することにより打上げコストの低減を図る,などである.
