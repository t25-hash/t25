# 7・5・2 H- ロケット

> **章**: 第7章 構造力学
> **節**: 7・5 H- /H- A ロケットの構造

a. 全体構成H-IIロケットの全機形状を図7・28に示す。機体サイズは、全長50m、コア機体の直径は4mであり、大型の固体ロケットブースタ(SRB : solid rocket booster)を2本もつ。第1段機体のポゴ振動の抑止(pogo suppression)および搭載機器に対する環境条件の緩和(機体が大型化し音響が増大してもH-Iロケットの搭載機器レベルと同等以下)を実現している。第1段機体は、液体酸素(LOX)、液体水素(LH)を推進薬とするLE-7エンジンを装備している。なお、初期推力を増強するため第1段機体の両側に2本のSRB を取付けている。第1段機体の構成は、重心を機体中心付近に確保するためLOX タンクを前方に、LHタンクを後方に配置している。両タンクをつなぐ円筒構造(中央部)でSRB の推力を受ける方式である。また、LHタンク後方の円筒構造(エンジン部)の後端部でLE-7エンジン推力を受ける。第2段機体は、H-Iロケットで国産開発したLOX/LHを推進薬とする第2段機体を図7・29に示すように大型化した構造である。機体構造には性能を最優先させた軽量化設計が採用されており、H-Iの共通隔壁(common bulkhead)構造を踏襲したLHタンクとLOX タンクの一体化による軽量構造と、H-Iで採用したコーン形状の衛星搭載構造(内部は機器搭載部)を廃止して、LHタンクの前方ドーム中間に衛星搭載構造を直接取付けた軽量構造が特徴である。エンジンはH-IのLE-5からLE-5A に改良されている。LHタンク上の衛星搭載構造に取付けられた衛星は、第2段機体LHタンク前端に取付けた衛星フェアリングによって保護される。図7・28 H-IIロケットの全機形状図7・29第2段の大型化b. 第1段機体構造LOX タンク8とLHタンク10は独立タンクであり、円筒の長さを除き基本的に同一構造のアルミニウム合金製の溶接構造である。タンク外表面全体に発泡断熱材が吹き付けられている。円筒壁は、アイソグリッドパネルであり、両バルクヘッドは曲面パネルを溶接で組立てた半球シェル構造である。中央部(center body section)9は、アルミニウム合金製のセミモノコック構造(図7・30)で、貫通ビームの両端にSRB を結合する構造である。ビームが貫通する円筒部には厚板から削り出したスラストパネルを配置し、SRB 推力を伝達分散している。エンジン部(engine section)쑦썬は、アルミニウム合金製のセミモノコック構造(図7・31)で、円筒部後端にLE-7の推力を受けるため十字に組んだビーム(クロスビーム)は円筒部にピン結合されており、箱型断面にして推進系機能部品の搭載場所にもなっている。機体後端には、エンジン部内部およびLE-7の機能品類をSRB の噴煙から保護するため、サンドイッチ構造のエンジンカバーと耐熱繊維製のブランケット쑦썭が取付けられる。c. 段間部段間部7は、第1段機体と第2段機体をつなぐ円筒構造で、上下に二分割されたアルミニウム合金製のセミモノコック構造である。下部段間部の内周に機器搭載用の防振パネルをもっている。上部段間部の前端にスプリングアクチュエータが取付けられており、第1・2段分離時に分離力を発生する。11―9 SRB쑦썮ノーズコーン、前方アダプタ、固体ロケットモータおよびスカートから構成される。モータケースは、超高力鋼製の円筒4セグメントをボルト結合によって組立てたものである。そのほかの構造はアルミニウム合金製であり、前方アダプタに推力伝達部が設けられており、中央部の貫通ビーム端に分離ボルトを介して結合される。スカート下端の周上4箇所(左右合計8箇所)で分離ナットにより発射台に結合される。なお、機体の組立は、初めにSRB 2本を組立て、その間に第1段機体を入れ、前方アダプタ部に中央部の貫通ビームを結合する順序である。SRB は分離可能なリンク機構を介して半径方向および周方向にコア機体に結合される。SRB は前方アダプタおよびスカートに分離用小型モータを装備している。f. 衛星分離部(payload attach fitting,payload adapter)2各種の衛星を搭載するため数種類に標準化して準備されている。衛星の結合方式には、クランプバンド方式と分離ボルト・ナット方式がある。構造様式は機器搭載部に取付けられるように頭裁コーン形状をしたセミモノコック構造で、アルミニウム合金製とCFRP(炭素繊維強化プラスチック)製がある。g. 衛星フェアリング(spacecraft fairing)1先端が丸くなったコーンと円筒からなる構造で、機軸方向に二分割した半殻から組立てられる。構造様式は、アルミニウム製ハニカムコアとCFRP面板からなるサンドイッチ構造である。分割面(機軸方向および第2段の前方スカートとの結合部の周方向)に分離機構が組込まれている。衛星フェアリングの種類は、直径が4mと5mの2種類、また各直径に対して衛星の打上げ個数によってシングル型とデュアル型がある。
