# 7・5・3 H- A ロケット

> **章**: 第7章 構造力学
> **節**: 7・5 H- /H- A ロケットの構造

a. 全体構成H-IIA ロケットの全機形状を図7・33に示す。機体サイズは全長53m、コア機体の直径は4mであり、アスペクト比の小さい大型固体ロケットブースタ(SRB-A)を2本もつ。なお、第2段には独立型のLH/LOX タンクが採用され、また衛星フェアリング側に衛星搭載構造を移して、衛星と第2段機体とが独立になっており、システムの簡素化が図られている。エンジンは、第1段にLE-7を改良したLE-7A エンジンが、また第2段にLE-5A を改良したLE-5B が採用されている。第1段機体は、H-IIで採用した中央部においてSRB の推力を受ける方式を変更して、SRB-A 推力はエンジン部で受けている。CFRP製一体成形の短いモータケースを採用し、LHタンクの中間部とエンジン部において半径方向、周方向に結合分離される。第2段機体は、性能よりもシステムの簡素化を優先した設計が採用されており、軽量化を優先したH-IIの一体型タンク構造を変更してLHタンクとLOXタンクを独立とし、システムを簡素化した構造となっている、衛星搭載構造を第2段機体側から衛星フェアリング側に移すことによって、衛星を取付けた衛星フェアリングを、第2段機体に一度に取付けられる方式が特徴である。b. 第1段機体構造LOX タンク10とLHタンクは、H-IIと基本的に同一構造であるが、両バルクヘッドにスピン成形によるだ円形一体ドーム(長軸:短軸=2:1)を採用し溶接線が削減されている。円筒部はアイソグリッド構造である。中央部(centerbody section)は、アルミニウム合金製のセミモノコック構造であり、地上風による機体の横揺れを抑えるため、整備塔から延びた機体支持構造と結合分離する機構が設けられている。エンジン部(engine section)は、アルミニウム合金製のセミモノコック構造で、厚板から削り出した曲面パネルが円筒全周に採用されている。後端部には、H-IIのクロスビームに貫通ビームの機能を加えた方式を採用し、中心でLE-7A 推力を受け、両端部でSRB-A 推力を受ける構造である。またクロスビームの両端部(4箇所)は発射固定台との結合部にもなっている。機体後端には、H-IIと同様にエンジンカバーとブランケットが取付けられている。c. 段間部段間部9は、発泡材コアとCFRP面板からなるサンドイッチ構造である。前端部に第1・2段分離用スプリングアクチュエータが取付けられている。なお、LOX タンクと結合される下端部に、極低温対策としてアルミニウム製円筒構造を設けている。d. 第2段機体構造LHタンク4は、第1段のタンクと共通設計である。LOX タンク6は、赤道部直径3.2mの回転だ円体シェル構造である。アルミニウム合金製のだ円ドーム二つを溶接結合した構造で、後端部にLE-5B エンジンが取付けられている。両タンクは、CFRP製のトラスサポート5で結合されている。LHタンクの上下端にアルミニウム合金製の前方スカート(4-1)と後方スカート(4-2)を設け、前方スカート前端に衛星フェアリングを結合し、後方スカート後端で段間部と結合する。後方スカート下端部は、H-IIと同様、第1・2段分離部になっている。LOX タンクには、トラスサポート7を介してリング状の機器搭載部8が結合されている。e. SRB-ASRB よりアスペクト比の小さいフィラメントワインディングによるCFRP製モータケース(全長10m、直径2.5m)が採用されている。推力は、モータケースの前端部からパイプ構造(スラストストラット)を介してコア機体のエンジン部に伝達される。モータケースの両端部にてリンク機構を介して半径方向、周方向にコア機体に拘束されている。リンク機構には分離機構が組込まれており、燃焼終了後コア機体から分離される。分離モータは後方にのみ取付けられている。f. 衛星分離部2商業衛星用に衛星結合部を欧米のロケットと共通化した3種類の衛星分離部が開発された。また3トン級の大型衛星を搭載できる衛星分離部が開発されている。なおH-IIA で追加された衛星分離部はアルミニウム合金製である。g. 衛星フェアリング1衛星フェアリング本体は、製造工程の簡素化が図られているが、基本的にはH-IIと同じ構造である。しかし、衛星フェアリングの後端部に衛星搭載構造3が追加されている。この変更により衛星が衛星フェアリング側に取付けられるため、衛星と衛星フェアリングを第2段機体に取付ける手順が簡素化されている。h. SSB(solid strapon booster)打上げ能力を増加させるた図7・33 H-IIA ロケットの全機形状め、SSB が2本または4本追加装備できる。SSB は、SRB-A と同様、第1段のLHタンク中間部とエンジン部の2箇所で結合分離される。
