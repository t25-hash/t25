# 7・6・2 モータケース

> **章**: 第7章 構造力学
> **節**: 7・6 M- ロケットの構造

固体燃料ロケットのモータケースは、その内部が固体燃料の燃焼室となるため、軽量化とともに高い耐圧性能が要求される。したがって、比強度の大きな材料が使われている。a. 金属製モータケース第1段M-14モータケースと第2段M-24モータケースは、その寸法や製造工程等の理由から金属製である。ただし、極力軽量化を図るために、従来のモータケース材よりも比強度の高い高張力マレージング鋼(maraging steel)HT-230M を新規開発して、これらの円筒部に用いている。M-14モータケースは、全長11.00m、円筒部内径2.50m、円筒部最小板厚6.4mmであり、上段側と下段側の二つのセグメントよりなる。M-24モータケースは、全長4.76m、円筒部内径2.50m、円筒部最小板厚5.8mmであり、一体型であることを除き、製法・材料はM-14モータケースと同一である。また、最大使用圧力は両方とも5.9MPaである。これらのモータケースの円筒部はHT-230M 材製の薄板を円筒状に巻いた後、電子ビーム溶接によって接合される。その後、二次溶体化処理および時効処理の熱処理を施す。前後の鏡部は、加工性のよいベイナイト鋼HT-150材製のプレス成形品から機械切削により作られ、熱処理後、電子ビーム溶接で円筒部に結合される。b. CFRP 製モータケース上段構造の重量は打上げ性能に対する感度が第1段、第2段に比べて大変大きいため、厳しい軽量化が要求される。そのため、さらに比強度の大きな材料を使用する必要がある。従来のミュー(Mu)シリーズロケットの上段モータケース材料としてはチタン(Ti)合金をおもに用いていたが、この第3段M-34モータケースはCFRPを採用した。これは、CFRPがTi合金に比べて潜在的性能が高いこととコスト面で有利なためである。本モータケースは全長2.26m、内径2.15mで、最大使用圧力5.9MPa,CFRPフィラメントワインディング(FW : filamentwinding)製である。図7・35に示すように、インプレーン層、フープ層、前・後部スカート、前・後部ボスで構成されており、また、前・後スカート間は機軸方向の荷重を受け持つ一方向繊維(UD)層が挿入されている。前部ドームの口元径および前部ボス11―9 7第II部基礎編外径は、ドームの繊維方向最大応力とボスの最大応力の制約下で軽量化を標定とした概略最適値になっている。また、後部ドームの口元径は、ドームの繊維方向最大応力を下げるために推進薬内孔の制約による限界まで小さく設計されている。インプレーン層で形成されるドーム部形状は網目理論により設計されている。ただし、製造上厚くならざるを得ないボス口元周りでは、網目理論では現れない曲げモーメントが発生し、層間せん断応力が大きくなる。これが強度上問題となる前部ドームでは、インプレーン層の最外層のみを前部ドームの半径500mm位置より巻き始めるずらし巻きにして、口元周りの層間せん断応力の緩和を図っている。M-V ロケット5号機からは、第2段モータケースを金属製のM-24モータケースからCFRPのFW 製M-25モータケースに変更した。本モータケースでは、FW の巻き方、局部補強等に新しい技術を適用し、高性能化を達成している。
