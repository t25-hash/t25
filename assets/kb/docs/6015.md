# 7・6・3 段間接手

> **章**: 第7章 構造力学
> **節**: 7・6 M- ロケットの構造

a. 1・2段接手M-V ロケットの打上げ能力向上のために,第1段モータは推力を極力増加させて,重力損失の低減を図っている.その結果,1段燃焼終了時の動圧が大きく,分離された2段目は分離直後から強力な姿勢制御が必要となる.2段目の姿勢制御はモータの主推力を利用するために,分離後,ただちに2段モータ推力を立ち上げる必要がある.そのため,1・2段分離と2段モータ点火を同時に行うファイア・イン・ザ・ホール(fire in the hole)方式が1・2段接手に採用されている.本接手は図7・36に示すように,4340鋼製の1段側構造物,アルミニウム合金製の開傘パネルおよび2段側構造物よりなる.1段側構造物は空げき率72%のラティス構造で,ファイア・イン・ザ・ホール点火された第2段モータの噴射ガスが無理なく通過できる構造となっている.中間の開傘パネルは1/3周円筒パネル3枚から成り,各パネルは内側の1/3周リングと外側の縦通材により補強されている.各開傘パネルと1段側および2段側構造物とは,アルミニウム合金製リング上にFLSC(flexible linear shaped charge,V 形成形爆破線)を周状に配した分離接手(FLSC接手)を介して結合されている.FLSCの起爆により,このリングが切断された後に,3枚の開傘パネルは2段ノズルとの衝突を避けるために花弁状に開傘す図7・35 M-34モータケース概要図7・34 M-Vロケット主要構造要素(M-V1号機)11―9 8宇宙機器・システム図7・36 1・2段接手概要図7・37 前部FLSC接手断面図る.前部FLSC接手は,この部分に作用する引張力が圧縮力に比べて約半分と小さいことを利用して,図7・37に示すように,大きな圧縮力を厚い圧縮側リングで受け持ち,小さな引張力は薄い引張側リングで受け持つ構成となっている.圧縮側リングの下端は,開傘パネルに押し当てられているだけである.この構成によって,FLSCが薄い引張側リングのみを切断することで2段側構造と開傘パネルとを分離できることになり,薬量の低減,ひいては分離衝撃の低減を実現している.b. 2・3段接手ンストリンガ構造(skin-stringer structure)である.その中ほど内側には,第2段計器搭載部が設けられている.本接手の外板は,軽量化のために設計制限荷重内の弾性座屈を許容する設計になっており,座屈後はせん断荷重のみを支えるせん断場として扱われている.ただし,地上での荷重では座屈しないように配慮されている.本接手は,衛星に近いために,1・2段接手の採用された衝撃の大きいFLSC接手ではなく,作動時の衝撃が比較的小さいマルマンクランプ方式の分離接手が採用されている.
