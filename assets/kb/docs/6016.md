# 7・6・4 ノーズフェアリング

> **章**: 第7章 構造力学
> **節**: 2・3 段接手は、アルミニウム合金製のスキ

ノーズフェアリングは、全長9.19m、外径2.5mのハニカムサンドイッチ殻構造であり、分離機構で結合されている二つの開頭片で構成されている。サンドイッチ殻の表面板には、軽量化だけでなく、外荷重による構体の変形を抑えて、衛星搭載領域を広くとることを目的として、比強度・比剛性の高いCFRPが適用されている。二つの開頭片どうしおよびこれらとロケット本体とは、連続した分離機構で結合されている。図7・38にその断面図を示す。プレートは、シアピンで上部ハウジングに固定されており、かつ、下部ハウジングからのボルトに結合されている。このボルトを締め付けると、ボルトに生じた引張力に見合う圧縮力で、上部ハウジングと下部ハウジングとが互いに接触面(分離面)で押し付けられる設計になっている。上部ハウジングとプレートとに囲まれる部分には、テフロンサポートで固定されたひも状の火薬MDF(mild detonat-ing fuse、金属被覆導爆線)を装てんした偏平なステンレス管が挿入されている。点火によるMDFの爆発力は、ステンレス管を押し図7・38 ノーズフェアリング分離機構断面図広げ、プレートを押し出す力となり、その力によりシアピンが切断され、分離する仕組みになっている。本分離機構の特徴は、ボルトとプレートという小さい部品の移動で分離が完了することであり、開頭片全体を移動させなければ分離できない機構に比べて、発生する衝撃が小さい。分離後、各開頭片は下端のヒンジ回りに回転し、60°でヒンジから外れ投棄されるクラムシェル方式の開頭になっている。
