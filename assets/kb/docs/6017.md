# 7・6・5 後部筒

> **章**: 第7章 構造力学
> **節**: 2・3 段接手は,アルミニウム合金製のスキ

後部筒は外板外形2.61m,高さ1.8mのアルミニウム合金製の円筒殻で,縦通材とリングとで補強されている.外板にはアクセス窓があり,第1段ノズル周りの搭載機器にアクセスできる.外側4箇所にはSMRC(solid motor roll control)を搭載するためのマウントとカウリングが設けられている.陸域観測技術衛星(ALOS : Advanced Land Observing Satel-lite)の開発は,(1) 国内およびアジア太平洋地域などの諸外国の地図作成および更新(地図作成)(2) 世界各地域の持続可能な開発に必要な地域観測(地域観測)(3) 国内外の大規模災害の迅速な状況把握(災害状況把握)(4) 国内外の資源探査(資源探査)のため,高分解能の陸域観測データの収集を行うとともに,将来の地球観測に必要な技術開発を行うことを目的としている.この目的を達成するためALOS は,(1) パンクロマチック立体視センサ(PRISM,約550kg)(2) 高性能可視近赤外放射計2型(AVNIR-2,約200kg)(3) フェーズドアレイ方式Lバンド合成開口レーダ(PAL-SAR,約660kg)の3種類の大型センサを搭載した4トン級太陽同期地球観測衛星である.打上げ時のALOS コンフィギュレーションを図7・39に示す.この中で特にPRISM は2.5m地上分解能,3~5m標高抽出精度を得る3式の放射計で構成(立体視)される高分解能地球観測センサであり,指向軸精度に対する要求が従来の観測衛星に比べて1オーダから2オーダ厳しいものとなっている.軌道上における衛星のセンサ指向軸に対する主要誤差要因は以下の項目である.(1) 衛星の姿勢制御誤差(2) センサ機器アライメント誤差(3) その他軌道上変動誤差等衛星構造の観点からは,振動を含めた軌道上変動誤差がクリティカル項目である.その中でも特に熱ひずみによる指向軸誤差変動は,最も予測困難な現象であり,従来衛星では誤差配分上マージンを大きく見込むことで性能を満足させてきたが,さらなる高精度を図7・39 ALOS コンフィギュレーション(打上げ時)要求されるALOS は,従来の強度,剛性要求等のほかに衛星主要構造の熱変形を抑制することが必須の技術となっている.
