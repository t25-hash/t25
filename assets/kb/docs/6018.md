# 7・7・1 構造系技術的課題

> **章**: 第7章 構造力学
> **節**: 7・7 ALOS 衛星の構造

高分解能観測衛星は1/1000°オーダの指向決定精度が必要であり,これに伴い熱ひずみによる指向変動(ランダム変動)も,全軌道上において1/10000°オーダが必要となってきている.一般に構造物の熱ひずみを抑制するためには,熱設計,構造設計の両面からアプローチされる.(1) 構造物の温度分布および変動量を最小限に抑制する.(2) 温度変動があっても伸縮しない材料・構造様式を採用する.衛星構造各部の温度変化を適切な温度制御により抑制することが設計の前提となる.この温度変化は,地上常温ポテンシャルと,軌道上での平均温度ポテンシャルの差,および各部の温度ばらつきの時系列変化を意味する.前者の温度変化はバイアス熱ひずみを起こすが,光学系の性能劣化に注意すれば軌道上校正によりキャンセルできる.ALOS では,後者に起因するランダム変動に対する構造設計手法について検討された.ALOS 構造設計上の技術的課題は以下のとおりである.(1) 観測センサ指向精度要求を達成するための,軌道上構体変形の最小化(熱ひずみ最小化).(2) 大質量ミッション機器搭載に対応した,強度・剛性の確保.(3) 並行開発の機器設計変更に対応可能な柔軟性の確保.
