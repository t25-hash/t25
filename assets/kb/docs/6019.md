# 7・7・2 設計方針

> **章**: 第7章 構造力学
> **節**: 7・7 ALOS 衛星の構造

これらの技術課題を踏まえ，ALOS構造系の構成，機能配分は
以下のとおり設定された．
(１)軌道上での衛星姿勢基準となる恒星センサ（STT:Star
Tracker），ジャイロとPRISMを高剛性の光学ベンチ面上
に搭載したモジュール化を採用し，PRISMの指向軸変動を
最小化．
(２)主構体に積層設計の自由度が大きいCFRPトラス構造を採
用し，大荷重を堅牢に支持するとともに，軸方向の線膨張係
数をほぼ零（－0.06ppm/℃）に設定．
(３)構造系の主要性能である強度・剛性・低熱ひずみの機能をト
ラス構造に，機器実装機能をパネル構造に機能分けすること
で，機器インタフェース変更に左右されにくい構体系の実
現．
7・7・3インテグレーション設計
ALOSはモジュール構成を採用することにより，それぞれのモ
ジュール機能に応じた最適化を図るとともに，良好なインテグレー
ション性および輸送性が確保されている．モジュール単位の並行組
立作業によるインテグレーション期間の短縮化（スケジュールの柔
軟性）にも適応している．ALOSの構成を図7･40に示す．
