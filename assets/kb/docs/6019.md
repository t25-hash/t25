# 7・7・2 設計方針

> **章**: 第7章 構造力学
> **節**: 7・7 ALOS 衛星の構造

これらの技術課題を踏まえ,ALOS 構造系の構成,機能配分は以下のとおり設定された.(1) 軌道上での衛星姿勢基準となる恒星センサ(STT : StarTracker),ジャイロとPRISM を高剛性の光学ベンチ面上に搭載したモジュール化を採用し,PRISM の指向軸変動を最小化.(2) 主構体に積層設計の自由度が大きいCFRPトラス構造を採用し,大荷重を堅牢に支持するとともに,軸方向の線膨張係数をほぼ零(-0.06ppm/°C)に設定.(3) 構造系の主要性能である強度・剛性・低熱ひずみの機能をトラス構造に,機器実装機能をパネル構造に機能分けすることで,機器インタフェース変更に左右されにくい構体系の実現.
