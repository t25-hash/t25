# 7・7・5 熱ひずみ解析結果

> **章**: 第7章 構造力学
> **節**: 7・7 ALOS 衛星の構造

衛星構造の熱ひずみによる軌道上1周回中(約100分)の指向軸変動は、地上零ひずみ状態を基準としたときの軌道上高温、低温ケース各10分刻みに対する各節点の変位より、姿勢系機器(STT)を基準とした観測機器(PRISM)指向軸の相対変動を求めている。軌道周回中の温度はフライト解析用熱数学モデル(有限差分法、約5700節点)により求められた温度条件を用い、構造の変位は各部材の物性値、締結条件、温度分布等を、衛星応力解析用数学モデル(有限要素法、約5万3000節点)を用いて計算される。図7・42に光学ベンチの代表的変形解析結果を示し、図7・43に軌図7・44 軌道上指向軸変動(STT 基準)道上指向軸変動解析結果(各機器ごと)、図7・44に軌道上指向軸変動解析結果(STT 基準)を示す。a. STT挙動STT は光学、放熱視野確保のため光学ベンチの角部に搭載されており、主構体(PRISM トラス)の影響を受けやすく、光学ベンチ面外方向の挙動に支配されている。b. PRISM 挙動STT を基準としたPRISM 3放射計の軌道上指向軸変動はランダム成分に支配されることなく、サインカーブに近い挙動を示している。これは衛星の日照/日陰の温度フェーズの遷移にほぼ沿ったものとなっている。また、変動値は満足する結果が得られている。
