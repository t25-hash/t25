# 7・8・3 往還機構造

> **章**: 第7章 構造力学
> **節**: 7・8 構造力学の最近の話題

アメリカのスペースシャトル(space shuttle)のような宇宙往還機は、大気圏への再突入時に厳しい空力加熱環境にさらされる。この環境から機体を保護するために、熱防護システム(TPS : ther-mal protection system)が機体外表面に装着されている。往還機機体構造は、構造自身がTPS を兼ねている耐熱構造(hot struc-ture、ホットストラクチャ)と、外部にTPS を装着して構造温度を比較的低く抑えた通常構造とに大きく分類することができる。ホットストラクチャとは、空力加熱環境に十分耐えることの可能な耐熱材料を機体構造に用いた構造様式であり、外部に断熱材を必要としない。このため、構造内部に断熱材(機内断熱材)を装着して機体内の温度を減じる必要があるが、機体内が比較的高温になることが許容される部位(例えば舵面構造など)においては、断熱材料の質量を小さく抑えることが可能であり、機体質量の軽減に有効な構造様式となりうる。一方、構造内部の温度を比較的低く抑える必要がある部位については、機内断熱材の必要質量が増加することと、一般的な構造用材料と比較して耐熱材料の比強度が低いことから、外部にTPS を装着した通常構造のほうが質量面では有利なことが多い。そこで本項では、耐熱構造およびTPS+通常構造について、アメリカのスペースシャトルや最近の試験機、およびわが国で開発を実施してきた宇宙往還技術試験機(HOPE-X)を例に挙げながら、それぞれに特有の制約条件・技術課題等についてまとめることとする。a. 耐熱構造(ホットストラクチャ)再突入時の最高温度が1500°C程度に達するアメリカのスペースシャトルの機体先端(ノーズ)部および主翼前縁部には、この高温に耐えうる軽量材料であるカーボン/カーボン複合材(carbon/carbon composites、以下、C/C材)が適用されている。C/C材とはマトリックスとその強化繊維の両方とも炭素である繊維強化複合材料であり、非酸化雰囲気中では2000°C程度の高温まで強度低下がなく、また、構造部材として十分な強度を有することから、ノーズ部および主翼前縁部はホットストラクチャとして設計されている。わが国で開発が進められていたHOPE-X 機体構造においても、ノーズ部と主翼前縁部はC/C材によるホットストラクチャが採用されていた(図7・48)。これらの部位は加わる空力(飛行)荷重も小さいことから、補強材を含む一体成形で製造されており、断熱材を介して機体構造の他の部分に結合されている。図7・49,7・50にHOPE-X 用に開発されたノーズ部および主翼前縁部の試作供試体を示す。ただし、C/C材は酸化雰囲気において500°C以上で著しい酸化損耗が生じるため、耐酸化コーティング(代表的コーティング材:SiC-SiO系)が必須であり、一般にはコーティング材の耐熱温度のほうがC/C材のそれより低いため、C/C材の使用可能上限温度は耐酸化コーティングの耐熱温度で決まってしまう。そのため、より高耐熱の耐酸化コーティングの開発は現在でも重要課題の一つである。また、耐熱温度はC/C材よりも劣るが、1300°C程度まで使用可能であり、耐酸化コーティングが不要となりうるセラミックス基複合材料(CMC)であるSiC繊維強化SiC材(SiC/SiC材)を用いたホットストラクチャに関する研究開発も盛んに行われている。HOPE-X 開発においても、主翼前縁部のスケールモデルの試作を実施している。以上のように往還機構造のノーズ部、主翼前縁部はホットストラ図7・48 宇宙往還技術試験機(HOPE-X)熱防護材配置構想図7・49 HOPE-X C/Cノーズ部試作供試体図7・50 HOPE-X 熱防護システム(TPS)+通常構造宇宙往還機の胴体部等で搭載物が存在する部位の機内温度は、例えば燃料搭載部はその燃料の自己発火温度以下に、電子機器搭載部についてはそれらの動作環境条件から60~70°C以下に抑える必要がある。また、有人機に対しては、より低い機内上限温度が要求される。このような部位に対しては、TPS を外部に装着した通常構造が一般に適用される。これは、機内上限温度が存在するため、その温度から空力加熱にさらされる機体外表面の最高温度までの間に耐熱上限温度がある構造材料を使用すれば、外部環境から構造材料耐熱温度までの断熱を外部TPS、構造温度から機内温度までの断熱を機内断熱材が受け持つような熱設計が可能であるからである。機体外部に装着されるTPS としては、スペースシャトルに適用されたセラミックタイル(ceramic tiles、図7・53)や可撓断熱材(flexible external insulations、図7・54)がよく知られており、HOPE-X へも同様の材料の適用が想定されていた(図7・48)。これらの材料は機体構造に接着剤で装着されるため、飛行中の機体変形に起因するTPS の脱離や破壊を防止することが、往還機の通常構造部分には要求される。すなわち、通常の航空機やロケットのセミモノコック構造においては、軽量化の観点から機体外板は制限荷重以下での座屈が許容され、せん断力と座屈後の有効幅相当分の図7・51 X-38実験機(モックアップ)図7・52 X-38舵面供試体(ドイツ、MAN においては、セラミックタイルを装着したことによる座屈荷重向上効果を取り入れた設計を実施している。しかし、このようなTPS 保護のための制約により、機体構造質量は通常の航空機ベースの質量推算値と比較し、どうしても増加してしまうことは避けられない。設計の初期段階でこの質量増加分を認識し、質量配分に余裕をもつことが、往還機構造設計においては特に重要である。また、近年、セラミックタイル等の運用性の悪さを解消するために、外側を耐熱金属もしくはCMC材で覆い、内側に断熱材を配したスタンドオフTPS(stand-offtype TPS)が盛んに提案されている。その一例を図7・55に示す。このようなスタンドオフTPS は外表面が硬い材料で覆われているため、セラミックタイルのようにもろくなく、また、機体構造に機械的に結合されているため、接着結合のような不確定さを排除できる。このような理由から破損・交換の可能性は相当低下することが期待できるが、構成が複雑であるため、セラミックタイル等と比べて質量は増加してしまう。そこで、実際には飛行には至らなかったが、アメリカの試験機X-33においては、機体構造をフレームと縦通材のみで構成し、スタンドオフTPS の表面板に機体構造の外板としての機能をもたせ、機体質量を削減するエアロシェルという構造およびTPS 構想が提案された(図7・56)。このように、スタンドオフTPS にお図7・53 セラミックタイル図7・54 可撓断熱材図7・55 スタンドオフTPS の一例11―106宇宙機器・システム図7・56 X-33のエアロシェルいては、TPS
