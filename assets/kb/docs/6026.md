# 8・1・1 機構の分類

> **章**: 第8章 機構
> **節**: 8・1 機構設計の方法と手法

人工衛星に使われている機構を機能別に分類すると，以下のよう
になる．
(１) 姿勢制御機構（衛星バス機器）
･ジャイロスコープ：衛星の角速度を検出
･リアクションホイール：姿勢制御トルクの発生
･スラスタ：速度，角速度の付与
(２) 保持解放機構（hold and release mechanism）
･保持機構：ロケット振動加速度に対する構体への固縛
･解放機構：固縛力の解放，放出
(３) 展開機構（deployment mechanism）
･太陽電池パドル展開同期機構：複数パネルの整列展開
･アンテナ展開機構：折り畳み収納機器の展開
･マスト伸展機構：長尺構造物の伸展展開
･ラッチ機構：展開した機器の安定保持，剛性付与
(４) 追尾機構（tracking mechanism）
･太陽電池パドル駆動機構：太陽追尾
･ポインティング機構，ジンバル機構：指向角度制御
一方，これらの代表的な機構を使用するアクチュエータで分類す
ると，以下のようになる．
･火工品：火薬の爆発圧力を利用した保持解放機構
･ばね応用機構：弾性反発力を利用した展開機構
･モータ駆動機構：電磁力を利用した運動制御機構
･パラフィン：加熱相変化の体積膨張力を利用した直動機構
･形状記憶合金：形状記憶効果を利用した保持解放機構
･圧電素子：電歪（わい）効果を利用した微調整機構
