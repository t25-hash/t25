# 8・1・2 機構の設計

> **章**: 第8章 機構
> **節**: 8・1 機構設計の方法と手法

前述のように、設計における検討項目は地上の製品と変わらないが、展開構造物が多いことと無重力場であることが宇宙機器の特徴であり、これらに関する設計については十分注意する必要がある。本項では、なかでも重要な機構設計解析と制御設計および多用される基本機構について述べている。a. 機構設計解析機構設計の例として、通信衛星に搭載される展開型アンテナ反射鏡面(deployable antenna reflector)の展開機構の設計と、基本設計パラメータの決定に必要な解析について述べる。通信衛星に搭載されるアンテナ反射鏡面は、必要な開口直径の増大に伴い、ロケットフェアリングへの収納寸法の制限から、展開構造や機構を採用して収納することが必要になる。開口直径が2~4m程度の反射鏡面の場合は、シェル構造を用いたソリッド鏡面を、一軸回りの全体の跳ね上げ、あるいは分割した鏡面の部分展開と全体の跳ね上げによって展開させる様式をとる。開口直径が5m以上になると、反射鏡面をソリッドで構成することは不可能であり、例えば反射面となる金属メッシュと、金属メッシュをパラボラ形状に近似するケーブル構造、そして反射面を広げ支持するための背面支持構造とで構成する。このように、展開構造はさまざまな形態をとるが、展開型アンテナ反射鏡における機構設計は以下のような共通の基本的考え方で進めることができる。. 制御設計機構の制御には、展開機構を地上試験で検証した展開角度、角速度プロファイルで駆動し展開信頼性を維持するように制御する展開制御と、通信用アンテナのように姿勢擾乱に対してつねに指向方向を一定の精度で維持する追尾制御がある。展開制御は一般的にオープン制御で行われ、トルクバジェットの配分により受動的に制御する場合と、モータを駆動回路によって制御することによって能動的に行う場合がある。展開制御を行うことによって、展開角度が一定の速度で変化するように、あるいは慣性質量が大きくなる姿勢で加速度が最小になるように制御し、展開信頼性を向上させることができる。追尾制御の例として、通信衛星におけるアンテナ指向方向制御を挙げることができる。大型のアンテナ反射鏡面を用いた場合にはビームが小さく絞られるため、指向方向精度の要求条件が厳しくなり、衛星本体の姿勢精度では不十分となるため、アンテナ指向方向制御システムが必要になる。指向方向制御システムは、駆動回路と駆動機構、ビーコン波の受信装置で構成される。アンテナ指向方向制御への擾乱として大きいものは熱変形と軌道制御時、アンローディング時の姿勢制御力である。これらの擾乱に対して、太陽電池パドルの振動、アンテナ反射鏡面の振動を考慮してダイナミクスを導出し、制御系を設計する。また、機構の展開制御はコンティンジェンシーを考慮することも必要であり、運用プランの策定と、適切なコマンド/テレメトリ項目の設定が重要である。c. 熱設計と熱制御機構には衛星構体内部で使用されるものより、衛星外部の厳しい温度環境で使用されるものが多い。したがって、熱変形に留意するのが大切で、温度予測によるクリアランス管理が重要である。ここでは機構としての熱設計と熱制御について述べる。. 可動部機構部に存在する可動部には適切なクリアランスあるいはプリロードが必要であるが、相対する材料の線膨張係数と温度の差によってクリアランスと荷重が変化する。したがって、可動部が所望の機能を保つためには、部材温度によるクリアランス変動を正確に予測しておく必要がある。クリアランス管理は、(1) 製造公差を考慮した初期すきま(2) 常温からの温度変化(3) 材料の線膨張係数を考慮して行う。設定クリアランスが大きすぎて本来の可動部の機能上の問題がある場合は、材料を変更するか、もしくはヒータによる温度制御を採用する。. モータモータには巻き線、ロータとステータのクリアランス、軸受などの温度変化による特性変化、あるいは焼損の観点からの温度制限がある。したがって、動作時にはサーモスタットとヒータによる温度制御を必要とする場合が多い。. ダンパばねの蓄積エネルギーを動力源とする展開機構では、展開に従って弾性エネルギーが運動エネルギーに変換され、展開完了時にストッパに衝突する。この衝突のエネルギーを低減するためにダンパを使用することが多い。多くのダンパは粘性流体を使用するものであり、温度による特性変化が大きい。そのため、モータと同様にサーモスタットとヒータによる温度制御を必要とする。f. 強度設計機構部に作用する荷重は、打上げ時の振動荷重と使用時の荷重に大別される。前者は振動応答解析によって求められ、軌道上で正常に機能するために特に可動部に損傷が起きないかどうかを検討する必要がある。後者には動力学的に発生する伝達系駆動力と支持系反力のほかに、熱変形荷重も考慮する必要がある。伝達系には歯車や減速機、あるいは回転軸や引張圧縮リンク、およびリンクどうしを結合する継手部材などがあるが、それぞれに作用する力を正確に算定し、強度マージンや寿命を評価する。一方、支持系では特に軸受負荷に対して注意が必要である。軸受にはいろいろな形式があるが、作用する荷重、すなわちプリロードやラジアル荷重とアキシアル荷重などを正確に算出し、許容荷重に対するマージンや寿命を評価する。また、構造部材の熱変形による荷重が機構系、特に軸受部に作用する場合があり、可動部以外の部材変形にも注意する必要がある。g. 剛性設計軌道上で展開後に問題になる固有振動数は、展開構造物を支持している展開機構や駆動系であることが多い。したがって、強度のみでなく剛性にも十分注意しなければならない。特に、駆動系では減速機の特性、支持系ではクリアランスを必要とする軸受の固定方法(予圧方法)に注意が必要である。展開構造物の11―112宇宙機器・システム場合は前述のようにラッチ部の剛性が問題になることが多いが、ストッパにばねで押し付けるいわゆるあて止めの場合は、接触変形や構造変形を正しく評価しておくことが大切である。なお、展開機構に限らずアンテナポインティング機構や太陽電池パドル駆動機構のようにつねに駆動する場合も、剛性が問題になることが多い。その設計と評価についてはc・iii項に述べたモータ駆動展開機構の設計と基本的には同じである。
