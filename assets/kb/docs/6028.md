# 8・2・1 機械インタフェース

> **章**: 第8章 機構
> **節**: 8・2 システムインタフェース設計

機構設計の中でシステムの機械インタフェースに関わる項目としては、(1)インタフェース荷重と姿勢擾乱、(2)アライメント、(3)システムあるいは他のアペンデージとの干渉、が挙げられる。a. インタフェース(interface)荷重と姿勢擾乱技術試験衛星ETS-VIのアンテナ反射鏡面の場合には、跳ね上げ機構とアンテナタワーとのインタフェース面荷重が規定された。反射鏡がこのインタフェース面荷重を超過するようなラッチ速度で展開を完了した場合、アンテナ反射鏡面を片持状態で支えるタワーの根元が破壊する。技術試験衛星VIII型(ETS-VIII)のような巨大なアンテナ反射鏡面を搭載する衛星では、システムへの取付インタフェース点における荷重に加え、展開の開始時、展開途中のシステム姿勢制御への擾乱も考慮することが必要となる。ETS-VIIIでは、打上げ時に反射鏡面構造を固縛している保持解放機構が展開を開始する場合の反動トルク、展開を終了する場合のラッチ衝撃トルクがシステムの許容擾乱トルクを超えないように、ダンパ等の速度制御要素を利用して初期展開速度およびラッチ速度の適正化を行っている。b. アライメント展開構造が機構を介してシステムに固定される場合、そのインタフェース面のアライメント規定と管理がミッション性能に大きな影響を及ぼす。展開構造が機構を介してシステムに固定される場合、そのインタフェース面は展開構造全体のアライメント基準となる。このアライメント基準面を機構の取付面そのものにしてしまうと組立時に基準を設定することができないため、計測コンフィギュレーションを考慮した適当な位置にアライメントミラーを取付けたり、取付精度をリーマボルトで決まるアライメント精度で保証するといった方法がとられる。c. システムあるいは他のアペンデージとの干渉展開構造物が大型になってくると、展開途中の経路で他の展開構造や衛星バス構造との干渉がないように、あるいは展開完了時に他の通信機器のレイパスとの干渉、姿勢制御用センサの視野への干渉を起こさないように慎重に設計を行うことが必要である。
