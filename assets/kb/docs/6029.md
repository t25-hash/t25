# 8・2・2 電気インタフェース

> **章**: 第8章 機構
> **節**: 8・2 システムインタフェース設計

機構設計上、考慮すべきシステム電気インタフェースとしては電源(駆動回路、ヒータ)、コマンド(駆動回路、ヒータ)、テレメトリ(駆動回路、温度、マイクロスイッチ)、電磁干渉(EMI: elec-tromagnetic interference)、静電気放電(ESD : electrostatic dis-charge)等が挙げられる。電源については、使用電圧、リップル電圧の許容量、電圧ドリフト変動の許容量、電力等について、システムの仕様、制約条件との調整が必要であり、コマンド・テレメトリに関しては通信プロトコル、信号規格、チャネル数、サンプルレートなどの調整が必要であり、データを蓄積伝送する場合にはその運用のタイミングや蓄積容量の調整が必要となる。さらに、システムに有害な影響を与える電気のノイズの発生の防止、また駆動回路の異常動作時のシステム保護のための遮断回路の設置が求められる。
