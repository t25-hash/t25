# 8・3・1 地上展開試験方法(ground deployment test method)

> **章**: 第8章 機構
> **節**: 8・3 試験評価法

近年の論文に見られるアンテナ鏡面、太陽電池パドル等、展開構造の展開機構特性の評価方法には以下のようなものがある。(1) 展開時間および最終(ラッチ固定時)展開角速度の測定と設計値との比較(2) 軌道上展開特性の解析予測(3) 軌道上展開信頼性予測これらの評価方法は、単独ではアンテナ展開特性を正確に予測するには不十分である。最初の二つの評価方法は多くの報告書の中で地上展開試験の主たる評価項目として見られるが、地上試験によって得られた結果は重力および大気の影響を受けており、これらの地上試験環境要因が定量的に見積もられない限り、試験によって得られた展開特性は実際の軌道上特性とは異なった値となる。言い換えれば、展開機構の駆動力が設計値より小さい場合でも重力によって外力トルクが発生すれば、展開運動は加速され、結果として設計予測値とよい一致を示し、誤判断をする可能性もある。逆に駆動力が過大であったにもかかわらず大気によって減速され、ラッチ固定時に過度な衝撃を許すこともありうる。機構レベルの試験に基づいた軌道上展開特性の予測は、動作環境の模擬をかなり正確に行うことができるという点で有利である。展開構造物の軌道上展開シミュレーションは、その展開構造の初期設計および展開機構の初期展開信頼性評価において重要な意味をもつ。展開構造の初期設計においては、ラッチ速度に対するラッチ衝撃応答解析をもとに剛性および強度検討を行い、ラッチ速度の許容値を定める。また、初期展開信頼性予測は部品個々のトルク特性の積み上げ値によって駆動力マージンを求める。三澤らは統計的な解析手法により展開機構の展開信頼性(deployment reliability)を予測する手法を提案している。しかしながら、これらの駆動トルクや抵抗トルクは組立工程や環境試験の実施によって大きく変化する可能性がある。したがって、組立工程、環境試験の実施後において機構特性に変化がないことを地上試験によって確認する必要がある。このように、展開機構の信頼性を地上試験評価で保証するために、各開発レベルに応じた適切な試験評価を一貫した評価基準で積み上げていくことが極めて重要である。a. 衛星システムレベル試験プログラムが次の試験レベルに移行すると、供試体展開機構(あるいは機構部品)は組立工程を経たり、打上げ振動、音響、軌道上熱真空環境にさらされることになる。これらの組立工程による影響や振動、音響、熱真空環境による影響により、展開機構特性や機構部品特性に変化が生じることが考えられる。したがって、各試験レベルにおいて展開試験を実施し、特性の評価を行う必要がある。ところが、試験レベルがシステムに近づくにつれて評価できる項目が大きく制約を受けることになる。例えば、鏡面コンポーネントレベルやアンテナモジュールレベル、衛星システムレベルにおいては重力や大気の影響を大きく受け、正確な地上展開シミュレーションは極めて困難である。したがって、われわれは各試験レベルに応じた最適な試験評価方法を検討する必要がある。b. 機構アセンブリーの展開試験図8・17に一対の鏡面展開機構に対する機構アセンブリーレベルでの展開試験装置を示した。液体窒素を用いた冷却装置によって低温条件を与え、電熱線ヒータを用いて高温条件を与える。真空容器の中に設置したそれぞれの展開機構の展開軸は連結アームを用いてお互いに結合されている。展開軸の回転運動は連結アームに設けられた回転軸を介して真空容器の壁面の外側へ導いており、この際に磁気流体型カップリングによって真空容器のシールを行う。実際の鏡面の展開軸回りの慣性モーメントを模擬するために、慣性質量ダミーとなるはずみ車を、チェーンを用いた回転伝達機構を介して取付けている。また、展開角速度を計測するためのポテンショメータを回転軸の終端部分に取付けてある。さらに、モータを用いた外部駆動機構がトルク変換器を介して接続されており、この外部からの回転駆動によって展開機構の静的な駆動トルクの角度プロファイルが測定できる。動的挙動を測定する際にはクラッチ装置を用い、トルク変換器と慣性質量ダミーへの回転伝達機構の歯車との間を切断する。c. 大型展開構造(large deployable structure)の地上展開試験構造寸法が10mを超えるような大型の展開構造の場合、機構の部品点数も膨大になり、前述のような部品個々の特性を積み上げて展開信頼性を補償する方法は現実的とはいえない。また、大型構造を地上において試験しても、地上試験環境の影響、試験装置の影響が支配的になり、試験結果から機構特性を推定することは容易なことではない。そのような場合、ETS-VIII搭載用大型展開アンテナ(large deployable antenna)が採用したモジュール構造(modu-lar structure)による展開特性試験が有効な方法である。これは、展開構造を精度よく評価できる構造寸法まで分割し、詳細な評価試験を実施して、その基本モジュールを組み上げていく過程で基本構造と全体構造との差異(モジュール結合により拘束され、発生する内力など)を把握し、解析モデルを高精度化して軌道上の展開特性を予測する方法である。図8・22はETS-VIII搭載用大型展開アンテナのモジュール数と試験評価精度との関係を推定したものである。図8・20で困難性指標値は試験の困難性の指標を示す、重力と展開力の比であり、評価精度は試験の推定精度である。モジュール数の増加により試験精度が極度に低下していることがわかる。この精度劣化の大きな原因は、展開中に変化していく質量特性の同定精度図8・21つり下げ追従試験装置の重力補償誤差図8・22
