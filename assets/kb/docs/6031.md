# 8・3・2 形状精度計測方法

> **章**: 第8章 機構
> **節**: 8・3 試験評価法

形状精度(surface accuracy)測定には、表面形状測定とアライメント精度測定がある。近年の宇宙機の構造は大型で、かつ柔軟な構造になっているため、これらの測定を行う場合には、従来の剛体の測定方法に加え、重力による変形を考慮した測定方法の工夫が必要になる。a. 表面形状の測定望遠鏡やアンテナの反射鏡面などでは、表面形状が性能を支配するため、高精度な表面形状の測定が必要である。表面が比較的硬い構造の場合には接触式のプローブをテーブル上で走査して、鏡面の凹凸を測定する方法がよくとられる。表面が金属メッシュなどの軟らかい面の場合や大型の構造の場合には、写真測量を利用した三次元形状測定装置(photometrysystem)やビデオカメラを利用した三次元形状測定装置(video-metry system)を使用して鏡面形状を測定する。このとき、重力の影響で大きく自重変形するような柔軟で大型の構造の場合には、ミューゼス(MUSES)B に搭載された高精度メッシュアンテナのように、鏡面精度をいくつかの境界条件で測定、数学モデルを綿密にアップデートして無重力の形状を推定する方法や、技術試験衛星VIII型(ETS-VIII)に搭載された大型アンテナのように構造をモジュール分割し、それぞれのモジュールを重力方向、重力と反対方向それぞれの姿勢で測定し、その計測結果から無重力の形状を算出し、かつ各モジュールの変形量を数学モデルに反映して全体構造の形状精度を推定するという方法がとられる。b. アライメント精度(alignment accuracy)測定アライメントの測定は角度の高精度な測定が基本である。測定対象を高精度に位置決めするロータリテーブル、基準面となるコーナキューブ、レベル計、オートコメリメータやセオドライドといった高精度角度測定装置を利用して行う。これらの機器を用いた測定結果に基づき、シム(調整分解能に対応した基準厚みをもつ金属片)を利用してアライメント調整を行う。アライメント測定、調整の場合も構造物が大型で柔軟になった場合、重力の影響を大きく受けることになる。この場合、構造物を高い剛性のジグでつり下げ、変形を最小にすると同時に、最終的なアライメント精度は構造解析モデルにより推定する。
