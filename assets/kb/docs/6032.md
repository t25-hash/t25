# 8・4・1 ETS- 搭載展開アンテナ跳ね上げ・展開機構

> **章**: 第8章 機構
> **節**: 8・4 機構設計の実例

図8・26に技術試験衛星VI型(ETS-VI「きく」6号、engineer-ing test satelliteVI)に搭載したアンテナモジュールを示す。ETS-VIには20GHz-S 帯用開口直径3.5mの展開・跳ね上げ式アンテナ主反射鏡面(20GHz-S 帯主反射鏡面)と30GHz-C帯用開口直径2.5mの跳ね上げ式アンテナ主反射鏡面(30GHz-C帯主反射鏡面)が搭載された。ロケットによる打上げ時には20GHz-S帯主反射鏡面は全体を三つの部分に分割して両翼を折り畳み、かつ鏡面全体を根元で折り曲げてタワーモジュールに固定し、30GHz-C帯主反射鏡面は鏡面の根元のみで折り曲げて固定して収納する。これらのアンテナ鏡面は、軌道上においてアンテナタワーとの固定部が解放されると、ヒンジ部に取付けたばねの力によって展開を開始し、所定の位置においてラッチ固定される。図8・27,8・28にはそれぞれの主反射鏡の構造および外形寸法を示す。反射鏡面はアルミニウムハニカムコアをCFRP(炭素繊維強化プラスチック)のスキンで挟んだ薄いサンドイッチ構造になっており、パラボラ面を形成している。この薄い反射鏡面構造に剛性を与えるため、円柱形状を近似したトラス構造体で多数のスタンドオフを介して支持して11―115第II部基礎編いる。図8・29に20GHz-S 帯主反射鏡面の跳ね上げ機構を示す。機構は二つで一組になっており、1ハウジング、2回転軸、3転がり軸受、4ねじりコイルばね、5コイルばね案内筒、6ラッチアーム、7ラッチ爪から構成されている。展開力はねじりコイルばねによって与え、展開の終了時にはラッチ爪によって鏡面支持トラス構造体を構成するトラス部材の一部をつかんでアンテナ鏡面を固定する。図8・30には30GHz-C帯主反射鏡面の跳ね上げ機構を示す。二つで一組となって取付けられ、1ハウジング、2回転アーム、3滑り図8・26 技術試験衛星VI型(ETS-VI「きく」6号)に搭載したアンテナモジュール図8・27 20GHz-S 帯用開口直径3.5m展開・跳ね上げ式アンテナ主反射鏡面図8・28 30GHz-C帯用開口直径2.5m跳ね上げ式アンテナ主反射鏡面11―116宇宙機器・システム図8・2920GHz-S 帯主反射鏡面跳ね上げ機構図8・30 30GHz-C帯主反射鏡面跳ね上げ機構表8・1 Y 翼鏡面展開機構の常温大気中の機構単体特性抵抗トルク展開トルク収納展開収納展開平均値N・m2.4751.4410.1860.100+Y 翼鏡面標準偏差N・m0.1440.0920.0550.058平均値N・m2.7331.6820.1280.107-Y 翼鏡面標準偏差N・m0.0980.0890.0710.056表8・2 Y 翼鏡面展開機構鏡面展開試験より評価した展開特性駆動トルク収納展開平均値N・m2.3841.284+Y 翼鏡面標準偏差N・m0.0970.234平均値N・m2.8061.276-Y 翼鏡面標準偏差N・m0.0720.364軸受、4ねじりコイルばね、5コイルばね案内筒、6ラッチピン、7スライダによって構成されている。両者を比較して異なる点は軸受の形式およびラッチの形式である。特にラッチ形式に関しては30GHz-C帯主反射鏡面の場合、ラッチピンが展開中にピンスライダ上を滑り、展開終了時にピン先が溝に落ちて回転アームをラッチ固定する方式であるため、機構展開トルクに占める摩擦抵抗トルクの割合が比較的大きい。表8・1には機構単体試験によって得られた20GHz-S 帯主反射鏡面Y 翼展開機構の特性を示す。また、表8・21サーフェスケーブル2金属メッシュ3タイケーブル4バックケーブル5スタンドオフ6展開フレーム図8・31 モジュール型展開メッシュアンテナ鏡面基本モジュールの構成にはアンテナモジュール展開試験によって得られた20GHz-S 帯主反射鏡面Y 翼展開機構の特性を示す。機構が構造に組込まれ、サブシステムに組込まれた後でも特性をばらつきの範囲で維持していることがわかる。
