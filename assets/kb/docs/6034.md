# 8・4・3 ADEOS- 太陽電池パドル

> **章**: 第8章 機構
> **節**: 8・4 機構設計の実例

環境観測技術衛星ADEOS-IIに搭載されている太陽電池パドルの収納状態と展開状態を図8・37,8・38に示す。このパドルでは通常のリジッドパネルとは異なり、収納効率の高いフレキシブル太陽電池パネル(flexible solar array panel)を採用している。ポリイミドフィルムをベースにしたパネルにセルを張った厚さ約0.7mmの薄膜構造である。幅が約2.6mで、長さが約0.46mのパネル50枚をヒンジで連結したもので、展開時の長さは約23mになる。50枚のパネルは収納時はカーテンのように折り重ねられ、衛星構体のコーナ部に図8・37のようにトグルリンク保持解放機構とプレッシャボードで予圧保持しており、ケーブルリンクの切断によって保持解放を行う。展開シーケンスは以下のようになっている。a. ブーム展開50枚のパネルを保持したまま衛星構体から所定の離隔を行うため、二つに折り畳んだアームを展開同期機構によってばね展開する。展開速度の制御にはダンパを使用している。b. パネル展開(マスト伸展)パネルの伸展には伸展マストと呼ばれる一次元展開機構が用いられる。これは図8・38のような三角断面トラスの長手部材すべてを屈曲して折り畳んだものを1段ずつ引き伸ばしながら伸展し、オーバセンタ状態をダイアゴナルロッドのテンションによるプリロード保持するものである。伸展マストは展開完了時には横断面がパネルを平らに伸ばすために所定の張力を付与する機能ももっている。太陽電池パドルは地球周回に伴って温度がおよそ±100°Cと大きく変動するため、長いパネルとマストの間に大きな熱変形差を生じる。このため、パネルに付与している張力が変動しないように一定に保つテンションコントロール機構(TCM : tension controlmechanism)を装備している。TCM は一定の張力を得るために渦図8・37 ADEOS-II太陽電池パドル(収納状態)図8・40 PALSAR のオフナディア展開機構図8・38 ADEOS-II太陽電池パドル(展開状態)巻ばねとカム機構を採用してケーブルでパネルにテンションを付与している。パネル端が直線的に動くためのリニアガイドにはパンタグラフ機構を採用している。
