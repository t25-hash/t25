# 8・5・1 柔軟展開構造(flexible deployable structure)の機構

> **章**: 第8章 機構
> **節**: 8・5 機構に関する最近の話題

設計機構設計は基本的に剛体の機構運動と線形の構造微小変形を前提に設計できるものが多いが、近年の大型展開宇宙構造の場合は、地上で自立できないくらい柔軟な構造になっている。このような展開構造は剛体の機構に微小変形する弾性構造が結合しているという考え方で設計することができず、展開構造そのものが機構であり、機構運動する弾性構造といえる。また、構成要素もはり要素のみならず、膜要素やケーブル要素のような可撓性の要素も含まれる場合がある。可撓性の要素は初期形状をもたず、形状や反力を求めるには平衡形状を解析する必要がある。このような展開構造の機構設計を行う場合には、可撓性の要素も含んだ構造解析モデルで機構運動解析を実施して駆動力、抵抗力の計算を行う必要がある。図8・42は図8・40の基本モジュールの展開解析の例である。ある位置まで収納した時点でケーブルが緩み、展開抵抗が不連続に変化している様子も解析できている。図8・42中、消えているケーブルが緩んだケーブルを示す。
