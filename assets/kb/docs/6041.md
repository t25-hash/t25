# 9・1・4 まとめ

> **章**: 第9章 国際宇宙ステーション「きぼう」(JEM)のシステム
> **節**: 9・1 環境制御系の動向

環境制御系(ECLSS)を分類する場合、宇宙航空研究開発機構(JAXA)でも検討されていたが、図9・9のような分類が目安になる。つまり環境制御系の機能は、その対象となるミッション規模によって必要なシステムが決まってくるものと考えられる。そのとき決め手となるのは効率と質量、サイズ、リソース、開発コストであり、これらを比較検討することで最適なシステムを構築していく必要がある。きぼう」ばく露部は、船外実験プラットホームとしてさまざまな実験用ペイロードを搭載し、軌道上で実験運用を行う。その実験機器および搭載機器を許容温度範囲内に維持するため、熱制御が必要となる。ばく露部の熱制御系は、熱制御材として断熱材、塗料、ヒータ等を用いて温度を制御する受動熱制御系と、冷媒を用いて高発熱機器から排熱させる能動熱制御系から構成される。
