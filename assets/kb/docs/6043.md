# 9・2・2 ばく露部の能動熱制御系

> **章**: 第9章 国際宇宙ステーション「きぼう」(JEM)のシステム
> **節**: 9・2 きぼう」ばく露部の熱制御系

宇宙ステーションの能動熱制御系(ATCS : active thermal con-trol system)は、冷媒を用いて各機器の温度制御を行うサブシステムを指し、衛星で使われているヒータ等を用いての熱制御系とは定義が異なるので、注意が必要である。図9・11 ISS 統合熱数学モデルの一例a. 系統構成能動熱制御系は、9・3・3項に記述されるばく露部側装置交換機構(EFU)を介してばく露部に結合される実験ペイロードおよびばく露部システム機器からの排熱を冷媒により吸収し、与圧部側に置かれたばく露部熱交換器まで移送して水系流体ループへ排熱を行うシステムである。この系統構成を図9・12に示す。系統は、冷媒を循環制御するポンプ、系統全体の圧力を維持・調整するアキュムレータ、システム機器からの排熱吸収を行うコールドプレート、および配管・弁・センサ類、クイックディスコネクト(QD : quick disconnect)等から構成される。図9・12に示すとおりEFU を介してのペイロード接続は並列のため、直列型の系統設計に見られる、後流ほど供給冷媒の温度が高くなる欠点を排除し、ペイロードの取付位置によるサービス内容の差をほとんど生じない利点がある。ペイロードに提供される能動熱制御系サービス条件は、表9・10に示すとおりである。b. 運転制御方式ペイロードへの冷媒の供給遮断は、各EFUの出入り口に設置した遮断弁の開閉制御により行われる。一方、供給流量に関する制御は、流体ループのほぼ中央に設置した差圧センサにより計測した供給側・戻り側冷媒間の差圧が一定になるようにポンプの回転数を操作し、各ペイロードにかかる出入り口差圧(通常運用時の差圧制御目標値はノミナルで67.7kPa)を一定に維持することで行われる。各ペイロードは、自身にかかる差圧が一定であることを利用し、ペイロード内部の圧力損失を調整することにより、必要な冷媒流量の供給を受けることができる設計となっている。本流量制御方式は、各ペイロードへの流量制御をペイロード内の圧力損失の調整にて決定するため、EFU ごとの流量調整弁、流量センサ等の設置が不要となり、コスト、重量等の面で優れたシステム設計であるといえる。c. 有人宇宙機システムとして特徴的な設計有人システム特有の安全性等に関わる特徴的な設計について述べる。. 圧力システムの安全性設計スペースシャトルでの打上げや宇宙ステーションの軌道上運用においては、圧力上昇による機器の破壊、人命の喪失を防止することが設計上重要である。要求は、いかなる二つの故障の組合せによっても、流体圧力を機器の最大設計圧力(ばく露部の場合、1.18MPa)以下に維持することである。ばく露部の流体系の設計においては、系統全体について圧力を調整する2台冗長構成のアキュムレータに対し、冷媒体積の膨張・収縮を吸収する内部スライド機構の潤滑やクリアランスの設計、認定試験結果等のデータから、この部分の故障発生確率が十分低いことを実証し、逃がし弁等の第三の圧力調整デバイスなしに、要求を満たす設計としている。また、系統図に示すEFU 分岐ライン部の遮断弁から先(ペイロード側)で発生する可能性のある冷媒の封じ込み部分の圧力逃し設計について、遮断弁内にばねとポペットという単純な構成の逃し機構を設けて遮断弁閉による冷媒の封じ込めの発生を回避した設計を行い、ばく露部内で可能性のある数十表9・10実験ペイロードに提供される能動熱制御系サービス条件項目サービス条件備考最大排熱量3kW(EFU #1,2は6kW)ただし、ペイロード合計で最大10kW に制限最大冷媒流量465kg/h(ノミナル)(EFU #1,2は930kg/h)冷媒はパーフルオロカーボン(製品名:FC-72)単相液体を使用供給冷媒温度16~24°C(ノミナル)ばく露部への戻り冷媒温度は16~50°Cに制限箇所への圧力調整デバイスを追加せずに要求を満足させている。. 流体QD の信頼性設計QD は、有人システムとしての特徴である機器の交換、保全等のために必須な部品である。なかでもばく露部の流体QD については、ロボットやEVA による保全を前提に、冷媒リークに対する信頼性確保、内部しゅう動部品の固着回避、過大結合力回避等、システム全体設計とのバランスを考慮した設計を行っている。. 使用冷媒ばく露系の冷媒としては、NASA で用いているアンモニアやスペースラブで実績のあるフロン(CFC-114)等、選択範囲が広い。ばく露部では当初CFC-114で計画されたが、オゾン層破壊防止に対する世界的な潮流が考慮され、流体QDシール材等の各種材料との適合性やポンプ性能への影響等の試験確認も経て、オゾン層破壊係数ODPが零である現在のパーフルオロカーボン(perfluorocarbon)に変更されている。冷媒選定においては、基本的な熱流動特性、安定性等とともに、このような環境への影響にも配慮された設計となっている。
