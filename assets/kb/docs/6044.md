# 9・3・1 エアロック

> **章**: 第9章 国際宇宙ステーション「きぼう」(JEM)のシステム
> **節**: 9・3 きぼう」の機構系

エアロック(airlock)は、「きぼう」の船内実験室左舷端部に取付けられたシリンダ状構造物で、船内の与圧環境を保持したまま船内と船外間の物資(ペイロードと称する)の通過を行うサブシステムである。エアロックは、その内部を船外環境に減圧、あるいは船内環境まで加圧する機能をもち、それぞれの圧力状況に応じて、船11―125第II部基礎編図9・12ばく露部能動制御系の系統構成図外側の外側ハッチ、船内側の内側ハッチを開閉することでペイロードの通過を可能としている。a. 基本構成と概要図9・13にエアロック概要、図9・14に外観写真を示す。エアロックはシリンダ構体、外側ハッチ(outerhatch)、内側ハッチ(inner hatch)、移動テーブル(slidetable)、加減圧機器、監視駆動装置から構成される。シリンダ構体は円筒状のアルミニウム鍛造材を、ワッフルグリッド状にNC(numerical control、数値制御)加工した構造物を溶接することにより製造している。108本のファスナにて船内実験室端部を貫通するように取付けられ、接合部は気密のために溶接シールが施されている。エアロックのすべての部品はシリンダ構体にぎ装される。シリンダ構体の船外側にはアクチュエータにて開閉する機構をもつ外側ハッチ、船内側には手動にて開閉する内側ハッチが取付けられており、各ハッチにより密閉されたシリンダ構体内部を減圧・加圧する加減圧機器として真空ポンプ、バルブ、および配管類をもつ。さらに、船内側と船外側に伸展する移動テーブル、移動テーブル上にペイロードを固定する着脱機構(capture mechanism)をもつ。エアロックの制御は監視駆動装置によって行われ、運用時は監視駆動装置とインタフェースをもつ船内実験室のラップトップ端末(SLT : station laptop terminal)、エアロック上部に設置される操作表示盤とバルブボックスにより操作する。エアロックを通して通過させることができるペイロードの最大寸法を図9・15に示す。b. 手順4(表9・1110~)外側ハッチを開け、移動テーブルとペイロードを船外に伸展させる。把持終了後に着脱機構の解除を行い、移動テーブルからペイロードを取り外す。充てん完了後にシリンダ構体内圧力変化をモニタすることにより、リークチェック(船外~シリンダ構体内)を行う。その後、エアロックへの電源供給を停止する。c. 機構部の概要エアロックがもつ移動テーブル機構、着脱機構、外側ハッチ開閉機構について以下に概説する。. 移動テーブル(slide table)機構図9・16に移動テーブル機構駆動方法を示す。移動テーブル機構は、上部・中間・固定の三層のテーブルから構成される。シリンダ構体に設置されている固定テーブルにはアクチュエータが設置されており、ケーブル、ケーブルドラムおよびプーリを用いて中間テーブルを船外側、および船内側に伸展させる。また、上部テーブルは中間テーブルに設置されたプーリを介して固定テーブルとケーブルにて接続されている。これにより、前述したアクチュエータによる中間テーブル伸展動作が行われると、上部テーブルは中間テーブルの上面を同方向にさらに伸展する仕組みとなっている。. 着脱機構図9・17に着脱機構概要を示す。着脱機構は移動テーブルの上部テーブルに取付けられており、上部テーブル上をスライドする着脱機構可動側ウェッジ(凸部)と上部テーブル端に設置されている固定側ウェッジとの間にペイロードを挟み込んで把持する機構である。着脱機構可動側は、アクチュエータとピニオンギヤ、および上部テーブルのラックギヤによってスライド動作を行う。把持完了のモニタは可動側ウェッジ部のリミットスイッチにて検知され、アクチュエータを自動停止する。. 外側ハッチ開閉機構図9・18に外側ハッチ開閉機構を示す。外側ハッチの駆動は構体シリンダ内に設置されたアクチュエータとボールねじにて行い、開閉時にシリンダ構体内に入れられるペイロードのエンベローブを可能な限り大きくできるように、外側ハッチ移動レールによって開閉軌道を設定している。ハッチ閉時には、エアロック船内側から4本のラッチにてハッチの上下左右4箇所をロックする。なお、軌道上ではつねに船外側が真空環境であることから、内側・外側ハッチともに船内側からシリンダ構体に押し付けるように閉まる構造として気密の信頼性を向上させている。このため、外側ハッチはシリンダ構体内側方向に開く構造となっている。d. 潤滑剤(lubricant)の選定エアロックの機構部品は船内環境(湿度25~70%RH)、および船外環境(真空、放射線、原子図9・16 移動テーブル機構駆動方法11―127第II部基礎編状酸素)の両環境にさらされるのが特徴であり、これらの使用環境、使用方法、母材などから最適な潤滑方法を選定し、処理を施す必要がある。潤滑要素試験を実施したデータをもとに潤滑剤の選定を行った。. 移動テーブル機構ケーブルケーブルプーリ側に自己潤滑性を有する複合材を使用し、ケーブル側には耐放射線性の良好なPFPE(パーフルオロポリエーテル、perfluoropolyether)オイルを塗布する方式を採用した。. 着脱機構ラックアンドピニオン歯車船外環境にさらされる移動テーブル上面に設置される着脱機構ラックアンドピニオン歯車の潤滑剤には、耐放射線性、および耐原子状酸素性が良好なMoS焼成膜を採用した。. 外ハッチ開閉機構ボールねじ、アクチュエータ内歯車シリンダ構体内に設置されるボールねじ、アクチュエータケース内に密封される歯車については、放射線、原子状酸素の影響は少ないため、潤滑剤として低蒸発特性を示し、耐湿度環境性も良好なPFPEグリースを採用した。e. 安全性設計きぼう」の船内空気が船外に漏洩することによる急激な減圧は、搭乗員の致命的な傷害となるため、ハッチ部の気密性低下を招くすべての事象を考慮し、対策する必要がある。エアロック気密性に対する安全性設計は次のとおりとなっている。. 外側ハッチ・内側ハッチシール部の安全性設計(1) Oリングの設計・製造管理:使用するOリングは、気密性能低下に対して材料の選択、設計、製造工程を管理し、故障の発生を可能な限り低減している。(2) 二重シール:外側ハッチ・内側ハッチシールともにOリングを2本使用した二重シールとしている。また、エアロック内圧力変化の計測と、2本のOリング間の圧力計測が可能なリークチェックポートを設けたことにより、各Oリングの気密性確認を行えるようにしている。(3) シールカバー:ハッチを開けた状態で作業するときに、シールおよびシール当たり面を保護するカバーを用意している。(4) シール交換:各シールともに交換できる設計としている。. 故障に対する安全設計なんらかの故障により片側のハッチを閉められないという事象は、ただちに気密を損ってしまうものではないが、速やかに復旧させることが望ましい。このような事象を想定して採用した対処方法、および設計を次に示す。(1) 電装品の故障:外側ハッチ、移動テーブルは、エアロック船内側から手動で動かすことができるハンドルを装備している(マニュアルバックアップ機能)。これにより電装品の故障時はもとより、エアロックに供給される電源が断となった場合にも、クルーの手動操作で移動テーブルを引き込み、外側ハッチ閉の動作を行うことができる。(2) 移動テーブル機構部の故障:移動テーブル機構部が船外に図9・18 外側ハッチ開閉機構概要図9・17 着脱機構概要11―128宇宙機器・システム伸展中になんらかの故障を起こし、上記マニュアルバックアップ機能を使用しても動作しない状況では、外側ハッチの開閉を妨げる結果となる。よって、船外に伸展した移動テーブルを宇宙飛行士の船外活動により取外しできる設計とした。取り外した移動テーブルを船外側に撤去することにより、外側ハッチを閉めることを可能としている。
