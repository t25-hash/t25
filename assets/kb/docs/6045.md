9・3・3ばく露部装置交換機構
a.概要 船外実験プラットホーム（JEMばく露部）上での実
験装置（ペイロード）の交換は，ロボットアーム（JEMRMS:
Japanese Experiment Module Remote Manipulator System）と
装置交換機構（EEU:equipment exchange unit）との協調作業に
より行われる（図9･24）．装置交換機構は，ロボットアームにより
把持，搬送され，所定の場所に位置決めされた実験装置をプラット
ホーム上に機械的に結合するとともに，実験に必要な電力，信号，
熱制御用冷媒を供給するためのリソースインタフェースを提供す
る．
ロード側装置交換機構（PIU:payload interface unit）と，プラ
ットホーム上に搭載されているばく露部側装置交換機構（EFU:
exposed facility unit）の二つの機構から構成されている（図9･
25）．
EFUはロボットアームにより位置決めされたPIUを捕捉し，ロ
ボットアームの抵抗荷重に打ち勝ってその位置決め誤差を吸収し，
PIUと結合（または分離）することができる．そのための3本の
固定アーム，およびそれらを同期させて駆動するためのリンク機
構，ボールねじ，歯車，モータ，固定アーム位置検出センサなどを
備えている．
両機構（EFU，PIU）の結合面には，実験装置に電力，信号，
熱制御用冷媒を供給するための電気・光コネクタ，および流体コネ
クタが搭載されており，これらは装置交換機構の機械的な結合，分
離と同時に一括して着脱される．
軌道上での10年以上の寿命を保証するため，両機構は，これら
のコネクタ類を非結合時に放射線，デブリ，原子状酸素といった宇
宙環境から保護するためのカバー機構を備えている．この保護カバ
ーは，両機構の着脱に伴って自動的に開閉する．
高い信頼性を実現するため，センサ類はすべて待機冗長系として
いる．さらに，万一，モータが故障した場合を考慮し，宇宙飛行士
が船外活動（EVA）によりはん用ツールを用いて分離できるよう
なツール受け口を備えている．
