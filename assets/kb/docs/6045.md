# 9・3・2 ばく露部結合機構

> **章**: 第9章 国際宇宙ステーション「きぼう」(JEM)のシステム
> **節**: 9・3 きぼう」の機構系

ばく露部結合機構(EFBM : exposed facility berthing mecha-nism)は、「きぼう」船内実験室と船外実験プラットホームを軌道上で結合・分離する機構サブシステムである。結合作業は、ISS のマニピュレータ(以下、SSRMS : space station remote manipu-lator system)にて船外実験プラットホームをハンドリングした状態で実施し、初期結合、構造結合、リソースライン接続の三つの工程からなる。図9・19にばく露部結合機構の運用形態を示す。a. 基本構成と概要ばく露部結合機構は船内実験室左舷端部に取付けられるアクティブ機構(EFBM active half)と、船外実験プラットホームに取付けられるパッシブ機構(EFBM passivehalf)、および船内実験室内に設置される監視駆動装置、操作表示盤から構成される。図9・20にばく露部結合機構概要を示す。アクティブ機構は、4組のフックにてパッシブ機構を捕獲して引き込むための初期ラッチ機構(capture latch mechanism)、軌道上荷重に耐える強度・剛性で構造結合する4組の構造ラッチ機構(structural latch mechanism)、結合した船外実験プラットホームに必要な電力、信号(電気、光)、熱制御流体のリソースラインを接続するアンビリカル機構(umbilical mechanism,2組、最大20系統)の駆動側をもち、パッシブ機構は、これらと対応する受動側の機構をもつ。またアクティブ機構・パッシブ機構のおのおのに図9・21 初期ラッチ機構の動作概要図9・19ばく露部結合機構の運用形態は、初期ラッチによる結合動作時に相対位置・姿勢のミスアライメント修正を行わせるための8本のガイド、および2組のアライメントピンとソケットをもつ。船内実験室内の操作表示盤を搭乗員が操作することにより、監視駆動装置が機構の駆動・制御を行う。b. 運用手順ばく露部結合機構の運用手順は次のとおりである。(1) SSRMS による船外実験プラットホームの把持、位置決め。(2) パッシブ機構がアクティブ機構の把持可能範囲にあることを確認。(3) 初期ラッチ機構にてパッシブ機構を捕獲し、初期結合。(4) 構造ラッチ機構にて構造結合を行う。(5) アンビリカル機構にて、リソースラインの結合。(6) SSRMS による船外実験プラットホームの把持を解除。c. 機構概要。 初期ラッチ機構図9・21に初期ラッチ機構と作動シーケンスを示す。初期ラッチ機構はアクティブ機構側に設置されるアクチュエータ、捕獲フック、ローラ、駆動リンク等から構成され、結合動作時には、収納状態→展開→捕獲完了→引込み→初期結合と動作する。初期ラッチ機構による捕獲・初期結合が可能となる条件は、アクティブ機構とパッシブ機構の相対姿勢(ロールおよびウォブル)が±1.5°以内であり、機構中心どうしの相対位置が、接近方向で152mm以内、横方向で±76mm以内である。4組の初期ラッチ機構は同期して作動し、位置誤差が大きい場合はガイドどうし、もしくはガイドと結合面周囲を接触させながら引き込み、最終的には双方の結合面に設けた2組のアライメントピンとソケットがかん合することによって軸ずれ方向±1.1mm以内、ロール角±0.18°以内で初期結合される。. アンビリカル機構概要(単位:mm)ルト・ナット機構であり、4組設置されている。結合作動時は、結合面の締付け力バランスをよくするために対角線上の2組ずつ2回の作動に分けて行われる。構造ラッチ機構の結合部詳細を図9・22に示す。結合動作時はアクティブ側のラッチボルトが伸展し、ラッチボルトを駆動するDCモータの回転数と電流値の二つの条件がそろったところで回転を停止させることにより、適正な締付け力を得る。また、ラッチボルトとラッチナット端部のテーパガイド化、ラッチナットのフローティング化等により、初期ラッチ機構による初期結合時のミスアライメントがある場合にも結合可能である。. アンビリカル機構図9・23にアンビリカル機構を示す。アクティブ機構側のアンビリカル機構は、10個のコネクタを固定している移動台を駆動スクリュー・駆動ナットにより伸展させ、パッシブ機構側のコネクタに押し付けてかん合させる。パッシブ機構側のコネクタはフローティングマウントとすることによりミスアライメント(軌道上熱変形、構造ラッチ結合相対誤差)を修正し、各コネクタの同時かん合を可能にしている。
