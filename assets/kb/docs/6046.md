# 9・3・3 ばく露部装置交換機構

> **章**: 第9章 国際宇宙ステーション「きぼう」(JEM)のシステム
> **節**: 9・3 쓕きぼう」の機構系

a. 概要船外実験プラットホーム(JEM ばく露部)上での実験装置(ペイロード)の交換は、ロボットアーム(JEMRMS :Japanese Experiment Module Remote Manipulator System)と装置交換機構(EEU : equipment exchange unit)との協調作業により行われる(図9・24)。装置交換機構は、ロボットアームにより把持、搬送され、所定の場所に位置決めされた実験装置をプラットホーム上に機械的に結合するとともに、実験に必要な電力、信号、熱制御用冷媒を供給するためのリソースインタフェースを提供する。b. 特徴装置交換機構は、実験装置側に組み付けられるペイ11―129第II部基礎編図9・24 船外実験プラットホームにおける実験装置の交換図9・25 装置交換機構(EEU,PIU)の概要ロード側装置交換機構(PIU : payload interface unit)と、プラットホーム上に搭載されているばく露部側装置交換機構(EFU :exposed facility unit)の二つの機構から構成されている(図9・25)。EFU はロボットアームにより位置決めされたPIU を捕捉し、ロボットアームの抵抗荷重に打ち勝ってその位置決め誤差を吸収し、PIU と結合(または分離)することができる。そのための3本の固定アーム、およびそれらを同期させて駆動するためのリンク機構、ボールねじ、歯車、モータ、固定アーム位置検出センサなどを備えている。両機構(EFU,PIU)の結合面には、実験装置に電力、信号、熱制御用冷媒を供給するための電気・光コネクタ、および流体コネクタが搭載されており、これらは装置交換機構の機械的な結合、分離と同時に一括して着脱される。軌道上での10年以上の寿命を保証するため、両機構は、これらのコネクタ類を非結合時に放射線、デブリ、原子状酸素といった宇宙環境から保護するためのカバー機構を備えている。この保護カバーは、両機構の着脱に伴って自動的に開閉する。高い信頼性を実現するため、センサ類はすべて待機冗長系としている。さらに、万一、モータが故障した場合を考慮し、宇宙飛行士が船外活動(EVA)によりはん用ツールを用いて分離できるようなツール受け口を備えている。文献(1) Hienerwandel,K.and Kring,G.,ESA SP-288(1988)、45.(2) 小鑓幸雄・和田勝、日本航空宇宙学会誌、49 (2001)、273.→ Next Chapter
