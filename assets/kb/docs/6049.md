# 10・1・3 宇宙ステーション上のマニピュレータ

> **章**: 第10章 ロボティクス
> **節**: 10・1 宇宙ロボットの開発状況と今後の展望

国際宇宙ステーション(ISS : International Space Station)上には宇宙ステーションの建設、運用のために複数のロボットアームが取付けられる。a. 宇宙ステーション用マニピュレータ(SSRMS : space sta-tion remote manipulator system、通称カナダアーム2)年4月に国際宇宙ステーションに取付けられた最初のロボットアームで、スペースシャトルで運ばれてくる宇宙ステーション構成モジュール等を国際宇宙ステーション本体に取付ける作業に使用されている。7自由度を有し、全長は約17mである。また、カナダは宇宙ステーション用マニピュレータの先端に取付ける双腕のロボット(SPDM : special purpose dexterous manipulator)も開発している。本ロボットは国際宇宙ステーション上の軌道上交換ユニット(ORU)の交換等の作業に使用される。図10・6にSSRMS/SPDMの外観を示す。b. 日本実験モジュール(JEM)用マニピュレータを参照のこと。c. ヨーロッパロボットアーム(ERA : European robotic arm)7自由度、全長約11.3mのロボットアームで、国際宇宙ステーションのロシア担当部分(科学電力プラットホーム)に取付けられる。本ロボットアームは当初、ヨーロッパ宇宙機関(ESA : Euro-pean Space Agency)が開発していた宇宙往還機(HERMES)搭載用に研究されていたものを転用したものである。開発はオランダの企業が担当している。d. その他の国国際宇宙ステーション上には上記のロボットのほかにも実験用のロボットがいくつか計画・構想されている。ドイツはロシアと共同で小型2自由度のロボットアームの実験(RO-図10・4マニピュレータ飛行実証実験(MFD)の様子図10・5 ETS-VII搭載ロボット実験機器(図中の団体名は実験当時のもの)11―131KUISS)国際宇宙ステーションのロシアモジュールの外部で実施した(2005年12月~2006年3月)。イタリアもばく露パレット上の実験装置の操作・交換等を行うための小型のロボットアームの開発を予定している。
