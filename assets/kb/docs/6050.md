# 10・1・4 月惑星探査ローバ

> **章**: 第10章 ロボティクス
> **節**: 10・2 節

代表的な月惑星探査ローバとしては、旧ソ連が1970,1973年に月面に軟着陸させた無人移動探査機ルノホート、NASA がアポロ計画で使用した月面移動車(LRV : lunar roving vehicle)、NASA JPLが1997年7月4日のアメリカ独立記念日に火星に着陸させた探査機(マーズ・パスファインダ、Mars Pathfinder)に搭載された小型ローバ(ソジャーナ、Sojourner)、2004年1月4日、同年1月25日に相次いで火星に着陸したNASA の双子の火星探査ローバ[スピリット(Spirit)、オポチュニティ(Opportu-nity)]などがある。マーズ・パスファインダは打上げ時の衛星全体の質量が894図10・6 宇宙ステーション用マニピュレータ(SSRMS)およびSPDM図10・7 火星探査ローバ「ソジャーナ」(NASA JPL/Caltech)図10・8 火星探査ローバ「スピリット」(NASA JPL/Caltech)11―132kg、衛星から分離後の着陸機が370kg、同着陸機に搭載された小型ローバ(ソジャーナ)の質量は10.6kg、走行速度は0.01m/sであった。ローバは6輪独立モータ駆動される。地球とローバとの通信には約10分かかるため、着陸機からの画像をもとに地上のオペレータがローバの動作経路を指示し、ローバはレーザ光、CCDカメラを利用して障害物を避けながら指示された経路にできるだけ近い経路を自律的に移動した。一方、2004年1月に火星に着陸したスピリット、オポチュニティは質量がおのおの約180kg と大型で、ソジャーナが120日間で100mしか走行できなかったのに対して1日に40mの走行が可能である。図10・7にソジャーナを、図10・8にスピリットを示す。
