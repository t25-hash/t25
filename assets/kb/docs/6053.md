# 10・2・1 概要

> **章**: 第10章 ロボティクス
> **節**: 10・2 JEM マニピュレータ(「きぼう」のロボットア

JEM マニピュレータ(JEMRMS)は図10・9に示すように,国際宇宙ステーション(ISS : International Space Station)に取付けられる「きぼう」(日本の実験モジュール,JEM)のロボットアームであり,おもな作業として実験ペイロードの船外実験プラットホームへの取付け・交換を実施する.また,船外実験プラットホームの保守作業として,そのORU の交換を行う.一方,カナダアームはISS の組立,保守等の支援を行うロボットアームで,ISS 全体を対象として作業を行う(10・3節参照).「きぼう」のロボットアームは,図10・10に示す実験ペイロードと「きぼう」のエレメントをハンドリングする親アーム(main arm)と,船外実験プラットホームのORU 交換等の細密作業を行う子アーム(small finearm)から構成される.宇宙飛行士は,船内実験室に設置される制御卓であるコンソール(図10・11)からアームの操作を行う.親アームは,6個の関節,エンドエフェクタ,基部,2式の視覚装置と3本のブームから構成される6自由度のマニピュレータである.各関節には関節制御エレクトロニクス(JEU : joint elec-tronics unit)が搭載されており,伸展姿勢で基部から先端までの長さは約10m,質量は約780kg である.子アームは,6個の関節,力トルクセンサと手首カメラを搭載したツールと呼ばれるエンドエフェクタ,関節を制御する子アームエレクトロニクス(SFAE: small fine arm electronics)と2本のブームから構成される6自由度のマニピュレータである.伸展姿勢で基部から先端までの長さは約2m,質量は約180kg であり,運用時には親アームの先端に把持される.コンソールの計算機としては,ロボットアーム全体を管理する管図10・9쓕きぼう」(JEM)概観図図10・10 親アームと子アーム図10・11 コンソール理計算機(MDP:management data processor),アームを制御するアーム制御計算機(ACU : arm control unit),宇宙飛行士にGUI(graphical user interface)を提供するラップトップワークステーション(RLT : RMS laptop terminal)と,ソフトウェアとデータを保存する外部記憶装置(MSD : mass storage device)が搭載されている.宇宙飛行士が操作・モニタする機器としては,マニュアルでアームを操作するときに使用するジョイスティックである並進用/回転用ハンドコントローラ(THC/RHC: translationalhand controller/rotational hand controller),カメラを操作するカメラ制御パネル(CCP: camera control panel),アームの関節ブレーキ等を操作する管理装置操作盤(RIP: remote interfacepanel)と,カメラの画像とターゲットのオーバレイを表示する2台のTV モニタ(TVM : TV monitor)が搭載されている.そのほか,親アーム,子アーム,コンソールの各機器に電源を供給する電力変換装置(PDB : power distribution box),空冷機器に冷却用アビオエアを供給するアビオニクスエアアセンブリー(AAA :avionics air assembly),火災の煙をモニタする煙センサ(SD :smoke detector)が搭載されている.また,コンソールメンテナンス時に電源を切るスイッチと煙センサのインジケータを有するメンテナンススイッチパネル(MSP: maintenance switch panel),与圧部エンドコーンに取付けられた親アームを解放する保持解放エレクトロニクス(HREL: hold and release electronics),ISS の状況を宇宙飛行士に知らせる警告警報パネル(C& WP: cautionand warning panel)と,ISS や地上との連絡時に使用する音声固定端末(ATU : audio terminal unit)が搭載され,コンソールの質量は約430kg である.
