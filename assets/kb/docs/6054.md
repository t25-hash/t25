# 10・2・2 特徴

> **章**: 第10章 ロボティクス
> **節**: 10・2 JEM マニピュレータ(「きぼう」のロボットア

有人の「きぼう」に対応する宇宙ロボットとして,地上ロボットや無人宇宙ロボットと比較し,以下の特徴を有する.(1) 微小重力環境で動作すればよいので,軽量化のためアームの剛性は低く,設計・解析上,柔軟性の考慮が必要である.また地上で試験するためには,重力の補償が必要である.(2) 打上げ時の厳しい機械環境,軌道上の厳しい熱環境に耐えなければならない.(3) 軌道上の熱真空環境で動作できる潤滑を採用しなければならない.(4) 宇宙飛行士が軌道上で操作するうえで,操作性がよいことが重要である.また,宇宙飛行士訓練の簡素化のため,シャトルマニピュレータ(SRMS)とカナダアーム2(SSRMS)との操作の共通化が要望される.(5) ロボットアームのORU 機器を宇宙飛行士の船外活動(EVA : extra-vehicular activity)と船内活動(IVA :intra-vehicular activity)で交換することにより,保守が可能である.(6) 有人プロジェクトであるため,厳しい安全設計が要求され,宇宙飛行士の生命を脅かす危険とISS の損失に至る危険は,二重の故障が発生しても制止しなければならない.
