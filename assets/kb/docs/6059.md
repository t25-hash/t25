# 10・2・7 地上試験

> **章**: 第10章 ロボティクス
> **節**: 10・2 JEM マニピュレータ(「きぼう」のロボットア

定盤上で親アーム、子アームの二次元機能性能試験が実施され、その結果を用いてアームの数学モデルの評価が行われている。定盤上では、親アームと子アームは重力の影響を補償するため、空気ベアリングで浮上させたジグで支持されている。地上試験では確認できない軌道上での三次元動作は、試験結果を反映させた数学モデルを用いた解析で検証されている。第1世代の有人宇宙ロボティクスであるSRMS(shuttle remotemanipulator system、通称カナダアーム)はカナダ製で20年以上の運用実績をもち、軌道上での衛星の放出や回収、国際宇宙ステーション(ISS)モジュールの取付け、船外活動の支援およびスペースシャトル(space shuttle)オービタ外観検査等に使用されてきている(図10・13)。ISS にはすでにカナダのSSRMS(space sta-tion remote manipulator system、通称カナダアーム2)が搭載されており、今後日本実験棟「きぼう」のJEMRMS の親アームと子アーム、カナダのSPDM、およびヨーロッパのERA 等のロボットアームが搭載される予定であるが、第1世代の有人宇宙ロボティクスであるSRMS の運用経験がこれらのISS ロボティクスの設計や運用にも生かされている。SRMS はその基部がシャトル貨物室の左舷前方に固定されており、自重は約410kg、肩部2関節、ひじ部1関節、手首部3関節の計6関節からなり、全長は15.3m、マヌーバ可能なペイロード(payload)質量は約265tである。シャトルの質量は搭載ペイロードにもよるが100t程度であるため、SRMS はISS 側の構造を把持した状態でシャトル自体をマヌーバできる能力を有する。ペイロードの把持は、アーム先端のエンドエフェクタ内にある3本の金属ワイヤをペイロード側に取付けられた把持機構(グラプルフィクスチャ、図10・14)のグラプルピンに巻き付け、引き込んで固定することにより行われる。SRMS のマヌーバ制御モードには大きく分けて、自動制御、単軸制御、およびジョイスティック(図10・2)を用いたマニュアル制御がある。マニュアル制御と自動制御モードは参照点(point ofresolution)に関する並進および回転の6自由度同時制御が可能であり、単軸モードは1関節ごとに操作者が変位量を制御するモードである。このうちマニュアル制御、単軸制御の一部と自動制御はSRMS 用の制御計算機を使用するモードであるが、制御計算機に不具合が生じ使用不能になった場合でも単軸制御の一部は操作可能である。また電源系についても冗長なシステムとなっており、主系電源が使用できない場合でも副系の電源により単軸制御の一部が可能である。通常の運用では、アームの格納状態からの始動および格納は単軸制御を使用し、ペイロードの把持や移動、宇宙ステーション側への取付け、人工衛星の捕獲、さらに船外活動する宇宙飛行士図10・13 スペースシャトルとSRMS図10・14 エンドエフェクタ部(左)およびグラプルフィクスチャ(右)をアーム先端に載せた状態でのマヌーバ等はマニュアル制御で行う。マニュアル制御時のアーム先端最大速度はペイロードを把持していない状態で並進0.6m/s、回転4.7°/sである。自動制御はその軌跡全体にわたって、つねにアームおよび把持するペイロードが周囲構造物と1.5m以上の距離を維持できる場合に使用可能である。なお地上からのアームの遠隔操作はできない。マニュアル操作用ジョイスティックは図10・2のように並進用と回転用の二つがある。SRMS 操作パネルはシャトル操縦室後部に位置し、シャトルの船尾を臨む方向および後部操縦室天井部に窓が取付けられている。また、二つのテレビモニタがSRMS 操作パネル近傍にあり、操作者は窓越しやテレビモニタからの視覚情報を確認しながらアームの操縦を行う。テレビカメラはシャトル貨物室に通常四つ以上、SRMS のひじ部とエンドエフェクタにそれぞれ一つずつ取付けられている。また、ISS 組立作業時には、ISS 側のテレビカメラ映像もシャトル操縦室内のテレビモニタで使用できる。人工衛星捕獲時のアーム操作では、エンドエフェクタのテレビカメラによる衛星側グラプルフィクスチャ上のターゲットの映像が相対位置姿勢の検知のための主要な視覚情報である。3630kg 以上のペイロードを把持した状態や、人工衛星を捕獲する際のアームのマヌーバはシャトル側の姿勢制御系を停止して行う。ほとんどのSRMS の運用は2人の操作者により行われる。主操作者は操作パネルを使用したアームの操縦等を行い、副操作者はテレビカメラの操作や操作パネルとは別の場所にあるコンピュータディスプレイ上に表示されるSRMS関連テレメトリのモニタ等を行いながら主操作者の操作支援を行う。各関節のエンコーダによるSRMS 先端部の位置姿勢情報の精度は並進約5cm、回転1°であるが、把持したペイロード上の参照点ではこの誤差はさらに大きくなり、SRMS によるISS 各モジュールの取付け時には、把持したモジュールに許容される位置姿勢誤差がSRMS 各関節エンコーダによる位置姿勢情報の精度より小さいため、SRMS とは独立した、より精度のよい位置姿勢測定システム(取付作業用カメラや画像処理による位置姿勢測定装置等)も使用されている。システムや運用手順に関する知識に加え、SRMS の操作訓練において強調される点は、視覚情報や計器類からのデータによりシステム全体の状態を常時把握し、安全、ミッション達成のための適切な操作を行う能力、ジョイスティックへのスムーズなコマンド入力11―135第II部基礎編および停止や多自由度同時コマンド入力を行えること、ロボティクス操縦座標系の理解等を含めたアーム、マヌーバに関する技量、単軸制御の技量、さらにコックピットリソースマネジメント(主・副操作者間の協調運用)等であり、航空機の操縦者に要求される資質に概念的に近いものといえる。SRMS は1981年の初飛行以来、ハードウェア、ソフトウェア的にも改修が進められてきている。SRMS のコアシステムからは独立した形で、SRMS の制御計算機からの関節角データをラップトップパーソナルコンピュータに取込むことにより、アームの周囲構造物との相対位置情報を含めたアーム全体の状態や各関節角の状態、さらにマニュアル制御時の必要なジョイスティックコマンド入力量等を操作パネル付近に設置したラップトップパーソナルコンピュータ上にグラフィックス表示するシステムも使用されており、SRMS
