# 11・1・2 飛行例および今後の動向

> **章**: 第11章 気体力学
> **節**: 11・1 高高度飛翔体

a. 衛星の姿勢・軌道変動地球低軌道にある衛星では、姿勢・軌道変動に大気の影響が現れる。逆に、大気の影響を観測する目的で、アメリカのエクスプローラ衛星が打ち上げられており、大気密度の観測データを提供している。それらによれば、大気は特に太陽活動の影響を受けており、その他の影響も考慮された大気密度モデルが提案・実用化されている。第14号科学衛星「ようこう」は、軟X 線領域での太陽の撮像を目的とした衛星で、高度1000kmに打ち上げられ、その姿勢は高精度で太陽指向された衛星である。この制御系の制御データより姿勢変動力が逆に測定可能であり、姿勢変動力の予測精度の向上に利用することが可能であった。姿勢変動力には大きく分けて三つあり、空気力、重力傾斜、太陽ふく射圧である。高度1000km付近では、空気力は弱いものの、無視できない程度の寄与があることがわかっている。b. エアロブレーキ地球を含む、大気をもつ惑星周りを回る衛星は、その軌道を変更するために、大気をかすめる際に得られる空力制動力を利用することが可能である。軌道を変更するためには通常は推進器を用いるが、空力制動力を利用すれば推進器の占める質量を節約できるため、特に質量制限の厳しい惑星探査機においては有益である。このような技術をエアロブレーキといい、原理的には以前から知られていたものの、実飛行における実証が行われていなかった技術である。この技術の実証は、第13号科学衛星「ひてん」を用いて1990年に行われている。「ひてん」は、質量約150kg の円筒形のスピン安定の衛星であり、遠地点はほぼ月軌道に達するだ円軌道で、地球近地点での高度120kmの大気を用いて、空気力による減速を行ったものである。쓕ひてん」での成功をもとに、アメリカのマゼラン衛星は金星周回軌道における空力減速に成功している。さらに、アメリカのマー図11・3 ウェーク構造11―138ズ・サーベイヤは火星周回軌道において空力制動を行っているなど、今後の惑星探査において利用されるものと思われる。c. 今後の動向エアロブレーキ技術は、今後の惑星探査ミッションにおいて極めて有望な技術であり、さまざまな惑星探査において用いることが可能である。またこの技術は、地球を周回する衛星・宇宙機にとっても、軌道変換技術として有望である。大気に軌道上から再突入する飛翔体は、アメリカのスペースシャトルやロシアのソユーズカプセルなど飛行士を地上に帰還させることばかりでなく、宇宙基地からの物資の地上への輸送など、地球近傍での宇宙活動に必須なものであり、また惑星探査では、惑星表面への探査プローブの投入や惑星サンプルの地球への持ち帰りの際のサンプルプローブなど、その用途は多岐にわたる。
