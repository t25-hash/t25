# 11・2・1 抗力・揚力

> **章**: 第11章 気体力学
> **節**: 11・2 大気再突入飛翔体(reentry vehicle)

飛翔体は,大気飛行中の抗力や揚力を利用して必要な減速をしつつ地上に達する.そのような飛行を計画する際には,全飛行領域にわたり,機体のもつ抗力・揚力係数を把握する必要があり,それらはデータパッケージとして機体ごとにそろえられる.例えば,高層飛行では希薄効果のある高マッハ数領域であり,希薄気体効果について把握する必要がある.すなわち,遷移領域についても対象とする必要があるが,そのため,自由分子流での空力係数と連続流での空力係数を連続的に接続するような経験式も用いられる.希薄気体効果については,自由分子流領域から連続流に至る全クヌッセン数領域について,空力係数を把握する必要が出てくる.高度が下がるにつれ,連続流としての高マッハ数領域になる.さらに高度が下がるにつれ減速されるので,超音速領域,遷音速領域,さらに亜音速領域の低速領域を経るため,結局のところ,全マッハ数領域での空力特性を把握する必要がある.
