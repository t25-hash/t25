# 11・2・2 極超音速流れでの空力加熱

> **章**: 第11章 気体力学
> **節**: 11・2 大気再突入飛翔体(reentry vehicle)

一般に再突入する飛翔体の速度は音速より速く、機体の前面には強い衝撃波が形成され、大気は衝撃波により圧縮加熱されている。加熱の度合いは飛翔体の速度、つまり衝撃波の強さに依存するが、正の相関があり、速度の増加とともに増加する。初期突入速度を高度120kmでの速度とすると、地球周回軌道からの再突入では約7.8km/sである。また、深宇宙からの直接再突入では、最低でも約12km/sとなる。地上に達するまでには、初期の突入速度から、速度零までの速度範囲を経験することになる。衝撃波背後で大気が加熱される度合いは飛行速度によるが、加熱の度合いに応じて大気の成分である分子・原子は、解離反応、電離反応、電子励起反応などを受ける。地球大気の場合は、主成分は窒素分子、酸素分子であり、飛行速度により図11・4に示すような反応が進行することになる。図11・4は、飛行速度に対応した衝撃波背後での平衡温度に対して生じる化学種のモル分率を示す。飛行速度7.8,12.5km/sでは、平衡温度はそれぞれ、約7000,11000K図11・4衝撃波背後での平衡状態(温度)での化学種の割合程度に相当する。このような反応は一般に有限の反応速度をもつが、それらは分子-原子間衝突が基本であるため、高度の増加に伴う大気密度の減少に比例して遅くなる傾向がある。このため、流れの時間スケールに対して反応速度が同程度となるような高高度ではそれらの反応時間の有限性を考慮する必要があり、反応は一般に非平衡な状態となる。飛翔体の設計には、これらの高温大気からの飛翔体表面への熱流入量、つまり空力加熱(aerodynamic heating)を知る必要がある。機体表面に生じる熱境界層を通して熱が流入するためである。このためには、上記のような現象を取り入れた流体モデルに基づく解析が可能であるが、簡便な推算法として、いくつかの推算モデルが知られている。特に、鈍頭物体のよどみ点での加熱量として、フェイ(Fay)とリデル(Riddel)によれば以下の式のようになる。욌=0.763욥ρ욄μ욄d욄dρ욌μ욌ρ욄μ욄-욌× 1+욄-1(11・20)ただし、욄はルイス数で、そのべき数αは平衡流では0.52、凍結流では0.63,は境界層端での単位質量当たりの解離エネルギーである。d욄dは、d욄d=2-ρ(11・21)となる。ここで、は鈍頭の曲率半径である。デトラ・ケンプ・リデル(Detra-Kemp-Riddel)によれば、実用的には各種実験データや理論式に基づく相関式として、次のようになる。욌=110 ρρ욂-욌-욌(11・22)ただし、욂=7.925km/s,ρは海面上空気密度、はエンタルピー、添字は壁面上、0は境界層外縁、300は300K 空気を示す。また、それぞれの単位は、(m)、(M Pa)、(kms)、ρ(kgm)、욌(MWm)である。高速の再突入では、上記の熱伝達以外に、高温大気が放出するふく射熱による加熱も増大してくる。タウバー(Tauber)によって各種実験、理論式の相関が提案されている。
