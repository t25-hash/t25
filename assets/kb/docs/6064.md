# 11・2・3 耐熱構造

> **章**: 第11章 気体力学
> **節**: 11・2 大気再突入飛翔体(reentry vehicle)

空力加熱から機体内部を保護するため、機体表面は耐熱構造にする必要がある。さまざまな方法があるが、実用的なもののうち代表的なのは、アブレーション(ablation)法と熱放射法である。熱放射法は、空力加熱により表面温度が高まると表面からふく射熱としてエネルギーが放出されることを利用した方法である。両者のバランスにより表面温度が定まるが、表面を構成する耐熱材料はふく射係数が大きく、また、バランスにより定まる表面温度で破壊されないことが要求される。その表面温度を内部に伝達しにくくするため、断熱材を用いた構造となる。この方法による耐熱構造を利用している代表的機体は、スペースシャトルである。アブレーション法は、吸収した熱によりアブレーション材を気化させ、その潜熱放出により冷却する方法である。また、表面の温度上昇に伴い、表面からのふく射熱放出も利用される。アブレーションによる熱の放出には、表面物質の損耗を伴うため、アブレーション材の選択は、損耗の度合いを適度にする必要がある。このため、地球低軌道からの再突入など加熱量の少ない場合のアブレーション材と、深宇宙からの再突入など高速の再突入では、おのずとアブレーション材の選択は異なる。アメリカにおいては、1960年代よりさまざまな実飛行実験を遂行するために、アブレーション材の開発が行われてきた。近年においても、軽量化や耐熱性の増強の努力が続けられており、シリカ系のSIRCA、カーボンフェノール系のPICA などが開発されている。わが国においては、ミューゼスC(MUSES-C)計画に用いられる地球大気再突入機の開発の一環として、カーボンフェノール系の材料が開発されている。図11・5 OREX 宇宙機(提供:宇宙航空研究開発機構)図11・6 HYFLEX 宇宙機(提供:宇宙航空研究開発機構)
