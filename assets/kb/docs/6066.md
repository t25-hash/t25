# 12・1・1 数値シミュレーションの概要

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・1 シミュレーション技術

a. 役割宇宙工学の分野における数値シミュレーションの役割として、以下のようなものが挙げられる。(1) 実験的には実現が難しい、あるいは全く不可能な条件下での現象を模擬する。宇宙という特殊な空間を地上において実験的に模擬するには大掛かりな装置を要し、また機能的な制約も大きいが、これを数値的に模擬することは比較的容易である。例えば、微小重力環境下や外乱環境下での宇宙機の姿勢運動解析、大型構造物の構造解析、大型宇宙機の熱入出力環境およびコンポーネント機器の温度分布などである。(2) 宇宙機全体は大規模かつ複雑なシステムであり、複数のサブシステムとそれを構成するコンポーネント機器で構成され、さまざまな機能を実現する。サブシステムあるいはコンポーネント機器ごとに得られた要素モデルを数値的に結合することで、容易に全体シミュレーションが可能である。また、任意の要素の設計変更による全システムへの影響も比較的容易に解析できる。(3) 設計パラメータ、初期条件、境界条件の相違による影響を解析する場合に、数値的にさまざまなケースを設定でき、任意の要因に対する結果の相違を明確に調べることができる。このことは、ロバスト性の解析や最適設計の有効な手段となりうる。上記(1)~(3)の特徴は、解析に要する時間およびコストの大幅な削減につながる。b. 手順宇宙機の運動を数値シミュレーションする手順は、一般の数値シミュレーションと同様に、物理モデルの決定、数学モデルの導出(定式化)、プログラムの作成・実行、計算結果の可視化・考察に分割される。以下、各項目について概説する。. 物理モデルの決定解析対象とする現象の自由度、境界条件、拘束条件、幾何学的条件を規定する物理モデル(physicalmodel)を構築する。物理モデルは実態の近似にすぎないが、適切なモデルを設定することで実現象を精度よく再現・予測することができる。自由度、境界条件、拘束条件、幾何学的条件が適切に設定され、かつ設定したモデルに関する定量的パラメータの入手性・精度が十分である必要がある。特に設計・開発を目的とする物理モデルは、コスト的要因や解析時間も考慮する必要がある。. 数学モデルの導出(定式化)物理モデルの状態を表す変数の時間変化を支配する関係式を導出する。物理モデルは物理法則に従って機械的に数学モデル(mathematical model)に変換される。数学モデルは、微分方程式、拘束条件式などで構成される。. プログラムの作成・実行数学モデルを解くための数値計算アルゴリズムを構築し、これをプログラミングして計算機コードの作成、実行を行う。. 計算結果の可視化・考察計算結果であるデータ列は、グラフやアニメーションなど人間が理解しやすい形に可視化することで、全体像や詳細の把握が容易になる。さらに、物理モデルや数学モデルが適切であったか、プログラムにバグが含まれないか、モデル化誤差や数値計算誤差の影響は許容範囲内であるかなど数値シミュレーションそのものの妥当性を評価し、必要に応じてこれまでの手順を繰返す。妥当であることが判断されれば、数値シミュレーション解析のそもそもの目的である現象の考察を行う。
