# 12・1・2 宇宙機の運動のモデル化

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・1 シミュレーション技術

宇宙機の運動に関する物理モデルや数学モデルの重要性が強く認識されるようになったのは、1950年代からであろう。初期に打ち上げられたスピン衛星において、軌道上で姿勢が不安定になる不具合が発生し、その原因を解析した結果、現在では常識的に知られている事実であるが、なんらかのエネルギー消散効果を有する衛星の最小慣性主軸回りのスピン運動は力学的に不安定であることが確かめられた。スピン衛星の安定性を数学モデルを利用して確かめる場合、エネルギー消散効果をモデルに反映させない場合は最大および最小慣性主軸回りに安定であるが、消散効果を考慮する場合は最大慣性主軸回りについてのみ安定となる。この事実から、モデル化の重要性とモデルが妥当かどうかを判断することの難しさが認識される。その後宇宙機は、単一のスピン衛星から二重スピン衛星、三軸姿勢制御衛星、国際宇宙ステーションに代表される大規模宇宙構造物(LSS : large space structures)へと複雑化、多様化し、解析的安定性の保証は不可能となり、ますます数値シミュレーションの重要性が高まった。解析対象に応じて本質的に異なる部分はモデル化であり、以下、これについて概説する。a. 基礎事項。 キネマティクスとダイナミクスシステムの運動を表現するための変数として、物体の速度や角速度が用いられることが多いが、これらの変数と慣性空間内での位置や角度の関係は、キネマティクス(kinematics)と呼ばれ、通常は力やトルクを含まない微分方程式で記述される。一方、システムの動的挙動を表す運動方程式は加速度、角加速度と力、トルクとの関係を表すものであり、このような関係はダイナミクス(dynamics)と呼ばれる。. 絶対座標と相対座標絶対座標(absolute coordi-nate)では、各物体を独立に扱い、各物体の運動はすべて慣性空間に固定した座標系で規定する。一方、相対座標(relative coor-dinate)では、ある物体を基準として選択し、その基準物体のみ慣性空間固定座標系で規定するが、その他の物体は基準物体から出発して隣り合う物体との相対的な変数で随時表す。. 真座標と擬座標各物体の運動を記述する独立変数のとり方は、その変数の力学的意義から二とおりある。一つは、真座標(true coordinate)と呼ばれる変数であり、これは系の位置を定めることのできるもので、通常、その物体的意義は明確である。真座標の時間微分を運動方程式の変数として用いた場合、その定式化の過程において、多くの三角関数を扱うこととなり、面倒な計算を強いられる。しかし、変数の時間微分から変位を求めるための運動学的な微分方程式は簡単である。これに対し、擬座標(quasi coordinate)は、その時間微分に意味があって、それ自身は系の位置を定めることができない。したがって、擬座標の時間微分と変位の時間微分を関係付ける運動学的な微分方程式は複雑なものとなる。しかし、擬座標を運動方程式の変数として採用すると、定式化に三角関数を含まないようにできる。擬座標の微分の変数の例としては、機体座標系で定義されたその物体の速度、角速度などが挙げられる。b. 柔軟体としての宇宙機のモデル化と定式化人工衛星が剛体であると仮定できる場合、並進運動および回転運動をおのおの3個の変数で表すことができる。しかし、柔軟体では、それ自身第II部基礎編の変形を許容するから、剛体運動に加えて変形を表す適当な変数が必要となり、運動方程式の次数は高くなる。柔軟体のモデル化のポイントは、一般の力学系のモデル化と同様に、実際に存在する対象のふるまいを可能な限り簡潔なパラメータセットによって数学的に表現することである。また、その目的は系のふるまいの特徴を定性的に把握すると同時に、定量的に評価することである。.
