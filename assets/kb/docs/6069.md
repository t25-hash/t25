# 12・2・1 環境試験(environmental test)の歴史

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

ロケットおよび人工衛星等は,打上げからミッション達成までの間,過酷な環境にさらされる.打上げ上昇時にはロケットエンジンの排気流や空力的な影響による振動,音響,空力加熱などの環境を受ける.軌道上においては真空環境とともに,太陽に当たる部分はふく射熱を受けて百数十°Cになる反面,陰の部分においてはマイナス百数十°Cの低温環境となる.これら人工衛星等の環境試験は,アメリカにおける宇宙開発の歴史とともに発展してきた.特にNASA GSFC(Goddard Space Flight Center)においては,1960年代から1970年代初頭にかけ,衛星の軌道上の不具合を分析することにより環境試験項目および設計仕様等の妥当性を評価した.これらの活動に基づき,GSFCでは環境試験仕様書(G-320-G-1)を確立した.NASA JPL(Jet Propulsion Laboratory)では,深宇宙探査機の試験および実フライト時の不具合を蓄積し分析することにより,環境試験に反映させた.また,ヒューズ社,ボーイング社,TRW 社等,民間会社においても,不具合データを分析することにより試験の有効性を評価した.現在に至っても欧米においてはフライトデータを取得するとともに,それらに基づく環境試験を実施することにより試験の信頼性向上に努めている.以下では,宇宙航空研究開発機構筑波宇宙センター[旧:宇宙開発事業団(NASDA)筑波宇宙センター]における環境試験を中心に紹介する.旧宇宙開発事業団の発足当初,G-320-G-1およびMIL-STD-810に基づく試験が計画されていたが,試験設備の図12・6 Simulink による制御系モデルの一例規模および試験環境等がアメリカと違うことなどから,レベルや許容誤差などが必ずしも適合するものでなかった.そのため,1970年から1980年代にかけ,G-320-G-1をもとに,宇宙開発事業団の試験設備に適合する試験共通仕様書(人工衛星編:ESPC-7)を制定するとともに,試験機打上げ時の環境測定を行い,これらの妥当性を確認した.その後,ロケットの大型化とともに見直され,1994年,現在の衛星一般試験標準(NASDA-STD-15)となったが,当初の実現可能な試験スペックという考え方は次第に薄れた.1994年以降人工衛星やロケットの打上げ失敗が続いたため,データベースの活用,試験検証の重要性が唱えられた.これに基づき1998年より試験データのデータベース化を始めるとともに,2000年からは試験スペックの裏付けをとるべく打上げ時および地上燃焼時等の実測データの取得を再開した.この間,人工衛星の打上げ時質量は82.5kg から4000kg 以上に増加するとともに形状も大型化し,初期の試験設備では対応できなくなった.このため,以下のフェーズにおいて増改築を行い運用してきている.(1) 第1期整備(1975年)N-Iロケットで打ち上げられる静止130kg 級の衛星等の試験をするために音響試験設備,7.9t振動試験設備,8mφスペースチャンバ等を整備した.(2) 第2期整備(1980年)N-IIロケットで打ち上げられる静止350kg 級の衛星などの試験をするために13.6t振動試験設備,910m音響試験設備等を整備した.また,衛星の寸法も大きくなったため,天井高の高いクリーンルームなどを新設した.(3) 第3期整備(1987年)H-Iロケットで打ち上げられる静止550kg 級の衛星などの試験をするため,13.6t振動試験設備などの改修を行った.このロケットのフェアリングはN-IIロケットと変わらないため,建物の改修までは必要としなかった.(4) 第4期整備(1989年)H-IIロケットで打ち上げられる静止2t級の衛星などを試験するために,総合環境試験棟を整備した.この衛星は,質量,寸法とも大型化し既設の試験設備では試験できなくなったため,試験設備,建物とも新設した.また,試験を効率化するために試験ごとに棟間移動することは避け,同じ建物内で試験できるようにした.(5) 第5期整備の検討(1997年)(4)で整備した総合環境試験棟は,第3期までに整備された試験設備と併用することにより,静止2t級衛星を年間2機打ち上げるために必要な試験を実施できるように整備された.しかしながら,最近はこれを超える打上げスケジュールが設定され,試験スケジュールが過密化したため,第2総合環境試験棟の整備について検討を行った.その結果,実現するまでには至らなかったものの8mφスペースチャンバ,大型振動試験設備など,現有試験設備の老朽化対策等を行い,過密化対策の一部とした.今後の展開としては,欧米に格差をつけられているフライトデータ等,実測データの蓄積に努めるとともに,これらを用いた試験スペックの見直し,試験結果と解析結果の比較などの検証業務を充実することにより,データベースに基づく試験の効率化あるいは試験の省略など試験期間の短縮化を目指した検討を進める.しかしながら欧米においては,これらの検討を進めるだけでなく,過密化する試験スケジュールを消化するために試験設備等の増設が行われている.わが国において試験設備の整備については,これまでは質量や寸法などの物理的な問題で必要性を説明できたが,最近はスケジュールという目に見えない問題であるとともに,これらは打上げの数年前に問題化するものであるため,せっぱ詰まった問題としては考えられていないのが現状である.
