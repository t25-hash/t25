# 12・2・2 振動試験(vibration test)

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・2 ロケット・人工衛星などの環境試験

a. 振動試験の目的振動試験設備は,ロケットで打ち上げられる衛星等の振動環境を地上で模擬する設備である.振動試験の目的をおおまかに分類すると,以下のようになる.(1) 設計・試作した構造物が,予想される振動環境に耐えることができるかを確認する.(2) 設計・試作した構造物の材料・部品に欠陥がないかを確認する.(3) 構造物の応答を確認する.b. 振動試験設備(vibration test facility)の加振方式振動設備は,加振方式,設備の構造などにより,以下のように分類できる.(1) 油圧(式)型サーボバルブを使用して,油圧アクチュエータを動作させる方式で,構造物の質量が大きく,振幅(ストローク)が長大な設備として建設されている.油圧式は,動電型に比較して大きな変位,大きな加振力,三次元同時加振が可能なことが特徴である.加振周波数は,DC(直流)~50Hz 前後と低く,振幅は,約100~300mmで,質量10~1000tの構造物を二次元,または三次元で同時加振することが可能である.また,大型の加振設備の加振時間は,油圧アキュムレータの容量により決定される場合があり,長くても1分程度の加振しか実施できない.(2) 動電(式)型筑波宇宙センターに設置されている設備は,動電型振動発生器の代表的なものである.動電式は油圧式に比べ,高い周波数(5~2000Hz)まで加振可能であり,加振波形精度が良好であるため,衛星等などの振動試験設備に採用されている.また,加振時間も特に制約がなく,長時間の加振が可能である.. シングル加振(SESA : single exciter,single axis)加振する構造物に対して一方向に加振を行う仕組みで,おもに1台の動電型振動発生器を使用した振動試験設備として国内外に多く設置されている.水平および垂直振動試験設備は振動台の上に構造物を据え付け,振動台横(水平振動試験設備の場合)および振動台の下(垂直振動試験設備の場合)に配置された振動発生器により加振する.この種の試験設備では振動台に高い剛性が求められるとともに,垂直振動試験設備の場合には振動台および構造物の質量を振動発生器で支持する必要があるため,軽量な構造が必要となる.そのため,国内外のシングル加振設備は小型のものが主流である.. 多点一軸加振(MESA : multiple exciter,single axis)(1) 水平振動試験設備水平振動台の上に構造物を据え付け,振動台横に配置された複数の動電型振動発生器,もしくは油圧アクチュエータにより加振する.シンプルな構造であるが,複数台の振動発生器を使用するため,位相制御が使用されている.振動台の大きさ,振動発生器の配置により,国内外の設備でそれぞれ特色がある.図12・8に多点水11―145第II部基礎編平振動試験設備の構成例を示す.(2) 垂直振動試験設備垂直振動台の上に構造物を据え付け,振動台下に配置された複数の動電型振動発生器,もしくは油圧アクチュエータにより加振する.振動台,構造物の質量を動電型振動発生器,もしくは油圧アクチュエータで支持する必要があるため,特殊な構造となる.そのため,国内外の多軸加振設備はバラエティに富んでいる.図12・9に多点垂直振動試験設備の構成例を示す.. マルチ加振(MEMA : multiple exciter,multiple axis)振動台に対して,複数の振動発生器(油圧式ないし動電式)が接続され構造物を加振する仕組みで,質量が大きく,転倒モーメントなどが大きい構造物を加振するための方式として採用される.・多点多軸(三次元)振動試験設備振動台へ,複数の振動発生器(油圧式ないし動電式)を接続することにより,数多くのユニークな振動試験設備が建設されている.二次元,三次元,6自由度の加振が可能である.振動発生器には,大きな質量の支持が可能な油圧アクチュエータが用いられる.図c. 制御方法振動試験設備の加振能力・構造により,振動台および構造物の振動を制御する方式は多岐にわたる.以下におもなものを示す.(1) オープン制御規定の振動波形を振動台に加えることを目的とし,構造物からの応答(フィードバック)信号は制御に使用しない.地震波を再現す図12・8 多点水平振動試験設備の構成例図12・9多点垂直振動試験設備の構成例図12・10 多点多軸振動試験設備の構成例11―146宇宙機器・システムる耐震性の確認用に用いられる.この場合,一般的には構造物の破損・破壊などは考慮しない.(2) クローズドループ制御振動台に規定の振動を加え,振動台および構造物の応答(フィードバック)信号により,構造物の応答を規定値内に収めるように制御する.衛星など高価な構造物の振動試験に向いており,高い制御精度と応答性および高速性を要求される.1SISO(single input,single output)制御制御を行う信号と応答信号が1対1で対応する制御で,小型・一軸の振動試験設備に使用されるが,あまり一般的ではない.2MISO(multiple input,single output)制御通常,平均値制御と呼ばれ,振動台上の複数の制御点(通常4点)を平均し,振動発生器(1台)に駆動信号を送る制御方式である(図12・11).国内外の多くの振動試験設備は本制御方式を使用しており,宇宙開発事業団の設備においても本方式を採用している.3MIMO(multiple input,multiple output)制御通常,多点多軸制御装置器と呼ばれ,振動台上の複数点の制御応答により,各制御点を規定の応答値に収束するよう各軸各部位の振動発生器(複数台)に駆動信号を送る制御方式である(図12・12).制御のアルゴリズムは,振動試験設備の振動台形状,振動発生器(油圧式ないし動電式)の配置などと密接に関係しているため,国内外個々の振動試験設備で微妙に異なる.4リミット制御おもに衛星システムの振動試験に使用される制御で,構造物の特定部位(コンポーネント,サブシステムなど)に張り付けた加速度計の応答により,振動発生器への入力信号抑制を行う.衛星システム振動試験では,構造物が大きいため,特定部位(コンポーネント,サブシステムなど)に過大な負荷が加わる場合があり,これを避けるためにリミット制御を行う.
