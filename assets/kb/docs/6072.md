# 12・2・4 衝撃試験(shock test)

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・10 に多点多軸振動試験設備の構成例を示す。

a. 衝撃試験の目的人工衛星は、火工品を用いてロケットから分離される。この際に高レベル、高周波の衝撃が人工衛星とロケットの分離面から人工衛星に負荷される。また、太陽電池パドルやアンテナ類を展開する場合にも火工品が用いられ、同様に展開衝撃による負荷が人工衛星にかかる。このような高レベル、高周波(場合によれば300万m/s以上、1MHz以上)の衝撃負荷に対して、人工衛星の各機器が耐えうるかを確認するのが衝撃試験である。b. 衝撃試験設備(shock test facility)上記のような衝撃環境を地上で模擬する設備が衝撃試験設備である。筑波宇宙センター所有の衝撃試験設備のシステム構成図を図12・18に示す。また、表図12・17 衛星フェアリング内部の音圧分布11―148宇宙機器・システム図12・18 衝撃試験設備システム構成図表12・3 衝撃試験設備概要点火電流5~10A点火回路数計測時間320ms計測周波数範囲DC~10kHz計測チャネル数加速度センサ200ch点火電流1chサンプリング周波数50kHz(a) 通常の加速度波形(b) ゼロシフトした加速度波形(d) ゼロシフトした加速度SRS(c) 通常の加速度SRS図12・19加速度波形とSRS の例衝撃試験設備は、おもに爆管制御部・計測部・データ収録部等から構成されている。爆管制御部は、衛星を分離させる火工品に点火電流を流すとともにデータ収録部および高速度カメラの制御を行うことができる。衛星に取付けられた加速度センサからの信号は、データ収録部に記録され、フーリエ変換・衝撃応答スペクトル解析(SRS : shock response spectrum)の処理を行うことができる。また、分離の様子は、高速度カメラによって撮影することができる。c. 衝撃試験データ計測の問題点衝撃試験データ計測時、計測データに実際には発生していない偽信号(pseudo signal)が混入する現象(ゼロシフト、zero shift)が確認されている。これはおもにセンサ内部の圧電素子の共振による。通常衝撃応答波形は、図12・19(a)に示すように衝撃が発生した後、ゼロレベルを中心に正負同程度の振幅をもったまま減衰していく形となる。これに対し、ゼロシフトが発生すると、図12・19(b)に示すように衝撃発生直後に正方向または負方向のどちらか一方に急激なシフト成分が図12・20 ウェーブレットを用いた衝撃データ補正結果発生し、時間の経過とともにゼロレベルに戻っていく。この結果、図12・19(c)(d)に示すように衝撃応答加速度の評価方法であるSRS 解析結果において、特に低周波成分で大きな相違を示し、試験データとしては不適なデータとなる。実際の衛星開発では、コスト、累積疲労の面から再試験を行うことが非常に困難である。また、再試験できたとしても有効な対策をとらないと同じようなデータが再びとれるだけである。よって、以下に示すようなゼロシフトを含むデータから偽信号を取り除き、評価に耐えうる試験データとすることが望まれる。d. ウェーブレット(wavelet)を用いた衝撃データ補正方法偽信号を含む試験データから真の計測信号(true sig-nal)を得るためには、ゼロシフトした計測データから偽信号成分を除去することが必要である。フーリエ変換ベースまたはハイパスフィルタ(HPF: high pass filter)を使った信号処理では、真の計測信号の低周波成分も除去してしまうため、適用ができない。そのため、時間軸上で処理が可能なウェーブレットを用いる。補正結果の例を図12・20に示す。これより、ウェーブレットを用いたデータ処理方法は本問題を解決するのに非常に有効であることがわかる。
