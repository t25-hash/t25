# 12・2・9 電波試験設備(radiowave test facility)

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・3 に概要を示す。

電波試験設備は、人工衛星の電波系システム試験のほか、ロケット・人工衛星の搭載用アンテナ・搭載用電波系センサ等の試験を行い、その電波特性を測定するための設備である。アンテナ測定方法としては、遠方界測定と近傍界測定(NFM :near field measurement)に大別できる。遠方界測定は、電磁波源と受信側との測定距離を2/λ以上としてアンテナの特性測定を直接行う方法である。はアンテナの最大寸法、λは自由空間波長である。近傍界測定は電磁波源と受信側との測定距離を遠方界測定より短い状態でアンテナの特性を確認する方法である。近傍界測定は、測定した被測定アンテナの近傍界電界データを高速フーリエ変換(FFT : fast Fourier transform)し、遠方界特性を求める手法である。遠方界測定は測定時間が短いが、アンテナの大型化、高周波数化に伴い測定距離を確保することが難しい。近傍界測定は電波無反射室内で測定するため、妨害波のない、温・湿度や清浄度等管理された良好な環境下で大型のアンテナの測定が可能であるが、測定時間がかかる。遠方界測定の一種であるコンパクトレンジシステムによる測定は、反射鏡により限られた空間で電波を平面波にすることで遠方界測定を実現するもので、電波無反射室に設置することで、比較的大型のアンテナを良好な環境下で短時間に測定できる。筑波宇宙センターが所有する測定設備は遠方界測定によるものであり、3測定設備を有する。第1無反射室内には、送受信距離24.6mの遠方界測定システムと一枚鏡によるコンパクトレンジシステムの2設備を、第2無反射室は、最大450mの測定距離の遠方界測定システムを有する。第1無反射室は電界シールドされたクラスM 6.5のクリーンルームであり、衛星の電波干渉試験・電波系総合試験も可能である。第2無反射室は、一方を屋外に開放し、屋外の移動型電波測定塔と組合わせることにより、遠方界測定に必要な距離を確保している。試験設備概要を表12・8に示す。図12・26 旋回腕型加速度試験設備の外観11―151第II部基礎編
