# 12・2・11 磁気試験設備(magnetic test facility)

> **章**: 第12章 シミュレーション技術と試験法
> **節**: 12・3 に概要を示す。

地球磁場と人工衛星の有する残留磁気によって衛星の姿勢が乱れることを防止するため、磁気試験設備による磁気モーメント(magnetic moment)の測定、残留磁気の消去や磁気姿勢制御装置等の機能確認を行う必要がある。また、高精度の磁気センサ等の特性を確認する。本設備は測定室、消磁室、前室、等価コイル室および制御室から構成されている。測定室には三軸ブラウンベック方式の主コイルが設置されており、安定電流を流すことによって地磁気を消去する。この零磁場空間で、人工衛星や電子機器の磁気モーメントを近傍界測定により測定すること、および高性能の磁気センサの校正ができる。試験設備概要を表12・10に、システム構成を図12・28に示す。電車、自動車、建造物、電力線、通信線、設置線、電気機器、機第1無反射室第2無反射室送受信間距離24.6m固定式7.315m最大450m(電波測定塔使用)電波試験塔高さ:6.2~26mの任意長に設定可能試験周波数範囲450MHz~26.5GHz1~94GHz1250MHz~33.8GHz で電波法に基づき許可を受けている周波数図12・28 磁気試験設備システム構成図械設備、上水管、下水管、ガス管等に用いられる磁性材料やこれらに流れる電流等は地磁気変動の原因となる。これらの妨害物によって擾乱された磁気は距離により減衰し、300m以遠では100万分の1以下になるため、本試験装置の周囲は、半径300mにわたって磁気管理エリアとしている。
