# 13・1・1 おもな月(Moon)の探査

> **章**: 第13章 月・火星などへの展開
> **節**: 13・1 過去の探査衛星

ルナ2号(Luna2):月面に衝突[旧ソ連(旧ソビエト社会主義共和国連邦)、1959年]ルナ1~24まであり16,17,20,21,24は月に軟着陸。ルナ3号(Luna3):月面の裏側の写真撮影(旧ソ連、1959年)。サーベイヤ(Surveyor)1~7:無人月着陸し、写真撮影、土壌分析(アメリカ、1966~1967年)。アポロ(Apollo):11号が人類初の月面着陸。18号は中止、月面に7回着陸して6人が月面に降り立ち、月面からサンプルを持ち帰った(アメリカ、1969~1972)。アポロ12号はサーベイヤ3の近くに着陸し、サーベイヤ探査機からカメラの一部、スコップ等を持ち帰った。持ち帰ったガラスに付着した細菌は生きていた。ルナ16号(Luna16):自動的に月面からサンプルを持ち帰る(旧ソ連、1970年)。クレメンタイン(Clementine):1994年1月25日打上げ、月の周りを425kmから2950kmのだ円軌道で周回し、紫外(UV)から中間赤外(mid-IR)で2箇月間月面のマッピング(アメリカ、1994年)。ルナ・プロスペクタ(Lunar Prospector):1998年1月6日打上げ(アメリカ、1998年)、氷の存在を調べる。月面全体の精密な重力地図作成。スマート1(SMART-1):2003年9月28日打上げ、小型ミッションのための技術開発が目的(small missions for advancedresearch technology)。最大の特徴は電気推進を利用して月に向かっている点。カラーCCD カメラ、X 線スペクトロメータ、赤外線スペクトル探査装置を搭載し、月面の元素組成と鉱物組成を調べる。南極のサウスポール・エイトケン盆地が調査の重点。2005年2月に月に到達し、2006年9月に月面に衝突して予定どおり探査を終了した(ヨーロッパ宇宙機関、2006年)。ルナーA(LUNAR-A):日本、打上げ計画未定。月面地形の観測・月の内部構造調査、アポロ12号着陸地点近傍と、ほぼ反対側の2地点に投下したペネトレータによる月震・熱流量計測。母船は月の周りを周回し、月撮像カメラによる観測を行う。ミッション期間は1年を予定。セレーネ(SELENE):日本、2006年度打ち上げ計画。月面の全球観測、地形と誕生の歴史・極の氷の調査、着陸機の月面への軟着陸。
