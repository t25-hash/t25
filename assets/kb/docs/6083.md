# 13・1・3 その他のおもな惑星探査

> **章**: 第13章 月・火星などへの展開
> **節**: 13・1 過去の探査衛星

マリナー2号(Mariner2):金星(Venus)の周りをフライバイし、金星の温度測定(アメリカ、1962年)。パイオニア(Pioneer):10号は木星の周りをフライバイした最初の衛星(アメリカ、1973年)、そのあとを11号が続いた(アメリカ、1974年)。その後、土星を調査。小惑星帯と木星の磁場を測定を目的。最後のデータは1997年3月31日に得られた。マリナー10号(Mariner10):金星の大気の紫外線像を得、大気が4日で巡ることを明らかにした(アメリカ、1974年)。ベネラ7号(Venera7):金星表面のデータを得た(旧ソ連、1970年)。ベネラ9号(Venera9):金星に着陸し表面の画像を得た(旧ソ連、1975年)。パイオニア・ビーナス(Pioneer Venus):金星表面の地図を作った(旧ソ連、1978年)。ボイジャー1号(Voyager1):1977年9月5日打上げ、1979年3月5日に金星を通過、1980年11月13日に土星を通過、冥王星(プルート)と土星の衛星タイタンの映像撮影(アメリカ、1977年)。現在も動作中。ボイジャー2号(Voyager2):1986年8月20日打上げ、1979年8月7日に金星を通過、1981年8月26日に土星を通過、1986年1月24日に天王星を通過、1989年8月7日に海王星を通過、その後太陽系外を探査(アメリカ、1977年)。現在も動作中。ベガ(Vega):金星・ハレー彗星探査の国際プロジェクト、ハレー彗星の近傍を通過(旧ソ連、1984年)。ジオット(Giotto):1985年7月2日打上げ、1986年3月13日ハレー彗星の核に540kmまで接近(ヨーロッパ宇宙機関、1985年)。さきがけ:1985年1月7日打上げ(日本、1985年)。惑星空間の宇宙プラズマ・磁場観測。すいせい:1986年8月18日打上げ(日本、1986年)。ハレー彗星探査。フォボス(Phobos):打上げ失敗(旧ソ連、1988年)。マゼラン(Magellan):1989年5月打上げ(アメリカ、1989年)。金星表面の98%を300m以上の分解能で地図を作成、重力のデータを収集。マーズ96(Mars96):1986年11月17日打上げ失敗(ロシア、1996年)。ガリレオ(Galileo):1989年10月18日スペースシャトルから打上げ(アメリカ、1989年)。現在木星の周りの衛星軌道上。小惑星の映像、彗星SL-9の木星表面衝突の映像を撮影。ハッブル宇宙望遠鏡(Hubble Space Telescope):1990年4月打上げ(アメリカ、1990年)。ユリシーズ(Ulysses):1990年10月スペースシャトルから打上げ(ヨーロッパ宇宙機関/アメリカ、1990年)。木星重力でスイングバイし、太陽周りのだ円軌道上。太陽風の観測。ウィンド(Wind):1994年11月1日打上げ(アメリカ、1994年)。太陽風の観測。ニアー(NEAR):1996年2月17日打上げ(アメリカ、1996年)。小惑星エロスに1999年1月に接近し、2001年2月に軟着陸した。カッシーニ(Cassini):1997年10月15日打上げ(アメリカ、1997年)。金星・木星をフライバイして2004年7月に土星に到達。2005年1月には子機ホイヘンスを切り離し、土星の衛星タイタンに突入させる。2006年現在、タイタンの大気を計測中。ルナ・プロスペクタ(Lunar Prospector):1998年1月6日打上げ(アメリカ、1998年)。30年間にわたり月面観測。スターダスト(Stardust):1999年2月打上げ(アメリカ、1999年)。彗星に接近し、コマ部分の物質を持ち帰る。最初のちりの収集は2000年2月から5月まで行われた。2004年に彗星ヴィルト(Wild)2をフライバイし2006年に地球に帰る予定。ディープ・スペース1号(Deep Space1):1998年10月24日打上げ。2001年9月にボレリー(Borrelly)彗星に接近し、2001年12月にミッション終了。みどりII:2002年12月14日打上げ(日本、2002年)。地球環境技術衛星、海水温度分布の測定、水塊と漁場との関連調査等。第III部宇宙空間への進出11―155エウロパ・オービタ(Europa Orbiter):2010年打上げ計画(アメリカ)。木星の衛星エウロパの氷の厚さ・液体の海を探査予定。プルート・カイパー・エクスプレス(Pluto-Kuiper Express):2006年打上げ、2016年に冥王星と遭遇する計画だったが延期(アメリカ)。ミューゼスC(MUSES-C):쓕はやぶさ」2003年5月9日打上げ(日本、2002年)。小惑星探査・サンプルリターンを計画。小惑星「イトカワ」(1998SF36)に到達後詳細な観測を高性能カメラ、赤外線、X 線などの波長で調べ、「イトカワ」のかけらによって太陽系の起源を明らかにするミッションも含む。2005年12月に「イトカワ」にランデブーし、数十分間着地した後、離脱。その後の軌道については調査中。イオンエンジンによって飛行している。メッセンジャー(Messenger):2004年打上げ計画(アメリカ)。水星全体の詳細な画像と、大気・磁気の観測。
