# 13・2・2 火星の話題

> **章**: 第13章 月・火星などへの展開
> **節**: 13・2 惑星に関する最近の話題

・トピックス(水、ドライアイス)の存在。・有機物の存在調査(バイキング1号)。・火星より飛来した南極の隕石の中に生命体の痕跡発見。・アメリカ探査機マーズ・パスファインダによる観測で、20億年から30億年前に火山活動などの急激な環境変化によって氷が溶け洪水を起こした跡を思わせる地形が確認された(1997年7月5日)。・火星の表面近くの地下に大規模な氷原がある(火星探査機オデッセイによる観測、2002年5月29日)。・火星大気中に0.01ppm程度のメタンを検出(マーズ・エクスプレス、2004年)。・火星の南極の極冠が水の氷、ドライアイス、永久凍土で成り立っている(マーズ・エクスプレス、2004年)。・火星にかつて海があった強力な証拠(オポチュニティ、2004年)。・火星の岩から水の痕跡を発見(スピリット、2004年)。
