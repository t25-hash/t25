# 13・2・3 その他の惑星に関する話題

> **章**: 第13章 月・火星などへの展開
> **節**: 13・2 惑星に関する最近の話題

・外惑星(木星,土星,天王星,海王星)にはリングが存在する.・木星の衛星イオには火山が存在する(ボイジャーによる観測).・木星の衛星イオから溶岩の吹上げ,巨大なカルデラとクレータがとらえられた(木星探査機ガリレオのカメラ,赤外線マッピングスペクトロメータ).・土星の衛星タイタンには濃い大気(約2atm)が存在する.・土星の衛星テレスト(L4),テティス,カリプソ(L5)はラグランジュ点に位置する.・冥王星の質量は1948年にバビネットが地球質量の12倍と推定して以来,減少の一途をたどっている(実は彗星ではないかといわれている).・木星の衛星エウロパの表面の氷の下に水の海がある(ガリレオの磁気観測データ).・木星の衛星カリストとガニメデにも水が存在する可能性が出てきた.11―156・第10番惑星発見:セドナ(Sedna,2003VB 12).冥王星の2分の1から4分の3程度の大きさで,地球から90AU の距離(地球と冥王星の距離の倍以上)にあり,約1万年で太陽の周りを公転(小惑星2004DW は直径1400km程度).・冥王星は海王星の外側を回る太陽系の第9番惑星とされ,唯一探査衛星が到達していない天体である.直径2320km,軌道の離心率が大きくその一部は海王星の内側となり,平均軌道約59億kmである.1930年にトンボー(アメリカ,ローウエル観測所)によって発見された.衛星カロン,ヒドラ,ニクスをもっている.第10番惑星2003UB 313が発見されたことがきっかけとなって,2006年8月24日プラハでの国際天文学連合総会で惑星から除外されて小惑星(dwarf planet)に分類され,改めて小惑星番号134340が割り当てられた.これらはエッジワイス・カイパーベルト天体であり,2003UB 313も134340も似た性質をもつと考えられる.現在,冥王星探査機ニューホライズンズが向かっており,2015年ごろ接近の予定である.その調査から多くの知識が得られることが期待できる.
