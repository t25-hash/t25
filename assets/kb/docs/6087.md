# 13・3・1 探査形態

> **章**: 第13章 月・火星などへの展開
> **節**: 13・3 探査システム(exploration system)

21世紀は、人類が月や惑星など太陽系にまったく新しい文明圏を創り出す時代になるといわれている。これまでの宇宙探査ミッションにおいて、最も印象的な出来事は、なんといってもアポロ計画で人類が月面に降り立ったことである。また、ロボットが火星表面を移動しながら探査を行ったことも記憶に新しいところである。そのほかにも探査機がさまざまな惑星の鮮明な画像を地球に送って、人々に夢と感動を与えてきた。天体の衛星や小惑星など小天体にも接近することが可能となり、宇宙探査技術の進歩には目をみはるものがある。惑星探査(planetary exploration)が多くの人の関心を集めるのは、見たこともない光景が次々に展開されるからであり、知らないことを知りたいという人間本来の欲求によるものである。世界の宇宙探査ミッションを概観してみると、一時期大きく注目された有人の月・惑星探査に代わって、無人のロボティクスミッションに重点がおかれてきている。そして月・火星を集中的に深く研究する時期が訪れており、ロボティクス技術を用いた深宇宙探査が期待されている。深宇宙とは、地球重力圏よりも遠方の宇宙空間のことを指す。探査の対象にできる天体も多くなり、探査方法も惑星表面サンプルの回収など、精密かつ多面的な観測を可能とするようになっている。探査を行う対象の惑星や科学目的に応じて、さまざまな惑星探査が行われる。図13・1に深宇宙探査ミッションにおける探査形態を示す。フライバイオービタペネトレータ1回めの発見的探査磁気圏探査マッピング惑星内部探査地震計ネットワークランダローバサンプルリターン地球でサンプルの詳細分析表面探査移動探査(ローバ)図13・1 深宇宙探査形態惑星のそばを1回だけ飛行するのがフライバイ(接近通過)ミッションである。これはフライバイ時の短い時間にリモートセンシングを行うものである。より詳しく惑星を観測するためには、探査機のエンジンを噴射させ、惑星の周回軌道に入れるオービタ(周回衛星)ミッションがある。宇宙科学研究所(現:宇宙航空研究開発機構、JAXA)の火星探査機「のぞみ」がこのタイプである。惑星の上空から広域画像を撮って惑星全体を理解し、周辺の磁場や粒子観測をする計画であった。惑星の内部の様子を知るには、貫入型の観測器を用いたぺネトレータミッションがある。ぺネトレータを惑星全体に数本打ち込んでネットワークを形成し、地震波の伝わり方を測定することにより惑星の内部構造を調べることができる。惑星表面を探査するには、エンジンを逆噴射させながら着陸するランダミッションがある。約30年前に火星に着陸したバイキング探査機がよい例である。また比較的安全なところに着陸した着陸船から離れて、科学的に興味深い場所を広範囲にわたって惑星表面を移動しながら探査するのがローバミッションである。アポロ月着陸船は有人のランダであったが、日本の次期月探査ミッションでは無人ランダの検討を行っている。惑星の石や土壌のサンプルを採取して地球に帰還させることができれば、もっと詳しい分析が地球で行える。これがサンプルリターンミッションである。
