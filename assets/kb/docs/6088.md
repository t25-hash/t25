# 13・3・2 深宇宙探査(deep space exploration)ミッション

> **章**: 第13章 月・火星などへの展開
> **節**: 13・3 探査システム(exploration system)

深宇宙探査機の歴史は古く,すでに今から40年以上前の1959年には,アメリカ航空宇宙局(NASA)が初の人工衛星パイオニア4号を打ち上げている.それ以来,数多くの惑星探査機が打ち上げられてきたが,アポロ計画を除いて,すべて無人探査機である.太陽系探査では,冥王星を除く全惑星に探査機を送り込み,鮮明な画像を撮像した.ここでは,天体ごとに今までどのような探査が行われ,また今後どのようなミッションが行われるかについて,簡潔に述べる.a. 海外の探査ミッション太陽にいちばん近い惑星である水星ミッションでは,3回フライバイを行ったマリナー10号が,これまでで唯一の水星探査機である.水星表面の半球分の撮像が行われた.現在水星探査が注目されており,NASA はディスカバリー計画の一つに水星探査機メッセンジャーを選定した.ヨーロッパ宇宙機関(ESA : European Space Agency)とJAXA は,ベピコロンボ計画を検討している.周回衛星により水星に対してグローバルに高精度かつ高分解能な探査を行い,また着陸機を用いた水星表面や内部構造の探査を行う計画である.金星は地球に近い惑星ということから,マリナー,べネラ,パイオニア,ビーナス探査機などにより金星大気の探査が行われた.また,マゼランによりレーダ地形探査が行われた.地球の衛星である月に関してはさまざまな探査が行われている.アメリカのアポロ計画と旧ソ連によるルナ計画は有名である.米ソの冷戦が宇宙開発競争に拍車をかけ,1970年代には数多くの探査機が月面に送り込まれ,移動探査車やサンプルリターンなど積極的な探査が行われた.月は,表面のサンプル回収が実施された唯一の天体である.それ以後月探査の熱が冷めたかに見えたが,1990年代に入って再び月探査が注目され,ルナ・プロスペクタやクレメンタインにより月の全表面探査が行われた.火星はNASA が積極的に探査を推進している天体で,特に地球外生命の発見が注目されている.火星探査(Mars exploration)ミッションでは,驚くべきことに今から約30年前に2機の火星探査機バイキングが軟着陸に成功し,火星表面のマッピングと着陸機による直接探査を行った.その後,数多くの探査機が打ち上げられたが,その多くが火星に到達する前に音信不通やなんらかの不具合で火星探査ミッションを実現できないということがしばらくの間続いた.1997年になってようやく,マーズ・パスファインダ(MarsPathfinder)が火星着陸に成功し,ソジャーナが火星表面の移動探査を行った.マーズ・グローバル・サーベイヤは火星周回衛星となり,高度や重力などの全球マッピングを行っている.木星やその衛星は,パイオニア,ボイジャー,ガリレオによって探査が行われた.木星の衛星であるエウロパの周回探査も計画されている.土星,天王星,海王星の写真もボイジャーによって撮像された.1997年にはNASA が土星探査機カッシーニを打ち上げ,2004年に到着した.カッシーニには,ホイヘンスというプローブが搭載され,パラシュートを使って大気をもつ衛星タイタンに着陸した.彗星や小惑星などの小天体探査(small body exploration)も積極的に行われている.1986年のハレー彗星探査は有名であり,ジオット,ベガ,そして日本の「さきがけ」や「すいせい」が打ち上げられた.また,ガリレオ探査機によりガスパラやアイダなどの小惑星の撮像が行われた.最近ではNASA のディープ・スペース1号(Deep Space1)が,自律機能や電気推進技術など深宇宙探査技術の検証を行い,1999年7月に小惑星1992KD に接近してフライバイ観測を行った.NASA のニアー(NEAR)ミッションでは,2001年2月12日に小惑星エロスに軟着陸を果たし,エロスの組成や構造に関するさまざまな観測を行った.b. 日本の探査ミッション宇宙科学研究所は,1998年7月に火星周回探査機「のぞみ」を打ち上げた.火星探査機「のぞみ」は,火星の大気の散逸過程など上層大気と太陽風の相互作用や磁場の測定を行う予定であった.機器の不具合により火星周回を断念したが,月のスイングバイ時に月の裏側の鮮明な写真撮像に成功した(図13・2).また,距離は556万kmほど離れていたが,火星の自律撮像に成功した.日本で初めて軌道上から撮像した火星の写真はインタネットで公開されている.2003年に,小惑星探査機ミューゼスC(MUSES-C)を打ち上げ,この探査機は「はやぶさ」と命名された.1998SF36という直径1km程度の小惑星に探査機を送り込み,表面のサンプルを採取する,サンプルリターンミッションである.さらに,ぺネトレータと呼ばれる観測プローブを月面の表裏に1本ずつ貫入させるミッションの検討をしている.宇宙開発事業団と宇宙科学研究所(現在はともに宇宙航空研究開発機構,JAXA)は共同で月周回探査ミッション,セレーネ(SELENE)を推進している.図13・3に示すように,月の極周回軌道からグローバルに高精度かつ高分解能で観測し,元素・鉱物分析や重力場・三次元地形などの探査を行う.さらにセレーネミッシ図13・2火星探査機「のぞみ」による月面画像(提供:ISAS/JAXA)図13・4
