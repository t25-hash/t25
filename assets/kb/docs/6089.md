# 13・3・3 着陸・移動探査

> **章**: 第13章 月・火星などへの展開
> **節**: 13・3 探査システム(exploration system)

惑星探査は、惑星の地形などの外観を調べる初期の偵察型から、探査目的を絞って高精度・高分解能をねらう探求型に移行している。ここでは、惑星表面の詳細な探査を行う、着陸・移動探査ミッションについて述べる。a. 着陸探査(exploration by lander)現在まで行われた月・惑星の着陸探査には、アメリカと旧ソ連による月および火星探査ミッションがある。アメリカのアポロ計画は有人探査であったが、それ以外のミッションは無人で行われた。ここでは、実際に行われたミッションについて、着陸方法と着陸探査ミッションの内容について紹介する。. アポロ計画(Apollo Program)1969年7月20日は永遠に歴史に残る記念すべき日であった。この日、アメリカの宇宙船アポロ11号は月面着陸に成功し、人類が初めて月面に降り立った。着陸船は、コンピュータがあらかじめ選んだ「静かな海」のあたりには大きな岩がごろごろしていたうえ、クレータがあったため、月面上空150mのところまで降下したところで、手動による操縦に切り換え、着陸地点を求めて飛行を続けた。着陸船の月での滞在時間は21時間36分で、そのうちの2時間32分をニール・アームストロング船長とエドウィン・オルドリンが月面での活動を行った。アメリカ国旗を月面に立て、地震計、レーザ反射板、太陽風測定器を据え付け、写真を撮影し、石や土を約22kg ミッションの目的本ミッションの目的は、将来の火星探査に向けての主要な技術の実証である。またいくつかの科学探査機器を搭載して火星大気、気象学、地質学、岩石や砂の組成の調査を行う。さらに小型の移動ロボットを用いた探査技術を習得することを目的としている。マーズ・パスファインダは、NASA 線スペクトロメータがそれぞれ搭載されている。初期ミッションでは、30火星日間(1火星日は約24.6地球時間)にわたって、火星の大気と気象と地質について調査を行った。科学者が当初期待していた以上に豊富なデータを収集して、1997年8月3日に一つの区切りとなる30日間のミッションを完了した。地球に送信してきたデータの量は全部で1.2Gb(ギガビット)に達し、9669枚もの素晴らしい火星の画像が届いている。本ミッションは最大1火星年の観測が可能であり、初期ミッション後、引き続き運用された。マイクロローバは初期ミッション7火星日をはるかに上回り、30火星日たった後も十分に良好な状態にあった。30日間で着陸機ランダの周囲を約52mも走行しており、岩石やランダを撮影した画像は384点にものぼる。(3) 科学ミッション移動探査によるミッションとしては、月・惑星の起源や進化、現状の調査のために、環境調査、地形調査、地質調査、有機物探査、水探査などが挙げられる。地形調査では、カメラ画像によりクレータ、惑星性地形、地殻形成期の火山、断崖の断面の層状構造などが理学観測対象となる。地質調査では、近接カラー画像により土壌や岩石の分布、岩片のサイズや形、岩石のおおよその組成を知ることができる。また、地震計などによる内部構造解明や質量分析器を用いた岩石などの年代測定も重要である。蛍光X 線スペクトロメータ、α線後方散乱スペクトロメータを用いて、元素組成および鉱物組成を知ることができる。リモートセンシングで得られている科学データの校正データ取得もローバの重要な役目である。人類の宇宙探査における興味の中で最も大きな地球外生命の探査では、有機物の定量・定性分析や光合成反応実験、代謝反応実験が行われる。. ローバの役割ローバ(rover)の特徴は二つある。一つは移動できるということである。ランダミッションが着陸地点付近の探査であるのに対して、ローバミッションでは広範囲な探査が期待できる。また、クレータや断崖など地殻が露頭している地域など、調べてみたい場所の探査を可能にする。二つめは、ローバが月面に接触しているということである。そのため、サンプル採取やその場(in situ)分析、掘削等による地下探査や観測機器の設置、サンプルリターン用のサンプル収集などが可能である。ローバの基本的な作業は、周囲観測による調査地域の選定、目標地域への移動観測、科学観測データ取得などである。環境観測支援では、ローバおよびマニピュレータなどによる計測センサの位置姿勢制御が必要となる。表面のサンプルは比較的新しいものであり、宇宙線に汚染されている可能性がある。したがって、磨く・削る・割るなどの作業により岩石などのサンプルの表面や内部を露出することが必要である。地下探査では専用機器による掘削や計測センサの設置が要求される。地震波探査や地盤調査では、計測機器の設置や移動しながらのデータ取得などの作業がある。. ローバの技術課題ここでは、月・惑星探査ローバの設計や開発における主要な技術課題について述べる。(1) 宇宙環境適応技術ローバを月・惑星に送り込むには、ロケットによる打上げおよび着陸時の衝撃・振動に耐える構体系が必要である。太陽からの放射線や昼夜の大きな温度差、真空、レゴリスという過酷な環境で長期間活動できるロボットを開発するためには、解決すべき多くの技術課題がある。例えば、耐放射線機能を有する電子機器の開発が不可欠であり、高機能ローバを実現する宇宙用高速CPU(centralprocessing unit)が必要である。(2) 熱制御技術月や惑星では温度の差が激しいため、熱制御機能が特に重要となる。月では昼夜が約15日ごとであり、温度は最高120°C、最低-170°Cとなる。火星の1日はほぼ地球と同じであり、+15°Cから-100°Cと月ほどではないが温度差が大きい。したがって、月・惑星ローバの熱設計は重要な開発要素となっている。月は真空であることから空気による対流はなく、放射と熱伝導により温度が決まる。日中は太陽による放射と表面からの赤外線放射、車輪を介した熱伝導、ローバ搭載機器の発熱がある。夜間はローバからの放射と表面への伝導がある。ソジャーナ(マーズ・パスファインダの探査ローバ)では火星の夜を過ごすためにRHU(radioisotope heaterunit)が搭載されたが、環境に対する安全性が大きな課題である。受動的な熱制御では、蓄熱材とヒータで夜間保温を行う。昼間は放射面(ラジエータ)より機器の発熱を放出する。また内部を断熱構造とすること、外表面積を縮小すること、外部搭載機器を夜間収納することにより、熱散逸量の低減化が考えられている。そのほか、低温に耐える部品や材料、夜間の保温に効果のある断熱材、軽量で耐低温性のあるバッテリなどの開発が大きな課題である。また、急激な温度変化による構造材料や構成部品の熱ストレスへの考慮も必要となる。(3) 電源技術ローバの移動に伴い太陽方向が変わるため、太陽追尾メカニズムが必要となる。また、不整地を走行する際に生じる振動や衝撃に耐える太陽電池が必要である。小型軽量大容量バッテリの開発も必要である。(4) 走破技術高温、真空といった地球上とは大きく異なる月・惑星環境での走行駆動技術は、今後の本格的な月・惑星探査にとって重要である。広範囲を探査するミッションでは、ローバはできるだけ不整地走破性能の高いものが求められる。また、ミッションによっては、クレータや崖などの特定地形に特化した走破系を開発することも要求される。地表面の状況は地域により異なるが、月面の海ではレゴリスと呼ばれる細かい砂が厚さ数十cm程度積もっている。砂地でスタックしないようにタイヤの接地圧を小さくする工夫が必要である。月の極地域や裏側では大小さまざまなクレータが多数存在する。火星表面は、比較的障害物の少ないところに降りた2機のバイキングの周囲でも、30cmを超える径の岩石が数mの間隔をおいて分布している。小惑星は隕石などにより生成されたクレータが多数見られるが、その表面が硬い岩で覆われているのか、軟らかい砂からなるのかは不明である。このためローバの走行系設計においては、それぞれの特徴に対応したものが求められる。また直接表面と接地しながら走行するため、砂などの防じん対策も重要である。ローバの電力容量は限られており、省電力で効率よく走行する機構の開発が課題となる。(5) 航法誘導技術月や火星の場合、地球との通信時間が往復数秒から数十分かかる。また、非可視運用や通信リンクの容量確保に要するコストなどから、遠隔操縦だけでは長距離ミッションは難しい。したがって、遠隔操縦技術と自律化技術が併用される。未知の環境において、搭載センサを用いた障害物回避など知的な判断を行う、環境適応型の航法誘導手法が不可欠である。またローバの位置を同定する手法が必要である。(6) テレサイエンス(telescience)技術ローバを遠隔操作するためには、動画像を地上に送る必要があるが、通信回線の制約から画像圧縮技術は不可欠である。月面はコントラストの強い世界である。視認性が悪く特徴も少ない未知環境において画像を頼りにテレオペレーションするためには、ローバを含む周囲の状況を把握することが必要である。カメラの配置やどのシーンを優先的に送信するかなど環境提示システムが重要となる。通信時間遅れの問題を解決するために、予測画像や人工現実感を導入し、科学者の使い勝手のよいヒューマンインタフェースシステムを構築することが必要である。(7) マニピュレーション技術科学観測を支援するシステムの一つとしてマニピュレータがある。機能としては、遠方および周辺環境の観測、試料のサンプリング、分光カメラなど観測機器の表土や岩石などの試料への接近・接触、地面の掘削、試料の破砕、試料表面からのレゴリス等の除去、太陽電池パネルの清掃などのメンテナンス、三次元計測などが挙げられる。月・惑星探査用マニピュレータを開発するための要求としては、小型・軽量化、低消費電力化、長いリンク長(遠方の可視、作業空間の確保)、作業に十分な自由度(作業空間の確保)、カメラなどセンサの搭載(アクティブセンサ)、ミッションに合わせたエンドエフェクタの搭載などがある。(8)
