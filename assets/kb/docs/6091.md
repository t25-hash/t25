# 13・4・2 火星(Mars)利用の将来構想

> **章**: 第13章 月・火星などへの展開
> **節**: 13・4 惑星利用の将来構想

a. 現状と将来火星は太陽系の成り立ちだけではなく生命の誕生と進化に関わる科学探査の対象として注目されている。21世紀初頭の現在、火星に人類を送り込んでさまざまな探査を行うこと(有人火星探査)が当面の目標となっており、そのための予備調査としてすでにいくつかの無人探査機(自走式のローバなど)が火星表面に送り込まれ、さまざまなデータの収集が行われている。火星の大きさは地球のほぼ半分で、表面重力は地上の約3分の1である。表面温度は130~300K で自転速度や自転軸傾斜角は地球とほぼ同等である。わずかながら二酸化炭素を主成分とする大気が存在し、水の存在も示唆されている。ただし、大気が希薄で磁気圏がないため、火星表面には宇宙線が降り注いでいる。このように火星表面はそこで人間が活動するためには非常に過酷な環境であるが、月や他の惑星表面の環境と比較するとはるかに地球環境に近い。そのため、将来の人類の移住先として取り上げられることが多い。さらに遠い将来には火星の大気構造を操作して温暖化させ、地球と同等の環境を作り上げる(テラフォーミング、terra form-ing)構想も提案されている。火星の利用形態は、以下のように基本的には月の場合とほぼ同様である。(1) 位置の利用太陽系探査のための前哨基地地球からの遠さを生かした商業利用(観光)(2) 火星環境(希薄大気、低重力、低温、低ノイズなど)の利用火星天文台(火星からの観測)火星内部の科学探査材料などの新技術開発研究拠点11―171第III部宇宙空間への進出(3) 資源の利用水、酸素および炭素の抽出:有人活動支援、輸送系燃料(酸化剤)その他金属などの抽出:各種構造材料その他無機材料の利用:建設資材エネルギー生産:太陽エネルギー、火星面風力発電(4) 精神世界としての利用タイムカプセル、メモリアル火星は月に比べてはるかに遠いので、そこで使われる機器・装置類にはより高度な技術(自律性など)や信頼性が要求される。そのために、月面基地等で培われた技術が役立てられる。また、資源面において月と決定的に異なることは、火星では炭素や水素が得られる可能性があることである。これらは植物生産ならびにその延長線上にある食料生産を可能とするばかりでなく、燃料や薬品類など各種の有機系材料をも利用できるようにするため、火星では月に比べて自律性の高い活動が可能になる。火星利用のシナリオは、まだ詳細には検討されていない。21世紀初頭、アメリカをはじめとする各国の宇宙探査計画の中では本格的な火星探査のために月面基地を活用することが提案されているが、月面基地を経由せずに直接火星を目指すシナリオも検討されている。b. 火星利用の形態。 位置の利用(1) 太陽系探査のための前哨基地火星の脱出速度は約5.0km/sであり、火星から物資や探査機を打ち上げるために必要なエネルギーは、地球から打ち上げる場合の約5分の1である。そのため、太陽系探査の中心基地として火星は有効に利用できる。(2) 観光旅行という観点からは火星は少し遠すぎるが、地球以外の惑星に降り立って景色を見たいという願望をもつ人は多いかもしれない。このような火星観光を実現するためには、物資や人を運ぶための巨大な宇宙機が必要となる。直接人間が火星まで行くことなく、環境センサ(カメラ、マイク、温度計など)のみを火星表面の数箇所に設置し、地上からその様子をモニタして楽しむ仮想観光なども考えられている。. 火星環境の利用(1) 火星天文台火星にはわずかながら大気が存在し、砂嵐なども発生している。その表面に天文台を建設することはあまり得策ではないかもしれない。ただし、VLBI(very long baseline interferometry、超長基線干渉計)観測の拠点として火星を利用することによって、より精度の高い位置計測ができる可能性がある。(2) 火星内部の科学探査月の場合と同様に、太陽系の進化を明らかにするための科学探査ならびに火星利用のための地形学、力学、地質学、鉱物学的な探査が行われる。(3) 新技術開発研究拠点火星の表面環境は月よりも地球環境に近いので、月の場合ほどその環境を利用した技術開発は期待できないかもしれない。ただし、火星基地の自律性は月面基地よりも高いと予想されるので、火星ではより長期にわたる研究が可能となる。. 資源の利用火星の鉱物資源に関してはまだ十分なデータが得られていないが、同じ太陽系に属していることから、鉱物学的には月や地球と大きく変わらないことも予想される。したがって、月の場合と同様の資源利用が可能(ヘリウム3を除く)と考えられる。さらに炭素や水素を抽出すると、これらを用いた莫大な種類の有用物質を使えるので、地球上と変わらないレベルの資源利用が期待できる。. 精神世界としての利用月の場合と同様に、火星に対す11―172宇宙機器・システムる畏敬感や神秘性を具現化する手段の一つとして火星を利用することもありうる。
