# 13・4・3 その他の惑星利用の将来構想

> **章**: 第13章 月・火星などへの展開
> **節**: 13・4 惑星利用の将来構想

a. 位置計測利用太陽、地球および月以外の天体を人類が初めて利用したのは、航海における天文航法(celestial naviga-tion)である。これは、星の位置(方位角、仰角)と時間を測定することによって自分のいる位置を導き出す航法である。技術の進歩とともに測定機器の精度や機能は向上しているが、基本的な位置計測の原理は今も変わらない。現在では、人工衛星の位置や姿勢制御を行うために天体(恒星)が使われている(スターセンサ)。将来の太陽系航行においても天文航法が使われる。b. 重力利用月や火星のように、直接その天体表面に降り立ってその場所の環境や資源を利用するという方法以外に、すでに人類が天体を利用している方法がある。それはスイングバイ(swing-by)という宇宙機の航行法における天体の重力利用である。スイングバイは、宇宙機に天体近傍を通過させ、天体の重力を利用してその進行方向や速度を変える航行法である。これによって、効率よく宇宙機をより遠くにより早く送ることができる。すでにいくつかの惑星探査機(ボイジャー、ガリレオ、カッシーニなど)が、金星、地球、木星、土星などの重力を利用して航行している。将来の宇宙活動においてもスイングバイは多く使われるであろう。c. 環境・資源利用火星以遠の天体(惑星、小惑星および惑星の衛星など)は、現在の宇宙機の性能では非常に遠い存在である。したがって、その天体のみに存在する固有の物質を必要とする、あるいはさらなる航行に物資を補給する必要があるような場合以外に、その惑星の環境や資源を利用することが経済的に成立することはほとんどない。将来宇宙機の性能が向上し、より早く、安く、厳しい環境に行くことが可能になれば、高温高圧の金星表面や極寒の外惑星などの環境や資源を利用するようになるかもしれない。文献(1)
