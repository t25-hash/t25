# 14・2・1 方式

> **章**: 第14章 科学気球
> **節**: 14・2 成層圏気球

a. ゼロプレッシャ気球(zero-pressure balloon)気球の底部に浮力ガスの排気孔を設けた気球である。実際には、少し高い位置に孔をあけ、ダクトを底部の位置まで垂らす。総重量を上回る浮力(自由浮力、free lift)を与えられて上昇する気球は、満膨張となった後には自由浮力分のガスを排気孔から排出し、上昇が止まる。この方式の気球は、日没で浮力ガスの温度が低下して浮力が減少すると、高度が下がっても浮力は回復しない。一定高度を保つにはバラストを投下する必要がある。これを日没補償と呼ぶ。その量は、中緯度ではおおよそ総浮力の7~10%となる。このため、ペイロード質量のほとんどすべてをバラストに充てても、1週間ほどの飛翔が限界である。11―173b. スーパープレッシャ気球(super-pressure balloon)排気孔がなく、外気に閉じた気球である。満膨張になっても体積をほぼ一定に保って上昇を続け、大気密度の減少によって浮力が低下し、全荷重と釣り合う高度で水平浮遊に入る。を0.1、気球の高さを50mとすると、圧力比はおおよそ20倍となる。スーパープレッシャ気球はこのような高い圧力に耐えなければならないので、大重量を運搬して高高度まで到達できる気球を、ゼロプレッシャ気球と同等の質量で実現することは容易ではない。この方式の気球は、日没によって浮力ガス温度が低下しても、圧力差が残って気球の体積はほとんど変化しないので、バラストを投下することなく数箇月の長時間飛翔が可能になる。
