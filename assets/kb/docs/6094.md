# 14・2・2 形状設計

> **章**: 第14章 科学気球
> **節**: 14・2 成層圏気球

a. 自然型気球(natural shape balloon)の概念成層圏まで重い観測器を運ぶことのできる大型気球の形状には、ゼロプレッシャ気球とスーパープレッシャ気球のいずれにも、自然型と呼ばれる合理的な形状設計概念が適用されている。その基本的前提条件は、(1)鉛直軸回りの回転対称体、(2)経線方向の長さは一定、(3)フィルムの周方向の長さは必要量より余剰、である。(3)の条件から、フィルムには、余剰分でしわのある周方向には張力が発生せず、経線方向一軸にのみ発生する。このため、形状解析は静定問題(statical determinate)となり、図14・1に示すようにフィルム上の微小部分の力の釣合いを考えればよい。ペイロード重量とフィルム自重量の経線方向成分がフィルムの微小部分に作用する圧力と釣り合うように、フィルムは経線方向にわん曲する。気球形状は、この微小部分の曲率半径の連なりとして求まり、以下のように定式化される。式(14・4)~(14・6)により求めた気球の形状を図14・2に示す。をマイナス、ゼロ、プラスとすると、上昇中の部分的に膨張した気球(#1)、ゼロプレッシャ気球(#2)、スーパープレッシャ気球(#3)の形状が得られる。特にが著しく大きい場合は、経線の長さを一定とする拘束条件のもとで体積が最大となる、オイラーのエラスティカ(Eular’s elastica)と呼ばれる形状(#4)となる。b. ロードテープの導入上記a項で述べた設計概念では、頭部と底部は、経線方向の張力が集中して無限大となる特異点である。この問題を解決するために、図14・1の右半分に示すように、ロードテープと呼ぶ高張力繊維の入った補強テープを経線に沿って等間隔に挿入する。ロードテープは頭部と底部に集中するので、特異点問題は解消する。かつ底部につるされたペイロードの集中荷重は、ロードテープを通じてフィルムの分散荷重へとスムーズに引き渡される。通常のゼロプレッシャ気球は、製作工程の容易さから、14・2・3・a項で述べるように、平面状のフィルムの上に補強テープを取付けた構造となる(平面ゴア法)。その結果、立体形状は図14・3(a)のように、横断面が多角形の形状となり、フィルムには二軸の張力が発生し、静定問題として扱えなくなる。ロードテープによる補強効果も不十分である。c. 三次元ゴア設計法(3-D gore design concept)ロードテープを挿入する効果を最大限に高める方策である三次元ゴア設計法の概念を図14・3(b)に示す。フィルムは、ロードテープの間で伸びによらずに周方向に小さな局所曲率半径をもった膨らみ(バルジ)を形成している。かつ、経線方向にはフィルムに余剰があり、しわが発生しているとする。このような条件を作れば、フィルムの張力は、余剰でしわのある経線方向には発生せず周方向のみとなる。この条件を満たす形状設計は、a項で述べたロードテープがない場合と同一の静定問題となり、形状設計問題は統一的に理解できる。このフィルム張力は、高強度繊維を用いたロードテープの経線方向張力に引き渡される。
