# 14・2・3 製作

> **章**: 第14章 科学気球
> **節**: 14・2 成層圏気球

a. 強度平面ゴア法により製作されるゼロプレッシャ気球では,フィルム張力は二軸方向に複雑に発生し,気球としての実質的強度は同じ容積の球形気球とほぼ同一となる.標準的な例として,容積10万mの気球が高度35km(大気密度0.0085kg/m)で水平浮遊する場合を考える.球に換算した気球の半径はおおよそ30mであり,20 mの気球用フィルムの低温における破断強度は表よそ70Paとなる.一方,気球頭部付近のフィルムの内外圧力差は約5Paとなり,14程度の安全率が得られる.ゼロプレッシャ気球の場合,頭部に加わる圧力は大気圧の高い地上が最も大きく,地上と上空での大気圧の比の三乗根に比例して増大し,概略6倍強となる.地上では気球の頭部にのみガスが詰まった状態であるから,もし気球全体を地上強度に合わせて製作するとむだな重量増をもたらす.そこで,大型気球では頭部のみフィルムを二重,三重にして強度を確保する.三次元ゴア設計法を適用すれば,フィルムの幅が約3mであるから,ゴアの膨らみの周方向局所半径욂は1m程度に小さくできて,気球張力は大幅に減少する.かつ,気球の容積が増大してもゴアの枚数が増加するのみで,局所半径は変わらない.すなわち,気球の耐圧性は気球の容積に依存しない.b. 被膜材料つり下げるペイロード質量が数百kg から1tを超える大型ゼロプレッシャ気球の被膜材料には,厚さ20 m程度のポリエチレンフィルムが用いられる.単位面積当たりの質量は約20g/mである.ペイロード質量が数kg 程度と軽く,40kmを超える高い高度まで上昇することを主目的とする場合には,5 mや3 m強の薄く軽量なフィルムが用いられることもある.フィルムに要求される特性は,十分な強度,高い伸び率,高い引裂強度(tear strength),および脆性温度(brittleness tempera-ture)が十分低いことである.最後の事項は,上層大気が成層境界面(中緯度で約10km)で最も低くなるため,そこを通過する際にも柔軟性を失わない必要があるためである.最近のフィルムでは-100°C程度まで十分耐えられる.代表的なアメリカの大気球用フィルムの特性を表14・1に示す.低温域でも破断までに400%以上という大きな伸び率をもっていることが特徴である.c. 製作工程平面ゴア方式によれば,ゴアと呼ばれる構成要素は,14・2・2・a項で求めた自然型形状を縦に等分した紡錘型形状であり,ロールに巻かれたポリエチレンフィルムから切り出す.次にその縁どうしをロードテープとともに熱接着により接合して製作する.当然ゴアの最大幅は材料としてのフィルムの最大幅で決まる.代表的例として,20 m厚のフィルムを用いた大型気球の諸元を表14・2に示す.三次元ゴア設計法では,ゴアの幅は,膨らんだ状態の局所半径の円弧の長さであり,ゴアの縦の長さは,張り出して長くなった中心線の長さとなり,平面法によるゴアより一回り大きい.その縁の長さは,接合相手であるロードテープより長い.そこで,幾何学上の第III部宇宙空間への進出表14・2 大型気球の諸元例容積m最大直径m長さm質量kg1500050000100000200000図14・4上層風の東西成分の季節変化[図中の数値は風速(m/s).マイナスは東風]関係で求まる短縮率に応じて,フィルムにしわを寄せながらロードテープに接合する.14・2・2・c項で述べた経線方向のフィルムの余剰はこのしわとして与えられる.
