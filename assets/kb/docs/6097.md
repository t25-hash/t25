# 14・2・5 飛翔の安全・保安

> **章**: 第14章 科学気球
> **節**: 14・1 より約1100N/mであるので,気球が破壊する圧力はおお

科学気球は,上昇と飛翔終了時に航空機の運行高度を横切る.また気球バースト等,不測の事態では重量物が落下するおそれがある.このため,安全・保安には十分注意して飛翔させなければならない.航空管制関係の規制は,国際民間航空機関(ICAO: Interna-tional Civil Aviation Organization)が定めた,国際民間航空条約第2付属書の3.1「人身および財産の保護」中の3.1.8で無人自由気球への基本規定が記され,同書の付録-4で細かい条項が定められている.各国とも気球実験では,この規則に準拠して対応している.わが国でも,気球実験に際しては,国土交通省航空局および関連航空機関との事前協議の後,フライト計画を提出する.航空図14・8 恒星赤外線観測望遠鏡図14・9小型有翼飛翔体気球実験装置局はノータムに記載し,担当機関に周知を図り,飛翔中は航空管制の対象とする.海上の安全管理は海上保安庁の管轄であり,事前説明が行われる.おもに,ペイロードの海上着水時の安全対策と回収が対象となる.1971年に海洋汚染および海上災害の防止に関する法律が制定されたことにより,気球本体の海上での回収も義務付けられることとなった.欧米のように,陸上で回収するものも気球本体の回収には努力を払っている.
