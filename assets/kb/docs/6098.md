# 14・2・6 科学気球の利用

> **章**: 第14章 科学気球
> **節**: 14・1 より約1100N/mであるので、気球が破壊する圧力はおお

科学気球の利用分野を大別すると、(1)大気が希薄になったことで宇宙から到来する光や粒子が影響を受ける程度が減少する効果を利用する宇宙線観測、紫外線・赤外線領域の天文観測、(2)希薄になった上層大気そのものを観測対象とする大気物理や地球物理関係の観測がある。また、航空・宇宙工学に関連した実験としては、気球高度からの自由落下による無重力実験や超音速飛翔実験、近似的宇宙環境を利用した衛星搭載機器等の予備飛翔実験等がある。図工学実験の例として小型有翼飛翔体気球実験装置を図14・9に示す。いずれも宇宙科学研究所で飛翔実験が実施された。
