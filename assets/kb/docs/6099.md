# 14・3・1 惑星気球の概要

> **章**: 第14章 科学気球
> **節**: 14・3 惑星気球

a. 惑星気球の意義大気のある惑星には地球上と同様に気球を浮遊させることが可能である。ここでは、地球以外の惑星やその衛星の大気中に浮遊する気球を惑星気球(planetary balloon)と呼ぶことにする。大気をもつおもな惑星および衛星の大気組成、表面における圧力、温度、気球に使用可能な浮揚ガスを表14・3に示す。惑星探査の手段として惑星気球を用いることにより以下のことが可能になる。(1) 数十km以下の接近した位置から惑星表面を高精度に観測(2) 風に乗って惑星の広範囲を探査(3) 大気自体の組成等を直接測定(4) 気象ゾンデのように風の流れを直接測定逆に、飛翔経路や速度は通常風まかせで、その制御は困難である。b. 惑星気球の歴史惑星気球は1960年代から欧米を中心にさまざまな計画が立案、検討されたが、これまでに実際に気球を飛翔させた例は、旧ソ連(旧ソビエト社会主義共和国連邦)、フランスが共同で1985年に実施した2機の金星気球(Venus balloon)、VEGA-1およびVEGA-2のみである。これらの気球は、金星/ハレー彗星フライバイミッションの一部として、金星を過ぎる際に大気中に投入され、高度約54kmに約2日間浮遊させることに成功した。それ以降に実施されたミッションはないが、さまざまな惑星を対象とした種々のアイデアが提案されている。c. 惑星気球の特殊性惑星気球は、地球上の気球と異なり、表14・3 大気のあるおもな惑星と衛星惑星・衛星大気(表面または圧力10Pa相当の値)地球1.0×10N(77%)、O(21%)29.0H,He火星7.0×10CO(95%)、N(3%)43.5H,He昼夜の温度差大木星1.0×10H(90%)、He(10%)2.2H土星1.0×10H(97%)、He(3%)2.1Hタイタン1.5×10N(92%)、Ar(6%)28.6H,He低温天王星1.0×10H(83%)、He(15%)2.6H海王星1.0×10H(85%)、He(13%)2.6H探査機により運ばれ、通常、耐熱カプセルに収納された状態で目的とする大気に投入される。このカプセルの空力減速後にふたを開き、気球本体を伸張し、浮揚ガスにより膨張させ、最終的にカプセル、減速装置等の不要装置を切り離し、目標高度に浮遊させる。こうした過程は自律的に行われる必要がある。各惑星の大気環境(温度、圧力、組成)に応じた特殊な気球用被膜、気球の浮遊方式、浮揚ガスの選択をする必要がある。地球上の気球でも困難な数週間の長時間飛翔の実現も求められることから、スーパープレッシャ型気球が望ましい。木星型惑星のように水素が主成分の大気では、熱気球型になる。
