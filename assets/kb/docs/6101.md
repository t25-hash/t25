# 14・3・3 金星気球

> **章**: 第14章 科学気球
> **節**: 14・3 惑星気球

a. 金星の大気二酸化炭素を主成分とする金星大気には、高度50~70kmにかけて硫酸を含む雲があり、地表を覆い隠していて、これより上からの可視領域での観測はできない。大気圧、温度、密度はそれぞれ、表面で755K,9.2MPa,65kg/m、高度20kmで580K,2.2MPa,20kg/m、高度50kmで地球上と同程度の0.1MPa,350K,1.6kg/mである。金星の自転周期は243日と非常に長いが、風は東から西に非常に強く、雲の最上層部、高度70kmで100m/s以上、高度35kmで40m/sもの風が定常的に吹いている。これは4日で循環するスーパーローテーション(super rotation)と呼ばれる金星大気のなぞの一つである。この風により非常に広範囲を短時間で飛翔することができる。b. 金星気球(Venus balloon)の条件金星大気が濃密であるため、高度を低くとればかなりの小容積の気球でも十分な浮力(十分な搭載能力)が得られるが、その分高温環境となり、気球被膜の耐熱性が要求される。また、長時間の浮遊には耐硫酸性を考慮しなければならない。c. 高高度膨張型気球(VEGA)VEGA-1およびVEGA-2はランダから切り離された後、ヘリウムガスによって膨張させられ、赤道付近(それぞれ北緯7.1°および南緯6.5°)の真夜中側に浮遊した。その高度54kmの圧力540hPa、温度305K は地球表面とほぼ同じである。2機の気球は直径3.4mのテフロン製の球形スーパープレッシャ型気球である(標準圧力差30hPa)。浮遊総質量21kg のうち観測器とテレメトリシステムを含むゴンドラ(図速、後方散乱、太陽照度が測定された。信号は地球とフライバイモジュールの両方で受信され、地球上のアンテナを動員したVLBI(very long baseline interferometry)により気球の位置と速度が求められ、スーパーローテーションが確認された。1kg polymers)等がある。雲の下であることから、可視領域や高精度の近赤外線観測も可能である。VEGA VEGA のゴンドラや搭載機器の耐熱限界を超える。例えば、高度35kmに総浮遊質量10kg の気球を浮遊させるのに必要な水の量は4.3kg である。これを1時間程度で気化する必要がある。これまでの熱交換器を用いる方式に対し、水をあらかじめ気球内面に吸着させておき、降下中に周囲の高温大気を直接利用する方式が提案されている。この方式の気球投入シーケンスを図14・11に示す。大気からの熱流入を大きくするため、アスペクト比の大きい円筒形気球が適する。e. 高度可変方式例えば、水を用いると高度42km以下では気体となり浮力が増加し、42km以上の高度では液化して浮力を失う。このような性質を利用すれば、適当なバルブ操作により気球浮遊高度を制御することが可能となる。このために、大型の熱交換器を搭載する。表面まで降下してサンプルを採集する計画も提案されている。f. ドロップゾンデ方式高高度に浮遊している気球から複数のドロップゾンデを投下する方式である。気球はゾンデのプラットホームであると同時にゾンデと地球間の通信の中継を行う。g. 低高度金属球方式金星大気は濃密であるため、高度20km以下の高温高圧中では、小型の気球で十分な浮力が得られる。耐熱フィルムによる膨張型気球の代わりに、最初から浮揚ガスを封入した薄い金属球の気球を運搬する方式が適用可能である。高温、硫酸大気環境下でも気球の強度やガスバリア性に問題がなく、長期間飛翔が可能である。ただし、一重の気球では最初から球形高圧容器として運ばなくてはならないため、気球本体重量が増加する。これを大幅に減らすため、気球本体は保護のため耐圧容器に入れて運び、浮遊高度で分離させる二重カプセル方式が考案されている。例えば、直径1mのチタン製の気球は厚さが0.1mm程度で強度が保たれ、気球の質量は2kg 程度となる。高度13kmに飛翔させるとすると浮遊総質量は17kg で、浮揚ガスを除いた13kg の機器を搭載可能である。
