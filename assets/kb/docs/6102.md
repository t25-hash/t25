# 14・3・4 その他の気球

> **章**: 第14章 科学気球
> **節**: 14・10 )の質量は6.5kg である.周囲の大気圧,温度,相対鉛直風

a. タイタンタイタンでは,気球は降下中に膨らませることも,ランダで降下後に膨らませて上昇させる方式もとれる.低温で図14・11 金星気球の展開浮遊十分な性能を有する被膜(例えばテフロン)が必要となる.ヘリウムとアルゴンを用いた高度可変方式やガイドロープ付き気球が提案されている.b. 木星型の惑星水素が主成分なので浮揚ガスとしては水素以外にない.または,熱気球のように加熱するか,気球被膜が赤外線を吸収して上昇する方式(infrared Montgolfiere balloon)が考えられている.
