# 14・3・4 その他の気球

> **章**: 第14章 科学気球
> **節**: 14・10 )の質量は6.5kg である。周囲の大気圧、温度、相対鉛直風

a.タイタン タイタンでは，気球は降下中に膨らませること
も，ランダで降下後に膨らませて上昇させる方式もとれる．低温で
十分な性能を有する被膜（例えばテフロン）が必要となる．ヘリウ
ムとアルゴンを用いた高度可変方式やガイドロープ付き気球が提案
されている．
b.木星型の惑星 水素が主成分なので浮揚ガスとしては水素
以外にない．または，熱気球のように加熱するか，気球被膜が赤外
線を吸収して上昇する方式（infrared Montgolfiere balloon) が
考えられている．
14・3・5その他の課題
通信手段としては，惑星軌道上の衛星を利用する方法のほかに，
地球との直接通信も可能である．例えば，VEGAでは4.5Wの送
信出力で地球と4bpsの通信を行った．
金星のような高温大気中では，エレクトロニクスの動作温度が問
題となる．220℃程度まで使用可能な深地中探査やカーエレクトロ
ニクス用素子が利用できる．これ以上の温度や高温に耐えられない
素子がある場合には，冷却が必要となる．これには，物質の相変化
を利用した吸熱方式と電子冷却がある．高圧の場合は耐圧容器によ
る保護が必要になる場合もある．
長期間飛翔・観測するための電源には，通常，太陽電池が使用さ
れるが，このほかに気球の高度変更に伴う鉛直風を利用した風力発
電 も考えられている．
