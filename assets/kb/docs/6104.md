# 15・1・1 宇宙環境利用

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・1 쓕きぼう」の利用概説

放送・通信・観測・測位等における宇宙の開発および利用は,前世紀終盤には本格化し,また実用化されている.これらを目的とした人工衛星は,いまや多くの人々がその恩恵に浴し,日常生活において不可欠なインフラストラクチャとなっている.人々は毎日,衛星放送を楽しみ,当然のように気象衛星の画像データを目にしている.測位衛星による全地球測位システム(GPS : global position-ing system)は,カーナビゲーションや携帯電話に広く採用され,生活に深く浸透している.これらの宇宙利用は,宇宙空間という位置を利用したものである.すなわち,地表に対して高高度であること(目的により低・中高度軌道の400~500kmから静止衛星軌道の約3万6000kmまで)が本質的な要件であり,この高度を得るために,宇宙空間を利用している.これに対し,微小重力(microgravity)を代表とする宇宙空間ならではの環境を直接的に利用するという宇宙開発利用の概念は,一般に宇宙環境利用と呼ばれる.1971年,アメリカがアポロ14号で行った対流実験や複合材料製造実験が,微小重力環境を利用した最初の宇宙実験である.これは1970年代のスカイラブ計画に引き継がれ,1980年代には,スペースシャトルによる安定した宇宙飛行機会の確保により大きく拡大した.これらアメリカの計画は,通常2週間,長くても3週間程度という,いわば短期集中型の計画であったのに対し,旧ソ連(旧ソビエト社会主義共和国連邦,現ロシア)では,先駆的な宇宙ステーションであるサリュート宇宙船(1970年代)やミール(1980~1990年代)等による,人間の長期宇宙滞在を含めた宇宙環境利用の開拓が行われてきた.1984年,アメリカは,レーガン大統領(当時)の呼びかけで宇宙ステーションの概念検討に着手し,1987年,西側先進国14箇国(アメリカ,ヨーロッパ11箇国,カナダおよび日本)が参加する国際共同プロジェクトである宇宙ステーション計画が始まった.その後,旧ソ連の崩壊,東西対立構造の終焉という劇的な国際情勢の変革を経て,またアメリカの経済状況の悪化も相まって,ロシアもこの計画に参加するという計画変更(1995年)が行われ,名称も国際宇宙ステーション(ISS : International Space Station)計画と改められた.1986年のスペースシャトルチャレンジャー号の打上げ時の事故や2003年のコロンビア号の帰還時の事故等,有人プログラムであるがゆえの悲劇的な歴史を乗り越え,宇宙環境利用はISS の利用により,21世紀初頭にいよいよ本格化の段階を迎えつつある.
