# 15・1・2 きぼう」により獲得される環境

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・1 きぼう」の利用概説

9章に詳述されているように、「きぼう」(Kibo、またはJEM :Japanese Experiment Module)は日本が供給するISS の構成要素である。ISS により長期的に得られる特異な宇宙環境として、微小重力環境、高度・クリアな視界、真空、宇宙放射線環境等が挙げられる。a. 微小重力環境人工衛星等、地球を周回する飛翔体は地球に対し自由落下を続けることとなり、飛翔体自身の質量による引力が十分に小さければ、理想的にはその飛翔体の中では重力の加速度を感じない環境が得られる。ISS は比較的低軌道であり、また、それ自身が110×75m,450tという大構造物であることから、残留する大気によるドラッグや重力傾斜(潮汐力)の効果により、完全な重力零とはならない。図の範囲で、10~10(1=980cm/s、地表での重力の加速度)の重力環境となる。また、これら直流に近い成分の0からのずれ(準静的加速度)以外に、高周波の重力擾乱として、クルーの活動、太陽電池パネルや放熱板のロータリジョイント等のISS 本体の動作による振動による擾乱(-ジッタ)を受ける。微小重力環境は、「きぼう」により得られる代表的な環境であり、基礎物理化学、物質科学、材料工学、生命科学等の多くの宇宙環境利用実験はこれを活用することを目的としている。微小重力環境下では、実質的に重さがなくなることから、密度差による沈降・浮遊が起こらず、また対流が抑制される。これらは、一様均質な材料の生成、拡散や表面張力等の現象と、対流との混成を分離しての精密観察等に活用される。また、微小重力下では、流体の自重による静水圧が抑制され、地上重力下ではmmオーダの深度差による静水圧差が擾乱となるような、臨界点近傍流体の物理現象の巨視的スケールでの観測が期待される。さらに、無容器での物質位置制御が容易となり、深い過冷却状態の実現や高純度材料の創生等に利用される。一方、地上のさまざまな環境に適応しながら進化してきた生命にとって、1という地表の重力環境は一貫して変わらないものであったが、この重力という基本パラメータに対する生命の反応メカニズムの解明は、微小重力環境の獲得によって初めて可能となる。一般的に生命現象は時定数が比較的大きいため、特に「きぼう」により得られる長期間の微小重力環境は貴重である。この生命と重力の関係は、生命現象の基礎的な知見としても、また将来の人類の宇宙への進出にとっても大変重要なテーマである。b. は共用プラットホームであり、特定の観測ミッションに最適化された衛星とは異なり、軌道、姿勢、あるいは視野等からの制約から、観測に対する適合性に考慮を必要とする。ISS は軌道傾斜角51.6°、高度約400kmの円軌道で地球を約90分で周回する。しかし、降(昇)交点通過地方時や太陽との相対位置の同期がとれているわけではない。また、陸域のほとんどをカバーしているものの、両極域を直下に観測することは不可能である。したがって、地球観測範囲あるいは観測時点に対する考慮が必要である。ISS はトルク平衡姿勢(TEA : torque equivalent attitude)と呼ばれる、空力および重力傾度の外乱に対して力学的に平衡な姿勢で飛行する。この状態は、微小重力環境提供のためにアメリカ要素にあるジャイロにより維持・制御されるが、飛行経路下方に対する姿勢のずれの許容値は前後左右とも15°から20°ある。したがって、ISS から配信される姿勢や姿勢変動レート等の精度も考慮してミッションを計画する必要がある。このような条件から、姿勢実験装置ごとに固有の姿勢検出装置を備えるなどの対策をとるとしても、狭視野で精密な指向を要する観測ミッションに対しては限界があるといわざるを得ない。一方、広視野型の天体観測や指向要求が0.1°程度である高エネルギー宇宙線観測ミッション等には有効な利用が期待できる。ISS
