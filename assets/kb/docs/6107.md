# 15・1・2 ・b項に述べたように,共用

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・1 に示すように,ISS 重心周り,飛翔方向を中心としただ円柱

プラットホームであるISS は,軌道・姿勢の安定度の問題から,比較的狭い視野で特定の天体を高精度にポインティングするという一般的な天体観測には適合性が低い.全天X 線監視装置(MAXI:monitor of all sky X-ray image)は,スリットカメラを用いた全天スキャンにより,この観測ミッションにとって致命的ともいえる欠点を回避した,X 線放射天体の全天モニタ観測のミッションである.MAXIの概観を図15・6に示す.瞬時視野角が約1°×180°の扇形を成すスリットカメラが,前方(反PIU 側,図15・6中では左向き)および上方に向けて設置され,これがISS の周回に合わせ,全天球を走査する.カメラには,X 線センサとしてガス比例計数管とX 線CCD が組込まれ,おのおの2~30keV および0.5~10keV のエネルギー領域のX 線の検出を行う.天体からのX 線放射は,大気の吸収のない宇宙での観測が必須であり,これまでも多くの観測衛星が打ち上げられてきた.MAXIは,広視野モニタ観測としては従来にない高感度(~1mCrab:カニ星雲のX 線強度の1000分の1)で観測を行い,従来われわれの銀河の中までであったモニタ観測の範囲を,外銀河にまで飛躍的に拡大する.これにより,X 線放射天体のダイナミックなふるまいと,活動的な銀河の分布を調べることができ,ひいては宇宙の大構造や起源・進化の解明に大きく寄与するものと期待されている.. 超伝導サブミリ波リム放射サウンダ超伝導サブミリ波リム放射サウンダ(SMILES : superconducting submillimeter-wave limb-emission sounder)は,オゾン層破壊および地球温暖化の原因となる成層圏大気中の微量気体の三次元グローバル観測を可能とする,超伝導を利用したサブミリ波放射観測の宇宙における利用実証を行うミッションである.本装置および実験テーマの詳細図15・6 全天X 線監視装置[比例計数管とX 線CCD によるスリットカメラが,飛行方向前方(図中左側)および天頂方向(図中上方)に設けられている]11―184宇宙機器・システムは15・4節に詳述されているので,そちらを参照されたい.. 宇宙環境計測センサ宇宙環境計測センサ(SEDA :space environment data acquisition equipment)は,ISS 軌道における中性子,高エネルギー軽粒子,重イオン,ダスト,原子状酸素,プラズマ等の宇宙環境計測を行い,また,ISS 周辺の環境が部品・材料にどのような影響を与えるかについてのデータを取得する.SEDA は,これら複数のセンサを一つの船外実験ペイロードとして,標準エンベロープ内に搭載したミッションである.各搭載センサについては,提案者グループによるホームページ(SEES :Space Environment and Effects System)に詳しい.宇宙インフラストラクチャの開発を進めるうえで,宇宙機器設計の基礎データとして,宇宙放射線などの宇宙環境に関する実用的なモデルを構築し,これを広く提供していくことの意義は大きい.また,ISS 搭載機器や実験装置などがさらされている宇宙環境を定量的に把握することは,実験データの評価,機器設計への反映を行ううえでも重要である.本提案は,これらに必要な宇宙環境データを各種センサで総合的に取得するものである.また,材料ばく露および電子部品評価装置では,宇宙用新材料の劣化特性や電子部品の誤動作等のデータ蓄積を図ることができる.さらに,関連する科学研究やISS の運用に有用なデータとして用いることが期待される.得られたデータおよびそれにより構築された環境モデルは,ホームページに紹介されている.
