# 15・2・3 細胞培養装置の主要機能・性能

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・8 に示す。

細胞培養装置の物理的特性は以下のとおりである。軌道上質量:G/1G 部145.6kg制御部46.56kg付属品(キャニスタ) 約1kg/個寸法:G/1G 部1243×482×660(H×W×D) mm制御部310×482×660(H×W×D) mmここで、重力の加速度は通常で表すが、本装置の部分名称としてG/1G を通称としたため、この表示となっている。また、実験に関する性能は以下のとおりである。(1) 実験環境実験研究者の要求する温度、湿度、炭酸ガス濃度を下記の範囲で制御することができる。温度:15~40°C湿度:最大80±10Rh%炭酸ガス濃度:0~10vol.%(2) 加重力環境1G 部には回転式の人工重力発生器を搭載しており、微小重力と図15・7 CBEF 概観図15・8 CBEF 実機1とを直接対比し、微小重力影響を抽出するための対照実験を実施できる。なお回転数は20~140minで、2負荷相当までの過重力実験も可能である。(3) キャニスタ個別の実験に対して標準的なインタフェース(機械的、電気的)を与えるアルミニウム合金製の箱である。実験者は装置から供給される電力(DC,5,12,-12,15V)、アナログセンサ出力、ビデオ出力、シリアル通信の各機能を利用してキャニスタ内に自由に実験システムを構築することができる。軌道上で宇宙飛行士がキャニスタをCBEFに装着する作業を行い、実験は地上からのコマンドで自動実施することで、省力化を可能にした(15・2・9項参照)。(4) 自動実験運用機能実験計画に合わせた実験環境制御、キャニスタ内の個別実験装置へのコマンド出力などをユーザプログラムとして格納し、自動的に装置の運用を行うことが可能である。また、キャニスタ内部に格納した個別実験機器については、別途RS-485規格により外部に設置するLT(ラップトップコンピュータ)と接続することができ、LT を利用して実験システムを制御することができる。
