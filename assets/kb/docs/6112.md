# 15・2・4 主要構成要素とその仕様

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・8 に示す。

細胞培養装置の構成は図15・7で示したように、実験を遂行するG/1G の各部、キャニスタを装着するためのG トレーと1G ターンテーブル、および全体機能を制御しつつJEM、地上とデータを送受信する制御部からなる。ほかに、安全確保のための消火用ホールやのぞき窓が用意されている。a. インキュベータ部CBEFのインキュベータ部分は、軽量でかつ十分な構造強度をもつという打上げのための要求と、高い断熱性と気密性をもつという実験性能の要求および実験ラックとのインタフェース要求を同時に満たすために、図15・9に示すような設計を採用している。各要素の特徴は以下のとおりである。(1) 断熱ボデーインキュベータ部の外壁を構成し、保温性と構造強度を有する。アラミドハニカムをガラス繊維強化プラスチック(GFRP: glassfiber reinforced plastics)で挟んだハニカム構造のパネルと、アルミニウムフレームの補強材で構成される。ドア部分にはパッキンを図15・9インキュベータ部構成要素11―185第III部宇宙空間への進出設け、気密性を保てるよう大型のハンドルによりしっかりとロックできる構造となっている。(2) フレーム構造強度と剛性を確保する。フロントパネル、リヤフレームおよび補強フレームのそれぞれをボルト結合して構成している。(3) 培養部回転テーブルを有する1G 部および回転テーブルを有しないG部からなる、インキュベータ部の内側の領域である。内容積はそれぞれ70Lおよび60Lであり、両者の実験環境が均質となるように内部でエアの流通性をもたせてある。(4) ドロワコネクタパネル本実験装置は後方にQD(quick disconnector)結合のインタフェースをもち、実験ラックに装着することでリソースの供給を受ける。ドロワコネクタパネルはリヤフレームとボルト結合されており、SAIBOラック(15・2・6項参照)との機械的、電気的IFを受け持つ。(5) 温度調節器ペルチェ方式のヒートポンプである。JEM 側とのIFはコールドプレートによる低音冷却水への排熱。インキュベータ側にはシロッコファンによる空気循環を行い、インキュベータ内部の温度を制御する。(6) 加湿器、CO調整機、かくはんファン加湿器は自動シャッタによる開閉制御、COガスはJEM より供給されるので調圧弁制御を行い、インキュベータ内部の実験環境を制御する。(7) 各種センサ機能のフィードバック制御のために、温度センサ、湿度センサ、CO/Oセンサ、圧力センサを装備している。b. 制御部・駆動ユニット制御部はインキュベータ部から独立した筐体で、インキュベータ部とのデータ通信、電圧変換(JEM 本体から供給される120VDCを、装置自身やキャニスタで使う低電圧の直流に変換する)、JEM 本体のシステムとの通信、装置の制御、データ表示などを行う。駆動ユニットはインキュベータ部の後部の構造材に取付けられており、制御部からの指令を受けて、インキュベータ内部のアクチュエータの駆動制御を行う。11―186宇宙機器・システム図15・10 SAIBOラック
