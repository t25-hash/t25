# 15・2・5 ソフトウェア

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・8 に示す。

装置のソフトウェアは制御部に格納されており、三つの部分から構成されている。(1) BIOS(basic input output system)アプリケーションソフトウェアからサブルーチンとして呼び出され、ハードウェアの制御および通信等を行う。書換えは不可。(2) アプリケーションソフトウェアBIOS の機能を利用して、培養部の実験条件制御、実験データバスを通じての地上との通信制御、キーパッド、ディスプレイの入出力制御およびユーザプログラムの制御を行う。専用外部機器から書換えが可能。(3) ユーザプログラムアプリケーションソフトウェアに定義されているコマンドを用いて、実験個別にシーケンスを記述し、実行することができる。実験データバスを経由してアップロードすることが可能。
