# 15・2・6 ほかの実験装置との関係

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・8 に示す.

CBEFと連携して実験を実施する他の機器として,以下のものがある.(1) クリーンベンチ(CB : clean bench)クリーンベンチは,無菌操作のためのグローブボックス型の装置で,CBEFで培養した細胞の化学固定など手動の実験操作および位相差・蛍光顕微鏡観察ができる.図15・10に示すように,CBEFとCB は同じ実験ラック(SAIBOラック)に搭載され,一体の実験装置として運用される.図15・10中右の装置がCBEF,左がクリーンベンチである.(2) 冷凍冷蔵庫(MELFI: minus eighty degree Celsius labo-ratory freezer for ISS)生物試料を冷蔵または凍結して,ISS で保存するための装置である.開発担当はヨーロッパの宇宙機関(ESA).(3) 放射線計測機器(PADLES : passive dosimeter for life-science experiment in space)生物試料に影響を及ぼす宇宙放射線を計測する機器で,固体飛跡検出器(CR-39)と熱蛍光線量計を組合わせて積算の吸収線量とLET を測定できる受動型の線量計である.
