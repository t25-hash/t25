# 15・3・1 温度こう配炉(GHF)の概要

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・3 材料科学実験装置(温度こう配炉)

温度こう配炉(GHF: gradient heating furnace)は,微小重力下での半導体材料の結晶成長や気相(物質が気体となっている状態)成長などを調べるための実験装置であり,試料の加熱冷却プロセスを実施する炉体部,GHFの運用全体の制御および「きぼう」との通信等を行う制御装置,最大15本の試料を自動で交換可能な試料自動交換機構と,その制御装置から構成される多目的真空炉である.炉体部は,おのおの独立に制御可能な三つの加熱室をもち,これで実験要求に応じたさまざまな温度プロファイルを提供し,試料の溶融・一方向凝固(容器内の原料融液を一端から凝固させる)を行うことが可能である.本装置の特徴は,宇宙飛行士の軌道上での作業を軽減するために,試料自動交換機構によって実験ごとに自動的に試料を交換できるようにしている点である.また,加熱冷却中の試料温度測定用に,最大10本の熱電対を試料に設置することが可能である.図15・13に温度こう配炉の搭載状態を示す.a. 炉体部炉体部は,温度こう配炉の電気炉ユニットである.内部に抵抗加熱方式の三つの加熱室を有し,加熱室はそれぞれ独立に温度制御を行うことができる.一般的には,補助室で低温側(凝固側)の温度保持を行い,中央室および端部室の高温側(加熱溶融側)で試料の溶融状態を保持し,低温側と高温側のすきまの位置を制御することで試料の温度こう配を制御する使い方が想定される.同時に,低振動モータによりそれぞれ独立に軸方向移動させることが可能であり,実験要求に応じて,加熱室内の試料近傍の制御用熱電対を使用し,実験試料に対し多様な温度パターンの設定や,溶融試料の一方向凝固時に必要な移動加熱が実施できる.また,加熱室内部のヒータエレメント(抵抗)は,長時間真空状態での高温加熱に耐えうるために特殊コーティングを施してあり,その周囲を多層断熱フォイルで覆うことで,加熱室内部の温度均一性を確保する構造となっている.炉体部の構成を図15・14に示す.また,試料封入部最大径がφ36.1mmまでの大型試料カートリッジの取扱いを想定しているため,これまでの宇宙用材料実験装置に比して大型サンプルの加熱溶融プロセスが実施可能である.なお,作動寿命が制限されている加熱室や,加熱室の温度制御に不可欠な熱電対は軌道上での交換作業が可能な設計となっており,「きぼう」での長期運用中に故障が発生しても,ミッション遂行が阻害されない設計となっている.炉体部内部に設置された加熱室を図15・15に示す.図15・13 試料自動交換機構試料自動交換機構は15本の試料を搭載できるマガジン部と,マガジン上の所定の位置から試料を取出し,炉体部に試料を挿入・取出しを行うアーム部,および炉体部に挿入されたカートリッジボス部で炉体部チャンバを密封するクランプ部から構成される.また,アーム部はカートリッジを把持するハンド部,水平方向に動作させる水平アーム部,および上下方向に動作させる上下アーム部からなる.以上五つの駆動系は,レゾルバを位置検出素子としたモータ駆動のサーボで作動する.図15・15 試料自動交換機構用制御装置実験要求に基づき,試料自動交換機構に保管した試料を炉体部の真空チャンバへ搬入,搬出を行うように試料自動交換機構を駆動する.試料自動交換機構各部には,それぞれの動作方向の上限下限にマイクロスイッチが設置されている.試料自動交換機構の状態管理には,このマイクロスイッチの組合せを本制御装置が管理し,コマンドに対する動作可否を決める.また,軌道上での故障発生に対応して,本体をラックから引き出すことなく,ボード単位で交換することができるように設計されている.交換方法を図15・21に示す.e. 試料温度こう配炉を使って実験を行う際は,試料そのものを直接加熱すると溶融試料がヒータに接触してショートを引き起こし,炉体部を破壊してしまう.そのため,試料をるつぼに封入した状態で金属製カートリッジ(外層)に設置することで,温度こう配炉で加熱できるようにする.試料およびるつぼを内部に搭載した金属製カートリッジを,試料カートリッジと総称する.図15・22に試料カートリッジの概略図を示す.図15・19試料自動交換機構図15・20
