# 15・3・2 温度こう配炉の基本仕様

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・3 材料科学実験装置(温度こう配炉)

温度こう配炉を使用する際に必要な,基本仕様を以下に示す.a. 加熱温度範囲補助室:500~716°C中央室:500~1600°C端部室:500~1600°Cb. 温度安定性上記加熱最高温度で1時間保持したときの温度変動幅が±0.2°C.c. 温度こう配1450°Cにおいて150°C/cm(試料によって異なる).d. 温度設定精度上記加熱最高温度で,±0.4%以内.e. 加熱室移動距離および移動速度150mm以上,0.1~200mm/h.この際の移動速度安定性は±1%となる.ただし,0.1~10mm/hの精動モードでは±10%となる.f. 試料カートリッジサイズ直径36.1mm以下,長さ505mm以下.g. 温度計測点試料カートリッジ内に熱電対を設置することにより,標準タイプで5点の計測が可能.ただし,オプションとして最大10点まで拡大可能.h. 圧力モニタ真空チャンバ内に,ダイアフラム圧力計,ピラニゲージ真空計,イオンゲージ真空計の3種類を設置し,負圧から正圧までを連続的に計測可能.
