# 15・4・2 ミッション機器

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・4 船外プラットホームの利用(超伝導サブミリ波

a. 高精度サブミリ波アンテナ大気放射の観測システムへの入力部は小型のサブミリ波アンテナ(オフセットカセグレン形式,主鏡はアルミニウム製で400×200mmのだ円形)である.高度分解能は主鏡の口径で決まる(約0.1°)が,観測データの精度保証のために,サイドローブの低減,熱変形誤差の低減,回転角駆動精度等が重要である.また,システム全体の低雑音化のために,極低温に冷却された小型フィードホーンをアンテナ給電系としている.軌道上で大きく変化する温度環境下でアンテナの精度を維持するため,光学系全体をアルミニウム製とし,温度変化に対してこれを相似変形させる新しい設計手法を用いている.b. 低雑音サブミリ波受信機サブミリ波受信機は,サブミリ波光学系,超伝導ミキサ(周波数変換器),サブミリ波局発(基準信号),低雑音の中間周波増幅器,4K 級の極低温冷凍機等から構成されている(図15・25).冷凍機は,2段型スターリング冷凍機とジュール・トムソン冷凍機を組合わせたものである.受信機の信号入力部においては,サブミリ波減衰を極力減らすために,収束鏡を組合わせた準光学的伝送路が用いられている.他方,サブミリ波帯の準光学的伝送路は精度の高いアライメントを必要とする(典型的には位置で0.1mm,角度で0.05°程度の精度).SMILES ではシステム構体の中にアライメントパネルと称する基準パネルを設定し,アンテナおよび受信機の共通の構造的基準として対処している.図15・25 低雑音サブミリ波受信機の内部構造図15・24 SMILES が観測する成層圏大気の予想スペクトル(リム観測における接線高度への依存性を示す)11―19 0宇宙機器・システムc. 広帯域マイクロ波分光計分子種の存在量はその輝線スペクトルから求めることができる.このために周波数の関数として受信電力を分析する必要があり,マイクロ波分光計(microwavespectrometer)が用いられる.これにはいくつかの種類があるが,SMILES では広帯域性と高分解能を兼ね備えた方式として音響光学型分光計(AOS : acousto-optical spectrometer)を用いる(図つ,大気放射のパワースペクトルを検出するもので,瞬時帯域1200MHzをリアルタイムで約1500チャネルに分解・検出することができる.この場合,サブミリ波信号の検出感度σは,システム雑音温度읎윶읎を用いて,σ=읎윶읎τで与えられる(は各チャネルの検出バンド幅,τは積分時定数).읎윶읎を決める大きな要因はサブミリ波受信機の雑音レベルであり,超伝導センサを使えば,この値を常温の半導体センサに比べて数十倍小さくすることができる.SMILES のそれは約500K である.
