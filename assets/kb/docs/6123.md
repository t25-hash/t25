# 15・4・3 システム設計

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・26 )。これは、電波⇒音響波⇒光、と情報の担い手を変換しつ

a. 構造設計SMILES の構造は、「きぼう」船外プラットホームを利用するため、ISS への日本の輸送機であるHTV(H-IItransfer vehicle)への搭載条件、「きぼう」上で所定の実験ポートに移動するためのRMS(remote manipulator system)とのインタフェース条件、「きぼう」の実験ポートに固定し、電力、排熱、通信のサービスを受けるためのインタフェース条件等を満たさなければならない。HTV に結合する4本の脚部、RMS による把持機構、クイックカップリング方式による「きぼう」本体との結合機構等がそれである。SMILES の直方体形の構造は、最大許容エンベロープ(1850×1000×800mm)に対応している(図15・27)。ミッション要求に従い、アンテナ視線方向の高精度決定、アンテナとサブミリ波受信機との高精度光学アライメント等が設計に盛り込まれた。また、ISS 環境の高い電磁界レベルからSMILES の微弱な観測信号(中間周波数11~13GHz および1.55~2.75GHz)を守るために、システム構体パネルや受信機クライオスタット(いずれもアルミニウム製)には高い電磁シールド機能が施されている(両者で63dB 以上)。b. 熱設計SMILES の熱設計は、冷媒(フロリナートFC-図15・26 広帯域マイクロ波分光計図15・27 SMILES システムの全体構造72)による機器発熱の排出と、多層断熱材(MLI:multi-layerinsulation)等によるふく射熱および放射冷却の制御の2本柱で成り立っている。「きぼう」本体から供給されるフロリナートの温度は16~24°Cで制御されている。SMILES の極低温冷凍機は、極低温ステージ(4,20,100K)から常温部へ熱をくみ出すポンプ(冷媒はヘリウムガス)であるが、一定の冷却能力を得るために必要な消費電力は常温部(クライオスタット外壁およびヘリウム圧縮機)の温度に依存する。そこで、冷凍機常温部とフロリナートとの間に介在する熱抵抗を可能な限り小さくする工夫を施すことで、冷凍機の効率的運用を保証した。また、受信機器の利得や周波数の安定化のために、必要に応じてフロリナート配管に熱キャパシタ等を付加し、機器温度の安定化を図っている。c. 安全設計ISS は有人施設であるため、特別の安全設計が要求される。ペイロード構造の破損をはじめ、ヘリウムガスおよびフロリナートの圧力システムの破裂、予期しないアンテナ動作やシャープエッジの存在による宇宙飛行士への損傷等の可能性に対して、2FT(2個の独立故障が発生しても安全が保たれること)のレベルでの安全設計を行っている。
