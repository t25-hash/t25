# 15・4・4 観測データ処理

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・26 )。これは、電波⇒音響波⇒光、と情報の担い手を変換しつ

a. 軌道上データ校正大気の信号はアンテナで集められ、受信機による周波数変換と増幅の後、マイクロ波分光計で多数の周波数チャネルに分割・検出されディジタル化される。ディジタル出力はこれらすべての機器の性能パラメータ(利得、雑音の周波数特性等の装置関数)の影響を受ける。大気の輝度スペクトルを正確に評価するためには、基準となる白色雑音(常温の黒体ふく射源からのサブミリ波雑音)を定期的に受信機に入力し、機器パラメータの変動を校正する必要がある。軌道上のSMILES は、約1分周期で地球大気を高度方向に走査するとともに、受信電力強度と受信周波数の校正を行う。輝度スペクトルデータの発生量は約110kbpsとなる。b. 地上データ処理輝度スペクトルからO,ClO,HCl等の分子種ごとに大気中の混合比(高度分布)を求めるのは、地上データ処理の役割である。混合比の先験値ベクトル(高度分布)と観測ジオメトリーおよび装置関数を用いて予想される輝度スペクトルを計算し(フォワードモデル)、これを観測された輝度スペクトルと比較することで、より正確な混合比ベクトルを推定する(この処理をリトリーバルと呼ぶ)。具体的には、フォワードモデルを先験値ベクトルの近傍で線形化し、混合比ベクトルの微小な変化に対応する輝度スペクトルの変化を係数マトリックスとして求め、これを用いて混合比ベクトルを推定する方法が一般的である[ロジャーズ(Rodgers)法].その際、フォワードモデルの正確さが混合比の推定精度を左右するのは当然だが、サブミリ波観測では通常、観測システムの雑音による測定誤差が大きな比重を占める。超伝導センサを用いるSMILES の場合は、この誤差が非常に小さく、短時間に取得されたデータからも高い精度で微量分子の混合比を決定できるのが特徴である。宇宙ステーションは軌道上の実験室であり、長期間の微小重力状態が得られる。このことに着目して、日本実験モジュール「きぼう」与圧部や、ヨーロッパモジュールColumbus(コロンバス)などに設けられたさまざまな実験装置を利用して微小重力科学実験が計画されている。微小重力状態は図15・28に示すように、四つの要素現象によって特徴付けられる。すなわち、無静圧、無対流、無浮力・無沈降、無接触浮遊である。これらの結果として、物体の自重による変形の減少、物体における熱伝達の変化、物体中の物質分布の変化、物体における物質輸送の変化、界面現象の顕著化が生じる。微小重力環境において顕著化する現象は、これらのいずれか、あるいはその組合せによって説明できる。これまでに、スペースシャトル、小型ロケット、航空機の放物線図15・28 微小重力環境の特徴図15・29溶融スズの自己拡散飛行、落下実験施設など、さまざまな手段を用いて微小重力実験が実施され、先端科学としての優れた研究成果を上げてきている。宇宙ステーション上では、これまでの手段とは異なり極めて長時間の微小重力状態を利用した実験が提案されている。ここでは、これまでに実施された代表的な微小重力科学実験の例として熱物性測定、無容器材料プロセス、結晶成長、タンパク質結晶成長、マランゴニ対流および燃焼について述べ、長時間にわたる微小重力環境が得られる宇宙ステーション利用の意義について紹介する。ついで、ライフサイエンス分野および宇宙医学分野の研究についても併せて述べる。
