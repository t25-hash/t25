# 15・5・1 項に述べた微小重力科学と比べて新しい分野である。生物

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・5 与圧部の利用

科学の分野は対象とする領域が広く、個体レベル、器官レベル、細胞レベルおよび遺伝子レベルに至る生命現象の各階層において研究の対象が存在する。宇宙ステーションの利用は、微小重力環境、放射線環境、ならびに宇宙空間・閉鎖環境の利用としてとらえられる。微小重力環境の視点からは、重力感受性に関する研究、重力の効果に関する研究、生命科学の応用としてのバイオテクノロジーに関する研究が対象となっている。放射線環境ならびに宇宙空間・閉鎖環境としての利用の視点からは、宇宙環境と生命の関わりの解析が研究の対象となる。この分野は、従来は観察と記述が中心であったが、観察される現象を体系化していく研究が進展していくものと期待される。
