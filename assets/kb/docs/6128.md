# 15・5・3 宇宙医学

> **章**: 第15章 国際宇宙ステーション「きぼう」の利用例
> **節**: 15・5 与圧部の利用

宇宙医学(space medicine)の分野では,基礎医学分野の研究と臨床応用を視野に入れた研究が計画されている.前者は,基礎科学的な側面を重視した基礎医学関連を中心とした研究,ならびにヒトの生物学に関する生命科学現象としてのバイオメディカル研究である.後者は,骨からのカルシウムの溶け出しや尿路結石など,人体の宇宙環境への適応や,宇宙から地上に帰還した際の再適応を支援する研究など,宇宙飛行士の活動を支援する応用的側面が強調された研究として遂行が予定されている.文献(1) Penley, N., Microgravity Environment on ISS, (1999), SpaceStation Utilization Advisory Subcommittee (SSUAS, 口頭報告).(2) 高柳昌弘・ほか2名,信学技報,SANE98-103(1998),27.(3) 松岡勝,日本航空宇宙学会誌,48 (2000),455.(4) 宇宙航空研究開発機構のホームページ,http://sees.tksc.jaxa.jp/(5) 吉冨進, 電子情報通信学会宇宙航行エレクトロニクス研究会技術研究報告(信学技報),SANE2002-22(2002),19.11―19 3第III部宇宙空間への進出(6) Inatani,J.,ほか7名,Proc.
