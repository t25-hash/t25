# 16・2・1 基本的構造

> **章**: 第16章 テザーシステム
> **節**: 16・2 ひも付き衛星システム(TSS : shuttle/tethered sat

ひも付き衛星に関する大がかりな実験はスペースシャトル(space shuttle)によって行われたので、ここではこれに従い説明を加える。先端に付けられた衛星は図16・1に示されるように、(a) スペースシャトル内のテザーシステムの位置(b) テザーシステムの詳細図16・1 ひも付き衛星システムの基本的な構造第III部宇宙空間への進出スペースシャトルの上空にも、あるいは地球に向かっても伸展されうる。実験の目的によって衛星にはいろいろな型のものが考えられるが、先端衛星の高度の下限は空力加熱を考えると120kmであろう(後出図16・6参照)。衛星は、ブーム(長さ20~50m)の先端に取付けられている衛星捕獲のためのアダプタに固定されている。衛星とシャトルをつなぐテザーは直径約1mmで、合成繊維を材料としている。ブームの目的は、シャトルからの衛星放出時に衛星がスペースシャトルに巻き付くのを防ぐことと衛星回収である。システムはテザーの巻取りをするリールと、テザーの張力、長さおよびテザーの繰出し速度を検出するセンサ、テザー伸展、巻取り用の駆動モータより構成される。計算機によって必要な張力が常時テザーに与えられるように駆動モータが制御される。スペースシャトルを用いた上記のシステムは、特に大気科学研究のための有用な道具となろう。
