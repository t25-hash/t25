# 16・2・2 ひも付き衛星の運動

> **章**: 第16章 テザーシステム
> **節**: 16・2 ひも付き衛星システム(TSS : shuttle/tethered sat

ひも付き衛星の安定化は振り子の原理で説明される.もし,スプリングマス(spring mass)のパラメータが伸縮振動の周波数が揺れの周波数の2倍に等しいように調節されると,揺れのエネルギーは急速に伸縮振動のエネルギーに変換される(図16・2).伸縮振動の抑制は,テザーの伸展,引張込みを機械的に行うことにより実現する.テザーの伸展は,まずブームの先に付けた衛星を母船(一例としてスペースシャトル)より離れたところにおくか,母船上で母船の真下に向けて(今地球に向けて伸展する場合を考えると)初速度を与えることである.前者の場合はスペースシャトルと地球との重力差,衛星と地球の重力差を利用したことになるが,いずれの場合にも衛星はスペースシャトルより少し前に動くことになる[図16・3(a)].スペースシャトルより少し前に動いた時点でリールによって張力を加えると,シャトルよりいくぶん前にある衛星にはつねに衛星を減速させるような力が働くことになり,これがさらに衛星高度を下げることになる.もし,衛星をスペースシャトルより上方(地球から離れる方向)に伸展させたければ,初期の伸展速度は上向きで,衛星はスペースシャトル上空のやや後ろに位置するようになる.伸展を行う必要な張力の計算は,衛星の安定化に使われるのと同じ計算機でなされる.この場合,テザーに働く張力は衛星に働く重力こう配よりいくぶん小さくなければならない.張力が大きいと衛星は回収される方向に動いてしまう.テザーの伸展の間,テザー伸展および巻取り機構は発電機として働くので,擬似的負荷に消費させるか,実際に使うことができるであろう.衛星の回収は,前述のように衛星高度での平衡張力より大きな張力をテザーに加えること以外,衛星の伸展と似ている.このより大きな張力によって衛星はスペースシャトルに向かって引き戻されることになる.図16・3(b)に示すように,回収時における衛星は,スペースシャトルの下方にある場合はスペースシャトルの真下よりやや後方,スペースシャトルの上方に位置する場合にはスペースシャトルの真上よりやや前方に位置する.11―19 5回収時と伸展時における違いの一つは,望ましい軌道からのずれを修正するのに,伸展時より回収時のほうが難しく,より長い時間が必要とされることである.特に,この段階では,衛星のテザーが衛星に絡みつく可能性のあることが指摘されている.回収の最終段階では回収はゆっくりと行われ(~10cm/s),衛星捕獲は伸展ブームの捕獲機構によって行われる.その後,ペイロードベイの中にブームが引き寄せられ全行程が終了する.
