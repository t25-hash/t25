# 16・2・3 技術的検討

> **章**: 第16章 テザーシステム
> **節**: 16・2 ひも付き衛星システム(TSS : shuttle/tethered sat

現在までに行われてきた工学的な研究はおもに二つである。一つは衛星の最低高度に関するもの、もう一つはこれに一部関連するテザー張力(tether tension)の限界である。図16・4に示すように、質量200kg の衛星は100kmの長さのテザーの場合で100N(ニュートン)の重力差を生じることになる。テザーの長さが半分になれば、400kg の質量の衛星が同じ張力を生じる。点線は直径1mmのステンレス線に安全係数2.6をかけたときのステンレス線の耐えうる張力で、これによると、重力こう配のみを考えると、200kg の衛星を、高度200kmを飛ぶスペースシャトルから100kmの高度に保つことができることになる。もう一つ考えなければならない重要な要素は大気の空気抵抗(atmospheric drag)である。図16・5に示すように、200kmの高度を飛ぶスペースシャトルから高度120kmまで下ろされた直径1mmのテザーに働く大気抵抗は18N であるが、高度100kmまで下ろされると、大気による抗力は100N を超えてしまうことになる。したがって、テザーの引張強さは衛星の降下可能な高度を決める重要な要素である。現在、考えられている有力なテザー材料(a) 衛星伸展(b) design)に慎重な考慮を必要とすることになる。図16・6に示すように、球状衛星の外壁の温度は高度100kmで約600°Cに達する。したがって、衛星内の温度を低く保つために液体窒素を用いることも考えられる。例えば、衛星内温度を1週間、30°Cに保つためには50kg の液体窒素が必要とされる。空力加熱(aerodynamic heating)の影響はまたテザー材料の選定にも及ぶ。例えばアラミドを使った場合、800°Cで溶けてしまうので、テザーの先端のみを熱に強い、またいくぶん大きな直径の線にするなどの考慮が必要となる。図16・7に衛星の伸展-回収の1サイクルの軌跡が示されている。テザーの最高繰出し速度が約10m/sのとき、100km長のテザーの伸展-回収に約19時間が予定されている。これには大気抗力の効果は考慮されていないが、大気抗力の効果を入れると、図16・8に示すように、揺れ角が時間によって違うとともに衛星はさらに後方に押されることになる。高度が低くなるにつれて、1日の大気密度変動による揺れ運動は大きくなり、このことと空力加熱とが、ひも付き衛星の到達しうる最低高度を決めることになる。初期の研究はおもに衛星そのものの運動に限られているが、テザーそのもののふるまいがどうなるかは興味ある問題である。テザーの運動のふるまいを調べた最初の計算はクラ(Kulla)によってなされた。テザーを10m/sの一定速度で送り出すと、テザーは最終的にはスペースシャトルに衝突してしまうことになる。張力を制御しながら繰出すテザーの様子を比較のために示した。カラガン図16・7 大気の抗力を考慮したときの衛星伸展時における衛星軌道面における衛星の運動(スペースシャトルの高度200km)(Kalagan)はテザーをスペースシャトル下方に100km伸ばし、先端を放した状態からテザーがどのようにふるまうかを見た。大気研究者が希望している高度120km付近の研究のための衛星計画においては、小型衛星を搭載する母船からのテザー伸展から始まって衛星回収に至るまでのテザーのふるまいが計算されなければならない。テザーは、軽量、柔軟でありテザー方向への大きな慣性モーメントをもつことが構造上の特徴である。これにより重力場の中で安定な系を構成できる。この特徴を利用することにより、軌道上の軽量柔軟構造物(flexible structure)、姿勢安定システム、高品質重力場実験室、運動量変換システムへの工学的な応用が可能である。導電性のテザーを使用する場合は、地球磁場との電気力学的な作用を利用した発電や軌道制御システムに応用することができる。
