# 16・3・1 軽量柔軟構造物

> **章**: 第16章 テザーシステム
> **節**: 16・3 システムの工学への応用

軌道上の大型構造物は、国際宇宙ステーションに見られるように、現在ではトラスがその主要な構造部材として使用されている。宇宙ステーションの次の世代のより大型の構造物では、軽量で大きな構造を実現できるテザーが主要な役割を果たすと考えられる。地上でも大型構造物は、つり橋や大型ドームなどの例に見られるようにワイヤがその構造における主要な役割を果たしている。またテザーは、軌道上の施設本体から離して保持することが望ましい危険物貯蔵庫などの係留の手段としても有用である。この場合16・2・2項で示すように、軌道上の係留物は鉛直下方または上方に伸展した状態で安定する。テザーは剛性が低く構造体との接合部での運動の自由度が大きいため、トラスなどの剛性の高い構造物と比較して機械的な擾乱に対して耐性が高いという利点がある。
