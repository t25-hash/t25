# 16・3・2 姿勢安定システム

> **章**: 第16章 テザーシステム
> **節**: 16・3 システムの工学への応用

軌道上の構造物の重心付近では重力と遠心力が釣り合うため無重量状態と近似されるが、重心から離れた場所では重力と遠心力が釣り合わないため、大型構造物では実用上無視できない力(重力こう配力)が働く。この力は図16・9に示すように、テザーで結ばれた二機の衛星に対して鉛直上方または鉛直下方に作用する。重力こう配力は、テザーの長さ方向が鉛直方向からずれた場合には鉛直方向に復元する作用があるので、この力を利用して宇宙構造物の姿勢安定(attitude stability)を実現することができる。例えば図16・10に示すように、送電アンテナを常時地球方向に指向することが必要な太陽発電衛星、上下方向への物資の輸送システム、静止衛星資源を有効に使用するため多数の衛星を連結するシステムなどが応用として考えられている。
