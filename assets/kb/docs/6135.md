# 16・3・4 運動量変換システム

> **章**: 第16章 テザーシステム
> **節**: 16・3 システムの工学への応用

テザーで接続した二機の衛星を鉛直方向上下に展開させてもシス(a) テザー方式太陽発電衛星(b) テザーエレベータ(c) 連結衛星図16・10 重力安定を利用したミッションの例11―19 7第III部宇宙空間への進出テム全体の軌道角運動量は変化しないが、図16・13に示すように上方に伸展した衛星の角運動量は増加し、下方の衛星の角運動量は減少する。したがって、円軌道のシステムの場合、テザー伸展後にテザーを切断すると、上方の衛星は遠地点がより高い軌道に遷移し、下方の衛星は近地点がより低い軌道に遷移する。これを利用して、より高い軌道への衛星の投入や帰還船の大気圏再突入を行うことができる。
