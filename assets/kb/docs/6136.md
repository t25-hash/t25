# 16・3・5 電気力学的テザーシステム

> **章**: 第16章 テザーシステム
> **節**: 16・3 システムの工学への応用

テザーとして導電性のワイヤ(conductive wire)を使用することにより、テザーシステムによる発電および軌道制御(orbitmaneuver)を行うことができる。図16・14に示すように、地球磁場を横切って運動する導体は起電力を発生する。ワイヤの両端でのプラズマとのインピーダンスを低くすれば、宇宙空間プラズマを通して回路が形成され、電力を取得できる。この場合、テザーには軌図16・11 高品質の低重力場実験室の概念図16・12 可変重力場実験室の概念図16・9テザーシステムに働く重力こう配力図16・13 テザーを利用した運動量変換システム(a) 発電(b) 加速図16・14 エレクトロダイナミックテザーの概念11―19 8宇宙機器・システム道運動を減速する方向に力が働き、軌道高度は下降する。エネルギーの観点からは、テザーシステムの軌道運動のエネルギーが発電モータと同様な電気的作用で電気エネルギー(electric energy)に変換されると解釈される。一方、テザーシステム内の電源を用い起電力に抗してテザーに電流を流すとテザーには軌道方向への力が発生し、テザーシステムは加速され、軌道高度は上昇する。この場合は駆動モータと同様な電気的な作用で電気エネルギーが軌道運動のエネルギーに変換されることになる。これらの電気力学的機能をもつテザーはエレクトロダイナミックテザーと呼ばれ、宇宙ステーションの軌道維持や大型宇宙残留物の大気圏突入手段として注目されている。エレクトロダイナミックテザーの作用を有効にするためには、テザーの両端と周辺プラズマとのインピーダンスをできるだけ小さくする必要がある。その方法として、テザー端とプラズマとの接触面積を大きくするという受動的な方法以外に、電子ビーム発生装置やプラズマコンタクタと呼ばれるプラズマの発生装置の使用が考案されている。導電性のテザーは低周波(ULF/ELF/VLF)の通信アンテナとして使用することができる。非常に長い波長の電波を使用すれば、電離層のダクト効果により世界中との同時通信が可能である。また長い波長の電波は、従来減衰が大きくて困難であった地下や水面下への通信を可能にすると期待されている。
